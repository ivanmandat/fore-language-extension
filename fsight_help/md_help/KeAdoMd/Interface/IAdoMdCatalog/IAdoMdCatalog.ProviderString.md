# IAdoMdCatalog.ProviderString


## Синтаксис


ProviderString: String;


## Описание


Свойство ProviderString определяет
 дополнительные параметры подключения к серверу БД.


Примечание.
 Для баз данных, созданных на базе SAP NetWeaver BW в данном свойстве должен
 указываться номер пользователя, под которым осуществляется подключение.
 Номер пользователя задается в следующем формате: "SFC_CLIENT=<Номер>".
