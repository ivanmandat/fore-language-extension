# IAdoMdCatalog


## Описание


Интерфейс IAdoMdCatalog содержит
 свойства и методы объекта репозитория - [Каталог ADOMD](UiNavObj.chm::/ADOMD/UiDb_relational_ADOMD.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Authentication](IAdoMdCatalog.Authentication.htm)
		 Свойство Authentication
		 определяет тип аутентификации пользователя, используемый при подключении
		 к серверу БД.


		 ![](../../Property_Image.gif)
		 [AutoRefresh](IAdoMdCatalog.AutoRefresh.htm)
		 Свойство AutoRefresh
		 определяет, будет ли производиться автоматическое обновление структуры
		 кубов.


		 ![](../../Property_Image.gif)
		 [Credentials](IAdoMdCatalog.Credentials.htm)
		 Свойство Credentials
		 определяет учётные данные пользователя.


		 ![](../../Property_Image.gif)
		 [Driver](IAdoMdCatalog.Driver.htm)
		 Свойство Driver определяет
		 идентификатор драйвера, используемого при открытии каталога ADOMD.


		 ![](../../Property_Image.gif)
		 [LoginPrompt](IAdoMdCatalog.LoginPrompt.htm)
		 Свойство LoginPrompt
		 определяет, будет ли при открытии каталога ADOMD выводиться диалог
		 ввода учетных данных.


		 ![](../../Property_Image.gif)
		 [Name](IAdoMdCatalog.Name.htm)
		 Свойство Name определяет
		 наименование базы данных, к которой осуществляется подключение
		 при открытии каталога ADOMD.


		 ![](../../Property_Image.gif)
		 [ProviderString](IAdoMdCatalog.ProviderString.htm)
		 Свойство ProviderString
		 определяет дополнительные параметры подключения к серверу БД.


		 ![](../../Property_Image.gif)
		 [Server](IAdoMdCatalog.Server.htm)
		 Свойство Server определяет
		 наименование сервера, на котором расположена база данных, к которой
		 подключается каталог ADOMD при открытии.


		 ![](../../Property_Image.gif)
		 [UseMetabaseCredentials](IAdoMdCatalog.UseMetabaseCredentials.htm)
		 Свойство UseMetabaseCredentials
		 определяет, будут ли для подключения к базе данных использоваться
		 учетные данные, указанные при входе в репозиторий.


		 ![](../../Property_Image.gif)
		 [WellKnownDriver](IAdoMdCatalog.WellKnownDriver.htm)
		 Свойство WellKnownDriver
		 определяет драйвер, используемый при открытии каталога ADOMD.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Open](IAdoMdCatalog.Open.htm)
		 Метод Open осуществляет
		 открытие каталога ADOMD.


		 ![](../../Sub_Image.gif)
		 [OpenWithCredentials](IAdoMdCatalog.OpenWithCredentials.htm)
		 Метод OpenWithCredentials
		 осуществляет открытие каталога ADOMD с указанными учетными данными.
