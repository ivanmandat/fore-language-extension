# IAdoMdCatalog.LoginPrompt


## Синтаксис


LoginPrompt: Boolean;


## Описание


Свойство LoginPrompt определяет,
 будет ли при открытии каталога ADOMD выводиться диалог ввода учетных данных.


## Комментарии


По умолчанию свойству установлено значение False,
 при этом запрос не выдается. Если свойству установить значение True,
 то при открытии будет выводиться диалог ввода учетных данных. Указанные
 учетные данные будут использованы для подключения к базе данных, на которую
 настроен каталог ADOMD.
