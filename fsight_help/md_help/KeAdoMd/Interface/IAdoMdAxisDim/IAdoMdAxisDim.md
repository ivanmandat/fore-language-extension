# IAdoMdAxisDim


Сборка: AdoMd;


## Описание


Интерфейс IAdoMdAxisDim содержит
 свойства для получения информации об измерении, которое используется при
 формировании оси.


## Иерархия наследования


IAdoMdAxisDim


## Комментарии


Информацию об измерении можно получить в свойстве [IAdoMdAxis.Dim](../IAdoMdAxis/IAdoMdAxis.Dim.htm).
