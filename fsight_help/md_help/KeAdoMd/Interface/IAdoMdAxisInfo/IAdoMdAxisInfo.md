# IAdoMdAxisInfo


## Описание


Интерфейс IAdoMdAxisInfo содержит
 свойства для работы с коллекцией осей, по которым сформированы данные
 при выполнении MDX-запроса.


## Комментарии


Данные, полученные при выполнении многомерного запроса, сформированы
 в виде среза. Для идентификации ячейки с данными в срезе выделяются оси,
 пересечение которых и является координатой ячейки. Каждая ось в свою очередь
 формируется из отдельных измерений, атрибуты которых описывают координаты,
 доступные на оси.


Информацию об осях можно получить в свойстве [IAdoMdCellset.AxisInfo](../IAdoMdCellset/IAdoMdCellset.AxisInfo.htm).
