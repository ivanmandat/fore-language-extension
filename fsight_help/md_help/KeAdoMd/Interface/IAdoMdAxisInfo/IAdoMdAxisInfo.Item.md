# IAdoMdAxisInfo.Item


## Синтаксис


Item(Index: Integer): [IAdoMdAxis](../IAdoMdAxis/IAdoMdAxis.htm);


## Параметры


Integer. Индекс оси.


## Описание


Свойство Item возвращает информацию
 об оси, индекс которой передается в качестве входного параметра.
