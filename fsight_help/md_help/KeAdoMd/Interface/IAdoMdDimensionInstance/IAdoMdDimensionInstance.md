# IAdoMdDimensionInstance


## Описание


Интерфейс IAdoMdDimensionInstance
 содержит свойства для работы с открытым экземпляром справочника ADOMD.


## Комментарии


Для получения открытого экземпляра куба ADOMD выполните для соответствующего
 объекта репозитория метод [IMetabaseObjectDescriptor.Open](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.Open.htm)
 либо [IMetabaseObjectDescriptor.OpenWithParam](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.OpenWithParam.htm).
