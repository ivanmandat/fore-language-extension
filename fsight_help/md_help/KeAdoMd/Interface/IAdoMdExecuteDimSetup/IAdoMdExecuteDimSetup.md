# IAdoMdExecuteDimSetup


Сборка: AdoMd;


## Описание


Интерфейс IAdoMdExecuteDimSetup
 содержит свойства для указания дополнительных настроек измерения, которые
 будут использоваться при расчете куба.


## Иерархия наследования


IAdoMdExecuteDimSetup


## Комментарии


Дополнительные настройки могут быть получены в свойстве [ICubeExecuteDimSetup.Extension](KeCubes.chm::/Interface/ICubeExecuteDimSetup/ICubeExecuteDimSetup.Extension.htm) путем
 приведения значения этого свойства к интерфейсу IAdoMdExecuteDimSetup.
