# IAdoMdDimCalculation


Сборка: AdoMd;


## Описание


Интерфейс IAdoMdDimCalculation
 содержит свойства для настройки параметров вычисляемого элемента измерения.


## Иерархия наследования


IAdoMdDimCalculation


## Комментарии


Параметры вычисляемого элемента можно получить, используя свойства или
 методы интерфейса [IAdoMdDimCalculations](../IAdoMdDimCalculations/IAdoMdDimCalculations.htm).
