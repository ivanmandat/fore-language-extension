# IAdoMdDimCalculation.Element


## Синтаксис

 Integer;


## Параметры


Dimension. Измерение, элемент
 которого необходимо указать.


## Описание


Свойство Element определяет
 элемент в измерении, для которого создается вычисляемый элемент.
