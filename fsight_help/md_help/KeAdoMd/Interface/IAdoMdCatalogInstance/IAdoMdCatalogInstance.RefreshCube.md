# IAdoMdCatalogInstance.RefreshCube


## Синтаксис


RefreshCube(CubeName: String; [Cube: [IAdoMdCube](../IAdoMdCube/IAdoMdCube.htm)
 = Null]);


## Параметры


CubeName. Наименование куба,
 структуру которого необходимо обновить;


Cube. Экземпляр куба, структуру
 которого необходимо обновить.


## Описание


Метод RefreshCube обновляет
 структуру указанного куба.


## Комментарии


Метод позволяет обновить структуру куба с указанными наименованием или
 для указанного экземпляра, если он получен.
