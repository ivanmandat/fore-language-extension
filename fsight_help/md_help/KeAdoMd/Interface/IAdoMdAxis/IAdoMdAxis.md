# IAdoMdAxis


## Описание


Интерфейс IAdoMdAxis содержит
 свойства и методы для получения информации об оси, по которой формируются
 данные при выполнении MDX-запроса.


## Комментарии


Каждая ось представляет собой набор координат, которые используются
 для идентификации ячеек с данными. Оси сформированы с помощью [измерений](../IAdoMdDimension/IAdoMdDimension.htm),
 атрибуты которых описывают каждую координату оси.


Информацию по какой-либо оси возвращает свойство [IAdoMdAxisInfo.Item](../IAdoMdAxisInfo/IAdoMdAxisInfo.Item.htm).

		 [Count](IAdoMdAxis.Count.htm)
		 Свойство Count возвращает
		 количество координат, доступных на оси.


		 ![](../../Property_Image.gif)
		 [Dim](IAdoMdAxis.Dim.htm)
		 Свойство Dim возвращает
		 информацию об указанном измерении.


		 ![](../../Property_Image.gif)
		 [DimCols](IAdoMdAxis.DimCols.htm)
		 Свойство DimCols возвращает
		 количество атрибутов указанного измерения, характеризующих каждую
		 координату оси.


		 ![](../../Property_Image.gif)
		 [DimCount](IAdoMdAxis.DimCount.htm)

		 количество измерений, используемых для формирования отдельных
		 координат на оси.


		 ![](../../Property_Image.gif)
		 [DimName](IAdoMdAxis.DimName.htm)
		 Свойство DimName возвращает
		 наименование измерения, которое задействовано для формирования
		 отдельных координат на оси.


		 ![](../../Property_Image.gif)
		 [Rowset](IAdoMdAxis.Rowset.htm)
		 Свойство Rowset возвращает
		 курсор с информацией о координатах, доступных на оси.
