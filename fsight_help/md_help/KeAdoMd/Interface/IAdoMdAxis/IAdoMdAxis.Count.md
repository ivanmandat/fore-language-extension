# IAdoMdAxis.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 координат, доступных на оси.


## Комментарии


Описание координат можно получить в курсоре, который возвращает метод
 [Open](IAdoMdAxis.Open.htm).


## Пример


Пример использования приведен в описании метода [Open](IAdoMdAxis.Open.htm).
