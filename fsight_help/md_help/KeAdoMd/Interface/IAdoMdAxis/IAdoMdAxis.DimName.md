# IAdoMdAxis.DimName


## Синтаксис


DimName(Index: Integer): String;


## Параметры


Index. Индекс измерения.


## Описание


Свойство DimName возвращает
 наименование измерения, которое задействовано для формирования отдельных
 координат на оси.


## Комментарии


Общее количество измерений можно получить в свойстве [DimCount](IAdoMdAxis.DimCount.htm).
