# IAdoMdAxis.Rowset


## Синтаксис


Rowset: [IDalCursor](Dal.chm::/Interface/IDalCursor/IDalCursor.htm);


## Описание


Свойство Rowset возвращает курсор
 с информацией о координатах, доступных на оси.


## Комментарии


Свойство возвращает курсор, аналогичный курсору, который возвращает
 метод [IAdoMdAxis.Open](IAdoMdAxis.Open.htm).
