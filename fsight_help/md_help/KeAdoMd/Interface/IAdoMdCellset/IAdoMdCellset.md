# IAdoMdCellset


## Описание


Интерфейс IAdoMdCellset содержит
 свойства для работы с аналитическими данными, полученными в результате
 выполнения MDX-запроса к кубам на сервере.


## Комментарии


Для получения аналитических данных выполните метод [IAdoMdConnection.Cellset](../IAdoMdConnection/IAdoMdConnection.Cellset.htm).
