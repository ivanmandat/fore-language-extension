# IAdoMdDimension.Hierarchy


## Синтаксис


Hierarchy: String;


## Описание


Свойство Hierarchy определяет
 наименование иерархии уровней, которая используется в справочнике ADOMD.


## Комментарии


Иерархия определяет какие уровни будут доступны в справочнике. На разных
 уровнях располагаются данные различной степени детализации, уровни используются
 при настройке агрегации и при моделировании.
