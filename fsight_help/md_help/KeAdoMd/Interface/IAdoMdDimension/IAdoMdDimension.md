# IAdoMdDimension


## Описание


Интерфейс IAdoMdDimension содержит
 свойства и методы для работы со структурой справочника ADOMD.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Catalog](IAdoMdDimension.Catalog.htm)
		 Свойство Catalog возвращает
		 родительский каталог, в котором хранится справочник ADOMD.


		 ![](../../Property_Image.gif)
		 [Cube](IAdoMdDimension.Cube.htm)
		 Свойство Cube определяет
		 куб по умолчанию, с которым связано текущий справочник ADOMD.


		 ![](../../Property_Image.gif)
		 [Cubes](IAdoMdDimension.Cubes.htm)
		 Свойство Cubes возвращает
		 коллекцию кубов, в структуру которых входит текущий справочник
		 ADOMD.


		 ![](../../Property_Image.gif)
		 [Hierarchy](IAdoMdDimension.Hierarchy.htm)
		 Свойство Hierarchy
		 определяет наименование иерархии уровней, которая используется
		 в справочнике ADOMD.


		 ![](../../Property_Image.gif)
		 [NativeName](IAdoMdDimension.NativeName.htm)
		 Свойство NativeName
		 определяет физическое имя справочника ADOMD на сервере.


		 ![](../../Property_Image.gif)
		 [Type](IAdoMdDimension.Type.htm)
		 Свойство Type определяет
		 тип справочника ADOMD.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Refresh](IAdoMdDimension.Refresh.htm)
		 Метод Refresh обновляет
		 структуру справочника ADOMD в репозитории на основании информации
		 о справочнике, полученной с сервера.
