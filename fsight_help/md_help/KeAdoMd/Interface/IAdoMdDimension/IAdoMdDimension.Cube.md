# IAdoMdDimension.Cube


## Синтаксис


Cube: String;


## Описание


Свойство Cube определяет куб
 по умолчанию, с которым связан текущий справочник ADOMD.


## Комментарии


Свойство возвращает первый куб из [списка
 кубов](IAdoMdDimension.Cubes.htm), в состав которых входит данное измерение.
