# IAdoMdDimension.NativeName


## Синтаксис


NativeName: String;


## Описание


Свойство NativeName определяет
 физическое имя справочника ADOMD на сервере.


## Комментарии


Физическое имя уникальным образом идентифицирует измерение на сервере,
 может состоять из латинских букв и цифр. Физическое имя измерения на сервере
 может не совпадать с наименованием или идентификатором соответствующего
 объекта репозитория.


Примечание.
 В «Форсайт. Аналитическая платформа»
 изменение физического имени справочника ADOMD из прикладного куба не доступно.
