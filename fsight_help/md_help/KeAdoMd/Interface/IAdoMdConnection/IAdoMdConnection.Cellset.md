# IAdoMdConnection.Cellset


## Синтаксис


Cellset(Text: String): [IAdoMdCellset](../IAdoMdCellset/IAdoMdCellset.htm);


## Параметры


Text. Текст
 MDX-запроса для извлечения аналитических данных.


## Описание


Метод Cellset возвращает аналитические
 данные, полученные в результате выполнения MDX-запроса к кубам на сервере.

Пример использования приведен в описании свойств [IAdoMdCellset.AxisInfo](../IAdoMdCellset/IAdoMdCellset.AxisInfo.htm),
 [IAdoMdCellset.Cells](../IAdoMdCellset/IAdoMdCellset.Cells.htm).
