# IAdoMdDimCalculations


Сборка: AdoMd;


## Описание


Интерфейс IAdoMdDimCalculations
 содержит свойства и методы для работы с коллекцией вычисляемых элементов
 измерения.


## Иерархия наследования


IAdoMdDimCalculations


## Комментарии


Параметры коллекции могут быть получены в свойстве [IAdoMdExecuteDimSetup.Calculations](../IAdoMdExecuteDimSetup/IAdoMdExecuteDimSetup.Calculations.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IAdoMdDimCalculations.Count.htm)
		 Свойство Count возвращает
		 количество вычисляемых элементов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IAdoMdDimCalculations.Item.htm)
		 Свойство Item возвращает
		 параметры указанного вычисляемого элемента.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IAdoMdDimCalculations.Add.htm)
		 Метод Add создает новый
		 вычисляемый элемент и возвращает его параметры.


		 ![](../../Sub_Image.gif)
		 [AddFormula](IAdoMdDimCalculations.AddFormula.htm)
		 Метод зарезервирован на будущее.


		 ![](../../Sub_Image.gif)
		 [Clear](IAdoMdDimCalculations.Clear.htm)
		 Метод Clear удаляет
		 все вычисляемые элементы из коллекции.


		 ![](../../Sub_Image.gif)
		 [FindById](IAdoMdDimCalculations.FindById.htm)
		 Метод FindById осуществляет
		 поиск вычисляемого элемента по идентификатору и возвращает его
		 параметры в случае удачного поиска.


		 ![](../../Sub_Image.gif)
		 [Move](IAdoMdDimCalculations.Move.htm)
		 Метод Move осуществляет
		 перемещение вычисляемых элементов в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IAdoMdDimCalculations.Remove.htm)
		 Метод Remove удаляет
		 вычисляемый элемент по его индексу в коллекции и возвращает результат
		 удаления.


		 ![](../../Sub_Image.gif)
		 [RemoveById](IAdoMdDimCalculations.RemoveById.htm)
		 Метод RemoveById удаляет
		 вычисляемый элемент по его идентификатору в коллекции и возвращает
		 результат удаления.
