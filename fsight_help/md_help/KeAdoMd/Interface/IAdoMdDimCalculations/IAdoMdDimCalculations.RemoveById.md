# IAdoMdDimCalculations.RemoveById


## Синтаксис


RemoveById(Id: String): Boolean;


## Параметры


Id. Идентификатор удаляемого
 вычисляемого элемента.


## Описание


Метод RemoveById удаляет вычисляемый
 элемент по его идентификатору в коллекции и возвращает результат удаления.


## Комментарии


Метод возвращает значение True,
 если удаление прошло удачно и False,
 если удаление по каким-либо причинам невозможно.
