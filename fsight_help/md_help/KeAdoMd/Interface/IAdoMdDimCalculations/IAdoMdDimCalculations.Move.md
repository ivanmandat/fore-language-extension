# IAdoMdDimCalculations.Move


## Синтаксис


Move(IndexFrom: Integer; IndexTo: Integer);


## Параметры


IndexFrom. Индекс позиции,
 из которой необходимо переместить вычисляемый элемент.


IndexTo. Индекс позиции, в
 которую необходимо переместить вычисляемый элемент.


## Описание


Метод Move осуществляет перемещение
 вычисляемых элементов в коллекции.
