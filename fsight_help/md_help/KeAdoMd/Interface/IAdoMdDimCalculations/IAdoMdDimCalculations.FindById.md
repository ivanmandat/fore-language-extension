# IAdoMdDimCalculations.FindById


## Синтаксис


FindById(Id: String): [IAdoMdDimCalculation](../IAdoMdDimCalculation/IAdoMdDimCalculation.htm);


## Параметры


Id. Идентификатор вычисляемого
 элемента, по которому осуществляется поиск.


## Описание


Метод FindById осуществляет
 поиск вычисляемого элемента по идентификатору и возвращает его параметры
 в случае удачного поиска.


## Комментарии


Метод возвращает значение Null,
 если вычисляемый элемент с указанным идентификатором не существует.
