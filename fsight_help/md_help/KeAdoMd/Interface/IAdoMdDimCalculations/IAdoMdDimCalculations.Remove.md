# IAdoMdDimCalculations.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемого вычисляемого
 элемента.


## Описание


Метод Remove удаляет вычисляемый
 элемент по его индексу в коллекции и возвращает результат удаления.


## Комментарии


Метод возвращает значение True,
 если удаление прошло удачно и False,
 если удаление по каким-либо причинам невозможно.
