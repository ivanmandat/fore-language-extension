# IAdoMdDimCalculations.Add


## Синтаксис


Add(Id: String): [IAdoMdDimCalculation](../IAdoMdDimCalculation/IAdoMdDimCalculation.htm);


## Параметры


Id. Идентификатор вычисляемого
 элемента.


## Описание


Метод Add создает новый вычисляемый
 элемент и возвращает его параметры.
