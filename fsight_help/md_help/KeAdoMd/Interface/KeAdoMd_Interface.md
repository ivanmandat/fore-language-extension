# Интерфейсы сборки AdoMd


		 Интерфейс
		 Краткое описание


		 ![](../Interface_Image.gif)
		 [IAdoMdAxis](IAdoMdAxis/IAdoMdAxis.htm)


		 Интерфейс IAdoMdAxis
		 содержит свойства и методы для получения информации об оси, по
		 которой формируются данные при выполнении MDX-запроса.


		 ![](../Interface_Image.gif)
		 [IAdoMdAxisInfo](IAdoMdAxisInfo/IAdoMdAxisInfo.htm)


		 Интерфейс IAdoMdAxisInfo
		 содержит свойства для работы с коллекцией осей, по которым сформированы
		 данные при выполнении MDX-запроса.


		 ![](../Interface_Image.gif)
		 [IAdoMdCatalog](IAdoMdCatalog/IAdoMdCatalog.htm)


		 Интерфейс IAdoMdCatalog
		 содержит свойства и методы объекта репозитория - [Каталог
		 ADOMD](UiNavObj.chm::/ADOMD/UiDb_relational_ADOMD.htm).


		 ![](../Interface_Image.gif)
		 [IAdoMdCatalogInstance](IAdoMdCatalogInstance/IAdoMdCatalogInstance.htm)


		 Интерфейс IAdoMdCatalogInstance
		 содержит свойства и методы для работы с содержимым экземпляра
		 каталога ADOMD.


		 ![](../Interface_Image.gif)
		 [IAdoMdCellset](IAdoMdCellset/IAdoMdCellset.htm)


		 Интерфейс IAdoMdCellset
		 содержит свойства для работы с аналитическими данными, полученными
		 в результате выполнения MDX-запроса к кубам на сервере.


		 ![](../Interface_Image.gif)
		 [IAdoMdConnection](IAdoMdConnection/IAdoMdConnection.htm)


		 Интерфейс IAdoMdConnection
		 содержит свойства и методы для работы с соединением, которое создается
		 при открытии каталога ADOMD.


		 ![](../Interface_Image.gif)
		 [IAdoMdCube](IAdoMdCube/IAdoMdCube.htm)


		 Интерфейс IAdoMdCube
		 содержит свойства и методы для работы со структурой куба ADOMD.


		 ![](../Interface_Image.gif)
		 [IAdoMdCubeExecutor](IAdoMdCubeExecutor/IAdoMdCubeExecutor.htm)


		 Интерфейс IAdoMdCubeExecutor
		 содержит свойства и методы для выполнения различных MDX-запросов
		 к кубу.


		 ![](../Interface_Image.gif)
		 [IAdoMdCubeInstance](IAdoMdCubeInstance/IAdoMdCubeInstance.htm)


		 Интерфейс IAdoMdCubeInstance
		 содержит свойства для работы с открытым экземпляром куба ADOMD.


		 ![](../Interface_Image.gif)
		 [IAdoMdDimCalculation](IAdoMdDimCalculation/IAdoMdDimCalculation.htm)


		 Интерфейс IAdoMdDimCalculation
		 содержит свойства для настройки параметров вычисляемого элемента
		 измерения.


		 ![](../Interface_Image.gif)
		 [IAdoMdDimCalculations](IAdoMdDimCalculations/IAdoMdDimCalculations.htm)


		 Интерфейс IAdoMdDimCalculations
		 содержит свойства и методы для работы с коллекцией вычисляемых
		 элементов измерения.


		 ![](../Interface_Image.gif)
		 [IAdoMdDimension](IAdoMdDimension/IAdoMdDimension.htm)


		 Интерфейс IAdoMdDimension
		 содержит свойства и методы для работы со структурой справочника
		 ADOMD.


		 ![](../Interface_Image.gif)
		 [IAdoMdDimensionInstance](IAdoMdDimensionInstance/IAdoMdDimensionInstance.htm)


		 Интерфейс IAdoMdDimensionInstance
		 содержит свойства для работы с открытым экземпляром справочника
		 ADOMD.


		 ![](../Interface_Image.gif)
		 [IAdoMdExecuteDimSetup](IAdoMdExecuteDimSetup/IAdoMdExecuteDimSetup.htm)


		 Интерфейс IAdoMdExecuteDimSetup
		 содержит свойства для указания дополнительных настроек измерения,
		 которые будут использоваться при расчете куба.
