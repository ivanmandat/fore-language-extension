# IAdoMdCube.NativeName


## Синтаксис


NativeName: String;


## Описание


Свойство NativeName определяет
 физическое имя исходного куба на сервере, к которому подключается каталог
 ADOMD.


## Комментарии


Физическое имя уникальным образом идентифицирует куб на сервере, может
 состоять из латинских букв и цифр. Физическое имя куба на сервере может
 не совпадать с наименованием или идентификатором соответствующего объекта
 репозитория.


Примечание.
 В «Форсайт. Аналитическая платформа»
 изменение физического имени куба ADOMD из прикладного куба не доступно.
