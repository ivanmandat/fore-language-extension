# IAdoMdCube


## Описание


Интерфейс IAdoMdCube содержит
 свойства и методы для работы со структурой куба ADOMD.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AutoRefresh](IAdoMdCube.AutoRefresh.htm)
		 Свойство AutoRefresh
		 определяет, будет ли производиться автоматическое обновление структуры
		 текущего куба.


		 ![](../../Property_Image.gif)
		 [Catalog](IAdoMdCube.Catalog.htm)
		 Свойство Catalog возвращает
		 родительский каталог, в котором хранится куб ADOMD.


		 ![](../../Property_Image.gif)
		 [Dimensions](IAdoMdCube.Dimensions.htm)
		 Свойство Dimensions
		 возвращает коллекцию измерений куба ADOMD, описываемую интерфейсом
		 [ICubeModelDimensions](KeCubes.chm::/Interface/ICubeModelDimensions/ICubeModelDimensions.htm).


		 ![](../../Property_Image.gif)
		 [NativeName](IAdoMdCube.NativeName.htm)
		 Свойство NativeName
		 определяет физическое имя исходного куба на сервере, к которому
		 подключается каталог ADOMD.


		 ![](../../Property_Image.gif)
		 [RefreshStamp](IAdoMdCube.RefreshStamp.htm)
		 Свойство RefreshStamp
		 определяет дату и время обновления структуры куба.


		 ![](../../Property_Image.gif)
		 [SqlText](IAdoMdCube.SqlText.htm)
		 Свойство SqlText определяет
		 текст MDX-запроса, на котором был создан куб.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Refresh](IAdoMdCube.Refresh.htm)
		 Метод Refresh обновляет
		 структуру куба в репозитории на основании информации о кубе на
		 сервере, к которому подключается каталог ADOMD.
