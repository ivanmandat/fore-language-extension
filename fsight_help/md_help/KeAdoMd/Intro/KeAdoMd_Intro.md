# Введение в сборку AdoMd


В данном разделе представлено описание сборки AdoMd,
 предназначенной для работы с каталогом ADOMD, а также его дочерними объектами:
 кубами и справочниками ADOMD. Используя свойства и методы интерфейсов
 можно узнать о структуре объектов, изменить какие-либо параметры, а также
 получить доступ к данным многомерных структур.


Основными интерфейсами сборки AdoMd
 являются:


	- [IAdoMdCatalog](../Interface/IAdoMdCatalog/IAdoMdCatalog.htm).
	 Интерфейс, описывающий каталог ADOMD и позволяющий определить его
	 параметры подключения;


	- [IAdoMdCatalogInstance](../Interface/IAdoMdCatalogInstance/IAdoMdCatalogInstance.htm).
	 Интерфейс для работ с открытым экземпляром каталога ADOMD, используется
	 для обновления содержимого и получения параметров текущего подключения;


	- [IAdoMdCube](../Interface/IAdoMdCube/IAdoMdCube.htm).
	 Интерфейс, описывающий структуру куба ADOMD;


	- [IAdoMdCubeInstance](../Interface/IAdoMdCubeInstance/IAdoMdCubeInstance.htm).
	 Интерфейс для работ с открытым экземпляром куба ADOMD, используется
	 для получения доступа к данным;


	- [IAdoMdDimension](../Interface/IAdoMdDimension/IAdoMdDimension.htm).
	 Интерфейс, описывающий структуру справочника ADOMD;


	- [IAdoMdDimensionInstance](../Interface/IAdoMdDimensionInstance/IAdoMdDimensionInstance.htm).
	 Интерфейс для работ с открытым экземпляром справочника ADOMD, используется
	 для получения доступа к данным;


	- [IAdoMdCellset](../Interface/IAdoMdCellset/IAdoMdCellset.htm).
	 Интерфейс для работы с данными, полученными в результате выполнения
	 MDX-запроса.


Более подробно работа с каталогом ADOMD рассмотрена в следующих подразделах:


	- [Создание каталога ADOMD](KeAdoMd_CreateCatalog.htm);


	- [Работа с кубами и справочниками
	 ADOMD](KeAdoMd_WorkWithData.htm).
