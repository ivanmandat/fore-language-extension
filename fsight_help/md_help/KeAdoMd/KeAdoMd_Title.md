## С чего начать


		 Сборка AdoMd
		 предназначена для работы с объектом репозитория - [Каталог
		 ADOMD](UiNavObj.chm::/ADOMD/UiDb_relational_ADOMD.htm),
		 и его дочерними объектами. Изучив данный раздел, вы сможете с
		 применением языка Fore:


			- создавать каталоги ADOMD, настраивать их для подключения
			 к многомерным источникам данных, которые созданы на основе
			 Microsoft Analysis Services, SAP NetWeaver BW;


			- получать доступ к дочерним объектам каталога: кубам
			 и справочникам ADOMD, работать с их данными;


			- использовать кубы и справочники ADOMD в других инструментах
			 продукта «Форсайт. Аналитическая платформа».


		Более подробно с возможностями блока можно ознакомиться в разделе
		 «[Введение в сборку AdoMd](Intro/KeAdoMd_Intro.htm)».


		Описание ресурсов сборки представлено в следующих подразделах:


			- [Интерфейсы
			 сборки AdoMd](Interface/KeAdoMd_Interface.htm)


			- [Перечисления сборки
			 AdoMd](Enums/KeAdoMd_Enums.htm)


		 Перед изучением сборки AdoMd рекомендуется ознакомиться
		 с архитектурой той системы, к которой будет осуществляться подключение
		 с помощью каталога ADOMD. Также рекомендуется изучить язык запросов
		 MDX, который используется для доступа к многомерным структурам
		 данных.
