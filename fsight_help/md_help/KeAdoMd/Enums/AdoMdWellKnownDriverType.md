# AdoMdWellKnownDriverType


## Описание


Перечисление AdoMdWellKnownDriverType
 содержит тип драйвера, который будет использоваться при подключении к
 базе данных.


Используется следующими свойствами и методами:


	- [IAdoMdCatalog.WellKnownDriver](../Interface/IAdoMdCatalog/IAdoMdCatalog.WellKnownDriver.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Драйвер не определен.


		 1
		 MSOLAP. Драйвер для
		 подключения к базам данных на базе Microsoft SQL Server Analysis
		 Services.


		 2
		 SAP. Драйвер для подключения
		 к базам данных на базе SAP NetWeaver BW.


		 3
		 SAPHana. Драйвер для
		 подключения к базам данных на базе SAPHana.
