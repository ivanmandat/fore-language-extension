# IUiCommandTarget.CreateExecutionContext


## Синтаксис


		CreateExecutionContext: [IUiCommandExecutionContext](../IUiCommandExecutionContext/IUiCommandExecutionContext.htm);


## Описание


Метод CreateExecutionContext
 создает контекст для выполнения команды.


## Комментарии


Контекст выполнения необходим для определения окружения, в котором будет
 выполняться команда: родительское окно (форма), режим выполнения и параметры.
