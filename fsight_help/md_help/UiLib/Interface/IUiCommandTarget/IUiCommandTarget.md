# IUiCommandTarget


## Описание


Интерфейс IUiCommandTarget содержит
 методы, осуществляющие выполнение команд над объектами репозитория.


## Комментарии


Используется только в настольном приложении.


Все объекты репозитория платформы поддерживают стандартные операции,
 такие как открытие и редактирование. Также каждый класс объектов может
 поддерживать специфические только для него операции. Например, вычисляемый
 куб поддерживает операцию редактирования формул. Список всех доступных
 операций над объектом можно увидеть, вызвав контекстное меню в навигаторе
 над объектом. Из кода прикладной системы можно выполнить любую из этих
 операций.
