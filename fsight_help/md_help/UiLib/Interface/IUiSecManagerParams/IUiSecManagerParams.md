# IUiSecManagerParams


## Описание


Интерфейс IUiSecManagerParams
 содержит свойства и методы для определения параметров отображения менеджера
 безопасности.


Используется плагином: [Adm:
 OpenNavigator](../IUiCommandTarget/Command/Command_OpenNavigator_Adm.htm).
