# IWinApplicationClass


Сборка: Ui;


## Описание


Интерфейс IWinApplicationClass
 содержит статические свойства и методы класса [WinApplication](../../Class/WinApplication/WinApplication.htm),
 предназначенные для управления приложением, для обработки сообщений Windows
 и свойства для получения сведений о приложении.
