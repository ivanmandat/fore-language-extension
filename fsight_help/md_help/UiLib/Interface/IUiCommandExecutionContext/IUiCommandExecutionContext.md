# IUiCommandExecutionContext


## Описание


Интерфейс IUiCommandExecutionContext
 содержит свойства для задания дополнительных параметров выполнения команды.


## Комментарии


Используется только в настольном приложении.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CommandActionCustomize](IUiCommandExecutionContext.CommandActionCustomize.htm)
		 Свойство CommandActionCustomize
		 возвращает настройки для действий, которые доступны у объекта
		 репозитория при его открытии на просмотр или редактирование.


		 ![](../../Property_Image.gif)
		 [Data](IUiCommandExecutionContext.Data.htm)
		 Свойство Data определяет
		 данные, необходимые для выполнения команды.


		 ![](../../Property_Image.gif)
		 [ExecutionMode](IUiCommandExecutionContext.ExecutionMode.htm)
		 Свойство ExecutionMode
		 определяет тип выполнения команды.


		 ![](../../Property_Image.gif)
		 [Modal](IUiCommandExecutionContext.Modal.htm)
		 Свойство Modal определяет
		 признак открытия окна диалога модально, относительно окна, установленного
		 в свойстве [ParentWindow](IUiCommandExecutionContext.ParentWindow.htm).


		 ![](../../Property_Image.gif)
		 [ParentWindow](IUiCommandExecutionContext.ParentWindow.htm)
		 Свойство ParentWindow
		 определяет родительское окно для выполнения команды.


		 ![](../../Property_Image.gif)
		 [Silent](IUiCommandExecutionContext.Silent.htm)
		 Свойство Silent определяет
		 способ открытия объекта.
