# IUiCommandActionCustomize.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс действия.


## Описание


Метод Remove осуществляет удаление параметров действия из коллекции по индексу и возвращает признак удачного удаления.


## Комментарии


После удаления параметры действия будут определяться системными настройками платформы. Метод возвращает значение True, если параметры действия были удалены, и False если удаление не произошло.
