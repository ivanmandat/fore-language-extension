# IUiCommandActionCustomize


## Описание


Интерфейс IUiCommandActionCustomize
 содержит свойства и методы для работы с коллекцией параметров действий,
 которые доступны для объектов репозитория.


## Комментарии


Коллекцию параметров действий возвращает свойство [IWinApplication.CommandActionCustomize](../IWinApplication/IWinApplication.CommandActionCustomize.htm)
 и [IUiCommandExecutionContext.CommandActionCustomize](../IUiCommandExecutionContext/IUiCommandExecutionContext.CommandActionCustomize.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IUiCommandActionCustomize.Count.htm)
		 Свойство Count возвращает
		 количество действий, для которых имеются настройки параметров.


		 ![](../../Property_Image.gif)
		 [Item](IUiCommandActionCustomize.Item.htm)
		 Свойство Item возвращает
		 параметры действия по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IUiCommandActionCustomize.Add.htm)
		 Метод Add добавляет
		 новое действие в коллекцию и возвращает параметры добавленного
		 действия.


		 ![](../../Sub_Image.gif)
		 [Clear](IUiCommandActionCustomize.Clear.htm)
		 Метод Clear осуществляет
		 очистку коллекции параметров действий.


		 ![](../../Sub_Image.gif)
		 [Find](IUiCommandActionCustomize.Find.htm)
		 Метод Find осуществляет
		 поиск действия по наименованию.


		 ![](../../Sub_Image.gif)
		 [Remove](IUiCommandActionCustomize.Remove.htm)
		 Метод Remove осуществляет
		 удаление параметров действия из коллекции по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByName](IUiCommandActionCustomize.RemoveByName.htm)
		 Метод RemoveByName
		 осуществляет удаление параметров действия из коллекции по наименованию.
