# IWinApplicationParams


## Описание


Интерфейс IWinApplicationParams
 содержит свойства возвращающие значения параметров, с которыми было запущено
 приложение.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IWinApplicationParams.Count.htm)
		 Свойство Count возвращает
		 количество параметров, с которыми осуществлен запуск платформы.


		 ![](../../Property_Image.gif)
		 [Item](IWinApplicationParams.Item.htm)
		 Свойство Item возвращает
		 значение параметра, с которыми осуществлен запуск приложения.


		 ![](../../Property_Image.gif)
		 [ParamIndex](IWinApplicationParams.ParamIndex.htm)
		 Свойство ParamIndex
		 возвращает индекс ключа в массиве входных параметров, с которым
		 осуществлен запуск платформы.


		 ![](../../Property_Image.gif)
		 [ParamValue](IWinApplicationParams.ParamValue.htm)
		 Свойство ParamValue
		 возвращает параметры ключа, передаваемого посредством входного
		 параметра.
