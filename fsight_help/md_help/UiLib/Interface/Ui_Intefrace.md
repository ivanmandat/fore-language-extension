# Интерфейсы сборки Ui


		 Интерфейс
		 Краткое описание


		 ![](../Interface_Image.gif)
		 [IApplicationGlobals](IApplicationGlobals/IApplicationGlobals.htm)
		 Интерфейс IApplicationGlobals
		 содержит свойства и методы для работы с глобальными переменными
		 приложения.


		 ![](../Interface_Image.gif)
		 [IApplicationWindows](IApplicationWindows/IApplicationWindows.htm)
		 Интерфейс IApplicationWindows
		 содержит свойства и методы коллекции окон приложения.


		 ![](../Interface_Image.gif)
		 IMemoryStatus
		 Интерфейс IMemoryStatus
		 содержит свойства для получения информации о памяти, доступной
		 в операционной системе.


		 ![](../Interface_Image.gif)
		 [IOperatingSystemVersion](IOperatingSystemVersion/IOperatingSystemVersion.htm)
		 Интерфейс IOperatingSystemVersion
		 содержит свойства для получения информации об операционной системе.


		 ![](../Interface_Image.gif)
		 [IUiCommandAction](IUiCommandAction/IUiCommandAction.htm)
		 Интерфейс IUiCommandAction
		 содержит свойства для определения параметров действия.


		 ![](../Interface_Image.gif)
		 [IUiCommandActionCustomize](IUiCommandActionCustomize/IUiCommandActionCustomize.htm)
		 Интерфейс IUiCommandActionCustomize
		 содержит свойства и методы для работы с коллекцией параметров
		 действий, которые доступны для объектов репозитория.


		 ![](../Interface_Image.gif)
		 [IUiCommandExecutionContext](IUiCommandExecutionContext/IUiCommandExecutionContext.htm)
		 Интерфейс IUiCommandExecutionContext
		 содержит свойства для задания дополнительных параметров выполнения
		 команды.


		 ![](../Interface_Image.gif)
		 [IUiCommandTarget](IUiCommandTarget/IUiCommandTarget.htm)
		 Интерфейс IUiCommandTarget
		 содержит методы, осуществляющие выполнение команд над объектами
		 репозитория.


		 ![](../Interface_Image.gif)


		 [IUiLicenseRequestSettings](IUiLicenseRequestSettings/IUiLicenseRequestSettings.htm)
		 Интерфейс IUiLicenseRequestSettings
		 содержит свойства для работы с [мастером
		 активации модуля](setup.chm::/08_Licensing/RMS/Intro_LicMngr_ActivateModule.htm).


		 ![](../Interface_Image.gif)
		 [IWinApplication](IWinApplication/IWinApplication.htm)
		 Интерфейс IWinApplication
		 содержит свойства и методы для управления запущенным приложением.


		 ![](../Interface_Image.gif)
		 [IWinApplicationClass](IWinApplicationClass/IWinApplicationClass.htm)
		 Интерфейс IWinApplicationClass
		 содержит статические свойства и методы класса [WinApplication](../Class/WinApplication/WinApplication.htm),
		 предназначенные для управления приложением, для обработки сообщений
		 Windows и свойства для получения сведений о приложении.


		 ![](../Interface_Image.gif)
		 [IWinApplicationHelp](IWinApplicationHelp/IWinApplicationHelp.htm)
		 Интерфейс IWinApplicationHelp
		 содержит свойства и методы для работы со справкой приложения.


		 ![](../Interface_Image.gif)
		 [IWinApplicationParams](IWinApplicationParams/IWinApplicationParams.htm)
		 Интерфейс IWinApplicationParams
		 содержит свойства возвращающие значения параметров, с которыми
		 было запущено приложение.


		 ![](../Interface_Image.gif)
		 [IWinApplicationSettings](IWinApplicationSettings/IWinApplicationSettings.htm)
		 Интерфейс IWinApplicationSettings
		 содержит свойства для определения параметров оформления, используемого
		 в приложении.


		 ![](../Interface_Image.gif)
		 [IWinProcess](IWinProcess/IWinProcess.htm)
		 Интерфейс IWinProcess
		 содержит свойства и методы для запуска приложений, имеющих визуальный
		 интерфейс, и получения результата их выполнения.


		 ![](../Interface_Image.gif)
		 [IWinShellExecuteInfo](IWinShellExecuteInfo/IWinShellExecuteInfo.htm)
		 Интерфейс IWinShellExecuteInfo
		 содержит свойства, позволяющие настроить параметры открытия файла
		 (приложения).
