# IWinApplicationSettings


Сборка: Ui;


## Описание


Интерфейс IWinApplicationSettings
 содержит свойства для определения параметров оформления, используемого
 в приложении.


## Иерархия наследования


           IWinApplicationSettings


## Комментарии


Изменение настроек оформления доступно только в версиях АК «Прогноз-5».
 В «Форсайт. Аналитическая платформа»
 используется постоянная схема оформления, настройки оформления доступны
 только на чтение.
