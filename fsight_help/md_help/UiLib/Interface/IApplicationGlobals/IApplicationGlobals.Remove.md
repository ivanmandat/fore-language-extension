# IApplicationGlobals.Remove


## Синтаксис


Remove(Name: String): Boolean;


## Параметры


Name. Наименование глобальной
 переменной.


## Описание


Метод Remove осуществляет удаление
 глобальной переменной с указанными наименованием.


## Комментарии


Метод возвращает значение True,
 если удаление глобальной переменной завершилось успешно.
