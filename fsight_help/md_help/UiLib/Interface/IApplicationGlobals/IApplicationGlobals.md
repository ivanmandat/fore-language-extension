# IApplicationGlobals


## Описание


Интерфейс IApplicationGlobals
 содержит свойства и методы для работы с глобальными переменными приложения.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IApplicationGlobals.Count.htm)
		 Свойство Count возвращает
		 количество глобальных переменных приложения.


		 ![](../../Property_Image.gif)
		 [Item](IApplicationGlobals.Item.htm)
		 Свойство Item определяет
		 значение указанной глобальной переменной.


		 ![](../../Property_Image.gif)
		 [Name](IApplicationGlobals.Name.htm)
		 Свойство Name возвращает
		 наименование глобальной переменной с указанным индексом.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IApplicationGlobals.Add.htm)
		 Метод Add добавляет
		 в список глобальных переменных новую переменную с указанными наименованием
		 и значением.


		 ![](../../Sub_Image.gif)
		 [Clear](IApplicationGlobals.Clear.htm)
		 Метод Clear очищает
		 весь список глобальных переменных.


		 ![](../../Sub_Image.gif)
		 [Remove](IApplicationGlobals.Remove.htm)
		 Метод Remove осуществляет
		 удаление глобальной переменной с указанными наименованием.


		 ![](../../Sub_Image.gif)
		 [RemoveAt](IApplicationGlobals.RemoveAt.htm)
		 Метод RemoveAt осуществляет
		 удаление глобальной переменную с указанным индексом.
