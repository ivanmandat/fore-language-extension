# IApplicationGlobals.Add


## Синтаксис


Add(Name: String; Value: Variant);


## Параметры


Name. Наименование глобальной
 переменной;


Value. Значение глобальной
 переменной.


## Описание


Метод Add добавляет в список
 глобальных переменных новую переменную с указанными наименованием и значением.
