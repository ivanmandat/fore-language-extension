# IApplicationGlobals.RemoveAt


## Синтаксис


RemoveAt(Index: Integer): Boolean;


## Параметры


Index. Индекс глобальной переменной.


## Описание


Метод RemoveAt осуществляет
 удаление глобальной переменную с указанным индексом.


## Комментарии


Метод возвращает значение True,
 если удаление глобальной переменной завершилось успешно.
