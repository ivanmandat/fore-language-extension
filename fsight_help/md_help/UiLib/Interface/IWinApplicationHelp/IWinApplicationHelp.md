# IWinApplicationHelp


## Описание


Интерфейс IWinApplicationHelp
 содержит свойства и методы для работы со справкой приложения.


## Комментарии


Описание процесса подключения и использования справки в пользовательском
 приложении представлено в подразделе базы знаний: [Использование
 пользовательской справки](KnowledgeBase.chm::/01_Fore/KnowledgeBase_KB000025.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [FileName](IWinApplicationHelp.FileName.htm)
		 Свойство FileName определяет
		 путь и наименование файла справки.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ALinkLookup](IWinApplicationHelp.ALinkLookup.htm)
		 Метод ALinkLookup осуществляет
		 поиск по ключевым словам, сопоставленным страницам справки, но
		 не входящим в указатель и открывает справку с позиционированием
		 на странице, которая сопоставлена ключевому слову.


		 ![](../../Sub_Image.gif)
		 [Context](IWinApplicationHelp.Context.htm)
		 Метод Context осуществляет
		 поиск страницы по числовому идентификатору и в случае удачного
		 поиска открывает файл справки с позиционирование на данной странице.


		 ![](../../Sub_Image.gif)
		 [DisplayContents](IWinApplicationHelp.DisplayContents.htm)
		 Метод DisplayContents
		 открывает файл справки на вкладке «Содержание».


		 ![](../../Sub_Image.gif)
		 [DisplayIndex](IWinApplicationHelp.DisplayIndex.htm)
		 Метод DisplayIndex
		 открывает файл справки на вкладке «Указатель».


		 ![](../../Sub_Image.gif)
		 [DisplaySearch](IWinApplicationHelp.DisplaySearch.htm)
		 Метод DisplaySearch
		 открывает файл справки на вкладке «Поиск».


		 ![](../../Sub_Image.gif)
		 [KeywordLookup](IWinApplicationHelp.KeywordLookup.htm)
		 Метод KeywordLookup
		 осуществляет поиск по словам в указателе и
		 открывает справку с позиционированием на странице, сопоставленой
		 ключевому слову.
