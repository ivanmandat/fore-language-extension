# IWinShellExecuteInfo


Сборка: Ui;


## Описание


Интерфейс IWinShellExecuteInfo
 содержит свойства, позволяющие настроить параметры открытия файла (приложения).


## Комментарии


Для открытия файла (приложения) предназначен метод [IWinApplicationClass.ShellExecute](../IWinApplicationClass/IWinApplicationClass.ShellExecute.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ShowCommand](IWinShellExecuteInfo.ShowCommand.htm)
		 Свойство ShowCommand
		 определяет состояние окна открываемого файла (приложения).


		 ![](../../Property_Image.gif)
		 [Silent](IWinShellExecuteInfo.Silent.htm)
		 Свойство Silent определяет,
		 будут ли отображаться окна об ошибках, которые могут
		 возникнуть при открытии файла (приложения).


		 ![](../../Property_Image.gif)
		 [Verb](IWinShellExecuteInfo.Verb.htm)
		 Свойство Verb определяет
		 команду, которую необходимо использовать при открытии приложения
		 или документа.
