# IWinShellExecuteInfo.Silent


## Синтаксис


Silent: Boolean;


## Описание


Свойство Silent определяет,
 будут ли отображаться окна об ошибках, которые могут возникнуть
 при открытии файла (приложения).


## Комментарии


Если свойству установлено значение True,
 то при возникновении ошибок окна, генерируемые операционной
 системой, на экран выводиться не будут. По умолчанию свойству установлено
 значение False.
