# IUiSecManagerUsersGroupParams.PropertiesEnabled


## Синтаксис


PropertiesEnabled: Boolean;


## Описание


Свойство PropertiesEnabled определяет
 видимость меню «Свойства» для
 пользователей.


## Комментарии


При значении True - меню доступно.
 При значении False - меню не доступно.


Значение по умолчанию True.
