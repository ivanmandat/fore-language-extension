# IUiSecManagerUsersGroupParams.OperationsEnabled


## Синтаксис


OperationsEnabled: Boolean;


## Описание


Свойство OperationsEnabled определяет
 доступность команд для работы с пользователями.


## Комментарии


При значении True - команды
 доступны. При значении False -
 команды не доступны.


Значение по умолчанию True.
