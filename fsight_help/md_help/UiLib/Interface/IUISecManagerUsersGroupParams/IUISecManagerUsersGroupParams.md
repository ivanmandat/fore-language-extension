# IUISecManagerUsersGroupParams


Сборка: Ui;


## Описание


Интерфейс IUISecManagerUsersGroupParams
 содержит свойства и методы для определения параметров отображения вкладок
 «Пользователи» и «Группы»
 менеджера безопасности.
