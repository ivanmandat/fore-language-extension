# IOperatingSystemVersion.FullName


## Синтаксис


FullName: String;


## Описание


Свойство FullName возвращает
 полное наименование операционной системы.


## Комментарии


Полное наименование включает в себя наименование, версию, номер сборки
 и наименование последнего обновления операционной системы.
