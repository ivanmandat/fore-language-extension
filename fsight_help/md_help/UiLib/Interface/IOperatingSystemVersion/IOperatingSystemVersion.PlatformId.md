# IOperatingSystemVersion.PlatformId


## Синтаксис


PlatformId: Integer;


## Описание


Свойство PlatformId возвращает
 числовой идентификатор, соответствующий платформе операционной системы.


## Комментарии


Данное свойство возвращает следующие значения:


		 Значение
		 Платформа


		 0
		 Windows 3.1.


		 1
		 Windows 95, Windows 98 или Windows Me.


		 2
		 Windows NT и более поздние версии.
