# IOperatingSystemVersion.MinorVersion


## Синтаксис


MinorVersion: Integer;


## Описание


Свойство MinorVersion возвращает
 номер промежуточной версии операционной системы.


## Комментарии


Например, если версия операционной системы - 5.1.2600, то номер промежуточной
 версии будет 1.
