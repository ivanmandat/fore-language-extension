# IOperatingSystemVersion.MajorVersion


## Синтаксис


MajorVersion: Integer;


## Описание


Свойство MajorVersion возвращает
 номер основной версии операционной системы.


## Комментарии


Например, если версия операционной системы - 5.1.2600, то номер основной
 версии будет 5.
