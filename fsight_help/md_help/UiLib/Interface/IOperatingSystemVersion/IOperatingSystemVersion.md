# IOperatingSystemVersion


## Описание


Интерфейс IOperatingSystemVersion
 содержит свойства для получения информации об операционной системе.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CSDVersion](IOperatingSystemVersion.CSDVersion.htm)
		 Свойство CSDVersion
		 возвращает наименование последнего обновления, установленного
		 в операционной системе.


		 ![](../../Property_Image.gif)
		 [FullName](IOperatingSystemVersion.FullName.htm)
		 Свойство FullName возвращает
		 полное наименование операционной системы.


		 ![](../../Property_Image.gif)
		 [MajorVersion](IOperatingSystemVersion.MajorVersion.htm)
		 Свойство MajorVersion
		 возвращает номер основной версии операционной системы.


		 ![](../../Property_Image.gif)
		 [MinorVersion](IOperatingSystemVersion.MinorVersion.htm)
		 Свойство MinorVersion
		 возвращает номер промежуточной версии операционной системы.


		 ![](../../Property_Image.gif)
		 [Name](IOperatingSystemVersion.Name.htm)
		 Свойство Name возвращает
		 наименование операционной системы.


		 ![](../../Property_Image.gif)
		 [PlatformId](IOperatingSystemVersion.PlatformId.htm)
		 Свойство PlatformId
		 возвращает числовой идентификатор, соответствующий платформе операционной
		 системы.
