# IUiCommandAction.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет признак доступности действия на выполнение.


## Комментарии


Если свойству установлено значение True, то действие будет доступно. При этом в открытом объекте будет доступна соответствующая команда меню и кнопка на панели инструментов.


Если свойству установлено значение False, то действие будет недоступно. При этом в открытом объекте соответствующая команда меню и кнопка на панели инструментов будет подсвечена серым цветом.
