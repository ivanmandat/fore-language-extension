# IUiCommandAction.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет видимость элементов управления, используемых для выполнения текущего действия.


## Комментарии


Если свойству установлено значение True, то этом в открытом объекте будет присутствовать соответствующая команда меню и кнопка на панели инструментов.


Если свойству установлено значение False, то в открытом объекте соответствующая команда меню и кнопка на панели инструментов будут скрыты.
