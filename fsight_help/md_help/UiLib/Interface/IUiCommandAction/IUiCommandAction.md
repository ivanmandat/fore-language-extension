# IUiCommandAction


## Описание


Интерфейс IUiCommandAction содержит
 свойства для определения параметров действия.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Enabled](IUiCommandAction.Enabled.htm)
		 Свойство Enabled определяет
		 признак доступности действия на выполнение.


		 ![](../../Property_Image.gif)
		 [Name](IUiCommandAction.Name.htm)
		 Свойство Name возвращает
		 наименование действия.


		 ![](../../Property_Image.gif)
		 [Visible](IUiCommandAction.Visible.htm)
		 Свойство Visible определяет
		 видимость элементов управления, используемых для выполнения текущего
		 действия.
