# IUISecManagerTabParams


Сборка: Ui;


## Описание


Интерфейс IUISecManagerTabParams
 содержит свойства и методы для определения параметров отображения вкладки
 менеджера безопасности.


Используется плагином: [Adm:
 OpenNavigator](../IUiCommandTarget/Command/Command_OpenNavigator_Adm.htm).


## Свойства


			 Имя свойства
			 Краткое описание


			 ![](../../Property_Image.gif)
			 [ReadOnly](IUISecManagerTabParams.ReadOnly.htm)
			 Свойство ReadOnly
			 определяет доступность элементов вкладки только для просмотра.


			 ![](../../Property_Image.gif)
			 [Visible](IUISecManagerTabParams.Visible.htm)
			 Свойство Visible
			 определяет видимость вкладки.
