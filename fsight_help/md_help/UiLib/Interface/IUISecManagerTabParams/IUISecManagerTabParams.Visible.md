# IUiSecManagerTabParams.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет
 видимость вкладки.


## Комментарии


При значении True - вкладка
 будет видна, при значении False
 - вкладка будет скрыта.


Значение по умолчанию True.
