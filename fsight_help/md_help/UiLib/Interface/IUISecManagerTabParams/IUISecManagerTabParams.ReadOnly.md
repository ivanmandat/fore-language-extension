# IUiSecManagerTabParams.ReadOnly


## Синтаксис


ReadOnly: Boolean;


## Описание


Свойство ReadOnly определяет
 доступность элементов вкладки только для просмотра.


## Комментарии


При значении True - все элементы
 вкладки доступны только для просмотра, без возможности вносить изменения.
 При значении False - доступно
 редактирование.


Значение по умолчанию False.
