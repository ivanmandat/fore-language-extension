# IUISecManagerPolicyEditorParams.PasswordPageReadOnly


## Синтаксис


PasswordPageReadOnly: Boolean;


## Описание


Свойство PasswordPageReadOnly
 определяет использование режима «только чтение» для вкладки «Парольная
 политика».


## Комментарии


При значении True - вкладка
 доступна только для просмотра. При значении False
 - вкладка доступна для редактирования.


Значение по умолчанию False.
