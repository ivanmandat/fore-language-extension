# IUISecManagerPolicyEditorParams.GeneralPageReadOnly


## Синтаксис


GeneralPageReadOnly: Boolean;


## Описание


Свойство GeneralPageReadOnly
 определяет использование режима «только чтение» для вкладки «Общая
 политика».


## Комментарии


При значении True - вкладка
 доступна только для просмотра. При значении False
 - вкладка доступна для редактирования.


Значение по умолчанию False.
