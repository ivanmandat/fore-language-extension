# IUiLicenseRequestSettings


Интерфейс IUiLicenseRequestSettings
 содержит свойства для работы с [мастером
 активации модуля](setup.chm::/08_Licensing/RMS/Intro_LicMngr_ActivateModule.htm).


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ShowActivationWizard](IUiLicenseRequestSettings.ShowActivationWizard.htm)
		 Свойство ShowActivationWizard
		 отвечает за появление окна «[Активация модуля](setup.chm::/08_Licensing/RMS/Intro_LicMngr_ActivateModule.htm)»,
		 когда запрашиваемая единица лицензирования недоступна.


## Свойства, унаследованные от [ILicenseRequestSettings](KeSomHost.chm::/Interface/ILicenseRequestSettings/ILicenseRequestSettings.htm)


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [FeatureType](KeSomHost.chm::/Interface/ILicenseRequestSettings/ILicenseRequestSettings.FeatureType.htm)
		 Свойство FeatureType предназначено
		 для запроса одной единицы лицензирования, не указывая при этом
		 наименование этой единицы лицензирования.
