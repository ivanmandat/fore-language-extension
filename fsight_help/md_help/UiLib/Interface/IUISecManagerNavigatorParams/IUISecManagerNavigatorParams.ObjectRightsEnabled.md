# IUiSecManagerTabParams.ObjectRightsEnabled


## Синтаксис


ObjectRightsEnabled: Boolean;


## Описание


Свойство ObjectRightsEnabled
 определяет видимость меню «Права доступа»
 объектов вкладки.


## Комментарии


При значении True - меню доступно.
 При значении False - меню не доступно.


Значение по умолчанию True.
