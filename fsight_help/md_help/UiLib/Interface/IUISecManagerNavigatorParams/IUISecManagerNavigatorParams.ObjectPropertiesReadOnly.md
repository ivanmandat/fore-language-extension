# IUiSecManagerTabParams.ObjectPropertiesReadOnly


## Синтаксис


ObjectPropertiesReadOnly: Boolean;


## Описание


Свойство ObjectPropertiesReadOnly
 определяет использование режима «только чтение» для окна «Свойства» объектов репозитория.


## Комментарии


При значении True - окно доступно только для просмотра. При значении False
 - окно доступно для редактирования.


Значение по умолчанию False.
