# IUISecManagerNavigatorParams.ObjectPropertiesEnabled


## Синтаксис


ObjectPropertiesEnabled: Boolean;


## Описание


Свойство ObjectPropertiesEnabled
 определяет видимость меню «Свойства»
 объектов репозитория.


## Комментарии


При значении True - меню доступно.
 При значении False - меню не доступно.


Значение по умолчанию True.
