# WindowTabStyle


## Описание


Перечисление WindowTabStyle
 содержит стили оформления вкладок, которые могут использоваться в платформе.


Используется следующим свойством:


	- [IWinApplicationSettings.TabStyles](../Interface/IWinApplicationSettings/IWinApplicationSettings.TabStyles.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Standard. Стандартные
		 вкладки.

![](WindowTabStyle_Standard.gif)


		 1
		 OfficeOneNote. Стиль
		 Microsoft Office OneNote.

![](WindowTabStyle_OfficeOneNote.gif)


		 2
		 VisualStudio2005. Стиль
		 Microsoft Visual Studio 2005.

![](WindowTabStyle_VisualStudio2005.gif)


		 3
		 Chamfered. Вкладки
		 со скругленными краями.

![](WindowTabStyle_Chamfered.gif)


Примечание.
 Цветовое оформление вкладок зависит от используемой темы оформления. В
 указанных примерах использована тема оформления Microsoft Visual Studio
 2005.
