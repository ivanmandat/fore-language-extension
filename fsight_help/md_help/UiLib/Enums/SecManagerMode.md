# SecManagerMode


## Описание


Перечисление SecManagerMode
 содержит типы разделов менеджера безопасности.


Используется следующими свойствами и методами:


	- [IUiSecManagerParams.Active](../Interface/IUiSecManagerParams/IUiSecManagerParams.Active.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None.
		 Раздел не определен.


		 1
		 Users. Раздел «Пользователи».


		 2
		 Groups. Раздел «Группы».


		 4
		 Navigator. Раздел «Навигатор».


		 8
		 Privelege. Раздел «Привилении».


		 16
		 Mandatory. Раздел «Мандатный
		 доступ».


		 32
		 PolicyEditor. Раздел
		 «Редактор политики».


		 64
		 Classes. Раздел «Классы
		 объектов».


		 128
		 LogViewer. Раздел «Протокол
		 досутпа».


		 256
		 SecurityWatch. Раздел
		 «Мониторинг нарушений защиты».


		 512
		 Service. Раздел «Сервис».
