# WinShellExecuteInfo


Сборка: Ui;


## Описание


Класс WinShellExecuteInfo реализует
 объект, используемый для открытия внешних файлов и приложений.


## Свойства объекта класса, унаследованные от [IWinShellExecuteInfo](../../Interface/IWinShellExecuteInfo/IWinShellExecuteInfo.htm)

		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ShowCommand](../../Interface/IWinShellExecuteInfo/IWinShellExecuteInfo.ShowCommand.htm)
		 Свойство ShowCommand
		 определяет состояние окна открываемого файла (приложения).


		 ![](../../Property_Image.gif)
		 [Silent](../../Interface/IWinShellExecuteInfo/IWinShellExecuteInfo.Silent.htm)
		 Свойство Silent определяет,
		 будут ли отображаться окна об ошибках, которые могут
		 возникнуть при открытии файла (приложения).


		 ![](../../Property_Image.gif)
		 [Verb](../../Interface/IWinShellExecuteInfo/IWinShellExecuteInfo.Verb.htm)
		 Свойство Verb определяет
		 команду, которую необходимо использовать при открытии приложения
		 или документа.
