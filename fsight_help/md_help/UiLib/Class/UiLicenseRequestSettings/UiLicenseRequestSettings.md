# UiLicenseRequestSettings


Сборка: Ui;


## Описание


Класс UiLicenseRequestSettings
 реализует объект, предназначенный для работы с [мастером
 активации модуля](setup.chm::/08_Licensing/RMS/Intro_LicMngr_ActivateModule.htm).


## Свойства объекта класса, унаследованные от [IUiLicenseRequestSettings](../../Interface/IUiLicenseRequestSettings/IUiLicenseRequestSettings.htm)


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ShowActivationWizard](../../Interface/IUiLicenseRequestSettings/IUiLicenseRequestSettings.ShowActivationWizard.htm)
		 Свойство ShowActivationWizard
		 отвечает за появление окна «[Активация модуля](setup.chm::/08_Licensing/RMS/Intro_LicMngr_ActivateModule.htm)»,
		 когда запрашиваемая единица лицензирования недоступна.

		 ![](../../Property_Image.gif)

		 Свойство FeatureType предназначено
		 для запроса одной единицы лицензирования, не указывая при этом
		 наименование этой единицы лицензирования.
