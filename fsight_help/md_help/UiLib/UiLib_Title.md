## С чего начать


		 Сборка Ui
		 предназначена для работы с «Форсайт. Аналитическая платформа»
		 как с приложением Windows. Интерфейсы сборки позволяют управлять
		 пунктами меню, доступными в инструментах продукта «Форсайт. Аналитическая платформа»,
		 выполнять какие-либо команды в приложении, работать с лицензиями,
		 получать системную информацию об операционной системе.


		Описание ресурсов сборки представлено в следующих подразделах:


			- [Интерфейсы сборки
			 Ui](Interface/Ui_Intefrace.htm)


			- [Перечисления сборки Ui](Enums/Ui_Enums.htm)


			- [Классы сборки Ui](Class/Ui_Class.htm)


			- [Примеры](Samples/Example_built_Ui.htm)


		 Перед изучением сборки Ui рекомендуется ознакомиться
		 с разделами: «[Разработка
		 прикладного приложения](UiDevEnv.chm::/01_Development_Environment/01_Purpose_of_the_constructor/Purpose_of_the_constructor.htm)», «[Руководство по языку Fore](Fore.chm::/Fore_Title.htm)».
