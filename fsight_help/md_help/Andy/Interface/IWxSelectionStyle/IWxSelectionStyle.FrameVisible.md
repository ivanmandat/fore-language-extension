# IWxSelectionStyle.FrameVisible


## Синтаксис


		FrameVisible: Boolean;


## Описание


Свойство FrameVisible определяет
 признак отображения рамки у выделенного объекта. Если свойство имеет значение
 True, то рамка будет отображаться,
 при значении False - нет.


По умолчанию свойство имеет значение True.
