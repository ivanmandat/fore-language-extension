# IWxSelectionStyle.GroupFrameVisible


## Синтаксис


		GroupFrameVisible: Boolean;


## Описание


Свойство GroupFrameVisible определяет
 признак отображения рамки у выделенной группы. Если свойство имеет значение
 True, то рамка будет отображаться,
 при значении False - нет.


По умолчанию свойство имеет значение True.
