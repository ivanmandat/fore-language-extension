# IWxShapeExtension


Сборка: Andy;


## Описание


Интерфейс IWxShapeExtension
 не содержит собственных свойств и методов и является базовым интерфейсом
 для объектов, являющихся расширением для фигур рабочего пространства.


## Иерархия наследования


           IWxShapeExtension


## Комментарии


В текущей реализации расширение используется для размещения на рабочем
 пространстве фигур, соответствующих объектам репозитория. Для работы с
 расширением фигур предназначено свойство [IWxShape.Extension](../IWxShape/IWxShape.Extension.htm).
 Примеры по работе с расширением фигур можно найти в описании указанного
 свойства, а также в описании сборок [Etl](KeEtl.chm::/Interface/KeEtl_Interface.htm)
 и [Dt](KeDt.chm::/KeDt_Introduction.htm).
