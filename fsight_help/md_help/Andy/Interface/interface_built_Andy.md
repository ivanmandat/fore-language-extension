# Интерфейсы сборки Andy


В рамках сборки Andy существуют
 следующие интерфейсы:


		 Интерфейс
		 Краткое описание


		 ![](../Interface_Image.gif)
		 [IWxArrow](IWxArrow/IWxArrow.htm)


		 Интерфейс IWxArrow
		 определяет параметры фигурной стрелки.


		 ![](../Interface_Image.gif)
		 [IWxBaloon](IWxBaloon/IWxBaloon.htm)


		 Интерфейс IWxBaloon
		 определяет параметры выноски.


		 ![](../Interface_Image.gif)
		 [IWxBoxArrow](IWxBoxArrow/IWxBoxArrow.htm)


		 Интерфейс IWxBoxArrow
		 определяет параметры выноски со стрелкой.


		 ![](../Interface_Image.gif)
		 [IWxConnector](IWxConnector/IWxConnector.htm)


		 Интерфейс IWxConnector
		 содержит свойства и методы соединяющего объекта.


		 ![](../Interface_Image.gif)
		 [IWxContainer](IWxContainer/IWxContainer.htm)


		 Интерфейс IWxContainer
		 содержит свойства и методы контейнера фигур.


		 ![](../Interface_Image.gif)
		 [IWxCurveLink](IWxCurveLink/IWxCurveLink.htm)


		 Интерфейс IWxCurveLink используется
		 для работы со скругленной линией.


		 ![](../Interface_Image.gif)
		 [IWxDoubleArrow](IWxDoubleArrow/IWxDoubleArrow.htm)


		 Интерфейс IWxDoubleArrow
		 определяет параметры двойной стрелки.


		 ![](../Interface_Image.gif)
		 [IWxDoubleBoxArrow](IWxDoubleBoxArrow/IWxDoubleBoxArrow.htm)


		 Интерфейс IWxDoubleBoxArrow
		 определяет параметры выноски с двумя стрелками.


		 ![](../Interface_Image.gif)
		 [IWxEllipse](IWxEllipse/IWxEllipse.htm)


		 Интерфейс IWxEllipse
		 определяет параметры эллипса.


		 ![](../Interface_Image.gif)
		 [IWxEllipseBaloon](IWxEllipseBaloon/IWxEllipseBaloon.htm)


		 Интерфейс IWxEllipseBaloon
		 определяет параметры овальной выноски.


		 ![](../Interface_Image.gif)
		 [IWxGroup](IWxGroup/IWxGroup.htm)


		 Интерфейс IWxGroup
		 определяет параметры группы объектов.


		 ![](../Interface_Image.gif)
		 [IWxLink](IWxLink/IWxLink.htm)


		 Интерфейс IWxLink определяет
		 параметры ломанной линии.


		 ![](../Interface_Image.gif)
		 [IWxLinkedBaloon](IWxLinkedBaloon/IWxLinkedBaloon.htm)


		 Интерфейс IWxLinkedBaloon
		 определяет параметры выноски с точками.


		 ![](../Interface_Image.gif)
		 [IWxObject](IWxObject/IWxObject.htm)


		 Интерфейс IWxObject
		 является базовым для всех объектов.


		 ![](../Interface_Image.gif)
		 [IWxPageSettings](IWxPageSettings/IWxPageSettings.htm)


		 Интерфейс IWxPageSettings
		 определяет параметры страницы для рабочего пространства.


		 ![](../Interface_Image.gif)
		 [IWxRectangle](IWxRectangle/IWxRectangle.htm)


		 Интерфейс IWxRectangle
		 определяет параметры прямоугольника.


		 ![](../Interface_Image.gif)
		 [IWxRectangleBaloon](IWxRectangleBaloon/IWxRectangleBaloon.htm)


		 Интерфейс IWxRectangleBaloon
		 определяет параметры прямоугольной выноски.


		 ![](../Interface_Image.gif)
		 [IWxRegulargon](IWxRegulargon/IWxRegulargon.htm)


		 Интерфейс IWxRegulargon
		 определяет параметры многоугольника.


		 ![](../Interface_Image.gif)
		 [IWxRoundedRectangle](IWxRoundedRectangle/IWxRoundedRectangle.htm)


		 Интерфейс IWxRoundedRectangle
		 определяет параметры прямоугольника со скругленными углами.


		 ![](../Interface_Image.gif)
		 [IWxRoundedRectangleBaloon](IWxRoundedRectangleBaloon/IWxRoundedRectangleBaloon.htm)


		 Интерфейс IWxRoundedRectangleBaloon
		 определяет параметры скругленной прямоугольной выноски.


		 ![](../Interface_Image.gif)
		 [IWxSelectedShapes](IWxSelectedShapes/IWxSelectedShapes.htm)


		 Интерфейс IWxSelectedShapes
		 определяет параметры коллекции выделенных фигур.


		 ![](../Interface_Image.gif)
		 [IWxSelectionStyle](IWxSelectionStyle/IWxSelectionStyle.htm)


		 Интерфейс IWxSelectionStyle
		 определяет параметры оформления выделенного объекта на рабочем
		 пространстве.


		 ![](../Interface_Image.gif)
		 [IWxShape](IWxShape/IWxShape.htm)


		 Интерфейс IWxShape
		 содержит свойства и методы фигуры.


		 ![](../Interface_Image.gif)
		 [IWxShapeExtension](IWxShapeExtension/IWxShapeExtension.htm)


		 Интерфейс IWxShapeExtension
		 не содержит собственных свойств и методов и является базовым интерфейсом
		 для объектов, являющихся расширением для фигур рабочего пространства.


		 ![](../Interface_Image.gif)
		 [IWxShapes](IWxShapes/IWxShapes.htm)


		 Интерфейс IWxShapes
		 содержит свойства и методы коллекции фигур рабочего пространства.


		 ![](../Interface_Image.gif)
		 [IWxSmoothBaloon](IWxSmoothBaloon/IWxSmoothBaloon.htm)


		 Интерфейс IWxSmoothBaloon
		 определяет параметры выноски с фигурной стрелкой.


		 ![](../Interface_Image.gif)
		 [IWxStraightLink](IWxStraightLink/IWxStraightLink.htm)


		 Интерфейс IWxStraightLink
		 определяет параметры соединительной линии.


		 ![](../Interface_Image.gif)
		 [IWxStyle](IWxStyle/IWxStyle.htm)


		 Интерфейс IWxStyle
		 предназначен для настройки формата (параметров оформления) объектов.


		 ![](../Interface_Image.gif)
		 [IWxStyles](IWxStyles/IWxStyles.htm)


		 Интерфейс IWxStyles
		 содержит свойства и методы коллекции стилей.


		 ![](../Interface_Image.gif)
		 [IWxTableShape](IWxTableShape/IWxTableShape.htm)


		 Интерфейс IWxTableShape
		 содержит свойства и методы для работы с табличной фигурой.


		 ![](../Interface_Image.gif)
		 [IWxView](IWxView/IWxView.htm)


		 Интерфейс IWxView определяет
		 настройки отображения рабочего пространства.


		 ![](../Interface_Image.gif)
		 [IWxViewExporter](IWxViewExporter/IWxViewExporter.htm)


		 Интерфейс IWxViewExporter
		 содержит свойства и методы для экспорта рабочего пространства.


		 ![](../Interface_Image.gif)
		 [IWxViews](IWxViews/IWxViews.htm)


		 Интерфейс IWxViews
		 содержит свойства и методы коллекции отображений рабочего пространства.


		 ![](../Interface_Image.gif)
		 [IWxWorkspace](IWxWorkspace/IWxWorkspace.htm)


		 Интерфейс IWxWorkspace
		 содержит свойства и методы рабочего пространства.
