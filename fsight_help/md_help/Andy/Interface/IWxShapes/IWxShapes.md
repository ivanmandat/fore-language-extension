# IWxShapes


Сборка: Andy;


## Описание


Интерфейс IWxShapes содержит
 свойства и методы коллекции фигур рабочего пространства.


## Иерархия наследования


           IWxShapes


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IWxShapes.Count.htm)
		 Свойство Count возвращает
		 количество объектов рабочего пространства.


		 ![](../../Property_Image.gif)
		 [Item](IWxShapes.Item.htm)
		 Свойство Item возвращает
		 фигуру по заданному индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ClearAll](IWxShapes.ClearAll.htm)

		 Метод ClearAll удаляет
		 все фигуры с рабочего пространства.


		 ![](../../Sub_Image.gif)
		 [FindById](IWxShapes.FindById.htm)
		 Метод FindById осуществляет
		 поиск фигуры по идентификатору.


		 ![](../../Sub_Image.gif)
		 [RemoveById](IWxShapes.RemoveById.htm)
		 Метод RemoveById осуществляет
		 удаление фигуры по идентификатору и возвращает признак удачного
		 удаления.
