# IWxObject.EndUpdate


## Синтаксис


		EndUpdate;


## Описание


Метод EndUpdate снимает запрет
 на перерисовку объекта. После вызова EndUpdate
 будут отрисованы все изменения свойств объекта, произведенные с ним в
 промежуток между вызовами методов [IWxObject.BeginUpdate](IWxObject.BeginUpdate.htm)
 и EndUpdate.

Пример использования приведен в описании [IWxLinkedBaloon.PointCount](../IWxLinkedBaloon/IWxLinkedBaloon.PointCount.htm).
