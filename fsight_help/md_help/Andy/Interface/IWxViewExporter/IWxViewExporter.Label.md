# IWxViewExporter.Label


## Синтаксис


		Label: [IGxDocumentLabel](ModDrawing.chm::/Interface/IGxDocumentLabel/IGxDocumentLabel.htm);


## Описание


Свойство Label возвращает параметры
 маркировки отчета.


## Комментарии


Данное свойство автоматически используется при включенном мандатном
 доступе, в случае если для объекта задана маркировка (уровень безопасности).
