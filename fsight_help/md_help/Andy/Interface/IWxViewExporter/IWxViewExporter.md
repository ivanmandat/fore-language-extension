# IWxViewExporter


Сборка: Andy;


## Описание


Интерфейс IWxViewExporter содержит
 свойства и методы для экспорта рабочего пространства.

## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [FullLengthTitle](IWxViewExporter.FullLengthTitle.htm)
		 Свойство FullLengthTitle
		 определяет, будет ли обрезаться заголовок при экспорте если его
		 ширина превышает ширину рабочего пространства.


		 ![](../../Property_Image.gif)
		 [Label](IWxViewExporter.Label.htm)
		 Свойство Label возвращает
		 параметры маркировки отчета.


		 ![](../../Property_Image.gif)
		 [PixelHeight](IWxViewExporter.PixelHeight.htm)
		 Свойство PixelHeight
		 определяет высоту, которую будет иметь изображение после экспорта.


		 ![](../../Property_Image.gif)
		 [PixelWidth](IWxViewExporter.PixelWidth.htm)
		 Свойство PixelWidth
		 определяет ширину, которую будет иметь изображение после экспорта.


		 ![](../../Property_Image.gif)
		 [WxView](IWxViewExporter.WxView.htm)
		 Свойство WxView определяет
		 представление рабочего пространства.
