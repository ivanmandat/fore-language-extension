# IWxWorkspace.UndoEnabled


## Синтаксис


		UndoEnabled: Boolean;


## Описание


Свойство UndoEnabled включает/отключает
 ведение истории отменяемых/возвращаемых действий. Если значение свойства
 True, то история ведется, при
 значении False - история не ведется
 и отменить/возвратить действия будет невозможно.


По умолчанию свойство имеет значение True.
