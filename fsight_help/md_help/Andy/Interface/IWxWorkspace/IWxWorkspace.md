## Описание


Интерфейс IWxWorkspace содержит
 свойства и методы рабочего пространства.


## Иерархия наследования


           [IWxObject](../IWxObject/IWxObject.htm)


           [IWxContainer](../IWxContainer/IWxContainer.htm)


           IWxWorkspace


## Комментарии


Рабочее пространство является удобным инструментом:


	- для визуального размещения объектов системы на едином рабочем
	 поле;


	- для организации доступа к функциям и свойствам объектов системы;


	- для отображения связей между объектами и структуры их зависимостей.
