# IWxWorkspace.Undo


## Синтаксис


		Undo([Steps: Integer = 1]);


## Параметры


Steps. Количество отменяемых
 шагов. По умолчанию параметр равен единице.


## Описание


Метод Undo отменяет заданное
 количество совершенных действий. В случае если параметр не задан, то метод
 отменяет последнее совершенное действие.
