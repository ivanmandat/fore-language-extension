# IWxShape.TransparentColor


## Синтаксис


TransparentColor: [IGxColor](ModDrawing.chm::/Interface/IGxColor/IGxColor.htm);


## Описание


Свойство TransparentColor определяет
 цвет, который будет прозрачным на изображении, которое используется в
 качестве заливки для фигуры.
