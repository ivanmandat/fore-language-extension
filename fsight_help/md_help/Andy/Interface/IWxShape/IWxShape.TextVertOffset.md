# IWxShape.TextVertOffset


## Синтаксис


TextVertOffset: Double;


## Описание


Свойство TextVertOffset определяет
 вертикальное смещение текста.


## Комментарии


Данное свойство можно использовать только для линии, описываемой интерфейсом
 [IWxLink](../IWxLink/IWxLink.htm).
 При положительных значениях свойства сдвиг текста производится вниз от
 центра линии, при отрицательных - вверх от центра линии.
