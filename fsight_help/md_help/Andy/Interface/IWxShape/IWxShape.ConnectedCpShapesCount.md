# IWxShape.ConnectedCpShapesCount


## Синтаксис


		ConnectedCpShapesCount(ConnectionPointIndex:
		 Integer): Integer;


## Параметры


ConnectionPointIndex. Индекс
 точки соединения объекта.


## Описание


Свойство ConnectedCpShapesCount
 возвращает количество точек соединения, соединенных с точкой соединения,
 заданной по индексу.
