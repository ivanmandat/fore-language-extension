## Описание


Интерфейс IWxShape содержит
 свойства и методы фигуры рабочего пространства.

## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AllowedInteractions](IWxShape.AllowedInteractions.htm)


		 Свойство AllowedInteractions
		 определяет действия, которые будут доступны для фигуры рабочего
		 пространства.


		 ![](../../Property_Image.gif)
		 [Angle](IWxShape.Angle.htm)


		 Свойство Angle определяет
		 угол поворота фигуры.


		 ![](../../Property_Image.gif)
		 [ConnectedCPs](IWxShape.ConnectedCPs.htm)


		 Свойство ConnectedCP
		 возвращает индекс точки соединения фигуры, соединенной с исходным
		 объектом в заданной точке.


		 ![](../../Property_Image.gif)
		 [ConnectedCpShapesCount](IWxShape.ConnectedCpShapesCount.htm)


		 Свойство ConnectedCpShapesCount
		 возвращает количество точек соединения, соединенных с точкой соединения
		 заданной по индексу.


		 ![](../../Property_Image.gif)
		 [ConnectedShapes](IWxShape.ConnectedShapes.htm)


		 Свойство ConnectedShapes
		 позволяет получить фигуру, соединенную с исходным объектом в заданной
		 точке.


		 ![](../../Property_Image.gif)
		 [ConnectionPoints](IWxShape.ConnectionPoints.htm)


		 Свойство ConnectionPoints
		 возвращает координаты точек соединения объекта.


		 ![](../../Property_Image.gif)
		 [ConnectionPointsCount](IWxShape.ConnectionPointsCount.htm)


		 Свойство ConnectionPointsCount
		 возвращает количество точек соединения объекта.


		 ![](../../Property_Image.gif)
		 [Cursor](IWxShape.Cursor.htm)


		 Свойство Cursor определяет
		 вид курсора над объектом.


		 ![](../../Property_Image.gif)
		 [Extension](IWxShape.Extension.htm)


		 Свойство Extension
		 определяет объект, являющийся расширением фигуры.


		 ![](../../Property_Image.gif)
		 [FormattedText](IWxShape.FormattedText.htm)


		 Свойство FormattedText
		 определяет форматированный текст фигуры в формате RTF.


		 ![](../../Property_Image.gif)
		 [GroupBehaviour](IWxShape.GroupBehaviour.htm)


		 Свойство GroupBehaviour
		 определяет поведение объекта при изменении размера группы, в которую
		 он включен.


		 ![](../../Property_Image.gif)
		 [Id](IWxShape.Id.htm)


		 Свойство Id определяет
		 идентификатор объекта.


		 ![](../../Property_Image.gif)
		 [LocalPinPositionCoeff](IWxShape.LocalPinPositionCoeff.htm)


		 Свойство LocalPinPositionCoeff
		 определяет смещение точки вращения относительно размеров фигуры.


		 ![](../../Property_Image.gif)
		 [Parent](IWxShape.Parent.htm)


		 Свойство Parent возвращает
		 родительский контейнер фигуры.


		 ![](../../Property_Image.gif)
		 [Picture](IWxShape.Picture.htm)


		 Свойство Picture определяет
		 изображение, которое будет использоваться в качестве заливки для
		 фигуры.


		 ![](../../Property_Image.gif)
		 [PinPosition](IWxShape.PinPosition.htm)


		 Свойство PinPosition
		 определяет координату положения фигуры.


		 ![](../../Property_Image.gif)
		 [SelectionStyle](IWxShape.SelectionStyle.htm)


		 Свойство SelectionStyle
		 определяет параметры оформления выделенного фрагмента текста фигуры.


		 ![](../../Property_Image.gif)
		 [Size](IWxShape.Size.htm)


		 Свойство Size определяет
		 размеры фигуры.


		 ![](../../Property_Image.gif)
		 [Style](IWxShape.Style.htm)


		 Свойство Style определяет
		 параметры оформления фигуры.


		 ![](../../Property_Image.gif)
		 [Tag](IWxShape.Tag.htm)

		 ![](../../Property_Image.gif)
		 [Text](IWxShape.Text.htm)


		 Свойство Text определяет
		 текст, который будет содержать фигура.


		 ![](../../Property_Image.gif)
		 [TextHorzOffset](IWxShape.TextHorzOffset.htm)


		 Свойство TextHorzOffset
		 определяет горизонтальное смещение текста.


		 ![](../../Property_Image.gif)
		 [TextVertOffset](IWxShape.TextVertOffset.htm)


		 Свойство TextVertOffset
		 определяет вертикальное смещение текста.


		 ![](../../Property_Image.gif)
		 [TransparentColor](IWxShape.TransparentColor.htm)


		 Свойство TransparentColor
		 определяет цвет, который будет прозрачным на изображении, которое
		 используется в качестве заливки для фигуры.


		 ![](../../Property_Image.gif)
		 [Type](IWxShape.Type.htm)


		 Свойство Type возвращает

		 ![](../../Property_Image.gif)
		 [Workspace](IWxShape.Workspace.htm)


		 Свойство Workspace
