# IWxView.DisplayIntersections


## Синтаксис


		DisplayIntersections: Boolean;


## Описание


Свойство DisplayIntersections
 определяет признак отображения пересечений связей.


## Комментарии


Если значение свойства True,
 то пересечения отображаются, при значении False
 - нет.


По умолчанию свойство имеет значение True.
