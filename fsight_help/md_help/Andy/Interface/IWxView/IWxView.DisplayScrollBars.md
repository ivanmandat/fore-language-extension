# IWxView.DisplayScrollBars


## Синтаксис


		DisplayScrollBars: Boolean;


## Описание


Свойство DisplayScrollBars определяет
 признак отображения полос прокрутки.


## Комментарии


Если значение свойства True,
 то полосы прокрутки отображаются, при значении False
 - нет.


По умолчанию свойство имеет значение True.
