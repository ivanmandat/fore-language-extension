# IWxView.DisplayFocusFrame


## Синтаксис


		DisplayFocusFrame: Boolean;


## Описание


Свойство DisplayFocusFrame определяет
 признак отображения фокусной рамки, которая вызывается при нажатии на
 клавишу TAB.


## Комментарии


Если значение свойства True,
 то рамка отображается, при значении False
 - нет.


По умолчанию свойство имеет значение True.
