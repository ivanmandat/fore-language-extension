# IWxView.Scale


## Синтаксис


		Scale: Double;


## Описание


Свойство Scale определяет масштаб
 отображения.


## Комментарии


Допустимые значения берутся из диапазона [0.01, 10]. Значение «0.01»
 соответствует 1% от натуральной величины, а значение «10» - 1000% от натуральной
 величины.
