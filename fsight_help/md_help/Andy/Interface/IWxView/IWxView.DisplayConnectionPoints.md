# IWxView.DisplayConnectionPoints


## Синтаксис


		DisplayConnectionPoints: Boolean;


## Описание


Свойство DisplayConnectionPoints
 определяет признак отображения точек соединения.


## Комментарии


Если значение свойства True,
 то точки соединения отображаются, при значении False
 - нет.


По умолчанию свойство имеет значение True.
