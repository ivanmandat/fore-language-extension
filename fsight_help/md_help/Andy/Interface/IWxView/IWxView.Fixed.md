# IWxView.Fixed


## Синтаксис


		Fixed: Boolean;


## Описание


Свойство Fixed определяет признак
 блокировки рабочего пространства для прокрутки в область отрицательных
 координат.


## Комментарии


Рабочее пространство условно разделено на четыре квадранта. Центр системы
 координат и оси не отображаются, но они учитываются при размещении фигур
 на рабочем пространстве. Если свойству Fixed
 установить значение True, то прокрутка
 в область отрицательных координат с помощью полос прокрутки или мыши будет
 запрещена.


По умолчанию свойству установлено значение False,
 при этом прокрутка допустима в любую область системы координат.
