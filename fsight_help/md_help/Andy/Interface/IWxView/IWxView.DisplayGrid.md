# IWxView.DisplayGrid


## Синтаксис


		DisplayGrid: Boolean;


## Описание


Свойство DisplayGrid определяет
 признак отображения сетки.


## Комментарии


Если значение свойства True,
 то сетка отображается, при значении False
 - нет.


По умолчанию свойство имеет значение True.
