# IWxView.DisplaySelections


## Синтаксис


		DisplaySelections: Boolean;


## Описание


Свойство DisplaySelections определяет
 признак отображения выделения объекта.


## Комментарии


Если значение свойства True,
 то выделение объекта отображается, при значении False
 - нет.


По умолчанию свойство имеет значение True.
