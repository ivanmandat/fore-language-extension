# IWxPageSettings.PageWidth


## Синтаксис


		PageWidth: Integer;


## Описание


Свойство PageWidth определяет
 количество страниц по горизонтали (в ширину), используемых для размещения
 и печати рабочего пространства. По умолчанию значение свойства равно «1».


Свойство актуально, если свойство [IWxPageSettings.FitToPages](IWxPageSettings.FitToPages.htm)
 установлено в True.
