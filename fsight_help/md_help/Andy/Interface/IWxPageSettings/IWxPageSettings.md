# IWxPageSettings


Сборка: Andy;


## Описание


Интерфейс IWxPageSettings определяет
 параметры страницы для рабочего пространства.


## Иерархия наследования


           [IGxPageSettings](ModDrawing.chm::/Interface/IGxPageSettings/IGxPageSettings.htm)


           IWxPageSettings


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [FitToPages](IWxPageSettings.FitToPages.htm)
		 Свойство FitToPages
		 определяет размещение и печать рабочего пространства на заданном
		 количестве страниц по горизонтали и вертикали.


		 ![](../../Property_Image.gif)
		 [PageHeight](IWxPageSettings.PageHeight.htm)
		 Свойство PageHeight
		 определяет количество страниц по вертикали (в высоту), используемых
		 для размещения и печати рабочего пространства.


		 ![](../../Property_Image.gif)
		 [PageWidth](IWxPageSettings.PageWidth.htm)
		 Свойство PageWidth
		 определяет количество страниц по горизонтали (в ширину), используемых
		 для размещения и печати рабочего пространства.


		 ![](../../Property_Image.gif)
		 [Scale](IWxPageSettings.Scale.htm)
		 Свойство Scale определяет
		 масштаб рабочего пространства при печати.
