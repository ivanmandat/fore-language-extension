# IWxStyle.LinePenWxDashStyle


## Синтаксис


		LinePenWxDashStyle: [WxDashStyle](../../Enums/WxDashStyle.htm);


## Описание


Свойство LinePenWxDashStyle
 определяет стиль (начертание) линии. По умолчанию для линии используется
 сплошное начертание.


## Комментарии


Для определения толщины линии используйте свойство [IWxStyle.LinePenWidth](IWxStyle.LinePenWidth.htm).
