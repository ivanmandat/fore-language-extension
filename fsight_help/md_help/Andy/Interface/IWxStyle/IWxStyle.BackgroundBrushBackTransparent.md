# IWxStyle.BackgroundBrushBackTransparent


## Синтаксис


BackgroundBrushBackTransparent: Integer;


## Описание


Свойство BackgroundBrushBackTransparent
 определяет степень прозрачности узора для фона объекта.


## Комментарии


Допустимые значения берутся из диапазона [0,100]. Значение «100» соответствует
 полной прозрачности, значение «0» соответствует полной непрозрачности.
