# IWxStyle.IsPrivate


## Синтаксис


IsPrivate: Boolean;


## Описание


Свойство IsPrivate определяет
 признак частного стиля. В случае если свойство имеет значение True,
 то стиль не будет отображен в списке [стилей](UiWsp.chm::/2_WorkSpace/Objects_Work/UiWsp_Objects_Styles.htm),
 при значении False - будет отображен
 в данном списке.


По умолчанию свойство имеет значение True.
