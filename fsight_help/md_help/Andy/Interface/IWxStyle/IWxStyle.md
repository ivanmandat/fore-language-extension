# IWxStyle


Сборка: Andy;


## Описание


Интерфейс IWxStyle предназначен
 для настройки формата (параметры оформления) объектов.


## Иерархия наследования


           [IWxObject](../IWxObject/IWxObject.htm)


           IWxStyle


## Комментарии


Для работы с коллекцией стилей используйте интерфейс [IWxStyles](../IWxStyles/IWxStyles.htm).
