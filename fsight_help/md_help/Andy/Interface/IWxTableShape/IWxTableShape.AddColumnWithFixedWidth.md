# IWxTableShape.AddColumnWithFixedWidth


## Синтаксис


		AddColumnWithFixedWidth(Value: Integer):
		 Boolean;


## Параметры


Value.
 Ширина создаваемого столбца в пикселях.


## Описание


Метод AddColumnWithFixedWidth
 добавляет в таблицу столбец фиксированной ширины.
