# IWxTableShape.AddRowWithFixedHeight


## Синтаксис


		AddRowWithFixedHeight(Value: Integer): Boolean;


## Параметры


Value.
 Высота создаваемой строки в пикселях.


## Описание


Метод AddRowWithFixedHeight
 добавляет в таблицу строку фиксированной высоты.
