# GxAlignment


## Описание


Перечисление GxAlignment содержит типы выравнивания изображения по горизонтали/по вертикали в элементе управления. Данное перечисление используется свойствами:


-
[IWxStyle.PictureAlignmentHorizontal](../Interface/IWxStyle/IWxStyle.PictureAlignmentHorizontal.htm)


-
[IWxStyle.PictureAlignmentVertical](../Interface/IWxStyle/IWxStyle.PictureAlignmentVertical.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Near. По левому краю (по горизонтали); по верхнему краю (по вертикали).


 1
 Center. По центру.


 2
 Far. По правому краю (по горизонтали); по нижнему краю (по вертикали).


 -1
 Undefined. Не определен.


 10
 Justify. По ширине.
