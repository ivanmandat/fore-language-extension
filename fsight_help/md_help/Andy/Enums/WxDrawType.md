# WxDrawType


## Описание


Перечисление WxDrawType содержит типы отрисовки фигуры. Перечисление используется свойством IWxShape.DrawType.


## Допустимые значения


 Значение
 Краткое описание


 0
 ShapeDefault. Обычная фигура.


 1
 GroupDefault. Обычная группа.


 2
 LinkDefault. Обычная связь.
