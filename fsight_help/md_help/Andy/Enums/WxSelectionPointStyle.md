# WxSelectionPointStyle


## Описание


Перечисление WxSelectionPointStyle содержит типы выделения объекта. Используется свойством [IWxSelectionStyle.SelectionPointStyle](../Interface/IWxSelectionStyle/IWxSelectionStyle.SelectionPointStyle.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Не определено.


 1
 Corners. 4 точки.


 2
 All. 8 точек.
