# WxShapesAlignment


## Описание


Перечисление WxShapesAlignment содержит типы выравниваний объектов. Перечисление используется методом:


-
[IWxSelectedShapes.Align](../Interface/IWxSelectedShapes/IWxSelectedShapes.Align.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Left. Слева. Фигуры в коллекции будут выровнены по левому краю относительно крайней левой фигуры.


 1
 Right. Справа. Фигуры в коллекции будут выровнены по правому краю относительно крайней правой фигуры.


 2
 HorizontalCenter. По горизонтальному центру. Центр рассчитывается исходя из положения крайней верхней и крайней нижней фигуры в коллекции, с учетом размеров самих фигур.


 3
 Top. Сверху. Фигуры в коллекции будут выровнены по верхнему краю относительно крайней верхней фигуры.


 4
 Bottom. Снизу. Фигуры в коллекции будут выровнены по нижнему краю относительно крайней нижней фигуры.


 5
 VerticalCenter. По вертикальному центру. Центр рассчитывается исходя из положения крайней левой и крайней правой фигуры в коллекции, с учетом размеров самих фигур.


 6
 DistributeHorizontal. Распределить по горизонтали. Фигуры в коллекции будут отстоять друг от друга по горизонтали на одинаковом расстоянии.


 7
 DistributeVertical. Распределить по вертикали. Фигуры в коллекции будут отстоять друг от друга по вертикали на одинаковом расстоянии.
