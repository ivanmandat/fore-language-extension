# WxWordWrap


## Описание


Перечисление WxWordWrap содержит варианты переноса текста. Перечисление используется свойством [IWxStyle.WordWrap](../Interface/IWxStyle/IWxStyle.WordWrap.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Не переносить.


 1
 BreakWords. Переносить по словам.
