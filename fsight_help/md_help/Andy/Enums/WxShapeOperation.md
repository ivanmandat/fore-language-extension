# WxShapeOperation


## Описание


Перечисление WxShapeOperation
 содержит варианты действий, выполняемые с фигурой, при которых наступает
 событие [OnShapeChanged](KeWsp.chm::/Class/WorkspaceBox/WorkspaceBox.OnShapeChanged.htm).


Используется следующими свойствами и методами:


	- [IShapeChangedEventArgs.ShapeOperation](KeWsp.chm::/Interface/IShapeChangedEventArgs/IShapeChangedEventArgs.ShapeOperation.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Действие не определено.


		 1
		 AddShape. Создание
		 фигуры.


		 2
		 DeleteShape. Удаление
		 фигуры.


		 3
		 IconChanged. Изменение
		 изображения для фигуры.


		 4
		 TextChanged. Изменение
		 текста фигуры.


		 5
		 FontChanged. Изменение
		 оформления фигуры.
