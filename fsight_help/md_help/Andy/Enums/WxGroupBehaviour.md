# WxGroupBehaviour


## Описание


Перечисление WxGroupBehaviour содержит типы поведения объекта при изменении размера группы, в которую он включен. Перечисление используется свойством [IWxShape.GroupBehaviour](../Interface/IWxShape/IWxShape.GroupBehaviour.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 RepositionOnly.При изменении размера группы объект будет изменять свое положение (размер объекта остается неизменным).


 1
 ScaleWithGrou. При изменении размера группы размер объекта будет масштабироваться.
