# WxAdjustMode


## Описание


Перечисление WxAdjustMode содержит варианты подгонки размеров объекта. Перечисление используется свойством [IWxStyle.AdjustMode](../Interface/IWxStyle/IWxStyle.AdjustMode.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Отсутствует.


 1
 Width. В ширину.


 2
 Height. В высоту.
