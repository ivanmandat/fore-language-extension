# Перечисления сборки Andy


В рамках сборки Andy существуют
 следующие перечисления:


		 Перечисление
		 Краткое описание


		 ![](../Enums_Image.gif)
		 [GxAlignment](GxAlignment.htm)


		 Перечисление GxAlignment
		 содержит типы выравнивания изображения по горизонтали/по вертикали
		 в элементе управления.


		 ![](../Enums_Image.gif)
		 [GxBrushPattern](GxBrushPattern.htm)


		 Перечисление GxBrushPattern
		 содержит шаблоны заливки тени/фона объекта.


		 ![](../Enums_Image.gif)
		 [WxAdjustMode](WxAdjustMode.htm)


		 Перечисление WxAdjustMode
		 содержит варианты подгонки размеров объекта.


		 ![](../Enums_Image.gif)
		 [WxAllowedInteractions](WxAllowedInteractions.htm)


		 Перечисление WxAllowedInteractions
		 содержит действия, которые могут быть доступны для фигуры рабочего
		 пространства.


		 ![](../Enums_Image.gif)
		 [WxCursor](WxCursor.htm)


		 Перечисление WxCursor
		 содержит виды курсора.


		 ![](../Enums_Image.gif)
		 [WxDashStyle](WxDashStyle.htm)


		 Перечисление WxDashStyle
		 используется для определения стиля начертания линии.


		 ![](../Enums_Image.gif)
		 [WxDrawType](WxDrawType.htm)


		 Перечисление WxDrawType
		 содержит типы отрисовки фигуры.


		 ![](../Enums_Image.gif)
		 [WxFigureType](wxfiguretype.htm)


		 Перечисление WxFigureType
		 используется для определения типа фигуры.


		 ![](../Enums_Image.gif)
		 [WxGroupBehaviour](WxGroupBehaviour.htm)


		 Перечисление WxGroupBehaviour
		 содержит типы поведения объекта при изменении размера группы,
		 в которою он включен.


		 ![](../Enums_Image.gif)
		 [WxLineCap](WxLineCap.htm)


		 Перечисление WxLineCap
		 используется для определения видов наконечников у линии.


		 ![](../Enums_Image.gif)
		 [WxObjectProperty](WxObjectProperty.htm)


		 Перечисление WxObjectProperty
		 содержит свойства объекта, которые можно защитить от изменений.


		 ![](../Enums_Image.gif)
		 [WxSelectionPointStyle](WxSelectionPointStyle.htm)


		 Перечисление WxSelectionPointStyle
		 содержит типы выделения объекта.


		 ![](../Enums_Image.gif)
		 [WxShapeOperation](WxShapeOperation.htm)


		 Перечисление WxShapeOperation
		 содержит варианты действий, выполняемые с фигурой, при которых
		 наступает событие [OnShapeChanged](KeWsp.chm::/Class/WorkspaceBox/WorkspaceBox.OnShapeChanged.htm).


		 ![](../Enums_Image.gif)
		 [WxShapesAlignment](WxShapesAlignment.htm)


		 Перечисление WxShapesAlignment
		 содержит типы выравниваний объектов.


		 ![](../Enums_Image.gif)
		 [WxShapesSizing](WxShapesSizing.htm)


		 Перечисление WxShapesSizing
		 содержит типы изменения размеров объектов.


		 ![](../Enums_Image.gif)
		 [WxShapeType](WxShapeType.htm)


		 Перечисление WxShapeType
		 содержит типы фигур объектов.


		 ![](../Enums_Image.gif)
		 [WxTextPosition](WxTextPosition.htm)


		 Перечисление WxTextPosition
		 содержит варианты выравнивания текста.


		 ![](../Enums_Image.gif)
		 [WxTool](WxTool.htm)


		 Перечисление WxTool
		 содержит типы инструментов рабочего пространства.


		 ![](../Enums_Image.gif)
		 [WxWordWrap](WxWordWrap.htm)


		 Перечисление WxWordWrap
		 содержит варианты переноса текста.
