# GxBrushPattern


## Описание


Перечисление GxBrushPattern содержит шаблоны заливки тени/фона объекта. Перечисление используется свойствами:


-
[IWxStyle.ShadowBrushPattern](../Interface/IWxStyle/IWxStyle.ShadowBrushPattern.htm)


-
[IWxStyle.BackgroundBrushPattern](../Interface/IWxStyle/IWxStyle.BackgroundBrushPattern.htm)


## Допустимые значения


 Значение
 Краткое описание


 -1
 Undefined. Неопределен.


 100
 None. Отсутствует.


 101
 Solid. Сплошной. Для использования данного шаблона необходимо определить цвет фона.


 102
 Dots. Точки. Для использования данного шаблона необходимо определить цвет узора и цвет фона.


 103
 Diagonals. Диагонали. Для использования данного шаблона необходимо определить цвет узора и цвет фона.


 104
 Gradient. Градиент. Для использования данного шаблона необходимо определить цвет узора и цвет фона.
