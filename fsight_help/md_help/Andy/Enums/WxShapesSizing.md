# WxShapesSizing


## Описание


Перечисление WxShapesSizing содержит типы изменения размеров объектов. Перечисление используется методом:


-
[IWxSelectedShapes.Size](../Interface/IWxSelectedShapes/IWxSelectedShapes.Size.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 MaxWidth. По самому широкому объекту, ширина фигур в коллекции будет изменена на ширину самой широкой фигуры.


 1
 MinWidth. По самому узкому объекту, ширина фигур в коллекции будет изменена на ширину самой узкой фигуры.


 2
 MaxHeight. По самому высокому объекту, высота фигур в коллекции будет изменена на высоту самой высокой фигуры.


 3
 MinHeight. По самому низкому объекту, высота фигур в коллекции будет изменена на высоту самой низкой фигуры.
