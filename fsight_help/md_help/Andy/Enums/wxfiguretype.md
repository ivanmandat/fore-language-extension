# WxFigureType


## Описание


Перечисление WxFigureType используется
 для определения типа фигуры.


Используется следующими свойствами и методами:


	- [IWxWorkspace.Replace](../Interface/IWxWorkspace/IWxWorkspace.Replace.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Rectangle. Прямоугольник.


		 1
		 Arrow. Фигурная стрелка
		 с одним наконечником.


		 2
		 DoubleArrow. Фигурная
		 стрелка с наконечниками на обоих концах.


		 3
		 BoxArrow. Фигурная
		 стрелка в виде прямоугольника с одним наконечником.


		 4
		 DoubleBoxArrow. Фигурная
		 стрелка в виде прямоугольника с наконечниками на обоих концах.


		 5
		 Ellipse. Эллипс.


		 6
		 RoundedRectangle. Скругленный
		 прямоугольник.


		 7
		 Regulargon. Прямоугольник
		 с невидимым контуром.


		 8
		 RectangleBaloon. Прямоугольная
		 выноска.


		 9
		 EllipseBaloon. Скругленная
		 выноска.


		 10
		 RoundedRectangleBaloon.
		 Скругленная прямоугольная выноска.
