# WxTextPosition


## Описание


Перечисление WxTextPosition содержит варианты выравнивания текста. Перечисление используется свойством [IWxStyle.TextPosition](../Interface/IWxStyle/IWxStyle.TextPosition.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Over. Поверх.


 1
 Left. Слева.


 2
 Top. Сверху.


 3
 Right. Справа.


 4
 Bottom. Снизу.


 -1
 Undefined. Не определен.
