# WxObjectProperty


## Описание


Перечисление WxObjectProperty содержит свойства объекта, которые можно защитить от изменений. Перечисление используется свойством [IWxObject.Protection](../Interface/IWxObject/IWxObject.Protection.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 EntityFree. Существование объекта. При защите данного свойства нельзя будет удалить объект.
