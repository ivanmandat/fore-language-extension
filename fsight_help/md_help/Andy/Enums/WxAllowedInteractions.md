# WxAllowedInteractions


## Описание


Перечисление WxAllowedInteractions содержит действия, которые могут быть доступны для фигуры рабочего пространства.


Используется следующими свойствами и методами:


-
[IWxShape.AllowedInteractions](../Interface/IWxShape/IWxShape.AllowedInteractions.htm)


## Допустимые значения


 Значение
 Краткое описание


 1
 Move. Можно перемещать.


 2
 Rotate. Можно поворачивать.


 4
 HorizontalLeft. Можно изменять ширину путем перемещения только левой границы.


 8
 HorizontalRight. Можно изменять ширину путем перемещения только правой границы.


 12
 HorizontalSizing. Можно изменять ширину.


 16
 VerticalTop. Можно изменять высоту путем перемещения только верхней границы.


 32
 VerticalBottom. Можно изменять высоту путем перемещения только нижней границы.


 48
 VerticalSizing. Можно изменять высоту.


 64
 AllowCornerSizing. Можно изменять углы.


 128
 AllowSelection. Можно выделять.


 256
 AllowActivation. Можно активировать.


 511
 AllAllowed. Доступны любые действия.
