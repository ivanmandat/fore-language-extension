# IAdhocParamSynchronization.FindById


## Синтаксис


FindById(Id: String): [IAdhocParamSyncIndex](../IAdhocParamSyncIndex/IAdhocParamSyncIndex.htm);


## Параметры


Id. Идентификатор искомой связи.


## Описание


Метод FindById выполняет
 поиск связи с заданным идентификатором.


## Комментарии


Если связь с указанным идентификатором не найдена, то метод вернет значение
 Null.
