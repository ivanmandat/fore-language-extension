# IAdhocParamSynchronization.RemoveById


## Синтаксис


RemoveById(Id: String): Boolean;


## Параметры


Id. Идентификатор удаляемой
 связи.


## Описание


Метод RemoveById удаляет
 из коллекции связь с указанным идентификатором.


## Комментарии


Если удаление выполнено успешно метод возвращает значение True,
 в противном случае - False.
