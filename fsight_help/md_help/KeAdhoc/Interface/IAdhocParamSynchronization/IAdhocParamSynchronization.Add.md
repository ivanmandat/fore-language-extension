# IAdhocParamSynchronization.Add


## Синтаксис


Add(Id: String): [IAdhocParamSyncIndex](../IAdhocParamSyncIndex/IAdhocParamSyncIndex.htm);


## Параметры


Id. Идентификатор добавляемой
 связи.


## Описание


Метод Add добавляет связь
 в коллекцию.


## Комментарии


Для проверки, присутствует ли связь с определенным идентификатором в
 коллекции, используйте метод [IAdhocSyncParams.FindById](../IAdhocSyncParams/IAdhocSyncParams.FindById.htm).
