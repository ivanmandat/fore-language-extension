# IAdhocParamSynchronization


Сборка: Adhoc;


## Описание


Интерфейс IAdhocParamSynchronization предназначен
 для работы с коллекцией связей для синхронизации параметров источников
 данных аналитической панели.


## Иерархия наследования


           IAdhocParamSynchronization


## Комментарии


Для получения коллекции связей используйте свойство [IAdhocSynchronization.Params](../IAdhocSynchronization/IAdhocSynchronization.Params.htm).
