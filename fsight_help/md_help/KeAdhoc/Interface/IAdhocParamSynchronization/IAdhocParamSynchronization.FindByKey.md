# IAdhocParamSynchronization.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IAdhocParamSyncIndex](../IAdhocParamSyncIndex/IAdhocParamSyncIndex.htm);


## Параметры


Key. Ключ искомой связи.


## Описание


Метод FindByKey выполняет поиск
 связи с заданным ключом.


## Комментарии


Если связь с указанным ключом не найдена, то метод вернет значение Null.
