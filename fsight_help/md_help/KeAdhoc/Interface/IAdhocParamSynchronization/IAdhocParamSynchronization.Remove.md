# IAdhocParamSynchronization.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемой связи.


## Описание


Метод Remove удаляет указанную
 связь из коллекции.


## Комментарии


Индексация связей в коллекции сквозная и начинается с нуля. Для получения
 количества связей в коллекции используйте свойство [IAdhocSyncParams.Count](../IAdhocSyncParams/IAdhocSyncParams.Count.htm).


Если удаление выполнено успешно метод возвращает значение True,
 в противном случае - False.
