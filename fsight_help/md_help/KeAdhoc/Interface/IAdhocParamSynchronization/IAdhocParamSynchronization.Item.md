# IAdhocParamSynchronization.Item


## Синтаксис


Item(Index: Integer): [IAdhocParamSyncIndex](../IAdhocParamSyncIndex/IAdhocParamSyncIndex.htm);


## Параметры


Index. Индекс связи в коллекции.


## Описание


Свойство Item возвращает
 указанную связь из коллекции.


## Комментарии


Индексация связей в коллекции сквозная и начинается с нуля. Для получения
 количества связей в коллекции используйте свойство [IAdhocSyncParams.Count](../IAdhocSyncParams/IAdhocSyncParams.Count.htm).
