# IAdhocParamSynchronization.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Идентификатор удаляемой
 связи.


## Описание


Метод RemoveByKey удаляет из
 коллекции связь с указанным ключом.


## Комментарии


Если удаление выполнено успешно метод возвращает значение True,
 в противном случае - False.
