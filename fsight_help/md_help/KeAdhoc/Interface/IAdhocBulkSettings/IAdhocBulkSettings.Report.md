# IAdhocBulkSettings.Report


## Синтаксис


Report: [IAdhocReport](../IAdhocReport/IAdhocReport.htm);


## Описание


Свойство Report возвращает
 отчет, с которым выполняется пакетная операция.


## Комментарии


Для получения отметки, с которой будет выполнятся пакетная операция
 используйте свойство [IAdhocBulkSettings.SelectionSet](IAdhocBulkSettings.SelectionSet.htm).
