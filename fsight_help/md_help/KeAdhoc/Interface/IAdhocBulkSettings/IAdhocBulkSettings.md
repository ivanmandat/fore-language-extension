# IAdhocBulkSettings


Сборка: Adhoc;


## Описание


Интерфейс IAdhocBulkSettings
 предназначен для определения параметров пакетных операций, выполняемых
 для аналитической панели.


## Иерархия наследования


           [IGxBulkSettings](ModDrawing.chm::/Interface/IGxBulkSettings/IGxBulkSettings.htm)


           IAdhocBulkSettings


## Комментарии


Пакетная операция - это операция, выполняемая с несколькими вариантами
 аналитической панели. Варианты формируются по отметке фиксированных измерений
 источников данных.


## Свойства


		 Имя свойства
		 Краткое описание

		 [Report](IAdhocBulkSettings.Report.htm)


		 Свойство Report
		 возвращает отчет, с которым выполняется пакетная операция.


		 ![](../../Property_Image.gif)
		 [SelectionSet](IAdhocBulkSettings.SelectionSet.htm)


		 Свойство SelectionSet
		 возвращает отметку фиксированных измерений, в соответствии с которой
		 будут созданы варианты аналитической панели для выполнения пакетной
		 операции.
