# IAdhocSyncDimensions


Сборка: Adhoc;


## Описание


Интерфейс IAdhocSyncDimensions предназначен
 для работы с коллекцией измерений, входящих в индекс синхронизации.


## Иерархия наследования


           IAdhocSyncDimensions


## Комментарии


Для получения коллекции используйте свойство [IAdhocDimSyncIndex.Dimensions](../IAdhocDimSyncIndex/IAdhocDimSyncIndex.Dimensions.htm).
