# IAdhocSyncDimensions.FindById


## Синтаксис


FindById(Id: String): [IAdhocDimSyncItem](../IAdhocDimSyncItem/IAdhocDimSyncItem.htm);


## Параметры


Id. Идентификатор измерения,
 входящего в индекс синхронизации.


## Описание


Метод FindById выполняет поиск
 измерения по его идентификатору.


## Комментарии


Если измерение не найдено, то метод возвращает значение Null.
