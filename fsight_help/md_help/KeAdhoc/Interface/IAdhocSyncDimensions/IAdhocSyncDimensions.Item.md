# IAdhocSyncDimensions.Item


## Синтаксис


Item(Index: Integer): [IAdhocDimSyncItem](../IAdhocDimSyncItem/IAdhocDimSyncItem.htm);


## Параметры


Index. Индекс измерения в коллекции.


## Описание


Свойство Item возвращает
 указанное измерение, входящее в коллекцию.


## Комментарии


Количество измерений в коллекции возвращает свойство [IAdhocSyncDimensions.Count](IAdhocSyncDimensions.Count.htm).
 Индексация измерений сквозная и начинается с нуля.
