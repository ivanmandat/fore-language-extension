# IAdhocSyncDimensions.Add


## Синтаксис


Add:(Dimension: [IAdhocDsoDimension](../IAdhocDsoDimension/IAdhocDsoDimension.htm)):
 [IAdhocDimSyncItem](../IAdhocDimSyncItem/IAdhocDimSyncItem.htm);


## Параметры


Dimension. Измерение источника
 данных.


## Описание


Метод Add добавляет измерение
 источника данных в индекс синхронизации.


## Комментарии


В случае успешного выполнения метод возвращает объект синхронизации,
 соответствующий измерению.
