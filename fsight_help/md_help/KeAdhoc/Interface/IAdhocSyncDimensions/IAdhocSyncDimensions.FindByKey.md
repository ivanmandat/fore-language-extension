# IAdhocSyncDimensions.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IAdhocDimSyncItem](../IAdhocDimSyncItem/IAdhocDimSyncItem.htm);


## Параметры


Id. Ключ измерения, входящего
 в индекс синхронизации.


## Описание


Метод FindByKey выполняет поиск
 измерения по его ключу.


## Комментарии


Если измерение не найдено, то метод возвращает значение Null.
