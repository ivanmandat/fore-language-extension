# IAdhocSyncDimensions.Clear


## Синтаксис


Clear;


## Описание


Метод Clear очищает коллекцию
 измерений, входящих в индекс синхронизации.


## Комментарии


После вызова метода значение свойства [IAdhocSyncDimensions.Count](IAdhocSyncDimensions.Count.htm)
 будет обнулено.
