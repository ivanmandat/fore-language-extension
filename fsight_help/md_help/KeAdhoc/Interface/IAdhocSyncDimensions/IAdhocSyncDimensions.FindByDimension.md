# IAdhocSyncDimensions.FindByDimension


## Синтаксис


FindByDimension(AdhocDim: [IAdhocDsoDimension](../IAdhocDsoDimension/IAdhocDsoDimension.htm)):
 [IAdhocDimSyncItem](../IAdhocDimSyncItem/IAdhocDimSyncItem.htm);


## Параметры


AdhocDim. Измерение, входящее
 в индекс синхронизации.


## Описание


Метод FindByDimension выполняет
 поиск измерения, входящего в индекс синхронизации.


## Комментарии


Если измерение не найдено, то метод возвращает значение Null.
