# IUiDashboard.Report


## Синтаксис


Report: [IAdhocReport](../IAdhocReport/IAdhocReport.htm);


## Описание


Свойство Report определяет аналитическую
 панель репозитория, к которой необходимо получить доступ на форме.


## Комментарии


Указанная аналитическая панель будет отображаться в визуальном компоненте,
 для которого текущий компонент UiDashboard
 установлен в качестве источника данных.

Пример использования приведен в описании свойства [IDashboardDocumentViewerBox.Dashboard](../IDashboardDocumentViewerBox/IDashboardDocumentViewerBox.Dashboard.htm).
