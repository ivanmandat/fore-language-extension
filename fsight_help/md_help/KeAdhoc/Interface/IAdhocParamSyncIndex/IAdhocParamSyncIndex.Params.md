# IAdhocParamSyncIndex.Params


## Синтаксис


Params: [IAdhocSyncParams](../IAdhocSyncParams/IAdhocSyncParams.htm);


## Описание


Свойство Params возвращает
 коллекцию параметров, входящих в связь.


## Комментарии


Для установки значения всех параметров, входящих в связь, используйте
 свойство [IAdhocParamSyncIndex.Value](IAdhocParamSyncIndex.Value.htm).
