# IAdhocParamSyncIndex


Сборка: Adhoc;


## Описание


Интерфейс IAdhocParamSyncIndex
 предназначен для работы со связью для синхронизации параметров источников
 данных аналитической панели.


## Иерархия наследования


           [INamedEntity](kesom.chm::/interface/inamedentity/inamedentity.htm)


           IAdhocParamSyncIndex


## Комментарии


Для работы с коллекцией параметров используйте интерфейс [IAdhocParamSynchronization](../IAdhocParamSynchronization/IAdhocParamSynchronization.htm).
