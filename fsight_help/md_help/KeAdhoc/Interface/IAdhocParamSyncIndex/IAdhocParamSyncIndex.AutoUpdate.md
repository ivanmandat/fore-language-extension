# IAdhocParamSyncIndex.AutoUpdate


## Синтаксис


AutoUpdate: Boolean;


## Описание


Свойство AutoUpdate определяет,
 выполняется ли автоматическое значение параметра.


## Комментарии


Допустимые значения:


	- True. Значение параметра
	 применяется автоматически. При применении значения параметра происходит
	 переоткрытие зависимых объектов с заданным значением параметра. Значение
	 параметра определяет свойство [IAdhocParamSyncIndex.Value](IAdhocParamSyncIndex.Value.htm);


	- False. Значение параметра
	 применяется вручную. Для применения значения параметра используйте
	 метод [IAdhocParamSyncIndex.Apply](IAdhocParamSyncIndex.Apply.htm).
