# IAdhocParamSyncIndex.Value


## Синтаксис


Value: Variant;


## Описание


Свойство Value определяет
 значение параметра.


## Комментарии


Для автоматического применения значения параметра установите значение
 свойства [IAdhocParamSyncIndex.AutoUpdate](IAdhocParamSyncIndex.AutoUpdate.htm)
 в True. Для применения значения
 параметра вручную используйте метод [IAdhocParamSyncIndex.Apply](IAdhocParamSyncIndex.Apply.htm).
