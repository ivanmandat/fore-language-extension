# IAdhocParamSyncIndex.Apply


## Синтаксис


Apply;


## Описание


Метод Apply применяет
 значение параметра.


## Комментарии


Метод Apply необходимо применять,
 если свойство [IAdhocParamSyncIndex.AutoUpdate](IAdhocParamSyncIndex.AutoUpdate.htm)
 имеет значение False, то есть
 не происходит автоматическое применение значения параметра.


При применении значения параметра происходит переоткрытие зависимых
 объектов с заданным значением параметра. Значение параметра определяет
 свойство [IAdhocParamSyncIndex.Value](IAdhocParamSyncIndex.Value.htm).
