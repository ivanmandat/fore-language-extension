# IAdhocPageSettings


Сборка: Adhoc;


## Описание


Интерфейс IAdhocPageSettings
 содержит свойства и методы для настройки страницы аналитической панели.


## Иерархия наследования


           [IGxPageSettings](ModDrawing.chm::/Interface/IGxPageSettings/IGxPageSettings.htm)


           IAdhocPageSettings


## Комментарии


Настройки применяются только при печати аналитической панели и не влияют
 на её отображение в рабочей области. Для получения настроек используйте
 свойство [IAdhocReport.PageSettings](../IAdhocReport/IAdhocReport.PageSettings.htm).
