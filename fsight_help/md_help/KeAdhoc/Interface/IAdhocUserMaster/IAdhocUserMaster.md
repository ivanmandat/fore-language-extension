# IAdhocUserMaster


Сборка: Adhoc;


## Описание


Интерфейс IAdhocUserMaster
 предназначен для работы с формой-мастером для плагина.


## Иерархия наследования


           IAdhocUserMaster


## Комментарии


Форма-мастер содержит компоненты, образующие вкладки боковой панели
 для настройки плагина, и код, описывающий логику применения настроек к
 плагину.
