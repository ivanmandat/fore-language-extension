# IAdhocUserView.LoadSettings


## Синтаксис


LoadSettings(xml: Variant);


## Параметры


Xml.
 Загружаемые настройки в формате xml.


## Описание


Метод LoadSettings загружает
 настройки плагина из xml.


## Комментарии


Для сохранения настроек плагина используйте метод [IAdhocUserView.SaveSettings](IAdhocUserView.SaveSettings.htm).
