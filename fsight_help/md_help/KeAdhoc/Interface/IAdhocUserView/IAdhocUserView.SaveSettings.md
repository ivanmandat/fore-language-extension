# IAdhocUserView.SaveSettings


## Синтаксис


SaveSettings(xml: Variant);


## Параметры


Xml. Сохраняемые настройки
 плагина a формате xml.


## Описание


Метод SaveSettings сохраняет
 настройки плагина в формате xml.


## Комментарии


Для загрузки настроек плагина используйте метод [IAdhocUserView.LoadSettings](IAdhocUserView.LoadSettings.htm).
