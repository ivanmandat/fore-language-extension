# IAdhocUserView.SupportedSource


## Синтаксис


SupportedSource: Array;


## Описание


Свойство SupportedSource
 возвращает типы источников данных, поддерживаемых плагином.


## Комментарии


Поддерживаемые типы данных представлены в виде целочисленного массива.
 Каждый элемент массива соответствует типу поддерживаемого источник данных
 и содержит [ClassId](KeSom.chm::/Enums/MetabaseObjectClass.htm), соответствующий данному типу.
