# IAdhocReportExportSources.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемого элемента.


## Описание


Метод Remove удаляет указанный
 элемент из коллекции.


## Комментарии


Индексация элементов сквозная и начинается с нуля. Для получения количества
 элементов в коллекции используйте свойство [IAdhocReportExportSources.Count](IAdhocReportExportSources.Count.htm).


Если удаление выполнено успешно, то метод возвращает значение True, в обратном случае - False.
