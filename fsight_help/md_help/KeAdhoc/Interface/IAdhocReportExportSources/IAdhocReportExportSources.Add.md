# IAdhocReportExportSources.Add


## Синтаксис


Add(Source: [IAdhocDataSourceObject](../IAdhocDataSourceObject/IAdhocDataSourceObject.htm));


## Параметры


Source. Добавляемый источник
 данных.


## Описание


Метод Add добавляет новый
 источник данных в коллекцию.


## Комментарии


Для удаления определенного источника из коллекции используйте метод
 [IAdhocReportExportSources.Remove](IAdhocReportExportSources.Remove.htm),
 для удаления всех источников - метод [IAdhocReportExportSources.Clear](IAdhocReportExportSources.Clear.htm).
