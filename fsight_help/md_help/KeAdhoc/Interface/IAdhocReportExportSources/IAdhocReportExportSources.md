# IAdhocReportExportSources


Сборка: Adhoc;


## Описание


Интерфейс IAdhocReportExportSources
 предназначен для работы с коллекцией экспортируемых источников данных
 аналитической панели.


## Иерархия наследования


           IAdhocReportExportSources


## Комментарии


Для получения коллекции используйте свойство [IAdhocReportExporter.Sources](../IAdhocReportExporter/IAdhocReportExporter.Sources.htm).
