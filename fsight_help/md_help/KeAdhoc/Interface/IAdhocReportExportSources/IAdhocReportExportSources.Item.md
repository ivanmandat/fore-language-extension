# IAdhocReportExportSources.Item


## Синтаксис


Item(Index: Integer): [IAdhocReportExportSource](../IAdhocReportExportSource/IAdhocReportExportSource.htm);


## Параметры


Index. Индекс элемента.


## Описание


Свойство Item возвращает
 указанный элемент из коллекции.


## Комментарии


Индексация элементов сквозная и начинается с нуля. Для получения количества
 элементов в коллекции используйте свойство [IAdhocReportExportSources.Count](IAdhocReportExportSources.Count.htm).
