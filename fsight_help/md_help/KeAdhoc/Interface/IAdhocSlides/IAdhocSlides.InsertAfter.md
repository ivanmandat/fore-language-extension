# IAdhocSlides.InsertAfter


## Синтаксис


InsertAfter(Index: Integer; Name: String): [IAdhocSlide](../IAdhocSlide/IAdhocSlide.htm);


## Параметры


Index. Индекс слайда, после
 которого будет вставлен новый слайд;


Name. Название слайда.


## Описание


Метод InsertAfter добавляет
 слайд в коллекцию слайдов аналитической панели после указанного слайда
 и возвращает слайд аналитической панели.
