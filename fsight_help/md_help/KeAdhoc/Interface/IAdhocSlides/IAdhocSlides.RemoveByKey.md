# IAdhocSlides.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ слайда аналитической
 панели.


## Описание


Метод RemoveByKey удаляет
 слайд аналитической панели по его ключу.


## Комментарии


Допустимые значения:


	- True. Удаление прошло
	 успешно;


	- False. Удаление прошло
	 с ошибкой.
