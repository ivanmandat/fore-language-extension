# IAdhocSlides.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс слайда аналитической
 панели.


## Описание


Метод Remove удаляет слайд
 аналитической панели.


## Комментарии


Допустимые значения:


	- True. Удаление прошло
	 успешно;


	- False. Удаление прошло
	 с ошибкой.
