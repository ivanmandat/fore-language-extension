# IAdhocSlides.FindById


## Синтаксис


FindById(Id: String): [IAdhocSlide](../IAdhocSlide/IAdhocSlide.htm);


## Параметры


Id. Идентификатор слайда.


## Описание


Метод FindById выполняет
 поиск слайда в коллекции слайдов аналитической панели по указанному идентификатору.
