# IAdhocSlides.Move


## Синтаксис


Move(OldIndex: Integer; NewIndex: Integer);


## Параметры


OldIndex. Индекс позиции слайда
 аналитической панели, который необходимо переместить;


NewIndex. Индекс позиции, в
 которую необходимо переместить слайд аналитической панели.


## Описание


Метод Move перемещает
 слайд коллекции слайдов аналитической панели на указанную позицию.
