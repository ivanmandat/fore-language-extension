# IAdhocSlides


Сборка: Adhoc;


## Описание


Интерфейс IAdhocSlides предназначен
 для работы с коллекцией слайдов аналитической панели.


## Иерархия наследования


           IAdhocSlides


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IAdhocSlides.Count.htm)
		 Свойство Count возвращает
		 количество слайдов аналитической панели.


		 ![](../../Property_Image.gif)
		 [Item](IAdhocSlides.Item.htm)
		 Свойство Item возвращает
		 указанный слайд коллекции аналитической панели.


		 ![](../../Property_Image.gif)
		 [Report](IAdhocSlides.Report.htm)
		 Свойство Report возвращает
		 родительскую аналитическую панель.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IAdhocSlides.Add.htm)
		 Метод Add добавляет
		 слайд в коллекцию слайдов аналитической панели.


		 ![](../../Sub_Image.gif)
		 [Clear](IAdhocSlides.Clear.htm)
		 Метод Clear очищает
		 коллекцию слайдов аналитической панели.


		 ![](../../Sub_Image.gif)
		 [FindById](IAdhocSlides.FindById.htm)
		 Метод FindById выполняет
		 поиск слайда в коллекции слайдов аналитической панели по указанному
		 идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IAdhocSlides.FindByKey.htm)
		 Метод FindByKey выполняет
		 поиск слайда в коллекции слайдов аналитической панели по указанному
		 ключу.


		 ![](../../Sub_Image.gif)
		 [IndexOf](IAdhocSlides.IndexOf.htm)
		 Метод IndexOf возвращает
		 индекс указанного слайда аналитической панели.


		 ![](../../Sub_Image.gif)
		 [Insert](IAdhocSlides.Insert.htm)
		 Метод Insert вставляет
		 слайд на аналитическую панель.


		 ![](../../Sub_Image.gif)
		 [InsertAfter](IAdhocSlides.InsertAfter.htm)
		 Метод InsertAfter вставляет
		 слайд на аналитическую панель после указанного слайда и возвращает
		 слайд аналитической панели.


		 ![](../../Sub_Image.gif)
		 [Move](IAdhocSlides.Move.htm)
		 Метод Move перемещает
		 слайд коллекции слайдов аналитической панели на указанную позицию.


		 ![](../../Sub_Image.gif)
		 [Remove](IAdhocSlides.Remove.htm)
		 Метод Remove удаляет
		 слайд аналитической панели.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IAdhocSlides.RemoveByKey.htm)
		 Метод RemoveByKey удаляет
		 слайд аналитической панели по его ключу.
