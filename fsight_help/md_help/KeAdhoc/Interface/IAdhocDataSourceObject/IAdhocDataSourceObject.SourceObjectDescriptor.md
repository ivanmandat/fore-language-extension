# IAdhocDataSourceObject.SourceObjectDescriptor


## Синтаксис


SourceObjectDescriptor: [IMetabaseObjectDescriptor](kesom.chm::/interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm);


## Описание


Свойство SourceObjectDescriptor
 возвращает источник данных, как объект репозитория.


## Комментарии


Для работы с коллекцией значений параметров источника данных используйте
 свойство [IAdhocDataSourceObject.ParamValues](IAdhocDataSourceObject.ParamValues.htm).
