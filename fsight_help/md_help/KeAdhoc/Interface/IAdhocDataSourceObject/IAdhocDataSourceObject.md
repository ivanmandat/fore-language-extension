# IAdhocDataSourceObject


Сборка: Adhoc;


## Описание


Интерфейс IAdhocDataSourceObject
 предназначен для работы с источником данных аналитической панели.


## Иерархия наследования


           IAdhocDataSourceObject


## Комментарии


Для получения всех источников данных аналитической панели используйте
 свойство [IAdhocReport.DataSourceObjects](../IAdhocReport/IAdhocReport.DataSourceObjects.htm).
