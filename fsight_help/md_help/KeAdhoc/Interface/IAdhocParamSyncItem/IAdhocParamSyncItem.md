# IAdhocParamSyncItem


Сборка: Adhoc;


## Описание


Интерфейс IAdhocParamSyncItem
 предназначен для работы с параметром, входящим в связь, предназначенную
 для синхронизации параметров источников данных аналитической панели.


## Иерархия наследования


           [INamedEntity](kesom.chm::/interface/inamedentity/inamedentity.htm)


           IAdhocParamSyncItem


## Комментарии


Для работы с коллекцией параметров, входящих в связь, используйте интерфейс
 [IAdhocParamSyncItem](IAdhocParamSyncItem.htm).
