# IAdhocDimsSynchronization.Add


## Синтаксис


Add(Id: String): [IAdhocDimSyncIndex](../IAdhocDimSyncIndex/IAdhocDimSyncIndex.htm);


## Параметры


Id. Идентификатор измерения.


## Описание


Метод Add добавляет указанное
 измерение в коллекцию.


## Комментарии


В случае успешного выполнения метод возвращает индекс синхронизации,
 соответствующий измерению.
