# IAdhocDimsSynchronization.Clear


## Синтаксис


Clear;


## Описание


Метод Clear очищает коллекцию
 синхронизируемых измерений.


## Комментарии


После вызова метода значение свойства [IAdhocDimsSynchronization.Count](IAdhocDimsSynchronization.Count.htm)
 будет обнулено.
