# IAdhocDimsSynchronization


Сборка: Adhoc;


## Описание


Интерфейс IAdhocDimsSynchronization
 предназначен для работы с коллекцией синхронизируемых измерений.


## Иерархия наследования


           IAdhocDimsSynchronization


## Комментарии


Для получения коллекции используйте свойство [IAdhocSynchronization.Dimensions](../IAdhocSynchronization/IAdhocSynchronization.Dimensions.htm).
