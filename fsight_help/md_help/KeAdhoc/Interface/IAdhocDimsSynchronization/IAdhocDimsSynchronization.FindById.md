# IAdhocDimsSynchronization.FindById


## Синтаксис


FindById(Id: String): [IAdhocDimSyncIndex](../IAdhocDimSyncIndex/IAdhocDimSyncIndex.htm);


## Параметры


Id. Идентификатор измерения
 в коллекции.


## Описание


Метод FindById выполняет поиск
 измерения по его идентификатору.


## Комментарии


Если измерение найдено, то метод возвращает индекс синхронизации, соответствующий
 измерению.
