# IAdhocDimsSynchronization.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IAdhocDimSyncIndex](../IAdhocDimSyncIndex/IAdhocDimSyncIndex.htm);


## Параметры


Key. Ключ измерения.


## Описание


Метод FindByKey выполняет поиск
 измерения по его ключу.


## Комментарии


Если измерение найдено, то метод возвращает индекс синхронизации, соответствующий
 измерению.
