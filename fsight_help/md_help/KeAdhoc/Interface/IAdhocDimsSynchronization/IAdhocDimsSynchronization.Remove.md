# IAdhocDimsSynchronization.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс измерения в коллекции.


## Описание


Метод Remove удаляет указанное
 измерение из коллекции.


## Комментарии


Количество измерений в коллекции возвращает свойство [IAdhocDimsSynchronization.Count](IAdhocDimsSynchronization.Count.htm).
 Если удаление выполнено успешно, то метод возвращает значение True,
 в обратном случае - False.
