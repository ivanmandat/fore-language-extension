# IAdhocDimsSynchronization.RemoveById


## Синтаксис


RemoveById(Id: String): Boolean;


## Параметры


Id. Идентификатор измерения.


## Описание


Метод RemoveById удаляет
 измерение с указанным идентификатором из коллекции.


## Комментарии


Если удаление выполнено успешно, то метод возвращает значение True, в обратном случае - False.
