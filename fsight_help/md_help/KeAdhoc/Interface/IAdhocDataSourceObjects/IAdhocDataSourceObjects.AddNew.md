# IAdhocDataSourceObjects.AddNew


## Синтаксис


AddNew(Id: String; CreateInfo: [IMetabaseObjectCreateInfo](KeSom.chm::/Interface/IMetabaseObjectCreateInfo/IMetabaseObjectCreateInfo.htm)):
 [IAdhocDataSourceObject](../IAdhocDataSourceObject/IAdhocDataSourceObject.htm);


## Параметры


Id. Идентификатор источника
 данных;


CreateInfo. Параметры создаваемого
 объекта репозитория.


## Описание


Метод AddNew создает новый объект
 репозитория и добавляет его в коллекцию источников данных аналитической
 панели.
