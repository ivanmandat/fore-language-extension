# IAdhocDataSourceObjects.FindById


## Синтаксис


FindById(Id:String):[IAdhocDataSourceObject](../IAdhocDataSourceObject/IAdhocDataSourceObject.htm);


## Параметры


Id. Идентификатор искомого
 источника данных.


## Описание


Метод FindById выполняет поиск
 источника данных по его идентификатору.


## Комментарии


Если источник данных не найден, то метод возвращает значение Null.
