# IAdhocDataSourceObjects.Item


## Синтаксис


Item(Index: Integer): [IAdhocDataSourceObject](../IAdhocDataSourceObject/IAdhocDataSourceObject.htm);


## Параметры


Index. Индекс элемента.


## Описание


Свойство Item возвращает
 указанный элемент из коллекции.


## Комментарии


Индексация элементов сквозная и начинается с нуля. Для получения количества
 элементов в коллекции используйте свойство [IAdhocDataSourceObjects.Count](IAdhocDataSourceObjects.Count.htm).
