# IAdhocDataSourceObjects.RemoveById


## Синтаксис


RemoveById(Id: String): Boolean;


## Параметры


Id. Идентификатор источника
 данных.


## Описание


Метод RemoveById удаляет источник
 данных из коллекции по его идентификатору.


## Комментарии


Если удаление выполнено успешно, то метод возвращает значение True, в обратном случае - False.
