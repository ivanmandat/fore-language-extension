# IAdhocDataSourceObjects


Сборка: Adhoc;


## Описание


Интерфейс IAdhocDataSourceObjects
 предназначен для работы с коллекцией источников данных аналитической
 панели.


## Иерархия наследования


           IAdhocDataSourceObjects


## Комментарии


Для получения коллекции используйте свойство [IAdhocReport.DataSourceObjects](../IAdhocReport/IAdhocReport.DataSourceObjects.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AutoSyncNewSource](IAdhocDataSourceObjects.AutoSyncNewSource.htm)
		 Свойство AutoSyncNewSource
		 определяет, выполнять ли автоматическое связывание и синхронизацию
		 измерений источников данных.


		 ![](../../Property_Image.gif)
		 [AutoUpdateSources](IAdhocDataSourceObjects.AutoUpdateSources.htm)
		 Свойство AutoUpdateSources
		 определяет, обновлять ли автоматически источники данных аналитической
		 панели.


		 ![](../../Property_Image.gif)
		 [Count](IAdhocDataSourceObjects.Count.htm)
		 Свойство Count возвращает
		 количество элементов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IAdhocDataSourceObjects.Item.htm)
		 Свойство Item возвращает
		 указанный элемент из коллекции.


		 ![](../../Property_Image.gif)
		 [IsLinked](IAdhocDataSourceObjects.IsLinked.htm)
		 Свойство IsLinked возвращает
		 признак того, является ли источник связанным.


		 ![](../../Property_Image.gif)
		 [Report](IAdhocDataSourceObjects.Report.htm)
		 Свойство Report возвращает
		 родительскую аналитическую панель.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IAdhocDataSourceObjects.Add.htm)

		 объект репозитория в коллекцию источников данных аналитической

		 ![](../../Sub_Image.gif)
		 [AddNew](IAdhocDataSourceObjects.AddNew.htm)
		 Метод AddNew создает
		 новый объект репозитория и добавляет его в коллекцию источников
		 данных аналитической панели.


		 ![](../../Sub_Image.gif)
		 [Clear](IAdhocDataSourceObjects.Clear.htm)
		 Метод Clear очищает
		 коллекцию.


		 ![](../../Sub_Image.gif)
		 [FindById](IAdhocDataSourceObjects.FindById.htm)
		 Метод FindById выполняет
		 поиск источника данных по его идентификатору.


		 ![](../../Sub_Image.gif)
		 [Remove](IAdhocDataSourceObjects.Remove.htm)
		 Метод Remove удаляет
		 указанный источника данных из коллекции.


		 ![](../../Sub_Image.gif)
		 [RemoveById](IAdhocDataSourceObjects.RemoveById.htm)
		 Метод RemoveById удаляет
		 источник данных из коллекции по его идентификатору.
