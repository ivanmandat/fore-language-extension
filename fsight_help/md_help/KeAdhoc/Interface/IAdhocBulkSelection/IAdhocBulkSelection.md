# IAdhocBulkSelection


Сборка: Adhoc;


## Описание


Интерфейс IAdhocBulkSelection предназначен для
 работы с отметкой измерений.


## Иерархия наследования


           IAdhocBulkSelection


## Комментарии


Свойства интерфейса определяют измерения и отметку, которые используются
 для выполнения пакетной операции.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dimension](IAdhocBulkSelection.Dimension.htm)


		 Свойство Dimension
		 определяет измерение источника данных аналитической панели, которое
		 используется для выполнения пакетной операции.


		 ![](../../Property_Image.gif)
		 [Selection](IAdhocBulkSelection.Selection.htm)


		 Свойство Selection
		 возвращает отметку измерения, которая используется для выполнения
		 пакетной операции.
