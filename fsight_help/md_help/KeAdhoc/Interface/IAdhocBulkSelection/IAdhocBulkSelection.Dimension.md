# IAdhocBulkSelection.Dimension


## Синтаксис


Dimension: [IAdhocDsoDimension](../IAdhocDsoDimension/IAdhocDsoDimension.htm);


## Описание


Свойство Dimension определяет
 измерение источника данных аналитической панели, которое используется
 для выполнения пакетной операции.


## Комментарии


Для выбора отметки измерения используйте свойство [IAdhocBulkSelection.Selection](IAdhocBulkSelection.Selection.htm).
