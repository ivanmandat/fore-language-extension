# IAdhocBulkSelection.Selection


## Синтаксис


Selection: [IDimSelection](KeDims.chm::/interface/IDimSelection/IDimSelection.htm);


## Описание


Свойство Selection возвращает
 отметку измерения, которая используется для выполнения пакетной операции.


## Комментарии


Для выбора измерения используйте свойство [IAdhocBulkSelection.Dimension](IAdhocBulkSelection.Dimension.htm).

Использование свойства приведено в примере для [IAdhocBulkSettings.SelectionSet](../IAdhocBulkSettings/IAdhocBulkSettings.SelectionSet.htm).
