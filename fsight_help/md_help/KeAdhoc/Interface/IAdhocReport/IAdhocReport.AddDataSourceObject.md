# IAdhocReport.AddDataSourceObject


## Синтаксис


AddDataSourceObject(Id: String; Object: [IMetabaseObject](KeSom.chm::/Interface/IMetabaseObject/IMetabaseObject.htm)):
 [IAdhocDataSourceObject](../IAdhocDataSourceObject/IAdhocDataSourceObject.htm);


## Параметры


Id. Идентификатор объекта аналитической
 панели;


Object. Объект репозитория.


## Описание


Метод AddDataSourceObject добавляет
 источник данных для визуализаторов.


## Комментарии


Для удаления источника данных используйте метод [IAdhocDataSourceObjects.RemoveById](../IAdhocDataSourceObjects/IAdhocDataSourceObjects.RemoveById.htm).
