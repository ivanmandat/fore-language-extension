# IAdhocReport.MetabaseObject


## Синтаксис


MetabaseObject: [IMetabaseObject](KeSom.chm::/Interface/IMetabaseObject/IMetabaseObject.htm);


## Описание


Свойство MetabaseObject возвращает
 аналитическую панель как объект репозитория, реализуемый интерфейсом [IMetabaseObject](KeSom.chm::/Interface/IMetabaseObject/IMetabaseObject.htm).


## Комментарии


Полученный контекст может использоваться для изменения различных параметров
 аналитической панели как объекта репозитория, например настройки прав
 доступа.
