# IAdhocReport.CopyDataSourceObjectEx


## Синтаксис


CopyDataSourceObjectEx(Source: [IAdhocDataSourceObject](../IAdhocDataSourceObject/IAdhocDataSourceObject.htm);
 Id: String): [IAdhocDataSourceObject](../IAdhocDataSourceObject/IAdhocDataSourceObject.htm);


## Параметры


Source. Копируемый источник
 данных;


Id. Идентификатор копии
 источника данных


## Описание


Метод CopyDataSourceObjectEx
 копирует указанный источник данных, создавая для копии идентификатор,
 заданный пользователем.


## Комментарии


Для добавления источника данных используйте метод [IAdhocReport.AddDataSourceObject](IAdhocReport.AddDataSourceObject.htm).
