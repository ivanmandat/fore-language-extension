# IAdhocReport.DataSourceObjects


## Синтаксис


DataSourceObjects: [IAdhocDataSourceObjects](../IAdhocDataSourceObjects/IAdhocDataSourceObjects.htm);


## Описание


Свойство DataSourceObjects возвращает
 набор источников данных для визуализаторов аналитической панели.


## Комментарии


Каждый элемент в наборе является блоком, для которого задан источник
 данных. Для возвращения источников данных для блока типа «Экспресс-отчет»,
 «Регламентный отчет» или «Аналитическая панель» используйте
 свойство [IAdhocReport.DataSourceLinkedObjects](IAdhocReport.DataSourceLinkedObjects.htm).
