# IAdhocReport.SupportMultyLanguage


## Синтаксис


SupportMultyLanguage: Boolean;


## Описание


Свойство SupportMultyLanguage
 определяет, является ли аналитическая панель мультиязычной.


## Комментарии


Допустимые значения:


	- True. Аналитическая
	 панель является [мультиязычной](UiNav.chm::/Multilanguage/Multilanguage_tuning.htm).
	 Объект «Ресурсы», используемый для хранения переводов, определяет
	 свойство [IAdhocReport.Resources](IAdhocReport.Resources.htm).
	 Если объект не задан, то он будет создан автоматически в качестве
	 дочернего объекта аналитической панели;


	- False. Аналитическая
	 панель является одноязычной.
