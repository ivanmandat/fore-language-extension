# IAdhocReport.RefreshDataSources


## Синтаксис


RefreshDataSources(Value: Object);


## Параметры


Value. Источник данных аналитической
 панели.


## Описание


Метод RefreshDataSources обновляет
 метаданные указанного источника данных аналитической панели.


## Комментарии


Для обновления данных указанного источника полностью используйте свойство
 [IAdhocReport.Refresh](IAdhocReport.Refresh.htm).
