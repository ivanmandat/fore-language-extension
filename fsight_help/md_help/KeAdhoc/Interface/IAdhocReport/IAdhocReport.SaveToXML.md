# IAdhocReport.SaveToXML


## Синтаксис


SaveToXML(Destination: Variant);


## Параметры


Destination. Путь и наименование
 XML-файла, в который будут сохранены параметры аналитической панели.


## Описание


Метод SaveToXML осуществляет
 сохранение настроек аналитической панели в указанный XML-файл.


## Комментарии


В дальнейшем сохраненные настройки можно будет загрузить используя метод
 [LoadFromXML](IAdhocReport.LoadFromXML.htm).
