# IAdhocReport.DataSourceLinkedObjects


## Синтаксис


DataSourceLinkedObjects: [IAdhocDataSourceObjects](../IAdhocDataSourceObjects/IAdhocDataSourceObjects.htm);


## Описание


Свойство DataSourceLinkedObjects возвращает набор
 связанных источников данных аналитической панели.


## Комментарии


К связанным источникам данных относятся источники данных для блоков типа
 «Экспресс-отчет», «Регламентный
 отчет» или «Аналитическая панель».
