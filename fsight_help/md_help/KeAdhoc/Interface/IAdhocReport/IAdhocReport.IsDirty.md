# IAdhocReport.IsDirty


## Синтаксис


IsDirty: Boolean;


## Описание


Свойство IsDirty определяет,
 присутствуют ли изменения в параметрах аналитической панели.


## Комментарии


Допустимые значения


	- True. Параметры аналитической
	 панели изменены;


	- False. Изменения отсутствуют.
