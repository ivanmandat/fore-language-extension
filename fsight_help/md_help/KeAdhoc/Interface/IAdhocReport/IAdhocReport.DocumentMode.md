# IAdhocReport.DocumentMode


## Синтаксис


DocumentMode: [AdhocDocumentModeType](../../Enums/AdhocDocumentModeType.htm);


## Описание


Свойство DocumentMode определяет
 тип аналитической панели.


## Комментарии


Аналитическая панель может быть со [слайдами](UiAdhoc.chm::/Presentation/Presentation.htm)
 и без слайдов.


Примечание.
 Отображение слайдов и работа с ними доступна только веб-приложении.


От заданного типа аналитической панели зависит отображение [панели слайдов](UIAdhoc.chm::/UI/Slaide_Panel.htm).
 Для изменения типа аналитической панели на панели слайдов должно содержаться
 не более одного слайда.
