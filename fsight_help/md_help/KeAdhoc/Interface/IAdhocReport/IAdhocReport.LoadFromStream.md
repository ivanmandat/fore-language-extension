# IAdhocReport.LoadFromStream


## Синтаксис


LoadFromStream(Stream: [IIOStream](ModIo.chm::/Interface/IIOStream/IIOStream.htm));


## Параметры


Stream. Поток, из которого
 будет загруженена аналитическая панель.


## Описание


Метод LoadFromStream осуществляет
 загрузку аналитической панели из потока.


## Комментарии


Для сохранения аналитической панели в поток используйте метод [IAdhocReport.SaveToStream](IAdhocReport.SaveToStream.htm).
