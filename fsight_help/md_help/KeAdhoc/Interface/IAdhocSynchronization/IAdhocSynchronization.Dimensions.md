# IAdhocSynchronization.Dimensions


## Синтаксис


Dimensions: [IAdhocDimsSynchronization](../IAdhocDimsSynchronization/IAdhocDimsSynchronization.htm);


## Описание


Свойство Dimensions возвращает
 коллекцию синхронизируемых измерений.


## Комментарии


При добавлении нового источника данных, его измерения автоматически
 добавляются данную коллекцию.
