# IAdhocSynchronization


Сборка: Adhoc;


## Описание


Интерфейс IAdhocSynchronization предназначен
 для работы с параметрами синхронизации измерений аналитической панели.


## Иерархия наследования


           IAdhocSynchronization


## Комментарии


Для получения параметров синхронизации используйте свойство [IAdhocReport.Synchronization](../IAdhocReport/IAdhocReport.Synchronization.htm).
