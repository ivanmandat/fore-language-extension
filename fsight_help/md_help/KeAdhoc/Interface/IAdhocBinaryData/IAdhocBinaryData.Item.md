# IAdhocBinaryData.Item


## Синтаксис


Item(Index: Integer): [IAdhocBinaryItem](../IAdhocBinaryItem/IAdhocBinaryItem.htm);


## Параметры


Index. Индекс элемента коллекции.


## Описание


Свойство Item возвращает элемент
 коллекции бинарных данных.


## Комментарии


Количество элементов коллекции возвращает свойство [IAdhocBinaryData.Count](IAdhocBinaryData.Count.htm).
 Нумерация сквозная и начинается с нуля.
