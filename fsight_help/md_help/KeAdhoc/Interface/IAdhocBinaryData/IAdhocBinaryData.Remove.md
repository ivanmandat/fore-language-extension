# IAdhocBinaryData.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс элемента коллекции.


## Описание


Метод Remove удаляет указанный
 элемент коллекции бинарных данных аналитической панели.


## Комментарии


Для добавления нового элемента используйте метод [IAdhocBinaryData.Add](IAdhocBinaryData.Add.htm).


Допустимые значения:


	- True. Удаление прошло
	 успешно;


	- False. Удаление прошло
	 с ошибками.
