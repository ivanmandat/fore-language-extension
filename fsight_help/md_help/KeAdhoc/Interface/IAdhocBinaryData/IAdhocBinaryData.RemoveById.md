# IAdhocBinaryData.RemoveById


## Синтаксис


RemoveById(Id: String): Boolean;


## Параметры


Id. Идентификатор элемента
 коллекции бинарных данных аналитической панели.


## Описание


Метод RemoveById удаляет элемент
 коллекции бинарных данных с указанным идентификатором.


## Комментарии


Допустимые значения:


	- True. Удаление выполнено
	 успешно;


	- False. Удаление выполнено
	 с ошибками.
