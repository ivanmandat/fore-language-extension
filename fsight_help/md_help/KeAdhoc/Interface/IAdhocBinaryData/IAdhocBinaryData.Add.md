# IAdhocBinaryData.Add


## Синтаксис


Add: [IAdhocBinaryItem](../IAdhocBinaryItem/IAdhocBinaryItem.htm);


## Описание


Метод Add добавляет элемент
 в коллекцию бинарных данных аналитической панели.


## Комментарии


Для удаления определенного объекта используйте метод [IAdhocBinaryData.RemoveById](IAdhocBinaryData.RemoveById.htm),
 для удаления всех элементов - метод [IAdhocBinaryData.Clear](IAdhocBinaryData.Clear.htm).
