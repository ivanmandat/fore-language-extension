# IAdhocBinaryData


Сборка: Adhoc;


## Описание


Интерфейс IAdhocBinaryData предназначен
 для работы с бинарными данными аналитической панели.


## Иерархия наследования


           IAdhocBinaryData


## Комментарии


Для получения свойств бинарных данных используйте свойство [IAdhocReport.BinaryData](../IAdhocReport/IAdhocReport.BinaryData.htm).


При добавлении в коллекцию бинарных данных создаются связи данных с
 аналитической панелью, а сами данные сохраняются как дочерний документ
 панели.
