# IAdhocBinaryData.Clear


## Синтаксис


Clear;


## Описание


Метод Clear выполняет очистку
 коллекции бинарных данных аналитической панели.


## Комментарии


После выполнения метода свойство [IAdhocBinaryData.Count](IAdhocBinaryData.Count.htm)
 принимает значение «0».
