# IAdhocBulkSelectionSet


Сборка: Adhoc;


## Описание


Интерфейс IAdhocBulkSelectionSet предназначен для
 работы с коллекцией фиксированных измерений.


## Иерархия наследования


           IAdhocBulkSelectionSet


## Комментарии


Для получения коллекции фиксированных измерений используйте свойство
 [IAdhocBulkSettings.SelectionSet](../IAdhocBulkSettings/IAdhocBulkSettings.SelectionSet.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IAdhocBulkSelectionSet.Count.htm)


		 Свойство Count возвращает
		 количество измерений в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IAdhocBulkSelectionSet.Item.htm)


		 Свойство Item возвращает
		 указанное фиксированное измерение, входящее в коллекцию.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IAdhocBulkSelectionSet.Add.htm)


		 Метод Add добавляет
		 новое фиксированное измерение, которое используется для выполнения
		 пакетной операции.


		 ![](../../Sub_Image.gif)
		 [Clear](IAdhocBulkSelectionSet.Clear.htm)


		 Метод Clear удаляет
		 все измерения из коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IAdhocBulkSelectionSet.Remove.htm)


		 Метод Remove удаляет
		 измерение из коллекции.
