## Синтаксис

## Описание


Метод Clear удаляет все измерения
 из коллекции.


## Комментарии


После выполнения метода свойство [IAdhocBulkSelectionSet.Count](IAdhocBulkSelectionSet.Count.htm)
 принимает значение «0».
