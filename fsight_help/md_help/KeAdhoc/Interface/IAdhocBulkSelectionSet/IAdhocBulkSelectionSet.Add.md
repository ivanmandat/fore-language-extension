# IAdhocBulkSelectionSet.Add


## Синтаксис


Add(Value: [IAdhocDsoDimension](../IAdhocDsoDimension/IAdhocDsoDimension.htm)): [IAdhocBulkSelection](../IAdhocBulkSelection/IAdhocBulkSelection.htm);


## Параметры


Value. Измерение источника
 данных аналитической панели.


## Описание


Метод Add добавляет новое фиксированное
 измерение, которое используется для выполнения пакетной операции.


## Комментарии


Для получения свойств определенного измерения используйте [IAdhocBulkSelectionSet.Item](IAdhocBulkSelectionSet.Item.htm).
