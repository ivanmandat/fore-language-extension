# IAdhocBulkSelectionSet.Item


## Синтаксис


Item(Index: Integer): [IAdhocBulkSelection](../IAdhocBulkSelection/IAdhocBulkSelection.htm);


## Параметры


Index. Индекс измерения в коллекции.


## Описание


Свойство Item возвращает указанное
 фиксированное измерение, входящее в коллекцию.


## Комментарии


Количество измерений в коллекции возвращает свойство [IAdhocBulkSelectionSet.Count](IAdhocBulkSelectionSet.Count.htm).
 Индексация измерений сквозная и начинается с нуля.
