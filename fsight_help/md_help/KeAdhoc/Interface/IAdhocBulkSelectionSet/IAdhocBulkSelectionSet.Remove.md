# IAdhocBulkSelectionSet.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемого
 измерения.


## Описание


Метод Remove удаляет измерение
 из коллекции.


## Комментарии


Количество измерений в коллекции возвращает свойство [IAdhocBulkSelectionSet.Count](IAdhocBulkSelectionSet.Count.htm).
 Индексация измерений сквозная и начинается с нуля.


Допустимые значения:


	- True. Удаление выполнено
	 успешно;


	- False. Удаление выполнено
	 с ошибкой
