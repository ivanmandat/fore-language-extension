# IAdhocDsoDimensions.FindById


## Синтаксис


FindById(Id: String): [IAdhocDsoDimension](../IAdhocDsoDimension/IAdhocDsoDimension.htm);


## Параметры


Id. Идентификатор измерения.


## Описание


Метод FindById выполняет
 поиск измерения по его идентификатору.


## Комментарии


Если измерение не найдено, то метод возвращает значение Null.
