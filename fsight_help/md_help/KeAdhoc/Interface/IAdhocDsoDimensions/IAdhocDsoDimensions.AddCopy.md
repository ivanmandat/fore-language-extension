# IAdhocDsoDimensions.AddCopy


## Синтаксис


AddCopy(AdhocDimKey: Integer; CopySelection: Boolean):
 [IAdhocDsoDimension](../IAdhocDsoDimension/IAdhocDsoDimension.htm);


## Параметры


AdhocDimKey. Ключ измерения;


CopySelection. Признак того,
 что измерение копируется вместе с отметкой элементов.


## Описание


Метод AddCopy добавляет
 в коллекцию копию указанного измерения.


## Комментарии


Допустимые значения параметра CopySelection:


	- True. Измерение копируется
	 вместе с отметкой элементов;


	- False. Измерение копируется
	 без отметки элементов.
