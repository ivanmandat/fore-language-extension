# IAdhocDsoDimensions.FindByKey


## Синтаксис


FindByKey(Key: String): [IAdhocDsoDimension](../IAdhocDsoDimension/IAdhocDsoDimension.htm);


## Параметры


Key. Ключ измерения.


## Описание


Метод FindByKey выполняет поиск
 измерения по его ключу.


## Комментарии


Если измерение не найдено, то метод возвращает значение Null.
