# IAdhocDsoDimensions.Item


## Синтаксис


Item(Index: Integer): [IAdhocDsoDimension](../IAdhocDsoDimension/IAdhocDsoDimension.htm);


## Параметры


Index. Индекс измерения в коллекции.


## Описание


Свойство Item возвращает
 указанное измерение, входящее в коллекцию.


## Комментарии


Количество измерений в коллекции возвращает свойство [IAdhocDsoDimensions.Count](IAdhocDsoDimensions.Count.htm).
 Индексация измерений сквозная и начинается с нуля.
