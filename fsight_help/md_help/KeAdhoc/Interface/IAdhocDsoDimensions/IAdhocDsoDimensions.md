# IAdhocDsoDimensions


Сборка: Adhoc;


## Описание


Интерфейс IAdhocDsoDimensions
 предназначен для работы с коллекцией измерений для источника данных
 аналитической панели.


## Иерархия наследования


           IAdhocDsoDimensions


## Комментарии


Для получения коллекции измерений используйте свойство [IAdhocDataSourceObject.Dimensions](../IAdhocDataSourceObject/IAdhocDataSourceObject.Dimensions.htm).
