# IAdhocDsoDimensions.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ измерения.


## Описание


Метод RemoveByKey удаляет измерение
 из коллекции по его ключу.


## Комментарии


Если удаление выполнено успешно, то метод возвращает значение True, в обратном случае - False.
