# IAdhocDsoDimensions.RemoveById


## Синтаксис


RemoveById(Id: String): Boolean;


## Описание


Id. Идентификатор измерения.


## Описание


Метод RemoveById удаляет измерение
 из коллекции по его идентификатору.


## Комментарии


Если удаление выполнено успешно, то метод возвращает значение True, в обратном случае - False.
