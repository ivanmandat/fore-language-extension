# IAdhocDimSyncItem.SyncAttribute


## Синтаксис


SyncAttribute: String;


## Описание


Свойство SyncAttribute определяет
 атрибут, по которому выполняется синхронизация.


## Комментарии


Для получения измерения, по которому выполняется синхронизация, используйте
 свойство [IAdhocDimSyncItem.Dimension](IAdhocDimSyncItem.Dimension.htm).
