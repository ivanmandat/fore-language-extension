# IAdhocDimSyncItem.Dimension


## Синтаксис


Dimension: [IAdhocDsoDimension](../IAdhocDsoDimension/IAdhocDsoDimension.htm);


## Описание


Свойство Dimension возвращает
 измерение источника данных, соответствующее синхронизируемому измерению.


## Комментарии


Для получения атрибута, по которому выполняется синхронизация, используйте
 свойство [IAdhocDimSyncItem.SyncAttribute](IAdhocDimSyncItem.SyncAttribute.htm).
