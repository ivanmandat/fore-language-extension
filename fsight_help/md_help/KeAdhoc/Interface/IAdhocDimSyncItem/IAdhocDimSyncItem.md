# IAdhocDimSyncItem


Сборка: Adhoc;


## Описание


Интерфейс IAdhocDimSyncItem
 предназначен для работы с синхронизируемым измерением.


## Иерархия наследования


           [INamedEntity](kesom.chm::/interface/inamedentity/inamedentity.htm)


           IAdhocDimSyncItem


## Комментарии


Синхронизация выполняется с помощью индексов синхронизации: [IAdhocDimSyncIndex](../IAdhocDimSyncIndex/IAdhocDimSyncIndex.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dimension](IAdhocDimSyncItem.Dimension.htm)


		 Свойство Dimension
		 возвращает измерение источника данных, соответствующее синхронизируемому
		 измерению.


		 ![](../../Property_Image.gif)
		 [Direction](IAdhocDimSyncItem.Direction.htm)


		 Свойство Direction
		 определяет тип связи синхронизируемого измерения.


		 ![](../../Property_Image.gif)
		 [IsGeneral](IAdhocDimSyncItem.IsGeneral.htm)


		 Свойство IsGeneral
		 определяет, является ли синхронизируемое измерение основным.


		 ![](../../Property_Image.gif)
		 [SyncAttribute](IAdhocDimSyncItem.SyncAttribute.htm)


		 Свойство SyncAttribute
		 определяет атрибут, по которому выполняется синхронизация.


		 ![](../../Property_Image.gif)
		 [SyncIndex](IAdhocDimSyncItem.SyncIndex.htm)


		 Свойство SyncIndex
		 возвращает индекс синхронизации, в который используется
		 для синхронизации измерения.
