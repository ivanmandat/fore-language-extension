# IAdhocSyncParams.RemoveById


## Синтаксис


RemoveById(Id: String): Boolean;


## Параметры


Id. Идентификатор удаляемого
 параметра.


## Описание


Метод RemoveById удаляет
 из коллекции параметр с указанным идентификатором.


## Комментарии


Если удаление выполнено успешно метод возвращает значение True,
 в противном случае - False.
