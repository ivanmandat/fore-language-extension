# IAdhocSyncParams.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемого параметра.


## Описание


Метод Remove удаляет указанный
 параметр из коллекции.


## Комментарии


Индексация параметров в коллекции сквозная и начинается с нуля. Для
 получения количества параметров в коллекции используйте свойство [IAdhocSyncParams.Count](IAdhocSyncParams.Count.htm).


Если удаление выполнено успешно метод возвращает значение True,
 в противном случае - False.
