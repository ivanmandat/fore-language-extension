# IAdhocSyncParams


Сборка: Adhoc;


## Описание


Интерфейс IAdhocSyncParams предназначен
 для работы с коллекцией параметров, входящих в связь, предназначенную
 для синхронизации параметров источников данных аналитической панели.


## Иерархия наследования


           IAdhocSyncParams


## Комментарии


Для получения коллекции параметров используйте свойство [IAdhocParamSyncIndex.Params](../IAdhocParamSyncIndex/IAdhocParamSyncIndex.Params.htm).
