# IAdhocSyncParams.FindById


## Синтаксис


FindById(Id: String): [IAdhocParamSyncItem](../IAdhocParamSyncItem/IAdhocParamSyncItem.htm);


## Параметры


Id. Идентификатор искомого
 параметра.


## Описание


Метод FindById выполняет
 поиск параметра с заданным идентификатором.


## Комментарии


Если параметр с указанным идентификатором не найден, то метод вернет
 значение Null.
