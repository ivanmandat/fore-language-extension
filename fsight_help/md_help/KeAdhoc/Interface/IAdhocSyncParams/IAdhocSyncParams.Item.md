# IAdhocSyncParams.Item


## Синтаксис


Item(Index: Integer): [IAdhocParamSyncItem](../IAdhocParamSyncItem/IAdhocParamSyncItem.htm);


## Параметры


Index. Индекс параметра в коллекции.


## Описание


Свойство Item возвращает
 указанны параметр из коллекции.


## Комментарии


Индексация параметров в коллекции сквозная и начинается с нуля. Для
 получения количества параметров в коллекции используйте свойство [IAdhocSyncParams.Count](IAdhocSyncParams.Count.htm).
