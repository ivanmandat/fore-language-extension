# IAdhocSyncParams.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IAdhocParamSyncItem](../IAdhocParamSyncItem/IAdhocParamSyncItem.htm);


## Параметры


Key. Ключ искомого параметра.


## Описание


Метод FindByKey выполняет поиск
 параметра с заданным ключом.


## Комментарии


Если параметр с указанным ключом не найдена, то метод вернет значение
 Null.
