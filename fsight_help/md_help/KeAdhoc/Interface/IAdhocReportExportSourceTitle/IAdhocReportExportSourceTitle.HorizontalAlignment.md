# IAdhocReportExportSourceTitle.HorizontalAlignment


## Синтаксис


HorizontalAlignment: [AdhocHorizontalAlignment](../../Enums/AdhocHorizontalAlignment.htm);


## Описание


Свойство HorizontalAlignment
 определяет горизонтальное выравнивание заголовка относительно границ блока.


## Комментарии


По умолчанию заголовок выравнен по левой границе блока, т.е. HorizontalAlignment = AdhocHorizontalAlignment.Left.
