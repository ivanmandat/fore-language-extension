# IAdhocReportExportSourceTitle


Сборка: Adhoc;


## Описание


Интерфейс IAdhocReportExportSourceTitle
 предназначен для работы с параметрами экспорта заголовка блока.


## Иерархия наследования


           IAdhocReportExportSourceTitle


## Комментарии


Для получения параметров экспорта заголовка используйте свойство [IAdhocReportExportSource.TitleSettings](../IAdhocReportExportSource/IAdhocReportExportSource.TitleSettings.htm).
