# IAdhocReportExportSource.DataSource


## Синтаксис


DataSource: [IAdhocDataSourceObject](../IAdhocDataSourceObject/IAdhocDataSourceObject.htm);


## Описание


Свойство DataSource возвращает
 источник данных блока.


## Комментарии


Значение свойства устанавливается автоматически после добавления источника
 данных в коллекцию экспортируемых источников.


Для добавления источника данных в коллекцию используйте метод [IAdhocReportExportSources.Add](../IAdhocReportExportSources/IAdhocReportExportSources.Add.htm).
