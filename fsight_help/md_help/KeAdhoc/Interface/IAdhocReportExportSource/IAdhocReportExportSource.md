# IAdhocReportExportSource


Сборка: Adhoc;


## Описание


Интерфейс IAdhocReportExportSource
 предназначен для работы с параметрами экспорта источника данных
 аналитической панели.


## Иерархия наследования


           IAdhocReportExportSource


## Комментарии


Экспортируемый источник данных - это блок аналитической панели, для
 которого задан источник данных.


Для работы с коллекцией экспортируемых источников данных аналитической
 панели используйте интерфейс [IAdhocReportExportSources](../IAdhocReportExportSources/IAdhocReportExportSources.htm).
