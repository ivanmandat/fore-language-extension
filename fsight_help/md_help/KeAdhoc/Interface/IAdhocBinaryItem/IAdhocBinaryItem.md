# IAdhocBinaryItem


Сборка: Adhoc;


## Описание


Интерфейс IAdhocBinaryItem предназначен для
 работы с элементом коллекции бинарных данных аналитической панели.


## Иерархия наследования


           [INamedEntity](KeSom.chm::/Interface/INamedEntity/INamedEntity.htm)


           IAdhocBinaryItem


## Комментарии


Для получения параметров элемента коллекции бинарных данных используйте
 свойство [IAdhocBinaryData.Item](../IAdhocBinaryData/IAdhocBinaryData.Item.htm).
