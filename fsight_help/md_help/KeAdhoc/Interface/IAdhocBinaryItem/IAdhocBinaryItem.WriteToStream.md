# IAdhocBinaryItem.WriteToStream


## Синтаксис


WriteToStream(Stream: [IIOStream](ModIo.chm::/Interface/IIOStream/IIOStream.htm));


## Параметры


Stream. Поток для записи данных.


## Описание


Метод WriteToStream выполняет
 запись элемента коллекции бинарных данных в поток.


## Комментарии


Для чтения данных, записанных в поток, используйте метод [IAdhocBinaryItem.ReadFromStream](IAdhocBinaryItem.ReadFromStream.htm).
