# IAdhocDsoDimension


Сборка: Adhoc;


## Описание


Интерфейс IAdhocDsoDimension
 предназначен для работы с измерением источника данных аналитической
 панели.


## Иерархия наследования


           [INamedEntity](kesom.chm::/interface/inamedentity/inamedentity.htm)


           IAdhocDsoDimension


## Комментарии


Для работы с источником данных аналитической панели используйте интерфейс
 [IAdhocDataSourceObject](../IAdhocDataSourceObject/IAdhocDataSourceObject.htm).
