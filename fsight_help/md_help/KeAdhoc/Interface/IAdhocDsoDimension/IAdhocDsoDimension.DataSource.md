## Описание


Свойство DataSource возвращает
 источник данных аналитической панели, содержащий измерение.


## Комментарии


Для получения доступа к данным измерения используйте свойство [IAdhocDsoDimension.Dimension](IAdhocDsoDimension.Dimension.htm).
