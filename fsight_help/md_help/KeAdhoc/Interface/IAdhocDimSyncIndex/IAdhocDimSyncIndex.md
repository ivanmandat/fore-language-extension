# IAdhocDimSyncIndex


Сборка: Adhoc;


## Описание


Интерфейс IAdhocDimSyncIndex
 предназначен для работы с индексом синхронизации.


## Иерархия наследования


           [INamedEntity](kesom.chm::/interface/inamedentity/inamedentity.htm)


           IAdhocDimSyncIndex


## Комментарии


Индексы синхронизации создаются автоматически при добавлении источника
 данных в аналитическую панель.

		 [ControlledBy](IAdhocDimSyncIndex.ControlledBy.htm)

		 Свойство Dimensions

		 ![](../../Property_Image.gif)
		 [GeneralDimSyncItem](IAdhocDimSyncIndex.GeneralDimSyncItem.htm)


		 Свойство GeneralDimSyncItem

		 ![](../../Property_Image.gif)
		 [Selection](IAdhocDimSyncIndex.Selection.htm)


		 Свойство Selection
		 возвращает отметку элементов для измерений, входящих в индекс

		 ![](../../Property_Image.gif)
		 [SelectionProperty](IAdhocDimSyncIndex.SelectionProperty.htm)


		 Свойство SelectionProperty
		 определяет, сохранять ли группы и схемы отметки, примененные к
		 измерениям, входящим в индекс синхронизации.
