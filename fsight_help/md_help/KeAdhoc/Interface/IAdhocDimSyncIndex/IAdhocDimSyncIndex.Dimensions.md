# IAdhocDimSyncIndex.Dimensions


## Синтаксис

## Описание


Свойство Dimensions возвращает
 коллекцию измерений, входящих в индекс синхронизации.


## Комментарии


Отметку измерений возвращает свойство [IAdhocDimSyncIndex.Selection](IAdhocDimSyncIndex.Selection.htm).
