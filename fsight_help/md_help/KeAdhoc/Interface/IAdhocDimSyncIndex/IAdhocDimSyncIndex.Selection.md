# IAdhocDimSyncIndex.Selection


## Синтаксис


Selection: [IDimSelection](KeDims.chm::/Interface/IDimSelection/IDimSelection.htm);


## Описание


Свойство Selection возвращает
 отметку элементов для измерений, входящих в индекс синхронизации.


## Комментарии


Для получения измерений, входящих в индекс синхронизации, используйте
 свойство [IAdhocDimSyncIndex.Dimensions](IAdhocDimSyncIndex.Dimensions.htm).
