# IAdhocDimSyncIndex.GeneralDimSyncItem


## Синтаксис


GeneralDimSyncItem: [IAdhocDimSyncItem](../IAdhocDimSyncItem/IAdhocDimSyncItem.htm);


## Описание


Свойство GeneralDimSyncItem
 определяет основное измерение в индексе синхронизации.


## Комментарии


В индексе синхронизации может быть только одно основное измерение.
