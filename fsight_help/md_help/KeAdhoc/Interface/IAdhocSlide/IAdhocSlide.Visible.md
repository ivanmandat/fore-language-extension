# IAdhocSlide.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет
 отображение слайда аналитической панели.


## Комментарии


Допустимые значения:


	- True. Слайд аналитической
	 панели отображается;


	- False. Слайд аналитической
	 панели скрыт.
