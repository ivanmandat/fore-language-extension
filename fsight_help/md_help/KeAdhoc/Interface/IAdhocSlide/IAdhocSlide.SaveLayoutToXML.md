# IAdhocSlide.SaveLayoutToXML


## Синтаксис


SaveLayoutToXML(Destination: Variant);


## Параметры


Destination. Путь и наименование
 XML-файла, в который будет сохранена разметка слайда аналитической панели.


## Описание


Метод SaveLayoutToXML сохраняет
 разметку слайда в XML-файл.


## Комментарии


Для загрузки разметки слайда из XML-файла используйте метод [IAdhocSlide.LoadLayoutFromXML](IAdhocSlide.LoadLayoutFromXML.htm).
