# IAdhocSlide


Сборка: Adhoc;


## Описание


Интерфейс IAdhocSlide предназначен
 для работы со слайдами аналитической панели.


## Иерархия наследования


           [INamedEntity](KeSom.chm::/Interface/INamedEntity/INamedEntity.htm)


           IAdhocSlide


## Комментарии


Содержит XML разметку текущей страницы конструктора аналитической панели.
