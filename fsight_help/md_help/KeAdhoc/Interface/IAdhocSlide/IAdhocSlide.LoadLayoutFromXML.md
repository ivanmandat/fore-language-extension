# IAdhocSlide.LoadLayoutFromXML


## Синтаксис


LoadLayoutFromXML(Src: Variant);


## Параметры


Src. Путь и наименование XML-файла,
 содержащего разметку слайда аналитической панели.


## Описание


Метод LoadLayoutFromXML загружает
 разметку слайда из XML-файла.


## Комментарии


Для сохранения разметки слайда в XML-файл используйте метод [IAdhocSlide.SaveLayoutToXML](IAdhocSlide.SaveLayoutToXML.htm).
