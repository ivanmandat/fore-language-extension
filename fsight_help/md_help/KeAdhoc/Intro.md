# Введение в сборку Adhoc


В данном разделе представлено описание сборки Adhoc,
 содержащей средства для работы с аналитическими панелями.


Аналитическая панель - это набор визуальных компонентов, используемых
 пользователем продукта «Форсайт. Аналитическая платформа»
 для отображения сводной аналитической информации из различных источников.


Основными интерфейсами сборки Adhoc
 являются:


	- [IAdhocReport](Interface/IAdhocReport/IAdhocReport.htm).
	 Предназначен для работы с объектом репозитория «Аналитическая
	 панель»;


	- [IAdhocDataSourceObject](Interface/IAdhocDataSourceObject/IAdhocDataSourceObject.htm).
	 Предназначен для работы с источником данных аналитической панели;


	- [IAdhocReportExporter](Interface/IAdhocReportExporter/IAdhocReportExporter.htm).
	 Предназначен для экспорта аналитической панели.
