# AdhocChatUserShareType


## Описание


Перечисление AdhocChatUserShareType
 предназначено для определения режима обмена сообщениями между пользователями
 аналитической панели при совместной работе.


Используется следующими свойствами и методами:


	- IAdhocChatUser.ShareType.


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 View. Режим просмотра.


		 1
		 Edit. Режим редактирования.
