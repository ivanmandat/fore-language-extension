# AdhocChatUserState


## Описание


Перечисление AdhocChatUserState
 предназначено для определения текущего состояния обмена сообщениями пользователей
 аналитической панели.


Используется следующими свойствами и методами:


	- IAdhocChatUser.State.


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Online. Обмен сообщений
		 между пользователями аналитической панели активен.


		 1
		 Inactive. Обмен сообщений
		 между пользователями аналитической панели не активен.
