# AdhocDocumentModeType


## Описание


Перечисление AdhocDocumentModeType
 используется для определения типа аналитической панели.


Используется свойством [IAdhocReport.DocumentMode](../Interface/IAdhocReport/IAdhocReport.DocumentMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Document. Аналитическая
		 панель без слайдов.


		 1
		 Slides. Аналитическая
		 панель со [слайдами](UiAdhoc.chm::/Presentation/Presentation.htm).
