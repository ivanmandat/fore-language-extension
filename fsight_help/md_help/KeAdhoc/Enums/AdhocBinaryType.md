# AdhocBinaryType


## Описание


Перечисление AdhocBinaryType
 предназначено для определения формата чтения и записи бинарных данных
 аналитической панели.


Используется свойством:


	- [IAdhocBinaryItem.Type](../Interface/IAdhocBinaryItem/IAdhocBinaryItem.Type.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Custom. Пользовательский
		 формат.


		 1
		 Jpeg. Формат *.jpeg.


		 2
		 Png. Формат *.png.


		 3
		 Bmp. Формат *.bmp.


		 4
		 Emf. Формат *.emf.


		 5
		 Wmf. Формат *.wmf.
