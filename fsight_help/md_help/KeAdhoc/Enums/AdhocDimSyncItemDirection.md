# AdhocDimSyncItemDirection


## Описание


Перечисление AdhocDimSyncItemDirection
 используется для определения типа связи синхронизируемого измерения.


Используется следующим свойством:


	- [IAdhocDimSyncItem.Direction](../Interface/IAdhocDimSyncItem/IAdhocDimSyncItem.Direction.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Двусторонняя. Связь
		 может как принимать, так и передавать отметку в управляемые блоки.


		 1
		 Односторонняя. Связь
		 может только принимать отметку от других блоков, с которыми настроена
		 синхронизация.
