# AdhocHorizontalAlignment


## Описание


Перечисление AdhocHorizontalAlignment
 используется для определения горизонтального выравнивания объекта.


Используется следующим свойством:


	- [IAdhocReportExportSourceTitle.HorizontalAlignment](../Interface/IAdhocReportExportSourceTitle/IAdhocReportExportSourceTitle.HorizontalAlignment.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Left. По левому краю.


		 1
		 Center. По центру.


		 2
		 Right. По правому краю.


		 3
		 Justify. По ширине.
