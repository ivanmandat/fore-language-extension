# AdhocChatHistoryActionType


## Описание


Перечисление AdhocChatHistoryActionType
 предназначено для определения типа действия над сообщением.


Используется свойством:


	- IAdhocChatHistoryAction.Type.


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Действие не производится.


		 1
		 AddMessage. Добавление
		 сообщения в историю.
