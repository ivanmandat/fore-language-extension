# AdhocDimensionType


## Описание


Перечисление AdhocDimensionType
 используется для определения типа синхронизированного измерения.


Используется следующим свойством:


	- [IAdhocDsoDimension.Type](../Interface/IAdhocDsoDimension/IAdhocDsoDimension.Type.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 DataSource. Отметка
		 измерения источника. Количество таких измерений соответствует
		 количеству измерений в кубе источника.


		 1
		 Additional. Копия
		 измерения источника.


		 2
		 SelectedArea.
		 Отметка, определяющая выбранную область визуализатора. Актуально,
		 если свойство [IAdhocDataSourceObject.TrackAreaSelection](../Interface/IAdhocDataSourceObject/IAdhocDataSourceObject.TrackAreaSelection.htm)
		 принимает значение True.
