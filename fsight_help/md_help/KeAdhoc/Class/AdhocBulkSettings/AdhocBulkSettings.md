# AdhocBulkSettings


Сборка: Adhoc;


## Описание


Класс AdhocBulkSettings предназначен
 для настроек пакетных данных аналитической панели.


## Свойства, унаследованные от [IAdhocBulkSettings](../../Interface/IAdhocBulkSettings/IAdhocBulkSettings.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [SelectionSet](../../Interface/IAdhocBulkSettings/IAdhocBulkSettings.SelectionSet.htm)


		 Свойство SelectionSet
		 определяет отметку фиксированных измерений, в соответствии с которой
		 будут созданы варианты аналитической панели для выполнения пакетной
		 операции.

		 [Enabled](ModDrawing.chm::/Interface/IGxBulkSettings/IGxBulkSettings.Enabled.htm)


		 Свойство Enabled определяет,
		 будет ли выполняться пакетная операция.
