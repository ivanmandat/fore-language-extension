# AdhocUserMasterForm


Сборка: Adhoc;


## Описание


Класс AdhocUserMasterForm предназначен
 для работы с формой-мастером для плагина.


## Комментарии


Форма-мастер содержит компоненты, образующие вкладки боковой панели
 для настройки плагина, и код, описывающий логику применения настроек к
 плагину.


Готовая форма-мастер должна наследоваться от класса AdhocUserMasterForm.


Пример создания плагина приведен в разделе «[Создание плагина](DataAnalysis.chm::/Plugins/Create_Plugin.htm)».
