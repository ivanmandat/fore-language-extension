# DatasetFailPolicy


## Описание


Перечисление DatasetFailPolicy содержит виды действия при возникновении ошибки.


Используется следующими свойствами и методами:


-
[IDatasetBatchUpdate.FailPolicy](../Interface/IDatasetBatchUpdate/IDatasetBatchUpdate.FailPolicy.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Rollback - откат изменений.


 1
 CommitByRow - сохранить изменения в текущей строке.
