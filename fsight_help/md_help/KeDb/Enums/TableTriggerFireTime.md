# TableTriggerFireTime


## Описание


Перечисление TableTriggerFireTime содержит периоды времени срабатывания триггера таблицы.


Используется следующими свойствами и методами:


-
[ITableTrigger.FireTime](../Interface/ITableTrigger/ITableTrigger.FireTime.htm)
