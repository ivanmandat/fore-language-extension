# DbAlterType


## Описание


Перечисление DbAlterType определяет способ, которым изменили объект.


Используется следующим свойством:


-
[ITableForeignKey.AlterType](../Interface/ITableForeignKey/ITableForeignKey.AlterType.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Нет.


 1
 Added. Добавили.


 2
 Dropped. Удалили.


 3
 Renamed. Сменили идентификатор.


 4
 Altered. Изменили.
