# DatabaseApplicationRoleMode


## Описание


Перечисление DatabaseApplicationRoleMode содержит варианты применения ролевого режима базы данных.


Используется следующими свойствами и методами:


-
[IDatabaseApplicationRole.Mode](../Interface/IDatabaseApplicationRole/IDatabaseApplicationRole.Mode.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Off - Роль приложения не используется.


 1
 On_ - Роль приложения используется.
