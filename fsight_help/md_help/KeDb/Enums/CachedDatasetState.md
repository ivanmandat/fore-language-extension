# CachedDatasetState


## Описание


Перечисление CachedDatasetState содержит варианты состояния кеша источника данных.


Используется следующими свойствами и методами:


-
[ICachedDataset.State](../Interface/ICachedDataset/ICachedDataset.State.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Browse - просмотр.


 1
 Edit - редактирование.


 2
 Insert - вставка записи.


 3
 Inactive - закрыт.
