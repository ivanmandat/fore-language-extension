# DatasetBatchCommand


## Описание


Перечисление DatasetBatchCommand содержит способы обновления записей в источнике данных.


Используется следующими свойствами и методами:


-
[IDatasetBatchUpdate.BatchCommand](../Interface/IDatasetBatchUpdate/IDatasetBatchUpdate.BatchCommand.htm)

 Значение
 Краткое описание


 0

 Update – обновление.


 2
 Insert – вставка.


 3
 Delete – удаление.
