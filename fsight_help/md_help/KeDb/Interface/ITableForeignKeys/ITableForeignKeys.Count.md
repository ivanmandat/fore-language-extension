# ITableForeignKeys.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество внешних ключей в коллекции.


## Комментарии


Для получения ключа из коллекции используйте свойство [ITableForeignKeys.Item](ITableForeignKeys.Item.htm).
