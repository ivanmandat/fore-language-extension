# ITableForeignKeys.Clear


## Синтаксис


Clear;


## Описание


Метод Clear очищает коллекцию внешних ключей.


## Комментарии


Для удаления внешнего ключа по его индексу используйте метод [ITableForeignKeys.Remove](ITableForeignKeys.Remove.htm), для удаления по его ключу - метод [ITableForeignKeys.RemoveByKey](ITableForeignKeys.RemoveByKey.htm).
