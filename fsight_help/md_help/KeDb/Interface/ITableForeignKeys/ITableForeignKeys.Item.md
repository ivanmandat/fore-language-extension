# ITableForeignKeys.Item


## Синтаксис


Item(Index: Integer): [ITableForeignKey](../ITableForeignKey/ITableForeignKey.htm);


## Параметры


Index. Индекс внешнего ключа в коллекции.


## Описание


Свойство Item возвращает внешний ключ из коллекции по его индексу.


## Комментарии


Индексация внешних ключей в коллекции сквозная и начинается с нуля. Количество внешних ключей в коллекции возвращает свойство [ITableForeignKeys.Count](ITableForeignKeys.Count.htm).
