# ITableForeignKeys


Сборка: Db;


## Описание


Интерфейс ITableForeignKeys
 используется для работы с коллекцией внешних ключей таблицы.


## Иерархия наследования


           ITableForeignKeys


## Комментарии


Коллекцию внешних ключей таблицы возвращает свойство [ITable.ForeignKeys](../ITable/ITable.ForeignKeys.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITableForeignKeys.Count.htm)
		 Свойство Count возвращает
		 количество внешних ключей в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ITableForeignKeys.Item.htm)
		 Свойство Item возвращает
		 внешний ключ из коллекции по его индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ITableForeignKeys.Add.htm)
		 Метод Add добавляет
		 внешний ключ.


		 ![](../../Sub_Image.gif)
		 [Clear](ITableForeignKeys.Clear.htm)
		 Метод Clear очищает
		 коллекцию внешних ключей.


		 ![](../../Sub_Image.gif)
		 [FindById](ITableForeignKeys.FindById.htm)
		 Метод FindById выполняет
		 поиск внешнего ключа в коллекции по его идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ITableForeignKeys.FindByKey.htm)
		 Метод FindByKey выполняет
		 поиск внешнего ключа в коллекции по его ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](ITableForeignKeys.Remove.htm)
		 Метод Remove удаляет
		 из коллекции внешний ключ с указанным индексом.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](ITableForeignKeys.RemoveByKey.htm)
		 Метод RemoveByKey удаляет
		 внешний ключ из коллекции по его ключу.
