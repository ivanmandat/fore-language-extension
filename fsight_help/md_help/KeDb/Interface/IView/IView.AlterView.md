# IView.AlterView


## Синтаксис


AlterView(OldView: [IView](IView.htm));


## Параметры


OldView - представление, структуру которого необходимо обновить в соответствии со структурой текущей процедуры.


## Описание


Метод AlterView осуществляет обновление структуры представления OldView на сервере БД в соответствии со структурой текущего представления.


## Пример


	Sub UserProc;

	Var

	    MB: IMetabase;

	    View, OldView: IView;

	Begin

	    Mb := MetaBaseClass.Active;

	    View := Mb.ItemById("View_1").Edit As IView; //представление, в соответствии со структурой которого, будем обновлять

	    OldView := Mb.ItemById("View_2").Bind As IView; //представление, структуру на сервере БД которого, будем обновлять

	    View.NativeName := OldView.NativeName; //временно устанавливаем одинаковые физические имена.

	    //Необходимо для сохранения уникальности физ. имени представления OldView

	    View.AlterView(OldView);

	End Sub UserProc;


После выполнения примера будет обновлена структура представления на сервере БД, на которое ссылается представление репозитория "View_2". Структура будет установлена в соответствии со структурой представления, на которое ссылается журнал репозитория "View_1".
