# ISQLComponents.SubQuery


## Синтаксис


SubQuery: Boolean;


## Описание


Свойство SubQuery возвращает
 признак использования подзапросов в запросе.


## Комментарии


Признак использования подзапросов в запросе задаётся для источника данных
 куба в свойстве [IStandardCubeDataset.UseSubqueryOptimization](KeCubes.chm::/Interface/IStandardCubeDataset/IStandardCubeDataset.UseSubqueryOptimization.htm),
 а свойство SubQuery вернёт установленное
 там значение.


## Пример


Пример использования приведён в описании метода [ISQLCallback.BeforeConstruction](../ISQLCallback/ISQLCallback.BeforeConstruction.htm).
