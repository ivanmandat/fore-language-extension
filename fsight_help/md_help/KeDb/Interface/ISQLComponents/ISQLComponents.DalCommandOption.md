# ISQLComponents.DalCommandOption


## Синтаксис


DalCommandOption: Integer;


## Описание


Свойство DalCommandOption определяет
 режим выполнения SQL-запроса.


## Комментарии


В текущей реализации в качестве значения свойства может быть установлено
 значение перечисления [DalCommandOption.NoCursorInQuery](dal.chm::/Enums/DalCommandOption.htm).
