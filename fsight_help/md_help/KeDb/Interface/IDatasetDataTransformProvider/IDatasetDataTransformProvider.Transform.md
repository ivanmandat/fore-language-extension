# IDatasetDataTransformProvider.Transform


## Синтаксис


Transform: [IDatasetDataTransform](../IDatasetDataTransform/IDatasetDataTransform.htm);


## Описание


Свойство Transform определяет объект, осуществляющий преобразование данных перед загрузкой.


## Комментарии


В качестве значения данного свойства должен быть указан объект пользовательского класса, реализующий интерфейс [IDatasetDataTransform](../IDatasetDataTransform/IDatasetDataTransform.htm).


## Пример


Пример использования свойства приведен в описании метода [IDatasetDataTransform.OnFetch](../IDatasetDataTransform/IDatasetDataTransform.OnFetch.htm).
