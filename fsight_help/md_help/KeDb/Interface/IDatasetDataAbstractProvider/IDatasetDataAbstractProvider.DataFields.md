# IDatasetDataAbstractProvider.DataFields


## Синтаксис


DataFields: [ITableFields](../ITableFields/ITableFields.htm);


## Описание


Свойство DataFields возвращает коллекцию полей для дополнительной настройки.


## Комментарии


Данное свойство возвращает коллекцию полей, аналогичную коллекции [FieldDefs](../IDatasetDataProvider/IDatasetDataProvider.FieldDefs.htm), но поля в данной коллекции будут доступны для редактирования.
