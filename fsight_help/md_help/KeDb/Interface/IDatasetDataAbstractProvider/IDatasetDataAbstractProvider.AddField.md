# IDatasetDataAbstractProvider.AddField


## Синтаксис


AddField(Name: String; DataType: [DbDataType](Dal.chm::/Enums/DbDataType.htm)): [IDatasetModelField](../IDatasetModelField/IDatasetModelField.htm);


## Параметры


Name. Наименование создаваемого поля.


DataType. Тип данных поля.


## Описание


Метод AddField создает поле в пользовательском источнике данных.
