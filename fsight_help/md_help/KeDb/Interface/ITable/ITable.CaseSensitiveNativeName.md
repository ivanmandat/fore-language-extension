# ITable.CaseSensitiveNativeName


## Синтаксис


CaseSensitiveNativeName: String;


## Описание


Свойство CaseSensitiveNativeName
 определяет физическое имя таблицы в базе данных с учётом регистра символов.


## Комментарии


Используется при работе с присоединёнными таблицами, процедурами, запросами,
 если они основаны на базе данных, учитывающей регистр символов в наименованиях
 таблиц.
