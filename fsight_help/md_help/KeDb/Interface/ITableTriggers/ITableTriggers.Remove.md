# ITableTriggers.Remove


## Синтаксис


Remove(Trigger: Integer): Boolean;


## Параметры


Trigger — индекс триггера.


## Описание


Метод Remove осуществляет удаление триггера таблицы. Индекс триггера передается посредством параметра Trigger.
