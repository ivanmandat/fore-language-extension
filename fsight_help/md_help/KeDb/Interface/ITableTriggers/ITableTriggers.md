# ITableTriggers


## Описание


Интерфейс ITableTriggers содержит
 свойства и методы коллекции триггеров таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITableTriggers.Count.htm)
		 Свойство Count возвращает
		 количество триггеров таблицы.


		 ![](../../Property_Image.gif)
		 [Item](ITableTriggers.Item.htm)
		 Свойство Item возвращает
		 параметры триггера таблицы.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ITableTriggers.Add.htm)
		 Метод Add осуществляет
		 добавление триггера в таблицу.


		 ![](../../Sub_Image.gif)
		 [Clear](ITableTriggers.Clear.htm)
		 Метод Clear осуществляет
		 очистку списка триггеров таблицы.


		 ![](../../Sub_Image.gif)
		 [FindById](ITableTriggers.FindById.htm)
		 Метод FindById осуществляет
		 поиск триггера по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ITableTriggers.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск триггера по ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](ITableTriggers.Remove.htm)
		 Метод Remove осуществляет
		 удаление триггера таблицы.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](ITableTriggers.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление триггера по ключу.
