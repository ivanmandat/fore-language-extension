# ITableTriggers.RemoveByKey


## Синтаксис


RemoveByKey(TriggerKey: Integer): Boolean;


## Параметры


TriggerKey — ключ триггера.


## Описание


Метод RemoveByKey осуществляет удаление триггера таблицы. Ключ триггера передается посредством параметра TriggerKey.
