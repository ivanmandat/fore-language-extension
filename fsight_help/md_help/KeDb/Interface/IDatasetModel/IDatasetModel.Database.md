# IDatasetModel.Database


## Синтаксис


Database: [IDatabase](../IDatabase/IDatabase.htm);


## Описание


Свойство Database возвращает параметры базы данных репозитория, в которой содержится данный источник данных.


## Комментарии


Данное свойство вернет ту базу данных, которая была указана при создании объекта. Например, для таблицы свойство вернет базу данных, которая была задана в свойстве [ITable.Database](../ITable/ITable.Database.htm).
