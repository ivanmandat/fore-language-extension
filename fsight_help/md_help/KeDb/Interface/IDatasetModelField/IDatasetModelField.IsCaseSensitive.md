# IDatasetModelField.IsCaseSensitive


## Синтаксис


IsCaseSensitive: Boolean;


## Описание


Свойство IsCaseSensitive возвращает
 признак регистрозависимости содержимого поля.


## Комментарии


Свойство возвращает значение True,
 если содержимое поля является регистрозависимым. Зависимость от регистра
 определяется типом и настройками СУБД.
