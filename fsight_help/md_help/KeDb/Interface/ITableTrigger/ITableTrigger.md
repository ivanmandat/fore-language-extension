# ITableTrigger


## Описание


Интерфейс ITableTrigger содержит
 свойства и методы триггера таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Body](ITableTrigger.Body.htm)
		 Свойство Body определяет
		 текст триггера для выбранного драйвера базы данных.


		 ![](../../Property_Image.gif)
		 [FireEvent](ITableTrigger.FireEvent.htm)
		 Свойство FireEvent
		 определяет события, при которых будет срабатывать триггер.


		 ![](../../Property_Image.gif)
		 [FireTime](ITableTrigger.FireTime.htm)
		 Свойство FireTime определяет
		 время выполнения триггера.


		 ![](../../Property_Image.gif)
		 [ForEachRow](ITableTrigger.ForEachRow.htm)
		 Свойство ForEachRow
		 определяет признак выполнения триггера для каждой строки таблицы.


		 ![](../../Property_Image.gif)
		 [Text](ITableTrigger.Text.htm)
		 Свойство Text определяет
		 текст триггера.
