# ITableIndexes


## Описание


Интерфейс ITableIndexes содержит
 свойства и методы коллекции индексов таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITableIndexes.Count.htm)
		 Свойство Count возвращает
		 количество индексов таблицы.


		 ![](../../Property_Image.gif)
		 [Item](ITableIndexes.Item.htm)
		 Свойство Item возвращает
		 параметры индекса таблицы.


		 ![](../../Property_Image.gif)
		 [Primary](ITableIndexes.Primary.htm)
		 Свойство Primary возвращает
		 параметры первичного индекса таблицы.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ITableIndexes.Add.htm)
		 Метод Add осуществляет
		 добавление индекса таблицы.


		 ![](../../Sub_Image.gif)
		 [Clear](ITableIndexes.Clear.htm)
		 Метод Clear осуществляет
		 очистку списка индексов таблицы.


		 ![](../../Sub_Image.gif)
		 [FindById](ITableIndexes.FindById.htm)
		 Метод FindById осуществляет
		 поиск индекса по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ITableIndexes.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск индекса по ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](ITableIndexes.Remove.htm)
		 Метод Remove осуществляет
		 удаление индекса по номеру.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](ITableIndexes.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление индекса по ключу.
