# IDatabaseApplicationRole


## Описание


Интерфейс IDatabaseApplicationRole определяет
 настройки использования роли приложения.


## Комментарии


Использование [роли
 приложения](Admin.chm::/03_Admin/Access_control_settings.htm#role) доступно, если репозиторий создан на базе
 СУБД [Мicrosoft
 SQL Server](Setup.chm::/01_SysReq/database_Support.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DerivedPassword](IDatabaseApplicationRole.DerivedPassword.htm)
		 Свойство DerivedPassword
		 определяет использование производного пароля при создании подключения
		 к базе данных.


		 ![](../../Property_Image.gif)
		 [Mode](IDatabaseApplicationRole.Mode.htm)
		 Свойство Mode определяет
		 вариант применения роли приложения.


		 ![](../../Property_Image.gif)
		 [Name](IDatabaseApplicationRole.Name.htm)
		 Свойство Name определяет
		 наименование для роли приложения.


		 ![](../../Property_Image.gif)
		 [Password](IDatabaseApplicationRole.Password.htm)
		 Свойство Password задает
		 пароль для роли приложения.
