# IDatabaseApplicationRole.Name


## Синтаксис


Name: String;


## Описание


Свойство Name определяет наименование для роли приложения.


## Комментарии


По умолчанию для роли приложения используется имя P5DBAPPROLE. Использование роли доступно только для СУБД типа Microsoft SQL Server.
