# ITableIndexFields.RemoveByKey


## Синтаксис


RemoveByKey(Value: Integer): Boolean;


## Параметры


Value — ключ поля.


## Описание


Метод RemoveByKey осуществляет удаление поля из индекса таблицы. Ключ поля передается посредством параметра Value.
