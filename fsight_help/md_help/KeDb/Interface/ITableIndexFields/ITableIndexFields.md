# ITableIndexFields


## Описание


Интерфейс ITableIndexFields
 содержит свойства и методы коллекции полей индекса таблицы репозитория.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITableIndexFields.Count.htm)
		 Свойство Count возвращает
		 количество полей индекса таблицы.


		 ![](../../Property_Image.gif)
		 [Item](ITableIndexFields.Item.htm)
		 Свойство Item возвращает
		 поле таблицы, по которому создан индекс.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ITableIndexFields.Add.htm)
		 Метод Add осуществляет
		 добавление поля таблицы в индекс.


		 ![](../../Sub_Image.gif)
		 [Move](ITableIndexFields.Move.htm)
		 Метод Move осуществляет
		 перемещение поля таблицы в индексе.


		 ![](../../Sub_Image.gif)
		 [Remove](ITableIndexFields.Remove.htm)
		 Метод Remove осуществляет
		 удаление поля таблицы из индекса.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](ITableIndexFields.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление поля таблицы из индекса по ключу.
