# IDatabase.AutoAdjustRights


## Синтаксис


AutoAdjustRights: Boolean;


## Описание


Свойство AutoAdjustRights определяет, будут ли раздаваться права на все доступные объекты при подключении к базе данных. Если свойство установлено в True - права будут раздаваться.
