# IDatabase.LoginPrompt


## Синтаксис


LoginPrompt: Boolean;


## Описание


Свойство LoginPrompt определяет,
 будет ли производиться автоматическое подключение к базе данных при входе
 в репозиторий.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Не производить автоматическое подключение, запрашивать учётные данные
	 при первом использовании базы данных;


	- False. Использовать
	 автоматическое подключение. Учётные данные для подключения указываются
	 в свойстве [IDatabase.Credentials](IDatabase.Credentials.htm).
