# IProcedure.ForceCreate


## Синтаксис


ForceCreate: Boolean;


## Описание


Свойство ForceCreate определяет возможность создания некорректной процедуры.


## Комментарии


Процедура считается некорректной, если во время компилирования SQL-запроса возникают ошибки. Если свойству установлено значение True, то будет возможность сохранения процедуры, содержащей ошибки в SQL-запросе. По умолчанию значение данного свойства False.
