# IProcedure.Refresh


## Синтаксис


Refresh([Connection: [ISecurityConnection](KeSom.chm::/Interface/ISecurityConnection/ISecurityConnection.htm) = Null]);


## Параметры


Connection. Необязательный параметр, позволяющий напрямую указать соединение с базой данных.


## Описание


Метод Refresh заканчивает обновление, сохраняя все изменения.


## Комментарии


Используется вместе с методами [AttachProcedure](IProcedure.AttachProcedure.htm) и [UpdateProcedure](IProcedure.UpdateProcedure.htm)
