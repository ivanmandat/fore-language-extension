# IDatasetInstanceFields


## Описание


Интерфейс IDatasetInstanceFields
 содержит свойства и методы коллекции значений полей источника данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDatasetInstanceFields.Count.htm)
		 Свойство Count возвращает
		 количество полей источника данных.


		 ![](../../Property_Image.gif)
		 [Item](IDatasetInstanceFields.Item.htm)
		 Свойство Item возвращает
		 параметры поля источника данных.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](IDatasetInstanceFields.FindById.htm)
		 Метод FindById осуществляет
		 поиск поля источника данных по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IDatasetInstanceFields.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск поля источника данных по ключу.


		 ![](../../Sub_Image.gif)
		 [IndexOf](IDatasetInstanceFields.IndexOf.htm)
		 Метод IndexOf возвращает
		 индекс поля источника данных.
