# IDatasetInstanceFields.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IDatasetInstanceField](../IDatasetInstanceField/IDatasetInstanceField.htm);


## Параметры


Key — ключ поля.


## Описание


Метод FindByKey осуществляет поиск поля источника данных по ключу. Ключ, по которому осуществляется поиск, передается посредством параметра Key.
