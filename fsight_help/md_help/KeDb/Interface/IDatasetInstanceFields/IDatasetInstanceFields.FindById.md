# IDatasetInstanceFields.FindById


## Синтаксис


FindById(Id: String): [IDatasetInstanceField](../IDatasetInstanceField/IDatasetInstanceField.htm);


## Параметры


Id — идентификатор поля.


## Описание


Метод FindById осуществляет поиск поля источника данных по идентификатору. Идентификатор, по которому осуществляется поиск, передается посредством параметра Id.
