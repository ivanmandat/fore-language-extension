# IDatabaseInstance


## Описание


Интерфейс IDatabaseInstance
 содержит свойства для получения экземпляров объектов, содержащихся на
 сервере БД.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Connection](IDatabaseInstance.Connection.htm)
		 Свойство Connection
		 возвращает параметры соединения с сервером БД.


		 ![](../../Property_Image.gif)
		 [Database](IDatabaseInstance.Database.htm)
		 Свойство Database возвращает
		 свойства базы данных.


		 ![](../../Property_Image.gif)
		 [Tables](IDatabaseInstance.Tables.htm)
		 Свойство Tables возвращает
		 объект, содержащий коллекцию всех таблиц, хранящихся на сервере
		 БД.


		 ![](../../Property_Image.gif)
		 [Views](IDatabaseInstance.Views.htm)
		 Свойство Views возвращает
		 объект, содержащий коллекцию всех представлений, хранящихся на
		 сервере БД.
