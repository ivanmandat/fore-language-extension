# IQueryStorage


## Описание


Интерфейс IQueryStorage содержит свойства и методы, используемые для настройки параметров сохранения данных.


## Свойства


  Имя свойства
 Краткое описание


 ![](../../Property_Image.gif)
 [Enabled](IQueryStorage.Enabled.htm)
 Свойство Enabled определяет признак наличия возможности сохранения данных.


 ![](../../Property_Image.gif)
 [Table](IQueryStorage.Table.htm)
 Свойство Table определяет таблицу репозитория, в которую будет осуществляться сохранение данных.


 ![](../../Property_Image.gif)
 [TableName](IQueryStorage.TableName.htm)
 Свойство TableName определяет физическое имя таблицы, в которую будет осуществляться сохранение данных.


## Методы


  Имя метода
 Краткое описание


 ![](../../Sub_Image.gif)
 [DefineBySql](IQueryStorage.DefineBySql.htm)
 Метод DefineBySql определяет настройки для сохранения данных по SQL-запросу.
