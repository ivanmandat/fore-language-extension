# IQueryStorage.Table


## Синтаксис


Table: [ITable](../ITable/ITable.htm);


## Описание


Свойство Table определяет таблицу репозитория, в которую будет осуществляться сохранение данных.


## Комментарии


Данное свойство актуально, если свойству [Enabled](IQueryStorage.Enabled.htm) установлено значение True.


## Пример


Пример использования данного свойства приведен в примере для свойства [Enabled](IQueryStorage.Enabled.htm).
