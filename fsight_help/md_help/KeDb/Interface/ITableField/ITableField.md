# ITableField


Сборка: Db;


## Описание


Интерфейс ITableField содержит
 свойства поля таблицы.


## Иерархия наследования


           [INamedEntityModel](ForeSys.chm::/Interface/INamedEntityModel/INamedEntityModel.htm)


           ITableFields


## Комментарии


Для работы со всеми полями таблицы интерфейс [ITableFields](../ITableFields/ITableFields.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CaseSensitiveId](ITableField.CaseSensitiveId.htm)
		 Свойство CaseSensitiveId
		 определяет идентификатор поля таблицы.


		 ![](../../Property_Image.gif)
		 [DataDomain](ITableField.DataDomain.htm)
		 Свойство DataDomain
		 определяет тип данных поля таблицы, если оно предназначено для
		 хранения многобайтовой информации.


		 ![](../../Property_Image.gif)
		 [DataType](ITableField.DataType.htm)
		 Свойство DataType определяет
		 тип поля.


		 ![](../../Property_Image.gif)
		 [DefaultValue](ITableField.DefaultValue.htm)
		 Свойство DefaultValue
		 определяет значение по умолчанию для поля.


		 ![](../../Property_Image.gif)
		 [Expression](ITableField.Expression.htm)
		 Свойство Expression
		 возвращает выражение вычисляемого поля.


		 ![](../../Property_Image.gif)
		 [IsCalculated](ITableField.IsCalculated.htm)
		 Свойство IsCalculated
		 определяет, будет ли поле вычисляемым.


		 ![](../../Property_Image.gif)
		 [IsIdentity](ITableField.IsIdentity.htm)
		 Свойство IsIdentity
		 указывает, что значения данного поля уникальным образом идентифицируют
		 каждую строку данных.


		 ![](../../Property_Image.gif)
		 [IsSparse](ITableField.IsSparse.htm)
		 Свойство IsSparse
		 возвращает признак того, используются ли хранения данных поля
		 разреженные столбцы.


		 ![](../../Property_Image.gif)
		 [Mandatory](ITableField.Mandatory.htm)
		 Свойство Mandatory
		 определяет, допускаются ли пустые значения в поле.


		 ![](../../Property_Image.gif)
		 [Precision](ITableField.Precision.htm)
		 Свойство Precision
		 определяет количество десятичных знаков поля.


		 ![](../../Property_Image.gif)
		 [Size](ITableField.Size.htm)
		 Свойство Size определяет
		 общую длину поля.
