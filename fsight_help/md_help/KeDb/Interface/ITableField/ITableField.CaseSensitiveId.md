# ITableField.CaseSensitiveId


## Синтаксис


CaseSensitiveId: String;


## Описание


Свойство CaseSensitiveId определяет идентификатор поля таблицы. Данное свойство используется при работе с базами данных, учитывающих регистр символов в идентификаторах полей таблицы.
