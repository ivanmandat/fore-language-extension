# ITableForeignKey.RTable


## Синтаксис


RTable: String;


## Описание


Свойство RTable определяет
 идентификатор таблицы, на которую ссылается внешний ключ.


## Комментарии


Поле в данной таблице, на которое ссылается внешний ключ, определяет
 свойство [ITableForeignKey.RColumns](ITableForeignKey.RColumns.htm).
