# ITableForeignKey.ComparedWith


## Синтаксис


ComparedWith: Integer;


## Описание


Свойство ComparedWith определяет индекс элемента, с которым сравнивается внешний ключ.


## Комментарии


По умолчанию сравнение не выполняется.
