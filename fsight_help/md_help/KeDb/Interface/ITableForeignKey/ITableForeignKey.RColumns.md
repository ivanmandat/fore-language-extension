# ITableForeignKey.RColumns


## Синтаксис


RColumns: String;


## Описание


Свойство RColumns определяет поле, на которое ссылается внешний ключ.


## Комментарии


Таблицу, в которой расположено данное поле, определяет свойство [ITableForeignKey.RTable](ITableForeignKey.RTable.htm).
