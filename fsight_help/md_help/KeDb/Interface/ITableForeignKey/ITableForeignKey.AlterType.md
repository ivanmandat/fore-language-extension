# ITableForeignKey.AlterType


## Синтаксис


AlterType: [DbAlterType](../../Enums/DbAlterType.htm);


## Описание


Свойство AlterType определяет способ обновления ключа.


## Комментарии


По умолчанию свойство имеет значение DbAlterType.None.
