# ITableForeignKey.Columns


## Синтаксис


Columns: String;


## Описание


Свойство Columns определяет поле таблицы, которое является внешним ключом.


## Комментарии


Поле, на которое ссылается внешний ключ, определяет свойство [ITableForeignKey.RColumns](ITableForeignKey.RColumns.htm).
