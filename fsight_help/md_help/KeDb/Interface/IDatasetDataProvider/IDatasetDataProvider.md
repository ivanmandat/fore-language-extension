# IDatasetDataProvider


Сборка: Db;


## Описание


Интерфейс IDatasetDataProvider содержит базовые свойства источника данных, из которого осуществляется загрузка показателей в базу данных временных рядов.


## Иерархия наследования


IDatasetDataProvider


## Комментарии


Данный интерфейс поддерживают различные источники данных, реализованные в [сборке Dt](KeDt.chm::/KeDt_Introduction.htm), а также пользовательские источники данных, описываемые интерфейсами [IDatasetDataArrayProvider](../IDatasetDataArrayProvider/IDatasetDataArrayProvider.htm) и [IDatasetDataTransformProvider](../IDatasetDataTransformProvider/IDatasetDataTransformProvider.htm).
