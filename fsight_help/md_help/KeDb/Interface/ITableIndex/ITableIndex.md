# ITableIndex


## Описание


Интерфейс ITableIndex содержит
 свойства индекса таблицы репозитория.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AllowNulls](ITableIndex.AllowNulls.htm)
		 Свойство AllowNulls
		 определяет, допускает ли индекс пустые значения.


		 ![](../../Property_Image.gif)
		 [FieldNames](ITableIndex.FieldNames.htm)
		 Свойство FieldNames
		 возвращает символьную строку, содержащую список полей индекса.


		 ![](../../Property_Image.gif)
		 [FieldNamesWithSeparator](ITableIndex.FieldNamesWithSeparator.htm)
		 Свойство FieldNamesWithSeparator
		 возвращает символьную строку, содержащую список полей, отделенных
		 указанным разделителем.


		 ![](../../Property_Image.gif)
		 [Fields](ITableIndex.Fields.htm)
		 Свойство Fields возвращает
		 коллекцию полей таблицы, по которым создан индекс.


		 ![](../../Property_Image.gif)
		 [Primary](ITableIndex.Primary.htm)
		 Свойство Primary определяет
		 признак первичного индекса.


		 ![](../../Property_Image.gif)
		 [Unique](ITableIndex.Unique.htm)
		 Свойство Unique определяет,
		 будет ли индекс уникальным.
