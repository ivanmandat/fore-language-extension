# ITableChecks


## Описание


Интерфейс ITableChecks содержит
 свойства и методы коллекции ограничений таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITableChecks.Count.htm)
		 Свойство Count возвращает
		 количество ограничений таблицы.


		 ![](../../Property_Image.gif)
		 [Item](ITableChecks.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий ограничение таблицы.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ITableChecks.Add.htm)
		 Метод Add осуществляет
		 добавление ограничения таблицы.


		 ![](../../Sub_Image.gif)
		 [Clear](ITableChecks.Clear.htm)
		 Метод Clear осуществляет
		 очистку списка ограничений таблицы.


		 ![](../../Sub_Image.gif)
		 [FindById](ITableChecks.FindById.htm)
		 Метод FindById осуществляет
		 поиск ограничения по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ITableChecks.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск ограничения по ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](ITableChecks.Remove.htm)
		 Метод Remove осуществляет
		 удаление ограничения таблицы по номеру.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](ITableChecks.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление ограничения таблицы по ключу.
