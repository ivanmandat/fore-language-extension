# ISequence.StartValue


## Синтаксис


StartValue: Integer;


## Описание


Свойство StartValue определяет начальное значение, используемое для генерации ключей.


## Комментарии


Шаг, с которым генерируются значения, определяется свойством [ISequence.IncrementStep](ISequence.IncrementStep.htm).
