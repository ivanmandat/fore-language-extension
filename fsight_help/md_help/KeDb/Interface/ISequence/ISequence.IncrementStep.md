# ISequence.IncrementStep


## Синтаксис


IncrementStep: Integer;


## Описание


Свойство IncrementStep определяет
 шаг, используемый для генерации ключей.


## Комментарии


Начальное значение определяет свойство [ISequence.StartValue](ISequence.StartValue.htm).
