# ISequence.NativeName


## Синтаксис


NativeName: String;


## Описание


Свойство NativeName определяет физическое имя таблицы, используемой для генерации ключей.


## Комментарии


Базу данных, используемую таблицей, определяет свойство [ISequence.Database](ISequence.Database.htm).
