# ISequence


Сборка: Db;


## Описание


Интерфейс ISequence используется
 для генерации ключей объектов.


## Иерархия наследования


           ISequence


## Комментарии


Возможность генерировать ключи показателей доступна для баз данных временных
 рядов.
