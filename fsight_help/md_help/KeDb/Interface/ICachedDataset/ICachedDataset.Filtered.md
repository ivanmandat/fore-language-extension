# ICachedDataset.Filtered


## Синтаксис


Filtered: Boolean;


## Описание


Свойство Filtered определяет
 текущее состояние фильтрации данных в кеше.


## Комментарии


Если свойству установлено значение True
 - данные фильтруются по установленному в свойстве [Filter](ICachedDataset.Filter.htm)
 условию, если False - фильтрация
 не производится.
