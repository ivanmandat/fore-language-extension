# ICachedDataset.AddCalculatedField


## Синтаксис


AddCalculatedField(Id: String): [IDatasetInstanceField](../IDatasetInstanceField/IDatasetInstanceField.htm);


## Параметры


Id — идентификатор поля.


## Описание


Метод AddCalculatedField создает вычисляемое поле в кеше. Идентификатор поля передается посредством параметра Id.
