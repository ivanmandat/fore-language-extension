# ICachedDataset.FetchToRecord


## Синтаксис


FetchToRecord(Value: Integer): Integer;


## Параметры


Value — количество записей, которое необходимо загрузить в кеш.


## Описание


Метод FetchToRecord осуществляет загрузку записей из источника данных в кеш. Количество записей передается посредством параметра Value. После выполнения возвращает количество записей, находящихся в кеше.
