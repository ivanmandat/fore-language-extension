# IDatasetModelFields.FindByKey


## Синтаксис


FindByKey(FieldKey: Integer): [IDatasetModelField](../IDatasetModelField/IDatasetModelField.htm);


## Параметры


FieldKey — ключ поля.


## Описание


Метод FindByKey осуществляет поиск поля и возвращает объект, содержащий структуру поля источника данных. Ключ поля, по которому осуществляется поиск, передается посредством параметра Key.
