# IDatasetModelFields.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 полей источника данных.


## Комментарии


Для получения определённого поля источника данных используйте свойство
 [IDatasetModelFields.Item](IDatasetModelFields.Item.htm).
