# IDatasetModelFields


## Описание


Интерфейс IDatasetModelFields
 содержит свойства и методы структуры полей источника данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDatasetModelFields.Count.htm)
		 Свойство Count возвращает
		 количество полей источника данных.


		 ![](../../Property_Image.gif)
		 [Item](IDatasetModelFields.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий поле источника данных.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](IDatasetModelFields.FindById.htm)
		 Метод FindById осуществляет
		 поиск поля источника данных по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IDatasetModelFields.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск поля источника данных по ключу.
