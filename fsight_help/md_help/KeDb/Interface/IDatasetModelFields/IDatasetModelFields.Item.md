# IDatasetModelFields.Item


## Синтаксис


Item(Index: Integer): [IDatasetModelField](../IDatasetModelField/IDatasetModelField.htm);


## Параметры


Index. Индекс
 поля.


## Описание


Свойство Item возвращает структуру
 поля источника данных.


## Комментарии


Индексация полей сквозная и начинается с нуля. Для получения количества
 полей используйте свойство [IDatasetModelFields.Count](IDatasetModelFields.Count.htm).
