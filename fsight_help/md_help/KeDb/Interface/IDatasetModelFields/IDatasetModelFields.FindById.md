# IDatasetModelFields.FindById


## Синтаксис


FindById(Id: String): [IDatasetModelField](../IDatasetModelField/IDatasetModelField.htm);


## Параметры


Id — идентификатор поля.


## Описание


Метод FindById осуществляет поиск поля и возвращает объект, содержащий структуру поля источника данных. Идентификатор поля, по которому осуществляется поиск, передается посредством параметра Id.
