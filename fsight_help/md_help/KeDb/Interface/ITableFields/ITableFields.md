# ITableFields


## Описание


Интерфейс ITableFields содержит
 свойства и методы коллекции полей таблицы репозитория.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITableFields.Count.htm)
		 Свойство Count возвращает
		 количество полей таблицы.


		 ![](../../Property_Image.gif)
		 [Item](ITableFields.Item.htm)
		 Свойство Item возвращает
		 параметры поля таблицы.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ITableFields.Add.htm)
		 Метод Add осуществляет
		 создание нового поля таблицы.


		 ![](../../Sub_Image.gif)
		 [Clear](ITableFields.Clear.htm)
		 Метод Clear осуществляет
		 очистку списка полей таблицы.


		 ![](../../Sub_Image.gif)
		 [FieldByKey](ITableFields.FieldByKey.htm)
		 Метод FieldByKey осуществляет
		 переход к полю таблицы.


		 ![](../../Sub_Image.gif)
		 [FindById](ITableFields.FindById.htm)
		 Метод FindById осуществляет
		 поиск и возвращает объект, содержащий поле таблицы.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ITableFields.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск и возвращает объект, содержащий поле таблицы.


		 ![](../../Sub_Image.gif)
		 [Remove](ITableFields.Remove.htm)
		 Метод Remove осуществляет
		 удаление поля таблицы по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](ITableFields.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление поля таблицы по ключу.
