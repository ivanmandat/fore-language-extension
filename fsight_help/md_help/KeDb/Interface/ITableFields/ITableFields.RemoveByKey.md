# ITableFields.RemoveByKey


## Синтаксис


RemoveByKey(FieldKey: Integer): Boolean;


## Параметры


FieldKey - ключ поля.


## Описание


Метод RemoveByKey осуществляет удаление поля таблицы, ключ которого передается посредством параметра Fieldkey.
