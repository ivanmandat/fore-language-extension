# ICachedDatasetLookup.Lookup


## Синтаксис


Lookup(ResultFields: String): [IDatasetInstanceFields](../IDatasetModelFields/IDatasetModelFields.htm);


## Параметры


ResultFields.
 Список результирующих полей, значения которых необходимо получить. Поля
 в списке разделяются символом «;».


## Описание


Метод Lookup возвращает коллекцию
 значений полей найденной записи.


## Комментарии


Метод используется, если метод [Locate](ICachedDatasetLookup.Locate.htm)
 вернул значение True.

Пример использования приведен в описании метода [ICachedDatasetLookup.Locate](ICachedDatasetLookup.Locate.htm).
