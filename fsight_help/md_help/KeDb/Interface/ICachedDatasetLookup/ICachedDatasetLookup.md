# ICachedDatasetLookup


## Описание


Интерфейс ICachedDatasetLookup
 содержит свойства и методы для поиска данных в кеше источника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dataset](ICachedDatasetLookup.Dataset.htm)
		 Свойство Dataset возвращает
		 кеш источника данных.


		 ![](../../Property_Image.gif)
		 [Fields](ICachedDatasetLookup.Fields.htm)
		 Свойство Fields возвращает
		 коллекцию полей, по которым будет осуществляться поиск.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Locate](ICachedDatasetLookup.Locate.htm)
		 Метод Locate осуществляет
		 поиск и возвращает признак удачного поиска в кеше источника данных.


		 ![](../../Sub_Image.gif)
		 [Lookup](ICachedDatasetLookup.Lookup.htm)
		 Метод Lookup возвращает
		 коллекцию значений полей найденной записи.
