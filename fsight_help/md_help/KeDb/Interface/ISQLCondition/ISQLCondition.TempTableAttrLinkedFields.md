# ISQLCondition.TempTableAttrLinkedFields


## Синтаксис


TempTableAttrLinkedFields: String;


## Описание


Свойство TempTableAttrLinkedFields
 возвращает список полей временной таблицы, которые связываются с атрибутами
 измерения.


## Комментарии


Список полей возвращается в том же порядке, в котором осуществляется
 связь с атрибутами. Поля в списке разделяются запятой.
