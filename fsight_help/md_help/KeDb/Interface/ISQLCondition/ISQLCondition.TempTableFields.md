# ISQLCondition.TempTableFields


## Синтаксис


TempTableFields: String;


## Описание


Свойство TempTableFields возвращает
 список полей временной таблицы.


## Комментарии


Данный список может отличаться от списка, доступного в [TempTableAttrLinkedFields](ISQLCondition.TempTableAttrLinkedFields.htm),
 наличием служебных полей. Поля в списке разделяются запятой.
