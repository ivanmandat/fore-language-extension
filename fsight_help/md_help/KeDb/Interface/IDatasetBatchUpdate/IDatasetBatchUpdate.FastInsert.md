# IDatasetBatchUpdate.FastInsert


## Синтаксис


FastInsert: Boolean;


## Описание


Свойство FastInsert
 определяет, используется ли режим быстрой вставки для обновления записей
 источника данных.


## Комментарии


Допустимые значения:


	- True. Режим быстрой
	 вставки используется;


	- False. Режим быстрой
	 вставки не используется.
