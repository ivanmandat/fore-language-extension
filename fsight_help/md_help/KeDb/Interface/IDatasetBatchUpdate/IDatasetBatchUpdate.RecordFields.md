# IDatasetBatchUpdate.RecordFields


## Синтаксис


RecordFields(RecNo: Integer): Array;


## Параметры


RecNo.
 Индекс записи в буфере.


## Описание


Свойство RecordFields возвращает
 массив значений записи, находящейся в буфере.


## Комментарии


Индекс записи передается посредством параметра RecNo.
 Добавление записей в буфер осуществляется методом [AddRecord](IDatasetBatchUpdate.AddRecord.htm).
