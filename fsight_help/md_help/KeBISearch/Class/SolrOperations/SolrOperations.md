# SolrOperations


Сборка: BISearch;


## Описание


Класс SolrOperations реализует
 объект, предназначенный для работы с поисковым сервисом на базе Apache
 Solr.


## Методы объекта класса, унаследованные от [ISolrOperations](../../Interface/ISolrOperations/ISolrOperations.htm)

		 [ClearAll](../../Interface/ISolrOperations/ISolrOperations.ClearAll.htm)
		 Метод ClearAll удаляет
		 все документы с индексированными данными для указанного экземпляра
		 Solr.


		 ![](../../Sub_Image.gif)
		 [GetOverview](../../Interface/ISolrOperations/ISolrOperations.GetOverview.htm)
		 Метод GetOverview извлекает
