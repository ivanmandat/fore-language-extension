# ISourceDataImportInfos


Сборка: BISearch;


## Описание


Интерфейс ISourceDataImportInfos
 содержит свойства и методы для работы с коллекцией настроек индексации
 источников данных.


## Иерархия наследования


ISourceDataImportInfos


## Комментарии


Настройки индексации источников данных можно получить в свойстве [ISourceDataImportContext.SourceInfos](../ISourceDataImportContext/ISourceDataImportContext.SourceInfos.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ISourceDataImportInfos.Count.htm)
		 Свойство Count возвращает
		 количество источников, для которых задаются настройки индексации.


		 ![](../../Property_Image.gif)
		 [Item](ISourceDataImportInfos.Item.htm)
		 Свойство Item возвращает
		 указанные настройки индексации источника данных.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ISourceDataImportInfos.Add.htm)
		 Метод Add добавляет
		 источник данных и возвращает его настройки индексации.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ISourceDataImportInfos.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск настроек индексации по ключу источника данных.


		 ![](../../Sub_Image.gif)
		 [Remove](ISourceDataImportInfos.Remove.htm)
		 Метод Remove удаляет
		 настройки по индексу и возвращает признак удачного удаления.
