# ISourceDataImportInfos.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс элемента с настройками
 индексации.


## Описание


Метод Remove удаляет настройки
 по индексу и возвращает признак удачного удаления.


## Комментарии


Метод возвращает значение True,
 если настройки индексации были удалены и False,
 если удаление по каким-либо причинам невозможно.
