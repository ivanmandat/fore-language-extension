# ISourceDataImportInfos.FindByKey


## Синтаксис


FindByKey(Key: Integer): [ISourceDataImportInfo](../ISourceDataImportInfo/ISourceDataImportInfo.htm);


## Параметры


Key. Ключ источника данных.


## Описание


Метод FindByKey осуществляет
 поиск настроек индексации по ключу источника данных.


## Комментарии


Метод возвращает настройки индексации в случае удачного поиска и Null, если источник с указанным
 ключом не включен в список индексируемых.
