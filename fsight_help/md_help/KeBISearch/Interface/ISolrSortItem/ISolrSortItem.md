# ISolrSortItem


Сборка: BISearch;


## Описание


Интерфейс ISolrSortItem содержит
 свойства для настройки параметров сортировки результатов поиска по определенному
 полю.


## Иерархия наследования


ISolrSortItem


## Комментарии


Параметры сортировки по определенному полю можно получить, используя
 свойства и методы интерфейса [ISolrSortContext](../ISolrSortContext/ISolrSortContext.htm).
