# ISourceDataImportDestinationInfos


Сборка: BISearch;


## Описание


Интерфейс ISourceDataImportDestinationInfos
 содержит свойства и методы для работы с настройками индексации вариантов
 отображения источника данных.


## Иерархия наследования


ISourceDataImportDestinationInfos


## Комментарии


Настройки индексации вариантов отображения возвращает свойство [ISourceDataImportInfo.DestinationInfos](../ISourceDataImportInfo/ISourceDataImportInfo.DestinationInfos.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ISourceDataImportDestinationInfos.Count.htm)
		 Свойство Count возвращает
		 количество вариантов отображения, для которых заданы настройки
		 индексации.


		 ![](../../Property_Image.gif)
		 [Item](ISourceDataImportDestinationInfos.Item.htm)
		 Свойство Item возвращает
		 указанные настройки индексации варианта отображения источника
		 данных.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindByKey](ISourceDataImportDestinationInfos.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск настроек индексации по ключу варианта отображения источника
		 данных.


		 ![](../../Sub_Image.gif)
		 [Remove](ISourceDataImportDestinationInfos.Remove.htm)
		 Метод Remove удаляет
		 настройки индексации и возвращает признак удачного удаления.
