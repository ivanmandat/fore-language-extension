# ISourceDataImportDestinationInfos.FindByKey


## Синтаксис


FindByKey(Key: Integer): [ISourceDataImportDestinationInfo](../ISourceDataImportDestinationInfo/ISourceDataImportDestinationInfo.htm);


## Параметры


Key. Ключ варианта отображения.


## Описание


Метод FindByKey осуществляет
 поиск настроек индексации по ключу варианта отображения источника данных.


## Комментарии


Метод возвращает настройки индексации в случае удачного поиска и Null, если для указанного варианта
 отображения настройки не заданы.
