# ISourceDataImportDestinationInfos.Remove


## Синтаксис


Remove(ItemIndex: Integer): Boolean;


## Параметры


ItemIndex.
 Индекс элемента с настройками индексации.


## Описание


Метод Remove удаляет настройки
 индексации и возвращает признак удачного удаления.


## Комментарии


Метод возвращает значение True,
 если удаления настроек индексации прошло удачно и False,
 если удаление по каким-либо причинам невозможно.
