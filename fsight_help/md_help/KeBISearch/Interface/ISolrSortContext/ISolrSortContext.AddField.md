# ISolrSortContext.AddField


## Синтаксис


AddField(Value: [ISolrField](../ISolrField/ISolrField.htm)):
 [ISolrSortItem](../ISolrSortItem/ISolrSortItem.htm);


## Параметры


Value. Поле, для которого необходимо
 добавить параметры сортировки.


## Описание


Метод AddField добавляет для
 указанного поля параметры сортировки.
