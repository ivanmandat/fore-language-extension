# ISolrSortContext.RemoveBySolrField


## Синтаксис


RemoveBySolrField(Value: [ISolrField](../ISolrField/ISolrField.htm)):
 Boolean;


## Параметры


Value. Поле, по которому необходимо
 удалить параметры сортировки.


## Описание


Метод RemoveBySolrField удаляет
 параметры сортировки по указанному полю и возвращает признак удачного
 удаления.


## Комментарии


Метод возвращает значение True,
 если параметры сортировки были удалены и False,
 если удаление по каким-либо причинам невозможно.
