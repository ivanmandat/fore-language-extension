# ISolrSortContext.FindBySolrField


## Синтаксис


FindBySolrField(Value: [ISolrField](../ISolrField/ISolrField.htm)):
 [ISolrSortItem](../ISolrSortItem/ISolrSortItem.htm);


## Параметры


Value. Поле, по которому необходимо
 найти параметры сортировки.


## Описание


Метод FindBySolrField осуществляет
 поиск параметров сортировки по указанному полю.


## Комментарии


Метод возвращает параметры сортировки по полю в случае удачного поиска
 и Null, если по указанному полю
 сортировка не производится.
