# ISolrFacetFieldValue


Сборка: BISearch;


## Описание


Интерфейс ISolrFacetFieldValue
 содержит свойства для работы с результатом фасетного поиска.


## Иерархия наследования


ISolrFacetFieldValue


## Комментарии


Результат фасетного поиска возвращает свойство [ISolrFacetFieldResults.Item](../ISolrFacetFieldResults/ISolrFacetFieldResults.Item.htm).
