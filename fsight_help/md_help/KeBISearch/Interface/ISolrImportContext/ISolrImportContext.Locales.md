# ISolrImportContext.Locales


## Синтаксис


Locales: Array;


## Описание


Свойство Locales определяет
 список языков, на которых осуществляется индексация данных.


## Комментарии


В качестве значения свойства указывается массив, состоящий из элементов
 перечисления [LocaleCodeID](ForeSys.chm::/Enums/LocaleCodeID.htm).
