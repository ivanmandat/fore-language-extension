# ISolrImportContext


Сборка: BISearch;


## Описание


Интерфейс ISolrImportContext
 содержит свойства для задания базовых настроек индексации данных.


## Иерархия наследования


           ISolrImportContext


## Комментарии


Для создания контекста используйте метод IImportExecutor.CreateContext.
 Для задания настроек индексации приведите результат метода к интерфейсу
 ISolrImportContext.
