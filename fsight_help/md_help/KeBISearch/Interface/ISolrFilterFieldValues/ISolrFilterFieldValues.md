# ISolrFilterFieldValues


Сборка: BISearch;


## Описание


Интерфейс ISolrFilterFieldValues
 содержит свойства для указания значений, по которым будет осуществляться
 фильтрация результата поиска.


## Иерархия наследования


ISolrFilterFieldValues


## Комментарии


Доступ к значениям, по которым осуществляется фильтрация, предоставляет
 свойство [ISolrFilterCondition.Values](../ISolrFilterCondition/ISolrFilterCondition.Values.htm).
