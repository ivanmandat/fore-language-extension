# ISolrField


Сборка: BISearch;


## Описание


Интерфейс ISolrField содержит
 базовые свойства для настройки параметров поля экземпляра Solr.


## Иерархия наследования


ISolrField


## Комментарии


Интерфейс является базовым для интерфейса [ISolrSearchField](../ISolrSearchField/ISolrSearchField.htm).

		 Имя свойства

		 [DataType](ISolrField.DataType.htm)

		 [Fields](ISolrField.Fields.htm)
		 Свойство Fields возвращает

		 [MultiValue](ISolrField.MultiValue.htm)

		 значений.

		 [Type](ISolrField.Type.htm)
