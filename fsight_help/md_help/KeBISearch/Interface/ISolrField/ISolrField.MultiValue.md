# ISolrField.MultiValue


## Синтаксис


MultiValue: Boolean;


## Описание


Свойство MultiValue определяет
 признак возможности хранения по полю множественных значений.


## Комментарии


Допустимые значения:


	- True. По полю могут
	 храниться множественные значения;


	- False. По полю может
	 храниться одиночное значение.


Данный признак может быть изменен только для пользовательских полей.
