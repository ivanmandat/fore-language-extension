# ISolrField.IncludeInSearchResult


## Синтаксис


IncludeInSearchResult: Boolean;


## Описание


Свойство IncludeInSearchResult
 определяет признак включения индексированных значений поля в результаты
 поиска.


## Комментарии


Допустимые значения:


	- True. Включать значения
	 поля в результат поиска;


	- False. Не включать значения
	 поля в результат поиска.


Данный признак может быть изменен только для пользовательских полей.
