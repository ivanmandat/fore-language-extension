# ISolrFacetFieldResults


Сборка: BISearch;


## Описание


Интерфейс ISolrFacetFieldResults
 содержит свойства для работы с коллекцией результатов фасетного поиска,
 сгруппированных по определенному полю.


## Иерархия наследования


ISolrFacetFieldResults


## Комментарии


Результаты фасетного поиска, сгруппированные по определенному полю,
 можно получить, используя свойства и методы интерфейса [ISolrFacetSearchResults](../ISolrFacetSearchResults/ISolrFacetSearchResults.htm).
