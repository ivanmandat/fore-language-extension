# ISourceDataImportInfo


Сборка: BISearch;


## Описание


Интерфейс ISourceDataImportInfo
 содержит свойства для работы с настройками индексации источника данных.


## Иерархия наследования


ISourceDataImportInfo


## Комментарии


Получить настройки индексации можно используя свойства и методы интерфейса
 [ISourceDataImportInfos](../ISourceDataImportInfos/ISourceDataImportInfos.htm).
