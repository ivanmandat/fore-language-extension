# ISolrImportDocumentExtensionInfo


Сборка: BISearch;


## Описание


Интерфейс ISolrImportDocumentExtensionInfo
 содержит свойства для указания настроек, которые используются при заполнении
 пользовательских полей из внешнего источника данных.


## Иерархия наследования


ISolrImportDocumentExtensionInfo


## Комментарии


Настройки возвращает свойство [ISolrSourceDataImportDestinationInfo.DocumentExtensionInfo](../ISolrSourceDataImportDestinationInfo/ISolrSourceDataImportDestinationInfo.DocumentExtensionInfo.htm).


## Свойства


		 Имя свойства

		 [Cube](ISolrImportDocumentExtensionInfo.Cube.htm)
		 Свойство Cube определяет
		 куб, который будет являться внешним источником данных.


		 ![](../../Property_Image.gif)
		 [Dimension](ISolrImportDocumentExtensionInfo.Dimension.htm)
		 Свойство Dimension
		 определяет измерение, по которому будут браться значения для заполнения
		 пользовательского поля.


		 ![](../../Property_Image.gif)
		 [Params](ISolrImportDocumentExtensionInfo.Params.htm)
		 Свойство Params определяет
		 значения параметров, которые будут использоваться для открытия
		 внешнего источника данных.
