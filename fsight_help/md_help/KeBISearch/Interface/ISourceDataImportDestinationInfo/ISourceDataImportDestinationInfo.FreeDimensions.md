# ISourceDataImportDestinationInfo.FreeDimensions


## Синтаксис


FreeDimensions: [IDimensionSet](KeDims.chm::/interface/IDimensionSet/IDimensionSet.htm);


## Описание


Свойство FreeDimensions возвращает
 список свободных измерений.


## Комментарии


Свободное измерение - это измерение, по которому производится поиск,
 но элементы этого измерения не включаются в наименование результата поиска.
 Календарное измерение всегда индексируется как свободное. При необходимости
 добавьте в список свободных другие измерения.
