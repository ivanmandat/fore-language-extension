# ISourceDataImportDestinationInfo.KeyWordsDimensions


## Синтаксис


KeyWordsDimensions: [IDimensionSet](KeDims.chm::/interface/IDimensionSet/IDimensionSet.htm);


## Описание


Свойство KeyWordsDimensions
 возвращает список измерений, для которых будут генерироваться ключевые
 слова.


## Комментарии


Ключевые слова повышают релевантность поиска по измерению, для которого
 они сгенерированы. Включите необходимые измерения в список KeyWordsDimensions.
 Генерация ключевых слов доступна для любых измерений, не включенных в
 состав [свободных](ISourceDataImportDestinationInfo.FreeDimensions.htm).
