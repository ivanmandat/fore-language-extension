# ISourceDataImportDestinationInfo.GroupByDl


## Синтаксис


GroupByDl: Boolean;


## Описание


Свойство GroupByDl определяет
 признак группировки элементов календарного измерения по уровням.


## Комментарии


Допустимые значения:


	- True. Группировать элементы
	 по уровням;


	- False. Значение по умолчанию.
	 Не группировать элементы, индексировать одним списком.
