# ISolrSearchField.BoostFactor


## Синтаксис


BoostFactor: Integer;


## Описание


Свойство BoostFactor определяет
 коэффициент значимости поля.


## Комментарии


Коэффициент значимости влияет на релевантность результатов поиска по
 этому полю.
