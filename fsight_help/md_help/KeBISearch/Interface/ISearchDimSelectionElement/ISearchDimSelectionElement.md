# ISearchDimSelectionElement


Сборка: BISearch;


## Описание


Интерфейс ISearchDimSelectionElement
 содержит свойства для получения информации об индексированном элементе
 измерения.


## Иерархия наследования


ISearchDimSelectionElement


## Комментарии


Параметры индексированного элемента измерения возвращает свойство [ISearchDimSelectionElements.Item](../ISearchDimSelectionElements/ISearchDimSelectionElements.Item.htm).


Каждый элемент индексируется в виде пары: «ключ
 элемента» - «наименование элемента».
