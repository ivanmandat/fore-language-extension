# ISolrFields.FindById


## Синтаксис


FindById(Id: String): [ISolrField](../ISolrField/ISolrField.htm);


## Параметры


Id. Идентификатор поля.


## Описание


Метод FindById осуществляет
 поиск поля по идентификатору.


## Комментарии


Идентификатор поля регистрозависимый. Метод возвращает параметры поля
 в случае удачного поиска и Null,
 если поле с указанным идентификатором отсутствует в коллекции.
