# ISolrFields


Сборка: BISearch;


## Описание


Интерфейс ISolrFields содержит
 свойства и методы для работы с коллекцией полей экземпляра Solr.


## Иерархия наследования


ISolrFields


## Комментарии


Получить коллекцию полей можно используя следующие свойства:


	- [ISolrField.Fields](../ISolrField/ISolrField.Fields.htm);


	- [ISolrSearchEngineSchema.Fields](../ISolrSearchEngineSchema/ISolrSearchEngineSchema.Fields.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ISolrFields.Count.htm)
		 Свойство Count возвращает
		 количество полей в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ISolrFields.Item.htm)
		 Свойство Item возвращает
		 параметры указанного поля.


		 ![](../../Property_Image.gif)
		 [TargetType](ISolrFields.TargetType.htm)
		 Свойство TargetType
		 возвращает назначение экземпляра Solr, которому принадлежит коллекция
		 полей.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddField](ISolrFields.AddField.htm)
		 Метод AddField добавляет
		 поле в коллекцию в соответствии с указанными параметрами.


		 ![](../../Sub_Image.gif)
		 [FindById](ISolrFields.FindById.htm)
		 Метод FindById осуществляет
		 поиск поля по идентификатору.


		 ![](../../Sub_Image.gif)
		 [Remove](ISolrFields.Remove.htm)
		 Метод Remove удаляет
		 поле из коллекции по индексу и возвращает признак удачного удаления.


		 ![](../../Sub_Image.gif)
		 [RemoveById](ISolrFields.RemoveById.htm)
		 Метод RemoveById удаляет
		 поле из коллекции по идентификатору и возвращает признак удачного
		 удаления.
