# ISearchResults


Сборка: BISearch;


## Описание


Интерфейс ISearchResults содержит
 базовые свойства для работы с коллекцией результатов поиска.


## Иерархия наследования


ISearchResults


## Комментарии


Результаты поиска возвращает метод [ISearchExecutor.Search](../ISearchExecutor/ISearchExecutor.Search.htm).


Данный интерфейс является базовым для интерфейса [ISolrSearchResults](../ISolrSearchResults/ISolrSearchResults.htm).
