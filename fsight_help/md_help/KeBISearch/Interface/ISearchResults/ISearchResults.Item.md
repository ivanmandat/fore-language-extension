# ISearchResults.Item


## Синтаксис


Item(ItemIndex: Integer): [ISearchResult](../ISearchResult/ISearchResult.htm);


## Параметры


ItemIndex. Индекс записи с
 результатами поиска.


## Описание


Свойство Item возвращает указанную
 запись с результатами поиска.
