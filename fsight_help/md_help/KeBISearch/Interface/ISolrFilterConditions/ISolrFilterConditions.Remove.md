# ISolrFilterConditions.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс условия фильтрации.


## Описание


Метод Remove удаляет условие
 фильтрации по индексу и возвращает признак удачного удаления.


## Комментарии


Метод возвращает значение True,
 если удаление прошло удачно и False,
 если удаление по каким-либо причинам невозможно.
