# ISolrFilterConditions


Сборка: BISearch;


## Описание


Интерфейс ISolrFilterConditions
 содержит свойства и методы для работы с коллекцией условий фильтрации
 результатов поиска.


## Иерархия наследования


ISolrFilterConditions


## Комментарии


Коллекцию условий фильтрации возвращает свойство [ISolrFilterCondition.Nested](../ISolrFilterCondition/ISolrFilterCondition.Nested.htm).

		 Имя свойства
