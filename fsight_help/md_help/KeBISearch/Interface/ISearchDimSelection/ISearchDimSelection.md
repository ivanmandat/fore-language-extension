# ISearchDimSelection


Сборка: BISearch;


## Описание


Интерфейс ISearchDimSelection
 содержит свойства для работы с индексированным измерением источника данных.


## Иерархия наследования


ISearchDimSelection


## Комментарии


Параметры индексированного измерения возвращает свойство [ISearchDimSelectionSet.Item](../ISearchDimSelectionSet/ISearchDimSelectionSet.Item.htm).
