# ISearchDimSelection.Elements


## Синтаксис


Elements: [ISearchDimSelectionElements](../ISearchDimSelectionElements/ISearchDimSelectionElements.htm);


## Описание


Свойство Elements возвращает
 коллекцию индексированных элементов измерения.


## Комментарии


Каждый элемент индексируется в виде пары: «ключ
 элемента» - «наименование элемента».
