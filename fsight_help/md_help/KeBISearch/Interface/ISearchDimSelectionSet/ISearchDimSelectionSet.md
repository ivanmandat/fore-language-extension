# ISearchDimSelectionSet


Сборка: BISearch;


## Описание


Интерфейс ISearchDimSelectionSet
 содержит свойства для работы с коллекцией индексированных комбинаций измерений
 источников данных.


## Иерархия наследования


ISearchDimSelectionSet


## Комментарии


Коллекцию индексированных комбинаций измерений можно получить, используя
 следующие свойства:


	- [ISourceMetadata.FreeDimensionsSelection](../ISourceMetadata/ISourceMetadata.FreeDimensionsSelection.htm);


	- [ISourceMetadata.Selection](../ISourceMetadata/ISourceMetadata.Selection.htm).
