# ICalendarInfo


Сборка: BISearch;


## Описание


Интерфейс ICalendarInfo содержит
 свойства для получения информации о календарном измерении источника данных.


## Иерархия наследования


ICalendarInfo


## Комментарии


Календарное измерение можно получить в свойстве [ICalendarInfos.Item](../ICalendarInfos/ICalendarInfos.Item.htm).
