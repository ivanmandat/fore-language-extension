# ISolrFilterCondition.Values


## Синтаксис


Values: [ISolrFilterFieldValues](../ISolrFieldValues/ISolrFieldValues.htm);


## Описание


Свойство Values возвращает значения,
 по которым осуществляется фильтрация.


## Комментарии


В результирующем наборе останутся только те записи, значения которых
 по полю [ISolrFilterCondition.SolrField](ISolrFilterCondition.SolrField.htm)
 соответствуют значениям, указанным в свойстве Values.
