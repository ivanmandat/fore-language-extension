# ISolrFilterCondition.Operator_


## Синтаксис


Operator_: [LogicalOperator](../../Enums/LogicalOperator.htm);


## Описание


Свойство Operator_ определяет
 оператор, который будет использоваться для объединения текущего условия
 и вложенных условий.


## Комментарии


Вложенные условия задаются в свойстве [ISolrFilterCondition.Nested](ISolrFilterCondition.Nested.htm).


## Пример


Пример использования приведён в описании свойства [ISolrSearchContext.FilterContext](../ISolrSearchContext/ISolrSearchContext.FilterContext.htm).
