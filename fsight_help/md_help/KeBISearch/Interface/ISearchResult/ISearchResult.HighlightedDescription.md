# ISearchResult.HighlightedDescription


## Синтаксис


HighlightedDescription: String;


## Описание


Свойство HighlightedDescription
 возвращает описание источника данных с подсвеченным искомым текстом в
 нем.


## Комментарии


Значение свойства будет доступно, если искомый текст был найден в индексированном
 [описании](ISearchResult.Description.htm) источника данных.
 Искомый текст будет обрамлён тегом <hlt>...</hlt>.
