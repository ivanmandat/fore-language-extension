# ISearchResult


Сборка: BISearch;


## Описание


Интерфейс ISearchResult содержит
 базовые свойства записи с результатами поиска.


## Иерархия наследования


ISearchResult


## Комментарии


Результаты поиска возвращает свойство [ISearchResults.Item](../ISearchResults/ISearchResults.Item.htm).


Данный интерфейс является базовым для следующих интерфейсов:


	- [ISearchMbObjectResult](../ISearchMbObjectResult/ISearchMbObjectResult.htm);


	- [ISearchSourceDataResult](../ISearchSourceDataResult/ISearchSourceDataResult.htm).
