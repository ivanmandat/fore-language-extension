# ISearchResult.Name


## Синтаксис


Name: String;


## Описание


Свойство Name возвращает наименование
 элемента с результатом поиска.


## Комментарии


В зависимости от того, какой тип поискового сервиса был выбран (значение
 параметра Type в методе [ISearchExecutor.CreateContext](../ISearchExecutor/ISearchExecutor.CreateContext.htm)), свойство
 Name возвращает различные наименования:


	- При поиске по индексированным структурам источников данных -
	 свойство возвращает наименование отдельного среза сформированное из
	 наименований элементов измерений, при пересечении которых формируется
	 срез;


	- При поиске по индексированным метаданным объектов репозитория
	 - свойство возвращает наименование индексированного объекта репозитория.
