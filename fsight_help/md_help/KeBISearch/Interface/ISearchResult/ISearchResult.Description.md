# ISearchResult.Description


## Синтаксис


Description: String;


## Описание


Свойство Description возвращает
 описание источника данных.


## Комментарии


Свойство будет доступно, если для источника в свойстве [IMetabaseObjectDescriptor.Description](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.Description.htm)
 было задано какое-либо описание.
