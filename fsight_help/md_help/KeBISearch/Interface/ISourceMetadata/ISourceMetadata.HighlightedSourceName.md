# ISourceMetadata.HighlightedSourceName


## Синтаксис


HighlightedSourceName: String;


## Описание


Свойство HighlightedSourceName
 возвращает наименование источника данных с учетом подсветки искомого текста.


## Комментарии


Значение свойства будет доступно, если искомый текст был найден в индексированном
 [наименовании](ISourceMetadata.SourceName.htm) источника данных.
 Искомый текст будет обрамлён тегом <hlt>...</hlt>.
