# ISourceMetadata


Сборка: BISearch;


## Описание


Интерфейс ISourceMetadata содержит
 свойства для получения информации о метаданных источника данных.


## Иерархия наследования


ISourceMetadata


## Комментарии


Для получения доступа к метаданным используйте свойство [ISearchSourceDataResult.SourceMetadata](../ISearchSourceDataResult/ISearchSourceDataResult.SourceMetadata.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CalendarInfo](ISourceMetadata.CalendarInfo.htm)
		 Свойство CalendarInfo
		 возвращает информацию о календарных измерениях источника данных.


		 ![](../../Property_Image.gif)
		 [DestinationKey](ISourceMetadata.DestinationKey.htm)
		 Свойство DestinationKey
		 возвращает ключ варианта отображения куба, который был использован
		 при расчете матрицы с данными.


		 ![](../../Property_Image.gif)
		 [FreeDimensionsSelection](ISourceMetadata.FreeDimensionsSelection.htm)
		 Свойство FreeDimensionsSelection
		 возвращает информацию об индексированных комбинациях свободных
		 измерений.


		 ![](../../Property_Image.gif)
		 [HighlightedSourceName](ISourceMetadata.HighlightedSourceName.htm)
		 Свойство HighlightedSourceName
		 возвращает наименование источника данных с учетом подсветки искомого
		 текста.


		 ![](../../Property_Image.gif)
		 [Selection](ISourceMetadata.Selection.htm)
		 Свойство Selection
		 возвращает информацию об индексированных комбинациях элементов
		 измерений.


		 ![](../../Property_Image.gif)
		 [SourceClassId](ISourceMetadata.SourceClassId.htm)
		 Свойство SourceClassId
		 возвращает идентификатор класса источника данных.


		 ![](../../Property_Image.gif)
		 [SourceId](ISourceMetadata.SourceId.htm)
		 Свойство SourceId возвращает
		 идентификатор источника данных.


		 ![](../../Property_Image.gif)
		 [SourceKey](ISourceMetadata.SourceKey.htm)
		 Свойство SourceKey
		 возвращает ключ источника данных.


		 ![](../../Property_Image.gif)
		 [SourceName](ISourceMetadata.SourceName.htm)
		 Свойство SourceName
		 возвращает наименование источника данных.
