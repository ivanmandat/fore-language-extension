# ISolrFacetSearchResults


Сборка: BISearch;


## Описание


Интерфейс ISolrFacetSearchResults
 содержит свойства и методы для работы с результатами [фасетного поиска](UiNav.chm::/Glossary.htm#facetsearch).


## Иерархия наследования


ISolrFacetSearchResults


## Комментарии


Результаты фасетного поиска доступны в свойстве [ISolrSearchResults.FacetSearchResults](../ISolrSearchResults/ISolrSearchResults.FacetSearchResults.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ISolrFacetSearchResults.Count.htm)
		 Свойство Count возвращает
		 количество результатов фасетного поиска.


		 ![](../../Property_Image.gif)
		 [Item](ISolrFacetSearchResults.Item.htm)
		 Свойство Item возвращает
		 результаты фасетного поиска, сгруппированные по какому-либо полю.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](ISolrFacetSearchResults.FindById.htm)
		 Метод FindById осуществляет
		 поиск и возвращает результаты фасетного поиска, сгруппированные
		 по указанному полю.
