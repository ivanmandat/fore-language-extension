# ISolrFacetSearchResults.FindById


## Синтаксис


FindById(Id: String): [ISolrFacetFieldResults](../ISolrFacetFieldResults/ISolrFacetFieldResults.htm);


## Параметры


Id. Идентификатор поля, по
 которому необходимо найти результаты фасетного поиска.


## Описание


Метод FindById осуществляет
 поиск и возвращает результаты фасетного поиска, сгруппированные по указанному
 полю.


## Комментарии


Идентификатор поля регистрозависимый.

Пример работы с коллекцией ISolrFacetSearchResults приведён в описании
 свойства [ISolrSearchResults.FacetSearchResults](../ISolrSearchResults/ISolrSearchResults.FacetSearchResults.htm).
