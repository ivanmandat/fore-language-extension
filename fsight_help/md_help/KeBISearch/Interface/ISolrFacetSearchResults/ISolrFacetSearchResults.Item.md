# ISolrFacetSearchResults.Item


## Синтаксис


Item(ItemIndex: Integer): [ISolrFacetFieldResults](../ISolrFacetFieldResults/ISolrFacetFieldResults.htm);


## Параметры


ItemIndex. Индекс набора результатов
 фасетного поиска, сгруппированных по какому-либо полю.


## Описание


Свойство Item возвращает результаты
 фасетного поиска, сгруппированные по какому-либо полю.


## Пример


Пример работы с коллекцией ISolrFacetSearchResults приведён в описании
 свойства [ISolrSearchResults.FacetSearchResults](../ISolrSearchResults/ISolrSearchResults.FacetSearchResults.htm).
