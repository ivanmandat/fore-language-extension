# ISolrImportProcessOptions.ThreadsCount


## Синтаксис


ThreadsCount: Integer;


## Описание


Свойство ThreadsCount определяет
 число потоков, в рамках которых будет осуществляться построение матрицы
 с данными.


## Комментарии


По умолчанию число потоков равно числу логических процессоров.
