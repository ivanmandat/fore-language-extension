# ISolrImportProcessOptions.NeedSmartSearch


## Синтаксис


NeedSmartSearch: Boolean;


## Описание


Свойство NeedSmartSearch определяет
 признак необходимости индексации свободных измерений для использования
 смарт-поиска.


## Комментарии


Допустимые значения:


	- True. Индексировать
	 свободные измерения, чтобы была возможность использовать эти данные
	 в дальнейшем при поиске;


	- False. Значение по умолчанию.
	 Не индексировать свободные измерения.
