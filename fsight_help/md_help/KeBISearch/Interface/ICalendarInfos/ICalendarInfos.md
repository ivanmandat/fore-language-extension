# ICalendarInfos


Сборка: BISearch;


## Описание


Интерфейс ICalendarInfos содержит
 свойства для работы с коллекцией календарных измерений источника данных.


## Иерархия наследования


ICalendarInfos


## Комментарии


Коллекцию календарных измерений можно получить в свойстве [ISourceMetadata.CalendarInfo](../ISourceMetadata/ISourceMetadata.CalendarInfo.htm).

		 Имя свойства
		 Краткое описание

		 [Count](ICalendarInfos.Count.htm)

		 ![](../../Property_Image.gif)
		 [Item](ICalendarInfos.Item.htm)
		 Свойство Item возвращает
