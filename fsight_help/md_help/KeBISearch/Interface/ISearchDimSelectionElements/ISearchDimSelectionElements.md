# ISearchDimSelectionElements


Сборка: BISearch;


## Описание


Интерфейс ISearchDimSelectionElements
 содержит свойства для работы с коллекцией индексированных элементов измерения.


## Иерархия наследования


ISearchDimSelectionElements


## Комментарии


Коллекцию индексированных элементов возвращает свойство [ISearchDimSelection.Elements](../ISearchDimSelection/ISearchDimSelection.Elements.htm).
