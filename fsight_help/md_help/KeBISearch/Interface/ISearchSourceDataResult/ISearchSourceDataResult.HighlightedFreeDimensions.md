# ISearchSourceDataResult.HighlightedFreeDimensions


## Синтаксис


HighlightedFreeDimensions: Array;


## Описание


Свойство HighlightedFreeDimensions
 возвращает список свободных измерений источника с учетом подсветки в наименованиях
 искомого текста.


## Комментарии


Искомое значение будет обрамлено тегом <hlt>...</hlt>.
