# ISearchSourceDataResult.HighlightedTime


## Синтаксис


HighlightedTime: Array;


## Описание


Свойство HighlightedTime возвращает
 список годов, по которым имеются данные с подсвеченным искомым текстом.


## Комментарии


Значение свойства будет доступно, если искомый текст был найден в индексированных
 [номерах годов](ISearchSourceDataResult.Time.htm), по которым
 имеются данные. Искомый текст будет обрамлён тегом <hlt>...</hlt>.
