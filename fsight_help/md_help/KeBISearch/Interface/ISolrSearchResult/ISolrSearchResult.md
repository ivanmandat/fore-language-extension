# ISolrSearchResult


Сборка: BISearch;


## Описание


Интерфейс ISolrSearchResult
 содержит свойства для получения результатов поиска по пользовательским
 полям.


## Иерархия наследования


ISolrSearchResult


## Комментарии


Для работы с результатами поиска по пользовательским полям приведите
 значение свойства [ISearchResults.Item](../ISearchResults/ISearchResults.Item.htm)
 к интерфейсу ISolrSearchResult.
