# ISearchEngineSchema.SearchExecutor


## Синтаксис


SearchExecutor(Type: [SearchEngineTargetType](../../Enums/SearchEngineTargetType.htm)):
 [ISearchExecutor](../ISearchExecutor/ISearchExecutor.htm);


## Параметры


Type. Назначения поискового
 сервиса.


## Описание


Свойство SearchExecutor возвращает
 объект, используемый для поиска по индексированным данным, в зависимости
 от заданного типа поискового сервиса.
