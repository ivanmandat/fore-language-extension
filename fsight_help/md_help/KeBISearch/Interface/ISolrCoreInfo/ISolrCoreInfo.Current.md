# ISolrCoreInfo.Current


## Синтаксис


Current: Boolean;


## Описание


Свойство Current возвращает
 признак выполнения каких-либо действий с экземпляром Solr в текущий момент.


## Комментарии


Свойство возвращает значение True,
 если в текущий момент выполняются какие-либо действия и False,
 если экземпляр Solr в текущий момент не используется.
