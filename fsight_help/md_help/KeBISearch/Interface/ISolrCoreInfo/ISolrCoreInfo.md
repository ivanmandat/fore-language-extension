# ISolrCoreInfo


Сборка: BISearch;


## Описание


Интерфейс ISolrCoreInfo содержит
 свойства для получения информации об экземпляре Solr.


## Иерархия наследования


ISolrCoreInfo


## Комментарии


Информацию об указанном экземпляре Solr возвращает метод [ISolrOperations.GetOverview](../ISolrOperations/ISolrOperations.GetOverview.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Current](ISolrCoreInfo.Current.htm)
		 Свойство Current возвращает
		 признак выполнения каких-либо действий с экземпляром Solr в текущий
		 момент.


		 ![](../../Property_Image.gif)
		 [DeletedDocs](ISolrCoreInfo.DeletedDocs.htm)
		 Свойство DeletedDocs
		 возвращает количество документов Solr, которые были удалены.


		 ![](../../Property_Image.gif)
		 [HasDeletions](ISolrCoreInfo.HasDeletions.htm)
		 Свойство HasDeletions
		 возвращает признак наличия удаленных документов экземпляра Solr.


		 ![](../../Property_Image.gif)
		 [LastModified](ISolrCoreInfo.LastModified.htm)
		 Свойство LastModified
		 возвращает дату и время последнего изменения документов, связанных
		 с текущим экземпляром Solr.


		 ![](../../Property_Image.gif)
		 [MaxDoc](ISolrCoreInfo.MaxDoc.htm)
		 Свойство MaxDoc возвращает
		 максимальное допустимое количество документов, которые могут быть
		 созданы для экземпляра Solr.


		 ![](../../Property_Image.gif)
		 [NumDocs](ISolrCoreInfo.NumDocs.htm)
		 Свойство NumDocs возвращает
		 текущее количество документов, созданных для экземпляра Solr.
