# ISearchMbObjectResult.HighlightedClassName


## Синтаксис


HighlightedClassName: String;


## Описание


Свойство HighlightedClassName
 возвращает наименование класса объекта с подсвеченным искомым текстом.


## Комментарии


Значение свойства будет доступно, если искомый текст был найден в индексированном
 [наименовании класса](ISearchMbObjectResult.ClassName.htm)
 объекта. Искомый текст будет обрамлён тегом <hlt>...</hlt>.
