# ISearchMbObjectResult.HighlightedKey


## Синтаксис


HighlightedKey: String;


## Описание


Свойство HighlightedKey возвращает
 ключ объекта репозитория с подсвеченным искомым текстом.


## Комментарии


Значение свойства будет доступно, если искомый текст был найден в индексированном
 [ключе](ISearchMbObjectResult.Key.htm) объекта. Искомый текст
 будет обрамлён тегом <hlt>...</hlt>.
