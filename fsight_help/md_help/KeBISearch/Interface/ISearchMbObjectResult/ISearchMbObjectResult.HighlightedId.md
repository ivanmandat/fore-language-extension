# ISearchMbObjectResult.HighlightedId


## Синтаксис


HighlightedId: String;


## Описание


Свойство HighlightedId возвращает
 идентификатор объекта репозитория с подсвеченным искомым текстом.


## Комментарии


Значение свойства будет доступно, если искомый текст был найден в индексированном
 [идентификаторе](ISearchMbObjectResult.Id.htm) объекта. Искомый
 текст будет обрамлён тегом <hlt>...</hlt>.
