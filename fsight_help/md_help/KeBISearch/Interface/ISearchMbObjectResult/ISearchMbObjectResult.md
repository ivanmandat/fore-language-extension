# ISearchMbObjectResult


Сборка: BISearch;


## Описание


Интерфейс ISearchMbObjectResult
 содержит свойства записи с результатами поиска по метаданным объектов
 репозитория.


## Иерархия наследования


           [ISearchResult](../ISearchResult/ISearchResult.htm)


           ISearchMbObjectResult


## Комментарии


При поиске по метаданным объектов репозитория для получения результата
 поиска приведите элемент, описываемый интерфейсом [ISearchResult](../ISearchResult/ISearchResult.htm),
 к интерфейсу ISearchMbObjectResult.
