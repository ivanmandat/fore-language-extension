# ISolrCustomFieldSearchResults.Item


## Синтаксис


Item(Index: Integer): [ISolrCustomFieldSearchResult](../ISolrCustomFieldSearchResult/ISolrCustomFieldSearchResult.htm);


## Параметры


Index. Индекс элемента с результатами
 поиска по какому-либо пользовательскому полю.


## Описание


Свойство Item возвращает результаты
 поиска по пользовательскому полю.
