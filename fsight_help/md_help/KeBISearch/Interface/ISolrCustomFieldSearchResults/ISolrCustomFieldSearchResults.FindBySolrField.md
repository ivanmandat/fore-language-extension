# ISolrCustomFieldSearchResults.FindBySolrField


## Синтаксис


FindBySolrField(Value: [ISolrField](../ISolrField/ISolrField.htm)):
 [ISolrCustomFieldSearchResult](../ISolrCustomFieldSearchResult/ISolrCustomFieldSearchResult.htm);


## Параметры


Value. Пользовательское поле,
 для которого необходимо найти результаты поиска.


## Описание


Метод FindBySolrField осуществляет
 поиск и возвращает результаты для указанного пользовательского поля.


## Комментарии


Метод возвращает результаты в случае удачного поиска и Null,
 если по указанному пользовательскому полю ни чего не найдено.
