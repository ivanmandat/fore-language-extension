# ISolrCustomFieldSearchResults


Сборка: BISearch;


## Описание


Интерфейс ISolrCustomFieldSearchResults
 содержит свойства и методы для получения результатов поиска по пользовательским
 полям.


## Иерархия наследования


ISolrCustomFieldSearchResults


## Комментарии


Результаты поиска по пользовательским полям возвращает свойство [ISolrSearchResult.CustomFieldSearchResults](../ISolrSearchResult/ISolrSearchResult.CustomFieldSearchResults.htm).

		 Имя метода
