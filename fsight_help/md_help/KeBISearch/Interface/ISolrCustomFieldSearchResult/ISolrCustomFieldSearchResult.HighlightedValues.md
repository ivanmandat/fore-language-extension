# ISolrCustomFieldSearchResult.HighlightedValues


## Синтаксис


HighlightedValues: Array;


## Описание


Свойство HighlightedValues возвращает
 массив найденных значений поля с учетом подсветки искомого значения.


## Комментарии


Искомое значение будет обрамлено тегом <hlt>...</hlt>.
