# ISolrCustomFieldSearchResult


Сборка: BISearch;


## Описание


Интерфейс ISolrCustomFieldSearchResult
 содержит свойства для работы с результатом поиска по пользовательскому
 полю.


## Иерархия наследования


ISolrCustomFieldSearchResult


## Комментарии


Результат поиска по пользовательскому полю можно получить, используя
 свойства и методы интерфейса [ISolrCustomFieldSearchResults](../ISolrCustomFieldSearchResults/ISolrCustomFieldSearchResults.htm).
