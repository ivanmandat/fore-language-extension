# ISolrSearchOptions.Operator_


## Синтаксис


Operator_: [LogicalOperator](../../Enums/LogicalOperator.htm);


## Описание


Свойство Operator_ определяет
 оператор, который будет использоваться для объединения слов в поисковом
 запросе.


## Комментарии


По умолчанию свойству установлено значение [LogicalOperator.And](../../Enums/LogicalOperator.htm).
 Допустимо указывать значения [LogicalOperator.And](../../Enums/LogicalOperator.htm)
 или [LogicalOperator.Or](../../Enums/LogicalOperator.htm).


## Пример


Пример использования приведён в описании свойства [ISolrSearchEngineSchema.SearchOptions](../ISolrSearchEngineSchema/ISolrSearchEngineSchema.SearchOptions.htm).
