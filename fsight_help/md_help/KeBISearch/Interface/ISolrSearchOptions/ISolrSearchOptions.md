# ISolrSearchOptions


Сборка: BISearch;


## Описание


Интерфейс ISolrSearchOptions
 содержит свойства для настройки параметров, которые используются при поиске
 данных.


## Иерархия наследования


ISolrSearchOptions


## Комментарии


Параметры, используемые при поиске данных, можно получить в свойстве
 [ISolrSearchEngineSchema.SearchOptions](../ISolrSearchEngineSchema/ISolrSearchEngineSchema.SearchOptions.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Operator_](ISolrSearchOptions.Operator_.htm)
		 Свойство Operator_
		 определяет оператор, который будет использоваться для объединения
		 слов в поисковом запросе.


		 ![](../../Property_Image.gif)
		 [RelevanceImpactFormula](ISolrSearchOptions.RelevanceImpactFormula.htm)
		 Свойство RelevanceImpactFormula
		 определяет формулу влияния на релевантность поиска.


		 ![](../../Property_Image.gif)
		 [UseSmartSearch](ISolrSearchOptions.UseSmartSearch.htm)
		 Свойство UseSmartSearch
		 определяет признак поиска по свободным измерениям.
