# ISolrImportCustomBinding


Сборка: BISearch;


## Описание


Интерфейс ISolrImportCustomBinding
 содержит свойства для определения настроек, которые используются при индексации
 пользовательских полей.


## Иерархия наследования


ISolrImportCustomBinding


## Комментарии


Настройки, используемые при индексации пользовательских полей, возвращает
 свойство [ISolrImportProcessOptions.ImportBinding](../ISolrImportProcessOptions/ISolrImportProcessOptions.ImportBinding.htm).
