# ISolrImportCustomBinding.Assembly


## Синтаксис


Assembly: [IMetabaseObjectDescriptor](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm);


## Описание


Свойство Assembly определяет
 описание объекта среды разработки содержащего реализацию класса, который
 будет использоваться при обработке пользовательских полей.


## Комментарии


В качестве значения свойства необходимо указать описание сборки.

Пример использования приведён в описании свойства [ISolrSearchEngineSchema.ImportProcessOptions](../ISolrSearchEngineSchema/ISolrSearchEngineSchema.ImportProcessOptions.htm).
