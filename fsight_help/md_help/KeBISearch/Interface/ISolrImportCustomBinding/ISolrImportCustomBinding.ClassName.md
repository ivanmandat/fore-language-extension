# ISolrImportCustomBinding.ClassName


## Синтаксис


ClassName: String;


## Описание


Свойство ClassName определяет
 наименование класса, который будет использоваться при обработке пользовательских
 полей.


## Комментарии


В качестве значения свойства необходимо указать наименование класса,
 который реализует интерфейс [ISolrImportCustomValues](../ISolrImportCustomValues/ISolrImportCustomValues.htm).


## Пример


Пример использования приведён в описании свойства [ISolrSearchEngineSchema.ImportProcessOptions](../ISolrSearchEngineSchema/ISolrSearchEngineSchema.ImportProcessOptions.htm).
