# ISolrFieldValue


Сборка: BISearch;


## Описание


Интерфейс ISolrFieldValue содержит
 свойства для задания значения пользовательского поля при индексации.


## Иерархия наследования


ISolrFieldValue


## Комментарии


Доступ к элементу, используемому для задания значения пользовательского
 поля, предоставляют свойства и методы интерфейса [ISolrFieldValues](../ISolrFieldValues/ISolrFieldValues.htm).
