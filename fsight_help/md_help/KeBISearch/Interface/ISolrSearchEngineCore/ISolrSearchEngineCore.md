# ISolrSearchEngineCore


Сборка: BISearch;


## Описание


Интерфейс ISolrSearchEngineCore
 содержит свойства, отвечающие за параметры экземпляра Solr.


## Иерархия наследования


ISolrSearchEngineCore


## Комментарии


Для получения параметров экземпляра Solr используйте свойства или методы
 интерфейса [ISolrSearchEngineCores](../ISolrSearchEngineCores/ISolrSearchEngineCores.htm).
