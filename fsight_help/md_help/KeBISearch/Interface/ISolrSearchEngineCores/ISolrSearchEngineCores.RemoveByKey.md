# ISolrSearchEngineCores.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ удаляемого экземпляра
 Solr.


## Описание


Метод RemoveByKey удаляет настройки
 экземпляра Solr по ключу.


## Комментарии


Метод возвращает значение True,
 если удаление прошло удачно и False,
 если удаление по каким-либо причинам невозможно.
