# ISolrSearchEngineCores.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемого экземпляра
 Solr.


## Описание


Метод Remove удаляет настройки
 экземпляра Solr по индексу.


## Комментарии


Метод возвращает значение True,
 если удаление прошло удачно и False,
 если удаление по каким-либо причинам невозможно.
