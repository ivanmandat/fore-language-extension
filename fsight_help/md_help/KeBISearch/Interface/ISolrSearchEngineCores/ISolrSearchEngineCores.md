# ISolrSearchEngineCores


Сборка: BISearch;


## Описание


Интерфейс ISolrSearchEngineCores
 содержит свойства и методы для работы с коллекцией экземпляров Solr.


## Иерархия наследования


ISolrSearchEngineCores


## Комментарии


Для получения коллекции используйте свойство [ISolrSearchEngineService.Cores](../ISolrSearchEngineService/ISolrSearchEngineService.Cores.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ISolrSearchEngineCores.Count.htm)
		 Свойство Count возвращает
		 количество экземпляров Solr в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ISolrSearchEngineCores.Item.htm)
		 Свойство Item возвращает
		 параметры указанного экземпляра Solr.
