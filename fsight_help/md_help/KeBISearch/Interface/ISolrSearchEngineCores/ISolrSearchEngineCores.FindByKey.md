# ISolrSearchEngineCores.FindByKey


## Синтаксис


FindByKey(Key: Integer): [ISolrSearchEngineCore](../ISolrSearchEngineCore/ISolrSearchEngineCore.htm);


## Параметры


Key. Ключ экземпляра Solr.


## Описание


Метод FindByKey осуществляет
 поиск экземпляра Solr по ключу.


## Комментарии


Метод возвращает параметры экземпляра Solr в случае удачного поиска
 и Null, если экземпляр не найден.
