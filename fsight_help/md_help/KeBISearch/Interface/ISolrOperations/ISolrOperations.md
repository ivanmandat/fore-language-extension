# ISolrOperations


Сборка: BISearch;


## Описание


Интерфейс ISolrOperations содержит
 методы для работы с поисковым сервисом на базе Apache Solr.


## Иерархия наследования


ISolrOperations


## Комментарии


Для работы с поисковым сервисом на базе Apache Solr инициализируйте
 объект класса [SolrOperations](../../Class/SolrOperations/SolrOperations.htm).
