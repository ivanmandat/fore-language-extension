# ISolrImportFieldBindings.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемых параметров
 привязки пользовательского поля.


## Описание


Метод Remove удаляет параметры
 привязки по индексу и возвращает признак удачного удаления.


## Комментарии


Метод возвращает значение True,
 если параметры привязки были удалены и False,
 если параметры привязки по каким-либо причинам не могут быть удалены.
