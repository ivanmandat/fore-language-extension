# ISolrImportFieldBindings.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ удаляемых параметров
 привязки пользовательского поля.


## Описание


Метод RemoveByKey удаляет параметры
 привязки по ключу и возвращает признак удачного удаления.


## Комментарии


Метод возвращает значение True,
 если параметры привязки были удалены и False,
 если параметры привязки по каким-либо причинам не могут быть удалены.
