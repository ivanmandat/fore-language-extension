# ISolrImportFieldBindings.RemoveByField


## Синтаксис


RemoveByField(Field: [ISolrField](../ISolrField/ISolrField.htm)):
 Boolean;


## Параметры


Field. Пользовательское поле,
 для которого необходимо удалить параметры привязки.


## Описание


Метод RemoveByField удаляет
 параметры привязки по полю и возвращает признак удачного удаления.


## Комментарии


Метод возвращает значение True,
 если параметры привязки были удалены и False,
 если параметры привязки по каким-либо причинам не могут быть удалены.
