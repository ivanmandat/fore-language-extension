## С чего начать


		 Сборка BISearch
		 предназначена для управления процессами индексирования и полнотекстового
		 поиска данных по ключевым словам в структуре многомерных источников
		 в репозитории. Указанные процессы лежат в основе [сервиса BI-поиска](Setup.chm::/BISearch/BISearch_Setup.htm),
		 используемого в различных инструментах продукта «Форсайт. Аналитическая платформа».


		Изучив данный раздел, вы сможете с применением языка Fore:


			- настраивать репозиторий для взаимодействия с поисковой
			 платформой Apache Solr;


			- производить индексацию или индексировать метаданные
			 любых типов объектов репозитория;


			- производить поиск по индексированным данным в различных
			 режимах;


			- настраивать задачу обновления поискового индекса, которая
			 позволяет автоматизировать процесс индексации источников.


		Более подробно с возможностями блока можно ознакомиться в разделе
		 «[Введение в сборку BISearch](Intro/KeBISearch_Intro.htm)».
