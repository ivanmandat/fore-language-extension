# SearchSortMode


## Описание


Перечисление SearchSortMode
 содержит возможные варианты направления сортировки.


Используется следующими свойствами и методами:


	- [ISolrSortItem.SortDirection](../Interface/ISolrSortItem/ISolrSortItem.SortDirection.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Ascending. Сортировать
		 по возрастанию значений.


		 1
		 Descending. Сортировать
		 по убыванию значений.
