# LogicalOperator


## Описание


Перечисление LogicalOperator
 содержит операторы, которые могут использоваться для объединения различных
 элементов при настройке поискового сервиса.


Используется следующими свойствами и методами:


	- [ISolrFilterFieldValues.Operator_](../Interface/ISolrFilterFieldValues/ISolrFilterFieldValues.Operator_.htm);


	- [ISolrFilterCondition.Operator_](../Interface/ISolrFilterCondition/ISolrFilterCondition.Operator_.htm);


	- [ISolrSearchOptions.Operator_](../Interface/ISolrSearchOptions/ISolrSearchOptions.Operator_.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 -1
		 None. Оператор не используется.


		 0
		 And_. Логический AND.


		 1
		 Or_. Логический OR.


		 2
		 Not_. Логический NOT.
