# SearchEngineTargetType


## Описание


Перечисление SearchEngineTargetType
 содержит варианты назначения различных экземпляров поискового сервиса.


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 MbObject. Для хранения
		 и поиска по индексированным метаданным источников данных.


		 1
		 SourceData. Для хранения
		 и поиска по структуре источников данных (элементы измерений).


		 2
		 FreeDimensions. Для
		 хранения и поиска по свободным измерениям источников (измерениям,
		 элементы которых индексируются, но не отображаются в результате
		 поиска).
