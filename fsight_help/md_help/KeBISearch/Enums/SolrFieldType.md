# SolrFieldType


## Описание


Перечисление SolrFieldType содержит
 возможные типы полей экземпляра Solr.


Используется следующими свойствами и методами:


	- [ISolrField.Type](../Interface/ISolrField/ISolrField.Type.htm);


	- [ISolrFields.AddField](../Interface/ISolrFields/ISolrFields.AddField.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Meta. Обычное поле.


		 1
		 System. Системное поле.


		 2
		 Search. Поле для поиска.


		 3
		 SystemSearch. Системное
		 поле для поиска.


		 4
		 Facet. Поле для фасетного
		 поиска.


		 5
		 SystemFacet. Системное
		 поле для фасетного поиска.
