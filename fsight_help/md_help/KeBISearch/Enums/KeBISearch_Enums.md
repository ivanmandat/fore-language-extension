# Перечисления сборки BISearch


		 Перечисление
		 Краткое описание


		 ![](../Enums_Image.gif)
		 [LogicalOperator](LogicalOperator.htm)
		 Перечисление LogicalOperator
		 содержит операторы, которые могут использоваться для объединения
		 различных элементов при настройке поискового сервиса.


		 ![](../Enums_Image.gif)
		 [SearchEngineTargetType](SearchEngineTargetType.htm)
		 Перечисление SearchEngineTargetType
		 содержит варианты назначения различных экземпляров поискового
		 сервиса.


		 ![](../Enums_Image.gif)
		 [SearchSortMode](SearchSortMode.htm)
		 Перечисление SearchSortMode
		 содержит возможные варианты направления сортировки.


		 ![](../Enums_Image.gif)
		 [SolrFieldBindingType](SolrFieldBindingType.htm)
		 Перечисление SolrFieldBindingType содержит
		 типы привязок для автоматического заполнения пользовательских
		 полей при индексации.


		 ![](../Enums_Image.gif)
		 [SolrFieldType](SolrFieldType.htm)
		 Перечисление SolrFieldType
		 содержит возможные типы полей экземпляра Solr.


		 ![](../Enums_Image.gif)
		 [SolrImportCleanType](SolrImportCleanType.htm)
		 Перечисление SolrImportCleanType содержит
		 варианты очистки поискового индекса перед обновлением.


		 ![](../Enums_Image.gif)
		 [SolrSearchType](SolrSearchType.htm)
		 Перечисление SolrSearchType
		 содержит возможные варианты поиска.


		 ![](../Enums_Image.gif)
		 [SourceImportMode](SourceImportMode.htm)
		 Перечисление SourceImportMode определяет
		 режим индексации варианта отображения источника данных.
