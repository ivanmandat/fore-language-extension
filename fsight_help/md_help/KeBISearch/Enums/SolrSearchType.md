# SolrSearchType


## Описание


Перечисление SolrSearchType
 содержит возможные варианты поиска.


Используется следующими свойствами и методами:


	- [ISolrSearchContext.SearchType](../Interface/ISolrSearchContext/ISolrSearchContext.SearchType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 SearchOnly. Только
		 обычный поиск. После поиска будут получены записи с результатами
		 поиска. Каждая запись содержит информацию о найденном срезе данных/метаданных.


		 2
		 FacetOnly. Только фасетный
		 поиск. После поиска будут доступны результаты только фасетного
		 поиска: результаты сгруппированные по фасетным полям. Каждый результат
		 содержит только общую информацию, включающую значения фасетных
		 полей и количество сгруппированных по ним результатов поиска.
		 В дальнейшем по значениям фасетных полей можно организовать дополнительный
		 поиск для получения конкретных результатов поиска.

Результаты фасетного поиска возвращает свойство
		 [ISolrSearchResults.FacetSearchResults](../Interface/ISolrSearchResults/ISolrSearchResults.FacetSearchResults.htm).


		 3
		 FacetAndSearch. Обычный
		 и фасетный поиск. После поиска будут доступны записи с информацией
		 о найденных срезах данных/метаданных и результаты фасетного поиска,
		 содержащие общую информацию сгруппированную по фасетным полям.
