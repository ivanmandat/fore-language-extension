# SolrImportCleanType


## Описание


Перечисление SolrImportCleanType содержит
 варианты очистки поискового индекса перед обновлением.


Используется следующими свойствами и методами:


	- [ISolrImportContext.CleanType](../Interface/ISolrImportContext/ISolrImportContext.CleanType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 -1
		 None. Не очищать индекс.


		 0
		 All. Очистить индекс
		 полностью.


		 1
		 Specified. Очистить
		 индексированные данные для указанных источников.
