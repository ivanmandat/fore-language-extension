# ForeGC


Сборка: Fore;


## Описание


Класс ForeGC содержит статические
 методы для запуска процесса сборки мусора.


## Комментарии


Класс ForeGC является аналогом класса [ForeGarbageCollector](../ForeGarbageCollector/ForeGarbageCollector.htm),
 но не требует инициализации нового объекта. При работе «Форсайт. Аналитическая платформа»
 сборка мусора выполняется автоматически с заданной периодичностью, но
 при необходимости стратегия сборки мусора может быть изменена. Более подробно
 читайте в подразделе «[Сборка мусора](KnowledgeBase.chm::/01_Fore/KB000030/KnowledgeBase_KB000030_2.htm)».
