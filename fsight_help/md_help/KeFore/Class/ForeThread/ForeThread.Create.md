# ForeThread.Create


## Синтаксис


		Create(Method: [IForeMethod](../../Interface/IForeMethod/IForeMethod.htm));


## Параметры


Method. Пользовательский метод,
 который будет выполняться в отдельном потоке.


## Описание


Конструктор Create создаёт поток,
 связанный с указанным пользовательским методом.
