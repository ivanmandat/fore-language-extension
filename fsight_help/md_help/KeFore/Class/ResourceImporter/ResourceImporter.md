# ResourceImporter


## Описание


Класс ResourceImporter содержит
 свойства и методы, применяемые для [импорта
 ресурсов](UiNav.chm::/Multilanguage/Resources/Import_Resource.htm).


## Свойства объекта класса, унаследованные от [IResourceImporter](../../Interface/IResourceImporter/IResourceImporter.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Locales](../../Interface/IResourceImporter/IResourceImporter.Locales.htm)
		 Свойство Locales определяет
		 языки ресурсов для импорта.


		 ![](../../Property_Image.gif)
		 [Log](../../Interface/IResourceImporter/IResourceImporter.Log.htm)
		 Свойство Log возвращает
		 отчет о ходе импорта.


		 ![](../../Property_Image.gif)
		 [Method](../../Interface/IResourceImporter/IResourceImporter.Method.htm)
		 Свойство Method определяет
		 режим импорта ресурсов.


		 ![](../../Property_Image.gif)
		 [Provider](../../Interface/IResourceImporter/IResourceImporter.Provider.htm)
		 Свойство Provider определяет
		 объект, из которого будет произведен импорт.


		 ![](../../Property_Image.gif)
		 [Resource](../../Interface/IResourceImporter/IResourceImporter.Resource.htm)
		 Свойство Resource определяет
		 объект-ресурсы, в который должен быть произведен импорт.
