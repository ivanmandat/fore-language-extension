# AppServerClass


## Описание


Класс AppServerClass реализует
 объект, позволяющий управлять работой планировщика задач продукта «Форсайт. Аналитическая платформа».


## Методы объекта класса, унаследованные от [IAppServerClass](../../Interface/IAppServerClass/IAppServerClass.htm)


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [IsRunning](../../Interface/IAppServerClass/IAppServerClass.IsRunning.htm)
		 Метод IsRunning осуществляет
		 проверку наличия запущенного планировщика задач с указанными настройками.


		 ![](../../Sub_Image.gif)
		 [Refresh](../../Interface/IAppServerClass/IAppServerClass.Refresh.htm)
		 Метод Refresh осуществляет
		 обновление настроек запущенного планировщика задач и возвращает
		 результат обновления.


		 ![](../../Sub_Image.gif)
		 [Run](../../Interface/IAppServerClass/IAppServerClass.Run.htm)
		 Метод Run осуществляет
		 запуск планировщика задач в соответствии с указанными параметрами.
