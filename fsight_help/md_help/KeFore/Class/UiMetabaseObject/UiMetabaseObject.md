# UiMetabaseObject


## Описание


Класс UiMetabaseObject является
 базовым для компонентов, осуществляющих доступ к объектам репозитория.


## Свойства объекта класса, унаследованные от [IUiMetabaseObject](../../Interface/IUiMetabaseObject/IUiMetabaseObject.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Active](../../Interface/IUiMetabaseObject/IUiMetabaseObject.Active.htm)
		 Свойство Active определяет,
		 будет ли активным компонент.


		 ![](../../Property_Image.gif)
		 [Instance](../../Interface/IUiMetabaseObject/IUiMetabaseObject.Instance.htm)
		 Свойство Instance определяет
		 данные объекта репозитория, загруженного в компонент.


		 ![](../../Property_Image.gif)
		 [Object](../../Interface/IUiMetabaseObject/IUiMetabaseObject.Object.htm)
		 Свойство Object определяет
		 объект репозитория, к которому необходимо получить доступ.
