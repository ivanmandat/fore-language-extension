# IScheduledTaskAuditAlert.Operation


## Синтаксис


Operation: Integer;


## Описание


Свойство Operation определяет
 операцию, при выполнении которой будет генерироваться событие.


## Комментарии


Для определения результата выполнения операции, при котором будет генерироваться
 событие, используйте свойство [IScheduledTaskAuditAlert.Result](IScheduledTaskAuditAlert.Result.htm).
