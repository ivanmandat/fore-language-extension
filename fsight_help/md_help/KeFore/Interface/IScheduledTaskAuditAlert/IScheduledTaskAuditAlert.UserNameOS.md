# IScheduledTaskAuditAlert.UserNameOS


## Синтаксис


UserNameOS: String;


## Описание


Свойство UserNameOS определяет
 пользователя операционной системы, генерирующего событие.


## Комментарии


Для определения рабочей станции, на которой генерируется событие, используйте
 свойство [IScheduledTaskAuditAlert.Station](IScheduledTaskAuditAlert.Station.htm).
