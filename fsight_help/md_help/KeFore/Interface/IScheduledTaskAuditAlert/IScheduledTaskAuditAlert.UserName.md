## Описание


Свойство UserName определяет
 пользователя репозитория, генерирующего событие.


## Комментарии


Для определения пользователя операционной системы, генерирующего событие,
 используйте свойство [IScheduledTaskAuditAlert.UserNameOS](IScheduledTaskAuditAlert.UserNameOS.htm).
