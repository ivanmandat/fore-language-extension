# IScheduledTaskAuditAlert.Station


## Синтаксис


Station: String;


## Описание


Свойство Station определяет
 рабочую станцию, на которой генерируется событие.


## Комментарии


Для определения пользователя репозитория, генерирующего событие, используйте
 свойство [IScheduledTaskAuditAlert.UserName](IScheduledTaskAuditAlert.UserName.htm).
