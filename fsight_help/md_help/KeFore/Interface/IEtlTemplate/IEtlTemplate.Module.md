# IEtlTemplate.Module


## Синтаксис


Module: [IMetabaseObjectDescriptor](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm);


## Описание


Свойство Module определяет объект
 Fore, в котором содержится класс для реализации необходимых интерфейсов.


## Комментарии


К объектам Fore относятся:


	- модуль;


	- сборка;


	- форма.


Для использования данного свойства укажите класс для реализации необходимых
 интерфейсов в свойстве [IEtlTemplate.Class](IEtlTemplate.Class.htm).
