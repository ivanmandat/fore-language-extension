# IEtlTemplate.Class


## Синтаксис


Class_: String;


## Описание


Свойство Class определяет
 класс объектов Fore, реализующий необходимые интерфейсы.


## Комментарии


К объектам Fore относятся:


	- модуль;


	- сборка;


	- форма.


Для использования данного свойства укажите объект Fore в свойстве [IEtlTemplate.Module](IEtlTemplate.Module.htm).
