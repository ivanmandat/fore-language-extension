# IEtlTemplate


Сборка: Fore;


## Описание


Интерфейс IEtlTemplate используется
 для работы с пользовательским шаблоном источника/приёмника данных задачи
 ETL.


## Иерархия наследования


           [INamedEntity](KeSom.chm::/Interface/INamedEntity/INamedEntity.htm)


           IEtlTemplate


## Комментарии


Для работы с коллекцией пользовательских шаблонов источников/приёмников
 данных используйте интерфейс [IEtlTemplates](../IEtlTemplates/IEtlTemplates.htm).
