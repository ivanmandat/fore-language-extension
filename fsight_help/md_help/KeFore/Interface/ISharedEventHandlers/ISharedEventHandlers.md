# ISharedEventHandlers

## Описание


Интерфейс SharedEventHandlers
 используется для работы с [обработчиками
 событий](UiDevEnv.chm::/04_NavigatorSetting/EventHandler.htm) отчетов.

## Комментарии


Для получения объекта данного типа используйте [ISharedParams.EventHandlers](../ISharedParams/ISharedParams.EventHandlers.htm).
