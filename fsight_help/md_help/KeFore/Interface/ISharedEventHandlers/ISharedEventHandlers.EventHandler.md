# ISharedEventHandlers.EventHandler


## Синтаксис


EventHandler(Type: [SharedEventHandlerType](../../Enums/SharedEventHandlerType.htm)):
 [ISharedEventHandler](../ISharedEventHandler/ISharedEventHandler.htm);


## Параметры


Type. Тип отчетов, использующих
 обработчик событий.


## Описание


Свойство EventHandler возвращает
 объект для обработки событий отчёта, заданного в параметре.


## Комментарии


Данное свойство актуально, если свойство [ISharedEventHandlers.Enabled](ISharedEventHandlers.Enabled.htm)
 принимает значение True.
