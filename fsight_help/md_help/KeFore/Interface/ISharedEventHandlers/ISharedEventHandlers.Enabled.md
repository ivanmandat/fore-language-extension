# ISharedEventHandlers.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет,
 используется ли [обработчики
 событий](UiDevEnv.chm::/04_NavigatorSetting/EventHandler.htm) отчетов.


## Комментарии


Допустимые значения:


	- True. Обработчики используется;


	- False. Обработчики не
	 используется.
