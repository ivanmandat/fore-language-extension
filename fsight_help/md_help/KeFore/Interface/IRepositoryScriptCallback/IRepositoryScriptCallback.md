# IRepositoryScriptCallback


Сборка: Fore;


## Описание


Интерфейс IRepositoryScriptCallback
 содержит методы, реализующие события, возникающие во время выполнения
 операций над репозиторием. Данный интерфейс должен быть переопределен
 в пользовательском классе.
