# IScheduledTaskValidationChecker.ExceptionCount


## Синтаксис


ExceptionCount: Integer;


## Описание


Свойство ExceptionCount определяет
 число исключений из правила.


## Комментарии


Для определения типа исключения используйте свойство [IScheduledTaskValidationChecker.Condition](IScheduledTaskValidationChecker.Condition.htm).
