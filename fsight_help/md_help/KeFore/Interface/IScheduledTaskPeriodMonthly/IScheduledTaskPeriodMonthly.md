# IScheduledTaskPeriodMonthly


## Описание


Интерфейс IScheduledTaskPeriodMonthly
 содержит свойства и методы, определяющие параметры ежемесячного расчета
 задачи.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Day](IScheduledTaskPeriodMonthly.Day.htm)
		 Свойство Day определяет
		 день месяца, в который будет производиться расчет.


		 ![](../../Property_Image.gif)
		 [DayOfWeek](IScheduledTaskPeriodMonthly.DayOfWeek.htm)
		 Свойство DayOfWeek
		 определяет день недели, в который будет производиться расчет задачи.


		 ![](../../Property_Image.gif)
		 [Months](IScheduledTaskPeriodMonthly.Months.htm)
		 Свойство Months определяет,
		 будет ли производиться расчет в указанном месяце.


		 ![](../../Property_Image.gif)
		 [StartTime](IScheduledTaskPeriodMonthly.StartTime.htm)
		 Свойство StartTime
		 определяет время запуска задачи.


		 ![](../../Property_Image.gif)
		 [WeekOfMonth](IScheduledTaskPeriodMonthly.WeekOfMonth.htm)
		 Свойство WeekOfMonth
		 определяет неделю месяца, в которой будет производиться расчет.

## Методы, унаследованные от [IScheduledTaskPeriod](../IScheduledTaskPeriod/IScheduledTaskPeriod.htm)


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
