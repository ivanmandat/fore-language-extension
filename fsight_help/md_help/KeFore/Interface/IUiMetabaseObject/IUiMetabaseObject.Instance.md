# IUiMetabaseObject.Instance


## Синтаксис


Instance: Object;


## Описание


Свойство Instance определяет
 данные объекта репозитория, загруженного в компонент.


## Комментарии


При чтении значения данного свойства можно получить экземпляр открытого
 объекта, описание которого задано в свойстве [Object](IUiMetabaseObject.Object.htm).


Задавая значение данного свойства можно подключить к компоненту открытый
 экземпляр объекта.


При чтении/записи значение данного свойства должно быть приведено к
 интерфейсу, описывающему открытый экземпляр объекта, для работы с которым
 предназначен используемый компонент.


## Пример


Пример использования приведен в описании свойства [OperationMode](IUiMetabaseObject.OperationMode.htm).
