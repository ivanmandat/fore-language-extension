# IUiMetabaseObject.Active


## Синтаксис


Active: Boolean;


## Описание


Свойство Active определяет, будет ли активным компонент, т.е. будут ли доступны данные объекта репозитория, загруженного в компонент.


## Комментарии


Для открытия доступа к объекту репозитория необходимо свойству установить значение True. Если свойству установлено значение False, компонент будет связан с объектом репозитория, но передаваться данные в компонент не будут.
