# IUiMetabaseObject.ObjectClassId


## Синтаксис


ObjectClassId: Integer;


## Описание


Свойство ObjectClassId возвращает числовой идентификатор, соответствующий классу объекта, загруженного в компонент. Значения идентификатора класса объектов содержатся в перечислимом типе [MetabaseObjectClass](KeSom.chm::/Enums/MetabaseObjectClass.htm).
