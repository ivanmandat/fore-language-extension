# IRepositoryScriptManager.VersionMajor


## Синтаксис


VersionMajor: Integer;


## Описание


Свойство VersionMajor возвращает
 номер основной версии репозитория.


## Комментарии


Для получения версии необходимо выполнить метод [FetchVersion](IRepositoryScriptManager.FetchVersion.htm).
