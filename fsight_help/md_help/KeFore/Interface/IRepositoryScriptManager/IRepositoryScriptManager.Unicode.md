# IRepositoryScriptManager.Unicode


## Синтаксис


		Unicode: Boolean;


## Описание


Свойство Unicode определяет
 признак создания юникод версии репозитория.


## Комментарии


Если свойству установлено значение True,
 то при создании репозитория все строковые поля таблиц в базе данных будут
 создаваться с типом NVARCHAR.
