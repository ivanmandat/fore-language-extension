# IRepositoryScriptManager


Сборка: Fore;


## Описание


Интерфейс IRepositoryScriptManager
 содержит свойства и методы для работы с менеджером репозитория.


## Иерархия наследования


IRepositoryScriptManager


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Connection](IRepositoryScriptManager.Connection.htm)
		 Свойство Connection
		 определяет соединение с сервером БД в рамках которого осуществляется
		 работа.


		 ![](../../Property_Image.gif)
		 [Driver](IRepositoryScriptManager.Driver.htm)
		 Свойство Driver определяет
		 драйвер, используемый при работе с репозиторием.


		 ![](../../Property_Image.gif)
		 [Operation](IRepositoryScriptManager.Operation.htm)
		 Свойство Operation
		 определяет операцию, которую необходимо произвести над репозиторием.


		 ![](../../Property_Image.gif)
		 [Repository](IRepositoryScriptManager.Repository.htm)
		 Свойство Repository
		 определяет тип репозитория, для которого необходимо произвести
		 [указанную
		 операцию](IRepositoryScriptManager.Operation.htm).


		 ![](../../Property_Image.gif)
		 [Unicode](IRepositoryScriptManager.Unicode.htm)
		 Свойство Unicode определяет
		 признак создания юникод версии репозитория.


		 ![](../../Property_Image.gif)
		 [VersionMajor](IRepositoryScriptManager.VersionMajor.htm)
		 Свойство VersionMajor
		 возвращает номер основной версии репозитория.


		 ![](../../Property_Image.gif)
		 [VersionMinor](IRepositoryScriptManager.VersionMinor.htm)
		 Свойство VersionMinor
		 возвращает номер промежуточной версии репозитория.


		 ![](../../Property_Image.gif)
		 [VersionUnicode](IRepositoryScriptManager.VersionUnicode.htm)
		 Свойство VersionUnicode
		 возвращает признак юникод версии репозитория.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Execute](IRepositoryScriptManager.Execute.htm)
		 Метод Execute осуществляет
		 выполнение операции с репозиторием.


		 ![](../../Sub_Image.gif)
		 [FetchVersion](IRepositoryScriptManager.FetchVersion.htm)
		 Метод FetchVersion
		 осуществляет получение текущей версии репозитория.


		 ![](../../Sub_Image.gif)
		 [LoadScript](IRepositoryScriptManager.LoadScript.htm)
		 Метод LoadScript загружает
		 скрипт, используемый для осуществления [указанной
		 операции](IRepositoryScriptManager.Operation.htm) над репозиторием.
