# IRepositoryScriptManager.VersionUnicode


## Синтаксис


VersionUnicode: Boolean;


## Описание


Свойство VersionUnicode возвращает
 признак юникод версии репозитория.


## Комментарии


Свойство возвращает значение True,
 если репозиторий создан с признаком юникод версии.


Для получения версии необходимо выполнить метод [FetchVersion](IRepositoryScriptManager.FetchVersion.htm).


Примечание.
 Данное свойство вычисляется только для репозиториев НСИ.
