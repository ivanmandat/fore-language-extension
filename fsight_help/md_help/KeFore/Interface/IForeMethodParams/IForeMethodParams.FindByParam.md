# IForeMethodParams.FindByParam


## Синтаксис


		FindByParam(Param: String): [IForeMethodParam](../IForeMethodParam/IForeMethodParam.htm);


## Параметры


Param. Наименование параметра.


## Описание


Метод FindByParam осуществляет
 поиск параметра по его наименованию.


## Комментарии


Метод возвращает параметр в случае удачного поиска и Null,
 если указанный параметр не найден.
