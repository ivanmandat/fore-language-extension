# IForeMethodParams


## Описание


Интерфейс IForeMethodParams
 содержит свойства и методы для работы с коллекцией параметров пользовательского
 метода.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IForeMethodParams.Count.htm)
		 Свойство Count возвращает
		 количество параметров в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IForeMethodParams.Item.htm)
		 Свойство Item возвращает
		 параметр из коллекции по его индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindByParam](IForeMethodParams.FindByParam.htm)
		 Метод FindByParam осуществляет
		 поиск параметра по его наименованию.
