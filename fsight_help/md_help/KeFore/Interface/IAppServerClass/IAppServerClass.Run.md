# IAppServerClass.Run


## Синтаксис


Run(Instance: String; Arguments: String);


## Параметры


Instance. Идентификатор запускаемого
 экземпляра планировщика задач. В качестве значения параметра необходимо
 указать наименование XML-файла, содержащего настройки планировщика задач.
 Если указана пустая строка, то будет осуществляться запуск планировщика
 задач с настройками из файла «Scheduler.xml».


Примечание.
 Файлы настроек для запуска планировщика задач должны храниться в папке
 установки «Форсайт. Аналитическая платформа».
 Наименование файла в данном параметре указывается без расширения «xml».


Arguments. Дополнительные параметры
 запуска планировщика задач. Параметры, с которыми может осуществляться
 запуск планировщика задач, представлены в разделе [Запуск планировщика
 задач](UiAppSrv.chm::/1_Work_AppSrv/UiAppSrv_Work_Run.htm).


## Описание


Метод Run осуществляет запуск
 планировщика задач в соответствии с указанными параметрами.


## Комментарии


Настройки, указанные в параметре Instance
 будут переданы в ключ «-Instance» при запуске планировщика задач. Все
 остальные ключи передаются в параметре Arguments.


## Пример


Для выполнения примера в папке установки «Форсайт. Аналитическая платформа»
 должен иметься файл «Test.xml», содержащий настройки для запуска планировщика
 задач.


			Sub UserProc;

Var

    AppSvr: IAppServerClass;

Begin

    AppSvr := New AppServerClass.Create;

    AppSvr.Run("Test", "-Silent");

End Sub UserProc;


При выполнении примера будет запущен планировщик задач с настройками
 из указанного файла.
