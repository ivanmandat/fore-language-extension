# IForeProperty


Сборка: Fore;


## Описание


Интерфейс IForeProperty предназначен
 для работы с пользовательскими свойствами.


## Иерархия наследования


IForeProperty


## Комментарии


Интерфейс позволяет получить информацию о структуре пользовательских
 свойств, описанных в интерфейсах или реализованных в классах.


Получить свойство можно с помощью методов: [IForeClass.BindToProperty](../IForeClass/IForeClass.BindToProperty.htm),
 [IForeClass.PropertyItem](../IForeClass/IForeClass.PropertyItem.htm).
