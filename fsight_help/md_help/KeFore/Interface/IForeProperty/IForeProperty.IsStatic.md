# IForeProperty.IsStatic


## Синтаксис


IsStatic: Boolean;


## Описание


Свойство IsStatic возвращает
 признак соответствия статическому свойству.


## Комментарии


Свойство возвращает значение True,
 если пользовательское свойство в классе описано с директивой [Shared](Fore.chm::/02_GeneralInfo/Fore_ClassesAndObjects.htm#shared).
