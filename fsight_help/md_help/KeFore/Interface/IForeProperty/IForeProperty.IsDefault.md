# IForeProperty.IsDefault


## Синтаксис


		IsDefault: Boolean;


## Описание


Свойство IsDefault возвращает
 признак соответствия свойству по умолчанию.


## Комментарии


Свойство возвращает значение True,
 если пользовательское свойство в классе описано с директивой [Default](Fore.chm::/06_SyntRules/Description_Property.htm).
