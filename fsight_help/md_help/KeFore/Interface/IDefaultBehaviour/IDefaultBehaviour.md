# IDefaultBehaviour


Сборка: Fore;


## Описание


Интерфейс IDefaultBehaviour
 содержит свойства для работы с [настройками по умолчанию](UiNav.chm::/02_Navigator/Repo_Default.htm).


## Иерархия наследования


           IDefaultBehaviour


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [RDSElementCard](IDefaultBehaviour.RDSElementCard.htm)
		 Свойство RDSElementCard
		 предоставляет возможность выбора вида карточки элемента справочника
		 НСИ.


		 ![](../../Property_Image.gif)
		 [SelectionBorderColor](IDefaultBehaviour.SelectionBorderColor.htm)
		 Свойство SelectionBorderColor
		 определяет цвет границы выделения ячеек.


		 ![](../../Property_Image.gif)
		 [SelectionColor](IDefaultBehaviour.SelectionColor.htm)
		 Свойство SelectionColor
		 определяет цвет выделенных ячеек.


		 ![](../../Property_Image.gif)
		 [TreatEmptySelectionAsFull](IDefaultBehaviour.TreatEmptySelectionAsFull.htm)
		 Свойство TreatEmptySelectionAsFull
		 предоставляет возможность рассматривать [пустую
		 отметку как полную](UiNav.chm::/02_Navigator/Repo_Default.htm#behavior) при настройке поведения,
		 которое будет использоваться в инструментах продукта «Форсайт. Аналитическая платформа».


## Метод


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [IsSelectionColorDefault](IDefaultBehaviour.IsSelectionColorDefault.htm)
		 Метод IsSelectionColorDefault
		 проверяет, используются ли настройки оформления в отчётах по умолчанию.


		 ![](../../Sub_Image.gif)
		 [ResetSelectionColors](IDefaultBehaviour.ResetSelectionColors.htm)
		 Метод ResetSelectionColors
		 сбрасывает настройки оформления в отчётах и устанавливает значения
		 настроек по умолчанию.
