# IDefaultBehaviour.IsSelectionColorDefault


## Синтаксис


		IsSelectionColorDefault: Boolean;


## Описание


Метод IsSelectionColorDefault
 проверяет, используются ли настройки оформления в отчётах по умолчанию.


## Комментарии


Метод возвращает значение True,
 если используются настройки по умолчанию и False,
 если настройки были изменены.
