# IScheduledTaskProperties.Alerts


## Синтаксис


Alerts: [IScheduledTaskAlerts](../IScheduledTaskAlerts/IScheduledTaskAlerts.htm);


## Описание


Свойство Alerts возвращает событие,
 по наступлению которого будет выполняться задача.


## Комментарии


Для определения идентификатора системного события используйте свойство
 [IScheduledTaskProperties.EventId](IScheduledTaskProperties.EventId.htm).
