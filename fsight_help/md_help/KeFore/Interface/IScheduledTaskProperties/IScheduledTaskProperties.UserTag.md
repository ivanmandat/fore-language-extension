# IScheduledTaskProperties.UserTag


## Синтаксис


UserTag: String;


## Описание


Свойство UserTag доступно только через Fore, не используется компилятором и может содержать любые пользовательские данные. По умолчанию в данное свойство записывается наименование пользователя платформы, создавшего задачу.
