# IScheduledTaskProperties.HiddenMailRecipients


## Синтаксис


HiddenMailRecipients: String;


## Описание


Свойство HiddenMailRecipients
 определяет электронные адреса, по которым будет осуществляться отправка
 скрытой копии уведомления о выполнении задачи.


## Комментарии


Адреса указываются через разделитель «;».
