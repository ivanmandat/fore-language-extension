# IScheduledTaskProperties.Active


## Синтаксис


Active: Boolean;


## Описание


Свойство Active определяет активность
 задачи.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Выполнение задачи разрешено;


	- False. Выполнение задачи
	 запрещено.
