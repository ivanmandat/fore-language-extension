# IScheduledTaskProperties.DynamicMailListModule


## Синтаксис


DynamicMailListModule: [IMetabaseObjectDescriptor](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm);


## Описание


Свойство DynamicMailListModule
 определяет объект среды разработки, содержащий реализацию макроса, который
 будет формировать динамический список адресатов для отправки уведомления
 о завершении задачи.


## Комментарии


В качестве значения свойства необходимо указать описание Fore-модуля/формы/сборки,
 в котором реализован макрос, заданный в [IScheduledTaskProperties.DynamicMailListMacro](IScheduledTaskProperties.DynamicMailListMacro.htm).
