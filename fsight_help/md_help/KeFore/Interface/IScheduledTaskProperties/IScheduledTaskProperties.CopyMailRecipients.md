# IScheduledTaskProperties.CopyMailRecipients


## Синтаксис


CopyMailRecipients: String;


## Описание


Свойство CopyMailRecipients
 определяет электронные адреса, по которым будет осуществляться отправка
 копии уведомления о выполнении задачи.


## Комментарии


Адреса указываются через разделитель «;».
