# IScheduledTaskProperties.EventId


## Синтаксис


EventId: String;


## Описание


Свойство EventId определяет
 идентификатор события.


## Комментарии


Для получения события, по наступлению которого будет выполняться задача,
 используйте свойство [IScheduledTaskProperties.Alerts](IScheduledTaskProperties.Alerts.htm).
