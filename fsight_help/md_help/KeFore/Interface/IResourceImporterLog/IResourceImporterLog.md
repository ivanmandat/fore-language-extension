# IResourceImporterLog


## Описание


Интерфейс IResourceImporterLog
 содержит свойства для работы с отчетом о ходе импорта ресурсов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IResourceImporterLog.Count.htm)
		 Свойство Count возвращает
		 количество строк ресурсов, которые были импортированы.


		 ![](../../Property_Image.gif)
		 [Item](IResourceImporterLog.Item.htm)
		 Свойство Item возвращает
		 импортированную строку ресурсов по ее индексу.


		 ![](../../Property_Image.gif)
		 [State](IResourceImporterLog.State.htm)
		 Свойство State возвращает
		 статус импорта строки ресурсов по ее индексу.
