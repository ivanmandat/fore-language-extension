# IForeObjectsGraph.ObjectIndexing


## Синтаксис


		ObjectIndexing: Boolean;


## Описание


Свойство ObjectIndexing определяет,
 будут ли нумероваться объекты на графе.


## Комментарии


Если свойству установлено значение True,
 то объекты будут нумероваться при выводе графа в файл.
