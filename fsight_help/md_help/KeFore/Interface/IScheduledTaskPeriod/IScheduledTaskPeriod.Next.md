# IScheduledTaskPeriod.Next


## Синтаксис


Next(Current: DateTime): DateTime;


## Параметры


Current - дата, относительно которой необходимо рассчитать.


## Описание


Метод Next осуществляет расчет следующей даты и времени запуска задачи относительно даты Current с учетом параметров периода.
