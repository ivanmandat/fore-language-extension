# IScheduledTaskPeriod


## Описание


Интерфейс IScheduledTaskPeriod
 содержит базовые свойства и методы, определяющие параметры периода расчета
 задачи.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [StopDateTime](IScheduledTaskPeriod.StopDateTime.htm)
		 Свойство StopDateTime
		 определяет дату окончания действия периода, в который производится
		 расчет задачи.


		 ![](../../Property_Image.gif)
		 [Type](IScheduledTaskPeriod.Type.htm)
		 Свойство Type возвращает
		 способ задания периода.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Assign](IScheduledTaskPeriod.Assign.htm)
		 Метод Assign устанавливает
		 параметры периода расчета задачи в соответствии с параметрами
		 указанного периода.


		 ![](../../Sub_Image.gif)
		 [Next](IScheduledTaskPeriod.Next.htm)
		 Метод Next осуществляет
		 расчет следующей даты и времени запуска задачи относительно указанной
		 даты с учетом параметров периода.
