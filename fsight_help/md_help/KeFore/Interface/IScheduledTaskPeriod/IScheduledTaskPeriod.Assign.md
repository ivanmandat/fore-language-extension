# IScheduledTaskPeriod.Assign


## Синтаксис


Assign(Value: [IScheduledTaskPeriod](IScheduledTaskPeriod.htm));


## Параметры


Value - параметры периода расчета задачи, которые необходимо установить для данного периода.


## Описание


Метод Assign устанавливает параметры периода расчета задачи в соответствии с параметрами, передаваемыми посредством значения Value.
