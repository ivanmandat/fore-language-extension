# IForeSub.ModuleName


## Синтаксис


		ModuleName: String;


## Описание


Свойство ModuleName возвращает
 наименование модуля, в котором находится непосредственная реализация метода.


## Комментарии


Свойство актуально, если метод реализован в одном из модулей/форме в
 сборке.
