# IForeSub.ResultValue


## Синтаксис


		ResultValue: Variant;


## Описание


Свойство ResultValue возвращает
 результат выполнения функции.


## Комментарии


Свойство актуально, если свойство [IsResult](IForeSub.IsResult.htm)
 возвращает значение True. Значение
 свойства будет доступно после выполнения функции с помощью метода [Invoke](IForeSub.Invoke.htm).
