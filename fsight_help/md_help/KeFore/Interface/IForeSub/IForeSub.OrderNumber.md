# IForeSub.OrderNumber


## Синтаксис


		OrderNumber: Integer;


## Описание


Свойство OrderNumber возвращает
 порядковый номер метода в коде.


## Комментарии


Нумерация всех методов в коде производится по порядку, сверху вниз.
