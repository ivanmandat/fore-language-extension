# IForeSub.IsStatic


## Синтаксис


		IsStatic: Boolean;


## Описание


Свойство IsStatic возвращает
 признак соответствия статической процедуре/функции.


## Комментарии


Свойство возвращает значение True,
 если процедура/функция описана в глобальном пространстве имён сборки или
 описана в классе и имеет в своей сигнатуре директиву [Shared](Fore.chm::/02_GeneralInfo/Fore_ClassesAndObjects.htm#shared). Во всех
 других случаях свойство возвращает значение False.
