# IForeSub.IsConstructor


## Синтаксис


		IsConstructor: Boolean;


## Описание


Свойство IsConstructor возвращает
 признак соответствия конструктору класса.


## Комментарии


Допустимые значения:


	- True. Конструктор класса;


	- False. Процедура или
	 функция, для проверки используйте свойство [IForeSub.IsResult](IForeSub.IsResult.htm).
