# IForeSub.ResultName


## Синтаксис


		ResultName: String;


## Описание


Свойство ResultName возвращает
 наименование типа у результата функции.


## Комментарии


Свойство актуально, если свойство [IsResult](IForeSub.IsResult.htm)
 возвращает значение True.
