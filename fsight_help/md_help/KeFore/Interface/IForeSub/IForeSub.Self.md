# IForeSub.Self


## Синтаксис


		Self: [IForeObject](../IForeObject/IForeObject.htm);


## Описание


Свойство Self определяет экземпляр
 родительского класса, в рамках которого будет выполняться метод.


## Комментарии


Свойство Self доступно только
 на запись.


Свойство необходимо задать, если осуществляется работа с процедурой/функцией/конструктором
 класса. Экземпляр класса возвращает метод [IForeClass.CreateObject](../IForeClass/IForeClass.CreateObject.htm).
