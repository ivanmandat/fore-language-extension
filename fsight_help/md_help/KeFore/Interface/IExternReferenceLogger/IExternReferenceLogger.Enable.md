# IExternReferenceLogger.Enable


## Синтаксис


		Enable: Boolean;


## Описание


Свойство Enable определяет,
 осуществляется ли логирование изменений счётчика внешних ссылок.


## Комментарии


Для включения логирования установите свойству значение True,
 для остановки логирования - False.
