# IExternReferenceLogger


Сборка: Fore;


## Описание


Интерфейс IExternReferenceLogger
 предназначен для управления логированием изменения счётчика внешних ссылок.


## Иерархия наследования


IExternReferenceLogger


## Комментарии


Доступ для управления логированием изменения счётчика внешних ссылок
 предоставляет свойство [IForeRuntime.ExternReferenceLogger](../IForeRuntime/IForeRuntime.ExternReferenceLogger.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Collapse](IExternReferenceLogger.Collapse.htm)
		 Свойство Collapse определяет,
		 будут ли удаляться из лога идентичные записи.


		 ![](../../Property_Image.gif)
		 [Enable](IExternReferenceLogger.Enable.htm)
		 Свойство Enable определяет,
		 осуществляется ли логирование изменений счётчика внешних ссылок.


		 ![](../../Property_Image.gif)
		 [ObjectIndexing](IExternReferenceLogger.ObjectIndexing.htm)
		 Свойство ObjectIndexing
		 определяет, будет ли осуществляться нумерация объектов при выводе
		 лога в файл.
