# IExternReferenceLogger.Collapse


## Синтаксис


		Collapse: Boolean;


## Описание


Свойство Collapse определяет,
 будут ли удаляться из лога идентичные записи.


## Комментарии


Если свойству установить значение True,
 то из лога будут удаляться идентичные записи, за исключением тех, где
 в первой происходит увеличение ссылок, а во второй уменьшение.
