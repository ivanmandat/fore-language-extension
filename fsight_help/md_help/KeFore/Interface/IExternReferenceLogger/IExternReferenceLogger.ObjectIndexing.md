# IExternReferenceLogger.ObjectIndexing


## Синтаксис


		ObjectIndexing: Boolean;


## Описание


Свойство ObjectIndexing определяет,
 будет ли осуществляться нумерация объектов при выводе лога в файл.


## Комментарии


Допустимые значения:


	- True.
	 Осуществляется нумерация объектов;


	- False. Нумерация объектов
	 не осуществляется.
