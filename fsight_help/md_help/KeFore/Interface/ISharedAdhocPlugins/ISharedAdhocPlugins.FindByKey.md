# ISharedAdhocPlugins.FindByKey


## Синтаксис


FindById(Key: Integer): [ISharedAdhocPlugin](../ISharedAdhocPlugin/ISharedAdhocPlugin.htm);


## Параметры


Key. Ключ искомого плагина.


## Описание


Метод FindByKey выполняет поиск
 плагина с заданным ключом.


## Комментарии


Если плагин с указанным ключом не найден, то метод вернет значение Null.
