# ISharedAdhocPlugins.Add


## Синтаксис


Add: [ISharedAdhocPlugin](../ISharedAdhocPlugin/ISharedAdhocPlugin.htm);


## Описание


Метод Add добавляет новый
 плагин в коллекцию.


## Комментарии


Для удаления определенного плагина из коллекции используйте методы [ISharedAdhocPlugins.Remove](ISharedAdhocPlugins.Remove.htm)
 и [ISharedAdhocPlugins.RemoveByKey](ISharedAdhocPlugins.RemoveByKey.htm),
 для удаления всех плагинов - метод [ISharedAdhocPlugins.Clear](ISharedAdhocPlugins.Clear.htm).
