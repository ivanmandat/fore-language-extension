# ISharedAdhocPlugins.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс плагина в коллекции.


## Описание


Метод Remove удаляет указанный
 плагин из коллекции.


## Комментарии


Индексация плагинов в коллекции сквозная и начинается с нуля. Для получения
 количества плагинов в коллекции используйте свойство [ISharedAdhocPlugins.Count](ISharedAdhocPlugins.Count.htm).


Если удаление плагина выполнено успешно, то метод возвращает значение
 True, в противном случае - False.
