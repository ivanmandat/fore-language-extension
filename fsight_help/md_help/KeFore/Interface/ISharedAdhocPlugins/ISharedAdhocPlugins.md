# ISharedAdhocPlugins


Сборка: Fore;


## Описание


Интерфейс ISharedAdhocPlugins
 предназначен для работы с коллекцией плагинов, подключенных в инструменте
 «Аналитические панели».


## Иерархия наследования


           ISharedAdhocPlugins


## Комментарии


Для работы с отдельным плагином из коллекции используйте интерфейс [ISharedAdhocPlugin](../ISharedAdhocPlugin/ISharedAdhocPlugin.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ISharedAdhocPlugins.Count.htm)
		 Свойство Count
		 возвращает количество плагинов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ISharedAdhocPlugins.Item.htm)
		 Свойство Item
