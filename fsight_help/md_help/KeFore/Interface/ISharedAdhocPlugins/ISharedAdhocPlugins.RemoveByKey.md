# ISharedAdhocPlugins.RemoveByKey


## Синтаксис


Remove(Key: Integer): Boolean;


## Параметры


Key. Ключ плагина.


## Описание


Метод RemoveByKey удаляет из
 коллекции плагин с указанным ключом.


## Комментарии


Если удаление плагина выполнено успешно, то метод возвращает значение
 True, в противном случае - False.
