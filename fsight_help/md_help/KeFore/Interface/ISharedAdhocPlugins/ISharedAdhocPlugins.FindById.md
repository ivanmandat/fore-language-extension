# ISharedAdhocPlugins.FindById


## Синтаксис


FindById(Id: String): [ISharedAdhocPlugin](../ISharedAdhocPlugin/ISharedAdhocPlugin.htm);


## Параметры


Id. Идентификатор искомого
 плагина.


## Описание


Метод FindById выполняет
 поиск плагина с заданным идентификатором.


## Комментарии


Если плагин с указанным идентификатором не найден, то метод вернет значение
 Null.
