# IScheduledTaskAlerts.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ удаляемого события.


## Описание


Метод RemoveByKey удаляет событие
 из коллекции по ключу.


## Комментарии


В случае успешного удаления события из коллекции метод возвращает True, иначе - False.


Для поиска события в коллекции по ключу используйте метод [IScheduledTaskAlerts.FindByKey](IScheduledTaskAlerts.FindByKey.htm).
