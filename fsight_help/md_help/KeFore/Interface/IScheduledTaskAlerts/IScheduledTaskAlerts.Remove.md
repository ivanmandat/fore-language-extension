# IScheduledTaskAlerts.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемого события.


## Описание


Метод Remove удаляет событие
 из коллекции по индексу.


## Комментарии


В случае успешного удаления события из коллекции метод возвращает True, иначе - False.


Для поиска события в коллекции по идентификатору используйте метод [IScheduledTaskAlerts.FindById](IScheduledTaskAlerts.FindById.htm).
