# IScheduledTaskAlerts.Add


## Синтаксис


Add(Type: [ScheduledTaskAlertType](../../Enums/ScheduledTaskAlertType.htm)):
 [IScheduledTaskAlert](../IScheduledTaskAlert/IScheduledTaskAlert.htm);


## Параметры


Type. Тип события.


## Описание


Метод Add добавляет новое событие
 в коллекцию.


## Комментарии


Для получения количества событий в коллекции используйте свойство [IScheduledTaskAlerts.Count](IScheduledTaskAlerts.Count.htm).
