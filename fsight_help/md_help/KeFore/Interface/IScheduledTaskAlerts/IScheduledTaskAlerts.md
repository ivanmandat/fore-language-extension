# IScheduledTaskAlerts


Сборка: Fore;


## Описание


Интерфейс IScheduledTaskAlerts
 предназначен для работы с коллекцией событий, по наступлению которых будет
 выполнена задача.


## Иерархия наследования


           IScheduledTaskAlerts


## Комментарии


Для работы с событием, на поступлению которого будет выполнена задача,
 используйте интерфейс [IScheduledTaskAlert](../IScheduledTaskAlert/IScheduledTaskAlert.htm).
