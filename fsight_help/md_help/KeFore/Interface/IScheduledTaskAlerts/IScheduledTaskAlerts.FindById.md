# IScheduledTaskAlerts.FindById


## Синтаксис


FindById(Id: String): [IScheduledTaskAlert](../IScheduledTaskAlert/IScheduledTaskAlert.htm);


## Параметры


Id. Идентификатор события.


## Описание


Метод FindById осуществляет
 поиск события в коллекции по идентификатору.


## Комментарии


Для удаления события из коллекции по индексу используйте метод [IScheduledTaskAlerts.Remove](IScheduledTaskAlerts.Remove.htm).
