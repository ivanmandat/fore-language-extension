# IScheduledTaskAlerts.Item


## Синтаксис


Item(Index: Integer): [IScheduledTaskAlert](../IScheduledTaskAlert/IScheduledTaskAlert.htm);


## Параметры


Index. Индекс события в коллекции.


## Описание


Свойство Item возвращает событие
 из коллекции по его индексу.


## Комментарии


Для получения количества событий в коллекции используйте свойство [IScheduledTaskAlerts.Count](IScheduledTaskAlerts.Count.htm).
