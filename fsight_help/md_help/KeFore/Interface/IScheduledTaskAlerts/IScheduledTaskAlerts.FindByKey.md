# IScheduledTaskAlerts.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IScheduledTaskAlert](../IScheduledTaskAlert/IScheduledTaskAlert.htm);


## Параметры


Key. Ключ события.


## Описание


Метод FindByKey осуществляет
 поиск события в коллекции по ключу.


## Комментарии


Для удаления события из коллекции по ключу используйте метод [IScheduledTaskAlerts.RemoveByKey](IScheduledTaskAlerts.RemoveByKey.htm).
