# IScheduledTaskAlerts.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 событий в коллекции.


## Комментарии


Для получения события из коллекции по его индексу используйте свойство
 [IScheduledTaskAlerts.Item](IScheduledTaskAlerts.Item.htm).
