# ISharedEventHandler.EventsClass


## Синтаксис


EventsClass: String;


## Описание


Свойство EventsClass определяет
 класс [обработчика
 событий](UiExpress.chm::/setup_express_report/eventhandler.htm) отчетов.


## Комментарии


Данное свойство актуально, если свойства [ISharedEventHandler.Enabled](ISharedEventHandler.Enabled.htm)
 принимает значение True.
