# ISharedEventHandler.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет,
 используется ли [обработчик
 событий](UiDevEnv.chm::/04_NavigatorSetting/EventHandler.htm) отчетов.


## Комментарии


Допустимые значения:


	- True. Обработчик используется;


	- False. Обработчик не
	 используется.
