# ISharedEventHandler


Сборка: Fore;


## Описание


Интерфейс ISharedEventHandler
 используется для работы с [обработчиком
 событий](UiExpress.chm::/setup_express_report/eventhandler.htm) отчетов.


## Иерархия наследования


           ISharedEventHandler


## Комментарии


Для работы с [обработчиком
 событий](UiExpress.chm::/setup_express_report/eventhandler.htm) отчетов необходимо создать специальный
 модуль, который будет обрабатывать события.


Для получения объекта данного типа используйте [ISharedEventHandlers.EventHandler](../ISharedEventHandlers/ISharedEventHandlers.EventHandler.htm).
