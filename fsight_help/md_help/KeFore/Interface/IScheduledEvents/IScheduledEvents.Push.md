# IScheduledEvents.Push


## Синтаксис


Push(Value: [IScheduledEvents](IScheduledEvents.htm));


## Параметры


Value - коллекция событий выполнения задач, которую необходимо добавить в данную коллекцию.


## Описание


Метод Push осуществляет добавление в текущую коллекцию событий, события передаваемые посредством параметра Value.
