# IScheduledEvents


## Описание


Интерфейс IScheduledEvents содержит
 свойства и методы для работы с коллекцией событий выполнения задач, которые
 запланированы на будущее.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IScheduledEvents.Count.htm)
		 Свойство Count возвращает
		 количество запланированных на будущее событий выполнения задач.


		 ![](../../Property_Image.gif)
		 [Item](IScheduledEvents.Item.htm)
		 Свойство Item возвращает
		 параметры события выполнения задачи.


## Свойства, унаследованные от [IScheduledInvoke](../IScheduledInvoke/IScheduledInvoke.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Value](../IScheduledInvoke/IScheduledInvoke.Value.htm)
		 Свойство Value возвращает
		 параметры события выполнения задачи, представленные в виде массива
		 значений.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Pop](IScheduledEvents.Pop.htm)
		 Метод Pop осуществляет
		 удаление из коллекции запланированных событий первого события
		 и возвращает его параметра.


		 ![](../../Sub_Image.gif)
		 [Push](IScheduledEvents.Push.htm)
		 Метод Push осуществляет
		 добавление в текущую коллекцию событий указанной коллекции событий.


## Методы, унаследованные от [IScheduledInvoke](../IScheduledInvoke/IScheduledInvoke.htm)


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Invoke](../IScheduledInvoke/IScheduledInvoke.Invoke.htm)
		 Метод Invoke вызывает
		 наступление события выполнения задачи.


		 ![](../../Sub_Image.gif)
		 [Next](../IScheduledInvoke/IScheduledInvoke.Next.htm)
		 Метод Next возвращает
		 параметры событий выполнения задач, которые запланированы на будущее.
