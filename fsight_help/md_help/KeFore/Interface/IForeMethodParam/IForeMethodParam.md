# IForeMethodParam


## Описание


Интерфейс IForeMethodParam содержит
 свойства для работы с параметром пользовательского метода.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DefaultValue](IForeMethodParam.DefaultValue.htm)
		 Свойство DefaultValue
		 определяет значение параметра по умолчанию.


		 ![](../../Property_Image.gif)
		 [Description](IForeMethodParam.Description.htm)
		 Свойство Description
		 определяет описание параметра.


		 ![](../../Property_Image.gif)
		 [Param](IForeMethodParam.Param.htm)
		 Свойство Param возвращает
		 наименование параметра.


		 ![](../../Property_Image.gif)
		 [Tag](IForeMethodParam.Tag.htm)
		 Свойство Tag не используется
		 компилятором. Можно определить и использовать свойство по своему
		 усмотрению.


		 ![](../../Property_Image.gif)
		 [Type](IForeMethodParam.Type.htm)
		 Свойство Type возвращает
		 тип данных параметра.
