# IForeMethodParam.Description


## Синтаксис


Description: String;


## Описание


Свойство Description определяет
 описание параметра.


## Комментарии


В качестве описания может быть указан любой текст, который содержит
 какие-либо пояснения относительно назначения и возможных значений параметра.
