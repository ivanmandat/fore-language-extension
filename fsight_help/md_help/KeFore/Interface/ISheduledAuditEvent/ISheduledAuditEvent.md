# ISheduledAuditEvent


Сборка: Fore;


## Описание


Интерфейс ISheduledAuditEvent
 предназначен для работы с системным событием, по наступлению которого
 будет выполнена задача.


## Иерархия наследования


           ISheduledAuditEvent


## Комментарии


Для настройки расписания выполнения задачи по наступлению системного
 события используйте интерфейс [IScheduledTaskAuditAlert](../IScheduledTaskAuditAlert/IScheduledTaskAuditAlert.htm).
