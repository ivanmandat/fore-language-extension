# ISheduledAuditEvent.Result


## Синтаксис


Result: [ScheduledAlertAuditResult](../../Enums/ScheduledAlertAuditResult.htm);


## Описание


Свойство Result определяет результат
 выполнения операции, при котором будет генерироваться событие.


## Комментарии


Для определения операции, при выполнении которой будет генерироваться
 событие, используйте свойство [ISheduledAuditEvent.Operation](ISheduledAuditEvent.Operation.htm).
