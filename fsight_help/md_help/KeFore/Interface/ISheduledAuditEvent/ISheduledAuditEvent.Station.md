# ISheduledAuditEvent.Station


## Синтаксис


Station: String;


## Описание


Свойство Station определяет
 рабочую станцию, на которой генерируется событие.


## Комментарии


Для определения пользователя репозитория, генерирующего событие, используйте
 свойство [ISheduledAuditEvent.UserName](ISheduledAuditEvent.UserName.htm).
