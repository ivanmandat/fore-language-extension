# IForeMethods.FindByKey


## Синтаксис


		FindByKey(Key: Integer): [IForeMethod](../IForeMethod/IForeMethod.htm);


## Параметры


Key. Ключ пользовательского
 метода.


## Описание


Метод FindByKey осуществляет
 поиск пользовательского метода в коллекции по его ключу.


## Комментарии


Метод возвращает параметры пользовательского метода в случае удачного
 поиска и Null, если пользовательский
 метод в коллекции не найден. Ключ пользовательского метода можно получить
 после [добавления](IForeMethods.Add.htm) метода в коллекцию.
