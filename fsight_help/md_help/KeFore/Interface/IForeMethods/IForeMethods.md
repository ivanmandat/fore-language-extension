# IForeMethods


Сборка: Fore;


## Описание


Интерфейс IForeMethods содержит
 свойства и методы для работы с коллекцией пользовательских методов.


## Иерархия наследования


IForeMethods


## Комментарии


Пользовательские методы, содержащиеся в данной коллекции, будут доступны
 для использования в:


	- [редакторе
	 выражения](uinav.chm::/GUI/ExpressionEditor.htm);


	- [методах
	 расчёта вычисляемых рядов](UiDw.chm::/Workbook/CalculatedSeries/UiDw_cs_UserMethods.htm) в инструменте «[Анализ
	 временных рядов](UiDw.chm::/UiDw_Title.htm)»;


	- [методах
	 расчёта моделей](UiModelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_2_Model/Specification/UiModelling_SpecificationUserMethod.htm) в инструменте «[Моделирование
	 и прогнозирование](UiModelling.chm::/UiModelling_TitlePage.htm)»;


	- [мастере
	 функций](UIReport.chm::/Desktop/organizational_management/UiReport_Organizational_master_function.htm) в инструменте «[Отчёты](UIReport.chm::/UiReport_purpose.htm)»;


	- [редакторе
	 формул](CalculationAlgorithm.chm::/Desktop/Calculation_block/Formula.htm) в инструменте «[Алгоритмы
	 расчёта](CalculationAlgorithm.chm::/Purpose.htm)».

## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IForeMethods.Add.htm)
		 Метод Add добавляет
		 пользовательский метод в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](IForeMethods.Clear.htm)
		 Метод Clear очищает
		 коллекцию пользовательских методов.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IForeMethods.FindByKey.htm)
		 Метод FindByKey осуществляет

		 ![](../../Sub_Image.gif)
		 [Remove](IForeMethods.Remove.htm)

		 пользовательский метод из коллекции по его индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IForeMethods.RemoveByKey.htm)
		 Метод RemoveByKey удаляет
		 пользовательский метод из коллекции по его ключу.
