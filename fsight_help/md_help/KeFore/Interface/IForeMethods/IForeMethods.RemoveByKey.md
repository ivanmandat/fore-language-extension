# IForeMethods.RemoveByKey


## Синтаксис


		RemoveByKey(Key: Integer);


## Параметры


Key. Ключ пользовательского
 метода.


## Описание


Метод RemoveByKey удаляет пользовательский
 метод из коллекции по его ключу.


## Комментарии


Ключ пользовательского метода можно получить после [добавления](IForeMethods.Add.htm)
 метода в коллекцию.
