# IScheduledInvoke


## Описание


Интерфейс IScheduledInvoke содержит свойства и методы для работы с событием выполнения задачи.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Value](IScheduledInvoke.Value.htm)
		 Свойство Value возвращает
		 параметры события выполнения задачи, представленные в виде массива
		 значений.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Invoke](IScheduledInvoke.Invoke.htm)
		 Метод Invoke вызывает
		 наступление события выполнения задачи.


		 ![](../../Sub_Image.gif)
		 [Next](IScheduledInvoke.Next.htm)
		 Метод Next возвращает
		 параметры событий выполнения задач, которые запланированы на будущее.
