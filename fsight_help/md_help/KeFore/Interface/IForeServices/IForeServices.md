# IForeServices


Сборка: Fore;


## Описание


Интерфейс IForeServices используется
 для получения доступа к среде выполнения Fore-кода.


## Иерархия наследования


IForeServices


## Комментарии


Для работы со средой выполнения Fore-кода приведите соединение с репозиторием,
 описываемое интерфейсом [IMetabase](KeSom.chm::/Interface/IMetabase/IMetabase.htm),
 к интерфейсу IForeServices. Среда
 выполнения позволяет получить доступ и работать с кодом, который реализован
 в Fore-модулях/формах/сборках репозитория.
