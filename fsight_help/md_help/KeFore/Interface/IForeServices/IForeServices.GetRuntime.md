# IForeServices.GetRuntime


## Синтаксис


		GetRuntime: [IForeRuntime](../IForeRuntime/IForeRuntime.htm);


## Описание


Метод GetRuntime возвращает
 контекст среды выполнения Fore-кода.


## Комментарии


Контекст среды выполнения позволяет работать с содержимым Fore-сборок
 репозитория. В среде выполнения можно загружать сборки, получать их метаданные
 и списки реализованных в сборках конструкций (списки классов, интерфейсов
 и других конструкций). В среде выполнения можно работать с системными
 сборками, объектом среды разработки [Сборка](UiDevEnv.chm::/01_Development_Environment/02_Work_in_Development_Environment/DevEnv_Object/DevEnv_Assembly.htm),
 а также с отдельными формами или модулями, которые реализованы вне сборок.
