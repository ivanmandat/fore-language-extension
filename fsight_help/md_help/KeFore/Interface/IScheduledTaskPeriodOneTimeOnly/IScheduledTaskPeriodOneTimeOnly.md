# IScheduledTaskPeriodOneTimeOnly


## Описание


Интерфейс IScheduledTaskPeriodOneTimeOnly
 содержит свойства и методы, определяющие параметры однократного расчета
 задачи.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [StartDateTime](IScheduledTaskPeriodOneTimeOnly.StartDateTime.htm)
		 Свойство StartDateTime
		 определяет время начала расчета задачи.


		 ![](../../Property_Image.gif)
		 [StartMode](IScheduledTaskPeriodOneTimeOnly.StartMode.htm)
		 Свойство StartMode
		 определяет, когда будет производиться одноразовый запуск задачи.


## Свойства, унаследованные от [IScheduledTaskPeriod](../IScheduledTaskPeriod/IScheduledTaskPeriod.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [StopDateTime](../IScheduledTaskPeriod/IScheduledTaskPeriod.StopDateTime.htm)
		 Свойство StopDateTime
		 определяет дату окончания действия периода, в который производится
		 расчет задачи.


		 ![](../../Property_Image.gif)
		 [Type](../IScheduledTaskPeriod/IScheduledTaskPeriod.Type.htm)
		 Свойство Type возвращает
		 способ задания периода.


## Методы, унаследованные от [IScheduledTaskPeriod](../IScheduledTaskPeriod/IScheduledTaskPeriod.htm)


		 Имя метода
		 Краткое описание

		 [Assign](../IScheduledTaskPeriod/IScheduledTaskPeriod.Assign.htm)

		 ![](../../Sub_Image.gif)
