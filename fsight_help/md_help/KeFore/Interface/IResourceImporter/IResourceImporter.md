# IResourceImporter


## Описание


Интерфейс IResourceImporter
 содержит свойства и методы, применяемые для [импорта
 ресурсов](UiNav.chm::/Multilanguage/Resources/Import_Resource.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Locales](IResourceImporter.Locales.htm)
		 Свойство Locales определяет
		 языки ресурсов для импорта.


		 ![](../../Property_Image.gif)
		 [Log](IResourceImporter.Log.htm)
		 Свойство Log возвращает
		 отчет о ходе импорта.


		 ![](../../Property_Image.gif)
		 [Method](IResourceImporter.Method.htm)
		 Свойство Method определяет
		 режим импорта ресурсов.


		 ![](../../Property_Image.gif)
		 [Provider](IResourceImporter.Provider.htm)
		 Свойство Provider определяет
		 объект, из которого будет произведен импорт.


		 ![](../../Property_Image.gif)
		 [Resource](IResourceImporter.Resource.htm)
		 Свойство Resource определяет
		 объект-ресурсы, в который должен быть произведен импорт.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Import](IResourceImporter.Import.htm)
		 Метод Import осуществляет
		 импорт ресурсов.
