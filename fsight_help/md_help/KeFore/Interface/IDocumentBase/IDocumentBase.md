# IDocumentBase


## Описание


Интерфейс IDocumentBase содержит
 базовые методы, используемые для загрузки и сохранения данных в [Документ](UiNavObj.chm::/UiNavObj_document.htm).


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [GetAsStream](IDocumentBase.GetAsStream.htm)
		 Метод GetAsStream создает
		 новый поток, связанный с данным документом.


		 ![](../../Sub_Image.gif)
		 [LoadFromFile](IDocumentBase.LoadFromFile.htm)
		 Метод LoadFromFile
		 осуществляет загрузку указанного файла в документ.


		 ![](../../Sub_Image.gif)
		 [LoadFromStream](IDocumentBase.LoadFromStream.htm)
		 Метод LoadFromStream
		 осуществляет загрузку содержимого потока в документ.


		 ![](../../Sub_Image.gif)
		 [SaveToFile](IDocumentBase.SaveToFile.htm)
		 Метод SaveToFile осуществляет
		 сохранение содержимого документа в файл на диске.


		 ![](../../Sub_Image.gif)
		 [SaveToStream](IDocumentBase.SaveToStream.htm)
		 Метод SaveToStream
		 осуществляет сохранение содержимого документа в поток.
