# IDocumentBase.LoadFromStream


## Синтаксис


LoadFromStream(Stream: [IIOStream](ModIo.chm::/Interface/IIOStream/IIOStream.htm));


## Параметры


Stream. Поток, из которого
 осуществляется загрузка в документ.


## Описание


Метод LoadFromStream осуществляет
 загрузку содержимого потока, передаваемого посредством входного параметра,
 в документ.
