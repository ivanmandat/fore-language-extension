# IBaseMethod.Method


## Синтаксис


		Method: String;


## Описание


Свойство Method определяет уникальный
 идентификатор пользовательского метода.


## Комментарии


Данный идентификатор должен совпадать с идентификатором функции/процедуры,
 содержащейся в модуле, указанном в свойстве [Assembly](IBaseMethod.Assembly.htm).
 Функция/процедура должна быть объявлена с модификатором доступа Public.
