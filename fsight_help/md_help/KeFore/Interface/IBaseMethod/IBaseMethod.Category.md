# IBaseMethod.Category


## Синтаксис


		Category: String;


## Описание


Свойство Category определяет
 категорию пользовательского метода.


## Комментарии


Указываемая категория будет использоваться для группировки методов в
 пользовательских метаданных. При работе с инструментами платформы все
 методы будут группироваться в одну категорию - Пользовательские функции.
