# IBaseMethod


Сборка: Fore;


## Описание


Интерфейс IBaseMethod предоставляет
 базовые настройки пользовательских методов.


## Иерархия наследования


IBaseMethod


## Комментарии


Интерфейс является базовым для интерфейса [IForeMethod](../IForeMethod/IForeMethod.htm).
