# IBaseMethod.Name


## Синтаксис


		Name: String;


## Описание


Свойство Name определяет наименование
 пользовательского метода.


## Комментарии


Под данным наименованием метод будет отображаться в контейнере моделирования
 и базе данных временных рядов.
