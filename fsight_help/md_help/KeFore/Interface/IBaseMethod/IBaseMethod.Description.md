# IBaseMethod.Description


## Синтаксис


		Description: String;


## Описание


Свойство Description определяет
 описание пользовательского метода.


## Комментарии


В описании может быть указана любая информация, связанная с пользовательским
 методом, например, назначение метода, описание его параметров и возвращаемый
 результат. Описание будет отображаться рядом с сигнатурой метода при его
 выборе в [редакторе
 выражения](UiNav.chm::/GUI/ExpressionEditor.htm).
