# IForeAssemblyBinary.CheckCompiled


## Синтаксис


		CheckCompiled: Boolean;


## Описание


Метод CheckCompiled осуществляет
 проверку компиляции сборки.


## Комментарии


Метод возвращает значение True,
 если компиляция сборки завершилась успешно и False,
 если в сборке имеются ошибки.
