# IForeAssemblyBinary.ObjectName


## Синтаксис


		ObjectName: String;


## Описание


Свойство ObjectName возвращает
 наименование объекта, являющегося сборкой.


## Комментарии


Для системных сборок свойство вернёт путь и наименование библиотеки,
 в которой находится реализация сборки. Для пользовательских сборок репозитория
 свойство вернёт идентификатор и ключ.
