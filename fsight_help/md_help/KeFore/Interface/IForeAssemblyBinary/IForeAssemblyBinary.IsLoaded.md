# IForeAssemblyBinary.IsLoaded


## Синтаксис


		IsLoaded: Boolean;


## Описание


Свойство IsLoaded возвращает
 признак загрузки сборки в среду выполнения.


## Комментарии


Свойство возвращает значение True,
 если сборка загружена в среду выполнения и False,
 если не загружена.
