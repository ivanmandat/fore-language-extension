# IForeAssemblyBinary


Сборка: Fore;


## Описание


Интерфейс IForeAssemblyBinary
 предназначен для работы с метаданными сборки.


## Иерархия наследования


IForeAssemblyBinary


## Комментарии


Метаданные сборки можно получить, используя метод [IForeRuntime.ResolveAssembly](../IForeRuntime/IForeRuntime.ResolveAssembly.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Builtin](IForeAssemblyBinary.Builtin.htm)
		 Свойство Builtin возвращает
		 признак соответствия системной сборке среды разработки.


		 ![](../../Property_Image.gif)
		 [HiddenReferences](IForeAssemblyBinary.HiddenReferences.htm)
		 Свойство HiddenReferences
		 возвращает массив скрытых ссылок.


		 ![](../../Property_Image.gif)
		 [IsLoaded](IForeAssemblyBinary.IsLoaded.htm)
		 Свойство IsLoaded возвращает
		 признак загрузки сборки в среду выполнения.


		 ![](../../Property_Image.gif)
		 [Name](IForeAssemblyBinary.Name.htm)
		 Свойство Name возвращает
		 наименование сборки.


		 ![](../../Property_Image.gif)
		 [Namespace_](IForeAssemblyBinary.Namespace_.htm)
		 Свойство Namespace_
		 возвращает идентификатор контейнера, в рамках которого реализована
		 сборка.


		 ![](../../Property_Image.gif)
		 [ObjectName](IForeAssemblyBinary.ObjectName.htm)
		 Свойство ObjectName
		 возвращает наименование объекта, являющегося сборкой.


		 ![](../../Property_Image.gif)
		 [References](IForeAssemblyBinary.References.htm)
		 Свойство References
		 возвращает список ссылок на другие сборки.


		 ![](../../Property_Image.gif)
		 [TimeStamp](IForeAssemblyBinary.TimeStamp.htm)
		 Свойство TimeStamp
		 возвращает дату и время последнего изменения сборки.


		 ![](../../Property_Image.gif)
		 [Version](IForeAssemblyBinary.Version.htm)
		 Свойство Version возвращает
		 внутреннюю версию сборки «Форсайт. Аналитическая платформа»,
		 в которой производится работа со сборкой среды разработки.
