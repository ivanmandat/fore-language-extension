# IForeAssemblyBinary.Namespace_


## Синтаксис


		Namespace_: String;


## Описание


Свойство Namespace_ возвращает
 идентификатор контейнера, в рамках которого реализована сборка.


## Комментарии


Свойство актуально для сборок, которые реализованы в рамках какого-либо
 объекта репозитория - [Контейнер](UiNavObj.chm::/Container/Container.htm).


Для всех других сборок свойство Namespace_
 возвращает пустое значение.
