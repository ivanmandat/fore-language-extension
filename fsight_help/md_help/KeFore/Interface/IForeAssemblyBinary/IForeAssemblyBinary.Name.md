# IForeAssemblyBinary.Name


## Синтаксис


		Name: String;


## Описание


Свойство Name возвращает наименование
 сборки.


## Комментарии


Свойство возвращает наименование системной сборки или идентификатор
 сборки репозитория, которые были указаны в методе [IForeRuntime.ResolveAssembly](../IForeRuntime/IForeRuntime.ResolveAssembly.htm).
