# ISharedPlugins.AdhocPlugins


## Синтаксис


AdhocPlugins: [ISharedAdhocPlugins](../ISharedAdhocPlugins/ISharedAdhocPlugins.htm);


## Описание


Свойство AdhocPlugins
 возвращает коллекцию плагинов, подключенных в инструменте «Аналитические панели».


## Комментарии


В инструменте «Аналитические панели»
 отсутствуют плагины, подключенные по умолчанию.
