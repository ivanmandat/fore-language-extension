# ISharedPlugins


Сборка: Fore;


## Описание


Интерфейс ISharedPlugins предназначен
 для работы с плагинами.


## Иерархия наследования


           ISharedPlugins


## Комментарии


Данный интерфейс позволяет настроить общие параметры плагинов. Для настройки
 параметров конкретного плагина используйте интерфейс [ISharedAdhocPlugin](../ISharedAdhocPlugin/ISharedAdhocPlugin.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AdhocPlugins](ISharedPlugins.AdhocPlugins.htm)
		 Свойство AdhocPlugins
		 возвращает коллекцию плагинов, подключенных в инструменте «Аналитические панели».


		 ![](../../Property_Image.gif)
		 [LargeImages](ISharedPlugins.LargeImages.htm)
		 Свойство LargeImages
		 возвращает коллекцию больших изображений, используемых плагинами
		 в качестве пиктограмм.


		 ![](../../Property_Image.gif)
		 [Resource](ISharedPlugins.Resource.htm)
		 Свойство Resource определяет
		 объект «Ресурсы», используемый плагинами.


		 ![](../../Property_Image.gif)
		 [SmallImages](ISharedPlugins.SmallImages.htm)
		 Свойство SmallImages
		 возвращает коллекцию маленьких изображений, используемых
		 плагинами в качестве пиктограмм.
