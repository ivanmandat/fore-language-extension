# ISharedPlugins.LargeImages


## Синтаксис


LargeImages: [IGxImageList](ModDrawing.chm::/Interface/IGxImageList/IGxImageList.htm);


## Описание


Свойство LargeImages возвращает
 коллекцию больших изображений, используемых плагинами в качестве пиктограмм.


## Комментарии


Для корректного хранения коллекции изображений должен быть задан объект
 «Ресурсы», используемый плагинами. Для его задания используйте свойство
 [ISharedPlugins.Resource](ISharedPlugins.Resource.htm).


Для получения коллекции маленьких изображений используйте свойство [ISharedPlugins.SmallImages](ISharedPlugins.SmallImages.htm).
