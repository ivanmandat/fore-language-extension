# ISharedPlugins.Resource


## Синтаксис


Resource: [IResourceObject](ModIo.chm::/Interface/IResourceObject/IResourceObject.htm);


## Описание


Свойство Resource определяет
 объект «Ресурсы», используемый плагинами.


## Комментарии


Объект «Ресурсы» используется для хранения коллекций изображений, используемых
 плагинами в качестве пиктограмм, и локализованных имен плагинов.
