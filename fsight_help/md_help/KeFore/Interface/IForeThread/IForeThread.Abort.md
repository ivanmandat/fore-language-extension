# IForeThread.Abort


## Синтаксис


		Abort;


## Описание


Метод Abort завершает выполнение
 потока.


## Комментарии


При вызове метода Abort останавливается
 пользовательский метод, который выполнялся в потоке. Необходимо быть осторожным
 с вызовом метода Abort, так как
 при этом могут быть утеряны данные, с которыми осуществлялась работа в
 потоке.
