# IForeThread.Join


## Синтаксис


		Join;


## Описание


Метод Join останавливает выполнение
 кода до завершения потока.


## Комментарии


Вызов метода Join останавливает
 выполнение потока приложения, в котором был создан и запущен текущий поток.
 После завершения выполнения потока управление возвращается к следующему
 коду приложения.
