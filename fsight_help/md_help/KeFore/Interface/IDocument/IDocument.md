# IDocument


## Описание


Интерфейс IDocument содержит
 свойства и методы объекта репозитория - [Документ](UiNavObj.chm::/UiNavObj_document.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CreationDate](IDocument.CreationDate.htm)
		 Свойство CreationDate
		 определяет дату создания файла, хранящегося в документе.


		 ![](../../Property_Image.gif)
		 [FileNam](IDocument.FileName.htm)[e](IDocument.CreationDate.htm)
		 Свойство FileName определяет
		 путь и наименование файла, хранящегося в документе.


		 ![](../../Property_Image.gif)
		 [MimeType](IDocument.MimeType.htm)
		 Свойство MimeType определяет
		 тип содержимого, хранящегося в документе.


		 ![](../../Property_Image.gif)
		 [ModificDate](IDocument.ModificDate.htm)
		 Свойство ModificDate
		 определяет дату изменения файла, хранящегося в документе.


		 ![](../../Property_Image.gif)
		 [Size](IDocument.Size.htm)
		 Свойство Size возвращает
		 размер файла, хранящегося в документе.


## Методы, унаследованные от [IDocumentBase](../IDocumentBase/IDocumentBase.htm)


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [GetAsStream](../IDocumentBase/IDocumentBase.GetAsStream.htm)
		 Метод GetAsStream создает
		 новый поток, связанный с данным документом.


		 ![](../../Sub_Image.gif)
		 [LoadFromFile](../IDocumentBase/IDocumentBase.LoadFromFile.htm)
		 Метод LoadFromFile
		 осуществляет загрузку указанного файла в документ.


		 ![](../../Sub_Image.gif)
		 [LoadFromStream](../IDocumentBase/IDocumentBase.LoadFromStream.htm)
		 Метод LoadFromStream
		 осуществляет загрузку содержимого потока в документ.


		 ![](../../Sub_Image.gif)
		 [SaveToFile](../IDocumentBase/IDocumentBase.SaveToFile.htm)
		 Метод SaveToFile осуществляет
		 сохранение содержимого документа в файл на диске.


		 ![](../../Sub_Image.gif)
		 [SaveToStream](../IDocumentBase/IDocumentBase.SaveToStream.htm)
		 Метод SaveToStream
		 осуществляет сохранение содержимого документа в поток.
