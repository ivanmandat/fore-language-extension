# IDocument.Size


## Синтаксис


Size: Variant;


## Описание


Свойство Size возвращает размер файла, хранящегося в документе.


## Комментарии


При создании документа в данном свойстве будет установлен размер загружаемого файла в байтах. При загрузке новых версий файла значение свойства обновляется автоматически.
