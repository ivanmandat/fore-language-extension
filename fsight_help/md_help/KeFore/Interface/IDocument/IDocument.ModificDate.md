# IDocument.ModificDate


## Синтаксис


ModificDate: DateTime;


## Описание


Свойство ModificDate определяет дату изменения файла, хранящегося в документе.


## Комментарии


При загрузке документа из файла в данном свойстве будет установлена дата последнего изменения загружаемого файла. При загрузке документа из потока в свойстве ModificDate будет установлена нулевая дата и время. При загрузке новых версий файла значение свойства обновляется автоматически.
