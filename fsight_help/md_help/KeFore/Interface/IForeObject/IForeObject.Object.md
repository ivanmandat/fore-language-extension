# IForeObject.Object


## Синтаксис


		Object: Variant;


## Описание


Свойство Object возвращает экземпляр
 класса, приведённый к общему типу данных.


## Комментарии


Экземпляр класса, полученный с помощью свойства Object,
 может использоваться для указания экземпляра класса в качестве значения
 параметров в свойстве [IForeSubParam.Value](../IForeSubParam/IForeSubParam.Value.htm).


## Пример


Пример использования приведён в описании метода [IForeObject.SetFieldValue](IForeObject.SetFieldValue.htm).
