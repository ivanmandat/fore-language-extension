# IForeObject


Сборка: Fore;


## Описание


Интерфейс IForeObject предназначен
 для работы с экземпляром Fore-класса.


## Иерархия наследования


IForeObject


## Комментарии


Для создания экземпляра класса предназначен метод [IForeClass.CreateObject](../IForeClass/IForeClass.CreateObject.htm).
 Экземпляр класса необходимо указывать в свойстве [IForeSub.Self](../IForeSub/IForeSub.Self.htm),
 если предполагается выполнение метода класса.


## Свойства


		 Имя свойства
		 Краткое описание

		 [ClassName](IForeObject.ClassName.htm)

		 [Object](IForeObject.Object.htm)
		 Свойство Object возвращает
