# IForeObject.ClassName


## Синтаксис


		ClassName: String;


## Описание


Свойство ClassName возвращает
 наименование класса, которому соответствует текущий экземпляр.


## Комментарии


Для создания экземпляра класса используется метод [IForeClass.CreateObject](../IForeClass/IForeClass.CreateObject.htm).
