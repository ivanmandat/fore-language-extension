# IScheduledEvent


## Описание


Интерфейс IScheduledEvent содержит свойства и методы для работы с событием выполнения задачи, запланированным на будущее.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [IsLast](IScheduledEvent.IsLast.htm)
		 Свойство не реализовано.


		 ![](../../Property_Image.gif)
		 [Period](IScheduledEvent.Period.htm)
		 Свойство Period возвращает
		 параметры периода расчета задачи, для которой запланировано данное
		 событие.


		 ![](../../Property_Image.gif)
		 [StartDateTime](IScheduledEvent.StartDateTime.htm)
		 Свойство StartDateTime
		 возвращает дату и время, на которое запланирован запуск события.


		 ![](../../Property_Image.gif)
		 [TaskId](IScheduledEvent.TaskId.htm)
		 Свойство TaskId возвращает
		 идентификатор задачи, для выполнения которой запланировано данное
		 событие.


		 ![](../../Property_Image.gif)
		 [TaskKey](IScheduledEvent.TaskKey.htm)
		 Свойство TaskKey возвращает
		 ключ задачи, для выполнения которой запланировано данное событие.
