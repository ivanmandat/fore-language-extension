# IScheduledEvent.Bind


## Синтаксис


Bind(Mb: [IMetabase](KeSom.chm::/Interface/IMetabase/IMetabase.htm)): [IScheduledTask](../IScheduledTask/IScheduledTask.htm);


## Параметры


Mb - репозиторий, в рамках которого осуществляется выполнения задачи.


## Описание


Свойство Bind возвращает параметры задачи, для выполнения которой запланировано данное событие.
