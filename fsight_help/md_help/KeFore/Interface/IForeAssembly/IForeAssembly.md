# IForeAssembly


Сборка: Fore;


## Описание


Интерфейс IForeAssembly предназначен
 для работы с конструкциями сборки.


## Иерархия наследования


IForeAssembly


## Комментарии


Получить доступ к текущем интерфейсу для дальнейшей работы с конструкциям
 сборки можно с помощью метода [IForeRuntime.BindToAssembly](../IForeRuntime/IForeRuntime.BindToAssembly.htm).


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [BindToClass](IForeAssembly.BindToClass.htm)
		 Метод BindToClass возвращает
		 информацию о классе/интерфейсе/перечислении с указанным именем.


		 ![](../../Sub_Image.gif)
		 [BindToSub](IForeAssembly.BindToSub.htm)
		 Метод BindToSub возвращает
		 информацию о процедуре/функции с указанным именем.


		 ![](../../Sub_Image.gif)
		 [ClassesCount](IForeAssembly.ClassesCount.htm)
		 Метод ClassesCount
		 возвращает количество классов/интерфейсов/перечислений, реализованных
		 в сборке.


		 ![](../../Sub_Image.gif)
		 [ClassItem](IForeAssembly.ClassItem.htm)
		 Метод ClassItem возвращает
		 информацию о конструкции сборки с указанным индексом.


		 ![](../../Sub_Image.gif)
		 [SubItem](IForeAssembly.SubItem.htm)
		 Метод SubItem возвращает
		 информацию о процедуре/функции с указанным индексом.


		 ![](../../Sub_Image.gif)
		 [SubsCount](IForeAssembly.SubsCount.htm)
		 Метод SubsCount возвращает
		 количество процедур/функций, реализованных в глобальном пространстве
		 имён сборки.
