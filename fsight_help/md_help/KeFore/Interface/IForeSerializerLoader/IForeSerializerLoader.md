# IForeSerializerLoader


Сборка: Fore;


## Описание


Интерфейс IForeSerializerLoader содержит свойства и методы для загрузки сборок/модулей/форм среды разработки.


## Иерархия наследования


IForeSerializerLoader


## Свойства


 Имя свойства
 Краткое описание


 ![](../../Property_Image.gif)
 [CreateInfo](IForeSerializerLoader.CreateInfo.htm)
 Свойство CreateInfo возвращает информацию о загружаемом объекте.


 ![](../../Property_Image.gif)
 [Metabase](IForeSerializerLoader.Metabase.htm)
 Свойство Metabase определяет репозиторий, в рамках которого осуществляется загрузка сборок/форм/модулей из файла.
