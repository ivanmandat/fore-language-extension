## Синтаксис

## Описание


Свойство TaskChecker определяет
 условие выполнения задачи.


## Комментарии


Для создания нового условия выполнения задачи контейнера запланированных
 задач используйте метод [IScheduledTask.CreateChecker](IScheduledTask.CreateChecker.htm).
