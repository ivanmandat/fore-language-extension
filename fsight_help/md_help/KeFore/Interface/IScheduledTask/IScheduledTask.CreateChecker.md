# IScheduledTask.CreateChecker


## Синтаксис


CreateChecker(Type: [ScheduledTaskCheckerType](../../Enums/ScheduledTaskCheckerType.htm)):
 [IScheduledTaskChecker](../IScheduledTaskChecker/IScheduledTaskChecker.htm);


## Параметры


Type. Тип условия выполнения
 задачи.


## Описание


Метод CreateChecker создает
 новое условие выполнения задачи контейнера запланированных задач.


## Комментарии


Для определения условия выполнения задачи используйте свойство [IScheduledTask.TaskChecker](IScheduledTask.TaskChecker.htm).
