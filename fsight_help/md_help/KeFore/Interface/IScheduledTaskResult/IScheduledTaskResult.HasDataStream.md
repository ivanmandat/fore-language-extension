# IScheduledTaskResult.HasDataStream


## Синтаксис


HasDataStream: Boolean;


## Описание


Свойство HasDataStream возвращает признак наличия непустого потока данных, который можно прочитать с помощью метода [ReadDataStream](IScheduledTaskResult.ReadDataStream.htm).
