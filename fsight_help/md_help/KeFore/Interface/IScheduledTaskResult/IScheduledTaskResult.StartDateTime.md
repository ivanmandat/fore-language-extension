# IScheduledTaskResult.StartDateTime


## Синтаксис


StartDateTime: DateTime;


## Описание


Свойство StartDateTime возвращает дату и время запуска задачи на выполнение.


## Комментарии


Для получения даты и времени завершения выполнения задачи используйте свойство [IScheduledTaskResult.FinishDateTime](IScheduledTaskResult.FinishDateTime.htm).
