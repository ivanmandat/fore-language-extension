# IScheduledTaskResult.FinishDateTime


## Синтаксис


FinishDateTime: DateTime;


## Описание


Свойство FinishDateTime возвращает дату и время завершения выполнения задачи.


## Комментарии


Для получения даты и времени запуска задачи на выполнение используйте свойство [IScheduledTaskResult.StartDateTime](IScheduledTaskResult.StartDateTime.htm).
