# IForeSubParam.Value


## Синтаксис


		Value: Variant;


## Описание


Свойство Value определяет значение
 параметра.


## Комментарии


Указываемое значение должно соответствовать [типу
 значения](IForeSubParam.ParamType.htm), который может принимать параметр.
