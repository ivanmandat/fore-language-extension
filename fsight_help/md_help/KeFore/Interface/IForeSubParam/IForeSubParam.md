# IForeSubParam


Сборка: Fore;


## Описание


Интерфейс IForeSubParam предназначен
 для работы с параметром процедуры/функции/конструктора.


## Иерархия наследования


IForeSubParam


## Комментарии


Доступ к параметру предоставляют методы [IForeSubParams.Item](../IForeSubParams/IForeSubParams.Item.htm),
 [IForeSubParams.ParamByName](../IForeSubParams/IForeSubParams.ParamByName.htm).
