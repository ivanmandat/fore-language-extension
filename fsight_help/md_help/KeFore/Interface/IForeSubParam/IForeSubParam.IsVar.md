# IForeSubParam.IsVar


## Синтаксис


		IsVar: Boolean;


## Описание


Свойство IsVar возвращает способ
 передачи параметра.


## Комментарии


Свойство возвращает значение True,
 если параметр передаётся по ссылке, и False,
 если параметр передаётся по значению
