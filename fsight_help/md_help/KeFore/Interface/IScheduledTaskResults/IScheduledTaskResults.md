# IScheduledTaskResults


## Описание


Интерфейс IScheduledTaskResults
 содержит свойства и методы для работы с коллекцией записей в истории выполнения
 задачи.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IScheduledTaskResults.Count.htm)
		 Свойство Count возвращает
		 количество записей в истории выполнения задачи.


		 ![](../../Property_Image.gif)
		 [Item](IScheduledTaskResults.Item.htm)
		 Свойство Item возвращает
		 запись в истории выполнения задачи.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindByKey](IScheduledTaskResults.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск записи в истории задачи по ключу.


		 ![](../../Sub_Image.gif)
		 [FindCurrent](IScheduledTaskResults.FindCurrent.htm)
		 Метод FindCurrent осуществляет
		 поиск записи в истории выполнения задачи на указанную дату и время.
