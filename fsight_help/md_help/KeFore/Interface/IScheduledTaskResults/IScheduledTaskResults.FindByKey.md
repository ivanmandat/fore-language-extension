# IScheduledTaskResults.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IScheduledTaskResult](../IScheduledTaskResult/IScheduledTaskResult.htm);


## Параметры


Key - ключ записи в истории выполнения задачи.


## Описание


Метод FindByKey осуществляет поиск записи в истории задачи. Ключ записи, по которому осуществляется поиск, передается посредством параметра Key.
