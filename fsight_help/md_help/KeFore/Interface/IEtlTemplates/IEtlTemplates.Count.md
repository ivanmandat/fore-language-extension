# IEtlTemplates.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает
 количество пользовательских шаблонов источников/приёмников задачи ETL.


## Комментарии


Для получения пользовательского шаблона источника/приёмника задачи ETL
 по его индексу используйте [IEtlTemplates.Item](IEtlTemplates.Item.htm).
