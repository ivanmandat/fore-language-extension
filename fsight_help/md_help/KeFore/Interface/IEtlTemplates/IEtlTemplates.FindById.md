# IEtlTemplates.FindById


## Синтаксис


FindById(Id: String): [IEtlTemplate](../IEtlTemplate/IEtlTemplate.htm);


## Параметры


Id. Идентификатор пользовательского
 шаблона объекта задачи ETL.


## Описание


Метод FindById осуществляет
 поиск пользовательского шаблона источника/приёмника задачи ETL по его
 идентификатору.


## Комментарии


Для поиска пользовательского шаблона источника/приёмника задачи ETL
 по его ключу используйте [IEtlTemplates.FindByKey](IEtlTemplates.FindByKey.htm).
