# IEtlTemplates


Сборка: Fore;


## Описание


Интерфейс IEtlTemplates используется
 для работы с коллекцией пользовательских шаблонов источников/приёмников
 задачи ETL.


## Иерархия наследования


           IEtlTemplates


## Комментарии


Для получения объекта данного типа используйте [ISharedParams.EtlTemplates](../ISharedParams/ISharedParams.EtlTemplates.htm).


Для работы с пользовательским шаблоном источника/приёмника задачи ETL
 используйте интерфейс [IEtlTemplate](../IEtlTemplate/IEtlTemplate.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IEtlTemplates.Count.htm)
		 Свойство Count возвращает
		 количество пользовательских шаблонов источников/приёмников задачи
		 ETL.


		 ![](../../Property_Image.gif)
		 [Item](IEtlTemplates.Item.htm)
		 Свойство Item возвращает
		 пользовательский шаблон источника/приёмника задачи ETL по его
		 индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IEtlTemplates.Add.htm)
		 Метод Add добавляет
		 новый пользовательский шаблон источника/приёмника задачи ETL в
		 коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](IEtlTemplates.Clear.htm)
		 Метод Clear очищает
		 коллекцию пользовательских шаблонов источников/приёмников задачи
		 ETL.


		 ![](../../Sub_Image.gif)
		 [FindById](IEtlTemplates.FindById.htm)
		 Метод FindById осуществляет
		 поиск пользовательского шаблона источника/приёмника задачи ETL
		 по его идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IEtlTemplates.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск пользовательского шаблона источника/приёмника задачи ETL
		 по его ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](IEtlTemplates.Remove.htm)
		 Метод Remove удаляет
		 пользовательский шаблон источника/приёмника задачи ETL по его
		 индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveById](IEtlTemplates.RemoveById.htm)
		 Метод RemoveById удаляет
		 пользовательский шаблон источника/приёмника задачи ETL по его
		 идентификатору.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IEtlTemplates.RemoveByKey.htm)
		 Метод RemoveByKey удаляет
		 пользовательский шаблон источника/приёмника задачи ETL по его
		 ключу.
