# IEtlTemplates.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс пользовательского
 шаблона источника/приёмника задачи ETL.


## Описание


Метод Remove удаляет пользовательский
 шаблон источника/приёмника задачи ETL по его индексу.


## Комментарии


Если пользовательский шаблон будет удален, то метод вернет значение
 True.


Для удаления пользовательского шаблона источника/приёмника задачи ETL
 по его идентификатору используйте [IEtlTemplates.RemoveById](IEtlTemplates.RemoveById.htm).


Для удаления пользовательского шаблона источника/приёмника задачи ETL
 по его ключу используйте [IEtlTemplates.RemoveByKey](IEtlTemplates.RemoveByKey.htm).


Для добавления нового пользовательского источника/приёмника объекта
 задачи ETL в коллекцию используйте [IEtlTemplates.Add](IEtlTemplates.Add.htm).
