# IEtlTemplates.FindByKey


## Синтаксис


RemoveByKey(Key: Integer): [IEtlTemplate](../IEtlTemplate/IEtlTemplate.htm);


## Параметры


Key. Ключ пользовательского
 шаблона объекта задачи ETL.


## Описание


Метод FindByKey осуществляет
 поиск пользовательского шаблона источника/приёмника задачи ETL по его
 ключу.


## Комментарии


Для поиска пользовательского шаблона источника/приёмника задачи ETL
 по его идентификатору используйте [IEtlTemplates.FindById](IEtlTemplates.FindById.htm).
