# IEtlTemplates.Add


## Синтаксис


Add: [IEtlTemplate](../IEtlTemplate/IEtlTemplate.htm);


## Описание


Метод Add добавляет новый пользовательский
 шаблон источника/приёмника задачи ETL в коллекцию.


## Комментарии


Для очищения коллекции пользовательских шаблонов источников/приёмников
 задачи ETL используйте [IEtlTemplates.Clear](IEtlTemplates.Clear.htm).
