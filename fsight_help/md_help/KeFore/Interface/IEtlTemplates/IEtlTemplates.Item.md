# IEtlTemplates.Item


## Синтаксис


Item(Index: Integer): [IEtlTemplate](../IEtlTemplate/IEtlTemplate.htm);


## Параметры


Index. Индекс пользовательского
 шаблона объекта задачи ETL.


## Описание


Свойство Item возвращает пользовательский
 шаблон источника/приёмника задачи ETL по его индексу.


## Комментарии


Для получения количества пользовательских шаблонов источников/приёмников
 задачи ETL используйте [IEtlTemplates.Count](IEtlTemplates.Count.htm).
