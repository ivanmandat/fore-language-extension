# IEtlTemplates.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ пользовательского
 шаблона объекта задачи ETL.


## Описание


Метод RemoveByKey удаляет
 пользовательский шаблон источника/приёмника задачи ETL по его ключу.


## Комментарии


Если пользовательский шаблон будет удален, то метод вернет значение
 True.


Для удаления пользовательского шаблона источника/приёмника задачи ETL
 по его идентификатору используйте [IEtlTemplates.RemoveById](IEtlTemplates.RemoveById.htm)
 .


Для удаления пользовательского шаблона источника/приёмника задачи ETL
 по его индексу используйте [IEtlTemplates.Remove](IEtlTemplates.Remove.htm).


Для добавления нового пользовательского шаблона источника/приёмника
 задачи ETL в коллекцию используйте [IEtlTemplates.Add](IEtlTemplates.Add.htm).
