# IScheduledTaskModuleChecker.Macro


## Синтаксис


Macro: String;


## Описание


Свойство Macro определяет
 функцию, определённую в сборке, модуле или форме.


## Комментарии


[Сборка, модуль или
 форма](IScheduledTaskModuleChecker.Module.htm) должны содержать функции с сигнатурой Function
 <name> (<param>: IScheduledTask): Boolean.


Для определения ожидаемого результата выполнения функции используйте
 свойство [IScheduledTaskModuleChecker.ExpectedResult](IScheduledTaskModuleChecker.ExpectedResult.htm).
