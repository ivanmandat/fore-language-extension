# IScheduledTaskModuleChecker.ExpectedResult


## Синтаксис


ExpectedResult: Boolean;


## Описание


Свойство ExpectedResult определяет
 ожидаемый результат выполнения функции.


## Комментарии


Допустимые значения:


	- True. Результат выполнения
	 функции «истина»;


	- False. Результат выполнения
	 функции «ложь».
