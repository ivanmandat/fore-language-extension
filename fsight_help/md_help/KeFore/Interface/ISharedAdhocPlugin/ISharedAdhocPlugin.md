# ISharedAdhocPlugin


Сборка: Fore;


## Описание


Интерфейс ISharedAdhocPlugin
 предназначен для работы с плагином в инструменте «Аналитические
 панели».


## Иерархия наследования


           [INamedEntity](KeSom.chm::/Interface/INamedEntity/INamedEntity.htm)


           ISharedAdhocPlugin


## Комментарии


Для работы с коллекцией плагинов инструмента «Аналитические
 панели» используйте интерфейс [ISharedAdhocPlugins](../ISharedAdhocPlugins/ISharedAdhocPlugins.htm).
