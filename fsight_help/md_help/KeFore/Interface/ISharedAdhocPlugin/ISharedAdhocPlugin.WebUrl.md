# ISharedAdhocPlugin.WebUrl


## Синтаксис


WebUrl: String;


## Описание


Свойство WebUrl определяет js-скрипт
 плагина, который будет использоваться в веб-приложении.


## Комментарии


Скрипт должен содержаться в папке «Build»,
 расположенной в папке с установленным веб-приложением «Форсайт. Аналитическая платформа».
