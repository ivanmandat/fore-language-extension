# ISharedAdhocPlugin.MasterForm


## Синтаксис


MasterForm: [IMetabaseObjectDescriptor](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm);


## Описание


Свойство MasterForm определяет
 форму-мастер для плагина.


## Комментарии


Форма-мастер содержит компоненты, образующие вкладки боковой панели
 для настройки плагина, и код, описывающий логику применения настроек к
 плагину. Данная форма должна наследоваться от класса [AdhocUserMasterForm](KeAdhoc.chm::/Class/AdhocUserMasterForm/AdhocUserMasterForm.htm).
