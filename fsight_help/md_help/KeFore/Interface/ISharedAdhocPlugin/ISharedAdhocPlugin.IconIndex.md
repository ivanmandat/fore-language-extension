# ISharedAdhocPlugin.IconIndex


## Синтаксис


IconIndex: Integer;


## Описание


Свойство IconIndex определяет
 индекс пиктограммы плагина.


## Комментарии


Для работы с коллекцией больших пиктограмм используйте свойство [ISharedPlugins.LargeImages](../ISharedPlugins/ISharedPlugins.LargeImages.htm),
 с коллекцией маленьких - [ISharedPlugins.SmallImages](../ISharedPlugins/ISharedPlugins.SmallImages.htm).
