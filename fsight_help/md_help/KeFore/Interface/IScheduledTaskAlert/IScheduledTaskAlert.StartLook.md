# IScheduledTaskAlert.StartLook


## Синтаксис


StartLook;


## Описание


Метод StartLook устанавливает
 дату начала наступления события.


## Комментарии


В качестве даты начала берётся текущая дата минус месяц.


Для установки даты окончания наступления события используйте метод [IScheduledTaskAlert.StopLook](IScheduledTaskAlert.StopLook.htm).
