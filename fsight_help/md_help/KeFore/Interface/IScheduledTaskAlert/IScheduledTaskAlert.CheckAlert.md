# IScheduledTaskAlert.CheckAlert


## Синтаксис


CheckAlert: Boolean;


## Описание


Метод CheckAlert выполняет проверку
 выполнения события.


## Комментарии


Допустимые значения:


	- True. Событие выполнено;


	- False. Событие не выполнено.
