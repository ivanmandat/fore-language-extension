# IScheduledTaskAlert.StopLook


## Синтаксис


StopLook;


## Описание


Метод StopLook устанавливает
 дату окончания наступления события.


## Комментарии


В качестве даты окончания берётся текущая дата.


Для установки даты начала наступления события используйте метод [IScheduledTaskAlert.StartLook](IScheduledTaskAlert.StartLook.htm).
