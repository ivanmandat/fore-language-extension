# IForeSubParams


Сборка: Fore;


## Описание


Интерфейс IForeSubParams предназначен
 для управления параметрами процедур/функций/конструкторов, которые реализованы
 в сборках репозитория.


## Иерархия наследования


IForeSubParams


## Комментарии


Коллекцию параметров можно получить, используя свойство [IForeSub.Params](../IForeSub/IForeSub.Params.htm).
