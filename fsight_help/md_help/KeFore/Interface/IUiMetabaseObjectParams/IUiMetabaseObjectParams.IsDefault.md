# IUiMetabaseObjectParams.IsDefault


## Синтаксис


IsDefault: Boolean;


## Описание


Свойство IsDefault определяет,
 будут ли при открытии объекта использоваться значения параметров, установленные
 по умолчанию.


## Комментарии


Если свойству установлено значение True,
 то в качестве значений параметров будут браться значения соответствующих
 [элементов управления](IUiMetabaseObjectParams.ParamControl.htm).
 Если свойству установлено значение False,
 то используются значения параметров, установленные по умолчанию.
