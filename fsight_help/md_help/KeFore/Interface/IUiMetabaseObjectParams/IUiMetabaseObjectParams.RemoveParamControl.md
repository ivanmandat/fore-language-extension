# IUiMetabaseObjectParams.RemoveParamControl


## Синтаксис


RemoveParamControl(Value: [IControl](ModForms.chm::/Interface/IControl/IControl.htm));


## Параметры


Value - элемент управления,
 по которому необходимо сбросить привязку к параметрам объекта.


## Описание


Метод RemoveParamControl осуществляет
 удаление привязки элемента управления к параметру объекта.


## Комментарии


Элемент управления, для которого необходимо сбросить привязку, передается
 посредством параметра Value. После
 удаления привязки значения элемента управления ни будут учитываться при
 открытии [объекта](IUiMetabaseObjectParams.UiObject.htm).
