# IResourceExporter


## Описание


Интерфейс IResourceExporter
 содержит свойства и методы, применяемые для [экспорта
 ресурсов](UiNav.chm::/Multilanguage/Resources/Export_Resource.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Consumer](IResourceExporter.Consumer.htm)
		 Свойство Consumer определяет
		 объект, в который будет произведен экспорт.


		 ![](../../Property_Image.gif)
		 [Locales](IResourceExporter.Locales.htm)
		 Свойство Locales определяет
		 языки ресурсов для экспорта.


		 ![](../../Property_Image.gif)
		 [Resource](IResourceExporter.Resource.htm)
		 Свойство Resource определяет
		 объект-ресурсы, который должен быть экспортирован.


		 ![](../../Property_Image.gif)
		 [Strings](IResourceExporter.Strings.htm)
		 Свойство Strings определяет
		 строки ресурсов для экспорта.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Export](IResourceExporter.Export.htm)
		 Метод Export осуществляет
		 экспорт ресурсов.
