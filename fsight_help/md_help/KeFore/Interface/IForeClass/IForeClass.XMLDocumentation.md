# IForeClass.XMLDocumentation


## Синтаксис


		XMLDocumentation: [IXmlDomDocument](ModXml.chm::/develop.htm);


## Описание


Свойство XMLDocumentation возвращает
 XML-комментарий, который задан для текущей конструкции.


## Комментарии


XML-комментарий формируется следующим образом:


	- Корневым элементом будет элемент, состоящий из открывающего
	 и закрывающего тэга. Имя элемента совпадает с наименованием текущей
	 конструкции.


	- Дочерними элементами будут все комментарии, начинающиеся с трёх
	 слешей.


XML-комментарий не будет возвращён, если имеются какие-либо ошибки,
 не позволяющие сформировать XML-структуру:


	- Для конструкции имеются комментарии,
	 начинающиеся с двух слешей или многострочные комментарии в фигурных
	 скобках;


	- В рамках комментария имеются тэги
	 с синтаксическими ошибками, отсутствует один из тегов или какие-либо
	 другие ошибки.
