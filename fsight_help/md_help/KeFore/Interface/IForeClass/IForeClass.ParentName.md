# IForeClass.ParentName


## Синтаксис


		ParentName: String;


## Описание


Свойство ParentName возвращает
 наименование родительского класса/интерфейса.


## Комментарии


Значение свойства будет доступно, если для текущего класса/интерфейса
 в коде определён родительский класс/интерфейс.
