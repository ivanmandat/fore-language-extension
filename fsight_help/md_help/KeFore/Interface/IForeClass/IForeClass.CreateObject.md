# IForeClass.CreateObject


## Синтаксис


		CreateObject: [IForeObject](../IForeObject/IForeObject.htm);


## Описание


Метод CreateObject создаёт экземпляр
 объекта текущего класса.


## Комментарии


Метод актуален, если текущая конструкция является классом. Созданный
 экземпляр указывается в свойстве [IForeSub.Self](../IForeSub/IForeSub.Self.htm)
 для процедур/функций, которые не являются статическими и могут выполняться
 только у экземпляра класса.
