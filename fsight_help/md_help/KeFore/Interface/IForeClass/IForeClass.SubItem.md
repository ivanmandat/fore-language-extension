# IForeClass.SubItem


## Синтаксис


		SubItem(Index: Integer): [IForeSub](../IForeSub/IForeSub.htm);


## Параметры


SubIndex. Индекс процедуры/функции.
 Значение индекса указывается в диапазоне [0, [SubsCount](IForeClass.SubsCount.htm)-1].


## Описание


Метод SubItem возвращает информацию
 о процедуре/функции с указанным индексом.

Пример использования приведён в описании метода [IForeAssembly.BindToClass](../IForeAssembly/IForeAssembly.BindToClass.htm).
