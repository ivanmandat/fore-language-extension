# IForeMethod.Params


## Синтаксис


Params: [IForeMethodParams](../IForeMethodParams/IForeMethodParams.htm);


## Описание


Свойство Params возвращает коллекцию
 параметров пользовательского метода.


## Комментарии


Список параметров формируется на основании сигнатуры процедуры/функции,
 которые лежат в основе пользовательского метода. Если сигнатура может
 измениться, то для обновления списка параметров выполните метод [IBaseMethod.Bind](../IBaseMethod/IBaseMethod.Bind.htm).
