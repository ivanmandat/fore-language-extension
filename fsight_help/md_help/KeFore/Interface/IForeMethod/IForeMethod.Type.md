# IForeMethod.Type


## Синтаксис


Type: [ForeMethodType](../../Enums/ForeMethodType.htm);


## Описание


Свойство Type определяет тип
 пользовательского метода.


## Комментарии


Режимы расчета в базах данных временных рядов и контейнерах моделирования
 определены заранее, поэтому данное свойство не влияет на режим, в котором
 будет производиться расчет при использовании пользовательского метода.
 Значение данного свойства используется для классификации пользовательских
 методов, и определяет каким образом будет рассматриваться соответствующая
 процедура/функция при использовании.


В контейнере моделирования для использования будут доступны пользовательские
 методы, для которых данному свойству установлено значение [ForeMethodType.Serie](../../Enums/ForeMethodType.htm).
