# IForeCriticalSection


Сборка: Fore;


## Описание


Интерфейс IForeCriticalSection
 предназначен для работы с критической секцией, используемой при работе
 с потоками.


## Иерархия наследования


IForeCriticalSection


## Комментарии


Критическая секция - это участок
 кода, в котором осуществляется доступ к ресурсам, которые не должны быть
 одновременно использованы более чем одним [потоком](../IForeThread/IForeThread.htm).
 Использование критической секции позволяет предотвратить изменение глобальных
 переменных, содержащих какие-либо объекты и к которым могут обращаться
 различные потоки при выполнении кода.


Несколько потоков, выполняющих однотипные действия и использующих одни
 и те же переменные, должны использовать одну общую критическую секцию.
