## Синтаксис

## Описание


Свойство Type возвращает тип
 условия выполнения задачи.


## Комментарии


Свойство доступно только для чтения.


Для установки признака выполнения задачи по наступлению события используйте
 метод [IScheduledTaskChecker.Check](IScheduledTaskChecker.Check.htm).
