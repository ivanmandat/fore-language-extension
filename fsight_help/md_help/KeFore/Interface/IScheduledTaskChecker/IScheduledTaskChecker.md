# IScheduledTaskChecker


Сборка: Fore;


## Описание


Интерфейс IScheduledTaskChecker
 предназначен для работы с условиями выполнения задачи, выполняемой в контейнере
 запланированных задач.


## Иерархия наследования


           IScheduledTaskChecker


## Комментарии


Для настройки задачи, выполняемой в контейнере запланированных задач,
 используйте свойства и методы интерфейса [IScheduledTask](../IScheduledTask/IScheduledTask.htm).
