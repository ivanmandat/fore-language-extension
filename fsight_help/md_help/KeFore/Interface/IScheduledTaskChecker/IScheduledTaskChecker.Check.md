# IScheduledTaskChecker.Check


## Синтаксис


Check: Boolean;


## Описание


Метод Check устанавливает признак
 выполнения задачи по наступлению события.


## Комментарии


Допустимые значения:


	- True. Задача выполняется
	 по наступлению события;


	- False. Задача не выполняется
	 по наступлению события.
