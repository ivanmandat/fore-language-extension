# IScheduledTaskPeriodDaily


## Описание


Интерфейс IScheduledTaskPeriodDaily
 содержит свойства и методы, определяющие параметры ежедневного расчета
 задачи.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [EveryDays](IScheduledTaskPeriodDaily.EveryDays.htm)
		 Свойство EveryDays
		 определяет интервал между запусками в днях.


		 ![](../../Property_Image.gif)
		 [StartDateTime](IScheduledTaskPeriodDaily.StartDateTime.htm)
		 Свойство StartDateTime
		 определяет дату и время начала отсчета.


## Свойства, унаследованные от [IScheduledTaskPeriod](../IScheduledTaskPeriod/IScheduledTaskPeriod.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [StopDateTime](../IScheduledTaskPeriod/IScheduledTaskPeriod.StopDateTime.htm)
		 Свойство StopDateTime
		 определяет дату окончания действия периода, в который производится
		 расчет задачи.


		 ![](../../Property_Image.gif)
		 [Type](../IScheduledTaskPeriod/IScheduledTaskPeriod.Type.htm)
		 Свойство Type возвращает
		 способ задания периода.


## Методы, унаследованные от [IScheduledTaskPeriod](../IScheduledTaskPeriod/IScheduledTaskPeriod.htm)


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Assign](../IScheduledTaskPeriod/IScheduledTaskPeriod.Assign.htm)
		 Метод Assign устанавливает
		 параметры периода расчета задачи в соответствии с параметрами
		 указанного периода.


		 ![](../../Sub_Image.gif)
		 [Next](../IScheduledTaskPeriod/IScheduledTaskPeriod.Next.htm)
		 Метод Next осуществляет
		 расчет следующей даты и времени запуска задачи относительно указанной
		 даты с учетом параметров периода.
