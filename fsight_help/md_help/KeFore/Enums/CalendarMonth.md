# CalendarMonth


## Описание


Перечисление CalendarMonth содержит
 месяцы, по которым необходимо установить признак расчета.


Используется следующим свойством:


	- [IScheduledTaskPeriodMonthly.Months](../Interface/IScheduledTaskPeriodMonthly/IScheduledTaskPeriodMonthly.Months.htm).
