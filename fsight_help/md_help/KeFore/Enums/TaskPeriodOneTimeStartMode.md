# TaskPeriodOneTimeStartMode


## Описание


Перечисление TaskPeriodOneTimeStartMode
 содержит периоды, когда будет производиться одноразовый запуск задачи.


Используется следующим свойством:


	- [IScheduledTaskPeriodOneTimeOnly.StartMode](../Interface/IScheduledTaskPeriodOneTimeOnly/IScheduledTaskPeriodOneTimeOnly.StartMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Queued. В порядке очереди.


		 1
		 Immediate. Немедленно.


		 2
		 ByTime. В указанное
		 время.


		 3
		 OnLogon. При открытии
		 репозитория.
