# ScheduledAlertAuditResult


## Описание


Перечисление ScheduledAlertAuditResult
 используется для определения типа результата выполнения задачи.


Используется следующими свойствами:


	- [IScheduledTaskAuditAlert.Result](../Interface/IScheduledTaskAuditAlert/IScheduledTaskAuditAlert.Result.htm);


	- [ISheduledAuditEvent.Result](../Interface/ISheduledAuditEvent/ISheduledAuditEvent.Result.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 -1
		 None. Любой результат
		 выполнения задачи.


		 0
		 Failed. Результат выполнения
		 задачи «неуспешно».


		 1
		 Succeeded. Результат
		 выполнения задачи «успешно».
