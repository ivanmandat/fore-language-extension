# ForeClassType


## Описание


Перечисление ForeClassType содержит
 типы конструкций, которые могут использоваться в Fore для написания кода
 приложений.


Используется следующими свойствами и методами:


	- [IForeClass.ClassType](../Interface/IForeClass/IForeClass.ClassType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Class_. Класс.


		 1
		 Enum_. Перечислимый
		 тип.


		 2
		 Interface_. Интерфейс.


		 3
		 Sub_. Процедура/функция.
