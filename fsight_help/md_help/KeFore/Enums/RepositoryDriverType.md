# RepositoryDriverType


## Описание


Перечисление RepositoryDriverType
 содержит список драйверов, которые могут использоваться для работы с репозиторием
 платформы.


Используется следующим свойством:


	- [IRepositoryScriptManager.Driver](../Interface/IRepositoryScriptManager/IRepositoryScriptManager.Driver.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Unknown. Неизвестный
		 тип драйвера.


		 2
		 ORCL9. Тип драйвера
		 Оracle.


		 7
		 MSSQL2012. Тип драйвера
		 Microsoft SQL Server.


		 8
		 Postgres. Тип драйвера
		 PostgreSQL.


		 10
		 SQLite. Тип драйвера
		 SQLite.
