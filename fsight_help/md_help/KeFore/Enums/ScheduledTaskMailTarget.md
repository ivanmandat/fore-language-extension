# ScheduledTaskMailTarget


## Описание


Перечисление ScheduledTaskMailTarget
 содержит виды адресов электронной почты, на которые может быть отправлен
 результат выполнения задач.


Используется следующими свойствами и методами:


	- [IScheduledTaskProperties.MailTargetType](../Interface/IScheduledTaskProperties/IScheduledTaskProperties.MailTargetType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Default_. Электронная
		 почта, установленная для пользователя по умолчанию. Данная почта
		 указывается в свойстве [Email](KeSom.chm::/Interface/IUserProfile/IUserProfile.Email.htm)
		 профиля пользователя.


		 1
		 Mobile. Мобильная электронная
		 почта пользователя. Данная почта указывается в свойстве [Mobile](KeSom.chm::/Interface/IUserEmails/IUserEmails.Mobile.htm)
		 профиля пользователя.


		 2
		 Personal. Личная электронная
		 почта пользователя. Данная почта указывается в свойстве [Personal](KeSom.chm::/Interface/IUserEmails/IUserEmails.Personal.htm)
		 профиля пользователя.


		 4
		 Work. Рабочая электронная
		 почта пользователя. Данная почта указывается в свойстве [Working](KeSom.chm::/Interface/IUserEmails/IUserEmails.Working.htm)
		 профиля пользователя.
