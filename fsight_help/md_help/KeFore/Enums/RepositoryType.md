# RepositoryType


## Описание


Перечисление RepositoryType
 содержит типы репозиториев, с которыми можно работать в платформе.


Используется следующим свойством:


	- [IRepositoryScriptManager.Repository](../Interface/IRepositoryScriptManager/IRepositoryScriptManager.Repository.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Unknown. Неопределенный
		 тип репозитория.


		 1
		 Metabase. Основной
		 репозиторий платформы.


		 2
		 RDS. Репозиторий НСИ.
