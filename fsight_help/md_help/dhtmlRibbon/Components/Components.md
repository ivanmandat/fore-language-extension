# Компоненты для работы с лентой инструментов


## Описание


На данной странице приведены компоненты для работы с лентой инструментов.


## Компоненты


		 Имя компонента
		 Краткое описание


		 ![](../Class_Image.gif)
		 [Ribbon](Ribbon/Ribbon.htm)
		 Компонент Ribbon представляет
		 собой ленту приложения, которая объединяет в себе функциональность
		 главного меню и [панели
		 инструментов](dhtmlUi.chm::/Classes/ToolBar/ToolBar.htm).


		 ![](../Class_Image.gif)
		 [RibbonButton](RibbonButton/RibbonButton.htm)
		 Компонент RibbonButton
		 представляет собой кнопку ленты приложения.


		 ![](../Class_Image.gif)
		 [RibbonCategory](RibbonCategory/RibbonCategory.htm)
		 Компонент RibbonCategory
		 представляет собой вкладку [ленты приложения](Ribbon/Ribbon.htm).


		 ![](../Class_Image.gif)
		 [RibbonPanel](RibbonPanel/RibbonPanel.htm)
		 Компонент RibbonPanel представляет
		 собой панель [вкладки](../Classes/RibbonCategory/RibbonCategory.htm)
		 [ленты приложения](../Classes/Ribbon/Ribbon.htm).
