# RibbonPanel.LaunchButton


## Синтаксис


LaunchButton: {settings}


## Параметры


settings. JSON-объект со значениями свойств кнопки.


## Описание


Свойство LaunchButton устанавливает кнопку запуска для панели.


## Комментарии


Кнопка запуска расположена в правом нижнем углу панели и имеет следующий вид:![](../../Components/Ribbon/Ribbon8.gif).


Значением свойства является JSON-объект с настройками кнопки. Для данной кнопки нельзя настроить изображение или текст.
