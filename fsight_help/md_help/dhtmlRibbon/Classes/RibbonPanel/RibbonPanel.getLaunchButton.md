# RibbonPanel.getLaunchButton


## Синтаксис


getLaunchButton ();


## Описание


Метод getLaunchButton возвращает кнопку запуска.


## Комментарии


Кнопка запуска расположена в правом нижнем углу панели и имеет следующий вид:![](../../Components/Ribbon/Ribbon8.gif).


Добавить кнопку запуска можно из JSON-объекта при помощи свойства [LaunchButton](RibbonPanel.LaunchButton.htm)
