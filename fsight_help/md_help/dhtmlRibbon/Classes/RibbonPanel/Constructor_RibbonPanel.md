# Конструктор RibbonPanel


## Синтаксис


PP.Ui.RibbonPanel(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор RibbonPanel создает экземпляр компонента RibbonPanel.
