# RibbonPanel.addControl


## Синтаксис


addControl (value, isSmallControl);


## Параметры


value. Элемент управления, который необходимо добавить.


isSmallControl. Признак того, принимает ли элемент управления уменьшенные размеры. Если установлено значение true, компонент принимает уменьшенные размеры, если false - стандартные. Значение по умолчанию - false.


## Описание


Метод addControl добавляет элемент управления на панель.
