# RibbonCategory.getGroupName


## Синтаксис


getGroupName ();


## Описание


Метод getGroupName возвращает имя группы вкладок.


## Комментарии


В группы объединяются контекстные вкладки. Имя группы устанавливается при помощи свойства [RibbonCategory.GroupName](RibbonCategory.GroupName.htm).


## Пример


Пример использования метода приведен в описании свойства [Ribbon.SelectedIndex](../Ribbon/Ribbon.SelectedIndex.htm).
