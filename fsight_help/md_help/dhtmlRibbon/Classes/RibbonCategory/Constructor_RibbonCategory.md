# Конструктор RibbonCategory


## Синтаксис


PP.Ui.RibbonCategory(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор RibbonCategory создает экземпляр компонента [RibbonCategory](../../Components/RibbonCategory/RibbonCategory.htm).
