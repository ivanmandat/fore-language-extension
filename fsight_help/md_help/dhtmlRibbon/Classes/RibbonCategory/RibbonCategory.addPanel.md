# RibbonCategory.addPanel


## Синтаксис


addPanel (value: PP.Ui.[RibbonPanel](../RibbonPanel/RibbonPanel.htm),
 beforePanel: PP.Ui.[RibbonPanel](../RibbonPanel/RibbonPanel.htm));


## Параметры


value. Панель, которую нужно
 добавить, - экземпляр класса [RibbonPanel](../RibbonPanel/RibbonPanel.htm).


beforePanel. Панель, перед
 которой нужно добавить, - экземпляр класса [RibbonPanel](../RibbonPanel/RibbonPanel.htm).


## Описание


Метод addPanel добавляет панель
 на вкладку.
