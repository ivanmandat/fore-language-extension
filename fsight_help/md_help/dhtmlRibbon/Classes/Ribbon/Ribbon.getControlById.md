# Ribbon.getControlById


## Синтаксис


getControlById (value);


## Параметры


value. Идентификатор элемента
 управления.


## Описание


Метод getControlById возвращает
 элемент управления по идентификатору.


## Комментарии


Идентификатор - значение свойства [Id](dhtmlCommon.chm::/Classes/object/object.id.htm)
 - должен быть задан в JSON.
