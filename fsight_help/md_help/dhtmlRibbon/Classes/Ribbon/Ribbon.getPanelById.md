# Ribbon.getPanelById


## Синтаксис


getPanelById (value: String);


## Параметры


value. Идентификатор панели.


## Описание


Метод getPanelById возвращает
 панель по идентификатору.


## Комментарии


Идентификатор - значение свойства [Id](dhtmlCommon.chm::/Classes/object/object.id.htm)
 - должен быть задан в JSON.
