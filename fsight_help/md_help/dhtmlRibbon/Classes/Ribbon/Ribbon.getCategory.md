# Ribbon.getCategory


## Синтаксис


getCategory(index: Number);


## Параметры


index. Индекс вкладки.


## Описание


Метод getCategory возвращает
 вкладку ленты инструментов по индексу.


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [Ribbon](Ribbon.htm) c наименованием «ribbon» (см. «[Пример размещения
 компонента Ribbon из JSON](../../Components/Ribbon/Example_Ribbon_JSON.htm)»). Пример выполняется в консоли браузера.
 Получим заголовок первой вкладки ленты инструментов:


// Получаем заголовок первой вкладки ленты инструментов
console.log("Заголовок первой вкладки ленты инструментов: " + ribbon.getCategory(0).getCaption());
В результате в консоль будет выведен заголовок первой вкладки:


Заголовок первой вкладки ленты инструментов: Main
