# Ribbon.beginUpdate


## Синтаксис


beginUpdate();


## Описание


Метод beginUpdate отключает
 перерисовку ленты инструментов до тех пор, пока не будет вызван метод
 [endUpdate](Ribbon.endUpdate.htm).

Пример использования метода приведен в описании свойства [Ribbon.SelectedItem](Ribbon.SelectedItem.htm).
