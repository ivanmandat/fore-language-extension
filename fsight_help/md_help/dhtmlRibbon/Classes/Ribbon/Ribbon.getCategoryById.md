# Ribbon.getCategoryById


## Синтаксис


getCategoryById (value);


## Параметры


value. Идентификатор вкладки.


## Описание


Метод getCategoryById возвращает
 вкладку по идентификатору.


## Комментарии


Идентификатор - значение свойства [Id](dhtmlCommon.chm::/Classes/object/object.id.htm)
 - должен быть задан в JSON.
