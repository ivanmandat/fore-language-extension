# Ribbon.IsAutoHidingEnabled


## Синтаксис


IsAutoHidingEnabled: Boolean


## Описание


Свойство IsAutoHidingEnabled определяет, доступна ли кнопка сворачивания ленты приложения.


## Комментарии


Кнопка сворачивания ленты приложения расположена в правом верхнем углу компонента.


Если для свойства установлено значение false, то кнопка сворачивания не отображается. По умолчанию установлено значение true.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента [Ribbon](../../Components/Ribbon/Ribbon.htm) с наименованием «ribbon» (см «[Пример создания компонента Ribbon](../../Components/Ribbon/Example_Ribbon.htm)»). Скроем кнопку сворачивания:


ribbon.setIsAutoHidingEnabled(false);


После выполнения примера сворачивание или разворачивание содержимого вкладок ленты будет недоступным.
