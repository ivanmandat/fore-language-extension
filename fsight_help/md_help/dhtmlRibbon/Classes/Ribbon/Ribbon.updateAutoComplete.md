# Ribbon.updateAutoComplete


## Синтаксис


updateAutoComplete();


## Описание


Метод updateAutoComplete обновляет
 результат поиска операций, введенных в поле быстрого запуска.


## Пример


Пример использования метода приведен в описании свойства [Ribbon.UseIntellisense](Ribbon.UseIntellisense.htm).
