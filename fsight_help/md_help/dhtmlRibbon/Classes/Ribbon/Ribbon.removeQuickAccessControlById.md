# Ribbon.removeQuickAccessControlById


## Синтаксис


removeQuickAccessControlById (value);


## Параметры


value. Идентификатор кнопки, которую необходимо удалить.


## Описание


Метод removeQuickAccessControlById удаляет кнопку из панели быстрого доступа по идентификатору.


## Комментарии


Идентификатор задается в JSON-объекте с настройками кнопки.


Панель быстрого доступа находится справа от кнопки приложения.


Элементы управления добавляются на панель быстрого доступа с помощью метода [addCategory](Ribbon.addCategory.htm).

Пример использования метода аналогичен примеру, приведенному в описании метода [removeQuickAccessControl](Ribbon.removeQuickAccessControl.htm).
