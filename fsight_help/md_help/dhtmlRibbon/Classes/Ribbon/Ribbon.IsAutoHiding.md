# Ribbon.IsAutoHiding


## Синтаксис


IsAutoHiding: Boolean


## Описание


Свойство IsAutoHiding определяет, свернута ли лента приложения.


## Комментарии


Если для свойства установлено значение false, то лента приложения развернута. Если установлено значение true - свернута. По умолчанию установелно значение false.
