# Ribbon.getIsCategoryContentHidden


## Синтаксис


getIsCategoryContentHidden ();


## Описание


Метод getIsCategoryContentHidden возвращает признак того, является ли вкладка скрытой.


## Комментарии


Если свойство возвращает значение true, значит содержимое вкладок скрыто (свернуто). Если возвращается значение false - содержимое раскрыто.
