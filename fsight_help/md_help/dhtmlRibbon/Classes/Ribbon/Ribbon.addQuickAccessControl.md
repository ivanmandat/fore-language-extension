# Ribbon.addQuickAccessControl


## Синтаксис


addQuickAccessControl (value);


## Параметры


value. Кнопка, экземпляр класса PP.Ui.[RibbonButton](../RibbonButton/RibbonButton.htm), которая будет добавлена на панель быстрого доступа.


## Описание


Метод addQuickAccessControl добавляет кнопку на панель быстрого доступа.


## Комментарии


Панель быстрого доступа расположена справа от [кнопки приложения](Ribbon.MainButton.htm).
