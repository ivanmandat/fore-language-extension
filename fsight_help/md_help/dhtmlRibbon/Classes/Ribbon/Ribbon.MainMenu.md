# Ribbon.MainMenu


## Синтаксис


MainMenu: PP.Ui.[Menu](dhtmlUi.chm::/Classes/Menu/Menu.htm)


## Описание


Свойство MainMenu устанавливает
 главное меню ленты приложения.


## Комментарии


Главное меню ленты приложения раскрывается при нажатии на [кнопку
 приложения](Ribbon.getMainButton.htm), которая находится в верхнем правом углу.
