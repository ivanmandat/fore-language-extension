# Ribbon.Controls


## Синтаксис


Controls: Object


## Описание


Свойство Controls содержит набор
 элементов управления ленты инструментов.


## Комментарии


Значение свойства устанавливается из JSON, возвращается с помощью метода
 getControls.


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [Ribbon](Ribbon.htm) c наименованием «ribbon» (см. «[Пример размещения
 компонента Ribbon из JSON](../../Components/Ribbon/Example_Ribbon_JSON.htm)»). Пример выполняется в консоли браузера.
 Получим название первой кнопки ленты инструментов:


// Получаем название первой кнопки ленты инструментов
console.log("Название первой кнопки ленты инструментов: " + ribbon.getControls().but1.getContent());
В результате в консоль будет выведено название первой кнопки ленты инструментов:


Название первой кнопки ленты инструментов: Paste
