# Ribbon.getQuickAccessControls


## Синтаксис


getQuickAccessControls ();


## Описание


Метод getQuickAccessControls возвращает элементы управления панели быстрого доступа.


## Комментарии


Панель быстрого доступа расположена слева справа от [кнопки приложения](Ribbon.MainButton.htm).


Элементы добавляются на панель быстрого доступа при помощи метода [addQuickAccessControl](Ribbon.addQuickAccessControl.htm).
