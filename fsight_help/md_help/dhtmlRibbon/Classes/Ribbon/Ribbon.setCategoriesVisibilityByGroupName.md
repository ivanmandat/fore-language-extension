# Ribbon.setCategoriesVisibilityByGroupName


## Синтаксис


setCategoriesVisibilityByGroupName (name, flag);


## Параметры


name. Имя группы контекстных вкладок;


flag. Признак того, отображается ли группа контекстных вкладок. Если установлено значение true, то группа контекстных вкладок отображается, если false - не отображается. По умолчанию установлено значение false.


## Описание


Метод setCategoriesVisibilityByGroupName определяет, отображается ли группа контекстных вкладок.
