# Ribbon.endUpdate


## Синтаксис


endUpdate();


## Описание


Метод endUpdate включает перерисовку
 ленты инструментов и обновляет её.


## Комментарии


Использует в связке с методом [beginUpdate](Ribbon.beginUpdate.htm).

Пример использования метода приведен в описании свойства [Ribbon.SelectedItem](Ribbon.SelectedItem.htm).
