# Ribbon.getMainButton


## Синтаксис


getMainButton ();


## Описание


Метод getMainButton возвращает кнопку приложения.


## Комментарии


Кнопка приложения находится в левом верхнем углу. При нажатии на нее раскрывается [главное меню](Ribbon.MainMenu.htm) ленты приложения.
