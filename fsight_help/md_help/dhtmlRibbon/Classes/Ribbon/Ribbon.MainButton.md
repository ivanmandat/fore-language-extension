# Ribbon.MainButton


## Синтаксис


MainButton: {}


## Описание


Свойство MainButton устанавливает настройки для кнопки приложения.


## Комментарии


Свойство используется только из JSON.


Метод [getMainButton](Ribbon.getMainButton.htm) возвращает кнопку приложения.
