# Классы для работы с лентой инструментов


## Описание


На данной странице приведены классы для работы с лентой инструментов.


## Классы


		 Имя класса
		 Краткое описание


		 ![](../Class_Image.gif)
		 [Ribbon](Ribbon/Ribbon.htm)
		 Класс Ribbon реализует
		 компонент [Ribbon](../Components/Ribbon/Ribbon.htm),
		 представляющий собой ленту инструментов, компонент, объединяющий
		 в себе функциональность главного меню и [панели инструментов](dhtmlUi.chm::/Classes/toolbar/toolbar.htm).


		 ![](../Class_Image.gif)
		 [RibbonButton](RibbonButton/RibbonButton.htm)
		 Класс RibbonButton
		 реализует компонент [RibbonButton](../Components/RibbonButton/RibbonButton.htm),
		 представляющий собой кнопку [ленты
		 приложения](../Components/Ribbon/Ribbon.htm).


		 ![](../Class_Image.gif)
		 [RibbonCategory](RibbonCategory/RibbonCategory.htm)
		 Класс RibbonCategory
		 реализует компонент [RibbonCategory](../Components/RibbonCategory/RibbonCategory.htm),
		 представляющий собой вкладку [ленты
		 инструментов](../Components/Ribbon/Ribbon.htm).


		 ![](../Class_Image.gif)
		 [RibbonPanel](RibbonPanel/RibbonPanel.htm)
		 Класс RibbonPanel реализует
		 компонент [RibbonPanel](../Components/RibbonPanel/RibbonPanel.htm),
		 представляющий собой панель [вкладки](RibbonCategory/RibbonCategory.htm)
		 [ленты инструментов](Ribbon/Ribbon.htm).
