# JsBlock.getBlockType


## Синтаксис


getBlockType();


## Описание


Метод getBlockType возвращает
 тип блока.


## Комментарии


Допустимые значения:


	- Panel. Панель;


	- TabPanel. Вкладки;


	- Table. Таблица;


	- Chart. Диаграмма;


	- Map. Карта;


	- Bubbles. Пузырьковая
	 диаграмма;


	- BubbleTree. Пузырьковое
	 дерево;


	- TreeMap. Плоское дерево;


	- Gauge. Индикатор;


	- Report. Регламентный
	 отчёт;


	- Button. Кнопка;


	- Control. Список или
	 раскрывающийся список;


	- Param. Параметр;


	- Label. Надпись;


	- Image. Рисунок;


	- Frame. HTML;


	- Text. Текст.
