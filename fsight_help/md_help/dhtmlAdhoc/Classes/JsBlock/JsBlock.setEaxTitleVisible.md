# JsBlock.setEaxTitleVisible


## Синтаксис


setEaxTitleVisible(visible: Boolean);


## Параметры


visible. Определяет
 отображение заголовка визуализатора:


	- True. Заголовок отображается;


	- False. Заголовок скрыт.


## Описание


Метод setEaxTitleVisible скрывает/отображает
 заголовок визуализатора.


## Комментарии


Визуализаторы используются для отображения информации из источников
 данных с возможностью анализа и настройки отображаемых данных.


Доступные визуализаторы: таблица, диаграмма, карта, пузырьковая диаграмма,
 пузырьковое дерево, плоское дерево, индикатор.


## Пример


Для выполнения примера убедитесь, что в репозитории содержится аналитическая
 панель с ключом «88665».


Создайте HTML-страницу с [примером
 размещения компонента KapBox](../../Components/KapBox/KapBox_example.htm) и выполните следующие действия:


	- Откройте аналитическую панель с ключом «88665»:


		- выполните команду «Документ
		 > Открыть» в главном меню;


		- выполните в консоли:


kapBox.open({
    Key: 88665
});


	- Выделите блок аналитической панели с визуализатором и выполните
	 в консоли:


var eaxTitleCheckBox = new PP.Ui.CheckBox({
    ParentNode: "checkBox",
    Content: "Показать/скрыть заголовок визуализатора",
    CheckedChanged: function () {
        currentBlock.setEaxTitleVisible(eaxTitleCheckBox.getChecked());
    }
});

После выполнения примера будет добавлен флажок «Показать/скрыть
 заголовок визуализатора», при установке/снятии которого будет отображаться/скрываться
 заголовок выделенного блока с визуализатором.
