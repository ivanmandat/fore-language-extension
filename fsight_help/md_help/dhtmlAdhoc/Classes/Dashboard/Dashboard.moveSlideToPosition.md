# Dashboard.moveSlideToPosition


## Синтаксис


moveSlideToPosition(index);


## Параметры


index. Индекс слайда, на место
 которого необходимо переместить активный слайд.


## Описание


Метод moveSlideToPosition перемещает
 слайд аналитической панели на указанную позицию.


## Пример


Для выполнения примера убедитесь, что в репозитории содержится аналитическая
 панель с ключом «88665». В аналитической панели должно быть добавлено
 не менее четырёх [слайдов](UiAdhoc.chm::/Presentation/Presentation.htm).


Создайте HTML-страницу с [примером
 размещения компонента KapBox](../../Components/KapBox/KapBox_example.htm) и выполните следующие действия:


	- Откройте аналитическую панель с ключом «88665»:


		- выполните команду «Документ
		 > Открыть» в главном меню;


		- выполните в консоли:


kapBox.open({
    Key: 88665
});


	- Выполните в консоли:


// Получим четвертый слайд
kapBox.showSlide({
    SlideIndex: 3
})
// Перенесем слайд на место второго слайда
kapBox.moveSlideToPosition(1);

После выполнения примера четвертый слайд будет перемещен на место второго
 слайда.
