# Dashboard.setMultiLanguage


## Синтаксис


setMultiLanguage(isMultiLanguage: Boolean);


## Параметры


isMultiLanguage. Определяет
 использование мультиязычности:


	- True. Мультиязычность используется;


	- False.
	 Мультиязычность не используется.


## Описание


Метод setMultiLanguage управляет
 мультиязычностью аналитической панели.


## Комментарии


Если мультиязычность используется, то для текстовых полей аналитической
 панели возможен перевод на другие языки.
