# Dashboard.exportTo


## Синтаксис


exportTo({


    settings


});


## Параметры


settings. JSON-объект
 с параметром ExportData, в котором
 задается формат экспорта в параметре format.


## Описание


Метод exportTo осуществляет
 экспорт аналитической панели в заданном формате.


## Комментарии


Для экспорта доступны следующие форматы: *.xls, *.xlsx, *.pdf, *.rtf,
 *.html, *.mht, *.emf, *.pptx.


## Пример


Для выполнения примера убедитесь, что в репозитории содержится аналитическая
 панель с ключом «88665».


Создайте HTML-страницу с [примером
 размещения компонента KapBox](../../Components/KapBox/KapBox_example.htm) и выполните следующие действия:


	- Откройте аналитическую панель с ключом «88665»:


		- выполните команду «Документ
		 > Открыть» в главном меню;


		- выполните в консоли:


kapBox.open({
    Key: 88665
});


	- Выполните в консоли:


var expPPTXButton = new PP.Ui.Button({
    ParentNode: "but",
    Content: "Экспорт в *.pptx",
    Click: function () {
        kapBox.exportTo({ "ExportData": { "format": "pptx" } });
    }
});

После выполнения примера будет открыта аналитическая панель с ключом
 «88665» и добавлена кнопка «Экспорт в
 *.pptx», при нажатии на которую аналитическая панель будет экспортирована
 в файл формата *.pptx.
