# Dashboard.setPageSettings


## Синтаксис


setPageSettings({


    settings


});


## Параметры


settings. JSON-объект с параметрами
 страницы аналитической панели.


Доступны следующие параметры:


		 Имя параметра
		 Тип
		 Краткое описание


		 Обязательные
		 параметры:


		 PageSettings
		 string
		 Параметры страницы:


			- paperFormat.
			 Размер бумаги:


				- A4;


				- A5;


				- Letter.
				 Письмо;


				- Custom.
				 Произвольный.


			- paperOrientation.
			 Ориентация страницы:


				- Portrait.
				 Книжная;


				- Landscape.
				 Альбомная.


		 Необязательные
		 параметры:


		 Callback
		 function
		 Возвратная функция, которая срабатывает при вызове метода.


## Описание


Метод setPageSettings устанавливает
 параметры страницы.


## Пример


Для выполнения примера убедитесь, что в репозитории содержится аналитическая
 панель с ключом «88665».


Создайте HTML-страницу с [примером
 размещения компонента KapBox](../../Components/KapBox/KapBox_example.htm) и выполните следующие действия:


	- Откройте аналитическую панель с ключом «88665»:


		- выполните команду «Документ
		 > Открыть» в главном меню;


		- выполните в консоли:


kapBox.open({
    Key: 88665
});


	- Выполните в консоли:


kapBox.setPageSettings({
    "PageSettings": {
        "paperFormat": "A4",
        "paperOrientation": "Portrait"
    },
    "Callback": function () {
        var print = new PP.Ui.Button({
            ParentNode: "but",
            Content: "Печать",
            Click: function () {
                kapBox.print();
            }
        });
    }
});

После выполнения примера будут изменены параметры страницы аналитической
 панели на книжную ориентацию с размером бумаги A4 и добавлена кнопка «Печать», при нажатии на которую
 будет открыто окно для выбора параметров печати и предпросмотра аналитической
 панели.
