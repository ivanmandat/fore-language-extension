# Dashboard.showSlide


## Синтаксис


showSlide({


    params


});


## Параметры


params. JSON-объект с параметрами
 отображения слайда аналитической панели.


Доступны следующие параметры:


		 Имя параметра
		 Тип
		 Краткое описание


		 SlideKey
		 number
		 Ключ слайда аналитической панели.


		 SlideIndex
		 number
		 Индекс слайда аналитической панели.


## Описание


Метод showSlide отображает слайд
 аналитической панели по ключу или индексу.


## Комментарии


Для получения ключа слайда используйте всплывающую подсказку:


	- Выделите слайд аналитической панели.


	- Перейдите на заголовок боковой панели и наведите курсор на название
	 слайда.


После выполнения действий появится всплывающая подсказка, содержащая
 название слайда и его ключ.


Для получения индекса слайда используйте метод [Dashboard.getActiveSlideIndex](Dashboard.getActiveSlideIndex.htm).


## Пример


Для выполнения примера убедитесь, что в репозитории содержится аналитическая
 панель с ключом «88665». В аналитической панели должно быть добавлено
 не менее трёх [слайдов](UiAdhoc.chm::/Presentation/Presentation.htm).


Создайте HTML-страницу с [примером
 размещения компонента KapBox](../../Components/KapBox/KapBox_example.htm) и выполните следующие действия:


	- Откройте аналитическую панель с ключом «88665»:


		- выполните команду «Документ
		 > Открыть» в главном меню;


		- выполните в консоли:


kapBox.open({
    Key: 88665
});


	- Выполните в консоли:


kapBox.moveSlideTop();
kapBox.showSlide({
    SlideIndex: 1
})
kapBox.moveSlideBottom();
kapBox.refresh();

После выполнения примера активный слайд будет перемещен в начало, а
 второй слайд в конец.
