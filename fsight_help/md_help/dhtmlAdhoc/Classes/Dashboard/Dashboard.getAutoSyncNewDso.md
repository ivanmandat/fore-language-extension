# Dashboard.getAutoSyncNewDso


## Синтаксис


getAutoSyncNewDso();


## Описание


Метод getAutoSyncNewDso определяет,
 включено или отключено автоматическое связывание измерений.


## Комментарии


Допустимые значения:


	- True. Автоматическое
	 связывание измерений включено;


	- False. Автоматическое
	 связывание измерений отключено.
