# PivotEvents


Сборка: Pivot;


## Описание


Класс PivotEvents используется
 для реализации событий в обработчике событий таблицы.


## Комментарии


Используйте данный класс для реализации класса, который будет
 использоваться в качестве обработчика событий, при этом необязательно
 вызывать все методы класса PivotEvents,
 вызываем только необходимый метод.
