## С чего начать


		 Сборка Pivot
		 предназначена для работы с объектом репозитория «Аналитические
		 запросы (OLAP)».


		Изучив данный раздел, Вы сможете с применением языка Fore:


			- [работать
			 с распределением](Hierarchy/Hierarchy_Pareto.htm);


			- [работать
			 с подсветкой данных в таблице](Hierarchy/Hierarchy_Highlight.htm);


			- [работать
			 с вычисляемыми показателями](Hierarchy/Hierarchy_Evaluator.htm);


			- [работать
			 с данными таблицы](Hierarchy/Hierarchy_Tables.htm);


			- [работать
			 с заголовками таблицы](Hierarchy/Hierarchy_Header.htm);


			- [работать
			 с измерениями источников данных](Hierarchy/Hierarchy_Dimensions.htm);


			- [сортировать
			 данные таблицы](Hierarchy/Hierarchy_Sort.htm);


			- [фильтровать
			 данные таблицы](Hierarchy/Hierarchy_Filter.htm);


			- [обрабатывать
			 события таблицы](Hierarchy/Hierarchy_Events.htm).


		Ознакомиться с основными принципами работы с регламентными отчетами
		 можно в разделе «[Общие
		 принципы программирования с использованием сборки Pivot](KePivot_Programming.htm)».


		Более подробное описание возможностей блока приведено в разделе
		 «[Введение
		 в сборку Pivot](../KePivot_Introduction.htm)».


		Описание ресурсов сборки представлено в следующих подразделах:


			- [Интерфейсы
			 сборки Pivot](../Interface/KePivot_Interface.htm)


			- [Перечисления сборки
			 Pivot](../Enums/KePivot_Enums.htm)


			- [Классы сборки Pivot](../Class/kepivot_class.htm)


			- [Примеры](../Samples/KePivot_Sample.htm)


		 Перед изучением сборки Pivot рекомендуется ознакомиться
		 с разделами: «[Среда разработки](UiDevEnv.chm::/01_Development_Environment/01_Purpose_of_the_constructor/Purpose_of_the_constructor.htm)»
		 и «[Руководство по языку Fore](Fore.chm::/Fore_Title.htm)».


		Непосредственно программирование с использованием Fore
		 осуществляется в рамках объекта репозитория «[Аналитические
		 запросы (OLAP)](UiExpress.chm::/purpose/UiExpress_Purpose.htm)». Попробуйте работать
		 с регламентными отчетами, используя сборку Pivot.


### Вам потребуется


		Для работы Вам потребуется объект репозитория «Аналитические
		 запросы (OLAP)».
