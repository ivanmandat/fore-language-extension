# Общие принципы программирования
 с использованием сборки Pivot


Сборка Pivot предназначена для
 работы с основой для построения таблицы данных отчета.


Процесс работы с основой состоит из следующих блоков:


![](KePivot_Process.png)

[


Примечание.
 Блоки являются гиперссылками, для перехода к примерам, указанным в блоках,
 щелкните по ним мышью.


Для получения основы для построения таблицы используйте интерфейс IEaxAnalyzeCore.Pivot](Common_view_table.htm).


После получения основы будет доступна [настройка
 отображения данных таблицы](Common_view_table.htm).
