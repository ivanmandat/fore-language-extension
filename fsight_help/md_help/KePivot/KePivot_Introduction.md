# Введение в сборку Pivot


В данном разделе представлено описание сборки Pivot,
 содержащей средства для работы на языке Fore
 с таблицей данных объекта репозитория «Аналитические запросы (OLAP)».
 Подробное описание смотрите в разделах:


	- [Общие принципы программирования
	 с использованием сборки Pivot](Intro/KePivot_Programming.htm);


	- [Иерархия сборки Pivot](Intro/KePivot_Hierarchy.htm).


В интерфейсах сборки реализованы все необходимые средства для работы
 с таблицей экспресс-отчетов.


Основным интерфейсом сборки Pivot
 является [IPivot](Interface/IPivot/IPivot.htm).
