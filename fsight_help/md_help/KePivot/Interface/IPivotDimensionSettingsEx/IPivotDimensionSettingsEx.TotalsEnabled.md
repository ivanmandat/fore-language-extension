# IPivotDimensionSettingsEx.TotalsEnabled


## Синтаксис


TotalsEnabled: Boolean;


## Описание


Свойство TotalsEnabled определяет
 признак расчёта итогов по измерению.


## Комментарии


Допустимые значения:


	- True. Рассчитывать итоги
	 по измерению;


	- False. Не рассчитывать
	 итоги по измерению.
