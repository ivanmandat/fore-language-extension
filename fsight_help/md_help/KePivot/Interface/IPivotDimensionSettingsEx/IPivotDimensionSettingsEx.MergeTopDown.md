# IPivotDimensionSettingsEx.MergeTopDown


## Синтаксис


MergeTopDown: Boolean;


## Описание


Свойство MergeTopDown определяет
 признак поджатия объединённых элементов измерения в одну строку/столбец
 с элементами предыдущего измерения.


## Комментарии


Допустимые значения:


	- True. Поджатие объединённых
	 элементов измерения с элементами предыдущего измерения выполняется;


	- False. По умолчанию.
	 Поджатие объединённых элементов измерения с элементами предыдущего
	 измерения не выполняется.


Для получения подробной информации об объединённых элементах измерения
 и поджатии обратитесь к разделу «[Объединение
 измерений в боковике/шапке](UiAnalyticalArea.chm::/TableView/Layout/Merging_Dimensions.htm)».
