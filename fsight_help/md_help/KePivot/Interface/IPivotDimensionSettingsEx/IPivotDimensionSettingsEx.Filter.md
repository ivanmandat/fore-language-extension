# IPivotDimensionSettingsEx.Filter


## Синтаксис


Filter: [IPivotDimensionFilterSettings](../IPivotDimensionFilterSettings/IPivotDimensionFilterSettings.htm);


## Описание


Свойство Filter возвращает параметры
 специальной фильтрации данных по измерению.


## Комментарии


Фильтрация позволяет выделить подмножество данных из
 таблицы, т.к. в результате фильтрации отображаются только
 те данные, которые удовлетворяют заданным условиям.
