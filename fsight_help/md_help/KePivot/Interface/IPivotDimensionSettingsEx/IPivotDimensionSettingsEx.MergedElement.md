# IPivotDimensionSettingsEx.MergedElement


## Синтаксис


MergedElement: Integer;


## Описание


Свойство MergedElement определяет
 индекс элемента, наименование которого будет объединено с элементами измерения
 на верхнем уровне.


## Комментарии


Для отображения объединения в экспресс-отчёте необходимо установить
 размещение по всем измерениям (перейдите на вкладку «Конструктор»
 ленты инструментов, нажмите на кнопку «Размещение»
 и установите флажок «[Все измерения](UiAnalyticalArea.chm::/TableView/Layout/Merging_Dimensions.htm)»).


Настройка MergedElement может
 быть задана для любого измерения, кроме самого первого, расположенного
 по строкам/столбцам.
