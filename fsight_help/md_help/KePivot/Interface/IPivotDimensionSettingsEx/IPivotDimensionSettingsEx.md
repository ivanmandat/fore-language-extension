# IPivotDimensionSettingsEx


Сборка: Pivot;


## Описание


Интерфейс IPivotDimensionSettingsEx
 используется для работы с дополнительными настройками измерения источника
 данных.


## Иерархия наследования


           [IPivotDimensionSettings](../IPivotDimensionSettings/IPivotDimensionSettings.htm)


           IPivotDimensionSettingsEx


## Комментарии


Для получения объекта данного типа используйте свойство [IPivotDimension.Settings](../IPivotDimension/IPivotDimension.Settings.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Filter](IPivotDimensionSettingsEx.Filter.htm)
		 Свойство Filter возвращает
		 параметры специальной фильтрации данных по измерению.


		 ![](../../Property_Image.gif)
		 [MergedElement](IPivotDimensionSettingsEx.MergedElement.htm)
		 Свойство MergedElement
		 определяет индекс элемента, наименование которого будет объединено
		 с элементами измерения на верхнем уровне.


		 ![](../../Property_Image.gif)
		 [MergedSelection](IPivotDimensionSettingsEx.MergedSelection.htm)
		 Свойство MergedSelection
		 возвращает отметку объединённых элементов измерения.


		 ![](../../Property_Image.gif)
		 [MergeTopDown](IPivotDimensionSettingsEx.MergeTopDown.htm)
		 Свойство MergeTopDown определяет
		 признак поджатия объединённых элементов измерения в одну строку/столбец
		 с элементами предыдущего измерения.


		 ![](../../Property_Image.gif)
		 [TotalsEnabled](IPivotDimensionSettingsEx.TotalsEnabled.htm)
		 Свойство TotalsEnabled
		 определяет признак расчёта итогов по измерению.
