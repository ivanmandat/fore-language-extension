# IPivotDimensions.FindByKey


## Синтаксис


FindByKey(DimKey: Integer): [IPivotDimension](../IPivotDimension/IPivotDimension.htm);


## Параметры


DimKey. Ключ измерения, по
 которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает свойства измерения источника данных, в случае удачного
 поиска.

Использование свойства приведено в примере для [IPivotDimensionLevelNumbering.Enable](../IPivotDimensionLevelNumbering/IPivotDimensionLevelNumbering.Enable.htm).
