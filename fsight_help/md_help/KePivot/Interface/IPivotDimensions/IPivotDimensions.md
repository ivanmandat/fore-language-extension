# IPivotDimensions


## Описание


Интерфейс IPivotDimensions содержит
 свойства и методы для работы с коллекцией измерений источников данных,
 на основе которых формируется таблица.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPivotDimensions.Count.htm)
		 Свойство Count возвращает
		 количество измерений источников данных, на основе которых формируется
		 таблица.


		 ![](../../Property_Image.gif)
		 [Item](IPivotDimensions.Item.htm)
		 Свойство Item возвращает
		 свойства измерения.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindByKey](IPivotDimensions.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск измерения источника данных по ключу.
