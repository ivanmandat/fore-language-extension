# IPivotSorter.Remove


## Синтаксис


Remove(Index: Integer);


## Параметры


Index. Индекс сортировки в
 коллекции.


## Описание


Метод Remove удаляет сортировку
 из коллекции по ее индексу.


## Комментарии


Для добавления сортировки в коллекцию используйте [IPivotSorter.Add](IPivotSorter.Add.htm).


Для очищения коллекции сортировок используйте [IPivotSorter.Clear](IPivotSorter.Clear.htm).
