# IPivotSorter.Add


## Синтаксис


Add: [IPivotSortItem](../IPivotSortItem/IPivotSortItem.htm);


## Описание


Метод Add добавляет сортировку
 в таблицу.


## Комментарии


Для удаления сортировку из коллекции по ее индексу используйте [IPivotSorter.Remove](IPivotSorter.Remove.htm).


Для очищения коллекции сортировок используйте [IPivotSorter.Clear](IPivotSorter.Clear.htm).
