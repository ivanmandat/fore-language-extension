# IPivotSorter


Сборка: Pivot;


## Описание


Интерфейс IPivotSorter используется
 для работы с параметрами сортировки по различным элементам таблицы.


## Иерархия наследования


IPivotSorter


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPivotSorter.Count.htm)
		 Свойство Count возвращает
		 количество сортировок в коллекции.


		 ![](../../Property_Image.gif)
		 [Enabled](IPivotSorter.Enabled.htm)
		 Свойство Enabled определяет,
		 включена ли сортировка.


		 ![](../../Property_Image.gif)
		 [Item](IPivotSorter.Item.htm)
		 Свойство Item возвращает
		 сортировку по ее индексу в коллекции.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IPivotSorter.Add.htm)
		 Метод Add добавляет
		 сортировку в таблицу.


		 ![](../../Sub_Image.gif)
		 [Clear](IPivotSorter.Clear.htm)
		 Метод Clear очищает
		 коллекцию сортировок.


		 ![](../../Sub_Image.gif)
		 [Move](IPivotSorter.Move.htm)
		 Метод Move перемещает
		 сортировку внутри коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IPivotSorter.Remove.htm)
		 Метод Remove удаляет
		 сортировку из коллекции по ее индексу.
