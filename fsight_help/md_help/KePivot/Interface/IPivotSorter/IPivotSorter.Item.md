# IPivotSorter.Item


## Синтаксис


Item(Index: Integer): [IPivotSortItem](../IPivotSortItem/IPivotSortItem.htm);


## Параметры


Index. Индекс сортировки в
 коллекции.


## Описание


Свойство Item возвращает сортировку
 по ее индексу в коллекции.


## Комментарии


Для получения количества сортировок в коллекции используйте [IPivotSorter.Count](IPivotSorter.Count.htm).
