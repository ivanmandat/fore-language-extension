# IPivotSorter.Move


## Синтаксис


Move(IndexFrom: Integer; IndexTo: Integer);


## Параметры


IndexFrom. Индекс позиции,
 из которой необходимо переместить сортировку;


IndexTo. Индекс позиции, в
 которую необходимо переместить сортировку.


## Описание


Метод Move перемещает
 сортировку внутри коллекции.


## Комментарии


Для получения сортировки по ее индексу в коллекции используйте [IPivotSorter.Item](IPivotSorter.Item.htm).
