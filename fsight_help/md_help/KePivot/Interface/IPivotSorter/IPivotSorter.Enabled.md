# IPivotSorter.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет,
 включена ли сортировка.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Сортировка включена;


	- False. Сортировка выключена.
