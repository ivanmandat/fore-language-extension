# IPivotEvaluatorTotalsItem


## Описание


Интерфейс IPivotEvaluatorTotalsItem
 содержит свойства элемента итогов, рассчитываемых на определенном уровне
 измерения таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BaseLevel](IPivotEvaluatorTotalsItem.BaseLevel.htm)
		 Свойство BaseLevel
		 определяет индекс уровня [измерения](IPivotEvaluatorTotalsItem.Dim.htm),
		 на котором производится расчет итогов.


		 ![](../../Property_Image.gif)
		 [Dim](IPivotEvaluatorTotalsItem.Dim.htm)
		 Свойство Dim возвращает
		 измерение, по которому производится расчет итогов.


		 ![](../../Property_Image.gif)
		 [TotalTypes](IPivotEvaluatorTotalsItem.TotalTypes.htm)
		 Свойство TotalTypes
		 определяет тип итогов, рассчитываемых на заданном уровне измерения.
