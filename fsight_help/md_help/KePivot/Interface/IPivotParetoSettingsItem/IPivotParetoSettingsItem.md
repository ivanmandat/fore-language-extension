# IPivotParetoSettingsItem


Сборка: Pivot;


## Описание


Интерфейс IPivotParetoSettingsItem
 предназначен для работы c настройками отбора минимальных/максимальных
 значений при использовании анализа «[Первые/Последние](UiAnalyticalArea.chm::/Analysis/distribution.htm#distibution_firstlast)».


## Иерархия наследования


           IPivotParetoSettingsItem


## Комментарии


Для получения объекта данного типа используйте [IPivotParetoSettings.BottomSettings](../IPivotParetoSettings/IPivotParetoSettings.BottomSettings.htm),
 [IPivotParetoSettings.TopSettings](../IPivotParetoSettings/IPivotParetoSettings.TopSettings.htm).


Анализ «[Первые/Последние](UiAnalyticalArea.chm::/Analysis/distribution.htm#distibution_firstlast)»
 применяется только к одной строке или столбцу.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Display](IPivotParetoSettingsItem.Display.htm)
		 Свойство Display определяет,
		 отображать ли ячейки столбца/строки с максимальными/минимальными
		 значениями.


		 ![](../../Property_Image.gif)
		 [Kind](IPivotParetoSettingsItem.Kind.htm)
		 Свойство Kind определяет
		 тип расчета значений ячеек строки/столбца.


		 ![](../../Property_Image.gif)
		 [Threshold](IPivotParetoSettingsItem.Threshold.htm)
		 Свойство Threshold
		 определяет пороговое значение, относительно которого будет производиться
		 отбор значений ячеек.
