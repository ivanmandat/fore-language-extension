# IPivotParetoSettingsItem.Display


## Синтаксис


Display: Boolean;


## Описание


Свойство Display определяет,
 отображать ли ячейки столбца/строки с максимальными/минимальными значениями.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Отображать ячейки;


	- False. Не отображать
	 ячейки.
