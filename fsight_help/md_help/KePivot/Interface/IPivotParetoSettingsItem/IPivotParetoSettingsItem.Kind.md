# IPivotParetoSettingsItem.Kind


## Синтаксис


Kind: [PivotParetoKind](../../Enums/PivotParetoKind.htm);


## Описание


Свойство Kind определяет тип
 расчета значений ячеек строки/столбца.


## Комментарии


По умолчанию свойство Kind принимает
 значение [PivotParetoKind.Percent](../../Enums/PivotParetoKind.htm).


Свойство актуально, если свойство [IPivotParetoSettingsItem.Display](IPivotParetoSettingsItem.Display.htm)
 принимает значение True.
