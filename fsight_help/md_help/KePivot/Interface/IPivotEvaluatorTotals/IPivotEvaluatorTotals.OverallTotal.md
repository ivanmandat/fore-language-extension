# IPivotEvaluatorTotals.OverallTotal


## Синтаксис


OverallTotal: Boolean;


## Описание


Свойство OverallTotal определяет,
 будет ли рассчитываться общий итог.


## Комментарии


Если свойству установлено значение True,
 то при отображении итогов будет отображен общий итог по таблице. Если
 свойству установлено значение False,
 то итоги будут отображены без общего итога.


По умолчанию свойству установлено значение True.
