# IPivotEvaluatorTotals


Сборка: Pivot;


## Описание


Интерфейс IPivotEvaluatorTotals
 содержит свойства и методы для работы с коллекцией итогов, рассчитываемых
 по различным элементам таблицы.


## Иерархия наследования


IPivotEvaluatorTotals


## Комментарии


Для работы с настройками вывода итогов в строку родительского элемента
 используйте интерфейс [IPivotInsteadOwnerTotalsSettings](../IPivotInsteadOwnerTotalsSettings/IPivotInsteadOwnerTotalsSettings.htm).

## Методы

		 [RemoveDimElementTotal](IPivotEvaluatorTotals.RemoveDimElementTotal.htm)
		 Метод RemoveDimElementTotal
