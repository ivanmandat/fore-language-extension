# IPivotEvaluatorTotals.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет признак расчета итогов в таблице.


## Комментарии


Если свойству установлено значение True, то в таблице рассчитываются итоги, заданные в свойствах [ColumnTypes](IPivotEvaluatorTotals.ColumnTypes.htm), [HierarchyTotals](IPivotEvaluatorTotals.HierarchyTotals.htm), [LevelTotals](IPivotEvaluatorTotals.LevelTotals.htm), [RowTypes](IPivotEvaluatorTotals.RowTypes.htm), либо добавленные в коллекцию с помощью метода [Add](IPivotEvaluatorTotals.Add.htm). Если свойству Enabled установить значение False, то итоги рассчитываться не будут, при этом их параметры сброшены не будут.


По умолчанию свойству установлено значение True.
