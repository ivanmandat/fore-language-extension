# IPivotEvaluatorTotals.ColumnTotalsEnabled


## Синтаксис


ColumnTotalsEnabled: Boolean;


## Описание


Свойство ColumnTotalsEnabled
 определяет признак расчёта итогов по столбцам.


## Комментарии


Допустимые значения:


	- True. Рассчитывать итоги
	 по столбцам;


	- False. Не рассчитывать
	 итоги по столбцам.
