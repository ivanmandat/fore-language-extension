# IPivotEvaluatorTotals.RowTotalsEnabled


## Синтаксис


RowTotalsEnabled: Boolean;


## Описание


Свойство RowTotalsEnabled определяет
 признак расчёта итогов по строкам.


## Комментарии


Допустимые значения:


	- True. Рассчитывать итоги
	 по строкам;


	- False. Не рассчитывать
	 итоги по строкам.
