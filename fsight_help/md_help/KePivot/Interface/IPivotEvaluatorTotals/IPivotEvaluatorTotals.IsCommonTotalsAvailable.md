# IPivotEvaluatorTotals.IsCommonTotalsAvailable


## Синтаксис


IsCommonTotalsAvailable: Boolean;


## Описание


Свойство IsCommonTotalsAvailable
 возвращает признак расчёта итогов, настроенных в источнике.


## Комментарии


Свойство возвращает значение True,
 если в свойстве [ColumnTypes](IPivotEvaluatorTotals.ColumnTypes.htm)/[RowTypes](IPivotEvaluatorTotals.RowTypes.htm)
 задано значение [PivotEvaluatorElementType.Common](../../Enums/PivotEvaluatorElementType.htm)
 и в источнике настроен расчёт агрегации по измерениям. В других случаях
 свойство возвращает значение False.
