# IPivotEvaluatorDynamics


## Описание


Интерфейс IPivotEvaluatorDynamics
 содержит свойства для работы с динамикой показателей по различным элементам
 таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [LeftHeader](IPivotEvaluatorDynamics.LeftHeader.htm)
		 Свойство LeftHeader
		 возвращает настройки динамики показателей по строкам таблицы.


		 ![](../../Property_Image.gif)
		 [TopHeader](IPivotEvaluatorDynamics.TopHeader.htm)
		 Свойство TopHeader
		 возвращает настройки динамики показателей по столбцам таблицы.
