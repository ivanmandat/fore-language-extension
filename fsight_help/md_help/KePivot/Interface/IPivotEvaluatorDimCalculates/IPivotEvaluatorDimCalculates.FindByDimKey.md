# IPivotEvaluatorDimCalculates.FindByDimKey


## Синтаксис


FindByDimKey(DimKey: Integer): [IPivotEvaluatorDimCalculatesItem](../IPivotEvaluatorDimCalculatesItem/IPivotEvaluatorDimCalculatesItem.htm);


## Параметры


DimKey - ключ измерения.


## Описание


Метод FindByDimKey осуществляет
 поиск и возвращает коллекцию вычисляемых элементов, созданных по измерению,
 ключ которого передается посредством параметра DimKey.
