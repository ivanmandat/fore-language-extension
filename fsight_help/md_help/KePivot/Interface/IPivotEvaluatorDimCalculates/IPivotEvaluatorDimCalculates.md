# IPivotEvaluatorDimCalculates


## Описание


Интерфейс IPivotEvaluatorDimCalculates
 содержит свойства и методы для работы с коллекцией вычисляемых элементов
 по измерениям таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPivotEvaluatorDimCalculates.Count.htm)
		 Свойство Count возвращает
		 количество измерений, по которым могут быть созданы вычисляемые
		 элементы.


		 ![](../../Property_Image.gif)
		 [Item](IPivotEvaluatorDimCalculates.Item.htm)
		 Свойство Item возвращает
		 коллекцию вычисляемых элементов, созданных по измерению, индекс
		 которого передается посредством параметра Index.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Clear](IPivotEvaluatorDimCalculates.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех вычисляемых элементов, созданных по какому либо
		 измерению.


		 ![](../../Sub_Image.gif)
		 [FindByDimKey](IPivotEvaluatorDimCalculates.FindByDimKey.htm)
		 Метод FindByDimKey
		 осуществляет поиск измерения и возвращает коллекцию вычисляемых
		 элементов по данному измерению, в случае удачного поиска.
