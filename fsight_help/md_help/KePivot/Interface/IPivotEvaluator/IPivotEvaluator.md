# IPivotEvaluator


## Описание


Интерфейс IPivotEvaluator содержит
 свойства для работы с вычисляемыми показателями таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DimCalculates](IPivotEvaluator.DimCalculates.htm)
		 Свойство DimCalculates
		 возвращает объект, позволяющий работать с вычисляемыми элементами
		 по измерениям таблицы.


		 ![](../../Property_Image.gif)
		 [Dynamics](IPivotEvaluator.Dynamics.htm)
		 Свойство Dynamics возвращает
		 объект, позволяющий настраивать просмотр динамики изменения показателей
		 измерений.


		 ![](../../Property_Image.gif)
		 [Totals](IPivotEvaluator.Totals.htm)
		 Свойство Totals возвращает
		 объект, позволяющий работать с коллекцией итогов таблицы.
