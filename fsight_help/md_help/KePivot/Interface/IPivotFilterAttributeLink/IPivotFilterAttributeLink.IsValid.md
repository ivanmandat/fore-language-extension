# IPivotFilterAttributeLink.IsValid


## Синтаксис


IsValid: Boolean;


## Описание


Свойство IsValid возвращает
 признак работоспособности связи атрибутов.


## Комментарии


Свойство возвращает значение True,
 если связь между атрибутами установлена и False,
 если в соответствии с заданными значениями свойств связь между атрибутами
 установлена быть не может.
