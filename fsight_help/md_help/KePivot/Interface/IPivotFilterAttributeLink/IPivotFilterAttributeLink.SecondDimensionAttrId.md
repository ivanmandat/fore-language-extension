# IPivotFilterAttributeLink.SecondDimensionAttrId


## Синтаксис


SecondDimensionAttrId: String;


## Описание


Свойство SecondDimensionAttrId
 определяет идентификатор атрибута второго связываемого измерения.


## Комментарии


В качестве значения свойства SecondDimensionAttrId
 необходимо указать идентификатор атрибута измерения, которое задано
 в свойстве [SecondDimensionKey](IPivotFilterAttributeLink.SecondDimensionKey.htm).


Примечание.
 Значение свойства чувствительно к регистру. Идентификатор должен указываться
 заглавными буквами.
