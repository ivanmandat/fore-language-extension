# IPivotFilterAttributeLink.FirstDimensionAttrId


## Синтаксис


FirstDimensionAttrId: String;


## Описание


Свойство FirstDimensionAttrId
 определяет идентификатор атрибута первого связываемого измерения.


## Комментарии


В качестве значения свойства FirstDimensionAttrId
 необходимо указать идентификатор атрибута измерения, которое задано
 в свойстве [FirstDimensionKey](IPivotFilterAttributeLink.FirstDimensionKey.htm).


Примечание.
 Значение свойства чувствительно к регистру. Идентификатор должен указываться
 заглавными буквами.
