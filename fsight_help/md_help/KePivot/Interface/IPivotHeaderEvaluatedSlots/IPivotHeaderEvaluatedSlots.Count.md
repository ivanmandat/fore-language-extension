# IPivotHeaderEvaluatedSlots.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает
 количество столбцов/строк в коллекции.


## Комментарии


Для получения требуемого столбца/строки из коллекции используйте свойство
 [IPivotHeaderEvaluatedSlots.Item](IPivotHeaderEvaluatedSlots.Item.htm).
