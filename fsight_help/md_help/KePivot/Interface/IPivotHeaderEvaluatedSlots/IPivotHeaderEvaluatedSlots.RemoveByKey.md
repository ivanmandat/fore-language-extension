# IPivotHeaderEvaluatedSlots.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ вычисляемого столбца/строки.


## Описание


Метод RemoveByKey удаляет вычисляемый
 столбец/строку с указанным ключом.


## Комментарии


Если удаление выполнено успешно, то метод возвращает значение True, в противном случае - False.
