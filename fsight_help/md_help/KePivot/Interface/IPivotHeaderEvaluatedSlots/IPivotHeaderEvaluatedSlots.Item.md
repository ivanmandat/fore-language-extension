# IPivotHeaderEvaluatedSlots.Item


## Синтаксис


Item(Index: Integer): [IPivotHeaderEvaluatedSlot](../IPivotHeaderEvaluatedSlot/IPivotHeaderEvaluatedSlot.htm);


## Параметры


Index. Индекс вычисляемого
 столбца/строки.


## Описание


Свойство Item возвращает
 вычисляемый столбец/строку с указанным индексом.


## Комментарии


Индексация столбцов/строк сквозная и начинается с нуля. Для получения
 количества столбцов/строк в коллекции используйте свойство [IPivotHeaderEvaluatedSlots.Count](IPivotHeaderEvaluatedSlots.Count.htm).
