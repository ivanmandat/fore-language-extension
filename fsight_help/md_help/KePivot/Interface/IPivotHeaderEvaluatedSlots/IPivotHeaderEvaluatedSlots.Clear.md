# IPivotHeaderEvaluatedSlots.Clear


## Синтаксис


Clear;


## Описание


Метод Clear очищает коллекцию
 столбцов/строк.


## Комментарии


Для удаления определенного столбца/строки из коллекции используйте методы
 [IPivotHeaderEvaluatedSlots.Remove](IPivotHeaderEvaluatedSlots.Remove.htm)
 и [IPivotHeaderEvaluatedSlots.RemoveByKey](IPivotHeaderEvaluatedSlots.RemoveByKey.htm).

 вычисляемого столбца в боковике таблицы](../../Samples/EvaluatedSlot.htm)».
