# IPivotHeaderEvaluatedSlots.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс вычисляемого
 столбца/строки.


## Описание


Метод Remove удаляет вычисляемый
 столбец/строку с указанным индексом.


## Комментарии


Если удаление выполнено успешно, то метод возвращает значение True, в противном случае - False.


Индексация столбцов/строк сквозная и начинается с нуля. Для получения
 количества столбцов/строк в коллекции используйте свойство [IPivotHeaderEvaluatedSlots.Count](IPivotHeaderEvaluatedSlots.Count.htm).
