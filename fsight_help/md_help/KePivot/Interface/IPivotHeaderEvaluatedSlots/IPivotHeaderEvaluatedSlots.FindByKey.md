# IPivotHeaderEvaluatedSlots.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IPivotHeaderEvaluatedSlot](../IPivotHeaderEvaluatedSlot/IPivotHeaderEvaluatedSlot.htm);


## Параметры


Key. Ключ столбца/строки.


## Описание


Метод FindByKey выполняет
 поиск столбца/строки с указанным ключом в коллекции.


## Комментарии


Если столбец/стока с указанным ключом не найдены, то метод возвращает
 значение Null.
