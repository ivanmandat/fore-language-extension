# IPivotHeaderEvaluatedSlots.Add


## Синтаксис


Add: [IPivotHeaderEvaluatedSlot](../IPivotHeaderEvaluatedSlot/IPivotHeaderEvaluatedSlot.htm);


## Описание


Метод Add добавляет вычисляемый
 столбец/строку в коллекцию.


## Комментарии


Для работы с формулой столбца/строки приведите объект, возвращаемый
 данным методом, к интерфейсу [IDataAreaTransformationsGetter](ForeSys.chm::/Interface/IDataAreaTransformationsGetter/IDataAreaTransformationsGetter.htm).


## Пример


Использование метода приведено в разделе «[Создание
 вычисляемого столбца в боковике таблицы](../../Samples/EvaluatedSlot.htm)».
