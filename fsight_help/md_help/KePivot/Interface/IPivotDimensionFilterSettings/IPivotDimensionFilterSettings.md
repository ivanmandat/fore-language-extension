# IPivotDimensionFilterSettings


Сборка: Pivot;


## Описание


Интерфейс IPivotDimensionFilterSettings
 используется для работы с настройками специальной фильтрации данных по
 измерению.


## Иерархия наследования


           IPivotDimensionFilterSettings


## Комментарии


Для получения настроек специальной фильтрации используйте свойство [IPivotDimensionSettingsEx.Filter](../IPivotDimensionSettingsEx/IPivotDimensionSettingsEx.Filter.htm).


Задавать параметры фильтрации данных возможно с помощью команды [FilterSetup](UiLib.chm::/Interface/IUiCommandTarget/Command/Command_FilterSetup.htm) плагина
 [Express](UiLib.chm::/Interface/IUiCommandTarget/IUiCommandTarget.Execute.htm).
