# IPivotDimensionFilterSettings.UseCondition


## Синтаксис


UseCondition: Boolean;


## Описание


Свойство UseCondition определяет,
 будет ли применяться специальная фильтрация данных по измерению.


## Комментарии


Допустимые значения:


	- True. Применять специальную
	 фильтрацию данных. Для задания условие скрытия данных используйте
	 свойство [IPivotDimensionFilterSettings.Condition](IPivotDimensionFilterSettings.Condition.htm).


	- False. Значение по умолчанию.
	 Не применять специальную фильтрацию данных.
