# IPivotTable.IsHighlighted


## Синтаксис


IsHighlighted(Row: Integer; Column: Integer): Boolean;


## Параметры


Row. Индекс строки ячейки;


Column. Индекс столбца ячейки.


## Описание


Свойство IsHighlighted возвращает
 признак подсветки данных в ячейке, значение которой соответствует какому-либо
 установленному условию.


## Комментарии


Индекс строки и столбца ячейки передается посредством параметров Row и Column
 соответственно.
