# IPivotTable.Highlight


## Синтаксис


Highlight(Row: Integer; Column: Integer; HighlightIndex:
 Integer): Boolean;


## Параметры


Row. Индекс строки ячейки;


Column. Индекс столбца ячейки;


HighlightIndex. Индекс [условия
 подсветки данных](../IPivotHighlightItems/IPivotHighlightItems.Item.htm), на соответствие с которым необходимо проверить значение
 указанной ячейки.


## Описание


Свойство Highlight возвращает
 признак подсветки данных в ячейке, значение которой соответствует указанному
 условию.


## Комментарии


Индекс строки и столбца ячейки передается посредством параметров Row и Column
 соответственно.
