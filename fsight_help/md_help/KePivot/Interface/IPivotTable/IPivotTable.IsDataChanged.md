# IPivotTable.IsDataChanged


## Синтаксис


IsDataChanged(Value: [IDimSelectionSet](KeDims.chm::/interface/IDimSelectionSet/IDimSelectionSet.htm)):
 Boolean;


## Параметры


Value. Отметка элементов, по
 которым необходимо проверить наличие измененных данных.


## Описание


Свойство IsDataChanged возвращает
 признак наличия измененных данных по элементам, входящим в указанную отметку.


## Комментарии


Свойство возвращает значение True,
 если имеются измененные данные хотя бы по одному элементу любого измерения,
 входящего в отметку.
