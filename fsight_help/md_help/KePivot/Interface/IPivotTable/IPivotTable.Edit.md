# IPivotTable.Edit


## Синтаксис


Edit;


## Описание


Метод Edit переводит таблицу
 в режим редактирования данных.


## Комментарии


Для определения признака возможности редактирования таблицы используйте
 свойство [IPivotTable.CanEdit](IPivotTable.CanEdit.htm).


Для выхода из режима редактирования с сохранением изменённых данных
 таблицы используйте метод [IPivotTable.Post](IPivotTable.Post.htm),
 без сохранения изменённых данных - [IPivotTable.Cancel](IPivotTable.Cancel.htm).
