# IPivotTable.PostAttachments


## Синтаксис


PostAttachments(Row: Integer; Column: Integer; Value:
 [ICubeAttachmentsEdited](KeCubes.chm::/Interface/ICubeAttachmentsEdited/ICubeAttachmentsEdited.htm));


## Параметры


Row. Индекс строки ячейки;


Column. Индекс столбца ячейки;


Value. Коллекция вложений.


## Описание


Метод PostAttachments возвращает
 признак изменения коллекции вложений в указанной ячейке таблицы.
