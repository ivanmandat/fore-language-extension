# IPivotTable.HasAttachments


## Синтаксис


HasAttachments(Row: Integer; Column: Integer): Boolean;


## Параметры


Row. Индекс строки ячейки.


Column. Индекс столбца ячейки.


## Описание


Свойство HasAttachments возвращает
 признак наличия вложений для ячейки с указанными координатами.


## Комментарии


Свойство возвращает значение True,
 если для ячейки заданы какие-либо вложения и False,
 если вложений нет или работы с ними [не
 поддерживается](IPivotTable.HasAttachmentsSupport.htm).
