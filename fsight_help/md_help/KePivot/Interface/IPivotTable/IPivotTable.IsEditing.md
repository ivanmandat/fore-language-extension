# IPivotTable.IsEditing


## Синтаксис


IsEditing: Boolean;


## Описание


Свойство IsEditing возвращает
 признак режима редактирования таблицы. Свойство вернет значение True, если был вызван метод [Edit](IPivotTable.Edit.htm).
