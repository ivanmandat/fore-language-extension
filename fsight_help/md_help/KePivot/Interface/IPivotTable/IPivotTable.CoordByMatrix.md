# IPivotTable.CoordByMatrix


## Синтаксис


CoordByMatrix(SourceKey: Integer; MatrixCoord: [IMatrixModelCoord](ForeSys.chm::/Interface/IMatrixModelCoord/IMatrixModelCoord.htm)):
 Array;


## Параметры


SourceKey. Ключ источника данных;


MatrixCoord. Координаты в матрице
 с данными.


## Описание


Свойство CoordByMatrix возвращает
 индекс строки и столбца таблицы по координате в матрице с данными.
