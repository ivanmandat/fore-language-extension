# IPivotTable.DataType


## Синтаксис


DataType(Row: Integer; Column: Integer):
[CubeDataType](KeCubes.chm::/Enums/CubeDataType.htm);


## Параметры


Row. Индекс строки;


Column. Индекс столбца.


## Описание


Свойство DataType возвращает
 тип ячейки.


## Комментарии


Свойство позволяет определить способ формирования данных в ячейке: данные
 получены из источника или являются расчётными.
