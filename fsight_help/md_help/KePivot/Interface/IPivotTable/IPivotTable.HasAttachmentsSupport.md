# IPivotTable.HasAttachmentsSupport


## Синтаксис


HasAttachmentsSupport: Boolean;


## Описание


Свойство HasAttachmentsSupport
 возвращает признак возможности работы с вложениями для таблицы.


## Комментарии


Свойство возвращает значение True,
 если для источника таблицы заданы настройки для [хранения
 вложений](KeCubes.chm::/Interface/IStandardCubeDestination/IStandardCubeDestination.AttachmentsStorage.htm).
