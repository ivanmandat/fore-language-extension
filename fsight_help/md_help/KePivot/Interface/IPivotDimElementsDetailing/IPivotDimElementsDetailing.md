# IPivotDimElementsDetailing


Сборка: Pivot;


## Описание


Интерфейс IPivotDimElementsDetailing
 содержит свойства и методы для работы с коллекцией детализаций элементов
 измерений.


## Иерархия наследования


           IPivotDimElementsDetailing


## Комментарии


Для получения объекта данного типа используйте свойство [IPivot.DimElementsDetailing](../IPivot/IPivot.DimElementsDetailing.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPivotDimElementsDetailing.Count.htm)
		 Свойство Count возвращает
		 количество детализаций в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IPivotDimElementsDetailing.Item.htm)
		 Свойство Item возвращает
		 детализацию по индексу элемента измерения в коллекции.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IPivotDimElementsDetailing.Add.htm)

		 [Clear](IPivotDimElementsDetailing.Clear.htm)
		 Метод Clear осуществляет

		 ![](../../Sub_Image.gif)
		 [Remove](IPivotDimElementsDetailing.Remove.htm)
		 Метод Remove осуществляет
