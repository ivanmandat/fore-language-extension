# IPivotDimElementsDetailing.Clear


## Синтаксис


Clear;


## Описание


Метод Clear осуществляет очистку
 коллекции детализаций.


## Комментарии


Для добавления детализации в конец коллекции используйте метод [IPivotDimElementsDetailing.Add](IPivotDimElementsDetailing.Add.htm).


Для удаления детализации по индексу используйте метод [IPivotDimElementsDetailing.Remove](IPivotDimElementsDetailing.Remove.htm).
