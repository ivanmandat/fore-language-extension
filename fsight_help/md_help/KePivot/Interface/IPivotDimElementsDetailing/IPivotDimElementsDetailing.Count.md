# IPivotDimElementsDetailing.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 детализаций в коллекции.


## Комментарии


Для получения детализации по индексу элемента измерения в коллекции
 используйте свойство [IPivotDimElementsDetailing.Item](IPivotDimElementsDetailing.Item.htm).
