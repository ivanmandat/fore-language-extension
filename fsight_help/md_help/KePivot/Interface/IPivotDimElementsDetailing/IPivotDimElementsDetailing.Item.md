# IPivotDimElementsDetailing.Item


## Синтаксис


Item (Index: Integer): [IPivotDimElementsDetailingItem](../IPivotDimElementsDetailingItem/IPivotDimElementsDetailingItem.htm);


## Параметры


Index.
 Индекс детализации.


## Описание


Свойство Item возвращает детализацию
 по индексу элемента измерения в коллекции.


## Комментарии


Для получения количества детализаций в коллекции используйте свойство
 [IPivotDimElementsDetailing.Count](IPivotDimElementsDetailing.Count.htm).
