# IPivotDimElementsDetailing.Remove


## Синтаксис


Remove (Index: Integer);


## Параметры


Index. Индекс детализации.


## Описание


Метод Remove осуществляет удаление
 детализации по индексу.


## Комментарии


Для добавления детализации в конец коллекции используйте метод [IPivotDimElementsDetailing.Add](IPivotDimElementsDetailing.Add.htm).


Для очищения коллекции детализаций используйте метод [IPivotDimElementsDetailing.Clear](IPivotDimElementsDetailing.Clear.htm).
