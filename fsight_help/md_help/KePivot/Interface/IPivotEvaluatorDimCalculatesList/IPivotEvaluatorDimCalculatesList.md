# IPivotEvaluatorDimCalculatesList


## Описание


Интерфейс IPivotEvaluatorDimCalculatesList
 содержит свойства коллекции вычисляемых показателей, созданных относительно
 какого-либо элемента измерения.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPivotEvaluatorDimCalculatesList.Count.htm)
		 Свойство Count возвращает
		 количество вычисляемых показателей, созданных относительно элемента
		 [Element](IPivotEvaluatorDimCalculatesList.Element.htm).


		 ![](../../Property_Image.gif)
		 [DimCalculatesItem](IPivotEvaluatorDimCalculatesList.DimCalculatesItem.htm)
		 Свойство DimCalculatesItem
		 возвращает все вычисляемые показатели, созданные для измерения,
		 в котором расположен элемент [Element](IPivotEvaluatorDimCalculatesList.Element.htm).


		 ![](../../Property_Image.gif)
		 [Element](IPivotEvaluatorDimCalculatesList.Element.htm)
		 Свойство Element возвращает
		 индекс элемента измерения, относительно которого созданы вычисляемые
		 показатели.


		 ![](../../Property_Image.gif)
		 [Item](IPivotEvaluatorDimCalculatesList.Item.htm)
		 Свойство Item возвращает
		 индекс вычисляемого показателя в коллекции показателей всего измерения.
