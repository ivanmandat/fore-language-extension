# IPivotHeader.Clear


## Синтаксис


Clear;


## Описание


Метод Clear осуществляет удаление
 всех измерений из выбранного заголовка таблицы.


## Комментарии


Актуально для измерений, расположенных по [столбцам](../IPivot/IPivot.TopHeader.htm),
 либо [строкам](../IPivot/IPivot.LeftHeader.htm) таблицы. Все
 удаляемые измерения помещаются в список фиксированных измерений.
