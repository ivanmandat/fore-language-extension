# IPivotHeader.EvaluatedSlots


## Синтаксис


EvaluatedSlots: [IPivotHeaderEvaluatedSlots](../IPivotHeaderEvaluatedSlots/IPivotHeaderEvaluatedSlots.htm);


## Описание


Свойство EvaluatedSlots
 возвращает коллекцию вычисляемых столбцов, расположенных в заголовке таблицы.


## Комментарии


Большое количество вычисляемых строк и столбцов может привести к снижению
 производительности таблицы из-за расчета формул вычисляемых строк и столбцов.

Использование свойства приведено в разделе «[Создание
 вычисляемого столбца в боковике таблицы](../../Samples/EvaluatedSlot.htm)».
