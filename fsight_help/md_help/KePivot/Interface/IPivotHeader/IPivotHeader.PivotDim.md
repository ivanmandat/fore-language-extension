# IPivotHeader.PivotDim


## Синтаксис


PivotDim(Position: Integer): [IPivotDimension](../IPivotDimension/IPivotDimension.htm);


## Параметры


Position. Позиция, в которой
 расположено измерение в заголовке.


## Описание


Свойство PivotDim возвращает
 измерение, расположенное в заданной позиции заголовка таблицы.
