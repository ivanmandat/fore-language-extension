# IPivotHeader.DeleteDim


## Синтаксис


DeleteDim(Dim: [IDimInstance](KeDims.chm::/interface/IDimInstance/IDimInstance.htm)):
 Boolean;


## Параметры


Dim - измерение, которое необходимо
 удалить.


## Описание


Метод DeleteDim осуществляет
 удаление указанного измерения из выбранного заголовка таблицы.


## Комментарии


Актуально для измерений, расположенных по [столбцам](../IPivot/IPivot.TopHeader.htm),
 либо [строкам](../IPivot/IPivot.LeftHeader.htm) таблицы. Удаляемое
 измерение помещаются в список фиксированных измерений.
