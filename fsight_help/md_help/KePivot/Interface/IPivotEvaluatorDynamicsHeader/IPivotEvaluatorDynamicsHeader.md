# IPivotEvaluatorDynamicsHeader


## Описание


Интерфейс IPivotEvaluatorDynamicsHeader
 содержит свойства для построения динамики показателей по выбранному элементу
 таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AbsGrowth](IPivotEvaluatorDynamicsHeader.AbsGrowth.htm)
		 Свойство AbsGrowth
		 определяет признак расчета абсолютного прироста значений.


		 ![](../../Property_Image.gif)
		 [GrowthRate](IPivotEvaluatorDynamicsHeader.GrowthRate.htm)
		 Свойство GrowthRate
		 определяет признак расчета темпа роста значений.


		 ![](../../Property_Image.gif)
		 [IncreaseRate](IPivotEvaluatorDynamicsHeader.IncreaseRate.htm)
		 Свойство IncreaseRate
		 определяет признак расчета темпа прироста значений.


		 ![](../../Property_Image.gif)
		 [Slot](IPivotEvaluatorDynamicsHeader.Slot.htm)
		 Свойство Slot определяет
		 измерение, по которому проводится построение динамики показателей.
