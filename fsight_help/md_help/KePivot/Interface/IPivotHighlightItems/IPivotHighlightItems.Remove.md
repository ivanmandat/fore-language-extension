# IPivotHighlightItems.Remove


## Синтаксис


Remove(Index: Integer);


## Параметры


Index - индекс условия подсветки
 данных.


## Описание


Метод Remove осуществляет удаление
 условия подсветки данных таблицы, индекс которого передается посредством
 параметра Index.
