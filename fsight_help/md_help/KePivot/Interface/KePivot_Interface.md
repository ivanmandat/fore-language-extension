# Интерфейсы сборки Pivot


		 Интерфейс
		 Краткое описание


		 ![](../Interface_Image.gif)
		 [IPivot](IPivot/IPivot.htm)
		 Интерфейс IPivot содержит
		 свойства и методы для работы объектом, представляющим собой основу
		 для построения таблицы с данными.

		 [IPivotClass](IPivotClass/IPivotClass.htm)
		 Интерфейс IPivotClass
		 содержит статические свойства и методы класса [PivotClass](../Class/PivotClass/PivotClass.htm).

		 измерений.
