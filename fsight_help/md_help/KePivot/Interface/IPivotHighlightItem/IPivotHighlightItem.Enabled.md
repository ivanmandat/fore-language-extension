# IPivotHighlightItem.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет признак применения данного условия подсветки данных в таблице.


## Комментарии


Если свойству установлено значение True, то в таблице будет применена подсветка данных, удовлетворяющих данному условию. Параметры условия определяются в свойствах данного интерфейса.
