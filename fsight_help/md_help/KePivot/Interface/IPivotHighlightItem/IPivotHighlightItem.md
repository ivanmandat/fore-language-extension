# IPivotHighlightItem


## Описание


Интерфейс IPivotHighlightItem
 содержит свойства условия подсветки данных в таблице.


## Комментарии


Условие подсветки данных можно получить, используя свойства и методы
 интерфейса [IPivotHighlightItems](../IPivotHighlightItems/IPivotHighlightItems.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Area](IPivotHighlightItem.Area.htm)
		 Свойство Area определяет
		 область, для которой применяется подсветка данных.


		 ![](../../Property_Image.gif)
		 [ConditionType](IPivotHighlightItem.ConditionType.htm)
		 Свойство ConditionType
		 определяет тип условия, по которому будут отбираться подсвечиваемые
		 данные.


		 ![](../../Property_Image.gif)
		 [ConditionValueA](IPivotHighlightItem.ConditionValueA.htm)
		 Свойство ConditionValueA
		 определяет первое значение, используемое при определении подсвечиваемых
		 данных.


		 ![](../../Property_Image.gif)
		 [ConditionValueB](IPivotHighlightItem.ConditionValueB.htm)
		 Свойство ConditionValueB
		 определяет второе значение, используемое при определении подсвечиваемых
		 данных.


		 ![](../../Property_Image.gif)
		 [Enabled](IPivotHighlightItem.Enabled.htm)
		 Свойство Enabled определяет
		 признак применения данного условия подсветки данных в таблице.


		 ![](../../Property_Image.gif)
		 [Number](IPivotHighlightItem.Number.htm)
		 Свойство Number возвращает
		 номер условия.


		 ![](../../Property_Image.gif)
		 [Range](IPivotHighlightItem.Range.htm)
		 Свойство Range возвращает
		 диапазон, в котором будет осуществляться подсветка данных.
