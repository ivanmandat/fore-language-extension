# IPivotHeaderFilterSettings


Сборка: Pivot;


## Описание


Интерфейс IPivotHeaderFilterSettings
 используется для работы с настройками фильтрации для заголовков строк/столбцов.


## Иерархия наследования


           [IPivotFilterSettings](../IPivotFilterSettings/IPivotFilterSettings.htm)


           IPivotHeaderFilterSettings


## Комментарии


Для получения объекта данного типа используйте [IPivotHeaderSettings.Filter](../IPivotHeaderSettings/IPivotHeaderSettings.Filter.htm).

		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ExcludeElements](IPivotHeaderFilterSettings.ExcludeElements.htm)
		 Свойство ExcludeElements
		 определяет, будут ли исключаться элементы из настроек фильтрации.


		 ![](../../Property_Image.gif)
		 [ExcludeSelection](IPivotHeaderFilterSettings.ExcludeSelection.htm)
		 Свойство ExcludeSelection

		 ![](../../Property_Image.gif)
		 [ConditionValueA](../IPivotFilterSettings/IPivotFilterSettings.ConditionValueA.htm)
		 Свойство ConditionValueA

		 ![](../../Property_Image.gif)

		 [FilterSources](../IPivotFilterSettings/IPivotFilterSettings.FilterSources.htm)

		 ![](../../Property_Image.gif)
