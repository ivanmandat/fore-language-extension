# IPivotDimNamings


Сборка: Pivot;


## Описание


Интерфейс IPivotDimNamings содержит
 свойства и методы для работы с коллекцией наименований измерения.


## Иерархия наследования


           IPivotDimNamings


## Комментарии


Для работы с наименованием элементов измерения используйте свойство
 [IPivotDimNaming.Expression](../IPivotDimNaming/IPivotDimNaming.Expression.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPivotDimNamings.Count.htm)
		 Свойство Count возвращает
		 количество наименований в коллекции.


		 ![](../../Property_Image.gif)
		 [CustomNaming](IPivotDimNamings.CustomNaming.htm)
		 Свойство CustomNaming
		 возвращает пользовательское наименование элемента измерения.


		 ![](../../Property_Image.gif)
		 [Item](IPivotDimNamings.Item.htm)
		 Свойство Item возвращает
		 наименование из коллекции по заданному индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IPivotDimNamings.Add.htm)
		 Метод Add осуществляет
		 добавление наименования в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](IPivotDimNamings.Clear.htm)
		 Метод Clear осуществляет
		 очистку коллекции наименований.


		 ![](../../Sub_Image.gif)
		 [Move](IPivotDimNamings.Move.htm)
		 Метод Move осуществляет
		 перемещение наименования в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IPivotDimNamings.Remove.htm)
		 Метод Remove осуществляет
		 удаление наименование из коллекции по заданному индексу.
