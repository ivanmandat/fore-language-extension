# IPivotTableCellEventArgs.Cancel


## Синтаксис


Cancel: Boolean;


## Описание


Свойство Cancel определяет признак
 возможности изменения значения ячейки.


## Комментарии


Если свойству установить значение True,
 то для ячейки будет установлена отметка о запрете редактирования.
