# IPivotDimElementsDetailingItem


Сборка: Pivot;


## Описание


Интерфейс IPivotDimElementsDetailingItem
 содержит свойства и методы для работы с детализацией элементов измерения.


## Иерархия наследования


           IPivotDimElementsDetailingItem


## Комментарии


Для получения объекта данного типа используйте свойство [IPivotDimElementsDetailing.Item](../IPivotDimElementsDetailing/IPivotDimElementsDetailing.Item.htm).


Для создания объекта данного типа используйте свойство [IPivotDimElementsDetailing.Add](../IPivotDimElementsDetailing/IPivotDimElementsDetailing.Add.htm).
