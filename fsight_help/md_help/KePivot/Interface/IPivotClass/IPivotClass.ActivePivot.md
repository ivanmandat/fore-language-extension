# IPivotClass.ActivePivot


## Синтаксис


ActivePivot: [IPivot](../IPivot/IPivot.htm);


## Описание


Свойство ActivePivot возвращает
 активную таблицу отчета.


## Комментарии


Свойство ActivePivot может использоваться
 для получения текущей таблицы при обработке каких-либо [событий](../../Class/PivotEvents/PivotEvents.htm).
