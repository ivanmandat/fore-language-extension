# IPivotSortItem.UseTransformationValues


## Синтаксис


UseTransformationValues: Boolean;


## Описание


Свойство UseTransformationValues
 определяет, производится ли сортировка по формуле.


## Комментарии


Допустимые значения:


	- True. Сортировка производится
	 по формуле;


	- False. По умолчанию.
	 Сортировка производится [по
	 атрибуту](IPivotSortItem.AttributeKey.htm).
