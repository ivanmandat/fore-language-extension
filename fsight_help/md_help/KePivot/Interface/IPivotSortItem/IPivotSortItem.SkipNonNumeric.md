# IPivotSortItem.SkipNonNumeric


## Синтаксис


SkipNonNumeric: Boolean;


## Описание


Свойство SkipNonNumeric определяет,
 будут ли располагаться нечисловые значения в конце таблицы.


## Комментарии


Допустимые значения:


	- True. Нечисловые значения
	 будут располагаться в конце таблицы;


	- False. По умолчанию.
	 Нечисловые значения располагаются согласно своему месту в таблице.
