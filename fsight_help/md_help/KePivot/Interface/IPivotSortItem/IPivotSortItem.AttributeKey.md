# IPivotSortItem.AttributeKey


## Синтаксис


AttributeKey: Integer;


## Описание


Свойство AttributeKey определяет
 ключ атрибута измерения.


## Комментарии


Для использования данного свойства сначала определите ключ измерения
 в свойстве [IPivotSortItem.DimKey](IPivotSortItem.DimKey.htm).


Для использования свойства необходимо, чтобы свойство [IPivotSortItem.UseTransformationValues](IPivotSortItem.UseTransformationValues.htm)
 принимало значение False.
