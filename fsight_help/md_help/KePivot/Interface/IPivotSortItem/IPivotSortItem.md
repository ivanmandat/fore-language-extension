# IPivotSortItem


Сборка: Pivot;


## Описание


Интерфейс IPivotSortItem используется
 для работы с сортировкой таблицы.


## Иерархия наследования


IPivotSortItem


## Комментарии


Для получения объекта данного типа используйте [IPivotSorter.Item](../IPivotSorter/IPivotSorter.Item.htm).


Для создания объекта данного типа используйте [IPivotSorter.Add](../IPivotSorter/IPivotSorter.Add.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AttributeKey](IPivotSortItem.AttributeKey.htm)
		 Свойство AttributeKey определяет
		 ключ атрибута измерения.


		 ![](../../Property_Image.gif)
		 [DimKey](IPivotSortItem.DimKey.htm)
		 Свойство DimKey определяет
		 ключ измерения, по которому будет осуществляться сортировка.


		 ![](../../Property_Image.gif)
		 [Direction](IPivotSortItem.Direction.htm)
		 Свойство Direction
		 определяет направление сортировки.


		 ![](../../Property_Image.gif)
		 [IndexInHeader](IPivotSortItem.IndexInHeader.htm)
		 Свойство IndexInHeader
		 определяет в заголовке индекс элемента, по которому будет осуществляться
		 сортировка.


		 ![](../../Property_Image.gif)
		 [Kind](IPivotSortItem.Kind.htm)
		 Свойство Kind определяет
		 вид сортировки.


		 ![](../../Property_Image.gif)
		 [Selection](IPivotSortItem.Selection.htm)
		 Свойство Selection определяет
		 элемент отметки, по которому будет осуществляться сортировка.


		 ![](../../Property_Image.gif)
		 [SkipNonNumeric](IPivotSortItem.SkipNonNumeric.htm)
		 Свойство SkipNonNumeric определяет,
		 будут ли располагаться нечисловые значения в конце таблицы.


		 ![](../../Property_Image.gif)
		 [UseTransformationValues](IPivotSortItem.UseTransformationValues.htm)
		 Свойство UseTransformationValues
		 определяет, производится ли сортировка по формуле.
