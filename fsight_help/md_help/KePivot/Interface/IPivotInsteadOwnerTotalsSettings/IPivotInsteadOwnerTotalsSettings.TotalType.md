# IPivotInsteadOwnerTotalsSettings.TotalType


## Синтаксис


TotalType: [PivotEvaluatorElementType](../../Enums/PivotEvaluatorElementType.htm);


## Описание


Свойство TotalType определяет
 тип итогов, выводящихся в строку родительского элемента.


## Комментарии


В качестве значения данного параметра необходимо задавать десятичное
 число, соответствующее виду итогов, примененных в данной таблице. Значения
 итогов, которые можно рассчитать, содержатся в перечислении [PivotEvaluatorElementType](../../Enums/PivotEvaluatorElementType.htm).
