# IPivotInsteadOwnerTotalsSettings.NamingMode


## Синтаксис


NamingMode: [PivotTotalParentNamingMode](../../Enums/PivotTotalParentNamingMode.htm);


## Описание


Свойство NamingMode определяет
 вид наименования элементов, выводимых в строку родительского элемента.


## Комментарии


В качестве значения данного параметра необходимо задавать число, соответствующее
 типу наименования. Значения типов содержатся в перечислении [PivotTotalParentNamingMode](../../Enums/PivotTotalParentNamingMode.htm).
