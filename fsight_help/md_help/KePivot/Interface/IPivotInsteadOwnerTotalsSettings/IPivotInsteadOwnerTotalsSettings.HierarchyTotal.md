# IPivotInsteadOwnerTotalsSettings.HierarchyTotal


## Синтаксис


HierarchyTotal: Boolean;


## Описание


Свойство HierarchyTotal определяет,
 будут ли выводиться значения итогов по иерархии в строку родительского
 элемента.


## Комментарии


Допустимые значения:


	- False. Значения итогов
	 по иерархии не будут выводиться в строку родительского элемента. Значение
	 по умолчанию;


	- True. Значения итогов
	 по иерархии будут выводиться в строку родительского элемента.
