# IPivotInsteadOwnerTotalsSettings


Сборка: Pivot;


## Описание


Интерфейс IPivotInsteadOwnerTotalsSettings
 используется работы с настройками вывода итогов в строку родительского
 элемента.


## Иерархия наследования


           IPivotInsteadOwnerTotalsSettings


## Комментарии


Для работы с коллекцией итогов таблицы используйте свойства и методы
 интерфейса [IPivotEvaluatorTotals](../IPivotEvaluatorTotals/IPivotEvaluatorTotals.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [HierarchyTotal](IPivotInsteadOwnerTotalsSettings.HierarchyTotal.htm)
		 Свойство HierarchyTotal
		 определяет, будут ли выводиться значения итогов по иерархии в
		 строку родительского элемента.


		 ![](../../Property_Image.gif)
		 [NameSeparator](IPivotInsteadOwnerTotalsSettings.NameSeparator.htm)
		 Свойство NameSeparator
		 определяет вид разделителя между наименованием элемента и методом
		 итога, выводимого в строке родительского элемента.


		 ![](../../Property_Image.gif)
		 [NamingMode](IPivotInsteadOwnerTotalsSettings.NamingMode.htm)
		 Свойство NamingMode
		 определяет вид наименования элементов, выводимых в строку родительского
		 элемента.


		 ![](../../Property_Image.gif)
		 [OverallTotal](IPivotInsteadOwnerTotalsSettings.OverallTotal.htm)
		 Свойство OverallTotal
		 определяет, будут ли выводиться значения общих итогов в строку
		 родительского элемента.


		 ![](../../Property_Image.gif)
		 [TotalType](IPivotInsteadOwnerTotalsSettings.TotalType.htm)
		 Свойство Totaltype
		 определяет тип итогов, выводящихся в строку родительского элемента.
