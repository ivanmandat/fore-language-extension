# IPivotInsteadOwnerTotalsSettings.NameSeparator


## Синтаксис


NameSeparator: String;


## Описание


Свойство NameSeparator определяет
 вид разделителя между наименованием элемента и методом итога, выводимого
 в строке родительского элемента.


## Комментарии


Для использования свойства необходимо, чтобы свойство [IPivotInsteadOwnerTotalsSettings.NamingMode](IPivotInsteadOwnerTotalsSettings.NamingMode.htm)
 принимало значение 2.
