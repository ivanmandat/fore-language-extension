# IPivotTables


Сборка: Pivot;


## Описание


Интерфейс IPivotTables содержит
 свойства для работы с коллекцией вариантов таблиц с данными.


## Комментарии


           IPivotTables


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPivotTables.Count.htm)
		 Свойство Count возвращает
		 количество вариантов таблиц с данными.


		 ![](../../Property_Image.gif)
		 [Item](IPivotTables.Item.htm)
		 Свойство Item возвращает
		 вариант таблицы с данными.


		 ![](../../Property_Image.gif)
		 [ItemByElement](IPivotTables.ItemByElement.htm)
		 Свойство ItemByElement
		 возвращает вариант таблицы с данными по индексу элемента.
