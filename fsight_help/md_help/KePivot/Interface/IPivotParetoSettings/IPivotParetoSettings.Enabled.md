# IPivotParetoSettings.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет,
 применяется ли распределение.


## Комментарии


Допустимые значения:


	- True. Применить распределение
	 в столбце/строке;


	- False. Не применять
	 распределение в столбце/строке.
