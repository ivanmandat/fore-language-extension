# IPivotParetoSettings


Сборка: Pivot;


## Описание


Интерфейс IPivotParetoSettings
 содержит свойства для определения параметров распределения.


## Иерархия наследования


IPivotParetoSettings


## Комментарии


Для получения объекта данного типа используйте [IPivotPareto.ColumnsSettings](../IPivotPareto/IPivotPareto.ColumnsSettings.htm),
 [IPivotPareto.RowsSettings](../IPivotPareto/IPivotPareto.RowsSettings.htm).


Распределение применяется только к одной строке или столбцу.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BottomSettings](IPivotParetoSettings.BottomSettings.htm)
		 Свойство BottomSettings
		 возвращает настройки отбора минимальных значений в строке/столбце
		 при использовании анализа «[Первые/Последние](UiAnalyticalArea.chm::/Analysis/distribution.htm#distibution_firstlast)».


		 ![](../../Property_Image.gif)
		 [ElementIndex](IPivotParetoSettings.ElementIndex.htm)
		 Свойство ElementIndex
		 определяет индекс столбца/строки, для которой применяется распределение.


		 ![](../../Property_Image.gif)
		 [Enabled](IPivotParetoSettings.Enabled.htm)
		 Свойство Enabled определяет,
		 применяется ли распределение.


		 ![](../../Property_Image.gif)
		 [OtherAggregation](IPivotParetoSettings.OtherAggregation.htm)
		 Свойство OtherAggregation
		 определяет способ агрегации, применяемый к тем данным, которые
		 не попадают в результирующий набор распределения.


		 ![](../../Property_Image.gif)
		 [TopSettings](IPivotParetoSettings.TopSettings.htm)
		 Свойство TopSettings
		 возвращает настройки отбора максимальных значений в строке/столбце
		 при использовании анализа «[Первые/Последние](UiAnalyticalArea.chm::/Analysis/distribution.htm#distibution_firstlast)».
