# IPivotParetoSettings.BottomSettings


## Синтаксис


BottomSettings: [IPivotParetoSettingsItem](../IPivotParetoSettingsItem/IPivotParetoSettingsItem.htm);


## Описание


Свойство BottomSettings возвращает
 настройки отбора минимальных значений в строке/столбце при использовании
 анализа «[Первые/Последние](UiAnalyticalArea.chm::/Analysis/distribution.htm#distibution_firstlast)».


## Комментарии


Для получения настроек отбора максимальных значений в строке/столбце
 при использовании анализа «[Первые/Последние](UiAnalyticalArea.chm::/Analysis/distribution.htm#distibution_firstlast)»
 используйте [IPivotParetoSettings.TopSettings](IPivotParetoSettings.TopSettings.htm).
