# IPivotParetoSettings.ElementIndex


## Синтаксис


ElementIndex: Integer;


## Описание


Свойство ElementIndex определяет
 индекс столбца/строки, для которой применяется распределение.


## Комментарии


По умолчанию для параметров анализа по [столбцам](../IPivotPareto/IPivotPareto.ColumnsSettings.htm)
 свойству ElementIndex установлено
 значение «0», по [строкам](../IPivotPareto/IPivotPareto.RowsSettings.htm)
 - «-1».
