# IPivotParetoSettings.OtherAggregation


## Синтаксис


OtherAggregation: [PivotEvaluatorElementType](../../Enums/PivotEvaluatorElementType.htm);


## Описание


Свойство OtherAggregation определяет
 способ агрегации, применяемый к тем данным, которые не попадают в результирующий
 набор распределения.


## Комментарии


По умолчанию для агрегации данных, которые не попали в результирующий
 набор, применяется метод - Сумма.
