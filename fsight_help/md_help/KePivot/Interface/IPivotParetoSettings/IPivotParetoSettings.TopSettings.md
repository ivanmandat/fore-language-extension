# IPivotParetoSettings.TopSettings


## Синтаксис


TopSettings: [IPivotParetoSettingsItem](../IPivotParetoSettingsItem/IPivotParetoSettingsItem.htm);


## Описание


Свойство TopSettings возвращает
 настройки отбора максимальных значений в строке/столбце при использовании
 анализа «[Первые/Последние](UiAnalyticalArea.chm::/Analysis/distribution.htm#distibution_firstlast)».


## Комментарии


Для получения настроек отбора минимальных значений в строке/столбце
 при использовании анализа «[Первые/Последние](UiAnalyticalArea.chm::/Analysis/distribution.htm#distibution_firstlast)»
 используйте [IPivotParetoSettings.BottomSettings](IPivotParetoSettings.BottomSettings.htm).
