# IPivotDimensionSettings.LeafElementsOnly


## Синтаксис


LeafElementsOnly: [TriState](ForeSys.chm::/Enums/TriState.htm);


## Описание


Свойство LeafElementsOnly определяет,
 будут ли отображаться только листьевые элементы измерения.


## Комментарии


Листьевые элементы - это элементы, у которых нет отмеченных дочерних
 элементов.


Для определения признака того, отображаются ли только листьевые элементы,
 используйте [IPivotDimensionSettings.IsLeafElementsOnly](IPivotDimensionSettings.IsLeafElementsOnly.htm).
