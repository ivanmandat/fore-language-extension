# IPivotDimensionSettings.IsLeafElementsOnly


## Синтаксис


IsLeafElementsOnly: Boolean;


## Описание


Свойство IsLeafElementsOnly
 возвращает признак того, отображаются ли только листьевые элементы измерения.


## Комментарии


Допустимые значения:


	- True. Отображаются только
	 листьевые элементы измерения;


	- False. Листьевые элементы
	 измерения не отображаются.


Листьевые элементы - это элементы, у которых нет отмеченных дочерних
 элементов.


Для определения, будут ли отображаться только листьевые элементы измерения,
 используйте [IPivotDimensionSettings.LeafElementsOnly](IPivotDimensionSettings.LeafElementsOnly.htm).
