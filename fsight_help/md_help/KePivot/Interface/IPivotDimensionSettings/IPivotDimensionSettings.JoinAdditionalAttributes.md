# IPivotDimensionSettings.JoinAdditionalAttributes


## Синтаксис


JoinAdditionalAttributes: [TriState](ForeSys.chm::/Enums/TriState.htm);


## Описание


Свойство JoinAdditionalAttributes
 определяет, будут ли объединены дополнительные наименования измерений.


## Комментарии


Для определения признака того, объединяются ли дополнительные наименования,
 используйте [IPivotDimensionSettings.IsJoinAdditionalAttributes](IPivotDimensionSettings.IsJoinAdditionalAttributes.htm).
