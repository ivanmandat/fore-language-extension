# IPivotDimensionSettings


Сборка: Pivot;


## Описание


Интерфейс IPivotDimensionSettings
 используется для работы с настройками измерения источника данных.


## Иерархия наследования


           IPivotDimensionSettings


## Комментарии


Для получения объекта данного типа используйте [IPivotDimension.Settings](../IPivotDimension/IPivotDimension.Settings.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [GetKeepLeafParents](IPivotDimensionSettings.GetKeepLeafParents.htm)
		 Свойство GetKeepLeafParents
		 возвращает признак того, отображаются ли родительские элементы
		 для листьевых элементов.


		 ![](../../Property_Image.gif)
		 [IsJoinAdditionalAttributes](IPivotDimensionSettings.IsJoinAdditionalAttributes.htm)
		 Свойство IsJoinAdditionalAttributes
		 возвращает признак того, объединены ли дополнительные наименования
		 измерений.


		 ![](../../Property_Image.gif)
		 [IsLeafElementsOnly](IPivotDimensionSettings.IsLeafElementsOnly.htm)
		 Свойство IsLeafElementsOnly
		 возвращает признак того, отображаются ли только листьевые элементы
		 измерения.


		 ![](../../Property_Image.gif)
		 [JoinAdditionalAttributes](IPivotDimensionSettings.JoinAdditionalAttributes.htm)
		 Свойство JoinAdditionalAttributes
		 определяет, будут ли объединены дополнительные наименования измерений.


		 ![](../../Property_Image.gif)
		 [KeepLeafParents](IPivotDimensionSettings.KeepLeafParents.htm)
		 Свойство KeepLeafParents
		 определяет, будут ли отображаться родительские элементы для листьевых
		 элементов измерения.


		 ![](../../Property_Image.gif)
		 [LeafElementsOnly](IPivotDimensionSettings.LeafElementsOnly.htm)
		 Свойство LeafElementsOnly
		 определяет, будут ли отображаться только листьевые элементы измерения.
