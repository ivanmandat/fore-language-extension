# IPivotDimensionSettings.GetKeepLeafParents


## Синтаксис


GetKeepLeafParents: Boolean;


## Описание


Свойство GetKeepLeafParents
 возвращает признак того, отображаются ли родительские элементы для листьевых
 элементов.


## Комментарии


Допустимые значения:


	- True. Родительские элементы
	 отображаются;


	- False. Родительские
	 элементы не отображаются.


Листьевые элементы - это элементы, у которых нет отмеченных дочерних
 элементов.


Для определения, будут ли отображаться родительские элементы для листьевых
 элементов измерения, используйте свойство [IPivotDimensionSettings.KeepLeafParents](IPivotDimensionSettings.KeepLeafParents.htm).
