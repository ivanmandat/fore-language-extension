# IPivotCustomDimNaming.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет,
 будут ли для элементов измерения использоваться пользовательские наименования.


## Комментарии


Допустимые значения:


	- True. Для элементов
	 измерения будут использоваться пользовательские наименования;


	- False. Для составления
	 наименований элементов измерения будут использоваться атрибуты справочника.
