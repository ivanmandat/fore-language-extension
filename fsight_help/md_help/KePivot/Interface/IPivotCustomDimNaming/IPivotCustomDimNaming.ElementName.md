# IPivotCustomDimNaming.ElementName


## Синтаксис


ElementName(Element: Integer): String;


## Параметры


Element. Индекс элемента измерения.


## Описание


Свойство ElementName возвращает
 пользовательское наименование элемента по его индексу.


## Комментарии


Для определения шаблона пользовательского наименования элементов используйте
 свойство [IPivotCustomDimNaming.Expression](IPivotCustomDimNaming.Expression.htm).
