# IPivotCustomDimNaming


Сборка: Pivot;


## Описание


Интерфейс IPivotCustomDimNaming
 предназначен для работы с пользовательскими наименованиями элементов измерений.


## Иерархия наследования


           IPivotCustomDimNaming


## Комментарии


Для работы с коллекцией наименований измерения используйте свойства
 и методы интерфейса [IPivotDimNamings](../IPivotDimNamings/IPivotDimNamings.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dimension](IPivotCustomDimNaming.Dimension.htm)
		 Свойство Dimension
		 возвращает измерение, для элементов которого требуется настроить
		 пользовательские наименования.


		 ![](../../Property_Image.gif)
		 [ElementName](IPivotCustomDimNaming.ElementName.htm)
		 Свойство ElementName
		 возвращает пользовательское наименование элемента по его индексу.


		 ![](../../Property_Image.gif)
		 [Enabled](IPivotCustomDimNaming.Enabled.htm)
		 Свойство Enabled определяет,
		 будут ли для элементов измерения использоваться пользовательские
		 наименования.


		 ![](../../Property_Image.gif)
		 [Expression](IPivotCustomDimNaming.Expression.htm)
		 Свойство Expression
		 определяет шаблон пользовательского наименования элементов.
