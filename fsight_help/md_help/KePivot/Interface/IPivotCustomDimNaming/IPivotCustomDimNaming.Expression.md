# IPivotCustomDimNaming.Expression


## Синтаксис


Expression: String;


## Описание


Свойство Expression определяет
 шаблон пользовательского наименования элементов.


## Комментарии


В качестве шаблонов пользовательского наименования элементов используйте
 доступные атрибуты измерения. Например:


	- %NAME%. Наименование
	 элемента;


	- %ID%. Идентификатор
	 элемента и другие.
