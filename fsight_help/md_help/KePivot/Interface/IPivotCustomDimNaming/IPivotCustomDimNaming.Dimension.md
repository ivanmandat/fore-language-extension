# IPivotCustomDimNaming.Dimension


## Синтаксис


Dimension: [IDimInstance](KeDims.chm::/interface/IDimInstance/IDimInstance.htm);


## Описание


Свойство Dimension возвращает
 измерение, для элементов которого требуется настроить пользовательские
 наименования.


## Комментарии


Для получения пользовательского наименования элемента измерения по его
 индексу используйте свойство [IPivotCustomDimNaming.ElementName](IPivotCustomDimNaming.ElementName.htm).
