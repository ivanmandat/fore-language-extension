# IPivotHeaderEvaluatedSlot


Сборка: Pivot;


## Описание


Интерфейс IPivotHeaderEvaluatedSlot
 предназначен для работы с вычисляемым столбцом/строкой в заголовке таблицы.


## Иерархия наследования


           [INamedEntity](KeSom.chm::/Interface/INamedEntity/INamedEntity.htm)


           IPivotHeaderEvaluatedSlot


## Комментарии


Для работы с формулой столбца приведите объект, возвращаемый данным
 методом, к интерфейсу [IDataAreaTransformationsGetter](ForeSys.chm::/Interface/IDataAreaTransformationsGetter/IDataAreaTransformationsGetter.htm).


Для работы со всеми вычисляемыми столбцами/заголовками таблицы используйте
 интерфейс [IPivotHeaderEvaluatedSlots](../IPivotHeaderEvaluatedSlots/IPivotHeaderEvaluatedSlots.htm).
