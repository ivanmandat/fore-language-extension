# IPivotDimensionLevelNumbering


Сборка: Pivot;


## Описание


Интерфейс IPivotDimensionLevelNumbering
 используется для работы с параметрами нумерации измерения/уровня таблицы.


## Иерархия наследования


           IPivotDimensionLevelNumbering


## Комментарии


Для получения объекта данного типа используйте свойства [IPivotDimensionLevelProperties.Numbering](../IPivotDimensionLevelProperties/IPivotDimensionLevelProperties.Numbering.htm).
