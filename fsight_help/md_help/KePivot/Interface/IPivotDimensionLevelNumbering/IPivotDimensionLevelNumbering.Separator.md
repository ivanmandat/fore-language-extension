# IPivotDimensionLevelNumbering.Separator


## Синтаксис


Separator: String;


## Описание


Свойство Separator определяет
 разделитель, который будет использоваться при нумерации измерения/уровня.


## Комментарии


По умолчанию разделитель отсутствует.
