# IPivotDimensionLevels


Сборка: Pivot;


## Описание


Интерфейс IPivotDemensionLevels
 используется для работы с коллекцией уровней измерения таблицы.


## Иерархия наследования


           IPivotDimensionLevels


## Комментарии


Для получения объекта данного типа используйте [IPivotDimension.Levels](../IPivotDimension/IPivotDimension.Levels.htm).
