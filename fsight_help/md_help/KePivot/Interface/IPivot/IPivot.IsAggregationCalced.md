# IPivot.IsAggregationCalced


## Синтаксис


IsAggregationCalced: Boolean;


## Описание


Свойство IsAggregationCalced
 возвращает признак расчёта агрегации данных таблицы.


## Комментарии


Допустимые значения:


	- True. Агрегация рассчитана;


	- False. Агрегация не
	 рассчитана. Например, если включена агрегация, и во всех фиксированных
	 измерениях установлена единичная отметка.


Для определения элемента, по которому будут устанавливаться данные по
 измерениям в матрицу, используйте [IPivot.AggregationElement](IPivot.AggregationElement.htm).
