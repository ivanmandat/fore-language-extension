# IPivot.BroughtOutDim


## Синтаксис


BroughtOutDim: [IDimInstance](KeDims.chm::/interface/IDimInstance/IDimInstance.htm);


## Описание


Свойство BroughtOutDim определяет
 измерение метрик.


## Комментарии


Для использования свойства необходимо, чтобы измерение было фиксированным.


Для определения индекса элемента измерения метрик при использовании
 атрибута «[Значение](UiAnalyticalArea.chm::/Work_with_dimensions/Table_metrics.htm)» используйте
 [IPivot.BroughtOutDimValue](IPivot.BroughtOutDimValue.htm).
