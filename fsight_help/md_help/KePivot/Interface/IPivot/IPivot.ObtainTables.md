# IPivot.ObtainTables


## Синтаксис


ObtainTables: [IPivotTables](../IPivotTables/IPivotTables.htm);


## Описание


Метод ObtainTables возвращает
 коллекцию вариантов таблиц данных.


## Комментарии


В текущей реализации поддерживается одна таблица данных, для доступа
 к ней рекомендуется использовать свойство [IPivot.ObtainTable](IPivot.ObtainTable.htm).
