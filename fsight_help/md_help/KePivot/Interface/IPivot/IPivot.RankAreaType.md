# IPivot.RankAreaType

## Описание


Свойство RankAreaType определяет
 область, по значениям которой будет осуществляться ранжирование данных
 таблицы.


## Комментарии


Способ ранжирования указывается в свойстве [IPivot.RankType](IPivot.RankType.htm).
