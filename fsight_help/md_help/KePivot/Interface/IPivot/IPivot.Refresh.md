# IPivot.Refresh


## Синтаксис


Refresh;


## Описание


Метод Refresh осуществляет обновление
 таблицы.


## Комментарии


После вызова метода Refresh,
 для работы с данными таблицы необходимо получать новый экземпляр данных
 с помощью свойства [IPivot.ObtainTable](IPivot.ObtainTable.htm).
