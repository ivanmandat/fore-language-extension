# IPivot.DimElementsDetailing


## Синтаксис


DimElementsDetailing: [IPivotDimElementsDetailing](../IPivotDimElementsDetailing/IPivotDimElementsDetailing.htm);


## Описание


Свойство DimElementsDetailing
 возвращает детализацию элементов измерений.


## Комментарии


Для получения коллекции измерений источников данных, на основе которых
 формируется таблица, используйте свойство [IPivot.Dimensions](IPivot.Dimensions.htm).
