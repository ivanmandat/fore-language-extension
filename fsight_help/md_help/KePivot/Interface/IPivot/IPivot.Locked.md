# IPivot.Locked


## Синтаксис


Locked: Boolean;


## Описание


Свойство Locked возвращает признак
 блокировки перерисовки таблицы.


## Комментарии


Свойство возвращает True, если
 был вызван метод [IPivot.BeginUpdate](IPivot.BeginUpdate.htm),
 но не была восстановлена перерисовка с помощью метода [IPivot.EndUpdate](IPivot.EndUpdate.htm).
