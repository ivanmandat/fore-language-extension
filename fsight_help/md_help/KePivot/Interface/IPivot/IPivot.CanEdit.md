# IPivot.CanEdit


## Синтаксис


CanEdit: Boolean;


## Описание


Свойство CanEdit возвращает
 признак, доступно ли редактирование данных в таблице.


## Комментарии


Допустимые значения:


	- True. Доступно редактирование
	 данных в таблице;


	- False. Недоступно редактирование
	 данных в таблице.


Для определения, доступно ли редактирование данных таблицы, используйте
 свойство [IPivot.IsEditEnabled](IPivot.IsEditEnabled.htm).
