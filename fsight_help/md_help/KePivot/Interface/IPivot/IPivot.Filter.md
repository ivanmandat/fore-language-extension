# IPivot.Filter


## Синтаксис


Filter: [IPivotFilter](../IPivotFilter/IPivotFilter.htm);


## Описание


Свойство Filter возвращает параметры
 фильтрации данных в таблице.


## Комментарии


Фильтрация позволяет отобразить данные, отвечающие определённым условиям.
 Отметка в измерении при этом не изменяется.
