# IPivot.IsDimEnabled


## Синтаксис


IsDimEnabled(DimKey: Integer): Boolean;


## Параметры


DimKey. Ключ измерения.


## Описание


Свойство IsDimEnabled определяет,
 будет ли измерение с указанным ключом отображаться в списке измерений
 для задания отметки.


## Комментарии


По умолчанию отображаются все измерения. Если для какого-либо измерения
 установить свойству IsDimEnabled
 значение False, то оно будет скрыто
 из списка измерений, по которым может быть изменена отметка. При этом
 уже установленная отметка будет использоваться для получения данных, также
 отметку можно будет изменить в свойстве [IPivot.Selection](IPivot.Selection.htm).
