# IPivot.HeadersAndData


## Синтаксис


HeadersAndData: Boolean;


## Описание


Свойство HeadersAndData определяет
 признак построения таблицы.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Отстраиваются заголовки и данные таблицы;


	- False. Таблица не отстраивается.
