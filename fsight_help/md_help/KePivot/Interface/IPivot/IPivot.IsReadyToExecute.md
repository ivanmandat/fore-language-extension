# IPivot.IsReadyToExecute


## Синтаксис


IsReadyToExecute: Boolean;


## Описание


Свойство IsReadyToExecute возвращает
 признак, готова ли таблица к расчёту данных.


## Комментарии


Допустимые значения:


	- True. Таблица готова
	 к расчёту;


	- False. Таблица не готова
	 к расчёту.


Для получения признака расчёта данных таблицы используйте свойство [IPivot.IsCalced](IPivot.IsCalced.htm).
