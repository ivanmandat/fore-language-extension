# IPivot.MergedNamingSeparator


## Синтаксис


MergedNamingSeparator: String;


## Описание


Свойство MergedNamingSeparator
 определяет разделитель, который будет использоваться при объединении наименований
 элементов различных измерений.


## Комментарии


Свойство актуально, если свойству [IPivot.MergedNamingMode](IPivot.MergedNamingMode.htm)
 установлено значение «PivotMergedNamingMode.UpperElement
 or PivotMergedNamingMode.LowerElement».


По умолчанию в качестве разделителя используется сочетание «, ».
