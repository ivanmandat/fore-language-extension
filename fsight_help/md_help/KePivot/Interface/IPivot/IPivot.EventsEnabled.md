# IPivot.EventsEnabled


## Синтаксис


EventsEnabled: Boolean;


## Описание


Свойство EventsEnabled определяет
 использование обработчика событий в таблице.


## Комментарии


Допустимые значения:


	- True. Используется обработчик
	 событий в таблице;


	- False. Не используется
	 обработчик событий в таблице.


Для добавления пользовательского обработчика используйте метод [IPivot.AddEventHandler](IPivot.AddEventHandler.htm),
 для отключения обработчика - метод [IPivot.RemoveEventHandler](IPivot.RemoveEventHandler.htm).
