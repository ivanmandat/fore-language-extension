# IPivot.ViewOptions


## Синтаксис


ViewOptions: [PivotViewOptions](../../Enums/PivotViewOptions.htm);


## Описание


Свойство ViewOptions определяет
 режим отображения составных измерений в таблице.


## Комментарии


На отображение наименований источников в наименованиях элементов составных
 измерений влияет свойство [ICompoundDimensionSources.ShowSourceName](KeDims.chm::/interface/ICompoundDimensionSources/ICompoundDimensionSources.ShowSourceName.htm).
