# IPivot.DataCacheEnabled


## Синтаксис


DataCacheEnabled:
 Boolean;


## Описание


Свойство DataCacheEnabled возвращает
 признак работы с кешированными данными источника.


## Комментарии


Свойство возвращает значение True,
 если для источника включено кеширование данных в базе репозитория и текущая
 работа осуществляется с загруженными из кеша данными, и False
 во всех остальных случаях.
