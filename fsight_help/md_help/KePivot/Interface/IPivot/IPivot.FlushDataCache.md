# IPivot.FlushDataCache


## Синтаксис


FlushDataCache;


## Описание


Метод FlushDataCache осуществляет
 очистку кеша данных.


## Комментарии


Данный метод очищает временный кеш данных, который создаётся при работе
 с таблицей. Метод актуален, если свойству [IPivot.UseDataCache](IPivot.UseDataCache.htm)
 установлено значение True.
