# IPivot.DrillThrough


## Синтаксис


DrillThrough: Boolean;


## Описание


Свойство DrillThrough определяет
 признак [детализации](UiAnalyticalArea.chm::/Working_with_table_data/Drill_down.htm#agregation) данных
 таблицы в реляционную таблицу.


## Комментарии


Допустимые значения:


	- True. Данные таблицы
	 будут детализированы в реляционную таблицу;


	- False. По умолчанию.
	 Стандартное отображение таблицы.
