# IPivot.RemoveHierarchy


## Синтаксис


RemoveHierarchy(Value: [IDimHierarchyInstance](KeDims.chm::/interface/IDimHierarchyInstance/IDimHierarchyInstance.htm));


## Параметры


Value. Данные альтернативной
 иерархии справочника.


## Описание


Метод RemoveHierarchy удаляет
 альтернативную иерархию из коллекции.


## Комментарии


Для добавления альтернативной иерархии в коллекцию используйте метод
 [IPivot.AddHierarchy](IPivot.AddHierarchy.htm).


Для получения признака, определяющего наличие указанной альтернативной
 иерархии в коллекции, используйте свойство [IPivot.ContainsHierarchy](IPivot.ContainsHierarchy.htm).
