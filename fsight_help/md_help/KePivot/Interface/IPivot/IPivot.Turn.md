# IPivot.Turn


## Синтаксис


Turn;


## Описание


Метод Turn осуществляет разворот
 таблицы.


## Комментарии


После вызова метода, измерения располагавшиеся по строкам будут располагаться
 по столбцам, а располагавшиеся по столбцам - будут располагаться по строкам.
 Фиксированные измерения останутся без изменений.
