# IPivot.RemoveEventHandler


## Синтаксис


RemoveEventHandler(Cookie: Integer);


## Параметры


Cookie. Уникальный индекс обработчика
 событий.


## Описание


Метод RemoveEventHandler удаляет
 обработчик событий таблицы.


## Комментарии


Для удаления обработчика событий, который использует таблица, необходимо
 получить значение, возвращаемое методом [AddEventHandler](IPivot.AddEventHandler.htm),
 затем подставить его в качестве значения параметра
 Cookie.
