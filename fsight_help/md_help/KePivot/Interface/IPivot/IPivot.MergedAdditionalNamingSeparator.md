# IPivot.MergedAdditionalNamingSeparator


## Синтаксис


MergedAdditionalNamingSeparator: String;


## Описание


Свойство MergedAdditionalNamingSeparator
 определяет разделитель, который будет использоваться при объединении дополнительных
 наименований элементов различных измерений.


## Комментарии


Для определения разделителя, который будет использоваться при объединении
 наименований элементов различных измерений, используйте свойство [IPivot.MergedNamingSeparator](IPivot.MergedNamingSeparator.htm).
