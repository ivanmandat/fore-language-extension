# IPivotPareto


Сборка: Pivot;


## Описание


Интерфейс IPivotPareto содержит
 свойства для получения параметров анализа 80/20 по различным элементам.


## Иерархия наследования


IPivotPareto


## Комментарии


Общие параметры анализа 80/20 могут быть получены в свойстве [IPivot.Pareto](../IPivot/IPivot.Pareto.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ColumnsSettings](IPivotPareto.ColumnsSettings.htm)
		 Свойство ColumnsSettings
		 возвращает параметры анализа 80/20, установленные по столбцам.


		 ![](../../Property_Image.gif)
		 [RowsSettings](IPivotPareto.RowsSettings.htm)
		 Свойство RowsSettings
		 возвращает параметры анализа 80/20, установленные по строкам.
