# IPivotEvaluatorDimCalculatesItem.Expression


## Синтаксис


Expression(Index: Integer): [IExpression](ForeSys.chm::/Interface/IExpression/IExpression.htm);


## Параметры


Index -
 индекс вычисляемого элемента по данному измерению.


## Описание


Свойство Expression возвращает
 выражение, по которому осуществляется расчет вычисляемого элемента, индекс
 которого передается посредством параметра Index.
