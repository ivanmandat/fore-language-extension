# IPivotEvaluatorDimCalculatesItem.Relation


## Синтаксис


Relation(Index: Integer): [DimRelation](KeDims.chm::/Enums/DimRelation.htm);


## Параметры


Index -
 индекс вычисляемого элемента по данному измерению.


## Описание


Свойство Relation определяет
 позицию вставки вычисляемого элемента, индекс которого передается посредством
 параметра Index, относительно
 элемента [RelationElement](IPivotEvaluatorDimCalculatesItem.RelationElement.htm)
 измерения.
