# IPivotEvaluatorDimCalculatesItem.RemoveCalculatedElement


## Синтаксис


RemoveCalculatedElement(Index: Integer);


## Параметры


Index -
 индекс вычисляемого элемента по данному измерению.


## Описание


Метод RemoveCalculatedElement
 осуществляет удаление вычисляемого элемента, индекс которого передается
 посредством параметра Index.
