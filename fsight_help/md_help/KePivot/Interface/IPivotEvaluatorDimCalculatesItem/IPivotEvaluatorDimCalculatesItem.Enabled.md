# IPivotEvaluatorDimCalculatesItem.Enabled


## Синтаксис


Enabled(Index: Integer): Boolean;


## Параметры


Index - индекс вычисляемого
 показателя по данному измерению.


## Описание


Свойство Enabled определяет
 признак расчета вычисляемого показателя, индекс которого передается посредством
 параметра Index.


## Комментарии


По умолчанию свойству установлено значение True,
 при этом производится расчет вычисляемого показателя. Если свойству установить
 значение False, то вычисляемый
 показатель не будет рассчитываться и отображаться в таблице.
