# IPivotEvaluatorDimCalculatesItem.RelationElement


## Синтаксис


RelationElement(Index: Integer): Integer;


## Параметры


Index -
 индекс вычисляемого элемента по данному измерению.


## Описание


Свойство RelationElement определяет
 элемент измерения, относительно которого осуществляется вставка вычисляемого
 элемента, индекс которого передается посредством параметра Index.
 Позиция вставки определяется свойством [Relation](IPivotEvaluatorDimCalculatesItem.Relation.htm).
