# IPivotEvaluatorDimCalculatesItem


## Описание


Интерфейс IPivotEvaluatorDimCalculatesItem
 содержит свойства и методы для работы с вычисляемыми элементами по заданному
 измерению.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CalculatedElement](IPivotEvaluatorDimCalculatesItem.CalculatedElement.htm)
		 Свойство CalculatedElement
		 возвращает уникальный идентификатор вычисляемого элемента в измерении.


		 ![](../../Property_Image.gif)
		 [CalculatedName](IPivotEvaluatorDimCalculatesItem.CalculatedName.htm)
		 Свойство CalculatedName
		 определяет наименование вычисляемого элемента по данному измерению.


		 ![](../../Property_Image.gif)
		 [Count](IPivotEvaluatorDimCalculatesItem.Count.htm)
		 Свойство Count возвращает
		 количество вычисляемых элементов, имеющихся по данному измерению.


		 ![](../../Property_Image.gif)
		 [Dim](IPivotEvaluatorDimCalculatesItem.Dim.htm)
		 Свойство Dim возвращает
		 данные измерения, для которого осуществляется работа с вычисляемыми
		 элементами.


		 ![](../../Property_Image.gif)
		 [Enabled](IPivotEvaluatorDimCalculatesItem.Enabled.htm)
		 Свойство Enabled определяет
		 признак расчета указанного вычисляемого показателя.


		 ![](../../Property_Image.gif)
		 [Expression](IPivotEvaluatorDimCalculatesItem.Expression.htm)
		 Свойство Expression
		 возвращает выражение, по которому осуществляется расчет вычисляемого
		 элемента.


		 ![](../../Property_Image.gif)
		 [Relation](IPivotEvaluatorDimCalculatesItem.Relation.htm)
		 Свойство Relation определяет
		 позицию вставки вычисляемого элемента относительно элемента [RelationElement](IPivotEvaluatorDimCalculatesItem.RelationElement.htm).


		 ![](../../Property_Image.gif)
		 [RelationElement](IPivotEvaluatorDimCalculatesItem.RelationElement.htm)
		 Свойство RelationElement
		 определяет элемент измерения, относительно которого осуществляется
		 вставка вычисляемого элемента.


		 ![](../../Property_Image.gif)
		 [Type](IPivotEvaluatorDimCalculatesItem.Type.htm)
		 Свойство Type возвращает
		 тип вычисляемого показателя.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddCalculatedElement](IPivotEvaluatorDimCalculatesItem.AddCalculatedElement.htm)
		 Метод AddCalculatedElement
		 осуществляет создание нового вычисляемого элемента для данного
		 измерения.


		 ![](../../Sub_Image.gif)
		 [AddPercentCalculate](IPivotEvaluatorDimCalculatesItem.AddPercentCalculate.htm)
		 Метод AddPercentCalculate
		 создает вычисляемый показатель, осуществляющий расчет процентного
		 значения ячейки относительно итогового значения.


		 ![](../../Sub_Image.gif)
		 [Clear](IPivotEvaluatorDimCalculatesItem.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех вычисляемых элементов по данному измерению.


		 ![](../../Sub_Image.gif)
		 [FindByDimElement](IPivotEvaluatorDimCalculatesItem.FindByDimElement.htm)
		 Метод FindByDimElement
		 осуществляет поиск всех вычисляемых показателей, созданных относительно
		 указанного элемента измерения.


		 ![](../../Sub_Image.gif)
		 [RemoveCalculatedElement](IPivotEvaluatorDimCalculatesItem.RemoveCalculatedElement.htm)
		 Метод RemoveCalculatedElement
		 осуществляет удаление вычисляемого элемента.
