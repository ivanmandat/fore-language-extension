# IPivotHeaderNumbering.CrossDimension


## Синтаксис


CrossDimension: Boolean;


## Описание


Свойство CrossDimension определяет,
 будет ли объединяться нумерация измерений в заголовках таблицы.


## Комментарии


Допустимые значения:


	- True. Элементы разных
	 измерений имеют единую нумерацию. Значение по умолчанию.


	- False. Элементы разных
	 измерений нумеруются независимо друг от друга.


Свойство актуально, если свойству [IsOn](IPivotHeaderNumbering.IsOn.htm)
 установлено значение True.
