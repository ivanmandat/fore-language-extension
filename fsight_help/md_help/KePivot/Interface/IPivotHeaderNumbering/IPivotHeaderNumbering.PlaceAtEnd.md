# IPivotHeaderNumbering.PlaceAtEnd


## Синтаксис


PlaceAtEnd: Boolean;


## Описание


Свойство PlaceAtEnd определяет
 место расположения номеров в заголовках таблицы.


## Комментарии


Допустимые значения:


	- True. В конце заголовка.


	- False. В начале заголовка.
	 Значение по умолчанию.


Свойство актуально, если свойству [IsOn](IPivotHeaderNumbering.IsOn.htm)
 установлено значение True.
