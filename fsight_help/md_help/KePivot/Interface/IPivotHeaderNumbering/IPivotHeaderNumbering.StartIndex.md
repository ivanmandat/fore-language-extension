# IPivotHeaderNumbering.StartIndex


## Синтаксис


StartIndex: Integer;


## Описание


Свойство StartIndex определяет
 начальное значение нумерации в заголовках таблицы.


## Комментарии


По умолчанию нумерация начинается с единицы.


Свойство
 актуально, если свойству [IsOn](IPivotHeaderNumbering.IsOn.htm)
 установлено значение True.

Пример использования приведен в описании свойства [IPivotHeaderNumbering.IsOn](IPivotHeaderNumbering.IsOn.htm).
