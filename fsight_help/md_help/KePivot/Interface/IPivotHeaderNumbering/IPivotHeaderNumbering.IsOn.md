# IPivotHeaderNumbering.IsOn


## Синтаксис


IsOn: Boolean;


## Описание


Свойство IsOn определяет, будет
 ли использоваться сквозная нумерация в заголовках таблицы.


## Комментарии


Допустимые значения:


	- True. В строках/столбцах
	 заголовков используется сквозная нумерация. При включенной нумерации
	 в заголовках добавляется дополнительная строка/столбец, в которых
	 указываются номера столбцов/строк с данными. Для дополнительной строки/столбца
	 применяется то же оформление, какое определено для заголовков.


	- False. Сквозная нумерация
	 не используется. Значение по умолчанию.


## Пример


Для выполнения примера предполагается наличие в репозитории экспресс-отчёта
 с идентификатором «EXPRESS».
 Данный отчёт должен содержать таблицу данных. В боковике и шапке отчёта
 должно быть по два измерения.


Добавьте ссылки на системные сборки: Metabase, Express, Pivot.


			Sub UserProc;

Var

    MB: IMetabase;

    Express: IEaxAnalyzer;

    Pivot: IPivot;

    Header: IPivotHeader;

    Numbering: IPivotHeaderNumbering;

Begin

    MB := MetabaseClass.Active;

    Express := MB.ItemById("EXPRESS").Edit As IEaxAnalyzer;

    Pivot := Express.Pivot;

    Header := Pivot.LeftHeader;

    //Параметры нумерации строк

    Numbering := Header.Numbering;

    //Включаем отображение нумерации

    Numbering.IsOn := True;

    //Учет в нумерации строк, расположенных в заголовке

    Numbering.ConsiderHeader := True;

    //Отображение нумерации в конце заголовка

    Numbering.PlaceAtEnd :=True;

    //Задаем нумерацию с учетом отступов элементов

    Numbering.AutoIdent:= True;

    //Задаем нумерацию измерений, независящую друг от друга

    Numbering.CrossDimension:= False;

    //Определяем начальное значение нумерации

    Numbering.StartIndex:= 3;

    //Добавление префикса нумерации

    Numbering.Prefix:= "№";

    // Сохраним изменения

    (Express As IMetabaseObject).Save;

End Sub UserProc;


После выполнения примера для строк будет включена сквозная нумерация,
 начинающаяся с цифры три и с префиксом «№». При нумерации будут учитываться количество
 строк в заголовке и отступы элементов измерений. Нумерация будет независимой
 для различных измерений и располагаться в конце заголовка каждой строки.
