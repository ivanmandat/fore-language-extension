# IPivotHeaderNumbering.AutoIdent


## Синтаксис


AutoIdent: Boolean;


## Описание


Свойство AutoIdent определяет,
 будут ли учитываться отступы элементов измерения при размещении нумерации
 в заголовках таблицы.


## Комментарии


Допустимые значения:


	- True. Нумерация размещается
	 в соответствии с отступами по иерархии элементов измерения.


	- False. Нумерация размещается
	 без отступов. Значение по умолчанию.


Свойство актуально, если свойству [IsOn](IPivotHeaderNumbering.IsOn.htm)
 установлено значение True.
