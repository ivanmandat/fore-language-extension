# IPivotHeaderNumbering


Сборка: Pivot;


## Описание


Интерфейс IPivotHeaderNumbering
 содержит свойства для управления нумерацией в заголовках таблицы.


## Иерархия наследования


IPivotHeaderNumbering


## Комментарии


Параметры управления нумерацией можно получить в свойстве [IPivotHeader.Numbering](../IPivotHeader/IPivotHeader.Numbering.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AutoIdent](IPivotHeaderNumbering.AutoIdent.htm)
		 Свойство AutoIdent
		 определяет, будут ли учитываться отступы элементов измерения при
		 размещении нумерации в заголовках таблицы.


		 ![](../../Property_Image.gif)
		 [ConsiderHeader](IPivotHeaderNumbering.ConsiderHeader.htm)
		 Свойство ConsiderHeader
		 определяет, будут ли при расчёте номера строки/столбца учитываться
		 строки/столбцы, расположенные в уголке.


		 ![](../../Property_Image.gif)
		 [CornerFormat](IPivotHeaderNumbering.CornerFormat.htm)
		 Свойство CornerFormat
		 возвращает формат нумерации в уголке таблицы.


		 ![](../../Property_Image.gif)
		 [CrossDimension](IPivotHeaderNumbering.CrossDimension.htm)
		 Свойство CrossDimension
		 определяет, будет ли объединяться нумерация измерений в
		 заголовках таблицы.


		 ![](../../Property_Image.gif)
		 [IsOn](IPivotHeaderNumbering.IsOn.htm)
		 Свойство IsOn определяет,
		 будет ли использоваться сквозная нумерация в заголовках таблицы.


		 ![](../../Property_Image.gif)
		 [PlaceAtEnd](IPivotHeaderNumbering.PlaceAtEnd.htm)
		 Свойство PlaceAtEnd
		 определяет место расположения номеров в заголовках таблицы.


		 ![](../../Property_Image.gif)
		 [Prefix](IPivotHeaderNumbering.Prefix.htm)
		 Свойство Prefix определяет
		 префикс нумерации в заголовках таблицы.


		 ![](../../Property_Image.gif)
		 [StartIndex](IPivotHeaderNumbering.StartIndex.htm)
		 Свойство StartIndex
		 определяет начальное значение нумерации в заголовках таблицы.
