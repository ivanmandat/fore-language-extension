# IPivotTableHeader.ElementDimLevel


## Синтаксис


ElementDimLevel(SlotNumber: Integer; ElementNumber:
 Integer): Integer;


## Параметры


SlotNumber. Индекс слота в
 заголовке.


ElementNumber. Индекс элемента
 в слоте заголовка.


## Описание


Свойство ElementDimLevel возвращает
 уровень элемента измерения, расположенного в определённой позиции слота
 заголовка.
