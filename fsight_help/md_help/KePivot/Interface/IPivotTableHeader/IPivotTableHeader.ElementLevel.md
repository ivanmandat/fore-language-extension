# IPivotTableHeader.ElementLevel


## Синтаксис


ElementLevel(SlotNumber: Integer; ElementNumber:
 Integer): Integer;


## Параметры


SlotNumber.
 Индекс слота в заголовке таблицы;


ElementNumber. Индекс элемента
 в заголовке таблицы.


## Описание


Свойство ElementLevel возвращает
 для элемента заголовка ElementNumber
 уровень соответствующего элемента в измерении.
