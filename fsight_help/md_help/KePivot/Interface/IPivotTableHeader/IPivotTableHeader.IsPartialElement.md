# IPivotTableHeader.IsPartialElement


## Синтаксис


IsPartialElement(SlotNumber: Integer; ElementNumber:
 Integer): Boolean;


## Параметры


SlotNumber. Индекс слота в
 заголовке.


ElementNumber. Индекс элемента
 в слоте заголовка.


## Описание


Свойство IsPartialElement возвращает
 признак, является ли сокращённым элемент, находящийся в определённой позиции
 слота заголовка.
