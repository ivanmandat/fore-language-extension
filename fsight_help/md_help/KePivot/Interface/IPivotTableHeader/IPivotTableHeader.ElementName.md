# IPivotTableHeader.ElementName


## Синтаксис


ElementName(SlotNumber: Integer; ElementNumber:
 Integer): String;


## Параметры


SlotNumber.
 Индекс слота в заголовке таблицы;


ElementNumber. Индекс элемента
 в заголовке таблицы.


## Описание


Свойство ElementName возвращает
 наименование элемента измерения, которому соответствует указанный элемент
 заголовка.
