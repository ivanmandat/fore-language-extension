# IPivotTableHeader


Сборка: Pivot;


## Описание


Интерфейс IPivotTableHeader
 содержит свойства для работы с заголовками таблицы.


## Иерархия наследования


           IPivotTableHeader


## Комментарии


Параметры заголовков таблицы можно получить в следующих свойствах: [IPivotTable.LeftHeader](../IPivotTable/IPivotTable.LeftHeader.htm)
 (заголовок строк) и [IPivotTable.TopHeader](../IPivotTable/IPivotTable.TopHeader.htm)
 (заголовок столбцов).
