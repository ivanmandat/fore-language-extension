# IPivotTableHeader.MergedDimKeys


## Синтаксис


MergedDimKeys(SlotNumber: Integer; ElementNumber:
 Integer): Array;


## Параметры


SlotNumber. Индекс слота в
 заголовке.


ElementNumber. Индекс элемента
 в слоте заголовка.


## Описание


Свойство MergedDimKeys возвращает
 массив ключей поджатого элемента, находящегося в определённой позиции
 слота заголовка.
