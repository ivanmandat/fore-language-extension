# IPivotTableHeader.SlotDim


## Синтаксис


SlotDim(SlotNumber: Integer): [IDimInstance](KeDims.chm::/interface/IDimInstance/IDimInstance.htm);


## Параметры


SlotNumber - индекс слота в
 заголовке таблицы.


## Описание


Свойство SlotDim возвращает
 данные измерения, из элементов которого формируется слот заголовка, индекс
 которого передается посредством параметра SlotNumber.
