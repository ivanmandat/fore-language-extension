# IPivotTableHeader.IsJoinSlot


## Синтаксис


IsJoinSlot (SlotNumber: Integer): Boolean;


## Параметры


SlotNumber. Индекс слота в
 заголовке таблицы.


## Описание


Свойство IsJoinSlot возвращает
 признак того, является ли элемент заголовка объединенным по дополнительному
 наименованию.


## Комментарии


Допустимые значения:


	- True. Элемент заголовка
	 объединён по дополнительному наименованию;


	- False. Элемент заголовка
	 не объединён по дополнительному наименованию.
