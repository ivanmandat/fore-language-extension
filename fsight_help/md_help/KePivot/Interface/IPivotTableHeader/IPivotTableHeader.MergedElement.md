# IPivotTableHeader.MergedElement


## Синтаксис


MergedElement(SlotNumber: Integer; ElementNumber:
 Integer; DimKey: Integer): Integer;


## Параметры


SlotNumber.
 Индекс слота в заголовке.


ElementNumber. Индекс элемента
 в слоте заголовка.


DimKey. Ключ измерения в слоте
 заголовка.


## Описание


Свойство MergedElement возвращает
 индекс поджатого элемента измерения в определённой позиции слота заголовка.
