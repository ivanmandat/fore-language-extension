# IPivotTableHeader.SlotCount


## Синтаксис


SlotCount: Integer;


## Описание


Свойство SlotCount возвращает
 количество слотов, на которые разбит заголовок таблицы.


## Комментарии


Количество слотов зависит от количества измерений, расположенных в выбранной
 области таблицы, а также от количества уровней в каждом измерении.
