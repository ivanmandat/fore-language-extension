# IPivotTableHeader.ElementDimKey


## Синтаксис


ElementDimKey(SlotNumber: Integer; ElementNumber:
 Integer): Integer;


## Параметры


SlotNumber.
 Индекс слота в заголовке таблицы;


ElementNumber. Индекс элемента
 в заголовке таблицы.


## Описание


Свойство ElementDimKey возвращает
 ключ измерения, которому соответствует указанный элемент заголовка.
