# IPivotTableHeader.Element


## Синтаксис


Element(SlotNumber: Integer; ElementNumber: Integer):
 Integer;


## Параметры


SlotNumber. Индекс слота в заголовке таблицы;


ElementNumber. Индекс элемента
 в заголовке таблицы.


## Описание


Свойство Element возвращает
 индекс элемента в измерении, которому соответствует элемент заголовка
 ElementNumber.
