# IPivotTableHeader.IsEvaluatedElement


## Синтаксис


IsEvaluatedElement(SlotNumber: Integer; ElementNumber:
 Integer): Boolean;


## Параметры


SlotNumber.
 Индекс слота в заголовке таблицы;


ElementNumber. Индекс элемента
 в заголовке таблицы.


## Описание


Свойство IsEvaluatedElement
 возвращает True, если элемент
 заголовка ElementNumber является
 вычисляемым элементом измерения.
