# IPivotTableHeader.DimensionElement


## Синтаксис


DimensionElement(DimensionKey: Integer; ElementNumber:
 Integer): Integer;


## Параметры


DimensionKey. Ключ измерения,
 содержащегося в заголовке таблицы;


ElementNumber. Индекс элемента
 в заголовке таблицы.


## Описание


Свойство DimensionElement определяет
 индекс элемента измерения, которое содержится в заголовке таблицы.


## Комментарии


Ключ измерения можно получить с помощью свойства [ElementDimKey](IPivotTableHeader.ElementDimKey.htm).
