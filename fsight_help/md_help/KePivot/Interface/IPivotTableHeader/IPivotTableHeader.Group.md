# IPivotTableHeader.Group


## Синтаксис


Group(ElementNumber: Integer): Integer;


## Параметры


ElementNumber - индекс элемента
 заголовка таблицы.


## Описание


Свойство Group возвращает индекс
 группы, в которую входит указанный элемент заголовка таблицы.
