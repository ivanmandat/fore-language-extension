# IPivotFilterSources.Move


## Синтаксис


Move(From_: Integer; To_: Integer);


## Параметры


From. Индекс позиции фильтра,
 который необходимо переместить, в коллекции.


To. Индекс позиции, в которую
 необходимо переместить фильтр в коллекции.


## Описание


Метод Move осуществляет перемещение
 фильтра в коллекции по индексу.
