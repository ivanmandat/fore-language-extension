# IPivotFilterSources


Сборка: Pivot;


## Описание


Интерфейс IPivotFilterSources
 содержит свойства и методы для работы с коллекцией фильтров.


## Иерархия наследования


           IPivotFilterSources


## Комментарии


Для получения свойств фильтра используйте интерфейс [IPivotFilterSource](../IPivotFilterSource/IPivotFilterSource.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPivotFilterSources.Count.htm)
		 Свойство Count возвращает
		 количество фильтров в коллекции.


		 ![](../../Property_Image.gif)
		 [Enable](IPivotFilterSources.Enable.htm)
		 Свойство Enable определяет
		 возможность применения фильтров.


		 ![](../../Property_Image.gif)
		 [Inverted](IPivotFilterSources.Inverted.htm)
		 Свойство Inverted
		 определяет признак применения фильтра в инвертированном режиме.


		 ![](../../Property_Image.gif)
		 [Item](IPivotFilterSources.Item.htm)
		 Свойство Item возвращает
		 фильтр по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IPivotFilterSources.Add.htm)
		 Метод Add осуществляет
		 добавление фильтра в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](IPivotFilterSources.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех фильтров из коллекции.


		 ![](../../Sub_Image.gif)
		 [Move](IPivotFilterSources.Move.htm)
		 Метод Move осуществляет
		 перемещение фильтра в коллекции по индексу.


		 ![](../../Sub_Image.gif)
		 [Remove](IPivotFilterSources.Remove.htm)
		 Метод Remove осуществляет
		 удаление фильтра из коллекции по индексу.
