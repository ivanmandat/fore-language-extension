# IPivotFilterSources.Enable


## Синтаксис


Enable: Boolean;


## Описание


Свойство Enable определяет возможность
 применения фильтров.


## Комментарии


Допустимые значения:


	- True. Все фильтры применяются.
	 Значение по умолчанию;


	- False. Применение всех
	 фильтров отключено.
