# IPivotHeaderSettings.Filter


## Синтаксис


Settings: [IPivotHeaderFilterSettings](../IPivotHeaderFilterSettings/IPivotHeaderFilterSettings.htm);


## Описание


Свойство Filter возвращает объект
 для работы с настройками фильтрации для заголовков строк/столбцов.


## Комментарии


Фильтрация позволяет выделить подмножество данных из
 таблицы, т.к. в результате фильтрации отображаются только
 те данные, которые удовлетворяют заданным условиям.
