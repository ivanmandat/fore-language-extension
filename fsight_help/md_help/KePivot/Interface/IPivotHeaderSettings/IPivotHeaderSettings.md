# IPivotHeaderSettings


Сборка: Pivot;


## Описание


Интерфейс IPivotHeaderSettings
 используется для работы с настройками заголовков строк/столбцов.


## Иерархия наследования


           [IPivotDimensionSettings](../IPivotDimensionSettings/IPivotDimensionSettings.htm)


           IPivotHeaderSettings


## Комментарии


Для получения объекта данного типа используйте [IPivotHeader.Settings](../IPivotHeader/IPivotHeader.Settings.htm).
