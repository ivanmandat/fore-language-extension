# IPivotDimensionLevelPropertiesCommon.IsChildrenBeforeParents


## Синтаксис


IsChildrenBeforeParents: Boolean;


## Описание


Свойство IsChildrenBeforeParents
 возвращает признак того, размещаются ли [дочерние
 элементы иерархии перед родительскими](UiAnalyticalArea.chm::/TableView/Layout/Layout.htm#locating_child_elements).


## Комментарии


Допустимые значения:


	- True. Дочерние элементы
	 размещаются перед родительскими;


	- False. По умолчанию.
	 Дочерние элементы размещаются после родительских.


Для определения, размещать ли [дочерние
 элементы иерархии перед родительскими](UiAnalyticalArea.chm::/TableView/Layout/Layout.htm#locating_child_elements), используйте
 [IPivotDimensionLevelPropertiesCommon.ChildrenBeforeParents](IPivotDimensionLevelPropertiesCommon.ChildrenBeforeParents.htm).
