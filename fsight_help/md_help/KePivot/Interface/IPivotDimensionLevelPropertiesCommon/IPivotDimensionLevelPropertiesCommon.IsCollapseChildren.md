# IPivotDimensionLevelPropertiesCommon.IsCollapseChildren


## Синтаксис


IsCollapseChildren: Boolean;


## Описание


Свойство IsCollapseChildren
 возвращает признак нахождения дочерних элементов измерения(ий) в свернутом
 состоянии.


## Комментарии


Допустимые значения:


	- True. Иерархия всех
	 элементов находится в свернутом состоянии;


	- False. Имеются элементы
	 с развернутой иерархией.
