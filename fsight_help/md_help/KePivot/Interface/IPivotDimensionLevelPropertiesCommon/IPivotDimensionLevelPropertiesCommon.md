# IPivotDimensionLevelPropertiesCommon


Сборка: Pivot;


## Описание


Интерфейс IPivotDimensionLevelPropertiesCommon
 используется для работы с базовыми свойствами уровня измерения.


## Иерархия наследования


           IPivotDimensionLevelPropertiesCommon


## Комментарии


Данный интерфейс является базовым для следующих интерфейсов:


	- [IPivotDimensionLevelProperties](../IPivotDimensionLevelProperties/IPivotDimensionLevelProperties.htm);


	- [IPivotHeader](../IPivotHeader/IPivotHeader.htm).
