# IPivotDimensionLevelPropertiesCommon.IsDuplicateParent


## Синтаксис


IsDuplicateParent: Boolean;


## Описание


Свойство IsDuplicateParent возвращает
 признак того, повторяются ли [элементы
 уровня для дочерних](UiAnalyticalArea.chm::/TableView/Layout/Layout.htm#parent_after).


## Комментарии


Допустимые значения:


	- True. Элементы уровня
	 повторяются для дочерних элементов;


	- False. По умолчанию.
	 Элементы уровня не повторяются для дочерних элементов.


Для определения, будут ли повторяться [элементы
 уровня для дочерних](UiAnalyticalArea.chm::/TableView/Layout/Layout.htm#parent_after), используйте [IPivotDimensionLevelPropertiesCommon.DuplicateParent](IPivotDimensionLevelPropertiesCommon.DuplicateParent.htm).
