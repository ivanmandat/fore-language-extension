# IPivotEvents.OnBeforeSelectionChangeSent

## Параметры


Value. Данные измерения, в
 котором меняется отметка.


## Описание


Метод OnBeforeSelectionChangeSent
 реализует событие, которое наступает перед изменением отметки измерения
 источника данных.
