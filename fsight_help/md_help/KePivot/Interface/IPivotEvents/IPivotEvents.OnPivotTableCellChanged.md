# IPivotEvents.OnPivotTableCellChanged


## Синтаксис


OnPivotTableCellChanged(Value: [IPivotTableCellChangeEventArgs](../IPivotTableCellChangeEventArgs/IPivotTableCellChangeEventArgs.htm));


## Параметры


Value. Аргумент события, предоставляющий
 информацию о ячейке.


## Описание


Метод OnPivotTableCellChanged
 реализует событие, которое наступает после изменения данных в ячейке таблицы.
