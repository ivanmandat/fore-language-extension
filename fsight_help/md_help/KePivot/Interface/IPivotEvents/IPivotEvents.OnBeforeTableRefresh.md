# IPivotEvents.OnBeforeTableRefresh


## Синтаксис


OnBeforeTableRefresh(Value: [IPivotTable](../IPivotTable/IPivotTable.htm));


## Параметры


Value. Данные обновляемой таблицы.


## Описание


Метод OnBeforeTableRefresh реализует
 событие, которое наступает перед обновлением таблицы.


## Комментарии


Для обновления таблицы используйте метод [IPivot.Refresh](../IPivot/IPivot.Refresh.htm).
