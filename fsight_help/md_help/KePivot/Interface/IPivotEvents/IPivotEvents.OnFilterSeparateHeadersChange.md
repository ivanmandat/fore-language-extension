# IPivotEvents.OnFilterSeparateHeadersChange


## Синтаксис


OnFilterSeparateHeadersChange;


## Описание


Метод OnFilterSeparateHeadersChange
 реализует событие, которое наступает при изменении признака раздельной
 фильтрации строк и столбцов.


## Комментарии


Признак раздельной фильтрации строк и столбцов определяет свойство [IPivotFilter.SeparateHeaders](../IPivotFilter/IPivotFilter.SeparateHeaders.htm).
