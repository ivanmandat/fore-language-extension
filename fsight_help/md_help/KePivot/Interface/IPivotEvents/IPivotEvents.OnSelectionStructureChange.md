# IPivotEvents.OnSelectionStructureChange


## Синтаксис


OnSelectionStructureChange(Value: [IDimInstance](KeDims.chm::/interface/IDimInstance/IDimInstance.htm));


## Параметры


Value. Данные измерения, в
 отметке которого меняется структура элементов.


## Описание


Метод OnSelectionStructureChange
 реализует событие, которое наступает при изменении структуры элементов
 отметки измерения.
