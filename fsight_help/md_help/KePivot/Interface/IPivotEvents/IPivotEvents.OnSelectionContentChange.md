# IPivotEvents.OnSelectionContentChange


## Синтаксис


OnSelectionContentChange(Value: [IDimInstance](KeDims.chm::/interface/IDimInstance/IDimInstance.htm));


## Параметры


Value. Данные измерения.


## Описание


Метод OnSelectionContentChange
 реализует событие, которое наступает при изменении содержимого элементов
 отметки измерения.


## Комментарии


Событие наступит, например, при переименовании элемента измерения, который
 отображается в шапке/боковике таблицы.
