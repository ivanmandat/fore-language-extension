# IPivotEvents.OnHighlightChange


## Синтаксис


OnHighlightChange;


## Описание


Метод OnHighlightChange реализует
 событие, которое наступает при изменении подсветки в таблице.


## Комментарии


Параметры подсветки задаются в свойстве [IPivot.Highlight](../IPivot/IPivot.Highlight.htm).
