# IPivotEvents.OnMatrixerChange


## Синтаксис


OnMatrixerChange;


## Описание


Метод OnMatrixerChange реализует
 событие, которое наступает при изменении матрицы с данными.


## Комментарии


Матрицу с данными можно получить, используя свойство [IPivot.Matrix](../IPivot/IPivot.Matrix.htm).
