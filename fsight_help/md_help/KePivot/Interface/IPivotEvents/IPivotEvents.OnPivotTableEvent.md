# IPivotEvents.OnPivotTableEvent


## Синтаксис


OnPivotTableEvent(Value: [PivotTableEventType](../../Enums/PivotTableEventType.htm));


## Параметры


Value. Тип события, сгенерированного
 в таблице.


## Описание


Метод OnPivotTableEvent реализует
 событие, которое наступает при генерации какого-либо события в таблице.


## Комментарии


Аргумент события позволяет определить какое именно событие сгенерировалось
 в таблице.
