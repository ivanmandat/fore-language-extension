# IPivotFilterRange


## Описание


Интерфейс IPivotFilterRange
 содержит свойства для определения области фильтрации (подсветки) данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Bottom](IPivotFilterRange.Bottom.htm)
		 Свойство Bottom определяет
		 индекс строки, являющейся нижней границей диапазона.


		 ![](../../Property_Image.gif)
		 [Left](IPivotFilterRange.Left.htm)
		 Свойство Left определяет
		 индекс столбца, являющегося левой границей диапазона.


		 ![](../../Property_Image.gif)
		 [Right](IPivotFilterRange.Right.htm)
		 Свойство Right определяет
		 индекс столбца, являющегося правой границей диапазона.


		 ![](../../Property_Image.gif)
		 [Top](IPivotFilterRange.Top.htm)
		 Свойство Top определяет
		 индекс строки, являющейся верхней границей диапазона.
