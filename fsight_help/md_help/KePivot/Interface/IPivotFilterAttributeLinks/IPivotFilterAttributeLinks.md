# IPivotFilterAttributeLinks


Сборка: Pivot;


## Описание


Интерфейс IPivotFilterAttributeLinks
 содержит свойства и методы для работы с коллекцией фильтров по связям
 атрибутов измерений.


## Иерархия наследования


IPivotFilterAttributeLinks


## Комментарии


Параметры фильтрации по связям атрибутов измерений можно получить в
 свойстве [IPivotFilterSettings.AttributeLinks](../IPivotFilterSettings/IPivotFilterSettings.AttributeLinks.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPivotFilterAttributeLinks.Count.htm)
		 Свойство Count возвращает
		 количество связей атрибутов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IPivotFilterAttributeLinks.Item.htm)
		 Свойство Item возвращает
		 параметры указанной связи атрибутов.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IPivotFilterAttributeLinks.Add.htm)
		 Метод Add добавляет
		 в коллекцию новую связь атрибутов и возвращает ее параметры.


		 ![](../../Sub_Image.gif)
		 [Clear](IPivotFilterAttributeLinks.Clear.htm)
		 Метод Clear удаляет
		 из коллекции все связи атрибутов измерений.


		 ![](../../Sub_Image.gif)
		 [RemoveAt](IPivotFilterAttributeLinks.RemoveAt.htm)
		 Метод RemoveAt удаляет
		 из коллекции указанную связь атрибутов.
