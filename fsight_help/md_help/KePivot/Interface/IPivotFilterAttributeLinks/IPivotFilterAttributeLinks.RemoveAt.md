# IPivotFilterAttributeLinks.RemoveAt


## Синтаксис


RemoveAt(Index: Integer);


## Параметры


Index. Индекс связи в коллекции.


## Описание


Метод RemoveAt удаляет из коллекции
 указанную связь атрибутов.
