# IPivotFilterSettings.ConditionValueB


## Синтаксис


ConditionValueB: Variant;


## Описание


Свойство ConditionValueB определяет
 второе число, используемое при фильтрации данных.


## Комментарии


Для использования свойства необходимо, чтобы:


	- свойство [IPivotFilterSettings.Enabled](IPivotFilterSettings.Enabled.htm)
	 принимало значение True.


	- свойство [IPivotFilterSettings.UseCondition](IPivotFilterSettings.UseCondition.htm)
	 принимало значение True.


	- свойство [IPivotFilterSettings.ConditionType](IPivotFilterSettings.ConditionType.htm)
	 принимало любое значение из перечисления [PivotFilterType](../../Enums/PivotFilterType.htm).


Для определения первого значения, используемого для фильтрации данных,
 используйте [IPivotFilterSettings.ConditionValueA](IPivotFilterSettings.ConditionValueA.htm).
