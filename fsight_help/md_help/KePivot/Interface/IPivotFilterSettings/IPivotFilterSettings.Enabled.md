# IPivotFilterSettings.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет,
 будут ли фильтроваться данные в таблице.


## Комментарии


Допустимые значения:


	- True. Данные в таблице
	 будут фильтроваться;


	- False. По умолчанию.
	 Данные в таблице фильтроваться не будут.
