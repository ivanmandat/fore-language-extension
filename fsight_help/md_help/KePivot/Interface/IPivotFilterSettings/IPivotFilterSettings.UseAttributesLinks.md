# IPivotFilterSettings.UseAttributesLinks


## Синтаксис


UseAttributesLinks: Boolean;


## Описание


Свойство UseAttributesLinks
 возвращает признак использования фильтрации по связям атрибутов измерений.


## Комментарии


Допустимые значения:


	- True. Используется фильтрация
	 по связям атрибутов измерений;


	- False. Фильтрация по
	 связям атрибутов измерений не используется.
