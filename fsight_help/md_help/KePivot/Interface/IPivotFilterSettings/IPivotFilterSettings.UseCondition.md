# IPivotFilterSettings.UseCondition


## Синтаксис


UseCondition: Boolean;


## Описание


Свойство UseCondition определяет,
 будет ли в таблице использоваться фильтрация данных по условию.


## Комментарии


Допустимые значения:


	- True. Фильтрация данных
	 по условию будет использоваться;


	- False. По умолчанию.
	 Фильтрация данных по условию использоваться не будет.


Для использования свойства необходимо, чтобы свойство [IPivotFilterSettings.Enabled](IPivotFilterSettings.Enabled.htm)
 принимало значение True.
