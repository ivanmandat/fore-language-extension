# IPivotFilterSettings.ConditionValueA


## Синтаксис


ConditionValueA: Variant;


## Описание


Свойство ConditionValueA определяет
 первое значение, используемое при фильтрации данных.


## Комментарии


Свойство актуально, если:


	- свойство [IPivotFilterSettings.Enabled](IPivotFilterSettings.Enabled.htm)
	 принимает значение True;


	- свойство [IPivotFilterSettings.UseCondition](IPivotFilterSettings.UseCondition.htm)
	 принимает значение True;


	- свойство [IPivotFilterSettings.ConditionType](IPivotFilterSettings.ConditionType.htm)
	 принимает любое значение из перечисления [PivotFilterType](../../Enums/PivotFilterType.htm).


Для определения второго значения, используемого для фильтрации данных,
 используйте [IPivotFilterSettings.ConditionValueB](IPivotFilterSettings.ConditionValueB.htm).
