# IPivotFilterSettings.UseAdditionalTableSource


## Синтаксис


UseAdditionalTableSource: Boolean;


## Описание


Свойство UseAdditionalTableSource
 определяет, будет ли использоваться фильтрация по дополнительной отметке,
 отличной от отметки в источнике.


## Комментарии


Допустимые значения:


	- True. Используется фильтрация
	 по дополнительной отметке;


	- False. Фильтрация по
	 дополнительной отметке не используется.


Указание дополнительной отметки позволяет произвести фильтрацию не по
 отображаемым в таблице данным, а по тем данным, которые имеются по этой
 отметке. Изменить отметку можно по любому измерению. Если в дополнительной
 отметке не задана отметка для какого-либо измерения, то используется его
 отметка, установленная в источнике данных. Для измерения с ролью «[Метрика](UiSelection.chm::/Selection/Dimension_type.htm)»
 может иметь только единичную отметку. Если для фиксированных измерений
 заданы настройки агрегации, то по ним может быть выставлена множественная
 отметка.


Возможность фильтрации по дополнительной отметке доступна для следующих
 видов фильтрации:


	- Скрыть пустые ячейки;


	- Скрыть ячейки с нулевыми значениями;


	- Скрыть ячейки с нечисловыми значениями;


	- По связям атрибутов.


Если таблица строится на нескольких источниках и для одного из источников
 задана дополнительная отметка, то фильтроваться будут общие измерения
 всех источников и фиксированные измерения того источника, для которого
 определена дополнительная отметка.
