# IPivotFilterSettings.UseFilterSelection


## Синтаксис


UseFilterSelection: Boolean;


## Описание


Свойство UseFilterSelection
 возвращает признак наличия отметки, по которой выполняется фильтрация.


## Комментарии


Свойство возвращает значение True,
 если отметка, заданная в свойстве [FilterSelection](IPivotFilterSettings.FilterSelection.htm),
 отличается от [отметки IPivot.Selection](../IPivot/IPivot.Selection.htm).
