# IPivotFilterSettings.SupressNoNumeric


## Синтаксис


SupressNoNumeric: Boolean;


## Описание


Свойство SupressNoNumeric определяет,
 будут ли фильтроваться ячейки с нечисловыми значениями.


## Комментарии


Допустимые значения:


	- True. Ячейки с нечисловыми
	 значениями будут фильтроваться;


	- False. По умолчанию.
	 Ячейки с нечисловыми значениями фильтроваться не будут.


Свойство актуально, если свойство [IPivotFilterSettings.Enabled](IPivotFilterSettings.Enabled.htm)
 принимает значение True.
