# PivotEvaluatorTotalDisposition


## Описание


Перечисление PivotEvaluatorTotalDisposition
 содержит виды итогов, относительно которых будет осуществляться расчет
 вычисляемого показателя.


Используется методом [IPivotEvaluatorDimCalculatesItem.AddPercentCalculate](../Interface/IPivotEvaluatorDimCalculatesItem/IPivotEvaluatorDimCalculatesItem.AddPercentCalculate.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Sub_. Промежуточные
		 итоги. Если в системе имеются промежуточные итоги, то значение
		 будет рассчитываться в процентах от промежуточного значения. Значение
		 промежуточного итога будет рассчитываться в процентах от общего
		 итога.


		 1
		 Overall. Общие итоги.
		 Значение и промежуточные итоги рассчитываются в процентах от общего
		 итога.
