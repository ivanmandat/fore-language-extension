# PivotTotalPosition


## Описание


Перечисление PivotTotalPosition
 используется для определения расположения итогов.


Используется следующими методами:


	- [IPivotClass.SubTotal](../Interface/IPivotClass/IPivotClass.SubTotal.htm);


	- [IPivotClass.Total](../Interface/IPivotClass/IPivotClass.Total.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Default_. По умолчанию.
		 Расположение не определено.


		 1
		 Rows. Расположение
		 итогов по строкам.


		 2
		 Columns. Расположение
		 итогов по столбцам.
