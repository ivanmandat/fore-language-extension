# PivotSortDiapason


## Описание


Перечисление PivotSortDiapason
 содержит режимы, в которых будет осуществляться сортировка в таблице.


Используется свойством [IPivot.SortDiapason](../Interface/IPivot/IPivot.SortDiapason.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 Owner. Внутри иерархии:


		![](PivotSortDiapason_Owner.gif)


		 2
		 Level. По уровням
		 иерархии:


		![](PivotSortDiapason_Level.gif)


		 3
		 Group. По всему
		 измерению внутри одной группы:


		![](PivotSortDiapason_Group.gif)


		 4
		 All. По всему измерению
		 без учета групп:


		![](PivotSortDiapason_All.gif)


		 5
		 HierarchyLevel.
		 По всем уровням:


		![](PivotSortDiapason_HierarchyLevel.png)
