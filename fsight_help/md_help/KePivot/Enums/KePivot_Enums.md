# Перечисления сборки Pivot


		 Перечисление
		 Краткое описание


		 ![](../Enums_Image.gif)
		 [PivotAggregationOperation](PivotAggregationOperation.htm)
		 Перечисление PivotAggregationOperation
		 содержит способы агрегации, применяемые в фиксированных измерениях
		 при множественной отметке элементов.


		 ![](../Enums_Image.gif)
		 [PivotDimensionLevelNumberingType](PivotDimensionLevelNumberingType.htm)
		 Перечисление PivotDimensionLevelNumberingType
		 содержит тип нумерации.


		 ![](../Enums_Image.gif)
		 [PivotDimPosition](PivotDimPosition.htm)
		 Перечисление PivotDimPosition
		 содержит позиции вставки измерения в выбранном заголовке таблицы.


		 ![](../Enums_Image.gif)
		 [PivotEvaluatorDimCalcType](PivotEvaluatorDimCalcType.htm)
		 Перечисление PivotEvaluatorDimCalcType
		 содержит типы вычисляемого показателя, рассчитываемого для экспресс-отчета.


		 ![](../Enums_Image.gif)
		 [PivotEvaluatorElementType](PivotEvaluatorElementType.htm)
		 Перечисление PivotEvaluatorElementType
		 содержит типы итогов, который необходимо рассчитать по строкам
		 и/или столбцам таблицы.


		 ![](../Enums_Image.gif)
		 [PivotEvaluatorTotalDisposition](PivotEvaluatorTotalDisposition.htm)
		 Перечисление PivotEvaluatorTotalDisposition
		 содержит виды итогов, относительно которых будет осуществляться
		 расчет вычисляемого показателя.


		 ![](../Enums_Image.gif)
		 [PivotExceptionType](PivotExceptionType.htm)
		 Перечисление PivotExceptionType
		 содержит типы ошибок, которые могут возникнуть в таблице.


		 ![](../Enums_Image.gif)
		 [PivotFilterArea](PivotFilterArea.htm)
		 Перечисление PivotFilterArea
		 содержит области в таблице с данными, для которых применяются
		 фильтрации (подсветки).


		 ![](../Enums_Image.gif)
		 [PivotFilterElements](PivotFilterElements.htm)
		 Перечисление PivotFilterElements
		 содержит фильтруемые элементы таблицы.


		 ![](../Enums_Image.gif)
		 [PivotFilterType](PivotFilterType.htm)
		 Перечисление PivotFilterType
		 содержит типы условия, по которому будут отбираться данные, которые
		 необходимо отфильтровать.


		 ![](../Enums_Image.gif)
		 [PivotHighlightType](PivotHighlightType.htm)
		 Перечисление PivotHighlightType
		 содержит условия, по которым будут отбираться подсвечиваемые данные.


		 ![](../Enums_Image.gif)
		 [PivotMergedNamingMode](PivotMergedNamingMode.htm)
		 Перечисление PivotMergedNamingMode
		 содержит варианты объединения наименований элементов разных измерений.


		 ![](../Enums_Image.gif)
		 [PivotNumberingFormat](PivotNumberingFormat.htm)
		 Перечисление PivotNumberingFormat
		 содержит форматы нумерации.


		 ![](../Enums_Image.gif)
		 [PivotObtainDataMode](PivotObtainDataMode.htm)
		 Перечисление PivotObtainDataMode
		 используется для определения режима преобразования данных в таблице.


		 ![](../Enums_Image.gif)
		 [PivotParetoKind](PivotParetoKind.htm)
		 Перечисление PivotParetoKind
		 содержит типы расчета количества значений, удовлетворяющих установленному
		 способу отбора при анализе 80/20.


		 ![](../Enums_Image.gif)
		 [PivotRankType](PivotRankType.htm)
		 Перечисление PivotRankType
		 содержит способы ранжирования данных в таблице.


		 ![](../Enums_Image.gif)
		 [PivotSortDiapason](PivotSortDiapason.htm)
		 Перечисление PivotSortDiapason
		 содержит режимы, в которых будет осуществляться сортировка в таблице.


		 ![](../Enums_Image.gif)
		 [PivotSortDirection](PivotSortDirection.htm)
		 Перечисление PivotSortDirection
		 содержит направления сортировки, которые необходимо установить
		 по указанному элементу таблицы.


		 ![](../Enums_Image.gif)
		 [PivotSortKind](PivotSortKind.htm)
		 Перечисление PivotSortKind
		 используется для определения вида сортировки.


		 ![](../Enums_Image.gif)
		 [PivotTableEventType](PivotTableEventType.htm)
		 Перечисление PivotTableEventType
		 содержит типы событий таблицы.


		 ![](../Enums_Image.gif)
		 [PivotTotalParentNamingMode](PivotTotalParentNamingMode.htm)
		 Перечисление PivotTotalParentNamingMode
		 содержит типы наименований элементов, выводимых в строку родительского
		 элемента.


		 ![](../Enums_Image.gif)
		 [PivotTotalPosition](PivotTotalPosition.htm)
		 Перечисление PivotTotalPosition
		 используется для определения расположения итогов.


		 ![](../Enums_Image.gif)


		 [PivotTotalType](PivotTotalType.htm)
		 Перечисление PivotTotalType
		 используется для хранения типа итогов.


		 ![](../Enums_Image.gif)
		 [PivotValueArea](PivotValueArea.htm)
		 Перечисление PivotValueArea
		 содержит области представления данных.


		 ![](../Enums_Image.gif)
		 [PivotValueType](PivotValueType.htm)
		 Перечисление PivotValueType
		 содержит форматы представления данных в таблицы.


		 ![](../Enums_Image.gif)
		 [PivotViewMode](PivotViewMode.htm)
		 Перечисление PivotViewMode
		 содержит режимы отображения таблицы с данными.


		 ![](../Enums_Image.gif)
		 [PivotViewOptions](PivotViewOptions.htm)
		 Перечисление PivotViewOptions
		 содержит режимы отображения таблицы с данными при наличии составных
		 измерений в источнике.
