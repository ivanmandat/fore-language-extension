# PivotSortDirection


## Описание


Перечисление PivotSortDirection
 содержит направления сортировки, которые необходимо установить по указанному
 элементу таблицы.


Используется свойством [IPivotSortItem.Direction](../Interface/IPivotSortItem/IPivotSortItem.Direction.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Нет сортировки.


		 1
		 Asc. Сортировка по
		 возрастанию значений.


		 2
		 Desc. Сортировка по
		 убыванию значений.
