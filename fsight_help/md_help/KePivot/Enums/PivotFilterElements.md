# PivotFilterElements


## Описание


Перечисление PivotFilterElements
 содержит фильтруемые элементы таблицы.


Используется свойством [IPivotFilter.Elements](../Interface/IPivotFilter/IPivotFilter.Elements.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 RowsAndColumns. По
		 строкам и столбцам.


		 1
		 Rows. По строкам.


		 2
		 Columns. По столбцам.
