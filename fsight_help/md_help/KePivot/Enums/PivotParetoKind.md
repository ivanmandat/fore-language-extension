# PivotParetoKind


## Описание


Перечисление PivotParetoKind
 содержит типы расчета количества значений, удовлетворяющих установленному
 способу отбора при анализе 80/20.


Используется свойством [IPivotParetoSettingsItem.Kind](../Interface/IPivotParetoSettingsItem/IPivotParetoSettingsItem.Kind.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 Percent. Отбирать процент
		 значений, заданный в свойстве [Threshold](../Interface/IPivotParetoSettingsItem/IPivotParetoSettingsItem.Threshold.htm).


		 2
		 Count. Отбирать количество
		 значений, заданное в свойстве [Threshold](../Interface/IPivotParetoSettingsItem/IPivotParetoSettingsItem.Threshold.htm).


		 3
		 Sum. Отбирать значения,
		 сумма которых не превышает значение, заданное в свойстве [Threshold](../Interface/IPivotParetoSettingsItem/IPivotParetoSettingsItem.Threshold.htm).
