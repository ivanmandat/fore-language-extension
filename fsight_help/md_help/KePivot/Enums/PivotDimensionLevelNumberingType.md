# PivotDimensionLevelNumberingType


## Описание


Перечисление PivotDimensionLevelNumberingType
 содержит тип нумерации.


Используется свойством [IPivotDimensionLevelNumbering.Type](../Interface/IPivotDimensionLevelNumbering/IPivotDimensionLevelNumbering.Type.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Default. Наследовать
		 от родителя.


		 1
		 Through. Сквозной.


		 2
		 Hierarchical. Иерархический.


		 3
		 HierarchicalWithPrefix.
		 Иерархический с включением родителя в префикс.
