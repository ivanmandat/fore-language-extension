# PivotAggregationOperation


## Описание


Перечисление PivotAggregationOperation
 содержит способы агрегации, применяемые в фиксированных измерениях при
 множественной отметке элементов.


Используется свойством [IPivot.FixedElementsAggregation](../Interface/IPivot/IPivot.FixedElementsAggregation.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None.
		 Не использовать агрегацию в фиксированных измерениях.


		 1
		 Sum. Сумма
		 значений по выделенным элементам.


		 2
		 DataSource.
		 Используется способ агрегации, настроенный для [измерений](UiNavObj.chm::/Cube/CreateCube/Master_Standart/UiMd_Cube_CreateCube_Master_Standart_7.htm)
		 источника данных.
		Если в источнике данных по фиксированному измерению настроена
		 агрегация, то рассчитываются и выводятся агрегированные данные.
		 Если в источнике данных по фиксированному измерению не настроена
		 агрегация, то будут выводиться значения первого отмеченного элемента.


		 3
		 Max. Максимальное значение
		 по выделенным элементам.


		 4
		 Min. Минимальное
		 значение по выделенным элементам.


		 5
		 Avg. Среднее значение
		 по выделенным элементам.


		 6
		 Facts. Используется
		 способ агрегации, настроенный для [фактов](UiNavObj.chm::/CreateCube/Master_Standart/UiMd_Cube_CreateCube_Master_Standart_2.htm#agregation)
		 источника данных.
