# PivotDimPosition


## Описание


Перечисление PivotDimPosition
 содержит позиции вставки измерения в выбранном заголовке таблицы.


Используется методом [IPivotHeader.InsertDim](../Interface/IPivotHeader/IPivotHeader.InsertDim.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 First. Первым в списке.


		 1
		 Change. Взамен указанного
		 измерения.


		 2
		 After. После указанного
		 измерения.


		 3
		 Last. Последним в списке.
