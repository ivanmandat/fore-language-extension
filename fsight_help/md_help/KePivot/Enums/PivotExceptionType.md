# PivotExceptionType


## Описание


Перечисление PivotExceptionType
 содержит типы ошибок, которые могут возникнуть в таблице.


Используется свойством [IPivotException.ExceptionType](../Interface/IPivotException/IPivotException.ExceptionType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Unknown. Тип ошибки
		 не определен.


		 1
		 SelectionLimit. Ошибка,
		 связанная с превышением допустимого количества ячеек таблицы (декартово
		 произведение отмеченных элементов измерений источника).


		 2
		 MatrixLimit. Ошибка,
		 связанная с превышением допустимого размера матрицы (количество
		 непустых ячеек).


		 3
		 DimSelectionLimit.
		 Ошибка, связанная с превышением допустимого количества элементов
		 в отметке измерений.


		 4
		 DimAttrDelete. Ошибка,
		 связанная с удалением одного или нескольких атрибутов измерения.
