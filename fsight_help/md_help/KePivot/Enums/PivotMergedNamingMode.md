# PivotMergedNamingMode


## Описание


Перечисление PivotMergedNamingMode
 содержит варианты объединения наименований элементов разных измерений.


Используется свойствами:


	- [IPivot.MergedNamingMode](../Interface/IPivot/IPivot.MergedNamingMode.htm);


	- [IPivot.MergedAdditionalNamingMode](../Interface/IPivot/IPivot.MergedAdditionalNamingMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 UpperElement. В качестве
		 наименования использовать наименование элемента верхнего измерения.


		 2
		 LowerElement. В качестве
		 наименования использовать наименование элемента нижнего измерения.


## Комментарии
