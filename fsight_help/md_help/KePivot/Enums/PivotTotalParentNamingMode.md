# PivotTotalParentNamingMode


## Описание


Перечисление PivotTotalParentNamingMode
 содержит типы наименований элементов, выводимых в строку родительского
 элемента.


Используется свойством [IPivotInsteadOwnerTotalsSettings.NamingMode](../Interface/IPivotInsteadOwnerTotalsSettings/IPivotInsteadOwnerTotalsSettings.NamingMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Source. Наименование
		 исходного элемента.


		 1
		 TotalMethod. Наименование
		 метода итога.


		 2
		 Combined. Комбинированное
		 наименование исходного элемента и метода итогов.
