# PivotObtainDataMode


## Описание


Перечисление PivotObtainDataMode
 используется для определения режима преобразования данных в таблице.


Используется свойством [IPivot.ObtainDataMode](../Interface/IPivot/IPivot.ObtainDataMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Normal. По отмасштабированным
		 данным. Актуально, если в качестве источника данных таблицы
		 используется куб или база данных временных рядов с настроенными
		 [единицами
		 измерения](UiNavObj.chm::/Units/Units.htm).


		 1
		 Raw. По умолчанию.
		 По исходным данным.
