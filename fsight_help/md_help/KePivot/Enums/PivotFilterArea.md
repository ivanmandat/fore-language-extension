# PivotFilterArea


## Описание


Перечисление PivotFilterArea
 содержит области в таблице с данными, для которых применяются фильтрации
 (подсветки).


Используется следующими свойствами:


	- [IPivotFilter.Area](../Interface/IPivotFilter/IPivotFilter.Area.htm);


	- [IPivotHighlightItem.Area](../Interface/IPivotHighlightItem/IPivotHighlightItem.Area.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Table. Вся таблица.


		 1
		 Range. Выделенная область.
