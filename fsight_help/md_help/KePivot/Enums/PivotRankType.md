# PivotRankType


## Описание


Перечисление PivotRankType содержит способы ранжирования данных в таблице.


Используется свойством [IPivot.RankType](../Interface/IPivot/IPivot.RankType.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Ранжирование не используется.


 1
 Asc. Ранг по возрастанию (в таблице, строке, столбце).
![](PivotRankType_Asc.gif)


 2
 Desc. Ранг по убыванию (в таблице, строке, столбце).
![](PivotRankType_Desc.gif)
