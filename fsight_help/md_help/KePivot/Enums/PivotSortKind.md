# PivotSortKind


## Описание


Перечисление PivotSortKind используется
 для определения вида сортировки.


Используется свойством [IPivotSortItem.Kind](../Interface/IPivotSortItem/IPivotSortItem.Kind.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Без сортировки.


		 1
		 Selection. Сортировка
		 по элементу отметки.


		 2
		 Row. Сортировка по
		 строке.


		 3
		 Column. Сортировка
		 по столбцу.


		 4
		 Dimension. Сортировка
		 по измерению.
