# PivotTableEventType


## Описание


Перечисление PivotTableEventType
 содержит типы событий таблицы.


Используется методом [IPivotEvents.OnPivotTableEvent](../Interface/IPivotEvents/IPivotEvents.OnPivotTableEvent.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 DataSaved. Сохранение
		 данных.


		 2
		 CancelChanges. Отмена
		 изменённых значений в ячейках таблицы.


		 3
		 SavedPartially. Частичное
		 сохранение данных.


		 4
		 CellsChanged. Изменение
		 значения в ячейке таблицы.
