# Введение в сборку ProjectPlanning


Календарно-сетевое планирование – это инструмент составления плана реализации
 проекта и анализа сроков выполнения задач для оптимизации общей продолжительности
 реализации проекта. Сетевое планирование позволяет составить наилучший
 план выполнения задач по проекту в ограниченные сроки и определить задачи,
 которые являются критическими и негативно влияют на общую календарную
 продолжительность проекта.


Проект – это система взаимосвязанных задач, ограниченных сроком выполнения
 и определяющих цель проекта.


В данном разделе представлено описание сборки ProjectPlanning,
 предназначенной для выполнения календарно-сетевого планирования на языке
 Fore.


Основными интерфейсами сборки ProjectPlanning
 являются:


	- [IPrjProject](../Interface/IPrjProject/IPrjProject.htm).
	 Интерфейс используется для настройки параметров проекта;


	- [IPrjTask](../Interface/IPrjTask/IPrjTask.htm).
	 Интерфейс используется для настройки параметров задачи;


	- [IPrjTaskDependency](../Interface/IPrjTaskDependency/IPrjTaskDependency.htm).
	 Интерфейс используется для настройки связи между задачами.
