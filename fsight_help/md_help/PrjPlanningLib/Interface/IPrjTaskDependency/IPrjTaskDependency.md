# IPrjTaskDependency


## Описание


Интерфейс IPrjTaskDependency
 предназначен для определения связи между задачами.

		 [PredecessorTaskKey](IPrjTaskDependency.PredecessorTaskKey.htm)
		 Свойство PredecessorTaskKey
		 определяет ключ задачи-предшественника.

		 Свойство Type
