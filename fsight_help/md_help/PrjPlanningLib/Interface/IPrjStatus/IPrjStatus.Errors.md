# IPrjStatus.Errors


## Синтаксис


Errors: [IPrjStatusCollection](../IPrjStatusCollection/IPrjStatusCollection.htm);


## Описание


Свойство Errors возвращает коллекцию
 ошибок.


## Комментарии


Ошибки могут возникнуть при построении плана проекта. Для проверки плана
 проекта на наличие ошибок или предупреждений используйте метод [IPrjProject.CheckConstraints](../IPrjProject/IPrjProject.CheckConstraints.htm).
