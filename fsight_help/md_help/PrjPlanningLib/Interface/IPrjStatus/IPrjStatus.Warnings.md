# IPrjStatus.Warnings


## Синтаксис


Warnings: [IPrjStatusCollection](../IPrjStatusCollection/IPrjStatusCollection.htm);


## Описание


Свойство Warnings возвращает
 коллекцию предупреждений.


## Комментарии


Предупреждения могут возникнуть при построении плана проекта. Для проверки
 плана проекта на наличие ошибок или предупреждений используйте метод [IPrjProject.CheckConstraints](../IPrjProject/IPrjProject.CheckConstraints.htm).
