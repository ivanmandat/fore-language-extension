# IPrjTaskPeriod


## Описание


Интерфейс IPrjTaskPeriod предназначен
 для определения срока выполнения задачи.


## Комментарии


Используется при определении [фактического](../IPrjTask/IPrjTask.FactPeriod.htm),
 [планируемого](../IPrjTask/IPrjTask.PlanPeriod.htm) и [прогнозируемого](../IPrjTask/IPrjTask.ForecastPeriod.htm)
 срока выполнения задачи.


## Свойства


		 Имя свойства

		 ![](../../Property_Image.gif)
		 [FinishDate](IPrjTaskPeriod.FinishDate.htm)
		 Свойство FinishDate
		 определяет дату окончания задачи.


		 ![](../../Property_Image.gif)
		 [StartDate](IPrjTaskPeriod.StartDate.htm)
		 Свойство StartDate
		 определяет дату начала задачи.
