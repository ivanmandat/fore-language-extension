# IPrjTaskPeriod.StartDate


## Синтаксис


StartDate: [DateTime](foresys.chm::/class/datetime/datetime.htm);


## Описание


Свойство StartDate определяет
 дату начала задачи.


## Комментарии


При определении планируемого срока выполнения задачи данное свойство
 актуально, если используется [ручной
 тип планирования](../../Enums/PrjTaskWorkOutTypes.htm).
