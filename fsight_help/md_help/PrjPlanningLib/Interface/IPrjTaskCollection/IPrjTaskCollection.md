# IPrjTaskCollection


## Описание


Интерфейс IPrjTaskCollection
 предназначен для работы с коллекцией задач.


## Комментарий


Для определения свойств задачи используйте интерфейс [IPrjTask](../IPrjTask/IPrjTask.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPrjTaskCollection.Count.htm)
		 Свойство Count
		 возвращает количество задач в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IPrjTaskCollection.Item.htm)
		 Свойство Item возвращает
		 задачу из коллекции по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IPrjTaskCollection.Add.htm)
		 Метод Add добавляет
		 задачу в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](IPrjTaskCollection.Clear.htm)
		 Метод Clear удаляет
		 все задачи из коллекции.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IPrjTaskCollection.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск задачи в коллекции по ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](IPrjTaskCollection.Remove.htm)
		 Метод Remove удаляет
		 задачу из коллекции по индексу.
