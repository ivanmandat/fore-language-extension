# IPrjTaskCollection.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IPrjTask](../IPrjTask/IPrjTask.htm);


## Параметры


Key. Уникальный ключ задачи.


## Описание


Метод FindByKey осуществляет
 поиск задачи в коллекции по ключу.


## Комментарии


Для определения уникального ключа задачи используйте свойство [IPrjTask.Key](../IPrjTask/IPrjTask.Key.htm).
