# IPrjCalendarOptions


## Описание


Интерфейс IPrjCalendarOptions
 предназначен для настройки календаря.


## Комментарий


Настройки календаря учитываются при [автоматическом](../../Enums/PrjTaskWorkOutTypes.htm)
 планировании задач.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ExcludedDates](IPrjCalendarOptions.ExcludedDates.htm)
		 Свойство ExcludedDates
		 определяет набор дат в качестве нерабочих дней.


		 ![](../../Property_Image.gif)
		 [IncludeWeekend](IPrjCalendarOptions.IncludeWeekend.htm)
		 Свойство IncludeWeekend
		 определяет, является ли суббота и воскресенье рабочими днями.
