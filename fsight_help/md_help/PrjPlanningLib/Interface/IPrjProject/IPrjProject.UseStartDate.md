# IPrjProject.UseStartDate


## Синтаксис


UseStartDate: Boolean;


## Описание


Свойство UseStartDate определяет
 использование даты начала проекта при добавлении первой задачи проекта.


## Комментарии


Допустимые значения:


	- True. Заданная дата
	 начала используется при добавлении первой задачи;


	- False. Заданная дата
	 начала не используется при добавлении первой задачи.
