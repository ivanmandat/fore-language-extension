# IPrjProject


## Описание


Интерфейс IPrjProject предназначен
 для работы с проектом.


## Комментарий


Проект – это система взаимосвязанных задач, ограниченных сроком выполнения
 и определяющих цель проекта. Для каждой задачи в проекте должен быть определен
 уникальный ключ с помощью свойства [IPrjTask.Key](../IPrjTask/IPrjTask.Key.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CalendarOptions](IPrjProject.CalendarOptions.htm)
		 Свойство CalendarOptions
		 возвращает настройки календаря.


		 ![](../../Property_Image.gif)
		 [Contradictions](IPrjProject.Contradictions.htm)
		 Свойство Contradictions
		 возвращает коллекцию расхождений срока выполнения задачи.


		 ![](../../Property_Image.gif)
		 [EndDate](IPrjProject.EndDate.htm)
		 Свойство EndDate возвращает
		 дату окончания проекта на основе календарно-сетевого планирования.


		 ![](../../Property_Image.gif)
		 [StartDate](IPrjProject.StartDate.htm)
		 Свойство StartDate
		 определяет дату начала проекта.


		 ![](../../Property_Image.gif)
		 [Status](IPrjProject.Status.htm)
		 Свойство Status возвращает
		 найденные ошибки или предупреждения при построении плана проекта.


		 ![](../../Property_Image.gif)
		 [Tasks](IPrjProject.Tasks.htm)
		 Свойство Tasks возвращает
		 коллекцию задач.


		 ![](../../Property_Image.gif)
		 [UseStartDate](IPrjProject.UseStartDate.htm)
		 Свойство UseStartDate
		 определяет использование даты начала проекта при добавлении первой
		 задачи проекта.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CheckConstraints](IPrjProject.CheckConstraints.htm)
		 Метод CheckConstraints
		 выполняет проверку на наличие ошибок или предупреждений при построении
		 плана проекта.


		 ![](../../Sub_Image.gif)
		 [Plan](IPrjProject.Plan.htm)
		 Метод Plan выполняет
		 расчёт прогнозируемых сроков выполнения задач по заданному плану.
