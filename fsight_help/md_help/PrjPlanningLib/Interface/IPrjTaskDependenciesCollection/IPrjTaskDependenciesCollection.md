# IPrjTaskDependenciesCollection


## Описание


Интерфейс IPrjTaskDependenciesCollection
 предназначен для работы с коллекцией связей между задачами.


## Комментарий


Для определения связи между задачами используйте интерфейс [IPrjTaskDependency](../IPrjTaskDependency/IPrjTaskDependency.htm).
