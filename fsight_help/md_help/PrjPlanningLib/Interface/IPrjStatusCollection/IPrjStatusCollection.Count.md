# IPrjStatusCollection.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 ошибок или предупреждений в коллекции.


## Комментарии


Ошибки или предупреждения могут возникнуть при построении плана проекта.
 Для проверки плана проекта на наличие ошибок или предупреждений используйте
 метод [IPrjProject.CheckConstraints](../IPrjProject/IPrjProject.CheckConstraints.htm).


Примечание.
 Свойство принимает значение «1» и возвращает только первую найденную ошибку
 или предупреждение.
