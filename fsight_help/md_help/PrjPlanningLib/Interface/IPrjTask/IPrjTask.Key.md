# IPrjTask.Key


## Синтаксис


Key: Integer;


## Описание


Свойство Key определяет уникальный
 ключ задачи в проекте.


## Комментарии


Для каждой задачи в проекте должен быть определен уникальный ключ, принимающий
 целое значение большее или равное нулю. Если значение ключа меньше нуля,
 то при выполнении [расчёта](../IPrjProject/IPrjProject.Plan.htm)
 проекта будет выдано соответствующее сообщение.
