# IPrjTask


## Описание


Интерфейс IPrjTask предназначен
 для определения свойств задачи.


## Комментарии


Для каждой задачи в проекте должен быть определен уникальный ключ с
 помощью свойства [IPrjTask.Key](IPrjTask.Key.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ConstraintDate](IPrjTask.ConstraintDate.htm)
		 Свойство ConstraintDate
		 определяет дату ограничения выполнения задачи.


		 ![](../../Property_Image.gif)
		 [ConstraintType](IPrjTask.ConstraintType.htm)
		 Свойство ConstraintType
		 определяет тип ограничения задачи по времени.


		 ![](../../Property_Image.gif)
		 [Dependencies](IPrjTask.Dependencies.htm)
		 Свойство Dependencies
		 возвращает коллекцию связей задачи с другими задачами.


		 ![](../../Property_Image.gif)
		 [FactPeriod](IPrjTask.FactPeriod.htm)
		 Свойство FactPeriod
		 определяет фактический срок выполнения задачи.


		 ![](../../Property_Image.gif)
		 [ForecastPeriod](IPrjTask.ForecastPeriod.htm)
		 Свойство ForecastPeriod
		 определяет прогнозируемый срок выполнения задачи.


		 ![](../../Property_Image.gif)
		 [Key](IPrjTask.Key.htm)
		 Свойство Key определяет
		 уникальный ключ задачи в проекте.


		 ![](../../Property_Image.gif)
		 [Name](IPrjTask.Name.htm)
		 Свойство Name определяет
		 наименование задачи.


		 ![](../../Property_Image.gif)
		 [ParentKey](IPrjTask.ParentKey.htm)
		 Свойство ParentKey
		 определяет ключ родительской задачи.


		 ![](../../Property_Image.gif)
		 [PlanPeriod](IPrjTask.PlanPeriod.htm)
		 Свойство PlanPeriod
		 определяет планируемый срок выполнения задачи.


		 ![](../../Property_Image.gif)
		 [WorkOutType](IPrjTask.WorkOutType.htm)
		 Свойство WorkOutType
		 определяет тип планирования задач.
