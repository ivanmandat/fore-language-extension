# IPrjTask.ConstraintType


## Синтаксис


ConstraintType: [PrjTaskConstraintType](../../Enums/PrjTaskConstraintType.htm);


## Описание


Свойство ConstraintType определяет
 тип ограничения задачи по времени.


## Комментарии


Данное свойство используется вместе со свойством [IPrjTask.ConstraintDate](IPrjTask.ConstraintDate.htm),
 если необходимо задать дату ограничения выполнения задачи.
