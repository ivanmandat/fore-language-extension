# IPrjTask.FactPeriod


## Синтаксис


FactPeriod: [IPrjTaskPeriod](../IPrjTaskPeriod/IPrjTaskPeriod.htm);


## Описание


Свойство FactPeriod определяет
 фактический срок выполнения задачи.


## Комментарии


Если задан фактический срок выполнения задачи, то при расчёте проекта
 [прогнозируемый](IPrjTask.ForecastPeriod.htm) срок выполнения
 задачи будет совпадать с фактическим.
