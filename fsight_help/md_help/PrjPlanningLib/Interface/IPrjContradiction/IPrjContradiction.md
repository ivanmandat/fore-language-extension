# IPrjContradiction


## Описание


Интерфейс IPrjContradiction предназначен
 для определения расхождения срока выполнения задачи.


## Комментарий


Расхождение срока возникает при прогнозируемом построении сроков выполнения
 задач с разными [типами
 связей](../../Enums/PrjTaskDependencyType.htm) между ними.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dependency](IPrjContradiction.Dependency.htm)
		 Свойство Dependency
		 определяет связь между задачами.


		 ![](../../Property_Image.gif)
		 [Difference](IPrjContradiction.Difference.htm)
		 Свойство Difference
		 возвращает количество дней, на которое расходится срок выполнения
		 задачи.


		 ![](../../Property_Image.gif)
		 [Follower](IPrjContradiction.Follower.htm)
		 Свойство Follower определяет
		 задачу, следующую за задачей-предшественником.


		 ![](../../Property_Image.gif)
		 [Predecessor](IPrjContradiction.Predecessor.htm)
		 Свойство Predecessor
		 определяет задачу-предшественника.
