# PrjTaskWorkOutTypes


## Описание


Перечисление PrjTaskWorkOutTypes
 используется для выбора типа планирования задач.


Используется свойством [IPrjTask.WorkOutType](../Interface/IPrjTask/IPrjTask.WorkOutType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 ManualPlanning. Ручной
		 тип планирования - сроки выполнения задач задаются вручную.


		 1
		 AutoPlanning. По умолчанию.
		 Автоматический тип планирования - сроки выполнения задач задаются
		 автоматически.
