# Репозиторий и объекты репозитория


Репозиторий - это хранилище объектов платформы.


Объект репозитория - это некоторая сущность, обладающая определенным
 набором характеристик, имеющая специфическую структуру и использование.
 В «Форсайт. Аналитическая платформа»
 объекты предназначены для работы с отдельными типами
 данных или выполнения типовых задач. Примерами объектов являются
 таблицы, справочники, кубы, отчёты и так далее.


Один репозиторий может использоваться несколькими пользователями одновременно.
 Любой пользователь может подключиться к репозиторию и использовать существующие
 объекты, а также создавать новые объекты. По умолчанию объекты, добавляемые
 в репозиторий, становятся доступными всем пользователям. Если требуется,
 можно настроить индивидуальные [права доступа](UiNav.chm::/02_Navigator/UiNav_PrDo.htm)
 для каждого объекта или создать [личные папки](Admin.chm::/03_Admin/Users/Admin_UserFolder.htm)
 для пользователей.
