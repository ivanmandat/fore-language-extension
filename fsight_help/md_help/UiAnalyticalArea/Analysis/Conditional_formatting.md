# Условное форматирование


Условное форматирование - это форматирование, которое применяется только
 при выполнении определенного условия. Оно позволяет обеспечить наглядность
 при анализе данных.


Примечание.
 Настройка условного форматирования для инструмента «Отчёты»
 доступна только в настольном приложении.


Условное форматирование может применяться к нескольким элементам таблицы
 и состоять из нескольких условий.


Предусмотрены следующие подходы к настройке условного форматирования:


	- [быстрая
	 настройка](Format/Working_with_conditional_formatting.htm);


	- [расширенная
	 настройка](Format/UiReport_Table_Attribute_Format_conditionParams.htm).


## Элементы для условного форматирования


Применение условного форматирования доступно для следующих элементов:


	- Элемент измерения (строка/столбец).


	- Вычисляемый элемент.


	- Уровень.


	- Измерение.


	- Вся таблица (данные).


	- Итоги по столбцам.


	- Итоги по строкам.


## Порядок применения условного форматирования


Настройки условного форматирования разных типов (цветовые схемы, цветовые
 шкалы, значки, гистограммы) накладываются друг на друга. Если для элементов
 таблицы заданы одинаковые типы форматирования, то применение настроек
 происходит в следующем порядке:


	- Вся таблица (данные).


	- Измерение.


	- Уровень.


	- Элемент измерения.


	- Вычисляемый элемент.


### Особенности применения условного форматирования


При использовании условного форматирования существуют следующие особенности:


	- условное форматирование [перекрывает](UiNav.chm::/GUI/applystyle.htm)
	 [стиль](../TableView/Formatting/Table_styles.htm) таблицы;


	- условное форматирование, заданное для вычисляемых элементов
	 (вычисляемой иерархии) или альтернативной иерархии, приоритетнее условного
	 форматирования, заданного для элементов измерения;


	- условное форматирование итогов по строкам (отображаются в столбце)
	 приоритетнее итогов по столбцам (отображаются в строке);


	- если условное форматирование в строке пересекается с условным
	 форматированием в столбце, то на пересечении строки и столбца приоритетнее
	 условное форматирование по столбцу. Данное правило по аналогии распространяется
	 и на другие элементы таблицы;


	- если в шапке или боковике размещено несколько измерений, для
	 которых задано условное форматирование, то приоритетнее условное форматирование,
	 заданное для элементов последнего измерения, размещенного в шапке
	 или боковике. Данное правило по аналогии распространяется и на другие
	 элементы таблицы;


	- условное форматирование применяется с учетом выбранной [единицы измерения](UiNavObj.chm::/Units/Units_dbts.htm);


	- проверка правил применения значков осуществляется сверху вниз
	 для [условного
	 форматирования](Format/uireport_table_attribute_format_condition_1.htm#panel_icons), заданного для разделения значений в ячейках по
	 нескольким группам.
