# Настройка границ


Настройка границ [выделенного
 элемента](../../Select_areas.htm) таблицы осуществляется при помощи:


	- кнопки ![](border.gif)
	 «Граница» на вкладке «Формат» ленты инструментов. В раскрывающемся
	 меню кнопки выберите тип и цвет границы;


	- вкладки «Оформление»
	 боковой панели.


[![](../../opened.gif)![](../../closed.gif)Для открытия
 вкладки «Оформление»](javascript:TextPopup(this))


	- [Выделите элемент таблицы](../../Select_areas.htm), [измерения](../../Work_with_dimensions/Select_dimensions_elements_in_table.htm#select_dim),
	 [уровня](../../Work_with_dimensions/Select_dimensions_elements_in_table.htm#select_level)
	 или итогов.


	- Перейдите в группу вкладок «Таблица»
	 боковой панели и выберите вкладку «Оформление»/«Оформление данных».


Примечание.
 При выделении элемента таблицы группа вкладок боковой панели «Таблица»
 будет переименована в соответствии с выделенным элементом:

    • Заголовок.
 При выделении боковика или шапки;

    • Уголок.
 При выделении уголка;

    • Таблица.
 При выделении ячейки с данными или всей таблицы;

    • Данные.
 При выделении всех ячеек с данными;

    • Формат. При выделении столбца или строки
 целиком;

    • Уровень.
 При выделении наименований элементов уровня измерения;

    • Измерение.
 При выделении наименований элементов измерения;

    • Итоги.
 При выделении ячеек, столбцов или строк итогов.


Оформление элементов таблицы применяется в соответствии с [порядком оформления](UiNav.chm::/GUI/applystyle.htm)
 и может быть перекрыто другим оформлением.


Для отображения границы установите флажок «Специальная
 граница». Будут доступны настройки границ для выделенного элемента
 таблицы:


![](formattingBorders.gif)


Доступные настройки:


	- Выбор типов границ.
	 Установите флажки напротив типов границ, которые необходимо отобразить:


		- Верхняя;


		- Левая;


		- Правая;


		- Нижняя;


		- Горизонтальная сетка;


		- Вертикальная сетка;


		- Правая диагональ;


		- Левая диагональ.


Примечание.
 В веб-приложении не поддерживается отображение диагоналей для ячейки таблицы.


	- Оформление границ. Выберите
	 в списке одну или несколько границ;


	- Свойства выбранного.
	 Область содержит два раскрывающихся списка:


		- Линия. В раскрывающемся
		 списке выберите тип линии;


		- Цвет. В раскрывающейся
		 палитре выберите цвет границы.
