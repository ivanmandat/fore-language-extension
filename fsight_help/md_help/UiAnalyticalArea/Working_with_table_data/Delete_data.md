# Исключение данных из таблицы


При работе с таблицей аналитической области данных доступны операции
 исключения данных из таблицы с помощью изменения отметки в измерениях
 по строкам и/или столбцам и выполнения команд контекстного меню для различных
 элементов таблицы.


Примечание.
 Исключение данных из таблицы доступно для инструмента «Отчёты»
 только в настольном приложении.


Рассмотрим более подробно операции исключения данных из таблицы, выполняемые
 с помощью команд контекстного меню.


## Исключение строк


Для исключения из таблицы лишней строки:


	- Выделите ячейку или заголовок строки, которую необходимо исключить.


Совет.
 Чтобы выделить несколько ячеек, удерживайте клавишу CTRL при выделении.


	- Выполните команду «Исключить»
	 в контекстном меню выделенной строки.


Примечание.
 В инструменте «Аналитические панели»
 выполните команду «Таблица > Исключить»
 в контекстном меню выделенной строки.


После выполнения действий из таблицы будет исключена выделенная строка.


## Исключение столбцов


Для исключения из таблицы лишнего столбца:


	- Выделите заголовок столбца, который необходимо исключить.


Совет.
 Чтобы выделить несколько столбцов, удерживайте клавишу CTRL при выделении.


	- Выполните команду «Исключить»
	 в контекстном меню выделенного столбца.


Примечание.
 В инструменте «Аналитические панели»
 выполните команду «Таблица > Исключить»
 в контекстном меню выделенного столбца.


После выполнения действий из таблицы будет исключен выделенный столбец.


## Исключение всех данных, кроме выделенных


Для исключения из таблицы всех данных, кроме выделенных:


	- Выделите ячейку, диапазон ячеек/строк/столбцов в таблице, данные
	 из которых должны отображаться в таблице.


	- Выполните команду «Исключить
	 остальные» в контекстном меню выделенного диапазона ячеек.


Примечание.
 В инструменте «Аналитические панели»
 выполните команду «Таблица > Исключить
 остальные» в контекстном меню выделенного диапазона ячеек.


После выполнения действий из таблицы будут исключены все данные, кроме
 выделенных.
