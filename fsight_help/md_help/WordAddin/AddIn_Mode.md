# Начало работы с надстройкой Foresight
 Add-in for Word


Для построения отчётов откройте Microsoft Word, надстройка запустится
 автоматически.


Примечание.
 Поддерживаются версии Microsoft Office 2016, Microsoft Office 2019.


Для построения отчётов смотрите
 статью «[Построение
 отчёта с использованием надстройки](Work/Word_Work.htm)».


Основное окно Microsoft Word с надстройкой Foresight
 Add-in for Word:


![](interface/wordaddin_1.png)


[![](Opened.gif)![](Closed.gif)Элементы окна](javascript:TextPopup(this))


		- Главное меню. Содержит
		 список команд, предназначенных для выполнения различных операций
		 над документом;


		- Лента инструментов.
		 Содержит вкладку «FORESIGHT».
		 Предназначена для работы с надстройкой Foresight
		 Add-in for Word;


		- Рабочая область.
		 Предназначена для построения отчета;


		- Панель «Заголовок».
		 Предназначена для отображения заголовка. Доступна только при добавлении
		 экспресс-отчётов;


		- Панель «[Объекты
		 отчета](Work/ObjectsWork.htm)». Предназначена для работы с объектами, вставленными
		 в отчет;


		- Панель «[Элементы
		 управления](Work/Controls/Controls_work.htm)». Предназначена для работы с элементами
		 управления;


		- Строка статуса.
		 Предназначена для отображения состояния текущего слайда, выбранного
		 режима и масштаба.


[![](Opened.gif)![](Closed.gif)Языковые настройки
 Foresight Add-in for Word](javascript:TextPopup(this))


	Язык надстройки совпадает с языком Microsoft Office. Язык внутренних
	 диалогов (таких как окно регистрации, окно выбора файла из репозитория)
	 совпадает с языком интерфейса продукта «Форсайт. Аналитическая платформа».


[![](Opened.gif)![](Closed.gif)Возможные проблемы
 при запуске](javascript:TextPopup(this))


		- Запуск надстройки может быть заблокирован центром управления
		 безопасностью Microsoft Word. Для разблокировки надстройки и получения
		 дополнительной информации обратитесь к описанию Microsoft Word.


		- Надстройка не будет запущена, если её версия отличается
		 от установленной версии «Форсайт. Аналитическая платформа».
		 Установите надстройку той же версии, что и «Форсайт. Аналитическая платформа».


		- После установки надстройки в приложении Microsoft Office
		 на ленте инструментов может не отображаться вкладка «FORESIGHT». Решение проблемы
		 приведено в разделе «[Решение возможных
		 проблем при работе](FAQ.chm::/Errors/Error_while_working.htm#7_5)».
