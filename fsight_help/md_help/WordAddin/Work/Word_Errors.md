# Сообщения об ошибках


Раздел содержит информацию о способах устранения возможных ошибок, возникающих
 при попытке выполнить какую-либо операцию.


Текст ошибки:


[![](../Opened.gif)![](../Closed.gif)Для пакетной
 печати требуется наличие элементов управления, общих для всех листов отчета](javascript:TextPopup(this))


	Причина появления: возникает
	 при попытке пакетной печати
	 отчета, не содержащего элементов управления фиксированными измерениями.


	Решение: создайте элементы
	 управления для фиксированных измерений. На основе отметки элементов
	 в данных измерениях будут сформированы варианты печати отчета. Повторите
	 пакетную печать.


[![](../Opened.gif)![](../Closed.gif)Для пакетного
 экспорта требуется наличие элементов управления, общих для всех листов
 отчета](javascript:TextPopup(this))


	Причина появления: возникает
	 при попытке пакетного
	 экспорта отчета, не содержащего элементов управления фиксированными
	 измерениями.


	Решение: создайте элементы
	 управления для фиксированных измерений. На основе отметки элементов
	 в данных измерениях будут сформированы варианты экспорта отчета. Повторите
	 пакетный экспорт.


[![](../Opened.gif)![](../Closed.gif)Невозможно установить
 синхронизацию отметки измерений, так как измерения несовместимы](javascript:TextPopup(this))


	Причина появления: возникает
	 при попытке настроить синхронизацию отметки элементов для различных
	 измерений.


	Решение: настройте синхронизацию
	 отметки элементов только для одинаковых измерений.


[![](../Opened.gif)![](../Closed.gif)Некорректный
 диапазон для вставки объекта](javascript:TextPopup(this))


	Причина появления: возникает
	 при попытке вставить объект на основании некорректно выделенного диапазона
	 данных существующего объекта.


	Решения:


		- если требуется вставить объект с индивидуальными настройками
		 измерений и отметки элементов, то снимите все выделение в тексте
		 отчета и на панели «Объекты отчета».
		 Повторите вставку объекта;


		- если требуется вставить объект, который соответствует по
		 настройкам измерений и отметке элементов уже существующему объекту,
		 то снимите все выделение в тексте отчета и в панели «Объекты
		 отчета». Затем выделите необходимый объект (или его элементы)
		 на панели «Объекты отчета»
		 и повторите вставку объекта.


[![](../Opened.gif)![](../Closed.gif)Этот файл уже
 используется. Укажите другое имя файла или закройте файл в другом приложении](javascript:TextPopup(this))


	Причина появления: возникает
	 при попытке вставить из файла в отчет объект, который уже используется
	 в каком-либо отчете.


	Решения:


		- закройте отчет, использующий объект из данного файла. Повторите
		 вставку объекта;


		- выберите для вставки другой объект, сохраненный в виде файла.
