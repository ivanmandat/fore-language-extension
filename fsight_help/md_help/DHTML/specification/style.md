# Рекомендации к CSS коду


Рекомендации к написанию CSS кода:


	- Следует избегать глобальной установки стилей для HTML-тегов
	 в CSS-файлах, содержащих описание стилей отдельных компонентов.


	- Следует избегать использования «!important».


	- Допускается переопределение стилей компонента, но это при этом
	 не гарантируется его корректное отображение.
