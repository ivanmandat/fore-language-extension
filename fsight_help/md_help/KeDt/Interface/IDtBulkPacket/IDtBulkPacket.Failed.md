# IDtBulkPacket.Failed


## Синтаксис


Failed(RecNo: Integer): Boolean;


## Параметры


RecNo. Индекс записи.


## Описание


Свойство Failed возвращает признак
 неудачного экспорта записи, индекс которой передается посредством входного
 параметра.


## Комментарии


Если экспорт записи прошел неудачно, то свойство возвращает значение
 True, иначе - False.
