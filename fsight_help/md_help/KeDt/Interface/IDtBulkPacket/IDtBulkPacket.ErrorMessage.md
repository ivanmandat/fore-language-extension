# IDtBulkPacket.ErrorMessage


## Синтаксис


ErrorMessage(RecNo: Integer): String;


## Параметры


RecNo. Индекс записи.


## Описание


Свойство ErrorMessage возвращает
 текст сообщения об ошибке после вставки пакета при сохранении указанной
 записи.
