# IDtBulkPacket


## Описание


Интерфейс IDtBulkPacket содержит
 свойства пакета данных для транзакционного приёмника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Data](IDtBulkPacket.Data.htm)
		 Свойство Data определяет
		 двумерный массив данных, которые необходимо экспортировать в рамках
		 одной транзакции.


		 ![](../../Property_Image.gif)
		 [ErrorMessage](IDtBulkPacket.ErrorMessage.htm)
		 Свойство ErrorMessage
		 возвращает текст сообщения об ошибке после вставки пакета.


		 ![](../../Property_Image.gif)
		 [Failed](IDtBulkPacket.Failed.htm)
		 Свойство Failed возвращает
		 признак неудачного экспорта указанной записи.


		 ![](../../Property_Image.gif)
		 [FailedRows](IDtBulkPacket.FailedRows.htm)
		 Свойство FailedRows
		 возвращает количество записей, во время экспорта которых произошла
		 ошибка.


		 ![](../../Property_Image.gif)
		 [FailPolicy](IDtBulkPacket.FailPolicy.htm)
		 Свойство FailPolicy
		 определяет вид действия, осуществляемый при возникновении ошибки
		 во время экспорта данных.


		 ![](../../Property_Image.gif)
		 [StopLimit](IDtBulkPacket.StopLimit.htm)


		 Свойство StopLimit
		 определяет количество ошибочных записей, при достижении которого
		 выполнение операции прекращается.


		 ![](../../Property_Image.gif)
		 [Successful](IDtBulkPacket.Successful.htm)
		 Свойство Successful
		 возвращает признак успешного завершения экспорта пакета данных.
