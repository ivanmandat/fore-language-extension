# IDtExcelConsumer.HasHeader


## Синтаксис


HasHeader: Boolean;


## Описание


Свойство HasHeader определяет
 признак выгрузки в первую строку файла списка заголовков полей.


## Комментарии


В текущей реализации для корректной работы экспортера данному свойству
 должно быть установлено значение True.
