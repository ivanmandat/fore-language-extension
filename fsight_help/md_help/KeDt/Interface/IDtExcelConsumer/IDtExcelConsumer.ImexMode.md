# IDtExcelConsumer.ImexMode


## Синтаксис


ImexMode: [DtExcelImexMode](../../Enums/DtExcelImexMode.htm);


## Описание


Свойство ImexMode определяет
 режим, в котором будет производиться экспорт столбцов, содержащих данные
 разных типов.


## Комментарии


В текущей реализации для корректной работы экспортера данному свойству
 должно быть установлено значение DtExcelImexMode.Export.
