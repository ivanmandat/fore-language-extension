# IDtExcelConsumerEx.HasHeader


## Синтаксис


HasHeader: Boolean;


## Описание


Свойство HasHeader определяет,
 выгружать ли файл-приёмник строку с заголовками данных.


## Комментарии


Допустимые значения:


	- True. В файле-приёмнике
	 первая строка будет содержать заголовки данных;


	- False. В файле-приёмнике
	 не создается строка с заголовками данных.
