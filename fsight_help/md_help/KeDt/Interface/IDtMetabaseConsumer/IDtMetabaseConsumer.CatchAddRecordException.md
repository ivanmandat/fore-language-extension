# IDtMetabaseConsumer.CatchAddRecordException


## Синтаксис


CatchAddRecordException: Boolean;


## Описание


Свойство CatchAddRecordException
 определяет поведение при возникновении ошибки при добавлении записи в
 буфер.


## Комментарии


Допустимые значения:


	- True. Операция продолжается;


	- False. Значение по умолчанию.
	 Операция прерывается.
