# IDtUserProviderEx.CustomProvider


## Синтаксис


CustomProvider: [IDtCustomProvider](../IDtCustomProvider/IDtCustomProvider.htm);


## Описание


Свойство CustomProvider возвращает
 источник данных, полученный с помощью пользовательского алгоритма.


## Комментарии


Источник данных доступен только после вызова метода [IDtObject.Open](../IDtObject/IDtObject.Open.htm).
