# IDtUserProviderEx.Macro


## Синтаксис


Macro: String;


## Описание


Свойство Macro определяет наименование
 класса, реализующего пользовательский алгоритм получения данных.


## Комментарии


Класс должен быть унаследован от интерфейса
 [IDtCustomProvider](../IDtCustomProvider/IDtCustomProvider.htm)
 и содержать реализацию его свойств и методов. Свойство Macro
 используется в комбинации со свойством [Module](IDtUserProviderEx.Module.htm).


## Пример


Пример использования привёден в описании свойства [IDtUserProviderEx.Module](IDtUserProviderEx.Module.htm).
