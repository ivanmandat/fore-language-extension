# IDtExcelProviderEx.HeaderRow


## Синтаксис


HeaderRow: Integer;


## Описание


Свойство HeaderRow определяет
 индекс строки в файле-источнике, содержащей заголовки данных.


## Комментарии


Свойство актуально, если свойство [IDtExcelProvider.HasHeader](../IDtExcelProvider/IDtExcelProvider.HasHeader.htm)
 имеет значение True.


Индексация строк в файле Excel сквозная и начинается с нуля. Индекс
 первой строки, содержащей данные, определяет свойство [IDtExcelProviderEx.DataRow](IDtExcelProviderEx.DataRow.htm).
