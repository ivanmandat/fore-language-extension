# IDtExcelProviderEx.Format


## Синтаксис


Format: String;


## Описание


Свойство Format определяет
 формат файла-источника.


## Комментарии


Допустимые значения:


	- XLS. Формат двоичных
	 файлов Microsoft Excel 97 - Excel 2003 (BIFF8);


	- XLSX. Стандартный формат
	 файлов Microsoft Excel 2010 и Microsoft Excel 2007, основанный на
	 языке XML. В этом формате нельзя сохранять код макросов Microsoft
	 Visual Basic для приложений (VBA) и листы макросов Microsoft Office
	 Excel 4.0 (XLM);


	- XLSM. Формат Microsoft
	 Excel 2010 и Microsoft Excel 2007 на основе XML с поддержкой макросов.
	 Позволяет сохранять код макросов VBA и листы макросов Microsoft Excel
	 4.0 (XLM);


	- XLT. Формат двоичных
	 файлов Microsoft Excel 97 - Microsoft Excel 2003 (BIFF8) для хранения
	 шаблонов Excel;


	- XLTX. Стандартный формат
	 файлов Microsoft Excel 2010 и Microsoft Excel 2007 для шаблонов Microsoft
	 Excel. Не позволяет сохранять код макросов VBA, а также листы макросов
	 Microsoft Excel 4.0 (XLM);


	- XLTM. Формат файлов
	 Microsoft Excel 2010 и Microsoft Excel 2007 для шаблонов Microsoft
	 Excel с поддержкой макросов. Позволяет сохранять код макросов VBA
	 и листы макросов Microsoft Excel 4.0 (XLM);
