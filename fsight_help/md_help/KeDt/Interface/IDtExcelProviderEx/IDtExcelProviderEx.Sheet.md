# IDtExcelProviderEx.Sheet


## Синтаксис


Sheet: String;


## Описание


Свойство Sheet определяет
 наименование листа, на котором расположены данные.


## Комментарии


В качестве значения свойства задайте наименование одного или нескольких
 листов. Если указываются несколько листов, то наименования разделяются
 сочетанием символов «\;», например
 «Sheet1\;Sheet2». Для корректного импорта структура данных на всех листах
 должна быть одинаковой.


Для определения файла источника данных используйте свойство [IDtExcelProviderEx.File](IDtExcelProviderEx.File.htm).


Для определения индекса первых строк листов, начиная с которых будут
 импортироваться данные, используйте свойство [IDtExcelProviderEx.DataSheetRow](IDtExcelProviderEx.DataSheetRow.htm).
