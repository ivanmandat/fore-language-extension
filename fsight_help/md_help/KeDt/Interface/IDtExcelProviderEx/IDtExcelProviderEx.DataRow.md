# IDtExcelProviderEx.DataRow


## Синтаксис


DataRow: Integer;


## Описание


Свойство DataRow определяет
 индекс первой строки, содержащей данные.


## Комментарии


Индексация строк в файле Excel сквозная и начинается с нуля. Индекс
 первой строки, содержащей заголовки данных, определяет свойство [IDtExcelProviderEx.HeaderRow](IDtExcelProviderEx.HeaderRow.htm).
