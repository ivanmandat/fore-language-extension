# IDtCalcFieldErrors


Сборка: Ms;


## Описание


Интерфейс IDtCalcFieldErrors
 предназначен для работы с коллекцией ошибок, возникающих при расчете вычисляемого
 поля.


## Иерархия наследования


           IDtCalcFieldErrors


           [IDtBaseErrors](../IDtBaseErrors/IDtBaseErrors.htm)


## Комментарии


Для получения коллекции ошибок используйте свойства [IDtConsumer.CalcFieldsErrors](../IDtConsumer/IDtConsumer.CalcFieldsErrors.htm)
 и [IDtProvider.CalcFieldsErrors](../IDtProvider/IDtProvider.CalcFieldsErrors.htm).
