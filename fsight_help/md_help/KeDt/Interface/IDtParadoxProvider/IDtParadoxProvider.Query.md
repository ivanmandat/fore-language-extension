# IDtParadoxProvider.Query


## Синтаксис


Query: String;


## Описание


Свойство Query определяет запрос
 к таблицам, хранящимся по указанному [пути](IDtParadoxProvider.Path.htm).


## Комментарии


С помощью данного запроса будут отбираться импортируемые данные.
