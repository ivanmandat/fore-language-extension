# IDtUnpivotProvider.DataWidth


## Синтаксис


DataWidth: Integer;


## Описание


Свойство DataWidth определяет
 количество столбцов в блоке данных, расположенном в области A4.


## Комментарии


По умолчанию DataWidth
 = 1.


Если DataWidth не задано,
 то считается, что в блоке данных один столбец.
