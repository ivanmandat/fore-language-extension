# IDtUnpivotProvider.HeadBottom


## Синтаксис


HeadBottom: Integer;


## Описание


Свойство HeadBottom определяет
 строку, задающую нижнюю границу области A2.


## Комментарии


По умолчанию HeadBottom = 0.


Если HeadBottom не задано, то
 считается, что высота области A2
 - одна строка.
