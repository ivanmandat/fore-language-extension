# IDtUnpivotProvider.DataTop


## Синтаксис


DataTop: Integer;


## Описание


Свойство DataTop определяет
 строку, задающую начало данных в области A4.


## Комментарии


По умолчанию DataTop = 1.


Если DataTop не задано, то началом
 данных считается первая строка после области A2.
