# IDtUnpivotProvider.DataRight


## Синтаксис


DataRight: Integer;


## Описание


Свойство DataRight определяет
 столбец, задающий правую границу области A4.


## Комментарии


По умолчанию DataRight = -1.


Если для области A4 не заданы
 правая и нижняя границы окончания данных (DataRight
 и [IDtUnpivotProvider.DataBottom](IDtUnpivotProvider.DataBottom.htm)),
 то нижней правой границей считается окончание данных (файла).
