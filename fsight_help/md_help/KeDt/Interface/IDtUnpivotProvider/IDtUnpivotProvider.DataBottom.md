# IDtUnpivotProvider.DataBottom


## Синтаксис


DataBottom: Integer;


## Описание


Свойство DataBottom определяет
 строку, задающую нижнюю границу области A4.


## Комментарии


По умолчанию DataBottom = -1.


Если для области A4 не заданы
 правая и нижняя границы окончания данных ([IDtUnpivotProvider.DataRight](IDtUnpivotProvider.DataRight.htm)
 и DataBottom), то нижней правой
 границей считается окончание данных (файла).
