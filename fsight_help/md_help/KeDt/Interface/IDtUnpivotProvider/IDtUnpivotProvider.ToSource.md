# IDtUnpivotProvider.ToSource


## Синтаксис


FromSource(Row: Integer;

           Column:
 Integer;

           Var SourceRow:
 Integer;

           Var SourceColumn:
 Integer);


## Параметры


Row. Индекс строки в итоговой
 таблице;


Column. Индекс столбца в итоговой
 таблице;


SourceRow. В данный параметр
 метод вернет индекс строки в исходной таблице;


SourceColumn. В данный параметр
 метод вернет индекс столбца в исходной таблице.


## Описание


Метод ToSource позволяет
 определить координаты ячейки в исходной таблице по координатам ячейки
 в итоговой таблице.


## Комментарии


Для определения координат ячейки в итоговой таблице по координатам ячейки
 в исходной таблице используйте метод [IDtUnpivotProvider.FromSource](IDtUnpivotProvider.FromSource.htm).
