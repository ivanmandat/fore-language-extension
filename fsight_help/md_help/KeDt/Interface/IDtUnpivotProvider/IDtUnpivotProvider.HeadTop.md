# IDtUnpivotProvider.HeadTop


## Синтаксис


HeadTop: Integer;


## Описание


Свойство HeadTop определяет
 строку, задающую верхнюю границу области A2.


## Комментарии


По умолчанию HeadTop =
 0;


Если HeadTop не задано, то считается,
 что области A2 нет.
