# IDtUnpivotProvider.FixedCells


## Синтаксис


FixedCells: Array;


## Описание


Свойство FixedCells определяет
 двумерный массив с координатами фиксированных ячеек.


## Комментарии


Для каждой ячейки обязательно должны быть заданы оба параметра: строка
 и столбец.


Для каждой фиксированной ячейки в результирующую таблицу будет добавлен
 отдельный столбец. Значение, расположенное в фиксированной ячейке, будет
 повторено во всех строках данного столбца.
