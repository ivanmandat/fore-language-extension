# IDtUnpivotProvider.LeftBegin


## Синтаксис


LeftBegin: Integer;


## Описание


Свойство LeftBegin определяет
 первый столбец с данными, задающий левую границу области A3.


## Комментарии


По умолчанию LeftBegin
 = 0.


Если LeftBegin не задано,
 то левой границей считается начало данных (файла).
