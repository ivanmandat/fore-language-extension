# IDtUnpivotProvider.FromSource


## Синтаксис


FromSource(SourceRow: Integer;

           SourceColumn:
 Integer;

           Var Row:
 Integer;

           Var Column:
 Integer);


## Параметры


SourceRow. Индекс строки в
 исходной таблице;


SourceColumn. Индекс столбца
 в исходной таблице;


Row. В данный параметр метод
 вернет индекс строки в итоговой таблице;


Column. В данный параметр метод
 вернет индекс столбца в итоговой таблице.


## Описание


Метод FromSource позволяет
 определить координаты ячейки в итоговой таблице по координатам ячейки
 в исходной таблице.


## Комментарии


Для определения координат ячейки в исходной таблице по координатам ячейки
 в итоговой таблице используйте метод [IDtUnpivotProvider.ToSource](IDtUnpivotProvider.ToSource.htm).
