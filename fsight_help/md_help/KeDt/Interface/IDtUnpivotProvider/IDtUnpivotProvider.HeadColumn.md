# IDtUnpivotProvider.HeadColumn


## Синтаксис


HeadColumn: Integer;


## Описание


Свойство HeadColumn определяет
 столбец в области A2, соответствующий
 блоку данных, по которому будут формироваться столбцы результирующей таблицы.


## Комментарии


По умолчанию HeadColumn = -1.


Если HeadColumn не задано, то
 все столбцы области A2 будут участвовать
 в формировании столбцов результирующей таблицы.
