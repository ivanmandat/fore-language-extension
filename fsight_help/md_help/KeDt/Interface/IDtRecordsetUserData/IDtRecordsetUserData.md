# IDtRecordsetUserData


Сборка: Dt;


## Описание


Интерфейс IDtRecordsetUserData
 используется для работы с дополнительными настройками пользовательских
 источников/приёмников данных.


## Иерархия наследования


           IDtRecordsetUserData


## Комментарии


Примечание.
 Используйте данный интерфейс совместно с [IDtRecordsetProvider](../IDtRecordsetProvider/IDtRecordsetProvider.htm)
 или [IDtRecordsetConsumer](../IDtRecordsetConsumer/IDtRecordsetConsumer.htm).
