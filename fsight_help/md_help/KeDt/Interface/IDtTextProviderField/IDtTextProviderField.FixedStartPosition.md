# IDtTextProviderField.FixedStartPosition


## Синтаксис


FixedStartPosition: Integer;


## Описание


Свойство FixedStartPosition
 определяет начальную позицию в файле, ограничивающую данные для текущего
 поля.


## Комментарии


Актуально, если свойству [FormatType](../IDtTextProvider/IDtTextProvider.FormatType.htm)
 установлено значение DtTextFormatType.Fixed.
