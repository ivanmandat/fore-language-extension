# IDtTextProviderField


## Описание


Интерфейс IDtTextProviderField
 содержит свойства поля источника данных, необходимые при работе с файлами,
 данные в которых расположены в фиксированных позициях.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [FixedEndPosition](IDtTextProviderField.FixedEndPosition.htm)


		 Свойство FixedEndPosition
		 определяет конечную позицию в файле, ограничивающую данные для
		 текущего поля.


		 ![](../../Property_Image.gif)
		 [FixedStartPosition](IDtTextProviderField.FixedStartPosition.htm)


		 Свойство FixedStartPosition
		 определяет начальную позицию в файле, ограничивающую данные для
		 текущего поля.
