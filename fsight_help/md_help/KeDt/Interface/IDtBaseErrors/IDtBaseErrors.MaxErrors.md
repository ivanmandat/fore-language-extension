# IDtBaseErrors.MaxErrors


## Синтаксис


MaxErrors: Integer;


## Описание


Свойство MaxErrors определяет
 максимальное допустимое количество ошибок при импорте данных.


## Комментарии


Допустимые значения:


	- -1. Допускается неограниченное
	 количество ошибок. При возникновении ошибки импорт продолжается;


	- 0. Значение по умолчанию.
	 Ошибки не допускаются. При возникновении ошибки импорт прерывается;


	- больше 0. Допускается
	 ограниченное количество ошибок. Если количество ошибок превышает значение
	 MaxErrors, то импорт прерывается.


Для задания количества хранимых записей об ошибках используйте свойство
 [IDtBaseErrors.MaxItems](IDtBaseErrors.MaxItems.htm).
