# IDtBaseErrors.MaxItems


## Синтаксис


MaxItems: Integer;


## Описание


Свойство MaxItems определяет
 максимально допустимое количество сохраняемых записей об ошибках.


## Комментарии


Допустимые значения:


	- -1. Допускается неограниченное
	 количество записей об ошибках;


	- 0. Значение по умолчанию.
	 Записи об ошибках не сохраняются;


	- больше 0. Допускается
	 ограниченное количество записей об ошибках. Если количество записей
	 превышает значение MaxItems,
	 то ошибки перестают сохраняться.


Для задания максимально допустимого количества ошибок используйте свойство
 [IDtBaseErrors.MaxErrors](IDtBaseErrors.MaxErrors.htm).
