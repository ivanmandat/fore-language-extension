# IDtBaseErrors


Сборка: Dt;


## Описание


Интерфейс IDtBaseErrors предназначен
 для настройки базовых параметров работы с коллекцией ошибок.


## Иерархия наследования


           IDtBaseErrors


## Комментарии


Данный интерфейс является базовым для интерфейса [IDtErrors](../IDtErrors/IDtErrors.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDtBaseErrors.Count.htm)


		 Свойство Count
		 возвращает количество ошибок в коллекции.


		 ![](../../Property_Image.gif)
		 [MaxErrors](IDtBaseErrors.MaxErrors.htm)


		 Свойство MaxErrors
		 определяет максимальное допустимое количество ошибок при импорте
		 данных.


		 ![](../../Property_Image.gif)
		 [MaxItems](IDtBaseErrors.MaxItems.htm)


		 Свойство MaxItems
		 определяет максимально допустимое количество сохраняемых записей
		 об ошибках.
