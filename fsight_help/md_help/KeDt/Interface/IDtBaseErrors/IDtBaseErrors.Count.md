# IDtBaseErrors.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает
 количество ошибок в коллекции.


## Комментарии


Для задания максимального количества хранимых ошибок используйте свойство
 [IDtBaseErrors.MaxItems](IDtBaseErrors.MaxItems.htm).
