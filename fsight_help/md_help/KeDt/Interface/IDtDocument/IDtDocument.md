# IDtDocument


Сборка: Dt;


## Описание


Интерфейс IDtDocument предназначен
 для работы с источником данных «Репозиторий»,
 если в качестве источника выступает документ репозитория.


## Иерархия наследования


           IDtDocument


## Комментарии


Документ предназначен для хранения и получения доступа к внешним файлам
 различных расширений.
