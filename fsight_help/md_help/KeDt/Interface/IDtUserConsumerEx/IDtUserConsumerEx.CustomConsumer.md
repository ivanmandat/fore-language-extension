# IDtUserConsumerEx.CustomConsumer


## Синтаксис


CustomConsumer: [IDtCustomConsumer](../IDtCustomConsumer/IDtCustomConsumer.htm);


## Описание


Свойство CustomConsumer возвращает
 приёмник данных, полученный с помощью пользовательского алгоритма.


## Комментарии


Приёмник данных доступен только после вызова метода [IDtObject.Open](../IDtObject/IDtObject.Open.htm).


## Пример


Пример использования привёден в описании свойства [IDtUserConsumerEx.Module](IDtUserConsumerEx.Module.htm).
