# IDtUserConsumerEx.Macro


## Синтаксис


Macro: String;


## Описание


Свойство Macro определяет наименование
 класса, реализующего пользовательский алгоритм получения данных.


## Комментарии


Класс должен быть унаследован от интерфейса [IDtCustomConsumer](../IDtCustomConsumer/IDtCustomConsumer.htm)
 и содержать реализацию его свойств и методов. Свойство Macro
 используется в комбинации со свойством [Module](IDtUserConsumerEx.Module.htm).


## Пример


Пример использования привёден в описании свойства [IDtUserConsumerEx.Module](IDtUserConsumerEx.Module.htm).
