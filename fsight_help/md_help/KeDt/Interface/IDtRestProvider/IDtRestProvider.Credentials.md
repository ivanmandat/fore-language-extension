# IDtRestProvider.Credentials


## Синтаксис


Credentials: Object;


## Описание


Свойство Credentials возвращает
 параметры учётных данных, которые будут использоваться при подключении
 к REST-сервису.


## Комментарии


Для настройки параметров учётных данных приведите значение свойства
 к интерфейсу [INetHttpCommandCredential](ModNet.chm::/Interface/INetHttpCommandCredential/INetHttpCommandCredential.htm).
 Настройка учётных данных доступна только при разработке на языке Fore.
