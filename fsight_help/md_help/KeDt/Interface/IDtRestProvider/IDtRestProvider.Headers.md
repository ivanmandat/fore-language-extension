# IDtRestProvider.Headers


## Синтаксис


Headers: [IStringList](ModCollections.chm::/Interface/IStringList/IStringList.htm);


## Описание


Свойство Headers возвращает
 коллекцию HTTP-заголовков.


## Комментарии


Заданные заголовки будут отправляться вместе с запросом к REST-сервису.
