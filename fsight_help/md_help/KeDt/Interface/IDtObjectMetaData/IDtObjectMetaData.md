# IDtObjectMetaData


## Описание


Интерфейс IDtObjectMetaData
 содержит методы для получения информации о структуре источника/приёмника
 данных, использующего для подключения OLE DB драйвера, установленные в
 операционной системе.


## Методы


		 Имя метода
		 Краткое описание

		 [TableFields](IDtObjectMetaData.TableFields.htm)

		 строковый массив, содержащий наименования полей указанной таблицы
		 источника/приёмника данных.


		 ![](../../Sub_Image.gif)
		 [Tables](IDtObjectMetaData.Tables.htm)


		 Метод Tables возвращает

		 данных.


		 ![](../../Sub_Image.gif)
		 [Views](IDtObjectMetaData.Views.htm)


		 Метод Views возвращает
		 строковый массив, содержащий наименования представлений источника/приёмника
		 данных.
