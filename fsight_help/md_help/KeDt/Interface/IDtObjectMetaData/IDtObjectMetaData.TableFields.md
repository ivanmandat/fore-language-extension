# IDtObjectMetaData.TableFields


## Синтаксис


TableFields(TableName: String): Array;


## Параметры


TableName. Наименование таблицы
 источника/приёмника данных, список полей которой необходимо получить.


## Описание


Метод TableFields возвращает
 строковый массив, содержащий наименования полей таблицы источника/приёмника
 данных, наименование которой передается посредством входного параметра.
