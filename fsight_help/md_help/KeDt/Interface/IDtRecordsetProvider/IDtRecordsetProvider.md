# IDtRecordsetProvider


## Описание


Интерфейс IDtRecordsetProvider
 описывает метод, используемый для импорта данных с использованием пользовательского
 алгоритма.


## Комментарии


Методы данного интерфейса должны быть переопределены в пользовательском
 классе.
