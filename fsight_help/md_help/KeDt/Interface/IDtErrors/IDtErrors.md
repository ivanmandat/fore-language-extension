# IDtErrors


Сборка: Dt;


## Описание


Интерфейс IDtErrors предназначен
 для работы с коллекцией ошибок.


## Иерархия наследования


           [IDtBaseErrors](../IDtBaseErrors/IDtBaseErrors.htm)


           IDtErrors


## Комментарии


Для получения коллекции ошибок используйте свойство [IDtProvider.FetchErrors](../IDtProvider/IDtProvider.FetchErrors.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Item](IDtErrors.Item.htm)


		 Метод Item возвращает
		 ошибку из коллекции по её индексу.


## Свойства, унаследованные от [IDtBaseErrors](../IDtBaseErrors/IDtBaseErrors.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](../IDtBaseErrors/IDtBaseErrors.Count.htm)


		 Свойство Count
		 возвращает количество ошибок в коллекции.


		 ![](../../Property_Image.gif)
		 [MaxErrors](../IDtBaseErrors/IDtBaseErrors.MaxErrors.htm)


		 Свойство MaxErrors
		 определяет максимальное допустимое количество ошибок при импорте
		 данных.


		 ![](../../Property_Image.gif)
		 [MaxItems](../IDtBaseErrors/IDtBaseErrors.MaxItems.htm)


		 Свойство MaxItems
		 определяет максимально допустимое количество сохраняемых записей
		 об ошибках.
