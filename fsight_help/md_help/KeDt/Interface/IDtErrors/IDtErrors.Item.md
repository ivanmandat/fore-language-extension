# IDtErrors.Item


## Синтаксис


Item(Index: Integer): [IDtError](../IDtError/IDtError.htm);


## Параметры


Index. Индекс ошибки.


## Описание


Метод Item возвращает
 ошибку из коллекции по её индексу.


## Комментарии


Индексация ошибок в коллекции сплошная и начинается с нуля. Для получения
 количества ошибок в коллекции используйте свойство [IDtBaseErrors.Count](../IDtBaseErrors/IDtBaseErrors.Count.htm).
