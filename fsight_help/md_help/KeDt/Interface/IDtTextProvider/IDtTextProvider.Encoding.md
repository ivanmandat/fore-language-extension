# IDtTextProvider.Encoding


## Синтаксис


Encoding: String;


## Описание


Свойство Encoding определяет
 кодировку символов, используемую при импорте данных.


## Комментарии


В качестве значений данного свойства допустимы следующие значения:


	- WIN. При этом
	 будет использоваться кодировка ASCII;


	- DOS. При этом
	 будет использоваться кодировка OEM;


	- UNICODE. При
	 этом будет использоваться кодировка UTF-16 (LE).
