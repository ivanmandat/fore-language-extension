# IDtTextProvider.RangeHasHeader


## Синтаксис


RangeHasHeader: Boolean;


## Описание


Свойство RangeHasHeader определяет
 признак расположения заголовков столбцов в первой строке файла.


## Комментарии


По умолчанию свойству установлено значение False,
 при этом наименования полей будут генерироваться автоматически (FIELD0,
 FIELD1 и т.д.).
