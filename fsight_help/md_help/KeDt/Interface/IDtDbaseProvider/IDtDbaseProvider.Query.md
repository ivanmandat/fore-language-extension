# IDtDbaseProvider.Query


## Синтаксис


Query: String;


## Описание


Свойство Query определяет запрос
 к таблицам, хранящимся по указанному [пути](IDtDbaseProvider.Path.htm).
 С помощью данного запроса будут отбираться импортируемые данные.
