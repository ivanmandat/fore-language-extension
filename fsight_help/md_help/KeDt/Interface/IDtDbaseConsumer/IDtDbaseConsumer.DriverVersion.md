# IDtDbaseConsumer.DriverVersion


## Синтаксис


DriverVersion: String;


## Описание


Свойство DriverVersion определяет
 версию драйвера, используемого при подключении.


В качестве значений данного свойства доступны следующие значения:


	- dBASE III


	- dBASE IV


	- dBASE 5.0
