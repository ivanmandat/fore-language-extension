# IDtJsonConsumer.RootNodeName


## Синтаксис


RootNodeName: String;


## Описание


Свойство RootNodeName определяет
 имя корневого элемента, относительно которого данные будут экспортированы
 в виде дочерних элементов.


## Комментарии


Во время экспорта будет получена JSON-структура следующего вида: {"Data":[{…},{…},…]},
 где «Data» - имя заданного корневого элемента.
