# IDtTextConsumer.RowDelimiter


## Синтаксис


RowDelimiter: String;


## Описание


Свойство RowDelimiter определяет символ, используемый в качестве разделителя строк. По умолчанию свойству установлено значение {Возврат каретки}{Перевод строки}.


В качестве значений данного свойства допустимы следующие значения:


-
{Возврат каретки}{Перевод строки} - #13+#10


-
{Перевод строки} - код #10


-
{Возврат каретки} - код #13


-
{Табуляция} - код #9


-
,


-
;


-
.


-
:


-
|


-
Любой другой символ, либо сочетание символов, по желанию пользователя.
