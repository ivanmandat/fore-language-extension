# IDtTextConsumer.FormatType


## Синтаксис


FormatType: [DtTextFormatType](../../Enums/DtTextFormatType.htm);


## Описание


Свойство FormatType определяет
 формат структуры файла, в соответствии с которым будет производиться экспорт
 данных.


## Комментарии


По умолчанию используется формат с разделителями.
