# IDtTextConsumer.WriteHeader


## Синтаксис


WriteHeader: Boolean;


## Описание


Свойство WriteHeader определяет
 признак экспорта заголовков столбцов в первую строку файла.


## Комментарии


По умолчанию свойству установлено значение False,
 при этом наименования полей будут генерироваться автоматически (FIELD0,
 FIELD1 и т.д.).
