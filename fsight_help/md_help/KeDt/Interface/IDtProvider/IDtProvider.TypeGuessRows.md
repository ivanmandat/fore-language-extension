# IDtProvider.TypeGuessRows


## Синтаксис


TypeGuessRows: Integer;


## Описание


Свойство TypeGuessRows определяет
 количество строк, по которым будет определяться тип данных каждого поля.


## Комментарии


Свойство актуально, если свойству [IDtProvider.AutoFillFieldsMode](IDtProvider.AutoFillFieldsMode.htm)
 установлено значение [DtAutoFillFieldsMode.DataRow](../../Enums/DtAutoFillFieldsMode.htm).
