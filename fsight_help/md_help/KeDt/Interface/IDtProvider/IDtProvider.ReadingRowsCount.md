# IDtProvider.ReadingRowsCount


## Синтаксис


ReadingRowsCount: Integer;


## Описание


Метод ReadingRowsCount возвращает
 количество записей, считанных из источника данных с учетом условия загрузки.


## Комментарии


Условие загрузки определяется свойством [IDtProvider.FilterIf](IDtProvider.FilterIf.htm).
 Для получения записей, считанных без учета условия загрузки, используйте
 свойство [IDtProvider.ReadingRowsTotalCount](IDtProvider.ReadingRowsTotalCount.htm).
