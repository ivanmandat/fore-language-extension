# IDtProvider.FetchErrors


## Синтаксис


FetchErrors: [IDtErrors](../IDtErrors/IDtErrors.htm);


## Описание


Свойство FetchErrors возвращает
 ошибки, возникшие при импорте данных.


## Комментарии


Для задания максимального количества ошибок при импорте используйте
 свойство [IDtBaseErrors.MaxErrors](../IDtBaseErrors/IDtBaseErrors.MaxErrors.htm),
 для задания максимального количества хранимых ошибок - свойство [IDtBaseErrors.MaxItems](../IDtBaseErrors/IDtBaseErrors.MaxItems.htm).
