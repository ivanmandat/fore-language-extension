# IDtProvider.Fetch


## Синтаксис


Fetch(Var Values: Array);


## Параметры


Values. Одномерный массив,
 в который будут помещены считанные значения текущей записи в источнике
 данных.


## Описание


Метод Fetch осуществляет чтение
 данных текущей записи из источника данных.


## Комментарии


После чтения данных курсор в источнике перемещается на следующую запись.
