# IDtProvider.ReadingRowsTotalCount


## Синтаксис


ReadingRowsTotalCount: Integer;


## Описание


Метод ReadingRowsTotalCount
 возвращает количество записей, считанных из источника данных без учета
 условия загрузки.


## Комментарии


Условие загрузки определяется свойством [IDtProvider.FilterIf](IDtProvider.FilterIf.htm).
 Для получения записей, считанных с учетом условия загрузки, используйте
 свойство [IDtProvider.ReadingRowsCount](IDtProvider.ReadingRowsCount.htm).
