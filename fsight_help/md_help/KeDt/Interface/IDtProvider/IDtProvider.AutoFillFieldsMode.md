# IDtProvider.AutoFillFieldsMode


## Синтаксис


AutoFillFieldsMode: [DtAutoFillFieldsMode](../../Enums/DtAutoFillFieldsMode.htm);


## Описание


Свойство AutoFillFieldsMode
 определяет режим распознавания типа полей источника данных.


## Комментарии


По умолчанию считается, что все поля источника данных - текстовые.
