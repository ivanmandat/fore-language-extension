# IDtObjectDataFormat.DecimalDelimiter


## Синтаксис


DecimalDelimiter: String;


## Описание


Свойство DecimalDelimiter определяет
 разделитель целой и дробной части, который будет использоваться для идентификации
 вещественных чисел.
