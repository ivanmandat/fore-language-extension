# IDtObjectDataFormat


## Описание


Интерфейс IDtObjectDataFormat
 содержит свойства для определения различных настроек, используемых при
 идентификации данных.


## Комментарии


Настройки, используемые при идентификации данных, возвращает свойство
 [IDtProvider.StringDataFormat](../IDtProvider/IDtProvider.StringDataFormat.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DateFormat](IDtObjectDataFormat.DateFormat.htm)


		 Свойство DateFormat
		 определяет формат даты.


		 ![](../../Property_Image.gif)
		 [DecimalDelimiter](IDtObjectDataFormat.DecimalDelimiter.htm)


		 Свойство DecimalDelimiter
		 определяет разделитель целой и дробной части, который будет использоваться
		 для идентификации вещественных чисел.


		 ![](../../Property_Image.gif)
		 [UsePredefinedDateFormats](IDtObjectDataFormat.UsePredefinedDateFormats.htm)
		 Свойство UsePredefinedDateFormats
		 определяет использование предопределения формата даты поля, согласно
		 заданному формату через свойство [IDtFieldDefinition.DateFormat](../IDtFieldDefinition/IDtFieldDefinition.DateFormat.htm).
