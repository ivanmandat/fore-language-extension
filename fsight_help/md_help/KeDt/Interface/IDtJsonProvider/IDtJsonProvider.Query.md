# IDtJsonProvider.Query


## Синтаксис


Query: String;


## Описание


Свойство Query определяет JSONPath-запрос,
 в соответствии с которым будут отбираться импортируемые данные.


## Комментарии


При формировании различных условий в запросе учитывайте указываемый
 тип данных. Неявное преобразование типов не поддерживается. Более подробную
 информацию о формировании JSONPath-запросов можно найти в специализированной
 документации, например, в MSDN.
