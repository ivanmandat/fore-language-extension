# IDtError.Row


## Синтаксис


Row: Integer;


## Описание


Свойство Row возвращает
 номер строки, в которой произошла ошибка.


## Комментарии


Нумерация строк начинается с нуля. Для получения текста ошибки используйте
 свойство [IDtError.Message](IDtError.Message.htm).
