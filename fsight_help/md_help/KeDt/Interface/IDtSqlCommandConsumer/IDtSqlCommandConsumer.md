# IDtSqlCommandConsumer


## Описание


Интерфейс IDtSqlCommandConsumer
 содержит свойства и методы преобразователя данных - [Команда
 СУБД](UiETL.chm::/Desktop/03_ETLObjects/Other/DB_command.htm).

		 Имя свойства
		 Краткое описание

		 [Command](IDtSqlCommandConsumer.Command.htm)


		 Свойство Command определяет

		 данные.
