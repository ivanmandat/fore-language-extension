# IDtObjectCreator


## Описание


Интерфейс IDtObjectCreator содержит
 методы для создания источников/приёмников данных различных типов.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CreateDtObject](IDtObjectCreator.CreateDtObject.htm)


		 Метод CreateDtObject
		 создает источник/приёмник данных указанного типа.


		 ![](../../Sub_Image.gif)
		 [CreateDtObjectFromFile](IDtObjectCreator.CreateDtObjectFromFile.htm)


		 Метод CreateDtObjectFromFile
		 создает источник связанный с указанным файлом.
