# IDtObjectCreator.CreateDtObject


## Синтаксис


CreateDtObject(ObjectType: [DtObjectType](../../Enums/DtObjectType.htm)):
 [IDtObject](../IDtObject/IDtObject.htm);


## Параметры


ObjectType. Тип источника/приёмника
 данных.


## Описание


Метод CreateDtObject создает
 источник/приёмник данных указанного типа.


## Комментарии


Действие данного метода аналогично вызову конструктора Create
 для [класса](../../Class/KeDt_Class.htm), реализующего необходимый
 источник/приёмник данных.
