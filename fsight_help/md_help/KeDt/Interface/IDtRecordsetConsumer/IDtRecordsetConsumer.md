# IDtRecordsetConsumer


## Описание


Интерфейс IDtRecordsetConsumer
 описывает методы, используемые для экспорта данных с использованием пользовательского
 алгоритма.


## Комментарии


Методы данного интерфейса должны быть переопределены в пользовательском
 классе.
