# IDtCustomConsumer.ImplementPut


## Синтаксис


ImplementPut: Boolean;


## Описание


Метод ImplementPut возвращает
 признак того, реализован ли метод [IDtCustomConsumer.Put](IDtCustomConsumer.Put.htm).


## Комментарии


Допустимые значения:


	- True. Метод [IDtCustomConsumer.ImplementPut](IDtCustomConsumer.ImplementPut.htm)
	 реализован и доступен для использования;


	- False. Метод [IDtCustomConsumer.Put](IDtCustomConsumer.Put.htm)
	 не реализован. Используйте [IDtCustomConsumer.PutRow](IDtCustomConsumer.PutRow.htm).
