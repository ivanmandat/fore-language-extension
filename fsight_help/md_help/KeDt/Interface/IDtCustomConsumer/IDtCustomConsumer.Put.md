# IDtCustomConsumer.Put


## Синтаксис


Put(Values: Array);


## Параметры


Values. Двумерный массив значений,
 который необходимо выгрузить в приёмник данных.


## Описание


Метод Put выполняет выгрузку
 массива значений в приёмник данных.


## Комментарии


Для выгрузки одной строки данных используйте метод [IDtCustomConsumer.PutRow](IDtCustomConsumer.PutRow.htm).
