# IDtCustomConsumer.WritingRowsCount


## Синтаксис


WritingRowsCount: Integer;


## Описание


Метод WritingRowsCount возвращает
 число строк, выгруженных в приёмник данных.


## Комментарии


Для выгрузки данных в источник используйте методы [IDtCustomConsumer.Put](IDtCustomConsumer.Put.htm)
 и [IDtCustomConsumer.PutRow](IDtCustomConsumer.PutRow.htm).
