# IDtObject.Metabase


## Синтаксис


Metabase: [IMetabase](KeSom.chm::/Interface/IMetabase/IMetabase.htm);


## Описание


Свойство Metabase определяет
 репозиторий, в котором выполняется импорт или экспорт данных.


## Комментарии


Данное свойство необходимо обязательно задавать для использования фильтрации
 при импорте данных. Для задания условия фильтрации используйте свойство
 [IDtProvider.FilterIf](../IDtProvider/IDtProvider.FilterIf.htm).
