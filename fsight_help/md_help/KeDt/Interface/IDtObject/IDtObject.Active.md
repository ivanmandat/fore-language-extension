# IDtObject.Active


## Синтаксис


Active: Boolean;


## Описание


Свойство Active возвращает состояние
 источника/приёмника данных.


## Комментарии


Свойство возвращает True, если
 объект [открыт](IDtObject.Open.htm) и False,
 если объект [закрыт](IDtObject.Close.htm).
