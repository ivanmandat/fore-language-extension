# IDtFieldDefinition.IsCalculated


## Синтаксис


IsCalculated: Boolean;


## Описание


Свойство IsCalculated определяет,
 является ли поле вычисляемым.


## Комментарии


Допустимые значения:


	- True. Поле является
	 вычисляемым. Выражение для расчета поля определяет свойство [IDtFieldDefinition.Expression](IDtFieldDefinition.Expression.htm);


	- False. Поле не является
	 вычисляемым.
