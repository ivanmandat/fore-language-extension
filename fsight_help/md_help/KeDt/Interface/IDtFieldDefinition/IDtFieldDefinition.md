# IDtFieldDefinition


Сборка: Dt;


## Описание


Интерфейс IDtFieldDefinition
 содержит настройки поля источника/приёмника данных.


## Иерархия наследования


           IDtFieldDefinition


## Комментарии


Настройки отдельного поля можно получить, используя свойства и методы
 интерфейса [IDtFieldDefinitions](../IDtFieldDefinitions/IDtFieldDefinitions.htm).
