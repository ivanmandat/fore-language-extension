# IDtCalcFieldError


Сборка: Ms;


## Описание


Интерфейс IDtCalcFieldError
 предназначен для работы с ошибкой, возникшей при расчете вычисляемого
 поля.


## Иерархия наследования


           IDtCalcFieldError


           [IDtError](../IDtError/IDtError.htm)


## Комментарии


Для работы с коллекцией ошибок используйте интерфейс [IDtCalcFieldErrors](../IDtCalcFieldErrors/IDtCalcFieldErrors.htm).
