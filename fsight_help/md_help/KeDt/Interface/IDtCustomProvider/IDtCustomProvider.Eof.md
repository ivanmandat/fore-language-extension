# IDtCustomProvider.Eof


## Синтаксис


Eof: Boolean;


## Описание


Свойство Eof возвращает признак
 того, что курсор расположен на последней записи источника данных.


## Комментарии


Если Eof возвращает значение
 True, то достигнут конец данных.
