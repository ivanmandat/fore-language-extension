# IDtCustomProvider.ImplementFetchRows


## Синтаксис


ImplementFetchRows: Boolean;


## Описание


Метод ImplementFetchRows возвращает
 признак того, реализован ли метод [IDtCustomProvider.FetchRows](IDtCustomProvider.FetchRows.htm).


## Комментарии


Допустимые значения:


	- True. Метод [IDtCustomProvider.FetchRows](IDtCustomProvider.FetchRows.htm)
	 реализован и доступен для использования;


	- False. Метод [IDtCustomProvider.FetchRows](IDtCustomProvider.FetchRows.htm)
	 не реализован. Используйте [IDtCustomProvider.Fetch](IDtCustomProvider.Fetch.htm).
