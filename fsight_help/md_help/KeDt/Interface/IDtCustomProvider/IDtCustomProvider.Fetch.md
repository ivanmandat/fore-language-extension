# IDtCustomProvider.Fetch


## Синтаксис


Fetch(Var Values: Array)


## Параметры


Values. Одномерный массив,
 в который будут помещены считанные значения.


## Описание


Метод Fetch выполняет чтение
 текущей записи источника данных.


## Комментарии


После чтения данных курсор перемещается на следующую запись в источнике.
