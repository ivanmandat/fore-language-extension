# DtTextFormatType


## Описание


Перечисление DtTextFormatType
 содержит форматы структуры файла, в соответствии с которым будет производиться
 импорт данных.


Используется следующими свойствами и методами:


	- [IDtTextProvider.FormatType](../Interface/IDtTextProvider/IDtTextProvider.FormatType.htm)


	- [IDtTextConsumer.FormatType](../Interface/IDtTextConsumer/IDtTextConsumer.FormatType.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Fixed - фиксированный.
		 Поля данных имеют фиксированную позицию и размер.


		 1
		 Delimited - с разделителями.
		 Поля данных разделены текстовыми символами.
