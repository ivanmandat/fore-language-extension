# DtExcelRangeType


## Описание


Перечисление DtExcelRangeType
 содержит типы диапазона данных на листе Microsoft Excel.


Используется внутренними свойствами и методами.


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 0
		 Sheet. Весь лист.


		 1
		 Range. Диапазон.


		 2
		 RangeFromColumns.
		 Диапазон по столбцам и до конца данных.


		 3
		 RangeFromCell. Диапазон
		 с ячейки и до конца данных.


## Комментарии


Примеры диапазонов:


	- Range. E6:E12;G6:G12;


	- RangeFromColumns. A:С;
	 E; G;


	- RangeFromCell. E6.
