# DtRestProviderResultType


## Описание


Перечисление DtRestProviderResultType
 содержит форматы данных, в которых может быть получен ответ от REST-сервиса.


Используется следующими свойствами и методами:


	- [IDtRestProvider.ResultType](../Interface/IDtRestProvider/IDtRestProvider.ResultType.htm).

		 XML.
