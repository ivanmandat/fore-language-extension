# Перечисления сборки Dt


		 Перечисление
		 Краткое описание


		 ![](../Enums_Image.gif)
		 [DtAutoFillFieldsMode](DtAutoFillFieldsMode.htm)


		 Перечисление DtAutoFillFieldsMode
		 содержит режимы распознавания типа полей источника данных.


		 ![](../Enums_Image.gif)
		 [DtExcelImexMode](DtExcelImexMode.htm)


		 Перечисление DtExcelImexMode
		 содержит режимы, в которых будет производиться импорт/экспорт
		 значений из файлов Excel.


		 ![](../Enums_Image.gif)
		 [DtExcelRangeType](DtExcelRangeType.htm)


		 Перечисление DtExcelRangeType
		 содержит типы диапазона данных на листе Microsoft Excel.


		 ![](../Enums_Image.gif)
		 [DtObjectType](DtObjectType.htm)


		 Перечисление DtObjectType
		 используется для определения типа источника/приёмника данных.


		 ![](../Enums_Image.gif)
		 [DtRestProviderResultType](DtRestProviderResultType.htm)


		 Перечисление DtRestProviderResultType
		 содержит форматы данных, в которых может быть получен ответ от
		 REST-сервиса.


		 ![](../Enums_Image.gif)
		 [DtTextFormatType](DtTextFormatType.htm)


		 Перечисление DtTextFormatType
		 содержит форматы структуры файла, в соответствии с которым будет
		 производиться импорт данных.


		 ![](../Enums_Image.gif)
		 [DtUnpivotCellPosition](DtUnpivotCellPosition.htm)


		 Перечисление DtUnpivotCellPosition
		 содержит варианты расположения ячейки в исходной таблице источника
		 до преобразования кросс-таблицы.


		 ![](../Enums_Image.gif)
		 [DtWebTableIdentificationType](DtWebTableIdentificationType.htm)


		 Перечисление DtWebTableIdentificationType
		 содержит способы идентификации таблицы на Веб-странице.


		 ![](../Enums_Image.gif)
		 [DtXmlFormatType](DtXmlFormatType.htm)


		 Перечисление DtXmlFormatType
		 содержит форматы структуры файла, в соответствии с которым будет
		 производиться импорт данных.
