# DtWebTableIdentificationType


## Описание


Перечисление DtWebTableIdentificationType
 содержит способы идентификации таблицы на Веб-странице.


Используется следующим свойством:


	- [IDtWebProvider.TableIdentificationType](../Interface/IDtWebProvider/IDtWebProvider.TableIdentificationType.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 ByNumber - идентификация
		 таблицы по номеру (порядковый номер тега <Table> в странице).


		 1
		 ByAttribute - идентификация
		 таблицы по значению определенного атрибута, входящего в тег <Table>.
