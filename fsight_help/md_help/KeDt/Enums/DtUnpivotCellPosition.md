# DtUnpivotCellPosition


## Описание


Перечисление DtUnpivotCellPosition
 содержит варианты расположения ячейки в исходной таблице источника до
 преобразования кросс-таблицы.


Используется следующими свойствами и методами:


	- [IDtUnpivotProvider.SourceCellPosition](../Interface/IDtUnpivotProvider/IDtUnpivotProvider.SourceCellPosition.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Unknown. Расположение
		 ячейки не определено.


		 1
		 Left. Ячейка располагалась
		 в заголовке строк.


		 2
		 Head. Ячейка располагалась
		 в заголовке столбцов.


		 3
		 Data. Ячейка располагалась
		 в области данных.


		 4
		 Fixed. Ячейка располагалась
		 среди фиксированных.
