# DtExcelImexMode


## Описание


Перечисление DtExcelImexMode
 содержит режимы, в которых будет производиться импорт/экспорт значений
 из файлов Excel.


Используется следующими свойствами и методами:


	- [IDtExcelConsumer.ImexMode](../Interface/IDtExcelConsumer/IDtExcelConsumer.ImexMode.htm)


	- [IDtExcelProvider.ImexMode](../Interface/IDtExcelProvider/IDtExcelProvider.ImexMode.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Export.
		 Не поддерживается.


		 1
		 Import.
		 Осуществляется импорт/экспорт значений различных типов.

Тип данных, применяемый при импорте/экспорте,
		 определяется по первым n
		 ячейкам. Значение n соответствует
		 значению ключа TypeGuessRows.


В данном режиме используются следующие правила
		 определения типа данных:


			- В первых n
			 ячейках содержатся данные разных типов - при импорте/экспорте
			 все значения будут преобразовываться к типу, указанному в
			 ключе ImportMixedTypes
			 (по умолчанию тип данных Text).

			- В первых n
			 ячейках содержатся данные одного типа - при импорте/экспорте
			 все значения будут преобразовываться к типу, который имеют
			 первые n ячеек. Значения,
			 для которых преобразование невозможно, будут заменяться пустыми
			 значениями.


		 2
		 Linked.
		 Тип данных определяется по значениям первых ячеек.

Тип данных, применяемый при импорте/экспорте,
		 определяется по первым n
		 ячейкам. Значение n соответствует
		 значению ключа TypeGuessRows.


Значения, не соответствующие определенному типу,
		 будут обнуляться.

В данном режиме используются следующие правила
		 определения типа данных:


			- В первых n
			 ячейках содержатся данные разных типов - при импорте/экспорте
			 будет использоваться тип данных значений, количество которых
			 больше. Значения всех других типов будут заменяться пустыми
			 значениями.

			- В первых n
			 ячейках содержатся данные одного типа - данный тип будет использоваться
			 при импорте/экспорте. Значения всех других типов будут заменяться
			 пустыми значениями.
