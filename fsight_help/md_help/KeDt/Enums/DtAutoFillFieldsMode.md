# DtAutoFillFieldsMode


## Описание


Перечисление DtAutoFillFieldsMode
 содержит режимы распознавания типа полей источника данных.


Используется следующим свойством:


	- [IDtProvider.AutoFillFieldsMode](../Interface/IDtProvider/IDtProvider.AutoFillFieldsMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Strings. Все поля источника
		 данных текстовые.


		 1
		 DataRow. Тип поля определяется
		 исходя из данных, которые в нем содержатся.


		 2
		 Default_. Используются
		 типы данных, заданные в источнике.


		 3
		 DataRowEx. Тип поля
		 определяется исходя из данных, которые в нем содержатся. При этом
		 игнорируется формат даты «YYYY», чтобы целые числа не распознавались,
		 как годы.
