# IBusinessApplication


Сборка: DBA;


## Описание


Интерфейс IBusinessApplication
 описывает объект репозитория - [Бизнес-приложение](Constructor.chm::/Intro/Designer_business_applications.htm).


## Иерархия наследования


IBusinessApplication


## Комментарии


К интерфейсу IBusinessApplication могут быть приведены объекты, которые
 созданы с помощью конструкторов класса BusinessApplication, который реализован
 в [сборке](../../Intro/ModDBA_Programming.htm) компонента «Бизнес-приложение».
