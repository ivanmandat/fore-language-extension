# IBusinessApplication.IsEdited


## Синтаксис


IsEdited: Boolean;


## Описание


Свойство IsEdited возвращает
 признак нахождения бизнес-приложения в режиме редактирования.


## Комментарии


Свойство возвращает значение True,
 если бизнес-приложение открыто на редактирование и False,
 если на просмотр.
