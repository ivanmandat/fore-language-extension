# IMenuIconEditor.IsSvg


## Синтаксис


IsSvg: Boolean;


## Описание


Свойство IsSvg возвращает признак
 задания пиктограммы в формате SVG.


## Комментарии


Свойство возвращает значение True,
 если используется пиктограмма в SVG формате и False,
 если используется другой графический формат.
