# Редактор формул


Редактор формул используется для создания различных формул, выражений
 и условий, необходимых при создании различных объектов или их параметров.


Важно. Редактор
 формул устарел, по умолчанию вместо него используется [редактор выражения](UiNav.chm::/GUI/ExpressionEditor.htm).
 Если вы хотите использовать редактор формул, то установите значение «1»
 для ключа реестра [HKEY_CURRENT_USER\Software\Foresight\Foresight Analytics Platform\10.0\UiTs\OldFormulaEditor].
 По умолчанию ключ имеет значение «0».


![](equation%20editor.png)


Окно «Редактор формул» включает
 в себя несколько областей:


[![](../Opened.gif)![](../Closed.gif)Область «Формула»](javascript:TextPopup(this))


	В данном поле отображается формируемое выражение, формула или условие.
	 Формировать выражение можно набором с клавиатуры необходимых символов
	 и знаков или используя область данных и соответствующие кнопки в окне
	 редактора формул.


	В выражениях допускается использование:


		- [глобальных
		 переменных](UiNav.chm::/02_Navigator/UiNav_GlobalVariables.htm). Для этого в формульном поле следует
		 указать идентификатор глобальной переменной;


		- прикладных функций. Для использования прикладных функций
		 из модуля следует указать полный путь к функции в формате: <идентификатор
		 модуля>.<имя функции>.[(значения параметров)];


		- статических методов прикладных и системных классов
		 (см. также: [Классы](fore.chm::/02_generalinfo/fore_classesandobjects.htm),
		 [Классы
		 сборки System](foresys.chm::/class/system_classes.htm),
		 [IStatistics](statlib.chm::/interface/istatistics/istatistics.htm)
		 и [MathFin](mathlib.chm::/interface/interfaces.htm)).
		 Для использования методов необходимо указать полный путь к методу
		 в формате: <имя модуля>.<имя класса>.<имя метода>;


		- условного оператора. Для использования условного оператора
		 следует ввести строку следующей структуры: <условие>
		 ? <значение,
		 если условие выполняется> :
		 <значение, если условие не
		 выполняется>.


[![](../Opened.gif)![](../Closed.gif)Область кнопок](javascript:TextPopup(this))


	Кнопки реализуют вставку математических функций и операций в формулу.
	 Для появления необходимого знака операции или функции в области отображения
	 формул необходимо нажать соответствующую кнопку.


			 Кнопка
			 Описание


			 +
			 Операция сложения.


			 -
			 Операция вычитания.


			 *
			 Операция умножения.


			 /
			 Операция деления.


			 =
			 Знак «равно».


			 <>
			 Знак «не равно».


			 <
			 Знак «меньше».


			 >
			 Знак «больше».


			 <=
			 Знак «меньше или равно».


			 >=
			 Знак «больше или равно».


			 ( )
			 Скобки.


			 DIV
			 Частное двух операндов.


			 MOD
			 Остаток от деления двух операндов.


			 AND
			 Союз «и».


			 OR
			 Союз «или».


			 NOT
			 Союз «отрицание».


	Примечание.
	 При выполнении операций MOD
	 и DIV для операндов целого
	 типа результат также будет целого типа.


[![](../Opened.gif)![](../Closed.gif)Область «Данные»](javascript:TextPopup(this))


	Содержание области данных зависит от того, откуда открыт редактор
	 формул. Область данных может содержать в виде дерева поля, атрибуты,
	 параметры объекта (куба, таблицы, справочника и др.) или дополнительные
	 функции, которые можно использовать для составления выражения. Для
	 появления необходимого идентификатора в области формул необходимо
	 дважды щёлкнуть по нему.
