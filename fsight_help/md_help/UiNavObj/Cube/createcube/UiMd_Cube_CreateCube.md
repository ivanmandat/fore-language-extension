# Куб


Для создания куба в [навигаторе
 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm):


	- в веб-приложении нажмите кнопку ![](Master_Standart/add_button.png) «Создать» в главном меню и выберите
	 тип объекта на боковой панели «[Новый объект](UiNav.chm::/02_Navigator/General_Principles_of_Work.htm#add_object)»;


	- в настольном приложении выполните одно из действий:


		- нажмите кнопку «Новый объект
		 > Куб» в группе «Создать»
		 на вкладке «Главная» ленты
		 инструментов;


		- выполните команду «Создать
		 > Куб» в контекстном меню навигатора объектов.


Выберите тип создаваемого куба на странице
 «Тип объекта» в мастере куба.


Примечание.
 В настольном приложении куб можно создать также с помощью [мастера
 импорта данных](../../Data_import_wizard/Data_import_wizard.htm).


Доступно создание следующих типов
 кубов:


	- [стандартный
	 куб](Master_Standart/UiMd_Cube_CreateCube_Master_Standart.htm);


	- [представление-куб](Master_Conception/UiMd_Cube_CreateCube_Master_Conception.htm);


	- [виртуальный
	 куб](Master_Virtual_Cube/UiMd_Cube_CreateCube_Master_Virtual.htm);


	- [вычисляемый
	 куб](Master_Calculation/UiMd_Cube_CreateCube_Master_Calculation.htm);


	- [автоматический
	 куб](Master_Auto/UiMd_Cube_CreateCube_Master_Auto.htm).


Примечание.
 Создание автоматических кубов доступно только в настольном приложении.
