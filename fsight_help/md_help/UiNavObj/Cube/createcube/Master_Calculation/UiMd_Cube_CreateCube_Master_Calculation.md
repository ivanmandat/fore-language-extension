# Вычисляемый куб


Вычисляемый куб предназначен для получения новых данных, вычисленных
 на основе данных из других кубов. Для создания вычисляемого куба требуется
 один или несколько [кубов](../../UiMd_Cube_Type.htm) любого
 вида. Измерения в вычисляемом кубе могут быть скопированы из кубов-источников
 или созданы на основе произвольных [справочников](../../../reference_book/Create.htm).
 Факты могут быть сформированы пользователем или также скопированы из куба-источника.
 Все формулы для расчёта данных куба задаются пользователем.


Мастер вычисляемого куба содержит
 страницы:


	- [Базовые
	 настройки/Базовые свойства](base_settings.htm). Задаются наименование
	 объекта, идентификатор и примечание;


	- [Параметры](parameters.htm).
	 Формируется список параметров;


	- [Факты](UiMd_Cube_CreateCube_Master_Calculation_1.htm).
	 Формируется список фактов куба, которые в дальнейшем будут отражать
	 непосредственные данные куба;


	- [Структура
	 кубов](UiMd_Cube_CreateCube_Master_Calculation_2.htm). Указываются кубы-источники, данные которых
	 будут использоваться при расчете, и формируется структура вычисляемого
	 куба;


	- [Альтернативный куб](UiMd_Cube_CreateCube_Master_Calculation_alternative_cube.htm).
	 Указывается куб, в который будут сохраняться рассчитанные значения,
	 в случае если исходный куб не поддерживает сохранение;


	- [Единицы
	 измерения и показатели](UiMd_Cube_CreateCube_Master_Calculation_unit.htm). Указываются справочники
	 для организации процесса автоматического пересчета данных из одних
	 единиц измерения в другие;


	- [Управление
	 формулами](UiMd_Cube_CreateCube_Master_Calculation_3.htm). Указывается измерение и атрибут измерения,
	 по значениям которого будет определяться период действия формул;


	- [Управление
	 параметрами](UiMd_Cube_CreateCube_Master_Calculation_manage_param.htm). Настраивается передача значений параметров
	 для открытия объектов, которые используются в кубе;


	- [Управляющие
	 измерения](UiMd_Cube_CreateCube_Master_Calculation_managed_dim.htm). Настраивается взаимодействие параметрических
	 измерений куба с другими измерениями;


	- [Редактор
	 формул](../../../equation_editor/UiMd_Equation_editor_CalculationCube.htm). Настраиваются формулы расчёта. Страница доступна
	 только в веб-приложении.


Для сохранения настроек в мастере куба:


	- в веб-приложении:


		- нажмите кнопку «Сохранить»;


		- выполните команду «Сохранить
		 и закрыть» в раскрывающемся меню кнопки «Сохранить».
		 После этого будут сохранены заданные настройки, закрыт мастер
		 и выполнен переход в [навигатор
		 объектов](getstarted.chm::/Interface/Interface_Navigator.htm).


Примечание.
 Команда «Сохранить и закрыть»
 доступна только при [редактировании
 настроек](../../Work_Cube/UiMd_Cube_Work_Cube.htm#edit).


	- в настольном приложении нажмите кнопку «Готово».


Для задания/редактирования формул вычисляемого куба выполните команду
 «Редактировать формулы» в контекстном
 меню. Будет открыт [редактор
 формул](../../../equation_editor/UiMd_Equation_editor_CalculationCube.htm). Возможность доступна только в настольном приложении.
