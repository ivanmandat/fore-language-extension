# Стандартный куб


Стандартный куб предназначен для представления данных из нескольких
 различных источников. Для создания куба требуются [справочники](../../../reference_book/Create.htm)
 и [наборы данных](../../../Table/Master/object_type_page.htm).
 Особенность данного вида куба в том, что данные для фактов могут быть
 получены из набора данных или вычислены по формуле, заданной пользователем.


Мастер стандартного куба содержит страницы:


	- [Базовые
	 настройки/Базовые свойства](base_settings.htm). Задаются наименование объекта,
	 идентификатор и примечание;


	- [Параметры](parameters.htm).
	 Формируется список параметров;


	- [Варианты
	 отображения](Display_Version_Options.htm). Формируется список вариантов отображения куба,
	 при переключении между которыми задаются соответствующие настройки
	 на страницах ниже. В зависимости от выбранного настраиваемого варианта
	 отображения настройки отличаются. Страница доступна только в веб-приложении;


	- [Факты](UiMd_Cube_CreateCube_Master_Standart_1.htm).
	 Формируется список фактов куба, которые в дальнейшем будут отражать
	 непосредственные данные куба;


	- [Привязка
	 фактов](UiMd_Cube_CreateCube_Master_Standart_2.htm). Осуществляется привязка фактов к полям источников данных,
	 в которых хранятся значения ячеек куба;


	- [Измерения](UiMd_Cube_CreateCube_Master_Standart_3.htm).
	 Формируется многомерная структура куба и осуществляется привязка каждого
	 измерения к полям источников данных, по значениям которых будут формироваться
	 координаты ячеек куба;


	- [Единицы измерения и показатели](UiMd_Cube_CreateCube_Master_Standart_unit_dim.htm).
	 Указываются справочники для организации процесса автоматического пересчёта
	 данных из одних единиц измерения в другие;


	- [Отношения](UiMd_Cube_CreateCube_Master_Standart_4.htm).
	 Формируется список специальных элементов, которые могут использоваться
	 в вычисляемых фактах и позволяют в каждой точке расчёта получать и
	 использовать значения точек с другими координатами;


	- [Вычисляемые
	 факты](UiMd_Cube_CreateCube_Master_Standart_5.htm). Настраиваются факты куба, которые напрямую не связаны с
	 полями источников данных, а рассчитываются по формулам;


	- [Дополнительные
	 настройки](Additional_Settings.htm). Настраиваются параметры для работы с данными
	 стандартного куба;


	- [Управление параметрами](UiMd_Cube_CreateCube_Master_Standart_manage_param.htm).
	 Настраивается передача значений параметров для открытия объектов,
	 которые используются в кубе;


	- [Управляющие измерения](UiMd_Cube_CreateCube_Master_Standart_managed_dim.htm).
	 Настраивается взаимодействие параметрических измерений куба с другими
	 измерениями;


	- [Агрегация
	 данных](UiMd_Cube_CreateCube_Master_Standart_7.htm). Настраивается агрегация данных по различным уровням измерений
	 куба;


	- [Обработчик
	 событий](uimd_cube_createcube_master_standart_8.htm). Настраивается обработчик событий, которые будут
	 происходить в кубе. Страница доступна только в веб-приложении.


Для сохранения настроек в мастере куба:


	- в веб-приложении:


		- нажмите кнопку «Сохранить»;


		- выполните команду «Сохранить
		 и закрыть» в раскрывающемся меню кнопки «Сохранить».
		 После этого будут сохранены заданные настройки, закрыт мастер
		 и выполнен переход в [навигатор
		 объектов](getstarted.chm::/Interface/Interface_Navigator.htm).


Примечание.
 Команда «Сохранить и закрыть»
 доступна только при [редактировании
 настроек](../../Work_Cube/UiMd_Cube_Work_Cube.htm#edit).


	- в настольном приложении нажмите кнопку «Готово».


Дополнительно для стандартного куба могут быть настроены [сегменты](../../segments/segments.htm),
 которые определят права на чтение/запись данных в кубе. Настройка сегментов
 доступна только в веб-приложении.
