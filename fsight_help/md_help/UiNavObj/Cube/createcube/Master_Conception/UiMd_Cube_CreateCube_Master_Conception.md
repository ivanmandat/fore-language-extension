# Представление-куб


Представление-куб предназначен для усечения объёма данных, предоставляемых
 каким-либо другим кубом. Для создания представления-куба требуется [куб](../../UiMd_Cube_Type.htm) любого вида. Изначально все
 измерения исходного куба доступны в представлении-кубе. Для усечения объёма
 данных выполняется «фиксация» требуемых измерений, то есть задание фиксированной
 отметки для элементов измерений. Таким образом, происходит уменьшение
 размерности итогового представления-куба.


Примечание.
 Если в качестве исходного куба был выбран куб с [настроенной
 возможностью привязки вложений](../Master_Standart/UiMd_Cube_CreateCube_Master_Standart_2.htm), то и представление-куб тоже будет
 иметь такую возможность.


При открытии куба-представления в экспресс-отчете, проверяется куб-источник
 на наличие новых, отсутствующих в кубе-представлении, измерений.


Уже открытый куб-представление обновляется к новому списку измерений,
 таким образом при повторном открытии источников данных, будет обновлен
 актуальный список измерений.


Мастер представления-куба содержит
 страницы:


	- [Базовые
	 настройки/Базовые свойства](base_settings.htm). Задаются наименование
	 объекта, идентификатор и примечание;


	- [Параметры](parameters.htm).
	 Формируется список параметров;


	- [Исходный
	 куб](UiMd_Cube_CreateCube_Master_Conception_1.htm). Задается куб, на базе которого будет строиться
	 текущий представление-куб;


	- [Фиксация
	 измерений](UiMd_Cube_CreateCube_Master_Conception_2.htm). Осуществляется фиксация элементов измерений
	 для уменьшения структуры формируемого представления-куба;


	- [Единицы
	 измерения и показатели](UiMd_Cube_CreateCube_Master_Conception_unit.htm). Указываются справочники
	 для организации процесса автоматического пересчета данных из одних
	 единиц измерения в другие;


	- [Управление
	 параметрами](UiMd_Cube_CreateCube_Master_Conception_manage_dim.htm). Настраивается передача значений параметров
	 для открытия объектов, которые используются в кубе;


	- [Управляющие
	 измерения](UiMd_Cube_CreateCube_Master_Conception_managed_dim.htm). Настраивается взаимодействие параметрических измерений
	 куба с другими измерениями.


Для сохранения настроек в мастере куба:


	- в веб-приложении:


		- нажмите кнопку «Сохранить»;


		- выполните команду «Сохранить
		 и закрыть» в раскрывающемся меню кнопки «Сохранить».
		 После этого будут сохранены заданные настройки, закрыт мастер
		 и выполнен переход в [навигатор
		 объектов](getstarted.chm::/Interface/Interface_Navigator.htm).


Примечание.
 Команда «Сохранить и закрыть»
 доступна только при [редактировании
 настроек](../../Work_Cube/UiMd_Cube_Work_Cube.htm#edit).


	- в настольном приложении нажмите кнопку «Готово».
