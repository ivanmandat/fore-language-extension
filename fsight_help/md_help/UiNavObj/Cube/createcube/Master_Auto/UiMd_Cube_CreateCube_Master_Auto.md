# Автоматический куб


Автоматический куб предназначен для формирования источника данных по
 принципу «сверху-вниз». Для создания автоматического куба требуются [база данных](../../../database/UiDb_database.htm) и [справочники](../../../reference_book/Create.htm).
 На основе справочников будут созданы измерения, а для хранения данных
 будет использоваться указанная БД. Данные в куб могут быть внесены пользователем
 или загружены из указанного куба.


Примечание.
 Перед созданием автоматического куба убедитесь, что репозиторий метаданных
 НСИ [создан](Setup.chm::/05_RepoMngr/Setup_repoMngr_CreateRDSRepo.htm)
 и [обновлен](Setup.chm::/05_repomngr/setup_repomngr_updaterdsrepo.htm).


Мастер автоматического куба содержит
 страницы:


	- [Базовые
	 свойства](base_settings.htm). Задаются наименование объекта,
	 идентификатор и примечание;


	- [Параметры](parameters.htm).
	 Формируется список параметров;


	- [Структура](UiMd_Cube_CreateCube_Master_Auto_1.htm).
	 Указывается база данных, в которой будут храниться данные куба, и
	 с помощью справочников формируется многомерная структура куба;


	- [Единицы
	 измерения и показатели](uimd_cube_createcube_common_page4.htm). Указываются справочники
	 для организации процесса автоматического пересчета данных из одних
	 единиц измерения в другие;


	- [Управляющие
	 измерения](uimd_cube_createcube_common_page3.htm). Настраивается взаимодействие параметрических
	 измерений куба с другими измерениями;


	- [Агрегация
	 данных](UiMd_Cube_CreateCube_Master_Auto_Aggreg.htm). Настраивается агрегация данных по различным
	 уровням измерений куба;


	- [Источник
	 данных](UiMd_Cube_CreateCube_Master_Auto_2.htm). Указывается альтернативный источник, данные
	 которого будут использоваться автоматическим кубом;


	- [Управление
	 параметрами](uimd_cube_createcube_common_page2.htm). Настраивается передача значений параметров
	 для открытия объектов, которые используются в кубе;


	- [Фиксация
	 измерений](UiMd_Cube_CreateCube_Master_Auto_3.htm). Осуществляется фиксация измерений куба-источника, которые
	 не были сопоставлены с измерениями автоматического куба.


Примечание.
 Мастер автоматического куба доступен только в настольном приложении.
