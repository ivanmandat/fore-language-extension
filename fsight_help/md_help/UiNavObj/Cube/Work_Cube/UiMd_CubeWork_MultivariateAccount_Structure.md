# Структура


На странице «Структура» определяется
 куб-назначение и при необходимости фиксируются измерения:


![](MultivariateAccount%20Structure.png)


Древовидный список «Кубы-источники»
 отображает все кубы-источники, добавленные на предыдущей странице, вместе
 с их измерениями. В дочерних элементах измерений отображается привязка
 к полям таблиц, а в дочерних элементах измерений «Показатели»
 отображаются факторы кубов.


В поле «Куб-назначение» производится
 выбор куба, в который будут занесены результаты расчетов. При выборе куба-назначения
 (приёмник) необходимо выполнение следующих условий:


	- источники и приёмник должны иметь одинаковый тип СУБД;


	- если источники и приёмники находятся на одном сервере, но в
	 разных схемах, для их базы данных должен быть установлен флажок «Использовать учетные данные, указанные
	 при входе в систему».


Если условия не выполнены, то будет выдано сообщение о невозможности
 выбора данного куба в качестве приёмника.


После выбора куба-назначения ниже в древовидном списке будут отражены
 его измерения.


Для календарных измерений-источников и для измерений с привязкой к полю
 таблицы типа «Дата» существует
 возможность при задании формул использовать сдвиг по дате (функция [Shift](UiMd_CubeWork_MultivariateAccount_Functions.htm)).
 Для этого у выбранного измерения в контекстном меню установите флажок
 «Возможность сдвига по дате (SHIFT)».


Для задания наименований, используемых для отображения элементов справочника
 в [редакторе
 формул многомерного расчета](UiMd_CubeWork_MultivariateAccount_EditFormula.htm), выполните команду контекстного меню «Наименование в измерении» и во всплывающем
 меню выберите атрибут, значение которого будет использоваться для формирования
 наименований элементов.


Для задания наименований, используемых для отображения элементов справочника
 в выражениях [редактора
 выражения](UiNav.chm::/GUI/ExpressionEditor.htm), выполните команду контекстного меню «Наименование для формул» и во всплывающем
 меню выберите атрибут, значение которого будет использоваться для формирования
 наименований. Для скрытия наименований элементов в выражениях выберите
 пункт меню «Скрыть в формулах»,
 для восстановления отображения наименований – «Отобразить
 в формулах». Пункт «Наименование
 для формул» отображается только для измерений-источников.


Для перемещения фокуса в окне навигатора к выбранному измерению выполните
 команду контекстного меню «Перейти к
 объекту».


Для запрета [суммирования
 по элементам](UiMd_CubeWork_MultivariateAccount_EditFormula.htm) кубов-источников установите в контекстном меню флажок
 «Запретить суммирование». По умолчанию
 флажок снят (суммирование разрешено).


Примечание.
 Если в кубе-источнике нет свободных рекурсивных измерений, то для него
 данный флажок будет установлен и недоступен для изменения значения. Если
 в таком кубе появляется свободное рекурсивное измерение, то после обновления
 структуры куба флажок будет снят.


Для обновления структуры кубов-источников нажмите кнопку «Обновить
 структуру». Операция необходима для обновления состояния флажка
 «Запретить суммирование», если
 изменялся состав зафиксированных измерений.


Настройка способа отображения кубов и измерений производится через контекстное
 меню:


	- Отображать наименования.
	 В дереве будут отображены только наименования;


	- Отображать идентификаторы.
	 В дереве будут отображены только идентификаторы;


	- Отображать наименования и идентификаторы.
	 В дереве будут отображены только наименования и идентификаторы.


Для перехода на следующую страницу мастера необходимо воспользоваться
 кнопкой «Далее».


## Фиксация измерений


На данной странице следует зафиксировать измерения, элементы которых
 не будут использоваться в формулах расчета. Фиксация позволяет установить
 соответствие между элементами справочников.


Примечание.
 Зафиксированные измерения в кубе-назначении должны быть связаны с измерениями
 в источнике.


Для фиксации измерений:


	- дважды щёлкните по измерению. Данное действие применимо только
	 для измерений куба-источника;


	- установите флажок напротив пункта контекстного меню «Зафиксировать
	 измерение».


Примечание.
 Нельзя зафиксировать измерения «Показатели».


После выполнения одного из действий для измерений кубов-источников будет
 открыто окно, выберите справочник куба-назначения, с которым будет установлено
 соответствие:


![](MultivariateAccount%20FixDim.png)


Соответствие между элементами измерений устанавливается по индексу.
 В списке «Измерение назначения»
 будут отображаться только те измерения, которые не участвуют в связях
 с другими измерениями куба-источника, для измерения которого было открыто
 данное окно.


При фиксации справочников возможны варианты:


	- справочник куба-источника и сопоставленный ему справочник куба-назначения
	 зафиксированы - элементы данных справочников нельзя будет выбрать
	 при составлении формул расчета, расчет будет производиться для всех
	 точек измерения назначения;


	- справочник куба-источника зафиксирован, а сопоставленный ему
	 справочник куба-назначения не зафиксирован - при создании формул можно
	 использовать элементы справочника куба-назначения, расчет будет производиться
	 только для точек измерения назначения, для которых задана формула
	 расчета.


При выборе зафиксированного измерения куба-назначения, связанные с ним
 измерения кубов-источников будут подсвечиваться, тем самым будут видны
 связи между измерениями.


Для снятия фиксации измерения:


	- дважды щёлкните по измерению. Данное действие применимо только
	 для измерений куба-источника;


	- снимите флажок напротив пункта контекстного меню «Зафиксировать
	 измерение».
