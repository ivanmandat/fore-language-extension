# Расшифровка таблицы формул


Таблица формул содержит следующие поля:


		 Идентификатор поля
		 Описание


		 FACT
		 Код вычисляемого показателя (FACT TO)


		 FACF
		 Код исходного показателя (FACT FROM)


		 OPE
		 Код операции


		 OPE_RAND
		 Операнд


		 OPE_ORD
		 Порядок операции


		 CODT, CODT1, CODT2, ...
		 Результирующие измерения (назначение)


		 CODF, CODF1, CODF2, ...

CODF_B, CODF1_B, CODF2_B, ...

CODF_C, CODF1_C, CODF2_C, ...

CODF_D, CODF1_D, CODF2_D, ...

...
		 Свободные измерения кубов-источников


		 CUB
		 Куб-источник


		 DST
		 Дата начала периода


		 DFN
		 Дата окончания периода
