# Работа с кубами


По умолчанию открытие куба приводит к созданию его OLAP-представления
 или [экспресс-отчёта](uiexpress.chm::/purpose/uiexpress_purpose.htm)
 в терминах «Форсайт. Аналитическая платформа».


После настройки этот экспресс-отчёт может быть использован в качестве
 представления куба по умолчанию. Он будет открываться автоматически при
 открытии куба.


Примечание.
 При использовании куба в качестве [источника
 для рабочей книги](uidw.chm::/Purpose/UiDw_Purpose_Object.htm#select_ds) не требуется привязка справочников
 к репозиторию НСИ.


[![](../../Opened.gif)![](../../Closed.gif)Открытие
 куба на редактирование](javascript:TextPopup(this))


	Для открытия куба на редактирование в [навигаторе
	 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm) выделите куб и выполните одно из действий:


		- в веб-приложении:


			- выполните команду «Редактировать»
			 в контекстном меню куба;


			- выполните команду «Редактировать
			 в новой вкладке» в контекстном меню куба;


			- выполните команду «Редактировать
			 в новом окне» в контекстном меню куба;


			- нажмите клавишу F4;


		- в настольном приложении:


			- нажмите кнопку ![](../../reference_book/Work/b_edit.png)
			 «Редактировать» в
			 группе «Открыть» на
			 вкладке «Главная»
			 ленты инструментов;


			- выполните команду «Редактировать»
			 в контекстном меню куба;


			- нажмите клавишу F4.


	После выполнения одного из действий будет открыт [мастер
	 куба](../UiMd_Cube_Type.htm).


[![](../../Opened.gif)![](../../Closed.gif)Сохранение
 куба](javascript:TextPopup(this))


	Для сохранения нового куба или при открытии
	 куба на [редактирование](UiMd_Cube_Work_Cube.htm#edit)
	 в [мастере куба](../UiMd_Cube_Type.htm):


		- в веб-приложении:


			- нажмите кнопку «Сохранить»;


			- выполните команду «Сохранить
			 и закрыть» в раскрывающемся меню кнопки «Сохранить».
			 После этого будут сохранены заданные настройки, закрыт мастер
			 куба и выполнен переход в [навигатор
			 объектов](getstarted.chm::/Interface/Interface_Navigator.htm);


	Примечание.
	 Команда «Сохранить и закрыть»
	 доступна только при редактировании куба.


		- в настольном
		 приложении нажмите кнопку «Готово».


[![](../../Opened.gif)![](../../Closed.gif)Открытие куба
 на просмотр](javascript:TextPopup(this))


	Для открытия куба на просмотр в [навигаторе
	 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm) выделите куб и выполните одно из действий:


		- в веб-приложении:


			- выполните команду «Открыть»
			 в контекстном меню куба;


			- выполните команду «Открыть
			 в новом окне» в контекстном меню куба;


			- дважды щёлкните по кубу;


			- нажмите клавишу ENTER;


		- в настольном приложении:


			- нажмите кнопку ![](../../table/work/b_open.png) «Открыть»
			 в группе «Открыть»
			 на вкладке «Главная»
			 ленты инструментов;


			- выполните команду «Открыть»
			 в контекстном меню куба.


	После выполнения одного из действий будет открыт экспресс-отчет,
	 который был ранее сохранен для источника.


	Для открытия куба в другом инструменте выполните соответствующую
	 команду в раскрывающемся меню кнопки ![](../../table/work/b_open.png)
	 «Открыть» в группе «Открыть» на вкладке «Главная»
	 ленты инструментов:


		- «В
		 аналитической панели»;


		- «В
		 отчётах».


При открытии куба-представления в экспресс-отчете проверяется куб-источник
 на наличие новых, отсутствующих в кубе-представлении, измерений.


Примечание.
 Для удаления дочернего экспресс-отчета следует выполнить команду контекстного
 меню куба «Удалить сохраненное представление».
 Данный пункт контекстного меню будет отсутствовать, если для источника
 не сохранялся экспресс-отчет.
