# Многомерный расчет на сервере БД


Многомерный расчет на сервере БД реализует возможности вычисляемого
 куба, но непосредственный расчет производится на сервере БД, что позволяет
 освободить ресурсы локального компьютера в настольном приложении.
 Многомерный расчет, используя заданные формулы, производит вычисление
 значений фактов одного куба на основе значений фактов других кубов. Формулы
 расчета фактов хранятся в специальной таблице.


Для создания многомерного расчета выполните в навигаторе объектов:


	- команду контекстного меню «Создать >
	 Другие > Куб > Многомерный расчет на сервере БД»;


	- команду «Новый объект >
	 Другие > Куб > Многомерный расчет на сервере БД»
	 в группе «Создать» на вкладке
	 ленты «[Главная](GetStarted.chm::/Interface/Interface_Description.htm#customize_ribbon)».


После выполнения одного из действий
 будет открыт мастер многомерного расчета. Мастер содержит страницы:


	- [Базовые
	 свойства](base_settings.htm). Задаются наименование объекта,
	 идентификатор и примечание;


	- [Параметры](parameters.htm).
	 Формируется список параметров;


	- [Свойства](UiMd_CubeWork_MultivariateAccount_MDProps.htm).
	 Указывается соединение с базой данных, в которой будет производиться
	 расчет;


	- [Источники](UiMd_CubeWork_MultivariateAccount_Source.htm).
	 Формируется список стандартных кубов, по данным которых будет производиться
	 расчет;


	- [Структура](UiMd_CubeWork_MultivariateAccount_Structure.htm).
	 Указывается куб-приёмник, а также осуществляется фиксация измерений;


	- [Таблица
	 формул](UiMd_CubeWork_MultivariateAccount_Tables.htm). Указывается таблица, в которой будут храниться
	 формулы для расчета;


	- [Управление
	 параметрами](../UiMd_CubeWork_MultivariateAccount_ParamsControl.htm). Настраивается передача значений параметров
	 для открытия объектов, которые используются в многомерном расчете;


	- [Управляющие
	 измерения](UiMd_CubeWork_MultivariateAccount_ControlDims.htm). Настраивается управление параметрами данных
	 измерений с помощью других измерений кубов.


Для задания/редактирования формул многомерного расчета необходимо в
 навигаторе объектов выполнить команду контекстного меню «[Редактировать
 формулы](UiMd_CubeWork_MultivariateAccount_EditFormula.htm)».


Запуск расчета производится:


	- из окна [редактора
	 формул](UiMd_CubeWork_MultivariateAccount_EditFormula.htm);


	- из навигатора объектов:


	-


		- при выполнении команды «Выполнить»
		 в контекстном меню многомерного расчета;


		- при двойном щелчке по многомерному расчету.


После выполнения одного из действий будет
 открыто окно «[Параметры расчета](UiMd_CubeWork_MultivariateAccount_Account.htm)».
