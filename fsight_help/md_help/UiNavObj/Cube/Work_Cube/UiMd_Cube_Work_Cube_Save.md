# Сохранение куба в файл


Функция сохранения кубов в файл актуальна для всех видов кубов в настольном
 приложении.


Для сохранения куба в файл в навигаторе объектов необходимо вызвать
 контекстное меню для выбранного куба и выполнить команду «Сохранить
 в файл», после чего появится диалог сохранения. В диалоге необходимо
 задать имя файла. По умолчанию автоматически будет подставлено наименование
 сохраняемого куба.


В файл будет сохранена структура куба (параметры всех измерений, их
 расположение по умолчанию) и данные по всем координатам куба.


Сохраненный в файл куб в дальнейшем может использоваться для построения
 различных отчетов.
