# Источник и приёмник


На странице «Источник и приёмник»
 выбираются куб, из которого будут получены данные, и куб, в который данные
 будут сохранены:


![](page%203.png)


Раскрывающиеся списки «Куб-источник»
 и «Куб-приёмник» содержат списки
 объектов репозитория. Выберите куб-источник, который будет являться источником
 данных, и куб-приёмник, который будет являться приёмником данных. Указанные
 кубы должны иметь схожую структуру. По возможности кубы должны быть созданы
 на основе одинаковых справочников.


Для удаления выбранного куба из поля следует нажать на кнопку ![](../../CreateCube/Master_Calculation/alternative%20cube%20button.png) «Очистить».
