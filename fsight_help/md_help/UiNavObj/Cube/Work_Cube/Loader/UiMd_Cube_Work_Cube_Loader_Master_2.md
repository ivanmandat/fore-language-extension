# Связи измерений


На странице «Связи измерений»,
 если необходимо, формируется связь измерений куба-источника с кубом-приёмником:


![](page%204.png)


Если в кубе-источнике и кубе-приёмнике имеются одинаковые измерения,
 то между ними автоматически устанавливается связь по индексу.


При нажатии на кнопку «Отвязать»
 разрывается связь измерения приёмника с измерением источника.


Для редактирования связи необходимо выбрать справочник
 приёмника и нажать на кнопку «Связать».
 Появится окно «Связь по измерению»:


![](page%204.1.png)


В данном окне определите следующие параметры:


	- Источник. В раскрывающемся
	 списке данного поля представлены все измерения источника, необходимо
	 выбрать с каким измерением будет связано измерение приёмника, для
	 которого редактируется связь;


	- Метод связи. Из раскрывающегося
	 списка необходимо выбрать один из методов связи:


	-


		- По
		 идентификатору;


		- По
		 индексу. При выборе данного метода связи необходимо указать:


		-


			- Индекс измерения-источника.
			 В данном поле отображаются все индексы измерения источника.
			 При наличии у измерения нескольких индексов в поле необходимо
			 выбрать один из индексов, по которому будет осуществляться
			 связь;


			- Индекс измерения-приёмника.
			 В данном поле отображаются все индексы измерения приёмника.
			 При наличии у измерения нескольких индексов в поле необходимо
			 выбрать один из индексов, по которому будет осуществляться
			 связь.
