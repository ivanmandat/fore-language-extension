# Загрузчик данных


Загрузчик данных позволяет заполнить куб данными из другого куба в
 настольном приложении. Загрузчик данных можно использовать между кубами
 разных типов и с разным количеством справочников. Если куб-приёмник и
 куб-источник содержат разное количество справочников, то загрузка данных
 осуществляется, если возможно отображение измерений куба-приёмника на
 измерения куба-источника с сопоставлением по кодам или по индексам элементов.


## Создание загрузчика данных


Для создания загрузчика выполните в навигаторе объектов:


	- команду контекстного меню «Создать >
	 Другие > Куб > Загрузчик данных»;


	- команду «Новый объект >
	 Другие > Куб > Загрузчик данных» в группе «Создать» на вкладке ленты «[Главная](GetStarted.chm::/Interface/Interface_Description.htm#customize_ribbon)».


После выполнения одного из действий
 будет открыт мастер загрузчика данных. Мастер содержит страницы:


	- [Базовые
	 свойства](base_settings.htm). Задаются наименование объекта,
	 идентификатор и примечание;


	- [Параметры](parameters.htm).
	 Формируется список параметров;


	- [Источник
	 и приёмник](UiMd_Cube_Work_Cube_Loader_Master_1.htm). Выбираются куб, из которого будут получены
	 данные, и куб, в который данные будут сохранены;


	- [Связи
	 измерений](UiMd_Cube_Work_Cube_Loader_Master_2.htm). Осуществляется настройка привязки измерений
	 куба-приёмника и куба-источника;


	- [Фиксация
	 измерений](UiMd_Cube_Work_Cube_Loader_Master_3.htm). Осуществляется фиксация измерений куба-источника,
	 которые не были сопоставлены с измерениями куба-приёмника;


	- [Управление
	 параметрами](UiMd_Cube_Work_Cube_Loader_Master_Parameters_Managemetn.htm). Настраивается передача значений параметров для открытия
	 объектов, которые используются в загрузчике.


## Загрузка данных


Для загрузки данных необходимо открыть загрузчик в куб. При открытии
 будет выдан диалог для установки отметки элементов, по которым будет производиться
 загрузка данных.


![](pic%20elements.png)


В данном окне устанавливается отметка элементов измерений куба-приёмника,
 по которым будет осуществляться загрузка данных. Для этого в поле «Измерения куба» необходимо выбрать
 соответствующее измерение и в поле «Элементы
 измерения» отметить необходимые элементы. Отметка элементов измерения
 осуществляется с помощью мыши или контекстного меню.


Примечание.
 Если в кубе-приёмнике имеются управляющие измерения, то управляемые измерения
 будут перестраиваться автоматически в соответствии с отметкой, установленной
 для управляющих измерений.


Флаг «Проверять наличие дублирующих
 записей при загрузке» по умолчанию установлен, то есть при загрузке
 данных осуществляется обязательный запрос на наличие дублирующих записей.
 При снятии флага проверка наличия дублирующих записей будет отключена.


Кнопка «Изменить значения параметров»
 позволяет переоткрыть приёмник с новыми значениями параметров. После нажатия
 на кнопку будет выведен диалог «[Установка значений параметров](uinav.chm::/02_Navigator/General_Principles_of_Work.htm#open_param_object)».


При нажатии на кнопку «Закрыть»
 окно будет закрыто, загрузка данных не будет осуществлена.


При нажатии на кнопку «Загрузить данные»
 будет произведена загрузка данных из куба-источника в куб-приёмник.


Если при загрузке данных произошла ошибка, то будет выдано соответствующее
 сообщение.
