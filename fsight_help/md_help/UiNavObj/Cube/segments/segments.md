# Сегменты куба


Сегмент куба - совокупность среза данных [куба](UiNavObj.chm::/Cube/CreateCube/Master_Standart/UiMd_Cube_CreateCube_Master_Standart.htm)
 и прав субъектов безопасности на эти данные.


Примечание.
 Настройка сегментов доступна только для стандартных кубов пользователям/группе
 пользователей, обладающих правами создания таблиц в базе данных.


Для управления сегментами выполните команду контекстного меню «Сегменты» выбранного куба в [навигаторе
 объектов](getstarted.chm::/Interface/Interface_Navigator.htm). Будет
 открыт мастер, содержащий следующие страницы:


	- [База данных](seg_bd.htm);


	- [Сегменты](seg_list.htm).


При создании сегмента куба происходит ограничение имеющихся прав на
 куб для пользователей/групп пользователей. Права доступа к сегментам куба
 определяются [дискреционным
 методом](admin.chm::/04_SecurityPolicy/Admin_PermSep_D.htm) разграничения доступа. Дополнительно с дискреционным
 методом возможно использовать [атрибутный метод](Admin.chm::/04_SecurityPolicy/Admin_ABAC.htm).


Подробное описание выбора методов разграничения доступа и их настройки
 приведено в статье «[Выбор
 методов разграничения доступа и их настройка](Admin.chm::/04_SecurityPolicy/Admin_SecPolicy.htm)».
