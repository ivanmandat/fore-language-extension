# Контейнер


Каждый создаваемый в репозитории объект имеет базовый набор свойств,
 описывающих назначение этого объекта: наименование, идентификатор и примечание.
 Наименования и примечания могут содержать любой текст, а также могут совпадать
 у двух и более объектов. Идентификатор объекта должен быть уникальным,
 это необходимо для поддержания целостности дерева всех объектов репозитория.


Объект «Контейнер» предоставляет
 в рамках репозитория отдельное пространство, где ведётся собственный учёт
 идентификаторов. Идентификаторы объектов внутри контейнера должны быть
 уникальными, но они могут совпадать с идентификаторами объектов внутри
 других контейнеров или с идентификаторами объектов, не входящих в контейнеры.
 Внутри контейнера могут быть созданы любые объекты репозитория, а также
 другие контейнеры.


Для создания контейнера выполните команду:


	- «Создать >
	 Другие > Контейнер» в контекстном меню;


	- «Новый объект > Другие >
	 Контейнер» в группе «Создать»
	 на вкладке «[Главная](GetStarted.chm::/Interface/Interface_Description.htm#customize_ribbon)» ленты
	 инструментов.


Примечание.
 Создание контейнера доступно только в настольном приложении.


После создания контейнера задайте его имя.
