# Выбор репозитория НСИ


Для задания репозитория НСИ и справочников мер и единиц измерений используйте
 страницу «Репозиторий НСИ»:


![](MDM_Repository.png)


Примечание.
 Репозиторий НСИ является устаревшим и используется для совместимости с
 прошлыми версиями.


Задайте параметры хранения данных:


[![](../Opened.gif)![](../Closed.gif)Репозиторий
 НСИ](javascript:TextPopup(this))


	В раскрывающемся списке выберите репозиторий НСИ, в базе данных
	 которого будут храниться данные временных рядов. Репозиторий НСИ является
	 устаревшим и используется для совместимости с прошлыми версиями. Вместо
	 репозитория НСИ выберите базу данных.


	Примечание.
	 Выбор используемого репозитория НСИ или базы данных доступен только
	 при создании базы данных временных рядов.


[![](../Opened.gif)![](../Closed.gif)Хранить
 данные в отдельной БД](javascript:TextPopup(this))


	Установите флажок, если требуется хранить данные временных рядов
	 в базе данных, отличной от базы данных репозитория НСИ, и в раскрывающемся
	 списке укажите необходимую базу данных.


	Примечание.
	 Выбор используемой БД доступен только при создании базы данных временных
	 рядов.


Совет. Указанный
 репозиторий НСИ и отдельная база данных, используемая для хранения временных
 рядов, могут быть расположены на различных серверах СУБД. Важно, чтобы
 используемые серверы СУБД были одного типа, например, только Oracle или
 только Microsoft SQL Server.

Использование различных серверов СУБД позволит ускорить выполнение проверки
 целостности БД и ускорить создание резервных копий.


Задайте параметры мер и единиц измерения:


[![](../Opened.gif)![](../Closed.gif)Справочник
 мер/единиц измерений](javascript:TextPopup(this))


	В раскрывающихся списках выберите необходимые [справочники
	 мер и единиц измерений](../Units/Units_dbts.htm#options).


	Если требуемый справочник отсутствует, то для его создания нажмите
	 кнопку «Создать».


	Пустой справочник требуемой структуры будет создан в указанном репозитории
	 НСИ.


[![](../Opened.gif)![](../Closed.gif)Является
 обязательным атрибутом ряда](javascript:TextPopup(this))


	Установите флажок, если требуется использовать единицы измерения
	 для однозначной идентификации временного ряда. Справочник единиц измерений
	 будет включён в состав уникального ключа БД временных рядов. При работе
	 с временными рядами в инструменте «Аналитические
	 запросы (OLAP)» или контейнере моделирования будет доступно
	 измерение «Единицы измерения».
