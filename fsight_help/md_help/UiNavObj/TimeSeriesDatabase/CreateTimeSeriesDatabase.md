# База данных временных рядов


Для создания базы данных временных рядов:


	- в веб-приложении нажмите кнопку ![](../Cube/CreateCube/Master_Standart/add_button.png) «Создать» в главном меню и выберите
	 тип объекта на боковой панели «[Новый объект](UiNav.chm::/02_Navigator/General_Principles_of_Work.htm#add_object)»;


	- в настольном приложении:


		- выполните команду «Новый
		 объект > БД временных рядов» в группе «Создать»
		 на вкладке «Главная» ленты
		 инструментов;


		- выполните команду «Создать
		 > БД
		 временных рядов» в контекстном меню навигатора объектов.


Будет открыт мастер базы данных временных рядов.


Примечание.
 По умолчанию создание базы данных временных рядов в веб-приложении недоступно,
 данный тип объектов не отображается в [навигаторе](GetStarted.chm::/Interface/Interface_Navigator.htm).
 Для доступа к объекту задайте поле [hideClasses](setup.chm::/UiWebSetup/03_Setup_Web/PP_config_Java.htm#metabase_config) в
 [config.json](setup.chm::/UiWebSetup/03_Setup_Web/PP_config_Java.htm#config_json).


Создание БД временных рядов с помощью
 мастера состоит из этапов:


	- [Задание базовых свойств](base_settings.htm).


	- [Выбор репозитория НСИ](ChooseNSI.htm).


	- [Настройка атрибутов](TS_Attributes.htm).


	- [Создание индексов](TS_Indexes.htm).


	- [Настройка
	 календаря](TS_Calendar.htm).


Перед запуском мастера убедитесь, что в текущем репозитории существует
 репозиторий НСИ, необходимый для создания базы данных временных рядов.
