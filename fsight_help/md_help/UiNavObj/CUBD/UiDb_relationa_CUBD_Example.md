# Пример создания команды СУБД


Рассмотрим пример создания команды СУБД, которая удаляет все поля таблицы,
 в настольном приложении.


Для создания объекта «Команда СУБД»
 в окне навигатора объектов выполните пункт контекстного меню «Создать >
 Другие > Команда СУБД».


Задайте стандартные параметры объекта:


	- Наименование: Команда СУБД-удаление
	 данных таблицы;


	- Идентификатор: DBMS_DELETE;


Перейдите на страницу «Команда СУБД»
 и укажите следующие параметры:


	- База данных: Данные;


	- Драйвер: Oracle;


	- Оператор: delete from TABLETEST,
	 где TABLETEST- идентификатор таблицы, которая будет очищена.


![](UiDb%20CUBD%20Example%20P3.png)


Для завершения работы мастера перейдите на следующую страницу и нажмите
 на кнопку «Готово». После завершения
 работы мастера в навигаторе объектов появится объект «Команда
 СУБД».


Для запуска команды СУБД выполните пункт контекстного меню «Открыть».
 По завершению выполнения команды СУБД будет выдано информационное сообщение,
 содержащее результат выполнения.
