# Свойства обычного атрибута


При [создании](../Attributes.htm#add)/[редактировании](../Attributes.htm#edit)
 обычного или системного атрибута задайте его свойства с помощью окна «Свойства атрибута».


[![](../../../Opened.gif)![](../../../Closed.gif)Для открытия
 окна](javascript:TextPopup(this))


	Для открытия окна при создании обычного атрибута:


		- в веб-приложении выполните команду «Добавить >
		 Обычный атрибут». Будет открыто окно «Свойства
		 атрибута»;


		- в настольном приложении:


			- Выполните одно из действий:


				- нажмите кнопку «Добавить»;


				- вызовите контекстное меню на свободной области вкладки
				 «Атрибуты»;


				- дважды щёлкните по свободной области вкладки «Атрибуты».


			- Выберите тип атрибута «Обычный
			 атрибут». Будет открыто окно «Свойства
			 атрибута».


	Для открытия окна при редактировании обычного атрибута выполните
	 одно из действий:


		- в веб-приложении нажмите кнопку ![](../../../database/edit.png) «Редактировать»;


		- в настольном приложении:


			- нажмите кнопку «Редактировать»;


			- выполните команду «Редактировать»
			 в контекстном меню атрибута;


			- дважды щёлкните по наименованию атрибута.


	 Веб-приложение Настольное приложение


		![](attribute_properties_web.png)


		![](../../Master_Composite_Table_reference_book/attribute_properties.png)


Примечание.
 Для системных атрибутов часть настроек недоступна для редактирования.


Задайте параметры:


	- Наименование. Введите
	 наименование атрибута;


	- Идентификатор. Введите
	 уникальный идентификатор атрибута. Допускается использование только
	 латинских букв, цифр, знаков «_», не допускается использование пробелов;


	- Тип данных. Выберите
	 в раскрывающемся списке тип данных атрибута:


		- ДатаВремя;


		- Строковый;


		- Целый;


		- Вещественный;


		- Дата;


		- Логический;


		- Длинный текст;


Примечание.
 Атрибуты типа «Длинный текст»
 допускают использование текстовой строки длиной свыше 4000 символов.


	- Общая длина. Определите
	 максимальную длину значения атрибута. Доступно только для строкового
	 и вещественного типов данных;


	- Десятичных знаков. Определите
	 количество знаков после запятой. Доступно только для вещественного
	 типа данных. Заданное количество знаков после запятой должно быть
	 меньше максимальной длины значения атрибута, заданной в параметре
	 «Общая длина»;


	- Доступен
	 для редактирования в карточке элемента. В раскрывающемся списке
	 выберите, в каком случае значение атрибута доступно для редактирования
	 в [карточке элемента](../Element_Card.htm):


	-


		- Всегда. Значение
		 атрибута всегда доступно для редактирования в карточке элемента;


		- Если не скрытый.
		 Используется по умолчанию. Значение атрибута доступно для редактирования
		 в карточке элемента, если атрибут не является скрытым;


		- Никогда. Значение
		 атрибута не доступно для редактирования в карточке элемента. Независимо
		 от состояния флажка «Скрытый
		 атрибут», флажок «Может
		 иметь пустое значение» установлен и недоступен для изменения;


	- Может иметь пустое значение. По
	 умолчанию флажок установлен. Если флажок снят, то каждый элемент справочника
	 должен иметь значение данного атрибута. В противном случае, атрибут
	 не является обязательным для заполнения. Опция доступна, если атрибут
	 не является скрытым;


	- Может
	 иметь множественные значения. По умолчанию флажок снят. При
	 установке флажка для элемента справочника можно задать несколько значений
	 данного атрибута. Набор значений по такому атрибуту при просмотре
	 отображается через точку с запятой. Для выбора нескольких значений
	 атрибута можно настроить [связь](../Link.htm) по данному
	 атрибуту с другим справочником, содержащим значения. При работе с
	 [инструментами
	 анализа данных и построения отчётов](DataAnalysis.chm::/DataAnalysis_Title.htm) сохранить данные
	 в справочники со множественными значениями атрибута можно, если в
	 качестве [источника
	 данных](../../../Cube/CreateCube/Master_Standart/UiMd_Cube_CreateCube_Master_Standart_2.htm) и [измерения
	 куба](../../../Cube/CreateCube/Master_Standart/UiMd_Cube_CreateCube_Master_Standart_3.htm), на котором строятся различные отчёты, указать данный справочник;


Примечание.
 Если установлен флажок «Может иметь множественные
 значения», то флажок «Альтернативная
 иерархия» должен быть снят, так как совместная работа данных флажков
 не поддерживается.


	- Скрытый атрибут. При
	 установке флажка атрибут будет участвовать в построении справочника,
	 но при этом в справочнике отображаться не будет. Флажок доступен,
	 если атрибут может иметь пустое значение. Скрытые атрибуты на вкладке
	 «[Атрибуты](../Attributes.htm)»
	 выделены серым цветом. Скрытые системные атрибуты дополнительно не
	 выделяются;


	- Переводить значение на другие
	 языки. По умолчанию флажок установлен. Установка флажка позволяет
	 создавать [мультиязычный
	 справочник НСИ/составной справочник НСИ](UiNav.chm::/Multilanguage/UiRds_Locale.htm). Он хранит
	 наименования всех атрибутов и значения строковых атрибутов не только
	 для текущего языка, но и для языков, на которых предполагается использовать
	 справочник;


Примечание.
 Флажок доступен только в настольном приложении только для атрибутов типа
 «Строка» и «Длинный
 текст», если на вкладке «[Описание](../Master_RDS.htm#structure)»
 установлен флажок «[Переводить
 справочник на другие языки](../base_settings.htm#translate_dictionary)».


	- Пиктограмма элемента.
	 При установке флажка значение атрибута будет определять индекс значка
	 для элемента справочника. Флажок доступен, если для репозитория установлен
	 объект, содержащий [пиктограммы
	 элементов измерений](UiNav.chm::/02_Navigator/Repo_Default.htm). Если данный атрибут задан,
	 то около наименования элементов будет выводиться изображение, индекс
	 которого соответствует значению данного атрибута. Данный флажок доступен
	 только для целочисленных атрибутов;


	- Альтернативная иерархия.
	 По умолчанию флажок не установлен. При установленном флажке атрибут
	 будет использоваться для настройки [альтернативной
	 иерархии](../../UiMd_reference_book_Hierarchy.htm#alternativehierarchy) справочника НСИ/составного справочника НСИ;


Примечание.
 Если установлен флажок «Альтернативная
 иерархия», то флажок «Может иметь
 множественные значения» должен быть снят, так как совместная работа
 данных флажков не поддерживается.


	- Вычислять
	 значение по умолчанию. По умолчанию флажок снят. В качестве
	 значения по умолчанию задайте фиксированное значение или Fore-выражение,
	 используя [редактор
	 выражений](UiNav.chm::/GUI/ExpressionEditor.htm).
	 Установите, когда будет вычисляться значение: единожды при создании
	 элемента или каждый раз при изменении элемента.


Примечание.
 Вычисляемое значение недоступно для изменения при [редактировании
 элемента справочника](../../Work/Work_Elements.htm#edit).


Для всех системных атрибутов в окне «Свойства
 атрибута» возможно редактирование наименования и назначения атрибута.
 Также для всех системных атрибутов кроме атрибута «Наименование
 (NAME)» можно изменить настройку отображения атрибута в справочнике
 «Скрытый атрибут». Для атрибута
 «Наименование (NAME)» доступна
 настройка «Вычислять значение по умолчанию».
 В поле «Выражение» установите
 формулу, с помощью которой будет формироваться наименование элемента.
