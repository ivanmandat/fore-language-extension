# Примеры выражений


## Условный оператор


Для использования условного оператора в окне «[Редактора
 выражения](UiNav.chm::/GUI/ExpressionEditor.htm)» следует ввести строку следующей структуры:


Iif(Condition: Boolean; V1: Variant; V2: Variant):
 Variant


Например, при задании выражения для уровня «Кварталы»:


Iif(Наименование="I квартал 1990", "Дата
 начала периода", "Наименование")


У элемента с наименованием «I квартал
 1990» будет выведено наименование «Дата
 начала периода», у остальных
 элементов данного уровня будет выведено стандартное наименование.


Примечание.
 При составлении условного оператора следует учитывать, что все поля календарного
 справочника (NAME, LEVEL, ORDER и т.д.) строкового типа.


## Модуль


Для использования функции из модуля в редакторе выражения следует указать:


<идентификатор модуля>.<имя
 функции>


Например, при задании для уровня «Годы»
 выражения как:


Module.MyFunc(NAME)


Будет использоваться функция «MyFunc»
 из модуля с идентификатором «Module»,
 функции будет передаваться значение параметра NAME.


Пример функции:


Function
 MyFunc(Str: Variant): Variant;


Begin


Return
 "Год " + (Str As String);


End Function
 MyFunc;


Для всех наименований элементов уровня «Годы»
 перед стандартными наименованиями будет добавлено «Год».
