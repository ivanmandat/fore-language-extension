# Работа с элементами справочника НСИ через буфер обмена


	При работе с элементами справочника НСИ через буфер обмена доступны
	 операции:


		- [копирование](#copy_element);


		- [перемещение](#cut_element);


		- [вставка](#put_element).


	При работе с буфером обмена в параметрическом справочнике НСИ ознакомьтесь
	 с [особенностями](#mdm_features).


	Примечание.
	 Работа с элементами через буфер обмена доступна только при [редактировании
	 справочника НСИ](Work_Dictionary.htm#edit) в настольном приложении.


## Копирование элемента в буфер обмена


	Для копирования выделенного элемента в буфер обмена:


		- нажмите кнопку ![](B_copy.gif) «Копировать»
		 на панели инструментов. Доступно только в настольном приложении;


		- выполните команду «Правка
		 > Копировать» в главном меню. Доступно только в веб-приложении;


		- выполните команду «Копировать»
		 в контекстном меню элемента;


		- нажмите сочетание клавиш CTRL+C или CTRL+INSERT.


	Копирование элементов осуществляется без дочерних элементов.


## Перемещение элемента в буфер обмена


	Для перемещения выделенного элемента в буфер обмена:


		- нажмите кнопку ![](B_cut.gif) «Вырезать»
		 на панели инструментов. Доступно только в настольном приложении;


		- выполните команду «Правка
		 > Вырезать» в главном меню. Доступно только в веб-приложении;


		- выполните команду «Вырезать»
		 в контекстном меню элемента;


		- нажмите сочетание клавиш CTRL+X или SHIFT+DELETE.


	Вырезание элементов осуществляется вместе с дочерними элементами.


## Вставка элемента из буфера обмена


	Для вставки элемента из буфера обмена в текущее место справочника:


		- нажмите кнопку ![](b_insert.gif) «Вставить»
		 на панели инструментов. Доступно в настольном приложении;


		- выполните команду «Правка
		 > Вставить» в главном меню. Доступно только в веб-приложении;


		- выполните команду «Вставить» в
		 контекстном меню элемента;


		- нажмите сочетание клавиш CTRL+V или SHIFT+INSERT.


	При вставке копии элемента будет открыто окно «[Свойства
	 элемента справочника](Work_Elements.htm)», в котором отображаются значения атрибутов
	 скопированного элемента. К наименованию исходного элемента будет добавлен
	 суффикс « - Копия (<номер копии>)».
	 Например, наименование копируемого элемента «Test», наименование скопированного
	 элемента - «Test - Копия (1)». В веб-приложении наименование не меняется.


### Особенности вставки элементов


		- Поддерживается вставка как текстовых, так и табличных данных,
		 при этом пустые строки или ячейки игнорируются.


		- Вставка осуществляется для атрибута «Наименование».
		 Если данные являются таблицей, то в качестве наименования будут
		 выбраны значения первого столбца. Если данные являются текстом,
		 то в качестве наименования будет выбран текст, ограниченный символом
		 табуляции или точкой с запятой для файлов CSV.


		- При вставке в параметрический справочник значение параметра
		 будет установлено автоматически, в соответствии с указанным в
		 данный момент в справочнике.


		- Если в справочнике есть атрибуты, значение которых вычисляется
		 по умолчанию при создании элемента, то при вставке в такой справочник
		 значения атрибутов будут вычислены автоматически.


		- При наличии в справочнике пользовательских атрибутов, обязательных
		 для заполнения, вставка не осуществляется и выводится сообщение
		 о наличии таких атрибутов со списком их идентификаторов.


## Особенности операций с элементами в
	 параметрическом справочнике НСИ


	В параметрическом справочнике возможно копирование и вставка элементов
	 справочника при изменении набора и значений параметров. Копирование
	 и вставка элементов осуществляется без дочерних элементов.


	Пример использования операций «Копирование»,
	 «Вставка»:


		- Устанавливаем значение параметра «1».


		- Копируем элементы справочника.


		- Устанавливаем значение параметра «2».


		- Вставляем элементы. Будут добавлены скопированные значения,
		 значение атрибута элемента параметра для скопированных элементов
		 изменится на «2». Элементы скопируются без дочерних элементов.
