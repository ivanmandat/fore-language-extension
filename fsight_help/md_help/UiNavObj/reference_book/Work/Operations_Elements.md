# Выполнение операций с элементами


При работе с элементами справочника доступны операции:


	- [выделение элементов](#select);


	- [перемещение элементов справочника НСИ](#replace);


	- [обновление элементов](#refresh). Доступно только
	 в настольном приложении;


	- [управление отображением скрытых элементов
	 справочника НСИ](#hidden_elements). Доступно только в настольном приложении;


	- [перенос элементов справочника НСИ между репозиториями](#update).


## Выделение элементов


Для выделения одного элемента справочника щёлкните по его наименованию.


Для выборочной отметки зажмите клавишу CTRL, после чего последовательно
 щёлкните по элементам, которые необходимо выделить.


Для выделения диапазона элементов зажмите клавишу SHIFT, после чего
 щёлкните первый и последний элемент диапазона. Элементы будут выделены
 со всеми подчинёнными элементами.


Примечание.
 Выделение диапазона элементов доступно только в настольном приложении.


Также возможна установка отметки элементов справочника, открытого на
 [просмотр](Work_Dictionary.htm#view):


[![](../../Opened.gif)![](../../Closed.gif)Отметить все](javascript:TextPopup(this))


	Для установки отметки у всех элементов справочника выполните команду
	 контекстного меню «Отметить все».


	Для снятия отметки со всех элементов справочника выполните команду
	 контекстного меню «Снять отметку
	 со всех».


[![](../../Opened.gif)![](../../Closed.gif)Отметить подчиненные](javascript:TextPopup(this))


	Для установки отметки для всех элементов, расположенных на одном
	 уровне с указанным элементом справочника, выполните команду контекстного
	 меню «Отметить подчиненные по уровню».


	Для установки отметки для всех подчинённых элементов справочника
	 выполните команду контекстного меню «Отметить
	 все подчиненные».


	Для снятия отметки со всех подчиненных элементов справочника выполните
	 команду контекстного меню «Снять
	 отметку с подчиненных».


[![](../../Opened.gif)![](../../Closed.gif)Отметить уровень](javascript:TextPopup(this))


	Для установки отметки для уровня справочника выполните команду контекстного
	 меню «Отметить уровень» и
	 выберите необходимый уровень справочника.


	Для снятия отметки с уровня справочника выполните команду контекстного
	 меню «Снять отметку с уровня».


## Перемещение элементов справочника НСИ


Структура элементов справочника НСИ используется для построения множества
 объектов. При перемещении элемента выполняется автоматическое сохранение
 изменений.


Примечание.
 Перемещение элементов доступно для справочника, открытого на [редактирование](Work_Dictionary.htm#edit).


[![](../../Opened.gif)![](../../Closed.gif)Перемещение
 элемента вверх/вниз](javascript:TextPopup(this))


	Для перемещения выбранного элемента вверх:


		- нажмите кнопку ![](b_up.png) «Переместить вверх» на панели
		 инструментов. После выполнения действия выбранный элемент будет
		 перемещён на одну позицию вверх;


		- выполните команду «Переместить
		 вверх»/«Вверх»
		 в контекстном меню элемента. После выполнения действия выбранный
		 элемент будет перемещён на одну позицию вверх;


		- используйте механизм перетаскивания Drag&Drop:


			- если перетаскивание элемента выполняется с зажатой клавишей
			 SHIFT, то выполняется копирование элементов;


			- если перетаскивание элемента выполняется на одном уровне
			 без зажатых клавиш, элемент будет перемещён на место указанного
			 элемента.


	Для перемещения выбранного элемента вниз:


		- нажмите кнопку ![](b_down.png) «Переместить вниз» на панели
		 инструментов. После выполнения действия выбранный элемент будет
		 перемещён на одну позицию вниз;


		- выполните команду «Переместить
		 вниз»/«Вниз» в
		 контекстном меню элемента. После выполнения действия выбранный
		 элемент будет перемещён на одну позицию вниз;


		- используйте механизм перетаскивания Drag&Drop:


			- если перетаскивание элемента выполняется с зажатой клавишей
			 SHIFT, то выполняется копирование элементов;


			- если перетаскивание элемента выполняется на одном уровне
			 без зажатых клавиш, элемент будет перемещён на место указанного
			 элемента.


[![](../../Opened.gif)![](../../Closed.gif)Перемещение
 элемента между уровнями](javascript:TextPopup(this))


	Для перемещения выбранного элемента на уровень вверх:


		- нажмите кнопку ![](b_level.png) «Переместить на уровень вверх»
		 на панели инструментов;


		- выполните команду «Переместить
		 на уровень вверх»/«На
		 уровень вверх» в контекстном меню элемента.


	После выполнения действия выбранный элемент будет перемещен на
	 уровень вверх.


	Для перемещения выбранного элемента между уровнями используйте механизм
	 перетаскивания Drag&Drop:


		- если перетаскивание элемента выполняется с зажатой клавишей
		 CTRL, перемещаемый элемент становится подчиненным для того элемента,
		 на который его перетащили;


		- если перетаскивание элемента выполняется между уровнями
		 без зажатых клавиш, элемент будет перемещён на тот же уровень,
		 что и указанный элемент.


При перемещении элементов учитывайте следующие особенности:


	- одновременное перемещение нескольких элементов доступно, если
	 все выделенные элементы расположены на одном уровне;


	- перемещение элемента-родителя под дочерний элемент запрещено;


	- при перемещении элемента под новый элемент-родитель вместе с
	 ним перемещаются все его дочерние элементы.


Также перемещать элементы можно, используя буфер обмена.
 Для получения подробной информации обратитесь к статье «[Работа с элементами справочника
 НСИ через буфер обмена](Copy_Elements.htm#cut_element)».


## Обновление элементов


Обновление элементов справочника доступно только в настольном приложении.


Для обновления списка элементов справочника, открытого на [просмотр](Work_Dictionary.htm#view):


	- выполните команду «Вид >
	 Обновить» в главном меню;


	- нажмите клавишу F5.


Если справочник параметрический, то отобразится диалог «[Установка значений параметров](uinav.chm::/02_Navigator/General_Principles_of_Work.htm#open_param_object)».
 В полях ввода значений параметров будут отражены предыдущие значения.


При работе с элементами справочника НСИ, открытого на [редактирование](Work_Dictionary.htm#edit),
 доступны операции обновления:


	- Обновление элементов справочника.
	 Для обновления элементов справочника нажмите кнопку ![](5.gif) «Обновить»
	 на панели инструментов;


	- Автоматическое обновление справочника. По
	 умолчанию автоматическое обновление справочника включено. Для
	 автоматического обновления справочников нажмите кнопку ![](5.1.gif) «Обновлять
	 автоматически» на панели инструментов. При изменении значения
	 параметров содержимое справочника автоматически обновится.


Примечание.
 Автоматическое обновление доступно только для [параметрического
 справочника НСИ](../Master_RDS_reference_book/Parameters.htm).


## Управление отображением скрытых элементов
 справочника НСИ


Управление отображением скрытых элементов справочника НСИ доступно только
 в настольном приложении.


Для управления отображением [скрытых](../Master_RDS_reference_book/base_settings.htm#hidden_attribute)
 элементов справочника НСИ, открытого на [редактирование](Work_Dictionary.htm#edit),
 используйте флажок «Показать скрытые
 элементы» в главном меню «Вид».


## Перенос элементов справочника НСИ между репозиториями


Элементы справочника НСИ, открытого на [редактирование](Work_Dictionary.htm#edit),
 можно добавлять в обновление для переноса между репозиториями.


[![](../../Opened.gif)![](../../Closed.gif)Добавление
 элемента справочника НСИ в обновление](javascript:TextPopup(this))


	Для добавления элемента справочника НСИ в обновление в веб-приложении:


		- Откройте справочник НСИ на редактирование.


		- Выполните команду «Добавить
		 в обновление» в контекстном меню выбранного элемента.


	В результате выполненных действий в состав обновления в менеджере
	 обновлений будет добавлен выбранный элемент. В
	 соответствующем поле в столбце «Перенос
	 данных» в [менеджере
	 обновлений](UpdManager.chm::/admin_upmbobj_runmanager.htm) будет указано, что обновление частичное:


	![](../Master_RDS_reference_book/partial_update_web.png)


	Для добавления элемента справочника в обновление в настольном приложении:


		- Откройте справочник НСИ на редактирование и перейдите на
		 вкладку «Элементы справочника».


		- Вызовите контекстное меню выбранного элемента.


		- Следующий шаг зависит от наличия дочерних элементов:


			- если выбранный элемент не имеет дочерних элементов,
			 выберите пункт «Добавить
			 в обновление»;


			- если выбранный элемент имеет дочерние элементы, выберите
			 пункт «Добавить в обновление»,
			 затем в раскрывшемся списке выберите:


				- Без дочерних элементов.
				 Элемент будет добавлен в обновление без дочерних элементов;


				- С дочерними элементами.
				 Элемент будет добавлен в обновление вместе с дочерними
				 элементами.


		- Если ранее справочник еще не был включен в обновление, будет
		 открыто окно «[Добавление в обновление](UpdManager.chm::/CreateUpdate/Features/Admin_CreateUpdate_Features_Tables.htm)».
		 Задайте в нём параметры обновления. Если ранее справочник уже
		 был включен в обновление, при добавлении новых элементов
		 настраивать параметры повторно не нужно.


	В результате выполненных действий в состав обновления в менеджере
	 обновлений будет добавлен выбранный элемент. В соответствующем поле
	 в столбце «Перенос данных»
	 в [менеджере
	 обновлений](UpdManager.chm::/admin_upmbobj_runmanager.htm) будет указано, что обновление частичное:


	![](../Master_RDS_reference_book/partial_update.png)


[![](../../Opened.gif)![](../../Closed.gif)Удаление элемента
 справочника НСИ из обновления](javascript:TextPopup(this))


	Для удаления элемента справочника НСИ из обновления в веб-приложении:


		- Откройте справочник НСИ на редактирование.


		- Выполните команду «Удалить
		 из обновления» в контекстном меню выбранного элемента.


	В результате выполненных действий выбранный элемент будет исключён
	 из структуры обновления.


	Для удаления элемента справочника из обновления в настольном приложении:


		- Откройте справочник НСИ на редактирование и перейдите на
		 вкладку «Элементы справочника».


		- Вызовите контекстное меню выбранного элемента.


		- Выберите пункт «Убрать
		 из обновления».


	Примечание.
	 Если из обновления удален родительский элемент, его дочерние элементы
	 останутся в обновлении. Для удаления дочерних элементов из обновления
	 выполните команду «Убрать из обновления»
	 в контекстном меню дочерних элементов.


	В результате выполненных действий выбранный элемент будет исключён
	 из состава обновления.
