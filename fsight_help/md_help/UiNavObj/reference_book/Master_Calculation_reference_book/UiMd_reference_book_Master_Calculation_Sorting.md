# Сортировка


На странице «Сортировка»
 настраивается порядок сортировки элементов, который будет применяться
 при открытии справочника.


	Веб-приложение Настольное приложение


		![](sorting_page.png)


		![](../Master_Table_reference_book/Sorted_calculation_directory.png)


В списке «Атрибуты справочника»
 отображены все имеющиеся [атрибуты
 справочника](UiMd_reference_book_Master_Calculation_Atribute.htm). В данном списке не отображаются атрибуты, которые добавлены
 в список «Выбранные атрибуты».


В списке «Выбранные атрибуты»
 отображаются атрибуты, по которым будет производиться сортировка.


По умолчанию всегда проводится возрастающая сортировка по атрибуту,
 для которого установлено [назначение](../Master_Table_reference_book/UiMd_reference_book_Master_Table_page1_AttProps.htm)
 «Порядок».


Для добавления атрибута из списка «Атрибуты
 справочника» в список «Выбранные
 атрибуты» выполните одно из следующих действий:


	- используйте кнопку ![](../Master_Table_reference_book/Add_web.png) «Добавить атрибут» в веб-приложении
	 или кнопку ![](../Master_Table_reference_book/b_right.png) «Добавить
	 атрибут» в настольном приложении;


	- перетащите атрибут справочника в список «Выбранные
	 атрибуты» с помощью механизма Drag&Drop.


Для возврата атрибута обратно в список «Атрибуты
 справочника» используйте кнопку ![](../Master_Table_reference_book/Return_web.png) «Вернуть атрибут» в веб-приложении
 или кнопку ![](../Master_Table_reference_book/b_left.png) «Вернуть
 атрибут» в настольном приложении.


Для настройки сортировки следует выбрать атрибут в списке «Выбранные
 атрибуты» и определить для него способ сортировки:


	- По убыванию. Используйте
	 кнопки ![](../Master_Table_reference_book/Descend_web.png) «По
	 убыванию» в веб-приложении или ![](../Master_Table_reference_book/b_descend.png) «По убыванию» в настольном приложении;


	- По возрастанию. Используйте
	 кнопки ![](../Master_Table_reference_book/Ascend_web.png) «По
	 возрастанию» в веб-приложении или ![](../Master_Table_reference_book/b_ascend.png) «По возрастанию» в настольном приложении.


При настройке сортировки важен порядок расположения атрибутов, так как
 именно в этом порядке будет проводиться сортировка. Для изменения порядка
 сортировки атрибутов используются кнопки ![](../Master_Table_reference_book/LevelUp_web.png) «Переместить на один уровень вверх»
 и ![](../Master_Table_reference_book/LevelDown_web.png) «Переместить
 на один уровень вниз» в веб-приложении или кнопки ![](../Master_Table_reference_book/b_up.png) «Переместить
 на один уровень вверх» и ![](../Master_Table_reference_book/b_down.png) «Переместить на один уровень вниз»
 в настольном приложении.


Примечание.
 При работе со справочником можно будет изменить [сортировку
 элементов](../Work/Sort_Elements.htm).
