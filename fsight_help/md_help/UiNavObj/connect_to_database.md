# Подключение к внешним базам данных


Для получения доступа к данным, которые хранятся в каких-либо внешних
 источниках, в продукте «Форсайт. Аналитическая платформа»
 предусмотрены следующие возможности:


	- [Подключение
	 к реляционным базам данных](database/UiDb_database.htm). Данное подключение использует объект
	 репозитория «База данных», который содержит фактические настройки
	 базы данных, в которой хранятся данные.


	Веб-приложение Настольное приложение


		![](database/db_properties_web.png)


		![](database/db_properties.png)


	- [Подключение
	 к другому репозиторию](Link_with/UiDb_relational_LinkWith.htm). Данное подключение позволяет из текущего
	 репозитория получить доступ к объектам из другого репозитория. Настройки
	 другого репозитория сохраняются в объекте «Связь с репозиторием».
	 В текущем репозитории создаются ярлыки, которые через «Связь с репозиторием»
	 предоставят доступ к объектам из другого репозитория и их данным.


		Веб-приложение Настольное приложение


			![](Link_with/linkwith_db_web.png)


			![](Link_with/linkwith_db.png)


	- [Подключение
	 к Microsoft Analysis Services](ADOMD/Connect_to_Microsoft_Analysis_Services.htm). Данное подключение использует каталог
	 ADOMD для доступа к многомерным базам Microsoft Analysis Services:


![](ADOMD/ADOMD%20master%20Page%203.png)


	- [Подключение
	 к SAP](ADOMD/UiDb_relational_ADOMD.htm). Данное подключение использует каталог ADOMD для доступа
	 к многомерным базам, которые построены на базе SAP NetWeaver BW или
	 SAP HANA:


![](ADOMD_parametrs.png)
