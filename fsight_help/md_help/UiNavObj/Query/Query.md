# Запрос


Запрос - это выборка из
 таблиц на основе заданного SQL-запроса. Запрос, так же как [таблица](../Table/UiDb_relational_table.htm)
 и [представление](../conception/UiDb_relational_conception.htm),
 может являться источником данных для измерений и кубов.


## Создание запроса


Для создания запроса в [навигаторе
 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm):


	- в веб-приложении нажмите кнопку ![](../Cube/CreateCube/Master_Standart/add_button.png) «Создать» в главном меню и выберите
	 тип объекта на боковой панели «[Новый объект](UiNav.chm::/02_Navigator/General_Principles_of_Work.htm#add_object)»;


	- в настольном приложении:


		- Выполните одно из действий:


			- выполните команду «Создать >
			 Таблица» в контекстном меню;


			- нажмите кнопку «Новый
			 объект», расположенную в группе «Создать»
			 на вкладке «[Главная](GetStarted.chm::/Interface/Interface_Description.htm#customize_ribbon)»
			 ленты инструментов, и выберите пункт «Таблица».


		- Выберите тип создаваемого
		 объекта на странице «Тип объекта».


После выполнения
 действий в мастере запроса задайте настройки на страницах:


	- [Базовые
	 настройки/Базовые свойства](Master/base_settings.htm). Задаются наименование
	 объекта, идентификатор и примечание;


	- [Параметры](Master/parameters.htm).
	 Формируется список параметров;


	- [Свойства](Master/UiDb_relational_query_master_Property.htm).
	 Указывается база данных, в которой будет создан запрос, а также текст
	 запроса и дополнительные настройки.


Для сохранения настроек в мастере запроса:


	- в веб-приложении:


		- нажмите кнопку «Сохранить»;


		- выполните команду «Сохранить
		 и закрыть» в раскрывающемся меню кнопки «Сохранить».
		 После этого будут сохранены заданные настройки, закрыт мастер
		 и выполнен переход в [режим
		 редактирования](../Table/Work/Changing_Database_Query.htm) данных;


Примечание.
 Команда «Сохранить и закрыть»
 доступна только при [редактировании настроек](Query.htm#edit).


	- в настольном приложении нажмите кнопку «Готово».


## Работа с запросом


[![](../Opened.gif)![](../Closed.gif)Редактирование
 настроек](javascript:TextPopup(this))


	Для редактирования выбранного в навигаторе объектов запроса:


		- в веб-приложении:


			- выполните команду «Редактировать»
			 в контекстном меню запроса;


			- выполните команду «Редактировать
			 в новой вкладке» в контекстном меню запроса;


			- выполните команду «Редактировать
			 в новом окне» в контекстном меню запроса;


			- нажмите клавишу F4.


	Затем нажмите кнопку ![](../b_settings_web.png) «Настройки»;


		- в настольном приложении:


			- нажмите кнопку ![](../reference_book/Work/b_edit.png)
			 «Редактировать» на
			 вкладке «Главная»
			 ленты инструментов;


			- выполните команду «Редактировать»
			 в контекстном меню запроса;


			- нажмите клавишу F4.


	После чего будет открыт мастер запроса. Измените настройки на соответствующих
	 страницах мастера.


	Для сохранения изменений настроек в [мастере
	 запроса](Query.htm#master):


		- в веб-приложении:


			- нажмите кнопку «Сохранить»;


			- нажмите кнопку «Сохранить
			 и закрыть». После нажатия на кнопку будут сохранены
			 изменения и выполнен переход в [режим
			 редактирования](../Table/Work/Changing_Database_Query.htm) данных;


		- в настольном приложении нажмите кнопку «Готово».


[![](../Opened.gif)![](../Closed.gif)Просмотр
 данных](javascript:TextPopup(this))


Для просмотра данных выделите соответствующий объект в [навигаторе
 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm) и выполните одно из действий:


	- в веб-приложении:


		- выполните команду «Открыть»
		 в контекстном меню объекта;


		- выполните команду «Открыть
		 в новой вкладке» в контекстном меню объекта;


		- выполните команду «Открыть
		 в новом окне» в контекстном меню объекта;


		- дважды щёлкните по объекту;


		- нажмите клавишу ENTER;


	- в настольном приложении:


		- нажмите кнопку ![](../table/work/b_open.png) «Открыть»
		 на вкладке «Главная»
		 ленты инструментов;


		- нажмите кнопку ![](../table/work/b_open.png) «Открыть
		 > В инструменте по умолчанию» на вкладке «Главная» ленты инструментов;


		- выполните команду «Открыть»
		 в контекстном меню объекта.


		- дважды щёлкните по объекту.


После выполнения одного из действий будет открыт соответствующий объект
 с данными. Для получения подробной информации о принципах работы с набором
 данных обратитесь к разделу «[Работа
 с набором данных](../Table/Work/UiDb_relational_table_work.htm)».


Для просмотра данных в отчете в настольном приложении:


	- нажмите кнопку «Открыть
	 > Конструктор регламентных отчётов» на вкладке «Главная» ленты инструментов;


	- выполните команду «Открыть
	 с помощью > Конструктор регламентных отчётов» в контекстном
	 меню объекта.


Для [импорта данных](../Data_import_wizard/Data_import_wizard.htm)
 в настольном приложении:


	- нажмите кнопку «Открыть >
	 Мастер импорта данных» на вкладке «Главная»
	 ленты инструментов;


	- выполните команду «Открыть
	 с помощью > Мастер импорта данных» в контекстном меню объекта.


[![](../Opened.gif)![](../Closed.gif)Обновление](javascript:TextPopup(this))


	Для обновления и корректного отображения информации о полях запроса
	 выполните команду ![](../Table/update.png) «Обновить
	 информацию о запросе» в контекстном меню запроса в навигаторе
	 объектов.


	Данные, содержащиеся в запросе, автоматически обновляются в соответствии
	 с полученной информацией о запросе при его открытии.


	Если запрос открыт на просмотр и обновлена информация о нём, то
	 для обновления данных:


		- в веб-приложении нажмите кнопку ![](../Table/update.png) «Обновить содержимое таблицы»
		 на панели инструментов;


		- в настольном приложении:


			- нажмите кнопку ![](../Table/Work/View_Panel_Button_12.gif) «Обновить содержимое таблицы»
			 на панели инструментов;


			- выполните команду «Таблица >
			 Обновить» в главном меню;


			- нажмите клавишу F5.


	После выполнения действий содержимое запроса будет обновлено.


[![](../Opened.gif)![](../Closed.gif)Конструирование](javascript:TextPopup(this))


	Для запуска окна конструктора запросов в настольном приложении выполните
	 команду «Конструктор» в контекстном
	 меню запроса. Будет открыто окно конструктора запросов в режиме редактирования.


[![](../Opened.gif)![](../Closed.gif)Управление версиями](javascript:TextPopup(this))


	Если репозиторий [подключен
	 к системе управления версиями](uidevenv.chm::/04_NavigatorSetting/VCS/Add_In_VCS.htm), то запрос может
	 быть добавлен в систему для контроля изменений его версии. Для этого
	 в навигаторе объектов в контекстном меню запроса выполните команду
	 «Добавить в VCS». После отправки
	 объекта на сервер VCS будет возможность отслеживать его изменение.
	 В системе управления версиями сохраняются текст запроса и настройки
	 параметров. Более подробно о работе с объектами, добавленными в систему
	 управления версиями, читайте в подразделе «[Разработка
	 приложений в команде](UiDevEnv.chm::/01_Development_Environment/06_Version_Control_System/VSC_Introduction.htm)».
