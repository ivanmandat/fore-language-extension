# Источник данных ODBC


В репозитории можно получить доступ к различным источникам данных, поддерживающим
 работу через ODBC-драйверы, установленные на компьютере. Для получения
 доступа предназначен объект репозитория - источник данных ODBC. Данный
 объект позволяет провести настройку подключения и извлечь данные. Данные
 будут представлены в виде таблицы и могут участвовать в формировании многомерных
 структур данных репозитория.


Примечание.
 Создание источника данных ODBC и работа с ним доступны только в настольном
 приложении.


## Создание источника данных ODBC


Для создания источника данных ODBC в [навигаторе
 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm):


	- Выполните одно из действий:


		- выполните команду «Создать >
		 Таблица» в контекстном меню;


		- нажмите кнопку «Новый объект»,
		 расположенную в группе «Создать»
		 на вкладке «[Главная](GetStarted.chm::/Interface/Interface_Description.htm#customize_ribbon)»
		 ленты инструментов, и выберите пункт «Таблица».


	- Выберите тип создаваемого объекта на странице «Тип
	 объекта».


После выполнения
 действий в мастере источника данных ODBC задайте настройки на страницах:


	- [Базовые
	 свойства](Master/base_settings.htm). Задаются наименование объекта,
	 идентификатор и примечание;


	- [Параметры](Master/parameters.htm).
	 Формируется список параметров;


	- [Источник
	 данных ODBC](Master/UiDb_relational_ODBC_master_databaseODBC.htm). Определяется источник, либо ODBC-драйвер,
	 на базе которого будет создан новый источник;


	- [Настройка](Master/UiDb_relational_ODBC_master_tuning.htm).
	 Определяются параметры подключения к источнику данных.


## Работа с источником данных ODBC


Для просмотра данных выделите соответствующий объект в [навигаторе
 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm) и выполните одно из действий:


	- выполните команду «Открыть»
	 в контекстном меню объекта;


	- дважды щёлкните по объекту;


	- нажмите клавишу ENTER;


	- выполните команду «Просмотр»
	 на вкладке «Главная»
	 ленты инструментов в настольном приложении.


После выполнения одного из действий будет открыт соответствующий объект
 с данными. Для получения подробной информации о принципах работы с набором
 данных обратитесь к разделу «[Работа
 с набором данных](../Table/Work/UiDb_relational_table_work.htm)».


Примечание.
 Если с использованием драйвера ODBC осуществляется подключение к файлу
 с данными (текстовый файл, файл Microsoft Excel и другие), то работа с
 записями будет производиться только в режиме чтения. Добавление/удаление/изменение
 записей не поддерживается.
