# Источник данных ODBC


На странице «Источник данных ODBC»
 создается новый источник данных или настраивается существующий DSN:


![](page_4_ODBC.png)


Для создания нового источника данных необходимо установить переключатель
 «Новый» и в раскрывающемся списке
 выбрать один из ODBC-драйверов. Для доступа к данным могут использоваться
 как собственные (встроенные) драйвера, так и любые ODBC-драйвера, установленные
 на клиентском компьютере. Настройка источника с выбранным драйвером производится
 на следующей странице мастера.


Для настройки существующего источника необходимо установить переключатель
 «На основе существующего», затем
 в раскрывающемся списке выбрать одно из наименований существующих источников
 данных (DSN). В списке будут доступны все пользовательские DSN, которые
 были настроены при администрировании источников ODBC в операционной системе.
 Настройка драйверов также производится на следующей странице «Мастера
 источника данных ODBC».


## Первоначальная настройка


При первом создании источника данных ODBC в репозитории после выбора
 на базе чего он будет создаваться и при нажатии кнопки «Далее»
 будет открыто окно минимальных настроек драйвера. Вид данного окна зависит
 от выбранного источника, описание которого следует смотреть в справочной
 системе соответствующего приложения (источника). В соответствии с настройками,
 которые будут указаны в этом окне, будет сформирована строка подключения.
 После этого будет осуществляться переход к странице «[Настройка
 источника данных ODBC](UiDb_relational_ODBC_master_tuning.htm)».


При редактировании источника данных ODBC в репозитории при нажатии кнопки
 «Далее» будет осуществляться переход
 к странице «[Настройка
 источника данных ODBC](UiDb_relational_ODBC_master_tuning.htm)». Если перед переходом требуется произвести
 минимальную настройку (например, для изменения источника), то установите
 флажок «Провести первоначальную настройку».


Примечание.
 При работе с существующим источником минимальная настройка осуществляется
 всегда.
