# Связь измерений с существующими справочниками


Страница отображается, если выбран [новый
 приёмник данных](Select_consumer.htm), и предназначена для связи импортируемых измерений
 с существующими справочниками репозитория.


![](fixing_dimensions.png)


Для связи измерения с существующим справочником выберите справочник
 из раскрывающегося списка, который расположен напротив необходимого измерения.
 Для выбора доступны только справочники НСИ, составные справочники НСИ
 и календарные справочники.


Для дополнения списка элементов выбранного справочника из поля источника
 установите флажок «Добавлять новые записи».
 При снятом флажке элементы не добавляются и импорт данных будет производиться
 только по тем элементам, которые присутствуют в источнике и в выбранном
 справочнике. Если измерение приёмника не выбрано или привязка была сброшена
 пользователем, то в раскрывающемся списке измерений будет отображён текст
 «Не задано». Флажок «Добавлять
 новые записи» в этом случае будет недоступным. Для календарного
 измерения в раскрывающемся списке можно выбрать необходимую динамику.
 По умолчанию динамика определяется автоматически по первым загруженным
 записям. Если выбран существующий календарный справочник, то его параметры
 будут изменены.


Для привязки к данным будут использоваться только имеющиеся в справочнике
 элементы.


Примечание.
 Учитывайте имеющиеся особенности при импорте в существующие [иерархические
 или мультиязычные справочники](Features_Import_Into_An_Existing_Dimensions.htm).


Если измерение не связано с существующим справочником, то для него будет
 создан новый справочник НСИ.


При нажатии на кнопку «Импорт»
 будет создан новый стандартный куб, созданы необходимые справочники, таблицы
 и импортированы данные.


Если во время импорта возникли какие-либо ошибки, то по завершению будет
 открыто окно «[Конфликты
 импорта данных](conflict_imports.htm)». Если ошибок не было, то будет отображена последняя
 страница мастера импорта - «[Настройка
 справочников](Directories_Result.htm)».
