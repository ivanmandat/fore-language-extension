# Репозиторий НСИ


Объект «Репозиторий НСИ», отображаемый
 в навигаторе объектов, является устаревшим и используется для совместимости
 с прошлыми версиями.


Для создания репозитория метаданных НСИ используйте «[Менеджер репозитория](Setup.chm::/05_RepoMngr/RepoMngr_Windows.htm#rds)».


Примечание.
 Создание репозитория НСИ доступно только в настольном приложении.
