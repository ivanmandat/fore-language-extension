# Обработчики данных в СУБД


В данной статье приведено описание и примеры использования обработчиков
 данных в СУБД. В «Форсайт. Аналитическая платформа»
 для обработки данных в СУБД реализованы следующие обработчики:


	- [Команда
	 СУБД](../CUBD/UiDb_relationa_CUBD.htm):


	Веб-приложение Настольное приложение


		![](../CUBD/DBMS_Command_web.png)


		![](../cumb_command.png)


Объект «Команда
 СУБД» предназначен для выполнения следующих операций с объектами
 базы данных репозитория:


		- извлечение данных;


		- вставка, обновление и удаление строк таблицы;


		- создание, замена, изменение и удаление физических объектов.


В отличие от [процедур](../procedure/UiDb_relationa_procedure.htm)
 при создании в репозитории команды СУБД никаких дополнительных объектов
 на сервере не создаётся;


	- [Процедура](../procedure/UiDb_relationa_procedure.htm):


	Веб-приложение Настольное приложение


		![](../procedure/procedure_web.png)


		![](../procedure.png)


Объект «Процедура»
 позволяет создавать процедуры и функции для разных СУБД.


В отличие от [команды
 СУБД](../CUBD/UiDb_relationa_CUBD.htm) при создании в репозитории объекта «Процедура»
 на сервере будет создана хранимая процедура/функция. Несколько процедур
 репозитория могут ссылаться на одну и ту же хранимую процедуру на сервере
 базы данных. Изменение текста хранимой процедуры отразится на всех объектах,
 ссылающихся на нее. Результатом работы процедуры может являться количество
 обработанных записей либо конкретное значение, если создана не процедура,
 а функция.
