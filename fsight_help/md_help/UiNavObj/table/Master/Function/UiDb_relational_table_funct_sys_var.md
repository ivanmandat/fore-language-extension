# Функции, возвращающие системные переменные


## SYSDATE


### Синтаксис


SYSDATE


### Описание


Функция SYSDATE возвращает текущие
 дату и время. Аргументов нет.


Примечание.
 Функция доступна для использования только при работе с СУБД Oracle.


### Пример


BIRTHDATE <= SYSDATE


Значение поля «BIRTHDATE» типа «Дата» не может быть больше текущей даты.


## USERENV


### Синтаксис


USERENV('Option')


### Описание


Функция USERENV возвращает значение,
 содержащее информацию о текущем сеансе, на основе параметра Option.
 Допустимые значения параметра:


	- LANGUAGE. Возвращает
	 язык и территорию, используемые в данный момент сеансом, а также набор
	 символов базы данных. Возвращаемое значение имеет форму language_territory.characterset.
	 Например: RUSSIAN_CIS.CL8MSWIN1251;


	- TERMINAL. Возвращает
	 идентификатор терминала текущего сеанса;


	- SESSIONID. Возвращает
	 идентификатор сеанса аудита;


	- ENTRYID. Возвращает
	 доступный идентификатор записи аудита;


	- LANG. Возвращает сокращение
	 ISO для названия языка текущего сеанса. Например: RU.


Примечание.
 Функция доступна для использования только при работе с СУБД Oracle.


### Пример


USER_LNG = USERENV('LANG')


Значение поля «USER_LNG» типа «Строка» может содержать только название
 языка текущего сеанса.
