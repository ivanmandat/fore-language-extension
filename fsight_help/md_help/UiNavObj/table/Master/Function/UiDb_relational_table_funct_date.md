# Функции для работы с датами


## ADD_MONTHS


### Синтаксис


ADD_MONTHS(d, x)


### Описание


Функция ADD_MONTHS возвращает
 дату «d» увеличенную на «х» месяцев. Аргумент «х»
 может быть любым целым числом. Если в получаемом месяце число дней меньше,
 чем в месяце «d», то возвращается
 последний день месяца-результата. Если число дней не меньше, то день месяца-результата
 и день месяца «d» совпадают. Временные
 компоненты даты «d» и результата
 одинаковы.


### Пример


VAL < ADD_MONTHS(SYSDATE, 12)


Значение поля «VAL» типа «Дата» не может быть больше текущей даты больше
 чем на 12 месяцев.
