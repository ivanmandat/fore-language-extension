# Редактирование данных


	При [работе с табличным
	 набором данных](UiDb_relational_table_work.htm) доступно редактирование записей следующих объектов:


		- [таблица](../UiDb_relational_table.htm);


		- [присоединенная
		 таблица](../../Attach_table/UiDb_relational_AttachTable.htm);


		- [источник данных
		 ODBC](../../ODBC/UiDb_relational_ODBC.htm). Доступно только в настольном приложении;


		- [журнал](../../journal/UiDb_relational_journal.htm).


	Примечание.
	 Если для объекта не установлен [первичный
	 ключ](../Master/UiDb_relational_table_master_Index.htm), то при первой попытке редактирования данных будет выдано
	 предупреждение о том, что редактирование таблицы, не имеющей первичного
	 ключа или уникальных индексов, может привести к потере данных. Для
	 продолжения редактирования данных таблицы нажмите кнопку «Да».
	 Для открытия таблицы в режиме просмотра нажмите кнопку «Нет».


	Операции с данными:


		- [добавление записи](#add);


		- [импорт данных](#import). Доступно только в настольном
		 приложении;


		- [изменение значения записи](#edit);


		- [перемещение по записям](#moving);


		- [обновление содержимого таблицы](#update);


		- [удаление текущей записи](#delete);


		- [очистка таблицы](#cleaning).


## Добавление записи


	Для добавления записи в качестве новой строки таблицы:


		- в веб-приложении нажмите кнопку ![](AddRow_web.png) «Добавить строку» на панели инструментов;


		- в настольном приложении:


			- нажмите кнопку ![](View_Panel_Button_05.gif) «Вставить
			 запись в текущую позицию» или ![](View_Panel_Button_06.gif) «Вставить запись в конец таблицы»
			 на панели инструментов;


			- выполните команду «Таблица» в
			 главном меню и выберите вариант добавления записи:


				- ![](View_Panel_Button_05.gif) «Вставить запись в текущую позицию»;


				- ![](View_Panel_Button_06.gif) «Вставить запись в конец таблицы».


	После выполнения одного из действий в таблице будет добавлена новая
	 строка. Заполните пустые ячейки данными.


	Для сохранения изменений в веб-приложении:


		- нажмите кнопку «Сохранить»;


		- подтвердите производимое действие
		 в информационном окне при переходе в [навигатор
		 объектов](getstarted.chm::/Interface/Interface_Navigator.htm)/[мастер
		 табличного набора данных](../Master/object_type_page.htm).


## Импорт данных


	Примечание.
	 Доступно только в настольном приложении.


	Для [импорта
	 данных](../../Data_import_wizard/Data_import_wizard.htm) в куб выполните команду «Сервис >
	 Импорт данных» в главном меню.


## Изменение значения записи


	Для изменения значения записи целиком выделите ячейку таблицы и
	 задайте новое значение с помощью клавиатуры.


	Для частичного изменения значения записи переведите ячейку в режим
	 редактирования:


		- в веб-приложении дважды щёлкните по ячейке;


		- в настольном приложении щёлкните по выделенной ячейке.


	После выполнения одного из действий ячейка будет переведена в режим
	 редактирования. Измените значение записи.


## Перемещение по записям


	Для перемещения по записям в таблице:


		- в веб-приложении используйте мышь или клавиши управления
		 курсором на клавиатуре;


		- в настольном приложении:


			- используйте мышь или клавиши управления курсором на
			 клавиатуре;


			- для перехода к первой записи:


				- нажмите кнопку ![](View_Panel_Button_01.png) «Перейти к первой записи»
				 на панели инструментов;


				- выполните команду «Таблица >
				 Первая запись» в главном меню;


			- для перехода к предыдущей записи:


				- нажмите кнопку ![](View_Panel_Button_02.png) «Перейти к предыдущей записи»
				 на панели инструментов;


				- выполните команду «Таблица >
				 Предыдущая запись» в главном меню;


			- для перехода к следующей записи:


				- нажмите кнопку ![](View_Panel_Button_03.png) «Перейти к следующей записи»
				 на панели инструментов;


				- выполните команду «Таблица >
				 Следующая запись» в главном меню;


			- для перехода к последней записи:


				- нажмите кнопку ![](View_Panel_Button_04.png) «Перейти к последней записи»
				 на панели инструментов;


				- выполните команду «Таблица >
				 Последняя запись» в главном меню.


## Обновление содержимого таблицы


	Для обновления содержимого таблицы:


		- в веб-приложении нажмите кнопку ![](../update.png) «Обновить содержимое таблицы»
		 на панели инструментов;


		- в настольном приложении:


			- нажмите кнопку ![](View_Panel_Button_12.gif) «Обновить
			 содержимое таблицы» на панели инструментов;


			- выполните команду «Таблица >
			 Обновить» в главном меню;


			- нажмите клавишу F5.


	После выполнения одного из действий содержимое таблицы будет обновлено.
	 [Фильтрация данных](UiDb_relational_table_work_Filter.htm)
	 при обновлении содержимого таблицы сбрасывается.


## Удаление текущей записи


	Для удаления текущей выделенной записи:


		- в веб-приложении нажмите кнопку ![](b_delete_web.png) «Удалить текущую запись» на панели
		 инструментов;


		- в настольном приложении:


			- нажмите кнопку ![](../../TimeSeriesDatabase/b_delete.gif) «Удалить текущую запись»
			 на панели инструментов;


			- выполните команду «Таблица >
			 Удалить текущую запись» в главном меню.


	После выполнения одного из действий выделенная запись будет удалена
	 без запроса подтверждения выполняемого действия.


## Очистка таблицы


	Для очистки таблицы и удаления всех записей:


		- в веб-приложении нажмите кнопку ![](Delete_web.png) «Очистить таблицу» на панели
		 инструментов;


		- в настольном приложении:


			- нажмите кнопку ![](View_Panel_Button_08.gif) «Очистить
			 таблицу» на панели инструментов;


			- выполните команду «Таблица >
			 Очистить таблицу» в главном меню.


	После выполнения одного из действий будет запрошено подтверждение
	 о выполняемом действии. При положительном ответе таблица будет очищена.
