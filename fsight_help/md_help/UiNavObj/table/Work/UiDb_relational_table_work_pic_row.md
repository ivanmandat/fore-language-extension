# Выбор столбцов


При [работе с табличным набором
 данных](UiDb_relational_table_work.htm) доступен выбор отображаемых столбцов.


Примечание.
 Выбор столбцов доступен только в настольном приложении.


Для выбора отображаемых столбцов используйте:


	- контекстное меню заголовка столбца;


	- окно «Выбрать столбцы для отображения»,
	 которое вызывается при выполнении команды главного меню «Таблица >
	 Выбрать столбцы»:


![](UiDb%20Conception%20Select%20column.gif)


В окне «Выбрать столбцы для отображения»
 отображается список всех столбцов, входящих в набор данных. Слева от наименования
 столбцов расположены флажки, с помощью которых определяется необходимость
 отображения или скрытия столбцов. Установка флажков и определение порядка
 столбцов осуществляется с помощью кнопок:


	- Вверх. Перемещает столбец
	 вниз;


	- Вниз. Перемещает столбец
	 вверх;


	- Отобразить. Устанавливает
	 флажок, определяющий необходимость отображения столбца;


	- Скрыть. Снимает флажок,
	 определяющий необходимость отображения столбца;


	- Сбросить. Сбрасывает
	 настройки в исходные: все колонки будут отмечены и выстроены в том
	 порядке, в котором они хранятся в базе (или запрашиваются в запросе).


Также установка и снятие флажков возможна с помощью мыши.


Для определения ширины столбца его следует выделить и задать значение
 в редакторе чисел «Ширина выделенного
 столбца».
