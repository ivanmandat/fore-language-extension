# Оформление ячеек


Примечание.
 Для применения оформления соблюдайте правильный [порядок](UiNav.chm::/GUI/applystyle.htm).


Для настройки оформления ячеек в инструменте «Отчёты»
 предусмотрены следующие возможности:


	- настройка [атрибутов](../Attribute/UiReport_Table_Attribute.htm)
	 ячеек листа;


	- [условное
	 форматирование](UiAnalyticalArea.chm::/Analysis/Conditional_formatting.htm): позволяет настраивать оформление
	 ячеек в зависимости от их содержимого;


	- оформление диапазона ячеек в виде [таблицы
	 с чередующимся оформлением](../Attribute/UiReport_Table_Styles.htm) строк/столбцов;


	- сохранение оформления ячеек в виде [стиля](uianalyticalarea.chm::/TableView/Formatting/Table_Elements_Styles.htm)
	 и применение его к другим диапазонам ячеек или подключение внешних
	 стилей;


	- задание прочих [настроек](UiAnalyticalArea.chm::/TableView/Formatting/Formatting_table.htm)
	 оформления табличного визуализатора и его элементов;


	- настройка форматирования:


		- текста ячеек с помощью групп «Шрифт»
		 и «Выравнивание» на вкладке
		 «Главная» ленты инструментов;


		- части текста ячеек с помощью группы «Шрифт»
		 на вкладке «Главная» ленты
		 инструментов. Доступно для [текстового
		 формата данных](../../../Web/Table/Work/UiReport_Table_WorkStyle.htm#data_format).


Смешанное форматирование текста в ячейках
 сохраняется при использовании [автозаполнения
 ячеек данными](UiNav.chm::/GUI/UiExpress_Tabl_Operation_AutoFillData.htm).


Примечание.
 Для ячейки сохраняется форматирование первого символа текста при замене/удалении
 текста из неё.
