# Редактирование содержимого ячеек


Для настройки параметров форматирования содержимого ячеек перейдите
 в режим редактирования одним из способов:


	- выделите ячейку на листе отчета и введите необходимые данные;


	- дважды щёлкните по выбранной ячейке;


	- выделите ячейку и нажмите клавишу F2, после чего введите данные.


## Операции над содержимым ячеек


После перехода в режим редактирования с данными, содержащимися в ячейке,
 можно производить следующие операции:


[![](../../../opened.gif)![](../../../closed.gif)Настройка формата](javascript:TextPopup(this))


	Для настройки формата выбранных ячеек выполните команду контекстного
	 меню «Формат» ячейки. Откроется
	 окно «[Формат](../../Objects/UiReport_format.htm)»,
	 содержащее вкладку «Шрифт».
	 Задайте в окне параметры форматирования содержимого ячеек.


[![](../../../opened.gif)![](../../../closed.gif)Перемещение
 данных](javascript:TextPopup(this))


	Для перемещения содержимого ячейки в буфер обмена:


		- выполните команду контекстного меню «Вырезать»
		 ячейки;


		- нажмите сочетание клавиш CTRL+X или SHIFT+DELETE.


[![](../../../opened.gif)![](../../../closed.gif)Копирование
 данных](javascript:TextPopup(this))


	Для копирования содержимого выбранной ячейки в буфер обмена:


		- выполните команду «Копировать»,
		 расположенную в группе «Буфер
		 обмена» на вкладке «Главная»
		 ленты инструментов;


		- выполните команду контекстного меню «Копировать»
		 ячейки;


		- нажмите сочетание клавиш CTRL+С или CTRL+INSERT.


[![](../../../opened.gif)![](../../../closed.gif)Вставка
 данных](javascript:TextPopup(this))


	Для вставки данных в выбранную ячейку из буфера обмена:


		- выполните команду «Вставить»,
		 расположенную в группе «Буфер
		 обмена» на вкладке «Главная»
		 ленты инструментов;


		- выполните команду контекстного меню «Вставить»
		 ячейки;


		- нажмите сочетание клавиш CTRL+V или SHIFT+INSERT.


[![](../../../opened.gif)![](../../../closed.gif)Выделение данных](javascript:TextPopup(this))


	Для выделения всего содержимого выбранной ячейки выполните команду
	 контекстного меню «Выделить все».


Примечание.
 В режиме просмотра доступно только изменение и сохранение содержимого
 ячейки, но не форматирование.


После ввода данных для выхода из режима редактирования содержимого ячейки
 нажмите клавишу ENTER или клавишу TAB.


Для отмены и повтора действия с содержимым ячейки обратитесь к разделу
 «[Отмена и повтор операции](../../../Web/Reports/Undo_Redo.htm)».
