# Работа с данными на листе отчёта


Лист регламентного отчёта содержит элементы: строки, столбцы и их пересечение
 - ячейки. Множество строк и столбцов представляют данные в виде таблиц.
 Значение в числовой ячейке таблицы можно указать в явном виде, взять из
 источника данных, либо [рассчитать
 по формуле](../organizational_management/Function/UiReport_Function.htm).


С данными на листе отчёта доступны операции:


	- [ввод данных в ячейку
	 листа](Work/UiReport_Table_Work_Data.htm);


	- [редактирование
	 содержимого ячеек](Work/UiReport_Table_Work_Data_Edit.htm);


	- [копирование и вставка
	 данных](Work/UiReport_Table_Work_insert.htm);


	- [добавление примечания](Work/UiReport_Table_Work_Notice.htm);


	- [добавление
	 именованного диапазона](../Reports/OperationReport/UiReport_Reports_Operation_Name.htm);


	- [использование
