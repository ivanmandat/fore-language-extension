# Использование встроенной расширенной аналитики


Для изучения количественных и качественных экономических взаимосвязей
 с помощью математических и статистических методов и моделей, проверки
 данных на соответствие заданным условиям и ограничениям, выявления скрытых
 фактов и взаимосвязей в больших массивах данных используйте средства расширенной
 аналитики:


	- [эконометрику](UiAnalyticalArea.chm::/Analysis/AdvancedAnalytics.htm#econometrics);


	- [валидацию](UiAnalyticalArea.chm::/Analysis/AdvancedAnalytics.htm#data_validation);


	- [интеллектуальный
	 анализ](UiAnalyticalArea.chm::/Analysis/AdvancedAnalytics.htm#data_mining).


Для выбора средств расширенной аналитики перейдите на вкладку «Расширенная аналитика» ленты инструментов.
