# Работа с элементами листа отчёта


Лист регламентного отчёта содержит элементы: строки, столбцы и их пересечение
 - ячейки. Множество строк и столбцов представляют данные в виде таблиц.
 Значение в числовой ячейке таблицы можно указать в явном виде, взять из
 источника данных, либо [рассчитать
 по формуле](../organizational_management/Function/UiReport_Function.htm). На лист возможно добавление различных [объектов](../Objects/UiReport_Objects.htm).


## Адресация ячеек листа


Адресация ячеек листа может быть абсолютной и относительной:


	- Абсолютная адресация
	 предполагает запись адреса ячейки путем указания имени столбца и номера
	 строки, например F1, B50, FQD678 и т.д.


	- Относительная адресация
	 используется в формулах, когда необходимо указать ячейку относительно
	 текущей, для которой вычисляется формула. Например, формула =R[2]C[-2]
	 означает ссылку на ячейку, которая располагается на 2 столбца правее
	 и на две строки выше от текущей.


По левому краю листа располагается столбец с номерами строк, а по верхнему
 краю - строка с именами столбцов.


Строки листа нумеруются сверху вниз числами, начиная с нуля, а столбцы
 - слева направо буквами латинского алфавита A, B, … , Y, Z. Если количество
 столбцов превышает 26, то далее используются двухбуквенные номера: AA,
 AB, … AZ, BA, BB и т.д. вплоть до ZZ, потом трехбуквенные от AAA до ZZZ
 и т.д.


Адрес ячейки состоит из номера столбца и номера строки, записанных без
 пробела, например: A1, B50 или FQD678.


Прямоугольная область ячеек может быть задана с помощью диапазона адресов
 ячеек. Диапазон состоит из адресов левой верхней и правой нижней ячеек
 блока, разделенных двоеточием. Например, диапазон C12:AB14 включает ячейки,
 расположенные между столбцами от C до AB и строками 12, 13 и 14.


## Операции при работе с элементами листа


[![](../../opened.gif)![](../../closed.gif)Выделение
 ячеек](javascript:TextPopup(this))


	При выполнении операций с ячейками листа необходимо предварительно
	 [выделить
	 ячейку или группу ячеек](UiNav.chm::/GUI/Cells_Select.htm#select_list), над которыми будет производиться
	 операция.


[![](../../opened.gif)![](../../closed.gif)Операции с ячейками](javascript:TextPopup(this))


	Инструмент «Отчеты» позволяет
	 выполнять следующие [операции
	 с ячейками](Work/UiReport_Table_Work_Union.htm):


		- [добавление
		 ячеек](Work/UiReport_Table_Work_Union.htm#adding_cells);


		- [удаление
		 ячеек](Work/UiReport_Table_Work_Union.htm#deleting_cells);


		- [объединение
		 ячеек](Work/UiReport_Table_Work_Union.htm#merging_unmerging_cells);


		- [разъединение
		 ячеек](Work/UiReport_Table_Work_Union.htm#merging_unmerging_cells).


[![](../../opened.gif)![](../../closed.gif)Операции со
 строками](javascript:TextPopup(this))


	Инструмент «Отчеты» позволяет выполнять следующие [операции
	 над строками](OperationRow/UiReport_Table_Row.htm):


		- [добавление
		 строк](OperationRow/UiReport_Table_Row.htm#inserting_rows);


		- [изменение
		 высоты строк](OperationRow/UiReport_Table_Row.htm#changing_row_height);


		- [удаление
		 строк](OperationRow/UiReport_Table_Row.htm#deleting_rows);


		- [скрытие
		 строк](OperationRow/UiReport_Table_Row.htm#hiding_rows);


		- [отображение
		 скрытых строк](OperationRow/UiReport_Table_Row.htm#showing_hidden_rows);


		- закрепление первой строки;


		- снятие закрепления областей.


[![](../../opened.gif)![](../../closed.gif)Операции со
 столбцами](javascript:TextPopup(this))


	Инструмент «Отчеты» позволяет выполнять следующие [операции
	 над столбцами](OperationColumn/UiReport_Table_Column.htm):


		- добавление столбцов;


		- изменение ширины столбцов;


		- удаление столбцов;


		- скрытие столбцов;


		- отображение скрытых столбцов;


		- закрепление первого столбца;


		- снятие закрепления областей.


[![](../../opened.gif)![](../../closed.gif)Оформление
 ячеек](javascript:TextPopup(this))


	Для настройки [оформления
	 ячеек](Work/UiReport_Table_WorkStyle.htm) в инструменте «Отчеты» предусмотрено несколько гибких механизмов:


		- условное форматирование;


		- оформление диапазона ячеек в виде таблицы с чередующимся
		 оформлением строк/столбцов;


		- сохранение оформления ячейки в виде стиля и применение его
		 к другим ячейкам;


		- подключение набора внешних стилей ячеек к нескольким отчетам
		 с последующей синхронизацией при изменении этих стилей.
