# Настройка фильтрации


Параметры фильтрации строк и столбцов области данных аналогичны друг
 другу, но задаются на отдельных вкладках.


Примечание.
 Фильтрация пустых элементов производится до вычисления, но для элементов
 размещения, не привязанных к измерению, резервируется место.


Для настройки параметров фильтрации перейдите на вкладку «Фильтрация >
 Столбцы/Строки» окна «[Свойства области данных](UiReport_AreaData_Param.htm)».
 Окно примет вид:


![](2_Filter.png)


На данной вкладке определите параметры фильтрации данных:


[![](../../../opened.gif)![](../../../closed.gif)Скрыть пустые
 значения](javascript:TextPopup(this))


	При установленном флажке «Скрыть
	 пустые значения» будут скрываться все пустые строки (столбцы)
	 области данных. Под пустой строкой (столбцом) понимается строка (столбец),
	 не содержащая ни одного элемента данных.


[![](../../../opened.gif)![](../../../closed.gif)Скрыть нулевые
 значения](javascript:TextPopup(this))


	При установленном флажке «Скрыть
	 нулевые значения» будут скрываться все строки (столбцы) области
	 данных, содержащие только нулевые значения.


[![](../../../opened.gif)![](../../../closed.gif)Скрыть нечисловые
 значения](javascript:TextPopup(this))


	При установленном флажке «Скрыть
	 нечисловые значения» будут скрываться все строки (столбцы)
	 области данных, содержащие только нечисловые значения.


[![](../../../opened.gif)![](../../../closed.gif)Скрывать значения
 по условию](javascript:TextPopup(this))


	При установленном флажке «Скрывать
	 значения по условию» становится доступным поле «Выражение»,
	 введите в нем условие фильтрации. При формировании условия фильтрации
	 используется переменная «VALUE».
	 Данная переменная определяет содержимое ячейки. Например, при условии
	 «VALUE>100», будут скрыты
	 все строки (столбцы) области данных, содержащие значения более 100.


	Условия фильтрации можно вводить:


		- вручную. Введите условие в поле
		 ввода;


		- с помощью [редактора
		 выражения](UiNav.chm::/GUI/ExpressionEditor.htm). Для запуска редактора выражения
		 нажмите кнопку ![](../../Reports/option_button.png).


[![](../../../opened.gif)![](../../../closed.gif)Определить элементы,
 не участвующие в условиях фильтрации](javascript:TextPopup(this))


	При установленном флажке «Определить
	 элементы, не участвующие в условиях фильтрации» становится
	 доступным иерархический список «Элементы
	 измерения». Отметьте в списке элементы, которые не будут учитываться
	 при фильтрации. Если в срезе по строкам (столбцам) расположено несколько
	 измерений, то в списке отображаются элементы только последнего измерения.


Примечание.
 Если на вкладке «Размещение» установлены
 флажки «Отображать владельцев неисключенных
 строк/столбцов», «Использовать
 ступенчатую иерархию», то при использовании любого вида фильтрации
 у владельцев неисключенных строк не выводятся дочерние элементы, кроме
 тех, по которым есть данные.


При настройке фильтрации области данных можно настроить фильтрацию,
 [привязанную к измерениям](UiReport_AreaData_Param_FilterDim.htm).
