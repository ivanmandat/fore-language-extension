# Настройка общих параметров


Для настройки общих параметров области данных перейдите на вкладку «Параметры» окна «[Свойства области данных](UiReport_AreaData_Param.htm)».
 Окно примет вид:


![](parameters.png)


На данной вкладке определите общие параметры области данных:


[![](../../../opened.gif)![](../../../closed.gif)Идентификатор](javascript:TextPopup(this))


	Идентификатор - уникальная
	 последовательность букв латинского алфавита, цифр и знаков, которая
	 должна начинаться с буквы. Однозначно идентифицирует область
	 данных в пределах данного отчета. Каждой области данных по умолчанию
	 присваивается идентификатор, который отражается в данном поле.


	Для смены идентификатора введите в данное поле новое значение.


[![](../../../opened.gif)![](../../../closed.gif)Срез](javascript:TextPopup(this))


	Параметр определяет, какой из срезов отчета будет использоваться
	 в качестве источника данных.


	Выберите необходимый срез из раскрывающегося списка существующих
	 срезов отчета.


	При создании [шаблона
	 отчета](../../Reports/OperationReport/Report_Template.htm) срез можно не задавать.


[![](../../../opened.gif)![](../../../closed.gif)Границы области
 данных](javascript:TextPopup(this))


	Параметр определяет границы области листа, в которой будут размещаться
	 данные из среза, указанного в качестве параметра «Срез».
	 Значение параметра можно указать следующими способами:


		- Единичная ячейка.
		 Укажите адрес ячейки листа, которая будет являться верхней левой
		 ячейкой для области данных;


		- Диапазон ячеек.
		 Укажите адрес диапазона ячеек листа. Указанный диапазон определяет
		 границы размещения области данных.


	Адрес диапазона указывается следующим
	 образом:


	ячейка1:ячейка2


	где:


			- ячейка1. Адрес ячейки листа, которая будет являться
			 верхней левой ячейкой диапазона;


			- ячейка2. Адрес ячейки листа, которая будет являться
			 нижней правой ячейкой диапазона.


[![](../../../opened.gif)![](../../../closed.gif)Лист](javascript:TextPopup(this))


	Параметр определяет, на каком [листе
	 отчета](../../Sheets/UiReport_Sheets.htm) будет располагаться область данных.


	Выберите необходимый лист отчета из раскрывающегося списка существующих
	 листов отчета.


[![](../../../opened.gif)![](../../../closed.gif)Поведение строк](javascript:TextPopup(this))


	Параметр определяет, как будут вести себя строки области данных,
	 а также строки, расположенные ниже области данных, при изменении размеров
	 самой области данных. Возможны следующие варианты:


		- Фиксированное. Количество
		 строк области данных будет фиксированным (определенным в параметре
		 «Границы области данных»)
		 и изменяться не будет;


		- Захват смежных строк.
		 При увеличении количества строк области данных информация, расположенная
		 в строках ниже области данных, будет перекрываться данными из
		 соответствующего среза области данных;


		- Вставка строк. При
		 увеличении количества строк области данных на лист отчета будет
		 добавлено необходимое количество строк. При этом информация, расположенная
		 в строках ниже, будет смещена вниз.


[![](../../../opened.gif)![](../../../closed.gif)Поведение столбцов](javascript:TextPopup(this))


	Параметр определяет, как будут вести себя столбцы области данных,
	 а также столбцы, расположенные справа от области данных, при изменении
	 размеров самой области данных. Возможны следующие варианты:


		- Фиксированное. Количество
		 столбцов области данных будет фиксированным (определенным в параметре
		 «Границы области данных»)
		 и изменяться не будет;


		- Захват смежных столбцов.
		 При увеличении количества столбцов области данных информация,
		 расположенная в столбцах правее области данных, будет перекрываться
		 данными из соответствующего среза области данных;


		- Вставка столбцов.
		 При увеличении количества столбцов области данных на лист отчета
		 будет добавлено необходимое количество столбцов. При этом информация,
		 расположенная в столбцах правее области данных, будет смещена
		 вправо.


[![](../../../opened.gif)![](../../../closed.gif)Разрешить сохранение
 измененных данных в источник](javascript:TextPopup(this))


	При установленном флажке «Разрешить
	 сохранение измененных данных в источник» измененные данные
	 области будут сохраняться в источник данных. Если флажок снят, то
	 измененные данные не будут сохраняться в источник.


	Примечание.
	 Сохранение измененных данных в источник не доступно, если установлена
	 [защита
	 ячейки](UiNav.chm::/GUI/Format/UiReport_Table_Attribute_Others.htm).
