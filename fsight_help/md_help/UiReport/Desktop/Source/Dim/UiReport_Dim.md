# Работа со списком в измерении


В список измерения включены элементы справочника, на котором основано
 измерение источника.


При работе со списком в измерении доступны настройки списка, подобные
 настройкам справочника.


## Основные настройки


[![](../../../opened.gif)![](../../../closed.gif)Сортировка
 элементов измерения](javascript:TextPopup(this))


	Для настройки [сортировки](UiNavObj.chm::/reference_book/Work/Sort_Elements.htm) элементов
	 выполните команду контекстного меню «Сортировка»
	 элемента.


[![](../../../opened.gif)![](../../../closed.gif)Определение
 наименования уровня](javascript:TextPopup(this))


	Для определения атрибута, используемого для наименования уровней
	 элементов, выполните команду контекстного меню «Наименование»
	 элемента.


[![](../../../opened.gif)![](../../../closed.gif)Отображение
 в измерении элементов определенной группы](javascript:TextPopup(this))


	Для работы с группой элементов должны быть выполнены следующие условия:


		- для справочника должны быть настроены [группы
		 элементов](UiNavObj.chm::/reference_book/look-and-feel_Reference_book/UiMd_reference_book_look-and-feel_Group.htm);


		- измерение должно быть не зафиксировано, либо настроена [агрегация по фиксированным
		 измерениям](../Window/UiReport_Cache.htm).


	Примечание.
	 При совместном использовании «Группы
	 элементов» и «Схемы отметки»
	 для одного справочника необходимо учитывать, что результатом будет
	 являться их взаимодействие. В выбранной группе элементов будут отмечены
	 те элементы, которые соответствуют выбранной схеме отметки.


	Для отображения в измерении элементов определенной группы:


		- выполните команду контекстного меню «Отобразить
		 группу» элемента измерения. Во всплывающем меню выберите
		 группу элементов.


	Примечание.
	 Во всплывающем меню пункта «Отобразить
	 группу» отображается только десять последних использовавшихся
	 групп. Если для измерения определено более десяти групп, то в меню
	 будет присутствовать пункт «Другая
	 группа». В открывшемся диалоге «Выбор
	 группы элементов» выберите группу элементов.


		- выберите группу элементов в окне «[Параметры
		 измерения](UiReport_Dim_Change_mark.htm#dimension_parameters)» в списке «Группа
		 элементов».


	После этого все элементы, не включенные в выбранную группу элементов,
	 будут скрыты в измерении.


	Для отображения полного списка элементов в окне «[Параметры
	 измерения](UiReport_Dim_Change_mark.htm#dimension_parameters)» в списке «Группа элементов»
	 выберите пункт «нет».


## Работа с иерархией измерения


[![](../../../opened.gif)![](../../../closed.gif)Выбор
 альтернативной иерархии](javascript:TextPopup(this))


	Для выбора альтернативной иерархии выполните команду контекстного
	 меню «Альтернативная иерархия»
	 элемента.


	Примечание.
	 Команда доступна, если для измерения настроена [альтернативная
	 иерархия](UiNavObj.chm::/reference_book/UiMd_reference_book_Hierarchy.htm#alternativehierarchy).


[![](../../../opened.gif)![](../../../closed.gif)Сворачивание
 иерархии списка](javascript:TextPopup(this))


	Для сворачивания иерархии списка:


		- выполните команду контекстного меню «Свернуть
		 все» элемента;


		- нажмите клавишу LEFT, переходя по элементам с помощью клавиш
		 UP и DOWN.


[![](../../../opened.gif)![](../../../closed.gif)Разворачивание
 иерархии списка](javascript:TextPopup(this))


	Для разворачивания иерархии списка:


		- выполните команду «Развернуть
		 все» контекстного меню элемента;


		- нажмите клавишу LEFT, переходя по элементам с помощью клавиш
		 UP и DOWN.
