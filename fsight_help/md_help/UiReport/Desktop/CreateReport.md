# Построение отчёта


Для построения отчёта выполните шаги:


	- Добавьте [табличный лист](Sheets/UiReport_Sheets.htm).


	- Выберите [источник данных](Source/UiReport_Source.htm)
	 отчёта.


	- Создайте [срез
	 данных](Source/Window/UiReport_Source_Window_AreaSource.htm#add_slice_aa1).


	- Визуализируйте [данные на
	 листе](Reports/Visualization.htm) отчёта.


	- Используйте [встроенную
	 расширенную аналитику](UiAnalyticalArea.chm::/Analysis/AdvancedAnalytics.htm).


	- Настройте [параметры
	 отчёта](Reports/UiReport_Reports_Param.htm).


	- Добавьте интерактивные [элементы
	 управления](Source/Controls.htm) содержимым отчёта.


	- Вставьте [иллюстрации,
	 фигуры и текстовые блоки](Objects/UiReport_Objects.htm).


	- Выберите [стиль
	 оформления](Reports/OperationReport/FormattingReport.htm) отчёта.


	- Создайте [шаблон
	 отчёта](Reports/OperationReport/Report_Template.htm), если требуется.


	- Настройте [отчёт
	 под печать](Tuning/ParamPage/UiReport_Tuning_ParamPage.htm).


	- [Просмотрите готовый
	 отчёт](Reports/OperationReport/Preview.htm).


	- [Ускорьте работу объемного отчёта](Report_in_cache.htm),
	 если требуется.


После выполнения действий сохраните изменения регламентного
 отчёта для этого:


	- выполните команду «Отчет >
	 Сохранить» в главном меню;


	- нажмите сочетание клавиш CTRL+S.


После завершения построения отчёта перейдите к шагу «[Работа
 с готовым отчётом](Reports/OperationReport/Work_witn_report.htm)».
