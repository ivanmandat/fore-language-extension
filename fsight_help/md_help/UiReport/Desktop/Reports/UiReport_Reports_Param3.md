# Настройка общих параметров отчёта


Для настройки общих параметров отчёта перейдите на вкладку «Настройки»
 окна «[Параметры](UiReport_Reports_Param.htm)».


[![](../../opened.gif)![](../../closed.gif)Для открытия
 окна](javascript:TextPopup(this))


	Выполните команду главного меню «Сервис
	 > Параметры».


![](settings_parametr.png)


Задайте на вкладке следующие параметры отчёта:


[![](../../opened.gif)![](../../closed.gif)Не спрашивать
 подтверждения при закрытии измененного экземпляра отчёта](javascript:TextPopup(this))


	При установленном флажке «Не спрашивать
	 подтверждения при закрытии измененного экземпляра отчёта» при
	 закрытии отчёта, содержащего изменённые данные, не будет предложено
	 сохранить отчёт. Все внесенные изменения при этом будут потеряны.
	 По умолчанию флажок снят.


[![](../../opened.gif)![](../../closed.gif)Не
 вычислять отчет при открытии](javascript:TextPopup(this))


	При установленном флажке «Не вычислять
	 отчет при открытии» отчёт не будет вычисляться при открытии
	 в режиме просмотра, при изменении значения ячейки листа будут пересчитываться
	 только зависимые ячейки. По умолчанию флажок снят.


	С включенной опцией отчёт будет быстрее открываться на просмотр.
	 В режиме редактирования отчёт всегда открывается без обновления.


	Примечание.
	 Редактирование данных в таблице и реляционной области данных доступно
	 только после обновления отчёта.


	Опция будет игнорироваться, если было задано значение в свойстве
	 [IMetabaseObjectParamValuesEx.StateOptions](KeSom.chm::/Interface/IMetabaseObjectParamValuesEx/IMetabaseObjectParamValuesEx.StateOptions.htm):


		- при установленном значении StateOptions = 1,
		 будет считаться, что флажок «Не
		 вычислять отчет при открытии» был установлен;


		- при установленном значении StateOptions = 2,
		 будет считаться, что флажок «Не
		 вычислять отчет при открытии» не был установлен.


[![](../../opened.gif)![](../../closed.gif)Отображать параметры
 вставки](javascript:TextPopup(this))


	При установленном флажке «Отображать
	 параметры вставки» после вставки данных из буфера обмена в
	 правом нижнем углу от вставленных ячеек появляется значок ![](../Table/Work/insert_no_format_2.png), информирующий пользователя,
	 что у данных настроен формат.


	Подробное описание приведено в разделе «[Параметры
	 форматирования при вставке из буфера обмена](../Table/Work/UiReport_Table_Insert_not_formatted.htm)».


[![](../../opened.gif)![](../../closed.gif)Переход к другой
 ячейке после нажатия клавиши ENTER](javascript:TextPopup(this))


	В группе «Переход к другой ячейке
	 после нажатия клавиши ENTER» в раскрывающемся списке выберите
	 направление перехода к другой ячейке после нажатия клавиши ENTER:


		- Вниз;


		- Вверх;


		- Влево;


		- Вправо;


		- Без перехода.


[![](../../opened.gif)![](../../closed.gif)Оформление](javascript:TextPopup(this))


	В группе «Оформление» определите
	 параметры общего оформления отчёта:


		- Набор стилей оформления.
		 В раскрывающемся списке выберите [таблицу
		 стилей](UiNav.chm::/StylesTable/StylesTablePurpose.htm), которая будет использоваться для общего
		 стилевого оформления отчётов. Для удаления таблицы стилей нажмите
		 кнопку ![](../organizational_management/Toolbars/delete16.png), расположенную
		 рядом со списком;


		- Использовать единое оформление
		 гиперссылок. При установленном флажке для всех гиперссылок
		 отчёта будет использовано заданное оформление. При снятом флажке
		 для гиперссылок используются настройки по умолчанию (подчеркнутый
		 текст синего цвета) или уникальные настройки, заданные для ячейки
		 на вкладке «[Шрифт](UiNav.chm::/GUI/Format/UiReport_Table_Attribute_Type.htm)». Для настройки
		 единого формата гиперссылок используется компонент редактора стиля.
		 Подробнее о редакторе стиля см. раздел «[Настройка
		 стиля](UiNav.chm::/GUI/Style_setup.htm)».
