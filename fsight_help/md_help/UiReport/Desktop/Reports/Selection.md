# Управление отметкой


Для выбора данных, представляемых визуализаторами, используйте отметку.
 Для [работы
 с измерениями](UiSelection.chm::/Selection/Dimension.htm)
 источника данных предназначена группа вкладок «Отметка»
 боковой панели.


## Работа с измерениями


При работе с измерениями доступны следующие операции:


	- [выбор
	 варианта расположения измерения](UiSelection.chm::/Selection/Dimension.htm#variant_of_location) ;


	- [фиксация измерения](UiSelection.chm::/Selection/Dimension.htm#fixing);


	- [настройка роли
	 измерения для всех визуализаторов](UiSelection.chm::/Selection/Dimension_type.htm);


	- [изменение
	 положения вкладок](UiSelection.chm::/Selection/Dimension.htm#change_of_pads_position);


	- [отображение/скрытие
	 измерений](UiSelection.chm::/Selection/Dimension.htm#show_dims);


	- [комбинирование
	 измерений](UiSelection.chm::/Selection/Dimension_combine.htm);


	- [редактирование
	 измерения](UiSelection.chm::/Selection/Dimension_edit.htm);


	- [настройка
	 отображения измерения](UiSelection.chm::/Selection/Setting_view_of_the_dimension.htm).


При работе с измерениями в регламентном отчёте доступны следующие возможности:


	- [работа со списком в
	 измерении](../Source/Dim/UiReport_Dim.htm);


	- [измерения
	 области данных](../Source/Dim/UiReport_Dim_Change_mark.htm).


## Работа с элементами измерения


При работе с элементами измерения доступны следующие операции:


	- [отметка
	 элементов измерения](UiSelection.chm::/Selection/Selection_of_the_dimension_elements.htm);


	- [поиск
	 элемента измерения](UiSelection.chm::/Selection/Selection_of_the_dimension_elements.htm#search).


При работе с элементами измерения в регламентном отчёте доступны следующие
 возможности:


	- [синхронизация измерений
	 в срезах данных](../Source/Dim/Symc_dimensions.htm);


	- [синхронизация
	 измерений отчета](../Source/Dim/Sync_dimentions_report.htm).
