# Работа с интерактивными элементами управления


Элементы управления располагаются рядом с рабочей областью и позволяют изменять отметку измерений или параметра отчёта.


Для отображения панели «[Элементы
 управления](../../organizational_management/Starting.htm)» нажмите кнопку ![](../../Source/Controls_Add_button.gif) «Элементы управления» группы «Вид» вкладки «Главная»
 ленты инструментов.


[Создание элементов управления](../../Source/Controls.htm)
 производится в [режиме
 редактирования](../../organizational_management/Starting.htm#open).


Элемент управления отображает выбранное измерение [в
 заданном виде](../../Source/UiReport_Source_ControlProp.htm#parameters_of_displaying) (например, в виде раскрывающегося списка), с возможностью
 [поиска](UiSelection.chm::/Selection/Dimension.htm#search)
 по измерению.


Для изменения данных, отображаемых на [визуализаторах](../Visualization.htm),
 выберите другой отмеченный элемент или параметр отчёта.
