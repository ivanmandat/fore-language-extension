# Работа с готовым отчётом


Для работы с готовым отчётом выполните шаги:


	- [Обновление
	 данных отчёта](UiReport_Reports_Operation_Execute.htm).


	- [Работа с интерактивными элементами управления](Controls.htm).


	- [Масштабирование листа](Scale.htm).


	- [Работа с данными в таблице](Work_with_data.htm).


	- Использование [встроенной
	 расширенной аналитики](UiAnalyticalArea.chm::/Analysis/AdvancedAnalytics.htm).


	- [Экспорт отчёта](UiReport_Reports_Operation_Export.htm).


	- [Предварительный
	 просмотр и печать отчёта](UiReport_Reports_Operation_Print.htm).


	- [Сохранение измененных данных](Save_Report.htm).


	- [Создание копии отчёта](Save_Report.htm#copy).


	- [Публикация в социальных сетях и обсуждение
	 отчёта](../Publish.htm).


После работы с готовым отчётом нажмите кнопку  «Отчет
 > Закрыть» главного
 меню. Сам отчёт будет закрыт, окно просмотра отчёта останется открытым.


Для закрытия окна просмотра нажмите кнопку  «Отчет
 > Выход» главного меню.


Если имеются несохранённые изменения, то будет предложено их сохранить.
