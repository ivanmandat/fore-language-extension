# Масштабирование листа


Для изменения масштаба листа:


	- нажмите кнопку ![](../../organizational_management/Toolbars/zoom16.png) «Масштаб» на вкладке «Вид»
	 ленты инструментов. После чего будет открыто раскрывающееся меню:


![](zoom_settings_desktop.png)


Выполните одно из действий:


		- установите переключатель напротив одного из фиксированных
		 значений масштаба;


		- выполните команду «Произвольный».
		 После чего будет открыто окно «Масштаб»
		 для указания произвольного значения масштаба:


![](Scale.png)


	- установите фокус на листе и, удерживая клавишу CTRL, прокрутите
	 колесико мыши. При прокручивании колесика вверх масштаб будет увеличиваться,
	 при прокручивании вниз - уменьшаться.


Для отображения полноразмерного листа нажмите кнопку ![](zoom100.png)
 «Отображение документа в масштабе 1:1».
