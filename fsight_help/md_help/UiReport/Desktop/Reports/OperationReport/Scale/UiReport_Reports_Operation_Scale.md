# Шкалы отчета


Шкалы отчета - правила оформления различных показателей на картах в
 регламентном отчете в зависимости от выбранных данных.


Шкалы отчета используются в регламентных отчетах для настройки атрибутов
 отображения показателей в режиме карты. Созданная шкала подключается к
 показателю карты и после вычисления отчета карта оформляется с учетом
 указанных в шкале правил. Каждая шкала задает набор числовых интервалов
 и в зависимости от типа сопоставляет им оформление.


В отчетах предусмотрены следующие виды шкал:


	- [Цветовая
	 шкала](UiReport_Reports_Operation_Scale_Colorful.htm). Сопоставляет интервалу значений цвет. Используется для
	 настройки атрибута «Заливка»
	 показателей карты;


	- [Штриховая
	 шкала](UiReport_Reports_Operation_Scale_Stroke.htm). Сопоставляет интервалу значений штриховку. Используется
	 для настройки атрибута «Штриховка»
	 показателей карты;


	- [Числовая
	 шкала](UiReport_Reports_Operation_Scale_Numeric.htm). Сопоставляет интервалу значений радиусы и высоты частей
	 диаграммы. Используется в круговых и столбиковых показателях для определения
	 радиуса и высоты частей диаграммы соответственно.


Для работы со шкалами отчета нажмите кнопку ![](../../../organizational_management/Toolbars/legend16.png) «Шкалы», расположенную в группе «Инструменты и панели» на вкладке
 «Вид» ленты инструментов. Откроется
 окно «Шкалы отчета»:


![](numeral_scale.png)


В открывшемся окне выберите шкалу отчета из списка доступных шкал или
 добавьте новые шкалы.


Для добавления шкал нажмите кнопку «Добавить».
 Появится раскрывающийся список, содержащий типы шкал.


Для удаления выбранной шкалы нажмите кнопку «Удалить».
