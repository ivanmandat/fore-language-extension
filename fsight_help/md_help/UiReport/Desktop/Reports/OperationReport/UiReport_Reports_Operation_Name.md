# Добавление именованного диапазона


Именованные диапазоны используются для быстрого поиска необходимого
 диапазона ячеек, а также при создании формул.


Для создания именованного диапазона нажмите кнопку ![](named_ranges.png)
 «Именованные диапазоны», расположенную
 в группе «Именованные диапазоны»
 на вкладке «Формулы» ленты инструментов.
 Появится окно:


![](main%20window%20name.png)


## Работа с именованными диапазонами


При работе с именованными диапазонами доступны следующие операции:


[![](../../../opened.gif)![](../../../closed.gif)Добавление именованного
 диапазона](javascript:TextPopup(this))


	Для добавления нового именованного диапазона:


		- В окне «Именованные диапазоны»
		 установите курсор в поле ввода «Диапазон»
		 и выделите необходимый диапазон ячеек.


		- Введите имя диапазона в поле ввода «Имя».
		 Имя диапазона может состоять из букв латинского алфавита, цифр
		 и символа подчеркивания.


	Примечание.
	 В качестве имени диапазона не могут быть использованы комбинации букв
	 и цифр в форматах A1 и R1C1, так как они воспринимаются как [стили
	 ссылок](../../organizational_management/Function/UiReport_Function.htm#reference_styles). При указании имени диапазона в одном из форматов кнопка
	 «Добавить» становится недоступна.


		- Нажмите кнопку «Добавить».


	Диапазон будет добавлен в список именованных диапазонов.


[![](../../../opened.gif)![](../../../closed.gif)Удаление именованного
 диапазона](javascript:TextPopup(this))


	Для удаления выбранного диапазона нажмите кнопку «Удалить»
	 в окне «Именованные диапазоны».


В процессе работы с отчётом список именованных диапазонов отображается
 в виде комбинированного списка в строке адреса:


![](list%20diapason.png)


При выборе одного из имен диапазонов в списке происходит выделение данного
 диапазона другим цветом на листе отчёта.
