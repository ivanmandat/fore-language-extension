# Просмотр готового отчёта


Закончив оформление отчёта для придания ему требуемого вида, просмотрите
 готовый отчёт. Для этого нажмите кнопку ![](Page_Layout/View_report.png)«Просмотреть
 отчет» вкладки «Главная»
 ленты инструментов. Отчёт будет открыт в [режиме
 просмотра](../../organizational_management/Starting.htm#open).


Если [автообновление
 отчёта](UiReport_Reports_Operation_Execute.htm#upgrade) включено, то при открытии отчёта отобразится диалог «[Период автообновления](UiReport_Reports_Operation_Execute.htm#auto_period)», в котором
 нужно указать время обновления.
