# Отмена и повтор операции


Для отмены последней операции в отчёте:


	- нажмите кнопку ![](../organizational_management/Toolbars/undo16.png)
	 «Отменить» на панели быстрого
	 доступа ленты инструментов;


	- нажмите сочетание клавиш CTRL+Z.


Для повтора последней отмененной операции:


	- нажмите кнопку ![](../organizational_management/Toolbars/redo16.png)
	 «Вернуть» на панели быстрого
	 доступа ленты инструментов;


	- нажмите сочетание клавиш CTRL+Y.


Примечание.
 Если в настольном приложении на панели быстрого доступа ленты инструментов
 не отображаются кнопки ![](../organizational_management/Toolbars/undo16.png)
  «Отменить» и ![](../organizational_management/Toolbars/redo16.png) «Вернуть»,
 то выполните [настройку
 панели](GetStarted.chm::/Interface/Interface_Description.htm#quick_access_toolbar): добавьте команды «Повторить»
 и «Отменить» из категории «Главная».
