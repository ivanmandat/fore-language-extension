# Удаление листа


Для удаления листа:


	- выполните команду «Удалить
	 > Удалить лист» в группе «Ячейка»
	 на вкладке «Главная» ленты
	 инструментов;


	- выполните команду «Удалить
	 лист» контекстного меню вкладки выбранного листа.


После выполнении одного из действий на экране появится запрос на подтверждение
 процедуры удаления.


При удалении листа осуществляется автоматическая корректировка формул,
 ссылающихся на удаленный лист. Например, формула «=B0+1+"Лист2"!A0»
 после удаления листа «Лист2» принимает
 вид: «=B0+1+#ССЫЛКА!A0».
