# Параметры 3D-сцены


Для настройки параметров сцены выполните команду контекстного меню «Параметры 3D-сцены». После выполнения
 действия будет открыт диалог:


![](3DParameters.png)


Данное окно в зависимости от наличия многомерных объектов на сцене может
 содержать следующие вкладки:


	- [Сцена](UiReport_Objects_3DParametersScena.htm).
	 Определяет общие параметры сцены: настройка внешнего вида сцены, координатных
	 плоскостей и осей;


	- [Поверхности](UiReport_Objects_3DParametersSurface.htm).
	 Определяет настройки для поверхностей трехмерной сцены;


	- [Плоскости
	 уровня](UiReport_Objects_3DParametersLevels.htm). Определяет настройки для плоскостей уровня;


	- [Сферы](UiReport_Objects_3DParametersSpheres.htm).
	 Определяет настройки сфер;


	- [Группы](UiReport_Objects_3DParametersGroups.htm).
	 Определяет настройки групп именованных сфер;


	- [Замкнутые
	 поверхности](UiReport_Objects_3DParametersClosedSurface.htm). Определяет настройки для замкнутых поверхностей;


	- [Текстуры](UiReport_Objects_3DParametersTexture.htm).
	 Список всех текстур, загруженных в сцену.


## Образец трехмерной сцены


На всех вкладках окна «Параметры 3D-сцены»
 слева расположен образец трехмерной сцены. Редактирование любых параметров
 приводит к изменению вида образца.


При настройке 3D-сцены доступны следующие операции над образцом сцены:


[![](../../opened.gif)![](../../closed.gif)Поворот образца
 сцены](javascript:TextPopup(this))


	Определите необходимый поворот образца сцены с помощью перемещения
	 мыши.


[![](../../opened.gif)![](../../closed.gif)Изменение масштаба
 образца сцены](javascript:TextPopup(this))


	С помощью перемещения мыши определите необходимый масштаб просмотра
	 образца сцены.


[![](../../opened.gif)![](../../closed.gif)Изменение масштаба](javascript:TextPopup(this))


	Для увеличения масштаба образца сцены нажмите управляющую кнопку
	 ![](zoom%20in.gif). При однократном нажатии кнопки масштаб увеличивается
	 на 10%.


	Для уменьшения масштаба образца сцены нажмите управляющую кнопку
	 ![](zoom%20out.gif). При однократном нажатии кнопки масштаб уменьшается
	 на 10%.


[![](../../opened.gif)![](../../closed.gif)Поворот сцены](javascript:TextPopup(this))


	Для поворота образца сцены вокруг вертикальной оси влево нажмите
	 управляющую кнопку ![](rotate%201.gif). При
	 нажатии и удерживании кнопки происходит непрерывный поворот сцены
	 влево вокруг вертикальной оси.


	Для поворота образца сцены вокруг вертикальной оси вправо нажмите
	 управляющую кнопку ![](rotate%202.gif). При
	 нажатии и удерживании кнопки происходит непрерывный поворот сцены
	 вправо вокруг вертикальной оси.


	Для поворота образца сцены вокруг горизонтальной оси вниз нажмите
	 управляющую кнопку ![](rotate%203.gif). При
	 нажатии и удерживании кнопки происходит непрерывный поворот сцены
	 вниз вокруг горизонтальной оси.


	Для поворота образца сцены вокруг горизонтальной оси вверх нажмите
	 управляющую кнопку ![](rotate%204.gif). При
	 нажатии и удерживании кнопки происходит непрерывный поворот сцены
	 вверх вокруг горизонтальной оси.


[![](../../opened.gif)![](../../closed.gif)Освещение сцены](javascript:TextPopup(this))


	Для включения/отключения освещения сцены нажмите управляющую кнопку
	 ![](light.gif). При нажатой кнопке освещение на сцене включается,
	 при повторном нажатии - выключается.
