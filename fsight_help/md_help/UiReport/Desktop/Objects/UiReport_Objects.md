# Вставка иллюстраций, фигур и текстовых
 блоков


Кроме размещения на [листах отчета](../Table/UiReport_Table.htm)
 областей данных различных типов, инструмент «Отчеты»
 поддерживает возможность добавления различных объектов.


Объекты могут быть:


	- статичными. Например, рисунок;


	- интерактивными. Например, спидометр.


Для управления интерактивными объектами их необходимо активировать,
 то есть выполнить определенное действие, например, дважды щелкнуть по
 объекту или навести на него указатель мыши.


Инструмент «Отчеты» поддерживает
 размещение следующих типов объектов:


	- [Рисунок](Picture/UiReport_Objects_Picture.htm).
	 Растровые и векторные изображения форматов BMP, GIF, PNG, JPG и т.д.;


	- [Фигуры](AutoObjects/UiReport_AutoObject.htm).
	 Предустановленный набор фигур: линии, многоугольники, фигурные стрелки,
	 выноски и т.д.;


	- [Спидометр](UiReport_Objects_Spedometer.htm).
	 Представляет собой индикатор для отображения каких-либо данных на
	 числовой панели в виде спидометра;


	- [3D-сцена](UiReport_Objects_3D.htm).
	 Отображение данных в виде плоскостей, поверхностей, замкнутых поверхностей,
	 сфер и т.д. на трехмерной сцене;


	- [Текстовый
	 блок](UiReport_Objects_formatted_text.htm). Блок форматированного текста для оформления сопроводительного
	 текста к отчету.


Для всех объектов существуют как [стандартные
 свойства](UiReport_Objects_StandartProperty.htm), так и специальные, зависящие от типа объекта.
