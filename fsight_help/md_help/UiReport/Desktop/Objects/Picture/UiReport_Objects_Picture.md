# Рисунок


	Инструмент «Отчеты» поддерживает
	 вставку изображений из файлов следующих форматов:


		- точечный рисунок Windows (*.bmp);


		- формат JPEG (*.jpg, *.jpeg , *.jfif);


		- формат GIF (*.gif);


		- формат PNG (*.png);


		- метафайл Windows (*.wmf);


		- расширенный метафайл Windows (*.emf).


## Добавление рисунка


	Для добавления рисунка на лист регламентного отчета выделите диапазон
	 ячеек, в который необходимо вписать рисунок, и:


		- выполните команду «Иллюстрации
		 > Рисунок», расположенную в группе «Иллюстрации»
		 на вкладке «Вставка» ленты
		 инструментов;


		- нажмите сочетание клавиш CTRL+ALT+P.


	После выполнения одного из действий будет открыто окно «Загрузка
	 картинки из файла». Укажите в нем имя и тип загружаемого файла.
	 Нажмите кнопку «Открыть» для
	 добавления выбранного рисунка на лист отчета.


### Добавление рисунка при копировании
	 из внешних источников


	Инструмент «Отчёты» поддерживает
	 вставку объектов, скопированных из внешних источников:


		- Microsoft Excel.
		 Доступны копирование и вставка диаграмм, фигур, текстовых блоков,
		 формул, изображений, объектов WordArt и SmartArt;


		- LibreOffice Calc.
		 Доступны копирование и вставка диаграмм, фигур, текстовых блоков,
		 текстовых эффектов, формул, изображений;


		- OpenOffice Calc.
		 Доступны копирование и вставка фигур, текстовых блоков, изображений;


		- Google Sheets. Доступны
		 копирование и вставка диаграмм, изображений;


		- МойОфис Таблица.
		 Доступны копирование и вставка изображений. Доступно только из
		 веб-приложения продукта «МойОфис»;


		- Р7-Офис. Доступны
		 копирование и вставка диаграмм, фигур, изображений, объектов TextArt.


	Все объекты вставляются в виде PNG-изображений.


	[![](../../../opened.gif)![](../../../closed.gif)Добавление
	 объекта из внешнего источника](javascript:TextPopup(this))


		Для добавления объекта из внешнего источника на лист регламентного
		 отчёта:


			- Выделите ячейку или диапазон ячеек на листе.


			- Скопируйте объект из внешнего источника.


			- Вставьте скопированный объект одним из способов:


				- нажмите сочетание клавиш CTRL+V;


				- нажмите сочетание клавиш SHIFT+INSERT;


				- выполните команду «Вставить»
				 в группе «Буфер обмена»
				 на вкладке «Главная»
				 ленты инструментов;


				- выполните команду «Вставить»
				 в контекстном меню ячейки или диапазона ячеек.


		В результате выполненных действий на лист регламентного отчёта
		 будет вставлен объект из внешнего источника в виде PNG-изображения.
		 Левый верхний угол изображения совпадает с левым верхним углом
		 выделенной ячейки или выделенного диапазона.


## Работа с рисунком


	Работа с рисунком осуществляется с помощью:


		- вкладок «Вставка»,
		 «Объект.Рисунок» и «Объект.Формат» ленты инструментов;


		- контекстного меню объекта.


	При работе с рисунком кроме [стандартных
	 операций](../UiReport_Objects_Operation.htm) доступно:


	[![](../../../opened.gif)![](../../../closed.gif)Подгонка
	 размера объекта под размер изображения](javascript:TextPopup(this))


		Для автоматической подгонки размера объекта под размер изображения:


			- выполните команду контекстного меню «Исходный
			 размер» рисунка;


			- нажмите на кнопку «Исходный
			 размер», расположенную в группе «Масштаб»
			 на вкладке «Объект. Рисунок»
			 ленты инструментов.


	[![](../../../opened.gif)![](../../../closed.gif)Редактирование
	 изображения](javascript:TextPopup(this))


		Для редактирования изображения выполните одно из действий:


			- выполните команду контекстного меню «Редактировать»
			 рисунка;


			- нажмите кнопку, расположенную
			 в группе «Фиксированные области»
			 на вкладке «Объект. Рисунок»
			 ленты инструментов.


		Откроется окно «Редактор изображения»,
		 аналогичное вкладке «[Заливка](UiNav.chm::/GUI/Format/UiReport_Table_Attribute_View.htm)»
		 (при выборе типа заливки «С изображением»)
		 окна «Формат ячейки».


		Примечание.
		 При настройке в окне «Редактор
		 изображения» фиксированных областей для векторного изображения,
		 оно преобразуется в растровое.


	[![](../../../opened.gif)![](../../../closed.gif)Масштабирование
	 изображения](javascript:TextPopup(this))


		Для включения масштабирования изображения в объекте:


			- установите флажок «Разрешить
			 масштабирование», расположенный в группе «Масштаб»
			 на вкладке «Объект. Рисунок»
			 ленты инструментов;


			- вызовите контекстное меню рисунка и установите флажок
			 «Растянуть». Размер
			 изображения остается исходным независимо от изменения размеров
			 объекта.


		Для пропорционального масштабирования изображения в объекте
		 включите масштабирование, как описано выше, и установите флажок:


			- «Пропорционально»
			 в контекстном меню рисунка;


			- «Сохранять пропорции»
			 в группе «Масштаб»
			 на вкладке «Объект. Рисунок»
			 ленты инструментов.


		Для отключения масштабирования изображения в объекте:


			- снимите флажок «Разрешить
			 масштабирование», расположенный в группе «Масштаб»
			 на вкладке «Объект. Рисунок»
			 ленты инструментов;


			- вызовите контекстное меню рисунка и установите флажок
			 «Без масштабирования».


		Размер изображения остается исходным независимо от изменения
		 размеров объекта.


	[![](../../../opened.gif)![](../../../closed.gif)Замена изображения](javascript:TextPopup(this))


		Для замены изображения:


			- выполните команду контекстного меню «Редактировать»
			 рисунка. В открывшемся окне нажмите кнопку «Загрузить»;


			- выполните команду контекстного меню «Загрузить
			 из файла» рисунка;


			- выполните команду «Загрузить
			 из файла», расположенную в группе «Изображение»
			 на вкладке «Объект. Рисунок».


		После выполнения одного из действий будет открыто стандартное
		 окно выбора файла.


	[![](../../../opened.gif)![](../../../closed.gif)Сохранение
	 изображения во внешний файл](javascript:TextPopup(this))


		Для сохранения изображения во внешний файл:


			- выполните команду контекстного меню «Редактировать»
			 рисунка. В открывшемся окне нажмите кнопку «Сохранить»;


			- выполните команду контекстного меню «Выгрузить
			 в файл» рисунка;


			- выполните команду «Сохранить
			 в файл», расположенную в группе «Изображение»
			 на вкладке «Объект. Рисунок».


		После выполнения одного из действий будет открыто стандартное
		 окно выбора папки для сохранения файла.
