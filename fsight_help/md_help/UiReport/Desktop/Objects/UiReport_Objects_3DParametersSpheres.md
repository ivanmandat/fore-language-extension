# Сферы


Для настройки параметров сферы перейдите на вкладку «Сферы»
 окна «[Параметры
 3D-сцены](UiReport_Objects_3DParameters.htm)»:


![](Spheres.png)


Примечание.
 Вкладка присутствует в окне «[Параметры
 3D-сцены](UiReport_Objects_3DParameters.htm)», если в [источниках
 данных](UiReport_Objects_3DSource.htm) был создан ряд типа «Сферы».


Установленные флажки с названиями сфер включают видимость сферы на сцене.
 По умолчанию все имеющиеся поверхности отображаются на сцене.


При выделении сферы в списке происходит ее выделение на сцене, если
 для нее была включена видимость на сцене.


Для настройки выделенной в списке сферы задайте следующие параметры:


[![](../../opened.gif)![](../../closed.gif)Цвет](javascript:TextPopup(this))


	Задайте цвет заливки, выбрав его из раскрывающейся палитры цветов.


[![](../../opened.gif)![](../../closed.gif)Прозрачность](javascript:TextPopup(this))


	Задайте прозрачность заливки в процентах.
