# Работа с объектами на листе отчета


При работе с объектами [любого типа](UiReport_Objects.htm)
 на листе отчета доступны следующие операции:


[![](../../opened.gif)![](../../closed.gif)Добавление
 объекта](javascript:TextPopup(this))


	Для добавления объекта на лист регламентного отчета выберите необходимый
	 [тип объекта](UiReport_Objects.htm) на вкладке «Вставка»
	 ленты инструментов.


[![](../../opened.gif)![](../../closed.gif)Выбор
 нескольких объектов](javascript:TextPopup(this))


	Для выделения нескольких объектов удерживайте нажатой клавишу CTRL.


[![](../../opened.gif)![](../../closed.gif)Создание группы
 объектов](javascript:TextPopup(this))


	Для объединения объектов в группу выделите несколько объектов, последовательно
	 щелкая по ним с зажатой клавишей CTRL, и выполните:


		- команду «Группировать >
		 Группировать», расположенную в группе «Упорядочить»
		 на вкладке «Объект. Формат»
		 ленты инструментов;


		- команду контекстного меню «Группировать
		 > Группировать» для выделенных объектов.


	После выполнения одного из действий выбранные объекты будут сгруппированы
	 в группу.


При работе с группой объектов доступны операции:


[![](../../opened.gif)![](../../closed.gif)Выравнивание
 объектов в группе](javascript:TextPopup(this))


	Для [выравнивания/распределения
	 выделенных объектов](../Table/Work/UiReport_Table_Work_Arrange.htm):


		- выполните команду «Выровнять»,
		 расположенную в группе «Упорядочить»
		 на вкладке «Объект. Формат»
		 ленты инструментов;


		- выполните команду контекстного меню «Выровнять/Распределить»
		 группы объектов.


[![](../../opened.gif)![](../../closed.gif)Разгруппировка
 объектов](javascript:TextPopup(this))


	Для разгруппировки выделенной группы объектов выполните:


		- команду «Группировать >
		 Разгруппировать», расположенную в группе «Упорядочить»
		 на вкладке «Объект. Формат»
		 ленты инструментов;


		- команду «Группировать >
		 Разгруппировать» в контекстном меню для выделенных объектов.


	После выполнения одного из действий выбранные объекты будут разгруппированы
	 в отдельные объекты.


При работе с объектами [любого типа](UiReport_Objects.htm)
 и с группой объектов доступны операции:


[![](../../opened.gif)![](../../closed.gif)Копирование,
 вставка и вырезание объекта](javascript:TextPopup(this))


	При работе с объектами доступна стандартная [работа
	 с буфером обмена](UiNav.chm::/02_Navigator/UiNav_ObjectsOperations.htm).


	В инструменте «Отчеты» реализована возможность копирования и вставки
	 одновременно нескольких выделенных объектов на лист отчета. Исключением
	 являются объекты, для которых установлен режим активации «Активировать
	 при открытии отчета», для них данная функция недоступна.


	Относительное позиционирование объектов в копируемой группе при
	 вставке сохраняется. Группа скопированных объектов вставляется в позицию
	 выделенного диапазона ячеек (позиция левого верхнего угла вставляемой
	 области совпадает с левым верхним углом активной ячейки).


[![](../../opened.gif)![](../../closed.gif)Поворот объекта](javascript:TextPopup(this))


	Для поворота выделенного объекта:


		- выполните команду «Отразить/повернуть»,
		 расположенную в группе «Упорядочить»
		 на вкладке «Объект. Формат»
		 ленты инструментов, и выберите необходимый вариант поворота объекта:


			- Повернуть вправо на
			 90o. К объекту будет
			 применен поворот на 90o по часовой
			 стрелке;


			- Повернуть влево на
			 90o. К объекту будет
			 применен поворот на 90o против
			 часовой стрелки.


	Для задания другого варианта выберите
	 «Другие параметры поворота».
	 Будет открыто окно «[Свойства объекта](UiReport_Objects_StandartProperty.htm)».
	 Задайте угол поворота в соответствующем поле;


		- выполните команду контекстного меню «Свойства
		 объекта». В открывшемся [окне](UiReport_Objects_StandartProperty.htm)
		 задайте угол поворота в соответствующем поле;


		- вручную (не доступно при выделении группы объектов).


[![](../../opened.gif)![](../../closed.gif)Отражение объекта](javascript:TextPopup(this))


	Для отражения выделенного объекта выполните команду «Отразить/повернуть»,
	 расположенную в группе «Упорядочить»
	 на вкладке «Объект. Формат»
	 ленты инструментов, и выберите способ отражения:


		- Отразить сверху вниз;


		- Отразить слева направо.


[![](../../opened.gif)![](../../closed.gif)Настройка свойств
 объекта](javascript:TextPopup(this))


	Для объекта любого типа можно настроить следующие свойства:


		- идентификатор объекта;


		- всплывающая подсказка;


		- способ перемещения объекта;


		- способ активации объекта;


		- вывод объекта на печать;


		- прозрачность объекта;


		- размеры объекта.


	Подробное описание настройки свойств объекта приведено в разделе
	 «[Стандартные
	 свойства объектов](UiReport_Objects_StandartProperty.htm)».


[![](../../opened.gif)![](../../closed.gif)Перемещение
 объекта/группы объектов строго по горизонтали или по вертикали](javascript:TextPopup(this))


	Для перемещения объекта/группы объектов строго по горизонтали или
	 по вертикали удерживайте нажатой клавишу SHIFT.


Изменение положения объекта:


[![](../../opened.gif)![](../../closed.gif)Настройка
 порядка расположения объекта](javascript:TextPopup(this))


	Для настройки порядка размещения объекта на листе отчета:


		- выполните команду контекстного меню «Порядок»
		 объекта и выберите вариант размещения;


		- выберите вариант размещения в группе «Упорядочить»
		 на вкладке «Объект. Формат»
		 ленты инструментов.


	Для настройки порядка размещения объектов на листе отчета доступные
	 следующие варианты:


		- Переместить вперед.
		 Меняет порядок расположения объекта на листе отчета, перемещая
		 его на один план вперед;


		- Переместить назад.
		 Меняет порядок расположения объекта на листе отчета, перемещая
		 его на один план назад;


		- На передний план.
		 Меняет порядок расположения объекта на листе отчета, перемещая
		 его на передний план;


		- На задний план.
		 Меняет порядок расположения объекта на листе отчета, перемещая
		 его на задний план.


[![](../../opened.gif)![](../../closed.gif)Размещение
 объекта в заданном диапазоне ячеек](javascript:TextPopup(this))


	Для настройки размещения объекта в заданном диапазоне ячее:


		- выполните команду контекстного меню «Положение
		 объекта» объекта и выберите способ размещения;


		- нажмите кнопку «Переместить
		 объект», расположенную в группе «Размещение»
		 на вкладке «Объект. Формат»
		 ленты инструментов, и выберите способ размещения.


	Для настройки размещения объекта на листе отчета доступные следующие
	 варианты:


		- Вписать в выделенный диапазон.
		 Помещает выбранный объект в выделенном диапазоне ячеек;


		- Выбрать диапазон и вписать.
		 Открывает диалог, в котором можно задать диапазон ячеек текущего
		 отчета.
