# Сцена


Для настройки общих параметров 3D-сцены используйте вкладку «Сцена»:


![](3DParameters.png)


Задайте на вкладке следующие параметры:


[![](../../opened.gif)![](../../closed.gif)Заголовок](javascript:TextPopup(this))


	Установленный флажок «Заголовок»
	 включает отображение заголовка сцены.


	Введите необходимый текст заголовка сцены в поле ввода, расположенное
	 под флажком.


	Для настройки параметров заголовка сцены нажмите кнопку «Формат».
	 Будет открыт диалог «Формат заголовка»,
	 аналогичный диалогу «[Формат надписей](UiReport_Objects_3DParametersScena3.htm)».


	По умолчанию флажок «Заголовок»
	 снят, поэтому поле ввода заголовка и кнопка «Формат»
	 недоступны.


[![](../../opened.gif)![](../../closed.gif)Координатные
 плоскости](javascript:TextPopup(this))


	Группа параметров «Координатные
	 плоскости» отвечает за настройку отображения координатных плоскостей.


	При установленных флажках с наименованиями плоскостей все координатные
	 плоскости отображаются на 3D-сцене. По умолчанию флажки установлены.


	Для настройки соответствующей координатной плоскости нажмите кнопку
	 «Настроить». Будет открыт
	 диалог «[Свойства
	 координатной плоскости](UiReport_Objects_3DParametersScena1.htm)».


[![](../../opened.gif)![](../../closed.gif)Координатные
 оси](javascript:TextPopup(this))


	Группа параметров «Координатные
	 оси» отвечает за настройку параметров отображения координатных
	 осей:


		- Настроить ось. При
		 нажатии кнопки будет открыт [диалог
		 настройки формата](UiReport_Objects_3DParametersScena2.htm) соответствующей оси;


		- Формат выделенных делений.
		 При нажатии кнопки будет открыт диалог для настройки [параметров
		 надписей](UiReport_Objects_3DParametersScena3.htm) делений, которые выделяются на осях при выделении
		 объектов сцены;


		- Число делений. Поле
		 ввода определяет количество делений на осях: минимальное значение
		 «1», максимальное - не ограничено.


[![](../../opened.gif)![](../../closed.gif)Внешний вид](javascript:TextPopup(this))


	Группа параметров «Внешний вид»
	 отвечает за настройку внешнего вида сцены:


		- Тип проекции. Установите
		 соответствующий переключатель типа проекции трехмерной сцены:
		 перспективная или ортогональная;


		- Цвет фона. Укажите
		 цвет фона сцены, выбрав его из раскрывающейся палитры цветов.
