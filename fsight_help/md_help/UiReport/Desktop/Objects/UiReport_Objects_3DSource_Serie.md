# Свойства ряда данных


При добавлении/редактировании источников данных 3D-сцены открывается
 окно «Свойства ряда данных». Вид
 окна зависит от выбранного типа ряда.


Для рядов всех типов поля, содержащие значения по осям, являются обязательными
 для заполнения. Все значения параметров задаются в виде диапазонов.


Примечание.
 При открытии окна на редактирование параметров смена типа ряда недоступна.


При работе с источниками данных 3D-сцены возможны следующие варианты
 выбора типов рядов:


[![](../../opened.gif)![](../../closed.gif)Сферы](javascript:TextPopup(this))


	Данный тип ряда позволяет создать на сцене группу точек (сфер):


	![](3D%20SourceSerie.png)


	Для настройки данного типа ряда задайте следующие параметры:


		- Значения по осям.
		 Определите положение точек в пространстве;


		- Значения радиусов.
		 Определите размеры радиусов для точек.


	Примечание.
	 На сцене возможно создание только одного ряда данного типа.


[![](../../opened.gif)![](../../closed.gif)Группа
 именованных сфер](javascript:TextPopup(this))


	Данный тип ряда позволяет создать на сцене группу точек (сфер),
	 соединенных стрелками:


	![](3D%20SourceSerie1.png)


	Для настройки данного типа ряда задайте следующие параметры:


		- Наименование ряда.
		 В данном поле укажите только одну ячейку, содержащую наименование
		 группы создаваемых сфер. Заданное имя будет отображено на вкладке
		 «[Группы](UiReport_Objects_3DParametersGroups.htm)»
		 в параметрах сцены;


		- Значения по осям.
		 Задайте диапазон ячеек с координатами точек в пространстве;


		- Значения радиусов.
		 Задайте диапазон ячеек с радиусами точек;


		- Наименования сфер.
		 Задайте диапазон ячеек с наименованиями для сфер.


[![](../../opened.gif)![](../../closed.gif)Поверхность](javascript:TextPopup(this))


	Данный тип ряда позволяет создавать на сцене поверхность:


	![](3D%20SourceSerie3.png)


	Для настройки данного типа ряда задайте следующие параметры:


		- Значения по осям.
		 Задайте диапазон ячеек с координатами точек в пространстве.


	Примечание.
	 Необходимо помнить, что значения по оси Z должны быть заданы для каждой
	 возможной пары значений X и Y. Например, рассматривая два значения
	 по оси X и два значения по оси Y необходимо задать четыре значения
	 по оси Z.


[![](../../opened.gif)![](../../closed.gif)Замкнутая
 поверхность](javascript:TextPopup(this))


	Данный тип ряда позволяет создать на сцене группу соединенных точек,
	 образуя замкнутую поверхность:


	![](3D%20SourceSerie4.png)


	Для настройки данного типа ряда задайте следующие параметры:


		- Наименование ряда.
		 Задайте ячейку, содержащую наименование создаваемой поверхности.
		 Заданное имя будет отображено на вкладке «[Замкнутые
		 поверхности](UiReport_Objects_3DParametersClosedSurface.htm)» в параметрах сцены;


		- Значения по осям.
		 Задайте диапазон ячеек с координатами точек в пространстве;


		- Значения радиусов.
		 Задайте диапазон ячеек с радиусами точек;


		- Наименования сфер.
		 Задайте диапазон ячеек с наименованиями для сфер.
