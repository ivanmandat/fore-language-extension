# Текстовый блок


Инструмент «Отчеты» позволяет
 добавлять на табличные листы блоки сопроводительного текста с произвольным
 оформлением. Такие блоки, как и любые другие объекты листа, поддерживают
 вставку в заданный диапазон ячеек и активацию.


Для добавления на лист отчета объекта «Текстовый
 блок»:


	- в указанный диапазон:


		- Задайте диапазон ячеек, в котором будет располагаться создаваемый
		 объект.


		- Нажмите кнопку «Текстовый
		 блок», расположенную в группе «Текст»
		 на вкладке «Вставка» ленты
		 инструментов.


После выполнения данных действий в заданный
 диапазон будет добавлен объект «Текстовый
 блок»;


	- без указания диапазона:


		- Выделите ячейку.


		- Нажмите кнопку «Текстовый
		 блок», расположенную в группе «Текст»
		 на вкладке «Вставка» ленты
		 инструментов.


После выполнения данных действий на лист
 будет добавлен объект «Текстовый блок»
 левый верхний угол блока будет совпадать с левым верхним углом выделенной
 ячейки.


После активации объекта доступны следующие операции:


[![](../../opened.gif)![](../../closed.gif)Ввод и редактирование
 текста](javascript:TextPopup(this))


	Для ввода или редактирования уже имеющегося текста в текстовом блоке
	 дважды щелкните по объекту. Появится поле ввода. В данном поле разместите
	 или отредактируйте текст. Для закрытия поля ввода щелкните за пределами
	 текстового блока.


[![](../../opened.gif)![](../../closed.gif)Форматирование
 текста](javascript:TextPopup(this))


	Для настройки параметров форматирования введенного в текстовый блок
	 текста:


		- выполните команду контекстного меню «Формат»
		 содержимого объекта;


		- нажмите сочетание клавиш CTRL+1.


	Откроется окно «Формат»,
	 содержащее вкладки «[Шрифт](UiReport_format.htm)»
	 и «[Отсутпы
	 и интервалы](UiReport_format_interval.htm)».


	Также для настройки форматирования текста объекта можно задать параметры
	 форматирования в группе «Шрифт»,
	 расположенной на вкладке «Главная»
	 ленты инструментов.


[![](../../opened.gif)![](../../closed.gif)Отмена и повтор
 действия](javascript:TextPopup(this))


	Для отмены последнего выполняемого действия над содержимым текстового
	 блока:


		- выполните команду контекстного меню «Отменить»;


		- нажмите сочетание клавиш CTRL+Z.


	Для возврата отмененного действия:


		- выполните команду контекстного меню «Повторить»;


		- нажмите сочетание клавиш CTRL+Y.


[![](../../opened.gif)![](../../closed.gif)Работа с буфером
 обмена](javascript:TextPopup(this))


	Для перемещения выделенного
	 содержимого текстового блока в буфер обмена:


		- нажмите кнопку «Вырезать»,
		 расположенную в группе «Буфер
		 обмена» на вкладке «Главная»
		 ленты инструментов;


		- выполните команду контекстного меню «Вырезать»
		 содержимого объекта;


		- нажмите сочетание клавиш CTRL+X.


	Содержимое объекта будет перемещено в буфер обмена.


	Для копирования выделенного
	 содержимого текстового блока в буфер обмена:


		- нажмите кнопку «Копировать»,
		 расположенную в группе «Буфер
		 обмена» на вкладке «Главная»
		 ленты инструментов;


		- выполните команду контекстного меню «Копировать»
		 содержимого объекта;


		- нажмите сочетание клавиш CTRL+C.


	Содержимое объекта будет скопировано в буфер обмена.


	Для вставки данных в текстовый
	 блок из буфера обмена:


		- нажмите кнопку «Вставить»,
		 расположенную в группе «Буфер
		 обмена» на вкладке «Главная»
		 ленты инструментов;


		- выполните команду контекстного меню «Вставить»;


		- нажмите сочетание клавиш CTRL+V.


	В текстовый блок будут вставлены данные из буфера обмена.


[![](../../opened.gif)![](../../closed.gif)Удаление](javascript:TextPopup(this))


	Для удаления выделенного содержимого текстового блока:


		- выполните команду контекстного меню «Удалить»;


		- нажмите клавишу DELETE.


	Выделенное содержимое объекта будет удалено.


[![](../../opened.gif)![](../../closed.gif)Выделение всего
 текста](javascript:TextPopup(this))


	Для выделения всего текста, содержащегося в текстовом блоке:


		- выполните команду контекстного меню «Выделить
		 все»;


		- нажмите сочетание клавиш CTRL+A.


	Весь текст объекта будет выделен.


[![](../../opened.gif)![](../../closed.gif)Выравнивание
 текста](javascript:TextPopup(this))


	Для выравнивания содержимого текстового блока выберите способ выравнивания
	 в группе «Выравнивание», расположенной
	 на вкладке «Главная» ленты
	 инструментов.


	Доступны способы выравнивания текста фигуры:


		- выравнивание текста по левому краю;


		- выравнивание текста по центру;


		- выравнивание текста по правому краю.


	Примечание.
	 Выравнивание распространяется на весь текст, содержащийся в объекте.
