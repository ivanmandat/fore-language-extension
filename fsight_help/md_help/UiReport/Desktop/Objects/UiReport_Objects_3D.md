# 3D-сцена


3D-сцена служит для представления наборов данных в виде различных многомерных
 объектов: простые и замкнутые поверхности, сферы (точки) и группы именованных
 сфер:


![](3D%20example.gif)


Для добавления 3D-сцены на лист отчета:


	- Задайте диапазон ячеек, в котором будет располагаться сцена.


	- Выполните команду «Другие >
	 Трехмерная сцена», расположенную
	 в группе «Диаграммы» на вкладке
	 «Вставка» ленты инструментов.


	- В открывшемся окне «[Источники
	 данных 3D-сцены](UiReport_Objects_3DSource.htm)» задайте исходные данные 3D-сцены.


## Работа с трехмерной сценой


При работе с 3D-сценой доступны следующие операции:


[![](../../opened.gif)![](../../closed.gif)Настройка параметров
 сцены](javascript:TextPopup(this))


	Для настройки параметров трехмерной сцены выполните команду контекстного
	 меню «[Параметры
	 3D-сцены](UiReport_Objects_3DParameters.htm)» сцены.


[![](../../opened.gif)![](../../closed.gif)Настройка источников
 данных](javascript:TextPopup(this))


	Для настройки источников данных трехмерной сцены выполните команду
	 контекстного меню «[Источники
	 данных 3D-сцены](UiReport_Objects_3DSource.htm)» сцены.


[![](../../opened.gif)![](../../closed.gif)Задание обработчика
 событий](javascript:TextPopup(this))


	Для задания обработчика событий, позволяющего выполнять определенные
	 действия при обращении к трехмерной сцене, выполните команду контекстного
	 меню «[Обработчик
	 событий](UiReport_Objects_3DEventHandler.htm)» сцены.


После активации 3D-сцены доступны следующие действия:


[![](../../opened.gif)![](../../closed.gif)Изменение
 ракурса просмотра](javascript:TextPopup(this))


	Для изменения ракурса просмотра 3D-сцены зажмите кнопку
	 мыши и с помощью перемещения мыши определите необходимый поворот сцены.


[![](../../opened.gif)![](../../closed.gif)Изменение
 масштаба](javascript:TextPopup(this))


	Для изменения масштаба 3D-сцены зажмите кнопку мыши и клавишу
	 SHIFT, после чего с помощью перемещения мыши определите необходимый
	 масштаб просмотра.


[![](../../opened.gif)![](../../closed.gif)Изменение положения
 плоскости уровня](javascript:TextPopup(this))


	Для изменения положения плоскости уровня выделите плоскость с помощью
	 мыши, зажмите кнопку мыши и клавишу CTRL, после чего с помощью
	 перемещения мыши определите смещение плоскости уровня вдоль оси, перпендикулярной
	 ей.
