# Источники данных 3D-сцены


Каждый объект 3D-сцены (поверхность, сферы) представляется рядом данных,
 для которого необходимо определить источник данных. В источнике по строкам
 размещаются точки ряда (или вершины многомерной фигуры), а по столбцам
 значения координат (X, Y, Z) и радиус точек.


После добавления на лист отчета [3D-сцены](UiReport_Objects_3D.htm)
 будет открыто окно:


![](3D%20Source.png)


Задайте следующие настройки источника данных 3D-сцены:


	- Лист с данными. В раскрывающемся
	 списке выберите лист, на котором расположены исходные данные. По умолчанию
	 выбран лист отчета, на котором была добавлена 3D-сцена;


	- Автоматически изменять диапазоны.
	 Установленный флажок позволяет изменять размеры указанных диапазонов:
	 при вставке строк/столбцов и при удалении строк/столбцов.


При работе с источниками данных 3D-сцены доступны следующие операции:


[![](../../opened.gif)![](../../closed.gif)Добавление ряда
 данных](javascript:TextPopup(this))


	Для добавления ряда данных:


		- нажмите кнопку «Добавить»;


		- нажмите на клавишу INSERT;


		- дважды щёлкните по пустой области в списке.


	После выполнения одного из действий будет открыто окно «[Свойства ряда данных](UiReport_Objects_3DSource_Serie.htm)».


[![](../../opened.gif)![](../../closed.gif)Редактирование
 ряда данных](javascript:TextPopup(this))


	Для редактирования выбранного ряда данных:


		- нажмите кнопку «Редактировать»;


		- дважды щелкните кнопкой мыши.


	После выполнения одного из действий будет открыто окно «[Свойства ряда данных](UiReport_Objects_3DSource_Serie.htm)», внесите в нем
	 требуемые изменения.


[![](../../opened.gif)![](../../closed.gif)Удаление ряда
 данных](javascript:TextPopup(this))


	Для удаления одного или нескольких выбранных рядов данных:


		- нажмите кнопку «Удалить»;


		- нажмите на клавишу DELETE.


	После выполнения одного из действий будет выдан диалог подтверждения
	 выполняемого действия.
