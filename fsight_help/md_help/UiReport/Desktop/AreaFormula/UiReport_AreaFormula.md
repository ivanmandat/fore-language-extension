# Область формул


Область формул - диапазон ячеек
 на листе отчета, значения которого вычисляются по единой формуле.


В отчете может быть несколько областей формул. Области формул не должны
 пересекаться.


## Операции над областью формул


При работе с областью формул доступны следующие операции:


[![](../../opened.gif)![](../../closed.gif)Создание
 области формул](javascript:TextPopup(this))


	Для создания новой области формул выделите ячейку листа отчета либо
	 диапазон ячеек и выполните команду «Другие
	 области > Область формул», расположенную в группе «Области данных» на вкладке «Вставка» ленты инструментов.


	Будет отображено окно «Параметры
	 области формул»:


	![](property%20domain.gif)


	Задайте параметры области формулы:


		- Идентификатор. Уникальная
		 последовательность букв латинского алфавита, цифр и знаков, которая
		 должна начинаться с буквы. Однозначно идентифицирует область
		 формул в пределах данного отчета. Каждой области формул по умолчанию
		 присваивается идентификатор, который отражается в данном поле.
		 Для смены идентификатора введите новое значение;


		- Лист. Из раскрывающегося
		 списка выберите лист, на котором будет размещаться область формул;


		- Диапазон ячеек.
		 Определите границы размещения области формул;


		- Базовая ячейка.
		 Задайте базовую ячейку. Базовая ячейка влияет на вектор расчета
		 и может находиться как в пределах области формул, так и за ее
		 пределами. По умолчанию в качестве базовой ячейки предлагается
		 верхняя левая угловая ячейка области формул;


		- Формула. Задайте
		 формулу расчета значений. Если в формуле используются относительные
		 ссылки на другие ячейки, то для каждой ячейки области формул формула
		 будет содержать смещенную относительно базовой ячейки ссылку;


		- Оформление. Задайте
		 параметры оформления области формул с помощью [стандартного компонента](uinav.chm::/GUI/Style_setup.htm)
		 настройки оформления.


[![](../../opened.gif)![](../../closed.gif)Создание
 копии области формул](javascript:TextPopup(this))


	Для создания копии области формул отметьте любую ячейку области
	 формул и выполните команду «Создать
	 копию» в контекстном меню.


	Будет предложено ввести диапазон ячеек, в который будет помещена
	 копия области формул.


[![](../../opened.gif)![](../../closed.gif)Редактирование
 области формул](javascript:TextPopup(this))


	Для редактирования параметров области формул выделите любую ячейку
	 области формул и выполните команду «Параметры
	 области формул» в контекстном меню.


	После выполнения команды появится окно «[Параметры области формул](#create)».


[![](../../opened.gif)![](../../closed.gif)Вычисление
 области формул](javascript:TextPopup(this))


	Для вычисления области формул выделите любую ячейку области формул
	 и выполните команду «Вычислить область
	 формул» в контекстном меню.


	Также можно настроить вычисление области формул в [автоматическом
	 режиме](../Reports/OperationReport/UiReport_Reports_Operation_Execute.htm#upgrade).


[![](../../opened.gif)![](../../closed.gif)Удаление
 области формул](javascript:TextPopup(this))


	Для удаления области формул отметьте любую ячейку области формул
	 и выполните команду «Удалить область
	 формул» в контекстном меню.


	После выполнения команды появится предупреждающее сообщение.
