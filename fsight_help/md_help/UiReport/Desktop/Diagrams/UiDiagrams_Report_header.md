# Редактирование заголовков на диаграмме


Для редактирования содержимого заголовка диаграммы/осей непосредственно
 на диаграмме перейдите в режим редактирования текста. Для этого дважды
 щелкните по заголовку диаграммы/оси. После этого весь текст редактируемого
 объекта будет выделен.


Примечание.
 Также для перехода редактирования заголовков в регламентном отчете для
 перехода щелкните на соответствующем поле на вкладке «[Заголовки](uidiagrams.chm::/Params_diagram/UiDiagrams_titles.htm)»
 в окне «Параметры диаграммы».


В режиме редактирования доступны следующие операции:


[![](../../opened.gif)![](../../closed.gif)Форматирование
 текста заголовков](javascript:TextPopup(this))


	Для настройки параметров форматирования текста заголовков выполните
	 команду контекстного меню «Формат»
	 заголовка. Откроется окно «Формат».
	 На вкладке «[Шрифт](UiDiagrams.chm::/Tuning_format/Format_Title/UiDiagrams_FormatTitle_type.htm)» определите
	 необходимые параметры форматирования.


[![](../../opened.gif)![](../../closed.gif)Отмена/повтор
 действия](javascript:TextPopup(this))


	Для отмены последнего действия выполните команду контекстного меню
	 «Отменить».


	Для возврата последнего отмененного действия выполните команду контекстного
	 меню «Повторить».


[![](../../opened.gif)![](../../closed.gif)Работа с буфером
 обмена](javascript:TextPopup(this))


	Для перемещения выделенного текста заголовка в буфер обмена выполните
	 команду контекстного меню «Вырезать».


	Для копирования выделенного текста заголовка в буфер обмена выполните
	 команду контекстного меню «Копировать».


	Для выставки данных из буфера обмена в заголовок выполните команду
	 контекстного меню «Вставить».


[![](../../opened.gif)![](../../closed.gif)Выделение содержимого
 заголовка](javascript:TextPopup(this))


	Для выделения всего содержимого выбранного заголовка выполните команду
	 контекстного меню «Выделить все».


В режиме редактирования для выделенного текста заголовка действуют все
 команды панели инструментов «Форматирование
 текста», кроме команд выравнивания, действие которых распространяется
 на весь текст целиком.
