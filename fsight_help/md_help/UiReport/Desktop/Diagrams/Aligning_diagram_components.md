# Настройка режима выравнивания


Для выравнивания компонентов диаграммы выполните последовательность
 действий:


	- Активируйте диаграмму.


	- Выполните команду контекстного меню «Режим
	 выравнивания» диаграммы и установите флажок.


	- Выделите компоненты диаграммы, которые необходимо выровнять,
	 последовательно щелкая по ним с зажатой клавишей CTRL.


	- Нажмите кнопку «Выровнять»,
	 расположенную в группе «Упорядочить»
	 на вкладке «Объект. Формат»
	 ленты инструментов, и выберите способ выравнивания из раскрывающегося
	 списка.


Примечание.
 Режимы выравнивания компонентов диаграммы аналогичны вариантам [выравнивания
 объектов](../Table/Work/UiReport_Table_Work_Arrange.htm), расположенным на листе отчета.
