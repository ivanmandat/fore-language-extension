# Настройка параметров реляционной
 области данных


При создании или редактировании [реляционной
 области данных](../UIReport_Relational_data_area.htm) будет открыто окно «Реляционная
 область данных»:


![](../Relation_Example_M1.png)


Для настройки свойств реляционной области данных задайте параметры на
 следующих вкладках:


	- [Параметры](UiReport_Relation_Param.htm).
	 Определите общие параметры для определения размещения, поведения и
	 оформления реляционной области данных;


	- [Фильтрация](Relation_Filtering.htm).
	 Фильтрация реляционной области данных позволяет отобразить только
	 те данные, которые соответствуют заданным условиям;


	- [Сортировка](Relation_Sorting.htm).
	 Сортировка дает возможность производить упорядочивание строк реляционной
	 области;


	- [Размещение](UiReport_Relation_Placing.htm).
	 Настройте шаблон для вывода одной записи из источника. Для этого задайте
	 поля из источника и порядок их вывода;


	- [Группировка](UiReport_Relation_Grouping.htm).
	 Задайте способ группировки данных;


	- [Итоги](UiReport_Relation_Totals.htm).
	 Определите настройки вывода итоговых значений для группы по определенному
	 полю.


Примечание.
 Наименование окна «Реляционная область
 данных» будет отличаться в зависимости от активной вкладки.
