# Реляционная область данных


Реляционная область данных отображает
 данные напрямую из реляционных объектов, таких как таблицы данных, запросы
 и представления.


При работе с реляционными областями данных доступны следующие основные
 возможности:


	- [фильтрация](Parameter/Relation_Filtering.htm) данных;


	- [сортировка](Parameter/Relation_Sorting.htm) данных;


	- произвольная настройка [размещения](Parameter/UiReport_Relation_Placing.htm);


	- [группировка](Parameter/UiReport_Relation_Grouping.htm)
	 данных;


	- расчет [итоговых
	 значений](Parameter/UiReport_Relation_Totals.htm).


Все настройки реляционной области данных выполняются с помощью мастера.


## Операции над реляционной областью данных


При работе в регламентном отчете доступны следующие операции над реляционной
 областью данных:


[![](../../opened.gif)![](../../closed.gif)Создание реляционной
 области данных](javascript:TextPopup(this))


	Для создания реляционной области данных:


		- перетащите реляционный объект из панели «Источники
		 и срезы данных» на лист отчета, используя механизм Drag&Drop;


		- отметьте диапазон ячеек и поместите в данный диапазон новую
		 область, используя механизм Drag&Drop;


		- выделите ячейку на листе отчета и выполните команду «Другие области > Реляционная область»,
		 расположенную в группе «Области
		 данных» на вкладке «Вставка»
		 ленты инструментов. В раскрывающемся списке выберите реляционный
		 источник данных: таблица, запрос или представление.


	При создании реляционной области данных определите её [параметры](Parameter/UiReport_Relation_Parameters.htm)
	 в окне «Реляционная область
	 данных».


	Примечание.
	 Для вызова окна выполните команду контекстного меню «Параметры реляционной области данных»
	 для реляционной области, созданной с помощью ленты инструментов.


	Для того чтобы настройки вступили в силу выполните команду «Вычислить реляционную область данных»
	 контекстного меню.


	Созданная реляционная область данных отображается аналогично [области данных](../AreaData/UiReport_AreaData.htm).


[![](../../opened.gif)![](../../closed.gif)Редактирование
 реляционной области данных](javascript:TextPopup(this))


	Для редактирования параметров реляционной области данных выполните
	 команду контекстного меню «Параметры
	 реляционной области данных» для любой ячейки реляционной области
	 данных.


	Будет открыто окно «[Реляционная область
	 данных](Parameter/UiReport_Relation_Parameters.htm)».
