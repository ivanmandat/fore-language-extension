# Настройка набора цветов


Если для атрибутов показателей ([картографического](UiMaps_Report_property_1_atr_1.htm),
 [столбикового](UiMaps_Report_property_1_atr_3.htm), [кругового](UiMaps_Report_property_1_atr_2.htm))
 не используется имеющаяся в отчете [шкала](../Reports/OperationReport/Scale/UiReport_Reports_Operation_Scale.htm)
 (цветовая или штриховая), то будет использоваться набор цветов. Для его
 настройки щёлкните по значению атрибута «Набор
 цветов» в области «Атрибуты»
 окна «[Свойства
 карты](UiMaps_Report_property.htm)». Откроется окно «Редактирование
 цветов показателя»:


![](UiMaps%20Edit%20Color.gif)


По умолчанию в качестве набора цветов определен один цвет - «белый».


При работе в окне «Редактирование
 цветов показателя» доступны следующие операции:


[![](../../opened.gif)![](../../closed.gif)Добавление
 цвета](javascript:TextPopup(this))


	Для добавления цвета показателя в набор нажмите кнопку «Добавить».
	 В открывшемся окне «Настройка цвета»
	 выберите цвет для набора.


[![](../../opened.gif)![](../../closed.gif)Редактирование
 цвета](javascript:TextPopup(this))


	Для изменения цвета показателя нажмите кнопку «Редактировать».
	 В открывшемся окне «Настройка цвета»
	 выберите новый цвет для набора.


[![](../../opened.gif)![](../../closed.gif)Удаление цвета](javascript:TextPopup(this))


	Для удаления выделенного цвета показателя из набора нажмите на кнопку
	 «Удалить».


Также настройку набора цветов можно произвести в области «Атрибуты»
 окна «[Свойства
 карты](UiMaps_Report_property.htm)»:


[![](../../opened.gif)![](../../closed.gif)Редактирование
 набора цветов](javascript:TextPopup(this))


	Для редактирования набора цветов щёлкните по гиперссылке «Редактировать» в области атрибута
	 «Набор цветов». Откроется
	 окно «Редактирование цветов
	 показателя».


[![](../../opened.gif)![](../../closed.gif)Копирование
 набора цветов](javascript:TextPopup(this))


	Для копирования набора цветов из атрибутов другого показателя
	 щёлкните по гиперссылке «Скопировать
	 из» в области атрибута «Набор
	 цветов». Откроется окно «Выбор
	 показателя»:


	![](UiMaps%20Edit%20Color%20palette%20select.gif)


	Укажите в данном окне показатель, из которого будет скопирован набор
	 цветов.
