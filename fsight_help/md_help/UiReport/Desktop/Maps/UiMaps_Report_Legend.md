# Расширенная настройка легенды


Для сопоставления и отображения значений показателей и атрибутов на
 карте используется легенда карты.


Примечание.
 При изменении размера легенды происходит ее автоматическое масштабирование
 с учетом количества заданных колонок, размер шрифта не будет превышать
 установленного на вкладке «[Оформление](UiMaps_Report_Legend3.htm)».


Для карты можно настроить внутреннюю
 и внешнюю легенды. Отличие внутренней
 легенды от внешней заключается в том, что внутренняя легенда может располагаться
 только внутри области карты, тогда как внешнюю можно вставить в любое
 место отчета.


[![](../../opened.gif)![](../../closed.gif)Отображение
 внутренней легенды](javascript:TextPopup(this))


	Для отображения внутренней легенды:


		- установите флажок «Отображать
		 легенду» на вкладке «[Параметры](UiMaps_Report_property_param.htm)»
		 диалога «[Свойства
		 карты](UiMaps_Report_property.htm)»;


		- нажмите кнопку ![](../organizational_management/Toolbars/legend16.png)
		 «Легенда», расположенную
		 в группе «Вид» на вкладке
		 «Объект. Карта» ленты
		 инструментов, и выберите вариант расположения легенды в раскрывающемся
		 меню кнопки.


	Для скрытия внутренней легенды:


		- снимите флажок «Отображать
		 легенду» на вкладке «[Параметры](UiMaps_Report_property_param.htm)»
		 диалога «[Свойства
		 карты](UiMaps_Report_property.htm)»;


		- нажмите кнопку ![](../organizational_management/Toolbars/legend16.png)
		 «Легенда», расположенную
		 в группе «Вид» на вкладке
		 «Объект. Карта» ленты
		 инструментов, и выберите вариант ![](legendNo16.png) в раскрывающемся
		 меню кнопки.


[![](../../opened.gif)![](../../closed.gif)Отображение
 внешней легенды](javascript:TextPopup(this))


	Для отображения внешней легенды нажмите кнопку ![](../organizational_management/Toolbars/legend16.png)
	 «Легенда», расположенную в
	 группе «Карты» на вкладке
	 «Вставка» ленты инструментов.
	 Будет открыто окно «Легенда карты»
	 для настройки легенды выбранной карты.


	Для произвольного размещения легенды нажмите кнопку «Переместить
	 объект», расположенную в группе «Размещение»
	 на вкладке ленты «Объект. Формат».


Для настройки параметров:


	- внутренней легенды выполните
	 команду контекстного меню «Параметры
	 легенды» карты;


	- внешней легенды выполните
	 команду контекстного меню «Параметры
	 легенды» легенды.


Будет открыто окно «Легенда карты».


Примечание.
 При создании внешней легенды данное окно открывается автоматически.
