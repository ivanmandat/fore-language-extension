# Настройка показателей


Показатели используются для отображения информации на карте. Для настройки
 показателей используйте вкладку «Атрибуты»
 в диалоге «Свойства карты».


[![](../../opened.gif)![](../../closed.gif)Для отображения
 диалога «Свойства карты»](javascript:TextPopup(this))


		- [Добавьте
		 карту на лист отчета](svgmap.htm#add_and_locate_map) или выберите существующую карту.


		- Выполните команду «Свойства
		 карты» в контекстном меню карты.


![](map%20property.gif)


Задайте следующие настройки:


	- Слои и
	 показатели. Отображается список
	 всех слоев карты. Для каждого слоя можно добавить показатель или стрелку.
	 По умолчанию все слои и добавленные стрелки и показатели отображаются
	 на карте. Для скрытия слоя, стрелки или показателя снимите флажок,
	 который ему соответствует. При установке/снятии флажка у родительского
	 элемента будут устанавливаться/сниматься флажки у дочерних элементов;


	- Атрибуты. Отображаются
	 атрибуты для выделенного слоя, показателя или стрелки. Набор атрибутов
	 зависит от выбранного элемента. Доступна настройка следующих атрибутов:


	-


		- [атрибуты слоя](UiReport_layer_atr.htm);


		- [атрибуты картографических
		 показателей](UiMaps_Report_property_1_atr_1.htm);


		- [атрибуты столбиковых
		 показателей](UiMaps_Report_property_1_atr_3.htm);


		- [атрибуты круговых
		 показателей](UiMaps_Report_property_1_atr_2.htm).


Значения атрибутов отображаются в виде гиперссылок.


Для редактирования значения атрибута щёлкните
 по гиперссылке. Значение атрибута изменится на противоположное
 (да/нет) или в нижней части диалога «Свойства
 карты» будут отображены поля для ввода значения атрибута.


Если слева от названия атрибута располагается
 знак «+», то для этого атрибута
 существует список вложенных атрибутов, который раскрывается щелчком по
 знаку «+».


[![](../../opened.gif)![](../../closed.gif)Для
 добавления показателя](javascript:TextPopup(this))


	Для добавления показателя выделите требуемый слой, нажмите кнопку
	 «Добавить» и в отобразившемся
	 меню выберите тип показателя:


		- Картографический показатель.
		 Заливка региона задается в зависимости от значения показателя;


		- Столбиковый показатель.
		 Значение показателя для областей данного слоя будут отображаться
		 в виде столбиковых диаграмм. От значений показателей могут зависеть
		 размеры диаграммы, количество столбиков, их цвет и штриховка;


		- Круговой показатель.
		 Значения показателя для областей данного слоя будут отображаться
		 в виде круговых диаграмм. От значений показателей могут зависеть
		 радиус диаграммы, количество частей, их цвет и штриховка.


[![](../../opened.gif)![](../../closed.gif)Для
 удаления показателя](javascript:TextPopup(this))


	Для удаления выделенного показателя нажмите кнопку «Удалить».
	 Будет отображен запрос подтверждения выполняемого действия.
