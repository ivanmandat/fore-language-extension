# Настройка источников легенды


Для настройки источника легенды карты перейдите на вкладку «Источники»
 окна «[Легенда
 карты](UiMaps_Report_Legend.htm)»:


![](legend_map%20sourses.gif)


Выберите карту из раскрывающегося списка «Карта».
 Список содержит идентификаторы карт, [добавленных](svgmap.htm#add_and_locate_map)
 в текущем отчете.


Примечание.
 Данная настройка доступна только для внешней легенды. Подробнее см. раздел
 «[Расширенная
 настройка легенды](UiMaps_Report_Legend.htm)».
