# Возможности карты в регламентном
 отчете


В инструменте «Отчеты» карта
 имеет более широкие возможности по сравнению с другими инструментами продукта
 «Форсайт. Аналитическая платформа».


Для карты в регламентном отчете доступно:


	- добавление [показателей](UiMaps_Report_property_1.htm),
	 которые будут использоваться для отображения информации, и настройка
	 их атрибутов;


	- упрощенное создание [стрелок](UiMaps_Report_property_1_arrow.htm)
	 между территориями;


	- создание [расшифровки
	 территорий](TerritoryInfo/TerritoryInfo_setup.htm);


	- расширенная настройка [легенды
	 карты](UiMaps_Report_Legend.htm);


	- настройка [оформления
	 подписей](UiMaps_Report_property_appearance_tips.htm) территорий;


	- настройка [обработчика](UiMaps_Report_property_param.htm)
	 для событий карты;


	- изменение [состава карты](Map_Structure.htm).
