# Ссылки и массивы


Список предустановленных функций для работы со ссылками и массивами
 приведен в таблице:


		 Функция
		 Краткое описание


		 [HLookUp](UiReport_Func_LinkArray_HLookUp.htm)
		 Выполняет поиск значения в первой строке таблицы и возвращает
		 значение ячейки, находящейся в том же столбце в заданной строке.


		 [Match](UiReport_Func_LinkArray_Match.htm)
		 Выполняет поиск указанного элемента в диапазоне ячеек и возвращает
		 относительную позицию этого элемента в диапазоне.


		 [Offset](UiReport_Func_LinkArray_Offset.htm)
		 Возвращает ссылки на диапазон ячеек, смещенный относительно
		 заданной ссылки на указанное число строк и столбцов.


		 [Range](UiReport_Func_LinkArray_Range.htm)
		 Возвращает диапазон ячеек по ссылке.


		 [Rng](UiReport_Func_LinkArray_Rng.htm)
		 Возвращает диапазон по ссылке на ячейку.


		 [VLookUp](UiReport_Func_LinkArray_VLookUp.htm)
		 Выполняет поиск значения в крайнем левом столбце таблицы и
		 возвращает значение ячейки, находящейся в том же столбце в заданной
		 строке.
