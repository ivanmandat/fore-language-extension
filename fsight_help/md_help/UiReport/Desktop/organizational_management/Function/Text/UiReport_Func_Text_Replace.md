![](Replace.gif)


## Синтаксис


Replace(Old_text, Start_num, Num_chars, New_text)


## Параметры


Old_text.
 Исходная строка, в которой необходимо произвести замену;


Start_num.
 Позиция символа в исходной строке, с которого необходимо начать замену;


Num_chars.
 Количество символов в исходной строке, которые необходимо заменить;


New_text.
 Заменяющая строка.


Примечание.
 В качестве параметра можно указывать как непосредственно значение параметра,
 так и адрес ячейки, в которой оно располагается.


## Описание


Заменяет часть одной строки другой строкой.


## Комментарии


Нумерация символов в строке начинается с нуля.


## Пример


		 Формула
		 Результат
		 Описание


		 =Replace("asdfgh", 1, 2, "12")
		 a12fgh
		 Заменяет два знака, начиная с первого, в строке asdfgh на строку
		 12.


		 =Replace(B6, 0, 5, C6)
		 +56
		 Заменяет пять знаков, с начала строки, расположенной в ячейке
		 B6, на строку в ячейке C6. Ячейка B6 содержит значение 0123456,
		 C6 содержит значение +.
