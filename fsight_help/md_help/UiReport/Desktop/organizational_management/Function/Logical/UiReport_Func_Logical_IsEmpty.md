 образом:


![](IsEmpty.gif)


## Синтаксис


IsEmpty(Value)


## Параметры


Value. Определяет значение
 первого аргумента.


## Описание


Возвращает TRUE, если аргументом
 является ссылка на пустую ячейку, в противном случае возвращает FALSE.
