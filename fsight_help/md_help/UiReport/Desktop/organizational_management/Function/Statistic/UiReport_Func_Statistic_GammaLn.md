# GammaLn


[Мастер функций](../../UiReport_Organizational_master_function.htm)
 для функции GammaLn выглядит следующим
 образом:


![](GammaLn%20.gif)


## Синтаксис


GammaLn(Value)


## Параметры


Value. Значение, для которого
 вычисляется GammaLn. Значение
 должно удовлетворять ограничению: Value
 > 0.


## Описание


Метод возвращает натуральный логарифм гамма-функции.


## Комментарии


Число e, возведенное в степень GammaLn(i), где i
 — целое число, возвращает такой же результат, как и (i
 - 1)!.
