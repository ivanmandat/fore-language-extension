# Covar


[Мастер функций](../../UiReport_Organizational_master_function.htm)
 для функции Covar выглядит следующим
 образом:


![](Covar.gif)


## Синтаксис


Covar(A1, A2)


## Параметры


A1. Первый массив значений;


A2. Второй массив значений.


Примечание.
 A1 и A2
 должны иметь равное количество точек данных.


## Описание


Метод возвращает ковариацию, то есть среднее произведений отклонений
 для каждой пары точек данных.


## Комментарии


Ковариация используется для определения связи между двумя множествами
 данных.
