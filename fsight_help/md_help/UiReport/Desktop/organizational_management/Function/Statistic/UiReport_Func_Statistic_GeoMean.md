# GeoMean


[Мастер функций](../../UiReport_Organizational_master_function.htm)
 для функции GeoMean выглядит следующим
 образом:


![](GeoMean%20.gif)


## Синтаксис


GeoMean(Values,…)


## Параметры


Values1, Values2, …, ValuesN.
 Массив данных, для которых вычисляется среднее. Значения в массиве должны
 быть больше нуля.


## Описание


Метод возвращает среднее геометрическое значений массива положительных
 чисел.
