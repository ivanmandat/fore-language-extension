## Синтаксис


FDist(Value, DegOfFreedomNumer, DegOfFreedomDenomin)


## Параметры


Value. Значение, для которого
 вычисляется функция. Значение должно удовлетворять ограничению: Value >= 0;


DegOfFreedomNumer. Числитель
 степеней свободы. Допустимые значения берутся из диапазона [1, 10^10);


DegOfFreedomDenomin. Знаменатель
 степеней свободы. Допустимые значения берутся из диапазона [1, 10^10).


## Описание


Метод возвращает F-распределение вероятности.


## Комментарии


Функцию можно использовать, чтобы определить, имеют ли два множества
 данных различные степени разброса результатов.
