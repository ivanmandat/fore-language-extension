# FTest


[Мастер функций](../../UiReport_Organizational_master_function.htm)
 для функции FTest выглядит следующим
 образом:


![](FTest%20.gif)


## Синтаксис


FTest(A1, A2)


## Параметры


A1. Первый массив. Длина массива
 должна быть >= 2;


A2. Второй массив. Длина массива
 должна быть >= 2.


## Описание


Метод возвращает результат F-теста.


## Комментарии


F-тест возвращает одностороннюю вероятность того, что дисперсии параметров
 A1 и A2
 различаются несущественно. Эта функция используется для того, чтобы определить,
 имеют ли две выборки различные дисперсии.
