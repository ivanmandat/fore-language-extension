# Статистические функции


Список предустановленных статистических функций приведен в таблице:


		 Функция
		 Краткое описание


		 [_Frequency](UiReport_Func_Statistic_Frequency.htm)
		 Метод вычисляет частоту появления значений в интервале значений
		 и возвращает массив цифр.


		 [AveDev](UiReport_Func_Statistic_AveDev.htm)
		 Метод возвращает среднее абсолютных значений отклонений точек
		 данных от среднего.


		 [BetaDist](UiReport_Func_Statistic_BetaDist.htm)
		 Метод возвращает интегральную функцию плотности бета-вероятности.


		 [BetaInv](UiReport_Func_Statistic_BetaInv.htm)
		 Метод возвращает обратную функцию к интегральной функции плотности
		 бета-вероятности.


		 [BinomDist](UiReport_Func_Statistic_BinomDist.htm)
		 Метод возвращает отдельное значение биномиального распределения.


		 [ChiDist](UiReport_Func_Statistic_ChiDist.htm)
		 Метод возвращает одностороннюю вероятность распределения хи-квадрат.


		 [ChiInv](UiReport_Func_Statistic_ChiInv.htm)
		 Метод возвращает значение, обратное к односторонней вероятности
		 распределения хи-квадрат.


		 [ChiTest](UiReport_Func_Statistic_ChiTest.htm)
		 Метод возвращает значение для распределения хи-квадрат.


		 [Confidence](UiReport_Func_Statistic_Confidence.htm)
		 Метод возвращает доверительный интервал для среднего генеральной
		 совокупности.


		 [Correl](UiReport_Func_Statistic_Correl.htm)
		 Метод возвращает коэффициент корреляции между A1
		 и A2.


		 [Covar](UiReport_Func_Statistic_Covar.htm)
		 Метод возвращает ковариацию, то есть среднее произведений отклонений
		 для каждой пары точек данных.


		 [CritBinom](UiReport_Func_Statistic_CritBinom.htm)
		 Метод возвращает наименьшее значение, для которого интегральное
		 биномиальное распределение больше или равно заданному критерию.


		 [DevSq](UiReport_Func_Statistic_DevSq.htm)
		 Метод возвращает сумму квадратов отклонений точек данных от
		 их среднего.


		 [Dispersion](UiReport_Func_Statistic_Dispersion.htm)
		 Метод оценивает дисперсию по выборке.


		 [DispersionP](UiReport_Func_Statistic_DispersionP.htm)
		 Метод вычисляет дисперсию для генеральной совокупности.


		 [ExponDist](UiReport_Func_Statistic_ExponDist.htm)
		 Метод возвращает экспоненциальное распределение.


		 [FDist](UiReport_Func_Statistic_FDist.htm)
		 Метод возвращает F-распределение вероятности.


		 [FInv](UiReport_Func_Statistic_FInv.htm)
		 Метод возвращает обратное значение для F-распределения вероятностей.


		 [Fisher](UiReport_Func_Statistic_Fisher.htm)
		 Метод возвращает преобразование Фишера для аргумента Value.


		 [FisherInv](UiReport_Func_Statistic_FisherInv.htm)
		 Метод возвращает обратное преобразование Фишера.


		 [Forecast](UiReport_Func_Statistic_Forecast.htm)
		 Метод вычисляет будущее значение по существующим значениям.


		 [FTest](UiReport_Func_Statistic_FTest.htm)
		 Метод возвращает результат F-теста.


		 [GammaDist](UiReport_Func_Statistic_GammaDist.htm)
		 Метод возвращает гамма-распределение.


		 [GammaInv](UiReport_Func_Statistic_GammaInv.htm)
		 Метод возвращает обратное гамма-распределение.


		 [GammaLn](UiReport_Func_Statistic_GammaLn.htm)
		 Метод возвращает натуральный логарифм гамма-функции.


		 [GeoMean](UiReport_Func_Statistic_GeoMean.htm)
		 Метод возвращает среднее геометрическое значений массива положительных
		 чисел.


		 [HarMean](UiReport_Func_Statistic_HarMean.htm)
		 Метод возвращает среднее гармоническое множества данных.


		 [HypGeomDist](UiReport_Func_Statistic_HypGeomDist.htm)
		 Метод возвращает гипергеометрическое распределение.


		 [Intercept](UiReport_Func_Statistic_Intercept.htm)
		 Метод вычисляет точку пересечения линии с осью y, используя
		 KnownXs и KnownYs.


		 [JarqueBeraStat](UiReport_Func_Statistic_JarqueBeraStat.htm)
		 Метод возвращает статистику Жака-Бера.


		 [Kurt](UiReport_Func_Statistic_Kurt.htm)
		 Метод возвращает эксцесс множества данных.


		 [Large](UiReport_Func_Statistic_Large.htm)
		 Метод возвращает k-ое
		 по величине значение из множества данных.


		 [Linest](UiReport_Func_Statistic_Linest.htm)
		 Метод рассчитывает статистику для ряда с применением метода
		 наименьших квадратов, чтобы вычислить прямую линию, которая наилучшим
		 образом аппроксимирует имеющиеся данные.


		 [LogInv](UiReport_Func_Statistic_LogInv.htm)
		 Метод возвращает обратную функцию логарифмического нормального
		 распределения.


		 [LogNormDist](UiReport_Func_Statistic_LogNormDist.htm)
		 Метод возвращает интегральное логарифмическое нормальное распределение.


		 [Median](UiReport_Func_Statistic_Median.htm)
		 Метод возвращает медиану заданных чисел.


		 [Mode](UiReport_Func_Statistic_Mode.htm)
		 Метод возвращает моду - наиболее часто встречающееся значение
		 в массиве данных.


		 [NegBinomDist](UiReport_Func_Statistic_NegBinomDist.htm)
		 Метод возвращает отрицательное биномиальное распределение.


		 [NormDist](UiReport_Func_Statistic_NormDist.htm)
		 Метод возвращает нормальную функцию распределения для указанного
		 среднего и стандартного отклонения.


		 [NormInv](UiReport_Func_Statistic_NormInv.htm)
		 Метод возвращает обратное нормальное распределение.


		 [NormsDist](UiReport_Func_Statistic_NormsDist.htm)
		 Метод возвращает стандартное нормальное интегральное распределение.


		 [NormsInv](UiReport_Func_Statistic_NormsInv.htm)
		 Метод возвращает обратное значение стандартного нормального
		 распределения.


		 [Pearson](UiReport_Func_Statistic_Pearson.htm)
		 Метод возвращает коэффициент корреляции Пирсона (r).


		 [Percentile](UiReport_Func_Statistic_Percentile.htm)
		 Метод возвращает k-ую
		 процентиль для значений из интервала.


		 [PercentRank](UiReport_Func_Statistic_PercentRank.htm)
		 Метод возвращает категорию значения в наборе данных как процентное
		 содержание в наборе данных.


		 [Permut](UiReport_Func_Statistic_Permut.htm)
		 Метод возвращает количество перестановок для заданного числа
		 объектов.


		 [Poisson](UiReport_Func_Statistic_Poisson.htm)
		 Метод возвращает распределение Пуассона.


		 [Prob](UiReport_Func_Statistic_Prob.htm)
		 Метод возвращает вероятность того, что значение из интервала
		 находится внутри заданных пределов.


		 [Quartile](UiReport_Func_Statistic_Quartile.htm)
		 Метод возвращает квартиль множества данных.


		 [Rank](UiReport_Func_Statistic_Rank.htm)
		 Метод возвращает ранг числа в массиве чисел.


		 [Rsq](UiReport_Func_Statistic_Rsq.htm)
		 Метод возвращает квадрат коэффициента корреляции [Пирсона](UiReport_Func_Statistic_Pearson.htm)
		 (r^2).


		 [Skew](UiReport_Func_Statistic_Skew.htm)
		 Метод возвращает асимметрию распределения.


		 [Slope](UiReport_Func_Statistic_Slope.htm)
		 Метод возвращает наклон линии линейной регрессии.


		 [Small](UiReport_Func_Statistic_Small.htm)
		 Метод возвращает k-ое
		 наименьшее значение в множестве данных.


		 [Standardize](UiReport_Func_Statistic_Standardize.htm)
		 Метод возвращает нормализованное значение для распределения,
		 характеризуемого средним и стандартным отклонением.


		 [StDev](UiReport_Func_Statistic_StDev.htm)
		 Метод оценивает стандартное отклонение по выборке.


		 [StDevP](UiReport_Func_Statistic_StDevP.htm)
		 Метод вычисляет стандартное отклонение по генеральной совокупности.


		 [SteYX](UiReport_Func_Statistic_SteYX.htm)
		 Метод возвращает стандартную ошибку предсказанных значений
		 Y для каждого значения
		 X в регрессии.


		 [TDist](UiReport_Func_Statistic_TDist.htm)
		 Метод возвращает процентные точки (вероятность) для t-распределения
		 Стьюдента, где численное значение (x)
		 - это вычисленное значение, для которого должны быть вычислены
		 вероятности.


		 [TInv](UiReport_Func_Statistic_TInv.htm)
		 Метод возвращает t-значение распределения Стьюдента
		 как функцию вероятности и числа степеней свободы.


		 [Trend](UiReport_Func_Statistic_Trend.htm)
		 Метод возвращает значения в соответствии с линейным трендом.


		 [TrimMean](UiReport_Func_Statistic_TrimMean.htm)
		 Метод возвращает среднее внутренности множества данных.


		 [TTest](UiReport_Func_Statistic_TTest.htm)
		 Метод возвращает вероятность, соответствующую критерию Стьюдента.


		 [Weibull](UiReport_Func_Statistic_Weibull.htm)
		 Метод возвращает распределение Вейбулла.


		 [ZTest](UiReport_Func_Statistic_ZTest.htm)
		 Метод возвращает двустороннее P-значение z-теста.
