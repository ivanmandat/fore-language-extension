# Математические функции


При работе с формулами возможно использование предустановленных математических
 функций, приведенных в таблице:


		 Функция
		 Краткое описание


		 [Abs](UiReport_Func_Math_Abs.htm)
		 Возвращает модуль (абсолютную величину) числа.


		 [ACos](UiReport_Func_Math_ACos.htm)
		 Возвращает арккосинус числа.


		 [ACosH](UiReport_Func_Math_ACosH.htm)
		 Возвращает гиперболический арккосинус числа.


		 [ACot](ACot.htm)
		 Возвращает арккотангенс числа.


		 [ACotH](ACotH.htm)
		 Возвращает гиперболический арккотангенс числа.


		 [Aggregate](Aggregate.htm)
		 Возвращает результат агрегации заданным методом, с возможностью
		 не учитывать скрытые значения при расчете.


		 [ASin](UiReport_Func_Math_ASin.htm)
		 Возвращает арксинус числа.


		 [ASinH](UiReport_Func_Math_ASinH.htm)
		 Возвращает гиперболический арксинус числа.


		 [ATan](UiReport_Func_Math_ATan.htm)
		 Возвращает арктангенс числа.


		 [ATan2](UiReport_Func_Math_ATan2.htm)
		 Возвращает арктангенс для заданных координат X и Y.


		 [ATanH](UiReport_Func_Math_ATanH.htm)
		 Возвращает гиперболический арктангенс числа.


		 [Average](UiReport_Func_Math_Average.htm)
		 Возвращает среднее значение для массива ячеек.


		 [AverageIf](UiReport_Func_Math_AverageIf.htm)
		 Возвращает среднее значение ячеек, удовлетворяющих заданному
		 условию.


		 [AverageIfS](UiReport_Func_Math_AverageIfS.htm)
		 Возвращает среднее значение ячеек, удовлетворяющих заданному
		 набору условий.


		 [Ceiling](UiReport_Func_Math_Ceiling.htm)
		 Возвращает результат округления с избытком до ближайшего целого
		 числа, кратного значению точности.


		 [Combin](UiReport_Func_Math_Combin.htm)
		 Возвращает количество комбинаций для заданного числа объектов.


		 [Cos](UiReport_Func_Math_Cos.htm)
		 Возвращает косинус заданного угла.


		 [CosH](UiReport_Func_Math_CosH.htm)
		 Возвращает гиперболический косинус числа.


		 [Cot](Cot.htm)
		 Возвращает котангенс угла.


		 [CotH](CotH.htm)
		 Возвращает гиперболический котангенс числа.


		 [Count](Count.htm)
		 Возвращает количество элементов в массиве ячеек.


		 [CountIf](UiReport_Func_Math_CountIf.htm)
		 Возвращает количество ячеек, удовлетворяющих заданному условию.


		 [CountIfS](UiReport_Func_Math_CountIfS.htm)
		 Возвращает количество ячеек, удовлетворяющих заданному набору
		 условий.


		 [Degrees](UiReport_Func_Math_Degrees.htm)
		 Возвращает значение, преобразованное из радиан в градусы.


		 [Even](UiReport_Func_Math_Even.htm)
		 Возвращает результат округления числа до ближайшего четного
		 целого.


		 [Exp](UiReport_Func_Math_Exp.htm)
		 Возвращает результат возведения числа е в указанную степень.


		 [Fact](UiReport_Func_Math_Fact.htm)
		 Возвращает факториал указанного числа.


		 [Floor](UiReport_Func_Math_Floor.htm)
		 Возвращает результат округления числа до ближайшего меньшего
		 по модулю целого.


		 [Gcd](UiReport_Func_Math_Gcd.htm)
		 Возвращает наибольший общий делитель двух или более целых чисел.


		 [Int](UiReport_Func_Math_Int.htm)
		 Возвращает результат округления числа до ближайшего меньшего
		 целого.


		 [Lcm](UiReport_Func_Math_Lcm.htm)
		 Возвращает наименьшее общее кратное двух и более целых чисел.


		 [Ln](UiReport_Func_Math_Ln.htm)
		 Возвращает натуральный логарифм числа.


		 [Log](UiReport_Func_Math_Log.htm)
		 Возвращает логарифм числа по заданному основанию.


		 [Log10](UiReport_Func_Math_Log10.htm)
		 Возвращает десятичный логарифм заданного числа.


		 [Max](UiReport_Func_Math_Max.htm)
		 Возвращает максимальное число в массиве ячеек.


		 [Maxa](Maxa.htm)
		 Возвращает максимальное значение в массиве ячеек, содержащем
		 не только числовые значения.


		 [MDeterm](UiReport_Func_Math_MDeterm.htm)
		 Возвращает определитель матрицы, хранящейся в массиве ячеек.


		 [Min](UiReport_Func_Math_Min.htm)
		 Возвращает минимальное число в массиве ячеек.


		 [Mina](Mina.htm)
		 Возвращает минимальное число в массиве ячеек, содержащем не
		 только числовые значения.


		 [MInverse](UiReport_Func_Math_MInverse.htm)
		 Возвращает обратную матрицу для указанной матрицы.


		 [MMult](UiReport_Func_Math_MMult.htm)
		 Возвращает матрицу чисел, равную произведению двух матриц.


		 [Mod_](UiReport_Func_Math_Mod_.htm)
		 Возвращает остаток от деления числа на делитель.


		 [MRound](UiReport_Func_Math_MRound.htm)
		 Возвращает число, округленное с желаемой точностью.


		 [Multinomial](UiReport_Func_Math_Multinomial.htm)
		 Возвращает отношение факториала суммы значений к произведению
		 факториалов.


		 [Odd](UiReport_Func_Math_Odd.htm)
		 Возвращает результат округления числа до ближайшего нечетного
		 целого.


		 [Power](UiReport_Func_Math_Power.htm)
		 Возвращает результат возведения вещественного числа в вещественную
		 степень.


		 [Product](UiReport_Func_Math_Product.htm)
		 Возвращает произведение значений массива ячеек.


		 [Quotient](UiReport_Func_Math_Quotient.htm)
		 Возвращает целую часть результата деления с остатком.


		 [Radians](UiReport_Func_Math_Radians.htm)
		 Возвращает значение, преобразованное из градусов в радианы.


		 [Rand](UiReport_Func_Math_Rand.htm)
		 Возвращает равномерно распределенное случайное число, большее
		 либо равное 0 и меньшее 1.


		 [RandBetween](UiReport_Func_Math_RandBetween.htm)
		 Возвращает случайное число между двумя заданными числами.


		 [RndPermutation](RndPermutation.htm)
		 Возвращает массив, являющийся произвольной перестановкой другого
		 массива, элементами которого являются целые числа от 1 до значения
		 передаваемого параметра.


		 [RndSample](RndSample.htm)
		 Возвращает произвольную выборку из M элементов из массива,
		 элементами которого являются целые числа от 1 до N.


		 [Roman](UiReport_Func_Math_Roman.htm)
		 Возвращает результат преобразования арабского числа в римское
		 в виде текста.


		 [Round](UiReport_Func_Math_Round.htm)
		 Возвращает результат округления числа до указанного количества
		 десятичных разрядов.


		 [RoundDown](UiReport_Func_Math_RoundDown.htm)
		 Возвращает результат округления числа до ближайшего меньшего
		 по модулю значения.


		 [RoundUp](UiReport_Func_Math_RoundUp.htm)
		 Возвращает результат округления числа до ближайшего большего
		 по модулю значения.


		 [SeriesSum](UiReport_Func_Math_SeriesSum.htm)
		 Возвращает сумму степенного ряда.


		 [Sign](UiReport_Func_Math_Sign.htm)
		 Возвращает знак числа.


		 [Sin](UiReport_Func_Math_Sin.htm)
		 Возвращает синус заданного угла.


		 [SinH](UiReport_Func_Math_SinH.htm)
		 Возвращает гиперболический синус заданного числа.


		 [Sqrt](UiReport_Func_Math_Sqrt.htm)
		 Возвращает положительное значение квадратного корня.


		 [SqrtPi](UiReport_Func_Math_SqrtPi.htm)
		 Возвращает квадратный корень из значения выражения: Число * Пи.


		 [Sum](UiReport_Func_Math_Sum.htm)
		 Возвращает сумму вещественных чисел в заданном интервале ячеек.


		 [Sumif](Sumif.htm)
		 Возвращает сумму чисел в заданном интервале ячеек, удовлетворяющих
		 указанному условию.


		 [SumIfS](UiReport_Func_Math_SumIfS.htm)
		 Возвращает сумму чисел в заданном интервале ячеек, удовлетворяющие
		 заданному набору условий.


		 [SumSq](UiReport_Func_Math_SumSq.htm)
		 Возвращает сумму квадратов чисел в заданном интервале ячеек.


		 [SumX2MY2](UiReport_Func_Math_SumX2MY2.htm)
		 Возвращает сумму разностей квадратов соответствующих значений
		 в двух массивах.


		 [SumX2PY2](UiReport_Func_Math_SumX2PY2.htm)
		 Возвращает сумму сумм квадратов соответствующих значений в
		 двух массивах.


		 [SumXMY2](UiReport_Func_Math_SumXMY2.htm)
		 Возвращает сумму квадратов разностей соответствующих значений
		 в двух массивах.


		 [Tan](UiReport_Func_Math_Tan.htm)
		 Возвращает тангенс заданного угла.


		 [TanH](UiReport_Func_Math_TanH.htm)
		 Возвращает гиперболический тангенс заданного угла.


		 [Trunc](UiReport_Func_Math_Trunc.htm)
		 Усекает число до указанного количества десятичных разрядов.
