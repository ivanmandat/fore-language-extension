# Degrees


[Мастер функций](../../UiReport_Organizational_master_function.htm)
 для функции Degrees выглядит следующим
 образом:


![](Degrees_m.gif)


## Синтаксис


Degrees(Value)


## Параметры


Value. Угол в радианах, преобразуемый
 в градусы.


Примечание.
 В качестве параметра можно указывать как непосредственно число, так и
 адрес ячейки, в которой оно располагается.


## Описание


Возвращает значение, преобразованное из радиан в градусы.


## Пример


		 Формула
		 Результат
		 Описание


		 =Degrees(3.141592)
		 180
		 Угол 3,141592 радиан в градусах.


		 =Degrees(B6)
		 229,1831
		 Значение угла, расположенное в ячейке B6, в градусах. Ячейка
		 B6 содержит число 4.
