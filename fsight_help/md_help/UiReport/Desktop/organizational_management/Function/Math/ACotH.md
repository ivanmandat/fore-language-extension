# ACotH


[Мастер функций](../../UiReport_Organizational_master_function.htm)
 для функции ACotH выглядит следующим
 образом:


![](ACotH.gif)


## Синтаксис


ACotH(Value)


## Параметры


Value. Действительное число,
 от которого требуется найти гиперболический арккотангенс. Число должно
 удовлетворять условию: меньше -1 либо больше 1 (исключая -1 и 1).


Примечание.
 В качестве параметра можно указывать как непосредственно число, так и
 адрес ячейки, в которой оно располагается.


## Описание


Возвращает гиперболический арккотангенс.


## Пример


		 Формула
		 Результат
		 Описание


		 = ACotH(A1)
		 0,5493
		 Гиперболический арккотангенс числа, расположенного в ячейке
		 A1, в радианах. Ячейка A1 содержит число 2.


		 = ACotH(1.1)
		 1,5223
		 Гиперболический арккосинус числа 1.1 в радианах.


Примечание.
 Если нужно преобразовать результат из радиан в градусы, используйте функцию
 [Degrees](UiReport_Func_Math_Degrees.htm).
