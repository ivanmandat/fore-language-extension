# Nominal


[Мастер функций](../../UiReport_Organizational_master_function.htm)
 для функции Nominal выглядит следующим
 образом:


![](Nominal.gif)


## Синтаксис


Nominal(EffectRate, PeriodCount)


## Параметры


EffectRate. Фактическая процентная
 ставка. Значение параметра должно быть больше нуля;


PeriodCount. Количество периодов
 в году, за которое начисляются сложные проценты. Значение параметра должно
 быть больше либо равно нулю.


Примечание.
 В качестве параметра можно указывать как непосредственно значение параметра,
 так и адрес ячейки, в которой оно располагается.


## Описание


Возвращает номинальную годовую ставку, если заданы эффективная (фактическая)
 ставка и число периодов в году, за которые начисляются сложные проценты.


## Пример


		 Формула
		 Результат
		 Описание


		 =Nominal(0.25, 4)
		 0,23
		 Номинальная годовая процентная ставка при годовой ставке 0,25
		 и четырех периодах в год, за которые начисляются сложные проценты.


		 =Nominal(B6, 12)
		 0,14
		 Номинальная годовая процентная ставка при годовой ставке, указанной
		 в ячейке В6, и двенадцати периодах в год, за которые начисляются
		 сложные проценты. Ячейка B6 содержит значение 0,15.
