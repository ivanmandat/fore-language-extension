# XIrr


[Мастер функций](../../UiReport_Organizational_master_function.htm)
 для функции XIrr выглядит следующим
 образом:


![](XIrr.gif)


## Синтаксис


XIrr(Values, DateTimes[, Guess])


## Параметры


Values. Диапазон ячеек, содержащий
 значения денежных потоков, соответствующие графику платежей, приведенному
 в параметре DateTimes. Первый
 платеж является необязательным и соответствует выплате в начале инвестиции.
 Все последующие выплаты дисконтируются на основе 365-дневного года. Ряд
 значений должен содержать, по крайней мере, одно положительное и одно
 отрицательное значение;


DateTimes. Диапазон ячеек,
 содержащий расписание дат платежей, которое соответствует ряду денежных
 потоков. Первая дата - начальная в графике платежей. Все другие даты должны
 быть позже этой даты, но могут идти в произвольном порядке;


Примечание.
 Количество значений в параметрах Values
 и DateTimes должны совпадать.


Guess. Предполагаемое значение
 результата функции XIrr. Данный
 параметр не является обязательным. Значение по умолчанию: 0,1. В большинстве
 случаев нет необходимости задавать данный параметр.


Примечание.
 В качестве параметра можно указывать как непосредственно значение параметра,
 так и адрес ячейки, в которой оно располагается.


## Описание


Возвращает внутреннюю ставку доходности для графика денежных потоков,
 которые не обязательно носят периодический характер.


## Комментарии


Используется итеративный метод для вычисления XIrr.
 Используя меняющуюся ставку, начиная со значения параметра Guess,
 функция XIrr производит циклические
 вычисления, пока не получит результат с точностью до 0,000001 процента.
 Если функции XIrr не удается найти
 результат за 100 попыток, то возвращается значение ошибки. Ставка меняется
 до тех пор, пока не достигается равенства:


![](XIrr_f.gif),


где:


	- di.
	 Дата i-ой или последней выплаты;


	- d1. Дата нулевой или
	 начальной выплаты;


	- Pi.
	 Сумма i-ой или последней
	 выплаты;


	- Rate. Скорость.


## Пример


		 Формула
		 Результат
		 Описание


		 =XIrr(A0:E0, A1:E1,0.09)
		 2086,65
		 Внутренняя ставка доходности для ряда потоков денежных средств,
		 указанных в диапазоне A0:E0. Расписание дат платежей приведено
		 в диапазоне ячеек A1:E1. В диапазоне A0:E0 содержатся следующие
		 значения: -10000, 2750, 4250, 3250, 2750. В диапазоне A0:E0 содержатся
		 следующие значения: 01.01.2008, 01.03.2008, 03.10.2008, 15.02.2009,
		 01.04.2009.
