# Column

 для функции Column:


![](Column.png)


## Синтаксис


Column(Reference)


## Параметры


Reference. Ячейка или диапазон.


## Описание


Возвращает номер столбца, на который указывает ссылка.


## Комментарии


Нумерация столбцов начинается с нуля.


## Пример


		 Формула
		 Результат
		 Описание


		 =Column(V4)
		 21
		 Номер столбца.


		 =Column( A1:C1)
		 0
		 Номер первого столбца диапазона.
