## Синтаксис


Columns(Reference)


## Параметры


Reference. Ссылка на диапазон.


## Описание


Возвращает количество столбцов в ссылке.


## Пример


		 Формула
		 Результат
		 Описание


		 =Column( A1:C1)
		 3
		 Количество столбцов в диапазоне.
