# Row

![](Row.png)


## Синтаксис


Row(Reference)


## Параметры


Reference. Ячейка или диапазон.


## Описание


Возвращает номер строки, на которую указывает ссылка.


## Пример


		 Формула
		 Результат
		 Описание


		 =Row(V4)
		 4
		 Номер строки.


		 =Row( A1:A4)
		 1
		 Номер первой строки диапазона.
