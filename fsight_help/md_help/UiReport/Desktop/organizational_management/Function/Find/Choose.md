## Синтаксис


Choose(NumElement,Value, …)


## Параметры


NumElement. Индекс значения
 в списке;


Value - 1,Value - 1, …, Value - N.
 Список значений;


## Описание


Выбирает значение из списка значений по номеру индекса.


## Комментарии


Индекс значения должен быть числом от 1 до 254; формулой, результатом
 которой является число от 1 до 254, или ссылкой на ячейку, которая содержит
 число от 1 до 254.


В списке значений необходимо задать минимум одно значение, максимум
 - 254. Значения могут быть числами, ссылками на ячейки, формулами, функциями,
 определенными именами или текстом.


Индекс значения в списке должен быть равен или меньше количества элементов
 списка значений.


## Пример


		 Формула
		 Результат
		 Описание


		 =Choose(2, "первый", "второй",
		 "третий")
		 второй
		 Выбор второго значения из заданных.


		 =Choose(1, A1; B1; C1)
		 5
		 Выбор первого значения - A1. Значение ячейки «5».
