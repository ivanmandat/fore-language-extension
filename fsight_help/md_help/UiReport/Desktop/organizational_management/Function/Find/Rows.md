## Синтаксис


Rows(Reference)


## Параметры


Reference. Ссылка на диапазон.


## Описание


Возвращает количество строк в ссылке.


## Пример


		 Формула
		 Результат
		 Описание


		 =Rows( A1:A11)
		 11
		 Количество строк в диапазоне.
