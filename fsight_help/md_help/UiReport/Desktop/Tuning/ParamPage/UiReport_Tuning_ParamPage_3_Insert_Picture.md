# Использование изображений


Для повышения наглядности отчётов в колонтитулах можно использовать
 изображения из файла или из репозитория.


В текущий раздел колонтитула можно добавить только одно изображение.
 Если в разделе колонтитула уже содержится изображение, будет выдано соответствующее
 сообщение.


## Изображение из репозитория


Для добавления в колонтитул изображения из репозитория нажмите кнопку
 ![](3_Kolontituls_edit_b9.png)
 в [окне редактирования колонтитула](UiReport_Tuning_ParamPage_3.htm#edit)
 и выполните команду «Изображение из репозитория».


После выполнения команды будет открыт диалог выбора объекта репозитория.
 Выберите в нем объект репозитория
 «[Документ](UiNavObj.chm::/UiNavObj_document.htm)»,
 являющийся изображением, и нажмите кнопку «ОК».
 Если выбранный документ не является изображением, будет выдано сообщение
 об ошибке.


Примечание.
 Если было загружено новое изображение из внешнего файла в документ, используемый
 в колонтитулах, то при открытии отчёта в режиме просмотра или редактирования
 в колонтитулах будет отображен новый рисунок.


## Изображение из файла


Для добавления в колонтитул изображения из файла нажмите кнопку ![](3_Kolontituls_edit_b9.png)
 в [окне редактирования колонтитула](UiReport_Tuning_ParamPage_3.htm#edit)
 и выполните команду «Изображение из файла».


После выполнения команды будет открыт стандартный диалог выбора файла.
 Укажите в нем файл с изображением и нажмите кнопку «Открыть».
 В качестве изображений в колонтитулах поддерживаются изображения следующих
 форматов: JPEG, PNG, GIF, BMP, TIF.


После добавления изображения из файла или репозитория в окне редактирования
 колонтитула будет отображен текст «&[Picture]»:


![](3_Kolontituls_edit1.png)


Для настройки добавленного в колонтитул изображения
 нажмите кнопку ![](3_Kolontituls_edit_b10.png). Будет открыт диалог «Формат
 изображения»:


![](3_Kolontituls_Add_Pic_Format.png)


В открывшемся окне задайте параметры изображения:


	- Размер. С помощью клавиатуры
	 или редакторов чисел «Ширина
	 (мм)» и «Высота (мм)»
	 задайте высоту и ширину изображения в миллиметрах. Значения по умолчанию
	 соответствуют ширине и высоте изображения;


	- Масштабирование. С помощью
	 клавиатуры или редакторов чисел «Ширина
	 (%)» и «Высота (%)» определите
	 масштаб изображения относительно его исходных размеров. Значения по
	 умолчанию 100%. Также в группе параметров «Масштабирование»
	 доступна настройка следующих параметров:


		- Сохранять пропорции.
		 При установленном флажке появляется возможность изменять размер
		 изображения с соблюдением пропорций;


		- Использовать как фон.
		 При установленном флажке появляется возможность выводить
		 изображение под текстом раздела колонтитула;


	- Прозрачность. Из раскрывающегося
	 списка набора цветов выберите цвет, который будет считаться прозрачным
	 для данного изображения. Прозрачный цвет не выводится на экран и печать.


Все изменения параметров изображения отображаются в области «Предварительный
 просмотр».


Примечание.
 [Экспорт](../../Reports/OperationReport/UiReport_Reports_Operation_Export.htm)
 изображений в колонтитулах возможен только в документы формата PDF и RTF.
