# Визуализация данных в виде таблицы


Таблица данных представляет
 собой электронную таблицу, построенную на основе [источника
 данных](../Source/UiReport_Source.htm).


## Структура таблицы


Структура таблицы состоит из нескольких основных элементов:


![](tableStructure.png)


Структура таблицы включает следующие элементы:


	- Уголок. Верхняя левая
	 ячейка таблицы, позволяющая отображать наименования измерений по строкам
	 или столбцам;


	- Боковик. Элемент таблицы,
	 который отображает наименования элементов измерения, размещенного
	 по строкам;


	- Шапка. Элемент таблицы,
	 который отображает наименования элементов измерения, размещенного
	 по столбцам;


	- Данные. Элемент таблицы,
	 состоящий из ячеек с данными.


## Основные операции, доступные для таблицы


[![](../../opened.gif)![](../../closed.gif)Добавление таблицы](javascript:TextPopup(this))


	Для создания таблицы выполните одно из действий:


		- с помощью механизма Drag&Drop:


			- Наведите курсор на нужный источник/срез. Курсор примет
			 вид указателя ![](../Sheets/cursor.png).


			- Переместите источник/срез в рабочую область. Левая верхняя
			 граница выбранной ячейки будет левой верхней границей таблицы
			 или диаграммы, построенной на основе созданного среза.


	Примечание.
	 При перемещении источника в рабочую область будет автоматически добавлен
	 новый срез.


			- В меню выбора укажите ![](../Source/Window/b_table.png)«Таблица».


		- с помощью контекстного меню:


			- Выделите пустую ячейку на листе отчёта. Левая верхняя
			 граница выделенной ячейки будет левой верхней границей таблицы
			 или диаграммы, построенной на основе требуемого среза данных.


			- Выполните одно из действий:


				- выполните команду «Добавить > Таблицу»
				 в контекстном меню [источника
				 данных](../Source/UiReport_Source.htm) на панели «Источники».
				 Для источника будет создан новый срез данных, на основе
				 которого на листе отчёта будет сформирована таблица;


				- выполните команду «Добавить > Таблицу»
				 в контекстном меню среза данных на панели «Источники».


	После выполнения одного из действий:


		- на лист будет добавлена таблица;


		- на боковой панели «Источники»
		 добавится таблица с наименованием по умолчанию «Таблица
		 N», где N - порядковый
		 номер таблицы.


[![](../../opened.gif)![](../../closed.gif)Переименование
 таблицы](javascript:TextPopup(this))


	Для переименования таблицы выполните команду «Переименовать»
	 контекстного меню выбранной таблицы на [боковой
	 панели «Источники»](../Source/UiReport_Source.htm).


[![](../../opened.gif)![](../../closed.gif)Удаление таблицы](javascript:TextPopup(this))


	Для удаления таблицы выполните команду «Удалить»
	 контекстного меню выбранной таблицы на [боковой
	 панели «Источники»](../Source/UiReport_Source.htm).
