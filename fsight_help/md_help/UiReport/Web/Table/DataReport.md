# Работа с данными на листе отчёта


Лист регламентного отчёта содержит элементы: строки, столбцы и их пересечение
 - ячейки. Множество строк и столбцов представляют данные в виде таблиц.
 Значение в числовой ячейке таблицы можно указать в явном виде, взять из
 источника данных, либо [рассчитать
 по формуле](../organizational_management/Function/UiReport_Function.htm).
