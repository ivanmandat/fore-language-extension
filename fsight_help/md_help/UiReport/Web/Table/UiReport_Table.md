# Работа с элементами листа отчёта


Лист регламентного отчёта содержит элементы: строки, столбцы и их пересечение
 - ячейки. Множество строк и столбцов представляют данные в виде таблиц.
 Значение в числовой ячейке таблицы можно указать в явном виде, взять из
 источника данных, либо [рассчитать
 по формуле](../organizational_management/Function/UiReport_Function.htm).


## Адресация ячеек листа


Адресация ячеек листа может быть абсолютной и относительной:


	- Абсолютная адресация.
	 Предполагает запись адреса ячейки путем указания имени столбца и номера
	 строки. Например: F1, B50, FQD678;


	- Относительная адресация.
	 Используется в формулах, когда необходимо указать ячейку относительно
	 текущей, для которой вычисляется формула. Например, формула =R[2]C[-2]
	 означает ссылку на ячейку, которая располагается на 2 столбца левее
	 и на две строки ниже от текущей. Для включения относительной адресации
	 установите флажок «[Стиль
	 ссылок R1C1](../Reports/UiReport_Reports_Param4.htm#r1c1)».


По левому краю листа располагается столбец с номерами строк, а по верхнему
 краю - строка с именами столбцов.


Строки листа нумеруются сверху вниз числами, начиная с нуля, а столбцы
 - слева направо буквами латинского алфавита A, B, … , Y, Z. Если количество
 столбцов превышает 26, то далее используются двухбуквенные номера: AA,
 AB, … AZ, BA, BB и т.д. вплоть до ZZ, потом трехбуквенные от AAA до ZZZ
 и т.д.


Адрес ячейки состоит из номера столбца и номера строки, записанных без
 пробела, например: A1, B50 или FQD678.


Прямоугольная область ячеек может быть задана с помощью диапазона адресов
 ячеек. Диапазон состоит из адресов левой верхней и правой нижней ячеек
 блока, разделенных двоеточием. Например, диапазон C12:AB14 включает ячейки,
 расположенные между столбцами от C до AB и строками 12, 13 и 14.

## Выделение элемента


Принцип выделения ячеек областей и ячеек листа отчёта одинаковый.


Способы выделения ячеек:


[![](../../opened.gif)![](../../closed.gif)Единичная ячейка](javascript:TextPopup(this))


	Для выделения произвольной ячейки:


		- щёлкните по ней;


		- перейдите к ней с помощью клавиш/сочетания клавиш UP, DOWN,
		 RIGHT/TAB, LEFT/SHIFT+TAB.


	Для выделения крайней ячейки:


		- крайней левой ячейки текущей строки. Используйте сочетание
		 клавиш CTRL+LEFT;


		- крайней правой ячейки текущей строки. Используйте сочетание
		 клавиш CTRL+RIGHT;


		- крайней верхней ячейки текущего столбца. Используйте сочетание
		 клавиш CTRL+UP;


		- крайней нижней ячейки текущего столбца. Используйте сочетание
		 клавиш CTRL+DOWN.


[![](../../opened.gif)![](../../closed.gif)Столбцы целиком](javascript:TextPopup(this))


	Для выделения одного столбца:


		- щёлкните по заголовку столбца;


		- щёлкните по любой ячейке в требуемом столбце и нажмите сочетание
		 клавиш CTRL+ПРОБЕЛ.


	Для выделения нескольких столбцов:


		- щёлкните по заголовку необходимого столбца и, удерживая
		 зажатой кнопку мыши, переместите курсор к заголовку желаемого
		 столбца;


		- щёлкните по заголовку необходимого столбца и, удерживая
		 зажатой клавишу SHIFT, щёлкните по заголовку желаемого столбца;


		- выделите произвольный диапазон ячеек, содержащийся в требуемом
		 диапазоне столбцов, и нажмите сочетание клавиш CTRL+ПРОБЕЛ.


[![](../../opened.gif)![](../../closed.gif)Строки целиком](javascript:TextPopup(this))


	Для выделения одной строки:


		- щёлкните по заголовку строки;


		- щёлкните по любой ячейке в требуемой строке и нажмите сочетание
		 клавиш SHIFT+ПРОБЕЛ.


	Для выделения нескольких строк:


		- щёлкните по заголовку необходимой строки и, удерживая зажатой
		 кнопку мыши, переместите курсор к заголовку желаемой строки;


		- щёлкните по заголовку необходимой строки и, удерживая зажатой
		 клавишу SHIFT, щёлкните по заголовку желаемой строки;


		- выделите произвольный диапазон ячеек, содержащийся в требуемом
		 диапазоне строк, и нажмите сочетание клавиш SHIFT+ПРОБЕЛ.


[![](../../opened.gif)![](../../closed.gif)Диапазон ячеек](javascript:TextPopup(this))


	Для выделения диапазона ячеек:


		- щёлкните по требуемой ячейке в начале диапазона и, удерживая
		 зажатой кнопку мыши, переместите курсор к желаемой ячейке в конце
		 диапазона:


		- выделите требуемую ячейку в начале диапазона и, удерживая
		 зажатой клавишу SHIFT, щёлкните по желаемой ячейке в конце диапазона;


		- выделите требуемую ячейку в начале диапазона и, удерживая
		 зажатой клавишу SHIFT, перейдите к желаемой ячейке в конце диапазона
		 с помощью клавиш UP, DOWN, LEFT, RIGHT.


	Для выделения диапазона от текущей ячейки до крайней ячейки:


		- до крайней левой ячейки текущей строки. Используйте сочетание
		 клавиш CTRL+SHIFT+LEFT;


		- до крайней правой ячейки текущей строки. Используйте сочетание
		 клавиш CTRL+SHIFT+RIGHT;


		- до крайней верхней ячейки текущего столбца. Используйте
		 сочетание клавиш CTRL+SHIFT+UP;


		- до крайней нижней ячейки текущего столбца. Используйте сочетание
		 клавиш CTRL+SHIFT+DOWN.


	Для выделения всех ячеек на листе отчёта используйте сочетание клавиш
	 CTRL+A.


[![](../../opened.gif)![](../../closed.gif)Комбинированная
 область](javascript:TextPopup(this))


	Комбинированная область - область, состоящая из нескольких
	 непересекающихся диапазонов и/или единичных ячеек.


	Для получения комбинированной области:


		- Выделите необходимый диапазон ячеек или единичную ячейку.


		- Удерживая зажатой клавишу CTRL, выделите другие необходимые
		 диапазоны/единичные ячейки.


	После выполнения указанных действий будет выделена комбинированная
	 область.


[![](../../opened.gif)![](../../closed.gif)Весь лист](javascript:TextPopup(this))


	Для выделения всего листа данных:


		- щёлкните по левому верхнему углу таблицы, находящемуся на
		 пересечении заголовков строк и столбцов;


		- нажмите сочетание клавиш CTRL+SHIFT+SPACE.


	После выполнения одного из действий будет выделен весь лист данных.


Примечание.
 Для перемещения по ячейкам выделенной некомбинированной области используйте
 клавишу ENTER. При этом снятие рамки выделенной области не произойдет.
