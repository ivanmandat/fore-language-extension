# Исключение данных из таблицы


При работе с таблицей доступны операции исключения данных с помощью
 изменения отметки в измерениях по строкам и/или столбцам. Изменение отметки
 в измерениях доступно:


	- на [панели
	 отметки](../../Reports/Visualization.htm#selection_panel);


	- с помощью команд контекстного меню в боковике/шапке таблицы
	 на рабочей области:


		- для исключения строк/столбцов:


			- Выделите один или несколько элементов боковика/шапки.


			- Выполните команду «Исключить»
			 контекстного меню.


		- для исключения строк/столбцов, кроме выделенных:


			- Выделите один или несколько элементов боковика/шапки,
			 данные из которых должны отображаться в таблице.


			- Выполните команду «Исключить
			 остальные» контекстного меню.


Команды также доступны для элементов одного
 или разных измерений боковика/шапки таблицы, если выполнено [объединение
 измерений](../Attribute/Merging_Dim.htm#merging) или настроено [поджатие](../Attribute/Merging_Dim.htm#tucking).
