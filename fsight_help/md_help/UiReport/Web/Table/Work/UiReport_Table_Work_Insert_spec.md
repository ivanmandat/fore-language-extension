# Специальная вставка


При вставке данных из буфера обмена существует возможность корректировки
 вставляемых данных:


	- установите флажок  ![](../../Reports/OperationReport/b_paste_special.png)
	 «Специальная вставка»  в
	 контекстном меню ячейки;


	- нажмите сочетание клавиш CTRL+ALT+V. Доступно при работе веб-приложения
	 по [протоколу
	 HTTPS](setup.chm::/UiWebSetup/04_FAQ/setting_https.htm).


При [вставке](Insert.htm#paste) из буфера обмена будет открыто
 окно «Специальная вставка»:


![](insertion.png)


Задайте параметры:


	- Вставлять из буфера обмена.
	 Определите параметры специальной вставки:


	-


		- Значения ячеек.
		 По умолчанию флажок установлен и вставляются значения ячеек. При
		 снятии флажка значения ячеек вставляться не будут;


		- Формулы ячеек. По
		 умолчанию флажок установлен и вставляются формулы, содержащиеся
		 в ячейках. При снятии флажка формулы, содержащиеся в ячейках,
		 вставляться не будут;


		- Оформление ячеек.
		 По умолчанию флажок установлен и вставляется оформление ячеек.
		 При снятии флажка будет использован формат, заданный для ячеек,
		 в которые осуществляется вставка;


		- Примечания. По умолчанию
		 флажок установлен и вставляются примечания ячеек. При снятии флажка
		 примечания ячеек вставляться не будут;


	- Пропускать пустые ячейки.
	 Если вставляемые данные содержат пустые значения, то при установке
	 данного флажка они вставляться не будут. То есть в ячейках, в которые
	 должны были вставляться пустые значения, останутся старые данные;


	- Транспонировать данные.
	 При установке флажка вставляемые данные будут транспонированы, то
	 есть столбцы будут представлены в строках, а строки в столбцах.


Для возвращения к вставке данных из буфера обмена без корректировки
 снимите флажок ![](../../Reports/OperationReport/b_paste_special.png) «Специальная вставка» в контекстном
 меню ячейки или повторно нажмите сочетание клавиш CTRL+ALT+V.
