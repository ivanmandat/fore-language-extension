# Редактирование содержимого ячеек


Для настройки параметров форматирования содержимого ячеек перейдите
 в режим редактирования одним из способов:


	- выделите ячейку на листе отчёта;


	- дважды щёлкните по выбранной ячейке;


	- выделите ячейку и установите курсор в строку формул;


	- выделите ячейку и нажмите клавишу F2.


После выполнения действия введите необходимые данные в ячейку.


## Операции над содержимым ячеек


После перехода в режим редактирования с данными, содержащимися в ячейке,
 можно производить следующие операции:


[![](../../../opened.gif)![](../../../closed.gif)Настройка
 формата](javascript:TextPopup(this))


	Для настройки формата выбранных ячеек:


		- выполните команду ![](../../Reports/OperationReport/b_format.png) «Оформление» контекстного меню
		 ячейки. Откроется панель настройки [формата
		 ячеек](../Attribute/Format.htm). Задайте параметры форматирования содержимого ячеек;


		- используйте вкладку «Оформление»
		 панели инструментов.


[![](../../../opened.gif)![](../../../closed.gif)Настройка условного
 форматирования](javascript:TextPopup(this))


	Для настройки условного форматирования выбранных ячеек выполните
	 команду ![](../../Reports/OperationReport/b_format_condition.png) «Условное форматирование» контекстного
	 меню ячейки. Откроется панель настройки [условного
	 форматирования ячеек](../Attribute/Conditional_Formatting.htm). Задайте параметры условного форматирования
	 ячеек.


[![](../../../opened.gif)![](../../../closed.gif)Перемещение
 данных](javascript:TextPopup(this))


	Более подробно перемещение данных описано в статье «[Копирование
	 и вставка данных](Insert.htm)».


[![](../../../opened.gif)![](../../../closed.gif)Копирование
 данных](javascript:TextPopup(this))


	Более подробно копирование данных описано в статье «[Копирование
	 и вставка данных](Insert.htm)».


[![](../../../opened.gif)![](../../../closed.gif)Вставка
 данных](javascript:TextPopup(this))


	Более подробно вставка данных описана в статье «[Копирование
	 и вставка данных](Insert.htm)».


[![](../../../opened.gif)![](../../../closed.gif)Выделение данных](javascript:TextPopup(this))


	Для выделения всего содержимого выбранной ячейки перейдите в режим
	 редактирования ячейки и выполните одно из действий:


		- выполните команду контекстного меню «Выделить
		 всё»;


		- нажмите сочетание клавиш CTRL+A.


[![](../../../opened.gif)![](../../../closed.gif)Очистка
 данных](javascript:TextPopup(this))


	Для очистки данных ячейки используйте кнопку ![](b_clear.png)
	 «Очистить» вкладки «Данные» [панели
	 инструментов](../../organizational_management/Starting.htm#structure_window).


	Доступны следующие варианты очистки данных:


		- очистка примечания.
		 Для очистки только [примечания](UiReport_Table_Work_Notice.htm)
		 к ячейке выполните команду «Очистить
		 примечание» в раскрывающемся меню кнопки ![](b_clear.png) «Очистить»;


		- очистка содержимого.
		 Для очистки только содержимого ячейки выполните команду «Очистить содержимое» в раскрывающемся
		 меню кнопки ![](b_clear.png) «Очистить».
		 Доступно также по нажатию клавиши DELETE;


		- очистка формата.
		 Для очистки только [формата данных
		 в ячейке](../Attribute/Format.htm) выполните команду «Очистить
		 формат» в раскрывающемся меню кнопки ![](b_clear.png)
		 «Очистить». Доступно также
		 по нажатию сочетания клавиш CTRL+\;


		- полная очистка.
		 Для одновременной очистки содержимого ячейки, формата данных в
		 ячейке и примечания к ячейке нажмите на кнопку ![](b_clear.png)
		 «Очистить» или выполните
		 команду «Очистить все»
		 в её раскрывающемся меню.


	При использовании [фильтрации](../../Reports/OperationReport/Work_with_data.htm#filter)
	 диапазона данных на листе отчёта или таблицы очистка данных применяется
	 только к видимым строкам/столбцам.


После ввода данных для выхода из режима редактирования содержимого ячейки:


	- нажмите клавишу ENTER;


	- нажмите клавишу TAB;


	- нажмите клавишу перемещения курсора, при этом осуществится переход
	 на соседнюю ячейку в заданном направлении. Данный метод не работает,
	 если ячейка редактируется с помощью клавиши F2 или из строки формул.


Для отмены и повтора действия с содержимым ячейки обратитесь к разделу
 «[Отмена и повтор операции](../../../Desktop/Reports/Undo_Redo.htm)».
