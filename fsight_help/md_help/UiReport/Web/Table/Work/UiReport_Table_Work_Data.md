# Ввод данных в ячейку листа


Для ввода данных в ячейку выполните одно из действий:


	- выделите ячейку на листе отчёта;


	- дважды щёлкните по выбранной ячейке;


	- выделите ячейку и установите курсор в строку формул;


	- выделите ячейку и нажмите клавишу F2.


После выполнения действия введите необходимые данные в ячейку.


После ввода данных для выхода из режима редактирования содержимого ячейки:


	- нажмите клавишу ENTER;


	- нажмите клавишу TAB;


	- нажмите клавишу перемещения курсора, при этом осуществится переход
	 на соседнюю ячейку в заданном направлении. Данный метод не работает,
	 если ячейка редактируется из строки формул.


Примечание.
 При вводе в ячейку значения с большим количеством знаков после запятой
 это число будет преобразовано в Decimal,
 если ранее в ячейке уже было значение Decimal.
 Если в ячейке было значение другого типа, преобразование будет только
 при установлении значения Decimal
 с использованием [Fore](Dal.chm::/Enums/DbDataType.htm).


Если при редактировании содержимого ячейки нажать на точку цифровой
 клавиатуры, в ячейку будет введён символ разделителя целой и дробной части
 из текущих системных настроек.


Примечание.
 Данными способами нельзя редактировать содержимое ячейки, если для нее
 настроен [редактор
 значений](uinav.chm::/GUI/valueeditorparameters.htm).
 Например, раскрывающийся список справочника.
