# Объединение измерений и поджатие элементов измерений


Если
 в боковик/шапку выведено несколько измерений, то для более компактного
 отображения боковика/шапки табличной области используйте настройки объединения
 и поджатия.


Объединение и поджатие измерений в боковике/шапке настраивается при
 помощи области «Измерения» в группе
 параметров «[Размещение](Layout.htm)»
 боковика/шапки.


[![](../../../opened.gif)![](../../../closed.gif)Для открытия
 группы параметров «Размещение»](javascript:TextPopup(this))


	Для открытия на панели параметров группы параметров «Размещение»:


		- Выделите ячейку или диапазон ячеек, принадлежащих табличной
		 области.


		- Нажмите кнопку ![](parameters.png) «Параметры» панели инструментов.


		- Выберите требуемый тип области в раскрывающемся меню заголовка
		 панели параметров.


## Объединение


Доступные варианты объединения измерений в таблице с несколькими измерениями:


	- Полное
	 объединении. Все измерения боковика/шапки объединены в одном
	 столбце/строке;


	- Частичное
	 объединение. Объединены только выбранные измерения боковика/шапки
	 в одном столбце/строке.


Примечание.
 Частичное объединение возможно, если таблица содержит три или более измерений
 в боковике/шапке.


Для каждого из вариантов доступно [поджатие элементов](#tucking).


[![](../../../opened.gif)![](../../../closed.gif)Объединение/разделение
 измерений в боковике/шапке](javascript:TextPopup(this))


	Для объединения измерения с предыдущим измерением выполните команду
	 «Объединить с предыдущим»
	 в его контекстном меню. В результате выполнения действий выбранное
	 измерение будет объединено с предыдущим, а команда будет помечена
	 флажком.


	Для разделения измерений снимите флажок с команды «Объединить
	 с предыдущим».


	После объединения измерения с предыдущим станет доступно его [поджатие с другими измерениями](#tucking).


## Поджатие


Для объединения элементов одного измерения с элементами другого используйте
 поджатие измерений. При использовании поджатия элемент измерения, который
 поджимают, перестанет отображаться в отдельной строке, а его значения
 будут отображены в строке элемента измерения, к которому поджимают.


Примечание.
 Поджатие доступно только для [объединённых измерений](#merging)
 боковика/шапки таблицы.


[![](../../../opened.gif)![](../../../closed.gif)Поджатие
 измерений в боковике/шапке](javascript:TextPopup(this))


	Для поджатия измерения:


		- Выполните команду «Поджимать
		 элементы» в контекстном меню измерения. В контекстном меню
		 для измерения станет доступна команда «Настроить»,
		 а команда «Поджимать элементы»
		 будет помечена флажком.


		- Выполните команду «Настроить»
		 в контекстном меню измерения. В результате появится дополнительная
		 панель для выбора измерений.


		- Отметьте измерения для поджатия, на появившейся дополнительной
		 панели.


	Для каждого элемента измерения, для которого
	 настраивается поджатие, проверяется наличие поджимаемых элементов
	 нижележащего измерения:


			- если поджимаемый элемент один, то происходит поджатие;


			- если поджимаемых элементов несколько, то поджимается
			 только один, выделенный первым.


	Примечание.
	 Выбор нескольких элементов необходим для ситуаций, когда в результате
	 применения фильтрации общих измерений или фильтрации с помощью показателя,
	 будут оставаться различные элементы поджимаемого измерения под каждым
	 элементом верхнего измерения.


	В результате выполнения действий для выбранного измерения будет
	 произведено поджатие.


	Наименование выбранного измерения после поджатия отображается в
	 исходном виде и состоит из наименования исходного элемента. Для ячейки
	 на табличной области будет применяться [оформление](Format.htm),
	 заданное для измерения, которое находилось в этой ячейке до поджатия.
