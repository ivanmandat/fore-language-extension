# Отмена и повтор операции


Для отмены последней операции в отчёте:


	- нажмите кнопку ![](../organizational_management/b_cancel.png) «Отменить» [главного
	 меню](../organizational_management/Starting.htm#structure_window);


	- нажмите сочетание клавиш CTRL+Z.


Если [измененные данные](OperationReport/Work_with_data.htm#edit)
 были сохранены, а затем сохранение было отменено, то будут возвращены
 предыдущие значения. Эти значения будут выделены как измененные, и для
 дальнейшего их использования требуется выполнить их сохранение.


Для повтора последней отменённой операции:


	- нажмите кнопку ![](../organizational_management/b_repeat.png)
	 «Повторить» главного меню;


	- нажмите сочетание клавиш CTRL+Y.


Кнопки ![](../organizational_management/b_cancel.png) «Отменить»
 и ![](../organizational_management/b_repeat.png) «Повторить»
 доступны:


	- при [вводе](../../Desktop/Table/Work/UiReport_Table_Work_Data.htm)
	 данных в ячейку;


	- при [форматировании](../Table/Work/UiReport_Table_Work_Data_Edit.htm#format)
	 данных;


	- при [вставке/удалении](../../Desktop/Table/Work/UiReport_Table_Work_Data_Edit.htm#paste)
	 данных;


	- при [автозаполнении](UiNav.chm::/GUI/UiExpress_Tabl_Operation_AutoFillData.htm)
	 ячеек данными;


	- при [добавлении](../Table/Work/UiReport_Table_Work_Union.htm#adding_cells)/[удалении](../Table/Work/UiReport_Table_Work_Union.htm#deleting_cells)
	 ячеек;


	- при [создании/редактировании](../organizational_management/Function/UiReport_Function.htm#creating_formula)
	 формул;


	- при выполнении операций со [строками](../Table/OperationRow/UiReport_Table_Row.htm)/[столбцами](../Table/OperationColumn/UiReport_Table_Column.htm),
	 кроме операции [закрепления](../Table/Work/UiReport_Table_WorkStyle.htm#freeze)
	 строки/столбца;


	- при добавлении/удалении, перемещении [диаграммы](../Diagrams/UiReport_Diagrams_appointment.htm),
	 [фигуры](../Objects/Shapes/Shapes.htm) и [изображения](../Objects/Picture/UiReport_Objects_Picture.htm).


При [обновлении](OperationReport/UiReport_Reports_Operation_Execute.htm)
 отчёта очищается история операций и кнопки ![](../organizational_management/b_cancel.png) «Отменить» и ![](../organizational_management/b_repeat.png) «Повторить» главного меню становятся
 недоступными.
