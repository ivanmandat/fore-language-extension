# Просмотр готового отчёта


Закончив оформление отчёта, просмотрите готовый отчёт.


Для открытия готового отчёта в [режиме
 просмотра](../../organizational_management/Starting.htm#open) в [навигаторе
 объектов](getstarted.chm::/Interface/interface_navigator.htm) выполните одно из действий:


	- выполните команду «Открыть»
	 контекстного меню отчёта;


	- дважды щелкните по отчёту.


См. также:

 отчёта](../../CreateReport.htm)
