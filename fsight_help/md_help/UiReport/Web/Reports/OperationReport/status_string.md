# Расчёт базовых агрегатов


Настройка параметров расчёта базовых агрегатов выделенного диапазона
 ячеек таблицы, отображение результатов расчёта, а также отображение информации
 о выделенных объектах выполняется с помощью [строки
 статуса](../../organizational_management/Starting.htm#structure_window):


[![](../../../opened.gif)![](../../../closed.gif)Для отображения
 строки статуса](javascript:TextPopup(this))


	Убедитесь, что отображаются [вкладки
	 листов](../../organizational_management/Starting.htm#structure_window).


	Выделите несколько ячеек таблицы, содержащих данные.


	Примечание.
	 Если среди выделенных ячеек только одна ячейка содержит данные, то
	 строка статуса не отображается.


![](status_string.png)


Расчёт базовых агрегатов выделенного диапазона ячеек производится с
 помощью команд раскрывающегося меню строки статуса:


	- Среднее. Среднее значение
	 числовых данных выделенных ячеек. Устанавливается по умолчанию для
	 диапазонов ячеек, содержащих числовые или смешанные данные;


	- Количество. Количество
	 ячеек, которые содержат данные. Устанавливается по умолчанию для диапазонов
	 ячеек, содержащих строковые данные или даты;


	- Количество чисел. Количество
	 ячеек, которые содержат числовые данные;


	- Минимум. Наименьшее
	 из выделенных числовых значений;


	- Максимум. Наибольшее
	 из выделенных числовых значений;


	- Сумма. Сумма числовых
	 значений выделенных ячеек;


	- Нечисловые данные. Количество
	 ячеек, которые содержат нечисловые данные.


Выбранные агрегаты сохраняются для текущего отчёта и при следующем открытии
 отчёта отображаются в строке статуса.
