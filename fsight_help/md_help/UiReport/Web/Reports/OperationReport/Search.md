# Поиск по командам меню


Для ускорения нахождения и применения команд доступен поиск по командам
 меню. Поиск доступен при редактировании регламентного отчёта.


Примечание.
 Поиск доступен по наиболее востребованным командам. Команды пользовательских
 кнопок не участвуют в поиске.


Для поиска:


	- Активируйте строку поиска с помощью мыши или используйте сочетание
	 клавиш  ALT + /.


	- Введите команду.


	- Выберите нужную команду в выведенном списке с помощью мыши.
	 Если найдено несколько команд, то будут выведены первые четыре команды.
	 Сортировка команд в списке производится по количеству совпадающих
	 символов.


В результате будет выполнена выбранная команда: открыто соответствующая
 группа настроек, окно или выполнено действие.


При последующем поиске в списке доступных команд отображаются последние
 три использованные команды.


В списке доступных команд доступен переход к онлайн-справке. Онлайн-справка
 открывается в новой вкладке браузера. По умолчанию используется адрес
 онлайн-справки: https://help.fsight.ru.
