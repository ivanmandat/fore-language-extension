# Круговая диаграмма


Круговая диаграмма демонстрирует
 размер элементов одного ряда данных пропорционально сумме элементов.


Каждая группа данных отображается в виде круга, разделенного на секторы,
 угловые размеры которых пропорциональны значениям элементов данных.


Пример отображения круговой диаграммы с абсолютными значениями и такой
 же трехмерной:


![](Type%20pics/pie_chart.png)


Доступные типы круговых диаграмм:


	- ![](b_pie_chart.png)
	 абсолютные значения;


	- ![](b_pie_chart_secondary.png) вторичная круговая;


	- ![](b_pie_chart_secondary_bar.png) вторичная гистограмма;


	- ![](b_pie_chart_3d.png)
	 абсолютные значения трехмерная;


	- ![](b_pie_chart_doughnut.png) кольцевая.


Вторичная круговая диаграмма - круговая диаграмма
 с частью значений, вынесенных во вторую диаграмму:


![](Type%20pics/secondary_pie_chart.png)


Вторичная гистограмма - круговая диаграмма с
 частью значений, вынесенных в гистограмму:


![](Type%20pics/secondary_histogram.png)


Кольцевая диаграмма демонстрирует размер элементов
 одного ряда данных пропорционально сумме элементов. Каждая группа данных
 отображается в виде кольца, разделенного на секторы. Первый ряд данных
 отображается в центре диаграммы. Если в подписях данных отображаются проценты,
 то каждое кольцо в сумме должно давать 100 %.
