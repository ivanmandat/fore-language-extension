# Коробчатая диаграмма с линейными рядами


Коробчатая диаграмма с линейными рядами разновидность [коробчатой](Box_chart.htm).


На диаграмме отображаются статистические характеристики набора данных,
 рассчитанных «вручную». Ряды для такой диаграммы должны располагаться
 в строго определенном порядке. Обязательными являются: «Минимум»,
 «Нижняя квартиль», «Медиана»,
 «Верхняя квартиль», «Максимум»,
 «Среднее значение». Дополнительные
 статистические характеристики отображаются линейными рядами.


Для построения диаграммы с отображением рассчитанных статистических
 характеристик используем исходные данные «Динамика
 курса акций, долл»:


![](Type%20pics/box_chart_table.png)


Все статистические данные должны быть рассчитаны и расположены в таблице
 в строгом порядке:


![](Type%20pics/box_chart1.png)


Отображение данных в виде коробчатой диаграммы с линейными рядами:


![](Type%20pics/box_chart.png)
