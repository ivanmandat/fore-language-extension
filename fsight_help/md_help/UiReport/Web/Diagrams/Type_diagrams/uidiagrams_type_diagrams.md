# Типы диаграмм


Диаграммы используются для наглядного представления табличных данных.
 Каждый ряд данных на диаграмме обозначается отдельным цветом, указанным
 в [легенде](../Legend.htm), или способом представления. В отчётах
 могут использоваться следующие типы диаграмм:


	- ![](b_bar_chart.png)
	 [Гистограмма](UiDiagrams_Bar.htm);


	- ![](b_pie_chart.png)
	 [Круговая
	 диаграмма](UiDiagrams_round.htm);


	- ![](b_line_chart.png)
	 [Линейная
	 диаграмма](UiDiagrams_lenear.htm);


	- ![](b_area_chart.png)
	 [Диаграмма
	 с областями](uidiagrams_area.htm);


	- ![](b_mixed_chart.png)
	 [Смешанная
	 диаграмма](UiDiagrams_mixed_diagram.htm);


	- ![](b_other.png)
	 Другие. К данной группе относятся
	 следующие типы диаграмм:
