# Смешанная диаграмма


Смешанная диаграмма позволяет
 для каждой группы данных определить свой тип диаграммы: столбики, линии
 или области. Например, можно объединить график, содержащий данные о ценах,
 с гистограммой, отображающей объемы продаж.


Более подробную информацию о настройке типа ряда данных смотрите в разделе
 «[Ряды данных](../Data_Series.htm)».


Пример отображения смешанной диаграммы:


![](Type%20pics/mixed_chart.png)


Доступные типы смешанных диаграмм:


	- ![](b_mixed_chart.png)
	 абсолютные значения;


	- ![](b_mixed_chart_cumulative.png) накопительная;


	- ![](b_mixed_chart_percentage.png) процентная;


	- ![](b_mixed_chart_3d.png) абсолютные значения трехмерная;


	- ![](b_mixed_chart_cumulative_3d.png) накопительная трехмерная;


	- ![](b_mixed_chart_percentage_3d.png) процентная трехмерная.
