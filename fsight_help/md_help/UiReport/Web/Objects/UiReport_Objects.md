# Вставка объектов


Кроме размещения на [листах отчёта](../Table/UiReport_Table.htm)
 областей данных различных типов, инструмент «Отчёты»
 поддерживает возможность добавления объектов следующих типов:


	- [Рисунок](Picture/UiReport_Objects_Picture.htm).
	 Растровые и векторные изображения форматов BMP, GIF, PNG, JPG и т.д.;


Примечание.
 [Диаграммы](../Diagrams/UiReport_Diagrams_appointment.htm)
 также являются одним из типов объектов.


	- [Фигуры](Shapes/Shapes.htm).
	 Предустановленный набор фигур: линии, прямоугольники, правильные многоугольники
	 и т.д.;


Для работы с объектами обратитесь к статье «[Работа
 с объектами на листе отчёта](UiReport_Objects_Operation.htm)».


Для всех объектов существуют как [стандартные
 свойства](UiReport_Objects_StandartProperty.htm), так и специальные, зависящие от типа объекта.
