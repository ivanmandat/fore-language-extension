# Перемещение/копирование листа


Для перемещения листа:


	- Наведите указатель мыши на вкладку соответствующего листа, при
	 этом указатель сменит вид на ![](cursor.png).


	- Щёлкните по вкладке и, удерживая её, переместите лист в нужную
	 позицию.


Для копирования листа:


	- Нажмите кнопку ![](sheet_actions.png) «Действия»
	 на вкладке соответствующего листа.


	- В раскрывшемся списке выберите пункт «Создать
	 копию».


Рядом с копируемым листом будет добавлена его копия - новый лист с наименованием
 по умолчанию. По умолчанию наименование листа «ЛистN(M)»,
 где N - порядковый номер копируемого листа, M - номер копии.


## Особенности при копировании листов отчета


При копировании листа отчета существует возможность создания копий всех
 объектов, расположенных на листе отчета.


При копировании ячейки с формулой производится корректировка формулы:
 если в формуле вида «"Лист1"!A0»
 есть ссылка на этот же лист, то формула будет приводиться к виду «"Лист1(1)"!A0». При наличии
 ссылки на какой-либо другой лист формула не корректируется.
