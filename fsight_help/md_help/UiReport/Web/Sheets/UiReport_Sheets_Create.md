# Добавление листа отчёта


Для быстрого добавления листа в отчёт нажмите кнопку ![](../Table/Attribute/b_add.png) «Добавить лист», расположенную рядом
 с [вкладками листов](UiReport_Sheets.htm#sheets), или нажмите
 сочетание клавиш SHIFT+F11. Будет добавлен лист с наименованием по умолчанию.
 По умолчанию наименование листа «Лист
 N», где N - порядковый номер добавляемого листа.
