# Скрытие листа


Скрытие применяют для листов, содержащих промежуточные итоги. Вкладки
 скрытых листов не отображаются.


Для скрытия листа:


	- Нажмите кнопку ![](sheet_actions.png) «Действия»
	 на вкладке соответствующего листа.


	- В раскрывшемся списке выберите пункт «Скрыть».
	 Вкладка листа будет скрыта.


Для отображения скрытого листа:


	- Нажмите кнопку ![](all_sheets.png) «Все
	 листы».


	- В раскрывшемся списке щёлкните по листу, который нужно отобразить.
	 Вкладка соответствующего листа будет отображена.
