# Удаление листа


Для удаления листа:


	- Нажмите кнопку ![](sheet_actions.png) «Действия»
	 на вкладке соответствующего листа.


	- В раскрывшемся списке выберите пункт «Удалить».
	 Лист будет удалён.


При удалении листа осуществляется автоматическая корректировка формул,
 ссылающихся на удаленный лист. Например, формула «=B0+1+"Лист2"!A0»
 после удаления листа «Лист2» принимает
 вид: «=B0+1+#ССЫЛКА!A0».
