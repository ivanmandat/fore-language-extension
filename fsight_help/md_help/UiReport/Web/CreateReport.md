# Построение отчёта


Для построения отчёта выполните шаги:


Примечание.
 Для ускорения нахождения и применения команд доступен [поиск
 по командам меню](Reports/OperationReport/Search.htm).


	- Настройте [панель
	 инструментов](organizational_management/Toolbar_settings.htm).


	- Настройте [параметры
	 отчёта](Reports/UiReport_Reports_Param1.htm).


	- Добавьте [новый
	 лист](Sheets/UiReport_Sheets.htm).


	- Выберите [источник данных](Source/UiReport_Source.htm)
	 отчёта.


	- Создайте [срез
	 данных](Source/Window/UiReport_Source_Window_AreaSource.htm#add_slice).


	- Визуализируйте [данные на
	 листе](Reports/Visualization.htm) отчёта.


	- Добавьте интерактивные [элементы
	 управления](Source/AddControls.htm) содержимым отчёта.


	- Вставьте [объекты](Objects/UiReport_Objects.htm).


	- Настройте [оформление](Table/Work/UiReport_Table_WorkStyle.htm)
	 отчёта.


	- Настройте [обработчики
	 событий отчёта](Reports/Event/UiReport_Reports_Event.htm).


	- Создайте [копию
	 отчёта](Reports/OperationReport/UiReport_Reports_Operation_Save.htm).


После выполнения действий сохраните изменения регламентного
 отчёта с помощью кнопки «Сохранить»
 [главного
 меню](organizational_management/Starting.htm#structure_window). Сохранение нового отчёта аналогично [созданию
 копии](Reports/OperationReport/UiReport_Reports_Operation_Save.htm).


Если данные отчёта были изменены, то при сохранении отчёта будет предложено
 [сохранить изменённые
 данные в источник](Reports/OperationReport/Work_with_data.htm#edit).
