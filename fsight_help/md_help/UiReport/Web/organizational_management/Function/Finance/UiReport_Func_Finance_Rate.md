-


# Rate


## Синтаксис


Rate(PeriodCount, PeriodPayment, CurrentCost, FutureValue,
 Type[, Guess])


## Параметры


PeriodCount. Общее число периодов
 платежей по аннуитету;


PeriodPayment. Регулярный платеж
 (один раз в период), величина которого остается постоянной в течение всего
 срока аннуитета. Обычно состоит из платежа основной суммы и платежа процентов,
 но не включает других сборов или налогов;


CurrentCost. Приведенная к
 текущему моменту стоимость или общая сумма, которая на текущий момент
 равноценна ряду будущих платежей;


FutureValue. Требуемое значение
 будущей стоимости или остатка средств после последней выплаты;


Type. Выбор времени платежа.
 Данный параметр может принимать следующие значения:


	- 0. Платеж в конце периода;


	- 1. Платеж в начале периода;


Guess. Необязательный параметр.
 Предполагаемая величина ставки. Значение по умолчанию 0,01.


Примечание.
 В качестве параметра можно указывать как непосредственно значение параметра,
 так и адрес ячейки, в которой оно располагается.


## Описание


Возвращает процентную ставку по аннуитету за один период.


## Комментарии


Если функция Rate не сходится,
 попробуйте подставить различные значения для параметра Guess.
 Rate обычно сходится, если значение
 Guess находится между числами
 0 и 1.


Функция Rate вычисляется путем
 итерации и может давать нулевое значение или несколько значений. Если
 последовательные результаты функции не сходятся с точностью 0,0000001
 после 20-ти итераций, то Rate
 возвращает сообщение об ошибке.


## Пример


		 Формула
		 Результат
		 Описание


		 =Rate(12, -100, 1000, 0, 1, 0.01)
		 0,04
		 Процентная ставка по аннуитету за один период, в соответствии
		 со следующими условиями:


			- общее число периодов 12;


			- регулярный платеж -100;


			- текущая стоимость 1000;


			- будущая стоимость 0;


			- платеж совершается в начале периода;


			- предполагаемая величина ставки 0,01.


		 =Rate(12, A0, B0, 0, 0, 0.03)
		 0,03
		 Процентная ставка по аннуитету за один период, в соответствии
		 со следующими условиями:


			- общее число периодов 12;


			- регулярный платеж указан в ячейке A0, значение -154.1;


			- текущая стоимость указана в ячейке B0, значение 1500;


			- будущая стоимость 0;


			- платеж совершается в конце периода;


			- предполагаемая величина ставки 0,03.
