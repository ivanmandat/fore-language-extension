# Финансовые функции


Список предустановленных финансовых функций приведен в таблице:


		 Функция
		 Краткое описание


		 [Accrint](Accrint.htm)
		 Возвращает накопленный процент по ценным бумагам с периодической
		 выплатой процентов.


		 [AccrintM](UiReport_Func_Finance_AccrintM.htm)
		 Возвращает накопленный процент по ценным бумагам, процент по
		 которым выплачивается в срок погашения.


		 [AmorDegrC](AmorDegrC.htm)
		 Возвращает величину амортизации для каждого периода.


		 [AmorLinC](AmorLinC.htm)
		 Возвращает величину амортизации для каждого периода.


		 [CoupDayBs](UiReport_Func_Finance_CoupDayBs.htm)
		 Возвращает количество дней от начала действия купона до даты
		 соглашения.


		 [CoupDays](UiReport_Func_Finance_CoupDays.htm)
		 Возвращает число дней в периоде купона, который содержит дату
		 расчета.


		 [CoupDaysNc](UiReport_Func_Finance_CoupDaysNc.htm)
		 Возвращает число дней от даты расчета до срока следующего купона.


		 [CoupNcd](UiReport_Func_Finance_CoupNcd.htm)
		 Возвращает число, представляющее следующую дату купона после
		 даты расчета.


		 [CoupNum](UiReport_Func_Finance_CoupNum.htm)
		 Возвращает количество купонов, которые могут быть оплачены
		 между датой расчета и сроком погашения, округленное до ближайшего
		 целого количества купонов.


		 [CoupPcd](UiReport_Func_Finance_CoupPcd.htm)
		 Возвращает число, соответствующее предыдущей дате купона перед
		 датой расчета.


		 [CumIpmt](UiReport_Func_Finance_Cumipmt.htm)
		 Возвращает кумулятивную (нарастающим итогом) величину процентов,
		 выплачиваемых по займу в промежутке между двумя периодами выплат.


		 [Cumprinc](UiReport_Func_Finance_Cumprinc.htm)
		 Возвращает кумулятивную, нарастающим итогом, сумму, выплачиваемую
		 в погашение основной суммы займа в промежутке между двумя периодами.


		 [Db](UiReport_Func_Finance_Db.htm)
		 Возвращает величину амортизации актива для заданного периода,
		 рассчитанную методом фиксированного уменьшения остатка.


		 [Ddb](UiReport_Func_Finance_Ddb.htm)
		 Возвращает значение амортизации актива за данный период, используя
		 метод двойного уменьшения остатка или иной явно указанный метод.


		 [Disc](UiReport_Func_Finance_Disc.htm)
		 Возвращает ставку дисконтирования для ценных бумаг.


		 [DollarDe](UiReport_Func_Finance_DollarDe.htm)
		 Преобразует цену в рублях, выраженную в виде дроби, в цену
		 в рублях, выраженную в десятичным числом.


		 [DollarFr](UiReport_Func_Finance_DollarFr.htm)
		 Преобразует цену в рублях, выраженную десятичным числом, в
		 цену в рублях, выраженную в виде дроби.


		 [Duration](Duration.htm)
		 Возвращает продолжительность Маколея для предполагаемой номинальной
		 стоимости 100 рублей.


		 [Effect](UiReport_Func_Finance_Effect.htm)
		 Возвращает эффективную, фактическую, годовую процентную ставку,
		 если заданы номинальная годовая процентная ставка и количество
		 периодов в году, за которые начисляются сложные проценты.


		 [Fv](UiReport_Func_Finance_Fv.htm)
		 Возвращает будущую стоимость инвестиции на основе периодических
		 постоянных (равных по величине сумм) платежей и постоянной процентной
		 ставки.


		 [FvSchedule](UiReport_Func_Finance_FvSchedule.htm)
		 Возвращает будущую стоимость первоначальной основной суммы
		 после применения ряда (плана) ставок сложных процентов.


		 [Intrate](UiReport_Func_Finance_Intrate.htm)
		 Возвращает процентную ставку для полностью инвестированных
		 ценных бумаг.


		 [Ipmt](UiReport_Func_Finance_Ipmt.htm)
		 Возвращает сумму платежей процентов по инвестиции за данный
		 период на основе постоянства сумм периодических платежей и постоянства
		 процентной ставки.


		 [Irr](UiReport_Func_Finance_Irr.htm)
		 Возвращает внутреннюю ставку доходности для ряда потоков денежных
		 средств, представленными их численными значениями.


		 [Ispmt](UiReport_Func_Finance_Ispmt.htm)
		 Возвращает проценты, выплачиваемые за определенный инвестиционный
		 период.


		 [MDuration](MDuration.htm)
		 Возвращает модифицированную продолжительность Маколея для ценных
		 бумаг с предполагаемой номинальной стоимостью 100 руб.


		 [MIrr](UiReport_Func_Finance_MIrr.htm)
		 Возвращает модифицированную внутреннюю ставку доходности для
		 ряда периодических денежных потоков.


		 [Nominal](UiReport_Func_Finance_Nominal.htm)
		 Возвращает номинальную годовую ставку, если заданы эффективная
		 (фактическая) ставка и число периодов в году, за которые начисляются
		 сложные проценты.


		 [NPer](UiReport_Func_Finance_NPer.htm)
		 Возвращает общее количество периодов выплаты для инвестиции
		 на основе периодических постоянных выплат и постоянной процентной
		 ставки.


		 [Npv](UiReport_Func_Finance_Npv.htm)
		 Возвращает величину чистой приведенной стоимости инвестиции,
		 используя ставку дисконтирования, а также стоимости будущих выплат
		 (отрицательные значения) и поступлений (положительные значения).


		 [OddfPrice](OddfPrice.htm)
		 Возвращает цену за 100 рублей номинальной стоимости ценных
		 бумаг для нерегулярного, короткого или длинного, первого периода.


		 [OddfYield](OddfYield.htm)
		 Возвращает доход по ценным бумагам с нерегулярным (коротким
		 или длинным) первым периодом.


		 [OddlPrice](OddlPrice.htm)
		 Возвращает цену за 100 рублей номинальной стоимости ценных
		 бумаг для нерегулярного (короткого или длинного) последнего периода
		 купона.


		 [OddlYield](OddlYield.htm)
		 Возвращает доход по ценным бумагам с нерегулярным (коротким
		 или длинным) последним периодом.


		 [Pmt](UiReport_Func_Finance_Pmt.htm)
		 Возвращает сумму периодического платежа для аннуитета на основе
		 постоянства сумм платежей и постоянства процентной ставки.


		 [Ppmt](UiReport_Func_Finance_Ppmt.htm)
		 Возвращает величину платежа в погашение основной суммы по инвестиции
		 за данный на основе постоянства периодических платежей и постоянства
		 процентной ставки.


		 [Price](UiReport_Func_Finance_Price.htm)
		 Возвращает цену за 100 рублей номинальной стоимости ценных
		 бумаг, по которым выплачивается периодический процент.


		 [PriceDisc](UiReport_Func_Finance_PriceDisc.htm)
		 Возвращает цену за 100 рублей номинальной стоимости ценных
		 бумаг, на которые сделана скидка.


		 [PriceMat](UiReport_Func_Finance_PriceMat.htm)
		 Возвращает цену за 100 рублей номинальной стоимости ценных
		 бумаг, по которым процент выплачивается в срок погашения.


		 [Pv](UiReport_Func_Finance_Pv.htm)
		 Возвращает приведенную (к текущему моменту) стоимость инвестиции.


		 [Rate](UiReport_Func_Finance_Rate.htm)
		 Возвращает процентную ставку по аннуитету за один период.


		 [Received](UiReport_Func_Finance_Received.htm)
		 Возвращает сумму, полученную к сроку погашения полностью обеспеченных
		 ценных бумаг.


		 [RedemptionYield](RedemptionYield.htm)
		 Возвращает доход по ценной бумаге при ее погашении с нерегулярными
		 периодами выплат.


		 [Sln](UiReport_Func_Finance_Sln.htm)
		 Возвращает величину амортизации актива за один период, рассчитанную
		 линейным методом.


		 [Syd](UiReport_Func_Finance_Syd.htm)
		 Возвращает величину амортизации актива за данный период, рассчитанную
		 методом «суммы (годовых) чисел».


		 [TBillEq](UiReport_Func_Finance_TBillEq.htm)
		 Возвращает эквивалентный облигации доход по казначейскому векселю.


		 [TBillPrice](UiReport_Func_Finance_TBillPrice.htm)
		 Возвращает цену на 100 руб. номинальной стоимости для
		 казначейского векселя.


		 [TBillYield](UiReport_Func_Finance_TBillYield.htm)
		 Возвращает доход по казначейскому векселю.


		 [Vdb](UiReport_Func_Finance_Vdb.htm)
		 Возвращает величину амортизации актива для любого выбранного
		 периода, в том числе для частных периодов, с использованием метода
		 двойного уменьшения остатка или иного указанного метода.


		 [XIrr](UiReport_Func_Finance_XIrr.htm)
		 Возвращает внутреннюю ставку доходности для графика денежных
		 потоков, которые не обязательно носят периодический характер.


		 [Xnpv](UiReport_Func_Finance_Xnpv.htm)
		 Возвращает чистую приведенную стоимость для денежных потоков,
		 которые не обязательно являются периодическими.


		 [YieldDisc](YieldDisc.htm)
		 Возвращает годовую доходность по ценным бумагам, на которые
		 сделана скидка.


		 [YieldF](UiReport_Func_Finance_YieldF.htm)
		 Возвращает доходность ценных бумаг, по которым производятся
		 периодические выплаты процентов.


		 [YieldMat](UiReport_Func_Finance_YieldMat.htm)
		 Возвращает годовую доходность ценных бумаг, по которым проценты
		 выплачиваются при наступлении срока погашения.
