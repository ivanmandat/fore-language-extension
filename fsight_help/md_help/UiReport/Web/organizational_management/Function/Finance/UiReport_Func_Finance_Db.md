# Db: Регламентный отчёт, веб-приложение

Db: Регламентный отчёт, веб-приложение
-


# Db


## Синтаксис


Db(Cost, Salvage, Life, Period[, Month])


## Параметры


Cost. Затраты на приобретение
 актива;


Salvage. Стоимость в конце
 периода амортизации;


Life. Количество периодов,
 за которые собственность амортизируется;


Period. Период, для которого
 требуется вычислить амортизацию;


Month. Количество месяцев в
 первом году. Необязательный параметр. Значение по умолчанию 12.


Примечание.
 В качестве параметра можно указывать как непосредственно значение параметра,
 так и адрес ячейки, в которой оно располагается.


## Описание


Возвращает величину амортизации актива для заданного периода, рассчитанную
 методом фиксированного уменьшения остатка.


## Комментарии


Параметр Period должен быть
 задан в тех же единицах измерения, что и параметр Life.


Метод вычисляет амортизацию, используя фиксированную процентную ставку.
 Метод использует следующие формулы для вычисления амортизации за период:


(Cost - суммарная амортизация за предшествующие
 периоды) * Ставка,


где:


	- Ставка = 1 - ((Salvage / Cost)
	 ^ (1 / Life)), округленное до трех десятичных знаков после
	 запятой.


Особым случаем является амортизация за первый и последний периоды. Для
 первого периода Db использует
 такую формулу:


Cost * Ставка * Month / 12.


Для последнего периода Db использует
 такую формулу:


((Cost - суммарная амортизация за
 предшествующие периоды) * Ставка * (12 - Month)) / 12.


## Пример


		 Формула
		 Результат
		 Описание


		 =Db(5000, 100, 12, 1)
		 1390,00
		 Амортизация за первый период.


		 =Db(54000, 1200, 24, B6,7)
		 3842,13
		 Амортизация за период, указанный в ячейке B6. Ячейка B6
		 содержит значение 6.
