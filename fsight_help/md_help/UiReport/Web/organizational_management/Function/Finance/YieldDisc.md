# YieldDisc


## Синтаксис


YieldDisc(Settlement, Maturity, Price, Redemption[,
 Basis])


## Параметры


Settlement. Дата расчёта за
 ценные бумаги;


Maturity. Срок погашения ценных
 бумаг;


Price. Цена ценных бумаг за
 100 руб. номинальной стоимости. Значение параметра должно быть больше
 нуля;


Redemption. Выкупная стоимость
 ценных бумаг за 100 руб. номинальной стоимости. Значение параметра должно
 быть больше нуля;


Basis. Используемый способ
 вычисления дня. Задается в интервале от 0 до 4:


	- 0.
	 Способ вычисления дня американский. 360 дней (метод NSAD). Значение
	 по умолчанию;


	- 1.
	 Способ вычисления дня фактический/фактический;


	- 2.
	 Способ вычисления дня фактический/360 дней;


	- 3.
	 Способ вычисления дня фактический/365 дней;


	- 4.
	 Способ вычисления дня европейский 30/360 дней.


Необязательный параметр.


Примечание.
 В качестве параметра можно указывать как непосредственно значение параметра,
 так и адрес ячейки, в которой оно располагается.


## Описание


Возвращает годовую доходность по ценным бумагам, на которые сделана
 скидка.


## Комментарии


Значение параметра Settlement
 должно быть меньше, либо равно значению параметра Maturity.


## Пример


		 Формула
		 Результат
		 Описание


		 =YieldDisc("16.02.2008", "01.03.2008",
		 199, 200, 3)
		 0,131012
		 Годовая доходность по ценным бумагам, на которые сделана
		 скидка, на следующих условиях:


			- дата расчёта за ценные бумаги 16.02.2008;


			- срок погашения ценных бумаг 01.03.2008;


			- цена ценных бумаг 199;


			- выкупная стоимость ценных бумаг 200;


			- способ вычисления дня «фактический/365 дней».


		 =YieldDisc(A11, A2, A3, 100, 2)
		 0,052823
		 Годовая доходность по ценным бумагам, на которые сделана
		 скидка, на следующих условиях:


			- дата расчёта за ценные бумаги указана в ячейке A1, значение
			 16.02.2008;


			- срок погашения ценных бумаг указан в ячейке A2, значение
			 01.03.2008;


			- цена ценных бумаг указана в ячейке A3, значение 99,795;


			- выкупная стоимость ценных бумаг 100;


			- способ вычисления дня «фактический».
