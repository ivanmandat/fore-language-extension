# FvSchedule


## Синтаксис


FvSchedule(Principal, Schedule)


## Параметры


Principal. Стоимость инвестиций
 на текущий момент;


Schedule. Диапазон ячеек, содержащий
 массив переменных процентных ставок.


Примечание.
 В качестве параметра можно указывать как непосредственно значение параметра,
 так и адрес ячейки, в которой оно располагается.


## Описание


Возвращает будущую стоимость первоначальной основной суммы после применения
 ряда (плана) ставок сложных процентов.


## Комментарии


Функция используется для вычисления будущей стоимости инвестиции с переменной
 процентной ставкой.


## Пример


		 Формула
		 Результат
		 Описание


		 =FvSchedule(800,A0:C0)
		 1161,50
		 Будущая стоимость первоначальной основной суммы после применения
		 ряда (плана) ставок сложных процентов, указанных в диапазоне ячеек
		 A0:C0. В диапазоне A0:C0 находятся следующие значения: 0,01, 0,15,
		 0,25.


		 =FvSchedule(B6,A0:C0)
		 1797,21
		 Будущая стоимость первоначальной основной суммы, указанной
		 в ячейке B6, после применения ряда (плана) ставок сложных процентов,
		 указанных в диапазоне ячеек A0:C0. В диапазоне A0:C0 находятся
		 следующие значения: 0,1, 0,25, 0,25. Ячейка B6 содержит значение
		 1045,65.
