-


# Vdb


## Синтаксис


Vdb(Cost, Salvage, Life, StartPeriod, EndPeriod,
 Factor, NoSwitch


## Параметры


Cost. Затраты на приобретение
 актива;


Salvage. Стоимость в конце
 периода амортизации;


Life. Количество периодов,
 за которые актив амортизируется;


StartPeriod. Начальный период,
 для которого вычисляется амортизация;


EndPeriod. Конечный период,
 для которого вычисляется амортизация;


Factor. Процентная ставка снижающегося
 остатка;


NoSwitch. Логическое значение,
 определяющее, следует ли использовать линейную амортизацию в том случае,
 когда амортизация превышает величину, рассчитанную методом снижающегося
 остатка:


	- True. Использовать линейную
	 амортизацию;


	- False. Не использовать
	 линейную амортизацию.


Примечание.
 В качестве параметра можно указывать как непосредственно значение параметра,
 так и адрес ячейки, в которой оно располагается.


## Описание


Возвращает величину амортизации актива для любого выбранного периода,
 в том числе для частичных периодов, с использованием метода двойного уменьшения
 остатка или иного указанного метода.


## Комментарии


Параметры StartPeriod и EndPeriod должны быть заданы в
 тех же единицах измерения, что и параметр Life.


## Пример


		 Формула
		 Результат
		 Описание


		 =Vdb(5000, 100, 12, 1, 3, 2, true)
		 1273,15
		 Амортизация за три первых периода.


		 =Vdb(5000, 100, 12, B6, C7, 1.5, true)
		 740,69
		 Амортизация за диапазон периодов. Начало диапазона указано
		 в ячейке B6, окончание - в ячейке C7. Ячейка B6 содержит значение
		 6, C7 содержит значение 9.
