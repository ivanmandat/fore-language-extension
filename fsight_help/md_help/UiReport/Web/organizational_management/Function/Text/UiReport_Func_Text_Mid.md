# Mid


## Синтаксис


Mid(Text, Start_num, Num_chars)


## Параметры


Text. Исходная строка, из которой
 необходимо извлечь подстроку;


Start_num. Начальная позиция
 подстроки в исходной строке;


Num_chars. Количество символов
 в подстроке, которые необходимо извлечь из исходной строки.


Примечание.
 В качестве параметра можно указывать как непосредственно значение параметра,
 так и адрес ячейки, в которой оно располагается.


## Описание


Возвращает подстроку из строки по заданной начальной позиции и длине.


## Комментарии


Нумерация элементов в исходной строке начинается с нуля.


## Пример


		 Формула
		 Результат
		 Описание


		 =Mid("qwerty", 3, 2)
		 rt
		 Два символа из строки qwerty, начиная с третьего.


		 =Mid(B6, 0, 3)
		 asd
		 Подстрока длиной в три символа от начала исходной строки. Исходная
		 строка находится в ячейке B6. Ячейка B6 содержит значение asdfgh.
