# Right


## Синтаксис


Right(Text[, Num_chars])


## Параметры


Text. Исходная строка, из которой
 извлекаются символы;


Num_chars. Количество символов
 с конца строки, которое необходимо извлечь. Необязательный параметр. Значение
 по умолчанию 1.


Примечание.
 В качестве параметра можно указывать как непосредственно значение параметра,
 так и адрес ячейки, в которой оно располагается.


## Описание


Возвращает заданное число символов с конца строки.


## Пример


		 Формула
		 Результат
		 Описание


		 =Right("0123", 3)
		 123
		 Последние три символа строки 0123.


		 =Right(B6)
		 *
		 Последний символ строки в ячейке B6. Ячейка B6 содержит значение
		 12*.
