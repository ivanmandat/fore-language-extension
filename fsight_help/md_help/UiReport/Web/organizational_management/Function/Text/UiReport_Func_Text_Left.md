# Left


## Синтаксис


Left(Text[, Num_chars])


## Параметры


Text. Исходная строка, из которой
 извлекаются символы;


Num_chars. Количество символов,
 которое необходимо извлечь. Необязательный параметр. Значение по умолчанию
 1.


Примечание.
 В качестве параметра можно указывать как непосредственно значение параметра,
 так и адрес ячейки, в которой оно располагается.


## Описание


Возвращает указанное число символов от начала строки.


## Пример


		 Формула
		 Результат
		 Описание


		 =Left("zxcv", 2)
		 zx
		 Первые два символа строки zxcv.


		 =Left(B6)
		 М
		 Первый символ строки в ячейке B6. Ячейка B6 содержит значение
		 «Май».
