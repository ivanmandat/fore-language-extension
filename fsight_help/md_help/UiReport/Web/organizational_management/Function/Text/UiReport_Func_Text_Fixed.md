# Fixed


## Синтаксис


Fixed(Number[, Decimals[, No_commas])


## Параметры


Number. Число, которое необходимо
 округлить и конвертировать в текст;


Decimals. Число знаков после
 запятой. Количество десятичных разрядов, до которого требуется округлить
 число. Необязательный параметр. Значение по умолчанию 2;


No_commas. Определяет, отображать
 ли разделители групп разрядов в возвращаемом значении. Необязательный
 параметр. Допустимые значения:


	- TRUE. По умолчанию.
	 Разделители групп разрядов в возвращаемом значении не будут отображаться;


	- FALSE. Разделители групп
	 разрядов в возвращаемом значении будут отображаться.


Примечание.
 В качестве параметра можно указывать как непосредственно значение параметра,
 так и адрес ячейки, в которой оно располагается.


## Описание


Округляет число до заданного числа знаков после запятой и возвращает
 результат в виде строки с разделителями групп разрядов или без.


## Комментарии


Если Decimals больше 0, то
 число округляется до указанного количества десятичных разрядов справа
 от запятой. Если Decimals равно
 0, то число округляется до ближайшего целого. Если Decimals
 меньше 0, то число округляется слева от десятичной запятой.


## Пример


		 Формула
		 Результат
		 Описание


		 =Fixed(20.14,-1)
		 20
		 Округляет число 20,14 на 1 знак слева от десятичной запятой.


		 =Fixed(B6,C6)
		 20,1
		 Округляет число в ячейке B6 на количество знаков, указанных
		 в ячейке C6. Ячейка B6 содержит число 20,14, C6 содержит число
		 1.
