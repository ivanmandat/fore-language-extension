# Upper


## Синтаксис


Upper(Text)


## Параметры


Text. Строка, приводимая к
 верхнему регистру.


Примечание.
 В качестве параметра можно указывать как непосредственно значение параметра,
 так и адрес ячейки, в которой оно располагается.


## Описание


Возвращает результат преобразования строки к верхнему регистру.


## Пример


		 Формула
		 Результат
		 Описание


		 =Upper("AsDfGh")
		 ASDFGH
		 Строка AsDfGh, преобразованная к верхнему регистру.


		 =Upper("B6")
		 QWERTY
		 Строка в ячейке B6, преобразованная к верхнему регистру. Ячейка
		 B6 содержит значение QwertY.
