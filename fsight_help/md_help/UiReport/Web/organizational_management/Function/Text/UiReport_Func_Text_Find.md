# Find


## Синтаксис


Find(Find_text, Within_text[, Start_num])


## Параметры


Find_text. Подстрока, которую
 необходимо найти в исходной строке;


Within_text. Исходная строка,
 по которой осуществляется поиск;


Start_num. Необязательный параметр.
 Позиция элемента исходной строки, с которой осуществляется поиск. Если
 он не задан, поиск подстроки в исходной строке будет осуществляться с
 начала исходной строки.


Примечание.
 В качестве параметра можно указывать как непосредственно значение параметра,
 так и адрес ячейки, в которой оно располагается.


## Описание


Возвращает позицию одной строки в другой.


## Комментарии


Поиск осуществляется, начиная с указанного элемента исходной строки
 с учетом регистра. Если исходной строки нет в искомой, функция возвращает
 «#ЗНАЧ». Нумерация элементов в исходной строке начинается с 1.


## Пример


		 Формула
		 Результат
		 Описание


		 =Find("b", "BcBcab")
		 6
		 Возвращает индекс первого вхождения строки b в исходную строку
		 BcBcab, начиная с первого элемента исходной строки.


		 =Find("b", "zzzzZ")
		 #ЗНАЧ
		 В исходной строке zzzzZ не найдена подстрока b.


		 =Find(B6, C6, E6)
		 5
		 Возвращает индекс первого вхождения строки в ячейке B6 в исходную
		 строку C6, начиная с элемента исходной строки, номер которого
		 расположен в ячейке E6. Ячейка B6 содержит значение A, C6 - AcbaA,
		 E6 содержит число 2.
