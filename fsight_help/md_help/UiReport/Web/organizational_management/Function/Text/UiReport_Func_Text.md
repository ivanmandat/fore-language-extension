# Текстовые функции


Список предустановленных текстовых функций приведен в таблице:


		 Функция
		 Краткое описание


		 [_T](UiReport_Func_Text_T.htm)
		 Определяет, может ли значение быть преобразовано к строковому
		 типу.


		 [Concatenate](UiReport_Func_Text_Concatenate.htm)
		 Возвращает результат объединения двух или более текстовых строк
		 в одну.


		 [Exact](UiReport_Func_Text_Exact.htm)
		 Возвращает результат проверки полного совпадения двух строк
		 текста.


		 [Find](UiReport_Func_Text_Find.htm)
		 Возвращает позицию одной строки в другой.


		 [Fixed](UiReport_Func_Text_Fixed.htm)
		 Возвращает результат округления числа до заданного количества
		 знаков после запятой.


		 [Left](UiReport_Func_Text_Left.htm)
		 Возвращает указанное число символов от начала строки.


		 [Len](UiReport_Func_Text_Len.htm)
		 Возвращает длину строки.


		 [Lower](UiReport_Func_Text_Lower.htm)
		 Возвращает результат преобразования символов строки к нижнему
		 регистру.


		 [Mid](UiReport_Func_Text_Mid.htm)
		 Возвращает подстроку из строки по заданной начальной позиции
		 и длине.


		 [Replace](UiReport_Func_Text_Replace.htm)
		 Заменяет часть одной строки другой строкой.


		 [Rept](UiReport_Func_Text_Rept.htm)
		 Возвращает строку, повторенную заданное число раз.


		 [Right](UiReport_Func_Text_Right.htm)
		 Возвращает заданное число символов с конца строки.


		 [Trim](UiReport_Func_Text_Trim.htm)
		 Удаляет пробелы в начале и в конце указанной строки.


		 [Upper](UiReport_Func_Text_Upper.htm)
		 Возвращает результат преобразования строки к верхнему регистру.


		 [Value](UiReport_Func_Text_Value.htm)
		 Возвращает результат преобразования текстового представления
		 числа к числовому виду.
