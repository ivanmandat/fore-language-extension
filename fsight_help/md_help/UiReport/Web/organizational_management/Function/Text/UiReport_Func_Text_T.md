# _T


## Синтаксис


_T(Value)


## Параметры


Value. Проверяемое значение.


Примечание.
 В качестве параметра можно указывать как непосредственно значение параметра,
 так и адрес ячейки, в которой оно располагается.


## Описание


Определяет, может ли значение быть преобразовано к строковому типу.


## Комментарии


Если да, то возвращает текст, если нет - пустую строку.


## Пример


		 Формула
		 Результат
		 Описание


		 =_T(19)
		 Пустая строка
		 Возвращает значение 19 в текстовом виде.


		 =_T(B6)
		 Мир
		 Возвращает текст «Мир», так как параметр Value,
		 заданный значением в ячейке B6, является текстом. Ячейка B6 содержит
		 значение «Мир».
