# VLookUp


## Синтаксис


VLookUp(LookupValue, TableArray, ColumnIndexNum[,
 RangeLookup])


## Параметры


LookupValue. Значение, используемое
 при поиске нужного значения в массиве;


TableArray. Диапазон ячеек,
 в котором производится поиск;


ColumnIndexNum. Номер столбца
 в диапазоне, из которого должно быть возвращено сопоставляемое значение;


RangeLookup. Необязательный
 параметр. Указывает, как должен производиться поиск:


	- True.
	 Поиск производится приближенно. Значение по умолчанию;


	- False. Поиск производится
	 точно.


## Описание


Выполняет поиск значения в крайнем левом столбце таблицы и возвращает
 значение ячейки, находящейся в том же столбце в заданной строке.


## Комментарии


Функция работает по столбцам. Нумерация столбцов начинается с 1.


Значение, используемое при поиске, может быть числом, текстом или логическим
 значением, либо ссылкой на один из этих типов.


## Пример


		 Формула
		 Результат
		 Описание


		 =VLookUp(A0, B0:C0, 2)
		 8
		 Возвращает значение ячейки, находящейся в определённой строке,
		 заданной А0 в столбце 2 в диапазоне ячеек от B0 до C0. В диапазоне
		 A0:C0 расположены следующие числа: 6, 4, 8.
