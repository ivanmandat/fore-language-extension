# Range


## Синтаксис


Range(Address)


## Параметры


Address. Адрес диапазона.


## Описание


Возвращает диапазон ячеек по ссылке.


## Комментарии


Если необходимо получить диапазон одной ячейки, то используйте функцию
 [Rng](UiReport_Func_LinkArray_Rng.htm).


## Пример


		 Формула
		 Результат
		 Описание


		 =sum(offset(Range(A0:C0),1,0))
		 6
		 Сумма чисел в диапазоне ячеек от A1 до C1. Данный диапазон
		 смещен на строку вниз от диапазона ячеек от A0 до C0 и содержит
		 следующие числа: 1, 2, 3.
