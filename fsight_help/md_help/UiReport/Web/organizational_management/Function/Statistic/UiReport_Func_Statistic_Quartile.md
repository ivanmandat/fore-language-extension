# Quartile


## Синтаксис


Quartile(Data, Quart)


## Параметры


Data. Интервал числовых значений,
 с которыми связаны вероятности. Значения в массиве данных должны быть
 больше нуля;


Quart. Значение, которое нужно
 вернуть. Допустимые значения берутся из диапазона [0, 4]:


	- 0. Минимальное значение;


	- 1. Первая квартиль (25-ая
	 процентиль);


	- 2. Значение медианы
	 (50-ая процентиль);


	- 3. Третья квартиль (75-ая
	 процентиль);


	- 4. Максимальное значение.


## Описание


Метод возвращает квартиль множества данных.


## Комментарии


Квартили часто используются при анализе продаж, чтобы разбить генеральную
 совокупность на группы.
