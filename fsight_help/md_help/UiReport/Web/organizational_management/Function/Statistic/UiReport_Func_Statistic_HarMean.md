# HarMean


## Синтаксис


HarMean(Values,…)


## Параметры


Values1, Values2, …, ValuesN.
 Множество аргументов, для которых вычисляется среднее.


## Описание


Метод возвращает среднее гармоническое множества данных.


## Комментарии


Среднее гармоническое - это величина, обратная к среднему арифметическому
 обратных величин. Среднее гармоническое всегда меньше среднего геометрического,
 которое всегда меньше среднего арифметического.
