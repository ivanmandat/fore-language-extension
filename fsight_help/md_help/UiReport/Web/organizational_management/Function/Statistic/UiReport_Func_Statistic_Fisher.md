# Fisher


## Синтаксис


Fisher(Value)


## Параметры


Value. Значение, которое требуется
 преобразовать. Допустимые значения берутся из диапазона (-1; 1).


## Описание


Метод возвращает преобразование Фишера для параметра Value.


## Комментарии


Это преобразование строит функцию, которая имеет нормальное, а не асимметричное
 распределение. Эта функция используется для тестирования гипотез с помощью
 коэффициента корреляции.
