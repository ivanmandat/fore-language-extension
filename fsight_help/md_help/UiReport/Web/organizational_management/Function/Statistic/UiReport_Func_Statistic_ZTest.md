# ZTest


## Синтаксис


ZTest(Data, Value, Sigma)


## Параметры


Data. Массив данных, с которыми
 сравнивается Value;


Value. Проверяемое значение;


Sigma. Известное стандартное
 отклонение генеральной совокупности.


## Описание


Метод возвращает двустороннее P-значение z-теста.


## Комментарии


Z-тест определяет стандартную
 оценку для Value по отношению
 к массиву данных и возвращает двустороннюю вероятность для нормального
 распределения.


Можно использовать эту функцию, чтобы оценить вероятность того, что
 конкретное наблюдение взято из конкретной генеральной совокупности.
