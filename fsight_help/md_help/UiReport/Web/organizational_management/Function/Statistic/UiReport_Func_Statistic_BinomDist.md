# BinomDist


## Синтаксис


BinomDist(Successes, Trials, Probability, Cumulative)


## Параметры


Successes. Количество успешных
 испытаний;


Trials. Число независимых испытаний;


Probability. Вероятность успеха
 каждого испытания. Значение должно удовлетворять ограничению: 1 >=
 Probability >= 0;


Cumulative. Логическое значение,
 определяющее форму функции:


	- True. Функция BinomDist возвращает интегральную
	 функцию распределения, т.е. вероятность того, что число успешных испытаний
	 не менее значения параметра Successes;


	- False. Функция
	 BinomDist возвращает функцию
	 распределения, т.е. вероятность того, что число успешных испытаний
	 в точности равно значению параметра Successes.


## Описание


Метод возвращает отдельное значение биномиального распределения.


## Комментарии


Функция используется в задачах с фиксированным числом тестов или испытаний,
 когда результатом любого испытания может быть только успех или неудача,
 испытания независимы, и вероятность успеха постоянна на протяжении всего
 эксперимента.
