# Correl


## Синтаксис


Correl(A1, A2)


## Параметры


A1. Первый массив значений;


A2. Второй массив значений.


Примечание.
 A1 и A2
 должны иметь равное количество точек данных.


## Описание


Метод возвращает коэффициент корреляции между A1
 и A2.


## Комментарии


Коэффициент корреляции используется для определения наличия взаимосвязи
 между двумя свойствами.
