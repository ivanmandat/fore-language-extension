# AveDev


## Синтаксис


AveDev(Values,…)


## Параметры


Values1, Values2, …, ValuesN.
 Набор чисел, для которых определяется среднее абсолютных отклонений.


## Описание


Метод возвращает среднее абсолютных значений отклонений точек данных
 от среднего.


## Комментарии


Данное значение является мерой разброса множества данных.
