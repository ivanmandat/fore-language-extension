# Standardize


## Синтаксис


Standardize(Value, Mean, StandartDeviation)


## Параметры


Value. Нормализуемое значение;


Mean. Среднее арифметическое
 распределения;


StandartDeviation. Стандартное
 отклонение распределения. Значение должно удовлетворять ограничению: StandartDeviation > 0.


## Описание


Метод возвращает нормализованное значение для распределения, характеризуемого
 средним и стандартным отклонением.
