# DispersionP


## Синтаксис


DispersionP(Values,…)


## Параметры


Values1, Values2, …, ValuesN.
 Массив числовых аргументов, соответствующих генеральной совокупности.


## Описание


Метод вычисляет дисперсию для генеральной совокупности.


## Комментарии


При вычислении предполагается, что аргументы представляют всю генеральную
 совокупность.
