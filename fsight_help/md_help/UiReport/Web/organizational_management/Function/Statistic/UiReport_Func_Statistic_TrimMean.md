# TrimMean


## Синтаксис


TrimMean(Data, Percent)


## Параметры


Data. Массив усредняемых значений;


Percent. Доля точек данных,
 исключаемых из вычислений. Допустимые значения берутся из диапазона [0,
 1].


## Описание


Метод возвращает среднее внутренности множества данных.


## Комментарии


Функция вычисляет среднее, отбрасывая заданный процент данных с экстремальными
 значениями. Можно использовать эту функцию, чтобы исключить из анализа
 выбросы.


Функция округляет количество выбрасываемых точек данных с недостатком
 до ближайшего целого, кратного 2. Если доля равна 0,1, то 10 процентов
 от 30 точек данных составляют 3 точки, но из соображений симметрии функция
 TrimMean исключит по одному значению
 из начала и конца множества.
