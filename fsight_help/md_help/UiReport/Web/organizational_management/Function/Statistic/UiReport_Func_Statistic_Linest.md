# Linest


## Синтаксис


Linest(KnownYs, KnownXs, HasConstant, ReturnStatistics)


## Параметры


KnownYs. Множество значений
 y, которые уже известны для соотношения y = m*x+b.
 Значения должны быть больше нуля;


KnownXs. Множество значений
 x, которые уже известны для соотношения y = m*x+b;


HasConstant. Логическое значение,
 которое указывает, требуется ли, чтобы константа b
 была равна нулю:


	- True. Константа b вычисляется обычным образом;


	- False. Константа b полагается равной 0, а значения
	 m подбираются так, чтобы
	 выполнялось равенство: y = mx;


ReturnStatistics. Логическое
 значение, которое указывает, требуется ли вернуть дополнительную статистику
 по регрессии:


	- True. Функция Linest возвращает дополнительную
	 регрессионную статистику;


	- False. Функция Linest возвращает только коэффициенты
	 m и постоянную b.


## Описание


Метод рассчитывает статистику для ряда с применением метода наименьших
 квадратов, чтобы вычислить прямую линию, которая наилучшим образом аппроксимирует
 имеющиеся данные.


## Комментарии


Функция возвращает массив, который описывает полученную прямую.


Если массив KnownYs имеет один
 столбец, то каждый столбец массива KnownXs
 интерпретируется как отдельная переменная. Если массив KnownYs
 имеет одну строку, то каждая строка массива KnownXs
 интерпретируется как отдельная переменная.


Массив KnownXs может содержать
 одно или несколько множеств переменных. Если используется только одна
 переменная, то KnownYs и KnownXs могут иметь любую форму,
 при условии, что они имеют одинаковую размерность. Если используется более
 одной переменной, то KnownYs
 должен быть вектором (то есть интервалом высотой в одну строку или шириной
 в один столбец)
