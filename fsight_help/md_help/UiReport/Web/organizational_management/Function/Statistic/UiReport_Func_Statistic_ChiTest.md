# ChiTest


## Синтаксис


ChiTest(ActualRange, ExpectedRange)


## Параметры


ActualRange. Интервал данных,
 которые содержат наблюдения, подлежащие сравнению с ожидаемыми значениями;


ExpectedRange. Интервал данных,
 который содержит отношение произведений итогов по строкам и столбцам к
 общему итогу.


Примечание.
 Параметры ActualRange и ExpectedRange должны совпадать
 по размерам.


## Описание


Метод возвращает значение для распределения хи-квадрат.


## Комментарии


Критерий χ2
 используется для определения того, подтверждается ли гипотеза экспериментом.
 Критерий χ2
 сначала вычисляет χ2-статистику,
 а затем суммирует разности между фактическими значениями и ожидаемыми
 значениями.
