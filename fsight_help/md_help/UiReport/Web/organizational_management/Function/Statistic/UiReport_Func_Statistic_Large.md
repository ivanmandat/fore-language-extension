# Large


## Синтаксис


Large(Data, Position)


## Параметры


Data. Массив данных, для которых
 определяется k-ое наибольшее
 значение. Длина массива должна быть больше нуля;


Position. Позиция (начиная
 с наибольшей) в массиве данных. Допустимые значения берутся из диапазона
 (0, длина Data].


## Описание


Метод возвращает k-ое по величине
 значение из множества данных.


## Комментарии


Функция позволяет выбрать значение по его относительному местоположению.
