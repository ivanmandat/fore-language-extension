# Percentile


## Синтаксис


Percentile(Data,Percent)


## Параметры


Data. Массив данных, который
 определяет относительное положение;


Percent. Значение процентили.
 Допустимые значения берутся из диапазона [0, 1].


## Описание


Метод возвращает k-ую процентиль
 для значений из интервала.


## Комментарии


Функция используется для определения порога приемлемости.


Если k не кратно 1/(n
 - 1), то функция Percentile производит
 интерполяцию для определения значения k-ой
 процентили.
