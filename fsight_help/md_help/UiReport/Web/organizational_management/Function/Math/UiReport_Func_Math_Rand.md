# Rand


## Синтаксис


Rand: Регламентный отчёт, веб-приложение()


## Параметры


Данная функция не содержит параметров.


## Описание


Возвращает равномерно распределённое случайное число, большее либо равное
 0 и меньшее 1.


## Комментарии


Новое случайное вещественное число возвращается при каждом вычислении
 листа.
