# Int


## Синтаксис


Int(Value)


## Параметры


Value. Вещественное число,
 округляемое до ближайшего меньшего целого.


Примечание.
 В качестве параметра можно указывать как непосредственно число, так и
 адрес ячейки, в которой оно располагается.


## Описание


Возвращает результат округления числа до ближайшего меньшего целого.


## Пример


		 Формула
		 Результат
		 Описание


		 =Int(9.7)
		 9
		 Округляет число 9,7 до ближайшего меньшего целого.


		 =Int(B6)
		 -7
		 Округляет число в ячейке B6 до ближайшего меньшего целого.
		 Ячейка B6 содержит число -6,1.
