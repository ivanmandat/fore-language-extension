# RndPermutation


## Синтаксис


RndPermutation(Value)


## Параметры


Value. Размерность искомого
 массива, наибольшее значение в нем. Значение параметра должно быть не
 меньше, чем единица.


Примечание.
 В качестве параметра можно указывать как непосредственно число, так и
 адрес ячейки, в которой оно располагается.


## Описание


Возвращает массив, являющийся произвольной перестановкой другого массива,
 элементами которого являются целые числа от 1 до значения передаваемого
 параметра.


## Пример


		 Формула
		 Результат
		 Описание


		 = RndPermutation(A1)
		 Меняется.
		 Случайное число между 1 и значением, содержащимся в ячейке
		 A1.


		 = RndPermutation(5)
		 Меняется.
		 Случайное число между 1 и 5.
