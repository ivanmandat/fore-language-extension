# Max


## Синтаксис


Max(Values1,...)


## Параметры


Values1, Values2,
 …, ValuesN. Диапазон ячеек, для
 которого требуется определить максимальное значение.


Примечание.
 В качестве параметра можно указывать как непосредственно число, так и
 адрес ячейки или диапазона ячеек, в которых оно располагается.


## Описание


Возвращает максимальное число в массиве ячеек.


## Комментарии


Если в настройках формул отчёта не установлен флажок «[Проверять
 ссылки на пустые ячейки](../../../Reports/UiReport_Reports_Param4.htm#error_checking)» и:


	- диапазон ячеек, для которого требуется вычислить максимальное
	 значение, полностью состоит из пустых значений, то функция возвращает
	 пустое значение;


	- диапазон ячеек, для которого требуется вычислить максимальное
	 значение, содержит несколько пустых значений, то вычисление функции
	 происходит без учёта пустых значений.


При установленном флажке «[Проверять
 ссылки на пустые ячейки](../../../Reports/UiReport_Reports_Param4.htm#error_checking)» пустые значения в ячейках расцениваются как
 нулевые.


## Пример


		 Формула
		 Результат
		 Описание


		 =Max(A7:A9, A11, A6)
		 270
		 Максимальное число в массиве, расположенного в ячейках A7:A9,
		 A11, A6. Ячейка A6 содержит число 180, A7 содержит число 2, A8
		 содержит число 270, A9 содержит число 50, A11 содержит число
		 100.


		 =Max(10,5,28,13)
		 28
		 Максимальное число в заданном массиве.
