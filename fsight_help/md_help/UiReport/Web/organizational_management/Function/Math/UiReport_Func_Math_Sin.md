# Sin


## Синтаксис


Sin(Value)


## Параметры


Value. Угол в радианах, синус
 которого требуется вычислить.


Примечание.
 В качестве параметра можно указывать как непосредственно число, так и
 адрес ячейки, в которой оно располагается.


## Описание


Возвращает синус числа.


## Пример


		 Формула
		 Результат
		 Описание


		 =Sin(-16.1)
		 0,3821
		 Синус угла -16.1 радиан.


		 =Sin(B6)
		 0,9906
		 Синус угла, значение которого расположено в ячейке B6. Ячейка
		 B6 содержит число 14.


Примечание.
 Если угол задан в градусах, используйте функцию [Radians](UiReport_Func_Math_Radians.htm),
 чтобы преобразовать его в радианы.
