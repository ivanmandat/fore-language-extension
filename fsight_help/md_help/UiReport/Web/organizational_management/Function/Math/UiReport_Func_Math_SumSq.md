# SumSq


## Синтаксис


SumSq(Values1, …)


## Параметры


Values1, Values2,
 …, ValuesN. Диапазон ячеек, для
 которого требуется определить сумму квадратов.


## Описание


Возвращает сумму квадратов аргументов в заданном интервале ячеек.


## Пример


		 Формула
		 Результат
		 Описание


		 =SumSq(B6:E6)
		 38
		 Сумма квадратов чисел в диапазоне ячеек от B6 до E6. В данном
		 диапазоне расположены следующие числа: 3; 4; 3,6; -0,2.
