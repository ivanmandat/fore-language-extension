# Log


## Синтаксис


Log(Value, Base)


## Параметры


Value. Положительное число,
 для которого вычисляется логарифм;


Base. Положительное число,
 которое является основанием логарифма.


Примечание.
 В качестве параметра можно указывать как непосредственно число, так и
 адрес ячейки, в которой оно располагается.


## Описание


Возвращает логарифм числа по заданному основанию.


## Пример


		 Формула
		 Результат
		 Описание


		 =Log(16, 2.1)
		 3,7370
		 Логарифм числа 16 по основанию 2,1.


		 =Log(B6, C6)
		 3
		 Логарифм числа в ячейке B6 по основанию, расположенному в ячейке
		 С6. Ячейка B6 содержит число 8, C6 содержит число 2.
