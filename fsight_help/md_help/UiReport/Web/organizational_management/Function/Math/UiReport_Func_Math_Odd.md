# Odd


## Синтаксис


Odd(Value)


## Параметры


Value. Округляемое значение.


Примечание.
 В качестве параметра можно указывать как непосредственно число, так и
 адрес ячейки, в которой оно располагается.


## Описание


Возвращает результат округления числа до ближайшего нечетного целого.


## Пример


		 Формула
		 Результат
		 Описание


		 =Odd(-1.7)
		 -3
		 Округляет число -1,7 до ближайшего нечетного целого.


		 =Odd(B6)
		 3
		 Округляет число в ячейке B6 до ближайшего нечетного целого.
		 Ячейка B6 содержит число 2,1.
