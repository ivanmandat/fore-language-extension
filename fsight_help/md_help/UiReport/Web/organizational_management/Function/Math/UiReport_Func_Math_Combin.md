# Combin


## Синтаксис


Combin(SetSize, SampleSize)


## Параметры


SetSize. Общее количество элементов
 множества. Допустимые значения находятся в диапазоне (0, ∞);


SampleSize. Количество элементов,
 выбираемых из множества. Допустимые значения находятся в диапазоне (0, SetSize].


Примечание.
 В качестве параметра можно указывать как непосредственно число, так и
 адрес ячейки, в которой оно располагается.


## Описание


Возвращает количество комбинаций для заданного числа объектов.


## Комментарии


Комбинацией считается любое множество или подмножество объектов, безотносительно
 к их порядку. Комбинации отличаются от перестановок, для которых порядок
 существен.


## Пример


		 Формула
		 Результат
		 Описание


		 =Combin(B6, C1)
		 15
		 Число объектов задано в ячейке B6, число объектов в каждой
		 комбинации задано в ячейке C1. В ячейке B6 находится число 15,
		 в ячейке C1 находится число 1.


		 =Combin(8, 2)
		 24
		 Возможные команды по два человека, которые могут быть сформированы
		 из восьми кандидатов.
