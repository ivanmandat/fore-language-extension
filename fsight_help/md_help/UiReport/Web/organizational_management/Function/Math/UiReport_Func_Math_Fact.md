# Fact


## Синтаксис


Fact(Value)


## Параметры


Value. Неотрицательное число,
 факториал которого нужно вычислить.


Примечание.
 В качестве параметра можно указывать как непосредственно число, так и
 адрес ячейки, в которой оно располагается.


## Описание


Возвращает факториал указанного числа.


## Комментарии


Если число, указанное в параметре, не является целым, то оно округляется
 до ближайшего целого числа.


## Пример


		 Формула
		 Результат
		 Описание


		 =Fact(5)
		 120
		 Факториал числа 5.


		 =Fact(B6)
		 2
		 Факториал числа, расположенного в ячейке B6. Ячейка B6 содержит
		 число 1,5.
