# Log10


## Синтаксис


Log10(Value)


## Параметры


Value. Положительное число,
 для которого вычисляется десятичный логарифм.


Примечание.
 В качестве параметра можно указывать как непосредственно число, так и
 адрес ячейки, в которой оно располагается.


## Описание


Возвращает десятичный логарифм заданного числа.


## Пример


		 Формула
		 Результат
		 Описание


		 =Log10(14.1)
		 1,1492
		 Десятичный логарифм числа 14,1.


		 =Log10(B6)
		 1
		 Десятичный логарифм числа в ячейке B6. Ячейка B6 содержит число
		 10.
