# Aggregate


## Синтаксис


Aggregate(FuncNumber, Reference1, ...)


## Параметры


FuncNumber. Номер функции метода
 агрегации, доступные номера функций:


	- [Average](UiReport_Func_Math_Average.htm).
	 1 для учёта скрытых значений, 101 для игнорирования скрытых значений;


	- [Max](UiReport_Func_Math_Max.htm).
	 2 для учёта скрытых значений, 102 для игнорирования скрытых значений;


	- [Min](UiReport_Func_Math_Min.htm).
	 3 для учёта скрытых значений, 103 для игнорирования скрытых значений;


	- [Sum](UiReport_Func_Math_Sum.htm).
	 4 для учёта скрытых значений, 104 для игнорирования скрытых значений;


Reference1, Reference2, …, ReferenceN.
 Именованный диапазон или ссылка, для которых требуется вычислить функцию.


Примечание.
 В качестве параметра можно указывать как непосредственно число, так и
 адрес ячейки или диапазона ячеек, в которых оно располагается.


## Описание


Возвращает результат агрегации заданным методом, с возможностью не учитывать
 скрытые значения при расчёте.


## Комментарии


Доступны следующие методы агрегации:


	- [Average](UiReport_Func_Math_Average.htm);


	- [Max](UiReport_Func_Math_Max.htm);


	- [Min](UiReport_Func_Math_Min.htm);


	- [Sum](UiReport_Func_Math_Sum.htm).


При установленном флажке «[Проверять ссылки
 на пустые ячейки](../../../Reports/UiReport_Reports_Param4.htm#error_checking)» пустые значения в ячейках расцениваются как нулевые.


## Пример


		 Формула
		 Результат
		 Описание


		 =Aggregate(103, A7:A9)
		 270
		 Максимальное число в массиве, расположенного в ячейках A7:A9,
		 без учета скрытых ячеек. Ячейка A7 содержит число 180, A8 (скрыта)
		 содержит число 350, A9 содержит число 270.


		 =Aggregate(4, A4:A10, B4)
		 1590
		 Сумма чисел в массиве A4:A10, плюс значение ячейки B4. Даже
		 при частичном скрытии ячеек.
