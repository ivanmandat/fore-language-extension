# ACosH


## Синтаксис


ACosH(Value)


## Параметры


Value. Любое вещественное число
 большее или равное 1, от которого требуется найти гиперболический арккосинус.


Примечание.
 В качестве параметра можно указывать как непосредственно число, так и
 адрес ячейки, в которой оно располагается.


## Описание


Возвращает гиперболический арккосинус числа.


## Комментарии


Гиперболический арккосинус числа - это значение, гиперболический косинус
 которого равен числу. Число должно быть больше или равно 1.


## Пример


		 Формула
		 Результат
		 Описание


		 = ACosH(В6)
		 1,4750
		 Гиперболический арккосинус числа, расположенного в ячейке B6.
		 Ячейка B6 содержит число 2,3.


		 = ACosH(10)
		 2,9932
		 Гиперболический арккосинус числа 10.
