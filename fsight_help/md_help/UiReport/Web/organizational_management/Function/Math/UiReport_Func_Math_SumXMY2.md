# SumXMY2


## Синтаксис


SumXMY2(X, Y)


## Параметры


X. Диапазон ячеек, в которых
 расположены числа первого массива;


Y. Диапазон ячеек, в которых
 расположены числа второго массива.


Примечание.
 Размерности первого и второго массива должны совпадать, иначе функция
 возвращает значение 0.


## Описание


Возвращает сумму квадратов разностей соответствующих значений в двух
 массивах.


## Пример


		 Формула
		 Результат
		 Описание


		 =SumXMY2(B6:E6, B7:E7)
		 39,96
		 Сумма квадратов разностей соответствующих значений в диапазонах
		 ячеек от B6 до E6 и от B7 до E7. В диапазоне B6:E6 расположены
		 следующие числа: 3; 4; 3,6; -0,2; а в диапазоне B7:E7 - 2; -1,4;
		 0,8; 1,2.
