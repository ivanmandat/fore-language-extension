# Gcd


## Синтаксис


Gcd(Values1, …)


## Параметры


Values1, Values2,
 …, ValuesN. Диапазон ячеек, наибольший
 общий делитель которых нужно найти.


Примечание.
 Если любое из чисел диапазона не является целым, то оно усекается до целого.


## Описание


Возвращает наибольший общий делитель двух или более целых чисел.


## Пример


		 Формула
		 Результат
		 Описание


		 =Gcd(B6:E6)
		 7
		 Наибольший общий делитель чисел, расположенных в диапазоне
		 ячеек от B6 до E6. В данном диапазоне расположены следующие числа:
		 21, 7, 14, 28.
