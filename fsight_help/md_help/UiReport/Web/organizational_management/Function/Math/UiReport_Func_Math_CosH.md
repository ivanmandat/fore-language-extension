# CosH


## Синтаксис


CosH(Value)


## Параметры


Value. Вещественное число,
 от которого требуется найти гиперболический косинус. Допустимые значение
 находятся в диапазоне [1, ∞).


Примечание.
 В качестве параметра можно указывать как непосредственно число, так и
 адрес ячейки, в которой оно располагается.


## Описание


Возвращает гиперболический косинус числа.


## Пример


		 Формула
		 Результат
		 Описание


		 =CosH(4)
		 27,3082
		 Гиперболический косинус числа 4.


		 =CosH(B6)
		 3,1075
		 Гиперболический косинус числа, расположенного в ячейке B6.
		 Ячейка B6 содержит число 1,8.
