# Radians


## Синтаксис


Radians(Value)


## Параметры


Value. Величина угла в градусах,
 которую требуется преобразовать.


Примечание.
 В качестве параметра можно указывать как непосредственно число, так и
 адрес ячейки, в которой оно располагается.


## Описание


Возвращает значение, преобразованное из градусов в радианы.


## Пример


		 Формула
		 Результат
		 Описание


		 =Radians(45)
		 0,7854
		 Угол 45 градусов в радианах.


		 =Radians(B6)
		 4,7124
		 Угол, значение которого расположено в ячейке B6, в радианах.
		 Ячейка B6 содержит число 270.
