# Cos


## Синтаксис


Cos(Angle)


## Параметры


Angle. Угол в радианах, для
 которого определяется косинус.


Примечание.
 В качестве параметра можно указывать как непосредственно число, так и
 адрес ячейки, в которой оно располагается.


## Описание


Возвращает косинус числа.


## Пример


		 Формула
		 Результат
		 Описание


		 =Cos(1.047)
		 0,5002
		 Косинус угла 1,047 радиан.


		 =Cos(B6)
		 0,4536
		 Косинус угла, значение которого расположено в ячейке B6. Ячейка
		 B6 содержит число -1,1.


Примечание.
 Если угол задан в градусах, используйте функцию [Radians](UiReport_Func_Math_Radians.htm),
 чтобы преобразовать его в радианы.
