# CotH


## Синтаксис


CotH(Value)


## Параметры


Value. Угол в радианах, гиперболический
 котангенс которого требуется вычислить. Гиперболический котангенс имеет
 точку разрыва 0о.


Примечание.
 В качестве параметра можно указывать как непосредственно число, так и
 адрес ячейки, в которой оно располагается.


## Описание


Возвращает гиперболический котангенс числа.


## Пример


		 Формула
		 Результат
		 Описание


		 = CotH(A1)
		 1,000
		 Гиперболический котангенс числа, расположенного в ячейке A1.
		 Ячейка A1 содержит число 6,2832.


		 = CotH(1,5708)
		 1,0903
		 Гиперболический котангенс числа 1,5708.


Примечание.
 Если угол задан в градусах, используйте функцию [Radians](UiReport_Func_Math_Radians.htm),
 чтобы преобразовать его в радианы.
