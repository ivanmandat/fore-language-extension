# Cot


## Синтаксис


Cot(Value)


## Параметры


Value. Угол в радианах, для
 которого определяется котангенс. Котангенс имеет точки разрыва: 0о, ± 180о,± 360о,...


Примечание.
 В качестве параметра можно указывать как непосредственно число, так и
 адрес ячейки, в которой оно располагается.


## Описание


Возвращает котангенс числа.


## Комментарии


Котангенс угла - это отношение косинуса этого угла к синусу этого угла.


## Пример


		 Формула
		 Результат
		 Описание


		 = Cot(A1)
		 1,000
		 Котангенс числа, расположенного в ячейке A1, в радианах. Ячейка
		 A1 содержит число 0,7854.


		 = Cot(1,5708)
		 0,000
		 Котангенс числа 1,5708.


Примечание.
 Если угол задан в градусах, используйте функцию [Radians](UiReport_Func_Math_Radians.htm),
 чтобы преобразовать его в радианы.
