# Count


## Синтаксис


Count(Values1, …)


## Параметры


Values1, Values2, …, ValuesN.
 Диапазон ячеек, для которого будет определено среднее значение.


Примечание.
 В качестве параметра можно указывать как непосредственно число, так и
 адрес ячейки или диапазона ячеек, в которых оно располагается.


## Описание


Возвращает количество элементов в массиве ячеек.


## Пример


		 Формула
		 Результат
		 Описание


		 = Count(A1:A3, A5:A6, B7)
		 6
		 Количество элементов в ячейках A1:A3, A5:A6 и B7.


		 = Count(10, 82, 13, 4, 51)
		 5
		 Количество элементов в массиве заданных чисел.
