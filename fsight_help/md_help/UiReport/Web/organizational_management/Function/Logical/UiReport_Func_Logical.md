# Логические функции


Список предустановленных логических функций приведен в таблице:


		 Функция
		 Краткое описание


		 [_And](UiReport_Func_Logical_And.htm)
		 Проверяет, все ли аргументы имеют значение «ИСТИНА»,
		 и возвращает значение «ИСТИНА», если истинны все аргументы.


		 [_Not](UiReport_Func_Logical_Not.htm)
		 Меняет логическое значение своего аргумента на противоположное.


		 [_Or](UiReport_Func_Logical_Or.htm)
		 Проверяет, имеет ли хотя бы один из аргументов значение «ИСТИНА»,
		 и возвращает значение «ИСТИНА» или «ЛОЖЬ».


		 [False](UiReport_Func_Logical_False.htm)
		 Возвращает логическое значение FALSE.


		 [IfError](IfError.htm)
		 Возвращает выражение, если оно не содержит ошибки, иначе указанное
		 значение.


		 [Ifs](Ifs.htm)
		 Проверяет соответствие одному или нескольким условиям и возвращает
		 значение для первого условия, принимающего значение ИСТИНА.


		 [IIf](UiReport_Func_Logical_IIf.htm)
		 Выполняет проверку условия.


		 [IsEmpty](UiReport_Func_Logical_IsEmpty.htm)
		 Возвращает TRUE, если
		 аргументом является ссылка на пустую ячейку, в противном случае
		 возвращает FALSE.


		 [IsErr](IsErr.htm)
		 Проверяет, является ли значение ошибкой и отличается ли от
		 недоступного #Н/Д.


		 [IsError](IsError.htm)
		 Проверяет, является ли значение ошибкой.


		 [IsLogical](IsLogical.htm)
		 Проверяет, является ли значение логическим.


		 [IsNA](IsNa.htm)
		 Проверяет, является ли значение недоступным #Н/Д.


		 [IsNonText](IsNonText.htm)
		 Проверяет, не является ли значение текстовым.


		 [IsNumber](IsNumber.htm)
		 Проверяет, является ли значение числовым.


		 [IsText](IsText.htm)
		 Проверяет, является ли значение текстовым.


		 [True](UiReport_Func_Logical_True.htm)
		 Возвращает логическое значение TRUE.
