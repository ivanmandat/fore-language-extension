# IsError


## Синтаксис


IsError(Value)


## Параметры


Value. Проверяемое значение.


## Описание


Проверяет, является ли значение ошибкой.


## Комментарии


Возвращает TRUE, если значение
 является ошибкой, в противном случае возвращает FALSE.


В качестве значения параметра могут быть числа, пустые ячейки, логические
 значения, текст, ссылки на ячейки, формулы.


## Пример


		 Формула
		 Результат
		 Описание


		 =IsError(C2:E9)
		 FALSE
		 Диапазон ячеек C2:E9 не содержит ошибок.
