# _And


## Синтаксис


_And(Logical_Value1, …)


## Параметры


Logical Value1, Logical Value2, …
 Logical ValueN. Определяет значения аргументов, которые проверяются
 данной функцией.


## Описание


Проверяет, все ли аргументы имеют значение «ИСТИНА», и возвращает значение
 «ИСТИНА», если истинны все аргументы.
