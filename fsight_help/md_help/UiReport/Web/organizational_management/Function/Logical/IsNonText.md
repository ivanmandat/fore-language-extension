# IsNonText


## Синтаксис


IsNonText(Value)


## Параметры


Value. Проверяемое значение.


## Описание


Проверяет, не является ли значение текстовым.


## Комментарии


Возвращает TRUE, если значение
 не является текстовым, в противном случае возвращает FALSE.


В качестве значения параметра могут быть числа, пустые ячейки, логические
 значения, текст, ссылки на ячейки, формулы.


Пустые ячейки не являются текстовыми.


## Пример


		 Формула
		 Результат
		 Описание


		 =IsNonText(C2:E9)
		 TRUE
		 Диапазон ячеек C2:E9 не содержит только текстовые значения.
