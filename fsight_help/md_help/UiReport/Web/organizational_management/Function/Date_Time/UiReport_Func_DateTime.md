# Дата и время


Список предустановленных функций для работы с датой и временем приведен
 в таблице:


		 Функция
		 Краткое описание


		 [Date](UiReport_Func_DateTime_Date.htm)
		 Возвращает дату по заданным году, месяцу и дню.


		 [DateValue](UiReport_Func_DateTime_DateValue.htm)
		 Возвращает результат преобразования строки в дату.


		 [Day](UiReport_Func_DateTime_Day.htm)
		 Возвращает номер дня в месяце.


		 [Edate](UiReport_Func_DateTime_Edate.htm)
		 Возвращает дату, увеличенную/уменьшенную на заданное число
		 месяцев.


		 [Hour](UiReport_Func_DateTime_Hour.htm)
		 Возвращает час, соответствующий заданному времени.


		 [Minute](UiReport_Func_DateTime_Minute.htm)
		 Возвращает минуты, соответствующие заданному времени.


		 [Month](UiReport_Func_DateTime_Month.htm)
		 Возвращает месяц в заданной дате.


		 [Now](UiReport_Func_DateTime_Now.htm)
		 Возвращает текущую дату и время.


		 [Second](UiReport_Func_DateTime_Second.htm)
		 Возвращает секунды, соответствующие заданному времени.


		 [Time](UiReport_Func_DateTime_Time.htm)
		 Возвращает время по указанным часам, минутам и секундам.


		 [TimeValue](UiReport_Func_DateTime_TimeValue.htm)
		 Возвращает результат преобразования строкового значения к типу
		 [TimeSpan](Fore.chm::/03_dataTypes/Fore_DataTypes.htm).


		 [Today](UiReport_Func_DateTime_Today.htm)
		 Возвращает дату на текущий момент времени.


		 [Weekday](UiReport_Func_DateTime_Weekday.htm)
		 Возвращает номер дня недели по указанной дате.


		 [Year](UiReport_Func_DateTime_Year.htm)
		 Возвращает год по заданной дате.
