# Time


## Синтаксис


Time(Hour, Minute, Second)


## Параметры


Hour. Положительное целое число
 в диапазоне от 0 до 23, представляющее часы;


Minute. Положительное целое
 число в диапазоне от 0 до 59, представляющее минуты;


Second. Положительное целое
 число в диапазоне от 0 до 59, представляющее секунды.


Примечание.
 В качестве параметра можно указывать как непосредственно значение параметра,
 так и адрес ячейки, в которой оно располагается.


## Описание


Возвращает время по указанным часам, минутам и секундам.


## Пример


		 Формула
		 Результат
		 Описание


		 =Time(23,15, 21);
		 23:15:21
		 Время, составленное из указанных часа (23), минут (15) и секунд
		 (21).


		 =Time(B6, B7, B8)
		 14:01:18
		 Время, составленное из часа, указанного в ячейке B6, минут,
		 указанных в ячейке B7 и секунд, указанных в ячейке B8. Ячейка
		 B6 содержит число 14, B7 содержит число 1, B8 содержит число 18.
