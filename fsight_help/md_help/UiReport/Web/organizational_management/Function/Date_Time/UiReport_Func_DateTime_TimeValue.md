# TimeValue


## Синтаксис


TimeValue(Date_text)


## Параметры


Date_text. Строка, которую
 необходимо преобразовать. Строка должна иметь следующий формат: «<Кол-во
 дней>.<Кол-во часов>:<Кол-во минут>:<Кол-во секунд>.<Кол-во
 миллисекунд>».


Примечание.
 В качестве параметра можно указывать как непосредственно значение параметра,
 так и адрес ячейки, в которой оно располагается.


## Описание


Возвращает результат преобразования строкового значения к типу [TimeSpan](Fore.chm::/03_dataTypes/Fore_DataTypes.htm).


## Комментарии


Особенности работы функции:


	- количество дней отсчитывается от даты 30.12.1899. Если указано
	 положительное количество дней, то полученная дата будет больше 30.12.1899,
	 если отрицательное - меньше;


	- количество часов должно быть задано числом в диапазоне [0, 23];


	- количество минут и секунд должно быть задано числом в диапазоне
	 [0, 59].


## Пример


		 Формула
		 Результат
		 Описание


		 =TimeValue("44444.23:47:45.441")
		 5 сентября 2021
		 Результат преобразования строки 44444.23:47:45.441 к типу [TimeSpan](Fore.chm::/03_dataTypes/Fore_DataTypes.htm).


		 =TimeValue(B6)
		 1 января 1990
		 Результат преобразования текста в ячейке B6 к типу [TimeSpan](Fore.chm::/03_dataTypes/Fore_DataTypes.htm).
		 Ячейка B6 содержит значение 2.0:0:0.0.
