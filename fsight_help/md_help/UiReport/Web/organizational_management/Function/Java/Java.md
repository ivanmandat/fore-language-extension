# Функции для работы с Java


Список предустановленных функций для работы с функциями, написанными
 на языке Java, приведён в таблице:


		 Функция
		 Краткое описание


		 [JavaInvoke](JavaInvoke.htm)
		 Возвращает результат выполнения статического Java-метода.


		 [JavaInvokeModule](JavaInvokeModule.htm)
		 Возвращает результат выполнения статического метода, хранящегося
		 в Java-модуле.


Примечание.
 Перед использованием функций, написанных на языке Java, убедитесь, что
 выполнена [интеграция](uinav.chm::/02_Navigator/CommonSettings/Integration.htm)
 Java и «Форсайт. Аналитическая платформа».
