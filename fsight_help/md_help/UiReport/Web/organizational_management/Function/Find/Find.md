# Поиск


Список предустановленных поисковых функций приведен в таблице:


		 Функция
		 Краткое описание


		 [Choose](Choose.htm)
		 Выбирает значение из списка значений по номеру индекса.


		 [Column](Column.htm)
		 Возвращает номер столбца, на который указывает ссылка.


		 [Columns](Columns.htm)
		 Возвращает количество столбцов в ссылке.


		 [Row](Row.htm)
		 Возвращает номер строки, на которую указывает ссылка.


		 [Rows](Rows.htm)
		 Возвращает количество строк в ссылке.
