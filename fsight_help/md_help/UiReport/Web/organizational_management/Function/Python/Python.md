# Функции для работы с Python


Список предустановленных функций для работы с функциями, написанными
 на языке Python, приведен в таблице:


		 Функция
		 Краткое описание


		 [PythonInvoke](PythonInvoke.htm)
		 Возвращает результат выполнения функции на языке Python.


		 [PythonInvokeModule](PythonInvokeModule.htm)
		 Возвращает результат выполнения функции, хранящейся в Python-модуле.


Примечание.
 Перед использованием функций, написанных на языке Python, убедитесь, что
 выполнена [интеграция](uinav.chm::/02_Navigator/CommonSettings/Integration.htm)
 Python и «Форсайт. Аналитическая платформа».
