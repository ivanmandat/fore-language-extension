# Корректировка ссылок в формулах


Если формула ссылается на диапазон [области
 данных](../Desktop/AreaData/UiReport_AreaData.htm) (в том числе [реляционной](../Desktop/Relational_data_area/UIReport_Relational_data_area.htm)),
 то:


	- при расширении диапазона области данных диапазон формулы не
	 расширяется и в формуле не происходит корректировка ссылок;


	- при сужении диапазона области данных происходит сужение диапазона
	 формулы и корректировка ссылок.


## Причина


При вычислении области данных вставка строк происходит ниже диапазона
 старого размещения.


## Решение


Вместо формул, ссылающихся на диапазон области данных, используйте механизм
 итогов.


Подробное описание приведено в разделах:


	- [Область
	 данных: Настройка итогов](../Desktop/AreaData/Param/UiReport_AreaData_Param_Totals.htm);


	- [Реляционная
	 область данных: Настройка итогов](../Desktop/Relational_data_area/Parameter/UiReport_Relation_Totals.htm).
