# Какие функции экспортируются в Microsoft Excel?


## Вопрос


Какие функции экспортируются в Microsoft Excel?


## Область применения


Формулы, настроенные в регламентном отчете, могут быть экспортированы
 в Microsoft Excel в виде формул. Если [функция](../Desktop/organizational_management/UiReport_Organizational_master_function.htm),
 входящая в формулу, не поддерживается Microsoft Excel, то в ячейке с формулой
 будет отображаться текстовое или числовое значение.


## Исходные данные


Регламентный отчет, содержащий [формулы](../Desktop/organizational_management/Function/UiReport_Function.htm).


## Решение


В Microsoft Excel [экспортируются](../Desktop/Reports/OperationReport/UiReport_Reports_Operation_Export.htm#export_to_excel)
