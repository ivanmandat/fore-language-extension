# Общие принципы программирования
 с использованием сборки Metabase


Сборка Metabase предназначена
 для получения доступа к репозиторию, управления его содержимым, администрирования,
 формирования и установки обновлений. Программирование на Fore
 с использованием данной сборки заключается в получении доступа к репозиторию
 и выполнения требуемых действий с объектами. Также, при работе с объектами,
 после их открытия или редактирования, можно получить доступ к интерфейсам
 со специализированными настройками в соответствии с типом объекта. Описание
 этих интерфейсов представлено в описании других сборок, доступных в языке
 Fore.
