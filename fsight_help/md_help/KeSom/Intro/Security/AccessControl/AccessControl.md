# Методы разграничения доступа


Методы разграничения доступа формируют правила, по которым для субъектов
 безопасности будет определяться возможность выполнения каких-либо действий
 в репозитории, доступ к объектам и их данным. Комбинированное использование
 различных методов доступа позволяет повысить надёжность всей системы и
 произвести настройки в соответствии с заявленными требованиями.


Перед использованием какого-либо метода доступа рекомендуется ознакомиться
 с из назначением, механизмами работы и особенностями в подразделе «[Выбор методов
 разграничения доступа и их настройка](Admin.chm::/04_SecurityPolicy/Admin_SecPolicy.htm)».
