# IMetabaseObjectCache.Refresh


## Синтаксис


Refresh;


## Описание


Метод Refresh осуществляет обновление списка записей в кеше объекта.


## Комментарии


При выполнении метода для каждой записи в кеше будет обновлен соответствующий экземпляр объекта. Обновление объекта осуществляется путем переоткрытия объекта репозитория с [параметрами](../IMetabaseObjectCacheItem/IMetabaseObjectCacheItem.ParamValues.htm), которым соответствуют запись в кеше.
