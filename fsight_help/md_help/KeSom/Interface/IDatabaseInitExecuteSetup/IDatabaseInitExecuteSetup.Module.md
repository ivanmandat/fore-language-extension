# IDatabaseInitExecuteSetup.Module


## Синтаксис


Module: [IMetabaseObjectDescriptor](../IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm);


## Описание


Свойство Module определяет объект
 среды разработки, содержащий реализацию метода, который будет запущен
 после подключения к базе данных.


## Комментарии


В качестве значения свойства необходимо указать описание модуля/формы/сборки.
 Наименование выполняемого метода задается в свойстве [Macro](IDatabaseInitExecuteSetup.Macro.htm).
