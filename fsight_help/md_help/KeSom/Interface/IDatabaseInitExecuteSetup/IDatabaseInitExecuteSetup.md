# IDatabaseInitExecuteSetup


Сборка: Metabase;


## Описание


Интерфейс IDatabaseInitExecuteSetup
 содержит свойства для определения параметров базы данных, при подключении
 к которой будет выполняться прикладной макрос.


## Иерархия наследования


IDatabaseInitExecuteSetup


## Комментарии


Данный интерфейс поддерживают все объекты репозитория - [База данных](UiNavObj.chm::/database/UiDb_database.htm).
