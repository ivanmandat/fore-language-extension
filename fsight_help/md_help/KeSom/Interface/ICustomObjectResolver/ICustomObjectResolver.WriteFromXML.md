# ICustomObjectResolver.WriteFromXML


## Синтаксис


WriteFromXML(Var Xml: Variant);


## Параметры


Xml. Объект, из которого будут
 получены данные для записи.


## Описание


Метод WriteFromXML записывает
 данные в пользовательский объект из формата XML.


## Комментарии


Для чтения данных из пользовательского объекта в формат XML используйте
 метод [ICustomObjectResolver.ReadToXML](ICustomObjectResolver.ReadToXML.htm).
