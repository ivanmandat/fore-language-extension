# ICustomObjectResolver.ReadToFile


## Синтаксис


ReadToFile(FileName: String);


## Параметры


FileName. Полное наименование
 файла, в который должны быть прочитаны данные.


## Описание


Метод ReadToFile читает
 данные пользовательского объекта в указанный файл.


## Комментарии


Для записи данных из файла в пользовательский объект используйте метод
 [ICustomObjectResolver.WriteFromFile](ICustomObjectResolver.WriteFromFile.htm).
