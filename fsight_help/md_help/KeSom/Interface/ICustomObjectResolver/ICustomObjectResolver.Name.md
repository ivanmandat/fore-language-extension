# ICustomObjectResolver.Name


## Синтаксис


Name: String;


## Описание


Свойство Name возвращает
 наименование объекта пользовательского класса.


## Комментарии


Для получения идентификатора объекта пользовательского класса используйте
 свойство [ICustomObjectResolver.Id](ICustomObjectResolver.Id.htm).
