# ICustomObjectResolver.WriteFromFile


## Синтаксис


WriteFromFile(FileName: String);


## Параметры


FileName. Полное наименование
 файла, из которого будут получены данные для записи.


## Описание


Метод WriteFromFile записывает
 данные в пользовательский объект из указанного файла.


## Комментарии


Для чтения данных из пользовательского объекта в указанный файл используйте
 метод [ICustomObjectResolver.ReadToFile](ICustomObjectResolver.ReadToFile.htm).
