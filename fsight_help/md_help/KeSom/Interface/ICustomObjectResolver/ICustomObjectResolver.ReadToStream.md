# ICustomObjectResolver.ReadToStream


## Синтаксис


ReadToStream(Stream: [IIOStream](ModIo.chm::/Interface/IIOStream/IIOStream.htm));


## Параметры


Stream. Поток, в который должны
 быть прочитаны данные.


## Описание


Метод ReadToStream читает данные
 пользовательского объекта в поток.


## Комментарии


Для записи данных из потока в пользовательский объект используйте метод
 [ICustomObjectResolver.WriteFromStream](ICustomObjectResolver.WriteFromStream.htm).
