# ICustomObjectResolver.ClassId


## Синтаксис


ClassId: Integer;


## Описание


Свойство ClassId возвращает
 индекс класса, которому принадлежит пользовательский объект.


## Комментарии


Индексы классов объектов содержатся в перечислении [MetabaseObjectClass](../../Enums/MetabaseObjectClass.htm).
