# ICustomObjectResolver.ReadToXML


## Синтаксис


ReadToXML(Var Xml: Variant);


## Параметры


Xml. Объект, в который будут
 прочитаны данные.


## Описание


Метод ReadToXML читает
 данные пользовательского объекта в формат XML.


## Комментарии


Для записи данных в пользовательский объект из формата XML используйте
 метод [ICustomObjectResolver.WriteFromXML](ICustomObjectResolver.WriteFromXML.htm).
