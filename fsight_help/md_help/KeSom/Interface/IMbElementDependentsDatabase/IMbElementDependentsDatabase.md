# IMbElementDependentsDatabase


Сборка: Metabase;


## Описание


Интерфейс IMbElementDependentsDatabase
 предназначен для работы с зависимостями элементов справочников НСИ от
 объектов репозитория.


## Иерархия наследования


           IMbElementDependentsDatabase


## Комментарии


Для работы с коллекцией записей зависимости элементов справочника НСИ
 от объектов репозитория используйте свойства и методы интерфейса [IMbElementDependents](../IMbElementDependents/IMbElementDependents.htm).
