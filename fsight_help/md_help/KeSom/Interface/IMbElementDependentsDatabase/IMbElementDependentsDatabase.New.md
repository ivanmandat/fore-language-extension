# IMbElementDependentsDatabase.New_


## Синтаксис


New_: [IMbElementDependents](../IMbElementDependents/IMbElementDependents.htm);


## Описание


Метод New_ создает новую коллекцию
 зависимостей элементов справочников НСИ от объектов репозитория.


## Комментарии


Для получения параметров зависимостей, соответствующих указанному шаблону,
 используйте метод [IMbElementDependentsDatabase.Select_](IMbElementDependentsDatabase.Select.htm).
