# IMbElementDependentsDatabase.Exists


## Синтаксис


Exists(Where: [IMbElementDependentTemplate](../IMbElementDependentTemplate/IMbElementDependentTemplate.htm)):
 Boolean;


## Параметры


Where. Шаблон, в соответствии
 с которым будет осуществлена проверка.


## Описание


Метод Exists осуществляет проверку
 наличия зависимостей элементов справочников НСИ от объектов репозитория.


## Комментарии


Метод возвращает значение True,
 если существует информация о зависимостях, соответствующих указанному
 шаблону.
