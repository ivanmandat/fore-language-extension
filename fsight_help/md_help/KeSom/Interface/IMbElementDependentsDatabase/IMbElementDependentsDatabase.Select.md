# IMbElementDependentsDatabase.Select_


## Синтаксис


Select_(Where: [IMbElementDependentTemplate](../IMbElementDependentTemplate/IMbElementDependentTemplate.htm)):
 [IMbElementDependents](../IMbElementDependents/IMbElementDependents.htm);


## Параметры


Where. Шаблон, в соответствии
 с которым осуществляется отбор зависимостей.


## Описание


Метод Select_ возвращает параметры
 зависимостей, соответствующих указанному шаблону.


## Комментарии


Если зависимостей, соответствующих указанному шаблону, нет, то метод
 вернет пустую коллекцию.
