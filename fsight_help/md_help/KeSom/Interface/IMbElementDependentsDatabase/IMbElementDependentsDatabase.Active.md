# IMbElementDependentsDatabase.Active


## Синтаксис


Active: Boolean;


## Описание


Свойство Active возвращает признак
 доступности сохранения зависимостей элементов справочников НСИ от объектов
 репозитория.


## Комментарии


Допустимые значения:


	- True. Доступно сохранение
	 зависимостей элементов справочников НСИ от объектов репозитория;


	- False. Сохранение зависимостей
	 элементов справочников НСИ от объектов репозитория не доступно.
