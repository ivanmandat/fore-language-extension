# IUserProfileProperties.Add


## Синтаксис


Add(Name: String; Value: Variant): [IUserProfileProperty](../IUserProfileProperty/IUserProfileProperty.htm);


## Параметры


Name. Наименование дополнительного свойства.


Value. Значение дополнительного свойства.


## Описание


Метод Add добавляет дополнительное свойство пользователя в коллекцию.


## Комментарии


В коллекции не может существовать несколько дополнительных свойств с одинаковым наименованием. Если при добавлении будет указано наименование существующего свойства, то будет изменено его значение.
