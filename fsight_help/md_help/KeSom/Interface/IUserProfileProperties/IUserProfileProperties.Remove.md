# IUserProfileProperties.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс дополнительного свойства.


## Описание


Метод Remove удаляет указанное дополнительное свойство и возвращает признак удачного удаления.


## Комментарии


Метод возвращает значение True, если удаление прошло удачно и False, если дополнительное свойство удалить не удалось.
