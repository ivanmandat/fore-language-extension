# IUserProfileProperties


Сборка: Metabase;


## Описание


Интерфейс IUserProfileProperties
 содержит свойства и методы для работы с коллекцией дополнительных свойств
 пользователя.


## Иерархия наследования


IUserProfileProperties


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IUserProfileProperties.Count.htm)
		 Свойство Count возвращает
		 количество дополнительных свойств в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IUserProfileProperties.Item.htm)
		 Свойство Item возвращает
		 параметры дополнительного свойства пользователя.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IUserProfileProperties.Add.htm)
		 Метод Add добавляет
		 дополнительное свойство пользователя в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Remove](IUserProfileProperties.Remove.htm)
		 Метод Remove удаляет
		 указанное дополнительное свойство и возвращает признак удачного
		 удаления.
