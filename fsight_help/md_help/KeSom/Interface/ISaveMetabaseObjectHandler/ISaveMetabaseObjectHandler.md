# ISaveMetabaseObjectHandler


Сборка: Metabase;


## Описание


Интерфейс ISaveMetabaseObjectHandler
 предназначен для настройки обработчика событий, наступающих при изменении
 метаданных объектов.


## Иерархия наследования


ISaveMetabaseObjectHandler


## Комментарии


Настройки обработчика событий можно получить, используя свойства и методы
 интерфейса [ISaveMetabaseObjectHandlers](../ISaveMetabaseObjectHandlers/ISaveMetabaseObjectHandlers.htm).
