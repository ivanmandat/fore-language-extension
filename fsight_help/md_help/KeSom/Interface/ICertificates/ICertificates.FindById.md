# ICertificates.FindById


## Синтаксис


FindById(Id: String): [ICertificate](../ICertificate/ICertificate.htm);


## Параметры


Id. Идентификатор сертификата.


## Описание


Метод FindById осуществляет
 поиск сертификата безопасности по идентификатору.


## Комментарии


Метод возвращает параметры сертификата в случае удачного поиска и Null, если сертификат с указанным
 идентификатором отсутствует.
