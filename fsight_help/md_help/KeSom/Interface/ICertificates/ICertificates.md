# ICertificates


Сборка: Metabase;


## Описание


Интерфейс ICertificates содержит
 свойства и методы для работы с коллекцией сертификатов безопасности.


## Иерархия наследования


ICertificates


## Комментарии


Для получения коллекции сертификатов безопасности используйте свойство
 [IMetabasePolicy.Certificates](../IMetabasePolicy/IMetabasePolicy.Certificates.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICertificates.Count.htm)
		 Свойство Count возвращает
		 количество сертификатов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ICertificates.Item.htm)
		 Свойство Item возвращает
		 параметры сертификата по его индексу в коллекции.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ICertificates.Add.htm)
		 Метод Add добавляет
		 сертификат в коллекцию и возвращает его параметры.


		 ![](../../Sub_Image.gif)
		 [Clear](ICertificates.Clear.htm)
		 Метод Clear удаляет
		 все сертификаты из коллекции.


		 ![](../../Sub_Image.gif)
		 [FindById](ICertificates.FindById.htm)
		 Метод FindById осуществляет
		 поиск сертификата безопасности по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ICertificates.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск сертификата безопасности по ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](ICertificates.Remove.htm)
		 Метод Remove удаляет
		 сертификат по индексу и возвращает признак удачного удаления.


		 ![](../../Sub_Image.gif)
		 [RemoveById](ICertificates.RemoveById.htm)
		 Метод RemoveById удаляет
		 сертификат по идентификатору и возвращает признак удачного удаления.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](ICertificates.RemoveByKey.htm)
		 Метод RemoveByKey удаляет
		 сертификат по ключу и возвращает признак удачного удаления.
