# ICertificates.RemoveById


## Синтаксис


RemoveById(Id: String): Boolean;


## Параметры


Id. Идентификатор сертификата.


## Описание


Метод RemoveById удаляет сертификат
 по идентификатору и возвращает признак удачного удаления.


## Комментарии


Метод возвращает значение True,
 если сертификат был удален и False,
 если удаление по каким-либо причинам невозможно.
