# ICertificates.FindByKey


## Синтаксис


FindByKey(Key: Integer): [ICertificate](../ICertificate/ICertificate.htm);


## Параметры


Key. Ключ сертификата.


## Описание


Метод FindByKey осуществляет
 поиск сертификата безопасности по ключу.


## Комментарии


Метод возвращает параметры сертификата в случае удачного поиска и Null, если сертификат с указанным
 ключом отсутствует.
