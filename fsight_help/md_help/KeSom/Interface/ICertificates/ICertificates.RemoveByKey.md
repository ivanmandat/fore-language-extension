# ICertificates.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ сертификата.


## Описание


Метод RemoveByKey удаляет сертификат
 по ключу и возвращает признак удачного удаления.


## Комментарии


Метод возвращает значение True,
 если сертификат был удален и False,
 если удаление по каким-либо причинам невозможно.
