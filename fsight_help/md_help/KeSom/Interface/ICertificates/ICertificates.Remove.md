# ICertificates.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс сертификата в
 коллекции.


## Описание


Метод Remove удаляет сертификат
 по индексу и возвращает признак удачного удаления.


## Комментарии


Метод возвращает значение True,
 если сертификат был удален и False,
 если удаление по каким-либо причинам невозможно.
