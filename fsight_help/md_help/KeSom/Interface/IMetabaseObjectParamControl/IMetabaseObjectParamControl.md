# IMetabaseObjectParamControl


Сборка: Metabase;


## Описание


Интерфейс IMetabaseObjectParamControl
 содержит свойства для настройки привязки параметра вложенного объекта
 к параметрам текущего объекта.


## Иерархия наследования


IMetabaseObjectParamControl


## Комментарии


Параметры привязки можно получить, используя свойства и методы интерфейса
 [IMetabaseObjectControl](../IMetabaseObjectControl/IMetabaseObjectControl.htm).

		 Имя свойства

		 [ControlledParam](IMetabaseObjectParamControl.ControlledParam.htm)
		 Свойство ControlledParam
		 возвращает параметр вложенного объекта, для которого настраивается

		 ![](../../Property_Image.gif)
		 [Expression](IMetabaseObjectParamControl.Expression.htm)
		 Свойство Expression
		 возвращает выражение, определяющее привязку текущего параметра
