# ICryptoPackage.Decrypt


## Синтаксис


Decrypt(passPhrase: String; cipherText: String):
 String;


## Параметры


passPhrase. Секретная фраза,
 используемая при дешифровании текста;


cipherText. Зашифрованный текст,
 который необходимо дешифровать.


## Описание


Метод Decrypt осуществляет дешифрование
 указанного зашифрованного текста и возвращает полученный результат.


## Комментарии


Если дешифрация текста cipherText
 с использованием фразы passPhrase
 недопустима, то метод возвращает пустое значение.
