# IMetabaseCustomClassOperation


Сборка: Metabase;


## Описание


Интерфейс IMetabaseCustomClassOperation
 содержит свойства, предназначенные для работы с операцией пользовательского
 класса.


## Иерархия наследования


IMetabaseCustomClassOperation


## Комментарии


Интерфейс описывает как стандартные, так и пользовательские операции
 пользовательского класса. Получить операцию можно, используя свойства
 и методы интерфейсов [IMetabaseCustomClassStandardOperations](../IMetabaseCustomClassStandardOperations/IMetabaseCustomClassStandardOperations.htm),
 [IMetabaseCustomClassOperations](../IMetabaseCustomClassOperations/IMetabaseCustomClassOperations.htm).
