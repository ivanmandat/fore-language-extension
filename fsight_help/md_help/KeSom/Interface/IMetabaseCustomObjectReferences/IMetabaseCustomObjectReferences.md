# IMetabaseCustomObjectReferences


Сборка: Metabase;


## Описание


Интерфейс IMetabaseCustomObjectReferences
 содержит свойства и методы для работы с коллекцией зависимостей объекта
 пользовательского класса.


## Иерархия наследования


           IMetabaseCustomObjectReferences


## Комментарии


Для получения коллекции зависимостей объекта пользовательского класса
 используйте свойство [IMetabaseCustomObjectSerializer.References](../IMetabaseCustomObjectSerializer/IMetabaseCustomObjectSerializer.References.htm)
 или [ICustomObjectResolver.References](../ICustomObjectResolver/ICustomObjectResolver.References.htm).
