# IMetabaseCustomObjectReferences.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс зависимости в
 коллекции.


## Описание


Метод Remove осуществляет удаление
 зависимости из коллекции по индексу.


## Комментарии


Метод возвращает значение True,
 если удаление прошло успешно.
