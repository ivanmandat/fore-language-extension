# IConnectedUserInfo


Сборка: Metabase;


## Описание


Интерфейс IConnectedUserInfo
 содержит свойства пользователя, подключенного к репозиторию.


## Иерархия наследования


           IConnectedUserInfo


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [HostName](IConnectedUserInfo.HostName.htm)
		 Свойство HostName возвращает
		 наименование рабочей станции, с которой осуществлено подключение
		 к репозиторию.


		 ![](../../Property_Image.gif)
		 [NetAddress](IConnectedUserInfo.NetAddress.htm)
		 Свойство NetAddress
		 возвращает МАС-адрес компьютера, с которого осуществлено подключение
		 к репозиторию.


		 ![](../../Property_Image.gif)
		 [UserName](IConnectedUserInfo.UserName.htm)
		 Свойство UserName возвращает
		 наименование пользователя, который подключен к репозиторию.


		 ![](../../Property_Image.gif)
		 [UserOsName](IConnectedUserInfo.UserOsName.htm)
		 Свойство UserOsName
		 возвращает наименование пользователя операционной системы, с использованием
		 которого осуществлено подключение к репозиторию.
