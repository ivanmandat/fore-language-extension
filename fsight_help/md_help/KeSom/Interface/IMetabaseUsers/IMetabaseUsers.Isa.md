# IMetabaseUsers.Isa


## Синтаксис


Isa: [IMetabaseUser](../IMetabaseUser/IMetabaseUser.htm);


## Описание


Свойство Isa возвращает параметры пользователя, являющегося администратором информационной безопасности репозитория.


## Комментарии


Актуально, если в платформе включено разделение ролей между администратором информационной безопасности и прикладным администратором.
