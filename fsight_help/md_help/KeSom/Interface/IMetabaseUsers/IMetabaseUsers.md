# IMetabaseUsers


## Описание


Интерфейс IMetabaseUsers содержит
 свойства для работы с коллекцией пользователей репозитория.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Administrator](IMetabaseUsers.Administrator.htm)
		 Свойство Administrator
		 возвращает параметры пользователя, являющегося прикладным администратором
		 репозитория.


		 ![](../../Property_Image.gif)
		 [Attributes](IMetabaseUsers.Attributes.htm)
		 Свойство Attributes
		 возвращает коллекцию атрибутов пользователей.


		 ![](../../Property_Image.gif)
		 [Count](IMetabaseUsers.Count.htm)
		 Свойство Count возвращает
		 количество пользователей репозитория.


		 ![](../../Property_Image.gif)
		 [Isa](IMetabaseUsers.Isa.htm)
		 Свойство Isa возвращает
		 параметры пользователя, являющегося администратором информационной
		 безопасности репозитория.


		 ![](../../Property_Image.gif)
		 [Item](IMetabaseUsers.Item.htm)
		 Свойство Item возвращает
		 параметры пользователя репозитория.
