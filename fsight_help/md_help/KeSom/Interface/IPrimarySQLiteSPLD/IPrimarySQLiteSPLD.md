# IPrimarySQLiteSPLD


Сборка: Metabase;


## Описание


Интерфейс IPrimarySQLiteSPLD
 содержит свойства и методы модуля безопасности, используемого при подключении
 к репозиторию, созданному во встраиваемой СУБД SQLite.
