# ICertProvider.NewCode


## Синтаксис


NewCode(Var Cookie: String; Var Code: String);


## Параметры


Cookie. Переменная, в которую
 будет помещено случайное значение, используемое для установления соответствия
 между цифровой подписью и полученным блоком данных.


Code. Переменная, в которую
 будет помещён случайный блок данных, который должен быть подписан цифровой
 подписью.


## Описание


Метод NewCode генерирует пару
 «блок данных-случайное значение», которые в дальнейшем используются для
 цифровой подписи.
