# ICertProvider.Code


## Синтаксис


Code(Cookie: String): String;


## Параметры


Cookie. Случайное значение,
 используемое для установления соответствия между цифровой подписью и блоком
 данных.


## Описание


Свойство Code определяет блок
 данных, которому будет сопоставлено указанное случайное значение.


## Комментарии


Случайное значение Cookie может
 быть сгенерировано методом [ICertProvider.NewCode](ICertProvider.NewCode.htm)
 или получено каким-либо другим способом.

Примеры использования приведены в описании методов [ICertProvider.OpenSigner](ICertProvider.OpenSigner.htm),
 [ICertProvider.OpenSignerBase64](ICertProvider.OpenSignerBase64.htm).
