# IMetabaseCheckList.SaveToFile


## Синтаксис


SaveToFile(FileName: String);


## Параметры


FileName. Файл, в который будет
 сохранен список объектов репозитория и их рассчитанные контрольные суммы.


## Описание


Метод SaveToFile осуществляет
 сохранение в файл списка объектов репозитория и рассчитанных для них контрольных
 сумм.
