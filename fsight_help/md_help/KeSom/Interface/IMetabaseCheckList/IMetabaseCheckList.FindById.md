# IMetabaseCheckList.FindById


## Синтаксис


FindById(Id: String): [IMetabaseCheckListItem](../IMetabaseCheckListItem/IMetabaseCheckListItem.htm);


## Параметры


Id - идентификатор объекта, который необходимо найти в списке проверки контрольных сумм.


## Описание


Метод FindById осуществляет поиск объекта в списке проверки контрольных сумм по идентификатору. В случае удачного поиска метод возвращает параметры объекта, иначе Null.
