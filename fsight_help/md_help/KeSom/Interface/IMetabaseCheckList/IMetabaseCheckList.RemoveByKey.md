# IMetabaseCheckList.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key - ключ объекта в списке для проверки контрольных сумм.


## Описание


Метод RemoveByKey осуществляет удаление объекта из списка проверки контрольных сумм, и возвращает признак удачного удаления.


## Комментарии


Ключ объекта передается посредством параметра Key. Метод возвращает значение True в случае удачного удаления.
