# IMetabaseCheckList.Move


## Синтаксис


Move(IndexFrom: Integer; IndexTo: Integer);


## Параметры


IndexFrom - индекс позиции объекта, который необходимо переместить в списке.


IndexTo - индекс позиции, в которую необходимо переместить объект.


## Описание


Метод Move осуществляет перемещение объекта в списке расчета контрольных сумм.
