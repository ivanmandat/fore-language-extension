# IMetabaseCheckList.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IMetabaseCheckListItem](../IMetabaseCheckListItem/IMetabaseCheckListItem.htm);


## Параметры


Key - ключ объекта, который необходимо найти в списке проверки контрольных сумм.


## Описание


Метод FindByKey осуществляет поиск объекта в списке проверки контрольных сумм по ключу. В случае удачного поиска метод возвращает параметры объекта, иначе Null.
