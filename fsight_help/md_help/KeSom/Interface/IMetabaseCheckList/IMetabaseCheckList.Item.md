# IMetabaseCheckList.Item


## Синтаксис


Item(Index: Integer): [IMetabaseCheckListItem](../IMetabaseCheckListItem/IMetabaseCheckListItem.htm);


## Параметры


Index - индекс объекта в списке.


## Описание


Свойство Item возвращает параметры объекта, содержащегося в списке проверки контрольных сумм. Индекс объекта передается посредством параметра Index.
