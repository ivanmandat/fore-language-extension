# IMetabaseCheckList.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс объекта в списке для проверки контрольных сумм.


## Описание


Метод Remove осуществляет удаление объекта из списка проверки контрольных сумм, и возвращает признак удачного удаления.


## Комментарии


Индекс объекта передается посредством параметра Index. Метод возвращает значение True в случае удачного удаления.
