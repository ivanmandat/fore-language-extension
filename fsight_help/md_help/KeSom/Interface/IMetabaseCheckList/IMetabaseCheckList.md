# IMetabaseCheckList


## Описание


Интерфейс IMetabaseCheckList
 содержит свойства и методы для работы со списком объектов, для которых
 осуществляется проверка контрольных сумм.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMetabaseCheckList.Count.htm)
		 Свойство Count возвращает
		 количество объектов, содержащихся в списке проверки контрольных
		 сумм.


		 ![](../../Property_Image.gif)
		 [Item](IMetabaseCheckList.Item.htm)
		 Свойство Item возвращает
		 параметры объекта, содержащегося в списке проверки контрольных
		 сумм.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IMetabaseCheckList.Add.htm)
		 Метод Add осуществляет
		 добавление объекта репозитория в список проверки контрольных сумм.


		 ![](../../Sub_Image.gif)
		 [Clear](IMetabaseCheckList.Clear.htm)
		 Метод Clear осуществляет
		 очистку списка объектов, для которых осуществляется проверка контрольных
		 сумм.


		 ![](../../Sub_Image.gif)
		 [FindById](IMetabaseCheckList.FindById.htm)
		 Метод FindById осуществляет
		 поиск объекта в списке проверки контрольных сумм по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IMetabaseCheckList.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск объекта в списке проверки контрольных сумм по ключу.


		 ![](../../Sub_Image.gif)
		 [LoadFromFile](IMetabaseCheckList.LoadFromFile.htm)
		 Метод LoadFromFile
		 осуществляет загрузку из файла и проверку контрольных сумм объектов
		 репозитория.


		 ![](../../Sub_Image.gif)
		 [LoadFromStream](IMetabaseCheckList.LoadFromStream.htm)
		 Метод LoadFromStream
		 осуществляет загрузку из потока и проверку контрольных сумм объектов
		 репозитория.


		 ![](../../Sub_Image.gif)
		 [Move](IMetabaseCheckList.Move.htm)
		 Метод Move осуществляет
		 перемещение объекта в списке расчета контрольных сумм.


		 ![](../../Sub_Image.gif)
		 [Remove](IMetabaseCheckList.Remove.htm)
		 Метод Remove осуществляет
		 удаление объекта из списка проверки контрольных сумм по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IMetabaseCheckList.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление объекта из списка проверки контрольных сумм по ключу.


		 ![](../../Sub_Image.gif)
		 [SaveToFile](IMetabaseCheckList.SaveToFile.htm)
		 Метод SaveToFile осуществляет
		 сохранение в файл списка объектов репозитория и рассчитанных для
		 них контрольных сумм.


		 ![](../../Sub_Image.gif)
		 [SaveToStream](IMetabaseCheckList.SaveToStream.htm)
		 Метод SaveToStream
		 осуществляет сохранение в поток списка объектов репозитория и
		 рассчитанных для них контрольных сумм.
