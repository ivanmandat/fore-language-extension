# IMetabaseObjectCopyInfo.WithData


## Синтаксис


WithData: Boolean;


## Описание


Свойство WithData определяет
 признак копирования объекта вместе с данными.


## Комментарии


По умолчанию свойству установлено значение False,
 при этом осуществляется копирование объекта без данных.
