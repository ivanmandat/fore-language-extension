# IMetabaseCustomClassStandardOperations.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IMetabaseCustomClassOperation](../IMetabaseCustomClassOperation/IMetabaseCustomClassOperation.htm);


## Параметры


Key. Ключ стандартной операции.


## Описание


Метод FindByKey осуществляет
 поиск по ключу и возвращает стандартную операцию в случае удачного поиска.


## Комментарии


Метод возвращает значение Null,
 если операция с указанным ключом не найдена.
