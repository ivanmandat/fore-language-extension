# IMetabaseCustomClassStandardOperations


Сборка: Metabase;


## Описание


Интерфейс IMetabaseCustomClassStandardOperations
 предназначен для работы с коллекцией стандартных операций пользовательского
 класса.


## Иерархия наследования


IMetabaseCustomClassStandardOperations


## Комментарии


Коллекцию стандартных операций можно получить, используя свойство [IMetabaseCustomClass.StandardOperations](../IMetabaseCustomClass/IMetabaseCustomClass.StandardOperations.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMetabaseCustomClassStandardOperations.Count.htm)
		 Свойство Count возвращает
		 количество стандартных операций в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMetabaseCustomClassStandardOperations.Item.htm)
		 Свойство Item возвращает
		 стандартную операцию с указанным индексом.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)

		 Метод FindById осуществляет
		 поиск по идентификатору и возвращает стандартную операцию в случае

		 ![](../../Sub_Image.gif)
		 [FindByKey](IMetabaseCustomClassStandardOperations.FindByKey.htm)
		 Метод FindByKey осуществляет

		 поиска.
