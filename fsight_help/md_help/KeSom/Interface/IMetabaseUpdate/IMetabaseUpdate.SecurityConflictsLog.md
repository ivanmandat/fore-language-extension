# IMetabaseUpdate.SecurityConflictsLog


## Синтаксис


		SecurityConflictsLog: String;


## Описание


Свойство SecurityConflictsLog
 возвращает лог конфликтов субъектов безопасности, для которых переносятся
 права доступа на объекты и конфликты при обновлении мандатного доступа.


## Комментарии


Перед использованием свойства требуется присвоить свойству [IMetabaseUpdate.AllowReplaceSD](IMetabaseUpdate.AllowReplaceSD.htm) значение
 True.

Использование свойства приведено в примере для свойства [IMetabaseUpdateProgress.OnSkip](../IMetabaseUpdateProgress/IMetabaseUpdateProgress.OnSkip.htm).
