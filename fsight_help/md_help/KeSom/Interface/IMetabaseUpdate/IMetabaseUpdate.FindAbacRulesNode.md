# IMetabaseUpdate.FindAbacRulesNode


## Синтаксис


AbacRulesNode: [IMetabaseUpdateAbacRulesNode](../IMetabaseUpdateAbacRulesNode/IMetabaseUpdateAbacRulesNode.htm);


## Описание


Метод FindAbacRulesNode осуществляет
 поиск структуры правил [атрибутного
 доступа](admin.chm::/04_securitypolicy/Admin_AttributeAccess.htm) в обновлении.


## Комментарии


Если структура правил атрибутного доступа не найдена, то возвращается
 пустое значение и структура не создается.


Для создания структуры правил атрибутного доступа в обновлении используйте
 свойство [IMetabaseUpdate.AbacRulesNode](IMetabaseUpdate.AbacRulesNode.htm).
