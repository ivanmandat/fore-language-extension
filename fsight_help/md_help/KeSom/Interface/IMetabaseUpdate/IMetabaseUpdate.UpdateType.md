# IMetabaseUpdate.UpdateType


## Синтаксис


UpdateType: [MetabaseObjectUpdateType](../../Enums/MetabaseObjectUpdateType.htm);


## Описание


Свойство UpdateType определяет
 тип обновления объектов репозитория.


## Комментарии


Установленный тип обновления будет использоваться для всех объектов,
 которых тип обновления которых не выставлен явно (используется тип обновления
 по умолчанию).
