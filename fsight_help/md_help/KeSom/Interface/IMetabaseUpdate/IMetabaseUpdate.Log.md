# IMetabaseUpdate.Log


## Синтаксис


Log: [IMetabaseUpdateLog](../IMetabaseUpdateLog/IMetabaseUpdateLog.htm);


## Описание


Свойство Log возвращает журнал
 применения обновления.


## Комментарии


В журнале содержится общая информация об установке обновления, а также
 детальная информация об установке каждого отдельного элемента обновления.
