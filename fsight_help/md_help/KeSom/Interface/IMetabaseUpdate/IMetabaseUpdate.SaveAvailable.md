# IMetabaseUpdate.SaveAvailable


## Синтаксис


SaveAvailable (FileName: String;[Options: [MetabaseUpdateCopyType](../../Enums/MetabaseUpdateCopyType.htm)
 = 1]);


## Параметры


FileName. Полный путь и имя
 файла, в который будет сохранено обновление;


Options. Тип копирования обновления.


## Описание


Метод SaveAvailable сохраняет
 обновление с доступными пользователю объектами.
