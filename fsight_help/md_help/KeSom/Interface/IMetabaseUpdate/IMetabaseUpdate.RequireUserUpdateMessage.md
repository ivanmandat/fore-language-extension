# IMetabaseUpdate.RequireUserUpdateMessage


## Синтаксис


		RequireUserUpdateMessage: Boolean;


## Описание


Свойство RequireUserUpdateMessage
 возвращает признак вывода сообщения с требованием об обновлении пользователей.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Сообщение с требованием об обновлении пользователей после установки
	 обновления выводится;


	- False. Сообщение с требованием
	 об обновлении прав пользователей не выводится.
