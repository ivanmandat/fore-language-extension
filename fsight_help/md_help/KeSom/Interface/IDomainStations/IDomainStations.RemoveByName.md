# IDomainStations.RemoveByName


## Синтаксис


		RemoveByName(Name: String): Boolean;


## Параметры


Name. Наименование рабочей
 станции, которую необходимо удалить.


## Описание


Метод RemoveByName удаляет указанную
 станцию из коллекции.


## Комментарии


Метод возвращает значение True,
 если удаление прошло удачно, иначе False.
