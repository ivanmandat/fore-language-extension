# IDomainStations.Remove


## Синтаксис


		Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс станции в коллекции.


## Описание


Метод Remove удаляет станцию
 из коллекции по индексу.


## Комментарии


Метод возвращает значение True,
 если удаление адреса с указанным индексом прошло удачно, иначе False.
