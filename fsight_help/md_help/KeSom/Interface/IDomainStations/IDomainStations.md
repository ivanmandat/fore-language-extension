# IDomainStations


## Описание


Интерфейс IDomainStations содержит
 свойства и методы для работы коллекцией рабочих станций, принадлежащих
 домену.


## Свойства, унаследованные от [IStations](../IStations/IStations.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Access](../IStations/IStations.Access.htm)
		 Свойство Access определяет
		 тип доступа для станций из коллекции.


		 ![](../../Property_Image.gif)
		 [Count](../IStations/IStations.Count.htm)
		 Свойство Count возвращает
		 количество рабочих станций в коллекции.


		 ![](../../Property_Image.gif)
		 [Domains](../IStations/IStations.Domains.htm)
		 Свойство Domains возвращает
		 домены, к которым относятся рабочие станции из коллекции.


		 ![](../../Property_Image.gif)
		 [Item](../IStations/IStations.Item.htm)
		 Свойство Item возвращает
		 рабочую станцию из коллекции.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IDomainStations.Add.htm)
		 Метод Add добавляет
		 станцию в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Remove](IDomainStations.Remove.htm)
		 Метод Remove удаляет
		 станцию из коллекции по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByName](IDomainStations.RemoveByName.htm)
		 Метод RemoveByName
		 удаляет указанную станцию из коллекции.


## Методы, унаследованные от [IStations](../IStations/IStations.htm)


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindByName](../IStations/IStations.FindByName.htm)
		 Метод FindByName возвращает
		 рабочую станцию из коллекции по заданному наименованию.


		 ![](../../Sub_Image.gif)
		 [Refresh](../IStations/IStations.Refresh.htm)
		 Метод Refresh производит
		 обновление коллекции.
