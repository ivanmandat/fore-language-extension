# ISecurityLevel.ExportAllowed


## Синтаксис


		ExportAllowed: Boolean;


## Описание


Свойство ExportAllowed определяет,
 будет ли возможность экспортировать объекты с данным уровнем безопасности.


## Комментарии


По умолчанию свойство принимает значение True.
