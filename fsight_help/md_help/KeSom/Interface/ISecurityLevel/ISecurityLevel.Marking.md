# ISecurityLevel.Marking


## Синтаксис


		Marking: Boolean;


## Описание


Свойство Marking определяет,
 будут ли маркироваться объекты с данным уровнем безопасности при экспорте
 и печати.


## Комментарии


По умолчанию свойство принимает значение False.
