# ISecurityLevel


## Описание


Интерфейс ISecurityLevel предназначен
 для работы со свойствами уровня безопасности.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Category](ISecurityLevel.Category.htm)
		 Свойство Category возвращает
		 категорию безопасности, в которой создан данный уровень.


		 ![](../../Property_Image.gif)
		 [CriticalityLabel](ISecurityLevel.CriticalityLabel.htm)
		 Свойство CriticalityLabel
		 возвращает метку критичности уровня в категории безопасности.


		 ![](../../Property_Image.gif)
		 [Description](ISecurityLevel.Description.htm)
		 Свойство Description
		 определяет описание, которое будет использовано при маркировке
		 объектов при экспорте и печати.


		 ![](../../Property_Image.gif)
		 [ExportAllowed](ISecurityLevel.ExportAllowed.htm)
		 Свойство ExportAllowed
		 определяет, будет ли возможность экспортировать объекты с данным
		 уровнем безопасности.


		 ![](../../Property_Image.gif)
		 [Index](ISecurityLevel.Index.htm)
		 Свойство Index возвращает
		 индекс уровня в категории безопасности.


		 ![](../../Property_Image.gif)
		 [Marking](ISecurityLevel.Marking.htm)
		 Свойство Marking определяет,
		 будут ли маркироваться объекты с данным уровнем безопасности при
		 экспорте и печати.


		 ![](../../Property_Image.gif)
		 [Name](ISecurityLevel.Name.htm)
		 Свойство Name возвращает
		 наименование уровня в категории безопасности.


		 ![](../../Property_Image.gif)
		 [PrintAllowed](ISecurityLevel.PrintAllowed.htm)
		 Свойство PrintAllowed
		 определяет, будет ли возможность печатать объекты с данным уровнем
		 безопасности.


		 ![](../../Property_Image.gif)
		 [TreatSectionAsDocument](ISecurityLevel.TreatSectionAsDocument.htm)
		 Свойство TreatSectionAsDocument
		 определяет, необходимо ли при маркировке документа считать секции
		 объекта отдельными документами.
