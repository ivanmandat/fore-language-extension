# IMetabaseUpdateRemapping.OldId


## Синтаксис


OldId: String;


## Описание


Свойство OldId возвращает
 идентификатор сопоставляемого элемента в обновлении.


## Комментарии


Для получения ключа сопоставляемого элемента в обновлении используйте
 свойство [IMetabaseUpdateRemapping.OldKey](IMetabaseUpdateRemapping.OldKey.htm).
