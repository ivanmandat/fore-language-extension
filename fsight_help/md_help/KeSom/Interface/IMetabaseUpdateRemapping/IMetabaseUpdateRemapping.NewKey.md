# IMetabaseUpdateRemapping.NewKey


## Синтаксис


NewKey: Integer;


## Описание


Свойство NewKey возвращает
 ключ сопоставляемого элемента в репозитории-приёмнике.


## Комментарии


Для получения идентификатора сопоставляемого элемента в репозитории-приёмнике
 используйте свойство [IMetabaseUpdateRemapping.NewId](IMetabaseUpdateRemapping.NewId.htm).
