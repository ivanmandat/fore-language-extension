# IMetabaseUpdateRemapping.Type


## Синтаксис


Type: [MetabaseUpdateRemappingType](../../Enums/MetabaseUpdateRemappingType.htm);


## Описание


Свойство Type возвращает
 тип сопоставляемого элемента.


## Комментарии


Для получения более подробной информации о параметрах повторного сопоставления
 элемента необходимо привести интерфейс [IMetabaseUpdateRemapping](IMetabaseUpdateRemapping.htm)
 к одному из дочерних интерфейсов, зависящих от типа сопоставляемого элемента.
