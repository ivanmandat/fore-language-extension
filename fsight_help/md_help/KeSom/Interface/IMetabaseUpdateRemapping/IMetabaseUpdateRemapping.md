# IMetabaseUpdateRemapping


Сборка: Metabase;


## Описание


Интерфейс IMetabaseUpdateRemapping
 используется для работы с параметрами повторного сопоставления элементов
 в обновлении и в репозитории-приёмнике.


## Иерархия наследования


           IMetabaseUpdateRemapping


## Комментарии


Данный интерфейс является базовым для интерфейса [IMetabaseUpdateObjectRemapping](../IMetabaseUpdateObjectRemapping/IMetabaseUpdateObjectRemapping.htm),
 использующимся для работы с параметрами повторного сопоставления объектов
 в обновлении и в репозитории-приёмнике.
