# IMetabaseUpdateRemapping.NewId


## Синтаксис


NewId: String;


## Описание


Свойство NewId возвращает
 идентификатор сопоставляемого элемента в репозитории-приёмнике.


## Комментарии


Для получения ключа сопоставляемого объекта в репозитории-приёмнике
 используйте свойство [IMetabaseUpdateRemapping.NewKey](IMetabaseUpdateRemapping.NewKey.htm).
