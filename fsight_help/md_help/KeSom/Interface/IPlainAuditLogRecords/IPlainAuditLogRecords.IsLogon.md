# IPlainAuditLogRecords.IsLogon


## Синтаксис


IsLogon: Boolean;


## Описание


Свойство IsLogon возвращает
 значение, указывающее, соответствует ли текущая запись операции подключения
 к репозиторию.


## Комментарии


Свойство возвращает значение True,
 если текущая запись соответствует операции подключения к репозиторию,
 и False для всех остальных операций.
