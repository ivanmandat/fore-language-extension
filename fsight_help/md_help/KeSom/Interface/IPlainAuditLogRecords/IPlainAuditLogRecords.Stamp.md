# IPlainAuditLogRecords.Stamp


## Синтаксис


Stamp: Double;


## Описание


Свойство Stamp возвращает дату
 и время выполнения действия.


## Комментарии


Значение представляется в системном формате дат. Для преобразования
 используйте метод [DateTime.FromDouble](ForeSys.chm::/Class/DateTime/DateTime.FromDouble.htm).
