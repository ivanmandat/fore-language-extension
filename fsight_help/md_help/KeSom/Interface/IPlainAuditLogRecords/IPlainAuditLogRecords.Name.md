# IPlainAuditLogRecords.Name


## Синтаксис


Name: String;


## Описание


Свойство Name возвращает наименование
 операции над объектом.


## Комментарии


Свойство актуально только для записей, соответствующих операциям над
 объектами. Если текущая запись соответствует подключению к репозиторию,
 то свойство возвращает пустое значение.
