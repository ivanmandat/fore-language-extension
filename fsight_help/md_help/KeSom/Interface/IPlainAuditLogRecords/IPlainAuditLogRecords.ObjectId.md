# IPlainAuditLogRecords.ObjectId


## Синтаксис


ObjectId: String;


## Описание


Свойство ObjectId возвращает
 идентификатор объекта, над которым произведена операция.


## Комментарии


Если текущая запись соответствует подключению к репозиторию, то свойство
 возвращает пустое значение.
