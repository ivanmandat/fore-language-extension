# IPlainAuditLogRecords.ObjectName


## Синтаксис


ObjectName: String;


## Описание


Свойство ObjectName возвращает
 наименование объекта, над которым произведена операция.


## Комментарии


Если текущая запись соответствует подключению к репозиторию, то свойство
 возвращает пустое значение.
