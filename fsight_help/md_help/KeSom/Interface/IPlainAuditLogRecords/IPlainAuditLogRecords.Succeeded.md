# IPlainAuditLogRecords.Succeeded


## Синтаксис


Succeeded: Boolean;


## Описание


Свойство Succeeded возвращает
 признак успешного выполнения действия.


## Комментарии


Свойство возвращает значение True,
 если операция/подключение к репозиторию завершены успешно и False,
 если операция/подключение было отменено.
