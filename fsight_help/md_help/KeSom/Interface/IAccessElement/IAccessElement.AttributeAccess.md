# IAccessElement.AttributeAccess


## Синтаксис


AttributeAccess(Attribute: [AccessElementAttributes](../../Enums/AccessElementAttributes.htm)):
 [IAccessAttributeValue](../IAccessAttributeValue/IAccessAttributeValue.htm);


## Описание


Свойство AttributeAccess определяет
 значение атрибута прав доступа элемента.


## Комментарии


После получения значения свойства AttributeAccess
 и изменения прав доступа для необходимых субъектов, значение свойства
 AttributeAccess необходимо переприсвоить.
