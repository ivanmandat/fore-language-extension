# IAccessElement


## Описание


Интерфейс IAccessElement предназначен
 для работы с правами доступа элемента.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AccessToken](IAccessElement.AccessToken.htm)
		 Свойство AccessToken
		 возвращает параметры мандатного контроля доступа для элемента.


		 ![](../../Property_Image.gif)
		 [AttributeAccess](IAccessElement.AttributeAccess.htm)
		 Свойство AttributeAccess
		 определяет значение атрибута прав доступа элемента.


		 ![](../../Property_Image.gif)
		 [Key](IAccessElement.Key.htm)
		 Свойство Key возвращает
		 ключ элемента.


		 ![](../../Property_Image.gif)
		 [Name](IAccessElement.Name.htm)
		 Свойство Name возвращает
		 наименование элемента.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Apply](IAccessElement.Apply.htm)
		 Метод Apply сохраняет
		 заданные значения атрибутов доступа и применяет данные права в
		 соответствии с выбранными параметрами.


		 ![](../../Sub_Image.gif)
		 [ApplyAccessToken](IAccessElement.ApplyAccessToken.htm)
		 Метод ApplyAccessToken
		 применяет указанные параметры мандатного контроля доступа
		 для элемента.


		 ![](../../Sub_Image.gif)
		 [Edit](IAccessElement.Edit.htm)
		 Метод Edit открывает
		 на редактирование права доступа элемента.
