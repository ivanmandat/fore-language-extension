# IAccessElement.ApplyAccessToken


## Синтаксис


ApplyAccessToken(Options: [AccessElementApplyOptions](../../Enums/AccessElementApplyOptions.htm));


## Параметры


Options. Параметры применения прав доступа для элемента.


## Описание


Метод ApplyAccessToken сохраняет заданные категории секретности для мандатного доступа и применяет их в соответствии с выбранными параметрами.


## Комментарии


Для получения параметров мандатного контроля доступа для элемента используйте свойство [IAccessElement.AccessToken](IAccessElement.AccessToken.htm).
