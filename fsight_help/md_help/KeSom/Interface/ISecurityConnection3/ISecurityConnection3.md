# ISecurityConnection3


Сборка: Metabase;


## Описание


Интерфейс ISecurityConnection3
 описывает соединение с сервером БД, после установления которого был выполнен
 прикладной макрос.


## Иерархия наследования


ISecurityConnection3


## Комментарии


Данный интерфейс поддерживают подключения, произведенные с помощью базы
 данных репозитория, для которой определены [дополнительные
 настройки подключения](UiNavObj.chm::/database/database_additional_adjustment.htm).
