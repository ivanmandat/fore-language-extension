# ISecurityConnection3.InitExecuteSetup


## Синтаксис


InitExecuteSetup: [IDatabaseInitExecuteSetup](../IDatabaseInitExecuteSetup/IDatabaseInitExecuteSetup.htm);


## Описание


Свойство InitExecuteSetup возвращает дополнительные настройки базы данных репозитория.


## Комментарии


В дополнительных настройках базы данных указывается макрос, который будет выполнен после подключением к БД.
