# IDomains.Remove


## Синтаксис


		Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс домена в коллекции.


## Описание


Метод Remove удаляет домен из
 коллекции по индексу. Возвращает значение True,
 если удаление домена с индексом Index
 прошло удачно.
