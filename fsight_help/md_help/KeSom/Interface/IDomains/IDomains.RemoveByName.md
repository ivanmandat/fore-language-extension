# IDomains.RemoveByName


## Синтаксис


		RemoveByName(Name: String): Boolean;


## Параметры


Name. Наименование домена,
 который нужно удалить.


## Описание


Метод RemoveByName удаляет указанный
 домен из коллекции.


## Комментарии


Метод возвращает значение True,
 если удаление прошло удачно, иначе - False.
