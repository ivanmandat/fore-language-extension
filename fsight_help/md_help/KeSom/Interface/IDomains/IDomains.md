# IDomains


## Описание


Интерфейс IDomains содержит
 свойства и методы для работы с коллекцией доменов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDomains.Count.htm)
		 Свойство Count возвращает
		 количество доменов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IDomains.Item.htm)
		 Свойство Item возвращает
		 домен из коллекции по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IDomains.Add.htm)
		 Метод Add добавляет
		 домен в коллекцию.


		 ![](../../Sub_Image.gif)
		 [FindByName](IDomains.FindByName.htm)
		 Метод FindByName возвращает
		 домен из коллекции по заданному наименованию.


		 ![](../../Sub_Image.gif)
		 [Refresh](IDomains.Refresh.htm)
		 Метод Refresh производит
		 обновление коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IDomains.Remove.htm)
		 Метод Remove удаляет
		 домен из коллекции по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByName](IDomains.RemoveByName.htm)
		 Метод RemoveByName
		 удаляет указанный домен из коллекции.
