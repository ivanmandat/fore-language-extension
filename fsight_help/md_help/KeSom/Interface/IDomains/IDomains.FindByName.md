# IDomains.FindByName


## Синтаксис


FindByName(Name: String): [IDomain](../IDomain/IDomain.htm);


## Параметры


Name. Наименование домена.


## Описание


Метод FindByName возвращает
 домен из коллекции по заданному наименованию.


## Комментарии


Метод возвращает значение Null,
 если домен с указанным наименование отсутствует в локальной сети.
