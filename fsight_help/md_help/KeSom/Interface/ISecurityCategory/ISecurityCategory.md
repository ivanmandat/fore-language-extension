# ISecurityCategory


## Описание


Интерфейс ISecurityCategory
 содержит свойства и методы категорий безопасности мандатного доступа.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Hierarchical](ISecurityCategory.Hierarchical.htm)
		 Свойство Hierarchical
		 определяет признак иерархической категории безопасности.


		 ![](../../Property_Image.gif)
		 [Key](ISecurityCategory.Key.htm)
		 Свойство Key возвращает
		 ключ категории безопасности.


		 ![](../../Property_Image.gif)
		 [Level](ISecurityCategory.Level.htm)
		 Свойство Level возвращает
		 параметры указанного уровня безопасности.


		 ![](../../Property_Image.gif)
		 [LevelCount](ISecurityCategory.LevelCount.htm)
		 Свойство LevelCount
		 возвращает количество уровней в текущей категории.


		 ![](../../Property_Image.gif)
		 [Name](ISecurityCategory.Name.htm)
		 Свойство Name определяет
		 наименование категории безопасности.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddLevel](ISecurityCategory.AddLevel.htm)
		 Метод AddLevel создает
		 уровень в категории безопасности.


		 ![](../../Sub_Image.gif)
		 [FindLevelByLabel](ISecurityCategory.FindLevelByLabel.htm)
		 Метод FindLevelByLabel
		 осуществляет поиск уровня по метке критичности.


		 ![](../../Sub_Image.gif)
		 [RemoveLevel](ISecurityCategory.RemoveLevel.htm)
		 Метод RemoveLevel осуществляет
		 удаление уровня безопасности и возвращает признак удачного удаления.
