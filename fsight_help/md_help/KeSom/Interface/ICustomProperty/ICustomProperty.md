# ICustomProperty


Сборка: Metabase;


## Описание


Интерфейс ICustomProperty содержит
 свойства для работы с пользовательским свойством объекта обновления.


## Иерархия наследования


           ICustomProperty


## Комментарии


Для получения объекта данного типа используйте свойство [ICustomProperties.Item](../ICustomProperties/ICustomProperties.Item.htm).
