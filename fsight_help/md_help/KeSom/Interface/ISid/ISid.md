# ISid


## Описание


Интерфейс ISid содержит свойства идентификатора субъекта безопасности.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AsString](ISid.AsString.htm)
		 Свойство AsString определяет
		 идентификатор субъекта безопасности в строковом виде.


		 ![](../../Property_Image.gif)
		 [Valid](ISid.Valid.htm)
		 Свойство Valid возвращает
		 признак корректности (валидности) идентификатора субъекта безопасности.
