# IMetabaseUpdateResolver


## Описание


Интерфейс IMetabaseUpdateResolver
 содержит свойства для работы с коллекцией зависимостей обновления от объектов
 репозитория-источника, которые отсутствуют в репозитории назначения.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMetabaseUpdateResolver.Count.htm)
		 Свойство Count возвращает
		 количество зависимостей в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMetabaseUpdateResolver.Item.htm)
		 Свойство Item возвращает
		 зависимость из коллекции по ее индексу.
