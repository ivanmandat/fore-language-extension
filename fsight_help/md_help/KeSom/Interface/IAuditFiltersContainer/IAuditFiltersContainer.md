# IAuditFiltersContainer


Сборка: Metabase;


## Описание


Интерфейс IAuditFiltersContainer
 содержит свойства и методы для работы с контейнером фильтров протокола
 доступа.


## Иерархия наследования


           IAuditFiltersContainer


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Callback](IAuditFiltersContainer.Callback.htm)
		 Свойство Callback определяет
		 действие, выполняемое при возникновении конфликта во время загрузки
		 контейнера.


		 ![](../../Property_Image.gif)
		 [DefaultFileName](IAuditFiltersContainer.DefaultFileName.htm)
		 Свойство DefaultFileName
		 возвращает полное имя файла контейнера по умолчанию.


		 ![](../../Property_Image.gif)
		 [FileName](IAuditFiltersContainer.FileName.htm)
		 Свойство FileName определяет
		 имя файла, из которого был загружен контейнер.


		 ![](../../Property_Image.gif)
		 [Filters](IAuditFiltersContainer.Filters.htm)
		 Свойство Filters возвращает
		 фильтры, которые содержит контейнер.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Append](IAuditFiltersContainer.Append.htm)
		 Метод Append дополняет
		 контейнер новыми фильтрами.


		 ![](../../Sub_Image.gif)
		 [Assign](IAuditFiltersContainer.Assign.htm)
		 Метод Assign выполняет
		 копирование контейнера.


		 ![](../../Sub_Image.gif)
		 [Load](IAuditFiltersContainer.Load.htm)
		 Метод Load осуществляет
		 загрузку контейнера из файла по умолчанию.


		 ![](../../Sub_Image.gif)
		 [LoadFromFile](IAuditFiltersContainer.LoadFromFile.htm)
		 Метод LoadFromFile
		 осуществляет загрузку контейнера из файла.


		 ![](../../Sub_Image.gif)
		 [SaveToFile](IAuditFiltersContainer.SaveToFile.htm)
		 Метод SaveToFile осуществляет
		 сохранение контейнера в файл.
