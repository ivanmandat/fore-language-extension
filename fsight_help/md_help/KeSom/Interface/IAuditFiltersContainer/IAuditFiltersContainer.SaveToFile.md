# IAuditFiltersContainer.SaveToFile


## Синтаксис


SaveToFile(FileName: String);


## Параметры


FileName. Полное имя файла,
 из которого должен быть загружен контейнер.


## Описание


Метод SaveToFile осуществляет
 сохранение контейнера в файл.
