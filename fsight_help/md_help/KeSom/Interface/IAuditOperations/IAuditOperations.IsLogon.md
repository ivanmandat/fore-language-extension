# IAuditOperations.IsLogon


## Синтаксис


IsLogon: Boolean;


## Описание


Свойство IsLogon возвращает
 значение, указывающее, соответствует ли текущая запись операции подключения
 к репозиторию.


## Комментарии


Свойство возвращает True, если
 данная запись соответствует операции входа в систему, и False
 для всех остальных операций.
