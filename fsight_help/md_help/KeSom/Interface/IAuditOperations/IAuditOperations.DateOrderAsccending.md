# IAuditOperations.DateOrderAsccending


## Синтаксис


		DateOrderAsccending: Boolean;


## Описание


Свойство DateOrderAsccending
 возвращает порядок сортировки протокола доступа по дате.


## Комментарии


Свойство возвращает True, если
 протокол доступа отсортирован в порядке увеличения даты, и False,
 если в порядке уменьшения даты операции.
