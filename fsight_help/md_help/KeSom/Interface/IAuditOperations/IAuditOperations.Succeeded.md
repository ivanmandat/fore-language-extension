# IAuditOperations.Succeeded


## Синтаксис


Succeeded: Boolean;


## Описание


Свойство Succeeded возвращает
 признак удачного выполнения операции.


## Комментарии


Свойство возвращает True, если
 операция завершена успешно и False,
 если операция была запрещена.
