# ILogonHoursPeriod


## Описание


Интерфейс ILogonHoursPeriod
 содержит свойства периода, в котором осуществляется ограничение времени
 доступа пользователя.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Begin_](ILogonHoursPeriod.Begin_.htm)
		 Свойство Begin_ возвращает
		 начальную дату периода, в котором осуществляется ограничение времени
		 доступа пользователя.


		 ![](../../Property_Image.gif)
		 [End_](ILogonHoursPeriod.End_.htm)
		 Свойство End_ возвращает
		 конечную дату периода, в котором осуществляется ограничение времени
		 доступа пользователя.


		 ![](../../Property_Image.gif)
		 [Hours](ILogonHoursPeriod.Hours.htm)
		 Свойство Hours возвращает
		 настройки доступа по часам в течение дня.
