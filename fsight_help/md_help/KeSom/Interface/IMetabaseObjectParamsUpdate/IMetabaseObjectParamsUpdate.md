# IMetabaseObjectParamsUpdate


Сборка: Metabase;


## Описание


Интерфейс IMetabaseObjectParamsUpdate
 используется для работы с параметрами объекта репозитория после вызова
 метода [IMetabaseObjectParams.BeginUpdate](../IMetabaseObjectParams/IMetabaseObjectParams.BeginUpdate.htm).
