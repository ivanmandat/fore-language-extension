# IAuditFilterCondition.MinDate


## Синтаксис


		MinDate: DateTime;


## Описание


Свойство MinDate определяет
 начальное значение даты и времени, используемое при фильтрации записей.


По умолчанию устанавливается значение даты на 3 дня раньше текущей.
