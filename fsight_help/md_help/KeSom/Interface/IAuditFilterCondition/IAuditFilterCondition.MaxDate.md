# IAuditFilterCondition.MaxDate


## Синтаксис


		MaxDate: DateTime;


## Описание


Свойство MaxDate определяет
 конечное значение даты и времени, используемое при фильтрации записей.


По умолчанию устанавливается значение текущей даты и времени.
