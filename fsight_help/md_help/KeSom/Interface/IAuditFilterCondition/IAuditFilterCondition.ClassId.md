# IAuditFilterCondition.ClassId


## Синтаксис


		ClassId(Index: Integer): Integer;


## Параметры


Index. Индекс
 класса. Количество классов определяется свойством [IAuditFilterCondition.ClassCount](IAuditFilterCondition.ClassCount.htm).


## Описание


Свойство ClassId определяет
 класс объекта, для которого будут отображены записи в протоколе доступа.
 Класс определяется с помощью перечисления [MetabaseObjectClass](../../Enums/MetabaseObjectClass.htm).
