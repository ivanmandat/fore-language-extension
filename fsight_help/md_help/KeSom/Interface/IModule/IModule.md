# IModule


## Описание


Интерфейс IModule содержит базовые
 свойства модулей и форм среды разработки.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Assembly](IModule.Assembly.htm)
		 Свойство Assembly возвращает
		 параметры временной сборки, соответствующей данному объекту.


		 ![](../../Property_Image.gif)
		 [Modified](IModule.Modified.htm)
		 Свойство Modified определяет
		 признак наличия несохраненных изменений.


		 ![](../../Property_Image.gif)
		 [ParentAssembly](IModule.ParentAssembly.htm)
		 Свойство ParentAssembly
		 возвращает описание родительской сборки.


		 ![](../../Property_Image.gif)
		 [Standalone](IModule.Standalone.htm)
		 Свойство Standalone
		 возвращает признак нахождения объекта в составе сборки.


		 ![](../../Property_Image.gif)
		 [Text](IModule.Text.htm)
		 Свойство Text определяет
		 текст модуля.
