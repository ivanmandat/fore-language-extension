# ICertSigner


Сборка: Metabase;


## Описание


Интерфейс ICertSigner содержит
 свойства и методы подписчика цифровой подписью.


## Иерархия наследования


ICertSigner


## Комментарии


Подписчик цифровой подписью можно получить, используя метод [ICertProvider.OpenSigner](../ICertProvider/ICertProvider.OpenSigner.htm)
 или [ICertProvider.OpenSignerBase64](../ICertProvider/ICertProvider.OpenSignerBase64.htm).
