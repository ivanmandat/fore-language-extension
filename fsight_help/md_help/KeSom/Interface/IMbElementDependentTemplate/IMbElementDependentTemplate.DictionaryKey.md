# IMbElementDependentTemplate.DictionaryKey


## Синтаксис


DictionaryKey: Integer;


## Описание


Свойство DictionaryKey возвращает
 ключ справочника НСИ, зависящего от объекта репозитория.


## Комментарии


Для получения ключа элемента справочника НСИ, зависящего от объекта
 репозитория, используйте свойство [IMbElementDependentTemplate.ElementKey](IMbElementDependentTemplate.ElementKey.htm).
