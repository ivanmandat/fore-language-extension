# IMbElementDependentTemplate.ObjectKeysCount


## Синтаксис


ObjectKeysCount: Integer;


## Описание


Свойство ObjectKeysCount возвращает
 количество ключей объектов репозитория, зависящих от объекта репозитория.


## Комментарии


Для получения массива ключей объектов репозитория, зависящего от объекта
 репозитория, используйте свойство [IMbElementDependentTemplate.ObjectKeys](IMbElementDependentTemplate.ObjectKeys.htm).
