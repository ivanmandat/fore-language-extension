# IMbElementDependentTemplate


Сборка: Metabase;


## Описание


Интерфейс IMbElementDependentTemplate
 предназначен для работы с шаблоном зависимостей объектов и элементов справочников
 НСИ от объектов репозитория.


## Иерархия наследования


IMbElementDependentTemplate


## Комментарии


Для создания шаблона зависимостей объектов и элементов справочников
 НСИ от объектов репозитория используйте конструкторы класса [MbElementDependentTemplateClass](../../Class/MbElementDependentTemplateClass/MbElementDependentTemplateClass.htm).
