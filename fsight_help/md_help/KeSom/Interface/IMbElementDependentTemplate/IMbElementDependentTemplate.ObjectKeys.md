# IMbElementDependentTemplate.ObjectKeys


## Синтаксис


ObjectKeys: Array;


## Описание


Свойство ObjectKeys возвращает
 массив ключей объектов репозитория, зависящих от объекта репозитория.


## Комментарии


Для получения количества ключей объектов репозитория, зависящего от
 объекта репозитория, используйте свойство [IMbElementDependentTemplate.ObjectKeysCount](IMbElementDependentTemplate.ObjectKeysCount.htm).
