# IMbElementDependentTemplate.ElementKeys


## Синтаксис


ElementKeys: Array;


## Описание


Свойство ElementKeys возвращает
 массив ключей элементов справочника НСИ, зависящего от объекта репозитория.


## Комментарии


Для получения количества ключей элементов справочника НСИ, зависящего
 от объекта репозитория, используйте свойство [IMbElementDependentTemplate.ElementKeysCount](IMbElementDependentTemplate.ElementKeysCount.htm).
