# IMbElementDependentTemplate.ElementKeysCount


## Синтаксис


ElementKeysCount: Integer;


## Описание


Свойство ElementKeysCount возвращает
 количество ключей элементов справочника НСИ, зависящего от объекта репозитория.


## Комментарии


Для получения массива ключей элементов справочника НСИ, зависящего от
 объекта репозитория, используйте свойство [IMbElementDependentTemplate.ElementKeys](IMbElementDependentTemplate.ElementKeys.htm).
