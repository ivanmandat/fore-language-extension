# IMbElementDependent.Object


## Синтаксис


Object: [IMetabaseObjectDescriptor](../IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm);


## Описание


Свойство Object определяет объект
 репозитория, от которого зависят элементы справочника НСИ.


## Комментарии


Для получения справочника НСИ, элементы которого зависят от объекта
 репозитория, используйте свойство [IMbElementDependent.Dictionary](IMbElementDependent.Dictionary.htm).
