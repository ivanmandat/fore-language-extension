# IMbElementDependent


Сборка: Metabase;


## Описание


Интерфейс IMbElementDependent
 предназначен для работы с записями зависимости элементов справочника НСИ
 от объектов репозитория.


## Иерархия наследования


           [IMbElementDependentTemplate](../IMbElementDependentTemplate/IMbElementDependentTemplate.htm)


           IMbElementDependent


## Комментарии


Для работы с шаблоном зависимостей объектов и элементов справочников
 НСИ от объектов репозитория используйте свойства интерфейса [IMbElementDependentTemplate](../IMbElementDependentTemplate/IMbElementDependentTemplate.htm).
