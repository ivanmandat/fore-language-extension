# ICredentials.Authentication


## Синтаксис


Authentication: [AuthenticationMode](../../Enums/AuthenticationMode.htm);


## Описание


Свойство Authentication возвращает
 тип аутентификации, используемый при подключении с текущими учетными данными.


## Комментарии


Свойство вернёт тот тип, который был указан при создании учётных данных
 с помощью метода [ISecurityPackage.CreateCredentials](../ISecurityPackage/ISecurityPackage.CreateCredentials.htm).
