# ICredentials.SlotValue


## Синтаксис


SlotValue(SlotName: String): Variant;


## Параметры


SlotName. Наименование столбца
 таблицы учетных данных репозитория.


## Описание


Свойство SlotValue определяет
 значение столбца таблицы учетных данных репозитория, наименование которого
 передается с помощью параметра SlotName.
