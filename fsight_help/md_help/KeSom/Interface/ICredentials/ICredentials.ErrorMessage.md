# ICredentials.ErrorMessage


## Синтаксис


ErrorMessage: String;


## Описание


Свойство ErrorMessage определяет комментарий для заведомо неудачного входа.


## Комментарии


Заданный комментарий будет отображен в поле «Примечание» протокола доступа. Заведомо неудачный вход в систему определяется свойством [ICredentials.ForceError](ICredentials.ForceError.htm).
