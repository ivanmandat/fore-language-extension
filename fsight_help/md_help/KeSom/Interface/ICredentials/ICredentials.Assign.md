# ICredentials.Assign


## Синтаксис


Assign(Source: [ICredentials](ICredentials.htm));


## Параметры


Source - учетные
 данные репозитория, используемые в качестве источника.


## Описание


Метод Assign заполняет учетные
 данные пользователя, учетными данными, которые передаются посредством
 параметра Source.
