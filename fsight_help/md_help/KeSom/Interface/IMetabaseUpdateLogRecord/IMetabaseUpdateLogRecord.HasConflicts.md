# IMetabaseUpdateLogRecord.HasConflicts


## Синтаксис


HasConflicts: Boolean;


## Описание


Свойство HasConflicts возвращает
 признак наличия конфликта при определении соответствия объекта в обновлении
 и репозитории.


## Комментарии


Свойство возвращает значение True,
 если тип объекта в обновлении не соответствует типу объекта в репозитории.
