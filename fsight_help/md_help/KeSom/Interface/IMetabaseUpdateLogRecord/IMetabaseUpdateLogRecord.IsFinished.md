# IMetabaseUpdateLogRecord.IsFinished


## Синтаксис


IsFinished: Boolean;


## Описание


Свойство IsFinished возвращает
 признак успешного применения элемента обновления.


## Комментарии


Свойство возвращает значение True,
 если установка элемента обновления завершилась успешно и False,
 если при установке возникли какие-либо ошибки.
