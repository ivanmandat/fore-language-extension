# IMetabaseUpdateLogRecord.ErrorSkipped


## Синтаксис


ErrorSkipped: Boolean;


## Описание


Свойство ErrorSkipped возвращает
 признак пропуска ошибки в процессе применения элемента обновления.


## Комментарии


Свойство возвращает значение True,
 если ошибка была пропущена и False,
 если ошибка требует обработки.
