# IMetabaseUpdateLogRecord.Error


## Синтаксис


Error: [IException](ForeSys.chm::/Interface/IException/IException.htm);


## Описание


Свойство Error возвращает ошибку,
 которая возникла в процессе применения элемента обновления.


## Комментарии


Свойство актуально, если свойство [HasError](IMetabaseUpdateLogRecord.HasError.htm)
 возвращает значение True.
