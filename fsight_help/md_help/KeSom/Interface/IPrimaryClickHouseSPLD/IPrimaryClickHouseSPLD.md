# IPrimaryClickHouseSPLD


Сборка: Metabase;


## Описание


Интерфейс IPrimaryClickHouseSPLD
 содержит свойства и методы модуля безопасности, используемого при подключении
 к серверам на базе СУБД ClickHouse.
