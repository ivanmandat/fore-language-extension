# ISaveMetabaseObjectHandlers


Сборка: Metabase;


## Описание


Интерфейс ISaveMetabaseObjectHandlers
 предназначен для работы с коллекций обработчиков событий, наступающих
 при изменении метаданных объектов.


## Иерархия наследования


ISaveMetabaseObjectHandlers


## Комментарии


Коллекцию обработчиков событий можно получить, используя метод [ISpecialObjects.SaveMetabaseObjectHandlers](../ISpecialObjects/ISpecialObjects.SaveMetabaseObjectHandlers.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ISaveMetabaseObjectHandlers.Count.htm)
		 Свойство Count возвращает
		 количество обработчиков событий в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ISaveMetabaseObjectHandlers.Item.htm)
		 Свойство Item возвращает
		 настройки обработчика событий с указанным индексом.

		 [ItemByClassId](ISaveMetabaseObjectHandlers.ItemByClassId.htm)
		 Свойство ItemByClassId
		 возвращает настройки обработчика событий для указанного класса
		 объектов.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ISaveMetabaseObjectHandlers.Add.htm)

		 ![](../../Sub_Image.gif)
		 [Clear](ISaveMetabaseObjectHandlers.Clear.htm)
		 Метод Clear удаляет

		 ![](../../Sub_Image.gif)
		 [Remove](ISaveMetabaseObjectHandlers.Remove.htm)
		 Метод Remove удаляет

		 ![](../../Sub_Image.gif)
		 [RemoveByClassId](ISaveMetabaseObjectHandlers.RemoveByClassId.htm)
		 Метод RemoveByClassId
		 удаляет из коллекции обработчик событий для указанного класса
		 объектов.
