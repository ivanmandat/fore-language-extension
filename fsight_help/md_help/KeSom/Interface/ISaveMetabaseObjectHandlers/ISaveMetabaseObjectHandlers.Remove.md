# ISaveMetabaseObjectHandlers.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс обработчика событий.


## Описание


Метод Remove удаляет из коллекции
 обработчик событий с указанным индексом.


## Комментарии


Метод возвращает значение True,
 если удаление завершилось успешно и False,
 если удаление по каким-либо причинам невозможно.
