# IAuditFilter


Сборка: Metabase;


## Описание


Интерфейс IAuditFilter содержит
 свойства и методы фильтра протокола доступа менеджера безопасности.


## Иерархия наследования


           IAuditFilter


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IAuditFilter.Count.htm)
		 Свойство Count возвращает
		 количество условий фильтрации.


		 ![](../../Property_Image.gif)
		 [DateOrderAsccending](IAuditFilter.DateOrderAsccending.htm)
		 Свойство DateOrderAsccending
		 определяет порядок сортировки по дате.


		 ![](../../Property_Image.gif)
		 [Item](IAuditFilter.Item.htm)
		 Свойство Item возвращает
		 объект, позволяющий редактировать условие фильтрации.


		 ![](../../Property_Image.gif)
		 [Top](IAuditFilter.Top.htm)
		 Свойство Top определяет
		 количество рассматриваемых записей протокола доступа.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IAuditFilter.Add.htm)
		 Метод Add добавляет
		 условие фильтрации.


		 ![](../../Sub_Image.gif)
		 [Append](IAuditFilter.Append.htm)
		 Метод Append дополняет
		 фильтр новыми условиями.


		 ![](../../Sub_Image.gif)
		 [Assign](IAuditFilter.Assign.htm)
		 Метод Assign копирует
		 условие фильтрации.


		 ![](../../Sub_Image.gif)
		 [Clear](IAuditFilter.Clear.htm)
		 Метода Clear позволяет
		 очистить фильтр от всех условий.


		 ![](../../Sub_Image.gif)
		 [Move](IAuditFilter.Move.htm)
		 Метод Move перемещает
		 условие с одной позиции на другую.


		 ![](../../Sub_Image.gif)
		 [Remove](IAuditFilter.Remove.htm)
		 Метод Remove позволяет
		 удалить условие фильтра.
