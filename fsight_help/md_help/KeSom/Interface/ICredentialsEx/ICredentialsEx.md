# ICredentialsEx


Сборка: Metabase;


## Описание


Интерфейс ICredentialsEx содержит
 расширенные свойства учетных данных пользователя.


## Иерархия наследования


ICredentialsEx


## Комментарии


Для получения расширенных свойств приведите учетные данные, описываемые
 интерфейсом [ICredentials](../ICredentials/ICredentials.htm),
 к интерфейсу ICredentialsEx.
