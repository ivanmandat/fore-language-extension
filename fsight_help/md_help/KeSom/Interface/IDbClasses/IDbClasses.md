# IDbClasses


## Описание


Интерфейс IDbClasses содержит
 свойства и методы коллекции классов объектов, доступных в платформе.


## Комментарии


Коллекцию классов можно получить, используя одно из следующих свойств:


	- [IMetabase.Classes](../IMetabase/IMetabase.Classes.htm);


	- [IMetabaseManager.Classes](../IMetabaseManager/IMetabaseManager.Classes.htm).
