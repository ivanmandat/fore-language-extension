# IDbClasses.Item


## Синтаксис


Item(Index: Integer): [IDbClass](../IDbClass/IDbClass.htm);


## Параметры


Index.
 Индекс класса объекта.


## Описание


Свойство Item возвращает информацию
 о классе объекта, индекс которого передается в качестве входного параметра
 Index.
