# IDbClasses.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IDbClass](../IDbClass/IDbClass.htm);


## Параметры


Key. Ключ
 класса, по которому будет осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает информацию о классе объекта, ключ которого передается
 посредством входного параметра.


## Комментарии


Если класс объекта с указанным ключом не найден, то метод возвращает
 значение Null.
