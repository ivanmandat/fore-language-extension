# IMetabaseCustomClassOperationsWeb.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс операции в коллекции.


## Описание


Метод Remove осуществляет удаление
 операции из коллекции по индексу.


## Комментарии


Метод возвращает значение True,
 если удаление прошло успешно.
