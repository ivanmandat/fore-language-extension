# IMetabaseCustomClassOperationsWeb


Сборка: Metabase;


## Описание


Интерфейс IMetabaseCustomClassOperationsWeb
 предназначен для работы с коллекцией пользовательских операций пользовательского
 класса, доступных в веб-приложении.


## Иерархия наследования


IMetabaseCustomClassOperationsWeb


## Комментарии


Коллекцию пользовательских операций, доступных в веб-приложении можно
 получить, используя свойство [IMetabaseCustomClass.OperationsWeb](../IMetabaseCustomClass/IMetabaseCustomClass.OperationsWeb.htm).
