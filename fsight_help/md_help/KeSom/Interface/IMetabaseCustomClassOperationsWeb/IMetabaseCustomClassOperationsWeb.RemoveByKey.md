# IMetabaseCustomClassOperationsWeb.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ удаляемой операции.


## Описание


Метод RemoveByKey осуществляет
 удаление операции из коллекции по ключу.


## Комментарии


Метод возвращает значение True,
 если удаление прошло успешно.
