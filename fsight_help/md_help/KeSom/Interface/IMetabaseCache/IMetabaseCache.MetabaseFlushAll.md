# IMetabaseCache.MetabaseFlushAll


## Синтаксис


MetabaseFlushAll;


## Описание


Метод MetabaseFlushAll осуществляет
 очистку кеша ссылок на объекты сборок и их зависимостей.


## Комментарии


При выполнении метода будут очищены хранящиеся в кеше репозитория кешированные
 данные объектов. Кеш со ссылками на объекты среды разработки и их зависимостями
 очищен не будет.
