# IMetabaseCache.Flush


## Синтаксис


Flush(Key: Integer): Boolean;


## Параметры


Key. Ключ объекта репозитория.


## Описание


Метод Flush очищает ссылки на
 объект репозитория по его ключу.


## Комментарии


Метод возвращает значение True,
 если очистка прошла успешно.
