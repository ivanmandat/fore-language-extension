# IUserEmail.IsDefault


## Синтаксис


IsDefault: Boolean;


## Описание


Свойство IsDefault возвращает признак электронного почтового адреса по умолчанию.


## Комментарии


Свойство возвращает значение True, если текущая запись содержит электронный почтовый адрес, используемый по умолчанию.


Значение электронной почты по умолчанию будет доступно в профиле в свойстве [IUserProfile.Email](../IUserProfile/IUserProfile.Email.htm). Для установки адреса по умолчанию используйте метод [SetDefault](IUserEmail.SetDefault.htm).
