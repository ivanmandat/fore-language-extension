# IUserEmail.SetDefault


## Синтаксис


SetDefault;


## Описание


Метод SetDefault устанавливает электронный почтовый адрес, указанный в текущей записи, адресом по умолчанию.


## Комментарии


Значение электронного почтового адреса будет доступно в профиле в свойстве [IUserProfile.Email](../IUserProfile/IUserProfile.Email.htm).
