# IUserEmail.Name


## Синтаксис


Name: String;


## Описание


Свойство Name определяет наименование записи о электронном почтовом адресе.


## Комментарии


Значение данного свойства должно быть уникальным в рамках коллекции [IUserEmails](../IUserEmails/IUserEmails.htm).
