# IForm


## Описание


Интерфейс IForm содержит свойства
 объекта среды разработки - [Форма](UiDevEnv.chm::/01_Development_Environment/02_Work_in_Development_Environment/DevEnv_Object/DevEnv_Form.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Content](IForm.Content.htm)
		 Свойство Content определяет
		 содержимое и параметры формы, представленные в формате XML.


## Свойства, унаследованные от [IModule](../IModule/IModule.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Assembly](../IModule/IModule.Assembly.htm)
		 Свойство Assembly возвращает
		 параметры временной сборки, соответствующей данному объекту.


		 ![](../../Property_Image.gif)
		 [Modified](../IModule/IModule.Modified.htm)
		 Свойство Modified определяет
		 признак наличия несохраненных изменений.


		 ![](../../Property_Image.gif)
		 [ParentAssembly](../IModule/IModule.ParentAssembly.htm)
		 Свойство ParentAssembly
		 возвращает описание родительской сборки.


		 ![](../../Property_Image.gif)
		 [Standalone](../IModule/IModule.Standalone.htm)
		 Свойство Standalone
		 возвращает признак нахождения объекта в составе сборки.


		 ![](../../Property_Image.gif)
		 [Text](../IModule/IModule.Text.htm)
		 Свойство Text определяет
		 текст модуля.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CreateFormControl](IForm.CreateFormControl.htm)
		 Метод CreateFormControl
		 осуществляет создание новой формы и возвращает контекст для управления
		 данной формой.
