# IForm.Content


## Синтаксис


Content: String;


## Описание


Свойство Content определяет
 содержимое и параметры формы, представленные в формате XML.


## Комментарии


В данном виде форма хранится в БД репозитория.


Примечание.
 Не рекомендуется изменять значения данного свойство для избежания потери
 работоспособности формы.
