# IMetabaseUpdateSqlNode.ClearSql


## Синтаксис


ClearSql;


## Описание


Метод ClearSql очищает текст SQL-запроса для всех драйверов.


## Комментарии


Установить текст SQL-запроса, используемого для всех драйверов по умолчанию, можно используя свойство [IMetabaseUpdateSqlNode.SetDefaultSql](IMetabaseUpdateSqlNode.SetDefaultSql.htm). Установить текст SQL-запроса для определенного драйвера можно используя свойство [IMetabaseUpdateSqlNode.Sql](IMetabaseUpdateSqlNode.Sql.htm).

Использование метода приведено в описании [IMetabaseUpdateSqlNode.SetDefaultSql](IMetabaseUpdateSqlNode.SetDefaultSql.htm).
