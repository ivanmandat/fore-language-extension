# IMetabaseUpdateSqlNode.Sql


## Синтаксис


Sql(Driver: String): String;


## Параметры


Driver. Идентификатор драйвера
 СУБД.


## Описание


Свойство Sql определяет текст
 запроса SQL-оператора, выполняемого при обновлении.


## Комментарии


Список идентификаторов, которые могут использоваться в качестве значения
 параметра Driver, представлен
 в разделе «[Сводный
 список идентификаторов](Setup.chm::/06_AK_Client_Config/Configuring_repository_list_in_the_system_registry.htm#id)». Укажите идентификатор той
 СУБД, которая установлена на сервере, где будет применяться обновление.
