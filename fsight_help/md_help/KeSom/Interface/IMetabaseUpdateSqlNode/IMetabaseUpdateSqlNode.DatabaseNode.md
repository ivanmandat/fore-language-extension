# IMetabaseUpdateSqlNode.DatabaseNode


## Синтаксис


DatabaseNode: [IMetabaseUpdateObjectNode](../IMetabaseUpdateObjectNode/IMetabaseUpdateObjectNode.htm);


## Описание


Свойство DatabaseNode возвращает
 БД, к которой будет выполняться SQL-запрос в виде объекта обновления.


## Комментарии


Свойство будет доступно после того, как было определено значение свойства
 [IMetabaseUpdateSqlNode.Database](IMetabaseUpdateSqlNode.Database.htm).


Изменить базу данных можно, используя свойство [IMetabaseUpdateObjectNode.Object](../IMetabaseUpdateObjectNode/IMetabaseUpdateObjectNode.Object.htm)
 или [IMetabaseUpdateSqlNode.Database](IMetabaseUpdateSqlNode.Database.htm).
