# IMbElementDependents.Where


## Синтаксис


Where: [IMbElementDependentTemplate](../IMbElementDependentTemplate/IMbElementDependentTemplate.htm);


## Описание


Свойство Where определяет шаблон,
 на котором строится список зависимостей элементов справочника НСИ от объектов
 репозитория.


## Комментарии


Для добавления в коллекцию записи зависимости элементов справочника
 НСИ от объектов репозитория используйте метод [IMbElementDependents.Add](IMbElementDependents.Add.htm).
