# IMbElementDependents.Database


## Синтаксис


Database: [IMbElementDependentsDatabase](../IMbElementDependentsDatabase/IMbElementDependentsDatabase.htm);


## Описание


Свойство Database возвращает
 репозиторий, в котором осуществляется работа с зависимостями элементов
 справочника НСИ от объектов репозитория.


## Комментарии


Для определения шаблона, на котором строится список зависимостей элементов
 справочника НСИ от объектов репозитория, используйте свойство [IMbElementDependents.Where](IMbElementDependents.Where.htm).
