# IMbElementDependents.Clear


## Синтаксис


Clear;


## Описание


Метод Clear удаляет все записи
 зависимостей элементов справочника НСИ от объектов репозитория.


## Комментарии


Для удаления записи зависимостей элементов справочника НСИ от объектов
 репозитория по её индексу используйте метод [IMbElementDependents.Remove](IMbElementDependents.Remove.htm).
