# IMbElementDependents.Add


## Синтаксис


Add(Value: [IMbElementDependentTemplate](../IMbElementDependentTemplate/IMbElementDependentTemplate.htm)):
 Integer;


## Параметры


Value. Шаблон зависимости элементов
 справочников НСИ от объектов репозитория.


## Описание


Метод Add добавляет в коллекцию
 запись зависимости элементов справочника НСИ от объектов репозитория.


## Комментарии


Для получения значения записи зависимости элементов справочника НСИ
 от объектов репозитория по её индексу используйте свойство [IMbElementDependents.Item](IMbElementDependents.Item.htm).
