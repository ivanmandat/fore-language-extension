# IMbElementDependents.Item


## Синтаксис


Item(Index: Integer): [IMbElementDependent](../IMbElementDependent/IMbElementDependent.htm);


## Параметры


Index. Индекс записи зависимости
 элементов справочника НСИ от объектов репозитория.


## Описание


Свойство Item возвращает значение
 записи зависимости элементов справочника НСИ от объектов репозитория по
 её индексу.


## Комментарии


Для получения количества записей зависимости элементов справочника НСИ
 от объектов репозитория используйте свойство [IMbElementDependents.Count](IMbElementDependents.Count.htm).
