# IMbElementDependents


Сборка: Metabase;


## Описание


Интерфейс IMbElementDependents
 предназначен для работы с коллекцией записей зависимости объектов и элементов
 справочника НСИ от объектов репозитория.


## Иерархия наследования


           IMbElementDependents


## Комментарии


Для работы с конкретной записью зависимости элементов справочника НСИ
 от объектов репозитория используйте свойства интерфейса [IMbElementDependent](../IMbElementDependent/IMbElementDependent.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMbElementDependents.Count.htm)
		 Свойство Count возвращает
		 количество записей зависимости элементов справочника НСИ от объектов
		 репозитория.


		 ![](../../Property_Image.gif)
		 [Database](IMbElementDependents.Database.htm)
		 Свойство Database возвращает
		 репозиторий, в котором осуществляется работа с зависимостями элементов
		 справочника НСИ от объектов репозитория.


		 ![](../../Property_Image.gif)
		 [Item](IMbElementDependents.Item.htm)
		 Свойство Item возвращает
		 значение записи зависимости элементов справочника НСИ от объектов
		 репозитория по её индексу.


		 ![](../../Property_Image.gif)
		 [Modified](IMbElementDependents.Modified.htm)
		 Свойство Modified определяет
		 признак изменения записей зависимости элементов справочника НСИ
		 от объектов репозитория.


		 ![](../../Property_Image.gif)
		 [Where](IMbElementDependents.Where.htm)
		 Свойство Where определяет
		 шаблон, на котором строится список зависимостей элементов справочника
		 НСИ от объектов репозитория.
