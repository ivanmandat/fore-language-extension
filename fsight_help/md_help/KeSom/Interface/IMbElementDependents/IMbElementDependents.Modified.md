# IMbElementDependents.Modified


## Синтаксис


Modified: Boolean;


## Описание


Свойство Modified определяет
 признак изменения записей зависимости элементов справочника НСИ от объектов
 репозитория.


## Комментарии


Допустимые значения:


	- True. Записи зависимости
	 элементов справочника НСИ от объектов репозитория были изменены;


	- False. Записи зависимости
	 элементов справочника НСИ от объектов репозитория не были изменены.
