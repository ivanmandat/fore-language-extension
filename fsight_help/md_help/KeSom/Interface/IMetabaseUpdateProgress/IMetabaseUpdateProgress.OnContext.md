# IMetabaseUpdateProgress.OnContext


## Синтаксис


OnContext(Context: [IMetabaseUpdateContext](../IMetabaseUpdateContext/IMetabaseUpdateContext.htm));


## Параметры


OnContext. Дополнительные настройки
 обновления.


## Описание


Метод OnContext реализует
 событие, возникающее при обновлении с использованием дополнительных настроек.


## Комментарии


Данное событие генерируется во время применения обновления.
