# IPrimaryMsSqlSPLD.Server


## Синтаксис


Server: String;


## Описание


Свойство Server определяет псевдоним, под которым зарегистрирован сервер MS SQL: название сервера, на котором хранится подключаемая база данных (репозиторий).
