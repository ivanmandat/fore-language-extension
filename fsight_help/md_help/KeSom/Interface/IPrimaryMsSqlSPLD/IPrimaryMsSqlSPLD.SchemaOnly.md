# IPrimaryMsSqlSPLD.SchemaOnly


## Синтаксис


SchemaOnly: String;


## Описание


Свойство SchemaOnly определяет
 идентификатор пользовательской схемы в [базе
 данных](IPrimaryMsSqlSPLD.DatabaseOnly.htm).


## Комментарии


Свойство актуально для использования, если на сервере используется тип
 СУБД Microsoft SQL Server. По умолчанию используется схема «dbo».
