# IScPacks.FindById


## Синтаксис


FindById(Id: String): [IScPack](../IScPack/IScPack.htm);


## Параметры


Id. Идентификатор
 пакета безопасности, по которому будет производиться поиск.


## Описание


Метод FindById осуществляет
 поиск пакета безопасности, идентификатор которого передается посредством
 параметра Id.


## Комментарии


В случае удачного поиска метод возвращает параметры поиск пакета, иначе
 возвращает значение Null.
