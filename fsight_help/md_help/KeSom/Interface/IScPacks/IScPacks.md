# IScPacks


## Описание


Интерфейс IScPacks содержит
 свойства и методы коллекции пакетов безопасности платформы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IScPacks.Count.htm)
		 Свойство Count возвращает
		 количество пакетов безопасности, доступных в платформе.


		 ![](../../Property_Image.gif)
		 [Item](IScPacks.Item.htm)
		 Свойство Item возвращает
		 параметры пакета безопасности.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](IScPacks.FindById.htm)
		 Метод FindById осуществляет
		 поиск пакета безопасности по идентификатору.
