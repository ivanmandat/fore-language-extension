# IMetabaseCustomClassOperations.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс операции в коллекции.


## Описание


Метод Remove осуществляет удаление операции из коллекции по индексу. Метод возвращает значение True, если удаление прошло успешно.
