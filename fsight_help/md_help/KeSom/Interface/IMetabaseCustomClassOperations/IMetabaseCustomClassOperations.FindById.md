# IMetabaseCustomClassOperations.FindById


## Синтаксис


FindById(Id: String): [IMetabaseCustomClassOperation](../IMetabaseCustomClassOperation/IMetabaseCustomClassOperation.htm);


## Параметры


Id - идентификатор операции.


## Описание


Метод FindById осуществляет поиск операции в коллекции по идентификатору.


В случае удачного поиска метод возвращает параметры операции, иначе возвращает значение Null.
