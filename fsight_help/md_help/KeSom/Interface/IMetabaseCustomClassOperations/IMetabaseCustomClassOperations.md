# IMetabaseCustomClassOperations


Сборка: Metabase;


## Описание


Интерфейс IMetabaseCustomClassOperations
 содержит свойства и методы для работы с коллекцией пользовательских операций
 пользовательского класса.


## Иерархия наследования


IMetabaseCustomClassOperations


## Комментарии


Коллекцию пользовательских операций можно получить, используя свойство
 [IMetabaseCustomClass.Operations](../IMetabaseCustomClass/IMetabaseCustomClass.Operations.htm).
