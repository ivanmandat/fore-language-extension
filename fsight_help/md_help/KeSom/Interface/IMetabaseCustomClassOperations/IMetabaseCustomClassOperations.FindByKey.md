# IMetabaseCustomClassOperations.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IMetabaseCustomClassOperation](../IMetabaseCustomClassOperation/IMetabaseCustomClassOperation.htm);


## Параметры


Key - ключ операции.


## Описание


Метод FindByKey осуществляет поиск операции в коллекции по ключу.


В случае удачного поиска метод возвращает параметры операции, иначе возвращает значение Null.
