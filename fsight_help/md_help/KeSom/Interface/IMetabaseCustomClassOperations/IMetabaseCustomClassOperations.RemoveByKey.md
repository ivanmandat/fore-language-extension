# IMetabaseCustomClassOperations.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key - ключ операции.


## Описание


Метод RemoveByKey осуществляет удаление операции из коллекции по ключу. Метод возвращает значение True, если удаление прошло успешно.
