# IMetabaseObjectCacheItem


## Описание


Интерфейс IMetabaseObjectCacheItem
 содержит свойства и методы для работы с записью в кеше объекта.


## Комментарии


Запись в кеше объекта содержит экземпляр объекта, открытого с определенными
 значениями параметров.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ParamValues](IMetabaseObjectCacheItem.ParamValues.htm)
		 Свойство ParamValues
		 возвращает коллекцию значений параметров, с которыми был открыт
		 объект при создании записи в кеше.


		 ![](../../Property_Image.gif)
		 [Timestamp](IMetabaseObjectCacheItem.Timestamp.htm)
		 Свойство Timestamp
		 возвращает время создания записи в кеше объекта.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Flush](IMetabaseObjectCacheItem.Flush.htm)
		 Метод Flush осуществляет
		 очистку записи в кеше объекта.


		 ![](../../Sub_Image.gif)
		 [Open](IMetabaseObjectCacheItem.Open.htm)
		 Метод Open возвращает
		 экземпляр объекта, которому соответствует запись в кеше.


		 ![](../../Sub_Image.gif)
		 [Refresh](IMetabaseObjectCacheItem.Refresh.htm)
		 Метод Refresh осуществляет
		 обновление записи в кеше объекта.


		 ![](../../Sub_Image.gif)
		 [Replace](IMetabaseObjectCacheItem.Replace.htm)
		 Метод Replace осуществляет
		 замену экземпляра объекта в кеше.
