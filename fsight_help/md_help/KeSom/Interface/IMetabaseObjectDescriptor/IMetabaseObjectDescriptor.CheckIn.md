# IMetabaseObjectDescriptor.CheckIn


## Синтаксис


		CheckIn;


## Описание


Метод CheckIn осуществляет снятие
 блокировки на редактирование с объекта.


## Комментарии


Если объект добавлен в систему управления версиями, то вызов метода
 приведет к публикации изменений. Если при публикации изменений возникли
 ошибки, то будет сгенерирована исключительная ситуация.
