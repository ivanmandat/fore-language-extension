# IMetabaseObjectDescriptor.GetSecurity


## Синтаксис


GetSecurity: [IAccessObjectSecurity](../IAccessObjectSecurity/IAccessObjectSecurity.htm);


## Описание


Метод GetSecurity получает политику
 доступа объекта.


## Комментарии


К политике доступа объекта относится получение пользователей/групп,
 элементов доступа и прав по элементам.
