# IMetabaseObjectDescriptor.TrackElementDependents


## Синтаксис


		TrackElementDependents: Boolean;


## Описание


Свойство TrackElementDependents
 определяет признак отслеживания объектов, зависящих от элементов текущего
 объекта.


## Комментарии


Свойство актуально для справочников НСИ. Изменение данного свойства
 приведет к автоматическому изменению свойства [IMetabaseObjectDescriptor.ElementDependenciesTrackingType](IMetabaseObjectDescriptor.ElementDependenciesTrackingType.htm).
