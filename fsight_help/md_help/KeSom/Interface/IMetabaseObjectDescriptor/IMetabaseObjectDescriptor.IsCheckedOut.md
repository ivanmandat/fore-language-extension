# IMetabaseObjectDescriptor.IsCheckedOut


## Синтаксис


		IsCheckedOut: Boolean;


## Описание


Метод IsCheckedOut возвращает
 признак блокировки объекта на редактирование.


## Комментарии


Свойство возвращает значение True,
 если для объекта был вызван метод [IMetabaseObjectDescriptor.CheckOut](IMetabaseObjectDescriptor.CheckOut.htm),
 но не был вызван метод [IMetabaseObjectDescriptor.CheckIn](IMetabaseObjectDescriptor.CheckIn.htm).
