# IMetabaseObjectDescriptor.UpdateTimestamp


## Синтаксис


		UpdateTimestamp;


## Описание


Метод UpdateTimestamp обновляет
 временную отметку [справочника](UiNavObj.chm::/reference_book/Create.htm).


## Комментарии


Для получения даты и времени последнего сохранения справочника используйте
 свойство [IMetabaseObjectDescriptor.Timestamp](IMetabaseObjectDescriptor.Timestamp.htm).
