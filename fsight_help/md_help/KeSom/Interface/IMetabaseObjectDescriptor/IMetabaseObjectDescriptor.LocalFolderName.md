# IMetabaseObjectDescriptor.LocalFolderName


## Синтаксис


LocalFolderName: String;


## Описание


Свойство LocalFolderName возвращает
 путь к папке на диске, в которой хранятся локальные файлы объекта, добавленного
 в систему управления версиями.


## Комментарии


Свойство актуально для объектов, добавленных в систему управления версиями.
