# IMetabaseObjectDescriptor.CheckInEx


## Синтаксис


		CheckInEx(Var ResId: Integer): Boolean;


## Параметры


ResId. Переменная, в которую
 будет помещен код ошибки.


## Описание


Метод CheckInEx осуществляет
 публикацию изменений объекта.


## Комментарии


Данный метод актуально использовать, если объект добавлен в систему
 управления версиями.


При вызове метода будет открыто окно публикации изменений объекта. При
 удачной публикации метод возвращает значение True.
 Код ошибки при этом будет «0».


Если публикация изменений не произошла, то метод вернет значение False. В параметр ResId
 будет помещен код ошибки.
