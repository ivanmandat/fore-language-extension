# IMetabaseObjectDescriptor.LocalFileNames


## Синтаксис


LocalFileNames: Array;


## Описание


Свойство LocalFileNames возвращает
 строковый массив, содержащий список локальных файлов, в которых хранятся
 данные объекта.


## Комментарии


Свойство актуально для объектов, добавленных в систему управления версиями.


В данных файлах хранится локальная версия объекта, добавленного в систему
 управления версиями.
