# IMetabaseObjectDescriptor.IsShared


## Синтаксис


		IsShared: Boolean;


## Описание


Свойство IsShared возвращает
 признак наличия объекта в системе управления версиями.


## Комментарии


Свойство возвращает значение True,
 если объект был помещен в систему управления версиями. Для помещения объекта
 в систему управления версиями используйте метод [IMetabaseObjectDescriptor.Share](IMetabaseObjectDescriptor.Share.htm).
