# IMetabaseObjectDescriptor.HistoryVCS


## Синтаксис


		HistoryVCS;


## Описание


Метод HistoryVCS вызывает окно
 просмотра истории публикаций объекта в системе управления версиями.


## Комментарии


Данный метод актуально использовать, если объект добавлен в систему
 управления версиями.
