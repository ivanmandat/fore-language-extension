# IMetabaseAuditPolicy


## Описание


Интерфейс IMetabaseAuditPolicy
 содержит свойства, используемые для настройки аудита по определенным классам
 объектов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [FilterClass](IMetabaseAuditPolicy.FilterClass.htm)
		 Свойство FilterClass
		 определяет маску операций, для которых будет вестись аудит доступа.


		 ![](../../Property_Image.gif)
		 [LogRecordCountThreshold](IMetabaseAuditPolicy.LogRecordCountThreshold.htm)
		 Свойство LogRecordCountThreshold
		 для указанного порога определяет количество записей протокола
		 доступа, при достижении которого пользователю будет выводиться
		 предупреждающее сообщение.


		 ![](../../Property_Image.gif)
		 [TrackClassHistory](IMetabaseAuditPolicy.TrackClassHistory.htm)
		 Свойство TrackClassHistory
		 определяет маску операций, для которых будет вестись история.
