# ISearchEngineServiceBase.Metabase


## Синтаксис


Metabase: [IMetabase](../IMetabase/IMetabase.htm);


## Описание


Свойство Metabase возвращает
 репозиторий, для работы с которым предназначена поисковая платформа.


## Комментарии


Свойство возвращает параметры того репозитория, который был указан в
 качестве параметра в методе [ISearchEngineServiceFactory.CreateSearchEngine](../ISearchEngineServiceFactory/ISearchEngineServiceFactory.CreateSearchEngine.htm).
