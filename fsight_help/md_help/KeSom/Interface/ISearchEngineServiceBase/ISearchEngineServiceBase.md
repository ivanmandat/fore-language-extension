# ISearchEngineServiceBase


Сборка: Metabase;


## Описание


Интерфейс ISearchEngineServiceBase
 содержит базовые свойства и методы для работы с поисковым сервисом.


## Иерархия наследования


ISearchEngineServiceBase


## Комментарии


Интерфейс является базовым для интерфейса [ISearchEngineService](KeBISearch.chm::/Interface/ISearchEngineService/ISearchEngineService.htm).
 Объект для работы с поисковым сервисом можно получить при выполнении метода
 [ISearchEngineServiceFactory.CreateSearchEngine](../ISearchEngineServiceFactory/ISearchEngineServiceFactory.CreateSearchEngine.htm).
