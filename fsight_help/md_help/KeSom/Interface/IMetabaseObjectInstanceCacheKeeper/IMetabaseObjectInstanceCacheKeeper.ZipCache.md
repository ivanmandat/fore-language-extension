# IMetabaseObjectInstanceCacheKeeper.ZipCache


## Синтаксис


ZipCache;


## Описание


Метод ZipCache осуществляет
 сжатие двоичных данных в хранилище закешированных объектов.


Примечание.
 Данный метод предназначен для сжатия данных, сохраненных в хранилище закешированных
 объектов в версиях Prognoz Platform ранее версии 8.0.
