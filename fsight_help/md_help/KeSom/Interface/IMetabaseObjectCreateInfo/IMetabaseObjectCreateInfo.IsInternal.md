# IMetabaseObjectCreateInfo.IsInternal


## Синтаксис


IsInternal: Boolean;


## Описание


Свойство IsInternal определяет
 признак создания внутреннего объекта репозитория.


## Комментарии


Допустимые значения:


	- True. Созданный объект
	 будет внутренним и не будет доступен для пользователей репозитория;


	- False. По умолчанию.
	 Созданный объект не будет внутренним и будет доступен пользователям
	 репозитория.
