# IMetabaseObjectCreateInfo.DefaultName


## Синтаксис


DefaultName: String;


## Описание


Свойство DefaultName возвращает
 наименование объекта репозитория по умолчанию.


## Комментарии


Наименование, возвращаемое данным свойством, будет использоваться для
 формирования нового наименования, если не задано свойство [IMetabaseObjectCreateInfo.Name](IMetabaseObjectCreateInfo.Name.htm).
 Новое наименование формируется путем добавления к наименованию
 по умолчанию ключа вновь созданного объекта. Наименование по умолчанию
 зависит от класса создаваемого объекта.
