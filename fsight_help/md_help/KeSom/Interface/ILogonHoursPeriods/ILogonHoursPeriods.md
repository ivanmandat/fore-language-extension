# ILogonHoursPeriods


## Описание


Интерфейс ILogonHoursPeriods
 содержит свойства и методы для работы с коллекцией периодов, в которые
 необходимо ограничить время доступа пользователя.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ILogonHoursPeriods.Count.htm)
		 Свойство Count возвращает
		 количество периодов, в которых осуществляется ограничение времени
		 доступа пользователя.


		 ![](../../Property_Image.gif)
		 [Item](ILogonHoursPeriods.Item.htm)
		 Свойство Item возвращает
		 параметры периода, в который осуществляется ограничение времени
		 доступа пользователя.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Clear](ILogonHoursPeriods.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех периодов, в которых осуществляется ограничение времени
		 доступа пользователя.


		 ![](../../Sub_Image.gif)
		 [CreateCopy](ILogonHoursPeriods.CreateCopy.htm)
		 Метод CreateCopy создает
		 копии всех периодов, в которых осуществляется ограничение времени
		 доступа пользователя.


		 ![](../../Sub_Image.gif)
		 [CreateHours](ILogonHoursPeriods.CreateHours.htm)
		 Метод CreateHours создает
		 объект, используемый для настройки доступа по часам в течение
		 дня.


		 ![](../../Sub_Image.gif)
		 [FindByDate](ILogonHoursPeriods.FindByDate.htm)
		 Метод FindByDate осуществляет
		 поиск периода, содержащего указанную дату.


		 ![](../../Sub_Image.gif)
		 [Remove](ILogonHoursPeriods.Remove.htm)
		 Метод Remove осуществляет
		 удаление периода, в который осуществляется ограничение времени
		 доступа пользователя.


		 ![](../../Sub_Image.gif)
		 [SetPeriodAccess](ILogonHoursPeriods.SetPeriodAccess.htm)
		 Метод SetPeriodAccess
		 устанавливает параметры доступа пользователя для указанного периода.
