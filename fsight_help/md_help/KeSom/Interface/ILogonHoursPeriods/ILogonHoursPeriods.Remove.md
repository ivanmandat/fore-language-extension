# ILogonHoursPeriods.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс периода.


## Описание


Метод Remove осуществляет удаление периода, в который осуществляется ограничение времени доступа пользователя. Индекс периода передается посредством параметра Index.
