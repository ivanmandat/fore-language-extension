# ILogonHoursPeriods.Item


## Синтаксис


Item(Index: Integer): [ILogonHoursPeriod](../ILogonHoursPeriod/ILogonHoursPeriod.htm);


## Параметры


Index - индекс периода.


## Описание


Свойство Item возвращает параметры периода, в который осуществляется ограничение времени доступа пользователя. Индекс периода передается посредством параметра Index.
