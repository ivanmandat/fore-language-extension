# IMetabaseClass


## Описание


Интерфейс IMetabaseClass содержит
 статические свойства и методы класса [MetabaseClass](../../Class/MetabaseClass/MetabaseClass.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Active](IMetabaseClass.Active.htm)
		 Свойство Active возвращает
		 данные текущего репозитория.


		 ![](../../Property_Image.gif)
		 [CommonClassName](IMetabaseClass.CommonClassName.htm)
		 Свойство CommonClassName
		 возвращает наименование указанного класса объекта репозитория
		 в заданном падеже.


		 ![](../../Property_Image.gif)
		 [IconIndex](IMetabaseClass.IconIndex.htm)
		 Свойство IconIndex
		 возвращает индекс пиктограммы, установленной для указанного класса
		 объекта.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [GetMetabaseObjectClass](IMetabaseClass.GetMetabaseObjectClass.htm)
		 Метод GetMetabaseObjectClass
		 возвращает класс объекта, реализуемый перечислимым типом [MetabaseObjectClass](../../Enums/MetabaseObjectClass.htm),
		 по числовому значению идентификатора класса.
