# IMetabaseUpdateUnresolved.KeepResolve


## Синтаксис


KeepResolve: Boolean;


## Описание


Свойство KeepResolve определяет,
 применять ли замену объекта репозитория-источника на объект [ResolveObject](IMetabaseUpdateUnresolved.ResolveObject.htm).


## Комментарии


Допустимые значения:


	- True. Заменять объект;


	- False. Не заменять объект.
