# IMetabaseUpdateUnresolved


## Описание


Интерфейс IMetabaseUpdateUnresolved содержит свойства для работы с зависимостью обновления от объекта репозитория-источника, который отсутствует в репозитории назначения.


## Свойства


 Имя свойства
 Краткое описание


 ![](../../Property_Image.gif)
 [ClassId](IMetabaseUpdateUnresolved.ClassId.htm)
 Свойство ClassId возвращает идентификатор класса зависимого объекта, который отсутствует в репозитории назначения.


 ![](../../Property_Image.gif)
 [Description](IMetabaseUpdateUnresolved.Description.htm)
 Свойство Description возвращает описание зависимого объекта, который отсутствует в репозитории назначения.


 ![](../../Property_Image.gif)
 [Id](IMetabaseUpdateUnresolved.Id.htm)
 Свойство Id возвращает идентификатор зависимого объекта, который отсутствует в репозитории назначения.


 ![](../../Property_Image.gif)
 [IsRequired](IMetabaseUpdateUnresolved.IsRequired.htm)
 Свойство IsRequired определяет, будет ли обработана ошибка (неразрешенная ссылка) при обновлении.


 ![](../../Property_Image.gif)
 [KeepResolve](IMetabaseUpdateUnresolved.KeepResolve.htm)
 Свойство KeepResolve определяет, применять ли замену объекта репозитория-источника на объект [ResolveObject](IMetabaseUpdateUnresolved.ResolveObject.htm).


 ![](../../Property_Image.gif)
 [Key](IMetabaseUpdateUnresolved.Key.htm)
 Свойство Key возвращает ключ зависимого объекта, который отсутствует в репозитории назначения.


 ![](../../Property_Image.gif)
 [Name](IMetabaseUpdateUnresolved.Name.htm)
 Свойство Name возвращает наименование зависимого объекта, который отсутствует в репозитории назначения.


 ![](../../Property_Image.gif)
 [ResolveObject](IMetabaseUpdateUnresolved.ResolveObject.htm)
 Свойство ResolveObject определяет объект в репозитории назначения, заменяющий объект репозитория-источника.
