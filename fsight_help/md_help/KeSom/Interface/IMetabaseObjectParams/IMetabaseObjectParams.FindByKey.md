# IMetabaseObjectParams.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IMetabaseObjectParam](../IMetabaseObjectParam/IMetabaseObjectParam.htm);


## Параметры


Key. Ключ параметра объекта репозитория.


## Описание


Метод FindByKey позволяет найти
 параметр по указанному ключу Key.


## Комментарии


В случае удачного поиска метод возвращает настройки параметра, иначе
 возвращает значение Null.
