# IMetabaseObjectParams.Move


## Синтаксис


Move(OldIndex: Integer, NewIndex: Integer);


## Параметры


OldIndex. Индекс позиции, из
 которой необходимо переместить параметр;


NewIndex. Индекс позиции, в
 которую необходимо переместить параметр.


## Описание


Метод Move осуществляет перемещение
 параметра объекта репозитория в указанную позицию.


## Коментарии


Если указанный индекс позиции, в которую необходимо переместить параметр,
 за пределами коллекции, то перемещение параметра объекта репозитория не
 будет выполнено.
