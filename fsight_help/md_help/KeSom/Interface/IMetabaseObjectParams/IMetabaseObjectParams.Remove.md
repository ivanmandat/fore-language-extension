# IMetabaseObjectParams.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс параметра объекта репозитория.


## Описание


Метод Remove осуществляет удаление
 параметра, индекс которого передается посредством входного параметра,
 и возвращает признак удачного удаления.
