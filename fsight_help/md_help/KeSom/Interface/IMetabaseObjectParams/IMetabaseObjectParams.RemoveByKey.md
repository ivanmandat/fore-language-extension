# IMetabaseObjectParams.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ параметра объекта репозитория.


## Описание


Метод RemoveByKey осуществляет
 удаление параметра, ключ которого передается посредством входного параметра.
