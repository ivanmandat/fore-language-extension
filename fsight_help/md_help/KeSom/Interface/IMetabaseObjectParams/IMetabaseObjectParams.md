# IMetabaseObjectParams


Сборка: Metabase;


## Описание


Интерфейс IMetabaseObjectParams
 содержит свойства и методы коллекции параметров объектов репозитория.


## Иерархия наследования

		 Краткое описание

		 [ControlInfo](IMetabaseObjectParams.ControlInfo.htm)
		 Свойство ControlInfo
		 возвращает информацию об управлении зависящими параметрами.


		 ![](../../Property_Image.gif)
		 [Count](IMetabaseObjectParams.Count.htm)
