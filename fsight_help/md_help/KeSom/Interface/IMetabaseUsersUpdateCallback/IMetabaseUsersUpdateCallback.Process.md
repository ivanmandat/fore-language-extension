# IMetabaseUsersUpdateCallback.Process


## Синтаксис


Process(Subject: [ISecuritySubject](../ISecuritySubject/ISecuritySubject.htm));


## Параметры


Subject. Субъект безопасности, при обновлении которого произошла ошибка.


## Описание


Метод Process возвращает пользователя, который обновляется в данный момент.


## Комментарии


Метод должен быть переопределен в пользовательском классе.
