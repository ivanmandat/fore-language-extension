# IUserProfile.Name


## Синтаксис


Name: String;


## Описание


Свойство Name возвращает наименование профиля.


## Комментарии


Если профиль соответствует пользователю репозитория, то в качестве наименования будет выступать [наименование пользователя](../ISecuritySubject/ISecuritySubject.Name.htm). Если профиль локальный, то свойство Name вернет то значение, которое было указано при выполнении метода [IUserProfiles.AddStr](../IUserProfiles/IUserProfiles.AddStr.htm).
