# IUserProfile.Edit


## Синтаксис


Edit: [IUserProfile](IUserProfile.htm);


## Описание


Метод Edit открывает профиль пользователя на редактирование.


## Комментарии


Метод возвращает экземпляр профиля, открытый на редактирование. Администратор репозитория может изменять профили любых пользователе. Отдельные пользователи могут изменять только свой профиль. Для сохранения изменений необходимо у экземпляра профиля вызвать метод [Apply](IUserProfile.Apply.htm).
