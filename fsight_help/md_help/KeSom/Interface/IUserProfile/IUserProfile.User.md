# IUserProfile.User


## Синтаксис


User: [IMetabaseUser](../IMetabaseUser/IMetabaseUser.htm);


## Описание


Свойство User возвращает параметры пользователя, которому соответствует данный профиль.


## Комментарии


Если текущий профиль является локальным и не имеет привязки к пользователю репозитория, то свойство вернет значение Null.
