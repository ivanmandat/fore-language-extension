# IUserProfile.Apply


## Синтаксис


Apply;


## Описание


Метод Apply применяет изменения, внесенные в профиль.


## Комментарии


Данный метод должен быть вызван после вызова метода [Edit](IUserProfile.Edit.htm) и внесения каких-либо изменений в профиль.

Пример использования приведен в описании метода [IUserProfiles.AddStr](../IUserProfiles/IUserProfiles.AddStr.htm).
