# IUserProfile


Сборка: Metabase;


## Описание


Интерфейс IUserProfile содержит
 свойства и методы для определения параметров профиля пользователя.


## Иерархия наследования


IUserProfile


## Комментарии


В профиле пользователя хранится информация об адресах электронной почты
 и дополнительные свойства пользователя.


## Свойства


		 Имя свойства
		 Краткое описание


		 [![](../../Property_Image.gif)](IUserProfile.Email.htm)
		 [Email](IUserProfile.Email.htm)
		 Свойство Email определяет
		 электронный почтовый адрес по умолчанию.


		 ![](../../Property_Image.gif)
		 [Emails](IUserProfile.Emails.htm)
		 Свойство Emails возвращает
		 коллекцию записей об адресах электронной почты.


		 [![](../../Property_Image.gif)](IUserProfile.Email.htm)
		 [Name](IUserProfile.Name.htm)
		 Свойство Name возвращает
		 наименование профиля.


		 ![](../../Property_Image.gif)
		 [PrivateFolder](IUserProfile.PrivateFolder.htm)
		 Свойство PrivateFolder
		 определяет личную папку пользователя.


		 ![](../../Property_Image.gif)
		 [Prop](IUserProfile.Prop.htm)
		 Свойство Prop определяет
		 значение дополнительного свойства пользователя.


		 [![](../../Property_Image.gif)](IUserProfile.Email.htm)
		 [Properties](IUserProfile.Properties.htm)
		 Свойство Properties
		 возвращает коллекцию с дополнительными параметрами пользователя.


		 [![](../../Property_Image.gif)](IUserProfile.Email.htm)
		 [StringSid](IUserProfile.StringSid.htm)
		 Свойство StringSid
		 возвращает символьное значение идентификатора безопасности.


		 ![](../../Property_Image.gif)
		 [User](IUserProfile.User.htm)
		 Свойство User возвращает
		 параметры пользователя, которому соответствует данный профиль.
