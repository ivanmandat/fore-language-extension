# IMetabaseObject.GetModelDispatch


## Синтаксис


		GetModelDispatch: Object;


## Описание


Метод GetModelDispatch осуществляет
 преобразование структуры объекта к общей форме и возвращает полученную
 структуру.


## Комментарии


Данный метод предназначен для применения в языках программирования,
 в которых отсутствует возможность явного преобразования типов. Результатом
 работы метода будет необходимый тип объекта.


Примечание.
 Использование данного метода в языке Fore не актуально. Подробнее о преобразовании
 типов смотрите в описании оператора [As](Fore.chm::/07_Operations/Operations_of_the_creation_object.htm#as).
