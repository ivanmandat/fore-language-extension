# IMetabaseObject.IsDirty


## Синтаксис


		IsDirty: Boolean;


## Описание


Свойство IsDirty возвращает
 признак наличия изменений в структуре объекта репозитория в рамках текущего
 соединения с сервером БД.


## Комментарии


Свойство возвращает True, если
 присутствуют изменения, иначе False.
