# IMetabaseUpdateSecurityCategory


Сборка: Metabase;


## Описание


Интерфейс IMetabaseUpdateSecurityCategory
 содержит свойства и методы категории безопасности мандатного доступа для
 обновления.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Exists](IMetabaseUpdateSecurityCategory.Exists.htm)
		 Свойство Exists возвращает
		 признак существования категории безопасности мандатного доступа
		 из обновления в базе, на которую оно устанавливается.


		 ![](../../Property_Image.gif)
		 [IsValid](IMetabaseUpdateSecurityCategory.IsValid.htm)
		 Свойство IsValid возвращает
		 признак доступности категории безопасности.


		 ![](../../Property_Image.gif)
		 [Key](IMetabaseUpdateSecurityCategory.Key.htm)
		 Свойство Key возвращает
		 ключ категории безопасности мандатного доступа.


		 ![](../../Property_Image.gif)
		 [Level](IMetabaseUpdateSecurityCategory.Level.htm)
		 Свойство Level
		 возвращает уровень безопасности по индексу.


		 ![](../../Property_Image.gif)
		 [LevelCount](IMetabaseUpdateSecurityCategory.LevelCount.htm)
		 Свойство LevelCount
		 возвращает количество уровней безопасности.


		 ![](../../Property_Image.gif)
		 [Name](IMetabaseUpdateSecurityCategory.Name.htm)
		 Свойство Name
		 возвращает наименование категории безопасности.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindLevelByLabel](IMetabaseUpdateSecurityCategory.FindLevelByLabel.htm)
		 Метод FindLevelByLabel
		 возвращает уровень безопасности по метке критичности.
