# ISpecialObjects


## Описание


Интерфейс ISpecialObjects содержит
 свойства для работы с коллекцией [специальных
 объектов репозитория](UpdManager.chm::/CreateUpdate/admin_createupdate_specialobjects.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [SpecialObject](ISpecialObjects.SpecialObject.htm)
		 Свойство SpecialObject
		 определяет объект, являющийся специальным объектом репозитория.


		 ![](../../Property_Image.gif)
		 [VcsParams](ISpecialObjects.VcsParams.htm)
		 Свойство VcsParams
		 возвращает настройки подключения репозитория к системе управления
		 версиями.


## Методы


		 Имя свойства
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Operation](ISpecialObjects.Operation.htm)
		 Метод Operation осуществляет
		 выполнение указанной операции со специальным объектом репозитория.


		 ![](../../Sub_Image.gif)
		 [SaveMetabaseObjectHandlers](ISpecialObjects.SaveMetabaseObjectHandlers.htm)
		 Метод SaveMetabaseObjectHandlers
		 возвращает коллекцию обработчиков событий, наступающих при изменении
		 метаданных объектов.
