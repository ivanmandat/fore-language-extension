# ISpecialObjects.SaveMetabaseObjectHandlers


## Синтаксис


SaveMetabaseObjectHandlers: [ISaveMetabaseObjectHandlers](../ISaveMetabaseObjectHandlers/ISaveMetabaseObjectHandlers.htm);


## Описание


Метод SaveMetabaseObjectHandlers
 возвращает коллекцию обработчиков событий, наступающих при изменении метаданных
 объектов.


## Комментарии


События наступают непосредственно в момент сохранения изменений в метаданных
 объекта, позволяют получить объект до и после изменения. Более подробно
 сигнатура обработчиков событий представлена в описании свойства [ISaveMetabaseObjectHandler.Class_](../ISaveMetabaseObjectHandler/ISaveMetabaseObjectHandler.Class_.htm).
