# IPrivilege.Grant


## Синтаксис


		Grant(Subject: [ISecuritySubject](../ISecuritySubject/ISecuritySubject.htm));


## Параметры


Subject. Субъект безопасности, для которого необходимо
 раздать привилегию.


## Описание


Метод Grant дает привилегию
 субъекту безопасности, передаваемому посредством параметра Subject.

Использование метода приведено в примере для [IMetabaseSecurity.AddGroup](../IMetabaseSecurity/IMetabaseSecurity.AddGroup.htm).
