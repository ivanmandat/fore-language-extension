# IPrivilege


## Описание


Интерфейс IPrivilege содержит
 свойства и методы привилегии менеджера безопасности.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Description](IPrivilege.Description.htm)
		 Свойство Description
		 возвращает описание привилегии.


		 ![](../../Property_Image.gif)
		 [Holder](IPrivilege.Holder.htm)
		 Свойство Holder возвращает
		 параметры субъекта безопасности, обладающего данной привилегией.


		 ![](../../Property_Image.gif)
		 [HolderCount](IPrivilege.HolderCount.htm)
		 Свойство HolderCount
		 возвращает количество субъектов безопасности, обладающих данной
		 привилегией.


		 ![](../../Property_Image.gif)
		 [Name](IPrivilege.Name.htm)
		 Свойство Name возвращает
		 наименование привилегии.


		 ![](../../Property_Image.gif)
		 [Predefined](IPrivilege.Predefined.htm)
		 Свойство Predefined
		 возвращает системную привилегию, используя значения перечисления
		 [MetabasePolicyPredefinedPrivilege](../../Enums/MetabasePolicyPredefinedPrivilege.htm).


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Grant](IPrivilege.Grant.htm)
		 Метод Grant назначает
		 привилегию указанному субъекту безопасности.


		 ![](../../Sub_Image.gif)
		 [Revoke](IPrivilege.Revoke.htm)
		 Метод Revoke изымает
		 привилегию у указанного субъекта безопасности.


		 ![](../../Sub_Image.gif)
		 [SubjectGranted](IPrivilege.SubjectGranted.htm)
		 Метод SubjectGranted
		 возвращает признак наличия данной привилегии у указанного субъекта
		 безопасности.
