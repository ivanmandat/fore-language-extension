# IMetabaseObjectParam.DefaultValue


## Синтаксис


DefaultValue: Variant;


## Описание


Свойство DefaultValue определяет
 значение для параметра по умолчанию.


## Комментарии


Данное значение будет подставляться в редактор, и использоваться при
 открытии объекта из навигатора объектов. Если для всех параметров указано
 значение по умолчанию, то при открытии объекта из навигатора объектов
 диалог ввода значений параметров не отображается.
