# IMetabaseObjectParam


## Описание


Интерфейс IMetabaseObjectParam
 содержит свойства параметра объекта репозитория.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Binding](IMetabaseObjectParam.Binding.htm)
		 Свойство Binding возвращает
		 настройки отображения редактора значения параметра объекта.


		 ![](../../Property_Image.gif)
		 [DataType](IMetabaseObjectParam.DataType.htm)
		 Свойство DataType определяет
		 тип данных, которые может принимать параметр.


		 ![](../../Property_Image.gif)
		 [DefaultValue](IMetabaseObjectParam.DefaultValue.htm)
		 Свойство DefaultValue
		 определяет значение для параметра по умолчанию.


		 ![](../../Property_Image.gif)
		 [Object](IMetabaseObjectParam.Object.htm)
		 Свойство Object возвращает
		 описание объекта репозитория, которому принадлежит используемый
		 параметр.


		 ![](../../Property_Image.gif)
		 [Params](IMetabaseObjectParam.Params.htm)
		 Свойство Params возвращает
		 коллекцию параметров, которой принадлежит данный параметр.


		 ![](../../Property_Image.gif)
		 [SharedParamLink](IMetabaseObjectParam.SharedParamLink.htm)
		 Свойство SharedParamLink
		 возвращает настройки связи параметра с глобальной переменной.


## Свойства, унаследованные от [INamedEntity](../INamedEntity/INamedEntity.htm)
