# IMetabaseObjectParamLink


Сборка: Metabase;


## Описание


Интерфейс IMetabaseObjectParamLink
 предназначен для настройки связи параметра с глобальной переменной.


## Иерархия наследования


IMetabaseObjectParamLink


## Комментарии


Настройки связи параметра можно получить, используя свойство [IMetabaseObjectParam.SharedParamLink](../IMetabaseObjectParam/IMetabaseObjectParam.SharedParamLink.htm).

		 Свойство ReadWriteMode
