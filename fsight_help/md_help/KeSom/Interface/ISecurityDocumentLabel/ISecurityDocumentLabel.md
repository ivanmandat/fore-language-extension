# ISecurityDocumentLabel


## Описание


Интерфейс ISecurityDocumentLabel
 предназначен для работы с меткой безопасности, которой могут маркироваться
 объекты во время печати и экспорта при включенном мандатном доступе.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [IsON](ISecurityDocumentLabel.IsON.htm)
		 Свойство IsON возвращает
		 признак включенной маркировки объектов.


		 ![](../../Property_Image.gif)
		 [Label](ISecurityDocumentLabel.Label.htm)
		 Свойство Label возвращает
		 наименование метки.


		 ![](../../Property_Image.gif)
		 [LabelDescription](ISecurityDocumentLabel.LabelDescription.htm)
		 Свойство LabelDescription
		 возвращает описание, которое будет использовано при маркировке
		 объектов при экспорте и печати.


		 ![](../../Property_Image.gif)
		 [Metabase](ISecurityDocumentLabel.Metabase.htm)
		 Свойство Metabase возвращает
		 параметры репозитория, из которого осуществляется печать или экспорт
		 объекта.


		 ![](../../Property_Image.gif)
		 [TreatSectionAsDocument](ISecurityDocumentLabel.TreatSectionAsDocument.htm)
		 Свойство TreatSectionAsDocument
		 возвращает признак печати отдельных секций документа как отдельных
		 документов.


		 ![](../../Property_Image.gif)
		 [User](ISecurityDocumentLabel.User.htm)
		 Свойство User возвращает
		 параметры пользователя, осуществляющего печать либо экспорт объекта.
