# IMbElementDependency


Сборка: Metabase;


## Описание


Интерфейс IMbElementDependency содержит свойства для работы с записью о зависимости объекта от элементов справочника НСИ.


## Иерархия наследования


           [IMbElementDependencyTemplate](../IMbElementDependencyTemplate/IMbElementDependencyTemplate.htm)


           IMbElementDependency


## Свойства


 Имя свойства
 Краткое описание


 ![](../../Property_Image.gif)
 [Dictionary](IMbElementDependency.Dictionary.htm)
 Свойство Dictionary определяет справочник НСИ, в котором содержатся элементы, от которых зависит объект [Object](IMbElementDependency.Object.htm).


 ![](../../Property_Image.gif)
 [Object](IMbElementDependency.Object.htm)
 Свойство Object определяет объект репозитория, который зависит от элементов справочника НСИ, указанного в свойстве [Dictionary](IMbElementDependency.Dictionary.htm).
