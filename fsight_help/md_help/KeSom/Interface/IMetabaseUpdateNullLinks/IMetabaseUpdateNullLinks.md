# IMetabaseUpdateNullLinks


## Описание


Интерфейс IMetabaseUpdateNullLinks
 содержит свойства для работы с коллекцией ссылок, которые отсутствуют
 в репозитории назначения.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMetabaseUpdateNullLinks.Count.htm)
		 Свойство Count возвращает
		 количество ссылок в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMetabaseUpdateNullLinks.Item.htm)
		 Свойство Item возвращает
		 ссылку из коллекции по ее индексу.
