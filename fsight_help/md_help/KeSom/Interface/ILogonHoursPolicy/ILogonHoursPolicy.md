# ILogonHoursPolicy


## Описание


Интерфейс ILogonHoursPolicy
 содержит свойства и методы для управления политикой доступа пользователя
 к платформе по времени.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Periods](ILogonHoursPolicy.Periods.htm)
		 Свойство Periods определяет
		 параметры доступа пользователя в определенные интервалы дат.


		 ![](../../Property_Image.gif)
		 [WeekDayHours](ILogonHoursPolicy.WeekDayHours.htm)
		 Свойство WeekDayHours
		 определяет параметры доступа пользователя по дням недели.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindByDate](ILogonHoursPolicy.FindByDate.htm)
		 Метод FindByDate осуществляет
		 поиск периода, содержащего указанную дату.
