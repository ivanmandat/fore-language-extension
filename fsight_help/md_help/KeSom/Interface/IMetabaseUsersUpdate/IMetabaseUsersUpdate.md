# IMetabaseUsersUpdate


Сборка: Metabase;


## Описание


Интерфейс IMetabaseUsersUpdate
 содержит свойства и методы для работы с коллекцией пользователей, о которых
 необходимо обновить информацию.


## Иерархия наследования


           [IMetabaseUsers](../IMetabaseUsers/IMetabaseUsers.htm)


           IMetabaseUsersUpdate
