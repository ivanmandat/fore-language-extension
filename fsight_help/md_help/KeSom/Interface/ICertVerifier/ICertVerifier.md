# ICertVerifier


Сборка: Metabase;


## Описание


Интерфейс ICertVerifier содержит
 свойства и методы контролера цифровой подписи.


## Иерархия наследования


ICertVerifier


## Комментарии


Контролер цифровой подписи можно получить, используя метод [ICertProvider.OpenVerifier](../ICertProvider/ICertProvider.OpenVerifier.htm)
 или [ICertProvider.OpenVerifierBase64](../ICertProvider/ICertProvider.OpenVerifierBase64.htm).
