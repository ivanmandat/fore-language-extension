# IMetabaseSecurity.AddNTSubject


## Синтаксис


AddNTSubject(Subject: [ISecuritySubject](../ISecuritySubject/ISecuritySubject.htm)): [ISecuritySubject](../ISecuritySubject/ISecuritySubject.htm);


## Параметры


Subject. Доменный субъект безопасности.


## Описание


Метод AddNTSubject осуществляет добавление доменного субъекта безопасности в репозиторий.
