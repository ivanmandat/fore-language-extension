# IMetabaseSecurity.UpdateUsers


## Синтаксис


UpdateUsers(Value: [IMetabaseUsersUpdate](../IMetabaseUsersUpdate/IMetabaseUsersUpdate.htm));


## Параметры


Value.
 Список пользователей, для которых необходимо обновить информацию.


## Описание


Метод UpdateUsers производит
 обновление информации о пользователях, список которых передается посредством
 параметра Value.
