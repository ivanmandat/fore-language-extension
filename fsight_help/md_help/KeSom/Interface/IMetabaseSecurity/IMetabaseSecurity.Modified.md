# IMetabaseSecurity.Modified


## Синтаксис


Modified: Boolean;


## Описание


Свойство Modified возвращает признак наличия несохраненных изменений в настройках менеджера безопасности.


## Комментарии


Метод возвращает значение True, если были изменены какие-либо параметры менеджера безопасности и не был вызван метод [Apply](IMetabaseSecurity.Apply.htm) либо [Revert](IMetabaseSecurity.Revert.htm).
