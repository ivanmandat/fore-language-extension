# IMetabaseCustomClasses


## Описание


Интерфейс IMetabaseCustomClasses
 содержит свойства и методы для работы с коллекцией пользовательских классов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMetabaseCustomClasses.Count.htm)
		 Свойство Count возвращает
		 количество классов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMetabaseCustomClasses.Item.htm)
		 Свойство Item возвращает
		 класс из коллекции по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IMetabaseCustomClasses.Add.htm)
		 Метод Add осуществляет
		 добавление класса в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](IMetabaseCustomClasses.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех классов из коллекции.


		 ![](../../Sub_Image.gif)
		 [FindByClassId](IMetabaseCustomClasses.FindByClassId.htm)
		 Метод FindByClassId
		 выполняет поиск класса по числовому идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindById](IMetabaseCustomClasses.FindById.htm)
		 Метод FindById осуществляет
		 поиск класса в коллекции по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IMetabaseCustomClasses.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск класса в коллекции по ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](IMetabaseCustomClasses.Remove.htm)
		 Метод Remove осуществляет
		 удаление класса из коллекции по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IMetabaseCustomClasses.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление класса из коллекции по ключу.
