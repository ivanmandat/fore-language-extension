# IMetabaseCustomClasses.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс класса в коллекции.


## Описание


Метод Remove осуществляет удаление класса из коллекции по индексу. Метод возвращает значение True, если удаление прошло успешно.
