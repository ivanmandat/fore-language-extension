# IMetabaseCustomClasses.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key - ключ класса.


## Описание


Метод RemoveByKey осуществляет удаление класса из коллекции по ключу. Метод возвращает значение True, если удаление прошло успешно.
