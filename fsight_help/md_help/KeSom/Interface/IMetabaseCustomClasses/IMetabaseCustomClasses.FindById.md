# IMetabaseCustomClasses.FindById


## Синтаксис


FindById(Id: String): [IMetabaseCustomClass](../IMetabaseCustomClass/IMetabaseCustomClass.htm);


## Параметры


Id - идентификатор класса.


## Описание


Метод FindById осуществляет поиск класса в коллекции по идентификатору.


## Комментарии


В случае удачного поиска метод возвращает параметры пользовательского класса, иначе возвращает значение Null.
