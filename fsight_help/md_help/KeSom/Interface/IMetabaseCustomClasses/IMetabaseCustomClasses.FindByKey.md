# IMetabaseCustomClasses.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IMetabaseCustomClass](../IMetabaseCustomClass/IMetabaseCustomClass.htm);


## Параметры


Key - ключ класса.


## Описание


Метод FindByKey осуществляет поиск класса в коллекции по ключу.


## Комментарии


В случае удачного поиска метод возвращает параметры пользовательского класса, иначе возвращает значение Null.
