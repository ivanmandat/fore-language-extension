# IMetabaseDeviceDescriptor.SecurityDescriptor


## Синтаксис


SecurityDescriptor: [ISecurityDescriptor](../ISecurityDescriptor/ISecurityDescriptor.htm);


## Описание


Свойство SecurityDescriptor возвращает параметры управления доступом к текущему устройству.


## Комментарии


Свойство используется, если для пользователей установлен режим ограничения на используемые для печати принтеры.
