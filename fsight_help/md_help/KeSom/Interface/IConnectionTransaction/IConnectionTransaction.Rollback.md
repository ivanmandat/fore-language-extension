# IConnectionTransaction.Rollback


## Синтаксис


Rollback;


## Описание


Метод Rollback отменяет все
 изменения и завершает транзакцию.


## Комментарии


При отмене изменений внешней транзакции результаты всех операций будут
 отменены.
