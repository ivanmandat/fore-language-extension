# IUserEmails.Personal


## Синтаксис


Personal: String;


## Описание


Свойство Personal устанавливает личный электронный почтовый адрес.


## Комментарии


Мобильный электронный почтовый адрес устанавливается в окне «Личные данные». Данное окно открывается при выполнении команды главного меню навигатора объектов «Сервис > Личные данные».
