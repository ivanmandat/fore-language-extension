# IUserEmails


Сборка: Metabase;


## Описание


Интерфейс IUserEmails содержит
 свойства и методы для работы с коллекцией записей об электронных почтовых
 адресах.


## Иерархия наследования


IUserEmails


## Комментарии


Электронные адреса хранятся в профиле пользователей. Доступ к коллекции
 адресов осуществляется посредством свойства [IUserProfile.Emails](../IUserProfile/IUserProfile.Emails.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IUserEmails.Count.htm)
		 Свойство Count возвращает
		 количество записей в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IUserEmails.Item.htm)
		 Свойство Item возвращает
		 параметры указанного электронного почтового адреса.


		 ![](../../Property_Image.gif)
		 [Mobile](IUserEmails.Mobile.htm)
		 Свойство Mobile устанавливает
		 мобильный электронный почтовый адрес.


		 ![](../../Property_Image.gif)
		 [Personal](IUserEmails.Personal.htm)
		 Свойство Personal устанавливает
		 личный электронный почтовый адрес.


		 ![](../../Property_Image.gif)
		 [Working](IUserEmails.Working.htm)
		 Свойство Working устанавливает
		 рабочий электронный почтовый адрес.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IUserEmails.Add.htm)
		 Метод Add добавляет
		 в коллекцию запись о пользовательской электронной почте и возвращает
		 ее параметры.


		 ![](../../Sub_Image.gif)
		 [Remove](IUserEmails.Remove.htm)
		 Метод Remove удаляет
		 запись об электронной почте из коллекции.
