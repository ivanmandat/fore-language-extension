# IUserEmails.Remove


## Синтаксис


Remove(Index: Integer); Boolean;


## Параметры


Index. Индекс электронного почтового адреса.


## Описание


Метод Remove удаляет запись об электронной почте из коллекции.


Примечание. Удаление фиксированных адресов с индексами 0-2 запрещено.
