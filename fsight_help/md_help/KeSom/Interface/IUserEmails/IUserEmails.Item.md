# IUserEmails.Item


## Синтаксис


Item(Index: Integer): [IUserEmail](../IUserEmail/IUserEmail.htm);


## Параметры


Index. Индекс электронного почтового адреса.


## Описание


Свойство Item возвращает параметры указанной записи.


## Комментарии


В коллекции IUserEmails хранятся три фиксированных электронных адреса. Значения этих адресов также можно узнать в свойствах [Mobile](IUserEmails.Mobile.htm), [Personal](IUserEmails.Personal.htm) и [Working](IUserEmails.Working.htm).


Для доступа к записям о фиксированных электронных адресах с помощью свойства Item необходимо указать следующие значения параметра Index:


-
0. Рабочий адрес.


-
1. Мобильный адрес.


-
2. Личный адрес.
