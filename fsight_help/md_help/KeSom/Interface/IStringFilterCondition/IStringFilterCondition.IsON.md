# IStringFilterCondition.IsON


## Синтаксис


IsON: Boolean;


## Описание


Свойство IsON определяет признак
 использования фильтрации протокола доступа по конкретному объекту.


## Комментарии


При значении True фильтрация
 по конкретному объекту используется, при значении False
 - нет. При включении данного условия протокол доступа будет содержать
 записи только для объектов, которые найдены по указанному наименованию
 и/или идентификатору.
