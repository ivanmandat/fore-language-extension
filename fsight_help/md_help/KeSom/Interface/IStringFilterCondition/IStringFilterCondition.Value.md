# IStringFilterCondition.Value


## Синтаксис


Value: String;


## Описание


Свойство Value определяет текстовую
 строку для поиска в наименованиях/идентификаторах объектов.


## Комментарии


При применении данного фильтра протокол будет содержать записи для объектов,
 наименование/идентификатор которых содержит заданный текст.
