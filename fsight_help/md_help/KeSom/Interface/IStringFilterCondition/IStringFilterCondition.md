# IStringFilterCondition


## Описание


Интерфейс IStringFilterCondition
 предназначен для работы с условием фильтрации протокола доступа по конкретному
 объекту.


## Комментарии


При включении данного условия протокол доступа будет содержать записи
 только для объектов, которые найдены по указанному наименованию и/или
 идентификатору.
