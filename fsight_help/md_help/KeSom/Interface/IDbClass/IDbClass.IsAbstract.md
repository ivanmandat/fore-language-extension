# IDbClass.IsAbstract


## Синтаксис


IsAbstract: Boolean;


## Описание


Свойство IsAbstract возвращает признак класса, не поддерживающего создание.


## Комментарии


При попытке создать объект класса, не поддерживающего создание (например: KE_CLASS_SHORTCUT_AUDIT), будет сгенерирована исключительная ситуация «Класс объекта не найден».
