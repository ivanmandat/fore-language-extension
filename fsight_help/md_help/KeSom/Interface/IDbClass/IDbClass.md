# IDbClass


## Описание


Интерфейс IDbClass содержит
 свойства класса объекта, доступного в платформе.


## Комментарии


Класс объекта можно получить, используя свойства и методы интерфейса
 [IDbClasses](../IDbClasses/IDbClasses.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [IsAbstract](IDbClass.IsAbstract.htm)
		 Свойство IsAbstract
		 возвращает признак класса, не поддерживающего создание.


		 ![](../../Property_Image.gif)
		 [IsObsolete](IDbClass.IsObsolete.htm)
		 Свойство IsObsolete
		 возвращает признак того, является ли класс устаревшим.


		 ![](../../Property_Image.gif)
		 [SupportsElementDependencies](IdbClass.SupportsElementDependencies.htm)
		 Свойство SupportsElementDependencies
		 возвращает признак того, отслеживает ли класс изменения
		 в справочниках НСИ.


		 ![](../../Property_Image.gif)
		 [SupportsElementDependents](IdbClass.SupportsElementDependents.htm)
		 Свойство SupportsElementDependents
		 возвращает признак того, отслеживает ли класс изменения
		 его элементов.
