# IMetabaseUsersUpdateError.Message


## Синтаксис


Message: String;


## Описание


Свойство Message возвращает сообщение об ошибке при обновлении пользователя.


## Комментарии


Сообщение содержит информацию о пользователе, при обновлении которого произошла ошибка, причине возникновении ошибки и моменте ее возникновения, например «Ошибка при создании пользователя 'ADMIN_BASE' на сервере БД».
