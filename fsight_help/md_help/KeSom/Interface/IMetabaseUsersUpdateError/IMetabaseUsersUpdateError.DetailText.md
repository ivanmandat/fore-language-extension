# IMetabaseUsersUpdateError.DetailText


## Синтаксис


DetailText: String;


## Описание


Свойство DetailText возвращает подробную информацию об ошибке при обновлении пользователя.


## Комментарии


Информация содержит класс ошибки и источник, например «Класс ошибки: CPiObjException. Источник: Som\SomPackStd.cpp, строка 6520».
