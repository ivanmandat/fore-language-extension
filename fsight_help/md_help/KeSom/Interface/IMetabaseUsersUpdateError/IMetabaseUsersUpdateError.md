# IMetabaseUsersUpdateError


## Описание


Интерфейс IMetabaseUsersUpdateError
 содержит свойства ошибки при обновлении пользователя.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DetailText](IMetabaseUsersUpdateError.DetailText.htm)
		 Свойство DetailText
		 возвращает подробную информацию об ошибке при обновлении
		 пользователя.


		 ![](../../Property_Image.gif)
		 [Message](IMetabaseUsersUpdateError.Message.htm)
		 Свойство Message возвращает
		 сообщение об ошибке при обновлении пользователя.


		 ![](../../Property_Image.gif)
		 [SecuritySubject](IMetabaseUsersUpdateError.SecuritySubject.htm)
		 Свойство SecuritySubject
		 возвращает субъект безопасности, в процессе обновления которого
		 произошла ошибка.


		 ![](../../Property_Image.gif)
		 [Type](IMetabaseUsersUpdateError.Type.htm)
		 Свойство Type возвращает
		 тип ошибки при обновлении пользователя.
