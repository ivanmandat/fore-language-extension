# IConnectedUserInfoEx


Сборка: Metabase;


## Описание


Интерфейс IConnectedUserInfoEx
 содержит свойства активных пользователей, подключённых к репозиторию.


## Иерархия наследования


           IConnectedUserInfoEx


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [HostName](IConnectedUserInfoEx.HostName.htm)
		 Свойство HostName возвращает
		 наименование рабочей станции, с которой осуществлено подключение
		 к репозиторию.


		 ![](../../Property_Image.gif)
		 [NetAddress](IConnectedUserInfoEx.NetAddress.htm)
		 Свойство NetAddress
		 возвращает сетевой МАС-адрес компьютера, с которого осуществлено
		 подключение к репозиторию.


		 ![](../../Property_Image.gif)
		 [SessionId](IConnectedUserInfoEx.SessionId.htm)
		 Свойство SessionId
		 возвращает идентификатор сессии пользователя, подключённого к
		 репозиторию.


		 ![](../../Property_Image.gif)
		 [UserName](IConnectedUserInfoEx.UserName.htm)
		 Свойство UserName возвращает
		 наименование пользователя платформы, подключённого к репозиторию.


		 ![](../../Property_Image.gif)
		 [UserOsName](IConnectedUserInfoEx.UserOsName.htm)
		 Свойство UserOsName
		 возвращает наименование пользователя операционной системы, подключённого
		 к репозиторию.
