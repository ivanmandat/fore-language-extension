# IAssembly.Permanent


## Синтаксис


Permanent: Boolean;


## Описание


Свойство Permanent возвращает
 признак постоянной сборки.


## Комментарии


Для сборок репозитория данное свойство возвращает значение True.


Модули и формы, не включенные в сборку, реализуют временные сборки.
 Для них свойство [Assembly](../IModule/IModule.Assembly.htm)
 вернет параметры временной сборки, а данное свойство вернет значение False.
