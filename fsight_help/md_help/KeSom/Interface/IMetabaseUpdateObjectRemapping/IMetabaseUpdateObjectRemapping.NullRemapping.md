# IMetabaseUpdateObjectRemapping.NullRemapping


## Синтаксис


NullRemapping: Boolean;


## Описание


Свойство NullRemapping
 возвращает признак того, что для объекта используется нулевое повторное
 сопоставление.


## Комментарии


Допустимые значения:


	- True. Для объекта используется
	 нулевое повторное сопоставление;


	- False. Для объекта используется
	 обычное повторное сопоставление.
