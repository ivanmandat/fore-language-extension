# IMetabaseUpdateObjectRemapping.Map


## Синтаксис


Map(Type: [MetabaseUpdateRemappingType](../../Enums/MetabaseUpdateRemappingType.htm)):
 [IMetabaseUpdateRemappings](../IMetabaseUpdateRemappings/IMetabaseUpdateRemappings.htm);


## Параметры


Type. Тип повторного сопоставления.


## Описание


Свойство Map возвращает
 коллекцию повторных сопоставлений указанного типа.


## Комментарии


Если повторные сопоставления указанного типа отсутствуют, то свойство
 [IMetabaseUpdateRemappings.Count](../IMetabaseUpdateRemappings/IMetabaseUpdateRemappings.Count.htm)
 будет иметь нулевое значение.
