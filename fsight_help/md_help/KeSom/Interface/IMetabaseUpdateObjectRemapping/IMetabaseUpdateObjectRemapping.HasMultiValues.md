# IMetabaseUpdateObjectRemapping.HasMultiValues


## Синтаксис


HasMultiValues: Boolean;


## Описание


Свойство HasMultiValues
 возвращает признак того, что объект участвует в нескольких повторных сопоставлениях.


## Комментарии


Допустимые значения:


	- True. Объект участвует
	 в нескольких повторных сопоставлениях;


	- False. Для объекта задано
	 одно повторное сопоставление.
