# ISecurityConnection.UserNameDb


## Синтаксис


UserNameDb: String;


## Описание


Свойство UserNameDb возвращает
 наименование пользователя, под которым осуществлено подключение, в соответствии
 с параметрами пользователя в базе данных.


## Комментарии


Данное наименование берется из менеджера безопасности платформы.
