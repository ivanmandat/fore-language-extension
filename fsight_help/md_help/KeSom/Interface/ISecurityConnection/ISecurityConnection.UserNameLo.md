# ISecurityConnection.UserNameLo


## Синтаксис


UserNameLo: String;


## Описание


Свойство UserNameLo возвращает
 наименование пользователя, под которым осуществлено подключение, в соответствии
 с параметрами пользователя в менеджере безопасности.


## Комментарии


Данное наименование берется из менеджера безопасности платформы.
