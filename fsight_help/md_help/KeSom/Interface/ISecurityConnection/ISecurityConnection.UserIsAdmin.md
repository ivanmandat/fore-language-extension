# ISecurityConnection.UserIsAdmin


## Синтаксис


UserIsAdmin: Boolean;


## Описание


Свойство UserIsAdmin возвращает признак создания текущего соединения с использованием учетных данных администратора.


## Комментарии


Свойство возвращает значение True, если пользователь под которым создано соединение, обладает правами администратора.
