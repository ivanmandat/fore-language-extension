# IMbElementDependencies.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс записи о зависимости объекта, которую необходимо удалить.


## Описание


Метод Remove осуществляет удаление записи о зависимости объекта из коллекции.
