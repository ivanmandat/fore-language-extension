# IMbElementDependencies.Modified


## Синтаксис


Modified: Boolean;


## Описание


Свойство Modified возвращает признак наличия изменений в коллекции.


## Комментарии


По умолчанию свойство имеет значение False. Если в коллекцию добавлены новые зависимости, либо удалены существующие, то свойство вернет значение True. При наличии каких-либо изменений в коллекции, для сохранения всех изменений в репозитории необходимо вызвать метод [IMbElementDependenciesDatabase.Update](../IMbElementDependenciesDatabase/IMbElementDependenciesDatabase.Update.htm).
