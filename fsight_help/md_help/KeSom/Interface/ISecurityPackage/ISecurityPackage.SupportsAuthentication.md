# ISecurityPackage.SupportsAuthentication


## Синтаксис


SupportsAuthentication(Authentication: [AuthenticationMode](../../Enums/AuthenticationMode.htm)):
 Boolean;


## Параметры


Authentication. Тип
 аутентификации.


## Описание


Метод SupportsAuthentication
 возвращает признак поддержки текущим модулем безопасности указанного вида
 аутентификации.


## Комментарии


Метод возвращает значение True,
 если модуль безопасности поддерживает указанный вид аутентификации, и
 False, если не поддерживает.
