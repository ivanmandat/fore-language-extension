# IMetabaseObjectDescriptors.Item


## Синтаксис


		Item(Index: Integer): [IMetabaseObjectDescriptor](../IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm);


## Параметры


Index. Индекс описания объекта базы метаданных.


## Описание


Свойство Item возвращает описание
 объекта репозитория, индекс которого передается в качестве входного параметра
 Index.


## Пример


Пример использования приведён в описании свойства [IMetabaseObjectFindInfo.CaseSensitive](../IMetabaseObjectFindInfo/IMetabaseObjectFindInfo.CaseSensitive.htm).
