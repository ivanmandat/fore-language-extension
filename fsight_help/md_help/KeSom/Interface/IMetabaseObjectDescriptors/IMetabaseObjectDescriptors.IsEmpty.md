# IMetabaseObjectDescriptors.IsEmpty


## Синтаксис


		IsEmpty: Boolean;


## Описание


Свойство IsEmpty возвращает
 признак того, является ли описание объекта пустым.


## Комментарии


Если описание объекта не является пустым, свойство возвращает значение
 False, иначе - True.
