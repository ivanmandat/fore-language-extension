# ISecurityConnection2


Сборка: Metabase;


## Описание


Интерфейс ISecurityConnection2
 предназначен для работы со специализированным соединением с сервером БД.


## Комментарии


Интерфейс ISecurityConnection2
 реализован для получения возможности создания нового подключения к БД
 на базе существующего с возможностью указания типа нового подключения.
