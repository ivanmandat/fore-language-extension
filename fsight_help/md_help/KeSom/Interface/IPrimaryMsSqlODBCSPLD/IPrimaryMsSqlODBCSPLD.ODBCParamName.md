# IPrimaryMsSqlODBCSPLD.ODBCParamName


## Синтаксис


ODBCParamName(Index: Integer): String;


## Параметры


Index. Индекс параметра.


## Описание


Свойство ODBCParamName возвращает
 наименование параметра с указанным индексом.


## Комментарии


Список параметров формируется при задании для них значений с помощью
 свойства [ODBCParams](../IPrimaryMsSqlSPLD/IPrimaryMsSqlSPLD.ODBCParams.htm)
 или [ODBCParamValue](IPrimaryMsSqlODBCSPLD.ODBCParamValue.htm).
