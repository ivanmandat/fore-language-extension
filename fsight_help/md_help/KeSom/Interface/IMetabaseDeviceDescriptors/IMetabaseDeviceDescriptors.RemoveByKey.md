# IMetabaseDeviceDescriptors.RemoveByKey


## Синтаксис


RemoveByKey (Key:Integer);


## Параметры


Key. Ключ описания устройства.


## Описание


Метод RemoveByKey удаляет описание принтера из списка доступных для печати принтеров по указанному ключу.


## Комментарии


Метод возвращает значение True, если удаление прошло успешно.
