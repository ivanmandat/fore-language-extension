# IMetabaseDeviceDescriptors


## Описание


Интерфейс IMetabaseDeviceDescriptors предназначен для работы с коллекцией описаний устройств.


## Свойства


 Имя свойства
 Краткое описание


 ![](../../Property_Image.gif)
 [Count](IMetabaseDeviceDescriptors.Count.htm)
 Свойство Count возвращает количество устройств в списке доступных для печати принтеров.


 ![](../../Property_Image.gif)
 [Item](IMetabaseDeviceDescriptors.Item.htm)
 Свойство Item возвращает описание устройства по индексу.


 ![](../../Property_Image.gif)
 [ItemByKey](IMetabaseDeviceDescriptors.ItemByKey.htm)
 Свойство ItemByKey возвращает описание устройства по ключу.


## Методы


 Имя метода
 Краткое описание


 ![](../../Sub_Image.gif)
 [Add](IMetabaseDeviceDescriptors.Add.htm)
 Метод Add добавляет описание принтера в список доступных для печати принтеров по указанному наименованию.


 ![](../../Sub_Image.gif)
 [Remove](IMetabaseDeviceDescriptors.Remove.htm)
 Метод Remove удаляет описание принтера из списка доступных для печати принтеров по его индексу.


 ![](../../Sub_Image.gif)
 [RemoveAll](IMetabaseDeviceDescriptors.RemoveAll.htm)
 Метод RemoveAll удаляет все принтеры из списка доступных для печати принтеров.


 ![](../../Sub_Image.gif)
 [RemoveByKey](IMetabaseDeviceDescriptors.RemoveByKey.htm)
 Метод RemoveByKey удаляет описание принтера из списка доступных для печати принтеров по указанному ключу.
