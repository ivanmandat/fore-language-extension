# IMetabaseDeviceDescriptors.Remove


## Синтаксис


Remove (Index:Integer): Boolean;


## Параметры


Index. Индекс описания устройства в списке.


## Описание


Метод Remove удаляет описание принтера из списка доступных для печати принтеров по его индексу.
