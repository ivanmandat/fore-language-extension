# IMetabaseDeviceDescriptors.Add


## Синтаксис


		Add(DeviceName:String): [IMetabaseDeviceDescriptor;](../IMetabaseDeviceDescriptor/IMetabaseDeviceDescriptor.htm)


## Параметры


DeviceName. Наименование устройства.


## Описание


Метод Add добавляет описание
 принтера в список доступных для печати принтеров по указанному наименованию.
