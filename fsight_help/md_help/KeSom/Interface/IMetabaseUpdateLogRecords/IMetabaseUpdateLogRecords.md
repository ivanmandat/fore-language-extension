# IMetabaseUpdateLogRecords


Сборка: Metabase;


## Описание


Интерфейс IMetabaseUpdateLogRecords
 предназначен для работы с коллекцией записей журнала процесса обновления.


## Иерархия наследования


IMetabaseUpdateLogRecords


## Комментарии


Коллекцию записей можно получить, используя свойство [IMetabaseUpdateLog.Records](../IMetabaseUpdateLog/IMetabaseUpdateLog.Records.htm).
