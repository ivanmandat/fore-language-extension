# IObjectCheckOutInfo


## Описание


Интерфейс IObjectCheckOutInfo
 содержит свойства для работы с коллекцией записей о блокировании объекта
 репозитория.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IObjectCheckOutInfo.Count.htm)
		 Свойство Count возвращает
		 количество записей о блокировании объекта.


		 ![](../../Property_Image.gif)
		 [Item](IObjectCheckOutInfo.Item.htm)
		 Свойство Item возвращает
		 информацию о блокировке объекта репозитория.
