# IMetabaseUpdateNode.Index


## Синтаксис


Index: Integer;


## Описание


Свойство доступно только для чтения.


Свойство Index возвращает индекс объекта обновления.


## Комментарии


Index возвращает «-1» для корневого элемента структуры обновления.
