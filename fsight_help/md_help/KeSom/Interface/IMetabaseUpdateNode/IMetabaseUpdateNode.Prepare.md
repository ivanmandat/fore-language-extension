# IMetabaseUpdateNode.Prepare


## Синтаксис


Prepare;


## Описание


Метод Prepare осуществляет подготовку объекта к обновлению.


## Комментарии


При выполнении метода осуществляется проверка соответствия параметров объекта в обновлении и соответствующего объекта в репозитории. Для объекта в обновлении будет выставлено свойство [ApplyState](../IMetabaseUpdateObjectNode/IMetabaseUpdateObjectNode.ApplyState.htm).


Примечание. Данный метод актуально использовать только для объектов обновления, связанных с объектами репозитория.
