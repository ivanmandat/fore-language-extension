# IAccessControlList


## Описание


Интерфейс IAccessControlList
 содержит свойства и методы списка контроля доступа.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IAccessControlList.Count.htm)
		 Свойство Count возвращает
		 количество элементов в списке контроля.


		 ![](../../Property_Image.gif)
		 [Item](IAccessControlList.Item.htm)
		 Свойство Item возвращает
		 элемент контроля доступа.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddAce](IAccessControlList.AddAce.htm)
		 Метод AddAce добавляет
		 элемент контроля доступа.


		 ![](../../Sub_Image.gif)
		 [Append](IAccessControlList.Append.htm)
		 Метод Append добавляет
		 элемент контроля доступа из другого списка.


		 ![](../../Sub_Image.gif)
		 [Assign](IAccessControlList.Assign.htm)
		 Метод Assign заменяет
		 элементы списка элементами другого списка контроля доступа.


		 ![](../../Sub_Image.gif)
		 [Clear](IAccessControlList.Clear.htm)
		 Метод Clear очищает
		 список элементов контроля доступа.


		 ![](../../Sub_Image.gif)
		 [Remove](IAccessControlList.Remove.htm)
		 Метод Remove удаляет
		 элемент контроля доступа и возвращает признак удачного удаления.
