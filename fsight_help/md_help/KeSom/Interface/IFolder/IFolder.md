# IFolder


## Описание


Интерфейс IFolder является базовым
 интерфейсом объекта репозитория - [Папка](UiNavObj.chm::/UiNavObj_Folder.htm).


## Комментарии


Интерфейс не содержит собственных свойств и методов. Для работы со свойствами
 папки и ее содержимым используются обобщенные интерфейсы всех объектов
 репозитория: [IMetabaseObjectDescriptor](../IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm)
 и [IMetabaseObject](../IMetabaseObject/IMetabaseObject.htm).
