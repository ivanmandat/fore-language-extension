# IMetabaseUpdateObjectExParams


## Описание


Интерфейс IMetabaseUpdateObjectExParams является базовым для настройки расширенных параметров обновления объекта.


## Комментарии


В текущей реализации расширенные параметры обновления доступны для справочников НСИ. Интерфейсом, содержащим расширенные параметры для обновления справочников НСИ и поддерживающим данный интерфейс, является [IRdsDictionaryUpdateExParams](KeRds.chm::/Interface/IRdsDictionaryUpdateExParams/IRdsDictionaryUpdateExParams.htm).


Для создания расширенных параметров используйте метод [IMetabaseObject.CreateUpdateExParams](../IMetabaseObject/IMetabaseObject.CreateUpdateExParams.htm).
