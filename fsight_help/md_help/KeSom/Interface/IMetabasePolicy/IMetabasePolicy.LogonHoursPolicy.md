# IMetabasePolicy.LogonHoursPolicy


## Синтаксис


		LogonHoursPolicy(Subject: [ISecuritySubject](../ISecuritySubject/ISecuritySubject.htm)):
		 [ILogonHoursPolicy](../ILogonHoursPolicy/ILogonHoursPolicy.htm);


## Параметры


Subject. Субъект безопасности,
 для которого необходимо распределить доступ к платформе по времени.


## Описание


Свойство LogonHoursPolicy возвращает
 параметры доступа к платформе по времени для указанного субъекта безопасности.
