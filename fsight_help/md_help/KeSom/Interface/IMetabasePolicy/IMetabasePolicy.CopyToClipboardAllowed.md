# IMetabasePolicy.CopyToClipboardAllowed


## Синтаксис


		CopyToClipboardAllowed: Boolean;


## Описание


Свойство CopyToClipboardAllowed
 определяет признак возможности копирования данных в буфер обмена и копирования
 экрана.


## Комментарии


По умолчанию свойство принимает значение True,
 при этом есть возможность использовать буфер обмена при работе в платформе.
 Если установлено значение False,
 то при работе в платформе копирование данных и использование кнопки «Print
 Screen» для копирования экрана недоступно.
