# IMetabasePolicy.IsaPresent


## Синтаксис


IsaPresent: Boolean;


## Описание


Свойство IsaPresent возвращает признак включенного режима разделения ролей между администратором информационной безопасности и прикладным администратором.


## Комментарии


Свойство возвращает значение True, если был активирован администратор информационной безопасности, и False, если администрирование пользователей и раздача прав на объекты осуществляется прикладным администратором.
