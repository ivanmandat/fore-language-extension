# IMetabasePolicy.Privilege


## Синтаксис


Privilege(Name: String): [IPrivilege](../IPrivilege/IPrivilege.htm);


## Параметры


Name. Наименование привилегии.


## Описание


Свойство Privilege возвращает
 параметры привилегии по указанному наименованию.


## Комментарии


Доступные наименования привилегий можно получить, используя свойство
 [PrivilegeName](IMetabasePolicy.PrivilegeName.htm).

Использование свойства приведено в примере для [IMetabaseSecurity.AddGroup](../IMetabaseSecurity/IMetabaseSecurity.AddGroup.htm).
