# IMetabasePolicy.UserPrivateFolderRoot


## Синтаксис


UserPrivateFolderRoot: [IMetabaseObjectDescriptor](../IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm);


## Описание


Свойство UserPrivateFolderRoot
 определяет корневую папку для [личных папок](Admin.chm::/03_Admin/Users/Admin_UserFolder.htm)
 пользователей.
