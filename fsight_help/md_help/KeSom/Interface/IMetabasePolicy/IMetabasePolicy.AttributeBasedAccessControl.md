# IMetabasePolicy.AttributeBasedAccessControl


## Синтаксис


AttributeBasedAccessControl: Boolean;


## Описание


Свойство AttributeBasedAccessControl
 определяет признак использования [атрибутного метода](admin.chm::/04_SecurityPolicy/Admin_ABAC.htm) разграничения доступа к объектам.
