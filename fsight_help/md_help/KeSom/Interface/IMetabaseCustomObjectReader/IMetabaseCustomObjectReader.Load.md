# IMetabaseCustomObjectReader.Load


## Синтаксис


Load;


## Описание


Метод Load загружает данные
 из репозитория в объект в памяти ПК.


## Комментарии


Данный метод должен быть вызван перед записью в конечный приёмник с
 помощью методов: [ReadToFile](IMetabaseCustomObjectReader.ReadToFile.htm),
 [ReadToStream](IMetabaseCustomObjectReader.ReadToStream.htm)
 или [ReadToXML](IMetabaseCustomObjectReader.ReadToXML.htm).
