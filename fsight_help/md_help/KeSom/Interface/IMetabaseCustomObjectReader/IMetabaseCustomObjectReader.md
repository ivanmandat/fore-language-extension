# IMetabaseCustomObjectReader


## Описание


Интерфейс IMetabaseCustomObjectReader
 содержит свойства и методы для чтения данных из объекта пользовательского
 класса.


## Свойства, унаследованные от [IMetabaseCustomObjectSerializer](../IMetabaseCustomObjectSerializer/IMetabaseCustomObjectSerializer.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ElementDependencies](../IMetabaseCustomObjectSerializer/IMetabaseCustomObjectSerializer.ElementDependencies.htm)
		 Свойство ElementDependencies
		 определяет коллекцию зависимостей объекта пользовательского класса
		 от элементов справочников НСИ.


		 ![](../../Property_Image.gif)
		 [Object](../IMetabaseCustomObjectSerializer/IMetabaseCustomObjectSerializer.Object.htm)
		 Свойство Object возвращает
		 объект, с данными которого ведется работа.


		 ![](../../Property_Image.gif)
		 [References](../IMetabaseCustomObjectSerializer/IMetabaseCustomObjectSerializer.References.htm)
		 Свойство References
		 возвращает зависимости объекта.


		 ![](../../Property_Image.gif)
		 [Selections](../IMetabaseCustomObjectSerializer/IMetabaseCustomObjectSerializer.Selections.htm)
		 Свойство Selections
		 возвращает коллекцию отметок измерений, связанных с объектом пользовательского
		 класса.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Load](IMetabaseCustomObjectReader.Load.htm)
		 Метод Load загружает
		 данные из репозитория в объект в памяти ПК.


		 ![](../../Sub_Image.gif)
		 [ReadToFile](IMetabaseCustomObjectReader.ReadToFile.htm)
		 Метод ReadToFile считывает
		 данные из объекта пользовательского класса и загружает их в указанный
		 файл.


		 ![](../../Sub_Image.gif)
		 [ReadToStream](IMetabaseCustomObjectReader.ReadToStream.htm)
		 Метод ReadToStream
		 считывает данные из объекта пользовательского класса и загружает
		 их в указанный поток.


		 ![](../../Sub_Image.gif)
		 [ReadToXML](IMetabaseCustomObjectReader.ReadToXML.htm)
		 Метод ReadToXML считывает
		 данные из объекта пользовательского класса в XML-формате и загружает
		 их в указанную переменную.
