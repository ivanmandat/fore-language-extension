# IDbDrivers


## Описание


Интерфейс IDbDrivers содержит
 свойства и методы коллекции драйверов СУБД, поддерживаемых в платформе.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDbDrivers.Count.htm)
		 Свойство Count возвращает
		 количество драйверов СУБД, поддерживаемых в платформе.


		 ![](../../Property_Image.gif)
		 [Item](IDbDrivers.Item.htm)
		 Свойство Item возвращает
		 информацию о драйвере СУБД, поддерживаемом в платформе.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](IDbDrivers.FindById.htm)
