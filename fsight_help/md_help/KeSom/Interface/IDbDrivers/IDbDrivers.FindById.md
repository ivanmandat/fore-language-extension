# IDbDrivers.FindById


## Синтаксис


FindById(Id: String): [IDbDriver](../IDbDriver/IDbDriver.htm);


## Параметры


Id. Идентификатор драйвера, по которому будет
 осуществляться поиск.


## Описание


Метод FindById осуществляет
 поиск и возвращает информацию о драйвере СУБД, идентификатор которого
 передается посредством входного параметра.


## Комментарии


Если драйвер с указанным идентификатором не найдет, то метод возвращает
 значение Null.
