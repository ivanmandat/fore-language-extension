# IAccessAttributeValue.ToString


## Синтаксис


ToString: String;


## Описание


Свойство ToString возвращает строку, соответствующую битовой маске доступа.


## Комментарии


Вся маска разбивается на группы по 32 бита, каждой группе в строке соответствует подстрока из 12 символов, в которые входит десятичное число, представляющее группу маски, и пробелы.
