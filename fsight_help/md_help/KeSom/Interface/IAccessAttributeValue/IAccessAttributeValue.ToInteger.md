# IAccessAttributeValue.ToInteger


## Синтаксис


ToInteger: Integer;


## Описание


Свойство ToInteger возвращает число в десятичном виде, соответствующее битовой маске доступа. Каждому биту маски сопоставлен субъект безопасности.
