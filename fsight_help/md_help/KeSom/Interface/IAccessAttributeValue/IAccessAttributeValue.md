# IAccessAttributeValue


## Описание


Интерфейс IAccessAttributeValue предназначен для работы со значением атрибута доступа.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BitCount](IAccessAttributeValue.BitCount.htm)


		 Свойство BitCount возвращает
		 количество субъектов безопасности, для которых могут быть определены
		 права.


		 ![](../../Property_Image.gif)
		 [ToInteger](IAccessAttributeValue.ToInteger.htm)


		 Свойство ToInteger
		 возвращает число в десятичном виде, соответствующее битовой маске
		 доступа.


		 ![](../../Property_Image.gif)
		 [ToString](IAccessAttributeValue.ToString.htm)


		 Свойство ToString возвращает
		 строку, соответствующую битовой маске доступа.


		 ![](../../Property_Image.gif)
		 [Value](IAccessAttributeValue.Value.htm)


		 Свойство Value определяет
		 значение права доступа. При значении True
		 соответствующие права имеются, при значении False
		 - прав нет.
