# ICertCredentials.Cookie


## Синтаксис


Cookie: String;


## Описание


Свойство Cookie определяет случайное
 значение, используемое для установления соответствия между цифровой подписью
 и полученным блоком данных.


## Комментарии


Блок данных, подписанный цифровой подписью, указывается в свойстве [ICertCredentials.Signature](ICertCredentials.Signature.htm).

Пример использования приведен в описании свойства [ICertCredentials.Signature](ICertCredentials.Signature.htm).
