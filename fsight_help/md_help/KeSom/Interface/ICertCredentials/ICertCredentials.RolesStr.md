# ICertCredentials.RolesStr


## Синтаксис


RolesStr: String;


## Описание


Свойство RolesStr определяет
 группы пользователей, в которые будет включен пользователь.


## Комментарии


В качестве значения свойства необходимо указать наименование одной или
 нескольких групп, разделенных символом «;». Если свойство не указано,
 то пользователь будет включен в группу «Встроенная
 группа пользователей».
