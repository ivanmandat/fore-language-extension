# ISecuritySubjectsSearch.AbandonSearch


## Синтаксис


AbandonSearch;


## Описание


Метод AbandonSearch прекращает
 поиск субъектов безопасности.


## Комментарии


При выполнении метода список найденных субъектов безопасности, возвращаемый
 свойством [ISecuritySubjectsSearch.Subjects](ISecuritySubjectsSearch.Subjects.htm),
 не очищается.


Для осуществления поиска используйте метод [ISecuritySubjectsSearch.ExecuteSearch](ISecuritySubjectsSearch.ExecuteSearch.htm).
