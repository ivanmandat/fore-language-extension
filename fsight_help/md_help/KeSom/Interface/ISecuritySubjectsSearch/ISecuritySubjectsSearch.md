# ISecuritySubjectsSearch


## Описание


Интерфейс ISecuritySubjectsSearch
 содержит свойства и методы предназначенные для поиска субъектов безопасности.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AreaIncludeDB](ISecuritySubjectsSearch.AreaIncludeDB.htm)
		 Свойство AreaIncludeDB
		 определяет: производить ли поиск среди субъектов СУБД.


		 ![](../../Property_Image.gif)
		 [AreaIncludeMetabaseNT](ISecuritySubjectsSearch.AreaIncludeMetabaseNT.htm)
		 Свойство AreaIncludeMetabaseNT
		 определяет признак поиска среди доменных субъектов безопасности,
		 которые были зарегистрированы в репозитории и для которых был
		 установлен признак отложенной загрузки описания.


		 ![](../../Property_Image.gif)
		 [AreaIncludeNT](ISecuritySubjectsSearch.AreaIncludeNT.htm)
		 Свойство AreaIncludeNT
		 определяет: производить ли поиск среди доменных субъектов безопасности.


		 ![](../../Property_Image.gif)
		 [DomainCredentials](ISecuritySubjectsSearch.DomainCredentials.htm)
		 Свойство DomainCredentials
		 определяет учетные данные, под которыми будет осуществляться авторизация
		 в домене при поиске.


		 ![](../../Property_Image.gif)
		 [DomainSelectCriteria](ISecuritySubjectsSearch.DomainSelectCriteria.htm)
		 Свойство DomainSelectCriteria
		 определяет способ задания домена, в котором производится поиск
		 субъектов безопасности.


		 ![](../../Property_Image.gif)
		 [NameCriteria](ISecuritySubjectsSearch.NameCriteria.htm)
		 Свойство NameCriteria
		 определяет наименование искомого субъекта безопасности.


		 ![](../../Property_Image.gif)
		 [ProtocolSelectCriteria](ISecuritySubjectsSearch.ProtocolSelectCriteria.htm)
		 Свойство ProtocolSelectCriteria
		 определяет протокол, который будет использован при поиске в службе
		 каталогов.


		 ![](../../Property_Image.gif)
		 [SubjectCriteria](ISecuritySubjectsSearch.SubjectCriteria.htm)
		 Свойство SubjectCriteria
		 определяет типы субъектов безопасности, которые необходимо найти.


		 ![](../../Property_Image.gif)
		 [Subjects](ISecuritySubjectsSearch.Subjects.htm)
		 Свойство Subjects возвращает
		 найденные субъекты безопасности.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AbandonSearch](ISecuritySubjectsSearch.AbandonSearch.htm)
		 Метод AbandonSearch
		 прекращает поиск субъектов безопасности.


		 ![](../../Sub_Image.gif)
		 [ExecuteSearch](ISecuritySubjectsSearch.ExecuteSearch.htm)
		 Метод ExecuteSearch
		 осуществляет поиск субъектов безопасности.
