# ISecuritySubjectsSearch.AreaIncludeNT


## Синтаксис


AreaIncludeNT: Boolean;


## Описание


Свойство AreaIncludeNT определяет признак поиска среди доменных субъектов безопасности.


## Комментарии


Если свойству установить значение True, то будет производиться поиск среди доменных субъектов безопасности.


Поиск будет производиться среди доменных пользователей, добавленных в менеджере безопасности, а также среди пользователей, для которых включен признак [отложенной загрузки описания](../IMetabaseUser/IMetabaseUser.IsDeferred.htm).
