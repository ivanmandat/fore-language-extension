# ISecuritySubjectsSearch.AreaIncludeDB


## Синтаксис


		AreaIncludeDB: Boolean;


## Описание


Свойство AreaIncludeDB определяет
 признак поиска среди субъектов СУБД.


## Комментарии


Если свойству установлено значение True,
 то будет производиться поиск среди субъектов СУБД.


Поиск будет производиться среди пользователей, добавленных в менеджере
 безопасности, а также среди пользователей, для которых включен признак
 [отложенной загрузки
 описания](../IMetabaseUser/IMetabaseUser.IsDeferred.htm).
