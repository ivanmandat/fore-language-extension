# ILogonHours.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс временного промежутка.


## Описание


Метод Remove осуществляет удаление временного промежутка, в котором имеется ограничение времени доступа пользователя. Индекс периода передается посредством параметра Index.
