# ILogonHours


## Описание


Интерфейс ILogonHours содержит
 свойства и методы для ограничения времени доступа пользователя в течение
 дня.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ILogonHours.Count.htm)
		 Свойство Count возвращает
		 количество временных промежутков, на которых присутствует ограничение
		 доступа пользователя, в течение дня.


		 ![](../../Property_Image.gif)
		 [Item](ILogonHours.Item.htm)
		 Свойство Item возвращает
		 параметры временного промежутка.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Clear](ILogonHours.Clear.htm)
		 Метод Clear осуществляет
		 очистку списка временных промежутков, на которых имеется ограничение
		 доступа пользователя.


		 ![](../../Sub_Image.gif)
		 [CreateCopy](ILogonHours.CreateCopy.htm)
		 Метод CreateCopy создает
		 копию списка временных промежутков, на которых имеется ограничение
		 доступа пользователя.


		 ![](../../Sub_Image.gif)
		 [FindByDate](ILogonHours.FindByDate.htm)
		 Метод FindByDate осуществляет
		 поиск временного промежутка, содержащего указанное время.


		 ![](../../Sub_Image.gif)
		 [Remove](ILogonHours.Remove.htm)
		 Метод Remove осуществляет
		 удаление временного промежутка.


		 ![](../../Sub_Image.gif)
		 [SetPeriodAccess](ILogonHours.SetPeriodAccess.htm)
		 Метод SetPeriodAccess
		 устанавливает параметры доступа пользователя для указанного промежутка
		 времени.
