# ILogonHours.FindByDate


## Синтаксис


FindByDate(Current: DateTime): [ILogonPeriod](../ILogonPeriod/ILogonPeriod.htm);


## Параметры


Current - время, для которого необходимо найти временной промежуток, в котором имеется ограничение времени доступа пользователя.


## Описание


Метод FindByDate осуществляет поиск временного промежутка, содержащего время, передаваемое посредством параметра Current. Метод возвращает параметры временного промежутка в случае удачного поиска, и Null, если указанная дата не попадает ни в один из периодов.
