# IMetabaseUpdateObjectNode.ObjectName


## Синтаксис


ObjectName: String;


## Описание


Свойство ObjectName определяет наименование обновляемого объекта.


## Комментарии


По умолчанию значение свойства соответствует наименованию объекта, добавленного в обновление из репозитория. Значение данного свойства может быть изменено для изменения наименования обновляемого объекта.
