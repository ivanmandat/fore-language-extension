# IMetabaseUpdateObjectNode.ApplyState


## Синтаксис


ApplyState: [IMetabaseUpdateObjectApplyState](../IMetabaseUpdateObjectApplyState/IMetabaseUpdateObjectApplyState.htm);


## Описание


Свойство ApplyState возвращает
 параметры готовности объекта к обновлению.


## Комментарии


Значение свойства заполняется после вызова метода [IMetabaseUpdate.Prepare](../IMetabaseUpdate/IMetabaseUpdate.Prepare.htm)
 всего обновления, либо метода [IMetabaseUpdateNode.Prepare](../IMetabaseUpdateNode/IMetabaseUpdateNode.Prepare.htm)
 конкретного объекта в обновлении.
