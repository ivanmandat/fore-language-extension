# IMetabaseUpdateObjectNode.SDApplyLog


## Синтаксис


SDApplyLog: String;


## Описание


Свойство SDApplyLog возвращает лог изменения прав доступа.


## Комментарии


Необходимо свойству [IMetabaseUpdateObjectNode.UpdatePart](../../Enums/MetabaseObjectUpdatePart.htm) присвоить значение ReplaceSecurityDescriptor.

Использование свойства приведено в примере для свойства [IMetabaseUpdateProgress.OnSkip](../IMetabaseUpdateProgress/IMetabaseUpdateProgress.OnSkip.htm).
