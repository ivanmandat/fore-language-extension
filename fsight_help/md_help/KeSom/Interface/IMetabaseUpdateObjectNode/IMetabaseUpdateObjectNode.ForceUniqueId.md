# IMetabaseUpdateObjectNode.ForceUniqueId


## Синтаксис


ForceUniqueId: Boolean;


## Описание


Свойство ForceUniqueId определяет признак генерирования нового уникального идентификатора объекта при совпадении идентификаторов объектов в обновлении и в репозитории.


## Комментарии


По умолчанию свойству установлено значение False, при этом вне зависимости от выбранного типа обновления (по ключам, либо по идентификаторам) идентификатор обновляемого объекта будет установлен такой, какой имеет объект в обновлении.


Если свойству установить значение True, то после применения обновления для обновленного объекта в репозитории будет сгенерирован новый уникальный идентификатор.
