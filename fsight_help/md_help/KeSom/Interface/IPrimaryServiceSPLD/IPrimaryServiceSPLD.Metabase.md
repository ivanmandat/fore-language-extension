# IPrimaryServiceSPLD.Metabase


## Синтаксис


Metabase: String;


## Описание


Свойство Metabase определяет
 идентификатор описания репозитория, к которому необходимо подключиться
 через веб-сервис платформы.


## Комментарии


В качестве значения свойства необходимо указать идентификатор описания
 репозитория, который зарегистрирован в реестре там, где установлен веб-сервис
 платформы. Настройки из этого описания будут использоваться для фактического
 подключения при подключении через веб-сервис.
