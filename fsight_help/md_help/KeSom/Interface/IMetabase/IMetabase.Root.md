# IMetabase.Root


## Синтаксис


		Root: [IMetabaseObjectDescriptor](../IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm);


## Описание


Свойство Root возвращает описание
 корневой папки репозитория.


## Комментарии


Корневая папка репозитория - это папка самого верхнего уровня, содержимое
 которой доступно пользователю.


По умолчанию корневой папкой является виртуальная папка, создаваемая
 при входе в репозиторий. В рамках этот папки отстраивается дерево всех
 объектов репозитория. Наименование и идентификатор данной папке соответствуют
 наименованию репозитория, ключ папки - «0». Изменение описания или прав
 доступа к данной папке запрещено, она может выступать только в качестве
 родительского объекта для других объектов.


Также, используя свойство [IMetabasePolicy.RootObject](../IMetabasePolicy/IMetabasePolicy.RootObject.htm),
 в качестве корневой папки может быть задана любая папка репозитория. Указанная
 папка будет использована в качестве корневой, если вход осуществляется
 под пользователем, который не является администратором или администратором
 информационной безопасности (АИБ). Объекты, которые не являются дочерними
 для указанной папки, будут недоступны.
