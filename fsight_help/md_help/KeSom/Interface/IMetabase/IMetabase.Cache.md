# IMetabase.Cache


## Синтаксис


Cache: [IMetabaseCache](../IMetabaseCache/IMetabaseCache.htm);


## Описание


Свойство Cache возвращает содержимое
 кеша репозитория.


## Комментарии


В кеше репозитория хранятся кешированные данные открываемых объектов.
 Данные в кеше хранятся только в рамках текущего соединения с репозиторием.
 Кеширование данных позволяет уменьшить количество запросов к базе репозитория,
 что ускоряет работу с объектами на локальном компьютере. Если осуществляется
 работа с параметрическими объектами, то кеширование осуществяется для
 каждого набора параметров.
