# IMetabase.CheckInEx


## Синтаксис


		CheckInEx(Objects: [IMetabaseObjectDescriptors](../IMetabaseObjectDescriptors/IMetabaseObjectDescriptors.htm);
		 Var ResId: Integer): Boolean;


## Параметры


Objects. Объекты, изменения
 которых необходимо опубликовать;


ResId. Переменная, в которую
 будет помещен код ошибки.


## Описание


Метод CheckInEx осуществляет
 публикацию изменений группы объектов.


## Комментарии


Данный метод актуально использовать, если объекты добавлены в систему
 управления версиями.


При вызове метода будет открыто окно публикации изменений объектов.
 При удачной публикации метод возвращает значение True.
 Код ошибки при этом будет «0».


Если публикация изменений не произошла, то метод вернет значение False. В параметр ResId
 будет помещен код ошибки.
