# IMetabase.SingleUserMode


## Синтаксис


		SingleUserMode: Boolean;


## Описание


Свойство SingleUserMode возвращает
 признак возможности работы в репозитории только под одной учетной записью.


## Комментарии


Свойство возвращает значение True, если текущий репозиторий создан на
 базе СУБД, которая не поддерживает разграничение прав доступа и организацию
 политики безопасности. В текущей реализации такой СУБД является - SQLite.


Для всех других видов СУБД свойство вернет значение False.
