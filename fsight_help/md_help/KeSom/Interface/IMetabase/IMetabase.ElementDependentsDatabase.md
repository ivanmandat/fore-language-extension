# IMetabase.ElementDependentsDatabase


## Синтаксис


		ElementDependentsDatabase: [IMbElementDependentsDatabase](../IMbElementDependentsDatabase/IMbElementDependentsDatabase.htm);


## Описание


Свойство ElementDependentsDatabase
 возвращает ссылки элементов справочников на объекты репозитория.


## Комментарии


Для получения объекта, предназначенного для работы с зависимостями объектов
 от элементов справочников НСИ, используйте свойство [IMetabase.ElementDependenciesDatabase](IMetabase.ElementDependenciesDatabase.htm).
