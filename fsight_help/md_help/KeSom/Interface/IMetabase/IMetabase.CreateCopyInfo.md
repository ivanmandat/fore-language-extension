# IMetabase.CreateCopyInfo


## Синтаксис


		CreateCopyInfo: [IMetabaseObjectCopyInfo](../IMetabaseObjectCopyInfo/IMetabaseObjectCopyInfo.htm);


## Описание


Метод CreateCopyInfo позволяет
 создать объект для осуществления копирования по определенным параметрам.


Копирование осуществляется методом [IMetabase.CopyObject](IMetabase.CopyObject.htm).
