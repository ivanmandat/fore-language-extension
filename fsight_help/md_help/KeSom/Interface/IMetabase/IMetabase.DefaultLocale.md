# IMetabase.DefaultLocale


## Синтаксис


DefaultLocale: [LocaleCodeID](ForeSys.chm::/Enums/LocaleCodeID.htm);


## Описание


Свойство DefaultLocale определяет
 язык репозитория по умолчанию.


## Комментарии


Язык по умолчанию используется для отображения наименований объектов,
 для которых отсутствует наименование на языке, используемом при [подключении
 к репозиторию](IMetabase.CurrentLocale.htm). Если для объекта отсутствует наименование на языке
 по умолчанию и языке, используемом при подключении к репозиторию, то для
 отображения наименования будет использоваться язык репозитория, который
 был установлен при создании объекта.


## Пример


Добавьте ссылку на системную сборку Metabase.


			Sub UserProc;

Var

    MB: IMetabase;

Begin

    MB := MetabaseClass.Active;

    MB.DefaultLocale := LocaleCodeID.English_UnitedStates;

End Sub UserProc;


После выполнения примера для текущего репозитория в качестве языка по
 умолчанию будет выбран Английский (США). Данное значение будет применено
 и сохранено в БД репозитория.
