# IMetabase.UndoCheckOutEx


## Синтаксис


		UndoCheckOutEx(Objects: [IMetabaseObjectDescriptors](../IMetabaseObjectDescriptors/IMetabaseObjectDescriptors.htm);
		 Var ResId: Integer): Boolean;


## Параметры


Objects. Объекты, изменения
 которых необходимо отменить;


ResId. Переменная, в которую
 будет помещен код ошибки.


## Описание


Метод UndoCheckOutEx осуществляет
 отмену изменений и возвращает предыдущую версию объектов.


## Комментарии


Данный метод актуально использовать, если объекты добавлены в систему
 управления версиями.


При вызове метода будет открыт диалог отмены изменений объектов. При
 удачной отмене и возврату к предыдущим версиям объектов метод возвращает
 значение True. Код ошибки при
 этом будет «0».


Если отмена изменений не произошла, то метод вернет значение False.
 В параметр ResId будет помещен
 код ошибки.
