# IMetabase.IsScreenshotSupported


## Синтаксис


		IsScreenshotSupported: Boolean;


## Описание


Свойство IsScreenshotSupported
 возвращает признак поддержки отображения объектов в виде значков.


## Комментарии


Свойство возвращает значение True,
 если в менеджере безопасности установлен флажок «[Разрешить отображение объектов в виде значков
 без учета прав доступа на объекты](Admin.chm::/03_Admin/Access_control_settings.htm#extralargeicons)», и False, если флажок снят.
