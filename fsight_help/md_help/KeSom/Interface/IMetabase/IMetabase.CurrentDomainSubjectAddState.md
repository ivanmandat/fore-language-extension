# IMetabase.CurrentDomainSubjectAddState


## Синтаксис


		CurrentDomainSubjectAddState(Type: [SecuritySubjectType](../../Enums/SecuritySubjectType.htm)):
		 [DomainSubjectAddState](../../Enums/DomainSubjectAddState.htm);


## Параметры


Type. Тип субъектов безопасности,
 для которых необходимо определить параметры добавления.


## Описание


Свойство CurrentDomainSubjectAddState
 определяет настройки добавления доменных субъектов безопасности.


## Комментарии


Настройки, установленные в данном свойстве, будут применяться для всех
 добавляемых пользователей. Данное свойство является аналогом свойства
 [IMetabaseSecurity.CurrentDomainSubjectAddState](../IMetabaseSecurity/IMetabaseSecurity.CurrentDomainSubjectAddState.htm).
