# IMetabase.CurrentLocale


## Синтаксис


		CurrentLocale: [LocaleCodeID](ForeSys.chm::/Enums/LocaleCodeID.htm);


## Описание


Свойство CurrentLocale определяет
 текущий язык репозитория.


## Комментарии


Данное свойство используется для определения языка, с которым осуществлено
 подключение к репозиторию, и для отображения наименований объектов. Если
 для объектов отсутствует наименование на текущем языке репозитория, то
 для них будет использоваться язык, установленный в качестве [языка
 по умолчанию](IMetabase.DefaultLocale.htm).
