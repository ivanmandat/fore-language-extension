# IMetabaseCustomClass.Data


## Синтаксис


Data: String;


## Описание


Свойство Data определяет текстовые
 пользовательские данные для класса.


## Комментарии


Используется для сохранения текста произвольного формата в пользовательском
 классе. Заданный текст восстанавливается с метаданными класса.
