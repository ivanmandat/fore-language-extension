# IMetabaseCustomClass.ImplementationURL


## Синтаксис


ImplementationURL: String;


## Описание


Свойство ImplementationURL устанавливает
 URL-адрес скрипта, реализующего операции объекта класса, которые будут
 выполняться из веб-приложения.
