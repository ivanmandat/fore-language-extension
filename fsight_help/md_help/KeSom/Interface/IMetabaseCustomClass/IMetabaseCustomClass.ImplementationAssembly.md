# IMetabaseCustomClass.ImplementationAssembly


## Синтаксис


ImplementationAssembly: [IMetabaseObjectDescriptor](../IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm);


## Описание


Свойство ImplementationAssembly
 определяет сборку, содержащую реализацию класса, который будет обрабатывать
 [операции](IMetabaseCustomClass.Operations.htm) пользовательского
 класса в настольном приложении.


## Комментарии


В качестве значения данного свойства необходимо указать описание модуля/формы/сборки
 репозитория. Сам класс задаётся в свойстве [ImplementationClass](IMetabaseCustomClass.ImplementationClass.htm).
