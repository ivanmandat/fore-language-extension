# IMetabaseCustomClass.LargeImageIndex


## Синтаксис


LargeImageIndex: Integer;


## Описание


Свойство LargeImageIndex определяет
 для класса индекс крупной пиктограммы в коллекции.


## Комментарии


Для получения коллекции крупных пиктограмм используйте свойство [IMetabaseCustomExtender.LargeImages](../IMetabaseCustomExtender/IMetabaseCustomExtender.LargeImages.htm).
