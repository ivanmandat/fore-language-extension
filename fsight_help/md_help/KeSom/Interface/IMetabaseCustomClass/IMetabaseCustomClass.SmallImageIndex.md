# IMetabaseCustomClass.SmallImageIndex


## Синтаксис


SmallImageIndex: Integer;


## Описание


Свойство SmallImageIndex определяет
 для класса индекс маленькой пиктограммы в коллекции.


## Комментарии


Для получения коллекции маленьких пиктограмм используйте свойство [IMetabaseCustomExtender.SmallImages](../IMetabaseCustomExtender/IMetabaseCustomExtender.SmallImages.htm).
