# IMetabaseCustomClass.StandardOperationsWeb


## Синтаксис


StandardOperationsWeb: [IMetabaseCustomClassStandardOperationsWeb](../IMetabaseCustomClassStandardOperationsWeb/IMetabaseCustomClassStandardOperationsWeb.htm)


## Описание


Свойство StandardOperationsWeb
 возвращает коллекцию стандартных операций класса, доступных при работе
 в веб-приложении.


## Комментарии


Для объектов существуют следующие стандартные операции при работе в
 веб-приложении:


		 Наименование операция
		 Идентификатор операции


		 Создать
		 CREATEOBJECT


		 Открыть
		 OPEN


		 Редактировать
		 EDIT


		 Удалить
		 DELETE


		 Вставить
		 PASTE
