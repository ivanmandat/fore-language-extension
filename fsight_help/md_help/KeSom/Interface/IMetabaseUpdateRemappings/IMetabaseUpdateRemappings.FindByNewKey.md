# IMetabaseUpdateRemappings.FindByNewKey


## Синтаксис


FindByNewKey(Key: Integer): [IMetabaseUpdateRemapping](../IMetabaseUpdateRemapping/IMetabaseUpdateRemapping.htm);


## Параметры


Key. Ключ элемента в репозитории-приёмнике.


## Описание


Метод FindByNewKey выполняет
 поиск параметров сопоставления элемента по его ключу в репозитории-приёмнике.


## Комментарии


Если параметры не найдены, то метод вернёт значение Null.
