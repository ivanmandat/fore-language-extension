# IMetabaseUpdateRemappings


Сборка: Metabase;


## Описание


Интерфейс IMetabaseUpdateRemappings
 предназначен для работы с коллекцией повторных сопоставлений элементов
 в обновлении и в репозитории-приёмнике.


## Иерархия наследования


           IMetabaseUpdateRemappings


## Комментарии


Для получения коллекции используйте свойство [IMetabaseUpdateObjectRemapping.Map](../IMetabaseUpdateObjectRemapping/IMetabaseUpdateObjectRemapping.Map.htm).
