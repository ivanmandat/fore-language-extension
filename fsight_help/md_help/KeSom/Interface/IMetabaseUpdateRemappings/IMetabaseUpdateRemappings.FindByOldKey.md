# IMetabaseUpdateRemappings.FindByOldKey


## Синтаксис


FindByOldKey(Key: Integer): [IMetabaseUpdateRemapping](../IMetabaseUpdateRemapping/IMetabaseUpdateRemapping.htm);


## Параметры


Key. Ключ элемента в обновлении.


## Описание


Метод FindByOldKey выполняет
 поиск параметров сопоставления элемента по его ключу в обновлении.


## Комментарии


Если параметры не найдены, то метод вернёт значение Null.
