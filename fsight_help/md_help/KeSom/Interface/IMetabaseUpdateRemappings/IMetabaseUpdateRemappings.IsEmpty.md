# IMetabaseUpdateRemappings.IsEmpty


## Синтаксис


IsEmpty: Boolean;


## Описание


Свойство IsEmpty возвращает
 признак того, пустая ли коллекция.


## Комментарии


Допустимые значения:


	- True. Коллекция пустая
	 и не содержит элементов;


	- False. Коллекция содержит
	 элементы.
