# IAuditFilterObjects.Remove


## Синтаксис


		Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс фильтра, который
 необходимо удалить.


## Описание


Метод Remove осуществляет удаление
 фильтра из коллекции по индексу.
