# IAuditFilterObjects.FindByKey


## Синтаксис


		FindByKey(Key: String): [IAuditFilterObject](../IAuditFilterObject/IAuditFilterObject.htm);


## Параметры


Key. Ключ искомого фильтра.


## Описание


Метод FindByKey осуществляет
 поиск фильтра в коллекции по ключу.


## Комментарии


Метод возвращает параметры фильтра в случае удачного поиска и Null, если фильтр с указанным ключом
 не найден.
