# IAuditFilterObjects.FindById


## Синтаксис


		FindById(Id: String): [IAuditFilterObject](IAuditFilterObjects.htm);


## Параметры


Id. Идентификатор искомого
 фильтра.


## Описание


Метод FindById осуществляет
 поиск фильтра в коллекции по идентификатору.


## Комментарии


Метод возвращает параметры фильтра в случае удачного поиска и Null, если фильтр с указанным идентификатором
 не найден.
