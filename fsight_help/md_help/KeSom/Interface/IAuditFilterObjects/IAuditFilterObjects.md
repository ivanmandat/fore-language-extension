# IAuditFilterObjects


Сборка: Metabase;


## Описание


Интерфейс IAuditFilterObjects
 содержит свойства и методы для работы с коллекцией объектов, хранящих
 фильтры протокола доступа.


## Иерархия наследования


           IAuditFilterObjects


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IAuditFilterObjects.Count.htm)
		 Свойство Count возвращает
		 количество объектов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IAuditFilterObjects.Item.htm)
		 Свойство Item возвращает
		 объект из коллекции по его индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IAuditFilterObjects.Add.htm)
		 Метод Add выполняет
		 добавление нового фильтра в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](IAuditFilterObjects.Clear.htm)
		 Метод Clear осуществляет
		 очистку коллекции фильтров.


		 ![](../../Sub_Image.gif)
		 [FindById](IAuditFilterObjects.FindById.htm)
		 Метод FindById осуществляет
		 поиск фильтра в коллекции по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IAuditFilterObjects.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск фильтра в коллекции по ключу.


		 ![](../../Sub_Image.gif)
		 [Move](IAuditFilterObjects.Move.htm)
		 Метод Move осуществляет
		 перемещение фильтров в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IAuditFilterObjects.Remove.htm)
		 Метод Remove осуществляет
		 удаление фильтра из коллекции по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IAuditFilterObjects.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
