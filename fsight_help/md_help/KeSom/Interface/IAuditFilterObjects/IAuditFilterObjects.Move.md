# IAuditFilterObjects.Move


## Синтаксис


		Move(IndexFrom: Integer; IndexTo: Integer);


## Параметры


IndexFrom. Индекс позиции фильтра,
 который необходимо переместить;


IndexTo. Индекс позиции, в
 которую необходимо переместить фильтр.


## Описание


Метод Move осуществляет перемещение
 фильтров в коллекции.
