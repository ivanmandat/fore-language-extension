# IAuditFilterObjects.RemoveByKey


## Синтаксис


		RemoveByKey(Key: String): Boolean;


## Параметры


Key. Ключ фильтра, который
 необходимо удалить.


## Описание


Метод RemoveByKey осуществляет
 удаление фильтра из коллекции по ключу.
