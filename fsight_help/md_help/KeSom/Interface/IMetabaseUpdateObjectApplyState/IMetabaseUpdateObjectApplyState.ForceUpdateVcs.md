# IMetabaseUpdateObjectApplyState.ForceUpdateVcs


## Синтаксис


ForceUpdateVcs: Boolean;


## Описание


Свойство ForceUpdateVcs определяет,
 применяется ли обновление к объекту, находящемуся под управлением VCS.


## Комментарии


Если для свойства установлено значение True,
 то объект, находящийся под управлением VCS, обновляется. Если установлено
 значение False, обновление к указанному
 объекту применяться не будет.
