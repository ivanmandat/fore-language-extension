# IMetabaseUpdateObjectApplyState


## Описание


Интерфейс IMetabaseUpdateObjectApplyState
 содержит свойства для получения информации о готовности объекта к обновлению.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ApplyAfterParent](IMetabaseUpdateObjectApplyState.ApplyAfterParent.htm)
		 Свойство ApplyAfterParent
		 устанавливает, будет ли применяться обновление к объекту
		 после обновления его родительского объекта.


		 ![](../../Property_Image.gif)
		 [ConflictObject](IMetabaseUpdateObjectApplyState.ConflictObject.htm)
		 Свойство ConflictObject
		 возвращает объект репозитория, с которым возник конфликт при подготовке
		 объекта в обновлении.


		 ![](../../Property_Image.gif)
		 [CreatedObject](IMetabaseUpdateObjectApplyState.CreatedObject.htm)
		 Свойство CreatedObject
		 возвращает описание объекта, созданного при обновлении.


		 ![](../../Property_Image.gif)
		 [ForceParent](IMetabaseUpdateObjectApplyState.ForceParent.htm)
		 Свойство ForceParent
		 определяет описание объекта, который является родительским
		 по отношению к заданному объекту обновления.


		 ![](../../Property_Image.gif)
		 [ForceUpdateObject](IMetabaseUpdateObjectApplyState.ForceUpdateObject.htm)
		 Свойство ForceUpdateObject
		 определяет объект репозитория, к которому будет применено обновление
		 вне зависимости от настроек, установленных для объекта в самом
		 обновлении.


		 [![](../../Property_Image.gif)](IMetabaseUpdateObjectApplyState.ForceUpdateVcs.htm)
		 [ForceUpdateVcs](IMetabaseUpdateObjectApplyState.ForceUpdateVcs.htm)
		 Свойство ForceUpdateVcs
		 определяет, применяется ли обновление к объекту, находящемуся
		 под управлением VCS.


		 ![](../../Property_Image.gif)
		 [State](IMetabaseUpdateObjectApplyState.State.htm)
		 Свойство State определяет
		 состояние готовности объекта к обновлению.


		 ![](../../Property_Image.gif)
		 [UpdateObject](IMetabaseUpdateObjectApplyState.UpdateObject.htm)
		 Свойство UpdateObject
		 определяет объект репозитория, к которому будет применено обновление.
