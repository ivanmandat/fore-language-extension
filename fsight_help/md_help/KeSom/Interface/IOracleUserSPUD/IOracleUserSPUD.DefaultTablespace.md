# IOracleUserSPUD.DefaultTablespace


## Синтаксис


DefaultTablespace: String;


## Описание


Свойство DefaultTablespace определяет наименование табличного пространства на сервере, в рамках которого будут храниться объекты пользователя.


Примечание. Информацию о допустимых наименованиях табличных пространств можно узнать у администратора сервера СУБД.
