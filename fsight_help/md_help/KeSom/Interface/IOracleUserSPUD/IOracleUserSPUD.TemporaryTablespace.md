# IOracleUserSPUD.TemporaryTablespace


## Синтаксис


TemporaryTablespace: String;


## Описание


Свойство TemporaryTablespace определяет наименование временного табличного пространства на сервере, в рамках которого будут создаваться временные объекты пользователя.


Примечание. Информацию о допустимых наименованиях временных табличных пространств можно узнать у администратора сервера СУБД.
