# ILogonPeriod


## Описание


Интерфейс ILogonPeriod содержит
 свойства временного промежутка, на котором имеется ограничение доступа
 пользователя.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Allowed](ILogonPeriod.Allowed.htm)
		 Свойство Allowed возвращает
		 признак возможности доступа в данный промежуток времени.


		 ![](../../Property_Image.gif)
		 [Begin_](ILogonPeriod.Begin_.htm)
		 Свойство Begin_ возвращает
		 начальное время временного промежутка, в котором осуществляется
		 ограничение времени доступа пользователя.


		 ![](../../Property_Image.gif)
		 [End_](ILogonPeriod.End_.htm)
		 Свойство End_ возвращает
		 конечное время временного промежутка, в котором осуществляется
		 ограничение времени доступа пользователя.
