# ILogonPeriod.Allowed


## Синтаксис


Allowed: Boolean;


## Описание


Свойство Allowed возвращает признак возможности доступа в данный промежуток времени. Свойство возвращает True, если в текущем промежутке времени доступ пользователю разрешен и False, если доступ запрещен
