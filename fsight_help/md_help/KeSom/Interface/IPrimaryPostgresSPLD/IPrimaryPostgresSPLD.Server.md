# IPrimaryPostgresSPLD.Server


## Синтаксис


Server: String;


## Описание


Свойство Server определяет наименование
 сервера, на котором расположена СУБД PostgreSQL.


## Комментарии


В качестве значения свойства необходимо указать сетевое имя или IP-адрес
 компьютера, на котором установлена СУБД.
