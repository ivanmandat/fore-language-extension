# IPrimaryPostgresSPLD.SchemaOnly


## Синтаксис


SchemaOnly: String;


## Описание


Свойство SchemaOnly определяет
 идентификатор пользовательской схемы в базе данных.


## Комментарии


Для определения идентификатора базы данных на сервере PostgreSQL используйте
 свойство [IPrimaryPostgresSPLD.Database](IPrimaryPostgresSPLD.Database.htm)
 или [IPrimaryPostgresSPLD.DatabaseOnly](IPrimaryPostgresSPLD.DatabaseOnly.htm).
 При задании идентификатора базы данных с помощью свойства [IPrimaryPostgresSPLD.Database](IPrimaryPostgresSPLD.Database.htm)
 можно указать идентификатор пользовательской схемы через точку.
