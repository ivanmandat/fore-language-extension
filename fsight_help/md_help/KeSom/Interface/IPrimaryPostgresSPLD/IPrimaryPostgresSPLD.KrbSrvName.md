# IPrimaryPostgresSPLD.KrbSrvName


## Синтаксис


KrbSrvName: Boolean;


## Описание


Свойство KrbSrvName определяет
 наименование сервиса MIT Kerberos, с использованием которого осуществляется
 подключение.
