# IUserProfiles.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс профиля.


## Описание


Метод Remove удаляет профиль пользователя, индекс которого передается посредством параметра Index, и возвращает признак удачного удаления.


## Комментарии


Метод возвращает значение True, если профиль был удален и False, если профиль удалить не удалось.
