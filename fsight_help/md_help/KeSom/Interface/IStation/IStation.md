# IStation


## Описание


Интерфейс IStations содержит
 свойства и методы для работы с рабочей станцией.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Access](IStation.Access.htm)
		 Свойство Access определяет
		 тип доступа для станции.


		 ![](../../Property_Image.gif)
		 [Comment](IStation.Comment.htm)
		 Свойство Comment возвращает
		 описание рабочей станции.


		 ![](../../Property_Image.gif)
		 [Domain](IStation.Domain.htm)
		 Свойство Domain возвращает
		 домен рабочей станции.


		 ![](../../Property_Image.gif)
		 [FullName](IStation.FullName.htm)
		 Свойство FullName возвращает
		 полное имя рабочей станции.


		 ![](../../Property_Image.gif)
		 [Included](IStation.Included.htm)
		 Свойство Included определяет,
		 будет ли использоваться рабочая станция.


		 ![](../../Property_Image.gif)
		 [Name](IStation.Name.htm)
		 Свойство Name возвращает
		 имя рабочей станции.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [IsAllowed](IStation.IsAllowed.htm)
		 Метод IsAllowed возвращает
		 признак доступа у пользователя с рабочей станцией.
