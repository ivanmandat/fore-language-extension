# IMetabaseObjectControl.Item


## Синтаксис


Item(Index: Integer): [IMetabaseObjectParamControl](../IMetabaseObjectParamControl/IMetabaseObjectParamControl.htm);


## Параметры


Index. Индекс параметра вложенного объекта.


## Описание


Свойство Item возвращает привязку
 параметра вложенного объекта к параметрам текущего объекта.
