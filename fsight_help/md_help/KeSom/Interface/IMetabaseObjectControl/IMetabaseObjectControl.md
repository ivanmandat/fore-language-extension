# IMetabaseObjectControl


Сборка: Metabase;


## Описание


Интерфейс IMetabaseObjectControl
 содержит свойства и методы для управления параметрами вложенного объекта.


## Иерархия наследования


IMetabaseObjectControl


## Комментарии


Для получения параметров вложенного объекта используйте свойства и методы
 интерфейса [IMetabaseObjectControlInfo](../IMetabaseObjectControlInfo/IMetabaseObjectControlInfo.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ControlledObject](IMetabaseObjectControl.ControlledObject.htm)


		 Свойство ControlledObject
		 возвращает описание вложенного объекта, для которого настраивается
		 управление параметрами.


		 ![](../../Property_Image.gif)
		 [Count](IMetabaseObjectControl.Count.htm)


		 Свойство Count возвращает
		 количество параметров вложенного объекта, для которых можно настроить
		 привязку.


		 ![](../../Property_Image.gif)
		 [IsEmpty](IMetabaseObjectControl.IsEmpty.htm)


		 Свойство IsEmpty возвращает
		 признак отсутствия настроек по управлению параметрами вложенного
		 объекта.


		 ![](../../Property_Image.gif)
		 [Item](IMetabaseObjectControl.Item.htm)


		 Свойство Item возвращает
		 привязку параметра вложенного объекта к параметрам текущего объекта.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindByKey](IMetabaseObjectControl.FindByKey.htm)


		 Метод FindByKey осуществляет
		 поиск и возвращает привязку параметра вложенного объекта к параметрам
		 текущего объекта.
