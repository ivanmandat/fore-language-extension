# IMbElementDependenciesDatabase.Exists


## Синтаксис


Exists(Where: [IMbElementDependencyTemplate](../IMbElementDependencyTemplate/IMbElementDependencyTemplate.htm)):
 Boolean;


## Параметры


Where - шаблон, в соответствии
 с которым осуществляется проверка.


## Описание


Метод Exists осуществляет проверку
 наличия зависимостей.


## Комментарии


Метод возвращает значение True,
 если существует информация о зависимостях, соответствующих указанному
 шаблону.
