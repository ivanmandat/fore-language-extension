# IMbElementDependenciesDatabase.Active


## Синтаксис


Active: Boolean;


## Описание


Свойство Active возвращает признак
 доступности сохранения зависимостей объектов от элементов справочников
 НСИ.


## Комментарии


Допустимые значения:


	- True.
	 Версия репозитория допускает хранение зависимостей объектов
	 от элементов справочников НСИ;


	- False.
	 Версия репозитория не допускает хранение зависимостей объектов
	 от элементов справочников НСИ.
