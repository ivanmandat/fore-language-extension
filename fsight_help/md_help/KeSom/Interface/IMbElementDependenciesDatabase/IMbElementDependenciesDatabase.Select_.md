# IMbElementDependenciesDatabase.Select_


## Синтаксис


Select_(Where: [IMbElementDependencyTemplate](../IMbElementDependencyTemplate/IMbElementDependencyTemplate.htm)): [IMbElementDependencies](../IMbElementDependencies/IMbElementDependencies.htm);


## Параметры


Where - шаблон, в соответствии с которым осуществляется отбор зависимостей.


## Описание


Метод Select_ возвращает параметры зависимостей, соответствующих указанному шаблону.


## Комментарии


Если зависимостей, соответствующих указанному шаблону, нет, то метод вернет пустую коллекцию.
