# IMetabaseCodeBlock.References


## Синтаксис


References: String;


## Описание


Свойство References определяет
 список подключаемых сборок, в которых хранятся необходимые ресурсы.


## Комментарии


В свойстве указываются наименования [системных сборок](Assembly.chm::/System_Assembly.htm)
 или идентификаторы пользовательских Fore-модулей/форм/сборок. Подключаемые
 сборки в списке разделяются символом «;», например: "Metabase;Report;MOD_REPORT".

Пример использования свойства приведён в описании метода [Execute](IMetabaseCodeBlock.Execute.htm).
