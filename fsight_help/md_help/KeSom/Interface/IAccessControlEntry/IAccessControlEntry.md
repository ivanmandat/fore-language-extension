# IAccessControlEntry


## Описание


Интерфейс IAccessControlEntry
 содержит свойства элемента контроля доступа.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AccessMask](IAccessControlEntry.AccessMask.htm)
		 Свойство AccessMask
		 определяет маску доступа к объекту.


		 ![](../../Property_Image.gif)
		 [Sid](IAccessControlEntry.Sid.htm)
		 Свойство Sid возвращает
		 идентификатор субъекта безопасности, к которому относится данный
		 элемент.


		 ![](../../Property_Image.gif)
		 [Type](IAccessControlEntry.Type.htm)
		 Свойство Type определяет
		 тип доступа к объекту.
