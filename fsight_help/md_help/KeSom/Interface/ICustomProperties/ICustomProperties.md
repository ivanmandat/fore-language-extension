# ICustomProperties


Сборка: Metabase;


## Описание


Интерфейс ICustomProperties
 содержит свойства и методы для работы с коллекцией пользовательских свойств
 обновления.


## Иерархия наследования


           ICustomProperties


## Комментарии


Для получения объекта данного типа используйте свойства [IMetabaseUpdate.CustomProperties](../IMetabaseUpdate/IMetabaseUpdate.CustomProperties.htm),
 [IMetabaseUpdateNode.CustomProperties](../IMetabaseUpdateNode/IMetabaseUpdateNode.CustomProperties.htm).
