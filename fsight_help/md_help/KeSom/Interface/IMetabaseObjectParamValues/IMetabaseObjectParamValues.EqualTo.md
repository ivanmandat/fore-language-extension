# IMetabaseObjectParamValues.EqualTo


## Синтаксис


EqualTo(Values: [IMetabaseObjectParamValues](IMetabaseObjectParamValues.htm)): Boolean;


## Параметры


Values - значения параметров, которые сравниваются с существующими значениями параметров данного объекта.


## Описание


Метод EqualTo осуществляет сравнение существующих значений параметров со значениями, передаваемыми посредством параметра Values.
