# IMetabaseObjectParamValues.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IMetabaseObjectParamValue](../IMetabaseObjectParamValue/IMetabaseObjectParamValue.htm);


## Параметры


Key — ключ параметра объекта репозитория.


## Описание


Метод FindByKey осуществляет поиск параметра по указанному ключу Key.


## Комментарии


В случае удачного поиска метод возвращает настройки параметра, иначе возвращает значение Null.
