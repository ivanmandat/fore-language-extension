# IMetabaseObjectParamValues.FindById


## Синтаксис


FindById(Id: String): [IMetabaseObjectParamValue](../IMetabaseObjectParamValue/IMetabaseObjectParamValue.htm);


## Параметры


Id - идентификатор параметра объекта репозитория.


## Описание


Метод FindById осуществляет поиск параметра по указанному идентификатору Id.


## Комментарии


В случае удачного поиска метод возвращает настройки параметра, иначе возвращает значение Null.
