# IMetabaseObjectParamValues


## Описание


Интерфейс IMetabaseObjectParamValues
 содержит свойства и методы коллекции значений параметров объекта репозитория.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMetabaseObjectParamValues.Count.htm)
		 Свойство Count возвращает
		 количество параметров объекта репозитория.


		 ![](../../Property_Image.gif)
		 [Item](IMetabaseObjectParamValues.Item.htm)
		 Свойство Item возвращает
		 значение параметра.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CreateCopy](IMetabaseObjectParamValues.CreateCopy.htm)
		 Метод CreateCopy осуществляет
		 создание копии значений параметров объекта.


		 ![](../../Sub_Image.gif)
		 [EqualTo](IMetabaseObjectParamValues.EqualTo.htm)
		 Метод EqualTo осуществляет
		 сравнение существующих значений параметров со значениями, которые
		 передаются в качестве параметра метода.


		 ![](../../Sub_Image.gif)
		 [FindById](IMetabaseObjectParamValues.FindById.htm)
		 Метод FindById осуществляет
		 поиск параметра по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IMetabaseObjectParamValues.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск параметра по ключу.
