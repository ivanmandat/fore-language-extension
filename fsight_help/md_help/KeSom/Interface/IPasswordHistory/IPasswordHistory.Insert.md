# IPasswordHistory.Insert


## Синтаксис


		Insert(PasswordHash: String);


## Параметры


PasswordHash.
 Пользовательский пароль, который был хеширован.


## Описание


Метод Insert осуществляет добавление
 пароля в начало истории.
