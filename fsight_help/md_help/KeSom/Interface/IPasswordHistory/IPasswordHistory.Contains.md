# IPasswordHistory.Contains


## Синтаксис


		Contains(PasswordHash: String): Boolean;


## Параметры


PasswordHash. Пользовательский
 пароль, который был хеширован.


## Описание


Метод Contains возвращает признак
 того, содержится ли пароль в истории.


## Комментарии


Допустимые значения:


	- True. Пароль содержится
	 в истории;


	- False. Пароль отсутствует
	 в истории.
