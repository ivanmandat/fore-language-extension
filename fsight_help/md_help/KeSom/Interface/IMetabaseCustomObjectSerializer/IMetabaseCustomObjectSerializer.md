# IMetabaseCustomObjectSerializer


## Описание


Интерфейс IMetabaseCustomObjectSerializer
 содержит базовые свойства для чтения и записи данных объекта пользовательского
 класса.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ElementDependencies](IMetabaseCustomObjectSerializer.ElementDependencies.htm)
		 Свойство ElementDependencies
		 определяет коллекцию зависимостей объекта пользовательского класса
		 от элементов справочников НСИ.


		 ![](../../Property_Image.gif)
		 [Object](IMetabaseCustomObjectSerializer.Object.htm)
		 Свойство Object возвращает
		 объект, с данными которого ведется работа.


		 ![](../../Property_Image.gif)
		 [References](IMetabaseCustomObjectSerializer.References.htm)
		 Свойство References
		 возвращает зависимости объекта.


		 ![](../../Property_Image.gif)
		 [Selections](IMetabaseCustomObjectSerializer.Selections.htm)
		 Свойство Selections
		 возвращает коллекцию отметок измерений, связанных с объектом пользовательского
		 класса.
