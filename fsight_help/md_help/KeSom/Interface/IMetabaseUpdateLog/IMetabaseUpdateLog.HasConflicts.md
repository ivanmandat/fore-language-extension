# IMetabaseUpdateLog.HasConflicts


## Синтаксис


HasConflicts: Boolean;


## Описание


Свойство HasConflicts возвращает
 признак наличия конфликтов при определении соответствия объектов в обновлении
 и репозитории.


## Комментарии


Свойство возвращает значение True,
 если в списке [записей](IMetabaseUpdateLog.Records.htm) об
 обновлении объектов имеются записи, в которых возник конфликт соответствия
 типа объекта в обновлении и репозитории.
