# IMetabaseUpdateLog.HasErrorSkipped


## Синтаксис


HasErrorSkipped: Boolean;


## Описание


Свойство HasErrorSkipped возвращает
 признак наличия пропусков ошибок в записях журнала.


## Комментарии


Свойство возвращает значение True,
 если в списке [записей](IMetabaseUpdateLog.Records.htm) имеются
 пропущенные ошибки обновления объектов.
