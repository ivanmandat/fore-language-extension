# IMetabaseUpdateLog.InProgress


## Синтаксис


InProgress: Boolean;


## Описание


Свойство InProgress возвращает
 признак незавершенности процесса применения обновления.


## Комментарии


Свойство возвращает значение True,
 если происходит установка обновления и False,
 если установка обновления завершена.
