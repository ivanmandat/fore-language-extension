# IMetabaseUpdateLog.HasErrors


## Синтаксис


HasErrors: Boolean;


## Описание


Свойство HasErrors возвращает
 признак наличия ошибок в записях журнала.


## Комментарии


Свойство возвращает значение True,
 если в списке [записей](IMetabaseUpdateLog.Records.htm) имеются
 ошибки обновления объектов.
