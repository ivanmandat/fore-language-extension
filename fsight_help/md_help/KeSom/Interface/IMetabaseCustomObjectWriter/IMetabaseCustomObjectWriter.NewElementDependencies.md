# IMetabaseCustomObjectWriter.NewElementDependencies


## Синтаксис


NewElementDependencies: [IMbElementDependencies](../IMbElementDependencies/IMbElementDependencies.htm);


## Описание


Метод NewElementDependencies создает новую коллекцию зависимостей объекта пользовательского класса от элементов справочников НСИ.


## Комментарии


При создании новой коллекции ключ текущего объекта пользовательского класса автоматически будет установлен в шаблоне коллекции [ObjectKey](../IMbElementDependencyTemplate/IMbElementDependencyTemplate.ObjectKey.htm).
