# IMetabaseCustomObjectWriter.WriteFromFile


## Синтаксис


WriteFromFile(FileName: String);


## Параметры


FileName. Полное имя файла, из которого будут получены
 данные для записи.


## Описание


Метод WriteFromFile осуществляет
 запись данных в объект из файла.
