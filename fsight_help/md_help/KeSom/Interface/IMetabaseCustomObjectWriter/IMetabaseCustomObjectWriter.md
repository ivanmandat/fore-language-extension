# IMetabaseCustomObjectWriter


## Описание


Интерфейс IMetabaseCustomObjectWriter
 содержит свойства и методы для записи данных в объект пользовательского
 класса.


## Свойства, унаследованные от [IMetabaseCustomObjectSerializer](../IMetabaseCustomObjectSerializer/IMetabaseCustomObjectSerializer.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ElementDependencies](../IMetabaseCustomObjectSerializer/IMetabaseCustomObjectSerializer.ElementDependencies.htm)
		 Свойство ElementDependencies
		 определяет коллекцию зависимостей объекта пользовательского класса
		 от элементов справочников НСИ.


		 ![](../../Property_Image.gif)
		 [Object](../IMetabaseCustomObjectSerializer/IMetabaseCustomObjectSerializer.Object.htm)
		 Свойство Object возвращает
		 объект, с данными которого ведется работа.


		 ![](../../Property_Image.gif)
		 [References](../IMetabaseCustomObjectSerializer/IMetabaseCustomObjectSerializer.References.htm)
		 Свойство References
		 возвращает зависимости объекта.


		 ![](../../Property_Image.gif)
		 [Selections](../IMetabaseCustomObjectSerializer/IMetabaseCustomObjectSerializer.Selections.htm)
		 Свойство Selections
		 возвращает коллекцию отметок измерений, связанных с объектом пользовательского
		 класса.

		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [NewElementDependencies](IMetabaseCustomObjectWriter.NewElementDependencies.htm)

		 создает новую коллекцию зависимостей объекта пользовательского
		 класса от элементов справочников НСИ.


		 ![](../../Sub_Image.gif)

		 ![](../../Sub_Image.gif)
		 [WriteFromFile](IMetabaseCustomObjectWriter.WriteFromFile.htm)
		 Метод WriteFromFile
		 осуществляет запись данных в объект из файла.


		 ![](../../Sub_Image.gif)
		 [WriteFromStream](IMetabaseCustomObjectWriter.WriteFromStream.htm)
		 Метод WriteFromStream
		 осуществляет запись данных в объект из потока.


		 ![](../../Sub_Image.gif)
		 [WriteFromXML](IMetabaseCustomObjectWriter.WriteFromXML.htm)
		 Метод WriteFromXML
		 осуществляет запись данных в объект из формата XML.
