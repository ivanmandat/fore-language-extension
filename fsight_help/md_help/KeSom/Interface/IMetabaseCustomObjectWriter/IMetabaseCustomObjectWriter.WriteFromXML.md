# IMetabaseCustomObjectWriter.WriteFromXML


## Синтаксис


WriteFromXML(Xml: Variant);


## Параметры


Xml. Объект, из которого будут
 получены данные для записи.


## Описание


Метод WriteFromXML осуществляет
 запись данных в объект из формата XML.
