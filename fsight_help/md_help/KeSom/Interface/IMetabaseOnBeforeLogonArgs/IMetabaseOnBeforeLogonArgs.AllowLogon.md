# IMetabaseOnBeforeLogonArgs.AllowLogon


## Синтаксис


		AllowLogon: Boolean;


## Описание


Свойство AllowLogon определяет,
 разрешено ли пользователю подключение к репозиторию.


## Комментарии


По умолчанию свойство имеет значение True,
 при этом пользователю разрешено подключение. Для запрета подключения установите
 свойству значение False.
