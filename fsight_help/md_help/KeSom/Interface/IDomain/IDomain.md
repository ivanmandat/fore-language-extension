# IDomain


## Описание


Интерфейс IDomain содержит свойства
 для работы с доменом.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Comment](IDomain.Comment.htm)
		 Свойство Comment возвращает
		 описание домена.


		 ![](../../Property_Image.gif)
		 [Name](IDomain.Name.htm)
		 Свойство Name возвращает
		 наименование домена.


		 ![](../../Property_Image.gif)
		 [Stations](IDomain.Stations.htm)
		 Свойство Stations определяет
		 рабочие станции, принадлежащие домену.
