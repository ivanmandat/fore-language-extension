# IMetabaseCustomClassStandardOperationsWeb


Сборка: Metabase;


## Описание


Интерфейс IMetabaseCustomClassStandardOperationsWeb
 предназначен для работы с коллекцией стандартных операций пользовательского
 класса, доступных в веб-приложении.


## Иерархия наследования


IMetabaseCustomClassStandardOperationsWeb


## Комментарии


Коллекцию стандартных операций, доступных в веб-приложении, можно получить,
 используя свойство [IMetabaseCustomClass.StandardOperationsWeb](../IMetabaseCustomClass/IMetabaseCustomClass.StandardOperationsWeb.htm).


## Свойства


		 Имя свойства
