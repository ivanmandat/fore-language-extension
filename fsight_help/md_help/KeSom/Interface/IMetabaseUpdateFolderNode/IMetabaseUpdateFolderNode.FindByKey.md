# IMetabaseUpdateFolderNode.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IMetabaseUpdateNode](../IMetabaseUpdateNode/IMetabaseUpdateNode.htm);


## Параметры


Key. Ключ объекта обновления.


## Описание


Метод FindByKey осуществляет поиск и возвращает объект обновления. Поиск осуществляется по ключу объекта обновления, который передается посредством параметра Key.
