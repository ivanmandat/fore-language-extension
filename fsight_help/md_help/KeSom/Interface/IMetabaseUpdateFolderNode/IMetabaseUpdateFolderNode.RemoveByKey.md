# IMetabaseUpdateFolderNode.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ объекта обновления.


## Описание


Метод RemoveByKey осуществляет удаление объекта обновления, ключ которого передается посредством параметра Key, и возвращает признак удачного удаления.
