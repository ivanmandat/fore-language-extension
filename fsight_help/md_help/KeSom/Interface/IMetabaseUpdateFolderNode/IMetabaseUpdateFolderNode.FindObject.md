# IMetabaseUpdateFolderNode.FindObject


## Синтаксис


FindObject(ObjKey: Integer): [IMetabaseUpdateObjectNode](../IMetabaseUpdateObjectNode/IMetabaseUpdateObjectNode.htm);


## Параметры


ObjKey. Ключ объекта репозитория.


## Описание


Метод FindObject осуществляет поиск и возвращает объект обновления. Поиск осуществляется по ключу объекта репозитория, который передается посредством параметра Objkey.
