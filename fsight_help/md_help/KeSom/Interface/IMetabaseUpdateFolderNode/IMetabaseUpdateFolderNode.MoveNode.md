# IMetabaseUpdateFolderNode.MoveNode


## Синтаксис


MoveNode (Index: Integer, Parent: [IMetabaseUpdateFolderNode](IMetabaseUpdateFolderNode.htm), InsertAt: Integer);


## Параметры


Index. Индекс объекта, который нужно переместить.


Parent. Родительская структура объекта обновления.


Insert. Позиция, на которую следует переместить объект обновления. При установке значения «0» для данного параметра объект будет перемещен в начало списка.


## Описание


Метод MoveNode перемещает объект в списке объектов обновления.
