# IMetabaseUpdateFolderNode.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс объекта обновления.


## Описание


Метод Remove осуществляет удаление объекта обновления, индекс которого передается посредством параметра Index, и возвращает признак удачного удаления.
