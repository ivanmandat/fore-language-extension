# IObjectCheckOutInfoUser


## Описание


Интерфейс IObjectCheckOutInfoUser
 содержит свойства для получения информации о блокировке объекта репозитория.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Comment](IObjectCheckOutInfoUser.Comment.htm)
		 Свойство Comment возвращает
		 примечание к записи о блокировании объекта.


		 ![](../../Property_Image.gif)
		 [MachineName](IObjectCheckOutInfoUser.MachineName.htm)
		 Свойство MachineName
		 возвращает наименование рабочей станции, с который был заблокирован
		 объект.


		 ![](../../Property_Image.gif)
		 [Stamp](IObjectCheckOutInfoUser.Stamp.htm)
		 Свойство Stamp возвращает
		 дату и время блокирования объекта.


		 ![](../../Property_Image.gif)
		 [UserName](IObjectCheckOutInfoUser.UserName.htm)
		 Свойство UserName возвращает
		 наименование пользователя, который заблокировал объект.
