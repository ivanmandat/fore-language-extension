# ISecurityDescriptor.Edit


## Синтаксис


		Edit;


## Описание


Метод Edit открывает на редактирование
 параметры безопасности объекта.


## Комментарии


После внесения изменений для сохранения используйте метод [Apply](ISecurityDescriptor.Apply.htm),
 для отмены изменений - метод [Revert](ISecurityDescriptor.Revert.htm).
