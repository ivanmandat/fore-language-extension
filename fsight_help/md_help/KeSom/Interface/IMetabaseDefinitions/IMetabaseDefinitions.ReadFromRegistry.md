# IMetabaseDefinitions.ReadFromRegistry


## Синтаксис


ReadFromRegistry;


## Описание


Метод ReadFromRegistry осуществляет
 чтение настроек подключения репозиториев из файла Metabases.xml и реестра.


Примечание.
 Данный метод может применяться при разработке приложений, использующих
 ресурсы платформы.


При работе
 в платформе список описаний репозиториев загружается автоматически (при
 загрузке файлов Studio.exe либо PP.exe). Повторный вызов метода в данном
 случае будет приводить к ошибке из-за дублирования идентификаторов описаний,
 содержащихся в реестре и загруженных при запуске платформы.
