# IMetabaseDefinitions


## Описание


Интерфейс IMetabaseDefinitions
 содержит свойства и методы коллекции описаний репозиториев, зарегистрированных
 на рабочей станции.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMetabaseDefinitions.Count.htm)
		 Свойство Count возвращает
		 количество описаний репозиториев, зарегистрированных на рабочей
		 станции.


		 ![](../../Property_Image.gif)
		 [Item](IMetabaseDefinitions.Item.htm)
		 Свойство Item возвращает
		 описание репозитория.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IMetabaseDefinitions.Add.htm)
		 Метод Add осуществляет
		 добавление нового описания репозитория.


		 ![](../../Sub_Image.gif)
		 [FindById](IMetabaseDefinitions.FindById.htm)
		 Метод FindById осуществляет
		 поиск описания репозитория по идентификатору.


		 ![](../../Sub_Image.gif)
		 [ReadFromFile](IMetabaseDefinitions.ReadFromFile.htm)
		 Метод ReadFromFile
		 осуществляет чтение описания репозиториев из файла.


		 ![](../../Sub_Image.gif)
		 [ReadFromRegistry](IMetabaseDefinitions.ReadFromRegistry.htm)
		 Метод ReadFromRegistry
		 осуществляет чтение настроек подключения репозиториев из файла
		 Metabases.xml и реестра.


		 ![](../../Sub_Image.gif)
		 [RemoveById](IMetabaseDefinitions.RemoveById.htm)
		 Метод RemoveById осуществляет
		 удаление описания репозитория по идентификатору.
