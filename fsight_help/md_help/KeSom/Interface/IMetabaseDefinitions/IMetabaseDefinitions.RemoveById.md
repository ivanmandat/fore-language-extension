# IMetabaseDefinitions.RemoveById


## Синтаксис


RemoveById(Id: String): Boolean;


## Параметры


Id. Идентификатор
 описания репозитория.


## Описание


Метод RemoveById осуществляет
 удаление описания репозитория, идентификатор которого передается посредством
 входного параметра.
