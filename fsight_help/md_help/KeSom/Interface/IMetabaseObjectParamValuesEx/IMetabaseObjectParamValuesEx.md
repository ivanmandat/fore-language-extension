# IMetabaseObjectParamValuesEx


Сборка: Metabase;


## Описание


Интерфейс IMetabaseObjectParamValuesEx
 предназначен для передачи дополнительного состояния, используемого при
 открытии регламентного отчета на просмотр.


## Иерархия наследования


           IMetabaseObjectParamValuesEx


## Комментарии


Для работы с коллекцией значений параметров регламентного отчета используйте
 свойства и методы интерфейса [IMetabaseObjectParamValues](../IMetabaseObjectParamValues/IMetabaseObjectParamValues.htm).
