# IMetabaseUpdateDataObjectNode.IsPartialUpdate


## Синтаксис


IsPartialUpdate: Boolean;


## Описание


Свойство IsPartialUpdate возвращает
 признак частичного обновления объекта.


## Комментарии


Свойство возвращает значение True,
 если для объекта определён список полей/атрибутов в свойстве [FieldsOnly](IMetabaseUpdateDataObjectNode.FieldsOnly.htm)
 или список ключей отдельных записей в свойстве [PrimaryKeysOnly](IMetabaseUpdateDataObjectNode.PrimaryKeysOnly.htm).
