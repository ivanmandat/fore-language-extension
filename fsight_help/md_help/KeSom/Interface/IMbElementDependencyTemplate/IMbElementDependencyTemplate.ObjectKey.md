# IMbElementDependencyTemplate.ObjectKey


## Синтаксис


ObjectKey: Integer;


## Описание


Свойство ObjectKey определяет ключ объекта, с зависимостями которого осуществляется работа.


## Комментарии


Для получения ключа объекта используйте свойство [IMetabaseObjectDescriptor.Key](../IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.Key.htm), либо метод [IMetabase.GetObjectKeyById](../IMetabase/IMetabase.GetObjectKeyById.htm).
