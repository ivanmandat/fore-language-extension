# IMbElementDependencyTemplate.DictionaryKey


## Синтаксис


DictionaryKey: Integer;


## Описание


Свойство DictionaryKey определяет ключ справочника НСИ, с зависимостями от элементов которого осуществляется работа.


## Комментарии


Для получения ключа справочника используйте свойство [IMetabaseObjectDescriptor.Key](../IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.Key.htm), либо метод [IMetabase.GetObjectKeyById](../IMetabase/IMetabase.GetObjectKeyById.htm).
