# IMbElementDependencyTemplate.ElementKeys


## Синтаксис


ElementKeys: Array;


## Описание


Свойство ElementKeys определяет массив ключей элементов справочника НСИ, с зависимостями от которых осуществляется работа.


## Комментарии


Данное свойство используется при извлечении/удалении зависимостей от нескольких элементов справочника НСИ.
