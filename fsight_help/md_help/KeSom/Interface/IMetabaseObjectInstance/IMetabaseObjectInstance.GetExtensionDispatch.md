# IMetabaseObjectInstance.GetExtensionDispatch


## Синтаксис


		GetExtensionDispatch: Object;


## Описание


Метод GetExtensionDispatch осуществляет
 преобразование к общей форме объекта, от которой в дальнейшем можно перейти
 к необходимому классу объектов.


## Комментарии


Данный метод предназначен для применения в языках программирования,
 в которых отсутствует возможность явного преобразования типов. Результатом
 работы метода будет необходимый тип объекта.


Использование данного метода в языке Fore
 неактуально. Подробнее о преобразовании типов смотрите в описании оператора
 [As](Fore.chm::/07_Operations/Operations_of_the_creation_object.htm#as).
