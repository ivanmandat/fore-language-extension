# IPrimaryOracleSPLD


## Описание


Интерфейс IPrimaryOracleSPLD
 содержит свойства и методы модуля безопасности, используемого при подключении
 к репозиторию, расположенному на сервере Oracle.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [OsAuthentPrefix](IPrimaryOracleSPLD.OsAuthentPrefix.htm)
		 Свойство OsAuthentPrefix
		 определяет значение префикса, используемого при создании доменных
		 пользователей в менеджере безопасности платформы.


		 ![](../../Property_Image.gif)
		 [Scheme](IPrimaryOracleSPLD.Scheme.htm)
		 Свойство Scheme определяет
		 наименование схемы на сервере Oracle.


		 ![](../../Property_Image.gif)
		 [Server](IPrimaryOracleSPLD.Server.htm)
		 Свойство Server определяет
		 наименование сервера Oracle, на котором расположен схема.
