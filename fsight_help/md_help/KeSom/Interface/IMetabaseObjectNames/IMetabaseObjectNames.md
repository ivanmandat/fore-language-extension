# IMetabaseObjectNames


Сборка: Metabase;


## Описание


Интерфейс IMetabaseObjectNames
 предназначен для работы с наименованиями объекта.


## Иерархия наследования


           IMetabaseObjectNames


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMetabaseObjectNames.Count.htm)
		 Свойство Count возвращает
		 количество наименований объекта.


		 ![](../../Property_Image.gif)
		 [Item](IMetabaseObjectNames.Item.htm)
		 Свойство Item определяет
		 параметры наименования объекта.


		 ![](../../Property_Image.gif)
		 [Name](IMetabaseObjectNames.Name.htm)
		 Свойство Name определяет
		 наименование объекта для заданного языка.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IMetabaseObjectNames.Add.htm)
		 Метод Add добавляет
		 наименования объекта для заданного языка.


		 ![](../../Sub_Image.gif)
		 [FindByLocale](IMetabaseObjectNames.FindByLocale.htm)
		 Метод FindByLocale
		 осуществляет поиск наименования по заданному языку среди всех
		 наименований объекта.


		 ![](../../Sub_Image.gif)
		 [Remove](IMetabaseObjectNames.Remove.htm)
		 Метод Remove удаляет
		 наименование по заданному языку.
