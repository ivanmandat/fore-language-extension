# IMetabaseCustomExtender.IsShared


## Синтаксис


IsShared: Boolean;


## Описание


Свойство IsShared определяет,
 доступны ли классы контейнера для использования.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Классы контейнера доступны для использования;


	- False. Классы
	 контейнера недоступны для использования.
