# IMetabaseCustomExtender.SmallImages


## Синтаксис


SmallImages: [IGxImageList](ModDrawing.chm::/Interface/IGxImageList/IGxImageList.htm);


## Описание


Свойство SmallImages определяет
 источник маленьких пиктограмм для объектов классов, входящих в контейнер.


## Комментарии


Для определения индекса маленькой пиктограммы в коллекции используйте
 свойство [IMetabaseCustomClass.SmallImageIndex](../IMetabaseCustomClass/IMetabaseCustomClass.SmallImageIndex.htm).
