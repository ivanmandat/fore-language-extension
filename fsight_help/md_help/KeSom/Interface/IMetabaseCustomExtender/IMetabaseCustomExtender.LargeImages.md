# IMetabaseCustomExtender.LargeImages


## Синтаксис


LargeImages: [IGxImageList](ModDrawing.chm::/Interface/IGxImageList/IGxImageList.htm);


## Описание


Свойство LargeImages определяет
 источник крупных пиктограмм для объектов классов, входящих в контейнер.


## Комментарии


Для определения индекса крупной пиктограммы в коллекции используйте
 свойство [IMetabaseCustomClass.LargeImageIndex](../IMetabaseCustomClass/IMetabaseCustomClass.LargeImageIndex.htm).
