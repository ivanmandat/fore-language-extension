# IMetabaseUpdateKeyMap.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемого элемента
 коллекции.


## Описание


Метод Remove удаляет элемент
 из коллекции по его индексу.


## Комментарии


Индексация элементов в коллекции сплошная и начинается с нуля.


Если удаление выполнено успешно, то метод вернёт значение True,
 иначе - False.
