# IMetabaseUpdateKeyMap.IsEmpty


## Синтаксис


IsEmpty: Boolean;


## Описание


Свойство IsEmpty возвращает
 признак того, пустая ли карта ключей.


## Комментарии


Допустимые значения:


	- True. Карта пустая и
	 не содержит ключей;


	- False. Карта содержит
	 ключи.
