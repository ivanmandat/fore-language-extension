# IMetabaseUpdateKeyMap.FindByOldKey


## Синтаксис


FindByOldKey(Key: Integer): [IMetabaseUpdateObjectRemapping](../IMetabaseUpdateObjectRemapping/IMetabaseUpdateObjectRemapping.htm);


## Параметры


Key. Ключ объекта в обновлении.


## Описание


Метод FindByOldKey выполняет
 поиск параметров сопоставления объекта по его ключу в обновлении.


## Комментарии


Если параметры не найдены, то метод вернёт значение Null.
