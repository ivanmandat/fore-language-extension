# IMetabaseUpdateKeyMap.FindByNewKey


## Синтаксис


FindByNewKey(Key: Integer): [IMetabaseUpdateObjectRemapping](../IMetabaseUpdateObjectRemapping/IMetabaseUpdateObjectRemapping.htm);


## Параметры


Key. Ключ объекта в репозитории-приёмнике.


## Описание


Метод FindByNewKey выполняет
 поиск параметров сопоставления объекта по его ключу в репозитории-приёмнике.


## Комментарии


Если параметры не найдены, то метод вернёт значение Null.
