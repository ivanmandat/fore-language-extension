# IApplicationRole


## Описание


Интерфейс IApplicationRole предназначен
 для работы с параметрами роли приложения.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Active](IApplicationRole.Active.htm)
		 Свойство Active
		 определяет использование роли приложения.


		 ![](../../Property_Image.gif)
		 [Name](IApplicationRole.Name.htm)
		 Свойство Name определяет
		 наименование роли.


		 ![](../../Property_Image.gif)
		 [Password](IApplicationRole.Password.htm)
		 Свойство Password определяет
		 пароль роли.


		 ![](../../Property_Image.gif)
		 [PasswordSalt](IApplicationRole.PasswordSalt.htm)
		 Свойство PasswordSalt
		 определяет начальное значение для создания производного пароля.


		 ![](../../Property_Image.gif)
		 [Privilegies](IApplicationRole.Privilegies.htm)
		 Свойство Privilegies
		 возвращает привилегию роли.


		 ![](../../Property_Image.gif)
		 [UseDerivedPassword](IApplicationRole.UseDerivedPassword.htm)
		 Свойство UseDerivedPassword
		 определяет использование производного пароля при создании подключения.
