# IApplicationRole.UseDerivedPassword


## Синтаксис


UseDerivedPassword: Boolean;


## Описание


Свойство UseDerivedPassword определяет использование производного пароля при создании подключения.


## Комментарии


При значении True пароль будет шифроваться, при значении False – нет. По умолчанию свойство имеет значение True.
