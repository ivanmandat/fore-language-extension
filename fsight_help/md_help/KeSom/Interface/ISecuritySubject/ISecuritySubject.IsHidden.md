# ISecuritySubject.IsHidden


## Синтаксис


IsHidden: Boolean;


## Описание


Свойство IsHidden возвращает
 признак скрытого субъекта безопасности.


## Комментарии


Свойство актуально для доменных субъектов безопасности.


Свойство возвращает True, если
 субъект безопасности имеет права на какие-либо объекты репозитория, но
 не включен в список субъектов в менеджере безопасности платформы.
