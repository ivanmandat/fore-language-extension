# ISecuritySubject.IsNT


## Синтаксис


IsNT: Boolean;


## Описание


Свойство IsNT возвращает признак
 доменного субъекта безопасности.


## Комментарии


Допустимые значения:


	- True. Субъект безопасности
	 является доменным и добавлен в менеджере безопасности репозитория;


	- False. Субъект безопасности
	 создан в менеджере безопасности репозитория.
