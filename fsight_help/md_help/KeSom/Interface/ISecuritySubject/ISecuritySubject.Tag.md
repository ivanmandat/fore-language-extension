# ISecuritySubject.Tag


## Синтаксис


Tag: Variant;


## Описание


Свойство Tag предназначено для хранения дополнительной системной информации по субъектам безопасности.


## Комментарии


В свойстве Tag хранится пользовательская строка XML с дополнительными настройками, возможно сохранение свойства Tag в файл обновления (*.pefx).
