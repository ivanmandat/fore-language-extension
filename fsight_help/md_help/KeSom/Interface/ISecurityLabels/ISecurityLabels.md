# ISecurityLabels


## Описание


Интерфейс ISecurityLabels содержит
 свойства параметров меток безопасности.


## Комментарии


Метки безопасности устанавливаются для табличных справочников, справочников
 НСИ и составных справочников НСИ.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BitCount](ISecurityLabels.BitCount.htm)
		 Свойство BitCount определяет
		 количество субъектов безопасности, для которых могут быть сопоставлены
		 биты в маске доступа.


		 ![](../../Property_Image.gif)
		 [Mapping](ISecurityLabels.Mapping.htm)
		 Свойство Mapping определяет
		 карту соответствия битов маски субъектам безопасности.


		 ![](../../Property_Image.gif)
		 [Value](ISecurityLabels.Value.htm)
		 Свойство Value возвращает
		 значение маски для субъекта безопасности в десятичном виде. Используется
		 при сопоставлении битов маски не более чем 32 субъектам безопасности.


		 ![](../../Property_Image.gif)
		 [ValueStr](ISecurityLabels.ValueStr.htm)
		 Свойство ValueStr возвращает
		 значение маски для субъекта безопасности в строковом виде. Используется
		 при сопоставлении битов маски более чем 32 субъектам безопасности.
