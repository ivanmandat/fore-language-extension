# IMetabaseMandatoryAccess


## Описание


Интерфейс IMetabaseMandatoryAccess
 содержит свойства и методы мандатного доступа.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Category](IMetabaseMandatoryAccess.Category.htm)
		 Свойство Category возвращает
		 параметры указанной категории безопасности.


		 ![](../../Property_Image.gif)
		 [CategoryCount](IMetabaseMandatoryAccess.CategoryCount.htm)
		 Свойство CategoryCount
		 возвращает количество категорий безопасности.


		 ![](../../Property_Image.gif)
		 [IsSimple](IMetabaseMandatoryAccess.IsSimple.htm)
		 Свойство IsSimple определяет
		 признак использования метода разграничения доступа по уровням.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddCategory](IMetabaseMandatoryAccess.AddCategory.htm)
		 Метод AddCategory создает
		 категорию безопасности.


		 ![](../../Sub_Image.gif)
		 [RemoveCategory](IMetabaseMandatoryAccess.RemoveCategory.htm)
		 Метод RemoveCategory
		 осуществляет удаление категории безопасности по индексу.
