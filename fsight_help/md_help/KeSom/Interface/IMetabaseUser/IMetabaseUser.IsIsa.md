# IMetabaseUser.IsIsa


## Синтаксис


IsIsa: Boolean;


## Описание


Свойство IsIsa возвращает признак соответствия пользователя администратору информационной безопасности репозитория.


## Комментарии


Свойство возвращает значение True, если пользователь является администратором информационной безопасности репозитория, и False во всех других случаях. Актуально, если в платформе включено разделение ролей между администратором информационной безопасности и прикладным администратором.
