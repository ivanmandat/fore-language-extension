# IMetabaseUser.MustChangePasswordAtNextLogon


## Синтаксис


		MustChangePasswordAtNextLogon: Boolean;


## Описание


Свойство MustChangePasswordAtNextLogon
 определяет установку требования на смену пароля при следующем входе в
 систему для данного пользователя.
