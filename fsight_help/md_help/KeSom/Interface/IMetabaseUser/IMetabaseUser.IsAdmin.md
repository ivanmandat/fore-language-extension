# IMetabaseUser.IsAdmin


## Синтаксис


IsAdmin: Boolean;


## Описание


Свойство IsAdmin возвращает признак соответствия пользователя прикладному администратору репозитория.


## Комментарии


Свойство возвращает значение True, если пользователь является прикладным администратором платформы, и False во всех других случаях.
