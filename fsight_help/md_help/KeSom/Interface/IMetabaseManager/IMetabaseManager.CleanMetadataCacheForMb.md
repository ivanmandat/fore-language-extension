# IMetabaseManager.CleanMetadataCacheForMb


## Синтаксис


CleanMetadataCacheForMb(Metabase: [IMetabaseDefinition](../IMetabaseDefinition/IMetabaseDefinition.htm));


## Параметры


Metabase. Описание репозитория,
 в котором необходимо сбросить кеш метаданных.


## Описание


Метод CleanMetadataCacheForMb
 сбрасывает кеш метаданных репозитория с указанным описанием.
