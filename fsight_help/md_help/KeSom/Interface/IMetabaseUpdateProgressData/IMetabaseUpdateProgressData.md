# IMetabaseUpdateProgressData


## Описание


Интерфейс IMetabaseUpdateProgressData
 содержит свойства для получения информации о процессе обновления.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Current](IMetabaseUpdateProgressData.Current.htm)
		 Свойство Current определяет
		 индекс объекта, находящегося на стадии обновления.


		 ![](../../Property_Image.gif)
		 [Error](IMetabaseUpdateProgressData.Error.htm)
		 Свойство Error определяет
		 информацию об ошибке, которая может произойти во время процесса
		 обновления.


		 ![](../../Property_Image.gif)
		 [Node](IMetabaseUpdateProgressData.Node.htm)
		 Свойство Node определяет
		 объект обновления, в соответствии с которым обновляется объект
		 репозитория.


		 ![](../../Property_Image.gif)
		 [Object](IMetabaseUpdateProgressData.Object.htm)
		 Свойство Object определяет
		 объект репозитория, который будет обновлен.


		 ![](../../Property_Image.gif)
		 [Stage](IMetabaseUpdateProgressData.Stage.htm)
		 Свойство Stage возвращает
		 стадию обновления объекта.


		 ![](../../Property_Image.gif)
		 [Total](IMetabaseUpdateProgressData.Total.htm)
		 Свойство Total возвращает
		 общее количество объектов, для которых происходит обновление.
