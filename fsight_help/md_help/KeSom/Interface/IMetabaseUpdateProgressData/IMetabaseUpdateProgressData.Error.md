# IMetabaseUpdateProgressData.Error


## Синтаксис


Error: [IException](ForeSys.chm::/Interface/IException/IException.htm);


## Описание


Свойство Error определяет информацию об ошибке, которая может произойти во время процесса обновления.


Примечание. Данное свойство доступно только при синхронизации объектов обновления, во время возникновения ошибок. Для обработки ошибок используйте событие [OnError](../IMetabaseUpdateProgress/IMetabaseUpdateProgress.OnError.htm).
