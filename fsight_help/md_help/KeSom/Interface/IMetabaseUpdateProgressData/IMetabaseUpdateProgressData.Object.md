# IMetabaseUpdateProgressData.Object


## Синтаксис


Object: [IMetabaseObject](../IMetabaseObject/IMetabaseObject.htm);


## Описание


Свойство Object определяет объект репозитория, который будет обновлен.


Примечание. Данное свойство доступно после подготовки объектов к обновлению.
