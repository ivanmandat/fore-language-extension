# ILogonSession.OnLogon


## Синтаксис


OnLogon(Data: [ILogonParams](../ILogonParams/ILogonParams.htm));


## Параметры


Data - параметры, значения которых будут выставлены после проверки возможности подключения.


## Описание


Метод OnLogon осуществляет проверку возможности подключения и устанавливает соответствующие значения параметров.
