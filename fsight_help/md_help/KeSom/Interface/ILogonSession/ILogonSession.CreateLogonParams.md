# ILogonSession.CreateLogonParams


## Синтаксис


CreateLogonParams: [ILogonParams](../ILogonParams/ILogonParams.htm);


## Описание


Метод CreateLogonParams создает параметры, используемые при проверке возможности подключения.


## Комментарии


Проверка подключения осуществляется с помощью метода [OnLogon](ILogonSession.OnLogon.htm). После проверки подключения будут установлены соответствующие значения параметров.
