# ILogonSession


## Описание


Интерфейс ILogonSession содержит
 свойства и методы текущей сессии.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Credentials](ILogonSession.Credentials.htm)
		 Свойство Credentials
		 определяет учетные данные, установленные в рамках текущей сессии,
		 которые будут использоваться при работе с объектами.


		 ![](../../Property_Image.gif)
		 [Groups](ILogonSession.Groups.htm)
		 Свойство Groups возвращает
		 группы пользователей, в которые входит текущий пользователь репозитория.


		 ![](../../Property_Image.gif)
		 [Key](ILogonSession.Key.htm)
		 Свойство Key возвращает
		 ключ текущей сессии с репозиторием.


		 ![](../../Property_Image.gif)
		 [LogonCredentials](ILogonSession.LogonCredentials.htm)
		 Свойство LogonCredentials
		 возвращает учетные данные, использованные при входе.


		 ![](../../Property_Image.gif)
		 [LogonData](ILogonSession.LogonData.htm)
		 Свойство LogonData
		 возвращает параметры модуля безопасности, использованные при входе
		 в репозиторий.


		 ![](../../Property_Image.gif)
		 [PrimaryConnection](ILogonSession.PrimaryConnection.htm)
		 Свойство PrimaryConnection
		 возвращает текущее соединение с репозиторием.


		 ![](../../Property_Image.gif)
		 [User](ILogonSession.User.htm)
		 Свойство User возвращает
		 данные о текущем пользователе репозитория.


		 ![](../../Property_Image.gif)
		 [UserDescription](ILogonSession.UserDescription.htm)
		 Свойство UserDescription
		 возвращает полное имя пользователя для занесения в протокол доступа.


		 ![](../../Property_Image.gif)
		 [UserIP](ILogonSession.UserIP.htm)
		 Свойство UserIP возвращает
		 IP-адрес рабочей станции для занесения в протокол доступа.


		 ![](../../Property_Image.gif)
		 [UserOS](ILogonSession.UserOS.htm)
		 Свойство UserOS возвращает
		 имя пользователя ОС для занесения в протокол доступа.


		 ![](../../Property_Image.gif)
		 [UserStation](ILogonSession.UserStation.htm)
		 Свойство UserStation
		 возвращает название рабочей станции пользователя, используемое
		 для занесения в протокол доступа.


		 ![](../../Property_Image.gif)
		 [UserToken](ILogonSession.UserToken.htm)
		 Свойство UserToken
		 возвращает параметры мандатного доступа текущего пользователя.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CreateLogonParams](ILogonSession.CreateLogonParams.htm)
		 Метод CreateLogonParams
		 создает параметры, используемые при проверке возможности подключения.


		 ![](../../Sub_Image.gif)
		 [OnLogon](ILogonSession.OnLogon.htm)
		 Метод OnLogon осуществляет
		 проверку возможности подключения и устанавливает соответствующие
		 значения параметров.
