# ILogonSession.UserDescription


## Синтаксис


UserDescription: String;


## Описание


Свойство UserDescription возвращает полное имя пользователя для занесения в протокол доступа.


## Комментарии


Полное имя пользователя используется в маркировке документов при печати или экспорте, если используется мандатный доступ.
