# IUpdateCallback


Сборка: Metabase;


## Описание


Интерфейс IUpdateCallback предоставляет
 методы, предназначенные для обработки событий обновлений.


## Иерархия наследования


IUpdateCallback


## Комментарии


Методы интерфейса должны быть переопределены в пользовательском классе.
 Экземпляр класса для обработки событий обновлений задаётся в свойстве
 [IMetabase.UpdateHandler](../IMetabase/IMetabase.UpdateHandler.htm).
