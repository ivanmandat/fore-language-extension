# IMetabaseSecurityExporter.FileAppend


## Синтаксис


FileAppend: Boolean;


## Описание


Свойство FileAppend определяет, будет ли дополнен файл, в который производится экспорт.


## Комментарии


По умолчанию свойство имеет значение False и файл при экспорте перезаписывается. Если необходимо дописать файл, то следует свойство установить в True. Путь к файлу определяет свойство [FileName](IMetabaseSecurityExporter.FileName.htm).
