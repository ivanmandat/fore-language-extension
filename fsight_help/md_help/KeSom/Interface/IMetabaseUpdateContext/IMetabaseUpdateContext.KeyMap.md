# IMetabaseUpdateContext.KeyMap


## Синтаксис


KeyMap: [IMetabaseUpdateKeyMap](../IMetabaseUpdateKeyMap/IMetabaseUpdateKeyMap.htm);


## Описание


Свойство KeyMap возвращает карту
 ключей, используемых для повторного сопоставления объектов обновления.


## Комментарии


Карта позволяет получить ключи и идентификаторы объектов в репозитории-приёмнике,
 которые будут использоваться для обновления объектов.
