# IMetabasePasswordPolicy.WarningMaxAge


## Синтаксис


WarningMaxAge: Integer;


## Описание


Свойство WarningMaxAge определяет количество дней, за которое пользователь будет уведомлен о необходимости смены пароля.


## Комментарии


Для установки задайте максимальное количество дней до смены пароля в свойстве [IMetabasePasswordPolicy.MaxAge](IMetabasePasswordPolicy.MaxAge.htm).
