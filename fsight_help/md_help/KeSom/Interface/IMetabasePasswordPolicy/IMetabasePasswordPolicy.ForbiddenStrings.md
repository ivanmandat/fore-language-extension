# IMetabasePasswordPolicy.ForbiddenStrings


## Синтаксис


		ForbiddenStrings: [IStringList](ModCollections.chm::/Interface/IStringList/IStringList.htm);


## Описание


Свойство ForbiddenStrings возвращает
 список, содержащий запрещенные строки, наличие которых будет проверяться
 в пароле.


## Комментарии


Актуально, если свойству [CheckForbiddenStrings](IMetabasePasswordPolicy.CheckForbiddenStrings.htm)
 установлено значение True.
