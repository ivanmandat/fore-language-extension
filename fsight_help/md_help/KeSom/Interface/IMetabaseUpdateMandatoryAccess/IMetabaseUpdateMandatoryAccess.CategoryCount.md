# IMetabaseUpdateMandatoryAccess.CategoryCount


## Синтаксис


CategoryCount: Integer;


## Описание


Свойство CategoryCount возвращает количество категорий.


## Комментарии


При выключенном мандатном доступе возвращает 0.
