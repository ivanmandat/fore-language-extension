# IVcsParams


Сборка: Metabase;


## Описание


Интерфейс IVcsParams содержит
 свойства для определения настроек подключения репозитория к системе управления
 версиями.


## Иерархия наследования


IVcsParams


## Комментарии


Данные настройки хранятся на сервере СУБД, где создан репозиторий.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AuxPathLabel](IVcsParams.AuxPathLabel.htm)
		 Свойство AuxPathLabel
		 определяет вспомогательную метку, в которой система управлениями
		 версиями хранит путь, используемый для подключения.


		 ![](../../Property_Image.gif)
		 [AuxProjPath](IVcsParams.AuxProjPath.htm)
		 Свойство AuxProjPath
		 определяет проект на сервере TFS, в рамках которого осуществляется
		 контроль версий объектов.


		 ![](../../Property_Image.gif)
		 [IsShared](IVcsParams.IsShared.htm)
		 Свойство IsShared определяет
		 признак подключения текущего репозитория к системе управления
		 версиями.


		 ![](../../Property_Image.gif)
		 [ProjName](IVcsParams.ProjName.htm)
		 Свойство ProjName определяет
		 путь к папке в проекте, в рамках которой будут храниться файлы
		 объектов.


		 ![](../../Property_Image.gif)
		 [SccName](IVcsParams.SccName.htm)
		 Свойство SccName определяет
		 наименование системы управления версиями.
