# IVcsParams.IsShared


## Синтаксис


IsShared: Boolean;


## Описание


Свойство IsShared определяет признак подключения текущего репозитория к системе управления версиями.


## Комментарии


Если свойству установлено значение True, то репозиторий подключен к системе управления версиями, иначе нет.


Данный признак устанавливается для репозитория, хранится на сервере СУБД и проверяется при работе с репозиторием на разных ПК. Если репозиторий подключен к системе управления версиями, то для использования возможностей работы с версиями объектов на конкретном ПК в описании репозитория должны быть установлены свойства [AuxProjPath](../IMetabaseDefinition/IMetabaseDefinition.AuxProjPath.htm) и [VcsLocalPath](../IMetabaseDefinition/IMetabaseDefinition.VcsLocalPath.htm).
