# IMetabaseObjectControlInfo


Сборка: Metabase;


## Описание


Интерфейс IMetabaseObjectControlInfo
 содержит свойства и методы для работы с коллекцией вложенных объектов.


## Иерархия наследования


IMetabaseObjectControlInfo


## Комментарии


Коллекцию вложенных объектов можно получить, используя следующие свойства:


	- [IEaxDataSources.ControlInfo](KeExpress.chm::/Interface/IEaxDataSources/IEaxDataSources.ControlInfo.htm);


	- [IMetabaseObjectParams.ControlInfo](../IMetabaseObjectParams/IMetabaseObjectParams.ControlInfo.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMetabaseObjectControlInfo.Count.htm)


		 Свойство Count возвращает
		 количество вложенных объектов, параметрами которых можно управлять.


		 ![](../../Property_Image.gif)
		 [Item](IMetabaseObjectControlInfo.Item.htm)


		 Свойство Item возвращает
		 параметры вложенного объекта с указанным индексом.


## Методы


		 Имя метода
		 Краткое описание

		 [FindByKey](IMetabaseObjectControlInfo.FindByKey.htm)

		 ![](../../Sub_Image.gif)
