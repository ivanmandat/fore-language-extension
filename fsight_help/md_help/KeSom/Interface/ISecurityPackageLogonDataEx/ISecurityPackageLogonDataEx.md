# ISecurityPackageLogonDataEx


Сборка: Metabase;


## Описание


Интерфейс ISecurityPackageLogonDataEx
 возвращает дополнительные свойства параметров модуля безопасности репозитория.


## Иерархия наследования


ISecurityPackageLogonDataEx


## Комментарии


К интерфейсу ISecurityPackageLogonDataEx
 могут быть приведены любые свойства, которые возвращает интерфейс [ISecurityPackageLogonData](../ISecurityPackageLogonData/ISecurityPackageLogonData.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [IsLocal](ISecurityPackageLogonDataEx.IsLocal.htm)


		 Свойство IsLocal возвращает
		 признак возможности использования параметра в глобальных переменных.


		 ![](../../Property_Image.gif)
		 [IsRequired](ISecurityPackageLogonDataEx.IsRequired.htm)


		 Свойство IsRequired
		 возвращает признак обязательного параметра, отображающегося в
		 диалоге авторизации в базе данных.


		 ![](../../Property_Image.gif)
		 [ParamCountEx](ISecurityPackageLogonDataEx.ParamCountEx.htm)


		 Свойство ParamCountEx
		 возвращает количество параметров модуля безопасности вместе с
		 дополнительными параметрами.


		 ![](../../Property_Image.gif)
		 [ParamDescription](ISecurityPackageLogonDataEx.ParamDescription.htm)


		 Свойство ParamDescription
		 возвращает описание параметра модуля безопасности.


		 ![](../../Property_Image.gif)
		 [ParamIndex](ISecurityPackageLogonDataEx.ParamIndex.htm)


		 Свойство ParamIndex
		 возвращает индекс параметра модуля безопасности.


		 ![](../../Property_Image.gif)
		 [ServiceLogon](ISecurityPackageLogonDataEx.ServiceLogon.htm)


		 Свойство ServiceLogon
		 определяет признак подключения через веб-сервис платформы без
		 создания соединения с репозиторием.


		 ![](../../Property_Image.gif)
		 [ServiceLogonData](ISecurityPackageLogonDataEx.ServiceLogonData.htm)


		 Свойство ServiceLogonData
		 возвращает дополнительные параметры модуля безопасности, содержащего
		 настройки для подключения к СУБД через веб-сервис платформы.
