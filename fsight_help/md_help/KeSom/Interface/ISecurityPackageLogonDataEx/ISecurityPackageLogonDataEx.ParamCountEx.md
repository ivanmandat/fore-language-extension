# ISecurityPackageLogonDataEx.ParamCountEx


## Синтаксис


ParamCountEx: Integer;


## Описание


Свойство ParamCountEx возвращает количество параметров модуля безопасности вместе с дополнительными параметрами.


## Комментарии


Дополнительными параметрами модуля безопасности могут являться глобальные переменные.
