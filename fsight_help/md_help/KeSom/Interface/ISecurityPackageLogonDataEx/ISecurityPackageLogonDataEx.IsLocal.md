# ISecurityPackageLogonDataEx.IsLocal


## Синтаксис


IsLocal(Index: Integer): Boolean;


## Параметры


Index. Индекс параметра модуля безопасности.


## Описание


Свойство IsLocal возвращает признак возможности использования параметра в глобальных переменных.


## Комментарии


Если свойство возвращает True, то параметр можно использовать в качестве глобальной переменной.


Если свойство возвращает False, то отсутствует возможность использовать параметр в качестве глобальной переменной.
