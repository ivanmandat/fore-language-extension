# ISecurityPackageLogonDataEx.IsRequired


## Синтаксис


IsRequired(Index: Integer): Boolean;


## Параметры


Index. Индекс параметра модуля безопасности.


## Описание


Свойство IsRequired возвращает
 признак обязательного параметра, отображающегося в диалоге авторизации
 в базе данных.
