# IMetabaseClassesSecurityDescriptors.Count


## Синтаксис


		Count: Integer;


## Описание


Свойство Count возвращает количество
 дескрипторов безопасности в коллекции.


## Комментарии


Коллекция дескрипторов безопасности определяется набором дескрипторов,
 которые заданы для [типов
 объектов](Admin.chm::/04_SecurityPolicy/Admin_Object_Classes.htm).


Свойство доступно только для чтения.
