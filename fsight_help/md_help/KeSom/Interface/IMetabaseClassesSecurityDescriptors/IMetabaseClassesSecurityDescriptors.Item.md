# IMetabaseClassesSecurityDescriptors.Item


## Синтаксис


		Item(Index: Integer): [IMetabaseClassSecurityDescriptor](../IMetabaseClassSecurityDescriptor/IMetabaseClassSecurityDescriptor.htm);


## Параметры


Index. Индекс дескриптора безопасности
 в коллекции.


## Описание


Свойство Item возвращает параметры
 дескриптора безопасности по индексу.


## Комментарии


Коллекция дескрипторов безопасности определяется набором дескрипторов,
 которые заданы для [типов
 объектов](Admin.chm::/04_SecurityPolicy/Admin_Object_Classes.htm).


Свойство доступно только для чтения.
