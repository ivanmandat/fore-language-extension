# IMetabaseClassesSecurityDescriptors.RemoveByClassId


## Синтаксис


		RemoveByClassId(ClassId: [MetabaseObjectClass](../../Enums/MetabaseObjectClass.htm)):
		 Boolean;


## Параметры


ClassId. Тип объектов.


## Описание


Метод RemoveByClassId удаляет
 дескриптор безопасности из коллекции по типу объектов.


## Комментарии


Коллекция дескрипторов безопасности определяется набором дескрипторов,
 которые заданы для [типов
 объектов](Admin.chm::/04_SecurityPolicy/Admin_Object_Classes.htm).
