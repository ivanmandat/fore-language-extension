# IMetabaseObjectSecurity


## Описание


Интерфейс IMetabaseObjectSecurity предназначен для работы с политикой доступа к объекту.


## Свойства


 Имя свойства
 Краткое описание


 ![](../../Property_Image.gif)
 [AccessSecured](IMetabaseObjectSecurity.AccessSecured.htm)
 Свойство AccessSecured возвращает признак использования прав доступа для элементов справочника.

 Краткое описание
