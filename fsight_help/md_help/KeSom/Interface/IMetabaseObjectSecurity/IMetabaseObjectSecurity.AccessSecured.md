# IMetabaseObjectSecurity.AccessSecured


## Синтаксис


AccessSecured: Boolean;


## Описание


Свойство AccessSecured возвращает признак использования прав доступа для элементов справочника НСИ.


## Комментарии


Возвращает True, в случае если элементы справочника НСИ имеют права доступа, в противном случае - False. Значение данного свойства будет соответствовать значению свойства [ISecurityDescriptor.AccessSecured](../ISecurityDescriptor/ISecurityDescriptor.AccessSecured.htm).
