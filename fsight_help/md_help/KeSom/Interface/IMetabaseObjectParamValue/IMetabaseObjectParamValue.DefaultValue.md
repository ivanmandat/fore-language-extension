# IMetabaseObjectParamValue.DefaultValue


## Синтаксис


DefaultValue: Variant;


## Описание


Свойство DefaultValue возвращает
 значение для параметра по умолчанию.


## Комментарии


Данное свойство возвращает значение, установленное в свойстве [DefaultValue](../IMetabaseObjectParam/IMetabaseObjectParam.DefaultValue.htm)
 при настройке параметра.
