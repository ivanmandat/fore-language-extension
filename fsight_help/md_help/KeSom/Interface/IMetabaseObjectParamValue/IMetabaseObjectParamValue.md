# IMetabaseObjectParamValue


## Описание


Интерфейс IMetabaseObjectParamValue
 содержит свойства значения параметра объекта репозитория.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Binding](IMetabaseObjectParamValue.Binding.htm)
		 Свойство Binding возвращает
		 настройки отображения редактора значения параметра объекта.


		 ![](../../Property_Image.gif)
		 [DataType](IMetabaseObjectParamValue.DataType.htm)
		 Свойство DataType возвращает
		 тип данных, которые можно указывать в качестве значения параметра.


		 ![](../../Property_Image.gif)
		 [DefaultValue](IMetabaseObjectParamValue.DefaultValue.htm)
		 Свойство DefaultValue
		 возвращает значение для параметра по умолчанию.


		 ![](../../Property_Image.gif)
		 [IsArray](IMetabaseObjectParamValue.IsArray.htm)
		 Свойство IsArray возвращает
		 True, если параметр указанного
		 объекта репозитория принимает множество значений.


		 ![](../../Property_Image.gif)
		 [Value](IMetabaseObjectParamValue.Value.htm)
		 Свойство Value определяет
		 значение параметра.
