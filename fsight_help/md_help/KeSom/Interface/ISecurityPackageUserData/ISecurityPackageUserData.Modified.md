# ISecurityPackageUserData.Modified


## Синтаксис


Modified: Boolean;


## Описание


Свойство Modified определяет
 признак наличия изменений в параметрах безопасности пользователя.


## Комментарии


Свойство возвращает значение True,
 если были изменены параметры безопасности пользователя.
