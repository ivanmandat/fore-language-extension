# IMetabaseCustomObjectSelSets.KeyByIndex


## Синтаксис


KeyByIndex(Index: Integer): String;


## Параметры


Index. Индекс элемента в коллекции.


## Описание


Метод KeyByIndex возвращает
 ключ элемента по его индексу в коллекции.


## Комментарии


Значение параметра Index указывается в диапазоне [0, [Count](IMetabaseCustomObjectSelSets.Count.htm)-1].
