# IMetabaseCustomObjectSelSets.RemoveByIndex


## Синтаксис


RemoveByIndex(Index: Integer): Boolean;


## Параметры


Index. Индекс элемента в коллекции.


## Описание


Метод RemoveByIndex удаляет
 элемент из коллекции по индексу и возвращает признак удачного удаления.


## Комментарии


Метод возвращает значение True,
 если удаление элемента прошло удачно.
