# IMetabaseCustomObjectSelSets.IndexByKey


## Синтаксис


IndexByKey(Key: String): Integer;


## Параметры


Key. Ключ
 элемента в коллекции.


## Описание


Метод IndexByKey возвращает
 индекс элемента с отметкой измерений в коллекции по его ключу.


## Комментарии


Метод возвращает значение «-1», если элемент с указанным ключом отсутствует
 в коллекции.
