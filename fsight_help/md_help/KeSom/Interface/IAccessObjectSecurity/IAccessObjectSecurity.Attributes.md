# IAccessObjectSecurity.Attributes


## Синтаксис


Attributes: [AccessElementAttributes](../../Enums/AccessElementAttributes.htm);


## Описание


Свойство Attributes возвращает доступный набор атрибутов доступа.


## Комментарии


Данное свойство возвращает число в десятичной форме, которое в двоичной форме будет соответствовать значению, либо комбинации значений, содержащихся в перечислении [AccessElementAttributes](../../Enums/AccessElementAttributes.htm).
