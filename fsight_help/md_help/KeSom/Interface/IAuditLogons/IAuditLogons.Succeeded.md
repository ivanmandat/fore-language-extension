# IAuditLogons.Succeeded


## Синтаксис


Succeeded: Boolean;


## Описание


Свойство Succeeded возвращает
 признак удачного подключения к репозиторию.


## Комментарии


Свойство возвращает значение True,
 если не было запрета на подключение к репозиторию.
