# IMetabaseCustomForeEvent.InvokeEvent


## Синтаксис


InvokeEvent(SesConn: [ISecurityConnection](../ISecurityConnection/ISecurityConnection.htm));


## Параметры


SesConn. Соединение с сервером
 СУБД.


## Описание


Метод InvokeEvent вызывает наступление
 пользовательского события при соединении с сервером базы данных.


## Комментарии


Для вызова наступления пользовательского события используйте метод [IMetabaseCustomForeEvent.Invoke](IMetabaseCustomForeEvent.Invoke.htm).
