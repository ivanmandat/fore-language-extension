# IMetabaseCustomForeEvent.Invoke


## Синтаксис


Invoke;


## Параметры


Context. Контекст репозитория,
 в рамках которого генерируется событие.


## Описание


Метод Invoke вызывает наступление
 пользовательского события.


## Комментарии


При работе в текущем репозитории в качестве значения параметра Context можно передать значение
 Null.
