# IMetabaseUpdateAccessSubject


Сборка: Metabase;


## Описание


Интерфейс IMetabaseUpdateAccessSubject
 содержит свойства и методы для работы с субъектом безопасности в обновлении,
 по которым происходит перенос прав доступа на объекты.


## Иерархия наследования


IMetabaseUpdateAccessSubject.
