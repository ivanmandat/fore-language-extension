# IMetabaseUpdateAccessSubject.AlternativeSid


## Синтаксис


AlternativeSid: [ISid](../ISid/ISid.htm);


## Описание


Свойство AlternativeSid возвращает соответствующий идентификатор субъекта безопасности при поиске по имени субъекта безопасности.


## Комментарии


Используется в том случае, если [при поиске по идентификатору](IMetabaseUpdateAccessSubject.RemappingSid.htm) субъект безопасности не был найден.
