# IPrimaryODBCSPLD.ODBCParamName


## Синтаксис


ODBCParamName(Index: Integer): String;


## Параметры


Index. Индекс параметра.


## Описание


Свойство ODBCParamName возвращает
 наименование параметра с указанным индексом.


## Комментарии


Список параметров формируется при задании для них значений с помощью
 свойства [ODBCParams](../../../Dal/Interface/IDalODBCConnectionDescriptor/IDalODBCConnectionDescriptor.ODBCParams.htm)
 или [ODBCParamValue](../../../Dal/Interface/IDalODBCConnectionDescriptor/IDalODBCConnectionDescriptor.ODBCParamValue.htm).
