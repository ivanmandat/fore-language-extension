# IPrimaryODBCSPLD.ODBCParamCount


## Синтаксис


ODBCParamCount: Integer;


## Описание


Свойство ODBCParamCount возвращает
 количество параметров, для которых определены значения.


## Комментарии


Значение свойства изменяется при задании значений параметров с помощью
 свойства [ODBCParams](IPrimaryODBCSPLD.ODBCParams.htm) или
 [ODBCParamValue](IPrimaryODBCSPLD.ODBCParamValue.htm).
