# IPrimaryODBCSPLD


Сборка: Metabase;


## Описание


Интерфейс IPrimaryODBCSPLD содержит
 свойства и методы модуля безопасности, используемого при подключении к
 источникам данных с помощью одного из ODBC драйверов, которые установлены
 в операционной системе.
