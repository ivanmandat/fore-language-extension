# IMetabaseGroups


## Описание


Интерфейс IMetabaseGroups содержит
 свойства коллекции групп пользователей.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attributes](IMetabaseGroups.Attributes.htm)
		 Свойство Attributes
		 возвращает коллекцию атрибутов групп пользователей.


		 ![](../../Property_Image.gif)
		 [Count](IMetabaseGroups.Count.htm)
		 Свойство Count возвращает
		 количество групп пользователей активного репозитория.


		 ![](../../Property_Image.gif)
		 [Item](IMetabaseGroups.Item.htm)
		 Свойство Item возвращает
		 группу пользователей активного репозитория.
