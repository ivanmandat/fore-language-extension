# ISysLogSettings.Save


## Синтаксис


Save;


## Описание


Метод Save сохраняет настройки
 подключения syslog-сервера.


## Комментарии


Место сохранения настроек определяется свойством [ISysLogSettings.Scope](ISysLogSettings.Scope.htm).


## Пример


Использование свойства приведено в примере для [ISysLogSettings.IsActive](ISysLogSettings.IsActive.htm).
