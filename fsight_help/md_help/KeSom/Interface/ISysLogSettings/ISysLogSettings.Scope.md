# ISysLogSettings.Scope


## Синтаксис


Scope: [SysLogSettingsScope](../../Enums/SysLogSettingsScope.htm);


## Описание


Свойство Scope определяет расположение
 настроек подключения syslog-сервера.


## Комментарии


Если расположение не задано явно, то расположение настроек совпадает
 с местом их прочтения.
