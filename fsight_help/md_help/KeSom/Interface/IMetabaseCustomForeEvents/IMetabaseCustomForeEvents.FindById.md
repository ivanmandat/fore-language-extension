# IMetabaseCustomForeEvents.FindById


## Синтаксис


FindById(Id: String): [IMetabaseCustomForeEvent](../IMetabaseCustomForeEvent/IMetabaseCustomForeEvent.htm);


## Параметры


Id. Идентификатор пользовательского
 события.


## Описание


Метод FindById осуществляет
 поиск пользовательского события в коллекции по его идентификатору.


## Комментарии


Для удаления пользовательского события из коллекции по индексу используйте
 метод [IMetabaseCustomForeEvents.Remove](IMetabaseCustomForeEvents.Remove.htm).
