# IMetabaseCustomForeEvents.Item


## Синтаксис


Item(Index: Integer): [IMetabaseCustomForeEvent](../IMetabaseCustomForeEvent/IMetabaseCustomForeEvent.htm);


## Параметры


Index. Индекс пользовательского
 события в коллекции.


## Описание


Свойство Item возвращает пользовательское
 событие из коллекции по его индексу.


## Комментарии


Для получения количества пользовательских событий в коллекции используйте
 свойство [IMetabaseCustomForeEvents.Count](IMetabaseCustomForeEvents.Count.htm).
