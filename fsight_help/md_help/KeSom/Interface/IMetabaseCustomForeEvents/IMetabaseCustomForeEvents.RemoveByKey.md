# IMetabaseCustomForeEvents.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ удаляемого пользовательского
 события.


## Описание


Метод RemoveByKey удаляет пользовательское
 событие из коллекции по его ключу.


## Комментарии


В случае успешного удаления пользовательского события из коллекции метод
 возвращает True, иначе - False.
