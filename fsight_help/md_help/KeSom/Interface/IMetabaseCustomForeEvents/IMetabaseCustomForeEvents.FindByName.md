# IMetabaseCustomForeEvents.FindByName


## Синтаксис


FindByName(Name: String): [IMetabaseCustomForeEvent](../IMetabaseCustomForeEvent/IMetabaseCustomForeEvent.htm);


## Параметры


Name. Наименование пользовательского
 события.


## Описание


Метод FindByName осуществляет
 поиск пользовательского события в коллекции по его наименованию.


## Комментарии


Для очистки коллекции пользовательских событий используйте метод [IMetabaseCustomForeEvents.Clear](IMetabaseCustomForeEvents.Clear.htm).
