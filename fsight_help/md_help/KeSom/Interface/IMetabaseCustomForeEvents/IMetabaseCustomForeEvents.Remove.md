# IMetabaseCustomForeEvents.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемого пользовательского
 события.


## Описание


Метод Remove удаляет пользовательское
 событие из коллекции по его индексу.


## Комментарии


В случае успешного удаления пользовательского события из коллекции метод
 возвращает True, иначе - False.
