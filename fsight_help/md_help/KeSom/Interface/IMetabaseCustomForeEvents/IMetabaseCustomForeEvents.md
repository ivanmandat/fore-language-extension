# IMetabaseCustomForeEvents


Сборка: Metabase;


## Описание


Интерфейс IMetabaseCustomForeEvents
 предназначен для работы с коллекцией пользовательских событий.


## Иерархия наследования


           IMetabaseCustomForeEvents


## Комментарии


Коллекцию пользовательских событий возвращает свойство [IMetabaseCustomExtender.Events](../IMetabaseCustomExtender/IMetabaseCustomExtender.Events.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMetabaseCustomForeEvents.Count.htm)


		 Свойство Count возвращает
		 количество пользовательских событий в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMetabaseCustomForeEvents.Item.htm)


		 Свойство Item возвращает
		 пользовательское событие из коллекции по его индексу.
