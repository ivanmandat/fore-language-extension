# IMetabaseCustomForeEvents.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IMetabaseCustomForeEvent](../IMetabaseCustomForeEvent/IMetabaseCustomForeEvent.htm);


## Параметры


Key. Ключ пользовательского
 события.


## Описание


Метод FindByKey осуществляет
 поиск пользовательского события в коллекции по его ключу.


## Комментарии


Для удаления пользовательского события из коллекции по ключу используйте
 метод [IMetabaseCustomForeEvents.RemoveByKey](IMetabaseCustomForeEvents.RemoveByKey.htm).
