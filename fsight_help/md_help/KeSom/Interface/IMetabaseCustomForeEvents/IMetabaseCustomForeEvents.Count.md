## Синтаксис

## Описание


Свойство Count возвращает количество
 пользовательских событий в коллекции.


## Комментарии


Для получения пользовательского события из коллекции используйте свойство
 [IMetabaseCustomForeEvents.Item](IMetabaseCustomForeEvents.Item.htm).
