# ILogonParams.OK


## Синтаксис


OK: Boolean;


## Описание


Свойство OK определяет признак нажатия кнопки «OK» в диалоге авторизации.


## Комментарии


Если свойство возвращает значение True, значит в диалоге авторизации была нажата кнопка «OK», иначе - «Отмена».
