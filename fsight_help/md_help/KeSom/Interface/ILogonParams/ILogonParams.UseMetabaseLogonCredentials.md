# ILogonParams.UseMetabaseLogonCredentials


## Синтаксис


UseMetabaseLogonCredentials: [TriState](ForeSys.chm::/Enums/TriState.htm);


## Описание


Свойство UseMetabaseLogonCredentials определяет, будут ли использоваться для соединения с базой данных учетные данные пользователя, указанные при входе в систему.


## Комментарии


Данное свойство соответствует опции «Учетные данные, указанные при входе в систему» в диалоге «[Авторизация в БД](UiNavObj.chm::/database/UiDb_database_authorization.htm)». В случае если свойство было установлено в OnOption, то при последующих подключениях будут использоваться данные, указанные при входе в систему.
