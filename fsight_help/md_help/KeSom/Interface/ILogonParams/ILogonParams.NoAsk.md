# ILogonParams.NoAsk


## Синтаксис


NoAsk: Boolean;


## Описание


Свойство NoAsk определяет признак автоматического подключения.


## Комментарии


Если свойству установлено значение True, то при проверке будет осуществляться автоматическое подключение с учетными данными, указанными в свойстве [Creds](ILogonParams.Creds.htm). Если свойству установлено значение False, то при проверке будет выводиться окно авторизации пользователя, учетные данные которого, указаны в свойстве [Creds](ILogonParams.Creds.htm).
