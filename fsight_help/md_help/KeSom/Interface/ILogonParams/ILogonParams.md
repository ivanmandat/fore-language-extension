# ILogonParams


## Описание


Интерфейс ILogonParams содержит
 свойства для определения параметров подключения.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Creds](ILogonParams.Creds.htm)
		 Свойство Creds определяет
		 учетные данные пользователя, возможность подключения с которыми
		 необходимо проверить.


		 ![](../../Property_Image.gif)
		 [Data](ILogonParams.Data.htm)
		 Свойство Data определяет
		 данные, используемые модулем безопасности при проверке подключения.


		 ![](../../Property_Image.gif)
		 [Descr](ILogonParams.Descr.htm)
		 Свойство Descr определяет
		 описание репозитория, к которому осуществляется проверка подключения.


		 ![](../../Property_Image.gif)
		 [ErrorException](ILogonParams.ErrorException.htm)
		 Свойство ErrorException
		 определяет описание исключительной ситуации, возникшей при
		 входе в систему.


		 ![](../../Property_Image.gif)
		 [MetabaseCompatible](ILogonParams.MetabaseCompatible.htm)
		 Свойство MetabaseCompatible
		 возвращает, можно ли использовать учетные данные пользователя,
		 указанные при входе в систему, для соединения с базой данных.


		 ![](../../Property_Image.gif)
		 [NoAsk](ILogonParams.NoAsk.htm)
		 Свойство NoAsk определяет
		 признак автоматического подключения.


		 ![](../../Property_Image.gif)
		 [NoCache](ILogonParams.NoCache.htm)
		 Свойство NoCache определяет
		 признак кеширования учетных данных.


		 ![](../../Property_Image.gif)
		 [OK](ILogonParams.OK.htm)
		 Свойство OK определяет
		 признак нажатия кнопки «OK» в диалоге авторизации.


		 ![](../../Property_Image.gif)
		 [Pack](ILogonParams.Pack.htm)
		 Свойство Pack определяет
		 модуль безопасности, используемый при проверке подключения.


		 ![](../../Property_Image.gif)
		 [Realm](ILogonParams.Realm.htm)
		 Свойство Realm определяет
		 область действия подключения.


		 ![](../../Property_Image.gif)
		 [ReportError](ILogonParams.ReportError.htm)
		 Свойство ReportError
		  определяет признак того, настроен ли вывод сообщения
		 об ошибке при неверном вводе имени пользователя/пароля.


		 ![](../../Property_Image.gif)
		 [ToAdmin](ILogonParams.ToAdmin.htm)
		 Свойство ToAdmin определяет
		 признак использования указанных учетных данных в качестве учетных
		 данных администратора при проверке подключения.


		 ![](../../Property_Image.gif)
		 [UseMetabaseLogonCredentials](ILogonParams.UseMetabaseLogonCredentials.htm)
		 Свойство UseMetabaseLogonCredentials
		 определяет, будут ли использоваться для соединения с базой данных
		 учетные данные пользователя, указанные при входе в систему.
