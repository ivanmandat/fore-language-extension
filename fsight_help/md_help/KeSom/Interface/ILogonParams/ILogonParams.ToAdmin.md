# ILogonParams.ToAdmin


## Синтаксис


ToAdmin: Boolean;


## Описание


Свойство ToAdmin определяет признак использование указанных учетных данных в качестве учетных данных администратора при проверке подключения.


## Комментарии


Если свойству установлено значение False, то учетные данные используются как учетные данные обычного пользователя. Если свойству установлено значение True, то учетные данные будут использоваться как учетные данные администратора.
