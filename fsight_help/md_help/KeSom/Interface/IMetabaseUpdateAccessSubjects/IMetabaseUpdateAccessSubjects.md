# IMetabaseUpdateAccessSubjects


Сборка: Metabase;


## Описание


Интерфейс IMetabaseUpdateAccessSubjects
 содержит свойства и методы для работы с субъектами безопасности в обновлении,
 по которым происходит перенос прав доступа на объекты.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMetabaseUpdateAccessSubjects.Count.htm)
		 Свойство Count возвращает
		 количество субъектов безопасности в обновлении, по которым происходит
		 перенос прав доступа на объекты.


		 ![](../../Property_Image.gif)
		 [Item](IMetabaseUpdateAccessSubjects.Item.htm)
		 Свойство Item возвращает
		 субъект безопасности по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindBySidString](IMetabaseUpdateAccessSubjects.FindBySidString.htm)
		 Метод FindBySidString
		 выполняет поиск субъекта безопасности по идентификатору.
