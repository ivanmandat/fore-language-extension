# ICertificate


Сборка: Metabase;


## Описание


Интерфейс ICertificate содержит
 свойства и методы для работы с сертификатом безопасности.


## Иерархия наследования


ICertificate


## Комментарии


Параметры сертификата можно получить, используя свойства и методы интерфейса
 [ICertificates](../ICertificates/ICertificates.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Data](ICertificate.Data.htm)
		 Свойство Data возвращает
		 содержимое сертификата в виде текстовой строки формата Base64.

## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)

		 [SetData](ICertificate.SetData.htm)
