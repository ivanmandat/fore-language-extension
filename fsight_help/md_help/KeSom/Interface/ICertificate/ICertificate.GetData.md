# ICertificate.GetData


## Синтаксис


GetData(Stream: [IIOStream](ModIo.chm::/Interface/IIOStream/IIOStream.htm));


## Параметры


Stream. Поток, в который будет
 помещено содержимое сертификата.


## Описание


Метод GetData загружает содержимое
 текущего сертификата в указанный поток.

Пример использования приведен в описании метода [ICertProvider.OpenVerifier](../ICertProvider/ICertProvider.OpenVerifier.htm).
