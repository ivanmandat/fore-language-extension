# IMetabaseObjectFindInfo.WholeWordsOnly


## Синтаксис


		WholeWordsOnly: Boolean;


## Описание


Свойство WholeWordsOnly определяет,
 будет ли производиться поиск по заданной строке целиком.


## Комментарии


Для задания строки поиска используйте свойство [IMetabaseObjectFindInfo.Text](IMetabaseObjectFindInfo.Text.htm).


Допустимые значения:


	- True. Значение по умолчанию.
	 Поиск будет производиться по заданной строке целиком;


	- False. Осуществляется
	 поиск объектов, содержащих в своем наименовании/идентификаторе заданную
	 строку.
