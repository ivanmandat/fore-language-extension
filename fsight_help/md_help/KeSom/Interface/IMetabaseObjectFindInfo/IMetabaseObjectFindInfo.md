# IMetabaseObjectFindInfo


Сборка: Metabase;


## Описание


Интерфейс IMetabaseObjectFindInfo
 содержит свойства, используемые для задания условий поиска объектов репозитория.


## Иерархия наследования


           IMetabaseObjectFindInfo


## Комментарии


Условия поиска можно получить, используя метод [IMetabase.CreateFindInfo](../IMetabase/IMetabase.CreateFindInfo.htm).


Заданные условия поиска используются методом [IMetabase.Find](../IMetabase/IMetabase.Find.htm)
 для поиска объектов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attribute](IMetabaseObjectFindInfo.Attribute.htm)
		 Свойство Attribute
		 определяет атрибут, по значениям которого будет осуществляться
		 поиск.


		 ![](../../Property_Image.gif)
		 [AttributeEx](IMetabaseObjectFindInfo.AttributeEx.htm)
		 Свойство AttributeEx
		 определяет атрибут, по значениям которого будет осуществляться
		 расширенный поиск.


		 ![](../../Property_Image.gif)
		 [CaseSensitive](IMetabaseObjectFindInfo.CaseSensitive.htm)
		 Свойство CaseSensitive
		 определяет признак учета регистра символов при поиске.


		 ![](../../Property_Image.gif)
		 [ClassId](IMetabaseObjectFindInfo.ClassId.htm)
		 Свойство ClassId определяет
		 класс объектов, которые будут проверяться в процессе поиска.


		 ![](../../Property_Image.gif)
		 [ContainersContent](IMetabaseObjectFindInfo.ContainersContent.htm)
		 Свойство ContainersContent
		 определяет признак поиска среди объектов в объекта-контейнерах.


		 ![](../../Property_Image.gif)
		 [InternalObjects](IMetabaseObjectFindInfo.InternalObjects.htm)
		 Свойство InternalObjects
		 определяет признак поиска скрытых объектов.


		 ![](../../Property_Image.gif)
		 [ScanHiddenFolders](IMetabaseObjectFindInfo.ScanHiddenFolders.htm)
		 Свойство ScanHiddenFolders
		 определяет, будет ли производиться поиск в скрытых папках.


		 ![](../../Property_Image.gif)
		 [ScanNestedNamespaces](IMetabaseObjectFindInfo.ScanNestedNamespaces.htm)
		 Свойство ScanNestedNameSpaces
		 определяет, будут ли просматриваться вложенные пространства имен.


		 ![](../../Property_Image.gif)
		 [Scope](IMetabaseObjectFindInfo.Scope.htm)
		 Свойство Scope определяет
		 область для осуществления поиска.


		 ![](../../Property_Image.gif)
		 [Text](IMetabaseObjectFindInfo.Text.htm)
		 Свойство Text определяет
		 текст для осуществления поиска.


		 ![](../../Property_Image.gif)
		 [WholeWordsOnly](IMetabaseObjectFindInfo.WholeWordsOnly.htm)
		 Свойство WholeWordsOnly
		 определяет, будет ли производиться поиск по заданной строке целиком.
