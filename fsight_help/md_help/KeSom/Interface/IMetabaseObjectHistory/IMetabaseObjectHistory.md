# IMetabaseObjectHistory


Сборка: Metabase;


## Описание


Интерфейс IMetabaseObjectHistory
 содержит свойства истории изменения объекта репозитория.


## Иерархия наследования


           IMetabaseObjectHistory


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMetabaseObjectHistory.Count.htm)
		 Свойство Count возвращает
		 количество записей в истории изменения объекта.


		 ![](../../Property_Image.gif)
		 [Item](IMetabaseObjectHistory.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий запись в истории объекта.


		 ![](../../Property_Image.gif)
		 [Key](IMetabaseObjectHistory.Key.htm)
		 Свойство Key возвращает
		 ключ объекта репозитория, которому соответствует данная история
		 изменений.
