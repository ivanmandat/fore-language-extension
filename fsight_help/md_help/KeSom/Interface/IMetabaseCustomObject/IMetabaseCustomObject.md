# IMetabaseCustomObject


## Описание


Интерфейс IMetabaseCustomObject
 содержит свойства и методы для работы с объектом пользовательского класса.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CustomClass](IMetabaseCustomObject.CustomClass.htm)


		 Свойство CustomClass
		 определяет пользовательский класс объекта.


		 ![](../../Property_Image.gif)
		 [Extender](IMetabaseCustomObject.Extender.htm)


		 Свойство Extender определяет
		 контейнер пользовательского класса.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CheckAndAudit](IMetabaseCustomObject.CheckAndAudit.htm)


		 Метод CheckAndAudit
		 проверяет права доступа текущего пользователя на указанную операцию
		 пользовательского класса и заносит запись в протокол доступа с
		 указанным комментарием.


		 ![](../../Sub_Image.gif)
		 [CheckAndAuditOperation](IMetabaseCustomObject.CheckAndAuditOperation.htm)


		 Метод CheckAndAuditOperation
		 проверяет права доступа текущего пользователя на указанную операцию
		 пользовательского класса и протоколирует результат проверки в
		 соответствии с выбранным вариантом аудита.


		 ![](../../Sub_Image.gif)
		 [CheckAndAuditOperationLabel](IMetabaseCustomObject.CheckAndAuditOperationLabel.htm)


		 Метод CheckAndAuditOperationLabel
		 проверяет права доступа текущего пользователя на указанную операцию
		 и протоколирует результат проверки в соответствии с выбранным
		 вариантом аудита, а также вычисляет параметры метки безопасности,
		 используемой при печати и экспорте объектов репозитория.


		 ![](../../Sub_Image.gif)
		 [CreateReader](IMetabaseCustomObject.CreateReader.htm)


		 Метод CreateReader
		 создает объект, позволяющий читать данные из объекта.


		 ![](../../Sub_Image.gif)
		 [CreateWriter](IMetabaseCustomObject.CreateWriter.htm)


		 Метод CreateWriter
		 создает объект, позволяющий писать данные в объект.
