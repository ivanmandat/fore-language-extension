# IMetabaseGroup.Member


## Синтаксис


Member(Index: Integer): [ISecuritySubject](../ISecuritySubject/ISecuritySubject.htm);


## Параметры


Index -
 индекс члена встроенной группы пользователей.


## Описание


Свойство Member возвращает параметры
 субъекта безопасности, входящего в данную [встроенную группу
 пользователей](Admin.chm::/03_Admin/Groups/Admin_Groups.htm).


Индекс субъекта безопасности передается посредством параметра Index.
