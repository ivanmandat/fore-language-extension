# IMetabaseGroup.RemoveMember


## Синтаксис


		RemoveMember(Subject: [ISecuritySubject](../ISecuritySubject/ISecuritySubject.htm));


## Параметры


Subject. Субъект безопасности, который необходимо
 удалить.


## Описание


Метод RemoveMember удаляет субъект
 безопасности, передаваемый посредством параметра Subject,
 из данной группу пользователей.
