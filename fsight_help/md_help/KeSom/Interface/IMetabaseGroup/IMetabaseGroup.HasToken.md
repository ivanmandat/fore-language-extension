# IMetabaseGroup.HasToken


## Синтаксис


		HasToken: Boolean;


## Описание


Свойство HasToken определяет
 возможность настройки параметров мандатного метода разграничения доступа
 для группы пользователей.


## Комментарии


Свойство актуально только при работе с доменной группой пользователей,
 которая включена в список субъектов безопасности репозитория. В репозитории
 должен быть включён [мандатный
 метод разграничения доступа](../IMetabasePolicy/IMetabasePolicy.MandatoryAccessControl.htm) или [метод
 разграничения доступа по уровням](../IMetabaseMandatoryAccess/IMetabaseMandatoryAccess.IsSimple.htm).


Допустимые значения:


	- True. Настройки параметров
	 мандатного метода разграничения доступа доступны для доменной группы
	 с помощью свойства [IMetabaseGroup.Token](IMetabaseGroup.Token.htm).
	 При входе в репозиторий с использованием интегрированной доменной
	 аутентификации осуществляется проверка пользователя. Если пользователь
	 состоит только в одной группе пользователей с назначенным уровнем
	 доступа, то вход в репозиторий будет разрешен, иначе - запрещен;


	- False. Настройки параметров
	 мандатного метода разграничения доступа для группы сбрасываются и
	 не используются при работе в репозитории.
