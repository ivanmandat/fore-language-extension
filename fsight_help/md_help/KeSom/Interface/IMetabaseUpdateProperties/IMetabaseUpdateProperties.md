# IMetabaseUpdateProperties


## Описание


Интерфейс IMetabaseUpdateProperties
 содержит свойства и методы для работы с коллекцией флагов обновления.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMetabaseUpdateProperties.Count.htm)
		 Свойство Count возвращает
		 количество флагов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMetabaseUpdateProperties.Item.htm)
		 Свойство Item возвращает
		 флаг из коллекции по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IMetabaseUpdateProperties.Add.htm)
		 Метод Add осуществляет
		 добавление флага в коллекцию.


		 ![](../../Sub_Image.gif)
		 [FindById](IMetabaseUpdateProperties.FindById.htm)
		 Метод FindById осуществляет
		 поиск флага по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IMetabaseUpdateProperties.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск флага по ключу.


		 ![](../../Sub_Image.gif)
		 [Move](IMetabaseUpdateProperties.Move.htm)
		 Метод Move осуществляет
		 перемещение флагов в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IMetabaseUpdateProperties.Remove.htm)
		 Метод Remove осуществляет
		 удаление флага из коллекции по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IMetabaseUpdateProperties.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление флага из коллекции по ключу.
