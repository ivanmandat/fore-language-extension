# IMetabaseUpdateProperties.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс флага в коллекции.


## Описание


Метод Remove осуществляет удаление флага из коллекции по индексу. Метод возвращает значение True, если удаление прошло успешно.
