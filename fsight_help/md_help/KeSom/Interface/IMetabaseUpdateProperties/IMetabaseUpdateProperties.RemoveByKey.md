# IMetabaseUpdateProperties.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key - ключ флага в коллекции.


## Описание


Метод RemoveByKey осуществляет удаление флага из коллекции по ключу. Метод возвращает значение True, если удаление прошло успешно.
