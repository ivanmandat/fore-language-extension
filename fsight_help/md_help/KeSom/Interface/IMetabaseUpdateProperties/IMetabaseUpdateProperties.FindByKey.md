# IMetabaseUpdateProperties.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IMetabaseUpdateProperty](../IMetabaseUpdateProperty/IMetabaseUpdateProperty.htm);


## Параметры


Key - ключ флага.


## Описание


Метод FindByKey осуществляет поиск флага по ключу.


В случае удачного поиска метод возвращает свойства флага обновления, иначе возвращает значение Null.
