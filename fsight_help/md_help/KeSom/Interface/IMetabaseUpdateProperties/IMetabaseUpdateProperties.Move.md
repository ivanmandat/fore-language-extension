# IMetabaseUpdateProperties.Move


## Синтаксис


Move(IndexFrom: Integer; IndexTo: Integer);


## Параметры


IndexFrom - индекс позиции флага, который необходимо переместить.


IndexTo - индекс позиции, в которую необходимо переместить флаг.


## Описание


Метод Move осуществляет перемещение флагов в коллекции.
