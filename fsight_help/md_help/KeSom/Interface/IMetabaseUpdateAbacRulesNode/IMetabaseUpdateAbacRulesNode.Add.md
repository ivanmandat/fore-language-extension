# IMetabaseUpdateAbacRulesNode.Add


## Синтаксис


Add(Object: [IABACEvaluatedObject](KeABAC.chm::/Interface/IABACEvaluatedObject/IABACEvaluatedObject.htm);
 WithChildren: Boolean);


## Параметры


Object.
 Элемент структуры атрибутного доступа.


WithChildren. Параметр, определяющий
 будут ли добавлены дочерние элементы. Допустимые значения:


	- True - дочерние элементы
	 будут добавлены вместе с выбранным;


	- False - дочерние элементы
	 не будут добавлены.


## Описание


Метод Add позволяет добавить
 элемент структуры атрибутного доступа с дочерними элементами в обновление.
