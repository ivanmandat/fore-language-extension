# IMetabaseUpdateAbacRulesNode.KeepActive


## Синтаксис


KeepActive: Boolean;


## Описание


Свойство KeepActive определяет
 сохранение в обновлении статусов элементов [структуры
 атрибутного доступа](Admin.chm::/04_SecurityPolicy/Admin_AttributeAccess.htm).


## Комментарии


Допустимые значения:


	- True. Статусы элементов
	 структуры атрибутного доступа сохраняются в обновлении;


	- False. По умолчанию.
	 Статусы элементов структуры атрибутного доступа не сохраняются в обновлении.


Статус определяет разрешение на выполнение проверки доступа существующим
 элементом и задается с помощью флажков напротив элементов [структуры
 атрибутного доступа](Admin.chm::/04_SecurityPolicy/Admin_AttributeAccess.htm) в менеджере безопасности.


Для получения подробной информации обратитесь к разделу «[Добавление
 правил атрибутного доступа](UpdManager.chm::/CreateUpdate/Admin_CreateUpdate_ABACRule.htm)».
