# MetabaseUsersUpdateCallbackResult


## Описание


Перечисление MetabaseUsersUpdateCallbackResult содержит действия при работе с ошибками, возникшими при обновлении пользователей.


Используется следующими свойствами и методами:


-
[IMetabaseUsersUpdateCallback.CallbackResult](../Interface/IMetabaseUsersUpdateCallback/IMetabaseUsersUpdateCallback.CallbackResult.htm);


-
[IMetabaseUsersUpdateCallback.RequestAction](../Interface/IMetabaseUsersUpdateCallback/IMetabaseUsersUpdateCallback.RequestAction.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Default_. Действие по умолчанию. Продолжить до следующей ошибки.


 1
 SilentContinue. Продолжить обновление без уведомления об ошибке.


 2
 Abort. Прервать обновление.
