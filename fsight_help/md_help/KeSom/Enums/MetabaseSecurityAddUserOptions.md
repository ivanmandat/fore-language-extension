# MetabaseSecurityAddUserOptions


## Описание


Перечисление MetabaseSecurityAddUserOptions
 содержит опции создания пользователей в платформе.


Используется следующими свойствами и методами:


	- [IMetabaseSecurity.AddUserEx](../Interface/IMetabaseSecurity/IMetabaseSecurity.AddUserEx.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 Internal. Признак создания
		 внутреннего пользователя платформы.


		 2
		 External. Признак добавления
		 пользователя, подключаемого с сервера.


## Комментарии


При создании внутреннего пользователя его имя принудительно приводится
 в верхний регистр. Подключение будет возможно только при указании имени
 в верхнем регистре.


При добавлении пользователя, подключаемого с сервера, пользователь добавляется
 с тем именем, с каким он создан на уровне СУБД.
