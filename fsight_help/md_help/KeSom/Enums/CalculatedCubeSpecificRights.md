# CalculatedCubeSpecificRights


## Описание


Перечисление CalculatedCubeSpecificRights
 содержит список специфических операций, доступных для вычисляемых кубов.


## Комментарии


Список основных и дополнительных операций доступен в перечислении [MetabaseObjectPredefinedRights](MetabaseObjectPredefinedRights.htm).
