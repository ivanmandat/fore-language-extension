# ApplyStateType


## Описание


Перечисление ApplyStateType содержит типы состояний восстановления субъектов безопасности.


Используется следующими свойствами и методами:


-
[ISecuritySnapshotCallback.OnSubjectApply](../Interface/ISecuritySnapshotCallback/ISecuritySnapshotCallback.OnSubjectApply.htm);


-
[ISecuritySnapshotLog.ApplyState](../Interface/ISecuritySnapshotLog/ISecuritySnapshotLog.ApplyState.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Нет изменений.


 1
 Created. Создан.


 2
 Changed. Изменен.


 3
 Error. Ошибка.


 4
 Deleted. Удален.
