# SecuritySubjectType


## Описание


Перечисление SecuritySubjectType
 содержит типы субъектов безопасности.


Используется следующими свойствами и методами:


	- [ISecuritySnapshotLog.SubjectType](../Interface/ISecuritySnapshotLog/ISecuritySnapshotLog.SubjectType.htm);


	- [ISecuritySubjectsSearch.SubjectCriteria](../Interface/ISecuritySubjectsSearch/ISecuritySubjectsSearch.SubjectCriteria.htm);


	- [Команда
	 SelectSecSubject](UiLib.chm::/Interface/IUiCommandTarget/Command/SelectSecSubject.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Unknown. Неизвестный
		 тип субъекта.


		 1
		 User. Пользователь.


		 2
		 Group. Группа.
