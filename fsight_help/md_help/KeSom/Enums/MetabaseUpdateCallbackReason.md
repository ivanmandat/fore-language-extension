# MetabaseUpdateCallbackReason


## Описание


Перечисление MetabaseUpdateCallbackReason
 содержит возможные действия, являющиеся причиной возникновения события,
 связанного с обновлением.


Используется следующими свойствами и методами:


	- [IUpdateCallback.OnCallback](../Interface/IUpdateCallback/IUpdateCallback.OnCallback.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 ApplyUI. Вызов мастера
		 применения обновления из визуального интерфейса приложения.


		 2
		 ApplyCoreStart. Вызов
		 применения обновления ядром платформы.


		 3
		 ApplyCoreEnd. Окончание
		 применения обновления ядром платформы.


		 4
		 SaveUI. Вызов сохранения
		 обновления из визуального интерфейса приложения.


		 5
		 SaveCore. Вызов сохранения
		 обновления ядром платформы.


		 6
		 LoadCoreEnd. Окончание
		 чтения обновления ядром платформы.
