# MetabaseUpdateAccessType


## Описание


Перечисление MetabaseUpdateAccessType
 содержит тип доступа к обновлению.


Используется следующими свойствами и методами:


	- [IMetabaseUpdate.AccessAllowed](../Interface/IMetabaseUpdate/IMetabaseUpdate.AccessAllowed.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Значение не определено.


		 1
		 Full. Полный доступ
		 к обновлению.


		 2
		 Restricted. Нет доступа
		 к обновлению.


		 4
		 PartAccess. Доступ
		 к части объектов обновления.
