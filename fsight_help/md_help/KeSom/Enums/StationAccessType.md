# StationAccessType


## Описание


Перечисление StationAccessType содержит типы доступа с рабочих станций.


Используется следующими свойствами и методами:


-
[IStation.Access](../Interface/IStation/IStation.Access.htm);


-
[IStations.Access](../Interface/IStations/IStations.Access.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Default. Доступ по умолчанию.


 1
 Forbidden. Запрещен доступ.


 2
 Allowed. Разрешен доступ.
