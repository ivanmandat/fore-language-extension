# MetabaseUpdateCopyType


## Описание


Перечисление MetabaseUpdateCopyType
 содержит тип копирования обновления.


Используется следующими свойствами и методами:


	- [IMetabaseUpdateNode.AccessAllowed](../Interface/IMetabaseUpdateNode/IMetabaseUpdateNode.AccessAllowed.htm);


	- [IMetabaseUpdate.Copy](../Interface/IMetabaseUpdate/IMetabaseUpdate.Copy.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Full. Полная копия.


		 1
		 Available. Копия доступная
		 текущему пользователю.


		 2
		 AvailableAdmin. Копия
		 доступная прикладному администратору при [разделении
		 ролей](Admin.chm::/04_SecurityPolicy/Editor_of_Politicy/Security_EditorPoliticy_Adm.htm) администраторов.


		 3
		 AvailableISA. Копия
		 доступная администратору информационной безопасности при
		 [разделении
		 ролей](Admin.chm::/04_SecurityPolicy/Editor_of_Politicy/Security_EditorPoliticy_Adm.htm) администраторов.
