# MetabaseCheckStatus


## Описание


Перечисление MetabaseCheckStatus
 возвращает статус объекта после проверки контрольной суммы.


Используется следующим свойством:


	- [IMetabaseCheckListItem.Status](../Interface/IMetabaseCheckListItem/IMetabaseCheckListItem.Status.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Undefined. Статус не
		 определен.


		 1
		 Identical. Объект в
		 репозитории идентичен источнику.


		 2
		 Different. Объект в
		 репозитории отличен от источника.


		 3
		 Absent. Объект в репозитории
		 отсутствует.
