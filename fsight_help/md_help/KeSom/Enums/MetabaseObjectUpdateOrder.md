# MetabaseObjectUpdateOrder


## Описание


Перечисление MetabaseObjectUpdateOrder
 используется для определения порядка обновления данных объектов.


Используется следующим свойством:


	- [IMetabaseUpdateObjectNode.UpdateOrder](../Interface/IMetabaseUpdateObjectNode/IMetabaseUpdateObjectNode.UpdateOrder.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Default. Порядок по
		 умолчанию.


		 1
		 BeforeParent. Перед
		 родительским объектом.


		 2
		 AfterParent. После
		 родительского объекта.


		 3
		 AfterParentData. После
		 обновления данных родительского объекта.
