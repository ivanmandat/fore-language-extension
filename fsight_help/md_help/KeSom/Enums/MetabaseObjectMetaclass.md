# MetabaseObjectMetaclass


## Описание


Перечисление MetabaseObjectMetaclass
 содержит классы объектов репозитория.


Используется следующими свойствами и методами:


	- [IMetabaseObjectFindInfo.ClassId](../Interface/IMetabaseObjectFindInfo/IMetabaseObjectFindInfo.ClassId.htm);


	- [IMetabaseDialogMetaclassFilter.ObjectMetaclass](KeExtCtrls.chm::/Interface/IMetabaseDialogMetaclassFilter/IMetabaseDialogMetaclassFilter.ObjectMetaclass.htm).
