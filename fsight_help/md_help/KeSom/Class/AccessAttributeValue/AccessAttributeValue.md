# AccessAttributeValue


## Описание


Класс AccessAttributeValue предназначен для создания атрибута доступа.


## Конструкторы


 Имя конструктора
 Краткое описание


 ![](../../Sub_Image.gif)
 [Create](AccessAttributeValue.Create.htm)
 Конструктор Create создает атрибут доступа.


## Свойства, унаследованные от [IAccessAttributeValue](../../Interface/IAccessAttributeValue/IAccessAttributeValue.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BitCount](../../Interface/IAccessAttributeValue/IAccessAttributeValue.BitCount.htm)


		 Свойство BitCount возвращает
		 количество субъектов безопасности, для которых могут быть определены
		 права.


		 ![](../../Property_Image.gif)
		 [ToInteger](../../Interface/IAccessAttributeValue/IAccessAttributeValue.ToInteger.htm)


		 Свойство ToInteger
		 возвращает число в десятичном виде, соответствующее битовой маске
		 доступа.


		 ![](../../Property_Image.gif)
		 [ToString](../../Interface/IAccessAttributeValue/IAccessAttributeValue.ToString.htm)


		 Свойство ToString возвращает
		 строку, соответствующую битовой маске доступа.


		 ![](../../Property_Image.gif)
		 [Value](../../Interface/IAccessAttributeValue/IAccessAttributeValue.Value.htm)


		 Свойство Value определяет
		 значение права доступа. При значении True
		 соответствующие права имеются, при значении False
		 - прав нет.
