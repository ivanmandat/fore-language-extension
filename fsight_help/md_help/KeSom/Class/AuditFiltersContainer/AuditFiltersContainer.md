# AuditFiltersContainer


## Описание


Класс AuditFiltersContainer
 содержит свойства и методы для работы с контейнером фильтров протокола
 доступа.


## Свойства, унаследованные от [IAuditFiltersContainer](../../Interface/IAuditFiltersContainer/IAuditFiltersContainer.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Callback](../../Interface/IAuditFiltersContainer/IAuditFiltersContainer.Callback.htm)
		 Свойство Callback определяет
		 действие, выполняемое при возникновении конфликта во время загрузки
		 контейнера.


		 ![](../../Property_Image.gif)
		 [DefaultFileName](../../Interface/IAuditFiltersContainer/IAuditFiltersContainer.DefaultFileName.htm)
		 Свойство DefaultFileName
		 возвращает полное имя файла контейнера по умолчанию.


		 ![](../../Property_Image.gif)
		 [FileName](../../Interface/IAuditFiltersContainer/IAuditFiltersContainer.FileName.htm)
		 Свойство FileName определяет
		 имя файла, из которого был загружен контейнер.


		 ![](../../Property_Image.gif)
		 [Filters](../../Interface/IAuditFiltersContainer/IAuditFiltersContainer.Filters.htm)
		 Свойство Filters возвращает
		 фильтры, которые содержит контейнер.
