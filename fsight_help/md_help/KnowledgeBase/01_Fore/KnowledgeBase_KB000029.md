# Использование ресурсов продукта
 «Форсайт. Аналитическая платформа»
 в сторонних приложениях


Номер статьи: KB000029


## Общие сведения


Для использования библиотек продукта «Форсайт. Аналитическая платформа»
 при разработке необходимо:


	- На компьютере прикладного разработчика установить «Форсайт. Аналитическая платформа»
	 с помощью одного из доступных [дистрибутивов](Setup.chm::/02_AK_Install/DistributionKit.htm).
	 Дистрибутив не должен использовать технологию Reg-free
	 COM registration.


	- В зависимости от того, на каком языке предполагается разработка,
	 установите один из следующих дистрибутивов:


		- FP10DevTools.exe, если разработка
		 будет осуществляться на языке C# или других языках, ориентированных
		 на .NET Framework;


		- FP10JavaInterops.exe, если разработка будет осуществляться
		 на языке Java;


		- FP10PythonInterops.exe, если разработка будет осуществляться
		 на языке Python.


После этого можно переходить к разработке приложения с использованием
 ресурсов из сборок «Форсайт. Аналитическая платформа».


При распространении приложения, разработанного в сторонней среде разработки
 и использующего сборки платформы, на компьютере каждого пользователя также
 необходимо будет установить «Форсайт. Аналитическая платформа»
 с помощью инсталлятора, не использующего технологию Reg-free
 COM registration. Также необходимо будет вручную прописать в системной
 переменной Path путь к папке с «Форсайт. Аналитическая платформа».


Примечание.
 Дистрибутив «FP10DevTools.exe» не должен применяться
 на компьютерах пользователей.
