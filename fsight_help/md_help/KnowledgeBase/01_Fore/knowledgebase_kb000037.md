# Настройки, влияющие на функциональность
 продукта «Форсайт. Аналитическая платформа»


Номер статьи: KB000037


## Введение


Кроме настроек, которые задаются в различных инструментах платформы
 и вынесены в визуальный интерфейс, существует ряд настроек, указываемых
 в реестре. Данные настройки по умолчанию могут отсутствовать, но при их
 создании (изменении их значения) изменится поведение отдельного инструмента
 или всего приложения «Форсайт. Аналитическая платформа».
 Ниже приведен список этих настроек в зависимости от инструмента, на который
 они влияют.


[![](../Opened.gif)![](../Closed.gif)Общие параметры](javascript:TextPopup(this))


			 Параметр
			 Описание


			 Ветка реестра: [HKEY_LOCAL_MACHINE\SOFTWARE\Foresight\Foresight Analytics Platform\10.0\Exception]


Параметр: EMail
			 Строковый параметр, определяющий адрес электронной почты,
			 который будет подставляться в поле адресата при отправке сообщений
			 об ошибках платформы.


[![](../Opened.gif)![](../Closed.gif)Среда разработки](javascript:TextPopup(this))


			 Параметр
			 Описание


			 Ветка реестра: [HKEY_CURRENT_USER\Software\Foresight\Foresight Analytics Platform\10.0\Ui\]

Параметр: MenuExpandTimeout
			 Параметр типа REG_DWORD, определяющий время задержки раскрытия пунктов
			 [главного
			 меню](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/MainMenu.htm), [контекстных
			 меню](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/PopupMenu.htm) и меню, реализованного на [ленте](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/Ribbon.htm).
			 Время указывается в миллисекундах. Если параметр не указан,
			 то используется значение, определенное в настройках операционной
			 системны.


			 Ветка реестра: [HKEY_CURRENT_USER\Software\Foresight\Foresight Analytics Platform\10.0\DevEnv\Debug]


Параметр: ToolTipEvaluation
			 Параметр типа REG_DWORD, указывающий на необходимость в режиме отладки
			 кода вычислять текущее значение переменных при наведении на
			 них курсора мыши. Если параметр не указан или имеет значение
			 «1», то при наведении курсора мыши на переменную будет отображена
			 всплывающая подсказка с текущим значением переменной. Для
			 отмены вычисления значений переменных создайте параметр ToolTipEvaluation со значением
			 «0».


			 Ветка реестра: [HKEY_CURRENT_USER\Software\Foresight\Foresight Analytics Platform\10.0\Fore]

Параметр: CompileWebRestrictedAsm
			 Параметр типа REG_DWORD, позволяющий разрешить [компилирование
			 сборок](UiDevEnv.chm::/01_Development_Environment/02_Work_in_Development_Environment/Way_to_compiling.htm), запрещённых для использования в
			 веб. По умолчанию параметр отсутствует, при этом во время
			 компиляции, если имеется ссылка на запрещённую для веба сборку,
			 то будет генерироваться сообщение об ошибке. Если создать
			 параметр CompileWebRestrictedAsm
			 и установить ему значение «1», то компиляция сборок будет
			 разрешена.
