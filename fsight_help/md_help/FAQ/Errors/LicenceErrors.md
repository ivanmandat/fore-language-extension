# Проблемы при проверке лицензии


Ниже описаны наиболее распространенные проблемы при
 проверке лицензий, а также причины их появления и пути решения:


[![](../opened.gif)![](../closed.gif)Отсутствует

	Проблема


	При запуске продукта «Форсайт. Аналитическая платформа»
	 появляется одно их следующих сообщений:


	Файл с лицензией не
	 найден.


	Пробный период лицензии
	 истёк.


	Причина


	Возможные причины:


		- файл лицензии отсутствует;


		- истёк срок действия лицензии;


		- сервер лицензий не доступен.


	Решение


	Обратитесь в техническую поддержку по адресу [support@fsight.ru](mailto:support@fsight.ru)
	 или через [сервисы
	 технической поддержки](https://www.fsight.ru/support/), которые доступны после регистрации на сайте.

	Проблема


	В процессе работы приложения появляется сообщение об ограничении
	 лицензии:


	Для
	 выполнения этого действия требуется активировать лицензию «Аналитические
	 панели»/«Конструктор хранилища данных» и т.д.


	Причина


	Выполняемое действие требует [дополнительную единицу
	 лицензирования](Setup.chm::/08_Licensing/AboutFeatures.htm).


	Решение


	Необходимо активировать единицу лицензирования, разрешающую выполнение
	 требуемого действия.


	Для получения дополнительных единиц лицензирования обратитесь в
	 техническую поддержку по адресу [support@fsight.ru](mailto:support@fsight.ru)
	 или через [сервисы
	 технической поддержки](https://www.fsight.ru/support/), которые доступны после регистрации на сайте.


	Для обновления лицензии в настольном приложении:


		- Объедините используемую лицензию с полученной единицей лицензирования.
		 Для этого откройте с помощью любого текстового редактора используемый
		 файл лицензии и полученный, скопируйте их содержимое в один файл
		 и сохраните с именем lservrc без расширения.


	Совет. Используемую
	 лицензию можно найти по следующему пути: %PROGRAMDATA%\Foresight\Foresight Analytics Platform\


		- Примените объединенный файл лицензии одним из способов:


			- подмените файл лицензии по следующему пути: %PROGRAMDATA%\Foresight\Foresight Analytics Platform\


			- вызовите мастер активации с помощью команды в главном
			 меню Windows «Форсайт. Аналитическая платформа
			 10.9
			 > Менеджер лицензий». В открывшемся окне выберите
			 в списке «Лицензия»
			 единицу лицензирования «Форсайт. Аналитическая платформа»,
			 нажмите напротив нее кнопку «Активировать».
			 В открывшемся мастере активации выберите объединенный файл
			 лицензии. Для завершения активации нажмите кнопку «Активировать».


	Для работы в веб-приложении активируйте «Форсайт. Аналитическая платформа»
	 на BI-сервере. Для этого обратитесь к администратору BI-сервера.

	Проблема


	При запуске продукта «Форсайт. Аналитическая платформа»
	 появляется сообщение о неправильном коде активации:


	Файл лицензии не содержит
	 правильного кода активации.


	Причина


	Возможные причины:


		- Если используется временная лицензия, то, возможно, время,
		 установленное на рабочей станции, не попадает в промежуток, определенный
		 в лицензии.


		- Вместо автономной лицензии выбрана сетевая лицензия или
		 наоборот.


		- Цифровой отпечаток компьютера, на котором активируется приложение,
		 не совпадает со цифровым отпечатком, который задан в лицензии.


		- Лицензия содержит код активации от предыдущих версий платформы.


	Решение


	Во всех случаях требуется замена файла лицензии. Для получения новой
	 лицензии обратитесь в техническую поддержку по адресу [support@fsight.ru](mailto:support@fsight.ru)
	 или через [сервисы
	 технической поддержки](https://www.fsight.ru/support/), которые доступны после регистрации на сайте.
