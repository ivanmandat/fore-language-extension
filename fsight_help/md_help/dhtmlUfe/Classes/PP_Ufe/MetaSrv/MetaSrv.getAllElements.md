# MetaSrv.getAllElements


## Синтаксис


getAllElements(dim: PP.Exp.PvtDimension, args: Object,
 callback: Function | PP.Delegate, errorCallback: Function | PP.Delegate);


## Параметры


dim. Измерение, из которого
 извлекаются элементы;


args. Аргументы;


callback. Функция обратного
 вызова;


errorCallback. Функция обратного
 вызова после ошибки.


## Описание


Метод getAllElements возвращает
 все элементы измерения.
