# MetaSrv.ChildLoaded


## Синтаксис


ChildLoaded: function(sender, args);


## Параметры


sender. Источник события;


args. Информация
 о событии. Аргументы: MetaDim
 - модель дерева метаданных; OpArgs - переданные аргументы, предназначенные
 для загрузки элементов.


## Описание


Событие ChildLoaded наступает
 после загрузки дочерних элементов в дерево.
