# MetaSource.getDefaultFilter


## Синтаксис


getDefaultFilter();


## Описание


Метод getDefaultFilter возвращает
 базовый фильтр элементов модели дерева.


## Комментарии


Возвращаемое значение - экземпляр класса PP.Mb.ElsFilter.
