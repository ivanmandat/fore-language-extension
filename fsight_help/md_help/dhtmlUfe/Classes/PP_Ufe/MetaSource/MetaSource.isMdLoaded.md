# MetaSource.isMdLoaded


## Синтаксис


isMdLoaded();


## Описание


Метод isMdLoaded проверяет,
 все ли необходимые данные содержит модель дерева.


## Комментарии


Метод возвращает значение true,
 если модель содержит все необходимые данные, иначе - значение false.
