# MetaSource.AllowHasData


## Синтаксис


AllowHasData: Boolean;


## Описание


Свойство AllowHasData определяет,
 будет ли проводиться проверка наличия новых данных для модели.


## Комментарии


Допустимые значения:


	- true.
	 Проверка наличия новых данных будет проводиться;


	- false.
	 Проверка наличия новых данных проводиться не будет (по умолчанию).


Значение свойства устанавливается с помощью метода setAllowHasData,
 а возвращается с помощью метода getAllowHasData.
 Из JSON задать значение нельзя.
