# MetaSource.getElem


## Синтаксис


getElem(key: String, filter: PP.Mb.ElsFilter, odId:
 Object);


## Параметры


key. Ключ возвращаемого элемента.
 Если данный параметр не указан, то будет возвращен корневой элемент источника
 данных.


filter. Фильтр, которому удовлетворяют
 элементы.


odId. Моникёр измерения.


## Описание


Метод getElem возвращает элемент
 модели по ключу.


## Пример


Для выполнения примера необходимо наличие на html-странице объекта типа
 [MetaTree](../../PP_Ufe_Ui/MetaTree/MetaTree.htm) с наименованием
 «metaTree» (см. «[Конструктор
 MetaTree](../../PP_Ufe_Ui/MetaTree/Constructor_MetaTree.htm)»). Проверим, имеет ли модель все необходимые для работы метаданные
 и получим наименование первого дочернего элемента модели:


// Получаем модель дерева метаданных
var metaSource = metaTree.getSource();
// Проверяем, содержит ли модель все необходимые данные
if (metaSource.isMdLoaded()) {
    console.log("Модель содержит все необходимые данные");
} else {
    console.log("Модель получила не все данные");
}
// Получаем наименование первого дочернего элемента модели
console.log("Наименование первого дочернего элемента модели: " + metaSource.getElem().children[0].n);
В результате в консоль будет выведен результат проверки и наименование
 первого дочернего элемента модели:


Модель содержит все необходимые данные


Наименование первого дочернего элемента модели:
 express1
