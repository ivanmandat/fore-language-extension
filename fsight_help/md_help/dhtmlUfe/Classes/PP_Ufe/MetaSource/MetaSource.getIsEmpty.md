# MetaSource.getIsEmpty


## Синтаксис


getIsEmpty();


## Описание


Метод getIsEmpty проверяет наличие
 элементов в модели дерева метаданных.


## Комментарии


Метод возвращает значение true,
 если модель не содержит элементов, иначе - значение false.
