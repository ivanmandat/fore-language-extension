# MetaSource.IsDataLoading


## Синтаксис


IsDataLoading: Boolean;


## Описание


Свойство IsDataLoading определяет,
 все ли элементы модели дерева загружены.


## Комментарии


Допустимые значения:


	- true.
	 Не все элементы модели загружены (по умолчанию);


	- false.
	 Все элементы модели загружены.


Значение свойства устанавливается с помощью метода setIsDataLoading,
 а возвращается с помощью метода getIsDataLoading.
 Из JSON задать значение нельзя.
