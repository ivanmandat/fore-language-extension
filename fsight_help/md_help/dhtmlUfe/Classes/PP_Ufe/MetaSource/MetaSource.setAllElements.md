# MetaSource.setAllElements


## Синтаксис


setAllElements(els: Array, filter: PP.Mb.ElsFilter,
 odId: Object);


## Параметры


els. Загружаемые элементы;


filter. Фильтр, которому соответствуют
 загруженные элементы;


odId. Моникёр отметки загружаемых
 элементов.


## Описание


Метод setAllElements добавляет
 новые элементы в модель дерева метаданных.
