# MetaSource.AllowSelection


## Синтаксис


AllowSelection: Boolean;


## Описание


Свойство AllowSelection определяет
 установку отметки элементам модели дерева метаданных.


## Комментарии


Допустимые значения:


	- true.
	 Отметку можно устанавливать элементам модели;


	- false.
	 Отметку нельзя устанавливать элементам модели (по умолчанию).


Значение свойства устанавливается с помощью метода setAllowSelection,
 а возвращается с помощью метода getAllowSelection.
 Из JSON задать значение нельзя.
