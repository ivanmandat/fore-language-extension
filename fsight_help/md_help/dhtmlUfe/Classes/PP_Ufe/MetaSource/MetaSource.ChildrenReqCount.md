# MetaSource.ChildrenReqCount


## Синтаксис


ChildrenReqCount: Number;


## Описание


Свойство ChildrenReqCount определяет
 количество вершин дерева, которые будут загружены после его создания.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setChildrenReqCount,
 а возвращается с помощью метода getChildrenReqCount.


По умолчанию для свойства установлено значение 100.
