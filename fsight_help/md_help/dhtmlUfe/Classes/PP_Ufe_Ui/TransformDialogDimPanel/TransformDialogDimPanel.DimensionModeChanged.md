# TransformDialogDimPanel.DimensionModeChanged


## Синтаксис


DimensionModeChanged: function(sender, args);


## Параметры


sender. Источник события;


args. Информация
 о событии. Аргументы: DimIndex
 - индекс открытого измерения; Mode - режим отображения, при значении Parameters отображаются параметры, при значении
 DimElements отображаются элементы измерения.


## Описание


Событие DimensionModeChanged
 наступает после изменения режима отображения элементов дерева панели.
