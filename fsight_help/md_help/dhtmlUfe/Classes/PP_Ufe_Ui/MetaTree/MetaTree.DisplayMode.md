# MetaTree.DisplayMode


## Синтаксис


DisplayMode: PP.Ufe.[MetaTreeDisplayMode](../../../Enums/PP.Ufe.MetaTreeDisplayMode.htm);


## Описание


Свойство DisplayMode определяет
 режим отображения вершин дерева метаданных.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDisplayMode,
 а возвращается с помощью метода getDisplayMode.
