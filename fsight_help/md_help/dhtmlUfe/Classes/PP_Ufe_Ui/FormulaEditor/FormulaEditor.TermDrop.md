# FormulaEditor.TermDrop


## Синтаксис


TermDrop: function(sender, args);


## Параметры


sender. Источник
 события;


args. Информация о событии.
 Аргументы: FormulaEditor - редактор
 выражений,в который добавляется терм, ddData
 - информация о добавляемом терме.


## Описание


Событие TermDrop наступает после
 добавления терма из списков функций или операторов.

Пример использования события приведен в описании события [FormulaEditor.TermsChanged](FormulaEditor.TermsChanged.htm).
