# FormulaEditor.MinCenterColumnWidth


## Синтаксис


MinCenterColumnWidth: Number;


## Описание


Свойство MinCenterColumnWidth
 определяет минимальную ширину центрального столбца редактора выражения.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMinCenterColumnWidth,
 а возвращается с помощью метода getMinCenterColumnWidth.
