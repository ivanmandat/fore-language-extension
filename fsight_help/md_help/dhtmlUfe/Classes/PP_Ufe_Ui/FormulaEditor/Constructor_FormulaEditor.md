# Конструктор FormulaEditor


## Синтаксис


PP.Ui.FormulaEditor(settings);


## Параметры


settings. JSON-объект со значениями
 свойств компонента.


## Описание


Конструктор FormulaEditor создает
 экземпляр компонента FormulaEditor.
