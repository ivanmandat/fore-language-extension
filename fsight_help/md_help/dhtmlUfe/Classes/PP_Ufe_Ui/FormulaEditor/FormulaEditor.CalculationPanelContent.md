# FormulaEditor.CalculationPanelContent


## Синтаксис


CalculationPanelContent: Object;


## Описание


Свойство CalculationPanelContent
 определяет содержимое панели расчета параметров редактора выражения.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setCalculationPanelContent,
 а возвращается с помощью метода getCalculationPanelContent.
