# FormulaEditor.DescriptionTemplate


## Синтаксис


DescriptionTemplate: String;


## Описание


Свойство DescriptionTemplate
 определяет шаблон описания операторов и функций редактора выражения.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDescriptionTemplate,
 а возвращается с помощью метода getDescriptionTemplate.
