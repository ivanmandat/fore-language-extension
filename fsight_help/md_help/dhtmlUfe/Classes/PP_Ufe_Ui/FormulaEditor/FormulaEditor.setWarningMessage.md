# FormulaEditor.setWarningMessage


## Синтаксис


setWarningMessage(msg: String, pos: Number);


## Параметры


msg. Сообщение устанавливаемое
 в кнопку;


pos. Позиция в редакторе термов, в которую будет установлен курсор после
 нажатия на кнопку.


## Описание


Метод setWarningMessage отображает
 кнопку «Предупреждение» в редакторе выражения.

Пример использования метода приведен в описании свойства [FormulaEditor.NamingTextBox](FormulaEditor.NamingTextBox.htm).
