# FormulaEditor.IsNamingVisible


## Синтаксис


IsNamingVisible: Boolean;


## Описание


Свойство IsNamingVisible определяет,
 будет ли отображаться контейнер «Наименование».


## Комментарии


Допустимые значения:


	- true.
	 Контейнер «Наименование» будет отображен;


	- false.
	 Контейнер «Наименование» не будет отображен (по умолчанию).


Значение свойства устанавливается из JSON и с помощью метода setIsNamingVisible,
 а возвращается с помощью метода getIsNamingVisible.
