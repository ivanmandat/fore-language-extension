# FormulaEditor.InitialTerms


## Синтаксис


InitialTerms: Array;


## Описание


Свойство InitialTerms определяет
 исходные термы, которые будут добавлены в редактор выражения во время
 инициализации компонента.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setInitialTerms,
 а возвращается с помощью метода getInitialTerms.
