# FormulaEditor.setFilterPanelVisibility


## Синтаксис


setFilterPanelVisibility(value: Boolean);


## Параметры


value. Значение признака видимости.
 Если для параметра установлено значение true,
 то панель станет видимой.


## Описание


Метод setFilterPanelVisibility
 устанавливает признак видимости панели фильтрации.
