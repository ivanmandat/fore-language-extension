# FormulaEditor.addFunction


## Синтаксис


addFunction(settings: Object);


## Параметры


settings. JSON-объект с параметрами
 добавляемой функции. Объект должен содержать следующие поля: Text
 - текстовое значение функции; Formula
 - формула; Desc - описание формулы.


## Описание


Метод addFunction добавляет
 функцию в список функций.
