# FormulaEditor.LeftColumnContent


## Синтаксис


LeftColumnContent: Object;


## Описание


Свойство LeftColumnContent определяет
 содержимое левого столбца редактора выражения.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLeftColumnContent,
 а возвращается с помощью метода getLeftColumnContent.
