# FormulaEditor.getCalcParamsContainer


## Синтаксис

## Описание


Метод getCalcParamsContainer возвращает
 панель расчета параметров редактора выражения.


## Комментарии


Возвращаемое значение - экземпляр PP.Ui.Spoiler.
