# FormulaEditor.CaretPosition


## Синтаксис


CaretPosition: Number;


## Описание


Свойство CaretPosition устанавливает
 позицию курсора в редакторе выражения.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setCaretPosition,
 а возвращается с помощью метода getCaretPosition.
