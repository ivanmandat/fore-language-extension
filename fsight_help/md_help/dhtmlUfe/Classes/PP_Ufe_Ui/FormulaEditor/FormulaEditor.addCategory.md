# FormulaEditor.addCategory


## Синтаксис


addCategory(settings: Object);


## Параметры


settings. JSON-объект с параметрами
 добавляемой категории. Объект должен содержать следующие поля: Key
 - ключ категории; Text - текстовое
 значение категории.


## Описание


Метод addCategory добавляет
 категорию в список функций.
