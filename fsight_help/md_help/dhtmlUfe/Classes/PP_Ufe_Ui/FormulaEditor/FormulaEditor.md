## Описание


Класс FormulaEditor реализует
 компонент [FormulaEditor](../../../Components/FormulaEditor/FormulaEditor.htm),
 представляющий собой редактор выражения.


## Синтаксис


PP.initClass(PP.Ufe.Ui.FormulaEditor, PP.Ui.[Control](dhtmlUi.chm::/Classes/control/control.htm),
 "PP.Ufe.Ui.FormulaEditor");
