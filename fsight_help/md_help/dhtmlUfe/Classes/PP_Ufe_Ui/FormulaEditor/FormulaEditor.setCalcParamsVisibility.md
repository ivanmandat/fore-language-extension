# FormulaEditor.setCalcParamsVisibility


## Синтаксис


setCalcParamsVisibility(value: Boolean);


## Параметры


value. Значение признака видимости.
 Если для параметра установлено значение true,
 то панель станет видимой.


## Описание


Метод setCalcParamsVisibility
 устанавливает признак видимости панели расчета параметров.
