# FormulaEditor.InitialOperators


## Синтаксис


InitialOperators: Array;


## Описание


Свойство InitialOperators определяет
 исходные операторы, которые будут добавлены в редактор выражения во время
 инициализации компонента.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setInitialOperators,
 а возвращается с помощью метода getInitialOperators.
