## Синтаксис


getFilterMenuButton();


## Описание


Метод getFilterMenuButton возвращает
 кнопку фильтрации редактора выражения.


## Комментарии


Возвращаемое значение - объект класса PP.Ui.[RibbonButton](dhtmlRibbon.chm::/Classes/RibbonButton/RibbonButton.htm).
