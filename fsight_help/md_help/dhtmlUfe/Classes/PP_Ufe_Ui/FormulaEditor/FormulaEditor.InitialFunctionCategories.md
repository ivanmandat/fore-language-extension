# FormulaEditor.InitialFunctionCategories


## Синтаксис


InitialFunctionCategories: Array;


## Описание


Свойство InitialFunctionCategories
 определяет исходные категории функций, которые будут добавлены в редактор
 выражения во время инициализации компонента.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setInitialFunctionCategories,
 а возвращается с помощью метода getInitialFunctionCategories.
