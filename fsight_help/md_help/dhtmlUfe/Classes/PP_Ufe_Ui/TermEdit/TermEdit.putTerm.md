# TermEdit.putTerm


## Синтаксис


putTerm (term, position);


## Параметры


term. Терм, который будет помещаться
 в коллекцию;


position. Необязательный
 параметр, определяющий позицию, в которую будет помещаться терм.


## Описание


Метод putTerm добавляет терм
 в редактор термов.
