# TermEdit.CaretPos


## Синтаксис


CaretPos: Number;


## Описание


Свойство CaretPos определяет
 позицию курсора в редакторе термов.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setCaretPos,
 а возвращается с помощью метода getCaretPos.


Свойство аналогично свойству [TermEdit.CaretPosition](TermEdit.CaretPosition.htm).


## Пример

 пример использования которого приведено на странице [Term.Splitted](../Term/Term.Splitted.htm).
