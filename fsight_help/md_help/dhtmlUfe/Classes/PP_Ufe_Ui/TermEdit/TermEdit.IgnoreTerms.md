# TermEdit.IgnoreTerms


## Синтаксис


IgnoreTerms: Array;


## Описание


Свойство IgnoreTerms определяет
 термы, которые будут проигнорированы при выводе формулы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIgnoreTerms,
 а возвращается с помощью метода getIgnoreTerms.
