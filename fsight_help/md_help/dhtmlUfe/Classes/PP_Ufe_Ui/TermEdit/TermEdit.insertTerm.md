# TermEdit.insertTerm


## Синтаксис


insertTerm(value: String || Object || PP.Ui.[Term](../Term/Term.htm),
 index: Number);


## Параметры


value. Добавляемая строка,
 объект с параметрами терма или объект класса PP.Ui.[Term](../Term/Term.htm);


index. Позиция элемента, после
 которого необходимо произвести вставку.


## Описание


Метод insertTerm добавляет строку
 или терм в редактор термов.

Метод находится в разработке.
