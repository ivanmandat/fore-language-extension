# TermEdit.putLiteral


## Синтаксис


putLiteral(text, position);


## Параметры


text. Литеральный терм, который
 будет помещаться в коллекцию;


position. Необязательный параметр,
 определяющий позицию, в которую будет помещаться литеральный терм.


## Описание


Метод putLiteral добавляет литеральный
 терм в редактор термов.


## Комментарии


Под литеральным термом понимается терм, для свойства Term.IsLiteral
 которого установлено значение true.
