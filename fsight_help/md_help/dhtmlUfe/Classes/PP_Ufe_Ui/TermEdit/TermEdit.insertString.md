# TermEdit.insertString


## Синтаксис


insertString(string: String, pos: Number, isUpdateTerms:
 Boolean);


## Параметры


string. Строка с термами;


pos. Позиция элемента, после
 которого необходимо произвести вставку;


isUpdateTerms. Признак обновления
 содержимого редактора термов после вставки. Если для параметра установлено
 значение true, то редактор термов
 будет обновлен. По умолчанию установлено значение параметра true.


## Описание


Метод insertString добавляет
 строку с термами в редактор термов.
