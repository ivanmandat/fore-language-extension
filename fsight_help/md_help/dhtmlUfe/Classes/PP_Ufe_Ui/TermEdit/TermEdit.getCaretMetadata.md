# TermEdit.getCaretMetadata


## Синтаксис


getCaretMetadata();


## Описание


Метод getCaretMetadata возвращает
 данные о курсоре.


## Комментарии


Возвращаемый объект содержит следующие поля: Position
 - позиция курсора относительно всего выражения; Offset
 - позиция курсора относительно найденного терма; Term
 - терм, в области которого находится курсор; Index
 - позиция текущего терма или следующего, если курсор вне какого-либо терма.
