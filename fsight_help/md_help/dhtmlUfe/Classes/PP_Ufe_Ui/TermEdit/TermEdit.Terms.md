# TermEdit.Terms


## Синтаксис


Terms: Array;


## Описание


Свойство Terms устанавливает
 список термов.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTerms,
 а возвращается с помощью метода getTerms.
