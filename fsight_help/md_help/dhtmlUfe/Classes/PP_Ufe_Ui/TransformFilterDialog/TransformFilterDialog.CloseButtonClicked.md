# TransformFilterDialog.CloseButtonClicked


## Синтаксис


CloseButtonClicked: function(sender, args);


## Параметры


sender. Источник события;


args. Информация
 о событии. Аргументы: Break
 - признак отмены действия, если для аргумента установлено значение true,
 то действие было отменено; Element - элемент, у которого вызвано событие;
 Event - информация о событии.


## Описание


Событие CloseButtonClicked наступает
 после нажатия на кнопку закрытия окна редактирования формул.
