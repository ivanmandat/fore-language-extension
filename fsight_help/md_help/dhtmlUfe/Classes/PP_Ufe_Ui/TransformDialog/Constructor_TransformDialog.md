# Конструктор TransformDialog


## Синтаксис


PP.Ufe.Ui.TransformDialog(settings);


## Параметры


settings. JSON-объект со значениями
 свойств компонента.


## Описание


Конструктор TransformDialog
 создает экземпляр компонента TransformDialog.
