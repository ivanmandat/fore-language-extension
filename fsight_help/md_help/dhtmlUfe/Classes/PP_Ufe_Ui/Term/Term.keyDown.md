# Term.keyDown


## Синтаксис


keyDown(event: Event, offset: Number);


## Параметры


event. Событие, которое необходимо
 обработать;


offset. Позиция курсора.


## Описание


Метод keyDown обрабатывает событие
 зажатия клавиши клавиатуры.

Пример использования метода приведен в описании события [Term.StartBackspacePressed](Term.StartBackspacePressed.htm).
