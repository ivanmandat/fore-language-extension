# Term.EndOffset


## Синтаксис


EndOffset: Number;


## Описание


Свойство EndOffset содержит
 позицию последнего элемента терма в выражении.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setEndOffset,
 а возвращается с помощью метода getEndOffset.
