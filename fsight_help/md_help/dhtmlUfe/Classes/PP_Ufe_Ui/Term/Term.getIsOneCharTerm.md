# Term.getIsOneCharTerm


## Синтаксис


getIsOneCharTerm();


## Описание


Метод getIsOneCharTerm проверяет,
 является ли терм односимвольным.


## Комментарии


Метод возвращает true, если
 терм является односимвольным, иначе false.
