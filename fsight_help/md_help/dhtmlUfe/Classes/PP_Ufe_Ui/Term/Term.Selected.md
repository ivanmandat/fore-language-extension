# Term.Selected


## Синтаксис


Selected: Boolean;


## Описание


Свойство Selected определяет,
 выделен терм или нет.


## Комментарии


Допустимые значения:


	- true.
	 Терм выделен;


	- false.
	 Терм не выделен (по умолчанию).


Значение свойства устанавливается из JSON и с помощью метода setSelected,
 а возвращается с помощью метода getSelected.
