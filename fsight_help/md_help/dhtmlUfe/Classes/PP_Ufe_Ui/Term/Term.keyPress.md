# Term.keyPress


## Синтаксис


keyPress(event: Event, offset: Number);


## Параметры


event. Событие, которое необходимо
 обработать;


offset. Позиция курсора.


## Описание


Метод keyPress обрабатывает
 событие нажатия клавиши клавиатуры.
