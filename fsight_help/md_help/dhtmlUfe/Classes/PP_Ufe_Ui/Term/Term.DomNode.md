# Term.DomNode


## Синтаксис


DomNode: Boolean;


## Описание


Свойство DomNode устанавливает
 DOM-вершину терма.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDomNode,
 а возвращается с помощью метода getDomNode.


Используется для отображения подсказки при автодополнении.

Пример использования свойства приведен в описании метода [TermEdit.addACTerms](../TermEdit/TermEdit.addACTerms.htm).
