# Term.StartOffset


## Синтаксис


StartOffset: Number


## Описание


Свойство StartOffset содержит
 позицию первого элемента терма в выражении.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setStartOffset,
 а возвращается с помощью метода getStartOffset.
