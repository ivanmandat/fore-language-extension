# Term.slice


## Синтаксис


slice();


## Описание


Метод slice переопределяет нативный
 метод slice для манипулирования содержимым терма.


## Пример


Для выполнения примера необходимо наличие ссылок на файлы сценариев
 PP.js и PP.Ufe.js. Приведенный ниже код выполняется в консоли браузера.
 Создадим терм с содержимым «Term» и получаем первые два символа:


// Создаем терм
var term = new PP.Ui.Term({
    Content: "Term"
});
// Получаем подстроку
console.log(term.slice(0, 2));
В результате в консоль будет выведены первые два символа терма:


"Te"
