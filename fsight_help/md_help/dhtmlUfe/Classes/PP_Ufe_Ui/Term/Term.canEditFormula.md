# Term.canEditFormula


## Синтаксис


canEditFormula();


## Описание


Метод canEditFormula определяет,
 можно ли редактировать формулу терма.


## Комментарии


Метод возвращает значение true,
 если редактирование формулы терма разрешено, иначе - значение false.
