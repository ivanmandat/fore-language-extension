# WbkTransformDialog.ApplyFormula


## Синтаксис


ApplyFormula: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные
 аргументы: DialogResult - объект с данными, установленными в редакторе
 выражения.


## Описание


Событие ApplyFormula наступает
 после применения формулы в соответствующем диалоге рабочей книги.
