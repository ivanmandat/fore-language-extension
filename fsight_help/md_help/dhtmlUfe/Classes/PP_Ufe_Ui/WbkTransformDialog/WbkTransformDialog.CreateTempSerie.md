# WbkTransformDialog.CreateTempSerie


## Синтаксис


CreateTempSerie: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные
 аргументы: DialogResult - объект с данными, установленными в редакторе
 выражения.


## Описание


Событие CreateTempSerie наступает
 после установки в редакторе выражения ряда данных по умолчанию.
