# IUiDataSet.Active


## Синтаксис


Active: Boolean;


## Описание


Свойство Active определяет,
 будет ли активным компонент, т.е. будут ли данные из источника передаваться
 в компонент [UiDataSource](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiDataSource.htm).


## Комментарии


Для открытия доступа к наборам данных необходимо свойству установить
 значение True. Если свойству установлено
 значение False, компонент [UiDataSource](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiDataSource.htm)
 будет связан с источником данных, но передаваться данные в компонент не
 будут.
