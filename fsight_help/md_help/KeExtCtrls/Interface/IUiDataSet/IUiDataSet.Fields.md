# IUiDataSet.Fields


## Синтаксис


Fields: [IUiFields](../IUiFields/IUiFields.htm);


## Описание


Свойство Fields возвращает коллекцию
 полей набора данных.


## Комментарии


В зависимости от используемого источника данных коллекция может состоять
 из полей исходного реляционного источника, либо из созданных новых полей.
 Добавление новых полей осуществляется в окне «Список полей», которое вызывается
 щелчком кнопки мыши на кнопке ![](../knopka_dialog.gif)
 свойства Fields компонента в [инспекторе
 объектов](UiDevEnv.chm::/01_Development_Environment/03_Windows_of_Development_Environment/Object_Inspector.htm).
