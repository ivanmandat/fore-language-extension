# IUiDataSet.IsEmpty


## Синтаксис


IsEmpty: Boolean;


## Описание


Метод IsEmpty осуществляет проверку
 наличия данных. Если в источнике данных имеется хотя бы одна запись
 - метод возвращает значение True.
