# IUiDataSet.Eof


## Синтаксис


Eof: Boolean;


## Описание


Свойство Eof возвращает признак
 расположения курсора в источнике данных на последней записи.


## Комментарии


Если курсор установлен на последней записи, то свойство возвращает значение
 True, иначе - False.
