# IUiDataSet.Bof


## Синтаксис


Bof: Boolean;


## Описание


Свойство Bof возвращает признак
 расположения курсора в источнике данных на первой записи. Если курсор
 установлен на первой записи, то свойство возвращает значение True,
 иначе - False.
