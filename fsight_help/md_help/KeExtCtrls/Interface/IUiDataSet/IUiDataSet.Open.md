# IUiDataSet.Open


## Синтаксис


Open([Values: [IMetabaseObjectParamValues](KeSom.chm::/Interface/IMetabaseObjectParamValues/IMetabaseObjectParamValues.htm)
 = Null]);


## Параметры


Values. Значения параметров,
 используемых при открытии источника данных.


## Описание


Метод Open осуществляет открытие
 источника данных, при этом значение свойства [Active](IUiDataSet.Active.htm)
 устанавливается в True и появляется
 возможность изменения данных.
