# IUiDataSet.Filtered


## Синтаксис


Filtered: Boolean;


## Описание


Свойство Filtered определяет
 признак фильтрации источника данных.


## Комментарии


При установке свойству значения True,
 будет включена фильтрация при этом будут отображаться данные, удовлетворяющие
 условию фильтра. При установке False
 фильтрация отключена. Текст текущего фильтра задается свойством [Filter](IUiDataSet.Filter.htm).
