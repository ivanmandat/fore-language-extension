# IUiDataSet.MoveBy


## Синтаксис


MoveBy(Value: Integer): Integer;


## Параметры


Value.
 Количество
 записей, на которое перемещается курсор.


## Описание


Метод MoveBy осуществляет перемещение
 курсора в источнике данных и возвращает количество записей, на которое
 переместился курсор.


## Комментарии


Если параметр Value больше 0,
 то курсор перемещается вперед, если меньше 0 - назад. При достижении конца/начала
 источника данных курсор останавливается. При этом свойствам [IUiDataSet.Bof](IUiDataSet.Bof.htm)/[IUiDataSet.Eof](IUiDataSet.Eof.htm)
 устанавливается значение True,
 а метод MoveBy при этом вернет
 значение меньше, чем указано в параметре Value.
