# IDataGridStyle.ImageAlign


## Синтаксис


ImageAlign: [GxImageAlign](ModDrawing.chm::/Enums/GxImageAlign.htm);


## Описание


Свойство ImageAlign определяет
 положение изображения относительно текста в ячейке.


## Комментарии


Для указания используемого изображения используйте свойство [IDataGridRows.ImageIndex](../IDataGridRows/IDataGridRows.ImageIndex.htm).
 Для использования изображений в ячейках у компонента должно быть установлено
 свойство [IDataGrid.Images](../IDataGrid/IDataGrid.Images.htm).
