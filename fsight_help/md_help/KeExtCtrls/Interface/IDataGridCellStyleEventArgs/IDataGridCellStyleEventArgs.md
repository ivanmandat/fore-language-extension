# IDataGridCellStyleEventArgs


Сборка: ExtCtrls;


## Описание


Интерфейс IDataGridCellStyleEventArgs
 содержит свойства аргумента события, наступающего при отрисовке ячейки
 таблицы.


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [StyleIndex](IDataGridCellStyleEventArgs.StyleIndex.htm)
		 Свойство StyleIndex
		 определяет индекс стиля, который необходимо применить для текущей
		 ячейки.
