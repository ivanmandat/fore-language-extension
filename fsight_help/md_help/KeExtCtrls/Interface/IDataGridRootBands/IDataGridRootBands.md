# IDataGridRootBands


Сборка: ExtCtrls;


## Описание


Интерфейс IDataGridRootBands
 содержит свойства для работы с фиксированными контейнерами компонента
 [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).


## Комментарии


Доступ к фиксированным контейнерам предоставляет свойство [IDataGrid.RootBands](../IDataGrid/IDataGrid.RootBands.htm).
