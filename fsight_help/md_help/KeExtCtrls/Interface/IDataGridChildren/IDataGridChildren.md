# IDataGridChildren


Сборка: ExtCtrls;


## Описание


Интерфейс IDataGridChildren
 содержит свойства дочерних объектов, у которых родительские объекты входят
 в состав компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).

		 [Item](IDataGridChildren.Item.htm)
		 Свойство Item возвращает
		 параметры дочернего объекта.
