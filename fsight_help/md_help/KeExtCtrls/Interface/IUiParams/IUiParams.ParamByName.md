# IUiParams.ParamByName


## Синтаксис


ParamByName(ParamName: String): [IUiParam](../IUiParam/IUiParam.htm);


## Параметры


ParamName.
 Параметр, определяющий наименование параметра, к которому необходимо обратиться.


## Описание


Метод ParamByName позволяет
 обратиться к параметру по его наименованию.
