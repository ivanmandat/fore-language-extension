# IUiParams.IsEqual


## Синтаксис


IsEqual(Params: [IUiParams](IUiParams.htm)):
 Boolean;


## Параметры


Params.
 Параметры, которые необходимо проверить на соответствие текущим.


## Описание


Метод IsEqual осуществляет сравнение
 текущих параметров и параметров, которые передаются в качестве входного
 значения Params, после чего метод
 возвращает признак соответствия параметров.
