# IUiParams.AddParam


## Синтаксис


AddParam(Param: [IUiParam](../IUiParam/IUiParam.htm));


## Параметры


Param. Параметр, который необходимо
 добавить в коллекцию.


## Описание


Метод AddParam осуществляет
 добавление в коллекцию параметра, передаваемого посредством параметра
 Param.
