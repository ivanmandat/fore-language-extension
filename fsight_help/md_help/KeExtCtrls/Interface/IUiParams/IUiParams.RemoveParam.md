# IUiParams.RemoveParam


## Синтаксис


RemoveParam(Param: [IUiParam](../IUiParam/IUiParam.htm));


## Параметры


Param.
 Параметр, определяющий удаляемый параметр.


## Описание


Метод RemoveParam позволяет
 удалить параметр, передаваемый посредством входного параметра Param.
