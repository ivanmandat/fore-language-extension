# IUiParams


Сборка: ExtCtrls;


## Описание


Интерфейс IUiParams содержит
 свойства и методы для работы с коллекцией параметров компонента [UiQuery](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiQuery.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IUiParams.Count.htm)
		 Свойство Count возвращает
		 количество параметров в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IUiParams.Item.htm)
		 Свойство Item возвращает
		 параметр, индекс которого передается в качестве входного значения.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddParam](IUiParams.AddParam.htm)
		 Метод AddParam осуществляет
		 добавление в коллекцию указанного параметра.


		 ![](../../Sub_Image.gif)
		 [CreateParam](IUiParams.CreateParam.htm)
		 Метод CreateParam осуществляет
		 создание параметра.


		 ![](../../Sub_Image.gif)
		 [IsEqual](IUiParams.IsEqual.htm)
		 Метод IsEqual осуществляет
		 сравнение текущих параметров и параметров, которые передаются
		 в качестве входного значения.


		 ![](../../Sub_Image.gif)
		 [ParamByName](IUiParams.ParamByName.htm)
		 Метод ParamByName позволяет
		 обратиться к параметру по его наименованию.


		 ![](../../Sub_Image.gif)
		 [RemoveParam](IUiParams.RemoveParam.htm)
		 Метод RemoveParam позволяет
		 удалить параметр.
