# IUiField.AsBoolean


## Синтаксис


AsBoolean: Boolean;


## Описание


Свойство AsBoolean определяет
 значение данного поля текущей записи, представленное в логическом типе.


## Комментарии


Если преобразование невозможно, то генерируется исключительная ситуация.
