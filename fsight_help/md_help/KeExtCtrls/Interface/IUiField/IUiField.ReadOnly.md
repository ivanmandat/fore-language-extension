# IUiField.ReadOnly


## Синтаксис


ReadOnly: Boolean;


## Описание


Свойство ReadOnly определяет,
 будет ли пользователь иметь возможность изменять содержимое поля.


## Комментарии


Если свойству установить значение True,
 то поле будет использоваться только для отображения данных, изменять содержимое
 поля будет запрещено. При установке значения False
 набор данных будет находиться в режиме редактирования, и пользователь
 будет иметь возможность редактирования содержимого поля.
