# IUiField


Сборка: ExtCtrls;


## Описание


Интерфейс IUiField содержит
 свойства и методы поля источника данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AsBoolean](IUiField.AsBoolean.htm)
		 Свойство AsBoolean
		 определяет значение данного поля текущей записи, представленное
		 в логическом типе.


		 ![](../../Property_Image.gif)
		 [AsChar](IUiField.AsChar.htm)
		 Свойство AsChar определяет
		 значение данного поля текущей записи, представленное в символьном
		 типе.


		 ![](../../Property_Image.gif)
		 [AsDateTime](IUiField.AsDateTime.htm)
		 Свойство AsDateTime
		 определяет значение данного поля текущей записи, представленное
		 как значение даты/времени.


		 ![](../../Property_Image.gif)
		 [AsDouble](IUiField.AsDouble.htm)
		 Свойство AsDouble определяет
		 значение данного поля текущей записи, представленное как вещественное
		 число.


		 ![](../../Property_Image.gif)
		 [AsInteger](IUiField.AsInteger.htm)
		 Свойство AsInteger
		 определяет значение данного поля текущей записи, представленное
		 как целое число.


		 ![](../../Property_Image.gif)
		 [AsString](IUiField.AsString.htm)
		 Свойство AsString определяет
		 значение данного поля текущей записи, представленное как символьная
		 строка.


		 ![](../../Property_Image.gif)
		 [Dataset](IUiField.Dataset.htm)
		 Свойство Dataset возвращает
		 источник данных, которому принадлежит данное поле.


		 ![](../../Property_Image.gif)
		 [DataType](IUiField.DataType.htm)
		 Свойство DataType возвращает
		 тип данных, содержащихся в данном поле источника данных.


		 ![](../../Property_Image.gif)
		 [FieldName](IUiField.FieldName.htm)
		 Свойство FieldName
		 определяет наименование поля реляционного источника данных, с
		 которым связано данное поле компонента доступа к данным.


		 ![](../../Property_Image.gif)
		 [Index](IUiField.Index.htm)
		 Свойство Index определяет
		 индекс поля, с которым связано данное поле.


		 ![](../../Property_Image.gif)
		 [IsNull](IUiField.IsNull.htm)
		 Свойство IsNull возвращает
		 признак наличия по данному полю в текущей записи пустого значения.


		 ![](../../Property_Image.gif)
		 [ReadOnly](IUiField.ReadOnly.htm)
		 Свойство ReadOnly определяет,
		 будет ли пользователь иметь возможность изменять содержимое поля
		 таблицы.


		 ![](../../Property_Image.gif)
		 [Value](IUiField.Value.htm)
		 Свойство Value определяет
		 значение данного поля в текущей записи источника данных.


## Свойства, унаследованы от [IComponent](ModForms.chm::/Interface/IComponent/IComponent.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ComponentCount](ModForms.chm::/Interface/IComponent/IComponent.ComponentCount.htm)
		 Свойство ComponentCount
		 возвращает количество дочерних компонентов, расположенных на родительском
		 компоненте.


		 ![](../../Property_Image.gif)
		 [Components](ModForms.chm::/Interface/IComponent/IComponent.Components.htm)
		 Свойство Components
		 возвращает дочерний компонент, индекс которого передается
		 посредством входного параметра.


		 ![](../../Property_Image.gif)
		 [Data](ModForms.chm::/Interface/IComponent/IComponent.Data.htm)
		 Свойство Data предназначено
		 для хранения любых пользовательских данных.


		 ![](../../Property_Image.gif)
		 [Name](ModForms.chm::/Interface/IComponent/IComponent.Name.htm)
		 Свойство Name определяет
		 наименование компонента.


		 ![](../../Property_Image.gif)
		 [Tag](ModForms.chm::/Interface/IComponent/IComponent.Tag.htm)
		 Свойство Tag не используется
		 компилятором. Пользователь может изменить значение данного свойства
		 и использовать его по своему усмотрению.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Clear](IUiField.Clear.htm)
		 Метод Clear осуществляет
		 очистку содержимого данного поля в текущей записи источника данных.
