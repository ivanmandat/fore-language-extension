# IUiField.AsChar


## Синтаксис


AsChar: Char;


## Описание


Свойство AsChar определяет значение
 данного поля текущей записи, представленное в символьном типе.


## Комментарии


Если преобразование невозможно, то генерируется исключительная ситуация.
