# IUiField.IsNull


## Синтаксис


IsNull: Boolean;


## Описание


Свойство IsNull возвращает признак
 наличия по данному полю в текущей записи пустого значения.


## Комментарии


Свойство возвращает значение True,
 если по данному полю в текущей записи содержится пустое значение и False, если имеются какие-либо данные.
