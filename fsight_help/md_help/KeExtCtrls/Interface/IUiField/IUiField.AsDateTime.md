# IUiField.AsDateTime


## Синтаксис


AsDateTime: DateTime;


## Описание


Свойство AsDateTime определяет
 значение данного поля текущей записи, представленное как значение даты/времени.


## Комментарии


Если преобразование невозможно, то генерируется исключительная ситуация.
