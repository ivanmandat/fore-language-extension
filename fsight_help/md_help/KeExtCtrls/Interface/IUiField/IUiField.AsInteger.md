# IUiField.AsInteger


## Синтаксис


AsInteger: Integer;


## Описание


Свойство AsInteger определяет
 значение данного поля текущей записи, представленное как целое число.


## Комментарии


Если преобразование невозможно, то генерируется исключительная ситуация.
