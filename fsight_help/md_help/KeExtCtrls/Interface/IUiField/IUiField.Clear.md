# IUiField.Clear


## Синтаксис


Clear;


## Описание


Метод Clear осуществляет очистку
 содержимого данного поля в текущей записи источника данных.


## Комментарии


Для очистки необходимо, чтобы источник данных находился в режиме редактирования.
