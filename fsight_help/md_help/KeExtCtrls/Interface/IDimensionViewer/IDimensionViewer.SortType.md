# IDimensionViewer.SortType


## Синтаксис


SortType: [ControlSortType](ModForms.chm::/Enums/ControlSortType.htm);


## Описание


Свойство SortType определяет
 тип сортировки для компонента.


## Комментарии


Если свойству установлено значение ControlSortType.Custom, то для компонента
 будет генерироваться событие [OnCompareItems](IDimensionViewer.OnCompareItems.htm),
 в котором осуществляется пользовательская сортировка элементов. Используется
 вместе со свойствами [Sorted](IDimensionViewer.Sorted.htm)
 и [SortColumn](IDimensionViewer.SortColumn.htm).
