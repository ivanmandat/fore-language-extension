# IDimensionViewer.OnCollapsing


## Синтаксис


OnCollapsing(Sender: Object; Args: [IDimensionAllowActionEventArgs](../IDimensionAllowActionEventArgs/IDimensionAllowActionEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnCollapsing наступает
 перед сворачиванием иерархии потомков вершины дерева.
