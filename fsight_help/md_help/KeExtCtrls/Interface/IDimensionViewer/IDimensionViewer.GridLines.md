# IDimensionViewer.GridLines


## Синтаксис


GridLines: Boolean;


## Описание


Свойство GridLines определяет,
 будет ли отображаться сетка. Если свойству установлено значение True, то в области компонента будет
 отображаться сетка. При установке значения False,
 сетки не будет.
