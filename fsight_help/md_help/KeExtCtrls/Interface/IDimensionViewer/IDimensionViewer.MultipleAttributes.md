# IDimensionViewer.MultipleAttributes


## Синтаксис


MultipleAttributes: Boolean;


## Описание


Свойство MultipleAttributes
 определяет, используется ли отображение нескольких атрибутов справочника.


## Комментарии


Допустимые значения:


	- True. Отображение используется;


	- False. Отображение не
	 используется.
