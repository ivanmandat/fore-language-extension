# IDimensionViewer.OnCollapsed


## Синтаксис


OnCollapsed(Sender: Object; Args: [IDimensionNodeEventArgs](../IDimensionNodeEventArgs/IDimensionNodeEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnCollapsed наступает
 после сворачивания иерархии потомков вершины дерева.


## Комментарии


Событие OnCollapsed наступает
 только в том случае, если произошло событие [OnCollapsing](IDimensionViewer.OnCollapsing.htm).
