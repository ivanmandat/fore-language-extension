# IDimensionViewer.OnGetHint


## Синтаксис


OnGetHint(Sender: Object; Args: [IDimensionViewerGetHintEventArgs](../IDimensionViewerGetHintEventArgs/IDimensionViewerGetHintEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnGetHint наступает
 при появлении всплывающей подсказки над элементом, над которым остановился
 курсор мыши.
