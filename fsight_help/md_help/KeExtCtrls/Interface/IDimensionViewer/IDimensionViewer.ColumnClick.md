# IDimensionViewer.ColumnClick


## Синтаксис


ColumnClick: Boolean;


## Описание


Свойство ColumnClick определяет
 возможность отображения заголовков столбцов в виде кнопок.


## Комментарии


Если свойству установлено значение True,
 то заголовок примет форму кнопки, нажатие на которую будет разрешено.
 При нажатии на заголовок будет генерироваться событие [OnColumnClick](IDimensionViewer.OnColumnClick.htm).
