# IDimensionViewer.OnCompareItems


## Синтаксис


OnCompareItems(Sender: Object; Args: [IDimensionViewerCompareEventArgs](../IDimensionViewerCompareEventArgs/IDimensionViewerCompareEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnCompareItems наступает
 при сравнении двух элементов во время пользовательской сортировки элементов.
