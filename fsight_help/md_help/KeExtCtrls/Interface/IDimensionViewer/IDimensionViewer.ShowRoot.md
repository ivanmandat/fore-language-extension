# IDimensionViewer.ShowRoot


## Синтаксис


ShowRoot: Boolean;


## Описание


Свойство ShowRoot определяет
 признак отображения кнопок разворачивания и сворачивания корневых элементов
 деревьев.


## Комментарии


Если свойству установлено значение True,
 то слева от корневых элементов будут располагаться кнопки «+» и «-». При
 установке значения False, разворачивание
 и сворачивание дерева потомков вершин будет производиться двойным щелчком
 мыши на вершине.
