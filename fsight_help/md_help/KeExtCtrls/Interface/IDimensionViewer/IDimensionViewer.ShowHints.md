# IDimensionViewer.ShowHints


## Синтаксис


ShowHints: Boolean;


## Описание


Свойство ShowHints определяет,
 будет ли появляться подсказка при задержке мыши над элементами дерева
 компонента.


## Комментарии


Если свойству установлено значение True,
 то при задержке мыши над элементом появится небольшая рамка, в которой
 располагается текст подсказки. Текст подсказки соответствует тексту элемента
 компонента. При установке значения False,
 подсказки не будет. Свойство актуально в том случае, когда текст элемента
 компонента полностью не помещается в окне компонента.
