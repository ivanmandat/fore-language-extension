# IDimensionViewer.MakeVisible


## Синтаксис


MakeVisible(Node: Integer; [SetFocus: Boolean =
 False]);


## Параметры


Node. Входной параметр, определяющий
 номер элемента, который необходимо сделать видимым;


SetFocus. Входной параметр,
 определяющий признак установки фокуса для элемента Node.
 При установке параметру значения True
 - фокус устанавливается, при установке False
 - фокус не устанавливается.


## Описание


Метод MakeVisible осуществляет
 прокрутку дерева элементов таким образом, чтобы элемент Node
 попал в видимую область компонента.
