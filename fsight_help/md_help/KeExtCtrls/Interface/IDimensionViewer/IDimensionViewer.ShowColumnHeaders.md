# IDimensionViewer.ShowColumnHeaders


## Синтаксис


ShowColumnHeaders: Boolean;


## Описание


Свойство ShowColumnHeaders определяет,
 будут ли отображаться заголовки столбцов.


## Комментарии


Если свойству установлено значение True,
 то заголовки столбцов будут отображаться. При установке значения False, столбцы не будут иметь заголовков.
