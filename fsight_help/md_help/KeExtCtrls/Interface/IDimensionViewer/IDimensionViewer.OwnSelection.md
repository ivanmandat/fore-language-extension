# IDimensionViewer.OwnSelection


## Синтаксис


OwnSelection: Boolean;


## Описание


Свойство OwnSelection определяет
 признак использования собственной отметки элементов в компоненте.


## Комментарии


По умолчанию свойству установлено значение False,
 при этом отметка данного компонента будет соответствовать отметке в компоненте
 [UiDimension](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiDimension.htm).
 Если свойству установить значение True,
 то будет создаваться копия отметки независящая от отметки элементов в
 компоненте [UiDimension](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiDimension.htm).
