# IDimensionViewer.Collapse


## Синтаксис


Collapse(Node: Integer; [Recursively: Boolean =
 False]);


## Параметры


Node. Вершина,
 иерархия потомков которой будет свернута;


Recursively. Признак рекурсивного
 сворачивания иерархии потомков. Если параметр принимает значение True, то происходит сворачивание
 всей иерархии всех потомков. При значении False
 произойдет сворачивание иерархии только выбранного элемента.


## Описание


Метод Collapse осуществляет
 сворачивание иерархии потомков дерева.
