# IDimensionViewer.Expand


## Синтаксис


Expand(Node: Integer; [Recursively: Boolean = False]);


## Параметры


Node. Вершина,
 иерархия потомков которой будет развернута;


Recursively. Признак рекурсивного
 разворачивания иерархии потомков. Если параметр принимает значение True, то происходит разворачивание
 всей иерархии всех потомков. При значении False
 произойдет разворачивание иерархии только выбранного элемента.


## Описание


Метод Expand осуществляет разворачивание
 иерархии потомков дерева.
