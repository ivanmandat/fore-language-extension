# IDimensionViewer.OnChecking


## Синтаксис


OnChecking(Sender: Object; Args: [IDimensionNodeCheckingEventArgs](../IDimensionNodeCheckingEventArgs/IDimensionNodeCheckingEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnChecking наступает
 перед изменением состояния отметки флага элемента.
