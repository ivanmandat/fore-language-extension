# IDimensionViewer.OnExpanding


## Синтаксис


OnExpanding(Sender: Object; Args: [IDimensionAllowActionEventArgs](../IDimensionAllowActionEventArgs/IDimensionAllowActionEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnExpanding наступает
 непосредственно перед развертыванием иерархии потомков вершины дерева.
