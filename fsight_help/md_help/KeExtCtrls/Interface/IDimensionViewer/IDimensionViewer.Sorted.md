# IDimensionViewer.Sorted


## Синтаксис


Sorted: Boolean;


## Описание


Свойство Sorted определяет возможность
 сортировки по столбцу, указанному в свойстве [SortColumn](IDimensionViewer.SortColumn.htm).


## Комментарии


Если свойству установлено значение True
 - сортировка осуществляется по столбцу [SortColumn](IDimensionViewer.SortColumn.htm),
 иначе - осуществляется сортировка по атрибуту «Порядок».
