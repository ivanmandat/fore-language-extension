# IDimensionViewer.OnColumnClick


## Синтаксис


OnColumnClick(Sender: Object; Args: [IEventArgs](ForeSys.chm::/Interface/IEventArgs/IEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnColumnClick наступает
 при щелчке кнопкой мыши по одному из столбцов компонента.
