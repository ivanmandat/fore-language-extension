# IDimensionViewer.Sort


## Синтаксис


Sort([Column: [IDimensionViewerColumn](../IDimensionViewerColumn/IDimensionViewerColumn.htm)
 = Null;][Direction: [SortDirection](foresys.chm::/Enums/SortDirection.htm)
 = 0]);


## Параметры


Column. Столбец, по которому
 необходимо осуществить сортировку. Если параметр не указан, то осуществляется
 сортировка по столбцу, указанному в свойстве [SortColumn](IDimensionViewer.SortColumn.htm);


Direction. Направление сортировки,
 которое будет установлено в столбце. Если параметр не указан, то будет
 установлена сортировка по возрастанию значений.


## Описание


Метод Sort выполняет сортировку
 по значениям указанного столбца.


## Комментарии


Направление сортировки определяется в параметре Direction.
