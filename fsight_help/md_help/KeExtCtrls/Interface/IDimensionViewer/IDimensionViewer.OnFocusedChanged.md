# IDimensionViewer.OnFocusedChanged


## Синтаксис


OnFocusedChanged(Sender: Object; Args: [IDimensionViewerEventArgs](../IDimensionViewerEventArgs/IDimensionViewerEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnFocusedChanged наступает
 при передаче фокуса другому элементу компонента.
