# IDimensionViewer.OnHover


## Синтаксис


OnHover(Sender: Object; Args: [IDimensionNodeColumnEventArgs](../IDimensionNodeColumnEventArgs/IDimensionNodeColumnEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnHover наступает в
 момент прохождения курсора мыши над элементом и отметки подчеркиванием
 элемента компонента.


## Комментарии


Непременным условием для наступления события является установка свойству
 [IDimensionViewer.HotTrack](IDimensionViewer.HotTrack.htm)
 компонента значения True.
