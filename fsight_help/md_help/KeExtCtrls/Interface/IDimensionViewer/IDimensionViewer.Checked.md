# IDimensionViewer.Checked


## Синтаксис


Checked(Element: Integer): Boolean;


## Параметры


Element.
 Индекс элемента измерения.


## Описание


Свойство Checked определяет
 признак установки флага у элемента измерения, индекс которого передается
 в качестве входного параметра Element.
