# IDimensionViewer.ShowButtons


## Синтаксис


ShowButtons: Boolean;


## Описание


Свойство ShowButtons определяет
 признак отображения кнопок разворачивания и сворачивания дерева потомков
 вершин.


## Комментарии


Если свойству установлено значение True,
 то слева от вершины будут располагаться кнопки «+» и «-». При установке
 значения False, разворачивание
 и сворачивание дерева потомков вершин будет производиться двойным щелчком
 мыши по вершине.
