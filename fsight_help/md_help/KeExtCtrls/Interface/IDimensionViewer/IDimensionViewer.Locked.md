# IDimensionViewer.Locked


## Синтаксис


Locked: Boolean;


## Описание


Свойство Locked определяет возможность
 изменения отметки элементов в компоненте.


## Комментарии


Если свойству установлено значение True,
 то будет блокироваться возможность изменения отметки и редактирования
 элементов.


Примечание.
 Данное свойство блокирует возможность изменения отметки и редактирования
 элементов в компоненте только с помощью мыши или клавиатуры. Возможность
 изменить отметку и отредактировать текст элементов с помощью соответствующих
 свойств и методов в ядре будет доступна при любых значениях свойства
 Locked.
