# IDimensionViewer.HotTrackAutoExpand


## Синтаксис


HotTrackAutoExpand: Boolean;


## Описание


Свойство HotTrackAutoExpand
 определяет признак изменения состояния дерева дочерних элементов при щелчке
 по выделенному элементу.


## Комментарии


Если свойству установлено значение True,
 то при щелчке по выделенному элементу будет свернута/развернута иерархия
 его дочерних элементов, иначе изменение состояние иерархии дочерних элементов
 не производится.


Также при щелчке по выделенной вершине происходит изменение ее отметки
 в соответствии с настройками, установленными в свойстве [SelectionMode](IDimensionViewer.SelectionMode.htm).


Свойство HotTrackAutoExpand
 актуально, если свойству [HotTrack](IDimensionViewer.HotTrack.htm)
 установлено значение True.


Примечание.
 Если свойству HotTrackAutoExpand
 установлено значение True, то
 изменение состояния дерева происходит после щелчка по элементу с небольшой
 задержкой. Изменение состояния дерева при двойном щелчке по элементу будет
 недоступно.
