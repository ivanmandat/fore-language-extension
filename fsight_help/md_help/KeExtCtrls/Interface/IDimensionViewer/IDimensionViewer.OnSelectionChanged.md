# IDimensionViewer.OnSelectionChanged


## Синтаксис


OnSelectionChanged(Sender: Object; Args: [IDimensionViewerEventArgs](../IDimensionViewerEventArgs/IDimensionViewerEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnSelectionChanged наступает
 при изменении отметки элементов компонента.
