# IDimensionViewer.OnExpanded


## Синтаксис


OnExpanded(Sender: Object; Args: [IDimensionNodeEventArgs](../IDimensionNodeEventArgs/IDimensionNodeEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnExpanded наступает
 после развертывания иерархии потомков вершины дерева.


## Комментарии


Событие OnExpanded наступает
 только в том случае, если произошло событие [OnExpanding](IDimensionViewer.OnExpanding.htm).
