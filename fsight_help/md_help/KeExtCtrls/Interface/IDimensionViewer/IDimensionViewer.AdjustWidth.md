# IDimensionViewer.AdjustWidth


## Синтаксис


AdjustWidth(Column: Integer);


## Параметры


Column. Индекс колонки, в которой
 необходимо осуществить автоматический подгон ширины.


## Описание


Метод AdjustWidth осуществляет
 автоматический подгон ширины колонки с учетом её содержимого.


## Комментарии


Режим автоподгонки определяет свойство [IDimensionViewer.AdjustWidthOnlyVisible](IDimensionViewer.AdjustWidthOnlyVisible.htm).
