# IDimensionViewer.RowSelect


## Синтаксис


RowSelect: Boolean;


## Описание


Свойство RowSelect определяет
 необходимость выделения выбранного элемента подсветкой всей строки.


## Комментарии


Если свойству установлено значение True,
 то подсветкой будет выделяться вся строка, в которой находится выбранный
 элемент. При установке значения False,
 будет выделяться только сам элемент.
