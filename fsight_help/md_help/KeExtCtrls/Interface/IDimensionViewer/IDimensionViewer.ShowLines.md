# IDimensionViewer.ShowLines


## Синтаксис


ShowLines: Boolean;


## Описание


Свойство ShowLines определяет
 признак отображения соединительной линии между вершиной и ее потомками.


## Комментарии


Если свойству установлено значение True,
 линии будут соединять вершины с их потомками. При установке значения False, линии отображаться не будут.
