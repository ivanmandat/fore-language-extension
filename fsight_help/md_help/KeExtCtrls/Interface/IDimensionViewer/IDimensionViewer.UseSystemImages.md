# IDimensionViewer.UseSystemImages


## Синтаксис


UseSystemImages: Boolean;


## Описание


Свойство UseSystemImages определяет
 признак использования системных пиктограмм.


## Комментарии


По умолчанию свойству установлено значение False,
 при этом для элементов отображаются пиктограммы из документа, который
 [задан
 в репозитории](KeSom.chm::/Interface/IMetabase/IMetabase.SpecialObject.htm) в качестве списка пиктограмм элементов
 по умолчанию или компонента, который может быть указан в свойстве [Images](IDimensionViewer.Images.htm).


При установке свойству UseSystemImages
 значения True для элементов
 будут отображены системные пиктограммы, которые хранятся в ресурсах продукта
 «Форсайт. Аналитическая платформа».
