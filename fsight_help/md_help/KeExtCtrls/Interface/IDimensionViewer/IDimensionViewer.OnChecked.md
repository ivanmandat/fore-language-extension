# IDimensionViewer.OnChecked


## Синтаксис


OnChecked(Sender: Object; Args: [IDimensionNodeCheckedEventArgs](../IDimensionNodeCheckedEventArgs/IDimensionNodeCheckedEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnChecked наступает
 после изменения состояния отметки флага элемента.
