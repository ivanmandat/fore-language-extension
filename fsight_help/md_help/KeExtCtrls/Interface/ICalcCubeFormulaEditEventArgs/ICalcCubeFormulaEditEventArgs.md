# ICalcCubeFormulaEditEventArgs


Сборка: ExtCtrls;


## Описание


Интерфейс ICalcCubeFormulaEditEventArgs
 содержит свойства аргумента события, возникающего при создании терма,
 соответствующего координате куба.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Coord](ICalcCubeFormulaEditEventArgs.Coord.htm)
		 Свойство Coord возвращает
		 координату куба, для которой сгенерировано событие.


		 ![](../../Property_Image.gif)
		 [Result](ICalcCubeFormulaEditEventArgs.Result.htm)
		 Свойство Result определяет
		 текста терма, который будет соответствовать указанной координате
		 куба.
