# IDBComboBox.DataSource


## Синтаксис


DataSource: [IUiDataSource](../IUiDataSource/IUiDataSource.htm);


## Описание


Свойство DataSource связывает
 компонент [DbComboBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DBComboBox.htm)
 с набором данных, который содержит поле [DataField](IDBComboBox.DataField.htm).


## Комментарии


Список допустимых значений включает имена всех компонентов [UiDataSource](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiDataSource.htm),
 расположенных на форме.
