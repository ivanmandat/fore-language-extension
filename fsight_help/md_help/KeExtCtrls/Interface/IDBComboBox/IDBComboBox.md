# IDBComboBox


Сборка: ExtCtrls;


## Описание


Интерфейс IDBComboBox содержит
 свойства и методы компонента среды разработки - [DBComboBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DBComboBox.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DataField](IDBComboBox.DataField.htm)
		 Свойство DataField
		 определяет имя поля в источнике данных, с которым связан компонент.


		 ![](../../Property_Image.gif)
		 [DataSource](IDBComboBox.DataSource.htm)
		 Свойство DataSource
		 связывает компонент [DBComboBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DBComboBox.htm)
		 с набором данных, который содержит поле [DataField](IDBComboBox.DataField.htm).


		 ![](../../Property_Image.gif)
		 [Field](IDBComboBox.Field.htm)
		 Свойство Field возвращает
		 параметры поля, с которым связан компонент.


		 ![](../../Property_Image.gif)
		 [ReadOnly](IDBComboBox.ReadOnly.htm)
		 Свойство ReadOnly определяет,
		 будет ли пользователь иметь возможность изменять значение поля
		 текущей записи.
