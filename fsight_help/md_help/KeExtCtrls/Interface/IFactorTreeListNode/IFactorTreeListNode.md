# IFactorTreeListNode


Сборка: ExtCtrls;


## Описание


Интерфейс IAttributeValueTreeListNode
 содержит свойства и методы для работы с показателями в иерархической структуре
 базы данных временных рядов компонента [MetaAttributesTreeList](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/MetaAttributesTreeList.htm).
