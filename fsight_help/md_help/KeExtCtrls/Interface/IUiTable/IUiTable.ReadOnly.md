# IUiTable.ReadOnly


## Синтаксис


ReadOnly: Boolean;


## Описание


Свойство ReadOnly определяет
 признак возможности редактирования данных таблицы.


## Комментарии


Свойству по умолчанию установлено значение False,
 при этом пользователь имеет возможность изменения данных таблицы (обновление,
 вставка и удаление).
