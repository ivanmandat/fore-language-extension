# IUiTable.TableName


## Синтаксис


TableName: String;


## Описание


Свойство TableName определяет
 физическое имя таблицы, к которой необходимо получить доступ.


## Комментарии


Список допустимых значений включает имена всех таблиц базы данных, установленной
 в свойстве [Database](IUiTable.Database.htm).


В инспекторе объектов для свойства TableName
 реализован раскрывающийся список, в котором после установки свойства [Database](IUiTable.Database.htm) будет содержаться полный
 список физических имен таблиц, имеющихся в выбранной базе данных.
