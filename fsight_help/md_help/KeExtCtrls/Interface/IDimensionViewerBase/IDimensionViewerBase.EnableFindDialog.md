# IDimensionViewerBase.EnableFindDialog


## Синтаксис


EnableFindDialog: Boolean;


## Описание


Свойство EnableFindDialog определяет,
 доступен ли для вызова стандартный диалог поиска элементов.


## Комментарии


Стандартный диалог поиска элементов вызывается с помощью горячих клавиш
 CTRL+F или F3.


Допустимые значения:


	- True. По умолчанию.
	 Доступен вызов диалога поиска;


	- False. Вызов диалога
	 поиска недоступен. Актуально, если необходимо переназначить указанные
	 горячие клавиши.
