# IDimensionViewerBase.ShowRoot


## Синтаксис


ShowRoot: Boolean;


## Описание


Свойство ShowRoot определяет,
 будут ли отображаться кнопки сворачивания и разворачивания корневых элементов
 деревьев.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Слева от вершины будут располагаться кнопки разворачивания «+» и сворачивания
	 «-»;


	- False. Кнопки разворачивания
	 «+» и сворачивания «-» будут скрыты. Сворачиввание/разворачивание
	 дерева потомков вершин будет производиться двойным щелчком мыши по
	 вершине.
