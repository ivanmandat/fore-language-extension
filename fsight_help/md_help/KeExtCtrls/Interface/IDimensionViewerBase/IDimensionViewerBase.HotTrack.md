# IDimensionViewerBase.HotTrack


## Синтаксис


HotTrack: Boolean;


## Описание


Свойство HotTrack определяет,
 будет ли подчеркиваться элемент при наведении указателя мыши.


## Комментарии


Допустимые значения:


	- True. Элемент, над которым
	 находится указатель мыши, будет подчеркиваться;


	- False. По умолчанию.
	 Элемент, над которым находится указатель мыши, подчеркиваться не будет.


Для определения, будет ли развернуто дерево дочерних элементов при щелчке
 по выделенному элементу, используйте [IDimensionViewerBase.HotTrackAutoExpand](IDimensionViewerBase.HotTrackAutoExpand.htm).
