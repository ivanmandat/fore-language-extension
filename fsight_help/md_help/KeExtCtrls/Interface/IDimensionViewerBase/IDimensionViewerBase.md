# IDimensionViewerBase


Сборка: ExtCtrls;


## Описание


Интерфейс IDimensionViewerBase
 используется для работы с раскрывающимся списком компонента [MetaAttributesBreadcrumb](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/MetaAttributesBreadcrumb.htm).


## Иерархия наследования


           IDimensionViewerBase


## Комментарии


Данный интерфейс используется для работы с раскрывающимся списком цепочки
 навигации по временным рядам, содержащимся в базе данных временных рядов.


Для получения объекта данного типа используйте свойство [IMetaAttributesBreadcrumb.DimensionViewer](../IMetaAttributesBreadcrumb/IMetaAttributesBreadcrumb.DimensionViewer.htm).

## Методы

		 флажки.


		 ![](../../Sub_Image.gif)
		 [GetItemAt](IDimensionViewerBase.GetItemAt.htm)
		 Метод GetItemAt позволяет
		 получить элемент компонента в заданной точке.


		 ![](../../Sub_Image.gif)
		 [HitTest](IDimensionViewerBase.HitTest.htm)
		 Метод HitTest является
		 расширением метода [IDimensionViewerBase.GetItemAt](IDimensionViewerBase.GetItemAt.htm).


		 ![](../../Sub_Image.gif)
		 [MakeVisible](IDimensionViewerBase.MakeVisible.htm)
		 Метод MakeVisible осуществляет
		 прокрутку дерева элементов таким образом, чтобы указанный элемент
		 попал в видимую область компонента.
