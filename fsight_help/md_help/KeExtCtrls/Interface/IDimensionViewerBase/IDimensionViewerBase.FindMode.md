# IDimensionViewerBase.FindMode


## Синтаксис


FindMode: [FindModeType](ModForms.chm::/Enums/FindModeType.htm);


## Описание


Свойство FindMode определяет
 режим отметки найденных элементов.


## Комментарии


Свойство актуально, если включена [множественная
 отметка](IDimensionViewerBase.SelectionMode.htm) элементов.
