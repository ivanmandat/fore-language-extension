# IDimensionViewerBase.HighlightTrack


## Синтаксис


HighlightTrack: Boolean;


## Описание


Свойство HighlightTrack определяет,
 будут ли подсвечиваться элементы при наведении указателя мыши.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 При наведении указателя мыши элемент подсвечивается;


	- False. При наведении
	 указателя мыши элемент не подсвечивается.
