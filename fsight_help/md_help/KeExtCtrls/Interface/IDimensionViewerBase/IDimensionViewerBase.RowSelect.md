# IDimensionViewerBase.RowSelect


## Синтаксис


RowSelect: Boolean;


## Описание


Свойство RowSelect определяет,
 будет ли выделяться вся строка, в которой находится выбранный элемент.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Будет выделена вся строка, в которой находится выбранный элемент;


	- False. Будет выделен
	 только текст элемента.
