# IDimensionViewerBase.ShowButtons


## Синтаксис


ShowButtons: Boolean;


## Описание


Свойство ShowButtons определяет,
 будут ли отображаться кнопки сворачивания и разворачивания дерева потомков
 вершин.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Слева от вершины будут располагаться кнопки разворачивания «+» и сворачивания
	 «-»;


	- False. Сворачивание/разворачивание
	 будет производиться двойным щелчком мыши по вершине.
