# IDimensionViewerBase.CheckState


## Синтаксис


CheckState(Element: Integer): [NodeCheckState](ModForms.chm::/Enums/NodeCheckState.htm);


## Параметры


Element. Индекс вершины дерева.


## Описание


Свойство CheckState определяет
 состояние флажка вершины дерева.


## Комментарии


Свойство актуально, если свойству [IDimensionViewerBase.Checkboxes](IDimensionViewerBase.Checkboxes.htm)
 установлено значение True.
