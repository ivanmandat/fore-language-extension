# IDimensionViewerBase.AdjustWidth


## Синтаксис


AdjustWidth(Column: Integer);


## Параметры


Column. Индекс колонки, в которой
 необходимо осуществить автоматический подгон ширины.


## Описание


Метод AdjustWidth осуществляет
 автоматический подгон ширины колонки с учетом её содержимого.


## Комментарии


Для определения, будет ли автоматический подгон учитывать ширину
 элементов, которые видны в компоненте, используйте свойство [IDimensionViewerBase.AdjustWidthOnlyVisible](IDimensionViewerBase.AdjustWidthOnlyVisible.htm).
