# IDimensionViewerBase.Locked


## Синтаксис


Locked: Boolean;


## Описание


Свойство Locked определяет,
 будет ли доступно изменение отметки элементов в компоненте.


## Комментарии


Допустимые значения:


	- True. Изменение отметки
	 заблокировано;


	- False. По умолчанию.
	 Изменение отметки доступно.


Примечание.
 Данное свойство блокирует возможность изменения отметки только с помощью
 мыши или клавиатуры. Возможность изменить отметку с помощью соответствующих
 свойств и методов в Fore будет
 доступна при любых значениях свойства Locked.
