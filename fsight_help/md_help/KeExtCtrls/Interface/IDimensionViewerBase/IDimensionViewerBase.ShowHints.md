# IDimensionViewerBase.ShowHints


## Синтаксис


ShowHints: Boolean;


## Описание


Свойство ShowHints определяет,
 будут ли всплывать подсказки при наведении указателя мыши на элементы
 в компоненте.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Подсказки будут всплывать при наведении указателя мыши на элементы;


	- Flase. Подсказки всплывать
	 не будут при наведении указателя мыши на элементы.
