# IDimensionViewerBase.Checkboxes


## Синтаксис


Checkboxes: Boolean;


## Описание


Свойство Checkboxes определяет,
 будут ли отображаться флажки слева от элементов в компоненте.


## Комментарии


Допустимые значения:


	- True. Рядом с элементами
	 отображаются флажки;


	- False. По умолчанию.
	 Флажки у элементов не отображаются.


Для определения, будет ли у флажков родительских вершин дерева автоматически
 устанавливаться неопределенное состояние, если отмечены не все дочерние
 элементы, используйте [IDimensionViewerBase.AutoCheckParent](IDimensionViewerBase.AutoCheckParent.htm).


Для определения состояния флажка вершин используйте свойство [IDimensionViewerBase.CheckState](IDimensionViewerBase.CheckState.htm) или
 [IDimensionViewerBase.Checked](IDimensionViewerBase.Checked.htm).
