# IDimensionViewerBase.GridLines


## Синтаксис


GridLines: Boolean;


## Описание


Свойство GridLines определяет,
 будет ли отображаться сетка в раскрывающемся списке компонента.


## Комментарии


Допустимые значения:


	- True. В раскрывающемся
	 списке между элементами отображается горизонтальная сетка;


	- False. По умолчанию.
	 Сетка в списке элементов не отображается.
