# IDimensionViewerBase.ColumnClick


## Синтаксис


ColumnClick: Boolean;


## Описание


Свойство ColumnClick определяет,
 будет ли разрешен щелчок по заголовку столбца.


## Комментарии


Допустимые значения:


	- True. Щелчок по заголовку
	 столбца разрешён, каждый щелчок приводит к смене направления сортировки
	 элементов;


	- Flase. По умолчанию.
	 Щелчок по заголовку столбца запрещён.


Свойство актуально, если свойству [IMetaAttributesBreadcrumb.ShowColumnHeaders](../IMetaAttributesBreadcrumb/IMetaAttributesBreadcrumb.ShowColumnHeaders.htm)
 установлено значение True.
