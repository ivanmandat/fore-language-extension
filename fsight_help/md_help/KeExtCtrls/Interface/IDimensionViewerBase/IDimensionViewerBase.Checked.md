# IDimensionViewerBase.Checked


## Синтаксис


Checked(Element: Integer): Boolean;


## Параметры


Element. Индекс вершины дерева.


## Описание


Свойство Checked определяет
 признак установки флажка у вершины дерева.


## Комментарии


Допустимые значения:


	- True. Флажок элемента
	 установлен;


	- False. Флажок элемента
	 снят.


Свойство актуально, если свойству [IDimensionViewerBase.Checkboxes](IDimensionViewerBase.Checkboxes.htm)
 установлено значение True.
