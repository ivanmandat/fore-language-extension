# IDimensionViewerBase.Columns


## Синтаксис


Columns: [IDimensionViewerColumns](../IDimensionViewerColumns/IDimensionViewerColumns.htm);


## Описание


Свойство Columns возвращает
 коллекцию столбцов раскрывающегося списка компонента.


## Комментарии


Количество столбцов соответствует количеству атрибутов в измерении,
 на базе которого строится атрибут базы данных временных рядов.
