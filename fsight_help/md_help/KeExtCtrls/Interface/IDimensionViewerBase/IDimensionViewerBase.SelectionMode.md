# IDimensionViewerBase.SelectionMode


## Синтаксис


SelectionMode: [SelectionModeType](ModForms.chm::/Enums/SelectionModeType.htm);


## Описание


Свойство SelectionMode определяет
 режим отметки элементов в компоненте.


## Комментарии


По умолчанию используется режим множественной отметки: [SelectionModeType.MultiSelectStandard](modforms.chm::/Enums/SelectionModeType.htm).
