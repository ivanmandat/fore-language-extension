# IDimensionViewerBase.FocusedElement


## Синтаксис


FocusedElement: Integer;


## Описание


Свойство FocusedElement определяет
 индекс элемента, на котором установлен фокус.


## Комментарии


Чтобы элемент находился в фокусе необходимо, чтобы необходимый элемент
 находился в развернутой иерархии. Для этого используйте свойство [IDimensionViewerBase.AutoExpandDepth](IDimensionViewerBase.AutoExpandDepth.htm)
 или метод [IDimensionViewerBase.Expand](IDimensionViewerBase.Expand.htm).
