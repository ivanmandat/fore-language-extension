# IMetabaseListViewAfterSearchEventArgs


Сборка: ExtCtrls;


## Описание


Интерфейс IMetabaseListViewAfterSearchEventArgs
 содержит свойства аргумента события, наступающего для компонента, если
 в диалоге поиска осуществлялся поиск всех объектов, удовлетворяющих заданным
 условиям.
