# IMetabaseDialog.InitialFolder


## Синтаксис


InitialFolder: [IMetabaseObjectDescriptor](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm);


## Описание


Свойство InitialFolder определяет
 исходный каталог.


## Комментарии


Исходный каталог - это каталог, содержимое которого отображается при
 [инициализации](IMetabaseDialog.Execute.htm) диалога. Исходный
 каталог не может располагаться выше [корневого
 каталога](IMetabaseDialog.Root.htm).
