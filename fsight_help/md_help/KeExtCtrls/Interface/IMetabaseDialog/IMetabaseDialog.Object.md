# IMetabaseDialog.Object


## Синтаксис


Object: [IMetabaseObjectDescriptor](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm);


## Описание


Свойство Object возвращает описание
 выбранного объекта репозитория.


## Комментарии


Если в компоненте выбрано несколько объектов, то свойство Object
 вернёт описание только первого выбранного объекта. Описания всех объектов
 возвращает свойство [IMetabaseDialog.Objects](IMetabaseDialog.Objects.htm).
