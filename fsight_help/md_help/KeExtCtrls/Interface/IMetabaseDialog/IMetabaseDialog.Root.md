# IMetabaseDialog.Root


## Синтаксис


Root: [IMetabaseObjectDescriptor](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm);


## Описание


Свойство Root определяет корневой
 каталог для диалога.


## Комментарии


Корневой каталог - это папка репозитория, выше которой просмотр содержимого
 будет недоступен. По умолчанию для диалога корневым каталогом является
 корневая папка текущего репозитория.
