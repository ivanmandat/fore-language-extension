# IMetabaseDialog.Title


## Синтаксис


Title: String;


## Описание


Свойство Title определяет заголовок
 диалога.


## Комментарии


По умолчанию в данном свойстве установлена пустая строка. При этом после
 инициализации диалога в заголовке будет отображаться текст «Выбор объекта».
 Данный текст локализован в системных ресурсах продукта «Форсайт. Аналитическая платформа»,
 поэтому будет зависеть от используемого в текущий момент языка.
