# IDimensionCombo.CloseOnClick


## Синтаксис


CloseOnClick: Boolean;


## Описание


Свойство CloseOnClick определяет,
 будет ли производиться автоматическое закрытие списка при выборе
 одного элемента из этого списка.


## Комментарии


При установке свойству значения True
 производится закрытие, при установке False
 - список закрываться не будет.
