# IDimensionCombo.CancelOnEscape


## Синтаксис


CancelOnEscape: Boolean;


## Описание


Свойство CancelOnEscape определяет,
 будет ли производиться отмена установленной отметки и закрытие списка
 при нажатии клавиши ESC.


## Комментарии


При установке свойству значения True
 производится отмена установленной отметки и сворачивание списка, при установке
 False - установленная отметка
 не отменяется.
