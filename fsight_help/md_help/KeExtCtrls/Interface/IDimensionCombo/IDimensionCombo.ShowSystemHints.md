# IDimensionCombo.ShowSystemHints


## Синтаксис


ShowSystemHints: Boolean;


## Описание


Свойство ShowSystemHints определяет,
 будет ли отображаться системная всплывающая подсказка для различных элементов
 компонента.


## Комментарии


Допустимые значения:


	- True. Отображается системная
	 всплывающая подсказка;


	- False. Системная всплывающая
	 подсказка не отображается. Будет отображаться подсказка указанная
	 в свойстве [Hint](ModForms.chm::/Interface/IControl/IControl.Hint.htm),
	 если свойству [ShowHint](ModForms.chm::/Interface/IControl/IControl.ShowHint.htm)
	 установлено значение True.


В текущей реализации системная всплывающая подсказка доступна для кнопки
 поиска.
