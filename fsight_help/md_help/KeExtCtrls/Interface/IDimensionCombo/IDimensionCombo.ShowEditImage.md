# IDimensionCombo.ShowEditImage


## Синтаксис


ShowEditImage: Boolean;


## Описание


Свойство ShowEditImage определяет
 признак отображения пиктограммы элемента в редакторе компонента [DimensionCombo](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DimensionCombo.htm).


## Комментарии


Если свойству установлено значение True,
 то в редакторе компонента будет отображаться пиктограмма и наименование
 выделенного элемента. В режиме множественной отметки - пиктограмма и наименование
 первого выделенного объекта, и общее количество выделенных элементов.


Если свойству установлено значение False,
 то отображается только наименование выделенного элемента (наименование
 элемента и общее количество выделенных элементов).


Свойство актуально, если установлено свойство [Images](../IDimensionViewer/IDimensionViewer.Images.htm).
