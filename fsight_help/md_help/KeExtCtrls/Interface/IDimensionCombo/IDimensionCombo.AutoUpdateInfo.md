# IDimensionCombo.AutoUpdateInfo


## Синтаксис


AutoUpdateInfo: Boolean;


## Описание


Свойство AutoUpdateInfo определяет
 признак автоматического обновления информации об отметке в редакторе компонента
 [DimensionCombo](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DimensionCombo.htm).


## Комментарии


Если свойству установлено значения True,
 то при изменении отметки будет производиться автоматическое обновление
 об отметке в редакторе компонента. Если свойству установлено значение
 False, то информация об отметке
 обновится после вызова метода [UpdateInfo](IDimensionCombo.UpdateInfo.htm).
