# ISearchDataSourcesSelectionInfo


Сборка: ExtCtrls;


## Описание


Интерфейс ISearchDataSourcesSelectionInfo
 содержит свойства для работы с коллекцией индексированных элементов измерения.


## Иерархия наследования


           ISearchDataSourcesSelectionInfo


## Комментарии


Параметры коллекции можно получить в свойстве [ISearchDataSourcesItem.Selection](../ISearchDataSourcesItem/ISearchDataSourcesItem.Selection.htm).
