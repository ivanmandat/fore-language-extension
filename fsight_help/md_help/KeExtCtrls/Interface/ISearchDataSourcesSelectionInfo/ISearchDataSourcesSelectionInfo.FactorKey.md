# ISearchDataSourcesSelectionInfo.FactorKey


## Синтаксис


FactorKey: Integer;


## Описание


Свойство FactorKey возвращает
 уникальный ключ измерения.


## Комментарии


Если индексированное измерение соответствует базе данных временных рядов,
 то свойство вернет ключ ряда.


Если индексированное измерение соответствует стандартному кубу, то свойство
 вернет уникальный индекс для отметки измерения.
