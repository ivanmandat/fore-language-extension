# IMetabaseDialogCombiFilter


Сборка: ExtCtrls;


## Описание


Интерфейс IMetabaseDialogCombiFilter
 содержит свойства и методы комбинированного фильтра, включающего в себя
 другие виды фильтров.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Filters](IMetabaseDialogCombiFilter.Filters.htm)
		 Свойство Filters возвращает
		 коллекцию фильтров комбинированного фильтра.


## Свойства, унаследованные от [IMetabaseDialogFilter](../IMetabaseDialogFilter/IMetabaseDialogFilter.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Description](../IMetabaseDialogFilter/IMetabaseDialogFilter.Description.htm)
		 Свойство Description
		 определяет описание фильтра, которое будет отображаться в комбинированном
		 списке фильтров диалога.


		 ![](../../Property_Image.gif)
		 [IsVisible](../IMetabaseDialogFilter/IMetabaseDialogFilter.IsVisible.htm)
		 Свойство IsVisible
		 возвращает признак видимости объекта репозитория в диалоге.
