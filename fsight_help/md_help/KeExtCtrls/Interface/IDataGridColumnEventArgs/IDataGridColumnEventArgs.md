# IDataGridColumnEventArgs


Сборка: ExtCtrls;


## Описание


Интерфейс IDataGridColumnEventArgs
 содержит свойства аргумента событий, возникающих для какой-либо колонки
 компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).
