# IDimensionAllowActionEventArgs


Сборка: ExtCtrls;


## Описание


Интерфейс IDimensionAllowActionEventArgs
 содержит свойства аргумента событий, возникающих перед изменением отображения
 дерева дочерних элементов какой-либо вершины дерева.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Allow](IDimensionAllowActionEventArgs.Allow.htm)
		 Свойство Allow определяет
		 признак изменения отображения дерева дочерних элементов вершины.
