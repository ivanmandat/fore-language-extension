# IDimensionAllowActionEventArgs.Allow


## Синтаксис


Allow: Boolean;


## Описание


Свойство Allow определяет признак
 изменения отображения дерева дочерних элементов вершины.


## Комментарии


Если свойству установлено значение True,
 то изменение будет произведено, иначе отменено.
