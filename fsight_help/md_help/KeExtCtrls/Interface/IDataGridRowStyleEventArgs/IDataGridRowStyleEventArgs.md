# IDataGridRowStyleEventArgs


Сборка: ExtCtrls;


## Описание


Интерфейс IDataGridRowStyleEventArgs
 содержит свойства аргумента события, наступающего при отрисовке строки
 таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Row](IDataGridRowStyleEventArgs.Row.htm)
		 Свойство Row возвращает
		 индекс строки, для которой сгенерировано событие.


		 ![](../../Property_Image.gif)
		 [StyleIndex](IDataGridRowStyleEventArgs.StyleIndex.htm)
		 Свойство StyleIndex
		 определяет индекс стиля, который необходимо применить для текущей
		 строки таблицы.
