# IDimensionColumnEventArgs


Сборка: ExtCtrls;


## Описание


Интерфейс IDimensionColumnEventArgs
 содержит свойства аргумента события, возникающего при щелчке по заголовку
 колонки компонентов [DimensionCombo](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DimensionCombo.htm)
 и [DimensionTree](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DimensionTree.htm).
