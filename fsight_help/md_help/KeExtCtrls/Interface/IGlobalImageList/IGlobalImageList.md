# IGlobalImageList

## Описание


Интерфейс IGlobalImageList содержит
 свойства и методы компонента среды разработки - [GlobalImageList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/GlobalImageList.htm).


## Свойства


		 Имя свойства
		 Краткое описание

		 [Document](IGlobalImageList.Document.htm)
		 Свойство Document определяет
		 документ репозитория, являющийся источником изображения для компонента.
