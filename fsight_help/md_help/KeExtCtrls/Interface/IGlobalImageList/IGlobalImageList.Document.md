# IGlobalImageList.Document


## Синтаксис


Document: [IDocument](KeFore.chm::/Interface/IDocument/IDocument.htm);


## Описание


Свойство Document определяет
 документ репозитория, являющийся источником изображения для компонента.


## Комментарии


Документ должен представлять собой импортированное изображение в формате
 BMP, состоящее из горизонтально расположенных пиктограмм. При загрузке
 изображение будет разбиваться на пиктограммы, в соответствии с размерами,
 установленными в свойствах [Height](ModForms.chm::/Interface/IImageList/IImageList.Height.htm) и [Width](ModForms.chm::/Interface/IImageList/IImageList.Width.htm)
 компонента [GlobalImageList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/GlobalImageList.htm).
