# IDataGridCompareEventArgs


Сборка: ExtCtrls;


## Описание


Интерфейс IDataGridCompareEventArgs
 содержит свойства аргумента события, возникающего при сравнении значений
 колонки во время пользовательской сортировки компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).
