# IDimensionNodeCheckingEventArgs.Allow


## Синтаксис


Allow: Boolean;


## Описание


Свойство Allow определяет, будет
 ли изменена отметка флага элемента. Если свойству установить значение
 True, то изменение отметки флага
 элемента будет разрешено, иначе - отменено.
