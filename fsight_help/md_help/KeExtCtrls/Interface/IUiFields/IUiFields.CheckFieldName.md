# IUiFields.CheckFieldName


## Синтаксис


CheckFieldName(FieldName: String): Boolean;


## Параметры


FieldName. Наименование поля,
 наличие которого необходимо проверить.


## Описание


Метод CheckFieldName осуществляет
 проверку наличия поля с наименованием FieldName
 в списке полей источника данных.
