# IUiFields.FieldByName


## Синтаксис


FieldByName(FieldName: String): [IUiField](../IUiField/IUiField.htm);


## Параметры


FieldName.
 Параметр, определяющий наименование поля таблицы, к которому необходимо
 обратиться.


## Описание


Метод FieldByName позволяет
 обратиться к полю источника данных по его имени, указанному в качестве
 входного параметра FieldName.
