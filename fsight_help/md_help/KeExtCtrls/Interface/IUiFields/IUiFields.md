# IUiFields


Сборка: ExtCtrls;


## Описание


Интерфейс IUiFields содержит
 свойства и методы коллекции полей источника данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IUiFields.Count.htm)
		 Свойство Count возвращает
		 количество полей, содержащихся в коллекции.


		 ![](../../Property_Image.gif)
		 [Dataset](IUiFields.Dataset.htm)
		 Свойство Dataset возвращает
		 источник данных коллекции полей.


		 [Item](IUiFields.Item.htm)
		 Свойство Item возвращает
		 параметры поля источника данных.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CheckFieldName](IUiFields.CheckFieldName.htm)
		 Метод CheckFieldName
		 осуществляет проверку наличия указанного поля в списке полей источника
		 данных.


		 ![](../../Sub_Image.gif)
		 [FieldByName](IUiFields.FieldByName.htm)
		 Метод FieldByName позволяет
		 обратиться к полю источника данных по его имени.
