# IUiDimension


Сборка: ExtCtrls;


## Описание


Интерфейс IUiDimension содержит
 свойства и методы компонента среды разработки - [UiDimension](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiDimension.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dimension](IUiDimension.Dimension.htm)
		 Свойство Dimension
		 определяет справочник, являющийся источником данных для компонента.


		 ![](../../Property_Image.gif)
		 [DimInstance](IUiDimension.DimInstance.htm)
		 Свойство DimInstance
		 возвращает данные справочника, подключенного к компоненту.


		 ![](../../Property_Image.gif)
		 [Selection](IUiDimension.Selection.htm)
		 Свойство Selection
		 определяет отметку элементов справочника, являющегося источником
		 данных для компонента.
