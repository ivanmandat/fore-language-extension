# ISearchDataSourceSelectionChangeEventArgs.Cancel


## Синтаксис


Cancel: Boolean;


## Описание


Свойство Cancel определяет,
 будет ли осуществлена отметка элемента в области компонента.


## Комментарии


Допустимые значения:


	- True. Отметка элемента
	 будет запрещена;


	- False. Элемент будет
	 отмечен.
