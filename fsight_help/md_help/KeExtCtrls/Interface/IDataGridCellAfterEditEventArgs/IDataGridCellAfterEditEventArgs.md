# IDataGridCellAfterEditEventArgs


Сборка: ExtCtrls;


## Описание


Интерфейс IDataGridCellAfterEditEventArgs
 содержит свойства аргумента события, наступающего после завершения редактирования
 значения ячейки.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Changed](IDataGridCellAfterEditEventArgs.Changed.htm)
		 Свойство Changed возвращает
		 True, если значение ячейки
		 изменилось.
