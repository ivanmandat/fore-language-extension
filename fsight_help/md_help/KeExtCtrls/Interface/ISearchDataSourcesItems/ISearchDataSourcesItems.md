# ISearchDataSourcesItems


Сборка: ExtCtrls;


## Описание


Интерфейс ISearchDataSourcesItems
 содержит свойства и методы для работы с коллекцией элементов, содержащих
 результаты BI-поиска.


## Иерархия наследования


           ISearchDataSourcesItems


## Комментарии


Коллекцию элементов можно получить в свойстве [ISearchDataSourcesBox.Items](../ISearchDataSourcesBox/ISearchDataSourcesBox.Items.htm).
