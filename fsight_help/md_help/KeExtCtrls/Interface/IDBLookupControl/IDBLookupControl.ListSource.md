# IDBLookupControl.ListSource


## Синтаксис


ListSource: [IUiDataSource](../IUiDataSource/IUiDataSource.htm);


## Описание


Свойство ListSource определяет
 источник данных для компонента поиска, то есть источник, содержащий поля
 [KeyField](IDBLookupControl.KeyField.htm) и [ListField](IDBLookupControl.ListField.htm).


## Комментарии


Список допустимых значений включает имена всех источников [UiDataSource](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiDataSource.htm),
 расположенных на форме.
