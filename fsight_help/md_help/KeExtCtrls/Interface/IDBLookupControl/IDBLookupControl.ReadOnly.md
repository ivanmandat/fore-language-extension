# IDBLookupControl.ReadOnly


## Синтаксис


ReadOnly: Boolean;


## Описание


Свойство ReadOnly определяет
 возможность редактирования данных компонента.


## Комментарии


Если свойству ReadOnly установлено
 значение True, то изменение данных
 компонента запрещено. При установке значения False,
 изменение разрешено.
