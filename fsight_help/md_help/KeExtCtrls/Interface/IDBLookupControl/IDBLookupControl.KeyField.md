# IDBLookupControl.KeyField


## Синтаксис


KeyField: String;


## Описание


Свойство KeyField определяет
 поле из источника данных [ListSource](IDBLookupControl.ListSource.htm),
 значение которого должно соответствовать значению поля [DataField](IDBLookupControl.DataField.htm).


## Комментарии


При помощи KeyField набор-справочник
 [ListSource](IDBLookupControl.ListSource.htm) связывается с
 набором [DataSource](IDBLookupControl.DataSource.htm). Необходимым
 условием установки связи является совпадение значений полей KeyField
 и [DataField](IDBLookupControl.DataField.htm), совпадение имен
 этих полей необязательно.
