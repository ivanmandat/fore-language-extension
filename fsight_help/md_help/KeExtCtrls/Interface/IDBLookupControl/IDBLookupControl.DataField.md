# IDBLookupControl.DataField


## Синтаксис


DataField: String;


## Описание


Свойство DataField определяет
 поле источника данных, которое будет отображаться данным компонентом.


## Комментарии


Данные для отображения берутся из текущей записи набора. Поле DataField не отображается, а указывает
 на поле источника [DataSource](IDBLookupControl.DataSource.htm).
 Это поле расшифровывается списком значений, а именно в это поле вносятся
 изменения при выборе другого элемента списка.
