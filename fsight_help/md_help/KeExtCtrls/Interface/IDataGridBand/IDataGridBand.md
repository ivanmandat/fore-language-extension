# IDataGridBand


Сборка: ExtCtrls;


## Описание


Интерфейс IDataGridBand содержит
 свойства и методы контейнеров компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AutoFillDown](IDataGridBand.AutoFillDown.htm)
		 Свойство AutoFillDown
		 определяет, будет ли расширяться заголовок контейнера вниз, если
		 это возможно.


		 ![](../../Property_Image.gif)
		 [Children](IDataGridBand.Children.htm)
		 Свойство Children возвращает
		 коллекцию дочерних объектов контейнера.


		 ![](../../Property_Image.gif)
		 [HasChildren](IDataGridBand.HasChildren.htm)
		 Свойство HasChildren
		 возвращает признак наличия дочерних объектов у контейнера.


		 ![](../../Property_Image.gif)
		 [RowCount](IDataGridBand.RowCount.htm)
		 Свойство RowCount возвращает
		 количество строк, в которых расположены дочерние объекты.
