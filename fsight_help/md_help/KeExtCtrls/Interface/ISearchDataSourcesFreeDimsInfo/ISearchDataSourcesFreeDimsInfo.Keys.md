# ISearchDataSourcesFreeDimsInfo.Keys


## Синтаксис


Keys: Array;


## Описание


Свойство Keys возвращает массив,
 содержащий ключи справочников репозитория, используемых в качестве свободных
 измерений в источнике.


## Комментарии


Для получения информации об элементах по ключу свободного измерения
 используйте свойство [ItemByKey](ISearchDataSourcesFreeDimsInfo.ItemByKey.htm).
