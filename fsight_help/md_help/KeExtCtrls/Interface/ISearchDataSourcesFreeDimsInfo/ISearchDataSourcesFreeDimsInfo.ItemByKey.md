# ISearchDataSourcesFreeDimsInfo.ItemByKey


## Синтаксис


ItemByKey(Key: Integer): [ISearchDataSourcesFreeDimElements](../ISearchDataSourcesFreeDimElements/ISearchDataSourcesFreeDimElements.htm);


## Параметры


Key. Ключ свободного измерения.


## Описание


Свойство ItemByKey возвращает
 информацию об элементах свободного измерения по его ключу.


## Комментарии


Ключ необходимого свободного измерения можно получить в массиве, который
 возвращает свойство [Keys](ISearchDataSourcesFreeDimsInfo.Keys.htm).
