# ISearchDataSourcesFreeDimsInfo


Сборка: ExtCtrls;


## Описание


Интерфейс ISearchDataSourcesFreeDimsInfo
 содержит свойства и методы для работы с коллекцией свободных измерений
 источника данных.


## Иерархия наследования


           ISearchDataSourcesFreeDimsInfo


## Комментарии


Информацию о свободных измерениях можно получить в свойстве [ISearchDataSourcesItem.FreeDims](../ISearchDataSourcesItem/ISearchDataSourcesItem.FreeDims.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ISearchDataSourcesFreeDimsInfo.Count.htm)
		 Свойство Count возвращает
		 количество свободных измерений, информация о которых имеется в
		 коллекции.


		 ![](../../Property_Image.gif)
		 [ItemByKey](ISearchDataSourcesFreeDimsInfo.ItemByKey.htm)
		 Свойство ItemByKey
		 возвращает информацию об элементах свободного измерения по его
		 ключу.


		 ![](../../Property_Image.gif)
		 [Keys](ISearchDataSourcesFreeDimsInfo.Keys.htm)
		 Свойство Keys возвращает
		 массив, содержащий ключи справочников репозитория, используемых
		 в качестве свободных измерений в источнике.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Contains](ISearchDataSourcesFreeDimsInfo.Contains.htm)
		 Метод Contains осуществляет
		 проверку и возвращает признак наличия информации об указанном
		 измерении в коллекции.
