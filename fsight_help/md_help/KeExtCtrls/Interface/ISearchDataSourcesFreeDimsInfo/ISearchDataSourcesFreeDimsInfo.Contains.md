# ISearchDataSourcesFreeDimsInfo.Contains


## Синтаксис


Contains(Key: Integer): Boolean;


## Параметры


Key. Ключ измерения.


## Описание


Метод Contains осуществляет
 проверку и возвращает признак наличия информации об указанном измерении
 в коллекции.


## Комментарии


Метод возвращает значение True,
 если измерение с указанным ключом используется в источнике данных в качестве
 свободного измерения и информация о нем имеется в текущей коллекции.
