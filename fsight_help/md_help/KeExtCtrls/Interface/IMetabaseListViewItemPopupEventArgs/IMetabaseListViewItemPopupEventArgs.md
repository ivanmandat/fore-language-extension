# IMetabaseListViewItemPopupEventArgs


Сборка: ExtCtrls;


## Описание


Интерфейс IMetabaseListViewItemPopupEventArgs
 содержит свойства аргумента события, наступающего при вызове контекстного
 меню для элемента компонента.


## Иерархия наследования


           [IEventArgs](ForeSys.chm::/Interface/IEventArgs/IEventArgs.htm)


           IMetabaseListViewItemPopupEventArgs


## Комментарии


Интерфейс используется в событии [MetabaseListView.OnGetItemPopup](../../Class/MetabaseListView/MetabaseListView.OnGetItemPopup.htm).
