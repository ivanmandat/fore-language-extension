# IMetabaseTreeList.ShowEmptyFolders


## Синтаксис


ShowEmptyFolders: Boolean;


## Описание


Свойство ShowEmptyFolders определяет
 признак отображения пустых папок.


## Комментарии


По умолчанию свойству установлено значение True,
 при этом в дереве отображаются все папки репозитория. Если свойству установить
 значение False, то в дереве будут
 отображаться только те папки, в которых содержатся объекты, удовлетворяющие
 установленным [фильтрам объектов](IMetabaseTreeList.Filters.htm).
