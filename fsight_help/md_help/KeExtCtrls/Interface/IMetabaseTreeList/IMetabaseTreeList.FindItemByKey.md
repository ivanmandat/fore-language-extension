# IMetabaseTreeList.FindItemByKey


## Синтаксис


FindItemByKey(Key: Integer): [IMetabaseTreeListNode](../IMetabaseTreeListNode/IMetabaseTreeListNode.htm);


## Параметры


Key.
 Ключ объекта репозитория, для которого необходимо найти соответствующий
 элемент в дереве компонента.


## Описание


Метод FindItemByKey осуществляет
 поиск вершины в дереве компонента, соответствующей объекту репозитория,
 ключ которого передается посредством параметра Key.
 В случае удачного поиска метод возвращает параметры вершины, иначе значение
 Null.
