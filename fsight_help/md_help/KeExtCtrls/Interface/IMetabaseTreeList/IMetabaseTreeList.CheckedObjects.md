# IMetabaseTreeList.CheckedObjects


## Синтаксис


CheckedObjects: [IMetabaseObjectDescriptors](KeSom.chm::/Interface/IMetabaseObjectDescriptors/IMetabaseObjectDescriptors.htm);


## Описание


Свойство CheckedObjects возвращает
 коллекцию описаний объектов, флаги которых отмечены в компоненте. Актуально,
 если свойству [Checkboxes](ModForms.chm::/Interface/ITreeControl/ITreeControl.Checkboxes.htm)
 установлено значение True.
