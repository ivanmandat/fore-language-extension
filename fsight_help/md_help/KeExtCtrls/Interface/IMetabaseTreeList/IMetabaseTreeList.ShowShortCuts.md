# IMetabaseTreeList.ShowShortCuts


## Синтаксис


ShowShortCuts: Boolean;


## Описание


Свойство ShowShortCuts определяет
 признак отображения ярлыков в дереве элементов компонента.


## Комментарии


По умолчанию свойству установлено значение True,
 при этом в дереве элементов отображаются как сами объекты репозитория,
 так и объекты, являющиеся ярлыками на объекты репозитория. Если свойству
 установить значение False, то
 в дереве элементов ярлыки на объекты репозитория отображаться не будут.
