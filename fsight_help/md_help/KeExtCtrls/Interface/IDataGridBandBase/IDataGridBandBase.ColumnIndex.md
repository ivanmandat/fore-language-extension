# IDataGridBandBase.ColumnIndex


## Синтаксис


ColumnIndex: Integer;


## Описание


Свойство ColumnIndex возвращает
 индекс столбца или контейнера на текущем уровне вложенности. Контейнеры
 и столбцы индексируются отдельно.
