# IDataGridBandBase.RowIndex


## Синтаксис


RowIndex: Integer;


## Описание


Свойство RowIndex возвращает
 индекс строки, в которой находится заголовок объекта, для которого вызвано
 свойство. Контейнеры и столбцы индексируются отдельно.
