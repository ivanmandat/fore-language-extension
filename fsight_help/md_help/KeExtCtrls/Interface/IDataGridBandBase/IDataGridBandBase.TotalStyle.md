# IDataGridBandBase.TotalStyle


## Синтаксис


TotalStyle: [DataGridTotalStyle](../../Enums/DataGridTotalStyle.htm);


## Описание


Свойство TotalStyle определяет
 вид итогов, рассчитываемых для данного объекта.


## Комментарии


Если свойству установлено значение [DataGridTotalStyle.Value](../../Enums/DataGridTotalStyle.htm),
 то для здания значения итогов необходимо использовать свойство [TotalValue](IDataGridBandBase.TotalValue.htm).
