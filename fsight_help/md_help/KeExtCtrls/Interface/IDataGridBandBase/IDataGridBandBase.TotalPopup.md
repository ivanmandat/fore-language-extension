# IDataGridBandBase.TotalPopup


## Синтаксис


TotalPopup: [IPopupMenu](ModForms.chm::/Interface/IPopupMenu/IPopupMenu.htm);


## Описание


Свойство TotalPopup определяет
 контекстное меню, которое будет появляться по щелчку дополнительной кнопки
 мыши в области итогов данного объекта.


## Комментарии


В список допустимых значений входят все компоненты [PopupMenu](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/PopupMenu.htm),
 расположенные на форме.
