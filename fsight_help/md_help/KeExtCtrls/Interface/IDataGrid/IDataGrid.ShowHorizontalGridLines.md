# IDataGrid.ShowHorizontalGridLines


## Синтаксис


ShowHorizontalGridLines: Boolean;


## Описание


Свойство ShowHorizontalGridLines
 определяет признак отображения горизонтальных линий сетки таблицы.


## Комментарии


Если свойству установлено значение True,
 то горизонтальные линии сетки отображаются, иначе не отображаются. По
 умолчанию свойству установлено значение True.
