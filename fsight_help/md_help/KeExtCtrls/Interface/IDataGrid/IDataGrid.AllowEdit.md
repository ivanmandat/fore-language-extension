# IDataGrid.AllowEdit


## Синтаксис


AllowEdit: Boolean;


## Описание


Свойство AllowEdit определяет
 возможность редактирования записей в таблице.


## Комментарии


Если свойству установлено значение True,
 то будет возможность редактировать записи в таблице.
