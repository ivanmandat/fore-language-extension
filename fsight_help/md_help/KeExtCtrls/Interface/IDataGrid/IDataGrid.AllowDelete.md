# IDataGrid.AllowDelete


## Синтаксис


AllowDelete: Boolean;


## Описание


Свойство AllowDelete определяет
 возможность удаления записей в таблице.


## Комментарии


По умолчанию свойству установлено значение True,
 при этом в таблице есть возможность удалять строки с помощью методов [IDataGridRows.Delete](../IDataGridRows/IDataGridRows.Delete.htm)
 или [IDataGridRow.Delete](../IDataGridRow/IDataGridRow.Delete.htm).
 Если свойству установить значение False,
 то будет установлен запрет на удаление строк таблицы.
