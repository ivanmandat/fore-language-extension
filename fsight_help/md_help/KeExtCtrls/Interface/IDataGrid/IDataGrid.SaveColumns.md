# IDataGrid.SaveColumns


## Синтаксис


SaveColumns: String;


## Описание


Метод SaveColumns осуществляет
 сохранение параметров столбцов таблицы.


## Комментарии


Результатом работы метода является XML-срока, содержащая параметры столбцов
 таблицы. Сохраняются следующие параметры: размеры, видимость, расположение
 столбцов. В дальнейшем для восстановления сохраненных параметров используйте
 метод [LoadColumns](IDataGrid.LoadColumns.htm).
