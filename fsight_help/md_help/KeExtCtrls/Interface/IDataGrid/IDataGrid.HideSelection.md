# IDataGrid.HideSelection


## Синтаксис


HideSelection: Boolean;


## Описание


Свойство HideSelection определяет
 видимость установленной отметки при потере фокуса компонентом. Если свойству
 установлено значение False, то
 отметка не скрывается.
