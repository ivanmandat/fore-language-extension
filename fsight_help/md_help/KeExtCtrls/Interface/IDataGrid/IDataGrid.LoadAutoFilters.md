# IDataGrid.LoadAutoFilters


## Синтаксис


LoadAutoFilters(Value: String);


## Параметры


Value - XML-строка, содержащая
 параметры настройки автофильтров таблицы.


В качестве значения данного параметра необходимо указывать значение,
 полученное в результате работы метода [SaveAutoFilters](IDataGrid.SaveAutoFilters.htm).


## Описание


Метод LoadAutoFilters загружает
 параметры автофильтров таблицы.


## Комментарии


При выполнении метода загруженные параметры будут применены к столбцам
 таблицы. Также загружаются и устанавливаются параметры сортировки столбцов.
