# IDataGrid.VisualFocusedRow


## Синтаксис


VisualFocusedRow: Integer;


## Описание


Свойство VisualFocusedRow определяет
 индекс видимой сфокусированной строки таблицы.


## Комментарии


Данный индекс устанавливается только для видимых строк, рассчитывается
 в зависимости от установленной сортировки и фильтрации.
