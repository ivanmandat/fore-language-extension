# IDataGrid.AutoFilter


## Синтаксис


AutoFilter: [AutoFilterType](../../Enums/AutoFilterType.htm);


## Описание


Свойство AutoFilter определяет
 признак использования автофильтра в столбцах.


## Комментарии


Если свойству установлено значение AutoFilterType.Inplace, появится
 возможность настроить автофильтр для столбцов. При этом в заголовке столбца
 будет отображаться кнопка ![](../knopka_drop.gif),
 используемая для настройки условий автофильтра.
