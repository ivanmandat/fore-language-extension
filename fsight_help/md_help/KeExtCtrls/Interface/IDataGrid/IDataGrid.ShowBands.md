# IDataGrid.ShowBands


## Синтаксис


ShowBands: Boolean;


## Описание


Свойство ShowBands определяет
 признак отображения заголовков контейнеров. Контейнеры будут видны, если
 свойства [ShowHeaders](IDataGrid.ShowHeaders.htm) и ShowBands
 будут установлены в True.
