# IDataGrid.VisualToReal


## Синтаксис


VisualToReal(Index: Integer): Integer;


## Параметры


Index - индекс строки, среди
 фактически отображаемых строк.


## Описание


Метод VisualToReal возвращает
 абсолютный индекс строки, по индексу строки среди фактически отображаемых
 строк.


## Комментарии


Метод учитывает установленную фильтрацию и сортировку.
