# IDataGrid.HeaderHeight


## Синтаксис


HeaderHeight: Integer;


## Описание


Свойство HeaderHeight определяет
 высоту заголовков столбцов и контейнеров компонента.


## Комментарии


Значение высоты указывается в пикселях. По умолчанию свойству установлено
 значение 24 пикселя. Допустимое значение свойства находится в диапазоне
 [1, 100000].
