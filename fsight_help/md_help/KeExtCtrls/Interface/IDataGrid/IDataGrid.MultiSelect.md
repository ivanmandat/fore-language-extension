# IDataGrid.MultiSelect


## Синтаксис


MultiSelect: Boolean;


## Описание


Свойство MultiSelect определяет возможность множественной отметки строк компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).


## Комментарии


Если свойству установлено значение True, то для выделения множества строк можно использовать клавиши SHIFT и CTRL:


При удерживании клавишу SHIFT и щелчке по строке N - происходит отметка всех строк с текущей строки по строку N (предыдущая отметка снимается).


При удерживании клавиши SHIFT+CTRL и щелчке по строке N - происходит отметка всех строк с текущей строки по строку N (предыдущее отметка сохраняется).


При отметке строки, удерживая нажатой клавишу CTRL, эта отметка будет инвертироваться. Если строка сфокусирована но не выделенна - она подсвечиваться голубым цветом.
