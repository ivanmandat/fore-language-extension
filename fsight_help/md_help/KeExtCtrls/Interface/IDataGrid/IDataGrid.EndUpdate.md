# IDataGrid.EndUpdate


## Синтаксис


EndUpdate;


## Описание


Метод EndUpdate отключает режим
 запрета перерисовки компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).
 Метод используется для избежания мерцания компонента при изменении его
 свойств. Данный метод используется совместно с методом [BeginUpdate](IDataGrid.BeginUpdate.htm).
