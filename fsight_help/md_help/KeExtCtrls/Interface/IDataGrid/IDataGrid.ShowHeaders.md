# IDataGrid.ShowHeaders


## Синтаксис


ShowHeaders: Boolean;


## Описание


Свойство ShowHeaders определяет
 признак отображения заголовков контейнеров и столбцов таблицы.


## Комментарии


Если свойству установлено значение False,
 то заголовки контейнеров и столбцов отображаться не будут. Если свойству
 установлено значение True, то
 заголовки столбцов отображаются, а видимость контейнеров отпределяет свойством
 [ShowBands](IDataGrid.ShowBands.htm).
