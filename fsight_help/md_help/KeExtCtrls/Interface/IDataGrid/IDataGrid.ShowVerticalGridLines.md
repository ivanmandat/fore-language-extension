# IDataGrid.ShowVerticalGridLines


## Синтаксис


ShowVerticalGridLines: Boolean;


## Описание


Свойство ShowVerticalGridLines
 определяет признак отображения вертикальных линий сетки таблицы.


## Комментарии


Если свойству установлено значение True,
 то вертикальные линии сетки отображаются, иначе не отображаются. По умолчанию
 свойству установлено значение True.
