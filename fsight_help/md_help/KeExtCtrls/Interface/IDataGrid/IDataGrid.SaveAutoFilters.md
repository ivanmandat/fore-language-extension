# IDataGrid.SaveAutoFilters


## Синтаксис


SaveAutoFilters: String;


## Описание


Метод SaveAutoFilters осуществляет
 сохранение параметров автофильтров, установленных для столбцов таблицы.


## Комментарии


Результатом работы метода является XML-строка, содержащая параметры
 автофильтров, а также параметры сортировки, установленной по столбцам
 таблицы. В дальнейшем для восстановления сохраненных параметров используйте
 метод [LoadAutoFilters](IDataGrid.LoadAutoFilters.htm).
