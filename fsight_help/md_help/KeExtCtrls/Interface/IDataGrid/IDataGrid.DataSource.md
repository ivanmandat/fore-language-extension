# IDataGrid.DataSource


## Синтаксис


DataSource: [IUiDataSource](../IUiDataSource/IUiDataSource.htm);


## Описание


Свойство DataSource указывает
 источник данных, с которым связан компонент.


## Комментарии


Список допустимых значений включает имена всех компонентов [UiDataSource](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiDataSource.htm),
 расположенных на форме. Данное свойство связывает визуальный компонент
 доступа к данным с набором данных, который указан в свойстве [DataSet](../IUiDataSource/IUiDataSource.Dataset.htm).
