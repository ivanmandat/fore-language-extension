# IDataGrid.LoadColumns


## Синтаксис


LoadColumns(Value: String);


## Параметры


Value - XML-строка, содержащая
 параметры столбцов таблицы.


В качестве значения данного параметра необходимо указывать значение,
 полученное в результате работы метода [SaveColumns](IDataGrid.SaveColumns.htm).


## Описание


Метод LoadColumns загружает
 параметры столбцов таблицы.


## Комментарии


При выполнении метода загруженные параметры будут установлены для столбцов
 таблицы.
