# IDataGrid.FixedBorderWidth


## Синтаксис


FixedBorderWidth: Integer;


## Описание


Свойство FixedBorderWidth определяет
 толщину границы между фиксированными контейнерами.


## Комментарии


Доступ к фиксированным контейнерам предоставляет свойство [RootBands](IDataGrid.RootBands.htm).
