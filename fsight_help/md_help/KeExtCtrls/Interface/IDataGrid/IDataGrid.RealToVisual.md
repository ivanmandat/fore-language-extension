# IDataGrid.RealToVisual


## Синтаксис


RealToVisual(Index: Integer): Integer;


## Параметры


Index - абсолютный индекс строки.


## Описание


Метод RealToVisual возвращает
 индекс видимой строки по абсолютному индексу строки в таблице.


## Комментарии


Метод учитывает установленную фильтрацию и сортировку, возвращает "-1"
 если строка не входит в число видимых строк.
