# IDataGrid.VisualRowCount


## Синтаксис


VisualRowCount: Integer;


## Описание


Свойство VisualRowCount возвращает
 количество строк, фактически отображаемых в таблице.


## Комментарии


Данное свойство зависит от установленной фильтрации в таблице.
