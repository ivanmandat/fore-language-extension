# IDataGrid.AllowAppend


## Синтаксис


AllowAppend: Boolean;


## Описание


Свойство AllowAppend определяет
 возможность добавления записей.


## Комментарии


Если свойству установлено значение True,
 то будет возможность добавить запись в таблицу. Если источник данных допускает
 редактирование, то в него также будет добавляться новая запись.
