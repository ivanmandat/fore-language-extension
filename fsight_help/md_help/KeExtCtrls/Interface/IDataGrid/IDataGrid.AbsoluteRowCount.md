# IDataGrid.AbsoluteRowCount


## Синтаксис


AbsoluteRowCount: Integer;


## Описание


Свойство AbsoluteRowCount определяет
 абсолютное количество строк, присутствующих в таблице. При установленном
 источнике данных, количество строк таблицы определяется количеством записей
 в источнике данных. Данное свойство не зависит от установленной фильтрации
 в таблице.
