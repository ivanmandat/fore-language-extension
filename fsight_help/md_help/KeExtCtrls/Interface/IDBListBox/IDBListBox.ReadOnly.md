# IDBListBox.ReadOnly


## Синтаксис


ReadOnly: Boolean;


## Описание


Свойство ReadOnly определяет,
 будет ли пользователь иметь возможность изменять значение поля текущей
 записи.


## Комментарии


Если свойству установить значение True,
 то компонент будет использоваться только для отображения данных, изменять
 содержимое поля запрещено. При установке в False,
 набор данных находится в режиме редактирования и пользователь имеет возможность
 редактирования содержимого поля.
