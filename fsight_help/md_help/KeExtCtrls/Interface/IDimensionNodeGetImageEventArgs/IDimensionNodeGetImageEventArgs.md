# IDimensionNodeGetImageEventArgs


Сборка: ExtCtrls;


## Описание


Интерфейс IDimensionNodeGetImageEventArgs
 содержит свойства аргумента события, возникающего при получении изображения
 элемента дерева компонентов [DimensionCombo](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DimensionCombo.htm)
 и [DimensionTree](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DimensionTree.htm).
