# IMetaAttributesBreadcrumb.RootText


## Синтаксис


RootText: String;


## Описание


Свойство RootText определяет
 текст для дополнительной кнопки объединенного элемента.


## Комментарии


Свойство актуально, если свойству [ExtendedRootItem](IMetaAttributesBreadcrumb.ExtendedRootItem.htm)
 установлено значение True.


Если текст длинный, то длина объединенного элемента будет ограничена
 и составит не более 75 единиц окна (при использовании стандартного
 шрифта компонентов платформы - приблизительно 128 пикселей). При щелчке
 по дополнительной кнопке длина объединенного элемента будет увеличен таким
 образом, чтобы был отображен остальной текст.


Примечание.
 При установке текста в свойстве RootText
 длина объединенного элемента изменяется автоматически. Длина объединенного
 элемента, которая будет установлена после щелчка по дополнительной кнопке,
 составляет не более 75% от длины компонента.
