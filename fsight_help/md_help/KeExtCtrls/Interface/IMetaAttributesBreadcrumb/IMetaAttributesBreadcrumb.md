 содержит свойства и методы компонента среды разработки - [MetaAttributesBreadcrumb](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/MetaAttributesBreadcrumb.htm).

## Комментарии


Компонент [MetaAttributesBreadcrumb](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/MetaAttributesBreadcrumb.htm) используется для создания цепочки навигации по
 временным рядам, содержащимся в базе данных временных рядов.
