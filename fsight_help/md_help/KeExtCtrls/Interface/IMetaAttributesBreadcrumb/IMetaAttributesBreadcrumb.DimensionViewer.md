# IMetaAttributesBreadcrumb.DimensionViewer


## Синтаксис


DimensionViewer: [IDimensionViewerBase](../IDimensionViewerBase/IDimensionViewerBase.htm);


## Описание


Свойство DimensionViewer возвращает
 объект для работы с раскрывающимся списком компонента.


## Комментарии


Данное свойство возвращает объект для работы с раскрывающимся списком
 цепочки навигации по временным рядам, содержащимся в базе данных временных
 рядов.
