# IMetaAttributesBreadcrumb.HistoryButton


## Синтаксис


HistoryButton: Boolean;


## Описание


Свойство HistoryButton определяет признак отображения в компоненте кнопки, позволяющей выбрать комбинацию и отметку атрибутов, которая была установлена ранее.


## Комментарии


Запись в историю компонента осуществляется после изменения комбинации отображаемых атрибутов, их отметки и очистки содержимого компонента с помощью кнопки ![](../knopka_clear.gif). По умолчанию свойству установлено значение False, при этом данная кнопка не отображается.
