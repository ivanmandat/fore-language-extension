# IMetaAttributesBreadcrumb.SelectionMode


## Синтаксис


SelectionMode: [SelectionModeType](ModForms.chm::/Enums/SelectionModeType.htm);


## Описание


Свойство SelectionMode определяет
 режим отметки показателей по атрибутам, отображаемым в компоненте.


## Комментарии


По умолчанию свойству установлено значение SelectionModeType.MultiSelect.
