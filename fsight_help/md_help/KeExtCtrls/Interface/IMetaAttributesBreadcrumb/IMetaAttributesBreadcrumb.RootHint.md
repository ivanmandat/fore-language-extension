# IMetaAttributesBreadcrumb.RootHint


## Синтаксис


RootHint: String;


## Описание


Свойство RootHint определяет
 текст всплывающей подсказки для дополнительной кнопки объединенного элемента.


## Комментарии


Свойство актуально, если свойству [ExtendedRootItem](IMetaAttributesBreadcrumb.ExtendedRootItem.htm)
 установлено значение True.


## Пример


Пример использования свойства приведен в описании свойства [ExtendedRootItem](IMetaAttributesBreadcrumb.ExtendedRootItem.htm).
