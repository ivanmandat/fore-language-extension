# IMetaAttributesBreadcrumb.LabelDropDown


## Синтаксис


LabelDropDown: Boolean;


## Описание


Свойство LabelDropDown определяет признак отображения раскрывающегося списка, содержащего список показателей атрибута, при выборе элемента в списке компонента.


## Комментарии


По умолчанию свойству установлено значение True. В этом режиме при выборе элемента компонента, будет раскрываться список показателей атрибута, которому соответствует данный элемент. Если свойству установить значение False, то при выборе элемента компонента будет выдаваться контекстное меню, используемое для управления списком атрибутов, отображаемых в компоненте.
