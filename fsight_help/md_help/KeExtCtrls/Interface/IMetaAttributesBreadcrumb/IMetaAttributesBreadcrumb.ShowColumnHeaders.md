# IMetaAttributesBreadcrumb.ShowColumnHeaders


## Синтаксис


ShowColumnHeaders: Boolean;


## Описание


Свойство ShowColumnHeaders определяет
 признак отображения заголовков столбцов в раскрывающемся списке значений
 атрибутов базы данных временных рядов.


## Комментарии


Если свойству установлено значение True,
 то в раскрывающемся списке, выводимом при выборе соответствующего элемента
 компонента, будут отображаться заголовки столбцов. Если атрибут базы данных
 временных рядов ссылается на какой-либо справочник НСИ, то будет возможность
 отобразить дополнительные атрибуты соответствующего справочника НСИ.


По умолчанию данному свойству установлено значение False.
