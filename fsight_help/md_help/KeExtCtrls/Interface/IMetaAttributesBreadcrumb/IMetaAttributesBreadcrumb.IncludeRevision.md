# IMetaAttributesBreadcrumb.IncludeRevision


## Синтаксис


IncludeRevision: Boolean;


## Описание


Свойство IncludeRevision определяет признак отображения атрибута с ревизиями в компоненте.


## Комментарии


По умолчанию свойству установлено значение False, при этом атрибут с ревизиями не отображается.


Примечание. При изменении значения данного свойства во время выполнения формы будет необходимо переподключение к базе данных временных рядов.
