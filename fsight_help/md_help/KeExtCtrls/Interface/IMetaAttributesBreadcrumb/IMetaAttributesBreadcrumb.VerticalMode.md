# IMetaAttributesBreadcrumb.VerticalMode


## Синтаксис


VerticalMode: Boolean;


## Описание


Свойство VerticalMode определяет
 режим расположения элементов в компоненте.


## Комментарии


Допустимые значения:


	- True. Элементы располагаются
	 вертикально, в виде дерева.

	![](IMetaAttributesBreadcrumb_VerticalMode_True.gif)


	- False. Элементы располагаются
	 горизонтально в виде цепочки навигации.

	![](IMetaAttributesBreadcrumb_VerticalMode_False.gif)
