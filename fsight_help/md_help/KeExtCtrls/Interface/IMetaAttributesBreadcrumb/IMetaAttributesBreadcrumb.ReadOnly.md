# IMetaAttributesBreadcrumb.ReadOnly


## Синтаксис


ReadOnly: Boolean;


## Описание


Свойство ReadOnly определяет
 запрет изменения настроек компонента через визуальный интерфейс.


## Комментарии


Допустимые значения:


	- True. В компоненте нельзя
	 изменить набор отображаемых атрибутов, их порядок и отметку, через
	 визуальный интерфейс. Данные изменения могут быть произведены через
	 прикладной код с использованием соответствующих свойств и методов
	 компонента.


	- False.
	 В компоненте можно изменить набор отображаемых атрибутов, их порядок
	 и отметку, через визуальный интерфейс.
