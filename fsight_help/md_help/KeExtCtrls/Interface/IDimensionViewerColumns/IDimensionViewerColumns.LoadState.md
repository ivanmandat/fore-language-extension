# IDimensionViewerColumns.LoadState


## Синтаксис


LoadState(Value: String);


## Параметры


Value. XML-структура с параметрами
 состояния колонок.


## Описание


Метод LoadState загружает параметры
 состояния колонок компонента.


## Комментарии


В качестве значения параметра Value
 передаются параметры состояния, полученные при выполнении метода [SaveState](IDimensionViewerColumns.SaveState.htm).
