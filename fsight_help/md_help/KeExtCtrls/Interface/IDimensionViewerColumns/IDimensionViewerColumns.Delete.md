# IDimensionViewerColumns.Delete


## Синтаксис


Delete(ColumnIndex: Integer);


## Параметры


ColumnIndex.
 Входной параметр, опредялющий индекс столбца, который необходимо удалить.


## Описание


Метод Delete осуществляет удаление
 столбца, индекс которого передается посредством входного параметра ColumnIndex.
