# IDimensionViewerColumns


Сборка: ExtCtrls;


## Описание


Интерфейс IDimensionViewerColumns
 содержит свойства и методы для работы с коллекцией столбцов компонентов
 [DimensionCombo](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DimensionCombo.htm)
 и [DimensionTree](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DimensionTree.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDimensionViewerColumns.Count.htm)
		 Свойство Count возвращает
		 количество столбцов в коллекции.


		 ![](../../Property_Image.gif)
		 [DimensionViewer](IDimensionViewerColumns.DimensionViewer.htm)
		 Свойство DimensionViewer
		 возвращает объект-родитель.


		 ![](../../Property_Image.gif)
		 [Item](IDimensionViewerColumns.Item.htm)
		 Свойство Item возвращает
		 параметры указанного столбца.

		 Имя метода

		 [Add](IDimensionViewerColumns.Add.htm)
