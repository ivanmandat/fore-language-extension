# IMetabaseDialogClassFilter


Сборка: ExtCtrls;


## Описание


Интерфейс IMetabaseDialogClassFilter
 содержит свойства и методы фильтра, позволяющего отобразить в диалоге
 объекты определенного типа.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ObjectClass](IMetabaseDialogClassFilter.ObjectClass.htm)
		 Свойство ObjectClass
		 определяет тип объектов, которые будут отображаться в диалоге
		 при использовании данного фильтра.


## Свойства, унаследованные от [IMetabaseDialogFilter](../IMetabaseDialogFilter/IMetabaseDialogFilter.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Description](../IMetabaseDialogFilter/IMetabaseDialogFilter.Description.htm)
		 Свойство Description
		 определяет описание фильтра, которое будет отображаться в комбинированном
		 списке фильтров диалога.


		 ![](../../Property_Image.gif)
		 [IsVisible](../IMetabaseDialogFilter/IMetabaseDialogFilter.IsVisible.htm)
		 Свойство IsVisible
		 возвращает признак видимости объекта репозитория в диалоге.
