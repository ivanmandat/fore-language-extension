# IMetaAttributesEdit.RemoveAttribute


## Синтаксис


RemoveAttribute(Value: [IMetaAttribute](KeRds.chm::/Interface/IMetaAttribute/IMetaAttribute.htm));


## Параметры


Value. Атрибут базы данных временных рядов, значения которой необходимо удалить из списка подстановок компонента.


## Описание


Метод RemoveAttribute осуществляет удаление значений атрибута, передаваемого посредством параметра Value, из списка подстановок компонента.
