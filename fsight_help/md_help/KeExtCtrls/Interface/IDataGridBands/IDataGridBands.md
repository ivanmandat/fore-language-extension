# IDataGridBands


Сборка: ExtCtrls;


## Описание


Интерфейс IDataGridBands содержит
 свойства и методы для работы с коллекцией контейнеров компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDataGridBands.Count.htm)
		 Свойство Count возвращает
		 количество контейнеров в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IDataGridBands.Item.htm)
		 Свойство Item возвращает
		 параметры контейнера.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IDataGridBands.Add.htm)
		 Метод Add осуществляет
		 добавление нового контейнера.


		 ![](../../Sub_Image.gif)
		 [Clear](IDataGridBands.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех контейнеров компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).


		 ![](../../Sub_Image.gif)
		 [Delete](IDataGridBands.Delete.htm)
		 Метод Delete осуществляет
		 удаление контейнера.
