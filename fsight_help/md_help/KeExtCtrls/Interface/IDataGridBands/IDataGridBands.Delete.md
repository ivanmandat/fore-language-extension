# IDataGridBands.Delete


## Синтаксис


Delete(BandIndex: Integer);


## Параметры


BandIndex -
 номер контейнера, который необходимо удалить.


## Описание


Метод Delete осуществляет удаление
 контейнера, индекс которого передается посредством параметра BandIndex.
