# IDataGridBands.Add


## Синтаксис


Add(Title: String): [IDataGridBand](../IDataGridBand/IDataGridBand.htm);

Title -
 заголовок создаваемого контейнера.


## Описание


Метод Add осуществляет добавление
 нового контейнера, заголовок которого передается в качестве входного параметра
 Title.
