# ISearchDataSourcesCalendarElement


Сборка: ExtCtrls;


## Описание


Интерфейс ISearchDataSourcesCalendarElement
 содержит свойства для получения информации об индексированном календарном
 измерении.


## Иерархия наследования


           ISearchDataSourcesCalendarElement


## Комментарии


Параметры календарного измерения можно получить в свойстве [ISearchDataSourcesCalendarsInfo.ItemByKey](../ISearchDataSourcesCalendarsInfo/ISearchDataSourcesCalendarsInfo.ItemByKey.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CalendarKey](ISearchDataSourcesCalendarElement.CalendarKey.htm)
		 Свойство CalendarKey
		 возвращает ключ календарного справочника репозитория, который
		 используется в качестве измерения в источнике данных.


		 ![](../../Property_Image.gif)
		 [CalendarLevel](ISearchDataSourcesCalendarElement.CalendarLevel.htm)
		 Свойство CalendarLevel
		 возвращает уровень календаря, элементы которого проиндексированы.


		 ![](../../Property_Image.gif)
		 [EndDate](ISearchDataSourcesCalendarElement.EndDate.htm)
		 Свойство EndDate возвращает
		 конечную дату, на которую имеются индексированные данные измерения.


		 ![](../../Property_Image.gif)
		 [StartDate](ISearchDataSourcesCalendarElement.StartDate.htm)
		 Свойство StartDate
		 возвращает начальную дату, на которую имеются индексированные
		 данные измерения.
