# ISearchDataSourcesCalendarsInfo.ItemByKey


## Синтаксис


ItemByKey(Key: Integer): [ISearchDataSourcesCalendarElement](../ISearchDataSourcesCalendarElement/ISearchDataSourcesCalendarElement.htm);


## Параметры


Key. Ключ календарного измерения.


## Описание


Свойство ItemByKey возвращает
 параметры индексированного календарного измерения по его ключу.


## Комментарии


Ключ необходимого календарного измерения можно получить в массиве, который
 возвращает свойство [Keys](ISearchDataSourcesCalendarsInfo.Keys.htm).
