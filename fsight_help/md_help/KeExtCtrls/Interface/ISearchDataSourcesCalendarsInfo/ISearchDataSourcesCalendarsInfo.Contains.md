# ISearchDataSourcesCalendarsInfo.Contains


## Синтаксис


Contains(Key: Integer): Boolean;


## Параметры


Key. Ключ календарного измерения.


## Описание


Метод Contains осуществляет
 проверку и возвращает признак наличия информации об указанном календарном
 измерении в коллекции.


## Комментарии


Метод возвращает значение True,
 если календарное измерение с указанным ключом используется в источнике
 данных и информация о нем имеется в текущей коллекции.
