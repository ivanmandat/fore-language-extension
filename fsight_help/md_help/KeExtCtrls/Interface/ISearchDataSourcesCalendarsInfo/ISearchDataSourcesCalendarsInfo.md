# ISearchDataSourcesCalendarsInfo


Сборка: ExtCtrls;


## Описание


Интерфейс ISearchDataSourcesCalendarsInfo
 содержит свойства и методы для работы с коллекций календарных измерений,
 которые используются в источнике данных и информация о них индексирована.


## Иерархия наследования


           ISearchDataSourcesCalendarsInfo


## Комментарии


Информацию о календарных измерениях можно получить в свойстве [ISearchDataSourcesItem.Calendars](../ISearchDataSourcesItem/ISearchDataSourcesItem.Calendars.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ISearchDataSourcesCalendarsInfo.Count.htm)
		 Свойство Count возвращает
		 количество календарных измерений, информация о которых имеется
		 в коллекции.


		 ![](../../Property_Image.gif)
		 [ItemByKey](ISearchDataSourcesCalendarsInfo.ItemByKey.htm)
		 Свойство ItemByKey
		 возвращает параметры индексированного календарного измерения по
		 его ключу.


		 ![](../../Property_Image.gif)
		 [Keys](ISearchDataSourcesCalendarsInfo.Keys.htm)
		 Свойство Keys возвращает
		 массив, содержащий ключи календарных справочников репозитория,
		 используемых в качестве измерений в источнике.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Contains](ISearchDataSourcesCalendarsInfo.Contains.htm)
		 Метод Contains осуществляет
		 проверку и возвращает признак наличия информации об указанном
		 календарном измерении в коллекции.
