# IMetabaseTreeListNodePopupEventArgs


Сборка: ExtCtrls;


## Описание


Интерфейс IMetabaseTreeListNodePopupEventArgs
 содержит свойства аргумента события, возникающего при вызове контекстного
 меню для какого-либо элемента дерева компонентов [MetabaseTreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/MetabaseTreeCombo.htm)
 и [MetabaseTreeList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/MetabaseTreeList.htm).
