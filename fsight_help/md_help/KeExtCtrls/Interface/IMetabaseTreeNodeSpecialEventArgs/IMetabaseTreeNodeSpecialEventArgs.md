# IMetabaseTreeNodeSpecialEventArgs


Сборка: ExtCtrls;


## Описание


Интерфейс IMetabaseTreeNodeSpecialEventArgs
 содержит свойства аргумента различных событий, связанных с настройкой
 отображения элементов в дереве компонентов [MetabaseTreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/MetabaseTreeCombo.htm)
 и [MetabaseTreeList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/MetabaseTreeList.htm).
