# IDataGridBandBaseOptions.AllowPress


## Синтаксис


AllowPress: Boolean;


## Описание


Свойство AllowPress определяет
 возможность заголовка столбца или контейнера быть нажатым.


## Комментарии


Если свойству установлено значение False,
 то нажатие по заголовку будет запрещено. При этом перемещение объекта
 так же будет запрещено. Изменение места расположения будет осуществляться
 с помощью метода [SetParentBand](../IDataGridBandBase/IDataGridBandBase.SetParentBand.htm).
