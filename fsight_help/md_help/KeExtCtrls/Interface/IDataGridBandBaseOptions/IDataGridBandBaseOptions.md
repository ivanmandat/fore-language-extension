# IDataGridBandBaseOptions


Сборка: ExtCtrls;


## Описание


Интерфейс IDataGridbandBaseOptions
 содержит дополнительные свойства столбцов и контейнеров компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AllowPress](IDataGridBandBaseOptions.AllowPress.htm)
		 Свойство AllowPress
		 определяет возможность заголовка столбца или контейнера быть нажатым.


		 ![](../../Property_Image.gif)
		 [FixedWidth](IDataGridBandBaseOptions.FixedWidth.htm)
		 Свойство FixedWidth
		 определяет возможность фиксации ширины столбцов или контейнеров
		 при подборе ширины столбцов.


		 ![](../../Property_Image.gif)
		 [HotTrack](IDataGridBandBaseOptions.HotTrack.htm)
		 Свойство HotTrack определяет
		 признак выделения заголовка объекта другим цветом при наведении
		 на него курсора мыши.


		 ![](../../Property_Image.gif)
		 [Moveable](IDataGridBandBaseOptions.Moveable.htm)
		 Свойство Moveable определяет
		 возможность перемещения столбца или контейнера с помощью
		 мыши.


		 ![](../../Property_Image.gif)
		 [Resizable](IDataGridBandBaseOptions.Resizable.htm)
		 Свойство Resizable
		 определяет возможность изменения размера столбца или контейнера
		 с помощью мыши.
