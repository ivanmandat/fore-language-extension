# IMetabaseListView.SelectedObjects


## Синтаксис


SelectedObjects: [IMetabaseObjectDescriptors](KeSom.chm::/Interface/IMetabaseObjectDescriptors/IMetabaseObjectDescriptors.htm);


## Описание


Свойство SelectedObjects возвращает
 коллекцию описаний объектов, которые выделены в списке компонента.


## Комментарии


Полный список объектов, отображаемых в текущий момент в компоненте,
 можно получить используя свойство [Items](ModForms.chm::/Interface/IListView/IListView.Items.htm).
 Каждый элемент списка может быть приведён к интерфейсу [IMetabaseListViewItem](../IMetabaseListViewItem/IMetabaseListViewItem.htm).
