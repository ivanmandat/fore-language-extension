# IMetabaseListView.Tree


## Синтаксис


Tree: [IMetabaseTreeList](../IMetabaseTreeList/IMetabaseTreeList.htm);


## Описание


Свойство Tree определяет компонент,
 который будет управлять содержимым текущего компонента.


## Комментарии


В качестве значения свойства необходимо указать один из доступных компонентов
 [MetabaseTreeList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/MetabaseTreeList.htm).
 После установки свойства Tree
 в текущем компоненте [MetabaseListView](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/MetabaseListView.htm)
 будет отображаться содержимое объекта, который будет выбран в [MetabaseTreeList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/MetabaseTreeList.htm).
 Свойство [Root](IMetabaseListView.Root.htm) при этом игнорируется.
