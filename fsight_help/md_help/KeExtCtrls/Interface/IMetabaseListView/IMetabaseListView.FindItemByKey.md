# IMetabaseListView.FindItemByKey


## Синтаксис


FindItemByKey(Key: Integer): [IMetabaseListViewItem](../IMetabaseListViewItem/IMetabaseListViewItem.htm);


## Параметры


Key. Ключ объекта репозитория,
 по которому необходимо найти элемент в компоненте.


## Описание


Метод FindItemByKey осуществляет
 поиск элемента компонента по ключу объекта репозитория.


## Комментарии


Если поиск прошёл успешно, то метод возвращает элемент компонента. Если
 элемент не найден, то метод вернёт значение Null.
