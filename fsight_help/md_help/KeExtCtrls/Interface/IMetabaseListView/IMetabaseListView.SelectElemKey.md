# IMetabaseListView.SelectElemKey


## Синтаксис


SelectElemKey(Key: Integer, [Focus: Boolean = False]);


## Параметры


Object. Ключ объекта репозитория;


Focus. Признак необходимости
 передать фокус элементу.


## Описание


Метод SelectElemKey выделяет
 элемент компонента, который соответствует объекту репозитория с указанным
 ключом.
