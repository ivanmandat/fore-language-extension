# IMetabaseListView.Filters


## Синтаксис


Filters: [IMetabaseDialogFilters](../IMetabaseDialogFilters/IMetabaseDialogFilters.htm);


## Описание


Свойство Filters возвращает
 коллекцию фильтров, используемых в компоненте.


## Комментарии


Фильтры данной коллекции позволяют определить, какие типы объектов будут
 отображаться в компоненте.
