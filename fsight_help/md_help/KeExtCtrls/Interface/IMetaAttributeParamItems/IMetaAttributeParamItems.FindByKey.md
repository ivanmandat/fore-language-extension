# IMetaAttributeParamItems.FindByKey


## Синтаксис


FindByKey(StubKey: Integer): [IMetaAttributeParamItem](../IMetaAttributeParamItem/IMetaAttributeParamItem.htm);


## Параметры


StubKey. Ключ параметра, по
 которому производится поиск.


## Описание


Метод FindByKey осуществляет
 поиск по ключу и возвращает информацию о параметре в случае удачного поиска.


## Комментарии


Если параметр не найден, то метод возвращает значение Null.
