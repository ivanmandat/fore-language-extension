# IMetaAttributeParamItems.FindById


## Синтаксис


FindById(StubId: String): [IMetaAttributeParamItem](../IMetaAttributeParamItem/IMetaAttributeParamItem.htm);


## Параметры


StubId. Идентификатор параметра,
 по которому производится поиск.


## Описание


Метод FindById осуществляет
 поиск по идентификатору и возвращает информацию о параметре в случае удачного
 поиска.


## Комментарии


Если параметр не найден, то метод возвращает значение Null.
