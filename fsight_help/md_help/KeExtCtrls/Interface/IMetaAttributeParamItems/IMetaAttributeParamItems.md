# IMetaAttributeParamItems


Сборка: ExtCtrls;


## Описание


Интерфейс IMetaAttributeParamItems
 содержит свойства и методы для работы с коллекций параметров атрибута.


## Иерархия наследования


           IMetaAttributeParamItems


## Комментарии


Параметры, добавленные в данную коллекцию, будут доступны для установки
 в качестве значения атрибута.

		 [Count](IMetaAttributeParamItems.Count.htm)

## Методы


		 Имя метода
		 Краткое описание
