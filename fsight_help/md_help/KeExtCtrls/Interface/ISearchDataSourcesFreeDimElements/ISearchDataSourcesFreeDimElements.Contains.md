# ISearchDataSourcesFreeDimElements.Contains


## Синтаксис


Contains(Key: String): Boolean;


## Параметры


Key. Ключ элемента.


## Описание


Метод Contains осуществляет
 проверку и возвращает признак наличия указанного ключа элемента в коллекции.


## Комментарии


Метод возвращает значение True,
 если элемент с указанным ключом присутствует в текущей коллекции.
