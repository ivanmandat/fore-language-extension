# ISearchDataSourcesFreeDimElements.ItemByKey


## Синтаксис


ItemByKey(Key: String): String;


## Параметры


Key. Ключ элемента измерения.


## Описание


Свойство ItemByKey возвращает
 наименование элемента с указанным ключом.


## Комментарии


Ключ необходимого элемента можно получить в массиве, который возвращает
 свойство [Keys](ISearchDataSourcesFreeDimElements.Keys.htm).
