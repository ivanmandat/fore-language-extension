# ISearchDataSourcesFreeDimElements


Сборка: ExtCtrls;


## Описание


Интерфейс ISearchDataSourcesFreeDimElements
 содержит свойства и методы для работы с коллекцией элементов свободного
 измерения.


## Иерархия наследования


           ISearchDataSourcesFreeDimElements


## Комментарии


Коллекцию элементов свободного измерения можно получить в свойстве [ISearchDataSourcesFreeDimsInfo.ItemByKey](../ISearchDataSourcesFreeDimsInfo/ISearchDataSourcesFreeDimsInfo.ItemByKey.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ISearchDataSourcesFreeDimElements.Count.htm)
		 Свойство Count возвращает
		 количество элементов в коллекции.


		 ![](../../Property_Image.gif)
		 [DimensionKey](ISearchDataSourcesFreeDimElements.DimensionKey.htm)
		 Свойство DimensionKey
		 возвращает ключ свободного измерения, которому принадлежит текущая
		 коллекция элементов.


		 ![](../../Property_Image.gif)
		 [ItemByKey](ISearchDataSourcesFreeDimElements.ItemByKey.htm)
		 Свойство ItemByKey
		 возвращает наименование элемента с указанным ключом.


		 ![](../../Property_Image.gif)
		 [Keys](ISearchDataSourcesFreeDimElements.Keys.htm)
		 Свойство Keys возвращает
		 массив ключей элементов, содержащихся в коллекции.
