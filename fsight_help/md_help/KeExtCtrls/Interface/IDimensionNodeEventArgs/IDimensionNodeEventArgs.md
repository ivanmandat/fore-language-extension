# IDimensionNodeEventArgs


Сборка: ExtCtrls;


## Описание


Интерфейс IDimensionNodeEventArgs
 содержит базовые свойства аргумента событий, возникающих при каких-либо
 действиях, связанных с элементами дерева компонентов [DimensionCombo](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DimensionCombo.htm)
 и [DimensionTree](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DimensionTree.htm).
