# IDataGridBandEventArgs

## Описание


Интерфейс IDataGridBandEventArgs
 содержит свойства аргумента событий, связанных с каким-либо контейнером
 компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).
