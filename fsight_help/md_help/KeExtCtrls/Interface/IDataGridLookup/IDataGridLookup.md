# IDataGridLookup


Сборка: ExtCtrls;


## Описание


Интерфейс IDataGridLookup содержит
 свойства для настройки отображения ячейки при использовании справочников.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dimension](IDataGridLookup.Dimension.htm)
		 Свойство Dimension
		 определяет справочник, по элементам которого будет осуществляться
		 поиск введенных значений.
