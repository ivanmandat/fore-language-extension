# IUiParam.IsNull


## Синтаксис


IsNull: Boolean;


## Описание


Свойство IsNull позволяет определить,
 является ли значение параметра ненулевым.


## Комментарии


Свойство возвращает True, если
 значение параметра пустое. В противном случае возвращает False.
