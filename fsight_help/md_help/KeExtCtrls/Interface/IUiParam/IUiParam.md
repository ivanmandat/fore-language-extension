# IUiParam


Сборка: ExtCtrls;


## Описание


Интерфейс IUiParam содержит
 свойства и методы для работы с параметром компонента [UiQuery](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiQuery.htm).
