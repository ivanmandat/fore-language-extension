# IUiParam.AssignField


## Синтаксис


AssignField(Field: [IUiField](../IUiField/IUiField.htm));


## Параметры


Field.
 Поле источника данных, в соответствии с параметрами которого, необходимо
 настроить свойства параметра запроса.


## Описание


Метод AssignField настраивает
 свойства параметра в соответствии с параметрами поля Field.


## Комментарии


Для параметра будут установлены наименование, тип данных и значение
 поля. Значение берется из текущей записи поля источника данных.
