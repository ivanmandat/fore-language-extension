# IUiParam.AssignFieldValue


## Синтаксис


AssignFieldValue(Field: [IUiField](../IUiField/IUiField.htm));


## Параметры


Field. Поле источника данных,
 в соответствии с параметрами которого, необходимо настроить значение параметра
 запроса.


## Описание


Метод AssignFieldValue настраивает
 значение параметра в соответствии с параметрами поля Field.


## Комментарии


Для параметра будут установлены тип данных и значение поля. Значение
 берется из текущей записи поля источника данных.
