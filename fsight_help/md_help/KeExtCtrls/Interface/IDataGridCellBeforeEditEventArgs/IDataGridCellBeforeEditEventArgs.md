# IDataGridCellBeforeEditEventArgs


Сборка: ExtCtrls;


## Описание


Интерфейс IDataGridCellBeforeEditEventArgs
 содержит свойства аргумента события, наступающего перед редактированием
 значения ячейки.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Cancel](IDataGridCellBeforeEditEventArgs.Cancel.htm)
		 Свойство Cancel определяет,
		 будет ли возможность редактировать значение ячейки.
