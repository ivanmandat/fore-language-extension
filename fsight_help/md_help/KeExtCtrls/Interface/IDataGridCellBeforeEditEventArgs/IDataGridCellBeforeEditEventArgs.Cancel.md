# IDataGridCellBeforeEditEventArgs.Cancel


## Синтаксис


Cancel: Boolean;


## Описание


Свойство Cancel определяет признак
 возможности редактирования значения ячейки.


## Комментарии


Если свойству установлено значение True,
 то редактирование ячейки запрещено.
