## Описание


Интерфейс IDBLookupComboBox
 содержит свойства и методы компонента среды разработки - [DBLookupComboBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DBLookupComboBox.htm).

		 Свойство DropDownRows

		 списка.


		 ![](../../Property_Image.gif)

		 Свойство DataSource

		 ![](../../Property_Image.gif)

		 Свойство Field возвращает

		 ![](../../Property_Image.gif)
		 [KeyField](../IDBLookupControl/IDBLookupControl.KeyField.htm)
		 Свойство KeyField определяет
		 поле из набора данных [ListSource](../IDBLookupControl/IDBLookupControl.ListSource.htm),
		 значение которого должно соответствовать значению поля [DataField](../IDBLookupControl/IDBLookupControl.DataField.htm).


		 ![](../../Property_Image.gif)
		 [KeyValue](../IDBLookupControl/IDBLookupControl.KeyValue.htm)
		 Свойство KeyValue определяет
		 значение ключевого поля в активной строке источника данных.


		 ![](../../Property_Image.gif)
		 [ListField](../IDBLookupControl/IDBLookupControl.ListField.htm)
		 Свойство ListField
		 определяет одно или несколько полей, значения которых отображаются
		 в данном компоненте.


		 ![](../../Property_Image.gif)
		 [ListSource](../IDBLookupControl/IDBLookupControl.ListSource.htm)
		 Свойство ListSource
		 определяет источник данных для компонента поиска, то есть источник,
		 содержащий поля [KeyField](../IDBLookupControl/IDBLookupControl.KeyField.htm)
		 и [ListField](../IDBLookupControl/IDBLookupControl.ListField.htm).


		 ![](../../Property_Image.gif)
		 [ReadOnly](../IDBLookupControl/IDBLookupControl.ReadOnly.htm)
		 Свойство ReadOnly определяет
		 возможность редактирования данных компонента.
