# IDBLookupComboBox.ListVisible


## Синтаксис


ListVisible: Boolean;


## Описание


Свойство ListVisible возвращает
 значение видимости элементов раскрывающегося списка.


## Комментарии


Если свойству установлено значение True,
 то раскрывающийся список развернут, иначе - свернут.
