# IDataGridRows.AbsoluteCount


## Синтаксис


AbsoluteCount: Integer;


## Описание


Свойство AbsoluteCount возвращает
 абсолютное количество строк, присутствующих в таблице.


## Комментарии


При установленном источнике данных, количество строк таблицы определяется
 количеством записей в источнике данных. Данное свойство не зависит от
 установленной фильтрации в таблице.
