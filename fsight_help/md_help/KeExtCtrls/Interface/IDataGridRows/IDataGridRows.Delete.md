# IDataGridRows.Delete


## Синтаксис


Delete(Index: Integer);


## Параметры


Index.
 Индекс строки, которую необходимо удалить.


## Описание


Метод Delete удаляет строку,
 индекс которой передается в качестве входного параметра Index.
