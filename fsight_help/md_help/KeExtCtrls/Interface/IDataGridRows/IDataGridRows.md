# IDataGridRows


Сборка: ExtCtrls;


## Описание


Интерфейс IDataGridRows реализует
 свойства и методы коллекции строк компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).


## Иерархия наследования


           IDataGridRows


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AbsoluteCount](IDataGridRows.AbsoluteCount.htm)
		 Свойство AbsoluteCount
		 возвращает абсолютное количество строк, присутствующих в таблице.


		 ![](../../Property_Image.gif)
		 [DataGrid](IDataGridRows.DataGrid.htm)
		 Свойство DataGrid возвращает
		 объект-родитель.


		 ![](../../Property_Image.gif)
		 [Focused](IDataGridRows.Focused.htm)
		 Свойство Focused определяет
		 строку, на которой установлен фокус.


		 ![](../../Property_Image.gif)
		 [ImageIndex](IDataGridRows.ImageIndex.htm)
		 Свойство ImageIndex
		 определяет индекс изображения, которое будет использоваться в
		 ячейке таблицы.


		 ![](../../Property_Image.gif)
		 [Item](IDataGridRows.Item.htm)
		 Свойство Item возвращает
		 параметры указанной строки.


		 ![](../../Property_Image.gif)
		 [VisualCount](IDataGridRows.VisualCount.htm)
		 Свойство VisualCount
		 возвращает количество строк, фактически отображаемых в таблице.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IDataGridRows.Add.htm)
		 Метод Add добавляет
		 новую строку и возвращает ссылку на эту строку.


		 ![](../../Sub_Image.gif)
		 [AdjustHeight](IDataGridRows.AdjustHeight.htm)
		 Метод AdjustHeight
		 изменяет высоту строк таким образом, чтобы содержимое полностью
		 помещалось в ячейки.


		 ![](../../Sub_Image.gif)
		 [Clear](IDataGridRows.Clear.htm)
		 Метод Clear удаляет
		 все строки таблицы.


		 ![](../../Sub_Image.gif)
		 [Delete](IDataGridRows.Delete.htm)
		 Метод Delete удаляет
		 строку, индекс которой передается в качестве входного параметра.
