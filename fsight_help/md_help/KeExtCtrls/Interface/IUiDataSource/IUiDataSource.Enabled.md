# IUiDataSource.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет
 признак передачи данных в компоненты визуального отображения данных.


## Комментарии


Если свойству установлено значение True,
 то данные будут передаваться во все визуальные компоненты, для которых
 в качестве источника данных установлен данный компонент [UiDataSource](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiDataSource.htm).
