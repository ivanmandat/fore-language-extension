# IUiDataSource.IsLinkedTo


## Синтаксис


IsLinkedTo(Dataset: [IUiDataSet](../IUiDataSet/IUiDataSet.htm)):
 Boolean;


## Параметры


Dataset.
 Набор данных, с которым будет проверяться связь источника данных.


## Описание


Метод IsLinkedTo осуществляет
 проверку наличия связи источника данных [UiDataSource](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiDataSource.htm)
 с набором данных [Dataset](IUiDataSource.Dataset.htm).
 Если связь существует, то возвращает True,
 иначе - False.
