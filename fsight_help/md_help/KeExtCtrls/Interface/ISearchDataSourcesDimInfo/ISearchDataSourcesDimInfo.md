# ISearchDataSourcesDimInfo


Сборка: ExtCtrls;


## Описание


Интерфейс ISearchDataSourcesDimInfo
 содержит свойства для получения информации об индексированном элементе
 измерения.


## Иерархия наследования


           ISearchDataSourcesDimInfo


## Комментарии


Информацию об индексированном элементе измерения можно получить в свойстве
 [ISearchDataSourcesSelectionInfo.Item](../ISearchDataSourcesSelectionInfo/ISearchDataSourcesSelectionInfo.Item.htm).
