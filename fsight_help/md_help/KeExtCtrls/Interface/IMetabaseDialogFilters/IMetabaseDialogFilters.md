# IMetabaseDialogFilters


Сборка: ExtCtrls;


## Описание


Интерфейс IMetabaseDialogFilters
 содержит свойства и методы для работы с коллекцией фильтров, используемых
 для фильтрации объектов репозитория.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMetabaseDialogFilters.Count.htm)
		 Свойство Count возвращает
		 количество фильтров в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMetabaseDialogFilters.Item.htm)
		 Свойство Item возвращает
		 фильтр из коллекции.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddFilter](IMetabaseDialogFilters.AddFilter.htm)
		 Метод AddFilter добавляет
		 в коллекцию фильтр.


		 ![](../../Sub_Image.gif)
		 [Clear](IMetabaseDialogFilters.Clear.htm)
		 Метод Clear очищает
		 всю коллекцию фильтров.


		 ![](../../Sub_Image.gif)
		 [Remove](IMetabaseDialogFilters.Remove.htm)
		 Метод Remove удаляет
		 фильтр из коллекции.
