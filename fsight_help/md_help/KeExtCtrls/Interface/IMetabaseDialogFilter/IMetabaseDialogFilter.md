# IMetabaseDialogFilter


Сборка: ExtCtrls;


## Описание


Интерфейс IMetabaseDialogFilter
 содержит базовые свойства и методы фильтра объектов репозитория.


## Комментарии


Данный интерфейс является базовым для следующих интерфейсов:


	- [IMetabaseDialogAllFilter](../IMetabaseDialogAllFilter/IMetabaseDialogAllFilter.htm);


	- [IMetabaseDialogClassFilter](../IMetabaseDialogClassFilter/IMetabaseDialogClassFilter.htm);


	- [IMetabaseDialogCombiFilter](../IMetabaseDialogCombiFilter/IMetabaseDialogCombiFilter.htm);


	- [IMetabaseDialogMetaclassFilter](../IMetabaseDialogMetaclassFilter/IMetabaseDialogMetaclassFilter.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Description](IMetabaseDialogFilter.Description.htm)
		 Свойство Description
		 определяет описание фильтра, которое будет отображаться в комбинированном
		 списке фильтров диалога.


		 ![](../../Property_Image.gif)
		 [IsVisible](IMetabaseDialogFilter.IsVisible.htm)
		 Свойство IsVisible
		 возвращает признак видимости объекта репозитория в диалоге.
