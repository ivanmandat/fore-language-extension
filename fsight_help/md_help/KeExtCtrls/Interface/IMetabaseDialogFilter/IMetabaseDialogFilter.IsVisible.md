# IMetabaseDialogFilter.IsVisible


## Синтаксис


IsVisible(Object: [IMetabaseObjectDescriptor](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm)):
 Boolean;


## Параметры


Object.
 Объект репозитория, для которого необходимо определить видимость.


## Описание


Свойство IsVisible возвращает
 признак видимости объекта репозитория в диалоге. Описание объекта передается
 посредством входного параметра Object.
