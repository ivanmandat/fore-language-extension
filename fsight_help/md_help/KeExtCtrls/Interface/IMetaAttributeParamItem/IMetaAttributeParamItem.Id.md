# IMetaAttributeParamItem.Id


## Синтаксис


Id: String;


## Описание


Свойство Id определяет идентификатор
 параметра.


## Комментарии


Идентификатор должен быть уникален в рамках коллекции, в которую добавлен
 параметр.
