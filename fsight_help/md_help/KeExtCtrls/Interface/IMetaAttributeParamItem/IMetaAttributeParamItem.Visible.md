# IMetaAttributeParamItem.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет
 признак отображения параметра в списке параметров, которые можно установить
 в качестве значения атрибута.


## Комментарии


По умолчанию свойству установлено значение True,
 при этом параметр доступен для выбора. Если свойству установить значение
 False, то параметр будет скрыт
 из списка.
