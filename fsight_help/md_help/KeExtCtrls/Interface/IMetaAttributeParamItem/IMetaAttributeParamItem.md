# IMetaAttributeParamItem


Сборка: ExtCtrls;


## Описание


Интерфейс IMetaAttributeParamItem
 содержит свойства параметра, используемого в атрибутах, отображаемых в
 компоненте [MetaAttributesBreadcrumb](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/MetaAttributesBreadcrumb.htm).
