# IMetaAttributeParamItem.Tag


## Синтаксис


Tag: Variant;


## Описание


Свойство Tag определяет данные,
 связанные с данным параметром.


## Комментарии


В данном свойстве могут храниться любые пользовательские данные. Эти
 данные можно будет получить в событии [OnMetaAttributeParam](../../Class/MetaAttributesBreadcrumb/MetaAttributesBreadcrumb.OnMetaAttributeParam.htm)
 при установке параметра в качестве значения атрибута.


## Пример


Пример использования данного свойства приведен в описании метода [IMetaAttributeParamItems.Add](../IMetaAttributeParamItems/IMetaAttributeParamItems.Add.htm).
