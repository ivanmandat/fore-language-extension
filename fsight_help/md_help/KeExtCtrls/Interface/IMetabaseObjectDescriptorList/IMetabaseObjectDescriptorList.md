# IMetabaseObjectDescriptorList


Сборка: ExtCtrls;


## Описание


Интерфейс IMetabaseObjectDescriptorList
 содержит свойства для работы с коллекций описаний объектов, полученной
 в результате работы диалога открытия или сохранения объектов репозитория.


## Иерархия наследования


           IMetabaseObjectDescriptorList


## Комментарии


Коллекцию объектов, выбранных в диалоге, возвращает свойство [IMetabaseDialog.Objects](../IMetabaseDialog/IMetabaseDialog.Objects.htm).
