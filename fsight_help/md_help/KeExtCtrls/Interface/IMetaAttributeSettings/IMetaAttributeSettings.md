# IMetaAttributeSettings


Сборка: ExtCtrls;


## Описание


Интерфейс IMetaAttributeSettings
 содержит свойства и методы для работы с коллекцией параметров атрибутов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMetaAttributeSettings.Count.htm)
		 Свойство Count возвращает
		 количество элементов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMetaAttributeSettings.Item.htm)
		 Свойство Item возвращает
		 параметры атрибута из коллекции по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindByAttribute](IMetaAttributeSettings.FindByAttribute.htm)
		 Метод FindByAttribute
		 осуществляет поиск параметров для указанного атрибута.


		 ![](../../Sub_Image.gif)
		 [FindById](IMetaAttributeSettings.FindById.htm)
		 Метод FindById осуществляет
		 поиск параметров атрибута по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IMetaAttributeSettings.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск параметров атрибута по ключу.


		 ![](../../Sub_Image.gif)
		 [IndexOf](IMetaAttributeSettings.IndexOf.htm)
		 Метод IndexOf возвращает
		 индекс элемента в коллекции.


		 ![](../../Sub_Image.gif)
		 [Move](IMetaAttributeSettings.Move.htm)
		 Метод Move осуществляет
		 перемещение элементов в коллекции.
