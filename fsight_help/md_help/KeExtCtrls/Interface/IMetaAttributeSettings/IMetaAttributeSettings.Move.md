# IMetaAttributeSettings.Move


## Синтаксис


Move(IndexFrom: Integer; IndexTo: Integer);


## Параметры


IndexFrom. Индекс позиции элемента,
 который необходимо переместить;


IndexTo. Индекс позиции, в
 которую необходимо переместить элемент.


## Описание


Метод Move осуществляет перемещение
 элементов в коллекции.
