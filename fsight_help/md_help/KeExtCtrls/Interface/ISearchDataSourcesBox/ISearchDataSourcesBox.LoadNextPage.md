# ISearchDataSourcesBox.LoadNextPage


## Синтаксис


LoadNextPage;


## Описание


Метод LoadNextPage осуществляет
 продолжение поиска и загружает в область компонента следующий набор элементов
 с результатами поиска.


## Комментарии


Метод используется, если свойство [IsAllDataLoaded](ISearchDataSourcesBox.IsAllDataLoaded.htm)
 возвращает значение False. После
 [поиска](ISearchDataSourcesBox.StartSearch.htm) в коллекцию
 [Items](ISearchDataSourcesBox.Items.htm)
 загружаются первые сто записей с результатом. Для продолжения поиска и
 загрузки следующих записей используйте метод LoadNextPage.
