# ISearchDataSourcesBox.CancelSearch


## Синтаксис


CancelSearch;


## Описание


Метод CancelSearch прерывает
 процесс BI-поиска.


## Комментарии


Если в репозитории проиндексировано большое количество источников, то
 процесс BI-поиска может занимать продолжительное время. При необходимости
 поиск можно прервать, используя метод CancelSearch.
