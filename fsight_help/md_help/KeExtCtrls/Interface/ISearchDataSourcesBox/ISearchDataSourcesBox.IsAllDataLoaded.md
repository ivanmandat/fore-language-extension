# ISearchDataSourcesBox.IsAllDataLoaded


## Синтаксис


IsAllDataLoaded: Boolean;


## Описание


Свойство IsAllDataLoaded возвращает
 флаг, указывающий все ли элементы с результатами поиска загружены в область
 компонента.


## Комментарии


Допустимые значения:


	- True. В области компонента
	 отображаются все элементы с результатами поиска.


	- False. В области компонента
	 отображаются не все элементы с результатами поиска, следующий набор
	 элементов может быть подгружен с помощью метода [LoadNextPage](ISearchDataSourcesBox.LoadNextPage.htm).
