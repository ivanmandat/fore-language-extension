# IDataGridRow.ColumnValue


## Синтаксис


ColumnValue(ColumnIndex: Integer): Variant;


## Параметры


ColumnIndex.
 Индекс столбца, с которым пересекается текущая строка.


## Описание


Свойство ColumnValue определяет
 значение строки в столбце, индекс которого передается в качестве входного
 параметра ColumnIndex.
