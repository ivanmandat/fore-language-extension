# IDataGridRow.Index


## Синтаксис


Index: Integer;


## Описание


Свойство Index возвращает индекс
 строки в компоненте [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).


Примечание.
 Данное значение соответствует индексу строки среди видимых строк компонента.
 Зависит от установленной сортировки и фильтрации. Если строка не отображается
 в компоненте, то свойство возвращает значение «-1».
