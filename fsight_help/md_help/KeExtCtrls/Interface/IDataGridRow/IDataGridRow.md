# IDataGridRow


Сборка: ExtCtrls;


## Описание


Интерфейс IDataGridRow содержит
 свойства и методы строки компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AbsoluteIndex](IDataGridRow.AbsoluteIndex.htm)
		 Свойство AbsoluteIndex
		 возвращает абсолютный индекс строки в таблице.


		 ![](../../Property_Image.gif)
		 [ColumnValue](IDataGridRow.ColumnValue.htm)
		 Свойство ColumnValue
		 определяет значение строки в столбце, индекс которого передается
		 в качестве входного параметра.


		 ![](../../Property_Image.gif)
		 [DataGrid](IDataGridRow.DataGrid.htm)
		 Свойство DataGrid возвращает
		 объект-родитель данной строки.


		 ![](../../Property_Image.gif)
		 [Filtered](IDataGridRow.Filtered.htm)
		 Свойство Filtered возвращает
		 признак отображения текущей строки после применения фильтрации
		 в компоненте [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).


		 ![](../../Property_Image.gif)
		 [Focused](IDataGridRow.Focused.htm)
		 Свойство Focused определяет
		 признак установки фокуса на текущей строке.


		 ![](../../Property_Image.gif)
		 [Height](IDataGridRow.Height.htm)
		 Свойство Height определяет
		 высоту строки таблицы.


		 ![](../../Property_Image.gif)
		 [Index](IDataGridRow.Index.htm)
		 Свойство Index возвращает
		 индекс строки в компоненте [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).


		 ![](../../Property_Image.gif)
		 [NextRow](IDataGridRow.NextRow.htm)
		 Свойство NextRow возвращает
		 следующую строку в таблице.


		 ![](../../Property_Image.gif)
		 [PreviousRow](IDataGridRow.PreviousRow.htm)
		 Свойство PreviousRow
		 возвращает предыдущую строку в таблице.


		 ![](../../Property_Image.gif)
		 [Selected](IDataGridRow.Selected.htm)
		 Свойство Selected определяет
		 признак отметки строки.


		 ![](../../Property_Image.gif)
		 [StyleIndex](IDataGridRow.StyleIndex.htm)
		 Свойство StyleIndex
		 определяет, для текущей строки, индекс стиля из коллекции установленных
		 стилей для компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AdjustHeight](IDataGridRow.AdjustHeight.htm)
		 Метод AdjustHeight
		 изменяет высоту строки таким образом, чтобы содержимое полностью
		 помещалось в ячейки.


		 ![](../../Sub_Image.gif)
		 [BeginEdit](IDataGridRow.BeginEdit.htm)
		 Метод BeginEdit переводит
		 строку в режим редактирования.


		 ![](../../Sub_Image.gif)
		 [CancelEdit](IDataGridRow.CancelEdit.htm)
		 Метод CancelEdit прекращает
		 режим редактирования строки без сохранения изменений.


		 ![](../../Sub_Image.gif)
		 [Delete](IDataGridRow.Delete.htm)
		 Метод Delete удаляет
		 строку, для которой был вызван метод.


		 ![](../../Sub_Image.gif)
		 [EndEdit](IDataGridRow.EndEdit.htm)
		 Метод EndEdit прекращает
		 режим редактирования строки с сохранением изменений.


		 ![](../../Sub_Image.gif)
		 [IsEdit](IDataGridRow.IsEdit.htm)
		 Метод IsEdit возвращает
		 True, если данная строка
		 находится в режиме редактирования.


		 ![](../../Sub_Image.gif)
		 [IsNew](IDataGridRow.IsNew.htm)
		 Метод IsNew возвращает
		 True, если строка является
		 новой.
