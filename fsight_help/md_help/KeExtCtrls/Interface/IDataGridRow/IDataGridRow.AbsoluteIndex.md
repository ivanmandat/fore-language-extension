# IDataGridRow.AbsoluteIndex


## Синтаксис


AbsoluteIndex: Integer;


## Описание


Свойство AbsoluteIndex возвращает
 абсолютный индекс строки в таблице.


Примечание.
 Данный индекс всегда постоянный, соответствует индексу строки в источнике
 данных и не зависит от установленной сортировки и фильтрации.
