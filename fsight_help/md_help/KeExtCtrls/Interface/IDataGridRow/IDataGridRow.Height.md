# IDataGridRow.Height


## Синтаксис


Height: Integer;


## Описание


Свойство Height определяет высоту
 строки таблицы. Актуально, если свойству [RowSizingMode](../IDataGrid/IDataGrid.RowSizingMode.htm)
 установлено значение Single.
