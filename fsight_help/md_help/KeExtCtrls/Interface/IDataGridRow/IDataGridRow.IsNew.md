# IDataGridRow.IsNew


## Синтаксис


IsNew: Boolean;


## Описание


Метод IsNew возвращает True, если строка является новой.
 Строка считается новой, если она была добавлена и не редактировалась.


## Комментарии


Метод IsNew актуален только
 в том случае, если для компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm)
 в свойстве [DataSource](../IDataGrid/IDataGrid.DataSource.htm)
 определен источник данных.
