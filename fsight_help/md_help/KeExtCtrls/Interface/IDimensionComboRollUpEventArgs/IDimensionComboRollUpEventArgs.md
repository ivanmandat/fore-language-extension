# IDimensionComboRollUpEventArgs

## Описание


Интерфейс IDimensionComboRollUpEventArgs
 содержит свойства аргумента события, возникающего при сворачивании раскрывающегося
 списка компонента [DimensionCombo](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DimensionCombo.htm).
