# IDimensionViewerColumn


Сборка: ExtCtrls;


## Описание


Интерфейс IDimensionViewerColumn
 содержит свойства и методы столбца компонентов [DimensionTree](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DimensionTree.htm) и [DimensionCombo](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DimensionCombo.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Alignment](IDimensionViewerColumn.Alignment.htm)
		 Свойство Alignment
		 устанавливает способ расположения надписей элементов дерева в
		 столбце.


		 ![](../../Property_Image.gif)
		 [Attribute](IDimensionViewerColumn.Attribute.htm)
		 Свойство Attribute
		 определяет атрибут, данные которого будут отображаться в данном
		 столбце.


		 ![](../../Property_Image.gif)
		 [Caption](IDimensionViewerColumn.Caption.htm)
		 Свойство Caption определяет
		 заголовок столбца.


		 ![](../../Property_Image.gif)
		 [Columns](IDimensionViewerColumn.Columns.htm)
		 Свойство Columns возвращает
		 объект-родитель.


		 ![](../../Property_Image.gif)
		 [ImageIndex](IDimensionViewerColumn.ImageIndex.htm)
		 Свойство ImageIndex
		 определяет номер изображения из списка свойства [Images](../IDimensionViewer/IDimensionViewer.Images.htm).


		 ![](../../Property_Image.gif)
		 [Index](IDimensionViewerColumn.Index.htm)
		 Свойство Index определяет

		 ![](../../Property_Image.gif)
		 [LevelAttribute](IDimensionViewerColumn.LevelAttribute.htm)
		 Свойство LevelAttribute
		 определяет отображение атрибута измерения для уровня, который
		 передается в качестве входного параметра Level.


		 ![](../../Property_Image.gif)
		 [MaxWidth](IDimensionViewerColumn.MaxWidth.htm)
		 Свойство MaxWidth определяет
		 максимальную ширину столбца.


		 ![](../../Property_Image.gif)
		 [MinWidth](IDimensionViewerColumn.MinWidth.htm)
		 Свойство MinWidth определяет
		 минимальную ширину столбца.


		 ![](../../Property_Image.gif)
		 [SortAscending](IDimensionViewerColumn.SortAscending.htm)
		 Свойство SortAscending
		 определяет направление сортировки.


		 ![](../../Property_Image.gif)
		 [Visible](IDimensionViewerColumn.Visible.htm)
		 Свойство Visible определяет
		 видимость столбца в области компонента.


		 ![](../../Property_Image.gif)
		 [Width](IDimensionViewerColumn.Width.htm)
		 Свойство Width определяет
		 ширину столбца в пикселях.
