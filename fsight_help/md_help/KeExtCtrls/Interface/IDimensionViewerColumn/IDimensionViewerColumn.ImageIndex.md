# IDimensionViewerColumn.ImageIndex


## Синтаксис


ImageIndex: Integer;


## Описание


Свойство ImageIndex определяет
 номер изображения из списка свойства [Images](../IDimensionViewer/IDimensionViewer.Images.htm).


## Комментарии


Изображение отображается с левого края, перед наименованием столбца.
 По умолчанию свойству установлено значение -1, при этом столбец не содержит
 изображения.
