# IUiQuery.SQL


## Синтаксис

## Описание


Свойство SQL определяет текст
 SQL-запроса к базе данных, заданной свойством [Database](IUiQuery.Database.htm).

 щелчком кнопки мыши на кнопке ![](../knopka_dialog.gif)
