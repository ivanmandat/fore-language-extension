# IUiQuery.ParamCheck


## Синтаксис


ParamCheck: Boolean;


## Описание


Свойство ParamCheck определяет,
 будет ли заново генерироваться список параметров запроса (Свойство [Params](IUiQuery.Params.htm)), если свойства [SQL](IUiQuery.SQL.htm)
 и [Database](IUiQuery.Database.htm) изменились в процессе выполнения
 программы.


## Комментарии


По умолчанию свойству установлено значение True,
 при этом необходимо автоматическое заполнение свойства [Params](IUiQuery.Params.htm)
 во время выполнения.
