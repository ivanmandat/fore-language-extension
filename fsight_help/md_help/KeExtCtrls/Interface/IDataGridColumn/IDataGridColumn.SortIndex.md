# IDataGridColumn.SortIndex


## Синтаксис


SortIndex: Integer;


## Описание


Свойство SortIndex определяет
 индекс установленной сортировки.


## Комментарии


Допустимые значения: "-1" — сортировка не установлена; "0-n"
 — сортировка установлена, чем больше значение, тем больше приоритет.
