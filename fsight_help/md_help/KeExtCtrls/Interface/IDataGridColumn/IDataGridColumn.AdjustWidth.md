# IDataGridColumn.AdjustWidth


## Синтаксис


AdjustWidth([MinWidth: Integer = -1;] [MaxWidth:
 Integer = -1]);


## Параметры


MinWidth. Минимальная ширина
 столбца в пикселях при автоматической подгонке. По умолчанию - «-1», ограничение
 отсутствует;


MaxWidth. Максимальная ширина
 столбца в пикселях при автоматической подгонке. По умолчанию - «-1», ограничение
 отсутствует.


## Описание


Метод AdjustWidth выполняет
 автоматическую подгонку ширины столбца, учитывая параметры MinWidth
 и MaxWidth.
