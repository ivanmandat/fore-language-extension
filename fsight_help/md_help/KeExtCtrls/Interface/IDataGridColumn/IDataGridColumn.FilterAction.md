# IDataGridColumn.FilterAction


## Синтаксис


FilterAction: Integer;


## Описание


Свойство FilterAction возвращает
 индекс установленной фильтрации столбца.


## Комментарии


Данное свойство возвращает следующие значения:


		 Значение
		 Краткое описание


		 0
		 «Все».


		 1
		 «Первые n...».


		 2
		 «Условие...».


		 3...n-2
		 В зависимости от количества записей, выбирается фильтр по конкретной
		 записи.


		 n-1
		 «Пустые» (Доступен, если содержатся пустые значения).


		 n
		 «Непустые» (Доступен, если содержатся пустые значения).


Примечание.
 Для настройки автофильтра используется свойство [IDataGrid.TableFilter](../IDataGrid/IDataGrid.TableFilter.htm).
 При использовании нового способа задания автофильтра данное свойство будет
 возвращать значения только для фильтров «Все», «Первые n...» и «Условие...».
 При фильтрации по значениям конкретной записи данное свойство также вернет
 значение, соответствующее фильтру «Условие...».
