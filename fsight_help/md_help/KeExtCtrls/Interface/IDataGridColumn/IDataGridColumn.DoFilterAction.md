# IDataGridColumn.DoFilterAction


## Синтаксис


DoFilterAction(Value: Integer);


## Параметры


Value.
 Индекс метода фильтрации, который необходимо применить.


## Описание


Метод DoFilterAction применяет
 фильтрацию, индекс которой передается в качестве входного параметра Value.


## Комментарии


Если указанный метод фильтрации предусматривает настройку дополнительных
 параметров, то будет выведено соответствующее окно. В окне необходимо
 указать условия фильтрации и нажать кнопку «ОК».


Примечание.
 Для настройки автофильтра используется свойство [IDataGrid.TableFilter](../IDataGrid/IDataGrid.TableFilter.htm).
 Данный метод необходимо использовать только для установки фильтрации по
 конкретной записи.


Допустимые значения параметра Value:


		 Значение
		 Краткое описание


		 0
		 «Все».


		 1
		 «Первые n...».


		 2
		 «Настроить фильтр...».


		 3...n-2
		 В зависимости от количества записей, выбирается фильтр по конкретной
		 записи.


		 n-1
		 «Пустые» (Доступен, если содержатся пустые значения).


		 n
		 «Непустые» (Доступен, если содержатся пустые значения).
