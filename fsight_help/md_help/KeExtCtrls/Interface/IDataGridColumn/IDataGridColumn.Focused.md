# IDataGridColumn.Focused


## Синтаксис


Focused: Boolean;


## Описание


Свойство Focused возвращает
 признак нахождения фокуса в данной колонке.


## Комментарии


Свойство возвращает значение True,
 если сфокусированная ячейка находится на пересечении какой-либо строки
 и текущей колонки.
