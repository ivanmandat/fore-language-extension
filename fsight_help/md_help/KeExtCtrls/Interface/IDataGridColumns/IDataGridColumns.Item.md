# IDataGridColumns.Item


## Синтаксис


Item(Index: Integer): [IDataGridColumn](../IDataGridColumn/IDataGridColumn.htm);


## Параметры


Index.
 Индекс столбца, который необходимо получить.


## Описание


Свойство Item возвращает столбец
 таблицы, индекс которого передается в качестве входного параметра Index.
