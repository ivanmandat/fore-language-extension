# IDataGridColumns.Delete


## Синтаксис


Delete(ColumnIndex: Integer);


## Параметры


ColumnIndex.
 Индекс столбца, который необходимо удалить.


## Описание


Метод Delete удаляет столбец,
 индекс которого передается посредством входного параметра ColumnIndex.
