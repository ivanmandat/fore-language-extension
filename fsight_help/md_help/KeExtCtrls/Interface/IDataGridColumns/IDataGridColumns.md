# IDataGridColumns


Сборка: ExtCtrls;


## Описание


Интерфейс IDataGridColumns содержит
 свойства и методы коллекции столбцов компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDataGridColumns.Count.htm)
		 Свойство Count возвращает
		 количество столбцов компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).


		 ![](../../Property_Image.gif)
		 [DataGrid](IDataGridColumns.DataGrid.htm)
		 Свойство DataGrid возвращает
		 объект-родитель.


		 ![](../../Property_Image.gif)
		 [Item](IDataGridColumns.Item.htm)
		 Свойство Item возвращает
		 столбец таблицы.


		 ![](../../Property_Image.gif)
		 [Width](IDataGridColumns.Width.htm)
		 Свойство Width определяет
		 ширину всех столбцов в коллекции компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).


## Методы


		  Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IDataGridColumns.Add.htm)
		 Метод Add добавляет
		 новый столбец.


		 ![](../../Sub_Image.gif)
		 [Clear](IDataGridColumns.Clear.htm)
		 Метод Clear удаляет
		 все столбцы из коллекции столбцов компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).


		 ![](../../Sub_Image.gif)
		 [Delete](IDataGridColumns.Delete.htm)
		 Метод Delete удаляет
		 столбец, индекс которого передается посредством входного параметра.
