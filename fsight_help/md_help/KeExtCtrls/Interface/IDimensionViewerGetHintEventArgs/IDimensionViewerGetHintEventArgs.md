# IDimensionViewerGetHintEventArgs


Сборка: ExtCtrls;


## Описание


Интерфейс IDimensionViewerGetHintEventArgs
 содержит свойства аргумента события, возникающего при появлении всплывающей
 подсказки для элемента, над которым остановился курсор мыши.
