# IDataGridSelRowStyleEventArgs.Focused


## Синтаксис


Focused: Boolean;


## Описание


Свойство Focused возвращает
 признак нахождения фокуса в ячейке текущей строки.


## Комментарии


Свойство возвращает значение True,
 если фокус находится в одной из ячеек текущей строки.
