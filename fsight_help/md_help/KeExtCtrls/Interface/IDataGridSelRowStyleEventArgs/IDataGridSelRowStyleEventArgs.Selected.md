# IDataGridSelRowStyleEventArgs.Selected


## Синтаксис


Selected: Boolean;


## Описание


Свойство Selected возвращает
 признак выделения строки.


## Комментарии


Свойство возвращает значение True,
 если текущая строка выделена.
