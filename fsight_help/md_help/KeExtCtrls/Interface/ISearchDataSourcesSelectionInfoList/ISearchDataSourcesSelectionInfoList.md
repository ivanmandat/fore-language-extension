# ISearchDataSourcesSelectionInfoList


Сборка: ExtCtrls;


## Описание


Интерфейс ISearchDataSourcesSelectionInfoList
 содержит свойства для работы с коллекцией, в которой хранится информация
 об отметке индексированных измерений.


## Иерархия наследования


           ISearchDataSourcesSelectionInfoList


## Комментарии


Параметры коллекции можно получить в свойстве [ISearchDataSourcesItem.Selection](../ISearchDataSourcesItem/ISearchDataSourcesItem.Selection.htm).
