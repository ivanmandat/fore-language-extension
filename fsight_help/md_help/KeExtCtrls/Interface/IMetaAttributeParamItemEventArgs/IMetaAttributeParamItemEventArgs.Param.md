# IMetaAttributeParamItemEventArgs.Param


## Синтаксис


Param: [IMetaAttributeParamItem](../IMetaAttributeParamItem/IMetaAttributeParamItem.htm);


## Описание


Свойство Param возвращает параметр,
 который был выбран в качестве значения атрибута.


## Комментарии


Если параметр был сброшен, то свойство возвращает значение Null.
