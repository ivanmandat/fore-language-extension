# IMetaAttributeParamItemEventArgs


Сборка: ExtCtrls;


## Описание


Интерфейс IMetaAttributeParamItemEventArgs
 содержит свойства аргумента события, возникающего при присвоении атрибуту
 какого-либо параметра.
