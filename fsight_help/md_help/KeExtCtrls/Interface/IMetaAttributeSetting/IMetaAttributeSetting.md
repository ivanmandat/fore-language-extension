# IMetaAttributeSetting


Сборка: ExtCtrls;


## Описание


Интерфейс IMetaAttributeSetting
 содержит свойства и методы для настройки отображения атрибута.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AttributeCommands](IMetaAttributeSetting.AttributeCommands.htm)
		 Свойство AttributeCommands
		 определяет признак доступности для атрибута команд по изменению
		 его места расположения в цепочке навигации.


		 ![](../../Property_Image.gif)
		 [DimHierarchy](IMetaAttributeSetting.DimHierarchy.htm)
		 Свойство DimHierarchy
		 определяет иерархию атрибута.


		 ![](../../Property_Image.gif)
		 [MetaAttribute](IMetaAttributeSetting.MetaAttribute.htm)
		 Свойство MetaAttribute
		 возвращает атрибут, с параметрами которого ведется работа.


		 ![](../../Property_Image.gif)
		 [MetaParamItemsCollection](IMetaAttributeSetting.MetaParamItemsCollection.htm)
		 Свойство MetaParamItemsCollection
		 возвращает коллекцию параметров, которые будут доступны для установки
		 в качестве значения атрибута.


		 ![](../../Property_Image.gif)
		 [NamingFormat](IMetaAttributeSetting.NamingFormat.htm)
		 Свойство NamingFormat
		 определяет формат наименования атрибута.


		 ![](../../Property_Image.gif)
		 [OpenedDimension](IMetaAttributeSetting.OpenedDimension.htm)
		 Свойство OpenedDimension
		 определяет открытое измерение атрибута.


		 ![](../../Property_Image.gif)
		 [SelectedParam](IMetaAttributeSetting.SelectedParam.htm)
		 Свойство SelectedParam
		 определяет параметр, который установлен в качестве значения атрибута.


		 ![](../../Property_Image.gif)
		 [ShowAttribute](IMetaAttributeSetting.ShowAttribute.htm)
		 Свойство ShowAttribute
		 определяет видимость указанного атрибута справочника НСИ в раскрывающемся
		 списке значений атрибута базы данных временных рядов.


		 ![](../../Property_Image.gif)
		 [SortAttribute](IMetaAttributeSetting.SortAttribute.htm)
		 Свойство SortAttribute
		 определяет атрибут, по которому осуществляется сортировка.


		 ![](../../Property_Image.gif)
		 [SortDirectionType](IMetaAttributeSetting.SortDirectionType.htm)
		 Свойство SortDirectionType
		 определяет направление сортировки.


		 ![](../../Property_Image.gif)
		 [SortType](IMetaAttributeSetting.SortType.htm)
		 Свойство SortType определяет
		 тип сортировки.


		 ![](../../Property_Image.gif)
		 [Visible](IMetaAttributeSetting.Visible.htm)
		 Свойство Visible определяет
		 видимость атрибута.


## Методы

		 Краткое описание

		 [Refresh](IMetaAttributeSetting.Refresh.htm)

		 обновление данных по текущему атрибуту.
