# IMetaAttributeSetting.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет
 видимость атрибута.


## Комментарии


Если свойству установлено значение True,
 то атрибут будет присутствовать в списке доступных атрибутов компонента.
