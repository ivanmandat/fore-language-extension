# IMetaAttributeSetting.Refresh


## Синтаксис


Refresh;


## Описание


Метод Refresh осуществляет обновление
 данных по текущему атрибуту.


## Комментарии


Вызов метода приводит к сбросу кешированных данных по данному атрибуту.
 Новые данные будут подгружены при первом обращении к атрибуту.
