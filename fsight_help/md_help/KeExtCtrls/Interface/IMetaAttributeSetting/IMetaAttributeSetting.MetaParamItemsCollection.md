# IMetaAttributeSetting.MetaParamItemsCollection


## Синтаксис


MetaParamItemsCollection: [IMetaAttributeParamItems](../IMetaAttributeParamItems/IMetaAttributeParamItems.htm);


## Описание


Свойство MetaParamItemsCollection
 возвращает коллекцию параметров, которые будут доступны для установки
 в качестве значения атрибута.

Пример использования данного свойства приведен в описании метода [IMetaAttributeParamItems.Add](../IMetaAttributeParamItems/IMetaAttributeParamItems.Add.htm).
