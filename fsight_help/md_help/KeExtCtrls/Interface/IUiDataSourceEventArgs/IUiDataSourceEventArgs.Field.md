# IUiDataSourceEventArgs.Field


## Синтаксис


Field: [IUiField](../IUiField/IUiField.htm);


## Описание


Свойство Field возвращает поле,
 данные которого были изменены. При переходе по записям данное свойство
 возвращает значение Null.
