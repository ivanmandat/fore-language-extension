# IUiDataSourceEventArgs


Сборка: ExtCtrls;


## Описание


Интерфейс IUiDataSourceEventArgs
 содержит свойства аргумента события, возникающего при переходе на другую
 запись, либо при редактировании текущей записи источника данных.
