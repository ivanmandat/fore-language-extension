# IMetabaseOpenDialog.MultiSelect


## Синтаксис


MultiSelect: Boolean;


## Описание


Свойство MultiSelect определяет
 признак выбора в диалоге нескольких объектов. Если свойству установлено
 значение False, то в диалоге можно
 выбрать только один объект.
