# ISearchDataSourcesItem.IsRubricator


## Синтаксис


IsRubricator: Boolean;


## Описание


Свойство IsRubricator возвращает
 признак соответствия текущего элемента данным из базы данных временных
 рядов.


## Комментарии


Свойство возвращает значение True,
 если текущий элемент содержит индексированные данные базы данных временных
 рядов и False, если данные принадлежат
 стандартному кубу.
