# ISearchDataSourcesItem


Сборка: ExtCtrls;


## Описание


Интерфейс ISearchDataSourcesItem
 содержит свойства для получения параметров элемента с результатами BI-поиска.


## Иерархия наследования


           ISearchDataSourcesItem


## Комментарии


Параметры элемента можно получить в свойстве [ISearchDataSourcesItems.Item](../ISearchDataSourcesItems/ISearchDataSourcesItems.Item.htm).
