# ISearchDataSourcesItem.Selected


## Синтаксис


Selected: Boolean;


## Описание


Свойство Selected возвращает
 признак отметки текущего элемента в списке.


## Комментарии


Допустимые значения:


	- True. Текущий элемент
	 отмечен в списке.


	- False. Текущий элемент
	 не отмечен в списке.
