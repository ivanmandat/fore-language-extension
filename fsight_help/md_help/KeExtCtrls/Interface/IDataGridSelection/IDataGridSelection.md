# IDataGridSelection


Сборка: ExtCtrls;


## Описание


Интерфейс IDataGridSelection
 содержит свойства и методы коллекции отмеченных строк компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDataGridSelection.Count.htm)
		 Свойство Count возвращает
		 количество отмеченных строк в таблице.


		 ![](../../Property_Image.gif)
		 [DataGrid](IDataGridSelection.DataGrid.htm)
		 Свойство DataGrid возвращает
		 объект-родитель.


		 ![](../../Property_Image.gif)
		 [Item](IDataGridSelection.Item.htm)
		 Свойство Item возвращает
		 строку из коллекции отмеченных строк.
