# IDataGridSelection.Item


## Синтаксис


Item(Index: Integer): [IDataGridRow](../IDataGridRow/IDataGridRow.htm);


## Параметры


Index.
 Индекс отмеченной строки, которую необходимо получить.


## Описание


Свойство Item возвращает строку
 из коллекции отмеченных строк, индекс которой передается в качестве входного
 параметра Index.
