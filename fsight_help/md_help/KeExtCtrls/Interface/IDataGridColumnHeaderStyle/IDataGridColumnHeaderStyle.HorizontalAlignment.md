# IDataGridColumnHeaderStyle.HorizontalAlignment


## Синтаксис


HorizontalAlignment: [GxStringAlignment](ModDrawing.chm::/Enums/GxStringAlignment.htm);


## Описание


Свойство HorizontalAlignment
 определяет выравнивание текста заголовка по горизонтали.


## Комментарии


При выравнивании текста будет учитываться наличие знака направления
 сортировки и кнопки автофильтра. Текст будет выравниваться относительно
 области заголовка, свободной от знака сортировки и кнопки автофильтра.
