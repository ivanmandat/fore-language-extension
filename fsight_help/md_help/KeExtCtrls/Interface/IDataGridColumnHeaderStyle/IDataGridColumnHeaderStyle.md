# IDataGridColumnHeaderStyle


Сборка: ExtCtrls;


## Описание


Интерфейс IDataGridColumnHeaderStyle
 содержит свойства, определяющие параметры оформления заголовка контейнера,
 либо колонки компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Brush](IDataGridColumnHeaderStyle.Brush.htm)
		 Свойство Brush определяет
		 кисть, используемую для заливки области заголовка.


		 ![](../../Property_Image.gif)
		 [Font](IDataGridColumnHeaderStyle.Font.htm)
		 Свойство Font определяет
		 параметры шрифта, используемого для отображения текста заголовка.


		 ![](../../Property_Image.gif)
		 [HorizontalAlignment](IDataGridColumnHeaderStyle.HorizontalAlignment.htm)
		 Свойство HorizontalAlignment
		 определяет выравнивание текста заголовка по горизонтали.


		 ![](../../Property_Image.gif)
		 [VerticalAlignment](IDataGridColumnHeaderStyle.VerticalAlignment.htm)
		 Свойство VerticalAlignment
		 определяет выравнивание текста заголовка по вертикали.
