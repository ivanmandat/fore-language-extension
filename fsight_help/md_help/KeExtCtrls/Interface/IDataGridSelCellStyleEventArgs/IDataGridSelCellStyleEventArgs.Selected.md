# IDataGridSelCellStyleEventArgs.Selected


## Синтаксис


Selected: Boolean;


## Описание


Свойство Selected возвращает
 признак выделения ячейки.


## Комментарии


Свойство возвращает значение True,
 если текущая ячейка выделена.
