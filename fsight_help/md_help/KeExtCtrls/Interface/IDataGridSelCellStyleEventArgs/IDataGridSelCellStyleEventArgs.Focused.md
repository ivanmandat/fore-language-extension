# IDataGridSelCellStyleEventArgs.Focused


## Синтаксис


Focused: Boolean;


## Описание


Свойство Focused возвращает
 признак нахождения фокуса в ячейке.


## Комментарии


Свойство возвращает значение True,
 если текущая ячейка сфокусирована.
