# IMetaAttributesTreeList.ShowEmptyFolders


## Синтаксис


ShowEmptyFolders: Boolean;


## Описание


Свойство ShowEmptyFolders определяет
 признак отображения пустых папок в дереве показателей.


## Комментарии


По умолчанию свойству установлено значение False,
 при этом пустые папки не отображаются.
