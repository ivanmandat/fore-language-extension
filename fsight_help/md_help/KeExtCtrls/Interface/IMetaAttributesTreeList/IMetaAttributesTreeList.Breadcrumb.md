# IMetaAttributesTreeList.Breadcrumb


## Синтаксис


Breadcrumb: [IMetaAttributesBreadcrumb](../IMetaAttributesBreadcrumb/IMetaAttributesBreadcrumb.htm);


## Описание


Свойство Breadcrumb определяет
 компонент [MetaAttributesBreadcrumb](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/MetaAttributesBreadcrumb.htm),
 с которым связан данный компонент.


## Комментарии


Содержимое текущего компонента будет изменяться в соответствии с атрибутами
 и показателями, отображаемыми в установленном компоненте [MetaAttributesBreadcrumb](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/MetaAttributesBreadcrumb.htm).
