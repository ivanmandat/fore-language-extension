# IDataGridStyles


Сборка: ExtCtrls;


## Описание


Интерфейс IDataGridStyles содержит
 свойства и методы коллекции стилей компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDataGridStyles.Count.htm)
		 Свойство Count возвращает
		 количество стилей, которые содержатся в коллекции стилей.


		 ![](../../Property_Image.gif)
		 [Default_](IDataGridStyles.Default_.htm)
		 Свойство Default_ возвращает
		 параметры стиля, используемого по умолчанию.


		 ![](../../Property_Image.gif)
		 [Item](IDataGridStyles.Item.htm)
		 Свойство Item возвращает
		 стиль, индекс которого передается в качестве входного параметра.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IDataGridStyles.Add.htm)
		 Метод Add добавляет
		 новый стиль в коллекцию стилей компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).


		 ![](../../Sub_Image.gif)
		 [Delete](IDataGridStyles.Delete.htm)
		 Метод Delete удаляет
		 стиль из коллекции стилей, индекс которого передается в качестве
		 входного параметра.
