# IDataGridStyles.Delete


## Синтаксис


Delete(StyleIndex: Integer);


## Параметры


StyleIndex.
 Индекс стиля, который необходимо удалить.


## Описание


Метод Delete удаляет стиль из
 коллекции стилей, индекс которого передается в качестве входного параметра
 StyleIndex.
