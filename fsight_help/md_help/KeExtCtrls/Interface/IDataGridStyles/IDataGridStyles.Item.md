# IDataGridStyles.Item


## Синтаксис


Item(Index: Integer): [IDataGridStyle](../IDataGridStyle/IDataGridStyle.htm);


## Параметры


Index.
 Индекс стиля, который необходимо получить.


## Описание


Свойство Item возвращает стиль,
 индекс которого передается в качестве входного параметра Index.
