# IMetabaseTreeCombo.DelayedFill


## Синтаксис


DelayedFill: Boolean;


## Описание


Свойство DelayedFill определяет
 режим построения дерева объектов в компоненте.


## Комментарии


Если свойству установлено значение True,
 то дерево дочерних объектов будет отстраиваться при инициализации компонента.


Если свойству установлено значение False,
 то при инициализации компонента отстраиваются только корневые вершины.
 Дочерние вершины будут отстраиваться по мере разворачивания иерархии папок.
 По умолчанию свойству установлено значение True.


Примечание.
 Для настройки режима построения дерева также необходимо учитывать работу
 события [OnIsDynamicElem](../../Class/MetabaseTreeCombo/MetabaseTreeCombo.OnIsDynamicElem.htm).
