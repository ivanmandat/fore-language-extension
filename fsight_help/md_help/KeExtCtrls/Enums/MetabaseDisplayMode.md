# MetabaseDisplayMode


## Описание


Перечисление MetabaseDisplayMode
 содержит режимы отображения объектов в дереве компонента.


Используется следующими свойствами и методами:


	- [IMetabaseTreeCombo.DisplayMode](../Interface/IMetabaseTreeCombo/IMetabaseTreeCombo.DisplayMode.htm);


	- [IMetabaseTreeList.DisplayMode](../Interface/IMetabaseTreeList/IMetabaseTreeList.DisplayMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Names. В дереве компонента
		 будут отображаться наименования объектов репозитория.


		 1
		 Idents. В дереве компонента
		 будут отображаться идентификаторы объектов репозитория.


		 2
		 Both. В дереве компонента
		 будут отображаться наименования и идентификаторы объектов репозитория.
