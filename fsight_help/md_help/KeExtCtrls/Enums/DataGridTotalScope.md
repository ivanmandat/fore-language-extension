# DataGridTotalScope


## Описание


Перечисление DataGridTotalScope
 содержит типы данных, по которым будут подсчитываться итоги.


Используется следующим свойством:


	- [IDataGrid.TotalScope](../Interface/IDataGrid/IDataGrid.TotalScope.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 AllData. Подсчет итогов
		 по всем данным контейнера, вне зависимости от установленной фильтрации.


		 1
		 FilteredData. Подсчет
		 итогов по только по отфильтрованным данным контейнера.
