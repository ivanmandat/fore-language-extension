# DimensionViewerNodeState


## Описание


Перечисление DimensionViewerNodeState
 содержит варианты состояния элемента дерева.


Используется следующим свойством:


	- [IDimensionNodeGetImageEventArgs.State](../Interface/IDimensionNodeGetImageEventArgs/IDimensionNodeGetImageEventArgs.State.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Элемент никак
		 не задействован.


		 1
		 Expanded. Для элемента
		 развернуто дерево дочерних элементов.


		 2
		 Selected. Элемент отмечен.


		 3
		 ExpandedSelected. Элемент
		 отмечен и развернуто дерево его дочерних элементов.


		 4
		 Focused. Элемент находится
		 в фокусе.


		 5
		 ExpandedFocused. Элемент
		 находится в фокусе и развернуто дерево его дочерних элементов.


		 6
		 SelectedFocused. Элемент
		 отмечен и находится в фокусе.


		 7
		 ExpandedSelectedFocused.
		 Элемент отмечен, находится в фокусе и развернуто дерево его дочерних
		 элементов.
