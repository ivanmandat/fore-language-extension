# DataGridTotalStyle


## Описание


Перечисление DataGridTotalStyle
 содержит виды итогов, рассчитываемых для данного контейнера.


Используется следующим свойством:


	- [IDataGridBandBase.TotalStyle](../Interface/IDataGridBandBase/IDataGridBandBase.TotalStyle.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Расчет итогов
		 не производится.


		 1
		 Sum. Сумма значений
		 контейнера.


		 2
		 Average. Среднее значение
		 контейнера.


		 3
		 Min. Минимальное значение
		 контейнера.


		 4
		 Max. Максимальное значение
		 контейнера.


		 8
		 Value. Для установки
		 значения итога используется свойство [TotalValue](../Interface/IDataGridBandBase/IDataGridBandBase.TotalValue.htm).
