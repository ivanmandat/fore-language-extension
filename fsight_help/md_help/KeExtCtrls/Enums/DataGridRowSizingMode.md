# DataGridRowSizingMode


## Описание


Перечисление DataGridRowSizingMode
 содержит режимы изменения высоты строк компонента.


Используется следующим свойством:


	- [IDataGrid.RowSizingMode](../Interface/IDataGrid/IDataGrid.RowSizingMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 All. Для всех строк
		 устанавливается одинаковая высота.


		 1
		 Single. Для каждой
		 строки высота изменяется индивидуально.
