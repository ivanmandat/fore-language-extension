# AutoFilterType


## Описание


Перечисление AutoFilterType
 содержит варианты возможности использования автофильтра в колонках таблицы.


Используется следующим свойством:


	- [IDataGrid.AutoFilter](../Interface/IDataGrid/IDataGrid.AutoFilter.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Автофильтр не
		 используется.


		 1
		 Inplace. Использование
		 автофильтра в колонках разрешено.
