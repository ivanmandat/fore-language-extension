# FixedState


## Описание


Перечисление FixedState содержит
 фиксированные контейнеры, в которых может располагаться объект.


Используется следующим свойством:


	- [IDataGridBandBase.Fixed](../Interface/IDataGridBandBase/IDataGridBandBase.Fixed.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Расположение
		 по умолчанию в основном центральном контейнере таблицы.


		 1
		 Left. Левый фиксированный
		 контейнер таблицы.


		 2
		 Right. Правый фиксированный
		 контейнер таблицы.


## Комментарии


Доступ к контейнерам предоставляет свойство [IDataGrid.RootBands](../Interface/IDataGrid/IDataGrid.RootBands.htm).
