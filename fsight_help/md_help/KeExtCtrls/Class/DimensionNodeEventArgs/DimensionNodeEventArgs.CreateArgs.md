# DimensionNodeEventArgs.CreateArgs


## Синтаксис


CreateArgs(Viewer: [IDimensionViewer](../../Interface/IDimensionViewer/IDimensionViewer.htm);
 Node: Integer);


## Параметры


Viewer. Компонент, для которого
 сгенерировано событие;


Node. Индекс элемента, для
 которого сгенерировано событие.


## Описание


Конструктор CreateArgs создает
 аргумент событий, возникающих при каких-либо действиях, связанных с элементами
 дерева компонента.
