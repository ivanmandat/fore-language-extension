# DataGridRowStyleEventArgs.Create


## Синтаксис


Create(Grid: [IDataGrid](../../Interface/IDataGrid/IDataGrid.htm);
 Row: Integer; StyleIndex: Integer);


## Параметры


Grid. Таблица, для которой
 сгенерировано событие;


Row. Индекс строки, для которой
 сгенерировано событие;


StyleIndex. Индекс стиля, который
 необходимо применить для строки таблицы.


## Описание


Конструктор Create создает аргумент
 события, наступающего при отрисовке строки таблицы.
