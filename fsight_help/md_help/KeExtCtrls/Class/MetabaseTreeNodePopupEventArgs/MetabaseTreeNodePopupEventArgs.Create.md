# MetabaseTreeNodePopupEventArgs.Create


## Синтаксис


Create(MetabaseTreeList: [IMetabaseTreeList](../../Interface/IMetabaseTreeList/IMetabaseTreeList.htm);
 Node: [IMetabaseTreeListNode](../../Interface/IMetabaseTreeListNode/IMetabaseTreeListNode.htm));


## Параметры


MetabaseTreeList. Компонент,
 для которого генерируется событие;


Node. Вершина дерева компонента,
 для которой генерируется событие.


## Описание


Конструктор Create создает аргумент
 события, возникающего при вызове контекстного меню для какого-либо элемента
 дерева.
