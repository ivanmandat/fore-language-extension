# DimensionNodeColumnEventArgs.CreateDimNodeColumnArgs


## Синтаксис


CreateDimNodeColumnArgs(Viewer: [IDimensionViewer](../../Interface/IDimensionViewer/IDimensionViewer.htm);
 Node: Integer; Column: Integer);


## Параметры


Viewer. Компонент, для которого
 сгенерировано событие;


Node. Элемент, для которого
 сгенерировано событие;


Column. Индекс колонки, в области
 которой для элемента было сгенерировано событие.


## Описание


Конструктор CreateDimNodeColumnArgs
 создает аргумент событий, происходящих для элементов дерева в области
 какой-либо колонки компонента.
