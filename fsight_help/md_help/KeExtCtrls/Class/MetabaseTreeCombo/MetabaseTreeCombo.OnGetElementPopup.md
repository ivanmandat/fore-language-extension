# MetabaseTreeCombo.OnGetElementPopup


## Синтаксис


OnGetElementPopup(Sender: Object; Args: [IMetabaseTreeListNodePopupEventArgs](../../Interface/IMetabaseTreeListNodePopupEventArgs/IMetabaseTreeListNodePopupEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnGetElementPopup наступает
 при вызове контекстного меню для какого-либо элемента дерева.


## Комментарии


В свойстве [IMetabaseTreeListNodePopupEventArgs.PopupMenu](../../Interface/IMetabaseTreeListNodePopupEventArgs/IMetabaseTreeListNodePopupEventArgs.PopupMenu.htm)
 аргумента события определяется какое-именно контекстное меню будет вызвано
 для элемента.
