# MetaAttributesBreadcrumb


Сборка: ExtCtrls;


## Описание


Класс MetaAttributesBreadcrumb
 реализует компонент среды разработки [MetaAttributesBreadcrumb](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/MetaAttributesBreadcrumb.htm),
 используемый для создания цепочки навигации по показателям, содержащимся
 в базе данных временных рядов.
