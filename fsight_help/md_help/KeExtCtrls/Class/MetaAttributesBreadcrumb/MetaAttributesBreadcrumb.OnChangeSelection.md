# MetaAttributesBreadcrumb.OnChangeSelection


## Синтаксис


OnChangeSelection(Sender: Object; Args: [IBreadcrumbItemEventArgs](ModForms.chm::/Interface/IBreadcrumbItemEventArgs/IBreadcrumbItemEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnChangeSelection наступает
 при изменении отметки элементов по какому-либо атрибуту.


## Комментарии


Данное событие происходит непосредственно перед событием [MetaAttributesBreadcrumb.OnRollUp](MetaAttributesBreadcrumb.OnRollUp.htm).
