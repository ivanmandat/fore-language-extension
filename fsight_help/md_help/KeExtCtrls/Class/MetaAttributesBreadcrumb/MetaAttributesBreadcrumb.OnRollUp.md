# MetaAttributesBreadcrumb.OnRollUp


## Синтаксис


OnRollUp(Sender: Object; Args: [IBreadcrumbItemEventArgs](ModForms.chm::/Interface/IBreadcrumbItemEventArgs/IBreadcrumbItemEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnRollUp наступает после
 сворачивания списка элементов атрибута, которому соответствует какой-либо
 элемент компонента.
