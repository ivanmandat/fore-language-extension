# CalcCubeFormulaEdit.OnFormatCoord


## Синтаксис


OnFormatCoord(Sender: [ICalcCubeFormulaEdit](../../Interface/ICalcCubeFormulaEdit/ICalcCubeFormulaEdit.htm);
 Args: [ICalcCubeFormulaEditEventArgs](../../Interface/ICalcCubeFormulaEditEventArgs/ICalcCubeFormulaEditEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnFormatCoord возникает
 при создании терма, соответствующего координате куба.
