# DataGridColumnEventArgs.Create


## Синтаксис


Create(Grid: [IDataGrid](../../Interface/IDataGrid/IDataGrid.htm);
 Column: [IDataGridColumn](../../Interface/IDataGridColumn/IDataGridColumn.htm));


## Параметры


Grid. Таблица, для которой
 сгенерировано событие;


Column. Колонка, для которой
 сгенерировано событие.


## Описание


Конструктор Create создает аргумент
 событий, возникающих для какой-либо колонки компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).
