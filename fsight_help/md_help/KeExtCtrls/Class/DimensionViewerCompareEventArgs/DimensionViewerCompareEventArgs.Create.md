# DimensionViewerCompareEventArgs.Create


## Синтаксис


Create(DimensionViewer: [IDimensionViewer](../../Interface/IDimensionViewer/IDimensionViewer.htm);
 Column: [IDimensionViewerColumn](../../Interface/IDimensionViewerColumn/IDimensionViewerColumn.htm);
 LeftItem: Integer; RightItem: Integer);


## Параметры


DimensionViewer. Компонент,
 для которого сгенерировано событие;


Column. Столбец компонента,
 по которому осуществляется сортировка;


LeftItem. Первый сравниваемый
 элемент компонента;


RightItem. Второй сравниваемый
 элемент компонента.


## Описание


Конструктор Create аргумент
 события, наступающего при пользовательской сортировке элементов компонента.
