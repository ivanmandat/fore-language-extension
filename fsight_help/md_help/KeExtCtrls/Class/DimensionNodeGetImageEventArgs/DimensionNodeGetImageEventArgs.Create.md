# DimensionNodeGetImageEventArgs.Create


## Синтаксис


Create(Viewer: [IDimensionViewer](../../Interface/IDimensionViewer/IDimensionViewer.htm);
 Node: Integer; State: Integer; ImageIndex: Integer);


## Параметры


Viewer. Компонент, для которого
 сгенерировано событие;


Node. Индекс элемента, для
 которого сгенерировано событие;


State. Состояние элемента,
 для которого сгенерировано событие;


ImageIndex. Индекс изображения,
 которое будет располагаться слева от текста элемента.


## Описание


Конструктор Create создает аргумент
 события, возникающего при получении изображения элемента дерева компонента.
