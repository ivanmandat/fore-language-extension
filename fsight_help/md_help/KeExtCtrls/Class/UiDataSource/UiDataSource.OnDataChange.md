# UiDataSource.OnDataChange


## Синтаксис


OnDataChange(Sender: [IUiDataSource](../../Interface/IUiDataSource/IUiDataSource.htm);
 Args: [IUiDataSourceEventArgs](../../Interface/IUiDataSourceEventArgs/IUiDataSourceEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnDataChange происходит
 во время перехода на другую запись и при изменении значения поля.
