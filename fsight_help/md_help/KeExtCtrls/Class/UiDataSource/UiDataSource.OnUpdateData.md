# UiDataSource.OnUpdateData


## Синтаксис


OnUpdateData(Sender: [IUiDataSource](../../Interface/IUiDataSource/IUiDataSource.htm);
 Args: [IEventArgs](ForeSys.chm::/Interface/IEventArgs/IEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnUpdateData происходит
 перед занесением измененных данных в БД.
