# DimensionAllowActionEventArgs.Create


## Синтаксис


Create(Viewer: [IDimensionViewer](../../Interface/IDimensionViewer/IDimensionViewer.htm);
 Node: Integer; Allow: Boolean);


## Параметры


Viewer. Компонент, для которого
 генерируется событие;


Node. Индекс элемента, для
 которого сгенерировано событие;


Allow. Признак изменения отображения
 дерева дочерних элементов вершины.


## Описание


Конструктор Create создает аргумент
 событий, возникающих перед изменением отображения дерева дочерних элементов
 какой-либо вершины дерева.
