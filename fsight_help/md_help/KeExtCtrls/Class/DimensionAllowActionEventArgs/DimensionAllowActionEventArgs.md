# DimensionAllowActionEventArgs


Сборка: ExtCtrls;


## Описание


Класс DimensionAllowActionEventArgs
 реализует аргумент событий, возникающих перед изменением отображения дерева
 дочерних элементов какой-либо вершины дерева.
