# DataGridCellBeforeEditEventArgs.Create


## Синтаксис


Create(Grid: [IDataGrid](../../Interface/IDataGrid/IDataGrid.htm);
 Row: Integer; Column: Integer; Cancel: Boolean);


## Параметры


Grid. Таблица, для которой
 сгенерировано событие;


Row. Номер строки ячейки, для
 которой сгенерировано событие;


Column. Сквозной номер столбца
 ячейки, для которой сгенерировано событие;


Cancel. Признак редактирования
 ячейки.


## Описание


Конструктор Create создает аргумент
 события, наступающего перед редактированием значения ячейки.
