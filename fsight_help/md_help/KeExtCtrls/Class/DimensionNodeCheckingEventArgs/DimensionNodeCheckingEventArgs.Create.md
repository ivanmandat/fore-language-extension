# DimensionNodeCheckingEventArgs.Create


## Синтаксис


Create(Viewer: [IDimensionViewer](../../Interface/IDimensionViewer/IDimensionViewer.htm);
 Node: Integer; Value: Boolean; Allow: Boolean);


## Параметры


Viewer. Компонент, для которого
 сгенерировано событие;


Node. Элемент, для которого
 сгенерировано событие;


Value. Значение отметки флага,
 которое будет установлено;


Allow. Признак изменения отметки
 флага элемента.


## Описание


Конструктор Create создает аргумент
 события, возникающего перед изменением состояния отметки флага элемента
 компонента.
