# SearchDataSourceSelectionChangeEventArgs.Create


## Синтаксис


Create(Index: Integer; Cancel: Boolean);


## Параметры


Index. Индекс элемента с результатами
 поиска, для которого генерируется событие;


Cancel. Признак отмены перехода
 к элементу с результатами поиска.


## Описание


Конструктор Create создает аргумент
 события, возникающего при переходе к элементу с результатами поиска.
