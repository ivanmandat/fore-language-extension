# SearchDataSourceSelectionChangeEventArgs


Сборка: ExtCtrls;


## Описание


Класс SearchDataSourceSelectionChangeEventArgs
 реализует аргумент событий компонента [SearchDataSourcesBox](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/SearchDataSourcesBox.htm).


## Комментарии


Аргумент данного класса используется в событиях [SearchDataSourcesBox.OnAfterSelectionChange](../SearchDataSourcesBox/SearchDataSourcesBox.OnAfterSelectionChange.htm)
 и [SearchDataSourcesBox.OnBeforeSelectionChange](../SearchDataSourcesBox/SearchDataSourcesBox.OnBeforeSelectionChange.htm).
