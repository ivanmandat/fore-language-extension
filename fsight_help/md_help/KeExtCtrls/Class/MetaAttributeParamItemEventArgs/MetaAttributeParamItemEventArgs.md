# MetaAttributeParamItemEventArgs


Сборка: ExtCtrls;


## Описание


Класс MetaAttributeParamItemEventArgs
 реализует аргумент события, возникающего при присвоении атрибуту какого-либо
 параметра.


## Комментарии


При присвоении атрибуту какого-либо параметра для компонента [MetaAttributesBreadcrumb](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/MetaAttributesBreadcrumb.htm)
 будет сгенерировано событие [OnMetaAttributeParam](../MetaAttributesBreadcrumb/MetaAttributesBreadcrumb.OnMetaAttributeParam.htm).
