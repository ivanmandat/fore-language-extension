# MetaAttributeParamItemEventArgs.CreateArgs


## Синтаксис


CreateArgs(Attribute: String; Param: [IMetaAttributeParamItem](../../Interface/IMetaAttributeParamItem/IMetaAttributeParamItem.htm));


## Параметры


Attribute. Наименование атрибута;


Param. Параметр, который будет
 установлен в качестве значения атрибута.


## Описание


Конструктор CreateArgs создает
 экземпляр аргумента события, возникающего при присвоении атрибуту какого-либо
 параметра.
