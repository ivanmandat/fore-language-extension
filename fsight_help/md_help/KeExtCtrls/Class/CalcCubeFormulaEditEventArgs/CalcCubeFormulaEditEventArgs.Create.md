# CalcCubeFormulaEditEventArgs.Create


## Синтаксис


Create(Coord: [ICalculatedCubeInstanceCoord](KeCubes.chm::/Interface/ICalculatedCubeInstanceCoord/ICalculatedCubeInstanceCoord.htm); Result: String);


## Параметры


Coord - координата в вычисляемом кубе.


Result - текста терма.


## Описание


Конструктор Create создает аргумент события, возникающего при создании терма, соответствующего координате куба.
