# MetabaseListViewItemPopupEventArgs.Create


## Синтаксис


Create(MetabaseListView: [IMetabaseListView](../../Interface/IMetabaseListView/IMetabaseListView.htm),
 Item: [IMetabaseListViewItem](../../Interface/IMetabaseListViewItem/IMetabaseListViewItem.htm),
 Menu: [IPopupMenu](ModForms.chm::/Interface/IPopupMenu/IPopupMenu.htm));


## Параметры


MetabaseListView. Компонент,
 для которого генерируется событие;


Item. Элемент компонента, для
 которого генерируется событие;


Menu. Контекстное меню, которое
 будет соответствовать элементу.


## Описание


Конструктор Create создает аргумент
 события с указанными параметрами.
