# DataGridCompareEventArgs


Сборка: ExtCtrls;


## Описание


Класс DataGridCompareEventArgs
 реализует аргумент события, возникающего при сравнении значений колонки
 во время пользовательской сортировки компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).
