# DimensionComboRollUpEventArgs.Create


## Синтаксис


Create(Combo: [IDimensionCombo](../../Interface/IDimensionCombo/IDimensionCombo.htm);
 Cancel: Boolean);


## Параметры


Combo. Компонент, для которого
 сгенерировано событие;


Cancel. Признак закрытия раскрывающегося
 списка компонента.


## Описание


Конструктор Create создает аргумент
 события, возникающего при сворачивании раскрывающегося списка компонента
 [DimensionCombo](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DimensionCombo.htm).
