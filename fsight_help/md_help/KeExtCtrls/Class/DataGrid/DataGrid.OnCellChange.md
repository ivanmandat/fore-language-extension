# DataGrid.OnCellChange


## Синтаксис


OnCellChange(Sender: Object; Args: [IDataGridCellChangeEventArgs](../../Interface/IDataGridCellChangeEventArgs/IDataGridCellChangeEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnCellChange наступает
 после изменения значения ячейки.
