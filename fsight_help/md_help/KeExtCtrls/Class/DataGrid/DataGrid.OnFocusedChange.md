# DataGrid.OnFocusedChange


## Синтаксис


OnFocusedChange(Sender: Object; Args: [IDataGridCellEventArgs](../../Interface/IDataGridCellEventArgs/IDataGridCellEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnFocusedChange наступает
 при передаче фокуса другой ячейке компонента.
