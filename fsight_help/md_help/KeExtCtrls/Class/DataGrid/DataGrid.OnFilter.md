# DataGrid.OnFilter


## Синтаксис


OnFilter(Sender: Object; Args: [IDataGridColumnEventArgs](../../Interface/IDataGridColumnEventArgs/IDataGridColumnEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnFilter наступает после
 установки фильтра в каком-либо столбце таблицы.
