# DataGrid.OnBeforeCellChange


## Синтаксис

## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnBeforeCellChange наступает
 перед присвоением ячейке нового значения.
