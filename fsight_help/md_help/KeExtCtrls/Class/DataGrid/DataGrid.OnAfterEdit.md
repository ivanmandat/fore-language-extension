# DataGrid.OnAfterEdit


## Синтаксис


OnAfterEdit(Sender: Object; Args: [IDataGridCellAfterEditEventArgs](../../Interface/IDataGridCellAfterEditEventArgs/IDataGridCellAfterEditEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnAfterEdit происходит
 после попытки изменить значение ячейки.
