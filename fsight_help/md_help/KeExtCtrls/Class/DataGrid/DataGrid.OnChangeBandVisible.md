# DataGrid.OnChangeBandVisible


## Синтаксис


OnChangeBandVisible(Sender: Object; Args: [IDataGridBandEventArgs](../../Interface/IDataGridBandEventArgs/IDataGridBandEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnChangeBandVisible
 наступает при изменении видимости у какого-либо контейнера/колонки компонента.
