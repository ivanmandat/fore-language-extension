# DataGrid


Сборка: ExtCtrls;


## Описание


Класс DataGrid реализует компонент
 среды разработки [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm),
 позволяющий отображать и редактировать данные в виде таблицы.
