# DataGridBandEventArgs.CreateArgs


## Синтаксис


CreateArgs(Grid: [IDataGrid](../../Interface/IDataGrid/IDataGrid.htm);
 BandBase: [IDataGridBandBase](../../Interface/IDataGridBandBase/IDataGridBandBase.htm));


## Параметры


Grid. Таблица, для которой
 сгенерировано событие;


BandBase. Контейнер таблицы,
 для которого сгенерировано событие.


## Описание


Конструктор CreateArgs создает
 аргумент событий, связанных с каким-либо контейнером компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).
