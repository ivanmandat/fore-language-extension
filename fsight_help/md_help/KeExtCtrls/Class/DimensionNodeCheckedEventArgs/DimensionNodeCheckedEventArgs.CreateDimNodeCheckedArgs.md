# DimensionNodeCheckedEventArgs.CreateDimNodeCheckedArgs


## Синтаксис


CreateDimNodeCheckedArgs(Viewer: [IDimensionViewer](../../Interface/IDimensionViewer/IDimensionViewer.htm);
 Node: Integer; Value: Boolean);


## Параметры


Viewer. Компонент, для которого
 сгенерировано событие;


Node. Индекс элемента, для
 которого сгенерировано событие;


Value. Признак отметки флага.


## Описание


Конструктор CreateDimNodeCheckedArgs
 создает аргумент события, возникающего после изменения состояния отметки
 флага элемента компонента.
