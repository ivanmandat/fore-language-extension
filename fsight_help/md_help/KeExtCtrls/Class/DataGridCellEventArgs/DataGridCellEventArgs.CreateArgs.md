# DataGridCellEventArgs.CreateArgs


## Синтаксис


CreateArgs(Grid: [IDataGrid](../../Interface/IDataGrid/IDataGrid.htm);
 Row: Integer; Column: Integer);


## Параметры


Grid. Таблица, для которой
 сгенерировано событие;


Row. Номер строки ячейки, для
 которой сгенерировано событие;


Column. Сквозной номер столбца
 ячейки, для которой сгенерировано событие.


## Описание


Конструктор CreateArgs создает
 аргумент событий, связанных с ячейками таблицы.
