# DimensionTree


Сборка: ExtCtrls;


## Описание


Класс DimensionTree реализует
 компонент среды разработки [DimensionTree](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DimensionTree.htm),
 используемый для отображения данных справочника в виде дерева.


## Комментарии


Для работы с компонентом [DimensionCombo](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DimensionCombo.htm)
 используйте класс [DimensionCombo](../DimensionCombo/DimensionCombo.htm).
