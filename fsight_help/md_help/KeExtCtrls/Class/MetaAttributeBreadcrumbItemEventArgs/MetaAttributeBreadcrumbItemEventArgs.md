## Описание


Класс MetaAttributeBreadcrumbItemEventArgs
 реализует аргумент событий, связанных с атрибутами базы данных временных
 рядов, работа с которой производится в компоненте [MetaAttributeBreadcrumb](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/MetaAttributesBreadcrumb.htm).
