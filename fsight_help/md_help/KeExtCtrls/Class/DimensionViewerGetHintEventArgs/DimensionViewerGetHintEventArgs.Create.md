# DimensionViewerGetHintEventArgs.Create


## Синтаксис


Create(Viewer: [IDimensionViewer](../../Interface/IDimensionViewer/IDimensionViewer.htm);
 Node: Integer; Column: Integer; Text: String);


## Параметры


Viewer. Компонент, для которого
 сгенерировано событие;


Node. Индекс элемента, для
 которого сгенерировано событие;


Column. Индекс колонки, в области
 которой для элемента было сгенерировано событие;


Text. Текст всплывающей подсказки
 для элемента, над которым остановился курсор мыши.


## Описание


Конструктор Create создает аргумент
 события, возникающего при появлении всплывающей подсказки для элемента,
 над которым остановился курсор мыши.
