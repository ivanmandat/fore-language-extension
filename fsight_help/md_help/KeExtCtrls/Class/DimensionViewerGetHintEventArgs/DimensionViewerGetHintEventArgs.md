# DimensionViewerGetHintEventArgs


Сборка: ExtCtrls;


## Описание


Класс DimensionViewerGetHintEventArgs
 реализует аргумент события, возникающего при появлении всплывающей подсказки
 для элемента, над которым остановился курсор мыши.
