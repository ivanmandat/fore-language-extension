# SearchDataSourcesBox.OnCancelSearch


## Синтаксис


OnCancelSearch(Sender: Object; Args: [IEventArgs](ForeSys.chm::/Interface/IEventArgs/IEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnCancelSearch наступает
 в момент принудительной остановки процесса BI-поиска.


## Комментарии


Для принудительной остановки процесса BI-поиска используйте метод [ISearchDataSourcesBox.CancelSearch](../../Interface/ISearchDataSourcesBox/ISearchDataSourcesBox.CancelSearch.htm).
