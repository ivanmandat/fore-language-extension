# DimensionColumnEventArgs.Create


## Синтаксис


Create(Viewer: [IDimensionViewer](../../Interface/IDimensionViewer/IDimensionViewer.htm);
 Column: [IDimensionViewerColumn](../../Interface/IDimensionViewerColumn/IDimensionViewerColumn.htm));


## Параметры


Viewer. Компонент, для которого
 сгенерировано событие;


Column. Колонка, для которой
 сгенерировано событие.


## Описание


Конструктор Create создает аргумент
 события, возникающего при щелчке колонки компонента.
