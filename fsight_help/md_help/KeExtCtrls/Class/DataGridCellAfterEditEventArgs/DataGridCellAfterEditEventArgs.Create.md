# DataGridCellAfterEditEventArgs.Create


## Синтаксис


Create(Grid: [IDataGrid](../../Interface/IDataGrid/IDataGrid.htm);
 Row: Integer; Column: Integer; Changed: Boolean);


## Параметры


Grid. Таблица, для которой
 сгенерировано событие;


Row. Номер строки ячейки, для
 которой сгенерировано событие;


Column. Сквозной номер столбца
 ячейки, для которой сгенерировано событие;


Changed. Признак изменения
 значения ячейки.


## Описание


Конструктор Create создает аргумента
 события, наступающего после завершения редактирования значения ячейки.
