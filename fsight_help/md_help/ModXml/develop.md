## С чего начать


		 Сборка XML
		 предназначена для работы с XML-документами и данными посредством
		 набора служб MSXML (Microsoft XML Core Services).


		При использовании сборки XML
		 в среде разработки продукта «Форсайт. Аналитическая платформа»
		 можно руководствоваться описанием MSXML, доступным в Microsoft
		 Learn:


			- Общее описание служб
			 MSXML. [https://learn.microsoft.com/en-us/previous-versions/windows/desktop/ms763742(v=vs.85)](https://learn.microsoft.com/en-us/previous-versions/windows/desktop/ms763742(v%3dvs.85));


			- Интерфейсы для работы
			 XML DOM. [https://learn.microsoft.com/en-us/previous-versions/windows/desktop/ms760218(v=vs.85)](https://learn.microsoft.com/en-us/previous-versions/windows/desktop/ms760218(v%3dvs.85)).


		Описанные там типы и типы сборки XML идентичны по назначению
		 и функциональным возможностям.


		В статье «[Примеры работы
		 с XML-файлами](Samples/ModXml_Sample1.htm)» представлены несколько простых примеров по
		 работе со сборкой.


		Важно.
		 В «Форсайт. Аналитическая платформа»
		 используется библиотека для работы со службами MSXML версии 6.0.
		 Использование MSXML 6.0 накладывает определенные требования к
		 структуре XML-файлов в плане использования пространств имён. Если
		 в структуре имеется пространство по умолчанию XMLNS, то для работы
		 с элементами может потребоваться [изменение
		 кода приложения](Samples/work_using_msxml6.htm) или [структуры](Samples/work_using_msxml6.htm)
		 XML-файла.


		 Перед использованием сборки
		 XML рекомендуется изучить
		 расширяемый язык разметки XML, его области применения, лексику
		 и терминологию, а также объектные модели документов DOM различных
		 уровней.


		Спецификации по языку XML и объектным моделям документов DOM
		 доступны на сайте консорциума W3C (World Wide Web Consortium)
		 - [https://www.w3.org/](https://www.w3.org/).
