# Настройка ролевой модели


Ролевая модель задаёт права пользователей на доступ к данным, используемым
 в построении процесса. Права доступа определяются с помощью объектов полномочий,
 которые задают связь субъектов безопасности (пользователей/групп пользователей)
 со срезами данных куба.


При выполнении настройки ролевой модели задаются статические и динамические
 права доступа к информационным объектам:


	- статические права доступа к данным действуют постоянно на протяжении
	 всего времени работы пользователя с источником данных;


	- динамические права доступа к данным раздаются только во время
	 выполнения отдельных шагов, к которым привязаны объекты полномочий.


Для настройки прав доступа выполните следующие шаги:


	- Создайте
	 группы пользователей.


	- Включите
	 пользователей в группы.


	- [Создайте](Create_Authorization_Object.htm) и [настройте](Configuring_the_Authorization_Object.htm) объекты
	 полномочий.
