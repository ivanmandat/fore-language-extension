# Построение процесса


Для построения бизнес-процесса выполните шаги:


	- Создайте бизнес-процесс.


	- Определите основные [настройки
	 и параметры](Setting_Process_Parameters.htm) процесса.


	- [Настройте
	 права пользователей](../Facility_Agreement/Role_model.htm), действующие в рамках выполнения
	 шагов процесса, с помощью [объектов
	 полномочий](../Facility_Agreement/Create_Authorization_Object.htm).


	- Настройте для [объектов
	 полномочий](../Facility_Agreement/Configuring_the_Authorization_Object.htm) связи с параметрами процесса.


	- Создайте [этапы процесса](Create_Stage.htm).


	- Создайте [шаги
	 этапа](StepsProcess/CreateStepsProcess.htm).


	- Настройте [стартовые](Setting_up_Events.htm) и [промежуточные события
	 по времени](Setting_up_Timer_Intermediate_Events.htm).


	- Настройте [условия
	 выполнения шагов этапа](Setting_up_steps_conditions.htm).


После выполненных действий сохраните созданный процесс или внесенные
 изменения в процесс, выполнив одно из действий:


	- нажмите кнопку ![](save16.png) «Сохранить»
	 на [панели
	 быстрого доступа](../Starting/Starting.htm#setting_business_process);


	- выполните команду «Бизнес-процесс
	 > Сохранить» главного меню.


После завершения построения процесса:


	- Запустите [симуляцию](BPM_Simulation_Process.htm)
	 выполнения отдельных шагов процесса, если требуется.


	- Запустите [процесс](Start_Process.htm), задав необходимые
	 параметры запуска.


Для дальнейшей работы по выполнению и мониторингу шагов запущенного
 процесса, откройте окно «[Мониторинг
 процессов](../Starting/Monitoring_process_execution.htm)».
