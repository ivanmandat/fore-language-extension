# Создание этапов процесса


Процесс может состоять из неограниченного количества последовательных
 этапов, которые состоят в свою очередь из шагов, выполняемых последовательно
 или параллельно.


Для создания этапа процесса в окне [настройки
 бизнес-процесса](../Starting/Starting.htm#setting_business_process):


	- нажмите кнопку ![](button_create_step0.png) «Добавить этап» на вкладке «Процесс» ленты инструментов;


	- выполните команду ![](button_create_step0.png) «Добавить
	 этап» контекстного меню в рабочей области или дереве процессов.


Будет открыт мастер «Создание этапа»:


![](Create_Stage.png)


В окне мастера:


	- В поле «Наименование»
	 введите наименование этапа процесса.


	- В списке «Предшественник»
	 выберите наименование предшествующего этапа или выберите элемент «- В начало -». При выбранном элементе
	 «- В начало -» новый этап
	 добавляется в начало процесса.


	- Нажмите кнопку «Готово».
