# Настройка параметров процесса


Параметры процесса предназначены для запуска процесса с заданными значениями
 параметров, например: единицы организационной структуры, версии планирования,
 валюты. Настройка параметров процесса осуществляется в окне «Настройки
 и параметры процесса».


[![](../../opened.gif)![](../../closed.gif)Для открытия
 окна](javascript:TextPopup(this))


	Выполните одно из действий:


		- нажмите кнопку ![](settings_button_and_options.png) «Настройки и параметры» в группе
		 «Настройка» на вкладке
		 «Процесс» ленты инструментов
		 окна «[Настройка
		 бизнес-процесса](../Starting/Starting.htm#setting_business_process)»;


		- выполните команду ![](settings_button_and_options.png) «Настройки и параметры» контекстного
		 меню:


			- рабочей области окна настройки бизнес-процесса;


			- корневого элемента с названием процесса на боковой панели
			 окна.


![](Setting_Process_Parameters.png)


Задайте настройки на вкладках/страницах:


	- [Базовые
	 свойства](Basic_properties.htm). Задаются базовые свойства процесса;


	- [Управление
	 параметрами](Params_control.htm). Формируется список параметров процесса из
	 справочников, которые будут использоваться в качестве параметров;


	- [События](Events.htm).
	 Формируется список выполняемых действий для событий процесса для автоматического
	 выполнения Fore-методов или отправки E-mail при необходимости.


После настройки параметров процесса перейдите к [настройке
 параметров и администрирования бизнес-процессов](Setting_and_administrating_process_parameters.htm).
