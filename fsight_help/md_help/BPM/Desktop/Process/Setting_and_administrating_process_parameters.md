# Настройка параметров и администрирование
 бизнес-процессов


Для настройки параметров и администрирования бизнес-процессов откройте
 на просмотр утилиту «Настройка параметров
 и администрирование бизнес-процессов» с идентификатором BP_UTIL,
 которая входит в состав компонента «Бизнес-процессы».


После выполнения действия будет открыто окно «Настройка
 параметров и администрирование бизнес-процессов»:


![](Setting_and_administrating_parameters.png)


Задайте настройки на страницах:


	- [Планировщик](Setting_planner.htm).
	 Определяется тип исполнения бизнес-процессов, а также задаётся адрес
	 BI-сервера;


	- [Параметры](Parameters_setting.htm).
	 Задаются параметры отправки e-mail оповещений, параметры [мониторинга](../Starting/Monitoring_process_execution.htm)
	 и дополнительные параметры;


	- [Отладка](Setting_debug.htm).
	 Устанавливается функциональность очистки мониторинга, перегенерации
	 процессов и инициализации контейнера задач;


	- [Служебные
	 задачи](Creating_service_tasks.htm). Создаются служебные задачи для обновления метаданных процессов
	 и пересохранения [объектов
	 полномочий](../Facility_Agreement/Create_Authorization_Object.htm);


	- [Администрирование](Administrating.htm).
	 Выполняются операции добавления прав доступа к бизнес-процессам, повторного
	 создания группы пользователей, удаления всей истории процессов.


Примечание.
 Настройка параметров на страницах «Планировщик»
 и «Параметры отправки e-mail/Параметры»
 доступна только пользователям из [группы](Admin.chm::/03_Admin/Groups/Admin_Groups.htm)
 «Администраторы». Для работы со
 всеми разделами, включая разделы «Отладка»,
 «Служебные задачи» и «Администрирование»,
 необходимо обладать правами администратора ADMIN, который является владельцем
 схемы.


После настройки параметров и администрирования бизнес-процессов перейдите
 к [настройке ролевой
 модели](../Facility_Agreement/Role_model.htm).
