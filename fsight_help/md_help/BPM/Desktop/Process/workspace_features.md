# Настройки рабочей области


При редактировании настроек рабочей области доступны следующие действия:


	- [управление расположением элементов](#location);


	- [масштабирование рабочей области](#scale).


## Управление расположением элементов


Для визуальной настройки рабочей области доступны следующие режимы:


	- Указатель.
	 Режим указателя предназначен
	 для выделения элементов рабочей области. Используется по умолчанию;


	- Рука.
	 Режим руки предназначен для перемещения по рабочей области.


Для задания режима выберите требуемый режим с помощью кнопки ![](Cursor.png) «Указатель»/
 ![](Hand.png) «Рука» группы кнопок «Инструменты»
 на вкладке «Главная».


## Масштабирование рабочей области


Для изменения масштаба рабочей области используйте группу кнопок «Масштаб» на вкладке «Главная»:


	- ![](Scale.png) «Выбрать
	 масштаб». Выбирается процент
	 масштабирования из списка или задается вручную;


	- ![](Full-size.png) «100
	 %». Возвращает масштабирование рабочей области к 100%;


	- ![](Adapt.png)
	 «Подогнать». Подгоняет размер
	 рабочей области под размер открытого окна.
