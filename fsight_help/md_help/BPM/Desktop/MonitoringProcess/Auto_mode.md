# Выполнение шагов процесса в автоматическом режиме


Для выполнения шагов процесса в автоматическом режиме откройте [мониторинг
 процессов](../Starting/Monitoring_process_execution.htm) после [запуска процесса](../Process/Start_Process.htm).
 Запуск шага процесса с автоматическим способом исполнения выполняется
 автоматически после выполнения предыдущего шага.


Автоматический способ исполнения доступен для шагов:


	- [Расчёт](../Process/StepsProcess/Calculation.htm);


	- [Вызов
	 подпроцесса](../Process/StepsProcess/Subprocess.htm);


	- [Вызов Fore-метода](../Process/StepsProcess/Fore.htm);


	- [Задача ETL](../Process/StepsProcess/ETL.htm);


	- [Отправка e-mail
	 оповещения](../Process/StepsProcess/E-mail.htm).


Для задания автоматического выполнения шагов выполните одно из действий:


	- укажите способ исполнения «Автоматический»
	 при [создании](../Process/StepsProcess/CreateStepsProcess.htm)
	 или [редактировании](../Process/BPM_Edit_Process.htm#editing_step)
	 шага на странице мастера «Управление
	 параметрами»;


	- выполните команду «Автоматическое
	 выполнение шага» контекстного меню шага. Изменить способ исполнения
	 шага можно только при редактировании процесса.


Примечание.
 В рабочей области шаги процесса с автоматическим способом исполнения
 обозначаются значком ![](../Process/Auto16.png).


Особенности выполнения шагов в автоматическом режиме:


	- если после [завершения шага](Manual_mode.htm) в ручном
	 режиме следующий шаг запускается в автоматическом режиме, то выполнение
	 процесса произойдет до шага, который запускается в ручном режиме;


	- если при [отклонении шага](Reject_step.htm) в ручном
	 режиме предыдущий шаг выполняется в автоматическом режиме, то отклонение
	 произойдет до шага, который запускается в ручном режиме.
