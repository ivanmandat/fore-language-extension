# Обновление мониторинга процесса


Обновление мониторинга процесса доступно при отображении в рабочей области
 с[писка запущенных
 экземпляров процесса/назначенных шагов процесса](../Starting/Starting.htm#legend_monitoring_web).


Для мониторинга процессов доступно:


	- разовое обновление списка запущенных процессов/подпроцессов
	 на боковой панели и списка экземпляров выбранного процесса/назначенных
	 шагов процесса в рабочей области. Для этого нажмите кнопку ![](b_refresh.png) «Обновить»
	 на панели инструментов;


	- обновление списка запущенных процессов/подпроцесса на боковой
	 панели и списка экземпляров выбранного процесса/назначенных шагов
	 процесса в рабочей области с заданной периодичностью. Для этого нажмите
	 кнопку ![](b_refresh_auto.png) «Автообновление».
	 Будет включено автоматическое обновление каждую 1 минуту.


Для изменения периода автоматического обновления
 выполните команду «Задать период»
 в раскрывающемся меню кнопку ![](b_refresh_auto.png) «Автообновление».
 Будет открыт диалог «Период автообновления»:


![](update_period.png)


В диалоге задайте период обновления и нажмите
 кнопку «Применить». Обновление
 может выполняться каждые N секунд,
 минут или часов.
