# Настройка рабочей области


Для настройки рабочей области доступны следующие операции:


[![](../../opened.gif)![](../../closed.gif)Перемещение
 по рабочей области](javascript:TextPopup(this))


	Для перемещения по рабочей области выполните одно из действий:


		- выберите режим ![](Hand_web.png) «Рука»
		 на [панели инструментов](../Starting/Starting.htm#legend_web)
		 и зажмите кнопку мыши. Если выбран режим ![](Cursor_web.png)
		 «Курсор» для [выделения
		 объектов](BPM_Edit_Process.htm#selection), то используйте раскрывающийся список для смены режима;


		- зажмите клавишу ПРОБЕЛ и кнопку мыши;


		- используйте стрелки на клавиатуре;


		- прокрутите колёсико мыши для перемещения рабочей области
		 по вертикали. Для перемещения рабочей области по горизонтали дополнительно
		 зажмите клавишу SHIFT.


[![](../../opened.gif)![](../../closed.gif)Масштабирование
 рабочей области](javascript:TextPopup(this))


	Для изменения масштаба рабочей области прокрутите колесо мыши при
	 зажатой клавише CTRL.


[![](../../opened.gif)![](../../closed.gif)Изменение
 размеров панели параметров/боковой панели](javascript:TextPopup(this))


	Для изменения размеров [панели
	 параметров](../Starting/Starting.htm#legend_web)/[боковой
	 панели](../Starting/Starting.htm#legend_monitoring_web):


		- Наведите курсор мыши на границу между рабочей областью и
		 панелью параметров/боковой панелью. Указатель мыши примет вид
		 двунаправленной стрелки.


		- Зажмите кнопку мыши и переместите границу в нужное положение.
