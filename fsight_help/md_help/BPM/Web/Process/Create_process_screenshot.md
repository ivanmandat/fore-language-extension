# Построение процесса


Для построения бизнес-процесса выполните шаги:


	- Настройте [параметры
	 администрирования процессов](Setting_and_administrating_process_parameters.htm) при необходимости.


	- [Создайте
	 бизнес-процесс](../Starting/Starting.htm#creating_a_new_process).


	- Настройте [ролевую
	 модель процесса](../Facility_Agreement/Role_model.htm).


	- Определите основные [настройки
	 и параметры](Setting_Process_Parameters.htm) процесса.


	- Создайте [этапы процесса](Create_Stage.htm).


	- Создайте [шаги
	 этапа](StepsProcess/CreateStepsProcess.htm).


	- Настройте [стартовые](Setting_up_Events.htm) и [промежуточные события
	 по времени](Setting_up_Timer_Intermediate_Events.htm).


	- Настройте [условия
	 выполнения шагов этапа](Setting_up_steps_conditions.htm).


После выполненных действий сохраните созданный процесс или внесенные
 изменения в процесс с помощью кнопки «Сохранить»
 главного меню.


Примечание.
 Для сохранения промежуточного результата доступно сохранение процесса
 с шагами ошибочного и/или неполного заполнения настроек.


После завершения построения процесса [запустите
 процесс](Start_Process.htm), задав необходимые параметры запуска. Для дальнейшей работы
 по выполнению и мониторингу шагов запущенного процесса, откройте окно
 «[Мониторинг
 процессов](../Starting/Monitoring_process_execution.htm)».
