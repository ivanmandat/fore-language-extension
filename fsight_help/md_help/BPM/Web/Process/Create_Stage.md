# Создание этапов процесса


Процесс может состоять из неограниченного количества последовательных
 этапов, которые состоят в свою очередь из шагов, выполняемых последовательно
 или параллельно.


Для создания этапа процесса в окне [настройки
 бизнес-процесса](../Starting/Starting.htm#setting_business_process):


	- в [рабочей области](../Starting/Starting.htm#legend_web):


		- нажмите кнопку ![](b_create_step.png) «Добавить
		 этап», если в процессе не создан ни один этап;


		- нажмите кнопку ![](b_add.png) «Добавить
		 этап» для добавления этапа в конец последовательности этапов:


![](StepsProcess/stage.png)


	- на [панели инструментов](../Starting/Starting.htm#legend_web)
	 нажмите кнопку ![](b_create_step.png) «Добавить
	 этап».


Особенности добавления этапов с помощью панели инструментов:


	- если в процессе уже содержатся этапы и ни один из этапов не
	 выделен, то новый этап будет добавлен в конец последовательности этапов;


	- если в процессе уже содержатся этапы и один из них выделен,
	 то новый этап будет добавлен после выделенного этапа.


Для переименования этапа дважды щёлкните по его наименованию в рабочей
 области и задайте новое наименование.
