# Настройка параметров процесса


Параметры процесса предназначены для запуска процесса с заданными значениями
 параметров, например: единицы организационной структуры, версии планирования,
 валюты. Настройка параметров процесса осуществляется в окне «Настройки».


[![](../../opened.gif)![](../../closed.gif)Для открытия
 окна](javascript:TextPopup(this))


	Нажмите кнопку ![](b_param_settings.png) «Настройки»
	 в главном меню [окна
	 настройки бизнес-процесса](../Starting/Starting.htm#setting_business_process).


![](Params_control_web.png)


Задайте настройки на вкладках:


	- [Базовые
	 свойства](Basic_properties.htm). Задаются базовые свойства процесса;


	- [Параметры](Params_control.htm).
	 Формируется список параметров процесса из справочников, которые будут
	 использоваться в качестве параметров;


	- [События](Events.htm).
	 Формируется список выполняемых действий для событий процесса для автоматического
	 выполнения Fore-методов или отправки E-mail при необходимости.


После настройки параметров процесса перейдите к [настройке
 параметров и администрирования бизнес-процессов](Setting_and_administrating_process_parameters.htm).
