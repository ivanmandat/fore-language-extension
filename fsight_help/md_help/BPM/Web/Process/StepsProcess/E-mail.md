# Создание шага «Отправка e-mail оповещения»


Примечание.
 Создание шага «Отправка e-mail оповещения»
 доступно только в настольном приложении. Для получения подробной информации
 о создании шага обратитесь к статье «[Создание
 шага «Отправка e-mail оповещения»](../../../Desktop/Process/StepsProcess/E-mail.htm)».


Для шага по формированию автоматических e-mail оповещений веб-приложении
 доступен просмотр настроек шага. Для этого в рабочей области выделите
 шаг ![](b_message.png)
 «Отправка e-mail оповещения» и
 нажмите кнопку ![](../b_settings.png) «Настройки»
 на панели инструментов. Будет выведено информационное окно о том, что
 настройки шага доступны только для просмотра, и открыта панель параметров
 «Отправка e-mail оповещения»;


Для отправки e-mail оповещений должны быть настроены параметры отправки
 на странице «[Параметры
 отправки e-mail/Параметры](../Parameters_setting.htm)» в [мастере
 администрирования бизнес-процессов](../Setting_and_administrating_process_parameters.htm) .


После создания шагов процесса перейдите к [настройке
 условий выполнения шагов](../Setting_up_steps_conditions.htm).
