# Масштабирование рабочей области


Текущий масштаб рабочей области отображается в правом нижнем углу окна
 алгоритма расчёта. Для изменения масштаба рабочей области используйте
 группу «Масштаб» на вкладке «Вид» ленты инструментов. Доступные
 операции:


	- установка произвольного масштаба:


		- нажмите кнопку ![](zoom.png) «Масштаб»
		 и в раскрывающемся меню выберите масштаб. Если ни один из масштабов
		 не подходит, то установите свой собственный масштаб. Для этого
		 выберите пункт «Произвольный»
		 и в отобразившемся окне введите требуемое вам значение масштаба;


		- зажмите клавишу CTRL, затем наведите указатель мыши на место,
		 относительно которого будет изменяться масштаб, и прокрутите колёсико
		 мыши от себя для увеличения масштаба, на себя - для уменьшения;


		- используйте ползунок и кнопки «+»
		 и «-» в правом нижнем
		 углу окна алгоритма расчёта;


	- установка стопроцентного масштаба.
	 Нажмите кнопку ![](100.png)
	 «100%»;


	- установка масштаба по размеру
	 рабочей области. Нажмите кнопку ![](FitScale.png) «Подогнать».
	 Масштаб рабочей области будет изменён таким образом, чтобы все объекты
	 были видимы.
