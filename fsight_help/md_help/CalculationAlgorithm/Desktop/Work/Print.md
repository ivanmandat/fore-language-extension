# Печать алгоритма расчёта


Для печати алгоритма расчёта используйте команды в главном меню «Алгоритм расчёта»:


	- Печать. Открывает диалог
	 «[Печать](uinav.chm::/GUI/Print_setup.htm)»
	 для настройки печати и выбора используемого принтера;


	- Печать > Предварительный
	 просмотр. Отображает алгоритм расчёта в режиме предварительного
	 просмотра перед выводом на печать. Принципы работы в режиме предварительного
	 просмотра приведены в разделе «[Предварительный просмотр
	 и печать объекта](uinav.chm::/GUI/Object_preview.htm)»;


	- Печать > Предварительный
	 просмотр. Отображает диалог «Параметры
	 страницы» для настройки параметров печати алгоритма расчёта.
	 Работа с данным диалогом подобна работе с диалогом «[Параметры страницы](uireport.chm::/desktop/tuning/parampage/uireport_tuning_parampage.htm)»
	 для табличного листа регламентного отчета.
