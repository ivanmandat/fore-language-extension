# Оформление алгоритма расчёта


Для визуального структурирования элементов алгоритма расчёта используйте
 следующие объекты:


	- Текст. Предназначен
	 для создания надписей, комментариев, подписей и тому подобных элементов,
	 разъясняющих назначение блоков и объектов;


	- Рисунок. Предназначен
	 для создания визуальных элементов;


	- Фигуры. Предназначены
	 для создания визуальных блоков и стрелок между элементами.


Для вставки объектов используйте кнопки, расположенные в группе «Фигуры» на вкладке «Главная»
 ленты инструментов. Работа с объектами аналогична работе одноимёнными
 объектами [рабочего
 пространства](UiWsp.chm::/1_intro/UiWsp_Intro.htm).


Для упрощения оформления в рабочей области можно отобразить дополнительные
 элементы:


	- сетку. Вертикальные
	 и горизонтальные линии, позволяющие более ровно выстраивать объекты
	 по горизонтали и вертикали;


	- точки соединения фигур и объектов.
	 Точки, к которым можно присоединять линии, соединяющие объекты;


	- пересечения линий. Места
	 пересечения линий, соединяющих объекты.


Для отображения дополнительных элементов используйте соответствующие
 флажки в группе «Показать» на
 вкладке «Вид» ленты инструментов.
