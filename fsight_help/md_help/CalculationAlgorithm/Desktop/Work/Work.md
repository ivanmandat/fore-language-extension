# Работа с готовым алгоритмом расчёта


Для работы с готовым алгоритмом расчёта в настольном приложении выполните
 шаги:


	- [Рассчитайте алгоритм](Perform_calculations.htm).


	- [Просмотрите рассчитанные данные](DataView.htm).


	- [Измените масштаб рабочей области](Scale.htm).


	- [Напечатайте алгоритм расчёта](Print.htm).


После работы с готовым алгоритмом расчёта выполните команду «Алгоритм
 расчёта > Выход» в [главном меню](Beginning_of_work.htm).
 Если имеются несохранённые изменения, то будет предложено их сохранить.
