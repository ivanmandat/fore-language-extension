# Вставка объектов репозитория


Вставка объектов репозитория позволяет добавить источники и приёмники
 данных в рабочую область. За счёт этого обеспечивается наглядность алгоритма
 расчёта.


[![](../../Opened.gif)![](../../Closed.gif)Вставка объекта
 репозитория](javascript:TextPopup(this))


	Для вставки объекта репозитория:


		- нажмите кнопку ![](b_addObject.png)
		 «Объект репозитория» на
		 вкладке «Главная» ленты
		 инструментов;


		- выполните команду «Добавить
		 > Объект репозитория» в контекстном меню рабочей области.


	В отобразившемся диалоге выберите вставляемый объект. Выбранный
	 объект будет вставлен в рабочую область.


[![](../../Opened.gif)![](../../Closed.gif)Редактирование
 объекта репозитория](javascript:TextPopup(this))


	Для редактирования выбранного объекта репозитория:


		- нажмите кнопку ![](../Calculation_block/b_edit.png) «Редактировать» на вкладке «Главная» ленты инструментов;


		- выполните команду «Редактировать»
		 в контекстном меню объекта;


		- нажмите клавишу F4;


		- дважды щёлкните по выбранному объекту.


	Выбранный объект будет открыт на редактирование.


[![](../../Opened.gif)![](../../Closed.gif)Просмотр свойств
 объекта репозитория](javascript:TextPopup(this))


	Для просмотра свойств объекта выполните команду «Свойства»
	 в контекстном меню объекта.


	Будет открыт диалог «[Свойства
	 объекта](UiNav.chm::/03_objects/uinav_obj_basicprop.htm)».
