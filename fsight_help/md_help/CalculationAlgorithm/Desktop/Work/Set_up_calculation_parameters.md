# Добавление и настройка параметров


Параметры позволяют динамически управлять расчётом алгоритма.


Для добавления и настройки параметров алгоритма расчёта нажмите кнопку
 ![](b_settings.png) «Настройка»
 в группе «Параметры расчёта» на
 вкладке «Расчёт» ленты инструментов.
 Будет открыт диалог «Параметры алгоритма
 расчёта»:


![](AlgorithmParams.png)


Для добавления и настройки параметров алгоритма расчёта:


	- [Добавьте
	 параметры алгоритма расчёта](Parameters_of_the_calculation_algorithm.htm).


	- [Задайте связи параметров алгоритма
	 расчёта с параметрами блоков](Manage_settings.htm).


	- [Назначьте параметры
	 алгоритма расчёта, управляющие периодом расчёта](Calculation_period_management.htm).


	- [Задайте связи параметров
	 алгоритма расчёта с параметрами объектов репозитория](Manage_objects_settings.htm).


	- [Задайте связи параметров
	 алгоритма расчёта с параметрами кубов](Manage_cubes_settings.htm).


	- Нажмите кнопку «Сохранить»
	 в диалоге «Параметры алгоритма расчёта».


В результате будут добавлены и настроены параметры алгоритма расчёта.
