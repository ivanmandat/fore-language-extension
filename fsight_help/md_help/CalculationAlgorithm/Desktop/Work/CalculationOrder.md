# Настройка порядка расчёта алгоритма


Блоки будут [рассчитаны](Perform_calculations.htm) в том
 порядке, в котором они следуют в дереве расчёта. Для изменения порядка
 следования блоков используйте кнопки ![](../Calculation_block/b_up.png) и ![](../Calculation_block/b_down.png) и аналогичные команды в
 контекстном меню блоков.


Важно. Учитывайте
 порядок [формул](../Calculation_block/Formula.htm), заданных
 для [блоков расчёта](../Calculation_block/Calculation_unit.htm)
 и [блоков контроля](../Control_Block/Control_Block.htm), в
 порядке расчёта алгоритма. Блок с формулами, результат которых используется
 в других формулах, должен стоять перед блоками, в которых формулы используют
 данный результат расчёта.


Для исключения отдельных блоков из расчёта выполните команду «Исключить из расчёта» в контекстном
 меню блока в рабочей области или в дереве расчёта. Исключённый блок меняет
 свой вид в рабочей области: он становится серым с зачёркнутой пиктограммой.
