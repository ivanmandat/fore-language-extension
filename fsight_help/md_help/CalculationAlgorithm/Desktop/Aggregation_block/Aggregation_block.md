# Вставка и настройка блоков агрегации


Блок агрегации предназначен для выполнения агрегации по заданным параметрам.


[![](../../Opened.gif)![](../../Closed.gif)Создание
 нового блока агрегации](javascript:TextPopup(this))


	Для создания нового блока агрегации:


		- в [навигаторе
		 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm):


			- нажмите кнопку «Новый
			 объект > Пользовательские объекты > Блок агрегации»
			 в группе «Создать»
			 на вкладке «Главная»
			 ленты инструментов;


			- выполните команду «Создать >
			 Пользовательские объекты > Блок агрегации»
			 в контекстном меню навигатора объектов.


	После выполнения одного из действий будет
	 открыто окно «Вставка объекта - параметры
	 - Блок агрегации». Введите наименование создаваемого блока
	 агрегации и измените автоматически сформированный идентификатор при
	 необходимости;


		- в [расширении «Алгоритмы расчёта»](../Work/Beginning_of_work.htm):


			- выполните команду «Блок
			 агрегации» в раскрывающемся меню кнопки ![](../Work/b_block.png)
			 «Блок расчёта» на
			 вкладке «Главная»
			 ленты инструментов;


			- выполните команду «Создать > Блок
			 агрегации» в контекстном меню рабочей области.


[![](../../Opened.gif)![](../../Closed.gif)Редактирование
 готового блока агрегации](javascript:TextPopup(this))


	Для редактирования готового блока агрегации:


		- в [навигаторе
		 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm) выделите блок агрегации и выполните
		 одно из действий:


			- нажмите кнопку ![](../Calculation_block/b_edit.png)
			 «Редактировать» в
			 группе «Открыть» на
			 вкладке «Главная»
			 ленты инструментов;


			- выполните команду «Редактировать»
			 в контекстном меню блока агрегации;


			- нажмите клавишу F4;


		- в [расширении «Алгоритмы расчёта»](../Work/Beginning_of_work.htm) выделите
		 блок агрегации в рабочей области:


			- нажмите кнопку ![](../Calculation_block/b_edit.png) «Редактировать» на вкладке
			 «Главная» ленты инструментов;


			- выполните команду «Редактировать»
			 в контекстном меню блока агрегации;


			- нажмите клавишу F4;


			- дважды щёлкните по блоку агрегации.


Создание и редактирование выполняется с помощью мастера редактирования
 блока агрегации, в котором настраиваются параметры блока.
