# Вставка и настройка блоков расчёта


Блок расчёта предназначен для выполнения расчётов по заданным параметрам.


[![](../../Opened.gif)![](../../Closed.gif)Создание
 нового блока расчёта](javascript:TextPopup(this))


	Для создания нового блока расчёта:


		- в [навигаторе
		 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm):


			- нажмите кнопку «Новый
			 объект > Пользовательские объекты > Блок расчёта»
			 в группе «Создать»
			 на вкладке «Главная»
			 ленты инструментов;


			- выполните команду «Создать >
			 Пользовательские объекты > Блок расчёта» в
			 контекстном меню навигатора объектов.


	После выполнения одного из действий будет
	 открыто окно «Вставка объекта - параметры
	 - Блок расчёта». Введите наименование создаваемого блока расчёта
	 и измените автоматически сформированный идентификатор при необходимости;


		- в [расширении «Алгоритмы расчёта»](../Work/Beginning_of_work.htm):


			- выполните команду «Блок
			 расчёта» в раскрывающемся меню кнопки ![](../Work/b_block.png)
			 «Блок расчёта» на
			 вкладке «Главная»
			 ленты инструментов;


			- выполните команду «Создать > Блок
			 расчёта» в контекстном меню рабочей области.


	Для создания нового блока расчёта на основе [блока
	 контроля](../Control_Block/Control_Block.htm) выполните команду «Копировать
	 как блок расчёта» в контекстном меню блока контроля. В результате
	 будет создан блок расчёта, содержащий те же настройки, что и исходный
	 блок контроля.


[![](../../Opened.gif)![](../../Closed.gif)Редактирование
 готового блока расчёта](javascript:TextPopup(this))


	Для редактирования готового блока расчёта:


		- в [навигаторе
		 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm) выделите блок расчёта и выполните
		 одно из действий:


			- нажмите кнопку ![](b_edit.png) «Редактировать»
			 в группе «Открыть»
			 на вкладке «Главная»
			 ленты инструментов;


			- выполните команду «Редактировать»
			 в контекстном меню блока расчёта;


			- нажмите клавишу F4;


		- в [расширении «Алгоритмы расчёта»](../Work/Beginning_of_work.htm) выделите
		 блок расчёта в рабочей области:


			- нажмите кнопку ![](b_edit.png) «Редактировать»
			 на вкладке «Главная»
			 ленты инструментов;


			- выполните команду «Редактировать»
			 в контекстном меню блока расчёта.


			- нажмите клавишу F4.


Создание и редактирование выполняется с помощью мастера редактирования
 блока расчёта, в котором настраиваются параметры блока.
