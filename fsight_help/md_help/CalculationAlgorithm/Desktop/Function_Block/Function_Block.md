# Вставка и настройка блоков функций


Блок функции предназначен для расчёта прикладной функции.


[![](../../Opened.gif)![](../../Closed.gif)Создание
 нового блока функции](javascript:TextPopup(this))


	Для создания нового блока функции:


		- в [навигаторе
		 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm):


			- нажмите кнопку «Новый
			 объект > Пользовательские объекты > Блок функции»
			 в группе «Создать»
			 на вкладке «Главная»
			 ленты инструментов;


			- выполните команду «Создать >
			 Пользовательские объекты > Блок функции» в
			 контекстном меню навигатора объектов.


	После выполнения одного из действий будет
	 открыто окно «Вставка объекта - параметры
	 - Блок функции». Введите наименование создаваемого блока функции
	 и измените автоматически сформированный идентификатор при необходимости;


		- в [расширении «Алгоритмы расчёта»](../Work/Beginning_of_work.htm):


			- выполните команду «Блок
			 функции» в раскрывающемся меню кнопки ![](../Work/b_block.png)
			 «Блок расчёта» на
			 вкладке «Главная»
			 ленты инструментов;


			- выполните команду «Создать > Блок
			 функции» в контекстном меню рабочей области.


[![](../../Opened.gif)![](../../Closed.gif)Редактирование
 готового блока функции](javascript:TextPopup(this))


	Для редактирования готового блока функции:


		- в [навигаторе
		 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm) выделите блок функции и выполните
		 одно из действий:


			- нажмите кнопку ![](../Calculation_block/b_edit.png)
			 «Редактировать» в
			 группе «Открыть» на
			 вкладке «Главная»
			 ленты инструментов;


			- выполните команду «Редактировать»
			 в контекстном меню блока функции;


			- нажмите клавишу F4;


		- в [расширении «Алгоритмы расчёта»](../Work/Beginning_of_work.htm) выделите
		 блок агрегации в рабочей области:


			- нажмите кнопку ![](../Calculation_block/b_edit.png) «Редактировать» на вкладке
			 «Главная» ленты инструментов;


			- выполните команду «Редактировать»
			 в контекстном меню блока функции;


			- нажмите клавишу F4;


			- дважды щёлкните по блоку функции.


[![](../../Opened.gif)![](../../Closed.gif)Редактирование
 прикладной функции](javascript:TextPopup(this))


	Для просмотра и редактирования функции, используемой готовым блоком,
	 выполните команду «Открыть модуль»
	 в контекстном меню блока. Функция будет открыта в среде разработки.


Создание и редактирование выполняется с помощью мастера редактирования
 блока функции, в котором настраиваются параметры блока.
