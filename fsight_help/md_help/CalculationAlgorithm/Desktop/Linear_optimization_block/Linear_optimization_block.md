# Вставка и настройка блоков линейной
 оптимизации


Блок линейной оптимизации предназначен для решения задачи оптимального
 использования ограниченных ресурсов.


В задаче управляющие переменные - это ресурсы, а целевая функция - это
 сущность, подлежащая оптимизации. Решение задачи - это нахождение таких
 значений управляющих переменных, которые удовлетворяют заданным ограничениям
 и при которых целевая функция принимает наибольшее или наименьшее значение.
 Условия, для которых составлялась задача, могут измениться. Для учёта
 этих условий без составления новой задачи вводятся коэффициенты целевой
 функции, позволяющие учитывать воздействие изменений условий на расчёт
 целевой функции.


[![](../../Opened.gif)![](../../Closed.gif)Пример задачи,
 решаемой с помощью линейной оптимизации](javascript:TextPopup(this))


	Например, фирма рекламирует свою продукцию с помощью интернета,
	 телевидения и газет. Каждый рубль, потраченный на рекламу в интернете
	 приводит к увеличению прибыли на десять рублей. Рубль, потраченный
	 на телевизионную рекламу даёт увеличение прибыли на семь рублей, на
	 рекламу в газетах - четыре рубля прибыли.


	Распределение рекламного бюджета имеет следующие ограничения:


		- полный рекламный бюджет не превышает 500000 рублей;


		- на рекламу в интернете можно потратить не более 60% всего
		 рекламного бюджета;


		- на телевизионную рекламу должно приходиться не менее 50%
		 всего рекламного бюджета;


		- на рекламу в газетах можно отвести не более 20% от всего
		 рекламного бюджета.


	С помощью блока линейной оптимизации можно рассчитать модель рекламного
	 бюджета, обеспечивающую фирме максимальную прибыль.


	Где:


		- управляющие переменные - это бюджет на рекламу в интернете,
		 бюджет на телевизионную рекламу и бюджет на рекламу в газетах;


		- коэффициенты целевой функции - это прибыль, получаемая с
		 каждого рубля, потраченного на рекламу;


		- ограничения - это распределение рекламного бюджета;


		- целевая функция - прибыль.


	Представим данную задачу в виде формул. Управляющие переменные x1,
	 x2
	 и x3
	 - это бюджет на рекламу в интернете, телевидении и в газетах соответственно.
	 На них наложены следующие ограничения:


		- x1 + x2 + x3 ≤ 500000
		 - полный рекламный бюджет не превышает 500000 рублей;


		- x1 ≤ 300000 - на рекламу в интернете
		 можно потратить не более 60% всего рекламного бюджета;


		- x2 ≥ 250000 - на телевизионную
		 рекламу должно приходиться не менее 50% всего рекламного бюджета;


		- x3 ≤ 100000 - на рекламу в газетах
		 можно отвести не более 20% от всего рекламного бюджета.


	Целевая функция - это получение максимальной прибыли:


	10x1 + 7x2 + 4x3
	 → max


	Где 10, 7 и 4 - это коэффициенты целевой функции для управляющих
	 переменных x1, x2 и x3
	 соответственно.


[![](../../Opened.gif)![](../../Closed.gif)Создание
 нового блока линейной оптимизации](javascript:TextPopup(this))


	Для создания нового блока линейной оптимизации:


		- в [навигаторе
		 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm):


			- нажмите кнопку «Новый
			 объект > Пользовательские объекты > Блок линейной
			 оптимизации» в группе «Создать»
			 на вкладке «Главная»
			 ленты инструментов;


			- выполните команду «Создать >
			 Пользовательские объекты > Блок линейной оптимизации»
			 в контекстном меню навигатора объектов.


	После выполнения одного из действий будет
	 открыто окно «Вставка объекта - параметры
	 - Блок линейной оптимизации». Введите наименование создаваемого
	 блока линейной оптимизации и измените автоматически сформированный
	 идентификатор при необходимости;


		- в [расширении «Алгоритмы расчёта»](../Work/Beginning_of_work.htm):


			- выполните команду «Блок
			 линейной оптимизации» в раскрывающемся меню кнопки
			 ![](../Work/b_block.png) «Блок
			 расчёта» на вкладке «Главная»
			 ленты инструментов;


			- выполните команду «Создать > Блок
			 линейной оптимизации» в контекстном меню рабочей области.


[![](../../Opened.gif)![](../../Closed.gif)Редактирование
 готового блока линейной оптимизации](javascript:TextPopup(this))


	Для редактирования готового блока линейной оптимизации:


		- в [навигаторе
		 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm) выделите блок функции и выполните
		 одно из действий:


			- нажмите кнопку ![](../Calculation_block/b_edit.png)
			 «Редактировать» в
			 группе «Открыть» на
			 вкладке «Главная»
			 ленты инструментов;


			- выполните команду «Редактировать»
			 в контекстном меню блока линейной оптимизации;


			- нажмите клавишу F4;


		- в [расширении «Алгоритмы расчёта»](../Work/Beginning_of_work.htm) выделите
		 блок линейной оптимизации в рабочей области:


			- нажмите кнопку ![](../Calculation_block/b_edit.png) «Редактировать» на вкладке
			 «Главная» ленты инструментов;


			- выполните команду «Редактировать»
			 в контекстном меню блока линейной оптимизации;


			- нажмите клавишу F4.


[![](../../Opened.gif)![](../../Closed.gif)Просмотр
 целевой функции](javascript:TextPopup(this))


	Для просмотра целевой функции выполните
	 команду «Целевая функция»
	 в контекстном меню блока линейной оптимизации. Будет открыт диалог
	 «[Внешний
	 вид целевой функции](ViewTagretFunction_LinOpt.htm)».


Создание и редактирование выполняется с помощью мастера редактирования
 блока линейной оптимизации, в котором настраиваются параметры блока.


При расчёте блока линейной оптимизации используется модуль для решения
 линейных задач LPSolve. Перед использованием блока предварительно должна
 быть проведена [интеграция
 с LPSolve](UiNav.chm::/02_Navigator/CommonSettings/Integration.htm#lpsolve).
