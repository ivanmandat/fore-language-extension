# Как использовать модуль расчёта Gurobi?


Gurobi - это внешний модуль решения задач с параллельными алгоритмами
 для задач линейного программирования, задач квадратичного программирования
 и смешанно-целочисленных задач. Модуль обладает повышенной производительностью
 при решении задач большого размера за счет использования современных
 алгоритмов и поддержки архитектуры последних многоядерных процессоров.


В продукте «Форсайт. Аналитическая платформа»
 модуль расчёта Gurobi может использоваться для решения задачи оптимального
 использования ограниченных ресурсов при настройке [блока
 линейной оптимизации](../Desktop/Linear_optimization_block/Linear_optimization_block.htm) алгоритма расчёта.


Для использования Gurobi предварительно с сайта разработчика [https://www.gurobi.com/](https://www.gurobi.com/) на компьютеры пользователей
 необходимо скачать и установить программное обеспечение Gurobi Optimizer.
 Используемая разрядность программного обеспечения должна совпадать с разрядностью
 продукта «Форсайт. Аналитическая платформа».
 Также для Gurobi должна быть получена лицензия, соответствующая решаемым
 задачам. После этого Gurobi будет доступен для использования, выбор внешнего
 модуля решения осуществляется на вкладке «[Дополнительные
 настройки](../Desktop/Linear_optimization_block/DataBase_LinOpt.htm)» в мастере блока линейной оптимизации.


Примечание.
 При установке Gurobi Optimizer используйте пути, предлагаемые по-умолчанию.
