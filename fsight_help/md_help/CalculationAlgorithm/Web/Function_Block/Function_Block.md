# Вставка и настройка блоков функций


Блок функции предназначен для расчёта прикладной функции.


[![](../../Opened.gif)![](../../Closed.gif)Создание
 нового блока функции](javascript:TextPopup(this))


	Для создания нового блока функции:


	- в [навигаторе
	 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm) нажмите кнопку ![](../Work/b_add.png) «Создать» в главном меню и выберите
	 тип объекта на боковой панели «[Новый
	 объект](UiNav.chm::/02_Navigator/General_Principles_of_Work.htm#add_object)».


	- в [расширении
	 «Алгоритмы расчёта»](../Work/Beginning_of_work.htm#create) выполните
	 одно из действий:


			- выполните команду ![](func_block.png) «Блок
			 функции» в раскрывающемся меню кнопки ![](../Work/b_block.png) «Добавить
			 блок» на [панели
			 инструментов](../Work/Beginning_of_work.htm);


			- выполните команду «Создать
			 > ![](func_block.png) Блок функции» в контекстном
			 меню [рабочей области](../Work/Beginning_of_work.htm).


[![](../../Opened.gif)![](../../Closed.gif)Редактирование
 готового блока функции](javascript:TextPopup(this))


	Для редактирования готового блока функции:


	- в [навигаторе
	 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm) выделите блок и выполните одно из действий:


		- выполните команду «Редактировать»
		 в контекстном меню блока;


		- выполните команду «Редактировать
		 в новой вкладке» в контекстном меню блока;


		- выполните команду «Редактировать
		 в новом окне» в контекстном меню блока;


		- нажмите клавишу F4;


		- в [расширении
		 «Алгоритмы расчёта»](../Work/Beginning_of_work.htm#create) выполните
		 команду «Редактировать»
		 в контекстном меню блока, расположенного в [дереве
		 расчёта](../Work/Beginning_of_work.htm) или на [рабочей
		 области](../Work/Beginning_of_work.htm).


Создание и редактирование выполняется с помощью мастера редактирования
 блока функции, в котором настраиваются параметры блока:


![](BasicProperties.png)


Настройка блока функций состоит из следующих шагов:


	- [Настройка базовых свойств](Basic_Properties_Func.htm).


	- [Добавление параметров](ParametersFunc.htm).


	- [Выбор пользовательской функции](Function_Block_Wizard.htm).


Для сохранения настроек:


	- нажмите кнопку «Сохранить»;


	- выполните команду «Сохранить
	 и закрыть» в раскрывающемся меню кнопки «Сохранить».
	 По завершению выполнения команды будут сохранены заданные настройки
	 и закрыт мастер редактирования блока функции.


Примечание.
 Команда «Сохранить и закрыть»
 доступна только при открытии блока функции на [редактирование](Function_Block.htm#edit).
