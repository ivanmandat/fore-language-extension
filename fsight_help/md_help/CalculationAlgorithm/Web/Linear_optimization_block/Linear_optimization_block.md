# Вставка и настройка блоков линейной
 оптимизации


Блок линейной оптимизации предназначен для решения задачи оптимального
 использования ограниченных ресурсов.


В задаче управляющие переменные - это ресурсы, а целевая функция - это
 сущность, подлежащая оптимизации. Решение задачи - это нахождение таких
 значений управляющих переменных, которые удовлетворяют заданным ограничениям
 и при которых целевая функция принимает наибольшее или наименьшее значение.
 Условия, для которых составлялась задача, могут измениться. Для учёта
 этих условий без составления новой задачи вводятся коэффициенты целевой
 функции, позволяющие учитывать воздействие изменений условий на расчёт
 целевой функции.


[![](../../Opened.gif)![](../../Closed.gif)Создание
 нового блока линейной оптимизации](javascript:TextPopup(this))


	Для создания нового блока линейной оптимизации:


	- в [навигаторе
	 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm) нажмите кнопку ![](../Work/b_add.png) «Создать» в главном меню и выберите
	 тип объекта на боковой панели «[Новый
	 объект](UiNav.chm::/02_Navigator/General_Principles_of_Work.htm#add_object)».


	- в [расширении
	 «Алгоритмы расчёта»](../Work/Beginning_of_work.htm#create) выполните
	 одно из действий:


			- выполните команду ![](Linear_optimization_block.png) «Блок
			 линейной оптимизации» в раскрывающемся меню кнопки
			 ![](../Work/b_block.png) «Добавить
			 блок» на [панели
			 инструментов](../Work/Beginning_of_work.htm);


			- выполните команду «Создать > ![](Linear_optimization_block.png) Блок линейной оптимизации» в контекстном
			 меню рабочей области.


[![](../../Opened.gif)![](../../Closed.gif)Редактирование
 готового блока линейной оптимизации](javascript:TextPopup(this))


	Для редактирования готового блока линейной оптимизации:


	- в [навигаторе
	 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm) выделите блок и выполните одно из действий:


		- выполните команду «Редактировать»
		 в контекстном меню блока;


		- выполните команду «Редактировать
		 в новой вкладке» в контекстном меню блока;


		- выполните команду «Редактировать
		 в новом окне» в контекстном меню блока;


		- нажмите клавишу F4;


		- в [расширении
		 «Алгоритмы расчёта»](../Work/Beginning_of_work.htm#create)
		 выполните команду «Редактировать»
		 в контекстном меню блока, расположенного в [дереве
		 расчёта](../Work/Beginning_of_work.htm) или на [рабочей
		 области](../Work/Beginning_of_work.htm).


[![](../../Opened.gif)![](../../Closed.gif)Просмотр
 целевой функции](javascript:TextPopup(this))


	Для просмотра целевой функции:


		- нажмите кнопку «Показать
		 внешний вид функции» в мастере редактирования блока линейной
		 оптимизации. Кнопка доступна, начиная со страницы «[Целевая функция](TargetFunction_LinOpt.htm)»;


		- выполните команду «Целевая
		 функция» в контекстном меню блока линейной оптимизации.


	Будет открыт диалог «[Внешний
	 вид целевой функции](ViewTagretFunction_LinOpt.htm)».


Создание и редактирование выполняется с помощью мастера редактирования
 блока линейной оптимизации, в котором настраиваются параметры блока.


![](Basic_Properties.png)


Настройка блока линейной оптимизации состоит из следующих шагов:


	- [Настройка базовых свойств](Basic_Properties_LinOpt.htm).


	- [Добавление параметров](Parameters_LinOpt.htm).


	- [Настройка целевой функции](TargetFunction_LinOpt.htm).


	- [Настройка простых
	 ограничений управляющих переменных](VariablesRestriction_LinOpt.htm).


При расчёте блока линейной оптимизации используется модуль для решения
 линейных задач LPSolve. Перед использованием блока предварительно должна
 быть проведена [интеграция
 с LPSolve](UiNav.chm::/02_Navigator/CommonSettings/Integration.htm#lpsolve).


Для сохранения настроек:


	- нажмите кнопку «Сохранить»;


	- выполните команду «Сохранить
	 и закрыть» в раскрывающемся меню кнопки «Сохранить».
	 По завершению выполнения команды будут сохранены заданные настройки
	 и закрыт мастер редактирования блока линейной оптимизации.


Примечание.
 Команда «Сохранить и закрыть»
 доступна только при открытии блока линейной оптимизации на [редактирование](Linear_optimization_block.htm#edit).
