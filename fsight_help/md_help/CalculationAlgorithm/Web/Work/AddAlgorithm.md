# Вставка готовых алгоритмов расчёта и блоков


Вставка готового алгоритма расчёта или блока позволяет встроить существующий
 расчёт в текущий расчёт.


[![](../../Opened.gif)![](../../Closed.gif)Вставка готового
 алгоритма расчёта](javascript:TextPopup(this))


	Для вставки готового алгоритма расчёта:


		- Выполните одно из действий:


			- нажмите кнопку ![](b_addTask.png)
			 «Добавить алгоритм расчёта»
			 на [панели инструментов](Beginning_of_work.htm);


			- выполните команду «Добавить
			 > Объект расчёта» в контекстном меню [рабочей
			 области](Beginning_of_work.htm).


		- Выберите готовый алгоритм расчёта на боковой панели и нажмите
		 кнопку «Применить».


	После выполнения действий готовый алгоритм расчёта будет добавлен
	 в [дерево расчёта](Beginning_of_work.htm) и на [рабочую
	 область](Beginning_of_work.htm).


[![](../../Opened.gif)![](../../Closed.gif)Вставка готового
 блока](javascript:TextPopup(this))


	Для вставки готового блока в алгоритм расчёта:


		- Выполните одно из действий:


			- выполните команду «Существующий
			 блок» в раскрывающемся меню кнопки ![](b_block.png)
			 «Добавить блок» на
			 [панели инструментов](Beginning_of_work.htm);


			- выполните команду «Добавить
			 > Объект расчёта» в контекстном меню [рабочей
			 области](Beginning_of_work.htm).


		- Выберите готовый блок на боковой панели и нажмите кнопку
		 «Применить».


	После выполнения действий готовый блок будет добавлен в [дерево
	 расчёта](Beginning_of_work.htm) и на [рабочую область](Beginning_of_work.htm).


Для просмотра блоков, входящих в добавленный алгоритм расчёта, используйте
 [дерево расчёта](Beginning_of_work.htm):
