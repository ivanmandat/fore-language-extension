# Построение алгоритма расчёта


Для построения алгоритма расчёта в веб-приложении выполните шаги:


	- [Вставьте
	 и настройте блоки расчёта](../Calculation_block/Calculation_unit.htm).


	- [Вставьте
	 и настройте блоки агрегации](../Aggregation_block/Aggregation_block.htm).


	- [Вставьте и настройте
	 блоки функций](../Function_Block/Function_Block.htm).


	- [Вставьте
	 и настройте блоки линейной оптимизации](../Linear_optimization_block/Linear_optimization_block.htm).


	- [Вставьте и настройте
	 блоки контроля](../Control_Block/Control_Block.htm).


	- [Вставьте готовые алгоритмы расчёта
	 и блоки](AddAlgorithm.htm).


	- [Задайте настройки
	 алгоритма расчёта](Set_up_calculation_parameters.htm).


	- [Настройте условия выполнения
	 расчёта алгоритма](CalculationConditions.htm).


	- [Настройте порядок расчёта алгоритма](CalculationOrder.htm).


	- [Выполните отладку расчёта алгоритма](Debug.htm).


	- [Вставьте объекты репозитория](AddRepoObject.htm).


	- [Управляйте элементами алгоритма
	 расчёта](Management_of_Elements.htm).


	- [Настройте оформление алгоритма
	 расчёта](AlgorithmDecor.htm).


После выполнения действий сохраните изменения алгоритма расчёта с помощью
 кнопки «Сохранить» в [главном
 меню](Beginning_of_work.htm). Сохранение доступно только при [создании](Beginning_of_work.htm#create)
 нового или [открытия на редактирование](Beginning_of_work.htm#open)
 готового алгоритма расчёта.
