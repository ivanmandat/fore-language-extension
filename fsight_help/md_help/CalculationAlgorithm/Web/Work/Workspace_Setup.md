# Настройка рабочей области


Рабочая область - контейнер, в котором расположены объекты расчёта.
 Рабочая область является удобным механизмом для:


	- визуального размещения объектов расчёта на едином рабочем поле;


	- организации доступа к свойствам объектов расчёта;


	- отображения заданного текста и добавленных фигур для объектов.


## Перемещение по рабочей области


Для перемещения по рабочей области выполните одно из действий:


	- выберите режим ![](b_hand.png) «Рука»
	 на [панели инструментов](Beginning_of_work.htm) и зажмите
	 кнопку мыши. Если выбран режим ![](b_cursor.png) «Курсор»
	 для [выделения объектов](Management_of_Elements.htm#select),
	 то используйте раскрывающийся список для смены режима;


	- зажмите клавишу ПРОБЕЛ и кнопку мыши;


	- используйте стрелки на клавиатуре;


	- прокрутите колёсико мыши для перемещения рабочей области по
	 вертикали. Для перемещения рабочей области по горизонтали дополнительно
	 зажмите клавишу SHIFT;


	- используйте миникарту. Для отображения миникарты нажмите кнопку
	 ![](b_mini_map.png)
	 «Включить миникарту» на панели
	 настроек отображения рабочей области:


![](minimap_example.png)


## Масштабирование рабочей области


В рабочей области может быть установлен масштаб отображения в диапазоне
 [1%, 400%].


Для масштабирования рабочей области:


	- используйте панель настроек отображения рабочей области:


![](zoom_panel.png)


На панели доступны следующие операции:


		- ![](b_adjust_for_all.png) «Подогнать
		 по всем объектам». Изменение масштаба рабочей области по
		 всем объектам;


		- ![](b_adjust_for_one.png) «Подогнать
		 по объекту». Изменение масштаба рабочей области по выделенному
		 объекту;


Примечание.
 Если ни один объект на рабочей области не выделен, то кнопка ![](b_adjust_for_one.png) «Подогнать по
 объекту» будет недоступна.


		- ![](b_scale_plus.png) «Увеличить
		 масштаб». Увеличение масштаба рабочей области на 25%;


		- ![](b_scale_minus.png) «Уменьшить
		 масштаб». Уменьшение масштаба рабочей области на 25%.


Для установки стопроцентного масштаба нажмите
 кнопку с отображением текущего масштаба;


	- прокрутите колёсико мыши с зажатой клавишей CTRL.
