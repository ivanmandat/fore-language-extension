# Настройки алгоритма расчёта


Настройка алгоритма расчёта выполняется с помощью мастера. Для открытия
 мастера редактирования настроек алгоритма расчёта нажмите кнопку ![](b_settings.png) «Настройки»
 на [панели базовых настроек](Beginning_of_work.htm).


После выполнения действия будет открыт мастер редактирования настроек
 алгоритма расчёта:


![](TaskBaseSettings.png)


Мастер позволяет задать настройки:


	- [Базовые
	 настройки](base_settings.htm). Задаются наименование алгоритма расчёта, его
	 идентификатор и примечание;


	- [Параметры](Parameters_of_the_calculation_algorithm.htm).
	 Задаётся список параметров для использования в алгоритме расчёта;


	- [Управление
	 параметрами](Manage_settings.htm). Задаются связи параметров алгоритма расчёта
	 с параметрами блоков;


	- [Управление
	 параметрами кубов](Manage_cubes_settings.htm). Задаются связи параметров алгоритма
	 расчёта с параметрами кубов;


	- [Управление
	 периодом расчёта](Calculation_period_management.htm). Задаются параметры алгоритма расчёта,
	 управляющие периодом расчёта;


	- [Управление
	 параметрами объектов репозитория](Manage_objects_settings.htm). Задаются связи параметров
	 алгоритма расчёта с параметрами объектов репозитория, вставленных
	 на рабочую область.


Для сохранения настроек:


	- нажмите кнопку «Сохранить»;


	- выполните команду «Сохранить
	 и закрыть» в раскрывающемся меню кнопки «Сохранить».
	 По завершению выполнения команды будут сохранены заданные настройки
	 и закрыт мастер редактирования настроек алгоритма расчёта.
