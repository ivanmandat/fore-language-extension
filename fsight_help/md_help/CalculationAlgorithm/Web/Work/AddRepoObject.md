# Вставка объектов репозитория


Вставка объектов репозитория позволяет добавить источники и приёмники
 данных в рабочую область. За счёт этого обеспечивается наглядность алгоритма
 расчёта.


Для вставки объекта репозитория:


	- Выполните одно из действий:


		- нажмите кнопку ![](b_addObject.png)
		 «Объект репозитория» на
		 [панели инструментов](Beginning_of_work.htm);


		- выполните команду «Добавить
		 > Объект репозитория» в контекстном меню [рабочей
		 области](Beginning_of_work.htm).


	- Выберите объект репозитория на боковой панели и нажмите кнопку
	 «Применить».


После выполнения действий выбранный объект репозитория будет вставлен
 на [рабочую область](Beginning_of_work.htm).


Примечание.
 Настройка оформления объектов репозитория аналогична [настройке
 оформления текстового блока](AlgorithmDecor.htm#p_text).
