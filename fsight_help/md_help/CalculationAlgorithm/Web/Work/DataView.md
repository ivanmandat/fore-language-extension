# Просмотр рассчитанных данных


Просмотр рассчитанных данных доступен для объектов, являющихся приёмниками
 [блоков расчёта](../Calculation_block/Calculation_unit.htm),
 [блоков агрегации](../Aggregation_block/Aggregation_block.htm),
 [блоков
 линейной оптимизации](../Linear_optimization_block/Linear_optimization_block.htm) и [блоков
 контроля](../Control_Block/Control_Block.htm):


	- при [открытии алгоритма
	 расчёта](Beginning_of_work.htm#open) данные в приёмниках можно посмотреть до выполнения расчёта;


	- при [отладке алгоритма расчёта](Debug.htm) в приёмниках
	 данных можно посмотреть промежуточные результаты расчёта;


	- при [расчёте алгоритма целиком](Perform_calculations.htm)
	 рассчитанные данные в приёмниках можно посмотреть только по окончании
	 расчёта.


Также просмотр данных доступен для [объектов
 репозитория](AddRepoObject.htm), вставленных в алгоритм расчёта.


Для просмотра:


	- данных объекта, являющегося приёмником:


		- Выделите блок расчёта, блок агрегации, блок линейной оптимизации
		 или блок контроля на [рабочей области](Beginning_of_work.htm).


		- Выполните команду «Открыть
		 приёмник в отчёте» в контекстном меню блока.


В результате объект будет открыт в инструменте
 «[Отчеты](UIReport.chm::/uireport_purpose.htm)».
 При отладке или расчёте алгоритма исходная отметка измерений куба меняется
 в зависимости от [настроек
 приёмника](../Calculation_block/Master_calculation_block_page_consumer.htm), в результате чего при открытии куба будет отображаться
 отметка измерений только по данным, рассчитанным соответствующим блоком.
 Если в блоке расчёта или блоке контроля настроена [фильтрация
 по данным](../Calculation_block/FilteringByData.htm), то она не будет влиять на отметку измерений. При необходимости
 [сохраните
 отчёт](UiReport.chm::/Web/Reports/OperationReport/UiReport_Reports_Operation_Save.htm).


	- данных объекта, вставленного в алгоритм расчёта:


		- выполните команду «Открыть»
		 в контекстном меню объекта;


		- дважды щёлкните по выбранному объекту.


В результате объект будет открыт на просмотр.
 Если объект содержит параметры, которые не связаны с параметрами алгоритма
 расчёта, то перед открытием объекта будет открыто окно «[Установка
 значений параметров](uinav.chm::/02_Navigator/General_Principles_of_Work.htm#open_param_object)». Для получения подробной
 информации о связях параметров объекта с параметрами алгоритма расчёта
 обратитесь к разделу «[Задание связей
 с параметрами объектов репозитория](Manage_objects_settings.htm)».
