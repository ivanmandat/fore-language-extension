# Перечисления сборки ABAC


В рамках сборки ABAC существуют
 следующие перечисления:


		 Перечисление
		 Краткое описание


		 ![](../Enums.gif)
		 [ABACCombineAlgorithm](ABACCombineAlgorithm.htm)
		 Перечисление ABACCombineAlgorithm
		 используется для выбора [алгоритма
		 комбинации](Admin.chm::/04_SecurityPolicy/Admin_AttributeAccess.htm#properties) корневого набора/политик/правил.


		 ![](../Enums.gif)
		 [ABACEvaluateResult](ABACEvaluateResult.htm)
		 Перечисление ABACEvaluateResult
		 используется для определения доступа субъекта к объекту.


		 ![](../Enums.gif)
		 [ABACRuleEffect](ABACRuleEffect.htm)
		 Перечисление ABACRuleEffect
		 используется для выбора [эффекта](Admin.chm::/04_SecurityPolicy/Admin_AttributeAccess.htm#properties)
		 правила при успешном выполнении [цели](Admin.chm::/04_SecurityPolicy/Target.htm)
		 и [условия](Admin.chm::/04_SecurityPolicy/Creating_conditions.htm).


		 ![](../Enums.gif)
		 [ABACTargetOperation](ABACTargetOperation.htm)
		 Перечисление ABACTargetOperation
		 используется для выбора условной операции в [цели](Admin.chm::/04_SecurityPolicy/Target.htm).
