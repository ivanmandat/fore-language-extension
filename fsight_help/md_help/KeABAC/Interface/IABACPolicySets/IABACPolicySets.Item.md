# IABACPolicySets.Item


## Синтаксис


Item(Index: Integer): [IABACPolicySet](../IABACPolicySet/IABACPolicySet.htm);


## Параметры


Index. Индекс набора политик
 в структуре атрибутного доступа.


## Описание


Свойство Item возвращает набор
 политик из атрибутного доступа по индексу.
