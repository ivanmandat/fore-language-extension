# IABACPolicySets


## Описание


Интерфейс IABACPolicySets предназначен
 для работы с [наборами
 политик](Admin.chm::/04_SecurityPolicy/Admin_AttributeAccess.htm#policies_set) в структуре атрибутного доступа.


## Комментарии


Для определения [набора
 политик](Admin.chm::/04_SecurityPolicy/Admin_AttributeAccess.htm#policies_set) в атрибутном доступе используйте интерфейс
 [IABACPolicySet](../IABACPolicySet/IABACPolicySet.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IABACPolicySets.Count.htm)
		 Свойство Count
		 возвращает количество наборов политик, содержащихся в атрибутном
		 доступе.


		 ![](../../Property_Image.gif)
		 [Item](IABACPolicySets.Item.htm)
		 Свойство Item возвращает
		 набор политик из атрибутного доступа по индексу.
