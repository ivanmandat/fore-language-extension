# IABACPolicySets.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ
 набора политик.


## Описание


Метод RemoveByKey осуществляет
 удаление набора политик в атрибутном доступе по ключу.


## Комментарии


Для определения ключа набора политик используйте свойство [IABACPolicySet.Key](../IABACPolicySet/IABACPolicySet.Key.htm).
