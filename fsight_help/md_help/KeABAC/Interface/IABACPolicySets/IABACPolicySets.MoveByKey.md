# IABACPolicySets.MoveByKey


## Синтаксис


MoveByKey(FromKey: Integer; ToKey: Integer): Boolean;


## Параметры


FromKey. Ключ набора политик
 в атрибутном доступе, который необходимо переместить;


ToKey. Ключ набора политик,
 на место которого необходимо переместить указанный набор политик в атрибутном
 доступе.


## Описание


Метод MoveByKey осуществляет
 перемещение набора политик среди существующих наборов политик в атрибутном
 доступе по ключу.


## Комментарии


Для определения ключа набора политик используйте свойство [IABACPolicySet.Key](../IABACPolicySet/IABACPolicySet.Key.htm).
