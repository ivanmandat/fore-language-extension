# IABACPolicySets.SwapById


## Синтаксис


SwapById(Id1: String; Id2: String): Boolean;


## Параметры


Id1. Идентификатор набора политик
 в атрибутном доступе, который необходимо переместить;


Id2. Идентификатор набора политик,
 вместо которого необходимо переместить указанный набор политик в атрибутном
 доступе.


Примечание.
 Наборы политик с указанными идентификаторами будут перемещены между собой.


## Описание


Метод SwapById меняет местами
 положение заданных наборов политик в атрибутном доступе по идентификатору.


## Комментарии


Для определения идентификатора набора политик используйте свойство [IABACPolicySet.Id](../IABACPolicySet/IABACPolicySet.Id.htm).
