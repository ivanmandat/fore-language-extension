# IABACPolicySets.FindById


## Синтаксис


FindById(Id: String): [IABACPolicySet](../IABACPolicySet/IABACPolicySet.htm);


## Параметры


Id. Идентификатор набора политик.


## Описание


Метод FindById осуществляет
 поиск набора политик в атрибутном доступе по идентификатору.


## Комментарии


Для определения идентификатора набора политик используйте свойство [IABACPolicySet.Id](../IABACPolicySet/IABACPolicySet.Id.htm).
