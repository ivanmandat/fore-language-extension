# IABACPolicySets.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IABACPolicySet](../IABACPolicySet/IABACPolicySet.htm);


## Параметры


Key. Ключ набора политик.


## Описание


Метод FindByKey осуществляет
 поиск набора политик в атрибутном доступе по ключу.


## Комментарии


Для определения ключа набора политик используйте свойство [IABACPolicySet.Key](../IABACPolicySet/IABACPolicySet.Key.htm).
