# IABACPolicySets.Move


## Синтаксис


Move(FromIndex: Integer; ToIndex: Integer): Boolean;


## Параметры


FromIndex. Индекс позиции набора
 политик в атрибутном доступе, который необходимо переместить;


ToIndex. Индекс позиции, в
 которую необходимо переместить набор политик в атрибутном доступе.


## Описание


Метод Move осуществляет перемещение
 набора политик среди существующих наборов политик в атрибутном доступе
 по индексу.
