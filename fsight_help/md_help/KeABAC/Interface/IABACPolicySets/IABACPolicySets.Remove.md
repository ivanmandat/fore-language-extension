# IABACPolicySets.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс набора политик.


## Описание


Метод Remove осуществляет удаление
 набора политик в атрибутном доступе по индексу.


## Комментарии


Метод возвращает значение True,
 если удаление прошло успешно.
