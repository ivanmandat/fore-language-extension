# IABACPolicySets.Swap


## Синтаксис


Swap(Index1: Integer; Index2: Integer): Boolean;


## Параметры


Index1. Индекс позиции набора
 политик в атрибутном доступе, который необходимо переместить;


Index2. Индекс позиции набора
 политик, вместо которого необходимо переместить указанный набор политик
 в атрибутном доступе.


Примечание.
 Наборы политик с указанными индексами будут перемещены между собой.


## Описание


Метод Swap меняет местами положение
 заданных наборов политик в атрибутном доступе по индексу.
