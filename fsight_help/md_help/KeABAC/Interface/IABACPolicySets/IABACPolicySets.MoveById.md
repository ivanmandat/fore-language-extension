# IABACPolicySets.MoveById


## Синтаксис


MoveById(FromId: String; ToId: String): Boolean;


## Параметры


FromId. Идентификатор набора
 политик в атрибутном доступе, который необходимо переместить;


ToId. Идентификатор набора
 политик, на место которого необходимо переместить указанный набор политик
 в атрибутном доступе.


## Описание


Метод MoveById осуществляет
 перемещение набора политик среди существующих наборов политик в атрибутном
 доступе по идентификатору.


## Комментарии


Для определения идентификатора набора политик используйте свойство [IABACPolicySet.Id](../IABACPolicySet/IABACPolicySet.Id.htm).
