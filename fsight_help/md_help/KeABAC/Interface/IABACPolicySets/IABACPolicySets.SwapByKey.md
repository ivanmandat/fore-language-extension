# IABACPolicySets.SwapByKey


## Синтаксис


SwapByKey(Id1: Integer; Id2: Integer): Boolean;


## Параметры


Id1. Ключ набора политик в
 атрибутном доступе, который необходимо переместить;


Id2. Ключ набора политик, вместо
 которого необходимо переместить указанный набор политик в атрибутном доступе.


Примечание.
 Наборы политик с указанными ключами будут перемещены между собой.


## Описание


Метод SwapByKey меняет местами
 положение заданных наборов политик в атрибутном доступе по ключу.


## Комментарии


Для определения ключа набора политик используйте свойство [IABACPolicySet.Key](../IABACPolicySet/IABACPolicySet.Key.htm).
