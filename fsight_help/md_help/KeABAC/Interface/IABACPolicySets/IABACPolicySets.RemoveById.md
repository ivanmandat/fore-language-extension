# IABACPolicySets.RemoveById


## Синтаксис


RemoveById(Id: String): Boolean;


## Параметры


Id. Идентификатор набора политик.


## Описание


Метод RemoveById осуществляет
 удаление набора политик в атрибутном доступе по идентификатору.


## Комментарии


Для определения идентификатора набора политик используйте свойство [IABACPolicySet.Id](../IABACPolicySet/IABACPolicySet.Id.htm).
