# IABACRules.RemoveById


## Синтаксис


RemoveById(Id: String): Boolean;


## Параметры


Id. Идентификатор правила.


## Описание


Метод RemoveById осуществляет
 удаление правила в политике по идентификатору.


## Комментарии


Для определения идентификатора правила используйте свойство [IABACRule.Id](../IABACRule/IABACRule.Id.htm).
