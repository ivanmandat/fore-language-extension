# IABACRules.FindById


## Синтаксис


FindById(Id: String): [IABACRule](../IABACRule/IABACRule.htm);


## Параметры


Id. Идентификатор правила.


## Описание


Метод FindById осуществляет
 поиск правила в политике по идентификатору.


## Комментарии


Для определения идентификатора правила используйте свойство [IABACRule.Id](../IABACRule/IABACRule.Id.htm).


## Пример


Использование метода приведено в примере для [IABACRules.AddCopy](IABACRules.AddCopy.htm).
