# IABACRules.Move


## Синтаксис


Move(FromIndex: Integer; ToIndex: Integer): Boolean;


## Параметры


FromIndex. Индекс позиции правила
 в политике, которое необходимо переместить;


ToIndex. Индекс позиции, в
 которую необходимо переместить правило в политике.


## Описание


Метод Move осуществляет перемещение
 правила среди существующих правил в политике по индексу.
