# IABACRules.MoveById


## Синтаксис


MoveById(FromId: String; ToId: String): Boolean;


## Параметры


FromId. Идентификатор правила
 в политике, которое необходимо переместить;


ToId. Идентификатор правила,
 на место которого необходимо переместить указанное правило в политике.


## Описание


Метод MoveById осуществляет
 перемещение правила среди существующих правил в политике по идентификатору.


## Комментарии


Для определения идентификатора правила используйте свойство [IABACRule.Id](../IABACRule/IABACRule.Id.htm).
