# IABACRules.SwapById


## Синтаксис


SwapById(Id1: String; Id2: String): Boolean;


## Параметры


Id1. Идентификатор правила
 в политике, которое необходимо переместить;


Id2. Идентификатор правила,
 вместо которого необходимо переместить указанное правило в политике.


Примечание.
 Правила с указанными идентификаторами будут перемещены между собой.


## Описание


Метод SwapById меняет местами
 положение заданных правил в политике по идентификатору.


## Комментарии


Для определения идентификатора правила используйте свойство [IABACRule.Id](../IABACRule/IABACRule.Id.htm).
