# IABACRules.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IABACRule](../IABACRule/IABACRule.htm);


## Параметры


Key. Ключ правила.


## Описание


Метод FindByKey осуществляет
 поиск правила в политике по ключу.


## Комментарии


Для определения ключа правила используйте свойство [IABACRule.Key](../IABACRule/IABACRule.Key.htm).
