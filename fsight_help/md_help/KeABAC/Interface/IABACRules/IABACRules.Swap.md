# IABACRules.Swap


## Синтаксис


Swap(Index1: Integer; Index2: Integer): Boolean;


## Параметры


Index1. Индекс позиции правила
 в политике, которое необходимо переместить;


Index2. Индекс позиции правила,
 вместо которого необходимо переместить указанное правило в политике.


Примечание.
 Правила с указанными индексами будут перемещены между собой.


## Описание


Метод Swap меняет местами положение
 заданных правил в политике по индексу.
