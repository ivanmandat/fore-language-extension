# IABACRules


## Описание


Интерфейс IABACRules предназначен
 для работы с [правилами](Admin.chm::/04_SecurityPolicy/Admin_AttributeAccess.htm#rule)
 в политике атрибутного доступа.


## Комментарии


Для определения [политики](Admin.chm::/04_SecurityPolicy/Admin_AttributeAccess.htm#policy)
 в атрибутном доступе используйте интерфейс [IABACPolicy](../IABACPolicy/IABACPolicy.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IABACRules.Count.htm)
		 Свойство Count
		 возвращает количество правил, содержащихся в политике.


		 ![](../../Property_Image.gif)
		 [Item](IABACRules.Item.htm)
		 Свойство Item возвращает
		 правило из политики по индексу.
