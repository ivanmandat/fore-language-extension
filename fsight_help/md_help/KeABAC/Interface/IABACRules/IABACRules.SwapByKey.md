# IABACRules.SwapByKey


## Синтаксис


SwapByKey(Id1: Integer; Id2: Integer): Boolean;


## Параметры


Id1. Ключ правила в политике,
 которое необходимо переместить;


Id2. Ключ правила, вместо которого
 необходимо переместить указанное правило в политике.


Примечание.
 Правила с указанными ключами будут перемещены между собой.


## Описание


Метод SwapByKey меняет местами
 положение заданных правил в политике по ключу.


## Комментарии


Для определения ключа правила используйте свойство [IABACRule.Key](../IABACRule/IABACRule.Key.htm).
