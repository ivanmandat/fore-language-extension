# IABACRules.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ
 правила.


## Описание


Метод RemoveByKey осуществляет
 удаление правила в политике по ключу.


## Комментарии


Для определения ключа правила используйте свойство [IABACRule.Key](../IABACRule/IABACRule.Key.htm).
