# IMetabaseSecurityClassesAttributes.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс типа объектов.


## Описание


Метод Remove осуществляет удаление
 атрибутов типа объектов по индексу.


## Комментарии


Метод возвращает значение True,
 если удаление прошло успешно.
