# IMetabaseSecurityClassesAttributes.Add


## Синтаксис


Add(ClassId: Integer): [IMetabaseSecurityClassAttributes](../IMetabaseSecurityClassAttributes/IMetabaseSecurityClassAttributes.htm);


## Параметры


ClassId. Идентификатор типа
 объекта.


## Описание


Метод Add осуществляет добавление
 атрибутов типу объектов по идентификатору.


## Комментарии


Идентификатор типа объектов определяется с помощью свойства [IMetabaseSecurityClassAttributes.ClassId](../IMetabaseSecurityClassAttributes/IMetabaseSecurityClassAttributes.ClassId.htm)
 или перечисления [MetabaseObjectClass](KeSom.chm::/Enums/MetabaseObjectClass.htm).
