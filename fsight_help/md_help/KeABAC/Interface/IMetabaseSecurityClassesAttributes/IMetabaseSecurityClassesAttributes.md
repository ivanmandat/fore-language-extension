# IMetabaseSecurityClassesAttributes


## Описание


Интерфейс IMetabaseSecurityClassesAttributes
 предназначен для работы с коллекцией [пользовательских](admin.chm::/04_SecurityPolicy/Admin_PermSep_Attributes.htm)
 атрибутов типа объектов.


## Комментарии


Для получения коллекции атрибутов и идентификатора типа объектов используйте
 интерфейс [IMetabaseSecurityClassAttributes](../IMetabaseSecurityClassAttributes/IMetabaseSecurityClassAttributes.htm).
