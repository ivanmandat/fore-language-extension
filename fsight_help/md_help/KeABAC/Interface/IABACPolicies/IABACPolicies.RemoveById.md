# IABACPolicies.RemoveById


## Синтаксис


RemoveById(Id: String): Boolean;


## Параметры


Id. Идентификатор политики.


## Описание


Метод RemoveById осуществляет
 удаление политики в атрибутном доступе по идентификатору.


## Комментарии


Для определения идентификатора политики используйте свойство [IABACPolicy.Id](../IABACPolicy/IABACPolicy.Id.htm).
