# IABACPolicies.MoveById


## Синтаксис


MoveById(FromId: String; ToId: String): Boolean;


## Параметры


FromId. Идентификатор политики
 в атрибутном доступе, которую необходимо переместить;


ToId. Идентификатор политики,
 на место которой необходимо переместить указанную политику в атрибутном
 доступе.


## Описание


Метод MoveById осуществляет
 перемещение политики среди существующих политик в атрибутном доступе по
 идентификатору.


## Комментарии


Для определения идентификатора политики используйте свойство [IABACPolicy.Id](../IABACPolicy/IABACPolicy.Id.htm).
