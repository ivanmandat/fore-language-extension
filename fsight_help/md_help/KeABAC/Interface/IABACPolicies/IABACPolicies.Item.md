# IABACPolicies.Item


## Синтаксис


Item(Index: Integer): [IABACPolicy](../IABACPolicy/IABACPolicy.htm);


## Параметры


Index. Индекс политики в структуре
 атрибутного доступа.


## Описание


Свойство Item возвращает политику
 из атрибутного доступа по индексу.
