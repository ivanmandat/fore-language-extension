# IABACPolicies.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IABACPolicy](../IABACPolicy/IABACPolicy.htm);


## Параметры


Key. Ключ политики.


## Описание


Метод FindByKey осуществляет
 поиск политики в атрибутном доступе по ключу.


## Комментарии


Для определения ключа политики используйте свойство [IABACPolicy.Key](../IABACPolicy/IABACPolicy.Key.htm).
