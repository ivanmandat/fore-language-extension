# IABACPolicies


## Описание


Интерфейс IABACPolicies предназначен
 для работы с [политиками](Admin.chm::/04_SecurityPolicy/Admin_AttributeAccess.htm#policy)
 в структуре атрибутного доступа.


## Комментарии


Для определения [политики](Admin.chm::/04_SecurityPolicy/Admin_AttributeAccess.htm#policy)
 в атрибутном доступе используйте интерфейс [IABACPolicy](../IABACPolicy/IABACPolicy.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IABACPolicies.Count.htm)
		 Свойство Count
		 возвращает количество политик, содержащихся в атрибутном
		 доступе.


		 ![](../../Property_Image.gif)
		 [Item](IABACPolicies.Item.htm)
		 Свойство Item возвращает
		 политику из атрибутного доступа по индексу.

		 Имя метода
		 Краткое описание
