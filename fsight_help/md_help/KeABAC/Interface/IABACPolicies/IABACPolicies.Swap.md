# IABACPolicies.Swap


## Синтаксис


Swap(Index1: Integer; Index2: Integer): Boolean;


## Параметры


Index1. Индекс позиции политики
 в атрибутном доступе, которую необходимо переместить;


Index2. Индекс позиции политики,
 вместо которой необходимо переместить указанную политику в атрибутном
 доступе.


Примечание.
 Политики с указанными индексами будут перемещены между собой.


## Описание


Метод Swap меняет местами положение
 заданных политик в атрибутном доступе по индексу.
