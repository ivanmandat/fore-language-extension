# IABACPolicies.SwapByKey


## Синтаксис


SwapByKey(Id1: Integer; Id2: Integer): Boolean;


## Параметры


Id1. Ключ политики в атрибутном
 доступе, которую необходимо переместить;


Id2. Ключ политики, вместо
 которой необходимо переместить указанную политику в атрибутном доступе.


Примечание.
 Политики с указанными ключами будут перемещены между собой.


## Описание


Метод SwapByKey меняет местами
 положение заданных политик в атрибутном доступе по ключу.


## Комментарии


Для определения ключа политики используйте свойство [IABACPolicy.Key](../IABACPolicy/IABACPolicy.Key.htm).
