# IABACPolicies.MoveByKey


## Синтаксис


MoveByKey(FromKey: Integer; ToKey: Integer): Boolean;


## Параметры


FromKey. Ключ политики в атрибутном
 доступе, которую необходимо переместить;


ToKey. Ключ политики, на место
 которой необходимо переместить указанную политику в атрибутном доступе.


## Описание


Метод MoveByKey осуществляет
 перемещение политики среди существующих политик в атрибутном доступе по
 ключу.


## Комментарии


Для определения ключа политики используйте свойство [IABACPolicy.Key](../IABACPolicy/IABACPolicy.Key.htm).
