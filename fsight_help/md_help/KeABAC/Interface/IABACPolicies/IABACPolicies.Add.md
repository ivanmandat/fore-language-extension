# IABACPolicies.Add


## Синтаксис


Add: [IABACPolicy](../IABACPolicy/IABACPolicy.htm);


## Описание


Метод Add осуществляет добавление
 политики в структуру атрибутного доступа.


## Пример


Использование метода приведено в примере для [IABACTarget.SecondAttrId](../IABACTarget/IABACTarget.SecondAttrId.htm)
 и [IABACRootPolicyObject.Policies](../IABACRootPolicyObject/IABACRootPolicyObject.Policies.htm).
