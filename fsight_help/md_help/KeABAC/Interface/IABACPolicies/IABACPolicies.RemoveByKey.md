# IABACPolicies.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ
 политики.


## Описание


Метод RemoveByKey осуществляет
 удаление политики в атрибутном доступе по ключу.


## Комментарии


Для определения ключа политики используйте свойство [IABACPolicy.Key](../IABACPolicy/IABACPolicy.Key.htm).
