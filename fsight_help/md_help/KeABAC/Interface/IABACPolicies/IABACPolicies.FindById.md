# IABACPolicies.FindById


## Синтаксис


FindById(Id: String): [IABACPolicy](../IABACPolicy/IABACPolicy.htm);


## Параметры


Id. Идентификатор политики.


## Описание


Метод FindById осуществляет
 поиск политики в атрибутном доступе по идентификатору.


## Комментарии


Для определения идентификатора политики используйте свойство [IABACPolicy.Id](../IABACPolicy/IABACPolicy.Id.htm).
