# IABACPolicies.SwapById


## Синтаксис


SwapById(Id1: String; Id2: String): Boolean;


## Параметры


Id1. Идентификатор политики
 в атрибутном доступе, которую необходимо переместить;


Id2. Идентификатор политики,
 вместо которой необходимо переместить указанную политику в атрибутном
 доступе.


Примечание.
 Политики с указанными идентификаторами будут перемещены между собой.


## Описание


Метод SwapById меняет местами
 положение заданных политик в атрибутном доступе по идентификатору.


## Комментарии


Для определения идентификатора политики используйте свойство [IABACPolicy.Id](../IABACPolicy/IABACPolicy.Id.htm).
