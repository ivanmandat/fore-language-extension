# IABACAttributes.Item


## Синтаксис


Item(Index: Integer): [IABACAttribute](../IABACAttribute/IABACAttribute.htm);


## Параметры


Index. Индекс атрибута в коллекции.


## Описание


Свойство Item возвращает атрибут
 из коллекции по индексу.


## Комментарии


Список всех атрибутов содержится в [цели](../../../admin/04_SecurityPolicy/Target.htm)
 и [дополнительных
 условиях](../../../admin/04_SecurityPolicy/Creating_conditions.htm) проверки доступа.
