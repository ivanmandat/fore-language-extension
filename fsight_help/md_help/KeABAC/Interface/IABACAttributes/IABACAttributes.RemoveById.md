# IABACAttributes.RemoveById


## Синтаксис


RemoveById(Id: String): Boolean;


## Параметры


Id. Идентификатор атрибута.


## Описание


Метод RemoveById осуществляет
 удаление атрибута из коллекции по идентификатору.


## Комментарии


Для определения идентификатора атрибута используйте свойство [IABACAttribute.Id](../IABACAttribute/IABACAttribute.Id.htm).
