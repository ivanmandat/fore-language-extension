# IABACAttributes.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс атрибута.


## Описание


Метод Remove осуществляет удаление
 атрибута из коллекции по индексу.


## Комментарии


Метод возвращает значение True,
 если удаление прошло успешно.
