# IABACAttributes.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ
 атрибута.


## Описание


Метод RemoveByKey осуществляет
 удаление атрибута из коллекции по ключу.


## Комментарии


Для определения ключа атрибута используйте свойство [IABACAttribute.Key](../IABACAttribute/IABACAttribute.Key.htm).
