# IABACAttributes.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IABACAttribute](../IABACAttribute/IABACAttribute.htm);


## Параметры


Key. Ключ атрибута.


## Описание


Метод FindByKey осуществляет
 поиск атрибута в коллекции по ключу.


## Комментарии


Для определения ключа атрибута используйте свойство [IABACAttribute.Key](../IABACAttribute/IABACAttribute.Key.htm).
