# IABACAttributes.Add


## Синтаксис


Add: [IABACAttribute](../IABACAttribute/IABACAttribute.htm);


## Описание


Метод Add осуществляет добавление
 атрибута в коллекцию.


## Комментарии


Данный метод актуален при добавлении нового атрибута пользователям,
 группам пользователей, классам объектов, отдельным типам объектов.
