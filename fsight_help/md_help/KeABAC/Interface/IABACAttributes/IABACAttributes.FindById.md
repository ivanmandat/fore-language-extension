# IABACAttributes.FindById


## Синтаксис


FindById(Id: String): [IABACAttribute](../IABACAttribute/IABACAttribute.htm);


## Параметры


Id. Идентификатор атрибута.


## Описание


Метод FindById осуществляет
 поиск атрибута в коллекции по идентификатору.


## Комментарии


Для определения идентификатора атрибута используйте свойство [IABACAttribute.Id](../IABACAttribute/IABACAttribute.Id.htm).
