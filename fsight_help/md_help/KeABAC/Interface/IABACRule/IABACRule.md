# IABACRule


## Описание


Интерфейс IABACRule описывает
 свойства [правила](Admin.chm::/04_SecurityPolicy/Admin_AttributeAccess.htm#rule).


## Комментарии


Определение свойств правила осуществляется после определения [политики](Admin.chm::/04_SecurityPolicy/Admin_AttributeAccess.htm#policy)
 в структуре атрибутного доступа с помощью интерфейса [IABACPolicy](../IABACPolicy/IABACPolicy.htm).
