# IABACPolicy


## Описание


Интерфейс IABACPolicy описывает
 свойства [политики](Admin.chm::/04_SecurityPolicy/Admin_AttributeAccess.htm#policy).


## Комментарии


Определение свойств политики осуществляется после определения [набора
 политик](Admin.chm::/04_SecurityPolicy/Admin_AttributeAccess.htm#policies_set) с помощью интерфейса [IABACPolicySet](../IABACPolicySet/IABACPolicySet.htm).
