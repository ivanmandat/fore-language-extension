# IABACAttributeInstances


## Описание


Интерфейс IABACAttributeInstances
 предназначен для работы с атрибутами конкретного пользователя, группы
 пользователей, класса объектов, отдельного типа объектов.


## Комментарии


Для определения и получения свойств атрибутов используйте интерфейсы
 [IABACAttribute](../IABACAttribute/IABACAttribute.htm), [IABACAttributeInstance](../IABACAttributeInstance/IABACAttributeInstance.htm).


Для работы с существующей коллекцией атрибутов используйте интерфейс
 [IABACAttributes](../IABACAttributes/IABACAttributes.htm).
