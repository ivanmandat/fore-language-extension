# IABACAttributeInstances.FindById


## Синтаксис


FindById(Id: String): [IABACAttributeInstance](../IABACAttributeInstance/IABACAttributeInstance.htm);


## Параметры


Id. Идентификатор атрибута.


## Описание


Метод FindById осуществляет
 поиск атрибута конкретного субъекта или объекта по идентификатору.


## Комментарии


Для получения идентификатора атрибута используйте свойство [IABACAttributeInstance.Id](../IABACAttributeInstance/IABACAttributeInstance.Id.htm).
