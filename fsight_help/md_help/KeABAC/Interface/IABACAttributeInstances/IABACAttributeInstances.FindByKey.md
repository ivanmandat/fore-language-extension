# IABACAttributeInstances.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IABACAttributeInstance](../IABACAttributeInstance/IABACAttributeInstance.htm);


## Параметры


Key. Ключ атрибута.


## Описание


Метод FindByKey осуществляет
 поиск атрибута конкретного субъекта или объекта по ключу.


## Комментарии


Для получения ключа атрибута используйте свойство [IABACAttributeInstance.Key](../IABACAttributeInstance/IABACAttributeInstance.Key.htm).
