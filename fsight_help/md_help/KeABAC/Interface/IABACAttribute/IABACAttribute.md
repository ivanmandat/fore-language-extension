# IABACAttribute


## Описание


Интерфейс IABACAttribute описывает
 свойства [пользовательских
 атрибутов](Admin.chm::/04_SecurityPolicy/Creating_attributes.htm) пользователей, групп пользователей, классов
 объектов, отдельных типов объектов.


## Комментарии


Используется при добавлении или редактировании свойств атрибутов.
