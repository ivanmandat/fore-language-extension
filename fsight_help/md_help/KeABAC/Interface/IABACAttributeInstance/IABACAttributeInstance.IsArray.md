# IABACAttributeInstance.IsArray


## Синтаксис


IsArray: Boolean;


## Описание


Свойство IsArray возвращает
 признак множественного значения атрибута.


## Комментарии


Данное свойство актуально при редактировании значения существующего
 атрибута.


Для изменения значения атрибута используйте свойство [IABACAttributeInstance.Value](IABACAttributeInstance.Value.htm).
