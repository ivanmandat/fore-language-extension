# IABACAttributeInstance


## Описание


Интерфейс IABACAttributeInstance предназначен
 для работы с существующим атрибутом пользователей, групп пользователей,
 классов объектов, отдельных типов объектов.


## Комментарии


Используется для получения свойств и изменения значения существующего
 атрибута.
