# IABACPolicySet


## Описание


Интерфейс IABACPolicySet описывает
 свойства [набора
 политик](Admin.chm::/04_SecurityPolicy/Admin_AttributeAccess.htm#policies_set).


## Иерархия наследования


[IABACEvaluatedObject](../IABACEvaluatedObject/IABACEvaluatedObject.htm)


IABACPolicySet


## Комментарии


Определение свойств набора политик осуществляется после определения
 [атрибутного
 доступа](Admin.chm::/04_SecurityPolicy/Admin_AttributeAccess.htm) с помощью интерфейса [IABACRootPolicyObject](../IABACRootPolicyObject/IABACRootPolicyObject.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Active](IABACPolicySet.Active.htm)
		 Свойство Active определяет
		 признак активности набора политик в структуре атрибутного доступа.


		 ![](../../Property_Image.gif)
		 [CombineAlgorithm](IABACPolicySet.CombineAlgorithm.htm)
		 Свойство CombineAlgorithm
		 определяет [алгоритм
		 комбинации](Admin.chm::/04_SecurityPolicy/Admin_AttributeAccess.htm#properties) набора политик.


		 ![](../../Property_Image.gif)
		 [Description](IABACPolicySet.Description.htm)
		 Свойство Description
		 определяет описание набора политик.


		 ![](../../Property_Image.gif)
		 [Id](IABACPolicySet.Id.htm)
		 Свойство Id определяет
		 идентификатор набора политик.


		 ![](../../Property_Image.gif)
		 [Key](IABACPolicySet.Key.htm)
		 Свойство Key возвращает
		 ключ набора политик.


		 ![](../../Property_Image.gif)
		 [Name](IABACPolicySet.Name.htm)
		 Свойство Name определяет
		 наименование набора политик.


		 ![](../../Property_Image.gif)
		 [Policies](IABACPolicySet.Policies.htm)
		 Свойство Policies
		 предназначено для обращения к коллекции политик.


		 ![](../../Property_Image.gif)
		 [PolicySets](IABACPolicySet.PolicySets.htm)
		 Свойство PolicySets
		 предназначено для работы с коллекцией наборов политик.


		 ![](../../Property_Image.gif)

		 Свойство Target предназначено
