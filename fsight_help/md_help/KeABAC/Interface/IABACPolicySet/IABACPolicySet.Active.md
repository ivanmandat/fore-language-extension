# IABACPolicySet.Active


## Синтаксис


Active: Boolean;


## Описание


Свойство Active определяет признак
 активности набора политик в структуре атрибутного доступа.


## Комментарии


Данное свойство актуально при определении структуры атрибутного доступа.
