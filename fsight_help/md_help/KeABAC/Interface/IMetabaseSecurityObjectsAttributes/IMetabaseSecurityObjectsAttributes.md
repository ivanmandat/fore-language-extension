# IMetabaseSecurityObjectsAttributes


## Описание


Интерфейс IMetabaseSecurityObjectsAttributes
 предоставляет обращение к классам и отдельным типам объектов, которые
 содержатся в разделе «[Классы
 объектов](Admin.chm::/04_SecurityPolicy/Admin_Object_Classes.htm)» менеджера безопасности.
