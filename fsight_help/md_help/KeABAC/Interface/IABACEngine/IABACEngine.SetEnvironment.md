# IABACEngine.SetEnvironment


## Синтаксис


SetEnvironment(EnvAttrs: [IABACAttributeInstances](../IABACAttributeInstances/IABACAttributeInstances.htm));


## Параметры


EnvAttrs. Коллекция атрибутов
 среды окружения.


## Описание


Метод SetEnvironment задает
 коллекцию [атрибутов
 среды окружения](Admin.chm::/04_SecurityPolicy/Creating_conditions.htm#environment).


## Комментарии


Метод используется с методом [IABACEngine.Evaluate](IABACEngine.Evaluate.htm)
 для вычисления доступа субъекта к объекту.


## Пример


Использование метода приведено в примере для [IABACEngine.Evaluate](IABACEngine.Evaluate.htm).
