## С чего начать


		 Сборка Report
		 предназначена для работы с объектом репозитория «Отчеты».


		Изучив данный раздел, Вы сможете с применением языка Fore:


			- [выбрать
			 источник данных отчета](KeReport_Programming_Source.htm);


			- [настроить
			 срез данных и область данных](KeReport_Programming_DataArea.htm);


			- [настроить
			 стилевое оформление области данных](KeReport_Programming_FormatConditions.htm);


			- [настроить
			 элементы управления](KeReport_Programming_Controls.htm);


			- [настроить
			 отображаемые на листе отчета объекты](KeReport_Programming_Objects.htm);


			- [настроить
			 параметры страницы для печати и экспорта отчета](KeReport_Programming_PageSettings.htm).


		Ознакомиться с основными принципами работы с регламентными отчетами
		 можно в разделе «[Общие принципы
		 программирования с использованием сборки Report](KeReport_Programming.htm)».


		Более подробное описание возможностей блока приведено в разделе
		 «[Введение в сборку Report](../KeReport_Introduction.htm)».


		Описание ресурсов сборки представлено в следующих подразделах:


			- [Интерфейсы
			 сборки Report](../Interface/KeReport_Interface.htm)


			- [Перечисления сборки
			 Report](../Enums/KeReport_Enums.htm)


			- [Классы сборки
			 Report](../Class/KeReport_Classes.htm)


			- [Делегаты сборки Report](../Delegate/Delegate.htm)


			- [Примеры](../Samples/KeReport_Example.htm)


		 Перед изучением сборки Report рекомендуется ознакомиться
		 с разделами: «[Среда разработки](UiDevEnv.chm::/01_Development_Environment/01_Purpose_of_the_constructor/Purpose_of_the_constructor.htm)»
		 и «[Руководство
		 по языку Fore](Fore.chm::/Fore_Title.htm)».


		Непосредственно программирование с использованием Fore
		 осуществляется в рамках объекта репозитория «[Отчёты](UIReport.chm::/UiReport_purpose.htm)».
		 Попробуйте работать с регламентными отчетами, используя сборку
		 Report.
