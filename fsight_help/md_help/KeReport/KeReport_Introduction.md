# Введение в сборку Report


В данном разделе представлено описание сборки Report,
 содержащей средства для работы на языке Fore
 с объектом репозитория «Отчеты».
 Подробное описание смотрите в разделах:


	- [Общие принципы программирования
	 с использованием сборки Report](Intro/KeReport_Programming.htm);


	- [Иерархия сборки Report](Intro/KeReport_Hierarchy.htm).


В интерфейсах сборки реализованы все необходимые средства для работы
 с регламентными отчетами.


Основным интерфейсом сборки Report
 является [IPrxReport](Interface/IPrxReport/IPrxReport.htm).
