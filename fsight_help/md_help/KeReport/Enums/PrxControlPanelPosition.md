# PrxControlPanelPosition


## Описание


Перечисление PrxControlPanelPosition
 используется для определения положения панели элементов управления.


Используется следующим свойством:


	- [IPrxReportOptions.ControlPanelPosition](../Interface/IPrxReportOptions/IPrxReportOptions.ControlPanelPosition.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Панель «Элементы управления» не отображается.


		 1
		 Left. Панель размещается
		 вертикально по левой границе отчета.


		 2
		 Top. Панель размещается
		 горизонтально по верхней границе отчета.


		 3
		 Right. Панель размещается
		 вертикально по правой границе отчета.


		 4
		 Bottom. Панель размещается
		 горизонтально по нижней границе отчета.
