# PrxDimensionSortKind


## Описание


Перечисление PrxDimensionSortKind содержит виды сортировки.


Используется следующим свойством:


-
[IPrxDataIslandDimensionSort.Kind](../Interface/IPrxDataIslandDimensionSort/IPrxDataIslandDimensionSort.Kind.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 ByElement. По элементу измерения.


 1
 ByColumnRow. По строке/столбцу.


 2
 ByName. По имени элемента.


 3
 ByAttribute. По значению атрибута элемента.
