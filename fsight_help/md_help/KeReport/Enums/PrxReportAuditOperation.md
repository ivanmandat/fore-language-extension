# PrxReportAuditOperation


## Описание


Перечисление PrxReportAuditOperation
 содержит типы операций, для которых можно добавить комментарий при занесении
 в протокол доступа. Используется свойством [IUiPrxReportAuditOperationEventArgs.Operation](../Interface/IUiPrxReportAuditOperationEventArgs/IUiPrxReportAuditOperationEventArgs.Operation.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Unknown. Не определено.


		 1
		 Print. Печать.


		 2
		 Export. Экспорт.
