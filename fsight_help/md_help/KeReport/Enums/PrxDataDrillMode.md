# PrxDataDrillMode


## Описание


Перечисление PrxDataDrillMode
 содержит способы модификации отметки в измерении.


Используется следующим свойством:


	- [IPrxDimensionDrill.Mode](../Interface/IPrxDimensionDrill/IPrxDimensionDrill.Mode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Отметка измерения
		 изменена не будет.


		 1
		 Replace. Вся имеющаяся
		 отметка в измерении будет снята и выделен только элемент, соответствующий
		 элементу, гиперссылка которого была нажата.


		 2
		 Invert. Будет инвертирована
		 отметка элемента, соответствующего элементу, гиперссылка которого
		 была нажата.


		 3
		 Add. В измерении будет
		 отмечен элемент, соответствующий элементу, гиперссылка которого
		 была нажата. Отметка будет произведена, только если ранее по данному
		 элементу отметка отсутствовала.
