# PrxOpenObjectParamType


## Описание


Перечисление PrxOpenObjectParamType
 содержит типы управления параметром открываемого объекта.


Используется следующими свойствами и методами:


	- [IPrxOpenObjectParam.Type](../Interface/IPrxOpenObjectParam/IPrxOpenObjectParam.Type.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Не передавать
		 значение в параметр.


		 1
		 Constant. Передавать
		 в параметр константное значение.


		 2
		 ReportParam. Передавать
		 в параметр открываемого объекта значение параметра текущего объекта.


		 3
		 DimensionAttribute.
		 Передавать в параметр открываемого объекта значение заданного
		 атрибута измерения.
