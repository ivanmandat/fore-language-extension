# PrxChartSourceType


## Описание


Перечисление PrxChartSourceType
 содержит типы источников данных для диаграммы.


Используется следующим свойством:


	- [IPrxChart.SourceType](../Interface/IPrxChart/IPrxChart.SourceType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Sheet. В качестве источника
		 данных для диаграммы используется область на листе отчета.


		 1
		 Slice. В качестве источника
		 данных для диаграммы используется срез данных.
		Примечание.
		 В случае использования данного источника определите свойство [IPrxChart.Slice](../Interface/IPrxChart/IPrxChart.Slice.htm).


		 2
		 EaxChart. В качестве
		 источника данных для диаграммы используется аналитическая область
		 данных.
