# PrxAdjustMode


## Описание


Перечисление PrxAdjustMode содержит способы подгонки размеров ячейки.


Используется следующими свойствами и методами:


-
[IPrxDataIsland.AdjustMode](../Interface/IPrxDataIsland/IPrxDataIsland.AdjustMode.htm)


-
[IPrxTableIsland.AdjustMode](../Interface/IPrxTableIsland/IPrxTableIsland.AdjustMode.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Default. По умолчанию.


 1
 ColumnsThenRows. Сначала столбцы, затем строки.


 2
 RowsThenColumns. Сначала строки, затем столбцы.


 3
 None. Нет подгонки.


 4
 Columns. Только столбцы: ширина каждого столбца выравнивается соответственно самой длинной строке, содержащейся в ячейке этого столбца.


 5
 Rows. Только строки: высота каждой строки выравнивается в соответствии с самой высокой ячейкой, содержащейся в данной строке.
