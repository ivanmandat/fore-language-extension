# PrxDimensionSortMethod


## Описание


Перечисление PrxDimensionSortMethod
 содержит методы сортировки.


Используется следующим свойством:


	- [IPrxDataIslandDimensionSort.Method](../Interface/IPrxDataIslandDimensionSort/IPrxDataIslandDimensionSort.Method.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 WithinBranchesOnly.
		 Внутри веток иерархии.


		 1
		 ByHierarchyLevel. По
		 уровням иерархии.


		 2
		 Destructive. С нарушением
		 иерархии.
		Примечание.
		 Под нарушением иерархии понимается нарушение иерархии текущего
		 измерения, а не всей совокупности измерений среза.
