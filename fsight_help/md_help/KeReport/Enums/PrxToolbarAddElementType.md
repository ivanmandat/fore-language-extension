# PrxToolbarAddElementType


## Описание


Перечисление PrxToolbarAddElementType
 содержит типы элементов, которые могут быть созданы на панели инструментов.


Используется следующими свойствами и методами:


	- [IPrxToolbarElements.Add](../Interface/IPrxToolbarElements/IPrxToolbarElements.Add.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 Separator. Разделитель.


		 2
		 UserGroup. Пользовательская
		 группа кнопок.
