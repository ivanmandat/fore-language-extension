# PrxNamingPlacement


## Описание


Перечисление PrxNamingPlacement
 содержит варианты расположения нумерации строк/столбцов.


Используется следующим свойством:


	- [IPrxDataIslandNaming.Placement](../Interface/IPrxDataIslandNaming/IPrxDataIslandNaming.Placement.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 AfterDimension.
		 После основного наименования измерения.


		 1
		 BeforeDimension.
		 Перед основным наименованием измерения.


		 2
		 AfterAll. В конце всего
		 заголовка.


		 3
		 BeforeAll. В начале
		 всего заголовка.
