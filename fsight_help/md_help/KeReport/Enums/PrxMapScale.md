# PrxMapScale


## Описание


Перечисление PrxMapScale содержит типы шкал.


Используется следующим методом:


-
[IPrxMapScales.Add](../Interface/IPrxMapScales/IPrxMapScales.Add.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Нет.


 1
 Color. Цветовая шкала.


 2
 Numeric. Числовая шкала.


 3
 Fill. Штриховая шкала.
