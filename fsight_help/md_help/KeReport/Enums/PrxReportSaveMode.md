# PrxReportSaveMode


## Описание


Перечисление PrxReportSaveMode содержит варианты (режимы) сохранения регламентного отчета.


Используется следующими свойствами и методами:


-
[IPrxReport.SaveToFile](../Interface/IPrxReport/IPrxReport.SaveToFile.htm);


-
[IPrxReport.SaveToStream](../Interface/IPrxReport/IPrxReport.SaveToStream.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Static. Статический отчет.


 1
 DefinitionOnly. Только определения срезов и источников данных.


 2
 Full. Полностью срезы с данными.
