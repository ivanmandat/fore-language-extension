# PrxDimensionSortFilter


## Описание


Перечисление PrxDimensionSortFilter содержит варианты поведения при наличии пустых и нечисловых значений.


Используется следующим свойством:


-
[IPrxDataIslandDimensionSort.Filter](../Interface/IPrxDataIslandDimensionSort/IPrxDataIslandDimensionSort.Filter.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Empty. Только пустые элементы не участвуют в сортировке. Нечисловые элементы помещаются в начало сортируемой области.


 1
 NonNumerical. Нечисловые элементы не участвуют в сортировке и помещаются в конец сортируемой области.
