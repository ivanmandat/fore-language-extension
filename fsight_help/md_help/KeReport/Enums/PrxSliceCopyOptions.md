# PrxSliceCopyOptions


## Описание


Перечисление PrxSliceCopyOptions содержит параметры копирования среза.


Используется следующим методом:


-
[IPrxSlice.Copy](../Interface/IPrxSlice/IPrxSlice.Copy.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 All. Полностью


 1
 NoSelection. Без отметки.


 2
 NoSyncIndex. Без синхронизации.


 3
 NoSelectionAndSyncIndex. Без отметки и синхронизации.
