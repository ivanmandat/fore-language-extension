# PrxUserButtonType


## Описание


Перечисление PrxUserButtonType
 содержит типы действия, выполняемого при щелчке по пользовательской кнопке.


Используется следующими свойствами и методами:


	- [IPrxUserButton.Type](../Interface/IPrxUserButton/IPrxUserButton.Type.htm);


	- [IPrxUserButtons.Add](../Interface/IPrxUserButtons/IPrxUserButtons.Add.htm);


	- [IPrxUserButtons.AddByType](../Interface/IPrxUserButtons/IPrxUserButtons.AddByType.htm).
