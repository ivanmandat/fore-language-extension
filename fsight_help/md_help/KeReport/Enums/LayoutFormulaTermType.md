# LayoutFormulaTermType


## Описание


Перечисление LayoutFormulaTermType содержит типы используемого терма.


Используется следующим свойством:


-
[IPrxDataIslandDimensionLayoutFormulaParser.TermType](../Interface/IPrxDataIslandDimensionLayoutFormulaParser/IPrxDataIslandDimensionLayoutFormulaParser.TermType.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Тип терма не определен.


 1
 Value. Значение.


 2
 Node. Элемент размещения. Элемент определяется значением свойства [IPrxDataIslandDimensionLayoutFormulaParser.TermNodeKey](../Interface/IPrxDataIslandDimensionLayoutFormulaParser/IPrxDataIslandDimensionLayoutFormulaParser.TermNodeKey.htm).


 3
 Element. Элемент измерения. Элемент определяется значением свойства [IPrxDataIslandDimensionLayoutFormulaParser.TermElementID](../Interface/IPrxDataIslandDimensionLayoutFormulaParser/IPrxDataIslandDimensionLayoutFormulaParser.TermElementID.htm).


 4
 OverallTotal. Итоговое значение.


 5
 SubTotal. Промежуточный итог.
