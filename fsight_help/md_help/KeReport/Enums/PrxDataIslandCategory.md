# PrxDataIslandCategory


## Описание


Перечисление PrxDataIslandCategory
 содержит группы настроек, которые будут доступны в диалоге настройки области
 данных.


Используется следующими свойствами и методами:


	- [Команда
	 DataIslandSetup](UiLib.chm::/Interface/IUiCommandTarget/Command/DataIslandSetup.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. В диалоге настройки
		 параметров области данных будет только раздел по настройке базовых
		 параметров.


		 1
		 Filter. В диалоге настройки
		 параметров области данных будет группа настроек фильтрации.


		 2
		 Sort. В диалоге настройки
		 параметров области данных будет группа настроек сортировки.


		 4
		 Placement. В диалоге
		 настройки параметров области данных будет группа настроек размещения
		 элементов области данных.


		 8
		 Totals. В диалоге настройки
		 параметров области данных будет группа настроек итогов.


		 16
		 Drill. В диалоге настройки
		 параметров области данных будет группа настроек расшифровки.


		 32
		 Format. В диалоге настройки
		 параметров области данных будет группа настроек оформления элементов
		 области данных.


		 63
		 All. В диалоге настройки
		 параметров области данных будут все группы настроек.
