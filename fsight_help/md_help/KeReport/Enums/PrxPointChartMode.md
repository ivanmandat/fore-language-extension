# PrxPointChartMode


## Описание


Перечисление PrxPointChartMode
 содержит способы передачи данных для диаграмм точечного и пузырькового
 типов.


Используется следующими свойствами и методами:


	- [IPrxChart.PointChartMode](../Interface/IPrxChart/IPrxChart.PointChartMode.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 XYY. Значения по оси
		 Х одинаковые и содержатся
		 либо в верхней строке (при размещении рядов в строках), либо в
		 первом столбце (при размещении рядов в столбцах). В остальных
		 строках / столбцах содержатся значения рядов по оси Y.


		 1
		 XYXY. Значения
		 по осям X
		 и Y чередуются
		 в строках / столбцах (в зависимости от расположения рядов). Если
		 ряды расположены в строках, то чередование происходит сверху вниз,
		 если ряды расположены в столбцах - слева направо.


		 2
		 YXX. Значения по оси
		 Y одинаковые и содержатся
		 либо в верхней строке (при размещении рядов в строках), либо в
		 первом столбце (при размещении рядов в столбцах). В остальных
		 строках / столбцах содержатся значения рядов по оси X.


		 3
		 YXYX. Значения по осям
		 Y и X
		 будут чередуются в строках / столбцах (в зависимости от расположения
		 рядов). Если ряды расположены в строках, то чередование происходит
		 сверху вниз, если ряды расположены в столбцах - слева направо.


		 4
		 XYZYZ. Значения по
		 оси X одинаковые и содержатся
		 либо в верхней строке (при размещении рядов в строках), либо в
		 первом столбце (при размещении рядов в столбцах). Значения по
		 осям Y, Z
		 чередуются в строках/столбцах (в зависимости от расположения рядов).
		 Если ряды расположены в строках, то чередование происходит
		 сверху вниз, если ряды расположены в столбцах - слева направо.


Где Z
		 - размер пузырьков.


		 5
		 XYZXYZ. Значения по
		 осям X, Y,
		 Z чередуются в строках/столбцах
		 (в зависимости от расположения рядов). Если ряды расположены
		 в строках, то чередование происходит сверху вниз, если ряды расположены
		 в столбцах - слева направо.

Где Z
		 - размер пузырьков.


		 6
		 YXZXZ. Значения по
		 оси Y одинаковые и
		 содержатся либо в верхней строке (при размещении рядов в строках),
		 либо в первом столбце (при размещении рядов в столбцах). Значения
		 по осям X, Z
		 чередуются в строках/столбцах (в зависимости от расположения
		 рядов). Если ряды расположены в строках, то чередование
		 происходит сверху вниз, если ряды расположены в столбцах - слева
		 направо.

Где Z
		 - размер пузырьков.


		 7
		 YXZYXZ. Значения по
		 осям Y, X,
		 Z чередуются в строках/столбцах
		 (в зависимости от расположения рядов). Если ряды расположены
		 в строках, то чередование происходит сверху вниз, если ряды расположены
		 в столбцах - слева направо.

Где Z
		 - размер пузырьков.


## Комментарии


Размещение рядов в столбцах или строках определяется свойством [IPrxChart.SeriesInRows](../Interface/IPrxChart/IPrxChart.SeriesInRows.htm).


Типы перечисления XYZYZ, XYZXYZ, YXZXZ
 и YXZYXZ доступны только при выборе [пузырьковой](Chart.chm::/Enums/ChartType.htm)
 диаграммы. По умолчанию для пузырькового типа диаграмм используется способ
 передачи данных XYZYZ.
