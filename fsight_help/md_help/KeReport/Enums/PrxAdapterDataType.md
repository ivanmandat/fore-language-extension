# PrxAdapterDataType


## Описание


Перечисление PrxAdapterDataType
 содержит варианты расположения исходных данных для карты.


Используется следующим свойством:


	- [IPrxMapDataAdapter.AdapterDataType](../Interface/IPrxMapDataAdapter/IPrxMapDataAdapter.AdapterDataType.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 0


		 None. Источник данных
		 не определён.


		 1


		 Slice. Данные берутся
		 из среза.


		 2


		 Ranges. Данные берутся
		 из заданного диапазона значений.
