# PamAssemblyMethod


## Описание


Перечисление PamAssemblyMethod
 используется для определения способа подключения объекта среды разработки
 к регламентному отчёту, входящему в состав формы ввода.


Используется следующим методом:


	- IPrxAssemblies.AddEx.


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 ReportAndTabsheet.
		 По умолчанию. Объект среды разработки подключён к регламентному
		 отчёту и таблице.


		 1
		 OnlyReport. Объект
		 среды разработки подключён только к регламентному отчёту.


## Комментарии


Если при построении формы ввода не планируется [обработка
 событий таблицы](KePivot.chm::/Intro/Hierarchy/Hierarchy_Events.htm), то используйте значение OnlyReport.
 За счёт данного способа подключения объекта среды разработки к регламентному
 отчёту сокращается время открытия формы ввода.
