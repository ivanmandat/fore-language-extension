# PrxToolbarSystemButtonType


## Описание


Перечисление PrxToolbarSystemButtonType
 содержит типы системных кнопок, которые могут отображаться на панели инструментов.


Используется следующими свойствами и методами:


	- [IPrxSystemButtonElement.SystemButtonType](../Interface/IPrxSystemButtonElement/IPrxSystemButtonElement.SystemButtonType.htm).
