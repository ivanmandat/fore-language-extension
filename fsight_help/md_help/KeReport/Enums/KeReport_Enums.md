# Перечисления сборки Report


В рамках сборки Report существуют
 следующие перечисления:


		 Перечисление
		 Краткое описание


		 ![](../Enums.gif)
		 [LayoutFormulaTermType](LayoutFormulaTermType.htm)
		 Перечисление LayoutFormulaTermType
		 содержит типы используемого терма.


		 ![](../Enums.gif)
		 [PamAssemblyMethod](PamAssemblyMethod.htm)
		 Перечисление PamAssemblyMethod
		 используется для определения способа подключения объекта среды
		 разработки к регламентному отчёту, входящему в состав формы ввода.


		 ![](../Enums.gif)
		 [PrxAdapterDataType](PrxAdapterDataType.htm)
		 Перечисление PrxAdapterDataType
		 содержит варианты источников данных для карты.


		 ![](../Enums.gif)
		 [PrxAdjustMode](PrxAdjustMode.htm)
		 Перечисление PrxAdjustMode
		 содержит способы подгонки размеров ячейки.


		 ![](../Enums.gif)
		 [PrxChartSourceType](PrxChartSourceType.htm)
		 Перечисление PrxChartSourceType
		 содержит типы источников данных для диаграммы.


		 ![](../Enums.gif)
		 [PrxControlPanelOrientation](PrxControlPanelOrientation.htm)
		 Перечисление PrxControlPanelOrientation
		 содержит варианты расположения элементов управления в компоненте
		 среды разработки [ReportControlPanel](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/ReportControlPanel.htm).


		 ![](../Enums.gif)
		 [PrxControlPanelPosition](PrxControlPanelPosition.htm)
		 Перечисление PrxControlPanelPosition
		 используется для определения положения панели элементов управления.


		 ![](../Enums.gif)
		 [PrxDataAdapterConsumer](PrxDataAdapterConsumer.htm)
		 Перечисление PrxDataAdapterConsumer
		 содержит типы объектов, которые использует источник данных.


		 ![](../Enums.gif)
		 [PrxDataDrillMode](PrxDataDrillMode.htm)
		 Перечисление PrxDataDrillMode
		 содержит способы модификации отметки в измерении.


		 ![](../Enums.gif)
		 [PrxDataIslandCategory](PrxDataIslandCategory.htm)
		 Перечисление PrxDataIslandCategory
		 содержит группы настроек, которые будут доступны в диалоге настройки
		 области данных.


		 ![](../Enums.gif)
		 [PrxDataIslandHeaderBehaviour](PrxDataIslandHeaderBehaviour.htm)
		 Перечисление PrxDataIslandHeaderBehaviour
		 содержит варианты поведения строк/столбцов при увеличении области
		 данных.


		 ![](../Enums.gif)
		 [PrxDataIslandHeaderPlacement](PrxDataIslandHeaderPlacement.htm)
		 Перечисление PrxDataIslandHeaderPlacement
		 содержит варианты размещения строк/столбцов.


		 ![](../Enums.gif)
		 [PrxDataIslandPart](PrxDataIslandPart.htm)
		 Перечисление PrxDataIslandPart
		 содержит части области данных.


		 ![](../Enums.gif)
		 [PrxDimensionSortFilter](PrxDimensionSortFilter.htm)
		 Перечисление PrxDimensionSortFilter
		 содержит варианты поведения при наличии пустых и нечисловых значений.


		 ![](../Enums.gif)
		 [PrxDimensionSortKind](PrxDimensionSortKind.htm)
		 Перечисление PrxDimensionSortKind
		 содержит виды сортировки.


		 ![](../Enums.gif)
		 [PrxDimensionSortMethod](PrxDimensionSortMethod.htm)
		 Перечисление PrxDimensionSortMethod
		 содержит методы сортировки.


		 ![](../Enums.gif)
		 [PrxHeaderDisposition](PrxHeaderDisposition.htm)
		 Перечисление PrxHeaderDisposition
		 содержит варианты положений измерения в срезе.


		 ![](../Enums.gif)
		 [PrxMapAdapter](PrxMapAdapter.htm)
		 Перечисление PrxMapAdapter
		 содержит типы адаптера для карты.


		 ![](../Enums.gif)
		 [PrxMapScale](PrxMapScale.htm)
		 Перечисление PrxMapScale
		 содержит типы шкал.


		 ![](../Enums.gif)
		 [PrxNamingPlaceArrangement](PrxNamingPlaceArrangement.htm)
		 Перечисление PrxNamingPlaceArrangement
		 содержит варианты размещения нумерации строк/столбцов относительно
		 элементов измерений и уровней.


		 ![](../Enums.gif)
		 [PrxNamingPlacement](tagPrxNamingPlacement.htm)
		 Перечисление PrxNamingPlacement
		 содержит варианты расположения нумерации строк/столбцов.


		 ![](../Enums.gif)
		 [PrxNumberingType](PrxNumberingType.htm)
		 Перечисление PrxNumberingType
		 содержит типы нумерации строк/столбцов.


		 ![](../Enums.gif)
		 [PrxOpenObjectParamType](PrxOpenObjectParamType.htm)
		 Перечисление PrxOpenObjectParamType
		 содержит типы управления параметром открываемого объекта.


		 ![](../Enums.gif)
		 [PrxOpenObjectSelectionType](PrxOpenObjectSelectionType.htm)
		 Перечисление PrxOpenObjectSelectionType
		 содержит типы отметки, в соответствии с которой будут передаваться
		 значения атрибута измерения в параметр открываемого объекта.


		 ![](../Enums.gif)
		 [PrxPointChartMode](PrxPointChartMode.htm)
		 Перечисление PrxPointChartMode
		 содержит способы передачи данных для диаграмм точечного и пузырькового
		 типов.


		 ![](../Enums.gif)
		 [PrxReportAuditOperation](PrxReportAuditOperation.htm)
		 Перечисление PrxReportAuditOperation
		 содержит типы операций, для которых можно добавить комментарий
		 при занесении в протокол доступа.


		 ![](../Enums.gif)
		 [PrxReportSaveMode](PrxReportSaveMode.htm)
		 Перечисление PrxReportSaveMode
		 содержит варианты (режимы) сохранения регламентного отчета.


		 ![](../Enums.gif)
		 [PrxReportSubstitutionContext](PrxReportSubstitutionContext.htm)
		 Перечисление PrxReportSubstitutionContext
		 содержит варианты действий при которых может быть осуществлена
		 подмена регламентного отчета.


		 ![](../Enums.gif)
		 [PrxReportUserEvents](PrxReportUserEvents.htm)
		 Перечисление PrxReportUserEvents
		 содержит пользовательские события регламентного отчёта.


		 ![](../Enums.gif)
		 [PrxShapeType](PrxShapeType.htm)
		 Перечисление PrxShapeType
		 содержит типы автофигур.


		 ![](../Enums.gif)
		 [PrxSheetType](PrxSheetType.htm)
		 Перечисление PrxSheetType
		 содержит типы листов регламентного отчета.


		 ![](../Enums.gif)
		 [PrxSliceCopyOptions](PrxSliceCopyOptions.htm)
		 Перечисление PrxSliceCopyOptions
		 содержит параметры копирования среза.


		 ![](../Enums.gif)
		 [PrxToolbarAddElementType](PrxToolbarAddElementType.htm)
		 Перечисление PrxToolbarAddElementType
		 содержит типы элементов, которые могут быть созданы на панели
		 инструментов.


		 ![](../Enums.gif)
		 [PrxToolbarElementType](PrxToolbarElementType.htm)
		 Перечисление PrxToolbarElementType
		 содержит тип элемента панели инструментов.


		 ![](../Enums.gif)
		 [PrxToolbarSystemButtonType](PrxToolbarSystemButtonType.htm)
		 Перечисление PrxToolbarSystemButtonType
		 содержит типы системных кнопок, которые могут отображаться на
		 панели инструментов.


		 ![](../Enums.gif)
		 [PrxToolbarSystemGroupType](PrxToolbarSystemGroupType.htm)
		 Перечисление PrxToolbarSystemGroupType
		 содержит типы системных вкладок панели инструментов.


		 ![](../Enums.gif)
		 [PrxUserButtonType](PrxUserButtonType.htm)
		 Перечисление PrxUserButtonType
		 содержит типы действия, выполняемого при щелчке по пользовательской
		 кнопке.
