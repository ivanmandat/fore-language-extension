# PrxReportUserEvents


## Описание


Перечисление PrxReportUserEvents
 содержит пользовательские события регламентного отчёта.


Используется следующими методами:


	- [IPrxReport.LockPrxUserEvents](../Interface/IPrxReport/IPrxReport.LockPrxUserEvents.htm);


	- [IPrxReport.UnLockUserPrxEvents](../Interface/IPrxReport/IPrxReport.UnLockUserPrxEvents.htm).
