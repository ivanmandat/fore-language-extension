# PrxToolbarElementType


## Описание


Перечисление PrxToolbarElementType
 содержит тип элемента панели инструментов.


Используется следующими свойствами и методами:


	- [IPrxToolbarElement.Type](../Interface/IPrxToolbarElement/IPrxToolbarElement.Type.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 Separator. Разделитель.


		 2
		 UserGroup. Пользовательская
		 или системная группа кнопок.


		 3
		 UserButton. Пользовательская
		 кнопка.


		 4
		 SystemButton. Системная
		 кнопка.
