# PrxNamingPlaceArrangement


## Описание


Перечисление PrxNamingPlaceArrangement
 содержит варианты размещения нумерации строк/столбцов относительно элементов
 измерений и уровней.


Используется следующим свойством:


	- [IPrxDataIslandNaming.PlaceArrangement](../Interface/IPrxDataIslandNaming/IPrxDataIslandNaming.PlaceArrangement.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 SingleRowColumn. В
		 одну строку/столбец.


		 1
		 ElementLike. Как элемент
		 (если элемент расположен в отдельном столбце, то нумерация так
		 же в отдельном столбце).
