# PrxDataIslandHeaderPlacement


## Описание


Перечисление PrxDataIslandHeaderPlacement содержит варианты размещения строк/столбцов.


Используется следующим свойством:


-
[IPrxDataIslandHeader.Placement](../Interface/IPrxDataIslandHeader/IPrxDataIslandHeader.Placement.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Simple. Простое размещение.


 1
 SkipDuble. Пропускать повторяющиеся.


 2
 Merge. Объединять ячейки.
