# PrxReportSubstitutionContext


## Описание


Перечисление PrxReportSubstitutionContext
 содержит варианты действий, при которых может быть осуществлена подмена
 регламентного отчета.


Используется следующими свойствами и методами:


	- [IUiPrxReportBaseSubstitutionEventArgs.Context](../Interface/IUiPrxReportBaseSubstitutionEventArgs/IUiPrxReportBaseSubstitutionEventArgs.Context.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Preview. Подмена при
		 предварительном просмотре.


		 1
		 Print. Подмена при
		 печати.


		 2
		 Export. Подмена при
		 экспорте в файл.


Примечание.
 Особенности использования значений Print
 и Export приведены в описании
 свойства [IUiPrxReportBaseSubstitutionEventArgs.Context](../Interface/IUiPrxReportBaseSubstitutionEventArgs/IUiPrxReportBaseSubstitutionEventArgs.Context.htm).
