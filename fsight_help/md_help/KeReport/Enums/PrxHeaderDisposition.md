# PrxHeaderDisposition


## Описание


Перечисление PrxHeaderDisposition содержит варианты положений измерения в срезе.


Используется следующими свойствами и методами:


-
[IPrxSliceDimension.Disposition](../Interface/IPrxSliceDimension/IPrxSliceDimension.Disposition.htm)


-
[IPrxSliceDimension.MoveFirst](../Interface/IPrxSliceDimension/IPrxSliceDimension.MoveFirst.htm)


-
[IPrxSliceDimension.MoveLast](../Interface/IPrxSliceDimension/IPrxSliceDimension.MoveLast.htm)


-
[IPrxDataIslandHeader.Disposition](../Interface/IPrxDataIslandHeader/IPrxDataIslandHeader.Disposition.htm)
