# PrxOpenObjectSelectionType


## Описание


Перечисление PrxOpenObjectSelectionType
 содержит типы отметки, в соответствии с которой будут передаваться значения
 атрибута измерения в параметр открываемого объекта.


Используется следующими свойствами и методами:


	- [IPrxOpenObjectDimAttributeParamControl.SelectionType](../Interface/IPrxOpenObjectDimAttributeParamControl/IPrxOpenObjectDimAttributeParamControl.SelectionType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 All. Значения атрибутов
		 всех отмеченных элементов.


		 1
		 First. Значение атрибута
		 первого отмеченного элемента.
