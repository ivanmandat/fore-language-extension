# IPrxUserButtonActionMethod.JsFunction


## Синтаксис


JsFunction: String;


## Описание


Свойство JsFunction определяет
 JavaScript-метод, который должен срабатывать при запуске обработчика.


## Комментарии


Установленный JavaScript-метод будет выполняться только в веб-приложении
 «Форсайт. Аналитическая платформа».
 Требуемая сигнатура для ввода наименования:


javascript: <Наименование метода ([пользовательские
 параметры], callback, [args])>


Параметры:


	- пользовательские параметры. Требуемые пользовательские параметры.
	 Необязательный параметр;


	- callback. Функция обратного вызова. Обязательный параметр;


	- args. Аргументы функции. Необязательный параметр.


Примечание.
 Наличие угловых скобок при указании наименования функции с параметрами
 обязательно.


Если указаны JavaScript-метод и Fore-метод, то сначала выполняется Fore-метод,
 затем JavaScript-метод.
