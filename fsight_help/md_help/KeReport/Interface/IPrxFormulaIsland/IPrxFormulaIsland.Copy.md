# IPrxFormulaIsland.Copy


## Синтаксис


Copy([Sheet: [IPrxSheet](../IPrxSheet/IPrxSheet.htm)
 = Null;] [Range: [ITabRange](TabSheet.chm::/Interface/ITabRange/ITabRange.htm)
 = Null]): [IPrxFormulaIsland](IPrxFormulaIsland.htm);


## Параметры


Sheet - лист регламентного
 отчета, на котором будет размещена копия области формул.


Range - диапазон ячеек, в котором
 будет расположена копия области формул.


## Описание


Метод Copy создает копию области
 формул.
