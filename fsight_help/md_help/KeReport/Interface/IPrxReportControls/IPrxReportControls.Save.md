# IPrxReportControls.Save


## Синтаксис


Save;


## Описание


Метод Save осуществляет сохранение
 изменений, внесенных в коллекцию элементов управления регламентного отчета.
 Данный метод используется совместно с методом [Edit](IPrxReportControls.Edit.htm).
