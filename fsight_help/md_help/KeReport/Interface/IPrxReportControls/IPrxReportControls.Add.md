# IPrxReportControls.Add


## Синтаксис


Add(Sheet: [IPrxSheet](../IPrxSheet/IPrxSheet.htm)):
 [IPrxControl](../IPrxControl/IPrxControl.htm);


## Параметры


Sheet - лист регламентного
 отчета.


## Описание


Метод Add осуществляет добавление
 элемента управления регламентного отчета.


## Комментарии


Лист, управление которым будет осуществляться с помощью нового элемента
 управления, передается посредством параметра Sheet
 (если Null - глобальный для отчета).
