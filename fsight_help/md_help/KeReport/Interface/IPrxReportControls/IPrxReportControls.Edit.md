# IPrxReportControls.Edit


## Синтаксис


Edit: [IPrxReportControls](IPrxReportControls.htm);


## Описание


Метод Edit открывает на редактирование
 коллекцию элементов управления регламентного отчета. Используется, если
 необходимо произвести какие-либо действия над большим количеством элементов
 управления. Данный метод используется совместно с методом [Save](IPrxReportControls.Save.htm).
