# IPrxReportControls


## Описание


Интерфейс IPrxReportControls
 предназначен для работы с коллекцией элементов управления регламентного
 отчета. Можно работать непосредственно с коллекцией, а можно с редактируемой
 копией. В случае работы с редактируемой копией для применения изменений
 необходимо ее сохранить.
