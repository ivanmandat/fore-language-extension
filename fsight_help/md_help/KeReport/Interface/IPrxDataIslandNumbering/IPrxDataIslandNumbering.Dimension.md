# IPrxDataIslandNumbering.Dimension


## Синтаксис


Dimension: (Dimension: [IPrxDataIslandDimension](../IPrxDataIslandDimension/IPrxDataIslandDimension.htm)):[IPrxDataIslandNumberingLevel](../IPrxDataIslandNumberingLevel/IPrxDataIslandNumberingLevel.htm);


## Параметры


IPrxDataIslandDimension. Измерение
 среза данных, на котором построена область данных.


## Описание


Свойство Dimension возвращает
 настройки нумерации для измерения.


## Комментарии


В [настройках
 нумерации](UiReport.chm::/desktop/AreaData/Param/Order/UiReport_AreaData_Param_numbering.htm) невозможно задать формат числа и начальное
 значение, по умолчанию нумерация начинается с единицы.
