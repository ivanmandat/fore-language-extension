Интерфейс IPrxOldObject описывает
 устаревшие объекты регламентного отчёта.


## Иерархия наследования

 в ядре устаревших объектов, которые необходимы для совместимости с предыдущими
 версиями «Форсайт. Аналитическая платформа».
