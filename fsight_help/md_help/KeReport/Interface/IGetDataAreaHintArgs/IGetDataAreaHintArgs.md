# IGetDataAreaHintArgs


Сборка: Report;


## Описание


Интерфейс IGetDataAreaHintArgs
 предназначен для работы с аргументами события, наступающего при изменении
 метаданных содержимого ячейки с формулой.

## Комментарии


Строка формул используется для
 ввода или изменения значений/формул в ячейках.
