# IGetDataAreaHintArgs.Text


## Синтаксис


Text: String;


## Описание


Свойство Text определяет текст,
 отображающийся справа от строки формул.


## Комментарии


По умолчанию справа от строки формул отображается текст:


	- «Наименование источника данных!
	 наименования элементов, образующий ячейку», если ячейка, содержащая
	 формулу, образована элементами источника данных. Например:


Куб социально-экономических
 показателей!Значение|2001|Численность населения|Ленинградская область|Данные
 регионов


	- «пусто», если ячейка
	 не содержит формулу.
