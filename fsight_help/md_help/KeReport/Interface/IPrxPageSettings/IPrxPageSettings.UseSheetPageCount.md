# IPrxPageSettings.UseSheetPageCount


## Синтаксис


UseSheetPageCount: Boolean;


## Описание


Свойство UseSheetPageCount определяет,
 будет ли нумерация страниц листа начинаться заново независимо от нумерации
 страниц других листов.


## Комментарии


Если свойство имеет значение True,
 то нумерация страниц листа начинается заново; при значении False
 для страниц листа используется сквозная нумерация.


По умолчанию свойство имеет значение False.
