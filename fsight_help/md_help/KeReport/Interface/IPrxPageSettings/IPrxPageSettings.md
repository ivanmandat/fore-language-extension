# IPrxPageSettings


Сборка: Report;


## Описание


Интерфейс IPrxPageSettings содержит
 свойства и методы параметров печати страниц листа регламентного отчета.


## Иерархия наследования


           [ITabPageSettings](TabSheet.chm::/Interface/ITabPageSettings/ITabPageSettings.htm)


           IPrxPageSettings


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [StartPageNumber](IPrxPageSettings.StartPageNumber.htm)


		 Свойство StartPageNumber
		 определяет номер, с которого будет начинаться нумерация страниц
		 листа.


		 ![](../../Property_Image.gif)
		 [UseSheetPageCount](IPrxPageSettings.UseSheetPageCount.htm)


		 Свойство UseSheetPageCount
		 определяет, будет ли нумерация страниц листа начинаться заново
		 независимо от нумерации страниц других листов.
