# IPrxScene3D


## Описание


Интерфейс IPrxScene3D определяет
 параметры 3D-сцены, размещенной на листе регламентного отчета.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AutoAdjust](IPrxScene3D.AutoAdjust.htm)
		 Свойство AutoAdjust
		 определяет автоматическое изменение размеров указанных диапазонов
		 для рядов.


		 ![](../../Property_Image.gif)
		 [SerieAction](IPrxScene3D.SerieAction.htm)
		 Свойство SerieAction
		 определяет макрос, который будет использоваться для обработки
		 событий рядов сцены.


		 ![](../../Property_Image.gif)
		 [Series](IPrxScene3D.Series.htm)
		 Свойство Series определяет
		 параметры универсальной коллекции рядов сцены.


		 ![](../../Property_Image.gif)
		 [Sheet](IPrxScene3D.Sheet.htm)
		 Свойство Sheet определяет
		 лист регламентного отчета, с которого будут использованы исходные
		 данные для рядов.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [DoSerieClick](IPrxScene3D.DoSerieClick.htm)
		 Метод DoSerieClick
		 генерирует событие OnSeriesClick
		 (двойной щелчок мыши по ряду сцены).
