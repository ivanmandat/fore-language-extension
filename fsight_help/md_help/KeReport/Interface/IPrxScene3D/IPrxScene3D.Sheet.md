# IPrxScene3D.Sheet


## Синтаксис


Sheet: [IPrxSheet](../IPrxSheet/IPrxSheet.htm);


## Описание


Свойство Sheet определяет лист
 регламентного отчета, с которого будут использованы исходные данные для
 рядов. В случае если свойство имеет значение Null,
 то используется лист, на котором расположена 3D-сцена.
