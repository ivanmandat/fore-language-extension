# IPrxScene3D.DoSerieClick


## Синтаксис


DoSerieClick(Args: [IScene3DSeriesClickArgs](Chart.chm::/Interface/IScene3DSeriesClickArgs/IScene3DSeriesClickArgs.htm));


## Параметры


Args - параметр события, возвращает
 описание графических объектов, для которых наступило событие.


## Описание


Метод DoSerieClick генерирует
 событие OnSeriesDoubleClick (двойной щелчок мыши по ряду сцены).
