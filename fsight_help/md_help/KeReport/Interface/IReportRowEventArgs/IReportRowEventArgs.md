# IReportRowEventArgs


## Описание


Интерфейс IReportRowEventArgs
 содержит свойства аргумента событий, связанных с какой-либо строкой таблицы.


## Комментарии


Данный аргумент используется при обработке следующих событий:


	- [OnRowClick](../../Class/ReportBox/ReportBox.OnRowClick.htm)


	- [OnBeforeRowResize](../../Class/ReportBox/ReportBox.OnBeforeRowResize.htm)
