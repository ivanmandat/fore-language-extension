# IPrxScene3DSeries


## Описание


Интерфейс IPrxScene3DSeries
 определяет параметры универсальной коллекции рядов 3D-сцены в регламентном
 отчете.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPrxScene3DSeries.Count.htm)
		 Свойство Count возвращает
		 количество рядов сцены.


		 ![](../../Property_Image.gif)
		 [Item](IPrxScene3DSeries.Item.htm)
		 Свойство Item возвращает
		 ряд сцены по заданному индексу.
