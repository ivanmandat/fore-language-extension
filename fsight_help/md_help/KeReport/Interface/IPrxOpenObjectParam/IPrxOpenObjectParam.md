# IPrxOpenObjectParam


Сборка: Report;


## Описание


Интерфейс IPrxOpenObjectParam
 предназначен для настройки управления параметром объекта, открываемого
 по пользовательской кнопке.


## Иерархия наследования


           [INamedEntityModel](ForeSys.chm::/Interface/INamedEntityModel/INamedEntityModel.htm)


           IPrxOpenObjectParam


## Комментарии


Настройки управления параметром можно получить, используя свойства и
 методы интерфейса [IPrxOpenObjectParams](../IPrxOpenObjectParams/IPrxOpenObjectParams.htm).
