# IPrxOpenObjectParam.Control


## Синтаксис


Control: [IPrxOpenObjectParamControl](../IPrxOpenObjectParamControl/IPrxOpenObjectParamControl.htm);


## Описание


Свойство Control возвращает
 настройки передачи значения в параметр открываемого объекта.


## Комментарии


В зависимости от [типа управления
 параметром](IPrxOpenObjectParam.Type.htm), приведите значение свойства Control к одному из следующих
 интерфейсов:


	- [IPrxOpenObjectConstantParamControl](../IPrxOpenObjectConstantParamControl/IPrxOpenObjectConstantParamControl.htm)


	- [IPrxOpenObjectDimAttributeParamControl](../IPrxOpenObjectDimAttributeParamControl/IPrxOpenObjectDimAttributeParamControl.htm)


	- [IPrxOpenObjectReportParamControl](../IPrxOpenObjectReportParamControl/IPrxOpenObjectReportParamControl.htm)


## Пример

 [IPrxOpenObjectDimAttributeParamControl.DimId](../IPrxOpenObjectDimAttributeParamControl/IPrxOpenObjectDimAttributeParamControl.DimId.htm),
 [IPrxOpenObjectReportParamControl.Param](../IPrxOpenObjectReportParamControl/IPrxOpenObjectReportParamControl.Param.htm).
