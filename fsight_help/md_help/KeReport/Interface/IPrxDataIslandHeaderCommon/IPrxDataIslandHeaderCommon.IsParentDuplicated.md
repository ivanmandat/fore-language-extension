# IPrxDataIslandHeaderCommon.IsParentDuplicated


## Синтаксис


IsParentDuplicated: Boolean;


## Описание


Свойство IsParentDuplicated
 определяет, установлено ли повторение элементов уровня для строк/столбцов.


## Комментарии


Допустимые значения:


	- True. Установлено повторение
	 элементов уровня для строк/столбцов;


	- False. Не установлено
	 повторение элементов уровня для строк/столбцов.
