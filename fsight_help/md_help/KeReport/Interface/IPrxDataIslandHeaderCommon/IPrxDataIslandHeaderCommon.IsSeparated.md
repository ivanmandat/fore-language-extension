# IPrxDataIslandHeaderCommon.IsSeparated


## Синтаксис


IsSeparated: Boolean;


## Описание


Свойство IsSeparated определяет,
 установлено ли размещение элементов уровня в отдельной строке/столбце.


## Комментарии


Допустимые значения:


	- True. Установлено размещение
	 элементов уровня в отдельном строке/столбце;


	- False. Не установлено
	 размещение элементов уровня в отдельном строке/столбце.
