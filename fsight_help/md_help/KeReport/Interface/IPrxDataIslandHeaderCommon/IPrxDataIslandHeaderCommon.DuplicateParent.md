# IPrxDataIslandHeaderCommon.DuplicateParent


## Синтаксис


DuplicateParent: [TriState](ForeSys.chm::/Enums/TriState.htm);


## Описание


Свойство DuplicateParent определяет
 состояние опции «Повторять элемент уровня».


## Комментарии


Для определения, установлена ли опция «Повторять элемент уровня», используйте
 свойство [IPrxDataIslandHeaderCommon.IsParentDuplicated](IPrxDataIslandHeaderCommon.IsParentDuplicated.htm).
