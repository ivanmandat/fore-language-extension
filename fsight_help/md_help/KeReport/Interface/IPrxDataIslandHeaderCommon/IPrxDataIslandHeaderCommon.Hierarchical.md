# IPrxDataIslandHeaderCommon.Hierarchical

## Описание


Свойство Hierarchical определяет
 состояние опции «Раскрывающаяся иерархия».


## Комментарии


Для определения, установлена ли раскрывающаяся иерархия, используйте
 свойство [IPrxDataIslandHeaderCommon.IsHierarchical](IPrxDataIslandHeaderCommon.IsHierarchical.htm).
