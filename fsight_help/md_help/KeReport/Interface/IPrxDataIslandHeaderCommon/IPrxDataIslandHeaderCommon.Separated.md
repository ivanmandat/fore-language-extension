# IPrxDataIslandHeaderCommon.Separated


## Синтаксис


Separated: [TriState](ForeSys.chm::/Enums/TriState.htm);


## Описание


Свойство Separated определяет
 состояние опции «Разместить элементы уровня отдельно».


## Комментарии


Для определения, установлено ли размещение элементов уровня в отдельном
 строке/столбце, используйте свойство [IPrxDataIslandHeaderCommon.IsSeparated](IPrxDataIslandHeaderCommon.IsSeparated.htm).
