# IPrxDataIslandHeaderCommon


## Описание


Интерфейс IPrxDataIslandHeaderCommon
 позволяет настраивать параметры заголовков области данных и является базовым
 для следующих интерфейсов:


	- [IPrxDataIslandHeader](../IPrxDataIslandHeader/IPrxDataIslandHeader.htm)


	- [IPrxDataIslandDimensionCommon](../IPrxDataIslandDimensionCommon/IPrxDataIslandDimensionCommon.htm)


## Комментарии


Интерфейс предназначен для работы с [областью данных](UiReport.chm::/desktop/AreaData/UiReport_AreaData.htm),
 которая является устаревшим типом таблицы. Для работы с аналитической
 областью данных используйте интерфейс [IEaxDatArea](KeExpress.chm::/Interface/IEaxDataArea/IEaxDataArea.htm).
