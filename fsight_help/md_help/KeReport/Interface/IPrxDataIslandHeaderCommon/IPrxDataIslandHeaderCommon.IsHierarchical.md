# IPrxDataIslandHeaderCommon.IsHierarchical


## Синтаксис


IsHierarchical: Boolean;


## Описание


Свойство IsHierarchical определяет,
 установлена ли раскрывающаяся иерархия.


## Комментарии


Допустимые значения:


	- True. Установлена раскрывающаяся
	 иерархия;


	- False. Не установлена
	 раскрывающаяся иерархия.
