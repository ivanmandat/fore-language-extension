# IPrxDataIslandHeaderCommon.IsChildrenBeforeParents


## Синтаксис


IsChildrenBeforeParents: Boolean;


## Описание


Свойство IsChildrenBeforeParents
 определяет, установлена ли опция «Дети перед родителями».


## Комментарии


Допустимые значения:


	- True. Опция «Дети перед
	 родителями» установлена;


	- False. Опция «Дети перед
	 родителями» не установлена.
