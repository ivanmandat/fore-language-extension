# IPrxDataIslandHeaderCommon.ChildrenBeforeParents


## Синтаксис


ChildrenBeforeParents: [TriState](ForeSys.chm::/Enums/TriState.htm);


## Описание


Свойство ChildrenBeforeParents
 определяет состояние опции «Дети перед родителями».


## Комментарии


Для определения, установлена ли опция «Дети перед родителями», используйте
 свойство [IPrxDataIslandHeaderCommon.IsChildrenBeforeParents](IPrxDataIslandHeaderCommon.IsChildrenBeforeParents.htm).
