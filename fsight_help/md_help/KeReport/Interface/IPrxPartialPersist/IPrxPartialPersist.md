# IPrxPartialPersist


## Описание


Интерфейс IPrxPartialPersist
 предназначен для частичной выгрузки отчета в поток с освобождением занимаемой
 памяти.

		 [SaveToStream](IPrxPartialPersist.SaveToStream.htm)
		 Метод SaveToStream
