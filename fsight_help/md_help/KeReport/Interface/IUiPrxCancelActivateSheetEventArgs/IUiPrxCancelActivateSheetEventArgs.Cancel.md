# IUiPrxCancelActivateSheetEventArgs.Cancel


## Синтаксис


Cancel: Boolean;


## Описание


Свойство Cancel определяет,
 будет ли производиться смена активного листа регламентного отчета.


## Комментарии


При значении данного свойства True
 лист регламентного отчета активирован не будет, при значении False
 - лист отчета будет активирован.
