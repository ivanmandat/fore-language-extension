# IUiPrxCancelActivateSheetEventArgs


## Описание


Интерфейс IUiPrxCancelActivateSheetEventArgs
 содержит свойства аргумента события, происходящего перед сменой листа
 регламентного отчета: [EventOnBeforeActivateSheet](../IPrxReportUserEvents/IPrxReportUserEvents.EventOnBeforeActivateSheet.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Cancel](IUiPrxCancelActivateSheetEventArgs.Cancel.htm)
		 Свойство Cancel определяет,
		 будет ли производиться смена активного листа регламентного отчета.
