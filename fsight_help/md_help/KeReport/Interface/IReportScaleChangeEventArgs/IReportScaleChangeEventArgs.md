# IReportScaleChangeEventArgs


## Описание


Интерфейс IReportScaleChangeEventArgs
 содержит свойства аргумента события, наступающего при изменении масштаба
 листа регламентного отчета: [ReportBox.OnScaleChange](../../Class/ReportBox/ReportBox.OnScaleChange.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Scale](IReportScaleChangeEventArgs.Scale.htm)
		 Свойство Scale возвращает
		 масштаб, который будет установлен для листа регламентного отчета.
