# IPrxDataIslands


## Описание


Интерфейс IPrxDataIslands содержит
 свойства и методы коллекции областей данных.


## Комментарии


Интерфейс предназначен для работы с [областью данных](UiReport.chm::/desktop/AreaData/UiReport_AreaData.htm),
 которая является устаревшим типом таблицы. Для работы с аналитической
 областью данных используйте интерфейс [IEaxDatArea](KeExpress.chm::/Interface/IEaxDataArea/IEaxDataArea.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Report](IPrxDataIslands.Report.htm)
		 Свойство Report возвращает
		 владельца - регламентный отчет.


		 ![](../../Property_Image.gif)
		 [Item](IPrxDataIslands.Item.htm)
		 Свойство Item возвращает
		 область данных регламентного отчета.


		 ![](../../Property_Image.gif)
		 [Count](IPrxDataIslands.Count.htm)
		 Свойство Count возвращает
		 количество областей данных регламентного отчета.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IPrxDataIslands.Add.htm)
		 Метод Add осуществляет
		 добавление новой области данных на листе регламентного отчета.


		 ![](../../Sub_Image.gif)
		 [Remove](IPrxDataIslands.Remove.htm)
		 Метод Remove осуществляет
		 удаление области данных по индексу.


		 ![](../../Sub_Image.gif)
		 [Clear](IPrxDataIslands.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех областей данных регламентного отчета.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IPrxDataIslands.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск по ключу и возвращает область данных регламентного отчета.


		 ![](../../Sub_Image.gif)
		 [FindById](IPrxDataIslands.FindById.htm)
		 Метод FindById осуществляет
		 поиск по идентификатору и возвращает область данных регламентного
		 отчета.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IPrxDataIslands.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление области данных по ключу.
