# IPrxDataIslands.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IPrxDataIsland](../IPrxDataIsland/IPrxDataIsland.htm);


## Параметры


Key. Ключ, по которому осуществляется
 поиск.


## Описание


Метод FindByKey осуществляет
 поиск по ключу и возвращает объект, содержащий область данных регламентного
 отчета.
