# IPrxDataIslands.FindById


## Синтаксис


FindById(Id: String): [IPrxDataIsland](../IPrxDataIsland/IPrxDataIsland.htm);


## Параметры


Id. Идентификатор, по которому
 осуществляется поиск.


## Описание


Метод FindById осуществляет
 поиск по идентификатору и возвращает объект, содержащий область данных
 регламентного отчета.
