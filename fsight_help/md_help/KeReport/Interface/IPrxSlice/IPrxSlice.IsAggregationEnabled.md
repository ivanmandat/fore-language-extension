# IPrxSlice.IsAggregationEnabled


## Синтаксис


IsAggregationEnabled: Boolean;


## Описание


Свойство IsAggregationEnabled возвращает признак наличия агрегации в срезе.


## Комментарии


Свойство возвращает True, если свойству AggregationEnabled установлено значение TriState.OnOption и выбран метод агрегации.
