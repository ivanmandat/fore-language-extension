# IPrxSlice.CreateMapDataAdapter


## Синтаксис


CreateMapDataAdapter(AdapterType: [PrxMapAdapter](../../Enums/PrxMapAdapter.htm)): [IPrxMapDataAdapter](../IPrxMapDataAdapter/IPrxMapDataAdapter.htm);


## Параметры


AdapterType. Тип адаптера для карты.


## Описание


Метод CreateMapDataAdapter создает источник данных для карты, на основе среза данных.
