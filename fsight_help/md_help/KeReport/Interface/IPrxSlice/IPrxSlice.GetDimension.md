# IPrxSlice.GetDimension


## Синтаксис


GetDimension(DimKey Integer): [IEaxDimensionInstance](KeExpress.chm::/Interface/IEaxDimensionInstance/IEaxDimensionInstance.htm);


## Параметры


DimKey. Ключ измерения.


## Описание


Метод GetDimension возвращает
 измерение заданного среза источника данных по его ключу.


## Комментарии


Для работы со срезами источника данных регламентного отчета используйте
 свойство [IPrxSlice.Slices](IPrxSlice.Slices.htm).
