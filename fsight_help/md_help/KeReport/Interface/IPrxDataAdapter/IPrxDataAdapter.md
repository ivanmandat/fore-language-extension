# IPrxDataAdapter


## Описание


Интерфейс IPrxDataAdapter содержит
 базовые свойства источника данных.


## Комментарии


Данный интерфейс является базовым для следующих интерфейсов:


	- [IPrxChartDataAdapter](../IPrxChartDataAdapter/IPrxChartDataAdapter.htm);


	- [IPrxMapDataAdapter](../IPrxMapDataAdapter/IPrxMapDataAdapter.htm).
