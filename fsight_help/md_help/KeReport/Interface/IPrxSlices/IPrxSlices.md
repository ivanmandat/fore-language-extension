# IPrxSlices


## Описание


Интерфейс IPrxSlices содержит
 свойства и методы коллекции срезов источника данных регламентного отчета.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPrxSlices.Count.htm)
		 Свойство Count возвращает
		 количество срезов источника данных регламентного отчета.


		 ![](../../Property_Image.gif)
		 [DataSource](IPrxSlices.DataSource.htm)
		 Свойство DataSource
		 возвращает объект-родитель.


		 ![](../../Property_Image.gif)
		 [Item](IPrxSlices.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий срез источника данных.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IPrxSlices.Add.htm)
		 Метод Add осуществляет
		 добавление нового среза данных.


		 ![](../../Sub_Image.gif)
		 [Clear](IPrxSlices.Clear.htm)
		 Метод Clear удаляет
		 все срезы данных.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IPrxSlices.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск и возвращает объект содержащий срез данных.


		 ![](../../Sub_Image.gif)
		 [Remove](IPrxSlices.Remove.htm)
		 Метод Remove удаляет
		 срез данных.
