# IPrxTableIslandLayout.FilterEnabled


## Синтаксис


FilterEnabled: Boolean;


## Описание


Свойство FilterEnabled определяет,
 будет ли использоваться фильтрация для строк реляционной области данных.


## Комментарии


Допустимые значения:


	- True. Для строк реляционной
	 области данных будет использоваться фильтрация;


	- False. Для строк реляционной
	 области данных не будет использоваться фильтрация.


Для задания условия фильтрации используйте свойство [IPrxTableIslandLayout.Filter](IPrxTableIslandLayout.Filter.htm).
