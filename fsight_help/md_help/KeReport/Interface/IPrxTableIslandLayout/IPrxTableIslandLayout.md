# IPrxTableIslandLayout


Сборка: Report;


## Описание


Интерфейс IPrxTableIslandLayout
 определяет структуру отображения реляционной области данных.


## Иерархия наследования


           IPrxTableIslandLayout


## Комментарии


Для определения параметров реляционной области данных используйте свойства
 и методы интерфейса [IPrxTableIsland](../IPrxTableIsland/IPrxTableIsland.htm).
