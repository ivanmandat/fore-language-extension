# IPrxTableIslandLayout.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index -
 индекс строки отображения.


## Описание


Метод Remove удаляет строку
 отображения.


## Комментарии


В случае успешного удаления метод возвращает True,
 в противном случае False.
