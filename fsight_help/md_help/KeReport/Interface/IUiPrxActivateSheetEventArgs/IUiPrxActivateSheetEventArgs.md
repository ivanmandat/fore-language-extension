# IUiPrxActivateSheetEventArgs


## Описание


Интерфейс IUiPrxActivateSheetEventArgs
 содержит свойства аргумента события, происходящего после смены листа регламентного
 отчета: [EventOnAfterActivateSheet](../IPrxReportUserEvents/IPrxReportUserEvents.EventOnAfterActivateSheet.htm).


## Свойства

		 лист регламентного отчета, который был активным.


		 ![](../../Property_Image.gif)
		 [NewSheet](IUiPrxActivateSheetEventArgs.NewSheet.htm)
		 Свойство NewSheet возвращает
