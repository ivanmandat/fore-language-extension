# IPrxSheetHeaders


Сборка: Report;


## Описание


Интерфейс IPrxSheetHeaders предназначен
 для работы с коллекцией колонтитулов страниц регламентного отчета.


## Иерархия наследования


           IPrxSheetHeaders


## Комментарии


Для настройки параметров выбранного колонтитула используйте свойства
 интерфейса [IPrxSheetHeader](../IPrxSheetHeader/IPrxSheetHeader.htm).
