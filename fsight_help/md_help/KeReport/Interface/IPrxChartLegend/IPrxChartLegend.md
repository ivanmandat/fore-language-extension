# IPrxChartLegend


## Описание


Интерфейс IPrxChartLegend предназначен
 для определения параметров внешней легенды диаграмм. Для доступа к свойствам
 и методам необходимо привести интерфейс IPrxChartLegend
 к [IChartExternLegendItem](Chart.chm::/Interface/IChartExternLegendItem/IChartExternLegendItem.htm).
