# IUiPrxFormulaIslandCancelEventArgs


## Описание


Интерфейс IUiPrxFormulaIslandCancelEventArgs
 предназначен для работы с параметрами событий области формул отчета до
 осуществления каких-либо действий (например, вычисления области формул).
