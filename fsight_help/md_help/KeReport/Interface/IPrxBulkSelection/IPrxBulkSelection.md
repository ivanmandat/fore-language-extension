# IPrxBulkSelection


Сборка: Report;


## Описание


Интерфейс IPrxBulkSelection
 содержит свойства для определения параметров допустимой отметки измерения.


## Иерархия наследования


IPrxBulkSelection


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Control](IPrxBulkSelection.Control.htm)
		 Свойство Control возвращает
		 элемент управления, связанный с измерением, которое используется
		 при пакетной операции.


		 ![](../../Property_Image.gif)
		 [Selection](IPrxBulkSelection.Selection.htm)
		 Свойство Selection
		 возвращает отметку измерения, в соответствии с которой будут формироваться
		 отчеты при выполнении пакетной операции.
