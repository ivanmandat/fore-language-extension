# IPrxBulkSelection.Selection


## Синтаксис


Selection: [IDimSelection](KeDims.chm::/interface/IDimSelection/IDimSelection.htm);


## Описание


Свойство Selection возвращает
 отметку измерения, в соответствии с которой будут формироваться отчеты
 при выполнении пакетной операции.


## Комментарии


Количество отчетов, для которых будет выполнена операция, равно декартовому
 произведению отмеченных элементов во всех допустимых измерениях, входящих
 в коллекцию [IPrxBulkSettings.SelectionSet](../IPrxBulkSettings/IPrxBulkSettings.SelectionSet.htm).
