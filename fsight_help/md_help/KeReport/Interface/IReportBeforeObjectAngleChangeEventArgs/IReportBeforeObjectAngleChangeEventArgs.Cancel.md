# IReportBeforeObjectAngleChangeEventArgs.Cancel


## Синтаксис


Cancel: Boolean;


## Описание


Свойство Cancel определяет,
 отменять ли вращение объекта.


## Комментарии


Допустимые значения:


	- True. Вращение объекта
	 будет отменено;


	- False. Вращение объекта
	 будет выполнено.
