# IReportBeforeObjectAngleChangeEventArgs.NewValue


## Синтаксис


NewValue: Double;


## Описание


Свойство NewValue определяет
 угол, на который был повёрнут объект.


## Комментарии


Значение угла всегда положительное и отсчитывается по часовой стрелке.
