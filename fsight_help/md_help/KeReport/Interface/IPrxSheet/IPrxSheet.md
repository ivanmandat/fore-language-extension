# IPrxSheet


Сборка: Report;


## Описание


Интерфейс IPrxSheet содержит
 свойства и методы листа регламентного отчета.


## Иерархия наследования


           IPrxSheet


## Комментарии


Для работы с коллекцией листов регламентного отчета используйте свойства
 и методы интерфейса [IPrxSheets](../IPrxSheets/IPrxSheets.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Active](IPrxSheet.Active.htm)


		 Свойство Active определяет,
		 будет ли данный лист активным.


		 ![](../../Property_Image.gif)
		 [HeadersFooters](IPrxSheet.HeadersFooters.htm)


		 Свойство HeadersFooters
		 возвращает коллекцию колонтитулов страниц регламентного отчета.


		 ![](../../Property_Image.gif)
		 [Index](IPrxSheet.Index.htm)


		 Свойство Index возвращает
		 индекс листа в списке.


		 ![](../../Property_Image.gif)
		 [IsLoaded](IPrxSheet.IsLoaded.htm)


		 Свойство IsLoaded определяет,
		 будет ли загружаться лист регламентного отчёта.


		 ![](../../Property_Image.gif)
		 [Key](IPrxSheet.Key.htm)


		 Свойство Key возвращает
		 ключ листа регламентного отчета.


		 ![](../../Property_Image.gif)
		 [Name](IPrxSheet.Name.htm)


		 Свойство Name определяет
		 наименование страницы регламентного отчета.


		 ![](../../Property_Image.gif)
		 [PageSettings](IPrxSheet.PageSettings.htm)


		 Свойство PageSettings
		 возвращает параметры листа регламентного отчета.


		 ![](../../Property_Image.gif)
		 [Printable](IPrxSheet.Printable.htm)


		 Свойство Printable
		 определяет, будет ли лист выведен на печать вместе с книгой.


		 ![](../../Property_Image.gif)
		 [ProtectFromGlobalRecalc](IPrxSheet.ProtectFromGlobalRecalc.htm)


		 Свойство ProtectFromGlobalRecalc
		 определяет состояние опции «Не вычислять при вычислении отчета».


		 ![](../../Property_Image.gif)
		 [Scale](IPrxSheet.Scale.htm)


		 Свойство Scale определяет
		 масштаб листа.


		 ![](../../Property_Image.gif)
		 [Sheets](IPrxSheet.Sheets.htm)


		 Свойство Sheets возвращает
		 объект-владелец данного листа.


		 ![](../../Property_Image.gif)
		 [Type](IPrxSheet.Type.htm)


		 Свойство Type возвращает
		 тип листа.


		 ![](../../Property_Image.gif)
		 [Visible](IPrxSheet.Visible.htm)


		 Свойство Visible определяет
		 видимость листа в рабочей области.
