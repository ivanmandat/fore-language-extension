# IPrxSheet.IsLoaded


## Синтаксис


IsLoaded: Boolean;


## Описание


Свойство IsLoaded определяет,
 будет ли загружаться лист регламентного отчёта.


## Комментарии


Свойство используется, если установлена отложенная загрузка листов регламентного
 отчёта: у свойства [IPrxReport.DelayedLoad](../IPrxReport/IPrxReport.DelayedLoad.htm)
 установлено значение True.


Допустимые значения:


	- True. Лист будет загружаться.
	 Значение по умолчанию для активного листа;


	- False. Лист не будет
	 загружаться. Значение по умолчанию для неактивных листов.
