# IPrxSheet.Load


## Синтаксис


Load;


## Описание


Метод Load загружает лист регламентного
 отчёта.


## Комментарии


Метод используется, если:


	- у отчёта для свойства [IPrxReport.DelayedLoad](../IPrxReport/IPrxReport.DelayedLoad.htm)
	 установлено значение True;


	- у листа для свойства [IPrxSheet.IsLoaded](IPrxSheet.IsLoaded.htm)
	 установлено значение True.
