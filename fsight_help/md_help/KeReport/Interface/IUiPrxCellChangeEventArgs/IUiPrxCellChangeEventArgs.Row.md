# IUiPrxCellChangeEventArgs.Row


## Синтаксис


Row: Integer;


## Описание


Свойство Row возвращает строку,
 в которой находится редактируемая ячейка.


## Комментарии


Для получения листа, на котором находится редактируемая ячейка, используйте
 свойство [IUiPrxCellChangeEventArgs.Sheet](IUiPrxCellChangeEventArgs.Sheet.htm).
