# IUiPrxCellChangeEventArgs.Sheet


## Синтаксис


Sheet: [IPrxSheet](../IPrxSheet/IPrxSheet.htm);


## Описание


Свойство Sheet возвращает лист
 отчёта, на котором находится редактируемая ячейка.


## Комментарии


Для получения столбца, в котором находится редактируемая ячейка, используйте
 свойство [IUiPrxCellChangeEventArgs.Column](IUiPrxCellChangeEventArgs.Column.htm).
