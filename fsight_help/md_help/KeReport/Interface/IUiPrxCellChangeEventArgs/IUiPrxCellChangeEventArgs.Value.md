# IUiPrxCellChangeEventArgs.Value


## Синтаксис


Value: Variant;


## Описание


Свойство Value возвращает новое
 значение в ячейке.


## Комментарии


Для получения строки, в которой находится редактируемая ячейка, используйте
 свойство [IUiPrxCellChangeEventArgs.Row](IUiPrxCellChangeEventArgs.Row.htm).
