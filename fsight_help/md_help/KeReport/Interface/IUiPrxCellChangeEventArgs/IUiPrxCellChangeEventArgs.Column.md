# IUiPrxCellChangeEventArgs.Column


## Синтаксис


Column: Integer;


## Описание


Свойство Column возвращает столбец,
 в котором находится редактируемая ячейка.


## Комментарии


Для получения нового значения в ячейке используйте свойство [IUiPrxCellChangeEventArgs.Value](IUiPrxCellChangeEventArgs.Value.htm).
