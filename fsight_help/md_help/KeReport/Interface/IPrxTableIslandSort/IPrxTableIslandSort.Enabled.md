# IPrxTableIslandSort.Enabled


## Синтаксис


Enabled: [TriState](ForeSys.chm::/Enums/TriState.htm);


## Описание


Свойство Enabled определяет,
 включена ли сортировка реляционной области данных.


## Комментарии


Для получения признака использования сортировки реляционной области
 данных используйте свойство [IPrxTableIslandSort.IsEnabled](IPrxTableIslandSort.IsEnabled.htm).
