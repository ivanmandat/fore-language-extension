# IPrxTableIslandSort


Сборка: Report;


## Описание


Интерфейс IPrxTableIslandSort
 предназначен для работы с коллекцией условий сортировки реляционной области
 данных.


## Иерархия наследования


           IPrxTableIslandSort


## Комментарии


Для работы с параметрами условия сортировки реляционной области
 данных используйте свойства и методы интерфейса [IPrxTableIslandSortItem](../IPrxTableIslandSortItem/IPrxTableIslandSortItem.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPrxTableIslandSort.Count.htm)
		 Свойство Count возвращает
		 количество условий сортировки реляционной области данных.


		 ![](../../Property_Image.gif)
		 [Enabled](IPrxTableIslandSort.Enabled.htm)
		 Свойство Enabled определяет,
		 включена ли сортировка реляционной области данных.


		 ![](../../Property_Image.gif)
		 [IsEnabled](IPrxTableIslandSort.IsEnabled.htm)
		 Свойство IsEnabled
		 возвращает признак использования сортировки реляционной области
		 данных.


		 ![](../../Property_Image.gif)
		 [Item](IPrxTableIslandSort.Item.htm)
		 Свойство Item возвращает
		 условие сортировки реляционной области данных по индексу.


		 ![](../../Property_Image.gif)
		 [Parent](IPrxTableIslandSort.Parent.htm)
		 Свойство Parent возвращает
		 параметры родительской реляционной области данных.
