# IPrxTableIslandSort.IsEnabled


## Синтаксис


IsEnabled: Boolean;


## Описание


Свойство IsEnabled возвращает
 признак использования сортировки реляционной области данных.


## Комментарии


Для определения, включена ли сортировка реляционной области данных,
 используйте свойство [IPrxTableIslandSort.Enabled](IPrxTableIslandSort.Enabled.htm).
