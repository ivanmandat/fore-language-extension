# IPrxUserButtonElement


Сборка: Report;


## Описание


Интерфейс IPrxUserButtonElement
 содержит свойства элемента панели инструментов, соответствующего пользовательской
 кнопке.


## Иерархия наследования


IPrxUserButtonElement


## Комментарии


Приведите к интерфейсу IPrxSystemButtonElement элемент панели инструментов.
 описываемый интерфейсом [IPrxToolbarElement](../IPrxToolbarElement/IPrxToolbarElement.htm),
 если свойство [IPrxToolbarElement.Type](../IPrxToolbarElement/IPrxToolbarElement.Type.htm)
 = [PrxToolbarElementType.UserButton](../../Enums/PrxToolbarElementType.htm).
