# IPrxDataIslandHeader.KeepParentCells


## Синтаксис


KeepParentCells: Boolean;


## Описание


Свойство KeepParentCells определяет
 состояние опции «Отображать владельцев неисключенных столбцов».


## Комментарии


Свойство позволяет отображать для дочерних элементов, имеющихся в области
 данных, старшие элементы, которые не отображались в результате фильтрации
 или сортировки.
