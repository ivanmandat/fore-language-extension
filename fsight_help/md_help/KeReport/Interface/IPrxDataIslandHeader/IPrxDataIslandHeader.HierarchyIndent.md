# IPrxDataIslandHeader.HierarchyIndent


## Синтаксис


HierarchyIndent: Double;


## Описание


Свойство HierarchyIndent определяет
 отступ в раскрывающейся иерархии.


## Комментарии


Для использования отступов необходимо свойство [IPrxDataIslandHeaderCommon.AutoIndent](../IPrxDataIslandHeaderCommon/IPrxDataIslandHeaderCommon.AutoIndent.htm)
 установить в OnOption.
