# IPrxDataIslandHeader.CanFitTo


## Синтаксис


CanFitTo(DataIsland: [IPrxDataIslandProperties](../IPrxDataIslandProperties/IPrxDataIslandProperties.htm)):
 Boolean;


## Параметры


DataIsland. Область данных,
 с которой проверяют возможность согласования.


## Описание


Метод CanFitTo осуществляет
 проверку возможности согласования по строкам/столбцам с областью данных.
