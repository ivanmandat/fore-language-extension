# IPrxReportRecalcContext


Сборка: Report;


## Описание


Интерфейс IPrxReportRecalcContext
 содержит свойства и методы для определения параметров вычисления регламентного
 отчета.


## Иерархия наследования


           IPrxReportRecalcContext


## Комментарии


Для работы с объектом репозитория «Отчеты»
 используйте свойства и методы интерфейса [IPrxReport](../IPrxReport/IPrxReport.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ParamValuesChanged](IPrxReportRecalcContext.ParamValuesChanged.htm)
		 Свойство ParamValuesChanged
		 определяет, изменились ли значения параметров.


		 ![](../../Property_Image.gif)
		 [RecalcDataIsland](IPrxReportRecalcContext.RecalcDataIsland.htm)
		 Свойство RecalcDataIsland
		 определяет, необходимо ли вычислить область данных.


		 ![](../../Property_Image.gif)
		 [RecalcEaxObject](IPrxReportRecalcContext.RecalcEaxObject.htm)

		 ![](../../Property_Image.gif)
		 [RecalcFormulaIsland](IPrxReportRecalcContext.RecalcFormulaIsland.htm)
		 Свойство RecalcFormulaIsland

		 ![](../../Property_Image.gif)
		 [RecalcReport](IPrxReportRecalcContext.RecalcReport.htm)
		 Свойство RecalcReport
		 определяет, вычислять ли отчет целиком.


		 ![](../../Property_Image.gif)
		 [RecalcSheet](IPrxReportRecalcContext.RecalcSheet.htm)
		 Свойство RecalcSheet
		 определяет, вычислять ли лист отчета.


		 ![](../../Property_Image.gif)
		 [RecalcTableIsland](IPrxReportRecalcContext.RecalcTableIsland.htm)
		 Свойство RecalcTableIsland
		 определяет, вычислять ли реляционную область данных.


		 ![](../../Property_Image.gif)
		 [Report](IPrxReportRecalcContext.Report.htm)
		 Свойство Report возвращает
		 регламентный отчет, для которого определяются параметры вычисления.
