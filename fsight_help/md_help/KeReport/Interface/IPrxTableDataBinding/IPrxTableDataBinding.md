# IPrxTableDataBinding


## Описание


Интерфейс IPrxTableDataBinding
 определяет привязку ячеек листа отчета.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AttachCell](IPrxTableDataBinding.AttachCell.htm)
		 Метод AttachCell осуществляет
		 привязку ячейки к срезу данных.


		 ![](../../Sub_Image.gif)
		 [AttachRange](IPrxTableDataBinding.AttachRange.htm)
		 Метод AttachRange осуществляет
		 привязку диапазона ячеек к срезу данных.


		 ![](../../Sub_Image.gif)
		 [DetachCell](IPrxTableDataBinding.DetachCell.htm)
		 Метод DetachCell осуществляет
		 отвязку ячейки от данных среза.


		 ![](../../Sub_Image.gif)
		 [DetachRange](IPrxTableDataBinding.DetachRange.htm)
		 Метод DetachRange осуществляет
		 отвязку диапазона ячеек от данных среза.


		 ![](../../Sub_Image.gif)
		 [DrillCell](IPrxTableDataBinding.DrillCell.htm)
		 Метод DrillCell возвращает

		 настроена или нет множественная отметка в фиксированных измерениях.
