# IPrxScene3DNamedPointSerie


## Описание


Интерфейс IPrxScene3DNamedPointSerie
 определяет параметры группы именованных сфер (точек) 3D-сцены.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Name](IPrxScene3DNamedPointSerie.Name.htm)
		 Свойство Name определяет
		 наименование ряда.


		 ![](../../Property_Image.gif)
		 [NameRange](IPrxScene3DNamedPointSerie.NameRange.htm)
		 Свойство NameRange
		 определяет параметры ячейки (диапазона ячеек), содержащей наименование
		 ряда.


		 ![](../../Property_Image.gif)
		 [Points](IPrxScene3DNamedPointSerie.Points.htm)
		 Свойство Points определяет
		 диапазон наименований точек ряда.


		 ![](../../Property_Image.gif)
		 [PointsInRows](IPrxScene3DNamedPointSerie.PointsInRows.htm)
		 Свойство PointsInRows
		 определяет расположение наименований сфер в указанном диапазоне
		 (в строках или столбцах).


		 ![](../../Property_Image.gif)
		 [PointsRange](IPrxScene3DNamedPointSerie.PointsRange.htm)
		 Свойство PointsRange
		 определяет параметры диапазона ячеек, содержащих наименования
		 точек ряда.
