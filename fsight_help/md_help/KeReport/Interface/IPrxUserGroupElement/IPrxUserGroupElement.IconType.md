# IPrxUserGroupElement.IconType


## Синтаксис


IconType: String;


## Описание


Свойство IconType определяет
 тип изображения, используемого для обозначения группы.


## Комментарии


В качестве значения укажите расширение файла, который загружается для
 группы в свойстве [Icon](IPrxUserGroupElement.Icon.htm).
