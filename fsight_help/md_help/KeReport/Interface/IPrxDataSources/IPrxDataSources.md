# IPrxDataSources


Сборка: Report;


## Описание


Интерфейс IPrxDataSources содержит
 свойства и методы коллекции источников данных.


## Иерархия наследования


           IPrxDataSources


## Комментарии


Для работы с источником данных регламентного отчета используйте свойства
 и методы интерфейса [IPrxDataSource](../IPrxDataSource/IPrxDataSource.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Report](IPrxDataSources.Report.htm)
		 Свойство Report возвращает
		 владельца - регламентный отчет.


		 ![](../../Property_Image.gif)
		 [Item](IPrxDataSources.Item.htm)
		 Свойство Item возвращает
		 источник данных.


		 ![](../../Property_Image.gif)
		 [Count](IPrxDataSources.Count.htm)
		 Свойство Count возвращает
		 количество источников данных.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IPrxDataSources.Add.htm)
		 Метод Add добавляет
		 источник данных в регламентный отчет.


		 ![](../../Sub_Image.gif)
		 [AddEx](IPrxDataSources.AddEx.htm)
		 Метод AddEx подключает
		 источник, дескриптор которого передается через параметр Value.


		 ![](../../Sub_Image.gif)
		 [Remove](IPrxDataSources.Remove.htm)
		 Метод Remove осуществляет
		 удаление источника данных по его индексу.


		 ![](../../Sub_Image.gif)
		 [Clear](IPrxDataSources.Clear.htm)
		 Метод Clear удаляет
		 все источники данных регламентного отчета.


		 ![](../../Sub_Image.gif)
		 [FindById](IPrxDataSources.FindById.htm)
		 Метод FindById осуществляет
		 поиск источника данных регламентного отчета по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IPrxDataSources.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск источника данных регламентного отчета по ключу.
