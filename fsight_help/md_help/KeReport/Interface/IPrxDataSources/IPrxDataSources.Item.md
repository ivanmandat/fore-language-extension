# IPrxDataSources.Item


## Синтаксис


Item(Index: Integer): [IPrxDataSource](../IPrxDataSource/IPrxDataSource.htm);


## Параметры


Index. Индекс источника данных.


## Описание


Свойство Item возвращает объект,
 содержащий источник данных, индекс которого передается посредством параметра
 Index.
