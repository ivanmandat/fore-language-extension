# IPrxVisualizerLegend.PlacedControl


## Синтаксис


PlacedControl: [IVZPlacedControlBase](KeVisualizators.chm::/Interface/IVZPlacedControlBase/IVZPlacedControlBase.htm);


## Описание


Свойство PlacedControl определяет
 базовые настройки легенды визуализатора как элемента управления.


## Комментарии


Для установки легенды для указанного визуализатора по его метрике используйте
 [IPrxVisualizerLegend.SetEaxObjectLegend](IPrxVisualizerLegend.SetEaxObjectLegend.htm).
