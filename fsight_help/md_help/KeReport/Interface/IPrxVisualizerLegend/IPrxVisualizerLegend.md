# IPrxVisualizerLegend


Сборка: Report;


## Описание


Интерфейс IPrxVisualizerLegend
 предназначен для работы с легендой визуализатора как отдельного элемента
 регламентного отчета.


## Иерархия наследования


IPrxVisualizerLegend


## Комментарии


Легенда визуализатора зависит от типа визуализатора и его метрики.
