# IPrxDataCache


Сборка: Report;


## Описание


Интерфейс IPrxDataCache предназначен
 для настройки параметров кеширования данных.


## Иерархия наследования


           IPrxDataCache


## Комментарии


Для работы с параметрами настройки регламентного отчета используйте
 свойства и методы интерфейса [IPrxReportOptions](../IPrxReportOptions/IPrxReportOptions.htm).
