# IPrxDataIslandDimensionLayoutNodes


## Описание


Интерфейс IPrxDataIslandDimensionLayoutNodes
 определяет параметры и структуру произвольного заголовка.


## Комментарии


Интерфейс предназначен для работы с [областью данных](UiReport.chm::/desktop/AreaData/UiReport_AreaData.htm),
 которая является устаревшим типом таблицы. Для работы с аналитической
 областью данных используйте интерфейс [IEaxDatArea](KeExpress.chm::/Interface/IEaxDataArea/IEaxDataArea.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPrxDataIslandDimensionLayoutNodes.Count.htm)
		 Свойство Count возвращает
		 количество элементов в произвольном заголовке на одном уровне.


		 ![](../../Property_Image.gif)
		 [Item](IPrxDataIslandDimensionLayoutNodes.Item.htm)
		 Свойство Item возвращает
		 элемент произвольного заголовка по заданному индексу, поиск элемента
		 производится на одном уровне заголовка.
