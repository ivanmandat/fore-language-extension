# IPrxDataIslandDimensionLayoutNodes.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс элемента произвольного
 заголовка.


## Описание


Метод Remove удаляет элемент
 произвольного заголовка по заданному индексу, поиск удаляемого элемента
 производится на одном уровне заголовка.
