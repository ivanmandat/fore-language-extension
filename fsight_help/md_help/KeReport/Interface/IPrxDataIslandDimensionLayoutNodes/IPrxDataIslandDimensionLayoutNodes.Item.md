# IPrxDataIslandDimensionLayoutNodes.Item


## Синтаксис


Item(Index: Integer): [IPrxDataIslandDimensionLayoutNode](../IPrxDataIslandDimensionLayoutNode/IPrxDataIslandDimensionLayoutNode.htm);


## Параметры


Index. Индекс элемента произвольного
 заголовка.


## Описание


Свойство Item возвращает элемент
 произвольного заголовка по заданному индексу, поиск элемента производится
 на одном уровне заголовка.
