# IPrxDataIslandDimensionLayoutNodes.RemoveByKey


## Синтаксис


RemoveByKey(NodeKey: Integer): Boolean;


## Параметры


NodeKey. Ключ элемента произвольного
 заголовка.


## Описание


Метод RemoveByKey удаляет элемент
 произвольного заголовка по заданному ключу, поиск удаляемого элемента
 производится на одном уровне заголовка.
