# IReportObjectEventArgs


## Описание


Интерфейс IReportObjectEventArgs
 содержит свойства аргумента события, происходящего при активации/деактивации
 объекта, расположенного на листе регламентного отчета: [ReportBox.OnObjectActivate](../../Class/ReportBox/ReportBox.OnObjectActivate.htm).
