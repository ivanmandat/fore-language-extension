# IPrxDataIslandTotals.Totals


## Синтаксис


Totals: [TriState](ForeSys.chm::/Enums/TriState.htm);


## Описание


Свойство Totals определяет состояние
 опции «Итоги».


## Комментарии


Данная опция позволяет производить вывод итоговых значений по строкам
 или столбцам на основании значений, расположенных в области данных.


По умолчанию опция отключена.
