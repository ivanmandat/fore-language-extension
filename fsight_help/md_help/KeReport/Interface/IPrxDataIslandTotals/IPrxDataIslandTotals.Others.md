# IPrxDataIslandTotals.Others


## Синтаксис


Others: [TriState](ForeSys.chm::/Enums/TriState.htm);


## Описание


Свойство Others определяет состояние
 опции «Отображать агрегат Other».


## Комментарии


Данная опция позволяет вычислять итоги для элементов измерений, не попавших
 в отметку. По умолчанию опция отключена.
