# IPrxDataIslandDimension.HierarchyPosition


## Синтаксис


HierarchyPosition: Integer;


## Описание


Свойство HierarchyPosition определяет
 позицию размещения иерархии среди дополнительных наименований (заголовков).


## Комментарии


Под иерархией в данном случае понимаются отступы, используемые для отображения
 иерархии.
