# IPrxDataIslandDimension.CustomLayout


## Синтаксис


CustomLayout : Boolean;


## Описание


Свойство CustomLayout определяет
 признак использования произвольного заголовка в области данных.


## Комментарии


Допустимые значения:


	- True. Использован произвольный
	 заголовок;


	- False. Произвольный
	 заголовок не использован. Значение по умолчанию.
