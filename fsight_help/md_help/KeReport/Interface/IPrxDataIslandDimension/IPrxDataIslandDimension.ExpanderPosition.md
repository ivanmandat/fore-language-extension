# IPrxDataIslandDimension.ExpanderPosition


## Синтаксис


ExpanderPosition : Integer;


## Описание


Свойство ExpanderPosition определяет
 позицию размещения экспандеров среди дополнительных наименований (заголовков).


## Комментарии


По умолчанию свойство имеет значение «0» и экспандеры располагаются
 у основного заголовка измерения.
