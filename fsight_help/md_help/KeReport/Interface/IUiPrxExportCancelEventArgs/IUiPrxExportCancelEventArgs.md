# IUiPrxExportCancelEventArgs


## Описание


Интерфейс IUiPrxExportCancelEventArgs
 предназначен для работы с параметрами события [EventOnBeforeExportReport](../IPrxReportUserEvents/IPrxReportUserEvents.EventOnBeforeExportReport.htm).

		 [Exporter](IUiPrxExportCancelEventArgs.Exporter.htm)

		 объект-экспортер, выполняющий экспорт.


		 ![](../../Property_Image.gif)
		 [Format](IUiPrxExportCancelEventArgs.Format.htm)
