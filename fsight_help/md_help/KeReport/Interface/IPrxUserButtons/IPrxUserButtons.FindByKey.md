# IPrxUserButtons.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IPrxUserButton](../IPrxUserButton/IPrxUserButton.htm);


## Параметры


Key. Ключ пользовательской
 кнопки.


## Описание


Метод FindByKey осуществляет
 поиск по ключу и возвращает настройки пользовательской кнопки в случае
 удачного поиска.


## Комментарии


Метод возвращает значение Null,
 если кнопка с указанным ключом отсутствует в коллекции.
