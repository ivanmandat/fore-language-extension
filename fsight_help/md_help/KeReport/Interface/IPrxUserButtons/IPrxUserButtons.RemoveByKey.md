# IPrxUserButtons.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ пользовательской
 кнопки.


## Описание


Метод RemoveByKey удаляет из
 коллекции пользовательскую кнопку с указанным ключом.


## Комментарии


Метод возвращает значение True,
 если удаление завершилось успешно и False,
 если удаление кнопки по каким-либо причинам недоступно.
