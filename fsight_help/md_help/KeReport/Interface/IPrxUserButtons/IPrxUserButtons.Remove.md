# IPrxUserButtons.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс пользовательской
 кнопки.


## Описание


Метод Remove удаляет из коллекции
 пользовательскую кнопку с указанным индексом.


## Комментарии


Метод возвращает значение True,
 если удаление завершилось успешно и False,
 если удаление кнопки по каким-либо причинам недоступно.
