# IPrxUserButtons


Сборка: Report;


## Описание


Интерфейс IPrxUserButtons предназначен
 для работы с коллекций пользовательских кнопок регламентного отчёта.


## Иерархия наследования


IPrxUserButtons


## Комментарии


Коллекцию пользовательских кнопок можно получить, используя свойство
 [IPrxReport.UserButtons](../IPrxReport/IPrxReport.UserButtons.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPrxUserButtons.Count.htm)
		 Свойство Count возвращает
		 количество пользовательских кнопок в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IPrxUserButtons.Item.htm)
		 Свойство Item возвращает
		 настройки пользовательской кнопки с указанным индексом.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IPrxUserButtons.Add.htm)
		 Метод Add создаёт пользовательскую
		 кнопку с заданным описанием и типом.


		 ![](../../Sub_Image.gif)
		 [AddByType](IPrxUserButtons.AddByType.htm)
		 Метод AddByType создаёт
		 пользовательскую кнопку с заданным типом.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IPrxUserButtons.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск по ключу и возвращает настройки пользовательской кнопки
		 в случае удачного поиска.


		 ![](../../Sub_Image.gif)
		 [Remove](IPrxUserButtons.Remove.htm)
		 Метод Remove удаляет
		 из коллекции пользовательскую кнопку с указанным индексом.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IPrxUserButtons.RemoveByKey.htm)
		 Метод RemoveByKey удаляет
		 из коллекции пользовательскую кнопку с указанным ключом.
