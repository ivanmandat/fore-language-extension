# IPrxVisualizer


Сборка: Report;


## Описание


Интерфейс IPrxVisualizer используется
 для работы с визуализатором в регламентном отчете.


## Иерархия наследования


           IPrxVisualizer


## Комментарии


Для получения объекта данного типа необходимо:


	- Для активного листа экспресс отчета создать объект с помощью
	 метода [ITabObjects.Add](TabSheet.chm::/Interface/ITabObjects/ITabObjects.Add.htm).


	- Привести объект типа [ITabObject](TabSheet.chm::/Interface/ITabObject/ITabObject.htm)
	 к IPrxVisualizer.
