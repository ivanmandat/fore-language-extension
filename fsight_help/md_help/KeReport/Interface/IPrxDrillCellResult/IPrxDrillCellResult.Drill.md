# IPrxDrillCellResult.Drill


## Синтаксис


Drill(Row: Integer; Column: Integer): Boolean;


## Параметры


Row. Номер строки ячейки;


Column. Номер столбца ячейки.


## Описание


Метод Drill возвращает True, если можно получить следующую
 ячейку в матрице.
