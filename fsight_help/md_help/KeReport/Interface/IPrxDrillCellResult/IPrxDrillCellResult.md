# IPrxDrillCellResult


## Описание


Интерфейс IPrxDrillCellResult
 содержит свойства привязки ячейки к данным среза.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Column](IPrxDrillCellResult.Column.htm)
		 Свойство Column возвращает
		 номер столбца ячейки, привязанной к срезу данных.


		 ![](../../Property_Image.gif)
		 [Coord](IPrxDrillCellResult.Coord.htm)
		 Свойство Coord определяет
		 координату элемента в матрице.


		 ![](../../Property_Image.gif)
		 [DataSource](IPrxDrillCellResult.DataSource.htm)
		 Свойство DataSource
		 возвращает источник данных, содержащий срез являющийся источником
		 данных для ячейки.


		 ![](../../Property_Image.gif)
		 [Row](IPrxDrillCellResult.Row.htm)
		 Свойство Row возвращает
		 номер строки ячейки, привязанной к срезу данных.


		 ![](../../Property_Image.gif)
		 [Selection](IPrxDrillCellResult.Selection.htm)
		 Свойство Selection
		 возвращает отметку среза, являющегося источником данных для ячейки.


		 ![](../../Property_Image.gif)
		 [Sheet](IPrxDrillCellResult.Sheet.htm)
		 Свойство Sheet возвращает
		 объект, содержащий лист регламентного отчета, на котором расположена
		 ячейка, привязанная к срезу данных.


		 ![](../../Property_Image.gif)
		 [Slice](IPrxDrillCellResult.Slice.htm)
		 Свойство Slice возвращает
		 объект, содержащий срез, являющийся источником данных для ячейки.


## Методы


		 Имя свойства
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Drill](IPrxDrillCellResult.Drill.htm)
		 Метод Drill возвращает
		 True, если можно получить
		 следующую ячейку в матрице.
