# IPrxMapDataAdapter.IdRange


## Синтаксис


IdRange: [ITabRange](tabsheet.chm::/interface/itabrange/Iitabrange.htm);


## Описание


Свойство IdRange определяет
 диапазон ячеек, содержащий идентификаторы регионов на карте.


## Комментарии


Диапазоны значений должны находиться на странице, указанной в свойстве
 [Sheet](IPrxMapDataAdapter.Sheet.htm).
