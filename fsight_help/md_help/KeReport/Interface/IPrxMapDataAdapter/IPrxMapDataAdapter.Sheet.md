# IPrxMapDataAdapter.Sheet


## Синтаксис


Sheet: [IPrxSheet](../IPrxSheet/IPrxSheet.htm);


## Описание


Свойство Sheet определяет лист
 с данными, которые будут отображены на карте.


## Комментарии


Свойство может быть указано при вызове метода [CreateRangeMapDataAdapter](../IPrxDataAdapterManager/IPrxDataAdapterManager.CreateRangeMapDataAdapter.htm).
 При использовании данного свойства страница с диапазонами данных может
 меняться.
