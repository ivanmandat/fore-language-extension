# IPrxMapDataAdapter.DataRange


## Синтаксис


DataRange: [ITabRange](tabsheet.chm::/interface/itabrange/Iitabrange.htm);


## Описание


Свойство DataRange определяет
 диапазон ячеек, содержащий отображаемые данные на карте.


## Комментарии


Диапазоны значений должны находится на странице, указанной в свойстве
 [Sheet](IPrxMapDataAdapter.Sheet.htm).
