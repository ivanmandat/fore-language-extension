# IPrxSliceHeader


## Описание


Интерфейс IPrxSliceHeader содержит
 свойства и методы коллекции измерений среза источника данных регламентного
 отчета, находящихся в определенном заголовке.


## Свойства, унаследованные от [IPrxSliceDimensions](../IPrxSliceDimensions/IPrxSliceDimensions.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](../IPrxSliceDimensions/IPrxSliceDimensions.Count.htm)
		 Свойство Count возвращает
		 количество измерений среза.


		 ![](../../Property_Image.gif)
		 [Item](../IPrxSliceDimensions/IPrxSliceDimensions.Item.htm)
		 Свойство Item возвращает
		 измерение среза источника данных регламентного отчета.


		 ![](../../Property_Image.gif)
		 [Slice](../IPrxSliceDimensions/IPrxSliceDimensions.Slice.htm)
		 Свойство Slice возвращает
		 объект-родитель.


## Методы, унаследованные от [IPrxSliceDimensions](../IPrxSliceDimensions/IPrxSliceDimensions.htm)


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](../IPrxSliceDimensions/IPrxSliceDimensions.FindById.htm)
		 Метод FindById осуществляет
		 поиск измерения среза по заданному идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](../IPrxSliceDimensions/IPrxSliceDimensions.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск измерения среза по заданному ключу.


		 ![](../../Sub_Image.gif)
		 [FindByName](../IPrxSliceDimensions/IPrxSliceDimensions.FindByName.htm)
		 Метод FindByName осуществляет
		 поиск измерения среза по заданному наименованию.
