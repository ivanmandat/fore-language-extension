# IReportBeforeExpanderChangedEventArgs.Cancel


## Синтаксис


Cancel: Boolean;


## Описание


Свойство Cancel определяет признак
 допустимости изменения состояния экспандера.


## Комментарии


Если свойству установить значение True,
 то изменение состояния экспандера будет запрещено, если False
 - разрешено.
