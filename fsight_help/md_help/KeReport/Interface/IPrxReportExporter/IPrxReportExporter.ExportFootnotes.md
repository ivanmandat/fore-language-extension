# IPrxReportExporter.ExportFootnotes


## Синтаксис


ExportFootnotes: Boolean;


## Описание


Свойство ExportFootnotes определяет,
 будут ли экспортироваться сноски.


## Комментарии


По умолчанию свойству установлено значение False,
 при этом сноски не экспортируются. Для экспорта сносок установите свойству
 значение True.
