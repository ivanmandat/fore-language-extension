# IPrxReportExporter.IsExportToPdfAvailable


## Синтаксис


IsExportToPdfAvailable: Boolean;


## Описание


Метод IsExportToPdfAvailable
 осуществляет проверку возможности экспорта в формат PDF.


## Комментарии


При выполнении метода осуществляется проверка наличия библиотеки, необходимой
 для экспорта в PDF. Если библиотека доступна, то экспорт в PDF возможен
 и метод IsExportToPdfAvailable возвращает
 значение True. При отсутствии
 библиотеки метод возвращает значение False.
