# IPrxReportExporter.ExportBreaksRtf


## Синтаксис


ExportBreaksRtf: Boolean;


## Описание


Свойство ExportBreaksRtf определяет,
 будут ли экспортироваться разрывы страниц при экспорте в формат RTF.


## Комментарии


По умолчанию свойству установлено значение True,
 при этом разрывы страниц экспортируются. При установке значения False разрывы экспортированы не
 будут, текст будет объединён.
