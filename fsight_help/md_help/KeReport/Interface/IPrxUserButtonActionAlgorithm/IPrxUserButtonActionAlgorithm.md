# IPrxUserButtonActionAlgorithm


Сборка: Report;


## Описание


Интерфейс IPrxUserButtonActionAlgorithm
 предназначен для настройки запуска алгоритма расчёта в качестве обработчика
 пользовательской кнопки.


## Иерархия наследования


           [IPrxUserButtonAction](../IPrxUserButtonAction/IPrxUserButtonAction.htm)


           IPrxUserButtonActionAlgorithm


## Комментарии


Для настройки запуска алгоритма приведите значение свойства [IPrxUserButton.Action](../IPrxUserButton/IPrxUserButton.Action.htm)
 к интерфейсу IPrxUserButtonActionAlgorithm.
