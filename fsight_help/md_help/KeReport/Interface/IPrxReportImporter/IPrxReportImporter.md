# IPrxReportImporter


Сборка: Report;


## Описание


Интерфейс IPrxReportImporter
 предназначен для импорта данных в регламентный отчёт из внешних файлов.


## Иерархия наследования


IPrxReportImporter


## Комментарии


В текущей реализации метод импорта данных поддерживается только из файлов
 Microsoft Excel (*.xlsx).
