# IPrxSliceDimension.IsDSAggregationSupported


## Синтаксис


IsDSAggregationSupported: Boolean;


## Описание


Свойство IsDSAggregationSupported
 возвращает признак наличия по данному измерению настроек агрегации в источнике
 данных.
