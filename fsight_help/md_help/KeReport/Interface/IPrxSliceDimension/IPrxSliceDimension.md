# IPrxSliceDimension


## Описание


Интерфейс IPrxSliceDimension
 содержит свойства и методы измерения среза источника данных регламентного
 отчета.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AggregationEnabled](IPrxSliceDimension.AggregationEnabled.htm)
		 Свойство AggregationEnabled
		 определяет признак использования агрегации для элементов измерения.


		 ![](../../Property_Image.gif)
		 [AggregatorOperation](IPrxSliceDimension.AggregatorOperation.htm)
		 Свойство AggregatorOperation
		 определяет метод агрегации данных, применяемый для выделенных
		 элементов.


		 ![](../../Property_Image.gif)
		 [Dimension](IPrxSliceDimension.Dimension.htm)
		 Свойство Dimension
		 возвращает данные измерения среза источника данных регламентного
		 отчета.


		 ![](../../Property_Image.gif)
		 [Dimensions](IPrxSliceDimension.Dimensions.htm)
		 Свойство Dimensions
		 возвращает объект-родитель.


		 ![](../../Property_Image.gif)
		 [Disposition](IPrxSliceDimension.Disposition.htm)
		 Свойство Disposition
		 возвращает положение измерения в срезе.


		 ![](../../Property_Image.gif)
		 [HeaderPosition](IPrxSliceDimension.HeaderPosition.htm)
		 Свойство HeaderPosition
		 возвращает положение измерения в заголовке среза.


		 ![](../../Property_Image.gif)
		 [Id](IPrxSliceDimension.Id.htm)
		 Свойство Id возвращает
		 идентификатор измерения среза.


		 ![](../../Property_Image.gif)
		 [InUpdate](IPrxSliceDimension.InUpdate.htm)
		 Свойство InUpdate возвращает
		 признак режима обновления измерения.


		 ![](../../Property_Image.gif)
		 [IsAggregationEnabled](IPrxSliceDimension.IsAggregationEnabled.htm)
		 Свойство IsAggregationEnabled
		 возвращает признак наличия агрегации по данному измерению.


		 ![](../../Property_Image.gif)
		 [IsDSAggregationSupported](IPrxSliceDimension.IsDSAggregationSupported.htm)
		 Свойство IsDSAggregationSupported
		 возвращает признак наличия по данному измерению настроек агрегации
		 в источнике данных.


		 ![](../../Property_Image.gif)
		 [Key](IPrxSliceDimension.Key.htm)
		 Свойство Key возвращает
		 ключ измерения среза.


		 ![](../../Property_Image.gif)
		 [Name](IPrxSliceDimension.Name.htm)
		 Свойство Name возвращает
		 наименование измерения среза.


		 ![](../../Property_Image.gif)
		 [Selection](IPrxSliceDimension.Selection.htm)
		 Свойство Selection
		 возвращает отметку измерения.


		 ![](../../Property_Image.gif)
		 [Slice](IPrxSliceDimension.Slice.htm)
		 Свойство Slice возвращает

		 ![](../../Property_Image.gif)
		 [SyncIndex](IPrxSliceDimension.SyncIndex.htm)
		 Свойство SyncIndex
		 определяет индекс синхронизации измерения среза.


		 ![](../../Property_Image.gif)
		 [UseDSAggregationValue](IPrxSliceDimension.UseDSAggregationValue.htm)
		 Свойство UseDSAggregationValue
		 определяет признак использования в регламентном отчете по данному
		 измерению настроек агрегации, имеющихся в источнике данных.


## Методы


		 Имя метода
		 Краткое описание

		 [MoveToPosition](IPrxSliceDimension.MoveToPosition.htm)

		 среза.
