# IPrxSliceDimension.InUpdate


## Синтаксис


InUpdate: Boolean;


## Описание


Свойство InUpdate возвращает
 признак режима обновления измерения. Свойство возвращает значение True, если был вызван метод [BeginUpdate](IPrxSliceDimension.BeginUpdate.htm)
 и объект, созданный им, еще не был уничтожен.
