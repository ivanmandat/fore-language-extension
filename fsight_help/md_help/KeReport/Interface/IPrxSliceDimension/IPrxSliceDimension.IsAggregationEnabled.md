# IPrxSliceDimension.IsAggregationEnabled


## Синтаксис


IsAggregationEnabled: Boolean;


## Описание


Свойство IsAggregationEnabled
 возвращает признак наличия агрегации по данному измерению. Свойство возвращает
 True, если свойству [AggregationEnabled](IPrxSliceDimension.AggregationEnabled.htm)
 установлено значение TriState.OnOption и выбран метод агрегации.
