# IPrxTableSources


## Описание


Интерфейс IPrxTableSources содержит
 свойства и методы для работы с коллекцией реляционных источников данных
 регламентного отчета.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPrxTableSources.Count.htm)
		 Свойство Count возвращает
		 количество реляционных источников.


		 ![](../../Property_Image.gif)
		 [Item](IPrxTableSources.Item.htm)
		 Свойство Item определяет
		 параметры источника данных по заданному индексу.


		 ![](../../Property_Image.gif)
		 [Report](IPrxTableSources.Report.htm)
		 Свойство Report возвращает
		 регламентный отчет, которому принадлежит коллекция реляционных
		 источников данных.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IPrxTableSources.Add.htm)
		 Метод Add добавляет
		 реляционный источник данных.


		 ![](../../Sub_Image.gif)
		 [Clear](IPrxTableSources.Clear.htm)
		 Метод Clear очищает
		 коллекцию реляционных источников.


		 ![](../../Sub_Image.gif)
		 [FindById](IPrxTableSources.FindById.htm)
		 Метод FindById осуществляет
		 поиск реляционного источника данных по заданному идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IPrxTableSources.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск реляционного источника данных по заданному ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](IPrxTableSources.Remove.htm)
		 Метод Remove удаляет
		 реляционный источник данных по заданному индексу.
