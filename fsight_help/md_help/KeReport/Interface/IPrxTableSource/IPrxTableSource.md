# IPrxTableSource


Сборка: Report;


## Описание


Интерфейс IPrxTableSource содержит
 свойства и методы для работы с реляционным источником данных.


## Иерархия наследования


           [INamedEntity](KeSom.chm::/Interface/INamedEntity/INamedEntity.htm)


           IPrxTableSource


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DataCache](IPrxTableSource.DataCache.htm)
		 Свойство DataCache
		 возвращает реляционную область данных, для которой требуется произвести
		 кеширование данных.


		 ![](../../Property_Image.gif)
		 [DataObject](IPrxTableSource.DataObject.htm)
		 Свойство DataObject
		 возвращает объект репозитория, соответствующий данному реляционному
		 источнику данных.


		 ![](../../Property_Image.gif)
		 [Dataset](IPrxTableSource.Dataset.htm)
		 Свойство Dataset определяет
		 реляционный источник данных.


		 ![](../../Property_Image.gif)
		 [TableSources](IPrxTableSource.TableSources.htm)
		 Свойство TableSources
		 возвращает коллекцию реляционных источников данных.
