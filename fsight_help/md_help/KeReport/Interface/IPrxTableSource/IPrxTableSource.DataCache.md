# IPrxTableSource.DataCache


## Синтаксис


DataCache: [IPrxDataCache](../IPrxDataCache/IPrxDataCache.htm);


## Описание


Свойство DataCache возвращает
 реляционную область данных, для которой требуется произвести кеширование
 данных.


## Комментарии


Для определения, включено ли кеширование данных реляционной области
 данных, используйте свойство [IPrxDataCache.UseDataCache](../IPrxDataCache/IPrxDataCache.UseDataCache.htm).
