# IPrxDimensionDrill.Condition


## Синтаксис


Condition: [IExpression](ForeSys.chm::/Interface/IExpression/IExpression.htm);


## Описание


Свойство Condition возвращает
 выражение условия фильтрации, по которому отбираются элементы, не участвующие
 в расшифровке. Если элемент удовлетворяет заданному условию фильтрации,
 расшифровка не применятся. Если условие фильтрации отсутствует, то поведение
 останется старым.
