# IPrxDimensionDrill.Index


## Синтаксис


Index: [IDimIndex](KeDims.chm::/interface/IDimIndex/IDimIndex.htm);


## Описание


Свойство Index определяет индекс
 для поиска элементов. Необходимо указывать индекс измерения, в котором
 будет модифицироваться отметка.


Данный индекс будет использоваться для поиска элемента, соответствующего
 элементу, гиперссылка которого была нажата.
