# IPrxDimensionDrill.Sheet


## Синтаксис


Sheet: [IPrxSheet](../IPrxSheet/IPrxSheet.htm);


## Описание


Свойство Sheet определяет лист
 отчета, который будет вычисляться при срабатывании гиперссылки. В случае
 если свойство имеет значение NULL, то будет вычислен весь отчет.
