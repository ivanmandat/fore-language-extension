# IPrxSheets.Item


## Синтаксис


Item(Index: Integer): [IPrxSheet](../IPrxSheet/IPrxSheet.htm);


## Параметры


Index. Индекс объекта.


## Описание


Свойство Item возвращает объект,
 содержащий лист регламентного отчета, индекс которого передается посредством
 параметра Index.


## Комментарии


Проверить тип возвращаемого методом Item
 листа регламентного отчета можно с помощью свойства [IPrxSheet.Type](../IPrxSheet/IPrxSheet.Type.htm).
