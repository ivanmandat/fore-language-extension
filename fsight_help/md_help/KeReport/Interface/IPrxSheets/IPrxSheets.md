# IPrxSheets


## Описание


Интерфейс IPrxSheets содержит
 свойства и методы коллекции листов регламентного отчета.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPrxSheets.Count.htm)
		 Свойство Count возвращает
		 количество листов в регламентном отчете.


		 ![](../../Property_Image.gif)
		 [Item](IPrxSheets.Item.htm)
		 Свойство Item возвращает
		 лист регламентного отчета.


		 ![](../../Property_Image.gif)
		 [Report](IPrxSheets.Report.htm)
		 Свойство Report возвращает
		 владельца - регламентный отчет.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IPrxSheets.Add.htm)
		 Метод Add добавляет
		 лист в регламентный отчет.


		 ![](../../Sub_Image.gif)
		 [Clear](IPrxSheets.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех листов регламентного отчета.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IPrxSheets.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск листа по заданному ключу.


		 ![](../../Sub_Image.gif)
		 [FindByName](IPrxSheets.FindByName.htm)
		 Метод FindByName осуществляет
		 поиск листа по заданному наименованию.


		 ![](../../Sub_Image.gif)
		 [Insert](IPrxSheets.Insert.htm)
		 Метод Insert осуществляет
		 вставку нового листа в заданную позицию.


		 ![](../../Sub_Image.gif)
		 [InsertAfter](IPrxSheets.InsertAfter.htm)
		 Метод InsertAfter осуществляет
		 вставку нового листа в позицию, следующую сразу за листом.


		 ![](../../Sub_Image.gif)
		 [Move](IPrxSheets.Move.htm)
		 Метод Move осуществляет
		 перемещение листа из одной позиции в другую.


		 ![](../../Sub_Image.gif)
		 [Remove](IPrxSheets.Remove.htm)
		 Метод Remove осуществляет
		 удаление листа.
