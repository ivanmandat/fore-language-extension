# IPrxSheets.Move


## Синтаксис


Move(OldIndex: Integer; NewIndex: Integer);


## Параметры


OldIndex - старая позиция.


NewIndex - новая позиция.


## Описание


Метод Move осуществляет перемещение
 листа из одной позиции в другую.
