# IPrxControlAppearance.SpecificRecalc


## Синтаксис


SpecificRecalc: Boolean;


## Описание


Свойство SpecificRecalc определяет
 признак вычисления листов регламентного отчета при изменении значения
 элемента управления.


## Комментарии


Допустимые значения:


	- True. Лист будет вычислен;


	- False. Лист вычислен
	 не будет. Значение по умолчанию.


Лист, который надо вычислить, укажите в параметре свойства [IPrxControlAppearance.SheetRecalc](IPrxControlAppearance.SheetRecalc.htm).

Пример использования свойства приведен в описании свойства [IPrxControlAppearance.SheetRecalc](IPrxControlAppearance.SheetRecalc.htm).
