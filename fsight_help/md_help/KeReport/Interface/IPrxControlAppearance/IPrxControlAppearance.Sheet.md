# IPrxControlAppearance.Sheet


## Синтаксис


Sheet(Sheet: [IPrxSheet](../IPrxSheet/IPrxSheet.htm)):
 Boolean;


## Параметры


Sheet - лист отчета.


## Описание


Свойство Sheet определяет признак
 использования элемента управления для определенного листа.


## Комментарии


Допустимые значения:


	- True. Элемент управления
	 будет отображен для листа;


	- False. Элемент управления
	 отображен не будет. Значение по умолчанию.
