# IPrxControlAppearance


Сборка: Report;


## Описание


Интерфейс IPrxControlAppearance
 предназначен для определения области применения элемента управления.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Report](IPrxControlAppearance.Report.htm)
		 Свойство Report определяет
		 признак использования элемента управления на всех листах.


		 ![](../../Property_Image.gif)
		 [Sheet](IPrxControlAppearance.Sheet.htm)
		 Свойство Sheet определяет
		 признак использования элемента управления для определенного листа.


		 ![](../../Property_Image.gif)
		 [SheetRecalc](IPrxControlAppearance.SheetRecalc.htm)
		 Свойство SheetRecalc
		 определяет признак вычисления листа, указанного в качестве параметра,
		 при изменении значения элемента управления.


		 ![](../../Property_Image.gif)
		 [SpecificRecalc](IPrxControlAppearance.SpecificRecalc.htm)
		 Свойство SpecificRecalc
		 определяет признак вычисления листов регламентного отчета при
		 изменении значения элемента управления.
