# IUiPrxTableIslandCancelEventArgs


## Описание


Интерфейс IUiPrxTableIslandCancelEventArgs
 содержит свойства аргумента события, происходящего перед вычислением реляционной
 области данных: [EventOnBeforeExecuteTableIsland](../IPrxReportUserEvents/IPrxReportUserEvents.EventOnBeforeExecuteTableIsland.htm).
