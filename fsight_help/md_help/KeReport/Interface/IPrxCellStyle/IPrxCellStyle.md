# IPrxCellStyle


## Описание


Интерфейс IPrxCellStyle содержит
 свойства и методы оформления ячеек таблицы регламентного отчета.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [IsDefault](IPrxCellStyle.IsDefault.htm)
		 Свойство IsDefault
		 определяет, используется ли значение оформления по умолчанию.


		 ![](../../Property_Image.gif)
		 [IsSet](IPrxCellStyle.IsSet.htm)
		 Свойство IsSet возвращает
		 признак изменения параметров оформления с момента открытия отчёта.
