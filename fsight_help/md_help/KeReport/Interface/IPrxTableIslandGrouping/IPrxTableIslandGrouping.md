# IPrxTableIslandGrouping


## Описание


Интерфейс IPrxTableIslandGrouping
 определяет параметры группы реляционной области данных. Группа определяет
 способ группировки данных, имеющихся в реляционном источнике.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Caption](IPrxTableIslandGrouping.Caption.htm)
		 Свойство Caption определяет
		 заголовок группы.


		 ![](../../Property_Image.gif)
		 [Field](IPrxTableIslandGrouping.Field.htm)
		 Свойство Field определяет
		 привязку к полю реляционного источника данных.


		 ![](../../Property_Image.gif)
		 [Format](IPrxTableIslandGrouping.Format.htm)
		 Свойство Format определяет
		 параметры оформления группы реляционной области данных.


		 ![](../../Property_Image.gif)
		 [Hierarchical](IPrxTableIslandGrouping.Hierarchical.htm)
		 Свойство Hierarchical
		 определяет состояние опции «Раскрывающаяся
		 иерархия» для группы.


		 ![](../../Property_Image.gif)
		 [Indent](IPrxTableIslandGrouping.Indent.htm)
		 Свойство Indent определяет
		 отступ заголовка группы от левого края ячейки.


		 ![](../../Property_Image.gif)
		 [IsHierarchical](IPrxTableIslandGrouping.IsHierarchical.htm)
		 Свойство IsHierarchical
		 возвращает True, если
		 опция «Раскрывающаяся иерархия»
		 включена.


		 ![](../../Property_Image.gif)
		 [Island](IPrxTableIslandGrouping.Island.htm)
		 Свойство Island возвращает
		 реляционную область данных, которой принадлежит группа.


		 ![](../../Property_Image.gif)
		 [Style](IPrxTableIslandGrouping.Style.htm)
		 Свойство Style определяет
		 параметры оформления для группы.


		 ![](../../Property_Image.gif)
		 [Totals](IPrxTableIslandGrouping.Totals.htm)
		 Свойство Totals определяет
		 параметры итогов для группы.


		 ![](../../Property_Image.gif)
		 [Visible](IPrxTableIslandGrouping.Visible.htm)
		 Свойство Visible определяет
		 возможность расположения заголовков группируемых данных в отдельном
		 столбце.
