# IPrxTableIslandGrouping.Hierarchical


## Синтаксис


Hierarchical: [TriState](ForeSys.chm::/Enums/TriState.htm);


## Описание


Свойство Hierarchical определяет
 состояние опции «Раскрывающаяся иерархия»
 для группы.


## Комментарии


Установка флажка «[Раскрывающаяся иерархия](UiReport.chm::/desktop/Relational_data_area/Parameter/UiReport_Relation_Grouping.htm#group_parameters)»
 доступна только при установленном флажке «[Отобразить группу в отдельном столбце](UiReport.chm::/desktop/Relational_data_area/Parameter/UiReport_Relation_Grouping.htm#group_parameters)»
 с помощью свойства [IPrxTableIslandGrouping.Visible](IPrxTableIslandGrouping.Visible.htm).
 По умолчанию флажки установлены.


При установленном флажке «Раскрывающаяся
 иерархия» отображаются экспандеры для сворачивания групп элементов
 перед заголовком группы.
