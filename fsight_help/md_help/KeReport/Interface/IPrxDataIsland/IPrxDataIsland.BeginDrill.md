# IPrxDataIsland.BeginDrill


## Синтаксис


BeginDrill: [IPrxDrillCellResult](../IPrxDrillCellResult/IPrxDrillCellResult.htm);


## Описание


Метод BeginDrill позволяет получить
 значение ячейки области данных из матрицы среза с возможностью подмены
 координаты.


## Комментарии


 Также может быть получена информация о привязке ячейки к источнику,
 для которого настроена множественная отметка в фиксированных измерениях.
