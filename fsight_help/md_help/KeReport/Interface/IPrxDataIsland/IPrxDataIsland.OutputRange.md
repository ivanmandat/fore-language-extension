# IPrxDataIsland.OutputRange


## Синтаксис


OutputRange: String;


## Описание


Свойство OutputRange определяет
 диапазон среза в формате R1C1, выводимый в области данных.


## Комментарии


 Свойство работает только при значении OutputPart = DataOnly.
