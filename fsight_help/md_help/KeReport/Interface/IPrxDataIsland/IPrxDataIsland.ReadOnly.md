# IPrxDataIsland.ReadOnly


## Синтаксис


ReadOnly: Boolean;


## Описание


Свойство ReadOnly определяет
 возможность сохранения измененных данных в источник.


## Комментарии


Допустимые значения:


	- True. Сохранить измененные
	 данные в источник невозможно;


	- False. Измененные данные
	 могут быть сохранены в источник. Значение по умолчанию.


Значение данного свойства влияет на состояние флажка «Разрешить
 изменения данных» в диалоге «[Параметры](UiReport.chm::/desktop/AreaData/Param/UiReport_AreaData_Param_Param.htm)»
 для области данных.
