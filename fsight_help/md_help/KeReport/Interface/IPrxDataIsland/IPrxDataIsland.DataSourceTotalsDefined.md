# IPrxDataIsland.DataSourceTotalsDefined


## Синтаксис


DataSourceTotalsDefined: Boolean;


## Описание


Свойство DataSourceTotalsDefined
 определяет, включена ли агрегация в источнике данных среза.


## Комментарии


Допустимые значения:


	- True. В источнике данных
	 среза есть агрегация, настроенная непосредственно для факта куба (
	 [серверная
	 агрегация](KeCubes.chm::/Interface/IStandardCubeFactBinding/IStandardCubeFactBinding.AggregationType.htm));


	- False. Агрегации в кубе
	 нет.
