# IPrxDataIsland.IsDrillAllowed


## Синтаксис


IsDrillAllowed: Boolean;


## Описание


Свойство IsDrillAllowed определяет,
 разрешена ли привязка данных ячеек к срезу.


## Комментарии


Допустимые значения:


	- True. Включена привязка
	 данных ячеек к срезу. Значение по умолчанию;


	- False. Отключена привязка
	 данных ячеек к срезу.
