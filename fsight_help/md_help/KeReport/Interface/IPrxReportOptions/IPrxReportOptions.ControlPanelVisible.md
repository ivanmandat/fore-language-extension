# IPrxReportOptions.ControlPanelVisible


## Синтаксис


ControlPanelVisible: Boolean;


## Описание


Свойство ControlPanelVisible
 определяет видимость панели элементов управления.


## Комментарии


Допустимые значения:


	- True. Панель элементов
	 управления отображается;


	- False. Панель элементов
	 управления не отображается.


Для определения положения панели элементов управления в отчете используйте
 свойство [IPrxReportOptions.ControlPanelPosition](IPrxReportOptions.ControlPanelPosition.htm).
