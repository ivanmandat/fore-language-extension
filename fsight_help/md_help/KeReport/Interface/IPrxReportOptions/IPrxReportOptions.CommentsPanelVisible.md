# IPrxReportOptions.CommentsPanelVisible


## Синтаксис


CommentsPanelVisible: Boolean;


## Описание


Свойство CommentsPanelVisible
 определяет видимость панели комментариев.


## Комментарии


Допустимые значения:


	- True. Панель комментариев
	 отображается;


	- False. Панель комментариев
	 не отображается.
