# IPrxTableIslandTotalValues.IsValid


## Синтаксис


IsValid: Boolean;


## Описание


Свойство IsValid возвращает
 признак валидности значений.


## Комментарии


Допустимые значения:


	- True. Итоговые значения
	 существуют;


	- False. Итоговые значения
	 не существуют.


Для получения итогового значения ячейки размещения используйте метод
 [IPrxTableIslandTotalValues.TotalValue](IPrxTableIslandTotalValues.TotalValue.htm).
