# IPrxTableIslandTotalValues


Сборка: Report;


## Описание


Интерфейс IPrxTableIslandTotalValues
 предназначен для работы с набором итоговых значений заданного размещения
 в реляционной области данных.

 и методы интерфейса [IPrxTableIsland](../IPrxTableIsland/IPrxTableIsland.htm).
