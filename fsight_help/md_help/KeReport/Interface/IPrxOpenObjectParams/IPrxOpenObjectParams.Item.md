# IPrxOpenObjectParams.Item


## Синтаксис


Item(Index: Integer): [IPrxOpenObjectParam](../IPrxOpenObjectParam/IPrxOpenObjectParam.htm);


## Параметры


Index. Индекс настроек управления
 параметром.


## Описание


Свойство Item возвращает настройки
 управления параметром с указанным индексом.
