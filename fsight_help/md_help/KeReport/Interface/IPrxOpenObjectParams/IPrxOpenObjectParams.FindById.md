# IPrxOpenObjectParams.FindById


## Синтаксис


FindById(Id: String): [IPrxOpenObjectParam](../IPrxOpenObjectParam/IPrxOpenObjectParam.htm);


## Параметры


Id. Идентификатор настроек
 управления параметром.


## Описание


Метод FindById осуществляет
 поиск настроек управления параметром по идентификатору.


## Комментарии


Если настройки с указанным идентификатором не найдены, то метод возвращает
 значение Null.
