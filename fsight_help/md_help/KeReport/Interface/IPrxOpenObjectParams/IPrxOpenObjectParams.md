# IPrxOpenObjectParams


Сборка: Report;


## Описание


Интерфейс IPrxOpenObjectParams
 предназначен для работы с коллекцией настроек управления параметрами объекта,
 открываемого по пользовательской кнопке.


## Иерархия наследования


IPrxOpenObjectParams


## Комментарии


Настройки управления параметрами можно получить, используя свойство
 [IPrxUserButtonActionOpenObject.Params](../IPrxUserButtonActionOpenObject/IPrxUserButtonActionOpenObject.Params.htm).


## Свойства


		 Имя свойства

		 [Count](IPrxOpenObjectParams.Count.htm)
