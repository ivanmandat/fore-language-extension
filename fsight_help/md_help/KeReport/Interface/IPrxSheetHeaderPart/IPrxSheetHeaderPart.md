# IPrxSheetHeaderPart


## Описание


Интерфейс IPrxSheetHeaderPart
 содержит свойства частей колонтитула/заголовка листов/регламентного отчета.
 Заголовок/колонтитул условно разделен на 3 части, отличающиеся способом
 выравнивания текста, и может состоять как из каждой части в отдельности,
 так и из всех сразу.
