# IPrxPicture


## Описание


Интерфейс IPrxPicture содержит
 свойства объекта регламентного отчета - «Рисунок».


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AutoSize](IPrxPicture.AutoSize.htm)
		 Свойство AutoSize определяет,
		 будет ли автоматически изменяться размер элемента управления до
		 размеров изображения, размещенного в нем.


		 ![](../../Property_Image.gif)
		 [Center](IPrxPicture.Center.htm)
		 Свойство Center определяет,
		 будет ли центрироваться рисунок при изменении размеров элемента
		 управления.


		 ![](../../Property_Image.gif)
		 [Image](IPrxPicture.Image.htm)
		 Свойство Image определяет
		 объект, соответствующий рисунку регламентного отчета.


		 ![](../../Property_Image.gif)
		 [Proportional](IPrxPicture.Proportional.htm)
		 Свойство Proportional
		 определяет, будут ли сохраняться пропорции изображения при изменении
		 размеров элемента управления.


		 ![](../../Property_Image.gif)
		 [Stretch](IPrxPicture.Stretch.htm)
		 Свойство Stretch определяет,
		 будет ли растягиваться изображение до размеров элемента управления,
		 в котором оно размещено.
