# IPrxPicture.Stretch


## Синтаксис


Stretch: Boolean;


## Описание


Свойство Stretch определяет,
 будет ли растягиваться изображение до размеров элемента управления, в
 котором оно размещено.


## Комментарии


При значении True изображение
 растягивается до размеров элемента управления, при значении False
 изменение размеров не происходит.


По умолчанию свойство имеет значение True.
