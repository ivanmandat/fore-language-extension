# IPrxSliceDimensions


## Описание


Интерфейс IPrxSliceDimensions
 содержит свойства и методы коллекции измерений среза источника данных
 регламентного отчета.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPrxSliceDimensions.Count.htm)
		 Свойство Count возвращает
		 количество измерений среза.


		 ![](../../Property_Image.gif)
		 [Item](IPrxSliceDimensions.Item.htm)
		 Свойство Item возвращает
		 измерение среза источника данных регламентного отчета.


		 ![](../../Property_Image.gif)
		 [Slice](IPrxSliceDimensions.Slice.htm)
		 Свойство Slice возвращает
		 объект-родитель.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](IPrxSliceDimensions.FindById.htm)
		 Метод FindById осуществляет
		 поиск измерения среза по заданному идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IPrxSliceDimensions.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск измерения среза по заданному ключу.


		 ![](../../Sub_Image.gif)
		 [FindByName](IPrxSliceDimensions.FindByName.htm)
		 Метод FindByName осуществляет
		 поиск измерения среза по заданному наименованию.
