# IPrxDataIslandDimensionLayoutNode.IsDimElement


## Синтаксис


IsDimElement : Boolean;


## Описание


Свойство IsDimElement возвращает
 признак привязки элемента произвольного заголовка к элементу измерения.


## Комментарии


Допустимые значения:


	- True. Элемент произвольного
	 заголовка имеет привязку к элементу измерения;


	- False. Элемент произвольного
	 заголовка не имеет привязку к элементу измерения.
