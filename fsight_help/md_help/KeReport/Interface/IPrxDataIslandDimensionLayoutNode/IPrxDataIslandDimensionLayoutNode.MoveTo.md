# IPrxDataIslandDimensionLayoutNode.MoveTo


## Синтаксис


MoveTo(Node: [IPrxDataIslandDimensionLayoutNode](IPrxDataIslandDimensionLayoutNode.htm);
 Mode: [NodeAttachMode](ForeSys.chm::/Enums/NodeAttachMode.htm));


## Параметры


Node. Элемент дерева произвольного
 заголовка, относительно которого будет перемещаться элемент;


Mode. Положение перемещаемого
 элемента;


## Описание


Метод MoveTo перемещает элемент
 произвольного заголовка в указанную позицию.
