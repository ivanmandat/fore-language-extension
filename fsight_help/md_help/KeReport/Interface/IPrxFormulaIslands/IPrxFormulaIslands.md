# IPrxFormulaIslands


## Описание


Интерфейс IPrxFormulaIslands
 содержит свойства и методы коллекции областей формул регламентного отчёта.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPrxFormulaIslands.Count.htm)
		 Свойство Count возвращает
		 количество областей формул регламентного отчёта.


		 ![](../../Property_Image.gif)
		 [Item](IPrxFormulaIslands.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий область формул.


		 ![](../../Property_Image.gif)
		 [Report](IPrxFormulaIslands.Report.htm)
		 Свойство Report возвращает
		 владельца - регламентный отчёт.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IPrxFormulaIslands.Add.htm)
		 Метод Add осуществляет
		 добавление области формул.


		 ![](../../Sub_Image.gif)
		 [Clear](IPrxFormulaIslands.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех областей формул регламентного отчёта.


		 ![](../../Sub_Image.gif)
		 [FindById](IPrxFormulaIslands.FindById.htm)
		 Метод FindById осуществляет
		 поиск области формул по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IPrxFormulaIslands.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск области формул по заданному ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](IPrxFormulaIslands.Remove.htm)
		 Метод Remove осуществляет
		 удаление области формул.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IPrxFormulaIslands.RemoveByKey.htm)
		 Метод RemoveByKey удаляет
		 область формул по заданному ключу.
