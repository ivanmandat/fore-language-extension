# IPrxFormulaIslands.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key - ключ области формул.


## Описание


Метод RemoveByKey удаляет область
 формул по заданному ключу.


## Комментарии


Метод возвращает True, в случае
 если область формул была удалена, в противном случае False.
