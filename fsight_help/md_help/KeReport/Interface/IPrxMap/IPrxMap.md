# IPrxMap


## Описание


Интерфейс IPrxMap содержит свойства
 объекта регламентного отчета - «Карта».


## Комментарии


Интерфейс предназначен для работы с картой старого формата (*.tbs).
 Для работы с картой нового формата (*.svg) используйте интерфейсы [IVzMapChart](KeVisualizators.chm::/Interface/IVZMapChart/IVZMapChart.htm)
 и [IEaxMapChart](KeExpress.chm::/Interface/IEaxMapChart/IEaxMapChart.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AreaAction](IPrxMap.AreaAction.htm)
		 Свойство AreaAction
		 определяет макрос для обработки событий карты.


		 ![](../../Property_Image.gif)
		 [AutoScaled](IPrxMap.AutoScaled.htm)
		 Свойство AutoScaled
		 определяет, масштабировать ли отображаемый фрагмент топоосновы.


		 ![](../../Property_Image.gif)
		 [OriginalTopobase](IPrxMap.OriginalTopobase.htm)
		 Свойство OriginalTopobase
		 определяет исходную топооснову карты.


		 ![](../../Property_Image.gif)
		 [SpecificXML](IPrxMap.SpecificXML.htm)
		 Свойство SpecificXML
		 определяет отображаемый фрагмент топоосновы в формате XML.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AdviseAdapter](IPrxMap.AdviseAdapter.htm)
		 Метод AdviseAdapter
		 информирует адаптер данных карты об изменении данных в источнике.


		 ![](../../Sub_Image.gif)
		 [DoAreaAction](IPrxMap.DoAreaAction.htm)
		 Метод DoAreaAction
		 генерирует событие щелчка кнопки мыши на территории карты.
