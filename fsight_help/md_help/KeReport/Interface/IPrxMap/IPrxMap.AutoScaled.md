# IPrxMap.AutoScaled


## Синтаксис


AutoScaled: Boolean;


## Описание


Свойство AutoScaled определяет,
 масштабировать ли отображаемый фрагмент топоосновы.


## Комментарии


Отображаемый фрагмент топоосновы определяет свойство [IPrxMap.SpecificXML](IPrxMap.SpecificXML.htm).


Допустимые значения:


	- True. Отображаемая топооснова
	 будет масштабирована по размеру карты;


	- False. Масштаб отображаемой
	 топоосновы изменён не будет.
