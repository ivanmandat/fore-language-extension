# IPrxControl.Width


## Синтаксис


Width: Integer;


## Описание


Свойство Width определяет ширину
 элемента управления. По умолчанию свойству установлено значение "-1",
 при этом ширина подгоняется автоматически, в зависимости от размера панели
 элементов управления.
