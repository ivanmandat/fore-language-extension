# IPrxControl.AutoRecalc


## Синтаксис


AutoRecalc: Boolean;


## Описание


Свойство AutoRecalc определяет,
 будет ли вычислен лист/отчет при смене значения элемента управления.


## Комментарии


Допустимые значения:


	- True. При смене значения
	 элемента управления будут перевычислены все листы, для которых используется
	 данный элемент. Значение по умолчанию;


	- False. Перевычисление
	 не осуществляется.
