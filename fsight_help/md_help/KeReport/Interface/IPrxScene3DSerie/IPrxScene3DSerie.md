# IPrxScene3DSerie


## Описание


Интерфейс IPrxScene3DSerie определяет
 параметры ряда 3D-сцены и является базовым для следующих интерфейсов:


	- [IPrxScene3DPointSerie](../IPrxScene3DPointSerie/IPrxScene3DPointSerie.htm);


	- [IPrxScene3DNamedPointSerie](../IPrxScene3DNamedPointSerie/IPrxScene3DNamedPointSerie.htm);


	- [IPrxScene3DSurfaceSerie](../IPrxScene3DSurfaceSerie/IPrxScene3DSurfaceSerie.htm);


	- [IPrxScene3DConvexHullSerie](../IPrxScene3DConvexHullSerie/IPrxScene3DConvexHullSerie.htm).
