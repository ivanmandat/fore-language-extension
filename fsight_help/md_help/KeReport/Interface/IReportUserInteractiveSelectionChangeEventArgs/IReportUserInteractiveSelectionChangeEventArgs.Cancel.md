# IReportUserInteractiveSelectionChangeEventArgs.Cancel


## Синтаксис


Cancel: Boolean;


## Описание


Свойство Cancel определяет,
 отменять ли изменение интерактивного диапазона.


## Комментарии


Если свойству установить значение True,
 то изменение интерактивного диапазона будет отменено.
