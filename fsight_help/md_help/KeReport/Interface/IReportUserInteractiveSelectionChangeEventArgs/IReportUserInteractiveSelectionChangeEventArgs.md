# IReportUserInteractiveSelectionChangeEventArgs


Сборка: Report;


## Описание


Интерфейс IReportUserInteractiveSelectionChangeEventArgs
 содержит свойства аргумента событий, наступающих при изменении интерактивного
 диапазона ячеек в компоненте ReportBox.
