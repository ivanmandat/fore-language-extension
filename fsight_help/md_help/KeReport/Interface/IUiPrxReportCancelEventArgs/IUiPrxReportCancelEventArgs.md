# IUiPrxReportCancelEventArgs


## Описание


Интерфейс IUiPrxReportCancelEventArgs
 предназначен для работы с параметрами следующих событий:


	- [EventOnBeforeExecuteReport](../IPrxReportUserEvents/IPrxReportUserEvents.EventOnBeforeExecuteReport.htm);


	- [EventOnBeforeOpenReport](../IPrxReportUserEvents/IPrxReportUserEvents.EventOnBeforeOpenReport.htm);


	- [EventOnBeforePrintReport](../IPrxReportUserEvents/IPrxReportUserEvents.EventOnBeforePrintReport.htm);


	- [EventOnBeforeTransferData](../IPrxReportUserEvents/IPrxReportUserEvents.EventOnBeforeTransferData.htm);


	- [EventOnBeforeSaveReport](../IPrxReportUserEvents/IPrxReportUserEvents.EventOnBeforeSaveReport.htm);


	- [UiReport.OnBeforeSaveReport](../../Class/UiReport/UiReport.OnBeforeSaveReport.htm);


	- [EventOnAuditOperation](../IPrxReportUserEvents/IPrxReportUserEvents.EventOnAuditOperation.htm).
