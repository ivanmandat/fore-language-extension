# IUiPrxControlCancelEventArgs


Сборка: Report;


## Описание


Интерфейс IUiPrxControlCancelEventArgs
 предназначен для работы с аргументами события, происходящего перед удалением
 элемента управления.

## Комментарии


Для работы с аргументом события, происходящего при изменении значения
 элемента управления, используйте свойства интерфейса [IUiPrxControlEventArgs](../IUiPrxControlEventArgs/IUiPrxControlEventArgs.htm).
