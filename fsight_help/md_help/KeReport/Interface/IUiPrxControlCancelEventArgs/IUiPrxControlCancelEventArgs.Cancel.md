# IUiPrxControlCancelEventArgs.Cancel


## Синтаксис


Cancel: Boolean;


## Описание


Свойство Cancel определяет,
 будет ли осуществляться удаление элемента управления.


## Комментарии


Допустимые значения:


	- True. Элемент управления
	 не будет удалён;


	- False. Элемент управления
	 будет удалён.
