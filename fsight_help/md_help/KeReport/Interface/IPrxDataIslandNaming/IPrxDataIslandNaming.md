# IPrxDataIslandNaming


## Описание


Интерфейс IPrxDataIslandNaming
 содержит свойства наименований элементов измерений и уровней измерений.


## Комментарии


Интерфейс предназначен для работы с [областью данных](UiReport.chm::/desktop/AreaData/UiReport_AreaData.htm),
 которая является устаревшим типом таблицы. Для работы с аналитической
 областью данных используйте интерфейс [IEaxDatArea](KeExpress.chm::/Interface/IEaxDataArea/IEaxDataArea.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AutoIndent](IPrxDataIslandNaming.AutoIndent.htm)
		 Свойство AutoIndent
		 определяет, в каком состоянии находится автоматическая нумерация
		 строк/столбцов.


		 ![](../../Property_Image.gif)
		 [Expression](IPrxDataIslandNaming.Expression.htm)
		 Свойство Expression
		 возвращает объект, содержащий выражение наименования элементов
		 измерения.


		 ![](../../Property_Image.gif)
		 [IsAutoIndent](IPrxDataIslandNaming.IsAutoIndent.htm)
		 Свойство IsAutoIndent
		 возвращает признак того, установлена ли автоматическая нумерация
		 строк/столбцов.


		 ![](../../Property_Image.gif)
		 [PlaceArrangement](IPrxDataIslandNaming.PlaceArrangement.htm)
		 Свойство PlaceArrangement
		 определяет расположение нумерации относительно наименования элемента.


		 ![](../../Property_Image.gif)
		 [Placement](IPrxDataIslandNaming.Placement.htm)
		 Свойство Placement
		 определяет варианты размещения нумерации строк/столбцов относительно
		 наименований измерений и заголовков.


		 ![](../../Property_Image.gif)
		 [Style](IPrxDataIslandNaming.Style.htm)
		 Свойство Style возвращает
		 объект, содержащий оформления ячеек наименований элементов.
