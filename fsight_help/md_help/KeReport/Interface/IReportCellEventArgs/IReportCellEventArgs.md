# IReportCellEventArgs


## Описание


Интерфейс IReportCellEventArgs
 содержит свойства аргумента событий, генерируемых для ячейки таблицы:


	- [ReportBox.OnCellClick](../../Class/ReportBox/ReportBox.OnCellClick.htm);


	- [ReportBox.OnCellPictureClick](../../Class/ReportBox/ReportBox.OnCellPictureClick.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Column](IReportCellEventArgs.Column.htm)
		 Свойство Column возвращает
		 индекс столбца, в котором находится ячейка, для которой генерируется
		 событие.


		 ![](../../Property_Image.gif)
		 [Row](IReportCellEventArgs.Row.htm)
		 Свойство Row возвращает
		 индекс строки, в которой находится ячейка, для которой генерируется
		 событие.
