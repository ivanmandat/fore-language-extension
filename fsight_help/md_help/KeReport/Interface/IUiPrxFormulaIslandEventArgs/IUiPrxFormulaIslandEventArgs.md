# IUiPrxFormulaIslandEventArgs


## Описание


Интерфейс IUiPrxFormulaIslandEventArgs
 предназначен для работы с параметрами событий области формул регламентного
 отчета:


	- [EventOnAfterExecuteFormulaIsland](../IPrxReportUserEvents/IPrxReportUserEvents.EventOnAfterExecuteFormulaIsland.htm);


	- [EventOnBeforeExecuteFormulaIsland](../IPrxReportUserEvents/IPrxReportUserEvents.EventOnBeforeExecuteFormulaIsland.htm).
