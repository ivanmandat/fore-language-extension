# IUiPrxEaxGridEventArgs


Сборка: Report;


## Описание


Интерфейс IUiPrxEaxGridEventArgs
 предназначен для работы с параметрами событий аналитической области данных,
 происходящих после осуществления определенного действия:


	- [EventOnAfterAddEaxGrid](../IPrxReportUserEvents/IPrxReportUserEvents.EventOnAfterAddEaxGrid.htm);


	- [EventOnAfterDeleteEaxGrid](../IPrxReportUserEvents/IPrxReportUserEvents.EventOnAfterDeleteEaxGrid.htm).
