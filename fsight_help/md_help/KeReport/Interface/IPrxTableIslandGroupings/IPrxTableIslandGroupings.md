# IPrxTableIslandGroupings


## Описание


Интерфейс IPrxTableIslandGroupings
 содержит свойства и методы для работы с коллекцией групп реляционной области
 данных. Группы определяют способ группировки данных, имеющихся в реляционном
 источнике.
