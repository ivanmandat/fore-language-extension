# IPrxToolbar


Сборка: Report;


## Описание


Интерфейс IPrxToolbar предназначен
 для работы с панелью инструментов регламентного отчёта.


## Иерархия наследования


IPrxToolbar


## Комментарии


Настройки панели инструментов возвращает свойство [IPrxReport.Toolbar](../IPrxReport/IPrxReport.Toolbar.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Elements](IPrxToolbar.Elements.htm)
		 Свойство Elements возвращает
		 коллекцию элементов, расположенных на верхнем уровне панели инструментов.


		 ![](../../Property_Image.gif)
		 [IsCustom](IPrxToolbar.IsCustom.htm)
		 Свойство IsCustom возвращает
		 признак наличия изменений в структуре панели инструментов.


		 ![](../../Property_Image.gif)
		 [Pinned](IPrxToolbar.Pinned.htm)
		 Свойство Pinned определяет
		 признак закрепления панели инструментов.
