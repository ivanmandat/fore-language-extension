# IPrxToolbar.IsCustom


## Синтаксис


IsCustom: Boolean;


## Описание


Свойство IsCustom возвращает
 признак наличия изменений в структуре панели инструментов.


## Комментарии


Свойство возвращает значение True,
 если в структуру панели инструментов внесены какие-либо изменения и False, если панель инструментов
 не менялась.
