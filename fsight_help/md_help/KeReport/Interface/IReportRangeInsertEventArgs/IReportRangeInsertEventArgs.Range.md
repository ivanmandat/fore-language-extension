# IReportRangeInsertEventArgs.Range


## Синтаксис

## Описание


Свойство Range возвращает параметры
 диапазона ячеек таблицы.


## Комментарии


Для получения способа, который использовался при удалении диапазона
 ячеек, используйте свойство [IReportRangeInsertEventArgs.ShiftDirection](IReportRangeInsertEventArgs.ShiftDirection.htm).
