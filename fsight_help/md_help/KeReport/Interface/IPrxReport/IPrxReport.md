# IPrxReport


Сборка: Report;


## Описание


Интерфейс IPrxReport содержит
 свойства и методы объекта репозитория «Регламентный
 отчёт».


## Иерархия наследования


           IPrxReport


## Комментарии


Для доступа к активному регламентному отчету используйте свойство [IPrxReportClass.ActiveReport](../IPrxReportClass/IPrxReportClass.ActiveReport.htm).

## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddEventHandler](IPrxReport.AddEventHandler.htm)
		 Метод AddEventHandler
		 добавляет обработчик событий в отчет.


		 ![](../../Sub_Image.gif)
		 [Append](IPrxReport.Append.htm)
		 Метод Append добавляет
		 к отчету листы отчета-шаблона.


		 ![](../../Sub_Image.gif)
		 [BeginRecalc](IPrxReport.BeginRecalc.htm)
		 Метод BeginRecalc создает
		 контекст для вычисления, ссылка на который сохраняется внутри
		 отчета при последующем вызове.


		 ![](../../Sub_Image.gif)
		 [CopyFrom](IPrxReport.CopyFrom.htm)
		 Метод CopyFrom
		 удаляет все имеющиеся в регламентном отчете листы, источники данных,
		 элементы управления и подключенные модули и копирует листы из
		 источника, передаваемого посредством параметра Source.


		 ![](../../Sub_Image.gif)
		 [CopyFromEx](IPrxReport.CopyFromEx.htm)
		 Метод CopyFromEx удаляет
		 все имеющиеся в отчёте листы, источники данных, элементы управления
		 и подключенные модули и копирует всё содержимое отчёта, который
		 необходимо скопировать.


		 ![](../../Sub_Image.gif)
		 [CreateRecalcContext](IPrxReport.CreateRecalcContext.htm)
		 Метод CreateRecalcContext
		 создает новый контекст для вычисления, не привязанный к отчету,
		 т.е. ссылка на него в отчете не хранится.


		 ![](../../Sub_Image.gif)
		 [GetBulkSelectionSet](IPrxReport.GetBulkSelectionSet.htm)
		 Метод GetBulkSelectionSet
		 возвращает допустимую отметку, используемую при выполнении пакетных
		 операций, для указанного диапазона листов.


		 ![](../../Sub_Image.gif)
		 [ParseCell](IPrxReport.ParseCell.htm)
		 Метод ParseCell осуществляет
		 разбор полного адреса ячейки.


		 ![](../../Sub_Image.gif)
		 [ParseRange](IPrxReport.ParseRange.htm)
		 Метод ParseRange осуществляет
		 разбор полного адреса диапазона ячеек.


		 ![](../../Sub_Image.gif)
		 [Recalc](IPrxReport.Recalc.htm)
		 Метод Recalc вычисляет
		 отчет целиком.


		 ![](../../Sub_Image.gif)
		 [RecalcC](IPrxReport.RecalcC.htm)
		 Метод RecalcC вычисляет
		 контекст отчета.


		 ![](../../Sub_Image.gif)
		 [RefreshDataSources](IPrxReport.RefreshDataSources.htm)
		 Метод RefreshDataSources
		 обновляет метаданные источников данных регламентного отчета.


		 ![](../../Sub_Image.gif)
		 [RemoveEventHandler](IPrxReport.RemoveEventHandler.htm)
		 Метод RemoveEventHandler
		 удаляет обработчик событий из отчета.


		 ![](../../Sub_Image.gif)
		 [LoadFromFile](IPrxReport.LoadFromFile.htm)
		 Метод LoadFromFile
		 загружает регламентный отчет из файла.


		 ![](../../Sub_Image.gif)
		 [LoadFromStream](IPrxReport.LoadFromStream.htm)
		 Метод LoadFromStream
		 загружает регламентный отчет из потока.


		 ![](../../Sub_Image.gif)
		 [LockPrxUserEvents](IPrxReport.LockPrxUserEvents.htm)
		 Метод LockPrxUserEvents
		 блокирует использование событий регламентного отчёта.


		 ![](../../Sub_Image.gif)
		 [MoveSelectedGridToSheet](IPrxReport.MoveSelectedGridToSheet.htm)
		 Метод MoveSelectedGridToSheet
		 осуществляет перемещение выделенной таблицы области данных на
		 другой лист.


		 ![](../../Sub_Image.gif)
		 [SaveToFile](IPrxReport.SaveToFile.htm)
		 Метод SaveToFile сохраняет
		 регламентный отчет в файл.


		 ![](../../Sub_Image.gif)
		 [SaveToStream](IPrxReport.SaveToStream.htm)
		 Метод SaveToStream
		 сохраняет регламентный отчет в поток.


		 ![](../../Sub_Image.gif)
		 [UnLockUserPrxEvents](IPrxReport.UnLockUserPrxEvents.htm)
		 Метод UnLockUserPrxEvents
		 разблокировывает использование событий регламентного отчёта.
