# IPrxReport.EnableNewPage


## Синтаксис


EnableNewPage: Boolean;


## Описание


Свойство EnableNewPage определяет возможность добавления новых листов в регламентный отчет.


## Комментарии


Если свойству EnableNewPage установлено значение False, то кнопка добавления листа в регламентном отчете будет скрыта.
