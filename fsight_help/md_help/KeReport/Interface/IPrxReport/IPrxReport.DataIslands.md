# IPrxReport.DataIslands


## Синтаксис


DataIslands: [IPrxDataIslands](../IPrxDataIslands/IPrxDataIslands.htm);


## Описание


Свойство DataIslands возвращает
 объект, содержащий все области данных регламентного отчета.


## Комментарии


Свойство предназначено для работы с [областью
 данных](UiReport.chm::/desktop/AreaData/UiReport_AreaData.htm), которая является устаревшим типом таблицы.
 Для работы с аналитической областью данных используйте интерфейс [IEaxDatArea](KeExpress.chm::/Interface/IEaxDataArea/IEaxDataArea.htm).
