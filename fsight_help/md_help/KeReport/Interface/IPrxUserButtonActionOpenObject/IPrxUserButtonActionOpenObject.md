# IPrxUserButtonActionOpenObject


Сборка: Report;


## Описание


Интерфейс IPrxUserButtonActionOpenObject
 предназначен для настройки открытия объекта репозитория в качестве обработчика
 пользовательской кнопки.


## Иерархия наследования


           [IPrxUserButtonAction](../IPrxUserButtonAction/IPrxUserButtonAction.htm)


           IPrxUserButtonActionOpenObject


## Комментарии


Для настройки открытия объекта репозитория приведите значение свойства
 [IPrxUserButton.Action](../IPrxUserButton/IPrxUserButton.Action.htm)
 к интерфейсу IPrxUserButtonActionOpenObject.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [JsFunction](IPrxUserButtonActionOpenObject.JsFunction.htm)
		 Свойство JsFunction
		 определяет наименование JavaScript-функции, которая будет срабатывать
		 при запуске объекта.


		 ![](../../Property_Image.gif)
		 [ObjectForOpen](IPrxUserButtonActionOpenObject.ObjectForOpen.htm)
		 Свойство ObjectForOpen
		 определяет описание открываемого объекта репозитория.


		 ![](../../Property_Image.gif)
		 [OpenInModal](IPrxUserButtonActionOpenObject.OpenInModal.htm)
		 Свойство OpenInModal
		 определяет признак открытия объекта в модальном режиме.


		 ![](../../Property_Image.gif)
		 [Params](IPrxUserButtonActionOpenObject.Params.htm)
		 Свойство Params возвращает
		 коллекцию настроек управления параметрами открываемого объекта.


		 ![](../../Property_Image.gif)
		 [SheetsForRefresh](IPrxUserButtonActionOpenObject.SheetsForRefresh.htm)
		 Свойство SheetsForRefresh
		 определяет массив ключей листов, которые будут обновлены после
		 выполнения обработчика кнопки.
