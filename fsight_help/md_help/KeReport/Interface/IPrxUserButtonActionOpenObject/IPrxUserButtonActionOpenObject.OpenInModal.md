# IPrxUserButtonActionOpenObject.OpenInModal


## Синтаксис


OpenInModal: Boolean;


## Описание


Свойство OpenInModal определяет
 признак открытия объекта в модальном режиме.


## Комментарии


Допустимые значения:


	- True. Открывать модально
	 относительно окна текущего объекта.


	- False. Значение по умолчанию.
	 Открывать в отдельной вкладке.
