# IPrxTableIslandLayoutRow.Height


## Синтаксис


Height: Double;


## Описание


Свойство Height определяет высоту
 строки отображения.


## Комментарии


При задании высоты строки необходимо свойство [HasHeight](IPrxTableIslandLayoutRow.HasHeight.htm)
 установить в True.
