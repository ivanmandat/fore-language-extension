# IPrxTableIslandLayoutRow


## Описание


Интерфейс IPrxTableIslandLayoutRow
 определяет параметры строки отображения.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Cells](IPrxTableIslandLayoutRow.Cells.htm)
		 Свойство Cells определяет
		 параметры ячейки строки отображения.


		 ![](../../Property_Image.gif)
		 [Count](IPrxTableIslandLayoutRow.Count.htm)
		 Свойство Count возвращает
		 количество ячеек строки отображения.


		 ![](../../Property_Image.gif)
		 [HasHeight](IPrxTableIslandLayoutRow.HasHeight.htm)
		 Свойство HasHeight
		 определяет признак использования пользовательской высоты строки
		 отображения.


		 ![](../../Property_Image.gif)
		 [Height](IPrxTableIslandLayoutRow.Height.htm)
		 Свойство Height определяет
		 высоту строки отображения.


		 ![](../../Property_Image.gif)
		 [Layout](IPrxTableIslandLayoutRow.Layout.htm)
		 Свойство Layout возвращает
		 структуру отображения, которой принадлежит строка.
