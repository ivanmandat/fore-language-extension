# IPrxDataElementStyle


## Описание


Интерфейс IPrxDataElementStyle
 содержит свойства оформления уровней измерения, на котором построена область
 данных.


## Комментарии


Интерфейс предназначен для работы с [областью
 данных](UiReport.chm::/desktop/AreaData/UiReport_AreaData.htm), которая является устаревшим типом таблицы.
