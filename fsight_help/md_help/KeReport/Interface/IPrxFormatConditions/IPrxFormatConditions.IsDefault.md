# IPrxFormatConditions.IsDefault


## Синтаксис


IsDefault : Boolean;


## Описание


Свойство IsDefault определяет
 признак наследования оформления по условию от родителя.


В случае если свойство имеет значение True,
 то настройки оформления будут унаследованы от родителя, при значение False - используются собственные
 настройки оформления.


По умолчанию свойство имеет значение False.


Допустим, для измерения области данных
 и для его уровня настроено различное условное форматирование, если для
 уровня установить свойство IsDefault
 в True, то настройки условного
 форматирования для уровня будут проигнорированы и на листе отчета будет
 отображено условное форматирование, только которое настроено для всего
 измерения.
