# IPrxFormatConditions


## Описание


Интерфейс IPrxFormatConditions
 определяет параметры коллекции условий форматирования.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [IsDefault](IPrxFormatConditions.IsDefault.htm)
		 Свойство IsDefault
		 определяет признак наследования оформления по условию от родителя.


		 ![](../../Property_Image.gif)
		 [IsSet](IPrxFormatConditions.IsSet.htm)
		 Свойство IsSet возвращает
		 True, в случае если настроено
		 условное форматирование, False
		 - если условное форматирование настроено не было.


## Свойства, унаследованные от [ITabFormatConditions](TabSheet.chm::/Interface/ITabFormatConditions/ITabFormatConditions.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](TabSheet.chm::/Interface/ITabFormatConditions/ITabFormatConditions.Count.htm)

		 количество условных форматов в коллекции.

		 [Item](TabSheet.chm::/Interface/ITabFormatConditions/ITabFormatConditions.Item.htm)
