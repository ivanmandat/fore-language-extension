# IReportProtectionFailEventArgs.Handled


## Синтаксис


Handled: Boolean;


## Описание


Свойство Handled определяет,
 обработано ли сработавшее нарушение защиты.


## Комментарии


Свойство доступно только для записи. При установке свойству значения
 True нарушение защиты считается
 обработанным и будет продолжена работа с отчётом.
