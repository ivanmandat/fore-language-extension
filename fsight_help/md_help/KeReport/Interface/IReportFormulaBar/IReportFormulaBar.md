# IReportFormulaBar


Сборка: Report;


## Описание


Интерфейс IReportFormulaBar
 предназначен для работы с компонентом среды разработки [ReportFormulaBar](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/ReportFormulaBar.htm).


## Иерархия наследования


           [IUiBar](ModForms.chm::/Interface/IUiBar/IUiBar.htm)


           IReportFormulaBar


## Комментарии


Для работы с компонентом среды разработки [ReportBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/ReportBox.htm)
 используйте свойства и методы интерфейса [IReportBox](../IReportBox/IReportBox.htm).
