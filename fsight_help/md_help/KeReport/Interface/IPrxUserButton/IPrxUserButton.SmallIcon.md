# IPrxUserButton.SmallIcon


## Синтаксис


SmallIcon: Boolean;


## Описание


Свойство SmallIcon определяет,
 будет ли для пользовательской кнопки использоваться маленькое изображение.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Будет использоваться маленькое изображение, сжатое до 16*16 пикселей;


	- False. Используется
	 стандартный размер загруженного изображения.
