# IPrxUserButton.IconType


## Синтаксис


IconType: String;


## Описание


Свойство IconType определяет
 тип хранимого изображения.


## Комментарии


Допустимые значения:


	- svg;


	- png;


	- jpg;


	- bmp;


	- gif.
