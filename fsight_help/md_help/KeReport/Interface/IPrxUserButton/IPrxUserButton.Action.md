# IPrxUserButton.Action


## Синтаксис


Action: [IPrxUserButtonAction](../IPrxUserButtonAction/IPrxUserButtonAction.htm);


## Описание


Свойство Action возвращает обработчик
 действия кнопки, который будет срабатывать при нажатии на кнопку.


## Комментарии


В зависимости от [типа кнопки](IPrxUserButton.Type.htm) значение
 свойства Action может быть приведено
 к одному из следующих интерфейсов:


	- [IPrxUserButtonActionAlgorithm](../IPrxUserButtonActionAlgorithm/IPrxUserButtonActionAlgorithm.htm)


	- [IPrxUserButtonActionForeMethod](../IPrxUserButtonActionForeMethod/IPrxUserButtonActionForeMethod.htm)


	- [IPrxUserButtonActionOpenObject](../IPrxUserButtonActionOpenObject/IPrxUserButtonActionOpenObject.htm)
