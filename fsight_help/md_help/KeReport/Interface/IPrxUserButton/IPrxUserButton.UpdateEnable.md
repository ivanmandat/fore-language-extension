# IPrxUserButton.UpdateEnable


## Синтаксис


UpdateEnable: [IPrxUserButtonActionForeMethod](../IPrxUserButtonActionForeMethod/IPrxUserButtonActionForeMethod.htm);


## Описание


Свойство UpdateEnable возвращает
 обработчик действия кнопки, который будет срабатывать при определении
 доступности кнопки.

 [IPrxUserButtons.AddByType](../IPrxUserButtons/IPrxUserButtons.AddByType.htm).
