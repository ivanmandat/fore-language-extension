# IPrxUserButton.Enable


## Синтаксис


Enable: Boolean;


## Описание


Свойство Enable возвращает признак
 доступности кнопки.


## Комментарии


Свойство возвращает значение True,
 если кнопка доступна и False,
 если недоступна. Доступность кнопки можно определить в обработчике [UpdateEnable](IPrxUserButton.UpdateEnable.htm).
