# IPrxUserButton


Сборка: Report;


## Описание


Интерфейс IPrxUserButton предназначен
 для настройки пользовательской кнопки в регламентном отчёте.


## Иерархия наследования


           [INamedEntity](KeSom.chm::/Interface/INamedEntity/INamedEntity.htm)


           IPrxUserButton


## Комментарии


Настройки пользовательской кнопки можно получить, используя свойства
 и методы интерфейса [IPrxUserButtons](../IPrxUserButtons/IPrxUserButtons.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Action](IPrxUserButton.Action.htm)
		 Свойство Action возвращает
		 обработчик действия кнопки, который будет срабатывать при нажатии
		 на кнопку.


		 ![](../../Property_Image.gif)
		 [AfterAction](IPrxUserButton.AfterAction.htm)
		 Свойство AfterAction
		 возвращает обработчик действия кнопки, который будет срабатывать
		 после нажатия на кнопку.


		 ![](../../Property_Image.gif)
		 [BeforeAction](IPrxUserButton.BeforeAction.htm)
		 Свойство BeforeAction
		 возвращает обработчик действия кнопки, который будет срабатывать
		 перед нажатием на кнопку.


		 ![](../../Property_Image.gif)
		 [Enable](IPrxUserButton.Enable.htm)
		 Свойство Enable возвращает
		 признак доступности кнопки.


		 ![](../../Property_Image.gif)
		 [ForeModule](IPrxUserButton.ForeModule.htm)
		 Свойство ForeModule
		 определяет Fore-модуль, методы которого будут использоваться для
		 обработки действий кнопки.


		 ![](../../Property_Image.gif)
		 [Icon](IPrxUserButton.Icon.htm)
		 Свойство Icon определяет
		 поток, содержащий изображение, отображаемое на месте пользовательской
		 кнопки.


		 ![](../../Property_Image.gif)
		 [IconType](IPrxUserButton.IconType.htm)
		 Свойство IconType определяет
		 тип хранимого изображения.


		 ![](../../Property_Image.gif)
		 [SmallIcon](IPrxUserButton.SmallIcon.htm)
		 Свойство SmallIcon
		 определяет, будет ли для пользовательской кнопки использоваться
		 маленькое изображение.


		 ![](../../Property_Image.gif)
		 [Type](IPrxUserButton.Type.htm)
		 Свойство Type определяет
		 тип обработчика пользовательской кнопки.


		 ![](../../Property_Image.gif)
		 [UpdateEnable](IPrxUserButton.UpdateEnable.htm)
		 Свойство UpdateEnable
		 возвращает обработчик действия кнопки, который будет срабатывать
		 при определении доступности кнопки.
