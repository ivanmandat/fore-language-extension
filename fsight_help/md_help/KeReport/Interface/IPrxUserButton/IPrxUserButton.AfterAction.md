# IPrxUserButton.AfterAction


## Синтаксис


AfterAction: [IPrxUserButtonActionMethod](../IPrxUserButtonActionMethod/IPrxUserButtonActionMethod.htm);


## Описание


Свойство AfterAction возвращает
 обработчик действия кнопки, который будет срабатывать после нажатия на
 кнопку.
