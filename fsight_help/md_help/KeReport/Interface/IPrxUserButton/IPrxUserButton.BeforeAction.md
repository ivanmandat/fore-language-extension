# IPrxUserButton.BeforeAction


## Синтаксис


BeforeAction: [IPrxUserButtonActionMethod](../IPrxUserButtonActionMethod/IPrxUserButtonActionMethod.htm);


## Описание


Свойство BeforeAction возвращает
 обработчик действия кнопки, который будет срабатывать перед нажатием на
 кнопку.

Использование свойства приведено в примерах для методов [IPrxUserButtons.Add](../IPrxUserButtons/IPrxUserButtons.Add.htm),
 [IPrxUserButtons.AddByType](../IPrxUserButtons/IPrxUserButtons.AddByType.htm).
