## Синтаксис

## Описание


Свойство Range возвращает параметры
 диапазона ячеек таблицы.


## Комментарии


Для получения способа, который использовался при удалении диапазона
 ячеек, используйте свойство [IReportRangeDeleteEventArgs.ShiftDirection](IReportRangeDeleteEventArgs.ShiftDirection.htm).
