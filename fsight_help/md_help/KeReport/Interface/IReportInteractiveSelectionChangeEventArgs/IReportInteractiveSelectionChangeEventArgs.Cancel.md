# IReportInteractiveSelectionChangeEventArgs.Cancel


## Синтаксис


Cancel: Boolean;


## Описание


Свойство Cancel определяет признак
 допустимости перемещения выделенной области ячеек.


## Комментарии


Если свойству установить значение True,
 то перемещение ячеек разрешено, иначе - запрещено.
