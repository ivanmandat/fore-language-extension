# IReportRowResizingEventArgs.Cancel


## Синтаксис


Cancel: Boolean;


## Описание


Свойство Cancel определяет допустимость
 изменения высоты строки.


## Комментарии


Если свойству установить значение True,
 то изменение высоты строки будет запрещено, если False
 - разрешено.
