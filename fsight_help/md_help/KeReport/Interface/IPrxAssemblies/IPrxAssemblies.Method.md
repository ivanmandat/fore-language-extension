# IPrxAssemblies.Method


## Синтаксис


Method(Index: Integer): [PamAssemblyMethod](../../Enums/PamAssemblyMethod.htm);


## Параметры


Index. Индекс объекта среды
 разработки, подключённого к регламентному отчёту.


## Описание


Свойство Method возвращает способ подключения объекта среды
 разработки к регламентному отчёту, входящему в состав формы ввода.


## Комментарии


Для получения коллекции объектов среды разработки, подключённых к регламентному
 отчёту, используйте свойство [IPrxReport.Assemblies](../IPrxReport/IPrxReport.Assemblies.htm).
