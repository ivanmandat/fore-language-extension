# IPrxAssemblies.EventsAssembly


## Синтаксис


EventsAssembly: Integer;


## Описание


Свойство EventsAssembly определяет
 индекс объекта, который будет использоваться для обработки событий регламентного
 отчёта.


## Комментарии


Объект, индекс которого указан в данном свойстве, должен содержать в
 себе реализацию класса, унаследованного от [ReportEvents](../../Class/ReportEvents/ReportEvents.htm).


Если обработка событий не осуществляется, то в данном свойстве необходимо
 установить значение «-1».
