# IPrxAssemblies


## Описание


Интерфейс IPrxAssemblies содержит
 свойства и методы для работы с коллекцией объектов среды разработки, подключенных
 к регламентному отчету.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPrxAssemblies.Count.htm)
		 Свойство Count возвращает
		 количество подключенных объектов.


		 ![](../../Property_Image.gif)
		 [EventsAssembly](IPrxAssemblies.EventsAssembly.htm)
		 Свойство EventsAssembly
		 определяет индекс объекта, который будет использоваться для обработки
		 событий регламентного отчёта.


		 ![](../../Property_Image.gif)
		 [EventsClass](IPrxAssemblies.EventsClass.htm)
		 Свойство EventsClass
		 определяет класс, используемый для обработки событий.


		 ![](../../Property_Image.gif)
		 [Item](IPrxAssemblies.Item.htm)
		 Свойство Item возвращает
		 описание объекта, подключенного к регламентному отчёту.


		 ![](../../Property_Image.gif)
		 [Method](IPrxAssemblies.Method.htm)
		 Свойство Method
		 возвращает способ
		 подключения объекта среды разработки к регламентному отчёту, входящему
		 в состав формы ввода.


		 ![](../../Property_Image.gif)
		 [Report](IPrxAssemblies.Report.htm)
		 Свойство Report возвращает
		 объект-родитель.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IPrxAssemblies.Add.htm)
		 Метод Add осуществляет
		 подключение объекта среды разработки к регламентному отчёту и
		 возвращает индекс подключённого объекта в списке.


		 ![](../../Sub_Image.gif)
		 [AddEx](IPrxAssemblies.AddEx.htm)
		 Метод AddEx осуществляет
		 подключение объекта среды разработки к регламентному отчёту, входящему
		 в состав формы ввода, и возвращает индекс подключённого объекта
		 в списке.


		 ![](../../Sub_Image.gif)
		 [Clear](IPrxAssemblies.Clear.htm)
		 Метод Clear осуществляет
		 очистку списка подключенных объектов.


		 ![](../../Sub_Image.gif)
		 [Edit](IPrxAssemblies.Edit.htm)
		 Метод Edit открывает
		 на редактирование коллекцию подключенных объектов.


		 ![](../../Sub_Image.gif)
		 [Save](IPrxAssemblies.Save.htm)
		 Метод Save осуществляет
		 сохранение изменений, внесенных в коллекцию подключенных объектов,
		 до сохранения всего отчета.


		 ![](../../Sub_Image.gif)
		 [Remove](IPrxAssemblies.Remove.htm)
		 Метод Remove осуществляет
		 удаление подключенного объекта.


		 ![](../../Sub_Image.gif)
		 [Revert](IPrxAssemblies.Revert.htm)
		 Метод Revert осуществляет
		 отмену изменений, внесенных в коллекцию подключенных объектов,
		 до сохранения всего отчета.
