# IPrxAssemblies.Save


## Синтаксис


Save;


## Описание


Метод Save осуществляет сохранение
 изменений, внесенных в коллекцию подключенных объектов, до сохранения
 всего отчета. Данный метод используется совместно с методом [Edit](IPrxAssemblies.Edit.htm).
