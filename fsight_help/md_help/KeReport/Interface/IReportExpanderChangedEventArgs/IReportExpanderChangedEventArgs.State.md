# IReportExpanderChangedEventArgs.State


## Синтаксис


State: [TriState](ForeSys.chm::/Enums/TriState.htm);


## Описание


Свойство State возвращает состояние
 экспандера, которое будет установлено.


## Комментарии


Свойство возвращает следующие значения:


	- [TriState.OnOption](ForeSys.chm::/Enums/TriState.htm)
	 - экспандер будет развернут.


	- [TriState.OffOption](ForeSys.chm::/Enums/TriState.htm) -
	 экспандер будет свернут.
