# IPrxDataIslandDimensionSort.UseActualColumnIndex


## Синтаксис


UseActualColumnIndex: Boolean;


## Описание


Свойство UseActualColumnIndex
 определяет необходимость производить сортировку только по индексам столбцов/строк,
 видимых в области данных.


## Комментарии


По умолчанию сортировка в области данных работает с учетом скрытых строк/столбцов,
 то есть, сначала размещается область данных, затем к указанному столбцу
 применяется сортировка, при этом скрываются столбцы и строки (например,
 когда выводятся только листьевые элементы). Соответственно при наложении
 сортировки, например, на второй столбец индекс «2» не будет являться индексом
 второго видимого столбца, а будет принадлежать столбцу, который в данный
 момент скрыт.
