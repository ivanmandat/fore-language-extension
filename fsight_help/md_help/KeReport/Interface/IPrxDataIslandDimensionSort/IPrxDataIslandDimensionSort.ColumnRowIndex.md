# IPrxDataIslandDimensionSort.ColumnRowIndex


## Синтаксис


ColumnRowIndex: Integer;


## Описание


Свойство ColumnRowIndex определяет
 номер строки/столбца при сортировке по строке/столбцу.


## Комментарии


При наличии в регламентном отчете скрытых/отфильтрованных строк/столбцов
 нумерация производится с учетом значения свойства [UseActualColumnIndex](IPrxDataIslandDimensionSort.UseActualColumnIndex.htm).
