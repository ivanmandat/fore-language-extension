# IPrxDataIslandDimensionSort.Filter


## Синтаксис


Filter: [PrxDimensionSortFilter](../../Enums/PrxDimensionSortFilter.htm);


## Описание


Свойство Filter определяет поведение
 при наличии пустых и нечисловых значений.


## Комментарии


В значении Empty только пустые
 элементы не участвуют в сортировке. Нечисловые элементы помещаются в начало
 сортируемой области.


В значении NonNumerical нечисловые
 элементы не участвуют в сортировке и помещаются в конец сортируемой области.


В интерфейсе свойство выведено в виде флажка «Не
 учитывать нечисловые значения» в окне «Параметры
 области данных» на вкладке «Сортировка».
