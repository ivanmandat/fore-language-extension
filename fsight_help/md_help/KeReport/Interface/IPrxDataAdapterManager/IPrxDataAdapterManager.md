# IPrxDataAdapterManager


Сборка: Report;


## Описание


Интерфейс IPrxDataAdapterManager
 позволяет устанавливать тип адаптера для отображения исходных данных на
 карте.


## Комментарии


Интерфейс предназначен для работы с картой старого формата (*.tbs).
 Для работы с картой нового формата (*.svg)
 используйте интерфейсы [IVzMapChart](KeVisualizators.chm::/Interface/IVZMapChart/IVZMapChart.htm)
 и [IEaxMapChart](KeExpress.chm::/Interface/IEaxMapChart/IEaxMapChart.htm).
