# IPrxDataElementStyleCommon


## Описание


Интерфейс IPrxDataElementStyleCommon
 определяет оформление измерений и уровней измерения. Данный интерфейс
 является базовым для интерфейса [IPrxDataElementStyle](../IPrxDataElementStyle/IPrxDataElementStyle.htm).


## Комментарии


Интерфейс предназначен для работы с [областью
 данных](UiReport.chm::/desktop/AreaData/UiReport_AreaData.htm), которая является устаревшим типом таблицы.
