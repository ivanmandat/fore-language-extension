# IPrxDataChange


## Описание


Интерфейс IPrxDataChange предназначен
 для работы с изменёнными данными области данных.


## Комментарии


Интерфейс IPrxDataChange поддерживается
 следующими интерфейсами:


	- Отчёт. [IPrxReport](../IPrxReport/IPrxReport.htm);


	- Листа отчёта. [IPrxSheet](../IPrxSheet/IPrxSheet.htm);


	- Область данных. [IPrxDataIsland](../IPrxDataIsland/IPrxDataIsland.htm).
	 [Область
	 данных](UiReport.chm::/desktop/AreaData/UiReport_AreaData.htm) является устаревшим типом таблицы. Для работы
	 с аналитической областью данных используйте интерфейс [IEaxDatArea](KeExpress.chm::/Interface/IEaxDataArea/IEaxDataArea.htm).
