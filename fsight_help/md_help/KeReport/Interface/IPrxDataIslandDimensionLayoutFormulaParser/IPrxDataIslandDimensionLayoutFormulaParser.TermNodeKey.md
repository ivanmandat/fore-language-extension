# IPrxDataIslandDimensionLayoutFormulaParser.TermNodeKey


## Синтаксис


TermNodeKey(Term: String): Integer;


## Параметры


Term. Терм.


## Описание


Свойство TermNodeKey определяет
 ключ элемента размещения для терма выражения.


## Комментарии


Данное свойство учитывается, если свойство [IPrxDataIslandDimensionLayoutFormulaParser.TermType](IPrxDataIslandDimensionLayoutFormulaParser.TermType.htm)
 = [Node](../../Enums/LayoutFormulaTermType.htm).
