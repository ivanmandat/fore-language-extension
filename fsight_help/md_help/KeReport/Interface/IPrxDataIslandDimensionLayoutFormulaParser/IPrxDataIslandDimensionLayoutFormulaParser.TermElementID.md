# IPrxDataIslandDimensionLayoutFormulaParser.TermElementID


## Синтаксис


TermElementID(Term: String): String;


## Параметры


Term. Терм.


## Описание


Свойство TermElementID определяет
 идентификатор элемента измерения для терма выражения.


## Комментарии


Данное свойство учитывается, если свойство [IPrxDataIslandDimensionLayoutFormulaParser.TermType](IPrxDataIslandDimensionLayoutFormulaParser.TermType.htm)
 = [Element](../../Enums/LayoutFormulaTermType.htm).
