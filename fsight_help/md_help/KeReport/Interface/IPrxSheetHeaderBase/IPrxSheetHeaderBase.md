# IPrxSheetHeaderBase


## Описание


Интерфейс IPrxSheetHeaderBase
 определяет параметры частей колонтитула/заголовка листов/регламентного
 отчета и является базовым для интерфейсов [IPrxReportTitle](../IPrxReportTitle/IPrxReportTitle.htm)
 и [IPrxSheetHeader](../IPrxSheetHeader/IPrxSheetHeader.htm).


Заголовок (колонтитул) условно разделен на 3 части, отличающихся способом
 выравнивания текста, и может состоять как из каждой части в отдельности,
 так и из всех сразу.
