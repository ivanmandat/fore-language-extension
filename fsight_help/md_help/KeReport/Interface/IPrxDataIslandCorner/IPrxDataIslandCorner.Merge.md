# IPrxDataIslandCorner.Merge


## Синтаксис


Merge: Boolean;


## Описание


Свойство Merge определяет объединять
 или нет уголок таблицы.


## Комментарии


Допустимые значения:


	- True. Уголок будет объединен;


	- False. Уголок объединен
	 не будет. Значение по умолчанию.
