# IUiPrxReportAuditOperationEventArgs.FromWeb


## Синтаксис


FromWeb: Boolean;


## Описание


Свойство FromWeb определяет,
 производиться ли операция из веб-приложения.


## Комментарии


По умолчанию свойство имеет значение False.
 Если операция производится из веб-приложения, то свойство принимает значение
 True.
