# IPrxDataIslandProperties.UseStyleFormatting


## Синтаксис


UseStyleFormatting: Boolean;


## Описание


Свойство UseStyleFormatting
 определяет, какое оформление будет применяться для области данных.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Для ячеек будет применено оформление, настроенное для области данных;


	- False. Для ячеек области
	 данных применяется оформление, заданное для листа.
