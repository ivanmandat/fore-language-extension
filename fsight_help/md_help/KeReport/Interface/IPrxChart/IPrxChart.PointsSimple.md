# IPrxChart.PointsSimple


## Синтаксис


PointsSimple: Boolean;


## Описание


Свойство PointsSimple определяет
 порядок формирования значений для наименования точек диаграммы.


## Комментарии


Допустимые значения:


	- True. При формировании
	 значений используется наименование последнего элемента;


	- False. При формировании
	 значений используется все наименования родителей.


Примечание.
 Совместная установка свойств PointsSimple
 и SeriesSimple определяет состояние
 опции «[Использовать
 простые наименования рядов и точек](UiReport.chm::/Desktop/Diagrams/UiDiagrams_Report_create_source.htm#simple_titles)».
