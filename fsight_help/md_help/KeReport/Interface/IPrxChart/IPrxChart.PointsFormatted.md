# IPrxChart. PointsFormatted


## Синтаксис


PointsFormatted: Boolean;


## Описание


Свойство PointsFormatted определяет,
 использовать ли форматированный текст ячеек для наименований точек диаграммы.


## Комментарии


Под форматированным текстом ячейки понимается формат отображения данных
 ([ITabRange.FormattedText](TabSheet.chm::/Interface/ITabRange/ITabRange.FormattedText.htm)).


Допустимые значения:


	- True. При формировании
	 наименований точек диаграммы будет использоваться форматированный
	 текст ячеек;


	- False. Будет использоваться
	 неформатированный текст.
