# IPrxChart.ExcludedPoints


## Синтаксис


ExcludedPoints: Array;


## Описание


Свойство ExcludedPoints определяет
 диапазон индексов точек, подписи которых исключены из списка отображаемых
 на оси категорий диаграммы.
