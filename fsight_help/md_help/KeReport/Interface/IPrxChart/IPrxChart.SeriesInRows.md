# IPrxChart.SeriesInRows


## Синтаксис


SeriesInRows: Boolean;


## Описание


Свойство SeriesInRows определяет
 способ интерпретации числовых рядов в области данных (диапазон), из которой
 берутся числовые значения для построения диаграммы.


## Комментарии


Допустимые значения:


	- True. Ряды в области
	 данных располагаются в строках. Значение по умолчанию;


	- False. Ряды располагаются
	 в столбцах.
