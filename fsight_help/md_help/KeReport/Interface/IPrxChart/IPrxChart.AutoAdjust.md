# IPrxChart.AutoAdjust


## Синтаксис


AutoAdjust: Boolean;


## Описание


Свойство AutoAdjust определяет,
 будут ли автоматически изменяться данные диаграммы в соответствии с изменениями
 размеров области данных, на которой она построена.


## Комментарии


Допустимые значения свойства:


	- True. При вставке строк/столбцов в область
	 данных, на которой построена диаграмма, диапазон диаграммы автоматически
	 увеличивается, при удалении строк/столбцов - уменьшается;


	- False.
	 Автоматическое изменение диапазонов не используется.


Данное свойство неактуально, если в качестве источника диаграммы используется
 срез.
