# IPrxChart.BeginUpdate


## Синтаксис


BeginUpdate;


## Описание


Метод BeginUpdate отключает
 обновление диаграммы.


## Комментарии


После вызова метода будет отключено любое обновление диаграммы. Также
 не будут генерироваться внутренние события.


Данный метод актуально использовать при изменении большого количества
 параметров/элементов диаграммы для избежания их постоянного обновления,
 которое может занять длительное время.


Применение всех изменений и возобновление обновления осуществляется
 после вызова метода [IPrxChart.EndUpdate](IPrxChart.EndUpdate.htm).
