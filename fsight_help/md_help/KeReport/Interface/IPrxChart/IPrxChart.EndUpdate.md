# IPrxChart.EndUpdate


## Синтаксис


EndUpdate;


## Описание


Метод EndUpdate возобновляет
 обновление диаграммы.


## Комментарии


После вызова метода будут применены все изменения, произведенные в диаграмме
 между вызовами методов [BeginUpdate](IPrxChart.BeginUpdate.htm)
 и EndUpdate.
