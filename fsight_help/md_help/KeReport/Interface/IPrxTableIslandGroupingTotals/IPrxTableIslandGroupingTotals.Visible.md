# IPrxTableIslandGroupingTotals.Visible


## Синтаксис


Visible : Boolean;


## Описание


Свойство Visible определяет
 признак отображения итогов. При значении True
 итоги будут отображены, при значении False
 - нет.


По умолчанию свойство имеет значение True.
