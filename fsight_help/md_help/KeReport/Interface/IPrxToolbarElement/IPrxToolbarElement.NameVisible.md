# IPrxToolbarElement.NameVisible


## Синтаксис


NameVisible: Boolean;


## Описание


Свойство NameVisible определяет
 признак отображения наименования кнопки.


## Комментарии


Допустимые значения:


	- True. Отображать наименование.


	- False. Скрывать наименование.
