# IPrxToolbarElement.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет
 видимость элемента на панели инструментов.


## Комментарии


Допустимые значения:


	- True. Элемент отображается;


	- False. Элемент скрыт.
