# IUiPrxExportEventArgs


## Описание


Интерфейс IUiPrxExportEventArgs
 содержит свойства аргумента события, происходящего после экспорта отчета:


[EventOnAfterExportReport](../IPrxReportUserEvents/IPrxReportUserEvents.EventOnAfterExportReport.htm).

		 информацию о формате, в который экспортируется отчет.
