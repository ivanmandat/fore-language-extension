# IReportObjectMovingEventArgs.Cancel


## Синтаксис


Cancel: Boolean;


## Описание


Свойство Cancel определяет,
 отменять ли перемещение объекта.


## Комментарии


Допустимые значения:


	- True. Перемещение объекта
	 будет отменено;


	- False. Перемещение объекта
	 будет выполнено. Значение по умолчанию.
