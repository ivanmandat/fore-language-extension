# IReportObjectMovingEventArgs.Rect


## Синтаксис

## Описание


Свойство Rect возвращает координаты
 прямоугольной области, занимаемой объектом.


## Комментарии


Для получения изменяемого объекта используйте свойство [IReportObjectMovingEventArgs.Object](IReportObjectMovingEventArgs.Object.htm).
