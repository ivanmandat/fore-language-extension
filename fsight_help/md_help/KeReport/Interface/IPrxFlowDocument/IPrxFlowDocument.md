# IPrxFlowDocument


## Описание


Интерфейс IPrxFlowDocument содержит
 методы объекта регламентного отчета - «Форматированный
 текст».


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [UndoRedo](IPrxFlowDocument.UndoRedo.htm)
		 Свойство UndoRedo определяет
		 параметры стека изменений (отмены/повтора) объекта регламентного
		 отчета - «Форматированный текст».
