# IPrxSliceDimensionInstance


Сборка: Report;


## Описание


Интерфейс IPrxSliceDimensionInstance
 предназначен для работы с измерением среза источника данных регламентного
 отчета.


## Иерархия наследования


           [IEaxDimensionInstance](KeExpress.chm::/Interface/IEaxDimensionInstance/IEaxDimensionInstance.htm)


           IPrxSliceDimensionInstance


## Комментарии


Для работы с коллекцией измерений среза источника данных регламентного
 отчета используйте свойства и методы интерфейса [IPrxSliceDimensions](../IPrxSliceDimensions/IPrxSliceDimensions.htm).
