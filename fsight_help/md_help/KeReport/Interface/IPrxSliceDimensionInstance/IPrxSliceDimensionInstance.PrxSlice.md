## Синтаксис

## Описание


Свойство PrxSlice возвращает
 срез источника данных регламентного отчета.


## Комментарии


Для получения измерения среза источника данных регламентного отчета
 используйте свойство [IPrxSliceDimensionInstance.SliceDimension](IPrxSliceDimensionInstance.SliceDimension.htm).
