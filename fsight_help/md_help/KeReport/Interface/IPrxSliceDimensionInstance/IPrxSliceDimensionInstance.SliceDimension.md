# IPrxSliceDimensionInstance.SliceDimension


## Синтаксис


SliceDimension: [IPrxSliceDimension](../IPrxSliceDimension/IPrxSliceDimension.htm);


## Описание


Свойство SliceDimension возвращает измерение
 среза источника данных регламентного отчета.


## Комментарии


Для получения среза источника данных регламентного отчета используйте
 свойство [IPrxSliceDimensionInstance.PrxSlice](IPrxSliceDimensionInstance.PrxSlice.htm).
