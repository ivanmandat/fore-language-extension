# IPrxTableIslandPager.ExpandAll


## Синтаксис


ExpandAll;


## Описание


Метод ExpandAll выключает режим
 постраничного вывода без необходимости перевычисления реляционной области.


## Комментарии


После выполнения метода значение свойства [Expanded](IPrxTableIslandPager.Expanded.htm)
 становится равным True, а [Enabled](IPrxTableIslandPager.Enabled.htm) - False.


Для проверки признака выключения режима с помощью ExpandAll
 используйте [IPrxTableIslandPager.Expanded](IPrxTableIslandPager.Expanded.htm).
