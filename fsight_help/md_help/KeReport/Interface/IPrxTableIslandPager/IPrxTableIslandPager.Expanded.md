# IPrxTableIslandPager.Expanded


## Синтаксис


Expanded: Boolean;


## Описание


Свойство Expanded возвращает
 признак использования [IPrxTableIslandPager.ExpandAll](IPrxTableIslandPager.ExpandAll.htm).


## Комментарии


После использования [IPrxTableIsland.Recalc](../IPrxTableIsland/IPrxTableIsland.Recalc.htm)
 возвращает False.
