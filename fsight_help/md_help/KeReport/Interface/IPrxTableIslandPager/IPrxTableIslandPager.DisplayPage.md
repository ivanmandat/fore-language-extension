# IPrxTableIslandPager.DisplayPage


## Синтаксис


DisplayPage: Integer;


## Описание


Свойство DisplayPage определяет
 страницу отображения.


## Комментарии


Первая страница соответствует значению 0. Для получения количества страниц
 отображения используйте свойство [IPrxTableIslandPager.Pages](IPrxTableIslandPager.Pages.htm).
