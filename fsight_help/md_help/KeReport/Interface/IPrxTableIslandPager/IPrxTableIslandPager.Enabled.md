# IPrxTableIslandPager.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет,
 установлен ли режим постраничного отображения.


## Комментарии


Включенный постраничный режим не сохраняется с отчетом.


Для использования остальных свойств и методов вычислите реляционную
 область ([IPrxTableIsland.Recalc](../IPrxTableIsland/IPrxTableIsland.Recalc.htm))
 до их применения.
