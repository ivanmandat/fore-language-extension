# IPrxTableIslandPager


Сборка: Report;


## Описание


Интерфейс IPrxTableIslandPager
 содержит параметры работы с режимом постраничного отображения реляционной
 области.


## Иерархия наследования


           IPrxTableIslandPager


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DisplayPage](IPrxTableIslandPager.DisplayPage.htm)


		 Свойство DisplayPage
		 определяет страницу отображения.


		 ![](../../Property_Image.gif)
		 [Enabled](IPrxTableIslandPager.Enabled.htm)


		 Свойство Enabled определяет,
		 установлен ли режим постраничного отображения.


		 ![](../../Property_Image.gif)
		 [Expanded](IPrxTableIslandPager.Expanded.htm)


		 Свойство Expanded возвращает
		 признак использования [IPrxTableIslandPager.ExpandAll](IPrxTableIslandPager.ExpandAll.htm).


		 ![](../../Property_Image.gif)
		 [Pages](IPrxTableIslandPager.Pages.htm)


		 Свойство Pages возвращает
		 количество страниц отображения реляционной области.


		 ![](../../Property_Image.gif)
		 [Rows](IPrxTableIslandPager.Rows.htm)


		 Свойство Rows определяет
		 количество строк в странице.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ExpandAll](IPrxTableIslandPager.ExpandAll.htm)


		 Метод ExpandAll выключает
		 режим постраничного вывода без необходимости перевычисления реляционной
		 области.


		 ![](../../Sub_Image.gif)
		 [Next](IPrxTableIslandPager.Next.htm)


		 Метод Next выполняет
		 переход к следующей странице отображения реляционной области.


		 ![](../../Sub_Image.gif)
		 [Prev](IPrxTableIslandPager.Prev.htm)


		 Метод Prev выполняет
		 переход к предыдущей странице отображения реляционной области.
