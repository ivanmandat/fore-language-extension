# IUiPrxEaxGridCancelEventArgs


Сборка: Report;


## Описание


Интерфейс IUiPrxEaxGridCancelEventArgs
 предназначен для работы с параметрами события [IPrxReportUserEvents.EventOnBeforeRefreshEaxGrid](../IPrxReportUserEvents/IPrxReportUserEvents.EventOnBeforeRefreshEaxGrid.htm).


## Иерархия наследования


           [IUiPrxEaxGridEventArgs](../IUiPrxEaxGridEventArgs/IUiPrxEaxGridEventArgs.htm)


           IUiPrxEaxGridCancelEventArgs


## Комментарии


Для работы с параметрами событий аналитической области данных, происходящих
 после осуществления определенного действия ([EventOnAfterAddEaxGrid](../IPrxReportUserEvents/IPrxReportUserEvents.EventOnAfterAddEaxGrid.htm)
 или [EventOnAfterDeleteEaxGrid](../IPrxReportUserEvents/IPrxReportUserEvents.EventOnAfterDeleteEaxGrid.htm)),
 используйте свойство [IUiPrxEaxGridEventArgs.EaxGrid](../IUiPrxEaxGridEventArgs/IUiPrxEaxGridEventArgs.EaxGrid.htm).
