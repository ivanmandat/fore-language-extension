# IUiPrxEaxGridCancelEventArgs.Cancel


## Синтаксис


Cancel : Boolean;


## Описание


Свойство Cancel определяет,
 будет ли производиться обновление аналитической области данных.


## Комментарии


Допустимые значения:


	- True. Будет произведено
	 обновление аналитической области данных;


	- False. Значение по умолчанию.
	 Не будет произведено обновление аналитической области данных.
