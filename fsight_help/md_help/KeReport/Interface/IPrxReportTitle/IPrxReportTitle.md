# IPrxReportTitle


## Описание


Интерфейс IPrxReportTitle содержит
 свойства заголовка регламентного отчета.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Height](IPrxReportTitle.Height.htm)
		 Свойство Height определяет
		 высоту заголовка регламентного отчета.


## Свойства,
 унаследованные от [IHeaderBase](KeExpress.chm::/Interface/IHeaderBase/IHeaderBase.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Center](KeExpress.chm::/Interface/IHeaderBase/IHeaderBase.Center.htm)
		 Свойство Center возвращает
		 центральную часть заголовка/колонтитула.


		 ![](../../Property_Image.gif)
		 [IsTop](KeExpress.chm::/Interface/IHeaderBase/IHeaderBase.IsTop.htm)
		 Свойство IsTop возвращает
		 тип используемого колонтитула.


		 ![](../../Property_Image.gif)
		 [Left](KeExpress.chm::/Interface/IHeaderBase/IHeaderBase.Left.htm)
		 Свойство Left возвращает
		 левую часть заголовка/колонтитула.


		 ![](../../Property_Image.gif)
		 [Printable](KeExpress.chm::/Interface/IHeaderBase/IHeaderBase.Printable.htm)
		 Свойство Printable
		 определяет признак вывода на печать заголовка/колонтитула.


		 ![](../../Property_Image.gif)
		 [Right](KeExpress.chm::/Interface/IHeaderBase/IHeaderBase.Right.htm)
		 Свойство Right возвращает
		 правую часть заголовка/колонтитула.
