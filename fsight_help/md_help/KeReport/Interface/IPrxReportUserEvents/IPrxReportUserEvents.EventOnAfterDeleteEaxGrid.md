# IPrxReportUserEvents.EventOnAfterDeleteEaxGrid


## Синтаксис


EventOnAfterDeleteEaxGrid(Args: [IUiPrxEaxGridEventArgs](../IUiPrxEaxGridEventArgs/IUiPrxEaxGridEventArgs.htm));


## Параметры


Args. Параметр, позволяющий
 работать с параметрами события.


## Описание


Метод EventOnAfterDeleteEaxGrid
 реализует событие, которое наступает после удаления аналитической области
 данных.


## Комментарии


Для реализации события, которое наступит после добавления аналитической
 области данных, используйте метод [EventOnAfterAddEaxGrid](IPrxReportUserEvents.EventOnAfterAddEaxGrid.htm).
