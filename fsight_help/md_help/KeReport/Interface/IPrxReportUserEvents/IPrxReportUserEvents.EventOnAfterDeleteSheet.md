# IPrxReportUserEvents.EventOnAfterDeleteSheet


## Синтаксис


EventOnAfterDeleteSheet(Args: [IUiPrxSheetEventArgs](../IUiPrxSheetEventArgs/IUiPrxSheetEventArgs.htm));


## Параметры


Args. Параметр, позволяющий
 работать с параметрами события.


## Описание


Метод EventOnAfterDeleteSheet
 реализует событие, которое наступает после удаления листа.


## Комментарии


Для реализации события, которое наступит после добавления листа, используйте
 метод [EventOnAfterAddSheet](IPrxReportUserEvents.EventOnAfterAddSheet.htm).
