# IPrxReportUserEvents.EventOnBeforeTransferDataSheet


## Синтаксис


EventOnBeforeTransferDataSheet(Args: [IUiPrxSheetCancelEventArgs](../IUiPrxSheetCancelEventArgs/IUiPrxSheetCancelEventArgs.htm));


## Параметры


Args - параметр, позволяющий
 определить параметры события.


## Описание


Метод EventOnBeforeTransferDataSheet
 реализует событие, происходящее перед сохранением измененных значений
 областей данных листа отчета в источник.


## Комментарии


Метод предназначен для работы с [областью
 данных](UiReport.chm::/desktop/AreaData/UiReport_AreaData.htm), которая является устаревшим типом таблицы.
 Для работы с аналитической областью данных используйте интерфейс [IEaxDatArea](KeExpress.chm::/Interface/IEaxDataArea/IEaxDataArea.htm).
