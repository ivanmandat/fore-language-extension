# IPrxReportUserEvents.EventOnAfterTransferDataEaxGrid


## Синтаксис


EventOnAfterTransferDataEaxGrid (Args: [IUiPrxEaxGridEventArgs](../IUiPrxEaxGridEventArgs/IUiPrxEaxGridEventArgs.htm));


## Параметры


Args. Параметр, позволяющий
 работать с параметрами события.


## Описание


Метод EventOnAfterTransferDataEaxGrid
 реализует событие, происходящее после сохранения изменённых значений аналитической
 области данных.


## Комментарии


Метод позволяет переопределить системное событие, которое наступает
 после события [ReportEvents.OnAfterTransferDataEaxGrid](../../Class/ReportEvents/ReportEvents.OnAfterTransferDataEaxGrid.htm).
