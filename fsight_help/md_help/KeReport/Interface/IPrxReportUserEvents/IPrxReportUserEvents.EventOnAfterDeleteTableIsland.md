# IPrxReportUserEvents.EventOnAfterDeleteTableIsland


## Синтаксис


EventOnAfterDeleteTableIsland(Args: [IUiPrxTableIslandEventArgs](../IUiPrxTableIslandEventArgs/IUiPrxTableIslandEventArgs.htm));


## Параметры


Args. Параметр, позволяющий
 работать с параметрами события.


## Описание


Метод EventOnAfterDeleteTableIsland
 реализует событие, которое наступает после удаления реляционной области
 данных.


## Комментарии


Для реализации события, которое наступит после добавления реляционной
 области данных, используйте метод [EventOnAfterAddTableIsland](IPrxReportUserEvents.EventOnAfterAddTableIsland.htm).
