# IPrxDataIslandNumberingLevel.IncludeTotals


## Синтаксис


IncludeTotals: [TriState](ForeSys.chm::/Enums/TriState.htm);


## Описание


Свойство IncludeTotals определяет
 необходимость включения нумерации для строк/столбцов, которые содержат
 итоговые данные.


## Комментарии


В [настройках
 нумерации](UiReport.chm::/desktop/AreaData/Param/Order/UiReport_AreaData_Param_numbering.htm) невозможно задать формат числа и начальное
 значение, по умолчанию нумерация начинается с единицы.
