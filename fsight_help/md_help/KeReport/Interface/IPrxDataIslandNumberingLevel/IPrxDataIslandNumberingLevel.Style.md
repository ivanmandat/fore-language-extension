# IPrxDataIslandNumberingLevel.Style


## Синтаксис


Style: [IPrxCellStyle](../IPrxCellStyle/IPrxCellStyle.htm);


## Описание


Свойство Style определяет стиль
 отображения ячеек, в которых располагается нумерация уровня измерения.


## Комментарии


В [настройках
 нумерации](UiReport.chm::/desktop/AreaData/Param/Order/UiReport_AreaData_Param_numbering.htm) невозможно задать формат числа и начальное
 значение, по умолчанию нумерация начинается с единицы.
