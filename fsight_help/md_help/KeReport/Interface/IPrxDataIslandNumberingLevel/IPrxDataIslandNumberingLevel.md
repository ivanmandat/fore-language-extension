# IPrxDataIslandNumberingLevel


## Описание


Интерфейс IPrxDataIslandNumberingLevel
 содержит свойства нумерации строк/столбцов для уровней измерений.


## Комментарии


Интерфейс предназначен для работы с [областью данных](UiReport.chm::/desktop/AreaData/UiReport_AreaData.htm),
 которая является устаревшим типом таблицы. Для работы с аналитической
 областью данных используйте интерфейс [IEaxDatArea](KeExpress.chm::/Interface/IEaxDataArea/IEaxDataArea.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [IncludeTotals](IPrxDataIslandNumberingLevel.IncludeTotals.htm)
		 Свойство IncludeTotals
		 определяет необходимость включения нумерации для строк/столбцов,
		 которые содержат итоговые данные.


		 ![](../../Property_Image.gif)
		 [IsOn](IPrxDataIslandNumberingLevel.IsOn.htm)
		 Свойство IsOn определяет,
		 включена ли автоматическая нумерация для уровня измерения.


		 ![](../../Property_Image.gif)
		 [Style](IPrxDataIslandNumberingLevel.Style.htm)
		 Свойство Style определяет
		 стиль отображения ячеек, в которых располагается нумерация уровня
		 измерения.


		 ![](../../Property_Image.gif)
		 [Type](IPrxDataIslandNumberingLevel.Type.htm)
		 Свойство Type определяет
		 тип нумерации строк/столбцов уровня измерения.
