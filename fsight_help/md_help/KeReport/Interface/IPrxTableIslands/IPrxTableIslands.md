# IPrxTableIslands


## Описание


Интерфейс IPrxTableIslands содержит
 свойства и методы для работы с коллекцией реляционных областей данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Report](IPrxTableIslands.Report.htm)
		 Свойство Report возвращает
		 регламентный отчет, которому принадлежит коллекция реляционных
		 областей данных.


		 ![](../../Property_Image.gif)
		 [Count](IPrxTableIslands.Count.htm)
		 Свойство Count возвращает
		 количество реляционных областей данных регламентного отчета.


		 ![](../../Property_Image.gif)
		 [Item](IPrxTableIslands.Item.htm)
		 Свойство Item возвращает
		 реляционную область данных регламентного отчета.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IPrxTableIslands.Add.htm)
		 Метод Add добавляет
		 реляционную область на лист регламентного отчета.


		 ![](../../Sub_Image.gif)
		 [Clear](IPrxTableIslands.Clear.htm)
		 Метод Clear удаляет
		 все реляционные области регламентного отчета.


		 ![](../../Sub_Image.gif)
		 [FindById](IPrxTableIslands.FindById.htm)
		 Метод FindById осуществляет
		 поиск реляционной области данных по заданному идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IPrxTableIslands.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск реляционной области данных по заданному ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](IPrxTableIslands.Remove.htm)
		 Метод Remove удаляет
		 реляционную область данных по заданному индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IPrxTableIslands.RemoveByKey.htm)
		 Метод RemoveByKey удаляет
		 реляционную область данных по заданному ключу.
