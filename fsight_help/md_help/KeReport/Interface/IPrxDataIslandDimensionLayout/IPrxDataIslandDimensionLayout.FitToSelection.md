# IPrxDataIslandDimensionLayout.FitToSelection


## Синтаксис


FitToSelection: Boolean;


## Описание


Свойство FitToSelection определяет
 признак учета отметки в измерении среза при выводе произвольного заголовка.


## Комментарии


Допустимые значения:


	- True. Выведены только
	 те заголовки, для которых присутствует отметка в срезе. Значение по
	 умолчанию;


	- False. Выведен весь
	 сформированный заголовок независимо от имеющейся отметки.
