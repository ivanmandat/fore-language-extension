# IPrxDataIslandDimensionLayout.MergeWithSelection


## Синтаксис


MergeWithSelection: Boolean;


## Описание


Свойство MergeWithSelection
 определяет признак соединения произвольного заголовка с элементами, по
 которым имеется отметка в срезе, но для которых произвольное наименование
 не определено.


## Комментарии


Допустимые значения:


	- True. К сформированному
	 произвольному заголовку будут также выведены заголовки элементов,
	 по которым не сформировано произвольное наименование, но имеется отметка
	 в срезе;


	- False. Выведен только
	 сформированный заголовок. Значение по умолчанию.
