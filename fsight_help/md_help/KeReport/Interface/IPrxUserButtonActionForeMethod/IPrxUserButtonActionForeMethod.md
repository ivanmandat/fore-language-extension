# IPrxUserButtonActionForeMethod


Сборка: Report;


## Описание


Интерфейс IPrxUserButtonActionForeMethod
 предназначен для настройки Fore-метода в качестве обработчика пользовательской
 кнопки.


## Иерархия наследования


           [IPrxUserButtonAction](../IPrxUserButtonAction/IPrxUserButtonAction.htm)


           IPrxUserButtonActionForeMethod


## Комментарии


Для настройки Fore-метода приведите значение свойства [IPrxUserButton.Action](../IPrxUserButton/IPrxUserButton.Action.htm)
 к интерфейсу IPrxUserButtonActionForeMethod.


Также Fore-метод используется при определении доступности кнопки. Соответствующие
 настройки возвращает свойство [IPrxUserButton.UpdateEnable](../IPrxUserButton/IPrxUserButton.UpdateEnable.htm).
