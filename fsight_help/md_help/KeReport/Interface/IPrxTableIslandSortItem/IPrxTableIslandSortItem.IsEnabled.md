# IPrxTableIslandSortItem.IsEnabled


## Синтаксис


IsEnabled: Boolean;


## Описание


Свойство IsEnabled возвращает
 признак, включена ли сортировка реляционной области данных по определённому
 полю.


## Комментарии


Допустимые значения:


	- True. Сортировка реляционной
	 области данных по определённому полю включена;


	- False. Сортировка реляционной
	 области данных по определённому полю выключена.
