# IPrxTableIslandSortItem.MoveFirst


## Синтаксис


MoveFirst;


## Описание


Метод MoveFirst перемещает условие
 сортировки на одну позицию выше.


## Комментарии


Для перемещения условия сортировки в последнюю позицию используйте метод
 [IPrxTableIslandSortItem.MoveLast](IPrxTableIslandSortItem.MoveLast.htm).
