# IPrxTableIslandSortItem


Сборка: Report;


## Описание


Интерфейс IPrxTableIslandSortItem
 предназначен для работы с параметрами условия сортировки реляционной
 области данных.


## Иерархия наследования


           IPrxTableIslandSortItem


## Комментарии


Для работы с коллекцией условий сортировки реляционной области данных
 используйте свойства и методы интерфейса [IPrxTableIslandSort](../IPrxTableIslandSort/IPrxTableIslandSort.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Descending](IPrxTableIslandSortItem.Descending.htm)
		 Свойство Descending
		 определяет, используется ли для данных реляционной области данных
		 сортировка по убыванию.


		 ![](../../Property_Image.gif)
		 [Enabled](IPrxTableIslandSortItem.Enabled.htm)
		 Свойство Enabled определяет,
		 включена ли сортировка реляционной области данных по определённому
		 полю.


		 ![](../../Property_Image.gif)
		 [Index](IPrxTableIslandSortItem.Index.htm)
		 Свойство Index возвращает
		 индекс условия сортировки в коллекции.


		 ![](../../Property_Image.gif)
		 [InGroups](IPrxTableIslandSortItem.InGroups.htm)
		 Свойство InGroups определяет,
		 используется ли сортировка внутри групп.


		 ![](../../Property_Image.gif)
		 [IsEnabled](IPrxTableIslandSortItem.IsEnabled.htm)
		 Свойство IsEnabled
		 возвращает признак, включена ли сортировка реляционной области
		 данных по определённому полю.


		 ![](../../Property_Image.gif)
		 [LayoutCell](IPrxTableIslandSortItem.LayoutCell.htm)
		 Свойство LayoutCell
		 определяет привязку к ячейке строки отображения.


		 ![](../../Property_Image.gif)
		 [Parent](IPrxTableIslandSortItem.Parent.htm)
		 Свойство Parent возвращает
		 параметры коллекции условий сортировки реляционной области данных.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [MoveFirst](IPrxTableIslandSortItem.MoveFirst.htm)
		 Метод MoveFirst перемещает
		 условие сортировки на одну позицию выше.


		 ![](../../Sub_Image.gif)
		 [MoveLast](IPrxTableIslandSortItem.MoveLast.htm)
		 Метод MoveLast перемещает
		 условие сортировки на одну позицию ниже.


		 ![](../../Sub_Image.gif)
		 [MoveToPosition](IPrxTableIslandSortItem.MoveToPosition.htm)
		 Метод MoveToPosition
		 перемещает условие сортировки в указанную позицию.
