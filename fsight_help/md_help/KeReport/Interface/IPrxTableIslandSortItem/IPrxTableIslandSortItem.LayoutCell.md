## Описание


Свойство LayoutCell определяет
 привязку к ячейке строки отображения.

Для определения, включена ли сортировка реляционной области данных,
 используйте свойство [IPrxTableIslandSortItem.Enabled](IPrxTableIslandSortItem.Enabled.htm).
