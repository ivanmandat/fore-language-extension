# IPrxTableIslandSortItem.Descending


## Синтаксис


Descending: Boolean;


## Описание


Свойство Descending определяет,
 используется ли для данных реляционной области данных сортировка по убыванию.


## Комментарии


Для перемещения условия сортировки в указанную позицию используйте метод
 [IPrxTableIslandSortItem.MoveToPosition](IPrxTableIslandSortItem.MoveToPosition.htm).
