# IPrxTableIslandSortItem.InGroups


## Синтаксис


InGroups: Boolean;


## Описание


Свойство InGroups определяет,
 используется ли сортировка внутри групп.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Используется сортировка внутри групп;


	- False. Не используется
	 сортировка внутри групп.
