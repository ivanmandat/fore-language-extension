# IPrxTableIslandSortItem.Parent


## Синтаксис


Parent: [IPrxTableIslandSort](../IPrxTableIslandSort/IPrxTableIslandSort.htm);


## Описание


Свойство Parent возвращает параметры
 коллекции условий сортировки реляционной области данных.


## Комментарии


Для определения, включена ли сортировка реляционной области данных,
 используйте свойство [IPrxTableIslandSortItem.Enabled](IPrxTableIslandSortItem.Enabled.htm).
