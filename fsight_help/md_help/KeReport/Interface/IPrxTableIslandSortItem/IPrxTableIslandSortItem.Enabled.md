# IPrxTableIslandSortItem.Enabled


## Синтаксис


Enabled: [TriState](ForeSys.chm::/Enums/TriState.htm);


## Описание


Свойство Enabled определяет,
 включена ли сортировка реляционной области данных по определённому
 полю.


## Комментарии


Для получения признака, включена ли сортировка реляционной области данных,
 используйте свойство [IPrxTableIslandSortItem.IsEnabled](IPrxTableIslandSortItem.IsEnabled.htm).
