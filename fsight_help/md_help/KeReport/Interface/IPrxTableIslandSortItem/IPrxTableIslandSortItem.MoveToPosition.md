# IPrxTableIslandSortItem.MoveToPosition


## Синтаксис


MoveToPosition(Position: Integer);


## Параметры


Position. Позиция условия сортировки
 в коллекции.


## Описание


Метод MoveToPosition перемещает
 условие сортировки в указанную позицию.


## Комментарии


Для перемещения условия сортировки в первую позицию используйте метод
 [IPrxTableIslandSortItem.MoveFirst](IPrxTableIslandSortItem.MoveFirst.htm).
