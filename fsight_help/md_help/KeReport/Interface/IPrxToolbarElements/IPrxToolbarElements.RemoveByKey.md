# IPrxToolbarElements.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ удаляемого элемента
 панели инструментов.


## Описание


Метод RemoveByKey удаляет элемента
 с указанным ключом из коллекции.


## Комментарии


Метод возвращает значение True,
 если удаление завершилось успешно и False,
 если удаление по каким-либо причинам недоступно.
