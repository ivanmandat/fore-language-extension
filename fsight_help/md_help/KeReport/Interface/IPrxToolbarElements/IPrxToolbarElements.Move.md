# IPrxToolbarElements.Move


## Синтаксис


Move(OldIndex: Integer; NewIndex: Integer);


## Параметры


OldIndex. Индекс позиции, из
 которой перемещается элемент панели инструментов.


NewIndex. Индекс позиции, в
 которую перемещается элемент панели инструментов.


## Описание


Метод Move осуществляет перемещение
 элемента панели инструментов внутри коллекции.
