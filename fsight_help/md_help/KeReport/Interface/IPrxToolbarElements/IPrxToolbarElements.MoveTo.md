# IPrxToolbarElements.MoveTo


## Синтаксис


MoveTo(OldIndex: Integer; NewIndex: Integer; NewElements:
 [IPrxToolbarElements](IPrxToolbarElements.htm));


## Параметры


OldIndex. Индекс позиции, из
 которой перемещается элемент панели инструментов.


NewIndex. Индекс позиции, в
 которую перемещается элемент панели инструментов.


NewElements. Коллекция элементов
 панели инструментов или пользовательской группы, в которую осуществляется
 перемещение.


## Описание


Метод MoveTo осуществляет перемещение
 элемента панели инструментов в заданную коллекцию элементов.
