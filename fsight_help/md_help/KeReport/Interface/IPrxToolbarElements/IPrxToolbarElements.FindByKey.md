# IPrxToolbarElements.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IPrxToolbarElement](../IPrxToolbarElement/IPrxToolbarElement.htm);


## Параметры


Key. Ключ элемента панели инструментов.


## Описание


Метод FindByKey осуществляет
 поиск элемента панели инструментов по ключу и возвращает его настройки
 в случае удачного поиска.


## Комментарии


Метод возвращает значение Null,
 если элемент с указанным ключом не найден.
