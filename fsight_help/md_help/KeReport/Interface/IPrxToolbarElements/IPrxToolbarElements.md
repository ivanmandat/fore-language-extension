# IPrxToolbarElements


Сборка: Report;


## Описание


Интерфейс IPrxToolbarElements
 предназначен для работы с коллекцией элементов панели инструментов.


## Иерархия наследования


IPrxToolbarElements


## Комментарии


Коллекцию элементов можно получить, используя следующие свойства:


	- [IPrxToolbar.Elements](../IPrxToolbar/IPrxToolbar.Elements.htm)


	- [IPrxUserGroupElement.Elements](../IPrxUserGroupElement/IPrxUserGroupElement.Elements.htm)


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPrxToolbarElements.Count.htm)
		 Свойство Count возвращает
		 количество элементов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IPrxToolbarElements.Item.htm)
		 Свойство Item возвращает
		 настройки элемента панели инструментов с указанным индексом.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IPrxToolbarElements.Add.htm)
		 Метод Add добавляет
		 элемент с указанным типом на панель инструментов.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IPrxToolbarElements.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск элемента панели инструментов по ключу и возвращает его настройки
		 в случае удачного поиска.


		 ![](../../Sub_Image.gif)
		 [Move](IPrxToolbarElements.Move.htm)
		 Метод Move осуществляет
		 перемещение элемента панели инструментов внутри коллекции.


		 ![](../../Sub_Image.gif)
		 [MoveTo](IPrxToolbarElements.MoveTo.htm)
		 Метод MoveTo осуществляет
		 перемещение элемента панели инструментов в заданную коллекцию
		 элементов.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IPrxToolbarElements.RemoveByKey.htm)
		 Метод RemoveByKey удаляет
		 элемента с указанным ключом из коллекции.
