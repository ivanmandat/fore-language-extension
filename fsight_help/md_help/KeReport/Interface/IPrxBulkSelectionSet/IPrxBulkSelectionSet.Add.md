# IPrxBulkSelectionSet.Add


## Синтаксис


Add(Value: [IPrxControl](../IPrxControl/IPrxControl.htm)):
 [IPrxBulkSelection](../IPrxBulkSelection/IPrxBulkSelection.htm);


## Параметры


Value. Элемент управления,
 измерение которого необходимо использовать при пакетной операции.


## Описание


Метод Add добавляет в коллекцию
 отметку измерения, которое связано с указанным элементом управления.
