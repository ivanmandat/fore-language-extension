# IPrxBulkSelectionSet.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс допустимой отметки
 измерения.


## Описание


Метод Remove удаляет указанную
 допустимую отметку измерения.


## Комментарии


Метод возвращает значение True,
 если удаление прошло удачно и False,
 если удаление не допустимо.
