# IUiPrxSheetCancelEventArgs


## Описание


Интерфейс IUiPrxSheetCancelEventArgs
 предназначен для работы с параметрами событий листа отчета, происходящих
 до осуществления определенного действия:


	- [EventOnBeforeRecalcSheet](../IPrxReportUserEvents/IPrxReportUserEvents.EventOnBeforeRecalcSheet.htm);


	- [EventOnBeforeTransferDataSheet](../IPrxReportUserEvents/IPrxReportUserEvents.EventOnBeforeTransferDataSheet.htm);


	- [UiReport.OnBeforeTransferDataSheet](../../Class/UiReport/UiReport.OnBeforeTransferDataSheet.htm).
