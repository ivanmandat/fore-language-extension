# IPrxDataIslandDimensionCommon.IsDrilled


## Синтаксис


IsDrilled : Boolean;


## Описание


Свойство IsDrilled определяет,
 включена ли опция «Расшифровка».


## Комментарии


Для определения состояния опции «Расшифровка» используйте свойство [IPrxDataIslandDimensionCommon.Drilled](IPrxDataIslandDimensionCommon.Drilled.htm).


Допустимые значения:


	- True. Опция «Расшифровка»
	 включена;


	- False. Опция «Расшифровка»
	 отключена.
