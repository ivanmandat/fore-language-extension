# IPrxDataIslandDimensionCommon.NamingCount


## Синтаксис


NamingCount: Integer;


## Описание


Свойство NamingCount определяет
 количество дополнительных наименований элементов измерения.


## Комментарии


Для получения объекта, содержащего наименование элементов измерения,
 используйте свойство [IPrxDataIslandDimensionCommon.Naming](IPrxDataIslandDimensionCommon.Naming.htm).
