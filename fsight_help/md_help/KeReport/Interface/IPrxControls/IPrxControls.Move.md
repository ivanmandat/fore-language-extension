# IPrxControls.Move


## Синтаксис


Move(OldIndex: Integer; NewIndex: Integer);


## Параметры


OldIndex. Индекс позиции элемента
 управления, который необходимо переместить;


NewIndex. Индекс позиции, в
 которую необходимо переместить элемент управления.


## Описание


Метод Move осуществляет перемещение
 элемента управления.
