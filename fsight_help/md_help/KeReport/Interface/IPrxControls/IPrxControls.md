# IPrxControls


## Описание


Интерфейс IPrxControls содержит
 свойства и методы коллекции элементов управления листа регламентного отчета.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPrxControls.Count.htm)
		 Свойство Count возвращает
		 количество элементов управления листа регламентного отчета.


		 ![](../../Property_Image.gif)
		 [Item](IPrxControls.Item.htm)
		 Свойство Item возвращает
		 элемент управления листа регламентного отчета.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindByKey](IPrxControls.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск элемента управления по ключу.


		 ![](../../Sub_Image.gif)
		 [FindById](IPrxControls.FindById.htm)
		 Метод FindById осуществляет
		 поиск элемента управления по идентификатору.


		 ![](../../Sub_Image.gif)
		 [IndexOf](IPrxControls.IndexOf.htm)
		 Метод IndexOf возвращает
		 индекс позиции элемента управления.


		 ![](../../Sub_Image.gif)
		 [Move](IPrxControls.Move.htm)
		 Метод Move осуществляет
		 перемещение элемента управления с текущей позиции на заданную.
