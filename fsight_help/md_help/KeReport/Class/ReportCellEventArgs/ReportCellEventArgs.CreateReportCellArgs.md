# ReportCellEventArgs.CreateReportCellArgs


## Синтаксис


CreateReportCellArgs(Report: [IReportBox](../../Interface/IReportBox/IReportBox.htm);
 Sheet: [IPrxSheet](../../Interface/IPrxSheet/IPrxSheet.htm);
 Row: Integer; Column: Integer);


## Параметры


Report. Компонент, для которого
 генерируется событие;


Sheet. Страница отчета, для
 которой генерируется событие;


Row. Индекс строки, в которой
 расположена ячейка;


Column. Индекс столбца, в которой
 расположена ячейка.


## Описание


Конструктор CreateReportCellArgs
 создает аргумент событий, связанных с ячейкой таблицы.
