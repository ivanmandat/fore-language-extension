# ReportRowEventArgs.CreateReportRowArgs


## Синтаксис


CreateReportRowArgs(Report: [IReportBox](../../Interface/IReportBox/IReportBox.htm); Sheet: [IPrxSheet](../../Interface/IPrxSheet/IPrxSheet.htm); Row: Integer);


## Параметры


Report. Компонент, для которого генерируется событие.


Sheet. Страница отчета, для которой генерируется событие.


Row. Индекс строки, для которого генерируется событие.


## Описание


Конструктор CreateReportRowArgs создает аргумент события, связанного с какой-либо строкой таблицы.
