# UiPrxReportAuditOperationEventArgs.CreateArgs


## Синтаксис


CreateArgs(UiRep: [IUiReport](../../Interface/IUiReport/IUiReport.htm);
 Report: [IPrxReport](../../Interface/IPrxReport/IPrxReport.htm));


## Параметры


UiRep. Компонент,
 для которого сгенерировано событие;


Report. Регламентный отчет,
 действия над которым будут протоколироваться.


## Описание


Конструктор CreateArgs создает
 аргументы события [EventOnAuditOperation](../../Interface/IPrxReportUserEvents/IPrxReportUserEvents.EventOnAuditOperation.htm),
 происходящего при аудите операций: экспорт, экспорт в веб, печать.
