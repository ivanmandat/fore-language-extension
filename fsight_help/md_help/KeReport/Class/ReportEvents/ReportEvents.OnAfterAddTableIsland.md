# ReportEvents.OnAfterAddTableIsland


## Синтаксис


OnAfterAddTableIsland(TableIsland: [IPrxTableIsland](../../Interface/IPrxTableIsland/IPrxTableIsland.htm));


## Параметры


TableIsland. Добавленная реляционная
 область данных.


## Описание


Метод OnAfterAddTableIsland
 реализует событие, наступающее после добавления реляционной области данных
 на лист отчёта.
