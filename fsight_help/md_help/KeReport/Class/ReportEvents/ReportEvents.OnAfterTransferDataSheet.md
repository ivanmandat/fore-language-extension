# ReportEvents.OnAfterTransferDataSheet


## Синтаксис


OnAfterTransferDataSheet(Sheet: [IPrxSheet](../../Interface/IPrxSheet/IPrxSheet.htm));


## Параметры


Sheet. Лист отчета, сгенерировавший
 событие.


## Описание


Метод OnAfterTransferDataSheet
 реализует событие, происходящее после сохранения измененных значений областей
 данных листа отчета.
