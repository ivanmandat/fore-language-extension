# ReportEvents.OnAfterTransferData


## Синтаксис


OnAfterTransferData(Report : [IPrxReport](../../Interface/IPrxReport/IPrxReport.htm));


## Параметры


Report.
 Регламентный отчет, сгенерировавший событие.


## Описание


Метод OnAfterTransferData реализует
 событие, происходящее после сохранения измененных значений всех областей
 данных регламентного отчета.
