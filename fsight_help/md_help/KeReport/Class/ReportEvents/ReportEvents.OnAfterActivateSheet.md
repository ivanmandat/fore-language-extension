# ReportEvents.OnAfterActivateSheet


## Синтаксис


OnAfterActivateSheet(OldSheet: [IPrxSheet](../../Interface/IPrxSheet/IPrxSheet.htm);
 NewSheet: [IPrxSheet](../../Interface/IPrxSheet/IPrxSheet.htm));


## Параметры


OldSheet. Лист регламентного
 отчета, который был активным;


NewSheet. Лист регламентного
 отчета, который стал активным.


## Описание


Метод OnAfterActivateSheet реализует
 событие, происходящее после смены активного листа регламентного отчета.
