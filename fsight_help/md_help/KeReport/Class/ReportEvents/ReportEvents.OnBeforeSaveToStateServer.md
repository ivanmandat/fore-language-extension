# ReportEvents.OnBeforeSaveToStateServer


## Синтаксис


OnBeforeSaveToStateServer(Report: [IPrxReport](../../Interface/IPrxReport/IPrxReport.htm));


## Параметры


Report. Регламентный отчёт,
 сохраняемый на сервер состояний.


## Описание


Метод OnBeforeSaveToStateServer
 реализует событие, наступающее перед сохранением регламентного отчёта
 на сервер состояний.
