## Синтаксис


CreateArgs(Report: [IReportBox](../../Interface/IReportBox/IReportBox.htm);
 Sheet: [IPrxSheet](../../Interface/IPrxSheet/IPrxSheet.htm);
 Editor: [ITabEditor](TabSheet.chm::/Interface/ITabEditor/ITabEditor.htm));


## Параметры


Report. Компонент, для которого
 генерируется событие;


Sheet. Страница отчета, для
 которой генерируется событие;


Editor. Параметры редактора
 ячейки.


## Описание


Конструктор CreateEditorArgs
 создает аргумент события, возникающего при редактировании текста в ячейке.
