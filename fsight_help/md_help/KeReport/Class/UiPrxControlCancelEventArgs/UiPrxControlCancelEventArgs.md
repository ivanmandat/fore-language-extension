# UiPrxControlCancelEventArgs


Сборка: Report;


## Описание


Класс UiPrxControlCancelEventArgs
 предназначен для работы с аргументами событий, происходящих перед удалением
 элемента управления.


## Комментарии


Для создания пользовательских аргументов события, происходящего при
 изменении значения элемента управления, используйте класс [UiPrxControlEventArgs](../UiPrxControlEventArgs/UiPrxControlEventArgs.htm).
