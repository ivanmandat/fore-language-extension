# ReportColumnEventArgs.CreateReportColumnArgs


## Синтаксис


CreateReportColumnArgs(Report: [IReportBox](../../Interface/IReportBox/IReportBox.htm);
 Sheet: [IPrxSheet](../../Interface/IPrxSheet/IPrxSheet.htm);
 Column: Integer);


## Параметры


Report. Компонент, для которого
 генерируется событие;


Sheet. Страница отчета, для
 которой генерируется событие;


Column. Индекс столбца, для
 которого генерируется событие.


## Описание


Конструктор CreateReportColumnArgs
 создает аргумент события, связанного с каким-либо столбцом таблицы.
