# ReportRangeInsertEventArgs


Сборка: Report;


## Описание


Класс ReportRangeInsertEventArgs предназначен
 для работы с аргументами событий, происходящих при добавлении диапазонов/строк/столбцов.


## Комментарии


Для работы с аргументами событий, происходящих при удалении диапазонов/строк/столбцов,
 используйте класс [ReportRangeDeleteEventArgs](../ReportRangeDeleteEventArgs/ReportRangeDeleteEventArgs.htm).
