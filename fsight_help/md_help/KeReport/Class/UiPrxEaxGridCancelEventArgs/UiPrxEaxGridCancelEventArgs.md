# UiPrxEaxGridCancelEventArgs


Сборка: Report;


## Описание


Класс UiPrxEaxGridCancelEventArgs
 предназначен для работы с аргументами событий, происходящих при обновлении
 аналитической области данных.


## Комментарии


Для работы с аргументами событий, происходящих при добавлении/удалении
 аналитической области данных, используйте класс [UiPrxEaxGridEventArgs](../UiPrxEaxGridEventArgs/UiPrxEaxGridEventArgs.htm).
