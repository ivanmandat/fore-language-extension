# ReportBox.OnTableClick


## Синтаксис


OnTableClick(Sender: Object; Args: [IReportEventArgs](../../Interface/IReportEventArgs/IReportEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnTableClick наступает
 при щелчке кнопки мыши по области, расположенной на пересечении заголовков
 строк и столбцов, осуществляющей выделение всей таблицы.
