# ReportBox.OnBeforeEdit


## Синтаксис


OnBeforeEdit(Sender: Object; Args: [IReportBeforeEditEventArgs](../../Interface/IReportBeforeEditEventArgs/IReportBeforeEditEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnBeforeEdit наступает
 перед редактированием данных какой-либо ячейки таблицы.
