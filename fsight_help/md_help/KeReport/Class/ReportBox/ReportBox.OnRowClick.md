# ReportBox.OnRowClick


## Синтаксис


OnRowClick(Sender: Object; Args: [IReportRowEventArgs](../../Interface/IReportRowEventArgs/IReportRowEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnRowClick наступает
 при щелчке кнопки мыши по заголовку одной из строк таблицы.
