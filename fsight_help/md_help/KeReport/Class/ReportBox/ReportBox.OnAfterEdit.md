# ReportBox.OnAfterEdit


## Синтаксис


OnAfterEdit(Sender: Object; Args: [IReportAfterEditEventArgs](../../Interface/IReportAfterEditEventArgs/IReportAfterEditEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnAfterEdit наступает
 после редактирования данных какой-либо ячейки таблицы.
