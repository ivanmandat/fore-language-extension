# ReportBox.OnRowResizing


## Синтаксис


OnRowResizing(Sender: Object; Args: [IReportRowResizingEventArgs](../../Interface/IReportRowResizingEventArgs/IReportRowResizingEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnRowResizing происходит
 при каждом видимом изменении размера строки таблицы.
