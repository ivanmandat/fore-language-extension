# ReportBox.OnCellClick


## Синтаксис


OnCellClick(Sender: Object; Args: [IReportCellEventArgs](../../Interface/IReportCellEventArgs/IReportCellEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnCellClick наступает
 при щелчке кнопки мыши по ячейке таблицы.
