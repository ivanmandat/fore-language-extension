# ReportBox.OnColumnClick


## Синтаксис


OnColumnClick(Sender: Object; Args: [IReportColumnEventArgs](../../Interface/IReportColumnEventArgs/IReportColumnEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnColumnClick наступает
 при щелчке кнопки мыши по заголовку одного из столбцов таблицы.
