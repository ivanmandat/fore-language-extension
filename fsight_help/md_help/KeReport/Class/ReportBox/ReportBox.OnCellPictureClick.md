# ReportBox.OnCellPictureClick


## Синтаксис


OnCellPictureClick(Sender: Object; Args: [IReportCellEventArgs](../../Interface/IReportCellEventArgs/IReportCellEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnCellPictureClick наступает
 при щелчке кнопки мыши по рисунку, расположенному в ячейке таблицы.
