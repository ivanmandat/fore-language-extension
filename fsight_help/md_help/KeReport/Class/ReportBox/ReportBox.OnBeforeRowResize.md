# ReportBox.OnBeforeRowResize


## Синтаксис


OnBeforeRowResize(Sender: Object; Args: [IReportRowEventArgs](../../Interface/IReportRowEventArgs/IReportRowEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnBeforeRowResize наступает
 перед изменением размера строки таблицы.
