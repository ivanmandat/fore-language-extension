# ReportBox.OnBeforeCellChange


## Синтаксис


OnBeforeCellChange(Sender: Object; Args: [IReportBeforeCellChangeEventArgs](../../Interface/IReportBeforeCellChangeEventArgs/IReportBeforeCellChangeEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnBeforeCellChange наступает
 перед изменением значения ячейки таблицы.


## Комментарии


Данное событие происходит перед событием [OnCellChange](ReportBox.OnCellChange.htm)
 и может использоваться для анализа и изменения данных.
