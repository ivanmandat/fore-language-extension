# ReportBox.OnObjectActivate


## Синтаксис


OnObjectActivate(Sender: Object; Args: [IReportObjectEventArgs](../../Interface/IReportObjectEventArgs/IReportObjectEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnObjectActivate наступает
 при активации/деактивации объекта, расположенного на листе регламентного
 отчета.
