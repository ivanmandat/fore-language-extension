# ReportBox.OnCellChange


## Синтаксис


OnCellChange(Sender: Object; Args: [IReportCellChangeEventArgs](../../Interface/IReportCellChangeEventArgs/IReportCellChangeEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnCellChange наступает
 после изменения значения ячейки.
