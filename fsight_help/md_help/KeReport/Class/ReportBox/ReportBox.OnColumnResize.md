# ReportBox.OnColumnResize


## Синтаксис


OnColumnResize(Sender: Object; Args: [IReportColumnResizeEventArgs](../../Interface/IReportColumnResizeEventArgs/IReportColumnResizeEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnColumnResize происходит
 после изменения размера столбца таблицы.
