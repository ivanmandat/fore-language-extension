# ReportBox.OnAutoFilter


## Синтаксис


OnAutoFilter(Sender: Object; Args: [IReportAutoFilterEventArgs](../../Interface/IReportAutoFilterEventArgs/IReportAutoFilterEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnAutoFilter наступает
 при изменении условия автофильтра.
