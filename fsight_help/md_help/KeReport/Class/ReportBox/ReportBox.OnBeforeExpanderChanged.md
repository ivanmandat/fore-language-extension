# ReportBox.OnBeforeExpanderChanged


## Синтаксис


OnBeforeExpanderChanged(Sender: Object; Args: [IReportBeforeExpanderChangedEventArgs](../../Interface/IReportBeforeExpanderChangedEventArgs/IReportBeforeExpanderChangedEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnBeforeExpanderChanged
 наступает перед изменением состояния экспандера в ячейке таблицы.
