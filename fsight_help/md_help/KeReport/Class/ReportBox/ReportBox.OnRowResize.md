# ReportBox.OnRowResize


## Синтаксис


OnRowResize(Sender: Object; Args: [IReportRowResizeEventArgs](../../Interface/IReportRowResizeEventArgs/IReportRowResizeEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnRowResize происходит
 после изменения размера строки таблицы.
