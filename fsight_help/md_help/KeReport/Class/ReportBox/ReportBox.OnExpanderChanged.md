# ReportBox.OnExpanderChanged


## Синтаксис


OnExpanderChanged(Sender: Object; Args: [IReportExpanderChangedEventArgs](../../Interface/IReportExpanderChangedEventArgs/IReportExpanderChangedEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnExpanderChanged наступает
 при изменении состояния экспандера в ячейке таблицы.
