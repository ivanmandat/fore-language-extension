# ReportBox.OnScaleChange


## Синтаксис


OnScaleChange(Sender: Object; Args: [IReportScaleChangeEventArgs](../../Interface/IReportScaleChangeEventArgs/IReportScaleChangeEventArgs.htm));


## Параметры


Sender - . Параметр, возвращающий
 компонент, сгенерировавший событие;


Args - . Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnScaleChange наступает
 при изменении масштаба таблицы.
