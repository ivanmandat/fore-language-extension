# ReportBox.OnSelectionChange


## Синтаксис


OnSelectionChange(Sender: Object; Args: [IReportEventArgs](../../Interface/IReportEventArgs/IReportEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnSelectionChange наступает
 при изменении выделения ячеек или групп ячеек таблицы.
