# ReportObjectMovingEventArgs


Сборка: Report;


## Описание


Класс ReportObjectMovingEventArgs
 реализует аргумент события, происходящего при перемещении объекта.


## Комментарии


Для реализации аргумента события, происходящего при каждом изменении
 размера объекта, используйте класс [ReportObjectResizingEventArgs](../ReportObjectResizingEventArgs/ReportObjectResizingEventArgs.htm).
