# UiReport.OnBeforeTransferDataSheet


## Синтаксис


OnBeforeTransferDataSheet(Sender: Object; Args:
 [IUiPrxSheetCancelEventArgs](../../Interface/IUiPrxSheetCancelEventArgs/IUiPrxSheetCancelEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnBeforeTransferDataSheet
 наступает перед сохранением измененных значений областей данных листа
 отчета.
