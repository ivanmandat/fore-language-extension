# UiReport.OnAfterActivateSheet


## Синтаксис


OnAfterActivateSheet(Sender: Object; Args: [IUiPrxActivateSheetEventArgs](../../Interface/IUiPrxActivateSheetEventArgs/IUiPrxActivateSheetEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnAfterActivateSheet
 наступает после изменения активного листа регламентного отчёта.
