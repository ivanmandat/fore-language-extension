# UiReport.OnCellChange


## Синтаксис


OnCellChange(Sender: Object; Args: [IUiPrxCellChangeEventArgs](../../Interface/IUiPrxCellChangeEventArgs/IUiPrxCellChangeEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnCellChange наступает
 после изменения значения в ячейке.
