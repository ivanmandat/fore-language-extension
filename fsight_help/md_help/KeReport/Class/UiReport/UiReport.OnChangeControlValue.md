# UiReport.OnChangeControlValue


## Синтаксис


OnChangeControlValue(Sender: Object; Args: [IUiPrxControlEventArgs](../../Interface/IUiPrxControlEventArgs/IUiPrxControlEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnChangeControlValue
 наступает при изменении значения элемента управления регламентного отчёта.
