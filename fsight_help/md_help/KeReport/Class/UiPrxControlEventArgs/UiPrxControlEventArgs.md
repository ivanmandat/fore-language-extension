# UiPrxControlEventArgs


Сборка: Report;


## Описание


Класс UiPrxControlEventArgs
 предназначен для создания пользовательских аргументов события, происходящего
 при изменении значения элемента управления.


## Комментарии


Для работы с аргументами событий, происходящих перед удалением элемента
 управления, используйте класс [UiPrxControlCancelEventArgs](../UiPrxControlCancelEventArgs/UiPrxControlCancelEventArgs.htm).
