# ReportRangeDeleteEventArgs


Сборка: Report;


## Описание


Класс ReportRangeDeleteEventArgs
 предназначен для работы с аргументами событий, происходящих при удалении
 диапазонов/строк/столбцов.


## Комментарии


Для работы с аргументами событий, происходящих при добавлении диапазонов/строк/столбцов,
 используйте класс [ReportRangeInsertEventArgs](../ReportRangeInsertEventArgs/ReportRangeInsertEventArgs.htm).
