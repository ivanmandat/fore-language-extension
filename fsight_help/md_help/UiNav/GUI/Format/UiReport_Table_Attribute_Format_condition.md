# Форматирование по условию


Вкладка «Форматирование по условию»
 предназначена для настройки расширенного форматирования ячеек по заданному
 условию.


Условное форматирование обеспечивает наглядность при анализе данных.


Примечание.
 Условное форматирование перекрывает оформление установленного [стиля
 таблицы](UiAnalyticalArea.chm::/TableView/Formatting/Table_styles.htm), но не перекрывает [настройки
 форматирования ячеек](UiReport.chm::/desktop/Table/Attribute/UiReport_Table_Attribute.htm). Сначала применяются настройки
 форматирования ячеек, затем условное форматирование.


Вкладка имеет вид:


![](Page%207.gif)


[![](../../Opened.gif)![](../../Closed.gif)Добавление нового
 условия в список](javascript:TextPopup(this))


	Для добавления нового условия в список:


		- нажмите кнопку «Добавить»;


		- нажмите клавишу INSERT.


	Будет открыт диалог настройки условия «[Параметры условия форматирования](UiAnalyticalArea.chm::/Analysis/Format/UiReport_Table_Attribute_Format_conditionParams.htm)».
	 Созданные условия применяются в порядке их следования в списке. Для
	 изменения порядка используйте кнопки «Вверх»
	 и «Вниз» или соответствующие
	 пункты контекстного меню. Также для изменения порядка можно воспользоваться
	 механизмом перетаскивания.


[![](../../Opened.gif)![](../../Closed.gif)Редактирование
 выбранного условия](javascript:TextPopup(this))


	Для редактирования выбранного условия из списка:


		- нажмите кнопку «Редактировать»;


		- дважды щёлкните по условию;


		- выполните команду контекстного меню «Редактировать».


	Будет открыт диалог настройки условия «[Параметры условия форматирования](UiAnalyticalArea.chm::/Analysis/Format/UiReport_Table_Attribute_Format_conditionParams.htm)».


[![](../../Opened.gif)![](../../Closed.gif)Удаление выбранного
 условия](javascript:TextPopup(this))


	Для удаления выбранного условия:


		- нажмите кнопку «Удалить»;


		- нажмите клавишу DELETE;


		- выполните команду контекстного меню «Удалить».
