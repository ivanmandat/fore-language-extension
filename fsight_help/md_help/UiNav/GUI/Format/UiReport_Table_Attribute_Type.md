# Шрифт


Вкладка «Шрифт» имеет вид:


![](Page%202.gif)


Задайте параметры:


	- Шрифт. Задаёт один из
	 установленных в операционной системе шрифтов;


Примечание.
 «Форсайт. Аналитическая платформа»
 поддерживает шрифты типа TrueType
 и ограниченно поддерживает шрифты типа OpenType.
 При попытке использовать неподдерживаемый шрифт или шрифт, не установленный
 на компьютере, где выполняется приложение, вместо него будет подставлен
 шрифт Microsoft Sans Serif. Данное правило распространяется на объекты
 native GDI+, класс Font и метод DrawString.


	- Размер. Задаёт размер
	 шрифта. Допускается ввод дробных значений. Если дробная часть меньше
	 0,25, то выполняется округление в меньшую сторону до ближайшего целого
	 значения. Если дробная часть равна 0,75 или больше, то выполняется
	 округление в большую сторону до ближайшего целого значения. В остальных
	 случаях дробная часть принимает значение 0,5;


	- Начертание. Задаёт вариант
	 начертания:


	-


		- жирный;


		- курсив;


		- подчёркнутый;


		- зачёркнутый;


	- Цвет. Задаёт цвет текста.
	 В раскрывающейся палитре выберите цвет или нажмите на кнопку «Авто» для выбора цвета по умолчанию.
	 При нажатии на кнопку «Другой»
	 будет открыта расширенная палитра цветов;


Примечание.
 Кнопка «Авто» доступна при настройке
 формата ячейки.


	- Набор. Задаёт набор
	 используемых символов, поддерживаемых выбранным шрифтом. Параметр
	 присутствует при настройке шрифта в регламентном отчёте;


	- Образец. В области «Образец» расположен текст, оформленный
	 согласно параметрам, установленным на вкладке «Шрифт»
	 в данный момент.


Параметры, доступные в определённых инструментах:


	- Прозрачность. Задаёт
	 процент прозрачности текста, где 100% соответствует полной прозрачности,
	 0% - полной непрозрачности текста. Параметр присутствует только для
	 объектов рабочего пространства.


Примечание.
 Если в ячейке расположена [гиперссылка](UiReport_Table_Attribute_Hyperlink.htm),
 то на вкладке будут отображены ее собственные настройки (по умолчанию
 подчеркнутый текст синего цвета) независимо от использования [единого
 оформления гиперссылок](uireport.chm::/desktop/Reports/UiReport_Reports_Param3.htm#formatting). Например, если для гиперссылки
 был задан зеленый цвет, а после включено единое оформление, то на вкладке
 по-прежнему будет отображен зеленый цвет, но в ячейке гиперссылка будет
 отображена в соответствии с глобальными настройками. После отключения
 режима единого оформления гиперссылка будет отображена зеленым цветом.
