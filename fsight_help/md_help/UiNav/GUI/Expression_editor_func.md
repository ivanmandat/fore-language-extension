# Функции, доступные в редакторе
 выражения


При создании и редактировании выражения допускается использование различных
 функций, которые разделены на категории:


	- [Агрегация](Function/Aggregation/Func_Aggregation.htm).
	 Функции агрегации;


	- [Арифметические](Function/Arithmetic/Func_Arithmetic.htm).
	 Арифметические функции;


	- [Атрибутный
	 доступ](Function/ABAC/Func_ABAC.htm). Функции для создания дополнительного условия проверки
	 доступа при использовании [атрибутного метода
	 разграничения доступа](Admin.chm::/04_SecurityPolicy/Admin_ABAC.htm);


Примечание.
 Категория доступна только при создании дополнительного условия проверки
 доступа в окне «[Создание условия](Admin.chm::/04_SecurityPolicy/Creating_conditions.htm)».


	- Все функции. Все существующие
	 функции;


	- [Итоги](Function/Totals/Func_Totals.htm).
	 Функции для работы с итогами;


	- [Логические](Function/Logical/Func_Logical.htm).
	 Логические функции;


	- [Методы
	 R](Function/R_methods/Func_R_methods.htm). Функции, при расчёте которых используется пакет
	 R;


Примечание.
 Расчёт методов R возможен, если в «Форсайт. Аналитическая платформа»
 [подключен пакет
 R](../02_Navigator/CommonSettings/Integration.htm#r).


	- [Преобразования](Function/Transformations/Func_Transformations.htm).
	 Функции преобразования;


	- [Прогнозирование](Function/Forecasting/Func_Forecasting.htm).
	 Функции прогнозирования;


	- [Прочие](Function/Other/Func_Other.htm).
	 Прочие функции;


	- [Работа
	 с Java](Function/Java/Methods_Java.htm). Статические методы, написанные на языке Java;


	- [Работа
	 с Python](Function/Python/Func_Python.htm). Функции, написанные на языке Python;


	- [Регрессия](Function/Regression/Func_Regression.htm).
	 Функции регрессии;


	- [Сглаживание](Function/Smoothing/Func_Smoothing.htm).
	 Функции сглаживания;


	- [Текстовые](Function/Text/Func_Text.htm).
	 Функции для работы со строками;


	- Пользовательские функции.
	 Пользовательские методики расчёта;


Примечание.
 Категория доступна, если в редактор выражения были подключены пользовательские
 функции. Для получения подробной информации
 обратитесь к разделу «[Подключение пользовательских
 функций](ExpressionEditor_UserFunc.htm)».
