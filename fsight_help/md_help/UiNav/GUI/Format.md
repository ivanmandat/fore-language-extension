# Работа с диалогом форматирования


Во многих инструментах продукта «Форсайт. Аналитическая платформа»
 для настройки параметров ячеек, текста, стиля, объектов и других элементов
 используется диалог форматирования, в котором задаются формат числа, оформление,
 параметры шрифта, заливку, тень, форматирование по условию и пр. Диалог
 форматирования может содержать различный набор вкладок в зависимости от
 того, откуда был открыт диалог.


Примечание.
 Диалог форматирования доступен только в настольном приложении.


Диалог форматирования для ячеек
 или стиля ячеек может содержать вкладки:


	- [Стили](Format/UiReport_Table_Attribute_Style.htm);


	- [Формат
	 числа](Format/UiReport_Table_Attribute_Format.htm);


	- [Шрифт](Format/UiReport_Table_Attribute_Type.htm);


	- [Заливка](Format/UiReport_Table_Attribute_View.htm);


	- [Выравнивание](Format/UiReport_Table_Attribute_Arrange.htm);


	- [Границы](Format/UiReport_Table_Attribute_Border.htm);


	- [Прочее](Format/UiReport_Table_Attribute_Others.htm);


	- [Форматирование
	 по условию](Format/UiReport_Table_Attribute_Format_condition.htm);


	- [Гиперссылка](Format/UiReport_Table_Attribute_Hyperlink.htm).
