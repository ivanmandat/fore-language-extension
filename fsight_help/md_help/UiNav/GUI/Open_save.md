# Открытие и сохранение объектов из инструментов продукта «Форсайт. Аналитическая платформа»


При работе в инструментах продукта «Форсайт. Аналитическая платформа»
 может потребоваться открыть различные файлы или сохранить результат работы
 в виде отчёта.


Для открытия объекта выполните команду «Открыть»
 в главном меню инструмента. В настольном приложении вместо команды можно
 использовать сочетание клавиш CTRL+O. Откроется диалог, в котором выберите
 открываемый объект. Выбранный объект будет открыт в инструменте, если
 у пользователя есть соответствующие права.


Для сохранения объекта выполните команду «Сохранить»
 в главном меню инструмента. В настольном приложении вместо команды можно
 использовать сочетание клавиш CTRL+S. Если объект сохраняется впервые,
 то будет открыт диалог, в котором выберите расположение сохраняемого объекта.
 Выбранный объект будет сохранён, если у пользователя есть соответствующие
 права.


Для сохранения объекта с новым наименованием выполните команду «Сохранить как» в главном меню инструмента.
 Будет открыт диалог, в котором выберите расположение сохраняемого объекта.
 Выбранный объект будет сохранён, если у пользователя есть соответствующие
 права.
