# Применение стилей оформления


Стили оформления - набор параметров, предназначенный для создания
 единого оформления объектов. Стиль можно применять как для отдельных объектов,
 так и для групп объектов. Изменения параметров стиля автоматически применяются
 ко всем объектам, использующим данный стиль.


Примечание.
 Работа со стилями оформления доступна только в настольном приложении.


Работа со стилями ведётся в окне «Стили»:


![](Styles_Common.gif)


В левой части окна расположен список существующих стилей, а в правой -
 описание и образец выбранного стиля.


Доступные операции со стилями:


	- Создание стиля. Нажмите
	 кнопку «Добавить». Будет открыт
	 диалог «Параметры стиля» для
	 редактирования параметров создаваемого стиля. Вкладки данного диалога
	 аналогичны вкладкам [диалога форматирования объектов](Format.htm);


	- Редактирование выбранного стиля.
	 Нажмите кнопку «Изменить».
	 Отредактируйте параметры стиля в открывшемся диалоге «Параметры
	 стиля»;


	- Удаление выбранного стиля.
	 Стиль можно удалить, если он не используется ни одним объектом. Для
	 удаления стиля нажмите кнопку «Удалить».
