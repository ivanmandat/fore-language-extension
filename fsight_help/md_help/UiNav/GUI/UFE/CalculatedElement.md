# Просмотр формулы вычисляемого элемента измерения


При вставке оператора в списке элементов присутствуют в том числе и
 [вычисляемые
 элементы измерения](UiSelection.chm::/Selection/Dimension_edit.htm#edit_dim_element). Соответственно, их тоже можно выбрать
 для формирования отметки операнда.


В настольном приложении доступен просмотр формулы вычисляемого элемента
 измерения, если он входит в состав операнда. Для этого:


	- Выполните команду «Показать
	 формулу» в контекстном меню операнда. Если команда отсутствует,
	 значит операнд не содержит вычисляемых элементов измерения.


Если операнд не был сохранён, то будет предложено
 его сохранить. После этого в области выражения будет отображена формула
 вычисляемого элемента.


	- Для возврата к первоначальному операнду выполните команду «Предыдущая формула» в контекстном
	 меню формулы вычисляемого элемента.
