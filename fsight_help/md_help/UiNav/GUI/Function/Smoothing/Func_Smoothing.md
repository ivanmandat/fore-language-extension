# Сглаживание


При работе с формулами возможно использование предустановленных функций,
 приведённых в таблице:


		 Наименование
		 Краткое описание


		 [Bpf](Func_Smoothing_Bpf.htm)


		 Осуществляет моделирование значений переменной фильтром Бакстера-Кинга.


		 [BpfR](../R_methods/Func_R_methods_BpfR.htm)


		  Осуществляет моделирование значений переменной фильтром
		 Бакстера-Кинга. Для расчёта используется R.


		 [Census1](Func_Smoothing_Census1.htm)


		 Осуществляет сглаживание переменной помощью метода Census1.


		 [Hpf](Func_Smoothing_Hpf.htm)


		 Осуществляет сглаживание переменной с использованием фильтра
		 Ходрика-Прескотта (параметр сглаживания - лямбда).


		 [HpfP](Func_Smoothing_HpfP.htm)


		 Осуществляет сглаживание переменной с использованием фильтра
		 Ходрика-Прескотта (параметр сглаживания - степень).


		 [HpfR](../R_methods/Func_R_methods_HpfR.htm)


		  Осуществляет сглаживание переменной с использованием
		 фильтра Ходрика-Прескотта (параметр сглаживания - степень). Для
		 расчёта используется R.


		 [Lrxf](Func_Smoothing_Lrxf.htm)


		 Осуществляет моделирование переменной с помощью LRX-фильтра.


		 [MedianSmooth](Func_Smoothing_MedianSmooth.htm)


		 Осуществляет моделирование переменной с помощью медианного
		 сглаживания.


		 [MedianSmoothR](../R_methods/Func_R_methods_MedianSmoothR.htm)


		  Осуществляет моделирование переменной с помощью медианного
		 сглаживания. Для расчёта используется R.


		 [MovAvg](Func_Smoothing_MovAvg.htm)


		 Осуществляет преобразование переменной методом скользящего
		 среднего.


		 [MovAvgR](../R_methods/Func_R_methods_MovAvgR.htm)


		  Осуществляет преобразование переменной методом скользящего
		 среднего. Для расчёта используется R.


		 [X11](Func_Smoothing_X11.htm)


		 Осуществляет сезонную декомпозицию и корректировку данных.

		Примечание.
		 Метод X11 поддерживается только в ОС Windows.
