# Replace


## Синтаксис


Replace(Source: String, OldValue: String, NewValue:
 String)


## Параметры


Source. Исходная строка;


OldValue. Подстрока, которую
 требуется заменить;


NewValue. Подстрока, на которую
 требуется заменить.


## Описание


Возвращает строку, в которой все вхождения
 указанной подстроки заменены на указанную строку.


## Комментарии


Если подстрока не найдена, то исходная строка не изменяется.


Метод является регистрозависимым.


## Пример


		 Формула
		 Результат
		 Применение


		 = Replace("Version 8.5", "8.5",
		 "9.0")
		 Version 9.0


		 Можно использовать в формулах моделей контейнера
		 моделирования, в формулах вычисляемых рядов базы данных временных
		 рядов, в формулах моделей контейнера моделирования, являющегося
		 дочерним по отношению к базе данных временных рядов и в редакторе
		 выражения.


		 = Replace("Version 8.5", "version",
		 "Edition")
		 Version 8.5
