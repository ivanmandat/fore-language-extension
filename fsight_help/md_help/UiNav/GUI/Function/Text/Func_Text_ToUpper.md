# ToUpper


## Синтаксис


ToUpper(Source: String)


## Параметры


Source. Строка.


## Описание


Возвращает указанную строку, преобразованную
 в верхний регистр.


## Комментарии


Для получения строки, преобразованной в нижний регистр, используйте
 функцию [ToLower](Func_Text_ToLower.htm).


## Пример


		 Формула
		 Результат
		 Применение


		 = ToUpper("CaLcUlAtOr")
		 CALCULATOR


		 Можно использовать в формулах моделей контейнера
		 моделирования, в формулах вычисляемых рядов базы данных временных
		 рядов, в формулах моделей контейнера моделирования, являющегося
		 дочерним по отношению к базе данных временных рядов и в редакторе
		 выражения.


		 = ToUpper("ProductVersion")
		 PRODUCTVERSION
