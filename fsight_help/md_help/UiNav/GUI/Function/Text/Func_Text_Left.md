# Left


## Синтаксис


Left(Source: String, Count: Integer)


## Параметры


Source. Строка;


Count. Количество символов с
 левого края строки, которое нужно получить.


## Описание


Возвращает указанное количество символов
 с левого края строки.


## Комментарии


Для получения символов с правого края строки используйте функцию [Right](Func_Text_Right.htm).


## Пример


		 Формула
		 Результат
		 Применение


		 = Left("Calculator", 4)
		 Calc


		 Можно использовать в формулах моделей контейнера
		 моделирования, в формулах вычисляемых рядов базы данных временных
		 рядов, в формулах моделей контейнера моделирования, являющегося
		 дочерним по отношению к базе данных временных рядов и в редакторе
		 выражения.


		 = Left("Calculator", 2)
		 Ca
