# Find


## Синтаксис


Find(Source: String, SubStr: String, Start: Integer)


## Параметры


Source. Исходная строка;


SubStr. Искомая подстрока;


Start. Номер позиции строки,
 с которой нужно начинать поиск. Необязательный параметр. По умолчанию
 параметр равен 1.


## Описание


Возвращает индекс позиции подстроки в строке.


## Комментарии


Если указанная подстрока в исходной строке не найдена, то метод вернёт
 0.


Если указан необязательный параметр Start,
 то функция игнорирует все экземпляры подстроки, которые появляются перед
 началом позиции с индексом Start.


Нумерация символов в строке начинается с единицы.


Поиск выполняется с учетом регистра.


## Пример


		 Формула
		 Результат
		 Применение


		 = Find("Calculator", "a",
		 4)
		 7

		 рядов, в формулах моделей контейнера моделирования, являющегося
		 дочерним по отношению к базе данных временных рядов и в редакторе
		 выражения.

		 2
