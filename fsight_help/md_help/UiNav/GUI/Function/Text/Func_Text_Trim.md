# Trim


## Синтаксис


Trim(Source: String)


## Параметры


Source. Строка.


## Описание


Возвращает строку с удаленными пробелами
 в начале и конце.


## Комментарии


Для удаления пробелов только в начале строки используйте функцию [TrimStart](Func_Text_TrimStart.htm),
 только в конце строки - [TrimEnd](Func_Text_TrimEnd.htm).


## Пример


		 Формула
		 Результат
		 Применение


		 = Trim("  Calculator   ")
		 «Calculator»


		 Можно использовать в формулах моделей контейнера
		 моделирования, в формулах вычисляемых рядов базы данных временных
		 рядов, в формулах моделей контейнера моделирования, являющегося
		 дочерним по отношению к базе данных временных рядов и в редакторе
		 выражения.


		 = Trim("Calculator   ")
		 «Calculator»
