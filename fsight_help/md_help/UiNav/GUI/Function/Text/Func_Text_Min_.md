# Min_


## Синтаксис


Min_(A: String, B: String)


## Параметры


A. Первая сравниваемая строка;


B. Вторая сравниваемая строка.


## Описание


Возвращает минимум из двух строк, используя
 посимвольное сравнение.


## Комментарии


Сравнение выполняется учетом регистра.


Функция возвращает Null, если
 хотя бы один из параметров равен Null.


Для получения максимума из двух строк используйте функцию [Max_](Func_Text_Max_.htm).


## Пример


		 Формула
		 Результат
		 Применение


		 = Min_("Apple", "Banana")
		 Apple


		 Можно использовать в формулах моделей контейнера
		 моделирования, в формулах вычисляемых рядов базы данных временных
		 рядов, в формулах моделей контейнера моделирования, являющегося
		 дочерним по отношению к базе данных временных рядов и в редакторе
		 выражения.


		 = Min_("Apple", "apple")
		 Apple
