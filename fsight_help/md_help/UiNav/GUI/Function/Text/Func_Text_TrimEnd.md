# TrimEnd


## Синтаксис


TrimEnd(Source: String)


## Параметры


Sourse. Строка.


## Описание


Возвращает строку с удаленными пробелами
 в конце строки.


## Комментарии


Для удаления пробелов в начале строки используйте функцию [TrimStart](Func_Text_TrimStart.htm), для удаления
 пробелов в начале и конце строки - функцию [Trim](Func_Text_Trim.htm).


## Пример


		 Формула
		 Результат
		 Применение


		 = TrimEnd("Calculator   ")
		 «Calculator»


		 Можно использовать в формулах моделей контейнера
		 моделирования, в формулах вычисляемых рядов базы данных временных
		 рядов, в формулах моделей контейнера моделирования, являющегося
		 дочерним по отношению к базе данных временных рядов и в редакторе
		 выражения.


		 = TrimEnd("   Calculator   ")
		 «    Calculator»
