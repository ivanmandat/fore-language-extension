# Length_


## Синтаксис


Length_(Source: String)


## Параметры


Source. Строка.


## Описание


Возвращает количество символов в указанной
 строке.


## Комментарии


Для получения определенного количества символов строки используйте функции
 [Left](Func_Text_Left.htm)
 и [Right](Func_Text_Right.htm).


## Пример


		 Формула
		 Результат
		 Применение


		 = Length_("Calculator")
		 10


		 Можно использовать в формулах моделей контейнера
		 моделирования, в формулах вычисляемых рядов базы данных временных
		 рядов, в формулах моделей контейнера моделирования, являющегося
		 дочерним по отношению к базе данных временных рядов и в редакторе
		 выражения.


		 = Length_("Monday")
		 6
