# Right


## Синтаксис


Right(Source: String, Count: Integer)


## Параметры


Source. Строка;


Count. Количество символов с
 правого края строки, которое нужно получить.


## Описание


Возвращает указанное количество символов
 с правого края строки.


## Комментарии


Для получения символов с левого края строки используйте функцию [Left](Func_Text_Left.htm).


## Пример


		 Формула
		 Результат
		 Применение


		 = Right("Calculator", 4)
		 ator


		 Можно использовать в формулах моделей контейнера
		 моделирования, в формулах вычисляемых рядов базы данных временных
		 рядов, в формулах моделей контейнера моделирования, являющегося
		 дочерним по отношению к базе данных временных рядов и в редакторе
		 выражения.


		 = Right("Calculator", 2)
		 or
