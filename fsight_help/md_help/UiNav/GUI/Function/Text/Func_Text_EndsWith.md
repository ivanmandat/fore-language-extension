# EndsWith_


## Синтаксис


EndsWith_(Source: String, SubStr: String)


## Параметры


Source. Исходная строка;


SubStr. Искомая подстрока.


## Описание


Возвращает признак того, что исходная строка
 оканчивается искомой подстрокой.


## Комментарии


Допустимые значения:


	- True. Исходная строка
	 оканчивается искомой подстрокой;


	- False. Исходная строка
	 не оканчивается искомой подстрокой.


Пробелы в конце строки и подстроки игнорируются. Поиск подстроки в исходной
 строке выполняется с учетом регистра.


Для получения признака того, что исходная строка начинается с искомой
 подстроки используйте функцию [StartsWith_](Func_Text_StartsWith.htm).
