# StartsWith_


## Синтаксис


StartsWith_(Source: String, SubStr: String)


## Параметры


Source. Исходная строка;


SubStr. Искомая подстрока.


## Описание


Возвращает признак того, что исходная
 строка начинается с искомой подстроки.


## Комментарии


Допустимые значения:


	- True. Исходная строка
	 оканчивается искомой подстрокой;


	- False. Исходная строка
	 не оканчивается искомой подстрокой.


Пробелы в конце строки и подстроки игнорируются. Поиск подстроки в исходной
 строке выполняется с учетом регистра.


Для получения признака того, что исходная строка оканчивается искомой
 подстрокой используйте функцию [EndsWith_](Func_Text_EndsWith.htm).


## Пример


		 Формула
		 Результат
		 Применение


		 = StartsWith_("Calculator", "calc")
		 False


		 Можно использовать в формулах моделей контейнера
		 моделирования, в формулах вычисляемых рядов базы данных временных
		 рядов, в формулах моделей контейнера моделирования, являющегося
		 дочерним по отношению к базе данных временных рядов и в редакторе
		 выражения.


		 = StartsWith_("Calculator", "    Calc")
		 True
