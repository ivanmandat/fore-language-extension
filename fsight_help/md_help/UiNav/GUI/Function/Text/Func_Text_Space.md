# Space


## Синтаксис


Space(Number: Integer)


## Параметры


Number. Требуемое количество
 пробелов.


## Описание


Возвращает строку, состоящую из указанного
 количества пробелов.


## Комментарии


Для исключения пробелов из исходной строки используйте функции [LTrim](Func_Text_TrimStart.htm),
 [RTrim](Func_Text_TrimEnd.htm)
 и [Trim](Func_Text_Trim.htm).


## Пример


		 Формула
		 Результат
		 Применение


		 = Space(2)
		 «  »


		 Можно использовать в формулах моделей контейнера
		 моделирования, в формулах вычисляемых рядов базы данных временных
		 рядов, в формулах моделей контейнера моделирования, являющегося
		 дочерним по отношению к базе данных временных рядов и в редакторе
		 выражения.


		 = Space(4)
		 «    »
