# ASCII


## Синтаксис


ASCII(Source: String)


## Параметры


Source. Строка, для которой
 необходимо получить код в ASCII для первого символа строки.


## Описание


Возвращает код в ASCII для первого символа
 строки.


## Комментарии


Для получения символа, который соответствует коду ASCII используйте
 функцию [Char](Func_Text_Chr.htm).


## Пример


		 Формула
		 Результат
		 Применение


		 = ASCII("String")
		 83
		 Можно использовать в формулах моделей контейнера
		 моделирования, в формулах вычисляемых рядов базы данных временных
		 рядов, в формулах моделей контейнера моделирования, являющегося
		 дочерним по отношению к базе данных временных рядов и в редакторе
		 выражения.


		 = ASCII("a")
		 97
