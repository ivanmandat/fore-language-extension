# TrimStart


## Синтаксис


Trim(Source: String)


## Параметры


Source. Строка.


## Описание


Возвращает строку с удаленными пробелами
 в начале строки.


## Комментарии


Для удаления пробелов в конце строки используйте функцию [TrimEnd](Func_Text_TrimEnd.htm), для удаления пробелов
 в начале и конце строки - функцию [Trim](Func_Text_Trim.htm).


## Пример


		 Формула
		 Результат
		 Применение


		 = TrimStart("   Calculator")
		 «Calculator»


		 Можно использовать в формулах моделей контейнера
		 моделирования, в формулах вычисляемых рядов базы данных временных
		 рядов, в формулах моделей контейнера моделирования, являющегося
		 дочерним по отношению к базе данных временных рядов и в редакторе
		 выражения.


		 = TrimStart("   Calculator   ")
		 «Calculator   »
