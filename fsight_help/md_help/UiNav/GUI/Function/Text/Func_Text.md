# Текстовые функции


При работе с формулами возможно использование предустановленных функций,
 приведённых в таблице:


		 Наименование
		 Краткое описание


		 [ASCII](Func_Text_ASCII.htm)


		 Возвращает код в ASCII для первого символа строки.


		 [Chr](Func_Text_Chr.htm)


		 Возвращает символ, соответствующий указанному ASCII-коду


		 [Contains](Func_Text_Contains.htm)


		 Возвращает признак наличия искомой подстроки в исходной
		 строке.


		 [EndsWith_](Func_Text_EndsWith.htm)


		 Возвращает признак того, что исходная строка оканчивается
		 искомой подстрокой.


		 [Find](Func_Text_Find.htm)


		 Возвращает индекс позиции подстроки в строке.


		 [Left](Func_Text_Left.htm)


		 Возвращает заданное количество символов от начала строки.


		 [Length_](Func_Text_Length.htm)


		 Возвращает длину строки.


		 [Max_](Func_Text_Max_.htm)


		 Возвращает максимум из a и b, используя посимвольное сравнение
		 строк.


		 [Mid](Func_Text_Mid.htm)


		 Возвращает строку, начиная с позиции с индексом start.


		 [Min_](Func_Text_Min_.htm)


		 Возвращает минимум из a и b, используя посимвольное сравнение
		 строк.


		 [Replace](Func_Text_Replace.htm)


		 Возвращает строку, в которой все вхождения указанной подстроки
		 заменены на другую строку.


		 [Right](Func_Text_Right.htm)


		 Возвращает заданное количество символов от конца строки.


		 [Space](Func_Text_Space.htm)


		 Возвращает строку, состоящую из указанного количества повторных
		 пробелов.


		 [StartsWith_](Func_Text_StartsWith.htm)


		 Возвращает признак того, что исходная строка начинается
		 с искомой подстроки.


		 [ToLower](Func_Text_ToLower.htm)


		 Возвращает строку, в которой все символы в нижнем регистре.


		 [ToUpper](Func_Text_ToUpper.htm)


		 Возвращает строку, в которой все символы в верхнем регистре.


		 [Trim](Func_Text_Trim.htm)


		 Возвращает строку с удалёнными пробелами в начале и конце.


		 [TrimEnd](Func_Text_TrimEnd.htm)


		 Возвращает строку с удаленными в конце пробелами.


		 [TrimStart](Func_Text_TrimStart.htm)


		 Возвращает строку с удаленными пробелами в начале строки.
