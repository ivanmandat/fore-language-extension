# ToLower


## Синтаксис


ToLower(Source: String)


## Параметры


Source. Строка.


## Описание


Возвращает указанную строку, преобразованную
 в нижний регистр.


## Комментарии


Для получения строки, преобразованной в верхней регистр, используйте
 функцию [ToUpper](Func_Text_ToUpper.htm).


## Пример


		 Формула
		 Результат
		 Применение


		 = ToLower("CaLcUlAtOr")
		 calculator


		 Можно использовать в формулах моделей контейнера
		 моделирования, в формулах вычисляемых рядов базы данных временных
		 рядов, в формулах моделей контейнера моделирования, являющегося
		 дочерним по отношению к базе данных временных рядов и в редакторе
		 выражения.


		 = ToLower("ProductVersion")
		 productversion
