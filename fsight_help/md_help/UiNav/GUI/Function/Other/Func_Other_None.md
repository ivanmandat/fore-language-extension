# None


## Синтаксис


None()


## Описание


Возвращает признак того, что константа
 не используется.


## Комментарии


None используется в функциях
 [Arima](../Forecasting/Func_Forecasting_Arima.htm),
 [Ols](../Regression/Func_Regression_Ols.htm)
 и [Tsls](../Regression/Func_Other_Tsls.htm).
 Если None применяется в функции [Expsmooth](../Forecasting/Func_Forecasting_ExpSmooth.htm),
 то это означает, что константа равна нулю.
