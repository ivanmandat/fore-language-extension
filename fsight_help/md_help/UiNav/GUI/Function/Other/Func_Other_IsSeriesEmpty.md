# IsSeriesEmpty


## Синтаксис


IsSeriesEmpty(Input: [ITimeSeries](kems.chm::/Interface/itimeseries/itimeseries.htm))


## Параметры


Input. Ряд данных, пустоту
 которого требуется определить.


## Описание


Возвращает признак пустоты ряда.


## Комментарии


Допустимые значения:


	- True. Ряд пустой;


	- False. Ряд содержит
	 данные.
