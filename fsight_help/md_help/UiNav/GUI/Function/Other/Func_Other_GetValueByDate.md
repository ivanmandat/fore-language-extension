# GetValueByDate


## Синтаксис


GetValueByDate(Input: Variant, Date: Variant)


## Параметры


Input. Ряд, значение которого
 необходимо получить;


Date. Дата, на которую надо
 получить значение.


## Описание


Возвращает значение ряда на заданную
 дату.


## Комментарии


В качестве значения параметра Input
 может передаваться константа, параметр или объект типа [TimeSeries](kems.chm::/Interface/ITimeSeries/ITimeSeries.htm), а
 в качестве значения параметра Date
 - число, строка или параметр.


## Пример


		 Формула
		 Результат
		 Применение


		 = GetValueByDate({ВВП|Анкоридж[t]}, 2005)*2
		 Значение ряда ВВП|Анкоридж
		 на 2005-й год будет умножено на два.


		 Можно использовать в формулах вычисляемых рядов базы данных
		 временных рядов и в формулах моделей контейнера моделирования,
		 основанных на атрибутах.


		 = GetValueByDate(X1, "2015A")*2
		 Значение фактора X1
		 на 2015-й год будет умножено на два.


		 Можно использовать в формулах моделей контейнера моделирования,
		 основанных на переменных.
