# Прочие функции


При работе с формулами возможно использование предустановленных функций,
 приведённых в таблице:


		 Наименование
		 Краткое описание


		 [Combine](Func_Other_Combine.htm)


		 Возвращает комбинированный ряд, состоящий из исходного ряда
		 на период идентификации и рассчитанного прогноза на периоде прогнозирования.


		 [Estimate](Func_Other_Estimate.htm)


		 Возвращает оцененное значение константы.


		 [GetValueByDate](Func_Other_GetValueByDate.htm)


		 Возвращает значение ряда на заданную дату.


		 [IsSeriesEmpty](Func_Other_IsSeriesEmpty.htm)


		 Возвращает признак пустоты ряда.


		 [None](Func_Other_None.htm)


		 Возвращает признак того, что константа не используется.


		 [Pi](Func_Other_Pi.htm)


		 Возвращает математическую константу «Пи».


		 [SetPeriod](Func_Other_SetPeriod.htm)


		 Формирует период по указанным датам.
