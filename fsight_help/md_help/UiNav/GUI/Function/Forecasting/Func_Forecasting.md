# Прогнозирование


При работе с формулами возможно использование предустановленных функций,
 приведённых в таблице:


		 Наименование
		 Краткое описание


		 [Arima](Func_Forecasting_Arima.htm)


		 Осуществляет моделирование значений переменной методом ARIMA.


		 [ArimaR](../R_methods/Func_R_methods_ArimaR.htm)


		 Осуществляет моделирование значений переменной методом ARIMA.
		 Для расчёта используется R.


		 [AutoTrend](Func_Forecasting_AutoTrend.htm)


		 Подбирает для переменной оптимальный тренд на заданном периоде.


		 [DynamicLowerConfidenceLevel](Func_Forecasting_DynamicLowerConfidenceLevel.htm)


		 Возвращает нижнюю динамическую доверительную границу
		 прогнозного ряда.


		 [DynamicUpperConfidenceLevel](Func_Forecasting_DynamicUpperConfidenceLevel.htm)


		 Возвращает верхнюю динамическую доверительную границу
		 прогнозного ряда.


		 [ExpSmooth](Func_Forecasting_ExpSmooth.htm)


		 Осуществляет преобразование переменной методом экспоненциального
		 сглаживания.


		 [ExpSmoothR](../R_methods/Func_R_methods_ExpSmoothR.htm)


		 Осуществляет преобразование переменной методом экспоненциального
		 сглаживания. Для расчёта используется R.


		 [Extrapolate](Func_Forecasting_Extrapolate.htm)


		 Осуществляет преобразование переменной с использованием тренда
		 с подбором функциональной зависимости.


		 [Fitted](Func_Forecasting_Fitted.htm)


		 Возвращает
		 модельный ряд.


		 [Forecast](Func_Forecasting_Forecast.htm)


		 Возвращает прогнозные значения для указанного ряда.


		 [GreyForecast](Func_Forecasting_GreyForecast.htm)


		 Осуществляет моделирование переменной с помощью Грей-метода.


		 [LowerConfidenceLevel](Func_Forecasting_LowerConfidenceLevel.htm)


		 Возвращает нижнюю доверительную границу прогнозного
		 ряда.


		 [Residuals](Func_Forecasting_Residuals.htm)


		 Возвращает ряд остатков.


		 [UpperConfidenceLevel](Func_Forecasting_UpperConfidenceLevel.htm)


		 Возвращает верхнюю доверительную границу прогнозного
		 ряда.
