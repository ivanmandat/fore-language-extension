# Sign


## Синтаксис


Sign(Input: Variant)


## Параметры


Input.
 Переменная.


## Описание


Возвращает знак точек
 указанной переменной. Может возвращать следующие значения:


	- 1.
	 Число положительное;


	- 0.
	 Число равно нулю;


	- -1.
	 Число отрицательное.


## Пример


		 Формула
		 Результат
		 Применение


		 = Sign({Brazil|BCA})
		 Рассчитаны знаки наблюдений показателя Brazil|BCA.
		 Можно использовать в формулах вычисляемых рядов базы данных
		 временных рядов и в формулах моделей контейнера моделирования,
		 являющегося дочерним по отношению к базе данных временных рядов.


		 = Sign(X1)
		 Рассчитаны знаки точек фактора X1.
		 Можно использовать в формулах моделей контейнера моделирования.
