# Работа с Python


При работе с формулами возможно использование предустановленных методов,
 приведённых в таблице:


		 Наименование
		 Краткое описание


		 [PythonInvoke_](Func_Python_Invoke.htm)
		 Выполняет функцию на языке Python и возвращает результат её
		 работы.


		 [PythonInvokeModule_](Func_Python_Invoke_Module.htm)
		 Выполняет функцию, хранящуюся в [Python-модуле](UiDevEnv.chm::/01_Development_Environment/02_Work_in_Development_Environment/DevEnv_Object/python_module.htm),
		 и возвращает результат её работы.


Примечание.
 Перед использованием методов убедитесь, что выполнена [интеграция](../../../02_Navigator/CommonSettings/Integration.htm#python)
 «Форсайт. Аналитическая платформа»
 с Python.
