# Регрессия


При работе с формулами возможно использование предустановленных функций,
 приведённых в таблице:


		 Наименование
		 Краткое описание


		 [Ecm](Func_Regression_Ecm.htm)


		 Осуществляет преобразование переменной с помощью модели коррекции
		 ошибок.


		 [Ols](Func_Regression_Ols.htm)


		 Осуществляет моделирование переменной с помощью линейной регрессии
		 (оценка МНК).


		 [OlsR](../R_methods/Func_R_methods_OlsR.htm)


		 Осуществляет моделирование переменной с помощью линейной регрессии
		 (оценка МНК). Для расчёта используется R.


		 [Tsls](Func_Other_Tsls.htm)


		 Осуществляет моделирование переменной с помощью линейной регрессии
		 (оценка методом инструментальных переменных).


		 [TslsR](../R_methods/Func_R_methods_TslsR.htm)


		 Осуществляет моделирование переменной с помощью линейной регрессии
		 (оценка методом инструментальных переменных). Для расчёта
		 используется R.
