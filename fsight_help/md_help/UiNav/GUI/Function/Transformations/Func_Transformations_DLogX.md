# DLogX


## Синтаксис


DLogX(Input: [ITimeSeries](kems.chm::/Interface/itimeseries/itimeseries.htm),Lag:
 Integer)


## Параметры


Input. Переменная;


Lag. Параметр задает сдвиг
 (лаг) точек переменной во временном периоде. Может принимать только положительные
 значения. Необязательный параметр. По умолчанию параметр равен 1 - предыдущий
 период.


## Описание


Осуществляет расчет прироста логарифма
 точек переменной к указанному периоду: (ln(x[t]/x[t-Lag])).


## Пример


		 Формула
		 Результат
		 Применение


		 = DLogX({Brazil|BCA}, 2)
		 Рассчитан прирост логарифма наблюдений показателя Brazil|BCA к периоду, сдвинутому
		 от текущего на две точки назад.
		 Можно использовать в формулах вычисляемых рядов базы данных
		 временных рядов и в формулах моделей контейнера моделирования,
		 являющегося дочерним по отношению к базе данных временных рядов.


		 = DLogX(X1, 3)
		 Рассчитан прирост логарифма точек фактора X1 к
		 периоду, сдвинутому от текущего на три точки назад.


		 Можно использовать в формулах моделей контейнера моделирования.
