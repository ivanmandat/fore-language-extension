# Ratio


## Синтаксис


Ratio(Input: [ITimeSeries](kems.chm::/Interface/itimeseries/itimeseries.htm),

     Period: [MsInversionLag](kems.chm::/enums/msinversionlag.htm),

     PeriodNumber: Integer)


## Параметры


Input. Моделируемая переменная;


Period. Период, в отношении
 к которому рассчитывается коэффициент роста. Необязательный параметр.
 По умолчанию параметр имеет значение MsInversionLag.PrecidingValue
 - предыдущий период;


PeriodNumber. Лаг для периода.
 Необязательный параметр. Минимальное значение и значение по умолчанию
 параметра равно 1.


## Описание


Осуществляет расчет коэффициента роста точек
 переменной.


## Пример


		 Формула
		 Результат
		 Применение


		 = Ratio({Brazil|BCA}, MsInversionLag.PrecidingYear)
		 Для показателя Brazil|BCA
		 будет рассчитан коэффициент роста значений наблюдений к соответствующему
		 периоду предыдущего года.


		 Можно использовать в формулах вычисляемых рядов базы данных
		 временных рядов и в формулах моделей контейнера моделирования,
		 являющегося дочерним по отношению к базе данных временных рядов.


		 = Ratio(X1,MsInversionLag.PrecidingValue,2)
		 Для фактора X1 будет
		 рассчитан коэффициент роста значений точек к периоду, сдвинутому
		 от текущего на две точки.


		 Можно использовать в формулах моделей контейнера моделирования.
