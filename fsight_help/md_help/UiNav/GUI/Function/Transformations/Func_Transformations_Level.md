# Level


## Синтаксис


Level(Input: [ITimeSeries](kems.chm::/Interface/itimeseries/itimeseries.htm),
 Method: [MsLvlMethod](kems.chm::/enums/mslvlmethod.htm),
 StartValue: Integer)


## Параметры


Input. Переменная;


Method. Метод преобразования;


StartValue. Начальное значение.
 Необязательный параметр. Параметр по умолчанию равен 100.


## Описание


Осуществляет сглаживание данных указанной переменной.


## Пример


		 Формула
		 Результат
		 Применение


		 = Level({Brazil|BCA}, MsLvlMethod.Log)
		 Выполняется сглаживание показателя Brazil|BCA с использованием
		 натурального логарифма.
		 Можно использовать в формулах вычисляемых рядов базы данных
		 временных рядов и в формулах моделей контейнера моделирования,
		 являющегося дочерним по отношению к базе данных временных рядов.


		 = Level(X1,MsLvlMethod.PchA)
		 Выполняется сглаживание фактора X1 с использованием
		 процентного изменения к предыдущему периоду с поправкой на сезонность.


		 Можно использовать в формулах моделей контейнера моделирования.
