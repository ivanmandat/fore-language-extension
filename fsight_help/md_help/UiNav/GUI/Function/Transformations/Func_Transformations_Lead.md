# Lead


## Синтаксис


Lead(Input: [ITimeSeries](kems.chm::/Interface/itimeseries/itimeseries.htm),
 Value: Integer)


## Параметры


Input. Переменная;


Value. Параметр задает сдвиг
 (лаг) точек переменной во временном периоде. Может принимать только положительные
 значения. Необязательный параметр. По умолчанию параметр равен 1 -
 сдвиг на одну точку.


## Описание


Осуществляет сдвиг переменной назад
 на заданное количество точек во временном периоде.


## Пример


		 Формула
		 Результат
		 Применение


		 = Lead({Brazil|BCA}, 2)
		 Наблюдения показателя Brazil|BCA
		 будут сдвинуты на две точки назад во временном периоде.
		 Можно использовать в формулах вычисляемых рядов базы данных
		 временных рядов и в формулах моделей контейнера моделирования,
		 являющегося дочерним по отношению к базе данных временных рядов.


		 = Lead(X1, 3)
		 Точки фактора X1
		 будут сдвинуты на три точки назад во временном периоде.


		 Можно использовать в формулах моделей контейнера моделирования.
