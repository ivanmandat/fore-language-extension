# PchX


## Синтаксис


PchX(Input: [ITimeSeries](kems.chm::/Interface/itimeseries/itimeseries.htm),
 Lag: Integer)


## Параметры


Input. Переменная;


Lag. Параметр задает сдвиг
 (лаг) точек переменной во временном периоде. Может принимать только положительные
 значения. Необязательный параметр. По умолчанию параметр равен 1 - предыдущий
 период.


## Описание


Осуществляет расчёт темпа прироста
 точек переменной к указанному периоду.


## Пример


		 Формула
		 Результат
		 Применение


		 = PchX({Brazil|BCA},3)
		 Рассчитан темп прироста наблюдений показателя Brazil|BCA к
		 периоду, сдвинутому от текущего на три точки.


		 Можно использовать в формулах вычисляемых рядов базы данных
		 временных рядов и в формулах моделей контейнера моделирования,
		 являющегося дочерним по отношению к базе данных временных рядов.


		 = PchX(X1,2)
		 Рассчитан темп прироста точек фактора X1 к
		 предыдущему периоду, сдвинутому от текущего на две точки.


		 Можно использовать в формулах моделей контейнера моделирования.
