# Rebase


## Синтаксис


Rebase(Input: [ITimeSeries](kems.chm::/Interface/itimeseries/itimeseries.htm),

      BaseDate: String,

      BasePeriod: [DimCalendarLevel](KeDims.chm::/Enums/DimCalendarLevel.htm),

      BaseValue: Integer)


## Параметры


Input. Переменная;


BaseDate. Дата, относительно
 которой рассчитывается функция;


BasePeriod. Календарная динамика
 для функции. Необязательный параметр. По умолчанию параметр имеет значение
 DimCalendarLevel.Year - годовая
 динамика;


BaseValue. Базовое значение
 для расчета функции. Необязательный параметр. По умолчанию параметр равен
 100.


## Описание


Осуществляет «перебазирование» данных указанной переменной с одной динамики на другую.


## Пример


		 Формула
		 Результат
		 Применение


		 = Rebase({Brazil|BCA},"2005A")
		 К показателю Brazil|BCA
		 будет применена функция Rebase.


		 Можно использовать в формулах вычисляемых рядов базы данных
		 временных рядов и в формулах моделей контейнера моделирования,
		 являющегося дочерним по отношению к базе данных временных рядов.


		 = Rebase(X1,"2000A")
		 К фактору X1 будет
		 применена функция Rebase.


		 Можно использовать в формулах моделей контейнера моделирования.
