# Преобразования


При работе с формулами возможно использование предустановленных функций,
 приведённых в таблице:


		 Наименование
		 Краткое описание


		 [Cumulative](Func_Transformations_Cumulative.htm)


		 Осуществляет преобразование переменной с применением указанного
		 метода накопления.


		 [CumulativeYTD](Func_Transformations_CumulativeYTD.htm)


		 Осуществляет преобразование переменной с применением указанного
		 метода накопления к началу года.


		 [Diff](Func_Transformations_Diff.htm)


		 Осуществляет расчет прироста точек переменной к предыдущему
		 периоду.


		 [DiffX](Func_Transformations_DiffX.htm)


		 Осуществляет расчет прироста точек переменной к указанному
		 периоду.


		 [DiffY](Func_Transformations_DiffY.htm)


		 Осуществляет расчет прироста точек переменной к соответствующему
		 периоду предыдущего года.


		 [DLog](Func_Transformations_DLog.htm)


		 Осуществляет расчет прироста логарифма точек переменной к предыдущему
		 периоду.


		 [DLogX](Func_Transformations_DLogX.htm)


		 Осуществляет расчет прироста логарифма точек переменной к указанному
		 периоду.


		 [Fill](Func_Transformations_Fill.htm)


		 Позволяет, с использованием различных методов, вычислить пустые
		 значения переменной.


		 [Lag](Func_Transformations_Lag.htm)


		 Осуществляет сдвиг переменной вперед на заданное количество
		 точек во временном периоде.


		 [Lead](Func_Transformations_Lead.htm)


		 Осуществляет сдвиг переменной назад на заданное количество
		 точек во временном периоде.


		 [Level](Func_Transformations_Level.htm)


		 Осуществляет сглаживание данных указанной переменной.


		 [Pch](Func_Transformations_Pch.htm)


		 Осуществляет расчет темпа прироста точек переменной к предыдущему
		 периоду.


		 [PchA](Func_Transformations_PchA.htm)


		 Осуществляет расчет темпа прироста точек переменной к предыдущему
		 периоду с поправкой на сезонность.


		 [PchX](Func_Transformations_PchX.htm)


		 Осуществляет расчет темпа прироста точек переменной к указанному
		 периоду.


		 [PchY](Func_Transformations_PchY.htm)


		 Осуществляет расчет темпа прироста точек переменной к соответствующему
		 периоду предыдущего года.


		 [Ratio](Func_Transformations_Ratio.htm)


		 Осуществляет расчет темпа роста точек переменной.


		 [Rebase](Func_Transformations_Rebase.htm)


		 Осуществляет «перебазирование» данных указанной переменной
		 с одной динамики на другую.


		 [Splice](Func_Transformations_Splice.htm)


		 Осуществляет совмещение переменных.


		 [SpliceP](Func_Transformations_SpliceP.htm)


		 Осуществляет преобразование переменной на основе совмещенных
		 переменных.


		 [Sum](Func_Transformations_Sum.htm)


		 Возвращает сумму соответствующих точек для двух и более переменных.


		 [SumI](Func_Transformations_SumI.htm)


		 Возвращает сумму соответствующих точек для двух и более переменных,
		 если они содержат только целые значения.


		 [SumSq](Func_Transformations_SumSq.htm)


		 Возвращает сумму квадратов для соответствующих точек для двух
		 и более переменных.


		 [Truncate](Func_Transformations_Truncate.htm)


		 Осуществляет усечение переменной по заданным параметрам.


Примечание.
 В модуле «Моделирование и прогнозирование»
 функцию SumI нельзя использовать
 в моделях, рассчитываемых в [задаче
 трансформации](uimodelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_4_Problem/uimodelling_problem_2.htm).
