# AbsI


## Синтаксис


AbsI(Input: Variant)


## Параметры


Input.
 Переменная.


## Описание


Возвращает абсолютное значение (модуль)
 точек указанной переменной, если она содержит только целые значения.


## Комментарии


Для расчёта модуля точек переменной, содержащей не только целые значения,
 используйте функцию [Abs](Func_Arithmetic_Abs.htm).


## Пример


		 Формула
		 Результат
		 Применение


		 = AbsI({Brazil|BCA})
		 Для каждого наблюдения показателя Brazil|BCA
		 будет рассчитано абсолютное значение. Если показатель содержит
		 вещественные значения, то возникнет ошибка.
		 Можно использовать в формулах вычисляемых рядов базы данных
		 временных рядов и в формулах моделей контейнера моделирования,
		 являющегося дочерним по отношению к базе данных временных рядов.


		 = AbsI(X1)
		 Для каждой точки фактора X1
		 будет рассчитано абсолютное значение.
		 Можно использовать в формулах моделей контейнера моделирования.


		 = AbsI(-7)
		 7
		 Можно использовать в формулах вычисляемых рядов базы данных
		 временных рядов и моделей контейнера моделирования.
