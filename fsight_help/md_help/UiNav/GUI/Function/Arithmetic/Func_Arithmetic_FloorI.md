# FloorI


## Синтаксис


FloorI(Input: Variant, Precision: Variant)


## Параметры


Input. Переменная, точки которой
 должны быть округлены;


Precision. Кратное, до которого
 требуется округлить.


## Описание


Возвращает результат округления точек переменной
 до кратного заданной точности с недостатком. Переменная должна содержать
 только целые значения.


## Комментарии


Для округления точек переменной, содержащей не только целые значения,
 используйте функцию [Floor](Func_Arithmetic_Floor.htm).


## Пример


		 Формула
		 Результат
		 Применение


		 = FloorI({Brazil|BCA},3)
		 Наблюдения показателя Brazil|BCA
		 округлены с недостатком до ближайшего числа, кратного трём. Если
		 показатель содержит вещественные значения, то возникнет ошибка.


		 Можно использовать в формулах вычисляемых рядов базы данных
		 временных рядов и в формулах моделей контейнера моделирования,
		 являющегося дочерним по отношению к базе данных временных рядов.


		 = FloorI(X1,2)
		 Точки фактора X1 округлены
		 с недостатком до ближайшего числа, кратного двум.


		 Можно использовать в формулах моделей контейнера моделирования.
