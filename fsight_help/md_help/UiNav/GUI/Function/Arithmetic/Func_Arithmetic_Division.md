# Division


## Синтаксис


Division(Division: Variant, Divisor: Variant)


## Параметры


Division. Делимое;


Divisor. Делитель.


## Описание


Возвращает частное от поточечного деления
 переменной Division на переменную
 Divisor.


## Пример


		 Формула
		 Результат
		 Применение


		 = Division({Brazil|BCA}, {Albania|BCA})
		 Данные показателя Brazil|BCA
		 будут поточечно разделены на данные показателя Albania|BCA.


		 Можно использовать в формулах вычисляемых рядов базы данных
		 временных рядов и в формулах моделей контейнера моделирования,
		 являющегося дочерним по отношению к базе данных временных рядов.


		 = Division(X1,7)
		 Каждая точка фактора X1
		 будет разделена на семь.
		 Можно использовать в формулах моделей контейнера моделирования.


		 = Division(100, 4)
		 25


		 Можно использовать в формулах вычисляемых рядов базы данных
		 временных рядов и моделей контейнера моделирования.
