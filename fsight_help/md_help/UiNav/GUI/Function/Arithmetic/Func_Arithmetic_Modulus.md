# Modulus


## Синтаксис


Modulus(Input1: Variant; Input2: Variant)


## Параметры


Input1. Переменная.


Input2. Делитель.


## Описание


Возвращает остаток от целочисленного
 деления значений точек переменной на заданное число. Результат имеет такой
 же знак, как и делимое.


Примечание.
 Методы Modulus и [Mod_](Func_Arithmetic_Mod_.htm)
 дублируют друг друга.


## Пример


		 Формула
		 Результат
		 Применение


		 = Modulus({Brazil|BCA},{China|BCA})
		 Рассчитан остаток от целочисленного деления наблюдений показателя
		 Brazil|BCA на соответствующие
		 наблюдения показателя China|BCA.


		 Можно использовать в формулах вычисляемых рядов базы данных
		 временных рядов и в формулах моделей контейнера моделирования,
		 являющегося дочерним по отношению к базе данных временных рядов.


		 = Modulus(X1,3)
		 Рассчитан остаток от целочисленного деления точек фактора
		 X1 на три.


		 Можно использовать в формулах моделей контейнера моделирования.


		 = Modulus(7,3)
		 1


		 Можно использовать в формулах вычисляемых рядов базы
		 данных временных рядов и моделей контейнера моделирования.
