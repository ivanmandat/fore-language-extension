# Power


## Синтаксис


Power(Input: Variant, PowerValue: Variant)


## Параметры


Input. Переменная, задающая
 основание степени;


PowerValue. Показатель степени.


## Описание


Возвращает результат возведения точек
 переменной в заданную степень.


## Комментарии


Функции Power и [ExpX](Func_Arithmetic_ExpX.htm) дублируют друг друга.


## Пример


		 Формула
		 Результат
		 Применение


		 = Power({Brazil|BCA},{Canada|BCA})
		 Наблюдения временного ряда Brazil|BCA будут
		 возведены в степень, заданную наблюдениями временного ряда Canada|BCA.


		 Можно использовать в формулах вычисляемых рядов базы данных
		 временных рядов и в формулах моделей контейнера моделирования,
		 являющегося дочерним по отношению к базе данных временных рядов.


		 = Power(X1,2)
		 Все точки фактора X1
		 будут возведены во вторую степень.


		 Можно использовать в формулах моделей контейнера моделирования.


		 = Power(2,4)
		 24=16


		 Можно использовать в формулах вычисляемых рядов базы
		 данных временных рядов и моделей контейнера моделирования.
