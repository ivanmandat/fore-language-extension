# Remainder


## Синтаксис


Remainder(Input1: Variant, Input2: Variant)


## Параметры


Input1. Переменная;


Input2. Делитель.


## Описание


Возвращает остаток от деления значений
 точек переменной на заданное число.


## Пример


		 Формула
		 Результат
		 Применение


		 = Remainder({Brazil|BCA}, {China|BCA})
		 Рассчитан остаток от деления наблюдений показателя Brazil|BCA на соответствующие
		 наблюдения показателя China|BCA.


		 Можно использовать в формулах вычисляемых рядов базы данных
		 временных рядов и в формулах моделей контейнера моделирования,
		 являющегося дочерним по отношению к базе данных временных рядов.


		 = Remainder(X1,3)
		 Рассчитан остаток от деления точек фактора X1
		 на три.


		 Можно использовать в формулах моделей контейнера моделирования.


		 = Remainder(7,3)
		 0,33


		 Можно использовать в формулах вычисляемых рядов базы данных
		 временных рядов и моделей контейнера моделирования.
