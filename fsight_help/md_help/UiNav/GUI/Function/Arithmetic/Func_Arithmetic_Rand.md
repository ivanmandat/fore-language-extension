# Rand


## Синтаксис


Rand()


## Описание


Возвращает равномерно распределённое
 случайное число из диапазона [0, 1].


## Пример


		 Формула
		 Результат
		 Применение


		 = Rand*{Brazil|BCA}
		 Каждое наблюдение показателя Brazil|BCA
		 будет умножено на случайное число из диапазона [0, 1].


		 Можно использовать в формулах вычисляемых рядов базы данных
		 временных рядов и в формулах моделей контейнера моделирования,
		 являющегося дочерним по отношению к базе данных временных рядов.


		 = Rand+X1
		 Каждая точка фактора X1
		 будет увеличена на случайное число из диапазона [0, 1].


		 Можно использовать в формулах моделей контейнера моделирования.
