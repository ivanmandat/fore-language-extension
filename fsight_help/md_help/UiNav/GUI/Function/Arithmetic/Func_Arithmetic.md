# Арифметические функции


При работе с формулами возможно использование предустановленных арифметических
 функций, приведённых в таблице:


		 Наименование
		 Краткое описание


		 [Abs](Func_Arithmetic_Abs.htm)


		 Возвращает абсолютное значение (модуль) точек указанной переменной.


		 [AbsI](Func_Arithmetic_AbsI.htm)


		 Возвращает абсолютное значение (модуль) точек указанной переменной,
		 если она содержит только целые значения.


		 [Average](Func_Arithmetic_Average.htm)


		 Возвращает среднее значение точек указанной переменной.


		 [AverageI](Func_Arithmetic_AverageI.htm)


		 Возвращает среднее значение точек указанной переменной, если
		 она содержит только целые значения.


		 [Cos](Func_Arithmetic_Cos.htm)


		 Возвращает косинус точек указанной переменной.


		 [Division](Func_Arithmetic_Division.htm)


		 Возвращает частное от поточечного деления двух переменных.


		 [Exp](Func_Arithmetic_Exp.htm)


		 Возвращает результат возведения числа «е» в степень, заданную
		 точкой переменной.


		 [ExpX](Func_Arithmetic_ExpX.htm)


		 Возвращает результат возведения точек переменной в заданную
		 степень.


		 [Fact](Func_Arithmetic_Fact.htm)


		 Возвращает факториал точек указанной переменной.


		 [Floor](Func_Arithmetic_Floor.htm)


		 Возвращает результат округления точек переменной до кратного
		 заданной точности с недостатком.


		 [FloorI](Func_Arithmetic_FloorI.htm)


		 Возвращает результат округления точек переменной до кратного
		 заданной точности с недостатком. Переменная должна содержать только
		 целые значения.


		 [Int](Func_Arithmetic_Int.htm)


		 Округляет значения точек указанной переменной до ближайшего
		 меньшего целого.


		 [Ln](Func_Arithmetic_Ln.htm)


		 Возвращает натуральный логарифм точек указанной переменной.


		 [Log](Func_Arithmetic_Log.htm)


		 Возвращает логарифм точек указанной переменной по заданному
		 основанию.


		 [Log10](Func_Arithmetic_Log10.htm)


		 Возвращает десятичный логарифм точек указанной переменной.


		 [Max](Func_Arithmetic_Max.htm)


		 Возвращает максимальное значение среди точек переменной.


		 [MaxI](Func_Arithmetic_MaxI.htm)


		 Возвращает максимальное значение среди точек переменной, если
		 она содержит только целые значения.


		 [Mean](Func_Arithmetic_Mean.htm)


		 Возвращает математическое ожидание для указанной переменной.


		 [Median](Func_Arithmetic_Median.htm)


		 Возвращает медиану для указанной переменной.


		 [Min](Func_Arithmetic_Min.htm)


		 Возвращает минимальное значение среди точек переменной.


		 [MinI](Func_Arithmetic_MinI.htm)


		 Возвращает минимальное значение среди точек переменной, если
		 она содержит только целые значения.


		 [Mod_](Func_Arithmetic_Mod_.htm)


		 Возвращает остаток от целочисленного деления значений точек
		 переменной на заданное число.


		 [Mode](Func_Arithmetic_Mode.htm)


		 Возвращает моду указанной переменной.


		 [Modulus](Func_Arithmetic_Modulus.htm)


		 Возвращает остаток от целочисленного деления значений точек
		 переменной на заданное число.


		 [Mult](Func_Arithmetic_Mult.htm)


		 Возвращает произведение соответствующих точек для двух и более
		 переменных.


		 [Power](Func_Arithmetic_Power.htm)


		 Возвращает результат возведения точек переменной в заданную
		 степень.


		 [PowerI](Func_Arithmetic_PowerI.htm)


		 Возвращает результат возведения точек переменной в заданную
		 степень. Все точки переменной и степень должны быть целыми числами.


		 [Rand](Func_Arithmetic_Rand.htm)


		 Возвращает равномерно распределенное случайное число из диапазона
		 [0, 1].


		 [RandBetween](Func_Arithmetic_RandBetween.htm)


		 Возвращает случайное вещественное число между двумя заданными
		 числами.


		 [RandBetweenI](Func_Arithmetic_RandBetweenI.htm)


		 Возвращает случайное целое число между двумя заданными целыми
		 числами.


		 [Remainder](Func_Arithmetic_Remainder.htm)


		 Возвращает остаток от деления значений точек переменной на
		 заданное число.


		 [Round](Func_Arithmetic_Round.htm)


		 Осуществляет округление значений точек переменной.


		 [RoundDown](Func_Arithmetic_RoundDown.htm)


		 Осуществляет округление с недостатком значений точек переменной.


		 [RoundUp](Func_Arithmetic_RoundUp.htm)


		 Осуществляет округление с избытком значений точек переменной.


		 [Sin](Func_Arithmetic_Sin.htm)


		 Возвращает синус точек указанной переменной.


		 [Sqrt](Func_Arithmetic_Sqrt.htm)


		 Возвращает квадратный корень точек указанной переменной.


		 [StDv](Func_Arithmetic_StDv.htm)


		 Возвращает стандартное отклонение для указанной переменной.


		 [Subtract](Func_Arithmetic_Subtract.htm)


		 Возвращает разность соответствующих точек для двух и более
		 переменных.


		 [Tan](Func_Arithmetic_Tan.htm)


		 Возвращает тангенс точек указанной переменной.


		 [Trunc](Func_Arithmetic_Trunc.htm)


		 Усекает точки переменной до указанного количества десятичных
		 разрядов.


		 [Variance](Func_Arithmetic_Variance.htm)


		 Возвращает дисперсию переменной.


Примечание.
 В модуле «Моделирование и прогнозирование»
 функции AbsI, AverageI,
 FloorI, PowerI,
 RandBetweenI, MinI,
 MaxI нельзя использовать в моделях,
 рассчитываемых в [задаче
 трансформации](uimodelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_4_Problem/uimodelling_problem_2.htm).
