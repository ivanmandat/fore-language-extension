# PowerI


## Синтаксис


PowerI(Input: Variant, PowerValue: Variant)


## Параметры


Input. Переменная, задающая
 основание степени;


PowerValue. Показатель степени.


## Описание


Возвращает результат возведения точек
 переменной в заданную степень. Все точки переменной и степень должны быть
 целыми числами.


## Комментарии


Для возведения в степень точек переменной, содержащей не только целые
 значения, используйте функцию [Power](Func_Arithmetic_Power.htm).


## Пример


		 Формула
		 Результат
		 Применение


		 = PowerI({Brazil|BCA},{Canada|BCA})
		 Наблюдения показателя Brazil|BCA будут
		 возведены в степень, заданную наблюдениями показателя Canada|BCA.
		 Если какой-либо из показателей содержит вещественные значения,
		 то возникнет ошибка.


		 Можно использовать в формулах вычисляемых рядов базы данных
		 временных рядов и в формулах моделей контейнера моделирования,
		 являющегося дочерним по отношению к базе данных временных рядов.


		 = PowerI(X1,2)
		 Все точки фактора X1
		 будут возведены во вторую степень.


		 Можно использовать в формулах моделей контейнера моделирования.


		 = PowerI(2,4)
		 24=16


		 Можно использовать в формулах вычисляемых рядов базы
		 данных временных рядов и моделей контейнера моделирования.
