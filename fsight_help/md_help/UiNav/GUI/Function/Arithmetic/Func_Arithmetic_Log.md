# Log


## Синтаксис


Log(Input: Variant, Base: Variant)


## Параметры


Input. Переменная;


Base. Основание логарифма.


## Описание


Возвращает логарифм точек указанной переменной
 по заданному основанию.


Примечание.
 Значение точки, для которой находится логарифм, и его основание должны
 быть больше 0.


## Пример


		 Формула
		 Результат
		 Применение


		 = Log({Brazil|BCA},2)
		 Рассчитан логарифм наблюдений показателя Brazil|BCA
		 по основанию два.


		 Можно использовать в формулах вычисляемых рядов базы данных
		 временных рядов и в формулах моделей контейнера моделирования,
		 являющегося дочерним по отношению к базе данных временных рядов.


		 = Log(X1,5)
		 Рассчитан логарифм точек фактора X1 по
		 основанию пять.


		 Можно использовать в формулах моделей контейнера моделирования.


		 = Log(10,2)
		  log210 =
		 3,32


		 Можно использовать в формулах вычисляемых рядов базы
		 данных временных рядов и моделей контейнера моделирования.
