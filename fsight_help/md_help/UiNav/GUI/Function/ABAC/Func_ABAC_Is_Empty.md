# Is_Empty


## Синтаксис


Is_Empty(Value: Variant)


## Параметры


Value. Атрибут:


	- SUBJECT.<идентификатор атрибута>.
	 Пользовательский атрибут субъекта;


	- OBJECT.<идентификатор атрибута>.
	 Пользовательский атрибут объекта.


Для получения подробной информации
 обратитесь к разделу «[Создание
 пользовательских атрибутов](Admin.chm::/04_SecurityPolicy/Creating_attributes.htm)».


## Описание


Определяет, является ли значение [пользовательского
 атрибута](admin.chm::/04_SecurityPolicy/Creating_attributes.htm) субъекта или объекта пустым, и возвращает
 значение:


	- True. Пользовательский
	 атрибут субъекта или объекта содержит пустое значение;


	- False.
	 Пользовательский атрибут субъекта или объекта не содержит пустое значение.


## Пример


		 Формула
		 Результат


		 = Is_Empty(OBJECT.ATTR_OBJ)
		 Функция возвращает значение True,
		 если пользовательский атрибут объекта с идентификатором «ATTR_OBJ»
		 содержит пустое значение, иначе False.


		 = Is_Empty(SUBJECT.ATTR_USER)
		 Функция возвращает значение True,
		 если пользовательский атрибут субъекта с идентификатором «ATTR_USER»
		 содержит пустое значение, иначе False.
