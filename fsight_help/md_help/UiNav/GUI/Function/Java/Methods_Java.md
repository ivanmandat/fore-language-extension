# Работа с Java


При работе с формулами возможно использование предустановленных методов,
 приведённых в таблице:


		 Наименование
		 Краткое описание


		 [JavaInvoke_](Method_Java_Invoke.htm)
		 Выполняет статический Java-метод и возвращает результат его
		 работы.


		 [JavaInvokeModule_](Method_Java_Invoke_Module.htm)
		 Выполняет статический метод, хранящийся в [Java-модуле](UiDevEnv.chm::/01_Development_Environment/02_Work_in_Development_Environment/DevEnv_Object/Java_module.htm),
		 и возвращает результат его работы.

 «Форсайт. Аналитическая платформа»
