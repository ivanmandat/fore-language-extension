# Методы R


При работе с формулами возможно использование функций, при расчёте которых
 используются пакет R, приведённых в таблице:


		 Наименование
		 Краткое описание


		 [ArimaR](Func_R_methods_ArimaR.htm)


		 Моделирует значения переменной методом ARIMA
		 с помощью пакета R.


		 [BpfR](Func_R_methods_BpfR.htm)


		 Моделирует значения переменной фильтром [Бакстера-Кинга](Lib.chm::/02_Time_series_analysis/UiModelling_BaxterKingFilter.htm)
		 с помощью пакета R.


		 [ExpSmoothR](Func_R_methods_ExpSmoothR.htm)


		 Преобразует данные переменной методом экспоненциального сглаживания
		 с помощью пакета R.


		 [HpfR](Func_R_methods_HpfR.htm)


		 Сглаживает данные переменной фильтром Ходрика-Прескотта с помощью
		 пакета R.


		 [MedianSmoothR](Func_R_methods_MedianSmoothR.htm)


		 Выполняет [медианное
		 сглаживание](Lib.chm::/02_Time_series_analysis/UiModelling_median_smoothing.htm) переменной с помощью пакета R.


		 [MovAvgR](Func_R_methods_MovAvgR.htm)


		 Преобразует данные переменной методом [скользящего
		 среднего](Lib.chm::/02_Time_series_analysis/UiModelling_SlideSmooth.htm) с помощью пакета R.


		 [OlsR](Func_R_methods_OlsR.htm)


		 Моделирует данные переменной с помощью линейной регрессии (оценка
		 МНК). Расчет выполняется с помощью пакета R.


		 [R](Func_R_methods_R.htm)


		 Выполняет преобразование данных с помощью методов пакета
		 R.


		 [TslsR](Func_R_methods_TslsR.htm)


		 Моделирует данные переменной с помощью линейной регрессии (оценка
		 методом инструментальных переменных). Расчет выполняется с помощью
		 пакета R.


Примечание.
 Расчёт методов R возможен, если в «Форсайт. Аналитическая платформа»
 подключен пакет R. Для получения подробной информации обратитесь к статье
 «[Подключение дополнительных модулей к «Форсайт. Аналитическая платформа](../../../02_Navigator/CommonSettings/Integration.htm#r)».
