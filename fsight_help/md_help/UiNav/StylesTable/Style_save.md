# Сохранение стиля


Для сохранения стиля:


	- в веб-приложении:


		- нажмите кнопку ![](b_save_web.png) «Сохранить»
		 на панели «Стили»;


		- выполните команду ![](b_save_web.png) «Сохранить»
		 в контекстном меню стиля на панели «Стили».


	- в настольном приложении нажмите кнопку
	 ![](b_save.gif) «Сохранить» на панели инструментов.


Все изменения в стиле будут сохранены.
