# Создание копии таблицы стилей


Для создания копии таблицы стилей выполните команду «Сохранить
 как» в главном меню «Таблица стилей».
 Будет отображен диалог для выбора места хранения копии таблицы стилей
 в текущем репозитории.


Примечание.
 Создание копии таблицы стилей доступно только в настольном приложении.


По умолчанию таблица стилей сохраняется со старым именем и с автоматически
 сгенерированным идентификатором. Если требуется измените имя или идентификатор.


Примечание.
 Идентификатор не должен содержать символы русского алфавита и пробелы.


При создании копии в репозитории создается новая таблица стилей, которая
 сразу же открывается для редактирования.


Примечание.
 Создание копии таблицы стилей доступно, если у пользователя есть [права](Admin.chm::/03_Admin/Admin_AdminObjects_Discretionary.htm) на
 изменение данного объекта репозитория.
