# Создание и настройка стиля


Стиль предназначен для автоматического изменения внешнего вида визуализаторов
 в отчётах, для которых [подключена](Connect.htm) таблица стилей.


[![](../Opened.gif)![](../Closed.gif)Создание
 нового стиля ячеек таблицы](javascript:TextPopup(this))


	Для создания нового стиля ячеек таблицы:


		- в веб-приложении:


			- нажмите кнопку ![](../02_Navigator/b_add.png) «Новый стиль» на панели «Стили»;


			- выполните команду «Новый
			 стиль» в контекстном меню панели «Стили»;


		- в настольном приложении:


			- нажмите кнопку ![](../02_Navigator/b_add.png) «Новый стиль» на панели инструментов;


			- выполните команду «Новый
			 стиль» в контекстном меню панели «Стили»;


			- выполните команду «Новый
			 стиль» в главном меню «Таблица
			 стилей».


	В результате выполнения одного из перечисленных действий в таблицу
	 стилей будет добавлен новый стиль ячеек таблицы.


[![](../Opened.gif)![](../Closed.gif)
 Создание нового стиля других визуализаторов](javascript:TextPopup(this))


	Создание новых стилей для других визуализаторов доступно только
	 в настольном приложении.


	Для создания нового стиля [диаграммы](UiDiagrams.chm::/Diagrams.htm),
	 [карты](UiMaps.chm::/Map.htm), [пузырьковой
	 диаграммы](UiVisualizators.chm::/BubbleChart/BubbleChart.htm), [пузырькового
	 дерева](UiVisualizators.chm::/BubbleTree/BubbleTree.htm), [плоского дерева](UiVisualizators.chm::/TreeMap/TreeMap.htm)
	 используйте инструменты: «[Аналитические
	 запросы (OLAP)](UIExpress.chm::/purpose/UiExpress_Purpose.htm)», «[Аналитические
	 панели](UIAdhoc.chm::/UiAdhoc_Purpose.htm)», «[Отчёты](UIReport.chm::/UiReport_purpose.htm)»,
	 «[Анализ
	 временных рядов](UiDw.chm::/UiDw_Title.htm)».


	Примечание.
	 В таблице стилей доступна только загрузка готовых стилей визуализаторов
	 из указанных инструментов.


	Для настройки стиля диаграммы
	 обратитесь к разделу «[Настройка стиля диаграммы](UiDiagrams.chm::/UiDiagram_Style.htm)».


	Для настройки стиля карты обратитесь
	 к разделу «[Настройка стиля карты](UiMaps.chm::/Configure/Map_style.htm)».


	Для настройки стиля пузырьковой
	 диаграммы обратитесь к разделу «[Настройка
	 стиля пузырьковой диаграммы](UiVisualizators.chm::/BubbleChart/BubbleChart_Style.htm)».


	Для настройки стиля пузырькового
	 дерева обратитесь к разделу «[Пузырьковое
	 дерево](UiVisualizators.chm::/BubbleTree/BubbleTree.htm#style)».


	Для настройки стиля плоского
	 дерева обратитесь к разделу «[Плоское дерево](UiVisualizators.chm::/TreeMap/TreeMap.htm#style)».


	Для добавления созданного стиля
	 в таблицу стилей обратитесь к разделу «[Загрузка
	 стилей](StylesTable_Import.htm)».


[![](../Opened.gif)![](../Closed.gif)Настройка
 стиля ячеек таблицы](javascript:TextPopup(this))


	Для настройки стиля ячеек:


		- На панели «Стили»
		 выберите стиль.


		- В области настройки задайте параметры стиля:


			- в веб-приложении параметры настройки стиля аналогичны
			 параметрам оформления ячеек в группе параметров «[Формат
			 ячеек](UIReport.chm::/Web/table/attribute/format.htm)» на панели параметров регламентного
			 отчёта;


			- в настольном приложении вкладки, отображаемые для настройки
			 стиля, аналогичны вкладкам [диалога
			 форматирования](../GUI/Format.htm).


		- [Сохраните](Style_save.htm) внесённые изменения.


	Примечание.
	 При попытке выбрать на панели «Стили»
	 другой стиль без сохранения изменений в текущем стиле появится окно
	 подтверждения действия с предложением сохранения изменений.


	В результате выполнения действий выбранный стиль будет настроен.
	 К ячейкам таблицы, использующим этот стиль, автоматически будет применено
	 новое оформление.


[![](../Opened.gif)![](../Closed.gif)Настройка
 стиля других визуализаторов](javascript:TextPopup(this))


	Настройка стилей для других визуализаторов доступна только в настольном
	 приложении.


	Для настройки стиля [диаграммы](UiDiagrams.chm::/Diagrams.htm),
	 [карты](UiMaps.chm::/Map.htm), [пузырьковой
	 диаграммы](UiVisualizators.chm::/BubbleChart/BubbleChart.htm), [пузырькового
	 дерева](UiVisualizators.chm::/BubbleTree/BubbleTree.htm), [плоского дерева](UiVisualizators.chm::/TreeMap/TreeMap.htm)
	 создайте [новый стиль](#add_style_visualizers) соответствующего
	 визуализатора.


	Примечание.
	 В таблице стилей загруженные стили визуализаторов доступны только
	 для просмотра.


[![](../Opened.gif)![](../Closed.gif)Переименование
 стиля](javascript:TextPopup(this))


	Для переименования стиля, выбранного на панели «Стили»:


		- в веб-приложении:


			- нажмите кнопку ![](b_rename_web.png) «Переименовать»
			 на панели «Стили»;


			- выполните команду ![](b_rename_web.png) «Переименовать» в контекстном
			 меню стиля на панели «Стили»;


		- в настольном приложении:


			- нажмите кнопку ![](b_rename.gif) «Переименовать»
			 на панели инструментов;


			- выполните команду «Переименовать»
			 в контекстном меню стиля на панели «Стили»;


			- выполните команду «Правка >
			 Переименовать» в главном меню;


			- нажмите клавишу F2.


	Наименование стиля будет переведено в режим редактирования. Введите
	 новое наименование стиля.


	Совет. Для быстрого
	 переименования выбранного стиля в настольном приложении введите новое
	 наименование стиля в поле ввода «Наименование».


[![](../Opened.gif)![](../Closed.gif)Создание
 копии стиля](javascript:TextPopup(this))


	Для создания копии стиля, выбранного на панели «Стили»:


		- в веб-приложении:


			- нажмите кнопку ![](b_duplicate_web.png) «Дублировать» на панели инструментов;


			- выполните команду ![](b_duplicate_web.png) «Дублировать» в контекстном
			 меню стиля на панели «Стили»;


		- в настольном приложении:


			- нажмите кнопку ![](b_Duplicate.gif) «Дублировать»
			 на панели инструментов;


			- выполните команду «Дублировать»
			 в контекстном меню стиля на панели «Стили»;


			- выполните команду «Правка >
			 Дублировать» в главном
			 меню окна «Таблица стилей»;


			- нажмите сочетание клавиш CTRL+D.


	В таблице стилей будет создана копия стиля.

	Для удаления стиля, выбранного на панели «Стили»:


		- в веб-приложении:


			- нажмите кнопку ![](b_delete_web.png) «Удалить»
			 на панели инструментов;


			- выполните команду ![](b_delete_web.png) «Удалить» в контекстном меню
			 стиля на панели «Стили»;


		- в настольном приложении:


			- нажмите кнопку ![](b_delete.gif) «Удалить»
			 на панели инструментов;


			- выполните команду «Удалить»
			 в контекстном меню стиля на панели «Стили»;


			- выполните команду «Правка >
			 Удалить» в главном
			 меню окна «Таблица стилей»;


			- нажмите клавишу DELETE.


	Будет запрошено подтверждение выполняемого действия.
