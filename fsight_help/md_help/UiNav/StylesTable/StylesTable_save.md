# Сохранение таблицы стилей


Для сохранения таблицы стилей:


	- в веб-приложении нажмите кнопку «Сохранить»
	 главного меню;


	- в настольном приложении выполните команду «Таблица
	 стилей > Сохранить» [главного
	 меню](StylesTable.htm#elements).


Все изменения в таблице стилей будут сохранены.
