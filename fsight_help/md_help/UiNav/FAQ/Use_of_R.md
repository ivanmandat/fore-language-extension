# Какие методы в «Форсайт. Аналитическая платформа»
 можно рассчитать с помощью R?


Статистический пакет R позволяет расширить функциональные возможности
 «Форсайт. Аналитическая платформа»
 в инструменте «[Моделирование и прогнозирование](UiModelling.chm::/UiModelling_Purpose.htm)», инструменте «[Анализ
 временных рядов](UiDw.chm::/UiDw_Title.htm)» и универсальном [редакторе
 выражения](../GUI/ExpressionEditor.htm). Статистический пакет R также доступен при расчете с помощью
 внутреннего языка программирования Fore.


После подключения к «Форсайт. Аналитическая платформа»
 статистического пакета R следующие методы могут быть рассчитаны как с
 помощью «Форсайт. Аналитическая платформа»,
 так и с помощью R:


[![](../Opened.gif)![](../Closed.gif)Линейная регрессия
 (МНК)](javascript:TextPopup(this))


		- Ограничения при расчете
		 с помощью R:


			- не поддерживается автоподбор факторов;


			- игнорируется тип расчета стандартных ошибок для доверительных
			 границ прогноза;


			- не поддерживается модель с лаговыми переменными.


		- Требуемый пакет R:
		 stats.


		- Интерфейс для расчета с
		 помощью внутреннего языка Fore: [ISmLinearRegress](StatLib.chm::/Interface/ISmLinearRegress/ISmLinearRegress.htm).


[![](../Opened.gif)![](../Closed.gif)Линейная регрессия
 (оценка методом инструментальных переменных)](javascript:TextPopup(this))


		- Ограничения при расчете
		 с помощью R: не поддерживается модель с авторегрессией
		 и скользящим средним.


		- Требуемый пакет R:
		 gmm.


		- Интерфейс для расчета с
		 помощью внутреннего языка Fore: [ISm2SLS](StatLib.chm::/Interface/ISm2SLS/ISm2SLS.htm).


[![](../Opened.gif)![](../Closed.gif)Нелинейная регрессия](javascript:TextPopup(this))


		- Ограничения при расчете
		 с помощью R:


			- недоступен в универсальном редакторе выражения;


			- выражение нелинейной регрессии должно быть написано
			 с использованием синтаксиса R.


		- Требуемый пакет R:
		 stats.


		- Интерфейс для расчета с
		 помощью внутреннего языка Fore: [ISmNonLinearLeastSquare](StatLib.chm::/Interface/ISmNonLinearLeastSquare/ISmNonLinearLeastSquare.htm).


[![](../Opened.gif)![](../Closed.gif)Фильтр Бакстера-Кинга](javascript:TextPopup(this))


		- Ограничения при расчете
		 с помощью R: нет.


		- Требуемый пакет R:
		 mFilter.


		- Интерфейс для расчета с
		 помощью внутреннего языка Fore: [ISmBandpassFilter](StatLib.chm::/Interface/ISmBandpassFilter/ISmBandpassFilter.htm).


[![](../Opened.gif)![](../Closed.gif)Фильтр Ходрика-Прескотта](javascript:TextPopup(this))


		- Ограничения при расчете
		 с помощью R: не поддерживается вычисление параметра сглаживания
		 с использованием значения степени.


		- Требуемый пакет R:
		 mFilter.


		- Интерфейс для расчета с
		 помощью внутреннего языка Fore: [ISmHodrickPrescottFilter](StatLib.chm::/Interface/ISmHodrickPrescottFilter/ISmHodrickPrescottFilter.htm).


[![](../Opened.gif)![](../Closed.gif)Экспоненциальное
 сглаживание](javascript:TextPopup(this))


		- Ограничения при расчете
		 с помощью R:


			- не поддерживается сочетание экспоненциальной модели
			 роста с аддитивным сезонным эффектом;


			- не поддерживается метод наилучшей пробы для автоподбора
			 параметров;


			- не поддерживается критерий «Среднее
			 остатков» для автоподбора параметров;


			- результаты расчета метода «экспоненциальное
			 сглаживание» через R могут отличаться от результатов
			 расчета данного метода в «Форсайт. Аналитическая платформа».


		- Требуемый пакет R:
		 forecast.


		- Интерфейс для расчета с
		 помощью внутреннего языка Fore: [ISmExponentialSmoothing](StatLib.chm::/Interface/ISmExponentialSmoothing/ISmExponentialSmoothing.htm).


[![](../Opened.gif)![](../Closed.gif)ARIMA](javascript:TextPopup(this))


		- Ограничения при расчете
		 с помощью R:


			- не поддерживается модель с произвольными порядками авторегрессии
			 и скользящего среднего: порядки авторегрессии и скользящего
			 среднего должны задаваться подряд с 1; порядки сезонных авторегрессии
			 и скользящего среднего должны быть кратны периоду сезонности;


			- не поддерживается модель с константой при использовании
			 порядка дифференцирования больше 2 или сезонного дифференцирования.
			 При расчете константа будет исключена из модели;


			- игнорируются начальные приближения;


			- игнорируются настройки максимального числа итераций
			 и точности;


			- игнорируется тип расчета начальных приближений;


			- игнорируется ретрополяция (BackCast);


			- игнорируется тип расчета стандартных ошибок для доверительных
			 границ прогноза.


			- результаты расчета метода «ARIMA»
			 через R могут отличаться от результатов расчета данного метода
			 в «Форсайт. Аналитическая платформа».


		- Требуемый пакет R:
		 forecast.


		- Интерфейс для расчета с
		 помощью внутреннего языка Fore: [ISmLinearRegress](StatLib.chm::/Interface/ISmLinearRegress/ISmLinearRegress.htm).


[![](../Opened.gif)![](../Closed.gif)Медианное сглаживание](javascript:TextPopup(this))


		- Ограничения при расчете
		 с помощью R: недоступен в инструменте «Анализ
		 временных рядов».


		- Требуемый пакет R:
		 stats.


		- Интерфейс для расчета с
		 помощью внутреннего языка Fore: [ISmMedianSmoothing](StatLib.chm::/Interface/ISmMedianSmoothing/ISmMedianSmoothing.htm).


[![](../Opened.gif)![](../Closed.gif)Скользящее среднее](javascript:TextPopup(this))


		- Ограничения при расчете
		 с помощью R: нет.


		- Требуемый пакет R:
		 forecast.


		- Интерфейс для расчета с
		 помощью внутреннего языка Fore: [ISmSlideSmoothing](StatLib.chm::/Interface/ISmSlideSmoothing/ISmSlideSmoothing.htm).


[![](../Opened.gif)![](../Closed.gif)Тест Гранжера](javascript:TextPopup(this))


		- Ограничения при расчете
		 с помощью R: нет.


		- Требуемый пакет R:
		 lmtest.


		- Интерфейс для расчета с
		 помощью внутреннего языка Fore: [ISmGrangerTest](StatLib.chm::/Interface/ISmGrangerTest/ISmGrangerTest.htm).


[![](../Opened.gif)![](../Closed.gif)Тест Рамсея](javascript:TextPopup(this))


		- Ограничения при расчете
		 с помощью R: не поддерживается модель с авторегрессией
		 и скользящим средним.


		- Требуемый пакет R:
		 lmtest.


		- Интерфейс для расчета с
		 помощью внутреннего языка Fore: [ISmRamseyRESSETTest](StatLib.chm::/Interface/ISmRamseyRESSETTest/ISmRamseyRESSETTest.htm).


[![](../Opened.gif)![](../Closed.gif)Расширенный
 тест Дики-Фуллера](javascript:TextPopup(this))


		- Ограничения при расчете
		 с помощью R: поддерживается только модель с константой
		 и трендом.


		- Требуемый пакет R:
		 tseries.


		- Интерфейс для расчета с
		 помощью внутреннего языка Fore: [ISmAugmentDickeyFullerTest](StatLib.chm::/Interface/ISmAugmentDickeyFullerTest/ISmAugmentDickeyFullerTest.htm).


[![](../Opened.gif)![](../Closed.gif)Тест Йохансена](javascript:TextPopup(this))


		- Ограничения при расчете
		 с помощью R:


			- доступен только в веб-версии инструмента «Моделирование
			 и прогнозирование»;


			- не поддерживается модель без лагов;


			- не поддерживаются следующие типы моделей:


				- модель без тренда в авторегрессии, без константы
				 в коинтеграционном уравнении;


				- модель с квадратичным трендом в авторегрессии, с
				 трендом в коинтеграционном уравнении.


		- Требуемый пакет R:
		 urca.


		- Интерфейс для расчета с
		 помощью внутреннего языка Fore: [ISmJohansenTest](StatLib.chm::/Interface/ISmJohansenTest/ISmJohansenTest.htm).

## Информация о пакетах R


Подробную информацию о пакете R можно получить по ссылке: http://cran.r-project.org/web/packages/<наименование
 пакета>/. Например: [http://cran.r-project.org/web/packages/lmtest/](http://cran.r-project.org/web/packages/lmtest/)


Важная информация на странице с описанием пакета:


	- строка «Depends»
	 содержит наименования дополнительных пакетов, которые требуется для
	 корректной работы пакета R;


	- строка «Imports» содержит
	 наименования пакетов, импортируемых при установке пакета;


	- раздел «Downloads»
	 содержит ссылки на загрузку: справочной документации к пакету, архивов
	 для установки пакета.


### Решение возможных проблем с пакетами при работе с методами R


При работе с методами R из «Форсайт. Аналитическая платформа»
 в отдельных случаях могут возникать различные проблемы, связанные с пакетами.
 Ниже описаны наиболее распространенные из них, а также причины их появления
 и пути решения:


[![](../Opened.gif)![](../Closed.gif)Ошибка при
 загрузке пакета <имя пакета>: there is no package called <имя
 пакета>](javascript:TextPopup(this))


	Причина появления: не установлен
	 пакет, требуемый для расчета метода.


	Решение: загрузите и установите
	 необходимый пакет.


[![](../Opened.gif)![](../Closed.gif)Ошибка при
 загрузке пакета <имя пакета>: package <имя другого пакета>
 could not be loaded](javascript:TextPopup(this))


	Причина появления: не установлен
	 дополнительный пакет, требуемый для расчета метода.


	Решение: загрузите и установите
	 необходимый дополнительный пакет.


[![](../Opened.gif)![](../Closed.gif)Ошибка при
 загрузке пакета <имя пакета>: package or namespace load failed for
 <имя пакета>](javascript:TextPopup(this))


	Причина появления: не удалось
	 импортировать нужный пакет.


	Решение: загрузите и установите
	 пакет, который не удалось импортировать.
