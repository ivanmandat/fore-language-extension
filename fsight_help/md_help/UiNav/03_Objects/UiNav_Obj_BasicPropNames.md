# Наименования


На вкладке «Наименования» содержится
 список наименований объекта для его перевода на разные языки. Выбор языка
 отображения объектов производится в [параметрах
 репозитория](Setup.chm::/06_AK_Client_Config/UiNav_RepoConfig_repo1.htm#current_repository_parameters).


Условия отображения вкладки:


	- в [настройках интерфейса навигатора
	 объектов](Translate_names.htm) включено отображение наименований на других языках;


	- для репозитория задан язык по умолчанию. Выбрать язык репозитория
	 по умолчанию можно один раз: при [создании
	 репозитория](setup.chm::/05_RepoMngr/Setup_RepoMngr_CreateRepo.htm) или позднее в [настройках
	 репозитория](../02_Navigator/Repo_Default.htm);


	- в [настройках
	 подключения к репозиторию](setup.chm::/06_AK_Client_Config/UiNav_RepoConfig_repo1.htm#more) в качестве языка репозитория
	 для текущего пользователя выбран «Язык
	 пользовательского интерфейса» или «Другой».


Примечание.
 Вкладка «Наименования» доступна
 только в настольном приложении.


[![](../Opened.gif)![](../Closed.gif)Для открытия
 вкладки](javascript:TextPopup(this))


	Выполните команду «Свойства»
	 в контекстном меню объекта.


![](BasicPropNames.png)


Наименование, используемое для [языка
 по умолчанию](../02_Navigator/Repo_Default.htm), выделено в списке. Данное наименование будет использовано
 для отображения наименования объекта, если для него отсутствует название
 на языке, выбранном в [параметрах
 репозитория](Setup.chm::/06_AK_Client_Config/UiNav_RepoConfig_repo1.htm#current_repository_parameters).


Для работы с наименованиями объекта доступны операции:


[![](../Opened.gif)![](../Closed.gif)Добавление нового
 наименования объекта](javascript:TextPopup(this))


	Для добавления нового наименования объекта в группе «Добавить
	 наименование»:


		- Выберите язык, для которого добавляется наименование. Раскрывающийся
		 список «Язык» содержит
		 только те языки, для которых еще не было добавлено наименование.


		- Введите наименование.


		- Нажмите кнопку «Добавить».


	В результате будет добавлено новое наименование объекта.


[![](../Opened.gif)![](../Closed.gif)Переименование
 объекта](javascript:TextPopup(this))


	Для переименования объекта:


		- нажмите кнопку «Переименовать»;


		- нажмите клавишу F2.


	Примечание.
	 Переименование объекта для текущего языка репозитория приведёт к изменению
	 наименования, отображаемого на вкладке «[Тип
	 объекта](UiNav_Obj_BasicPropGeneral.htm)».


[![](../Opened.gif)![](../Closed.gif)Удаление наименования
 объекта](javascript:TextPopup(this))


	Для удаления выбранного наименования объекта:


		- нажмите кнопку «Удалить»;


		- нажмите клавишу DELETE.


	Примечание.
	 Данная операция недоступна, если выбрано наименование для языка по
	 умолчанию.
