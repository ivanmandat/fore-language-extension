 общая информация об объекте.

При необходимости измените свойства объекта:


	- Наименование. Наименование
	 объекта на [текущем
	 языке репозитория](Setup.chm::/06_AK_Client_Config/UiNav_RepoConfig_repo1.htm);


Примечание.
 В настольном приложении изменение наименования приводит к изменению соответствующего
 наименования на вкладке «[Наименования](UiNav_Obj_BasicPropNames.htm)».


	- Идентификатор. Последовательность
	 букв латинского алфавита, цифр и знаков подчёркивания, начинающаяся
	 с буквы. Идентификатор должен быть уникальным в пределах репозитория;


	- Примечание. Дополнительная
	 информация или описание объекта.


Примечание.
 В настольном приложении заданное примечание отображается в столбце «[Примечание](../GUI/View.htm#view_columns)»
 [навигатора
 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm).


Свойства объекта, доступные только для просмотра:


	- Тип объекта. Тип объекта
	 репозитория;


	- Ключ. Уникальное числовое
	 значение, которое однозначно идентифицирует объект в репозитории;


	- Текущая версия. Номер
	 версии, который увеличивается на единицу при изменении объекта. Если
	 в менеджере безопасности настроен [контроль
	 версий объектов при обновлении](Admin.chm::/04_SecurityPolicy/Editor_of_Politicy/Security_EditorPoliticy_General.htm#version_control), то в свойстве будет
	 дополнительно указан заданный идентификатор репозитория;


	- Дата изменения. Дата
	 и время последнего редактирования объекта.
