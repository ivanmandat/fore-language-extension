# Работа с готовыми ресурсами


Для работы с готовыми ресурсами доступны операции:


	- [редактирование ресурса](#edit);


	- [сортировка ресурсов](#sorting). Доступно только
	 в настольном приложении;


	- [выделение ресурсов](#select);


	- [удаление ресурса](#delete).


## Редактирование ресурса


Для редактирования ресурса:


	- в веб-приложении:


		- дважды щёлкните по ячейке ресурса;


		- выполните команду «Редактировать»
		 в контекстном меню ячейки ресурса;


	- в настольном приложении:


		- дважды щёлкните по ячейке ресурса;


		- выполните команду «Редактировать»
		 в контекстном меню ячейки ресурса;


		- нажмите клавишу F2 при выбранной ячейке ресурса.


После выполнения одного из действий:


	- для строковых ресурсов выбранная ячейка будет переведена в режим
	 редактирования. Измените перевод ресурса и нажмите клавишу ENTER или
	 щёлкните за пределами ячейки;


	- для графических ресурсов будет открыта [боковая
	 панель](Work_Resources.htm#localized) в веб-приложении или окно «[Редактирование изображения](Work_Resources.htm#localized)»
	 в настольном приложении. Измените изображение с помощью кнопки ![](b_change.png) «Изменить».

## Выделение ресурсов


Для выделения одного ресурса щёлкните по нему.


Для выделения всех ресурсов:


	- в веб-приложении:


		- выполните команду «Выделить
		 все» в контекстном меню списка ресурсов;


		- выполните команду «Правка >
		 Выделить все» в главном меню;


	- в настольном приложении:


		- выполните команду «Выделить
		 все» в контекстном меню списка ресурсов;


		- выполните команду «Правка >
		 Выделить все» в главном меню;


		- нажмите сочетание клавиш CTRL+A.


Для выделения нескольких ресурсов зажмите клавишу CTRL и щёлкайте по
 ресурсам. Для выделения диапазона элементов зажмите клавишу SHIFT и щёлкните
 сначала по первому ресурсу, затем по последнему ресурсу диапазона.


Примечание.
 Выделение нескольких ресурсов доступно только в настольном приложении.


Для снятия выделения выполните команду «Правка > Снять
 выделение» в главном меню.


## Удаление ресурса


Для удаления выделенного ресурса:


	- в веб-приложении:


		- выполните команду «Правка > Удалить
		 ресурс» в главном меню;


		- выполните команду «Удалить»
		 в контекстном меню ресурса;


		- нажмите клавишу DELETE;


	- в настольном приложении:


		- нажмите кнопку ![](delete.gif) «Удалить
		 ресурс» на панели инструментов;


		- выполните команду «Правка > Удалить»
		 в главном меню;


		- выполните команду «Удалить»
		 в контекстном меню ресурса;


		- нажмите клавишу DELETE.


После выполнения одного из действий ресурс будет удалён.
