# DateTimePicker.NextMonthName


## Синтаксис


NextMonthName: String


## Описание


Свойство NextMonthName устанавливает текст всплывающей подсказки для кнопки «Следующий месяц».


## Комментарии


По умолчанию установлен текст «Next month»


## Пример


Для выполнения примера создайте компонент [DateTimePicker](../../Components/DateTimePicker/DateTimePicker.htm) (см. «[Пример создания компонента DateTimePicker](../../Components/DateTimePicker/Example_DateTimePicker.htm)») с наименованием «DTP». Чтобы изменить текст всплывающей подсказки используйте свойство NextMonthName:


DTP.setNextMonthName ("Следующий месяц");


Задать значение свойства через JSON-объект можно следующим образом:


Calendar: { NextMonthName: "Следующий месяц" }
