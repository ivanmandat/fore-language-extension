# DateTimePicker.TextAfterQuarter


## Синтаксис


TextAfterQuarter: String


## Описание


Свойство TextAfterQuarter устанавливает текст, который будет отображён в поле ввода после значения квартала.


## Комментарии


По умолчанию установлен текст « quarter ».


## Пример


Для выполнения примера создайте компонент [DateTimePicker](../../Components/DateTimePicker/DateTimePicker.htm) (см. «[Пример создания компонента DateTimePicker](../../Components/DateTimePicker/Example_DateTimePicker.htm)») с наименованием «DTP». Чтобы изменить текст в поле ввода после значения квартала используйте свойство TextAfterQuarter:


DTP.setTextAfterQuarter(" квартал ");


Задать значение свойства через JSON-объект можно следующим образом:


DateTimeEdit: {TextAfterQuarter: " квартал "}
