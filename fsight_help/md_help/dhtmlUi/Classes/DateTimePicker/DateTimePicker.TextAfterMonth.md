# DateTimePicker.TextAfterMonth


## Синтаксис


TextAfterMonth: String;


## Описание


Свойство TextAfterMonth устанавливает текст, который будет отображён в поле ввода после значения месяца.


## Комментарии


По умолчанию текст не установлен.


## Пример


Для выполнения примера создайте компонент [DateTimePicker](../../Components/DateTimePicker/DateTimePicker.htm) (см. «[Пример создания компонента DateTimePicker](../../Components/DateTimePicker/Example_DateTimePicker.htm)») с наименованием «DTP». Чтобы изменить текст в поле ввода после значения месяца используйте свойство TextAfterMonth:


DTP.setTextAfterMonth(" месяц ");


Задать значение свойства через JSON-объект можно следующим образом:


DateTimeEdit: {TextAfterMonth: " месяц "}
