# DateTimePicker.TextAfterYear


## Синтаксис


TextAfterYear: String


## Описание


Свойство TextAfterYear устанавливает текст, который будет отображён в поле ввода после значения года.


## Комментарии


По умолчанию текст не задан.


## Пример


Для выполнения примера создайте компонент [DateTimePicker](../../Components/DateTimePicker/DateTimePicker.htm) (см. «[Пример создания компонента DateTimePicker](../../Components/DateTimePicker/Example_DateTimePicker.htm)») с наименованием «DTP». Чтобы изменить текст в поле ввода после значения года используйте свойство TextAfterYear:


DTP.setTextAfterYear(" год");


Задать значение свойства через JSON-объект можно следующим образом:


DateTimeEdit: {TextAfterYear: " год"}
