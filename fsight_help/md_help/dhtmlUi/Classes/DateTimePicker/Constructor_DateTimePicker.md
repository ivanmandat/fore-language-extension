# Конструктор DateTimePicker


## Синтаксис


PP.DateTimePicker(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор DateTimePicker создает экземпляр компонента [DateTimePicker](../../Components/DateTimePicker/DateTimePicker.htm).
