## Описание


Класс DateTimePicker реализует
 компонент [DateTimePicker](../../Components/DateTimePicker/DateTimePicker.htm),
 представляющий собой редактор даты и времени с раскрывающимся календарем
 для выбора даты.


## Комментарии


Класс DateTimePicker является
 наследником класса [Combo](../Combo/Combo.htm).


## Синтаксис


PP.initClass(PP.Ui.DateTimePicker, PP.Ui.[Combo](../Combo/Combo.htm),
 "DateTimePicker");
