# DateTimePicker.PreviousMonthName


## Синтаксис


PreviousMonthName: String


## Описание


Свойство PreviousMonthName устанавливает текст всплывающей подсказки для кнопки «Предыдущий месяц».


## Комментарии


По умолчанию установлен текст «Previous month».


## Пример


Для выполнения примера создайте компонент [DateTimePicker](../../Components/DateTimePicker/DateTimePicker.htm) (см. «[Пример создания компонента DateTimePicker](../../Components/DateTimePicker/Example_DateTimePicker.htm)») с наименованием «DTP». Чтобы изменить текст всплывающей подсказки используйте свойство PreviousMonthName:


DTP.setPreviousMonthName ("Предыдущий месяц");


Задать значение свойства через JSON-объект можно следующим образом:


Calendar: { PreviousMonthName: "Предыдущий месяц" }
