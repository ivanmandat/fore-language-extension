# DateTimePicker.PreviousYearName


## Синтаксис


PreviousYearName: String;


## Описание


Свойство PreviousYearName устанавливает текст всплывающей подсказки для кнопки «Предыдущий год».


## Комментарии


По умолчанию установлен текст «Previous year».


## Пример


Для выполнения примера создайте компонент [DateTimePicker](../../Components/DateTimePicker/DateTimePicker.htm) (см. «[Пример создания компонента DateTimePicker](../../Components/DateTimePicker/Example_DateTimePicker.htm)») с наименованием «DTP». Чтобы изменить текст всплывающей подсказки используйте свойство PreviousYearName:


DTP.setPreviousYearName ("Предыдущий год");


Задать значение свойства через JSON-объект можно следующим образом:


Calendar: { PreviousYearName: "Предыдущий год" }
