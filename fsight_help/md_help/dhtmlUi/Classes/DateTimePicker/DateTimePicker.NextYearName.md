# DateTimePicker.NextYearName


## Синтаксис


NextYearName: String;


## Описание


Свойство NextYearName устанавливает текст всплывающей подсказки для кнопки «Следующий год».


## Комментарии


По умолчанию установлен текст «Next year».


## Пример


Для выполнения примера создайте компонент [DateTimePicker](../../Components/DateTimePicker/DateTimePicker.htm) (см. «[Пример создания компонента DateTimePicker](../../Components/DateTimePicker/Example_DateTimePicker.htm)») с наименованием «DTP». Чтобы изменить текст всплывающей подсказки используйте свойство NextYearName:


DTP.setNextYearName ("Следующий год");


Задать значение свойства через JSON-объект можно следующим образом:


Calendar: { NextYearName: "Следующий год" }
