# DateTimePicker.TextAfterHalfYear


## Синтаксис


TextAfterHalfYear: String


## Описание


Свойство TextAfterHalfYear устанавливает текст, который будет отображён в поле ввода после значения полугодия.


## Комментарии


По умолчанию установлен текст « half-year ».


## Пример


Для выполнения примера создайте компонент [DateTimePicker](../../Components/DateTimePicker/DateTimePicker.htm) (см. «[Пример создания компонента DateTimePicker](../../Components/DateTimePicker/Example_DateTimePicker.htm)») с наименованием «DTP». Чтобы изменить текст в поле ввода после значения полугодия используйте свойство TextAfterHalfYear:


DTP.setTextAfterHalfYear(" полугодие ")


Задать значение свойства через JSON-объект можно следующим образом:


DateTimeEdit: {TextAfterHalfYear: " полугодие "}
