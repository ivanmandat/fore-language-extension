# DateTimePicker.TextAfterWeek


## Синтаксис


TextAfterWeek: String


## Описание


Свойство TextAfterWeek устанавливает текст, который будет отображён в поле ввода после значения недели.


## Комментарии


По умолчанию установлен текст «-th week».


## Пример


Для выполнения примера создайте компонент [DateTimePicker](../../Components/DateTimePicker/DateTimePicker.htm) (см. «[Пример создания компонента DateTimePicker](../../Components/DateTimePicker/Example_DateTimePicker.htm)») с наименованием «DTP». Чтобы изменить текст в поле ввода после значения недели используйте свойство TextAfterWeek:


DTP.setTextAfterWeek(" неделя ");


Задать значение свойства через JSON-объект можно следующим образом:


DateTimeEdit: {TextAfterWeek: " неделя "}
