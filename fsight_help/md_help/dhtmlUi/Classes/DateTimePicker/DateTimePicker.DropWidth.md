# DateTimePicker.DropWidth


## Синтаксис


setDropWidth(value);


## Параметры


value. Ширина области, на которой расположена раскрывающаяся панель.


## Описание


Свойство DropWidth устанавливает ширину области, на которой расположена раскрывающаяся панель.
