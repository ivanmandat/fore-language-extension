# LineLabel.IsLineBlur


## Синтаксис


IsLineBlur: Boolean


## Описание


Свойство IsLineBlur определяет
 размытости линии.


## Комментарии


Допустимые значения:


	- true.
	 Линия будет размыта (по умолчанию);


	- false.
	 Линия размыта не будет.


Значение свойства устанавливается из JSON и с помощью метода setIsLineBlur,
 а возвращается с помощью метода getIsLineBlur.
