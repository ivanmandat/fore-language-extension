# LineLabel.LineHeight


## Синтаксис


LineHeight: Number


## Описание


Свойство LineHeight определяет
 высоту линии.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLineHeight,
 а возвращается с помощью метода getLineHeight.
