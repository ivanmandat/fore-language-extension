# LineLabel.LineBlurWidth


## Синтаксис


LineBlurWidth: Number


## Описание


Свойство LineBlurWidth определяет
 ширину размытости по краям линии.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLineBlurWidth,
 а возвращается с помощью метода getLineBlurWidth.
