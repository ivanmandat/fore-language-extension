# GradientSlider.ShowButtons


## Синтаксис


ShowButtons: Boolean


## Описание


Свойство ShowButtons** определяет, отображаются ли кнопки добавления и удаления [бегунков с возможностью выбора цвета](../SliderItemColor/SliderItemColor.htm).


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setShowButtons, а возвращается с помощью метода getShowButtons.


Если свойство имеет значение true, кнопки будут отображаться, иначе - кнопки отображаться не будут.


По умолчанию свойство имеет значение false.
