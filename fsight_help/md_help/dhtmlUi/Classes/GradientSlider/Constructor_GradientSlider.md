# Конструктор GradientSlider


## Синтаксис


PP.Ui.GradientSlider(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор GradientSlider создает экземпляр компонента [GradientSlider](../../Components/GradientSlider/GradientSlider.htm).
