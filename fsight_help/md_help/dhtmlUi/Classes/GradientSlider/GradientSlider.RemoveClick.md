# GradientSlider.RemoveClick


## Синтаксис


RemoveClick: function(sender, args, timeout);


## Параметры


sender. Источник события;


args. Информация о событии;


timeout. Промежуток времени в миллисекундах, через который вызовется событие.


## Описание


Событие RemoveClick** наступает при нажатии на кнопку удаления [бегунка с возможность выбора цвета](../SliderItemColor/SliderItemColor.htm).
