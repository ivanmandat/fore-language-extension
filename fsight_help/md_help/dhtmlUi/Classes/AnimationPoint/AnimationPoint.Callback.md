# AnimationPoint.Callback


## Синтаксис


Callback: PP.Delegate|function


## Описание


Свойство Callback** устанавливает функцию обратного вызова, которая выполняется по завершении воспроизведения анимации.


## Комментарии


Значение свойства устанавливается с помощью метода setCallback, а возвращается с помощью метода getCallback. Из JSON значение свойства установить нельзя.
