# AnimationPoint.Value


## Синтаксис


Value: Number


## Описание


Свойство Value** устанавливает и возвращает значение задержки анимации в миллисекундах.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setValue, а возвращается с помощью метода getValue.
