# AnimationPoint.Items


## Синтаксис


Items: Array|[PP.Ui.Animation](../Animation/Animation.htm)


## Описание


Свойство Items** устанавливает массив анимаций.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setItems, а возвращается с помощью метода getItems.


Свойство содержит массив объектов типа [PP.Ui.Animation](../Animation/Animation.htm) или JSON-объектов, описывающих [PP.Ui.Animation](../Animation/Animation.htm).
