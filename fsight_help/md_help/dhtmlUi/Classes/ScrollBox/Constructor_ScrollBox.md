# Конструктор ScrollBox


## Синтаксис


PP.ScrollBox(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор ScrollBox создает экземпляр компонента [ScrollBox](../../Components/ScrollBox/ScrollBox.htm).
