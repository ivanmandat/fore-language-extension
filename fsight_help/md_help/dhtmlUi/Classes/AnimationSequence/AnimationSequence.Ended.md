# AnimationSequence.Ended


## Синтаксис


Ended: function(sender, args)


## Параметры


sender. Источник события;


args. Информация о событии. Доступный аргумент: IsEnd - признак окончания воспроизведения анимации.


## Описание


Событие Ended** наступает при окончании воспроизведения анимации.
