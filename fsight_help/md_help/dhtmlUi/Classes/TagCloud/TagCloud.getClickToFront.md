# TagCloud.getClickToFront


## Синтаксис


getClickToFront();


## Описание


Метод getClickToFront определяет,
 будет ли элемент облака выдвигаться вперед при нажатии на него.


## Комментарии


Если метод возвращает значение true,
 то элементы будут выдвигаться вперед при нажатии на них кнопкой мыши.


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [TagCloud](../../Components/TagCloud/TagCloud.htm) c наименованием
 «tagCloud» (см. «[Пример
 создания компонента TagCloud](../../Components/TagCloud/Example_TagCloud.htm)»). Проверим, будут ли элементы облака
 выдвигаться вперед при нажатии на них:


// Проверяем, будут ли элементы облака выдвигаться вперед при нажатии на них.
if (tagCloud.getClickToFront()) {
    console.log("Элементы будут выдвигаться вперед при нажатие на них");
} else {
     console.log("Элементы не будут выдвигаться вперед при нажатие на них");


}
В консоль будет выведена информация о результате нажатия на элементы:


Элементы не будут выдвигаться вперед при нажатие
 на них
