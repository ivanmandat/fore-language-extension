# TagCloud.FreezeActive


## Синтаксис


FreezeActive: Boolean


## Описание


Свойство FreezeActive определяет,
 останавливается ли анимация элементов при наведении курсора на активную
 метку.


## Комментарии


Допустимые значения:


	- true.
	 При наведении курсора мыши на активную метку анимация будет остановлена;


	- false.
	 При наведении курсора мыши на активную метку анимация остановлена
	 не будет (по умолчанию).


Значение свойства устанавливается из JSON и с помощью метода setFreezeActive,
 а возвращается с помощью метода getFreezeActive.
