# TagCloud.IsDragControl


## Синтаксис


IsDragControl: Boolean


## Описание


Свойство IsDragControl определяет
 способ движения элементов облака меток.


## Комментарии


Допустимые значения:


	- true.
	 Элементы будут двигаться при зажатии кнопки мыши (по умолчанию);


	- false.
	 Элементы будут двигаться при перемещении мыши в области облака меток.


Значение свойства устанавливается из JSON и с помощью метода setIsDragControl,
 а возвращается с помощью метода getIsDragControl.
