# TagCloud.IsWheelZoom


## Синтаксис


IsWheelZoom: Boolean


## Описание


Свойство IsWheelZoom определяет,
 доступно ли масштабирование облака меток с помощью колесика мыши.


## Комментарии


Допустимые значения:


	- true.
	 Возможно масштабирование облака меток с помощью колесика мыши (по
	 умолчанию);


	- false.
	 Масштабирование облака меток с помощью колесика мыши не возможно.


Значение свойства устанавливается из JSON и с помощью метода setIsWheelZoom,
 а возвращается с помощью метода getIsWheelZoom.
