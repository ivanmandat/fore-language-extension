# Конструктор TagCloud


## Синтаксис


PP.Ui.TagCloud(settings);


## Параметры


settings. JSON-объект со значениями
 свойств компонента.


## Описание


Конструктор TagCloud создает
 экземпляр компонента [TagCloud](../../Components/TagCloud/TagCloud.htm).
