# TagCloud.SortItems


## Синтаксис


SortItems: Boolean;


## Описание


Свойство SortItems определяет,
 используется ли сортировка элементов облака меток.


## Комментарии


Допустимые значения:


	- true.
	 Элементы будут сортироваться;


	- false.
	 Сортировка производиться не будет (по умолчанию).


Значение свойства устанавливается из JSON и с помощью метода setSortItems,
 а возвращается с помощью метода getSortItems.
