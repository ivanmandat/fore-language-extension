# StackPanel.endUpdate


## Синтаксис


endUpdate ()


## Описание


Метод endUpdate возобновляет отрисовку компонента.


## Комментарии


Метод endUpdate выполняется в том случае, если перед установкой набора параметров использован метод [StackPanel.beginUpdate](StackPanel.beginUpdate.htm).
