# StackPanel.getItemByPoint


## Синтаксис


getItemByPoint(x, y);


## Параметры


x. Координата по горизонтали;


y. Координата по вертикали.


## Описание


Метод getItemByPoint возвращает
 элемент контейнера по заданным координатам.
