# StackPanel.AutoScrollSize


## Синтаксис


AutoScrollSize: Boolean;


## Описание


Свойство AutoScrollSize определяет
 признак того, что при обновлении компонента будет учитываться размер полос
 прокрутки.


## Комментарии


Значение свойства устанавливается из JSON и возвращается с помощью метода
 getAutoScrollSize.


Допустимые значения:


	- true.
	 При обновлении компонента будет учитываться размер полос прокрутки (по
	 умолчанию);


	- false.
	 При обновлении компонента размер полос прокрутки учитываться не будет.
