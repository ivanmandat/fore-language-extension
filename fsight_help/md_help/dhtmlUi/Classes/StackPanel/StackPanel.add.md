# StackPanel.add


## Синтаксис


add(ctrl, fullSize, width, height)


## Параметры


ctrl. Элемент, который необходимо
 добавить, экземпляр класса PP.Ui.[Control](../Control/Control.htm);


fullSize. Определяет, будет
 ли компонент занимать всю панель. По умолчанию установлено значение true, то есть компонент занимает
 всю панель;


width. Ширина компонента, которую
 нужно выставить после добавления;


height. Высота компонента,
 которую нужно выставить после добавления.


## Описание


Метод add добавляет элемент
 на панель.
