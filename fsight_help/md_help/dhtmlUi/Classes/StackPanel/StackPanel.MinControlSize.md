# StackPanel.MinControlSize


## Синтаксис


MinControlSize: Number


## Описание


Свойство MinControlSize определяет
 минимальный размер элемента в контейнере.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMinControlSize,
 а возвращается с помощью метода getMinControlSize.


По умолчанию значение свойства равно 0.
