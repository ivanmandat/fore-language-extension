# StackPanel.ScrollVisible


## Синтаксис


ScrollVisible: Boolean


## Описание


Свойство ScrollVisible определяет
 признак видимость полос прокрутки.


## Комментарии


Допустимые значения:


	- true.
	 Полосы прокрутки отображаются (по умолчанию);


	- false.
	 Полосы прокрутки не отображаются.


Значение свойства устанавливается из JSON и с помощью метода setScrollVisible,
 а возвращается с помощью метода getScrollVisible.
