# StackPanel.ItemSizeChanging


## Синтаксис


ItemSizeChanging: function (sender, args)


## Параметры


sender. Источник события;


args. Информация о событии.
 Доступный аргумент: Item - возвращает
 элемент, размер которого изменяется.


## Описание


Событие ItemSizeChanging наступает
 при изменении размеров панелей компонента StackPanel.


## Комментарии


Чтобы было доступно изменение размеров панелей, для свойства [StackPanel.IsContentResizable](StackPanel.IsContentResizable.htm)
 должно быть установлено значение true.


## Пример


Пример использования события приведен в описании свойства [StackPanel.DeferredResize](StackPanel.DeferredResize.htm).
