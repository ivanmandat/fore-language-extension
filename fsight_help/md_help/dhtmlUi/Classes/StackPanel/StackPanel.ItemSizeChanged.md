# StackPanel.ItemSizeChanged


## Синтаксис


ItemSizeChanged: function (sender, args)


## Параметры


sender. Источник события;


args. Информация о событии.
 Доступный аргумент - Item - возвращает элемент, размер которого был изменен.


## Описание


Событие ItemSizeChanged наступает
 после изменения размера панели компонента StackPanel.


## Комментарии


Чтобы было доступно изменение размеров панелей, для свойства [StackPanel.IsContentResizable](StackPanel.IsContentResizable.htm)
 должно быть установлено значение true.
