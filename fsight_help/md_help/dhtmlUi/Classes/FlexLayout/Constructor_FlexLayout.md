# Конструктор FlexLayout


## Синтаксис


PP.Ui.FlexLayout(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор FlexLayout создает экземпляр компонента [FlexLayout](../../Components/FlexLayout/FlexLayout.htm).
