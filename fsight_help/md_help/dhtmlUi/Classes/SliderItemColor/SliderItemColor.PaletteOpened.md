# SliderItemColor.PaletteOpened


## Синтаксис


PaletteOpened: function(sender, args, timeout);


## Параметры


sender. Источник события;


args. Информация о событии;


timeout. Промежуток времени в миллисекундах, через который вызовется событие.


## Описание


Событие PaletteOpened** наступает при открытии палитры выбора цвета [бегунка с возможностью выбора цвета](SliderItemColor.htm).
