# BorderTypeCombo.SplitLines


## Синтаксис


SplitLines: Boolean;


## Описание


Свойство SplitLines
 определяет отображение редактора в две строки.


## Комментарии


Значение свойства устанавливается из JSON
 и с помощью метода setSplitLines,
 а возвращается с помощью метода getSplitLines.
