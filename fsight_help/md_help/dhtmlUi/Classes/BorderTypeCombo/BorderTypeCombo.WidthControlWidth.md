# BorderTypeCombo.WidthControlWidth


## Синтаксис


WidthControlWidth: Number;


## Описание


Свойство WidthControlWidth
 определяет ширину элемента редактирования шириной линии.


## Комментарии


Значение свойства устанавливается из JSON
 и с помощью метода setWidthControlWidth,
 а возвращается с помощью метода getWidthControlWidth.


Значение свойства задается в пикселях.
 При значении 0 - ширина будет расчитана автоматически.
