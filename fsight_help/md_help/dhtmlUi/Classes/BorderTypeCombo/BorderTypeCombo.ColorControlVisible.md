# BorderTypeCombo.ColorControlVisible


## Синтаксис


ColorControlVisible: Boolean;


## Описание


Свойство ColorControlVisible
 определяет видимость элемента управления цветом.


## Комментарии


Значение свойства устанавливается из JSON
 и с помощью метода setColorControlVisible,
 а возвращается с помощью метода getColorControlVisible.
