# BorderTypeCombo.BorderType


## Синтаксис


BorderType: PP.Ui.LineTypes;


## Описание


Свойство BorderType
 определяет тип границы.


## Комментарии


Значение свойства устанавливается из JSON
 и с помощью метода setBorderType,
 а возвращается с помощью метода getBorderType.
