# BorderTypeCombo.MaxBorderWidth


## Синтаксис


MaxBorderWidth: Number;


## Описание


Свойство MaxBorderWidth
 определяет максимальную ширину границы.


## Комментарии


Значение свойства устанавливается из JSON
 и с помощью метода setMaxBorderWidth,
 а возвращается с помощью метода getMaxBorderWidth.


Значение свойства устанавливается в пикселях.
