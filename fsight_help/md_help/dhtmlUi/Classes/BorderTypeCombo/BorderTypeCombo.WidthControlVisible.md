# BorderTypeCombo.WidthControlVisible


## Синтаксис


WidthControlVisible: Boolean;


## Описание


Свойство WidthControlVisible
 определяет видимость элемента редактирования шириной линии.


## Комментарии


Значение свойства устанавливается из JSON
 и с помощью метода setWidthControlVisible,
 а возвращается с помощью метода getWidthControlVisible.
