# BorderTypeCombo.MinBorderWidth


## Синтаксис


MinBorderWidth: Number;


## Описание


Свойство MinBorderWidth
 определяет минимальную ширину границы.


## Комментарии


Значение свойства устанавливается из JSON
 и с помощью метода setMinBorderWidth,
 а возвращается с помощью метода getMinBorderWidth.


Значение свойства устанавливается в пикселях.
