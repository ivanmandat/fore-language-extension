# BorderTypeCombo.BorderWidth


## Синтаксис


BorderWidth: Number;


## Описание


Свойство BorderWidth
 определяет ширину границы.


## Комментарии


Значение свойства устанавливается из JSON
 и с помощью метода setBorderWidth,
 а возвращается с помощью метода getBorderWidth.


Свойство задается в пикселях.
