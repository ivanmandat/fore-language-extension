# BorderTypeCombo.TypeControlVisible


## Синтаксис


TypeControlVisible: Boolean;


## Описание


Свойство TypeControlVisible
 определяет видимость редактора типа границ.


## Комментарии


Значение свойства устанавливается из JSON
 и с помощью метода setTypeControlVisible,
 а возвращается с помощью метода getTypeControlVisible.
