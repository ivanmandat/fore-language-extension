# BorderTypeCombo.WidthControlSuffix


## Синтаксис


WidthControlSuffix: String;


## Описание


Свойство WidthControlSuffix
 определяет суффикс в элементе редактирования ширины линии.


## Комментарии


Значение свойства устанавливается из JSON
 и с помощью метода setWidthControlSuffix,
 а возвращается с помощью метода getWidthControlSuffix.
