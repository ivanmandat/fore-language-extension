# BorderTypeCombo.BorderEnabled


## Синтаксис


BorderEnabled: Boolean;


## Описание


Свойство BorderEnabled
 определяет доступность редактора границ.


## Комментарии


Значение свойства устанавливается из JSON
 и с помощью метода setBorderEnabled,
 а возвращается с помощью метода getBorderEnabled.
