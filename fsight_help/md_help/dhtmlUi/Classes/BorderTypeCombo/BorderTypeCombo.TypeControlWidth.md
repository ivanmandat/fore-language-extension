# BorderTypeCombo.TypeControlWidth


## Синтаксис


TypeControlWidth: Number;


## Описание


Свойство TypeControlWidth
 определяет ширину элемента управления типом границы.


## Комментарии


Значение свойства устанавливается из JSON
 и с помощью метода setTypeControlWidth,
 а возвращается с помощью метода getTypeControlWidth.


Значение свойства задается в пикселях.
 При значении 0 - ширина будет расчитана автоматически.
