# BorderTypeCombo.TypesMode


## Синтаксис


TypesMode: [PP.Ui.BorderCombo.TypesMode](../../Enums/TypesMode.htm);


## Описание


Свойство TypesMode
 определяет технологию отображения линии.


## Комментарии


Значение свойства устанавливается из JSON
 и с помощью метода setTypesMode,
 а возвращается с помощью метода getTypesMode.

Пример использования свойства приведен в описании свойства [BorderTypeCombo.BorderColor](BorderTypeCombo.BorderColor.htm).
