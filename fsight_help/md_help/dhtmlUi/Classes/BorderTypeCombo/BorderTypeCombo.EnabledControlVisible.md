# BorderTypeCombo.EnabledControlVisible


## Синтаксис


EnabledControlVisible: Boolean;


## Описание


Свойство EnabledControlVisible
 определяет видимость чекбокса включения редактора границ.


## Комментарии


Значение свойства устанавливается из JSON
 и с помощью метода setEnabledControlVisible,
 а возвращается с помощью метода getEnabledControlVisible.
