# SlidePanel.ExpandConsoleToolTip


## Синтаксис


ExpandConsoleToolTip: String


## Описание


Свойство ExpandConsoleToolTip устанавливает всплывающую подсказку для кнопки, при нажатии на которую раскрывается консоль.
