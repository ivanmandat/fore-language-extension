# SlidePanel.CloseToolTip


## Синтаксис


CloseToolTip: String


## Описание


Свойство CloseToolTip устанавливает текст всплывающей подсказки для кнопки закрытия консоли.


## Комментарии


Консоль - объединение панелей Header и Details.


По умолчанию текст подсказки не задан.
