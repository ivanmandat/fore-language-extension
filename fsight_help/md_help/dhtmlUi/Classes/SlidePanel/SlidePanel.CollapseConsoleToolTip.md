# SlidePanel.CollapseConsoleToolTip


## Синтаксис


CollapseConsoleToolTip: String


## Описание


Свойство CollapseConsoleToolTip устанавливает текст для всплывающей подсказки кнопки сворачивания консоли.


## Комментарии


Консоль - объединение панелей Header и Details.


По умолчанию текст подсказки не задан.
