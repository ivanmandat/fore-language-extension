# SlidePanel.DetailsTitleHeight


## Синтаксис


DetailsTitleHeight: Number


## Описание


Свойство DetailsTitleHeight
 определяет высоту заголовка панели Details.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDetailsTitleHeight,
 а возвращается с помощью метода getDetailsTitleHeight.
