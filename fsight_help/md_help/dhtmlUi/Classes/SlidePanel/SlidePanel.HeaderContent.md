# SlidePanel.HeaderContent


## Синтаксис


HeaderContent: Object


## Описание


Свойство HeaderContent устанавливает содержимое панели Header.


## Комментарии


Содержимым панели может быть произвольный объект. По умолчанию содержимое не задано.
