# SlidePanel.DetailsContent


## Синтаксис


DetailsContent: Object


## Описание


Свойство DetailsContent устанавливает содержимое для панели Details.


## Комментарии


Для панели может быть установлено произвольное содержимое. По умолчанию содержимое не задано.
