# SlidePanel.EnableResize


## Синтаксис


EnableResize: Boolean


## Описание


Свойство EnableResize определяет
 возможность изменения размера панелей Header
 и Details.


## Комментарии


Допустимые значения:


	- true.
	 Изменение размера панелей Header
	 и Details разрешено;


	- false.
	 Изменение размера панелей Header
	 и Details невозможно
	 (по умолчанию).


Значение свойства устанавливается из JSON и с помощью метода setEnableResize,
 а возвращается с помощью метода getEnableResize.
