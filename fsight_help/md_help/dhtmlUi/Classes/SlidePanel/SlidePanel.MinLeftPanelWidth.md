# SlidePanel.MinLeftPanelWidth


## Синтаксис


MinLeftPanelWidth: Number


## Описание


Свойство MinLeftPanelWidth определяет
 минимальную ширину панелей Header
 и Details.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMinLeftPanelWidth,
 а возвращается с помощью метода getMinLeftPanelWidth.
