# SlidePanel.Alignment


## Синтаксис


Alignment: PP.[Alignment](dhtmlCommon.chm::/Enums/Alignment.htm)


## Описание


Свойство Alignment устанавливает
 выравнивание консоли относительно центра панели.


## Комментарии


Консоль - объединение панелей
 Header и Details.


По умолчанию установлено значение LeftAlignment,
 то есть консоль расположена слева относительно центра панели.


Значение свойства устанавливается из JSON и с помощью метода setAlignment,
 а возвращается с помощью метода getAlignment.
