# SlidePanel.ConsoleOpened


## Синтаксис


ConsoleOpened: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие ConsoleOpened наступает при открытии консоли.


## Комментарии


Консоль - объединение панелей Header и Details.
