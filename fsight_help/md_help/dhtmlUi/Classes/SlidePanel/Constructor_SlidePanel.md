# Конструктор SlidePanel


## Синтаксис


PP.SlidePanel(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор SlidePanel создает экземпляр компонента [SlidePanel](../../Components/SlidePanel/SlidePanel.htm).
