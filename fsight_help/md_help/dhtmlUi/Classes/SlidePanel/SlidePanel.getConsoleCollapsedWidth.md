# SlidePanel.getConsoleCollapsedWidth


## Синтаксис


getConsoleCollapsedWidth();


## Описание


Метод getConsoleCollapsedWidth
 возвращает значение расстояния между содержимым панели Details
 и заголовком панели Details.
