# Конструктор Clipboard


## Синтаксис


PP.Ui.Clipboard(settings);


## Параметры


*settings.* JSON-объект со значениями свойств класса.


## Описание


Конструктор Clipboard создает экземпляр класса [Clipboard](Clipboard.htm).
