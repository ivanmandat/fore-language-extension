# Clipboard.copy


## Синтаксис


copy(text: String, htmlFormat: Boolean);


## Параметры


*text.* Текст, копируемый в буфер обмена;


*htmlFormat.* Признак того, является ли текст форматируемым (в HTML-формате). Если параметр равен значению true, то текст является форматируемым, иначе - обычным. Необязательный параметр, по умолчанию его значение не определено.


## Описание


Метод copy** копирует текст в буфер обмена.


## Комментарии


Метод возвращает объект типа [PP.Ui.Clipboard](Clipboard.htm).
