# CheckBox.Checked


## Синтаксис


Checked: Boolean


## Описание


Свойство Checked определяет,
 является ли компонент выделенным.


## Комментарии


Если для свойства установлено значение true,
 то компонент является выделенным. По умолчанию установлено значение false.
