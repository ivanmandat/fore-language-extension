# Конструктор CheckBox


## Синтаксис


PP.Ui.CheckBox (settings);


## Параметры


settings. JSON-объект со значениями
 свойств компонента.


## Описание


Конструктор CheckBox создает
 экземпляр компонента [CheckBox](../../Components/CheckBox/CheckBox.htm).
