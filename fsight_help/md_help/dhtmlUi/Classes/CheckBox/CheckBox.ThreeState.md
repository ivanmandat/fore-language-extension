# CheckBox.ThreeState


## Синтаксис


ThreeState: Boolean


## Описание


Свойство ThreeState определяет,
 установлен ли режим трех состояний компонента.


## Комментарии


Если для свойства установлено значение true,
 значит включен режим трех состояний компонента. По умолчанию установлено
 значение false. Режим трех состояний
 необходим для того чтобы [установить
 промежуточное состояние](CheckBox.CheckedState.htm) отметки компонента.
