# SourceView.PropertyChanged


## Синтаксис


PropertyChanged: function(sender, args, timeout);


## Параметры


sender. Источник события;


args. Информация о событии;


timeout. Промежуток времени
 в миллисекундах, через который вызовется событие.


## Описание


Событие PropertyChanged наступает
 после изменения значения свойства.
