# SourceView.loadFiles


## Синтаксис


loadFiles(callback: PP.Delegate, arguments: Object);


## Параметры


callback. Возвратная функция;


arguments. Аргументы возвратной
 функции.


## Описание


Метод loadFiles загружает
 файлы скриптов и стилей, основываясь на внутренних параметрах таблицы.


## Комментарии


Метод возвращает значение true,
 если был загружен хотя бы один файл стиля или скрипта, и значение false, если не был загружен ни один
 из файлов.
