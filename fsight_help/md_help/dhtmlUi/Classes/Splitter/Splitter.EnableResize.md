# Splitter.EnableResize


## Синтаксис


EnableResize: Boolean


## Описание


Свойство EnableResize определяет, могут ли изменяться размеры панелей.


## Комментарии


Если для свойства установлено значение false, то размеры панелей не изменяются. По умолчанию установлено значение true.
