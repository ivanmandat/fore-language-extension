# Splitter.getResizerSize


## Синтаксис


getResizerSize();


## Описание


Метод getResizerSize возвращает
 размер области, с помощью которой изменяются размеры панелей.


## Комментарии


Метод возвращает для вертикальной области её ширину, а для горизонтальной
 - её высоту в пикселях.
