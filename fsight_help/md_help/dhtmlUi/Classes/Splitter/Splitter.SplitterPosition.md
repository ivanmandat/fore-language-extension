# Splitter.SplitterPosition


## Синтаксис


SplitterPosition: Number


## Описание


Свойство SplitterPosition устанавливает позицию, в которой расположен разделитель.


## Комментарии


Значение свойства задается в пикселях. По умолчанию разделитель установлен в центре, то есть, если [ширина](../Control/Control.Width.htm)/[высота](../Control/Control.Height.htm) компонента - 80, то для свойства SplitterPosition установлено значение 40.
