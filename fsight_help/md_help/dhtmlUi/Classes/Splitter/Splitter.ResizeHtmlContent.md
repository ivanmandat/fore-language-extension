# Splitter.ResizeHtmlContent


## Синтаксис


ResizeHtmlContent: Boolean


## Описание


Свойство ResizeHtmlContent определяет, будет ли изменяться размер HTML-элемента, установленного в свойстве [Content](../Control/Control.Content.htm).


## Комментарии


Если установлено значение true (по умолчанию), размер HTML-элемент будет изменяться при перемещении разделителя:


![](Splitter.ResizeHtmlContent.gif)


Если установлено значение false, размер HTML-элемент при перемещении разделителя изменяться не будет:


![](Splitter.ResizeHtmlContent_false.gif)
