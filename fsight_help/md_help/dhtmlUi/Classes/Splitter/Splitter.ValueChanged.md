# Splitter.ValueChanged


## Синтаксис


ValueChanged: function(sender, args);


## Параметры


sender. Источник события;


args. Информация
 о событии. Аргумент Position возвращает
 текущую позицию разделителя.


## Описание


Событие ValueChanged наступает
 после изменения значения разделителя.
