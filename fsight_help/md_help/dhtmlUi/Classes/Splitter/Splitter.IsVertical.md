# Splitter.IsVertical


## Синтаксис


IsVertical: Boolean


## Описание


Свойство IsVertical определяет
 ориентацию компонента с разделителем.


## Комментарии


Допустимые значения:


	- true.
	 Компонент размещается вертикально, то есть его панели располагаются
	 сверху и снизу от горизонтального разделителя;


	- false.
	 Компонент размещается горизонтально, то есть его панели располагаются
	 слева и справа от вертикального разделителя (по умолчанию).


Значение свойства устанавливается из JSON и с помощью метода setIsVertical,
 а возвращается с помощью метода getIsVertical.
