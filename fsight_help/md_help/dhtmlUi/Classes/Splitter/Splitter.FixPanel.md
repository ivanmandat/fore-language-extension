# Splitter.FixPanel


## Синтаксис


FixPanel: Number


## Описание


Свойство FixPanel определяет
 индекс панели, размер которой не будет меняться при изменении размеров
 всего компонента.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setFixPanel,
 а возвращается с помощью метода getFixPanel.
