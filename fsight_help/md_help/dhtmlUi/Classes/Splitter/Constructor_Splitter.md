# Конструктор Splitter


## Синтаксис


PP.Splitter(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор Splitter создает экземпляр компонента [Splitter](../../Components/Splitter/Splitter.htm).
