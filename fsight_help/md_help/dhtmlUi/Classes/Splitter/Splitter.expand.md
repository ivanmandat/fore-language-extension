# Splitter.expand


## Синтаксис


expand ()


## Описание


Метод expand разворачивает свернутую панель компонента [Splitter](../../Components/Splitter/Splitter.htm).


## Комментарии


Панель сворачивается при помощи метода [Splitter.collapse](Splitter.collapse.htm).
