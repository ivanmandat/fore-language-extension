# ColorPanel.UpdateSize


## Синтаксис


UpdateSize: function(sender,args)


## Параметры


sender. Источник события;


args. Информация
 о событии.


## Описание


Событие UpdateSize** наступает
 при обновлении размеров компонента.


## Комментарии


Обновление размеров компонента осуществляется при помощи метода [Control.updateSize](../Control/Control.updateSize.htm).
