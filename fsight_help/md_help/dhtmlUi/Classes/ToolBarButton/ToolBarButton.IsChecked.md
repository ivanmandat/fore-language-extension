# ToolBarButton.IsChecked


## Синтаксис


IsChecked: Boolean


## Описание


Свойство IsChecked устанавливает состояние кнопки.


## Комментарии


Если установлено значение true, кнопка находится в нажатом состоянии. Если установлено значение false - в отжатом. По умолчанию установлено значение false.


Свойство актуально только при генерации события [StateChanged](ToolBarButton.StateChanged.htm).
