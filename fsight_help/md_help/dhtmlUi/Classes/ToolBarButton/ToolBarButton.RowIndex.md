# ToolBarButton.RowIndex


## Синтаксис


RowIndex: Number


## Описание


Свойство RowIndex устанавливает
 индекс строки, в котором размещается пиктограмма.


## Комментарии


Набор пиктограмм содержится в файле, путь к которому указан в значении
 свойства [Source](dhtmlCommon.chm::/Classes/imagelist/imagelist.source.htm)
 компонента [ImageList](../../Components/ImageList/ImageList.htm).
