# ToolBarButton.refreshCorners


## Синтаксис


refreshCorners();


## Описание


Метод refreshCorners скругляет все углы кнопки панели инструментов.


## Комментарии


Свойство актуально, если при помощи метода [setRoundedCorners](ToolBarButton.setRoundedCorners.htm) был изменен вариант скругления углов.
