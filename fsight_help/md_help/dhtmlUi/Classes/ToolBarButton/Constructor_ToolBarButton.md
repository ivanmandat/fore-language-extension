# Констуктор ToolBarButton


## Синтаксис


PP.Ui.ToolBarButton(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор ToolBarButton создает экземпляр компонента ToolBarButton.
