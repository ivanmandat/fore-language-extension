# Master.hideGroup


## Синтаксис


hideGroup (groupName: String||Array);


## Параметры


groupName. Имя группы для скрытия.


## Описание


Метод hideGroup скрывает группы вкладок по имени.


## Комментарии


Имя группы определяется в свойстве [MasterPanel.GroupName](../masterpanel/MasterPanel.GroupName.htm).


## Пример


Пример использования метода приведен в описании свойства [MasterPanel.GroupName](../masterpanel/MasterPanel.GroupName.htm).
