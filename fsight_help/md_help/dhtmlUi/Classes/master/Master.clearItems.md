# Master.clearItems


## Синтаксис


clearItems (notDispose);


## Параметры


notDispose. Определяет, будут ли панели уничтожены (false) или удалены из [списка](Master.getItems.htm) (true).


## Описание


Метод clearItems очищает список панелей.


## Пример


Для выполнения примера предполагается наличие на странице компонента [Master](Master.htm) с наименованием «master» (см. «[Конструктор Master](Constructor_Master.htm)»). Очистим список панелей мастера:


master.clearItems();


После выполнения примера будут удалены все панели мастера.
