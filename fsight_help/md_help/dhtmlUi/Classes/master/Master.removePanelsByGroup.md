# Master.removePanelsByGroup


## Синтаксис


removePanelsByGroup(groupName);


## Параметры


groupName. Имя удаляемой
 группы.


## Описание


Метод removePanelsByGroup удаляет
 панели по имени группы, к которой они относятся.
