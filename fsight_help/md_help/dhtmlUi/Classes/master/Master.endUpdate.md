# Master.endUpdate


## Синтаксис


endUpdate(animate: Boolean);


## Параметры


animate. Признак наличия анимации
 при раскрытии панели после обновления мастера. Если значение параметра
 равно true - то панели раскроются
 с анимацей, иначе без анимации.


## Описание


Метод endUpdate завершает обновление
 мастера.


## Комментарии


Работает в связке с методом [Master.beginUpdate](Master.beginUpdate.htm).
