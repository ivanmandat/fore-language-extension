# Master.addMasterPanel


## Синтаксис


addMasterPanel(mPanelItem, beforeItem);


## Параметры


mPanelItem. Добавляемая
 панель мастера;


beforeItem. Элемент,
 перед которым будет добавлена панель.


## Описание


Метод addMasterPanel добавляет
 панель мастера.


## Комментарии


Панель мастера отрисовывается через [NavigationItem](dhtmlCommon.chm::/Classes/NavigationItem/NavigationItem.htm).
