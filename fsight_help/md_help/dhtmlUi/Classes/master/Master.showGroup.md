# Master.showGroup


## Синтаксис


showGroup (groupName: String||Array, preserveCurrentItem);


## Параметры


groupName. Имя группы для отображения;


preserveCurrentItem. Определяет, будет ли запрещена активизация показываемой группы. По умолчанию установлено значение false.


## Описание


Метод showGroup отображает группы вкладок по имени.


## Комментарии


Имя группы панелей определяется в свойстве [MasterPanel.GroupName](../masterpanel/MasterPanel.GroupName.htm).

Пример использования метода приведен в описании свойства [MasterPanel.GroupName](../masterpanel/MasterPanel.GroupName.htm).
