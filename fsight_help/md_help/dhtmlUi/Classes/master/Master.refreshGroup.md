# Master.refreshGroup


## Синтаксис


refreshGroup (groupName);


## Параметры


groupName. Имя группы.


## Описание


Метод refreshGroup обновляет панели,
 относящиеся к определённой группе.


## Комментарии


Имя группы определяется в свойстве [MasterPanel.GroupName](../masterpanel/MasterPanel.GroupName.htm).

Пример использования аналогичен примеру, приведенному для метода [EaxDimBarView.refreshAll](dhtmlExpress.chm::/Classes/Express/EaxDimBarView/EaxDimBarView.refreshAll.htm).
