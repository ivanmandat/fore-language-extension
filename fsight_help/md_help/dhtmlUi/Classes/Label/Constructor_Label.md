# Конструктор Label


## Синтаксис


PP.Ui.Label (settings)


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор Label создает экземпляр компонента [Label](../../Components/Label/Label.htm).


## Пример


Для выполнения примера в теге HEAD подключите ссылки на библиотеку PP.js и визуальные стили PP.css. Для размещения компонента добавьте на страницу элемент DIV с идентификатором «label». В теге SCRIPT добавьте код создания компонента:


        var labelSettings = {
            Content: "Новая подпись",
            ParentNode: document.getElementById("label"),
            Width: 100,
            Height: 50
        };
        var label = new PP.Ui.Label(labelSettings);

После выполнения примера на странице будет размещена подпись:


Новая подпись
