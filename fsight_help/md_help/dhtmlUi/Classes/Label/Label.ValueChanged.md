# Label.ValueChanged


## Синтаксис


ValueChanged: function (sender, args)


## Параметры


sender. Источник события. PP.Ui.Label.


args. Информация
 о событии. Событие не содержит аргументов.


## Описание


Событие ValueChanged наступает
 после изменения значения подписи.
