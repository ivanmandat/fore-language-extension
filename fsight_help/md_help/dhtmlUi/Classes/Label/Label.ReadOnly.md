# Label.ReadOnly


## Синтаксис


ReadOnly: Boolean


## Описание


Свойство ReadOnly определяет,
 доступен ли компонент для редактирования.


## Комментарии


Допустимые значения:


	- true.
	 Компонент недоступен для редактирования (по умолчанию);


	- false.
	 Компонент доступен для редактирования.


Значение свойства устанавливается из JSON и с помощью метода setReadOnly,
 а возвращается с помощью метода getReadOnly.
