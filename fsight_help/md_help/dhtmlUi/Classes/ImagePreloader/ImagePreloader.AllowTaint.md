# ImagePreloader.AllowTaint


## Синтаксис


AllowTaint: Boolean


## Описание


Свойство AllowTaint определяет,
 будет ли использоваться прокси-сервер для загрузки изображений.


## Комментарии


Допустимые значения:


	- true.
	 Для загрузки изображений будет использован прокси-сервер;


	- false.
	 Для загрузки изображений прокси-сервер использоваться не будет (по
	 умолчанию).


Значение свойства устанавливается из JSON и с помощью метода setAllowTaint,
 а возвращается с помощью метода getAllowTaint.
