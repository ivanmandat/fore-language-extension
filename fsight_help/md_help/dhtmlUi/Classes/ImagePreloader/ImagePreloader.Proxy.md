# ImagePreloader.Proxy


## Синтаксис


Proxy: String


## Описание


Свойство Proxy определяет адрес
 прокси-сервера для загрузки изображений с другого домена.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setProxy,
 а возвращается с помощью метода getProxy.
