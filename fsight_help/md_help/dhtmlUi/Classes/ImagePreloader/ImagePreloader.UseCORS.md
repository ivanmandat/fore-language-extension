# ImagePreloader.UseCORS


## Синтаксис


UseCORS: Boolean


## Описание


Свойство UseCORS определяет
 использование технологии CORS для загрузки изображений с другого домена.


## Комментарии


Допустимые значения:


	- true.
	 Для загрузки изображений будет использована технология CORS;


	- false.
	 Для загрузки изображений технология CORS использоваться не будет (по
	 умолчанию).


Значение свойства устанавливается из JSON и с помощью метода setUseCORS,
 а возвращается с помощью метода getUseCORS.
