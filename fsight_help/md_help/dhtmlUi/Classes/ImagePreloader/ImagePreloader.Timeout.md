# ImagePreloader.Timeout


## Синтаксис


Timeout: Number


## Описание


Свойство Timeout определяет
 время ожидания загрузки изображений (в миллисекундах).


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTimeout,
 а возвращается с помощью метода getTimeout.
