# ScrollBar.DraggerMinSize


## Синтаксис


DraggerMinSize: Number;


## Описание


Свойство DraggerMinSize определяет
 минимальную ширину ползунка.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDraggerMinSize, а возвращается
 с помощью метода getDraggerMinSize.
