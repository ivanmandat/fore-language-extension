# Конструктор ScrollBar


## Синтаксис


PP.ScrollBar(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор ScrollBar создает экземпляр компонента [ScrollBar](../../Components/ScrollBar/ScrollBar.htm).
