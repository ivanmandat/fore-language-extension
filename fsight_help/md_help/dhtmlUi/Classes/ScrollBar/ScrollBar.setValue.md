# ScrollBar.setValue


## Синтаксис


setValue (position, abs, fireEvent);


## Параметры


position. Позиция
 ползунка;


abs. Определяет,
 какое значение задано в параметре value: false
 - процентное, true -
 абсолютное. По умолчанию задается абсолютное значение в пикселях;


fireEvent. Определяет,
 генерируется ли при выполнении метода событие [ScrollBar.MoveScroll](ScrollBar.MoveScroll.htm).
 Если установлено значение true, событие будет генерироваться. По умолчанию
 событие не генерируется.


## Описание


Метод setValue устанавливает
 позицию ползунка.
