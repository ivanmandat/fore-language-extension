# TextBox.BeforeChange


## Синтаксис


BeforeChange: function(sender, args, timeout);


## Параметры


sender. Источник события;


args. Информация о событии;


timeout. Промежуток времени в миллисекундах, через который вызовется событие.


## Описание


Событие BeforeChange** наступает при начале редактирования [текстового поля](TextBox.htm).
