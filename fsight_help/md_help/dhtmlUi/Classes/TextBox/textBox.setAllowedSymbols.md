# TextBox.setAllowedSymbols


## Синтаксис


setAllowedSymbols(value: Array || RegExp);


## Параметры


*value. Массив или регулярное выражение, определяющее разрешенные к вводу символы.*


## Описание


Метод setAllowedSymbols** устанавливает список символов, разрешенных к вводу.
