# TextBox.EnableEdit


## Синтаксис


EnableEdit: Boolean


## Описание


Свойство EnableEdit** определяет,
 разрешено ли редактирование [текстового поля](TextBox.htm).


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**EnableEdit**,
 а возвращается с помощью метода get**EnableEdit**.


Если для свойства установлено значение true,
 редактирование разрешено, иначе - запрещено.


По умолчанию установлено значение true.
