# TextBox.setRestrictedSymbols


## Синтаксис


setRestrictedSymbols(value: Array || RegExp);


## Параметры


*value. Массив или регулярное выражение, определяющее запрещенные к вводу символы.*


## Описание


Метод setRestrictedSymbols** устанавливает список символов, запрещенных к вводу.
