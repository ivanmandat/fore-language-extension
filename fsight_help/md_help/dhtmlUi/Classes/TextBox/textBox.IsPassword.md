# TextBox.IsPassword


## Синтаксис


IsPassword: Boolean


## Описание


Свойство IsPassword** определяет
 признак скрытия введенного текста.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**IsPassword**,
 а возвращается с помощью метода get**IsPassword****.**


Если для свойства установлено значение true,
 текст будет заменяться, иначе - не будет.


По умолчанию установлено значение false.
