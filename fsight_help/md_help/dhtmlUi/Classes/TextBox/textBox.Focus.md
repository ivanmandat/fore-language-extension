# TextBox.Focus


## Синтаксис


Focus: function(sender, args, timeout);


## Параметры


sender. Источник события;


args. Информация о событии;


timeout. Промежуток времени в миллисекундах, через который вызовется событие.


## Описание


Событие Focus** наступает при получении фокуса [текстовым полем](TextBox.htm).
