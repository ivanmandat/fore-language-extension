# TextBox.AfterTimeout


## Синтаксис


AfterTimeout: function(sender, args, timeout);


## Параметры


sender. Источник события;


args. Информация о событии;


timeout. Промежуток времени в миллисекундах, через который вызовется событие.


## Описание


Событие AfterTimeout** наступает после окончания редактирования [текстового поля](TextBox.htm) по истечении времени задержки между редактированием поля и вызовом события.
