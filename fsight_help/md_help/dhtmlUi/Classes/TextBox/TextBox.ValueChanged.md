# TextBox.ValueChanged


## Синтаксис


ValueChanged: function(sender, args, timeout);


## Параметры


sender. Источник события;


args. Информация о событии.
 Аргументы: Event - произошедшее
 событие; OldValue - значение текстового
 поля до изменения; OldContent
 - содержимое текстового поля до изменения; NewContent
 - содержимое текстового поля после изменения; NewValue
 - значение текстового поля после изменения;


timeout. Промежуток времени
 в миллисекундах, через который вызовется событие.


## Описание


Событие ValueChanged наступает
 после изменения значения [текстового поля](TextBox.htm).
