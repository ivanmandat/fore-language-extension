# TextBox.IsWatermark


## Синтаксис


IsWatermark: Boolean


## Описание


Свойство IsWatermark**
 определяет признак отрисовки прозрачного текста, если [текстовое
 поле](TextBox.htm) пустое и не имеет фокуса.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**IsWatermark**,
 а возвращается с помощью метода get**IsWatermark**.


Если для свойства установлено значение true,
 будет отрисован прозрачный текст, иначе - не будет.


По умолчанию установлено значение false.
