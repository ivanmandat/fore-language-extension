# TextBox.WatermarkText


## Синтаксис


WatermarkText: String


## Описание


Свойство WatermarkText** определяет текст, который будет отображаться, если [текстовое поле](TextBox.htm) пустое и не имеет фокуса.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**WatermarkText**, а возвращается с помощью метода get**WatermarkText**.


Текст будет отображаться, если значение свойства [IsWatermark](textBox.IsWatermark.htm) - true.
