# TextBox.MaxLength


## Синтаксис


MaxLength: Number


## Описание


Свойство MaxLength определяет максимальную длину текста в [текстовом поле](TextBox.htm)**.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**MaxLength**, а возвращается с помощью метода get**MaxLength**.
