# MenuItem.ColumnIndex


## Синтаксис


ColumnIndex: Number


## Описание


Свойство ColumnIndex устанавливает индекс столбца в [спрайте](../../Components/ImageList/ImageList.htm).


## Комментарии


Свойство используется, когда для меню [установлен спрайт](../Menu/Menu.ImageList.htm) (набор пиктограмм в одном графическом файле).


Пиктограммы спрайта располагаются в левой части меню.


Для установки индекса строки пиктограммы в спрайте используется свойство [RowIndex](MenuItem.RowIndex.htm).


Примечание. Свойство актуально, если для свойства [Menu.ShowIcons](../Menu/Menu.ShowIcons.htm) установлено значение true.
