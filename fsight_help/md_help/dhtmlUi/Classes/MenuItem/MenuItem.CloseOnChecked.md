# MenuItem.CloseOnChecked


## Синтаксис


CloseOnChecked: Boolean


## Описание


Свойство CloseOnChecked определяет, закрывается ли меню при выборе указанного элемента.


## Комментарии


Если для свойства установлено значение false, то при выборе элемента меню не закрывается. По умолчанию установлено значение true.


## Пример


Для выполнения примера предполагается наличие на html-странице компонента [Menu](../../Components/Menu/Menu.htm) с дочерним элементом «item2» (см. «[Пример создания компонента Menu](../../Components/Menu/Menu_example.htm)»). Чтобы при выборе элемента с наименованием «item2» меню не закрывалось, используйте свойство CloseOnChecked:


item2.setCloseOnChecked(false);


После выполнения примера при выборе элемента с наименованием «item2» меню закрываться не будет.
