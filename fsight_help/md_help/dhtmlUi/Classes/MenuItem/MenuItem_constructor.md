# Конструктор MenuItem


## Синтаксис


PP.Ui.MenuItem (settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор MenuItem создает экземпляр класса [MenuItem](MenuItem.htm).
