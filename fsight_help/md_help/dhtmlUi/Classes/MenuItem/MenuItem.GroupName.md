# MenuItem.GroupName


## Синтаксис


GroupName: String


## Описание


Свойство GroupName определяет имя группы для элемента меню.


## Комментарии


Для элементов с одинаковыми именами группы при установке значения для свойства [Checked](MenuItem.Checked.htm) будет настроен [переключатель](../../Components/RadioButton/RadioButton.htm). Данный переключатель устанавливается в левой части элемента меню при его выборе. Одновременно может быть установлен только один переключатель.


Свойство актуально, если для свойства [Menu.ShowIcons](../Menu/Menu.ShowIcons.htm) установлено значение true.
