# MenuItem.HotKey


## Синтаксис


HotKey: String


## Описание


Свойство HotKey устанавливает название «горячих клавиш» для пункта меню.


## Комментарии


Значение свойства можно установить только из JSON. Значение свойства возвращается при помощи метода [getHotKey](MenuItem.getHotKey.htm).
