# MenuItem.RowIndex


## Синтаксис


RowIndex: Number


## Описание


Свойство RowIndex устанавливает индекс строки в [спрайте](../../Components/ImageList/ImageList.htm).


## Комментарии


Свойство используется, когда для меню установлен [спрайт](../../Components/ImageList/ImageList.htm) (набор пиктограмм в одном графическом файле).


Пиктограммы [спрайта](../../Components/ImageList/ImageList.htm) располагаются в левой части меню.


Для установки индекса столбца пиктограммы в [спрайте](../../Components/ImageList/ImageList.htm) используется свойство [ColumnIndex](MenuItem.ColumnIndex.htm).


Примечание. Свойство актуально, если для свойства [Menu.ShowIcons](../Menu/Menu.ShowIcons.htm) установлено значение true.
