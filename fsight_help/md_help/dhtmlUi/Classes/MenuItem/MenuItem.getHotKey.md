# MenuItem.getHotKey


## Синтаксис


getHotKey ();


## Описание


Метод getHotKey возвращает название «горячих клавиш» для пункта меню.


## Комментарии


Название «горячих клавиш» задается из JSON при помощи свойства [HotKey](MenuItem.HotKey.htm).

Пример использования метода приведен в описании метода [Menu.getMenuItemById](../Menu/Menu.getMenuItemById.htm).
