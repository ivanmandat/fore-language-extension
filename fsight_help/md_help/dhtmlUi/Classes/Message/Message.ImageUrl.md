# Message.ImageUrl


## Синтаксис


ImageUrl: String


## Описание


Свойство ImageUrl определяет
 путь к пиктограмме окна.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setImageUrl,
 а возвращается с помощью метода getImageUrl.


Пиктограмма будет отображаться только если окно имеет тип
 "Custom".
