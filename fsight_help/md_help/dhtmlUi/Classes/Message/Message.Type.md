# Message.Type


## Синтаксис


Type: PP.Ui.[MessageType](../../Enums/MessageType.htm)


## Описание


Свойство Type определяет тип
 окна и его пиктограмму.


## Комментарии


Значение свойства - элемент перечисления PP.Ui.[MessageType](../../Enums/MessageType.htm).


Значение свойства устанавливается из JSON и с помощью метода setType,
 а возвращается с помощью метода getType.
