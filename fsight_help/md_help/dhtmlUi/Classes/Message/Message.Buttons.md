# Message.Buttons


## Синтаксис


Buttons: String


## Описание


Свойство Buttons определяет
 кнопки окна.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setButtons,
 а возвращается с помощью метода getButtons.


Допустимые значения содержатся в перечислении PP.Ui.MessageButtons:


		 Значение
		 Описание


		 Ok
		 Кнопка «Ок».


		 OkCancel
		 Кнопки «Ок» и «Отмена».


		 YesNo
		 Кнопки «Да» и «Нет».


		 YesNoCancel
		 Кнопки «Да», «Нет» и «Отмена».
