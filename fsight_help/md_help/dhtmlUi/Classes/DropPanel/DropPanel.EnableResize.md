# DropPanel.EnableResize


## Синтаксис


EnableResize: Boolean


## Описание


Свойство EnableResize определяет возможность изменять размеры компонента.


## Комментарии


По умолчанию установлено значение true, то есть размеры панели можно изменять.


## Пример


Для выполнения примера создайте компонент [DropPanel](../../Components/DropPanel/DropPanel.htm) (см. «[Пример создания компонента DropPanel](../../Components/DropPanel/Example_DropPanel.htm)») с наименованием «DP». Для того чтобы размеры созданной панели нельзя было изменять установите для свойства EnableResize значение false:


EnableResize: false
