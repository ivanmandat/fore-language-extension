# DropPanel.EnableResizeBottom


## Синтаксис


EnableResizeBottom: Boolean


## Описание


Свойство EnableResizeBottom определяет возможность изменения высоты компонента с помощью захвата и перетаскивания его нижней границы курсором мыши.


## Комментарии


По умолчанию установлено значение true, то есть высоту компонента можно изменять с помощью захвата и перетаскивания его нижней границы курсором мыши.


## Пример


Для выполнения примера создайте компонент [DropPanel](../../Components/DropPanel/DropPanel.htm) (см. «[Пример создания компонента DropPanel](../../Components/DropPanel/Example_DropPanel.htm)»). Для того чтобы размер нижней стороны панели нельзя было изменять установите для свойства EnableResizeBottom значение false:


EnableResizeBottom: false
