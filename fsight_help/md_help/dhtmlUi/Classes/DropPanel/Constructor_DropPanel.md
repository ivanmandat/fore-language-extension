# Конструктор DropPanel


## Синтаксис


PP.Ui.DropPanel(settings);


## Параметры


settings. JSON объект со значениями свойств компонента.


## Описание


Конструктор DropPanel создает экземпляр компонента [DropPanel](../../Components/DropPanel/DropPanel.htm).
