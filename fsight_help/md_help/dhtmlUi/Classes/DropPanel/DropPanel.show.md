# DropPanel.show


## Синтаксис


show (left: Number, top: Number, notCreateFrame: Boolean)


## Параметры


left. Левая координата компонента;


top. Верхняя координата компонента;


notCreateFrame. Параметр, определяющий, создается ли подкладка под панель. Если установлено значение true, то подкладка не создается. По умолчанию установлено значение false.


## Описание


Метод show раскрывает панель.
