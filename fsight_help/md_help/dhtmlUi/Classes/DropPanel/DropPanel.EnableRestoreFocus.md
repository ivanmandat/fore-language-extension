# DropPanel.EnableRestoreFocus


## Синтаксис


EnableRestoreFocus: Boolean


## Описание


Свойство EnableRestoreFocus
 определяет возможность восстановления фокуса при закрытии панели.


## Комментарии


При значении true последний
 элемент, на котором был установлен фокус до открытия панели будет сохранен.
 После закрытия панели фокус будет восстановлен на сохраненном элементе.


Значение свойства устанавливается из JSON и с помощью метода setEnableRestoreFocus,
 а возвращается с помощью метода getEnableRestoreFocus.
