# DropPanel.EnableResizeTop


## Синтаксис


EnableResizeTop: Boolean


## Описание


Свойство EnableResizeTop определяет возможность изменения высоты компонента с помощью захвата и перетаскивания его верхней границы курсором мыши.


## Комментарии


По умолчанию установлено значение true, то есть высота компонента может быть изменена с помощью захвата и перетаскивания его верхней границы курсором мыши.
