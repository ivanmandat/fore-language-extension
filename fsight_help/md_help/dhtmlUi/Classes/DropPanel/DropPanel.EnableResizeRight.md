# DropPanel.EnableResizeRight


## Синтаксис


EnableResizeRight: Boolean


## Описание


Свойство EnableResizeRight определяет возможность изменения ширины компонента с помощью захвата и перетаскивания его правой границы курсором мыши.


## Комментарии


По умолчанию установлено значение true, то есть ширина компонента может быть изменена с помощью захвата и перетаскивания его правой границы курсором мыши.


## Пример


Для выполнения примера создайте компонент [DropPanel](../../Components/DropPanel/DropPanel.htm) (см. «[Пример создания компонента DropPanel](../../Components/DropPanel/Example_DropPanel.htm)»). Для того чтобы размер правой стороны панели нельзя было изменять установите для свойства EnableResizeRightзначение false:


EnableResizeRight: false
