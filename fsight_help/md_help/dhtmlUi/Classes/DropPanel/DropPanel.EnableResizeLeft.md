# DropPanel.EnableResizeLeft


## Синтаксис


EnableResizeLeft: Boolean


## Описание


Свойство EnableResizeLeft определяет возможность изменения ширины компонента с помощью захвата и перетаскивания его левой границы курсором мыши.


## Комментарии


По умолчанию установлено значение true, то есть ширина компонента может быть изменена с помощью захвата и перетаскивания его левой границы курсором мыши.
