# DropPanel.isVisibleArea


## Синтаксис


isVisibleArea(top: Number, left: Number, doNotConsiderBorderValues: Boolean)


## Параметры


top Верхняя координата компонента;


left. Левая координата компонента;


doNotConsiderBorderValues. Параметр, определяющий необходимость учета минимальных/максимальных размеров компонента.


## Описание


Метод isVisibleArea определяет, входит ли раскрывающая часть в видимую область.


## Комментарии


Данный метод применяется при создании меню.
