# GridPanel.setRowHeight


## Синтаксис


setRowHeight (rowNumber, height);


## Параметры


rowNumber. Порядковый номер строки;


height. Высота строки.


## Описание


Метод setRowHeight устанавливает высоту строки таблицы.
