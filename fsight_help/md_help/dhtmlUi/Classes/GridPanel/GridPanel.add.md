# GridPanel.add


## Синтаксис


add (ppControl, left, top);


## Параметры


ppControl. Компонент, экземпляр класса [Control](../Control/Control.htm), который необходимо добавить в таблицу;


left. Левая координата компонента;


top. Верхняя координата компонента.


## Описание


Метод add добавляет компонент в таблицу.


## Комментарии


Метод используется при статическом размещении компонента. При динамическом размещении компоненты добавляются в таблицу при помощи свойства [Items](../Container/Container.Items.htm).
