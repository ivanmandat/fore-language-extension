# Конструктор GridPanel


## Синтаксис


GridPanel (args);


## Параметры


args. JSON-объект со значениями свойств компонента.


## Описание


Конструктор GridPanel создает компонент [GridPanel](../../Components/GridPanel/GridPanel.htm).


## Комментарии


При статическом размещении компонента дочерние элементы добавляются при помощи метода [add](GridPanel.add.htm), при динамическом - при помощи свойства [Container.Items](../Container/Container.Items.htm) в параметре конструктора.
