# Конструктор NumberEdit


## Синтаксис


PP.Ui.NumberEdit(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор NumberEdit создает экземпляр компонента [NumberEdit](../../Components/NumberEdit/NumberEdit.htm).
