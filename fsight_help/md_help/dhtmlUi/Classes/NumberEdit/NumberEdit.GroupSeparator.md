# NumberEdit.GroupSeparator


## Синтаксис


GroupSeparator: String


## Описание


Свойство GroupSeparator устанавливает разделитель тысяч.


## Комментарии


Значение свойства должно содержать один символ или пустую строку. В качестве значения свойства по умолчанию установлен пробел или значение, которое подтягивается из PP.getCurrentCulture().GroupSeparator.
