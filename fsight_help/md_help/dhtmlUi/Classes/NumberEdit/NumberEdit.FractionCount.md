# NumberEdit.FractionCount


## Синтаксис


FractionCount: Number


## Описание


Свойство FractionCount устанавливает количество знаков после [разделителя разрядов](NumberEdit.DecimalSeparator.htm).


## Комментарии


По умолчанию количество знаков после разделителя разрядов не установлено.
