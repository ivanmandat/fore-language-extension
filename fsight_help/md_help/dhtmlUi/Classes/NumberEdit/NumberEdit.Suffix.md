# NumberEdit.Suffix


## Синтаксис


Suffix: String;


## Описание


Свойство Suffix устанавливает
 суффикс для значения в компоненте.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSuffix.


## Пример


Для выполнения примера необходимо создать html-страницу и выполнить
 следующие действия:


1. Добавить ссылки на файлы PP.css и PP.js.


2. Внутри тега <body> разместить блок с идентификатором «NE1»:


<div id="NE1"></div>


3. Далее на страницу внутри тега <body> необходимо добавить сценарий,
 который создаёт числовой редактор с отображаемыми префиксом и суффиксом:


<script type="text/javascript">
// Создаём экземпляр компонента NumberEdit
var ne = new PP.Ui.NumberEdit({
    ParentNode: document.getElementById("NE1"), // Контейнер для компонента
    Width: "170px",
    Height: "21px",
    Value: 22, // Устанавливаем значение
    Suffix: " °C", // Суффикс
    Prefix: "t " // Префикс
});
</script>
В результате выполнения примера на html-странице будет размещен числовой
 редактор, значение которого имеет префикс «t » и суффикс « °C»:
