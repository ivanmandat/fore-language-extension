# NumberEdit.getUnroundValue


## Синтаксис


getUnroundValue();


## Описание


Метод getUnroundValue возвращает
 числовое значение компонента без округления в строковом виде.


## Комментарии


Числовое значение компонента задается с помощью свойства [NumberEdit.Value](NumberEdit.Value.htm).
 Для получения заданного значения используйте метод [NumberEdit.getValue](NumberEdit.Value.htm)
 или getUnroundValue. В первом
 случае значение округляется в зависимости от заданного значения свойства
 [NumberEdit.FractionCount](NumberEdit.FractionCount.htm), во
 втором случае значение возвращается без округления.
