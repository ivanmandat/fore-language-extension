# NumberEdit.EnableGroupSeparator


## Синтаксис


EnableGroupSeparator: Boolean;


## Описание


Свойство EnableGroupSeparator
 определяет разделение групп [разрядов](NumberEdit.DecimalSeparator.htm).


## Комментарии


При значении true - числа длинной четыре и более знаков будут разделены
 по разрядам.


По умолчанию разделение групп разрядов не установлено.
