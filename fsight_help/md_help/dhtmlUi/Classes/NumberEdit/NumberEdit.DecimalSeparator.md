# NumberEdit.DecimalSeparator


## Синтаксис


DecimalSeparator: String


## Описание


Свойство DecimalSeparator устанавливает символ разделителя разрядов.


## Комментарии


По умолчанию символ разделителя разрядов - запятая.


В качестве значения свойства может устанавливаться только один символ.
