# NumberEdit.IsNullable


## Синтаксис


IsNullable: Boolean;


## Описание


Свойство IsNullable определяет
 отображение пустого поля ввода.


## Комментарии


Допустимые значения:


	- True. Отображение пустого
	 поля ввода разрешено;


	- False. Отображение пустого
	 поля ввода запрещено.


По умолчанию свойство принимает значение False.
