# ProgressBar.DisplayValue


## Синтаксис


DisplayValue: Boolean


## Описание


Свойство DisplayValue определяет, будет ли отображаться значение на компоненте [ProgressBar](../../Components/ProgressBar/ProgressBar.htm).


## Комментарии


Если для свойства установлено true, то на компоненте [ProgressBar](../../Components/ProgressBar/ProgressBar.htm) отображается его текущее значение:


![](ProgressBar1.gif)


Если установлено false (по умолчанию), значение не отображается:


![](ProgressBar_DisplayValue.gif)
