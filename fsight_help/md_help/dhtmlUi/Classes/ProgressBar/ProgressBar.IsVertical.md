# ProgressBar.IsVertical


## Синтаксис


IsVertical: Boolean


## Описание


Свойство IsVertical определяет, размещается ли компонент вертикально.


## Комментарии


Если для свойства установлено true, компонент размещается вертикально:


![](ProgressBar_Vertical.gif)


Если для свойства установлено false (по умолчанию), компонент размещается горизонтально:


![](ProgressBar1.gif)
