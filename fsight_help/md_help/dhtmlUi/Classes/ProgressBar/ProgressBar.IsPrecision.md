# ProgressBar.IsPrecision


## Синтаксис


IsPrecision: Boolean


## Описание


Свойство IsPrecision определяет, будет ли осуществляться работа с тысячными долями.


## Комментарии


Если для свойства установлено true, то значение компонента будет изменяться с учетом тысячных долей:


![](ProgressBar_IsPrecision.gif)


Если для свойства установлено false (по умолчанию), значение компонента будет изменяться без учета тысячных долей:


![](ProgressBar1.gif)
