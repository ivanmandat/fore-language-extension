# ColorCombo.ColorChanged


## Синтаксис


ColorChanged: function (sender,args)


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие ColorChanged наступает при изменении цвета.

Пример использования события приведен в описании свойства MapLegend.HighlightBrush.
