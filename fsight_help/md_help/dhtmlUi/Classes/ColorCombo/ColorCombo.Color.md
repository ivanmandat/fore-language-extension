# ColorCombo.Color


## Синтаксис


Color: PP.[Color](dhtmlCommon.chm::/Classes/pp/color/color.htm);


## Описание


Свойство Color устанавливает
 текущий цвет, который также является цветом кнопки вызова раскрывающегося
 списка.


## Комментарии


Формат цвета может быть задан следующими способами:


	- «A=255;R=230;G=230;B=230»;


	- «#FFFFFF»;


	- «#FFFFFFFF»;


	- «White», «Red», «Black» и т.д.
