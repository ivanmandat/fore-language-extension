# ColorCombo.ValueChanged


## Синтаксис


ValueChanged: function(sender,args)


## Параметры


sender. Источник события;


args. Информация
 о событии. Аргументы: Event
 - произошедшее событие; OldColor - старое значение цвета; NewColor - новое значение цвета; Cancel - признак отмены действия: если значение
 равно true, то действие было отменено.


## Описание


Событие ValueChanged наступает
 при изменении значения компонента.
