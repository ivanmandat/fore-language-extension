# Wizard.removeStep


## Синтаксис


removeStep (step: PP.Ui.[WizardStep](../WizardStep/WizardStep.htm), withDispose: Boolean);


## Параметры


step. Шаг, который нужно удалить;


withDispose. Определяет, будет ли шаг уничтожен (true) или удален из списка шагов (false).


## Описание


Метод removeStep удаляет [шаг](../WizardStep/WizardStep.htm) из мастера.

Пример использования приведен в описании метода [Wizard.addStep](Wizard.addStep.htm).
