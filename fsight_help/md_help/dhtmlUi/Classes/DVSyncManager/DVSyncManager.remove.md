# DVSyncManager.remove


## Синтаксис


remove(visualizer_id: [PP.Ui.DataVisualizer](../DataVisualizer/DataVisualizer.htm)
 | String);


## Параметры


remove. Визуализатор или его
 идентификатор.


## Описание


Метод remove удаляет указанный
 визуализатор из коллекции синхронизируемых элементов.
