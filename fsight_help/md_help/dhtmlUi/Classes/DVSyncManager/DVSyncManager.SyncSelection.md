# DVSyncManager.SyncSelection


## Синтаксис


SyncSelection: Boolean


## Описание


Свойство SyncSelection
 определяет признак синхронизации выделений.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSyncSelection,
 а возвращается с помощью метода getSyncSelection.


Свойство содержит значение true,
 если выделение синхронизируется, и false
 - если выделение не синхронизируется.


По умолчанию свойство имеет значение true.
