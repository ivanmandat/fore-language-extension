# DVSyncManager.Visualizers


## Синтаксис


Visualizer: Object


## Описание


Свойство Visualizer определяет
 массив визуализаторов, которые будут синхронизироваться.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setVisualizer,
 а возвращается с помощью метода getVisualizer.
