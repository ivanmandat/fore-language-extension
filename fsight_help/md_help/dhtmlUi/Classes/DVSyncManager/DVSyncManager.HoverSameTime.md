# DVSyncManager.HoverSameTime


## Синтаксис


HoverSameTime: Boolean


## Описание


Свойство HoverSameTime
 определяет признак подсветки только тех элементов, для которых установлены
 одинаковые значения временной шкалы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHoverSameTime,
 а возвращается с помощью метода getHoverSameTime.


Свойство содержит значение true,
 если подсвечиваются только те элементы, для которых установлены одинаковые
 значения временной шкалы, и false
 - если подсвечиваются все элементы.


По умолчанию свойство имеет значение true.
