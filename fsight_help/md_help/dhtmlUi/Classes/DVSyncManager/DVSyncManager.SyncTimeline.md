# DVSyncManager.SyncTimeline


## Синтаксис


SyncTimeline: Boolean


## Описание


Свойство SyncTimeline
 определяет признак синхронизации временной шкалы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSyncTimeline,
 а возвращается с помощью метода getSyncTimeline.


Свойство содержит значение true,
 если временная шкала синхронизируется, и false
 - если временная шкала не синхронизируется.


По умолчанию свойство имеет значение true.
