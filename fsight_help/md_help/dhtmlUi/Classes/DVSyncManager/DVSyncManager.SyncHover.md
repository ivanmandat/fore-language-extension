# DVSyncManager.SyncHover


## Синтаксис


SyncHover: Boolean


## Описание


Свойство SyncHover определяет
 признак синхронизации подсветки.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSyncHover,
 а возвращается с помощью метода getSyncHover.


Свойство содержит значение true,
 если подсветка синхронизируется, и false
 - если подсветка не синхронизируется.


По умолчанию свойство имеет значение true.
