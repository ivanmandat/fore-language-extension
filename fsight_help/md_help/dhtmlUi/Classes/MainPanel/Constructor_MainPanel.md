# Конструктор MainPanel


## Синтаксис


PP.Ui.MainPanel(settings);


## Параметры


settings. JSON-объект со значениями
 свойств компонента.


## Описание


Конструктор MainPanel создает
 экземпляр компонента [MainPanel](../../Components/MainPanel/MainPanel.htm).
