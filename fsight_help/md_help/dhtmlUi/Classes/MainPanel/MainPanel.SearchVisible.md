# MainPanel.SearchVisible


## Синтаксис


SearchVisible: Boolean


## Описание


Свойство SearchVisible определяет
 видимость строки поиска элементов.


## Комментарии


Допустимые значения:


	- true.
	 Строка поиска элементов будет отображена (по умолчанию);


	- false.
	 Строка поиска элементов будет отсутствовать.


Значение свойства устанавливается из JSON и с помощью метода setSearchVisible,
 а возвращается с помощью метода getSearchVisible.
