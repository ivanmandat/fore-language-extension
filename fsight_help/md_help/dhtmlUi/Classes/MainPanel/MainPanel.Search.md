# MainPanel.Search


## Синтаксис


Search: function (sender, args)


## Параметры


sender. Источник события;


args. Информация
 о событии. Аргументы: Value
 - искомое значение.


## Описание


Событие Search наступает после
 нажатия клавиши ENTER в строке поиска.


## Пример


Для реализации примера необходимо создать раскрывающуюся панель с меню
 с наименованием «mainPanel» (см. «[Пример
 создания компонента MainPanel](../../Components/MainPanel/Example_MainPanel.htm)»). Добавим обработчик события нажатия
 клавиши ENTER в строке поиска:


// Добавляем обработчик события нажатия клавиши ENTER в строке поиска
mainPanel.Search.add(function (sender, args) {
    console.log("Искомое значение: " + args.Value);
});
После ввода текста в строку поиска и нажатия клавиши ENTER в консоль
 будет выведено сообщение о искомом значении, а панель будет скрыта:


Искомое значение: Элемент 1
