# MainPanel.insertItem


## Синтаксис


insertItem(item: PP.Ui.[MainPanelItem](../MainPanelItem/MainPanelItem.htm)
 || Object, index: Number);


## Параметры


item. Вставляемый элемент;


index. Индекс добавляемого
 элемента.


## Описание


Метод insertItem добавляет элемент
 в указанную позицию.
