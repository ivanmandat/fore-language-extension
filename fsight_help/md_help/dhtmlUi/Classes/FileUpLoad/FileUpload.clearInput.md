# FileUpload.clearInput


## Синтаксис


clearInput (index)


## Параметры


index. Индекс загрузки.


## Описание


Метод clearInput очищает загрузку по индексу.


## Пример


Для выполнения примера создайте компонент [FileUpload](../../Components/FileUpload/FileUpload.htm) с наименованием «fileUpload» (см. «[Конструктор FileUpload](Constructor_FileUpload.htm)»). Загрузите в него 2 файла. Чтобы очистить первую загрузку, используйте следующий код:


fileUpload.clearInput(0);


После выполнения примера будет очищена первая загрузка.
