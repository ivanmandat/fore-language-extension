# FileUpload.clearAllInputs


## Синтаксис


clearAllInputs ()


## Описание


Метод clearAllInputs осуществляет очистку всех загрузок.


## Пример


Для выполнения примера создайте компонент [FileUpload](../../Components/FileUpload/FileUpload.htm) с наименованием «fileUpload» (см. «[Конструктор FileUpload](Constructor_FileUpload.htm)»). Загрузите в него несколько файлов. Чтобы очистить все загрузки, используйте метод clearAllInputs:


fileUpload.clearAllInputs();


После выполнения примера загрузки будут очищены.
