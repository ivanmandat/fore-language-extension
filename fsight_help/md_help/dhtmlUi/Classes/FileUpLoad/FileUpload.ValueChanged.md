# FileUpload.ValueChanged


## Синтаксис


ValueChanged: function(sender, args)


## Параметры


sender. Источник события;


args. Информация о событии.
 Доступные аргументы: FileInput
 - DOM-вершина загрузчика, значение которого изменилось.


## Описание


Событие ValueChanged наступает
 после изменения значения загрузчика.
