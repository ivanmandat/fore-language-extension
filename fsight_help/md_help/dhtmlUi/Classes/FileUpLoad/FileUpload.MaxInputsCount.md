# FileUpload.MaxInputsCount


## Синтаксис


MaxInputsCount: Number


## Описание


Свойство MaxInputsCount устанавливает
 количество полей ввода для файлов.


## Комментарии


Свойство актуально, если для свойства [FileUpload.AutoExpandable](FileUpload.AutoExpandable.htm)
 установлено значение true.


По умолчанию установлено значение 3.
