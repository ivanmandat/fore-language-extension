# FileUpload.FileName


## Синтаксис


FileName: String


## Описание


Свойство FileName содержит имя
 последнего загруженного файла.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setFileName,
 а возвращается с помощью метода getFileName.
