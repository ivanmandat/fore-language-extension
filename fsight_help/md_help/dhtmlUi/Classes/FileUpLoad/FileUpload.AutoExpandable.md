# FileUpload.AutoExpandable


## Синтаксис


AutoExpandable: Boolean


## Описание


Свойство AutoExpandable определяет, будет ли при загрузке файла появляться дополнительное поле.


## Комментарии


Если установлено значение true, дополнительный элемент создается при выборе файла. Если false - не создается. По умолчанию установлено значение false.


Максимальное количество полей задается в свойстве [FileUpload.MaxInputsCount](FileUpload.MaxInputsCount.htm).
