# PP.Ui.getMarginsHeight


## Синтаксис


PP.Ui.getMarginsHeight(el: HTMLElement);


## Параметры


el. HTML-элемент;


## Описание


Метод getMarginsHeight возвращает
 сумму высот внутренних и внешних отступов и границ элемента в пикселях.
