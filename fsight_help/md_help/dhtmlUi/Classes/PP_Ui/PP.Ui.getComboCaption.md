# PP.Ui.getComboCaption


## Синтаксис


PP.Ui.getComboCaption(text1: String, text2: String,
 width: Number, isRtl: Boolean, count: Number, node: HTMlElement, args:
 Object);


## Параметры


text1. Первая
 проверяемая строка;


text2.
 Вторая проверяемая строка;


width. Проверяемая
 ширина;


isRtl. Порядок
 написания. Аргумент не используется, нужен для того, чтобы не менять сигнатуру
 метода;


count.
 Число в скобках в конце полученной строки;


node. DOM-вершина,
 в которую будет вставляться текст;


args.
 Аргументы.


## Описание


Метод getComboCaption возвращает
 строку полученную в результате соединения и обрезания двух строк при помещении
 их в элемент с заданной шириной.


## Пример


Для выполнения примера необходимо наличие ссылки на файл сценария PP.js.
 Приведенный ниже код выполняется из консоли браузера. Получаем строку
 из строк "Первая строка" и "Вторая строка", которую
 необходимо поместить в элемент с шириной 100:


console.log(PP.Ui.getComboCaption("Первая строка", "Вторая строка", 100, false, 1));
В результате в консоль будет выведена новая строка:


Перв... Втор...
