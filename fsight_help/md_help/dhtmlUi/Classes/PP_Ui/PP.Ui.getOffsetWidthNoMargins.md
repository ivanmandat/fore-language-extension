# PP.Ui.getOffsetWidthNoMargins


## Синтаксис


PP.Ui.getOffsetWidthNoMargins(el: HTMLElement);


## Параметры


el. HTML-элемент;


## Описание


Метод getOffsetWidthNoMargins
 возвращает ширину элемента в пискелях без учета внутренних и внешних отступов
 и границ.
