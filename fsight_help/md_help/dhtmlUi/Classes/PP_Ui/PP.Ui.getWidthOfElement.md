# PP.Ui.getWidthOfElement


## Синтаксис


PP.Ui.getWidthOfElement(text: String, node: HTMLElement);


## Параметры


text. Проверяемый текст.


node. HTML-элемент;


## Описание


Метод getWidthOfElement возвращает
 ширину строки в пикселях, занимаемую текстом в данном элементе.
