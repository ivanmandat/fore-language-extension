# PP.Ui.getConfig


## Синтаксис


PP.Ui.getConfig(selector);


## Параметры


selector. Наименование
 свойства вложенного объекта класса [PP.Ui.ConfigClass](dhtmlGraphicBase.chm::/Classes/PP_Ui/Config/PP.Ui.ConfigClass.htm),
 настройки которого будут возвращены. Если параметр не указан, будет возвращаться
 класс [PP.Ui.ConfigClass](dhtmlGraphicBase.chm::/Classes/PP_Ui/Config/PP.Ui.ConfigClass.htm).


## Описание


Метод getConfig возвращает объект
 глобальных настроек визуальных компонентов.

Пример использования приведен в описании метода [PP.Ui.ConfigClass.getChart](dhtmlGraphicBase.chm::/Classes/PP_Ui/Config/PP.Ui.ConfigClass.getChart.htm).
