# PP.Ui.setEnblTo


## Синтаксис


PP.Ui.setEnblTo(value: Boolean, nodes: Array);


## Параметры


values. Устанавливаемое
 значение. Допустимые значения: true - устанавливается признак доступности,
 false - снимается признак доступности;


nodes.
 Вершины, которым устанавливается/снимается признак доступности. Элементы
 массива - объекты типа HTMLElement.


## Описание


Метод setEnblTo предназначен
 для установки/снятия признака доступности вершин.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылок на
 файлы сценария PP.js и файл стилей PP.css, в теге <body> html-страницы
 элемента <div> с идентификатором «button». В событии onload тега
 <body> необходимо указать вызов функции createButton(). Добавим
 кнопку на страницу:


function createButton() {
    // Создаем кнопку
    button = new PP.Ui.Button({
        // Устанавливаем родительский элемент
        ParentNode: "button",
        // Устанавливаем содержимое кнопки
        Content: "Кнопка",
    });
    // Снимаем признак доступности с элемента кнопки
    PP.Ui.setEnblTo(false, [ button.getDomNode() ]);
}
В результате кнопка будет недоступна:
