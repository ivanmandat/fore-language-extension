# PP.Ui.getCtrlById


## Синтаксис


PP.Ui.getCtrlById(id: String);


## Параметры


id. Идентификатор
 главной DOM-вершины, принадлежащей элементу управления


## Описание


Метод getCtrlById возвращает
 элемент управления по идентификатору DOM-вершины, принадлежащей ему.


## Комментарии


Возвращаемое значение - значение, типа Object. Если DOM-вершина с указанным
 идентификатором не существует, то вернется null.
 Если элемент управления не будет найден, то также вернется null.
