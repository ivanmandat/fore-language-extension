# PP.Ui.getRetinaUrl


## Синтаксис


PP.Ui.getRetinaUrl(url: String);


## Параметры


url. Адрес;


## Описание


Метод getRetinaUrl подставляет
 приставку ".retina" к переданному адресу.


## Пример


Для выполнения примера необходимо наличие ссылки на файл сценария PP.js.
 Приведенный ниже код выполняется из консоли браузера. Получаем новый адрес
 с приставкой ".retina":


console.log("Новый адрес: " + PP.Ui.getRetinaUrl("data/file"));
В результате в консоль будет выведен новый адрес:


Новый адрес: retina.data/file
