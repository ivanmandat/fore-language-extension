# PP.Ui.getBordersWidth


## Синтаксис


PP.Ui.getBordersWidth(el: HTMLElement);


## Параметры


el. HTML-элемент;


## Описание


Метод getBordersWidth возвращает
 суммарную ширину внутренних отступов и границ элемента в пикселях.
