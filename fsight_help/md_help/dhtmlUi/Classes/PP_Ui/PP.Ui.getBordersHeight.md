# PP.Ui.getBordersHeight


## Синтаксис


PP.Ui.getBordersHeight(el: HTMLElement);


## Параметры


el. HTML-элемент;


## Описание


Метод getBordersHeight возвращает
 сумму высот внутренних отступов и границ элемента в пикселях.
