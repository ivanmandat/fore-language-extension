# PP.Ui.getMarginsWidth


## Синтаксис


PP.Ui.getMarginsWidth(el: HTMLElement);


## Параметры


el. HTML-элемент;


## Описание


Метод getMarginsWidth возвращает
 сумму ширин внутренних и внешних отступов и границ элемента в пикселях.
