# ToolBar.removeItemById


## Синтаксис


removeItemById (id);


## Параметры


id. Ключ дочернего элемента
 панели инструментов.


## Описание


Метод removeItemById удаляет
 дочерний элемент по указанному идентификатору.


## Комментарии


Идентификатор задается в свойстве [Id](dhtmlCommon.chm::/Classes/object/object.id.htm).


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 [ToolBar](ToolBar.htm) с наименованием «toolBar», который содержит
 элемент с идентификатором «b1». Создадим html-кнопку при нажатии на которую
 будет удален указанный элемент.


Javascript-код функции для удаления элемента панели управления:


function remove() {


        toolBar.removeItemById("b1");


    }


Html-кнопка для удаления элемента:


<button onclick="remove();">Удалить</button>


При нажатии на кнопку «Удалить» с панели управления будет удален элемент
 с указанным идентификатором.
