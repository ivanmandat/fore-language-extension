# ToolBar.ImageList


## Синтаксис


ImageList: PP.[ImageList](dhtmlCommon.chm::/Classes/imagelist/imagelist.htm)


## Описание


Свойство ImageList возвращает
 экземпляр невизуального компонента [ImageList](../../Components/ImageList/ImageList.htm).


## Комментарии


Компонент [ImageList](dhtmlCommon.chm::/Classes/imagelist/imagelist.htm)
 содержит путь к файлу с набором пиктограмм. Пиктограммы используются как
 фоновые изображения для [кнопок](../ToolBarButton/ToolBarButton.htm).
 Для выбора нужной пиктограммы из набора используйте свойства [ToolBarButton.ColumnIndex](../ToolBarButton/ToolBarButton.ColumnIndex.htm)
 и [ToolBarButton.RowIndex](../ToolBarButton/ToolBarButton.RowIndex.htm).


Значение свойства возвращает метод [getImageList](ToolBar.getImageList.htm).


Установить значение свойства можно только из JSON.
