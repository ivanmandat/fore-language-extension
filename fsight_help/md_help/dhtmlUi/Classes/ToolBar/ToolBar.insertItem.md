# ToolBar.insertItem


## Синтаксис


insertItem (item:  PP.Ui.[Control](../Control/Control.htm), index: Integer);


## Параметры


item. Элемент панели инструментов - экземпляр класса-наследника PP.Ui.[Control](../Control/Control.htm);


index. Позиция, в которую будет добавлено новый элемент. Нумерация начинается с 0.


## Описание


Метод insertItem вставляет новый элемент панели инструментов в указанную позицию.
