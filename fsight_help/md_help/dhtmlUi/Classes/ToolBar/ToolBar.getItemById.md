# ToolBar.getItemById


## Синтаксис


getItemById (id);


## Параметры


id. Идентификатор дочернего
 элемента панели инструментов.


## Описание


Метод getItemById возвращает
 дочерний элемент панели инструментов по указанному идентификатору.


## Комментарии


Идентификатор задается в свойстве [Id](dhtmlCommon.chm::/Classes/object/object.id.htm).
