# ToolBar.addItem


## Синтаксис


addItem (item: PP.Ui.[Control](../Control/Control.htm));


## Параметры


item. Элемент панели инструментов - экземпляр класса-наследника PP.Ui.[Control](../Control/Control.htm).


## Описание


Метод addItem добавляет в панель инструментов новый дочерний элемент.
