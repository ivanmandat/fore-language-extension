# ToolBar.beginUpdate


## Синтаксис


beginUpdate();


## Описание


Метод beginUpdate отключает
 перерисовку панели инструментов до тех пор, пока не будет вызван метод
 [endUpdate](ToolBar.endUpdate.htm).
