# ToolBar.removeItemByIndex


## Синтаксис


removeItemByIndex (index: Number, isDispose: Boolean);


## Параметры


index. Индекс дочернего элемента панели инструментов, который необходимо удалить;


isDispose. Определяет, удалять ли элемент полностью.


## Описание


Метод removeItemByIndex удаляет дочерний элемент панели инструментов по указанному индексу.
