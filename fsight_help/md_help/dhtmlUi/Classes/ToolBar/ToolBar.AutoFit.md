# ToolBar.AutoFit


## Синтаксис


AutoFit: Boolean


## Описание


Свойство AutoFit определяет
 автоматическое выравнивание элементов панели инструментов.


## Комментарии


Допустимые значения:


	- true. Элементы будут
	 выравнены;


	- false. По умолчанию.
	 Элементы не будут выравнены.


Значение свойства устанавливается из JSON и с помощью метода setAutoFit,
 а возвращается с помощью метода getAutoFit.
