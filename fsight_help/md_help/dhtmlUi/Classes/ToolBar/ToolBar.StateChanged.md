# ToolBar.StateChanged


## Синтаксис


StateChanged: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.Item - объект, состояние которого было изменено.


## Описание


Событие StateChanged наступает
 при смене состояния элемента панели инструмента.
