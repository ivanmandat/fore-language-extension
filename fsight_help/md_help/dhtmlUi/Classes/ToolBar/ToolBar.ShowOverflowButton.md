# ToolBar.ShowOverflowButton


## Синтаксис


ShowOverflowButton: Boolean


## Описание


Свойство ShowOverflowButton
 определяет наличие кнопки меню со скрытыми элементам.


## Комментарии


Допустимые значения:


	- true. Панель инструментов
	 будет содержать кнопку меню со скрытыми элементами;


	- false. По умолчанию.
	 Все элементы будут находиться на панели.


Значение свойства устанавливается из JSON и с помощью метода setShowOverflowButton,
 а возвращается с помощью метода getShowOverflowButton.
