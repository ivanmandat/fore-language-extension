# ToolBar.IsSplitGroups


## Синтаксис


IsSplitGroups: Boolean


## Описание


Свойство IsSplitGroups определяет,
 будет ли добавлен отступ между элементами группы панели инструментов.


## Комментарии


Допустимые значения:


	- true. Между элементами
	 группы будет добавлен отступ;


	- false. По умолчанию.
	 Между элементами группы отступ добавлен не будет.


Значение свойства устанавливается из JSON и с помощью метода setIsSplitGroups,
 а возвращается с помощью метода getIsSplitGroups.
