# ToolBar.endUpdate


## Синтаксис


endUpdate();


## Описание


Метод endUpdate включает перерисовку
 панели инструментов и обновляет её.


## Комментарии


Использует в связке с методом [beginUpdate](ToolBar.beginUpdate.htm).
