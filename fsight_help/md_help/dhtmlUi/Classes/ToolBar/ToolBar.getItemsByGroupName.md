# ToolBar.getItemsByGroupName


## Синтаксис


getItemsByGroupName (groupName);


## Параметры


groupName. Имя группы.


## Описание


Метод getItemsByGroupName возвращает список элементов управления панели по указанному имени группы.


## Комментарии


Имя группы указывается в свойстве [ToolBarButton.GroupName](../ToolBarButton/ToolBarButton.GroupName.htm).
