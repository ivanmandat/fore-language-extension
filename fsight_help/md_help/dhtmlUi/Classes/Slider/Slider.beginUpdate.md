# Slider.beginUpdate


## Синтаксис


beginUpdate();


## Описание


Метод beginUpdate** блокирует возможность обновления компонента [Slider](../../Components/Slider/Slider.htm).


## Комментарии


Метод для оптимизации обновления [компонента](../../Components/Slider/Slider.htm), который следует вызывать перед непрерывной серией его изменений. После завершения серии изменений [компонента](../../Components/Slider/Slider.htm) необходимо вызвать метод [endUpdate](Slider.endUpdate.htm).
