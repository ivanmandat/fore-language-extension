# Slider.MaxItemsCount


## Синтаксис


MaxItemsCount: Number


## Описание


Свойство MaxItemsCount** определяет максимальное количество [бегунков](../SliderItem/SliderItem.htm) у компонента [Slider](../../Components/Slider/Slider.htm).


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMaxItemsCount, а возвращается с помощью метода getMaxItemsCount.
