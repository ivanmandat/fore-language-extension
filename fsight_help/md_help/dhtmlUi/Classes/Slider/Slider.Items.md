# Slider.Items


## Синтаксис


Items: Array


## Описание


Свойство Items устанавливает массив [бегунков](../SliderItem/SliderItem.htm) компонента [Slider](../../Components/Slider/Slider.htm).


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setItems, а возвращается с помощью метода getItems.


Массив бегунков - массив JSON-объектов типа [PP.Ui.SliderItem](../SliderItem/SliderItem.htm).
