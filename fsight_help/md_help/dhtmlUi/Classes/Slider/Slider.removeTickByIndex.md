# Slider.removeTickByIndex


## Синтаксис


removeTickByIndex(index: Number);


## Параметры


*index. Индекс удаляемой метки.*


## Описание


Метод removeTickByIndex** удаляет метку компонента [Slider](../../Components/Slider/Slider.htm) по указанному индексу.
