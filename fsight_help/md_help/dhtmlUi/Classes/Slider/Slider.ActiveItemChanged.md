# Slider.ActiveItemChanged


## Синтаксис


ActiveItemChanged: function(sender, args, timeout);


## Параметры


sender. Источник события;


args. Информация о событии;


timeout. Промежуток времени в миллисекундах, через который вызовется событие.


## Описание


Событие ActiveItemChanged** наступает при изменении активного [бегунка](../SliderItem/SliderItem.htm) компонента [Slider](../../Components/Slider/Slider.htm).
