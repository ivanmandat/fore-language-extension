# Slider.removeItems


## Синтаксис


removeItems();


## Описание


Метод removeItems удаляет все [бегунки](../SliderItem/SliderItem.htm)** компонента [Slider](../../Components/Slider/Slider.htm).


## Пример


Для выполнения примера предполагается наличие на странице компонента [Slider](../../Components/Slider/Slider.htm) с наименованием «slider» (см. «[Пример создания компонента Slider](../../Components/Slider/Example_Slider.htm)» ). Удалим все **[бегунки](../SliderItem/SliderItem.htm)** и метки:


// Удалим все бегунки
slider.removeItems();
// Удалим все метки
slider.removeTicks();

В результате выполнения примера были удалены все **[бегунки](../SliderItem/SliderItem.htm)** и метки компонента [Slider](../../Components/Slider/Slider.htm):
