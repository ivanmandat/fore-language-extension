# Slider.removeItemById


## Синтаксис


removeItemById(id: String);


## Параметры


*id. Идентификатор удаляемого [бегунка](../SliderItem/SliderItem.htm).*


## Описание


Метод removeItemById** удаляет *[бегунок](../SliderItem/SliderItem.htm) компонента [Slider](../../Components/Slider/Slider.htm) с указанным идентификатором*.
