# Slider.MaxValue


## Синтаксис


MaxValue: Number


## Описание


Свойство MaxValue** определяет максимальное значение [бегунков](../SliderItem/SliderItem.htm) у компонента [Slider](../../Components/Slider/Slider.htm).


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**MaxValue**, а возвращается с помощью метода get**MaxValue**.
