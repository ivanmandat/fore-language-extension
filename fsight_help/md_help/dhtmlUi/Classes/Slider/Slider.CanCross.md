# Slider.CanCross


## Синтаксис


CanCross: Boolean


## Описание


Свойство CanCross определяет, могут ли [бегунки](../SliderItem/SliderItem.htm) компонента [Slider](../../Components/Slider/Slider.htm) пересекаться друг с другом.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setCanCross, а возвращается с помощью метода getCanCross.


Если свойство имеет значение true, [бегунки](../SliderItem/SliderItem.htm) могут пересекаться друг с другом, иначе - не могут.


По умолчанию свойство имеет значение true.
