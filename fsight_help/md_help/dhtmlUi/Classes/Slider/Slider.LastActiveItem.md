# Slider.LastActiveItem


## Синтаксис


LastActiveItem: [PP.Ui.SliderItem](../SliderItem/SliderItem.htm)


## Описание


Свойство LastActiveItem** определяет последний использованный [бегунок](../SliderItem/SliderItem.htm) у компонента Slider.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**LastActiveItem**, а возвращается с помощью метода get**LastActiveItem**.
