# Slider.endUpdate


## Синтаксис


endUpdate();


## Описание


Метод endUpdate** разблокировывает возможность обновления компонента [Slider](../../Components/Slider/Slider.htm) и обновляет его.


## Комментарии


Метод для оптимизации обновления [компонента](../../Components/Slider/Slider.htm). Вызывается после метода [beginUpdate](Slider.beginUpdate.htm), когда завершается серия изменений [компонента](../../Components/Slider/Slider.htm).
