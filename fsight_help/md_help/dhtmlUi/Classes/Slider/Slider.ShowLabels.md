# Slider.ShowLabels


## Синтаксис


ShowLabels: Boolean;


## Описание


Свойство ShowLabels определяет
 признак отображения меток в компоненте.


## Комментарии


Значение свойства можно установить только из JSON.


Свойство содержит значение true
 (по умолчанию), если в компоненте отображаются метки, и значение false, если они не отображаются.


## Пример


Для выполнения примера необходимо создать html-страницу и выполнить
 следующие действия:


1. Добавить ссылки на файлы PP.css и PP.js.


2. Внутри тега <body> разместить блок с идентификатором «slider»:


<div id="slider"></div>


3. Далее на страницу внутри тега <body> необходимо добавить сценарий,
 который создаёт элемент управления для выбора значения с помощью перемещения
 бегунков без отображения меток:


<script type="text/javascript">
var slider = new PP.Ui.Slider({
    ParentNode: document.getElementById("slider"), // Родительский контейнер
    // Бегунки
    Items: [{
        Value: 0.25,
        PPType: "SliderItem",
        Id: "SliderItem1"
    }, {
        Value: 0.75,
        PPType: "SliderItem",
        Id: "SliderItem2"
    }],
    Orientation: PP.Orientation.Horizontal, // Ориентация компонента
    Ticks: ["2001", "2002", "2003"], // Метки
    ShowLabels: false, // Метки отображать не будем
    Width: 500 // Ширина компонента
});
</script>
В результате выполнения примера на html-странице будет размещен элемент
 управления для выбора значения с помощью перемещения бегунков без отображения
 меток:
