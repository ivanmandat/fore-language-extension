# Slider.getTickByIndex


## Синтаксис


getTickByIndex(index: Number);


## Параметры


*index. Индекс возвращаемой метки.*


## Описание


Метод getTickByIndex** возвращает метку компонента [Slider](../../Components/Slider/Slider.htm) по указанному индексу.
