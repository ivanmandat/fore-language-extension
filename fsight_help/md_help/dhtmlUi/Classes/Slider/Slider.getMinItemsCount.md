# Slider.getMinItemsCount


## Синтаксис


getMinItemsCount();


## Описание


Метод getMinItemsCount** возвращает минимально возможное количество [бегунков](../SliderItem/SliderItem.htm) на компоненте [Slider](../../Components/Slider/Slider.htm).


## Комментарии


Метод возвращает значение типа Number.
