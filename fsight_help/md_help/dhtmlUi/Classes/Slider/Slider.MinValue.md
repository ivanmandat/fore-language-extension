# Slider.MinValue


## Синтаксис


MinValue: Number


## Описание


Свойство MinValue** определяет минимальное значение [бегунков](../SliderItem/SliderItem.htm) у компонента [Slider](../../Components/Slider/Slider.htm).


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**MinValue**, а возвращается с помощью метода get**MinValue**.
