# Slider.Ticks


## Синтаксис


Ticks: Array


## Описание


Свойство Ticks** устанавливает массив меток компонента [Slider](../../Components/Slider/Slider.htm).


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**Ticks**, а возвращается с помощью метода get**Ticks**.


Массив бегунков - массив объектов типа String.
