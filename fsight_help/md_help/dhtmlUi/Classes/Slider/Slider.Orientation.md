# Slider.Orientation


## Синтаксис


Orientation: [PP.Orientation](dhtmlCommon.chm::/Enums/PP.Orientation.htm);


## Описание


Свойство Orientation**
 определяет ориентацию компонента [Slider](../../Components/Slider/Slider.htm).


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**Orientation**,
 а возвращается с помощью метода get**Orientation**.
