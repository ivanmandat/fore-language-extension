# InfoWindow.removePanel


## Синтаксис


removePanel(panel, isDispose);


## Параметры


panel. Панель, экземпляр класса PP.Ui.[InfoWindowPanel](../InfoWindowPanel/InfoWindowPanel.htm);


isDispose. Определяет, будет ли панель уничтожена (true) или удалена из списка панелей (false).


## Описание


Метод removePanel удаляет панель информационного окна.
