# InfoWindow.addToButtonContent


## Синтаксис


addToButtonContent(control);


## Параметры


control. Элемент управления, наследник [PP.Ui.Control](../Control/Control.htm), который будет добавлен в информационное окно на месте кнопки


## Описание


Метод addToButtonContent добавляет элемент управления в информационное окно на место, отведенное для кнопок.


## Комментарии


Место, отведенное для кнопок, -левая нижняя часть окна.
