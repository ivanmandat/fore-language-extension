# InfoWindow.hidePanel


## Синтаксис


hidePanel (panel);


## Параметры


panel. Панель, экземпляр класса PP.Ui.[InfoWindowPanel](../InfoWindowPanel/InfoWindowPanel.htm), которую необходимо скрыть.


## Описание


Метод hidePanel скрывает панель в информационном окне.


## Комментарии


Отобразить панель можно при помощи метода [InfoWindow.showPanel](InfoWindow.showPanel.htm).
