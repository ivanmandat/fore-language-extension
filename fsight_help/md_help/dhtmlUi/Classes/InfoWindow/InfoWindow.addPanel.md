# InfoWindow.addPanel


## Синтаксис


addPanel(panel);


## Параметры


panel. Панель, экземпляр класса PP.Ui.[InfoWindowPanel](../InfoWindowPanel/InfoWindowPanel.htm), которую-нужно добавить в информационное окно.


## Описание


Метод addPanel добавляет панель в информационное окно.
