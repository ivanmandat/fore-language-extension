# InfoWindow.showPanel


## Синтаксис


showPanel (panel);


## Параметры


panel. Панель, экземпляр класса PP.Ui.[InfoWindowPanel](../InfoWindowPanel/InfoWindowPanel.htm), которую необходимо скрыть.


## Описание


Метод showPanel отображает панель в информационном окне.


## Комментарии


Панель скрывается при помощи метода [InfoWindow.hidePanel](InfoWindow.hidePanel.htm).
