# InfoWindow.showCenter


## Синтаксис


showCenter (infoWindow);


## Параметры


infoWindow. Информационное окно, которое необходимо открыть. Необязательный параметр.


## Описание


Метод showCenter открывает информационное окно в центре экрана.
