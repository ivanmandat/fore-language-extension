# MetabaseListItem.HasPrv


## Синтаксис


HasPrv: Boolean


## Описание


Свойство HasPrv определяет,
 включен ли предварительный просмотр для элемента.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHasPrv,
 а возвращается с помощью метода getHasPrv.


Для определения изображения предварительного просмотра используйте свойство
 [ListItem.ImageURL](dhtmlList.chm::/Classes/ListItem/ListItem.ImageURL.htm).
