# DateTimeEdit.TextAfterYear


## Синтаксис


TextAfterYear: String


## Описание


Свойство TextAfterYear устанавливает текст, который будет отображён в поле ввода после значения года.


## Комментарии


Свойство актуально, если для свойства [DateTimeEdit.Mode](DateTimeEdit.Mode.htm) установлено значение 5.

Пример использования свойства приведен в описании свойства [DateTimeEdit.Mode](DateTimeEdit.Mode.htm).
