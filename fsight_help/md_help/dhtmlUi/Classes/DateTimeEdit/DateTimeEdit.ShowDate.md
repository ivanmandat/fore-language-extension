# DateTimeEdit.ShowDate


## Синтаксис


ShowDate: Boolean


## Описание


Свойство ShowDate определяет, отображается ли дата в компоненте.


## Комментарии


По умолчанию установлено значение true, то есть в поле ввода отображается значение даты.


Свойство актуально, если для свойства [DateTimeEdit.Mode](DateTimeEdit.Mode.htm) установлено значение по умолчанию (0).
