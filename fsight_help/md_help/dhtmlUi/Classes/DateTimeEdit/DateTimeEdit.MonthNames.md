# DateTimeEdit.MonthNames


## Синтаксис


MonthNames: Array


## Описание


Свойство MonthNames задаёт массив с наименованиями месяцев.


## Комментарии


Свойство актуально, если для свойства [DateTimeEdit.Mode](DateTimeEdit.Mode.htm) установлено значение 1.

Пример использования свойства приведен в описании свойства [DateTimeEdit.Mode](DateTimeEdit.Mode.htm).
