# DateTimeEdit.TextAfterQuarter


## Синтаксис


TextAfterQuarter: String


## Описание


Свойство TextAfterQuarter устанавливает текст, который будет отображён в поле ввода после значения квартала.


## Комментарии


Свойство актуально, если для свойства [DateTimeEdit.Mode](DateTimeEdit.Mode.htm) установлено значение 3.
