# DateTimeEdit.TextAfterMonth


## Синтаксис


TextAfterMonth: String


## Описание


Свойство TextAfterMonth устанавливает текст, который будет отображён в поле ввода после значения месяца.


## Комментарии


Свойство актуально, если для свойства [DateTimeEdit.Mode](DateTimeEdit.Mode.htm) установлено значение 2.
