# DateTimeEdit.TextAfterWeek


## Синтаксис


TextAfterWeek: String


## Описание


Свойство TextAfterWeek устанавливает текст, который будет отображён в поле ввода после значения недели.


## Комментарии


Свойство актуально, если для свойства [DateTimeEdit.Mode](DateTimeEdit.Mode.htm) установлено значение 1.
