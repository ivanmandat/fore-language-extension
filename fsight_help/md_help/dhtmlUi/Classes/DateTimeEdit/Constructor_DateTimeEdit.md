# Конструктор DateTimeEdit


## Синтаксис


PP.DateTimeEdit (settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор DateTimeEdit создает экземпляр компонента [DateTimeEdit](../../Components/DateTimeEdit/DateTimeEdit.htm).
