# DateTimeEdit.getCurrentHalfYear


## Синтаксис


getCurrentHalfYear();


## Описание


Метод getCurrentHalfYear возвращает значение полугодия.


## Комментарии


Метод возвращает 0, если текущее полугодие -первое, и 1, если второе.
