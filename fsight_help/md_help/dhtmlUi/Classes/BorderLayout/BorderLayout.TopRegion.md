# BorderLayout.TopRegion


## Синтаксис


TopRegion: PP.Ui.BorderLayoutItem


## Описание


Свойство TopRegion содержит
 компонент верхней области макета.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTopRegion,
 а возвращается с помощью метода getTopRegion.
