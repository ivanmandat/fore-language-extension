# BorderLayout.BottomRegion


## Синтаксис


BottomRegion: PP.Ui.BorderLayoutItem


## Описание


Свойство BottomRegion содержит
 компонент нижней области макета.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setBottomRegion,
 а возвращается с помощью метода getBottomRegion.
