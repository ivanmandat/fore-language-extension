# BorderLayout.Region


## Синтаксис


Region: PP.Ui.BorderLayoutItem


## Описание


Свойство Region устанавливает
 заданную область макета.


## Комментарии


Значение свойства устанавливается с помощью метода setRegion(regionName,
 region), а возвращается с помощью метода getRegion(regionName).
 Из JSON значение свойства задать нельзя.


Параметры метода setRegion:


- regionName: PP.Ui.REGION.
 Имя области макета, куда устанавливается компонент;


- region: PP.Ui.BorderLayoutItem||JSON.
 Компонент, который устанавливается в область.


Параметры метода getRegion:


- regionName: PP.Ui.REGION.
 Имя области, из которой извлекается компонент.
