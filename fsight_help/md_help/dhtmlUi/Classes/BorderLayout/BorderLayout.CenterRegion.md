# BorderLayout.CenterRegion


## Синтаксис


CenterRegion: PP.Ui.BorderLayoutItem


## Описание


Свойство CenterRegion содержит
 компонент центральной области макета.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setCenterRegion,
 а возвращается с помощью метода getCenterRegion.
