# BorderLayout.LeftRegion


## Синтаксис


LeftRegion: PP.Ui.BorderLayoutItem


## Описание


Свойство LeftRegion содержит
 компонент левой области макета.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLeftRegion,
 а возвращается с помощью метода getLeftRegion.
