# Конструктор BorderLayout


## Синтаксис


PP.Ui.BorderLayout(settings);


## Параметры


settings. JSON-объект со значениями
 свойств компонента.


## Описание


Конструктор BorderLayout создает
 экземпляр компонента [BorderLayout](../../Components/BorderLayout/BorderLayout.htm).
