# BorderLayout.RightRegion


## Синтаксис


RightRegion: PP.Ui.BorderLayoutItem


## Описание


Свойство RightRegion содержит
 компонент правой области макета.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setRightRegion,
 а возвращается с помощью метода getRightRegion.
