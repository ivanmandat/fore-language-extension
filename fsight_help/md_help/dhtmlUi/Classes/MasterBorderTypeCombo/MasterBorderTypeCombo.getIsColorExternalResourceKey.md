# MasterBorderTypeCombo.IsColorExternalResourceKey


## Синтаксис


IsColorExternalResourceKey: String;


## Описание


Свойство IsColorExternalResourceKey
 определяет ресурс, используемый мастером редактора границ.


## Комментарии


Свойство возвращается при помощи метода getIsColorExternalResourceKey.


По умолчанию содержит «externalColor».
