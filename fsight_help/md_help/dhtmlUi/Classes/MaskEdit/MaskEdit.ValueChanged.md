# MaskEdit.ValueChanged


## Синтаксис


ValueChanged: function(sender, args)


## Параметры


sender. Источник события;


args. Информация
 о событии. Аргументы: Event
 - произошедшее событие, OldValue - старое значение редактора ввода, NewValue - новое значение редактора ввода.


## Описание


Событие ValueChanged наступает
 после изменения значения редактора ввода.
