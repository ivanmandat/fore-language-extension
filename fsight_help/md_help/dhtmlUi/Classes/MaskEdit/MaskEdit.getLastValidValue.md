# MaskEdit.getLastValidValue


## Синтаксис


getLastValidValue ();


## Описание


Метод getLastValidValue возвращает последнее допустимое значение редактора маски.


## Комментарии


Допустимое значение - значение, которое соответствует [регулярному выражению](MaskEdit.RegExpContent.htm).
