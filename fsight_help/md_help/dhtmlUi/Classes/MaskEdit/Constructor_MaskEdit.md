# Конструктор MaskEdit


## Синтаксис


PP.Ui.MaskEdit(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор MaskEdit создает экземпляр компонента [MaskEdit](../../Components/MaskEdit/MaskEdit.htm).
