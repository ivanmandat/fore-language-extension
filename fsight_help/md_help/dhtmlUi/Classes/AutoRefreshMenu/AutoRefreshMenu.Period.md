# AutoRefreshMenu.Period


## Синтаксис


Period: object;


## Описание


Свойство Period определяет параметры
 периодического обновления.


## Комментарии


Значение свойства устанавливается с помощью метода setPeriod,
 а возвращается с помощью метода getPeriod.
 Из JSON значение свойства установить нельзя.


Метод setPeriod принимает следующие
 параметры: value - интервал времени,
 через который выполняется автообновление, unit
 - единица измерения, элемент перечисления PP.Mb.TimeUnits.
