# AutoRefreshMenu.setMarkChangedRefresh


## Синтаксис


setMarkChangedRefresh(value: Boolean);


## Параметры


value. Параметр определяет,
 включить (true) или отключить
 (false) автообновление при смене
 отметки.


## Описание


Метод setMarkChangedRefresh
 включает/отключает автообновление при смене отметки.
