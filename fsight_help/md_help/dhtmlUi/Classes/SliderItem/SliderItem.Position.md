# SliderItem.Position


## Синтаксис


Position: Object


## Описание


Свойство Position определяет координаты [бегунка](SliderItem.htm).


## Комментарии


Значение свойства устанавливается с помощью метода setPosition, а возвращается с помощью метода getPosition. Из JSON свойство установить нельзя.


JSON-объект в значении свойства содержит следующие поля: X - значение координаты X [бегунка](SliderItem.htm), Y -значение координаты Y [бегунка](SliderItem.htm).


При вызове метода setPosition указываются следующие параметры: position - новое значение положения [бегунка](SliderItem.htm), fireEvent - признак генерации события.
