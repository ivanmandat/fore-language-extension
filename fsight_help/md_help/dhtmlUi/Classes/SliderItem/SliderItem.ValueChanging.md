# SliderItem.ValueChanging


## Синтаксис


ValueChanging: function(sender, args, timeout);


## Параметры


sender. Источник события;


args. Информация о событии;


timeout. Промежуток времени в миллисекундах, через который вызовется событие.


## Описание


Событие ValueChanging** наступает при изменении значения [бегунка](SliderItem.htm).
