# SliderItem.MinValue


## Синтаксис


MinValue: Number


## Описание


Свойство MinValue** определяет минимальное значение [бегунка](SliderItem.htm).


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**MinValue**, а возвращается с помощью метода get**MinValue**.
