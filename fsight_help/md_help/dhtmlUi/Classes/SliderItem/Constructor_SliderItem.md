# Конструктор SliderItem


## Синтаксис


PP.Ui.SliderItem(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор SliderItem создает экземпляр класса [SliderItem](SliderItem.htm).
