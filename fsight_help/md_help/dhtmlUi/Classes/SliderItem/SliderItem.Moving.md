# SliderItem.Moving


## Синтаксис


Moving: function(sender, args, timeout);


## Параметры


sender. Источник события;


args. Информация о событии;


timeout. Промежуток времени в миллисекундах, через который вызовется событие.


## Описание


Событие Moving** наступает при изменении положения [бегунка](SliderItem.htm).
