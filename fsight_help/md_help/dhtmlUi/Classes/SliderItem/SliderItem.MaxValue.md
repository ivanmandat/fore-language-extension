# SliderItem.MaxValue


## Синтаксис


MaxValue: Number


## Описание


Свойство MaxValue** определяет максимальное значение [бегунка](SliderItem.htm).


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**MaxValue**, а возвращается с помощью метода get**MaxValue**.
