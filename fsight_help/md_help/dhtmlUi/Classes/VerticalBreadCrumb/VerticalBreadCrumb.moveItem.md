# VerticalBreadCrumb.moveItem


## Синтаксис


moveItem(sender, direction);


## Параметр


sender. Элемент меняющий позицию.


direction. Направление. Может
 принимать значения 1 или -1.


## Описание


Метод moveItem меняет позицию
 элемента.
