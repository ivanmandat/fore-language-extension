# VerticalBreadCrumb.replaceItem


## Синтаксис


replaceItem(item, newitem);


## Параметр


item. Элемент который необходимо
 заменить.


newitem. Новый элемент.


## Описание


Метод replaceItem заменяет элемент
 на новый.

 для внутреннего использования классом PP.Exp.Ui.HiePanel.
