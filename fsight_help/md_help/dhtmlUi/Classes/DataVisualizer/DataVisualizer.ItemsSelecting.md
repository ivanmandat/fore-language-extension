# DataVisualizer.ItemsSelecting


## Синтаксис


ItemsSelecting: function (sender, args: Object);


## Параметры


sender. Источник события;


args. Информация о событии.
 Информация о выделенных элементах содержится в свойстве Items.


## Описание


Событие ItemsSelecting генерируется
 перед выделением элементов визуализатора.


## Пример


Пример использования события аналогичен событию [DataVisualizer.ItemsSelected](DataVisualizer.ItemsSelected.htm).
