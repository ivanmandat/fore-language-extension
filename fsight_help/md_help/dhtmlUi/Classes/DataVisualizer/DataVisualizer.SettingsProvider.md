# DataVisualizer.SettingsProvider


## Синтаксис


SettingsProvider: [PP.Ui.DefaultSettingsProvider](../DefaultSettingsProvider/DefaultSettingsProvider.htm);


## Описание


Свойство SettingsProvider определяет
 базовый класс для предоставления настроек.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSettingsProvider,
 а возвращается с помощью метода getSettingsProvider.


## Пример


Пример использования свойства приведен в описании метода [DefaultSettingsProvider.getDefaultLabelsMaskText](../DefaultSettingsProvider/DefaultSettingsProvider.getDefaultLabelsMaskText.htm).
