# DataVisualizer.AreaSelecting


## Синтаксис


AreaSelecting: function (sender, args: Object);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: AreaSelection - выделение области (значение является объектом типа PP.Ui.AreaSelection), Cancel - признак отмены нажатия на заголовок легенды диаграммы ( по умолчанию равен значению false).


## Описание


Событие AreaSelecting** генерируется в процессе выделения области диаграммы.
