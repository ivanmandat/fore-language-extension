# DataVisualizer.SelectActiveEffects


## Синтаксис


SelectActiveEffects: Array


## Описание


Свойство SelectActiveEffects
 определяет массив эффектов, которые применяются к выделенным элементам пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSelectActiveEffects,
 а возвращается с помощью метода getSelectActiveEffects.
