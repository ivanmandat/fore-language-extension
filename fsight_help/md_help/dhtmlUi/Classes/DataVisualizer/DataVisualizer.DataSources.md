# DataVisualizer.DataSources


## Синтаксис


DataSources: Object


## Описание


Свойство DataSources**
 устанавливает и возвращает объект с источниками данных для диаграмм.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDataSources,
 а возвращается с помощью метода getDataSources.


Свойство содержит JSON-объект, поля которого представляют собой объект
 типа PP.DataSource.
