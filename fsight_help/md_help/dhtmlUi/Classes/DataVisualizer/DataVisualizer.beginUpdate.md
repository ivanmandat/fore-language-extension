# DataVisualizer.beginUpdate


## Синтаксис


beginUpdate();


## Описание


Метод beginUpdate** включает режим обновления данных, запрещающий перерисовку компонента до выхода из этого режима.


## Комментарии


Выход из режима обновления данных осуществляется с помощью метода [endUpdate](DataVisualizer.endUpdate.htm).
