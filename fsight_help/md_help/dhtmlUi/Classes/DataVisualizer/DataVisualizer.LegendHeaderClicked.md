# DataVisualizer.LegendHeaderClicked


## Синтаксис


LegendHeaderClicked: function (sender, args: Object);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: Event - событие мыши, Break - признак отмены нажатия на заголовок легенды диаграммы (по умолчанию равен значению false), Type - тип легенды.


## Описание


Событие LegendHeaderClicked** генерируется при нажатии на заголовок легенды диаграммы.
