# DataVisualizer.LegendMapping


## Синтаксис


LegendMapping: Object


## Описание


Свойство LegendMapping определяет
 словарь связи легенд с привязками данных.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLegendMapping,
 а возвращается с помощью метода getLegendMapping.


Значение свойства представляет собой JSON-объект, элементы которого
 представлены в следующем формате: <идентификатор легенды: идентификатор
 привязки данных>.


Словарь предназначен для сопоставления данных визуализаторов экспресс-отчета
 с легендами.
