# DataVisualizer.getVisualById


## Синтаксис


getVisualById(id: String);


## Параметры


id. Идентификатор сопоставления
 данных визуализатора.


## Описание


Метод getVisualById возвращает
 сопоставление данных визуализатора по его идентификатору.


## Комментарии


Метод возвращает значение типа [PP.Visual](dhtmlGraphicBase.chm::/Classes/PP/Visual/Visual.htm).
