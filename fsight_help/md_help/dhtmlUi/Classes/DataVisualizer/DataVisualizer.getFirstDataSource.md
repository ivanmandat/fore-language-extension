# DataVisualizer.getFirstDataSource


## Синтаксис


getFirstDataSource();


## Описание


Метод getFirstDataSource** возвращает первый источник у визуализатора данных для диаграмм.


## Комментарии


Метод возвращает объект типа PP.DataSource.
