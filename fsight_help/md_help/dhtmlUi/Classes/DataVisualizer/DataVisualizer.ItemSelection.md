# DataVisualizer.ItemSelection


## Синтаксис


ItemSelection: Boolean;


## Описание


Свойство ItemSelection**
 определяет выделение элемента.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setItemSelection(elem_id,
 isActive), а возвращается с помощью метода getItemSelection(elem_id),
 где elem_id - идентификатор элемента
 или объект элемента, isActive
 - признак выделения.


Свойство содержит значение true
 - если элемент выделен, и false
 - если не выделен.
