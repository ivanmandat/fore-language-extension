# DataVisualizer.ActiveMappingItems


## Синтаксис


ActiveMappingItems: Object


## Описание


Свойство ActiveMappingItems
 определяет массив активных значений пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setActiveMappingItems,
 а возвращается с помощью метода getActiveMappingItems.**
