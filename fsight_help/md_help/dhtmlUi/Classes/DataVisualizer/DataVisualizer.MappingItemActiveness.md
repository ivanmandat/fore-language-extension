# DataVisualizer.MappingItemActiveness


## Синтаксис


MappingItemActiveness: Boolean;


## Описание


Свойство MappingItemActiveness
 определяет, активно ли сопоставление эффекта у элемента визуализатора.


## Комментарии


Значение свойства устанавливается с помощью метода setMappingItemActiveness(elem_id,
 mapping_id, isActive), а возвращается с помощью метода getMappingActiveEffects(mapping_id,
 elem_id), где elem_id
 - идентификатор или объект элемента визуализатора, mapping_id
 - идентификатор сопоставления эффекта, isActive
 - признак активности эффекта.
