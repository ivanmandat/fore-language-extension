# DataVisualizer.Scales


## Синтаксис


Scales: Object


## Описание


Свойство Scales** устанавливает
 и возвращает шкалы диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setScales,
 а возвращается с помощью метода getScales.


Свойство содержит JSON-объект с полями, являющимися объектами типа PP.ScaleBase.
