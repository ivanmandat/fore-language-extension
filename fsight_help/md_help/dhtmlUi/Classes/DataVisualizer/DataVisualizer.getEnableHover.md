# DataVisualizer.getEnableHover


## Синтаксис


getEnableHover();


## Описание


Метод getEnableHover возвращает
 признак использования эффектов при наведении курсора на пузырьки пузырьковой диаграммы.


## Комментарии


Метод возвращает значение типа Boolean.


Если эффекты наведения используются, то метод возвращает значение true, иначе - false.
