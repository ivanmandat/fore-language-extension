# DataVisualizer.ItemHoverness


## Синтаксис


ItemHoverness: Boolean


## Описание


Свойство ItemHoverness
 определяет, установлено ли наведение на элемент диаграммы.


## Комментарии


Значение свойства устанавливается с помощью метода setItemHoverness,
 а возвращается с помощью метода getItemHoverness.
 Из JSON значение свойства установить нельзя.


Метод getItemHoverness принимает
 параметр elem_id - идентификатор или элемент, признак наведения на который
 необходимо вернуть.


Метод setItemHoverness принимает
 следующие параметры: elem_id - идентификатор или элемент, на который устанавливается
 наведение, isActive - признак наведения на элемент.


Свойство содержит значение true,
 если для указанного элемента установлено наведение, и значение false,
 если наведение не установлено.
