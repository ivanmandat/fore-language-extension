# DataVisualizer.getLegendByDataMapping


## Синтаксис


getLegendByDataMapping(mapping_id: String);


## Параметры


mapping_id. Идентификатор настройки
 сопоставления данных визуализатора.


## Описание


Метод getLegendByDataMapping
 возвращает легенду визуализатора по идентификатору сопоставления данных.
