# DataVisualizer.getFirstLegend


## Синтаксис


getFirstLegend();


## Описание


Метод getFirstLegend** возвращает первую из всех легенд для диаграммы.


## Комментарии


Метод возвращает объект, базовым типом которого является PP.Ui.LegendBase.
