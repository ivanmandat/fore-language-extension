# DataVisualizer.getGlobalItemId


## Синтаксис


getGlobalItemId(item_id: Object | String);


## Параметры


item_id. Элемент или его внутренний
 идентификатор.


## Описание


Метод getGlobalItemId
 возвращает глобальный идентификатор элемента.
