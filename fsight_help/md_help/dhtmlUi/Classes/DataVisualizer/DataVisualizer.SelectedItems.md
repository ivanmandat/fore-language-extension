# DataVisualizer.SelectedItems


## Синтаксис


SelectedItems: Object


## Описание


Свойство SelectedItems
 определяет массив выделенных элементов пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSelectedItems,
 а возвращается с помощью метода getSelectedItems.


Метод getSelectedItems принимает
 параметр useGlobalIds - признак
 использования глобальных идентификаторов, вместо локальных.
