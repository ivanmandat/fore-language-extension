# DataVisualizer.endUpdate


## Синтаксис


endUpdate();


## Описание


Метод endUpdate** выключает режим обновления данных, запрещающий перерисовку компонента до выхода из этого режима.


## Комментарии


Режим обновления данных включается с помощью метода [beginUpdate](DataVisualizer.beginUpdate.htm).
