# DataVisualizer.addVisual


## Синтаксис


addVisual(visual: [PP.Visual](dhtmlGraphicBase.chm::/Classes/PP/Visual/Visual.htm),
 id: String);


## Параметры


visual. Сопоставление данных
 визуализатора, которое требуется установить;


id. Идентификатор сопоставления
 данных визуализатора.


## Описание


Метод addVisual устанавливает
 сопоставление данных визуализатора с указанным идентификатором.
