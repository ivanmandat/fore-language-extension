# DataVisualizer.getBackwardLegendMapping


## Синтаксис


getBackwardLegendMapping();


## Описание


Метод getBackwardLegendMapping
 возвращает обратный словарь связи легенд с привязками данных.


## Комментарии


Метод возвращает JSON-объект, элементы которого представлены в следующем
 формате: <идентификатор привязки данных: идентификатор легенды>.


Словарь с обычным порядком значений содержится в свойстве [DataVisualizer.LegendMapping](DataVisualizer.LegendMapping.htm).
