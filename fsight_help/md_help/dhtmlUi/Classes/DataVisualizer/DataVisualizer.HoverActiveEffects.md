# DataVisualizer.HoverActiveEffects


## Синтаксис


HoverActiveEffects: Array


## Описание


Свойство HoverActiveEffects
 определяет эффекты, применяющиеся к объектам при наведении на них курсора.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHoverActiveEffects,
 а возвращается с помощью метода getHoverActiveEffects.
