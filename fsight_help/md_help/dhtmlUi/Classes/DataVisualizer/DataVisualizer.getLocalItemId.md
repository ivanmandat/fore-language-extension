# DataVisualizer.getLocalItemId


## Синтаксис


getLocalItemId(global_id: Object | String);


## Параметры


global_id. Элемент или его
 глобальный идентификатор.


## Описание


Метод getLocalItemId возвращает
 внутренний идентификатор элемента.
