# DataVisualizer.updateData


## Синтаксис


updateData(index: Number);


## Параметр


index. Индекс для параметров сопоставления значений рядов данных осям диаграммы, цвету и размеру её элементов.


## Описание


Метод updateData** присваивает параметрам сопоставления значений рядов данных диаграммы новое значение индекса.


## Комментарии


Для диаграмм данный метод обновляет данные из источника.


Индекс, указываемый в качестве параметра, соответствует шагу временной шкалы диаграммы.
