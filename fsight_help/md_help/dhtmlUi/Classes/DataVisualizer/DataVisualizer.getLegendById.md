# DataVisualizer.getLegendById


## Синтаксис


getLegendById(legendId: String);


## Описание


Метод getLegendById возвращает
 легенду пузырьковой диаграммы по её идентификатору.


## Комментарии


Метод возвращает экземпляр класса [PP.Ui.LegendBase](dhtmlGraphicBase.chm::/Classes/LegendBase/LegendBase.htm).
