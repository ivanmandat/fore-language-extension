# DataVisualizer.SelectInactiveEffects


## Синтаксис


SelectInactiveEffects: Array


## Описание


Свойство SelectInactiveEffects
 определяет массив эффектов, которые применяются к невыделенным элементам
 пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSelectInactiveEffects,
 а возвращается с помощью метода getSelectInactiveEffects.
