# DataVisualizer.getBackwardGlobalItemsIds


## Синтаксис


getBackwardGlobalItemsIds();


## Описание


Метод getBackwardGlobalItemsIds
 возвращает обратный словарь глобальных идентификаторов элементов.


## Комментарии


Метод возвращает JSON-объект, элементы которого представлены в следующем
 формате: <глобальный идентификатор элемента: внутренний идентификатор
 элемента>.


Словарь с обычным порядком значений содержится в свойстве [DataVisualizer.GlobalItemsIds](DataVisualizer.GlobalItemsIds.htm).
