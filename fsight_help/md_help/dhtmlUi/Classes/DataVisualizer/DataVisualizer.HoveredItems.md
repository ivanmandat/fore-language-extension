# DataVisualizer.HoveredItems


## Синтаксис


HoveredItems: Array


## Описание


Свойство HoveredItems
 определяет массив элементов, для которых установлено наведение.


## Комментарии


Значение свойства устанавливается из JSON с помощью метода setHoveredItems,
 а возвращается с помощью метода getHoveredItems.


Метод getHoveredItems принимает
 параметр useGlobalIds - признак
 использования глобальных идентификаторов, вместо локальных.
