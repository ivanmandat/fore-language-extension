# DataVisualizer.MappingActiveEffects


## Синтаксис


MappingActiveEffects: Array [Object];


## Описание


Свойство MappingActiveEffects
 определяет сопоставление эффектов для активных элементов.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMappingActiveEffects,
 а возвращается с помощью метода getMappingActiveEffects.


## Пример


Использование свойства аналогично свойству [DataVisualizer.MappingInactiveEffects](DataVisualizer.MappingInactiveEffects.htm).
