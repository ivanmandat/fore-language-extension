# DataVisualizer.HoverInactiveEffects


## Синтаксис


HoverInactiveEffects: Array


## Описание


Свойство HoverInactiveEffects
 определяет эффекты, применяющиеся к объектам, на которые не наведен курсор.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHoverInactiveEffects,
 а возвращается с помощью метода getHoverInactiveEffects.
