# DataVisualizer.LegendsPackingMaxPart


## Синтаксис


LegendsPackingMaxPart: Number


## Описание


Свойство LegendsPackingMaxPart** устанавливает и возвращает максимальную часть легенд диаграммы из их компоновки.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLegendsPackingMaxPart, а возвращается с помощью метода getLegendsPackingMaxPart.


По умолчанию свойство содержит значение 0,5.
