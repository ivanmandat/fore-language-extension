# DataVisualizer.Legends


## Синтаксис


Legends: Array


## Описание


Свойство Legends** устанавливает
 и возвращает массив легенд диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLegends,
 а возвращается с помощью метода getLegends.


Свойство содержит массив объектов, базовым типом которых является PP.Ui.LegendBase.
