# Конструктор RadioButton


## Синтаксис


PP.RadioButton(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор RadioButton создает экземпляр компонента [RadioButton](../../Components/RadioButton/RadioButton.htm).
