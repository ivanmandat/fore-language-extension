# RadioButton.GroupName


## Синтаксис


GroupName: String


## Описание


Свойство GroupName устанавливает имя группы компонентов [RadioButton](../../Components/RadioButton/RadioButton.htm).


## Комментарии


Экземпляры компонента RadioButton обычно объединяются в группы при помощи свойства GroupName.
