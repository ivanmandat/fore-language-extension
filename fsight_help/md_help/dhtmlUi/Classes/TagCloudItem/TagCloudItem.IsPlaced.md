# TagCloudItem.IsPlaced


## Синтаксис


IsPlaced: Boolean


## Описание


Свойство IsPlaced определяет,
 отображен ли элемента облака меток.


## Комментарии


Допустимые значения:


	- true.
	 Элемент размещен;


	- false.
	 Элемент еще не размещен.


Значение свойства устанавливается из JSON и с помощью метода setIsPlaced,
 а возвращается с помощью метода getIsPlaced.
