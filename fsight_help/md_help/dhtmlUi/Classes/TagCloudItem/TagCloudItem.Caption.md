# TagCloudItem.Caption


## Синтаксис


Caption: String


## Описание


Свойство Caption определяет
 заголовок элемента облака меток.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setCaption,
 а возвращается с помощью метода getCaption.
