# TagCloudItem.Active


## Синтаксис


Active: Boolean


## Описание


Свойство Active определяет активность
 элемента облака меток.


## Комментарии


Допустимые значения:


	- true.
	 Элемент находится в активном состоянии;


	- false.
	 Элемент находится в неактивном состоянии.


Значение свойства устанавливается из JSON и с помощью метода setActive,
 а возвращается с помощью метода getActive.
