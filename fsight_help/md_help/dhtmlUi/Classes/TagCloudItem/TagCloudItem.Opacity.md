# TagCloudItem.Opacity


## Синтаксис


Opacity: Number


## Описание


Свойство Opacity определяет
 прозрачность элемента облака меток.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setOpacity,
 а возвращается с помощью метода getOpacity.
