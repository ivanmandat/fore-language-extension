# TagCloudItem.Size


## Синтаксис


Size: Nubmer


## Описание


Свойство Size определяет размер
 элемента облака меток.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSize,
 а возвращается с помощью метода getSize.
