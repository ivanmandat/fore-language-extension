# TagCloudItem.TargetColor


## Синтаксис


TargetColor: Object


## Описание


Свойство TargetColor определяет
 целевое значение цвета элемента облака меток.


## Комментарии


Целевое значение цвета - цвет метки на следующем шаге анимации.


Объект содержит следующие поля:


	- R.
	 Значение красной компоненты цвета;


	- G.
	 Значение зеленой компоненты цвета;


	- B.
	 Значение синей компоненты цвета.


Значение свойства устанавливается из JSON и с помощью метода setTargetColor,
 а возвращается с помощью метода getTargetColor.
