# ProxyControl.getControl


## Синтаксис


getControl();


## Описание


Метод getControl возвращает
 базовый элемент управления - наследник PP.Ui.[Control](../Control/Control.htm).


## Комментарии


С помощью метода getControl
 можно настроить свойства высокоуровневых компонентов, построенных на базовых.


## Пример


Пример использования метода приведен в описании свойства [ReportView.RibbonView](dhtmlReport.chm::/Classes/RegularReport/ReportView/ReportView.RibbonView.htm).
