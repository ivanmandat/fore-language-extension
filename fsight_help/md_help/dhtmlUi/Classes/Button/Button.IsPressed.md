# Button.IsPressed


## Синтаксис


IsPressed: Boolean


## Описание


Свойство IsPressed устанавливает признак того, что кнопка находится в нажатом состоянии.


## Комментарии


Если для свойства установлено значение true, то кнопка отображается в нажатом состоянии. По умолчанию установлено значение false.


Свойство актуально, если для свойства [Button.IsToggleButton](Button.IsToggleButton.htm) установлено значение true.


## Пример


Для выполнения примера создайте кнопку (см.«[Пример создания простой кнопки](../../Components/Button/Example_SimpleButton.htm)»). Для того чтобы при загрузке html-страницы кнопка находилась в нажатом состоянии, установите для свойства IsPressed значение true:


IsPressed: true
