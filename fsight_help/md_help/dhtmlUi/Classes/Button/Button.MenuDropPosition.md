# Button.MenuDropPosition


## Синтаксис


MenuDropPosition: [LTRB](dhtmlCommon.chm::/enums/ltrb.htm);


## Описание


Свойство MenuDropPosition устанавливает
 позицию открытия дочернего меню компонента.


## Комментарии


Значения свойства - элемент перечисления [LTRB](dhtmlCommon.chm::/enums/ltrb.htm).
 По умолчанию установлено значение Bottom.


Вместе с изменением позиции открытия дочернего компонента меняется и
 направление стрелки.


Если на html-странице нет места для разворачивания дочернего меню в
 направлении, указанном в свойстве MenuDropPosition,
 дочерний компонент будет развернут в противоположном направлении.
