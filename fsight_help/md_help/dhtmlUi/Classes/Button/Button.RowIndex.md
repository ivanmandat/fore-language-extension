# Button.RowIndex


## Синтаксис


RowIndex: Number;


## Описание


Свойство RowIndex устанавливает
 индекс строки с пиктограммой, выбранной из файла с картинками, реализованного
 классом [ImageList](dhtmlCommon.chm::/Classes/imagelist/imagelist.htm).


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setRowIndex.
 При вызове метода setRowIndex
 указываются два параметра: value - индекс строки с пиктограммой, release
 - признак того, что кнопка находится в ненажатом состоянии (необязательный
 параметр).
