# Button.IsRetinaAvailable


## Синтаксис


IsRetinaAvailable: Boolean


## Описание


Свойство IsRetinaAvailable определяет
 признак возможности работы на устройстве с дисплеем Retina.


## Комментарии


Допустимые значения:


	- true.
	 Возможна работа компонента на устройстве с дисплеем Retina;


	- false.
	 Работа компонента на устройстве с дисплеем Retina невозможна (по умолчанию).


Значение свойства устанавливается из JSON и с помощью метода setIsRetinaAvailable,
 а возвращается с помощью метода getIsRetinaAvailable.


## Пример


Для выполнения примера необходимо наличие компонента [Button](../../Components/Button/Button.htm)
 с наименованием «button» (см.«[Пример
 создания простой кнопки](../../Components/Button/Example_SimpleButton.htm)»). Проверим, имеет ли устройство дисплей Retina
 и возможна ли работа с данным устройством:


if (button.getIsRetinaAvailable()) {
    console.log("Дисплей является Retina и возможна работа с устройством с данными дисплеем");
} else {
    console.log("Дисплей не является Retina или невозможна работа с устройством с данным дисплеем");
}
В результате выполнения примера в консоли будет выведено сообщение о
 том, имеет ли устройство дисплей Retina и возможна ли работа с данным
 дисплеем.
