# Button.ContentChanged


## Синтаксис


ContentChanged: function (sender, args)


## Параметры


sender. Источник события;


args. Информация
 о событии.


## Описание


Событие ContentChanged наступает
 после редактирования содержимого кнопки при условии, если оно было изменено.


## Пример


Пример использования события приведен в описании свойства [Button.EnableEdit](Button.EnableEdit.htm).
