# Button.ColumnIndex


## Синтаксис


ColumnIndex: Number;


## Описание


Свойство ColumnIndex определяет
 индекс столбца с пиктограммой, выбранной из файла с картинками, реализованного
 классом [ImageList](dhtmlCommon.chm::/Classes/imagelist/imagelist.htm).


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setColumnIndex.
 При вызове метода setColumnIndex
 указываются два параметра: value - индекс столбца с пиктограммой, release
 - признак того, что кнопка находится в ненажатом состоянии (необязательный
 параметр).
