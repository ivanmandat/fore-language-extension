# Button.ImagePosition


## Синтаксис


ImagePosition: PP.[LTRB](dhtmlCommon.chm::/enums/ltrb.htm)


## Описание


Свойство ImagePosition определяет
 положение изображения кнопки.


## Комментарии


Значения свойства - элементы перечисления [LTRB](dhtmlCommon.chm::/enums/ltrb.htm).
 По умолчанию установлено значение Top.


Если создается кнопка с изображением и с раскрывающимся меню, значение
 свойства ImagePosition перекрывает
 значение свойства [.MenuArrowPosition](Button.MenuArrowPosition.htm).
