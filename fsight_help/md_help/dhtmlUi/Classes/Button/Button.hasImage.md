# Button.hasImage


## Синтаксис


hasImage ()


## Описание


Метод hasImage возвращает признак того, содержит ли кнопка изображение


## Комментарии


Если для кнопки установлено изображение, возвращается значение true, иначе - null.


Установить изображение для кнопки можно при помощи свойства [Button.ImageList](button.imagelist.htm) (выбрав нужную картинку из файла с несколькими картинками) или [Button.ImageUrl](Button.ImageUrl.htm) (указав адрес картинки).
