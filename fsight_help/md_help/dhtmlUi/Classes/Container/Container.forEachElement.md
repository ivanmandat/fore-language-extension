# Container.forEachElement


## Синтаксис


forEachElement(fn: Function);


## Параметры


fn. Функция, применяемая ко
 всем элементам контейнера.


## Описание


Метод forEachElement применяет
 функцию ко всем элементам контейнера.


## Комментарии


В качестве аргумента в функцию передается элемент контейнера.
