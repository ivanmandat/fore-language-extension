# Container.getItemById


## Синтаксис


getItemById (id)


## Параметры


id. Идентификатор компонента.


## Описание


Метод getItemById возвращает
 дочерний элемент контейнера по его идентификатору.


## Комментарии


Идентификатор задается в свойстве [Id](dhtmlCommon.chm::/Classes/object/object.id.htm).


## Пример


Метод используется аналогично методу [Menu.getMenuItemById](../Menu/Menu.getMenuItemById.htm).
