# Container.hideAt


## Синтаксис


hideAt (index);


## Параметры


index. Индекс элемента, который необходимо скрыть.


## Описание


Метод hideAt скрывает дочерний элемент контейнера по его индексу.


## Комментарии


Скрытые элементы отображаются при помощи метода [showAt](Container.showAt.htm).

Пример использования метода приведен в описании метода [showAt](Container.showAt.htm).
