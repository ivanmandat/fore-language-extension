# Container.clear


## Синтаксис


clear (notDispose);


## Параметры


notDispose. Полностью ли удалить
 элементы контейнера. По умолчанию установлено значение true,
 то есть элементы удаляются полностью.


## Описание


Метод clear удаляет компонент
 с html-страницы.


## Пример


Метод используется аналогично методу [List.clear](dhtmlList.chm::/Classes/list/list.clear.htm).
