# Container.remove


## Синтаксис


remove (ppControl, notDispose);


## Параметры


ppControl. Компонент, который необходимо удалить;


notDispose. Определяет, будет ли элемент удален полностью. Если установлено значение false, элемент будет удален полностью.


## Описание


Метод remove удаляет указанный дочерний элемент контейнера.
