# Control.TabIndex


## Синтаксис


TabIndex: Number;


## Описание


Свойство TabIndex определяет
 последовательность перехода элемента управления внутри контейнера.


## Комментарии


Индексом перехода по клавише TAB может служить любое целое число, величина
 которого больше или равна нулю. При этом меньшие числа располагаются ближе
 к началу последовательности табуляции.


Если для элемента управления не задано значение свойства TabIndex,
 то при нажатии клавиши TAB заданный элемент не выделяется.


## Пример


Для выполнения примера создайте [кнопку](../../Components/Button/Button.htm)
 (см.«[Пример
 создания простой кнопки](../../Components/Button/Example_SimpleButton.htm)»). В параметрах кнопки укажите значение для
 свойства TabIndex:


TabIndex: 1


После выполнения примера при нажатии клавиши TAB будет выделяться созданная
 кнопка. Данному элементу управления будет установлен индекс перехода по
 клавише TAB - 1.
