# Control.removeEventHandler


## Синтаксис


removeEventHandler(eventName: String, handler: PP.Delegate);


## Параметры


eventName. Имя события;


handler. Обработчик события.


## Описание


Метод removeEventHandler удаляет
 обработчик события DOM-вершины.
