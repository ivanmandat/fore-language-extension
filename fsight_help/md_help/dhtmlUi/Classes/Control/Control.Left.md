# Control.Left


## Синтаксис


Left: Integer;


## Описание


Свойство Left определяет отступ
 слева при размещении компонента внутри [GridPanel](../../Components/GridPanel/GridPanel.htm).


## Комментарии


При размещении компонента внутри [GridPanel](../../Components/GridPanel/GridPanel.htm)
 для свойства Left устанавливается
 индекс столбца, в котором будет размещён компонент.
