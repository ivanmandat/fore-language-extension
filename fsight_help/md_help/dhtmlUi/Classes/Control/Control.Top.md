# Control.Top


## Синтаксис


Top: Integer;


## Описание


Свойство Top определяет отступ
 сверху при размещении компонента внутри [GridPanel](../../Components/GridPanel/GridPanel.htm).


## Комментарии


При размещении компонента внутри [GridPanel](../../Components/GridPanel/GridPanel.htm)
 для свойства Top устанавливается
 индекс строки, в которой будет размещён компонент.
