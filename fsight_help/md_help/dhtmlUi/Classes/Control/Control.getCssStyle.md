# Control.getCssStyle


## Синтаксис


getCssStyle(node, styleName: String, toNumber: Boolean,
 noCache);


## Параметры


node.
 DOM-вершина, для которой нужно получить
 значение стиля;


styleName.
 Имя CSS-свойства;


toNumber.
 Признак попытки преобразования стиля в числовой формат;


noCache.
 Признак кеширования результатов получения стилей для каждой DOM-вершины.


## Описание


Метод getCssStyle возвращает
 стили для указанной вершины.


## Комментарии


Допустимые значения параметра toNumber:


	- true. Будет произведена
	 попытка преобразования стиля в числовой формат. Если преобразовать
	 не удалось, то метод вернёт значение 0;


	- false. Попытка преобразования
	 стиля в числовой формат производиться не будет.


Допустимые значения параметра noCache:


	- true. Результаты получения
	 стилей для каждой DOM-вершины будут кешироваться;


	- false. Результаты получения
	 стилей для каждой DOM-вершины кешироваться не будут.
