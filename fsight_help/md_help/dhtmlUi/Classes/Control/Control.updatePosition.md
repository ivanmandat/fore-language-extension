# Control.updatePosition


## Синтаксис


updatePosition(parent);


## Параметры


parent. Родительская DOM-вершина
 компонента.


## Описание


Метод updatePosition корректирует
 размер и положение при абсолютном позиционировании на основе текущих параметров.
