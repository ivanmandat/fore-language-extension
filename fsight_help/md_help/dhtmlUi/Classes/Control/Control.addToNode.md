# Control.addToNode


## Синтаксис


addToNode(node);


## Параметры


node. DOM-вершина, в которую
 будет добавлен компонент.


## Описание


Метод addToNode добавляет компонент
 в указанную вершину.


## Комментарии


Метод используется, если не задано значение свойства ParentNode.
