# Control.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет
 признак доступности компонента для использования.


## Комментарии


Допустимые значения:


	- true. По умолчанию.
	 Компонент доступен для использования;


	- false. Компонент недоступен
	 для использования.


## Пример


Для выполнения примера создайте кнопку (см.«[Пример
 создания простой кнопки](../../Components/Button/Example_SimpleButton.htm)»). Для того чтобы кнопка стала недоступной
 для пользователя в параметрах кнопки установите значение false
 для свойства Enabled:


Enabled: false


После выполнения примера созданная кнопка станет недоступной для пользователя.
