# Control.IsVisible


## Синтаксис


IsVisible: Boolean;


## Описание


Свойство IsVisible определяет
 признак отображения компонента.


## Комментарии


Допустимые значения:


	- true. Компонент отображается;


	- false. Компонент скрыт.
