# Control.Right


## Синтаксис


Right: Integer;


## Описание


Свойство Right определяет отступ
 справа при размещении компонента внутри LayoutPanel.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setRight,
 а возвращается с помощью метода getRight.


Свойство находится в разработке.
