# Control.ParentNode


## Синтаксис


ParentNode: String;


## Описание


Свойство ParentNode определяет
 родительскую DOM-вершину.


## Комментарии


Значение свойства возвращается при помощи метода getParentNode,
 а устанавливается из JSON.
