# Control.getIsBinded


## Синтаксис


getIsBinded();


## Описание


Метод getIsBinded возвращает
 признак подписи элемента на события DOM-вершины.


## Комментарии


Метод возвращает true, если
 элемент подписан на события DOM-вершины, иначе false.
