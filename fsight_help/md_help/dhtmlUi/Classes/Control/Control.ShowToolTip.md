# Control.ShowToolTip


## Синтаксис


ShowToolTip: Boolean;


## Описание


Свойство ShowToolTip определяет
 признак возможности отображения [всплывающей
 подсказки](Control.ToolTip.htm) компонента.


## Комментарии


Допустимые значения:


	- true. Возможность отображения
	 всплывающей подсказки компонента включена;


	- false. Возможность отображения
	 всплывающей подсказки компонента отключена.


Текст всплывающей подсказки устанавливается при помощи свойства [Control.ToolTip](Control.ToolTip.htm).
