# Control.Bottom


## Синтаксис


Bottom: Integer;


## Описание


Свойство Bottom определяет отступ
 снизу при размещении компонента внутри LayoutPanel.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setBottom,
 а возвращается с помощью метода getBottom.


Свойство находится в разработке.
