# Control.ToolTip


## Синтаксис


ToolTip: String;


## Описание


Свойство ToolTip определяет
 текст всплывающей подсказки компонента.


## Комментарии


Свойство актуально, только если для свойства [Control.ShowToolTip](Control.ShowToolTip.htm)
 установлено значение true.


## Пример


Пример использования приведён в описании свойства [Control.Opacity](Control.Opacity.htm).
