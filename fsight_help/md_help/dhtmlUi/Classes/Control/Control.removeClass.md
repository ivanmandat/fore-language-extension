# Control.removeClass


## Синтаксис


removeClass(element, value);


## Параметры


element. Элемент
 разметки, из поля className которого необходимо удалить указанное имя
 CSS-класса;


value. Имя
 CSS-класса.


## Описание


Метод removeClass удаляет CSS-класс
 из компонента.
