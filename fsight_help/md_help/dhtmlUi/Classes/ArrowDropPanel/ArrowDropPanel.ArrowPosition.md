# ArrowDropPanel.ArrowPosition


## Синтаксис


ArrowPosition: [PP.Ui.DropPanelArrowPosition](../../Enums/DropPanelArrowPosition.htm);


## Описание


Свойство ArrowPosition определяет
 позицию стрелки относительно панели.


## Комментарии


Значение свойства - элемент перечисления [PP.Ui.DropPanelArrowPosition](../../Enums/DropPanelArrowPosition.htm).


Значение свойства устанавливается из JSON и с помощью метода setArrowPosition,
 а возвращается с помощью метода getArrowPosition.
