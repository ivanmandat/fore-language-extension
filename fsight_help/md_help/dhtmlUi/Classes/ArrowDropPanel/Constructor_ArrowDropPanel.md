# Конструктор ArrowDropPanel


## Синтаксис


PP.Ui.ArrowDropPanel(settings);


## Параметры


settings. JSON объект со значениями
 свойств компонента.


## Описание


Конструктор ArrowDropPanel создает
 экземпляр компонента [ArrowDropPanel](../../Components/ArrowDropPanel/ArrowDropPanel.htm).
