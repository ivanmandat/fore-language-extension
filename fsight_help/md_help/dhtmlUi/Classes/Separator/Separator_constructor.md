# Конструктор Separator


## Синтаксис


PP.Ui.Separator(settings)


## Параметры


settings. JSON-объект со значениями
 свойств компонента.


## Описание


Конструктор Separator создает
 экземпляр класса [Separator](Separator.htm).


## Пример


Пример использования конструктора при динамическом размещении компонента
 [Menu](../../Components/Menu/Menu.htm) приведен в примере для
 свойства [Menu.ImageList](../Menu/Menu.ImageList.htm), при
 статическом - на странице «[Пример
 создания компонента Menu](../../Components/Menu/Menu_example.htm)».
