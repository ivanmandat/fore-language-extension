# Combo.IsDropHasCombosWidth


## Синтаксис


IsDropHasCombosWidth: Boolean


## Описание


Свойство IsDropHasCombosWidth
 устанавливает признак независимости ширины раскрывающейся панели от ширины
 редактора ввода.


## Комментарии


Допустимые значения:


	- true. По умолчанию.
	 Ширина раскрывающейся панели равна ширине редактора ввода;


	- false. Ширина раскрывающейся
	 панели не будет подогнана под ширину компонента.


Значение свойства устанавливается из JSON и с помощью метода setIsDropHasCombosWidth,
 а возвращается с помощью метода getIsDropHasCombosWidth.
