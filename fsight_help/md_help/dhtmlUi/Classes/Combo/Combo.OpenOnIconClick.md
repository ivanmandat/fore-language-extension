# Combo.OpenOnIconClick


OpenOnIconClick: Boolean


## Описание


Свойство OpenOnIconClick определяет
 возможность раскрытия панели компонента по нажатию на пиктограмму заголовка.


## Комментарии


Допустимые значения:


	- true.
	 При нажатии на пиктограмму заголовка панель раскроется;


	- false.
	 При нажатии на пиктограмму заголовка панель не раскроется (по умолчанию).


Значение свойства устанавливается из JSON и с помощью метода setOpenOnIconClick,
 а возвращается с помощью метода getOpenOnIconClick.
