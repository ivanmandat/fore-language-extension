# Combo.CloseOnContentClick


CloseOnContentClick: Boolean


## Описание


Свойство CloseOnContentClick
 определяет возможность закрытия панели компонента по нажатию на заголовок.


## Комментарии


Допустимые значения:


	- true.
	 При нажатии на заголовок панель компонента закроется (по умолчанию);


	- false.
	 При нажатии на заголовок панель не закроется.


Значение свойства устанавливается из JSON и с помощью метода setCloseOnContentClick,
 а возвращается с помощью метода getCloseOnContentClick.
