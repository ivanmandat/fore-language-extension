# Combo.OpenOnContentClick


## Синтаксис


OpenOnContentClick: Boolean;


## Описание


Свойство OpenOnContentClick
 определяет возможность раскрытия панели компонента по нажатию на заголовок.


## Комментарии


Допустимые значения:


	- true.
	 При нажатии на заголовок раскроется панель компонента (по умолчанию);


	- false.
	 При нажатии на заголовок панель компонента не раскроется.


Значение свойства устанавливается из JSON и с помощью метода setOpenOnContentClick,
 а возвращается с помощью метода getOpenOnContentClick.


При нажатии на редактор ввода в режиме редактирования панель компонента
 не раскроется.
