# Combo.EnableEdit


## Синтаксис


EnableEdit: Boolean


## Описание


Свойство EnableEdit устанавливает возможность ввода.


## Комментарии


Если для свойства EnableEdit установлено значение true, текст в строке ввода можно редактировать.
