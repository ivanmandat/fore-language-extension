# Combo.showDropPanel


## Синтаксис


showDropPanel();


## Описание


Метод showDropPanel отображает раскрывающуюся часть компонента.


## Пример


Для выполнения примера создайте компонент [Combo](../../Components/Combo/Combo.htm) (см. [Пример создания компонента Combo](../../Components/Combo/Example_Combo.htm)).


Создайте кнопку с наименованием «Открыть», при нажатии на которую будет выполняться метод showDropPanel:


<button onclick = "Combo.showDropPanel()">Открыть</button>


После выполнения примера при нажатии на кнопку будет отображаться раскрывающаяся часть компонента Combo.
