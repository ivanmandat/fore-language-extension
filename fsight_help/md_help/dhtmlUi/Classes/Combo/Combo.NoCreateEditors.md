# Combo.NoCreateEditors


## Синтаксис


NoCreateEditors: Boolean


## Описание


Свойство NoCreateEditors определяет
 признак того, что заголовок отобразится простым текстом, а не в редакторах
 компонентов [TextBox](../../Components/TextBox/TextBox.htm)
 или [MaskEdit](../../Components/MaskEdit/MaskEdit.htm).


## Комментарии


Допустимые значения:


	- true. Заголовок отобразится
	 в виде простого текста, а не в редакторе компонентов [TextBox](../../Components/TextBox/TextBox.htm)
	 или [MaskEdit](../../Components/MaskEdit/MaskEdit.htm);


	- false. По умолчанию.
	 Редакторы будут созданы.


Значение свойства устанавливается из JSON и с помощью метода setNoCreateEditors,
 а возвращается с помощью метода getNoCreateEditors.
