# Combo.UseMask


## Синтаксис


UseMask: Boolean


## Описание


Свойство UseMask устанавливает возможность использования редактора маски.


## Комментарии


Если установлено значение true, то для компонента [Combo](../../Components/Combo/Combo.htm) включена возможность использования маски. По умолчанию установлено значение false.


Для корректной работы метода [Combo.getMaskEdit](Combo.getMaskEdit.htm) для свойства должно быть установлено значение true.


## Пример


Пример использования свойства приведен в описании метода [Combo.getMaskEdit](Combo.getMaskEdit.htm).
