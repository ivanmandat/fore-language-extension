# Combo.IsHint


IsHint: Boolean


## Описание


Свойство IsHint определяет отображение
 подсказки при наведении курсора мыши на кнопку открытия раскрывающейся
 панели.


## Комментарии


Допустимые значения:


	- true.
	 Подсказка появится при наведении курсора мыши на кнопку открытия раскрывающейся
	 панели (по умолчанию);


	- false.
	 Подсказка не появится при наведении курсора мыши на кнопку открытия
	 раскрывающейся панели.


Значение свойства устанавливается из JSON и с помощью метода setIsHint,
 а возвращается с помощью метода getIsHint.
