# DiscussMenu.AutoShare


## Синтаксис


AutoShare: Boolean


## Описание


Свойство AutoShare определяет, будет ли при выборе пункта меню осуществляться переход на соответствующий веб-сайт.


## Комментарии


Если для свойства установлено значение true (по умолчанию), при выборе пункта меню осуществляется переход на соответствующий веб-сайт. Если установлено значение false, переход не осуществляется.


Значение свойства устанавливается при помощи метода setAutoShare и возвращается при помощи метода getAutoShare.
