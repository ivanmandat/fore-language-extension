# DiscussMenu.goPlusone


## Синтаксис


goPlusone (settings)


## Параметры


settings. JSON-объект, в котором задаются следующие параметры:


-
url. URL текущей страницы. Значение задано по умолчанию;


-
container. Контейнер, в котором размещается кнопка. Совпадает с параметром parentNode метода [DiscussMenu.appendPluseoneContent](DiscussMenu.appendPluseoneContent.htm);


-
pluseone. JSON-объект с настройками кнопки. Параметры кнопки см. на [developers.google.com](https://developers.google.com/%2b/plugins/%2b1button/)


## Описание


Метод goPlusone выполняет отрисовку кнопки Google+1.


## Комментарии


Для работы метода необходимо, чтобы на странице была [добавлена ссылка на plusone.js](DiscussMenu.appendGPlusLink.htm) и [размещена кнопка Google+1](DiscussMenu.appendPluseoneContent.htm).
