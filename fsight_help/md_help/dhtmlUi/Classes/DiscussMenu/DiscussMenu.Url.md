# DiscussMenu.Url


## Синтаксис


Url: String


## Описание


Свойство Url определяет ссылку,
 которую необходимо опубликовать.


## Комментарии


По умолчанию для свойства установлено значение window.location.href
 - ссылка на текущую страницу.


Значение свойства устанавливается при помощи метода setUrl
 и возвращается при помощи метода getUrl.

 и getUrl приведен в описании метода
