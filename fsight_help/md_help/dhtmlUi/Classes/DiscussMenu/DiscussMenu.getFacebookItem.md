# DiscussMenu.getFacebookItem


## Синтаксис


getFacebookItem()


## Описание


Метод getFacebookItem возвращает
 пункт меню «Facebook» (Компания Meta
 Platforms Inc., владеющая социальной сетью Facebook, признана экстремистской
 организацией, её деятельность на территории России запрещена).


## Комментарии


Метод возвращает объект класса PP.Ui.[MenuItem](../MenuItem/MenuItem.htm).
