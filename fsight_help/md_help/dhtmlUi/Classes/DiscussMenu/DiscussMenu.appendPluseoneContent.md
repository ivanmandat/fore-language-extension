# DiscussMenu.appendPluseoneContent


## Синтаксис


appendPluseoneContent (parentNode)


## Параметры


parentNode. DOM-вершина. Если значение не задано, размещает в DOM-вершине, указанной по умолчанию.


## Описание


Метод appendPluseoneContent вставляет кнопку Google+1 в указанную DOM-вершину.


## Комментарии


Метод работает, если на странице [добавлена ссылка на plusone.js](DiscussMenu.appendGPlusLink.htm).


## Пример


Пример использования приведен в описании метода [DiscussMenu.appendGPlusLink](DiscussMenu.appendGPlusLink.htm).
