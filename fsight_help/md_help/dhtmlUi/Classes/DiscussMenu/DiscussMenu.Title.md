# DiscussMenu.Title


## Синтаксис


Title: String


## Описание


Свойство Title определяет заголовок сообщения, которое будет размещено в социальной сети.


## Комментарии


По умолчанию значение свойства не определено.


Значение свойства устанавливается при помощи метода setTitle и возвращается при помощи метода getTitle. Из JSON значение свойства определить нельзя.
