# DiscussMenu.BeforeDiscuss


## Синтаксис


BeforeDiscuss: function (sender,args)


## Параметры


sender. Источник события;


args. Информация о событии. Доступный аргумент - ShareType - определяет выбранный пункт меню.


## Описание


Событие BeforeDiscuss наступает перед публикацией, то есть при выборе пункта меню.
