# Конструктор MainMenuButton


## Синтаксис


PP.Ui.MainMenuButton(settings)


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор MainMenu создает экземпляр класса [MainMenuButton](MainMenuButton.htm).
