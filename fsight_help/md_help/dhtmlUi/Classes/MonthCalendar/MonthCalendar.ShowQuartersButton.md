# MonthCalendar.ShowQuartersButton


## Синтаксис


ShowQuartersButton: String


## Описание


Свойство ShowQuartersButton устанавливает видимость кнопки «Кварталы».


## Комментарии


По умолчанию для свойства установлено значение true, то есть кнопка отображается.


## Пример


Для выполнения примера создайте компонент [MonthCalendar](../../Components/MonthCalendar/MonthCalendar.htm) (см. «[Пример создания компонента MonthCalendar](../../Components/MonthCalendar/Example_MonthCalendar.htm)»). Чтобы кнопка «Кварталы» не отображалась, в компоненте используйте свойство ShowQuartersButton:


ShowQuartersButton: false
