# Конструктор MonthCalendar


## Синтаксис


PP.MonthCalendar(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор MonthCalendar создает экземпляр компонента [MonthCalendar](../../Components/MonthCalendar/MonthCalendar.htm).
