# MonthCalendar.ShowWeeksButton


## Синтаксис


ShowWeeksButton: Boolean


## Описание


Свойство ShowWeeksButton устанавливает видимость кнопки «Недели».


## Комментарии


По умолчанию для свойства установлено значение true, то есть кнопка отображается.


## Пример


Для выполнения примера создайте компонент [MonthCalendar](../../Components/MonthCalendar/MonthCalendar.htm) (см. «[Пример создания компонента MonthCalendar](../../Components/MonthCalendar/Example_MonthCalendar.htm)»). Чтобы кнопка «Недели» не отображалась, в компоненте используйте свойство ShowWeeksButton:


ShowWeeksButton: false
