# MonthCalendar.ShowNowButton


## Синтаксис


ShowNowButton: Boolean


## Описание


Свойство ShowNowButton устанавливает видимость кнопки «Сегодня».


## Комментарии


По умолчанию для свойства установлено значение true, то есть кнопка отображается.


## Пример


Для выполнения примера создайте компонент [MonthCalendar](../../Components/MonthCalendar/MonthCalendar.htm) (см. «[Пример создания компонента MonthCalendar](../../Components/MonthCalendar/Example_MonthCalendar.htm)»). Чтобы кнопка «Сегодня» не отображалась, в компоненте используйте свойство ShowNowButton:


ShowNowButton: false
