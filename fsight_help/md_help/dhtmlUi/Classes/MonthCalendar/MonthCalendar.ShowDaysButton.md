# MonthCalendar.ShowDaysButton


## Синтаксис


ShowDaysButton: Boolean


## Описание


Свойство ShowDaysButton устанавливает видимость кнопки «Дни».


## Комментарии


По умолчанию для свойства установлено значение true, то есть кнопка отображается.


## Пример


Для выполнения примера создайте компонент MonthCalendar (см. «[Пример создания компонента MonthCalendar](../../Components/MonthCalendar/Example_MonthCalendar.htm)»). Чтобы кнопка «Дни» не отображалась, используйте свойство ShowDaysButton:


ShowDaysButton : false
