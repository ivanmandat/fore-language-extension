# MonthCalendar.ShowHalfYearsButton


## Синтаксис


ShowHalfYearsButton: String


## Описание


Свойство ShowHalfYearsButton устанавливает видимость кнопки «Полугодия».


## Комментарии


По умолчанию для свойства установлено значение true, то есть кнопка отображается.


## Пример


Для выполнения примера создайте компонент [MonthCalendar](../../Components/MonthCalendar/MonthCalendar.htm) (см. «[Пример создания компонента MonthCalendar](../../Components/MonthCalendar/Example_MonthCalendar.htm)»). Чтобы кнопка «Полугодия» не отображалась, используйте свойство ShowHalfYearsButton:


ShowHalfYearsButton : false
