# MonthCalendar.ShowTime


## Синтаксис


ShowTime: Boolean


## Описание


Свойство ShowTime устанавливает видимость элементов для ввода времени.


## Комментарии


По умолчанию для свойства установлено значение true, то есть элементы для ввода времени отображаются.


## Пример


Для выполнения примера создайте компонент [MonthCalendar](../../Components/MonthCalendar/MonthCalendar.htm) (см. «[Пример создания компонента MonthCalendar](../../Components/MonthCalendar/Example_MonthCalendar.htm)»). Чтобы элементы для ввода времени не отображались, в компоненте используйте свойство ShowTime:


ShowTime: false
