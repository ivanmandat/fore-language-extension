# MonthCalendar.ShowMonthsButton


## Синтаксис


ShowMonthsButton: Boolean


## Описание


Свойство ShowMonthsButton устанавливает видимость кнопки «Месяцы».


## Комментарии


По умолчанию для свойства установлено значение true, то есть кнопка отображается.


## Пример


Для выполнения примера создайте компонент [MonthCalendar](../../Components/MonthCalendar/MonthCalendar.htm) (см. «[Пример создания компонента MonthCalendar](../../Components/MonthCalendar/Example_MonthCalendar.htm)»). Чтобы кнопка «Полугодия» не отображалась в компоненте, используйте свойство ShowMonthsButton:


ShowMonthsButton: false
