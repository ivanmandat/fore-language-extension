# MonthCalendar.CurrentDate


## Синтаксис


CurrentDate: Date


## Описание


Свойство CurrentDate устанавливает дату в календаре.


## Комментарии


Значение месяца задается в диапазоне от 0 до 11.


Если значение даты и времени не установлено, то в редакторе отображаются текущие дата и время.


## Пример


Для выполнения примера создайте календарь (см. «[Пример создания компонента MonthCalendar](../../Components/MonthCalendar/Example_MonthCalendar.htm)»). Создадим переменную, содержащую дату и время:


var date = new Date (2011, 08, 29, 11, 0, 0, 0);


Установим эту переменную в качестве значения для свойства CurrentDate:


CurrentDate: date


После выполнения примера для установленного значения даты появится отметка во всех режимах отображения календаря.
