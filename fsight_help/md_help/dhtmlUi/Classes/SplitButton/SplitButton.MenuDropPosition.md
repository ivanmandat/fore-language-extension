# SplitButton.MenuDropPosition


## Синтаксис


MenuDropPosition: PP.[LTRB](dhtmlCommon.chm::/Enums/LTRB.htm)


## Описание


Свойство MenuDropPosition устанавливает
 позицию открытия дочернего меню компонента.


## Комментарии


Значения свойства - элементы перечисления [LTRB](dhtmlCommon.chm::/Enums/LTRB.htm).
 По умолчанию установлено значение Bottom.


Вместе с изменением позиции открытия дочернего компонента меняется и
 направление стрелки.


Значение свойства устанавливается при помощи метода setMenuDropPosition
 и возвращается при помощи метода getMenuDropPosition.
 Задать значение свойства из [конструктора](Constructor_SplitButton.htm)
 нельзя.

Пример использования метода setMenuDropPosition
 приведен в описании метода [SplitButton.getColumnIndex](SplitButton.getColumnIndex.htm).
