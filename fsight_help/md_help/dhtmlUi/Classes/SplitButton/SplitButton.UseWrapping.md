# SplitButton.UseWrapping


## Синтаксис


UseWrapping: Boolean


## Описание


Свойство UseWrapping осуществляет перенос слов на 2 строки.


## Комментарии


При установке значения true осуществляется перенос по словам. По умолчанию установлено значение false.
