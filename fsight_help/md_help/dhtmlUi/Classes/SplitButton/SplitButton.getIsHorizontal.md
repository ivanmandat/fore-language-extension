# SplitButton.getIsHorizontal


## Синтаксис


getIsHorizontal ();


## Описание


Метод getIsHorizontal возвращает признак того, что для кнопки применяется горизонтальное размещение.


## Комментарии


Если применяется горизонтальное размещение, то возвращается значение true, иначе - false (по умолчанию).


По умолчанию кнопка размещается горизонтально, то есть картинка сверху, надпись снизу:


![](SplitButton1.gif)


Чтобы кнопка размещалась горизонтально, задайте в [конструкторе](Constructor_SplitButton.htm) для свойства IsHorizontal значение true. Пример горизонтального размещения:


![](SplitButton_horizontal.gif)
