# SplitButton.getImageButton


## Синтаксис


getImageButton ();


## Описание


Метод getImageButton возвращает верхнюю часть кнопки (с картинкой).


## Комментарии


В верхней части кнопки отображается картинка, путь к которой задан в свойстве [SplitButton.ImageUrl](SplitButton.ImageUrl.htm).
