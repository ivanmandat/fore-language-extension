# SplitButton.getContentButton


## Синтаксис


getContentButton ();


## Описание


Метод getContentButton возвращает нижнюю часть кнопки (с раскрывающимся меню).


## Комментарии


В нижней части кнопки размещается текст, заданный в свойстве [Control.Content](../Control/Control.Content.htm).
