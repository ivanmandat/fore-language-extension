# SplitButton.IsPressed


## Синтаксис


IsPressed: Boolean


## Описание


Свойство IsPressed устанавливает признак того, что кнопка находится в нажатом состоянии.


## Комментарии


Если для свойства установлено значение true, то кнопка отображается в нажатом состоянии.


По умолчанию установлено значение false.


Свойство актуально, если для свойства [SplitButton.IsToggleButton](SplitButton.IsToggleButton.htm) установлено значение true.
