# SplitButton.getRowIndex


## Синтаксис


getRowIndex ();


## Описание


Метод getRowIndex возвращает индекс строки с пиктограммой, выбранной из [спрайта](SplitButton.ImageList.htm).


## Комментарии


Индекс столбца с пиктограммой можно задать в свойстве RowIndex из [конструктора](Constructor_SplitButton.htm).


## Пример


Пример использования свойства RowIndex приведен в описании метода [SplitButton.getColumnIndex](SplitButton.getColumnIndex.htm).
