# SplitButton.IsToggleButton


## Синтаксис


IsToggleButton: Boolean


## Описание


Свойство IsToggleButton определяет, можно ли переключать состояния верхней части кнопки (нажать/отжать).


## Комментарии


По умолчанию для свойства установлено значение false, то есть состояние кнопки изменить нельзя. Если установлено значение true, то кнопку можно нажать/отжать.
