# ColorLegendBase.GreaterFormat


## Синтаксис


GreaterFormat: String;


## Описание


Свойство GreaterFormat определяет формат записи интервалов с левой границей.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setGreaterFormat, а возвращается с помощью метода getGreaterFormat.**
