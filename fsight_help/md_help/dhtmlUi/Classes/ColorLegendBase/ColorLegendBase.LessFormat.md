# ColorLegendBase.LessFormat


## Синтаксис


LessFormat: String;


## Описание


Свойство LessFormat определяет формат записи интервалов с правой границей.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLessFormat, а возвращается с помощью метода getLessFormat.**
