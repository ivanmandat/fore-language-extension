# ColorLegendBase.NoDataPosition


## Синтаксис


NoDataPosition: [PP.Ui.NoDataPosition](dhtmlGraphicBase.chm::/Enums/NoDataPosition.htm);


## Описание


Свойство NoDataPosition определяет
 расположение элементов, соответствующих отсутствующим данным в визуализаторе.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setNoDataPosition,
 а возвращается с помощью метода getNoDataPosition.**
