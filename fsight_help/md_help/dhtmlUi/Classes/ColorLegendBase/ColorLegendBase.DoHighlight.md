# ColorLegendBase.DoHighlight


## Синтаксис


DoHighlight: Boolean;


## Описание


Свойство DoHighlight определяет признак, разрешающий подсветку элементов цветовой шкалы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDoHighlight, а возвращается с помощью метода getDoHighlight.**


Свойство содержит значение True, если подсветка элементов цветовой шкалы разрешена, и значение False, если запрещена.
