# ColorLegendBase.ActiveEffects


## Синтаксис


ActiveEffects: [PP.Brush](dhtmlCommon.chm::/Classes/PP/Brush/Brush.htm)


## Описание


Свойство ActiveEffects определяет
 массив эффектов, применяемых к активным элементам шкалы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setActiveEffects,
 а возвращается с помощью метода getActiveEffects.**
