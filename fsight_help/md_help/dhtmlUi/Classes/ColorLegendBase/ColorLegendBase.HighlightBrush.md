# ColorLegendBase.HighlightBrush


## Синтаксис


HighlightBrush: [PP.Brush](dhtmlCommon.chm::/Classes/PP/Brush/Brush.htm);


## Описание


Свойство HighlightBrush определяет
 кисть для подсветки элементов визуализатора.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHighlightBrush,
 а возвращается с помощью метода getHighlightBrush.**
