# ColorLegendBase.NoDataText


## Синтаксис


NoDataText: String;


## Описание


Свойство NoDataText определяет подпись для элемента, соответствующего отсутствующим данным в визуализаторе.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setNoDataText, а возвращается с помощью метода getNoDataText.**
