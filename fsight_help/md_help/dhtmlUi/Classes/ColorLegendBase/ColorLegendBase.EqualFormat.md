# ColorLegendBase.EqualFormat


## Синтаксис


EqualFormat: String;


## Описание


Свойство EqualFormat определяет формат записи интервалов с конкретным значением.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setEqualFormat, а возвращается с помощью метода getEqualFormat.**
