# ColorLegendBase.InactiveEffects


## Синтаксис


InactiveEffects: [PP.Brush](dhtmlCommon.chm::/Classes/PP/Brush/Brush.htm)


## Описание


Свойство InactiveEffects определяет
 массив эффектов, применяемых к неактивным элементам шкалы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setInactiveEffects,
 а возвращается с помощью метода getInactiveEffects.**
