# ColorLegendBase.NoDataVisible


## Синтаксис


NoDataVisible: Boolean;


## Описание


Свойство NoDataVisible определяет
 признак видимости элемента, соответствующего отсутствующим данным в визуализаторе.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setNoDataVisible,
 а возвращается с помощью метода getNoDataVisible.**


Свойство содержит значение True,
 если элемент, соответствующий отсутствующим данным, отображён в визуализаторе,
 и значение False, если не отображён.
