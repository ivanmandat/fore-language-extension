# ColorLegendBase.HighlightItemColor


## Синтаксис


HighlightItemColor (value: [PP.Color](dhtmlCommon.chm::/Classes/PP/Color/Color.htm)
 || String);


## Параметры


value. Цвет маркера и подписи
 элемента легенды.


## Описание


Свойство HighlightItemColor
 устанавливает цвет маркера и подписи подсвеченного элемента легенды.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHighlightItemColor.


В качестве значения параметра можно использовать:


	- JSON-объект со значениями свойств класса [PP.Color](dhtmlCommon.chm::/Classes/PP/Color/Color.htm);


	- строку с названием цвета (например: «White», «Red», «Black»
	 и т.д.);


	- строку со значением цвета в шестнадцатиричном формате со знаком
	 решетки в начале (например: «#FFFFFFFF»);


	- строку со значением цвета в формате RGB (например: «rgb(0, 255,
	 0)»).
