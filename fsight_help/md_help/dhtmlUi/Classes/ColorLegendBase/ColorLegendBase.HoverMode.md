# ColorLegendBase.HoverMode


## Синтаксис


HoverMode: [PP.HoverMode](dhtmlMapChart.chm::/Enums/HoverMode.htm);


## Описание


Свойство HoverMode определяет
 режим активации подсветки элементов легенды.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHoverMode,
 а возвращается с помощью метода getHoverMode.**
