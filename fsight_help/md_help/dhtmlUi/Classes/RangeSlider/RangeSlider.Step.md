# RangeSlider.Step


## Синтаксис


Step: Number


## Описание


Свойство Step определяет шаг изменения значения** [бегунка](../SliderItem/SliderItem.htm).


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setStep, а возвращается с помощью метода getStep.
