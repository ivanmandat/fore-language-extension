## Описание


Класс RangeSlider реализует
 [элемент управления,
 предназначенный для выбора интервала значений](../../Components/RangeSlider/RangeSlider.htm) с помощью перемещения
 [бегунков](../SliderItem/SliderItem.htm).


## Синтаксис


PP.initClass(PP.Ui.RangeSlider, [PP.Ui.Slider](../Slider/slider.htm),
 "PP.Ui.RangeSlider");
