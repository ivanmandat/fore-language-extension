# RangeSlider.MaxValue


## Синтаксис


MaxValue: Number


## Описание


Свойство MaxValue** определяет максимальное значение [бегунка](../SliderItem/SliderItem.htm).


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMaxValue, а возвращается с помощью метода getMaxValue.
