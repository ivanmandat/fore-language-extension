# RangeSlider.Range


## Синтаксис


Range: Object


## Описание


Свойство Range** определяет выбираемый [бегунками](../SliderItem/SliderItem.htm) диапазон значений.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setRange, а возвращается с помощью метода getRange.


Свойство содержит следующие поля: Begin - начальное значение интервала выбора значений, End - конечное значение интервала выбора значений.
