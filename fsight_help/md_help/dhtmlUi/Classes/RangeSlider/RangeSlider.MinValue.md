# RangeSlider.MinValue


## Синтаксис


MinValue: Number


## Описание


Свойство MinValue** определяет минимальное значение [бегунка](../SliderItem/SliderItem.htm).


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMinValue а возвращается с помощью метода getMinValue.
