# Конструктор RangeSlider


## Синтаксис


PP.Ui.RangeSlider(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор RangeSlider создает экземпляр компонента [RangeSlider](../../Components/RangeSlider/RangeSlider.htm).
