# RangeSlider.FractionCount


## Синтаксис


FractionCount: Number


## Описание


Свойство FractionCount** определяет количество знаков после запятой, до которых округляется значение шага изменения значения [бегунков](../SliderItem/SliderItem.htm).


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setFractionCount, а возвращается с помощью метода getFractionCount.
