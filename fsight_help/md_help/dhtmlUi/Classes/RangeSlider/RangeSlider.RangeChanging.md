# RangeSlider.RangeChanging


## Синтаксис


RangeChanging: function(sender, args, timeout);


## Параметры


sender. Источник события;


args. Информация о событии;


timeout. Промежуток времени в миллисекундах, через который вызовется событие.


## Описание


Событие RangeChanging** наступает при изменении выбираемого [бегунками](../SliderItem/SliderItem.htm) диапазона значений.
