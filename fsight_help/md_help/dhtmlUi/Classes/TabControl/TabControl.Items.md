# TabControl.Items


## Синтаксис


Items;


## Описание


Свойство Items содержит массив
 вкладок элемента управления.


## Комментарии


Элементы массива являются объектами класса [TabItem](../TabItem/TabItem.htm).


Значение свойства устанавливается из JSON и с помощью метода setItems,
 а возвращается с помощью метода getItems.
