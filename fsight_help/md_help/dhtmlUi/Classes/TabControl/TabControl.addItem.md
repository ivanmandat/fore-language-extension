# TabControl.addItem


## Синтаксис


addItem(value);


## Параметры


value. Вкладка, экземпляр класса
 [PP.TabItem](../TabItem/TabItem.htm).


## Описание


Метод addItem добавляет вкладку
 к компоненту.
