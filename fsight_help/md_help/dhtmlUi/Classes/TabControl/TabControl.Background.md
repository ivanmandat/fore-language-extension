# TabControl.Background


## Синтаксис


Background: String | PP.[Color](dhtmlCommon.chm::/Classes/pp/color/color.htm);


## Описание


Свойство Background определяет
 фон для панели компонента. на которой размещаются заголовки вкладок.


## Комментарии


В качестве значения свойства могут выступать:


	- цвет в виде объекта [PP.Color](dhtmlCommon.chm::/Classes/pp/color/color.htm);


	- строки;


	- URL-изображения.
