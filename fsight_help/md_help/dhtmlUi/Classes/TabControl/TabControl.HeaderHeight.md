# TabControl.HeaderHeight


## Синтаксис


HeaderHeight: Number;


## Описание


Свойство HeaderHeight определяет
 высоту вкладок элемента управления.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHeaderHeight,
 а возвращается с помощью метода getHeaderHeight.


По умолчанию значение свойства равно 26.
