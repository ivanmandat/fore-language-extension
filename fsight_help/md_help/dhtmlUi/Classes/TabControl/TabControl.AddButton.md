# TabControl.AddButton


## Синтаксис


AddButton: PP.Delegate | function (sender,args);


## Параметры


sender. Источник события;


args. Информация
 о событии.


## Описание


Событие AddButton наступает
 при добавлении кнопки в элемент управления вкладками.


## Комментарии


Добавление кнопки осуществляется при помощи метода [TabControl.addButton](TabControl.addButton_M.htm).
