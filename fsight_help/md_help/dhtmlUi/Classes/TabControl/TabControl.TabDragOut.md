# TabControl.TabDragOut


## Синтаксис


TabDragOut: function (sender, args);


## Параметры


sender. Источник
 события;


args. Информация о событии.


## Описание


Событие TabDragOut наступает
 при вынесении вкладки за пределы области элемента управления.


## Комментарии


Событие актуально, если для свойства [TabControl.EnableDragOutside](TabControl.EnableDragOutside.htm)
 установлено значение true.


## Пример


Пример использования события приведен в описании метода [TabControl.removeItemByIndex](TabControl.removeItemByIndex.htm).
