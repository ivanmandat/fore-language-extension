# TabControl.TabEditing


## Синтаксис


TabEditing: function (sender, args);


## Параметры


sender. Источник
 события;


args. Информация о событии.


## Описание


Событие TabEditing наступает
 при переходе в режим редактирования.


## Комментарии


Допустимые значения args:


	- TabIndex. Возвращает
	 индекс редактируемой вкладки;


	- Caption. Возвращает
	 заголовок редактируемой вкладки.


Для перехода в режим редактирования дважды щёлкните по заголовку
 вкладки.


## Пример


Пример использования события приведен в описании свойства [TabControl.EnableEdit](TabControl.EnableEdit.htm).
