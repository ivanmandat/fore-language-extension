# TabControl.UseToolBarSwitch


## Синтаксис


UseToolBarSwitch: Boolean;


## Описание


Свойство UseToolBarSwitch определяет
 признак стиля вкладок, как у панели инструментов.


## Комментарии


Допустимые значения:


	- true.
	 Стиль вкладок, как у панели инструментов;


	- false.
	 По умолчанию. Обычный стиль вкладок.


Значение свойства устанавливается из JSON и с помощью метода setUseToolBarSwitch,
 а возвращается с помощью метода getUseToolBarSwitch.
