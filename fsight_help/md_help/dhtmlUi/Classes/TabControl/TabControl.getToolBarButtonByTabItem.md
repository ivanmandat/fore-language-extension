# TabControl.getToolBarButtonByTabItem


## Синтаксис


getToolBarButtonByTabItem(item: [PP.Ui.TabItem](../TabItem/TabItem.htm));


## Параметры


item. Вкладка элемента управления.


## Описание


Метод getToolBarButtonByTabItem
 возвращает кнопку панели инструментов вкладки элемента управления.


## Комментарии


Возвращаемое значение - элемент класса [PP.Ui.ToolBarButton](../ToolBarButton/ToolBarButton.htm).


Метод работает, если свойство [TabControl.UseToolBarSwitch](TabControl.UseToolBarSwitch.htm)
 имеет значение true.
