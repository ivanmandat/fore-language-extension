# TabControl.DragDropEnabled


## Синтаксис


DragDropEnabled: Boolean;


## Описание


Свойство DragDropEnabled определяет
 признак возможности изменения положения вкладок при помощи механизма Drag&Drop.


## Комментарии


Допустимые значения:


	- true. По умолчанию.
	 Вкладки можно перетаскивать с помощью механизма Drag&Drop;


	- false. Вкладки нельзя
	 перетаскивать с помощью механизма Drag&Drop.
