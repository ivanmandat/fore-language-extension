# TabControl.MenuAddItems


## Синтаксис


MenuAddItems: Object;


## Описание


Свойство MenuAddItems определяет
 меню добавления вкладок.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMenuAddItems,
 а возвращается с помощью метода getMenuAddItems.


Свойство находится в разработке.
