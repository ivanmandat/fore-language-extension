# TabControl.TabDeleted


## Синтаксис


TabDeleted: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие TabDeleted наступает
 после удаления вкладки.


## Комментарии


Для удаления вкладки нажмите кнопку ![](../../Components/SlidePanel/SlidePanel4.gif)
 «Удалить» в её заголовке.


Удаление доступно, если для свойства [TabControl.EnableDelete](TabControl.EnableDelete.htm)
 установлено значение true.


## Пример


Пример использования события приведен в описании свойства [TabControl.EnableDelete](TabControl.EnableDelete.htm).
