# TabControl.SelectOnStartEditing


## Синтаксис


SelectOnStartEditing: Boolean;


## Описание


Свойство SelectOnStartEditing
 определяет признак выделения заголовка вкладки при двойном щелчке по нему.


## Комментарии


Допустимые значения:


	- true.
	 По умолчанию. При двойном щелчке по заголовку вкладки произойдёт выделение
	 текста заголовка;


	- false.
	 При двойном щелчке по заголовку вкладки произойдёт установка курсора
	 в конец текста заголовка.


Свойство актуально, только если для свойства [TabControl.EnableEdit](TabControl.EnableEdit.htm)
 установлено значение true.


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [TabControl](../../Components/TabControl/TabControl.htm)
 с наименованием «tabControl» (см. «[Пример
 создания компонента TabControl](../../Components/TabControl/Example_TabControl.htm)»). Разрешим редактирование заголовков
 вкладок и определим, что при двойном щелчке по заголовкам будет устанавливаться
 курсор в конец текста:


//разрешаем редактирование заголовка вкладки
tabControl.setEnableEdit(true);
//при двойном щелчке по заголовку будет установлен курсор в конец текста
tabControl.setSelectOnStartEditing(false);

После выполнения примера будет разрешено редактирование заголовков вкладки.
 При двойном щелчке по заголовку в конце текста будет устанавливаться курсор.
