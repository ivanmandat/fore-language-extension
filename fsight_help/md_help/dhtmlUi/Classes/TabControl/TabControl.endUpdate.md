# TabControl.endUpdate


## Синтаксис


endUpdate();


## Описание


Метод endUpdate включает перерисовку
 элемента управления и обновляет её.


## Комментарии


Использует в связке с методом [TabControl.beginUpdate](TabControl.beginUpdate.htm).

Пример использования метода приведён в описании метода [TabControl.removeItemByIndex](TabControl.removeItemByIndex.htm).
