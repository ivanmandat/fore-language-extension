# TabControl.EnableDragOutside


## Синтаксис


EnableDragOutside: Boolean;


## Описание


Свойство EnableDragOutside определяет
 признак возможности выноса вкладки из элемента управления.


## Комментарии


Допустимые значения:


	- true.
	 Вкладку можно вынести из элемента управления;


	- false.
	 По умолчанию. Вкладку нельзя вынести из элемента управления.


Значение свойства устанавливается из JSON и с помощью метода setEnableDragOutside,
 а возвращается с помощью метода getEnableDragOutside.
