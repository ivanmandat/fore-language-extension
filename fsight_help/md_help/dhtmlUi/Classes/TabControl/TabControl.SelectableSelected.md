# TabControl.SelectableSelected


## Синтаксис


SelectableSelected: function(sender, args);


## Параметры


sender. Источник
 события;


args. Информация о событии.


## Описание


Событие SelectableSelected наступает
 при нажатии кнопки мыши на элемент.


## Комментарии


Допустимые значения параметра args:


	- Event. Настройки события;


	- Element. DOM-элемент,
	 на котором произошло событие;


	- Control. Элемент, по
	 которому было выполнено нажатие.
