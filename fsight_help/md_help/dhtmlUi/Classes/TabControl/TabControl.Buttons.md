# TabControl.Buttons


## Синтаксис


Buttons: Object;


## Описание


Свойство Buttons определяет
 объект с массивами кнопок каждой стороны элемента управления.


## Комментарии


Свойство Buttons - объект
 следующего вида: { <Сторона элемента>: <Массив кнопок> }.


Элементы массивов являются объектами класса [PP.Ui.Button](../Button/Button.htm).


Значение свойства устанавливается из JSON и с помощью метода setButtons,
 а возвращается с помощью метода getButtons.
