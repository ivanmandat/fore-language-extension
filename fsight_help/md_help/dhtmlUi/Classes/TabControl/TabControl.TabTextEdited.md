# TabControl.TabTextEdited


## Синтаксис


TabTextEdited: function (sender, args);


## Параметры


sender. Источник
 события;


args. Информация о событии.


## Описание


Событие TabTextEdited наступает
 при выходе из режима редактирования, если был изменён текст заголовка.


## Комментарии


Допустимые значения параметра args:


	- TabIndex. Индекс редактируемой
	 вкладки;


	- Caption. Заголовок редактируемой
	 вкладки.


Для выхода из режима редактирования щёлкните мышкой по пространству
 вне редактируемого заголовка.
