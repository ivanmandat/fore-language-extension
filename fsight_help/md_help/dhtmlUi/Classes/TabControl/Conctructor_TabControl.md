# Конструктор TabControl


## Синтаксис


PP.TabControl(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор TabControl создает экземпляр компонента [TabControl](../../Components/TabControl/TabControl.htm).
