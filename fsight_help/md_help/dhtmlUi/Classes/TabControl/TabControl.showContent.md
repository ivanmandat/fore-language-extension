# TabControl.showContent


## Синтаксис


showContent();


## Описание


Метод showContent отображает
 содержимое вкладок.


## Комментарии


Для скрытия содержимого вкладок используйте метод [TabControl.hideContent](TabControl.hideContent.htm).


## Пример


Пример использования метода приведен в описании свойства [TabControl.EnableEdit](TabControl.EnableEdit.htm).
