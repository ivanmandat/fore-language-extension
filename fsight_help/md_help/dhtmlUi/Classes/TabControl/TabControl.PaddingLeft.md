# TabControl.PaddingLeft


## Синтаксис


PaddingLeft: Number;


## Описание


Свойство PaddingLeft определяет
 величину левого отступа панели вкладок элемента управления.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setPaddingLeft,
 а возвращается с помощью метода getPaddingLeft.


По умолчанию значение свойства равно 0.
