# TabControl.EnableDropFromOutside


## Синтаксис


EnableDropFromOutside: Boolean;


## Описание


Свойство EnableDropFromOutside
 определяет признак возможности вставки вкладки из другого элемента управления.


## Комментарии


Допустимые значения:


	- true.
	 Вкладку можно вставить из другого элемента управления;


	- false.
	 По умолчанию. Вкладку нельзя вставить из другого элемента управления.


Значение свойства устанавливается из JSON и с помощью метода setEnableDropFromOutside,
 а возвращается с помощью метода getEnableDropFromOutside.
