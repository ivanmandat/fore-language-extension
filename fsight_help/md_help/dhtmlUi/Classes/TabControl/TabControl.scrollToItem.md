# TabControl.scrollToItem


## Синтаксис


scrollToItem(item: [PP.Ui.TabItem](../TabItem/TabItem.htm));


## Параметры


item. Вкладка элемента управления.


## Описание


Метод scrollToItem прокручивает
 панель вкладок до указанной вкладки.
