# TabControl.AddTabButtonTemplate


## Синтаксис


AddTabButtonTemplate: String;


## Описание


Свойство AddTabButtonTemplate
 определяет шаблон вкладки при добавлении её в панель.


## Комментарии


Значение свойства устанавливается из JSON, а возвращается с помощью
 метода getAddTabButtonTemplate.
