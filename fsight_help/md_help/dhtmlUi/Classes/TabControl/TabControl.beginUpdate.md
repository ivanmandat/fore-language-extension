# TabControl.beginUpdate


## Синтаксис


beginUpdate();


## Описание


Метод beginUpdate отключает
 перерисовку элемента управления.


## Комментарии


Для повторного включения перерисовки элемента управления используйте
 метод [TabControl.endUpdate](TabControl.endUpdate.htm).


## Пример


Пример использования метода приведен в описании метода [TabControl.removeItemByIndex](TabControl.removeItemByIndex.htm).
