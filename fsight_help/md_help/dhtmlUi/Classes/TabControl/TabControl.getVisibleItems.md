# TabControl.getVisibleItems


## Синтаксис


getVisibleItems();


## Описание


Метод getVisibleItems возвращает
 массив видимых вкладок панели элемента управления.


## Комментарии


Элементы массива являются объектами класса [TabItem](../TabItem/TabItem.htm).
