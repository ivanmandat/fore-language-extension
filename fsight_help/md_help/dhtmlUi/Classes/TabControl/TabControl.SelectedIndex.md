# TabControl.SelectedIndex


## Синтаксис


SelectedIndex: Number;


## Описание


Свойство SelectedIndex определяет
 выделенную вкладку по индексу.


## Комментарии


Одновременно в компоненте может быть выделена только одна вкладка. Выделить
 вкладку значит раскрыть её содержимое.


По умолчанию выбранной вкладкой является вкладка с индексом 0.


Также вкладку можно выделить, используя свойство [TabControl.SelectedItem](TabControl.SelectedItem.htm).
