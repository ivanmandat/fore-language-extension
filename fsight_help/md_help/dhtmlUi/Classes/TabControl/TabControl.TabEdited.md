# TabControl.TabEdited


## Синтаксис


TabEdited: function (sender, args);


## Параметры


sender. Источник
 события;


args. Информация о событии.


## Описание


Событие TabEdited наступает
 после выхода из режима редактирования.


## Комментарии


Допустимые значения параметра args:


	- TabIndex. Индекс редактируемой
	 вкладки;


	- Caption. Заголовок редактируемой
	 вкладки.


Для выхода из режима редактирования щёлкните мышкой по пространству
 вне редактируемого заголовка.
