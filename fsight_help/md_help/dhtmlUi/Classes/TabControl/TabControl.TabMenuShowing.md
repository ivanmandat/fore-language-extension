# TabControl.TabMenuShowing


## Синтаксис


TabMenuShowing: function (sender, args);


## Параметры


sender. Источник
 события;


args. Информация о событии.


## Описание


Событие TabMenuShowing наступает
 во время вызова меню у вкладки.


## Комментарии


Допустимые значения параметра args:


	- Menu. Вызванное меню,
	 объект класса PP.Ui.[Menu](../Menu/Menu.htm);


	- TabItem. Вкладка, у
	 которой вызвали меню, объект класса PP.Ui.[TabItem](../TabItem/TabItem.htm).
