# TabControl.ContentAutoFit


## Синтаксис


ContentAutoFit: Boolean;


## Описание


Свойство ContentAutoFit определяет
 признак автоматической подгонки содержимого элемента управления под родительский
 элемент.


## Комментарии


Допустимые значения:


	- true.
	 Содержимое элемента управления будет подогнано под родительский элемент;


	- false.
	 По умолчанию. Содержимое элемента управления не будет подогнано под
	 родительский элемент.


Значение свойства устанавливается из JSON и с помощью метода setContentAutoFit,
 а возвращается с помощью метода getContentAutoFit.
