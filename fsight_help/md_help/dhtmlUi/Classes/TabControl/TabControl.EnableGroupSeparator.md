# TabControl.EnableGroupSeparator


## Синтаксис


EnableGroupSeparator: Boolean;


## Описание


Свойство EnableGroupSeparator
 определяет признак наличия разделителя между группами вкладок.


## Комментарии


Допустимые значения:


	- true.
	 Между группами вкладок есть разделитель;


	- false.
	 По умолчанию. Между группами вкладок разделителя нет.


Значение свойства устанавливается из JSON и с помощью метода setEnableGroupSeparator,
 а возвращается с помощью метода getEnableGroupSeparator.


Свойство находится в разработке.
