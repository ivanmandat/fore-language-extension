# TabControlArgs.Cancel


## Синтаксис


TabControlArgs.Cancel = Boolean


## Описание


Свойство Cancel определяет,отменяется ли открытие вкладки.


## Комментарии


Если установлено значение true, открытие вкладки отменяется. По умолчанию установлено значение false.
