# TabControlArgs.Item


## Синтаксис


TabControlArgs.Item == PP.[TabItem](../TabItem/TabItem.htm)


## Описание


Свойство Item возвращает активную вкладку.


## Комментарии


Под активной вкладкой понимается вкладка, над которой производится какое-либо действие, например, добавление или перемещение.
