# Конструктор ToolBarSeparator


## Синтаксис


PP.Ui.ToolBarSeparator(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор ToolBarSeparator создает экземпляр компонента ToolBarSeparator.
