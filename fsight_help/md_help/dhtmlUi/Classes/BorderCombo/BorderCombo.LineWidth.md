# BorderCombo.LineWidth


## Синтаксис


LineWidth: Number


## Описание


Свойство LineWidth определяет значение в числовом редакторе на вкладке «Толщина».


## Комментарии


Вкладка «Толщина» отображается, если для свойства [BorderCombo.BorderWidthVisible](BorderCombo.BorderWidthVisible.htm) установлено значение true.


По умолчанию значение для свойства LineWidth не задано.
