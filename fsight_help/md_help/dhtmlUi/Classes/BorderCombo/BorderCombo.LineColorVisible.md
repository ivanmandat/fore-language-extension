# BorderCombo.LineColorVisible


## Синтаксис


LineColorVisible: Boolean


## Описание


Свойство LineColorVisible определяет, отображается ли вкладка для выбора цвета линии.


## Комментарии


Если установлено значение false, вкладка для выбора цвета линии не отображется. По умолчанию установлено значение true.


## Пример


Для выполнения примера предполагается наличие на странице компонента [BorderCombo](../../Components/BorderCombo/BorderCombo.htm) с наименованием «borderC» (см. «[Пример создания компонента BorderCombo](../../Components/BorderCombo/BorderCombo_example.htm)»). Скроем вкладку для выбора цвета линии:


borderC.setLineColorVisible(false);
