# BorderCombo.ValueChanged


## Синтаксис


ValueChanged: function(sender, args)


## Параметры


sender. Источник события;


args. Информация
 о событии. Доступные аргументы: PropertyName - имя измененного свойства, Value
 - новое значение свойства, Border - объект, у которого изменилось свойство.


## Описание


Событие ValueChanged наступает
 после изменения свойства границы.
