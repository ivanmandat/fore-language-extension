# Конструктор BorderCombo


## Синтаксис


BorderCombo (settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор BorderCombo создает экземпляр компонента [BorderCombo](../../Components/BorderCombo/BorderCombo.htm).
