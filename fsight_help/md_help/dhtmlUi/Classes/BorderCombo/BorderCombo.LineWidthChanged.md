# BorderCombo.LineWidthChanged


## Синтаксис


LineWidthChanged: function (sender, args)


## Параметры


sender. Источник события;


args. Информация о событии. Доступный аргумент LineWidth - возвращает текущее установленное значение толщины линии.


## Описание


Событие LineWidthChanged наступает при изменении толщины линии.


## Комментарии


Толщина линии изменяется в числовом редакторе на вкладке «Толщина», которая доступна, если установлено значение true для свойства [BorderCombo.BorderWidthVisible](BorderCombo.BorderWidthVisible.htm).
