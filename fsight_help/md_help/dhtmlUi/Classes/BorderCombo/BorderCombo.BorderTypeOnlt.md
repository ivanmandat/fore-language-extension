# BorderCombo.BorderTypeOnlt


## Синтаксис


BorderTypeOnlt: Boolean


## Описание


Свойство BorderTypeOnlt определяет,
 отображается ли в раскрывающейся части компонента панель для выбора варианта
 расположения границ.


## Комментарии


Если для свойства установлено значение true,
 то в раскрывающаейся панели отображается панель для выбора типа расположения
 границы. Если установлено значение false,
 то данная панель не отображается.


Панель располагается в верхней части компонента:


![](../../Components/BorderCombo/BorderCombo1.gif)


Данная панель отображается и в режиме выбора цвета, и в режиме выбора
 типа линии. При отображении данной панели на раскрывающейся части компонента
 появляются полосы прокрутки.
