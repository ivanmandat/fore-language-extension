# BorderCombo.TypesMode


## Синтаксис


TypesMode: [PP.Ui.BorderComboTypesMode](../../Enums/TypesMode.htm)


## Описание


Свойство TypesMode определяет
 технологию, в которой будут отображаться типы границ.


## Комментарии


Значения свойства - элементы перечисления [BorderComboTypesMode](../../Enums/TypesMode.htm).
 По умолчанию установлено значение «Services».
