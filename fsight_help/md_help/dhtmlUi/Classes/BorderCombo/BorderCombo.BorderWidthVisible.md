# BorderCombo.BorderWidthVisible


## Синтаксис


BorderWidthVisible: Boolean


## Описание


Свойство BorderWidthVisible
 определяет, отображается ли вкладка для настройки толщины линии.


## Комментарии


Если установлено значение true,
 то вкладка для настройки толщины линии отображается. По умолчанию установлено
 значение false.
