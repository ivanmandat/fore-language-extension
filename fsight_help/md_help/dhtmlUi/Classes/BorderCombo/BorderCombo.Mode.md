# BorderCombo.Mode


## Синтаксис


Mode: [PP.Ui.BorderComboMode](../../Enums/Mode.htm);


## Описание


Свойство Mode
 определяет активный режим для выбора типа границ.


## Комментарии


Значения свойства - элементы перечисления [BorderComboMode](../../Enums/Mode.htm).


Доступны режимы для выбора цвета и для
 выбора типа линии. По умолчанию активен режим для выбора цвета.
