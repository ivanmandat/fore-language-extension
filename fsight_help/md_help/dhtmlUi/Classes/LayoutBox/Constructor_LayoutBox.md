# Конструктор LayoutBox


## Синтаксис


PP.Ui.LayoutBox(settings);


## Параметры


settings. JSON-объект со значениями
 свойств компонента.


## Описание


Конструктор LayoutBox создает
 экземпляр компонента [LayoutBox](../../Components/LayoutBox/LayoutBox.htm).
