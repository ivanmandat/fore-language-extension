# LayoutBox.insert


## Синтаксис


insert(ctrl: PP.Ui.[Control](../Control/Control.htm),
 index: Number);


## Параметры


ctrl. Добавляемый элемент;


index. Позиция, куда добавляется
 элемент.


## Описание


Метод insert добавляет элемент
 в таблицу по индексу.
