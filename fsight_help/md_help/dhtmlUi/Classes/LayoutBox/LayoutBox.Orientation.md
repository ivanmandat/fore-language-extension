# LayoutBox.Orientation


## Синтаксис


Orientation: PP.[Orientation](dhtmlCommon.chm::/enums/pp.orientation.htm)


## Описание


Свойство Orientation определяет
 ориентацию таблицы.


## Комментарии


Значением свойства являются элементы перечисления [Orientation](dhtmlCommon.chm::/enums/pp.orientation.htm).


Значение свойства устанавливается из JSON и с помощью метода setOrientation,
 а возвращается с помощью метода getOrientation.


По умолчанию установлено значение PP.[Orientation](dhtmlCommon.chm::/Enums/PP.Orientation.htm).Horizontal,
 то есть элементы таблицы располагаются горизонтально.
