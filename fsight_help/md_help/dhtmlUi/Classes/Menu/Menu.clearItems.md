# Menu.clearItems


## Синтаксис


clearItems(withDispose: Boolean);


## Параметры


withDispose. Признак полного
 уничтожения элементов. Если значение параметра равно true,
 то элементы будут полностью удалены, иначе элементы просто будут удалены
 из меню.


## Описание


Метод clearItems удаляет дочерние
 элементы меню.


## Пример


Для выполнения примера предполагается наличие на html-странице компонента
 [Menu](../../Components/Menu/Menu.htm) с наименованием «menu»
 (см. «[Пример создания
 компонента Menu](../../Components/Menu/Menu_example.htm)»). Чтобы удалить все дочерние элементы меню используйте
 метод clearItems:


menu.clearItems()


После выполнения примера будут удалены все дочерние элементы меню.
