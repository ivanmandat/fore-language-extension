# Menu.addItem


## Синтаксис


addItem (value);


## Параметры


value. Дочерний элемент меню: текст, html-разметка, DOM-узел или наследник класса [Control](../Control/Control.htm), в частности, экземпляр класса [MenuItem](../MenuItem/MenuItem.htm).


## Описание


Метод addItem добавляет дочерний элемент в [меню](../../Components/Menu/Menu.htm).


## Комментарии


Метод используется при статическом создании компонента. При помощи этого метода в меню также могут быть добавлены [заголовок](../MenuHeader/MenuHeader.htm) и [разделитель](../Separator/Separator.htm).
