# Конструктор Menu


## Синтаксис


PP.Ui.Menu (settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор Menu создает экземпляр компонента [Menu](../../Components/Menu/Menu.htm).
