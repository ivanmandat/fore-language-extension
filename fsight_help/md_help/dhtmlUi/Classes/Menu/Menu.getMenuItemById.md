# Menu.getMenuItemById


## Синтаксис


getMenuItemById (value, notRecursive);


## Параметры


value. Идентификатор;


notRecursive. Определять, выполняется
 ли поиск только в дочерних пунктах меню. Если установлено значение false, поиск выполняется и в подменю.
 Если установлено значение true,
 элемент ищется только среди дочерних пунктов меню. По умолчанию установлено
 значение true.


## Описание


Метод getMenuItemById возвращает
 дочерний элемент меню по его идентификатору.


## Комментарии


Идентификатор элемента задается при помощи свойства [Object.Id](dhtmlCommon.chm::/Classes/object/object.id.htm).


## Пример


Для выполнения примера предполагается наличие на html-странице компонента
 [Menu](../../Components/Menu/Menu.htm) с наименованием «menu1»
 (см. описание свойства [ImageList](Menu.ImageList.htm)), для
 одного из дочерних элементов которого установлен идентификатор «itemWithHotKey».
 С помощью приведенного ниже кода на страницу выдается сообщение с наименованием
 «горячих клавиш», установленных для элемента меню с заданным индексом.


alert (menu1.getMenuItemById("itemWithHotKey").getHotKey());


После выполнения примера на страницу будет выведено сообщение «Ctrl+N».
