# Menu.showContext


## Синтаксис


showContext (left, top, notCreateFrame)


## Параметры


left. Координата по оси X;


top. Координата по оси Y;


notCreateFrame. Определяет, будет ли создаваться подложка под меню. Если установлено значение true (по умолчанию), то подложка создаваться не будет, то есть при снятии фокуса с меню компонент будет скрываться.


## Описание


Метод showContext отображает меню как контекстное.


## Комментарии


При использовании метода showContext меню может раскрываться вверх/влево, если оно не влезает в видимую область браузера после раскрытия.
