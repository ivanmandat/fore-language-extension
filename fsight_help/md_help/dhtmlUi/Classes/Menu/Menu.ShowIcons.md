# Menu.ShowIcons


## Синтаксис


ShowIcons: Boolean


## Описание


Свойство ShowIcons определяет, отображается ли левая часть меню для пиктограмм.


## Комментарии


Если установлено значение false, левая часть меню для отображения пиктограмм скрыта. По умолчанию установлено значение true.


Определить, отображается ли пиктограмма для отдельного пункта меню можно при помощи метода [MenuItem.setShowIcon](../MenuItem/MenuItem.setShowIcon.htm).
