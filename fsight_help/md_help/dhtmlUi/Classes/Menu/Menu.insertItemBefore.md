# Menu.insertItemBefore


## Синтаксис


insertItemAfter (item: Object, beforeItem: PP.Ui.[MenuItem](../MenuItem/MenuItem.htm))


## Параметры


item. Пункт меню, который необходимо вставить;


beforeItem. Пункт меню, перед которым необходимо вставить новый пункт.


## Описание


Метод insertItemBefore вставляет пункт в меню перед указанным пунктом.
