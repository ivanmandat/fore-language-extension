# Menu.ScrollHeight


## Синтаксис


ScrollHeight: Number


## Описание


Свойство ScrollHeight определяет высоту прокручиваемого содержимого меню.


## Комментарии


По умолчанию для свойства установлено значение 100.


Высота содержимого меню не изменяется при изменении [высоты](../Control/Control.Height.htm) видимой части компонента.
