# Menu.Items


## Синтаксис


Items: Array


## Описание


Свойство Items устанавливает
 массив дочерних элементов меню.


## Комментарии


Используется при статическом размещении компонента, задается из JSON.


Значение свойства возвращается при помощи метода getItems.
