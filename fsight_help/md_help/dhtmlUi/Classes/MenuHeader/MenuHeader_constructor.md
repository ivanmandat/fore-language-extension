# Конструктор MenuHeader


## Синтаксис


PP.Ui.MenuHeader (settings)


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор MenuHeader создает экземпляр класса [MenuHeader](MenuHeader.htm).


## Пример


Пример использования конструктора при динамическом размещении компонента [Menu](../../Components/Menu/Menu.htm) приведен в примере для свойства [Menu.ImageList](../Menu/Menu.ImageList.htm), при статическом - на странице «[Пример создания компонента Menu](../../Components/Menu/Menu_example.htm)».
