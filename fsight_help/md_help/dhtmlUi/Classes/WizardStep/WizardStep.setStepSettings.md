# WizardStep.setStepSettings


## Синтаксис


setStepSettings (value: Object);


## Параметры


value. Настройки шага мастера.


## Описание


Метод setStepSettings устанавливает ссылку на настройки [мастера](../Wizard/Wizard.htm).


## Комментарии


Ссылка на эти настройки передается классом, реализующим абстрактный класс PP.Ui.Wizard при инстанцировании списка шагов мастера.
