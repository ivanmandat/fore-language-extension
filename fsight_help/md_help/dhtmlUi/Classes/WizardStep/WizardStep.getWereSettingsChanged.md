# WizardStep.getWereSettingsChanged


## Синтаксис


getWereSettingsChanged ();


## Описание


Метод getWereSettingsChanged определяет, были ли изменены настройки на шаге мастера.


## Комментарии


Если возвращает значение true, значит были изменения, false - не было.
