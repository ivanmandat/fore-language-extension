# WizardStep.accomplish


## Синтаксис


accomplish ()


## Описание


Реализация метода accomplish может выполнять дополнительные действия по инициализации настроек, которые хранятся по ссылке PP.Ui.WizardStep._StepSettings.


## Комментарии


Вызывается контейнерным классом (потомком PP.Ui.[Wizard](../Wizard/Wizard.htm)) по нажатию на кнопку «Далее».
