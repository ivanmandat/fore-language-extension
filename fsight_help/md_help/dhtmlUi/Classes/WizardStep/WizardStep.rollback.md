# WizardStep.rollback


## Синтаксис


rollback ();


## Описание


Реализация метода rollback может откатывать изменения, внесенные в рамках шага.


## Комментарии


Вызывается контейнерным классом (потомком PP.Ui.[Wizard](../Wizard/Wizard.htm)) по нажатию на кнопку «Назад».
