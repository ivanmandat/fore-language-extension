# WizardStep.getIsRequired


## Синтаксис


getIsRequired ();


## Описание


Метод getIsRequired возвращает признак того, является ли данный шаг обязательным для выполнения.


## Комментарии


Если шаг является обязательным для выполнения, возвращает true, иначе - false.
