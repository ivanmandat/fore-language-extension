# FontDialog.Font


## Синтаксис


Font: PP.[Font](dhtmlCommon.chm::/Classes/pp/font/font.htm)


## Описание


Свойство Font устанавливает
 текущие настройки шрифта.


## Комментарии


По умолчанию настроен шрифт Calibri, размер 12, цвет черный, дополнительные
 настройки не заданы.
