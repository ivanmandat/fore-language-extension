# Конструктор FontDialog


## Синтаксис


PP.Ui.FontDialog (settings);


## Параметры


settings. JSON объект со значениями свойств компонента.


## Описание


Конструктор FontDialog создает экземпляр компонента [FontDialog](../../Components/FontDialog/FontDialog.htm).
