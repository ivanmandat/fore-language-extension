# FontDialog.SampleTextString


## Синтаксис


SampleTextString: String;


## Описание


Свойство SampleTextString устанавливает текст строки, используемой для примера настройки шрифта.


## Комментарии


По умолчанию в качестве текста примера используется название выбранного шрифта.
