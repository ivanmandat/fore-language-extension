# Breadcrumb.getItems


## Синтаксис


getItems ();


## Описание


Метод getItems возвращает коллекцию элементов компонента.


## Комментарии


Коллекция элементов компонента устанавливается при помощи свойства [Items](Breadcrumb.Items.htm).

Пример использования метода приведен в описании события [FilterClick](../BreadcrumbItem/BreadcrumbItem.FilterClick.htm).
