# Breadcrumb.SelectedItems


## Синтаксис


SelectedItems: PP.Ui.[BreadcrumbSelectedItemsList](../BreadcrumbSelectedItemsList/BreadcrumbSelectedItemsList.htm)


## Описание


Свойство SelectedItems устанавливает
 коллекцию элементов, выбранных в компоненте.


## Комментарии


Коллекция элементов, выбранных в компоненте, возвращается при помощи
 метода [getSelectedItems](Breadcrumb.getSelectedItems.htm).
