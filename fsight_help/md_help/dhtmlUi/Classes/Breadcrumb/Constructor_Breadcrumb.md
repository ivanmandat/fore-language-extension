# Конструктор Breadcrumb


## Синтаксис


PP.Ui.Breadcrumb(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор Breadcrumb создает экземпляр компонента [Breadcrumb](../../Components/Breadcrumb/Breadcrumb.htm).
