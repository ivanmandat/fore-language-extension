# Breadcrumb.Items


## Синтаксис


Items: PP.Ui.[BreadcrumbItemsList](../BreadcrumbItemsList/BreadcrumbItemsList.htm)


## Описание


Свойство Items устанавливает коллекцию дочерних элементов компонента [Breadcrumb](../../Components/Breadcrumb/Breadcrumb.htm).


## Комментарии


Коллекция дочерних элементов компонента [Breadcrumb](../../Components/Breadcrumb/Breadcrumb.htm) возвращается при помощи метода [getItems](Breadcrumb.getItems.htm).
