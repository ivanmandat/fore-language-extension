# Breadcrumb.getSelectedItems


## Синтаксис


getSelectedItems ();


## Описание


Метод getSelectedItems возвращает коллекцию элементов, выбранных в компоненте.


## Комментарии


Коллекция элементов, выбранных в компоненте, устанавливается при помощи свойства [SelectedItems](Breadcrumb.SelectedItems.htm).
