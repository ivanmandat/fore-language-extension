# TextArea.ValueChanged


## Синтаксис


ValueChanged: function(sender, args)


## Параметры


sender. Источник события;


args. Информация о событии.
 Доступные аргументы: Value - значение
 текстовой области; OldValue -
 значение текстовой области до изменения; OldContent
 - содержимое текстовой области до изменения; NewContent
 - содержимое текстовой области после изменения; NewValue
 - значение текстовой области после изменения.


## Описание


Событие ValueChanged наступает
 после изменения значения [текстовой области](TextArea.htm).
