# TextArea.TextSelectable


## Синтаксис


TextSelectable: Boolean


## Описание


Свойство TextSelectable определяет
 возможность выделения текста [текстовой области](TextArea.htm).


## Комментарии


Допустимые значения:


	- true.
	 Возможно выделение текста в текстовой области (по умолчанию);


	- false.
	 Выделение текста в текстовой области не возможно.


Значение свойства устанавливается из JSON и с помощью метода setTextSelectable,
 а возвращается с помощью метода getTextSelectable.
