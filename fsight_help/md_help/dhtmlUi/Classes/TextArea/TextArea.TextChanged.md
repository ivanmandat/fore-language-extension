# TextArea.TextChanged


## Синтаксис


TextChanged: function(sender, args)


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: Value - новое значение текстового содержимого компонента.


## Описание


Событие TextChanged** наступает при потере фокуса компонентом при измененном текстовом содержимом.
