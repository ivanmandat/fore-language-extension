# TextArea.ReadOnly


## Синтаксис


ReadOnly: Boolean


## Описание


Свойство ReadOnly** определяет, доступен ли компонент для редактирования.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setReadOnly, а возвращается с помощью метода getReadOnly.
