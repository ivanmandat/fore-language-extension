# Animation.isExecute


## Синтаксис


isExecute();


## Описание


Метод isExecute** определяет, воспроизводится ли анимация в данный момент.


## Комментарии


Метод возвращает значение true, если в данный момент воспроизводится анимация, и значение false в противном случае.
