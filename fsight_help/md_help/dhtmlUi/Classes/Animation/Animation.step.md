# Animation.step


## Синтаксис


step(time: Number);


## Параметры


time. Время воспроизведения анимации в миллисекундах, по истечении которого нужно определить значение шага анимации.


## Описание


Метод step** возвращает значение шага анимации по истечении указанного времени.
