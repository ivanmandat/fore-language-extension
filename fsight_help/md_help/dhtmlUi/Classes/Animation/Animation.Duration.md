# Animation.Duration


## Синтаксис


Duration: Number


## Описание


Свойство Duration** устанавливает и возвращает длительность воспроизведения анимации в миллисекундах.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDuration, а возвращается - с помощью метода getDuration.
