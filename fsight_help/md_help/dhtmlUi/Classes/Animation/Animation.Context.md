# Animation.Context


## Синтаксис


Context: [PP.Object](dhtmlCommon.chm::/Classes/object/object.htm)


## Описание


Свойство Context** устанавливает и возвращает объект, свойство которого будет меняться при анимации.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setContext, а возвращается - с помощью метода getContext.
