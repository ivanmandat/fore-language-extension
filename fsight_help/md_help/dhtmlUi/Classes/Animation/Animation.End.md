# Animation.End


## Синтаксис


End: Number


## Описание


Свойство End** устанавливает и возвращает индекс последнего шага анимации.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setEnd, а возвращается - с помощью метода getEnd.
