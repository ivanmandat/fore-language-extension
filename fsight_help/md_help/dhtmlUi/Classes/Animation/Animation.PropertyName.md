# Animation.PropertyName


## Синтаксис


PropertyName: String


## Описание


Свойство PropertyName** устанавливает и возвращает наименование свойства, которое будет анимироваться.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setPropertyName, а возвращается - с помощью метода getPropertyName.
