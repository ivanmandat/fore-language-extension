# Animation.Start


## Синтаксис


Start: Number


## Описание


Свойство Start** устанавливает и возвращает индекс начального шага анимации.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setStart, а возвращается - с помощью метода getStart.
