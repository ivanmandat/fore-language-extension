# NumberEditValidateArgs.IsValid


## Синтаксис


args.IsValid = Boolean;


## Описание


Свойство IsValid определяет, проходит ли значение проверку.


## Комментарии


Если установлено значение false, то значение не проходит проверку, если true, то проходит. По умолчанию установлено значение false.
