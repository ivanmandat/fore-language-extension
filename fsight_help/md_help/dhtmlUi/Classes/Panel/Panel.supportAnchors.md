# Panel.supportAnchors


## Синтаксис


supportAnchors();


## Описание


Метод supportAnchors возвращает
 признак использования [Anchors](../Control/Control.Anchors.htm)
 в панели.


## Комментарии


Метод возвращает значение true, если в панели возможно использование
 [Anchors](../Control/Control.Anchors.htm), иначе false.


## Пример


Пример использования метода приведен в описании метода [Panel.toFront](Panel.toFront.htm).
