# Panel.getItemByPoint


## Синтаксис


getItemByPoint(x: Number, y: Number, excludeItems:
 Array || PP.Ui.[Control](../Control/Control.htm));


## Параметры


x. Координата по горизонтали;


y. Координата по вертикали;


excludeItems. Элементы, которые
 необходимо исключить из поиска.


## Описание


Метод getItemByPoint возвращает
 элемент панели по координатам.


## Комментарии


Возвращаемое значение - наследник класса PP.Ui.[Control](../Control/Control.htm).
