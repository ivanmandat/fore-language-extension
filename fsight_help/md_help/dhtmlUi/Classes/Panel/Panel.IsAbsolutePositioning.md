# Panel.IsAbsolutePositioning


## Синтаксис


IsAbsolutePositioning: Boolean


## Описание


Свойство IsAbsolutePositioning
 определяет позиционирование элементов панели.


## Комментарии


Допустимые значения:


	- true.
	 Автоматическое позиционирование элементов (по умолчанию);


	- false.
	 Пользовательское позиционирование элементов.


Значение свойства устанавливается из JSON и с помощью метода setIsAbsolutePositioning,
 а возвращается с помощью метода getIsAbsolutePositioning.
