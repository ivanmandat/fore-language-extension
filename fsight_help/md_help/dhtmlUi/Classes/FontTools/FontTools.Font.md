# FontTools.Font


## Синтаксис


Font: PP.[Font](dhtmlCommon.chm::/Classes/PP/Font/Font.htm);


## Описание


Свойство Font определяет настройки
 текста.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setFont,
 а возвращается с помощью метода getFont.


Метод setFont является аналогом
 метода [FontTools.init](FontTools.init.htm).
