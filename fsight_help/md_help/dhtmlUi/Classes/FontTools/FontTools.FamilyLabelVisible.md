# FontTools.FamilyLabelVisible


## Синтаксис


FamilyLabelVisible: Boolean;


## Описание


Свойство FamilyLabelVisible
 определяет отображение надписи «Шрифт» элемента выбора шрифта.


## Комментарии


Значение свойства устанавливается из JSON.
