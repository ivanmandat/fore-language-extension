# FontTools.FamilyVisible


## Синтаксис


FamilyVisible: Boolean;


## Описание


Свойство FamilyVisible определяет
 отображение элемента выбора шрифта.


## Комментарии


Значение свойства устанавливается из JSON, а возвращается с помощью
 метода getFont.
