# WebGLControl.getMesh


## Синтаксис


getMesh(id_index: String || Number);


## Параметры


id_index. Индекс или идентификатор
 поверхности.


## Описание


Метод getMesh возвращает поверхность
 многогранного объекта по индексу или идентификатору.
