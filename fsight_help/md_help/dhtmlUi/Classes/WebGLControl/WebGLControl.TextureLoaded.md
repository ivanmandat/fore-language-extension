# WebGLControl.TextureLoaded


## Синтаксис


TextureLoaded: function(sender, args);


## Параметры


sender. Источник события;


args. Информация
 о событии. Аргументы: Event
 - информация о событии; Element - загруженное изображение; Break
 - признак отмены события (если установлено значение true,
 то событие отменено, иначе - разрешено).


## Описание


Событие TextureLoaded наступает
 после загрузки текстуры.
