# WebGLControl.setDirectLight


## Синтаксис


setDirectLight(value: PP.[Color](dhtmlCommon.chm::/Classes/PP/Color/Color.htm)
 || String);


## Параметры


value. Цвет устанавливаемого
 света.


## Описание


Метод setDirectLight устанавливает
 цвет прямого света.


## Комментарии


Прямой свет - поток параллельных лучей в направлении свечения.
