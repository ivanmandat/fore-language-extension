# TextEditor.BackgroundColor


## Синтаксис


BackgroundColor: [PP.Color](dhtmlCommon.chm::/Classes/PP/Color/Color.htm);


## Описание


Свойство BackgroundColor определяет
 цвет фона выделенного текста.


## Комментарии


Значение свойства устанавливается при помощи метода setBackgroundColor,
 а возвращается при помощи метода getBackgroundColor.
 Из JSON задать значение свойства нельзя.


Метод setBackgroundColor принимает
 следующие параметры: value - цвет
 фона выделенного текста, updateControl
 - признак обновления компонента (необязательный параметр, по умолчанию
 содержит значение true).
