# TextEditor.IsUnderline


## Синтаксис


IsUnderline: Boolean;


## Описание


Свойство IsUnderline определяет, применяется ли подчеркивание к выделенному тексту.


## Комментарии


Значение свойства устанавливается при помощи метода setIsUnderline, а возвращается при помощи метода getIsUnderline. Из JSON задать значение свойства нельзя.


Метод IsUnderline принимает следующие параметры: value - признак использования подчеркивания, updateControl - признак обновления компонента (необязательный параметр, по умолчанию содержит значение true).


Если свойство имеет значение true, то используется подчеркивание текста, иначе - не используется.
