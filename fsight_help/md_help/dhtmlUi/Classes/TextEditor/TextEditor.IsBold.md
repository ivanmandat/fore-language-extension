# TextEditor.IsBold


## Синтаксис


IsBold: Boolean;


## Описание


Свойство IsBold определяет, используется ли полужирное начертание.


## Комментарии


Значение свойства устанавливается при помощи метода setIsBold, а возвращается при помощи метода getIsBold. Из JSON задать значение свойства нельзя.


Метод setIsBold принимает следующие параметры: value - признак использования полужирного начертания, updateControl - признак обновления компонента (необязательный параметр, по умолчанию содержит значение true).


Если свойство имеет значение true, то используется полужирное начертание, иначе - не используется.
