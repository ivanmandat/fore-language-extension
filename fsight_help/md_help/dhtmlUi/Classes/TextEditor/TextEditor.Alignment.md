# TextEditor.Alignment


## Синтаксис


Alignment: [PP.HorizontalAlignment](dhtmlCommon.chm::/Enums/PP.HorizontalAlignment.htm);


## Описание


Свойство Alignment определяет
 горизонтальное выравнивание текста.


## Комментарии


Значение свойства устанавливается при помощи метода setAlignment,
 возвращается при помощи метода getAlignment.
 Из JSON задать значение свойства нельзя.


Метод setAlignment принимает
 следующие параметры: value - значение
 выравнивания, updateControl -
 признак обновления компонента (необязательный параметр, по умолчанию содержит
 значение true).
