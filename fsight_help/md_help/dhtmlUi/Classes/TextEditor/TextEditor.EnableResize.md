# TextEditor.EnableResize


## Синтаксис


EnableResize: Boolean;


## Описание


Свойство EnableResize определяет возможность изменять размеры текстовой области.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setEnableResize, а возвращается с помощью метода getEnableResize.


Если для свойства установлено значение true, то размер текстовой области можно изменить, иначе - нельзя.


Изменение размера происходит при перемещении правого нижнего угла текстовой области редактора.
