# TextEditor.FontSize


## Синтаксис


FontSize: Number;


## Описание


Свойство FontSize определяет размер шрифта выделенного текста.


## Комментарии


Значение свойства устанавливается при помощи метода setFontSize, а возвращается при помощи метода getFontSize. Из JSON задать значение свойства нельзя.


Метод setFontSize принимает следующие параметры: value - размер шрифта выделенного текста, updateControl - признак обновления компонента (необязательный параметр, по умолчанию содержит значение true).
