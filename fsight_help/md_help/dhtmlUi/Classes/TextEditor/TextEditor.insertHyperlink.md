# TextEditor.insertHyperlink


## Синтаксис


insertHyperlink(url: String, target: [PP.Ui.TextEditorHyperlinkTarget](../../Enums/TextEditorHyperlinkTarget.htm));


## Параметры


url. Адрес гиперссылки;


target. Метод открытия ссылки.


## Описание


Метод insertHyperlink создает
 гиперссылку в выделенном тексте.
