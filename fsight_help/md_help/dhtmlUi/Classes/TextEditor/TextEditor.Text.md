# TextEditor.Text


## Синтаксис


Text: String;


## Описание


Свойство Text определяет текст, который отображается в текстовой области редактора.


## Комментарии


Значение свойства устанавливается при помощи метода setText, а возвращается при помощи метода getText. Из JSON задать значение свойства нельзя.
