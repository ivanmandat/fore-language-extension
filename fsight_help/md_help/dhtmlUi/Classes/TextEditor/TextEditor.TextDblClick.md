# TextEditor.TextDblClick


## Синтаксис


TextDblClick: function(sender, args, timeout);


## Параметры


sender. Источник события;


args. Информация о событии;


timeout. Промежуток времени в миллисекундах, через который вызовется событие.


## Описание


Событие TextDblClick наступает при двойном щелчке по содержимому текстовой области редактора.
