# TextEditor.FontColor


## Синтаксис


FontColor: [PP.Color](dhtmlCommon.chm::/Classes/PP/Color/Color.htm);


## Описание


Свойство FontColor определяет
 цвет шрифта выделенного текста.


## Комментарии


Значение свойства устанавливается при помощи метода setFontColor,
 а возвращается при помощи метода getFontColor.
 Из JSON задать значение свойства нельзя.


Метод setFontColor принимает
 следующие параметры: value - цвет
 шрифта выделенного текста, updateControl
 - признак обновления компонента (необязательный параметр, по умолчанию
 содержит значение true).
