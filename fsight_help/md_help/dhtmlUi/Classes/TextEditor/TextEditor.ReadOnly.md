# TextEditor.ReadOnly


## Синтаксис


ReadOnly: Boolean;


## Описание


Свойство ReadOnly определяет
 признак доступности редактирования текста.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setReadOnly,
 а возвращается с помощью метода getReadOnly.


Если для свойства установлено значение true,
 то текст доступен только для чтения, иначе - для чтения и редактирования.
