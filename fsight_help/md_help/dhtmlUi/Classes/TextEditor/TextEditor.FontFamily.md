# TextEditor.FontFamily


## Синтаксис


FontFamily: [PP.Font.Family](dhtmlCommon.chm::/Enums/Font.Family.htm);


## Описание


Свойство FontFamily определяет
 семейство шрифта выделенного текста.


## Комментарии


Значение свойства устанавливается при помощи метода setFontFamily,
 а возвращается при помощи метода getFontFamily.
 Из JSON задать значение свойства нельзя.


Метод setFontFamily принимает
 следующие параметры: value - семейство
 шрифта выделенного текста, updateControl
 - признак обновления компонента (необязательный параметр, по умолчанию
 содержит значение true).
