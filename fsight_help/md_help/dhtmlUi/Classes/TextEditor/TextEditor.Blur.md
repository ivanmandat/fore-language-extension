# TextEditor.Blur


## Синтаксис


Blur: function(sender, args, timeout);


## Параметры


sender. Источник события;


args. Информация о событии;


timeout. Промежуток времени в миллисекундах, через который вызовется событие.


## Описание


Событие Blur наступает при потере фокуса элемента, входящего в состав компонента [TextEditor](../../Components/TextEditor/TextEditor.htm).
