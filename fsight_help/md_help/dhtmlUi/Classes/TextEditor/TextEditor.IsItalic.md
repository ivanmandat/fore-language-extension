# TextEditor.IsItalic


## Синтаксис


IsItalic: Boolean;


## Описание


Свойство IsItalic определяет, используется ли курсивное начертание.


## Комментарии


Значение свойства устанавливается при помощи метода setIsItalic, а возвращается при помощи метода getIsItalic. Из JSON задать значение свойства нельзя.


Метод IsItalic принимает следующие параметры: value - признак использования курсивного начертания, updateControl - признак обновления компонента (необязательный параметр, по умолчанию содержит значение true).


Если свойство имеет значение true, то используется курсивное начертание, иначе - не используется.
