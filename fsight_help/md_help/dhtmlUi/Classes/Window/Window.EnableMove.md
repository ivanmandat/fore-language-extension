# Window.EnableMove


## Синтаксис


EnableMove: Boolean


## Описание


Свойство EnableMove определяет, можно ли перемещать окно.


## Комментарии


Если установлено значение false, значит окно нельзя перемещать. По умолчанию установлено значение true.
