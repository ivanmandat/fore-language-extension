# Autocomplete.showSuggestion


## Синтаксис


showSuggestion (target)


## Параметры


target. Идентификатор или экземпляр элемента управления, подключенного к системе автозавершения.


## Описание


Метод showSuggestion отображает меню с подсказками, настроенными для указанного элемента управления.
