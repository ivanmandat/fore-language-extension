# Autocomplete.Active


## Синтаксис


Active: Boolean


## Описание


Свойство Active определяет, является ли автозавершение активным для указанного элемента управления.


## Комментарии


Если установлено значение true, то автозавершение применяется для указанного элемента управления. По умолчанию установлено значение false, то есть автозавершение не применяется.
