# Autocomplete.MaxHeight


## Синтаксис


getMaxHeight (target);


setMaxHeight (target, value);


## Параметры


target. Элемент управления,
 для которого нужно вернуть/установить максимальную высоту меню с вариантами
 автозавершения.


value. Значение максимальной
 высоты меню с вариантами автозавершения для указанного элемента.


## Описание


Свойство MaxHeight устанавливает
 и возвращает максимальную высоту меню с вариантами автозавершения для
 указанного элемента управления.


## Комментарии


Значение свойства устанавливается при помощи метода setMaxHeight
 и возвращается при помощи метода getMaxHeight.
 Установить значение свойства из JSON нельзя.


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [Autocomplete](../../Components/Autocomplete/Autocomplete.htm)
 с наименованием «autocomplete» (см. «[Конструктор
 Autocomplete](Constructor_Autocomplete.htm)»). Установим для текстового редактора с идентификатором
 «TB1» максимальную высоту и ширину всплывающей подсказки:


autocomplete.setMaxHeight("TB1", 300);
autocomplete.setMaxWidth("TB1", 300);
После выполнения примера для компонента TextBox с идентификатором «TB1»
 были установлены максимальная высота и ширина всплывающей подсказки, равные
 300 пикселям.
