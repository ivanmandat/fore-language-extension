# Autocomplete.hideSuggestion


## Синтаксис


hideSuggestion (target)


## Параметры


target. Идентификатор или экземпляр элемента управления, подключенного к системе автозавершения.


## Описание


Метод hideSuggestion скрывает меню, в котором отображаются подсказки.


## Комментарии


Метод вызывается автоматически при выборе подсказки.
