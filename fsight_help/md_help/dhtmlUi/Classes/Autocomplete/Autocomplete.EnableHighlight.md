# Autocomplete.EnableHighlight


## Синтаксис


EnableHighlight: Boolean


## Описание


Свойство EnableHighlight управляет
 возможностью подсветки совпадающих символов в подсказке для автозавершения
 ввода текста.


## Комментарии


Допустимые значения:


	- true. Подсветка совпадающих
	 символов в всплывающей подсказке включена;


	- false. Подсветка совпадающих
	 символов в всплывающей подсказке выключена.
