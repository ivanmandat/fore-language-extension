# Autocomplete.MinAutoFindSymbolsCount


## Синтаксис


getMinAutoFindSymbolsCount();


setMinAutoFindSymbolsCount(value: Number);


## Параметры


value. Минимальное количество символов, по которым будут искаться варианты для автозаполнения.


## Описание


Свойство MinAutoFindSymbolsCount устанавливает минимальную длину строки, для которой будет показана подсказка.


## Комментарии


Значение свойства устанавливается при помощи метода setMinAutoFindSymbolsCount и возвращается при помощи метода getMinAutoFindSymbolsCount. Задать значение свойства из JSON нельзя.


По умолчанию минимальная длина строки, для которой будут отображаться подсказки, - 3 символа.
