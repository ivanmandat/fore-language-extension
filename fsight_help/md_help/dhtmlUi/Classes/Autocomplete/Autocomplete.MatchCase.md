# Autocomplete.MatchCase


## Синтаксис


MatchCase: Boolean


## Описание


Свойство MatchCase определяет, нужно ли при поиске вариантов автозавершения учитывать регистр.


## Комментарии


Если для свойства установлено значение true, то при поиске вариантов автозаврешения будет учитываться регистр. Если установлено false, поиск варианта автозавершения осуществляется без учета регистра.
