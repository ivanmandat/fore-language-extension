# Autocomplete.MaxWidth


## Синтаксис


getMaxWidth (target);


setMaxWidth (target, value);


## Параметры


target. Элемент управления,
 для которого нужно вернуть/установить максимальную ширину меню с вариантами
 автозавершения.


value. Значение максимальной ширины меню с вариантами автозавершения
 для указанного элемента.


## Описание


Метод MaxWidth устанавливает
 и возвращает максимальную ширину меню с вариантами автозавершения для
 указанного элемента управления.


## Комментарии


Значение свойства устанавливается при помощи метода setMaxHeight
 и возвращается при помощи метода getMaxHeight.
 Установить значение свойства из JSON нельзя.
