# Spoiler.MaxPanelSize


## Синтаксис


MaxPanelSize: Number;


## Описание


Свойство MaxPanelSize определяет
 максимальный размер панели.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMaxPanelSize,
 а возвращается с помощью метода getMaxPanelSize.
