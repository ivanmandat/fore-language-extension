# Spoiler.IsExpanded


## Синтаксис


IsExpanded: Boolean;


## Описание


Свойство IsExpanded определяет
 состояние панели.


## Комментарии


Допустимые значения:


	- true.
	 Панель развернута (по умолчанию);


	- false.
	 Панель свернута.


Значение свойства устанавливается из JSON и с помощью метода setIsExpanded,
 а возвращается с помощью метода getIsExpanded.
