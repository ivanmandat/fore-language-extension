# Spoiler.EnableCollapse


## Синтаксис


EnableCollapse: Boolean;


## Описание


Свойство EnableCollapse определяет,
 можно ли свернуть панель.


## Комментарии


Допустимые значения:


	- true.
	 Панель свернуть можно (по умолчанию);


	- false.
	 Панель свернуть нельзя.


Значение свойства устанавливается из JSON и с помощью метода setEnableCollapse,
 а возвращается с помощью метода getEnableCollapse.
