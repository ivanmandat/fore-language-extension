# Spoiler.DefaultContentMargin


## Синтаксис


DefaultContentMargin: Number;


## Описание


Свойство DefaultContentMargin
 определяет отступ содержимого от края панели.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDefaultContentMargin,
 а возвращается с помощью метода getDefaultContentMargin.
