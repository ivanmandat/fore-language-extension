# Spoiler.EnableResize


## Синтаксис


EnableResize: Boolean;


## Описание


Свойство EnableResize определяет,
 можно ли изменить размер панели.


## Комментарии


Допустимые значения:


	- true.
	 Можно изменить размер панели (по умолчанию);


	- false.
	 Размер панели изменить нельзя.


Значение свойства устанавливается из JSON и с помощью метода setEnableResize,
 а возвращается с помощью метода getEnableResize.
