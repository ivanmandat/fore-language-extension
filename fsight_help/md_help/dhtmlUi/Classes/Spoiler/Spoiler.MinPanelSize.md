# Spoiler.MinPanelSize


## Синтаксис


MinPanelSize: Number;


## Описание


Свойство MinPanelSize определяет
 минимальный размер панели.


## Комментарии


При изменении размера панели, после достижения данного значения панель
 будет свернута, если для свойства [Spoiler.EnableCollapse](Spoiler.EnableCollapse.htm)
 установлено значение true.


Значение свойства устанавливается из JSON и с помощью метода setMinPanelSize,
 а возвращается с помощью метода getMinPanelSize.
