# Spoiler.DefaultResizerSize


## Синтаксис


DefaultResizerSize: Number;


## Описание


Свойство DefaultResizerSize
 определяет минимальный отступ содержимого от края панели.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDefaultResizerSize,
 а возвращается с помощью метода getDefaultResizerSize.
