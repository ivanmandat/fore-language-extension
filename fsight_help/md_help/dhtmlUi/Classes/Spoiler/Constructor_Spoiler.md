# Конструктор Spoiler


## Синтаксис


PP.Ui.Spoiler(settings);


## Параметры


settings. JSON-объект со значениями
 свойств компонента.


## Описание


Конструктор Spoiler создает
 экземпляр компонента [Spoiler](../../Components/Spoiler/Spoiler.htm).
