# DataVisualizerContainer.getRealWidth


## Синтаксис


getRealWidth();


## Описание


Метод getRealWidth возвращает
 общую ширину контейнера визуальных элементов, включая поля и отступы.


## Комментарии


Метод возвращает значение типа Number.
