# DataVisualizerContainer.getIsOverlap


## Синтаксис


getIsOverlap();


## Описание


Метод getIsOverlap возвращает
 признак того, что контейнер не уменьшает свою рабочую область автоматически.


## Комментарии


Метод возвращает значение типа Boolean.
 Если контейнер не уменьшает рабочую область, то метод возвращает значение
 true, иначе - false.
