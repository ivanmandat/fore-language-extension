# DataVisualizerContainer.getVerticalAlignment


## Синтаксис


getVerticalAlignment()


## Описание


Метод getVerticalAlignment
 возвращает вертикальное выравнивание контейнера визуальных элементов.


## Комментарии


Метод возвращает значение типа [PP.VerticalAlignment](dhtmlCommon.chm::/Enums/PP.VerticalAlignment.htm).
