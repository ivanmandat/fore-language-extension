# DataVisualizerContainer.getHorizontalAlignment


## Синтаксис


getHorizontalAlignment();


## Описание


Метод getHorizontalAlignment
 возвращает горизонтальное выравнивание контейнера визуальных элементов.


## Комментарии


Метод возвращает значение типа [PP.HorizontalAlignment](dhtmlCommon.chm::/Enums/PP.HorizontalAlignment.htm).
