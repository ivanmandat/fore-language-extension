# DataVisualizerContainer.getRealHeight


## Синтаксис


getRealHeight();


## Описание


Метод getRealHeight возвращает
 общую высоту контейнера визуальных элементов, включая поля и отступы.


## Комментарии


Метод возвращает значение типа Number.
