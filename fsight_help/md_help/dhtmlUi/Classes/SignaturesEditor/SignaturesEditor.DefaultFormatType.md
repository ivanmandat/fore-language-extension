# SignaturesEditor.DefaultFormatType


## Синтаксис


DefaultFormatType: PP.Ui.SignEditorFormatType;


## Описание


Свойство DefaultFormatType определяет
 формат по умолчанию.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDefaultFormatType,
 а возвращается с помощью метода getDefaultFormatType.
