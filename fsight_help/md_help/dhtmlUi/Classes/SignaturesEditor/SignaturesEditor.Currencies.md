# SignaturesEditor.Currencies


## Синтаксис


Currencies: Array;


## Описание


Свойство Currencies определяет
 массив используемых единиц в денежном формате.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setCurrencies,
 а возвращается с помощью метода getCurrencies.


Содержит объекты с полями:


{
  Name: String, //Имя валюты.
  Prefix: String, //Префикс валюты.
  Postfix: String, //Постфикс валюты.
}
