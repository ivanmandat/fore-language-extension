# SignaturesEditor.DefaultDelimeter


## Синтаксис


DefaultDelimeter: PP.Ui.SignEditorThousandsDelimeterType;


## Описание


Свойство DefaultDelimeter определяет
 признак добавления разделителя разрядов по умолчанию.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDefaultDelimeter,
 а возвращается с помощью метода getDefaultDelimeter.
