# SignaturesEditor.DefaultDecimalPlaces


## Синтаксис


DefaultDecimalPlaces: Number;


## Описание


Свойство DefaultDecimalPlaces
 определяет число знаков после запятой по умолчанию.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDefaultDecimalPlaces,
 а возвращается с помощью метода getDefaultDecimalPlaces.
