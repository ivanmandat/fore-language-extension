# SignaturesEditor.Formula


## Синтаксис


Formula: String;


## Описание


Свойство Formula определяет
 содержание текстового поля.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setFormula,
 а возвращается с помощью метода getFormula.
