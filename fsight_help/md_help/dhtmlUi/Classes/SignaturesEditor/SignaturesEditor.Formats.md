# SignaturesEditor.Formats


## Синтаксис


Formats: Array of PP.Ui.SignEditorFormatType;


## Описание


Свойство Formats определяет
 массив используемых форматов.


## Комментарии


Доступные форматы: Валюта, Число, Проценты.


Значение свойства устанавливается из JSON и с помощью метода setFormats,
 а возвращается с помощью метода getFormats.
