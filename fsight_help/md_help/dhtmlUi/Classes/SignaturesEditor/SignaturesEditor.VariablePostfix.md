# SignaturesEditor.VariablePostfix


## Синтаксис


VariablePostfix: String;


## Описание


Свойство VariablePostfix определяет
 постфикс переменных.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setVariablePostfix,
 а возвращается с помощью метода getVariablePostfix.


По умолчанию содержит значение "}".
