# SignaturesEditor.Delimeter


## Синтаксис


Delimeter: PP.Ui.SignEditorThousandsDelimeterType;


## Описание


Свойство Delimeter определяет
 признак добавления разделителя разрядов.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDelimeter,
 а возвращается с помощью метода getDelimeter.
