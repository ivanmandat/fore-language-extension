# SignaturesEditor.DefaultCurrency


## Синтаксис


DefaultCurrency: Number;


## Описание


Свойство DefaultCurrency определяет
 денежный формат по умолчанию.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDefaultCurrency,
 а возвращается с помощью метода getDefaultCurrency.
