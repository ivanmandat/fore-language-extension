# SignaturesEditor.FormatDelimeter


## Синтаксис


FormatDelimeter: String;


## Описание


Свойство FormatDelimeter определяет
 разделитель форматов.


## Комментарии


По умолчанию содержит значение ":".


Значение свойства устанавливается из JSON и с помощью метода setFormatDelimeter,
 а возвращается с помощью метода getFormatDelimeter.
