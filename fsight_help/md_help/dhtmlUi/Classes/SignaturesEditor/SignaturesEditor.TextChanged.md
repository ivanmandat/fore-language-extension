# SignaturesEditor.TextChanged


## Синтаксис


TextChanged: function(sender, args);


## Параметры


sender. Источник события;


args. Информация
 о событии. Аргументы:


	- Value. Значение;


	- OldValue. Предыдущее
	 значение;


	- NewContent. Содержимое;


	- OldContent. Предыдущее
	 содержимое.


## Описание


Событие TextChanged наступает
 после изменения текста.
