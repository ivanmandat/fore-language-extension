# SignaturesEditor.VariablePrefix


## Синтаксис


VariablePrefix: String;


## Описание


Свойство VariablePrefix определяет
 префикс переменных.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setVariablePrefix,
 а возвращается с помощью метода getVariablePrefix.


По умолчанию содержит значение "{%".
