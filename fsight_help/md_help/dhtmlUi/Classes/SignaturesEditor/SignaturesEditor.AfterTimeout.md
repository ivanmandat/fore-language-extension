# SignaturesEditor.AfterTimeout


## Синтаксис


AfterTimeout: function(sender, args);


## Параметры


sender. Источник события;


args. Информация
 о событии. Не содержит аргументов.


## Описание


Событие AfterTimeout наступает
 после таймаута, при изменении значения поля ввода.

Пример использования события приведен в описании свойства [SignaturesEditor.Items](SignaturesEditor.Items.htm).
