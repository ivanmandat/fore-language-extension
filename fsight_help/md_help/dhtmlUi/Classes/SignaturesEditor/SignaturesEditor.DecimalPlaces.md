# SignaturesEditor.DecimalPlaces


## Синтаксис


DecimalPlaces: Number;


## Описание


Свойство DecimalPlaces определяет
 число знаков после запятой.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDecimalPlaces,
 а возвращается с помощью метода getDecimalPlaces.


По умолчанию содержит значение «4».
