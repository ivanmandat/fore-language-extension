# FontBox.getToolbar


## Синтаксис


getToolbar();


## Описание


Метод getToolbar возвращает
 компонент [ToolBar](../../Components/ToolBar/ToolBar.htm),
 который используется для выбора варианта выравнивания текста и начертания
 шрифта.

Пример использования приведен в описании метода [FontBox.getColorCombo](FontBox.getColorCombo.htm).
