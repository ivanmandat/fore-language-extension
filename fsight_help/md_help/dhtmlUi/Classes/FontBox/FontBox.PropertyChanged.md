# FontBox.PropertyChanged


## Синтаксис


PropertyChanged: function (sender, args)


## Параметры


sender. Источник
 события;


args. Информация о событии.
 Аргументы: PropertyName - возвращает
 название свойства, значение которого было изменено, Font
 - возвращает значение свойства [FontBox.Font](FontBox.Font.htm),
 Alignment - возвращает значение
 свойства [FontBox.Alignment](FontBox.Alignment.htm).


## Описание


Событие PropertyChanged наступает
 при изменении настроек отображения текста.
