# FontBox.IsSizeExtend


## Синтаксис


IsSizeExtend: Boolean


## Описание


Свойство IsSizeExtend определяет
 отображение размера больше 36 в раскрывающемся списке размеров текста.


## Комментарии


Допустимые значения:


	- true. По умолчанию.
	 В раскрывающемся списке размеров текста будут размеры больше 36;


	- false. В раскрывающемся
	 списке размеров текста не будет размеров больше 36.


Значение свойства устанавливается из JSON и с помощью метода setIsSizeExtend,
 а возвращается с помощью метода getIsSizeExtend.
