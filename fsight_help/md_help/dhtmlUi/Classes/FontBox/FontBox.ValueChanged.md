# FontBox.ValueChanged


## Синтаксис


ValueChanged: function(sender, args)


## Параметры


sender. Источник
 события;


args. Информация о событии.
 Аргументы: PropertyName - возвращает
 название свойства, значение которого было изменено, Value
 - измененное значение свойства, Font
 - возвращает значение свойства [FontBox.Font](FontBox.Font.htm).


## Описание


Событие ValueChanged наступает
 при изменении значения параметров текста.
