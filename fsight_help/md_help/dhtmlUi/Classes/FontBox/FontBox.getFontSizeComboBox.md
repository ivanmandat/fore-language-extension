# FontBox.getFontSizeComboBox


## Синтаксис


getFontSizeComboBox ()


## Описание


Метод getFontSizeComboBox возвращает компонент [NumberEdit](../NumberEdit/NumberEdit.htm), который используется для выбора размера шрифта.
