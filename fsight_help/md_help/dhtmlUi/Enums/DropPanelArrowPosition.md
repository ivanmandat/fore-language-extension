# DropPanelArrowPosition


Пространство имен: PP.Ui;


## Описание


Перечисление DropPanelArrowPosition содержит
 варианты расположения стрелки относительно панели.


Используется свойством [ArrowDropPanel.ArrowPosition](../Classes/ArrowDropPanel/ArrowDropPanel.ArrowPosition.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 TopBottom
		 Стрелка размещена по вертикали.


		 LeftRight
		 Стрелка размещена по горизонтали.
