# ColorPanelMode


Пространство имен: PP.Ui;


## Описание


Перечисление ColorPanelMode содержит
 режимы отображения компонента [ColorCombo](../Components/ColorCombo/ColorCombo.htm).


Используется свойством [ColorPanel.Mode](../Classes/ColorPanel/ColorPanel.Mode.htm).


## Допустимые значения


			 Значение
			 Краткое описание


			 0
			 Palette.
			 Режим отображения [списка
			 для выбора цвета](../Components/ColorCombo/ColorCombo.htm) в виде палитры.


			 1
			 Spectrum.
			 Режим отображения [списка
			 для выбора цвет](../Components/ColorCombo/ColorCombo.htm)а в виде спектра.
