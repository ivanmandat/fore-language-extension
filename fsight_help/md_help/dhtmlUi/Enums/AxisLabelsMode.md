# AxisLabelsMode


Пространство имен: PP.Ui;


## Описание


Перечисление AxisLabelsMode
 содержит режимы подписей оси.


Используется свойством [CylinderGaugeAxis.LabelsMode](dhtmlGraphicBase.chm::/Classes/Cylinder/CylinderGaugeAxis/CylinderGaugeAxis.LabelsMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Auto
		 Автоматический режим.


		 Custom
		 Пользовательский режим.
