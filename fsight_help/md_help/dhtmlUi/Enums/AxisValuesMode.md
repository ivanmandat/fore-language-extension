# AxisValuesMode


Пространство имен: PP.Ui;


## Описание


Перечисление AxisValuesMode
 содержит режимы построения меток шкалы.


Используется свойством [CylinderGaugeAxis.ValuesMode](dhtmlGraphicBase.chm::/Classes/Cylinder/CylinderGaugeAxis/CylinderGaugeAxis.ValuesMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 MinMax
		 Построение меток на основе минимального и максимального значений
		 шкалы на оси цилиндра.


		 Values
		 Построение меток на основе массива значений шкалы на оси цилиндра.
