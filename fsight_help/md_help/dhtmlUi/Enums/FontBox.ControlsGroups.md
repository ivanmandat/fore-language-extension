# FontBox.ControlsGroups


Пространство имен: PP.Ui;


## Описание


Перечисление FontBox.ControlsGroups содержит
 группы элементов управления компонента [FontBox](../Components/FontBox/FontBox.htm).


Используется методом [FontBox.setControlsGroupVisibility](../Classes/FontBox/FontBox.setControlsGroupVisibility.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 FontFamily
		 Раскрывающийся список для выбора семейства шрифтов.


		 FontSize
		 Числовой редактор для определения размера шрифта.


		 TextAlignment
		 Панель инструментов для выбора варианта выравнивания.


		 FontStyle
		 Панель инструментов для выбора начертания шрифта.


		 FontColor
		 Раскрывающийся список для выбора цвета шрифта.
