# FontBox.PropertyName


Пространство имен: PP.Ui;


## Описание


Перечисление FontBox.PropertyName содержит
 названия настроек шрифта.


Используется следующими аргументом события [TextEditor.PropertyChanged](../Classes/TextEditor/TextEditor.PropertyChanged.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Size
		 Размер шрифта.


		 FontFamily
		 Семейство шрифта.


		 Alignment
		 Выравнивание текста.


		 IsBold
		 Полужирное начертание.


		 IsItalic
		 Курсивное начертание.


		 IsUnderline
		 Подчеркнутое начертание.


		 Color
		 Цвет шрифта.


		 BackgroundColor
		 Цвет фона.
