# TSSelControlPos


Пространство имен: PP.Ui;


## Описание


Перечисление TSSelControlPos
 содержит позиции элемента, который будет отображаться при выборе ячейки
 [таблицы](dhtmlTabSheet.chm::/Classes/TabSheet/TabSheet/TabSheet.htm).


Используется методом [TabSheet.setSelectionControlPosition](dhtmlTabSheet.chm::/Classes/TabSheet/TabSheet/TabSheet.setSelectionControlPosition.htm).
