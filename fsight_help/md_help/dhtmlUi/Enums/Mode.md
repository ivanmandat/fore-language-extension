# BorderComboMode


Пространство имен: PP.Ui;


## Описание


Перечисление BorderComboMode
 содержит варианты отображения [раскрывающегося
 списка для выбора типа границы](../Components/BorderCombo/BorderCombo.htm).


Используется свойством [BorderCombo.Mode](../Classes/BorderCombo/BorderCombo.Mode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Color. При открытии
		 раскрывающейся панели активна вкладка для выбора цвета линии.


		 1
		 Type. При открытии
		 раскрывающейся панели активна вкладка для выбора вида линии.


		 2
		 Width. При открытии
		 раскрывающейся панели активна вкладка для выбора толщины линии.
