# TextWrapping


Пространство имен: PP.Ui;


## Описание


Перечисление TextWrapping содержит
 режимы переноса текста на следующую строку.


Используется следующими свойствами:


	- [ChartLabel.TextWrapping](dhtmlChart.chm::/Classes/ChartLabel/ChartLabel.TextWrapping.htm);


	- [LegendBase.TextWrapping](dhtmlGraphicBase.chm::/Classes/LegendBase/LegendBase.TextWrapping.htm);


	- [LegendToolTip.TextWrapping](dhtmlGraphicBase.chm::/Classes/LegendToolTip/LegendToolTip.TextWrapping.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Hidden
		 Отображается только текст, умещающийся
		 в контейнер.


		 NoWrap
		 Перенос не осуществляется.


		 Wrap
		 Перенос осуществляется.


		 WrapWithEllipsis
		 Текст обрезается по ширине контейнера,
		 и добавляется троеточие.
