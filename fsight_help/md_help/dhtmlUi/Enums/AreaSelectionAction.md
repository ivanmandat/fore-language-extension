# AreaSelectionAction


Пространство имен: PP.Ui;


## Описание


Перечисление AreaSelectionAction
 содержит действия при выделении.


Используется свойством [DataVisualizer.AreaSelectionAction](../Classes/DataVisualizer/DataVisualizer.AreaSelectionAction.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Selection
		 Выделение области.


		 ItemsSelection
		 Выделение элементов.


		 Scale
		 Масштабирование.


		 None
		 Нет действия.
