# MessageType

## Описание


Перечисление MessageType содержит
 типы окна.


Используется следующими свойствами и методами:


	- [Message.Type](../Classes/Message/Message.Type.htm);


	- [showMessage](../Classes/PP_Ui/PP.Ui.showMessage.htm).

		 Значение

		 Окно с пиктограммой предупреждения.


		 Information
		 Окно с пиктограммой информации.

		 Окно с пользовательской пиктограммой.
