# TextEditorHyperlinkTarget


Пространство имен: PP.Ui;


## Описание


Перечисление TextEditorHyperlinkTarget
 содержит методы открытия ссылки.


Используется методом [TextEditor.insertHyperlink](../Classes/TextEditor/TextEditor.insertHyperlink.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Current
		 Открытие ссылки в текущей вкладке.


		 New
		 Открытие ссылки в новой вкладке.


		 Popup
		 Открытие ссылки в новом всплывающем окне.
