# RoundedCorners


Пространство имен: PP.Ui;


## Описание


Перечисление RoundedCorners
 содержит варианты скругления углов.


Используется методом [ToolBarButton.setRoundedCorners](../Classes/ToolBarButton/ToolBarButton.setRoundedCorners.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 All
		 Скругляются все углы.


		 Left
		 Скругляются углы слева.


		 Right
		 Скругляются углы справа.


		 None
		 Углы не скругляются.
