## Описание


Перечисление ShareType содержит
 варианты социальных сетей, для которых доступна публикация.


Используется методом [DiscussMenu.share](../Classes/DiscussMenu/DiscussMenu.share.htm).

		 Значение
		 Краткое описание


		 LiveJournal
		 Публикация в LiveJournal.


		 GooglePlus
		 Публикация в Google+.
