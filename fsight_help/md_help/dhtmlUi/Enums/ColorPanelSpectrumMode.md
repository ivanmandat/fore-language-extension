# ColorComboSpectrumMode


Пространство имен: PP.Ui;


## Описание


Перечисление ColorPanelSpectrumMode
 содержит режимы отображения спектра компонента [ColorCombo](../Components/ColorCombo/ColorCombo.htm).


Используется свойством [ColorPanel.SpectrumMode](../Classes/ColorPanel/ColorPanel.SpectrumMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 RGB. Режим RGB.


		 1
		 HSV. Режим HSV.
