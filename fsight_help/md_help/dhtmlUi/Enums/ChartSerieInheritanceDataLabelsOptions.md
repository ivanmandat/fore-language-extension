# ChartSerieInheritanceDataLabelsOptions


Пространство имен: PP.Ui;


## Описание


Перечисление ChartSerieInheritanceDataLabelsOptions
 содержит настройки подписей на диаграмме.


## Допустимые значения


		 Значение
		 Краткое описание


		 All
		 Все доступные настройки подписей на диаграмме.


		 AutoPlaced
		 Автоматическое размещение подписей на диаграмме.


		 AutoSize
		 Автоматический подбор размера подписи данных диаграммы.


		 Background
		 Заливка фона подписей на диаграмме.


		 Cap
		 Тип окончания линии подписей на диаграмме.


		 CapLength
		 Длина наконечника линии подписей на диаграмме.


		 CustomFormat
		 Пользовательский формат числа для подписей на диаграмме.


		 DisplayBorder
		 Признак отображения границы подписей на диаграмме.


		 Font
		 Параметры шрифта текста подписей на диаграмме.


		 FontColor
		 Цвет шрифта подписей на диаграмме.


		 HideOnIntersect
		 Скрывать подпись при пересечении с другой подписью.


		 HorzHotspotAlignment
		 Положение подписей на диаграмме по горизонтали.


		 HorzTextAligment
		 Горизонтальное выравнивание подписей на диаграмме.


		 HotspotOffset
		 Положение подписей на диаграмме, то есть смещение подписи по
		 оси X и Y.


		 HotspotPosition
		 Положение подписей на диаграмме относительно столбика или сектора
		 диаграммы.


		 IncludeMarketOffset
		 Учёт и включение в смещение габариты маркера при смещении подписей
		 на диаграмме.


		 NoteLineInfo
		 Устарело. Используйте Type, NotePen,
		 Cap, CapLength.


		 NotePen
		 Перо линии выноски подписей на диаграмме.


		 Pen
		 Перо границы для подписей на диаграмме.


		 Position
		 Положение подписей на диаграмме относительно ряда (столбика,
		 сектора, линии и т. д.).


		 Shadowed
		 Параметры тени подписей на диаграмме.


		 ShowText
		 Признак вывода текста подписей на диаграмме.


		 Text
		 Текст подписей на диаграмме.


		 Type
		 Тип подписей на диаграмме.


		 VertHotspotAlignment
		 Положение подписей на диаграмме по вертикали.


		 Visible
		 Признак видимости подписей на диаграмме.
