# EaseMethods


Пространство имен: PP.Ui;


## Описание


Перечисление EaseMethods содержит
 методы сглаживания анимации при её воспроизведении.


## Комментарии


Каждый метод сглаживания анимации использует одно из уравнений сглаживания,
 а также один из 3 способов применения уравнения:


	- Сглаживание с замедлением.
	 Воспроизведение начинается с максимальной скоростью, после чего замедляется
	 до нуля;


	- Сглаживание с ускорением.
	 Воспроизведение начинается с нулевой скоростью, после чего ускоряется;


	- Сглаживание с ускорением и
	 замедлением. Воспроизведение начинается с нулевой скоростью,
	 ускоряется до тех пор, пока не будет воспроизведена половина анимации,
	 после чего опять замедляется до нуля.


## Допустимые значения


		 Значение
		 Краткое описание


		 easeInBack
		 Сглаживание с ускорением по уравнению плавности, которое осуществляет
		 незначительное движение в обратном направлении незадолго до того,
		 как оно начнет выполняться по указанному пути.


		 easeInBounce
		 Сглаживание с ускорением по уравнению колебательного движения
		 в анимации. Такой тип сглаживания создает анимацию, похожую на
		 падение и отскакивание от пола мяча с несколькими затухающими
		 отскоками.


		 easeInCirc
		 Сглаживание с ускорением по уравнению окружности.


		 easeInCubic
		 Сглаживание с ускорением по уравнению кубической кривой.


		 easeInElastic
		 Сглаживание с ускорением по уравнению экспоненциально затухающей
		 синусоиды. Такой тип сглаживания создает анимацию, похожую на
		 колебательное движение пружины назад и вперед до полной остановки.


		 easeInExpo
		 Сглаживание с ускорением по уравнению экспоненты.


		 easeInOutBack
		 Сглаживание с ускорением и замедлением по уравнению плавности,
		 которое осуществляет незначительное движение в обратном направлении
		 незадолго до того, как оно начнет выполняться по указанному пути.
