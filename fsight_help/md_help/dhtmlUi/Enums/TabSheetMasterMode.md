# TabSheetMasterMode


Пространство имен: PP.Ui;


## Описание


Перечисление TabSheetMasterMode
 указывает на тип мастера, который меняется и устанавливается при выборе
 листа таблицы.


## Допустимые значения


		 Значение
		 Краткое описание


		 Calculation
		 Выделяется вычисленный элемент измерения.


		 Corner
		 Выделяется уголок.


		 Data
		 Выделяются все данные исключая заголовки и уголки.


		 DimElement
		 Выделяется элемент измерения.


		 Dimension
		 Выделяется измерение.


		 LeftHeader
		 Выделяется боковой заголовок.


		 LeftHeaderTotals
		 Выделяются итоги по столбцам.


		 Level
		 Выделяется уровень.


		 Table
		 Выделяется ячейка с данными.


		 TopHeader
		 Выделяется верхний заголовок.


		 TopHeaderTotals
		 Выделяются итоги по строкам.
