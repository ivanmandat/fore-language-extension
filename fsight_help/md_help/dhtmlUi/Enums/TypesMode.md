# BorderComboTypesMode


Пространство имен: PP.Ui;


## Описание


Перечисление BorderComboTypesMode содержит
 технологии, в рамках которых доступно отображение разных типов границ.


Используется свойством [BorderCombo.TypesMode](../Classes/BorderCombo/BorderCombo.TypesMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 SVG
		 Доступны границы, поддерживаемые
		 технологией SVG.


		 SVGLight
		 Упрощенный режим SVG. Без линий со стилями long и short.


		 CSS
		 Доступны границы, поддерживаемые
		 технологией CSS.


		 Services
		 Доступны границы, поддерживаемые
		 технологиями SVG и CSS.
