# SpoilerPosition


Пространство имен: PP.Ui;


## Описание


Перечисление SpoilerPosition содержит
 направления раскрытия панели.


Используется свойством [Spoiler.Position](../Classes/Spoiler/Spoiler.Position.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Top
		 Панель раскрывается сверху вниз.


		 Bottom
		 Панель раскрывается снизу вверх.


		 Left
		 Панель раскрывается слева направо.


		 Right
		 Панель раскрывается справа налево.
