# Rect


Пространство имен**: PP.Ui;


## Иерархия наследования


PP.Ui.Rect


## Описание


Интерфейс Rect используется
 для определения параметров прямоугольника двумерного пространства.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [height](dhtmlUi.chm::/Interfaces/Rect/Rect.height.htm)
		 Свойство height
		 определяет значение высоты прямоугольника.


		 ![](../../Property_Image.gif)
		 [left](dhtmlUi.chm::/Interfaces/Rect/Rect.left.htm)
		 Свойство left
		 определяет значение координаты X левого верхнего угла прямоугольника.


		 ![](../../Property_Image.gif)
		 [top](dhtmlUi.chm::/Interfaces/Rect/Rect.top.htm)
		 Свойство top
		 определяет значение координаты Y левого верхнего угла прямоугольника.


		 ![](../../Property_Image.gif)
		 [width](dhtmlUi.chm::/Interfaces/Rect/Rect.width.htm)
		 Свойство width
		 определяет значение ширины прямоугольника.
