# ChartAxis.AxisCrossValue


## Синтаксис


AxisCrossValue: Number


## Описание


Свойство AxisCrossValue** определяет значение, в котором данная ось пузырьковой диаграммы пересекается с другой осью.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setAxisCrossValue, а возвращается с помощью метода getAxisCrossValue.
