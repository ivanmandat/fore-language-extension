# ChartAxis.AutoMin


## Синтаксис


AutoMin: Boolean


## Описание


Свойство AutoMin** определяет возможность автоматической установки минимального значения оси пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setAutoMin, а возвращается с помощью метода getAutoMin.


Свойство содержит значение true, если автоматическая установка минимального значения оси пузырьковой диаграммы разрешена, иначе - значение false.
