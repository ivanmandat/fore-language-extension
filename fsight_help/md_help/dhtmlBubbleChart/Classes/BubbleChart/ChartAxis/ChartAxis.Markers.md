# ChartAxis.Markers


## Синтаксис


Markers: PP.Ui.MarkerCollection


## Описание


Свойство Markers** определяет коллекцию меток оси пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMarkers, а возвращается с помощью метода getMarkers.
