# ChartAxis.MinValue


## Синтаксис


MinValue: Number


## Описание


Свойство MinValue** определяет минимальное значение оси пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается с помощью метода setMinValue, а возвращается с помощью метода getMinValue. Из JSON значение свойства установить нельзя.
