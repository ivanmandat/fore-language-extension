# ChartAxis.Caption


## Синтаксис


Caption: PP.Ui.[ChartLabel](dhtmlChart.chm::/Classes/ChartLabel/ChartLabel.htm)


## Описание


Свойство Caption** определяет заголовок оси пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setCaption, а возвращается с помощью метода getCaption.
