# ChartAxis.Stroke


## Синтаксис


Stroke: PP.[Brush](dhtmlCommon.chm::/Classes/PP/Brush/Brush.htm)


## Описание


Свойство Stroke** определяет
 кисть для отрисовки оси пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setStroke,
 а возвращается с помощью метода getStroke.
