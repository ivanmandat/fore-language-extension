# ChartAxis.Labels


## Синтаксис


Labels: [PP.Ui.ChartLabelCollection](../ChartLabelCollection/ChartLabelCollection.htm)


## Описание


Свойство Labels** определяет коллекцию меток для осей пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLabels, а возвращается - с помощью метода getLabels.
