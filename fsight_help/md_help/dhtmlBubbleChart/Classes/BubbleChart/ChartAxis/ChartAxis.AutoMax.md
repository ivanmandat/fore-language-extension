# ChartAxis.AutoMax


## Синтаксис


AutoMax: Boolean


## Описание


Свойство AutoMax** определяет возможность автоматической установки максимального значения оси пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setAutoMax, а возвращается с помощью метода getAutoMax.


Свойство содержит значение true, если автоматическая установка максимального значения оси пузырьковой диаграммы разрешена, иначе - значение false.
