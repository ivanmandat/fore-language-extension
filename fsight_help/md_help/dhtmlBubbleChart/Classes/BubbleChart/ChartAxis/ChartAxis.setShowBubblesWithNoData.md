# ChartAxis.setShowBubblesWithNoData


## Синтаксис


setShowBubblesWithNoData(value: Boolean);


## Параметры


value. Признак отображения пузырьков без данных. Если параметр равен значению true, то пузырьки без данных будут отображаться в пузырьковой диаграмме, если равен значению false, то не будут.


## Описание


Метод setShowBubblesWithNoData** позволяет отображать в пузырьковой диаграмме пузырьки без данных.
