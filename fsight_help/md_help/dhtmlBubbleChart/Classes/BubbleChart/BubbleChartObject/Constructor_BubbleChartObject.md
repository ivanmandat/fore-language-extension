# Конструктор BubbleChartObject


## Синтаксис


PP.Ui.BubbleChartObject (settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор BubbleChartObject создает экземпляр класса [BubbleChartObject](BubbleChartObject.htm).
