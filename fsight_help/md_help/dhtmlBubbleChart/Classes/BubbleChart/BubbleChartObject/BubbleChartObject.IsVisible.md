# BubbleChartObject.IsVisible


## Синтаксис


IsVisible: Boolean;


## Описание


Свойство IsVisible** определяет
 видимость элемента пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsVisible,
 а возвращается с помощью метода getIsVisible.


Если для свойства установлено значение true,
 то элемент пузырьковой диаграммы будет видимым, если значение false,
 то не будет.
