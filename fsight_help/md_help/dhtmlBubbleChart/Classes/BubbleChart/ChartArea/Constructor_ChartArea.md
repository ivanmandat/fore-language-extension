# Конструктор ChartArea


## Синтаксис


PP.Ui.ChartArea(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор ChartArea создает экземпляр класса [ChartArea](ChartArea.htm).


## Пример


Конструктор предназначен для внутреннего использования.
