# ChartArea.getPaper


## Синтаксис


getPaper();


## Описание


Метод getPaper** возвращает SVG-элемент, содержащий область построения пузырьковой диаграммы.


## Комментарии


Метод возвращает значение типа SVGElement.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента [BubbleChart](../../../Components/BubbleChart/BubbleChart.htm) с наименованием «bubbleChart» (см. «[Пример создания компонента BubbleChart](../../../Components/BubbleChart/BubbleChart_Example.htm)»). Получим SVG-элемент, содержащий область построения пузырьковой диаграммы и выведем размеры области построения:


// Получим область пузырьковой диаграммы
var chartArea = bubbleChart.getChartArea();
// Получим SVG-элемент, содержащий область построения пузырьковой диаграммы
var paper = chartArea.getPaper();
// Отобразим размеры области построения пузырьковой диаграммы
console.log("Ширина: " + paper.clientWidth + " Высота: " + paper.clientHeight);

В результате выполнения примера в консоли браузера были выведены размеры области построения пузырьковой диаграммы:


Ширина: 500 Высота: 243
