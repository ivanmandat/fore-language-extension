# ChartArea.getCrosshairLabelsLayer


## Синтаксис


getCrosshairLabelsLayer();


## Описание


Метод getCrosshairLabelsLayer
 возвращает SVG-элемент, предназначенный для отрисовки слоя меток, указывающих
 положение выделенных элементов.


## Комментарии


Метод возвращает значение типа SVGElement.
