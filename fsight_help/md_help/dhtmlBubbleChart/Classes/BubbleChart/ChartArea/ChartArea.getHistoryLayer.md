# ChartArea.getHistoryLayer


## Синтаксис


getHistoryLayer();


## Описание


Метод getHistoryLayer возвращает** SVG-элемент, содержащий историю значений пузырьков пузырьковой диаграммы.


## Комментарии


Метод возвращает значение типа SVGElement.
