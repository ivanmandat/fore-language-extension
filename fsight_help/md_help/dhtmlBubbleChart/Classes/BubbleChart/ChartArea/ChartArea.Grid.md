# ChartArea.Grid


## Синтаксис


Grid: [PP.Ui.ChartGrid](../ChartGrid/ChartGrid.htm)


## Описание


Свойство Grid** определяет сетку области построения пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setGrid, а возвращается с помощью метода getGrid.
