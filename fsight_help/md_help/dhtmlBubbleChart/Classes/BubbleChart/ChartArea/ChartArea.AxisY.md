# ChartArea.AxisY


## Синтаксис


AxisY: [PP.Ui.ChartAxis](../ChartAxis/ChartAxis.htm)


## Описание


Свойство AxisY** определяет
 ось Y области построения пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается с помощью метода setAxisY,
 а возвращается с помощью метода getAxisY.
 Из JSON свойство установить нельзя.
