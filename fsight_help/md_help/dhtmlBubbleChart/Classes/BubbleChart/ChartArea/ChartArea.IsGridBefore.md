# ChartArea.IsGridBefore


## Синтаксис


IsGridBefore: Boolean


## Описание


Свойство IsGridBefore** определяет, отображается ли сетка области построения пузырьковой диаграммы поверх пузырьков.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsGridBefore, а возвращается с помощью метода getIsGridBefore.


Свойство принимает значение true, если сетка области построения отображается поверх пузырьков, иначе - false.


По умолчанию свойство содержит значение false.
