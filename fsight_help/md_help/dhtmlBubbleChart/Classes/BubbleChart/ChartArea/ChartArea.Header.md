# ChartArea.Header


## Синтаксис


Header: [PP.Ui.ChartLabel](dhtmlChart.chm::/Classes/ChartLabel/ChartLabel.htm)


## Описание


Свойство Header** определяет
 заголовок для области построения пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHeader,
 а возвращается с помощью метода getHeader.
