# ChartArea.AngleLines


## Синтаксис


AngleLines: PP.Ui.[ChartAngleLineCollection](../ChartAngleLineCollection/ChartAngleLineCollection.htm);


## Описание


Свойство AngleLines определяет
 коллекцию наклонных линий области построения пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setAngleLines,
 а возвращается с помощью метода getAngleLines.
