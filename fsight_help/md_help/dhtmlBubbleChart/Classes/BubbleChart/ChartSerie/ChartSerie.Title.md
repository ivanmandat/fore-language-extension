# ChartSerie.Title


## Синтаксис


Title: String


## Описание


Свойство Title** определяет заголовок для ряда данных.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTitle, а возвращается с помощью метода getTitle.
