# ChartSerie.Points


## Синтаксис


Points: Array


## Описание


Свойство Points** определяет точки для ряда данных.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setPoints, а возвращается с помощью метода getPoints.


Свойство содержит массив объектов типа [PP.Ui.ChartSeriePoint](../ChartSeriePoint/ChartSeriePoint.htm).
