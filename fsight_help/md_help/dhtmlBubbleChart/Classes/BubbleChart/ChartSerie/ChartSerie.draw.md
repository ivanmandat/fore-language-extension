# ChartSerie.draw


## Синтаксис


draw(parentNode: SVGSVGElement|SVGGElement);


## Параметры


*parentNode.* Контейнер, в котором нужно отрисовать ряд данных.


## Описание


Метод draw** рисует ряд данных в пузырьковой диаграмме.
