# ChartSerie.Index


## Синтаксис


Index: Number


## Описание


Свойство Index** определяет индекс ряда данных.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIndex, а возвращается с помощью метода getIndex.
