# ChartLabelCollection.MaskText


## Синтаксис


MaskText: String


## Описание


Свойство MaskText** определяет маску текста в метках коллекции.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMaskText, а возвращается с помощью метода getMaskText.
