# ChartLabelCollection.Style


## Синтаксис


Style: Object


## Описание


Свойство Style** определяет
 стиль меток в коллекции.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setStyle,
 а возвращается с помощью метода getStyle.


Свойство содержит JSON-объект с полем Release,
 являющимся объектом типа [PP.Style](dhtmlCommon.chm::/Classes/PP/Style/Style.htm).
