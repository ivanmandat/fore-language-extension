# ChartLine.Value


## Синтаксис


Value: Number


## Описание


Свойство Value** определяет значение, соответствующее заданной линии уровня в пузырьковой диаграмме.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setValue, а возвращается с помощью метода getValue.
