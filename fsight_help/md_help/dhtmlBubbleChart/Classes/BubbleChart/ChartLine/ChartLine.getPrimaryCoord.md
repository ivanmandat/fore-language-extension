# ChartLine.getPrimaryCoord


## Синтаксис


getPrimaryCoord(visualType: [PP.Ui.VisualType](../../../Enums/VisualType.htm));


## Параметры


*visualType.* Тип координаты линии
 уровня. Если параметр равен значению PP.Ui.VisualType.X, то будет возвращено
 значение, соответствующее координате линии уровня по оси X, если значению
 PP.Ui.VisualType.Y - по оси Y.


## Описание


Метод getPrimaryCoord**
 возвращает значение, соответствующее указанной координате линии уровня.


## Комментарии


Метод возвращает значение типа Number.
