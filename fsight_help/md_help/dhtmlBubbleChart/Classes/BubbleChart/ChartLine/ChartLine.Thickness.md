# ChartLine.Thickness


## Синтаксис


Thickness: Number


## Описание


Свойство Thickness** определяет толщину линии уровня в пикселях.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setThickness, а возвращается с помощью метода getThickness.
