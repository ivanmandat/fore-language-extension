# ChartSeriePoint.Tooltip


## Синтаксис


Tooltip: [PP.Ui.ChartTooltip](../ChartSymbol/ChartSymbol.htm)


## Описание


Свойство Tooltip** определяет всплывающую подсказку для пузырька пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTooltip, а возвращается с помощью метода getTooltip.
