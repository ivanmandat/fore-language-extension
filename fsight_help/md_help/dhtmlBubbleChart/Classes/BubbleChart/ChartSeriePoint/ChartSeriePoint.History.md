# ChartSeriePoint.History


## Синтаксис


History: [PP.Ui.ChartHistory](../ChartHistory/ChartHistory.htm)


## Описание


Свойство History** определяет историю значений для ряда данных.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHistory, а возвращается с помощью метода getHistory.
