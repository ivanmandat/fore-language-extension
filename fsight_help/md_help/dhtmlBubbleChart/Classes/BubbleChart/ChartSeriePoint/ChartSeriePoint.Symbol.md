# ChartSeriePoint.Symbol


## Синтаксис


Symbol: [PP.Ui.ChartSymbol](../ChartSymbol/ChartSymbol.htm)


## Описание


Свойство Symbol** определяет символ для пузырька пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSymbol, а возвращается с помощью метода getSymbol.
