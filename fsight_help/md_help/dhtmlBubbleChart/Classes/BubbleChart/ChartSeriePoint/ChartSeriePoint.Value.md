# ChartSeriePoint.Value


## Синтаксис


Value: String


## Описание


Свойство Value** определяет наименование пузырька пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setValue, а возвращается с помощью метода getValue.
