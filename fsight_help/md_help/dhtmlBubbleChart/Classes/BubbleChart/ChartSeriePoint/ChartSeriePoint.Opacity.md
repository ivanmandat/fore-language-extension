# ChartSeriePoint.Opacity


## Синтаксис


Opacity: Number


## Описание


Свойство Opacity** определяет уровень прозрачности пузырька пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setOpacity, а возвращается с помощью метода getOpacity.


Свойство содержит числовое значение от 0 до 1 включительно. Значение 0 соответствует прозрачному пузырьку, 1 - непрозрачному.
