# ChartSeriePoint.getTooltipValues


## Синтаксис


getTooltipValues();


## Описание


Метод getTooltipValues** возвращает значения пузырька пузырьковой диаграммы по [всем измерениям](../../../Enums/VisualType.htm).


## Комментарии


Метод возвращает JSON-объект со следующими свойствами: ColorTitle и ColorValue - наименование и значение измерения по цвету пузырька ( PP.Ui.VisualType.Color), SizeTitle и SizeValue - по его размеру ( PP.Ui.VisualType.R), XTitle и XValue - по координате оси X ( PP.Ui.VisualType.X), YTitle и YValue - по координате оси Y ( PP.Ui.VisualType.Y), Name - наименование пузырька пузырьковой диаграммы.
