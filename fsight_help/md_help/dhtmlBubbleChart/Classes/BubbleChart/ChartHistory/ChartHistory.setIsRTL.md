# ChartHistory.setIsRTL


## Синтаксис


setIsRTL(value: Boolean);


## Параметры


*value.* Определяет положение меток истории значений пузырьков.
 Если установлено значение true,
 то метки истории будут выравнены по левому краю пузырька, иначе - по правому
 краю пузырька.


## Описание


Метод setIsRTL устанавливает
 положение меток истории значений пузырьков.
