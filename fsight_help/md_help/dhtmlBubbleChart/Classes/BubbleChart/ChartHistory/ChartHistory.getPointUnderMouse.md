# ChartHistory.getPointUnderMouse


## Синтаксис


getPointUnderMouse(x: Number, y: Number);


## Параметры


x. Координата X проверяемой
 точки;


y. Координата Y проверяемой
 точки


## Описание


Метод getPointUnderMouse проверяет
 существование истории пузырька в заданной точке.


## Комментарии


Метод возвращает экземпляр класса PP.Ui.[ChartSeriePoint](../ChartSeriePoint/ChartSeriePoint.htm),
 если история в заданной точке существует.
