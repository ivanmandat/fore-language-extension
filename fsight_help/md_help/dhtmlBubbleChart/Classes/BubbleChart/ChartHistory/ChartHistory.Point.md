# ChartHistory.Point


## Синтаксис


Point: [PP.Ui.ChartSeriePoint](../ChartSeriePoint/ChartSeriePoint.htm);


## Описание


Свойство Point** устанавливает
 точку ряда данных пузырьковой диаграммы, для которой необходимо отобразить историю значений
 данного ряда.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setPoint,
 а возвращается с помощью метода getPoint.
