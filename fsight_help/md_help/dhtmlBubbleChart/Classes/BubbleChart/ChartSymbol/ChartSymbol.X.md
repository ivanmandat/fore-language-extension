# ChartSymbol.X


## Синтаксис


X: Number;


## Описание


Свойство X** определяет
 координату X центра символа пузырька пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setX,
 а возвращается с помощью метода getX.
