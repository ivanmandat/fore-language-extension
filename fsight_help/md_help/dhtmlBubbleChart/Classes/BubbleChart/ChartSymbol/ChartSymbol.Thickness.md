# ChartSymbol.Thickness


## Синтаксис


Thickness: Number


## Описание


Свойство Thickness** определяет
 для границы символа пузырька пузырьковой диаграммы толщину в пикселях.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setThickness,
 а возвращается с помощью метода getThickness.
