# ChartSymbol.MarkerShape


## Синтаксис


MarkerShape: [PP.Ui.IntervalsLegendMarkerShape](../../../Enums/IntervalsLegendMarkerShape.htm)


## Описание


Свойство MarkerShape**
 определяет форму символа пузырька пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMarkerShape,
 а возвращается с помощью метода getMarkerShape.
