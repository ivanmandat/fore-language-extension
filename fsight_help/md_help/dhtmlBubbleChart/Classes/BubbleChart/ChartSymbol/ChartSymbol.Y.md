# ChartSymbol.Y


## Синтаксис


Y: Number;


## Описание


Свойство Y определяет координату
 Y центра символа пузырька пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setY,
 а возвращается с помощью метода getY.
