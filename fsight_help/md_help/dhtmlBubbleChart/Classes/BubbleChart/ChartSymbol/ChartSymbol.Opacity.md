# ChartSymbol.Opacity


## Синтаксис


Opacity: Number;


## Описание


Свойство Opacity определяет
 прозрачность символа пузырька пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setOpacity,
 а возвращается с помощью метода getOpacity.


Значения диапазона прозрачности изменяются от 0 до 1.
