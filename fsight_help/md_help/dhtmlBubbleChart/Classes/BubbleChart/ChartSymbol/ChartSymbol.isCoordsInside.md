# ChartSymbol.isCoordsInside


## Синтаксис


isCoordsInside(x: Number, y: Number);


## Параметры


x. Координата X проверяемой
 точки;


y. Координата Y проверяемой
 точки.


## Описание


Метод isCoordsInside определяет,
 попадает ли точка в область символа пузырька пузырьковой диаграммы.


## Комментарии


Метод возвращает значение true,
 если точка попадает в область символа пузырька пузырьковой диаграммы, иначе - значение
 false.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 [BubbleChart](../../../Components/BubbleChart/BubbleChart.htm)
 с наименованием «bubbleChart» (см. «[Пример
 создания компонента BubbleChart](../../../Components/BubbleChart/BubbleChart_Example.htm)»). Проверим попадание точки с координатами
 (150, 200) в область символа первого пузырька пузырьковой диаграммы:


// Получаем символ первого пузырька пузырьковой диаграммы
var symbol = bubbleChart.getSeries()[0].getPoints()[0].getSymbol();
// Проверяем попадание точки в область символа первого пузырька пузырьковой диаграммы
if (symbol.isCoordsInside(150, 200)) {
    console.log("Точка попадает в область символа первого пузырька пузырьковой диаграммы");
} else {
    console.log("Точка не попадает в область символа первого пузырька пузырьковой диаграммы");
}
В результате в консоль будет выведен результат проверки попадания точки
 в область символа:


Точка не попадает в область символа первого пузырька
 пузырьковой диаграммы
