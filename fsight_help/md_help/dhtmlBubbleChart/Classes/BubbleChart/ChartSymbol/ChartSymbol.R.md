# ChartSymbol.R


## Синтаксис


R: Number;


## Описание


Свойство R определяет радиус
 символа пузырька пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setR,
 а возвращается с помощью метода getR.
