# BubbleChart.UseBubbleColorForHistoryLines


## Синтаксис


UseBubbleColorForHistoryLines: Boolean;


## Описание


Свойство UseBubbleColorForHistoryLines
 устанавливает признак того, будут ли использоваться цвета пузырьков
 истории значений при отрисовке линий между ними.


## Комментарии


Значение свойства устанавливается с помощью метода setUseBubbleColorForHistoryLines,
 а возвращается с помощью метода getUseBubbleColorForHistoryLines.


Допустимые значения:


	- true.
	 Цвета пузырьков истории значений будут использоваться при отрисовке
	 линий между ними;


	- false.
	 Цвета пузырьков истории значений не будут использоваться при отрисовке
	 линий между ними.
