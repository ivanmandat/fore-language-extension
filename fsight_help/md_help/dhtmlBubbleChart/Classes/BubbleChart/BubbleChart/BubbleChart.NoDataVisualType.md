# BubbleChart.NoDataVisualType


## Синтаксис


NoDataVisualType: [PP.Ui.VisualType](../../../Enums/VisualType.htm);


## Описание


Свойство NoDataVisualType**
 определяет тип отображения элемента без данных.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setNoDataVisualType,
 а возвращается с помощью метода getNoDataVisualType.
