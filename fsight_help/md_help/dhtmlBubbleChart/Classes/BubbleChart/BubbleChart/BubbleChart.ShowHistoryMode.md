# BubbleChart.ShowHistoryMode


## Синтаксис


ShowHistoryMode: [PP.HoverMode](../../../Enums/PP.HoverMode.htm);


## Описание


Свойство ShowHistoryMode**
 определяет режим отображения истории значений для пузырьков пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**ShowHistoryMode**,
 а возвращается с помощью метода get**ShowHistoryMode**.
