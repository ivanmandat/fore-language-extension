# BubbleChart.ShowHistoryBubbleTooltip


## Синтаксис


ShowHistoryBubbleTooltip: Boolean;


## Описание


Свойство ShowHistoryBubbleTooltip**
 определяет отображение пузырьков истории значений при отображении всплывающей
 подсказки элемента.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**ShowHistory**BubbleTooltip, а возвращается с
 помощью метода get**ShowHistory**BubbleTooltip.
