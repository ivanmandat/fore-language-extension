# BubbleChart.UseInactiveItemsOpacity


## Синтаксис


UseInactiveItemsOpacity: Boolean;


## Описание


Свойство UseInactiveItemsOpacity**
 определяет, разрешена ли прозрачная заливка неактивных пузырьков пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setUseInactiveItemsOpacity,
 а возвращается - с помощью метода getUseInactiveItemsOpacity.


Свойство содержит значение true,
 если прозрачная заливка неактивных пузырьков пузырьковой диаграммы разрешена, и значение
 false, если не разрешена.
