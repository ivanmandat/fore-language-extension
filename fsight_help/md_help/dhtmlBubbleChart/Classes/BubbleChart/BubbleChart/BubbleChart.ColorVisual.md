# BubbleChart.ColorVisual


## Синтаксис


ColorVisual: PP.ColorVisual;


## Описание


Свойство ColorVisual**
 определяет цвета заливки пузырьков пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается с помощью метода setColorVisual,
 а возвращается с помощью метода getColorVisual.
 Из JSON значение задать нельзя.
