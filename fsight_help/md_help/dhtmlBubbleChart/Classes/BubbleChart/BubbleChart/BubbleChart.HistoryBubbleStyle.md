# BubbleChart.HistoryBubbleStyle


## Синтаксис


HistoryBubbleStyle: [PP.Ui.VisualizerItemStyle](../../../Enums/VisualizerItemStyle.htm);


## Описание


Свойство HistoryBubbleStyle устанавливает стиль отображения пузырьков истории
 значений пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается с помощью метода setHistoryBubbleStyle,
 а возвращается с помощью метода getHistoryBubbleStyle.
 Если значение свойства не задано, то используется стиль отображения
 обычных пузырьков.


## Пример


Пример использования свойства приведен в описании метода [BubbleChart.applyColorScheme](BubbleChart.applyColorScheme.htm).
