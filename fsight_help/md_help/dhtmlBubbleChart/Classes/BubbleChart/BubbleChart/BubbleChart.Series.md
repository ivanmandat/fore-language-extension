# BubbleChart.Series


## Синтаксис


Series: Array;


## Описание


Свойство Series** определяет
 ряды данных для пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSeries,
 а возвращается с помощью метода getSeries.


Свойство содержит массив объектов типа [PP.Ui.ChartSerie](../ChartSerie/ChartSerie.htm).
