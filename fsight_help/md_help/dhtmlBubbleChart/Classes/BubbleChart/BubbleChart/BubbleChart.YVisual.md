# BubbleChart.YVisual


## Синтаксис


YVisual: PP.CoordinateVisual;


## Описание


Свойство YVisual** содержит
 настройки сопоставления значений рядов оси Y пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается с помощью метода setYVisual,
 а возвращается с помощью метода getYVisual.
 Из JSON значение свойства задать нельзя.
