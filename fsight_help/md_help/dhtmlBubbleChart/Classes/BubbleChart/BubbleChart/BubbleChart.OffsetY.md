# BubbleChart.OffsetY


## Синтаксис


OffsetY: Number;


## Описание


Свойство OffsetY** определяет
 отступ пузырьков от верхней границы области построения пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода **setOffsetY**,
 а возвращается с помощью метода getOffsetY.
