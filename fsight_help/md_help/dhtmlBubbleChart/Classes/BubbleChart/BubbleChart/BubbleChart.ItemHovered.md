# BubbleChart.ItemHovered


## Синтаксис


ItemHovered: function (sender, args: Object, timeout: Number);


## Параметры


sender. Источник события;


args. Информация о событии.
 Доступные аргументы: Item - пузырек пузырьковой диаграммы, Hovered - признак, определяющий,
 наведен или отведен курсор;


timeout. Промежуток времени
 в миллисекундах, через который вызовется событие.


## Описание


Событие ItemHovered наступает
 при наведении указателя на пузырек пузырьковой диаграммы.
