# BubbleChart.isLoaded


## Синтаксис


isLoaded();


## Описание


Метод isLoaded** возвращает признак того, что пузырьковая диаграмма загружена.


## Комментарии


Если пузырьковая диаграмма загружена, то метод возвращает значение true, иначе - значение false.
