# BubbleChart.HistoryMode


## Синтаксис


HistoryMode: [PP.Ui.NoDataHistoryMode](../../../Enums/NoDataHistoryMode.htm);


## Описание


Свойство HistoryMode**
 определяет режим отображения истории значений для выбранного пузырька
 пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода **setHistoryMode**,
 а возвращается с помощью метода getHistoryMode.
