# BubbleChart.ShowBubblesWithNoData


## Синтаксис


ShowBubblesWithNoData: Boolean;


## Описание


Свойство ShowBubblesWithNoData**
 содержит признак отображения пузырьков пузырьковой диаграммы без данных.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**ShowBubblesWithNoData**,
 а возвращается с помощью метода get**ShowBubblesWithNoData**.


Свойство содержит значение true,
 если отображение пузырьков пузырьковой диаграммы без данных разрешено, и значение false, если запрещено.
