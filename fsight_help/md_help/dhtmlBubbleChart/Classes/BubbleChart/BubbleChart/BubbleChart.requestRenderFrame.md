# BubbleChart.requestRenderFrame


## Синтаксис


requestRenderFrame(renderPoints: Boolean, renderGrid:
 Boolean);


## Параметры


renderPoints. Признак необходимости
 перерисовки пузырьков и связанных с ними элементов (выделение, история).
 Необязательный параметр;


renderGrid. Признак необходимости
 перерисовки координатной сетки и осей пузырьковой диаграммы. Необязательный
 параметр.


## Описание


Метод requestRenderFrame вызывает
 асинхронную перерисовку пузырьков пузырьковой диаграммы.


## Комментарии


Допустимые значения параметра renderPoints:


	- true. Необходима перерисовка
	 пузырьков и связанных с ними элементов;


	- false. Перерисовка пузырьков
	 и связанных с ними элементов не требуется.


По умолчанию для параметра renderPoints устанавливается значение true.


Допустимые значения параметра renderGrid:


	- true. Необходима перерисовка координатной
	 сетки и осей пузырьковой диаграммы;


	- false. Перерисовка координатной
	 сетки и осей пузырьковой диаграммы не требуется.


По умолчанию для параметра renderGrid устанавливается значение true.
