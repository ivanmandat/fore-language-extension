# BubbleChart.CaptionClick


## Синтаксис


CaptionClick: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: Break - признак отмены события, Element - SVG-элемент, соответствующий пузырьковой диаграмме, Event - событие мыши, Type - тип отображаемого элемента, задаваемый с помощью перечисления [PP.Ui.VisualType](../../../Enums/VisualType.htm).


## Описание


Событие CaptionClick** наступает при нажатии на заголовок пузырьковой диаграммы и на заголовки её элементов.
