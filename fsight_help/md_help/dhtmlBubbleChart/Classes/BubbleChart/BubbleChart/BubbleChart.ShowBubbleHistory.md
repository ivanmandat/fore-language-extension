# BubbleChart.ShowBubbleHistory


## Синтаксис


ShowBubbleHistory: Boolean;


## Описание


Свойство ShowBubbleHistory**
 содержит признак отображения истории значений для пузырьков пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**ShowBubbleHistory**,
 а возвращается с помощью метода get**ShowBubbleHistory**.


Свойство содержит значение true,
 если отображение истории значений для пузырьков пузырьковой диаграммы разрешено, и значение
 false, если запрещено.
