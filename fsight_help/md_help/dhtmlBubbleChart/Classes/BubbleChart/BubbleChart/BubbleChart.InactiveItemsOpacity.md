# BubbleChart.InactiveItemsOpacity


## Синтаксис


InactiveItemsOpacity: Number;


## Описание


Свойство InactiveItemsOpacity**
 содержит коэффициент прозрачности неактивных пузырьков пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setInactiveItemsOpacity,
 а возвращается - с помощью метода getInactiveItemsOpacity.


Свойство может принимать значения от 0 до 1, где 0 соответствует абсолютно
 прозрачному пузырьку, а 1 - непрозрачному.
