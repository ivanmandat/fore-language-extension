# ChartTooltip.HasMask


## Синтаксис


HasMask: Boolean


## Описание


Свойство HasMask** определяет, формируется ли текст всплывающей подсказки по шаблону.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHasMask, а возвращается с помощью метода getHasMask.


Свойство содержит значение true, если текст всплывающей подсказки формируется по шаблону, иначе - значение false.
