# ChartSelection.HorizontalLine


## Синтаксис


HorizontalLine: Boolean


## Описание


Свойство HorizontalLine** определяет наличие горизонтальной линии уровня, соответствующей выделенному пузырьку пузырьковой диаграммы относительно оси Y.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHorizontalLine, а возвращается с помощью метода getHorizontalLine.


Свойство содержит значение true, если для выделенного пузырька пузырьковой диаграммы нарисована горизонтальная линия уровня вдоль оси Y, иначе - значение false.


По умолчанию свойство имеет значение true.
