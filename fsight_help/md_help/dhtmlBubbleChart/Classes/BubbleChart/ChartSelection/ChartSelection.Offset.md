# ChartSelection.Offset


## Синтаксис


Offset: Number


## Описание


Свойство Offset** определяет отступ окружности, соответствующей выделенному пузырьку пузырьковой диаграммы, от его границы в пикселях.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setOffset, а возвращается с помощью метода getOffset.
