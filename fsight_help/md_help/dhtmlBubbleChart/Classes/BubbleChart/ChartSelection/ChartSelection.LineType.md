# ChartSelection.LineType


## Синтаксис


LineType: [PP.Ui.LineTypes](dhtmlUi.chm::/Enums/LineTypes.htm);


## Описание


Свойство LineType** определяет тип линий окружности и уровней, соответствующих выделенному пузырьку пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLineType, а возвращается с помощью метода getLineType.
