# ChartSelection.VerticalLineExtension


## Синтаксис


VerticalLineExtension: Boolean


## Описание


Свойство VerticalLineExtension определяет наличие вертикальной линии уровня, соответствующей выделенному пузырьку пузырьковой диаграммы относительно оси X, после данного пузырька.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setVerticalLineExtension, а возвращается с помощью метода getVerticalLineExtension.


Свойство содержит значение true, если линия уровня по оси X продолжается и после выделенного пузырька пузырьковой диаграммы, иначе - значение false.


По умолчанию свойство имеет значение true и работает при условии наличия вертикальной линии уровня, соответствующей выделенному пузырьку пузырьковой диаграммы относительно оси X, т.е. свойство [VerticalLine](ChartSelection.VerticalLine.htm) должно иметь значение true.
