# ChartSelection.HorizontalLineExtension


## Синтаксис


HorizontalLineExtension: Boolean


## Описание


Свойство HorizontalLineExtension определяет наличие горизонтальной линии уровня, соответствующей выделенному пузырьку пузырьковой диаграммы относительно оси Y, после данного пузырька.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHorizontalLineExtension, а возвращается с помощью метода getHorizontalLineExtension.


Свойство содержит значение true, если линия уровня по оси Y продолжается и после выделенного пузырька пузырьковой диаграммы, иначе - значение false.


По умолчанию свойство имеет значение true и работает при условии наличия горизонтальной линии уровня, соответствующей выделенному пузырьку пузырьковой диаграммы относительно оси Y, т.е. свойство [HorizontalLine](ChartSelection.HorizontalLine.htm) должно иметь значение true.
