# ChartSelection.YLabel


## Синтаксис


YLabel: PP.Ui.[ChartTooltip](../ChartTooltip/ChartTooltip.htm)


## Описание


Свойство YLabel** определяет метку на пересечении горизонтальной линии уровня, соответствующей выделенному пузырьку пузырьковой диаграммы, с осью Y.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setYLabel, а возвращается с помощью метода getYLabel.
