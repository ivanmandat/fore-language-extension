# ChartSelection.XLabel


## Синтаксис


XLabel: PP.Ui.[ChartTooltip](../ChartTooltip/ChartTooltip.htm)


## Описание


Свойство XLabel** определяет метку на пересечении вертикальной линии уровня, соответствующей выделенному пузырьку пузырьковой диаграммы, с осью X.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setXLabel, а возвращается с помощью метода getXLabel.
