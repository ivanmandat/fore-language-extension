# ChartSelection.Stroke


## Синтаксис

## Описание


Свойство Stroke** определяет
 кисть для линий окружности и уровней, соответствующих выделенному пузырьку
 пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setStroke,
 а возвращается с помощью метода getStroke.
