# ChartSelection.VerticalLine


## Синтаксис


VerticalLine: Boolean


## Описание


Свойство VerticalLine определяет наличие вертикальной линии уровня, соответствующей выделенному пузырьку пузырьковой диаграммы относительно оси X.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setVerticalLine, а возвращается с помощью метода getVerticalLine.


Свойство содержит значение true, если для выделенного пузырька пузырьковой диаграммы нарисована вертикальная линия уровня вдоль оси X, иначе - значение false.


По умолчанию свойство имеет значение true.
