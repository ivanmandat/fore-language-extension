# ChartSelection.Thickness


## Синтаксис


Thickness: Number


## Описание


Свойство Thickness** определяет для линий окружности и уровней, соответствующих выделенному пузырьку пузырьковой диаграммы, толщину в пикселях.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setThickness, а возвращается с помощью метода getThickness.
