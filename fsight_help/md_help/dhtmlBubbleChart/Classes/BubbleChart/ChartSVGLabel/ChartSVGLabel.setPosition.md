# ChartSVGLabel.setPosition


## Синтаксис


setPosition(left: Number, top: Number);


## Параметры


*left. Значение левого отступа;*


*top. Значение верхнего отступа.*


## Описание


Метод setPosition** устанавливает новое положение текстовой метки SVG-элемента пузырьковой диаграммы.
