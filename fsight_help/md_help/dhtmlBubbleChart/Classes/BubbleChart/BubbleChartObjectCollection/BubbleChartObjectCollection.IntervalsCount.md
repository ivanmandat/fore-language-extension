# BubbleChartObjectCollection.IntervalsCount


## Синтаксис


IntervalsCount: Number


## Описание


Свойство IntervalsCount** определяет общее количество интервалов между объектами из коллекции.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIntervalsCount, а возвращается с помощью метода getIntervalsCount.
