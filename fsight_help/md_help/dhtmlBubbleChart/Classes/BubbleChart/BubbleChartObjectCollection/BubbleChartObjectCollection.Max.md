# BubbleChartObjectCollection.Max


## Синтаксис


Max: Number


## Описание


Свойство Max** определяет максимальное значение объектов в коллекции.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMax, а возвращается с помощью метода getMax.
