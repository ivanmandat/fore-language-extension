# BubbleChartObjectCollection.IsVisible


## Синтаксис


IsVisible: Boolean


## Описание


Свойство IsVisible** определяет видимость объектов из коллекции.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsVisible, а возвращается с помощью метода getIsVisible.


Если свойство содержит значение true, то объекты из коллекции будут отображаться в пузырьковой диаграмме, если значение false, то не будут.
