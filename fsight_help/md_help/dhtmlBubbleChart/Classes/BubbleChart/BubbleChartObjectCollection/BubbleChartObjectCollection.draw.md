# BubbleChartObjectCollection.draw


## Синтаксис


draw(parentNode: SVGElement, visualType: [PP.Ui.VisualType](../../../Enums/VisualType.htm), secondCoord: Number);


## Параметры


*parentNode.* Контейнер, в котором требуется отрисовать объекты из коллекции. Необязательный параметр;


*visualType.* Тип оси пузырьковой диаграммы, из которой необходимо взять данные. Необязательный параметр;


*secondCoord.* Координата, которую не нужно считывать из данных пузырьковой диаграммы. Необязательный параметр. По умолчанию равен 0.


## Описание


Метод draw** рисует в пузырьковой диаграмме объекты из коллекции.
