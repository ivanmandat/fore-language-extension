# BubbleChartObjectCollection.Min


## Синтаксис


Min: Number


## Описание


Свойство Min** определяет минимальное значение объектов в коллекции.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMin, а возвращается с помощью метода getMin.
