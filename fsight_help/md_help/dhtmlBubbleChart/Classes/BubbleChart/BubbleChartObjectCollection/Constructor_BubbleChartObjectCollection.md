# Конструктор BubbleChartObjectCollection


## Синтаксис


PP.Ui.BubbleChartObjectCollection (settings);


## Параметры

## Описание


Конструктор BubbleChartObjectCollection создает экземпляр класса [BubbleChartObjectCollection](BubbleChartObjectCollection.htm).
