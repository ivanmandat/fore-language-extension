# ChartLineCollection.Grid


## Синтаксис


Grid: [PP.Ui.ChartGrid](../ChartGrid/ChartGrid.htm);


## Описание


Свойство Grid** устанавливает
 сетку пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setGrid,
 а возвращается с помощью метода getGrid.

Пример использования свойства аналогичен использованию свойства [ChartArea.Grid](../ChartArea/ChartArea.Grid.htm).
