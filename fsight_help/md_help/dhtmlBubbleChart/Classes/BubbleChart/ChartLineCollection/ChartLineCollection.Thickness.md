# ChartLineCollection.Thickness


## Синтаксис


Thickness: Number


## Описание


Свойство Thickness** определяет для линий уровня в коллекции толщину в пикселях.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setThickness, а возвращается с помощью метода getThickness.
