# ChartLineCollection.LineType


## Синтаксис


LineType: [PP.Ui.LineTypes](dhtmlUi.chm::/Enums/LineTypes.htm);


## Описание


Свойство LineType** определяет тип линий уровня в коллекции.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLineType, а возвращается с помощью метода getLineType.


По умолчанию значение свойства равно значению PP.Ui.LineTypes.Solid, соответствующему сплошной линии.
