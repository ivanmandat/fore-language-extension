# ChartLineCollection.Stroke


## Синтаксис

## Описание


Свойство Stroke** определяет
 для линий уровня в коллекции кисть со сплошной заливкой.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setStroke,
 а возвращается с помощью метода getStroke.
