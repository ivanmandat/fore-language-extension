# ChartLineCollection.SubLines


## Синтаксис


SubLines: [PP.Ui.ChartGridLineCollection](../ChartGridLineCollection/ChartGridLineCollection.htm)


## Описание


Свойство SubLines** определяет вложенные линии уровня для каждой линии из заданной коллекции.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSubLines, а возвращается с помощью метода getSubLines.
