# ChartGrid.BorderThickness


## Синтаксис


BorderThickness: Number


## Описание


Свойство BorderThickness** определяет для границы координатной сетки пузырьковой диаграммы толщину в пикселях.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setBorderThickness, а возвращается с помощью метода getBorderThickness.
