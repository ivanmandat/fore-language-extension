# ChartGrid.SmoothDisappear


## Синтаксис


SmoothDisappear: Boolean;


## Описание


Свойство SmoothDisappear определяет,
 будет ли координатная сетка пузырьковой диаграммы плавно исчезать при изменении масштаба.


## Комментарии


Допустимые значения:


	- true.
	 При изменении масштаба координатной сетки линии будут исчезать плавно;


	- false.
	 При изменении масштаба координатной сетки линии будут исчезать не
	 плавно (по умолчанию).


Значение свойства устанавливается из JSON и с помощью метода setSmoothDisappear,
 а возвращается с помощью метода getSmoothDisappear.
