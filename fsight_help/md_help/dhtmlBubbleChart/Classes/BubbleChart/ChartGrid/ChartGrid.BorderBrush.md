# ChartGrid.BorderBrush


## Синтаксис


BorderBrush: [PP.SolidColorBrush](dhtmlCommon.chm::/Classes/PP/SolidColorBrush/SolidColorBrush.htm)


## Описание


Свойство BorderBrush**
 определяет кисть со сплошной заливкой для границы координатной сетки пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setBorderBrush,
 а возвращается с помощью метода getBorderBrush.
