# ChartGrid.Left


## Синтаксис


Left: Boolean


## Описание


Свойство Left** определяет видимость левой границы координатной сетки пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLeft, а возвращается с помощью метода getLeft.


Свойство содержит значение true, если левая граница координатной сетки отображается, и значение false, если она не отображается.
