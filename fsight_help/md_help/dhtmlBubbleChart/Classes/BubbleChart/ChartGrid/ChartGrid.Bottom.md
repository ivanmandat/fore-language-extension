# ChartGrid.Bottom


## Синтаксис


Bottom: Boolean


## Описание


Свойство Bottom** определяет видимость нижней границы координатной сетки пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setBottom, а возвращается с помощью метода getBottom.


Свойство содержит значение true, если нижняя граница координатной сетки отображается, и значение false, если она не отображается.
