# ChartGrid.getLeftBorderWidth


## Синтаксис


getLeftBorderWidth();


## Описание


Метод getLeftBorderWidth возвращает
 ширину левой границы координатной сетки пузырьковой диаграммы.


## Комментарии


Метод возвращает значение типа Number.
