# ChartGrid.Top


## Синтаксис


Top: Boolean


## Описание


Свойство Top** определяет видимость верхней границы координатной сетки пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTop, а возвращается с помощью метода getTop.


Свойство содержит значение true, если верхняя граница координатной сетки отображается, и значение false, если она не отображается.
