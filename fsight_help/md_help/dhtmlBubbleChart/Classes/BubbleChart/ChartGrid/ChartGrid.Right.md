# ChartGrid.Right


## Синтаксис


Right: Boolean


## Описание


Свойство Right** определяет видимость правой границы координатной сетки пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setRight, а возвращается с помощью метода getRight.


Свойство содержит значение true, если правая граница координатной сетки отображается, и значение false, если она не отображается.
