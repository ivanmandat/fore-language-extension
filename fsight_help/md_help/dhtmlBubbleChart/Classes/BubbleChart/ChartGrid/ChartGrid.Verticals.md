# ChartGrid.Verticals


## Синтаксис


Verticals: [PP.Ui.ChartGridLineCollection](../ChartGridLineCollection/ChartGridLineCollection.htm)


## Описание


Свойство Verticals** определяет коллекцию вертикальных линий координатной сетки пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setVerticals, а возвращается с помощью метода getVerticals.
