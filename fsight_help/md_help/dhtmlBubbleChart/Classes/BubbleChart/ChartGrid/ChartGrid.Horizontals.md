# ChartGrid.Horizontals


## Синтаксис


Horizontals: [PP.Ui.ChartGridLineCollection](../ChartGridLineCollection/ChartGridLineCollection.htm)


## Описание


Свойство Horizontals** определяет коллекцию горизонтальных линий координатной сетки пузырьковой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHorizontals, а возвращается с помощью метода getHorizontals.
