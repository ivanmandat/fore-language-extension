# ChartTooltipBase.setPosition


## Синтаксис


setPosition(x: Number, y: Number, color: [PP.Color](dhtmlCommon.chm::/Classes/PP/Color/Color.htm),
 borderColor: [PP.Color](dhtmlCommon.chm::/Classes/PP/Color/Color.htm));


## Параметры


x - Координата X;


y - Координата Y;


color - Цвет фона подсказки;


borderColor - Цвет границы
 подсказки.


## Описание


Метод setPosition устанавливает
 новую позицию всплывающей подсказки.
