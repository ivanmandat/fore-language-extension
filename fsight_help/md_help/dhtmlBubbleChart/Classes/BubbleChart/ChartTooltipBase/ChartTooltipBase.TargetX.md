# ChartTooltipBase.TargetX


## Синтаксис


TargetX: Number;


## Описание


Свойство TargetX определяет координату X точки привязки всплывающей подсказки.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTargetX, а возвращается с помощью метода getTargetX.**
