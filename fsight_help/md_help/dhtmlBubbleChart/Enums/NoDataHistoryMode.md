# NoDataHistoryMode


Пространство имен: PP.Ui;


## Описание


Перечисление NoDataHistoryMode
 содержит режимы отображения истории пузырька, не имеющего данных по указанным
 измерениям в определённых точках временной шкалы.


Используется свойством [BubbleChart.HistoryMode](../Classes/BubbleChart/BubbleChart/BubbleChart.HistoryMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Break
		 Разрыв соединения с другими пузырьками.


		 Continue
		 Соединение с другими пузырьками.
