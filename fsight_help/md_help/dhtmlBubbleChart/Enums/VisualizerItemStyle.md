# VisualizerItemStyle


Пространство имен: PP.Ui;


## Описание


Перечисление VisualizerItemStyle
 содержит виды стилей визуальных элементов пузырьковой диаграммы.


Используется свойствами [BubbleChart.BubbleStyle](../Classes/BubbleChart/BubbleChart/BubbleChart.BubbleStyle.htm),
 [BubbleChart.HistoryBubbleStyle](../Classes/BubbleChart/BubbleChart/BubbleChart.HistoryBubbleStyle.htm)
 и [BubbleTree.BubbleStyle](dhtmlBubbleTree.chm::/Classes/BubbleTree/BubbleTree.BubbleStyle.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Transparent
		 Полупрозрачный стиль.


		 RadialGradient
		 Объёмный стиль с радиальным градиентом.


		 Indented
		 Вдавленный стиль.
