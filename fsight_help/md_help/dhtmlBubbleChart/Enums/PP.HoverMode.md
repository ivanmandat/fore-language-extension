# HoverMode


Пространство имен: PP;


## Описание


Перечисление HoverMode содержит
 события мыши, при наступлении которых выделяются или отображаются элементы
 компонента.


Используется следующими свойствами:


	- BubbleChart.SelectionMode;


	- [BubbleChart.ShowHistoryMode](../Classes/BubbleChart/BubbleChart/BubbleChart.ShowHistoryMode.htm);


	- [ChartTooltipBase.HoverMode](../Classes/BubbleChart/ChartTooltipBase/ChartTooltipBase.HoverMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Default
		 Поведение по умолчанию.


		 None
		 Элементы не отображаются/не выделяются ни при каких событиях
		 мыши.


		 Click
		 Элементы отображаются/выделяются по щелчку мыши.


		 MouseOver
		 Элементы отображаются/выделяются при наведении курсора.


		 MouseOverCenter
		 Элементы отображаются/выделяются при наведении курсора по
		 центру элемента.
