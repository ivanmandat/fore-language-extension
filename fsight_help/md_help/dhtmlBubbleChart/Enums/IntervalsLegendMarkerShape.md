# IntervalsLegendMarkerShape


Пространство имен: PP.Ui;


## Описание


Перечисление IntervalsLegendMarkerShape
 содержит виды фигур для маркера.


Используется свойством [ChartSymbol.MarkerShape](../Classes/BubbleChart/ChartSymbol/ChartSymbol.MarkerShape.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Circle
		 Круг.


		 Rectangle
		 Квадрат.


		 Rhombus
		 Ромб.


		 TriangleDown
		 Перевёрнутый треугольник.


		 TriangleUp
		 Треугольник.


		 LineCircle
		 Круг с горизонтальной линией.


		 LineRectangle
		 Квадрат с горизонтальной линией.


		 Dash
		 Штрих.


		 LineRhombus
		 Ромб с горизонтальной линией.


		 LineTriangleDown
		 Перевёрнутый треугольник с горизонтальной линией.


		 LineTriangleUp
		 Треугольник с горизонтальной линией.


		 LineTriangleLeft
		 Треугольник с горизонтальной линией повёрнутый налево.


		 LineTriangleRight
		 Треугольник с горизонтальной линией повёрнутый направо.


		 Line
		 Горизонтальная линия.


		 Custom
		 Специальная форма.
