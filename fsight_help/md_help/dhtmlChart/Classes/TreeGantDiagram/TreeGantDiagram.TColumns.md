# TreeGantDiagram.TColumns


## Синтаксис


TColumns: Array of [PP.Ui.TreeColumn](dhtmlUiTree.chm::/classes/TreeColumn/TreeColumn.htm);


## Описание


Свойство TColumns определяет
 колонки для дерева диаграммы Ганта.


## Комментарии


Колонки задаются массивом JSON-объектов с полями, соответствующими свойствам
 класса [PP.Ui.TreeColumn](dhtmlUiTree.chm::/classes/TreeColumn/TreeColumn.htm).
