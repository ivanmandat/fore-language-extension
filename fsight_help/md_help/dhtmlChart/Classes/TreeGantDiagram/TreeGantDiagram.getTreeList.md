# TreeGantDiagram.getTreeList


## Синтаксис


getTreeList(): [PP.Ui.TreeList](dhtmlUiTree.chm::/Classes/TreeList/TreeList.htm);


## Описание


Метод getTreeList возвращает
 компонент «[Дерево
 элементов](dhtmlUiTree.chm::/Components/TreeList/TreeList.htm)».


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 [TreeGantDiagram](TreeGantDiagram.htm) (см. «[Пример
 создания компонента TreeGantDiagram](../../Components/TreeGantDiagram/TreeGantExample.htm)»).


// Получим дерево элементов
GantD.getTreeList();
// -> PP.Ui.TreeList {_Nodes: Array[2], _ChildrenCount: null, _Columns: PP.Ui.TreeColumns, _ImageList: null, _ImageCentered: true…}

В результате выполнения примера было получено дерево элементов.
