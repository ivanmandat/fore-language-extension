# TreeGantDiagram.DTimelineStartTime


## Синтаксис


DTimelineStartTime: Date;


## Описание


Свойство DTimelineStartTime
 определяет начало временного отрезка диаграммы.


## Комментарии


Значение свойства устанавливается из JSON.
