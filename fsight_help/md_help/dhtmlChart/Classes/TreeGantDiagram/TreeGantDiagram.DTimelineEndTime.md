# TreeGantDiagram.DTimelineEndTime


## Синтаксис


DTimelineEndTime: Date;


## Описание


Свойство DTimelineEndTime определяет
 конец временного отрезка диаграммы.


## Комментарии


Значение свойства устанавливается из JSON.
