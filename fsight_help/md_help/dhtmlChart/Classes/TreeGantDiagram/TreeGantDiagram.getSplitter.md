# TreeGantDiagram.getSplitter


## Синтаксис


getSplitter(): [PP.Ui.Splitter](dhtmlUi.chm::/Classes/Splitter/Splitter.htm);


## Описание


Метод getSplitter возвращает
 компонент «[Разделитель](dhtmlUi.chm::/Components/Splitter/Splitter.htm)».


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 [TreeGantDiagram](TreeGantDiagram.htm) (см. «[Пример
 создания компонента TreeGantDiagram](../../Components/TreeGantDiagram/TreeGantExample.htm)»).


// Получим разделитель
GantD.getSplitter();
// -> PP.Ui.Splitter {_Events: Object, ValueChanged: PP.Delegate, SplitterPositionChanged: PP.Delegate, _IsRTL: false, OnContextMenu: PP.Delegate…}

В результате выполнения примера был получен разделитель.
