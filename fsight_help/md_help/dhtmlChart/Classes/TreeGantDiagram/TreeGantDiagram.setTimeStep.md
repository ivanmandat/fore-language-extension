# TreeGantDiagram.setTimeStep


## Синтаксис


setTimeStep(value);


## Параметры


value. Number. Значение.


## Описание


Метод setTimeStep устанавливает
 шаг шкалы времени в диаграмме Ганта.


## Комментарии


Значение по умолчанию: 1000 мс.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 [TreeGantDiagram](TreeGantDiagram.htm) (см. «[Пример
 создания компонента TreeGantDiagram](../../Components/TreeGantDiagram/TreeGantExample.htm)»).


// Установим шаг шкалы времени
GantD.setTimeStep(2000);

В результате выполнения примера шкала времени стала поделена на отрезки
 по 2 секунды.
