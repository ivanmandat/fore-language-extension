# RatingChartHeader.ExpandLastColumn


## Синтаксис


ExpandLastColumn: Boolean;


## Описание


Свойство ExpandLastColumn определяет,
 будет ли последний столбец с текстом перекрывать следующий столбец при
 нехватке места.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setExpandLastColumn,
 а возвращается с помощью метода getExpandLastColumn.


Допустимые значения:


	- true.
	 Последний столбец с текстом будет перекрывать следующий столбец при
	 нехватке места;


	- false.
	 Значение по умолчанию. Последний столбец с текстом не будет перекрывать
	 следующий столбец при нехватке места.
