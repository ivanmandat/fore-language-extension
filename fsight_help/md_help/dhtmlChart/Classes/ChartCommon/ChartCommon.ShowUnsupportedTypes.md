# ChartCommon.ShowUnsupportedTypes


## Синтаксис


ShowUnsupportedTypes: Boolean


## Описание


Свойство ShowUnsupportedTypes** определяет, отображены ли в списке типов диаграммы неподдерживаемые типы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setShowUnsupportedType, а возвращается с помощью метода getShowUnsupportedTypes.


Свойство содержит значение true, если в списке типов диаграммы отображены также неподдерживаемые типы, и значение false, если они не отображены.
