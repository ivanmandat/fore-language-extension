# EaxRadiusAngleChartView.Props


## Синтаксис


Props: Object;


## Описание


Свойство Props определяет параметры
 углового показателя.


## Комментарии


Свойство содержит объект со следующей структурой:


Props : {


  AnglePointIndex.
 Индекс точки (столбца), чьи данные используются для отображения угла.


  AnglePointVisible.
 Видимость точки. При значении false
 ряд, используемый для определения углового показателя будет скрыт.


};
