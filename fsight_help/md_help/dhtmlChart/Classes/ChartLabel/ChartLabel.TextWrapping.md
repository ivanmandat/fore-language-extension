# ChartLabel.TextWrapping


## Синтаксис


TextWrapping: [PP.Ui.TextWrapping](dhtmlUi.chm::/Enums/TextWrapping.htm);


## Описание


Свойство TextWrapping**
 определяет признак переноса текста метки.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTextWrapping,
 а возвращается с помощью метода getTextWrapping.
