# ChartLabel.ShowHandCursor


## Синтаксис


ShowHandCursor: Boolean


## Описание


Свойство ShowHandCursor** определяет признак отображения курсора «Рука» при его наведении на метку.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setShowHandCursor, а возвращается с помощью метода getShowHandCursor.


Если значение свойства - true, то при наведении курсора на метку будет отображен курсор «Рука», иначе будет отображен курсор по умолчанию.


По умолчанию свойство имеет значение false.
