# ChartLabel.Direction


## Синтаксис


Direction: PP.Ui.TextDirection


## Описание


Свойство Direction** определяет направление текста метки.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDirection, а возвращается с помощью метода getDirection.
