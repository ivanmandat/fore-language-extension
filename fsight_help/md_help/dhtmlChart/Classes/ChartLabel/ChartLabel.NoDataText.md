# ChartLabel.NoDataText


## Синтаксис


NoDataText: String


## Описание


Свойство NoDataText** определяет текст, отображаемый меткой при отсутствии в ней текстового содержимого.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setNoDataText, а возвращается с помощью метода getNoDataText.
