# RatingChartColumn.IsHeaderVisible


## Синтаксис


IsHeaderVisible: Boolean;


## Описание


Свойство IsHeaderVisible определяет
 признак видимости заголовка столбца.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsHeaderVisible,
 а возвращается с помощью метода getIsHeaderVisible.


Допустимые значения:


	- true.
	 Значение по умолчанию. Заголовок будет отображаться;


	- false.
	 Заголовок отображаться не будет.
