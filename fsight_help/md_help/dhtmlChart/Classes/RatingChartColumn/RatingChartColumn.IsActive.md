# RatingChartColumn.IsActive


## Синтаксис


IsActive: Boolean;


## Описание


Свойство IsActive определяет,
 является ли столбец активным.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsActive,
 а возвращается с помощью метода getIsActive.


Активным является столбец, по которому производится сортировка рядов
 данных рейтинговой диаграммы.


Допустимые значения:


	- true.
	 Столбец является активным;


	- false.
	 Столбец не является активным.
