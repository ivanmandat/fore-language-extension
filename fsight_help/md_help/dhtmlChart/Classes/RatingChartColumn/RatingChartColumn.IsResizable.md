# RatingChartColumn.IsResizable


## Синтаксис


IsResizable: Boolean;


## Описание


Свойство IsResizable определяет,
 можно ли изменять ширину столбца с помощью мыши.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsResizable,
 а возвращается с помощью метода getIsResizable.


Допустимые значения:


	- true.
	 Ширину столбца можно изменять с помощью мыши;


	- false.
	 Значение по умолчанию. Ширину столбца нельзя изменять с помощью мыши.
