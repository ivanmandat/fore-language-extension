# RatingChartColumn.Title


## Синтаксис


Title: String;


## Описание


Свойство Title определяет текст
 заголовка столбца.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTitle,
 а возвращается с помощью метода getTitle.
