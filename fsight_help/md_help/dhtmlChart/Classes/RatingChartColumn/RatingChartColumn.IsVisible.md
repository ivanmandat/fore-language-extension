# RatingChartColumn.IsVisible


## Синтаксис


IsVisible: Boolean;


## Описание


Свойство IsVisible определяет
 признак видимости столбца.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsVisible,
 а возвращается с помощью метода getIsVisible.


Допустимые значения:


	- true.
	 Значение по умолчанию. Столбец является видимым;


	- false.
	 Столбец является скрытым.
