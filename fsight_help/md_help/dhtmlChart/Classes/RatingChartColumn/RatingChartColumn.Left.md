# RatingChartColumn.Left


## Синтаксис


Left: Number;


## Описание


Свойство Left возвращает расстояние
 от левого края диаграммы до левой границы столбца.


## Комментарии


Свойство доступно только для чтения. Значение свойства возвращается
 с помощью метода getLeft.
