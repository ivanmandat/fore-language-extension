## Синтаксис

 String;


## Описание


Свойство IndexBrush определяет
 цвет заливки фона номера ряда данных.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIndexBrush,
 а возвращается с помощью метода getIndexBrush.
