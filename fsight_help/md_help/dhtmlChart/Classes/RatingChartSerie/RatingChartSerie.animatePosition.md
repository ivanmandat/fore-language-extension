# RatingChartSerie.animatePosition


## Синтаксис


animatePosition(step: Number);


## Параметры


step. Шаг анимации. Может
 принимать значения от «0» до «1».


## Описание


Метод animatePosition отрисовывает
 промежуточный шаг анимации изменения позиции ряда данных.


## Комментарии


Если параметр step принимает
 значение «0», позиция ряда данных отображается в начале анимации, если
 «1» - в конце.
