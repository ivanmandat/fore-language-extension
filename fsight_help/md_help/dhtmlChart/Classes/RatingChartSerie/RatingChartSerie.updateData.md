# RatingChartSerie.updateData


## Синтаксис


updateData(index: Number);


## Параметры


index. Шаг временной шкалы,
 для которого обновляются значения. Значение по умолчанию - «0».


## Описание


Метод updateData обновляет значения
 ряда данных.
