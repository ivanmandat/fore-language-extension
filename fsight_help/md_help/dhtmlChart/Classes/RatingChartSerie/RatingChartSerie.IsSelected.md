# RatingChartSerie.IsSelected


## Синтаксис


IsSelected: Boolean;


## Описание


Свойство IsSelected определяет,
 выделен ли ряд данных.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsSelected,
 а возвращается с помощью метода getIsSelected.


Допустимые значения:


	- true.
	 Ряд данных выделен;


	- false.
	 Ряд данных не выделен.
