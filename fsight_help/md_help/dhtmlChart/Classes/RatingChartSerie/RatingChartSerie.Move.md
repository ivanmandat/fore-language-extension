# RatingChartSerie.Move


## Синтаксис


Move: Number;


## Описание


Свойство Move возвращает изменение
 позиции ряда данных в рейтинге.


## Комментарии


Свойство доступно только для чтения. Значение свойства возвращается
 с помощью метода getMove.
