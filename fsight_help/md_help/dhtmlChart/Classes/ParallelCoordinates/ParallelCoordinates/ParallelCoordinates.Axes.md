# ParallelCoordinates.Axes


## Синтаксис


Axes: Array;


## Описание


Свойство Axes определяет оси
 диаграммы.


## Комментарии


Элементы массива - объекты класса [PP.Ui.PCAxis](../PCAxis/PCAxis.htm).


Значение свойства устанавливается из JSON и с помощью метода setAxes,
 а возвращается с помощью метода getAxes.


Если значение для свойства Axes
 не указано, то оси диаграммы создаются автоматически по шаблону, определённому
 в свойстве [ParallelCoordinates.Axis](ParallelCoordinates.Axis.htm).
