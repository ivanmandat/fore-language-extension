# ParallelCoordinates.FilterInactiveEffects


## Синтаксис


FilterInactiveEffects: Array;


## Описание


Свойство FilterInactiveEffects
 определяет эффекты для линий диаграммы, не входящих в диапазон фильтрации.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setFilterInactiveEffects,
 а возвращается с помощью метода getFilterInactiveEffects.
