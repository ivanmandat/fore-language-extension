# ParallelCoordinates.refreshView


## Синтаксис


refreshView(args: Object, fireEvent: Boolean);


## Параметры


args. Аргументы вызова события
 Refreshed;


fireEvent. Признак вызова события
 Refreshed. Если установлено значение
 true, то будет вызвано событие
 Refreshed.


## Описание


Метод refreshView обновляет
 отображение диаграммы.
