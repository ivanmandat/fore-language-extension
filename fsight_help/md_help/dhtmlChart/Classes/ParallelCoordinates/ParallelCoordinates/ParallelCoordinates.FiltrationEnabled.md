# ParallelCoordinates.FiltrationEnabled


## Синтаксис


FiltrationEnabled: Boolean;


## Описание


Свойство FiltrationEnabled определяет,
 включена ли фильтрация линий диаграммы.


## Комментарии


Допустимые значения:


	- true.
	 Фильтрация линий включена (по умолчанию);


	- false.
	 Фильтрация линий выключена.


Значение свойства устанавливается из JSON и с помощью метода setFiltrationEnabled,
 а возвращается с помощью метода getFiltrationEnabled.
