# ParallelCoordinates.LinesColors


## Синтаксис


LinesColors: Array;


## Описание


Свойство LinesColors определяет
 цвета линий.


## Комментарии


Элементы массива - объекты типа [PP.Color](dhtmlCommon.chm::/Classes/PP/Color/Color.htm)
 или строки со значением цвета в шестнадцатиричном формате.


Цвета из данного массива устанавливаются для линий, у которых не был
 указан цвет.


Значение свойства устанавливается из JSON и с помощью метода setLinesColors,
 а возвращается с помощью метода getLinesColors.
