# ParallelCoordinates.Lines


## Синтаксис


Lines: Array;


## Описание


Свойство Lines определяет линии
 диаграммы.


## Комментарии


Элементы массива - объекты класса [PP.Ui.PCLine](../PCLine/PCLine.htm).


Значение свойства устанавливается из JSON и с помощью метода setLines,
 а возвращается с помощью метода getLines.


Если значение для свойства Lines
 не указано, то линии диаграммы создаются автоматически по шаблону, определённому
 в свойстве [ParallelCoordinates.Line](ParallelCoordinates.Line.htm).
