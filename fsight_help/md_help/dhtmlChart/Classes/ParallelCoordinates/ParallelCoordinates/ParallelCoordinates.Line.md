# ParallelCoordinates.Line


## Синтаксис


Line: Object;


## Описание


Свойство Line определяет шаблон
 настроек для линий диаграммы.


## Комментарии


Значение свойства устанавливается из JSON, а возвращается с помощью
 метода getLine.


Свойство Line используется для
 автоматического создания линий диаграммы, если не задано значение
 для свойства [ParallelCoordinates.Lines](ParallelCoordinates.Lines.htm).
 Шаблон настроек представляет собой JSON-объект, поля которого являются
 свойствами класса [PCLine](../PCLine/PCLine.htm).
