# PCAxis.BaseThickness


## Синтаксис


BaseThickness: Number;


## Описание


Свойство BaseThickness определяет
 толщину основной линии оси.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setBaseThickness,
 а возвращается с помощью метода getBaseThickness.
