# PCAxis.Header


## Синтаксис


Header: [PP.Ui.ChartSVGLabel](PCAxis.Header.htm);


## Описание


Свойство Header определяет заголовок
 оси диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHeader,
 а возвращается с помощью метода getHeader.

Пример использования свойства приведен в описании свойства [PCAxis.RangeMaxLabel](PCAxis.RangeMaxLabel.htm).
