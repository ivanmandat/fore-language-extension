# PCAxis.RangeMinValue


## Синтаксис


RangeMinValue: Number;


## Описание


Свойство RangeMinValue определяет
 минимальное значение диапазона фильтрации.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setRangeMinValue,
 а возвращается с помощью метода getRangeMinValue.
