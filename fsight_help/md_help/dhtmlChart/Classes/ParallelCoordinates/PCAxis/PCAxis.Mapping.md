# PCAxis.Mapping


## Синтаксис


Mapping: PP.[DataMapping](dhtmlGraphicBase.chm::/Classes/PP/DataMapping/DataMapping.htm);


## Описание


Свойство Mapping определяет
 объект для сопоставления данных из источника элементам диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMapping,
 а возвращается с помощью метода getMapping.
