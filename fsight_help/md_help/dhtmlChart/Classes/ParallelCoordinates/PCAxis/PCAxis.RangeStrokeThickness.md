# PCAxis.RangeStrokeThickness


## Синтаксис


RangeStrokeThickness: Number;


## Описание


Свойство RangeStrokeThickness
 определяет толщину контура диапазона фильтрации.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setRangeStrokeThickness,
 а возвращается с помощью метода getRangeStrokeThickness.
