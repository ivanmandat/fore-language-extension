# PCAxis.RangeMinLabel


## Синтаксис


RangeMinLabel: [PP.Ui.ChartSVGLabel](dhtmlBubbleChart.chm::/Classes/BubbleChart/ChartSVGLabel/ChartSVGLabel.htm);


## Описание


Свойство RangeMinLabel определяет
 метку оси для минимального значения диапазона фильтрации.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setRangeMinLabel,
 а возвращается с помощью метода getRangeMinLabel.
