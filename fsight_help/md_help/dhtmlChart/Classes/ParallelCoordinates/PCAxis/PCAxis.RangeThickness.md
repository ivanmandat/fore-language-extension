# PCAxis.RangeThickness


## Синтаксис


RangeThickness: Number;


## Описание


Свойство RangeThickness определяет
 ширину диапазона фильтрации.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setRangeThickness,
 а возвращается с помощью метода getRangeThickness.
