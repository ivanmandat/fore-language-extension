# PCAxis.RangeMaxValue


## Синтаксис


RangeMaxValue: Number;


## Описание


Свойство RangeMaxValue определяет
 максимальное значение диапазона фильтрации.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setRangeMaxValue,
 а возвращается с помощью метода getRangeMaxValue.
