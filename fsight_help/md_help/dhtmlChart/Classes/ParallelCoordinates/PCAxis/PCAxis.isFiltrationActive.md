# PCAxis.isFiltrationActive


## Синтаксис


isFiltrationActive();


## Описание


Метод isFiltrationActive проверяет,
 задан ли диапазон фильтрации.


## Комментарии


Метод возвращает значение true,
 если задан диапазон фильтрации, иначе - значение false.
