# PCAxis.moveRangeBy


## Синтаксис


moveRangeBy(dy: Number);


## Параметры


dy. Величина сдвига диапазона
 фильтрации.


## Описание


Метод moveRangeBy сдвигает диапазон
 фильтрации по вертикали.
