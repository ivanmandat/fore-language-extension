# PCAxis.RangeStroke


## Синтаксис


RangeStroke: PP.[Color](dhtmlCommon.chm::/Classes/PP/Color/Color.htm)
 || String;


## Описание


Свойство RangeStroke определяет
 цвет контура диапазона фильтрации.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setRangeStroke,
 а возвращается с помощью метода getRangeStroke.
