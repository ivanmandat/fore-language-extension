# PCAxis.moveResizerTo


## Синтаксис


moveResizerTo(coords: Object);


## Параметры


coords. Объект с новым значением
 Y-координаты для границы диапазона фильтрации оси. Объект должен содержать
 поле Y.


## Описание


Метод moveResizerTo устанавливает
 новое значение Y-координаты верхней или нижней границы диапазона фильтрации.
