# PCVisual.Mappings


## Синтаксис


Mappings: Object;


## Описание


Свойство Mappings определяет
 справочник объектов для сопоставления данных из источника элементам диаграммы.


## Комментарии


Свойство содержит набор пар «ключ-значение», где ключ - это идентификатор
 сопоставления данных, а значение - наследник класса [PP.Visual](dhtmlGraphicBase.chm::/Classes/PP/Visual/Visual.htm).


Значение свойства устанавливается из JSON и с помощью метода setMappings,
 а возвращается с помощью метода getMappings.

Примет использования свойства приведен в описании свойства [ParallelCoordinates.Axis](../ParallelCoordinates/ParallelCoordinates.Axis.htm).
