# Конструктор PCVisual


## Синтаксис


PP.Ui.PCVisual(settings);


## Параметры


settings. JSON-объект со значениями
 свойств компонента.


## Описание


Конструктор PCVisual создает
 экземпляр класса PCVisual.

Примет использования конструктора приведен в описании свойства [ParallelCoordinates.Axis](../ParallelCoordinates/ParallelCoordinates.Axis.htm).
