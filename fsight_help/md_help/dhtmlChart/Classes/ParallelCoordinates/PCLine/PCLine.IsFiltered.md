# PCLine.IsFiltered


## Синтаксис


IsFiltered: Boolean;


## Описание


Свойство IsFiltered определяет,
 применён ли фильтр к линии.


## Комментарии


Допустимые значения:


	- true. К линии применён
	 фильтр;


	- false. По умолчанию. К линии не применён фильтр.


Значение свойства устанавливается из JSON, а возвращается с помощью
 метода getIsFiltered.


Свойство предназначено только для чтения.
