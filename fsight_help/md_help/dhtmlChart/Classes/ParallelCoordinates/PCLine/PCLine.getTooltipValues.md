# PCLine.getTooltipValues


## Синтаксис


getTooltipValues(noDataText: String, x: Number);


## Параметры


noDataText. Текст, который
 устанавливается, если не задано текстовое значение всплывающей подсказки.


x. Координата для получения
 переменных для осей слева и справа.


## Описание


Метод getTooltipValues возвращает
 значения переменных всплывающей подсказки линии.

Пример использования метода приведен в описании метода [PCLine.toggleTooltip](PCLine.toggleTooltip.htm).
