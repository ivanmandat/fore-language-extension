# PCObject.IsVisible


## Синтаксис


IsVisible: Boolean;


## Описание


Свойство IsVisible определяет,
 отображён ли элемент.


## Комментарии


Допустимые значения:


	- true.
	 Элемент отображен;


	- false.
	 Элемент не отображен.


Значение свойства устанавливается из JSON и с помощью метода setIsVisible,
 а возвращается с помощью метода getIsVisible.
