## Описание


Класс PCObject представляет
 собой базовый класс для визуальных элементов диаграммы с параллельными
 координатами.


## Синтаксис


PP.initClass(PP.Ui.PCObject, PP.[Object](dhtmlCommon.chm::/Classes/object/object.htm),
 "PP.Ui.PCObject");
