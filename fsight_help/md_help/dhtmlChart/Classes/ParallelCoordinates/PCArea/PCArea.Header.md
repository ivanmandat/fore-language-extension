# PCArea.Header


## Синтаксис


Header: [PP.Ui.ChartSVGLabel](dhtmlBubbleChart.chm::/Classes/BubbleChart/ChartSVGLabel/ChartSVGLabel.htm);


## Описание


Свойство Header устанавливает
 верхний колонтитул области отрисовки диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHeader,
 а возвращается с помощью метода getHeader.
