# Конструктор PCArea


## Синтаксис


PP.Ui.PCArea(settings);


## Параметры


settings. JSON-объект со значениями
 свойств компонента.


## Описание


Конструктор PCArea создает экземпляр
 класса [PCArea](PCArea.htm).
