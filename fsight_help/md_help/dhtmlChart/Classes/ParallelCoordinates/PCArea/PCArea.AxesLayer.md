# PCArea.AxesLayer


## Синтаксис


AxesLayer: PP.Ui.Canvas;


## Описание


Свойство AxesLayer определяет
 слой осей области рисования диаграммы.


## Комментарии


Значение свойства устанавливается из JSON, а возвращается с помощью
 метода getAxesLayer.
