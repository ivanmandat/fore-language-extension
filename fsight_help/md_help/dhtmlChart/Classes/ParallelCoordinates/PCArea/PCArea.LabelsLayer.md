# PCArea.LabelsLayer


## Синтаксис


LabelsLayer: SVGElement;


## Описание


Свойство LabelsLayer определяет
 слой меток области рисования диаграммы.


## Комментарии


Значение свойства устанавливается из JSON, а возвращается с помощью
 метода getLabelsLayer.
