# PCArea.LinesLayer


## Синтаксис


LinesLayer: PP.Ui.Canvas;


## Описание


Свойство LinesLayer определяет
 слой линий области рисования диаграммы.


## Комментарии


Значение свойства устанавливается из JSON, а возвращается с помощью
 метода getLinesLayer.
