# ChartColumnSerie.Preview


## Синтаксис


Preview: Canvas;


## Описание


Свойство Preview возвращает
 канву с предпросмотром шаблона.


## Комментарии


Значение свойства возвращается с помощью метода getPreview.


Свойство актуально при использовании [пользовательских
 шаблонов](DataAnalysis.chm::/Plugins/Custom/CustomDiagram.htm).


## Пример


Свойство предназначено для внутреннего использования.


Пример использования свойства приведен в описании свойства [ChartColumnSerie.Template](ChartColumnSerie.Template.htm).
