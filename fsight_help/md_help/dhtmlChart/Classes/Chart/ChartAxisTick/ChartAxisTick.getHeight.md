# ChartAxisTick.getHeight


## Синтаксис


getHeight();


## Описание


Метод getHeight возвращает высоту
 подписи.


## Пример


Для выполнения примера необходимо наличие экземпляра класса [ChartAxisTick](ChartAxisTick.htm)
 с наименованием «axisTick» (см. «[Конструктор
 ChartAxisTick](Constructor_ChartAxisTick.htm)»). Получим размеры подписи:


// Получаем размеры созданной подписи оси
console.log("Высота подписи: " + axisTick.getHeight());
console.log("Ширина подписи: " + axisTick.getWidth());
В результате в консоль были выведены размеры подписи:


Высота подписи: 43.92914094082653


Ширина подписи: 52.08750404236562
