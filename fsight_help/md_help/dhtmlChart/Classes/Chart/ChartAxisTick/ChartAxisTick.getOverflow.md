# ChartAxisTick.getOverflow


## Синтаксис


getOverflow();


## Описание


Метод getOverflow проверяет,
 выходит ли текстовое значение подписи за границы максимальной ширины подписи.


## Комментарии


Метод возвращает значение true,
 если текстовое значение выходит за границы максимальной ширины подписи,
 иначе - значение false.
