# ChartAxisTick.Left


## Синтаксис


Left: Number;


## Описание


Свойство Left определяет отступ
 подписи слева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLeft,
 а возвращается с помощью метода getLeft.
