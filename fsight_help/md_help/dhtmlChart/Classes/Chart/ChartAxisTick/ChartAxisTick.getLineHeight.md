# ChartAxisTick.getLineHeight


## Синтаксис


getLineHeight();


## Описание


Метод getLineHeight возвращает
 высоту текста подписи.


## Пример


Для выполнения примера необходимо наличие экземпляра класса [ChartAxisTick](ChartAxisTick.htm)
 с наименованием «axisTick» (см. «[Конструктор
 ChartAxisTick](Constructor_ChartAxisTick.htm)»). Получим высоту текста подписи:


// Получаем высоту текста подписи
console.log("Высота текста подписи: " + axisTick.getLineHeight());
В результате в консоль была выведена высота текста подписи:


Высота текста подписи: 24
