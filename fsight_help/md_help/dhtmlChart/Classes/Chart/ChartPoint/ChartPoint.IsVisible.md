# ChartPoint.IsVisible


## Синтаксис


IsVisible: Boolean;


## Описание


Свойство IsVisible определяет,
 отображается ли точка ряда данных диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsVisible,
 а возвращается с помощью метода getIsVisible.


Допустимые значения:


	- true.
	 Точка ряда данных диаграммы отображается (по умолчанию);


	- false.
	 Точка ряда данных диаграммы скрыта.
