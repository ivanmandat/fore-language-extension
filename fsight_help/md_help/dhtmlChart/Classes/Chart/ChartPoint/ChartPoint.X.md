# ChartPoint.X


## Синтаксис


X: Number;


## Описание


Свойство X определяет значение
 точки ряда данных диаграммы по оси X.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setX,
 а возвращается с помощью метода getX.
