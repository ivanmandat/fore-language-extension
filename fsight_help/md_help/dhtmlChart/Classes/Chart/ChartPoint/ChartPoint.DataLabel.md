# ChartPoint.DataLabel


## Синтаксис


DataLabel: [PP.Ui.ChartText](../ChartText/ChartText.htm);


## Описание


Свойство DataLabel определяет
 подпись данных точки ряда данных диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDataLabel,
 а возвращается с помощью метода getDataLabel.
