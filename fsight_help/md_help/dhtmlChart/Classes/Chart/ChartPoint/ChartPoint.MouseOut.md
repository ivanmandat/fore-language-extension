# ChartPoint.MouseOut


## Синтаксис


MouseOut: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие MouseOut наступает после
 покидания курсором мыши точки ряда данных диаграммы.
