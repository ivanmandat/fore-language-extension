# ChartPoint.Y


## Синтаксис


Y: Number;


## Описание


Свойство Y определяет значение
 точки ряда данных диаграммы по оси Y.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setY,
 а возвращается с помощью метода getY.
