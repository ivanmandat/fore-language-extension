# ChartPoint.MouseOver


## Синтаксис


MouseOver: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие MouseOver наступает
 после наведения курсором мыши на точку ряда данных диаграммы.
