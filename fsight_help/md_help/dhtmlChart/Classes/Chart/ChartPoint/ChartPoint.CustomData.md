# ChartPoint.CustomData


## Синтаксис


CustomData: Number | String | Object;


## Описание


Свойство CustomData предназначено
 для хранения любых пользовательских данных.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setCustomData,
 а возвращается с помощью метода getCustomData.
