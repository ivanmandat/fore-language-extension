# ChartPoint.IsFictive


## Синтаксис


IsFictive: Boolean;


## Описание


Свойство IsFictive определяет,
 является ли точка ряда данных диаграммы фиктивной.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsFictive,
 а возвращается с помощью метода getIsFictive.


Допустимые значения:


	- true.
	 Точка является фиктивной;


	- false.
	 Точка не является фиктивной (по умолчанию).


Фиктивная точка используется для продления ряда прогнозных данных.
