# ChartPoint.ColorValue


## Синтаксис


ColorValue: Number;


## Описание


Свойство ColorValue определяет
 прозрачность заливки точки ряда данных диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setColorValue,
 а возвращается с помощью метода getColorValue.


Допустимые значения свойства: от 0 до 1.
