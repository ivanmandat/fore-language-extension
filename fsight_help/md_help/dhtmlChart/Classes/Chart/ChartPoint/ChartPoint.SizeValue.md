# ChartPoint.SizeValue


## Синтаксис


SizeValue: Object;


## Описание


Свойство SizeValue определяет
 размеры точки ряда данных диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSizeValue,
 а возвращается с помощью метода getSizeValue.
