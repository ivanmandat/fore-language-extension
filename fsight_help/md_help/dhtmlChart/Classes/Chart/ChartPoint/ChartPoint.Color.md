# ChartPoint.Color


## Синтаксис


Color: String | [PP.SolidColorBrush](dhtmlCommon.chm::/Classes/PP/SolidColorBrush/SolidColorBrush.htm);


## Описание


Свойство Color определяет цвет
 заливки точки ряда данных диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setColor,
 а возвращается с помощью метода getColor.
