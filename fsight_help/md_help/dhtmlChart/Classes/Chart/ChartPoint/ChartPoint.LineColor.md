# ChartPoint.LineColor


## Синтаксис


LineColor: String | [PP.SolidColorBrush](dhtmlCommon.chm::/Classes/PP/SolidColorBrush/SolidColorBrush.htm);


## Описание


Свойство LineColor определяет
 цвет границы точки ряда данных диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLineColor,
 а возвращается с помощью метода getLineColor.
