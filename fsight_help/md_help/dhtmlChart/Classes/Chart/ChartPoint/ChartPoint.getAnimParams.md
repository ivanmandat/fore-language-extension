# ChartPoint.getAnimParams


## Синтаксис


getAnimParams();


## Описание


Метод getAnimParams возвращает
 параметры анимации точки ряда данных.


## Комментарии


Метод возвращает JSON-объект со следующими полями: PlotX - экранная
 координата точки ряда данных диаграммы по оси X, PlotY - экранная координата
 для первого значения точки по оси Y, PlotYBottom - экранная координата
 для второго значения точки по оси Y.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 [Chart](../../../Components/Chart/Chart.htm)
 с наименованием «chart» (см. «[Пример
 создания гистограммы с накоплением](../../../Components/Chart/ChartPoint.htm)»). Определим экранные координаты
 первой точки второго ряда данных диаграммы до и после сброса параметров
 анимации данной точки:


// Выводим параметры анимации точки
function printAnimParams(point) {
    var params = point.getAnimParams();
    console.log("  Экранная координата точки по оси X: %s", params.PlotX);
    console.log("  Первое значение координаты точки по оси Y: %s", params.PlotY);
    console.log("  Второе значение координаты точки по оси Y: %s", params.PlotYBottom);
};
// Получаем второй ряд данных
var serie = chart.getSeries()[1];
// Получаем первую точку ряда
var point = serie.getPoints()[0];
// Выводим параметры анимации точки
console.log("Исходные параметры анимации точки:");
printAnimParams(point);
// Сбрасываем параметры анимации
point.resetAnimParams();
console.log("Параметры анимации точки после их сброса:");
printAnimParams(point);
