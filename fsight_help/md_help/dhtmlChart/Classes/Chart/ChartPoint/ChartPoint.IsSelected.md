# ChartPoint.IsSelected


## Синтаксис


IsSelected: Boolean;


## Описание


Свойство IsSelected определяет,
 выделена ли точка ряда данных диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsSelected,
 а возвращается с помощью метода getIsSelected.


Допустимые значения:


	- true.
	 Точка ряда данных диаграммы выделена;


	- false.
	 Точка ряда данных диаграммы не выделена (по умолчанию).
