# ChartPoint.Parent


## Синтаксис


Parent: [PP.Ui.ChartCanvasSerie](../ChartCanvasSerie/ChartCanvasSerie.htm);


## Описание


Свойство Parent определяет ряд
 данных диаграммы, к которому принадлежит точка.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setParent,
 а возвращается с помощью метода getParent.
