# ChartPoint.IsAfterNull


## Синтаксис


IsAfterNull: Boolean;


## Описание


Свойство IsAfterNull определяет
 признак того, что данная точка ряда данных диаграммы располагается после
 точки с пустым значением.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsAfterNull,
 а возвращается с помощью метода getIsAfterNull.


Допустимые значения:


	- true.
	 Точка ряда данных диаграммы располагается после точки с пустым значением;


	- false.
	 Точка ряда данных диаграммы не располагается после точки с пустым
	 значением (по умолчанию).
