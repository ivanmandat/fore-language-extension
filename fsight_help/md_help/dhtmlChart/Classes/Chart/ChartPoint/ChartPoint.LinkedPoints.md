# ChartPoint.LinkedPoints


## Синтаксис


LinkedPoints: Array;


## Описание


Свойство LinkedPoints определяет
 массив связанных точек ряда данных.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLinkedPoints,
 а возвращается с помощью метода getLinkedPoints.


Свойство содержит массив объектов типа [PP.Ui.ChartPoint](ChartPoint.htm).
