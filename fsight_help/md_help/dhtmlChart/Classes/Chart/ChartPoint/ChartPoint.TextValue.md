# ChartPoint.TextValue


## Синтаксис


TextValue: String;


## Описание


Свойство TextValue определяет
 текстовое значения для точки ряда данных диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTextValue,
 а возвращается с помощью метода getTextValue.
