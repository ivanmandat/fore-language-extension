# ChartPoint.getOriginalLineColorIsRGBA


## Синтаксис


getOriginalLineColorIsRGBA();


## Описание


Метод getOriginalLineColorIsRGBA
 возвращает признак того, задан ли цвет границы у точки ряда данных диаграммы
 в виде строки в формате RGBA.


## Комментарии


Метод возвращает значение типа Boolean.


Допустимые значения:


	- true. Цвет границы у
	 точки ряда данных диаграммы задан в виде строки в формате RGBA;


	- false. Цвет границы
	 у точки ряда данных диаграммы не задан в виде строки в формате RGBA.
