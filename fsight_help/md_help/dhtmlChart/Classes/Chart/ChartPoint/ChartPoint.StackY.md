# ChartPoint.StackY


## Синтаксис


StackY: Number;


## Описание


Свойство StackY определяет второе
 значение точки ряда данных диаграммы по оси Y.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setStackY,
 а возвращается с помощью метода getStackY.
