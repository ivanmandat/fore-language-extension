# ChartPoint.PercentPart


## Синтаксис


PercentPart: Number;


## Описание


Свойство PercentPart определяет
 отношение второго значения точки ряда данных диаграммы по оси Y к высоте
 данной оси.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setPercentPart,
 а возвращается с помощью метода getPercentPart.
