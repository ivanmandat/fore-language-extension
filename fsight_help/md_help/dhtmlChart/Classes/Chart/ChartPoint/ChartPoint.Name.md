# ChartPoint.Name


## Синтаксис


Name: String;


## Описание


Свойство Name определяет наименование точки
 ряда данных диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setName,
 а возвращается с помощью метода getName.
