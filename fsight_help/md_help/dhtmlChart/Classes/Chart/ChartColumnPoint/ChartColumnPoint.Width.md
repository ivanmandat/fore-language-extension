# ChartColumnPoint.Width


## Синтаксис


Width: Number;


## Описание


Свойство Width определяет ширину
 столбца гистограммы, соответствующего точке ряда данных.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setWidth,
 а возвращается с помощью метода getWidth.
