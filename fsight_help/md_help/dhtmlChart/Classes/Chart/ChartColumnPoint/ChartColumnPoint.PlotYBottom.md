# ChartColumnPoint.PlotYBottom


## Синтаксис


PlotYBottom: Number;


## Описание


Свойство PlotYBottom определяет
 экранную координату второго значения по оси Y для точки ряда данных.


## Комментарии


Свойство предназначено только для чтения.
