# Chart.ZAxis


## Синтаксис


ZAxis: [PP.Ui.ChartCanvasAxis](../ChartCanvasAxis/ChartCanvasAxis.htm);


## Описание


Свойство ZAxis определяет ось
 глубины диаграммы.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm)
 или с помощью метода setZAxis,
 а возвращается с помощью метода getZAxis.


Используется при отображении [объёмного
 вида диаграммы](UiDiagrams.chm::/Params_diagram/Setup_3DView.htm) (свойство [Chart.Is3D](Chart.Is3D.htm)
 принимает значение true).


Для определения длины оси Z используйте свойство [Chart.Deep](Chart.Deep.htm).
