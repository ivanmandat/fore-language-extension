# Chart.Polar


## Синтаксис


Polar: Boolean;


## Описание


Свойство Polar определяет признак
 того, использует ли диаграмма полярную систему координат.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm),
 а возвращается с помощью метода getPolar.


Допустимые значения:


	- true.
	 Диаграмма использует полярную систему координат;


	- false.
	 По умолчанию. Диаграмма не использует полярную систему координат.
