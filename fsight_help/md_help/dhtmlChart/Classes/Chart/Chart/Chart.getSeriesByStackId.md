# Chart.getSeriesByStackId


## Синтаксис


getSeriesByStackId(stackId: String, countInvisible:
 Boolean);


## Параметры


stackId. Уникальный идентификатор
 накоплений диаграммы;


countInvisible. Определение
 способа формирования массива рядов данных по уникальному идентификатору
 накоплений диаграммы: с учётом скрытых рядов данных или без учёта.


## Описание


Метод getSeriesByStackId возвращает
 массив рядов данных по уникальному идентификатору накоплений диаграммы.


## Комментарии


Для определения идентификатора накопления диаграммы используйте свойство
 [ChartCanvasSerie.StackId](../ChartCanvasSerie/ChartCanvasSerie.StackId.htm).
 Например, если два ряда данных имеют идентификатор Absolute, а один -
 Procent, то при вызове метода с идентификатором Absolute будет возвращён
 массив, содержащий два ряда данных.


Допустимые значения параметра countInvisible:


	- true. По умолчанию.
	 Скрытые ряды данных учитываются при формировании массива рядов данных
	 по уникальному идентификатору накоплений диаграммы;


	- false. Скрытые ряды
	 данных не учитываются при формировании массива рядов данных по уникальному
	 идентификатору накоплений диаграммы.
