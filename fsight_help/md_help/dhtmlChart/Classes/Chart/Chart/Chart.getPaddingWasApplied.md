# Chart.getPaddingWasApplied


## Синтаксис


getPaddingWasApplied();


## Описание


Метод getPaddingWasApplied возвращает
 объект, содержащий признаки применения отступов диаграммы.


## Комментарии


Метод возвращает объект, в котором содержатся следующие поля:


	- Bottom.
	 Признак применения нижнего отступа диаграммы;


	- Left.
	 Признак применения левого отступа диаграммы;


	- Right.
	 Признак применения правого отступа диаграммы;


	- Top.
	 Признак применения верхнего отступа диаграммы.


Поля возвращаемого объекта имеют тип Boolean.


Допустимые значения полей возвращаемого объекта:


	- true.
	 Отступ применен;


	- false.
	 Отступ не применен.
