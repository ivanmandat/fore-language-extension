# Chart.PaddingTop


## Синтаксис


PaddingTop: Number;


## Описание


Свойство PaddingTop определяет
 верхний отступ диаграммы.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm),
 а возвращается с помощью метода getPaddingTop.


По умолчанию значение свойства равно 10.
