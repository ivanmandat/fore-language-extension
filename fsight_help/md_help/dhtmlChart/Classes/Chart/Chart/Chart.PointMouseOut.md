# Chart.PointMouseOut


## Синтаксис


PointMouseOut: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Допустимый
 аргумент - Point. Представляет
 собой точку ряда данных, покинутую курсором мыши.


## Описание


Событие PointMouseOut наступает
 после покидания курсором мыши точки ряда данных диаграммы.

 создания гистограммы с накоплением](../../../Components/Chart/ChartPoint.htm).
