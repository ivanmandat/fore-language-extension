# Chart.Title


## Синтаксис


Title: {


    settings


};


## Параметры


settings. JSON-объект с параметрами
 отображения заголовка диаграммы.


## Описание


Свойство Title определяет параметры
 заголовка диаграммы.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm)
 или с помощью метода setTitle,
 а возвращается с помощью метода getTitle.


Для JSON-объекта параметра settings
 доступны следующие параметры:


	- AutoMargin. Тип параметра:
	 Boolean. Автоматический расчёт отступов заголовка диаграммы от краев
	 рабочего пространства (сверху, слева и справа) и от краев области
	 построения (снизу):


		- True. Отступы между
		 компонентами рассчитываются автоматически;


		- False. Отступы между
		 компонентами зависят от заданных параметров прямоугольника Rect;


	- AutoPlaced. Тип параметра:
	 Boolean. Автоматическое размещение заголовка диаграммы:


		- True. Заголовок
		 диаграммы размещается автоматически;


		- False. Размещение
		 заголовка диаграммы зависит от заданных параметров прямоугольника
		 Rect;


	- Rect. Параметры прямоугольника,
	 в котором будет расположен заголовок диаграммы:


		- BottomMargin. Тип
		 параметра: Number. Размер нижнего отступа;


		- Height. Тип параметра:
		 Number. Ширина прямоугольника;


		- LeftMargin. Тип
		 параметра: Number. Размер левого отступа;


		- RightMargin. Тип
		 параметра: Number. Размер правого отступа;


		- TopMargin. Тип параметра:
		 Number. Размер верхнего отступа;


		- Width. Тип параметра:
		 Number. Высота прямоугольника;


		- X. Тип параметра:
		 Number. Размер отступа относительно левой точки диаграммы по горизонтали;


		- Y. Тип параметра:
		 Number. Размер отступа относительно верхней точки диаграммы по
		 вертикали;


Примечание.
 Параметр используется, если AutoMargin
 и AutoPlaced принимают значение
 False.


	- Style. Параметры стиля
	 заголовка диаграммы:


		- BackgroundColor.
		 Цвет фона заголовка диаграммы;


		- BorderColor. Цвет
		 границы заголовка диаграммы;


		- BorderWidth. Толщина
		 границы заголовка диаграммы в пикселях;


		- Color. Цвет текста
		 заголовка диаграммы;


		- FontFamily. Шрифт
		 текст заголовка диаграммы;


		- FontSize. Размер
		 шрифта;


		- FontWeight. Начертание
		 шрифта;


	- Text. Тип параметра:
	 String. Текст заголовка диаграммы;


	- TextAlign. Тип параметра:
	 String. Способ выравнивания текста заголовка диаграммы:


		- Left. Выравнивание
		 текста по левому краю;


		- Center. Выравнивание
		 текста по центру;


		- Right. Выравнивание
		 текста по правому краю;


	- TextAngle. Тип параметра:
	 Number. Угол поворота заголовка диаграммы в градусах.
