# Chart.UseIndicatorColorForToolTipBorder


## Синтаксис


UseIndicatorColorForToolTipBorder: Boolean;


## Описание


Свойство UseIndicatorColorForToolTipBorder
 определяет окрашивание границы всплывающей подсказки в цвет ряда.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm)
 или с помощью метода setUseIndicatorColorForToolTipBorder,
 а возвращается с помощью метода getUseIndicatorColorForToolTipBorder.


Допустимые значения:


	- true. По умолчанию.
	 Цвет ряда используется в качестве цвета границы всплывающей подсказки.


	- false. Цвет всплывающей
	 подсказки задается независимо от цвета ряда.
