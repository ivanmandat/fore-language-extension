# Chart.UseSelectOnRightClick


## Синтаксис


UseSelectOnRightClick: Boolean;


## Описание


Свойство UseSelectOnRightClick определяет
 признак того, будет ли на диаграмме применяться выделение по правому щелчку
 мыши.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm)
 или с помощью метода setUseSelectOnRightClick,
 а возвращается с помощью метода getUseSelectOnRightClick.


Допустимые значения:


	- true.
	 По умолчанию. На диаграмме будет применяться выделение по правому
	 щелчку мыши;


	- false.
	 На диаграмме не будет применяться выделение по правому щелчку мыши.
