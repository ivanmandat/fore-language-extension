# Chart.ToolTipFormatter


## Синтаксис


ToolTipFormatter: String;


## Описание


Свойство ToolTipFormatter определяет
 текстовую маску всплывающей подсказки.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm)
 или с помощью метода setToolTipFormatter,
 а возвращается с помощью метода getToolTipFormatter.


По умолчанию содержит значение «%Autovalue».
