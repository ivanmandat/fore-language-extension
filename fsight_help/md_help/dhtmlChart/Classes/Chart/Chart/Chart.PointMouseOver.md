# Chart.PointMouseOver


## Синтаксис


PointMouseOver: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Допустимый
 аргумент - Point. Представляет
 собой точку ряда данных, на которую был наведён курсор мыши.


## Описание


Событие PointMouseOver наступает
 после наведения курсором мыши на точку ряда данных диаграммы.

 создания гистограммы с накоплением](../../../Components/Chart/ChartPoint.htm).
