# Chart.getPieSeriesCount


## Синтаксис


getPieSeriesCount(countInvisible: Boolean);


## Параметры


countInvisible. Определение
 способа расчёта количества рядов данных: с учётом скрытых рядов данных
 или без учёта.


## Описание


Метод getPieSeriesCount возвращает
 количество рядов данных [круговой
 диаграммы](UiDiagrams.chm::/Type_diagrams/UiDiagrams_round.htm).


## Комментарии


Допустимые значения параметра countInvisible:


	- true. По умолчанию.
	 Скрытые ряды данных учитываются при расчёте количества рядов данных
	 круговой диаграммы;


	- false. Скрытые ряды
	 данных не учитываются при расчёте количества рядов данных круговой
	 диаграммы.
