# Chart.PaddingBottom


## Синтаксис


PaddingBottom: Number;


## Описание


Свойство PaddingBottom определяет
 нижний отступ диаграммы.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm),
 а возвращается с помощью метода getPaddingBottom.


По умолчанию значение свойства равно 15.
