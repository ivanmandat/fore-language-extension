# Chart.getChartBounds


## Синтаксис


getChartBounds();


## Описание


Метод getChartBounds возвращает
 прямоугольную область контейнера, в котором расположена диаграмма.


## Комментарии


Метод возвращает значение типа [PP.Rect](dhtmlCommon.chm::/Classes/PP/Rect/Rect.htm).
