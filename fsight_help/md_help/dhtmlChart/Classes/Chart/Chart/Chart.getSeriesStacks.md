# Chart.getSeriesStacks


## Синтаксис


getSeriesStacks(filter: String);


## Параметры


filter. Тип диаграммы, для
 которой пересчитываются накопления.


## Описание


Метод getSeriesStacks пересчитывает
 все накопления диаграммы для дальнейших расчетов.


## Комментарии


Допустимые значения параметра filter:


	- column. Гистограмма;


	- linear. По умолчанию.
	 Линейная диаграмма.


Метод возвращает объект со следующими полями:


	- Primary.
	 Массив, содержащий суммы значений рядов данных для каждой точки оси
	 X, если используются значения рядов данных по основной оси значений;


	- Secondary.
	 Массив, содержащий суммы значений рядов данных для каждой точки оси
	 X, если используются значения рядов данных по дополнительной оси значений.
