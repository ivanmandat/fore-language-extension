## Описание


Свойство Legend определяет легенду
 диаграммы.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm)
 или с помощью метода setLegend,
 а возвращается с помощью метода getLegend.
