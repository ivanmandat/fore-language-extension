# Chart.InnerCircleSize


## Синтаксис


InnerCircleSize: Number;


## Описание


Свойство InnerCircleSize определяет
 размер отступа внутренней границы от центра [круговой
 диаграммы](UiDiagrams.chm::/Type_diagrams/UiDiagrams_round.htm).


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm)
 или с помощью метода setInnerCircleSize,
 а возвращается с помощью метода getInnerCircleSize.


По умолчанию размер отступа внутренней границы от центра круговой диаграммы
 0% - полностью залитый круг. Если свойство принимает значение, отличное
 от нуля, то будет отображаться кольцевая диаграмма.


Для указания размера отступа внутренней границы от центра круговой диаграммы
 в пикселях используйте свойство [Chart.InnerCircleSizeInPercent](Chart.InnerCircleSizeInPercent.htm)
 со значением false.


Для определения размера внешней границы круговой диаграммы используйте
 свойство [Chart.CircleSize](Chart.CircleSize.htm).
