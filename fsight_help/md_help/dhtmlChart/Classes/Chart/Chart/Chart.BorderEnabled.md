# Chart.BorderEnabled


## Синтаксис


BorderEnabled: Boolean;


## Описание


Свойство BorderEnabled определяет
 отображение границы диаграммы.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm)
 или с помощью метода setBorderEnabled.


Допустимые значения:


	- true. Граница диаграммы
	 отображается;


	- false. По умолчанию.
	 Граница диаграммы не отображается.
