# Chart.CircleSize


## Синтаксис


CircleSize: Number;


## Описание


Свойство CircleSize определяет
 размер внешней границы [круговой
 диаграммы](UiDiagrams.chm::/Type_diagrams/UiDiagrams_round.htm).


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm),
 а возвращается с помощью метода getCircleSize.


По умолчанию размер внешней границы 100% - полностью залитый круг. Если
 свойство [Chart.InnerCircleSize](Chart.InnerCircleSize.htm)
 принимает значение, отличное от нуля, то будет отображаться кольцевая
 диаграмма.


Для указания размера внешней границы круговой диаграммы в пикселях используйте
 свойство [Chart.CircleSizeInPercent](Chart.CircleSizeInPercent.htm)
 со значением false.
