# Chart.IsManhattan


## Синтаксис


IsManhattan: Boolean;


## Описание


Свойство IsManhattan устанавливает
 тип диаграммы - [объёмная
 гистограмма с группами](UiDiagrams.chm::/Type_diagrams/3DHistogram_with_Groups.htm).


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm),
 а возвращается с помощью метода getIsManhattan.


Допустимые значения:


	- true. Отображается объёмная
	 гистограмма с группами;


	- false. По умолчанию.
	 Отображается текущий тип диаграммы, заданный в свойстве [Chart.Type](Chart.Type.htm).
