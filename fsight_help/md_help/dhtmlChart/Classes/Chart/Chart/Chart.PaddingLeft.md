# Chart.PaddingLeft


## Синтаксис


PaddingLeft: Number;


## Описание


Свойство PaddingLeft определяет
 левый отступ диаграммы.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm),
 а возвращается с помощью метода getPaddingLeft.


По умолчанию значение свойства равно 10.
