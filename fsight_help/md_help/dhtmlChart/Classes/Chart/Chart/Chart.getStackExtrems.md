# Chart.getStackExtrems


## Синтаксис


getStackExtrems();


## Описание


Метод getStackExtrems возвращает
 рассчитанные значения накоплений диаграммы.


## Комментарии


Метод возвращает объект со следующими полями:


	- Primary.
	 Массив, содержащий суммы значений рядов данных для каждой точки оси
	 X, если используются значения рядов данных по основной оси значений;


	- Secondary.
	 Массив, содержащий суммы значений рядов данных для каждой точки оси
	 X, если используются значения рядов данных по дополнительной оси значений.


 Значения в массивах представляют собой объекты, содержащие целочисленное
 поле sum.


Массивы Primary и Secondary также содержат поля max и min, которые хранят
 максимальное и минимальное значения рядов данных с учетом накоплений.
