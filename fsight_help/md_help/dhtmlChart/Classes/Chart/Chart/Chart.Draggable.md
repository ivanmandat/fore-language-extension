# Chart.Draggable


## Синтаксис


Draggable: Boolean;


## Описание


Свойство Draggable определяет
 признак того, допускается ли перетаскивание диаграммы.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm),
 а возвращается с помощью метода getDraggable.


Допустимые значения:


	- true.
	 Перетаскивание диаграммы допускается;


	- false.
	 По умолчанию. Перетаскивание диаграммы не допускается.
