# Chart.removeSerie


## Синтаксис


removeSerie(serie: [PP.Ui.ChartCanvasSerie](../ChartCanvasSerie/ChartCanvasSerie.htm),
 redraw: Boolean);


## Параметры


serie. Ряд данных диаграммы,
 который требуется удалить;


redraw. Признак необходимости
 перерисовки диаграммы.


## Описание


Метод removeSerie удаляет указанный
 ряд данных диаграммы.


## Комментарии


Допустимые значения параметра redraw:


	- true. По умолчанию.
	 Необходима перерисовка диаграммы;


	- false. Перерисовка диаграммы
	 не требуется.
