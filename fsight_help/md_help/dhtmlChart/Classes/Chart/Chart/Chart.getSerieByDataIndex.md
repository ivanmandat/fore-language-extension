# Chart.getSerieByDataIndex


## Синтаксис


getSerieByDataIndex(dataIndex: Number);


## Параметры


dataIndex. Индекс ряда данных.


## Описание


Метод getSerieByDataIndex возвращает
 ряд данных диаграммы по индексу.


## Комментарии


Ряды данных отображаются на диаграмме по порядку слева направо. Индекс
 первого ряда данных - 0.
