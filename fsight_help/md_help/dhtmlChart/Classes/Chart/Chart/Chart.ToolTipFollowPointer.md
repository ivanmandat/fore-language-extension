# Chart.ToolTipFollowPointer


## Синтаксис


ToolTipFollowPointer: Boolean;


## Описание


Свойство ToolTipFollowPointer
 определяет поведение всплывающей подсказки при перемещении курсора мыши.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm)
 или с помощью метода setToolTipFollowPointer,
 а возвращается с помощью метода getToolTipFollowPointer.


Допустимые значения:


	- true. По умолчанию.
	 Всплывающая подсказка будет перерисована на новом месте, при перемещении
	 курсора.


	- false. Всплывающая подсказка
	 не будет перерисована при перемещении курсора в рамках текущего объекта.
