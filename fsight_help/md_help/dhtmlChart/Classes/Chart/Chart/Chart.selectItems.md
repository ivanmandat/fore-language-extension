# Chart.selectItems


## Синтаксис


selectItems (items, reset))


## Параметры


items. Json-объект, содержащий
 настройки следующего вида: <индекс
 ряда, который необходимо выделить>: true;


reset. Признак снятия выделения
 с рядов, выделенных ранее.


## Описание


Метод selectItems выделяет указанные
 ряды диаграммы.


## Комментарии


Допустимые значения параметра reset:


	- true. Выделение рядов
	 будет снято;


	- false. Выделение рядов
	 останется.


## Пример


Для выполнения примера необходимо сначала выполнить пример на странице
 описания метода [Chart.ItemsHovered](Chart.ItemsHovered.htm).
 Наведите курсор на ряд диаграммы с индексами 0 и 1:


chart.selectItems({1: true, 0: true}, true);
После выполнения примера будут выделены первый и второй ряд диаграммы.
 Сгенерируется событие [Chart.ItemsSelected](Chart.ItemsSelected.htm),
 в консоль браузера будут выведены индексы выделенных элементов.


Примечание.
 Если установлен линейный тип диаграммы, выделение будет незаметно. Чтобы
 выделение была заметно, можно установить тип диаграммы - гистограмма (Type:
 "Column").
