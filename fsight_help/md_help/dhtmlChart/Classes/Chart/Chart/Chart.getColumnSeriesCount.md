# Chart.getColumnSeriesCount


## Синтаксис


getColumnSeriesCount(countInvisible: Boolean);


## Параметры


countInvisible. Признак необходимости
 учитывать скрытые ряды данных диаграммы.


## Описание


Метод getColumnSeriesCount возвращает
 количество рядов данных диаграммы в виде столбцов.


## Комментарии


Метод возвращает значение типа Number.


Допустимые значения параметра countInvisible:


	- true. Необходимо
	 учитывать скрытые ряды данных диаграммы;


	- false. По умолчанию.
	 Учитывать скрытые ряды данных диаграммы не требуется.
