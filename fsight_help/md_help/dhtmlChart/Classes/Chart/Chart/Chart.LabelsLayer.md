# Chart.LabelsLayer


## Синтаксис


LabelsLayer: String;


## Описание


Свойство LabelsLayer определяет
 элемент, представляющий слой отрисовки подписей диаграммы.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm),
 а возвращается с помощью метода getLabelsLayer.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 [Chart](../../../Components/Chart/Chart.htm)
 с наименованием «chart» (см. «[Пример
 создания гистограммы](../../../Components/Chart/Chart_Example.htm)»). Получим слой отрисовки подписей диаграммы
 и установим для него новый идентификатор:


// Получаем элемент, представляющий слой, в котором отрисовываются подписи диаграммы
var layer = chart.getLabelsLayer();
// Устанавливаем идентификатор слоя
layer.setAttribute("id", "labelsLayer");
// Получаем в документе слой по идентификатору
var layer2 = document.getElementById("labelsLayer");
// Выводим в консоль браузера строку, представляющую полученный слой
console.log(layer2);
В результате выполнения примера в консоли браузера была выведена строка,
 представляющая слой отрисовки подписей диаграммы:


<canvas class=​"Graphic Labels" height=​"500"
 width=​"600" id=​"labelsLayer">​
