# Chart.UseMarkersThinning


## Синтаксис


UseMarkersThinning: Boolean;


## Описание


Свойство UseMarkersThinning
 определяет признак скрытия маркеров при условии, если хотя бы некоторые
 из них перекрывают друг друга.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm)
 или с помощью метода setUseMarkersThinning,
 а возвращается с помощью метода getUseMarkersThinning.


Допустимые значения:


	- true.
	 Все маркеры рядов данных диаграммы будут скрыты, если хотя бы некоторые
	 из них перекрывают друг друга;


	- false.
	 По умолчанию. Все маркеры рядов данных диаграммы будут отображены,
	 даже если некоторые из них перекрывают друг друга.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 [Chart](Chart.htm) с наименованием «chart» (см. «[Пример
 создания сглаженной диаграммы](../../../Components/Chart/ChartSplineSerie.htm)»). Установим признак, скрывающий маркеры
 рядов данных диаграммы при условии, если они перекрывают друг друга, и
 зададим новый размер маркеров:


// Устанавливаем новый размер маркеров
chart.setMarkersRadius(20);
// Устанавливаем признак скрытия маркеров при условии, если они перекрывают друг друга
chart.setUseMarkersThinning(true);
// Перерисовываем ряды данных диаграммы
chart.redrawSeries();
В результате выполнения примера маркеры у рядов данных диаграммы были
 скрыты:
