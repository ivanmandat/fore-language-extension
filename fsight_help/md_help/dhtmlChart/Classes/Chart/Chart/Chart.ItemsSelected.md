# Chart.ItemsSelected


## Синтаксис


ItemsHovered: function (sender, args);


## Параметры


sender. Источник события;


args. Информация
 о событии. Доступный аргумент -
 Items. Представляет собой json-объект, содержащий
 настройки следующего вида: <индекс выделенного ряда>:
 true.


## Описание


Событие ItemsSelected наступает
 при выделении ряда диаграммы.
