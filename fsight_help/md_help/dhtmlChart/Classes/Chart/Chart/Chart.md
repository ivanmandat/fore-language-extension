# Chart


Пространство имен: PP.Ui;


## Иерархия наследования


           [Object](dhtmlCommon.chm::/Classes/object/object.htm)


           [Control](dhtmlUi.chm::/Classes/control/control.htm)


           Chart


## Описание


Класс Chart предназначен для
 работы с диаграммой, построенной на основе элемента Canvas.


## Комментарии


Значения свойств задаются в конструкторе [Chart](Constructor_Chart.htm)
 или с помощью метода set<имя свойства>, а возвращаются с помощью метода
 get<имя
 свойства>.


Доступные методы указаны в описании конкретного свойства.


## Синтаксис


PP.initClass(PP.Ui.Chart, [PP.Ui.Control](dhtmlUi.chm::/Classes/control/control.htm),
 "PP.Ui.Chart", [PP.Ui.IDVSynchronizable]);
