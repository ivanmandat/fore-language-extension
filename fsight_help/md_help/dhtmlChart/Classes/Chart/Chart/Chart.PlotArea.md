# Chart.PlotArea


## Синтаксис

## Описание


Свойство PlotArea определяет
 область отрисовки диаграммы.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm),
 а возвращается с помощью метода getPlotArea.
