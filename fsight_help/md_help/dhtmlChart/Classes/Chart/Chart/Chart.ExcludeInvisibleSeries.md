# Chart.ExcludeInvisibleSeries


## Синтаксис


ExcludeInvisibleSeries: Boolean;


## Описание


Свойство ExcludeInvisibleSeries определяет
 признак необходимости исключения невидимых рядов данных диаграммы из расчетов.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm),
 а возвращается с помощью метода getExcludeInvisibleSeries.


Допустимые значения:


	- true.
	 Невидимые ряды данных диаграммы будут исключены из расчетов;


	- false.
	 По умолчанию. Невидимые ряды данных диаграммы не будут исключены из
	 расчетов.
