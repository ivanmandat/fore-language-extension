# Chart.Selected


## Синтаксис


Selected: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие Selected наступает при
 выделении элемента ряда диаграммы.


## Комментарии


Для наступления события зажмите клавишу SHIFT и с помощью мыши выделите
 один из элементов ряда диаграммы.
