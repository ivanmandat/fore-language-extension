# Chart.getSerie


## Синтаксис


getSerie(index: Number);


## Параметры


index. Индекс ряда данных диаграммы.


## Описание


Метод getSerie возвращает ряд
 данных диаграммы по индексу.


## Комментарии


Метод возвращает значение типа [ChartCanvasSerie](../ChartCanvasSerie/ChartCanvasSerie.htm).

Использование метода приведено в примере для [Chart.removeAllSeries](Chart.removeAllSeries.htm).
