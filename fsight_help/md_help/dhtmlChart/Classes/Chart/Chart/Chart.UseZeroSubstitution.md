# Chart.UseZeroSubstitution


## Синтаксис


UseZeroSubstitution: Boolean;


## Описание


Свойство UseZeroSubstitution
 определяет признак того, что пустые значения в рядах данных заменяются
 нулями.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm)
 или с помощью метода setUseZeroSubstitution,
 а возвращается с помощью метода getUseZeroSubstitution.


Допустимые значения:


	- true.
	 Пустые значения в рядах данных заменяются нулями;


	- false.
	 Пустые значения в рядах данных не заменяются нулями.
