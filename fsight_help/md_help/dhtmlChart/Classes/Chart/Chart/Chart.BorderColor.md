# Chart.BorderColor


## Синтаксис


BorderColor: String;


## Описание


Свойство BorderColor определяет
 цвет границы диаграммы.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm)
 в формате RGBA, например: rgba(237, 27, 35, 1).


Для отображения границы диаграммы используйте свойство [Chart.BorderEnabled](Chart.BorderEnabled.htm)
 со значением true.
