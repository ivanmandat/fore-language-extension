# Chart.getStackGroups


## Синтаксис


getStackGroups(countInvisible: Boolean);


## Параметры


countInvisible. Определение
 способа формирования массива уникальных идентификаторов накоплений диаграммы:
 с учётом скрытых рядов данных или без учёта.


## Описание


Метод getStackGroups возвращает
 массив уникальных идентификаторов накоплений диаграммы.


## Комментарии


Для определения идентификатора накопления диаграммы используйте свойство
 [ChartCanvasSerie.StackId](../ChartCanvasSerie/ChartCanvasSerie.StackId.htm).
 Например, если два ряда данных имеют идентификатор Absolute, а один -
 Procent, то при вызове метода будет возвращён массив, содержащий два идентификатора:
 Absolute и Procent.


Допустимые значения параметра countInvisible:


	- true. По умолчанию.
	 Скрытые ряды данных учитываются при формировании массива уникальных
	 идентификаторов накоплений диаграммы;


	- false. Скрытые ряды
	 данных не учитываются при формировании массива уникальных идентификаторов
	 накоплений диаграммы.
