# Chart.Options


## Синтаксис


Options: Object;


## Описание


Свойство Options определяет
 настройки диаграммы через [веб-сервис](ppsomservice.chm::/intro/pp7service_using.htm).


## Комментарии


Свойство предназначено для внутреннего использования.


Настройки диаграммы возвращаются в параметре hiChart
 при выполнении операции веб-сервиса [GetDefaultChart](ppsomservice.chm::/operations/visualizators/getdefaultchart.htm).
