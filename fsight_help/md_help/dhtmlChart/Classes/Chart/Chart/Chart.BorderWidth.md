# Chart.BorderWidth


## Синтаксис


BorderWidth: Number;


## Описание


Свойство BorderWidth определяет
 толщину границы диаграммы.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm).


Диапазон допустимых значений толщины границы диаграммы - [0, 13]. Толщина
 границы измеряется в пикселях.


Для отображения границы диаграммы используйте свойство [Chart.BorderEnabled](Chart.BorderEnabled.htm)
 со значением true.
