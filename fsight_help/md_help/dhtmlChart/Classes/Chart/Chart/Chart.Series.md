# Chart.Series


## Синтаксис


Series: Array;


## Описание


Свойство Series определяет массив
 рядов данных диаграммы.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm)
 или с помощью метода setSeries,
 а возвращается с помощью метода getSeries.


Массив содержит объекты типа [PP.Ui.ChartCanvasSerie](../ChartCanvasSerie/ChartCanvasSerie.htm).


Метод getSeries содержит параметр
 trendLines, задающий признак необходимости учитывать ряды тренда при получении
 массива, и параметр invisible, задающий признак необходимости исключения
 скрытых рядов.


Допустимые значения параметра trendLines:


	- true.
	 При получении массива будут учитываться ряды тренда;


	- false.
	 По умолчанию. При получении массива ряды тренда учитываться не будут.


Допустимые значения параметра invisible:


	- true.
	 При получении массива будут исключены скрытые ряды;


	- false.
	 По умолчанию. При получении массива скрытые ряды не будут исключены.
