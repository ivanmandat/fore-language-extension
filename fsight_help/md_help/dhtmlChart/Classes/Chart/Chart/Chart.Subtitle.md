# Chart.Subtitle


## Синтаксис


Subtitle: {


    settings


};


## Параметры


settings. JSON-объект с параметрами
 отображения дополнительного заголовка диаграммы.


## Описание


Свойство Subtitle определяет
 параметры дополнительного заголовка диаграммы.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm)
 или с помощью метода setSubtitle,
 а возвращается с помощью метода getSubtitle.


Для JSON-объекта используйте параметры, аналогичные параметрам JSON-объекта
 в свойстве [Chart.Title](Chart.Title.htm).
