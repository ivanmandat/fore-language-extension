# Chart.getPlotBorders


## Синтаксис


getPlotBorders(soft: Object);


## Параметры


soft. Объект, содержащий признаки
 применения отступов диаграммы. Необязательный параметр. Поля объекта:


	- bottom.
	 Признак применения нижнего отступа диаграммы;


	- left.
	 Признак применения левого отступа диаграммы;


	- right.
	 Признак применения правого отступа диаграммы;


	- top.
	 Признак применения верхнего отступа диаграммы.


## Описание


Метод getPlotBorders возвращает
 объект с размерами и отступами диаграммы.


## Комментарии


Метод возвращает объект со следующими полями:


	- X.
	 Координата X левого верхнего угла диаграммы;


	- Y.
	 Координата Y левого верхнего угла диаграммы;


	- W.
	 Ширина диаграммы;


	- H.
	 Высота диаграммы.


Если параметр soft не задан,
 то признаки применения отступов диаграммы не учитываются.
