# Chart.setHoverPoint


## Синтаксис


setHoverPoint(settings);


## Параметры


settings. JSON-объектс параметрами
 отображения выделения точки на диаграмме.


## Описание


Метод setHoverPoint устанавливает
 выделение точки на диаграмме.


## Комментарии


Для JSON-объекта параметра settings
 доступны следующие параметры:


	- Serie. Тип параметра:
	 Integer. Индекс ряда данных диаграммы;


	- Point. Тип параметра:
	 Integer. Индекс точки ряда данных диаграммы;


	- PreventToolTip. Тип
	 параметра: Boolean. Признак отображения всплывающей подсказки:


		- True.
		 Всплывающая подсказка не отображается;


		- False. Всплывающая
		 подсказка отображается.


## Пример


Для выполнения примера создайте html-страницу, на которой отображается
 [линейная диаграмма](../../../Components/Chart/ChartLine.htm).
 Настройки диаграммы задаются с помощью компонента [Chart](dhtmlChar.chm::/t/Components/Chart/Chart.htm).


Выделите одну точку ряда двойным нажатием мыши для снятия выделения
 с остальных точек и выполните в консоли:


// Выделим четвертую точку первого ряда
select = {serie:0, point:3, preventToolTip:false};
chart.setHoverPoint(select);
В результате выполнения примера четвертая точка первого ряда будет выделена
 вместе с точкой, выделенной вручную. Под заданной точкой ряда отобразится
 всплывающая подсказка.
