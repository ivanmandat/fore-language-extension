# Class.UseSoftPadding


## Синтаксис


UseSoftPadding: Boolean;


## Описание


Свойство UseSoftPadding определяет
 признак использования альтернативного расчета отступов диаграммы.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm)
 или с помощью метода setUseSoftPadding,
 а возвращается с помощью метода getUseSoftPadding.


Допустимые значения:


	- true.
	 По умолчанию. Используется альтернативный расчет отступов диаграммы;


	- false.
	 Альтернативный расчет отступов диаграммы не используется.


При альтернативном расчете отступов диаграммы соответствующий отступ
 не применяется, если с некоторой стороны имеются элементы диаграммы.
