# Chart.endUpdate


## Синтаксис


endUpdate();


## Описание


Метод endUpdate включает перерисовку
 диаграммы и обновляет её.


## Комментарии


Используется в связке с методом [Chart.beginUpdate](Chart.beginUpdate.htm).
