# Chart.redraw


## Синтаксис


redraw(recalc: boolean, noAnimation: boolean);


## Параметры


recalc. Признак перерасчёта
 компонентов диаграммы;


noAnimation. Признак использования
 анимации при перерисовке диаграммы.


## Описание


Метод redraw перерисовывает
 диаграмму.


## Комментарии


Допустимые значения параметра recalc:


	- true.
	 Компоненты диаграммы перерассчитываются с учётом их расположения;


	- false.
	 По умолчанию. Компоненты диаграммы не перерассчитываются.


Допустимые значения параметра noAnimation:


	- true.
	 Воспроизведение анимации при перерисовке диаграммы выключено;


	- false.
	 По умолчанию. Воспроизведение анимации при перерисовке диаграммы включено.
