# Chart.BackgroundColor


## Синтаксис


BackgroundColor: String;


## Описание


Свойство BackgroundColor определяет
 цвет фона диаграммы.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm)
 в формате RGBA, например: rgba(237, 27, 35, 1).


Для установки прозрачного фона диаграммы задайте свойству значение «transparent».
