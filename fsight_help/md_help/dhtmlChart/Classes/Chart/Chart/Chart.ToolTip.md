# Chart.ToolTip


## Синтаксис


ToolTip: PP.Ui.CanvasTooltip;


## Описание


Свойство ToolTip определяет
 параметры всплывающей подсказки.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm),
 а возвращается с помощью метода getToolTip.
