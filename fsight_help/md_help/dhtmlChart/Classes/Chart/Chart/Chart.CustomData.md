# Chart.CustomData


## Синтаксис


CustomData: Number | String | Object;


## Описание


Свойство CustomData определяет
 данные пользователя.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm)
 или с помощью метода setCustomData,
 а возвращается с помощью метода getCustomData.
