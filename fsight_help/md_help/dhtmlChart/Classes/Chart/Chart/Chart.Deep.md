# Chart.Deep


## Синтаксис


Deep: Number;


## Описание


Свойство Deep определяет длину
 оси Z при отображении [объёмного
 вида диаграммы](UiDiagrams.chm::/Params_diagram/Setup_3DView.htm).


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm),
 а возвращается с помощью метода getDeep.
 Длина оси Z должна быть больше нуля.


Для отображения объёмного вида диаграммы используйте свойство [Chart.Is3D](Chart.Is3D.htm)
 со значением true.
