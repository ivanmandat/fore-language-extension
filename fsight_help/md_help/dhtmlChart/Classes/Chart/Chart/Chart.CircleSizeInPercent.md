# Chart.CircleSizeInPercent


## Синтаксис


CircleSizeInPercent: Boolean;


## Описание


Свойство CircleSizeInPercent
 определяет способ задания размера внешней границы [круговой
 диаграммы](UiDiagrams.chm::/Type_diagrams/UiDiagrams_round.htm).


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm),
 а возвращается с помощью метода getCircleSizeInPercent.


Допустимые значения:


	- true. По умолчанию.
	 Размер внешней границы круговой диаграммы задаётся в процентах;


	- false. Размер внешней
	 границы круговой диаграммы задаётся в пикселях.


Для определения размера внешней границы круговой диаграммы используйте
 свойство [Chart.CircleSize](Chart.CircleSize.htm).
