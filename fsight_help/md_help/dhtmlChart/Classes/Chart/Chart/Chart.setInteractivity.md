# Chart.setInteractivity


## Синтаксис


setInteractivity(value: Boolean);


## Параметры


value. Признак использования
 интерактивности диаграммы.


## Описание


Метод setInteractivity определяет
 использование интерактивности диаграммы.


## Комментарии


Интерактивность позволяет выполнять события для работы с диаграммой,
 например: выделение ряда данных, наведение курсора на ряд данных и другие.


Допустимые значения параметра value:


	- true. Интерактивность
	 диаграммы включена;


	- false. Интерактивность
	 диаграммы отключена. Диаграмма доступна только на просмотр.
