# Chart.beginUpdate


## Синтаксис


beginUpdate();


## Описание


Метод beginUpdate отключает
 перерисовку диаграммы до тех пор, пока не будет вызван метод [endUpdate](Chart.endUpdate.htm).


## Комментарии


При отключении перерисовки диаграммы можно задать несколько разных настроек
 подряд для оптимизации времени, затрачиваемого на перерисовку диаграммы
 после каждого изменения настроек. В данном случае диаграмма перерисуется
 один раз после вызова метода [Chart.endUpate](Chart.endUpdate.htm)
 со всеми изменёнными настройками.
