# Chart.MarkersSymbol


## Синтаксис


MarkersSymbol: String;


## Описание


Свойство MarkersSymbol определяет
 форму маркера для рядов данных диаграммы.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm)
 или с помощью метода setMarkersSymbol,
 а возвращается с помощью метода getMarkersSymbol.


Допустимые значения:


	- circle.
	 По умолчанию. Маркер в форме круга;


	- diamond.
	 Маркер в форме ромба;


	- square.
	 Маркер в форме квадрата;


	- triangle.
	 Маркер в форме треугольника;


	- triangledown,
	 triangle-down. Маркер в форме
	 обратного треугольника.
