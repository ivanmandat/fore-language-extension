# Chart.Is3D


## Синтаксис


Is3D: Boolean;


## Описание


Свойство Is3D определяет отображение
 диаграммы в [объёмном
 виде](UiDiagrams.chm::/Params_diagram/Setup_3DView.htm).


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm),
 а возвращается с помощью метода getIs3D.


Допустимые значения:


	- true. Диаграмма отображается
	 в объёмном виде;


	- false. По умолчанию.
	 Диаграмма отображается в плоском виде.


Для определения длины оси Z используйте свойство [Chart.Deep](Chart.Deep.htm).
