# Chart.getSeriesTypes


## Синтаксис


getSeriesTypes();


## Описание


Метод getSeriesTypes возвращает
 массив типов рядов диаграммы.


## Комментарии


Массив типов рядов диаграммы содержит значения типа String.


Допустимые значения типов рядов данных приведены на странице описания
 свойства [ChartCanvasSerie.Type](../ChartCanvasSerie/ChartCanvasSerie.Type.htm).
