# Chart.Inverted


## Синтаксис


Inverted: Boolean;


## Описание


Свойство Inverted определяет
 признак того, является ли диаграмма инвертированной.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm),
 а возвращается с помощью метода getInverted.


На инвертированной диаграмме ось категорий и ось значений меняются местами.


Допустимые значения:


	- true.
	 Диаграмма является инвертированной;


	- false.
	 По умолчанию. Диаграмма не является инвертированной.
