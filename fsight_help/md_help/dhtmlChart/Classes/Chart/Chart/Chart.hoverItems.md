# Chart.hoverItems


## Синтаксис


hoverItems(items: Object);


## Параметры


items. JSON-объект, содержащий
 настройки следующего вида: <индекс
 ряда, на который наведен курсор>: true.


## Описание


Метод hoverItems устанавливает
 подсветку указанных рядов диаграммы.


## Пример


Для выполнения примера предполагается наличие на странице компонента
 Chart с наименованием «chart».
 Наведем курсор на ряд диаграммы с индексами 0 и 1:


chart.hoverItems({1: true, 0: true});
После выполнения примера для первого и второго ряда диаграммы будет
 установлена подсветка. Будет сгенерировано событие [Chart.ItemsHovered](Chart.ItemsHovered.htm):
 в консоль браузера будут выведены индексы подсвеченных элементов.


Примечание.
 Если установлен линейный тип диаграммы, подсветка будет незаметна. Чтобы
 подсветка была заметна, можно установить тип диаграммы - гистограмма (Type:
 "Column").
