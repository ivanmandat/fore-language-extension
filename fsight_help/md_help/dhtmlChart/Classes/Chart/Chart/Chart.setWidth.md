# Chart.setWidth


## Синтаксис


setWidth(width: Number, fireEvents: Boolean);


## Параметры


width. Значение ширины диаграммы
 в пикселях;


fireEvents. Признак использования
 события [Control.SizeChanging](dhtmlUi.chm::/Classes/Control/Control.SizeChanging.htm).


## Описание


Метод setWidth
 устанавливает ширину диаграммы.


## Комментарии


Допустимые значения параметра fireEvents:


	- true. Событие [Control.SizeChanging](dhtmlUi.chm::/Classes/Control/Control.SizeChanging.htm)
	 используется при изменении ширины диаграммы;


	- false. По умолчанию.
	 Событие [Control.SizeChanging](dhtmlUi.chm::/Classes/Control/Control.SizeChanging.htm)
	 не используется при изменении ширины диаграммы.
