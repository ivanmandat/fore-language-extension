# Chart.YAxis


## Синтаксис


YAxis: [PP.Ui.ChartCanvasAxis](../ChartCanvasAxis/ChartCanvasAxis.htm);


## Описание


Свойство YAxis определяет ось
 значений диаграммы.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm)
 или с помощью метода setYAxis,
 а возвращается с помощью метода getYAxis.
