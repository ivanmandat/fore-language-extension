# Chart.XAxis


## Синтаксис


XAxis: [PP.Ui.ChartCanvasAxis](../ChartCanvasAxis/ChartCanvasAxis.htm);


## Описание


Свойство XAxis определяет ось
 категорий диаграммы.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm)
 или с помощью метода setXAxis,
 а возвращается с помощью метода getXAxis.
