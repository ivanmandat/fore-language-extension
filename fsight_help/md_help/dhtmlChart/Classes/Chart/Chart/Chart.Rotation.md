# Chart.Rotation


## Синтаксис


Rotation: Number;


## Описание


Свойство Rotation определяет
 угол поворота [круговой
 диаграммы](UiDiagrams.chm::/Type_diagrams/UiDiagrams_round.htm).


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm)
 или с помощью метода setRotation,
 а возвращается с помощью метода getRotation.


Угол поворота круговой диаграммы задаётся в градусах.
