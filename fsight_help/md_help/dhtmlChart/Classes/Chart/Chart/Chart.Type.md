# Chart.Type


## Синтаксис


Type: String;


## Описание


Свойство Type определяет тип
 диаграммы.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm),
 а возвращается с помощью метода getType.


Допустимые значения:


	- Area.
	 Диаграмма с областями;


	- Column.
	 Гистограмма;


	- Doughnut.
	 Кольцевая диаграмма;


	- Line.
	 По умолчанию. Линейная диаграмма;


	- Mixed.
	 Смешанная диаграмма;


	- Pie.
	 Круговая диаграмма;


	- Radar.
	 Лепестковая диаграмма;


	- Scatter.
	 Точечная диаграмма;


	- Spline.
	 Сглаженная диаграмма;


	- Stepline.
	 Ступенчатая диаграмма.
