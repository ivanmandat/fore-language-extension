# Chart.ToolTipDataFormat


## Синтаксис


ToolTipDataFormat: String;


## Описание


Свойство ToolTipDataFormat определяет
 формат данных всплывающей подсказки.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm)
 или с помощью метода setToolTipDataFormat,
 а возвращается с помощью метода getToolTipDataFormat.


По умолчанию содержит значение «0.00».
