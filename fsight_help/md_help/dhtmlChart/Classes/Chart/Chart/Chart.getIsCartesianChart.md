# Chart.getIsCartesianChart


## Синтаксис


getIsCartesianChart();


## Описание


Метод getIsCartesianChart возвращает
 признак того, использует ли диаграмма декартову систему координат.


## Комментарии


Метод возвращает значение типа Boolean.


Допустимые значения:


	- true. Диаграмма использует
	 декартову систему координат;


	- false. Диаграмма не
	 использует декартову систему координат.
