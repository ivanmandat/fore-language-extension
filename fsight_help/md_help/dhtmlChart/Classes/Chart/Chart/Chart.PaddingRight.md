# Chart.PaddingRight


## Синтаксис


PaddingRight: Number;


## Описание


Свойство PaddingRight определяет
 правый отступ диаграммы.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm),
 а возвращается с помощью метода getPaddingRight.


По умолчанию значение свойства равно 10.
