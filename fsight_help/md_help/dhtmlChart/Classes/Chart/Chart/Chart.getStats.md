# Chart.getStats


## Синтаксис


getStats();


## Описание


Метод getStats возвращает статистические
 характеристики диаграммы.


## Комментарии


Возвращает JSON-объект со следующими параметрами:


	- min. Тип параметра:
	 Number. Минимум;


	- max. Тип параметра:
	 Number. Максимум;


	- summ. Тип параметра:
	 Number. Сумма;


	- median. Тип параметра:
	 Number. Медиана;


	- mean. Тип параметра:
	 Number. Среднее арифметическое.
