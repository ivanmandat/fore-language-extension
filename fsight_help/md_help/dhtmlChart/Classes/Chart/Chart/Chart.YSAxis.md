# Chart.YSAxis


## Синтаксис


YSAxis: [PP.Ui.ChartCanvasAxis](../ChartCanvasAxis/ChartCanvasAxis.htm);


## Описание


Свойство YSAxis определяет дополнительную
 ось значений диаграммы.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm)
 или с помощью метода setYSAxis,
 а возвращается с помощью метода getYSAxis.

 создания гистограммы](../../../Components/Chart/Chart_Example.htm).
