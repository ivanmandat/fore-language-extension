# Chart.BorderStyle


## Синтаксис


BorderStyle: String;


## Описание


Свойство BorderStyle определяет
 стиль линии границы диаграммы.


## Комментарии


Значение свойства задаётся в конструкторе [Chart](Constructor_Chart.htm).


Допустимые значения:


	- solid. Сплошная линия;


	- shortdash. Короткие
	 штрихи;


	- shortdot. Короткие пунктиры;


	- shortdashdot. Штрих-пунктирная
	 линия с короткими штрихами;


	- shortdashdotdot. Штрих-пунктирная
	 линия с короткими штрихами и с двумя точками.


Для отображения границы диаграммы используйте свойство [Chart.BorderEnabled](Chart.BorderEnabled.htm)
 со значением true.
