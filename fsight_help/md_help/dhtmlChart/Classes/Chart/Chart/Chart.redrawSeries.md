# Chart.redrawSeries


## Синтаксис


redrawSeries(recalc: boolean, noAnimation: boolean);


## Параметры


recalc. Признак перерасчёта
 рядов данных диаграммы;


noAnimation. Признак использования
 анимации при перерисовке рядов данных диаграммы.


## Описание


Метод redrawSeries перерисовывает
 ряды данных диаграммы.


## Комментарии


Допустимые значения параметра recalc:


	- true.
	 Ряды данных диаграммы перерассчитываются с учётом их расположения;


	- false.
	 По умолчанию. Ряды данных диаграммы не перерассчитываются. Например,
	 при изменении заливки перерасчёт рядов данных диаграммы не требуется.


Допустимые значения параметра noAnimation:


	- true.
	 Воспроизведение анимации при перерисовке рядов данных диаграммы выключено;


	- false.
	 По умолчанию. Воспроизведение анимации при перерисовке рядов данных
	 диаграммы включено. Например, при использовании гистограммы ряды данных
	 поднимаются снизу вверх.


Параметры метода являются необязательными.
