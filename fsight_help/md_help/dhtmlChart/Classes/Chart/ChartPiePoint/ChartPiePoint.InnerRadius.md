# ChartPiePoint.InnerRadius


## Синтаксис


InnerRadius: Number;


## Описание


Свойство InnerRadius определяет
 внутренний радиус сектора, соответствующего точке ряда круговой диаграммы.


## Комментарии


Свойство предназначено только для чтения.
