# ChartPiePoint.EndAngle


## Синтаксис


EndAngle: Number;


## Описание


Свойство EndAngle определяет
 конечный угол сектора, соответствующего точке ряда круговой диаграммы.


## Комментарии


Свойство предназначено только для чтения.
