# ChartAreaSerie.getPolyInRange


## Синтаксис


getPolyInRange(start: Object, end: Object);


## Параметры


start. Левая граница диапазона;


end. Правая граница диапазона.


## Описание


Метод getPolyInRange возвращает
 массив точек ряда в указанном диапазоне.


## Комментарии


Метод возвращает значение типа Array.
