# ChartAreaSerie.getIsAreaRange


## Синтаксис


getIsAreaRange();


## Описание


Метод getIsAreaRange возвращает
 признак того, представляет ли ряд данных область с накоплением.


## Комментарии


Метод возвращает значение типа Boolean.


Допустимые значения:


	- true. Ряд данных представляет
	 собой область с накоплением;


	- false. Ряд данных представляет
	 собой область без накопления.
