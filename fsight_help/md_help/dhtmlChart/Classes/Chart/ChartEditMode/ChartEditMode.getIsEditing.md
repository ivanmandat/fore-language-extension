# ChartEditMode.getIsEditing


## Синтаксис


getIsEditing();


## Описание


Метод getIsEditing возвращает
 признак редактирования элемента диаграммы.


## Комментарии


Метод возвращает значение true,
 если редактируется элемент диаграммы, иначе - значение false.
