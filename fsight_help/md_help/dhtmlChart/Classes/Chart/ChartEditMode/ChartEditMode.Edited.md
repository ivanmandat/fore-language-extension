# ChartEditMode.Edited


## Синтаксис


Edited: function(sender, args);


## Параметры


sender. Источник события;


args. Информация
 о событии. Аргументы: Point -
 редактируемая точка; Serie
 - редактируемая серия; Value - значение после редактирования.


## Описание


Событие Edited наступает после
 окончания редактирования значений диаграммы.
