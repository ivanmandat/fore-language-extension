# ChartEditMode.handleMouseMove


## Синтаксис


handleMouseMove(coords: Object, event: Object);


## Параметры


coords. Объект с координатами.
 Объект должен содержать поля: X
 - координата по оси X, Y - координата
 по оси Y;


event. Объект события.


## Описание


Метод handleMouseMove обрабатывает
 событие движения курсора по диаграмме в режиме редактирования.
