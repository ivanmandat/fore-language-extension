# ChartEditMode.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет
 активность режима редактирования диаграммы.


## Комментарии


Допустимые значения:


	- true.
	 Режим редактирования активен;


	- false.
	 Режим редактирования не активен (по умолчанию).


Значение свойства устанавливается из JSON и с помощью метода setEnabled,
 а возвращается с помощью метода getEnabled.
