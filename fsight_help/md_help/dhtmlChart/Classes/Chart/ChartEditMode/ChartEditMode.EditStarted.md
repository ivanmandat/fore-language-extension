# ChartEditMode.EditStarted


## Синтаксис


EditStarted: function(sender, args);


## Параметры


sender. Источник события;


args. Информация
 о событии. Аргументы: Point -
 редактируемая точка; Serie
 - редактируемая серия; Value - значение до начала редактирования.


## Описание


Событие EditStarted наступает
 перед началом редактирования значений диаграммы.
