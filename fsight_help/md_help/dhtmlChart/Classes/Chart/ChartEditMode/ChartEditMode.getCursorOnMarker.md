# ChartEditMode.getCursorOnMarker


## Синтаксис


getCursorOnMarker();


## Описание


Метод getCursorOnMarker возвращает
 признак нахождения курсора над маркером изменения размера столбца диаграммы.


## Комментарии


Метод возвращает значение true,
 если курсор находится над маркером, иначе - значение false.
