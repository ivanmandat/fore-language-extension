# ChartEditMode.State


## Синтаксис


State: Boolean;


## Описание


Свойство State определяет активность
 режима редактирования диаграммы.


## Комментарии


Допустимые значения:


	- true.
	 Режим редактирования активен;


	- false.
	 Режим редактирования не активен (по умолчанию).


Значение свойства устанавливается из JSON и с помощью метода setState,
 а возвращается с помощью метода getState.
