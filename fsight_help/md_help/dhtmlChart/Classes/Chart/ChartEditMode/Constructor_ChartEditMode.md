# Конструктор ChartEditMode


## Синтаксис


PP.Ui.ChartEditMode(settings);


## Параметры


settings. JSON-объект со значениями
 свойств компонента.


## Описание


Конструктор ChartEditMode создает
 экземпляр класса ChartEditMode.
