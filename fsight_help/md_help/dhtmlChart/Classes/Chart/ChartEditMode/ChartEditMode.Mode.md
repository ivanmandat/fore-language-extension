# ChartEditMode.Mode


## Синтаксис


Mode: String;


## Описание


Свойство Mode определяет способ
 редактирования элементов диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMode,
 а возвращается с помощью метода getMode.


Допустимые значения:


		 Значение


		 Краткое
		 описание


		 Pencil


		 Редактирование данных с помощью рисования.


		 Point


		 Редактирование по точкам.


		 Draw


		 Редактирование данных указанием положения с помощью курсора.
