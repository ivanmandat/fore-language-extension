# ChartAreaPoint.PlotYBottom


## Синтаксис


PlotYBottom: Number;


## Описание


Свойство PlotYBottom определяет
 экранную координату второго значения по оси Y для точки ряда данных.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setPlotYBottom,
 а возвращается с помощью метода getPlotYBottom.
