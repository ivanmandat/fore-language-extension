# ChartAreaPoint.YBottom


## Синтаксис


YBottom: Number;


## Описание


Свойство YBottom определяет
 второе значение по оси Y для точки ряда данных.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setYBottom,
 а возвращается с помощью метода getYBottom.
