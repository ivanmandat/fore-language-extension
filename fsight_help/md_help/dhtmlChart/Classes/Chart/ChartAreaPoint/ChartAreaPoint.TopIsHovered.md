# ChartAreaPoint.TopIsHovered


## Синтаксис


TopIsHovered: Boolean;


## Описание


Свойство TopIsHovered определяет,
 будет ли отображаться метка для первого значения точки ряда данных.


## Комментарии


Значение свойства устанавливается с помощью метода setTopIsHovered,
 а возвращается с помощью метода getTopIsHovered.


Допустимые значения:


	- true.
	 Метка отображается для первого значения точки ряда данных (по умолчанию);


	- false.
	 Метка отображается для второго значения точки ряда данных.
