# ChartAreaPoint.BottomDataLabel


## Синтаксис


BottomDataLabel: [PP.Ui.ChartText](../ChartText/ChartText.htm);


## Описание


Свойство BottomDataLabel определяет
 подпись данных для второго значения точки ряда данных.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setBottomDataLabel,
 а возвращается с помощью метода getBottomDataLabel.
