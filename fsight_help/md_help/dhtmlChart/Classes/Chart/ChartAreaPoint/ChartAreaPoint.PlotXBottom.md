# ChartAreaPoint.PlotXBottom


## Синтаксис


PlotXBottom: Number;


## Описание


Свойство PlotXBottom определяет
 экранную координату второго значения по оси X для точки ряда данных.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setPlotXBottom,
 а возвращается с помощью метода getPlotXBottom.
