# ChartCanvasAxis.lin2log


## Синтаксис


lin2log(value: Number);


## Параметры


value. Линейное значение оси
 диаграммы.


## Описание


Метод lin2log преобразует линейное
 значение оси диаграммы в логарифмическое.


## Комментарии


Метод возвращает значение типа Number.
