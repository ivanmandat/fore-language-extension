# ChartCanvasAxis.LineWidth


## Синтаксис


LineWidth: Number;


## Описание


Свойство LineWidth определяет
 толщину оси диаграммы.


## Комментарии


Значение свойства устанавливается из JSON, а возвращается с помощью
 метода getLineWidth.


По умолчанию значение свойства равно 1.
