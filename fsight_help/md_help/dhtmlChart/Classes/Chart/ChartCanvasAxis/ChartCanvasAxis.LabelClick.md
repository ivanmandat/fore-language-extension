# ChartCanvasAxis.LabelClick


## Синтаксис


LabelClick: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные
 аргументы: Event - событие мыши, Tick - подпись оси диаграммы.


## Описание


Событие LabelClick наступает
 после нажатия на подпись оси диаграммы.
