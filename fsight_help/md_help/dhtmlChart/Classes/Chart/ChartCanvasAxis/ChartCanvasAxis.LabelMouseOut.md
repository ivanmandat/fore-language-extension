# ChartCanvasAxis.LabelMouseOut


## Синтаксис


LabelMouseOut: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.
 Доступные аргументы: Event - событие мыши, Tick - подпись оси диаграммы.


## Описание


Событие LabelMouseOut наступает
 при отведении курсора от подписи оси диаграммы.
