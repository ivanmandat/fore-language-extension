# ChartCanvasAxis.TitleDblClick


## Синтаксис


TitleDblClick: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие TitleDblClick наступает
 при двойном нажатии указателя по заголовку оси диаграммы.
