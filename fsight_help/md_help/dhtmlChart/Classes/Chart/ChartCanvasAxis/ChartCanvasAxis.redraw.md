# ChartCanvasAxis.redraw


## Синтаксис


redraw(calcOnly: Boolean);


## Параметры


calcOnly. Признак того, что
 будут пересчитаны только свойства оси диаграммы. Необязательный параметр.
 Допустимые значения:


	- true.
	 Будут пересчитаны только свойства оси диаграммы;


	- false.
	 Ось диаграммы будет обновлена полностью (по умолчанию).


## Описание


Метод redraw перерисовывает
 ось диаграммы.
