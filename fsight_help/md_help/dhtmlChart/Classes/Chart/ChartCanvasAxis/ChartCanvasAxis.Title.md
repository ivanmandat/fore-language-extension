# ChartCanvasAxis.Title


## Синтаксис


Title: Object;


## Описание


Свойство Title определяет заголовок
 оси диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTitle,
 а возвращается с помощью метода getTitle.
