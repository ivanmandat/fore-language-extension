# ChartCanvasAxis.getDataExtremes


## Синтаксис


getDataExtremes();


## Описание


Метод getDataExtremes возвращает
 минимальное и максимальное значения оси диаграммы.


## Комментарии


Метод возвращает JSON-объект со следующими полями: min - минимальное
 значение оси, max - максимальное значение.
