# ChartCanvasAxis.getAxisValue


## Синтаксис


getAxisValue(value: Number, withOverflow: Boolean);


## Параметры


value. Координата оси диаграммы;


withoutLimits. Признак того,
 что допустимый диапазон значений оси не учитываются. Допустимые значения:


	- true.
	 Значение оси диаграммы будет определено без учёта её минимального
	 и максимального значений;


	- false.
	 Значение оси диаграммы будет определено с учётом её минимального и
	 максимального значений (по умолчанию).


## Описание


Метод getAxisValue возвращает
 значение, соответствующее указанной координате оси диаграммы.


## Комментарии


Метод возвращает значение типа Number.
