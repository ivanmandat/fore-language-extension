# ChartCanvasAxis.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет
 признак активности оси диаграммы.


## Комментарии


Значение свойства устанавливается из JSON, а возвращается с помощью
 метода getEnabled.


Допустимые значения:


	- true.
	 Ось диаграммы является активной (по умолчанию);


	- false.
	 Ось диаграммы не является активной.
