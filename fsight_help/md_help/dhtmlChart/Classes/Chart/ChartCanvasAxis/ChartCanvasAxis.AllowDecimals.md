# ChartCanvasAxis.AllowDecimals


## Синтаксис


AllowDecimals: Boolean;


## Описание


Свойство AllowDecimals определяет
 признак использования дробных значений для оси диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setAllowDecimals,
 а возвращается с помощью метода getAllowDecimals.


Допустимые значения:


	- true.
	 Для оси диаграммы используются дробные значения (по умолчанию);


	- false.
	 Для оси диаграммы используются целочисленные значения.
