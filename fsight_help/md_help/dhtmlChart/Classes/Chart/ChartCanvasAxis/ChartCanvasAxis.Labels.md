# ChartCanvasAxis.Labels


## Синтаксис


Labels: Object;


## Описание


Свойство Labels определяет объект
 с настройками подписей оси диаграммы.


## Комментарии


Значение свойства устанавливается из JSON, а возвращается с помощью
 метода getLabels.
