# ChartCanvasAxis.getLinkedSeries


## Синтаксис


getLinkedSeries();


## Описание


Метод getLinkedSeries возвращает
 массив рядов данных, связанных с осью диаграммы.


## Комментарии


Метод возвращает массив объектов типа [PP.Ui.ChartCanvasSerie](../ChartCanvasSerie/ChartCanvasSerie.htm).
