# ChartCanvasAxis.Categories


## Синтаксис


Categories: Array;


## Описание


Свойство Categories определяет
 категории для оси диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setCategories,
 а возвращается с помощью метода getCategories.


Массив содержит числовые или строковые значения, представляющие категории
 диаграммы.
