# ChartCanvasAxis.getPixelValue


## Синтаксис


getPixelValue(value: Number, withoutLimits: Boolean);


## Параметры


value. Значение оси диаграммы;


withoutLimits. Признак того,
 что допустимый диапазон значений оси не учитываются. Допустимые значения:


	- true.
	 Координата оси диаграммы будет определена без учёта её минимального
	 и максимального значений;


	- false.
	 Координата оси диаграммы будет определена с учётом её минимального
	 и максимального значений (по умолчанию).


## Описание


Метод getPixelValue возвращает
 координату оси диаграммы для указанного значения.


## Комментарии


Метод возвращает значение типа Number.
