# ChartCanvasAxis.Opposite


## Синтаксис


Opposite: Boolean;


## Описание


Свойство Opposite определяет
 признак того, располагается ли ось диаграммы зеркально относительно своего
 начального положения.


## Комментарии


Свойство предназначено только для чтения.


Допустимые значения:


	- true.
	 Ось располагается зеркально относительно своего начального положения;


	- false.
	 Ось располагается в начальном положении (по умолчанию).


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 [Chart](../../../Components/Chart/Chart.htm) с наименованием
 «chart» (см. «[Пример
 создания точечной диаграммы](../../../Components/Chart/ChartScatter.htm)»). Расположим ось Y справа от области
 построения диаграммы:


// Получаем ось Y диаграммы
var axis = chart.getYAxis();
// Располагаем ось справа от области построения диаграммы
var settings = axis.getSettings();
settings.Position = "max";
axis.setSettings(settings);
// Перерисовываем диаграмму
chart.redraw(true);
if (axis.getOpposite()) {
    console.log("Ось построения диаграммы расположена зеркально относительно её начального положения");
} else {
    console.log("Расположение оси диаграммы не изменилось");
};
В результате выполнения примера ось Y была расположена справа от области
 построения диаграммы, то есть зеркально относительно её начального положения.
 Об этом свидетельствует уведомление, выведенное в консоли браузера:


Ось построения диаграммы расположена зеркально относительно
 её начального положения
