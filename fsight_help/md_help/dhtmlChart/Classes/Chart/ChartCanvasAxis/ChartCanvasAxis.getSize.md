# ChartCanvasAxis.getSize


## Синтаксис


getSize(ignoreSize: Boolean, calcShift: Boolean);


## Параметры


ignoreSize. Признак, разрешающий
 определять ширину оси даже в тех случаях, когда она меньше минимального
 значения. Допустимые значения:


	- true.
	 Ширина оси определяется, даже если её значение для оси меньше минимального;


	- false.
	 Ширина оси не определяется, если значение для оси меньше минимального
	 (по умолчанию);


calcShift. Признак того, необходимо
 ли учитывать сдвиг оси при одностороннем размещении нескольких осей. Допустимые
 значения:


	- true.
	 Сдвиг оси при одностороннем размещении нескольких осей необходимо
	 учитывать;


	- false.
	 Сдвиг оси при одностороннем размещении нескольких осей не требуется
	 учитывать (по умолчанию).


## Описание


Метод getSize возвращает ширину
 оси диаграммы в пикселях.


## Комментарии


Метод возвращает значение типа Number.
