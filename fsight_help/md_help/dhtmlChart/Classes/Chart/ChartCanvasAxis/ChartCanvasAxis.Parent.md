# ChartCanvasAxis.Parent


## Синтаксис


Parent: [PP.Ui.Chart](../Chart/Chart.htm);


## Описание


Свойство Parent определяет диаграмму,
 к которой принадлежит ось.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setParent,
 а возвращается с помощью метода getParent.
