# ChartCanvasAxis.Max


## Синтаксис


Max: Number;


## Описание


Свойство Max определяет максимальное
 значение оси диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMax,
 а возвращается с помощью метода getMax.


Метод setMax содержит параметр
 value, задающий максимальное значение оси диаграммы, параметр redraw,
 задающий признак необходимости перерисовки диаграммы, и параметр noAnimation,
 задающий признак использования анимации при перерисовке диаграммы.


Допустимые значения параметра redraw:


	- true.
	 Необходима перерисовка диаграммы (по умолчанию);


	- false.
	 Перерисовка диаграммы не требуется.


Допустимые значения параметра noAnimation:


	- true.
	 Анимация используется;


	- false.
	 Анимация не используется (по умолчанию).
