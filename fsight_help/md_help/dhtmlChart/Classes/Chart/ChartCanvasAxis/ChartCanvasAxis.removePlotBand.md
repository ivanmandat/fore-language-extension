# ChartCanvasAxis.removePlotBand


## Синтаксис


removePlotBand(band: Object, recalc: Boolean);


## Параметры


band. Пользовательская область
 оси диаграммы;


recalc. Признак перерасчёта
 оси диаграммы. Допустимые значения:


	- true.
	 Ось диаграммы будет перерасчитана;


	- false.
	 Ось диаграммы не будет перерасчитана (по умолчанию).


## Описание


Метод removePlotBand удаляет
 пользовательскую область оси диаграммы.
