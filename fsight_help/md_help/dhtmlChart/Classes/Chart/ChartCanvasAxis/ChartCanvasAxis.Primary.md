# ChartCanvasAxis.Primary


## Синтаксис


Primary: Boolean;


## Описание


Свойство Primary определяет
 признак того, является ли текущая ось основной.


## Комментарии


Значение свойства устанавливается из JSON и возвращается с помощью метода
 getPrimary.


Допустимые значения:


	- true.
	 Текущая ось является основной;


	- false.
	 Текущая ось является дополнительной (по умолчанию).
