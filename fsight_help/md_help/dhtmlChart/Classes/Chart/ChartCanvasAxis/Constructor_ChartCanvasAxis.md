# Конструктор ChartCanvasAxis


## Синтаксис


PP.Ui.ChartCanvasAxis(settings);


## Параметры

 свойств класса.


## Описание


Конструктор ChartCanvasAxis создает
 экземпляр класса ChartCanvasAxis.
