# ChartCanvasAxis.LabelMouseOver


## Синтаксис


LabelMouseOver: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.
 Доступные аргументы: Event - событие мыши, Tick - подпись оси диаграммы.


## Описание


Событие LabelMouseOver наступает
 при наведении курсора на подпись оси диаграммы.
