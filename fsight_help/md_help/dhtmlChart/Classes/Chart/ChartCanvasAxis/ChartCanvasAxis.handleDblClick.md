# ChartCanvasAxis.handleDblClick


## Синтаксис


handleDblClick(coords: Object, event: MouseEvent);


## Параметры


coords. Координаты, соответствующие
 текущему положению указателя мыши над диаграммой;


event. Событие мыши.


## Описание


Метод handleDblClick генерирует
 событие двойного нажатия на заголовок оси диаграммы.
