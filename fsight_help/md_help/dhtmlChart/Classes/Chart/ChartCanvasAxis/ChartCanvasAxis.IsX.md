# ChartCanvasAxis.IsX


## Синтаксис


IsX: Boolean;


## Описание


Свойство IsX определяет, является
 ли текущая ось осью категорий.


## Комментарии


Значение свойства устанавливается из JSON и возвращается с помощью метода
 getIsX.


Допустимые значения:


	- true.
	 Текущая ось является осью категорий;


	- false.
	 Текущая ось не является осью категорий (по умолчанию).
