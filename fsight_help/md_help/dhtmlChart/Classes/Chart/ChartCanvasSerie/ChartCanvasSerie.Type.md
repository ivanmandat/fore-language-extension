# ChartCanvasSerie.Type


## Синтаксис


Type: String;


## Описание


Свойство Type определяет тип
 ряда данных.


## Комментарии


Значение свойства устанавливается из JSON, а возвращается с помощью
 метода getType.


По умолчанию значение свойства равно «Line».


Допустимые значения:


	- Area.
	 Ряд данных для диаграммы с областями;


	- Column.
	 Ряд данных в виде столбцов;


	- Doughnut.
	 Ряд данных для кольцевой диаграммы;


	- Line.
	 Ряд данных для линейной диаграммы;


	- Pie.
	 Ряд данных для круговой диаграммы;


	- Radar.
	 Ряд данных для лепестковой диаграммы;


	- Scatter.
	 Ряд данных для точечной диаграммы;


	- Spline.
	 Ряд данных для сглаженной диаграммы;


	- Stepline.
	 Ряд данных для ступенчатой диаграммы.
