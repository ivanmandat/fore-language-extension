# ChartCanvasSerie.LineColor


## Синтаксис


LineColor: String;


## Описание


Свойство LineColor определяет
 цвет линий ряда данных диаграммы.


## Комментарии


Значение свойства устанавливается из JSON, а возвращается с помощью
 метода getLineColor.


Строка со значением цвета может быть представлена одним из следующих
 форматов:


	- "red";


	- "#000";


	- "RGB(123, 12, 1)".
