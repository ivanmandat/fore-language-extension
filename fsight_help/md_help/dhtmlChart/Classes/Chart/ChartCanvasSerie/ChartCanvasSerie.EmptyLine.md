# ChartCanvasSerie.EmptyLine


## Синтаксис


EmptyLine: Object;


## Описание


Свойство EmptyLine определяет
 объект с настройками линии соединяющей прореженные данные.


## Комментарии


Значение свойства устанавливается только из JSON.


## Пример


Пример объекта с настройками:


{
  Enabled: true,
  LineWidth: 2,
  LineDashStyle: "shortdot",
  LineColor: "rgba(96, 161, 250, 1.000000)"
}
Пример использования приведён на странице [ChartCanvasSerie.LineDashStyle](ChartCanvasSerie.LineDashStyle.htm).
