# ChartCanvasSerie.OnContextMenu


## Синтаксис


OnContextMenu: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие OnContextMenu наступает
 после вызова контекстного меню для ряда данных диаграммы.
