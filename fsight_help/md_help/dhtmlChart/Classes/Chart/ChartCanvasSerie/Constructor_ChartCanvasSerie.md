# Конструктор ChartCanvasSerie


## Синтаксис


PP.Ui.ChartCanvasSerie(settings);


## Параметры


settings. JSON-объект со значениями
 свойств класса.


## Описание


Конструктор ChartCanvasSerie создает
 экземпляр класса ChartCanvasSerie.
