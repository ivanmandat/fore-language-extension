# ChartCanvasSerie.DataIndex


## Синтаксис


DataIndex: Number;


## Описание


Свойство DataIndex определяет
 индекс ряда данных диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и возвращается с помощью метода
 getDataIndex.
