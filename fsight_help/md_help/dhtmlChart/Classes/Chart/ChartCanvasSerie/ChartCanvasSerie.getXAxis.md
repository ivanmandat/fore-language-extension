# ChartCanvasSerie.getXAxis


## Синтаксис


getXAxis();


## Описание


Метод getXAxis возвращает ось
 категорий ряда данных диаграммы.


## Комментарии


Метод возвращает значение типа [PP.Ui.ChartCanvasAxis](../ChartCanvasAxis/ChartCanvasAxis.htm).


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 [Chart](../../../Components/Chart/Chart.htm)
 с наименованием «chart» (см. «[Пример
 создания гистограммы](../../../Components/Chart/Chart_Example.htm)»). Получим ряд данных диаграммы, выведем значение
 одного из его элементов, затем получим информацию об оси категорий и оси
 значений, которые использует полученный ряд:


var serieNumber = 0;
// Получаем ряд данных
var serie = chart.getSeries()[serieNumber];
// Получаем количество элементов ряда данных
var count = serie.getPoints().length;
// Получаем последний элемент ряда данных
var point = serie.getPoint(count - 1);
// Выводим значение последнего элемента ряда данных
console.log("Ряд: " + serieNumber + " элемент: " + point.getIndex() + " значение: " + point.getY());
// Выводим сообщение о том, инвертированна ли ось категорий диаграммы
var xAxis = serie.getXAxis();
var isXAxisInverted = xAxis.getInverted() ? "инвертирована" : "не инвертирована";
console.log("Ось категорий диаграммы " + isXAxisInverted);
// Выводим сообщение о том, является ли ось значений основной
var yAxis = serie.getYAxis();
var isYAxisPrimary = yAxis.getPrimary() ? "основной" : "дополнительной";
console.log("Используются значения " + isYAxisPrimary + " оси диаграммы");
В результате выполнения примера в консоли браузера было выведено значение
 одного из элементов ряда диаграммы, а также значение признака инвертированности
 оси категорий и признак того, использовались ли для отображения ряда данных
 значения основной оси:


Ряд: 0 элемент: 4 значение: -30


Ось категорий диаграммы не инвертирована


Используются значения основной оси диаграммы
