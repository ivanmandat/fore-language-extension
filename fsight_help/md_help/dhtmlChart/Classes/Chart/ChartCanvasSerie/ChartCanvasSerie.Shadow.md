# ChartCanvasSerie.Shadow


## Синтаксис


Shadow: Object;


## Описание


Свойство Shadow определяет объект
 с настройками тени ряда.


## Комментарии


Значение свойства устанавливается только из JSON.


## Пример


Пример объекта с настройками тени:


{
  Blur: 2,
  Color: PP.SolidColorBrush,
  Enabled: true,
  X: 1,
  Y: 1
}
Пример использования приведён на странице [ChartCanvasSerie.LineDashStyle](ChartCanvasSerie.LineDashStyle.htm).
