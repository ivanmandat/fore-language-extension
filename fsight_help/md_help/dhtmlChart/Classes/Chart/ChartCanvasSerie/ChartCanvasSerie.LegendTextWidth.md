# ChartCanvasSerie.LegendTextWidth


## Синтаксис


LegendTextWidth: Number;


## Описание


Свойство LegendTextWidth определяет
 допустимую длину подписей элементов легенды диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLegendTextWidth,
 а возвращается с помощью метода getLegendTextWidth.
