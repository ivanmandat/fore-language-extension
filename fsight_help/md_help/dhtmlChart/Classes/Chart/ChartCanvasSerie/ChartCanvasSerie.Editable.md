# ChartCanvasSerie.Editable


## Синтаксис


Editable: Boolean;


## Описание


Свойство Editable определяет
 признак того, разрешено ли редактирование точек ряда данных диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и возвращается с помощью метода
 getEditable.


Допустимые значения:


	- true.
	 Редактирование точки ряда данных разрешено (по умолчанию);


	- false.
	 Редактирование точки ряда данных запрещено.
