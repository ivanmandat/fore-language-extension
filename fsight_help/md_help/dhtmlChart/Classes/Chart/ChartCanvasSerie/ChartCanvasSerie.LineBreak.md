# ChartCanvasSerie.LineBreak


## Синтаксис


LineBreak: Number;


## Описание


Свойство LineBreak определяет
 коэффициент преломления линии.


## Комментарии


Значение свойства устанавливается из JSON, а возвращается с помощью
 метода getLineBreak.


Свойство принимает значение от 0 до 1. 0 - слева, 1 - справа.
