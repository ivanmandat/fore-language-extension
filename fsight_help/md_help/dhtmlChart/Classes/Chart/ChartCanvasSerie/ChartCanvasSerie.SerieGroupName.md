# ChartCanvasSerie.SerieGroupName


## Синтаксис


SerieGroupName: String;


## Описание


Свойство SerieGroupName определяет
 название группы рядов.


## Комментарии


Значение свойства содержится только в родительском элементе.


Значение свойства устанавливается из JSON, а возвращается с помощью
 метода getSerieGroupName.
