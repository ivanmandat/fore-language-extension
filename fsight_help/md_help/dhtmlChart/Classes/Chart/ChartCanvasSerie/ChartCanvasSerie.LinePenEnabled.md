# ChartCanvasSerie.LinePenEnabled


## Синтаксис


LinePenEnabled: Boolean;


## Описание


Свойство LinePenEnabled определяет,
 отображаются ли линии, соединяющие точки ряда данных диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и возвращается с помощью метода
 getLinePenEnabled.


Допустимые значения:


	- true.
	 Линии, соединяющие точки ряда данных диаграммы, отображаются (по
	 умолчанию);


	- false.
	 Линии, соединяющие точки ряда данных диаграммы, не отображаются.
