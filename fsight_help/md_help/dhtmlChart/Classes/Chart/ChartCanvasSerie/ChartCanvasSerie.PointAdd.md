# ChartCanvasSerie.PointAdd


## Синтаксис


PointAdd: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие PointAdd наступает после
 добавления точки в ряд данных диаграммы.
