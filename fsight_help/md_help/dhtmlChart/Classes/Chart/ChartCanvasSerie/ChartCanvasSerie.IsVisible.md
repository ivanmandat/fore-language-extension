# ChartCanvasSerie.IsVisible


## Синтаксис


IsVisible: Boolean;


## Описание


Свойство IsVisible определяет,
 отображается ли ряд данных диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsVisible,
 а возвращается с помощью метода getIsVisible.


Допустимые значения:


	- true.
	 Ряд данных диаграммы отображается (по умолчанию);


	- false.
	 Ряд данных диаграммы не отображается.


Метод setIsVisible содержит
 параметр value, задающий признак того, будет ли отображаться ряд данных
 диаграммы, параметр redraw, задающий признак необходимости перерисовки
 диаграммы, и параметр reInit, задающий признак необходимости принудительной
 установки значения свойства.


Допустимые значения параметра redraw:


	- true.
	 Необходима перерисовка диаграммы;


	- false.
	 Перерисовка диаграммы не требуется (по умолчанию).


Допустимые значения параметра reInit:


	- true.
	 Необходима принудительная установка нового значения свойства;


	- false.
	 Если новое значение свойства совпадает с текущим, то принудительная
	 установка свойства и перерисовка диаграммы производиться не будут
	 (по умолчанию).
