# ChartCanvasSerie.IsNewborn


## Синтаксис


IsNewborn: Boolean;


## Описание


Свойство IsNewborn определяет
 признак анимированного появления нового ряда.


## Комментарии


Значение свойства устанавливается из JSON, а возвращается с помощью
 метода getIsNewborn.
