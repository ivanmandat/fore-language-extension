# ChartCanvasSerie.redraw


## Синтаксис


redraw(local: Boolean);


## Параметры


local. Признак необходимости
 вызова перерисовки в контексте отдельного ряда данных диаграммы. Допустимые
 значения:


	- true.
	 Необходим вызов перерисовки в контексте отдельного ряда данных
	 диаграммы;


	- false.
	 Вызов перерисовки в контексте отдельного ряда данных диаграммы не
	 требуется (по умолчанию).


## Описание


Метод redraw перерисовывает
 ряд данных диаграммы.
