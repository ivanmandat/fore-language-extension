# ChartCanvasSerie.LineWidth


## Синтаксис


LineWidth: Number;


## Описание


Свойство LineWidth определяет
 толщину линий ряда данных диаграммы.


## Комментарии


Значение свойства устанавливается из JSON, а возвращается с помощью
 метода getLineWidth.


По умолчанию значение свойства равно 2.
