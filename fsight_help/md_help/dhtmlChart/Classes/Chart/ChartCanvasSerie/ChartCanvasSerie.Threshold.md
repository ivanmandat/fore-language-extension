# ChartCanvasSerie.Threshold


## Синтаксис


Threshold: Number;


## Описание


Свойство Threshold определяет
 значение, с которого начинается отчёт оси Y диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setThreshold,
 а возвращается с помощью метода getThreshold.
