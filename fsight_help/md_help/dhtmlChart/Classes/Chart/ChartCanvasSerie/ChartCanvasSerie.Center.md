# ChartCanvasSerie.Center


## Синтаксис


Center: Object;


## Описание


Свойство Center определяет координаты
 центра ряда.


## Комментарии


Значение свойства устанавливается только из JSON.


Свойство актуально для круговых диаграмм.


Объект содержит координаты, например:


{
  X: 50,
  Y: 50
}
