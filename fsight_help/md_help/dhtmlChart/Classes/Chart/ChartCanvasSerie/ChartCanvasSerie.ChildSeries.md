# ChartCanvasSerie.ChildSeries


## Синтаксис


ChildSeries: Array;


## Описание


Свойство ChildSeries определяет
 коллекцию дочерних рядов.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setChildSeries,
 а возвращается с помощью метода getChildSeries.


## Пример


Свойство предназначено для внутреннего использования.
