# ChartCanvasSerie.ShowInLegend


## Синтаксис


ShowInLegend: Boolean;


## Описание


Свойство ShowInLegend определяет
 признак отображения ряда данных в легенде диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setShowInLegend,
 а возвращается с помощью метода getShowInLegend.


Допустимые значения:


	- true.
	 Ряд данных отображается в легенде диаграммы (по умолчанию);


	- false.
	 Ряд данных не отображается в легенде диаграммы.


Метод setShowInLegend содержит
 параметр value, задающий признак отображения ряда данных в легенде диаграммы,
 и параметр redraw, задающий признак необходимости перерисовки диаграммы.


Допустимые значения параметра redraw:


	- true.
	 Необходима перерисовка диаграммы;


	- false.
	 Перерисовка диаграммы не требуется (по умолчанию).
