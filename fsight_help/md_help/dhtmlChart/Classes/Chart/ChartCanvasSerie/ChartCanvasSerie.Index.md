# ChartCanvasSerie.Index


## Синтаксис


Index: Number;


## Описание


Свойство Index определяет индекс
 ряда данных диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIndex,
 а возвращается с помощью метода getIndex.


Метод setIndex содержит параметр
 value, задающий индекс ряда данных диаграммы, и параметр redraw, задающий
 признак необходимости перерисовки диаграммы.


Допустимые значения параметра redraw:


	- true.
	 Необходима перерисовка диаграммы (по умолчанию);


	- false.
	 Перерисовка диаграммы не требуется.
