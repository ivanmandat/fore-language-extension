# ChartCanvasSerie.Trend


## Синтаксис


Trend: Boolean;


## Описание


Свойство Trend определяет признак
 того, является ли ряд данных диаграммы линией тренда.


## Комментарии


Для создания нового тренда используйте свойство [ChartCanvasSerie.TrendLine](ChartCanvasSerie.TrendLine.htm).


Значение свойства устанавливается из JSON, а возвращается с помощью
 метода getTrend.


Допустимые значения:


	- true.
	 Ряд данных диаграммы является линией тренда;


	- false.
	 Ряд данных диаграммы не является линией тренда (по умолчанию).
