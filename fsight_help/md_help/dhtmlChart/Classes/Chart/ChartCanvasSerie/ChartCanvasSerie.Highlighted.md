# ChartCanvasSerie.Highlighted


## Синтаксис


Highlighted: Boolean;


## Описание


Свойство Highlighted определяет,
 подсвечен ли ряд данных диаграммы.


## Комментарии


Свойство предназначено только для чтения.


Допустимые значения:


	- true.
	 Ряд данных диаграммы подсвечен;


	- false.
	 Ряд данных диаграммы не подсвечен (по умолчанию).
