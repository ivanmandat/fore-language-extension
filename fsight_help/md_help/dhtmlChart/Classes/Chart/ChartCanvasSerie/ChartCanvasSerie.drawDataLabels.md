# ChartCanvasSerie.drawDataLabels


## Синтаксис


drawDataLabels(indexes: Array, useDrawGraphicContext:
 Boolean);


## Параметры


indexes. Массив точек ряда
 данных, для которых требуется отрисовать подписи данных;


useDrawGraphicContext. Признак
 использования слоя ряда данных для отрисовки подписей данных. Допустимые
 значения:


	- true.
	 Для отрисовки текстовых меток используется слой ряда данных;


	- false.
	 Для отрисовки текстовых меток используется собственный слой (по умолчанию).


## Описание


Метод drawDataLabels отрисовывает
 подписи данных для ряда данных диаграммы.
