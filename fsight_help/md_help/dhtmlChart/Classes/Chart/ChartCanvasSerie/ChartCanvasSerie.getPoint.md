# ChartCanvasSerie.getPoint


## Синтаксис


getPoint(index: Integer);


## Параметры


index. Индекс точки ряда данных.


## Описание


Метод getPoint возвращает точку
 ряда данных диаграммы по индексу.


## Комментарии


Метод возвращает значение типа [PP.Ui.ChartPoint](../ChartPoint/ChartPoint.htm).
