# ChartCanvasSerie.checkPointOnPlot


## Синтаксис


checkPointOnPlot(point: Object, approx: Number);


## Параметры


point. Описание параметра;


approx. Коэффициент аппроксимации.


## Описание


Метод checkPointOnPlot проверяет,
 находится ли точка ряда данных в области отрисовки диаграммы.


## Комментарии


Метод возвращает значение типа Boolean.


Допустимые значения:


	- true. Точка ряда данных
	 находится в области отрисовки диаграммы;


	- false. Точка ряда данных
	 находится вне области отрисовки диаграммы.
