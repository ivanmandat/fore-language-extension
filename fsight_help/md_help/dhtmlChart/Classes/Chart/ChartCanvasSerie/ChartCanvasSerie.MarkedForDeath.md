# ChartCanvasSerie.MarkedForDeath


## Синтаксис


MarkedForDeath: Boolean;


## Описание


Свойство MarkedForDeath определяет
 признак анимированного удаления.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMarkedForDeath,
 а возвращается с помощью метода getMarkedForDeath.
