# ChartCanvasSerie.removePoint


## Синтаксис


removePoint(index: Integer, redraw: Boolean);


## Параметры


index. Индекс удаляемой точки
 ряда данных диаграммы;


redraw. Признак перерисовки
 диаграммы. Допустимые значения:


	- true.
	 Диаграмма будет перерисована;


	- false.
	 Диаграмма не будет перерисована (по умолчанию).


## Описание


Метод removePoint удаляет точку
 из ряда данных диаграммы.
