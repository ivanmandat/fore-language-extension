# ChartCanvasSerie.IsMasterSerie


## Синтаксис


IsMasterSerie: Boolean;


## Описание


Свойство IsMasterSerie определяет,
 является ли ряд данных диаграммы прогнозным.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsMasterSerie,
 а возвращается с помощью метода getIsMasterSerie.


Допустимые значения:


	- true.
	 Ряд данных диаграммы является прогнозируемым;


	- false.
	 Ряд данных диаграммы не является прогнозируемым (по умолчанию).
