# ChartCanvasSerie.State


## Синтаксис


State: String;


## Описание


Свойство State определяет, наведён
 ли указатель мыши на ряд данных диаграммы.


## Комментарии


Свойство предназначено только для чтения.


Допустимые значения:


	- Blur. На ряд данных
	 не наведен указатель мыши, при этом в массиве рядов данных есть ряд,
	 который был выбран с помощью щелчка мыши;


	- Hover. На ряд данных
	 диаграммы наведён указатель мыши;


	- Normal. По умолчанию.
	 На ряд данных диаграммы не наведён указатель мыши;


	- Unhover. На ряд данных
	 не наведен указатель мыши, при этом в массиве рядов данных есть ряд,
	 на который наведён указатель мыши.
