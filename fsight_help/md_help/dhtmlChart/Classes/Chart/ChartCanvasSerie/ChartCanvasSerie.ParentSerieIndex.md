# ChartCanvasSerie.ParentSerieIndex


## Синтаксис


ParentSerieIndex: Number;


## Описание


Свойство ParentSerieIndex определяет
 индекс родительского ряда.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setParentSerieIndex,
 а возвращается с помощью метода getLineDashStyle.


При значении -1 ряд не имеет родительского ряда.


## Пример


Свойство предназначено для внутреннего использования.
