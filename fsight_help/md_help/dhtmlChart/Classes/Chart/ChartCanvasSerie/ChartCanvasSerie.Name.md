# ChartCanvasSerie.Name


## Синтаксис


Name: String;


## Описание


Свойство Name определяет наименование
 ряда данных диаграммы.


## Комментарии


Значение свойства устанавливается из JSON, а возвращается с помощью
 метода getName.
