# ChartCanvasSerie.Data


## Синтаксис


Data: Array;


## Описание


Свойство Data определяет массив
 данных ряда диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setData,
 а возвращается с помощью метода getData.


Массив содержит числовые значения, представляющие данные ряда диаграммы.


Метод setData содержит параметр
 value, задающий массив данных ряда диаграммы, и параметр redraw,
 задающий признак необходимости перерисовки диаграммы.


Допустимые значения параметра redraw:


	- true.
	 Необходима перерисовка диаграммы (по умолчанию);


	- false.
	 Перерисовка диаграммы не требуется.
