# ChartCanvasSerie.Color


## Синтаксис


Color: String;


## Описание


Свойство Color определяет цвет
 заливки ряда данных диаграммы.


## Комментарии


Значение свойства устанавливается из JSON, а возвращается с помощью
 метода getColor.


Строка со значением цвета может быть представлена одним из следующих
 форматов:


	- "red";


	- "#000";


	- "RGB(123, 12, 1)".
