# ChartScroller.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет
 отображение элемента прокрутки.


## Комментарии


Допустимые значения:


	- true.
	 Элемент прокрутки будет отображен;


	- false.
	 Элемент прокрутки отображен не будет (по умолчанию).


Значение свойства устанавливается из JSON и с помощью метода setEnabled,
 а возвращается с помощью метода getEnabled.
