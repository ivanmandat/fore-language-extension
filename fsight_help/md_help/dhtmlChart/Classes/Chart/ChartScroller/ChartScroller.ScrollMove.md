# ChartScroller.ScrollMove


## Синтаксис


ScrollMove: function(sender, args);


## Параметры


sender. Источник события;


args. Информация
 о событии. Аргументы: Position -
 текущая позиция элемента прокрутки.


## Описание


Событие ScrollMove наступает
 во время движения элемента прокрутки.
