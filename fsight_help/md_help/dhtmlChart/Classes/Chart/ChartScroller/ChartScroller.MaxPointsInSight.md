# ChartScroller.MaxPointsInSight


## Синтаксис


MaxPointsInSight: Number;


## Описание


Свойство MaxPointsInSight определяет
 максимальное количество отображаемых элементов оси X диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMaxPointsInSight,
 а возвращается с помощью метода getMaxPointsInSight.
