# ChartScroller.redraw


## Синтаксис


redraw(recalc: Boolean);


## Параметры


recalc. Признак пересчета свойств
 элемента прокрутки перед перерисовкой. Если для параметра установлено
 значение true, то свойства элемента
 будут пересчитаны.


## Описание


Метод redraw перерисовывает
 элемент прокрутки.
