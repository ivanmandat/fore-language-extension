# ChartScroller.getSize


## Синтаксис


getSize();


## Описание


Метод getSize возвращает высоту
 элемента прокрутки.


## Пример


Для выполнения примера необходимо наличие экземпляра класса [ChartScroller](ChartScroller.htm)
 с наименованием «scroller» (см. «[Конструктор
 ChartScroller](Constructor_ChartScroller.htm)»). Получим высоту элемента прокрутки:


// Получаем высоту элемента прокрутки
console.log("Высота элемента прокрутки: " + scroller.getSize());
В результат в консоль была выведена высота элемента прокрутки:


Высота элемента прокрутки: 35.800000000000004
