# ChartScroller.ScrollStart


## Синтаксис


ScrollStart: function(sender, args);


## Параметры


sender. Источник события;


args. Информация
 о событии. Аргументы: Position -
 позиция начала движения элемента прокрутки.


## Описание


Событие ScrollStart наступает
 в начале движения элемента прокрутки.
