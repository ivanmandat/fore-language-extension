# ChartScroller.Scrolled


## Синтаксис


Scrolled: function(sender, args);


## Параметры


sender. Источник события;


args. Информация
 о событии. Аргументы: Position -
 конечная позиция элемента прокрутки.


## Описание


Событие Scrolled наступает после
 окончания прокрутки.
