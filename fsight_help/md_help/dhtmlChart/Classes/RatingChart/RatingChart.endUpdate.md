# RatingChart.endUpdate


## Синтаксис


endUpdate();


## Описание


Метод endUpdate включает перерисовку
 рейтинговой диаграммы и обновляет её.


## Комментарии


Используется в связке с методом [RatingChart.beginUpdate](RatingChart.beginUpdate.htm).
