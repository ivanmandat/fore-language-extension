# RatingChart.NegativeLineBrush


## Синтаксис


NegativeLineBrush: [PP.SolidColorBrush](dhtmlCommon.chm::/Classes/PP/SolidColorBrush/SolidColorBrush.htm)
 | String;


## Описание


Свойство NegativeLineBrush определяет
 цвет заливки рядов данных при отрицательных значениях.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setNegativeLineBrush,
 а возвращается с помощью метода getNegativeLineBrush.
