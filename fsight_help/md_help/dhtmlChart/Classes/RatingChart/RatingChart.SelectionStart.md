# RatingChart.SelectionStart


## Синтаксис


SelectionStart: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.
 Доступный аргумент: LineIndex - позиция выделяемого ряда данных в диаграмме.


## Описание


Событие SelectionStart**
 генерируется при выделении ряда данных в рейтинговой диаграмме.
