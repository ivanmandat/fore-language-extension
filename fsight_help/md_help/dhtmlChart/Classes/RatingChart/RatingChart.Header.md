# RatingChart.Header


## Синтаксис


Header: [PP.Ui.RatingChartHeader](../RatingChartHeader/RatingChartHeader.htm);


## Описание


Свойство Header определяет строку
 заголовков рейтинговой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHeader,
 а возвращается с помощью метода getHeader.


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [RatingChart](../../Components/RatingChart/RatingChart.htm)
 с наименованием «ratingChart» (см. «[Пример
 создания компонента RatingChart](../../Components/RatingChart/RatingChart_Example.htm)»). Получим ширину заголовков столбцов
 рейтинговой диаграммы, разрешим последнему столбцу с текстом перекрывать
 следующий столбец при нехватке места и перерисуем строку заголовков:


// Получим строку заголовков
header = ratingChart.getHeader();
// Получим ширину заголовка столбца с номерами рядов данных
header.getVisibleIndexColumnWidth();
// Получим ширину заголовка столбца изменения позиций рядов данных
header.getVisibleMoveColumnWidth();
// Получим ширину заголовка столбца наименований рядов данных
header.getVisibleNameColumnWidth();
// Получим ширину заголовка первого столбца значений рядов данных
header.getVisibleDataColumnWidth(0);
// Получим ширину заголовка второго столбца значений рядов данных
header.getVisibleDataColumnWidth(1);
// На время отключим перерисовку рейтинговой диаграммы
header.getChart().beginUpdate();
// Разрешим последнему столбцу с текстом перекрывать следующий столбец, если не хватает места
header.setExpandLastColumn(true);
// Перерисуем строку заголовков
header.draw();
// Обновим рейтинговую диаграмму
header.getChart().endUpdate();

В результате выполнения примера последний столбец с текстом будет перекрывать
 следующий столбец при нехватке места, в окно консоли браузера будет выведена
 ширина заголовков столбцов, строка заголовков будет перерисована.
