# RatingChart.ActiveColumn


## Синтаксис


ActiveColumn: [PP.Ui.RatingChartDataColumn](../RatingChartDataColumn/RatingChartDataColumn.htm);


## Описание


Свойство ActiveColumn определяет
 активный столбец значений рейтинговой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setActiveColumn,
 а возвращается с помощью метода getActiveColumn.


Свойство ActiveColumn используется
 для определения столбца значений, по которому будет выполняться сортировка
 рядов данных рейтинговой диаграммы. Также столбец значений можно сделать
 активным, выделив его заголовок вручную. Текст данного столбца имеет полужирное
 начертание.
