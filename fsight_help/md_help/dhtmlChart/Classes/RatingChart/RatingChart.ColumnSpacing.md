# RatingChart.ColumnSpacing


## Синтаксис


ColumnSpacing: Number;


## Описание


Свойство ColumnSpacing определяет
 расстояние между столбцами рейтинговой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setColumnSpacing,
 а возвращается с помощью метода getColumnSpacing.
