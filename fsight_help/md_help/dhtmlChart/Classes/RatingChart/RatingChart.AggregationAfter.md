# RatingChart.AggregationAfter


## Синтаксис


AggregationAfter: Number;


## Описание


Свойство AggregationAfter**
 определяет индекс строки, после которой значения рядов данных будут агрегированы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setAggregationAfter,
 а возвращается с помощью метода getAggregationAfter.


По умолчанию свойство содержит значение «0».
