# RatingChart.ShowValueTimeout


## Синтаксис


ShowValueTimeout: Number;


## Описание


Свойство ShowValueTimeout определяет
 длительность анимации изменения рейтинговой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setShowValueTimeout,
 а возвращается с помощью метода getShowValueTimeout.
