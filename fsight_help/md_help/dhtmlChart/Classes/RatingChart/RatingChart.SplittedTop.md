# RatingChart.SplittedTop


## Синтаксис


SplittedTop: Number;


## Описание


Свойство SplittedTop определяет
 количество отображаемых над разделителем рядов данных при первой отрисовке
 рейтинговой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON, а возвращается с помощью
 метода getSplittedTop.


Для отображения разделителя в рейтинговой диаграмме свойство [RatingChart.ShowSplitter](RatingChart.ShowSplitter.htm) должно принимать
 значение true.


По умолчанию свойство содержит значение «0».
