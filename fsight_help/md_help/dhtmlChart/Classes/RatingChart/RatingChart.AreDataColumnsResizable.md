# RatingChart.AreDataColumnsResizable


## Синтаксис


AreDataColumnsResizable: Boolean;


## Описание


Свойство AreDataColumnsResizable
 определяет, будет ли доступно изменение ширины столбцов значений с помощью
 мыши.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setAreDataColumnsResizable,
 а возвращается с помощью метода getAreDataColumnsResizable.


Допустимые значения:


	- true.
	 Ширину столбцов значений можно изменять с помощью мыши;


	- false.
	 Значение по умолчанию. Изменение ширины столбцов значений с помощью
	 мыши недоступно.
