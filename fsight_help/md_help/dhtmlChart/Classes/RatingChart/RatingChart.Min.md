# RatingChart.Min


## Синтаксис


Min: Number;


## Описание


Свойство Min возвращает минимальное
 значение активного столбца значений.


## Комментарии


Свойство доступно только для чтения. Значение свойства возвращается
 с помощью метода getMin.


Для получения максимального значения используйте свойство [RatingChart.Max](RatingChart.Max.htm).
