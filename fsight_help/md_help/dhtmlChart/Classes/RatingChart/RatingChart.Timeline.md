# RatingChart.Timeline


## Синтаксис


Timeline: [PP.Ui.Timeline](dhtmlGraphicBase.chm::/Classes/Timeline/Timeline.htm);


## Описание


Свойство Timeline** определяет
 временную шкалу рейтинговой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTimeline,
 а возвращается с помощью метода getTimeline.
