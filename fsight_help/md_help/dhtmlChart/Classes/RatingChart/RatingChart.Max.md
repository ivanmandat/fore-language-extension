# RatingChart.Max


## Синтаксис


Max: Number;


## Описание


Свойство Max возвращает максимальное
 значение активного столбца значений.


## Комментарии


Свойство доступно только для чтения. Значение свойства возвращается
 с помощью метода getMax.


Для получения минимального значения используйте свойство [RatingChart.Min](RatingChart.Min.htm).


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [RatingChart](../../Components/RatingChart/RatingChart.htm)
 с наименованием «ratingChart» (см. «[Пример
 создания компонента RatingChart](../../Components/RatingChart/RatingChart_Example.htm)»). Получим максимальное и минимальное
 значения активного столбца значений:


// Получим максимальное значение
ratingChart.getMax();
// Получим минимальное значение
ratingChart.getMin();

В результате выполнения примера в окно консоли браузера будут выведены
 максимальное и минимальное значения активного столбца значений рейтинговой
 диаграммы.
