# RatingChart.TrendLine


## Синтаксис


TrendLine: PP.Ui.RatingChartTrend;


## Описание


Свойство TrendLine определяет
 линию тренда в рейтинговой диаграмме.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTrendLine,
 а возвращается с помощью метода getTrendLine.
