# RatingChart.Sorted


## Синтаксис


Sorted: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.
 Доступные аргументы: TimeSlice - индекс периода времени, Method - режим
 сортировки элементов в рейтинговой диаграмме, определяемый с помощью перечисления
 [PP.Sorting](dhtmlCommon.chm::/Enums/Sorting.htm), ActiveColumn
 - индекс активного столбца диаграммы.


## Описание


Событие Sorted** генерируется
 после завершения сортировки элементов в рейтинговой диаграмме.
