# RatingChart.beginUpdate


## Синтаксис


beginUpdate();


## Описание


Метод beginUpdate отключает
 перерисовку рейтинговой диаграммы до тех пор, пока не будет вызван метод
 [endUpdate](RatingChart.endUpdate.htm).
