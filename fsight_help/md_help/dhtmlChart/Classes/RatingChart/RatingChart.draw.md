# RatingChart.draw


## Синтаксис


draw(index: Number, noAnimation: Boolean);


## Параметры


*index.* Индекс шага временной
 шкалы, на котором нужно перерисовать данные;


*noAnimation.* Признак
 немедленной перерисовки данных без анимации. Если принимает значение true, анимация при перерисовке будет
 пропущена, false - перерисовка
 будет выполнена с анимацией.


## Описание


Метод draw перерисовывает рейтинговую
 диаграмму.
