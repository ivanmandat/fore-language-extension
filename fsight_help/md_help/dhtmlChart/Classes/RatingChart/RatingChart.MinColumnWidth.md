# RatingChart.MinColumnWidth


## Синтаксис


MinColumnWidth: Number;


## Описание


Свойство MinColumnWidth определяет
 минимальную ширину столбцов рейтинговой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMinColumnWidth,
 а возвращается с помощью метода getMinColumnWidth.
