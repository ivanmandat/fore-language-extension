# RatingChart.UseAnimation


## Синтаксис


UseAnimation: Boolean;


## Описание


Свойство UseAnimation**
 определяет признак воспроизведения анимации в рейтинговой диаграмме.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setUseAnimation,
 а возвращается с помощью метода getUseAnimation.


Если значение свойства равно true
 (по умолчанию), то в рейтинговой диаграмме доступно воспроизведение анимации,
 а если false - то запрещено.
