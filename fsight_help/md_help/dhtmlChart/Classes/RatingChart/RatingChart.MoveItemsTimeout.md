# RatingChart.MoveItemsTimeout


## Синтаксис


MoveItemsTimeout: Number;


## Описание


Свойство MoveItemsTimeout определяет
 длительность анимации изменения порядка рядов данных.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMoveItemsTimeout,
 а возвращается с помощью метода getMoveItemsTimeout.
