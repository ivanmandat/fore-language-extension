# RatingChart.DelayDuration


## Синтаксис


DelayDuration: Number;


## Описание


Свойство DelayDuration определяет
 длительность задержки анимации смены шага временной шкалы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDelayDuration,
 а возвращается с помощью метода getDelayDuration.
