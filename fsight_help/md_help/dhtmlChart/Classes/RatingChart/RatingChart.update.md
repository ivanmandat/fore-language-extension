# RatingChart.update


## Синтаксис


update(header: Boolean, values: Boolean, sizes:
 Boolean);


## Параметры


*header.* Признак обновления
 строки заголовков;


*values.* Признак обновления
 значений;


*sizes.* Признак обновления
 размеров столбцов.


## Описание


Метод update обновляет рейтинговую
 диаграмму.


## Комментарии


Метод обновляет элементы рейтинговой диаграммы, для которых соответствующий
 параметр принимает значение true.
