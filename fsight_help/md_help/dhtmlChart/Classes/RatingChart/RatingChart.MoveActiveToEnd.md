# RatingChart.MoveActiveToEnd


## Синтаксис


MoveActiveToEnd: Boolean;


## Описание


Свойство MoveActiveToEnd определяет,
 будет ли активный столбец значений переноситься в конец.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMoveActiveToEnd,
 а возвращается с помощью метода getMoveActiveToEnd.


Допустимые значения:


	- true.
	 Активный столбец значений всегда будет автоматически переноситься
	 в конец;


	- false.
	 Значение по умолчанию. Расположение столбцов значений не будет изменяться
	 вне зависимости от того, какой столбец активен.


Для определения активного столбца значений используйте свойство [RatingChart.ActiveColumn](RatingChart.ActiveColumn.htm).
