# RatingChart.ShowToolTip


## Синтаксис


ShowToolTip: Boolean;


## Описание


Свойство ShowToolTip определяет
 признак отображения всплывающей подсказки в рейтинговой диаграмме.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setShowToolTip,
 а возвращается с помощью метода getShowToolTip.


Если значение свойства равно true
 (по умолчанию), то отображение всплывающей подсказки в рейтинговой диаграмме
 разрешено, а если false - то запрещено.
