# RatingChart.NumberFormat


## Синтаксис


NumberFormat: String;


## Описание


Свойство NumberFormat определяет
 формат данных рейтинговой диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setNumberFormat,
 а возвращается с помощью метода getNumberFormat.
