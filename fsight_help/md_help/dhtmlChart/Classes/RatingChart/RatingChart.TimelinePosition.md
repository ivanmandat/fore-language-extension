# RatingChart.TimelinePosition


## Синтаксис


TimelinePosition: [PP.LTRB](dhtmlCommon.chm::/Enums/LTRB.htm);


## Описание


Свойство TimelinePosition**
 определяет расположение временной шкалы в рейтинговой диаграмме.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTimelinePosition,
 а возвращается с помощью метода getTimelinePosition.


По умолчанию свойство содержит значение PP.LTRB.Bottom.
