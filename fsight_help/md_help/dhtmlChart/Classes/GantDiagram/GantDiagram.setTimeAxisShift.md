# GantDiagram.setTimeAxisShift


## Синтаксис


setTimeAxisShift(value);


## Параметры


value. Number. Значение в px.


## Описание


Метод setTimeAxisShift устанавливает
 сдвиг диаграммы относительно шкалы времени.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 [TreeGantDiagram](../TreeGantDiagram/TreeGantDiagram.htm) (см.
 «[Пример
 создания компонента TreeGantDiagram](../../Components/TreeGantDiagram/TreeGantExample.htm)»).


// Получим диаграмму
Diag = GantD.getDiagram();
// Установим сдвиг относительно шкалы
Diag.setTimeAxisShift(100);

В результате выполнения примера был установлен сдвиг диаграммы относительно
 шкалы времени.
