# GantDiagram.setShowStartTimeLabel


## Синтаксис


setShowStartTimeLabel(value);


## Параметры


value. Boolean. Значение.


## Описание


Метод setShowStartTimeLabel
 устанавливает признак отображения начального времени диаграммы.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 [TreeGantDiagram](../TreeGantDiagram/TreeGantDiagram.htm) (см.
 «[Пример
 создания компонента TreeGantDiagram](../../Components/TreeGantDiagram/TreeGantExample.htm)»).


// Получим диаграмму
Diag = GantD.getDiagram();
// Установим отображение начального времени
Diag.setShowStartTimeLabel(true);

В результате выполнения примера было установлено отображение начального
 времени.
