# Компоненты диаграмм


## Описание


На данной странице приведены компоненты для работы с плоским деревом.


## Классы


		 Имя класса
		 Краткое описание


		 ![](../Class_Image.gif)
		 [Chart](Chart/Chart.htm)
		 Компонент Chart предназначен
		 для работы с диаграммой, построенной на основе элемента Canvas.


		 ![](../Class_Image.gif)
		 [ChartMaster](ChartMaster/ChartMaster.htm)
		 Компонент ChartMaster
		 представляет собой мастер для настройки диаграмм.


		 ![](../Class_Image.gif)
		 [ParallelCoordinates](ParallelCoordinates/ParallelCoordinates.htm)
		 Компонент ParallelCoordinates
		 представляет собой диаграмму с параллельными координатами.


		 ![](../Class_Image.gif)
		 [RatingChart](RatingChart/RatingChart.htm)
		 Компонент RatingChart
		 представляет собой рейтинговую диаграмму.
