# DependenceCriterion


## Описание


Перечисление DependenceCriterion
 содержит типы критериев отбора лучшей формы зависимости.


Используется свойством [ISmCurveEstimation.Criterion](../Interface/ISmCurveEstimation/ISmCurveEstimation.Criterion.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 ![](../Enums.gif)
		 0
		 R2. Коэффициент детерминации.
		 Процент объясненной дисперсии зависимой (объясняемой) переменной.
		 Выбирается модель с наибольшим значением.


		 ![](../Enums.gif)
		 1
		 R2Adj. Исправленный
		 коэффициент детерминации. Коэффициент детерминации скорректированный
		 на число регрессоров. Выбирается модель с наибольшим значением.


		 ![](../Enums.gif)
		 2
		 SSR. Сумма квадратов
		 остатков. Выбирается модель с наименьшим значением.


		 ![](../Enums.gif)
		 3
		 Serr. Стандартная ошибка
		 регрессии. Выбирается модель с наименьшим значением.
