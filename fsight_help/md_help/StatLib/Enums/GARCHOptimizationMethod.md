# GARCHOptimizationMethod


## Описание


Перечисление GARCHOptimizationMethod
 используется для определения метода оптимизации.


Используется следующим свойством:


	- [ISmGARCH.OptimizationMethod](../Interface/ISmGARCH/ISmGARCH.OptimizationMethod.htm).
