# TradingDayEffectsX12Type


## Описание


Перечисление TradingDayEffectsX12Type
 используется для определения вида поправок на рабочие дни.


Примечание.
 Метод X12 поддерживается только в ОС Windows.


Используется следующим свойством:


	- [ISmx12arima.TradingDayEffects](../Interface/ISmx12arima/ISmx12arima.TradingDayEffects.htm).
