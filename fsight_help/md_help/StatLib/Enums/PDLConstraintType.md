# PDLConstraintType


## Описание


Перечисление PDLConstraintType
 содержит методы отсечения слагаемых для лаговой переменной.


Используется следующими свойствами и методами:


	- [IMsPDLTermOptions.PDLConstraint](KeMs.chm::/Interface/IMsPDLTermOptions/IMsPDLTermOptions.PDLConstraint.htm);


	- [ISlPDLTerm.PDLConstraint](../Interface/ISlPDLTerm/ISlPDLTerm.PDLConstraint.htm).
