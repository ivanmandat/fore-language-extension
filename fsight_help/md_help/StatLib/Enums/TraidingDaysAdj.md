# TraidingDaysAdj


## Описание


Перечисление TraidingDaysAdj
 содержит варианты поправки на число рабочих дней.


Используется следующими свойствами и методами:


	- [IModelling.X11](KeMs.chm::/Interface/IModelling/IModelling.X11.htm);


	- [IMsCensus2Transform.TraidingDaysAdjustment](KeMs.chm::/Interface/IMsCensus2Transform/IMsCensus2Transform.TraidingDaysAdjustment.htm);


	- [ISmCensus2.TraidingDaysAdjustment](../Interface/ISmCensus2/ISmCensus2.TraidingDaysAdjustment.htm).
