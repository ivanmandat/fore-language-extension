# F0SpectrumEstimationType


## Описание


Перечисление F0SpectrumEstimationType
 используется для определения метода расчета скорректированной дисперсии.


Используется следующими свойствами:


	- [ISmElliotRothenbergStockTest.F0SpectrumEstimation](../Interface/ISmElliotRothenbergStockTest/ISmElliotRothenbergStockTest.F0SpectrumEstimation.htm);


	- [ISmKwiatkowskiPhillipsSchmidtShinTest.F0SpectrumEstimation](../Interface/ISmKwiatkowskiPhillipsSchmidtShinTest/ISmKwiatkowskiPhillipsSchmidtShinTest.F0SpectrumEstimation.htm);


	- [ISmNgPerronTest.F0SpectrumEstimation](../Interface/ISmNgPerronTest/ISmNgPerronTest.F0SpectrumEstimation.htm);


	- [ISmPhillipsPerronTest.F0SpectrumEstimation](../Interface/ISmPhillipsPerronTest/ISmPhillipsPerronTest.F0SpectrumEstimation.htm).
