# TransformationLogLevelType


## Описание


Перечисление TransformationLogLevelType
 используется для определения вида преобразования исходного ряда.


Используется следующим свойством:


	- [ISmTramoSeats.Transformation](../Interface/ISmTramoSeats/ISmTramoSeats.Transformation.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 ![](../Element%20Enums.gif)
		 0
		 Yes. Есть преобразование
		 (логарифмическое).


		 ![](../Element%20Enums.gif)
		 1
		 None. Нет преобразования.


		 ![](../Element%20Enums.gif)
		 2
		 Auto. Автоматически
		 определяет, преобразовывать или нет.
