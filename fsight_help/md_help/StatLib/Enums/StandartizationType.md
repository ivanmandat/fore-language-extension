# StandartizationType


## Описание


Перечисление StandartizationType
 содержит способы стандартизации признаков.


Используется свойством [ISlDistance.Standartization](../Interface/ISlDistance/ISlDistance.Standartization.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 ![](../Enums.gif)
		 0
		 ScaleStd. Через стандартное
		 отклонение.


		 ![](../Enums.gif)
		 1
		 ScaleRange. Как разность
		 между максимальным и минимальным значением.


		 ![](../Enums.gif)
		 2
		 ScaleUser. Значения,
		 заданные пользователем.


		 ![](../Enums.gif)
		 3
		 NoScale. Не проводить
		 стандартизацию.


## Комментарии


ScaleUser. При использовании
 данного способа необходимо задавать коэффициенты стандартизации в соответствующем
 свойстве.
