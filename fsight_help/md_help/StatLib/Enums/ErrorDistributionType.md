# ErrorDistributionType


## Описание


Перечисление ErrorDistributionType
 используется для определения типа распределения ошибок.


Используется свойством:


	- [ISmCensoredTruncatedRegression.DistributionType](../Interface/ISmCensoredTruncatedRegression/ISmCensoredTruncatedRegression.DistributionType.htm).
