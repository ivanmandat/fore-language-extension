# ARMAEstimationApproach


## Описание


Перечисление ARMAEstimationApproach
 используется для определения метода оценки коэффициентов авторегрессии
 и скользящего среднего.


Используется следующим свойством:


	- [ISlARMA.EstimationApproach](../Interface/ISlARMA/ISlARMA.EstimationApproach.htm).
