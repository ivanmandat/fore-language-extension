# InformationCriterionType


## Описание


Перечисление InformationCriterionType
 используется для определения типа информационного критерия.


Используется следующим свойством:


	- [IAutomaticLagSelectionSettings.InformationCriterion](../Interface/IAutomaticLagSelectionSettings/IAutomaticLagSelectionSettings.InformationCriterion.htm).
