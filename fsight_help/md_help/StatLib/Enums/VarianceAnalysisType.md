# VarianceAnalysisType


## Описание


Перечисление VarianceAnalysisType
 содержит типы дисперсионного анализа.


Используется свойством [ISmVarianceAnalysis.AnalysisType](../Interface/ISmVarianceAnalysis/ISmVarianceAnalysis.AnalysisType.htm).
