# ClusterCentersType


## Описание


Перечисление ClusterCentersType
 содержит варианты выбора начальных центров кластеров.


Используется следующим свойством:


	- [ISmKmeansClusterAnalysis.InitCenters](../Interface/ISmKmeansClusterAnalysis/ISmKmeansClusterAnalysis.InitCenters.htm).
