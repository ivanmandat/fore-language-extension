# LinearEqSolutionType


## Описание


Перечисление LinearEqSolutionType
 содержит способ вычисления следующего приближения для решения линеаризованной
 системы уравнений. Данная система должна быть вычислена при решении системы
 нелинейных уравнений методом Ньютона.


Используется следующим свойством:


	- [ISmNonLinearEquations.LinearEqSolutionMethod](../Interface/ISmNonLinearEquations/ISmNonLinearEquations.LinearEqSolutionMethod.htm).
