# LRCBandwidthMethodType


## Описание


Перечисление LRCBandwidthMethodType
 используется для определения метода пропускной способности в долгосрочной
 ковариации.


Используется следующим свойством:


	- [ISmLongRunCovariance.LRCBandwidthMethod](../Interface/ISmLongRunCovariance/ISmLongRunCovariance.LRCBandwidthMethod.htm).
