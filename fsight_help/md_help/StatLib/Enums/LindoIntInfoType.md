# LindoIntInfoType


## Описание


Перечисление LindoIntInfoType
 используется для определения типа целочисленной информации о методе расчета
 LINDO.


Используется следующим методом:


	- [ILindoSettings.GetIntLindoInfo](../Interface/ILindoSettings/ILindoSettings.GetIntLindoInfo.htm).


## Допустимые значения


		 Краткое описание


		 11303
		 IINFO_MIP_SIM_ITER.
		 Количество итераций симплекс-метода, выполненных для нахождения
		 решения задачи целочисленной оптимизации.


		 11304
		 IINFO_MIP_BAR_ITER.
		 Количество итераций барьерной функции, выполненных для нахождения
		 решения задачи целочисленной оптимизации.


		 11305
		 IINFO_MIP_NLP_ITER.
		 Количество нелинейных итераций, выполненных для поиска решения
		 задачи целочисленной оптимизации.


		 11306
		 IINFO_MIP_BRANCHCOUNT.
		 Количество ветвей, сгенерированных для решения задачи целочисленной
		 оптимизации.


		 11307
		 IINFO_MIP_NEWIPSOL.
		 Возвращает значение, соответствующее True,
		 если было найдено целочисленное решение.


		 11308
		 IINFO_MIP_LPCOUNT.
		 Количество решенных задач с ослабленными ограничениями.


		 11309
		 IINFO_MIP_ACTIVENODES.
		 Оставшееся количество неисследованных вершин.


		 11310
		 IINFO_MIP_LTYPE. Метод,
		 которым было найдено последнее целочисленное решение. Для получения
		 подробного описания типов решений обратитесь к [документации
		 LINDO](http://www.lindo.com/index.php?option=com_content&view=article&id=147&Itemid=69).


		 11311
