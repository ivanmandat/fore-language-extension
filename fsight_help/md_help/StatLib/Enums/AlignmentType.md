# AlignmentType


## Описание


Перечисление AlignmentType используется
 для определения типа центрирования матрицы.


Используется следующими свойствами:


	- [ISmPrincipalComponentAnalysis.MatrixAlignmentType](../Interface/ISmPrincipalComponentAnalysis/ISmPrincipalComponentAnalysis.MatrixAlignmentType.htm);


	- [ISmSingularSpectrumAnalysis.MatrixAlignmentType](../Interface/ISmSingularSpectrumAnalysis/ISmSingularSpectrumAnalysis.MatrixAlignmentType.htm).
