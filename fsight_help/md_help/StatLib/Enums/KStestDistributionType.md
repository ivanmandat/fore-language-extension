# KStestDistributionType


## Описание


Перечисление KStestDistributionType
 содержит законы распределения наблюдаемой случайной величины.


Используется следующим свойством:


	- [ISmKolmogorovSmirnovTest.Distribution](../Interface/ISmKolmogorovSmirnovTest/ISmKolmogorovSmirnovTest.Distribution.htm).
