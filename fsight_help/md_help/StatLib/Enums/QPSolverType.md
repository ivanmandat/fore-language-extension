# QPSolverType


## Описание


Перечисление QPSolverType содержит
 типы модулей решения, которые могут использоваться в задаче квадратичного
 программирования.


Используется следующими свойствами и методами:


	- [ISmQuadraticProgramming.SolverType](../Interface/ISmQuadraticProgramming/ISmQuadraticProgramming.SolverType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 ![](../Element%20Enums.gif)
		 0
		 Platform. Модуль решения,
		 реализованный в «Форсайт. Аналитическая платформа».


		 ![](../Element%20Enums.gif)
		 1
		 Grb. Внешний модуль
		 решения Gurobi.


## Комментарии


Для использования внешнего модуля решений Gurobi соответствующее программное
 обеспечение должно быть предварительно установлено на компьютере пользователя.
