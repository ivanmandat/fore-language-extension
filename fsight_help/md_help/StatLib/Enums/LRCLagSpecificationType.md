# LRCLagSpecificationType


## Описание


Перечисление LRCLagSpecificationType
 используется для определения спецификации лагов в долгосрочной ковариации.


Используется следующим свойством:


	- [ISmLongRunCovariance.LRCLagSpecification](../Interface/ISmLongRunCovariance/ISmLongRunCovariance.LRCLagSpecification.htm).
