# LRWeightsType


## Описание


Перечисление LRWeightsType используется
 для определения типа весов для линейной регрессии.


Используется следующим свойством:


	- [ISmLinearRegress.WeightsType](../Interface/ISmLinearRegress/ISmLinearRegress.WeightsType.htm).
