# DendrogramOrientationType


## Описание


Перечисление DendrogramOrientationType
 содержит типы ориентации дендрограммы.


Используется свойством [ISmHierarchicalClusterAnalysis.DendrogramOrientation](../Interface/ISmHierarchicalClusterAnalysis/ISmHierarchicalClusterAnalysis.DendrogramOrientation.htm).
