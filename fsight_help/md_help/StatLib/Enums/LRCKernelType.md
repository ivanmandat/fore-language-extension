# LRCKernelType


## Описание


Перечисление LRCKernelType используется
 для определения спецификации ядра в долгосрочной ковариации.


Используется следующим свойством:


	- [ISmLongRunCovariance.LRCKernel](../Interface/ISmLongRunCovariance/ISmLongRunCovariance.LRCKernel.htm).
