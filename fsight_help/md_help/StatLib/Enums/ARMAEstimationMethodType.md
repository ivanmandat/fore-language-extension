# ARMAEstimationMethodType


## Описание


Перечисление ARMAEstimationMethodType
 используется для определения метода оптимизации.


Используется следующим свойством:


	- [ISlARMA.EstimationMethod](../Interface/ISlARMA/ISlARMA.EstimationMethod.htm).
