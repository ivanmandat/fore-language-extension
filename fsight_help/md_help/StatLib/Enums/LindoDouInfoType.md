# LindoDouInfoType


## Описание


Перечисление LindoDouInfoType
 используется для определения типа вещественной информации о методе расчета
 LINDO.


Используется следующим методом:


	- [ILindoSettings.GetDouLindoInfo](../Interface/ILindoSettings/ILindoSettings.GetDouLindoInfo.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 11301
		 DINFO_MIP_OBJ. Целевое
		 значение задачи целочисленной оптимизации.


		 11302
		 DINFO_MIP_BESTBOUND.
		 Лучшее ограничение целевой функции в задаче целочисленной оптимизации.


		 11347
		 DINFO_MIP_TOT_TIME.
		 Общее время расчета задачи целочисленной оптимизации, включая
		 время на загрузку и выгрузку модели, оптимизацию, эвристику.


		 11348

		 Процессорное время, затраченное на целочисленную оптимизацию.

		 оптимизации и на другие эвристики.


		 11351

		 DINFO_MIP_ROOT_OPT_TIME.
