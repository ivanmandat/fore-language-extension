# SeasonalFilterType


## Описание


Перечисление SeasonalFilterType
 используется для определения типа сезонных фильтров.


Примечание.
 Метод X12 поддерживается только в ОС Windows.


Используется следующим свойством:


	- [ISmx12arima.SeasonalFilter](../Interface/ISmx12arima/ISmx12arima.SeasonalFilter.htm).
