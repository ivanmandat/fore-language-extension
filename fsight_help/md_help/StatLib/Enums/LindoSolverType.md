# LindoSolverType


## Описание


Перечисление LindoSolverType
 используется для определения метода решения, используемого модулем LINDO.


Используется следующим свойством:


	- [ILindoSettings.SolverType](../Interface/ILindoSettings/ILindoSettings.SolverType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Free. Метод определяется
		 автоматически.


		 1
		 PSimplex. Симплекс-метод.


		 2
		 DSimplex. Двойственный
		 симплекс метод.


		 3
		 Barrier. Барьерная
		 функция.


		 4
		 Nlp. Нелинейный метод
		 решения.
