# AnalysisType


## Описание


Перечисление AnalysisType содержит типы анализа исходных данных.


Используется следующим свойством:


-
[ISlAnalysis.Type](../Interface/ISlAnalysis/ISlAnalysis.Type.htm).


## Допустимые значения


 Значение
 Краткое описание


 ![](../Enums.gif)
 0
 Correlation. Корреляционная матрица исходных данных.


 ![](../Enums.gif)
 1
 Standardised. Стандартизированная матрица исходных данных. При использовании данного типа анализа необходимо задать коэффициенты стандартизации ([ISlAnalysis.StandartizationCoefficients](../Interface/ISlAnalysis/ISlAnalysis.StandartizationCoefficients.htm)).


 ![](../Enums.gif)
 2
 SumSq. Матрица сумм квадратов и векторных произведений.


 ![](../Enums.gif)
 3
 VarCovar. Ковариационная матрица исходных данных.
