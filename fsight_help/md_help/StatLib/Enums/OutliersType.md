# OutliersType


## Описание


Перечисление OutliersType используется
 для определения режима выявления выбросов.


Используется следующим свойством:


	- [ISmTramoSeats.OutliersDetection](../Interface/ISmTramoSeats/ISmTramoSeats.OutliersDetection.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 ![](../Element%20Enums.gif)
		 0
		 None. Выбросы не подбираются
		 автоматически.


		 ![](../Element%20Enums.gif)
		 1
		 AutoDetectAll. Определение
		 выбросов всех типов.


		 ![](../Element%20Enums.gif)
		 2
		 AddTemp. Определяются
		 только аддитивные выбросы и выбросы типа «сдвиг уровня».


		 ![](../Element%20Enums.gif)
		 3
		 AddTempLevel. Определяются
		 аддитивные выбросы, выбросы типа «сдвиг уровня» и «временный сдвиг
		 уровня».


		 ![](../Element%20Enums.gif)
		 4
		 AddLevel. Определяются
		 только аддитивные выбросы и выбросы типа «временный сдвиг уровня».
