# DecisionTreePropertyType


## Описание


Перечисление DecisionTreePropertyType
 используется для определения типа классифицирующей переменной.


Используется следующим свойством:


	- [IBinaryTreeNode.PropertyType](../Interface/IBinaryTreeNode/IBinaryTreeNode.PropertyType.htm).
