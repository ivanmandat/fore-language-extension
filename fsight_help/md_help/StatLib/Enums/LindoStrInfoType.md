# LindoStrInfoType


## Описание


Перечисление LindoStrInfoType
 используется для определения типа строковой информации о методе расчета
 LINDO.


Используется следующим методом:


	- [ILindoSettings.GetStrLindoInfo](../Interface/ILindoSettings/ILindoSettings.GetStrLindoInfo.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 11368
		 SINFO_MIP_THREAD_LOAD.
		 Строка, содержащая поток рабочей нагрузки в последнем вызове LSsolveMIP.
