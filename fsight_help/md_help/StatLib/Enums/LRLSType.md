# LRLSType


## Описание


Перечисление LRLSType используется
 для определения типа метода наименьших квадратов.


Используется следующим свойством:


	- [ISmLinearRegress.LSType](../Interface/ISmLinearRegress/ISmLinearRegress.LSType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 ![](../Element%20Enums.gif)
		 0
		 GLS. Обобщенный метод
		 наименьших квадратов.


		 ![](../Element%20Enums.gif)
		 1
		 FGLS. Доступный обобщенный
		 метод наименьших квадратов.
