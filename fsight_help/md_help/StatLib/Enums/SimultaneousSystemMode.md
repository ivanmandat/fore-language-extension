# SimultaneousSystemMode


## Описание


Перечисление SimultaneousSystemMode
 используется для определения метода оценки линейных систем одновременных
 уравнений.


Используется следующим свойством:


	- [ISmSimultaneousSystem.SimultaneousSystemMethod](../Interface/ISmSimultaneousSystem/ISmSimultaneousSystem.SimultaneousSystemMethod.htm).
