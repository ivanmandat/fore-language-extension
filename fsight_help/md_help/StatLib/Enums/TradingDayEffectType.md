# TradingDayEffectType


## Описание


Перечисление TradingDayEffectsType
 используется для определения типа поправок на рабочие дни.


Используется следующим свойством:


	- [ISmTramoSeats.TradingDayEffects](../Interface/ISmTramoSeats/ISmTramoSeats.TradingDayEffects.htm).


## Допустимые значения
