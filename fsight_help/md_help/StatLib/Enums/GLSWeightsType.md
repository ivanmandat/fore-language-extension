# GLSWeightsType


## Описание


Перечисление GLSWeightsType
 используется для определения типа весов для регрессии на панельных данных.


Используется следующим свойством:


	- [ISmPooledModel.GLSWeights](../Interface/ISmPooledModel/ISmPooledModel.GLSWeights.htm).
