# WeightsScalingType


## Описание


Перечисление WeightsScalingType
 используется для определения метода масштабирования весов.


Используется следующим свойством:


	- [ISmLinearRegress.WeightsScaling](../Interface/ISmLinearRegress/ISmLinearRegress.WeightsScaling.htm).
