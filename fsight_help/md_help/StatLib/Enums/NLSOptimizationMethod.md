# NLSOptimizationMethod


## Описание


Перечисление NLSOptimizationMethod
 используется для определения метода оптимизации в нелинейном МНК.


Используется следующими свойствами:


	- [IMsNonLinearRegressionTransform.OptimizationMethod](KeMs.chm::/Interface/IMsNonLinearRegressionTransform/IMsNonLinearRegressionTransform.OptimizationMethod.htm);


	- [ISmNonLinearLeastSquare.OptimizationMethod](../Interface/ISmNonLinearLeastSquare/ISmNonLinearLeastSquare.OptimizationMethod.htm).
