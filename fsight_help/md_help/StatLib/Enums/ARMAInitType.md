# ARMAInitType


## Описание


Перечисление ARMAInitType содержит методы определения начальных приближений.


Используется следующим свойством:


-
[ISlARMA.CalcInitMode](../Interface/ISlARMA/ISlARMA.CalcInitMode.htm).


## Допустимые значения


 Значение
 Краткое описание


 ![](../Enums.gif)
 0
 Manual. Начальные приближения задаются вручную.


 ![](../Enums.gif)
 1
 Auto. Начальные приближения определяются автоматически.
