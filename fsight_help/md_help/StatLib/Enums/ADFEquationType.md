# ADFEquationType


## Описание


Перечисление ADFEquationType используется для определения авторегрессионной модели, применяемой в тесте Дики-Фуллера и тесте Филлипса-Перрона.


Используется следующими свойствами:


-
[IMsDFTestSettings.EquationType](KeMs.chm::/Interface/IMsDFTestSettings/IMsDFTestSettings.EquationType.htm);


-
[ISmAugmentDickeyFullerTest.EquationType](../Interface/ISmAugmentDickeyFullerTest/ISmAugmentDickeyFullerTest.EquationType.htm);


-
[ISmPhillipsPerronTest.Equation](../Interface/ISmPhillipsPerronTest/ISmPhillipsPerronTest.Equation.htm).
