# HCStoppingCriterionType


Перечисление HCStoppingCriterionType
 используется для определения критерия останова в иерархическом кластерном
 анализе.


Используется следующим свойством:


	- [ISmHierarchicalClusterAnalysis.HCStoppingCriterion](../Interface/ISmHierarchicalClusterAnalysis/ISmHierarchicalClusterAnalysis.HCStoppingCriterion.htm).
