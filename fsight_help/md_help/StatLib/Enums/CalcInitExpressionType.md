# CalcInitExpressionType


## Описание


Перечисление CalcInitExpressionType содержит режимы вычисления начальных значения искомых переменных.


Используется следующим свойством:


-
[ISmNonLinearEquations.CalcInitExpressionMode](../Interface/ISmNonLinearEquations/ISmNonLinearEquations.CalcInitExpressionMode.htm).


## Допустимые значения


 Значение
 Краткое описание


 ![](../Element%20Enums.gif)
 0
 Auto. Начальные значения определяются автоматически. В качестве начального значения искомой переменной принимается последняя точка идентификационного периода.


 ![](../Element%20Enums.gif)
 1
 Manual. Начальные значения определяются пользователем.
