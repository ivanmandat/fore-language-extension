# TrendSpecificationType


## Описание


Перечисление TrendSpecificationType используется
 для определения спецификации тренда в основной и дополнительной регрессиях.


Используется следующими свойствами:


	- [ISmCointegratingRegression.TrendSpecification](../Interface/ISmCointegratingRegression/ISmCointegratingRegression.TrendSpecification.htm);


	- [ISmEngleGrangerTest.TrendSpecification](../Interface/ISmEngleGrangerTest/ISmEngleGrangerTest.TrendSpecification.htm);


	- [ISmPhillipsOuliarisTest.TrendSpecification](../Interface/ISmPhillipsOuliarisTest/ISmPhillipsOuliarisTest.TrendSpecification.htm).
