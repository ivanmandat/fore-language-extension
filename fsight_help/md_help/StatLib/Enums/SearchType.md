# SearchType


## Описание


Перечисление SearchType содержит типы автоподборов параметров.


Используется следующим свойством:


-
[IExponentialSmoothingAutoSearch.Mode](../Interface/IExponentialSmoothingAutoSearch/IExponentialSmoothingAutoSearch.Mode.htm).


## Допустимые значения


 Значение
 Краткое описание


 ![](../Enums.gif)
 0
 NotSearch. Поиск не осуществляется.


 ![](../Enums.gif)
 1
 Grid. Поиск по сетке. В случае выбора необходимо настроить параметры метода наискорейшего спуска [QuickestDescentMethod](../Interface/IExponentialSmoothingAutoSearch/IExponentialSmoothingAutoSearch.QuickestDescentMethod.htm).


 ![](../Enums.gif)
 2
 Optimal. Автоматический поиск. В случае выбора необходимо настроить параметры метода наилучшей пробы [BestTrialMethod](../Interface/IExponentialSmoothingAutoSearch/IExponentialSmoothingAutoSearch.BestTrialMethod.htm).
