# StabilityAnalysisofSeasonalsX12Type


## Описание


Перечисление StabilityAnalysisofSeasonalsX12Type
 используется для определения способа анализа стабильности.


Примечание.
 Метод X12 поддерживается только в ОС Windows.


Используется следующим свойством:


	- [ISmx12arima.StabilityAnalysisofSeasonals](../Interface/ISmx12arima/ISmx12arima.StabilityAnalysisofSeasonals.htm).
