# KModesDistanceType


## Описание


Перечисление KModesDistanceType
 используется для определения способа расчета расстояния между точками
 при кластеризации методом к-мод.


Используется следующим свойством:


	- [ISmKmeansClusterAnalysis.KModesDistance](../Interface/ISmKmeansClusterAnalysis/ISmKmeansClusterAnalysis.KModesDistance.htm).
