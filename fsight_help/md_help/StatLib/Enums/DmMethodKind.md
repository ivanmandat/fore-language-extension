# DmMethodKind


## Описание


Перечисление DmMethodKind используется
 для определения метода интеллектуального анализа данных.


Используется свойством:


	- [IDmMethod.Kind](KeMs.chm::/Interface/IDmMethod/IDmMethod.Kind.htm).
