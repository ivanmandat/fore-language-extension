# SeasonalityCycleType


## Описание


Перечисление SeasonalityCycleType
 содержит типы дополнительной сезонности, которые задают период сезонности.


Используется следующими свойствами и методами:


	- [ISmCensus2.SeasonalComponentCycleType](../Interface/ISmCensus2/ISmCensus2.SeasonalComponentCycleType.htm);


	- [ISmx12arima.SeasonalComponentCycleType](../Interface/ISmx12arima/ISmx12arima.SeasonalComponentCycleType.htm);


	- [ISmTramoSeats.SeasonalComponentCycleType](../Interface/ISmTramoSeats/ISmTramoSeats.SeasonalComponentCycleType.htm).
