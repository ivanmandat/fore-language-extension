# AdjustmentOptionsX12Type


## Описание


Перечисление AdjustmentOptionsX12Type
 используется для определения способа учета поправок на праздничные и рабочие
 дни.


Примечание.
 Метод X12 поддерживается только в ОС Windows.


Используется следующим свойством:


	- [ISmx12arima.AdjustmentOptions](../Interface/ISmx12arima/ISmx12arima.AdjustmentOptions.htm).
