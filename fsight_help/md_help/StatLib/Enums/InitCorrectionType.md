# InitCorrectionType


## Описание


Перечисление InitCorrectionType
 содержит типы предварительной корректировки.


Используется свойством [ISmUnivariateSpectrumAnalysis.InitCorrection](../Interface/ISmUnivariateSpectrumAnalysis/ISmUnivariateSpectrumAnalysis.InitCorrection.htm).
