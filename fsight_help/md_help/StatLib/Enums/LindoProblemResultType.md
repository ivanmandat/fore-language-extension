# LindoProblemResultType


## Описание


Перечисление LindoProblemResultType
 используется для определения тип решения, который должен быть получен
 модулем LINDO


Используется следующим свойством:


	- [ILindoSettings.ProblemResultType](../Interface/ILindoSettings/ILindoSettings.ProblemResultType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Solution. Решение
		 задачи глобальной оптимизации.


		 1
		 MIPPrimalSolution.
		 Решение задачи целочисленной оптимизации.


		 2
		 PrimalSolution. Решение
		 задачи оптимизации.
