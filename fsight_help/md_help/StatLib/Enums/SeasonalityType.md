# SeasonalityType


## Описание


Перечисление SeasonalityType
 используется для определения типа сезонности.


Используется следующими свойствами и методами:


	- [IModelling.Expsmooth](KeMs.chm::/Interface/IModelling/IModelling.Expsmooth.htm);


	- [IModelling.Extrapolate](KeMs.chm::/Interface/IModelling/IModelling.Extrapolate.htm);


	- [IModelling.X11](KeMs.chm::/Interface/IModelling/IModelling.X11.htm);


	- [IMsCensus1Transform.Seasonality](KeMs.chm::/KeMs/Interface/IMsCensus1Transform/IMsCensus1Transform.Seasonality.htm);


	- [IMsCensus2Transform.Seasonality](KeMs.chm::/Interface/IMsCensus2Transform/IMsCensus2Transform.Seasonality.htm);


	- [ITsInversionInfo.Seasonality](KeTransform.chm::/Interface/ITsInversionInfo/ITsInversionInfo.Seasonality.htm);


	- [ISeasonal.Mode](../Interface/ISeasonal/ISeasonal.Mode.htm);


	- [IModelling.Census1](KeMs.chm::/Interface/IModelling/IModelling.Census1.htm).
