# SeasonalityTypeX12


## Описание


Перечисление SeasonalityTypeX12 используется
 для определения вида сезонности.


Примечание.
 Метод X12 поддерживается только в ОС Windows.


Используется следующим свойством:


	- [ISmx12arima.SeasonalAdjustmentMode](../Interface/ISmx12arima/ISmx12arima.SeasonalAdjustmentMode.htm).
