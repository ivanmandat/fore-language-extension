# LagLeadMethodType


## Описание


Перечисление LagLeadMethodType
 используется для определения типа оценки коинтеграционной регрессии методом
 DOLS.


Используется следующим свойством:


	- [ISmCointegratingRegression.LagLeadMethod](../Interface/ISmCointegratingRegression/ISmCointegratingRegression.LagLeadMethod.htm).
