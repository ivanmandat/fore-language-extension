# LRCWindowType


## Описание


Перечисление LRCWindowType используется
 для определения типа окна в долгосрочной ковариации.


Используется следующим свойством:


	- [ISmLongRunCovariance.LRCWindow](../Interface/ISmLongRunCovariance/ISmLongRunCovariance.LRCWindow.htm).
