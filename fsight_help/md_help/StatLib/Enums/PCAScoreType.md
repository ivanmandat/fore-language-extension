# PCAScoreType


## Описание


Перечисление PCAScoreType содержит типы условий, налагаемых на главные компоненты.


Используется следующим свойством:


-
[ISmPrincipalComponentAnalysis.ScoreType](../Interface/ISmPrincipalComponentAnalysis/ISmPrincipalComponentAnalysis.ScoreType.htm).


## Допустимые значения


 Значение
 Краткое описание


 ![](../Enums.gif)
 0
 Stand. Главные компоненты ортогональны.


 ![](../Enums.gif)
 1
 NotStand. Условие не накладывается.


 ![](../Enums.gif)
 2
 UnitVar. Главные компоненты имеют единичную дисперсию.


 ![](../Enums.gif)
 3
 Eigenval. Главные компоненты имеют дисперсию, равную соответствующим собственным числам.
