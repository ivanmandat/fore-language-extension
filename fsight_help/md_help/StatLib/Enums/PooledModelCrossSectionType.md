# PooledModelCrossSectionType


## Описание


Перечисление PooledModelCrossSectionType
 содержит типы моделей для регрессии на панельных данных.


Используется следующими свойствами и методами:


	- [IMsPooledModelTransform.CrossSection](KeMs.chm::/Interface/IMsPooledModelTransform/IMsPooledModelTransform.CrossSection.htm);
