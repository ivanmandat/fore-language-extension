# DataTransformationX12Type


## Описание


Перечисление DataTransformationX12Type
 используется для определения вида трансформации исходного ряда.


Примечание.
 Метод X12 поддерживается только в ОС Windows.


Используется следующим свойством:


	- [ISmx12arima.DataTransformation](../Interface/ISmx12arima/ISmx12arima.DataTransformation.htm).
