# CTModelType


## Описание


Перечисление CTModelType используется
 для определения типа модели регрессии.


Используется свойством:


	- [ISmCensoredTruncatedRegression.ModelType](../Interface/ISmCensoredTruncatedRegression/ISmCensoredTruncatedRegression.ModelType.htm).
