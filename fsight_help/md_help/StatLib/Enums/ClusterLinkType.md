# ClusterLinkType


## Описание


Перечисление ClusterLinkType
 содержит способы расчета расстояний между кластерами.


Используется свойством [ISmHierarchicalClusterAnalysis.ClusterLink](../Interface/ISmHierarchicalClusterAnalysis/ISmHierarchicalClusterAnalysis.ClusterLink.htm).
