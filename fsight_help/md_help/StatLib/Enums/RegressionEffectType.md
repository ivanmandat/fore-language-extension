# RegressionEffectType


## Описание


Перечисление RegressionEffectType
 используется для определения типа регрессора.


Используется следующими свойствами:


	- [ISlTramoSeatsExogenousVar.RegressionEffect](../Interface/ISlTramoSeatsExogenousVar/ISlTramoSeatsExogenousVar.RegressionEffect.htm);


	- [ISlTramoSeatsOutlier.RegressionEffect](../Interface/ISlTramoSeatsOutlier/ISlTramoSeatsOutlier.RegressionEffect.htm).
