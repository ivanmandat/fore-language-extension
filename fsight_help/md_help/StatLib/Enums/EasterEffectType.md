# EasterEffectType


## Описание


Перечисление EasterEffectType
 используется для определения типа поправок на праздник Пасхи.


Используется следующим свойством:


	- [ISmTramoSeats.EasterEffect](../Interface/ISmTramoSeats/ISmTramoSeats.EasterEffect.htm).

		 поправки.


		 ![](../Element%20Enums.gif)

		 Always. Всегда применять
		 поправки.
