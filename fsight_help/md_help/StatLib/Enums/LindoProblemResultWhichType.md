# LindoProblemResultWhichType


## Описание


Перечисление LindoProblemResultWhichType
 используется для определения типа задачи, решение которой должно быть
 извлечено модулем LINDO.


Используется следующим свойством:


	- [ILindoSettings.ProblemResultWhichType](../Interface/ILindoSettings/ILindoSettings.ProblemResultWhichType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 BasicPrimal. Задача
		 глобальной оптимизации. Применяется только в паре с элементом
		 [LindoProblemResultType.MIPPrimalSolution](LindoProblemResultType.htm).
