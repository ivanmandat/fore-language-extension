# BinaryOptimizationMethod


## Описание


Перечисление BinaryOptimizationMethod
 используется для определения метода оптимизации в бинарной регрессии.


Используется следующим свойством:


	- [ISmBinaryModel.OptimizationMethod](../Interface/ISmBinaryModel/ISmBinaryModel.OptimizationMethod.htm).
