# LindoProblemType


## Описание


Перечисление LindoProblemType
 используется для определения типа решаемой задачи в модуле LINDO.


Используется следующим свойством:


	- [ILindoSettings.ProblemType](../Interface/ILindoSettings/ILindoSettings.ProblemType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 GOP. Глобальная оптимизация.


		 1
		 MIP. Целочисленная
		 оптимизация.


		 2
		 Optimize. Обычная оптимизация.
