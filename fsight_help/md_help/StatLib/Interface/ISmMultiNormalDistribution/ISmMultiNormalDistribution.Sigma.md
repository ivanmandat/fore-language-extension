# ISmMultiNormalDistribution.Sigma


## Синтаксис


Sigma: Array;


## Описание


Свойство Sigma определяет ковариационную
 матрицу распределения.


## Комментарии


Матрица должна быть симметричная положительно определенная и иметь размерность
 K×K.
 Где K - это размерность распределения,
 определяемая значением свойства [ISmMultiNormalDistribution.K](ISmMultiNormalDistribution.K.htm).
