# ISmMultiNormalDistribution.Mu


## Синтаксис


Mu: Array;


## Описание


Свойство Mu определяет вектор
 математических ожиданий многомерного распределения.


## Комментарии


Размерность вектора: 1×K.
 Где K - это размерность распределения,
 определяемая значением свойства [ISmMultiNormalDistribution.K](ISmMultiNormalDistribution.K.htm).
