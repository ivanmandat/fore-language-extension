# ISmMultiNormalDistribution.MultiRandomVector


## Синтаксис


MultiRandomVector(Count: Integer): Array;


## Параметры


Count. Размер первого измерения
 матрицы.


## Описание


Метод MultiRandomVector возвращает матрицу,
 соответствующую распределению.


## Комментарии


Размерность вектора: Count×K. Где K
 - это размерность распределения, определяемая значением свойства [ISmMultiNormalDistribution.K](ISmMultiNormalDistribution.K.htm).
