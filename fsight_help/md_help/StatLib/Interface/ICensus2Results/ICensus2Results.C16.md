# ICensus2Results.C16


## Синтаксис


C16: Array;


## Описание


Свойство C16 возвращает поправки на число рабочих дней, полученные из коэффициентов регрессии - окончательный вариант. Свойство совпадает с [B16](ICensus2Results.B16.htm) с той разницей, что поправки вычитаются из ряда [C13](ICensus2Results.C13.htm) (аддитивный случай) или на них делятся (мультипликативный случай) значения ряда [C13](ICensus2Results.C13.htm).
