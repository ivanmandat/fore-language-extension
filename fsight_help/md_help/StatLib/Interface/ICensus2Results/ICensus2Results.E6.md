# ICensus2Results.E6


## Синтаксис


E6: Array;


## Описание


Свойство E6 возвращает разности (относительные изменения) окончательного варианта ряда с сезонной поправкой. Эти значения представляют собой разности от месяца к месяцу (квартала к кварталу) (аддитивная модель) или относительные изменения (мультипликативная модель) ряда [D11](ICensus2Results.D11.htm).
