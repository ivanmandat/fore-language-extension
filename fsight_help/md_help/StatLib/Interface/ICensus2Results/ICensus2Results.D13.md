# ICensus2Results.D13


## Синтаксис


D13: Array;


## Описание


Свойство D13 возвращает нерегулярную составляющую - окончательный вариант. Значения этого ряда получаются вычитанием значений [D12](ICensus2Results.D12.htm) из [D11](ICensus2Results.D11.htm) (аддитивная модель) или делением [D11](ICensus2Results.D11.htm) на [D12](ICensus2Results.D12.htm) (мультипликативная модель).
