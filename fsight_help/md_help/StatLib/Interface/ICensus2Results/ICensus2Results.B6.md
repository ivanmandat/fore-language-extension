# ICensus2Results.B6


## Синтаксис


B6: Array;


## Описание


Свойство B6 возвращает сезонную корректировку ряда. Предварительный вариант сезонно-скорректированного ряда получается вычитанием из [B1](ICensus2Results.B1.htm) сезонной составляющей [B5](ICensus2Results.B5.htm) (аддитивная модель) или делением [B1](ICensus2Results.B1.htm) на эту составляющую (мультипликативная модель).
