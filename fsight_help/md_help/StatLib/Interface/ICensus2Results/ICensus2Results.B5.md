# ICensus2Results.B5


## Синтаксис


B5: Array;


## Описание


Свойство B5 возвращает сезонную составляющую. Значения-выбросы в ряде [B3](ICensus2Results.B3.htm) заменяются соответствующими значениями из [B4](ICensus2Results.B4.htm). Из этого ряда получается предварительная сезонная составляющая взятием 5-точечного скользящего среднего отдельно для значений, соответствующих каждому месяцу; затем берется 12-точечное скользящее среднее для всего ряда, и полученные значения для каждого года корректируются на нулевую сумму (аддитивная модель) или на сумму 12.0 (мультипликативная модель).
