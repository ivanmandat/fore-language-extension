# ICensus2Results.C18


## Синтаксис


C18: Array;


## Описание


Свойство C18 возвращает поправки на число рабочих дней, полученные из комбинированных весов дней недели - окончательный вариант. Свойство аналогично таблице [B18](ICensus2Results.B18.htm), с той разницей, что здесь используются окончательные веса из таблицы [C15](ICensus2Results.C15.htm).
