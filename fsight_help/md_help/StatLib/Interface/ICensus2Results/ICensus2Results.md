# ICensus2Results


Сборка: Stat;


## Описание


Интерфейс ICensus2Results возвращает
 выходные ряды для метода «X11».


Окончательные результаты, находятся в таблицах:


[D10](ICensus2Results.D10.htm)
 – Сезонная составляющая;


[D11](ICensus2Results.D11.htm)
 – Сезонная корректировка;


[D12](ICensus2Results.D12.htm)
 – Тренд-циклическая компонента;


[D13](ICensus2Results.D13.htm)
 – Случайная составляющая временного ряда.


## Иерархия наследования


           ICensus2Results


## Комментарии


Метод X11 поддерживается только в ОС Windows.
