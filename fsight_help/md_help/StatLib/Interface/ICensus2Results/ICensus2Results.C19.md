# ICensus2Results.C19


## Синтаксис


C19: Array;


## Описание


Свойство C19 возвращает исходный ряд с поправками на рабочие дни и априорную вариацию. Значения из [C18](ICensus2Results.C18.htm) используются для корректировки исходного (или априорно-скорректированного) ряда ([A3](ICensus2Results.A3.htm) или [B1](ICensus2Results.B1.htm)). Конкретно, из значений исходного ряда вычитаются значения ряда [C18](ICensus2Results.C18.htm) (аддитивная модель) или он делится на значения ряда [C18](ICensus2Results.C18.htm) (мультипликативная модель).
