# ICensus2Results.D1


## Синтаксис


D1: Array;


## Описание


Свойство D1 возвращает исходный ряд, модифицированный с помощью окончательных весов, с поправкой на рабочие дни и априорную вариацию. Свойство аналогично [C1](ICensus2Results.C1.htm) с той разницей, что при вычислении ее значений используются веса [C17](ICensus2Results.C17.htm) и скорректированный ряд [C19](ICensus2Results.C19.htm).
