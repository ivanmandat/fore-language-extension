# ICensus2Results.C4


## Синтаксис


C4: Array;


## Описание


Свойство C4 возвращает модифицированные S-I разности (отношения). Уточненные S-I разности (отношения) получаются вычитанием значений ряда [C2](ICensus2Results.C2.htm) из (аддитивная модель) или делением на них (мультипликативная модель) модифицированного ряда [C1](ICensus2Results.C1.htm).
