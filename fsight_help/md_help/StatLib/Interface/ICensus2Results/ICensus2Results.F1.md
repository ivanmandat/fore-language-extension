# ICensus2Results.F1


## Синтаксис


F1: Array;


## Описание


Свойство F1 возвращает РМЦД
 (КЦД) скользящее среднее.


## Комментарии


Значения этого ряда получаются применением процедуры невзвешенного скользящего
 среднего к окончательному варианту ряда с сезонной поправкой ([D11](ICensus2Results.D11.htm)).
 Ширина окна сглаживания определяется месяцем (кварталом) циклического
 доминирования, сокращенно МЦД (КЦД). МЦД (или КЦД) вычисляется как средний
 отрезок времени, на котором изменения случайной составляющей становятся
 равными изменениям тренд-циклической компоненты.
