# ICensus2Results.C11


## Синтаксис


C11: Array;


## Описание


Свойство C11 возвращает сезонную корректировку ряда. Уточненный ряд с сезонной поправкой получается вычитанием ряда [C10](ICensus2Results.C10.htm) из ряда [B1](ICensus2Results.B1.htm) (аддитивная модель) или делением на [C10](ICensus2Results.C10.htm) (мультипликативная модель) ряда [B1](ICensus2Results.B1.htm).
