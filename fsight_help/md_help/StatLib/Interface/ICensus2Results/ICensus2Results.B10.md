# ICensus2Results.B10


## Синтаксис


B10: Array;


## Описание


Свойство B10 возвращает сезонную составляющую. После замены значений-выбросов на соответствующие значения из [B9](ICensus2Results.B9.htm), 7-точечное взвешенное скользящее среднее применяется к S-I разностям (отношениям) в [B8](ICensus2Results.B8.htm). Полученная оценка сезонной составляющей затем корректируется таким образом, чтобы сумма ее значений за каждый год равнялась нулю (аддитивная модель) или 12.0 (мультипликативная модель).
