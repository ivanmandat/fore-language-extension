# ICensus2Results.D12


## Синтаксис


D12: Array;


## Описание


Свойство D12 возвращает тренд-циклическую
 компоненту - окончательный вариант.


## Комментарии


 Значения этого ряда получаются вычитанием
 значений [D10](ICensus2Results.D10.htm) из [D1](ICensus2Results.D1.htm)
 (аддитивная модель) или делением [D1](ICensus2Results.D1.htm)
 на [D10](ICensus2Results.D10.htm) (мультипликативная модель).
