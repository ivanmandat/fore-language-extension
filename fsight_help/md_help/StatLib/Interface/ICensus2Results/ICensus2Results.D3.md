# ICensus2Results.D4


## Синтаксис


D4: Array;


## Описание


Свойство D4 возвращает модифицированные S-I разности (отношения). Модифицированные S-I разности (отношения) вычисляются вычитанием ряда [D2](ICensus2Results.D2.htm) из (аддитивная модель) [D1](ICensus2Results.D1.htm) или делением на [D2](ICensus2Results.D2.htm) (мультипликативная модель) значений ряда [D1](ICensus2Results.D1.htm).
