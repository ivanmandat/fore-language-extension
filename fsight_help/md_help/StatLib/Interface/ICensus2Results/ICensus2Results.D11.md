# ICensus2Results.D11


## Синтаксис


D11: Array;


## Описание


Свойство D11 возвращает сезонную корректировку ряда - окончательный вариант. Окончательный вариант ряда с сезонной поправкой получается вычитанием [D10](ICensus2Results.D10.htm) из [C19](ICensus2Results.C19.htm) (аддитивная модель) или делением [C19](ICensus2Results.C19.htm) на [D10](ICensus2Results.D10.htm) (мультипликативная модель).
