# ICensus2Results.C9


## Синтаксис


C9: Array;


## Описание


Свойство C9 возвращает модифицированные S-I разности (отношения). Модифицированные S-I разности (отношения) вычисляются путем вычитания ряда [C7](ICensus2Results.C7.htm) из ряда [C1](ICensus2Results.C1.htm), (аддитивная модель) или делением на [C7](ICensus2Results.C7.htm) (мультипликативная модель) ряда [C1](ICensus2Results.C1.htm).
