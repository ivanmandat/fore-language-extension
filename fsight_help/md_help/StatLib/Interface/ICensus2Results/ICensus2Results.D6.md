# ICensus2Results.D6


## Синтаксис


D6: Array;


## Описание


Свойство D6 возвращает сезонную корректировку ряда. Значения в этой таблице получаются вычитанием [D5](ICensus2Results.D5.htm) из [D1](ICensus2Results.D1.htm) (аддитивная модель) или делением [D1](ICensus2Results.D1.htm) на [D5](ICensus2Results.D5.htm) (мультипликативная модель).
