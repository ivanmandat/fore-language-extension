# ICensus2Results.B9


## Синтаксис


B9: Array;


## Описание


Свойство B9 возвращает значения для замены выбросов S-I разностей/отношений. Свойство B9 совпадает с [B4](ICensus2Results.B4.htm) с той разницей, что 7-точечное скользящее среднее применяется (для оценки сезонной компоненты) к разностям /отношениям, полученным в [B8](ICensus2Results.B8.htm).


Для аддитивной модели - разности, для мультипликативной - отношения.
