# ICensus2PeriodBegin


Сборка: Stat;


## Описание


Интерфейс ICensus2PeriodBegin
 определяет параметры начала периода для метода «X11».


## Иерархия наследования


           ICensus2PeriodBegin


## Комментарии


Метод X11 поддерживается только в ОС Windows.
