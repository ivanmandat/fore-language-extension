# IBinaryTreeGradientBoosted.Count2


## Синтаксис


Count2: Integer;


## Описание


Свойство Count2 возвращает
 количество столбцов в массиве деревьев.


## Комментарии


Количество столбцов соответствует количеству итераций при расчете градиентного
 бустинга. Для получения массива категорий используйте свойство [ISmGradientBoostedTree.CategoriesList](../ISmGradientBoostedTree/ISmGradientBoostedTree.CategoriesList.htm).
