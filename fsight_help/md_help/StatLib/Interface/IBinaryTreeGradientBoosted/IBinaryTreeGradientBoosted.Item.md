# IBinaryTreeGradientBoosted.Item


## Синтаксис


Item(Index1: Integer; Index2: Integer): [IBinaryTreeNode](../IBinaryTreeNode/IBinaryTreeNode.htm);


## Параметры


Index1. Индекс строки;


Index2. Индекс столбца.


## Описание


Свойство Item возвращает
 указанное дерево из массива деревьев.


## Комментарии


Количество строк и столбцов в массиве возвращают свойства [IBinaryTreeGradientBoosted.Count1](IBinaryTreeGradientBoosted.Count1.htm)
 и [IBinaryTreeGradientBoosted.Count2](IBinaryTreeGradientBoosted.Count2.htm).
