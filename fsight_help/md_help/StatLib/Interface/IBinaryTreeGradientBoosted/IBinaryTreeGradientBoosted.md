# IBinaryTreeGradientBoosted


Сборка: Stat;


## Описание


Интерфейс IBinaryTreeGradientBoosted
 предназначен для работы с двумерным массивом деревьев, рассчитанных градиентным
 бустингом.


## Иерархия наследования


           IBinaryTreeGradientBoosted


## Комментарии


Для получения массива деревьев используйте свойство [ISmGradientBoostedTree.Trees](../ISmGradientBoostedTree/ISmGradientBoostedTree.Trees.htm).
