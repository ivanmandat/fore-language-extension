# ISmPooledModel.IdiosyncraticSD


## Синтаксис


IdiosyncraticSD: Double;


## Описание


Свойство IdiosyncraticSD возвращает
 идиосинкразическое стандартное отклонение.


## Комментарии


Свойство актуально, если рассчитывается модель со случайными эффектами,
 т.е. [ISmPooledModel.CrossSection](ISmPooledModel.CrossSection.htm) = [PooledModelCrossSectionType](../../Enums/PooledModelCrossSectionType.htm).RandomEffect.


Для получения стандартного отклонения групповых ошибок свойство [ISmPooledModel.CrossSectionSD](ISmPooledModel.CrossSectionSD.htm).
