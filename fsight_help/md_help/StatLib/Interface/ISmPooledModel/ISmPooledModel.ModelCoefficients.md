# ISmPooledModel.ModelCoefficients


## Синтаксис


ModelCoefficients: [IModelCoefficients](../IModelCoefficients/IModelCoefficients.htm);


## Описание


Свойство ModelCoefficients возвращает
 параметры коэффициентов модели.


## Комментарии


Необходимо помнить, что порядок коэффициентов определяется порядком
 добавления в метод [факторов](ISmPooledModel.Explanatories.htm),
 которым соответствуют данные коэффициенты.
