# ISmPooledModel.Weights


## Синтаксис


Weights: Array;


## Описание


Свойство Weights возвращает
 веса, используемые при расчете модели.


## Комментарии


По умолчанию веса не заданы.


Веса задаются массивом вещественных чисел. Размерность данного массива
 должна совпадать с размерностью объясняемой переменной. Для работы с объясняемой
 переменной используйте [ISmPooledModel.Explained](ISmPooledModel.Explained.htm).
