## Синтаксис

## Описание


Свойство MissingData определяет
 параметры обработки пропусков.


## Комментарии


Обработка пропусков методом Casewise
 для регрессии на панельных данных учитывает пропуски в объясняемом и объясняющих
 рядах отдельно по каждому элементу.
