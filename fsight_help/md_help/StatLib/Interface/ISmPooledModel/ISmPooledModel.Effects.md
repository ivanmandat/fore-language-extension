# ISmPooledModel.Effects


## Синтаксис


Effects: Array;


## Описание


Свойство Effects возвращает
 рассчитанные эффекты.


## Комментарии


Эффекты будут рассчитаны, только если используется модель панельных
 данных с фиксированными эффектами: [ISmPooledModel.CrossSection](ISmPooledModel.CrossSection.htm) = [PooledModelCrossSectionType](../../Enums/PooledModelCrossSectionType.htm).FixedEffect.


Эффекты интерпретируются как мешающий параметр, и оценивание направлено
 на то, чтобы их исключить.
