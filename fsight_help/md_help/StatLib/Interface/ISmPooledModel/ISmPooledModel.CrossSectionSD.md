# ISmPooledModel.CrossSectionSD


## Синтаксис


CrossSectionSD: Double;


## Описание


Свойство CrossSectionSD возвращает
 стандартное отклонение групповых ошибок.


## Комментарии


Свойство актуально, если рассчитывается модель со случайными эффектами,
 т.е. [ISmPooledModel.CrossSection](ISmPooledModel.CrossSection.htm) = [PooledModelCrossSectionType](../../Enums/PooledModelCrossSectionType.htm).RandomEffect.


Для получения идиосинкразического стандартного отклонения используйте
 свойство [ISmPooledModel.IdiosyncraticSD](ISmPooledModel.IdiosyncraticSD.htm).
