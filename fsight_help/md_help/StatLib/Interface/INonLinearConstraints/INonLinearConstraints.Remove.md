# INonLinearConstraints.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс ограничения в коллекции.


## Описание


Метод Remove удаляет ограничение по заданному индексу.


## Комментарии


В случае успешного удаления метод возвращает True, в противном случае False.
