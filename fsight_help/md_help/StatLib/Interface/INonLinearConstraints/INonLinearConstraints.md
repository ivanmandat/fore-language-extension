# INonLinearConstraints


## Описание


Интерфейс INonLinearConstraints
 содержит свойства и методы для работы с коллекцией нелинейных ограничений.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property.gif)
		 [Count](INonLinearConstraints.Count.htm)


		 Свойство Count возвращает
		 количество нелинейных ограничений в коллекции.


		 ![](../../Property.gif)
		 [Item](INonLinearConstraints.Item.htm)


		 Свойство Item определяет
		 параметры нелинейного ограничения.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub.gif)
		 [Add](INonLinearConstraints.Add.htm)


		 Метод Add добавляет
		 нелинейное ограничение в коллекцию.


		 ![](../../Sub.gif)
		 [Clear](INonLinearConstraints.Clear.htm)


		 Метод Clear очищает
		 коллекцию.


		 ![](../../Sub.gif)
		 [Remove](INonLinearConstraints.Remove.htm)


		 Метод Remove удаляет
		 ограничение по заданному индексу.
