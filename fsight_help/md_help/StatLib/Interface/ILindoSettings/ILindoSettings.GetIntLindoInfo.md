# ILindoSettings.GetIntLindoInfo


## Синтаксис


GetIntLindoInfo(Type: [LindoIntInfoType](../../Enums/LindoIntInfoType.htm);
 Var ErrorCode: Integer): Integer;


## Параметры


Type. Тип целочисленной информации;


ErrorCode. Код ошибки.


## Описание


Метод GetIntLindoInfo возвращает
 указанную целочисленную информацию о методе расчета.


## Комментарии


Для получения вещественной и строковой информации используйте методы
 [ILindoSettings.GetDouLindoInfo](ILindoSettings.GetDouLindoInfo.htm)
 и [ILindoSettings.GetStrLindoInfo](ILindoSettings.GetStrLindoInfo.htm).


Полные сведения о возвращенной целочисленной информации можно получить
 в [документации
 LINDO](http://www.lindo.com/index.php?option=com_content&view=article&id=147&Itemid=69).
