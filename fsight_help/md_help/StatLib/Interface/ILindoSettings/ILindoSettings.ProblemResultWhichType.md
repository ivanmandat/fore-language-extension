## Синтаксис

## Описание


Свойство ProblemResultWhichType
 определяет тип задачи, решение которой должно быть извлечено.


## Комментарии


Для выбора типа решения, который должен быть получен, используйте свойство
 [ILindoSettings.ProblemResultType](ILindoSettings.ProblemResultType.htm).
