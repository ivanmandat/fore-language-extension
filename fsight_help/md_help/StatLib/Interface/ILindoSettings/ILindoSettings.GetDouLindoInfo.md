# ILindoSettings.GetDouLindoInfo


## Синтаксис


GetDouLindoInfo(Type: [LindoDouInfoType](../../Enums/LindoDouInfoType.htm);
 Var ErrorCode: Integer): Double;


## Параметры


Type. Тип вещественной информации;


ErrorCode. Код ошибки.


## Описание


Метод GetDouLindoInfo возвращает
 указанную вещественную информацию о методе расчета.


## Комментарии


Для получения целочисленной и строковой информации используйте методы
 [ILindoSettings.GetIntLindoInfo](ILindoSettings.GetIntLindoInfo.htm)
 и [ILindoSettings.GetStrLindoInfo](ILindoSettings.GetStrLindoInfo.htm).


Полные сведения о возвращенной вещественной информации можно получить
 в [документации
 LINDO](http://www.lindo.com/index.php?option=com_content&view=article&id=147&Itemid=69).
