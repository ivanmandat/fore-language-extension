# ILindoSettings.ProblemType


## Синтаксис


ProblemType: [LindoProblemType](../../Enums/LindoProblemType.htm);


## Описание


Свойство ProblemType определяет
 тип решаемой задачи.


## Комментарии


По умолчанию решается задача глобальной оптимизации, то есть свойство
 имеет значение [LindoProblemType.GOP](../../Enums/LindoProblemType.htm).
