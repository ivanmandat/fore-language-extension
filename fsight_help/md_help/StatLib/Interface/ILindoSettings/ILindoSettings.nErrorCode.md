# ILindoSettings.nErrorCode


## Синтаксис


nErrorCode: Integer;


## Описание


Свойство nErrorCode возвращает
 код ошибки LINDO, возникшей при расчете.


## Комментарии


Для получения текста ошибки, соответствующей коду, используйте свойство
 [ILindoSettings.MessageString](ILindoSettings.MessageString.htm).


Для получения предупреждений LINDO, возникших при расчете, используйте
 свойство [ILindoSettings.Warnings](ILindoSettings.Warnings.htm).
