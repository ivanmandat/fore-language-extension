# ILindoSettings.DisposeLindoModelAfterExecute


## Синтаксис


DisposeLindoModelAfterExecute: Boolean;


## Описание


Свойство DisposeLindoModelAfterExecute
 определяет, удалять ли модель LINDO сразу после выполнения.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Модель будет удалена сразу после выполнения;


	- False. Модель не будет
	 удалена сразу после выполнения.
