# ILindoSettings.GetStrLindoInfo


## Синтаксис


GetStrLindoInfo(Type: [LindoStrInfoType](../../Enums/LindoStrInfoType.htm);
 Var ErrorCode: Integer): String;


## Параметры


Type. Тип строковой информации;


ErrorCode. Код ошибки.


## Описание


Метод GetStrLindoInfo возвращает
 указанную строковую информацию о методе расчета.


## Комментарии


Для получения вещественной и целочисленной информации используйте методы
 [ILindoSettings.GetDouLindoInfo](ILindoSettings.GetDouLindoInfo.htm)
 и [ILindoSettings.GetIntLindoInfo](ILindoSettings.GetIntLindoInfo.htm).


Полные сведения о возвращенной строковой информации можно получить в
 [документации
 LINDO](http://www.lindo.com/index.php?option=com_content&view=article&id=147&Itemid=69).
