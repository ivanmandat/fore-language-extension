Свойство ModelParams возвращает
 параметры модели LINDO.

Для получения параметров среды LINDO используйте свойство [ILindoSettings.EnvParams](ILindoSettings.EnvParams.htm).


Описание параметров среды и модели приведено в [документации
 LINDO](http://www.lindo.com/index.php?option=com_content&view=article&id=147&Itemid=69).
