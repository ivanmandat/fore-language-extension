# ILindoSettings.MessageString


## Синтаксис


MessageString: String;


## Описание


Свойство MessageString возвращает
 текст ошибки LINDO, возникшей при расчете.


## Комментарии


Для получения кода LINDO, соответствующего ошибке, используйте свойство
 [ILindoSettings.nErrorCode](ILindoSettings.nErrorCode.htm).
