## Синтаксис

## Описание


Свойство SolverType определяет
 метод решения.


## Комментарии


По умолчанию метод решения выбирается автоматически, то есть свойство
 имеет значение [LindoSolverType.Free](../../Enums/LindoSolverType.htm).
