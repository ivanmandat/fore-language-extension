# ILindoSettings.Warnings


## Синтаксис


Warnings: Array;


## Описание


Свойство Warnings возвращает
 предупреждения LINDO, возникшие при расчете.


## Комментарии


Для получения кода ошибки LINDO, возникшей при расчете, используйте
 свойство [ILindoSettings.nErrorCode](ILindoSettings.nErrorCode.htm).
