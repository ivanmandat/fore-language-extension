## Описание


Свойство ProblemResultType определяет
 тип решения, который должен быть получен.


## Комментарии


Для выбора типа задачи, решение которой должно быть извлечено, используйте
 свойство [ILindoSettings.ProblemResultWhichType](ILindoSettings.ProblemResultWhichType.htm).
