# ILindoSettings.LicensePath


## Синтаксис


LicensePath: String;


## Описание


Свойство LicensePath определяет
 путь к лицензии LINDO.


## Комментарии


Путь должен включать название файла лицензии.


Лицензия LINDO по умолчанию расположена в папке S\license\,
 где S - это путь до папки, в
 которой установлен LINDO. Наименование файла лицензии - lndapiX.dll,
 где X - это номер версии LINDO. Например, для LINDO версии 8.0 лицензия
 будет иметь название lndapi80.dll,
 а для версии 9.0 - lndapi90.dll.
