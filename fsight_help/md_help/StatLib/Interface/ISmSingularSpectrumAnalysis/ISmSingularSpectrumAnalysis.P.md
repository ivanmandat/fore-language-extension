# ISmSingularSpectrumAnalysis.P


## Синтаксис


P: Array;


## Описание


Свойство P возвращает собственные
 векторы главных компонент.


## Комментарии


Свойство используется для получения собственных векторов только для
 основных главных компонент. Значения собственных векторов для дополнительных
 главных компонент недоступны.


Для получения количества собственных главных компонент используйте свойство
 [ISmSingularSpectrumAnalysis.PCCount](ISmSingularSpectrumAnalysis.PCCount.htm).
