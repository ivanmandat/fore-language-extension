# ISmSingularSpectrumAnalysis.Serie


## Синтаксис


Serie: Array;


## Описание


Свойство Serie возвращает
 исходный ряд.


## Комментарии


Индексация входных массивов должна начинаться с нуля.


Для получения восстановленного ряда и ряда остатков используйте свойства
 [ISmSingularSpectrumAnalysis.Fitted](ISmSingularSpectrumAnalysis.Fitted.htm)
 и [ISmSingularSpectrumAnalysis.Residuals](ISmSingularSpectrumAnalysis.Residuals.htm).
