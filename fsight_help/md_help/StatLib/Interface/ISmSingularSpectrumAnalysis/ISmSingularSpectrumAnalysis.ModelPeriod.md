# ISmSingularSpectrumAnalysis.ModelPeriod


## Синтаксис


ModelPeriod: [IStatPeriod](../IStatPeriod/IStatPeriod.htm);


## Описание


Свойство ModelPeriod возвращает
 параметры периода идентификации.


## Комментарии


По умолчанию длина периода идентификации совпадает с длиной входного
 ряда. Для получения входного ряда используйте свойство [ISmSingularSpectrumAnalysis.Serie](ISmSingularSpectrumAnalysis.Serie.htm).
