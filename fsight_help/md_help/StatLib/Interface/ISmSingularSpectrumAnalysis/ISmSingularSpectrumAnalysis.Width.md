# ISmSingularSpectrumAnalysis.Width


## Синтаксис


Width: Integer;


## Описание


Свойство Width определяет
 ширину окна.


## Комментарии


Ширина окна должна быть больше, чем количество главных компонент.


Для определения количества главных компонент используйте свойство [ISmSingularSpectrumAnalysis.PCCount](ISmSingularSpectrumAnalysis.PCCount.htm).
