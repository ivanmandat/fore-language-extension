# ISmSingularSpectrumAnalysis.PCCount


## Синтаксис


PCCount: Integer;


## Описание


Свойство PCCount определяет
 количество главных компонент.


## Комментарии


Количество используемых главных компонент должно быть положительным
 числом, не превышающим ширину окна.


Для определения ширины окна используйте свойство [ISmSingularSpectrumAnalysis.Width](ISmSingularSpectrumAnalysis.Width.htm).
