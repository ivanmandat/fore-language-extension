# ISmSingularSpectrumAnalysis.IncludedComponents


## Синтаксис


IncludedComponents: Array;


## Описание


Свойство IncludedComponents
 определяет, какие из главных компонент используются для восстановления
 входного ряда.


## Комментарии


Для восстановления входного ряда будут использоваться только те главные
 компоненты, индексы которых указаны в массиве IncludedComponents.
 Индексация главных компонент сквозная и начинается с нуля.


Если массив IncludedComponents
 пустой, то для восстановления входного ряда используются все главные
 компоненты.
