# ISmHierarchicalClusterAnalysis.ClusterCount


## Синтаксис


ClusterCount: Integer;


## Описание


Свойство ClusterCount определяет число кластеров, на которое нужно разбить множество объектов.


## Комментарии


Количество кластеров должно удовлетворять ограничению 2 ≤ k ≤ N - 1.


Где:


-
k. Количество кластеров;


-
N. Количество объектов.
