## Синтаксис

## Описание


Свойство Dord возвращает расстояния
 между кластерами в момент объединения.


## Комментарии


Для получения объектов в порядке построения дендрограммы используйте
 свойство [ISmHierarchicalClusterAnalysis.Iord](ISmHierarchicalClusterAnalysis.Iord.htm).
