# ISmHierarchicalClusterAnalysis.ClusterLink


## Синтаксис


ClusterLink: [ClusterLinkType](../../Enums/ClusterLinkType.htm);


## Описание


Свойство ClusterLink определяет способ расчета расстояния между кластерами.


## Комментарии


По умолчанию используется способ «Расстояние между ближайшими соседями».
