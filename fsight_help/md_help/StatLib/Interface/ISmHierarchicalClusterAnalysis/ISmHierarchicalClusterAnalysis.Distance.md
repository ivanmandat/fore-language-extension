# ISmHierarchicalClusterAnalysis.Distance


## Синтаксис


Distance: [ISlDistance](../ISlDistance/ISlDistance.htm);


## Описание


Свойство Distance определяет параметры расчета расстояний между объектами.


## Комментарии


Для определения способа расчета расстояния между кластерами используйте свойство [ISmHierarchicalClusterAnalysis.ClusterLink](ISmHierarchicalClusterAnalysis.ClusterLink.htm).
