# ISmHierarchicalClusterAnalysis.CreateDendrogram


## Синтаксис


CreateDendogram: Boolean;


## Описание


Свойство CreateDendogram определяет
 создание дендрограммы, расчёт массивов объектов и расстояний между кластерами.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Дендрограмма создается, массивы объектов и расстояний между кластерами
	 рассчитываются. Для получения массивов используйте свойства [ISmHierarchicalClusterAnalysis.Iord](ISmHierarchicalClusterAnalysis.Iord.htm)
	 и [ISmHierarchicalClusterAnalysis.Dord](ISmHierarchicalClusterAnalysis.Dord.htm);


	- False. Дендрограмма
	 не создается, массивы объектов и расстояний не рассчитываются.
