# ISlRecommendationRule.SupportCount


## Синтаксис


SupportCount: Integer;


## Описание


Свойство SupportCount возвращает
 поддержку правила, выраженную в количестве транзакций, для которых
 оно выполнилось.


## Комментарии


Поддержку, выраженную в процентах транзакций, для которых выполнилось
 правило, возвращает свойство [ISlRecommendationRule.SupportPercent](ISlRecommendationRule.SupportPercent.htm).
