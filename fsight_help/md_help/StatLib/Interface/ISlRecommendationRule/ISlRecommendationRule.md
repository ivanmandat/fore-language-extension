# ISlRecommendationRule


Сборка: Stat;


## Описание


Интерфейс ISlRecommendationRule используется
 для работы с правилом, полученным в результате интеллектуального анализа
 данных методом «Анализ ассоциаций».


## Иерархия наследования


           ISlRecommendationRule


## Комментарии


Для выполнения интеллектуального анализа данных методом «Анализ ассоциаций» используйте
 интерфейс [ISmAssociationRules](../ISmAssociationRules/ISmAssociationRules.htm).

		 ![](../../Property.gif)
		 [ExplainedItemset](ISlRecommendationRule.ExplainedItemset.htm)


		 Свойство ExplainedItemset

		 [ExplanatoryItemset](ISlRecommendationRule.ExplanatoryItemset.htm)


		 Свойство ExplanatoryItemset
		 возвращает условие правила.


		 ![](../../Property.gif)
		 [Importance](ISlRecommendationRule.Importance.htm)


		 Свойство Importance
		 возвращает уровень значимости правила.


		 ![](../../Property.gif)
		 [SupportCount](ISlRecommendationRule.SupportCount.htm)


		 Свойство SupportCount
		 возвращает поддержку правила, выраженную в количестве
		 транзакций, для которых оно выполнилось.


		 ![](../../Property.gif)
		 [SupportPercent](ISlRecommendationRule.SupportPercent.htm)


		 Свойство SupportPercent
		 возвращает поддержку правила, выраженную в проценте транзакций,
		 для которых оно выполнилось.
