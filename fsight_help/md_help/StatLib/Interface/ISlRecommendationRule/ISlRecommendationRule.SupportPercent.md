# ISlRecommendationRule.SupportPercent


## Синтаксис


SupportPercent: Double;


## Описание


Свойство SupportPercent возвращает
 поддержку правила, выраженную в проценте транзакций, для которых
 оно выполнилось.


## Комментарии


Поддержку, выраженную в количестве транзакций, для которых выполнилось
 правило, возвращает свойство [ISlRecommendationRule.SupportCount](ISlRecommendationRule.SupportCount.htm).
