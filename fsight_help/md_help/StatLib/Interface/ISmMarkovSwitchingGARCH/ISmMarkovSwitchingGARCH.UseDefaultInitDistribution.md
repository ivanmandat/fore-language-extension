# ISmMarkovSwitchingGARCH.UseDefaultInitDistribution


## Синтаксис


UseDefaultInitDistribution: Boolean;


## Описание


Свойство UseDefaultInitDistribution
 определяет, будут ли использоваться значения по умолчанию.


## Комментарии


Допустимые значения:


	- True. Будут использоваться
	 значения по умолчанию;


	- False. Не будут использоваться
	 значения по умолчанию.
