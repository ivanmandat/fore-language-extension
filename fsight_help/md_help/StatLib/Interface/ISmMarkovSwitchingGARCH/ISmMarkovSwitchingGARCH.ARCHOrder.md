# ISmMarkovSwitchingGARCH.ARCHOrder


## Синтаксис


ARCHOrder: Integer;


## Описание


Свойство ARCHOrder определяет
 порядок авторегрессии условной гетероскедастичности (ARCH).


## Комментарии


Значение по умолчанию - «1».


Для корректного расчета метода хотя бы один из порядков ARCHOrder
 и [GARCHOrder](ISmMarkovSwitchingGARCH.GARCHOrder.htm)
 должен быть ненулевым.
