# ISmMarkovSwitchingGARCH.ConditionalStateProbabilities


## Синтаксис


ConditionalStateProbabilities: Array;


## Описание


Свойство ConditionalStateProbabilities
 возвращает ряд вероятностей перехода для каждого момента времени.


## Комментарии


Для получения оценки параметров перехода используйте свойства [ISmMarkovSwitchingGARCH.P00](ISmMarkovSwitchingGARCH.P00.htm) и [ISmMarkovSwitchingGARCH.P11](ISmMarkovSwitchingGARCH.P11.htm).
