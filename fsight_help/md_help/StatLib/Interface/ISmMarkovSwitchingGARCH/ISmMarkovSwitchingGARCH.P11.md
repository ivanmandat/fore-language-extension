## Синтаксис

## Описание


Свойство P11 возвращает
 оценки параметров перехода из состояния 1 в 1.


## Комментарии


Параметры распределения для вероятности перехода из состояния 1 в 1
 можно определить, используя свойство [ISmMarkovSwitchingGARCH.P11Distribution](ISmMarkovSwitchingGARCH.P11Distribution.htm).
