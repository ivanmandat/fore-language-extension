## Описание


Свойство GARCHConst возвращает
 оценку константы GARCH μ0.

Для получения оценки константы μ1 используйте свойство [ISmMarkovSwitchingGARCH.GARCHConst1](ISmMarkovSwitchingGARCH.GARCHConst1.htm).
