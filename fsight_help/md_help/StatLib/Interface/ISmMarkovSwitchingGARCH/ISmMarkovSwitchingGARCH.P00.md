# ISmMarkovSwitchingGARCH.P00


## Синтаксис


P00: [IMSGARCHParameter](../IMSGARCHParameter/IMSGARCHParameter.htm);


## Описание


Свойство P00 возвращает оценки
 параметров перехода из состояния 0 в 0.


## Комментарии


Параметры распределения для вероятности перехода из состояния 0 в 0
 можно определить, используя свойство [ISmMarkovSwitchingGARCH.P00Distribution](ISmMarkovSwitchingGARCH.P00Distribution.htm).
