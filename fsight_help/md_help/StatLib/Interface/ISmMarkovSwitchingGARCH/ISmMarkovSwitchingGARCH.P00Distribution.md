# ISmMarkovSwitchingGARCH.P00Distribution


## Синтаксис


P00Distribution: [ISmBetaDistribution](../ISmBetaDistribution/ISmBetaDistribution.htm);


## Описание


Свойство P00Distribution определяет
 параметры распределения для вероятности перехода из состояния 0 в 0.


## Комментарии


Для получения оценок параметров перехода из состояния 0 в 0 используйте
 свойство [ISmMarkovSwitchingGARCH.P00](ISmMarkovSwitchingGARCH.P00.htm).
