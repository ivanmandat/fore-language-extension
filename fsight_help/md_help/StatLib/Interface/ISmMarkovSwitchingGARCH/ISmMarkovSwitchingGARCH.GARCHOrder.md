# ISmMarkovSwitchingGARCH.GARCHOrder


## Синтаксис


GARCHOrder: Integer;


## Описание


Свойство GARCHOrder определяет
 порядок обобщенной авторегрессии условной гетероскедастичности (GARCH).


## Комментарии


Значение по умолчанию - «1».


Для корректного расчета метода хотя бы один из порядков [ARCHOrder](ISmMarkovSwitchingGARCH.ARCHOrder.htm) и GARCHOrder
 должен быть ненулевым.
