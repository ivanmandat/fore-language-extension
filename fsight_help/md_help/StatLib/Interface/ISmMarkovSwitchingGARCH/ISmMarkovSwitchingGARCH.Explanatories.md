# ISmMarkovSwitchingGARCH.Explanatories


## Синтаксис


Explanatories: [ISlSeries](../ISlSeries/ISlSeries.htm);


## Описание


Свойство Explanatories определяет
 объясняющие ряды.


## Комментарии


В качестве объясняющих следует использовать ряды, коррелирующие с объясняемым
 рядом, но не коррелирующие друг с другом.
