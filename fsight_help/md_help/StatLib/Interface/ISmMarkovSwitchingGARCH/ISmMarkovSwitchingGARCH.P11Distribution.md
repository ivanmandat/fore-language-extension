# ISmMarkovSwitchingGARCH.P11Distribution


## Синтаксис


P11Distribution: [ISmBetaDistribution](../ISmBetaDistribution/ISmBetaDistribution.htm);


## Описание


Свойство P11Distribution определяет
 параметры распределения для вероятности перехода из состояния 1 в 1.


## Комментарии


Для получения оценок параметров перехода из состояния 1 в 1 используйте
 свойство [ISmMarkovSwitchingGARCH.P11](ISmMarkovSwitchingGARCH.P11.htm).
