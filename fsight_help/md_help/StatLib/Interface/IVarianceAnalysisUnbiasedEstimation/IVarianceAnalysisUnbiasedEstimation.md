# IVarianceAnalysisUnbiasedEstimation


## Описание


Интерфейс IVarianceAnalysisUnbiasedEstimation
 возвращает результаты дисперсионного анализа: несмещенную оценку суммы
 квадратов отклонений.


Для однофакторного анализа результаты возвращают свойства: [BetweenGroup](IVarianceAnalysisUnbiasedEstimation.BetweenGroup.htm)
 и [WithinGroup](IVarianceAnalysisUnbiasedEstimation.WithinGroup.htm).


Для двухфакторного анализа результаты возвращают свойства: [Accuracy](IVarianceAnalysisUnbiasedEstimation.Accuracy.htm),
 [Columns](IVarianceAnalysisUnbiasedEstimation.Columns.htm),
 [Rows](IVarianceAnalysisUnbiasedEstimation.Rows.htm).
