# IStatMethodValid


Сборка: Stat;


## Описание


Интерфейс IStatMethodValid
 предназначен для настройки и выполнения кросс-валидации.


## Иерархия наследования


           [IStatMethod](../IStatMethod/IStatMethod.htm)


           IStatMethodValid


## Комментарии


Кросс-валидация (англ. Cross-validation) - метод оценки качества классификации.


Для выполнения кросс-валидации:


	- Задайте параметры кросс-валидации с помощью свойства [IStatMethodValid.CrossValidation](IStatMethodValid.CrossValidation.htm).


	- Вызовите метод [IStatMethodValid.ExecuteValidation](IStatMethodValid.ExecuteValidation.htm).


	- Получите результаты в свойстве [IStatMethodValid.PerformanceScores](IStatMethodValid.PerformanceScores.htm).
