# ISmErrorCorrectionModel.CointegralEquationsTstat


## Синтаксис


CointegralEquationsTstat: [ISlSeries](../ISlSeries/ISlSeries.htm);


## Описание


Свойство CointegralEquationsTstat
 возвращает t-статистики коэффициентов коинтеграционных уравнений.


## Комментарии


Для получения стандартных ошибок коинтеграционных уравнений используйте
 свойство [ISmErrorCorrectionModel.CointegralEquationsSE](ISmErrorCorrectionModel.CointegralEquationsSE.htm).
