# IVarianceAnalysisDegreeOfFreedom


## Описание


Интерфейс IVarianceAnalysisDegreeOfFreedom
 возвращает результаты дисперсионного анализа: число степеней свободы.


Для однофакторного анализа результаты возвращают свойства: [BetweenGroup](IVarianceAnalysisDegreeOfFreedom.BetweenGroup.htm),
 [WithinGroup](IVarianceAnalysisDegreeOfFreedom.WithinGroup.htm),
 [Total](IVarianceAnalysisDegreeOfFreedom.Total.htm).


Для двухфакторного анализа результаты возвращают свойства: [Accuracy](IVarianceAnalysisDegreeOfFreedom.Accuracy.htm),
 [Columns](IVarianceAnalysisDegreeOfFreedom.Columns.htm), [Rows](IVarianceAnalysisDegreeOfFreedom.Rows.htm), [Total](IVarianceAnalysisDegreeOfFreedom.Total.htm).
