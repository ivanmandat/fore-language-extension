# ISmLogNormalDistribution


Сборка: Stat;


## Описание


Интерфейс ISmLogNormalDistribution
 позволяет генерировать выборку псевдослучайных чисел из логнормального
 распределения с заданными математическим ожиданием и дисперсией.


## Иерархия наследования


           [IDistribution](../IDistribution/IDistribution.htm)


           ISmLogNormalDistribution


## Комментарии


Если случайная величина имеет логнормальное распределение, то её логарифм
 имеет нормальное распределение.
