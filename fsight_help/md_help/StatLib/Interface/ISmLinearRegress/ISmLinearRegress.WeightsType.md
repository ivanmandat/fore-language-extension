# ISmLinearRegress.WeightsType


## Синтаксис


WeightsType: [LRWeightsType](../../Enums/LRWeightsType.htm);


## Описание


Свойство WeightsType определяет
 тип весов.


## Комментарии


В качестве весов используется матрица, элементами которой являются 1/S,
 где S может принимать значения:


	- 0 - дисперсии остатков;


	- 1 - стандартные отклонения
	 остатков.


В качестве весов используется матрица, элементами которой являются S,
 где S может принимать значения:


	- 2 - стандартные отклонение
	 остатков;


	- 3 - дисперсии остатков.
