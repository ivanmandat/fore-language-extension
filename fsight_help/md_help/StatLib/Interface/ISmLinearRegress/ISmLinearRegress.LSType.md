# ISmLinearRegress.LSType


## Синтаксис


LSType: [LRLSType](../../Enums/LRLSType.htm);


## Описание


Свойство LSType определяет тип
 метода наименьших квадратов.


## Комментарии


Метод наименьших квадратов - математический метод, применяемый для решения
 различных задач, основанный на минимизации суммы квадратов остатков целевой
 функции.
