# IExponentialSmoothingAutoSearch.GammaSearch


## Синтаксис


GammaSearch: Boolean;


## Описание


Свойство GammaSearch определяет,
 искать ли параметр «Гамма».


## Комментарии


GammaSearch = True. Параметр
 «Гамма» будет найден автоматически.


GammaSearch = False. Значение
 по умолчанию. Параметр «Гамма»
 определяется пользователем (используйте свойство [IExponentialSmoothingParameters.Gamma](../IExponentialSmoothingParameters/IExponentialSmoothingParameters.Gamma.htm)).
