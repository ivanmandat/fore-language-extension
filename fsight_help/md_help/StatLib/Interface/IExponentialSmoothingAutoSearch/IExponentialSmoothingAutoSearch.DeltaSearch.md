# IExponentialSmoothingAutoSearch.DeltaSearch


## Синтаксис


DeltaSearch: Boolean;


## Описание


Свойство DeltaSearch определяет,
 искать ли параметр «Дельта».


## Комментарии


DeltaSearch = True. Параметр
 «Дельта» будет найден автоматически.


DeltaSearch = False. Значение
 по умолчанию. Параметр «Дельта»
 определяется пользователем (используйте свойство [IExponentialSmoothingParameters.Delta](../IExponentialSmoothingParameters/IExponentialSmoothingParameters.Delta.htm)).
