# IExponentialSmoothingAutoSearch.PhiSearch


## Синтаксис


PhiSearch: Boolean;


## Описание


Свойство PhiSearch определяет,
 искать ли параметр «Фи».


## Комментарии


PhiSearch = True. Параметр «Фи» будет найден автоматически.


PhiSearch = False. Значение
 по умолчанию. Параметр «Фи» определяется
 пользователем (используйте свойство [IExponentialSmoothingParameters.Phi](../IExponentialSmoothingParameters/IExponentialSmoothingParameters.Phi.htm)).
