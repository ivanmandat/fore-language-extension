# IExponentialSmoothingAutoSearch


## Описание


Интерфейс IExponentialSmoothingAutoSearch
 определяет настройки автоподбора параметров для экспоненциального сглаживания.


## Комментарии


Метод «Наискорейшего спуска» осуществляет только поиск тех параметров
 которые требуются в конкретной модели.


Метод «Наилучшей пробы» осуществляет поиск всех параметров независимо
 от выбора модели. Но при построении использует только нужные в данной
 модели параметры. Остальные игнорируются.
