## Синтаксис


Criterion: [CriterionType](../../Enums/CriterionType.htm);


## Описание


Свойство Criterion определяет
 критерий поиска параметров. По умолчанию используется критерий «Сумма
 квадратов остатков» - сумма квадратов отклонений исходных значений зависимой
 переменной от модельных.
