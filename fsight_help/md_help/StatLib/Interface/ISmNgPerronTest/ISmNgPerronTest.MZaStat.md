## Синтаксис

## Описание


Свойство MZaStat
 возвращает значение модифицированной статистики Филлипса-Перрона MZdα.


## Комментарии


Для получения значения модифицированной статистики Эллиота-Розенберга-Стока
 используйте свойство [ISmNgPerronTest.MPTStat](ISmNgPerronTest.MPTStat.htm).
