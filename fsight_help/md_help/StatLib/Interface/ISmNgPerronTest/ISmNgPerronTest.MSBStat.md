# ISmNgPerronTest.MSBStat


## Синтаксис


MSBStat: [IUnitRootTestStatistic](../IUnitRootTestStatistic/IUnitRootTestStatistic.htm);


## Описание


Свойство MSBStat
 возвращает значение модифицированной статистики Бхаргава
 MSBd.


## Комментарии


Для получения значения модифицированной статистики Филлипса-Перрона
 используйте свойства [ISmNgPerronTest.MZaStat](ISmNgPerronTest.MZaStat.htm)
 и [ISmNgPerronTest.MZtStat](ISmNgPerronTest.MZtStat.htm).
