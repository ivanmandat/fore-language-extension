# ISmGeneralizedParetoDistribution


Сборка: Stat;


## Описание


Интерфейс ISmGeneralizedParetoDistribution
 предназначен для генерации выборки псевдослучайных величин из обобщенного
 распределения Парето.


## Иерархия наследования


           [IDistribution](../IDistribution/IDistribution.htm)


           ISmGeneralizedParetoDistribution


## Комментарии


Метод позволяет сгенерировать выборку из обобщенного распределения Парето
 и оценить параметры распределения по выборке с помощью метода максимального
 правдоподобия.
