# ICensus2SigmaLevel


Сборка: Stat;


## Описание


Интерфейс ICensus2SigmaLevel
 определяет границы Sigma для метода «X11».


## Иерархия наследования


           ICensus2SigmaLevel


## Комментарии


Метод X11 поддерживается только в ОС Windows.
