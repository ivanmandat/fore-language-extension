# ISmLinearEquations.MaxIteration


## Синтаксис


MaxIteration: Integer;


## Описание


Свойство MaxIteration определяет
 максимальное число итераций, за которое должен осуществиться поиск решения.


## Комментарии


Для определения точности решения используйте свойство [ISmLinearEquations.Tolerance](ISmLinearEquations.Tolerance.htm).
