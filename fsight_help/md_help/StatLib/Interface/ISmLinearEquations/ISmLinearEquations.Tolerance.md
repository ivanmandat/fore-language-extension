# ISmLinearEquations.Tolerance


## Синтаксис


Tolerance: Double;


## Описание


Свойство Tolerance определяет
 точность решения.


## Комментарии


Значение по умолчанию - «0.0000001».


Для определения максимального числа итераций, за которое должен осуществиться
 поиск решения, используйте свойство [ISmLinearEquations.MaxIteration](ISmLinearEquations.MaxIteration.htm).
