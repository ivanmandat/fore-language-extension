# ISmLinearEquations.SLEMethod


## Синтаксис


SLEMethod: [SLEMethodType](../../Enums/SLEMethodType.htm);


## Описание


Свойство SLEMethod определяет
 метод решения системы линейных уравнений.


## Комментарии


Для определения, приводить ли систему линейных уравнений к эквивалентной
 с положительно определённой симметричной матрицей, используйте свойство
 [ISmLinearEquations.SymmetricMatrixSet](ISmLinearEquations.SymmetricMatrixSet.htm).
