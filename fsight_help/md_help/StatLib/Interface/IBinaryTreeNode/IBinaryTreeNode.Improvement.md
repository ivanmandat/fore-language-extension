# IBinaryTreeNode.Improvement


## Синтаксис


Improvement: Double;


## Описание


Свойство Improvement возвращает
 коэффициент улучшения.


## Комментарии


Свойство значимо только при расчете ансамбля деревьев [ISmRandomForest](../ISmRandomForest/ISmRandomForest.htm)
 и бинарных деревьев [ISmCART](../ISmCART/ISmCART.htm).


Для получения проверяемого значения используйте свойство [IBinaryTreeNode.Value](IBinaryTreeNode.Value.htm).
