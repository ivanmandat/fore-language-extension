# IBinaryTreeNode.RightNodeCategories


## Синтаксис


RightNodeCategories: Array;


## Описание


Свойство RightNodeCategories
 возвращает список категорий, которые отсеиваются в правую ветку.


## Комментарии


Свойство значимо только для категориальной классифицирующей переменной.


Для получения списка категорий, которые отсеиваются в левую ветку используйте
 свойство [IBinaryTreeNode.LeftNodeCategories](IBinaryTreeNode.LeftNodeCategories.htm).
