# IBinaryTreeNode.Parent


## Синтаксис


Parent: [IBinaryTreeNode](IBinaryTreeNode.htm);


## Описание


Свойство Parent возвращает корневую
 вершину бинарного дерева.


## Комментарии


Для получения левой и правой вершин бинарного дерева используйте свойства
 [IBinaryTreeNode.LeftNode](IBinaryTreeNode.LeftNode.htm)
 и [IBinaryTreeNode.RightNode](IBinaryTreeNode.RightNode.htm)
 соответственно.
