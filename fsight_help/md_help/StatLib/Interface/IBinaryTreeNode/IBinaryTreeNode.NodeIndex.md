# IBinaryTreeNode.NodeIndex


## Синтаксис


NodeIndex: Integer;


## Описание


Свойство NodeIndex возвращает
 номер вершины в дереве.


## Комментарии


Нумерация вершин в дереве сквозная и начинается с «0».


Для получения корневой вершины бинарного дерева используйте свойство
 [IBinaryTreeNode.Parent](IBinaryTreeNode.Parent.htm).
