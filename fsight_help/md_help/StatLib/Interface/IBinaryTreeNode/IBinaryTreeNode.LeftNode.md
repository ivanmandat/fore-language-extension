# IBinaryTreeNode.LeftNode


## Синтаксис


LeftNode: [IBinaryTreeNode](IBinaryTreeNode.htm);


## Описание


Свойство LeftNode возвращает
 левую вершину бинарного дерева.


## Комментарии


Для получения правой вершины бинарного дерева используйте свойство [IBinaryTreeNode.RightNode](IBinaryTreeNode.RightNode.htm).
