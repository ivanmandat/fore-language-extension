# IBinaryTreeNode


Сборка: Stat;


## Описание


Интерфейс IBinaryTreeNode предназначен
 для работы с вершинами бинарного дерева.


## Иерархия наследования


           IBinaryTreeNode


## Комментарии


Для работы с коллекцией деревьев используйте свойства интерфейса [IBinaryTreeForest](../IBinaryTreeForest/IBinaryTreeForest.htm).
