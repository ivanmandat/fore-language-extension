# IBinaryTreeNode.Disp


## Синтаксис


Disp: Double;


## Описание


Свойство Disp возвращает дисперсию
 классифицирующей переменной.


## Комментарии


Свойство значимо только при расчете градиентного бустинга [ISmGradientBoostedTree](../ISmGradientBoostedTree/ISmGradientBoostedTree.htm).


Для получения среднего классифицирующей переменной используйте свойство
 [IBinaryTreeNode.Mean](IBinaryTreeNode.Mean.htm).
