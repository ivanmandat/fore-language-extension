# IBinaryTreeNode.LeftNodeCategories


## Синтаксис


LeftNodeCategories: Array;


## Описание


Свойство LeftNodeCategories
 возвращает список категорий, которые отсеиваются в левую ветку.


## Комментарии


Свойство значимо только для категориальной классифицирующей переменной.


Для получения списка категорий, которые отсеиваются в правую ветку используйте
 свойство [IBinaryTreeNode.RightNodeCategories](IBinaryTreeNode.RightNodeCategories.htm).
