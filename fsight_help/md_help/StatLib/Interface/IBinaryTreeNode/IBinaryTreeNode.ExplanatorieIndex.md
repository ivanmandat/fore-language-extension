# IBinaryTreeNode.ExplanatorieIndex


## Синтаксис


ExplanatorieIndex: Integer;


## Описание


Свойство ExplanatorieIndex возвращает
 номер объясняющего ряда, по которому происходит разбиение вершины.


## Комментарии


Для получения номера вершины в дереве используйте свойство [IBinaryTreeNode.NodeIndex](IBinaryTreeNode.NodeIndex.htm).
