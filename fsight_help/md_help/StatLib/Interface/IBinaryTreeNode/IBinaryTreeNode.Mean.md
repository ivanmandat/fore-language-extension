# IBinaryTreeNode.Mean


## Синтаксис


Mean: Double;


## Описание


Свойство Mean возвращает среднее
 классифицирующей переменной.


## Комментарии


Для получения дисперсии классифицирующей переменной используйте свойство
 [IBinaryTreeNode.Disp](IBinaryTreeNode.Disp.htm).


Свойство значимо только при расчете градиентного бустинга [ISmGradientBoostedTree](../ISmGradientBoostedTree/ISmGradientBoostedTree.htm).
