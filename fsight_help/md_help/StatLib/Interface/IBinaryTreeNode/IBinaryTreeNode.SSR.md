# IBinaryTreeNode.SSR


## Синтаксис


SSR: Double;


## Описание


Свойство SSR возвращает сумму
 квадратов остатков.


## Комментарии


Свойство значимо только при расчете градиентного бустинга [ISmGradientBoostedTree](../ISmGradientBoostedTree/ISmGradientBoostedTree.htm).


Сумма квадратов остатков рассчитывается только для регрессионных деревьев.
