# IBinaryTreeNode.Total


## Синтаксис


Total: Integer;


## Описание


Свойство Total возвращает количество
 наблюдений в вершине.


## Комментарии


Для получения количества наблюдений каждой категории в вершине используйте
 свойство [IBinaryTreeNode.Categories](IBinaryTreeNode.Categories.htm).
