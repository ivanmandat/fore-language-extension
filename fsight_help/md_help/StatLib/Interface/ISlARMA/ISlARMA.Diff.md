# ISlARMA.Diff


## Синтаксис


Diff: Integer;


## Описание


Свойство Diff определяет разность.


## Комментарии


Разность определяет порядок дифференцирования исходного временного ряда.


Diff используется при расчете
 авторегрессии и скользящего среднего в модели ARIMA.
