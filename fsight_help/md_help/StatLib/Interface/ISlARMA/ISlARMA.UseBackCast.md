# ISlARMA.UseBackCast


## Синтаксис


UseBackCast: Boolean;


## Описание


Свойство UseBackCast определяет,
 использовать ли ретрополяцию при оценке коэффициентов скользящего среднего.


## Комментарии


Значение свойства учитывается, если [ISlARMA.EstimationApproach](ISlARMA.EstimationApproach.htm) = [ARMAEstimationApproach](../../Enums/ARMAEstimationApproach.htm).LeastSquares.


Допустимые значения:


	- True. Ретрополяция используется;


	- False. Ретрополяция
	 не используется.
