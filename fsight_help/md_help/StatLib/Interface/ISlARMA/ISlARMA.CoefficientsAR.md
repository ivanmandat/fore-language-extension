# ISlARMA.CoefficientsAR


## Синтаксис


CoefficientsAR: [ICoefficients](../ICoefficients/ICoefficients.htm);


## Описание


Свойство CoefficientsAR возвращает
 коэффициенты авторегрессии.


## Комментарии


Коэффициенты будут рассчитаны, если в [ISlARMA.OrderAR](ISlARMA.OrderAR.htm) задан порядок
 авторегрессии.
