# ISlARMA.DiffSeas


## Синтаксис


DiffSeas: Integer;


## Описание


Свойство DiffSeas определяет
 сезонную разность.


## Комментарии


Разность определяет порядок дифференцирования сезонной составляющей.


DiffSeas используется при расчете
 авторегрессии и скользящего среднего в модели ARIMA.
