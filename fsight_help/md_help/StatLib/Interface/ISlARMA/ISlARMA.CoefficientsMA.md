# ISlARMA.CoefficientsMA


## Синтаксис


CoefficientsMA: [ICoefficients](../ICoefficients/ICoefficients.htm);


## Описание


Свойство CoefficientsMA возвращает
 коэффициенты скользящего среднего.


## Комментарии


Коэффициенты будут рассчитаны, если в [ISlARMA.OrderMA](ISlARMA.OrderMA.htm) задан порядок
 скользящего среднего.
