# ISlARMA.ARRootsRe


## Синтаксис


ARRootsRe: Array;


## Описание


Свойство ARRootsRe возвращает
 значения действительной части характеристических корней AR процесса.


## Комментарии


Для получения значений мнимой части характеристических корней AR процесса
 используйте свойство [ISlARMA.ARRootsIm](ISlARMA.ARRootsIm.htm).
