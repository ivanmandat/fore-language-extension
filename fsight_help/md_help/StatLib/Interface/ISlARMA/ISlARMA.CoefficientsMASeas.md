# ISlARMA.CoefficientsMASeas


## Синтаксис


CoefficientsMASeas: [ICoefficients](../ICoefficients/ICoefficients.htm);


## Описание


Свойство CoefficientsMASeas возвращает
 коэффициенты сезонного скользящего среднего.


## Комментарии


Коэффициенты будут рассчитаны, если в [ISlARMA.OrderMASeas](ISlARMA.OrderMASeas.htm)
 задан порядок сезонного скользящего среднего.
