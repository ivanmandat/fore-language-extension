# ISlARMA.UseARMAasInstrums


## Синтаксис


UseARMAasInstrums: Boolean;


## Описание


Свойство UseARMAasInstrums определяет,
 использовать ли лагированные значения объясняемой и объясняющей переменных
 в качестве дополнительных инструментов.


## Комментарии


Допустимые значения:


	- True. Использовать дополнительные
	 инструменты. Применяется только для [метода
	 инструментальных переменных](../ISm2SLS/ISm2SLS.htm);


	- False. Не использовать
	 дополнительные инструменты.
