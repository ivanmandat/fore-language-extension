# ISlARMA.MARootsRe


## Синтаксис


MARootsRe: Array;


## Описание


Свойство MARootsRe возвращает
 значения действительной части характеристических корней MA процесса.


## Комментарии


Для получения значений мнимой части характеристических корней MA процесса
 используйте свойство [ISlARMA.MARootsIm](ISlARMA.MARootsIm.htm).
