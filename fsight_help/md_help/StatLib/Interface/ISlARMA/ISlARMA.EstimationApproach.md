# ISlARMA.EstimationApproach


## Синтаксис


EstimationApproach: [ARMAEstimationApproach](../../Enums/ARMAEstimationApproach.htm);


## Описание


Свойство EstimationApproach
 определяет метод оценки коэффициентов.


## Комментарии


Доступно два метода оценки коэффициентов: метод наименьших квадратов
 и метод максимального правдоподобия. Первый позволяет выбрать оптимизационный
 метод: Левенберга-Маркгвардта или квазиньютоновский метод.


Метод оптимизации определяет свойство [ISlARMA.EstimationMethod](ISlARMA.EstimationMethod.htm).
