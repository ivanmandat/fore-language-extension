# IMSGARCHCoefficients


Сборка: Stat;


## Описание


Интерфейс IMSGARCHCoefficients
 предназначен для работы с коэффициентами MS-GARCH.


## Иерархия наследования


           IMSGARCHCoefficients


## Комментарии


Для работы с интерфейсом IMSGARCHCoefficients
 используйте свойство [ISmMarkovSwitchingGARCH.ARCHCoefficients](../ISmMarkovSwitchingGARCH/ISmMarkovSwitchingGARCH.ARCHCoefficients.htm).
