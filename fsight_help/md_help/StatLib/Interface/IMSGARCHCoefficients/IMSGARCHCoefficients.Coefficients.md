# IMSGARCHCoefficients.Coefficients


## Синтаксис


Coefficients: [ICoefficients](../ICoefficients/ICoefficients.htm);


## Описание


Свойство Coefficients возвращает
 оценки коэффициентов MS-GARCH и их характеристики.


## Комментарии


Для определения среднего значения и ковариационной матрицы для генерации
 используйте свойство [IMSGARCHCoefficients.InitMeanValues](IMSGARCHCoefficients.InitMeanValues.htm)
 и [IMSGARCHCoefficients.InitCovarianceMatrix](IMSGARCHCoefficients.InitCovarianceMatrix.htm).
