## Синтаксис

## Описание


Свойство BoundaryUpper определяет
 верхнюю границу.


## Комментарии


Для определения нижней границы линейного ограничения используйте свойство
 [ISlLinearConstraint.BoundaryLower](ISlLinearConstraint.BoundaryLower.htm).
