# ISlLinearConstraint


Сборка: Stat;


## Описание


Интерфейс ISlLinearConstraint
 определяет параметры линейного ограничения.


## Иерархия наследования


           ISlLinearConstraint


## Комментарии


Для решения задачи линейного программирования используйте интерфейс
 [ISmLinearProgramming](../ISmLinearProgramming/ISmLinearProgramming.htm).
