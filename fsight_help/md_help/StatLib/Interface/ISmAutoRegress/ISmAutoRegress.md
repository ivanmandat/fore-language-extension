# ISmAutoRegress


Сборка: Stat;


## Описание


Интерфейс ISmAutoRegress определяет
 параметры авторегрессии.


## Иерархия наследования


           [IStatMethod](../IStatMethod/IStatMethod.htm)


           ISmAutoRegress


## Комментарии


Авторегрессия предназначена для описания стационарных временных рядов.
