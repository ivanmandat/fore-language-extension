# ISmAutoRegress.AutoRegressionOrder


## Синтаксис


AutoRegressionOrder: Array;


## Описание


Свойство AutoRegressionOrder
 определяет порядки авторегрессий.


## Комментарии


Число элементов в массиве определяет число регрессоров, каждый элемент
 массива - порядок авторегрессии для каждого регрессора.
