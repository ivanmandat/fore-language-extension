# ISmDiscriminantAnalysis.WithinStdDev


## Синтаксис


WithinStdDev: Array;


## Описание


Свойство WithinStdDev возвращает
 внутригрупповое стандартное отклонение для каждого объясняющего ряда.


## Комментарии


Общее стандартное отклонение возвращает свойство [ISmDiscriminantAnalysis.TotalStdDev](ISmDiscriminantAnalysis.TotalStdDev.htm).
