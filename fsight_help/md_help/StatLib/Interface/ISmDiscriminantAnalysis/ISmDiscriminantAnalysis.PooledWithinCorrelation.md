# ISmDiscriminantAnalysis.PooledWithinCorrelation


## Синтаксис


PooledWithinCorrelation: Array;


## Описание


Свойство PooledWithinCorrelation
 возвращает усреднённую внутригрупповую корреляцию для каждого объясняющего
 ряда.


## Комментарии


Внутригрупповую корреляцию для каждого объясняющего ряда возвращает
 свойство [ISmDiscriminantAnalysis.WithinCorrelation](ISmDiscriminantAnalysis.WithinCorrelation.htm).
