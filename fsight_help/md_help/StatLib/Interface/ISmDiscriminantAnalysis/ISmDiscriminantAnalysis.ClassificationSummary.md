# ISmDiscriminantAnalysis.ClassificationSummary


## Синтаксис


ClassificationSummary: Array;


## Описание


Свойство ClassificationSummary
 возвращает сводную матрицу классификации.


## Комментарии


Матрица содержит только целочисленные значения и отражает, к каким классам
 были отнесены известные объекты. В последней строке располагаются объекты,
 подлежащие кластеризации, остальные строки соответствуют классам. В строках
 матрицы располагаются номера классов, к которым были отнесены объекты.
 Последний столбец - сводный: сумма всех элементов строки.


Количество классов возвращает свойство [ISmDiscriminantAnalysis.ClassCount](ISmDiscriminantAnalysis.ClassCount.htm).
