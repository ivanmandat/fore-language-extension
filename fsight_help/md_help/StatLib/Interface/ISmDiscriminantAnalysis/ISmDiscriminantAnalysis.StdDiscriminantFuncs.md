# ISmDiscriminantAnalysis.StdDiscriminantFuncs


## Синтаксис


StdDiscriminantFuncs: Array;


## Описание


Свойство StdDiscriminantFuncs
 возвращает стандартизованные коэффициенты канонических дискриминантных
 функций.


## Комментарии


Нестандартизованные коэффициенты канонических дискриминантных функций
 возвращает свойство [ISmDiscriminantAnalysis.NonStdDiscriminantFuncs](ISmDiscriminantAnalysis.NonStdDiscriminantFuncs.htm).
