# ISmDiscriminantAnalysis


Сборка: Stat;


## Описание


Интерфейс ISmDiscriminantAnalysis применяется
 для интеллектуального анализа данных методом «Дискриминантный
 анализ».


## Иерархия наследования


           [IStatMethod](../IStatMethod/IStatMethod.htm)


           [IStatMethodValid](../IStatMethodValid/IStatMethodValid.htm)


           [IDataMining](../IDataMining/IDataMining.htm)


           ISmDiscriminantAnalysis


## Комментарии


Данный метод выполняет поиск линейной комбинации переменных, наилучшим
 образом разделяющих (дискриминирующих) два или более класса.
