# ISmDiscriminantAnalysis.TotalStdDev


## Синтаксис


TotalStdDev: Array;


## Описание


Свойство TotalStdDev возвращает
 общее стандартное отклонение для каждого объясняющего ряда.


## Комментарии


Внутригрупповое стандартное отклонение возвращает свойство [ISmDiscriminantAnalysis.WithinStdDev](ISmDiscriminantAnalysis.WithinStdDev.htm).
