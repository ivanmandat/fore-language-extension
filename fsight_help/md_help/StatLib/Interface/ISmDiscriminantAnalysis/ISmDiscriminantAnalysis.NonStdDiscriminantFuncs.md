# ISmDiscriminantAnalysis.NonStdDiscriminantFuncs


## Синтаксис

## Описание


Свойство NonStdDiscriminantFuncs
 возвращает нестандартизованные коэффициенты канонических дискриминантных
 функций.


## Комментарии

 возвращает свойство [ISmDiscriminantAnalysis.StdDiscriminantFuncs](ISmDiscriminantAnalysis.StdDiscriminantFuncs.htm).
