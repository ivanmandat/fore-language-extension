# ISmDiscriminantAnalysis.WithinCovariance


## Синтаксис


WithinCovariance: Array;


## Описание


Свойство WithinCovariance возвращает
 внутригрупповую ковариацию для каждого объясняющего ряда.


## Комментарии


Усреднённую внутригрупповую ковариацию возвращает свойство [ISmDiscriminantAnalysis.PooledWithinCovariance](ISmDiscriminantAnalysis.PooledWithinCovariance.htm).
