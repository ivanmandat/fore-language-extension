# ISmDiscriminantAnalysis.PooledWithinCovariance


## Синтаксис


PooledWithinCovariance: Array;


## Описание


Свойство PooledWithinCovariance
 возвращает усреднённую внутригрупповую ковариацию для каждого объясняющего
 ряда.


## Комментарии


Внутригрупповую ковариацию для каждого объясняющего ряда возвращает
 свойство [ISmDiscriminantAnalysis.WithinCovariance](ISmDiscriminantAnalysis.WithinCovariance.htm).
