# ISmCurveEstimation.ForecastLastPoint


## Синтаксис


ForecastLastPoint: Integer;


## Описание


Свойство ForecastLastPoint определяет
 последнюю точку прогнозного ряда.


## Комментарии


Данная точка отсчитывается от единицы и должна быть больше, чем конечная
 точка периода идентификации. Последняя точка прогнозного ряда задаётся
 сразу для всех форм.
