# ISmCurveEstimation.PolynomOrder


## Синтаксис


PolynomOrder: Integer;


## Описание


Свойство PolynomOrder определяет
 степень полинома для полиномиальной формы зависимости.


## Комментарии


По умолчанию свойство имеет значение «3».
