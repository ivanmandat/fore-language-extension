# ISmCurveEstimation.BestDependenceForm


## Синтаксис


BestDependenceForm: Integer;


## Описание


Свойство BestDependenceForm
 возвращает номер лучшей формы зависимости.


## Комментарии


Лучшая форма зависимости определяется по значению критерия [IDependenceForm.CriterionValue](../IDependenceForm/IDependenceForm.CriterionValue.htm).
