# ISmCurveEstimation.DependenceForms


## Синтаксис


DependenceForms: [IDependenceForms](../IDependenceForms/IDependenceForms.htm);


## Описание


Свойство DependenceForms возвращает
 коллекцию форм зависимости.


## Комментарии


Используемые формы зависимостей: линейная, квадратичная, полиномиальная,
 составная, уравнение роста, логарифмическая, гиперболическая, экспоненциальная,
 обратная, степенная, логистическая.
