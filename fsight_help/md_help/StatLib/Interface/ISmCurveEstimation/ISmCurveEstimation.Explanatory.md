# ISmCurveEstimation.Explanatory


## Синтаксис


Explanatory: [ICurveEstimationExplanatory](../ICurveEstimationExplanatory/ICurveEstimationExplanatory.htm);


## Описание


Свойство Explanatory определяет
 параметры объясняющего ряда.


## Комментарии


Если объясняемый ряд содержит [пропуски](ISmCurveEstimation.MissingData.htm)
 (либо [FirstPoint](ISmCurveEstimation.ModelPeriod.htm) > 1),
 то заполнение ряда Explanatory
 начинается с точки, соответствующей первой непустой точке в объясняемом
 ряду.
