# ISmCurveEstimation.ForecastFirstPoint


## Синтаксис


ForecastFirstPoint: Integer;


## Описание


Свойство ForecastFirstPoint
 определяет первую точку прогнозного ряда.


## Комментарии


Для задания последней точки прогнозного ряда используйте свойство [ISmCurveEstimation.ForecastLastPoint](ISmCurveEstimation.ForecastLastPoint.htm).
