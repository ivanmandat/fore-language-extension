# ISmCurveEstimation.Criterion


## Синтаксис


Criterion: [DependenceCriterion](../../Enums/DependenceCriterion.htm);


## Описание


Свойство Criterion определяет
 тип критерия отбора лучшей формы зависимости.


## Комментарии


Для определения типа критерия отбора лучшей формы зависимости используйте
 перечисление [DependenceCriterion](../../Enums/DependenceCriterion.htm).
 По умолчанию используется критерий «Коэффициент
 детерминации».
