# ISmCurveEstimation.SortedModelList


## Синтаксис


SortedModelList: Array;


## Описание


Свойство SortedModelList возвращает
 список индексов форм зависимостей, отсортированный в порядке ухудшения
 значения критерия отбора лучшей зависимости.


## Комментарии


Лучшая форма зависимости определяется по значению критерия [IDependenceForm.CriterionValue](../IDependenceForm/IDependenceForm.CriterionValue.htm).
