# IBinningSettings.Clusters


## Синтаксис


Clusters: [IClustersType](../IClustersType/IClustersType.htm);


## Описание


Свойство Clusters возвращает
 последовательность кластеров.


## Комментарии


Результаты доступны после выполнения метода [IStatMethod.Execute](../IStatMethod/IStatMethod.Execute.htm).


Количество кластеров соответствует количеству итоговых категорий, заданных
 в свойстве [IBinningSettings.NumOfCategories](IBinningSettings.NumOfCategories.htm).
