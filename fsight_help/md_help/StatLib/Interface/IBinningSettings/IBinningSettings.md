# IBinningSettings


Сборка: Stat;


## Описание


Интерфейс IBinningSettings предназначен
 для работы с параметрами процедуры Binning.


## Иерархия наследования


           IBinningSettings


## Комментарии


Для получения параметров процедуры Binning используйте свойства:


	- [ISlSerie.BinningSettings](../ISlSerie/ISlSerie.BinningSettings.htm);


	- [ISmBinning.Settings](../ISmBinning/ISmBinning.Settings.htm).
