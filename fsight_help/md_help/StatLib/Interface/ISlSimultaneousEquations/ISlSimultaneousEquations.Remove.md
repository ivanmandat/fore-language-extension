# ISlSimultaneousEquations.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс уравнения в коллекции.


## Описание


Свойство Remove удаляет уравнение по заданному индексу.


## Комментарии


В случае успешного удаления ряда метод возвращает True, в противном случае False.
