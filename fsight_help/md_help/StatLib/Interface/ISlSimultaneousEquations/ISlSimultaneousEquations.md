# ISlSimultaneousEquations


Сборка: Stat;


## Описание


Интерфейс ISlSimultaneousEquations
 определяет параметры коллекции уравнений.


## Иерархия наследования


           ISlSimultaneousEquations


## Комментарии


Элементы коллекции реализованы интерфейсом [ISlSimultaneousEquation](../ISlSimultaneousEquation/ISlSimultaneousEquation.htm).
