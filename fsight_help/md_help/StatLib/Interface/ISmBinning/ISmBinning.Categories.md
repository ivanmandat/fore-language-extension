# ISmBinning.Categories


## Синтаксис


Categories: Array;


## Описание


Свойство Categories возвращает
 результаты выполнения процедуры Binning.


## Комментарии


Результаты доступны после выполнения метода [IStatMethod.Execute](../IStatMethod/IStatMethod.Execute.htm).


Значение элемента в результирующем массиве содержит номер категории,
 к которой относится соответствующий элемент исходного массива векторов.
 Для задания массива исходных векторов используйте свойство [ISmBinning.Values](ISmBinning.Values.htm).
