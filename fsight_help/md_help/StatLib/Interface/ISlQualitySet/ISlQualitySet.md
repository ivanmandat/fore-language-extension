# ISlQualitySet


## Описание


Интерфейс ISlQualitySet содержит
 свойства для работы с характеристиками, описывающим качество подбора значения
 бинарной модели.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property.gif)
		 [Actual](ISlQualitySet.Actual.htm)


		 Свойство Actual определяет
		 фактическое количество значений.


		 ![](../../Property.gif)
		 [CorrectPrediction](ISlQualitySet.CorrectPrediction.htm)


		 Свойство CorrectPrediction
		 определяет количество правильно предсказанных значений.


		 ![](../../Property.gif)
		 [Prediction](ISlQualitySet.Prediction.htm)


		 Свойство Prediction
		 определяет количество предсказанных значений.
