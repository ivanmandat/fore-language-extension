# IClusterType.ObjectsList


## Синтаксис


ObjectsList: Array;


## Описание


Свойство ObjectsList возвращает
 список объектов, принадлежащих кластеру.


## Комментарии


Для получения центра кластера используйте свойство [IClusterType.Center](IClusterType.Center.htm).
