# IClusterType.Center


## Синтаксис


Center: Array;


## Описание


Свойство Center возвращает центр
 кластера.


## Комментарии


Для получения числа объектов, принадлежащих данному кластеру, используйте
 свойство [IClusterType.Size](IClusterType.Size.htm).
