# IClusterType.Size


## Синтаксис


Size: Integer;


## Описание


Свойство Size возвращает число
 объектов, принадлежащих данному кластеру.


## Комментарии


Для получения списка объектов, принадлежащих кластеру, используйте свойство
 [IClusterType.ObjectsList](IClusterType.ObjectsList.htm).
