# ISmKolmogorovSmirnovTest.SuppliedParameters


## Синтаксис


SuppliedParameters: Boolean;


## Описание


Свойство SuppliedParameters
 определяет, будут ли параметры заданы пользователем или вычислены автоматически
 ([Parameter1](ISmKolmogorovSmirnovTest.Parameter1.htm) и/или
 [Parameter2](ISmKolmogorovSmirnovTest.Parameter2.htm)).


## Комментарии


При значении True, значения
 параметров распределения должны быть заданы пользователем, при значении
 False - значения параметров определяются
 автоматически. По умолчанию свойство имеет значение False.


Данное свойство не используется, если рассматривается случай двух выборок.
