# ISmKolmogorovSmirnovTest.Sample1


## Синтаксис


Sample1: [ISlSerie](../ISlSerie/ISlSerie.htm);


## Описание


Свойство Sample1 определяет
 рассматриваемый ряд. Выборка должна согласоваться с выбранным законом
 распределения.


		 Распределение
		 Ограничения на значение


		 Нормальное распределение
		 нет ограничений


		 Равномерное распределение
		 [Parameter1](ISmKolmogorovSmirnovTest.Parameter1.htm)
		 ≤ x[i] ≤ [Parameter2](ISmKolmogorovSmirnovTest.Parameter2.htm),
		 i = 1,…,n


		 Гамма-распределение
		 x[i] ≥ 0, i=1,…,n


		 Бета-распределение
		 0 ≤ x[i] ≤ 1, i=1,…,n


		 Биномиальное распределение
		 0 ≤ x[i] ≤ [Parameter1](ISmKolmogorovSmirnovTest.Parameter1.htm),
		 i=1


		 Экспоненциальное распределение
		 x[i] ≥ 0, i=1,…,n


		 Распределение Пуассона
		 x[i] ≥ 0, i=1,…,n
