# ILinearProgrammingInitApproximation


## Описание


Интерфейс ILinearProgrammingInitApproximation
 определяет начальные значения.


## Комментарии


Начальные значения используются для решения задач линейного и квадратичного
 программирования. Для получения параметров начальных значений используйте
 свойства:


	- [ISmLinearProgramming.InitialApproximation](../ISmLinearProgramming/ISmLinearProgramming.InitialApproximation.htm);


	- [ISmQuadraticProgramming.InitialApproximation](../ISmQuadraticProgramming/ISmQuadraticProgramming.InitalApproximation.htm).
