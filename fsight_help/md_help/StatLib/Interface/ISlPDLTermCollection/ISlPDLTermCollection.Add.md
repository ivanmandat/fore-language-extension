# ISlPDLTermCollection.Add


## Синтаксис


Add: [ISlPDLTerm](../ISlPDLTerm/ISlPDLTerm.htm);


## Описание


Метод Add добавляет переменную
 в коллекцию.


## Комментарии


Add после выполнения возвращает
 добавленную переменную. Переменная добавляется в конец коллекции, при
 этом значение свойства [ISlPDLTermCollection.Count](ISlPDLTermCollection.Count.htm)
 увеличивается на единицу.
