# ISlPDLTermCollection.Clear


## Синтаксис


Clear;


## Описание


Метод Clear очищает коллекцию
 лаговых переменных.


## Комментарии


После вызова Clear значение
 свойства [ISlPDLTermCollection.Count](ISlPDLTermCollection.Count.htm)
 становится равным нулю.
