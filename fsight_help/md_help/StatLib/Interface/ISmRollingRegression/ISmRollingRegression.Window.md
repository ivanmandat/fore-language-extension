# ISmRollingRegression.Window


## Синтаксис


Window: Integer;


## Описание


Свойство Window определяет ширину
 окна скользящей регрессии.


## Комментарии


Значение параметра «Ширина окна» не может превышать период идентификации
 и должно быть больше единицы.
