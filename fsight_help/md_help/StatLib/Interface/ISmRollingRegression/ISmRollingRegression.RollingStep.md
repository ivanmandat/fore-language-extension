# ISmRollingRegression.RollingStep


## Синтаксис


RollingStep: Integer;


## Описание


Свойство RollingStep определяет
 шаг сдвига окна.


## Комментарии


Значение параметра «Шаг сдвига окна» не может превышать значения параметра
 «Ширина окна» и должно быть больше единицы.
