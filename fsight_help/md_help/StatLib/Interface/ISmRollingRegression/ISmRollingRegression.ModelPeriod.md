# ISmRollingRegression.ModelPeriod


## Синтаксис


ModelPeriod: [IStatPeriod](../IStatPeriod/IStatPeriod.htm);


## Описание


Свойство ModelPeriod возвращает
 параметры периода идентификации.


## Комментарии


По умолчанию длина периода расчета модели совпадает с длиной объясняемого
 ряда.
