# ISmRollingRegression.StandartErrorMatrix


## Синтаксис


StandartErrorMatrix: Array;


## Описание


Свойство StandartErrorMatrix
 возвращает матрицу стандартных ошибок для коэффициентов.


## Комментарии


Для получения матрицы коэффициентов используйте свойство [ISmRollingRegression.CoefficientsMatrix](ISmRollingRegression.CoefficientsMatrix.htm).
