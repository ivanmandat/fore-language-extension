# IDistribution


## Описание


Интерфейс IDistribution предназначен
 для работы с распределением.


## Комментарии


Данный интерфейс является базовым для всех распределений.

## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property.gif)
		 [Cumulative](IDistribution.Cumulative.htm)
		 Свойство Cumulative
		 возвращает функцию распределения.


		 ![](../../Property.gif)
		 [Density](IDistribution.Density.htm)
		 Свойство Density возвращает
		 плотность.


		 ![](../../Property.gif)
		 [DisplayName](IDistribution.DisplayName.htm)
		 Свойство DisplayName
		 возвращает наименование распределения.


		 ![](../../Property.gif)
		 [Errors](IDistribution.Errors.htm)
		 Свойство Errors возвращает
		 сообщение обо всех ошибках и предупреждениях, возникших при расчетах.


		 ![](../../Property.gif)
		 [Name](IDistribution.Name.htm)
		 Свойство Name возвращает
		 идентификатор распределения.


		 ![](../../Property.gif)
		 [Quantile](IDistribution.Quantile.htm)
		 Свойство Quantile возвращает
		 квантиль.


		 ![](../../Property.gif)
		 [Random](IDistribution.Random.htm)
		 Свойство Random возвращает
		 случайное число, соответствующее распределению.


		 ![](../../Property.gif)
		 [Status](IDistribution.Status.htm)
		 Свойство Status возвращает
		 статус выполнения расчета.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub.gif)
		 [MaximumLikelihoodFitting](IDistribution.MaximumLikelihoodFitting.htm)

		 ![](../../Sub.gif)
		 [RandomVector](IDistribution.RandomVector.htm)
