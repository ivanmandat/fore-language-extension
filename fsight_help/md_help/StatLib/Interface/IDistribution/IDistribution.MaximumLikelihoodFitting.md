# IDistribution.MaximumLikelihoodFitting


## Синтаксис


MaximumLikelihoodFitting(Sample: Array): Integer;


## Параметры


Sample. Массив данных.


## Описание


Метод MaximumLikelihoodFitting
 возвращает оценки максимального правдоподобия параметров распределения.


## Комментарии


Метод максимального правдоподобия - способ оценивания неизвестных параметров.
