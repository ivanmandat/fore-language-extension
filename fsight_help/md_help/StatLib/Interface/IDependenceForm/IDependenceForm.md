# IDependenceForm


## Описание


Интерфейс IDependenceForm определяет
 параметры формы зависимости.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property.gif)
		 [CriterionValue](IDependenceForm.CriterionValue.htm)


		 Свойство CriterionValue
		 возвращает значение критерия для формы зависимости.


		 ![](../../Property.gif)
		 [DisplayName](IDependenceForm.DisplayName.htm)


		 Свойство DisplayName
		 возвращает наименование формы зависимости.


		 ![](../../Property.gif)
		 [Errors](IDependenceForm.Errors.htm)


		 Свойство Errors возвращает
		 сообщение обо всех ошибках и предупреждениях для формы зависимости.


		 ![](../../Property.gif)
		 [Failed](IDependenceForm.Failed.htm)


		 Свойство Failed возвращает
		 True, если при расчетах
		 для формы зависимости произошли ошибки, и False,
		 если ошибок не было.


		 ![](../../Property.gif)
		 [Fitted](IDependenceForm.Fitted.htm)


		 Свойство Fitted возвращает
		 массив значений модельного ряда формы зависимости без учета сезонности.


		 ![](../../Property.gif)
		 [FittedSeason](IDependenceForm.FittedSeason.htm)


		 Свойство FittedSeason
		 возвращает массив значений модельного ряда формы зависимости с
		 учетом сезонности.


		 ![](../../Property.gif)
		 [Forecast](IDependenceForm.Forecast.htm)


		 Свойство Forecast определяет
		 параметры прогнозного ряда формы зависимости без учета сезонности.


		 ![](../../Property.gif)
		 [ForecastSeasonality](IDependenceForm.ForecastSeasonality.htm)


		 Свойство ForecastSeasonality
		 определяет параметры прогнозного ряда формы зависимости с учетом
		 сезонности.


		 ![](../../Property.gif)
		 [Include](IDependenceForm.Include.htm)


		 Свойство Include определяет
		 признак включения формы зависимости в расчеты.


		 ![](../../Property.gif)
		 [ModelCoefficients](IDependenceForm.ModelCoefficients.htm)


		 Свойство ModelCoefficients
		 возвращает коэффициенты формы зависимости.


		 ![](../../Property.gif)
		 [Name](IDependenceForm.Name.htm)


		 Свойство Name возвращает
		 наименование формы зависимости.


		 ![](../../Property.gif)
		 [Residuals](IDependenceForm.Residuals.htm)


		 Свойство Residuals
		 возвращает ряд остатков формы зависимости без учета сезонности.


		 ![](../../Property.gif)
		 [ResidualsSeason](IDependenceForm.ResidualsSeason.htm)


		 Свойство ResidualsSeason
		 возвращает ряд остатков формы зависимости с учетом сезонности.


		 ![](../../Property.gif)
		 [SummaryStatistics](IDependenceForm.SummaryStatistics.htm)


		 Свойство SummaryStatistics
		 возвращает статистические характеристики формы зависимости.
