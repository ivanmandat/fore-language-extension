# ITreeSpecification


Сборка: Stat;


## Описание


Интерфейс ITreeSpecification
 предназначен для работы с параметрами, описывающими размер дерева.


## Иерархия наследования


           ITreeSpecification


## Комментарии


Для работы с ансамблем деревьев решений Random Forest используйте свойства
 интерфейса [ISmRandomForest](../ISmRandomForest/ISmRandomForest.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property.gif)
		 [MaximumNumberOfLevels](ITreeSpecification.MaximumNumberOfLevels.htm)
		 Свойство MaximumNumberOfLevels
		 определяет максимальное число уровней в дереве.


		 ![](../../Property.gif)
		 [MinimumNumberOfCases](ITreeSpecification.MinimumNumberOfCases.htm)
		 Свойство MinimumNumberOfCases
		 определяет минимальное количество наблюдений, которое может находиться
		 в вершине дерева.


		 ![](../../Property.gif)
		 [ReduceCategories](ITreeSpecification.ReduceCategories.htm)
		 Свойство ReduceCategories
		 определяет, будет ли каждая категория рассматриваться как отдельный
		 признак.
