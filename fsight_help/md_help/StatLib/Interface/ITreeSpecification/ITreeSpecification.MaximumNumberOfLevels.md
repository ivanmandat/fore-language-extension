# ITreeSpecification.MaximumNumberOfLevels


## Синтаксис


MaximumNumberOfLevels: Integer;


## Описание


Свойство MaximumNumberOfLevels
 определяет максимальное число уровней в дереве.


## Комментарии


Значение по умолчанию MaximumNumberOfLevels = 7.
