# ITreeSpecification.ReduceCategories


## Синтаксис


ReduceCategories: Boolean;


## Описание


Свойство ReduceCategories определяет,
 будет ли каждая категория рассматриваться как отдельный признак.


## Комментарии


Допустимые значения:


	- True. Каждая категория
	 рассматривается как отдельный признак;


	- False. Каждая категория
	 не рассматривается как отдельный признак. Значение по умолчанию.


Свойство предназначено только для категориальных признаков. При использовании
 метода сокращается число переборов и время расчёта, но уменьшается точность.
