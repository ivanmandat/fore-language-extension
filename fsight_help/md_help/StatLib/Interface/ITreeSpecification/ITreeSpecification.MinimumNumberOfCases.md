# ITreeSpecification.MinimumNumberOfCases


## Синтаксис


MinimumNumberOfCases: Integer;


## Описание


Свойство MinimumNumberOfCases
 определяет минимальное количество наблюдений, которое может находиться
 в вершине дерева.


## Комментарии


Значение по умолчанию MinimumNumberOfCases = 50.
