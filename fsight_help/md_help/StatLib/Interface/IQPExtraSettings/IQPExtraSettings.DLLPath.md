# IQPExtraSettings.DLLPath


## Синтаксис


		DLLPath: String;


## Описание


Свойство DLLPath определяет
 путь до библиотеки Gurobi.


## Комментарии


Путь должен содержать название файла библиотеки Gurobi.


Библиотека Gurobi по умолчанию расположена в папке S\win32\bin\
 или S\win64\bin\, в зависимости
 от разрядности используемой версии, где S
 - это путь до папки, в которой установлен внешний модуль Gurobi. Наименование
 файла библиотеки - gurobiX.dll, где X - это номер версии Gurobi.
