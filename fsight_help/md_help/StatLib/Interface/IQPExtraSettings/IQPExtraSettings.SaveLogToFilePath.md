# IQPExtraSettings.SaveLogToFilePath


## Синтаксис


SaveLogToFilePath: String;


## Описание


Свойство SaveLogToFilePath определяет
 путь и наименование файла, в который будет сохранён отчёт о расчёте.


## Комментарии


В качестве значения свойства задайте путь и наименование файла в формате
 *.LOG.
