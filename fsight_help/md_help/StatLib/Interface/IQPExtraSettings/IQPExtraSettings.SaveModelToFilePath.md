# IQPExtraSettings.SaveModelToFilePath


## Синтаксис


SaveModelToFilePath: String;


## Описание


Свойство SaveModelToFilePath
 определяет путь и наименование файла, в который будет сохранена модель.


## Комментарии


В качестве значения свойства задайте путь и наименование файла в формате
 *.MPS.
