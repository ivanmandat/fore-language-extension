# IQPExtraSettings


Сборка: Stat;


## Описание


Интерфейс IQPExtraSettings предназначен
 для задания дополнительных настроек, которые необходимы при использовании
 модуля решений Gurobi.


## Иерархия наследования


IQPExtraSettings


## Комментарии


Дополнительные настройки возвращает свойство [ISmQuadraticProgramming.ExtraSettings](../ISmQuadraticProgramming/ISmQuadraticProgramming.ExtraSettings.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property.gif)
		 [DLLPath](IQPExtraSettings.DLLPath.htm)


		 Свойство DLLPath определяет
		 путь до библиотеки Gurobi.


		 ![](../../Property.gif)
		 [MessageString](IQPExtraSettings.MessageString.htm)


		 Свойство MessageString
		 возвращает сообщение, генерируемое модулем решения после расчёта.


		 ![](../../Property.gif)
		 [nErrorCode](IQPExtraSettings.nErrorCode.htm)


		 Свойство nErrorCode
		 возвращает код выполнения, сгенерированный модулем решения.


		 ![](../../Property.gif)
		 [SaveLogToFilePath](IQPExtraSettings.SaveLogToFilePath.htm)


		 Свойство SaveLogToFilePath
		 определяет путь и наименование файла, в который будет сохранён
		 отчёт о расчёте.


		 ![](../../Property.gif)
		 [SaveModelToFilePath](IQPExtraSettings.SaveModelToFilePath.htm)


		 Свойство SaveModelToFilePath
		 определяет путь и наименование файла, в который будет сохранена
		 модель.
