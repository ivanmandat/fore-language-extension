# ISlEquations


Сборка: Stat;


## Описание


Интерфейс ISlEquations определяет
 параметры коллекции уравнений.


## Иерархия наследования


           ISlEquations


## Комментарии


Элементы коллекции реализованы интерфейсом [ISlEquation](../ISlEquation/ISlEquation.htm).
