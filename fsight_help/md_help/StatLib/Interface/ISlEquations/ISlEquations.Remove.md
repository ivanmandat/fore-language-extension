# ISlEquations.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс уравнения.


## Описание


Метод Remove удаляет уравнение по заданному индексу.


## Комментарии


В случае успешного удаления метод возвращает True, в противном случае False.
