# IMCMCParameters.NumOfIterations


## Синтаксис


NumOfIterations: Integer;


## Описание


Свойство NumOfIterations определяет
 количество итераций.


## Комментарии


По умолчанию задано 2000 итераций.


Для определения количества отбрасываемых итераций используйте свойство
 [IMCMCParameters.IterationsToDiscard](IMCMCParameters.IterationsToDiscard.htm).
