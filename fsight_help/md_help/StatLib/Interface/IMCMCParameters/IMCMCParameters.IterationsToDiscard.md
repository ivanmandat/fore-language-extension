# IMCMCParameters.IterationsToDiscard


## Синтаксис


IterationsToDiscard: Integer;


## Описание


Свойство IterationsToDiscard
 определяет количество отбрасываемых итераций.


## Комментарии


По умолчанию задано 300 отбрасываемых итераций.


Для определения количества итераций используйте свойство [IMCMCParameters.NumOfIterations](IMCMCParameters.NumOfIterations.htm).
