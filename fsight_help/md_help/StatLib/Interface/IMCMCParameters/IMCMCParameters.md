# IMCMCParameters


Сборка: Stat;


## Описание


Интерфейс IMCMCParameters предназначен
 для работы с параметрами алгоритма Марковской цепи Монте Карло.


## Иерархия наследования


           IMCMCParameters


## Комментарии


Цепью Маркова называют такую последовательность случайных событий, в
 которой вероятность каждого события зависит только от состояния, в котором
 процесс находится в текущий момент времени и не зависит от более ранних
 состояний.
