# ISmGrangerTest.ResValueMatrix


## Синтаксис


ResValueMatrix: Array;


## Описание


Свойство ResValueMatrix возвращает
 матрицу результатов о наличии или отсутствии обусловленности по Гранжеру.


## Комментарии


Матрица содержит только нули или единицы, в зависимости от сделанного
 вывода на основе расчета теста:


	- 0 - нет обусловленности;


	- 1
	 - есть обусловленности.
