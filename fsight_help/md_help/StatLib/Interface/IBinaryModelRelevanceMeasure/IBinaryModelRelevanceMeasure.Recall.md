# IBinaryModelRelevanceMeasure.Recall


## Синтаксис


Recall: Double;


## Описание


Свойство Recall возвращает значение
 полноты положительного результата.


## Комментарии


Свойство показывает, какая доля положительных результатов из всех положительных
 результатов найдена. Применяется в условиях несбалансированных
 выборок.
