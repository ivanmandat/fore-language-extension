# IBinaryModelRelevanceMeasure.Precision


## Синтаксис


Precision: Double;


## Описание


Свойство Precision возвращает
 значение точности положительного результата.


## Комментарии


Свойство показывает, какая доля найденных положительных результатов
 действительно является положительными. Применяется в условиях несбалансированных
 выборок.
