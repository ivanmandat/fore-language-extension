# IBinaryModelRelevanceMeasure


Сборка: Stat;


## Описание


Интерфейс IBinaryModelRelevanceMeasure
 предназначен для работы с критериями качества бинарной классификации.


## Иерархия наследования


           IBinaryModelRelevanceMeasure


## Комментарии


Интерфейс предназначен для работы с [ISmBinaryModel](../ISmBinaryModel/ISmBinaryModel.htm),
 [ISmCART](../ISmCART/ISmCART.htm),
 [ISmDecisionTree](../ISmDecisionTree/ISmDecisionTree.htm),
 [ISmDiscriminantAnalysis](../ISmDiscriminantAnalysis/ISmDiscriminantAnalysis.htm),
 [ISmGradientBoostedTree](../ISmGradientBoostedTree/ISmGradientBoostedTree.htm),
 [ISmLogisticRegression](../ISmLogisticRegression/ISmLogisticRegression.htm),
 [ISmRandomForest](../ISmRandomForest/ISmRandomForest.htm),
 [ISmBackPropagation](../ISmBackPropagation/ISmBackPropagation.htm).
