# ISmLIML.UseConstantAsInstrument


## Синтаксис


UseConstantAsInstrument: Boolean;


## Описание


Свойство UseConstantAsInstrument определяет,
 использовать ли константу в качестве инструментальной переменной.


## Комментарии


Допустимые значения:


	- True. Константа используется
	 в качестве инструментальной переменной;


	- False. Модель рассчитывается,
	 используя только инструментальные переменные, заданные при помощи
	 свойства [ISmLIML.Instrumental](ISmLIML.Instrumental.htm).
