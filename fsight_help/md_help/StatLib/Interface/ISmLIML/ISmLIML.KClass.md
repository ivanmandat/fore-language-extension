# ISmLIML.KClass


## Синтаксис


KClass: Double;


## Описание


Свойство KClass определяет параметр
 k для метода K-class.


## Комментарии


Если k = 0, то оценка
 совпадет с OLS (векторная регрессия методом наименьших квадратов), а если
 k = 1 - с методом 2SLS
 (векторная регрессия двухшаговым методом наименьших квадратов).


Свойство актуально, если [ISmLIML.UseKClass](ISmLIML.UseKClass.htm) = True.
