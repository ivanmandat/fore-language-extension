# ISmLIML.Instrumental


## Синтаксис

## Описание


Свойство Instrumental возвращает
 инструментальные переменные.


## Комментарии


Для определения, будет ли константа использоваться в качестве инструментальной
 переменной, используйте свойство [ISmLIML.UseConstantAsInstrument](ISmLIML.UseConstantAsInstrument.htm).
