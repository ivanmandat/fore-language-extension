# ISmLIML.ModelCoefficients


## Синтаксис


ModelCoefficients: [IModelCoefficients](../IModelCoefficients/IModelCoefficients.htm);


## Описание


Свойство ModelCoefficients возвращает
 оценки коэффициентов модели и их характеристики.


## Комментарии


Под коэффициентами модели подразумеваются коэффициенты при факторах
 и значение константы.
