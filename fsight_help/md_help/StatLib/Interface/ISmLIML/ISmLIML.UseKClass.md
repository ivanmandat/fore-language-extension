# ISmLIML.UseKClass


## Синтаксис


UseKClass: Boolean;


## Описание


Свойство UseKClass определяет,
 использовать ли метод K-class вместо LIML.


## Комментарии


LIML - метод максимального правдоподобия с ограниченной информацией.


Допустимые значения:


	- True. Используется метод
	 оценки K-class;


	- False. Метод оценки
	 K-class не используется.
