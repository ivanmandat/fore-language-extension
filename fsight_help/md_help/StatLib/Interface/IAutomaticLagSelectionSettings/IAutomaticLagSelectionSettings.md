# IAutomaticLagSelectionSettings


Сборка: Stat;


## Описание


Интерфейс IAutomaticLagSelectionSettings
 предназначен для работы с параметрами автоподбора лага.


## Иерархия наследования


           IAutomaticLagSelectionSettings


## Комментарии


Автоподбор лага осуществляется при помощи информационных критериев.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property.gif)
		 [InformationCriterion](IAutomaticLagSelectionSettings.InformationCriterion.htm)


		 Свойство InformationCriterion
		 определяет тип информационного критерия, используемого при автоподборе
		 лага.


		 ![](../../Property.gif)
		 [MaxLagLength](IAutomaticLagSelectionSettings.MaxLagLength.htm)


		 Свойство MaxLagLength
		 определяет максимальный порядок авторегрессии, используемый при
		 автоподборе.
