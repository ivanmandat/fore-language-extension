# IAutomaticLagSelectionSettings.InformationCriterion


## Синтаксис


InformationCriterion: [InformationCriterionType](../../Enums/InformationCriterionType.htm);


## Описание


Свойство InformationCriterion
 определяет тип информационного критерия, используемого при автоподборе
 лага.


## Комментарии


Для определения максимального лага, используемого при автоподборе, используйте
 свойство [IAutomaticLagSelectionSettings.MaxLagLength](IAutomaticLagSelectionSettings.MaxLagLength.htm).


## Пример


Использование свойства приведено в примере для [ISmDickeyFullerGLSTest.DFGLSStat](../ISmDickeyFullerGLSTest/ISmDickeyFullerGLSTest.DFGLSStat.htm).
