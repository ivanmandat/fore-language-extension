# IAutomaticLagSelectionSettings.MaxLagLength


## Синтаксис


MaxLagLength: Integer;


## Описание


Свойство MaxLagLength определяет
 максимальный порядок авторегрессии, используемый при автоподборе.


## Комментарии


Для определения типа информационного критерия, используемого при автоподборе
 лага, используйте свойство [IAutomaticLagSelectionSettings.InformationCriterion](IAutomaticLagSelectionSettings.InformationCriterion.htm).
