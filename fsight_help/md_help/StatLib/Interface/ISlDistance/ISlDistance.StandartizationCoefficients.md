# ISlDistance.StandartizationCoefficients


## Синтаксис


StandartizationCoefficients: Array;


## Описание


Свойство StandartizationCoefficients
 определяет коэффициенты стандартизации.


## Комментарии


Количество коэффициентов должно быть равно количеству признаков ([ISmHierarchicalClusterAnalysis.Objects](../ISmHierarchicalClusterAnalysis/ISmHierarchicalClusterAnalysis.Objects.htm)).
 Коэффициенты стандартизации необходимо задавать в случае, если был выбран
 способ стандартизации признаков «Значения, заданные пользователем» ([ISlDistance.Standartization](ISlDistance.Standartization.htm)).
 В качестве коэффициентов стандартизации можно использовать любые положительные
 значения.
