# ISlDistance.Values


## Синтаксис


Values: Array;


## Описание


Свойство Values возвращает матрицу
 расстояний между объектами.


## Комментарии


Матрица имеет размерность n x n, где n
 - количество объектов ([ISmHierarchicalClusterAnalysis.Objects](../ISmHierarchicalClusterAnalysis/ISmHierarchicalClusterAnalysis.Objects.htm)).
