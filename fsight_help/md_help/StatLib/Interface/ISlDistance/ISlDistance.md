# ISlDistance


Сборка: Stat;


## Описание


Интерфейс ISlDistance предназначен
 для работы с параметрами расчёта расстояний между объектами.


## Иерархия наследования


           ISlDistance


## Комментарии


Расчет расстояний между объектами используется в кластерном анализе.
 Для работы с кластерным анализом используйте интерфейс [ISmHierarchicalClusterAnalysis](../ISmHierarchicalClusterAnalysis/ISmHierarchicalClusterAnalysis.htm).
