# ISlTramoSeatsExogenousCollection.Item


## Синтаксис


Item(Index: Integer): [ISlTramoSeatsExogenousVar](../ISlTramoSeatsExogenousVar/ISlTramoSeatsExogenousVar.htm);


## Параметры


Index. Индекс элемента.


## Описание


Свойство Item возвращает уравнение
 коллекции по указанному индексу.


## Комментарии


Индексация элементов сквозная и начинается с нуля. Для получения количества
 элементов в коллекции используйте свойство [ISlTramoSeatsExogenousCollection.Count](ISlTramoSeatsExogenousCollection.Count.htm).
