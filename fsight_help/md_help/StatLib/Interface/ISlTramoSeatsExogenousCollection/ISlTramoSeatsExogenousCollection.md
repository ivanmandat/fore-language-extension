# ISlTramoSeatsExogenousCollection


Сборка: Stat;


## Описание


Интерфейс ISlTramoSeatsExogenousCollection предназначен
 для работы с коллекцией экзогенных уравнений.


## Иерархия наследования


           ISlTramoSeatsExogenousCollection


## Комментарии


Для получения коллекции экзогенных переменных (регрессоров) используйте
 свойство [ISmTramoSeats.Exogenous](../ISmTramoSeats/ISmTramoSeats.Exogenous.htm).
