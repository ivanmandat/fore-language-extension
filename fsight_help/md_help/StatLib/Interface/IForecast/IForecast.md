# IForecast


Сборка: Stat;


## Описание


Интерфейс IForecast определяет
 параметры прогнозного ряда.


## Иерархия наследования


           IForecast


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property.gif)
		 [CoefUncertaintyInSECalc](IForecast.CoefUncertaintyInSECalc.htm)


		 Свойство CoefUncertaintyInSECalc
		 определяет, учитывать ли при расчете доверительных границ, что
		 коэффициенты найдены приближенно.


		 ![](../../Property.gif)
		 [ConfidenceLevel](IForecast.ConfidenceLevel.htm)


		 Свойство ConfidenceLevel
		 определяет значимость доверительных границ прогнозного ряда, то
		 есть вероятность того, что неизвестное значение параметра будет
		 накрыто доверительным интервалом.


		 ![](../../Property.gif)
		 [FirstPoint](IForecast.FirstPoint.htm)


		 Свойство FirstPoint
		 определяет первую точку прогноза.


		 ![](../../Property.gif)
		 [LastPoint](IForecast.LastPoint.htm)


		 Свойство LastPoint
		 определяет последнюю точку прогноза.


		 ![](../../Property.gif)
		 [LowerConfidenceLevel](IForecast.LowerConfidenceLevel.htm)


		 Свойство LowerConfidenceLevel
		 возвращает значения нижней доверительной границы прогнозного ряда.


		 ![](../../Property.gif)
		 [StandardError](IForecast.StandardError.htm)


		 Свойство StandardError
		 возвращает значения стандартных ошибок прогнозного ряда.


		 ![](../../Property.gif)
		 [UpperConfidenceLevel](IForecast.UpperConfidenceLevel.htm)


		 Свойство UpperConfidenceLevel
		 возвращает значения верхней доверительной границы прогнозного
		 ряда.


		 ![](../../Property.gif)
		 [Value](IForecast.Value.htm)


		 Свойство Value возвращает
		 массив значений прогнозного ряда.
