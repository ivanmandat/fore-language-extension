# IForecast.StandardError


## Синтаксис


StandardError: Array;


## Описание


Свойство StandardError возвращает значения стандартных ошибок прогнозного ряда.


## Комментарии


Данный ряд рассчитывается только для методов ARIMA, линейная регрессия и универсальный тренд.
