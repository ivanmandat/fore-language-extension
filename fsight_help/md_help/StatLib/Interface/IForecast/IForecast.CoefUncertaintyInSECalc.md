# IForecast.CoefUncertaintyInSECalc


## Синтаксис


CoefUncertaintyInSECalc: Boolean;


## Описание


Свойство CoefUncertaintyInSECalc
 определяет, учитывать ли при расчете доверительных границ, что коэффициенты
 найдены приближенно.


## Комментарии


По умолчанию свойство принимает значение True.


Опция применяется для следующих методов:


	- [Линейная
	 регрессия (метод инструментальных переменных)](Lib.chm::/01_Regression_models/UiModelling_LinearRegr_Method_Instr_Var.htm).


	- [Линейная
	 регрессия (оценка МНК)](lib.chm::/01_Regression_models/UiModelling_LinearRegr_LSM.htm).


	- [ARIMA](Lib.chm::/02_Time_series_analysis/UiModelling_ARIMA.htm).
