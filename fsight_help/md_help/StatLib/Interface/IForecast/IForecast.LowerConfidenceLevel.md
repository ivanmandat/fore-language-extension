# IForecast.LowerConfidenceLevel


## Синтаксис


LowerConfidenceLevel: Array;


## Описание


Свойство LowerConfidenceLevel возвращает значения нижней доверительной границы прогнозного ряда.


## Комментарии


Для получения значения верхней доверительной границы прогнозного ряда используйте свойство [IForecast.UpperConfidenceLevel](IForecast.UpperConfidenceLevel.htm).
