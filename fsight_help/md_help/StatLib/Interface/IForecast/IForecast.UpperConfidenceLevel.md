# IForecast.UpperConfidenceLevel


## Синтаксис


UpperConfidenceLevel: Array;


## Описание


Свойство UpperConfidenceLevel возвращает значения верхней доверительной границы прогнозного ряда.


## Комментарии


Для получения значений нижней доверительной границы прогнозного ряда используйте свойство [IForecast.LowerConfidenceLevel](IForecast.LowerConfidenceLevel.htm).
