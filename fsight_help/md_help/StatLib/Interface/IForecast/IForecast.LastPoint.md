# IForecast.LastPoint


## Синтаксис


LastPoint: Integer;


## Описание


Свойство LastPoint определяет
 последнюю точку прогноза.


## Комментарии


Данная точка отсчитывается от единицы и должна быть больше, чем конечная
 точка периода идентификации. Используется, если необходимо рассчитать
 прогноз.


Для задания начальной точки прогноза используйте свойство [IForecast.FirstPoint](IForecast.FirstPoint.htm).
