# ISlTreeNodes.Item


## Синтаксис


Item(Index: Integer): [ISlTreeNode](../ISlTreeNode/ISlTreeNode.htm);


## Параметры


Index. Индекс узла в коллекции.


## Описание


Свойство Item возвращает
 узел из коллекции по его индексу.


## Комментарии


Индексация узлов в коллекции сквозная и начинается с нуля. Количество
 узлов в коллекции возвращает свойство [ISlTreeNodes.Count](ISlTreeNodes.Count.htm).
