# ISlTreeNodes


Сборка: Stat;


## Описание


Интерфейс ISlTreeNodes предназначен
 для работы с коллекцией узлов дерева решений.


## Иерархия наследования


           ISlTreeNodes


## Комментарии


Корневой узел дерева решений, использованного для заполнения пропусков,
 возвращает свойство [ISmDecisionTree.Tree](../ISmDecisionTree/ISmDecisionTree.Tree.htm).
