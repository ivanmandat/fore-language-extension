# IROCcurve.ConfidenceIntervalLower


## Синтаксис


ConfidenceIntervalLower: Double;


## Описание


Свойство ConfidenceIntervalLower
 возвращает нижнюю границу асимптотического доверительного интервала
 площади под кривой.


## Комментарии


Для получения верхней границы асимптотического доверительного интервала
 используйте свойство [IROCcurve.ConfidenceIntervalUpper](IROCcurve.ConfidenceIntervalUpper.htm).
