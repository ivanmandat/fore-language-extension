# IROCcurve.ConfidenceIntervalUpper


## Синтаксис


ConfidenceIntervalUpper: Double;


## Описание


Свойство ConfidenceIntervalUpper
 возвращает верхнюю границу асимптотического доверительного интервала площади
 под кривой.


## Комментарии


Для получения нижней границы асимптотического доверительного интервала
 используйте свойство [IROCcurve.ConfidenceIntervalLower](IROCcurve.ConfidenceIntervalLower.htm).
