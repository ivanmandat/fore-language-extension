# ISmDecisionTree.FilledLearningSample


## Синтаксис


FilledLearningSample: Array;


## Описание


Свойство FilledLearningSample
 возвращает результаты классификации обучающего множества.


## Комментарии


Данные результаты используются для оценки качества модели. Свойство
 FilledLearningSample возвращает
 классификацию для всех объектов.


Обратите внимание, что при расчете в свойство [IDataMining.FilledDependent](../IDataMining/IDataMining.FilledDependent.htm)
 будут скопированы только непустые значения из [IDataMining.Dependent](../IDataMining/IDataMining.Dependent.htm).
 Таким образом, [IDataMining.FilledDependent](../IDataMining/IDataMining.FilledDependent.htm)
 нельзя использовать для оценки качества модели.
