# ISm2SLS.UseConstantAsInstrument


## Синтаксис


UseConstantAsInstrument: Boolean;


## Описание


Свойство UseConstantAsInstrument
 определяет, использовать ли константу в качестве инструментальной переменной.


## Комментарии


Допустимые значения:


	- True. Константа используется
	 в качестве инструментальной переменной;


	- False. Константа используется
	 стандартным образом.
