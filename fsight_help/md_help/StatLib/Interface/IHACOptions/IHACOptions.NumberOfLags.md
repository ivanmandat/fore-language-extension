# IHACOptions.NumberOfLags


## Синтаксис


NumberOfLags: Integer;


## Описание


Свойство NumberOfLags определяет
 количество лагов.


## Комментарии


Свойство актуально, если для свойства [IHACOptions.LRCLagSpecification](IHACOptions.LRCLagSpecification.htm) не
 используется тип LRCLagSpecificationType.None.
