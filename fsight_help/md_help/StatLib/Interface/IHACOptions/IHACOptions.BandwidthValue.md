# IHACOptions.BandwidthValue


## Синтаксис


BandwidthValue: Double;


## Описание


Свойство BandwidthValue определяет
 значение пропускной способности ядра.


## Комментарии


Свойство актуально, если [IHACOptions.LRCBandwidthMethod](IHACOptions.LRCBandwidthMethod.htm) = LRCBandwidthMethodType.UserSpecified.
