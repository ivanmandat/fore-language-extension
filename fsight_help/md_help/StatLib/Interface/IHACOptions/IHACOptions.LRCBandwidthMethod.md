# IHACOptions.LRCBandwidthMethod


## Синтаксис


LRCBandwidthMethod: [LRCBandwidthMethodType](../../Enums/LRCBandwidthMethodType.htm);


## Описание


Свойство LRCBandwidthMethod
 определяет метод для расчета значения пропускной способности ядра.


## Комментарии


Если LRCBandwidthMethod = LRCBandwidthMethodType.UserSpecified,
 то значение пропускной способности ядра должно быть задано через свойство
 [IHACOptions.BandwidthValue](IHACOptions.BandwidthValue.htm).
