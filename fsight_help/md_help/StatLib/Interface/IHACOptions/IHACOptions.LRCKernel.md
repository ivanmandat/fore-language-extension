# IHACOptions.LRCKernel


## Синтаксис


LRCKernel: [LRCKernelType](../../Enums/LRCKernelType.htm);


## Описание


Свойство LRCKernel определяет
 тип ядра.


## Комментарии


Если LRCKernel = LRCKernelType.UserSpecified,
 то вектор ядра должен быть задан с использованием свойства [IHACOptions.KernelVector](IHACOptions.KernelVector.htm).
