# ISlPDLTerm.PolinomialDegreeP


## Синтаксис


PolinomialDegreeP: Integer;


## Описание


Свойство PolinomialDegreeP определяет
 степень полинома аппроксимации для лаговой переменной.


## Комментарии


Степень полинома должна быть не больше, чем количество факторов, участвующих
 в методе.
