# ISlPDLTerm.StdErrSum


## Синтаксис


StdErrSum: Double;


## Описание


Свойство StdErrSum возвращает
 сумму стандартных ошибок лаговых переменных.


## Комментарии


Свойство возвращает результат только после расчета метода, содержащего
 лаговые переменные. Обратите внимание, что StdErrSum
 возвращает сумму стандартных ошибок всех лаговых переменных метода.
