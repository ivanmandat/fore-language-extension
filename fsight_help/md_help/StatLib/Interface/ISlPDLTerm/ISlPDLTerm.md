# ISlPDLTerm


Сборка: Stat;


## Описание


Интерфейс ISlPDLTerm предназначен
 для работы с лаговой переменной.


## Иерархия наследования


           ISlPDLTerm


## Комментарии


Лаговые переменные используются
 для анализа корреляционных зависимостей между рядами во времени. Подробнее
 см. раздел «Модель распределенных
 лагов» в библиотеке методов и моделей.
