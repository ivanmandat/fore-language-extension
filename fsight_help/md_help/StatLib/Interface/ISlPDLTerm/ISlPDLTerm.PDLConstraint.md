# ISlPDLTerm.PDLConstraint


## Синтаксис


PDLConstraint: [PDLConstraintType](../../Enums/PDLConstraintType.htm);


## Описание


Свойство PDLConstraint определяет
 метод отсечения слагаемых для лаговой переменной.


## Комментарии


По умолчанию отсечение слагаемых для лаговой переменной не применяется.
