# ISlPDLTerm.EstimatesSum


## Синтаксис


EstimatesSum: Double;


## Описание


Свойство EstimatesSum возвращает
 сумму коэффициентов лаговых переменных.


## Комментарии


Свойство возвращает результат только после расчета метода, содержащего
 лаговые переменные. Обратите внимание, что EstimatesSum
 возвращает сумму коэффициентов всех лаговых переменных метода.
