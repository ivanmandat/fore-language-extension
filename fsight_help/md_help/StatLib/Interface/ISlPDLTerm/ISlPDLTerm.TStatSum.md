# ISlPDLTerm.TStatSum


## Синтаксис


TStatSum: Double;


## Описание


Свойство TStatSum возвращает
 сумму t-статистик лаговых переменных.


## Комментарии


Свойство возвращает результат только после расчета метода, содержащего
 лаговые переменные. Обратите внимание, что TStatSum
 возвращает сумму t-статистик всех лаговых переменных метода.
