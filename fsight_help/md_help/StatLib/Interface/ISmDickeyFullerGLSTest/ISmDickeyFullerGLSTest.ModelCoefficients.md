# ISmDickeyFullerGLSTest.ModelCoefficients


## Синтаксис


ModelCoefficients: [ICoefficients](../ICoefficients/ICoefficients.htm);


## Описание


Свойство ModelCoefficients возвращает
 оценки коэффициентов вспомогательной регрессии и их характеристики.


## Комментарии


Под коэффициентами вспомогательной модели подразумеваются коэффициенты
 при факторах и значение константы.
