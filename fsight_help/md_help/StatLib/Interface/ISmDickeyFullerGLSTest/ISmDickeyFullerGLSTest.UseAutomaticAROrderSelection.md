# ISmDickeyFullerGLSTest.UseAutomaticAROrderSelection


## Синтаксис


UseAutomaticAROrderSelection: Boolean;


## Описание


Свойство UseAutomaticAROrderSelection
 определяет, применять ли автоподбор лага.


## Комментарии


Для определения параметров автоподбора лага используйте свойство [ISmDickeyFullerGLSTest.AutomaticAROrderSelectionSettings](ISmDickeyFullerGLSTest.AutomaticAROrderSelectionSettings.htm).


Допустимые значения:


	- True. Автоподбор лага
	 применяется;


	- False. Автоподбор лага
	 не применяется.
