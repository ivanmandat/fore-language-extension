# ISmDickeyFullerGLSTest.AutomaticAROrderSelectionSettings


## Синтаксис


AutomaticAROrderSelectionSettings: [IAutomaticLagSelectionSettings](../IAutomaticLagSelectionSettings/IAutomaticLagSelectionSettings.htm);


## Описание


Свойство AutomaticAROrderSelectionSettings
 определяет параметры автоподбора лага.


## Комментарии


Для определения, применяется ли автоподбор лага, используйте свойство
 [ISmDickeyFullerGLSTest.UseAutomaticAROrderSelection](ISmDickeyFullerGLSTest.UseAutomaticAROrderSelection.htm).
