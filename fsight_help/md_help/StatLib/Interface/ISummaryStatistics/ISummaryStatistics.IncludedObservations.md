# ISummaryStatistics.IncludedObservations


## Синтаксис


IncludedObservations: Integer;


## Описание


Свойство IncludedObservations возвращает
 число наблюдений, фактически используемых для построения модели.


## Комментарии


Свойство отбрасывает пропуски в исходных рядах, а также наблюдения,
 которые не могут быть рассчитаны при применении авторегрессии и преобразований
 (например, дифференцирование в модели ARIMA).
