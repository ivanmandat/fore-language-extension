# ISummaryStatistics


Сборка: Stat;


## Описание


Интерфейс ISummaryStatistics
 возвращает статистические характеристики.


## Иерархия наследования


           ISummaryStatistics


## Комментарии


Статистические характеристики рассчитываются по общим формулам.
