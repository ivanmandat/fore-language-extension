# ISummaryStatistics.McFaddenRsquared


## Синтаксис


McFaddenRsquared: Double;


## Описание


Свойство McFaddenRsquared возвращает коэффициент
 детерминации МакФаддена.


## Комментарии


Коэффициент МакФаддена является аналогом [коэффициента
 детерминации](ISummaryStatistics.R2.htm).


Свойство актуально только для моделей бинарного выбора, содержащих константу.
