# ISummaryStatistics.SC


## Синтаксис


SC: Double;


## Описание


Свойство SC возвращает информационный
 критерий Шварца.


## Комментарии


Критерий используется для выбора набора объясняющих переменных. Предпочтительней
 модель с меньшим значением критерия.
