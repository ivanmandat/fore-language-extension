## Синтаксис

## Описание


Свойство AvgLogL возвращает
 среднее логарифма функции правдоподобия.


## Комментарии


Для получения логарифма функции правдоподобия используйте свойство [ISummaryStatistics.LogL](ISummaryStatistics.LogL.htm).
