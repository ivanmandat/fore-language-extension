# ISummaryStatistics.ProbJBStat


## Синтаксис


ProbJBStat: Double;


## Описание


Свойство ProbJBStat возвращает
 значение вероятности статистики Жака-Бера.


## Комментарии


Статистика Жака-Бера имеет распределение χ2.


Для получения значения статистики Жака-Бера используйте свойство [ISummaryStatistics.JBStat](ISummaryStatistics.JBStat.htm).
