# ISummaryStatistics.ProbFstat_2


## Синтаксис


ProbFstat_2: Double;


## Описание


Свойство ProbFstat_2 возвращает
 вероятность статистики Фишера, рассчитанной на основе нецентрированного
 коэффициента детерминации.


## Комментарии


Для получения статистики Фишера, рассчитанной на основе нецентрированного коэффициента детерминации,
 используйте свойство [ISummaryStatistics.Fstat_2](ISummaryStatistics.Fstat_2.htm).
