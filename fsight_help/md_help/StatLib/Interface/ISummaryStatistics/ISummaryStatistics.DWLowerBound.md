# ISummaryStatistics.DWLowerBound


## Синтаксис


DWLowerBound: Double;


## Описание


Свойство DWLowerBound возвращает
 значение вероятности для нижней границы интервала статистики Дарбина-Уотсона.


## Комментарии


Значения вероятности доверительных границ для статистики Дарбина-Уотсона
 рассчитываются только для моделей с регрессорами.


Для получения значения вероятности для верхней границы интервала статистки
 Дарбина-Уотсона используйте свойство [ISummaryStatistics.DWUpperBound](ISummaryStatistics.DWUpperBound.htm).
