# ISummaryStatistics.NumOfIter


## Синтаксис


NumOfIter: Integer;


## Описание


Свойство NumOfIter возвращает
 число итераций, за которое было получено решение.


## Комментарии


Для неитерационных методов свойство возвращает значение «-1».
