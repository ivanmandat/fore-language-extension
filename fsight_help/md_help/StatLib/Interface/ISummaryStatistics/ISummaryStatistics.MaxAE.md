# ISummaryStatistics.MaxAE


## Синтаксис


MaxAE: Double;


## Описание


Свойство MaxAE возвращает максимальную
 абсолютную величину остатков.


## Комментарии


Для получения среднего абсолютных величин остатков используйте свойство
 [ISummaryStatistics.MAE](ISummaryStatistics.MAE.htm).
