# ISummaryStatistics.Jstat


## Синтаксис


Jstat: Double;


## Описание


Свойство Jstat возвращает значение
 J-статистики.


## Комментарии


Значение J-статистики рассчитывается для регрессионной модели, использующей
 [метод инструментальных переменных](../ISm2SLS/ISm2SLS.htm)
 для оценки коэффициентов.


Значение вероятности для J-статистики возвращает свойство [ISummaryStatistics.ProbJstat](ISummaryStatistics.ProbJstat.htm).
