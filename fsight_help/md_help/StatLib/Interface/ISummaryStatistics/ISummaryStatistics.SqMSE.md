# ISummaryStatistics.SqMSE


## Синтаксис


SqMSE: Double;


## Описание


Свойство SqMSE возвращает корень
 из среднего квадратов остатков.


## Комментарии


Среднее квадратов остатков - это корень из среднего значения квадратов
 отклонений исходных значений зависимой переменной от модельных.


Для получения среднего квадратов остатков используйте свойство [ISummaryStatistics.MSE](ISummaryStatistics.MSE.htm).
