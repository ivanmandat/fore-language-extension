# ISummaryStatistics.MSE


## Синтаксис


MSE: Double;


## Описание


Свойство MSE возвращает среднее
 квадратов остатков.


## Комментарии


Среднее квадратов остатков - это среднее квадратов отклонений исходных
 значений зависимой переменной от модельных.
