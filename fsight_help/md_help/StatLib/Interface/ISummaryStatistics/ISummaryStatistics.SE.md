# ISummaryStatistics.SE


## Синтаксис


SE: Double;


## Описание


Свойство SE возвращает стандартную ошибку.


## Комментарии


Под стандартной ошибкой понимается мера среднего рассеяния значений зависимой переменной вокруг модельных значений.
