# ISummaryStatistics.Fstat_2


## Синтаксис


Fstat_2: Double;


## Описание


Свойство Fstat_2 возвращает
 статистику Фишера, рассчитанную на основе нецентрированного коэффициента
 детерминации.


## Комментарии


Для получения вероятности статистики Фишера, рассчитанной на основе нецентрированного коэффициента детерминации,
 используйте свойство [ISummaryStatistics.ProbFstat_2](ISummaryStatistics.ProbFstat_2.htm).
