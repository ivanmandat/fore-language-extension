# ISummaryStatistics.SSR


## Синтаксис


SSR: Double;


## Описание


Свойство SSR возвращает сумму
 квадратов остатков отклонений исходных данных от модельных.


## Комментарии


Для получения стандартного отклонения остатков используйте свойство
 [ISummaryStatistics.SEE](ISummaryStatistics.SEE.htm).
