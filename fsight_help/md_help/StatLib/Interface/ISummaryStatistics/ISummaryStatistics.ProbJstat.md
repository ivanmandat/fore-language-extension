# ISummaryStatistics.ProbJstat


## Синтаксис


ProbJstat: Double;


## Описание


Свойство ProbJstat возвращает
 значение вероятности J-статистики.


## Комментарии


Значение J-статистики возвращает свойство [ISummaryStatistics.Jstat](ISummaryStatistics.Jstat.htm).
 Значение J-статистики рассчитывается для регрессионной модели, использующей
 [метод инструментальных переменных](../ISm2SLS/ISm2SLS.htm)
 для оценки коэффициентов.
