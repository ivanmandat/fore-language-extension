Свойство RegressionEffect определяет
 тип экзогенной переменной (регрессора).

Для определения значения экзогенной переменной используйте свойство
 [ISlTramoSeatsExogenousVar.Exogenous](ISlTramoSeatsExogenousVar.Exogenous.htm).
