# ISlTramoSeatsExogenousVar


Сборка: Stat;


## Описание


Интерфейс ISlTramoSeatsExogenousVar
 предназначен для работы с настройками экзогенной переменной (регрессора).


## Иерархия наследования


           ISlTramoSeatsExogenousVar


## Комментарии


Для получения коллекции экзогенных переменных (регрессоров) используйте
 свойство [ISmTramoSeats.Exogenous](../ISmTramoSeats/ISmTramoSeats.Exogenous.htm).
