# ISlTramoSeatsExogenousVar.Exogenous


## Синтаксис


Exogenous: Array;


## Описание


Свойство Exogenous определяет
 значение экзогенной переменной (регрессора).


## Комментарии


Индексация входных массивов должна начинаться с нуля.


Для определения типа экзогенной переменной используйте свойство [ISlTramoSeatsExogenousVar.RegressionEffect](ISlTramoSeatsExogenousVar.RegressionEffect.htm).
