# ICrossValidation.SamplingType


## Синтаксис


SamplingType: [CrossValidationSamplingType](../../Enums/CrossValidationSamplingType.htm);


## Описание


Свойство SamplingType
 определяет метод кросс-валидации.


## Комментарии


По умолчанию используется метод кросс-валидация по K
 блокам, то есть свойство имеет значение [CrossValidationSamplingType.Kfold](../../Enums/CrossValidationSamplingType.htm).
