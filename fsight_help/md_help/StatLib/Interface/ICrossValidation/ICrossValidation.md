# ICrossValidation


Сборка: Stat;


## Описание


Интерфейс ICrossValidation
 предназначен для настройки параметров кросс-валидации.


## Иерархия наследования


           ICrossValidation


## Комментарии


Для получения параметров кросс-валидации используйте свойство [IStatMethodValid.CrossValidation](../IStatMethodValid/IStatMethodValid.CrossValidation.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property.gif)
		 [NumberOfFolds](ICrossValidation.NumberOfFolds.htm)
		 Свойство NumberOfFolds
		 определяет количество блоков для кросс-валидации по K блокам.


		 ![](../../Property.gif)
		 [NumberOfRandomTests](ICrossValidation.NumberOfRandomTests.htm)
		 Свойство NumberOfRandomTests
		 определяет число повторений процедуры кросс-валидации для кросс-валидации
		 последовательным случайным разбиением.


		 ![](../../Property.gif)
		 [SamplingType](ICrossValidation.SamplingType.htm)


		 Свойство SamplingType
		 определяет метод кросс-валидации.


		 ![](../../Property.gif)
		 [TrainingSetSize](ICrossValidation.TrainingSetSize.htm)


		 Свойство TrainingSetSize
		 определяет размер обучающего набора в процентах для кросс-валидации
		 последовательным случайным разбиением.
