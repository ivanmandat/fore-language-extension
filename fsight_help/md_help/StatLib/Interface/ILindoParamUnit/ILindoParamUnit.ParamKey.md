# ILindoParamUnit.ParamKey


## Синтаксис


ParamKey: Integer;


## Описание


Свойство ParamKey определяет
 ключ параметра.


## Комментарии


Описание возможных параметров приведено в [документации
 LINDO](http://www.lindo.com/index.php?option=com_content&view=article&id=147&Itemid=69). Ключи параметров приведены в файле lindo.h,
 расположенном по пути S\include\,
 где S - это папка установки модуля
 LINDO.
