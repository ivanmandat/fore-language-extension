# ISmHyperGeometricDistribution.N


## Синтаксис


N: Integer;


## Описание


Свойство N определяет объем
 одной выборки.


## Комментарии


Значение N должно удовлетворять
 условию: 1 ≤ N ≤ N1. Где N1
 - объем всей совокупности, определяемый значением свойства [ISmHyperGeometricDistribution.N1](ISmHyperGeometricDistribution.N1.htm).
