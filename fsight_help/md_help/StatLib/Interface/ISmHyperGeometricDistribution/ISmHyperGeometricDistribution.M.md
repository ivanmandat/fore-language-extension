# ISmHyperGeometricDistribution.M


## Синтаксис


M: Integer;


## Описание


Свойство M определяет количество
 «успешных» элементов во всей совокупности.


## Комментарии


Значение M должно удовлетворять
 условию: 1 ≤ M ≤ N1. Где N1
 - объем всей совокупности, определяемый значением свойства [ISmHyperGeometricDistribution.N1](ISmHyperGeometricDistribution.N1.htm).
