# ISmEngleGrangerTest.ZStat


## Синтаксис


ZStat: [ISpecificationTestStatistic](../ISpecificationTestStatistic/ISpecificationTestStatistic.htm);


## Описание


Свойство ZStat возвращает значение
 z-статистики теста Энгла-Гранжера.


## Комментарии


Для получения значения τ-статистики
 теста Энгла-Гранжера используйте свойство [ISmEngleGrangerTest.TauStat](ISmEngleGrangerTest.TauStat.htm).
