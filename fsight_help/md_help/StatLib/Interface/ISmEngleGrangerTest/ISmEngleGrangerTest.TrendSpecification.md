# ISmEngleGrangerTest.TrendSpecification


## Синтаксис


TrendSpecification: [TrendSpecificationType](../../Enums/TrendSpecificationType.htm);


## Описание


Свойство TrendSpecification
 определяет спецификацию тренда.


## Комментарии


Свойство соответствует свойству [ISmCointegratingRegression.TrendSpecification](../ISmCointegratingRegression/ISmCointegratingRegression.TrendSpecification.htm).
 Для теста Энгла-Гранжера значимыми являются следующие [элементы
 перечисления](../../Enums/TrendSpecificationType.htm): NoneNone, ConstNone, LinearNone
 и QuadraticNone.
