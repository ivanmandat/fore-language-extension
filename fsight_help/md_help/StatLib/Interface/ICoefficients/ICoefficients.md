# ICoefficients


Сборка: Stat;


## Описание


Интерфейс ICoefficients возвращает
 параметры коэффициентов модели.


## Иерархия наследования


           ICoefficients


## Комментарии


Необходимо помнить, что порядок коэффициентов определяется порядком
 добавления в метод факторов, которым соответствуют данные коэффициенты.
 Для разных статистических методов данный механизм работает по-разному.
 Например, для линейной регрессии факторы добавляются в явном виде, а для
 метода ARIMA - это коэффициенты при авторегрессионных факторах и скользящем
 среднем.
