# ICoefficients.StandardError


## Синтаксис


StandardError: Array;


## Описание


Свойство StandardError возвращает массив значений стандартных ошибок коэффициентов.


## Комментарии


Под стандартной ошибкой понимается приближённая величина отклонения оценки коэффициента от истинного значения. Чем больше значение стандартной ошибки, тем менее достоверной является оценка коэффициента.
