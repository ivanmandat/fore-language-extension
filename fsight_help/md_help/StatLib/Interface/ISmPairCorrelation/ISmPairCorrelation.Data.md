# ISmPairCorrelation.Data


## Синтаксис


Data: Array;


## Описание


Свойство Data определяет матрицу
 исходных данных.


## Комментарии


Индексация входных массивов должна начинаться с нуля.


Результаты расчёта возвращает свойство [ISmPairCorrelation.PairCorrelationMatrix](ISmPairCorrelation.PairCorrelationMatrix.htm).
