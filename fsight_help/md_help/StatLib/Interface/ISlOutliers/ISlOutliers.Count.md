# ISlOutliers.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 элементов в коллекции.


## Комментарии


Для получения элемента из коллекции используйте свойство [ISlOutliers.Item](ISlOutliers.Item.htm).
