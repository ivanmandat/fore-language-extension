# ISlOutliers


Сборка: Stat;


## Описание


Интерфейс ISlOutliers предназначен
 для работы с коллекцией выбросов в данных.


## Иерархия наследования


           ISlOutliers


## Комментарии


Для определения, учитываются ли выбросы, используйте свойство [ISmx12arima.OutliersDetection](../ISmx12arima/ISmx12arima.OutliersDetection.htm).
