# ISlOutliers.Clear


## Синтаксис


Clear;


## Описание


Метод Clear удаляет все элементы
 коллекции.


## Комментарии


После вызова метода значение свойства [ISlOutliers.Count](ISlOutliers.Count.htm)
 будет обнулено.
