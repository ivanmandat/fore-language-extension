# ISmMedianSmoothing.Width


## Синтаксис


Width: Integer;


## Описание


Свойство Width определяет размер
 окна.


## Комментарии


Под размером окна понимается интервал времени, на котором будет определятся
 среднее значение элементов. Чем больше размер окна, тем более «гладкий»
 получается ряд. Размер окна должен быть не меньше двух и не больше длины
 сглаживаемого ряда.


По умолчанию свойство имеет значение «5».
