# ISmElliotRothenbergStockTest.Equation


## Синтаксис


Equation: [EquationType](../../Enums/EquationType.htm);


## Описание


Свойство Equation определяет
 тип модели, рассматриваемой в тесте.


## Комментарии


Тестовая модель может содержать константу или константу и тренд.


Для определения типа проверяемого ряда используйте свойство [ISmElliotRothenbergStockTest.TestedSeries](ISmElliotRothenbergStockTest.TestedSeries.htm).
