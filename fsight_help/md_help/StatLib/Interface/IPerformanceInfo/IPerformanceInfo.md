# IPerformanceInfo


Сборка: Stat;


## Описание


Интерфейс IPerformanceInfo используется
 для работы с информацией о выполнении методов оптимизации.


## Иерархия наследования


           IPerformanceInfo


## Комментарии


Для решения задачи оптимизации квадратичной функции нескольких переменных
 при линейных ограничениях используйте интерфейс [ISmQuadraticProgramming](../ISmQuadraticProgramming/ISmQuadraticProgramming.htm).

		 Имя свойства

		 [ConstraintsCalls](IPerformanceInfo.ConstraintsCalls.htm)

		 ![](../../Property.gif)
		 [ObjFunCalls](IPerformanceInfo.ObjFunCalls.htm)


		 Свойство ObjFunCalls

		 [ObjValByIter](IPerformanceInfo.ObjValByIter.htm)


		 Свойство ObjValByIter
		 возвращает значение целевой функции, рассчитанной на каждой итерации.
