# IPerformanceInfo.ObjValByIter


## Синтаксис


ObjValByIter: Array;


## Описание


Свойство ObjValByIter возвращает
 значение целевой функции, рассчитанной на каждой итерации.


## Комментарии


Для получения фактического числа итераций используйте свойство [IPerformanceInfo.Iterations](IPerformanceInfo.Iterations.htm).
