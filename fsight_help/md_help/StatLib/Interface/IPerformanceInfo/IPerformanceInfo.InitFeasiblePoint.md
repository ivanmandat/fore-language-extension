# IPerformanceInfo.InitFeasiblePoint


## Синтаксис


InitFeasiblePoint: Array;


## Описание


Свойство InitFeasiblePoint возвращает
 найденную начальную точку, которая удовлетворяет заданным ограничениям.


## Комментарии


Для получения фактического числа итераций используйте свойство [IPerformanceInfo.Iterations](IPerformanceInfo.Iterations.htm).
