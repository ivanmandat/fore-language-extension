# ISmAssociationRules.UseSupportPercent


## Синтаксис


UseSupportPercent: Boolean;


## Описание


Свойство UseSupportPercent определяет,
 в каких единицах измерения задано значение минимальной поддержки.


## Комментарии


Допустимые значения:


	- True. Значение минимальной
	 поддержки задано в процентах;


	- False. Значение минимальной
	 поддержки задано количеством элементов.


Значение минимальной поддержки определяет свойство [ISmAssociationRules.MinimumSupport](ISmAssociationRules.MinimumSupport.htm).
