# ISmAssociationRules.Objects


## Синтаксис


Objects: Array;


## Описание


Свойство Objects определяет
 числовой ряд, содержащий анализируемые элементы.


## Комментарии


Индексация входных массивов должна начинаться с нуля.


Ряд, содержащий транзакции, определяет свойство [ISmAssociationRules.Transactions](ISmAssociationRules.Transactions.htm).
