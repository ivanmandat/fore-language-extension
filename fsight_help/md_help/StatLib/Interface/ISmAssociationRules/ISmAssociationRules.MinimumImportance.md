# ISmAssociationRules.MinimumImportance


## Синтаксис


MinimumImportance: Double;


## Описание


Свойство MinimumImportance определяет
 минимальный уровень значимости.


## Комментарии


Минимальный уровень доверия определяет свойство [ISmAssociationRules.MinimumConfidence](ISmAssociationRules.MinimumConfidence.htm),
 минимальную поддержку - [ISmAssociationRules.MinimumSupport](ISmAssociationRules.MinimumSupport.htm).
