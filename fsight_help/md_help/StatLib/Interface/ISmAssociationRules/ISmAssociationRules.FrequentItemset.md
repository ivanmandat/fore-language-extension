# ISmAssociationRules.FrequentItemset


## Синтаксис


FrequentItemset(Index: Integer): [ISlFrequentItemset](../ISlFrequentItemset/ISlFrequentItemset.htm);


## Параметры


Index. Индекс набора;


## Описание


Свойство FrequentItemset возвращает
 указанный набор часто встречающихся элементов, полученных в результате
 расчёта метода.


## Комментарии


Индексация наборов сквозная и начинается с нуля. Количество наборов
 возвращает свойство [ISmAssociationRules.FrequentItemsetsCount](ISmAssociationRules.FrequentItemsetsCount.htm).
