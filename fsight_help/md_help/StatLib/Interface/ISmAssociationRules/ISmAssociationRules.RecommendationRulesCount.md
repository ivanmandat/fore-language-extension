# ISmAssociationRules.RecommendationRulesCount


## Синтаксис


RecommendationRulesCount: Integer;


## Описание


Свойство RecommendationRulesCount
 возвращает количество правил, полученных в результате расчёта метода.


## Комментарии


Подробную информацию о правиле можно получить с помощью свойства [ISmAssociationRules.RecommendationRule](ISmAssociationRules.RecommendationRule.htm).
