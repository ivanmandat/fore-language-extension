# ISmAssociationRules.FrequentItemsetsCount


## Синтаксис


FrequentItemsetsCount: Integer;


## Описание


Свойство FrequentItemsetsCount
 возвращает количество наборов часто встречающихся элементов,
 полученных в результате расчёта метода.


## Комментарии


Подробную информацию о наборе элементов можно получить с помощью свойства
 [ISmAssociationRules.FrequentItemset](ISmAssociationRules.FrequentItemset.htm).
