# ISmPartialCorrelation.PartialCorrelationMatrix


## Синтаксис


PartialCorrelationMatrix: Array;


## Описание


Свойство PartialCorrelationMatrix
 возвращает матрицу частных коэффициентов корреляции.


## Комментарии


Для получения матрицы парных коэффициентов корреляции используйте свойство
 [ISmPartialCorrelation.PairCorrelationMatrix](ISmPartialCorrelation.PairCorrelationMatrix.htm).
