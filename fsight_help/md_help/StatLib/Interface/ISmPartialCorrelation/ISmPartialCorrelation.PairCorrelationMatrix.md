# ISmPartialCorrelation.PairCorrelationMatrix


## Синтаксис


PairCorrelationMatrix: Array;


## Описание


Свойство PairCorrelationMatrix
 возвращает матрицу парных коэффициентов корреляции.


## Комментарии


Для получения матрицы частных коэффициентов корреляции используйте свойство
 [ISmPartialCorrelation.PartialCorrelationMatrix](ISmPartialCorrelation.PartialCorrelationMatrix.htm).
