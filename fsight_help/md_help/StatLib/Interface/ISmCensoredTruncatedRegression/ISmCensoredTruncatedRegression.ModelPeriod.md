# ISmCensoredTruncatedRegression.ModelPeriod


## Синтаксис


ModelPeriod: [IStatPeriod](../IStatPeriod/IStatPeriod.htm);


## Описание


Свойство ModelPeriod возвращает
 параметры периода идентификации метода.


## Комментарии


Если начальная и конечная точки периода идентификации имеют значение
 «-1», то они будут определены автоматически.
