# ISmCensoredTruncatedRegression.SigmaSE


## Синтаксис


SigmaSE: Double;


## Описание


Свойство SigmaSE возвращает
 значение стандартной ошибки сигма.


## Комментарии


Другие результаты расчёта сигма (значение, ошибка Z-статистики, вероятность
 ошибки Z-статистики) возвращают свойства [ISmCensoredTruncatedRegression.Sigma](ISmCensoredTruncatedRegression.Sigma.htm),
 [ISmCensoredTruncatedRegression.SigmaZStatErr](ISmCensoredTruncatedRegression.SigmaZStatErr.htm),
 [ISmCensoredTruncatedRegression.SigmaZStatProbErr](ISmCensoredTruncatedRegression.SigmaZStatProbErr.htm).
