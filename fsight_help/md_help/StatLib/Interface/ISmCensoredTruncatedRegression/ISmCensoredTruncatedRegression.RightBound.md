# ISmCensoredTruncatedRegression.RightBound


## Синтаксис


RightBound: [ISlSerie](../ISlSerie/ISlSerie.htm);


## Описание


Свойство RightBound возвращает
 правую (верхнюю) границу отсечения для наблюдений.


## Комментарии


Граница отсечения должна быть задана для каждого наблюдения. Левую (нижнюю)границу
 отсечения возвращает свойство [ISmCensoredTruncatedRegression.LeftBound](ISmCensoredTruncatedRegression.LeftBound.htm).
