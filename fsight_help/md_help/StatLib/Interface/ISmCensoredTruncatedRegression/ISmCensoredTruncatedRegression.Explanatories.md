# ISmCensoredTruncatedRegression.Explanatories


## Синтаксис


Explanatories: [ISlSeries](../ISlSeries/ISlSeries.htm);


## Описание


Свойство Explanatories возвращает
 коллекцию объясняющих рядов.


## Комментарии


Для корректного расчета метода должен быть задан минимум один объясняющий
 ряд.


Объясняемый ряд определяет свойство [ISmCensoredTruncatedRegression.Explained](ISmCensoredTruncatedRegression.Explained.htm).
