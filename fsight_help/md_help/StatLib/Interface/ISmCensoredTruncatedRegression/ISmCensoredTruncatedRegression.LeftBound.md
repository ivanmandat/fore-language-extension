# ISmCensoredTruncatedRegression.LeftBound


## Синтаксис


LeftBound: [ISlSerie](../ISlSerie/ISlSerie.htm);


## Описание


Свойство LeftBound возвращает
 левую (нижнюю) границу отсечения для наблюдений.


## Комментарии


Граница отсечения должна быть задана для каждого наблюдения. Правую
 (верхнюю) границу отсечения возвращает свойство [ISmCensoredTruncatedRegression.RightBound](ISmCensoredTruncatedRegression.RightBound.htm).
