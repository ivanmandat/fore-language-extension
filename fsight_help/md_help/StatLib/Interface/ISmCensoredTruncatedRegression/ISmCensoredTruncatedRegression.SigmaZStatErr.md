# ISmCensoredTruncatedRegression.SigmaZStatErr


## Синтаксис


SigmaZStatErr: Double;


## Описание


Свойство SigmaZStatErr возвращает
 значение ошибки Z-статистики сигма.


## Комментарии


Другие результаты расчёта сигма (значение, стандартная ошибка, вероятность
 ошибки Z-статистики) возвращают свойства [ISmCensoredTruncatedRegression.Sigma](ISmCensoredTruncatedRegression.Sigma.htm),
 [ISmCensoredTruncatedRegression.SigmaSE](ISmCensoredTruncatedRegression.SigmaSE.htm),
 [ISmCensoredTruncatedRegression.SigmaZStatProbErr](ISmCensoredTruncatedRegression.SigmaZStatProbErr.htm).
