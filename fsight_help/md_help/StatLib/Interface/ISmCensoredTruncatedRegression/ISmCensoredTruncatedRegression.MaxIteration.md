# ISmCensoredTruncatedRegression.MaxIteration


## Синтаксис


MaxIteration: Integer;


## Описание


Свойство MaxIteration определяет
 максимальное число итераций, за которое должен быть рассчитан метод.


## Комментарии


По умолчанию метод должен быть рассчитан за 500 итераций.
