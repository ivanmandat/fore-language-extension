# ISmFisherDistribution


Сборка: Stat;


## Описание


Интерфейс ISmFisherDistribution
 позволяет генерировать выборку псевдослучайных чисел из распределения
 Фишера с двумя заданными степенями свободы.


## Иерархия наследования

           ISmFisherDistribution


## Комментарии


Распределение Фишера также называется F-распределением.
