# ISmNonLinearOptimization.MaxIteration


## Синтаксис


MaxIteration: Integer;


## Описание


Свойство MaxIteration определяет
 максимальное число итераций, за которое должен осуществиться поиск решения.


## Комментарии


Значение по умолчанию «100».
