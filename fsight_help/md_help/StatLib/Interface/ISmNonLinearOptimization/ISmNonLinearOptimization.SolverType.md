# ISmNonLinearOptimization.SolverType


## Синтаксис


SolverType: [NLOSolverType](../../Enums/NLOSolverType.htm);


## Описание


Свойство SolverType определяет
 программное обеспечение, используемое для решения задачи нелинейной оптимизации.


## Комментарии


По умолчанию используется «Форсайт. Аналитическая платформа».
 Если для решения задачи выбран модуль LINDO, то для задания его параметров
 используйте свойство [ISmNonLinearOptimization.LindoSettings](ISmNonLinearOptimization.LindoSettings.htm).


Примечание.
 Для использования модуля LINDO его необходимо предварительно установить.
 Получить подробную информацию о модуле LINDO и его условиях его использования
 можно на сайте [www.lindo.com](http://www.lindo.com/).
