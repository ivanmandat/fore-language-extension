# ISmNonLinearOptimization.NonLinearConstraints


## Синтаксис


NonLinearConstraints: [INonLinearConstraints](../INonLinearConstraints/INonLinearConstraints.htm);


## Описание


Свойство NonLinearConstraints
 определяет параметры нелинейных ограничений.


## Комментарии


Для задания линейных ограничений используйте свойство [ISmNonLinearOptimization.LinearConstraints](ISmNonLinearOptimization.LinearConstraints.htm).
