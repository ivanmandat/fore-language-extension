# ISlTramoSeatsOutlier


Сборка: Stat;


## Описание


Интерфейс ISlTramoSeatsOutlier
 предназначен для работы с выбросами TRAMO-SEATS.


## Иерархия наследования


           ISlTramoSeatsOutlier


## Комментарии


Для определения, учитываются ли выбросы, используйте свойство [ISmTramoSeats.OutliersDetection](../ISmTramoSeats/ISmTramoSeats.OutliersDetection.htm).
