# ISmAutoCorrelation.AutoCorrelationFunction


## Синтаксис


AutoCorrelationFunction: Array;


## Описание


Свойство AutoCorrelationFunction
 возвращает автокорреляционную функцию.


## Комментарии


Для получения частной автокорреляционной функции используйте свойство
 [ISmAutoCorrelation.PartialAutoCorrelationFunction](ISmAutoCorrelation.PartialAutoCorrelationFunction.htm).

Использование свойства приведено в примере для [ISmAutoCorrelation.MissingData](ISmAutoCorrelation.MissingData.htm).
