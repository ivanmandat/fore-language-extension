## Синтаксис


QStatistics: Array;


## Описание


Свойство QStatistics возвращает
 Q-статистику Льюнг-Бокса.


## Комментарии


Q-статистика Льюнг-Бокса - это статистический критерий для выявления
 автокоррелированности временных рядов.
