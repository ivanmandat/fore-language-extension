# ISmAutoCorrelation.ModelPeriod


## Синтаксис


ModelPeriod: [IStatPeriod](../IStatPeriod/IStatPeriod.htm);


## Описание


Свойство ModelPeriod определяет
 параметры периода идентификации.


## Комментарии


Для определения начальной и конечной точек периода идентификации используйте
 свойства [IStatPeriod.FirstPoint](../IStatPeriod/IStatPeriod.FirstPoint.htm)
 и [IStatPeriod.LastPoint](../IStatPeriod/IStatPeriod.LastPoint.htm).
