# ISmAutoCorrelation.PartialAutoCorrelationFunction


## Синтаксис


PartialAutoCorrelationFunction: Array;


## Описание


Свойство PartialAutoCorrelationFunction
 возвращает частную автокорреляционную функцию.


## Комментарии


Для получения автокорреляционной функции используйте свойство [ISmAutoCorrelation.AutoCorrelationFunction](ISmAutoCorrelation.AutoCorrelationFunction.htm).
