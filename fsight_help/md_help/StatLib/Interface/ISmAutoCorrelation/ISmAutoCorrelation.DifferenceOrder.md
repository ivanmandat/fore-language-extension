# ISmAutoCorrelation.DifferenceOrder


## Синтаксис


DifferenceOrder: Integer;


## Описание


Свойство DifferenceOrder определяет
 порядок разности, который определяет, сколько раз будет продифференцирован
 исходный временной ряд.


## Комментарии


Допустимые значения берутся из диапазона [0, 5]. В случае, если данный
 параметр равен нулю, то ряд не дифференцируется и рассматривается исходный.
