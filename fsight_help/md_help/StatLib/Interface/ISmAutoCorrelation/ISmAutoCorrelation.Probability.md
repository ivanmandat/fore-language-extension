## Описание


Свойство Probability возвращает
 вероятность для [Q-статистики
 Льюнг-Бокса](ISmAutoCorrelation.QStatistics.htm).

Q-статистика Льюнг-Бокса - это статистический критерий для выявления
 автокоррелированности временных рядов.
