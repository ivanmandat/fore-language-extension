# ISmLongRunCovariance.RemoveMeans


## Синтаксис


RemoveMeans: Boolean;


## Описание


Свойство RemoveMeans определяет,
 выполнять ли стандартизацию данных.


## Комментарии


Допустимые значения:


	- True. Стандартизация
	 данных выполняется;


	- False. Стандартизация
	 данных не выполняется.
