# ISmLongRunCovariance.DFAdjustment


## Синтаксис


DFAdjustment: Boolean;


## Описание


Свойство DFAdjustment определяет,
 учитывать ли количество степеней свободы.


## Комментарии


Допустимые значения:


	- True. Количество степеней
	 свободы будет учитываться;


	- False. Количество степеней
	 свободы не будет учитываться.
