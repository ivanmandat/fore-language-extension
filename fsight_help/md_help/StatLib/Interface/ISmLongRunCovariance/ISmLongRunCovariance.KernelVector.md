## Синтаксис


KernelVector: Array;


## Описание


Свойство KernelVector определяет
 вектор ядра.


## Комментарии


Свойство актуально, если [ISmLongRunCovariance.LRCKernel](ISmLongRunCovariance.LRCKernel.htm) = LRCKernelType.UserSpecified.


Первый элемент вектора ядра должен быть равен единице.
