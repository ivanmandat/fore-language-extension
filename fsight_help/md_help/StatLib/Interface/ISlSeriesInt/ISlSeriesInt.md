# ISlSeriesInt


Сборка: Stat;


## Описание


Интерфейс ISlSeriesInt предназначен
 для работы с параметрами коллекции категориальных рядов данных.


## Иерархия наследования


           ISlSeriesInt


## Комментарии


Категориальный ряд - набор данных, элементы которого нельзя упорядочить,
 и между которыми нельзя найти расстояние. Например, что обычно едят на
 завтрак люди из выборки: бутерброд, каша, яичница и т.д. Ни упорядочить,
 ни найти расстояние между элементами массива нельзя.


Для работы с параметрами категориального ряда данных используйте свойства
 интерфейса [ISlSerieInt](../ISlSerieInt/ISlSerieInt.htm).
