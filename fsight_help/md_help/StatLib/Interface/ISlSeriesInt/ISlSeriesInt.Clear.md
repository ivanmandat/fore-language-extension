# ISlSeriesInt.Clear


## Синтаксис


Clear;


## Описание


Метод Clear очищает коллекцию
 категориальных рядов данных.


## Комментарии


Для удаления из коллекции категориальный ряд данных по его индексу
 используйте метод [ISlSeriesInt.Remove](ISlSeriesInt.Remove.htm).
