# ISlSeriesInt.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс категориального
 ряда данных в коллекции.


## Описание


Метод Remove удаляет из коллекции
 категориальный ряд данных по его индексу.


## Комментарии


Для получения параметров категориального ряда данных по его индексу
 используйте свойство [ISlSeriesInt.Item](ISlSeriesInt.Item.htm).
