# ISlSeriesInt.Item


## Синтаксис


Item(Index: Integer): [ISlSerieInt](../ISlSerieInt/ISlSerieInt.htm);


## Параметры


Index. Индекс категориального
 ряда данных в коллекции.


## Описание


Свойство Item возвращает параметры
 категориального ряда данных по его индексу.


## Комментарии


Для получения количества категориальных рядов данных в коллекции используйте
 свойство [ISlSeriesInt.Count](ISlSeriesInt.Count.htm).


## Пример


Использование свойства приведено в примере для [ISmRandomForest.Forest](../ISmRandomForest/ISmRandomForest.Forest.htm).
