# ISlSeriesInt.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 категориальных рядов данных в коллекции.


## Комментарии


Для добавления в коллекцию категориального ряда данных используйте метод
 [ISlSeriesInt.Add](ISlSeriesInt.Add.htm).
