# IUnitRootTestStatistic


Сборка: Stat;


## Описание


Интерфейс IUnitRootTestStatistic
 предназначен для работы с тестовыми статистиками для тестов на единичные
 корни.


## Иерархия наследования


           IUnitRootTestStatistic


## Комментарии


Для выполнения тестов на единичные корни используйте следующие интерфейсы:


	- [ISmAugmentDickeyFullerTest](../ISmAugmentDickeyFullerTest/ISmAugmentDickeyFullerTest.htm).
	 Расширенный тест Дики-Фуллера;


	- [ISmDickeyFullerGLSTest](../ISmDickeyFullerGLSTest/ISmDickeyFullerGLSTest.htm).
	 Обобщенный тест Дики-Фуллера;


	- [ISmElliotRothenbergStockTest](../ISmElliotRothenbergStockTest/ISmElliotRothenbergStockTest.htm).
	 Тест Эллиота-Розенберга-Стока;


	- [ISmKwiatkowskiPhillipsSchmidtShinTest](../ISmKwiatkowskiPhillipsSchmidtShinTest/ISmKwiatkowskiPhillipsSchmidtShinTest.htm).
	 Тест Квятковского-Филлипса-Шмидта-Шина;


	- [ISmNgPerronTest](../ISmNgPerronTest/ISmNgPerronTest.htm).
	 Тест Нг-Перрона;


	- [ISmPhillipsPerronTest](../ISmPhillipsPerronTest/ISmPhillipsPerronTest.htm).
	 Тест Филлипса-Перрона.
