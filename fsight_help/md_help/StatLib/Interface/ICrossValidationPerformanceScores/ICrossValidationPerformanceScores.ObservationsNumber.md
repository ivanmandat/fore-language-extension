# ICrossValidationPerformanceScores.ObservationsNumber


## Синтаксис


ObservationsNumber: Integer;


## Описание


Свойство ObservationsNumber
 возвращает число объектов исходной выборки, участвующих в кросс-валидации.

Для получения корректной классификации используйте свойство [ICrossValidationPerformanceScores.PerformanceMatrix](ICrossValidationPerformanceScores.PerformanceMatrix.htm).
