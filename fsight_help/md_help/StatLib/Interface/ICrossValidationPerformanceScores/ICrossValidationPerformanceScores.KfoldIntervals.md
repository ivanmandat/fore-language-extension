# ICrossValidationPerformanceScores.KfoldIntervals


## Синтаксис


KfoldIntervals: Array;


## Описание


Свойство KfoldIntervals
 возвращает размер интервала для каждого блока.


## Комментарии


Свойство возвращает корректное значение, если используется кросс-валидация
 по K блокам, то есть свойство
 [ICrossValidation.SamplingType](../ICrossValidation/ICrossValidation.SamplingType.htm)
 имеет значение [CrossValidationSamplingType.Kfold](../../Enums/CrossValidationSamplingType.htm).
