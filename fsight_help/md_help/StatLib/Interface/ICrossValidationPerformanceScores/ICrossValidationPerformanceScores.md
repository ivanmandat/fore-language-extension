# ICrossValidationPerformanceScores


Сборка: Stat;


## Описание


Интерфейс ICrossValidationPerformanceScores предназначен
 для работы с результатами кросс-валидации.


## Иерархия наследования


           ICrossValidationPerformanceScores


## Комментарии


Для получения результатов кросс валидации используйте свойство [IStatMethodValid.PerformanceScores](../IStatMethodValid/IStatMethodValid.PerformanceScores.htm).
