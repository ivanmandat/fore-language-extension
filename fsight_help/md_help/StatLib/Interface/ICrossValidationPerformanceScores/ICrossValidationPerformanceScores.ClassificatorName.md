# ICrossValidationPerformanceScores.ClassificatorName


## Синтаксис


ClassificatorName: String;


## Описание


Свойство ClassificatorName
 возвращает наименование анализируемого признака.


## Комментарии


Для получения количества факторов, влияющих на анализируемый признак
 используйте свойство [ICrossValidationPerformanceScores.FactorsNumber](ICrossValidationPerformanceScores.FactorsNumber.htm).
