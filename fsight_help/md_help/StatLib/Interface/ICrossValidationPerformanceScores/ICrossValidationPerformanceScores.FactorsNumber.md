# ICrossValidationPerformanceScores.FactorsNumber


## Синтаксис


FactorsNumber: Integer;


## Описание


Свойство FactorsNumber
 возвращает количество факторов, влияющих на анализируемый признак.


## Комментарии


Для получения наименования анализируемого признака используйте свойство
 [ICrossValidationPerformanceScores.ClassificatorName](ICrossValidationPerformanceScores.ClassificatorName.htm).
