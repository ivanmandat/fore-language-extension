# ISmDerivative.DerivativeValues


## Синтаксис


DerivativeValues: Array;


## Описание


Свойство DerivativeValues возвращает
 массив значений производных.


## Комментарии


Количество элементов в массиве соответствует количеству переменных дифференцирования,
 заданных в свойстве [ISmDerivative.DiffVariables](ISmDerivative.DiffVariables.htm).
