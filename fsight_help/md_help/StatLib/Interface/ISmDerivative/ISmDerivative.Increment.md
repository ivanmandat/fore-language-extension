# ISmDerivative.Increment


## Синтаксис


Increment: Double;


## Описание


Свойство Increment определяет
 приращение аргумента для расчёта приближенных значений частных производных.


## Комментарии


Значение свойства учитывается, если [ISmDerivative.UseAnalyticCalc](ISmDerivative.UseAnalyticCalc.htm) = False.


Значение по умолчанию: 0.00000001 (1.0E-8).
