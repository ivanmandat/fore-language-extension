# ISmDerivative.UseAnalyticCalc


## Синтаксис


UseAnalyticCalc: Boolean;


## Описание


Свойство UseAnalyticCalc определяет,
 рассчитывать ли аналитические производные выражений.


## Комментарии


Допустимые значения:


	- True. Рассчитываются
	 аналитические производные выражений и их значения в заданной точке
	 (если точка задана);


	- False. Рассчитываются
	 только приближенные значения частных производных в заданной точке.
