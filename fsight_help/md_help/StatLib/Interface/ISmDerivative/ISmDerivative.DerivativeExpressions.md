# ISmDerivative.DerivativeExpressions


## Синтаксис


DerivativeExpressions: Array;


## Описание


Свойство DerivativeExpressions
 возвращает массив выражений производных.


## Комментарии


Каждый элемент массива является строкой. Количество элементов соответствует
 количеству переменных дифференцирования, заданных в свойстве [ISmDerivative.DiffVariables](ISmDerivative.DiffVariables.htm).
