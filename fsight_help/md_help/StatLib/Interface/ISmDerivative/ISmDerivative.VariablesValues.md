# ISmDerivative.VariablesValues


## Синтаксис


VariablesValues: Array;


## Описание


Свойство VariablesValues определяет
 значения аргументов в точке, в которой будет рассчитана производная.


## Комментарии


Количество элементов массива должно совпадать с количеством переменных,
 указанных в свойстве [ISmDerivative.CoefficientsOrder](ISmDerivative.CoefficientsOrder.htm).
