# IDataMining.FilledDependent


## Синтаксис


FilledDependent: [ISlSerie](../ISlSerie/ISlSerie.htm);


## Описание


Свойство FilledDependent
 возвращает ряд с результатами расчета.


## Комментарии


Для получения объясняемого ряда используйте свойство [IDataMining.Dependent](IDataMining.Dependent.htm).
