# ISmHodrickPrescottFilter.Frequency


## Синтаксис


Frequency: Integer;


## Описание


Свойство Frequency определяет
 частоту данных: количество периодов в году.


## Комментарии


Значение Frequency должно быть
 положительным.


В таблице приведено значение Frequency
 для различных данных:


		 Частота данных
		 Значение Frequency


		 Годовые данные
		 1


		 Квартальные данные
		 4


		 Месячные данные
		 12


		 Недельные данные
		 53


		 Суточные данные
		 365
