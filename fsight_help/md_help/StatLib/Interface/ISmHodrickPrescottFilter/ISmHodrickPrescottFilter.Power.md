# ISmHodrickPrescottFilter.Power


## Синтаксис


Power: Double;


## Описание


Свойство Power определяет значение
 степени, применяемой в качестве параметра сглаживания.


## Комментарии


Power должно быть неотрицательным
 числом, минимальное значение - ноль, значение по умолчанию - четыре.


Power учитывается, если [ISmHodrickPrescottFilter.SmoothingParameterMode](ISmHodrickPrescottFilter.SmoothingParameterMode.htm)
 имеет значение HPSmoothingParameterModeType.SetByFrequencyRule.
