# ISlInfluence.ExplanatoryId


## Синтаксис


ExplanatoryId: Integer;


## Описание


Свойство ExplanatoryId возвращает
 номер объясняющего ряда.


## Комментарии


Значение объясняющего ряда, которое оказывает влияние на значение объясняемого
 ряда, возвращает свойство [ISlInfluence.ExplanatoryValue](ISlInfluence.ExplanatoryValue.htm).


## Пример


Использование свойства приведено в примере для [ISmNaiveBayes.KeyInfluence](../ISmNaiveBayes/ISmNaiveBayes.KeyInfluence.htm).
