# ISlInfluence.ExplanatoryValue


## Синтаксис


ExplanatoryValue: Integer;


## Описание


Свойство ExplanatoryValue возвращает
 значение объясняющего ряда, которое оказывает влияние.


## Комментарии


Номер объясняющего ряда, которому принадлежит возвращаемое значение,
 возвращает свойство [ISlInfluence.ExplanatoryId](ISlInfluence.ExplanatoryId.htm).
 Значение объясняемого ряда, на которое оказывается влияние, возвращает
 свойство [ISlInfluence.ExplainedValue](ISlInfluence.ExplainedValue.htm).
