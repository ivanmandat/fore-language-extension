# ISlInfluence.ExplainedValue


## Синтаксис


ExplainedValue: Integer;


## Описание


Свойство ExplainedValue возвращает
 значение объясняемого ряда, на которое оказывается влияние.


## Комментарии


Значение объясняющего ряда, которое оказывает влияние, возвращает свойство
 [ISlInfluence.ExplanatoryValue](ISlInfluence.ExplanatoryValue.htm).
