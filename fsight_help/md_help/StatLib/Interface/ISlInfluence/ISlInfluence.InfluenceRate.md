# ISlInfluence.InfluenceRate


## Синтаксис


InfluenceRate: Double;


## Описание


Свойство InfluenceRate возвращает
 степень влияния значения объясняющего ряда на значение объясняемого ряда.


## Комментарии


Свойство возвращает значение в диапазоне [0, 100]. Значение объясняющего
 ряда, оказывающее влияние, возвращает свойство [ISlInfluence.ExplanatoryValue](ISlInfluence.ExplanatoryValue.htm).
 Значение объясняемого ряда, на которое оказывается влияние, возвращает
 свойство [ISlInfluence.ExplainedValue](ISlInfluence.ExplainedValue.htm).
