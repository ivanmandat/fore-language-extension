# ISlInfluence


Сборка: Stat;


## Описание


Интерфейс ISlInfluence используется
 для работы с характеристиками влияния фактора на зависимую переменную.


## Иерархия наследования


           ISlInfluence


## Комментарии


Влияние факторов на зависимую переменную вычисляется с помощью наивного
 классификатора Байеса, реализованного интерфейсом [ISmNaiveBayes](../ISmNaiveBayes/ISmNaiveBayes.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property.gif)
		 [ExplainedValue](ISlInfluence.ExplainedValue.htm)


		 Свойство ExplainedValue
		 возвращает значение объясняемого ряда, на которое происходит влияние.


		 ![](../../Property.gif)
		 [ExplanatoryId](ISlInfluence.ExplanatoryId.htm)


		 Свойство ExplanatoryId
		 возвращает номер объясняющего ряда.


		 ![](../../Property.gif)
		 [ExplanatoryValue](ISlInfluence.ExplanatoryValue.htm)


		 Свойство ExplanatoryValue
		 возвращает значение объясняющего ряда, которое оказывает влияние.


		 ![](../../Property.gif)
		 [InfluenceRate](ISlInfluence.InfluenceRate.htm)


		 Свойство InfluenceRate
		 возвращает степень влияния значения объясняющего ряда на
		 значение объясняемого ряда.
