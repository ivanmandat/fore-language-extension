# IBinningSettingsInt


Сборка: Stat;


## Описание


Интерфейс IBinningSettingsInt
 предназначен для работы с параметрами процедуры Binning, выполняющейся
 для категориальных рядов.


## Иерархия наследования


           [IBinningSettings](../IBinningSettings/IBinningSettings.htm)


           IBinningSettingsInt


## Комментарии


Для получения параметров процедуры Binning используйте свойство [ISlSerieInt.BinningSettings](../ISlSerieInt/ISlSerieInt.BinningSettings.htm).
