# IBinningSettingsInt.IntegerValue


## Синтаксис


IntegerValue: Array;


## Описание


Свойство IntegerValue возвращает
 результат выполнения процедуры Binning.


## Комментарии


Данное свойство используется, если процедура Binning применяется для
 объясняемого ряда при расчете [градиентного
 бустинга](../ISmGradientBoostedTree/ISmGradientBoostedTree.Dependent.htm), ансамбля деревьев решений [Random
 Forest](../ISmRandomForest/ISmRandomForest.Dependent.htm) или бинарных деревьев [ISmCART](../ISmCART/ISmCART.Dependent.htm).
