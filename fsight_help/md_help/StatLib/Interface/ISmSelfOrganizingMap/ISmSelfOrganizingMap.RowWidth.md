# ISmSelfOrganizingMap.RowWidth


## Синтаксис


RowWidth: Integer;


## Описание


Свойство RowWidth определяет
 ширину сети.


## Комментарии


Ширина сети должна быть больше нуля и не больше числа кластеров.
