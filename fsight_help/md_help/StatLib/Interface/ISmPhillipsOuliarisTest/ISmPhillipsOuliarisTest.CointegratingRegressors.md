## Описание


Свойство CointegratingRegressors
 возвращает коинтегрирующие регрессоры.


## Комментарии


Для получения детерминированных регрессоров используйте свойство [ISmPhillipsOuliarisTest.DeterministicRegressors](ISmPhillipsOuliarisTest.DeterministicRegressors.htm).
