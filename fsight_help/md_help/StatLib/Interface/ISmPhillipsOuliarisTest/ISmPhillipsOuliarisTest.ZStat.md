# ISmPhillipsOuliarisTest.ZStat


## Синтаксис

## Описание


Свойство ZStat возвращает значение
 z-статистики теста Филлипса-Оуляриса.


## Комментарии


Для получения значения τ-статистики
 теста Филлипса-Оуляриса используйте свойство [ISmPhillipsOuliarisTest.TauStat](ISmPhillipsOuliarisTest.TauStat.htm).
