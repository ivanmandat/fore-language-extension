# ISmPhillipsOuliarisTest.TauStat


## Синтаксис

## Описание


Свойство TauStat возвращает
 значение τ-статистики теста Филлипса-Оуляриса.


## Комментарии


Для получения значения z-статистики
 теста Филлипса-Оуляриса используйте свойство [ISmPhillipsOuliarisTest.ZStat](ISmPhillipsOuliarisTest.ZStat.htm).
