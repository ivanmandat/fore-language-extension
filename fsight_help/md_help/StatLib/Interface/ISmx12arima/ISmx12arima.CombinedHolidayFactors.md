# ISmx12arima.CombinedHolidayFactors


## Синтаксис


CombinedHolidayFactors: Array;


## Описание


Свойство CombinedHolidayFactors
 возвращает ряд, содержащий комбинацию поправок на праздничные и рабочие
 дни.


## Комментарии


Для определения способа учета поправок на праздничные и рабочие дни
 используйте свойство [ISmx12arima.AdjustmentOptions](ISmx12arima.AdjustmentOptions.htm).
