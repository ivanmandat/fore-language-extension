# ISmx12arima.ResidualsDiagnostic


## Синтаксис


ResidualsDiagnostic: Boolean;


## Описание


Свойство ResidualsDiagnostic
 определяет, проводить ли диагностику остатков.


## Комментарии


Допустимые значения:


	- True. Диагностика остатков
	 проводится;


	- False. Диагностика остатков
	 не проводится.
