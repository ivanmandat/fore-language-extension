# ISmx12arima.PredefinedVariableConst


## Синтаксис


PredefinedVariableConst: Boolean;


## Описание


Свойство PredefinedVariableConst
 определяет, добавлять ли в регрессоры константу при построении модели
 ARIMA.


## Комментарии


Допустимые значения:


	- True. Константа добавляется
	 в регрессоры;


	- False. Константа не
	 добавляется в регрессоры.
