# ISmx12arima.Output


## Синтаксис


Output: Array;


## Описание


Свойство Output определяет массив
 строк, содержащий выходной файл, формируемый внешней программой Х12.


## Комментарии


Для определения массива строк, содержащих сообщения об ошибках и предупреждениях,
 возникших при расчете, используйте свойство [ISmx12arima.ErrorFile](ISmx12arima.ErrorFile.htm).
