# ISmx12arima.IrregularComponent


## Синтаксис


IrregularComponent: Array;


## Описание


Свойство IrregularComponent
 возвращает ряд, содержащий нерегулярную компоненту.


## Комментарии


Для получения ряда, содержащего комбинацию сезонной составляющей и поправки
 на рабочие дни, используйте свойство [ISmx12arima.CombinedSeasonalFactors](ISmx12arima.CombinedSeasonalFactors.htm).
