# ISmx12arima.OrderDiff


## Синтаксис


OrderDiff: Integer;


## Описание


Свойство OrderDiff определяет
 порядок несезонного дифференцирования.


## Комментарии


Для определения порядка сезонного дифференцирования используйте свойство
 [ISmx12arima.OrderDiffSeas](ISmx12arima.OrderDiffSeas.htm).
