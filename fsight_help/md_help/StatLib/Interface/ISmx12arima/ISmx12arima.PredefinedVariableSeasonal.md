# ISmx12arima.PredefinedVariableSeasonal


## Синтаксис


PredefinedVariableSeasonal: Boolean;


## Описание


Свойство PredefinedVariableSeasonal
 определяет, добавлять ли в регрессоры сезонные фиктивные переменные при
 построении модели ARIMA.


## Комментарии


Допустимые значения:


	- True. Сезонные фиктивные
	 переменные добавляются в регрессоры;


	- False. Сезонные фиктивные
	 переменные не добавляются в регрессоры.
