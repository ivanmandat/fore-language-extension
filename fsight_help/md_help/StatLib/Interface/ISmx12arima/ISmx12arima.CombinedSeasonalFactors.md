# ISmx12arima.CombinedSeasonalFactors


## Синтаксис


CombinedSeasonalFactors: Array;


## Описание


Свойство CombinedSeasonalFactors
 возвращает ряд, содержащий комбинацию сезонной составляющей и поправки
 на рабочие дни.


## Комментарии


Для определения типа и периода сезонности используйте свойства [ISmx12arima.SeasonalAdjustmentMode](ISmx12arima.SeasonalAdjustmentMode.htm)
 и [ISmx12arima.SeasonalComponentCycleType](ISmx12arima.SeasonalComponentCycleType.htm).
