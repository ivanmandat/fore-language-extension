# ISmx12arima.BoxCoxPowerTransform


## Синтаксис


BoxCoxPowerTransform: Integer;


## Описание


Свойство BoxCoxPowerTransform
 определяет значение степени для трансформации Бокса-Кокса.


## Комментарии


Свойство актуально, если [ISmx12arima.DataTransformation](ISmx12arima.DataTransformation.htm) = DataTransformationX12Type.BoxCoxPower.


Значение по умолчанию - 0.
