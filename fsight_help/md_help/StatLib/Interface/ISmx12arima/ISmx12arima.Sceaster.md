# ISmx12arima.Sceaster


## Синтаксис


Sceaster: [IHolidays](../IHolidays/IHolidays.htm);


## Описание


Свойство Sceaster возвращает
 параметры поправок на праздник Пасхи (Канада).


## Комментарии


Для определения способа учета поправок на праздничные и рабочие дни
 используйте свойство [ISmx12arima.AdjustmentOptions](ISmx12arima.AdjustmentOptions.htm).
