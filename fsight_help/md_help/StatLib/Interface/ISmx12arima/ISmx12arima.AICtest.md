# ISmx12arima.AICtest


## Синтаксис


AICtest: Boolean;


## Описание


Свойство AICtest определяет,
 включен ли автоматический режим учета поправок на праздничные и рабочие
 дни.


## Комментарии


При включении автоматического режима различные варианты поправок сравниваются
 между собой по значению информационного критерия [Акаике](Lib.chm::/05_Statistics/Information_Criteria.htm).


Допустимые значения:


	- True. Автоматический
	 режим учета поправок на праздничные и рабочие дни включен;


	- False. Автоматический
	 режим учета поправок на праздничные и рабочие дни отключен.
