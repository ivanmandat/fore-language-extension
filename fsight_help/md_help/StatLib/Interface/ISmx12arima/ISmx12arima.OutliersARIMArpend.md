# ISmx12arima.OutliersARIMArpend


## Синтаксис


OutliersARIMArpend: [ISlOutliers](../ISlOutliers/ISlOutliers.htm);


## Описание


Свойство OutliersARIMArpend возвращает
 список выбросов типа «постепенная смена уровня» (дата окончания), учитываемых
 на шаге ARIMA.


## Комментарии


Для данного типа выброса должна быть задана дата начала с помощью свойства
 [ISmx12arima.OutliersARIMArpbegin](ISmx12arima.OutliersARIMArpbegin.htm).
