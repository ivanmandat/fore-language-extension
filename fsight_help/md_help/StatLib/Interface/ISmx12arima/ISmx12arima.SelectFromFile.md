# ISmx12arima.SelectFromFile


## Синтаксис


SelectFromFile: Boolean;


## Описание


Свойство SelectFromFile определяет,
 брать ли спецификацию параметров ARIMA из файла.


## Комментарии


Файл задается в виде строкового массива с помощью свойства [ISmx12arima.x12aMdlFile](ISmx12arima.x12aMdlFile.htm).


Допустимые значения:


	- True. Спецификация параметров
	 ARIMA берется из файла;


	- False. Спецификация
	 параметров ARIMA не берется из файла.
