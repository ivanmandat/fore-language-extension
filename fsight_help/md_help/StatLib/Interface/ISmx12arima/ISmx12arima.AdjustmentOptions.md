# ISmx12arima.AdjustmentOptions


## Синтаксис


AdjustmentOptions: [AdjustmentOptionsX12Type](../../Enums/AdjustmentOptionsX12Type.htm);


## Описание


Свойство AdjustmentOptions определяет
 способ учета поправок на праздничные и рабочие дни.


## Комментарии


По умолчанию учет поправок на праздничные и рабочие дни не производится.
