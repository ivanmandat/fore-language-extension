# ISmx12arima.SeasonalComponentCycleType


## Синтаксис

## Описание


Свойство SeasonalComponentCycleType
 определяет период сезонности (месяцы/кварталы).


## Комментарии


Для определения типа сезонности используйте свойство [ISmx12arima.SeasonalAdjustmentMode](ISmx12arima.SeasonalAdjustmentMode.htm).
