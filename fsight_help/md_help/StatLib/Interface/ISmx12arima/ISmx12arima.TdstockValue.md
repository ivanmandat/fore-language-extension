# ISmx12arima.TdstockValue


## Синтаксис


TdstockValue: Integer;


## Описание


Свойство TdstockValue определяет
 количество дней, рассматриваемых для получения поправки на рабочие дни.


## Комментарии


Свойство актуально, если [ISmx12arima.TradingDayEffects](ISmx12arima.TradingDayEffects.htm)
 = TradingDayEffectsX12Type.TdStock.
