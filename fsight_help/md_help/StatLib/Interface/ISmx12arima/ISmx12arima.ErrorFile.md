# ISmx12arima.ErrorFile


## Синтаксис


ErrorFile: Array;


## Описание


Свойство ErrorFile определяет
 массив строк файла с ошибками, приходящими из внешней программы.


## Комментарии


Для определения массива строк, содержащего выходной файл, используйте
 свойство [ISmx12arima.Output](ISmx12arima.Output.htm).
