# ISmx12arima.SP1frequency


## Синтаксис


SP1frequency: Array;


## Описание


Свойство SP1frequency возвращает
 ряд спектральных частот SP1.


## Комментарии


Для определения, строить ли графики спектральных плотностей и частот,
 используйте свойство [ISmx12arima.SpectralPlots](ISmx12arima.SpectralPlots.htm).
