# ISmx12arima.MissingData


## Синтаксис


MissingData: [IMissingData](../IMissingData/IMissingData.htm);


## Описание


Свойство MissingData возвращает
 параметры обработки пропусков в исходных данных.


## Комментарии


По умолчанию используется метод [MissingDataMethod.None](../../Enums/MissingDataMethod.htm),
 т.е. обработка пропусков не выполняется.


Метод обработки пропусков [MissingDataMethod.Casewise](../../Enums/MissingDataMethod.htm)
 не может быть использован из-за смещения данных.
