# ISmx12arima.Trendma


## Синтаксис


Trendma: Integer;


## Описание


Свойство Trendma определяет
 порядок скользящего среднего для трендового фильтра (фильтра Хендерсона).


## Комментарии


Значение параметра «скользящее среднее Хендерсона» должно быть нечетным
 числом больше 1 и меньше 102. Значение по умолчанию 0, означает, что фильтрация
 не применяется.
