## Описание


Свойство SeasonallyAdjusted
 возвращает ряд, скорректированный на сезонную компоненту.


## Комментарии


Для определения типа сезонности используйте свойство [ISmx12arima.SeasonalAdjustmentMode](ISmx12arima.SeasonalAdjustmentMode.htm).
