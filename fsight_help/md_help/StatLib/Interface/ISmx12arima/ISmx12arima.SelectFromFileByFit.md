# ISmx12arima.SelectFromFileByFit


## Синтаксис


SelectFromFileByFit: Boolean;


## Описание


Свойство SelectFromFileByFit
 определяет, брать ли спецификацию параметров ARIMA из файла, используя
 ту, что дает более близкие значения для данных вне периода идентификации.


## Комментарии


Файл задается в виде строкового массива с помощью свойства [ISmx12arima.x12aMdlFile](ISmx12arima.x12aMdlFile.htm).


Допустимые значения:


	- True. Спецификация параметров
	 ARIMA берется из файла, используя ту, что дает более близкие значения
	 для данных вне периода идентификации;


	- False. Спецификация
	 параметров ARIMA не берется из файла.
