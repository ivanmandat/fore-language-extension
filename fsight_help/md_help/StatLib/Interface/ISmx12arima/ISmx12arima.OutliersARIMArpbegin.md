# ISmx12arima.OutliersARIMArpbegin


## Синтаксис


OutliersARIMArpbegin: [ISlOutliers](../ISlOutliers/ISlOutliers.htm);


## Описание


Свойство OutliersARIMArpbegin возвращает
 список выбросов типа «постепенная смена уровня» (дата начала), учитываемых
 на шаге ARIMA.


## Комментарии


Для данного типа выброса должна быть задана дата окончания с помощью
 свойства [ISx12arima.OutliersARIMArpend](ISmx12arima.OutliersARIMArpend.htm).
