# ISmx12arima.SpectralPlots


## Синтаксис


SpectralPlots: Boolean;


## Описание


Свойство SpectralPlots определяет,
 строить ли графики спектральной плотности.


## Комментарии


Допустимые значения:


	- True. Графики спектральной
	 плотности строятся;


	- False. Графики спектральной
	 плотности не строятся.


Данные для построения графиков выгружаются в массивы [ISmx12arima.SP1frequency](ISmx12arima.SP1frequency.htm),
 [ISmx12arima.SP1spectr](ISmx12arima.SP1spectr.htm),
 [ISmx12arima.SP2frequency](ISmx12arima.SP2frequency.htm),
 [ISmx12arima.SP2spectr](ISmx12arima.SP2spectr.htm).
