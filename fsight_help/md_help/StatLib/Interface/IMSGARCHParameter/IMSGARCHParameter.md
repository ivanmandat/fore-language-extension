# IMSGARCHParameter


Сборка: Stat;


## Описание


Интерфейс IMSGARCHParameter
 предназначен для работы с MS-GARCH константами.


## Иерархия наследования

## Комментарии


Для работы с интерфейсом IMSGARCHParameter
 используйте свойства интерфейса [ISmMarkovSwitchingGARCH](../ISmMarkovSwitchingGARCH/ISmMarkovSwitchingGARCH.htm).
