# IArimaSpecification.SeasonalDifferenceOrder


## Синтаксис


SeasonalDifferenceOrder: Integer;


## Описание


Свойство SeasonalDifferenceOrder
 определяет порядок сезонной разности.


## Комментарии


По умолчанию свойство имеет значение «0».


Необходимо помнить, что при задании сезонной разности должен быть задан
 период сезонности: [IArimaSpecification.Cycle](IArimaSpecification.Cycle.htm).
