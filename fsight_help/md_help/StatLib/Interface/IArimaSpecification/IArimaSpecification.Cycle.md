# IArimaSpecification.Cycle


## Синтаксис


Cycle: Integer;


## Описание


Свойство Cycle определяет период
 сезонности. Например, 4 квартала или 12 месяцев.


## Комментарии


По умолчанию свойство имеет значение «0».
