# IArimaSpecification.SeasonalAutoRegressionOrder


## Синтаксис


SeasonalAutoRegressionOrder: Integer;


## Описание


Свойство SeasonalAutoRegressionOrder
 определяет порядок сезонной авторегрессии.


## Комментарии


По умолчанию свойство имеет значение «0».


Необходимо помнить, что при задании сезонной авторегрессии должен быть
 задан период сезонности: [IArimaSpecification.Cycle](IArimaSpecification.Cycle.htm).
