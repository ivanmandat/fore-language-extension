# IArimaSpecification.SeasonalMovingAverageOrder


## Синтаксис


SeasonalMovingAverageOrder: Integer;


## Описание


Свойство SeasonalMovingAverageOrder
 определяет порядок сезонного скользящего среднего.


## Комментарии


По умолчанию свойство имеет значение «0».


Необходимо помнить, что при задании сезонного скользящего среднего должен
 быть задан период сезонности: [IArimaSpecification.Cycle](IArimaSpecification.Cycle.htm).
