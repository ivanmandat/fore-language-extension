# ISmBoxConstrainedOptimization.UseDerivatives


## Синтаксис


UseDerivatives: Boolean;


## Описание


Свойство UseDerivatives определяет,
 будут ли при поиске решения использоваться аналитические производные.


## Комментарии


Допустимые значения:


	- True. При поиске решения
	 используются аналитические производные;


	- False. Значение по умолчанию.
	 При поиске решения используются численные производные.
