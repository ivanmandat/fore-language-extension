# ISmBinaryModel.ClassificationCutOff


## Синтаксис


ClassificationCutOff: Double;


## Описание


Свойство ClassificationCutOff
 определяет значение для деления на группы.


## Комментарии


По умолчанию ClassificationCutOff
 = 0.5.
