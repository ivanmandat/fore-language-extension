# ISmBinaryModel.BinaryFitted


## Синтаксис


BinaryFitted: Array;


## Описание


Свойство BinaryFitted возвращает
 бинарный модельный ряд.


## Комментарии


Для получения бинарного прогнозного ряда используйте свойство [ISmBinaryModel.BinaryForecast](ISmBinaryModel.BinaryForecast.htm).


Для получения модельного ряда используйте свойство [ISmBinaryModel.Fitted](ISmBinaryModel.Fitted.htm).


Для получения вероятностного модельного ряда используйте свойство [ISmBinaryModel.ProbabilityFitted](ISmBinaryModel.ProbabilityFitted.htm).
