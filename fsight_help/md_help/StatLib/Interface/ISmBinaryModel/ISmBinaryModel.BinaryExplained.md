# ISmBinaryModel.BinaryExplained


## Синтаксис


BinaryExplained: Array;


## Описание


Свойство BinaryExplained определяет
 объясняемый ряд.


## Комментарии


Объясняемый и объясняющие ряды задаются в виде вектора для обычной модели
 и в виде двумерного массива - для модели бинарных данных. Для объясняемого
 ряда допустимы значения 0 и 1, остальные значения игнорируются как пропуск.


Для получения объясняющих рядов используйте свойство [ISmBinaryModel.Explanatories](ISmBinaryModel.Explanatories.htm).
