# ISmBinaryModel.ModelCoefficients


## Синтаксис


ModelCoefficients: [IModelCoefficients](../IModelCoefficients/IModelCoefficients.htm);


## Описание


Свойство ModelCoefficients возвращает
 параметры коэффициентов модели.


## Комментарии


Для получения рассчитанных статистических характеристик используйте
 свойство [ISmBinaryModel.SummaryStatistics](ISmBinaryModel.SummaryStatistics.htm).


Под коэффициентами модели подразумеваются коэффициенты при факторах
 и значение константы.
