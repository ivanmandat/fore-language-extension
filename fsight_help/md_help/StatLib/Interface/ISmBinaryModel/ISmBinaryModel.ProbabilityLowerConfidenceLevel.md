# ISmBinaryModel.ProbabilityLowerConfidenceLevel


## Синтаксис


ProbabilityLowerConfidenceLevel: Array;


## Описание


Свойство ProbabilityLowerConfidenceLevel
 возвращает нижнюю доверительную границу для вероятностного прогнозного
 ряда.


## Комментарии


Для получения верхней доверительной границы вероятностного прогнозного
 ряда используйте свойство [ISmBinaryModel.ProbabilityUpperConfidenceLevel](ISmBinaryModel.ProbabilityUpperConfidenceLevel.htm).
