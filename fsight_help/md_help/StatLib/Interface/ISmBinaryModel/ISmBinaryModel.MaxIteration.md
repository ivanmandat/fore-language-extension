# ISmBinaryModel.MaxIteration


## Синтаксис


MaxIteration: Integer;


## Описание


Свойство MaxIteration определяет
 максимальное число итераций, за которое должен осуществиться поиск решения.


## Комментарии


Максимальное число итераций может принимать любые неотрицательные значения.
 По умолчанию для расчёта используется 500 итераций.
