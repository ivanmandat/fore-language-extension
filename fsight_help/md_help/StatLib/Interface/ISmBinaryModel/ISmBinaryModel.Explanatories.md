# ISmBinaryModel.Explanatories


## Синтаксис


Explanatories: [ISlSeries](../ISlSeries/ISlSeries.htm);


## Описание


Свойство Explanatories возвращает
 объясняющие ряды.


## Комментарии


Объясняемый и объясняющие ряды задаются в виде вектора для обычной модели
 и в виде двумерного массива - для модели бинарных данных.


Объясняемый ряд определяет свойство [ISmBinaryModel.BinaryExplained](ISmBinaryModel.BinaryExplained.htm).
