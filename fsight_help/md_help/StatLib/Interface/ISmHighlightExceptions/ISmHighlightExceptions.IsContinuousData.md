# ISmHighlightExceptions.IsContinuousData


## Синтаксис


IsContinuousData: Boolean;


## Описание


Свойство IsContinuousData определяет,
 являются данные количественными или категориальными.


## Комментарии


Допустимые значения:


	- True. Данные являются
	 количественными, то есть содержащими числовые данные;


	- False. Данные являются
	 категориальными, то есть содержащими только целые числовые данные.
