# ISmHighlightExceptions.MedianDeviationFraction


## Синтаксис


MedianDeviationFraction: Double;


## Описание


Свойство MedianDeviationFraction
 определяет долю медианного отклонения.


## Комментарии


Минимальное значение - «0,5».


Чем больше значение доли медианного отклонения, тем меньше исключений
 будет определяться.
