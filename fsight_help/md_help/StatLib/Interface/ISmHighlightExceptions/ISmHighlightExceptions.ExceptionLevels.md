# ISmHighlightExceptions.ExceptionLevels


## Синтаксис


ExceptionLevels: Array;


## Описание


Свойство ExceptionLevels возвращает
 массив уровней исключительности наблюдений (по всем признакам).


## Комментарии


Индексы признаков, имеющих максимальный уровень исключительности, возвращает
 свойство [ISmHighlightExceptions.Indexes](ISmHighlightExceptions.Indexes.htm).
 Если ExceptionLevels[i] = 0,
 то наблюдение не является исключительным и Indexes[i]
 возвращает значение « -1».
