# ISmHighlightExceptions.MissingData


## Синтаксис


MissingData: [IMissingData](../IMissingData/IMissingData.htm);


## Описание


Свойство MissingData определяет
 метод обработки пропусков, применяемый к исходным данным.


## Комментарии


По умолчанию обработка пропусков не выполняется.


Для определения исходных данных используйте свойство [ISmHighlightExceptions.Data](ISmHighlightExceptions.Data.htm).
