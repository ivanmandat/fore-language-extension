# ISmHighlightExceptions.Indexes


## Синтаксис


Indexes: Array;


## Описание


Свойство Indexes возвращает
 индексы признаков, имеющих максимальный уровень исключительности.


## Комментарии


Массив максимальных уровней исключительности наблюдений (по всем признакам)
 возвращает свойство [ISmHighlightExceptions.ExceptionLevels](ISmHighlightExceptions.ExceptionLevels.htm).


Если ExceptionLevels[i] = 0,
 то наблюдение не является исключительным и Indexes[i]
 возвращает значение « -1».
