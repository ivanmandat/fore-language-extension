# ISlFrequentItemset


Сборка: Stat;


## Описание


Интерфейс ISlFrequentItemset предназначен
 для работы с набором часто встречаемых элементов, полученных в результате
 интеллектуального анализа данных методом «Анализ
 ассоциаций».


## Иерархия наследования


           ISlFrequentItemset


## Комментарии


Для выполнения интеллектуального анализа данных методом «Анализ ассоциаций» используйте
 интерфейс [ISmAssociationRules](../ISmAssociationRules/ISmAssociationRules.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property.gif)
		 [Itemset](ISlFrequentItemset.Itemset.htm)

		 ![](../../Property.gif)
		 [SupportCount](ISlFrequentItemset.SupportCount.htm)


		 Свойство SupportCount
		 возвращает поддержку набора, выраженную в количестве

		 ![](../../Property.gif)
		 [SupportPercent](ISlFrequentItemset.SupportPercent.htm)


		 Свойство SupportPercent
		 возвращает поддержку правила, выраженную в проценте транзакций,
		 в которых он содержится.
