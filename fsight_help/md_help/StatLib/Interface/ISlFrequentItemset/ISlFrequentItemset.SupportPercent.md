# ISlFrequentItemset.SupportPercent


## Синтаксис


SupportPercent: Double;


## Описание


Свойство SupportPercent возвращает
 поддержку правила, выраженную в проценте транзакций, в которых он
 содержится.


## Комментарии


Поддержку, выраженную в количестве транзакций, в которых содержится
 набор, возвращает свойство [ISlFrequentItemset.SupportPercent](ISlFrequentItemset.SupportPercent.htm).
