# ISlFrequentItemset.SupportCount


## Синтаксис


SupportCount: Integer;


## Описание


Свойство SupportCount возвращает
 поддержку набора, выраженную в количестве транзакций, содержащих
 данный набор.


## Комментарии


Поддержку, выраженную в процентах транзакций, в которых содержится данный
 набор, возвращает свойство [ISlFrequentItemset.SupportPercent](ISlFrequentItemset.SupportPercent.htm).
