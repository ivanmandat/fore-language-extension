# ISlSerie.Id


## Синтаксис


Id: String;


## Описание


Свойство Id определяет идентификатор
 ряда данных.


## Комментарий


По умолчанию идентификатор ряда данных не задан. Для изменения наименования
 ряда используйте свойство [ISlSerie.Name](ISlSerie.Name.htm).
