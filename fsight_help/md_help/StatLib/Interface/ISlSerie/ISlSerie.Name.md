# ISlSerie.Name


## Синтаксис


Name: String;


## Описание


Свойство Name определяет наименование
 ряда данных.


## Комментарии


По умолчанию наименование ряда не задано. Для изменения идентификатора
 ряда используйте свойство [ISlSerie.Id](ISlSerie.Id.htm).
