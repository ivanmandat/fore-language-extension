# ISlSerie


Сборка: Stat;


## Описание


Интерфейс ISlSerie определяет
 параметры ряда данных.


## Иерархия наследования


           ISlSerie


## Комментарии


Во всех моделях, за исключением [ISmPooledModel](../ISmPooledModel/ISmPooledModel.htm)
 и [ISmBinaryModel](../ISmBinaryModel/ISmBinaryModel.htm), ряд
 данных должен задаваться одномерным массивом (вектором).


Индексация входных массивов должна начинаться с нуля.
