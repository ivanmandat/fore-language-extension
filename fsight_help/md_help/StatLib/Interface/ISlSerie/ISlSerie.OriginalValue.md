# ISlSerie.OriginalValue


## Синтаксис


OriginalValue: Array;


## Описание


Свойство OriginalValue возвращает
 оригинал значений ряда данных, то есть тех данных, которые были заданы
 с помощью свойства [Value](ISlSerie.Value.htm).


## Комментарии


Для расчетов статистических методов используются данные из OriginalValue.
