# ISlSerie.Value


## Синтаксис


Value: Array;


## Описание


Свойство Value определяет значения
 ряда данных.


## Комментарии


После задания значений ряда они будут установлены в свойство [ISlSerie.OriginalValue](ISlSerie.OriginalValue.htm).


Индексация входных массивов должна начинаться с нуля.
