# ISmLogisticDistribution


Сборка: Stat;


## Описание


Интерфейс ISmLogisticDistribution
 позволяет генерировать выборку псевдослучайных чисел из логистического
 распределения с заданными параметром сдвига (медианой) и параметром масштаба.
