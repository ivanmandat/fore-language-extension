# ISlQualityTable


## Описание


Интерфейс ISlQualityTable содержит
 свойства и методы для работой с коллекцию значений, описывающих качество
 подбора бинарной модели. Данные значения образуют следующую таблицу, например:


		 Фактически

		[ISlQualitySet.Actual](../ISlQualitySet/ISlQualitySet.Actual.htm)
		 Предсказано

		[ISlQualitySet.Prediction](../ISlQualitySet/ISlQualitySet.Prediction.htm)
		 Правильно предсказано

		[ISlQualitySet.CorrectPrediction](../ISlQualitySet/ISlQualitySet.CorrectPrediction.htm)


		 Количество «0»
		 3
		 3
		 2


		 Количество «1»
		 2
		 2
		 1
