# ISlQualityTable.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс строки в таблице.


## Описание


Метод Remove удаляет указанную строку из таблицы и возвращает значение True, если удаление прошло успешно.
