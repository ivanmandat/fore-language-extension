# IExponentialSmoothingBestTrialMethod


Сборка: Stat;


## Описание


Интерфейс IExponentialSmoothingBestTrialMethod
 определяет настройки метода наилучшей пробы для автоподбора параметров
 экспоненциального сглаживания.
