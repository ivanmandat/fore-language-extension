# IExponentialSmoothingBestTrialMethod.MaxIteration


## Синтаксис


MaxIteration: Integer;


## Описание


Свойство MaxIteration определяет
 максимальное число итераций (n) метода наилучшей пробы для поиска параметров.


## Комментарии


Для определения числа реализаций (S) в одной итерации метода наилучшей
 пробы используйте свойство [IExponentialSmoothingBestTrialMethod.Order](IExponentialSmoothingBestTrialMethod.Order.htm).
