# ISmLinearProgramming.IntVec


## Синтаксис


IntVec: Array;


## Описание


Свойство IntVec определяет числовой
 массив, в котором задается, какие переменные являются целочисленными.


## Комментарии


Длина массива IntVec должна
 быть равна количеству переменных.


Если значение элемента массива IntVec равно
 «0», то соответствующая переменная считается вещественной, а если не равно
 «0» - целочисленной.


Если массив IntVec не задан,
 все переменные считаются вещественными.
