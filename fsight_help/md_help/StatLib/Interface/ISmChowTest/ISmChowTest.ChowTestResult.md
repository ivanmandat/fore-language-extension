# ISmChowTest.ChowTestResult


## Синтаксис


ChowTestResult: Boolean;


## Описание


Свойство ChowTestResult возвращает
 результат о принятии нулевой гипотезы.


## Комментарии


	- True. Выборки
	 ОДНОРОДНЫЕ, нулевая гипотеза об отсутствии структурных изменений принимается;


	- False. Выборки
	 РАЗНОРОДНЫЕ, нулевая гипотеза об отсутствии структурных изменений
	 отвергается.
