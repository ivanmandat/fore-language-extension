## Описание


Свойство LR0 возвращает уравнение
 линейной регрессии для первой группы.


## Комментарии


Для получения уравнения второй группы и общего уравнения групп используйте
 свойства [ISmChowTest.LR1](ISmChowTest.LR1.htm)
 и [ISmChowTest.LR](ISmChowTest.LR.htm).
