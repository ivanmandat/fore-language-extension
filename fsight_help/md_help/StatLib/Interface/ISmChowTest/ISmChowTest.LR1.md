## Описание


Свойство LR1 возвращает уравнение
 линейной регрессии для второй группы.


## Комментарии


Для получения уравнения первой группы и общего уравнения групп используйте
 свойства [ISmChowTest.LR0](ISmChowTest.LR0.htm)
 и [ISmChowTest.LR](ISmChowTest.LR.htm).
