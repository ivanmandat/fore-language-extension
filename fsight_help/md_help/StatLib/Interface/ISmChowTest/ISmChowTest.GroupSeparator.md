# ISmChowTest.GroupSeparator


## Синтаксис


GroupSeparator: Array;


## Описание


Свойство GroupSeparator определяет
 группы наблюдений.


## Комментарии


Данный массив должен иметь размерность, совпадающую с размерностью исходного
 ряда, и состоять из:


	- 0 - определяет
	 наблюдение в первую группу.


	- 1 - определяет
	 наблюдение во вторую группу.


Именно созданные группы и будут проверяться на однородность.
