# ISmGradientBoostedTree.Dependent


## Синтаксис


Dependent: [ISlSerieInt](../ISlSerieInt/ISlSerieInt.htm);


## Описание


Свойство Dependent возвращает
 объясняемый ряд.


## Комментарии


Для задания объясняемого ряда используйте свойство Dependent.[Value](../ISlSerieInt/ISlSerieInt.Value.htm).
 Если планируется использовать процедуру Binning, то для задания объясняемого
 ряда используйте свойство Dependent.[BinningSettings](../ISlSerieInt/ISlSerieInt.BinningSettings.htm).[DoubleValue](../IBinningSettingsInt/IBinningSettingsInt.DoubleValue.htm).


Для задания объясняющих рядов используйте свойства [ISmGradientBoostedTree.ExplanatoriesCategorical](ISmGradientBoostedTree.ExplanatoriesCategorical.htm),
 [ISmGradientBoostedTree.ExplanatoriesContinuous](ISmGradientBoostedTree.ExplanatoriesContinuous.htm)
 и [ISmGradientBoostedTree.ExplanatoriesOrdered](ISmGradientBoostedTree.ExplanatoriesOrdered.htm).
