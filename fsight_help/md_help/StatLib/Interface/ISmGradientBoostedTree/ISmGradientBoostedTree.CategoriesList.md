# ISmGradientBoostedTree.CategoriesList


## Синтаксис


CategoriesList: Array;


## Описание


Свойство CategoriesList возвращает
 список категорий объясняемого ряда.


## Комментарии


Свойство CategoriesList
 содержит все уникальные значения из ряда [ISmGradientBoostedTree.Dependent](ISmGradientBoostedTree.Dependent.htm),
 расположенные в порядке возрастания.


Для получения итоговой классификации объектов используйте свойство [ISmGradientBoostedTree.ClassificationSummary](ISmGradientBoostedTree.ClassificationSummary.htm).
