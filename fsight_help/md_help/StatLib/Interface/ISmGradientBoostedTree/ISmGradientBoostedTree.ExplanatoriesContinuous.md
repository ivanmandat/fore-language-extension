# ISmGradientBoostedTree.ExplanatoriesContinuous


## Синтаксис


ExplanatoriesContinuous: [ISlSeries](../ISlSeries/ISlSeries.htm);


## Описание


Свойство ExplanatoriesContinuous
 определяет объясняющие количественные ряды.


## Комментарии


Для задания объясняемого ряда используйте свойство [ISmGradientBoostedTree.Dependent](ISmGradientBoostedTree.Dependent.htm),
 для задания объясняющих категориальных рядов - свойство [ISmGradientBoostedTree.ExplanatoriesCategorical](ISmGradientBoostedTree.ExplanatoriesCategorical.htm),
 а для задания объясняющих порядковых рядов - свойство [ISmGradientBoostedTree.ExplanatoriesOrdered](ISmGradientBoostedTree.ExplanatoriesOrdered.htm).


Количественный (вещественный) ряд - набор данных, элементы которого
 можно упорядочить, и между которыми можно найти расстояние. Например,
 расход топлива автомобилей. Можно упорядочить и сказать, у кого больше,
 а у кого меньше.
