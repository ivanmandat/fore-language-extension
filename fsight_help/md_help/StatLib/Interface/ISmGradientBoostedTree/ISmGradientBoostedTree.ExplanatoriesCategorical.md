# ISmGradientBoostedTree.ExplanatoriesCategorical


## Синтаксис


ExplanatoriesCategorical: [ISlSeriesInt](../ISlSeriesInt/ISlSeriesInt.htm);


## Описание


Свойство ExplanatoriesCategorical
 возвращает объясняющие категориальные ряды.


## Комментарии


Для задания объясняемого ряда используйте свойство [ISmGradientBoostedTree.Dependent](ISmGradientBoostedTree.Dependent.htm),
 для задания объясняющих количественных рядов - свойство [ISmGradientBoostedTree.ExplanatoriesContinuous](ISmGradientBoostedTree.ExplanatoriesContinuous.htm),
 а для задания объясняющих порядковых рядов - свойство [ISmGradientBoostedTree.ExplanatoriesOrdered](ISmGradientBoostedTree.ExplanatoriesOrdered.htm).


Категориальный ряд - набор данных, элементы которого нельзя упорядочить,
 и между которыми нельзя найти расстояние. Например, что обычно едят на
 завтрак люди из выборки: бутерброд, каша, яичница и т.д. Ни упорядочить,
 ни найти расстояние между элементами массива нельзя.
