# ISmCensus2.SigmaLevel


## Синтаксис


SigmaLevel: [ICensus2SigmaLevel](../ICensus2SigmaLevel/ICensus2SigmaLevel.htm);


## Описание


Свойство SigmaLevel определяет уровень Sigma.


## Комментарии


По умолчанию нижняя граница равна «1,5», верхняя - «2,5».


Задание нижней и верхней границ Sigma дает возможность уменьшить влияние выбросов на оценки, исключив из рассмотрения значения, выходящие за заданный диапазон, либо преобразовав такие значения.
