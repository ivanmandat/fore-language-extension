# ISmCensus2.DailyWeights


## Синтаксис


DailyWeights: Array;


## Описание


Свойство DailyWeights определяет
 веса для дней недели.


## Комментарии


Месяцы содержат различное число рабочих дней, это может повлиять на
 окончательный результат анализа. Используйте данное свойство для задания
 веса для каждого дня недели. В сумме веса должны дать «7». Если их сумма
 не равна семи, то для весов применяется нормирование.


По умолчанию веса всех дней недели равны «1». Если все элементы массива
 DailyWeights равны нулю, нормализовать
 такой массив нельзя и в расчете он не участвует.


Веса для дней недели задаются только для месячных данных.


Для получения нормированных весов для дней недели используйте свойство
 [ISmCensus2.NormDailyWeights](ISmCensus2.NormDailyWeights.htm).
