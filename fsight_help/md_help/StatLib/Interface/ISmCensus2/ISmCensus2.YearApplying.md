# ISmCensus2.YearApplying


## Синтаксис


YearApplying: Integer;


## Описание


Свойство YearApplying определяет
 начальную дату (год) для применения зависимости от рабочих дней недели.


## Комментарии


Начальную дату (год) для вычисления зависимости от рабочих дней недели
 определяет свойство [ISmCensus2.YearComputing](ISmCensus2.YearComputing.htm).
