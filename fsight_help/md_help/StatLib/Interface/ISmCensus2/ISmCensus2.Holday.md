# ISmCensus2.Holday


## Синтаксис


Holday: [Census2HolidayAdjType](../../Enums/Census2HolidayAdjType.htm);


## Описание


Свойство Holiday определяет,
 делать ли поправку на праздничные дни при расчете.


## Комментарии


По умолчанию поправка на праздничные дни не производится, то есть Holday
 = Census2HolidayAdjType.C2hatNone.
