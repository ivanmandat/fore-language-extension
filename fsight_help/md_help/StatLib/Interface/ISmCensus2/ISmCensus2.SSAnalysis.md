# ISmCensus2.SSAnalysis


## Синтаксис


SSAnalysis: Boolean;


## Описание


Свойство SSAnalysis определяет,
 применять ли в расчете скользящие интервалы.


## Комментарии


SSAnalysis = True.
 Скользящие интервалы применяются в расчете.


SSAnalysis = False.
 Значение по умолчанию. Скользящие интервалы не применяются в расчете.
