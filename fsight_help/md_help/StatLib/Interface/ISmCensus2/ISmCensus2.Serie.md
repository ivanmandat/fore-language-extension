# ISmCensus2.Serie


## Синтаксис


Serie: [ISlSerie](../ISlSerie/ISlSerie.htm);


## Описание


Свойство Serie определяет объясняемый ряд.


## Комментарии


Для месячных данных длина ряда должна быть не менее 36 месяцев (3 года), для квартальных - 12 кварталов (3 года).
