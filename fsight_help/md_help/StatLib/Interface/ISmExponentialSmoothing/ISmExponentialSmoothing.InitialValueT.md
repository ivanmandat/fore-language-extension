# ISmExponentialSmoothing.InitialValueT


## Синтаксис


InitialValueT: Double;


## Описание


Свойство InitialValueT определяет
 начальную точку линии тренда.


## Комментарии


Если данное значение не задается, то оно будет рассчитано автоматически.
 Формула расчета зависит от выбранных параметров модели.
