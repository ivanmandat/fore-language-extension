# ISlSeries.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс ряда в коллекции рядов.


## Описание


Метод Remove удаляет ряд данных.


## Комментарии


В случае успешного удаления ряда метод возвращает True, в противном случаеFalse.
