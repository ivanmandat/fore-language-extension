# ISlSeries


## Описание


Интерфейс ISlSeries определяет
 параметры коллекции рядов данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property.gif)
		 [Count](ISlSeries.Count.htm)


		 Свойство Count возвращает
		 количество рядов данных в коллекции.


		 ![](../../Property.gif)
		 [Item](ISlSeries.Item.htm)


		 Свойство Item определяет
		 параметры ряда данных.

		 [Clear](ISlSeries.Clear.htm)
