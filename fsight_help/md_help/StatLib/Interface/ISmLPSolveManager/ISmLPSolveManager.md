# ISmLPSolveManager


Сборка: Stat;


## Описание


Интерфейс ISmLPSolveManager
 предназначен для задания пути к установленному и интегрированному с «Форсайт. Аналитическая платформа»
 модулю LPSolve.


## Иерархия наследования


ISmLPSolveManager


## Комментарии


Для настройки пути создайте экземпляр класса [SmLPSolveManager](../../Class/SmLPSolveManager/SmLPSolveManager.htm)
 и приведите полученный объект к интерфейсу ISmLPSolveManager. Задайте
 путь в свойстве [UserLPSolvePath](ISmLPSolveManager.UserLPSolvePath.htm).
 Заданные настройки будут сохранены в параметрах репозитория.
