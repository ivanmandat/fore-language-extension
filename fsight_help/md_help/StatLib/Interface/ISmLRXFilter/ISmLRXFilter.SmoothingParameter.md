# ISmLRXFilter.SmoothingParameter


## Синтаксис


SmoothingParameter: Double;


## Описание


Свойство SmoothingParameter
 определяет меру сглаживания ряда.


## Комментарии


Диапазон допустимых значений: [0,001; 100000]. Значение по умолчанию
 - «100».


Чем больше значение меры, тем более «гладкий» получается ряд.
