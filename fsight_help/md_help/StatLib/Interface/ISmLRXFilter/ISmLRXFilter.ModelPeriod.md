# ISmLRXFilter.ModelPeriod


## Синтаксис


ModelPeriod: [IStatPeriod](../IStatPeriod/IStatPeriod.htm);


## Описание


Свойство ModelPeriod определяет
 параметры периода идентификации.


## Комментарии


По умолчанию период идентификации не задан и совпадает с границами исходного
 ряда.
