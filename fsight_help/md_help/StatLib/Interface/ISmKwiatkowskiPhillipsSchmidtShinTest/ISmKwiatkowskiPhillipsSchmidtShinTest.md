# ISmKwiatkowskiPhillipsSchmidtShinTest


Сборка: Stat;


## Описание


Интерфейс ISmKwiatkowskiPhillipsSchmidtShinTest
 предназначен для работы с параметрами теста Квятковского-Филлипса-Шмидта-Шина.


## Иерархия наследования


           [IStatMethod](../IStatMethod/IStatMethod.htm)


           ISmKwiatkowskiPhillipsSchmidtShinTest


## Комментарии


Тест возвращает статистику Квятковского-Филлипса-Шмидта-Шина.
