# IDependenceForms


## Описание


Интерфейс IDependenceForms возвращает
 коллекцию форм зависимости. Данная коллекция содержит все формы зависимостей,
 определенные в перечислении [DependenceType](../../Enums/DependenceType.htm).


## Свойства

		 число форм зависимостей в коллекции.


		 ![](../../Property.gif)
