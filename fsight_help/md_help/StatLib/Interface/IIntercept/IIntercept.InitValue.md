# IIntercept.InitValue


## Синтаксис


InitValue: Double;


## Описание


Свойство InitValue определяет
 начальное приближение для константы.


## Комментарии


Начальное приближение учитывается, если значение константы оценивается
 автоматически, и начальные приближения задаются вручную, т.е. [IIntercept.Mode](IIntercept.Mode.htm) = [InterceptModeInterceptMode](../../Enums/InterceptMode.htm).AutoEstimate
 и [ISlARMA.CalcInitMode](../ISlARMA/ISlARMA.CalcInitMode.htm) = [ARMAInitType](../../Enums/ARMAInitType.htm).Manual.
