# IIntercept.StandardError


## Синтаксис


StandardError: Double;


## Описание


Свойство StandardError возвращает
 значение стандартной ошибки константы.


## Комментарии


Под стандартной ошибкой понимается приближённая величина отклонения
 оценки коэффициента от истинного значения. Чем больше значение стандартной
 ошибки, тем менее достоверной является оценка коэффициента.
