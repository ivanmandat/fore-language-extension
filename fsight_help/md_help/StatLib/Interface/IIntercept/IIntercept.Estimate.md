# IIntercept.Estimate


## Синтаксис


Estimate: Double;


## Описание


Свойство Estimate определяет
 значение константы.


## Комментарии


Назначение свойства зависит от значения [IIntercept.Mode](IIntercept.Mode.htm). Если:


	- IIntercept.Mode = InterceptMode.InterceptMode,
	 т.е. значение константы задаётся вручную, то используйте свойство
	 Estimate для ввода значения
	 константы;


	- IIntercept.Mode = InterceptMode.AutoEstimate,
	 т.е. значение константы оценивается автоматически, то после расчёта
	 метода используйте свойство Estimate
	 для получения значения константы.
