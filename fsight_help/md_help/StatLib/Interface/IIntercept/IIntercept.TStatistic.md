# IIntercept.TStatistic


## Синтаксис


TStatistic: Double;


## Описание


Свойство TStatistic возвращает
 значение t-статистики константы.


## Комментарии


t-статистика используется для проверки нулевой гипотезы о том, что коэффициент
 равен нулю и соответственно не значим в рассматриваемой модели.
