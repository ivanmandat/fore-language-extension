# ISmGARCH.GARCHOrder


## Синтаксис


GARCHOrder: Integer;


## Описание


Свойство GARCHOrder определяет
 порядок обобщенной авторегрессии условной гетероскедастичности.


## Комментарии


Порядок авторегрессиии условной гетероскедастичности
 определяет свойство [ISmGARCH.ARCHOrder](ISmGARCH.ARCHOrder.htm).


Для корректного расчета метода хотя бы один из порядков [ISmGARCH.ARCHOrder](ISmGARCH.ARCHOrder.htm) и GARCHOrder
 должен быть ненулевым.
