# ISmGARCH.GARCHCoefficients


## Синтаксис


GARCHCoefficients: [ICoefficients](../ICoefficients/ICoefficients.htm);


## Описание


Свойство GARCHCoefficients возвращает
 оценки коэффициентов авторегрессии условной гетероскедастичности
 и обобщенной авторегрессии условной гетероскедастичности.


## Комментарии


Оценки коэффициентов доступны после расчёта метода. Коэффициенты идут
 в следующем порядке: константа обобщенной авторегрессиии условной гетероскедастичности,
 массив коэффициентов авторегрессии условной гетероскедастичности,
 массив коэффициентов обобщенной авторегрессиии условной гетероскедастичности.
