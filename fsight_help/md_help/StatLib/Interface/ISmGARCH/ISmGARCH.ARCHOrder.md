# ISmGARCH.ARCHOrder


## Синтаксис


ARCHOrder: Integer;


## Описание


Свойство ARCHOrder определяет
 порядок авторегрессии условной гетероскедастичности.


## Комментарии


Порядок обобщенной авторегрессиии условной гетероскедастичности
 определяет свойство [ISmGARCH.GARCHOrder](ISmGARCH.GARCHOrder.htm).


Для корректного расчета метода хотя бы один из порядков ARCHOrder
 и [ISmGARCH.GARCHOrder](ISmGARCH.GARCHOrder.htm)
 должен быть ненулевым.
