# ISmGARCH.UseDefaultInitValues


## Синтаксис


UseDefaultInitValues: Boolean;


## Описание


Свойство UseDefaultInitValues
 определяет, использовать ли начальные приближения, заданные по умолчанию.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Для расчёта используются начальные приближения, заданные по умолчанию;


	- False. Для расчёта используются
	 начальные приближения, заданные пользователем. Для объясняющих рядов
	 начальные приближения задаются с использованием свойства [ISlSerie.InitValue](../ISlSerie/ISlSerie.InitValue.htm), а для константы
	 - [IIntercept.InitValue](../IIntercept/IIntercept.InitValue.htm).
