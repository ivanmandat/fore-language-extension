# ISmGARCH.StationarityCondition


## Синтаксис


StationarityCondition: Boolean;


## Описание


Свойство StationarityCondition
 определяет использование условия стационарности.


## Комментарии


При значении True условие стационарности
 будет включено, при значении False
 – нет.


По умолчанию свойство имеет значение True.
