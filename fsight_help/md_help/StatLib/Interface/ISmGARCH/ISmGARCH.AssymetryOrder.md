# ISmGARCH.AssymetryOrder


## Синтаксис


AssymetryOrder: Integer;


## Описание


Свойство AssymetryOrder определяет
 порядок асимметрии.


## Комментарии


Если порядок равен 0, то будет рассчитана модель GARCH. Если порядок
 больше нуля, то будут рассчитаны модели EGARCH или GJR-GARCH в зависимости
 от [выбранного типа](ISmGARCH.GARCHSpec.htm).
