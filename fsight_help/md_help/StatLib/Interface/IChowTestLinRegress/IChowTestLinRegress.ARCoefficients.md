# IChowTestLinRegress.ARCoefficients


## Синтаксис


ARCoefficients: [IModelCoefficients](../IModelCoefficients/IModelCoefficients.htm);


## Описание


Свойство ARCoefficients возвращает
 коэффициенты авторегрессии.


## Комментарии


Коэффициенты рассчитываются, если с помощью свойства [ISmChowTest.ARMA](../ISmChowTest/ISmChowTest.ARMA.htm)
 задан порядок авторегрессии.
