# IChowTestLinRegress.MASeasCoefficients


## Синтаксис


MASeasCoefficients: [IModelCoefficients](../IModelCoefficients/IModelCoefficients.htm);


## Описание


Свойство MASeasCoefficients
 возвращает коэффициенты сезонного скользящего среднего.


## Комментарии


Коэффициенты рассчитываются, если с помощью свойства [ISmChowTest.ARMA](../ISmChowTest/ISmChowTest.ARMA.htm)
 задан порядок сезонного скользящего среднего.
