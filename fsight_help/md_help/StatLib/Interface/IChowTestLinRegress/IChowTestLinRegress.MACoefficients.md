# IChowTestLinRegress.MACoefficients


## Синтаксис


MACoefficients: [IModelCoefficients](../IModelCoefficients/IModelCoefficients.htm);


## Описание


Свойство MACoefficients возвращает
 коэффициенты скользящего среднего.


## Комментарии


Коэффициенты рассчитываются, если с помощью свойства [ISmChowTest.ARMA](../ISmChowTest/ISmChowTest.ARMA.htm)
 задан порядок скользящего среднего.
