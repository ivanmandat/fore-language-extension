# IBinaryTreeForest.Item


## Синтаксис


Item(Index: Integer): [IBinaryTreeNode](../IBinaryTreeNode/IBinaryTreeNode.htm);


## Параметры


Index. Индекс дерева в ансамбле.


## Описание


Свойство Item возвращает элементы
 одного из деревьев ансамбля по его индексу.


## Комментарии


Для получения количества деревьев в ансамбле используйте свойство [IBinaryTreeForest.Count](IBinaryTreeForest.Count.htm).
