# IBinaryTreeForest


Сборка: Stat;


## Описание


Интерфейс IBinaryTreeForest
 предназначен для работы с коллекцией деревьев.


## Иерархия наследования

## Комментарии


Для работы с бинарными вершинами дерева используйте свойства интерфейса
 [IBinaryTreeNode](../IBinaryTreeNode/IBinaryTreeNode.htm).
