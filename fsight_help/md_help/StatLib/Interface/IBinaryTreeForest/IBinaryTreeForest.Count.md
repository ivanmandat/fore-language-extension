# IBinaryTreeForest.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 деревьев в ансамбле.


## Комментарии


Для получения элементов одного из деревьев ансамбля по его индексу используйте
 свойство [IBinaryTreeForest.Item](IBinaryTreeForest.Item.htm).
