# ISlARMAGARCH.InitAR


## Синтаксис


InitAR: Array;


## Описание


Свойство InitAR определяет начальные
 приближения несезонной авторегрессии.


## Комментарии


Для определения начального приближения сезонного скользящего среднего
 используйте свойство [ISlARMAGARCH.InitMA](ISlARMAGARCH.InitMA.htm).
