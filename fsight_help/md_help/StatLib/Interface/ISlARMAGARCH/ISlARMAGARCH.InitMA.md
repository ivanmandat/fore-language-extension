# ISlARMAGARCH.InitMA


## Синтаксис


InitMA: Array;


## Описание


Свойство InitMA определяет начальные
 приближения сезонного скользящего среднего.


## Комментарии


Для определения начального приближения несезонной авторегрессии используйте
 свойство [ISlARMAGARCH.InitAR](ISlARMAGARCH.InitAR.htm).
