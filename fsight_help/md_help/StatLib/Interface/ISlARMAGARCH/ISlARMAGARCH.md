# ISlARMAGARCH


Сборка: Stat;


## Описание


Интерфейс ISlARMAGARCH предназначен
 для настройки ARIMA для модели GARCH.


## Иерархия наследования


           ISlARMAGARCH


## Комментарии


Модель GARCH (Generalized ARCH) - обобщение модели ARCH.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property.gif)
		 [CoefficientsAR](ISlARMAGARCH.CoefficientsAR.htm)


		 Свойство CoefficientsAR
		 возвращает коэффициенты несезонной авторегрессии.


		 ![](../../Property.gif)
		 [CoefficientsMA](ISlARMAGARCH.CoefficientsMA.htm)


		 Свойство CoefficientsMA возвращает
		 коэффициенты скользящего среднего.


		 ![](../../Property.gif)
		 [InitAR](ISlARMAGARCH.InitAR.htm)


		 Свойство InitAR определяет
		 начальные приближения несезонной авторегрессии.


		 ![](../../Property.gif)
		 [InitMA](ISlARMAGARCH.InitMA.htm)


		 Свойство InitMA определяет
		 начальные приближения сезонного скользящего среднего.


		 ![](../../Property.gif)
		 [OrderAR](ISlARMAGARCH.OrderAR.htm)


		 Свойство OrderAR определяет
		 порядок авторегрессии.


		 ![](../../Property.gif)
		 [OrderMA](ISlARMAGARCH.OrderMA.htm)


		 Свойство OrderMA определяет
		 порядок скользящего среднего.
