# ISmSimultaneousSystem.MaxIteration


## Синтаксис


MaxIteration: Integer;


## Описание


Свойство MaxIteration определяет
 максимальное число итераций.


## Комментарии


Задаваемое значение должно быть неотрицательным числом.


По умолчанию используется 100 итераций.


Для определения точности решения используйте свойство [ISmSimultaneousSystem.Tolerance](ISmSimultaneousSystem.Tolerance.htm).
