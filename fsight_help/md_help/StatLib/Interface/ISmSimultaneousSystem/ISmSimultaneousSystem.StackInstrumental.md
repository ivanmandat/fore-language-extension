# ISmSimultaneousSystem.StackInstrumental


## Синтаксис


StackInstrumental: [ISlSeries](../ISlSeries/ISlSeries.htm);


## Описание


Свойство StackInstrumental возвращает
 стековые инструментальные переменные.


## Комментарии


Стековые переменные используются одновременно во всех уравнениях.


Для определения, будет ли константа использоваться в качестве инструментальной
 переменной, используйте свойство [ISmSimultaneousSystem.UseConstantAsInstrument](ISmSimultaneousSystem.UseConstantAsInstrument.htm).
