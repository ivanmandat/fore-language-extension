# ISmSimultaneousSystem.SimultaneousSystemMethod


## Синтаксис


SimultaneousSystemMethod: [SimultaneousSystemMode](../../Enums/SimultaneousSystemMode.htm);


## Описание


Свойство SimultaneousSystemMethod определяет
 метод оценки для систем одновременных уравнений.


## Комментарии


По умолчанию используется линейная модель общего вида, то есть SimultaneousSystemMethod = SimultaneousSystemMode.General.
