# ISmSimultaneousSystem.Tolerance


## Синтаксис


Tolerance: Integer;


## Описание


Свойство Tolerance определяет
 точность решения.


## Комментарии


Задаваемое значение должно быть неотрицательным числом.


По умолчанию заданная точность ровна 0,0001.


Для определения максимального числа итераций используйте свойство [ISmSimultaneousSystem.MaxIteration](ISmSimultaneousSystem.MaxIteration.htm).
