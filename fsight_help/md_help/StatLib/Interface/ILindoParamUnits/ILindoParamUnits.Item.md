# ILindoParamUnits.Item


## Синтаксис


Item(Index: Integer): [ILindoParamUnit](../ILindoParamUnit/ILindoParamUnit.htm);


## Параметры


Index. Индекс параметра.


## Описание


Свойство Item возвращает указанный
 параметр из коллекции.


## Комментарии


Индексация параметров сквозная и начинается с нуля. Для получения количества
 параметров в коллекции используйте свойство [ILindoParamUnits.Count](ILindoParamUnits.Count.htm).
