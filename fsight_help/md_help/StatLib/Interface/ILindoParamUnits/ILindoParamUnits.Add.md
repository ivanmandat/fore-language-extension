# ILindoParamUnits.Add


## Синтаксис


Add: [ILindoParamUnit](../ILindoParamUnit/ILindoParamUnit.htm);


## Описание


Метод Add добавляет параметр
 в коллекцию.


## Комментарии


Для удаления одного параметра из коллекции используйте метод [ILindoParamUnits.Remove](ILindoParamUnits.Remove.htm),
 для удаления всех параметров - метод [ILindoParamUnits.Clear](ILindoParamUnits.Clear.htm).
