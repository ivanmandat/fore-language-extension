# ILindoParamUnits.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс параметра.


## Описание


Метод Remove удаляет указанный
 параметр из коллекции.


## Комментарии


Если удаление прошло успешно, то метод возвращает значение True,
 иначе - False.
