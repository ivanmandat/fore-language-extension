# ILindoParamUnits


Сборка: Stat;


## Описание


Интерфейс ILindoParamUnits
 предназначен для работы с коллекцией параметров в модуле LINDO.


## Иерархия наследования


           ILindoParamUnits


## Комментарии


Для получения коллекции параметров модели и параметров среды LINDO используйте
 свойства [ILindoSettings.ModelParams](../ILindoSettings/ILindoSettings.ModelParams.htm)
 и [ILindoSettings.EnvParams](../ILindoSettings/ILindoSettings.EnvParams.htm).
