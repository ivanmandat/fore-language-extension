# ISlBoundaryRegion


Сборка: Stat;


## Описание


Интерфейс ISlBoundaryRegion
 определяет параметры области определения.


## Иерархия наследования


           ISlBoundaryRegion


## Комментарии


Для решения задачи оптимизации квадратичной функции нескольких переменных
 при линейных ограничениях используйте интерфейс [ISmQuadraticProgramming](../ISmQuadraticProgramming/ISmQuadraticProgramming.htm).
