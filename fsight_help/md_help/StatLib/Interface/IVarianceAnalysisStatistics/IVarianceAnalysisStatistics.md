# IVarianceAnalysisStatistics


## Описание


Интерфейс IVarianceAnalysisStatistics
 возвращает результаты дисперсионного анализа.


Для однофакторного анализа результат возвращает свойство [OneFactorAnalysis](IVarianceAnalysisStatistics.OneFactorAnalysis.htm).


Для двухфакторного анализа результаты возвращают свойства: [Columns](IVarianceAnalysisStatistics.Columns.htm)
 и [Rows](IVarianceAnalysisStatistics.Rows.htm).
