# IHolidays


Сборка: Stat;


## Описание


Интерфейс IHolidays предназначен
 для учета поправок на праздничные дни.


## Иерархия наследования


           IHolidays


## Комментарии


Поправка на праздничные дни позволяет повысить точность расчетов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property.gif)
		 [Day](IHolidays.Day.htm)


		 Свойство Day определяет
		 количество дней, на которые влияет праздничный день.


		 ![](../../Property.gif)
		 [IsActive](IHolidays.IsActive.htm)


		 Свойство IsActive определяет,
		 учитывается ли поправка на праздничные дни.
