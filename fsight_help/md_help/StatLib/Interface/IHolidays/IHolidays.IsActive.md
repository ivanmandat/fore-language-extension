# IHolidays.IsActive


## Синтаксис


IsActive: Boolean;


## Описание


Свойство IsActive определяет,
 учитывается ли поправка на праздничные дни.


## Комментарии


Допустимые значения:


	- True. Поправка на праздничные
	 дни учитывается;


	- False. Поправка на праздничные
	 дни не учитывается.
