# IHolidays.Day


## Синтаксис


Day: Integer;


## Описание


Свойство Day определяет количество
 дней, на которые влияет праздничный день.


## Комментарии


Для определения, учитывается ли поправка на праздничные дни, используйте
 свойство [IHolidays.IsActive](IHolidays.IsActive.htm).

Использование свойства приведено в примере для [ISmx12arima.AdjustmentOptions](../ISmx12arima/ISmx12arima.AdjustmentOptions.htm).
