# ISlGDLTerms


Сборка: Stat;


## Описание


Интерфейс ISlGDLTerms используется
 для задания параметров расчёта распределенных лагов Койка.


## Иерархия наследования


           ISlGDLTerms


## Комментарии


Распределенные лаги Койка применяются в модели линейной регрессии (оценка
 МНК). Параметры расчёта распределенных лагов возвращает свойство [ISmLinearRegress.GDLTerms](../ISmLinearRegress/ISmLinearRegress.GDLTerms.htm).
