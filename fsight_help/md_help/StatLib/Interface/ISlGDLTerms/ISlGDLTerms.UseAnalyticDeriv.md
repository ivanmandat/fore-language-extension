# ISlGDLTerms.UseAnalyticDeriv


## Синтаксис


UseAnalyticDeriv: Boolean;


## Описание


Свойство UseAnalyticDeriv определяет,
 использовать ли аналитические производные.


## Комментарии


Допустимые значения:


	- True. Используются аналитические
	 производные;


	- False. Используются
	 численные производные.
