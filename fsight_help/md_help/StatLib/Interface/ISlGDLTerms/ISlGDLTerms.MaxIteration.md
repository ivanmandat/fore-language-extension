# ISlGDLTerms.MaxIteration


## Синтаксис


MaxIteration: Integer;


## Описание


Свойство MaxIteration определяет
 максимальное число итераций, используемых для расчёта распределенных лагов
 Койка.


## Комментарии


По умолчанию для расчёта используется 500 итераций.
