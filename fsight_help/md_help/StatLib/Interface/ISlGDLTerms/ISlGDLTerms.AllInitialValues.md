# ISlGDLTerms.AllInitialValues


## Синтаксис


AllInitialValues: Array;


## Описание


Свойство AllInitialValues определяет
 начальные значения коэффициентов модели.


## Комментарии


Количество коэффициентов должно совпадать с количеством экзогенных (объясняющих)
 переменных в модели.
