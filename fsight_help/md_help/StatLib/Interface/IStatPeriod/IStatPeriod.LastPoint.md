# IStatPeriod.LastPoint


## Синтаксис


LastPoint: Integer;


## Описание


Свойство LastPoint определяет
 конечную точку периода идентификации.


## Комментарии


Данная точка отсчитывается от единицы.


По умолчанию свойство имеет значение «-1», что соответствует автоматическому
 определению конечной точки периода идентификации.
