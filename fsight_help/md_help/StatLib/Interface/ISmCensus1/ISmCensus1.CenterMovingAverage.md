# ISmCensus1.CenterMovingAverage


## Синтаксис


CenterMovingAverage: Boolean;


## Описание


Свойство CenterMovingAverage
 определяет необходимость центрирования скользящего среднего.


## Комментарии


Допустимые значения:


	- True. Скользящее среднее будет центрировано;


	- False. Скользящее среднее не будет центрировано. Значение по умолчанию.
