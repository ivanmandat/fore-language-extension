# ISmCensus1.Seasonal


## Синтаксис


Seasonal: Array;


## Описание


Свойство Seasonal возвращает
 сезонную составляющую.


## Комментарий


При мультипликативной модели сезонности сезонная составляющая рассчитывается
 в процентах, при аддитивной - в тех же единицах, в которых задан исходный
 ряд.


Для задания типа модели сезонности используйте свойство [ISeasonal.Mode](../ISeasonal/ISeasonal.Mode.htm),
 для задания периода сезонности используйте свойство [ISeasonal.Cycle](../ISeasonal/ISeasonal.Cycle.htm).
