# ISmRManager.IsRExist


## Синтаксис


IsRExist: Boolean;


## Описание


Свойство IsRExist возвращает
 признак наличия на локальном рабочем месте установленного пакета R, на
 котором настроена интеграция с «Форсайт. Аналитическая платформа».


## Комментарии


Поиск установленного и интегрированного с «Форсайт. Аналитическая платформа»
 пакета R производится по пути, прописанному в системном реестре. Для указания
 пути к установленному и интегрированному с «Форсайт. Аналитическая платформа»
 пакету R используйте свойство [ISmRManager.UserRPath](ISmRManager.UserRPath.htm).


Допустимые значения:


	- True. Установленный
	 и интегрированный с «Форсайт. Аналитическая платформа»
	 пакет R найден на локальном рабочем месте;


	- False. Установленный
	 пакет R не найден на локальном рабочем месте, или не настроена интеграция
	 с «Форсайт. Аналитическая платформа».
