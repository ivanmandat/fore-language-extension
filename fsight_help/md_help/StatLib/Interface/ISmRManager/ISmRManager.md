# ISmRManager


Сборка: Stat;


## Описание


Интерфейс ISmRManager предназначен
 для задания пути к установленному и интегрированному с «Форсайт. Аналитическая платформа» пакету R.


## Иерархия наследования


           ISmRManager


## Комментарии


Для интеграции с R используйте интерфейс [ISmR](../ISmR/ISmR.htm).

		 признак наличия установленного и интегрированного с «Форсайт. Аналитическая платформа»

		 пакету R.
