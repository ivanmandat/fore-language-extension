# ISmRManager.Reset


## Синтаксис


Reset;


## Описание


Метод Reset закрывает текущее
 соединение с RServer и открывает новое.


## Комментарии


Для определения наличия установленного и интегрированного с «Форсайт. Аналитическая платформа» пакета R
 на локальном рабочем месте используйте свойство [ISmRManager.IsRExist](ISmRManager.IsRExist.htm).
