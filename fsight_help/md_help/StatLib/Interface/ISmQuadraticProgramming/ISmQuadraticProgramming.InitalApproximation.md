# ISmQuadraticProgramming.InitalApproximation


## Синтаксис


InitalApproximation: [ILinearProgrammingInitApproximation](../ILinearProgrammingInitApproximation/ILinearProgrammingInitApproximation.htm);


## Описание


Свойство InitialApproximation
 определяет параметры расчета начальных приближений.


## Комментарии


Необязательный параметр. По умолчанию начальные приближения подбираются
 автоматически.
