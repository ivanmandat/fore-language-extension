# ISmQuadraticProgramming.QuadraticForm


## Синтаксис


QuadraticForm: Array;


## Описание


Свойство QuadraticForm определяет
 квадратичную часть целевой функции (квадратичную форму).


## Комментарии


Индексация массива коэффициентов квадратичной формы должна начинаться
 с нуля.


Линейная часть целевой функции определяется свойством [ISmQuadraticProgramming.CriterionFunction](ISmQuadraticProgramming.CriterionFunction.htm).


## Пример


Использование свойства приведено в примерах для [ISmQuadraticProgramming.CriterionFunction](ISmQuadraticProgramming.CriterionFunction.htm).
