# ISlEquation.CointegralCoefficients


## Синтаксис


CointegralCoefficients: [ICoefficients](../ICoefficients/ICoefficients.htm);


## Описание


Свойство CointegralCoefficients
 возвращает коэффициенты коинтеграционных уравнений.


## Комментарии


Коэффициенты будут рассчитаны, если заданы коинтеграционные связи.
