# ISlEquation.Trend


## Синтаксис


Trend: [IIntercept](../IIntercept/IIntercept.htm);


## Описание


Свойство Trend возвращает
 параметры тренда уравнения.


## Комментарии


Параметры тренда учитываются при расчете модели коррекции ошибок с трендом
 и константой в авторегрессии и коинтеграционном уравнении, с квадратичным
 трендом в данных.
