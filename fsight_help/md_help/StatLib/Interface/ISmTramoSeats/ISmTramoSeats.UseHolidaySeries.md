# ISmTramoSeats.UseHolidaySeries


## Синтаксис


UseHolidaySeries: Boolean;


## Описание


Свойство UseHolidaySeries определяет,
 использовать ли в качестве поправки на праздничные/рабочие дни отдельный
 ряд.


## Комментарии


Допустимые значения:


	- True. Отдельный ряд
	 используется в качестве поправки на праздничные-рабочие дни;


	- False. Отдельный ряд
	 не используется в качестве поправки на праздничные-рабочие дни.
