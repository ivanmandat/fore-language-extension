# ISmTramoSeats.AR


## Синтаксис


AR: Integer;


## Описание


Свойство AR определяет порядок
 несезонной авторегрессии.


## Комментарии


Свойство актуально, если [ISmTramoSeats.ArimaOrderSearch](ISmTramoSeats.ArimaOrderSearch.htm) = ArimaOrderSearchType.FixOrder.
