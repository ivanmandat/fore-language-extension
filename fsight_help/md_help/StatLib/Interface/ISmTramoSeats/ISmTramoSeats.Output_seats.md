# ISmTramoSeats.Output_seats


## Синтаксис


Output_seats: Array;


## Описание


Свойство Output_seats возвращает
 массив строк, содержащий выходной файл, формируемый внешней программой
 SEATS.


## Комментарии


Для возвращения массива строк, содержащего выходной файл, формируемый
 внешней программой TRAMO, используйте свойство [ISmTramoSeats.Output_tramo](ISmTramoSeats.Output_tramo.htm).
