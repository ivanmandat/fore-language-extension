# ISmTramoSeats.MissingData


## Синтаксис


MissingData: [IMissingData](../IMissingData/IMissingData.htm);


## Описание


Свойство MissingData возвращает
 метод обработки пропусков, применяемый к исходным данным.


## Комментарии


По умолчанию используется метод [MissingDataMethod.None](../../Enums/MissingDataMethod.htm),
 т.е. обработка пропусков не выполняется.
