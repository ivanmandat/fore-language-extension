# ISmTramoSeats.EasterEffect


## Синтаксис


EasterEffect: [EasterEffectType](../../Enums/EasterEffectType.htm);


## Описание


Свойство EasterEffect определяет
 тип поправки на праздник Пасхи.


## Комментарии


Для определения, использовать ли в качестве поправки на праздничные/рабочие
 дни отдельный ряд, используйте свойство [ISmTramoSeats.UseHolidaySeries](ISmTramoSeats.UseHolidaySeries.htm).
