# ISmTramoSeats.HolidaySeries


## Синтаксис


HolidaySeries Array;


## Описание


Свойство HolidaySeries определяет
 ряд поправок на праздничные/рабочие дни.


## Комментарии


Свойство актуально, если [ISmTramoSeats.UseHolidaySeries](ISmTramoSeats.UseHolidaySeries.htm)
 = True.
