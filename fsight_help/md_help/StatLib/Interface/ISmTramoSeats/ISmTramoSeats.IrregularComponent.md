# ISmTramoSeats.IrregularComponent


## Синтаксис


IrregularComponent: Array;


## Описание


Свойство IrregularComponent
 возвращает ряд, содержащий нерегулярную компоненту.


## Комментарии


Для получения ряда, содержащего сезонный фактор, используйте свойство
 [ISmTramoSeats.SeasonalFactors](ISmTramoSeats.SeasonalFactors.htm).
