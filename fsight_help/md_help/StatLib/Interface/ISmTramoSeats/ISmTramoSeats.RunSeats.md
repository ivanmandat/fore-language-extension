# ISmTramoSeats.RunSeats


## Синтаксис


RunSeats: Boolean;


## Описание


Свойство RunSeats определяет,
 включить ли процедуру SEATS после выполнения TRAMO.


## Комментарии


Допустимые значения:


	- True. После выполнения
	 процедуры TRAMO процедура SEATS будет включена;


	- False. После выполнения
	 процедуры TRAMO процедура SEATS не будет включена.
