# ISmTramoSeats.TradingDayEffects


## Синтаксис


TradingDayEffects: [TradingDayEffectsType](../../Enums/TradingDayEffectType.htm);


## Описание


Свойство TradingDayEffects определяет
 тип поправок на рабочие дни.


## Комментарии


Для определения ряда поправок на праздничные/рабочие дни используйте
 свойство [ISmTramoSeats.HolidaySeries](ISmTramoSeats.HolidaySeries.htm).
