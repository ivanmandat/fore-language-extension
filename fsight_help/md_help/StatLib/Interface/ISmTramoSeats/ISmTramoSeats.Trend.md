# ISmTramoSeats.Trend


## Синтаксис


Trend: Array;


## Описание


Свойство Trend возвращает ряд,
 содержащий тренд-составляющую.


## Комментарии


Для получения ряда, содержащего циклическую составляющую, используйте
 свойство [ISmTramoSeats.Cycle](ISmTramoSeats.Cycle.htm).
