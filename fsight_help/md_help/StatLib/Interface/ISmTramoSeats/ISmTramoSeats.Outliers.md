# ISmTramoSeats.Outliers


## Синтаксис


Outliers: [ISlTramoSeatsOutliersCollection](../ISlTramoSeatsOutliersCollection/ISlTramoSeatsOutliersCollection.htm);


## Описание


Свойство Outliers возвращает
 коллекцию выбросов.


## Комментарии


Заданные выбросы учитываются в модели, если [ISmTramoSeats.OutliersDetection](ISmTramoSeats.OutliersDetection.htm)
 = OutliersType.None.
