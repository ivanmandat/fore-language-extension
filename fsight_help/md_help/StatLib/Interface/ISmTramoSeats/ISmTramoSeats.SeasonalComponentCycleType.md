# ISmTramoSeats.SeasonalComponentCycleType


## Синтаксис


SeasonalComponentCycleType: [SeasonalityCycleType](../../Enums/SeasonalityCycleType.htm);


## Описание


Свойство SeasonalComponentCycleType
 определяет период сезонности (месяцы/кварталы).


## Комментарии


Для получения ряда, содержащего сезонно скорректированный ряд, используйте
 свойство [ISmTramoSeats.SeasonallyAdjusted](ISmTramoSeats.SeasonallyAdjusted.htm).
