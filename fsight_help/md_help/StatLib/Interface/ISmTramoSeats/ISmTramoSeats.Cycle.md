# ISmTramoSeats.Cycle


## Синтаксис


Cycle: Array;


## Описание


Свойство Cycle возвращает ряд,
 содержащий циклическую составляющую.


## Комментарии


Для получения ряда, содержащего тренд-составляющую, используйте свойство
 [ISmTramoSeats.Trend](ISmTramoSeats.Trend.htm).
