# ISmTramoSeats.ArimaOrderSearch


## Синтаксис


ArimaOrderSearch: [ArimaOrderSearchType](../../Enums/ArimaOrderSearchType.htm);


## Описание


Свойство ArimaOrderSearch определяет
 способ спецификации модели ARIMA.


## Комментарии


Для определения порядка несезонного и сезонного дифференцирования используйте
 свойства [ISmTramoSeats.D](ISmTramoSeats.D.htm)
 и [ISmTramoSeats.SD](ISmTramoSeats.SD.htm),
 для определения порядка несезонной и сезонной авторегрессии используйте
 свойства [ISmTramoSeats.AR](ISmTramoSeats.AR.htm)
 и [ISmTramoSeats.SAR](ISmTramoSeats.SAR.htm),
 для определения порядка несезонного и сезонного скользящего среднего используйте
 свойства [ISmTramoSeats.MA](ISmTramoSeats.MA.htm)
 и [ISmTramoSeats.SMA](ISmTramoSeats.SMA.htm).
