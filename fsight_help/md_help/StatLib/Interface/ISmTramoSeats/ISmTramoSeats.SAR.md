# ISmTramoSeats.SAR


## Синтаксис


SAR: Integer;


## Описание


Свойство SAR определяет порядок
 сезонной авторегрессии.


## Комментарии


Свойство актуально, если [ISmTramoSeats.ArimaOrderSearch](ISmTramoSeats.ArimaOrderSearch.htm)
 = ArimaOrderSearchType.FixOrder.
