# ISmTramoSeats.TradingDayIfSignificant


## Синтаксис


TradingDayIfSignificant: Boolean;


## Описание


Свойство TradingDayIfSignificant
 определяет, включен ли автоматический режим учета поправок на рабочие
 дни.


## Комментарии


Для определения типа поправок на рабочие дни используйте свойство [ISmTramoSeats.TradingDayEffects](ISmTramoSeats.TradingDayEffects.htm).


Допустимые значения:


	- True. Автоматический
	 режим учета поправок на рабочие дни включен;


	- False. Автоматический
	 режим учета поправок на рабочие дни выключен.
