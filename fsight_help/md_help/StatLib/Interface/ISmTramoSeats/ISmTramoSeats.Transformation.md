# ISmTramoSeats.Transformation


## Синтаксис


Transformation: [TransformationLogLevelType](../../Enums/TransformationLogLevelType.htm);


## Описание


Свойство Transformation определяет
 вид преобразования исходного ряда.


## Комментарии


Значение по умолчанию TransformationLogLevelType.Yes.


Для автоматического преобразования исходного ряда используйте тип TransformationLogLevelType.Auto.
