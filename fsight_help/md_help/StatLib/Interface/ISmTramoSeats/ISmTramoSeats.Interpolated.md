# ISmTramoSeats.Interpolated


## Синтаксис


Interpolated: Array;


## Описание


Свойство Interpolated возвращает
 ряд, содержащий интерполированное значение исходного ряда.


## Комментарии


Для получения ряда, содержащего линеаризованное значение исходного ряда,
 используйте свойство [ISmTramoSeats.Linearized](ISmTramoSeats.Linearized.htm).
