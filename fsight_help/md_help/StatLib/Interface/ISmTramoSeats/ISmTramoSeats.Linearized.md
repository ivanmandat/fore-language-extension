# ISmTramoSeats.Linearized


## Синтаксис


Linearized: Array;


## Описание


Свойство Linearized возвращает
 ряд, содержащий линеаризованное значение исходного ряда.


## Комментарии


Для получения ряда, содержащего интерполированное значение исходного
 ряда, используйте свойство [ISmTramoSeats.Interpolated](ISmTramoSeats.Interpolated.htm).
