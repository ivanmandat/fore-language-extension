# ISmTramoSeats.ModelPeriod


## Синтаксис


ModelPeriod: [IStatPeriod](../IStatPeriod/IStatPeriod.htm);


## Описание


Свойство ModelPeriod возвращает
 настройки периода расчета модели.


## Комментарии


По умолчанию длина периода расчета модели совпадает с длиной объясняемого
 ряда. Для получения объясняемого ряда используйте свойство [ISmTramoSeats.Serie](ISmTramoSeats.Serie.htm).
