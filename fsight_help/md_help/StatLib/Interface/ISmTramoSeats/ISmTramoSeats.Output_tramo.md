# ISmTramoSeats.Output_tramo


## Синтаксис


Output_tramo: Array;


## Описание


Свойство Output_tramo возвращает
 массив строк, содержащий выходной файл, формируемый внешней программой
 TRAMO.


## Комментарии


Для возвращения массива строк, содержащего выходной файл, формируемый
 внешней программой SEATS, используйте свойство [ISmTramoSeats.Output_seats](ISmTramoSeats.Output_seats.htm).
