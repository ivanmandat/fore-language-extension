# ISmFillGapsProcedure


Сборка: Stat;


## Описание


Интерфейс ISmFillGapsProcedure
 определяет параметры обработки пропусков рядов данных.


## Иерархия наследования


           [IStatMethod](../IStatMethod/IStatMethod.htm)


           ISmFillGapsProcedure


## Комментарии


Пропуском в массиве данных является пустое значение - число равное «1#.QNAN». Получить его можно при
 помощи свойства [Double.Nan](ForeSys.chm::/Class/Double/Double.Nan.htm),
 проверить значение на пропуск можно при помощи метода [Double.IsNan](ForeSys.chm::/Class/Double/Double.IsNan.htm).
