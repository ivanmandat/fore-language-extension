# ISmGeometricExtrapolation


Сборка: Stat;


## Описание


Интерфейс ISmGeometricExtrapolation
 определяет параметры геометрической экстраполяции.


## Иерархия наследования


           [IStatMethod](../IStatMethod/IStatMethod.htm)


           ISmGeometricExtrapolation


## Комментарии


Экстраполяция - это особый тип аппроксимации (приближения), при котором
 функция аппроксимируется не между заданными значениями, а вне заданного
 интервала.
