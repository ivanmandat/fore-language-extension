# IStatistics.StDev


## Синтаксис


StDev(Values: Array): Double;


## Параметры


Values. Массив данных, соответствующих
 выборке из генеральной совокупности.


Примечание.
 Если данные представляют всю генеральную совокупность, то стандартное
 отклонение следует вычислять с помощью функции [StDevP](IStatistics.StDevP.htm).


## Описание


Метод StDev оценивает стандартное
 отклонение по выборке.


## Комментарии


Стандартное отклонение - это мера того, насколько широко разбросаны
 точки данных относительно их среднего. Стандартное отклонение вычисляется
 с использованием «несмещенного» или «n-1» метода.


## Пример


Для выполнения примера добавьте ссылку на системную сборку Stat.


			Sub UserProc;

Var

    st: Statistics;

    d0: Double;

    y: Array Of Double;

Begin

    y := New Double[10];

    y[00] := 6;

    y[01] := 17;

    y[02] := 9;

    y[03] := 15;

    y[04] := 21;

    y[05] := 24;

    y[06] := 20;

    y[07] := 21;

    y[08] := 14;

    y[09] := 25;

    st := New Statistics.Create;

    d0 := st.StDev(y);

    Debug.WriteLine("Стандартное отклонение: " + d0.ToString);

End Sub UserProc;


После выполнения примера в окно консоли будет выведено значение стандартного
 отклонения:


Выполнение модуля начато


Стандартное отклонение: 6.25033332444492


Выполнение модуля завершено
