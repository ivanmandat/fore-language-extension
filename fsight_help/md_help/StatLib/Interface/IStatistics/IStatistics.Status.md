# IStatistics.Status


## Синтаксис


Status: Integer;


## Описание


Свойство Status возвращает статус
 выполнения расчета.


## Комментарии


В случае выполнения расчета без ошибок данное свойство возвращает значение
 «0», иначе возвращает код ошибки.
