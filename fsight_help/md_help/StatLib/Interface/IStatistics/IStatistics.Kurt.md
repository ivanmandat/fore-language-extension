# IStatistics.Kurt


## Синтаксис


Kurt(Values: Array): Double;


## Параметры


Values. Ряд данных.


## Описание


Метод Kurt возвращает эксцесс
 множества данных.


## Комментарии


Эксцесс характеризует относительную остроконечность или сглаженность
 распределения по сравнению с нормальным распределением. Положительный
 эксцесс обозначает относительно остроконечное распределение. Отрицательный
 эксцесс обозначает относительно сглаженное распределение.


Для корректного расчета ряд данных Values должен
 быть неконстантный и содержать больше трех точек.


## Пример


Для выполнения примера добавьте ссылку на системную сборку Stat.


			Sub UserProc;

Var

    st: Statistics;

    d0: Double;

    y: Array Of Double;

Begin

    y := New Double[10];

    y[00] := 1.6;

    y[01] := 1.7;

    y[02] := 1.8;

    y[03] := 1.9;

    y[04] := 2;

    y[05] := 2.1;

    y[06] := 2.2;

    y[07] := 2.3;

    y[08] := 2.4;

    y[09] := 2.8;

    st := New Statistics.Create;

    d0 := st.kurt(y);

    Debug.WriteLine("Эксцесс: " + d0.ToString);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен эксцесс.
