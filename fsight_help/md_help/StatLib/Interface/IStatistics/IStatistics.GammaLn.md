# IStatistics.GammaLn


## Синтаксис


GammaLn(Value: Double): Double;


## Параметры


Value. Значение, для которого вычисляется GammaLn. Значение должно удовлетворять ограничению: Value > 0.


## Описание


Метод GammaLn возвращает натуральный логарифм гамма-функции.


## Комментарии


Значение параметра Value должно быть больше нуля.


Число e, возведенное в степень GammaLn(i), где i - целое число, возвращает такой же результат, как и (i - 1)!.
