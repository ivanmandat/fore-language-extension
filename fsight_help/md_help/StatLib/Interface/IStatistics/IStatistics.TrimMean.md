# IStatistics.TrimMean


## Синтаксис


TrimMean(Data: Array; Percent: Double): Double;


## Параметры


Data. Массив усредняемых значений;


Percent. Доля точек данных,
 исключаемых из вычислений. Допустимые значения берутся из диапазона [0, 1].


## Описание


Метод TrimMean возвращает среднее
 внутренности множества данных.


## Комментарии


Функция вычисляет среднее, отбрасывая заданный процент данных с экстремальными
 значениями. Можно использовать эту функцию, чтобы исключить из анализа
 выбросы.


Функция округляет количество выбрасываемых точек данных с недостатком
 до ближайшего целого, кратного «2». Если доля равна «0,1», то 10 процентов
 от 30 точек данных составляют 3 точки, но из соображений симметрии функция
 TrimMean исключит по одному значению
 из начала и конца множества.


## Пример


Для выполнения примера добавьте ссылку на системную сборку Stat.


			Sub UserProc;

Var

    st: Statistics;

    d0: Double;

    y: Array Of Double;

Begin

    y := New Double[10];

    y[00] := 6;

    y[01] := 17;

    y[02] := 9;

    y[03] := 15;

    y[04] := 21;

    y[05] := 24;

    y[06] := 20;

    y[07] := 21;

    y[08] := 14;

    y[09] := 25;

    st := New Statistics.Create;

    d0 := st.TrimMean(y,0.2);

    Debug.WriteLine("Среднее внутренности множества: " + d0.ToString);

End Sub UserProc;


После выполнения примера в окно консоли будет выведено среднее внутренности
 массива данных с исключением 20 процентов данных из вычислений:


Выполнение модуля начато


Среднее внутренности множества: 17.2


Выполнение модуля завершено
