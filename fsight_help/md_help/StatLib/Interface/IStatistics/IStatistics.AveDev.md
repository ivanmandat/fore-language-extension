# IStatistics.AveDev


## Синтаксис


AveDev(Value: Array): Double;


## Параметры


Value. Набор чисел, для которых
 определяется среднее абсолютных отклонений.


## Описание


Метод AveDev возвращает среднее
 абсолютных значений отклонений точек данных от среднего.


## Комментарии


Данное значение является мерой разброса множества данных.


## Пример


Для выполнения примера добавьте ссылку на системную сборку Stat.


			Sub UserProc;

Var

    st: Statistics;

    d0: Double;

    y: Array Of Double;

Begin

    y := New Double[10];

    y[00] := 1.6;

    y[01] := 1.7;

    y[02] := 1.8;

    y[03] := 1.9;

    y[04] := 2;

    y[05] := 2.1;

    y[06] := 2.2;

    y[07] := 2.3;

    y[08] := 2.4;

    y[09] := 2.8;

    st := New Statistics.Create;

    d0 := st.AveDev(y);

    Debug.WriteLine("Среднее абсолютных значений отклонений: " + d0.ToString);

End Sub UserProc;


После выполнения примера в окно консоли будет выведена мера разброса
 множества данных:


Выполнение модуля начато


Среднее абсолютных значений отклонений: 0.28


Выполнение модуля завершено
