# IStatistics.Large


## Синтаксис


Large(Data: Array; Position: Integer): Double;


## Параметры


Data - массив данных, для которых
 определяется k-ое наибольшее значение. Длина массива должна быть больше
 нуля (0);


Position - позиция (начиная
 с наибольшей) в массиве данных. Допустимые значения берутся из диапазона
 (0, длина Data].


## Описание


Метод Large возвращает k-ое
 по величине значение из множества данных. Эта функция позволяет выбрать
 значение по его относительному местоположению.


## Пример


Для выполнения примера добавьте ссылку на системную сборку Stat.


			Sub UserProc;

Var

    st: Statistics;

    d0: Double;

    y: Array Of Double;

Begin

    y := New Double[7];

    y[00] := 6;

    y[01] := 7;

    y[02] := 9;

    y[03] := 15;

    y[04] := 21;

    y[05] := 24;

    y[06] := 25;

    st := New Statistics.Create;

    d0 := st.Large(y,3);

    Debug.WriteLine("Третье по величине число: " + d0.ToString);

    d0 := st.Large(y,6);

    Debug.WriteLine("Шестое по величине число: " + d0.ToString);

End Sub UserProc;


После выполнения примера в окно консоли будут выведены результаты:


Выполнение модуля начато


Третье по величине число: 21


Шестое по величине число: 7


Выполнение модуля завершено
