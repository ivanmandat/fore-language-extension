# IStatistics.Skew


## Синтаксис


Skew(Values: Array): Double;


## Параметры


Values. Массив аргументов,
 для которых вычисляется асимметрия.


## Описание


Метод Skew возвращает асимметрию
 распределения.


## Комментарии


Асимметрия характеризует степень несимметричности распределения относительно
 его среднего. Положительная асимметрия указывает на отклонение распределения
 в сторону положительных значений. Отрицательная асимметрия указывает на
 отклонение распределения в сторону отрицательных значений.


## Пример


Для выполнения примера добавьте ссылку на системную сборку Stat.


			Sub UserProc;

Var

    st: Statistics;

    d0: Double;

    y: Array Of Double;

Begin

    y := New Double[10];

    y[00] := 1.6;

    y[01] := 1.7;

    y[02] := 1.8;

    y[03] := 1.9;

    y[04] := 2;

    y[05] := 2.1;

    y[06] := 2.2;

    y[07] := 2.3;

    y[08] := 2.4;

    y[09] := 2.8;

    st := New Statistics.Create;

    d0 := st.Skew(y);

    Debug.WriteLine("Асимметрия: " + d0.ToString);

End Sub UserProc;


После выполнения примера в окно консоли будет выведена асимметрия:


Выполнение модуля начато


Асимметрия: 0.660552553216138


Выполнение модуля завершено
