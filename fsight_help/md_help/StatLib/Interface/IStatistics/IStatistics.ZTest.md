# IStatistics.ZTest


## Синтаксис


ZTest(Data: Array; Value: Double; Sigma: Double):
 Double;


## Параметры


Data. Массив данных, с которыми
 сравнивается Value;


Value. Проверяемое значение;


Sigma. Известное стандартное
 отклонение генеральной совокупности.


## Описание


Метод ZTest возвращает двустороннее
 P-значение z-теста.


## Комментарии


Z-тест определяет стандартную оценку для Value
 по отношению к массиву данных и возвращает двустороннюю вероятность для
 нормального распределения. Можно использовать эту функцию, чтобы оценить
 вероятность того, что конкретное наблюдение взято из конкретной генеральной
 совокупности.


## Пример


Для выполнения примера добавьте ссылку на системную сборку Stat.


			Sub UserProc;

Var

    st: Statistics;

    d0: Double;

    y: Array Of Double;

Begin

    y := New Double[10];

    y[00] := 6;

    y[01] := 17;

    y[02] := 9;

    y[03] := 15;

    y[04] := 21;

    y[05] := 24;

    y[06] := 20;

    y[07] := 21;

    y[08] := 14;

    y[09] := 25;

    st := New Statistics.Create;

    d0 := st.ZTest(y,15,4);

    Debug.WriteLine("Z-тест: " + d0.ToString);

End Sub UserProc;


После выполнения примера в окно консоли будет выведено P-значение z-теста:


Выполнение модуля начато


Z-тест: 0.0409951605002064


Выполнение модуля завершено
