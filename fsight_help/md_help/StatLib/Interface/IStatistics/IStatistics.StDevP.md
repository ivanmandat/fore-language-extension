# IStatistics.StDevP


## Синтаксис


StDevP(Values: Array): Double;


## Параметры


Values. Массив данных, образующих
 всю генеральную совокупность.


## Описание


Метод StDevP вычисляет стандартное
 отклонение по генеральной совокупности.


## Комментарии


Стандартное отклонение - это мера того, насколько широко разбросаны
 точки данных относительно их среднего. Стандартное отклонение вычисляется
 с использованием «смещенного» или «n» метода.


## Пример


Для выполнения примера добавьте ссылку на системную сборку Stat.


			Sub UserProc;

Var

    st: Statistics;

    d0: Double;

    y: Array Of Double;

Begin

    y := New Double[10];

    y[00] := 6;

    y[01] := 17;

    y[02] := 9;

    y[03] := 15;

    y[04] := 21;

    y[05] := 24;

    y[06] := 20;

    y[07] := 21;

    y[08] := 14;

    y[09] := 25;

    st := New Statistics.Create;

    d0 := st.StDevP(y);

    Debug.WriteLine("Стандартное отклонение: " + d0.ToString);

End Sub UserProc;


После выполнения примера в окно консоли будет выведено значение стандартного
 отклонения:


Выполнение модуля начато


Стандартное отклонение: 5.92958683214944


Выполнение модуля завершено
