# IStatistics.HarMean


## Синтаксис


HarMean(Values: Array): Double;


## Параметры


Values. Множество аргументов,
 для которых вычисляется среднее.


## Описание


Метод HarMean возвращает среднее
 гармоническое множества данных.


## Комментарии


Среднее гармоническое - это величина, обратная к среднему арифметическому
 обратных величин. Среднее гармоническое всегда меньше среднего геометрического,
 которое всегда меньше среднего арифметического.


## Пример


Для выполнения примера добавьте ссылку на системную сборку Stat.


			Sub UserProc;

Var

    st: Statistics;

    d0: Double;

    y: Array Of Double;

Begin

    y := New Double[10];

    y[00] := 1.6;

    y[01] := 1.7;

    y[02] := 1.8;

    y[03] := 1.9;

    y[04] := 2;

    y[05] := 2.1;

    y[06] := 2.2;

    y[07] := 2.3;

    y[08] := 2.4;

    y[09] := 2.8;

    st := New Statistics.Create;

    d0 := st.HarMean(y);

    Debug.WriteLine("Среднее гармоническое: " + d0.ToString);

End Sub UserProc;


После выполнения примера в окно консоли будет выведено значение среднего:


Выполнение модуля начато


Среднее гармоническое: 2.02657459326811


Выполнение модуля завершено
