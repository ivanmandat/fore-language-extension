# IStatistics.PercentRank


## Синтаксис


PercentRank(Data: Array; Value: Double; Significance:
 Integer): Double;


## Параметры


Data. Массив данных, который
 определяет относительное положение;


Value. Значение, для которого
 определяется процентное содержание;


Significance. Необязательное
 значение, определяющее количество значащих цифр для возвращаемого процентного
 значения. Значение должно удовлетворять ограничению: Significance >= 1.


## Описание


Метод PercentRank возвращает
 категорию значения в наборе данных как процентное содержание в наборе
 данных.


## Комментарии


Эта функция используется для оценки относительного положения точки данных
 в множестве данных.


Если заданный Value не соответствует
 ни одному из значений аргумента Data,
 то функция PercentRank производит
 интерполяцию и возвращает корректное значение процентного содержания.


## Пример


Для выполнения примера добавьте ссылку на системную сборку Stat.


			Sub UserProc;

Var

    st: Statistics;

    d0: Double;

    y: Array Of Double;

Begin

    y := New Double[7];

    y[00] := 6;

    y[01] := 7;

    y[02] := 9;

    y[03] := 15;

    y[04] := 21;

    y[05] := 24;

    y[06] := 25;

    st := New Statistics.Create;

    d0 := st.PercentRank(y,7,3);

    Debug.WriteLine("Ранг в
 %: " + d0.ToString);

End Sub UserProc;


После выполнения примера в окно консоли будет выведен представленный
 в виде процента ранг числа «7» в приведенном массиве данных:


Выполнение модуля начато


Ранг в %: 0.166


Выполнение модуля завершено
