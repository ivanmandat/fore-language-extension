# IStatistics.Correl


## Синтаксис


Correl(A1: Array; A2: Array): Double;


## Параметры


A1. Первый ряд данных;


A2. Второй ряд данных.


## Описание


Метод Correl возвращает коэффициент
 корреляции между двумя рядами.


## Комментарии


Коэффициент корреляции используется для определения наличия взаимосвязи
 между двумя свойствами.


Для корректного расчета ряды A1
 и A2:


	- должны иметь равное количество точек;


	- должны быть неконстантными.


## Пример


Для выполнения примера добавьте ссылку на системную сборку Stat.


			Sub UserProc;

Var

    st: Statistics;

    d0: Double;

    y, x: Array Of Double;

Begin

    y := New Double[10];

    y[00] := 1.6; y[05] := 2.1;

    y[01] := 1.7; y[06] := 2.2;

    y[02] := 1.8; y[07] := 2.3;

    y[03] := 1.9; y[08] := 2.4;

    y[04] := 2; y[09] := 2.8;

    x := New Double[10];

    x[00] := 2; x[05] := 6;

    x[01] := 4; x[06] := 15;

    x[02] := 2; x[07] := 17;

    x[03] := 5; x[08] := 14;

    x[04] := 12; x[09] := 3;

    st := New Statistics.Create;

    d0 := st.Correl(y, x);

    Debug.WriteLine("Коэффициент корреляции: " + d0.ToString);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведено значение
 коэффициента корреляции.
