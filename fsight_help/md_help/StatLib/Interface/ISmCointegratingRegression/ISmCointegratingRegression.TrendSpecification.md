# ISmCointegratingRegression.TrendSpecification


## Синтаксис


TrendSpecification: [TrendSpecificationType](../../Enums/TrendSpecificationType.htm);


## Описание


Свойство TrendSpecification
 определяет спецификации основного и дополнительного трендов.


## Комментарии


Основной тренд задается для основной модели, дополнительный тренд задается
 для уравнений коинтегрирующих регрессоров.
