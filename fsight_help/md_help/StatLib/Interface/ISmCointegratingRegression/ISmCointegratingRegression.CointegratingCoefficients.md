# ISmCointegratingRegression.CointegratingCoefficients


## Синтаксис


CointegratingCoefficients: [ICoefficients](../ICoefficients/ICoefficients.htm);


## Описание


Свойство CointegratingCoefficients
 возвращает оценки коинтегрирующих коэффициентов модели и их характеристики.


## Комментарии


Для получения детерминированных коэффициентов модели используйте свойство
 [ISmCointegratingRegression.DeterministicCoefficients](ISmCointegratingRegression.DeterministicCoefficients.htm).
