# ISmCointegratingRegression.CointegratingRegressors


## Синтаксис


CointegratingRegressors: [ISlSeries](../ISlSeries/ISlSeries.htm);


## Описание


Свойство CointegratingRegressors
 возвращает коинтегрирующие регрессоры.


## Комментарии


Для получения детерминированных регрессоров используйте свойство [ISmCointegratingRegression.DeterministicRegressors](ISmCointegratingRegression.DeterministicRegressors.htm).
