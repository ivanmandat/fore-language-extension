# ISmCointegratingRegression.DeterministicCoefficients


## Синтаксис


DeterministicCoefficients: [ICoefficients](../ICoefficients/ICoefficients.htm);


## Описание


Свойство DeterministicCoefficients
 возвращает оценки детерминированных коэффициентов модели и их характеристики.


## Комментарии


Для получения коинтегрирующих коэффициентов модели используйте свойство
 [ISmCointegratingRegression.CointegratingCoefficients](ISmCointegratingRegression.CointegratingCoefficients.htm).
