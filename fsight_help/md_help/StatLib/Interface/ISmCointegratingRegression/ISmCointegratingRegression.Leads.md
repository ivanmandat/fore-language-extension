# ISmCointegratingRegression.Leads


## Синтаксис


Leads: Integer;


## Описание


Свойство Leads определяет количество
 опережений для метода DOLS.


## Комментарии


Метод DOLS (динамический метод наименьших квадратов) можно установить
 при помощи функции [ISmCointegratingRegression.EstimationMethod](ISmCointegratingRegression.EstimationMethod.htm)
 = CREstimationMethodType.DOLS.


Количество опережений должно быть больше либо равно нулю.
