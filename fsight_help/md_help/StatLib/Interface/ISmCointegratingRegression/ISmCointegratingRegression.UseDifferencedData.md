# ISmCointegratingRegression.UseDifferencedData


## Синтаксис


UseDifferencedData: Boolean;


## Описание


Свойство UseDifferencedData
 определяет, использовать ли конечные разности.


## Комментарии


Допустимые значения:


	- True. Конечные разности
	 используются;


	- False. Конечные разности
	 не используются.
