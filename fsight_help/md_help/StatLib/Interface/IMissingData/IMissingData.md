# IMissingData


Сборка: Stat;


## Описание


Интерфейс IMissingData определяет
 метод и параметры обработки пропусков.


## Иерархия наследования


           IMissingData


## Комментарии


В блоке статистики пропуском в массиве данных считается число равное
 1#.QNAN. Получить его можно при помощи свойства Double.Nan, проверить
 значение на пропуск можно при помощи метода Double.IsNan.
