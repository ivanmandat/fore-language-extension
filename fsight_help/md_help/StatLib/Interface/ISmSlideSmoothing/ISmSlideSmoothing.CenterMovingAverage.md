# ISmSlideSmoothing.CenterMovingAverage


## Синтаксис


CenterMovingAverage: Boolean;


## Описание


Свойство CenterMovingAverage
 определяет, будет ли центрировано скользящее среднее.


## Комментарии


Допустимые значения:


	- True. Скользящее среднее
	 будет центрировано;


	- False. Скользящее среднее
	 не будет центрировано. Значение по умолчанию.
