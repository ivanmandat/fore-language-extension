# ISmSlideSmoothing


Сборка: Stat;


## Описание


Интерфейс ISmSlideSmoothing
 определяет параметры скользящего среднего.


## Иерархия наследования


           [IStatMethod](../IStatMethod/IStatMethod.htm)


           ISmSlideSmoothing


## Комментарии


Метод скользящего среднего основан на представлении ряда в виде суммы
 достаточно гладкого тренда и случайной компоненты.
