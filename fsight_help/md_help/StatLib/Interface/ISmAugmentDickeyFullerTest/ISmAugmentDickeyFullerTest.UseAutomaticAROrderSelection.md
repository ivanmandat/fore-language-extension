# ISmAugmentDickeyFullerTest.UseAutomaticAROrderSelection


## Синтаксис


UseAutomaticAROrderSelection: Boolean;


## Описание


Свойство UseAutomaticAROrderSelection
 определяет, применять ли автоподбор лага.


## Комментарии


Для определения параметров автоподбора лага используйте свойство [ISmAugmentDickeyFullerTest.AutomaticAROrderSelectionSettings](ISmAugmentDickeyFullerTest.AutomaticAROrderSelectionSettings.htm).


Допустимые значения:


	- True. Автоподбор лага
	 применяется;


	- False. Автоподбор лага
	 не применяется.
