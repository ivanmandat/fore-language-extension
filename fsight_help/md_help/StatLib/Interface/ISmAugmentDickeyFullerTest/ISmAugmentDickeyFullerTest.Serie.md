# ISmAugmentDickeyFullerTest.Serie


## Синтаксис


Serie: [ISlSerie](../ISlSerie/ISlSerie.htm);


## Описание


Свойство Serie определяет тестируемый ряд данных.


## Комментарии


Если не задан исследуемый ряд, то корректный расчёт расширенного теста Дики-Фуллера невозможен.


Длина ряда должна быть не менее 15 элементов.
