## Описание


Свойство AutomaticAROrderSelectionSettings
 возвращает параметры автоподбора лага.


## Комментарии


Для определения, применяется ли автоподбор лага, используйте свойство
 [ISmAugmentDickeyFullerTest.UseAutomaticAROrderSelection](ISmAugmentDickeyFullerTest.UseAutomaticAROrderSelection.htm).
