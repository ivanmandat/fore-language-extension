# ISmAugmentDickeyFullerTest


Сборка: Stat;


## Описание


Интерфейс ISmAugmentDickeyFullerTest
 определяет параметры расширенного теста Дики-Фуллера.


## Иерархия наследования


           [IStatMethod](../IStatMethod/IStatMethod.htm)


           ISmAugmentDickeyFullerTest


## Комментарии


Расширенный тест возвращает статистику Дики-Фуллера. Тест является средством
 проверки стационарности временного ряда.
