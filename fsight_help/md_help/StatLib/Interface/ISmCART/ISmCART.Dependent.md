# ISmCART.Dependent


## Синтаксис


Dependent: [ISlSerieInt](../ISlSeriesInt/ISlSeriesInt.htm);


## Описание


Свойство Dependent возвращает
 объясняемый ряд.


## Комментарии


Объясняемый ряд является категориальным.


Для задания объясняющих рядов используйте свойства [ISmCART.ExplanatoriesCategorical](ISmCART.ExplanatoriesCategorical.htm),
 [ISmCART.ExplanatoriesContinuous](ISmCART.ExplanatoriesContinuous.htm)
 и [ISmCART.ExplanatoriesOrdered](ISmCART.ExplanatoriesOrdered.htm).
