# ISmCART.ExplanatoriesContinuous


## Синтаксис


ExplanatoriesContinuous: [ISlSeries](../ISlSeries/ISlSeries.htm);


## Описание


Свойство ExplanatoriesContinuous
 возвращает объясняющие количественные ряды.


## Комментарии


Для задания объясняемого ряда используйте свойство [ISmCART.Dependent](ISmCART.Dependent.htm), для задания
 объясняющих категориальных рядов - свойство [ISmCART.ExplanatoriesCategorical](ISmCART.ExplanatoriesCategorical.htm),
 а для задания объясняющих порядковых рядов - свойство [ISmCART.ExplanatoriesOrdered](ISmCART.ExplanatoriesOrdered.htm).


Количественный (вещественный) ряд - набор данных, элементы которого
 можно упорядочить, и между которыми можно найти расстояние. Например,
 расход топлива автомобилей. Можно упорядочить и сказать, у кого больше,
 а у кого меньше.
