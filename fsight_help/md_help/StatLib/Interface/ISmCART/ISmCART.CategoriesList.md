# ISmCART.CategoriesList


## Синтаксис


CategoriesList: Array;


## Описание


Свойство CategoriesList возвращает
 список категорий объясняемого ряда.


## Комментарии


Свойство CategoriesList
 содержит все уникальные значения из ряда [ISmCART.Dependent](ISmCART.Dependent.htm), расположенные
 в порядке возрастания.


Для получения итоговой классификации объектов используйте свойство [ISmCART.ClassificationSummary](ISmCART.ClassificationSummary.htm).
