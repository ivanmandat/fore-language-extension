# ISmNonLinearLeastSquare.OptimizationMethod


## Синтаксис


OptimizationMethod: [NLSOptimizationMethod](../../Enums/NLSOptimizationMethod.htm);


## Описание


Свойство OptimizationMethod
 определяет используемый метод оптимизации.


## Комментарии


По умолчанию используется метод Левенберга-Марквардта, т.е. OptimizationMethod = NLSOptimizationMethod.LevenbergMarquardt.
