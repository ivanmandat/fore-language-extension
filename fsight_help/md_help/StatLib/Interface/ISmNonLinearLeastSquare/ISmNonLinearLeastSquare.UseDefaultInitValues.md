# ISmNonLinearLeastSquare.UseDefaultInitValues


## Синтаксис


UseDefaultInitValues: Boolean;


## Описание


Свойство UseDefaultInitValues
 определяет, использовать ли начальные приближения, заданные по умолчанию.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Для расчёта используются начальные приближения, заданные по умолчанию;


	- False. Для расчёта используются
	 начальные приближения, заданные в свойстве [ISmNonLinearLeastSquare.InitApproximation](ISmNonLinearLeastSquare.InitApproximation.htm).
