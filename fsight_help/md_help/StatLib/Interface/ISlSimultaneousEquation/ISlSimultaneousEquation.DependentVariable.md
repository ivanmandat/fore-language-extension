# ISlSimultaneousEquation.DependentVariable


## Синтаксис


DependentVariable: [ISlSerie](../ISlSerie/ISlSerie.htm);


## Описание


Свойство DependentVariable определяет параметры зависимой переменной в уравнении.


## Комментарии


Для определения параметров эндогенных и экзогенных переменных в уравнении используйте свойства [ISlSimultaneousEquation.EndogenousVariables](ISlSimultaneousEquation.EndogenousVariables.htm) и [ISlSimultaneousEquation.ExogenousVariables](ISlSimultaneousEquation.ExogenousVariables.htm).
