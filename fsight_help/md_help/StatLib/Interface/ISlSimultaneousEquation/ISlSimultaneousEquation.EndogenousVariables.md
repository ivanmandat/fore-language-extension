# ISlSimultaneousEquation.EndogenousVariables


## Синтаксис


EndogenousVariables: [ISlSeries](../ISlSeries/ISlSeries.htm);


## Описание


Свойство EndogenousVariables определяет параметры эндогенных переменных в уравнении.


## Комментарии


Для определения параметров экзогенных переменных в уравнении используйте свойство [ISlSimultaneousEquation.ExogenousVariables](ISlSimultaneousEquation.ExogenousVariables.htm).
