# ISlSimultaneousEquation.ExogenousCoefficients


## Синтаксис


ExogenousCoefficients: [ICoefficients](../ICoefficients/ICoefficients.htm);


## Описание


Свойство ExogenousCoefficients возвращает оценки и характеристики коэффициентов экзогенных переменных уравнения.


## Комментарии


Для получения параметров коэффициентов эндогенных переменных уравнения используйте свойство [ISlSimultaneousEquation.EndogenousCoefficients](ISlSimultaneousEquation.EndogenousCoefficients.htm).
