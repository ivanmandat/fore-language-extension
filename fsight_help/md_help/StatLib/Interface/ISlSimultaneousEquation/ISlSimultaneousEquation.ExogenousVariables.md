# ISlSimultaneousEquation.ExogenousVariables


## Синтаксис


ExogenousVariables: [ISlSeries](../ISlSeries/ISlSeries.htm);


## Описание


Свойство ExogenousVariables определяет параметры экзогенных переменных в уравнении.


## Комментарии


Для определения параметров эндогенных переменных в уравнении используйте свойство [ISlSimultaneousEquation.EndogenousVariables](ISlSimultaneousEquation.EndogenousVariables.htm).
