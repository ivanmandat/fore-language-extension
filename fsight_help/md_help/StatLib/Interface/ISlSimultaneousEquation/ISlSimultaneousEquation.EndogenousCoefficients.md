# ISlSimultaneousEquation.EndogenousCoefficients


## Синтаксис


EndogenousCoefficients: [ICoefficients](../ICoefficients/ICoefficients.htm);


## Описание


Свойство EndogenousCoefficients возвращает оценки и характеристики коэффициентов эндогенных переменных уравнения.


## Комментарии


Для получения параметров коэффициентов эндогенных переменных в уравнении используйте свойство [ISlSimultaneousEquation.ExogenousCoefficients](ISlSimultaneousEquation.ExogenousCoefficients.htm).
