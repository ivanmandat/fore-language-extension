# ISlSimultaneousEquation


Сборка: Stat;


## Описание


Интерфейс ISlSimultaneousEquation
 определяет параметры одновременного уравнения.


## Иерархия наследования


           ISlSimultaneousEquation


## Комментарии


Система одновременных уравнений - совокупность эконометрических
 уравнений (часто линейных), определяющих взаимозависимость эконометрических
 переменных.
