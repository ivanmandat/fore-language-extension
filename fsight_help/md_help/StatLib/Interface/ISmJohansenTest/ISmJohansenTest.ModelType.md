# ISmJohansenTest.ModelType


## Синтаксис


ModelType: [ECMType](../../Enums/ECMType.htm);


## Описание


Свойство ModelType определяет
 тип модели коррекции ошибок.


## Комментарии


По умолчанию свойство имеет значение ECMType.NoTrendNoIntercept,
 то есть тип «Модель без тренда или константы в коинтеграционном уравнении
 и авторегрессии».
