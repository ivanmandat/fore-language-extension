# ISlLinearConstraints


## Описание


Интерфейс ISlLinearConstraints
 содержит свойства и методы для работы с коллекцией линейных ограничений.


## Комментарии


Линейные ограничения используются в решении задач [линейного](../ISmLinearProgramming/ISmLinearProgramming.htm)
 и [квадратичного](../ISmQuadraticProgramming/ISmQuadraticProgramming.htm)
 программирования и при [оптимизации
 функции](../ISmNonLinearOptimization/ISmNonLinearOptimization.htm) произвольного вида при нелинейных ограничениях. Для получения
 коллекции линейных ограничений используйте свойства:


	- [ISmLinearProgramming.LinearConstraints](../ISmLinearProgramming/ISmLinearProgramming.LinearConstraints.htm);


	- [ISmNonLinearOptimization.LinearConstraints](../ISmNonLinearOptimization/ISmNonLinearOptimization.LinearConstraints.htm);


	- [ISmQuadraticProgramming.LinearConstraints](../ISmQuadraticProgramming/ISmQuadraticProgramming.LinearConstraints.htm).
