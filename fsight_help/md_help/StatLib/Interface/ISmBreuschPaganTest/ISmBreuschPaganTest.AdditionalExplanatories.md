# ISmBreuschPaganTest.AdditionalExplanatories


## Синтаксис


AdditionalExplanatories: [ISlSeries](../ISlSeries/ISlSeries.htm);


## Описание


Свойство AdditionalExplanatories
 возвращает объясняющие ряды для вспомогательной регрессии.


## Комментарии


В качестве объясняющих рядов вспомогательной регрессии следует использовать
 ряды, коррелирующие с объясняемым рядом, но не коррелирующие друг с другом.
