# ISmFisherTest.FisherTestResult


## Синтаксис


FisherTestResult: Boolean;


## Описание


Свойство FisherTestResult возвращает
 результат принятия гипотезы.


## Комментарии


Допустимые значения:


	- True. Гипотеза
	 о равенстве дисперсий принимается;


	- False.
	 Гипотеза о равенстве дисперсий не принимается.
