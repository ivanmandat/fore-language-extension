# ISmFisherTest


Сборка: Stat;


## Описание


Интерфейс ISmFisherTest определяет
 параметры теста Фишера.


## Иерархия наследования


           [IStatMethod](../IStatMethod/IStatMethod.htm)


           ISmFisherTest


## Комментарии


Тест Фишера - это процедура проверки гипотезы о равенстве дисперсий
 двух нормальных распределений. В случае если статистика Фишера больше
 критического значения статистики Фишера, то гипотезу о равенстве дисперсий
 принимают.
