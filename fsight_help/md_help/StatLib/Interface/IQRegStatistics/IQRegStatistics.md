# IQRegStatistics


Сборка: Stat;


## Описание


Интерфейс IQRegStatistics определяет
 характеристики квантильной регрессии.


## Иерархия наследования


IQRegStatistics


## Комментарии


Для получения характеристик квантильной регрессии используйте свойство
 [ISmQuantileRegression.QRegStatistics](../ISmQuantileRegression/ISmQuantileRegression.QRegStatistics.htm).
