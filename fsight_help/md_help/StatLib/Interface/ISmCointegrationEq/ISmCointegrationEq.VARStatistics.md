# ISmCointegrationEq.VARStatistics


## Синтаксис


VARStatistics: [IVARStatistics](../IVARStatistics/IVARStatistics.htm);


## Описание


Свойство VARStatistics возвращает
 значения статистик векторной авторегрессии, рассчитанные для модели.


## Комментарии


Для работы с параметрами коэффициентов при коинтеграционном уравнении
 используйте свойство [ISmCointegrationEq.CointegralEquation](ISmCointegrationEq.CointegralEquation.htm).
