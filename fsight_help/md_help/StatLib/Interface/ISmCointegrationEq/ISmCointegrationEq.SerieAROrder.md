# ISmCointegrationEq.SerieAROrder


## Синтаксис


SerieAROrder: Array;


## Описание


Свойство SerieAROrder определяет
 порядок авторегрессии эндогенных переменных.


## Комментарии


Коэффициенты авторегрессии задаются в виде целочисленного массива. Количество
 элементов массива определяет порядок, а значения элементов определяю лаг
 для коэффициентов авторегрессии.
