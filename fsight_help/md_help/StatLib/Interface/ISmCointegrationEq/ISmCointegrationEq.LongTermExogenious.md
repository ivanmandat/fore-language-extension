# ISmCointegrationEq.LongTermExogenious


## Синтаксис


LongTermExogenious: Array;


## Описание


Свойство LongTermExogenious
 определяет массив индексов экзогенных переменных, входящих в группу переменных
 с долгосрочными коинтеграционными связями.


## Комментарии


Если экзогенная переменная не входит в группу долгосрочных (ISmCointegrationEq.LongTermExogenious)
 или краткосрочных ([ISmCointegrationEq.CommonExogenious](ISmCointegrationEq.CommonExogenious.htm))
 коинтеграционных связей, то она исключается из расчета модели.
