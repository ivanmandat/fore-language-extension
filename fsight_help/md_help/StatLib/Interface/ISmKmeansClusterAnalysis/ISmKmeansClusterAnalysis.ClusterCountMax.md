# ISmKmeansClusterAnalysis.ClusterCountMax


## Синтаксис


ClusterCountMax: Integer;


## Описание


Свойство ClusterCountMax определяет
 максимальное число кластеров при использовании автоподбора.


## Комментарии


Свойство актуально, если [ISmKmeansClusterAnalysis.ClusterCountSelection](ISmKmeansClusterAnalysis.ClusterCountSelection.htm) = [ClusterCountSelectionType](../../Enums/ClusterCountSelectionType.htm).VarDisAutoSelection
 или [ClusterCountSelectionType](../../Enums/ClusterCountSelectionType.htm).RAlgorithmAutoSelection.


Для определения минимального числа кластеров используйте свойство [ISmKmeansClusterAnalysis.ClusterCountMin](ISmKmeansClusterAnalysis.ClusterCountMin.htm).
