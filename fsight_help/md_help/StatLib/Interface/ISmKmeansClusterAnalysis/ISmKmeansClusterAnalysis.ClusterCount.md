# ISmKmeansClusterAnalysis.ClusterCount


## Синтаксис


ClusterCount: Integer;


## Описание


Свойство ClusterCount определяет
 число кластеров, на которое нужно разбить множество объектов.


## Комментарии


Количество кластеров должно удовлетворять ограничению 2 < k ≤ N.


Где:


	- k.
	 Количество кластеров;


	- N.
	 Количество рассматриваемых объектов. Определяется значением свойства [ISmSelfOrganizingMap.Data](../ISmSelfOrganizingMap/ISmSelfOrganizingMap.Data.htm).
