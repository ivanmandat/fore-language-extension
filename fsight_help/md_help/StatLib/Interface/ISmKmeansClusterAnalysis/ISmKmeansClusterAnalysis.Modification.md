# ISmKmeansClusterAnalysis.Modification


## Синтаксис


Modification: [KMeansModification](../../Enums/KMeansModification.htm);


## Описание


Свойство Modification определяет
 тип используемого алгоритма.


## Комментарии


Если Modification = KMeansModification.KMeans,
 то применяется стандартный алгоритм к-средних,
 предназначенный для кластеризации числовых значений. Если Modification = KMeansModification.KModes,
 то применяется алгоритм к-мод,
 предназначенный для кластеризации категориальных значений.
