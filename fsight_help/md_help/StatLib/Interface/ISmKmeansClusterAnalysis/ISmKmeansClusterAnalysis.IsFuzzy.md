# ISmKmeansClusterAnalysis.IsFuzzy


## Синтаксис


IsFuzzy: Boolean;


## Описание


Свойство IsFuzzy определяет,
 использовать ли метод нечёткой кластеризации.


## Комментарии


Допустимые значения:


	- True. Нечёткая кластеризация
	 включена;


	- False. Значение по умолчанию.
	 Нечёткая кластеризация отключена.
