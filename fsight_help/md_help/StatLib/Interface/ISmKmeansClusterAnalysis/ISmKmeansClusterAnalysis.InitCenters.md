# ISmKmeansClusterAnalysis.InitCenters


## Синтаксис


InitCenters: [ClusterCentersType](../../Enums/ClusterCentersType.htm);


## Описание


Свойство InitCenters определяет
 вариант выбора начальных центров кластеров.


## Комментарии


По умолчанию используется вариант «Максимальное расстояние до других
 центров кластеров».
