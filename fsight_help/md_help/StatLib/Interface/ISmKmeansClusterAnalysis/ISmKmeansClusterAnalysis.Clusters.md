# ISmKmeansClusterAnalysis.Clusters


## Синтаксис


Clusters: [IClustersType](../IClustersType/IClustersType.htm);


## Описание


Свойство Clusters возвращает
 информацию о разбиении на кластеры.


## Комментарии


Данное свойство возвращает список и число объектов, принадлежащих каждому
 кластеру, и координаты центров кластеров.
