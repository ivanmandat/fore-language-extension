# ISlMovingAverage


## Описание


Интерфейс ISlMovingAverage определяет
 параметры скользящего среднего.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property.gif)
		 [Backcast](ISlMovingAverage.Backcast.htm)


		 Свойство Backcast определяет
		 признак использования ретрополяции при оценивании коэффициентов
		 скользящего среднего.


		 ![](../../Property.gif)
		 [Coefficients](ISlMovingAverage.Coefficients.htm)


		 Свойство Coefficients
		 возвращает коэффициенты скользящего среднего.


		 ![](../../Property.gif)
		 [Epsilon](ISlMovingAverage.Epsilon.htm)


		 Свойство Epsilon определяет
		 точность сравнения коэффициентов скользящего среднего.


		 ![](../../Property.gif)
		 [MaxIteration](ISlMovingAverage.MaxIteration.htm)


		 Свойство MaxIteration
		 определяет максимальное число итераций для оценки коэффициентов
		 скользящего среднего.


		 ![](../../Property.gif)
		 [Order](ISlMovingAverage.Order.htm)


		 Свойство Order определяет
		 порядок скользящего среднего.
