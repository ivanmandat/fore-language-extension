# INonLinearRegressionExplanatories.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс регрессора в коллекции.


## Описание


Метод Remove удаляет ряд из коллекции по заданному индексу.


## Комментарии


В случае успешного удаления ряда метод возвращает True, в противном случае False.
