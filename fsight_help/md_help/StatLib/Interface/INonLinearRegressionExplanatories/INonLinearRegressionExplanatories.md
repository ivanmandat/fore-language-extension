# INonLinearRegressionExplanatories


Сборка: Stat;


## Описание


Интерфейс INonLinearRegressionExplanatories
 определяет параметры коллекции регрессоров для нелинейного МНК.


## Иерархия наследования


           INonLinearRegressionExplanatories


## Комментарий


Элементы коллекции реализованы интерфейсом [INonLinearRegressionExplanatory](../INonLinearRegressionExplanatory/INonLinearRegressionExplanatory.htm).
