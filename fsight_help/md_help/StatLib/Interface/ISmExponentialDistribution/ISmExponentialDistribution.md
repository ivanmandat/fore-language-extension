# ISmExponentialDistribution


Сборка: Stat;


## Описание


Интерфейс ISmExponentialDistribution
 позволяет генерировать выборку псевдослучайных чисел из экспоненциального
 распределения с заданным математическим ожиданием.


## Иерархия наследования


           [IDistribution](../IDistribution/IDistribution.htm)


           ISmExponentialDistribution


## Комментарии


Экспоненциальное распределение является частным случаем [распределения
 Вейбулла](Lib.chm::/05_Statistics/Distribution/Lib_WeibullDistribution.htm).
