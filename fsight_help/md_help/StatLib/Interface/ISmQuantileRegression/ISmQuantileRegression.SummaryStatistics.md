# ISmQuantileRegression.SummaryStatistics


## Синтаксис


SummaryStatistics: [ISummaryStatistics](../ISummaryStatistics/ISummaryStatistics.htm);


## Описание


Свойство SummaryStatistics возвращает
 описательные статистики модели.


## Комментарии


Описательные статистики рассчитываются по общим формулам. Статистики,
 не актуальные для данной модели, рассчитаны не будут.
