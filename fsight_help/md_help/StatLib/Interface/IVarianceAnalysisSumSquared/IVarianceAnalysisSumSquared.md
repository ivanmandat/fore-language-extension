# IVarianceAnalysisSumSquared


## Описание


Интерфейс IVarianceAnalysisSumSquared
 возвращает результаты дисперсионного анализа: сумму квадратов отклонений.


Для однофакторного анализа результаты возвращают свойства: [BetweenGroup](IVarianceAnalysisSumSquared.BetweenGroup.htm),
 [WithinGroup](IVarianceAnalysisSumSquared.WithinGroup.htm),
 [Total](IVarianceAnalysisSumSquared.Total.htm).


Для двухфакторного анализа результаты возвращают свойства: [Accuracy](IVarianceAnalysisSumSquared.Accuracy.htm),
 [Columns](IVarianceAnalysisSumSquared.Columns.htm), [Rows](IVarianceAnalysisSumSquared.Rows.htm),
 [Total](IVarianceAnalysisSumSquared.Total.htm).
