# INonLinearRegressionExplanatory


Сборка: Stat;


## Описание


Интерфейс INonLinearRegressionExplanatory
 определяет параметры объясняющего ряда для нелинейного МНК.


## Иерархия наследования


           INonLinearRegressionExplanatory


## Комментарии


Метод заключается в минимизации суммы квадратов отклонений наблюдаемых
 значений зависимой переменной от предсказанных значений.
