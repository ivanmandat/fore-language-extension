# ISmWhiteHeteroskedasticityTest.Explanatories


## Синтаксис


Explanatories: [ISlSeries](../ISlSeries/ISlSeries.htm);


## Описание


Свойство Explanatories определяет объясняющие ряды.


## Комментарии


При расчете теста будут рассмотрены объясняющие ряды и их квадраты. Например, если S1 и S2 - объясняющие, то будут рассмотрены S1, S2, S1*S1, S2*S2.
