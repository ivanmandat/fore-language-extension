# ISmUnivariateSpectrumAnalysis.InitCorrection


## Синтаксис


InitCorrection: [InitCorrectionType](../../Enums/InitCorrectionType.htm);


## Описание


Свойство InitCorrection определяет
 тип начальной корректировки.


## Комментарии


По умолчанию используется тип «Без предварительной коррекции».
