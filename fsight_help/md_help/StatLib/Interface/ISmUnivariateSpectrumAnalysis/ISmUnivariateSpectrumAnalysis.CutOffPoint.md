# ISmUnivariateSpectrumAnalysis.CutOffPoint


## Синтаксис


CutOffPoint: Integer;


## Описание


Свойство CutOffPoint определяет
 точку отсечения (длина исходного ряда на периоде идентификации).


## Пример


Использование свойства приведено в примере для [ISmUnivariateSpectrumAnalysis.InitCorrection](ISmUnivariateSpectrumAnalysis.InitCorrection.htm).


## Комментарии


Для корректного выполнения примера должно выполняться условие:


3<=(2*C+1)<=N/2


Где С - точка отсечения (CutOffPoint), N - число участвующих в расчете наблюдений.
