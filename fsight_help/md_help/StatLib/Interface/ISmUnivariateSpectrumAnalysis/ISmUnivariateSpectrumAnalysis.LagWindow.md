# ISmUnivariateSpectrumAnalysis.LagWindow


## Синтаксис


LagWindow: [LagWindowType](../../Enums/LagWindowType.htm);


## Описание


Свойство LagWindow определяет лаговое окно.


## Комментарии


По умолчанию используется «Прямоугольное», т.е. LagWindow = LagWindowType.Rectangular. Веса для лагового окна определяет свойство [ISmUnivariateSpectrumAnalysis.Weights](ISmUnivariateSpectrumAnalysis.Weights.htm).
