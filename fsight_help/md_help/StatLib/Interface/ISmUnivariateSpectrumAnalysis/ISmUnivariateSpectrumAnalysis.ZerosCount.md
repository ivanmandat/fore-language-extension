# ISmUnivariateSpectrumAnalysis.ZerosCount


## Синтаксис


ZerosCount: Integer;


## Описание


Свойство ZerosCount определяет
 количество нулевых значений, используемых для увеличения длины исходного
 ряда.


## Комментарии


Свойство учитывается, если [ISmUnivariateSpectrumAnalysis.PaddingType](ISmUnivariateSpectrumAnalysis.PaddingType.htm)
 = [SAPaddingType.PadByZeros](../../Enums/SAPaddingType.htm).
