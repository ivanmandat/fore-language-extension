# ISmUnivariateSpectrumAnalysis.UseFastFourierTransform


## Синтаксис


UseFastFourierTransform: Boolean;


## Описание


Свойство UseFastFourierTransform
 определяет, использовать ли быстрое преобразование Фурье.


## Комментарии


Допустимые значения:


	- True. Быстрое преобразование
	 Фурье используется. Порядок преобразования определяет свойство [ISmUnivariateSpectrumAnalysis.FastFourierTransformOrder](ISmUnivariateSpectrumAnalysis.FastFourierTransformOrder.htm);


	- False. Быстрое преобразование
	 Фурье не применяется.
