# ISmVarianceAnalysis.Data


## Синтаксис


Data: Array;


## Описание


Свойство Data определяет рассматриваемые наблюдения.


## Комментарии


Индексация входных массивов должна начинаться с нуля.


Если анализ однофакторный, то столбцы соответствуют уровням фактора, строки количеству испытаний. Если анализ двухфакторный, то столбцы соответствуют уровням фактора В, а строки уровням фактора А.
