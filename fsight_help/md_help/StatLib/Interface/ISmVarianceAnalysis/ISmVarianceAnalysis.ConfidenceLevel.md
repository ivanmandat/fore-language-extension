# ISmVarianceAnalysis.ConfidenceLevel


## Синтаксис


ConfidenceLevel: Double;


## Описание


Свойство ConfidenceLevel определяет уровень значимости, определяющий принятие/отвержение нулевой гипотезы.


## Комментарии


Допустимые значения берутся из диапазона (0, 1). По умолчанию свойство имеет значение «0,05».
