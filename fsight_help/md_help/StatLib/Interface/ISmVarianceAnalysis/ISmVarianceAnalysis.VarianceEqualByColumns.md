# ISmVarianceAnalysis.VarianceEqualByColumns


## Синтаксис


VarianceEqualByColumns: Boolean;


## Описание


Свойство VarianceEqualByColumns
 возвращает результат о принятии гипотезы.


## Комментарии


Допустимые значения:


	- True. Гипотеза
	 о равенстве дисперсий фактора B (по столбцам) принимается;


	- False. Гипотеза
	 о равенстве дисперсий фактора B (по столбцам) отклоняется.


Данное свойство используется только для двухфакторного анализа.
