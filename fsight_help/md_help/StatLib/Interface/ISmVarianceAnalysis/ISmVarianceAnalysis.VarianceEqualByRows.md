# ISmVarianceAnalysis.VarianceEqualByRows


## Синтаксис


VarianceEqualByRows: Boolean;


## Описание


Свойство VarianceEqualByRows
 возвращает результат о принятии гипотезы.


## Комментарии


Допустимые значения:


	- для однофакторного анализа:


		- True. Гипотеза о
		 равенстве групповых средних принимается;


		- False. Гипотеза
		 о равенстве групповых средних отклоняется;


	- для двухфакторного анализа:


		- True.
		 Гипотеза о равенстве дисперсий фактора A (по строкам) принимается;


		- False. Гипотеза о равенстве дисперсий
		 фактора A (по строкам) отклоняется.
