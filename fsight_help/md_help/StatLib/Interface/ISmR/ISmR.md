# ISmR


Сборка: Stat;


## Описание


Интерфейс ISmR предназначен
 для интеграции с R.


## Иерархия наследования


           [IStatMethod](../IStatMethod/IStatMethod.htm)


           ISmR


## Комментарии


Интеграция с R доступна, если в «Форсайт. Аналитическая платформа»
 [подключен
 пакет R](UiDw.chm::/FAQ/R_Integration.htm).


Интерфейс предназначен для работы с числовыми рядами и после выполнения
 возвращает числовой ряд. Использование интерфейса в других целях, например,
 для построения графиков через R, нежелательно, т.к. Rserve не поддерживает
 все возможности R и аварийно завершит работу, если попытаться воспользоваться
 неподдерживаемой RServe функциональностью.
