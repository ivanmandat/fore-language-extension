# ISmBinomialDistribution


Сборка: Stat;


## Описание


Интерфейс ISmBinomialDistribution
 позволяет генерировать выборку псевдослучайных целых чисел из дискретного
 биномиального распределения с заданными параметрами.


## Иерархия наследования


           [IDistribution](../IDistribution/IDistribution.htm)


           ISmBinomialDistribution


## Комментарии


Биномиальное распределение - это распределение числа успехов в серии
 из n экспериментов, каждый из
 которых завершается успехом с вероятностью p.
 Важными предельными случаями биномиального распределения являются [распределение
 Пуассона](Lib.chm::/05_Statistics/Distribution/Lib_PoissonDistribution.htm) и [нормальное
 распределение](Lib.chm::/05_Statistics/Distribution/Lib_NormalDistribution.htm).
