# ICallbackNLOptimization.GetConstraintValue


## Синтаксис


GetConstraintValue(Indx: Integer): Variant;


## Параметры


Indx. Индекс ограничения.


## Описание


Метод GetConstraintValue рассчитывает
 и возвращает значение i-го нелинейного
 ограничения.


## Комментарии


Для получения значения частной производной по j-ой
 переменной от i-го ограничения
 используйте метод [ICallbackNLOptimization.GetConstraintPartialDeriv](ICallbackNLOptimization.GetConstraintPartialDeriv.htm).
