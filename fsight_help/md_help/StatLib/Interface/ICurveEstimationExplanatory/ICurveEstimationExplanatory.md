# ICurveEstimationExplanatory


## Описание


Интерфейс ICurveEstimationExplanatory
 определяет параметры объясняющего ряда.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property.gif)
		 [IsTrend](ICurveEstimationExplanatory.IsTrend.htm)


		 Свойство IsTrend определяет,
		 каким образом задается объясняющий ряд - в виде автоматически
		 генерируемого ряда времени, или пользователем.


		 ![](../../Property.gif)
		 [ReferencePoint](ICurveEstimationExplanatory.ReferencePoint.htm)


		 Свойство ReferencePoint
		 определяет начальную точку тренда.


		 ![](../../Property.gif)
		 [Serie](ICurveEstimationExplanatory.Serie.htm)


		 Свойство Serie определяет
		 объясняющий ряд данных.
