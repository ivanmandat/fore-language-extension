# ILinearRegressionAutoSelection


## Описание


Интерфейс ILinearRegressionAutoSelection
 определяет параметры автоподбора регрессоров для линейной регрессии.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property.gif)
		 [Criterion](ILinearRegressionAutoSelection.Criterion.htm)


		 Свойство Criterion
		 определяет критерий поиска регрессора.


		 ![](../../Property.gif)
		 [IsActive](ILinearRegressionAutoSelection.IsActive.htm)


		 Свойство IsActive определяет
		 признак использования автоподбора регрессоров.


		 ![](../../Property.gif)
		 [Max](ILinearRegressionAutoSelection.Max.htm)


		 Свойство Max определяет
		 максимально допустимое число регрессоров.


		 ![](../../Property.gif)
		 [Min](ILinearRegressionAutoSelection.Min.htm)


		 Свойство Min определяет
		 минимально допустимое число регрессоров.


		 ![](../../Property.gif)
		 [Results](ILinearRegressionAutoSelection.Results.htm)


		 Свойство Results возвращает
		 результат автоподбора регрессоров.
