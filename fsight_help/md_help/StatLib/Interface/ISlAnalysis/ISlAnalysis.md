# ISlAnalysis


Сборка: Stat;


## Описание


Интерфейс ISlAnalysis определяет
 параметры анализа.


## Иерархия наследования


           [ISlAnalysis](ISlAnalysis.htm)


## Комментарии


Анализ применяется в методе главных компонент. Для получения параметров
 анализа используйте свойство [ISmPrincipalComponentAnalysis.Analysis](../ISmPrincipalComponentAnalysis/ISmPrincipalComponentAnalysis.Analysis.htm).
