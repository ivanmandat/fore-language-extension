# IJohansenTestEquations


## Описание


Интерфейс IJohansenTestEquations
 содержит свойства и методы для работы с коллекцией уравнений (эндогенных
 переменных) теста Йохансена.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property.gif)
		 [Count](IJohansenTestEquations.Count.htm)


		 Свойство Count возвращает
		 количество уравнений в коллекции.


		 ![](../../Property.gif)
		 [Item](IJohansenTestEquations.Item.htm)


		 Свойство Item определяет
		 параметры уравнения.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub.gif)
		 [Add](IJohansenTestEquations.Add.htm)


		 Метод Add добавляет
		 уравнение в коллекцию.


		 ![](../../Sub.gif)
		 [Clear](IJohansenTestEquations.Clear.htm)


		 Метод Clear очищает
		 коллекцию.


		 ![](../../Sub.gif)
		 [Remove](IJohansenTestEquations.Remove.htm)


		 Метод Remove удаляет
		 уравнение по заданному индексу.
