# IJohansenTestEquations.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс уравнения в коллекции.


## Описание


Метод Remove удаляет уравнение по заданному индексу. В случае успешного удаления метод возвращает True, в противном случае False.
