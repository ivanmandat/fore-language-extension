# IArimaSpecificationInitialValues.AutoRegression


## Синтаксис


AutoRegression: Array;


## Описание


Свойство AutoRegression определяет
 начальные значения для коэффициентов авторегрессии.


## Комментарии


Количество коэффициентов зависит от значения свойства [AutoRegressionOrder](../IArimaSpecification/IArimaSpecification.AutoRegressionOrder.htm).


По умолчанию начальные значения для коэффициентов имеют значение «0».
