# IArimaSpecificationInitialValues.SeasonalMovingAverage


## Синтаксис


SeasonalMovingAverage: Array;


## Описание


Свойство SeasonalMovingAverage
 определяет начальные значения для коэффициентов сезонного скользящего
 среднего.


## Комментарии


Количество коэффициентов зависит от значения свойства [SeasonalMovingAverageOrder](../IArimaSpecification/IArimaSpecification.SeasonalMovingAverageOrder.htm).


По умолчанию начальные значения для коэффициентов имеют значение «0».
