# IArimaSpecificationInitialValues.SeasonalAutoRegression


## Синтаксис


SeasonalAutoRegression: Array;


## Описание


Свойство SeasonalAutoRegression
 определяет начальные значения для коэффициентов сезонной авторегрессии.


## Комментарии


Количество коэффициентов зависит от значения свойства [SeasonalAutoRegression](IArimaSpecificationInitialValues.SeasonalAutoRegression.htm).


По умолчанию начальные значения для коэффициентов имеют значение «0».
