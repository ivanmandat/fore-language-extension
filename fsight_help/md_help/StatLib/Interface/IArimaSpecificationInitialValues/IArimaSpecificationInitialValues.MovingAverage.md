# IArimaSpecificationInitialValues.MovingAverage


## Синтаксис


MovingAverage: Array;


## Описание


Свойство MovingAverage определяет
 начальные значения для коэффициентов скользящего среднего.


## Комментарии


Количество коэффициентов зависит от значения свойства [MovingAverageOrder](../IArimaSpecification/IArimaSpecification.MovingAverageOrder.htm).


По умолчанию начальные значения для коэффициентов имеют значение «0».
