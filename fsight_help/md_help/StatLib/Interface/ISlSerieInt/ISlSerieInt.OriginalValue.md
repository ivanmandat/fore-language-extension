# ISlSerieInt.OriginalValue


## Синтаксис


OriginalValue: Array;


## Описание


Свойство OriginalValue возвращает
 оригинал значений категориального ряда данных, то есть тех данных, которые
 были заданы с помощью свойства [Value](ISlSerieInt.Value.htm).


## Комментарии


Для получения значений категориального ряда данных используйте свойство
 [ISlSerieInt.Value](ISlSerieInt.Value.htm).
