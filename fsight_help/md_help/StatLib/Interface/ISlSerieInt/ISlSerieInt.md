# ISlSerieInt


Сборка: Stat;


## Описание


Интерфейс ISlSerieInt предназначен
 для работы с параметрами категориального ряда данных.


## Иерархия наследования


           ISlSerieInt


## Комментарии


Категориальный ряд - набор данных, элементы которого нельзя упорядочить,
 и между которыми нельзя найти расстояние. Например, что обычно едят на
 завтрак люди из выборки: бутерброд, каша, яичница и т.д. Ни упорядочить,
 ни найти расстояние между элементами массива нельзя.


Для работы с параметрами коллекции категориальных рядов данных используйте
 свойства и методы интерфейса [ISlSeriesInt](../ISlSeriesInt/ISlSeriesInt.htm).
