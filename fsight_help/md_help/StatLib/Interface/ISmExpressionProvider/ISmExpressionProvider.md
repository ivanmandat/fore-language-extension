# ISmExpressionProvider


Сборка: Stat;


## Описание


Интерфейс ISmExpressionProvider
 предназначен для определения источника выражения.


## Иерархия наследования


           ISmExpressionProvider


## Комментарии


В качестве источника выражения, может быть использован пользовательский
 модуль.
