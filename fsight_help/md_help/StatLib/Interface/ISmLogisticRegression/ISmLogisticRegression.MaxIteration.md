# ISmLogisticRegression.MaxIteration


## Синтаксис


MaxIteration: Integer;


## Описание


Свойство MaxIteration определяет
 максимальное число итераций, за которое необходимо получить решение.


## Комментарии


Число итераций, за которое было получено решение, возвращает свойство
 [ISmLogisticRegression.NumOfIter](ISmLogisticRegression.NumOfIter.htm).
