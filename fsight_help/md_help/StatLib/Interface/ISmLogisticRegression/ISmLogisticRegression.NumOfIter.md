# ISmLogisticRegression.NumOfIter


## Синтаксис


NumOfIter: Integer;


## Описание


Свойство NumOfIter возвращает
 число итераций, за которое было получено решение.


## Комментарии


Максимальное число итераций, отведенных на решение метода, определяет
 свойство [ISmLogisticRegression.MaxIteration](ISmLogisticRegression.MaxIteration.htm).
