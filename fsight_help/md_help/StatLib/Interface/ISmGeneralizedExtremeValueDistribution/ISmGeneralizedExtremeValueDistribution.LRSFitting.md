# ISmGeneralizedExtremeValueDistribution.LRSFitting


## Синтаксис


LRSFitting(Sample: Array): Integer;


## Параметры


Sample. Массив данных.


## Описание


Метод LRSFitting осуществляет
 автоподбор параметров распределения методом LRS (Linear Ratio of Spacings).


## Комментарии


Метод LRS предназначен для оценки параметров обобщенного распределения
 экстремальных значений.
