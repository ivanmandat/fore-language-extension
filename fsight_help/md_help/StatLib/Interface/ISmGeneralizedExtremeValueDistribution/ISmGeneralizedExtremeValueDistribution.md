# ISmGeneralizedExtremeValueDistribution


Сборка: Stat;


## Описание


Интерфейс ISmGeneralizedExtremeValueDistribution предназначен
 для генерации выборки псевдослучайных величин из обобщенного распределения
 экстремальных значений.


## Иерархия наследования


           [IDistribution](../IDistribution/IDistribution.htm)


           ISmGeneralizedExtremeValueDistribution


## Комментарии


Метод позволяет сгенерировать выборку из обобщенного распределения экстремальных
 значений и оценить параметры распределения по выборке с помощью метода
 максимального правдоподобия.
