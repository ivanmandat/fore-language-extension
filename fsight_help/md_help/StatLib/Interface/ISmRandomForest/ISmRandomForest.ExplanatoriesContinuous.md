# ISmRandomForest.ExplanatoriesContinuous


## Синтаксис


ExplanatoriesContinuous: [ISlSeries](../ISlSeries/ISlSeries.htm);


## Описание


Свойство ExplanatoriesContinuous
 возвращает объясняющие количественные ряды.


## Комментарии


Для получения объясняющих категориальных рядов используйте свойство
 [ISmRandomForest.ExplanatoriesCategorical](ISmRandomForest.ExplanatoriesCategorical.htm),
 объясняющих порядковых рядов - свойство [ISmRandomForest.ExplanatoriesOrdered](ISmRandomForest.ExplanatoriesOrdered.htm).


Количественный (вещественный) ряд - набор данных, элементы которого
 можно упорядочить, и между которыми можно найти расстояние. Например,
 расход топлива автомобилей. Можно упорядочить и сказать, у кого больше,
 а у кого меньше.
