## Синтаксис


CategoriesList: Array;


## Описание


Свойство CategoriesList
 возвращает список категорий.


## Комментарии


Свойство CategoriesList содержит
 все уникальные значения из ряда [IDataMining.Dependent](../IDataMining/IDataMining.Dependent.htm),
 расположенные в порядке возрастания.
