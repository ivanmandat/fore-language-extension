# ISmRandomForest.NumberOfPredictors


## Синтаксис


NumberOfPredictors: Integer;


## Описание


Свойство NumberOfPredictors
 определяет число признаков, которые формируют случайное дерево.


## Комментарии


Для определения числа деревьев в случайном лесе используйте свойство
 [ISmRandomForest.ForestSize](ISmRandomForest.ForestSize.htm).
