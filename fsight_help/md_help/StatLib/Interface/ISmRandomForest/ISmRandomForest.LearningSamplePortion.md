# ISmRandomForest.LearningSamplePortion


## Синтаксис


LearningSamplePortion: Double;


## Описание


Свойство LearningSamplePortion
 определяет долю выборки для обучения дерева.


## Комментарии


Значение параметра должно принадлежать промежутку (0, 1).


Для определения числа деревьев в случайном лесе используйте свойство
 [ISmRandomForest.ForestSize](ISmRandomForest.ForestSize.htm).
