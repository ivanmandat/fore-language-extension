# ISlTramoSeatsOutliersCollection.Item


## Синтаксис


Item(Index: Integer): [ISlTramoSeatsOutliers](../ISlTramoSeatsOutlier/ISlTramoSeatsOutlier.htm);


## Параметры


Index. Индекс элемента.


## Описание


Свойство Item возвращает уравнение
 коллекции по указанному индексу.


## Комментарии


Индексация элементов сквозная и начинается с нуля. Для получения количества
 элементов в коллекции используйте свойство [ISlTramoSeatsOutliersCollection.Count](ISlTramoSeatsOutliersCollection.Count.htm).

Использование свойства приведено в примере для [ISmTramoSeats.D](../ISmTramoSeats/ISmTramoSeats.D.htm).
