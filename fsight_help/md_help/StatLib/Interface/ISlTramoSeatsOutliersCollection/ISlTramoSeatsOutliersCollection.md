# ISlTramoSeatsOutliersCollection


Сборка: Stat;


## Описание


Интерфейс ISlTramoSeatsOutliersCollection
 предназначен для работы с коллекцией выбросов.


## Иерархия наследования


           ISlTramoSeatsOutliersCollection


## Комментарии


Для определения, учитываются ли выбросы, используйте свойство [ISmTramoSeats.OutliersDetection](../ISmTramoSeats/ISmTramoSeats.OutliersDetection.htm).
