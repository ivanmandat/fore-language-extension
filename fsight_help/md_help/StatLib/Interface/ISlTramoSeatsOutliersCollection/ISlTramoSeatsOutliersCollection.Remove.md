# ISlTramoSeatsOutliersCollection.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемого элемента.


## Описание


Метод Remove удаляет уравнение
 из коллекции по указанному индексу.


## Комментарии


Индексация элементов сквозная и начинается с нуля. Для получения количества
 элементов в коллекции используйте свойство [ISlTramoSeatsOutliersCollection.Count](ISlTramoSeatsOutliersCollection.Count.htm).
 Если удаление выполнено успешно, то метод возвращает значение True,
 в обратном случае - False.
