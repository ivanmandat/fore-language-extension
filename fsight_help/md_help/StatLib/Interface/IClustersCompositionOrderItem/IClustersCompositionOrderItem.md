# IClustersCompositionOrderItem


## Описание


Интерфейс IClustersCompositionOrderItem
 возвращает параметры шага составления кластеров.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property.gif)
		 [ClusterDistance](IClustersCompositionOrderItem.ClusterDistance.htm)


		 Свойство ClusterDistance
		 возвращает расстояние между кластерами в момент объединения.


		 ![](../../Property.gif)
		 [IndexLowerCluster](IClustersCompositionOrderItem.IndexLowerCluster.htm)


		 Свойство IndexLowerCluster
		 возвращает номер кластера, соединённого с [IndexUpperCluster](IClustersCompositionOrderItem.IndexUpperCluster.htm).


		 ![](../../Property.gif)
		 [IndexUpperCluster](IClustersCompositionOrderItem.IndexUpperCluster.htm)


		 Свойство IndexUpperCluster
		 возвращает номер кластера, соединённого с [IndexLowerCluster](IClustersCompositionOrderItem.IndexLowerCluster.htm).
