# INonLinearConstraint


Сборка: Stat;


## Описание


Интерфейс INonLinearConstraint
 определяет параметры нелинейного ограничения.


## Иерархия наследования


           INonLinearConstraint


## Комментарии


Для решения задачи минимизации функции используйте интерфейс [ISmNonLinearOptimization](../ISmNonLinearOptimization/ISmNonLinearOptimization.htm).
