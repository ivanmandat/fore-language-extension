# INonLinearConstraint.BoundaryUpper


## Синтаксис


BoundaryUpper: Double;


## Описание


Свойство BoundaryUpper определяет
 верхнюю границу.


## Комментарии


Для определения нижней границы нелинейного ограничения используйте свойство
 [INonLinearConstraint.BoundaryLower](INonLinearConstraint.BoundaryLower.htm).
