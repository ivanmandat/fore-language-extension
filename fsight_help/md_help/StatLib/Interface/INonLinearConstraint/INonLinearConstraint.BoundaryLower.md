# INonLinearConstraint.BoundaryLower


## Синтаксис


BoundaryLower: Double;


## Описание


Свойство BoundaryLower определяет
 нижнюю границу.


## Комментарии


Для определения верхней границы нелинейного ограничения используйте
 свойство [INonLinearConstraint.BoundaryUpper](INonLinearConstraint.BoundaryUpper.htm).
