# INonLinearConstraint.LagrangeMultiplier


## Синтаксис


LagrangeMultiplier: Double;


## Описание


Свойство LagrangeMultiplier
 определяет значение множителя Лагранжа.


## Комментарии


Множитель Лагранжа характеризует в какой степени значение ограничения
 близко его границе.
