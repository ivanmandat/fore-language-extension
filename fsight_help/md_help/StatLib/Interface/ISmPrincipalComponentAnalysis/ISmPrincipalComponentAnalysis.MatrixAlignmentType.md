# ISmPrincipalComponentAnalysis.MatrixAlignmentType


## Синтаксис


MatrixAlignmentType: [AlignmentType](../../Enums/AlignmentType.htm);


## Описание


Свойство MatrixAlignmentType
 определяет тип центрирования исходной матрицы данных.


## Комментарии


По умолчанию применяется однократное центрирование, т.е. MatrixAlignmentType = AlignmentType.SingleAlignment.


Для задания исходных данных используйте свойство [ISmPrincipalComponentAnalysis.Objects](ISmPrincipalComponentAnalysis.Objects.htm).
