# ISmPrincipalComponentAnalysis.IncludeAverages


## Синтаксис


IncludeAverages: Boolean;


## Описание


Свойство IncludeAverages определяет,
 использовать ли межгрупповое и общегрупповое средние в качестве дополнительных
 компонентов.


## Комментарии


Допустимые значения.


	- True. Средние используются
	 в качестве дополнительных компонентов;


	- False. Значение по умолчанию.
	 Дополнительные компоненты не используются.
