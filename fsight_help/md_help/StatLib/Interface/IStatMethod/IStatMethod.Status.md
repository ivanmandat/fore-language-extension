# IStatMethod.Status


## Синтаксис


Status: Integer;


## Описание


Свойство Status возвращает статус выполнения метода.


## Комментарии


Если метод выполнен без ошибок, то Status возвращает значение «0», иначе возвращает код ошибки. Получить текст ошибки по её коду , просмотреть которую можно с помощью свойства [ErrorByStatus](IStatMethod.ErrorByStatus.htm).
