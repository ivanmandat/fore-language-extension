# IStatMethod.WarningsNumbers


## Синтаксис


WarningsNumbers: Array;


## Описание


Свойство WarningsNumbers возвращает
 номера предупреждений, возникших при расчёте метода.


## Комментарии


Номера предупреждений возвращаются в виде массива целочисленных значений.
 Для получения текста предупреждения по его номеру используйте свойство
 [IStatMethod.WarningByStatus](IStatMethod.WarningByStatus.htm).
