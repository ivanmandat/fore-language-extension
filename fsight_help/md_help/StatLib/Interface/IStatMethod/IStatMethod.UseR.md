# IStatMethod.UseR


## Синтаксис


UseR: Boolean;


## Описание


Свойство UseR определяет, будет
 ли расчет статистического метода производиться через пакет R.


## Комментарии


Допустимые значения:


	- True. Расчет статистического
	 метода будет производиться через пакет R;


	- False. Расчет статистического
	 метода не будет производиться через пакет R.
