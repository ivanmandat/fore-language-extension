# IStatMethod


Сборка: Stat;


## Описание


Интерфейс IStatMethod содержит
 свойства и методы статистического метода.


## Иерархия наследования


           IStatMethod


## Комментарии


Данный интерфейс является базовым для всех статистических методов.

		 Имя свойства
		 Краткое описание


		 ![](../../Property.gif)
		 [DisplayName](IStatMethod.DisplayName.htm)
		 Свойство DisplayName
		 возвращает внешнее наименование метода.


		 ![](../../Property.gif)
		 [ErrorByStatus](IStatMethod.ErrorByStatus.htm)
		 Свойство ErrorByStatus
		 возвращает сообщение об ошибке по ее номеру.


		 ![](../../Property.gif)
		 [Errors](IStatMethod.Errors.htm)
		 Свойство Errors возвращает
		 сообщение обо всех ошибках и предупреждениях.

		 [Name](IStatMethod.Name.htm)
		 Свойство Name возвращает

		 [PerformanceTime](IStatMethod.PerformanceTime.htm)
		 Свойство PerformanceTime
		 возвращает время выполнения метода.


		 ![](../../Property.gif)
		 [Status](IStatMethod.Status.htm)
		 Свойство Status возвращает
		 статус выполнения метода.


		 ![](../../Property.gif)
		 [SupportsR](IStatMethod.SupportsR.htm)
		 Свойство SupportsR
		 возвращает признак поддержки расчета статистического метода через

		 ![](../../Property.gif)
		 [UseR](IStatMethod.UseR.htm)
		 Свойство UseR определяет,
		 будет ли расчет статистического метода производиться через

		 ![](../../Property.gif)

		 [Warnings](IStatMethod.Warnings.htm)
		 Свойство Warnings возвращает предупреждения, возникшие при расчёте метода.


		 ![](../../Property.gif)
		 [WarningsCount](IStatMethod.WarningsCount.htm)
		 Свойство WarningsCount

		 ![](../../Property.gif)
		 [WarningsNumbers](IStatMethod.WarningsNumbers.htm)
		 Свойство WarningsNumbers возвращает номера предупреждений, возникших при расчёте метода.
