# IStatMethod.Execute


## Синтаксис


Execute: Integer;


## Описание


Метод Execute осуществляет выполнение статистического метода.


## Комментарии


В случае выполнения метода без ошибок возвращает значение «0», иначе возвращает код ошибки, просмотреть которую можно с помощью свойства [ErrorByStatus](IStatMethod.ErrorByStatus.htm).


Тело модуля для любого статистического метода должно быть следующей структуры:


-
создание объекта статистического метода;


-
заполнение входных данных;


-
настройка параметров метода. Следует помнить, что некоторые параметры статистического метода могут быть заданы по умолчанию;


-
выполнение метода - метод Execute;


-
выходные данные.


Перед запуском расчета статистического метода необходимо обязательно определить входные переменные и настройки самого статистического метода.
