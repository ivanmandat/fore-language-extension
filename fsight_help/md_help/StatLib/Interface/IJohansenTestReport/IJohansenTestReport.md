# IJohansenTestReport


## Описание


Интерфейс IJohansenTestReport
 возвращает результаты теста Йохансена.


Для определения количества коинтеграционных связей следует сравнить
 значение относительно максимального правдоподобия с N процентным (N=5%,
 1%) уровнем значимости: если данное значение больше N процентного уровня,
 то предполагают отклонение гипотезы о наличии соответствующего количества
 (нет, 1, 2, … n) коинтеграционных связей в сторону альтернативной гипотезы.
 Альтернативной гипотезой является гипотеза о том, что все ряды являются
 стационарными.
