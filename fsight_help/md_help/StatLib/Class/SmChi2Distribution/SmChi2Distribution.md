# SmChi2Distribution


Сборка: Stat;


## Описание


Класс SmChi2Distribution позволяет
 генерировать выборку псевдослучайных чисел из распределения χ2
 (хи-квадрат) с заданным количеством степеней свободы.


## Комментарии


Распределение χ2
 является частным случаем гамма-распределения.
