# SmGeneralizedExtremeValueDistribution


Сборка: Stat;


## Описание


Класс SmGeneralizedExtremeValueDistribution
 предназначен для оценки параметров распределения экстремальных значений
 методом максимального правдоподобия.


## Комментарии


Метод максимального правдоподобия - способ оценивания неизвестных параметров.
