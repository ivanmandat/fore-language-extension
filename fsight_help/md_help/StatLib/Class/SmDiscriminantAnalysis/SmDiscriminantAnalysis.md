# SmDiscriminantAnalysis


Сборка: Stat;


## Описание


Класс SmDiscriminantAnalysis
 применяется для интеллектуального анализа данных методом «Дискриминантный
 анализ».


## Комментарии


Данный метод выполняет поиск линейной комбинации переменных, наилучшим
 образом разделяющих (дискриминирующих) два или более класса.
