# SmR


Сборка: Stat;


## Описание


Класс SmR предназначен для интеграции
 с R.


## Комментарии


Интеграция с R доступна, если в «Форсайт. Аналитическая платформа»
 [подключен
 пакет R](UiDw.chm::/FAQ/R_Integration.htm).
