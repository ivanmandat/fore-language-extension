# SmBreuschPaganGodfreyTest


Сборка: Stat;


## Описание


Класс SmBreuschPaganGodfreyTest
 предназначен для работы с параметрами теста Бреуша-Пагана-Годфри на гетероскедастичность.


## Комментарии


Тест возвращает статистику Фишера, статистику Энгла и соответствующие
 значения вероятностей.
