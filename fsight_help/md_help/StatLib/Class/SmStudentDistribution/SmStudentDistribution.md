# SmStudentDistribution


Сборка: Stat;


## Описание


Класс SmStudentDistribution
 позволяет генерировать выборку псевдослучайных чисел из распределения
 Стьюдента с заданным количеством степеней свободы.


## Комментарии


Распределение Стьюдента также называется t-распределение.
