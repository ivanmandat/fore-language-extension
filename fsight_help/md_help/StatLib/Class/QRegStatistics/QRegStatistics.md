# QRegStatistics


Сборка: Stat;


## Описание


Класс QRegStatistics
 определяет характеристики квантильной регрессии.


## Комментарии


Для получения характеристик квантильной регрессии используйте свойство
 [ISmQuantileRegression.QRegStatistics](../../Interface/ISmQuantileRegression/ISmQuantileRegression.QRegStatistics.htm).
