# SmAugmentDickeyFullerTest


## Описание


Класс SmAugmentDickeyFullerTest предназначен для работы с параметрами расширенного теста Дики-Фуллера.


## Комментарии


Расширенный тест возвращает статистику Дики-Фуллера. Тест является средством проверки стационарности временного ряда.
