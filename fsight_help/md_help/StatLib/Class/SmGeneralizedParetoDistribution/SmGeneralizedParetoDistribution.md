# SmGeneralizedParetoDistribution


Сборка: Stat;


## Описание


Класс SmGeneralizedParetoDistribution
 предназначен для оценки параметров обобщенного распределения Парето
 методом максимального правдоподобия.


## Комментарии


Метод максимального правдоподобия - способ оценивания неизвестных параметров.
