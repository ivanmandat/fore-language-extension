# SmHyperGeometricDistribution


Сборка: Stat;


## Описание


Класс SmHyperGeometricDistribution
 позволяет по заданным параметрам генерировать выборку псевдослучайных
 целых чисел из дискретного гипергеометрического распределения количества
 «успешных элементов» в выборке из конечной совокупности, содержащей «успешные
 элементы».
