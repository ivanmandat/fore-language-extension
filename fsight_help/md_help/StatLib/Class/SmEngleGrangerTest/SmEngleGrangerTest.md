# SmEngleGrangerTest


Сборка: Stat;


## Описание


Класс SmEngleGrangerTest предназначен
 для работы с параметрами теста Энгла-Гранжера.


## Комментарии


Тест возвращает значение τ-статистики и Z-статистики.
