# SmPhillipsOuliarisTest


Сборка: Stat;


## Описание


Класс SmPhillipsOuliarisTest
 предназначен для работы с параметрами теста Филлипса-Оуляриса.


## Комментарии


Тест возвращает значение τ-статистики и Z-статистики.
