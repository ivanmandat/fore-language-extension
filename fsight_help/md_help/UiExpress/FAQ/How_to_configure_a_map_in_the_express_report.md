# Как настроить карту в экспресс-отчёте?


Для работы с картой в экспресс-отчёте в репозитории должны быть:


	- [сконвертированная
	 топооснова](UiMaps.chm::/Common/Topobase.htm);


	- куб, основанный на территориальном справочнике, который содержит
	 атрибут с идентификатором TERRID.
	 Атрибут TERRID используется
	 для установки соответствия между атрибутами топоосновы и атрибутами
	 справочника. Если в топооснове есть атрибут ISO со значениями по стандарту
	 ISO 3166-1, атрибут TERRID
	 должен быть строковым и содержать соответствующие значения.


Для [детализации
 территорий](UiMaps.chm::/Configure/MapConfigure.htm#drilldown) карты
 территориальный справочник должен удовлетворять следующим условиям:


	- должен содержать атрибут TOPO_ID;


	- должен быть иерархическим.


Для работы с картой в отчёте:


	- Откройте отчёт;


	- Подключите в источники куб;


	- [Настройте
	 измерения для работы с картой](UiMaps.chm::/Common/Metrics.htm);


	- Перейдите на визуализатор «[Карта](../Setup_express_report/UiExpress_control.htm)».
	 Здесь должна быть использована соответствующая топооснова. Если есть
	 необходимость, измените карту с помощью команды «[Изменить
	 карту](UiMaps.chm::/Configure/MapConfigure.htm#chngmap)».


Регионы на карте будут залиты цветом в соответствии с выбранным элементом
 метрик, только если в территориальном справочнике значения атрибута TERRID соответствуют значениям атрибута
 в топооснове.
