# Как ограничить права доступа для измерений и элементов измерений источника
 данных экспресс-отчёта?


Настройка прав доступа для измерений источника данных:


	- Выбрать в навигаторе нужный объект (справочник или справочник
	 НСИ);


	- Для выбранного справочника вызвать контекстное меню «Права
	 доступа»;


	- В открывшемся диалоге «[Параметры
	 управления доступом](Admin.chm::/03_Admin/Admin_AdminObjects.htm)» настроить права доступа
	 для выбранного объекта.


Настройка права доступа для элементов справочника НСИ:


	- Открыть справочник НСИ. На вкладке «[Описание](UiNavObj.chm::/reference_book/Master_RDS_reference_book/base_settings.htm)»
	 установить флажок «Элементы имеют
	 права доступа»;


	- [Открыть](Admin.chm::/01_RunSecManager/Admin_Organizational_Starting.htm)
	 менеджер безопасности, найти нужный справочник НСИ и настроить права
	 доступа для его элементов (для получения подробной информации обратитесь
	 к разделу «[Права
	 доступа на элементы справочников НСИ](Admin.chm::/03_Admin/Admin_ElementAccess.htm)»).
