# Построение экспресс-отчёта


Для построения экспресс-отчёта выполните шаги:


	- [Вставьте листы](../Setup_express_report/Sheets.htm)
	 экспресс-отчёта при необходимости.


	- [Выберите один](UiExpress_Tree_Multivariate_Object.htm)
	 или [несколько
	 источников данных](../ChangeNote/UiExpress_Tabl_ChangeNote_PrivDim.htm) экспресс-отчёта.


	- Разместите [измерения](UiSelection.chm::/Selection/Dimension.htm)
	 источника данных.


	- Отметьте необходимые [элементы
	 измерений](UiSelection.chm::/Selection/Selection_of_the_dimension_elements.htm).


	- [Выберите
	 визуализатор](../Setup_express_report/UiExpress_control.htm) для отображения данных.


	- [Настройте
	 заголовок](../Setup_express_report/UiExpress_Purpose_Toolbar_Title.htm) экспресс-отчёта как интерактивный элемент управления.


	- Выполните при необходимости [анализ
	 данных](UiAnalyticalArea.chm::/Analysis/Data_analysis_in_a_table.htm) экспресс-отчёта в таблице.


	- Используйте [встроенную
	 расширенную аналитику](UiAnalyticalArea.chm::/Analysis/AdvancedAnalytics.htm).


После выполнения действий сохраните изменения экспресс-отчёта:


	- выполните команду «Отчет >
	 Сохранить» в [главном
	 меню](UiExpress_Organizational_Starting.htm);


	- нажмите кнопку ![](../Express/save.png) «Сохранить»
	 на панели быстрого доступа;


	- нажмите сочетание клавиш CTRL+S.


Cохраняются все установленные фильтры (в том числе отметка источников
 данных), все выбранные и настроенные визуализаторы данных.
