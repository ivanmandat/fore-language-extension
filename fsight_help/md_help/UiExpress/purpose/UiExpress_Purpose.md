# Оперативный просмотр и анализ данных (OLAP)


Для оперативного просмотра и анализа данных используйте инструмент «Аналитические запросы (OLAP)»:


![](olap.png)


Инструмент «Аналитические запросы
 (OLAP)» для оперативного просмотра и анализа данных «на лету» с
 отображением данных средствами деловой графики.


Ключевые возможности:


	- формирование нерегламентированных запросов к данным и проведение
	 экспресс-анализа с применением различных аналитических функций;


	- настройки фильтрации, сортировки, вычислений, анализа 80/20,
	 построения трендов, условного форматирования и др.;


	- одновременный анализ нескольких источников в одном визуализаторе;


	- детализация и обобщение данных;


	- проверка данных на соответствие заданным условиям и ограничениям;


	- представление данных в виде интерактивных визуализаторов: таблиц,
	 диаграмм, карт;


	- интеграция с Microsoft Office.
