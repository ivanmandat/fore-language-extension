# Работа с готовым экспресс-отчётом


Для работы с готовым экспресс-отчётом выполните шаги:


	- [Обновление данных экспресс-отчёта](../Express/Update_data.htm).


	- [Работа с данными в таблице](../Express/Data.htm).


	- [Использование
	 встроенной расширенной аналитики](UiAnalyticalArea.chm::/Analysis/AdvancedAnalytics.htm).


	- [Отмена/повтор операции над объектами
	 экспресс-отчёта](Undo_Redo.htm).


	- [Экспорт экспресс-отчёта](../Express/UiExpress_Express_Export.htm).


	- [Предварительный
	 просмотр и печать экспресс-отчёта](../Express/UiExpress_Express_Print.htm).


	- [Сохранение изменённых
	 данных](../Express/UiExpress_Express_Save.htm).


	- [Публикация экспресс-отчёта](Publish.htm) из веб-приложения.


После работы с готовым экспресс-отчётом:


	- выполните команду «Отчет >
	 Закрыть» в [главном
	 меню](../purpose/UiExpress_Organizational_Starting.htm) для закрытия текущего экспресс-отчёта и продолжения работы
	 с инструментом «Аналитические запросы
	 (OLAP)»;


	- выполните команду «Отчет >
	 Выход» в главном меню для закрытия экспресс-отчёта и завершения
	 работы с инструментом «Аналитические
	 запросы (OLAP)».


Если имеются несохранённые изменения, то будет предложено их сохранить.
