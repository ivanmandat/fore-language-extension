# Публикация в социальных сетях и обсуждение экспресс-отчёта


Для публикации ссылки на экспресс-отчёт в социальной сети выполните
 команду «[Поделиться](uinav.chm::/GUI/Format/Publishing.htm)»
 в главном меню и в отобразившемся меню выберите необходимую социальную
 сеть.


Примечание.
 Возможность доступна только в веб-приложении.
