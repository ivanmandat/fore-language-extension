# Предварительный просмотр и печать
 экспресс-отчёта


Перед печатью отчёта:


	- [Настройте
	 параметры страниц](ParamPage/UiExpress_Express_ParamPage.htm), которые будут напечатаны.


	- Выполните [предварительный
	 просмотр](uinav.chm::/GUI/Object_preview.htm)
	 и убедитесь, что отчёт имеет требуемый вид.


Совет. Если отчёт,
 содержащий диаграммы, планируется печатать на черно-белом принтере, то
 для наилучшего качества печати диаграмм примените для них черно-белое
 оформление, установив для этого флажок «Черно-белая
 печать» на вкладке «[Печать
 объектов](ParamPage/UiExpress_Express_ParamPage4.htm)» диалога «[Параметры
 страницы](ParamPage/UiExpress_Express_ParamPage.htm)».


Для печати отчёта:


	- выполните команду «Отчет >
	 Печать» в главном меню;


	- нажмите сочетание клавиш CTRL+P. Возможность доступна только
	 в настольном приложении.


В настольном приложении будет открыт диалог «[Печать](uinav.chm::/GUI/Print_setup.htm)»
 для настройки параметров печати и выбора используемого принтера. В веб-приложении
 будет открыта страница для предварительного просмотра и печати отчёта.
 Внешний вид страницы и порядок работы зависит от используемого браузера.


Для печати отчёта в нескольких вариантах в зависимости от отметки фиксированных
 измерений, выполните команду главного меню «Отчет
 > Печать > Пакетная печать». Будет открыто окно «[Печать нескольких вариантов отчёта](UiExpress_Express_Select.htm)».


Примечание.
 При печати отчёта может быть отображен штамп № 1 в соответствии с
 Инструкцией № 0126-87, если в качестве политики безопасности используется
 [мандатный
 доступ](admin.chm::/04_securitypolicy/admin_mandataccess.htm) и для уровня безопасности данного экспресс-отчёта
 включена маркировка при экспорте и печати.
