# Сохранение изменённых данных


Для [сохранения
 измененных данных](UiAnalyticalArea.chm::/Working_with_table_data/Edit_and_save_data.htm) в источник нажмите кнопку ![](save.png) «Сохранить изменения», расположенную
 на вкладке «Данные» ленты инструментов.


## Создание копии экспресс-отчёта


Для сохранения копии экспресс-отчёта с новым именем в репозитории выполните
 команду «Отчет > Сохранить как»
 в [главном
 меню](../purpose/UiExpress_Organizational_Starting.htm). В открывшемся окне выберите месторасположение сохраняемого экспресс-отчёта,
 а также укажите наименование и уникальный идентификатор для сохраняемого
 экспресс-отчёта. По умолчанию копия отчёта создаётся со старым именем
 отчёта и автоматически сгенерированным идентификатором.
 Идентификатор сохраняемого объекта может состоять только из букв
 латинского алфавита, цифр и символа подчеркивания.


Примечание.
 Сохранение экспресс-отчёта недоступно, если у пользователя есть права
 только на просмотр и отсутствуют права на изменение.
