# ApplicationVersion


Сборка: Host;


## Описание


Класс ApplicationVersion реализует
 объект, предназначенный для получения информации о версии платформы.


## Свойства объекта класса, унаследованные от [IApplicationVersion](../../Interface/IApplicationVersion/IApplicationVersion.htm)


		 Имя свойства
		 Краткое описание

		 номер сборки платформы.


		 ![](../../Property_Image.gif)
		 [Debug](../../Interface/IApplicationVersion/IApplicationVersion.Debug.htm)
		 Свойство Debug возвращает
