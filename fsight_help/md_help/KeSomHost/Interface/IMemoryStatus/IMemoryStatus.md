# IMemoryStatus


## Описание


Интерфейс IMemoryStatus содержит
 свойства для получения информации о памяти, доступной в операционной системе.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AvailablePageFile](IMemoryStatus.AvailablePageFile.htm)
		 Свойство AvailablePageFile
		 возвращает размер доступного файла подкачки.


		 ![](../../Property_Image.gif)
		 [AvailablePhysicalMemory](IMemoryStatus.AvailablePhysicalMemory.htm)
		 Свойство AvailablePhysicalMemory
		 возвращает размер свободной физической памяти в байтах.


		 ![](../../Property_Image.gif)
		 [AvailableVirtualMemory](IMemoryStatus.AvailableVirtualMemory.htm)
		 Свойство AvailableVirtualMemory
		 возвращает размер свободной виртуальной памяти в байтах.


		 ![](../../Property_Image.gif)
		 [MemoryLoad](IMemoryStatus.MemoryLoad.htm)
		 Свойство MemoryLoad
		 возвращает общую загрузку памяти в процентах.


		 ![](../../Property_Image.gif)
		 [TotalPageFile](IMemoryStatus.TotalPageFile.htm)
		 Свойство TotalPageFile
		 возвращает общий размер файла подкачки в байтах.


		 ![](../../Property_Image.gif)
		 [TotalPhysicalMemory](IMemoryStatus.TotalPhysicalMemory.htm)
		 Свойство TotalPhysicalMemory
		 возвращает общий размер физической памяти в байтах.


		 ![](../../Property_Image.gif)
		 [TotalVirtualMemory](IMemoryStatus.TotalVirtualMemory.htm)
		 Свойство TotalVirtualMemory
		 возвращает общий размер виртуальной памяти в байтах.
