# ILicenseCommuterInfo


## Описание


Интерфейс ILicenseCommuterInfo
 используется для получения информации о временной локальной лицензии,
 которая была получена с сервера лицензий.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ExpireDate](ILicenseCommuterInfo.ExpireDate.htm)


		 Свойство ExpireDate
		 возвращает дату окончания действия локальной лицензии.


		 ![](../../Property_Image.gif)
		 [FeatureName](ILicenseCommuterInfo.FeatureName.htm)


		 Свойство FeatureName
		 возвращает имя единицы лицензирования, для которой получена локальная
		 лицензия.


		 ![](../../Property_Image.gif)
		 [IssuingServer](ILicenseCommuterInfo.IssuingServer.htm)


		 Свойство IssuingServer
		 возвращает IP-адрес сервера лицензий, с которого получена локальная
		 лицензия.


		 ![](../../Property_Image.gif)
		 [StartDate](ILicenseCommuterInfo.StartDate.htm)


		 Свойство StartDate
		 возвращает дату начала действия локальной лицензии.
