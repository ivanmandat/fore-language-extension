# IProcess.HasExited


## Синтаксис


HasExited: Boolean;


## Описание


Свойство HasExited возвращает
 признак завершения выполнения процесса.


## Комментарии


Если свойство вернуло значение True,
 значит процесс завершил свое выполнение. Код завершения можно получить
 с помощью свойства [IProcess.ExitCode](IProcess.ExitCode.htm).


## Пример


Использование свойства приведено в примере для [IProcess.Start](IProcess.Start.htm).
