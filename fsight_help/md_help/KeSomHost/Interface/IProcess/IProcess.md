# IProcess


Сборка: Host;


## Описание


Интерфейс IProcess содержит
 базовые свойства и методы для запуска и остановки локальных процессов.


## Комментарии


Интерфейс является базовым для следующих интерфейсов:


	- [IShellProcess](../IShellProcess/IShellProcess.htm);


	- [IWinProcess](UiLib.chm::/Interface/IWinProcess/IWinProcess.htm).
