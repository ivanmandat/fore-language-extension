# IApplicationVersion.Description


## Синтаксис


Description: String;


## Описание


Свойство Description возвращает
 полное описание версии платформы.


## Комментарии


Полное описание выглядит примерно следующим образом: Предварительная
 версия 7.0.7237 [юникод].
