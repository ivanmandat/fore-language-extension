# IApplicationVersion


Сборка: Host;


## Описание


Интерфейс IApplicationVersion
 содержит информацию о версии платформы.


## Иерархия наследования


IApplicationVersion


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Build](IApplicationVersion.Build.htm)
		 Свойство Build возвращает
		 номер сборки платформы.


		 ![](../../Property_Image.gif)
		 [Debug](IApplicationVersion.Debug.htm)
		 Свойство Debug возвращает
		 True, если данная сборка
		 является отладочной версией программы.


		 ![](../../Property_Image.gif)
		 [Description](IApplicationVersion.Description.htm)
		 Свойство Description
		 возвращает полное описание версии платформы.


		 ![](../../Property_Image.gif)
		 [IsRelease](IApplicationVersion.IsRelease.htm)
		 Свойство IsRelease
		 возвращает информацию о признаке выпущенной версии.


		 ![](../../Property_Image.gif)
		 [Major](IApplicationVersion.Major.htm)
		 Свойство Major возвращает
		 номер основной версии сборки.


		 ![](../../Property_Image.gif)
		 [MaxReleaseBuild](IApplicationVersion.MaxReleaseBuild.htm)
		 Свойство MaxReleaseBuild
		 возвращает информацию о максимальном номере сборки текущего релиза.


		 ![](../../Property_Image.gif)
		 [Minor](IApplicationVersion.Minor.htm)
		 Свойство Minor возвращает
		 номер промежуточной версии сборки.


		 ![](../../Property_Image.gif)
		 [MinReleaseBuild](IApplicationVersion.MinReleaseBuild.htm)
		 Свойство MinReleaseBuild
		 возвращает информацию о минимальном номере сборки текущего релиза.


		 ![](../../Property_Image.gif)
		 [PrivateBuild](IApplicationVersion.PrivateBuild.htm)
		 Свойство PrivateBuild
		 возвращает True, если
		 сборка является нетестированной.


		 ![](../../Property_Image.gif)
		 [QFE](IApplicationVersion.QFE.htm)
		 Свойство QFE возвращает
		 номер обновления.


		 ![](../../Property_Image.gif)
		 [ShortDescription](IApplicationVersion.ShortDescription.htm)
		 Свойство ShortDescription
		 возвращает краткое описание версии платформы.


		 ![](../../Property_Image.gif)
		 [VersionDescription](IApplicationVersion.VersionDescription.htm)
		 Свойство VersionDescription
		 возвращает описание для версии платформы.
