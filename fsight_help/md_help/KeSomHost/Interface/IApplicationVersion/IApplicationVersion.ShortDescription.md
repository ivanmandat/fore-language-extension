# IApplicationVersion.ShortDescription


## Синтаксис


ShortDescription: String;


## Описание


Свойство ShortDescription возвращает
 краткое описание версии платформы.


## Комментарии


Краткое описание выглядит примерно следующим образом: PB 7.0.7237
 U.
