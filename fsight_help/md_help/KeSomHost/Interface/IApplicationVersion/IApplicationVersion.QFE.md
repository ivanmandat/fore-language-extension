# IApplicationVersion.QFE


## Синтаксис


QFE: Integer;


## Описание


Свойство QFE возвращает номер
 обновления.


## Комментарии


Для не выпущенных версий возвращается значение 0, для выпущенных версий
 значение будет больше 0.
