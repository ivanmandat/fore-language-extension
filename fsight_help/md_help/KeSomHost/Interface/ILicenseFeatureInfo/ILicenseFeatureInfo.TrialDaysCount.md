# ILicenseFeatureInfo.TrialDaysCount


## Синтаксис


TrialDaysCount: Integer;


## Описание


Свойство TrialDaysCount возвращает
 количество дней использования типа лицензии с [пробным
 периодом](setup.chm::/08_Licensing/Admin_Licensing_Variants.htm#trial).


## Комментарии


Пробный период определяет свойство [ILicenseFeatureInfo.LicenseType](ILicenseFeatureInfo.LicenseType.htm).

Использование свойства приведено в примере для [ILicenseManager.FeatureInfo](../ILicenseManager/ILicenseManager.FeatureInfo.htm).
