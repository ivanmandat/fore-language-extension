# ILicenseFeatureInfo


## Описание


Интерфейс ILicenseFeatureInfo
 используется для получения информации о единице лицензирования.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CommuterAllowed](ILicenseFeatureInfo.CommuterAllowed.htm)


		 Свойство CommuterAllowed
		 возвращает True, в случае
		 если возможно получение локальной лицензии, и False,
		 если это не разрешено.


		 ![](../../Property_Image.gif)
		 [DaysLeft](ILicenseFeatureInfo.DaysLeft.htm)


		 Свойство DaysLeft
		 возвращает количество дней до окончания действия лицензии.


		 ![](../../Property_Image.gif)


		 [Description](ILicenseFeatureInfo.Description.htm)


		 Свойство Description
		 возвращает наименование системной единицы лицензирования.


		 ![](../../Property_Image.gif)
		 [ExpireDate](ILicenseFeatureInfo.ExpireDate.htm)


		 Свойство ExpireDate
		 возвращает дату окончания действия лицензии.


		 ![](../../Property_Image.gif)
		 [FeatureName](ILicenseFeatureInfo.FeatureName.htm)


		 Свойство FeatureName
		 возвращает имя единицы лицензирования.


		 ![](../../Property_Image.gif)


		 [LicenseIds](ILicenseFeatureInfo.LicenseIds.htm)


		 Свойство Licenselds возвращает
		 номер лицензии.


		 ![](../../Property_Image.gif)
		 [LicenseMode](ILicenseFeatureInfo.LicenseMode.htm)


		 Свойство LicenseMode
		 возвращает [тип
		 лицензии](setup.chm::/08_Licensing/Admin_Licensing_Variants.htm): сетевая или автономная.


		 ![](../../Property_Image.gif)
		 [LicenseType](ILicenseFeatureInfo.LicenseType.htm)


		 Свойство LicenseType
		 возвращает [тип
		 лицензии](setup.chm::/08_Licensing/Admin_Licensing_Variants.htm) по времени действия: пробный период,
		 конкретные даты, постоянная.


		 ![](../../Property_Image.gif)
		 [LockType](ILicenseFeatureInfo.LockType.htm)


		 Свойство LockType
		 возвращает тип привязки лицензии.


		 ![](../../Property_Image.gif)


		 [NoExpiration](ILicenseFeatureInfo.NoExpiration.htm)


		 Свойство NoExpiration
		 определяет, ограничена ли лицензия по времени действия.


		 ![](../../Property_Image.gif)
		 [StartDate](ILicenseFeatureInfo.StartDate.htm)


		 Свойство StartDate
		 возвращает дату начала действия лицензии.


		 ![](../../Property_Image.gif)
		 [TrialDaysCount](ILicenseFeatureInfo.TrialDaysCount.htm)


		 Свойство TrialDaysCount
		 возвращает количество дней использования типа лицензии с [пробным
		 периодом](setup.chm::/08_Licensing/Admin_Licensing_Variants.htm#trial).
