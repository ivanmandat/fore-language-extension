# ILicenseFeatureInfo.NoExpiration


## Синтаксис


NoExpiration: Boolean;


## Описание


Свойство NoExpiration определяет,
 ограничена ли лицензия по времени действия.


## Комментарии


Допустимые значения:


	- True. Лицензия не ограничена;


	- False. Лицензия ограничена.
