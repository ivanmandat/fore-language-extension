# ILicense.ServerName


## Синтаксис


ServerName: String;


## Описание


Свойство ServerName возвращает
 имя сервера лицензий, с которого используется лицензия.


## Комментарии


Если используется автономная лицензия, то свойство будет возвращать
 «no-net».
