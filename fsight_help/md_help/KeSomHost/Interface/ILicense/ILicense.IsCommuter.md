# ILicense.IsCommuter


## Синтаксис


IsCommuter: Boolean;


## Описание


Свойство IsCommuter возвращает
 информацию о использовании временной локальной лицензии, полученной с
 сервера лицензий.


## Комментарии


Допустимые значения:


	-  True. Используется
	 временная локальная лицензия (commuter), полученная с сервера лицензий;


	-  False. Временная
	 локальная лицензия с сервера лицензий не используется.


Лицензия может быть получена с сервера лицензий, но не использоваться,
 если сервер лицензий доступен.


Для получения информации о взятии временной локальной лицензии с сервера
 лицензий используйте свойство [ILicenseManager.IsCheckedOut](../ILicenseManager/ILicenseManager.IsCheckedOut.htm).


Примечание.
 Некоторые сетевые лицензии позволяют использовать приложение вне сетевого
 окружения на ограниченный срок до 30 дней, например, для использования
 на ноутбуках в командировках. Для этого сетевая лицензия позволяет получить
 [временную
 локальную лицензию](Setup.chm::/08_Licensing/Admin_Licensing_Variants.htm#network) (commuter).
