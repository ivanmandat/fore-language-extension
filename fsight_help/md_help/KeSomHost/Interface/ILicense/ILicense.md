# ILicense


## Описание


Интерфейс ILicense предназначен
 для работы с лицензиями.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [FeatureName](ILicense.FeatureName.htm)


		 Свойство FeatureName
		 возвращает имя единицы лицензирования.


		 ![](../../Property_Image.gif)
		 [IsCommuter](ILicense.IsCommuter.htm)


		 Свойство IsCommuter
		 возвращает информацию о использовании временной локальной лицензии,
		 полученной с сервера лицензий.


		 ![](../../Property_Image.gif)
		 [ServerName](ILicense.ServerName.htm)


		 Свойство ServerName
		 возвращает имя сервера лицензий, с которого используется лицензия.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [LicenseRelease](ILicense.LicenseRelease.htm)


		 Метод LicenseRelease
		 освобождает лицензию.


		 ![](../../Sub_Image.gif)
		 [LicenseUpdate](ILicense.LicenseUpdate.htm)


		 Метод LicenseUpdate
		 обновляет лицензию.
