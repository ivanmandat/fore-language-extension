# Интерфейсы сборки Host


		 Интерфейс
		 Краткое описание


		 ![](../Interface_Image.gif)
		 [IApplication](IApplication/iapplication.htm)
		 Интерфейс IApplication
		 содержит базовые свойства и методы для управления текущим приложением.


		 ![](../Interface_Image.gif)
		 [IApplicationClass](IApplicationClass/IApplicationClass.htm)


		 Интерфейс IApplicationClass
		 содержит статические свойства класса [Application](../Class/Application/Application.htm).


		 ![](../Interface_Image.gif)
		 [IApplicationVersion](IApplicationVersion/IApplicationVersion.htm)
		 Интерфейс IApplicationVersion
		 содержит информацию о версии платформы.


		 ![](../Interface_Image.gif)
		 [ILicense](ILicense/ILicense.htm)


		 Интерфейс ILicense
		 предназначен для работы с лицензиями.


		 ![](../Interface_Image.gif)
		 [ILicenseCommuterInfo](ILicenseCommuterInfo/ILicenseCommuterInfo.htm)


		 Интерфейс ILicenseCommuterInfo
		 используется для получения информации о временной локальной лицензии,
		 которая была получена с сервера лицензий.


		 ![](../Interface_Image.gif)
		 [ILicenseFeatureInfo](ILicenseFeatureInfo/ILicenseFeatureInfo.htm)


		 Интерфейс ILicenseFeatureInfo
		 используется для получения информации о единице лицензирования.


		 ![](../Interface_Image.gif)
		 [ILicenseManager](ILicenseManager/ILicenseManager.htm)


		 Интерфейс ILicenseManager
		 предназначен для управления лицензиями.


		 ![](../Interface_Image.gif)


		 [ILicenseRequestSettings](ILicenseRequestSettings/ILicenseRequestSettings.htm)


		 Интерфейс ILicenseRequestSettings
		 содержит свойство для запроса единицы лицензирования.


		 ![](../Interface_Image.gif)
		 [IMemoryStatus](IMemoryStatus/IMemoryStatus.htm)


		 Интерфейс IMemoryStatus
		 содержит свойства для получения информации о памяти, доступной
		 в операционной системе.


		 ![](../Interface_Image.gif)
		 [IProcess](IProcess/IProcess.htm)


		 Интерфейс IProcess
		 содержит базовые свойства и методы для запуска и остановки локальных
		 процессов.


		 ![](../Interface_Image.gif)


		 [IShellExecuteInfo](IShellExecuteInfo/IShellExecuteInfo.htm)


		 Интерфейс IShellExecuteInfo
		 предназначен для настройки параметров запуска приложений.


		 ![](../Interface_Image.gif)


		 [IShellProcess](IShellProcess/IShellProcess.htm)


		 Интерфейс IShellProcess
		 предназначен для запуска консольных приложений и получения результата
		 их выполнения.
