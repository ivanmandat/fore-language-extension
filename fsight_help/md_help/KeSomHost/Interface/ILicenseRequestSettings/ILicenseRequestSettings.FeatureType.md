# ILicenseRequestSettings.FeatureType


## Синтаксис


FeatureType: [UiLicenseFeatureType](ForeSys.chm::/Enums/UiLicenseFeatureType.htm);


## Описание


Свойство FeatureType предназначено
 для запроса одной единицы лицензирования, не указывая при этом наименование
 этой единицы лицензирования.


## Комментарии


Свойство актуально при использовании
 метода [ILicenseManager.RequestLicense](../ILicenseManager/ILicenseManager.RequestLicense.htm)
 без параметра FeatureName.
