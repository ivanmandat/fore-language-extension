# ILicenseRequestSettings


## Описание


Интерфейс ILicenseRequestSettings
 содержит свойство для запроса единицы лицензирования.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [FeatureType](ILicenseRequestSettings.FeatureType.htm)


		 Свойство FeatureType
		 предназначено для запроса одной единицы лицензирования, не указывая
		 при этом наименование этой единицы лицензирования.
