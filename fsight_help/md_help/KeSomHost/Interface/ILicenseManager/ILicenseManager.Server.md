# ILicenseManager.Server


## Синтаксис


Server: String;


## Описание


Свойство Server возвращает имя
 сервера лицензий, с которого используется лицензия.


## Комментарии


Если используется автономная лицензия, то свойство будет возвращать
 «no-net».
