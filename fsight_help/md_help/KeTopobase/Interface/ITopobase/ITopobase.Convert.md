# ITopobase.Convert


## Синтаксис


Convert(SourceFormat: [TopobaseFormat](../../Enums/TopobaseFormat.htm);
 DestFormat: [TopobaseFormat](../../Enums/TopobaseFormat.htm)):
 [IIOStream](modio.chm::/Interface/IIOStream/IIOStream.htm);


## Параметры


SourceFormat. Значение формата-источника;


DestFormat. Значение результирующего
 формата.


## Описание


Метод Convert преобразует топооснову
 из формата-источника в результирующий формат.


## Комментарии


Подробнее об особенностях результирующих форматов топооснов вы можете
 узнать из описания перечисления [TopobaseFormat](../../Enums/TopobaseFormat.htm).
