# ITopobase


Сборка: Topobase;


## Описание


Интерфейс ITopobase содержит
 методы, используемые при работе с картой репозитория.


## Иерархия наследования


           [IDocumentBase](KeFore.chm::/Interface/IDocumentBase/IDocumentBase.htm)


           ITopobase


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AvailableFormats](ITopobase.AvailableFormats.htm)
		 Свойство AvailableFormats
		 возвращает комбинацию значений форматов, в которые была сконвертирована
		 топооснова.


		 ![](../../Property_Image.gif)
		 [Tree](ITopobase.Tree.htm)
		 Свойство Tree
		 возвращает дерево-структуру топоосновы.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Convert](ITopobase.Convert.htm)
		 Метод Convert преобразует
		 топооснову из формата-источника в результирующий формат.


		 ![](../../Sub_Image.gif)
		 [FormatLoadFromFile](ITopobase.FormatLoadFromFile.htm)
		 Метод FormatLoadFromFile
		 загружает топооснову из файла.


		 ![](../../Sub_Image.gif)
		 [FormatLoadFromStream](ITopobase.FormatLoadFromStream.htm)
		 Метод FormatLoadFromStream
		 загружает топооснову из потока.


		 ![](../../Sub_Image.gif)
		 [FormatSaveToFile](ITopobase.FormatSaveToFile.htm)
		 Метод FormatSaveToFile
		 сохраняет топооснову в файл.


		 ![](../../Sub_Image.gif)
		 [FormatSaveToStream](ITopobase.FormatSaveToStream.htm)
		 Метод FormatSaveToStream
		 сохраняет топооснову в поток.


		 ![](../../Sub_Image.gif)
		 [GetSpecificTopobase](ITopobase.GetSpecificTopobase.htm)
		 Метод GetSpecificTopobase
		 создает топооснову по заданному дереву-структуре.


## Методы, унаследованные от [IDocumentBase](KeFore.chm::/Interface/IDocumentBase/IDocumentBase.htm)


		 Имя метода

		 [SaveToStream](kefore.chm::/interface/idocumentbase/idocumentbase.savetostream.htm)
		 Метод SaveToStream
		 осуществляет сохранение содержимого документа в поток.
