# ITopobase.FormatLoadFromStream


## Синтаксис


FormatLoadFromStream(SourceFormat: [TopobaseFormat](../../Enums/TopobaseFormat.htm),
 Stream: [IIOStream](modio.chm::/Interface/IIOStream/IIOStream.htm));


## Параметры


SourceFormat .Формат топоосновы;


Stream. Поток, из которого
 загружается топооснова.


## Описание


Метод FormatLoadFromStream загружает
 топооснову из потока.
