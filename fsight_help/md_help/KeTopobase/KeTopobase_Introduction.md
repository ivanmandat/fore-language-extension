## С чего начать


		 В данном разделе
		 представлено описание сборки Topobase,
		 содержащей средства для работы с картами репозитория. Карта -
		 это топооснова, на которой статистические данные отображены путем
		 задания атрибутов некоторых слоёв. В репозитории карта представляет
		 собой документ, содержащий в себе загруженный файл топоосновы
		 в формате TBS.


		Описание ресурсов сборки представлено в следующих подразделах:


			- [Интерфейсы сборки
			 Topobase](Interface/Interfaces.htm)


			- [Перечисления сборки Topobase](Enums/Enums.htm)


			- [Примеры](Samples/KeTopobase_Sample.htm)


		 Перед началом работы рекомендуется
		 ознакомиться с разделами справки: «[Среда
		 разработки](UiDevEnv.chm::/01_Development_Environment/01_Purpose_of_the_constructor/Purpose_of_the_constructor.htm)», «[Руководство по языку Fore](Fore.chm::/Fore_Title.htm)».
