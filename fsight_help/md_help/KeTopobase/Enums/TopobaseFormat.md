# TopobaseFormat


## Описание


Перечисление TopobaseFormat
 содержит форматы топоосновы карты.


Используется следующими методами:


	- [ITopobase.Convert](../Interface/ITopobase/ITopobase.Convert.htm);


	- [ITopobase.FormatLoadFromFile](../Interface/ITopobase/ITopobase.FormatLoadFromFile.htm);


	- [ITopobase.FormatLoadFromStream](../Interface/ITopobase/ITopobase.FormatLoadFromStream.htm);


	- [ITopobase.FormatSaveToFile](../Interface/ITopobase/ITopobase.FormatSaveToFile.htm);


	- [ITopobase.FormatSaveToStream](../Interface/ITopobase/ITopobase.FormatSaveToStream.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 Tbs. Формат для отображения
		 карты в настольном приложении в 2D режиме.

		Доступны: заливка карты, легенда, расшифровка территорий.


		 2
		 Svg. Формат для отображения
		 SVG карты в веб-приложении в 2D режиме.

		Доступны: заливка карты, легенда, расшифровка территорий, временная
		 шкала, масштабирование.


		 4
		 Google. Формат для
		 отображения [внешней
		 карты](UiMaps.chm::/Configure/MapConfigure.htm#externalservices) в веб-приложении в 2D режиме.

		Доступны: заливка карты, легенда, расшифровка территорий,
		 временная шкала, масштабирование.


		 8
		 Triangulate. Формат
		 для отображения SVG карты в веб-приложении в [3D режиме](UiMaps.chm::/Configure/MapConfigure.htm#3d).

		Доступны: заливка карты, легенда, расшифровка территорий,
		 временная шкала, масштабирование, объем территорий.


		 32
		 Sphere. Формат для
		 отображения SVG карты в веб-приложении в [3D режиме](UiMaps.chm::/Configure/MapConfigure.htm#3d).

		Доступны: заливка карты, легенда, расшифровка территорий,
		 временная шкала, масштабирование, объем территорий.

Все координаты в радианах, преобразованных из
		 физических координат.

Особенности использования топоосновы в веб-приложении:


	- в регламентном отчёте топооснова в формате TBS отображается в
	 виде изображения;


	- в экспресс-отчёте недоступна расшифровка территорий для топоосновы
	 в форматах: SVG, Triangulate, Sphere.
