# IPPSpeedometer.Opacity


## Синтаксис


Opacity: Double;


## Описание


Свойство Opacity определяет
 прозрачность спидометра.


## Комментарии


Диапазон допустимых значений [0, 1]. Значение по умолчанию - «1». Чем
 меньше значение, тем больше прозрачность.
