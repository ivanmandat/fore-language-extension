# IPPSpeedometer.BorderStyle


## Синтаксис


BorderStyle: [GxDashStyle](ModDrawing.chm::/Enums/GxDashStyle.htm);


## Описание


Свойство BorderStyle определяет
 стиль отрисовки границы области спидометра.


## Комментарии


Свойство BorderStyle актуально,
 если у свойства [IPPSpeedometer.EnableBorder](IPPSpeedometer.EnableBorder.htm)
 установлено значение True.
