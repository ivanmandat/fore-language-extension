# IPPSpeedometer.BorderBrush


## Синтаксис


BorderBrush: [IGxBrush](ModDrawing.chm::/Interface/IGxBrush/IGxBrush.htm);


## Описание


Свойство BorderBrush определяет
 кисть для отрисовки границы области спидометра.


## Комментарии


Свойство BorderBrush актуально,
 если у свойства [IPPSpeedometer.EnableBorder](IPPSpeedometer.EnableBorder.htm)
 установлено значение True.
