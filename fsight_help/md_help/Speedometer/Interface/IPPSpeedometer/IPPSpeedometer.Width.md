# IPPSpeedometer.Width


## Синтаксис


Width: Double;


## Описание


Свойство Width определяет ширину
 спидометра.


## Комментарии


Для определения высоты спидометра используйте свойство [IPPSpeedometer.Height](IPPSpeedometer.Height.htm).
