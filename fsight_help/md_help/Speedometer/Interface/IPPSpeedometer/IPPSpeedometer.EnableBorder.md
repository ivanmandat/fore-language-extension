# IPPSpeedometer.EnableBorder


## Синтаксис


EnableBorder: Boolean;


## Описание


Свойство EnableBorder определяет
 наличие границы области спидометра.


## Комментарии


Допустимые значения:


	- True. Область спидометра
	 имеет границу;


	- False. Значение по умолчанию.
	 Область спидометра не имеет границы.
