# IPPSpeedometer.Height


## Синтаксис

## Описание


Свойство Height определяет высоту
 спидометра.


## Комментарии


Для определения ширины спидометра используйте свойство [IPPSpeedometer.Width](IPPSpeedometer.Width.htm).
