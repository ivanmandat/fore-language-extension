# IPPSpeedometer.EnableBrush


## Синтаксис


EnableBrush: Boolean;


## Описание


Свойство EnableBrush определяет
 наличие заливки фона области спидометра.


## Комментарии


Допустимые значения:


	- True. Область спидометра
	 имеет заливку фона;


	- False. Значение по умолчанию.
	 Область спидометра не имеет заливки фона.
