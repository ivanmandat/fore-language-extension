# IPPSpeedometer.SaveToXml


## Синтаксис


SaveToXml: String;


## Описание


Метод SaveToXml сохраняет Xml-разметку
 спидометра в строковую переменную.


## Комментарии


Для загрузки Xml-разметки спидометра из строковой переменной используйте
 метод [IPPSpeedometer.LoadFromXml](IPPSpeedometer.LoadFromXml.htm).
