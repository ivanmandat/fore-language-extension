# ISpeedometer.IsDirty


## Синтаксис


IsDirty: Boolean;


## Описание


Свойство IsDirty определяет признак наличия изменений в параметрах спидометра.


## Комментарии


Свойство возвращает значение False, если значения параметров всех элементов спидометра не изменялись и True, если присутствуют какие-либо изменения.


Изменение любого параметра у элементов спидометра приведет к установке данному свойства значения True. Установка данному свойству значения False позволит в дальнейшем определить наличие новых изменений.
