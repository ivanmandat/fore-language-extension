# ISpeedometer


## Описание


Интерфейс ISpeedometer содержит
 свойства и методы для настройки параметров отображения спидометра.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [IsDirty](ISpeedometer.IsDirty.htm)
		 Свойство IsDirty определяет
		 признак наличия изменений в параметрах спидометра.


		 ![](../../Property_Image.gif)
		 [Scales](ISpeedometer.Scales.htm)
		 Свойство Scales возвращает
		 коллекцию шкал спидометра.


		 ![](../../Property_Image.gif)
		 [View](ISpeedometer.View.htm)
		 Свойство View возвращает
		 параметры отображения спидометра.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [BeginUpdate](ISpeedometer.BeginUpdate.htm)
		 Метод BeginUpdate отключает
		 любую перерисовку компонента.


		 ![](../../Sub_Image.gif)
		 [EndUpdate](ISpeedometer.EndUpdate.htm)
		 Метод EndUpdate возобновляет
		 перерисовку компонента.


		 ![](../../Sub_Image.gif)
		 [Refresh](ISpeedometer.Refresh.htm)
		 Метод Refresh осуществляет
		 обновление области спидометра.
