# ISpeedometer.BeginUpdate


## Синтаксис


BeginUpdate;


## Описание


Метод BeginUpdate отключает любую перерисовку спидометра.


## Комментарии


Метод используется при изменении параметров элементов спидометра для избежания мерцания компонента, отображающего спидометр. Возобновление перерисовки осуществляется методом [EndUpdate](ISpeedometer.EndUpdate.htm).
