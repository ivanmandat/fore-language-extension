# ISpeedometer.EndUpdate


## Синтаксис


EndUpdate;


## Описание


Метод EndUpdate возобновляет перерисовку компонента.


## Комментарии


После вызова данного метода будут отрисованы все изменения, произведенные между вызовами методов [BeginUpdate](ISpeedometer.BeginUpdate.htm) и EndUpdate.
