# IPPTimeLine.HorizontalAlignment


## Синтаксис


HorizontalAlignment: [PPSpeedometerHorizontalAlignment](../../Enums/PPSpeedometerHorizontalAlignment.htm);


## Описание


Свойство HorizontalAlignment
 определяет способ выравнивания временной шкалы по горизонтали.


## Комментарии


Для определения способа выравнивания временной шкалы по вертикали используйте
 свойство [IPPTimeLine.VerticalAlignment](IPPTimeLine.VerticalAlignment.htm).
