# IPPTimeLine


Сборка: Speedometer;


## Описание


Интерфейс IPPTimeLine содержит
 свойства для настройки параметров временной шкалы спидометра аналитической
 панели.


## Иерархия наследования


           IPPTimeLine


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CurrentIndex](IPPTimeLine.CurrentIndex.htm)
		 Свойство CurrentIndex
		 возвращает индекс текущего шага временной шкалы.


		 ![](../../Property_Image.gif)
		 [DelayDuration](IPPTimeLine.DelayDuration.htm)
		 Свойство DelayDuration
		 определяет длительность задержки анимации.


		 ![](../../Property_Image.gif)
		 [HorizontalAlignment](IPPTimeLine.HorizontalAlignment.htm)
		 Свойство HorizontalAlignment
		 определяет способ выравнивания временной шкалы по горизонтали.


		 ![](../../Property_Image.gif)
		 [IsOverlap](IPPTimeLine.IsOverlap.htm)
		 Свойство IsOverlap
		 определяет, пересекает ли временная шкала спидометр.


		 ![](../../Property_Image.gif)
		 [ReturnToInitialState](IPPTimeLine.ReturnToInitialState.htm)
		 Свойство ReturnToInitialState
		 определяет, нужно ли возвращать временную шкалу в начальное состояние
		 после завершения анимации.


		 ![](../../Property_Image.gif)
		 [StepDuration](IPPTimeLine.StepDuration.htm)
		 Свойство StepDuration
		 определяет длительность шага анимации.


		 ![](../../Property_Image.gif)
		 [StepsNames](IPPTimeLine.StepsNames.htm)
		 Свойство StepsNames
		 определяет массив наименований шагов временной шкалы.


		 ![](../../Property_Image.gif)
		 [VerticalAlignment](IPPTimeLine.VerticalAlignment.htm)
		 Свойство VerticalAlignment
		 определяет способ выравнивания временной шкалы по вертикали.


		 ![](../../Property_Image.gif)
		 [Visible](IPPTimeLine.Visible.htm)
		 Свойство Visible определяет
		 видимость временной шкалы.
