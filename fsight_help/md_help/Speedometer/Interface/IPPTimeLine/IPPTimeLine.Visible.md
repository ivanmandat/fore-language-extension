# IPPTimeLine.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет
 видимость временной шкалы.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Временная шкала отображается;


	- False. Временная шкала
	 не отображается.
