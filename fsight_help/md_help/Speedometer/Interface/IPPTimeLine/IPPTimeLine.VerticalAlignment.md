# IPPTimeLine.VerticalAlignment


## Синтаксис


VerticalAlignment: [PPSpeedometerVerticalAlignment](../../Enums/PPSpeedometerVerticalAlignment.htm);


## Описание


Свойство VerticalAlignment определяет
 способ выравнивания временной шкалы по вертикали.


## Комментарии


Для определения способа выравнивания временной шкалы по горизонтали
 используйте свойство [IPPTimeLine.HorizontalAlignment](IPPTimeLine.HorizontalAlignment.htm).
