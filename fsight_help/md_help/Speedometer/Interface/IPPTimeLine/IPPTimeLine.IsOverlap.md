# IPPTimeLine.IsOverlap


## Синтаксис


IsOverlap: Boolean;


## Описание


Свойство IsOverlap определяет,
 пересекает ли временная шкала спидометр.


## Комментарии


Допустимые значения:


	- True. Временная шкала
	 пересекает спидометр;


	- False. Значение по умолчанию.
	 Временная шкала не пересекает спидометр.
