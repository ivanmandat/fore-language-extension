# IPPTimeLine.ReturnToInitialState


## Синтаксис


ReturnToInitialState: Boolean;


## Описание


Свойство ReturnToInitialState
 определяет, нужно ли возвращать временную шкалу в начальное состояние
 после завершения анимации.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Временная шкала вернется к первому шагу после завершения анимации;


	- False. При завершении
	 анимации текущим шагом временной шкалы останется последний шаг.
