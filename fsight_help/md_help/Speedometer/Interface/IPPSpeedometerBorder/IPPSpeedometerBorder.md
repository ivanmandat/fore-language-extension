# IPPSpeedometerBorder


Сборка: Speedometer;


## Описание


Интерфейс IPPSpeedometerBorder
 содержит свойства для настройки параметров границы спидометра.


## Иерархия наследования


           IPPSpeedometerBorder


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Background](IPPSpeedometerBorder.Background.htm)
		 Свойство Background
		 определяет кисть для отрисовки границы.


		 ![](../../Property_Image.gif)
		 [BorderRadius](IPPSpeedometerBorder.BorderRadius.htm)
		 Свойство BorderRadius
		 определяет радиус скругления границы.


		 ![](../../Property_Image.gif)
		 [Bottom](IPPSpeedometerBorder.Bottom.htm)
		 Свойство Bottom определяет
		 толщину нижней границы.


		 ![](../../Property_Image.gif)
		 [Left](IPPSpeedometerBorder.Left.htm)
		 Свойство Left
		 определяет толщину левой границы.


		 ![](../../Property_Image.gif)
		 [Right](IPPSpeedometerBorder.Right.htm)
		 Свойство Right
		 определяет толщину правой границы.


		 ![](../../Property_Image.gif)
		 [Top](IPPSpeedometerBorder.Top.htm)
		 Свойство Top
		 определяет толщину верхней границы.


		 ![](../../Property_Image.gif)
		 [Width](IPPSpeedometerBorder.Width.htm)
		 Свойство Width определяет
		 толщину всех границ.
