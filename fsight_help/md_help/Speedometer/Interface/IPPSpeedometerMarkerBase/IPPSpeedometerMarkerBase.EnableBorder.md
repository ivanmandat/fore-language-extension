# IPPSpeedometerMarkerBase.EnableBorder


## Синтаксис


EnableBorder: Boolean;


## Описание


Свойство EnableBorder определяет
 наличие границы маркера.


## Комментарии


Допустимые значения:


	- True. Маркер имеет границу;


	- False. Значение по умолчанию.
	 Маркер не имеет границы.
