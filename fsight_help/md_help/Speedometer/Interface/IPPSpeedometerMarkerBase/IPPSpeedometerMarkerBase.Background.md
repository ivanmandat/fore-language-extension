# IPPSpeedometerMarkerBase.Background


## Синтаксис


Background: [IGxBrush](ModDrawing.chm::/Interface/IGxBrush/IGxBrush.htm);


## Описание


Свойство Background определяет
 кисть для заливки фона маркера.


## Комментарии


Свойство Background актуально,
 если у свойства [IPPSpeedometerMarkerBase.EnableBrush](IPPSpeedometerMarkerBase.EnableBrush.htm)
 установлено значение True.
