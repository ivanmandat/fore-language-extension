# IPPSpeedometerMarkerBase.EnableBrush


## Синтаксис


EnableBrush: Boolean;


## Описание


Свойство EnableBrush определяет
 наличие заливки фона маркера.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Маркер имеет заливку фона;


	- False. Маркер не имеет
	 заливки фона.
