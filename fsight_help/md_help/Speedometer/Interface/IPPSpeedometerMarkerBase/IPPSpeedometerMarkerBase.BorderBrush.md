# IPPSpeedometerMarkerBase.BorderBrush


## Синтаксис


BorderBrush: [IGxBrush](ModDrawing.chm::/Interface/IGxBrush/IGxBrush.htm);


## Описание


Свойство BorderBrush определяет
 кисть для отрисовки границы маркера.


## Комментарии


Свойство BorderBrush актуально,
 если у свойства [IPPSpeedometerMarkerBase.EnableBorder](IPPSpeedometerMarkerBase.EnableBorder.htm)
 установлено значение True.
