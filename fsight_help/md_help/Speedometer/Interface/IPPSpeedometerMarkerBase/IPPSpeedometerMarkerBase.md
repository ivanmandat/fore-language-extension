# IPPSpeedometerMarkerBase


Сборка: Speedometer;


## Описание


Интерфейс IPPSpeedometerMarkerBase
 содержит свойства для настройки параметров отрисовки маркера спидометра.


## Иерархия наследования


           IPPSpeedometerMarkerBase

		 определяет кисть для отрисовки границы маркера.

		 [BorderStyle](IPPSpeedometerMarkerBase.BorderStyle.htm)
		 Свойство BorderStyle

		 Свойство BorderWidth
		 определяет толщину границы маркера.


		 ![](../../Property_Image.gif)

		 Свойство EnableBorder

		 Свойство EnableBrush

		 Свойство LabelPlacementStyle

		 Свойство MarkerType

		 [Offset](IPPSpeedometerMarkerBase.Offset.htm)
