# IPPSpeedometerMarkerBase.BorderStyle


## Синтаксис


BorderStyle: [GxDashStyle](ModDrawing.chm::/Enums/GxDashStyle.htm);


## Описание


Свойство BorderStyle определяет
 стиль отрисовки границы маркера.


## Комментарии


Свойство BorderStyle актуально,
 если у свойства [IPPSpeedometerMarkerBase.EnableBorder](IPPSpeedometerMarkerBase.EnableBorder.htm)
 установлено значение True.
