# IPPSpeedometerScale.EnableBrush


## Синтаксис


EnableBrush: Boolean;


## Описание


Свойство EnableBrush определяет
 наличие заливки фона шкалы.


## Комментарии


Допустимые значения:


	- True. Шкала имеет заливку
	 фона;


	- False. Шкала не имеет
	 заливки фона.
