# IPPSpeedometerScale.DisplayLabels


## Синтаксис


DisplayLabels: Boolean;


## Описание


Свойство DisplayLabels определяет
 видимость подписей данных шкалы.


## Комментарии


Допустимые значения:


	- True. Подписи данных
	 отображаются;


	- False. Подписи данных
	 не отображаются.
