# IPPSpeedometerScale.BorderStyle


## Синтаксис


BorderStyle: [GxDashStyle](ModDrawing.chm::/Enums/GxDashStyle.htm);


## Описание


Свойство BorderStyle определяет
 стиль отрисовки границы шкалы.


## Комментарии


Свойство BorderStyle актуально,
 если у свойства [IPPSpeedometerScale.EnableBorder](IPPSpeedometerScale.EnableBorder.htm)
 установлено значение True.
