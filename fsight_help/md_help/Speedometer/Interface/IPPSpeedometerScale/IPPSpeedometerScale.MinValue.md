# IPPSpeedometerScale.MinValue


## Синтаксис


MinValue: Double;


## Описание


Свойство MinValue определяет
 минимальное значение шкалы.


## Комментарии


Для определения максимального значения шкалы используйте свойство [IPPSpeedometerScale.MaxValue](IPPSpeedometerScale.MaxValue.htm).
