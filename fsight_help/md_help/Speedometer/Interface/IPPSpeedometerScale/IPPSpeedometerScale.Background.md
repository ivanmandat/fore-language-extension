# IPPSpeedometerScale.Background


## Синтаксис


Background: [IGxBrush](ModDrawing.chm::/Interface/IGxBrush/IGxBrush.htm);


## Описание


Свойство Background определяет
 кисть для заливки фона шкалы.


## Комментарии


Свойство Background актуально,
 если у свойства [IPPSpeedometerScale.EnableBrush](IPPSpeedometerScale.EnableBrush.htm)
 установлено значение True.
