# IPPSpeedometerScale.EnableBorder


## Синтаксис


EnableBorder: Boolean;


## Описание


Свойство EnableBorder определяет
 наличие границы шкалы.


## Комментарии


Допустимые значения:


	- True. Шкала имеет границу;


	- False. Шкала не имеет
	 границы.
