# IPPSpeedometerScale.MinorTick


## Синтаксис


MinorTick: [IPPSpeedometerMarkerBase](../IPPSpeedometerMarkerBase/IPPSpeedometerMarkerBase.htm);


## Описание


Свойство MinorTick определяет
 параметры отрисовки промежуточных маркеров шкалы.


## Комментарии


Для определения параметров отрисовки основных маркеров шкалы используйте
 свойство [IPPSpeedometerScale.MajorTick](IPPSpeedometerScale.MajorTick.htm).
