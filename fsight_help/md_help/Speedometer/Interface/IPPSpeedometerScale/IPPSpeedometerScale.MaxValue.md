# IPPSpeedometerScale.MaxValue


## Синтаксис


MaxValue: Double;


## Описание


Свойство MaxValue определяет
 максимальное значение шкалы.


## Комментарии


Для определения минимального значения шкалы используйте свойство [IPPSpeedometerScale.MinValue](IPPSpeedometerScale.MinValue.htm).
