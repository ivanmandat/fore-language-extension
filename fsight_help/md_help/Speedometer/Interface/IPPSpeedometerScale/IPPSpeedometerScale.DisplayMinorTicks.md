# IPPSpeedometerScale.DisplayMinorTicks


## Синтаксис


DisplayMinorTicks: Boolean;


## Описание


Свойство DisplayMinorTicks определяет
 видимость промежуточных маркеров шкалы.


## Комментарии


Допустимые значения:


	- True. Промежуточные
	 маркеры отображаются;


	- False. Промежуточные
	 маркеры не отображаются.
