# IPPSpeedometerScale.PassedMajorTick


## Синтаксис


PassedMajorTick: [IPPSpeedometerMarkerBase](../IPPSpeedometerMarkerBase/IPPSpeedometerMarkerBase.htm);


## Описание


Свойство PassedMajorTick определяет
 параметры отрисовки пройденных указателем основных маркеров шкалы.


## Комментарии


Для определения параметров отрисовки пройденных указателем промежуточных
 маркеров шкалы используйте свойство [IPPSpeedometerScale.PassedMinorTick](IPPSpeedometerScale.PassedMinorTick.htm).
