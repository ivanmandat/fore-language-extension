# IPPSpeedometerScale.BorderBrush


## Синтаксис


BorderBrush: [IGxBrush](ModDrawing.chm::/Interface/IGxBrush/IGxBrush.htm);


## Описание


Свойство BorderBrush определяет
 кисть для отрисовки границы шкалы.


## Комментарии


Свойство BorderBrush актуально,
 если у свойства [IPPSpeedometerScale.EnableBorder](IPPSpeedometerScale.EnableBorder.htm)
 установлено значение True.
