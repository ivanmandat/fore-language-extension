# IPPSpeedometerScale.PassedMinorTick


## Синтаксис


PassedMinorTick: [IPPSpeedometerMarkerBase](../IPPSpeedometerMarkerBase/IPPSpeedometerMarkerBase.htm);


## Описание


Свойство PassedMinorTick определяет
 параметры отрисовки пройденных указателем промежуточных маркеров шкалы.


## Комментарии


Для определения параметров отрисовки пройденных указателем основных
 маркеров шкалы используйте свойство [IPPSpeedometerScale.PassedMajorTick](IPPSpeedometerScale.PassedMajorTick.htm).
