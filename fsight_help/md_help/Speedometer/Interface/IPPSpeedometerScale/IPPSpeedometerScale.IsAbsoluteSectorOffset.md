# IPPSpeedometerScale.IsAbsoluteSectorOffset


## Синтаксис


IsAbsoluteSectorOffset: Boolean;


## Описание


Свойство IsAbsoluteSectorOffset
 определяет единицу измерения смещения секторов.


## Комментарии


Допустимые значения:


	- True. Смещение измеряется
	 в пикселях;


	- False. Смещение измеряется
	 в процентах.
