# ISpeedometerArrowViews


## Описание


Интерфейс ISpeedometerArrowViews
 содержит свойства коллекции стилей оформления стрелок спидометра.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ISpeedometerArrowViews.Count.htm)
		 Свойство Count определяет
		 количество стилей оформления стрелок спидометра.


		 ![](../../Property_Image.gif)
		 [Item](ISpeedometerArrowViews.Item.htm)
		 Свойство Item возвращает
		 стиль оформления стрелки спидометра.
