# ISpeedometerBackground


## Описание


Интерфейс ISpeedometerBackground
 содержит свойства для настройки фона различных элементов спидометра.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Color](ISpeedometerBackground.Color.htm)
		 Свойство Color определяет
		 цвет заливки фона.


		 ![](../../Property_Image.gif)
		 [GradientAngle](ISpeedometerBackground.GradientAngle.htm)
		 Свойство GradientAngle
		 определяет угол наклона градиентной заливки.


		 ![](../../Property_Image.gif)
		 [GradientColor](ISpeedometerBackground.GradientColor.htm)
		 Свойство GradientColor
		 определяет второй цвет, используемый при градиентной заливке.


		 ![](../../Property_Image.gif)
		 [HatchStyle](ISpeedometerBackground.HatchStyle.htm)
		 Свойство HatchStyle
		 определяет тип штриховки, используемой для заливки фона.


		 ![](../../Property_Image.gif)
		 [Image](ISpeedometerBackground.Image.htm)
		 Свойство Image определяет
		 изображение, которое будет использоваться в качестве фона.


		 ![](../../Property_Image.gif)
		 [ImageDrawStyle](ISpeedometerBackground.ImageDrawStyle.htm)
		 Свойство ImageDrawStyle
		 определяет способ отрисовки изображения в области фона.


		 ![](../../Property_Image.gif)
		 [Type](ISpeedometerBackground.Type.htm)
		 Свойство Type определяет
		 тип фона спидометра.
