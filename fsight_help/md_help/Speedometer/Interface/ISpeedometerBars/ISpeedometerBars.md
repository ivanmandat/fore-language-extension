# ISpeedometerBars


## Описание


Интерфейс ISpeedometerBars содержит
 свойства коллекции сегментов цветной интервальной полосы спидометра.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ISpeedometerBars.Count.htm)
		 Свойство Count определяет
		 количество сегментов цветной интервальной полосы спидометра.


		 ![](../../Property_Image.gif)
		 [Item](ISpeedometerBars.Item.htm)
		 Свойство Item возвращает
		 параметры указанного сегмента цветной интервальной полосы спидометра.
