# IPPSpeedometerTrend.IsAbsoluteLabelOffset


## Синтаксис


IsAbsoluteLabelOffset: Boolean;


## Описание


Свойство IsAbsoluteLabelOffset
 определяет единицу измерения смещения текстовой метки тренда.


## Комментарии


Допустимые значения:


	- True. Смещение измеряется
	 в пикселях;


	- False. Смещение измеряется
	 в процентах.
