# IPPSpeedometerTrend.LabelOffset


## Синтаксис


LabelOffset: Double;


## Описание


Свойство LabelOffset определяет
 смещение подписи тренда.


## Комментарии


Для определения единицы измерения смещения используйте свойство [IPPSpeedometerTrend.IsAbsoluteLabelOffset](IPPSpeedometerTrend.IsAbsoluteLabelOffset.htm).
