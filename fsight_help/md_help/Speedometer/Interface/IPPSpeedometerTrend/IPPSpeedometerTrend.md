# IPPSpeedometerTrend


Сборка: Speedometer;


## Описание


Интерфейс IPPSpeedometerTrend
 содержит свойства для настройки параметров тренда шкалы данных спидометра.


## Иерархия наследования


           IPPSpeedometerTrend


## Комментарии


Тренд - отмеченное значение на шкале данных спидометра.
