# ISpeedometerTickInfo


## Описание


Интерфейс ISpeedometerTickInfo
 содержит свойства основных и дополнительных делений шкалы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Color](ISpeedometerTickInfo.Color.htm)
		 Свойство Color определяет
		 цвет заливки делений шкалы.


		 ![](../../Property_Image.gif)
		 [Height](ISpeedometerTickInfo.Height.htm)
		 Свойство Height определяет
		 высоту делений шкалы.


		 ![](../../Property_Image.gif)
		 [Pen](ISpeedometerTickInfo.Pen.htm)
		 Свойство Pen определяет
		 перо, которым отрисовывается граница деления шкалы спидометра.


		 ![](../../Property_Image.gif)
		 [Proportion](ISpeedometerTickInfo.Proportion.htm)
		 Свойство Proportion
		 определяет отношение высоты к ширине деления.


		 ![](../../Property_Image.gif)
		 [Shape](ISpeedometerTickInfo.Shape.htm)
		 Свойство Shape определяет
		 форму делений шкалы.
