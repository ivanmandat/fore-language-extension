# ISpeedometerView


## Описание


Интерфейс ISpeedometerView содержит
 свойства для настройки параметров отображения спидометра.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Background](ISpeedometerView.Background.htm)
		 Свойство Background
		 определяет параметры отображения фона спидометра.


		 ![](../../Property_Image.gif)
		 [BorderPen](ISpeedometerView.BorderPen.htm)
		 Свойство BorderPen
		 определяет перо, которым будет отрисована граница спидометра.


		 ![](../../Property_Image.gif)
		 [BottomMargin](ISpeedometerView.BottomMargin.htm)
		 Свойство BottomMargin
		 определяет отступ спидометра от нижнего края компонента.


		 ![](../../Property_Image.gif)
		 [CentralShape](ISpeedometerView.CentralShape.htm)
		 Свойство CentralShape
		 возвращает параметры круга, расположенного в центре спидометра.


		 ![](../../Property_Image.gif)
		 [LeftMargin](ISpeedometerView.LeftMargin.htm)
		 Свойство LeftMargin
		 определяет отступ спидометра от левого края компонента.


		 ![](../../Property_Image.gif)
		 [RightMargin](ISpeedometerView.RightMargin.htm)
		 Свойство RightMargin
		 определяет отступ спидометра от правого края компонента.


		 ![](../../Property_Image.gif)
		 [ScaleViews](ISpeedometerView.ScaleViews.htm)
		 Свойство ScaleViews
		 возвращает коллекцию стилей оформления шкал спидометра.


		 ![](../../Property_Image.gif)
		 [TopMargin](ISpeedometerView.TopMargin.htm)
		 Свойство TopMargin
		 определяет отступ спидометра от верхнего края компонента.
