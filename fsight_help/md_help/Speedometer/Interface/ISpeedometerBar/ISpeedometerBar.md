# ISpeedometerBar


## Описание


Интерфейс ISpeedometerBar содержит
 свойства сегмента цветной интервальной полосы спидометра.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [EndColor](ISpeedometerBar.EndColor.htm)
		 Свойство EndColor определяет
		 конечный цвет градиентной заливки сегмента.


		 ![](../../Property_Image.gif)
		 [Part](ISpeedometerBar.Part.htm)
		 Свойство Part определяет
		 долю шкалы, занимаемую данным сегментом.


		 ![](../../Property_Image.gif)
		 [Shape](ISpeedometerBar.Shape.htm)
		 Свойство Shape определяет
		 фигуру сегмента.


		 ![](../../Property_Image.gif)
		 [StartColor](ISpeedometerBar.StartColor.htm)
		 Свойство StartColor
		 определяет начальный цвет градиентной заливки сегмента.
