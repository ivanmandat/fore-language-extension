# ISpeedometerFilledBar


## Описание


Интерфейс ISpeedometerFilledBar
 содержит свойства цветной интервальной полосы спидометра.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Bars](ISpeedometerFilledBar.Bars.htm)
		 Свойство Bars возвращает
		 коллекцию сегментов цветной интервальной полосы спидометра.


		 ![](../../Property_Image.gif)
		 [EndRadius](ISpeedometerFilledBar.EndRadius.htm)
		 Свойство EndRadius
		 определяет конечный ограничивающий радиус полосы.


		 ![](../../Property_Image.gif)
		 [StartRadius](ISpeedometerFilledBar.StartRadius.htm)
		 Свойство StartRadius
		 определяет начальный ограничивающий радиус полосы.
