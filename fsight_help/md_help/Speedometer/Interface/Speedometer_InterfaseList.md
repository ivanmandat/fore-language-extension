# Интерфейсы сборки Speedometer


В «Форсайт. Аналитическая платформа»
 используется два вида спидометров:


	- Визуализатор «[Спидометр](UiReport.chm::/Desktop/Objects/UiReport_Objects_Spedometer.htm)»
	 используется в отчётах и может быть создан как через отчёты, так и
	 при работе с дизайнером форм Fore;


	- Визуализатор «[Индикатор](UiAdhoc.chm::/Blocks/Gadgets/Indicator.htm)»
	 используется в аналитических панелях. Создавать и просматривать данный
	 визуализатор можно только через веб-приложение, но есть возможность
	 редактирования параметров отображения при помощи языка Fore.
