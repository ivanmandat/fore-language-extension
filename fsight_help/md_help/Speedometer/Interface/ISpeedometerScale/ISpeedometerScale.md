# ISpeedometerScale


## Описание


Интерфейс ISpeedometerScale
 содержит свойства шкалы спидометра.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Arrows](ISpeedometerScale.Arrows.htm)
		 Свойство Arrows возвращает
		 коллекцию стрелок шкалы спидометра.


		 ![](../../Property_Image.gif)
		 [MajorTickCount](ISpeedometerScale.MajorTickCount.htm)
		 Свойство MajorTickCount
		 определяет количество основных делений шкалы.


		 ![](../../Property_Image.gif)
		 [MaxValue](ISpeedometerScale.MaxValue.htm)
		 Свойство MaxValue определяет
		 максимальное значение, которое может быть отображено на шкале.


		 ![](../../Property_Image.gif)
		 [MinorTickCount](ISpeedometerScale.MinorTickCount.htm)
		 Свойство MinorTickCount
		 определяет количество дополнительных делений шкалы.


		 ![](../../Property_Image.gif)
		 [MinValue](ISpeedometerScale.MinValue.htm)
		 Свойство MinValue определяет
		 минимальное значение, которое может быть отображено на шкале.


		 ![](../../Property_Image.gif)
		 [Precision](ISpeedometerScale.Precision.htm)
		 Свойство Precision
		 позволяет определить точность отображения значений шкалы.


		 ![](../../Property_Image.gif)
		 [View](ISpeedometerScale.View.htm)
		 Свойство View возвращает
		 параметры оформления шкалы.
