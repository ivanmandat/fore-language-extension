# IPPSpeedometerTrends


Сборка: Speedometer;


## Описание


Интерфейс IPPSpeedometerTrends
 предназначен для работы с коллекцией трендов шкалы спидометра.


## Иерархия наследования


           IPPSpeedometerTrends


## Комментарии


Тренд - отмеченное значение на шкале данных спидометра.
