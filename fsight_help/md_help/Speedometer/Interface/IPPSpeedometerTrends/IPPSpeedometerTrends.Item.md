# IPPSpeedometerTrends.Item


## Синтаксис


Item(Index: Integer): [IPPSpeedometerTrend](../IPPSpeedometerTrend/IPPSpeedometerTrend.htm);


## Параметры


Index. Индекс тренда, который
 требуется вернуть.


## Описание


Свойство Item возвращает тренд
 коллекции по его индексу.
