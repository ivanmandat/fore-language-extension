# IPPSpeedometerLabels.Item


## Синтаксис


Item(Index: Integer): [IPPSpeedometerLabel](../IPPSpeedometerLabel/IPPSpeedometerLabel.htm);


## Параметры


Index. Индекс текстовой метки,
 которую требуется вернуть.


## Описание


Свойство Item возвращает текстовую
 метку коллекции по её индексу.
