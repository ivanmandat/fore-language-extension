# IPPSpeedometerLabels


Сборка: Speedometer;


## Описание


Интерфейс IPPSpeedometerLabels
 предназначен для работы с коллекцией текстовых меток спидометра.


## Иерархия наследования


           IPPSpeedometerLabels


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPPSpeedometerLabels.Count.htm)
		 Свойство Count возвращает
		 размер коллекции текстовых меток.


		 ![](../../Property_Image.gif)
		 [Font](IPPSpeedometerLabels.Font.htm)
		 Свойство Font определяет
		 параметры шрифта текста, отображаемого на текстовых метках коллекции.


		 ![](../../Property_Image.gif)
		 [FontColor](IPPSpeedometerLabels.FontColor.htm)
		 Свойство FontColor
		 определяет цвет текста, отображаемого на текстовых метках коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IPPSpeedometerLabels.Item.htm)
		 Свойство Item возвращает
		 текстовую метку коллекции по её индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IPPSpeedometerLabels.Add.htm)
		 Метод Add добавляет
		 текстовую метку в коллекцию.


		 ![](../../Sub_Image.gif)
		 [RemoveAll](IPPSpeedometerLabels.RemoveAll.htm)
		 Метод RemoveAll удаляет
		 все текстовые метки из коллекции.
