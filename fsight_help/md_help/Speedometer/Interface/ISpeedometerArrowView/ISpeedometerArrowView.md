# ISpeedometerArrowView


## Описание


Интерфейс ISpeedometerArrowView
 содержит свойства, определяющие параметры оформления стрелки.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AxisOffset](ISpeedometerArrowView.AxisOffset.htm)
		 Свойство AxisOffset
		 определяет смещение стрелки от центра шкалы спидометра.


		 ![](../../Property_Image.gif)
		 [Background](ISpeedometerArrowView.Background.htm)
		 Свойство Background
		 определяет оформление фона стрелки.


		 ![](../../Property_Image.gif)
		 [Length](ISpeedometerArrowView.Length.htm)
		 Свойство Length определяет
		 длину стрелки.


		 ![](../../Property_Image.gif)
		 [Pen](ISpeedometerArrowView.Pen.htm)
		 Свойство Pen определяет
		 перо, которым будет отрисована граница стрелка.


		 ![](../../Property_Image.gif)
		 [Shape](ISpeedometerArrowView.Shape.htm)
		 Свойство Shape определяет
		 форму стрелки.


		 ![](../../Property_Image.gif)
		 [Width](ISpeedometerArrowView.Width.htm)
		 Свойство Width определяет
		 ширину стрелки.
