# ISpeedometerScaleViews


## Описание


Интерфейс ISpeedometerScaleViews
 содержит свойства коллекции стилей оформления шкал спидометра.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ISpeedometerScaleViews.Count.htm)
		 Свойство Count определяет
		 количество стилей оформления шкал спидометра.


		 ![](../../Property_Image.gif)
		 [Item](ISpeedometerScaleViews.Item.htm)
		 Свойство Item возвращает
		 параметры стиля оформления шкалы спидометра.
