# IPPSpeedometerArrowBase


Сборка: Speedometer;


## Описание


Интерфейс IPPSpeedometerArrowBase
 содержит свойства для настройки параметров стрелки шкалы спидометра.


## Иерархия наследования


           IPPSpeedometerArrowBase


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Background](IPPSpeedometerArrowBase.Background.htm)
		 Свойство Background
		 определяет кисть для заливки фона стрелки.


		 ![](../../Property_Image.gif)
		 [BorderBrush](IPPSpeedometerArrowBase.BorderBrush.htm)
		 Свойство BorderBrush
		 определяет кисть для отрисовки границы стрелки.


		 ![](../../Property_Image.gif)
		 [EnableBrush](IPPSpeedometerArrowBase.EnableBrush.htm)
		 Свойство EnableBrush
		 определяет наличие заливки фона стрелки.


		 ![](../../Property_Image.gif)
		 [ID](IPPSpeedometerArrowBase.ID.htm)
		 Свойство ID определяет
		 идентификатор стрелки.


		 ![](../../Property_Image.gif)
		 [Radius](IPPSpeedometerArrowBase.Radius.htm)
		 Свойство Radius определяет
		 длину стрелки.


		 ![](../../Property_Image.gif)
		 [Scale](IPPSpeedometerArrowBase.Scale.htm)
		 Свойство Scale определяет
		 шкалу, в которой расположена стрелка.


		 ![](../../Property_Image.gif)
		 [Size](IPPSpeedometerArrowBase.Size.htm)
		 Свойство Size определяет
		 толщину стрелки.


		 ![](../../Property_Image.gif)
		 [Tail](IPPSpeedometerArrowBase.Tail.htm)
		 Свойство Tail определяет
		 длину хвоста стрелки.


		 ![](../../Property_Image.gif)
		 [Type](IPPSpeedometerArrowBase.Type.htm)
		 Свойство Type определяет
		 форму стрелки.


		 ![](../../Property_Image.gif)
		 [Value](IPPSpeedometerArrowBase.Value.htm)
		 Свойство Value определяет
		 значение шкалы, на которое указывает стрелка.
