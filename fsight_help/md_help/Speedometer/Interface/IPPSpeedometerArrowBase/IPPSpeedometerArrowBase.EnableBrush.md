# IPPSpeedometerArrowBase.EnableBrush


## Синтаксис


EnableBrush: Boolean;


## Описание


Свойство EnableBrush определяет
 наличие заливки фона стрелки.


## Комментарии


Допустимые значения:


	- True. Стрелка имеет
	 заливку фона;


	- False. Значение по умолчанию.
	 Стрелка не имеет заливки фона.
