# IPPSpeedometerBorders.Item


## Синтаксис


Item(Index: Integer): [IPPSpeedometerBorder](../IPPSpeedometerBorder/IPPSpeedometerBorder.htm);


## Параметры


Index. Индекс границы, которую
 требуется вернуть.


## Описание


Свойство Item возвращает границу
 коллекции по её индексу.
