# IPPSpeedometerBorders


Сборка: Speedometer;


## Описание


Интерфейс IPPSpeedometerBorders
 предназначен для работы с коллекцией границ спидометра.


## Иерархия наследования


           IPPSpeedometerBorders


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPPSpeedometerBorders.Count.htm)
		 Свойство Count возвращает
		 размер коллекции границ.


		 ![](../../Property_Image.gif)
		 [Item](IPPSpeedometerBorders.Item.htm)
		 Свойство Item возвращает
		 границу коллекции по её индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IPPSpeedometerBorders.Add.htm)
		 Метод Add добавляет
		 границу в коллекцию.


		 ![](../../Sub_Image.gif)
		 [CreateBorder](IPPSpeedometerBorders.CreateBorder.htm)
		 Метод CreateBorder
		 создает границу с заданной толщиной.


		 ![](../../Sub_Image.gif)
		 [RemoveAll](IPPSpeedometerBorders.RemoveAll.htm)
		 Метод RemoveAll удаляет
		 все границы из коллекции.
