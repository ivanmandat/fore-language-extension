# ISpeedometerScales


## Описание


Интерфейс ISpeedometerScales
 содержит свойства коллекции шкал спидометра.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ISpeedometerScales.Count.htm)
		 Свойство Count определяет
		 количество шкал спидометра.


		 ![](../../Property_Image.gif)
		 [Item](ISpeedometerScales.Item.htm)
		 Свойство Item возвращает
		 параметры указанной шкалы спидометра.
