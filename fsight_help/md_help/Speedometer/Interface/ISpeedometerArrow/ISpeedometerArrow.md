# ISpeedometerArrow


## Описание


Интерфейс ISpeedometerArrow
 содержит свойства стрелки шкалы спидометра.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Value](ISpeedometerArrow.Value.htm)
		 Свойство Value определяет
		 значение, которое индицирует стрелка.


		 ![](../../Property_Image.gif)
		 [View](ISpeedometerArrow.View.htm)
		 Свойство View возвращает
		 параметры оформления стрелки спидометра.
