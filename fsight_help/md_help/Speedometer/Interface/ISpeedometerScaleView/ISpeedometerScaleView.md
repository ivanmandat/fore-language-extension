# ISpeedometerScaleView


## Описание


Интерфейс ISpeedometerScaleView
 содержит свойства, определяющие стиль оформления шкалы спидометра.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ArrowViews](ISpeedometerScaleView.ArrowViews.htm)
		 Свойство ArrowViews
		 возвращает коллекцию стилей оформления стрелок шкалы спидометра.


		 ![](../../Property_Image.gif)
		 [DeltaAngle](ISpeedometerScaleView.DeltaAngle.htm)
		 Свойство DeltaAngle
		 определяет угол рабочей области шкалы.


		 ![](../../Property_Image.gif)
		 [FilledBar](ISpeedometerScaleView.FilledBar.htm)
		 Свойство FilledBar
		 возвращает параметры цветной интервальной полосы спидометра.


		 ![](../../Property_Image.gif)
		 [Font](ISpeedometerScaleView.Font.htm)
		 Свойство Font определяет
		 шрифт, используемый для отображения значений делений шкалы спидометра.


		 ![](../../Property_Image.gif)
		 [FontColor](ISpeedometerScaleView.FontColor.htm)
		 Свойство FontColor
		 определяет цвет шрифта, используемого для отображения значений
		 делений шкалы спидометра.


		 ![](../../Property_Image.gif)
		 [MajorTickInfo](ISpeedometerScaleView.MajorTickInfo.htm)
		 Свойство MajorTickInfo
		 возвращает параметры основных делений шкалы.


		 ![](../../Property_Image.gif)
		 [MinorTickInfo](ISpeedometerScaleView.MinorTickInfo.htm)
		 Свойство MinorTickInfo
		 возвращает параметры дополнительных делений шкалы.


		 ![](../../Property_Image.gif)
		 [ScaleLinePen](ISpeedometerScaleView.ScaleLinePen.htm)
		 Свойство ScaleLinePen
		 определяет перо, с помощью которого будет отрисована линия, расположенная
		 в области значений делений шкалы.


		 ![](../../Property_Image.gif)
		 [StartAngle](ISpeedometerScaleView.StartAngle.htm)
		 Свойство StartAngle
		 определяет начальный угол шкалы.


		 ![](../../Property_Image.gif)
		 [TextPlacement](ISpeedometerScaleView.TextPlacement.htm)
		 Свойство TextPlacement
		 определяет размещение текста значений делений шкалы относительно
		 самих делений.
