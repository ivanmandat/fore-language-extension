# IPPSpeedometerSector


Сборка: Speedometer;


## Описание


Интерфейс IPPSpeedometerSector
 содержит свойства для настройки параметров сектора шкалы данных спидометра.


## Иерархия наследования


           IPPSpeedometerSector


## Свойства


		 Имя свойства
		 Краткое описание

		 Свойство Background

		 ![](../../Property_Image.gif)
		 [EnableBrush](IPPSpeedometerSector.EnableBrush.htm)
		 Свойство EnableBrush
		 определяет наличие заливки фона сектора.

		 Свойство EndSize определяет

		 Свойство EndValue определяет

		 [Offset](IPPSpeedometerSector.Offset.htm)
		 Свойство Offset определяет
		 смещение сектора.


		 ![](../../Property_Image.gif)
		 [StartSize](IPPSpeedometerSector.StartSize.htm)
		 Свойство StartSize
		 определяет толщину сектора в начале.


		 ![](../../Property_Image.gif)
		 [StartValue](IPPSpeedometerSector.StartValue.htm)
		 Свойство StartValue
