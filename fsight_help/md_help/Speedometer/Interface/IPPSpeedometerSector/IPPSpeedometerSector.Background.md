# IPPSpeedometerSector.Background


## Синтаксис


Background: [IGxBrush](ModDrawing.chm::/Interface/IGxBrush/IGxBrush.htm);


## Описание


Свойство Background определяет
 кисть для заливки фона сектора.


## Комментарии


Свойство Background актуально,
 если у свойства [IPPSpeedometerSector.EnableBrush](IPPSpeedometerSector.EnableBrush.htm)
 установлено значение True.
