# IPPSpeedometerSector.EnableBrush


## Синтаксис


EnableBrush: Boolean;


## Описание


Свойство EnableBrush определяет
 наличие заливки фона сектора.


## Комментарии


Допустимые значения:


	- True. Сектор имеет заливку
	 фона;


	- False. Сектор не имеет
	 заливки фона.
