# ISpeedometerCentralShape


## Описание


Интерфейс ISpeedometerCentralShape
 содержит свойства круга, расположенного в центре спидометра.


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Background](ISpeedometerCentralShape.Background.htm)
		 Свойство Background
		 определяет фон заливки круга, находящегося в центре поворота стрелки.


		 ![](../../Property_Image.gif)
		 [Radius](ISpeedometerCentralShape.Radius.htm)
		 Свойство Radius определяет
		 радиус круга, находящегося в центре поворота стрелки.


		 ![](../../Property_Image.gif)
		 [ShapeOrder](ISpeedometerCentralShape.ShapeOrder.htm)
		 Свойство ShapeOrder
		 определяет порядок расположения круга относительно стрелки.
