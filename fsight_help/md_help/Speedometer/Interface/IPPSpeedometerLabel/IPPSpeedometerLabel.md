# IPPSpeedometerLabel


Сборка: Speedometer;


## Описание


Интерфейс IPPSpeedometerLabel
 содержит свойства для настройки параметров текстовой метки спидометра.


## Иерархия наследования


           IPPSpeedometerLabel


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AutoSize](IPPSpeedometerLabel.AutoSize.htm)
		 Свойство AutoSize определяет,
		 устанавливать ли размер текстовой метки автоматически.


		 ![](../../Property_Image.gif)
		 [Background](IPPSpeedometerLabel.Background.htm)
		 Свойство Background
		 определяет кисть для заливки фона текстовой метки.


		 ![](../../Property_Image.gif)
		 [BorderBrush](IPPSpeedometerLabel.BorderBrush.htm)
		 Свойство BorderBrush
		 определяет кисть для отрисовки границы текстовой метки.


		 ![](../../Property_Image.gif)
		 [BorderStyle](IPPSpeedometerLabel.BorderStyle.htm)
		 Свойство BorderStyle
		 определяет стиль отрисовки границы текстовой метки.


		 ![](../../Property_Image.gif)
		 [BorderWidth](IPPSpeedometerLabel.BorderWidth.htm)
		 Свойство BorderWidth
		 определяет толщину границы текстовой метки.


		 ![](../../Property_Image.gif)
		 [EnableBorder](IPPSpeedometerLabel.EnableBorder.htm)
		 Свойство EnableBorder
		 определяет наличие границы текстовой метки.


		 ![](../../Property_Image.gif)
		 [EnableBrush](IPPSpeedometerLabel.EnableBrush.htm)
		 Свойство EnableBrush
		 определяет наличие заливки фона текстовой метки.


		 ![](../../Property_Image.gif)
		 [Font](IPPSpeedometerLabel.Font.htm)
		 Свойство Font определяет
		 параметры шрифта текста, отображаемого на текстовой метке.


		 ![](../../Property_Image.gif)
		 [FontColor](IPPSpeedometerLabel.FontColor.htm)
		 Свойство FontColor
		 определяет цвет текста, отображаемого на текстовой метке.


		 ![](../../Property_Image.gif)
		 [Height](IPPSpeedometerLabel.Height.htm)
		 Свойство Height определяет
		 высоту текстовой метки.


		 ![](../../Property_Image.gif)
		 [Offset](IPPSpeedometerLabel.Offset.htm)
		 Свойство Offset определяет
		 смещение текстовой метки.


		 ![](../../Property_Image.gif)
		 [Text](IPPSpeedometerLabel.Text.htm)
		 Свойство Text определяет
		 текст, отображаемый на текстовой метке.


		 ![](../../Property_Image.gif)
		 [TextAlign](IPPSpeedometerLabel.TextAlign.htm)
		 Свойство TextAlign
		 определяет способ выравнивания текста, отображаемого на текстовой
		 метке.


		 ![](../../Property_Image.gif)
		 [Width](IPPSpeedometerLabel.Width.htm)
		 Свойство Width определяет
		 ширину текстовой метки.
