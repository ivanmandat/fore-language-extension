# IPPSpeedometerLabel.EnableBorder


## Синтаксис


EnableBorder: Boolean;


## Описание


Свойство EnableBorder определяет
 наличие границы текстовой метки.


## Комментарии


Допустимые значения:


	- True. Текстовая метка
	 имеет границу;


	- False. Текстовая метка
	 не имеет границы.
