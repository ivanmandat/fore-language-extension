# IPPSpeedometerLabel.AutoSize


## Синтаксис


AutoSize: Boolean;


## Описание


Свойство AutoSize определяет
 признак автоматической установки размера текстовой метки.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Размер текстовой метки устанавливается автоматически;


	- False. Размер текстовой
	 метки устанавливается вручную.
