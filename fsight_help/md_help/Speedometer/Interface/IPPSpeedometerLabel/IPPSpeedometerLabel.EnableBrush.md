# IPPSpeedometerLabel.EnableBrush


## Синтаксис


EnableBrush: Boolean;


## Описание


Свойство EnableBrush определяет
 наличие заливки фона текстовой метки.


## Комментарии


Допустимые значения:


	- True. Текстовая метка
	 имеет заливку фона;


	- False. Текстовая метка
	 не имеет заливки фона.
