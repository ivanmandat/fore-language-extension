# IPPSpeedometerLabel.BorderBrush


## Синтаксис


BorderBrush: [IGxBrush](ModDrawing.chm::/Interface/IGxBrush/IGxBrush.htm);


## Описание


Свойство BorderBrush определяет
 кисть для отрисовки границы текстовой метки.


## Комментарии


Свойство BorderBrush актуально,
 если у свойства [IPPSpeedometerLabel.EnableBorder](IPPSpeedometerLabel.EnableBorder.htm)
 установлено значение True.
