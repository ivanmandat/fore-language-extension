# IPPSpeedometerLabel.Background


## Синтаксис


Background: [IGxBrush](ModDrawing.chm::/Interface/IGxBrush/IGxBrush.htm);


## Описание


Свойство Background определяет
 кисть для заливки фона текстовой метки.


## Комментарии


Свойство Background актуально,
 если у свойства [IPPSpeedometerLabel.EnableBrush](IPPSpeedometerLabel.EnableBrush.htm)
 установлено значение True.
