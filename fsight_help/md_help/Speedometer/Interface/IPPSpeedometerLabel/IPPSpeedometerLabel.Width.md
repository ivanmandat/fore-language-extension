# IPPSpeedometerLabel.Width


## Синтаксис


Width: Double;


## Описание


Свойство Width определяет ширину
 текстовой метки.


## Комментарии


Для определения высоты текстовой метки используйте свойство [IPPSpeedometerLabel.Height](IPPSpeedometerLabel.Height.htm).
