# IPPSpeedometerLabel.BorderStyle


## Синтаксис


BorderStyle: [GxDashStyle](ModDrawing.chm::/Enums/GxDashStyle.htm);


## Описание


Свойство BorderStyle определяет
 стиль отрисовки границы текстовой метки.


## Комментарии


Свойство BorderStyle актуально,
 если у свойства [IPPSpeedometerLabel.EnableBorder](IPPSpeedometerLabel.EnableBorder.htm)
 установлено значение True.
