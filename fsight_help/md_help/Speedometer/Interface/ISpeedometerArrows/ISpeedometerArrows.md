# ISpeedometerArrows


## Описание


Интерфейс ISpeedometerArrows
 содержит свойства коллекции стрелок шкалы спидометра.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ISpeedometerArrows.Count.htm)
		 Свойство Count определяет
		 количество стрелок шкалы спидометра.


		 ![](../../Property_Image.gif)
		 [Item](ISpeedometerArrows.Item.htm)
		 Свойство Item возвращает
		 параметры указанной стрелки.
