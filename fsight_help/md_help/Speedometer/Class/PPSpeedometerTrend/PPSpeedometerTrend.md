# PPSpeedometerTrend


Сборка: Speedometer;


## Описание


Класс PPSpeedometerTrend реализует
 объект, предназначенный для настройки параметров тренда шкалы данных спидометра.


## Комментарии


Тренд - отмеченное значение на шкале данных спидометра.
