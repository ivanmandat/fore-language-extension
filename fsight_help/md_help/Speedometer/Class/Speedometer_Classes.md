# Классы сборки Speedometer


В «Форсайт. Аналитическая платформа»
 используется два вида спидометров:


	- Визуализатор «[Спидометр](UiReport.chm::/Desktop/Objects/UiReport_Objects_Spedometer.htm)»
	 используется в отчетах и может быть создан как через отчеты, так и
	 при работе с дизайнером форм Fore;


	- Визуализатор «[Индикатор](UiAdhoc.chm::/Blocks/Gadgets/Indicator.htm)»
	 используется в аналитических панелях. Создавать и просматривать его
	 можно только через веб-приложение.


## Классы для работы с визуализатором «[Спидометр](UiReport.chm::/Desktop/Objects/UiReport_Objects_Spedometer.htm)»
 отчета


		 Класс
		 Краткое описание


		 ![](../Class_Image.gif)
		 [SpeedometerBox](SpeedometerBox/SpeedometerBox.htm)
		 Класс SpeedometerBox
		 реализует компонент среды разработки [SpeedometerBox](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/SpeedometerBox.htm),

		 оформленной в виде спидометра.
