# PPSpeedometer


Сборка: Speedometer;


## Описание


Класс PPSpeedometer реализует
 объект, предназначенный для настройки параметров отображения спидометра.


## Комментарии


Спидометр реализует визуализатор «[Индикатор](UiAdhoc.chm::/Blocks/Gadgets/Indicator.htm)»
 аналитической панели. Визуализатор «[Индикатор](UiAdhoc.chm::/Blocks/Gadgets/Indicator.htm)»
 доступен только в веб-приложении.
