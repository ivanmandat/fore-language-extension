# SpeedometerTextPlacementType


## Описание


Перечисление SpeedometerTextPlacementType содержит способы размещение текста относительно шкалы.


Используется следующим свойством:


-
[ISpeedometerScaleView.TextPlacement](../Interface/ISpeedometerScaleView/ISpeedometerScaleView.TextPlacement.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 OutsideRotated. Текст снаружи шкалы, ориентирован по линии шкалы.


 1
 InsideRotated. Текст внутри шкалы, ориентирован по линии шкалы.


 2
 OutsideHorizontal. Текст снаружи шкалы, ориентирован горизонтально.


 3
 InsideHorizontal. Текст внутри шкалы, ориентирован горизонтально.
