# PPSpeedometerVerticalAlignment


## Описание


Перечисление PPSpeedometerVerticalAlignment
 содержит варианты выравнивания по вертикали.


Используется следующим свойством:


	- [IPPTimeLine.VerticalAlignment](../Interface/IPPTimeLine/IPPTimeLine.VerticalAlignment.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Bottom. Выравнивание
		 по нижнему краю.


		 1
		 Center. Выравнивание
		 по центру.


		 2
		 Top. Выравнивание
		 по верхнему краю.
