# PPLabelPlacementStyle


## Описание


Перечисление PPLabelPlacementStyle
 содержит варианты расположения подписи маркера.


Используется следующим свойством:


	- [IPPSpeedometerMarkerBase.LabelPlacementStyle](../Interface/IPPSpeedometerMarkerBase/IPPSpeedometerMarkerBase.LabelPlacementStyle.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Outside. Подпись снаружи
		 маркера.


		 1
		 Inside. Подпись внутри
		 маркера.


		 2
		 Center. Подпись в
		 центре маркера.
