# SpeedometerBarSegmentShape


## Описание


Перечисление SpeedometerBarSegmentShape содержит формы сегмента цветной полосы.


Используется следующим свойством:


-
[ISpeedometerBar.Shape](../Interface/ISpeedometerBar/ISpeedometerBar.Shape.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Rectangle. Прямоугольный сегмент.


 1
 TriangleCenterRigh. Треугольник с вертикально центрированной вершиной, направленный вправо.


 2
 TriangleCenterLeft. Треугольник с вертикально центрированной вершиной, направленный влево.


 3
 TriangleTopRight. Треугольник с вершиной, прижатой вверх, направленный вправо.


 4
 TriangleBottomRight. Треугольник с вершиной, прижатой вниз, направленный вправо.


 5
 TriangleTopLeft. Треугольник с вершиной, прижатой вверх, направленный влево.


 6
 TriangleBottomLeft. Треугольник с вершиной, прижатой вниз, направленный влево.
