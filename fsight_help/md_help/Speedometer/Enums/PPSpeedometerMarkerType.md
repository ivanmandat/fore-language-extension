# PPSpeedometerMarkerType


## Описание


Перечисление PPSpeedometerMarkerType
 содержит варианты формы маркера.


Используется следующим свойствам:


	- [IPPSpeedometerMarkerBase.MarkerType](../Interface/IPPSpeedometerMarkerBase/IPPSpeedometerMarkerBase.MarkerType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Line. Линия.


		 1
		 Circle. Круг.


		 2
		 Triangle. Треугольник.


		 3
		 Rhomb. Ромб.
