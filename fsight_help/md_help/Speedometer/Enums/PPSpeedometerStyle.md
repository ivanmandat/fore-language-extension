# PPSpeedometerStyle


## Описание


Перечисление PPSpeedometerStyle
 содержит базовые стили оформления спидометра.


Используется следующим методом:


	- [IPPSpeedometer.SetStyle](../Interface/IPPSpeedometer/IPPSpeedometer.SetStyle.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Style1. Первый стиль
		 оформления.


		 1
		 Style2. Второй стиль
		 оформления.


		 2
		 Style3. Третий стиль
		 оформления.


		 3
		 Style4. Четвертый
		 стиль оформления.
