# SpeedometerArrowShapes


## Описание


Перечисление SpeedometerArrowShapes содержит типы стрелки спидометра.


Используется следующим свойством:


-
[ISpeedometerArrowView.Shape](../Interface/ISpeedometerArrowView/ISpeedometerArrowView.Shape.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Linear. Прямоугольная стрелка.


 1
 Triangle. Треугольная стрелка.
