# PPSpeedometerHorizontalAlignment


## Описание


Перечисление PPSpeedometerHorizontalAlignment
 содержит варианты выравнивания по горизонтали.


Используется следующим свойством:


	- [IPPTimeLine.HorizontalAlignment](../Interface/IPPTimeLine/IPPTimeLine.HorizontalAlignment.htm);


	- [IPPSpeedometerLabel.TextAlign](../Interface/IPPSpeedometerLabel/IPPSpeedometerLabel.TextAlign.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Right. Выравнивание
		 по правому краю.


		 1
		 Center. Выравнивание
		 по центру.


		 2
		 Left. Выравнивание
		 по левому краю.
