# SpeedometerArrowCircleOrder


## Описание


Перечисление SpeedometerArrowCircleOrder содержит варианты порядка расположения стрелки и круга.


Используется следующим свойством:


-
[ISpeedometerCentralShape.ShapeOrder](../Interface/ISpeedometerCentralShape/ISpeedometerCentralShape.ShapeOrder.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 AboveArrow. Центральный круг расположен поверх стрелок.


 1
 BelowArrow. Центральный круг расположен под стрелками.
