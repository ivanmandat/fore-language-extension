# Введение в сборку Orm


В данном разделе представлено описание сборки Orm,
 содержащей средства, предназначенные для создания условий и получения
 информации об объектах простых классов. К объектам простого класса относятся
 справочники и справочники НСИ.


Основными интерфейсами сборки Orm
 являются:


	- [IOrmCondition](Interface/IOrmCondition/IOrmCondition.htm).
	 Предназначен для работы с условием;


	- [IOrmRecord](Interface/IOrmRecord/IOrmRecord.htm).
	 Предназначен для работой с записью;


	- [IOrmSimpleClass](Interface/IOrmSimpleClass/IOrmSimpleClass.htm).
	 Предназначен для работы с объектом простого класса.
