# IOrmCondition.Operator_


## Синтаксис


Operator_: [OrmComparisonOperator](../../Enums/OrmComparisonOperator.htm);


## Описание


Свойство Operator_ определяет
 оператор сравнения, используемый в условии.


## Комментарии


По умолчанию используется следующий оператор сравнения: Operator_ = OrmComparisonOperator.Equal.


Для определения атрибута и сравниваемого значения используйте свойства
 [IOrmCondition.AttributeName](IOrmCondition.AttributeName.htm)
 и [IOrmCondition.Value](IOrmCondition.Value.htm).
