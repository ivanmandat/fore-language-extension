# IOrmCondition


Сборка: Orm;


## Описание


Интерфейс IOrmCondition предназначен для работы с условием.


## Иерархия наследования


           IOrmCondition


## Комментарии


Условие предназначено для поиска объекта по значениям его атрибутов.


Для работы с коллекцией условий используйте интерфейс [IOrmConditions](../IOrmConditions/IOrmConditions.htm).


## Свойства


 Имя свойства
 Краткое описание


 ![](../../Property_Image.gif)
 [AttributeName](IOrmCondition.AttributeName.htm)
 Свойство AttributeName определяет идентификатор атрибута, для которого создается условие.


 ![](../../Property_Image.gif)
 [ConditionJoin](IOrmCondition.ConditionJoin.htm)
 Свойство ConditionJoin определяет логический оператор, применяемый для объединения текущего условия с предыдущим.


 ![](../../Property_Image.gif)
 [JoinAttributeName](IOrmCondition.JoinAttributeName.htm)
 Свойство JoinAttributeName определяет идентификатор связанного атрибута.


 ![](../../Property_Image.gif)
 [Nested](IOrmCondition.Nested.htm)
 Свойство Nested возвращает коллекцию вложенных условий.


 ![](../../Property_Image.gif)
 [Operator_](IOrmCondition.Operator_.htm)
 Свойство Operator_ определяет оператор сравнения, используемый в условии.


 ![](../../Property_Image.gif)
 [Value](IOrmCondition.Value.htm)
 Свойство Value определяет сравниваемое значение.
