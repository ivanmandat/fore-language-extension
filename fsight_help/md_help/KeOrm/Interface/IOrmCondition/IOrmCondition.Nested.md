# IOrmCondition.Nested


## Синтаксис


Nested: [IOrmConditions](../IOrmConditions/IOrmConditions.htm);


## Описание


Свойство Nested возвращает коллекцию вложенных условий.


## Комментарии


Свойство актуально, если [IOrmCondition.Operator_](IOrmCondition.Operator_.htm) принимает следующие значения:


-
[OrmComparisonOperator.Exists](../../Enums/OrmComparisonOperator.htm);


-
[OrmComparisonOperator.InSelect](../../Enums/OrmComparisonOperator.htm);


-
[OrmComparisonOperator.Conditions](../../Enums/OrmComparisonOperator.htm).
