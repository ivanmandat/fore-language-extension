# IOrmCondition.AttributeName


## Синтаксис


AttributeName: String;


## Описание


Свойство AttributeName определяет идентификатор атрибута, для которого создается условие.


## Комментарии


Для определения значения атрибута используйте свойство [IOrmCondition.Value](IOrmCondition.Value.htm).
