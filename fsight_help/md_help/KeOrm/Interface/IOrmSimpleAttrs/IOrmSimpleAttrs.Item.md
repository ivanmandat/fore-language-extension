# IOrmSimpleAttrs.Item


## Синтаксис


Item(Index: Integer): [IOrmSimpleAttr](../IOrmSimpleAttr/IOrmSimpleAttr.htm);


## Параметры


Index. Индекс атрибута.


## Описание


Свойство Item возвращает параметры указанного атрибута.


## Комментарии


Индексация атрибутов сквозная и начинается с нуля. Для получения количества атрибутов используйте свойство [IOrmSimpleAttrs.Count](IOrmSimpleAttrs.Count.htm).
