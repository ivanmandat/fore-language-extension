# IOrmSimpleAttrs


Сборка: Orm;


## Описание


Интерфейс IOrmSimpleAttrs предназначен для работы с коллекцией атрибутов объекта простого класса.


## Иерархия наследования


           IOrmSimpleAttrs


## Комментарии


Для получения коллекции атрибутов используйте свойство [IOrmSimpleClass.Attrs](../IOrmSimpleClass/IOrmSimpleClass.Attrs.htm).
