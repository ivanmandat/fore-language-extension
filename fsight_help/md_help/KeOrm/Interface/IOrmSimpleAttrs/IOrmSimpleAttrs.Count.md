# IOrmSimpleAttrs.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество атрибутов объекта.


## Комментарии


Для получения определенного атрибута используйте свойство [IOrmSimpleAttrs.Item](IOrmSimpleAttrs.Item.htm).
