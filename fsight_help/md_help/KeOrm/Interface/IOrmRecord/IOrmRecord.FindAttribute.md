# IOrmRecord.FindAttribute


## Синтаксис


FindAttribute(AttributeName: String): Integer;


## Параметры


AttributeName. Идентификатор атрибута.


## Описание


Метод FindAttribute возвращает индекс указанного атрибута.


## Комментарии


Если атрибут с указанным идентификатором отсутствует, то метод возвращает значение «-1».
