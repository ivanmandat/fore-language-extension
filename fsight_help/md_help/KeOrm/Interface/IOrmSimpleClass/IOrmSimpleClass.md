# IOrmSimpleClass


Сборка: Orm;


## Описание


Интерфейс IOrmSimpleClass предназначен для работы с объектом простого класса.


## Иерархия наследования


           IOrmSimpleClass


## Комментарии


Интерфейс содержит свойства для получения информации об объекте простого класса. К объектам простого класса относятся справочники и справочники НСИ. Любой из этих объектов может быть приведен к данному интерфейсу. Информация об объекте может использоваться для поиска в нем данных.
