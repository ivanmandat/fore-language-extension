# IOrmConditions


Сборка: Orm;


## Описание


Интерфейс IOrmConditions предназначен
 для работы с коллекцией условий.


## Иерархия наследования


           IOrmConditions


## Комментарии


Каждое условие предназначено для поиска объекта по значениям его атрибутов.
 Для определения логического оператора, объединяющего условия в коллекции,
 используйте свойство [IOrmCondition.ConditionJoin](../IOrmCondition/IOrmCondition.ConditionJoin.htm).
