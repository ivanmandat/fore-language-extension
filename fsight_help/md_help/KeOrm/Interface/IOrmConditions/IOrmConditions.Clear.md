# IOrmConditions.Clear


## Синтаксис


Clear;


## Описание


Метод Clear очищает коллекцию условий.


## Комментарии


Для удаления одного условия из коллекции используйте метод [IOrmConditions.Remove](IOrmConditions.Remove.htm).
