# IOrmConditions.Item


## Синтаксис


Item(Index: Integer): [IOrmCondition](../IOrmCondition/IOrmCondition.htm);


## Параметры


Index. Индекс условия.


## Описание


Свойство Item возвращает указанное
 условие из коллекции.


## Комментарии


Индексация условий в коллекции сквозная и начинается с нуля. Для получения
 количества условий в коллекции используйте свойство [IOrmConditions.Count](IOrmConditions.Count.htm).
