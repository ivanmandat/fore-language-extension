# IOrmConditions.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемого условия.


## Описание


Метод Remove удаляет указанное условие из коллекции.


## Комментарии


Индексация условий в коллекции сквозная и начинается с нуля. Для получения количества условий в коллекции используйте свойство [IOrmConditions.Count](IOrmConditions.Count.htm).


Если удаление выполнено успешно, то метод возвращает значение True, иначе - False.
