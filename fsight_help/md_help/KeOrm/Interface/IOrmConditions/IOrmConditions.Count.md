# IOrmConditions.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество условий в коллекции.


## Комментарии


Для получения условия из коллекции используйте свойство [IOrmConditions.Item](IOrmConditions.Item.htm).
