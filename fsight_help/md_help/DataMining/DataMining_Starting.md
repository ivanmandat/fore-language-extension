# Начало работы с инструментом «Интеллектуальный анализ данных»


Запуск интеллектуального анализа данных доступен:


	- в веб-приложении из инструмента отчётности продукта «Форсайт. Аналитическая платформа»:
	 «Аналитические панели», «Аналитические запросы (OLAP)»,
	 «Анализ временных рядов»;


	- в настольном приложении:


		- из инструмента отчётности продукта «Форсайт. Аналитическая платформа»:
		 «Аналитические панели»,
		 «Аналитические запросы (OLAP)»,
		 «Отчеты», «Анализ
		 временных рядов»;


		- из [навигатора
		 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm);


		- из объекта репозитория «[Таблица](UiNavObj.chm::/Table/UiDb_relational_table.htm)».


Для запуска интеллектуального анализа данных из
 инструмента отчётности продукта «Форсайт. Аналитическая платформа»:


	- [Выберите данные
	 для анализа](Master/DataMining_Master1_Table.htm).


	- [Выберите
	 тип анализа](Master/Methods/Performing_DataMining.htm).


	- Настройте параметры анализа:


		- [Анализ
		 «Кластеризация»](Master/Methods/DataMining_CategDetect.htm);


		- [Анализ
		 «Поиск исключений»](Master/Methods/DataMining_ExceptionSearch.htm);


		- [Анализ
		 «Заполнение по шаблону»](Master/Methods/DataMining_PatternSubst.htm);


		- [Анализ «Ключевые
		 факторы»](Master/Methods/DataMining_KeyInf.htm);


		- [Анализ
		 «Анализ ассоциаций»](Master/Methods/DataMining_AssosiationAn.htm);


		- [Анализ
		 «Прогнозирование»](Master/Methods/DataMining_Forecasting.htm);


	- Выполните необходимые действия над [результатами
	 анализа](Master/DataMining_Result.htm).


Для запуска интеллектуального анализа данных из
 навигатора объектов и объекта репозитория «Таблица»:


	- Выполните команду главного меню «Сервис
	 > Интеллектуальный анализ данных».


	- Выполните все шаги, аналогичные шагам 2-4 для запуска интеллектуального
	 анализа данных из инструмента отчётности продукта «Форсайт. Аналитическая платформа».
