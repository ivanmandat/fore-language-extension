# Интеллектуальный анализ данных


Для выявления скрытых фактов и взаимосвязей в массивах данных используйте
 инструмент «Интеллектуальный анализ
 данных».


![](Master/DataMining.png)


Инструмент «Интеллектуальный анализ
 данных» для обработки и содержательной интерпретации больших массивов
 данных.


Ключевые возможности:


	- разделение объектов или наблюдений на заданное количество групп
	 на основе близости значений их признаков;


	- определение «степени исключительности» для каждого признака
	 каждого объекта на основании всей совокупности данных;


	- заполнение пропущенных значений одного признака в зависимости
	 от значений других признаков на основе имеющейся классификации;


	- выявление наиболее значимых факторов, выявление степени влияния
	 каждого фактора на зависимую переменную;


	- определение часто встречающихся совместно наборов элементов
	 на основе анализа множества повторяющихся транзакций;


	- продолжение указанного временного ряда выбранным методом прогнозирования
	 с использованием информации о характере его периодичности.


Для начала работы смотрите статью
