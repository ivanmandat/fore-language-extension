# Анализ «Поиск исключений»


Данный вид анализа определяет «степень исключительности» для каждого
 признака каждого объекта на основании всей совокупности данных. На основе
 полученной характеристики можно ранжировать объекты по их исключительности.


Для работы с анализом «Поиск исключений»:


	- [Выберите данные для
	 анализа](../DataMining_Master1_Table.htm).


Примечание.
 Для расчёта метода [выбранные
 данные](../DataMining_Master1_Table.htm) должны содержать минимум один признак и три наблюдения не считая
 заголовков. Если в качестве источника данных выступает:

   - регламентный отчет.
 Заголовками является первая строка в выбранном диапазоне;

   - рабочая книга.
 Заголовками являются имена рядов;

   - экспресс отчет,
 аналитическая панель, таблица
 данных. Заголовками являются заголовки столбцов.


	- [Выберите вид анализа](Performing_DataMining.htm).


	- Настройте параметры анализа:


![](ExceptionSearch.gif)


В окне доступны следующие операции:


[![](../../opened.gif)![](../../closed.gif)Задание
 признаков для поиска исключений](javascript:TextPopup(this))


	Отметьте флажками те столбцы таблицы,
	 данные в которых будут анализироваться на наличие исключений:


			- для отметки всех столбцов нажмите кнопку «Отметить
			 все»;


			- для снятия отметки со всех столбцов нажмите кнопку «Снять отметку»;


	Примечание.
	 Количество входных рядов должно быть больше одного.


[![](../../opened.gif)![](../../closed.gif)Задание
 порогового значения «степени исключительности»](javascript:TextPopup(this))


	Пороговое значение - максимальная доля
	 исключений, содержащихся в данных.


	Задайте максимальный процент исключений,
	 содержащихся в данных.


	Если установлено значение «0%», то пороговое
	 значение отсутствует и будут выявляться все исключения. Если установлено
	 значение «100%» - исключения не будут выявляться. Значение, определенное
	 в промежутке (0, 100), означает, что исключения будут выявляться согласно
	 установленному пороговому значению. По умолчанию установлено значение
	 «50%».


[![](../../opened.gif)![](../../closed.gif)Редактирование
 признака](javascript:TextPopup(this))


	Для редактирования выбранного признака
	 нажмите кнопку «Редактировать».
	 Будет открыто окно «[Редактировать признак](../Edit_Attribute.htm)».


	Примечание.
	 Редактирование признака для поиска исключений доступно только в настольном
	 приложении.


Для перехода к следующей странице мастера
 нажмите кнопку «Далее».


	- Выполните необходимые действия над результатами анализа.


Результаты анализа можно сохранить в файл,
 распечатать или вставить на лист регламентного отчета. Более подробные
 сведения приведены в разделе «[Работа
 с результатами анализа](../DataMining_Result.htm)».

 исключений»:

Для анализа «Поиск
 исключений» на странице «Результат
 анализа» отображаются вкладки:


		- Поиск исключений.
		 На вкладке выделены все найденные в исходных данных исключения;


		- Исключения по признакам.
		 На вкладке отображается общее количество исключений для каждого
		 признака;


		- [Предупреждения](../Warnings.htm).
		 Вкладка отображается, если во время выполнения анализа возникли
		 какие-либо предупреждения.
