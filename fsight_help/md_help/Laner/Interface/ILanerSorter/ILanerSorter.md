# ILanerSorter


Сборка: Laner;


## Описание


Интерфейс ILanerSorter предназначен
 для работы с параметрами сортировки рабочей книги.


## Иерархия наследования


           ILanerSorter


## Комментарии


Параметры сортировки рабочей книги возвращает свойство [ILaner.Sorter](../ILaner/ILaner.Sorter.htm).
