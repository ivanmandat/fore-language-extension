# ILanerSorter.SeriesDataSort


## Синтаксис


SeriesDataSort: [ILanerSorterData](../ILanerSorterData/ILanerSorterData.htm);


## Описание


Свойство SeriesDataSort возвращает параметры сортировки значений наблюдений показателей.


## Комментарии


По умолчанию в рабочей книге сортировка по значениям наблюдений показателей не производится.
