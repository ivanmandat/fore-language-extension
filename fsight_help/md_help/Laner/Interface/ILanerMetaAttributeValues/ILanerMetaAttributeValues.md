# ILanerMetaAttributeValues


## Описание


Интерфейс ILanerMetaAttributeValues
 содержит свойства и методы, предназначенные для работы с коллекцией атрибутов.


## Свойства, унаследованные от [ILanerMetaAttributeValueList](../ILanerMetaAttributeValueList/ILanerMetaAttributeValueList.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](../ILanerMetaAttributeValueList/ILanerMetaAttributeValueList.Count.htm)


		 Свойство Count возвращает
		 количество атрибутов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](../ILanerMetaAttributeValueList/ILanerMetaAttributeValueList.Item.htm)


		 Свойство Item возвращает
		 атрибут из коллекции по его индексу.
