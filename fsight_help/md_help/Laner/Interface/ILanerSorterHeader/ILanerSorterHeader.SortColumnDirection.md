# ILanerSorterHeader.SortColumnDirection


## Синтаксис


SortColumnDirection(Column: [ILanerColumn](../ILanerColumn/ILanerColumn.htm)): [LnSortDirection](../../Enums/LnSortDirection.htm);


## Параметры


Column. Столбец боковика рабочей книги, по значениям которого будет выполняться сортировка.


## Описание


Свойство SortColumnDirection определяет направление сортировки по значениям указанного атрибута ряда.


## Комментарии


Боковик включает в себя все отображаемые столбцы со значениями атрибутов ряда, кроме столбца с наименованием ряда и столбцов со значениями ряда.
