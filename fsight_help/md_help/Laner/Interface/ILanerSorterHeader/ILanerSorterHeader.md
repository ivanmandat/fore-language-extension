# ILanerSorterHeader


Сборка: Laner;


## Описание


Интерфейс ILanerSorterHeader
 предназначен для работы с параметрами сортировки рабочей книги по значениям
 атрибутов показателей.


## Иерархия наследования


           ILanerSorterHeader


## Комментарии


Параметры сортировки значений атрибутов показателей возвращает свойство
 [ILanerSorter.HeadersSort](../ILanerSorter/ILanerSorter.HeadersSort.htm).
