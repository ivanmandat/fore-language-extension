# ILanerSorterHeader.Clear


## Синтаксис


Clear;


## Описание


Метод Clear сбрасывает сортировку
 по значениям атрибута показателей.


## Комментарии


Для выполнения сортировки по значениям заданного атрибута используйте
 свойство [ILanerSorterHeader.SortColumnDirection](ILanerSorterHeader.SortColumnDirection.htm).
