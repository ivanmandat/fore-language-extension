# ILanerSeriesSet


Сборка: Laner;


## Описание


Интерфейс ILanerSeriesSet содержит
 свойства и методы для работы с набором рядов.


## Иерархия наследования


           [ILanerSerieList](../ILanerSerieList/ILanerSerieList.htm)


           ILanerSeriesSet


## Комментарии


Набор рядов позволяет ускорить выполнение операций. Например, можно
 сразу скопировать набор рядов, а не копировать каждый ряд по отдельности.
