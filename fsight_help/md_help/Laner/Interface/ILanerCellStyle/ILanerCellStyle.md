# ILanerCellStyle


## Описание


Интерфейс ILanerCellStyle содержит
 свойства и методы для работы со стилем оформления ячеек таблицы данных
 рабочей книги.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [TabStyle](ILanerCellStyle.TabStyle.htm)
		 Свойство TabStyle возвращает
		 стиль оформления ячейки рабочей книги.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Assign](ILanerCellStyle.Assign.htm)
		 Метод Assign устанавливает
		 стиль ячеек в соответствии с заданным стилем.


		 ![](../../Sub_Image.gif)
		 [SetDefault](ILanerCellStyle.SetDefault.htm)
		 Метод SetDefault устанавливает
		 стиль ячеек, используемый по умолчанию.
