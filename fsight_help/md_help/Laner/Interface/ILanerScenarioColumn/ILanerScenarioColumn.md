# ILanerScenarioColumn


Сборка: Laner;


## Описание


Интерфейс ILanerScenarioColumn содержит свойства для работы с колонкой в боковике рабочей книги, которая отображает наименование сценария, данные которого содержит ряд.


## Иерархия наследования


           [ILanerColumn](../ILanerColumn/ILanerColumn.htm)


           ILanerScenarioColumn


## Комментарии


Данные столбца с наименованием сценария доступны только для чтения.
