# ILanerDatabaseColumn


## Описание


Интерфейс ILanerDatabaseColumn
 предназначен для работы с колонкой в боковике рабочей книги, которая отображает
 информацию о базе данных рядов.


## Комментарии


Столбец с информацией о базе данных по умолчанию не отображается в боковике
 рабочей книги. Для отображения данного столбца используйте метод [ILanerAttributeColumns.AddByType](../ILanerAttributeColumns/ILanerAttributeColumns.AddByType.htm).
