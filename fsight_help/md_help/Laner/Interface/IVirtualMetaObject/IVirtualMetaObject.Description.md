# IVirtualMetaObject.Description


## Синтаксис


Description: String;


## Описание


Свойство Description определяет
 описание объекта.


## Комментарии


Для получения объекта репозитория, внутри которого хранится данный объект,
 используйте свойство [IVirtualMetaObject.MetabaseObject](IVirtualMetaObject.MetabaseObject.htm).
