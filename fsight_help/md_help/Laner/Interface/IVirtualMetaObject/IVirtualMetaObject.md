# IVirtualMetaObject


Сборка: Laner;


## Описание


Интерфейс IVirtualMetaObject
 предназначен для работы с объектами, которые не являются объектами репозитория,
 а являются дочерними объектами отчетов.


## Иерархия наследования


           [INamedEntity](KeSom.chm::/Interface/INamedEntity/INamedEntity.htm)


           IVirtualMetaObject


## Комментарии


Данными объектами могут быть модели, метамодели, задачи моделирования
 и правила валидации.
