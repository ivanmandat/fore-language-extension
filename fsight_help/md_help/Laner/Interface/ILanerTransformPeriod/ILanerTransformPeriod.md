# ILanerTransformPeriod


Сборка: Laner;


## Описание


Интерфейс ILanerTransformPeriod
 предназначен для работы с параметрами периода расчета вычисляемого ряда.


## Комментарии


По умолчанию период расчета вычисляемого ряда совпадет с периодом отображения
 в рабочей книге:


	- начало идентификации = [ILaner.StartDate](../ILaner/ILaner.StartDate.htm);


	- конец прогнозирования = [ILaner.EndDate](../ILaner/ILaner.EndDate.htm).
