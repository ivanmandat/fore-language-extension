# ILanerTransformPeriod.IsUsingForecastStart


## Синтаксис


IsUsingForecastStart: Boolean;


## Описание


Свойство IsUsingForecastStart
 определяет, использовать ли дату начала прогноза.


## Комментарии


Допустимые значения:


	- True. Использовать дату
	 начала прогноза, указанную в свойстве [ILanerTransformPeriod.ForecastStartDate](ILanerTransformPeriod.ForecastStartDate.htm);


	- False. Не использовать
	 дату начала прогноза. Начало прогноза будет определяться по дате окончания
	 идентификации, заданной свойством [ILanerTransformPeriod.IdentificationEndDate](ILanerTransformPeriod.IdentificationEndDate.htm).
