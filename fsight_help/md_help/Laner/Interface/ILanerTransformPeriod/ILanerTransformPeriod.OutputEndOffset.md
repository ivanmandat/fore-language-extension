# ILanerTransformPeriod.OutputEndOffset


## Синтаксис


OutputEndOffset: Integer;


## Описание


Свойство OutputEndOffset
 определяет количество периодов, на которое смещается дата окончания периода
 выгрузки данных.


## Комментарии


Смещение даты окончания периода используется, если период выгрузки данных
 задан с помощью условий, то есть задано значение свойства [ILanerTransformPeriod.OutputAutoPeriod](ILanerTransformPeriod.OutputAutoPeriod.htm).


Для задания количества периодов, на которое смещается дата начала периода
 выгрузки данных, используйте свойство [ILanerTransformPeriod.OutputStartOffset](ILanerTransformPeriod.OutputStartOffset.htm).
