# ILanerTransformPeriod.OutputStartOffset


## Синтаксис


OutputStartOffset: Integer;


## Описание


Свойство OutputStartOffset определяет
 количество периодов, на которое смещается дата начала периода выгрузки
 данных.


## Комментарии


Смещение даты начала периода используется, если период выгрузки данных
 задан с помощью условий, то есть задано значение свойства [ILanerTransformPeriod.OutputAutoPeriod](ILanerTransformPeriod.OutputAutoPeriod.htm).


Для задания количества периодов, на которое смещается дата окончания
 периода выгрузки данных, используйте свойство [ILanerTransformPeriod.OutputEndOffset](ILanerTransformPeriod.OutputEndOffset.htm)
 .
