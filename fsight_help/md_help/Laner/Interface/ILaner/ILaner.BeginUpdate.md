# ILaner.BeginUpdate


## Синтаксис


BeginUpdate;


## Описание


Метод BeginUpdate позволяет
 прекратить перерисовку рабочей книги базы данных временных рядов.


## Комментарии


Все изменения в рабочей книге не будут отображаться до тех пор, пока
 не будет вызван метод [ILaner.EndUpdate](ILaner.EndUpdate.htm).
