# ILaner.RefreshAttributesStructure


## Синтаксис


RefreshAttributesStructure;


## Описание


Метод RefreshAttributesStructure обновляет состав атрибутов рабочей книги в соответствии с родительской базой данных временных рядов.


## Комментарии


Если изменился тип атрибута и его текущее значение невозможно преобразовать к новому типу, то значение атрибута будет очищено.


Признак соответствия состава атрибутов рабочей книги и родительской базы данных временных рядов возвращает метод [ILaner.CheckAttributesCompliance](ILaner.CheckAttributesCompliance.htm).
