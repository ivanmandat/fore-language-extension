# ILaner.CommentAttributes


## Синтаксис


CommentAttributes: [ILanerCellAttributes](../ILanerCellAttributes/ILanerCellAttributes.htm);


## Описание


Свойство CommentAttributes возвращает
 коллекцию атрибутов, используемых для формирования комментариев к наблюдениям
 ряда.


## Комментарии


Для задания атрибутов, используемых для формирования префикса и суффикса
 наблюдений ряда, используйте свойства [ILaner.ValuePrefixAttributes](ILaner.ValuePrefixAttributes.htm)
 и [ILaner.ValueSuffixAttributes](ILaner.ValueSuffixAttributes.htm).
