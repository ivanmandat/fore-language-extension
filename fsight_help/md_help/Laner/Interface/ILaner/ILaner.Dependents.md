# ILaner.Dependents


## Синтаксис


Dependents: [IMetabaseObjectDescriptors](KeSom.chm::/Interface/IMetabaseObjectDescriptors/IMetabaseObjectDescriptors.htm);


## Описание


Свойство Dependents возвращает
 рабочие книги, зависящие от текущей рабочей книги.


## Комментарии


Рабочие книги, от которых зависит текущая книга, возвращает свойство
 [ILaner.Dependencies](ILaner.Dependencies.htm).
