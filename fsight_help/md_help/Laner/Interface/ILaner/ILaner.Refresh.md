# ILaner.Refresh


## Синтаксис


Refres;


## Описание


Метод Refresh обновляет
 значения, заданные параметрами.


## Комментарии


В рабочей книге следующие значения могут быть заданы параметрами:


	- период отображения данных в рабочей книге. Свойства: [ILaner.EndDateParamId](ILaner.EndDateParamId.htm),
	 [ILaner.StartDateParamId](ILaner.StartDateParamId.htm);


	- периоды расчета вычисляемого ряда. Свойства: [ILanerTransformPeriod.EndDateParamId](../ILanerTransformPeriod/ILanerTransformPeriod.EndDateParamId.htm),
	 [ILanerTransformPeriod.IdentificationEndDateParamId](../ILanerTransformPeriod/ILanerTransformPeriod.IdentificationEndDateParamId.htm),
	 [ILanerTransformPeriod.StartDateParamId](../ILanerTransformPeriod/ILanerTransformPeriod.StartDateParamId.htm);


	- значения атрибута рядов. Свойство: [ILanerMetaAttributeValue.ParamId](../ILanerMetaAttributeValue/ILanerMetaAttributeValue.ParamId.htm).
