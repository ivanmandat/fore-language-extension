# ILaner.Dependencies


## Синтаксис


Dependencies: [IMetabaseObjectDescriptors](KeSom.chm::/Interface/IMetabaseObjectDescriptors/IMetabaseObjectDescriptors.htm);


## Описание


Свойство Dependencies возвращает
 рабочие книги, от которых зависит данная рабочая книга.


## Комментарии


Рабочие книги, зависящие от текущей книги, возвращает свойство
 [ILaner.Dependents](ILaner.Dependents.htm).
