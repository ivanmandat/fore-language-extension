# ILaner.EndUpdate


## Синтаксис


EndUpdate;


## Описание


Метод EndUpdate позволяет возобновить
 перерисовку рабочей книги базы данных временных рядов.


## Комментарии


Данный метод отменяет действие метода [ILaner.BeginUpdate](ILaner.BeginUpdate.htm).
