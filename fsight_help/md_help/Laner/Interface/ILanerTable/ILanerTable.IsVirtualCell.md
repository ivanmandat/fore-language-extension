# ILanerTable.IsVirtualCell


## Синтаксис


IsVirtualCell(Row: Integer; Column: Integer): Boolean;


## Параметры


Row. Индекс
 строки;


Column.
 Индекс столбца.


## Описание


Свойство IsVirtualCell возвращает
 признак того, что ячейка не принадлежит динамике ряда.


## Комментарии


Допустимые значения:


	- True. Ячейка не принадлежит
	 динамике ряда;


	- False. Ячейка принадлежит
	 динамике ряда.


Например, ячейка, которая отображает данные за определенный квартал,
 не принадлежит динамике ряда, содержащего только данные по годам.
