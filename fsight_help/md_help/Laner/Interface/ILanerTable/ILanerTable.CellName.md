# ILanerTable.CellName


## Синтаксис


CellRevision(Row: Integer; Column: Integer): String;


## Параметры


Row.
 Индекс строки;


Column.
 Индекс столбца.


## Описание


Свойство CellName возвращает
 наименование ячейки.


## Комментарии


Наименование ячейки совпадает с её содержимым, возвращаемым свойством
 [ILanerTable.Cell](ILanerTable.Cell.htm).
