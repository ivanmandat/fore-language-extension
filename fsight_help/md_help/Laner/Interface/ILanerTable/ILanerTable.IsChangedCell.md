# ILanerTable.IsChangedCell


## Синтаксис


IsChangedCell(Row: Integer; Column: Integer): Boolean;


## Параметры


Row.
 Индекс строки;


Column.
 Индекс столбца.


## Описание


Свойство IsChangedCell возвращает
 признак того, что данные в ячейке были изменены.


## Комментарии


Допустимые значения:


	- True. Данные были изменены;


	- False. Данные не изменялись.
