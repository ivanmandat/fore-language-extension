# ILanerTable.IsEditableCell


## Синтаксис


IsEditableCell(Row: Integer; Column: Integer): Boolean;


## Параметры


Row. Индекс строки;


Column.
 Индекс столбца.


## Описание


Свойство IsEditableCell возвращает
 признак того, что ячейка может быть отредактирована в режиме редактирования
 данных в таблице.


## Комментарии


Допустимые значения:


	- True. Ячейка может быть
	 отредактирована;


	- False. Ячейка доступна
	 только для чтения.
