# ILanerTable.SuffixValues


## Синтаксис


SuffixValues(Row: Integer; Column: Integer): Array;


## Параметры


Row. Индекс строки;


Column. Индекс столбца.


## Описание


Свойство SuffixValues возвращает
 идентификаторы атрибутов, отвечающие за формирование суффиксов для наблюдения
 в указанной ячейке.


## Комментарии


Для получения идентификаторов, отвечающих за формирование префиксов
 и комментариев, используйте свойства [ILanerTable.PrefixValues](ILanerTable.PrefixValues.htm)
 и [ILanerTable.CommentValues](ILanerTable.CommentValues.htm).
