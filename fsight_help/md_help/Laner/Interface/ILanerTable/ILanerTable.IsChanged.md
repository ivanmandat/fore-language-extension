# ILanerTable.IsChanged


## Синтаксис


IsChanged: Boolean;


## Описание


Свойство IsChanged возвращает
 признак того, что данные в таблице были изменены.


## Комментарии


Допустимые значения:


	- True. Данные были изменены;


	- False. Данные не изменялись.
