# ILanerTable.IsEditing


## Синтаксис


IsEditing: Boolean;


## Описание


Свойство IsEditing возвращает
 признак того, что таблица находится в режиме редактирования.


## Комментарии


Допустимые значения:


	- True. Таблица находится
	 в режиме редактирования;


	- False. Таблица доступна
	 только для чтения.
