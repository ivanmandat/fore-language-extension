# ILanerTable.PrefixValues


## Синтаксис


PrefixValues(Row: Integer; Column: Integer): Array;


## Параметры


Row. Индекс строки;


Column. Индекс столбца.


## Описание


Свойство PrefixValues возвращает
 идентификаторы атрибутов, отвечающие за формирование префиксов для наблюдения
 в указанной ячейке.


## Комментарии


Для получения идентификаторов, отвечающих за формирование суффиксов
 и комментариев, используйте свойства [ILanerTable.SuffixValues](ILanerTable.SuffixValues.htm)
 и [ILanerTable.CommentValues](ILanerTable.CommentValues.htm).
