# ILanerTable


## Описание


Интерфейс ILanerTable содержит
 свойства и методы для работы с таблицей данных, полученной в результате
 расчета рабочей области.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Cell](ILanerTable.Cell.htm)
		 Свойство Cell определяет
		 значение в ячейке таблицы.


		 ![](../../Property_Image.gif)
		 [CellName](ILanerTable.CellName.htm)
		 Свойство CellName возвращает
		 наименование ячейки.


		 ![](../../Property_Image.gif)
		 [CellRevision](ILanerTable.CellRevision.htm)
		 Свойство CellRevision
		 возвращает последнюю ревизию, в которой были изменены данные в
		 ячейке.


		 ![](../../Property_Image.gif)
		 [CellRevisionDimension](ILanerTable.CellRevisionDimension.htm)
		 Свойство CellRevisionDimension
		 возвращает справочник, содержащий данные о [ревизиях](UiDw.chm::/Revision/Revision_Work.htm)
		 для указанной ячейки.


		 ![](../../Property_Image.gif)
		 [CellType](ILanerTable.CellType.htm)
		 Свойство CellType возвращает
		 тип указанной ячейки.


		 ![](../../Property_Image.gif)
		 [ColumnCount](ILanerTable.ColumnCount.htm)
		 Свойство ColumnCount
		 возвращает количество столбцов в таблице.


		 ![](../../Property_Image.gif)
		 [CommentValues](ILanerTable.CommentValues.htm)
		 Свойство CommentValues
		 возвращает идентификаторы атрибутов, отвечающие за формирование
		 комментариев для наблюдения в указанной ячейке.


		 ![](../../Property_Image.gif)
		 [DataRights](ILanerTable.DataRights.htm)
		 Свойство DataRights
		 возвращает права доступа на данные указанной ячейки.


		 ![](../../Property_Image.gif)
		 [FilterEndYear](ILanerTable.FilterEndYear.htm)
		 Свойство FilterEndYear
		 возвращает последнюю отображаемую дату.


		 ![](../../Property_Image.gif)
		 [FilterStartYear](ILanerTable.FilterStartYear.htm)
		 Свойство FilterStartYear
		 возвращает первую отображаемую дату.


		 ![](../../Property_Image.gif)
		 [FootnoteValues](ILanerTable.FootnoteValues.htm)
		 Свойство FootnoteValues
		 возвращает массив, содержащий значения сносок, сформированных
		 для ячейки с указанными координатами.


		 ![](../../Property_Image.gif)
		 [IsChanged](ILanerTable.IsChanged.htm)
		 Свойство IsChanged
		 возвращает признак того, что данные в таблице были изменены.


		 ![](../../Property_Image.gif)
		 [IsChangedCell](ILanerTable.IsChangedCell.htm)
		 Свойство IsChangedCell
		 возвращает признак того, что данные в ячейке были изменены.


		 ![](../../Property_Image.gif)
		 [IsEditableCell](ILanerTable.IsEditableCell.htm)
		 Свойство IsEditableCell
		 возвращает признак того, что ячейка может быть отредактирована.


		 ![](../../Property_Image.gif)
		 [IsEditing](ILanerTable.IsEditing.htm)
		 Свойство IsEditing
		 возвращает признак того, что таблица находится в режиме редактирования.


		 ![](../../Property_Image.gif)
		 [IsEvaluatedCell](ILanerTable.IsEvaluatedCell.htm)
		 Свойство IsEvaluatedCell
		 возвращает признак того, что ячейка является вычисляемой.


		 ![](../../Property_Image.gif)
		 [IsVirtualCell](ILanerTable.IsVirtualCell.htm)
		 Свойство IsVirtualCell
		 возвращает признак того, что ячейка не принадлежит динамике ряда.


		 ![](../../Property_Image.gif)
		 [LeftHeader](ILanerTable.LeftHeader.htm)
		 Свойство LeftHeader
		 возвращает боковик заголовка таблицы.


		 ![](../../Property_Image.gif)
		 [PrefixValues](ILanerTable.PrefixValues.htm)

		 возвращает идентификаторы атрибутов, отвечающие за формирование
		 префиксов для наблюдения в указанной ячейке.


		 ![](../../Property_Image.gif)
		 [RowCount](ILanerTable.RowCount.htm)
		 Свойство RowCount возвращает
		 количество строк в таблице.


		 ![](../../Property_Image.gif)
		 [SeriesList](ILanerTable.SeriesList.htm)
		 Свойство SeriesList
		 возвращает коллекцию рядов рабочей книги.


		 ![](../../Property_Image.gif)
		 [TopHeader](ILanerTable.TopHeader.htm)
		 Свойство TopHeader
		 возвращает шапку заголовка таблицы.


		 ![](../../Property_Image.gif)
		 [SuffixValues](ILanerTable.SuffixValues.htm)
		 Свойство SuffixValues
		 возвращает идентификаторы атрибутов, отвечающие за формирование
		 суффиксов для наблюдения в указанной ячейке.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Edit](ILanerTable.Edit.htm)
		 Метод Edit осуществляет
		 перевод таблицы данных в режим редактирования.


		 ![](../../Sub_Image.gif)
		 [Post](ILanerTable.Post.htm)
		 Метод Post осуществляет
		 выход из режима редактирования и сохранение данных.


		 ![](../../Sub_Image.gif)
		 [ReleaseCellValue](ILanerTable.ReleaseCellValue.htm)
		 Метод ReleaseCellValue
		 копирует вычисленное значение указанной ячейки в исходный
		 ряд.
