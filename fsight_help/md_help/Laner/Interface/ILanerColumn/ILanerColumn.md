# ILanerColumn


Сборка: Laner;


## Описание


Интерфейс ILanerColumn содержит
 свойства и методы для работы со столбцом боковика рабочей книги.


## Иерархия наследования


           ILanerColumn


## Комментарии


Боковик включает в себя все отображаемые столбцы со значениями атрибутов
 ряда, кроме столбцов со значениями ряда:
