# ILanerColumn.ReadOnly


## Синтаксис


ReadOnly: Boolean;


## Описание


Свойство ReadOnly определяет, доступны ли значения в столбце только для чтения.


## Комментарии


Допустимые значения:


-
True. Значения в столбце доступны только для чтения;


-
False. Значения в столбце доступны для изменения.
