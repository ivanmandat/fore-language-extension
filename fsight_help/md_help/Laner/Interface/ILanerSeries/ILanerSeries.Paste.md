# ILanerSeries.Paste


## Синтаксис


Paste(StartIndex: Integer;[ Parent: [ILanerGroupSerie](../ILanerGroupSerie/ILanerGroupSerie.htm)
 = Null])


## Параметры


StartIndex. Позиция, в которую
 будет вставлен набор рядов;


Parent. Родительская группа
 рядов. Необязательный параметр.


## Описание


Метод Paste вставляет набор
 рядов из буфера обмена.
