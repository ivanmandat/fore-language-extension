# ILanerSeries.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс ряда в коллекции.


## Описание


Метод Remove осуществляет удаление
 ряда из коллекции по его индексу.


## Комментарии


Метод возвращает True в случае
 удачного удаления.
