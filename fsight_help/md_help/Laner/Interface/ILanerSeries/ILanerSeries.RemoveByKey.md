# ILanerSeries.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ ряда в коллекции.


## Описание


Метод RemoveByKey осуществляет
 удаление ряда из коллекции по его ключу.


## Комментарии


Метод возвращает True в случае
 удачного удаления.
