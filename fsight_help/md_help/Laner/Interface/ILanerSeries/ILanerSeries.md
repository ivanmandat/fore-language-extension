# ILanerSeries


Сборка: Laner;


## Описание


Интерфейс ILanerSeries содержит
 свойства и методы для работы с коллекцией рядов рабочей области.


## Иерархия наследования


           [ILanerSerieList](../ILanerSerieList/ILanerSerieList.htm)


           ILanerSeries


## Комментарии


Для работы с отдельным рядом используйте интерфейс [ILanerSerie](../ILanerSerie/ILanerSerie.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [HaveUnsavedData](ILanerSeries.HaveUnsavedData.htm)
		 Свойство HaveUnsavedData
		 возвращает признак того, что в рабочей книге присутствуют ряды,
		 данные которых не сохранены.
