# ILanerAttributeColumn


## Описание


Интерфейс ILanerAttributeColumn
 содержит свойства для работы с отображаемым атрибутом ряда.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [MetaAttribute](ILanerAttributeColumn.MetaAttribute.htm)
		 Свойство MetaAttribute
		 возвращает отображаемый атрибут.


		 ![](../../Property_Image.gif)
		 [MetaAttributeEx](ILanerAttributeColumn.MetaAttributeEx.htm)
		 Свойство MetaAttributeEx
		 возвращает отображаемый атрибут для указанного ряда.


		 ![](../../Property_Image.gif)
		 [NamingFormat](ILanerAttributeColumn.NamingFormat.htm)
		 Свойство NamingFormat
		 определяет формат наименования отображаемого атрибута.
