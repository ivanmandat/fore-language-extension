# ILanerAttributes.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ атрибута в коллекции.


## Описание


Метод RemoveByKey осуществляет
 удаление атрибута из коллекции по ключу.


## Комментарии


Метод возвращает True в случае
 удачного удаления.
