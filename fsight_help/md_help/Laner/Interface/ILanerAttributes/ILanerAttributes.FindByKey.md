# ILanerAttributes.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IMetaAttribute](KeRds.chm::/Interface/IMetaAttribute/IMetaAttribute.htm);


## Параметры


Key. Ключ атрибута в коллекции.


## Описание


Метод FindByKey осуществляет
 поиск атрибута в коллекции по ключу и возвращает объект, содержащий атрибут
 в случае удачного поиска.
