# ILanerAttributes


## Описание


Интерфейс ILanerAttributes содержит
 свойства и методы для работы с коллекцией атрибутов ряда.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ILanerAttributes.Count.htm)
		 Свойство Count возвращает
		 количество атрибутов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ILanerAttributes.Item.htm)
		 Свойство Item возвращает
		 атрибут из коллекции по его индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ILanerAttributes.Add.htm)
		 Метод Add осуществляет
		 добавление атрибута в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](ILanerAttributes.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех атрибутов из коллекции.


		 ![](../../Sub_Image.gif)
		 [FindById](ILanerAttributes.FindById.htm)
		 Метод FindById осуществляет
		 поиск атрибута в коллекции по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ILanerAttributes.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск атрибута в коллекции по ключу.


		 ![](../../Sub_Image.gif)
		 [FindIndexByKey](ILanerAttributes.FindIndexByKey.htm)
		 Метод FindIndexByKey
		 осуществляет поиск индекса атрибута в коллекции по ключу.


		 ![](../../Sub_Image.gif)
		 [MoveTo](ILanerAttributes.MoveTo.htm)
		 Метод MoveTo осуществляет
		 перемещение атрибута в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](ILanerAttributes.Remove.htm)
		 Метод Remove осуществляет
		 удаление атрибута из коллекции по его индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](ILanerAttributes.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление атрибута из коллекции по ключу.
