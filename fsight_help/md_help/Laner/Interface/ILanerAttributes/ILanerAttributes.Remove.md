# ILanerAttributes.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс атрибута в коллекции.


## Описание


Метод Remove осуществляет удаление
 атрибута из коллекции по его индексу.


## Комментарии


Метод возвращает True в случае
 удачного удаления.
