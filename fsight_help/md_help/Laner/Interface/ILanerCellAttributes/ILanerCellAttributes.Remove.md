# ILanerCellAttributes.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс атрибута в коллекции.


## Описание


Метод Remove удаляет атрибут
 из коллекции по его индексу.


## Комментарии


Индексация атрибутов в коллекции и начинается с нуля. Для получения
 количества атрибутов в коллекции используйте свойство [ILanerCellAttributes.Count](ILanerCellAttributes.Count.htm).


Допустимые значения:


	- True. Удаление выполнено
	 успешно;


	- False. Удаление выполнено
	 с ошибкой.
