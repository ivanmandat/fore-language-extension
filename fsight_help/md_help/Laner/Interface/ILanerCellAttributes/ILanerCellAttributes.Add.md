# ILanerCellAttributes.Add


## Синтаксис


Add(AttributeId: String; AttributeType: [LnAttributeType](../../Enums/LnAttributeType.htm)):
 Integer;


## Параметры


AttributeId. Идентификатор
 атрибута, добавляемого в коллекцию;


AttributeType. Тип атрибута
 добавляемого в коллекцию.


## Описание


Метод Add добавляет указанный
 атрибут в коллекцию.


## Комментарии


В результате выполнения метод возвращает индекс добавленного атрибута
 в коллекции.
