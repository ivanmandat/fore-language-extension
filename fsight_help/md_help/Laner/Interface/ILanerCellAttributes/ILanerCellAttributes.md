# ILanerCellAttributes


Сборка: Laner;


## Описание


Интерфейс ILanerCellAttributes
 используется для работы с коллекцией атрибутов, используемых для формирования
 суффикса, префикса и комментария для наблюдений ряда данных.


## Иерархия наследования


           ILanerCellAttributes


## Комментарии


Для получения коллекции атрибутов используйте свойства:


	- [ILaner.CommentAttributes](../ILaner/ILaner.CommentAttributes.htm).
	 Используется для настройки комментариев;


	- [ILaner.ValuePrefixAttributes](../ILaner/ILaner.ValuePrefixAttributes.htm).
	 Используется для настройки префиксов;


	- [ILaner.ValueSuffixAttributes](../ILaner/ILaner.ValueSuffixAttributes.htm).
	 Используется для настройки суффиксов.
