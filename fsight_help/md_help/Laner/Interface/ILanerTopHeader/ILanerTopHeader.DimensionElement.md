# ILanerTopHeader.DimensionElement


## Синтаксис


DimensionElement(SlotIndex: Integer; ElementIndex:
 Integer): Integer;


## Параметры


SlotIndex. Индекс слота в заголовке;


ElementIndex. Индекс элемента
 в слоте.


## Описание


Свойство DimensionElement возвращает
 индекс календарного элемента в шапке заголовка.
