# ILanerTopHeader


## Описание


Интерфейс ILanerTopHeader содержит
 свойства, предназначенные для работы с шапкой заголовка.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DimensionDate](ILanerTopHeader.DimensionDate.htm)
		 Свойство DimensionDate
		 возвращает календарную точку, соответствующую указанному элементу
		 в шапке заголовка.


		 ![](../../Property_Image.gif)
		 [Dimension](ILanerTopHeader.DimensionElement.htm)[Element](ILanerTopHeader.DimensionLevel.htm)
		 Свойство DimensionElement
		 возвращает индекс календарного элемента в шапке заголовка.


		 ![](../../Property_Image.gif)
		 [DimensionLevel](ILanerTopHeader.DimensionLevel.htm)
		 Свойство DimensionLevel
		 возвращает уровень календаря, соответствующий указанному элементу
		 в шапке заголовка.


## Свойства, унаследованные от [ILanerHeader](../ILanerHeader/ILanerHeader.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ElementCount](../ILanerHeader/ILanerHeader.ElementCount.htm)


		 Свойство ElementCount
		 возвращает количество элементов в заголовке.


		 ![](../../Property_Image.gif)
		 [ElementName](../ILanerHeader/ILanerHeader.ElementName.htm)


		 Свойство ElementName
		 возвращает наименование элемента заголовка.


		 ![](../../Property_Image.gif)
		 [ElementValue](../ILanerHeader/ILanerHeader.ElementValue.htm)


		 Свойство ElementValue
		 определяет значение элемента заголовка.


		 ![](../../Property_Image.gif)
		 [ReadOnly](../ILanerHeader/ILanerHeader.ReadOnly.htm)


		 Свойство ReadOnly возвращает
		 признак того, возможно ли редактирование элемента заголовка.


		 ![](../../Property_Image.gif)
		 [SlotCount](../ILanerHeader/ILanerHeader.SlotCount.htm)


		 Свойство SlotCount
		 возвращает количество слотов, на которые разбит заголовок.


		 ![](../../Property_Image.gif)
		 [SlotName](../ILanerHeader/ILanerHeader.SlotName.htm)


		 Свойство SlotName возвращает
		 наименование слота заголовка.
