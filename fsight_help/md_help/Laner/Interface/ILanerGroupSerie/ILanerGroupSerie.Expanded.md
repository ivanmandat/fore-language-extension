# ILanerGroupSerie.Expanded


## Синтаксис


Expanded: Boolean;


## Описание


Свойство Expanded определяет
 состояние группы рядов.


## Комментарии


Если свойству установлено значение True,
 то группа рядов находится в развернутом состоянии, иначе группа свернута.
