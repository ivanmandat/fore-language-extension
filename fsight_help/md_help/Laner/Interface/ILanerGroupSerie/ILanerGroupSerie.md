# ILanerGroupSerie


## Описание


Интерфейс ILanerGroupSerie содержит
 свойства, предназначенные для работы с рядом, представляющим собой группу
 рядов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CanAddChildren](ILanerGroupSerie.CanAddChildren.htm)


		 Свойство CanAddChildren
		 возвращает признак того, возможно ли добавление дочернего ряда
		 в группу рядов.


		 ![](../../Property_Image.gif)
		 [Children](ILanerGroupSerie.Children.htm)


		 Свойство Children возвращает
		 ряды, образующие группу.


		 ![](../../Property_Image.gif)
		 [Expanded](ILanerGroupSerie.Expanded.htm)


		 Свойство Expanded определяет
		 состояние группы рядов.
