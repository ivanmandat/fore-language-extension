# ILanerFilter


Сборка: Laner;


## Описание


Интерфейс ILanerFilter предназначен
 для настройки фильтрации данных в рабочей книге.


## Иерархия наследования


           ILanerFilter


## Комментарии


По умолчанию используется фильтрация данных только справа и слева от
 ряда. Для смены режима используйте [SuppressEmptyArea](ILanerFilter.SuppressEmptyArea.htm).
