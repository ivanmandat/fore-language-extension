# ILanerFilter.SuppressEmptyArea


## Синтаксис


SuppressEmptyArea: [LnSuppressEmptyArea](../../Enums/LnSuppressEmptyArea.htm);


## Описание


Свойство SuppressEmptyArea определяет
 область фильтрации пустых значений.


## Комментарии


SuppressEmptyArea учитывается,
 если свойству [ILanerFilter.SuppressEmpty](ILanerFilter.SuppressEmpty.htm)
 установлено значение True.
