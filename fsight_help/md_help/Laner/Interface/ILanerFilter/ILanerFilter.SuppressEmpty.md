# ILanerFilter.SuppressEmpty


## Синтаксис


SuppressEmpty: Boolean;


## Описание


Свойство SuppressEmpty определяет,
 фильтровать ли пустые значения в ячейках.


## Комментарии


Допустимые значения:


	- True. Производится фильтрация
	 пустых ячеек;


	- False. Фильтрация пустых
	 ячеек не производится.


Режим фильтрации определяется свойством [ILanerFilter.SuppressEmptyArea](ILanerFilter.SuppressEmptyArea.htm).
