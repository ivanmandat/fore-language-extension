# ILanerCalculateSerie.UseTransform


## Синтаксис


UseTransform: Boolean;


## Описание


Свойство UseTransform определяет,
 использовать ли параметры вычисления ряда.


## Комментарии


Допустимые значения:


	- True. Параметры вычисления
	 используются. Для их настройки используйте свойство [ILanerCalculateSerie.Transform](ILanerCalculateSerie.Transform.htm).
	 Если параметры вычисления не заданы, то корректный расчет ряда
	 невозможен;


	- False. Параметры вычисления
	 не используются. Свойство [ILanerCalculateSerie.Transform](ILanerCalculateSerie.Transform.htm)
	 возвращает значение Null.
