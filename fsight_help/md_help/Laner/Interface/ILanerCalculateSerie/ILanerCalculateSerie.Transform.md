# ILanerCalculateSerie.Transform


## Синтаксис


Transform: [IFormulaTransformModel](KeCubes.chm::/Interface/IFormulaTransformModel/IFormulaTransformModel.htm);


## Описание


Свойство Transform возвращает
 объект, позволяющий настроить параметры вычисляемого ряда.


## Комментарии


Использование свойства доступно, если свойству [ILanerCalculateSerie.UseTransform](ILanerCalculateSerie.UseTransform.htm)
 установлено значение True.
