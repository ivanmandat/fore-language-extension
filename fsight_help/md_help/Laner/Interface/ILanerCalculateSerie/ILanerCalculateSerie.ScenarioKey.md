# ILanerCalculateSerie.ScenarioKey


## Синтаксис


ScenarioKey: Integer;


## Описание


Свойство ScenarioKey определяет
 ключ сценария, данные которого содержит ряд.


## Комментарии


Если ScenarioKey возвращает
 значение «-1», то ряд содержит фактические данные (сценарий «Факт»).
