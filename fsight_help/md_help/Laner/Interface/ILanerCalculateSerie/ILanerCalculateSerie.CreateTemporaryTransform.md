# ILanerCalculateSerie.CreateTemporaryTransform


## Синтаксис


CreateTemporaryTransform: [IFormulaTransformModel](KeCubes.chm::/Interface/IFormulaTransformModel/IFormulaTransformModel.htm);


## Описание


Метод CreateTemporaryTransform создает временный объект, позволяющий настроить параметры расчета вычисляемого ряда.


## Комментарии


Значения, вычисленные по данным настройкам, не влияют на параметры ряда. Значения будут записаны в ячейки рабочей книги с тем же флагом, который устанавливается при ручном вводе.
