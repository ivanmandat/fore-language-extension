# ILanerContext.EndLoad


## Синтаксис

## Описание


Метод EndLoad завершает
 загрузку рабочих книг в контекст.


## Комментарии


Для начала загрузки используйте метод [ILanerContext.BeginLoad](ILanerContext.BeginLoad.htm).
