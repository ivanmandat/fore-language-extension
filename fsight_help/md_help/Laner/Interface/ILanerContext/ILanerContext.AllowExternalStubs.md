# ILanerContext.AllowExternalStubs


## Синтаксис


AllowExternalStubs: Boolean;


## Описание


Свойство AllowExternalStubs
 определяет, разрешено ли использовать в расчетах ряды из различных рабочих
 книг.


## Комментарии


Допустимые значения:


	- True. Использование
	 рядов из различных рабочих книг разрешено;


	- False. Использование
	 рядов из различных рабочих книг запрещено.
