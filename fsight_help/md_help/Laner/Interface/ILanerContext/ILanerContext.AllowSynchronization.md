# ILanerContext.AllowSynchronization


## Синтаксис


AllowSynchronization: Boolean;


## Описание


Свойство AllowSynchronization
 определяет, разрешена ли синхронизация одинаковых рядов.


## Комментарии


Допустимые значения:


	- True. Синхронизация
	 одинаковых рядов разрешена;


	- False. Синхронизация
	 рядов запрещена.
