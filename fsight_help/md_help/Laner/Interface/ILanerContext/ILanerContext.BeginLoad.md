# ILanerContext.BeginLoad


## Синтаксис


BeginLoad;


## Описание


Метод BeginLoad начинает
 загрузку рабочих книг в контекст.


## Комментарии


Для завершения загрузки используйте метод [ILanerContext.EndLoad](ILanerContext.EndLoad.htm).
