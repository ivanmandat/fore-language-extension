# ILanerContext.DataLoadEndDate


## Синтаксис


DataLoadEndDate: DateTime;


## Описание


Свойство DataLoadEndDate определяет
 дату окончания загрузки данных для всех рабочих книг, содержащихся в контексте.


## Комментарии


Дату начала загрузки данных определяет свойство [ILanerContext.DataLoadStartDate](ILanerContext.DataLoadStartDate.htm).
