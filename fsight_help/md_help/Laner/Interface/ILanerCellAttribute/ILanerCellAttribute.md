# ILanerCellAttribute


Сборка: Laner;


## Описание


Интерфейс ILanerCellAttribute
 используется для работы с атрибутом, применяемым для формирования суффикса,
 префикса и комментария для наблюдений ряда данных.


## Иерархия наследования


           ILanerCellAttribute


## Комментарии


Для работы с коллекцией атрибутов используйте интерфейс [ILanerCellAttributes](../ILanerCellAttributes/ILanerCellAttributes.htm).
