# ILanerSerie.Hint


## Синтаксис


Hint: String;


## Описание


Свойство Hint определяет текст всплывающей подсказки для ряда данных.


## Комментарии


Признак наличия подсказки для ряда указывается в виде красного уголка в ячейке первой колонки боковика. Всплывающая подсказка отображается при наведении курсора мыши на эту ячейку.
