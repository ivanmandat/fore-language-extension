# ILanerAttributeColumns


## Описание


Интерфейс ILanerAttributeColumns
 содержит свойства и методы для работы с коллекцией отображаемых атрибутов
 ряда.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Column](ILanerAttributeColumns.Column.htm)
		 Свойство Column возвращает
		 колонку боковика рабочей книги, отображающую указанный атрибут.


		 ![](../../Property_Image.gif)
		 [Count](ILanerAttributeColumns.Count.htm)
		 Свойство Count возвращает
		 количество атрибутов в коллекции.


		 ![](../../Property_Image.gif)
		 [ShowFullNamesInHints](ILanerAttributeColumns.ShowFullNamesInHints.htm)
		 Свойство ShowFullNamesInHints
		 определяет, показывать ли иерархию значений атрибута.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddByType](ILanerAttributeColumns.AddByType.htm)
		 Метод AddByType добавляет
		 атрибут в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](ILanerAttributeColumns.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех атрибутов из коллекции.


		 ![](../../Sub_Image.gif)
		 [Move](ILanerAttributeColumns.Move.htm)
		 Метод Move осуществляет
		 перемещение атрибута в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](ILanerAttributeColumns.Remove.htm)
		 Метод Remove осуществляет
		 удаление атрибута из коллекции по его индексу.
