# ILanerSerieDisplayParams


## Описание


Интерфейс ILanerSerieDisplayParams
 содержит свойства для настройки параметров отображения данных ряда.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Inversion](ILanerSerieDisplayParams.Inversion.htm)
		 Свойство Inversion
		 определяет способ начального преобразования данных.


		 ![](../../Property_Image.gif)
		 [InversionLag](ILanerSerieDisplayParams.InversionLag.htm)
		 Свойство InversionLag
		 определяет период, на котором осуществляется начальное преобразование.


		 ![](../../Property_Image.gif)
		 [PreviousInversionLag](ILanerSerieDisplayParams.PreviousInversionLag.htm)
		 Свойство PreviousInversionLag
		 определяет лаг для периода, используемого при начальном преобразовании
		 данных среза.


		 ![](../../Property_Image.gif)
		 [SetDefault](ILanerSerieDisplayParams.SetDefault.htm)
		 Метод SetDefault сбрасывает
		 все дополнительные параметры отображения данных ряда.
