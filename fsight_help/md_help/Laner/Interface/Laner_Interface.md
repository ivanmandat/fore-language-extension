# Интерфейсы сборки Laner


		 Интерфейс
		 Краткое описание


		 ![](../Interface_Image.gif)
		 [ILaner](ILaner/ILaner.htm)
		 Интерфейс ILaner содержит
		 свойства и методы рабочей книги базы данных временных рядов в
		 рядном режиме.


		 ![](../Interface_Image.gif)
		 [ILanerAttributeColumn](ILanerAttributeColumn/ILanerAttributeColumn.htm)
		 Интерфейс ILanerAttributeColumn
		 содержит свойства для работы с отображаемым атрибутом ряда.


		 ![](../Interface_Image.gif)
		 [ILanerAttributeColumns](ILanerAttributeColumns/ILanerAttributeColumns.htm)
		 Интерфейс ILanerAttributeColumns
		 содержит свойства и методы для работы с коллекцией отображаемых
		 атрибутов ряда.


		 ![](../Interface_Image.gif)
		 [ILanerAttributes](ILanerAttributes/ILanerAttributes.htm)
		 Интерфейс ILanerAttributes
		 содержит свойства и методы для работы с коллекцией атрибутов ряда.


		 ![](../Interface_Image.gif)
		 [ILanerCalculateSerie](ILanerCalculateSerie/ILanerCalculateSerie.htm)
		 Интерфейс ILanerCalculateSerie
		 содержит свойства и методы, предназначенные для работы с вычисляемым
		 рядом.


		 ![](../Interface_Image.gif)
		 [ILanerCellAttribute](ILanerCellAttribute/ILanerCellAttribute.htm)
		 Интерфейс ILanerCellAttribute
		 используется для работы с атрибутом, применяемым для формирования
		 суффикса, префикса и комментария для наблюдений ряда данных.


		 ![](../Interface_Image.gif)
		 [ILanerCellAttributes](ILanerCellAttributes/ILanerCellAttributes.htm)
		 Интерфейс ILanerCellAttributes
		 используется для работы с коллекцией атрибутов, используемых для
		 формирования суффикса, префикса и комментария для наблюдений ряда
		 данных.


		 ![](../Interface_Image.gif)
		 [ILanerCellStyle](ILanerCellStyle/ILanerCellStyle.htm)
		 Интерфейс ILanerCellStyle
		 содержит свойства и методы для работы со стилем ячеек таблицы
		 данных.


		 ![](../Interface_Image.gif)
		 [ILanerInversionColumn](ILanerInversionColumn/ILanerInversionColumn.htm)
		 Интерфейс ILanerInversionColumn
		 содержит свойства для работы с колонкой в боковике рабочей книги,
		 которая отображает начальное преобразование ряда.


		 ![](../Interface_Image.gif)
		 [ILanerColumn](ILanerColumn/ILanerColumn.htm)
		 Интерфейс ILanerColumn
		 содержит свойства и методы для работы со столбцом боковика рабочей
		 книги.


		 ![](../Interface_Image.gif)
		 [ILanerContext](ILanerContext/ILanerContext.htm)
		 Интерфейс ILanerContext
		 содержит свойства и методы для работы с контекстом, в рамках которого
		 осуществляется синхронизация данных между рабочими книгами базы
		 данных временных рядов.


		 ![](../Interface_Image.gif)
		 [ILanerDatabaseColumn](ILanerDatabaseColumn/ILanerDatabaseColumn.htm)
		 Интерфейс ILanerDatabaseColumn
		 предназначен для работы с колонкой в боковике рабочей книги, которая
		 отображает информацию о базе данных рядов.


		 ![](../Interface_Image.gif)
		 [ILanerDerivedSerie](ILanerDerivedSerie/ILanerDerivedSerie.htm)
		 Интерфейс ILanerDerivedSerie
		 содержит свойства и методы для работы с дочерними рядами вычисляемого
		 ряда.


		 ![](../Interface_Image.gif)
		 [ILanerFilter](ILanerFilter/ILanerFilter.htm)
		 Интерфейс ILanerFilter
		 предназначен для настройки фильтрации данных в рабочей книге.


		 ![](../Interface_Image.gif)
		 [ILanerGroupSerie](ILanerGroupSerie/ILanerGroupSerie.htm)
		 Интерфейс ILanerGroupSerie
		 содержит свойства, предназначенные для работы с рядом, представляющим
		 собой группу рядов.


		 ![](../Interface_Image.gif)
		 [ILanerHeader](ILanerHeader/ILanerHeader.htm)
		 Интерфейс ILanerHeader
		 содержит свойства, предназначенные для работы с заголовком рабочей
		 области.


		 ![](../Interface_Image.gif)
		 [ILanerLeftHeader](ILanerLeftHeader/ILanerLeftHeader.htm)
		 Интерфейс ILanerLeftHeader
		 содержит свойства, предназначенные для работы с боковиком заголовка.


		 ![](../Interface_Image.gif)
		 [ILanerMetaAttributeValue](ILanerMetaAttributeValue/ILanerMetaAttributeValue.htm)
		 Интерфейс ILanerMetaAttributeValue
		 содержит свойства, предназначенные для работы с атрибутом.


		 ![](../Interface_Image.gif)
		 [ILanerMetaAttributeValueList](ILanerMetaAttributeValueList/ILanerMetaAttributeValueList.htm)
		 Интерфейс ILanerMetaAttributeValueList
		 содержит свойства и методы для работы с коллекцией значений атрибутов
		 без возможности добавления и удаления.


		 ![](../Interface_Image.gif)
		 [ILanerMetaAttributeValues](ILanerMetaAttributeValues/ILanerMetaAttributeValues.htm)
		 Интерфейс ILanerMetaAttributeValues
		 содержит свойства и методы, предназначенные для работы с коллекцией
		 атрибутов.


		 ![](../Interface_Image.gif)
		 [ILanerNameColumn](ILanerNameColumn/ILanerNameColumn.htm)
		 Интерфейс ILanerNameColumn
		 используется для работы со столбцом боковика рабочей книги,
		 содержащим наименование ряда.


		 ![](../Interface_Image.gif)
		 [ILanerScenarioColumn](ILanerScenarioColumn/ILanerScenarioColumn.htm)
		 Интерфейс ILanerScenarioColumn
		 содержит свойства для работы с колонкой в боковике рабочей книги,
		 которая отображает наименование сценария, данные которого содержит
		 ряд.


		 ![](../Interface_Image.gif)
		 [ILanerSerie](ILanerSerie/ILanerSerie.htm)
		 Интерфейс ILanerSerie
		 содержит свойства для работы с рядом рабочей области.


		 ![](../Interface_Image.gif)
		 [ILanerSerieDisplayParams](ILanerSerieDisplayParams/ILanerSerieDisplayParams.htm)
		 Интерфейс ILanerSerieDisplayParams
		 содержит свойства для настройки параметров отображения данных
		 ряда.


		 ![](../Interface_Image.gif)
		 [ILanerSerieList](ILanerSerieList/ILanerSerieList.htm)
		 Интерфейс ILanerSerieList
		 содержит свойства и методы для работы с коллекцией рядов рабочей
		 области без возможности добавления и удаления.


		 ![](../Interface_Image.gif)
		 [ILanerSeries](ILanerSeries/ILanerSeries.htm)
