# ILanerSorterData


Сборка: Laner;


## Описание


Интерфейс ILanerSorterData предназначен
 для работы с параметрами сортировки значений наблюдений показателей.


## Иерархия наследования


           ILanerSorterData


## Комментарии


Параметры сортировки значений наблюдений показателей возвращает свойство
 [ILanerSorter.SeriesDataSort](../ILanerSorter/ILanerSorter.SeriesDataSort.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CalendarElement](ILanerSorterData.CalendarElement.htm)
		 Свойство CalendarElement
		 возвращает идентификатор элемента календаря, соответствующий
		 полю сортировки.


		 ![](../../Property_Image.gif)
		 [Count](ILanerSorterData.Count.htm)
		 Свойство Count возвращает
		 число полей, участвующих в сортировке.


		 ![](../../Property_Image.gif)
		 [SortDirection](ILanerSorterData.SortDirection.htm)
		 Свойство SortDirection
		 определяет направление сортировки по значениям наблюдений
		 показателей.
