# ILanerSorterData.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает число полей, участвующих в сортировке.


## Комментарии


При сортировке по значениям наблюдений каждое поле является элементом календарного измерения базы данных временных рядов.
