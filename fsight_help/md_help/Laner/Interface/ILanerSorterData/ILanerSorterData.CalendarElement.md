# ILanerSorterData.CalendarElement


## Синтаксис


CalendarElement(Index: Integer): Integer;


## Параметры


Index. Индекс поля, по которому
 выполняется сортировка.


## Описание


Свойство CalendarElement возвращает
 идентификатор элемента календаря, соответствующий полю сортировки.


## Комментарии


Индексация полей сквозная и начинается с нуля. Число полей, по которым
 выполняется сортировка, возвращает свойство [ILanerSorterData.Count](ILanerSorterData.Count.htm).
