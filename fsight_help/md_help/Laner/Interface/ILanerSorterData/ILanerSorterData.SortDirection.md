# ILanerSorterData.SortDirection


## Синтаксис


SortDirection(Element: Integer): [LnSortDirection](../../Enums/LnSortDirection.htm);


## Параметры


Element. Идентификатор элемента
 календаря.


## Описание


Свойство SortDirection определяет
 направление сортировки по значениям наблюдений показателей.


## Комментарии


Для получения идентификатора элемента календаря, по которому уже выполняется
 сортировка, используйте свойство [ILanerSorterData.CalendarElement](ILanerSorterData.CalendarElement.htm).
