# ILanerMetaAttributeValueList


## Описание


Интерфейс ILanerMetaAttributeValueList
 содержит свойства и методы для работы с коллекцией значений атрибутов
 без возможности добавления и удаления.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ILanerMetaAttributeValueList.Count.htm)


		 Свойство Count возвращает
		 количество атрибутов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ILanerMetaAttributeValueList.Item.htm)


		 Свойство Item возвращает
		 атрибут из коллекции по его индексу.


## Методы


		 Имя метода


		 Краткое описание


		 ![](../../Sub_Image.gif)


		 [FindById](ILanerMetaAttributeValueList.FindById.htm)


		 Метод FindById
		 осуществляет поиск атрибута в коллекции по идентификатору.


		 ![](../../Sub_Image.gif)


		 [FindByKey](ILanerMetaAttributeValueList.FindByKey.htm)


		 Метод FindByKey
		 осуществляет поиск атрибута в коллекции по ключу.
