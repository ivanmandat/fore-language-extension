# ILanerSerieList


## Описание


Интерфейс ILanerSerieList содержит
 свойства и методы для работы с коллекцией рядов рабочей области без возможности
 добавления и удаления.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ILanerSerieList.Count.htm)


		 Свойство Count возвращает
		 количество рядов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ILanerSerieList.Item.htm)


		 Свойство Item возвращает
		 ряд из коллекции по его индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](ILanerSerieList.FindById.htm)


		 Метод FindById осуществляет
		 поиск ряда в коллекции по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ILanerSerieList.FindByKey.htm)


		 Метод FindByKey осуществляет
		 поиск ряда в коллекции по ключу.
