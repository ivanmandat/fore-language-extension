# IValidationFilterSettings


Сборка: Laner;


## Описание


Интерфейс IValidationFilterSettings
 содержит свойства для работы с настройками фильтра валидации.


## Иерархия наследования


           IValidationFilterSettings


## Комментарии


Для получения настроек фильтра валидации используйте свойство [IValidationFilterModel.Settings](../IValidationFilterModel/IValidationFilterModel.Settings.htm).
