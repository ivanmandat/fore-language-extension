# ILanerLeftHeader


## Описание


Интерфейс ILanerLeftHeader содержит
 свойства, предназначенные для работы с боковиком заголовка.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ElementFullName](ILanerLeftHeader.ElementFullName.htm)
		 Свойство ElementFullName
		 возвращает полное наименование указанного элемента боковика заголовка.


		 ![](../../Property_Image.gif)
		 [FootnoteValues](ILanerLeftHeader.FootnoteValues.htm)
		 Свойство FootnoteValues
		 возвращает массив, содержащий значения сносок, сформированных
		 для указанного элемента заголовка.


		 ![](../../Property_Image.gif)
		 [Serie](ILanerLeftHeader.Serie.htm)
		 Свойство Serie возвращает
		 ряд, соответствующий элементу заголовка.


## Свойства, унаследованные от [ILanerHeader](../ILanerHeader/ILanerHeader.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ElementCount](../ILanerHeader/ILanerHeader.ElementCount.htm)


		 Свойство ElementCount
		 возвращает количество элементов в заголовке.


		 ![](../../Property_Image.gif)
		 [ElementName](../ILanerHeader/ILanerHeader.ElementName.htm)


		 Свойство ElementName
		 возвращает наименование элемента заголовка.


		 ![](../../Property_Image.gif)
		 [ElementValue](../ILanerHeader/ILanerHeader.ElementValue.htm)


		 Свойство ElementValue
		 определяет значение элемента заголовка.


		 ![](../../Property_Image.gif)
		 [ReadOnly](../ILanerHeader/ILanerHeader.ReadOnly.htm)


		 Свойство ReadOnly возвращает
		 признак того, возможно ли редактирование элемента заголовка.


		 ![](../../Property_Image.gif)
		 [SlotCount](../ILanerHeader/ILanerHeader.SlotCount.htm)


		 Свойство SlotCount
		 возвращает количество слотов, на которые разбит заголовок.


		 ![](../../Property_Image.gif)
		 [SlotName](../ILanerHeader/ILanerHeader.SlotName.htm)


		 Свойство SlotName возвращает
		 наименование слота заголовка.
