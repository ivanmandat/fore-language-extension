# ILanerHeader


## Описание


Интерфейс ILanerHeader содержит
 свойства, предназначенные для работы с заголовком рабочей области.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ElementCount](ILanerHeader.ElementCount.htm)


		 Свойство ElementCount
		 возвращает количество элементов в заголовке.


		 ![](../../Property_Image.gif)
		 [ElementName](ILanerHeader.ElementName.htm)


		 Свойство ElementName
		 возвращает наименование элемента заголовка.


		 ![](../../Property_Image.gif)
		 [ElementValue](ILanerHeader.ElementValue.htm)


		 Свойство ElementValue
		 определяет значение элемента заголовка.


		 ![](../../Property_Image.gif)
		 [ReadOnly](ILanerHeader.ReadOnly.htm)


		 Свойство ReadOnly возвращает
		 признак того, возможно ли редактирование элемента заголовка.


		 ![](../../Property_Image.gif)
		 [SlotCount](ILanerHeader.SlotCount.htm)


		 Свойство SlotCount
		 возвращает количество слотов, на которые разбит заголовок.


		 ![](../../Property_Image.gif)
		 [SlotName](ILanerHeader.SlotName.htm)


		 Свойство SlotName возвращает
		 наименование слота заголовка.
