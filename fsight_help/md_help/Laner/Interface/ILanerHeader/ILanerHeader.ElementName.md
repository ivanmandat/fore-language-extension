# ILanerHeader.ElementName


## Синтаксис


ElementName(SlotIndex: Integer; ElementIndex: Integer):
 String;


## Параметры


SlotIndex. Индекс слота в заголовке;


ElementIndex. Индекс элементов
 в заголовке.


## Описание


Свойство ElementName возвращает
 наименование элемента заголовка.
