# LanerFactory


## Описание


Класс LanerFactory содержит
 свойства и методы рабочей области базы данных временных рядов.


## Конструкторы


		 Имя конструктора
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Create](LanerFactory.Create.htm)
		 Конструктор Create
		 создает объект, представляющий собой рабочую область базы данных
		 временных рядов.

## Методы, унаследованные от [ILaner](../../Interface/ILaner/ILaner.htm)


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [BeginUpdate](../../Interface/ILaner/ILaner.BeginUpdate.htm)
		 Метод BeginUpdate позволяет
		 прекратить перерисовку рабочей книги.


		 ![](../../Sub_Image.gif)
		 [CheckAttributesCompliance](../../Interface/ILaner/ILaner.CheckAttributesCompliance.htm)
		 Метод CheckAttributesCompliance
		 проверяет соответствие состава атрибутов рабочей книги и родительской
		 базы данных временных рядов.


		 ![](../../Sub_Image.gif)
		 [EndUpdate](../../Interface/ILaner/ILaner.EndUpdate.htm)
		 Метод EndUpdate позволяет
		 возобновить перерисовку рабочей книги.


		 ![](../../Sub_Image.gif)
		 [Execute](../../Interface/ILaner/ILaner.Execute.htm)
		 Метод Execute осуществляет
		 расчет рабочей области и возвращает таблицу, содержащую ее данные.


		 ![](../../Sub_Image.gif)
		 [FlushCache](../../Interface/ILaner/ILaner.FlushCache.htm)
		 Метод FlushCache сброса
		 кеша измерений рабочей книги базы данных временных рядов.


		 ![](../../Sub_Image.gif)
		 [Refresh](../../Interface/ILaner/ILaner.Refresh.htm)
		 Метод Refresh
		 обновляет значения, заданные параметрами.


		 ![](../../Sub_Image.gif)
		 [RefreshAttributesStructure](../../Interface/ILaner/ILaner.RefreshAttributesStructure.htm)
		 Метод RefreshAttributesStructure
		 обновляет состав атрибутов рабочей книги в соответствии с родительской
		 базой данных временных рядов.


		 ![](../../Sub_Image.gif)
		 [SaveChangedSeries](../../Interface/ILaner/ILaner.SaveChangedSeries.htm)
		 Метод SaveChangedSeries
		 осуществляет сохранение данных рядов рабочей области.


		 ![](../../Sub_Image.gif)
		 [SaveChangesToModels](../../Interface/ILaner/ILaner.SaveChangesToModels.htm)
		 Метод SaveChangesToModels
		 сохраняет изменения в данных показателей, полученных из модели
		 контейнера моделирования, в исходную модель.


		 ![](../../Sub_Image.gif)
		 [SaveProblem](../../Interface/ILaner/ILaner.SaveProblem.htm)
		 Метод SaveProblem осуществляет
		 сохранение данных рабочей области в виде задачи моделирования.


		 ![](../../Sub_Image.gif)
		 [Update](../../Interface/ILaner/ILaner.Update.htm)
		 Метод Update осуществляет
		 выход из режима редактирования без сохранения данных.
