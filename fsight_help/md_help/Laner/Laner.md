# Введение в сборку Laner


Основными интерфейсом сборки Laner
 является:


	- [ILaner](Interface/ILaner/ILaner.htm).
	 Содержит свойства и методы рабочей области базы данных временных рядов.


Интерфейсы необходимые для определения исходных данных, расчета и просмотра
 результатов:


	- [ILanerTable](Interface/ILanerTable/ILanerTable.htm).
	 Содержит свойства и методы для работы с таблицей данных, полученной
	 в результате расчета рабочей области;


	- [ILanerCalculateSerie](Interface/ILanerCalculateSerie/ILanerCalculateSerie.htm). Содержит
	 свойства и методы, предназначенные для работы с вычислимым рядом;


	- [ILanerDerivedSerie](Interface/ILanerDerivedSerie/ILanerDerivedSerie.htm). Содержит
	 свойства и методы для работы с дочерними рядами вычисляемого ряда;


	- [ILanerGroupSerie](Interface/ILanerGroupSerie/ILanerGroupSerie.htm).
	 Содержит свойства, предназначенные для работы с рядом, представляющим
	 собой группу рядов;


	- [ILanerMetaAttributeValue](Interface/ILanerMetaAttributeValue/ILanerMetaAttributeValue.htm).
	 Содержит свойства, предназначенные для работы с атрибутом;


	- [IValidationFilterSettings](Interface/IValidationFilterSettings/IValidationFilterSettings.htm).
	 Содержит свойства для работы с настройками фильтра валидации.
