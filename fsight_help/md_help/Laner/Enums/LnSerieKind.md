# LnSerieKind


## Описание


Перечисление LnSerieKind используется
 для определения типа ряда.


Используется следующим свойством:


	- [ILanerSerie.Kind](../Interface/ILanerSerie/ILanerSerie.Kind.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Source. Ряд, основанный
		 на показателе.


		 1
		 Calculate. Вычисляемый
		 ряд.


		 2
		 Group. Группа рядов.


		 3
		 CalculateGroup. Группа
		 вычисляемых рядов.


		 4
		 Derived. Дочерний ряд
		 вычисляемого ряда.


		 5
		 UserGroup. Пользовательская
		 группа рядов.


## Комментарии


Элемент LnSerieKind.Source
 не используется, оставлен для совместимости.
