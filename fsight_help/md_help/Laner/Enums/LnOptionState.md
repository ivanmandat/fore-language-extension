# LnOptionState


## Описание


Перечисление LnOptionState используется
 для определения того, отображать ли дочерние ряды у вычисляемых рядов.


Используется следующим свойством:


	- [ILaner.CreateDerivedSeries](../Interface/ILaner/ILaner.CreateDerivedSeries.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 Default_. Отображать
		 или скрывать ряд определяется его настройками.


		 0
		 OffOption. Скрывать
		 в рабочей книге все дочерние ряды у вычисляемых рядов.


		 -1
		 OnOption. Отображать
		 в рабочей книге все дочерние ряды у вычисляемых рядов.
