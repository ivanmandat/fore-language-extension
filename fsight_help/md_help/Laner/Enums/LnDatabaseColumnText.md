# LnDatabaseColumnText


## Описание


Перечисление LnDatabaseColumnText содержит варианты текста, который может отображаться в колонке боковика рабочей книги, которая отображает информацию о базе данных рядов.


Данное перечисление используется следующим свойством:


-
[ILanerDatabaseColumn.Text](../Interface/ILanerDatabaseColumn/ILanerDatabaseColumn.Text.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Name. Отображается наименование базы данных.


 1
 Path. Отображается путь к базе данных.
