# LnSerieFactorSearchResult


## Описание


Перечисление LnSerieFactorSearchResult содержит признаки наличия в базе данных временных рядов ряда с такими же значениями обязательных атрибутов, как и у текущего. Данное перечисление используется следующим свойством:


-
[ILanerSerie.SearchFactor](../Interface/ILanerSerie/ILanerSerie.SearchFactor.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 NotFound. Ряд с идентичными значениями обязательных атрибутов не существует.


 1
 Found. Ряд с идентичными значениями обязательных атрибутов существует.


 2
 Incomplete. Определены не все значения обязательных атрибутов ряда.
