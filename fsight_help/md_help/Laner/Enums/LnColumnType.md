# LnColumnType


## Описание


Перечисление LnColumnType содержит тип столбца в боковике рабочей книги.


Используется следующими свойствами и методами:


-
[ILanerAttributeColumns.AddByType](../Interface/ILanerAttributeColumns/ILanerAttributeColumns.AddByType.htm);


-
[ILanerColumn.Type](../Interface/ILanerColumn/ILanerColumn.Type.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Attribute. Столбец атрибута.


 1
 Inversion. Столбец преобразований.


 2
 Database. Столбец базы данных.


 3
 Scenario. Столбец со сценарием.


 4
 Name. Столбец с наименованием ряда.
