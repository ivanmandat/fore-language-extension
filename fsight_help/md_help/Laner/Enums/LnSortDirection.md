# LnSortDirection


## Описание


Перечисление LnSortDirection предназначено для определения направления сортировки.


Используется следующими свойствами и методами:


-
[ILanerSorterData.SortDirection](../Interface/ILanerSorterData/ILanerSorterData.SortDirection.htm);


-
[ILanerSorterHeader.SortColumnDirection](../Interface/ILanerSorterHeader/ILanerSorterHeader.SortColumnDirection.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Сортировка не производится.


 1
 Ascending. Сортировка по возрастанию значений.


 2
 Descending. Сортировка по убыванию значений.
