# LnDataDisplayType


## Описание


Перечисление LnDataDisplayType содержит варианты отображаемых значений ряда.


Данное перечисление используется следующими свойствами и методами:


-
[ILaner.DisplayDataAs](../Interface/ILaner/ILaner.DisplayDataAs.htm);


-
[ILanerCalculateSerie.DisplayDataAs](../Interface/ILanerCalculateSerie/ILanerCalculateSerie.DisplayDataAs.htm).


## Допустимые значения


 Значение
 Краткое описание


 1
 CalculatedValues. Отображать рассчитанные значения.


 2
 SourceValues. Отображать значения, хранимые в БД.
