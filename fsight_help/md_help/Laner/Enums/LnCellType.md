# LnCellType


## Описание


Перечисление LnCellType содержит тип ячеек в таблице данных рабочей книги.


Используется следующими свойствами и методами:


-
[ILaner.CellStyle](../Interface/ILaner/ILaner.CellStyle.htm);


-
[ILanerTable.CellType](../Interface/ILanerTable/ILanerTable.CellType.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Normal. Обычная ячейка.


 1
 Splice. Ячейка, являющаяся источником данных для результирующего совмещенного ряда.


 2
 DiffersFromSource. Ячейка, содержащая значение, которое отличается от значения ряда в БД.


 4
 Forecast. Ячейка, содержащая прогнозные значения.


## Комментарии


Перечисление можно использовать для настройки оформления различных типов ячеек.
