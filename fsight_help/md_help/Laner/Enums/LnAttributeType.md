# LnAttributeType


## Описание


Перечисление LnAttributeType
 используется для определения типа атрибута.


Используется следующими свойствами и методами:


	- [ILanerCellAttribute.Type](../Interface/ILanerCellAttribute/ILanerCellAttribute.Type.htm);


	- [ILanerCellAttributes.Add](../Interface/ILanerCellAttributes/ILanerCellAttributes.Add.htm).
