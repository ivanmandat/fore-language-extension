# ValidationSeverity


## Описание


Перечисление ValidationSeverity
 используется для определения важности фильтра валидации.


Используется следующим свойством:


	- [IValidationFilterSettings.Severity](../Interface/IValidationFilterSettings/IValidationFilterSettings.Severity.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Information. Информация.


		 1
		 Low. Низкая важность.


		 2
		 High. Высокая важность.


		 3
		 Critical. Критичный
		 фильтр.
