		 Перечисление LnDatabaseColumnText

		 боковика рабочей книги, которая отображает информацию о базе данных
		 рядов.


		 ![](../Enums_Image.gif)
		 [LnDataDisplayType](LnDataDisplayType.htm)

		 ![](../Enums_Image.gif)
		 [LnDerivedSerieType](LnDerivedSerieType.htm)
		 Перечисление LnDerivedSerieType
		 содержит типы дочернего ряда у вычисляемого ряда.


		 ![](../Enums_Image.gif)
		 [LnOptionState](LnOptionState.htm)
		 Перечисление LnOptionState
		 используется для определения того, отображать ли дочерние ряды
		 у вычисляемых рядов.


		 ![](../Enums_Image.gif)
		 [LnSaveOptions](LnSaveOptions.htm)
		 Перечисление LnSaveOptions
		 содержит варианты изменений, которые требуется сохранять.


		 ![](../Enums_Image.gif)
		 [LnSerieFactorSearchResult](LnSerieFactorSearchResult.htm)
		 Перечисление LnSerieFactorSearchResult
		 содержит признаки наличия в базе данных временных рядов ряда с
		 такими же значениями обязательных атрибутов, как и у текущего.


		 ![](../Enums_Image.gif)
		 [LnSerieKind](LnSerieKind.htm)

		 ![](../Enums_Image.gif)
		 [LnSortDirection](LnSortDirection.htm)

		 ![](../Enums_Image.gif)
