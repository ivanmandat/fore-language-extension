## С чего начать


		 В данном разделе
		 представлено описание сборки Laner.
		 Сборка содержит средства, предназначенные для работы с [рабочей областью
		 базы данных временных рядов](UiDw.chm::/Workbook/UiDw_Workbook.htm).


		Изучив данный раздел, вы научитесь:


			- работать с рядами в рабочей области (добавлять, перемещать,
			 удалять ряды);


			- настраивать параметры ряда (задавать значения атрибутов,
			 работать с данными ряда, настраивать оформления);


			- работать с вычисляемыми рядами, которые создаются на
			 базе уже имеющихся рядов.


		Более подробно с возможностями сборки можно ознакомиться в разделе
		 «[Введение в сборку Laner](Laner.htm)».


		Описание ресурсов сборки представлено в следующих подразделах:


			- [Интерфейсы сборки
			 Laner](Interface/Laner_Interface.htm)


			- [Перечисления сборки
			 Laner](Enums/Laner_Enums.htm)


			- [Классы сборки Laner](Class/Laner_Class.htm)


			- [Примеры](Samples/Laner_Samples.htm)


		 Перед началом работы рекомендуется
		 ознакомиться с разделами справки: «[Разработка
		 прикладного приложения](UiDevEnv.chm::/01_Development_Environment/01_Purpose_of_the_constructor/Purpose_of_the_constructor.htm)», «[Руководство по языку Fore](Fore.chm::/Fore_Title.htm)».
