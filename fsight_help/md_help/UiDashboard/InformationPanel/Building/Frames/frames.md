# Добавление
 слайдов и работа с ними


Слайд - объект для группировки
 визуализаторов и интерактивного взаимодействия с ними. Группировка применяется
 в следующих случаях:


	- для удобства экспорта проекта;


	- для наглядной иллюстрации конкретного среза данных;


	- для подготовки визуализаторов к последующему глубокому анализу
	 данных внутри группы.


## Операции со слайдами


Для всех слайдов доступны следующие операции:


[![](../../../opened.gif)![](../../../closed.gif)Добавление
 слайда](javascript:TextPopup(this))


	Информационная панель может содержать несколько вкладок, на каждой
	 из которых можно отобразить данные из различных источников, используя
	 разные визуализаторы.


	Для добавления нового слайда нажмите кнопку ![](../../../Picture/b_add.png) «Добавить слайд» на панели слайдов.
	 Будет добавлена вкладка нового слайда с наименованием по умолчанию
	 «Слайд N», где N - порядковый
	 номер добавляемого слайда.


	Примечание.
	 Добавление слайда доступно только при работе с информационной панелью
	 в [режиме редактирования](../../Starting.htm#modes).


[![](../../../opened.gif)![](../../../closed.gif)Переход
 по слайдам](javascript:TextPopup(this))


	Для перехода по слайдам:


		- выделите вкладку слайда на [панели
		 слайдов](../../Starting.htm#structure_window);


		- выберите слайд в раскрывающемся списке ![](../../../Picture/b_frames.png)
		 «Все слайды» на [панели
		 слайдов](../../Starting.htm#structure_window).


	Будет открыт выбранный слайд.


[![](../../../opened.gif)![](../../../closed.gif)Переименование
 слайда](javascript:TextPopup(this))


	Для задания заголовка выбранного слайда:


		- Выполните одно из действий:


			- нажмите кнопку ![](../../../Picture/b_actions.png)
			 «Действие» на вкладке
			 соответствующего слайда на [панели
			 слайдов](../../Starting.htm#structure_window) и в раскрывшемся меню выберите ![](../../../Picture/b_rename_frames.png)
			 «Переименовать»;


			- установите флажок «Отображать»
			 в группе «Заголовок»
			 на [панели
			 настроек](../Visualizers/Visualization/visualization_setting.htm#settings_panel). По умолчанию флажок снят, заголовок не отображается.

			Для отображения панели настроек переведите кнопку ![](../../../Picture/b_settings_off.png) «Настройки»
			 на [панели инструментов](../../Starting.htm#structure_window)
			 в активное состояние.


		- Задайте [настройки
		 заголовка](../Visualizers/Visualization/visualization_setting.htm#title).


	В результате у слайда будет отображаться заголовок. Заголовок слайда,
	 отображаемый в области заголовка, совпадает с заголовком, заданным
	 на панели настроек и на панели слайдов.


	Для изменения заголовка слайда выполните шаги по настройке заголовка
	 слайда, описанные выше, или выполните последовательность действий:


		- Щёлкните по области заголовка слайда.


		- Удалите старый заголовок и введите новый.


		- Нажмите клавишу ENTER или щёлкните вне области заголовка
		 слайда.


	В результате заголовок слайда будет изменён.


	Для получения подробной информации о настройке заголовка слайда
	 обратитесь к разделу «[Настройка
	 заголовка](../Visualizers/Visualization/visualization_setting.htm#title)».


[![](../../../opened.gif)![](../../../closed.gif)Настройка
 синхронизации визуализаторов](javascript:TextPopup(this))


	Подробное описание синхронизации визуализаторов смотрите в статье
	 «[Синхронизация
	 визуализаторов на слайде](visualizers_synchronization.htm)».


[![](../../../opened.gif)![](../../../closed.gif)Скрытие
 слайда](javascript:TextPopup(this))


	Для скрытия слайда:


		- Нажмите кнопку ![](../../../Picture/b_actions.png)
		 «Действие» на вкладке
		 соответствующего слайда.


		- В раскрывшемся меню выберите ![](../../../Picture/b_unhide.png) «Скрыть».


	Вкладка слайда будет скрыта.


	Для отображения скрытого слайда выберите его в раскрывающемся списке
	 ![](../../../Picture/b_frames.png) «Все слайды» на
	 [панели слайдов](../../Starting.htm#structure_window).


	Примечание.
	 При работе с информационной панелью в [режиме
	 просмотра](../../Starting.htm#modes) скрытые слайды не отображаются в списке всех слайдов.


[![](../../../opened.gif)![](../../../closed.gif)Смена порядка
 слайдов](javascript:TextPopup(this))


	Для смены порядка слайдов на [панели
	 слайдов](../../Starting.htm#structure_window):


		- Наведите курсор мыши на вкладку слайда. Указатель мыши примет
		 вид руки.


		- Зажмите левую кнопку мыши и переместите вкладку слайда в
		 требуемую позицию на панели слайдов.


[![](../../../opened.gif)![](../../../closed.gif)Удаление
 слайда](javascript:TextPopup(this))


	Для удаления слайда:


		- Нажмите кнопку ![](../../../Picture/b_actions.png)
		 «Действие» на вкладке
		 соответствующего слайда.


		- В раскрывшемся меню выберите ![](../../../Picture/b_delete.png) «Удалить».


	После подтверждения действия выделенный слайд будет удалён вместе
	 со всеми размещёнными на нём визуализаторами.


[![](../../../opened.gif)![](../../../closed.gif)Экспорт
 слайда](javascript:TextPopup(this))


	Для получения подробной информации обратитесь к разделу «[Экспорт визуализаторов
	 и слайдов](../../Work/export.htm)».


[![](../../../opened.gif)![](../../../closed.gif)Печать слайда](javascript:TextPopup(this))


	Для получения подробной информации обратитесь к разделу «[Печать визуализаторов и слайдов](../../Work/print.htm)».
