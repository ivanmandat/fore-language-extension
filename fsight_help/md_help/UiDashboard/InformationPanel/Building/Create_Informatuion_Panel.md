# Построение информационной панели


Для построения информационной панели выполните шаги:


	- Добавьте [источники данных](Source.htm).


	- Добавьте и настройте [визуализаторы](Visualizers/visualizers.htm).


	- Добавьте и настройте [слайды](Frames/frames.htm).


	- Настройте рабочую область.


	- Перейдите к [работе
	 с готовой информационной панелью](../Work/Work_with_information_bar.htm).


Для сохранения информационной панели:


	- нажмите кнопку «Сохранить»
	 [главного меню](../../DataModel/create_data_model.htm);


	- нажмите сочетание клавиш CTRL+S.


Сохранение новой
 информационной панели аналогично [созданию
 её копии](Save.htm).


При нажатии кнопки ![](../../Picture/b_back.png) «Назад»
 главного меню для сохранения измененных данных информационной панели подтвердите
 свои действия в информационном окне.


Примечание.
 Сохранение информационной панели доступно только пользователям, обладающим
 [правами](Admin.chm::/02_ComonFunction/Admin_SecManager_CommonFunction.htm)
 на запись объекта. Для пользователей, обладающими правами только на чтение
 объекта, доступно создание [копии информационной панели](Save.htm).
