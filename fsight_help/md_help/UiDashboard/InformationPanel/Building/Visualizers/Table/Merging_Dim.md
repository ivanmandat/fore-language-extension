# Объединение измерений и поджатие элементов измерений


Если
 в боковик/шапку выведено несколько измерений, то для более компактного
 отображения боковика/шапки таблицы используйте настройки объединения и
 поджатия.


Для настройки объединения и поджатия измерений в боковике/шапке используйте
 область «[Измерения](Layout.htm#dimension)»
 в группе параметров «Размещение»
 боковика/шапки.


[![](../../../../opened.gif)![](../../../../closed.gif)Для открытия
 группы параметров «Размещение»](javascript:TextPopup(this))


	Для открытия на панели параметров группы параметров «Размещение»:


		- Выделите [боковик](Select_Areas.htm#select_sidehead)
		 или [шапку](Select_Areas.htm#select_heading) таблицы.


		- Переведите кнопку ![](../../../../Picture/b_settings_off.png) «Настройки» на [панели
		 инструментов](../../../Starting.htm#structure_window) в активное состояние. По умолчанию панель настроек
		 скрыта.


		- Выберите требуемый тип элемента таблицы в раскрывающемся
		 меню заголовка панели настроек.


## Объединение


Для объединения измерения с предыдущим измерением выполните команду
 «Объединить с предыдущим» в его
 контекстном меню измерения в области «Измерения».
 В результате выполнения действий выбранное измерение будет объединено
 с предыдущим, а команда будет помечена флажком.


Для разделения измерений снимите флажок с команды «Объединить
 с предыдущим».


После объединения измерения с предыдущим станет доступно его [поджатие
 с другими измерениями](#tucking).


## Поджатие


Для объединения элементов одного измерения с элементами другого используйте
 поджатие измерений. При использовании поджатия элемент измерения, который
 поджимают, перестанет отображаться в отдельной строке, а его значения
 будут отображены в строке элемента измерения, к которому поджимают.


Примечание.
 Поджатие доступно только для [объединённых измерений](#merging)
 боковика/шапки таблицы.


Для поджатия измерений в боковике/шапке:


	- Выполните команду «Поджимать
	 элементы» в контекстном меню измерения. В контекстном меню
	 для измерения станет доступна команда «Настроить»,
	 а команда «Поджимать элементы»
	 будет помечена флажком.


	- Выполните команду «Настроить»
	 в контекстном меню измерения. В результате появится дополнительная
	 панель для выбора измерений.


	- Отметьте измерения для поджатия, на появившейся дополнительной
	 панели.


Для каждого элемента измерения, для которого
 настраивается поджатие, проверяется наличие поджимаемых элементов нижележащего
 измерения:


		- если поджимаемый элемент один, то происходит поджатие;


		- если поджимаемых элементов несколько, то поджимается только
		 один, выделенный первым.


Примечание.
 Выбор нескольких элементов необходим для ситуаций, когда в результате
 применения фильтрации общих измерений или фильтрации с помощью показателя,
 будут оставаться различные элементы поджимаемого измерения под каждым
 элементом верхнего измерения.


В результате выполнения действий для выбранного измерения будет произведено
 поджатие.


Наименование выбранного измерения после поджатия отображается в исходном
 виде и состоит из наименования исходного элемента. Для ячейки таблицы
 будет применяться [оформление](TableElemants_Format.htm), заданное
 для измерения, которое находилось в этой ячейке до поджатия.
