# Область построения


Область построения - это часть
 области диаграммы, на которой располагаются ряды данных.


![](../../../../Picture/settings_panel/construction_area_example.png)


Для настройки параметров области построения диаграммы используйте группу
 параметров «Область построения»
 [панели настроек](visualization_setting.htm#settings_panel).


Примечание.
 Настройка доступна для всех типов диаграмм, кроме [кольцевой](visualizer_types.htm#doughnut),
 [круговой](visualizer_types.htm#pie) и [лепестковой](visualizer_types.htm#radar).


[![](../../../../opened.gif)![](../../../../closed.gif)Для открытия
 группы параметров «Область построения»](javascript:TextPopup(this))


	Для открытия группы параметров «Область
	 построения»:


	- Выделите диаграмму.


	- Переведите кнопку ![](../../../../Picture/b_settings_off.png) «Настройки» на [панели
	 инструментов](../../../Starting.htm#structure_window) в активное состояние.


	- Выберите требуемую группу параметров.


![](../../../../Picture/settings_panel/construction_area.png)


Задайте параметры:


[![](../../../../opened.gif)![](../../../../closed.gif)Заливка](javascript:TextPopup(this))


	В раскрывающейся палитре выберите способ заливки фона подсказок:


		- Без заливки. По
		 умолчанию. Заливка не используется;


		- Сплошная заливка.
		 Выберите в раскрывающейся палитре цвет однотонной заливки области
		 построения. При необходимости укажите процент непрозрачности заливки;


		- Двухцветная заливка.
		 Выберите в раскрывающейся палитре начальный и конечный цвета двухцветного
		 градиента. При необходимости укажите угол наклона и процент непрозрачности
		 заливки;


		- Чередование цветов.
		 Выберите в раскрывающейся палитре первый и второй цвета, которые
		 будут чередоваться снизу вверх, начиная с первого цвета. При необходимости
		 укажите процент непрозрачности заливки;


		- Заливка со штриховкой.
		 Выполните действия:


			- Выберите цвет однотонной заливки в раскрывающейся палитре.
			 При необходимости укажите процент непрозрачности заливки и
			 цвета штриховки.


			- Выберите образец текстуры в раскрывающемся списке «Штриховка».


			- Выберите цвет штриховки в раскрывающейся палитре.


Для сброса заданных настроек и возврата к
 стилю по умолчанию нажмите кнопку ![](../../../../Picture/b_delete.png) «Удалить».


[![](../../../../opened.gif)![](../../../../closed.gif)Граница](javascript:TextPopup(this))


	Для настройки границы области построения задайте:


		- Тип линии. Выберите
		 тип линии из раскрывающегося списка;


		- Толщина линии. Задайте
		 требуемую толщину линии с помощью клавиатуры или редактора чисел.


Для сброса заданных настроек и возврата к
 стилю по умолчанию нажмите кнопку ![](../../../../Picture/b_delete.png) «Удалить».
