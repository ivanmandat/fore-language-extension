# Настройка визуализатора «Изображение»


Визуализатор «Изображение»
 предназначен для создания визуальных элементов.


Для добавления визуализатора на рабочую область:


	- выберите тип визуализатора ![](../../../../Picture/b_image.png) «Изображение»
	 в раскрывающемся меню кнопки ![](../../../../Picture/b_visualizers.png)
	 «Вставка» на панели инструментов
	 и щёлкните по ячейке слайда;


	- щёлкните по ячейке слайда и выберите тип визуализатора ![](../../../../Picture/b_image.png) «Изображение»
	 в раскрывающемся меню.


Для смены изображения:


	- [Выделите](../visualizers.htm#select) изображение.


	- Выполните команду ![](../../../../Picture/b_change_image.png) «Выбрать изображение» на всплывающей
	 панели настроек.


	- Выберите файл с изображением в открывшемся окне.


После выполнения последовательности действий изображение будет заменено
 на новое.


Для визуализатора «Изображение»
 доступны операции:


	- [изменение размеров визуализатора](../visualizers.htm#resize);


	- [перемещение визуализатора](../visualizers.htm#move);


	- [копирование визуализатора](../visualizers.htm#copy);


	- [дублирование визуализатора](../visualizers.htm#duplicate);


	- [удаление визуализатора](../visualizers.htm#delete).

[![](../../../../opened.gif)![](../../../../closed.gif)Настройка подсказки](javascript:TextPopup(this))


	Для отображения всплывающей подсказки при наведении курсора мыши
	 на изображение переведите переключатель «Подсказка»
	 на [панели настроек](../../../Starting.htm#structure_window)
	 в активное состояние и введите текст подсказки.


	Для скрытия всплывающей подсказки переведите переключатель «Подсказка» в неактивное состояние.
	 По умолчанию переключатель неактивен, и всплывающая подсказка не отображается.


[![](../../../../opened.gif)![](../../../../closed.gif)Настройка параметров
 изображения](javascript:TextPopup(this))


	Для сохранения пропорций при изменении размера визуализатора установите
	 флажок «Сохранять пропорции»
	 в группе «Параметры изображения» на
	 [панели настроек](../../../Starting.htm#structure_window).


	По умолчанию флажок «Сохранять
	 пропорции» установлен. Свободное пространство между границами
	 визуализатора и изображением заполняется фоновой заливкой.


[![](../../../../opened.gif)![](../../../../closed.gif)Настройка цвета фона](javascript:TextPopup(this))


	Для получения подробной информации о настройке обратитесь к разделу
	 «[Заливка
	 и фон](General_Settings.htm#background_color)».
