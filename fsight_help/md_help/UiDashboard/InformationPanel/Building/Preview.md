# Просмотр готовой информационной панели


Закончив оформление информационной панели, просмотрите готовую информационную
 панель.


Для открытия готовой информационной панели в навигаторе объектов выполните
 одно из действий:


	- выполните команду «Открыть»
	 контекстного меню информационной панели;


	- дважды щелкните по информационной панели.


Информационная панель будет переведена в режим просмотра, предназначенного
 для [работы с готовой информационной
 панелью](../Work/Work_with_information_bar.htm).


Для перехода между [режимами работы с
 информационной панелью](../Starting.htm#modes) используйте кнопки [главного
 меню](../Starting.htm#structure_window).
