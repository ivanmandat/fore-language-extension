# Работа с готовой информационной
 панелью


Для работы с готовой информационной панелью выполните шаги:


	- Используйте [интерактивные
	 элементы](../Building/Frames/visualizers_synchronization.htm#control) для управления отметкой.


	- Используйте [сортировку
	 данных таблицы](../Building/Visualizers/Table/Table_Formatting.htm#sort).


	- Измените размер строк и/или столбцов таблицы.


	- [Просмотрите готовую информационную
	 панель](../Building/Preview.htm).


	- [Экспортируйте
	 визуализаторы и/или слайды](export.htm).


	- [Распечатайте визуализаторы и/или
	 слайды](print.htm).


	- [Опубликуйте визуализаторы
	 и/или слайды](../../Project/Publishing.htm) в виде отдельных объектов.
