# Задание выражений


[Таблица предварительного просмотра](preview_table.htm) позволяет
 задавать выражения в специальном редакторе выражений. Выражения предназначены
 для:


	- задания правил фильтрации для ограничения выборки данных модели.
	 Правила фильтрации задаются в окне «Фильтр
	 модели данных»;


[![](../../opened.gif)![](../../closed.gif)Для открытия
 окна «Фильтр модели данных»](javascript:TextPopup(this))


	Нажмите кнопку ![](b_filter.png) «Задать фильтр» в таблице предварительного
	 просмотра.


	- создания дополнительных полей, вычисляемых по заданным выражениям.
	 Выражения вычисляемого поля задаются в окне «Вычисляемое
	 поле».


[![](../../opened.gif)![](../../closed.gif)Для открытия
 окна «Вычисляемое поле»](javascript:TextPopup(this))


	Нажмите кнопку ![](b_calculated_field.png) «Вычисляемое поле» в таблице предварительного
	 просмотра.


	Фильтр модели данных
	 Вычисляемое поле


		![](../../Picture/data_model_filter.png)


		![](../../Picture/calculated_field.png)


Сформируйте вычисляемое выражение в области формул. В формуле допускается
 использование операндов, арифметических операций, функций, символов.


Примечание.
 В окне «Фильтр модели данных»
 доступно задание логических выражений, по которым будет производиться
 фильтрация. В окне «Вычисляемое поле»
 доступно задание выражений любого типа. Если для вычисляемого поля задано
 логическое выражение, то поле может принимать значения «0» для ложного
 выражения и «1» для истинного выражения. В обоих окнах также доступно
 задание SQL-запросов. Синтаксис SQL-запросов зависит от типа СУБД источника
 данных.


[![](../../opened.gif)![](../../closed.gif)Работа с операндами](javascript:TextPopup(this))


	Нажмите кнопку ![](../../Picture/b_filter.png) «Задать
	 фильтр» в таблице предварительного просмотра.


[![](../../opened.gif)![](../../closed.gif)Работа с символами](javascript:TextPopup(this))


	В формулах допускается использование символов: букв, цифр, знаков
	 сравнения и круглых скобок.


	Для использования символов в выражении введите их с помощью клавиатуры.
	 Текстовые константы заключаются в одинарные кавычки (').


Для скрытия боковой панели нажмите кнопку ![](../../Picture/hide_left.png) «Скрыть панель полей».


## Пример


	Фильтр модели данных
	 Вычисляемое поле


		Пример выражения для фильтра:


		"Covid-19"."Выздоровлений за
		 день" > 100


		В указанной формуле выполняется проверка значения поля «Выздоровлений за день» таблицы
		 «Covid-19»:


			- если значение поля больше заданного значения, то условие
			 фильтра будет выполнено и поле будет отображаться в таблице
			 предварительного просмотра;


			- если значение поля меньше или равно заданному значению,
			 то условие фильтра не будет выполнено и поле будет скрыто
			 из таблицы предварительного просмотра и не будет участвовать
			 в дальнейшем [анализе
			 данных](../../InformationPanel/data_analysis.htm).


		В результате применения фильтра таблица предварительного просмотра
		 примет вид:


		![](../../Picture/preview_table_with_filter.png)


		Пример выражения для вычисляемого поля:


		"Covid-19"."Выздоровлений за
		 день" / "Covid-19"."Население страны" *
		 100


		В результате выполнения выражения в таблице предварительного
		 просмотра появится новое вычисляемое поле со значениями, посчитанными
		 по заданной формуле:


		![](../../Picture/preview_table_with_calculated_field.png)
