# Построение модели данных


Для построения модели данных:


	- [Настройте СУБД ClickHouse](ClickHouse.htm), если
	 требуется.


	- [Добавьте источники
	 данных](DataSources/add_data_sources.htm).


	- [Настройте режимы работы с данными](data_modes.htm).


	- [Добавьте объекты на рабочую
	 область и настройте связи между ними](Objects/add_objects.htm).


	- Отфильтруйте при необходимости сформированную модель данных
	 и/или добавьте в неё новые поля с помощью [таблицы
	 предварительного просмотра](Preview/preview_table.htm).


После выполнения действий сохраните модель данных:


	- нажмите кнопку «Сохранить»
	 [главного меню](create_data_model.htm);


	- нажмите сочетание клавиш CTRL+S.


Сохранение новой
 модели данных аналогично [созданию её копии](Save.htm).


При нажатии кнопки ![](../Picture/b_back.png) «Назад»
 главного меню для сохранения измененных данных модели данных подтвердите
 свои действия в информационном окне.


Примечание.
 Сохранение модели данных доступно только пользователям, обладающим [правами](Admin.chm::/02_ComonFunction/Admin_SecManager_CommonFunction.htm)
 на запись объекта. Для пользователей, обладающими правами только на чтение
 объекта, доступно создание [копии модели данных](Save.htm).
