## С чего начать


		 Сборка Dimensions
		 предназначена для работы с различными видами справочников репозитория.


		Изучив данный раздел, вы сможете:


			- создавать и настраивать параметры различных видов справочников с
			 помощью языка Fore;


			- настраивать параметры измерения фактов стандартного
			 куба и составного измерения виртуального куба;


			- работать с данными справочников;


			- управлять отметкой.


		Более подробно с возможностями сборки можно ознакомиться в разделе
		 «[Введение в сборку Dimensions](Intro/KeDims_Intro.htm)».


		Описание ресурсов сборки представлено в следующих подразделах:


			- [Интерфейсы
			 сборки Dimensions](interface/KeDims_Interface.htm)


			- [Перечисления сборки
			 Dimensions](Enums/KeDims_Enums.htm)


			- [Классы сборки Dimensions](Class/KeDims_Class.htm)


			- [Примеры](Samples/KeDims_Sample.htm)


		 Перед изучением сборки Dimensions рекомендуется ознакомиться
		 с разделами: [Среда
		 разработки](UiDevEnv.chm::/01_Development_Environment/01_Purpose_of_the_constructor/Purpose_of_the_constructor.htm) и [Руководство по языку Fore](Fore.chm::/Fore_Title.htm).
		 Также рекомендуется изучить раздел с описанием общего принципа
		 по созданию объектов с использованием языка Fore
		 и имеющиеся в «Форсайт. Аналитическая платформа»
		 [справочники](UiNavObj.chm::/reference_book/Create.htm).
