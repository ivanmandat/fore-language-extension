# Дополнительные объекты справочников


Кроме различных видов справочников в платформе доступны также дополнительные
 объекты, которые могут использоваться совместно с любым справочником.
 К дополнительным объектам справочников относятся:


	- [Альтернативные
	 иерархии](KeDims_Additional_Hierarchies.htm) - дополнительные настройки справочника, позволяющие при
	 работе отображать данные альтернативного справочника, связанного с
	 текущим по какому-либо признаку.


	- [Группы
	 элементов](KeDims_Additional_ElementGroup.htm) - наборы элементов справочника, объединенных по какому-либо
	 признаку с помощью примитивов. Могут применяться для ограничения списка
	 доступных элементов при построении справочника, либо для установки
	 отметки заданной группы элементов.


	- [Схемы
	 отметки](KeDims_Additional_SelectionSchema.htm) - набор готовых [отметок
	 элементов](../Selection/KeDims_Selection.htm), сформированные с помощью примитивов. Могут применяться
	 для установки заданной отметки при открытии объектов, либо автоматизации
	 отметки элементов, за счет использования динамических примитивов.
