# ICalendarDimAttribute


Сборка: Dimensions;


## Описание


Интерфейс ICalendarDimAttribute
 содержит свойства пользовательского атрибута календарного справочника.


## Иерархия наследования


[INamedEntity](KeSom.chm::/Interface/INamedEntity/INamedEntity.htm)
