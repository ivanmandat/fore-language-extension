# IDimAttributeHierarchies.FindByAttribute


## Синтаксис


FindByAttribute(AttributeKey: Integer): [IDimAttributeHierarchiesList](../IDimAttributeHierarchiesList/IDimAttributeHierarchiesList.htm);


## Параметры


AttributeKey. Ключ атрибута.


## Описание


Метод FindByAttribute выполняет
 поиск коллекции альтернативных иерархий по ключу атрибута.


## Комментарии


Для поиска коллекции альтернативных иерархий по уровню иерархии используйте
 [IDimAttributeHierarchies.FindByAttributeLevel](IDimAttributeHierarchies.FindByAttributeLevel.htm).
