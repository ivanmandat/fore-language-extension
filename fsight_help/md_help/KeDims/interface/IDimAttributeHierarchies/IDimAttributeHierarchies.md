# IDimAttributeHierarchies


Сборка: Dimensions;


## Описание


Интерфейс IDimAttributeHierarchies
 предназначен для работы с коллекцией альтернативных иерархий атрибута.


## Иерархия наследования


[IDimHierarchies](../IDimHierarchies/IDimHierarchies.htm)


IDimAttributeHierarchies


## Комментарии


Альтернативные иерархии позволяют для одного справочника настроить различные
 варианты отображения данных. В качестве альтернативных иерархий могут
 выступать другие справочники репозитория и их атрибуты.


## Свойства, унаследованные от [IDimHierarchies](../IDimHierarchies/IDimHierarchies.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](../IDimHierarchies/IDimHierarchies.Count.htm)
		 Свойство Count возвращает
		 число альтернативных иерархий справочника.


		 ![](../../Property_Image.gif)
		 [DefaultHierarchy](../IDimHierarchies/IDimHierarchies.DefaultHierarchy.htm)
		 Свойство DefaultHierarchy
		 устанавливает альтернативную иерархию, которая будет использоваться
		 по умолчанию при открытии справочника.


		 ![](../../Property_Image.gif)
		 [Item](../IDimHierarchies/IDimHierarchies.Item.htm)
		 Свойство Item возвращает
		 альтернативную иерархию из коллекции по индексу.


		 ![](../../Property_Image.gif)
		 [OriginalIndex](../IDimHierarchies/IDimHierarchies.OriginalIndex.htm)
		 Свойство OriginalIndex
		 определяет индекс основного справочника, содержащий ключи для
		 сопоставления элементов основного и альтернативного справочников.
