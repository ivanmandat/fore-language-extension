# IDimAttributeHierarchies.FindByAttributeLevel


## Синтаксис


FindByAttributeLevel(AttributeKey: Integer; LevelKey:
 Integer): [IDimAttributeHierarchy](../IDimAttributeHierarchy/IDimAttributeHierarchy.htm);


## Параметры


AttributeKey. Ключ атрибута;


LevelKey.
 Уровень иерархии атрибута.


## Описание


Метод FindByAttributeLevel выполняет
 поиск коллекции альтернативных иерархий по уровню иерархии атрибута.


## Комментарии


Для поиска коллекции альтернативных иерархий по атрибуту используйте
 [IDimAttributeHierarchies.FindByAttribute](IDimAttributeHierarchies.FindByAttribute.htm).
