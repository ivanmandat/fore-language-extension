# IDimElementsOrderSetup


Сборка: Dimensions;


## Описание


Интерфейс IDimElementsOrderSetup
 содержит свойства и методы для настройки пользовательской сортировки элементов
 справочника.


## Иерархия наследования


IDimElementsOrderSetup


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDimElementsOrderSetup.Count.htm)
		 Свойство Count возвращает
		 количество атрибутов, входящих в список сортировки справочника.


		 ![](../../Property_Image.gif)
		 [ReverseOrder](IDimElementsOrderSetup.ReverseOrder.htm)
		 Свойство ReverseOrder
		 определяет направление сортировки по указанному индексу атрибута.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IDimElementsOrderSetup.Add.htm)
		 Метод Add осуществляет
		 добавление атрибута справочника в список сортировки.


		 ![](../../Sub_Image.gif)
		 [AttributeKey](IDimElementsOrderSetup.AttributeKey.htm)
		 Метод AttributeKey
		 возвращает ключ атрибута по его индексу в списке сортировки.
