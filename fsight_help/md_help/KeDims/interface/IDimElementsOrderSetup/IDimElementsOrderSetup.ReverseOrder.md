# IDimElementsOrderSetup.ReverseOrder


## Синтаксис


ReverseOrder (Index: Integer): Boolean;


## Параметры


Index. Индекс атрибута в списке
 сортировки.


## Описание


Свойство ReverseOrder определяет
 направление сортировки по указанному индексу атрибута.


## Комментарии


Допустимые значения:


	- True. Сортировка по
	 убыванию;


	- False. Сортировка по
	 возрастанию.
