# IDimHierarchies.OriginalIndex


## Синтаксис


OriginalIndex: [IDimIndex](../IDimIndex/IDimIndex.htm);


## Описание


Свойство OriginalIndex определяет
 индекс основного справочника, содержащий ключи для сопоставления элементов
 основного и альтернативного справочников.


## Комментарии


Для определения альтернативной иерархии, которая будет использоваться
 по умолчанию при открытии справочника, используйте свойство [IDimHierarchies.DefaultHierarchy](IDimHierarchies.DefaultHierarchy.htm).

Использование свойства приведено в примере для [IDimHierarchies.Add](IDimHierarchies.Add.htm).
