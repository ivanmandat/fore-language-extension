# IDimHierarchies


Сборка: Dimensions;


## Описание


Интерфейс IDimHierarchies содержит
 свойства и методы для работы с коллекцией альтернативных иерархий справочника.


## Иерархия наследования


IDimHierarchies


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDimHierarchies.Count.htm)
		 Свойство Count возвращает
		 число альтернативных иерархий справочника.


		 ![](../../Property_Image.gif)
		 [DefaultHierarchy](IDimHierarchies.DefaultHierarchy.htm)
		 Свойство DefaultHierarchy
		 устанавливает альтернативную иерархию, которая будет использоваться
		 по умолчанию при открытии справочника.


		 ![](../../Property_Image.gif)
		 [Item](IDimHierarchies.Item.htm)
		 Свойство Item возвращает
		 альтернативную иерархию из коллекции по индексу.


		 ![](../../Property_Image.gif)
		 [OriginalIndex](IDimHierarchies.OriginalIndex.htm)
		 Свойство OriginalIndex
		 определяет индекс основного справочника, содержащий ключи для
		 сопоставления элементов основного и альтернативного справочников.
