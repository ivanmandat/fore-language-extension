# IDimCalendarPeriodSelection.Start


## Синтаксис


Start: DateTime;


## Описание


Свойство доступно только для чтения.


Свойство Start возвращает дату
 начала периода.


## Комментарии


По умолчанию дата начала периода совпадает с датой начала календарного
 справочника. Для изменения даты начала периода используйте метод [IDimCalendarPeriodSelection.Set](IDimCalendarPeriodSelection.Set_.htm).
