# IDimCalendarPeriodSelection.End_


## Синтаксис


End_: DateTime;


## Описание


Свойство доступно только для чтения.


Свойство End_ возвращает дату
 окончания периода.


## Комментарии


По умолчанию дата окончания периода совпадает с датой окончания календарного
 справочника. Для изменения даты окончания периода используйте метод [IDimCalendarPeriodSelection.Set_](IDimCalendarPeriodSelection.Set_.htm).
