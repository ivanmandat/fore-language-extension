# IDimCalendarPeriodSelection.Active


## Синтаксис


Active: Boolean;


## Описание


Свойство Active определяет,
 отмечены ли элементы в заданном периоде.


## Комментарии


Установите Active в True,
 чтобы применить отметку в периоде [[IDimCalendarPeriodSelection.Start](IDimCalendarPeriodSelection.Start.htm);
 [IDimCalendarPeriodSelection.End](IDimCalendarPeriodSelection.End_.htm)]
 для уровней [IDimCalendarPeriodSelection.Levels](IDimCalendarPeriodSelection.Levels.htm).


Свойство Active принимает значение
 False при отметке элемента вне
 периода или снятия отметки элемента в периоде. Для установки/снятия отметки
 элементов могут применяться методы интерфейса [IDimSelection](../IDimSelection/IDimSelection.htm).
