# IDimCalendarPeriodSelection.Set_


## Синтаксис


Set_(Start: DateTime; End_: DateTime);


## Параметры


Start. Дата начала периода.


End. Дата окончания периода.


## Описание


Метод Set_ устанавливает даты
 начала и окончания периода.


## Комментарии


Если значение параметра Start
 или End будет выходить
 за временные границы календарного справочника, для которого задается отметка,
 то значения, указанные вне периода календаря будут изменены на даты его
 начала/окончания.
