# IDimCalendarPeriodSelection


## Описание


Интерфейс IDimCalendarPeriodSelection
 предназначен для работы с отметкой элементов календарного справочника
 в определенный период.


## Комментарии


По умолчанию границы периода, в котором ведется отметка элементов, совпадают
 с датами начала и окончания календарного справочника.
