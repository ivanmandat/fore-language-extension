# IDimBlockLoader


## Описание


Интерфейс IDimBlockLoader содержит
 свойства и методы загрузчика данных блока справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ColCount](IDimBlockLoader.ColCount.htm)
		 Свойство ColCount возвращает
		 количество столбцов в загрузчике, в которых содержатся данные.


		 ![](../../Property_Image.gif)
		 [RowCapacity](IDimBlockLoader.RowCapacity.htm)
		 Свойство RowCapacity
		 определяет количество строк, для которых осуществляется резервирование
		 памяти под данные.


		 ![](../../Property_Image.gif)
		 [RowCount](IDimBlockLoader.RowCount.htm)
		 Свойство RowCount возвращает
		 количество строк в загрузчике.


		 ![](../../Property_Image.gif)
		 [Value](IDimBlockLoader.Value.htm)
		 Свойство Value определяет
		 значение элемента загрузчика.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddRow](IDimBlockLoader.AddRow.htm)
		 Метод AddRow осуществляет
		 добавление новой строки в загрузчик.
