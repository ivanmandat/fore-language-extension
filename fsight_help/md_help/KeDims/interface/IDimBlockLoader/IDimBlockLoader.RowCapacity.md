# IDimBlockLoader.RowCapacity


## Синтаксис


RowCapacity: Integer;


## Описание


Свойство RowCapacity определяет
 количество строк, для которых осуществляется резервирование памяти под
 данные.


## Комментарии


Память выделяется в соответствии с количеством строк и столбцов в загрузчике.
 Данное свойство актуально, если заранее известно количество строк, которые
 будут загружаться. По умолчанию память резервируется для 4096 строк данных.
