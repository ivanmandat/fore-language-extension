# ICustomDimAttributes.Order


## Синтаксис


Order: [ICustomDimAttribute](../ICustomDimAttribute/ICustomDimAttribute.htm);


## Описание


Свойство Order определяет атрибут
 с назначением «Порядок».


## Комментарии


По значениям атрибута с назначением «Порядок» осуществляется сортировка
 элементов при первом построении справочника.
