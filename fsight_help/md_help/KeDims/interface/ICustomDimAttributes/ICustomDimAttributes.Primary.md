# ICustomDimAttributes.Primary


## Синтаксис


Primary: [ICustomDimAttribute](../ICustomDimAttribute/ICustomDimAttribute.htm);


## Описание


Свойство Primary определяет
 атрибут, являющийся первичным ключом справочника.


## Комментарии


При создании измерения фактов, в основе которого лежит конструируемый
 справочник, автоматически создается атрибут с идентификатором «KEY».
 Данный атрибут имеет назначение «[Идентификатор](ICustomDimAttributes.Id.htm)»,
 также данный атрибут является первичным ключом. По первичному ключу создается
 уникальный первичный индекс. Этот индекс в дальнейшем используется для
 привязки элементов измерения фактов к полям источников данных.


При необходимости атрибут, являющийся первичным ключом, можно переопределить
 в свойстве Primary.
