# ICustomDimAttributes.FindByKey


## Синтаксис


FindByKey(AttributeKey: Integer): [ICustomDimAttribute](../ICustomDimAttribute/ICustomDimAttribute.htm);


## Параметры


AttributeKey. Ключ атрибута, по которому осуществляется
 поиск.


## Описание


Метод FindByKey осуществляет
 поиск атрибута по ключу.


## Комментарии


Если атрибут с указанным ключом не найден, то метод возвращает значение
 Null.
