# ICustomDimAttributes.Id


## Синтаксис


Id: [ICustomDimAttribute](../ICustomDimAttribute/ICustomDimAttribute.htm);


## Описание


Свойство Id определяет атрибут
 с назначением «Идентификатор».


## Комментарии


В свойстве задается атрибут справочника, значения которого являются
 идентификаторами элементов и позволяют уникальным образом определить каждый
 элемент.
