# ICustomDimAttributes.Orders


## Синтаксис


Orders: [ICustomDimOrders](../ICustomDimOrders/ICustomDimOrders.htm);


## Описание


Свойство Orders возвращает коллекцию
 атрибутов, по которым будет производиться сортировка элементов.


## Комментарии


Порядок сортировки определяется порядком расположения атрибутов в коллекции:
 сначала сортировка осуществляется по значениям первого атрибута, затем
 второго и так далее. По умолчанию в коллекцию сортировки включается атрибут
 с назначением «[Порядок](ICustomDimAttributes.Order.htm)».
