# IDimLevelInstance


## Описание


Интерфейс IDimLevelInstance
 содержит свойства уровня элементов справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Elements](IDimLevelInstance.Elements.htm)
		 Свойство Elements возвращает
		 объект, содержащий все элементы уровня справочника.


		 ![](../../Property_Image.gif)
		 [Level](IDimLevelInstance.Level.htm)
		 Свойство Level возвращает
		 объект, содержащий структуру уровня справочника.


		 ![](../../Property_Image.gif)
		 [LevelKey](IDimLevelInstance.LevelKey.htm)
		 Свойство LevelKey возвращает
		 ключ уровня справочника.


		 ![](../../Property_Image.gif)
		 [Name](IDimLevelInstance.Name.htm)
		 Свойство Name возвращает
		 наименование уровня справочника.


		 ![](../../Property_Image.gif)
		 [Number](IDimLevelInstance.Number.htm)
		 Свойство Number возвращает
		 порядковый номер уровня в справочнике.
