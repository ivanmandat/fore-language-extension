# IDimCalendarSelection


## Описание


Интерфейс IDimCalendarSelection
 предназначен для задания отметки элементов календарного справочника.


## Комментарии


Данный интерфейс служит для специфичной отметки элементов календарных
 справочников: в указанном временном диапазоне и по указанным уровням.
