# IDimAttributeProbe


## Описание


Интерфейс IDimAttributeProbe
 предназначен для обратного вызова с целью определения по каким атрибутам
 осуществлять поиск.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Probe](IDimAttributeProbe.Probe.htm)
		 Метод Probe проверяет
		 атрибут: если значение True,
		 то поиск будет осуществлен по данному атрибуту, в противном случае
		 - нет.
