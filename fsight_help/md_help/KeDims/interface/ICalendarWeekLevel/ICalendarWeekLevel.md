# ICalendarWeekLevel


## Описание


Интерфейс ICalendarWeekLevel
 содержит свойства для работы с уровнем «Недели» в календарном справочнике.

		 Свойство FirstDayOfWeek
		 определяет первый день недели.


		 ![](../../Property_Image.gif)
		 [RestDays](ICalendarWeekLevel.RestDays.htm)
		 Свойство RestDays определяет
		 выходные дни недели.
