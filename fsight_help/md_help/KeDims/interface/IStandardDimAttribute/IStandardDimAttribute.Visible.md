# IStandardDimAttribute.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет
 признак видимости значений по данному атрибуту.


## Комментарии


Свойство оставлено для совместимости. Может использоваться для определения
 видимости столбцов, при работе со справочниками на пользовательских формах.
