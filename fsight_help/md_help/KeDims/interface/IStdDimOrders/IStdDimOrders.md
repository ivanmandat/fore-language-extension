# IStdDimOrders


## Описание


Интерфейс IStdDimOrders содержит
 свойства и методы коллекции атрибутов, по которым производится сортировка
 табличного справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IStdDimOrders.Count.htm)
		 Свойство Count возвращает
		 количество атрибутов, входящих в список сортировки табличного
		 справочника.


		 ![](../../Property_Image.gif)
		 [Item](IStdDimOrders.Item.htm)
		 Свойство Item возвращает
		 настройки порядка сортировки атрибута, входящего в список сортировки
		 табличного справочника.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IStdDimOrders.Add.htm)
		 Метод Add осуществляет
		 добавление атрибута справочника в список сортировки.


		 ![](../../Sub_Image.gif)
		 [Delete](IStdDimOrders.Delete.htm)
		 Метод Delete осуществляет
		 удаление атрибута из списка сортировки табличного справочника.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IStdDimOrders.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск по ключу и возвращает атрибут, входящий в список сортировки.


		 ![](../../Sub_Image.gif)
		 [Move](IStdDimOrder.Move.htm)
		 Метод Move осуществляет
		 перемещение атрибутов в списке сортировки.
