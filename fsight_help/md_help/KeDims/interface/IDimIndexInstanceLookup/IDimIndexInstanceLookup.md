# IDimIndexInstanceLookup


Сборка: Dimensions;


## Описание


Интерфейс IDimIndexInstanceLookup
 содержит свойства и методы для поиска элементов справочника по индексу.


## Иерархия наследования


IDimIndexInstanceLookup


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attribute](IDimIndexInstanceLookup.Attribute.htm)
		 Свойство Attribute
		 возвращает структуру атрибута, по которому может производиться
		 поиск.


		 ![](../../Property_Image.gif)
		 [AttributeCount](IDimIndexInstanceLookup.AttributeCount.htm)
		 Свойство AttributeCount
		 возвращает количество атрибутов, по которым может производиться
		 поиск.


		 ![](../../Property_Image.gif)
		 [AttributeValue](IDimIndexInstanceLookup.AttributeValue.htm)
		 Свойство AttributeValue
		 определяет искомое значение по указанному атрибуту.


		 ![](../../Property_Image.gif)
		 [Index](IDimIndexInstanceLookup.Index.htm)
		 Свойство Index возвращает
		 объект-родитель.


		 ![](../../Property_Image.gif)
		 [Level](IDimIndexInstanceLookup.Level.htm)
		 Свойство Level определяет
		 уровень справочника, в котором выполняется поиск элемента.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [LookUp](IDimIndexInstanceLookup.LookUp.htm)
		 Метод LookUp осуществляет
		 поиск элементов, удовлетворяющих условию и возвращает массив найденных
		 элементов.


		 ![](../../Sub_Image.gif)
		 [LookUpFirst](IDimIndexInstanceLookup.LookUpFirst.htm)
		 Метод LookUpFirst осуществляет
		 поиск первого, удовлетворяющего условиям, элемента.
