# IDimIndexInstanceLookup.SetIndexAttributesValues


## Синтаксис


SetIndexAttributesValues(Value: Variant):


## Параметры


Value - Параметр определяет
 значение аргумента, которое необходимо найти. В качестве аргумента может
 использоваться массив значений (Array of Variant) размером, равным количеству
 атрибутов. Либо, если атрибут в индексе только один, просто значение.


## Описание


Метод SetIndexAttributesValues
 предназначен для установки значения поиска по всем атрибутам индекса.
