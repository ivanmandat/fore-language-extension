# IDimHierarchyCreator.RootElementName


## Синтаксис


RootElementName: String;


## Описание


Свойство RootElementName определяет
 имя корневого элемента.


## Комментарии


При указании свойства RootElementName,
 указание для свойства [RootElement](IDimHierarchyCreator.RootElement.htm)
 значения True необязательно.
