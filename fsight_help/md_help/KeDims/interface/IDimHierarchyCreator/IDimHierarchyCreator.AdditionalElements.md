# IDimHierarchyCreator.AdditionalElements


## Синтаксис


AdditionalElements: [IDimSelection](../IDimSelection/IDimSelection.htm);


## Описание


Свойство AdditionalElements
 определяет дополнительные элементы измерения.


## Комментарии


Дополнительные элементы измерения добавляются в конец альтернативной
 иерархии.
