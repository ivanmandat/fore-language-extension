# IDimHierarchyCreator.RootElement


## Синтаксис


RootElement: Boolean;


## Описание


Свойство RootElement определяет,
 будет ли создаваться корневой элемент.


## Комментарии


По умолчанию для свойства установлено значение False,
 то есть корневой элемент не создаётся.
