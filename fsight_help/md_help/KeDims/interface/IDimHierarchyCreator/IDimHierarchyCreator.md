# IDimHierarchyCreator


Сборка: Dimensions;


## Описание


Интерфейс IDimHierarchyCreator содержит
 свойства, используемые для динамической настройки альтернативной иерархии
 справочника.


## Иерархия наследования


IDimHierarchyCreator


## Комментарии


Альтернативная иерархия формируется с помощью отметки элементов, а также
 из элементов, по атрибутам которых осуществляется [группировка](DataEntryForms.chm::/Desktop/Table/Sidehead_Structure.htm#group).


## Свойства


		 Имя свойства


		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AdditionalAttributes](IDimHierarchyCreator.AdditionalAttributes.htm)


		 Свойство AdditionalAttributes
		 определяет дополнительные атрибуты элемента.


		 ![](../../Property_Image.gif)
		 [AdditionalElements](IDimHierarchyCreator.AdditionalElements.htm)


		 Свойство AdditionalElements
		 определяет дополнительные элементы.


		 ![](../../Property_Image.gif)
		 [Attributes](IDimHierarchyCreator.Attributes.htm)


		 Свойство Attributes
		 определяет массив атрибутов элемента.


		 ![](../../Property_Image.gif)
		 [BasicHierarchyInstance](IDimHierarchyCreator.BasicHierarchyInstance.htm)


		 Свойство BasicHierarchyInstance
		 определяет базовую альтернативную иерархию справочника.


		 ![](../../Property_Image.gif)
		 [HierarchyInstance](IDimHierarchyCreator.HierarchyInstance.htm)


		 Свойство HierarchyInstance
		 определяет альтернативную иерархию справочника.


		 ![](../../Property_Image.gif)
		 [LevelNumber](IDimHierarchyCreator.LevelNumber.htm)


		 Свойство LevelNumber
		 определяет индекс уровня справочника, для элементов которого будет
		 включена группировка.


		 ![](../../Property_Image.gif)
		 [RootElement](IDimHierarchyCreator.RootElement.htm)


		 Свойство RootElement
		 определяет, будет ли создаваться корневой элемент.


		 ![](../../Property_Image.gif)
		 [RootElementName](IDimHierarchyCreator.RootElementName.htm)


		 Свойство RootElementName
		 определяет имя корневого элемента.


		 ![](../../Property_Image.gif)
		 [Selection](IDimHierarchyCreator.Selection.htm)


		 Свойство Selection
		 определяет отметку, по которой формируется альтернативная иерархия.
