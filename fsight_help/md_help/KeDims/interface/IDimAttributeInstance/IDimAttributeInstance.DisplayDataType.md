# IDimAttributeInstance.DisplayDataType


## Синтаксис


DisplayDataType: [DbDataType](Dal.chm::/Enums/DbDataType.htm);


## Описание


Свойство DisplayDataType возвращает
 тип данных отображаемого значения.


## Комментарии


При использовании в справочниках НСИ связей, отображаемые значения атрибутов,
 используемых для связи, могут иметь тип данных, не совпадающий с типом
 данных самого атрибута. Тип отображаемых данных зависит от установленного
 [формата
 отображения](KeRds.chm::/Interface/IRdsLink/IRdsLink.DisplayFormat.htm). Получить тип отображаемого значения можно
 используя данное свойство.
