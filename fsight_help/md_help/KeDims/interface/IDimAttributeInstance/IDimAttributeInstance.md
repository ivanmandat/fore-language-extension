# IDimAttributeInstance


## Описание


Интерфейс IDimAttributeInstance
 содержит свойства и методы для работы со значениями атрибута справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attribute](IDimAttributeInstance.Attribute.htm)
		 Свойство Attribute
		 возвращает структуру атрибута справочника.


		 ![](../../Property_Image.gif)
		 [DataDomain](IDimAttributeInstance.DataDomain.htm)
		 Свойство DataDomain
		 возвращает тип данных для хранения в атрибуте многобайтовой информации.


		 ![](../../Property_Image.gif)
		 [DisplayDataDomain](IDimAttributeInstance.DisplayDataDomain.htm)
		 Свойство DisplayDataDomain
		 возвращает тип отображаемой в атрибуте многобайтовой информации.


		 ![](../../Property_Image.gif)
		 [DisplayDataType](IDimAttributeInstance.DisplayDataType.htm)
		 Свойство DisplayDataType
		 возвращает тип данных отображаемого значения.


		 ![](../../Property_Image.gif)
		 [DisplayDifferentValue](IDimAttributeInstance.DisplayDifferentValue.htm)
		 Свойство DisplayDifferentValue
		 возвращает признак отображения по данному атрибуту значения, не
		 соответствующего реальному значению.


		 ![](../../Property_Image.gif)
		 [DisplayValue](IDimAttributeInstance.DisplayValue.htm)
		 Свойство DisplayValue
		 возвращает отображаемое значение по текущему атрибуту для указанного
		 элемента.


		 ![](../../Property_Image.gif)
		 [Value](IDimAttributeInstance.Value.htm)
		 Свойство Value возвращает
		 значение атрибута элемента.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [LookupDisplayValue](IDimAttributeInstance.LookupDisplayValue.htm)
		 Метод LookupDisplayValue
		 осуществляет поиск отображаемого значения атрибута и возвращает
		 индекс элемента в случае удачного поиска.


		 ![](../../Sub_Image.gif)
		 [LookupValue](IDimAttributeInstance.LookupValue.htm)
		 Метод LookupValue осуществляет
		 поиск значения атрибута и возвращает индекс элемента в случае
		 удачного поиска.
