# IDimTextCriteria


## Описание


Интерфейс IDimTextCriteria определяет
 условия поиска текста.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CriteriaOptions](IDimTextCriteria.CriteriaOptions.htm)
		 Свойство CriteriaOptions
		 определяет параметры поиска.


		 ![](../../Property_Image.gif)
		 [Probe](IDimTextCriteria.Probe.htm)
		 Свойство Probe осуществляет
		 обратный вызов для определения, по каким атрибутам осуществлять
		 поиск.


		 ![](../../Property_Image.gif)
		 [SimpleProbe](IDimTextCriteria.SimpleProbe.htm)
		 Свойство SimpleProbe
		 определяет объект, с помощью которого осуществляется проверка
		 атрибутов, по значениям которых будет производиться поиск.


		 ![](../../Property_Image.gif)
		 [Text](IDimTextCriteria.Text.htm)
		 Свойство Text определяет
		 текст поиска.
