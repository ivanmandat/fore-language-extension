# IDimAsCalendarLevel


## Описание


Интерфейс IDimAsCalendarLevel
 содержит свойства уровня справочника для сопоставления, при использовании
 справочника в качестве календарного.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Key](IDimAsCalendarLevel.Key.htm)
		 Свойство Key возвращает
		 ключ уровня справочника, который сопоставлен уровню календаря.


		 ![](../../Property_Image.gif)
		 [Level](IDimAsCalendarLevel.Level.htm)
		 Свойство Level возвращает
		 уровень календаря, сопоставленный с уровнем текущего справочника.
