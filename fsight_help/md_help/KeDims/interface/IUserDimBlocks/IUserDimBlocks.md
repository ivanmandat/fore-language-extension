# IUserDimBlocks


## Описание


Интерфейс IUserDimBlocks содержит
 свойства и методы коллекции блоков вычисляемого справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IUserDimBlocks.Count.htm)
		 Свойство Count возвращает
		 количество блоков вычисляемого справочника.


		 ![](../../Property_Image.gif)
		 [Item](IUserDimBlocks.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий блок вычисляемого справочника.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IUserDimBlocks.Add.htm)
		 Метод Add осуществляет
		 добавление блока вычисляемого справочника.


		 ![](../../Sub_Image.gif)
		 [Clear](IUserDimBlocks.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех блоков вычисляемого справочника.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IUserDimBlocks.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск по ключу и возвращает объект, содержащий блок вычисляемого
		 справочника.


		 ![](../../Sub_Image.gif)
		 [Remove](IUserDimBlocks.Remove.htm)
		 Метод Remove осуществляет
		 удаление блока вычисляемого справочника по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IUserDimBlocks.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление блока вычисляемого справочника по ключу.
