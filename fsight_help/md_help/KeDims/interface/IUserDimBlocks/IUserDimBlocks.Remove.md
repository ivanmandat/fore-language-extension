# IUserDimBlocks.Remove


## Синтаксис


Remove(Index: Variant): Boolean;


## Параметры


Index — индекс
 блока.


## Описание


Метод Remove осуществляет удаление
 блока вычисляемого справочника и возвращает True
 в случае удачного удаления. Индекс блока передается посредством параметра
 Index.
