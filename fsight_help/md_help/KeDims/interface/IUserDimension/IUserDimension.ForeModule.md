# IUserDimension.ForeModule


## Синтаксис


ForeModule: [IModule](KeSom.chm::/Interface/IModule/IModule.htm);


## Описание


Свойство ForeModule определяет
 модуль/форму, в котором содержится макрос, осуществляющий построение справочника.


## Комментарии


Наименование макроса указывается в свойстве [IUserDimension.ForeSub](IUserDimension.ForeSub.htm).

Пример использования приведён в описании свойства [IUserDimension.ForeSub](IUserDimension.ForeSub.htm).
