# ICalendarSetup


Сборка: Dimensions;


## Описание


Интерфейс ICalendarSetup предназначен
 для работы параметрами календаря, элементы которого были построены динамически.


## Иерархия наследования


ICalendarSetup


## Комментарии


Текущие параметры календаря возвращает свойство [ICalendarDimension.Setup](../ICalendarDimension/ICalendarDimension.Setup.htm),
 для создания новых параметров используйте метод [IDynamicCalendarInstance.NewSetup](../IDynamicCalendarInstance/IDynamicCalendarInstance.NewSetup.htm).
