# ICalendarSetup.StartDay


## Синтаксис


StartDay: DateTime;


## Описание


Свойство StartDay определяет
 дату начала календаря.


## Комментарии


Дата начала календаря при задании свойства StartDay
 формируется: 01.01.ГГГГ, где ГГГГ - год, который указан в дате пользователя.


Дату окончания календаря определяет свойство [ICalendarSetup.EndDay](ICalendarSetup.EndDay.htm).
