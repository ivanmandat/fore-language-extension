# ICalendarSetup.Levels


## Синтаксис


Levels: [DimCalendarLevelSet](../../Enums/DimCalendarLevelSet.htm);


## Описание


Свойство Levels определяет
 уровни, присутствующие в календаре.


## Комментарии


Если необходимо, чтобы календарь содержал несколько уровней, то укажите
 необходимые элементы перечисления [DimCalendarLevelSet](../../Enums/DimCalendarLevelSet.htm)
 через «Or».
