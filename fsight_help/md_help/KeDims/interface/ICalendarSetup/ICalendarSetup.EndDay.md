# ICalendarSetup.EndDay


## Синтаксис


EndDay: DateTime;


## Описание


Свойство EndDay определяет
 дату окончания календаря.


## Комментарии


Дата окончания календаря при задании свойства EndDay
 формируется: 31.12.ГГГГ, где ГГГГ - год, который указан в дате пользователя.


Дату начала календаря определяет свойство [ICalendarSetup.StartDay](ICalendarSetup.StartDay.htm).
