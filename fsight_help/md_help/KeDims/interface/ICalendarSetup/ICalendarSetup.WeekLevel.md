# ICalendarSetup.WeekLevel


## Синтаксис


WeekLevel: [ICalendarWeekLevel](../ICalendarWeekLevel/ICalendarWeekLevel.htm);


## Описание


Свойство WeekLevel определяет
 параметры уровня «Недели» в календаре.


## Комментарии


Используя данное свойство можно изменить число выходных дней и первый
 день недели.
