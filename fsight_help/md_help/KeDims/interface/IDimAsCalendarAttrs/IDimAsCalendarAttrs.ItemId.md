# IDimAsCalendarAttrs.ItemId


## Синтаксис


ItemId(Index: Integer): String;


## Параметры


Index.
 Индекс атрибута календаря.


## Описание


Свойство ItemId возвращает идентификатор
 атрибута календаря с указанным индексом.


## Комментарии


Для сопоставления атрибута календаря с атрибутом текущего справочника
 используйте свойство [IDimAsCalendarAttrs.IdAttr](IDimAsCalendarAttrs.IdAttr.htm).
