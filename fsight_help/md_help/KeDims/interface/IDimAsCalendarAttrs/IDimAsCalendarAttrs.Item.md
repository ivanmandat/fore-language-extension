# IDimAsCalendarAttrs.Item


## Синтаксис


Item(Index: Integer): [IDimAsCalendarAttr](../IDimAsCalendarAttr/IDimAsCalendarAttr.htm);


## Параметры


index. Индекс атрибута календаря.


## Описание


Свойство Item возвращает параметры
 сопоставления атрибута текущего справочника с атрибутом календаря с указанным
 индексом.
