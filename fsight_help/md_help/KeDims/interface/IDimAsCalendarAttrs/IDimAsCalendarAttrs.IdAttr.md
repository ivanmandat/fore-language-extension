# IDimAsCalendarAttrs.IdAttr


## Синтаксис


IdAttr(Attr: String): [IDimAttribute](../IDimAttribute/IDimAttribute.htm);


## Параметры


Attr.
 Идентификатор атрибута.


## Описание


Свойство IdAttr сопоставляет
 атрибут справочника атрибуту календаря с заданным идентификатором.


## Комментарии


В текущей коллекции количество элементов соответствует количеству обязательных
 атрибутов календаря. Идентификатор атрибута, указываемый в параметре Attr, можно получить, используя
 свойство [IDimAsCalendarAttrs.ItemId](IDimAsCalendarAttrs.ItemId.htm).
