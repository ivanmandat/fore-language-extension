# IDimAsCalendarAttrs


## Описание


Интерфейс IDimAsCalendarAttrs
 содержит свойства и методы коллекции атрибутов для сопоставления при использовании
 справочника в качестве календарного.


## Комментарии


Коллекцию атрибутов можно получить, используя свойство [IDimAsCalendar.Attrs](../IDimAsCalendar/IDimAsCalendar.Attrs.htm).
 Коллекция содержит список обязательных атрибутов календаря, которые должны
 быть сопоставлены с атрибутами текущего справочника. Для сопоставления
 используйте свойство [IdAttr](IDimAsCalendarAttrs.IdAttr.htm).

		 Свойство ItemId возвращает
