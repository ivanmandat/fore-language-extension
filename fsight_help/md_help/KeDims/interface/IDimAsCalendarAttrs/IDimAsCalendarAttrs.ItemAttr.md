# IDimAsCalendarAttrs.ItemAttr


## Синтаксис


ItemAttr(Index: Integer): [IDimAttribute](../IDimAttribute/IDimAttribute.htm);


## Параметры


index. Индекс атрибута календаря.


## Описание


Свойство ItemAttr возвращает
 атрибут справочника, который сопоставлен с атрибутом календаря с указанным
 индексом.


## Комментарии


Свойство возвращает значение Null,
 если атрибут календаря не сопоставлен с атрибутом текущего справочника.
 Для сопоставления используйте свойство [IDimAsCalendarAttrs.IdAttr](IDimAsCalendarAttrs.IdAttr.htm).
