 свойства атрибута справочника.

		 ![](../../Property_Image.gif)
		 [DataType](IDimAttribute.DataType.htm)
		 Свойство DataType возвращает

		 ![](../../Property_Image.gif)
		 [DisplayDataDomain](IDimAttribute.DisplayDataDomain.htm)
		 Свойство DisplayDataDomain
		 возвращает тип отображаемой в атрибуте многобайтовой информации.


		 ![](../../Property_Image.gif)
		 [DisplayDataType](IDimAttribute.DisplayDataType.htm)
		 Свойство DisplayDataType
		 возвращает тип отображаемого значения атрибута.


		 ![](../../Property_Image.gif)
		 [HasHierarchy](IDimAttribute.HasHierarchy.htm)
		 Свойство HasHierarchy
		 возвращает признак того, используется ли атрибут в качестве альтернативной
		 иерархии.


		 ![](../../Property_Image.gif)
		 [HasMultipleValues](IDimAttribute.HasMultipleValues.htm)
		 Свойство HasMultipleValues
		 возвращает признак доступности множественной отметки элемента
		 справочника.


		 ![](../../Property_Image.gif)
		 [Hierarchies](IDimAttribute.Hierarchies.htm)
		 Свойство Hierarchies
		 возвращает коллекцию иерархий атрибута.


		 ![](../../Property_Image.gif)
		 [ObjectLinked](IDimAttribute.ObjectLinked.htm)
		 Свойство ObjectLinked
		 возвращает признак, является ли атрибут справочника ссылкой на
		 объект репозитория.


		 ![](../../Property_Image.gif)
		 [Visible](IDimAttribute.Visible.htm)
		 Свойство Visible определяет,
		 отображать ли значения атрибута.
