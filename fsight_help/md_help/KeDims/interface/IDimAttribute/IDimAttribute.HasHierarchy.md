# IDimAttribute.HasHierarchy


## Синтаксис


HasHierarchy: Boolean;


## Описание


Свойство HasHierarchy возвращает
 признак того, используется ли атрибут в качестве альтернативной иерархии.


## Комментарии


Допустимые значения:


	- True. Атрибут используется
	 в качестве альтернативной иерархии;


	- False. Атрибут не используется
	 в качестве альтернативной иерархии.
