# IUserDimIndexAttributes


## Описание


Интерфейс IUserDimIndexAttributes
 содержит свойства и методы коллекции атрибутов индекса вычисляемого справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IUserDimIndexAttributes.Count.htm)
		 Свойство Count возвращает
		 количество атрибутов вычисляемого справочника, по которым создан
		 индекс.


		 ![](../../Property_Image.gif)
		 [Item](IUserDimIndexAttributes.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий атрибут вычисляемого справочника, по которому
		 создан индекс.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IUserDimIndexAttributes.Add.htm)
		 Метод Add осуществляет
		 добавление атрибута в индекс.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IUserDimIndexAttributes.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск атрибута справочника в индексе.


		 ![](../../Sub_Image.gif)
		 [Remove](IUserDimIndexAttributes.Remove.htm)
		 Метод Remove осуществляет
		 удаление атрибута из индекса.
