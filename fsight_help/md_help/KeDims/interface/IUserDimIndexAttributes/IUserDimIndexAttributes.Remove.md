# IUserDimIndexAttributes.Remove


## Синтаксис


Remove(Value: [IUserDimAttribute](../IUserDimAttribute/IUserDimAttribute.htm));


## Параметры


Value — атрибут
 справочника, который необходимо удалить из индекса.


## Описание


Метод Remove осуществляет удаление
 атрибута, передаваемого посредством параметра Value,
 из индекса справочника.
