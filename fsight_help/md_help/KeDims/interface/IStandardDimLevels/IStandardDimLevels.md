# IStandardDimLevels


## Описание


Интерфейс IStandardDimLevels
 содержит свойства и методы коллекции уровней табличного справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IStandardDimLevels.Count.htm)
		 Свойство Count возвращает
		 количество уровней табличного справочника.


		 ![](../../Property_Image.gif)
		 [Item](IStandardDimLevels.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий уровень табличного справочника.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IStandardDimLevels.Add.htm)
		 Метод Add осуществляет
		 добавление уровня в справочник.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IStandardDimLevels.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск уровня справочника по ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](IStandardDimLevels.Remove.htm)
		 Метод Remove осуществляет
		 удаление уровня справочника по индексу.
