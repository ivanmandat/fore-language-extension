# IDynamicCalendarInstance


Сборка: Dimensions;


## Описание


Интерфейс IDynamicCalendarInstance
 предназначен для работы с календарем, элементы которого были построены
 динамически.


## Иерархия наследования


IDynamicCalendarInstance


## Комментарии


При открытии календаря, у которого [ICalendarDimension.DefaultOpenType](../ICalendarDimension/ICalendarDimension.DefaultOpenType.htm)
 = CalendarOpenType.Flat, либо методом [ICalendarDimension.OpenAs](../ICalendarDimension/ICalendarDimension.OpenAs.htm)(CalendarOpenType.Flat),
 будет возвращен объект, поддерживающий интерфейс IDynamicCalendarInstance.
