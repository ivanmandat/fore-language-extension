# IDynamicCalendarInstance.Iterator


## Синтаксис


Iterator(Setup: [ICalendarSetup](../ICalendarSetup/ICalendarSetup.htm)):
 [IDimIterator](../IDimIterator/IDimIterator.htm);


## Параметры


Setup. Параметры календаря.


## Описание


Свойство доступно только для чтения.


Свойство Iterator возвращает
 объект, предназначенный для навигации по элементам календаря.


## Комментарии


Для создания новых параметров календаря используйте метод [IDynamicCalendarInstance.NewSetup](IDynamicCalendarInstance.NewSetup.htm).
