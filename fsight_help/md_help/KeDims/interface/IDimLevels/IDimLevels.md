# IDimLevels


## Описание


Интерфейс IDimLevels содержит
 свойства и методы коллекции уровней справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDimLevels.Count.htm)
		 Свойство Count возвращает
		 количество уровней справочника.


		 ![](../../Property_Image.gif)
		 [Item](IDimLevels.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий уровень справочника.
