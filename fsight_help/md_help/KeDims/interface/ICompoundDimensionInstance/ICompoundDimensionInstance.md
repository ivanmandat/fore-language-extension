# ICompoundDimensionInstance


## Описание


Интерфейс ICompoundDimensionInstance
 содержит свойства и методы для работы с данными составного справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Selection](ICompoundDimensionInstance.Selection.htm)
		 Свойство Selection
		 возвращает отметку коллекции справочников, по которым построен
		 составной справочник.


		 ![](../../Property_Image.gif)
		 [Type](ICompoundDimensionInstance.Type.htm)
		 Свойство Type возвращает
		 тип составного измерения.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ComposeElement](ICompoundDimensionInstance.ComposeElement.htm)
		 Метод ComposeElement
		 возвращает индекс элемента составного справочника по его источникам.


		 ![](../../Sub_Image.gif)
		 [DecomposeElement](ICompoundDimensionInstance.DecomposeElement.htm)
		 Метод DecomposeElement
		 раскладывает элемент составного справочника на источники.
