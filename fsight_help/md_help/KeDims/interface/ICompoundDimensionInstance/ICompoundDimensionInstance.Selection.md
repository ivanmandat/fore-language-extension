# ICompoundDimensionInstance.Selection


## Синтаксис


Selection: [IDimSelectionSet](../IDimSelectionSet/IDimSelectionSet.htm);


## Описание


Свойство Selection возвращает
 отметку коллекции справочников, по которым построен составной справочник.


Примечание.
 Отметка справочника из коллекции ([IDimSelectionSet.Item](../IDimSelectionSet/IDimSelectionSet.Item.htm))
 будет являться составной отметкой ([ICompoundSelection](../ICompoundSelection/ICompoundSelection.htm)).
