# IUserDimIndexes


## Описание


Интерфейс IUserDimIndexes содержит
 свойства и методы коллекции индексов вычисляемого справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IUserDimIndexes.Count.htm)
		 Свойство Count возвращает
		 количество индексов вычисляемого справочника.


		 ![](../../Property_Image.gif)
		 [Item](IUserDimIndexes.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий индекс вычисляемого справочника.


		 ![](../../Property_Image.gif)
		 [PrimaryIndex](IUserDimIndexes.PrimaryIndex.htm)
		 Свойство PrimaryIndex
		 возвращает первичный индекс вычисляемого справочника.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IUserDimIndexes.Add.htm)
		 Метод Add осуществляет
		 добавление индекса в вычисляемый справочник.


		 ![](../../Sub_Image.gif)
		 [Clear](IUserDimIndexes.Clear.htm)
		 Метод Clear удаляет
		 все индексы вычисляемого справочника.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IUserDimIndexes.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск и возвращает объект, содержащий индекс вычисляемого справочника.


		 ![](../../Sub_Image.gif)
		 [Remove](IUserDimIndexes.Remove.htm)
		 Метод Remove осуществляет
		 удаление индекса вычисляемого справочника по номеру.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IUserDimIndexes.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление индекса вычисляемого справочника по ключу.
