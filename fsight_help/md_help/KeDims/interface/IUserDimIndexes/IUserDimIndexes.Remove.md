# IUserDimIndexes.Remove


## Синтаксис


Remove(Index: Variant): Boolean;


## Параметры


Index - номер
 индекс.


## Описание


Метод Remove осуществляет удаление
 индекса вычисляемого справочника и возвращает True
 в случае удачного удаления. Номер индекса передается посредством параметра
 Index.
