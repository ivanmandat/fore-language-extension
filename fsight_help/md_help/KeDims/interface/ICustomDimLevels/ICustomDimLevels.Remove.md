# ICustomDimLevels.Remove


## Синтаксис


Remove(Index: Variant): Boolean;


## Параметры


Index. Индекс
 измерения.


## Описание


Метод Remove осуществляет удаление
 уровня по индексу и возвращает признак удачного удаления.


## Комментарии


Метод возвращает значение True,
 если уровень был удален и False,
 если удаление уровня по каким-либо причинам невозможно.
