# ICustomDimLevels


## Описание


Интерфейс ICustomDimLevels содержит
 свойства и методы для работы с коллекцией уровней конструируемого справочника.


## Комментарии


Коллекцию уровней возвращает свойство [ICustomDimension.Levels](../ICustomDimension/ICustomDimension.Levels.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICustomDimLevels.Count.htm)
		 Свойство Count возвращает
		 количество уровней в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ICustomDimLevels.Item.htm)
		 Свойство Item возвращает
		 параметры указанного уровня.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ICustomDimLevels.Add.htm)
		 Метод Add осуществляет
		 добавление нового уровня.


		 ![](../../Sub_Image.gif)
		 [Clear](ICustomDimLevels.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех уровней справочника.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ICustomDimLevels.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск уровня по ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](ICustomDimLevels.Remove.htm)
		 Метод Remove осуществляет
		 удаление уровня по индексу.
