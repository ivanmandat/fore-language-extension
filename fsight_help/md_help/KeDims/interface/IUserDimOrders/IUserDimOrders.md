# IUserDimOrders


## Описание


Интерфейс IUserDimOrders содержит
 свойства и методы коллекции атрибутов, по которым производится сортировка
 вычисляемого справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IUserDimOrders.Count.htm)
		 Свойство Count возвращает
		 количество атрибутов, входящих в список сортировки вычисляемого
		 справочника.


		 ![](../../Property_Image.gif)
		 [Item](IUserDimOrders.Item.htm)
		 Свойство Item возвращает
		 настройки порядка сортировки атрибута, входящего в список сортировки
		 вычисляемого справочника.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IUserDimOrders.Add.htm)
		 Метод Add осуществляет
		 добавление атрибута справочника в список сортировки.


		 ![](../../Sub_Image.gif)
		 [Delete](IUserDimOrders.Delete.htm)
		 Метод Delete осуществляет
		 удаление атрибута из списка сортировки вычисляемого справочника.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IUserDimOrders.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск по ключу и возвращает атрибут, входящий в список сортировки.


		 ![](../../Sub_Image.gif)
		 [Move](IUserDimOrders.Move.htm)
		 Метод Move осуществляет
		 перемещение атрибутов в списке сортировки, изменяя тем самым,
		 приоритет сортировки по какому либо атрибуту.
