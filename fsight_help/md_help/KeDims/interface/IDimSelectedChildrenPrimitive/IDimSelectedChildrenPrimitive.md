# IDimSelectedChildrenPrimitive


## Описание


Интерфейс IDimSelectedChildrenPrimitive
 содержит свойства примитива, осуществляющего установку/снятие отметки
 у всех элементов, являющихся дочерними относительно выделенного элемента.


## Свойства


		 Имя свойства
		 Краткое описание

		 [Selection](IDimSelectedChildrenPrimitive.Selection.htm)
		 Свойство Selection
		 определяет отметку элементов примитива.
