# IUserDimIndex.Block


## Синтаксис


Block: [IUserDimBlock](../IUserDimBlock/IUserDimBlock.htm);


## Описание


Свойство Block возвращает объект,
 содержащий блок справочника, в котором создан индекс. Если индекс общий
 для всего справочника - возвращает Null.
