# IUserDimIndex


## Описание


Интерфейс IUserDimIndex содержит
 свойства структуры индекса вычисляемого справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attributes](IUserDimIndex.Attributes.htm)
		 Свойство Attributes
		 возвращает объект, содержащий коллекцию атрибутов, по которым
		 создан индекс.


		 ![](../../Property_Image.gif)
		 [Block](IUserDimIndex.Block.htm)
		 Свойство Block возвращает
		 объект, содержащий блок справочника, в котором создан индекс.


		 ![](../../Property_Image.gif)
		 [Primary](IUserDimIndex.Primary.htm)
		 Свойство Primary определяет,
		 является ли индекс первичным.


		 ![](../../Property_Image.gif)
		 [Unique](IUserDimIndex.Unique.htm)
		 Свойство Unique определяет,
		 является ли индекс уникальным.
