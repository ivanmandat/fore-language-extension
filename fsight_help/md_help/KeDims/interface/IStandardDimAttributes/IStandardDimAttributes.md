# IStandardDimAttributes


## Описание


Интерфейс IStandardDimAttributes
 содержит свойства и методы коллекции атрибутов табличного справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IStandardDimAttributes.Count.htm)
		 Свойство Count возвращает
		 количество атрибутов табличного справочника.


		 ![](../../Property_Image.gif)
		 [Id](IStandardDimAttributes.Id.htm)
		 Свойство Id определяет
		 атрибут табличного справочника, назначение которого - Идентификатор.


		 ![](../../Property_Image.gif)
		 [ImageIndex](IStandardDimAttributes.ImageIndex.htm)
		 Свойство ImageIndex
		 определяет атрибут табличного справочника, назначение которого
		 - Пиктограмма элементов.


		 ![](../../Property_Image.gif)
		 [Item](IStandardDimAttributes.Item.htm)
		 Свойство Item возвращает
		 параметры атрибута табличного справочника.


		 ![](../../Property_Image.gif)
		 [Name](IStandardDimAttributes.Name.htm)
		 Свойство Name определяет
		 атрибут справочника, назначение которого - Наименование.


		 ![](../../Property_Image.gif)
		 [Order](IStandardDimAttributes.Order.htm)
		 Свойство Order определяет
		 атрибут справочника, назначение которого - Порядок.


		 ![](../../Property_Image.gif)
		 [Orders](IStandardDimAttributes.Orders.htm)
		 Свойство Orders возвращает
		 коллекцию атрибутов, по которым будет производиться сортировка
		 табличного справочника.


		 ![](../../Property_Image.gif)
		 [ReadAccess](IStandardDimAttributes.ReadAccess.htm)
		 Свойство ReadAccess
		 определяет атрибут справочника, ограничивающий права на чтение
		 записей.


		 ![](../../Property_Image.gif)
		 [WriteAccess](IStandardDimAttributes.WriteAccess.htm)
		 Свойство WriteAccess
		 определяет атрибут справочника, ограничивающий права на изменение
		 записей.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IStandardDimAttributes.Add.htm)
		 Метод Add осуществляет
		 добавление атрибута табличного справочника.


		 ![](../../Sub_Image.gif)
		 [Clear](IStandardDimAttributes.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех атрибутов табличного справочника.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IStandardDimAttributes.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск по ключу и возвращает объект, содержащий атрибут табличного
		 справочника.


		 ![](../../Sub_Image.gif)
		 [Remove](IStandardDimAttributes.Remove.htm)
		 Метод Remove осуществляет
		 удаление атрибута табличного справочника по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IStandardDimAttributes.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление атрибута табличного справочника по ключу.
