# IDimAttributesList.Add


## Синтаксис


Add(Attribute: [IDimAttribute](../IDimAttribute/IDimAttribute.htm));


## Параметры


Attribute - атрибут справочника,
 который необходимо добавить в коллекцию.


## Описание


Метод Add осуществляет добавление
 указанного атрибута в коллекцию.


## Комментарии


После добавления атрибута в коллекцию его значения будут использовать
 при поиске элементов.
