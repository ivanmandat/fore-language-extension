# IDimAttributesList.RemoveByKey


## Синтаксис


RemoveByKey(Key: UInteger): Boolean;


## Параметры


Key. Ключ атрибута, который
 необходимо удалить.


## Описание


Метод RemoveByKey осуществляет
 удаление атрибута из коллекции по ключу и возвращает признак удачного
 удаления.


## Комментарии


Метод возвращает значение True,
 если удаление прошло успешно.
