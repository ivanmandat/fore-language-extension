# IDimAttributesList.Item


## Синтаксис


Item(Index: Integer): [IDimAttribute](../IDimAttribute/IDimAttribute.htm);


## Параметры


Index. Индекс атрибута в коллекции.


## Описание


Свойство Item возвращает параметры
 атрибута, добавленного в коллекцию.


## Комментарии


Индекс атрибута передается посредством параметра Index.
