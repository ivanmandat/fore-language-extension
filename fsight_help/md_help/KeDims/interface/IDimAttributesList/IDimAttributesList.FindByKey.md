# IDimAttributesList.FindByKey


## Синтаксис


FindByKey(AttributeKey: Integer): [IDimAttribute](../IDimAttribute/IDimAttribute.htm);


## Параметры


Key. Ключ искомого атрибута.


## Описание


Метод FindByKey осуществляет
 поиск атрибута в коллекции по ключу и возвращает его параметры в случае
 удачного поиска.


## Комментарии


Метод возвращает значение Null
 если атрибут не найден.
