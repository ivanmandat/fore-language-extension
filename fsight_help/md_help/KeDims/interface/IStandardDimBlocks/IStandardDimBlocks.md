# IStandardDimBlocks


## Описание


Интерфейс IStandardDimBlocks
 содержит свойства и методы коллекции блоков табличного справочника. В
 коллекции по порядку идут сначала все обычные, затем все рекурсивные блоки
 справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IStandardDimBlocks.Count.htm)
		 Свойство Count возвращает
		 количество блоков табличного справочника.


		 ![](../../Property_Image.gif)
		 [Item](IStandardDimBlocks.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий блок табличного справочника.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IStandardDimBlocks.Add.htm)
		 Метод Add осуществляет
		 добавление блока табличного справочника.


		 ![](../../Sub_Image.gif)
		 [Clear](IStandardDimBlocks.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех блоков табличного справочника.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IStandardDimBlocks.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск и возвращает объект, содержащий блок табличного справочника.


		 ![](../../Sub_Image.gif)
		 [Remove](IStandardDimBlocks.Remove.htm)
		 Метод Remove осуществляет
		 удаление блока табличного справочника по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IStandardDimBlocks.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление блока табличного справочника по ключу.
