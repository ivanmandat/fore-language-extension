# IStandardDimBlocks.FindByKey


## Синтаксис


FindByKey(BlockKey: Integer): [IStandardDimBlock](../IStandardDimBlock/IStandardDimBlock.htm);


## Параметры


BlockKey — ключ
 блока, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает объект, содержащий блок табличного справочника. Поиск
 осуществляется по ключу, передаваемому посредством параметра BlockKey.
