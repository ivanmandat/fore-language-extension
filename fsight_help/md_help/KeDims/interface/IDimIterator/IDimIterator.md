# IDimIterator


## Описание


Интерфейс IDimIterator содержит
 свойства и методы итератора, используемого для навигации по коллекции
 элементов справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DimElementArray](IDimIterator.DimElementArray.htm)
		 Свойство DimElementArray
		 возвращает коллекцию элементов, для которой создан итератор.


		 ![](../../Property_Image.gif)
		 [Element](IDimIterator.Element.htm)
		 Свойство Element возвращает
		 индекс элемента справочника, на котором установлен курсор итератора.


		 ![](../../Property_Image.gif)
		 [Position](IDimIterator.Position.htm)
		 Свойство Position возвращает
		 позицию курсора в итераторе.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [First](IDimIterator.First.htm)
		 Метод First осуществляет
		 переход на начало итератора.


		 ![](../../Sub_Image.gif)
		 [MoveTo](IDimIterator.MoveTo.htm)
		 Метод MoveTo осуществляет
		 установку курсора итератора на заданный элемент.


		 ![](../../Sub_Image.gif)
		 [NewIterator](IDimIterator.NewIterator.htm)
		 Метод NewIterator создает
		 новый итератор со смещением элементов.


		 ![](../../Sub_Image.gif)
		 [Next](IDimIterator.Next.htm)
		 Метод Next осуществляет
		 переход на следующий элемент итератора.
