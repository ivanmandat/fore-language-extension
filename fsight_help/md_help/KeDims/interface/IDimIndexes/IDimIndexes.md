# IDimIndexes


Сборка: Dimensions;


## Описание


Интерфейс IDimIndexes содержит
 свойства и методы коллекции индексов справочника.


## Иерархия наследования


IDimIndexes


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDimIndexes.Count.htm)
		 Свойство Count возвращает
		 количество индексов справочника.


		 ![](../../Property_Image.gif)
		 [Item](IDimIndexes.Item.htm)
		 Свойство Item возвращает
		 структуру индекса справочника.


		 ![](../../Property_Image.gif)
		 [PrimaryIndex](IDimIndexes.PrimaryIndex.htm)
		 Свойство PrimaryIndex
		 возвращает структуру первичного индекса справочника.


		 ![](../../Property_Image.gif)
		 [SelectionIndex](IDimIndexes.SelectionIndex.htm)
		 Свойство SelectionIndex
		 возвращает индекс, который используется для формирования
		 отметки.
