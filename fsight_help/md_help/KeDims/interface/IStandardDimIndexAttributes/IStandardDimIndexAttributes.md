# IStandardDimIndexAttributes


## Описание


Интерфейс IStandardDimIndexAttributes
 содержит свойства и методы коллекции атрибутов индекса справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IStandardDimIndexAttributes.Count.htm)
		 Свойство Count возвращает
		 количество атрибутов, по которым создан индекс табличного справочника.


		 ![](../../Property_Image.gif)
		 [Item](IStandardDimIndexAttributes.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий атрибут, по которому создан индекс.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IStandardDimIndexAttributes.Add.htm)
		 Метод Add осуществляет
		 добавление атрибута справочника в индекс.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IStandardDimIndexAttributes.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск атрибута справочника, по которому создан индекс.


		 ![](../../Sub_Image.gif)
		 [Remove](IStandardDimIndexAttributes.Remove.htm)
		 Метод Remove осуществляет
		 удаление атрибута из индекса справочника.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IStandardDimIndexAttributes.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление атрибута справочника из индекса по ключу.
