# IDimHierarchyInstance.IsGroupingElement


## Синтаксис


IsGroupingElement(Element: Integer): Boolean;


## Параметры


Element. Ключ элемента.


## Описание


Свойство IsGroupingElement возвращает
 признак того, что элемент является группировочным.


## Комментарии


Допустимые значения:


	- True. Элемент является
	 группировочным;


	- False. Элемент не является
	 группировочным.
