# IDimHierarchyInstance


Сборка: Dimensions;


## Описание


Интерфейс IDimHierarchyInstance
 содержит свойства, определяющие данные альтернативной иерархии справочника.


## Иерархия наследования


[INamedEntityModel](ForeSys.chm::/Interface/INamedEntityModel/INamedEntityModel.htm)


IDimHierarchyInstance


## Комментарии


Для обращения к свойствам интерфейса IDimHierarchyInstance
 используйте свойство [IDimHierarchiesInstance.Item](../IDimHierarchiesInstance/IDimHierarchiesInstance.Item.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [GroupingElements](IDimHierarchyInstance.GroupingElements.htm)


		 Свойство GroupingElements
		 возвращает отметку по группировочным элементам справочника.


		 ![](../../Property_Image.gif)
		 [Hierarchy](IDimHierarchyInstance.Hierarchy.htm)


		 Свойство Hierarchy
		 возвращает альтернативную иерархию справочника.


		 ![](../../Property_Image.gif)
		 [IsGroupingElement](IDimHierarchyInstance.IsGroupingElement.htm)


		 Свойство IsGroupingElement
		 возвращает признак того, что элемент является группировочным.


		 ![](../../Property_Image.gif)
		 [Original](IDimHierarchyInstance.Original.htm)


		 Свойство Original возвращает
		 данные основного справочника.


		 ![](../../Property_Image.gif)
		 [Source](IDimHierarchyInstance.Source.htm)


		 Свойство Source определяет
		 данные справочника, являющегося источником альтернативной иерархии.
