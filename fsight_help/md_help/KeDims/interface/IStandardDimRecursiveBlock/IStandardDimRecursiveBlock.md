# IStandardDimRecursiveBlock


## Описание


Интерфейс IStandardDimRecursiveBlock
 содержит свойства рекурсивного блока табличного справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ConnectBy](IStandardDimRecursiveBlock.ConnectBy.htm)
		 Свойство ConnectBy
		 возвращает объект, содержащий условие для атрибута индекса рекурсии.


		 ![](../../Property_Image.gif)
		 [ConnectByIndex](IStandardDimRecursiveBlock.ConnectByIndex.htm)
		 Свойство ConnectByIndex
		 определяет индекс блока справочника, по которому осуществляется
		 связь.


		 ![](../../Property_Image.gif)
		 [ConnectByWhere](IStandardDimRecursiveBlock.ConnectByWhere.htm)
		 Свойство ConnectByWhere
		 возвращает объект, содержащий условие для фильтрации элементов
		 рекурсивных уровней справочника.


		 ![](../../Property_Image.gif)
		 [StartWith](IStandardDimRecursiveBlock.StartWith.htm)
		 Свойство StartWith
		 возвращает объект, содержащий условие для верхнего уровня рекурсии
		 блока справочника.


		 ![](../../Property_Image.gif)
		 [StartWithWhere](IStandardDimRecursiveBlock.StartWithWhere.htm)
		 Свойство StartWithWhere
		 возвращает объект, содержащий условие для фильтрации элементов
		 первого уровня справочника.
