# ICalendarDimAttributes


## Описание


Интерфейс ICalendarDimAttributes
 содержит свойства и методы коллекции пользовательских атрибутов календарного
 справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICalendarDimAttributes.Count.htm)
		 Свойство Count возвращает
		 количество пользовательских атрибутов календарного справочника.


		 ![](../../Property_Image.gif)
		 [Item](ICalendarDimAttributes.Item.htm)
		 Свойство Item возвращает
		 пользовательский атрибут.


		 ![](../../Property_Image.gif)
		 [Name](ICalendarDimAttributes.Name.htm)
		 Свойство Name определяет
		 пользовательский атрибут, который будет использоваться для формирования
		 наименования элементов справочника.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ICalendarDimAttributes.Add.htm)
		 Метод Add осуществляет
		 создание нового пользовательского атрибута.


		 ![](../../Sub_Image.gif)
		 [Clear](ICalendarDimAttributes.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех пользовательских атрибутов.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ICalendarDimAttributes.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск пользовательского атрибута по ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](ICalendarDimAttributes.Remove.htm)
		 Метод Remove осуществляет
		 удаление пользовательского атрибута по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](ICalendarDimAttributes.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление пользовательского атрибута по ключу.
