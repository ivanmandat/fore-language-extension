# IDimDefaultElementSelectionPrimitive


Сборка: Dimensions;


## Описание


Интерфейс IDimDefaultElementSelectionPrimitive
 содержит свойства примитива, устанавливающего отметку на элемент по умолчанию.


## Иерархия наследования


IDimDefaultElementSelectionPrimitive

## Комментарии


Для установки элемента по умолчанию используйте свойство [IRdsDictionary.DefaultElementKey](KeRds.chm::/Interface/IRdsDictionary/IRdsDictionary.DefaultElementKey.htm).
