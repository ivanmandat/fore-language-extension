# ICalendarDimInstance


## Описание


Интерфейс ICalendarDimInstance
 содержит методы для работы с иерархией элементов в календарном справочнике.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Day](ICalendarDimInstance.Day.htm)
		 Метод Day возвращает
		 индекс дня для элемента календарного справочника.


		 ![](../../Sub_Image.gif)
		 [DownLevel](ICalendarDimInstance.DownLevel.htm)
		 Метод DownLevel возвращает
		 индекс элемента календарного справочника, находящегося на уровень
		 ниже данного элемента.


		 ![](../../Sub_Image.gif)
		 [FinishPeriod](ICalendarDimInstance.FinishPeriod.htm)
		 Метод FinishPeriod
		 возвращает индекс элемента, соответствующего окончанию периода,
		 в котором находится данный элемент.


		 ![](../../Sub_Image.gif)
		 [HalfYear](ICalendarDimInstance.HalfYear.htm)
		 Метод HalfYear возвращает
		 индекс полугодия для элемента календарного справочника.


		 ![](../../Sub_Image.gif)
		 [Level](ICalendarDimInstance.Level.htm)
		 Метод Level возвращает
		 уровень справочника, на котором расположен указанный элемент.


		 ![](../../Sub_Image.gif)
		 [Month](ICalendarDimInstance.Month.htm)
		 Метод Month возвращает
		 индекс месяца для элемента календарного справочника.


		 ![](../../Sub_Image.gif)
		 [NextPeriod](ICalendarDimInstance.NextPeriod.htm)
		 Метод NextPeriod возвращает
		 индекс элемента соответствующего следующему периоду, относительно
		 периода, в котором находится указанный элемент.


		 ![](../../Sub_Image.gif)
		 [NineMonths](ICalendarDimInstance.NineMonths.htm)
		 Метод NineMonths возвращает
		 индекс элемента, соответствующего периоду "9 месяцев",
		 в котором находится указанный элемент.


		 ![](../../Sub_Image.gif)
		 [PeriodLength](ICalendarDimInstance.PeriodLength.htm)
		 Метод PeriodLength
		 возвращает длительность периода, в котором находится данный элемент.


		 ![](../../Sub_Image.gif)
		 [PrevPeriod](ICalendarDimInstance.PrevPeriod.htm)
		 Метод PrevPeriod возвращает
		 индекс элемента соответствующего предыдущему периоду, относительно
		 периода, в котором находится указанный элемент.


		 ![](../../Sub_Image.gif)
		 [Quarter](ICalendarDimInstance.Quarter.htm)
		 Метод Quarter возвращает
		 индекс квартала для элемента календарного справочника.


		 ![](../../Sub_Image.gif)
		 [Shift](ICalendarDimInstance.Shift.htm)
		 Метод Shift возвращает
		 индекс элемента, полученного путем смещения относительно заданного
		 элемента на заданный лаг (смещение).


		 ![](../../Sub_Image.gif)
		 [StartPeriod](ICalendarDimInstance.StartPeriod.htm)
		 Метод StartPeriod возвращает
		 индекс элемента, соответствующего началу периода, в котором находится
		 данный элемент.


		 ![](../../Sub_Image.gif)
		 [ToLevel](ICalendarDimInstance.ToLevel.htm)
		 Метод ToLevel возвращает
		 индекс родительского элемента, расположенного на заданном уровне,
		 относительно которого данный элемент является дочерним.


		 ![](../../Sub_Image.gif)
		 [UpLevel](ICalendarDimInstance.UpLevel.htm)
		 Метод UpLevel возвращает
		 индекс элемента календарного справочника, находящегося на уровень
		 выше данного элемента.


		 ![](../../Sub_Image.gif)
		 [Week](ICalendarDimInstance.Week.htm)
		 Метод Week возвращает
		 индекс недели для элемента календарного справочника.


		 ![](../../Sub_Image.gif)
		 [Year](ICalendarDimInstance.Year.htm)
		 Метод Year возвращает
		 индекс года для элемента календарного справочника.
