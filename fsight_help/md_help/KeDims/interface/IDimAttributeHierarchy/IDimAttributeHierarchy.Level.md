# IDimAttributeHierarchy.Level


## Синтаксис


Level: Integer;


## Описание


Свойство Level определяет уровень
 атрибута-оригинала, на уникальных значениях которого строится альтернативная
 иерархия.


## Комментарии


Свойство актуально, если в свойстве [IDimAttributeHierarchy.BaseAttribute](IDimAttributeHierarchy.BaseAttribute.htm)
 задан атрибут-оригинал.
