# IDimAttributeHierarchy.AttributeIndex


## Синтаксис

## Описание


Свойство AttributeIndex
 возвращает индекс атрибута альтернативной иерархии.


## Комментарии


Свойство актуально, если в атрибуте присутствует альтернативная иерархия.
