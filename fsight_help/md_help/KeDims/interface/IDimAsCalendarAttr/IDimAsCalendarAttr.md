# IDimAsCalendarAttr


## Описание


Интерфейс IDimAsCalendarAttr
 содержит свойства атрибута справочника для сопоставления, при использовании
 справочника в качестве календарного.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attr](IDimAsCalendarAttr.Attr.htm)
		 Свойство Attr возвращает
		 атрибут справочника, который сопоставлен с атрибутом календаря.


		 ![](../../Property_Image.gif)
		 [Id](IDimAsCalendarAttr.Id.htm)
		 Свойство Id возвращает
		 идентификатор атрибута календаря, с которым сопоставлен атрибут
		 справочника.
