# IDimTreeIterator.GetParent


## Синтаксис


GetParent: Integer;


## Описание


Метод GetParent возвращает индекс
 владельца элемента справочника, на котором установлен курсор итератора.


## Комментарии


Для получения индекса элемента справочника, на котором установлен курсор
 итератора, используйте метод [IDimTreeIterator.GetElement](IDimTreeIterator.GetElement.htm).
