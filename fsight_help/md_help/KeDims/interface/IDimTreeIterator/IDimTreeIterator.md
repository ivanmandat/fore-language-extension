# IDimTreeIterator


Сборка: Dimensions;


## Описание


Интерфейс IDimTreeIterator содержит
 методы итератора, используемого для навигации по дереву отсортированных
 элементов справочника.


## Иерархия наследования


IDimTreeIterator


## Комментарии


Для получения итератора на дерево, построенного с учетом пользовательской
 сортировки, используйте метод [IDimInstance.ElementsByOrder](../IDimInstance/IDimInstance.ElementsByOrder.htm).
