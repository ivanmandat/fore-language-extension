# IDimTreeIterator.HasNext


## Синтаксис


HasNext: Integer;


## Описание


Метод HasNext осуществляет переход
 на следующий элемент итератора.


## Комментарии


Также метод возвращает уровень элемента справочника, на котором установлен
 курсор итератора.
