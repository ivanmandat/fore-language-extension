# IDimAsCalendarLevels


## Описание


Интерфейс IDimAsCalendarLevels
 содержит свойства и методы коллекции уровней для сопоставления при использовании
 справочника в качестве календарного.


## Комментарии


Коллекцию уровней можно получить, используя свойство [IDimAsCalendar.Levels](../IDimAsCalendar/IDimAsCalendar.Levels.htm).
 Коллекция содержит список всех уровней календаря. Уровни, которые будут
 использоваться, необходимо сопоставить с уровнями текущего справочника.
 Для сопоставления используйте свойство [KeyLevel](IDimAsCalendarLevels.KeyLevel.htm)
 или [LevelKey](IDimAsCalendarLevels.LevelKey.htm).
