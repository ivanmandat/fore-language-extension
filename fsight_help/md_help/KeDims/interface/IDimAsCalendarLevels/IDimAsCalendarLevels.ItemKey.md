# IDimAsCalendarLevels.ItemKey


## Синтаксис


ItemKey(Index: Integer): Variant;


## Параметры


Index. Индекс уровня календаря.


## Описание


Свойство ItemKey возвращает
 ключ уровня справочника, который сопоставлен с уровнем календаря с указанным
 индексом.
