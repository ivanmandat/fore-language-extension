# IDimAsCalendarLevels.KeyLevel


## Синтаксис


KeyLevel(Key: Variant): [DimCalendarLevel](../../Enums/DimCalendarLevel.htm);


## Параметры


Key.
 Ключ уровня справочника.


## Описание


Свойство KeyLevel сопоставляет
 уровень справочника с указанным уровнем календаря.


## Комментарии


В качестве значения параметра Key
 указывается значение свойства Key у [уровня](../IDimLevel/IDimLevel.htm)
 справочника.
