# IDynamicInstance.PresetChildRange


## Синтаксис


PresetChildRange(Parent: Integer; ChildCount: Integer):
 Boolean;


## Параметры


Parent. Индекс элемента-родителя;


ChildCount. Количество дочерних
 элементов.


## Описание


Метод PresetChildRange устанавливает
 начальное количество дочерних элементов указанного элемента справочника.


## Пример


Метод предназначен для внутреннего использования.
