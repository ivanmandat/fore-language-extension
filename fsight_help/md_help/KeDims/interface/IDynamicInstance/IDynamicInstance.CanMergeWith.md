# IDynamicInstance.CanMergeWith


## Синтаксис


CanMergeWith(Other: [IDynamicInstance](IDynamicInstance.htm)):
 Boolean;


## Параметры


Other. Справочник с которым
 нужно объединить основной справочник.


## Описание


Метод CanMergeWith определяет,
 можно ли объединить справочники.


## Комментарии


Метод возвращает true, если объединить
 справочники можно, и false, если
 нельзя.
