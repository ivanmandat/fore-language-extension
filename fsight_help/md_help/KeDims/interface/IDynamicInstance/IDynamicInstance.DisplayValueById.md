# IDynamicInstance.DisplayValueById


## Синтаксис


DisplayValueById(Element: Integer; AttributeId:
 String): Variant;


## Параметры


Element. Индекс элемента справочника;


AttributeId. Идентификатор
 атрибута.


## Описание


Свойство DisplayValueById определяет
 отображаемое значение атрибута элемента справочника по идентификатору
 атрибута.
