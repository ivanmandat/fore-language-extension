# IDynamicInstance.HasLoadedChildren


## Синтаксис


HasLoadedChildren(Element: Integer): Boolean;


## Параметры


Element. Индекс элемента справочника.


## Описание


Метод HasLoadedChildren возвращает
 признак наличия загруженных дочерних элементов указанного элемента справочника.


## Комментарии


Допустимые значения:


	- True. Указанный элемент
	 справочника имеет загруженные дочерние элементы;


	- False. Указанный элемент
	 справочника не имеет загруженных дочерних элементов.
