# IDynamicInstance.DisplayValueByKey


## Синтаксис


DisplayValueByKey(Element: Integer; AttributeKey:
 Integer): Variant;


## Параметры


Element. Индекс элемента справочника;


AttributeKey. Ключ атрибута.


## Описание


Свойство DisplayValueByKey определяет
 отображаемое значение атрибута элемента справочника по ключу атрибута.
