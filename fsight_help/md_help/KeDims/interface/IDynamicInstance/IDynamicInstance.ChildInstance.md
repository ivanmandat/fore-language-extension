# IDynamicInstance.ChildInstance


## Синтаксис


ChildInstance(idx: Integer): [IDynamicInstance](IDynamicInstance.htm);


## Параметры


idx. Индекс дочернего справочника.


## Описание


Свойство ChildInstance возвращает
 дочерний справочник по его индексу.


## Комментарии


Дочерний справочник - справочник, скопированный с помощью метода [IDynamicInstance.Clone](IDynamicInstance.Clone.htm).
