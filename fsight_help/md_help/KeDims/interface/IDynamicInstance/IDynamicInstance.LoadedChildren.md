# IDynamicInstance.LoadedChildren


## Синтаксис


LoadedChildren(Parent: Integer): Array;


## Параметры


Parent. Индекс элемента справочника.


## Описание


Свойство LoadedChildren возвращает
 массив загруженных дочерних элементов указанного элемента справочника.
