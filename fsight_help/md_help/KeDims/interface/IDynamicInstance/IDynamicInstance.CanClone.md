# IDynamicInstance.CanClone


## Синтаксис


CanClone(op: [DynamicInstanceOperationsVisibility](../../Enums/DynamicInstanceOperationsVisibility.htm);
 Mark: Variant): Boolean;


## Параметры


op. Параметр определяет, на
 какие справочники распространяется выполняемая операция;


Mark. Множество, которому будет
 принадлежать справочник.


## Описание


Метод CanClone определяет, можно
 ли создать копию справочника.


## Комментарии


Метод возвращает true, если
 создать копию справочника можно, и false,
 если нельзя.
