# IDynamicInstance.PreSetMark


## Синтаксис


PreSetMark: Variant;


## Описание


Свойство PreSetMark возвращает
 множество, которому принадлежит справочник.


## Комментарии


Определить принадлежность ко множеству справочников можно при создании
 справочника. Операции, производимые над одним справочником множества распространяются
 на другие справочники этого множества.
