# IDynamicInstance.HasLeavesAllowedToUnload


## Синтаксис


HasLeavesAllowedToUnload(Element: Integer): Boolean;


## Параметры


Element. Индекс элемента справочника.


## Описание


Метод HasLeavesAllowedToUnload возвращает
 признак наличия незагруженных дочерних элементов указанного элемента справочника.


## Комментарии


Допустимые значения:


	- True. Указанный элемент
	 справочника имеет незагруженные дочерние элементы;


	- False. Указанный элемент
	 справочника не имеет незагруженных дочерних элементов.
