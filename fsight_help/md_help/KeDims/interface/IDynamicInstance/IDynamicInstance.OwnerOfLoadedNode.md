# IDynamicInstance.OwnerOfLoadedNode


## Синтаксис


OwnerOfLoadedNode(Element: Integer): Integer;


## Параметры


Element. Индекс элемента справочника.


## Описание


Свойство OwnerOfLoadedNode возвращает
 индекс элемента, являющегося родителем по отношению к указанному загруженному
 элементу.


## Комментарии


Свойство возвращает «-1» если
 указанный элемент - корневой.
