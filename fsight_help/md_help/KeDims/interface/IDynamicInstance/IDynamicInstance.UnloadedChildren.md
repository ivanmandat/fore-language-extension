# IDynamicInstance.UnloadedChildren


## Синтаксис


UnloadedChildren(Parent: Integer): Array;


## Параметры


Parent. Индекс элемента справочника.


## Описание


Свойство UnloadedChildren возвращает
 массив незагруженных дочерних элементов указанного элемента справочника.
