# IDynamicInstance.ChildrenInstancesCount


## Синтаксис


ChildrenInstancesCount: Integer;


## Описание


Свойство ChildrenInstancesCount
 возвращает количество дочерних справочников.


## Комментарии


Дочерний справочник - справочник, скопированный с помощью метода [IDynamicInstance.Clone](IDynamicInstance.Clone.htm).
