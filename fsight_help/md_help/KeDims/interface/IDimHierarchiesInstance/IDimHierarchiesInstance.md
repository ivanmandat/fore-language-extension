# IDimHierarchiesInstance


Сборка: Dimensions;


## Описание


Интерфейс IDimHierarchiesInstance
 содержит свойства и методы для работы с коллекцией данных альтернативных
 иерархий справочника.


## Иерархия наследования


IDimSelection


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDimHierarchiesInstance.Count.htm)
		 Свойство Count возвращает
		 количество иерархий в коллекции.


		 ![](../../Property_Image.gif)
		 [DefaultHierarchy](IDimHierarchiesInstance.DefaultHierarchy.htm)
		 Свойство DefaultHierarchy
		 определяет альтернативную иерархию, которая будет использоваться
		 по умолчанию при открытии справочника.


		 ![](../../Property_Image.gif)
		 [Hierarchies](IDimHierarchiesInstance.Hierarchies.htm)
		 Свойство Hierarchies
		 возвращает объект для работы с коллекцией альтернативных иерархий
		 справочника.


		 ![](../../Property_Image.gif)
		 [Item](IDimHierarchiesInstance.Item.htm)
		 Свойство Item возвращает
		 данные альтернативной иерархии из коллекции по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindByKey](IDimHierarchiesInstance.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск данных альтернативной иерархии в коллекции по ключу.
