# IDimHierarchiesInstance.FindByKey


## Синтаксис


FindByKey(Value: Integer): [IDimHierarchyInstance](../IDimHierarchyInstance/IDimHierarchyInstance.htm);


## Параметры


Value - ключ альтернативной
 иерархии.


## Описание


Метод FindByKey осуществляет
 поиск данных альтернативной иерархии в коллекции по ключу.


## Комментарии


Для получения количества иерархий в коллекции используйте свойство [IDimHierarchiesInstance.Count](IDimHierarchiesInstance.Count.htm).
