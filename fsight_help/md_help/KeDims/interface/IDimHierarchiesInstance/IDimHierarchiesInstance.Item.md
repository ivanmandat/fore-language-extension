# IDimHierarchiesInstance.Item


## Синтаксис


Item(Index: Integer): [IDimHierarchyInstance](../IDimHierarchyInstance/IDimHierarchyInstance.htm);


## Параметры


Index - индекс альтернативной
 иерархии в коллекции.


## Описание


Свойство доступно только для чтения.


Свойство Item возвращает данные
 альтернативной иерархии из коллекции по индексу.


## Комментарии


Для получения количества иерархий в коллекции используйте свойство [IDimHierarchiesInstance.Count](IDimHierarchiesInstance.Count.htm).
