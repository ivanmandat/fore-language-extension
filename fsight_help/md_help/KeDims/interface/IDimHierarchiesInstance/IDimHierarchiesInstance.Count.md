# IDimHierarchiesInstance.Count


## Синтаксис


Count: Integer;


## Описание


Свойство доступно только для чтения.


Свойство Count возвращает количество
 иерархий в коллекции.


## Комментарии


Для получения данных альтернативной иерархии из коллекции по индексу
 используйте свойство [IDimHierarchiesInstance.Item](IDimHierarchiesInstance.Item.htm).
