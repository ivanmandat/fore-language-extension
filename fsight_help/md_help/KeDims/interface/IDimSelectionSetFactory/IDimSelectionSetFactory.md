# IDimSelectionSetFactory


## Описание


Интерфейс IDimSelectionSetFactory
 содержит методы для создания пустой коллекции отметок справочников.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CheckDimensionsIntegrity](IDimSelectionSetFactory.CheckDimensionsIntegrity.htm)
		 Метод CheckDimensionsIntegrity
		 осуществляет проверку построения всех справочников, входящих в
		 указанную комплексную отметку.


		 ![](../../Sub_Image.gif)
		 [CreateDimSelectionSet](IDimSelectionSetFactory.CreateDimSelectionSet.htm)
		 Метод CreateDimSelectionSet
		 создает коллекцию отметок справочников.
