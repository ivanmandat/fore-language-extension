# IDimAttributes.FindByKey


## Синтаксис


FindByKey(AttributeKey: Integer): [IDimAttribute](../IDimAttribute/IDimAttribute.htm);


## Параметры


AttributeKey - ключ
 атрибута, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск атрибута справочника по ключу, и возвращает его параметры в случае
 удачного поиска. Если атрибут не найден, то метод возвращает значение
 Null.
