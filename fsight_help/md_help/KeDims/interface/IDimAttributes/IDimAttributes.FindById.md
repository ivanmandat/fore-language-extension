# IDimAttributes.FindById


## Синтаксис


FindById(AttributeId: String): [IDimAttribute](../IDimAttribute/IDimAttribute.htm);


## Параметры


AttributeId - идентификатор
 атрибута, по которому осуществляется поиск.


## Описание


Метод FindById осуществляет
 поиск атрибута справочника по идентификатору, и возвращает его параметры
 в случае удачного поиска. Если атрибут не найден, то метод возвращает
 значение Null.
