# IDimAttributes


## Описание


Интерфейс IDimAttributes содержит
 свойства и методы для работы с коллекцией атрибутов справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDimAttributes.Count.htm)
		 Свойство Count возвращает
		 количество атрибутов справочника.


		 ![](../../Property_Image.gif)
		 [Id](IDimAttributes.Id.htm)
		 Свойство Id возвращает
		 атрибут справочника, назначение которого - Идентификатор.


		 ![](../../Property_Image.gif)
		 [ImageIndex](IDimAttributes.ImageIndex.htm)
		 Свойство ImageIndex
		 определяет атрибут справочника, который будет использован в качестве
		 пиктограммы элементов.


		 ![](../../Property_Image.gif)
		 [Item](IDimAttributes.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий атрибут справочника. Индекс атрибута передается
		 посредством параметра Index.


		 ![](../../Property_Image.gif)
		 [Name](IDimAttributes.Name.htm)
		 Свойство Name возвращает
		 атрибут справочника, назначение которого - Наименование.


		 ![](../../Property_Image.gif)
		 [Order](IDimAttributes.Order.htm)
		 Свойство Order возвращает
		 атрибут справочника, назначение которого - Порядок.


		 ![](../../Property_Image.gif)
		 [Orders](IDimAttributes.Orders.htm)
		 Свойство Orders возвращает
		 коллекцию атрибутов, по которым будет производиться сортировка
		 справочника.


		 ![](../../Property_Image.gif)
		 [ReadAccess](IDimAttributes.ReadAccess.htm)
		 Свойство ReadAccess
		 возвращает объект, содержащий атрибут справочника, ограничивающий
		 права на чтение записей.


		 ![](../../Property_Image.gif)
		 [WriteAccess](IDimAttributes.WriteAccess.htm)
		 Свойство WriteAccess
		 возвращает объект, содержащий атрибут справочника, ограничивающий
		 права на изменение записей.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](IDimAttributes.FindById.htm)
		 Метод FindById осуществляет
		 поиск атрибута справочника по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IDimAttributes.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск атрибута справочника по ключу.
