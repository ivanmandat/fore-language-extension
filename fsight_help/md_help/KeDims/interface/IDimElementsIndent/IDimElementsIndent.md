# IDimElementsIndent


## Описание


Интерфейс IDimElementsIndent
 содержит свойства и методы для получения информации о формировании составного
 измерения.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [IndentHigh](IDimElementsIndent.IndentHigh.htm)
		 Свойство IndentHigh
		 возвращает количество измерений, элементы которых использованы
		 при формировании наименования указанного элемента составного измерения.


		 ![](../../Property_Image.gif)
		 [IndentLow](IDimElementsIndent.IndentLow.htm)
		 Свойство IndentLow
		 возвращает индекс измерения, элемент которого был использован
		 для формирования наименования указанного элемента составного измерения.


		 ![](../../Property_Image.gif)
		 [IndentName](IDimElementsIndent.IndentName.htm)
		 Свойство IndentName
		 возвращает наименование элемента из измерения-источника, использованного
		 при формировании наименования указанного элемента составного измерения.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [GetIndentInfo](IDimElementsIndent.GetIndentInfo.htm)
		 Метод GetIndentInfo
		 возвращает информацию об элементе измерения-источника, который
		 был использован при формировании наименования указанного элемента
		 составного измерения.
