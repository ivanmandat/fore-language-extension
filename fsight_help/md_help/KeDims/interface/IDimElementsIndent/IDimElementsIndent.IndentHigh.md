# IDimElementsIndent.IndentHigh


## Синтаксис


IndentHigh(Element: Integer): Integer;


## Параметры


Element - элемент составного
 измерения.


## Описание


Свойство IndentHigh возвращает
 количество измерений, элементы которых использованы при формировании наименования
 элемента Element составного измерения.
