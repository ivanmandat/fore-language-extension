# ICustomDimTreeChildren


## Описание


Интерфейс ICustomDimTreeChildren
 содержит свойства для работы со списком элементов.


## Комментарии


Списки элементов возвращают свойства [ICustomDimTree.Children](../ICustomDimTree/ICustomDimTree.Children.htm)
 и [ICustomDimTree.RootChildren](../ICustomDimTree/ICustomDimTree.RootChildren.htm).


## Свойства

		 количество элементов в списке.


		 ![](../../Property_Image.gif)
		 [Item](ICustomDimTreeChildren.Item.htm)
		 Свойство Item возвращает
		 первичный ключ для элемента с указанными индексом.
