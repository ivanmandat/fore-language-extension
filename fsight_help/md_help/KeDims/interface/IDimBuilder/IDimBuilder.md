# IDimBuilder


## Описание


Интерфейс IDimBuilder содержит
 методы для подготовки вычисляемого справочника к загрузке данных.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CreateBlockLoader](IDimBuilder.CreateBlockLoader.htm)


		 Метод CreateBlockLoader
		 создает загрузчик элементов нижних уровней.


		 ![](../../Sub_Image.gif)
		 [CreateRootBlockLoader](IDimBuilder.CreateRootBlockLoader.htm)


		 Метод CreateRootBlockLoader
		 создает загрузчик элементов верхнего уровня.


		 ![](../../Sub_Image.gif)


		 [SetAttributeInstanceValuesLoader](IDimBuilder.SetAttributeInstanceValuesLoader.htm)


		 Метод SetAttributeInstanceValuesLoader
		 устанавливает загрузчик значений дополнительных атрибутов.
