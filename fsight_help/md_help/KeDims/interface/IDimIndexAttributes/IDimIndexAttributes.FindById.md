# IDimIndexAttributes.FindById


## Синтаксис


FindById(AttributeId: String): [IDimAttribute](../IDimAttribute/IDimAttribute.htm);


## Параметры


AttributeId. Идентификатор
 атрибута, по которому осуществляется поиск.


## Описание


Метод FindById осуществляет
 поиск атрибута в индексе по идентификатору.


## Комментарии


Метод возвращает параметры атрибута в случае удачного поиска и Null, если атрибут с указанным идентификатором
 не включён в индекс.
