# IDimIndexAttributes.Item


## Синтаксис


Item(Index: Variant): [IDimAttribute](../IDimAttribute/IDimAttribute.htm);


## Параметры


Index. Индекс атрибута.


## Описание


Свойство Item возвращает параметры
 атрибута, который включен в индекс.


## Комментарии


Индексация атрибутов сквозная и начинается с нуля. Для получения количества
 атрибутов в индексе используйте свойство [IDimIndexAttributes.Count](IDimIndexAttributes.Count.htm).
