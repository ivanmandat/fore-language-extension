# IDimIndexAttributes


Сборка: Dimensions;


## Описание


Интерфейс IDimIndexAttributes
 содержит свойства и методы коллекции атрибутов индекса справочника.


## Комментарии


Коллекцию атрибутов индекса можно получить в свойстве [IDimIndex.Attributes](../IDimIndex/IDimIndex.Attributes.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDimIndexAttributes.Count.htm)
		 Свойство Count возвращает
		 количество атрибутов, по которым создан индекс справочника.


		 ![](../../Property_Image.gif)
		 [Item](IDimIndexAttributes.Item.htm)
		 Свойство Item возвращает
		 параметры атрибута, который включен в индекс.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](IDimIndexAttributes.FindById.htm)
		 Метод FindById осуществляет
		 поиск атрибута в индексе по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IDimIndexAttributes.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск атрибута в индексе по ключу.
