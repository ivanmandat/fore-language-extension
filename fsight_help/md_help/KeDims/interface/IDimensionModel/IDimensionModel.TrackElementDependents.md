# IDimensionModel.TrackElementDependents


## Синтаксис


TrackElementDependents: Boolean;


## Описание


Свойство TrackElementDependents
 возвращает признак отслеживания ссылок на элементы текущего справочника.


## Комментарии


Если для справочника включено отслеживание ссылок на элементы, то свойство
 возвращает значение True. Для
 включения возможности отслеживания ссылок используйте свойство [ElementDependenciesTrackingType](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.ElementDependenciesTrackingType.htm),
 либо [TrackElementDependents](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.TrackElementDependents.htm).
