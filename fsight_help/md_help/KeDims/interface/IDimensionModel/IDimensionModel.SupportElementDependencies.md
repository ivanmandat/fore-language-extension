# IDimensionModel.SupportElementDependencies


## Синтаксис


SupportElementDependencies: Boolean;


## Описание


Свойство SupportElementDependencies
 возвращает признак возможности отслеживать ссылки на элементы текущего
 справочника.


## Комментарии


Данное свойство возвращает значение True
 для справочников НСИ и составных справочников НСИ. Для всех других видов
 справочников свойство возвращает значение False.
