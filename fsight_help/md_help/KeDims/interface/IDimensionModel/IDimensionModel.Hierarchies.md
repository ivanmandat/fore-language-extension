# IDimensionModel.Hierarchies


## Синтаксис


Hierarchies: [IDimHierarchies](../IDimHierarchies/IDimHierarchies.htm);


## Описание


Свойство Hierarchies возвращает
 коллекцию альтернативных иерархий справочника.


## Комментарии


Для получения объекта, содержащего коллекцию атрибутов справочника,
 используйте свойство [IDimensionModel.Attributes](IDimensionModel.Attributes.htm).
