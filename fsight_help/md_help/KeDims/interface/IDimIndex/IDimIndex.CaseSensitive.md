# IDimIndex.CaseSensitive


## Синтаксис


CaseSensitive: Boolean;


## Описание


Свойство CaseSensitive
 возвращает признак того, чувствителен ли индекс к регистру.


## Комментарии


Свойство используется, если индекс содержится в справочнике, используемом
 в качестве измерения.


Допустимые значения:


	- True. Индекс чувствителен
	 к регистру;


	- False. Индекс регистронезависимый.
