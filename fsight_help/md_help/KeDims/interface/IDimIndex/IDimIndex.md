# IDimIndex


Сборка: Dimensions;


## Описание


Интерфейс IDimIndex содержит
 свойства структуры индекса справочника.


## Иерархия наследования


[INamedEntityModel](ForeSys.chm::/Interface/INamedEntityModel/INamedEntityModel.htm)


IDimIndex


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attributes](IDimIndex.Attributes.htm)
		 Свойство Attributes
		 возвращает коллекцию атрибутов, по которым создан индекс.


		 ![](../../Property_Image.gif)
		 [Block](IDimIndex.Block.htm)
		 Свойство Block возвращает
		 объект, содержащий блок справочника, в котором создан индекс.


		 ![](../../Property_Image.gif)
		 [CaseSensitive](IDimIndex.CaseSensitive.htm)
		 Свойство CaseSensitive
		 возвращает признак того, чувствителен ли индекс к регистру.


		 ![](../../Property_Image.gif)
		 [Primary](IDimIndex.Primary.htm)
		 Свойство Primary возвращает
		 True, если индекс первичный.


		 ![](../../Property_Image.gif)
		 [Unique](IDimIndex.Unique.htm)
		 Свойство Unique возвращает
		 True, если индекс уникальный.


		 ![](../../Property_Image.gif)
		 [UseInSelection](IDimIndex.UseInSelection.htm)
		 Свойство UseInSelection
		 возвращает признак того, используется ли индекс при формировании
		 отметки.
