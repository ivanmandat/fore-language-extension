# IDimMacroSelectionPrimitive.ModuleMacro


## Синтаксис


ModuleMacro:
 String;


## Описание


Свойство ModuleMacro определяет
 привязку к макросу.


## Комментарии


Привязка к макросу указывается в следующем формате: <Наименование
 класса>.<Наименование процедуры>;


Используется вместе со свойством [IDimMacroSelectionPrimitive.Module](IDimMacroSelectionPrimitive.Module.htm).
