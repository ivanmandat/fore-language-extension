# ICompoundCalendarInstance


## Описание


Интерфейс ICompoundCalendarInstance
 предназначен для работы с экземпляром составного календарного справочника.


## Свойства, унаследованные от [ICompoundDimensionInstance](../ICompoundDimensionInstance/ICompoundDimensionInstance.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Selection](../ICompoundDimensionInstance/ICompoundDimensionInstance.Selection.htm)
		 Свойство Selection
		 возвращает отметку коллекции справочников, по которым построен
		 составной справочник.


		 ![](../../Property_Image.gif)
		 [Type](../ICompoundDimensionInstance/ICompoundDimensionInstance.Type.htm)
		 Свойство Type возвращает
		 тип составного измерения.


## Методы, унаследованные от [ICompoundDimensionInstance](../ICompoundDimensionInstance/ICompoundDimensionInstance.htm)


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ComposeElement](../ICompoundDimensionInstance/ICompoundDimensionInstance.ComposeElement.htm)
		 Метод ComposeElement
		 возвращает индекс элемента составного справочника по его источникам.


		 ![](../../Sub_Image.gif)
		 [DecomposeElement](../ICompoundDimensionInstance/ICompoundDimensionInstance.DecomposeElement.htm)
		 Метод DecomposeElement
		 раскладывает элемент составного справочника на источники.
