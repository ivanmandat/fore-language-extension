# IDimSelectionSet2


Сборка: Dimensions;


## Описание


Интерфейс IDimSelectionSet2
 содержит свойства для расширенной настройки параметров работы с коллекцией
 отметок справочников.


## Иерархия наследования


IDimSelectionSet2


## Комментарии


Объект, поддерживающий интерфейс [IDimSelectionSet](../IDimSelectionSet/IDimSelectionSet.htm),
 также может быть приведен к интерфейсу IDimSelectionSet2
 для задания расширенных настроек работы с отметками.
