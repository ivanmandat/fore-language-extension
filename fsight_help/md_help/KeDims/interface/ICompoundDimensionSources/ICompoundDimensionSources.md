# ICompoundDimensionSources


## Описание


Интерфейс ICompoundDimensionSources
 содержит свойства и методы для работы с коллекцией источников составного
 справочника.


## Комментарии


Коллекцию источников возвращает свойство [ICompoundDimension.Sources](../ICompoundDimension/ICompoundDimension.Sources.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICompoundDimensionSources.Count.htm)


		 Свойство Count возвращает
		 количество источников составного справочника.


		 [![](../../Property_Image.gif)](ICompoundDimensionSources.ForcedHideSourceName.htm)
		 [ForcedHideSourceName](ICompoundDimensionSources.ForcedHideSourceName.htm)


		 Свойство ForcedHideSourceName
		 управляет отображением названия составного измерения у нескольких
		 источников данных.


		 ![](../../Property_Image.gif)
		 [Item](ICompoundDimensionSources.Item.htm)


		 Свойство Item возвращает
		 источник из коллекции по индексу.


		 ![](../../Property_Image.gif)
		 [ShowSourceName](ICompoundDimensionSources.ShowSourceName.htm)


		 Свойство ShowSourceName
		 определяет признак отображения наименований источников в наименованиях
		 элементов составного справочника.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ICompoundDimensionSources.Add.htm)


		 Метод Add осуществляет
		 добавление источника в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](ICompoundDimensionSources.Clear.htm)


		 Метод Clear осуществляет
		 удаление всех источников из коллекции.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ICompoundDimensionSources.FindByKey.htm)


		 Метод FindByKey осуществляет
		 поиск по ключу и возвращает объект, содержащий источник составного
		 справочника.


		 ![](../../Sub_Image.gif)
		 [Remove](ICompoundDimensionSources.Remove.htm)


		 Метод Remove осуществляет
		 удаление источника составного справочника по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](ICompoundDimensionSources.RemoveByKey.htm)


		 Метод RemoveByKey осуществляет
		 удаление источника составного справочника по ключу.
