# ICompoundDimensionSources.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index -
 индекс источника.


## Описание


Метод Remove осуществляет удаление
 источника составного справочника по индексу. Возвращает значение True в случае удачного удаления.
