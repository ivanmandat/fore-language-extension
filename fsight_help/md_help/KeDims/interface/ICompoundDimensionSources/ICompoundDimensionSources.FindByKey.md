# ICompoundDimensionSources.FindByKey


## Синтаксис


FindByKey(Key: Integer): [ICompoundDimensionSource](../ICompoundDimensionSource/ICompoundDimensionSource.htm);


## Параметры


Key - ключ искомого источника.


## Описание


Метод FindByKey осуществляет
 поиск по ключу и возвращает объект, содержащий источник составного справочника.
