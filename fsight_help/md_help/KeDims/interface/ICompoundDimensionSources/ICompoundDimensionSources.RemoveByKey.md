# ICompoundDimensionSources.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key -
 ключ источника.


## Описание


Метод RemoveByKey осуществляет
 удаление источника составного справочника по ключу. Возвращает значение
 True в случае удачного удаления.
