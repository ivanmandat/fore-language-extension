# ICustomDimOrder


## Описание


Интерфейс ICustomDimOrder содержит
 свойства атрибута, по которому производится сортировка элементов в конструируемом
 справочнике.


## Комментарии


Параметры сортировки можно получить, используя свойства и методы интерфейса
 [ICustomDimOrders](../ICustomDimOrders/ICustomDimOrders.htm).
