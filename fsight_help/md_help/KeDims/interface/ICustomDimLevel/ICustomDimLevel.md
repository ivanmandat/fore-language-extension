# ICustomDimLevel


## Описание


Интерфейс ICustomDimLevel содержит
 свойства для определения параметров уровня конструируемого справочника.


## Комментарии


Параметры уровня можно получить, используя свойства и методы интерфейса
 [ICustomDimLevels](../ICustomDimLevels/ICustomDimLevels.htm).
