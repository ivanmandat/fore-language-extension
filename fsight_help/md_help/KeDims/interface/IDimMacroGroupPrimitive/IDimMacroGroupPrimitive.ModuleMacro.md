# IDimMacroGroupPrimitive.ModuleMacro


## Синтаксис


ModuleMacro: String;


## Описание


Свойство ModuleMacro определяет
 привязку к макросу.


## Комментарии


Привязка к макросу указывается в следующем формате: <Наименование
 класса>.<Наименование функции>.


Используется вместе с [IDimMacroGroupPrimitive.Module](IDimMacroGroupPrimitive.Module.htm).
