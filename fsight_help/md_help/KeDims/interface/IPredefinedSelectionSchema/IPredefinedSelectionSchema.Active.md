# IPredefinedSelectionSchema.Active


## Синтаксис


Active: Boolean;


## Описание


Свойство Active определяет,
 будет ли использоваться предопределенная схема отметки измерения.


## Комментарии


Допустимые значения:


	- True. Предопределенная
	 схема отметки измерения будет использоваться;


	- False. По умолчанию.
	 Предопределенная схема отметки использоваться не будет.
