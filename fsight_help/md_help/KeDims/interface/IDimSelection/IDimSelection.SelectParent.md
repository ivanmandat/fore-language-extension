# IDimSelection.SelectParent


## Синтаксис


SelectParent(Element: Integer; Recursive: Boolean);


## Параметры


Element. Индекс элемента, родительские
 элементы которого необходимо добавить в отметку;


Recursive. Признак добавления
 в отметку родительских элементов, находящихся на всех верхних уровнях.


## Описание


Метод SelectParent осуществляет
 добавление родительских элементов в отметку.


## Комментарии


Если в качестве значения параметра Recursive
 передаётся значение True,
 то в отметку добавляются все родительские элементы на всех уровнях. Если
 передаётся значение False, то
 добавляется непосредственный родительский элемент, расположенный на один
 уровень выше элемента Element.


## Пример


Для выполнения примера в репозитории предполагается наличие веб-формы,
 расположенной на ней кнопки с наименованием Button1 и компонента WebDimensionTree
 с наименованием DimensionTree1. К компоненту DimensionTree1 подключен
 справочник, имеющий список элементов на нескольких уровнях.


Добавьте ссылки на системные сборки: Dimensions, WebForms.


	Sub Button1OnClick;

	Begin

	    DimensionTree1.Selection.SelectParent(3, False);

	End Sub Button1OnClick;


При выполнении примера будет отмечен непосредственный родительский элемент
 для элемента с указанным индексом.
