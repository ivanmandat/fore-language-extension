# IDimSelection.LimitToHierarchy


## Синтаксис


LimitToHierarchy(Value: [IDimHierarchyInstance](../IDimHierarchyInstance/IDimHierarchyInstance.htm));


## Параметры


Value. Данные альтернативной
 иерархии справочника.


## Описание


Метод LimitToHierarchy оставляет
 отмеченными в отметке только те элементы, которые входят в указанную альтернативную
 иерархию.


## Комментарии


Для применения альтернативной иерархии в отметке справочника используйте
 [IDimSelection.SelectHierarchy](IDimSelection.SelectHierarchy.htm).
