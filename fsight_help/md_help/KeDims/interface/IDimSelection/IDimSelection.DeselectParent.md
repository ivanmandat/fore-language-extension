# IDimSelection.DeselectParent


## Синтаксис


DeselectParent(Element: Integer; Recursive: Boolean);


## Параметры


Element. Индекс элемента, с
 родительских элементов которого необходимо снять отметку;


Recursive. Признак снятия отметки
 с родительских элементов, находящихся на всех верхних уровнях.


## Описание


Метод DeselectParent осуществляет
 снятие отметки с родительских элементов.


## Комментарии


Если в качестве значения параметра Recursive
 передаётся значение True,
 то снимается отметка со всех родительских элементов на всех уровнях. Если
 передаётся значение False, то
 снимается отметка с непосредственного родительского элемента, расположенного
 на один уровень выше элемента Element.


## Пример


Для выполнения примера в репозитории предполагается наличие веб-формы,
 расположенной на ней кнопки с наименованием Button1 и компонента WebDimensionTree
 с наименованием DimensionTree1. К компоненту DimensionTree1 подключен
 справочник, имеющий список элементов на нескольких уровнях.


Добавьте ссылки на системные сборки: Dimensions, WebForms.


	Sub Button1OnClick;

	Begin

	    DimensionTree1.Selection.DeselectParent(3, False);

	End Sub Button1OnClick;


При выполнении примера будет снята отметка с непосредственного родительский
 элемента для элемента с указанным индексом.
