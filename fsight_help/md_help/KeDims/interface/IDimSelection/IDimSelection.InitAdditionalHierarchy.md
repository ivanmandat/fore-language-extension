# IDimSelection.InitAdditionalHierarchy


## Синтаксис


InitAdditionalHierarchy(Value: [IDimHierarchy](../IDimHierarchy/IDimHierarchy.htm));


## Параметры


Value. Альтернативная иерархия.


## Описание


Метод InitAdditionalHierarchy
 инициализирует альтернативную иерархию.


## Комментарии


Для инициализации альтернативной иерархии создайте отметку справочника
 с помощью метода [IDimInstance.CreateSelection](../IDimInstance/IDimInstance.CreateSelection.htm).
