# IDimSelection.ApplySelectionSchemaOnlyOnce


## Синтаксис


ApplySelectionSchemaOnlyOnce: Boolean;


## Описание


Свойство ApplySelectionSchemaOnlyOnce
 определяет признак применения выбранной схемы отметки только при открытии
 отчета.


## Комментарии


Допустимые значения:


	- True. Выбранная схема
	 отметки будет применяться только при открытии отчета. В дальнейшем
	 отметку элементов можно изменить;


	- False. Значение по умолчанию.
	 Выбранная схема отметки элементов будет применяться всегда и
	 у пользователя не будет возможности изменить отметку.
