# IDimSelection.SelectionSchemaOnceApplied


## Синтаксис


SelectionSchemaOnceApplied: Boolean;


## Описание


Свойство SelectionSchemaOnceApplied
 определяет признак того, что выбранная схема отметки применялась хотя
 бы раз.


## Комментарии


Допустимые значения:


	- True. Схема отметки
	 применялась;


	- False. Значение по умолчанию.
	 Схема отметки не применялась.


Примечание.
 Данное свойство поддерживается только при работе с измерениями экспресс-отчетов.
