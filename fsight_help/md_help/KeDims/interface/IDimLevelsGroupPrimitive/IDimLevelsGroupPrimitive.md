# IDimLevelsGroupPrimitive


## Описание


Интерфейс IDimLevelsGroupPrimitive
 содержит свойства и методы примитива группы элементов, оставляющего элементы
 уровня справочника.


## Свойства, унаследованные от [IDimElementGroupPrimitive](../IDimElementGroupPrimitive/IDimElementGroupPrimitive.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Selection](../IDimElementGroupPrimitive/IDimElementGroupPrimitive.Selection.htm)
		 Свойство Selection
		 определяет отметку примитива группы элементов справочника.


		 ![](../../Property_Image.gif)
		 [Type](../IDimElementGroupPrimitive/IDimElementGroupPrimitive.Type.htm)
		 Свойство Type возвращает
		 тип примитива группы элементов справочника.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddLevel](IDimLevelsGroupPrimitive.AddLevel.htm)
		 Метод AddLevel осуществляет
		 добавление в примитив уровня справочника, элементы которого необходимо
		 оставить.


		 ![](../../Sub_Image.gif)
		 [DeleteLevel](IDimLevelsGroupPrimitive.DeleteLevel.htm)
		 Метод DeleteLevel осуществляет
		 удаление уровня справочника из примитива группы элементов.


		 ![](../../Sub_Image.gif)
		 [IsLevelIncluded](IDimLevelsGroupPrimitive.IsLevelIncluded.htm)
		 Метод IsLevelIncluded
		 проверяет, отмечен ли указанный уровень справочника в примитиве
		 в списке уровней, чьи элементы необходимо оставить.
