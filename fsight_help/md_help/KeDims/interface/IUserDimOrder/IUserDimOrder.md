# IUserDimOrder


## Описание


Интерфейс IUserDimOrder содержит
 свойства атрибута, по которому производится сортировка вычисляемого справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attribute](IUserDimOrder.Attribute.htm)
		 Свойство Attribute
		 возвращает свойства атрибута, по которому осуществляется сортировка
		 вычисляемого справочника.


		 ![](../../Property_Image.gif)
		 [ReverseOrder](IUserDimOrder.ReverseOrder.htm)
		 Свойство ReverseOrder
		 определяет направление сортировки по данному атрибуту.
