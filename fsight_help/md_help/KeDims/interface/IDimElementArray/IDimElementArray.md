# IDimElementArray


## Описание


Интерфейс IDimElementArray содержит
 свойства и методы массива элементов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDimElementArray.Count.htm)
		 Свойство Count возвращает
		 количество элементов в массиве.


		 ![](../../Property_Image.gif)
		 [Element](IDimElementArray.Element.htm)
		 Свойство Element возвращает
		 индекс элемента в справочнике по индексу элемента в массиве.


		 ![](../../Property_Image.gif)
		 [Elements](IDimElementArray.Elements.htm)
		 Свойство Elements возвращает
		 объект, содержащий все элементы справочника.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IDimElementArray.Add.htm)
		 Метод Add осуществляет
		 добавление элемента справочника в массив.


		 ![](../../Sub_Image.gif)
		 [Clear](IDimElementArray.Clear.htm)
		 Метод Clear осуществляет
		 очистку массива элементов.


		 ![](../../Sub_Image.gif)
		 [CopyTo](IDimElementArray.CopyTo.htm)
		 Метод CopyTo осуществляет
		 копирование массива элементов в приёмник.


		 ![](../../Sub_Image.gif)
		 [CreateCopy](IDimElementArray.CreateCopy.htm)
		 Метод CreateCopy создает
		 копию массива элементов.


		 ![](../../Sub_Image.gif)
		 [Find](IDimElementArray.Find.htm)
		 Метод Find осуществляет
		 поиск элемента справочника в массиве.


		 ![](../../Sub_Image.gif)
		 [Iterator](IDimElementArray.Iterator.htm)
		 Метод Iterator создает
		 объект, содержащий итератор.


		 ![](../../Sub_Image.gif)
		 [Remove](IDimElementArray.Remove.htm)
		 Метод Remove осуществляет
		 удаление элемента с индексом из массива.


		 ![](../../Sub_Image.gif)
		 [Selection](IDimElementArray.Selection.htm)
		 Метод Selection создает
		 отметку справочника, состоящую из элементов массива.
