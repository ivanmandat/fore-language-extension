# IDimBlocks


## Описание


Интерфейс IDimBlocks содержит
 свойства и методы коллекции блоков справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDimBlocks.Count.htm)
		 Свойство Count возвращает
		 количество блоков справочника.


		 ![](../../Property_Image.gif)
		 [Item](IDimBlocks.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий блок справочника.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindByKey](IDimBlocks.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск по ключу и возвращает объект, содержащий блок справочника.
