# IDimBlocks.Item


## Синтаксис


Item(Index: Variant): [IDimBlock](../IDimBlock/IDimBlock.htm);


## Параметры


Index — индекс
 блока, который необходимо получить.


## Описание


Свойство Item возвращает объект,
 содержащий блок справочника. Индекс блока передается посредством параметра
 Index.
