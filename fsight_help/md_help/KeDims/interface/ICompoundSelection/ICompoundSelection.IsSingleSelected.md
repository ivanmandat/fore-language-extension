# ICompoundSelection.IsSingleSelected


## Синтаксис


IsSingleSelected: Boolean;


## Описание


Свойство IsSingleSelected определяет
 признак того, присутствуют ли источники в отметке. True
 - источники в отметке отсутствуют.


## Комментарии


Свойство имеет значение True,
 если источники не имеют частных измерений, на базе которых строится составной
 справочник.
