# ICompoundSelection.Selection


## Синтаксис


Selection: [IDimSelectionSet](../IDimSelectionSet/IDimSelectionSet.htm);


## Описание


Свойство доступно только для чтения.


Свойство Selection возвращает
 составную отметку.


## Комментарии


Для составной отметки, которая соответствует составному справочнику
 экспресс-отчёта, свойство Selection
 вернёт коллекцию составных отметок, число которых соответствует числу
 источников данных. Каждая отдельная составная отметка в свою очередь будет
 построена на частных измерениях отдельного источника данных.
