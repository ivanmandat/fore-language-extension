# ICompoundSelection


## Описание


Интерфейс ICompoundSelection
 содержит свойства и методы для работы с составной отметкой.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Filter](ICompoundSelection.Filter.htm)
		 Свойство Filter возвращает
		 параметры фильтрации составной отметки.


		 ![](../../Property_Image.gif)
		 [IsCompound](ICompoundSelection.IsCompound.htm)
		 Свойство IsCompound
		 определяет признак того, каким образом задана отметка: над элементами
		 составного справочника или над элементами источников составного
		 справочника.


		 ![](../../Property_Image.gif)
		 [IsDisplayCompound](ICompoundSelection.IsDisplayCompound.htm)
		 Свойство IsDisplayCompound
		 определяет, будут ли отображаться частные измерения в виде составного
		 справочника.


		 ![](../../Property_Image.gif)
		 [IsSingleSelected](ICompoundSelection.IsSingleSelected.htm)
		 Свойство IsSingleSelected
		 определяет признак того, присутствуют ли источники в отметке.


		 ![](../../Property_Image.gif)
		 [Key](ICompoundSelection.Key.htm)
		 Свойство Key возвращает
		 ключ составной отметки.


		 ![](../../Property_Image.gif)
		 [Selection](ICompoundSelection.Selection.htm)
		 Свойство Selection
		 возвращает составную отметку.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Decompose](ICompoundSelection.Decompose.htm)
		 Метод Decompose раскладывает
		 составную отметку по измерениям.
