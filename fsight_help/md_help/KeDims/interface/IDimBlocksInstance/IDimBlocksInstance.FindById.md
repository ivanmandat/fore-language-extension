# IDimBlocksInstance.FindById


## Синтаксис


FindById(Id: String): [IDimBlockInstance](../IDimBlockInstance/IDimBlockInstance.htm);


## Параметры


Id - идентификатор блока справочника,
 по которому осуществляется поиск.


## Описание


Метод FindById осуществляет
 поиск блока справочника по идентификатору, и возвращает данные блока в
 случае удачного поиска.
