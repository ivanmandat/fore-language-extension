# IDimBlocksInstance.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IDimBlockInstance](../IDimBlockInstance/IDimBlockInstance.htm);


## Параметры


Key - ключ
 блока справочника, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск блока справочника по ключу, и возвращает данные блока в случае удачного
 поиска.
