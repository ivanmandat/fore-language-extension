# IDimBlocksInstance


## Описание


Интерфейс IDimBlocksInstance
 содержит свойства и методы коллекции блоков справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDimBlocksInstance.Count.htm)
		 Свойство Count возвращает
		 количество блоков справочника.


		 ![](../../Property_Image.gif)
		 [Item](IDimBlocksInstance.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий блок справочника.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](IDimBlocksInstance.FindById.htm)
		 Метод FindById осуществляет
		 поиск блока справочника по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IDimBlocksInstance.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск блока справочника по ключу.
