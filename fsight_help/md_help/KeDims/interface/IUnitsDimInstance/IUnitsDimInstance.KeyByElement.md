# IUnitsDimInstance.KeyByElement


## Синтаксис


KeyByElement(Element: Integer): Variant;


## Параметры


Element. Индекс элемента справочника.


## Описание


Свойство KeyByElement возвращает
 ключ элемента.


## Комментарии


KeyByElement возвращает отрицательные
 значения, если элемент [является
 мерой](IUnitsDimInstance.IsUnit.htm).
