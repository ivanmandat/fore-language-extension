# IUnitsDimInstance.Scale


## Синтаксис


Scale(Element: Integer): Double;


## Параметры


Element. Индекс элемента справочника.


## Описание


Свойство Scale возвращает масштаб
 элемента.


## Комментарии


Масштаб может быть возвращен только для элемента, который является [единицей измерения](IUnitsDimInstance.IsUnit.htm).
