# IUnitsDimInstance.IsUnit


## Синтаксис


IsUnit(Element: Integer): Boolean;


## Параметры


Element. Индекс элемента справочника.


## Описание


Свойство IsUnit возвращает признак
 того, что элемент является единицей измерения.


## Комментарии


Допустимые значения:


	- True. Элемент является
	 единицей измерения;


	- False. Элемент является
	 мерой.
