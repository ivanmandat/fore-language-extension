# IUnitsDimInstance


Сборка: Dimensions;


## Описание


Интерфейс IUnitsDimInstance
 предназначен для работы с внутренним измерением единиц измерения у базы
 данных временных рядов .


## Иерархия наследования


IUnitsDimInstance


## Комментарии


IUnitsDimInstance также используется
 для получения базовой единицы измерения в базе данных временных рядов
 .
