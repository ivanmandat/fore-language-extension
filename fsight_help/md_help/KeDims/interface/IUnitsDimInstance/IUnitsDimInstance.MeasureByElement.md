# IUnitsDimInstance.MeasureByElement


## Синтаксис


MeasureByElement(Element: Integer): Integer;


## Параметры


Element. Индекс элемента справочника.


## Описание


Свойство MeasureByElement возвращает
 меру элемента.


## Комментарии


KeyByElement возвращает значение
 «-1», если элемент [является мерой](IUnitsDimInstance.IsUnit.htm).
