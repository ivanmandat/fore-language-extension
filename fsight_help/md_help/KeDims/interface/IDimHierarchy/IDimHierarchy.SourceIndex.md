# IDimHierarchy.SourceIndex


## Синтаксис


SourceIndex: [IDimIndex](../IDimIndex/IDimIndex.htm);


## Описание


Свойство SourceIndex определяет
 индекс альтернативного справочника, содержащий ключи для сопоставления
 элементов основного и альтернативного справочников.


## Комментарии


Для определения соответствия между атрибутом основного справочника и
 атрибутом альтернативного справочника используйте свойство [IDimHierarchy.SourceAttribute](IDimHierarchy.SourceAttribute.htm).
