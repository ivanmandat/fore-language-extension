# IDimHierarchy.SourceAttribute


## Синтаксис


SourceAttribute(OriginalAttribute: [IDimAttribute](../IDimAttribute/IDimAttribute.htm)):
 [IDimAttribute](../IDimAttribute/IDimAttribute.htm);


## Параметры


OriginalAttribute - атрибут
 основного справочника.


## Описание


Свойство SourceAttribute определяет
 соответствие между атрибутом основного справочника и атрибутом альтернативного
 справочника.


## Комментарии


Для определения индекса альтернативного справочника, содержащего ключи
 для сопоставления элементов основного и альтернативного справочников,
 используйте свойство [IDimHierarchy.SourceIndex](IDimHierarchy.SourceIndex.htm).
