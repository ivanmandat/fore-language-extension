# IUserDimLevels


## Описание


Интерфейс IUserDimLevels содержит
 свойства и методы коллекции уровней вычисляемого справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IUserDimLevels.Count.htm)
		 Свойство Count возвращает
		 количество уровней вычисляемого справочника.


		 ![](../../Property_Image.gif)
		 [Item](IUserDimLevels.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий уровень вычисляемого справочника.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IUserDimLevels.Add.htm)
		 Метод Add добавляет
		 уровень в вычисляемый справочник.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IUserDimLevels.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск по ключу и возвращает уровень вычисляемого справочника.


		 ![](../../Sub_Image.gif)
		 [Remove](IUserDimLevels.Remove.htm)
		 Метод Remove осуществляет
		 удаление уровня по индексу.
