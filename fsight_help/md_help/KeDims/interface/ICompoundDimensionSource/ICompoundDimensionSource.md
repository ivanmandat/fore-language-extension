# ICompoundDimensionSource


Сборка: Dimensions;


## Описание


Интерфейс ICompoundDimensionSource
 содержит свойства для работы с источником составного справочника.


## Иерархия наследования


ICompoundDimensionSource


[INamedEntity](KeSom.chm::/Interface/INamedEntity/INamedEntity.htm)


## Комментарии


Коллекцию источников составного справочника возвращает интерфейс [ICompoundDimensionSources](../ICompoundDimensionSources/ICompoundDimensionSources.htm).
