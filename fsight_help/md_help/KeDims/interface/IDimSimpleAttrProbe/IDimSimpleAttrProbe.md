# IDimSimpleAttrProbe


Сборка: Dimensions;


## Описание


Интерфейс IDimSimpleAttrProbe
 содержит методы для проверки атрибутов, по значениям которых будет производиться
 поиск.


## Иерархия наследования


IDimSimpleAttrProbe


## Комментарии


Методы данного интерфейса должны быть переопределены в пользовательском
 классе.
