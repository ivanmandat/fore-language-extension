# IUserDimAttributes


## Описание


Интерфейс IUserDimAttributes
 содержит свойства и методы коллекции атрибутов вычисляемого справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IUserDimAttributes.Count.htm)
		 Свойство Count возвращает
		 количество атрибутов вычисляемого справочника.


		 ![](../../Property_Image.gif)
		 [Id](IUserDimAttributes.Id.htm)
		 Свойство Id определяет
		 атрибут вычисляемого справочника, назначение которого - Идентификатор.


		 ![](../../Property_Image.gif)
		 [ImageIndex](IUserDimAttributes.ImageIndex.htm)
		 Свойство ImageIndex
		 определяет атрибут вычисляемого справочника, назначение которого
		 - Пиктограмма элементов.


		 ![](../../Property_Image.gif)
		 [Item](IUserDimAttributes.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий атрибут справочника.


		 ![](../../Property_Image.gif)
		 [Name](IUserDimAttributes.Name.htm)
		 Свойство Name определяет
		 атрибут справочника, назначение которого - Наименование.


		 ![](../../Property_Image.gif)
		 [Order](IUserDimAttributes.Order.htm)
		 Свойство Order определяет
		 атрибут справочника, назначение которого - Порядок.


		 ![](../../Property_Image.gif)
		 [Orders](IUserDimAttributes.Orders.htm)
		 Свойство Orders возвращает
		 коллекцию атрибутов, по которым будет производиться сортировка
		 вычисляемого справочника.


		 ![](../../Property_Image.gif)
		 [ReadAccess](IUserDimAttributes.ReadAccess.htm)
		 Свойство ReadAccess
		 определяет атрибут справочника, ограничивающий права на чтение
		 записей.


		 ![](../../Property_Image.gif)
		 [WriteAccess](IUserDimAttributes.WriteAccess.htm)
		 Свойство WriteAccess
		 определяет атрибут справочника, ограничивающий права на изменение
		 записей.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IUserDimAttributes.Add.htm)
		 Метод Add осуществляет
		 добавление атрибута вычисляемого справочника.


		 ![](../../Sub_Image.gif)
		 [Clear](IUserDimAttributes.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех атрибутов вычисляемого справочника.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IUserDimAttributes.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск и возвращает объект, содержащий атрибут вычисляемого справочника.


		 ![](../../Sub_Image.gif)
		 [Remove](IUserDimAttributes.Remove.htm)
		 Метод Remove осуществляет
		 удаление атрибута вычисляемого справочника по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IUserDimAttributes.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление атрибута вычисляемого справочника по ключу.
