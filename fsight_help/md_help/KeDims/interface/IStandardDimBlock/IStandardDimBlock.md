# IStandardDimBlock


## Описание


Интерфейс IStandardDimBlock
 содержит свойства структуры блока табличного справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Binding](IStandardDimBlock.Binding.htm)
		 Свойство Binding возвращает
		 объект, содержащий формулу, определяющую соответствие между атрибутами
		 блока и полями источника данных.


		 ![](../../Property_Image.gif)
		 [BlockType](IStandardDimBlock.BlockType.htm)
		 Свойство BlockType
		 возвращает тип блока табличного справочника.


		 ![](../../Property_Image.gif)
		 [ChildBlock](IStandardDimBlock.ChildBlock.htm)
		 Свойство ChildBlock
		 возвращает объект, содержащий дочерний блок табличного справочника.


		 ![](../../Property_Image.gif)
		 [ChildrenBlockCount](IStandardDimBlock.ChildrenBlockCount.htm)
		 Свойство ChildrenBlockCount
		 возвращает количество дочерних блоков.


		 ![](../../Property_Image.gif)
		 [Dataset](IStandardDimBlock.Dataset.htm)
		 Свойство Dataset определяет
		 объект, содержащий структуру источника данных блока справочника.


		 ![](../../Property_Image.gif)
		 [Indexes](IStandardDimBlock.Indexes.htm)
		 Свойство Indexes возвращает
		 объект, содержащий коллекцию индексов блока табличного справочника.


		 ![](../../Property_Image.gif)
		 [ParentBlock](IStandardDimBlock.ParentBlock.htm)
		 Свойство ParentBlock
		 возвращает блок-владелец.


		 ![](../../Property_Image.gif)
		 [ParentBlockBinding](IStandardDimBlock.ParentBlockBinding.htm)
		 Свойство ParentBlockBinding
		 возвращает объект, содержащий формулу, определяющую связь поля
		 источника данных и атрибута, входящего в уникальный индекс блока-владельца.


		 ![](../../Property_Image.gif)
		 [ParentBlockIndex](IStandardDimBlock.ParentBlockIndex.htm)
		 Свойство ParentBlockIndex
		 определяет уникальный индекс блока-владельца для связи с текущим
		 блоком.


		 ![](../../Property_Image.gif)
		 [Where](IStandardDimBlock.Where.htm)
		 Свойство Where возвращает
		 условие фильтрации всех элементов блока табличного справочника.
