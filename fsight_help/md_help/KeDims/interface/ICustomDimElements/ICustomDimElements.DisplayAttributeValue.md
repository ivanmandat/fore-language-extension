# ICustomDimElements.DisplayAttributeValue


## Синтаксис


DisplayAttributeValue(Row: Integer; AttributeIndex:
 Integer): Variant;


## Параметры


Row. Индекс
 строки, в которой находится необходимый элемент.


AttributeIndex. Индекс атрибута,
 отображаемое значение которого необходимо определить.


## Описание


Свойство DisplayAttributeValue
 определяет отображаемое значение атрибута элемента.
