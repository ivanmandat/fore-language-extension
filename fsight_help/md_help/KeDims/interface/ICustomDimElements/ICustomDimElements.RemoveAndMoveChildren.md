# ICustomDimElements.RemoveAndMoveChildren


## Синтаксис


RemoveAndMoveChildren(Row: Integer);


## Параметры


Row. Индекс строки, в которой
 находится удаляемый элемент.


## Описание


Метод RemoveAndMoveChildren
 осуществляет удаление указанного элемента со смещение всех дочерних элементов
 на одну строку вверх.
