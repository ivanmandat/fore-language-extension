# ICustomDimElements.DisplayAttributeValueByKey


## Синтаксис


DisplayAttributeValueByKey(Row: Integer; AttributeKey:
 Integer): Variant;


## Параметры


Row. Индекс строки, в которой
 находится необходимый элемент.


AttributeKey. Ключ атрибута,
 отображаемое значение которого необходимо получить.


## Описание


Свойство DisplayAttributeValue
 определяет отображаемое значение атрибута элемента по ключу атрибута.
