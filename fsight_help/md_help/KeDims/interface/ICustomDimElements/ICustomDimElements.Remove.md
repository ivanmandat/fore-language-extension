# ICustomDimElements.Remove


## Синтаксис


Remove(Row: Integer);


## Параметры


Row. Индекс
 строки, в которой находится удаляемый элемент.


## Описание


Метод Remove осуществляет удаление
 указанного элемента со всеми дочерними элементами.
