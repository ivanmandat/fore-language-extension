# IDimAsCalendar.IsCalendar


## Синтаксис


IsCalendar: Boolean;


## Описание


Свойство IsCalendar определяет,
 используется ли справочник в качестве календарного.


## Комментарии


Необходимые требования для использования справочника в качестве календаря
 указаны с описании свойства [IDimensionModel.IsCalendar](../IDimensionModel/IDimensionModel.IsCalendar.htm).
 После сопоставления [атрибутов](IDimAsCalendar.Attrs.htm) и
 [уровней](IDimAsCalendar.Levels.htm) текущего справочника укажите
 в свойстве IsCalendar значение
 True, чтобы справочник использовался
 в качестве календаря.
