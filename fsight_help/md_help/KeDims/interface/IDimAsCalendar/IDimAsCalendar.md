# IDimAsCalendar


## Описание


Интерфейс IDimAsCalendar предназначен
 для сопоставления атрибутов и уровней справочника для того, чтобы использовать
 его в качестве календаря.


## Комментарии


Параметры сопоставления возвращает свойство [IDimensionModel.AsCalendar](../IDimensionModel/IDimensionModel.AsCalendar.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attrs](IDimAsCalendar.Attrs.htm)
		 Свойство Attrs
		 возвращает коллекцию атрибутов для сопоставления при использовании
		 справочника в качестве календарного.


		 ![](../../Property_Image.gif)
		 [IsCalendar](IDimAsCalendar.IsCalendar.htm)
		 Свойство IsCalendar
		 определяет, используется ли справочник в качестве календарного.


		 ![](../../Property_Image.gif)
		 [Levels](IDimAsCalendar.Levels.htm)
		 Свойство Levels
		 возвращает коллекцию уровней для сопоставления при использовании
		 справочника в качестве календарного.
