# IDimElementGroup


## Описание


Интерфейс IDimElementGroup содержит
 свойства и методы групп элементов справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDimElementGroup.Count.htm)
		 Свойство Count возвращает
		 количество примитивов в группе элементов.


		 ![](../../Property_Image.gif)
		 [Dimension](IDimElementGroup.Dimension.htm)
		 Свойство Dimension
		 определяет справочник, для которого строится группа элементов.


		 ![](../../Property_Image.gif)
		 [DimensionInstance](IDimElementGroup.DimensionInstance.htm)
		 Свойство DimensionInstance
		 позволяет установить экземпляр справочника для группы элементов.


		 ![](../../Property_Image.gif)
		 [Item](IDimElementGroup.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий примитив группы элементов.


		 ![](../../Property_Image.gif)
		 [Selection](IDimElementGroup.Selection.htm)
		 Свойство Selection
		 возвращает объект, содержащий отметку группы элементов справочника.


		 ![](../../Property_Image.gif)
		 [UnionType](IDimElementGroup.UnionType.htm)
		 Свойство UnionType
		 определяет тип взаимодействия между собой нескольких примитивов
		 группы элементов.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddPrimitive](IDimElementGroup.AddPrimitive.htm)
		 Метод AddPrimitive
		 позволяет добавлять к группе элементов справочника примитивы.


		 ![](../../Sub_Image.gif)
		 [RemovePrimitive](IDimElementGroup.RemovePrimitive.htm)
		 Метод RemovePrimitive
		 осуществляет удаление примитивов из группы элементов справочника.
