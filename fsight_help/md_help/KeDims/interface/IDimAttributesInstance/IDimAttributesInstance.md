# IDimAttributesInstance


## Описание


Интерфейс IDimAttributesInstance
 содержит свойства и методы коллекции атрибутов справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDimAttributesInstance.Count.htm)


		 Свойство Count возвращает
		 количество атрибутов справочника.


		 ![](../../Property_Image.gif)
		 [Item](IDimAttributesInstance.Item.htm)


		 Свойство Item возвращает
		 объект, содержащий атрибут справочника.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)


		 [EnsureValuesLoaded](IDimAttributesInstance.EnsureValuesLoaded.htm)


		 Метод EnsureValuesLoaded
		 осуществляет проверку загрузки атрибутов справочника.


		 ![](../../Sub_Image.gif)
		 [FindById](IDimAttributesInstance.FindById.htm)


		 Метод FindById осуществляет
		 поиск атрибута по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IDimAttributesInstance.FindByKey.htm)


		 Метод FindByKey осуществляет
		 поиск атрибута по ключу.
