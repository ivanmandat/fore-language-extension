# ICustomDimAttribute.HasDisplayValues


## Синтаксис


HasDisplayValues: Boolean;


## Описание


Свойство HasDisplayValues определяет
 признак использования отображаемых значений атрибута.


## >Комментарии


Допустимые значения:


	- True. Отображаемые значения
	 используются;


	- False. Отображаемые
	 значения не используются.
