# ICustomDimAttribute


Сборка: Dimensions;


## Описание


Интерфейс ICustomDimAttribute
 содержит свойства атрибута конструируемого справочника.


## Иерархия наследования


           [INamedEntity](KeSom.chm::/Interface/INamedEntity/INamedEntity.htm)


           ICustomDimAttribute


## Комментарии


Свойства атрибута можно получить, используя свойства или методы интерфейса
 [ICustomDimAttributes](../ICustomDimAttributes/ICustomDimAttributes.htm).

		 Имя свойства
		 Краткое описание

		 [DataType](ICustomDimAttribute.DataType.htm)
		 Свойство DataType определяет
		 тип данных атрибута.


		 ![](../../Property_Image.gif)
		 [DisplayValueDataType](ICustomDimAttribute.DisplayValueDataType.htm)
		 Свойство DisplayValueDataType
		 определяет тип отображаемого значения атрибута.


		 ![](../../Property_Image.gif)
		 [Expression](ICustomDimAttribute.Expression.htm)
		 Свойство Expression
		 возвращает выражение, в соответствии с которым будет формироваться

		 ![](../../Property_Image.gif)
		 [HasDisplayValues](ICustomDimAttribute.HasDisplayValues.htm)
		 Свойство HasDisplayValues

		 ![](../../Property_Image.gif)
		 [HasHierarchy](ICustomDimAttribute.HasHierarchy.htm)
		 Свойство HasHierarchy
		 определяет, используется ли атрибут в качестве альтернативной
		 иерархии.


		 ![](../../Property_Image.gif)
		 [Visible](ICustomDimAttribute.Visible.htm)
		 Свойство Visible определяет
		 признак видимости значений по данному атрибуту.
