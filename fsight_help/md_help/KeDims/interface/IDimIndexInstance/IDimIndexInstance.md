# IDimIndexInstance


Сборка: Dimensions;


## Описание


Интерфейс IDimIndexInstance
 содержит свойства и методы для работы с индексированными данными справочника.


## Иерархия наследования


IDimIndexInstance


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Index](IDimIndexInstance.Index.htm)
		 Свойство Index возвращает
		 структуру индекса справочника.


		 ![](../../Property_Image.gif)
		 [Block](IDimIndexInstance.Block.htm)
		 Свойство Block возвращает
		 блок справочника, в котором создан индекс.


		 ![](../../Property_Image.gif)
		 [CaseSensitive](IDimIndexInstance.CaseSensitive.htm)
		 Свойство CaseSensitive
		 возвращает признак того, чувствителен ли индекс к регистру.


		 ![](../../Property_Image.gif)
		 [UseInSelection](IDimIndexInstance.UseInSelection.htm)
		 Свойство UseInSelection
		 возвращает признак того, используется ли индекс при формировании
		 отметки.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CreateDimIndexLookup](IDimIndexInstance.CreateDimIndexLookup.htm)
		 Метод CreateDimIndexLookup
		 создает объект для поиска элементов справочника.


		 ![](../../Sub_Image.gif)
		 [IndexAttributesValues](IDimIndexInstance.IndexAttributesValues.htm)
		 Метод IndexAttributesValues
		 возвращает значения атрибутов, включенных в индекс, по указанному
		 элементу.


		 ![](../../Sub_Image.gif)
		 [LookUp](IDimIndexInstance.LookUp.htm)
		 Метод LookUp осуществляет
		 поиск элемента справочника.


		 ![](../../Sub_Image.gif)
		 [LookUpLevel](IDimIndexInstance.LookUpLevel.htm)
		 Метод LookUpLevel выполняет
		 поиск элемента в указанном уровне справочника.
