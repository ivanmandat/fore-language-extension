# IDimOrmCondition


## Описание


Интерфейс IDimOrmCondition предназначен
 для работы с условиями.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Conditions](IDimOrmCondition.Conditions.htm)
		 Свойство Conditions
		 определяет условия поиска.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Evaluate](IDimOrmCondition.Evaluate.htm)
		 Метод Evaluate проверяет
		 удовлетворяет ли элемент измерения условиям.


		 ![](../../Sub_Image.gif)
		 [EvaluateAll](IDimOrmCondition.EvaluateAll.htm)
		 Метод EvaluateAll осуществляет
		 проверку всех элементов справочника и возвращает итератор, содержащий
		 элементы, удовлетворяющие условиям поиска.


		 ![](../../Sub_Image.gif)
		 [EvaluateRec](IDimOrmCondition.EvaluateRec.htm)
		 Метод EvaluateRec осуществляет
		 проверку значений атрибутов и возвращает признак соответствия
		 условиям поиска.


		 ![](../../Sub_Image.gif)
		 [SelectElements](IDimOrmCondition.SelectElements.htm)
		 Метод SelectElements
		 формирует отметку элементов, удовлетворяющих условиям.


		 ![](../../Sub_Image.gif)
		 [SetTextCriteria](IDimOrmCondition.SetTextCriteria.htm)
		 Метод SetTextCriteria
		 определяет условие поиска по тексту атрибутов.
