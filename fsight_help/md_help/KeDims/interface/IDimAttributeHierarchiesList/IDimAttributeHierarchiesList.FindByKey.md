# IDimAttributeHierarchiesList.FindByKey


## Синтаксис


FindByKey(hierarchyKey: Integer): [IDimAttributeHierarchy](../IDimAttributeHierarchy/IDimAttributeHierarchy.htm);


## Параметры


hierarchyKey. Ключ альтернативной
 иерархии.


## Описание


Метод FindByKey выполняет поиск
 альтернативной иерархии атрибута в списке по ключу.


## Комментарии


Если альтернативная иерархия не найдена, то метод возвращает значение
 Null.


Для поиска альтернативных иерархий в списке по уровню используйте [IDimAttributeHierarchiesList.FindByLevel](IDimAttributeHierarchiesList.FindByLevel.htm).
