# IDimAttributeHierarchiesList.FindById


## Синтаксис


FindById(hierarchyId: String): [IDimAttributeHierarchy](../IDimAttributeHierarchy/IDimAttributeHierarchy.htm);


## Параметры


hierarchyId. Идентификатор
 альтернативной иерархии атрибута.


## Описание


Метод FindById выполняет поиск
 альтернативной иерархии атрибута в списке по идентификатору.


## Комментарии


Если альтернативная иерархия не найдена, то метод возвращает значение
 Null.


Для поиска альтернативной иерархии в списке по ключу используйте [IDimAttributeHierarchiesList.FindByKey](IDimAttributeHierarchiesList.FindByKey.htm).
