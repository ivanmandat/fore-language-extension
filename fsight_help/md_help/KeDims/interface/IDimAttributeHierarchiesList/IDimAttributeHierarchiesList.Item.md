# IDimAttributeHierarchiesList.Item


## Синтаксис


Item(Index: Integer): [IDimAttributeHierarchy](../IDimAttributeHierarchy/IDimAttributeHierarchy.htm);


## Параметры


Index. Индекс иерархии атрибута.


## Описание


Свойство Item возвращает указанную
 альтернативную иерархию из списка.


## Комментарии


Для получения количества альтернативных иерархий в списке используйте
 [IDimAttributeHierarchiesList.Count](IDimAttributeHierarchiesList.count.htm).
