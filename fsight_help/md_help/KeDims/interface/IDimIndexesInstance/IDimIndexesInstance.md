# IDimIndexesInstance


Сборка: Dimensions;


## Описание


Интерфейс IDimIndexesInstance
 содержит свойства и методы для работы с коллекцией индексированных данных
 справочника.


## Иерархия наследования


IDimIndexesInstance


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDimIndexesInstance.Count.htm)
		 Свойство Count возвращает
		 количество индексов справочника.


		 ![](../../Property_Image.gif)
		 [Item](IDimIndexesInstance.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий индекс справочника.


		 ![](../../Property_Image.gif)
		 [PrimaryIndex](IDimIndexesInstance.PrimaryIndex.htm)
		 Свойство PrimaryIndex
		 возвращает данные первичного индекса справочника.


		 ![](../../Property_Image.gif)
		 [SelectionIndex](IDimIndexesInstance.SelectionIndex.htm)
		 Свойство SelectionIndex
		 возвращает индекс, который используется для формирования отметки.
