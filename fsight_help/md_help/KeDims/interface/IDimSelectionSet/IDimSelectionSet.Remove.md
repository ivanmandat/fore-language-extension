# IDimSelectionSet.Remove


## Синтаксис


Remove(Index: Variant): Boolean;


## Параметры


Index — индекс
 справочника в кубе.


## Описание


Метод Remove удаляет отметку
 справочника из коллекции. Индекс справочника передается посредством параметра
 Index.
