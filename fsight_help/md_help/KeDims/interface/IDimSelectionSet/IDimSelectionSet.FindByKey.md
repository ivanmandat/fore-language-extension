# IDimSelectionSet.FindByKey


## Синтаксис


FindByKey(DimensionKey: Integer): [IDimSelection](../IDimSelection/IDimSelection.htm);


## Параметры


DimensionKey — ключ
 справочника, отметку которого необходимо получить.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает объект, содержащий отметку справочника. Поиск осуществляется
 по ключу справочника, передаваемому посредством параметра DimensionKey.
