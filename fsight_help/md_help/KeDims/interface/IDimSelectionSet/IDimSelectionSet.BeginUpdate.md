# IDimSelectionSet.BeginUpdate


## Синтаксис


BeginUpdate;


## Описание


Метод BeginUpdate устанавливает
 запрет на выполнение событий.


## Комментарии


Все события, обрабатываемые после выполнения метода BeginUpdate,
 не будут выполняться до тех пор, пока не будет вызван метод [IDimSelectionSet.EndUpdate](IDimSelectionSet.EndUpdate.htm).
