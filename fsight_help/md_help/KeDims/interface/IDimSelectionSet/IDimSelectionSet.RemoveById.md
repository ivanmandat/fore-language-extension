# IDimSelectionSet.RemoveById


## Синтаксис


RemoveById(Id: String): Boolean;


## Параметры


Id — идентификатор
 справочника, отметку которого необходимо удалить.


## Описание


Метод RemoveById удаляет отметку
 справочника из коллекции. Идентификатор справочника передается посредством
 параметра Id.
