# IDimSelectionSet.RemoveByKey


## Синтаксис


RemoveByKey(DimensionKey: Integer): Boolean;


## Параметры


DimensionKey - ключ
 справочника, отметку которого необходимо удалить.


## Описание


Метод RemoveByKey удаляет отметку
 справочника из коллекции. Ключ справочника передается посредством параметра
 DimensionKey.
