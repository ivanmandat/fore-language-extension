# IDimSelectionSet


## Описание


Интерфейс IDimSelectionSet содержит
 свойства и методы для работы с коллекцией отметок справочников.


## Комментарии


Для задания дополнительных параметров работы с коллекцией отметок приведите
 имеющийся объект к интерфейсу [IDimSelectionSet2](../IDimSelectionSet2/IDimSelectionSet2.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AllSelected](IDimSelectionSet.AllSelected.htm)


		 Свойство AllSelected
		 возвращает признак выделения всех элементов в отметках всех справочников.


		 ![](../../Property_Image.gif)
		 [Count](IDimSelectionSet.Count.htm)


		 Свойство Count возвращает
		 количество отметок справочников.


		 ![](../../Property_Image.gif)
		 [Item](IDimSelectionSet.Item.htm)


		 Свойство Item возвращает
		 отметку справочника.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IDimSelectionSet.Add.htm)


		 Метод Add добавляет
		 справочник в коллекцию и возвращает его отметку.


		 ![](../../Sub_Image.gif)
		 [AddCompound](IDimSelectionSet.AddCompound.htm)


		 Метод AddCompound осуществляет
		 добавление составной отметки.


		 ![](../../Sub_Image.gif)
		 [BeginUpdate](IDimSelectionSet.BeginUpdate.htm)


		 Метод BeginUpdate
		 устанавливает запрет на выполнение событий.


		 ![](../../Sub_Image.gif)
		 [Clear](IDimSelectionSet.Clear.htm)


		 Метод Clear осуществляет
		 удаление всех отметок справочников из коллекции.


		 ![](../../Sub_Image.gif)
		 [Compare](IDimSelectionSet.Compare.htm)


		 Метод Compare осуществляет
		 сравнение текущей отметки с указанной отметкой.


		 ![](../../Sub_Image.gif)
		 [CopyTo](IDimSelectionSet.CopyTo.htm)


		 Метод CopyTo осуществляет
		 копирование отметки справочников в указанный приёмник.


		 ![](../../Sub_Image.gif)
		 [CopyToEx](IDimSelectionSet.CopyToEx.htm)


		 Метод CopyToEx
		 копирует группы элементов и отметки справочников в указанный
		 приёмник.


		 ![](../../Sub_Image.gif)
		 [CreateCopy](IDimSelectionSet.CreateCopy.htm)


		 Метод CreateCopy создает
		 копию отметок справочников.


		 ![](../../Sub_Image.gif)
		 [EndUpdate](IDimSelectionSet.EndUpdate.htm)


		 Метод EndUpdate снимает
		 запрет на выполнение событий.


		 ![](../../Sub_Image.gif)
		 [FindById](IDimSelectionSet.FindById.htm)


		 Метод FindById осуществляет
		 поиск отметки справочника по идентификатору справочника.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IDimSelectionSet.FindByKey.htm)


		 Метод FindByKey осуществляет
		 поиск отметки справочника по ключу справочника.


		 ![](../../Sub_Image.gif)
		 [IndexOfKey](IDimSelectionSet.IndexOfKey.htm)


		 Метод IndexOfKey возвращает
		 индекс отметки справочника в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IDimSelectionSet.Remove.htm)


		 Метод Remove удаляет
		 отметку справочника по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveById](IDimSelectionSet.RemoveById.htm)


		 Метод RemoveById удаляет
		 отметку справочника по идентификатору справочника.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IDimSelectionSet.RemoveByKey.htm)


		 Метод RemoveByKey удаляет
		 отметку справочника по ключу справочника.


		 ![](../../Sub_Image.gif)
		 [SaveElementDependencies](IDimSelectionSet.SaveElementDependencies.htm)


		 Метод SaveElementDependencies
		 осуществляет сохранение коллекции зависимостей от элементов, содержащихся
		 в отметках справочников.
