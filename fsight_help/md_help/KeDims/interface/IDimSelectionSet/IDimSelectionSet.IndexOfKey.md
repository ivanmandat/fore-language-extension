# IDimSelectionSet.IndexOfKey


## Синтаксис


IndexOfKey(Key: Integer): Integer;


## Параметры


Key - ключ
 справочника.


## Описание


Метод IndexOfKey возвращает
 индекс отметки справочника в коллекции. Ключ справочника передается посредством
 параметра Key. Свойство возвращает «-1», если справочника с ключом Key отсутствует в коллекции.
