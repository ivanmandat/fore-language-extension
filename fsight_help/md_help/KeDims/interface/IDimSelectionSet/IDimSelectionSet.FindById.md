# IDimSelectionSet.FindById


## Синтаксис


FindById(Id: String): [IDimSelection](../IDimSelection/IDimSelection.htm);


## Параметры


Id — идентификатор
 справочника, отметку которого необходимо получить.


## Описание


Метод FindById осуществляет
 поиск и возвращает объект, содержащий отметку справочника. Поиск осуществляется
 по идентификатору справочника, передаваемому посредством параметра Id.
