# IStandardDimIndexes


Сборка: Dimensions;


## Описание


Интерфейс IStandardDimIndexes
 содержит свойства и методы коллекции индексов табличного справочника.


## Иерархия наследования


IStandardDimIndexes


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IStandardDimIndexes.Count.htm)
		 Свойство Count возвращает
		 количество индексов табличного справочника.


		 ![](../../Property_Image.gif)
		 [Item](IStandardDimIndexes.Item.htm)
		 Свойство Item возвращает
		 индекс табличного справочника.


		 ![](../../Property_Image.gif)
		 [PrimaryIndex](IStandardDimIndexes.PrimaryIndex.htm)
		 Свойство PrimaryIndex
		 возвращает первичный индекс табличного справочника.


		 ![](../../Property_Image.gif)
		 [SelectionIndex](IStandardDimIndexes.SelectionIndex.htm)
		 Свойство SelectionIndex
		 возвращает индекс, который используется для формирования отметки.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IStandardDimIndexes.Add.htm)
		 Метод Add осуществляет
		 добавление индекса в табличный справочник.


		 ![](../../Sub_Image.gif)
		 [Clear](IStandardDimIndexes.Clear.htm)

		 все индексы табличного справочника.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IStandardDimIndexes.FindByKey.htm)

		 поиск и возвращает объект, содержащий индекс табличного справочника.


		 ![](../../Sub_Image.gif)
		 [Remove](IStandardDimIndexes.Remove.htm)
		 Метод Remove осуществляет
		 удаление индекса табличного справочника по номеру.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IStandardDimIndexes.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
