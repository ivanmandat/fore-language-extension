# ICalendarDimension.AllowedLevel


## Синтаксис


AllowedLevel(LevelIndex: Integer): Boolean;


## Параметры


LevelIndex — индекс уровня.


## Описание


Свойство AllowedLevel возвращает True, если уровень с индексом LevelIndex может использоваться в календарном справочнике.
