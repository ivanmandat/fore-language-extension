# ICalendarDimension.AllowedBlock


## Синтаксис


AllowedBlock(LevelIndex: Integer; BlockIndex: Integer): Boolean;


## Параметры


LevelIndex — индекс уровня.


BlockIndex — индекс блока.


## Описание


Свойство AllowedBlock возвращает True, если уровень, индекс которого передается посредством параметра LevelIndex, может использоваться в качестве блока в календарном справочнике.
