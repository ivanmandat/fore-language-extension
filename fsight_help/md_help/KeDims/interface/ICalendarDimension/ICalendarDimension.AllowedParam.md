# ICalendarDimension.AllowedParam


## Синтаксис


AllowedParam(ParamIndex: Integer): Boolean;


## Параметры


ParamIndex — индекс параметра.


## Описание


Свойство AllowedParam возвращает True, если параметр, индекс которого передается посредством параметра ParamIndex, может использоваться в календарном справочнике.
