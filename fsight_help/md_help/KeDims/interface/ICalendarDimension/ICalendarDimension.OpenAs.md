# ICalendarDimension.OpenAs


## Синтаксис


OpenAs(Type: [CalendarOpenType](../../Enums/CalendarOpenType.htm)):
 [IDimInstance](../IDimInstance/IDimInstance.htm);


## Параметры


Type. Вид элементов календаря.


## Описание


Метод OpenAs открывает
 календарь в указанном виде.


## Комментарии


Если Type = CalendarOpenType.Default,
 то календарь открывается в виде, указанном в свойстве [ICalendarDimension.DefaultOpenType](ICalendarDimension.DefaultOpenType.htm).
