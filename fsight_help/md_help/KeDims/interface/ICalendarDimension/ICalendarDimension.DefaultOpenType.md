# ICalendarDimension.DefaultOpenType


## Синтаксис


DefaultOpenType: [CalendarOpenType](../../Enums/CalendarOpenType.htm);


## Описание


Свойство DefaultOpenType определяет
 вид элементов, который используется по умолчанию при открытии календаря.


## Комментарии


Для открытия элементов календаря в требуемом виде используйте метод
 [ICalendarDimension.OpenAs](ICalendarDimension.OpenAs.htm).
