# ICalendarDimension.AllowedHierarchy


## Синтаксис


AllowedHierarchy(HierarchyIndex: Integer): Boolean;


## Параметры


HierarchyIndex — индекс иерархии.


## Описание


Свойство AllowedHierarchy возвращает True, если тип иерархии, передаваемый посредством параметра HierarchyIndex, может использоваться в календарном справочнике.
