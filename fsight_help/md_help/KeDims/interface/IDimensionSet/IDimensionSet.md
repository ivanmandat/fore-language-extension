# IDimensionSet


## Описание


Интерфейс IDimensionSet содержит
 свойства и методы коллекции обязательных измерений базы данных временных
 рядов и собственных измерений самих показателей.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDimensionSet.Count.htm)
		 Свойство Count возвращает
		 количество измерений.


		 ![](../../Property_Image.gif)
		 [Item](IDimensionSet.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий структуру измерения.


## Методы


		 Имя метода
		 Краткое описание
