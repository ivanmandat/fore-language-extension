# IDimensionSet.Add


## Синтаксис


Add(Dim: [IDimensionModel](../IDimensionModel/IDimensionModel.htm)):
 Integer;


## Параметры


Dim - справочник,
 которой необходимо добавить в список.


## Описание


Метод Add осуществляет добавление
 справочника в список измерений, и возвращает индекс в случае удачного
 добавления.
