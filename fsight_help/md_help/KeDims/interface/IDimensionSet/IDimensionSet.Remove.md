# IDimensionSet.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index — индекс
 измерения.


## Описание


Метод Remove осуществляет удаление
 измерения, индекс которого передается посредством параметра Index,
 и возвращает True в случае удачного
 удаления.
