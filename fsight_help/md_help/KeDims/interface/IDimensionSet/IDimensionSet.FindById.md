# IDimensionSet.FindById


## Синтаксис


FindById(Id: String): [IDimensionModel](../IDimensionModel/IDimensionModel.htm);


## Параметры


Id — идентификатор
 измерения, по которому осуществляется поиск.


## Описание


Метод FindById осуществляет
 поиск по идентификатору, и в случае удачного поиска возвращает объект,
 содержащий структуру справочника.
