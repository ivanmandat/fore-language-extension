# IDimensionSet.RemoveById


## Синтаксис


RemoveById(Id: String): Boolean;


## Параметры


Id — идентификатор
 измерения.


## Описание


Метод RemoveById осуществляет
 удаление измерения, идентификатор которого передается посредством параметра
 Id, и возвращает True
 в случае удачного удаления.
