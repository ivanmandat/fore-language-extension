# IDimensionSet.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IDimensionModel](../IDimensionModel/IDimensionModel.htm);


## Параметры


Key — ключ
 измерения, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск по ключу, и в случае удачного поиска возвращает объект, содержащий
 структуру справочника.
