# IDimOrder


## Описание


Интерфейс IDimOrder содержит
 свойства атрибута, по которому производится сортировка справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attribute](IDimOrder.Attribute.htm)
		 Свойство Attribute
		 возвращает свойства атрибута, по которому осуществляется сортировка
		 справочника.


		 ![](../../Property_Image.gif)
		 [ReverseOrder](IDimOrder.ReverseOrder.htm)
		 Свойство ReverseOrder
		 определяет направление сортировки по данному атрибуту.
