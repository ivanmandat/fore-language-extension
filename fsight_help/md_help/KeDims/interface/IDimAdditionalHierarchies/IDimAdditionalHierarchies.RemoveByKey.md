# IDimAdditionalHierarchies.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ дополнительной альтернативной
 иерархии.


## Описание


Метод RemoveByKey удаляет дополнительную
 альтернативную иерархию по ее ключу.


## Комментарии


Если дополнительная альтернативная иерархия будет удалена, то метод
 вернет значение True.


Для удаления дополнительной альтернативной иерархии по ее идентификатору
 используйте [IDimAdditionalHierarchies.RemoveById](IDimAdditionalHierarchies.RemoveById.htm).


Для удаления дополнительной альтернативной иерархии по ее индексу используйте
 [IDimAdditionalHierarchies.Remove](IDimAdditionalHierarchies.Remove.htm).


Для добавления дополнительной альтернативной иерархии в коллекцию используйте
 [IDimAdditionalHierarchies.Add](IDimAdditionalHierarchies.Add.htm).
