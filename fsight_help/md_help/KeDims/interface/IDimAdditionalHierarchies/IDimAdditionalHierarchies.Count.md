# IDimAdditionalHierarchies.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 дополнительных альтернативных иерархий в коллекции.


## Комментарии


Для получения дополнительной альтернативную иерархи по ее индексу используйте
 [IDimAdditionalHierarchies.Item](IDimAdditionalHierarchies.Item.htm).
