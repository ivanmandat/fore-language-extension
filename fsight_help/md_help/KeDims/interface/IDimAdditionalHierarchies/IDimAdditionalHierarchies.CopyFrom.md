# IDimAdditionalHierarchies.CopyFrom


## Синтаксис


CopyFrom(Source: [IDimAdditionalHierarchies](IDimAdditionalHierarchies.htm));


## Параметры


Source. Коллекция дополнительных
 альтернативных иерархий.


## Описание


Метод CopyFrom копирует коллекцию
 дополнительных иерархий из одного измерения в другое.


## Комментарии


Для добавления дополнительной альтернативной иерархии в коллекцию используйте
 [IDimAdditionalHierarchies.Add](IDimAdditionalHierarchies.Add.htm).
