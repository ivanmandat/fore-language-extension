# IDimAdditionalHierarchies.FidById


## Синтаксис


FidById(Id: String): [IDimHierarchyInstance](../IDimHierarchyInstance/IDimHierarchyInstance.htm);


## Параметры


Id. Идентификатор дополнительной
 альтернативной иерархии.


## Описание


Метод FidById осуществляет поиск
 дополнительной альтернативной иерархии по ее идентификатору.


## Комментарии


Для поиска дополнительной альтернативной иерархии по ее ключу во всех
 коллекциях используйте [IDimAdditionalHierarchies.FindByAllColl](IDimAdditionalHierarchies.FindByAllColl.htm).


Для поиска дополнительной альтернативной иерархии по ее ключу используйте
 [IDimAdditionalHierarchies.FindByKey](IDimAdditionalHierarchies.FindByKey.htm).
