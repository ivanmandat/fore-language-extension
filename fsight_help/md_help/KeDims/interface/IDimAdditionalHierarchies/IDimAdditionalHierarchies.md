# IDimAdditionalHierarchies


Сборка: Dimensions;


## Описание


Интерфейс IDimAdditionalHierarchies
 используется для работы с дополнительными альтернативными иерархиями.


## Иерархия наследования


IDimAdditionalHierarchies


## Комментарии


Для получения объекта данного типа используйте свойство [IDimSelection.AdditionalHierarchies](../IDimSelection/IDimSelection.AdditionalHierarchies.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDimAdditionalHierarchies.Count.htm)
		 Свойство Count возвращает
		 количество дополнительных альтернативных иерархий в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IDimAdditionalHierarchies.Item.htm)
		 Свойство Item возвращает
		 дополнительную альтернативную иерархию по ее индексу.
