# IDimAdditionalHierarchies.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IDimHierarchyInstance](../IDimHierarchyInstance/IDimHierarchyInstance.htm);


## Параметры


Key. Ключ дополнительной альтернативной
 иерархии.


## Описание


Метод FindByKey осуществляет
 поиск дополнительной альтернативной иерархии по ее ключу.


## Комментарии


Для поиска дополнительной альтернативной иерархии по ее ключу во всех
 коллекциях используйте [IDimAdditionalHierarchies.FindByAllColl](IDimAdditionalHierarchies.FindByAllColl.htm).


Для поиска дополнительной альтернативной иерархии по ее идентификатору
 используйте [IDimAdditionalHierarchies.FidById](IDimAdditionalHierarchies.FidById.htm).
