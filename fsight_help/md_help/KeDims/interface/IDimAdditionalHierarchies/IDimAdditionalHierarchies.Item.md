# IDimAdditionalHierarchies.Item


## Синтаксис


Item(Index: Integer): [IDimHierarchyInstance](../IDimHierarchyInstance/IDimHierarchyInstance.htm);


## Параметры


Index. Индекс дополнительной
 альтернативной иерархии в коллекции.


## Описание


Свойство Item возвращает дополнительную
 альтернативную иерархию по ее индексу.


## Комментарии


Для получения количества дополнительных альтернативных иерархий в коллекции
 используйте [IDimAdditionalHierarchies.Count](IDimAdditionalHierarchies.Count.htm).
