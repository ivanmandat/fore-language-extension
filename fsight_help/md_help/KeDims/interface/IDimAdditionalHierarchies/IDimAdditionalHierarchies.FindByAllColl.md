# IDimAdditionalHierarchies.FindByAllColl


## Синтаксис


FindByAllColl(Key: Integer): [IDimHierarchyInstance](../IDimHierarchyInstance/IDimHierarchyInstance.htm);


## Параметры


Key. Ключ дополнительной альтернативной
 иерархии.


## Описание


Метод FindByAllColl осуществляет
 поиск дополнительной альтернативной иерархии по ее ключу во всех коллекциях.


## Комментарии


Для поиска дополнительной альтернативной иерархии по ее идентификатору
 используйте [IDimAdditionalHierarchies.FidById](IDimAdditionalHierarchies.FidById.htm).


Для поиска дополнительной альтернативной иерархии по ее ключу используйте
 [IDimAdditionalHierarchies.FindByKey](IDimAdditionalHierarchies.FindByKey.htm).
