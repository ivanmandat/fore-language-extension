# IDimOrmManager.CreateCondition


## Синтаксис


CreateCondition(Dimension: [IDimensionModel](../IDimensionModel/IDimensionModel.htm);
 Criteria: [IDimTextCriteria](../IDimTextCriteria/IDimTextCriteria.htm)):
 [IDimOrmCondition](../IDimOrmCondition/IDimOrmCondition.htm);


## Параметры


Dimension - справочник, по
 элементам которого будет осуществляться поиск.


Criteria - критерий поиска
 элементов.


## Описание


Метод CreateCondition создает
 объект для работы с условиями поиска элементов справочника.
