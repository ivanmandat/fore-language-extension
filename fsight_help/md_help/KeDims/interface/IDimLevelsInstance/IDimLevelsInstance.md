# IDimLevelsInstance


## Описание


Интерфейс IDimLevelsInstance
 содержит свойства и методы коллекции уровней элементов справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDimLevelsInstance.Count.htm)
		 Свойство Count возвращает
		 количество уровней элементов справочника.


		 ![](../../Property_Image.gif)
		 [Item](IDimLevelsInstance.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий уровень элементов справочника.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindByKey](IDimLevelsInstance.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск уровня элементов по ключу.
