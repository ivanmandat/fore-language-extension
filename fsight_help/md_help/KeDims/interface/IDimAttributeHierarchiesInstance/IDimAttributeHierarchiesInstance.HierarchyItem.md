# IDimAttributeHierarchiesInstance.HierarchyItem


## Синтаксис


HierarchyItem(Index: Integer): [IDimAttributeHierarchyInstance](../IDimAttributeHierarchyInstance/IDimAttributeHierarchyInstance.htm);


## Параметры


Index. Индекс альтернативной
 иерархии.


## Описание


Свойство HierarchyItem возвращает
 данные альтернативной иерархии атрибута.


## Комментарии


Для поиска данных альтернативной иерархии атрибута по его уровню используйте
 [IDimAttributeHierarchiesInstance.FindByAttributeLevel](IDimAttributeHierarchiesInstance.FindByAttributeLevel.htm).
