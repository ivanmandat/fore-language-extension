# DimAttributesList.Create


## Синтаксис


Create(Dimension: [IDimensionModel](../../interface/IDimensionModel/IDimensionModel.htm);
 Attributes: String);


## Параметры


Dimension - справочник, в котором
 будет осуществляться поиск.


Attributes - атрибуты справочника,
 по значениям которых будет осуществляться поиск. Идентификаторы атрибутов
 разделяются знаком «;».


## Описание


Конструктор Create создает новую
 коллекцию атрибутов справочника, по значениям которых будет осуществляться
 поиск элементов.
