# DimAttributesList


Сборка: Dimensions;


## Описание


Класс DimAttributesList реализует
 коллекцию атрибутов справочника, по которым будет осуществляться поиск
 элементов.


## Конструкторы


		 Имя конструктора
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Create](DimAttributesList.Create.htm)
		 Конструктор Create
		 создает новую коллекцию атрибутов справочника, по значениям которых
		 будет осуществляться поиск элементов.
