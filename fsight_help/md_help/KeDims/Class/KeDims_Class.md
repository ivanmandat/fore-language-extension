# Классы сборки Dimensions


		 Класс
		 Краткое описание


		 ![](../Class_Image.gif)
		 [CalendarDimInstanceFactory](CalendarDimInstanceFactory/CalendarDimInstanceFactory.htm)
		 Класс CalendarDimInstanceFactory
		 реализует объект, используемый для навигации по иерархии элементов
		 календарного справочника.


		 ![](../Class_Image.gif)
		 [DimAttributesList](DimAttributesList/DimAttributesList.htm)
		 Класс DimAttributesList
		 реализует коллекцию атрибутов справочника, по которым будет осуществляться
		 поиск элементов.


		 ![](../Class_Image.gif)
		 [DimHierarchyCreator](DimHierarchyCreator/DimHierarchyCreator.htm)
		 Класс DimHierarchyCreator
		 реализует объект, используемый для создания группировочных элементов
		 измерений в виде альтернативной иерархии справочника.


		 ![](../Class_Image.gif)
		 [DimOrmManager](DimOrmManager/DimOrmManager.htm)
		 Класс DimOrmManager
		 реализует объект, используемый для работы с условиями.


		 ![](../Class_Image.gif)
		 [DimSelectionSetFactory](DimSelectionSetFactory/DimSelectionSetFactory.htm)
		 Класс DimSelectionSetFactory
		 реализует объект, содержащий отметку справочников.


		 ![](../Class_Image.gif)
		 [DimTextCriteria](DimTextCriteria/DimTextCriteria.htm)
		 Класс DimTextCriteria
		 реализует объект, предназначенный для создания критериев поиска.


		 ![](../Class_Image.gif)
		 [DynamicInstance](DynamicInstance/DynamicInstance.htm)
		 Класс DynamicInstance
		 реализует объект, предназначенный для работы с табличным справочником
		 НСИ с динамической загрузкой данных.
