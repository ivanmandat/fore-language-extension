# CalendarOpenType


## Описание


Перечисление CalendarOpenType
 содержит варианты открытия календаря.


Используется следующими свойствами и методами:


	- [ICalendarDimension.DefaultOpenType](../interface/ICalendarDimension/ICalendarDimension.DefaultOpenType.htm);


	- [ICalendarDimension.OpenAs](../interface/ICalendarDimension/ICalendarDimension.OpenAs.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Default. Вид по умолчанию,
		 заданный для календаря.


		 1
		 Flat. Элементы календаря
		 представлены в плоском виде.


		 2
		 Hierarchical. Элементы
		 календаря представлены в иерархическом виде.
