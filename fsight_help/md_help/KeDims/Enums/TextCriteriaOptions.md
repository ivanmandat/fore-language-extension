# TextCriteriaOptions


## Описание


Перечисление TextCriteriaOptions
 содержит параметры поиска.


Используется следующим свойством:


	- [IDimTextCriteria.CriteriaOptions](../interface/IDimTextCriteria/IDimTextCriteria.CriteriaOptions.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Параметры не
		 определены.


		 1
		 SearchName. Искать
		 в наименовании.


		 2
		 SearchID. Искать в
		 идентификаторе.


		 4
		 SearchStringAttributes.
		 Искать во всех строковых атрибутах.


		 8
		 SearchAllAttributes.
		 Искать во всех атрибутах.


		 16
		 SearchProbeAttributes.
		 Искать в определенных атрибутах. Поиск будет осуществлен по атрибутам,
		 для которых метод [IDimAttributeProbe.Probe](../interface/IDimAttributeProbe/IDimAttributeProbe.Probe.htm)
		 возвращает True.


		 256
		 FreeText. Искать свободный
		 текст. При сравнении ищется вхождение строки X в строку Y. Если
		 строка состоит из нескольких слов, то ищется вхождение всей фразы.
		 При поиске вхождений отбрасываются окончания существительных.
		 Например, для слов «Доход», «Доходы», «Доходов», «О доходах» будет
		 искаться вхождение «Доход».


		 512
		 LikeText. Искать
		 текст по маске с помощью символов подстановок «%» и «_»:


			- используйте символ «%» для поиска строки от нуля и более
			 символов. Например, по запросу «%область%» будут найдены все
			 сочетания, где до и после слова «область» есть какой-либо
			 текст и/или отдельное слово «область».


			-  используйте символ «_» для поиска по одному символу.
			 Например, по запросу «област_» будут найдены все слова
			 из семи букв, начинающиеся на «област».


		 4096
		 MatchCase. Искать с
		 учетом регистра.


		 8192
		 WholeWord. Искать слово
		 целиком (все слова разделенные пробелом).


		 16384
		 ForEachWord. Искать
		 вхождения как по каждому слову в отдельности, так и по вхождению
		 всех слов одновременно. Если строка состоит из нескольких слов,
		 то ищутся вхождения как по каждому слову в отдельности, так и
		 по вхождению всех слов. При поиске вхождений отбрасываются окончания
		 существительных.

Например, при поиске фразы «атрибуты справочника»
		 будут найдены элементы, содержащие как «атрибуты справочника»,
		 так и «атрибуты», «атрибут», «справочника», «справочник».
