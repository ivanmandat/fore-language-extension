# DimBlockType


## Описание


Перечисление DimBlockType содержит типы блока табличного справочника.


Используется следующими свойствами и методами:


-
[IStandardDimBlock.BlockType](../interface/IStandardDimBlock/IStandardDimBlock.BlockType.htm);


-
[IStandardDimBlocks.Add](../interface/IStandardDimBlocks/IStandardDimBlocks.Add.htm).


## Допустимые значения


 Значение
 Краткое описание


 1
 NotRecursive. Нерекурсивный блок.


 2
 Recursive. Рекурсивный блок.
