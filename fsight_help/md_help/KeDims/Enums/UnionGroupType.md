# UnionGroupType


## Описание


Перечисление UnionGroupType содержит типы взаимодействия между собой нескольких примитивов группы элементов.


Используется следующим свойством:


-
[IDimElementGroup.UnionType](../interface/IDimElementGroup/IDimElementGroup.UnionType.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Union. Объединение примитивов, установлено по умолчанию.


 1
 Intersection. Пересечение примитивов.
