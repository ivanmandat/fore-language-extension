# DimRelation


## Описание


Перечисление DimRelation содержит позиции вставки вычисляемого элемента.


Используется следующими свойствами и методами:


-
[IPivotEvaluatorDimCalculatesItem.AddCalculatedElement](KePivot.chm::/Interface/IPivotEvaluatorDimCalculatesItem/IPivotEvaluatorDimCalculatesItem.AddCalculatedElement.htm);


-
[IPivotEvaluatorDimCalculatesItem.Relation](KePivot.chm::/Interface/IPivotEvaluatorDimCalculatesItem/IPivotEvaluatorDimCalculatesItem.Relation.htm).
