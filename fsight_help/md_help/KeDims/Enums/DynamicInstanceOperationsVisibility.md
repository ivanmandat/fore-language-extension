# DynamicInstanceOperationsVisibility


## Описание


Перечисление DynamicInstanceOperationsVisibility
 содержит варианты распространения изменений в справочниках.


Используется следующими свойствами и методами:


	- [IDynamicInstance.CanClone](../interface/IDynamicInstance/IDynamicInstance.CanClone.htm);


	- [IDynamicInstance.Clone](../interface/IDynamicInstance/IDynamicInstance.Clone.htm);


	- [IDynamicInstance.InitInstance](../interface/IDynamicInstance/IDynamicInstance.InitInstance.htm);


	- [IDynamicInstance.OperationsVisibility](../interface/IDynamicInstance/IDynamicInstance.OperationsVisibility.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Local. Изменения будут
		 распространяться только на указанный справочник.


		 1
		 Global. Изменения будут
		 распространяться на все дочерние справочники.


		 2
		 PreSet. Изменения будут
		 распространяться на заданное множество справочников.
