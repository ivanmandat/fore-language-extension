# SelectionGroupType


## Описание


Перечисление SelectionGroupType содержит типы примитива группы элементов.


Используется следующими свойствами и методами:


-
[IDimElementGroupPrimitive.Type](../interface/IDimElementGroupPrimitive/IDimElementGroupPrimitive.Type.htm);


-
[IDimElementGroup.AddPrimitive](../interface/IDimElementGroup/IDimElementGroup.AddPrimitive.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 List. Список. Оставлять элементы по списку.


 1
 Children. Оставлять дочерние элементы.


 2
 ChildrenWithParent. Элементы по общему владельцу. Оставлять дочерние элементы с родителем.


 3
 Levels. Элементы уровня. Оставлять элементы уровня.


 4
 Macro. Использование прикладного макроса.
