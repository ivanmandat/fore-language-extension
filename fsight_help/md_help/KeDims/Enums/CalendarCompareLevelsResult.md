# CalendarCompareLevelsResult


## Описание


Перечисление CalendarCompareLevelsResult
 используется для определения результата сравнения уровней календарного
 справочника.


Используется следующим методом:


	- [ICalendarDimension.CompareLevels](../interface/ICalendarDimension/ICalendarDimension.CompareLevels.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 -2
		 NotApplicable. Сравнение
		 не применимо, так как в календарном справочнике нет указанного
		 типа уровня.


		 -1
		 Less. Уровень 1 находится
		 ниже по иерархии, чем уровень 2.


		 0
		 Equal. Уровни 1 и 2
		 календарного справочника располагаются на одном уровне иерархии.


		 1
		 Greater. Уровень 1
		 находится выше по иерархии, чем уровень 2.
