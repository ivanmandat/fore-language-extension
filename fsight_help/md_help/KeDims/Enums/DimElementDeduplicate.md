# DimElementDeduplicate


## Описание


Перечисление DimElementDeduplicate содержит методы отбора элементов при удалении дубликатов в справочнике.


Используется следующим свойством:


-
[IStandardDimension.DeduplicateCondition](../interface/IStandardDimension/IStandardDimension.DeduplicateCondition.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Оставлять дубликаты.


 1
 Min. По минимальному значению выражения.


 2
 Max. По максимальному значению выражения.
