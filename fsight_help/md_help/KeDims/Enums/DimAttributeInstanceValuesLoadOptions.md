# DimAttributeInstanceValuesLoadOptions


## Описание


Перечисление DimAttributeInstanceValuesLoadOptions
 содержит опции загрузки значений атрибута.


Используется следующими свойствами и методами:


	- [IDimAttributesInstance.EnsureValuesLoaded](../interface/IDimAttributesInstance/IDimAttributesInstance.EnsureValuesLoaded.htm);


	- [IDimAttributeInstanceValuesLoaderArguments.Options](../interface/IDimAttributeInstanceValuesLoaderArguments/IDimAttributeInstanceValuesLoaderArguments.Options.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Загрузка всех
		 значений.


		 1
		 Value. Загрузка реального
		 значения атрибута.


		 2
		 DisplayValue. Загрузка
		 отображаемого значения атрибута.
