# SelectionPrimitiveType


## Описание


Перечисление SelectionPrimitiveType
 содержит типы примитива схемы отметки.


Используется следующими свойствами и методами:


	- [IDimSelectionSchemaPrimitive.Type](../interface/IDimSelectionSchemaPrimitive/IDimSelectionSchemaPrimitive.Type.htm);


	- [IDimSelectionSchema.Add](../interface/IDimSelectionSchema/IDimSelectionSchema.Add.htm).


## Комментарии


При работе с [составными
 справочниками](UiNavObj.chm::/reference_book/Master_Composite_Table_reference_book/Master_CompositeTable.htm) доступен только примитив «Отметить
 все элементы».


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 All. Отметить все элементы.


		 1
		 Specified. Отметить
		 элементы по списку.


		 2
		 SpecifiedChildren.
		 Отметить подчиненные элементы.


		 3
		 Level. Отметить элементы
		 уровня.


		 4
		 SelectedSiblings. Отметить
		 весь уровень.


		 5
		 SelectedAncestors.
		 Отметить родительские элементы.


		 6
		 SelectedChildren. Отметить
		 дочерние элементы.


		 7
		 Macro. Использование
		 прикладного макроса.


		 8
		 Invert. Инвертировать
		 отметку.


		 9
		 DefaultElement. Отметить
		 элемент по умолчанию.
