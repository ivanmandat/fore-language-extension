# DimCompoundDimensionType


## Описание


Перечисление DimCompoundDimensionType
 используется для получения типа составного измерения.


Используется следующим свойством:


	- [ICompoundDimensionInstance.Type](../interface/ICompoundDimensionInstance/ICompoundDimensionInstance.Type.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 Compound. Простое составное
		 измерение.


		 2
		 MetaCompound. Составное
		 измерение метаиерархии.


		 3
		 SideHeaderCompound.
		 Измерение боковика.
