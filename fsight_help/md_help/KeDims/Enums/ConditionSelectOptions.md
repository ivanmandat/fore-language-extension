# ConditionSelectOptions


## Описание


Перечисление ConditionSelectOptions содержит варианты установок поиска.


Используется следующим свойством:


-
[IDimConditionSelect.SelectOptions](../interface/IDimConditionSelect/IDimConditionSelect.SelectOptions.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Производить поиск по всем элементам.


 1
 SelectedOnly. Проверять только отмеченные элементы.
