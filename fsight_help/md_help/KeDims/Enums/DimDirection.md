# DimDirection


## Описание


Перечисление DimDirection содержит
 варианты расположения элемента при добавлении или после переноса.


Используется следующими методами:


	- [IDynamicInstance.Add](../interface/IDynamicInstance/IDynamicInstance.Add.htm);


	- [IDynamicInstance.Move](../interface/IDynamicInstance/IDynamicInstance.Move.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Before. Элемент будет
		 расположен перед указанным элементом.


		 1
		 After. Элемент будет
		 расположен после указанного элемента.


		 2
		 FirstChild. Элемент
		 будет первым в списке дочерних элементов.


		 3
		 LastChild. Элемент
		 будет последним в списке дочерних элементов.


		 4
		 Swap. Указанный элемент
		 будет заменен новым.
