# IReportSheetData.OwnParamValues


## Синтаксис


OwnParamValues: [IMetabaseObjectParamValues](KeSom.chm::/Interface/IMetabaseObjectParamValues/IMetabaseObjectParamValues.htm);


## Описание


Свойство OwnParamValues возвращает
 коллекцию значений параметров отчета, заданных пользователем.


## Комментарии


Для получения коллекции значений параметров источника данных отчета,
 используйте свойство [IReportSheetData.DataSourceParamValues](IReportSheetData.DataSourceParamValues.htm).
