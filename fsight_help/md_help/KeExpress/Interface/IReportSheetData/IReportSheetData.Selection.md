# IReportSheetData.Selection


## Синтаксис

## Описание


Свойство Selection возвращает
 отметку элементов измерения отчета.


## Комментарии


Отметка элементов измерения предназначена для выбора данных, отображаемых
 в таблице, диаграмме и других представлениях.
