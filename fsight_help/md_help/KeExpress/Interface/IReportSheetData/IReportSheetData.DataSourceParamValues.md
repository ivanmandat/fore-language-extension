# IReportSheetData.DataSourceParamValues


## Синтаксис


DataSourceParamValues: [IMetabaseObjectParamValues](KeSom.chm::/Interface/IMetabaseObjectParamValues/IMetabaseObjectParamValues.htm);


## Описание


Свойство DataSourceParamValues
 возвращает коллекцию значений параметров источника данных отчета.


## Комментарии


Для получения коллекции значений параметров отчета, заданных пользователем,
 используйте свойство [IReportSheetData.OwnParamValues](IReportSheetData.OwnParamValues.htm).
