# IReportSheetData.DataSources


## Синтаксис


DataSources: [IReportDataSourcesData](../IReportDataSourcesData/IReportDataSourcesData.htm);


## Описание


Свойство DataSources возвращает
 коллекцию источников данных листа отчета.


## Комментарии


В качестве источников данных можно использовать:


	- все виды кубов;


	- базы данных временных рядов;


	- реляционные объекты: таблицы, запросы, представления, присоединенные
	 таблицы, источники данных ODBC;


	- экспресс-отчеты;


	- файлы;


	- другие источники: переменные моделирования и файловые кубы (файлы
	 *.ppcube).
