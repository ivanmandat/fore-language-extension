# IReportSheetData


Сборка: Express;


## Описание


Интерфейс IReportSheetData предназначен
 для работы с данными листа экспресс-отчета.


## Иерархия наследования


           IReportSheetData


## Комментарии


Для получения объекта данного типа используйте свойство [IReportData.SheetData](../IReportData/IReportData.SheetData.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DataSourceParamValues](IReportSheetData.DataSourceParamValues.htm)
		 Свойство DataSourceParamValues
		 возвращает коллекцию значений параметров источника данных отчета.


		 ![](../../Property_Image.gif)
		 [DataSources](IReportSheetData.DataSources.htm)
		 Свойство DataSources
		 возвращает коллекцию источников данных листа отчета.


		 ![](../../Property_Image.gif)
		 [Descriptor](IReportSheetData.Descriptor.htm)
		 Свойство Descriptor
		 возвращает описание листа отчета в качестве объекта репозитория.


		 ![](../../Property_Image.gif)
		 [Id](IReportSheetData.Id.htm)
		 Свойство Id возвращает
		 идентификатор листа отчета.


		 ![](../../Property_Image.gif)
		 [Name](IReportSheetData.Name.htm)
		 Свойство Name возвращает
		 наименование листа отчета.


		 ![](../../Property_Image.gif)
		 [OwnParamValues](IReportSheetData.OwnParamValues.htm)
		 Свойство OwnParamValues
		 возвращает коллекцию значений параметров отчета, заданных пользователем.


		 ![](../../Property_Image.gif)
		 [Selection](IReportSheetData.Selection.htm)
		 Свойство Selection
		 возвращает отметку элементов измерения отчета.
