# IEaxPivotDimensionInstance.PivotDimension


## Синтаксис


PivotDimension: [IPivotDimension](KePivot.chm::/Interface/IPivotDimension/IPivotDimension.htm);


## Описание


Свойство PivotDimension возвращает
 измерение таблицы, построенной на многомерном источнике данных.


## Комментарии


Для получения среза таблицы, построенной на многомерном источнике, используйте
 [IEaxPivotDimensionInstance.PivotSlice](IEaxPivotDimensionInstance.PivotSlice.htm).
