# IEaxPivotDimensionInstance


Сборка: Express;


## Описание


Интерфейс IEaxPivotDimensionInstance
 используется для работы с данными измерения таблицы.


## Иерархия наследования


           [IEaxDimensionDescriptor](../IEaxDimensionDescriptor/IEaxDimensionDescriptor.htm)


           [IEaxDimensionInstance](../IEaxDimensionInstance/IEaxDimensionInstance.htm)


           IEaxPivotDimensionInstance


## Комментарии


Для получения объекта данного типа необходимо объект типа [IEaxDimensionInstance](../IEaxDimensionInstance/IEaxDimensionInstance.htm)
 привести к IEaxPivotDimensionInstance.
