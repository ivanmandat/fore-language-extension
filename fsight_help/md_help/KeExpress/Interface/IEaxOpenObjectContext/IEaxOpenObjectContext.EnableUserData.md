# IEaxOpenObjectContext.EnableUserData


## Синтаксис


		EnableUserData: Boolean;


## Описание


Свойство EnableUserData определяет,
 будут ли применяться пользовательские настройки при открытии регламентного
 отчёта.


## Комментарии


Если свойству установлено значение True,
 то в свойстве [ExternalXML](IEaxOpenObjectContext.ExternalXML.htm)
 могут быть заданы пользовательские настройки. Эти настройки будут переданы
 в отчёт при открытии.
