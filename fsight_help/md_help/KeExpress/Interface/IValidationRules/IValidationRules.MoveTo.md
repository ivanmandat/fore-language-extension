# IValidationRules.MoveTo


## Синтаксис


MoveTo(FromIndex: Integer; ToIndex: Integer);


## Параметры


FromIndex. Индекс позиции правила,
 которое необходимо переместить;


ToIndex. Индекс позиции, в
 которую необходимо переместить правило.


## Описание


Метод MoveTo осуществляет перемещение
 правила в коллекции.
