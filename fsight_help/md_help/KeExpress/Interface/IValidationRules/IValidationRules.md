# IValidationRules


Сборка: Express;


## Описание


Интерфейс IValidationRules содержит
 свойства и методы для работы с коллекцией правил валидации.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IValidationRules.Count.htm)
		 Свойство Count возвращает
		 количество правил в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IValidationRules.Item.htm)
		 Свойство Item возвращает
		 правило из коллекции по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindByKey](IValidationRules.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск правила в коллекции по ключу.


		 ![](../../Sub_Image.gif)
		 [MoveTo](IValidationRules.MoveTo.htm)
		 Метод MoveTo осуществляет
		 перемещение правила в коллекции.
