# IReportDataSourceData.Descriptor


## Синтаксис


Descriptor: [IMetabaseObjectDescriptor](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm);


## Описание


Свойство Descriptor возвращает
 описание источника данных отчета в качестве объекта репозитория.


## Комментарии


В качестве источников данных можно использовать:


	- все виды кубов;


	- базы данных временных рядов;


	- реляционные объекты: таблицы, запросы, представления, присоединенные
	 таблицы, источники данных ODBC;


	- экспресс-отчеты;


	- файлы;


	- другие источники: переменные моделирования и файловые кубы (файлы
	 *.ppcube).
