# IReportDataSourceData


Сборка: Express;


## Описание


Интерфейс IReportDataSourceData
 предназначен для работы с данными источника данных экспресс-отчета.


## Иерархия наследования


           IReportDataSourceData


## Комментарии


Для получения объекта данного типа используйте свойство [IReportDataSourcesData.Item](../IReportDataSourcesData/IReportDataSourcesData.Item.htm).
