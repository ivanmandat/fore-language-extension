# IDataAreaCommentManager


Сборка: Express;


## Описание


Интерфейс IDataAreaCommentManager
 содержит свойства и методы менеджера работы с комментариями.


## Иерархия наследования


IDataAreaCommentManager


## Комментарии


Менеджер работы с комментариями можно получить, используя свойство [IEaxDataArea.CommentManager](../IEaxDataArea/IEaxDataArea.CommentManager.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AllComments](IDataAreaCommentManager.AllComments.htm)
		 Свойство AllComments
		 возвращает все комментарии к ячейкам с данными, имеющиеся в указанной
		 таблице.


		 ![](../../Property_Image.gif)
		 [Comments](IDataAreaCommentManager.Comments.htm)
		 Свойство Comments возвращает
		 комментарии к ячейке с указанными координатами.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddComment](IDataAreaCommentManager.AddComment.htm)
		 Метод AddComment добавляет
		 комментарий для указанной ячейки с данными.


		 ![](../../Sub_Image.gif)
		 [DeleteComment](IDataAreaCommentManager.DeleteComment.htm)
		 Метод DeleteComment
		 у указанной ячейки с данными удаляет комментарий с заданным ключом.
