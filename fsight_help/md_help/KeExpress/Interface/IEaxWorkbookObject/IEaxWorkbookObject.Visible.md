# IEaxWorkbookObject.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет
 видимость объекта.


## Комментарии


Допустимые значения:


	- True. Объект видимый;


	- False. Объект скрытый.
