# IEaxWorkbookObject.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет,
 доступен ли объект.


## Комментарии


Допустимые значения:


	- True. Объект доступен;


	- False. Объект недоступен.
