# IEaxRangeFormatCondition


Сборка: Express;


## Описание


Интерфейс IEaxRangeFormatCondition
 предназначен для работы со стилем условного форматирования таблицы.


## Комментарии


Для работы со всеми стилями уловного форматирования, применяемыми в
 таблице, используйте [IEaxRangeFormatConditions](../IEaxRangeFormatConditions/IEaxRangeFormatConditions.htm).
