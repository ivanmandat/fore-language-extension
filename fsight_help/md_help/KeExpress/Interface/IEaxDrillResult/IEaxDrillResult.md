# IEaxDrillResult


Сборка: Express;


## Описание


Интерфейс IEaxDrillResult содержит
 свойства для получения информации о привязке к данным источника.


## Комментарии


Интерфейс является базовым для следующих интерфейсов:


	- [IEaxDrillCellResult](../IEaxDrillCellResult/IEaxDrillCellResult.htm)


	- [IEaxDrillPointResult](../IEaxDrillPointResult/IEaxDrillPointResult.htm)


	- [IEaxDrillRangeResult](../IEaxDrillRangeResult/IEaxDrillRangeResult.htm)


	- [IEaxDrillSerieResult](../IEaxDrillSerieResult/IEaxDrillSerieResult.htm)
