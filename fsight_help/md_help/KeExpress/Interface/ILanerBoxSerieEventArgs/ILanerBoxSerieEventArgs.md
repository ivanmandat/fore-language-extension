# ILanerBoxSerieEventArgs


Сборка: Express;


## Описание


Интерфейс ILanerBoxSerieEventArgs
 предназначен для работы с аргументами событий, происходящих с рядами рабочей
 книги в компоненте LanerBox.


## Комментарии


События, происходящие с рядами:


	- [LanerBox.OnAddSerie](../../Class/LanerBox/LanerBox.OnAddSerie.htm);


	- [LanerBox.OnMoveSerie](../../Class/LanerBox/LanerBox.OnMoveSerie.htm);


	- [LanerBox.OnRemoveSerie](../../Class/LanerBox/LanerBox.OnRemoveSerie.htm).
