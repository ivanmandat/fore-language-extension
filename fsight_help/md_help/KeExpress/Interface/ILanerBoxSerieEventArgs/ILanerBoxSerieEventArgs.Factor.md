# ILanerBoxSerieEventArgs.Factor


## Синтаксис


Factor: [IRubricatorFactor](KeCubes.chm::/Interface/IRubricatorFactor/IRubricatorFactor.htm);


## Описание


Свойство Factor возвращает
 показатель, на котором основан ряд, сгенерировавший событие.


## Комментарии


Используется только в событии [LanerBox.OnAddSerie](../../Class/LanerBox/LanerBox.OnAddSerie.htm).


Если свойство возвращает Null, то ряд основан на показателе, который
 ещё сохранён в каталоге.
