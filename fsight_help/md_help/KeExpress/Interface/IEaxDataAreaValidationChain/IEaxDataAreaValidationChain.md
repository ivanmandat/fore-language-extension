# IEaxDataAreaValidationChain


Сборка: Express;


## Описание


Интерфейс IEaxDataAreaValidationChain
 используется для работы с цепочкой валидаций в аналитической области данных.


## Иерархия наследования


           IEaxDataAreaValidationChain


## Комментарии


Для получения объекта данного типа используйте [IDiagnosticReport.Chain](../IDiagnosticReport/IDiagnosticReport.Chain.htm).
