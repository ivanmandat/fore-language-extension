# IEaxDataAreaValidationChain.Item


## Синтаксис


Item(Index: Integer): [IEaxDataAreaValidationLink](../IEaxDataAreaValidationLink/IEaxDataAreaValidationLink.htm);


## Параметры


Index. Индекс среза данных.


## Описание


Свойство Item возвращает ссылку
 между цепочкой валидаций диагностического отчета и коллекцией правил валидации
 аналитической области данных.


## Комментарии


Количество правил в цепочке диагностического отчета равно количеству
 правил валидаций в аналитической области данных.
