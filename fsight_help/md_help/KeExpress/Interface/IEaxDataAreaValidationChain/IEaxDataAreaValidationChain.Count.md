# IEaxDataAreaValidationChain.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 валидаций в цепочке.


## Комментарии


Для получения ссылки между цепочкой валидаций диагностического отчета
 и коллекцией правил валидации аналитической области данных используйте
 [IEaxDataAreaValidationChain.Item](IEaxDataAreaValidationChain.Item.htm).
