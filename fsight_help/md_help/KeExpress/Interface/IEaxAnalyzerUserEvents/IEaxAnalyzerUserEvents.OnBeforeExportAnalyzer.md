# IEaxAnalyzerUserEvents.OnBeforeExportAnalyzer


## Синтаксис


OnBeforeExportAnalyzer(Args: [IEaxBeforeExportEventArgs](../IEaxBeforeExportEventArgs/IEaxBeforeExportEventArgs.htm));


## Параметры


Args. Аргументы события.


## Описание


Метод OnBeforeExportAnalyzer
 реализует событие, наступающее перед экспортом отчета.


## Комментарии


Для реализации события, наступающего после экспорта отчета, используйте
 [IEaxAnalyzerUserEvents.OnAfterExportAnalyzer](IEaxAnalyzerUserEvents.OnAfterExportAnalyzer.htm).
