## Синтаксис

## Параметры

## Описание


Метод OnAfterExportAnalyzer
 реализует событие, наступающее после экспорта отчета.


## Комментарии


Для реализации события, наступающего перед экспортом отчета, используйте
 [IEaxAnalyzerUserEvents.OnBeforeExportAnalyzer](IEaxAnalyzerUserEvents.OnBeforeExportAnalyzer.htm).
