# IEaxAnalyzerUserEvents


Сборка: Express;


## Описание


Интерфейс IEaxAnalyzerUserEvents
 используется для реализации событий в обработчике событий экспресс-отчета.


## Иерархия наследования


           IEaxAnalyzerUserEvents


## Комментарии


Подключение обработчика событий к экспресс-отчету представлено в примере
 для [IEaxAnalyzerOptions.UseSharedEventHandler](../IEaxAnalyzerOptions/IEaxAnalyzerOptions.UseSharedEventHandler.htm).
