# IEaxAnalyzerUserEvents.OnAfterPrintAnalyzer


## Синтаксис


OnAfterPrintAnalyzer(Args: [IEaxAfterPrintEventArgs](../IEaxAfterPrintEventArgs/IEaxAfterPrintEventArgs.htm));


## Параметры


Args. Аргументы события.


## Описание


Метод OnAfterPrintAnalyzer реализует
 событие, наступающее после печати отчета.


## Комментарии


Для реализации события, наступающего перед печатью отчета, используйте
 [IEaxAnalyzerUserEvents.OnBeforePrintAnalyzer](IEaxAnalyzerUserEvents.OnBeforePrintAnalyzer.htm).
