# IEaxAnalyzerUserEvents.OnBeforePrintAnalyzer


## Синтаксис


OnBeforePrintAnalyzer(Args: [IEaxBeforePrintEventArgs](../IEaxBeforePrintEventArgs/IEaxBeforePrintEventArgs.htm));


## Параметры


Args. Аргументы события.


## Описание


Метод OnBeforePrintAnalyzer
 реализует событие, наступающее перед печатью отчета.


## Комментарии


Для реализации события, наступающего после печати отчета, используйте
 [IEaxAnalyzerUserEvents.OnAfterPrintAnalyzer](IEaxAnalyzerUserEvents.OnAfterPrintAnalyzer.htm).
