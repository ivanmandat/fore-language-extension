# IEaxSheets.RemoveByIndex


## Синтаксис


RemoveByIndex(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемого листа.


## Описание


Метод RemoveByIndex удаляет указанный лист, индекс которого передается посредством параметра Index.


## Комментарии


Метод возвращает True, в случае успешного удаления, в противном случае - False. При удалении листа также будет удален дочерний экспресс-отчет, который лежит в основе листа.
