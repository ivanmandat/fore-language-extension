# IEaxSheets.ItemByKey


## Синтаксис


ItemByKey(Key: Integer): [IEaxSheet](../IEaxSheet/IEaxSheet.htm);


## Параметры


Key. Ключ листа.


## Описание


Свойство ItemByKey возвращает параметры листа, ключ которого передается посредством параметра Key.


## Комментарии


Если листа с ключом Key не существует, то свойство вернет значение Null.
