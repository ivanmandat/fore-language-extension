# IEaxSheets.AddNew


## Синтаксис


AddNew: [IEaxSheet](../IEaxSheet/IEaxSheet.htm);


## Описание


Метод AddNew создает лист на
 базе нового экспресс-отчета.


## Комментарии


При выполнении метода AddNew
 для исходного экспресс-отчета будет добавлен дочерний объект, являющийся
 пустым экспресс-отчетом.
