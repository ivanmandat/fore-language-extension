# IEaxSheets.Remove


## Синтаксис


Remove(Value: [IEaxSheet](../IEaxSheet/IEaxSheet.htm)): Boolean;


## Параметры


Value. Лист, который необходимо удалить.


## Описание


Метод Remove удаляет указанный лист и возвращает признак удачного удаления.


## Комментарии


Метод возвращает True, в случае успешного удаления, в противном случае - False. При удалении листа также будет удален дочерний экспресс-отчет, который лежит в основе листа.
