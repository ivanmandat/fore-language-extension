# IEaxSheets.IndexOf


## Синтаксис


IndexOf(Value: [IEaxSheet](../IEaxSheet/IEaxSheet.htm)): Integer;


## Параметры


Value. Лист, индекс которого необходимо получить.


## Описание


Метод IndexOf возвращает индекс указанного листа в коллекции.


## Комментарии


Если указанный лист отсутствует в коллекции, то метод возвращает значение «-1».
