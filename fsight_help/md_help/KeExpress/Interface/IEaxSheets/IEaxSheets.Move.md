# IEaxSheets.Move


## Синтаксис


Move(OldIndex: Integer; NewIndex: Integer);


## Параметры


OldIndex. Индекс позиции, из
 которой необходимо переместить лист;


NewIndex. Индекс позиции, в
 которую необходимо переместить лист.


## Описание


Метод Move осуществляет перемещение
 листа внутри коллекции.
