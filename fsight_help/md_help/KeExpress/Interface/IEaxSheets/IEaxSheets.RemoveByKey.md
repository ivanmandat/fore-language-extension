# IEaxSheets.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ удаляемого листа.


## Описание


Метод RemoveByKey удаляет указанный лист, ключ которого передается посредством параметра Key.


## Комментарии


Метод возвращает True, в случае успешного удаления, в противном случае - False. При удалении листа также будет удален дочерний экспресс-отчет, который лежит в основе листа.
