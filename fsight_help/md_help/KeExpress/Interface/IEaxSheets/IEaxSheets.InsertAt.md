# IEaxSheets.InsertAt


## Синтаксис


InsertAt(Value: [IEaxAnalyzer](../IEaxAnalyzer/IEaxAnalyzer.htm);
 Index: Integer): [IEaxSheet](../IEaxSheet/IEaxSheet.htm);


## Параметры


Value. Лист, который необходимо
 вставить в указанную позицию;


Index. Индекс позиции, в которую
 необходимо вставить лист.


## Описание


Метод InsertAt вставляет указанный
 лист в указанную позицию в коллекции.
