# IWorkbookConfigurationPanel.ActiveCategory


## Синтаксис


ActiveCategory: [IEaxConfigurationCategory](../IEaxConfigurationCategory/IEaxConfigurationCategory.htm);


## Описание


Свойство ActiveCategory определяет
 активную вкладку компонента.


## Комментарии


В качестве значения свойства может быть установлена одна из вкладок,
 доступная в коллекции [Categories](IWorkbookConfigurationPanel.Categories.htm).
