# IEaxDataAreaHierarchy


Сборка: Express;


## Описание


Интерфейс IEaxDataAreaHierarchy
 используется для работы с альтернативной иерархией аналитической области
 данных.


## Иерархия наследования


           [INamedEntityModel](ForeSys.chm::/Interface/INamedEntityModel/INamedEntityModel.htm)


           IEaxDataAreaHierarchy


## Комментарии


Для получения объекта данного типа используйте свойства и методы интерфейса
 [IEaxDataAreaHierarchies](../IEaxDataAreaHierarchies/IEaxDataAreaHierarchies.htm).
