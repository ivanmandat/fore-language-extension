# IEaxDataAreaHierarchy.CustomDimension


## Синтаксис


CustomDimension: [ICustomDimension](KeDims.chm::/interface/ICustomDimension/ICustomDimension.htm);


## Описание


Свойство CustomDimension возвращает
 объект для работы с измерением фактов.


## Комментарии


Измерение фактов - это внутренний справочник куба, который формирует
 одно из измерений. Если факт один - это случай вырожденного справочника.
