# IEaxDataAreaHierarchy.Filter


## Синтаксис


Filter;


## Описание


Метод Filter выполняет фильтрацию
 элементов альтернативной иерархии.


## Комментарии


Для получения коллекции формул преобразования данных в элементах альтернативной
 иерархии используйте [IEaxDataAreaHierarchy.FilterTransformations](IEaxDataAreaHierarchy.FilterTransformations.htm).


Для получения признака того, была ли произведена фильтрация элементов
 альтернативной иерархии, используйте [IEaxDataAreaHierarchy.IsFiltered](IEaxDataAreaHierarchy.IsFiltered.htm).
