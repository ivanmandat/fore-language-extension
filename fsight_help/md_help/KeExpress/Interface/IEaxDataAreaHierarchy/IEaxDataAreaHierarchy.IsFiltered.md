# IEaxDataAreaHierarchy.IsFiltered


## Синтаксис


IsFiltered: Boolean;


## Описание


Свойство IsFiltered возвращает
 признак того, была ли произведена фильтрация элементов альтернативной
 иерархии.


## Комментарии


Допустимые значения:


	- True. Фильтрация элементов
	 альтернативной иерархии была произведена;


	- False. Фильтрация элементов
	 альтернативной иерархии произведена не была.


Для выполнения фильтрации элементов альтернативной иерархии используйте
 [IEaxDataAreaHierarchy.Filter](IEaxDataAreaHierarchy.Filter.htm).
