# IEaxDataAreaHierarchy.AutoSort


## Синтаксис


AutoSort: [TriState](ForeSys.chm::/Enums/TriState.htm);


## Описание


Свойство AutoSort определяет,
 будет ли использоваться автоматическая сортировка элементов альтернативной
 иерархии.


## Комментарии


Для возвращения признака того, используется ли автоматическая сортировка
 элементов альтернативной иерархии, используйте [IEaxDataAreaHierarchy.IsAutoSort](IEaxDataAreaHierarchy.IsAutoSort.htm).
