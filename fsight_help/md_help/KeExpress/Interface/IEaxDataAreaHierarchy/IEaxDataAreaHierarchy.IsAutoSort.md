# IEaxDataAreaHierarchy.IsAutoSort


## Синтаксис


IsAutoSort: Boolean;


## Описание


Свойство IsAutoSort возвращает
 признак того, используется ли автоматическая сортировка элементов альтернативной
 иерархии.


## Комментарии


Допустимые значения:


	- True. Автоматическая
	 сортировка элементов альтернативной иерархии используется;


	- False. Автоматическая
	 сортировка элементов альтернативной иерархии не используется.


Для определения, будет ли использоваться автоматическая сортировка элементов
 альтернативной иерархии, используйте [IEaxDataAreaHierarchy.AutoSort](IEaxDataAreaHierarchy.AutoSort.htm).
