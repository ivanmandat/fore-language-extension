# IEaxDataAreaHierarchy.Sort


## Синтаксис


Sort;


## Описание


Метод Sort выполняет сортировку
 элементов альтернативной иерархии.


## Комментарии


Для получения коллекции формул преобразования данных у элементов альтернативной
 иерархии используйте [IEaxDataAreaHierarchy.SortTransformations](IEaxDataAreaHierarchy.SortTransformations.htm).


Для получения признака того, была ли произведена сортировка элементов
 альтернативной иерархии, используйте [IEaxDataAreaHierarchy.IsAutoSort](IEaxDataAreaHierarchy.IsAutoSort.htm).


Для определения направления сортировки элементов альтернативной иерархии,
 используйте [IEaxDataAreaHierarchy.SortDirection](IEaxDataAreaHierarchy.SortDirection.htm).
