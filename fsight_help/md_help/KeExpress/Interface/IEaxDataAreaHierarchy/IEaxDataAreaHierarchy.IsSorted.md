# IEaxDataAreaHierarchy.IsSorted


## Синтаксис


IsSorted: Boolean;


## Описание


Свойство IsSorted возвращает
 признак того, была ли произведена сортировка элементов альтернативной
 иерархии.


## Комментарии


Для выполнения сортировки элементов альтернативной иерархии используйте
 [IEaxDataAreaHierarchy.Sort](IEaxDataAreaHierarchy.Sort.htm).
