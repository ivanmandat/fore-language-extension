# IEaxDrillPointResult


Сборка: Express;


## Описание


Интерфейс IEaxDrillPointResult
 содержит свойства для получения информации о привязке точки к данным источника.


## Иерархия наследования


           [IEaxDrillResult](../IEaxDrillResult/IEaxDrillResult.htm)


           IEaxDrillPointResult


## Комментарии


Информацию о привязке точки можно получить, используя метод [IEaxAnalyzeCore.DrillPoint](../IEaxAnalyzeCore/IEaxAnalyzeCore.DrillPoint.htm).
