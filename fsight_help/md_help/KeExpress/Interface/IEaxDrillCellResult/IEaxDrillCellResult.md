# IEaxDrillCellResult


Сборка: Express;


## Описание


Интерфейс IEaxDrillCellResult
 содержит свойства для получения информации о привязке ячейки к данным
 источника.


## Иерархия наследования


           [IEaxDrillResult](../IEaxDrillResult/IEaxDrillResult.htm)


           IEaxDrillCellResult


## Комментарии


Информацию о привязке ячейки можно получить, используя метод [IEaxGrid.DrillCellResult](../IEaxGrid/IEaxGrid.DrillCellResult.htm).
