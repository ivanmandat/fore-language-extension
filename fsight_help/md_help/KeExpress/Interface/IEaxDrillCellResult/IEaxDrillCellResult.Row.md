# IEaxDrillCellResult.Row


## Синтаксис


Row: Integer;


## Описание


Свойство Row возвращает номер
 строки ячейки, привязанной к данным куба.


## Комментарии


Данное свойство возвращает тот номер строки, для которого был выполнен
 метод [IEaxGrid.DrillCellResult](../IEaxGrid/IEaxGrid.DrillCellResult.htm).
