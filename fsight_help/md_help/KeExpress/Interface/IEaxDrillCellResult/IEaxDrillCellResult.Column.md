# IEaxDrillCellResult.Column


## Синтаксис


Column: Integer;


## Описание


Свойство Column возвращает номер
 столбца ячейки, привязанной к данным куба.


## Комментарии


Данное свойство возвращает тот номер столбца, для которого был выполнен
 метод [IEaxGrid.DrillCellResult](../IEaxGrid/IEaxGrid.DrillCellResult.htm).
