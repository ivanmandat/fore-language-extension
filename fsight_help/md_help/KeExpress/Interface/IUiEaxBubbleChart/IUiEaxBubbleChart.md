# IUiEaxBubbleChart


Сборка: Express;


## Описание


Интерфейс IUiEaxBubbleChart
 содержит свойства компонента среды разработки [UiEaxBubbleChart](uidevenv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiEaxBubbleChart.htm),
 являющегося источником данных для компонента [EaxBubbleChartBox](uidevenv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/EaxBubbleChartBox.htm).
