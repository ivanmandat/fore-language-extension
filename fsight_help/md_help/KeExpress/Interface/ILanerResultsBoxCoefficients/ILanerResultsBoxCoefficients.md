# ILanerResultsBoxCoefficients


Сборка: Express;


## Описание


Интерфейс ILanerResultsBoxCoefficients
 предназначен для работы со значениями на вкладке «Коэффициенты»
 панели «Результаты вычислений»
 для компонента LanerResultsBox.


## Комментарии


Интерфейс применяется для получения коэффициентов. Изменение их значений
 невозможно.


Значения коэффициентов вычисляются только для рядов, рассчитанных с
 помощью методов моделирования (сглаживание, прогноз, регрессия).
