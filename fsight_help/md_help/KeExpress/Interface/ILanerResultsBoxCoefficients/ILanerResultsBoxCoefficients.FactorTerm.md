# ILanerResultsBoxCoefficients.FactorTerm


## Синтаксис


FactorTerm(TermIndex: Integer): [ITsCompositeFormulaTerm](KeTransform.chm::/Interface/ITsCompositeFormulaTerm/ITsCompositeFormulaTerm.htm);


## Параметры


TermIndex. Индекс элемента.


## Описание


Свойство FactorTerm возвращает элемент выражения, соответствующий фактору рассчитываемого метода.


## Комментарии


Данное свойство актуально только для рядов, рассчитанных с помощью линейной регрессии или линейной регрессии (оценка методом инструментальных переменных).
