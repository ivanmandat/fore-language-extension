# ILanerResultsBoxCoefficients.ColumnName


## Синтаксис


ColumnName(Index: Integer): String;


## Параметры


Index. Индекс колонки.


## Описание


Свойство ColumnName возвращает наименование колонки коэффициентов по ее индексу.


## Комментарии


Значение параметра Index должно входить в диапазон [0, [ILanerResultsBoxCoefficients.ColumnCount](ILanerResultsBoxCoefficients.ColumnCount.htm)-1].
