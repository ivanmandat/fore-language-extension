# ILanerResultsBoxCoefficients.Value


## Синтаксис


Value(TermIndex: Integer; ColumnIndex: Integer): Variant;


## Параметры


TermIndex. Индекс коэффициента.


ColumnIndex. Индекс колонки со значением коэффициента.


## Описание


Свойство Value возвращает значение указанного коэффициента.


## Комментарии


Значение параметра TermIndex должно входить в диапазон [0, [ILanerResultsBoxCoefficients.TermCount](ILanerResultsBoxCoefficients.TermCount.htm)-1].


Значение параметра ColumnIndex должно входить в диапазон [0, [ILanerResultsBoxCoefficients.ColumnCount](ILanerResultsBoxCoefficients.ColumnCount.htm)-1].
