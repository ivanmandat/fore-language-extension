# ILanerResultsBoxCoefficients.Equation


## Синтаксис


Equation: String;


## Описание


Свойство Equation возвращает идентифицированное уравнение, коэффициенты которого рассчитываются.


## Комментарии


Equation возвращает пустую строку, если вкладка «Коэффициенты» отсутствует на панели «Результаты вычислений».
