# ILanerResultsBoxCoefficients.ColumnCount


## Синтаксис


ColumnCount: Integer;


## Описание


Свойство ColumnCount возвращает количество колонок со значениями коэффициентов.


## Комментарии


ColumnCount является положительным числом и возвращает значение, даже если коэффициенты не были рассчитаны, т.е. [ILanerResultsBoxCoefficients.TermCount](ILanerResultsBoxCoefficients.TermCount.htm) = 0.
