# ILanerResultsBoxCoefficients.TermCount


## Синтаксис


TermCount: Integer;


## Описание


Свойство TermCount возвращает количество рассчитанных коэффициентов.


## Комментарии


TermCount возвращает положительное значение, только если вкладка «Коэффициенты» присутствует на панели «Результаты вычислений» и коэффициенты были рассчитаны.
