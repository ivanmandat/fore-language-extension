# IEaxDataAreaSlice.GenerateSourceParams


## Синтаксис


GenerateSourceParams: Boolean;


## Описание


Свойство GenerateSourceParams
 определяет будет ли использоваться автоматическая генерация параметров
 среза с идентификаторами, соответствующими параметрам источника для среза
 аналитической области.


## Комментарии


Допустимые значения:


	- True. Автоматическая
	 генерация параметров используется. Значение по умолчанию;


	- False. Автоматическая
	 генерация параметров не используется.
