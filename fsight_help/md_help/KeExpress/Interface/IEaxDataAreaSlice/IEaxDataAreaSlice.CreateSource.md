# IEaxDataAreaSlice.CreateSource


## Синтаксис


CreateSource;


## Описание


Метод CreateSource создает источник
 данных среза аналитической области в зависимости от его типа.


## Комментарии


Типы срезов аналитической области данных представлены в перечислении
 [EaxDataAreaSliceType](../../Enums/EaxDataAreaSliceType.htm).


Для построения среза аналитической области данных используйте [IEaxDataAreaSlice.Execute](IEaxDataAreaSlice.Execute.htm).
