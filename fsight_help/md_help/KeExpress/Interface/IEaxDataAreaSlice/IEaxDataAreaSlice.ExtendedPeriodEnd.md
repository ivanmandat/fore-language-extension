# IEaxDataAreaSlice.ExtendedPeriodEnd


## Синтаксис


ExtendedPeriodEnd: DateTime;


## Описание


Свойство ExtendedPeriodEnd определяет
 дату окончания периода загрузки данных.


## Комментарии


Период загрузки данных учитывается при расчете валидаций и трансформаций.


Свойство актуально, если [IEaxDataAreaSlice.UseExtendedPeriod](IEaxDataAreaSlice.UseExtendedPeriod.htm)
 имеет значение True. Для определения
 даты начала периода загрузки данных используйте свойство [IEaxDataAreaSlice.ExtendedPeriodStart](IEaxDataAreaSlice.ExtendedPeriodStart.htm).
