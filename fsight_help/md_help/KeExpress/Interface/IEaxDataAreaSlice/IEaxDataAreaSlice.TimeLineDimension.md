# IEaxDataAreaSlice.TimeLineDimension


## Синтаксис


TimeLineDimension: [IDimInstance](KeDims.chm::/interface/IDimInstance/IDimInstance.htm);


## Описание


Свойство TimeLineDimension определяет
 [измерение
 шкалы времени](UiMaps.chm::/Common/Metrics.htm)
 для карты.


## Комментарии


Измерение шкалы времени - календарное измерение, по отмеченным элементам
 которого будет формироваться временная шкала карты.


Для определения измерения метрик для карты используйте [IEaxDataAreaSlice.IndicatorsDimension](IEaxDataAreaSlice.IndicatorsDimension.htm).


Для определения измерения объектов наблюдения для карты используйте
 [IEaxDataAreaSlice.ObjectivesDimension](IEaxDataAreaSlice.ObjectivesDimension.htm).
