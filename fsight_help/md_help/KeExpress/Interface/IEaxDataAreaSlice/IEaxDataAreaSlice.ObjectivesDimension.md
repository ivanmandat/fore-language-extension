# IEaxDataAreaSlice.ObjectivesDimension


## Синтаксис


ObjectivesDimension: [IDimInstance](KeDims.chm::/interface/IDimInstance/IDimInstance.htm);


## Описание


Свойство ObjectivesDimension
 определяет [измерение
 объектов наблюдения](UiMaps.chm::/Common/Metrics.htm)
 для карты.


## Комментарии


Измерение объектов наблюдения - это [территориальное измерение](UiMaps.chm::/Common/Requirements.htm),
 содержащее атрибут «TERRID».


Для определения измерения метрик для карты используйте [IEaxDataAreaSlice.IndicatorsDimension](IEaxDataAreaSlice.IndicatorsDimension.htm).


Для определения измерения шкалы времени для карты используйте [IEaxDataAreaSlice.TimeLineDimension](IEaxDataAreaSlice.TimeLineDimension.htm).


Для определения измерения направления стрелок для карты используйте
 [IEaxDataAreaSlice.SecondaryObjectivesDimension](IEaxDataAreaSlice.SecondaryObjectivesDimension.htm).
