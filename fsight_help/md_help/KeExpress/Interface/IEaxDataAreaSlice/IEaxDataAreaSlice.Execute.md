## Синтаксис


Execute;


## Описание


Метод Execute выполняет построение
 среза аналитической области данных.


## Комментарии


Для создания источника данных среза используйте [IEaxDataAreaSlice.CreateSource](IEaxDataAreaSlice.CreateSource.htm).
