# IEaxDataAreaSlice.ExtendedPeriodStart


## Синтаксис


ExtendedPeriodStart: DateTime;


## Описание


Свойство ExtendedPeriodStart
 определяет дату начала периода загрузки данных.


## Комментарии


Период загрузки данных учитывается при расчете валидаций и трансформаций.


Свойство актуально, если [IEaxDataAreaSlice.UseExtendedPeriod](IEaxDataAreaSlice.UseExtendedPeriod.htm)
 имеет значение True. Для определения
 даты окончания периода загрузки данных используйте свойство [IEaxDataAreaSlice.ExtendedPeriodEnd](IEaxDataAreaSlice.ExtendedPeriodEnd.htm).
