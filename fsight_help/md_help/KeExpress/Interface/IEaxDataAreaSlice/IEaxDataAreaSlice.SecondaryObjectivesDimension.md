# IEaxDataAreaSlice.SecondaryObjectivesDimension


## Синтаксис


SecondaryObjectivesDimension: [IDimInstance](KeDims.chm::/interface/IDimInstance/IDimInstance.htm);


## Описание


Свойство SecondaryObjectivesDimension
 определяет измерение [направления
 стрелок](UiMaps.chm::/Common/Metrics.htm)
 для карты.


## Комментарии


Для использования данного свойства ознакомьтесь с [требованиями для
 отображения стрелок](UiMaps.chm::/Common/Arrows_requirements.htm) на карте.


Для определения объектов измерения для карты используйте [IEaxDataAreaSlice.ObjectivesDimension](IEaxDataAreaSlice.ObjectivesDimension.htm).
