# IEaxDataAreaSlice.Properties


## Синтаксис


Properties: [IEaxDataAreaSliceProperties](../IEaxDataAreaSliceProperties/IEaxDataAreaSliceProperties.htm);


## Описание


Свойство Properties возвращает
 свойства среза источника данных.


## Комментарии


В качестве источников данных можно использовать:


	- все виды кубов;


	- базы данных временных рядов;


	- реляционные объекты: таблицы, запроса, представления, присоединенной
	 таблицы, источника данных ODBC;


	- экспресс-отчеты;


	- файл;


	- другие источники: переменной моделирования и файлового
	 куба (файл *.ppcube).
