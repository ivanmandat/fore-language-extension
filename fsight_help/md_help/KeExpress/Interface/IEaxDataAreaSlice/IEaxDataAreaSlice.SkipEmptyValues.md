# IEaxDataAreaSlice.SkipEmptyValues


## Синтаксис


SkipEmptyValues: Boolean;


## Описание


Свойство SkipEmptyValues определяет,
 будут ли пропускаться пустые значения среза аналитической области данных.


## Комментарии


Допустимые значения:


	- True. Пустые значения
	 среза данных будут пропускаться;


	- False. По умолчанию.
	 Пустые значения среза данных пропускаться не будут.
