# IEaxDataAreaSlice.Copy


## Синтаксис


Copy(Options: [EaxSliceCopyOptions](../../Enums/EaxSliceCopyOptions.htm)):
 [IEaxDataAreaSlice](IEaxDataAreaSlice.htm);


## Параметры


Options. Параметры копирования
 среза аналитической области данных.


## Описание


Метод Copy создает копию среза
 аналитической области данных.


## Комментарии


По умолчанию копии будет присвоено имя [Срез]
 - Копия N, где Срез -
 имя оригинального среза, N -
 порядковый номер среза.


Для создания копии таблицы используйте [IEaxGrid.Copy](../IEaxGrid/IEaxGrid.Copy.htm).
