# IEaxDataAreaSlice.Load


## Синтаксис


Load;


## Описание


Метод Load загружает срез аналитической
 области.


## Комментарии


Метод используется, если:


	- у отчёта для свойства [IEaxDataArea.DelayedLoad](../IEaxDataArea/IEaxDataArea.DelayedLoad.htm)
	 установлено значение True;


	- у среза для свойства [IEaxDataAreaSlice.IsLoaded](IEaxDataAreaSlice.IsLoaded.htm)
	 установлено значение True.
