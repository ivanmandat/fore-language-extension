# IEaxDataAreaSlice.GetDimension


## Синтаксис


GetDimension(DimKey: Integer): [IEaxDimensionInstance](../IEaxDimensionInstance/IEaxDimensionInstance.htm);


## Параметры


DimKey. Ключ измерения.


## Описание


Метод GetDimension возвращает
 измерение среза аналитической области данных по его ключу.


## Комментарии


Для работы с коллекцией срезов аналитической области данных используйте
 [IEaxDataAreaSlice.Slices](IEaxDataAreaSlice.Slices.htm).
