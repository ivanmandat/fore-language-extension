# IEaxBulkSettings


Сборка: Express;


## Описание


Интерфейс IEaxBulkSettings содержит
 свойства для определения параметров пакетных операций, выполняемых для
 экспресс-отчета.


## Иерархия наследования


           [IGxBulkSettings](ModDrawing.chm::/Interface/IGxBulkSettings/IGxBulkSettings.htm)


           IEaxBulkSettings


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [FilesMapping](IEaxBulkSettings.FilesMapping.htm)
		 Свойство FilesMapping
		 определяет сопоставление ключей элементов отметки и соответствующих
		 им файлов.


		 ![](../../Property_Image.gif)
		 [SelectionSet](IEaxBulkSettings.SelectionSet.htm)
		 Свойство SelectionSet
		 определяет отметку фиксированных измерений, в соответствии с которой
		 будут созданы варианты экспресс-отчета для выполнения пакетной
		 операции.
