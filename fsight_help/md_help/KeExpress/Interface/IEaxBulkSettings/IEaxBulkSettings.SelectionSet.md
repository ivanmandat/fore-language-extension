# IEaxBulkSettings.SelectionSet


## Синтаксис


SelectionSet: [IDimSelectionSet](KeDims.chm::/interface/IDimSelectionSet/IDimSelectionSet.htm);


## Описание


Свойство SelectionSet определяет отметку фиксированных измерений, в соответствии с которой будут созданы варианты экспресс-отчета для выполнения пакетной операции.


## Комментарии


Количество отчетов, для которых будет выполнена операция, равно декартовому произведению отмеченных элементов в измерениях.
