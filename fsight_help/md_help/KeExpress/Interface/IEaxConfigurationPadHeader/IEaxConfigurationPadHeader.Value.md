# IEaxConfigurationPadHeader.Value


## Синтаксис


Value: Boolean;


## Описание


Свойство Value определяет состояние флажка, отображаемого в заголовке.


## Комментарии


Если свойство в значении True, то флажок установлен. Если в значении False - флажок снят.
