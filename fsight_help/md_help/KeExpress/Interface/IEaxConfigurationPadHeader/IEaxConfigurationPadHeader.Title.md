# IEaxConfigurationPadHeader.Title


## Синтаксис


Title: String;


## Описание


Свойство Title определяет текст,
 отображаемый в заголовке.


## Комментарии


Для определения состояния флага, отображаемого в заголовке, используйте
 свойство [IEaxConfigurationPadHeader.Value](IEaxConfigurationPadHeader.Value.htm).
