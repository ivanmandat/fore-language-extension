# IEaxConfigurationPadHeader


Сборка: Express;


## Описание


Интерфейс IEaxConfigurationPadHeader
 содержит свойства заголовка вкладки боковой панели экспресс-отчета.


## Иерархия наследования


           IEaxConfigurationPadHeader


## Комментарии


Параметры заголовка могут быть получены в свойстве [IEaxConfigurationPad.Header](../IEaxConfigurationPad/IEaxConfigurationPad.Header.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Content](IEaxConfigurationPadHeader.Content.htm)
		 Свойство Content определяет
		 тип элемента управления, который будет присутствовать в заголовке.


		 ![](../../Property_Image.gif)
		 [CustomContent](IEaxConfigurationPadHeader.CustomContent.htm)
		 Свойство CustomContent
		 определяет элемент управления, отображаемый в области заголовка.


		 ![](../../Property_Image.gif)
		 [Title](IEaxConfigurationPadHeader.Title.htm)
		 Свойство Title определяет
		 текст, отображаемый в заголовке.


		 ![](../../Property_Image.gif)
		 [Value](IEaxConfigurationPadHeader.Value.htm)
		 Свойство Value определяет
		 состояние флага, отображаемого в заголовке.
