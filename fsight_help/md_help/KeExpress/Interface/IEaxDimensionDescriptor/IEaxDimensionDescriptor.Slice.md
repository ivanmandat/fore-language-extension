# IEaxDimensionDescriptor.Slice


## Синтаксис


Slice: Object;


## Описание


Свойство Slice возвращает объект
 для работы со срезом данных, которому принадлежит измерение.


## Комментарии


Для дальнейшей работы с полученным объектом необходимо привести его
 к типу [IEaxDataAreaSlice](../IEaxDataAreaSlice/IEaxDataAreaSlice.htm)
 или [IPrxSlice](KeReport.chm::/Interface/IPrxSlice/IPrxSlice.htm).


Для получения типа источника данных измерения используйте [IEaxDimensionDescriptor.SliceType](IEaxDimensionDescriptor.SliceType.htm).
