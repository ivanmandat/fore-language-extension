# IEaxDimensionDescriptor


Сборка: Express;


## Описание


Интерфейс IEaxDimensionDescriptor
 используется для работы с описанием измерения.


## Иерархия наследования


           IEaxDimensionDescriptor


## Комментарии


Для работы с коллекцией описаний измерения используйте свойства интерфейса
 [IEaxDimensionDescriptors](../IEaxDimensionDescriptors/IEaxDimensionDescriptors.htm).
