# IEaxDimensionDescriptor.DimKey


## Синтаксис


DimKey: Integer;


## Описание


Свойство DimKey возвращает ключ
 измерения.


## Комментарии


Ключ - это уникальный идентификатор компонента в коллекции. Значение
 ключа генерируется автоматически и не может быть изменено пользователем.
