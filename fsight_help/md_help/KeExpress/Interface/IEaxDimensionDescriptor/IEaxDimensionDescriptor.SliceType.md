## Описание


Свойство SliceType возвращает
 тип источника данных измерения.


## Комментарии


Для получения объекта для работы со срезом данных, которому принадлежит
 измерение, используйте [IEaxDimensionDescriptor.Slice](IEaxDimensionDescriptor.Slice.htm).
