# IEaxRangeCellStyles


Сборка: Express;


## Описание


Интерфейс IEaxRangeCellStyles
 предназначен для работы с коллекцией стилей форматирования таблицы.


## Комментарии


Коллекцию стилей можно получить используя свойство [IEaxTableStyle.RangeCellStyles](../IEaxTableStyle/IEaxTableStyle.RangeCellStyles.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IEaxRangeCellStyles.Count.htm)
		 Свойство Count возвращает
		 количество стилей в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IEaxRangeCellStyles.Item.htm)
		 Свойство Item возвращает
		 стиль из коллекции по его индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IEaxRangeCellStyles.Add.htm)
		 Метод Add добавляет
		 указанный стиль в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](IEaxRangeCellStyles.Clear.htm)
		 Метод Clear очищает
		 коллекцию стилей.


		 ![](../../Sub_Image.gif)
		 [GetByRange](IEaxRangeCellStyles.GetByRange.htm)
		 Метод GetByRange возвращает
		 стиль выделенного диапазона ячеек.


		 ![](../../Sub_Image.gif)
		 [Remove](IEaxRangeCellStyles.Remove.htm)
		 Метод Remove удаляет
		 стиль из коллекции по его индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByRange](IEaxRangeCellStyles.RemoveByRange.htm)
		 Метод RemoveByRange
		 удаляет все стили оформления, применяемые в указанном диапазоне
		 ячеек.
