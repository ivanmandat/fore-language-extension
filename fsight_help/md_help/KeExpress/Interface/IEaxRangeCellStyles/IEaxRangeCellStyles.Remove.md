# IEaxRangeCellStyles.Remove


## Синтаксис


Remove(Index: Integer);


## Параметры


Index. Индекс стиля в коллекции.


## Описание


Метод Remove удаляет стиль из
 коллекции по его индексу.


## Комментарии


Значение параметра Index должно
 быть неотрицательным и меньше значения свойства [IEaxRangeCellStyles.Count](IEaxRangeCellStyles.Count.htm).
