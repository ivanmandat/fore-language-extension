# IAlgorithmParams.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 настроек управления параметрами в коллекции.


## Комментарии


Количество настроек соответствует количеству параметров, которые созданы
 в структуре алгоритма расчёта.
