# IAlgorithmParams


Сборка: Express;


## Описание


Интерфейс IAlgorithmParams предназначен
 для работы с коллекцией настроек управления параметрами алгоритма расчёта,
 выполняемого по пользовательской кнопке.


## Иерархия наследования


           IAlgorithmParams


## Комментарии


Настройки управления параметрами можно получить, используя следующие
 свойства:


	- [IEaxDataAreaExternalTransformation.Params](../IEaxDataAreaExternalTransformation/IEaxDataAreaExternalTransformation.Params.htm)


	- [IPrxUserButtonActionAlgorithm.Params](KeReport.chm::/Interface/IPrxUserButtonActionAlgorithm/IPrxUserButtonActionAlgorithm.Params.htm).
