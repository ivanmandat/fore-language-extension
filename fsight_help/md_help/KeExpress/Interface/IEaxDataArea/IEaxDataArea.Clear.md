# IEaxDataArea.Clear


## Синтаксис


Clear;


## Описание


Метод Clear очищает аналитическую
 область данных.


## Комментарии


Если метод выполняется для аналитической области регламентного отчета,
 то она будет преобразована в реляционную область данных.
