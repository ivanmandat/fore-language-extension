# IEaxDataArea.GenerateDimensionParams


## Синтаксис


GenerateDimensionParams: Boolean;


## Описание


Свойство GenerateDimensionParams
 определяет, будет ли использоваться автоматическая генерация параметров
 измерений для аналитической области данных.


## Комментарии


По умолчанию свойству установлено значение True,
 при этом будет производиться автоматическая генерация параметров измерений.
 Автоматически сгенерированные параметры позволяют в формулах использовать
 не отдельные элементы измерений, а какие-либо измерения полностью. При
 этом во время расчета формулы будет подставляться значение по необходимому
 элементу измерения.


При установке значения False
 автоматическая генерация параметров измерений не производится, в формулах
 можно использовать только отдельные элементы измерений.


Для определения, будет ли использоваться автоматическая генерация
 параметров измерения для среза данных, используйте свойство [IEaxDataAreaSlice.GenerateDimensionParams](../IEaxDataAreaSlice/IEaxDataAreaSlice.GenerateDimensionParams.htm).
