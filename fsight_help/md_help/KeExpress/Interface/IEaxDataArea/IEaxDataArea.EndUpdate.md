# IEaxDataArea.EndUpdate


## Синтаксис


EndUpdate;


## Описание


Метод EndUpdate возобновляет
 применение изменений, вносимых в настройки аналитической области данных.


## Комментарии


При вызове метода будут применены все изменения, которые вносились в
 настройки срезов данных и их представления после вызова метода [IEaxDataArea.BeginUpdate](IEaxDataArea.BeginUpdate.htm).
