# IEaxDataArea.Slices


## Синтаксис


Slices: [IEaxDataAreaSlices](../IEaxDataAreaSlices/IEaxDataAreaSlices.htm);


## Описание


Свойство Slices возвращает коллекцию
 срезов аналитической области данных.


## Комментарии


Срез данных - двумерная таблица данных, полученная путём фиксации одного
 или нескольких измерений многомерного куба.
