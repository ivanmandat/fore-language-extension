# IEaxDataArea.CommentManager


## Синтаксис


CommentManager: [IDataAreaCommentManager](../IDataAreaCommentManager/IDataAreaCommentManager.htm);


## Описание


Свойство CommentManager возвращает
 менеджер комментариев отчёта.


## Комментарии


Свойство актуально, если для отчёта в качестве источника выступает стандартный
 куб, для которого включено хранение комментариев.
