## Описание


Свойство ControlInfo возвращает
 объект, используемый для управления параметрами среза аналитической области.


## Комментарии


Для работы с параметрами среза аналитической области используйте интерфейс
 [IEaxDataAreaControl](../IEaxDataAreaControl/IEaxDataAreaControl.htm).
