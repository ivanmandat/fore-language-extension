# IEaxDataArea.IsInUpdate


## Синтаксис


IsInUpdate: Boolean;


## Описание


Свойство IsInUpdate возвращает
 признак включенного режима запрета обновления представлений аналитической
 области данных.


## Комментарии


Свойство возвращает значение True,
 если был включён режим блокировки обновлений с помощью метода [IEaxDataArea.BeginUpdate](IEaxDataArea.BeginUpdate.htm)
 и ещё не был вызван метод [IEaxDataArea.EndUpdate](IEaxDataArea.EndUpdate.htm).
