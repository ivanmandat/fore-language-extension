# IEaxDataAreaSlices.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс среза аналитической
 области данных.


## Описание


Метод Remove удаляет срез аналитической
 области данных по его индексу.


## Комментарии


Если срез аналитической области данных будет удален, то метод вернет
 значение True.


Для удаления среза аналитической области данных по его ключу используйте
 [IEaxDataAreaSlices.RemoveByKey](IEaxDataAreaSlices.RemoveByKey.htm).
