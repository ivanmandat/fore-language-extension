# IEaxDataAreaSlices.FindById


## Синтаксис


FindById(Id: String): [IEaxDataAreaSlice](../IEaxDataAreaSlice/IEaxDataAreaSlice.htm);


## Параметры


Id. Идентификатор среза аналитической
 области данных.


## Описание


Метод FindById осуществляет
 поиск среза аналитической области данных по его идентификатору.


## Комментарии


Если срез не будет найден, то метод вернёт значение Null.


Для поиска среза аналитической области данных по его ключу используйте
 [IEaxDataAreaSlices.FindByKey](IEaxDataAreaSlices.FindByKey.htm).
