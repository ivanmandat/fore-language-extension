# IEaxDataAreaSlices.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ среза аналитической
 области данных.


## Описание


Метод RemoveByKey удаляет срез
 аналитической области данных по его ключу.


## Комментарии


Если срез аналитической области данных будет удален, то метод вернет
 значение True.


Для удаления среза аналитической области данных по его индексу используйте
 [IEaxDataAreaSlices.Remove](IEaxDataAreaSlices.Remove.htm).
