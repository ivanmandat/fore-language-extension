# IEaxDataAreaSlices


Сборка: Express;


## Описание


Интерфейс IEaxDataAreaSlices
 используется для работы с коллекцией срезов аналитической области данных.


## Иерархия наследования


           IEaxDataAreaSlices


## Комментарии


Для получения коллекции срезов аналитической области данных используйте
 [IEaxDataArea.Slices](../IEaxDataArea/IEaxDataArea.Slices.htm).
