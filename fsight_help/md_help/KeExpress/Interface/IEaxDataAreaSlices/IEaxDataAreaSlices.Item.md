# IEaxDataAreaSlices.Item


## Синтаксис


Item(Index: Integer): [IEaxDataAreaSlice](../IEaxDataAreaSlice/IEaxDataAreaSlice.htm);


## Параметры


Index. Индекс среза аналитической
 области данных.


## Описание


Свойство Item возвращает срез
 аналитической области данных по его индексу.


## Комментарии


Для получения количества срезов аналитической области данных в коллекции
 используйте [IEaxDataAreaSlices.Count](IEaxDataAreaSlices.Count.htm).
