# IEaxDataAreaSlices.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IEaxDataAreaSlice](../IEaxDataAreaSlice/IEaxDataAreaSlice.htm);


## Параметры


Key. Ключ среза аналитической
 области данных.


## Описание


Метод FindByKey осуществляет
 поиск среза аналитической области данных по его ключу.


## Комментарии


Если срез не будет найден, то метод вернет значение Null.


Для поиска среза аналитической области данных по его идентификатору
 используйте [IEaxDataAreaSlices.FindById](IEaxDataAreaSlices.FindById.htm).
