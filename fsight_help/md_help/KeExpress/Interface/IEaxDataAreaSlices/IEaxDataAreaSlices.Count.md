# IEaxDataAreaSlices.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 срезов аналитической области данных.


## Комментарии


Для получения среза аналитической области данных по его индексу используйте
 [IEaxDataAreaSlices.Item](IEaxDataAreaSlices.Item.htm).
