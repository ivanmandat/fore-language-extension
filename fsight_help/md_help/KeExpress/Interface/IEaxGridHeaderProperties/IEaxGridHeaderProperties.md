# IEaxGridHeaderProperties


Сборка: Express;


## Описание


Интерфейс IEaxGridHeaderProperties
 используется для работы со свойствами заголовка.


## Иерархия наследования


           IEaxGridHeaderProperties


## Комментарии


Для получения объекта данного типа используйте [IEaxGridViewSettings.GetViewSettings](../IEaxGridViewSettings/IEaxGridViewSettings.GetViewSettings.htm).
