# IEaxRangeFormatConditions


Сборка: Express;


## Описание


Интерфейс IEaxRangeFormatConditions
 предназначен для работы с коллекцией стилей условного форматирования таблицы.


## Комментарии


Коллекцию стилей можно получить используя свойство [IEaxTableStyle.RangeFormatConditions](../IEaxTableStyle/IEaxTableStyle.RangeFormatConditions.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IEaxRangeFormatConditions.Count.htm)
		 Свойство Count возвращает
		 количество стилей в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IEaxRangeFormatConditions.Item.htm)
		 Свойство Item возвращает
		 стиль из коллекции по его индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IEaxRangeFormatConditions.Add.htm)
		 Метод Add добавляет
		 элемент в коллекцию стилей условного форматирования таблицы.


		 ![](../../Sub_Image.gif)
		 [Clear](IEaxRangeFormatConditions.Clear.htm)
		 Метод Clear очищает
		 коллекцию стилей.


		 ![](../../Sub_Image.gif)
		 [GetByRange](IEaxRangeFormatConditions.GetByRange.htm)
		 Метод GetByRange возвращает
		 стиль выделенного диапазона ячеек.


		 ![](../../Sub_Image.gif)
		 [Remove](IEaxRangeFormatConditions.Remove.htm)
		 Метод Remove удаляет
		 стиль из коллекции по его индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByRange](IEaxRangeFormatConditions.RemoveByRange.htm)
		 Метод RemoveByRange
		 удаляет все стили условного форматирования, применяемые в указанном
		 диапазоне ячеек.
