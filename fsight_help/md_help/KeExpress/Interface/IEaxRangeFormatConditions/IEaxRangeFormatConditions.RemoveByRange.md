# IEaxRangeFormatConditions.RemoveByRange


## Синтаксис


RemoveByRange(Range: [ITabRange](TabSheet.chm::/Interface/ITabRange/ITabRange.htm));


## Параметры


Range. Диапазон ячеек.


## Описание


Метод RemoveByRange удаляет
 все стили условного форматирования, применяемые в указанном диапазоне
 ячеек.


## Комментарии


Параметр Range не должен принимать
 значение Null.
