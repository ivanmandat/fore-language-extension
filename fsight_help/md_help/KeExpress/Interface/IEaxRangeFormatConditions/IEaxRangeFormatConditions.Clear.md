# IEaxRangeFormatConditions.Clear


## Синтаксис


Clear;


## Описание


Метод Clear очищает коллекцию
 стилей.


## Комментарии


После выполнения метода Clear
 значение [IEaxRangeFormatConditions.Count](IEaxRangeFormatConditions.Count.htm)
 становится равным нулю.
