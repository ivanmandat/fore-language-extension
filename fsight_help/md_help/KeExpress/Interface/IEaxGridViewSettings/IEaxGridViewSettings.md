# IEaxGridViewSettings


Сборка: Express;


## Описание


Интерфейс IEaxGridViewSettings
 предназначен для работы с [параметрами
 таблицы данных](UiAnalyticalArea.chm::/Parameters/Parameters.htm).


## Иерархия наследования


           IEaxGridViewSettings


## Комментарии


Для получения объекта для работы с параметрами таблицы данных используйте
 свойство [IEaxGrid.ViewSettings](../IEaxGrid/IEaxGrid.ViewSettings.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CreateRegion](IEaxGridViewSettings.CreateRegion.htm)
		 Свойство CreateRegion
		 определяет, будет ли создан регион таблицы в регламентном отчете.


		 ![](../../Property_Image.gif)
		 [GetStyleSettings](IEaxGridViewSettings.GetStyleSettings.htm)
		 Свойство GetStyleSettings
		 возвращает настройки стиля оформления для указанного элемента
		 заголовка.


		 ![](../../Property_Image.gif)
		 [GetViewSettings](IEaxGridViewSettings.GetViewSettings.htm)
		 Свойство GetViewSettings
		 возвращает настройки отображения для указанного элемента заголовка.


		 ![](../../Property_Image.gif)
		 [HyperlinkAsText](IEaxGridViewSettings.HyperlinkAsText.htm)
		 Свойство HyperlinkAsText
		 определяет, будут ли гиперссылки в таблице данных отображаться [как
		 простой текст](UiAnalyticalArea.chm::/Parameters/Parameters.htm).


		 ![](../../Property_Image.gif)
		 [OutputPart](IEaxGridViewSettings.OutputPart.htm)
		 Свойство OutputPart
		 определяет отображаемую область таблицы.


		 ![](../../Property_Image.gif)
		 [OutputRange](IEaxGridViewSettings.OutputRange.htm)
		 Свойство OutputRange
		 определяет выходной диапазон таблицы.


		 ![](../../Property_Image.gif)
		 [ShowRealBorderDataStyle](IEaxGridViewSettings.ShowRealBorderDataStyle.htm)
		 Свойство ShowRealBorderDataStyle
		 определяет признак отображения границ, которые пересекаются между
		 собой в ячейке таблицы.


		 ![](../../Property_Image.gif)
		 [UseStyleFormatting](IEaxGridViewSettings.UseStyleFormatting.htm)
		 Свойство UseStyleFormatting
		 определяет, будет ли использоваться собственное оформления для
		 таблицы данных.


		 ![](../../Property_Image.gif)
		 [UseTabSheetFormulas](IEaxGridViewSettings.UseTabSheetFormulas.htm)
		 Свойство UseTabSheetFormulas
		 определяет, будут ли использоваться формулы с текущего листа.
