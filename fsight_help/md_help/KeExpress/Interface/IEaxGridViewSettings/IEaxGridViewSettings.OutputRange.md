# IEaxGridViewSettings.OutputRange


## Синтаксис


OutputRange: String;


## Описание


Свойство OutputRange определяет
 выходной диапазон таблицы.


## Комментарии


Для использования свойства необходимо, чтобы свойство [IEaxGridViewSettings.OutputPart](IEaxGridViewSettings.OutputPart.htm)
 принимало значение [EaxGridOutputPart.Data](../../Enums/EaxGridOutputPart.htm).
