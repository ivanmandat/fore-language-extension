# IEaxGridViewSettings.UseTabSheetFormulas


## Синтаксис


UseTabSheetFormulas: Boolean;


## Описание


Свойство UseTabSheetFormulas
 определяет, будут ли использоваться формулы с текущего листа.


## Комментарии


Допустимые значения:


	- True. Будут использоваться
	 формулы текущего листа;


	- False. Будут использоваться
	 значения формул текущего листа. Значение по умолчанию.


Свойство актуально, если осуществляется работа с аналитической областью
 данных в регламентном отчете.
