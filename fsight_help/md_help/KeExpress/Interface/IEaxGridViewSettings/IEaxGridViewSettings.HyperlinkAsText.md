# IEaxGridViewSettings.HyperlinkAsText


## Синтаксис


HyperlinkAsText: Boolean;


## Описание


Свойство HyperlinkAsText определяет,
 будут ли гиперссылки в таблице данных отображаться [как простой
 текст](UiAnalyticalArea.chm::/Parameters/Parameters.htm).


## Комментарии


Допустимые значения:


	- True. Гиперссылки будут
	 отображаться как простой текст;


	- False. По умолчанию.
	 Гиперссылки будут отображаться как гиперссылки.
