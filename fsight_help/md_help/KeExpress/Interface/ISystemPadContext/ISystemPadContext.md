# ISystemPadContext


Сборка: Express;


## Описание


Интерфейс ISystemPadContext
 предназначен для работы с контекстом компонента боковой панели.


## Иерархия наследования


           ISystemPadContext


## Комментарии


Для работы с компонентом боковой панели в экспресс-отчете используйте
 свойства и методы интерфейса [IEaxConfigurationPanel](../IEaxConfigurationPanel/IEaxConfigurationPanel.htm),
 в рабочей книге - [IWorkbookConfigurationPanel](../IWorkbookConfigurationPanel/IWorkbookConfigurationPanel.htm).
