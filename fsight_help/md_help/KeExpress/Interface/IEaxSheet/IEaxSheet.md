# IEaxSheet


Сборка: Express;


## Описание


Интерфейс IEaxSheet содержит
 свойства и методы листа экспресс-отчета.


## Иерархия наследования


           IEaxSheet


## Комментарии


Коллекция листов экспресс-отчета доступна в свойстве [IEaxAnalyzeCore.Sheets](../IEaxAnalyzeCore/IEaxAnalyzeCore.Sheets.htm).
 Каждый лист представляет собой отдельный экспресс-отчет, имеющий собственные
 источники данных и настройки объектов.
