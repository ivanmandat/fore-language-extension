# IEaxSheet.Load


## Синтаксис


Load;


## Описание


Метод Load загружает лист экспресс-отчёта.


## Комментарии


Метод используется, если


	- у отчёта для свойства [IEaxDataArea.DelayedLoad](../IEaxDataArea/IEaxDataArea.DelayedLoad.htm)
	 установлено значение True;


	- у листа для свойства [IEaxSheet.IsLoaded](IEaxSheet.IsLoaded.htm)
	 установлено значение True.
