# IEaxSheet.Active


## Синтаксис


Active: Boolean;


## Описание


Свойство Active определяет,
 является ли текущий лист активным.


## Комментарии


Если свойству установлено значение True,
 то лист является активным. При этом отображаются объекты экспресс-отчёта,
 который лежит в основе листа.
