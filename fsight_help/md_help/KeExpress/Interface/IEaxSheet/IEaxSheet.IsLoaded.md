# IEaxSheet.IsLoaded


## Синтаксис


IsLoaded: Boolean;


## Описание


Свойство IsLoaded определяет,
 будет ли загружаться лист экспресс-отчёта.


## Комментарии


Свойство актуально, если установлена отложенная загрузка листов аналитической
 области: у свойства [IEaxDataArea.DelayedLoad](../IEaxDataArea/IEaxDataArea.DelayedLoad.htm)
 установлено значение True.


Допустимые значения:


	- True. Лист будет загружаться.
	 Значение по умолчанию для активного листа;


	- False. Лист не будет
	 загружаться. Значение по умолчанию.
