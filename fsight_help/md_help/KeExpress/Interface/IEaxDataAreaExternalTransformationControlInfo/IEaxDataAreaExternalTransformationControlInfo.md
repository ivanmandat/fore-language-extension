# IEaxDataAreaExternalTransformationControlInfo


Сборка: Express;


## Описание


Интерфейс IEaxDataAreaExternalTransformationControlInfo
 предназначен для работы с коллекцией параметров задачи моделирования.


## Иерархия наследования


           IEaxDataAreaExternalTransformationControlInfo


## Комментарии


Параметры задачи моделирования можно получить, используя свойство [IEaxDataAreaExternalTransformation.ControlInfo](../IEaxDataAreaExternalTransformation/IEaxDataAreaExternalTransformation.ControlInfo.htm).
