# IEaxDataAreaExternalTransformationControlInfo.FindById


## Синтаксис


FindById(Id: String): [IEaxDataAreaExternalTransformationControl](../IEaxDataAreaExternalTransformationControl/IEaxDataAreaExternalTransformationControl.htm);


## Параметры


Id. Идентификатор параметра.


## Описание


Метод FindById осуществляет
 поиск настроек по идентификатору параметра.


## Комментарии


Метод возвращает настройки привязки параметра в случае удачного поиска
 и Null, если задача моделирования не имеет параметра с указанным идентификатором.
