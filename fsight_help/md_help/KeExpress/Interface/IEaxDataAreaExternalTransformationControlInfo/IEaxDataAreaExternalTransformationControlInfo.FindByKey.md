# IEaxDataAreaExternalTransformationControlInfo.FindByKey


## Синтаксис


FindByKey(Value: Integer): [IEaxDataAreaExternalTransformationControl](../IEaxDataAreaExternalTransformationControl/IEaxDataAreaExternalTransformationControl.htm);


## Параметры


Value. Ключ параметра.


## Описание


Метод FindByKey осуществляет
 поиск настроек по ключу параметра.


## Комментарии


Метод возвращает настройки привязки параметра в случае удачного поиска
 и Null, если задача моделирования
 не имеет параметра с указанным ключом.
