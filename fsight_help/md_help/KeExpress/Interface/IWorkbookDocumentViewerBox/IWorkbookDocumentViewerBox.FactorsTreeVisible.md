# IWorkbookDocumentViewerBox.FactorsTreeVisible


## Синтаксис


FactorsTreeVisible: Boolean;


## Описание


Свойство FactorsTreeVisible
 определяет признак отображения дополнительных элементов управления в области
 компонента.


## Комментарии


Свойство FactorsTreeVisible
 управляет отображением дерева рядов, кнопки «Источник»
 и цепочки навигации по атрибутам выбранного источника.


Допустимые значения:


	- True. По умолчанию.
	 Элементы отображаются;


	- False. Элементы не отображаются.
