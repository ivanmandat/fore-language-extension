# IEaxObjectSelectionManager


Сборка: Express;


## Описание


Интерфейс IEaxObjectSelectionManager
 предназначен для работы с отметкой измерений.


## Иерархия наследования


           IEaxObjectSelectionManager


## Комментарии


Для получения объекта данного типа используйте [IEaxObject.SelectionManager](../IEaxObject/IEaxObject.SelectionManager.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AcceptDataSelection](IEaxObjectSelectionManager.AcceptDataSelection.htm)
		 Свойство AcceptDataSelection определяет,
		 конвертировать ли отметку по выделению в визуальную отметку.


		 ![](../../Property_Image.gif)
		 [ObjectSelection](IEaxObjectSelectionManager.ObjectSelection.htm)
		 Свойство ObjectSelection
		 возвращает текущую отметку измерений.


		 ![](../../Property_Image.gif)
		 [Selection](IEaxObjectSelectionManager.Selection.htm)
		 Свойство Selection
		 возвращает отметку объекта.


		 ![](../../Property_Image.gif)
		 [TrackObjectSelection](IEaxObjectSelectionManager.TrackObjectSelection.htm)
		 Свойство TrackObjectSelection
		 определяет, выставлять ли отметку в дереве измерений по выделению
		 в визуализаторе.
