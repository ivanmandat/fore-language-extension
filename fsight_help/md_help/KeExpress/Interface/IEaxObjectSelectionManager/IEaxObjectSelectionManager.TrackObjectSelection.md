# IEaxObjectSelectionManager.TrackObjectSelection


## Синтаксис


TrackObjectSelection: Boolean;


## Описание


Свойство TrackObjectSelection
 определяет, выставлять ли отметку в дереве измерений по выделению в визуализаторе.


## Комментарии


Допустимые значения:


	- True. Отметка выставляется;


	- False. Отметка не выставляется.
