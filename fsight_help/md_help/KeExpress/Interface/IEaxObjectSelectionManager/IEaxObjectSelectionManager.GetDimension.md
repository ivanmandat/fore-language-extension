# IEaxObjectSelectionManager.GetDimension


## Синтаксис


GetDimension(DimKey: Integer): [IEaxDimensionInstance](../IEaxDimensionInstance/IEaxDimensionInstance.htm);


## Параметры


DimKey. Ключ измерения.


## Описание


Метод GetDimension возвращает
 данные для измерения с указанным ключом.


## Комментарии


Для получения отметки измерений используйте свойство [IEaxObjectSelectionManager.ObjectSelection](IEaxObjectSelectionManager.ObjectSelection.htm).
