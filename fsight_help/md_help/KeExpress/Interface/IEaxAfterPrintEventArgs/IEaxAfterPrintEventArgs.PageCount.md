# IEaxAfterPrintEventArgs.PageCount


## Синтаксис


PageCount: Integer;


## Описание


Свойство PageCount возвращает
 количество напечатанных страниц.

Для определения количества экспортированных страниц используйте [IEaxAfterExportEventArgs.PageCount](../IEaxAfterExportEventArgs/IEaxAfterExportEventArgs.PageCount.htm).
