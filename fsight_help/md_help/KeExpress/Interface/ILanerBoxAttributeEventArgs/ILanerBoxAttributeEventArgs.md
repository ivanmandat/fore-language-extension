# ILanerBoxAttributeEventArgs


Сборка: Express;


## Описание


Интерфейс ILanerBoxAttributeEventArgs
 предназначен для работы с аргументами событий, происходящих с атрибутами
 базы данных временных рядов в компоненте LanerBox.
