# ILanerBoxAttributeEventArgs.Index


## Синтаксис


Index: Integer;


## Описание


Свойство Index возвращает индекс атрибута, с которым произошло событие.


## Комментарии


Если Index возвращает значение «-1», то события с атрибутом не происходило.


Индексация атрибутов сплошная и начинается с нуля.
