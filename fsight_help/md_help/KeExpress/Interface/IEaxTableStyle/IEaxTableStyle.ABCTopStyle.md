# IEaxTableStyle.ABCTopStyle


## Синтаксис


ABCTopStyle: [ITabCellStyle](TabSheet.chm::/Interface/ITabCellStyle/ITabCellStyle.htm);


## Описание


Свойство ABCTopStyle определяет
 стиль ячеек с максимальными значениями столбца/строки при использовании
 анализа «[Первые/Последние](UiAnalyticalArea.chm::/Analysis/distribution.htm#distibution_firstlast)».


## Комментарии


Для определения стиля ячеек с минимальными значениями столбца/строки
 используйте свойство [IEaxTableStyle.ABCBottomStyle](IEaxTableStyle.ABCBottomStyle.htm).
