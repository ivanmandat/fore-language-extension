# IEaxTableStyle


Сборка: Express;


## Описание


Интерфейс IEaxTableStyle содержит
 свойства и методы оформления таблицы экспресс-отчета.


## Иерархия наследования


           IEaxTableStyle


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ABCBottomStyle](IEaxTableStyle.ABCBottomStyle.htm)


		 Свойство ABCBottomStyle
		 определяет стиль ячеек с минимальными значениями столбца/строки
		 при использовании анализа «[Первые/Последние](UiAnalyticalArea.chm::/Analysis/distribution.htm#distibution_firstlast)».


		 ![](../../Property_Image.gif)
		 [ABCFormatConditions](IEaxTableStyle.ABCFormatConditions.htm)


		 Свойство ABCFormatConditions
		 возвращает коллекцию условных форматов применяемых для ячеек столбца,
		 по которому включен [анализ
		 80/20](../IEaxGrid/IEaxGrid.Activate8020.htm).


		 ![](../../Property_Image.gif)
		 [ABCHeaderStyle](IEaxTableStyle.ABCHeaderStyle.htm)


		 Свойство ABCHeaderStyle
		 определяет стиль оформления заголовка столбца, по которому включен
		 [анализ
		 80/20](../IEaxGrid/IEaxGrid.Activate8020.htm), в таблице экспресс-отчета.


		 ![](../../Property_Image.gif)
		 [ABCStyle](IEaxTableStyle.ABCStyle.htm)


		 Свойство ABCStyle определяет
		 стиль оформления ячеек столбца, по которому включен [анализ
		 80/20](../IEaxGrid/IEaxGrid.Activate8020.htm), в таблице экспресс-отчета.


		 ![](../../Property_Image.gif)
		 [ABCTopStyle](IEaxTableStyle.ABCTopStyle.htm)


		 Свойство ABCTopStyle
		 определяет стиль ячеек с максимальными значениями столбца/строки
		 при использовании анализа «[Первые/Последние](UiAnalyticalArea.chm::/Analysis/distribution.htm#distibution_firstlast)».


		 ![](../../Property_Image.gif)
		 [ChangedCellsStyle](IEaxTableStyle.ChangedCellsStyle.htm)


		 Свойство ChangedCellsStyle
		 определяет стиль оформления ячеек, содержащих измененные данные.


		 ![](../../Property_Image.gif)
		 [DefaultStyle](IEaxTableStyle.DefaultStyle.htm)
		 Свойство DefaultStyle
		 определяет стиль, применяемый к различным элементам таблицы по
		 умолчанию.


		 ![](../../Property_Image.gif)
		 [DimCalculateHeaderStyle](IEaxTableStyle.DimCalculateHeaderStyle.htm)
		 Свойство DimCalculateHeaderStyle
		 определяет стиль оформления заголовка вычисляемого показателя.


		 ![](../../Property_Image.gif)
		 [DimCalculateStyle](IEaxTableStyle.DimCalculateStyle.htm)


		 Свойство DimCalculateStyle
		 определяет стиль оформления ячеек, содержащих вычисляемые показатели.


		 ![](../../Property_Image.gif)
		 [FixedCornerStyle](IEaxTableStyle.FixedCornerStyle.htm)


		 Свойство FixedCornerStyle
		 определяет стиль оформления угловой ячейки таблицы.


		 ![](../../Property_Image.gif)
		 [HighlightedStyle](IEaxTableStyle.HighlightedStyle.htm)


		 Свойство HighlightedStyle
		 определяет стиль оформления подсвеченных ячеек таблицы.


		 ![](../../Property_Image.gif)
		 [InternalFormatConditions](IEaxTableStyle.InternalFormatConditions.htm)


		 Свойство InternalFormatConditions
		 возвращает коллекцию условных форматов применяемых для ячеек области
		 данных таблицы.


		 ![](../../Property_Image.gif)
		 [InternalStyle](IEaxTableStyle.InternalStyle.htm)


		 Свойство InternalStyle
		 определяет стиль оформления области данных таблицы.


		 ![](../../Property_Image.gif)
		 [LeftHeaderStyle](IEaxTableStyle.LeftHeaderStyle.htm)


		 Свойство LeftHeaderStyle
		 определяет стиль оформления заголовков строк таблицы.


		 ![](../../Property_Image.gif)
		 [RangeCellStyles](IEaxTableStyle.RangeCellStyles.htm)


		 Свойство RangeCellStyles
		 возвращает коллекцию стилей форматирования таблицы.


		 ![](../../Property_Image.gif)
		 [RangeFormatConditions](IEaxTableStyle.RangeFormatConditions.htm)


		 Свойство RangeFormatConditions
		 возвращает коллекцию стилей условного форматирования таблицы.


		 ![](../../Property_Image.gif)
		 [RevisionLegend](IEaxTableStyle.RevisionLegend.htm)


		 Свойство RevisionLegend
		 возвращает легенду ревизий для рядного режима базы данных временных
		 рядов.


		 ![](../../Property_Image.gif)
		 [TableStyle](IEaxTableStyle.TableStyle.htm)


		 Свойство TableStyle
		 возвращает стиль оформления областей элементов измерений в таблице
		 экспресс-отчета.


		 ![](../../Property_Image.gif)
		 [TopHeaderStyle](IEaxTableStyle.TopHeaderStyle.htm)


		 Свойство TopHeaderStyle
		 определяет стиль оформления заголовков столбцов таблицы.


		 ![](../../Property_Image.gif)
		 [ValueChangeStyle](IEaxTableStyle.ValueChangeStyle.htm)


		 Свойство ValueChangeStyle
		 определяет стиль оформления ячеек с настроенным преобразованием.


		 ![](../../Property_Image.gif)
		 [VirtualCellsStyle](IEaxTableStyle.VirtualCellsStyle.htm)


		 Свойство VirtualCellsStyle
		 определяет стиль ячеек, которые были получены в результате совмещения
		 рядов с различной календарной динамикой.


		 ![](../../Property_Image.gif)
		 [XmlDefinition](IEaxTableStyle.XmlDefinition.htm)


		 Свойство XmlDefinition
		 осуществляет экспорт/импорт оформления таблицы экспресс-отчета
		 в XML формате.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Assign](IEaxTableStyle.Assign.htm)


		 Метод Assign осуществляет
		 установку стиля оформления таблицы экспресс-отчета в соответствии
		 со стилем оформления, передаваемого посредством параметра.


		 ![](../../Sub_Image.gif)
		 [Clone](IEaxTableStyle.Clone.htm)


		 Метод Clone создает
		 копию оформления таблицы экспресс-отчета.


		 ![](../../Sub_Image.gif)
		 [Equal](IEaxTableStyle.Equal.htm)


		 Метод Equal осуществляет
		 сравнение оформлений таблицы экспресс-отчета с оформлением.
