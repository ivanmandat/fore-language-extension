## Описание


Интерфейс IWorkbookValidationPadContext
 предназначен для работы с контекстом вкладки с правилами валидации компонента
 [WorkbookConfigurationPanel](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/WorkbookConfigurationPanel.htm).
