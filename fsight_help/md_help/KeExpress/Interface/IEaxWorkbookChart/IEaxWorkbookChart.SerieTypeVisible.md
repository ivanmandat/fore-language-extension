# IEaxWorkbookChart.SerieTypeVisible


## Синтаксис


SerieTypeVisible(Serie: [EaxWorkbookChartSerieType](../../Enums/EaxWorkbookChartSerieType.htm)):
 Boolean;


## Параметры


Serie. Тип ряда.


## Описание


Свойство SerieTypeVisible определяет,
 отображается ли указанный тип ряда на диаграмме.


## Комментарии


Допустимые значения:


	- True. Ряд отображается.
	 Признак того, доступен ли ряд для отображения, возвращает свойство
	 [IEaxWorkbookChart.SerieTypeAvailable](IEaxWorkbookChart.SerieTypeAvailable.htm);


	- False. Ряд не отображается.
