# IEaxWorkbookChart.SerieTypeAvailable


## Синтаксис


SerieTypeAvailable(Serie: [EaxWorkbookChartSerieType](../../Enums/EaxWorkbookChartSerieType.htm)):
 Boolean;


## Параметры


Serie. Тип ряда.


## Описание


Свойство SerieTypeAvailable
 возвращает признак того, доступен ли указанный тип ряда для отображения
 на диаграмме.


## Комментарии


Допустимые значения:


	- True. Ряд может быть
	 отображен. Для отображения ряда используйте свойство [IEaxWorkbookChart.SerieTypeVisible](IEaxWorkbookChart.SerieTypeVisible.htm);


	- False. Ряд недоступен
	 для отображения.
