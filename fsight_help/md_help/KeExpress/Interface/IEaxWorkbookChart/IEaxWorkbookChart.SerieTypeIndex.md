# IEaxWorkbookChart.SerieTypeIndex


## Синтаксис


SerieTypeIndex(Serie: [EaxWorkbookChartSerieType](../../Enums/EaxWorkbookChartSerieType.htm)):
 Integer;


## Параметры


Serie. Тип ряда.


## Описание


Свойство SerieTypeIndex
 возвращает индекс для указанного типа ряда.


## Комментарии


Если указанный тип ряда отсутствует на диаграмме, то свойство вернет
 значение «-1».
