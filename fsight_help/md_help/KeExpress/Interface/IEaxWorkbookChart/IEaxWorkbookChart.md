# IEaxWorkbookChart


Сборка: Express;


## Описание


Интерфейс IEaxWorkbookChart
 предназначен для работы с параметрами диаграммы рабочей книги.


## Иерархия наследования


           IEaxWorkbookChart


## Комментарии


Для получения настроек диаграммы, используемой рабочей книгой, используйте
 свойство [IEaxWorkbook.Chart](../IEaxWorkbook/IEaxWorkbook.Chart.htm).
