# IEaxStatistics.CustomStatisticsVisible


## Синтаксис


CustomStatisticsVisible(Statistics: [EaxCustomStatisticsType](../../Enums/EaxCustomStatisticsType.htm)): Boolean;


## Параметры


Statistics. Статистическая характеристика.


## Описание


Свойство CustomStatisticsVisible определяет видимость указанной пользовательской статистической характеристики.


## Комментарии


Допустимые значения:


-
True. Статистическая характеристика отображается;


-
False. Статистическая характеристика не отображается.
