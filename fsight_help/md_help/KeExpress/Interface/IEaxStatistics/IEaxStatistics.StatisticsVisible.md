# IEaxStatistics.StatisticsVisible


## Синтаксис


StatisticsVisible(Statistics: [TsStatisticsType](KeTransform.chm::/Enums/TsStatisticsType.htm)): Boolean;


## Параметры


Statistics. Статистическая характеристика.


## Описание


Свойство StatisticsVisible определяет видимость указанной статистической характеристики.


## Комментарии


Допустимые значения:


-
True. Статистическая характеристика отображается;


-
False. Статистическая характеристика не отображается.
