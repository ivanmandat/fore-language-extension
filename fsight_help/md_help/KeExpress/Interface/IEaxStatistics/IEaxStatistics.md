# IEaxStatistics


Сборка: Express;


## Описание


Интерфейс IEaxStatistics предназначен
 для работы со статистическими характеристиками.


## Иерархия наследования


           IEaxStatistics


## Комментарии


Статистические характеристики возвращает свойство [IEaxAnalyzeCore.Statistics](../IEaxAnalyzeCore/IEaxAnalyzeCore.Statistics.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AllStatistics](IEaxStatistics.AllStatistics.htm)
		 Свойство AllStatistics
		 возвращает все статистические характеристики.


		 ![](../../Property_Image.gif)
		 [CustomStatisticsVisible](IEaxStatistics.CustomStatisticsVisible.htm)
		 Свойство CustomStatisticsVisible
		 определяет видимость указанной дополнительной статистической
		 характеристики.


		 ![](../../Property_Image.gif)
		 [Statistics](IEaxStatistics.Statistics.htm)
		 Свойство Statistics
		 возвращает только рассчитанные статистические характеристики.


		 ![](../../Property_Image.gif)
		 [StatisticsVisible](IEaxStatistics.StatisticsVisible.htm)
		 Свойство StatisticsVisible
		 определяет видимость указанной статистической характеристики.
