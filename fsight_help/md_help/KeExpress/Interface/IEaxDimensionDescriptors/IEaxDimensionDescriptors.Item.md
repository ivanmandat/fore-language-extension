# IEaxDimensionDescriptors.Item


## Синтаксис


Item(Index: Integer): [IEaxDimensionDescriptor](../IEaxDimensionDescriptor/IEaxDimensionDescriptor.htm);


## Параметры


Index. Индекс описания измерения.


## Описание


Свойство Item возвращает описание
 измерения по индексу.


## Комментарии


Для получения количества описаний измерения используйте [IEaxDimensionDescriptors.Count](IEaxDimensionDescriptors.Count.htm).
