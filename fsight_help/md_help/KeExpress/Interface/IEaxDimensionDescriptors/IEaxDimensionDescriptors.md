# IEaxDimensionDescriptors


Сборка: Express;


## Описание


Интерфейс IEaxDimensionDescriptors используется
 для работы с коллекцией описаний измерения.


## Иерархия наследования

## Комментарии


Для работы с описанием измерения используйте свойства и методы интерфейса
 [IEaxDimensionDescriptor](../IEaxDimensionDescriptor/IEaxDimensionDescriptor.htm).
