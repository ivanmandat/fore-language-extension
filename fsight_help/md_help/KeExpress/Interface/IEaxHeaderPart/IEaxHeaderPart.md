# IEaxHeaderPart


Сборка: Express;


## Описание


Интерфейс IEaxHeaderPart содержит
 свойства и методы части колонтитула экспресс-отчета.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Color](IEaxHeaderPart.Color.htm)
		 Свойство Color определяет
		 цвет шрифта части колонтитула экспресс-отчета.


		 ![](../../Property_Image.gif)
		 [Font](IEaxHeaderPart.Font.htm)
		 Свойство Font определяет
		 шрифт части колонтитула экспресс-отчета.


		 ![](../../Property_Image.gif)
		 [Image](IEaxHeaderPart.Image.htm)
		 Свойство Image определяет
		 изображение для части колонтитула.


		 ![](../../Property_Image.gif)
		 [ImageAsBackground](IEaxHeaderPart.ImageAsBackground.htm)
		 Свойство ImageAsBackground
		 определяет, использовать ли изображение в колонтитуле как фон.


		 ![](../../Property_Image.gif)
		 [ImageDocument](IEaxHeaderPart.ImageDocument.htm)
		 Свойство ImageDocument
		 определяет объект репозитория «Документ» для части колонтитула.


		 ![](../../Property_Image.gif)
		 [ImageHeight](IEaxHeaderPart.ImageHeight.htm)
		 Свойство ImageHeight
		 определяет высоту изображения для части колонтитула.


		 ![](../../Property_Image.gif)
		 [ImageTransparentColor](IEaxHeaderPart.ImageTransparentColor.htm)
		 Свойство ImageTransparentColor
		 определяет цвет, который будет прозрачным на изображении.


		 ![](../../Property_Image.gif)
		 [ImageWidth](IEaxHeaderPart.ImageWidth.htm)
		 Свойство ImageWidth
		 определяет ширину изображения для части колонтитула.


		 ![](../../Property_Image.gif)
		 [Text](IEaxHeaderPart.Text.htm)
		 Свойство Text определяет
		 текст части колонтитула экспресс-отчета.
