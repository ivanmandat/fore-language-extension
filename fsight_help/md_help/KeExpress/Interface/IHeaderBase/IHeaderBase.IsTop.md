# IHeaderBase.IsTop


## Синтаксис


IsTop: Boolean;


## Описание


Свойство IsTop возвращает тип
 используемого колонтитула.


## Комментарии


При значении True используется
 верхний заголовок/колонтитул, при значении False
 - нижний.
