# IHeaderBase


Сборка: Express;


## Описание


Интерфейс IHeaderBase содержит
 базовые свойства частей колонтитула/заголовка при печати.


## Иерархия наследования


IHeaderBase


## Комментарии


Интерфейс является базовым для интерфейса [IHeader](../IHeader/IHeader.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Center](IHeaderBase.Center.htm)
		 Свойство Center возвращает
		 центральную часть заголовка/колонтитула.


		 ![](../../Property_Image.gif)
		 [IsTop](IHeaderBase.IsTop.htm)
		 Свойство IsTop возвращает
		 тип используемого колонтитула.


		 ![](../../Property_Image.gif)
		 [Left](IHeaderBase.Left.htm)
		 Свойство Left возвращает
		 левую часть заголовка/колонтитула.


		 ![](../../Property_Image.gif)
		 [Printable](IHeaderBase.Printable.htm)
		 Свойство Printable
		 определяет признак вывода на печать заголовка/колонтитула.


		 ![](../../Property_Image.gif)
		 [Right](IHeaderBase.Right.htm)
		 Свойство Right возвращает
		 правую часть заголовка/колонтитула.
