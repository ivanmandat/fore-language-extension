# IReportDataSourcesData.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 источников данных отчета.


## Комментарии


Для получения источника данных используйте свойство [IReportDataSourcesData.Item](IReportDataSourcesData.Item.htm).
