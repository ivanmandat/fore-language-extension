# IReportDataSourcesData.Item


## Синтаксис


Item(Index: Integer): [IReportDataSourceData](../IReportDataSourceData/IReportDataSourceData.htm);


## Параметры


Index. Индекс источника данных.


## Описание


Свойство Item возвращает источник
 данных отчета.


## Комментарии


Для получения количества источников данных используйте свойство [IReportDataSourcesData.Count](IReportDataSourcesData.Count.htm).
