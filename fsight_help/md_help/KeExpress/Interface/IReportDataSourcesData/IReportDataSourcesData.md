# IReportDataSourcesData


Сборка: Express;


## Описание


Интерфейс IReportDataSourcesData
 предназначен для работы с коллекцией данных из источников экспресс-отчета.


## Иерархия наследования


           IReportDataSourcesData


## Комментарии


Для получения объекта данного типа используйте свойство [IReportSheetData.DataSources](../IReportSheetData/IReportSheetData.DataSources.htm).
