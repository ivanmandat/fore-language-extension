# IEaxDataAreaSliceProperties.MultipleAttributesEnabled


## Синтаксис


MultipleAttributesEnabled: Boolean;


## Описание


Свойство MultipleAttributesEnabled
 определяет, используется ли отображение нескольких атрибутов справочника.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Отображение используется;


	- False. Отображение не
	 используется.
