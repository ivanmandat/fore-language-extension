# IEaxDataAreaSliceProperties


Сборка: Express;


## Описание


Интерфейс IEaxDataAreaSliceProperties
 используется для работы со свойствами среза источника данных.


## Иерархия наследования


           IEaxDataAreaSliceProperties


## Комментарии


Для получения объекта данного типа используйте [IEaxDataAreaSlice.Properties](../IEaxDataAreaSlice/IEaxDataAreaSlice.Properties.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DimCalendarViewMode](IEaxDataAreaSliceProperties.DimCalendarViewMode.htm)
		 Свойство DimCalendarViewMode
		 определяет настройку режима отображения календарных измерений.


		 ![](../../Property_Image.gif)
		 [MultipleAttributesEnabled](IEaxDataAreaSliceProperties.MultipleAttributesEnabled.htm)
		 Свойство MultipleAttributesEnabled
		 определяет, используется ли отображение нескольких атрибутов справочника.


		 ![](../../Property_Image.gif)
		 [MultipleAttributesSeparator](IEaxDataAreaSliceProperties.MultipleAttributesSeparator.htm)
		 Свойство MultipleAttributesSeparator
		 определяет символ-разделитель между наименованиями атрибутов.


		 ![](../../Property_Image.gif)
		 [MultipleAttributesSpaced](IEaxDataAreaSliceProperties.MultipleAttributesSpaced.htm)
		 Свойство MultipleAttributesSpaced
		 определяет, используются ли пробелы между наименованиями атрибутов.


		 ![](../../Property_Image.gif)
		 [Slice](IEaxDataAreaSliceProperties.Slice.htm)
		 Свойство Slice
		 возвращает объект-родитель.


		 ![](../../Property_Image.gif)
		 [UseCustomDimensionNamingAttribute](IEaxDataAreaSliceProperties.UseCustomDimensionNamingAttribute.htm)
		 Свойство UseCustomDimensionNamingAttribute
		 определяет, будут ли для элементов измерения использоваться пользовательские
		 наименования.


		 ![](../../Property_Image.gif)
		 [UseGridSelection](IEaxDataAreaSliceProperties.UseGridSelection.htm)
		 Свойство UseGridSelection
		 определяет, будет ли выделенная область таблицы, построенная на
		 срезе данных, использоваться для построения других визуализаторов
		 (диаграмм, карт и прочих).
