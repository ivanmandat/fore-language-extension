# ITableSource


Сборка: Express;


## Описание


Интерфейс ITableSource предназначен
 для работы с параметрами источника-таблицы.


## Иерархия наследования


           ITableSource


## Комментарии


Интерфейс используется для передачи данных из компонентов [UiErAnalyzer](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiErAnalyzer.htm) и [UiReport](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiReport.htm) в компонент
 [TableWidget](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/TableWidget.htm).


Для работы с базовыми свойствами компонента [TableWidget](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/TableWidget.htm)
 используйте свойства интерфейса [ITableWidget](../ITableWidget/ITableWidget.htm).
