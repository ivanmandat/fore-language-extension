# IEaxDataAreaSources


Сборка: Express;


## Описание


Интерфейс IEaxDataAreaSources
 предназначен для работы с коллекцией источников данных аналитической области.


## Иерархия наследования


           IEaxDataAreaSources


## Комментарии


Для работы с конкретным источником данных аналитической области используйте
 интерфейс [IEaxDataAreaSource](../IEaxDataAreaSource/IEaxDataAreaSource.htm).
