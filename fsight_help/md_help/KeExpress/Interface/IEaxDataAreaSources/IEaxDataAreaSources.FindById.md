# IEaxDataAreaSources.FindById


## Синтаксис


		FindById(Id: String): [IEaxDataAreaSource](../IEaxDataAreaSource/IEaxDataAreaSource.htm);


## Параметры


Id. Идентификатор источника
 данных аналитической области.


## Описание


Метод FindById осуществляет
 поиск источника данных аналитической области по идентификатору.


## Комментарии


Для определения идентификатора источника данных аналитической области
 используйте свойство [IEaxDataAreaSource.Id](ForeSys.chm::/Interface/INamedEntityModel/INamedEntityModel.Id.htm).
