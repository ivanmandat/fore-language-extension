# IEaxDataAreaSources.FindByKey


## Синтаксис


		FindByKey(Key: Integer): [IEaxDataAreaSource](../IEaxDataAreaSource/IEaxDataAreaSource.htm);


## Параметры


Key. Ключ источника данных
 аналитической области.


## Описание


Метод FindByKey осуществляет
 поиск источника данных аналитической области по ключу.


## Комментарии


Для определения ключа источника данных аналитической области используйте
 свойство [IEaxDataAreaSource.Key](ForeSys.chm::/Interface/INamedEntityModel/INamedEntityModel.Key.htm).
