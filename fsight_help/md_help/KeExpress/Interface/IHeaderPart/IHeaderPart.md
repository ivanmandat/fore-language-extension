# IHeaderPart


Сборка: Express;


## Описание


Интерфейс IHeaderPart предназначен
 для работы с частью заголовка/колонтитула.


## Иерархия наследования


IHeaderPart


## Комментарии


Части заголовка/колонтитула можно получить с помощью свойств [IHeaderBase.Center](../IHeaderBase/IHeaderBase.Center.htm),
 [IHeaderBase.Left](../IHeaderBase/IHeaderBase.Left.htm), [IHeaderBase.Right](../IHeaderBase/IHeaderBase.Right.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Color](IHeaderPart.Color.htm)
		 Свойство Color определяет
		 цвет шрифта части заголовка/колонтитула.


		 ![](../../Property_Image.gif)
		 [Disposition](IHeaderPart.Disposition.htm)
		 Свойство Disposition
		 возвращает способ расположения части заголовка/колонтитула.


		 ![](../../Property_Image.gif)
		 [Font](IHeaderPart.Font.htm)
		 Свойство Font определяет
		 шрифт части заголовка/колонтитула.


		 ![](../../Property_Image.gif)
		 [Image](IHeaderPart.Image.htm)
		 Свойство Image определяет
		 изображение для части заголовка/колонтитула.


		 ![](../../Property_Image.gif)
		 [ImageAsBackground](IHeaderPart.ImageAsBackground.htm)
		 Свойство ImageAsBackground
		 определяет использование изображения в заголовке/колонтитуле в
		 качестве фона.


		 ![](../../Property_Image.gif)
		 [ImageDocument](IHeaderPart.ImageDocument.htm)
		 Свойство ImageDocument
		 определяет объект репозитория «[Документ](UiNavObj.chm::/UiNavObj_document.htm)»
		 для части заголовка/колонтитула.


		 ![](../../Property_Image.gif)
		 [ImageHeight](IHeaderPart.ImageHeight.htm)
		 Свойство ImageHeight
		 определяет высоту изображения для части заголовка/колонтитула.


		 ![](../../Property_Image.gif)
		 [ImageTransparentColor](IHeaderPart.ImageTransparentColor.htm)
		 Свойство ImageTransparentColor
		 определяет цвет, который будет прозрачным на изображении.


		 ![](../../Property_Image.gif)
		 [ImageWidth](IHeaderPart.ImageWidth.htm)
		 Свойство ImageWidth
		 определяет ширину изображения для части заголовка/колонтитула.


		 ![](../../Property_Image.gif)
		 [Text](IHeaderPart.Text.htm)
		 Свойство Text определяет
		 текст части заголовка/колонтитула.
