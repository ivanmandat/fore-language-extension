# IEaxDataAreaExternalTransformation.EnableArithmeticErrors


## Синтаксис


EnableArithmeticErrors: Boolean;


## Описание


Свойство EnableArithmeticErrors
 определяет результат выполнения задачи моделирования при делении на 0.


## Комментарии


Допустимые значения:


	- True. Доступно отображение
	 ошибок при делении на 0;


	- False. Недоступно отображение
	 ошибок при делении на 0.
