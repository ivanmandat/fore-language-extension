# IEaxDataAreaExternalTransformation.Problem


## Синтаксис


Problem: Object;


## Описание


Свойство Problem определяет
 задачу моделирования при обновлении регламентного отчёта или изменении
 данных.


## Комментарии


Задача моделирования задается к срезу аналитической области данных.
 Для получения коллекции срезов аналитической области данных используйте
 свойство [IEaxDataArea.Slices](../IEaxDataArea/IEaxDataArea.Slices.htm).


## Пример


Использование свойства приведено в примере для [IEaxDataAreaPivotSlice.ExternalTransformations](../IEaxDataAreaPivotSlice/IEaxDataAreaPivotSlice.ExternalTransformations.htm).
