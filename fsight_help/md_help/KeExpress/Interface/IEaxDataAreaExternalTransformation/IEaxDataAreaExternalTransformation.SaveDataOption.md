# IEaxDataAreaExternalTransformation.SaveDataOption


## Синтаксис


SaveDataOption: Boolean;


## Описание


Свойство SaveDataOption определят
 признак сохранения данных в источник при вычислении значений [алгоритмом
 расчёта на изменение](DataEntryForms.chm::/Desktop/Table/Calculation_Practices.htm#addalgorithm).


## Комментарии


Допустимые значения:


	- True. Значения, рассчитанные
	 данным алгоритмом, сохраняются в источнике данных. Значение по умолчанию;


	- False. Значения, рассчитанные
	 данным алгоритмом, не сохраняются в источнике данных.
