# IEaxDimensionInstance


Сборка: Express;


## Описание


Интерфейс IEaxDimensionInstance
 используется для работы с данными измерения, включая отметку.


## Иерархия наследования

## Комментарии


Для получения объекта данного типа используйте [IEaxDataAreaSlice.GetDimension](../IEaxDataAreaSlice/IEaxDataAreaSlice.GetDimension.htm).
