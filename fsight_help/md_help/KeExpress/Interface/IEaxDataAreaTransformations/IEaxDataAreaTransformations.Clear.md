# IEaxDataAreaTransformations.Clear


## Синтаксис


Clear;


## Описание


Метод Clear очищает коллекцию
 формул преобразования данных.


## Комментарии


После вызова данного метода значение свойства [IEaxDataAreaTransformations.Count](IEaxDataAreaTransformations.Count.htm)
 принимает значение «0».


Для добавления формулы преобразования данных в коллекцию используйте
 [IEaxDataAreaTransformations.Add](IEaxDataAreaTransformations.Add.htm).
