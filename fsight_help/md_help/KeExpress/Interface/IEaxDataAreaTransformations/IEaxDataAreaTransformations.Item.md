# IEaxDataAreaTransformations.Item


## Синтаксис


Item(Index: Integer): [IEaxDataAreaTransformation](../IEaxDataAreaTransformation/IEaxDataAreaTransformation.htm);


## Параметры


Index. Индекс формулы преобразования
 данных.


## Описание


Свойство Item возвращает формулу
 преобразования данных по ее индексу.


## Комментарии


Для получения количества формул преобразования данных в коллекции используйте
 [IEaxDataAreaTransformations.Count](IEaxDataAreaTransformations.Count.htm).


Для поиска формулы преобразования данных по ее ключу используйте [IEaxDataAreaTransformations.RemoveByKey](IEaxDataAreaTransformations.RemoveByKey.htm).
