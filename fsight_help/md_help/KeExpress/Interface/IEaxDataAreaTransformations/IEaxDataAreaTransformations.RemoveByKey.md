# IEaxDataAreaTransformations.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ формулы преобразования
 данных.


## Описание


Метод RemoveByKey удаляет формулу
 преобразования данных из коллекции по ее ключу.


## Комментарии


Если формула преобразования будет удалена, то данный метод вернет значение
 True.


Для удаления формулы преобразования данных из коллекции по ее индексу
 используйте [IEaxDataAreaTransformations.Remove](IEaxDataAreaTransformations.Remove.htm).
