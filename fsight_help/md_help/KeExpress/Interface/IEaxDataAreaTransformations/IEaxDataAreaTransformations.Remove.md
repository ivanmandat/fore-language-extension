# IEaxDataAreaTransformations.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс формулы преобразования
 данных.


## Описание


Метод Remove удаляет формулу
 преобразования данных из коллекции по ее индексу.


## Комментарии


Если формула преобразования будет удалена, то данный метод вернет значение
 True.


Для удаления формулы преобразования данных из коллекции по ее ключу
 используйте [IEaxDataAreaTransformations.RemoveByKey](IEaxDataAreaTransformations.RemoveByKey.htm).
