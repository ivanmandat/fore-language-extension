# IEaxDataAreaTransformations.MoveTo


## Синтаксис


MoveTo(FromIndex: Integer; ToIndex: Integer);


## Параметры


FromIndex. Индекс позиции формулы
 преобразования данных;


ToIndex. Индекс позиции, в
 которую необходимо переместить формулу преобразования данных.


## Описание


Метод MoveTo осуществляет перемещение
 формулы преобразования данных внутри коллекции.


## Комментарии


Значения параметров FromIndex, ToIndex должны быть неотрицательными
 и меньше значения [IEaxDataAreaTransformations.Count](IEaxDataAreaTransformations.Count.htm).


Для получения индекса указанной формулы преобразования данных используйте
 [IEaxDataAreaTransformations.IndexOf](IEaxDataAreaTransformations.IndexOf.htm).
