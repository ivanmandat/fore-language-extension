# IEaxDataAreaTransformations


Сборка: Express;


## Описание


Интерфейс IEaxDataAreaTransformations
 используется для работы с коллекцией формул преобразования данных аналитической
 области.


## Иерархия наследования


           IEaxDataAreaTransformations


## Комментарии


Для получения коллекции формул преобразования данных аналитической области
 используйте:


	- [IEaxDataAreaSlice.AdditionalFilterTransformations](../IEaxDataAreaSlice/IEaxDataAreaSlice.AdditionalFilterTransformations.htm);


	- [IEaxDataAreaSlice.CalcTransformations](../IEaxDataAreaSlice/IEaxDataAreaSlice.CalcTransformations.htm);


	- [IEaxDataAreaSlice.FilterTransformations](../IEaxDataAreaSlice/IEaxDataAreaSlice.FilterTransformations.htm);


	- [IEaxDataAreaSlice.SortTransformations](../IEaxDataAreaSlice/IEaxDataAreaSlice.SortTransformations.htm).


Для получения коллекции формул преобразования данных альтернативной
 иерархии аналитической области используйте:


	- [IEaxDataAreaHierarchy.FilterTransformations](../IEaxDataAreaHierarchy/IEaxDataAreaHierarchy.FilterTransformations.htm);


	- [IEaxDataAreaHierarchy.SortTransformations](../IEaxDataAreaHierarchy/IEaxDataAreaHierarchy.SortTransformations.htm).

## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IEaxDataAreaTransformations.Add.htm)
		 Метод Add добавляет
		 формулу преобразования данных в коллекцию.


		 ![](../../Sub_Image.gif)
		 [AddFunction](IEaxDataAreaTransformations.AddFunction.htm)
		 Метод AddFunction добавляет

		 итог.


		 ![](../../Sub_Image.gif)
		 [Arrange](IEaxDataAreaTransformations.Arrange.htm)
		 Метод Arrange автоматически
		 выстраивает итоги в цепочке расчета.


		 ![](../../Sub_Image.gif)
		 [ArrangeTransformation](IEaxDataAreaTransformations.ArrangeTransformation.htm)
		 Метод ArrangeTransformation
		 автоматически в цепочке расчета выстраивает итоги, используемые
		 в формулах преобразования данных.


		 ![](../../Sub_Image.gif)
		 [Clear](IEaxDataAreaTransformations.Clear.htm)
		 Метод Clear очищает
		 коллекцию формул преобразования данных.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IEaxDataAreaTransformations.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск формулы преобразования данных по ее ключу.


		 ![](../../Sub_Image.gif)
		 [IndexOf](IEaxDataAreaTransformations.IndexOf.htm)
		 Метод IndexOf возвращает
		 индекс указанной формулы преобразования данных.


		 ![](../../Sub_Image.gif)
		 [MoveTo](IEaxDataAreaTransformations.MoveTo.htm)
		 Метод MoveTo осуществляет
		 перемещение формулы преобразования данных внутри коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IEaxDataAreaTransformations.Remove.htm)
		 Метод Remove удаляет
		 формулу преобразования данных из коллекции по ее индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IEaxDataAreaTransformations.RemoveByKey.htm)
		 Метод RemoveByKey удаляет
		 формулу преобразования данных из коллекции по ее ключу.
