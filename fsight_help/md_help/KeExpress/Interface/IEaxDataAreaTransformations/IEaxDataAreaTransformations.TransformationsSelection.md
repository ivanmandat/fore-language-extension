# IEaxDataAreaTransformations.TransformationsSelection


## Синтаксис


TransformationsSelection: [IDimSelectionSet](KeDims.chm::/interface/IDimSelectionSet/IDimSelectionSet.htm);


## Описание


Свойство TransformationsSelection
 возвращает отметку элементов, для которых настроена формула преобразования
 данных.


## Комментарии


Для получения массива элементов измерения, для которых настроена формула
 преобразования данных, используйте [IEaxDataAreaTransformations.TransformationElements](IEaxDataAreaTransformations.TransformationElements.htm).
