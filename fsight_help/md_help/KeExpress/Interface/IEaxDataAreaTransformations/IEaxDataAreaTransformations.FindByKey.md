# IEaxDataAreaTransformations.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IEaxDataAreaTransformation](../IEaxDataAreaTransformation/IEaxDataAreaTransformation.htm);


## Параметры


Key. Ключ формулы преобразования
 данных.


## Описание


Метод FindByKey осуществляет
 поиск формулы преобразования данных по ее ключу.


## Комментарии


Для получения формулы преобразования данных по ее индексу используйте
 [IEaxDataAreaTransformations.Item](IEaxDataAreaTransformations.Item.htm).
