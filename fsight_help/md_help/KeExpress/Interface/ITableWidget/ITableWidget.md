# ITableWidget


Сборка: Express;


## Описание


Интерфейс ITableWidget предназначен
 для работы с базовыми свойствами компонента [TableWidget](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/TableWidget.htm).


## Иерархия наследования


           [IWidget](ModForms.chm::/Interface/IWidget/IWidget.htm)


           ITableWidget


## Комментарии


Для активации компонента [TableWidget](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/TableWidget.htm) в
 инспекторе объектов среды разработки вручную установите свойству Active значение True.
