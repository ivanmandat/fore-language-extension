# IEaxMapChart


Сборка: Express;


## Описание


Интерфейс IEaxMapChart предназначен
 для работы с картой, использующей для отрисовки технологию OpenGL.


## Иерархия наследования


           [IEaxObject](../IEaxObject/IEaxObject.htm)


           [IEaxVisualizer](../IEaxVisualizer/IEaxVisualizer.htm)


           IEaxMapChart


## Комментарии


OpenGL (Open Graphics Library - открытая графическая библиотека) - спецификация,
 определяющая независимый от языка программирования кросс-платформенный
 программный интерфейс для написания приложений, использующих двумерную
 и трёхмерную компьютерную графику.
