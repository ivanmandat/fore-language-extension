# IEaxMapChart.DoAction


## Синтаксис


DoAction(Argument: Variant);


## Параметры


Argument. Идентификатор территории.


## Описание


Метод DoAreaAction генерирует
 событие щелчка кнопки мыши на территории карты.


## Комментарии


Модуль для обработки событий устанавливается с помощью свойства [IEaxMapChart.Action](IEaxMapChart.Action.htm).
