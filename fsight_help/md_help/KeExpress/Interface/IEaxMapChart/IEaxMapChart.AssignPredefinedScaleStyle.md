# IEaxMapChart.AssignPredefinedScaleStyle


## Синтаксис


AssignPredefinedScaleStyle(Style: [EaxMapPredefinedScaleStyle](../../Enums/EaxMapPredefinedScaleStyle.htm));


## Параметры


Style. Стиль распределения
 цветов.


## Описание


Метод AssignPredefinedScaleStyle
 устанавливает для карты указанный стандартный стиль распределения цветов,
 которые будут использоваться для оформления территорий в зависимости от
 значений.
