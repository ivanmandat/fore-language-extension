# IEaxGridDimensionSettings.Drilled


## Синтаксис


Drilled: [TriState](ForeSys.chm::/Enums/TriState.htm);


## Описание


Свойство Drilled определяет,
 будет ли использоваться [расшифровка
 элементов измерения](UiAnalyticalArea.chm::/Hyperlinks/Working_with_hyperlinks.htm).


## Комментарии


Для получения признака того, используется ли [расшифровка
 элементов измерения](UiAnalyticalArea.chm::/Hyperlinks/Working_with_hyperlinks.htm), используйте [IEaxGridDimensionSettings.IsDrilled](IEaxGridDimensionSettings.IsDrilled.htm).
