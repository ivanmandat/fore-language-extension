# IEaxGridDimensionSettings.IsOnlyFirstNamingHyperlinkShown


## Синтаксис


IsOnlyFirstNamingHyperlinkShown: Boolean;


## Описание


Метод IsOnlyFirstNamingHyperlinkShown
 возвращает признак того, отображаются ли гиперссылки только у атрибута
 измерения, который размещен первым по порядку для [отображения
 наименований элементов измерений в рабочей области](UiSelection.chm::/selection/element_names.htm#user).


## Комментарии


Допустимые значения:


	- True. Гиперссылки отображаются
	 только у атрибута измерения, который размещен первым по порядку;


	- False. Гиперссылки отображаются
	 у всех атрибутов измерения.
