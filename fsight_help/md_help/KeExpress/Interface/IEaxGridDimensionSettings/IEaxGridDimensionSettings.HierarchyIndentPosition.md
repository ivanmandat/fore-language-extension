# IEaxGridDimensionSettings.HierarchyIndentPosition


## Синтаксис


HierarchyIndentPosition: Integer;


## Описание


Свойство HierarchyIndentPosition
 определяет позицию отступа для измерений.


## Комментарии


При использовании свойства HierarchyIndentPosition
 положение отступа изменится на «[Фиксированное](UiAnalyticalArea.chm::/TableView/Layout/Layout_Dim.htm#indent)». В качестве
 значения указывается номер столбца, в котором элементы различных уровней
 будут иметь отступы.
