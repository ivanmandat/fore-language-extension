# IEaxGridDimensionSettings.IsDrilled


## Синтаксис


IsDrilled: Boolean;


## Описание


Свойство IsDrilled возвращает
 признак того, используется ли [расшифровка
 элементов измерения](UiAnalyticalArea.chm::/Hyperlinks/Working_with_hyperlinks.htm)


## Комментарии


Допустимые значения:


	- True. Расшифровка используется;


	- False. По умолчанию.
	 Расшифровка не используется.


Для определения, будет ли использоваться [расшифровка
 элементов измерения](UiAnalyticalArea.chm::/Hyperlinks/Working_with_hyperlinks.htm), используйте [IEaxGridDimensionSettings.Drilled](IEaxGridDimensionSettings.Drilled.htm).
