# IEaxGridDimensionSettings.HierarchyIndent


## Синтаксис


HierarchyIndent: Double;


## Описание


Свойство HierarchyIndent определяет
 размер отступа.


## Комментарии


При использовании свойства HierarchyIndent
 способ задания размера отступа изменится на «[Вручную](UiAnalyticalArea.chm::/TableView/Layout/Layout_Dim.htm#indent)». Размер
 отступа указывается в миллиметрах. По умолчанию используется отступ 3
 миллиметра.
