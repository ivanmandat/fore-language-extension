# IEaxGridDimensionSettings.Drill


## Синтаксис


Drill: [IEaxDrillSettings](../IEaxDrillSettings/IEaxDrillSettings.htm);


## Описание


Свойство Drill возвращает объект
 для работы с [расшифровкой
 элементов измерения](UiAnalyticalArea.chm::/Hyperlinks/Working_with_hyperlinks.htm).


## Комментарии


Для работы с [расшифровкой
 элементов измерения](UiAnalyticalArea.chm::/Hyperlinks/Working_with_hyperlinks.htm) необходимо, чтобы свойство IEaxGridViewSettings.HyperlinkAsText
 принимало значение False.
