# IEaxGridDimensionSettings


Сборка: Express;


## Описание


Интерфейс IEaxGridDimensionSettings
 используется для работы с настройками измерения таблицы.


## Иерархия наследования


           IEaxGridDimensionSettings


## Комментарии


Для получения объекта данного типа необходимо привести объект типа [IEaxGridHeaderProperties](../IEaxGridHeaderProperties/IEaxGridHeaderProperties.htm)
 к IEaxGridDimensionSettings.
