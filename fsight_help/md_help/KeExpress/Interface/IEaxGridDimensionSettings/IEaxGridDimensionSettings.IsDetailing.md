# IEaxGridDimensionSettings.IsDetailing


## Синтаксис


IsDetailing: Boolean;


## Описание


Свойство IsDetailing возвращает
 признак того, используется ли [детализация
 данных](UiAnalyticalArea.chm::/Working_with_table_data/Drill_down.htm).


## Комментарии


Допустимые значения:


	- True. Детализация используется;


	- False. По умолчанию.
	 Детализация не используется.


Для определения, будет ли использоваться [детализация
 данных](UiAnalyticalArea.chm::/Working_with_table_data/Drill_down.htm), используйте свойство [IEaxGridDimensionSettings.Detailing](IEaxGridDimensionSettings.Detailing.htm).
