# ILanerResultsBoxResults


Сборка: Express;


## Описание


Интерфейс ILanerResultsBoxResults
 предназначен для работы с данными вкладок на панели «Результаты
 вычислений» компонента LanerResultsBox.


## Комментарии


Данный метод реализует доступ к значениям, выводимым на панель «Результаты вычислений». Все значения
 предназначены только для чтения.
