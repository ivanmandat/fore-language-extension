# ILanerResultsBoxResults.Statistics


## Синтаксис


Statistics: [IEaxStatistics](../IEaxStatistics/IEaxStatistics.htm);


## Описание


Свойство Statistics возвращает
 параметры для работы со статистиками на вкладке «Статистические
 характеристики».


## Комментарии


Если ряд, для которого рассчитаны статистики, вычисляемый (диапазон
 данных принадлежит только вычисляемому ряду) и получен с помощью методов
 моделирования (сглаживание, прогноз, регрессия), то набор характеристик
 подобен [статистическим
 характеристикам](uimodelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_2_Model/Standart_Model/statistical_features.htm) контейнера моделирования.
