# IAlgorithmConstantParamControl


Сборка: Express;


## Описание


Интерфейс IAlgorithmConstantParamControl
 содержит свойства для настройки управления параметром алгоритма с помощью
 константного значения.


## Иерархия наследования


           [IAlgorithmParamControl](../IAlgorithmParamControl/IAlgorithmParamControl.htm)


           IAlgorithmConstantParamControl


## Комментарии


Для получения настроек приведите значение свойства [IAlgorithmParam.Control](../IAlgorithmParam/IAlgorithmParam.Control.htm)
 к интерфейсу IAlgorithmConstantParamControl. Актуально, если свойство
 [IAlgorithmParam.Type](../IAlgorithmParam/IAlgorithmParam.Type.htm)
 имеет значение [AlgorithmParamType.Constant](../../Enums/AlgorithmParamType.htm).
