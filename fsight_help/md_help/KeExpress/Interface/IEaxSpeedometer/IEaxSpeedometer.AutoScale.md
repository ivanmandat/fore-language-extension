# IEaxSpeedometer.AutoScale


## Синтаксис


AutoScale: Boolean;


## Описание


Свойство AutoScale определяет
 признак автоматического расчета шкалы спидометра.


## Комментарии


Если свойство имеет значение True,
 то шкала спидометра рассчитывается автоматически в зависимости от фактических
 значений ряда (рядов).


При значении False шкала рассчитывается
 в зависимости от значений свойств [IEaxSpeedometer.MinValue](IEaxSpeedometer.MinValue.htm)
 и [IEaxSpeedometer.MaxValue](IEaxSpeedometer.MaxValue.htm).
