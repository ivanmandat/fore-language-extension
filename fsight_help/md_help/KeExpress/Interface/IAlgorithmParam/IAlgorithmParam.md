# IAlgorithmParam


Сборка: Express;


## Описание


Интерфейс IAlgorithmParam предназначен
 для настройки управления параметром алгоритма расчёта, запускаемого по
 пользовательской кнопке.


## Иерархия наследования


           [INamedEntityModel](ForeSys.chm::/Interface/INamedEntityModel/INamedEntityModel.htm)


           IAlgorithmParam


## Комментарии


Настройки управления параметром можно получить, используя свойства и
 методы интерфейса [IAlgorithmParams](../IAlgorithmParams/IAlgorithmParams.htm).
