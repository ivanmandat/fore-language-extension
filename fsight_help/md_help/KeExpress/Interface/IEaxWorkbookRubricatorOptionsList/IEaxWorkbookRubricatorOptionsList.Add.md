# IEaxWorkbookRubricatorOptionsList.Add


## Синтаксис


Add(RubricatorObject: [IMetabaseObjectDescriptor](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm)):
 [IEaxWorkbookRubricatorOptions](../IEaxWorkbookRubricatorOptions/IEaxWorkbookRubricatorOptions.htm);


## Параметры


RubricatorObject. Описание
 базы данных временных рядов как объекта репозитория.


## Описание


Метод Add добавляет настройки
 для указанной базы данных временных рядов в коллекцию.


## Комментарии


Для удаления настроек для указанной базы данных временных рядов используйте
 метод [IEaxWorkbookRubricatorOptionsList.Remove](IEaxWorkbookRubricatorOptionsList.Remove.htm).


## Пример

 настроек дерева объектов рабочей книги](../../Samples/KeExpress_EaxWorkbook_fore.htm)».
