# IEaxWorkbookRubricatorOptionsList


Сборка: Express;


## Описание


Интерфейс IEaxWorkbookRubricatorOptionsList
 используется для работы с коллекцией настроек баз данных временных рядов,
 которые использует [дерево
 показателей](UiDw.chm::/Purpose/UiDw_Purpose_Object.htm).


## Иерархия наследования


           IEaxWorkbookRubricatorOptionsList


## Комментарии


Для получения коллекции настроек используйте свойство [IEaxWorkbookMetaAttributesTree.RubricatorsOptions](../IEaxWorkbookMetaAttributesTree/IEaxWorkbookMetaAttributesTree.RubricatorsOptions.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Item](IEaxWorkbookRubricatorOptionsList.Item.htm)
		 Свойство Item возвращает
		 настройки из коллекции для указанной базы данных временных рядов.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IEaxWorkbookRubricatorOptionsList.Add.htm)
		 Метод Add добавляет
		 настройки для указанной базы данных временных рядов в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](IEaxWorkbookRubricatorOptionsList.Clear.htm)
		 Метод Clear удаляет
		 все настройки из коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IEaxWorkbookRubricatorOptionsList.Remove.htm)
		 Метод Remove удаляет
		 настройки для указанной базы данных временных рядов.
