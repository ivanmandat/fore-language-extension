# IEaxWorkbookRubricatorOptionsList.Clear


## Синтаксис


Clear;


## Описание


Метод Clear удаляет все
 настройки из коллекции.


## Комментарии


Для удаления настроек для указанной базы данных временных рядов используйте
 метод [IEaxWorkbookRubricatorOptionsList.Remove](IEaxWorkbookRubricatorOptionsList.Remove.htm).


## Пример


Использование метода приведено в разделе «[Сохранение и загрузка
 настроек дерева объектов рабочей книги](../../Samples/KeExpress_EaxWorkbook_fore.htm)».
