# IEaxWorkbookRubricatorOptionsList.Item


## Синтаксис


Item(RubricatorKey: Integer): [IEaxWorkbookRubricatorOptions](../IEaxWorkbookRubricatorOptions/IEaxWorkbookRubricatorOptions.htm);


## Параметры


RubricatorKey. Ключ базы данных
 временных рядов.


## Описание


Свойство Item возвращает
 настройки из коллекции для указанной базы данных временных рядов.


## Комментарии


Если настройки для указанной базы данных отсутствуют в коллекции, то
 свойство возвращает значение Null.

 настроек дерева объектов рабочей книги](../../Samples/KeExpress_EaxWorkbook_fore.htm)».
