# IEaxHierarchySettings.CellStyle


## Синтаксис


CellStyle(Element: String): [IEaxDataAreaCellStyle](../IEaxDataAreaCellStyle/IEaxDataAreaCellStyle.htm);


## Параметры


Element. Элемент иерархии.


## Описание


Свойство CellStyle определяет
 стиль для указанного элемента иерархии.


## Комментарии


Для создания стиля используйте класс [EaxDataAreaCellStyle](../../Class/EaxDataAreaCellStyle/EaxDataAreaCellStyle.htm).
