# IEaxDataAreaControl.FindById


## Синтаксис


FindById(ParamId: String): [IEaxDataAreaParamControl](../IEaxDataAreaParamControl/IEaxDataAreaParamControl.htm);


## Параметры


ParamId. Идентификатор параметра.


## Описание


Метод FindById осуществляет
 поиск и возвращает параметр среза аналитической области по его идентификатору.


## Комментарии


Для определения количества параметров среза данных аналитической области
 используйте свойство [IEaxDataAreaControl.Count](IEaxDataAreaControl.Count.htm).
