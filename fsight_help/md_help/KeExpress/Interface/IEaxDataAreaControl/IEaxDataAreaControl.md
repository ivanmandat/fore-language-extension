# IEaxDataAreaControl


Сборка: Express;


## Описание


Интерфейс IEaxDataAreaControl
 используется для работы с параметрами среза аналитической области.


## Иерархия наследования


           IEaxDataAreaControl


## Комментарии


Для работы с объектом настройки управления параметрами среза аналитической
 области используйте интерфейс [IEaxDataAreaControlInfo](../IEaxDataAreaControlInfo/IEaxDataAreaControlInfo.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ControlledObject](IEaxDataAreaControl.ControlledObject.htm)
		 Свойство ControlledObject
		 возвращает срез аналитической области, для которого настраивается
		 управление параметрами.


		 ![](../../Property_Image.gif)
		 [Count](IEaxDataAreaControl.Count.htm)
		 Свойство Count возвращает
		 количество параметров среза аналитической области.


		 ![](../../Property_Image.gif)
		 [IsEmpty](IEaxDataAreaControl.IsEmpty.htm)
		 Свойство IsEmpty возвращает
		 признак отсутствия настроек управления параметрами среза аналитической
		 области.


		 ![](../../Property_Image.gif)
		 [Item](IEaxDataAreaControl.Item.htm)
		 Свойство Item возвращает
		 параметр среза аналитической области.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddControl](IEaxDataAreaControl.AddControl.htm)
		 Метод AddControl добавляет
		 настройку по управлению для параметра среза с указанным идентификатором.


		 ![](../../Sub_Image.gif)
		 [FindById](IEaxDataAreaControl.FindById.htm)
		 Метод FindById осуществляет
		 поиск и возвращает параметр среза аналитической области по его
		 идентификатору.


		 ![](../../Sub_Image.gif)
		 [RemoveById](IEaxDataAreaControl.RemoveById.htm)
		 Метод RemoveById осуществляет
		 удаление параметра среза из коллекции по его идентификатору.
