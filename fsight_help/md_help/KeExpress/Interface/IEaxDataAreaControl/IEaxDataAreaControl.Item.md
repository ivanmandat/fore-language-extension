# IEaxDataAreaControl.Item


## Синтаксис


Item(Index: Integer): [IEaxDataAreaParamControl](../IEaxDataAreaParamControl/IEaxDataAreaParamControl.htm);


## Параметры


Index. Индекс параметра.


## Описание


Свойство Item возвращает параметр
 среза аналитической области.


## Комментарии


Для определения количества объектов настройки управления параметрами
 среза аналитической области используйте свойство [IEaxDataAreaControlInfo.Count](../IEaxDataAreaControlInfo/IEaxDataAreaControlInfo.Count.htm).
