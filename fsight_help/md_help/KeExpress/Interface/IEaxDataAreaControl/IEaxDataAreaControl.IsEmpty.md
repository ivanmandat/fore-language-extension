# IEaxDataAreaControl.IsEmpty


## Синтаксис


IsEmpty: Boolean;


## Описание


Свойство IsEmpty возвращает
 признак отсутствия настроек управления параметрами среза аналитической
 области.


## Комментарии


Допустимые значения:


	- True. Настройки управления
	 отсутствуют. Значение по умолчанию;


	- False. Управление настроено.
