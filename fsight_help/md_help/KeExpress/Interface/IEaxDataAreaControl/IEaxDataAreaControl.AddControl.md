# IEaxDataAreaControl.AddControl


## Синтаксис


AddControl(ParamId: String): [IEaxDataAreaParamControl](../IEaxDataAreaParamControl/IEaxDataAreaParamControl.htm);


## Параметры


ParamId. Идентификатор параметра.


## Описание


Метод AddControl добавляет настройку
 по управлению для параметра среза с указанным идентификатором.


## Комментарии


Для добавления объекта настройки управления параметрами среза используйте
 метод [IEaxDataAreaControlInfo.AddControl](../IEaxDataAreaControlInfo/IEaxDataAreaControlInfo.AddControl.htm).
