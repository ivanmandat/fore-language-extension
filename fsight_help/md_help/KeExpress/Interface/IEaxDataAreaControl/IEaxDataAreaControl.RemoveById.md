# IEaxDataAreaControl.RemoveById


## Синтаксис


RemoveById(ParamId: Srting);


## Параметры


ParamId. Идентификатор параметра.


## Описание


Метод RemoveById осуществляет
 удаление параметра среза из коллекции по его идентификатору.


## Комментарии


Для определения количества параметров среза данных аналитической области
 используйте свойство [IEaxDataAreaControl.Count](IEaxDataAreaControl.Count.htm).
