# IEaxGrid.CreateNewTableStyleName


## Синтаксис


CreateNewTableStyleName(Sheet: [ITabSheet](TabSheet.chm::/Interface/ITabSheet/ITabSheet.htm);
 Value: String): String;


## Параметры


Sheet. Таблица, для которой
 созданы стили;


Value. Наименование стиля,
 на базе которого необходимо сгенерировать новое уникальное наименование
 стиля.


## Описание


Метод CreateNewTableStyleName
 генерирует новое уникальное имя стиля таблицы на основе указанных данных.
