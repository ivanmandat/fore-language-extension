# IEaxGrid.DiagnosticReport


## Синтаксис


DiagnosticReport: [IDiagnosticReport](../IDiagnosticReport/IDiagnosticReport.htm);


## Описание


Свойство DiagnosticReport определяет
 диагностический отчет для таблицы.


## Комментарии


Диагностический отчет предназначен для отображения результатов
 выполнения валидаций.
