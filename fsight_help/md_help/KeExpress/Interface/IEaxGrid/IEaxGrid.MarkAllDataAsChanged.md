# IEaxGrid.MarkAllDataAsChanged


## Синтаксис


MarkAllDataAsChanged;


## Описание


Метод MarkAllDataAsChanged переводит
 все данные таблицы в статус измененных.


## Комментарии


Для обновления указанной области таблицы используйте [IEaxGrid.RefreshPart](IEaxGrid.RefreshPart.htm).
