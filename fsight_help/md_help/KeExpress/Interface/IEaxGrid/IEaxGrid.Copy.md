# IEaxGrid.Copy


## Синтаксис


Copy(Sheet: [ITabSheet](TabSheet.chm::/Interface/ITabSheet/ITabSheet.htm);
 Range: [ITabRange](TabSheet.chm::/Interface/ITabRange/ITabRange.htm)):
 [IEaxGrid](IEaxGrid.htm);


## Параметры


Sheet. Лист отчета, на котором
 находится таблица;


Range.
 Диапазон ячеек таблицы.


## Описание


Метод Copy создает копию таблицы.


## Комментарии


Для создания копии среза аналитической области данных используйте [IEaxDataAreaSlice.Copy](../IEaxDataAreaSlice/IEaxDataAreaSlice.Copy.htm).
