# IEaxGrid.AllowEditWhenNotCalculated


## Синтаксис


AllowEditWhenNotCalculated: Boolean;


## Описание


Свойство AllowEditWhenNotCalculated
 определяет, присутствует ли возможность редактировать таблицу до её вычисления.


## Комментарии


Допустимые значения:


	- True. Редактирование
	 таблицы разрешено.


	- False. Редактирование
	 таблицы запрещено.


Значение свойства не сохраняется, используется для включения возможности
 редактирования таблицы в прикладных макросах до вычисления отчета.
