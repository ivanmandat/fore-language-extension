# IEaxGrid.OutPutRect


## Синтаксис


OutPutRect: [IGxRect](ModDrawing.chm::/Interface/IGxRect/IGxRect.htm);


## Описание


Свойство OutPutRect определяет
 параметры прямоугольной области таблицы.


## Комментарии


К параметрам прямоугольной области таблицы относятся, например:


	- высота;


	- ширина;


	- левый и правый края;


	- верхний и нижний края и другие.


Для получения диапазона прямоугольной области таблицы используйте [IEaxGrid.Range](IEaxGrid.Range.htm).
