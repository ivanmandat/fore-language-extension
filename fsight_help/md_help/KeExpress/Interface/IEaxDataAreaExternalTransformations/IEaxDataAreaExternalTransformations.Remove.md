# IEaxDataAreaExternalTransformations.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс задачи моделирования
 в коллекции.


## Описание


Метод Remove удаляет из коллекции
 настройки задачи моделирования с указанным индексом.
