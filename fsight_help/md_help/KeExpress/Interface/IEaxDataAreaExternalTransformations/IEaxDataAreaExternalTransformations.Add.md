# IEaxDataAreaExternalTransformations.Add


## Синтаксис


Add: [IEaxDataAreaExternalTransformation](../IEaxDataAreaExternalTransformation/IEaxDataAreaExternalTransformation.htm);


## Описание


Метод Add добавляет настройки
 для задачи моделирования.


## Комментарии


После добавления в настройках в свойстве [IEaxDataAreaExternalTransformation.Problem](../IEaxDataAreaExternalTransformation/IEaxDataAreaExternalTransformation.Problem.htm)
 необходимо определить задачу моделирования, которая будет использоваться
 для преобразования данных.
