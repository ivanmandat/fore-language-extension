# IEaxDataAreaExternalTransformations.Move


## Синтаксис


Move(FromIndex: Integer; ToIndex: Integer);


## Параметры


FromIndex. Индекс позиции перемещаемой
 задачи моделирования.


ToIndex. Индекс позиции, в
 которую необходимо переместить задачу моделирования.


## Описание


Метод Move перемещает настройки
 задачи моделирования в коллекции.
