# IEaxDataAreaExternalTransformations.RemoveByKey


## Синтаксис


RemoveByKey(Value: Integer): Boolean;


## Параметры


Value. Ключ задачи моделирования
 в репозитории.


## Описание


Метод RemoveByKey удаляет из
 коллекции настройки задачи моделирования с указанным ключом.
