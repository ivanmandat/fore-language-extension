# IEaxDataAreaExternalTransformations.RemoveObject


## Синтаксис


RemoveObject(Value: [IEaxDataAreaExternalTransformation](../IEaxDataAreaExternalTransformation/IEaxDataAreaExternalTransformation.htm));


## Параметры


Value. Настройки задачи моделирования,
 которые необходимо удалить.


## Описание


Метод RemoveObject удаляет из
 коллекции указанные настройки задачи моделирования.
