# IEaxDataAreaParamControl.ControlledParam


## Синтаксис

## Описание


Свойство ControlledParam возвращает
 параметр среза аналитической области, для которого настраивается привязка.


## Комментарии


Для определения отсутствия настроек управления параметрами используйте
 свойство [IEaxDataAreaControl.IsEmpty](../IEaxDataAreaControl/IEaxDataAreaControl.IsEmpty.htm).
