# IEaxDataAreaParamControl


Сборка: Express;


## Описание


Интерфейс IEaxDataAreaParamControl
 используется для настройки привязки параметров различных срезов аналитической
 области между собой и параметров среза к параметру отчёта.


## Иерархия наследования


           IEaxDataAreaParamControl


## Комментарии


Для работы с параметрами среза аналитической области используйте интерфейс
 [IEaxDataAreaControl](../IEaxDataAreaControl/IEaxDataAreaControl.htm).
