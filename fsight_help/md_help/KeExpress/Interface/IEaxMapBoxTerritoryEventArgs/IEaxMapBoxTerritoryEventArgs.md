# IEaxMapBoxTerritoryEventArgs


Сборка: Express;


## Описание


Интерфейс IEaxMapBoxTerritoryEventArgs
 содержит свойства аргумента события [OnTerritoryClick](../../Class/EaxMapBox/EaxMapBox.OnTerritoryClick.htm)
 компонента [EaxMapBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/EaxMapBox.htm), возникающего
 при щелчке по какому-либо региону карты.
