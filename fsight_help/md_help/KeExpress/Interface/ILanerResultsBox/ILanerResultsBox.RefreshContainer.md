# ILanerResultsBox.RefreshContainer


## Синтаксис


RefreshContainer(Value: [LanerBoxContainerType](../../Enums/LanerBoxContainerType.htm));


## Параметры


Value. Панель.


## Описание


Метод RefreshContainer обновляет
 указанную панель компонента.


## Комментарии


Для обновления всех панелей компонента используйте метод [ILanerResultsBox.Refresh](ILanerResultsBox.Refresh.htm).
