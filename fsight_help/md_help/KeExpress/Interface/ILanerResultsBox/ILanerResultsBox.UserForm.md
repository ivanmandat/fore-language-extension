# ILanerResultsBox.UserForm


## Синтаксис


UserForm: String;


## Описание


Свойство UserForm определяет
 идентификатор пользовательской формы, подключаемой вместо одной из стандартных
 панелей компонента. Наименование панели с формой берется из названия пользовательской
 формы.
