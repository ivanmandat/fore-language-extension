# ILanerResultsBox.Refresh


## Синтаксис


Refresh;


## Описание


Метод Refresh обновляет панели
 компонента LanerResultsBox.


## Комментарии


Метод используется для принудительного обновления панелей компонента.
 Например, оно может потребоваться, если метод [ILanerResultsBox.SetUserForm](ILanerResultsBox.SetUserForm.htm)
 был вызван с параметром UpdatePanels,
 установленным в False.
