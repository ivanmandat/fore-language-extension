# IHeader


Сборка: Express;


## Описание


Интерфейс IHeader содержит свойства
 колонтитула для листа на печати.


## Иерархия наследования


           IHeaderBase


           IHeader


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Margin](IHeader.Margin.htm)
		 Свойство Margin определяет
		 отступ колонтитула.

		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Center](../IHeaderBase/IHeaderBase.Center.htm)
		 Свойство Center возвращает
