# IHeader.Margin


## Синтаксис


Margin: Double;


## Описание


Свойство Margin определяет отступ
 колонтитула.


## Комментарии


Значение указывается в миллиметрах. Для верхнего/нижнего колонтитула
 значение задаётся относительно верхнего/нижнего края страницы соответственно.
