# IEaxDataAreaLanerSlice


Сборка: Express;


## Описание


Интерфейс IEaxDataAreaLanerSlice
 используется для работы со срезом аналитической области данных, построенного
 на рабочей книге базы данных временных рядов в рядном режиме.


## Иерархия наследования


           IEaxDataAreaLanerSlice


## Комментарии


Для получения объекта данного типа необходимо [IEaxDataAreaSlice](../IEaxDataAreaSlice/IEaxDataAreaSlice.htm)
 привести к IEaxDataAreaLanerSlice.


Рядный режим для рабочей книги базы данных временных рядов устанавливается
 в свойстве [IEaxAnalyzeCore.Mode](../IEaxAnalyzeCore/IEaxAnalyzeCore.Mode.htm)
 переключением значения [EaxMode.MultiDimension](../../Enums/EaxMode.htm)
 в [EaxMode.Series](../../Enums/EaxMode.htm).
