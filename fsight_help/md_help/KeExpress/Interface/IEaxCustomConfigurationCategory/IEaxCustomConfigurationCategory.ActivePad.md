# IEaxCustomConfigurationCategory.ActivePad


## Синтаксис


ActivePad: [IEaxConfigurationPad](../IEaxConfigurationPad/IEaxConfigurationPad.htm);


## Описание


Свойство ActivePad определяет активную вкладку.


## Комментарии


Для активной вкладки отображается содержимое связанной с ней панели. При активации какой-либо вкладки содержимое остальных вкладок будет скрыто. В качестве значения для свойства ActivePad можно указать одну из доступных вкладок коллекции [Pads](IEaxCustomConfigurationCategory.Pads.htm).
