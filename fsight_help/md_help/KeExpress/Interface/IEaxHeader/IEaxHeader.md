# IEaxHeader


Сборка: Express;


## Описание


Интерфейс IEaxHeader содержит
 свойства и методы колонтитула экспресс-отчета.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Center](IEaxHeader.Center.htm)
		 Свойство Center возвращает
		 центральную часть колонтитула экспресс-отчета.


		 ![](../../Property_Image.gif)
		 [Left](IEaxHeader.Left.htm)
		 Свойство Left возвращает
		 левую часть колонтитула экспресс-отчета.


		 ![](../../Property_Image.gif)
		 [Margin](IEaxHeader.Margin.htm)
		 Свойство Margin определяет
		 отступ колонтитула экспресс-отчета.


		 ![](../../Property_Image.gif)
		 [Right](IEaxHeader.Right.htm)
		 Свойство Right возвращает
		 правую часть колонтитула экспресс-отчета.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Clear](IEaxHeader.Clear.htm)
		 Метод Clear осуществляет
		 очитку содержимого колонтитула.


		 ![](../../Sub_Image.gif)
		 [IsEmpty](IEaxHeader.IsEmpty.htm)
		 Метод IsEmpty возвращает
		 True если колонтитул пуст.
