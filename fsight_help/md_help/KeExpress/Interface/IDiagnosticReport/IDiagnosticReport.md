# IDiagnosticReport


Сборка: Express;


## Описание


Интерфейс IDiagnosticReport
 содержит свойства и методы для работы с диагностическим отчетом. Диагностический
 отчет предназначен для отображения данных выполнения валидации.


## Иерархия наследования


           IDiagnosticReport


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Chain](IDiagnosticReport.Chain.htm)


		 Свойство Chain возвращает
		 цепочку валидаций.


		 ![](../../Property_Image.gif)
		 [EaxAnalyzer](IDiagnosticReport.EaxAnalyzer.htm)


		 Свойство EaxAnalyzer
		 определяет экспресс-отчет, отображающий данные.


		 ![](../../Property_Image.gif)
		 [Exceptions](IDiagnosticReport.Exceptions.htm)


		 Свойство Exceptions
		 возвращает правила, выполняющиеся для заданной точки.


		 ![](../../Property_Image.gif)
		 [Explanation](IDiagnosticReport.Explanation.htm)


		 Свойство Explanation
		 определяет объяснение для указанной точки.


		 ![](../../Property_Image.gif)
		 [Grid](IDiagnosticReport.Grid.htm)


		 Свойство Grid определяет
		 таблицу, отображающую данные диагностического отчета.


		 ![](../../Property_Image.gif)
		 [MultipleExceptionImage](IDiagnosticReport.MultipleExceptionImage.htm)


		 Свойство MultipleExceptionImage
		 определяет изображение, которое отображается, если для точки выполняется
		 более одного правила.


		 ![](../../Property_Image.gif)
		 [NextException](IDiagnosticReport.NextException.htm)


		 Свойство NextException
		 возвращает следующую ячейку отчета, в которой выполняется правило.


		 ![](../../Property_Image.gif)
		 [PreviousException](IDiagnosticReport.PreviousException.htm)


		 Свойство PreviousException
		 возвращает предыдущую ячейку отчета, в которой выполняется правило.


		 ![](../../Property_Image.gif)
		 [Rules](IDiagnosticReport.Rules.htm)


		 Свойство Rules возвращает
		 правила, в результате выполнения которых получены данные отчета.


		 ![](../../Property_Image.gif)
		 [Run](IDiagnosticReport.Run.htm)


		 Свойство Run определяет
		 данные выполнения валидации, отображаемые в отчете.


		 ![](../../Property_Image.gif)
		 [ShowExplainedExceptions](IDiagnosticReport.ShowExplainedExceptions.htm)


		 Свойство ShowExplainedExceptions
		 определяет, отображать ли объяснение для точек в виде примечания.
