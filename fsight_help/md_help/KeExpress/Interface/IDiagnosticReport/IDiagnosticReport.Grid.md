# IDiagnosticReport.Grid


## Синтаксис


Grid: [IEaxGrid](../IEaxGrid/IEaxGrid.htm);


## Описание


Свойство Grid определяет таблицу,
 отображающую данные диагностического отчета.


## Комментарии


Диагностический отчет предназначен для отображения результатов
 выполнения валидаций.
