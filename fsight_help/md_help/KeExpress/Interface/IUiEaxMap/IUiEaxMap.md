# IUiEaxMap


Сборка: Express;


## Описание


Интерфейс IUiEaxMap содержит
 свойства компонента среды разработки [UiEaxMap](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiEaxMap.htm),
 являющегося источником данных для компонента [EaxMapBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/EaxMapBox.htm).
