# IEaxAnalyzerEventArgs.AnalyzerData


## Синтаксис


AnalyzerData: [IReportData](../IReportData/IReportData.htm);


## Описание


Свойство AnalyzerData возвращает
 данные отчета, использующего обработчик событий.


## Комментарии


Данные отчета состоят из:


	- описания отчета;


	- данных листа, для которого сгенерировано событие.
