# IEaxDataAreaHierarchies.FindById


## Синтаксис


FindById(Id: String): [IEaxDataAreaHierarchy](../IEaxDataAreaHierarchy/IEaxDataAreaHierarchy.htm);


## Параметры


Id. Идентификатор альтернативной
 иерархии.


## Описание


Метод FindById осуществляет
 поиск альтернативной иерархии по ее идентификатору.


## Комментарии


Если альтернативная иерархия не будет найдена, то данный метод вернет
 значение Null.


Для поиска альтернативной иерархии по ее ключу используйте [IEaxDataAreaHierarchies.FindByKey](IEaxDataAreaHierarchies.FindByKey.htm).
