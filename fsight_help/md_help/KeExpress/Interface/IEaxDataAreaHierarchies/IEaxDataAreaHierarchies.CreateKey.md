# IEaxDataAreaHierarchies.CreateKey


## Синтаксис


CreateKey(SliceKey: Integer; HKey: Integer): Integer;


## Параметры


SliceKey. Ключ среза аналитической
 области данных;


HKey. Ключ альтернативной иерархии.


## Описание


Метод CreateKey генерирует уникальный
 ключ аналитической области данных.


## Комментарии


Используйте полученный ключ для вызова метода [IEaxDataAreaHierarchies.GetDependants](IEaxDataAreaHierarchies.GetDependants.htm).
