# IEaxDataAreaHierarchies.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс альтернативной
 иерархии.


## Описание


Метод Remove удаляет альтернативную
 иерархию по ее индексу.


## Комментарии


Для удаления альтернативной иерархии по ее ключу используйте [IEaxDataAreaHierarchies.RemoveByKey](IEaxDataAreaHierarchies.RemoveByKey.htm).
