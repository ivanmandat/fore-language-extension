# IEaxDataAreaHierarchies.IsAutoSort


## Синтаксис


IsAutoSort: Boolean;


## Описание


Свойство IsAutoSort определяет,
 будет ли использоваться автоматическая сортировка иерархий при вычислении.


## Комментарии


Свойство актуально для иерархий, у которых задана сортировка через [формулу](../IEaxDataAreaHierarchy/IEaxDataAreaHierarchy.SortTransformations.htm).


Направление сортировки задаётся в свойстве [IEaxDataAreaHierarchy.SortDirection](../IEaxDataAreaHierarchy/IEaxDataAreaHierarchy.SortDirection.htm).


Допустимые значения:


	- True. По умолчанию.
	 Автоматическая сортировка будет использоваться;


	- False. Автоматическая
	 сортировка использоваться не будет.
