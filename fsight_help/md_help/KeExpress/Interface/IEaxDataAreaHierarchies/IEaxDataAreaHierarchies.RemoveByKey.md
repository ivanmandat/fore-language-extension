# IEaxDataAreaHierarchies.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ альтернативной иерархии.


## Описание


Метод RemoveByKey удаляет альтернативную
 иерархию по ее ключу.


## Комментарии


Для удаления альтернативной иерархии по ее индексу используйте [IEaxDataAreaHierarchies.Remove](IEaxDataAreaHierarchies.Remove.htm).
