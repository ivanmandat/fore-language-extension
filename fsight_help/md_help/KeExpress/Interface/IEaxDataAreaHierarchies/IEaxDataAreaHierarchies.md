# IEaxDataAreaHierarchies


Сборка: Express;


## Описание


Интерфейс IEaxDataAreaHierarchies
 используется для работы с коллекцией альтернативных иерархий аналитической
 области данных.


## Иерархия наследования


           IEaxDataAreaHierarchies


## Комментарии


Для получения объекта данного типа используйте [IEaxDataArea.Hierarchies](../IEaxDataArea/IEaxDataArea.Hierarchies.htm).
