# IEaxDataAreaHierarchies.Item


## Синтаксис


Item(Index: Integer): [IEaxDataAreaHierarchy](../IEaxDataAreaHierarchy/IEaxDataAreaHierarchy.htm);


## Параметры


Index. Индекс альтернативной
 иерархии.


## Описание


Свойство Item возвращает альтернативную
 иерархию аналитической области данных.


## Комментарии


Для определения количества альтернативных иерархий аналитической области
 данных используйте [IEaxDataAreaHierarchies.Count](IEaxDataAreaHierarchies.Count.htm).
