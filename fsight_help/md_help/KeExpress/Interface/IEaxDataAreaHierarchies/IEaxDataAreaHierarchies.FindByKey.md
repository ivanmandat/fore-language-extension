# IEaxDataAreaHierarchies.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IEaxDataAreaHierarchy](../IEaxDataAreaHierarchy/IEaxDataAreaHierarchy.htm);


## Параметры


Key. Ключ альтернативной иерархии.


## Описание


Метод FindByKey осуществляет
 поиск альтернативной иерархии по ее ключу.


## Комментарии


Если альтернативная иерархия не будет найдена, то данный метод вернет
 значение Null.


Для поиска альтернативной иерархии по ее идентификатору используйте
 [IEaxDataAreaHierarchies.FindById](IEaxDataAreaHierarchies.FindById.htm).
