# IEaxDataAreaHierarchies.ClearAll


## Синтаксис


ClearAll;


## Описание


Метод ClearAll очищает коллекцию
 альтернативных иерархий аналитической области данных.


## Комментарии


Для удаления коллекции альтернативных иерархий из измерения используйте
 [IEaxDataAreaHierarchies.Clear](IEaxDataAreaHierarchies.Clear.htm).


Для добавления альтернативной иерархии используйте [IEaxDataAreaHierarchies.Add](IEaxDataAreaHierarchies.Add.htm).
