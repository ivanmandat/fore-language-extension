# IEaxDataAreaControlInfo.FindByKey


## Синтаксис


FindByKey(DataAreaSliceKey: Integer): [IEaxDataAreaControl](../IEaxDataAreaControl/IEaxDataAreaControl.htm);


## Параметры


DataAreaSliceKey. Ключ объекта
 настройки среза аналитической области.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает объект настройки управления параметрами среза по его
 ключу.


## Комментарии


Для определения количества параметров среза аналитической области используйте
 свойство [IEaxDataAreaControl.Count](../IEaxDataAreaControl/IEaxDataAreaControl.Count.htm).
