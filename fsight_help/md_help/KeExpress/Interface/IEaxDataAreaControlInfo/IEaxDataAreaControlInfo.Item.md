# IEaxDataAreaControlInfo.Item


## Синтаксис


Item(Index: Integer): [IEaxDataAreaControl](../IEaxDataAreaControl/IEaxDataAreaControl.htm);


## Параметры


Index. Индекс параметра среза
 аналитической области.


## Описание


Свойство Item возвращает объект
 настройки управления параметрами среза.


## Комментарии


Для определения количества параметров среза аналитической области используйте
 свойство [IEaxDataAreaControl.Count](../IEaxDataAreaControl/IEaxDataAreaControl.Count.htm).
