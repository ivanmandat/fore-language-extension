# IEaxDataAreaControlInfo.AddControl


## Синтаксис


AddControl(Object: [IEaxDataAreaSlice](../IEaxDataAreaSlice/IEaxDataAreaSlice.htm)):
 [IEaxDataAreaControl](../IEaxDataAreaControl/IEaxDataAreaControl.htm);


## Параметры


Object. Срез аналитической
 области данных.


## Описание


Метод AddControl добавляет объект
 настройки управления параметрами среза в коллекцию.


## Комментарии


Для добавления настройки управления параметрами среза используйте метод
 [IEaxDataAreaControl.AddControl](../IEaxDataAreaControl/IEaxDataAreaControl.AddControl.htm).
