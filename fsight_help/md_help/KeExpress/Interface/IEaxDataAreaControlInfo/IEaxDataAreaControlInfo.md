# IEaxDataAreaControlInfo


Сборка: Express;


## Описание


Интерфейс IEaxDataAreaControlInfo
 используется для работы с коллекцией объектов настройки управления параметрами
 среза аналитической области.


## Иерархия наследования


           IEaxDataAreaControlInfo


## Комментарии


Для работы с параметрами среза аналитической области используйте интерфейс
 [IEaxDataAreaControl](../IEaxDataAreaControl/IEaxDataAreaControl.htm).
