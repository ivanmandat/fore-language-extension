# IEaxObject.Available


## Синтаксис


Available: Boolean;


## Описание


Свойство Available возвращает
 признак доступности объекта для использования в экспресс-отчете.


## Комментарии


Свойство возвращает значение True,
 если объект доступен для использования, в противном случае - False.
 Для доступного объекта можно менять все остальные базовые и специфические
 свойства, выполнять методы.


В текущей реализации данное свойство актуально для [карты](../IEaxMapChart/IEaxMapChart.htm).
 Карта будет доступна для использования, если в источнике данных есть территориальное
 измерение, структура которого подходит для работы с картой. Для этого
 в структуре измерения должен присутствовать атрибут с идентификатором
 TERRID.
