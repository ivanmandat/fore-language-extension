# IEaxObject.Load


## Синтаксис


Load;


## Описание


Метод Load загружает объект
 аналитической области.


## Комментарии


Метод используется, если:


	- у отчёта для свойства [IEaxDataArea.DelayedLoad](../IEaxDataArea/IEaxDataArea.DelayedLoad.htm)
	 установлено значение True;


	- у объекта для свойства [IEaxObject.IsLoaded](IEaxObject.IsLoaded.htm)
	 установлено значение True.
