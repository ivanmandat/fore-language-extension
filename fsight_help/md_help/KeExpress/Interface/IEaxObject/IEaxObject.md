# IEaxObject


Сборка: Express;


## Описание


Интерфейс IEaxObject содержит
 базовые свойства и методы объектов экспресс-отчета.


## Иерархия наследования


           IEaxObject


## Комментарии


Объектами экспресс-отчета являются:


	- [таблица](../IEaxGrid/IEaxGrid.htm);


	- [карта](../IEaxMapChart/IEaxMapChart.htm);


	- [диаграмма](../IEaxChart/IEaxChart.htm);


	- [пузырьковая
	 диаграмма](../IEaxBubbleChart/IEaxBubbleChart.htm);


	- [пузырьковое дерево](../IEaxBubbleTree/IEaxBubbleTree.htm);


	- [плоское дерево](../IEaxTreeMap/IEaxTreeMap.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Active](IEaxObject.Active.htm)
		 Свойство Active определяет
		 активный объект экспресс-отчета.


		 ![](../../Property_Image.gif)
		 [AllowSaveData](IEaxObject.AllowSaveData.htm)
		 Свойство AllowSaveData
		 определяет, будут ли данные объекта после изменения [сохраняться
		 в источник](UiAnalyticalArea.chm::/Parameters/Parameters.htm).


		 ![](../../Property_Image.gif)
		 [Available](IEaxObject.Available.htm)
		 Свойство Available
		 возвращает признак доступности объекта для использования в экспресс-отчете.


		 ![](../../Property_Image.gif)
		 [CombinedIndicatorSelection](IEaxObject.CombinedIndicatorSelection.htm)
		 Свойство CombinedIndicatorSelection
		 возвращает отметку всех метрик измерения в представлении.


		 ![](../../Property_Image.gif)
		 [Description](IEaxObject.Description.htm)
		 Свойство Description
		 возвращает описание объекта экспресс-отчета.


		 ![](../../Property_Image.gif)
		 [Enabled](IEaxObject.Enabled.htm)
		 Свойство Enabled определяет,
		 производится ли построение объекта.


		 ![](../../Property_Image.gif)
		 [Id](IEaxObject.Id.htm)
		 Свойство Id определяет
		 идентификатор объекта.


		 ![](../../Property_Image.gif)
		 [IndicatorLegendAssignment](IEaxObject.IndicatorLegendAssignment.htm)
		 Свойство IndicatorLegendAssignment
		 определяет назначение метрики в легенде визуализатора.


		 ![](../../Property_Image.gif)
		 [IndicatorSelection](IEaxObject.IndicatorSelection.htm)
		 Свойство IndicatorSelection
		 возвращает отметку метрики измерения в представлении.


		 ![](../../Property_Image.gif)
		 [IsLoaded](IEaxObject.IsLoaded.htm)
		 Свойство IsLoaded определят,
		 будет ли загружаться объект аналитической области.


		 ![](../../Property_Image.gif)
		 [Laner](IEaxObject.Laner.htm)
		 Свойство Laner определяет
		 объект для работы с рабочей книгой базы данных временных рядов
		 в рядном режиме.


		 ![](../../Property_Image.gif)
		 [Pivot](IEaxObject.Pivot.htm)
		 Свойство Pivot определяет
		 объект, представляющий основу для построения таблицы с данными.


		 ![](../../Property_Image.gif)
		 [PrintSettings](IEaxObject.PrintSettings.htm)
		 Свойство PrintSettings
		 возвращает параметры печати и экспорта объекта экспресс-отчета.


		 ![](../../Property_Image.gif)
		 [SelectionManager](IEaxObject.SelectionManager.htm)
		 Свойство SelectionManager
		 возвращает объект для работы с отметкой измерений.


		 ![](../../Property_Image.gif)
		 [SyncIndicators](IEaxObject.SyncIndicators.htm)
		 Свойство SyncIndicators определяет,
		 включена ли [синхронизация
		 метрик](UiSelection.chm::/Selection/Dimension_type.htm#sync) для визуализаторов.


		 ![](../../Property_Image.gif)
		 [Tag](IEaxObject.Tag.htm)
		 Свойство Tag определяет
		 любое значение, связанное с объектом.


		 ![](../../Property_Image.gif)
		 [Type](IEaxObject.Type.htm)
		 Свойство Type определяет
		 тип объекта экспресс-отчета.


		 ![](../../Property_Image.gif)
		 [UndoRedo](IEaxObject.UndoRedo.htm)
		 Свойство UndoRedo
		 возвращает параметры стека изменений объекта экспресс-отчета.


		 ![](../../Property_Image.gif)
		 [ViewScale](IEaxObject.ViewScale.htm)
		 Свойство ViewScale
		 определяет масштаб отображения окна объекта в экспресс-отчете.


		 ![](../../Property_Image.gif)
		 [ViewSize](IEaxObject.ViewSize.htm)
		 Свойство ViewSize возвращает
		 размер объекта экспресс-отчета.


		 ![](../../Property_Image.gif)
		 [Visible](IEaxObject.Visible.htm)
		 Свойство Visible определяет,
		 будет ли объект отображаться в окне экспресс-отчета и печататься
		 на принтере.
