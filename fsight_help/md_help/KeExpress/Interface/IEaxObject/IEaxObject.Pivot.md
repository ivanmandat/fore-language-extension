# IEaxObject.Pivot


## Синтаксис


Pivot: [IPivot](KePivot.chm::/Interface/IPivot/IPivot.htm);


## Описание


Свойство Pivot определяет объект,
 представляющий основу для построения таблицы с данными.


## Комментарии


Для определения объекта для работы с рабочей книгой базы данных временных
 рядов в рядном режиме используйте [IEaxObject.Laner](IEaxObject.Laner.htm).
