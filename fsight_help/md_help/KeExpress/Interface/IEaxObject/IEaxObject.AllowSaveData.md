# IEaxObject.AllowSaveData


## Синтаксис


AllowSaveData: Boolean;


## Описание


Свойство AllowSaveData определяет,
 будут ли данные объекта после изменения [сохраняться
 в источник](UiAnalyticalArea.chm::/Parameters/Parameters.htm).


## Комментарии


Возможные варианты:


	- True. По умолчанию.
	 Данные после изменения будут сохраняться в источник;


	- False. Данные после
	 изменения не будут сохраняться в источник.
