# IEaxObject.Refresh


## Синтаксис


Refresh;


## Описание


Метод Refresh осуществляет обновление
 объекта экспресс-отчета.


## Комментарии


При выполнении метода осуществляется полное обновление визуального представления
 объекта в соответствии с его настройками и настройками источников данных
 экспресс-отчета.
