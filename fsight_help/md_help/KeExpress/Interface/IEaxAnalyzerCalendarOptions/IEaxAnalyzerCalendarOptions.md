# IEaxAnalyzerCalendarOptions


Сборка: Express;


## Описание


Интерфейс IEaxAnalyzerCalendarOptions
 предназначен для настройки параметров отображения календарей отчёта.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ShowLevels](IEaxAnalyzerCalendarOptions.ShowLevels.htm)
		 Свойство ShowLevels
		 определяет необходимость отображения уровней календаря.


		 ![](../../Property_Image.gif)
		 [ShowPeriod](IEaxAnalyzerCalendarOptions.ShowPeriod.htm)
		 Свойство ShowPeriod
		 определяет необходимость отображения периода календаря.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Reset](IEaxAnalyzerCalendarOptions.Reset.htm)
		 Метод Reset сбрасывает
		 все настройки отображения всех календарных измерений отчёта.
