# IEaxDrillRangeResult


Сборка: Express;


## Описание


Интерфейс IEaxDrillRangeResult
 содержит свойства для получения информации о привязке диапазона ячеек
 к данным источника.


## Иерархия наследования


           [IEaxDrillResult](../IEaxDrillResult/IEaxDrillResult.htm)


           IEaxDrillRangeResult


## Комментарии


Информацию о привязке диапазона ячеек к данным можно получить, используя
 метод [IEaxGrid.DrillRangeResult](../IEaxGrid/IEaxGrid.DrillRangeResult.htm).
