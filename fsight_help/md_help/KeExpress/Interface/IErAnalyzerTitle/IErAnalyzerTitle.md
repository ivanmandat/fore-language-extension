## Описание


Интерфейс IErAnalyzerTitle содержит
 свойства и методы компонента среды разработки [ErAnalyzerTitle](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/ErAnalyzerTitle.htm).

## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Copy](IErAnalyzerTitle.Copy.htm)
		 Метод Copy помещает
		 выделенный фрагмент заголовка экспресс-отчета в буфер обмена,
		 оставляя заголовок без изменений.


		 ![](../../Sub_Image.gif)
		 [Cut](IErAnalyzerTitle.Cut.htm)
		 Метод Cut помещает
		 выделенный фрагмент заголовка экспресс-отчета в буфер обмена,
		 удаляя его из заголовка.


		 ![](../../Sub_Image.gif)
		 [EndEdit](IErAnalyzerTitle.EndEdit.htm)
		 Метод EndEdit выводит
		 компонент из режима редактирования с сохранением всех изменений.


		 ![](../../Sub_Image.gif)
		 [InsertDate](IErAnalyzerTitle.InsertDate.htm)
		 Метод InsertDate вставляет
		 текущую дату в заголовок экспресс-отчета.


		 ![](../../Sub_Image.gif)
		 [InsertName](IErAnalyzerTitle.InsertName.htm)
		 Метод InsertName вставляет
		 наименование текущего экспресс-отчета в заголовок.


		 ![](../../Sub_Image.gif)
		 [InsertSelection](IErAnalyzerTitle.InsertSelection.htm)
		 Метод InsertSelection
		 вставляет наименование текущего измерения в заголовок экспресс-отчета.


		 ![](../../Sub_Image.gif)
		 [InsertText](IErAnalyzerTitle.InsertText.htm)
		 Метод InsertText вставляет
		 текст, заданный пользователем, в заголовок экспресс-отчета.


		 ![](../../Sub_Image.gif)
		 [InsertTime](IErAnalyzerTitle.InsertTime.htm)
		 Метод InsertTime вставляет
		 текущее время в заголовок экспресс-отчета.


		 ![](../../Sub_Image.gif)
		 [Paste](IErAnalyzerTitle.Paste.htm)
		 Метод Paste вставляет
		 фрагмент, находящийся в буфере обмена, в заголовок экспресс-отчета.


		 ![](../../Sub_Image.gif)
		 [SelectAll](IErAnalyzerTitle.SelectAll.htm)
		 Метод SelectAll полностью
		 выделяет заголовок экспресс-отчета.


		 ![](../../Sub_Image.gif)
		 [SetSel](IErAnalyzerTitle.SetSel.htm)
		 Метод SetSel выделяет
		 необходимый фрагмент в заголовке экспресс-отчета.


		 ![](../../Sub_Image.gif)
		 [ShowFontDialog](IErAnalyzerTitle.ShowFontDialog.htm)
		 Метод ShowFontDialog
		 вызывает диалог настройки шрифта для заголовка.


		 ![](../../Sub_Image.gif)
		 [StartEdit](IErAnalyzerTitle.StartEdit.htm)
		 Метод StartEdit переводит
		 заголовок в режим редактирования.
