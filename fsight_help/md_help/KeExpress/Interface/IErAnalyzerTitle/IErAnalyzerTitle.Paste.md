# IErAnalyzerTitle.Paste


## Синтаксис


Paste;


## Описание


Метод Paste вставляет фрагмент,
 находящийся в буфере обмена, в заголовок экспресс-отчета.


## Комментарии


Для помещения выделенного фрагмента в буфер обмена, используйте методы
 [IErAnalyzerTitle.Copy](IErAnalyzerTitle.Copy.htm),
 [IErAnalyzerTitle.Cut](IErAnalyzerTitle.Cut.htm).
