# IErAnalyzerTitle.Copy


## Синтаксис


Copy;


## Описание


Метод Copy помещает выделенный
 фрагмент заголовка экспресс-отчета в буфер обмена, оставляя заголовок
 без изменений.


## Комментарии


Для вставки фрагмента, находящегося в буфере обмена, в заголовок экспресс-отчета
 используйте метод [IErAnalyzerTitle.Paste](IErAnalyzerTitle.Paste.htm).
