# IErAnalyzerTitle.Xaml


## Синтаксис


Xaml: String;


## Описание


Свойство Xaml определяет текст
 в заголовке экспресс-отчета, предназначенного для представления в формате
 XAML.


## Комментарии


Формат XAML более не поддерживается для хранения значений
 форматированного текста. При указании значений в формате XAML они
 автоматически будут преобразованы в формат RTF.
