# IErAnalyzerTitle.FontFamily


## Синтаксис


FontFamily: String;


## Описание


Свойство FontFamily определяет
 семейство шрифтов заголовка экспресс-отчета.


## Комментарии


Для определения размера шрифта заголовка экспресс-отчета используйте
 свойство [IErAnalyzerTitle.FontSize](IErAnalyzerTitle.FontSize.htm).
