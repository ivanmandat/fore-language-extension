# IErAnalyzerTitle.EndSel


## Синтаксис


EndSel: Integer;


## Описание


Свойство EndSel возвращает текущую
 конечную позицию выделенного фрагмента в заголовке экспресс-отчета.


## Комментарии


Для получения текущей начальной позиции выделенного фрагмента в заголовке
 экспресс-отчета используйте свойство [IErAnalyzerTitle.BeginSel](IErAnalyzerTitle.BeginSel.htm).
