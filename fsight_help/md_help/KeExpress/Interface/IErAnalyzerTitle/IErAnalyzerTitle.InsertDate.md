# IErAnalyzerTitle.InsertDate


## Синтаксис


InsertDate;


## Описание


Метод InsertDate вставляет текущую
 дату в заголовок экспресс-отчета.


## Комментарии


Для вставки текущего системного времени в заголовок экспресс-отчета
 используйте метод [IErAnalyzerTitle.InsertTime](IErAnalyzerTitle.InsertTime.htm).
