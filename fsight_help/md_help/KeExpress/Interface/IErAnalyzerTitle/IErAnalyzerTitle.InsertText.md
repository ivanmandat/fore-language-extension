# IErAnalyzerTitle.InsertText


## Синтаксис


InsertText(Value: String);


## Параметры


Value. Текст, заданный пользователем.


## Описание


Метод InsertText вставляет текст,
 заданный пользователем, в заголовок экспресс-отчета.


## Комментарии


В тексте заголовка может
 быть указана любая текстовая информация. Также в тексте допустимо использование
 подстановок:


	- &[Date] - текущая дата;


	- &[Time] - текущее
	 время;


	- &[Name] - название
	 экспресс-отчета;


	- &[Globals.<Идентификатор
	 переменной>] - значение [глобальной
	 переменной](UiNav.chm::/02_Navigator/UiNav_GlobalVariables.htm). Идентификатор глобальной переменной
	 чувствителен к регистру;


	- &[Dimensions.<Идентификатор
	 измерения>] - отметка по указанному измерению.
