# IErAnalyzerTitle.Bold


## Синтаксис


Bold: Boolean;


## Описание


Свойство Bold определяет, будет
 ли использоваться жирный шрифт заголовка экспресс-отчета.


## Комментарии


Допустимые значения:


	- True. Жирный шрифт заголовка
	 будет использоваться;


	- False. По умолчанию.
	 Жирный шрифт заголовка использоваться не будет.
