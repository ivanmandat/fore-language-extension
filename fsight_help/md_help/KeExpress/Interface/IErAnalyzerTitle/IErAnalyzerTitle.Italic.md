# IErAnalyzerTitle.Italic


## Синтаксис


Italic: Boolean;


## Описание


Свойство Italic определяет,
 будет ли использоваться курсивный шрифт заголовка экспресс-отчета.


## Комментарии


Допустимые значения:


	- True. Курсивный шрифт
	 заголовка будет использоваться;


	- False. По умолчанию.
	 Курсивный шрифт заголовка использоваться не будет.
