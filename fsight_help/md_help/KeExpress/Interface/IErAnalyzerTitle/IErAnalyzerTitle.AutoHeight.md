# IErAnalyzerTitle.AutoHeight


## Синтаксис


AutoHeight: Boolean;


## Описание


Свойство AutoHeight определяет,
 будет ли производиться автоматический подгон размеров компонента.


## Комментарии


Допустимые значения:


	- True. Осуществляется
	 автоматический подгон размеров компонента при изменении параметров
	 шрифта текста;


	- False. Автоматический
	 подгон размеров не производится.
