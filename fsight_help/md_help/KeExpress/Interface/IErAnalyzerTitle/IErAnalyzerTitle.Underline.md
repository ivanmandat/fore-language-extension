# IErAnalyzerTitle.Underline


## Синтаксис


Underline: Boolean;


## Описание


Свойство Underline определяет,
 будет ли использоваться подчеркнутый шрифт заголовка экспресс-отчета.


## Комментарии


Допустимые значения:


	- True. Подчеркнутый шрифт
	 заголовка будет использоваться;


	- False. По умолчанию.
	 Подчеркнутый шрифт заголовка использоваться не будет.
