# IErAnalyzerTitle.InsertTime


## Синтаксис


InsertTime;


## Описание


Метод InsertTime вставляет текущее
 время в заголовок экспресс-отчета.


## Комментарии


Для вставки текущей системной даты в заголовок экспресс-отчета используйте
 метод [IErAnalyzerTitle.InsertDate](IErAnalyzerTitle.InsertDate.htm).
