# IErAnalyzerTitle.InsertSelection


## Синтаксис


InsertSelection;


## Описание


Метод InsertSelection вставляет
 наименование текущего измерения в заголовок экспресс-отчета.


## Комментарии


Для вставки наименования экспресс-отчета в заголовок, используйте метод
 [IErAnalyzerTitle.InsertName](IErAnalyzerTitle.InsertName.htm).
