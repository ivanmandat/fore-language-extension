# IErAnalyzerTitle.Cut


## Синтаксис


Cut;


## Описание


Метод Cut помещает выделенный
 фрагмент заголовка экспресс-отчета в буфер обмена, удаляя его из заголовка.


## Комментарии


Для вставки фрагмента, находящегося в буфере обмена, в заголовок экспресс-отчета
 используйте метод [IErAnalyzerTitle.Paste](IErAnalyzerTitle.Paste.htm).
