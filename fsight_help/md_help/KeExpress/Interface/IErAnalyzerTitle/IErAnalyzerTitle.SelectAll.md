# IErAnalyzerTitle.SelectAll


## Синтаксис


SelectAll;


## Описание


Метод SelectAll полностью выделяет
 заголовок экспресс-отчета.


## Комментарии


Для выделения необходимого фрагмента в заголовке экспресс-отчета используйте
 метод [IErAnalyzerTitle.SetSel](IErAnalyzerTitle.SetSel.htm).
