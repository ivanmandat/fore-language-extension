# IErAnalyzerTitle.AutoEndEdit


## Синтаксис


AutoEndEdit: Boolean;


## Описание


Свойство AutoEndEdit определяет
 признак завершения редактирования заголовка при потере фокуса.


## Комментарии


Допустимые значения:


	- True. Редактирование
	 заголовка завершается при потере фокуса;


	- False. Редактирование
	 заголовка завершается только при вызове метода [IErAnalyzerTitle.EndEdit](IErAnalyzerTitle.EndEdit.htm).
