# IErAnalyzerTitle.ShowFontDialog


## Синтаксис


ShowFontDialog;


## Описание


Метод ShowFontDialog вызывает
 диалог настройки шрифта для заголовка.


## Комментарии


Если в диалоге нажать кнопку «ОК»,
 то заданные параметры шрифта будут применены в заголовке.
