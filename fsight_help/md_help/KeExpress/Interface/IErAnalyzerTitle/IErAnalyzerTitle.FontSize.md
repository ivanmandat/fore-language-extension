# IErAnalyzerTitle.FontSize


## Синтаксис


FontSize: Double;


## Описание


Свойство FontSize определяет
 размер шрифта заголовка экспресс-отчета.


## Комментарии


Для определения семейства шрифтов заголовка экспресс-отчета используйте
 свойство [IErAnalyzerTitle.FontFamily](IErAnalyzerTitle.FontFamily.htm).
