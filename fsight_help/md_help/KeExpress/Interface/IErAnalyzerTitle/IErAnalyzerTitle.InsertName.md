# IErAnalyzerTitle.InsertName


## Синтаксис


InsertName;


## Описание


Метод InsertName вставляет наименование
 текущего экспресс-отчета в заголовок.


## Комментарии


Для вставки наименования текущего измерения используйте метод [IErAnalyzerTitle.InsertSelection](IErAnalyzerTitle.InsertSelection.htm).
