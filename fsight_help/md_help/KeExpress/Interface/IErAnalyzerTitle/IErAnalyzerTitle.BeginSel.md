# IErAnalyzerTitle.BeginSel


## Синтаксис


BeginSel: Integer;


## Описание


Свойство BeginSel возвращает
 текущую начальную позицию выделенного фрагмента в заголовке экспресс-отчета.


## Комментарии


Для получения текущей конечной позиции выделенного фрагмента в заголовке
 экспресс-отчета используйте [IErAnalyzerTitle.EndSel](IErAnalyzerTitle.EndSel.htm).
