# IEaxDataAreaExternalTransformationControl


Сборка: Express;


## Описание


Интерфейс IEaxDataAreaExternalTransformationControl
 предназначен настройки привязки параметра задачи моделирования.


## Иерархия наследования


           IEaxDataAreaExternalTransformationControl


## Комментарии


Настройки привязки параметра можно получить, используя свойства и методы
 интерфейса [IEaxDataAreaExternalTransformationControlInfo](../IEaxDataAreaExternalTransformationControlInfo/IEaxDataAreaExternalTransformationControlInfo.htm).
