# IEaxDrillSerieResult


Сборка: Express;


## Описание


Интерфейс IEaxDrillSerieResult
 предназначен для получения информации о привязке ряда данных диаграммы
 к данным источника.


## Иерархия наследования


           [IEaxDrillResult](../IEaxDrillResult/IEaxDrillResult.htm)


           IEaxDrillSerieResult


## Комментарии


Информацию о привязке ряда данных можно получить, используя метод [IEaxAnalyzeCore.DrillSerie](../IEaxAnalyzeCore/IEaxAnalyzeCore.DrillSerie.htm).
