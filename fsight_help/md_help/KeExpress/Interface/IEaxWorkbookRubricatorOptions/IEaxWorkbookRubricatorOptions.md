# IEaxWorkbookRubricatorOptions


Сборка: Express;


## Описание


Интерфейс IEaxWorkbookRubricatorOptions
 используется для работы с настройками базы данных временных рядов.


## Иерархия наследования


           IEaxWorkbookRubricatorOptions


## Комментарии


Для получения настроек всех баз данных временных рядов, используемых
 рабочей книгой, используйте свойство [IEaxWorkbookMetaAttributesTree.RubricatorsOptions](../IEaxWorkbookMetaAttributesTree/IEaxWorkbookMetaAttributesTree.RubricatorsOptions.htm).
