# IEaxWorkbookRubricatorOptions.HierarchyAttributes


## Синтаксис


HierarchyAttributes: String;


## Описание


Свойство HierarchyAttributes
 определяет настройки [дерева
 показателей](UiDw.chm::/Purpose/UiDw_Purpose_Object.htm)
 рабочей книги.


## Комментарии


В HierarchyAttributes надо сохранять данные, полученные с помощью метода
 [MetaAttributesBreadcrumb.SaveState](ModForms.chm::/Interface/IBreadcrumb/IBreadcrumb.SaveState.htm).


Данные, сохраненные в HierarchyAttributes, надо загружать с помощью
 метода [MetaAttributesBreadcrumb.LoadState](ModForms.chm::/Interface/IBreadcrumb/IBreadcrumb.LoadState.htm).
