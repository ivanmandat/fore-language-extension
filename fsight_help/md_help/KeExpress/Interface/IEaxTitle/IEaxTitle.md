# IEaxTitle


Сборка: Express;


## Описание


Интерфейс IEaxTitle используется
 для работы с заголовком экспресс-отчета.


## Иерархия наследования


           [IGxTitle](moddrawing.chm::/Interface/IGxTitle/IGxTitle.htm)


           IEaxTitle


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [TagSelection](IEaxTitle.TagSelection.htm)
		 Свойство TagSelection определяет
		 подстановку для метрики визуализатора в заголовке экспресс-отчета.


## Свойства, унаследованные от [IGxTitle](moddrawing.chm::/Interface/IGxTitle/IGxTitle.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BackgroundColor](moddrawing.chm::/Interface/IGxTitle/IGxTitle.BackgroundColor.htm)
		 Свойство BackgroundColor
		 определяет цвет фона заголовка.


		 ![](../../Property_Image.gif)
		 [BlackAndWhite](moddrawing.chm::/Interface/IGxTitle/IGxTitle.BlackAndWhite.htm)
		 Свойство BlackAndWhite
		 определяет признак черно-белой печати заголовка.


		 ![](../../Property_Image.gif)
		 [Font](moddrawing.chm::/Interface/IGxTitle/IGxTitle.Font.htm)
		 Свойство Font определяет
		 шрифт заголовка.


		 ![](../../Property_Image.gif)
		 [Height](moddrawing.chm::/Interface/IGxTitle/IGxTitle.Height.htm)
		 Свойство Height определяет
		 высоту области заголовка.


		 ![](../../Property_Image.gif)
		 [IsControl](moddrawing.chm::/Interface/IGxTitle/IGxTitle.IsControl.htm)
		 Свойство IsControl
		 определяет признак отображения заголовка как элемента управления.


		 ![](../../Property_Image.gif)
		 [IsTagIncluded](moddrawing.chm::/Interface/IGxTitle/IGxTitle.IsTagIncluded.htm)
		 Свойство IsTagIncluded
		 определяет, присутствует ли в тексте заголовка указанная подстановка.


		 ![](../../Property_Image.gif)
		 [Text](moddrawing.chm::/Interface/IGxTitle/IGxTitle.Text.htm)
		 Свойство Text определяет
		 текст заголовка.


		 ![](../../Property_Image.gif)
		 [TextAlignment](moddrawing.chm::/Interface/IGxTitle/IGxTitle.TextAlignment.htm)
		 Свойство TextAlignment
		 определяет выравнивание текста в заголовке.


		 ![](../../Property_Image.gif)
		 [TextColor](moddrawing.chm::/Interface/IGxTitle/IGxTitle.TextColor.htm)
		 Свойство TextColor
		 определяет цвет текста заголовка.


		 ![](../../Property_Image.gif)
		 [Title](moddrawing.chm::/Interface/IGxTitle/IGxTitle.Title.htm)
		 Свойство Title возвращает
		 текст заголовка, полученный после обработки автоподстановок.


		 ![](../../Property_Image.gif)
		 [Visible](moddrawing.chm::/Interface/IGxTitle/IGxTitle.Visible.htm)
		 Свойство Visible определяет
		 признак видимости заголовка.
