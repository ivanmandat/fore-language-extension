# IEaxConfigurationPadPanel.Form


## Синтаксис


Form: [IFormControl](ModForms.chm::/Interface/IFormControl/IFormControl.htm);


## Описание


Свойство Form возвращает контекст формы отображаемой в области панели.


## Комментарии


Идентификатор формы, которая будет отображаться в области панели, задается в свойстве [IEaxConfigurationPadPanel.MetabaseObject](IEaxConfigurationPadPanel.MetabaseObject.htm).
