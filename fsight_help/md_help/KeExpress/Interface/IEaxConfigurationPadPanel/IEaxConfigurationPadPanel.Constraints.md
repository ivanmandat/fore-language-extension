# IEaxConfigurationPadPanel.Constraints


## Синтаксис


Constraints: [ISizeConstraints](ModForms.chm::/Interface/ISizeConstraints/ISizeConstraints.htm);


## Описание


Свойство Constraints определяет
 ограничения на размеры панели, начиная с которых будут отображаться полосы
 прокрутки.


## Комментарии


В свойстве можно задать минимальные размеры панели, начиная с которых
 будут отображаться полосы прокрутки.


Примечание: Панель
 вкладки поддерживает только следующие свойства: [ISizeConstraints.MinHeigh](ModForms.chm::/Interface/ISizeConstraints/ISizeConstraints.MinHeight.htm);
 [ISizeConstraints.MinWidth](ModForms.chm::/Interface/ISizeConstraints/ISizeConstraints.MinWidth.htm).
