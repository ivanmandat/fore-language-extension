# IEaxWorkbook.LanerContext


## Синтаксис


LanerContext: [ILanerContext](Laner.chm::/Interface/ILanerContext/ILanerContext.htm);


## Описание


Свойство LanerContext
 возвращает контекст рабочей книги.


## Комментарии


Свойство актуально, если свойство [IEaxWorkbook.UseLanerContext](IEaxWorkbook.UseLanerContext.htm) принимает значение True.
