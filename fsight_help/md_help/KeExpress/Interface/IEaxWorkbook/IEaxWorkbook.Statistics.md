# IEaxWorkbook.Statistics


## Синтаксис


Statistics: [IEaxWorkbookStatistics](../IEaxWorkbookStatistics/IEaxWorkbookStatistics.htm);


## Описание


Свойство Statistics возвращает
 параметры [панели
 статистик](UiDw.chm::/Workbook/UiDw_Description.htm).


## Комментарии


Свойство недоступно, если рабочая книга находится в многомерном режиме.
 Режим отображения рабочей книги определяет свойство [IEaxAnalyzeCore.Mode](../IEaxAnalyzeCore/IEaxAnalyzeCore.Mode.htm).

 и загрузка настроек дерева объектов рабочей книги](../../Samples/KeExpress_EaxWorkbook_fore.htm)».
