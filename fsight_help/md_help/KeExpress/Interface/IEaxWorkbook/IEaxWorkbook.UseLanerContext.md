# IEaxWorkbook.UseLanerContext


## Синтаксис


UseLanerContext: Boolean;


## Описание


Свойство UseLanerContext определяет,
 зарегистрирована ли рабочая книга в контексте.


## Комментарии


Если рабочие книги базы данных временных рядов зарегистрированы в одном
 контексте, то между ними будет производиться синхронизация данных. При
 изменении значений ряда одной рабочей книги соответствующие данные будут
 обновлены во всех рядах с теми же значениями обязательных атрибутов.


Допустимые значения:


	- True. Рабочая книга
	 зарегистрирована в контексте;


	- False. Рабочая книга
	 исключена из контекста.


При создании рабочей книги из Fore она не будет автоматически зарегистрирована
 в контексте. При открытии или создании книги в репозитории она автоматически
 будет зарегистрирована в контексте. При включенном контексте, он будет
 сохраняться/загружаться вместе с рабочей книгой. При добавлении/удалении
 нового листа, он будет автоматически регистрироваться/исключаться в/из
 контекста.
