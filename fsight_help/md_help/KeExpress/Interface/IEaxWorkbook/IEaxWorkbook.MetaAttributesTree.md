# IEaxWorkbook.MetaAttributesTree


## Синтаксис


MetaAttributesTree: [IEaxWorkbookMetaAttributesTree](../IEaxWorkbookMetaAttributesTree/IEaxWorkbookMetaAttributesTree.htm);


## Описание


Свойство MetaAttributesTree
 возвращает параметры [дерева
 рядов](UiDw.chm::/Purpose/UiDw_Purpose_Object.htm).


## Комментарии


Свойство недоступно, если рабочая книга находится в многомерном режиме.
 Режим отображения рабочей книги определяет свойство [IEaxAnalyzeCore.Mode](../IEaxAnalyzeCore/IEaxAnalyzeCore.Mode.htm).
