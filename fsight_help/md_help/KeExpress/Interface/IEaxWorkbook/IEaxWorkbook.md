# IEaxWorkbook


Сборка: Express;


## Описание


Интерфейс IEaxWorkbook используется
 для работы с параметрами рабочей книги базы данных временных рядов.


## Иерархия наследования


           IEaxWorkbook


## Комментарии


Для работы с объектами рабочей книги приведите интерфейс [IEaxAnalyzer](../IEaxAnalyzer/IEaxAnalyzer.htm) к интерфейсу IEaxWorkbook.
