# IDataAreaComment


Сборка: Express;


## Описание


Интерфейс IDataAreaComment предназначен
 для работы с комментарием, создаваемым для ячеек с данными.


## Иерархия наследования


IDataAreaComment


## Комментарии


Комментарий для ячеек с данными можно получить, используя свойства и
 методы интерфейса [IDataAreaComments](../IDataAreaComments/IDataAreaComments.htm).
