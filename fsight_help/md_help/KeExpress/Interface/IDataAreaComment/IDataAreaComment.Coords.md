# IDataAreaComment.Coords


## Синтаксис


Coords: Array;


## Описание


Свойство Coords возвращает массив
 с координатами ячеек, в которых присутствует данный комментарий.


## Комментарии


Каждая нечётная координата представляет строку, каждая чётная - столбец.
