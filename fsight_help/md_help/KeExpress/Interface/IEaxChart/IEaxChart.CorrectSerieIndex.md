# IEaxChart.CorrectSerieIndex


## Синтаксис


CorrectSerieIndex(SerieIndex: Integer): Integer;


## Параметры


SerieIndex. Индекс ряда диаграммы.


## Описание


Метод CorrectSerieIndex осуществляет
 корректировку индекса указанного ряда диаграммы.


## Комментарии


Для детализации данных по ряду диаграммы используйте метод [IEaxChart.DrillSerie](IEaxChart.DrillSerie.htm).
