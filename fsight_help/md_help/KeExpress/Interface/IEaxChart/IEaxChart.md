# IEaxChart


Сборка: Express;


## Описание


Интерфейс IEaxChart содержит
 свойства и методы объекта экспресс-отчета - Диаграмма.


## Иерархия наследования


           [IEaxObject](../IEaxObject/IEaxObject.htm)


           IEaxChart


## Свойства

		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BuildPointNameHierarchy](IEaxChart.BuildPointNameHierarchy.htm)

		 включает или выключает построение иерархии точек для диаграммы.

		 [Chart](IEaxChart.Chart.htm)

		 ![](../../Property_Image.gif)
		 [DisplayHidden](IEaxChart.DisplayHidden.htm)
		 Свойство DisplayHidden
		 определяет, отображать ли на диаграмме скрытые строки/столбцы.


		 ![](../../Property_Image.gif)
		 [EvaluatedElements](IEaxChart.EvaluatedElements.htm)
		 Свойство EvaluatedElements
		 определяет тип вычисляемых элементов, отображаемых на диаграмме.


		 ![](../../Property_Image.gif)
		 [GetDataRange](IEaxChart.GetDataRange.htm)
		 Свойство GetDataRange
		 возвращает диапазон ячеек, по которому строится диаграмма.


		 ![](../../Property_Image.gif)
		 [GetSerieTabRange](IEaxChart.GetSerieTabRange.htm)
		 Свойство GetSerieTabRange
		 возвращает диапазон ячеек, на основании которого построен указанный
		 ряд.


		 ![](../../Property_Image.gif)
		 [PointChartMode](IEaxChart.PointChartMode.htm)
		 Свойство PointChartMode
		 определяет способ передачи данных при построении точечной диаграммы.


		 ![](../../Property_Image.gif)
		 [Range](IEaxChart.Range.htm)
		 Свойство Range определяет
		 диапазон выделенной области таблицы, по которому строится диаграмма.


		 ![](../../Property_Image.gif)
		 [SerieDrillable](IEaxChart.SerieDrillable.htm)
		 Свойство SerieDrillable
		 определяет, возможно ли произвести детализацию данных по указанному
		 ряду диаграммы.


		 ![](../../Property_Image.gif)
		 [SeriesInRows](IEaxChart.SeriesInRows.htm)
		 Свойство SeriesInRows
		 определяет, каким образом будут браться ряды с данными.


		 ![](../../Property_Image.gif)
		 [SeriesLimit](IEaxChart.SeriesLimit.htm)
		 Свойство SeriesLimit
		 определяет максимальное количество рядов, отображаемых на диаграмме.


		 ![](../../Property_Image.gif)
		 [SingleCellRangeMode](IEaxChart.SingleCellRangeMode.htm)
		 Свойство SingleCellRangeMode
		 определяет режим построения диаграммы при одной выделенной ячейке.


		 ![](../../Property_Image.gif)
		 [UseNormalization](IEaxChart.UseNormalization.htm)
		 Свойство UseNormalization
		 определяет, будут ли выводиться нормализованные данные.
