# IEaxChart.GetPointTabRange


## Синтаксис


GetPointTabRange(SerieIndex: Integer, PointIndex:
 Integer): [ITabRange](TabSheet.chm::/Interface/ITabRange/ITabRange.htm);


## Параметры


SerieIndex. Индекс ряда данных.


PointIndex. Индекс точки ряда
 данных.


## Описание


Метод GetPointTabRange возвращает
 диапазон данных, на основании которого построена указанная точка.
