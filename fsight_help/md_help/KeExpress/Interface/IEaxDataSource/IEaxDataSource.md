# IEaxDataSource


Сборка: Express;


## Описание


Интерфейс IEaxDataSource содержит
 свойства источника данных экспресс-отчета.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DisplaySourceName](IEaxDataSource.DisplaySourceName.htm)
		 Свойство DisplaySourceName
		 определяет, будут ли отображаться наименования источников.


		 ![](../../Property_Image.gif)
		 [Id](IEaxDataSource.Id.htm)
		 Свойство Id возвращает
		 идентификатор объекта репозитория, являющегося источником данных
		 для экспресс-отчета.


		 ![](../../Property_Image.gif)
		 [Key](IEaxDataSource.Key.htm)
		 Свойство Key возвращает
		 ключ объекта репозитория, являющегося источником данных для экспресс-отчета.


		 ![](../../Property_Image.gif)
		 [VirtualCubeSource](IEaxDataSource.VirtualCubeSource.htm)
		 Свойство VirtualCubeSource
		 возвращает параметры источника данных экспресс-отчета.
