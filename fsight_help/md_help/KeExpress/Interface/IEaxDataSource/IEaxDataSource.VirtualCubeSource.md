# IEaxDataSource.VirtualCubeSource


## Синтаксис


VirtualCubeSource: [IVirtualCubeSource](KeCubes.chm::/Interface/IVirtualCubeSource/IVirtualCubeSource.htm);


## Описание


Свойство VirtualCubeSource возвращает
 параметры источника данных экспресс-отчета.


## Комментарии


Так как при создании экспресс-отчета на базе нескольких источников создается
 виртуальный куб, то работа с каждым отдельным источником осуществляется
 так же, как и с отдельными источниками виртуального куба.
