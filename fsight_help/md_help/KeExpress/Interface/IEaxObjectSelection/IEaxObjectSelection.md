# IEaxObjectSelection


Сборка: Express;


## Описание


Интерфейс IEaxObjectSelection
 предназначен для работы с отметкой объекта.


## Иерархия наследования


           IEaxObjectSelection


## Комментарии


Отметка объекта предназначена для получения отметки измерений, соответствующей
 выделению областей в объекте.


Для получения отметки объекта используйте свойство [IEaxObjectSelectionManager.Selection](../IEaxObjectSelectionManager/IEaxObjectSelectionManager.Selection.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Combined](IEaxObjectSelection.Combined.htm)
		 Свойство Combined возвращает
		 совмещенную отметку, соответствующую всем областям, выделенным в таблице.


		 ![](../../Property_Image.gif)
		 [Count](IEaxObjectSelection.Count.htm)
		 Свойство Count
		 возвращает количество областей, выделенных в объекте.


		 ![](../../Property_Image.gif)
		 [Item](IEaxObjectSelection.Item.htm)
		 Свойство Item
		 возвращает отметку, соответствующей указанной выделенной области
		 объекта.


		 ![](../../Property_Image.gif)
		 [Matrix](IEaxObjectSelection.Matrix.htm)
		 Свойство Matrix
		 возвращает матрицу данных, соответствующую областям, которые выделены
		 в объекте.
