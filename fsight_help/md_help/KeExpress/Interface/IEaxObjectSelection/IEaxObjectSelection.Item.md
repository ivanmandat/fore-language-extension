# IEaxObjectSelection.Item


## Синтаксис


Item(Index: Integer): [IDimSelectionSet](KeDims.chm::/interface/IDimSelectionSet/IDimSelectionSet.htm);


## Параметры


Index. Индекс выделенной области.


## Описание


Свойство Item возвращает
 отметку, соответствующей указанной выделенной области объекта.


## Комментарии


Индексация выделенных областей сквозная и начинается с нуля. Для получения
 количества областей, выделенных в объекте, используйте свойство [IEaxObjectSelection.Count](IEaxObjectSelection.Count.htm).
