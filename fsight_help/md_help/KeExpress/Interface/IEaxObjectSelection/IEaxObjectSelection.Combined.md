# IEaxObjectSelection.Combined


## Синтаксис


Combined: [IDimSelectionSet](KeDims.chm::/interface/IDimSelectionSet/IDimSelectionSet.htm);


## Описание


Свойство Combined возвращает
 совмещенную отметку, соответствующую всем областям, выделенным в таблице.


## Комментарии


Для получения отдельных отметок областей используйте свойство [IEaxObjectSelection.Item](IEaxObjectSelection.Item.htm).
