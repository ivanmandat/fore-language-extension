# ILanerResultsBoxCorrelation.TermCount


## Синтаксис


TermCount: Integer;


## Описание


Свойство TermCount возвращает
 количество переменных, участвовавших в расчете корреляции.


## Комментарии


TermCount возвращает положительное
 значение, только если вкладка «Корреляция»
 присутствует на панели «Результаты вычислений»
 и корреляция была рассчитана.
