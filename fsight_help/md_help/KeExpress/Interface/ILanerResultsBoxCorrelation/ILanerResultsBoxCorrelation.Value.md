# ILanerResultsBoxCorrelation.Value


## Синтаксис


Value(Row: Integer; Col: Integer): Double;


## Параметры


Row. Индекс переменной в строке;


Col. Индекс переменной в столбце.


## Описание


Свойство Value возвращает
 значение корреляции для указанных переменных.


## Комментарии


Количество столбцов и строк в таблице корреляции совпадает и соответствует
 количеству переменных.


Значение параметров Row и Col должно входить в диапазон [0,
 [ILanerResultsBoxCorrelation.TermCount](ILanerResultsBoxCorrelation.TermCount.htm)-1].
