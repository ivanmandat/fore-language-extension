# ILanerResultsBoxCorrelation


Сборка: Express;


## Описание


Интерфейс ILanerResultsBoxCorrelation предназначен
 для работы со значениями на вкладке «Корреляция»
 панели «Результаты вычислений»
 для компонента LanerResultsBox.


## Комментарии


ILanerResultsBoxCorrelation применяется
 для получения таблицы корреляции. Изменение их значений невозможно. Количество
 столбцов и строк в таблице совпадает и соответствует количеству переменных,
 для которых находилась корреляция. Значения вычисляются только для рядов,
 рассчитанных с помощью метода моделирования «Линейная регрессия».
