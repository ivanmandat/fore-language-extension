# ILanerResultsBoxCorrelation.Comment


## Синтаксис


Comment(Index: Integer): String;


## Параметры


Index. Индекс переменной.


## Описание


Свойство Comment возвращает
 комментарий к переменной по ее индексу.


## Комментарии


Значение параметра Index должно
 входить в диапазон [0, [ILanerResultsBoxCorrelation.TermCount](ILanerResultsBoxCorrelation.TermCount.htm)-1].
