# IEaxDataAreaTransformation.Element


## Синтаксис


Element: Integer;


## Описание


Свойство Element возвращает
 индекс вычисляемого элемента в измерении, которому соответствует формула
 преобразования.


## Комментарии


Для получения идентификатора вычисляемого элемента используйте [IEaxDataAreaTransformation.ElementId](IEaxDataAreaTransformation.ElementId.htm).
