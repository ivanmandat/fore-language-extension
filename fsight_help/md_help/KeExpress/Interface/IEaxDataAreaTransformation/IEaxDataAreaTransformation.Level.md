# IEaxDataAreaTransformation.Level


## Синтаксис


Level: [DimCalendarLevel](KeDims.chm::/Enums/DimCalendarLevel.htm);


## Описание


Свойство Level определяет календарную
 динамику формулы.


## Комментарии


Для настройки периода расчета формулы и периода выгрузки данных используйте
 свойства [IEaxDataAreaTransformation.Period](IEaxDataAreaTransformation.Period.htm)
 и [IEaxDataAreaTransformation.OutputPeriod](IEaxDataAreaTransformation.OutputPeriod.htm).
