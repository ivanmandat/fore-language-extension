# IEaxDataAreaTransformation.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет,
 будет ли активна формула преобразования данных.


## Комментарии


Допустимые значения:


	- True. Формула преобразования
	 данных будет активна;


	- False. Формула преобразования
	 данных активна не будет.


Для получения выражения, преобразующего данные аналитической области,
 используйте [IEaxDataAreaTransformation.Expression](IEaxDataAreaTransformation.Expression.htm).
