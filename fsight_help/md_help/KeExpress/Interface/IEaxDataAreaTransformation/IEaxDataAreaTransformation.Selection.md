# IEaxDataAreaTransformation.Selection


## Синтаксис


Selection: [IDimSelectionSet](KeDims.chm::/interface/IDimSelectionSet/IDimSelectionSet.htm);


## Описание


Свойство Selection определяет
 отметку измерения, в котором находится формула преобразования данных.


## Комментарии


Для задания ключа измерения, в котором находится формула преобразования
 данных, используйте [IEaxDataAreaTransformation.DimKey](IEaxDataAreaTransformation.DimKey.htm).
