# IEaxDataAreaTransformation.DimKey


## Синтаксис


DimKey: Integer;


## Описание


Свойство DimKey определяет ключ
 измерения, в котором находится формула преобразования данных.


## Комментарии


Для определения отметки измерения, в котором находится формула преобразования
 данных, используйте [IEaxDataAreaTransformation.Selection](IEaxDataAreaTransformation.Selection.htm).
