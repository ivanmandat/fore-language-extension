# IEaxDataAreaTransformation.GeneratedName


## Синтаксис


GeneratedName: String;


## Описание


Свойство GeneratedName возвращает
 сгенерированное наименование формулы преобразования данных.


## Комментарии


Данное свойство актуально, если в свойстве [IEaxDataAreaTransformation.Selection](IEaxDataAreaTransformation.Selection.htm)
 указана отметка.
