# IEaxDataAreaTransformation.Key


## Синтаксис


Key: Integer;


## Описание


Свойство Key возвращает ключ
 формулы преобразования данных в рамках коллекции, в которой она создана.


## Комментарии


Ключ - это уникальный идентификатор компонента в коллекции. Значение
 ключа генерируется автоматически и не может быть изменено пользователем.
