# IEaxDataAreaTransformation


Сборка: Express;


## Описание


Интерфейс IEaxDataAreaTransformation
 используется для работы с формулой преобразования данных в аналитической
 области.


## Иерархия наследования


           IEaxDataAreaTransformation


## Комментарии


Для получения объекта данного типа используйте свойства и методы интерфейса
 [IEaxDataAreaTransformations](../IEaxDataAreaTransformations/IEaxDataAreaTransformations.htm).
