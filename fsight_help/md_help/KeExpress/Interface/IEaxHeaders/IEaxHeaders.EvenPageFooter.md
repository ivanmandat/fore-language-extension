# IEaxHeaders.EvenPageFooter


## Синтаксис


EvenPageFooter: [IEaxHeader](../IEaxHeader/IEaxHeader.htm);


## Описание


Свойство EvenPageFooter возвращает
 параметры нижнего колонтитула чётных страниц.


## Комментарии


Для получения параметров верхнего колонтитула чётных страниц используйте
 свойство [IEaxHeaders.EvenPageHeader](IEaxHeaders.EvenPageHeader.htm).


Для использования разных колонтитулов для чётных и нечётных страниц
 используйте свойство [IEaxHeaders.OddAndEvenPagesHeaderFooter](IEaxHeaders.OddAndEvenPagesHeaderFooter.htm).
