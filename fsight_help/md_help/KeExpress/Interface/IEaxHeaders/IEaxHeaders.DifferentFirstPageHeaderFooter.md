# IEaxHeaders.DifferentFirstPageHeaderFooter


## Синтаксис


DifferentFirstPageHeaderFooter: Boolean;


## Описание


Свойство DifferentFirstPageHeaderFooter
 определяет использование особого колонтитула для первой страницы.


## Комментарии


Допустимые значения:


	- True. Для первой страницы
	 будут использованы особые настройки колонтитулов;


	- False. Для первой страницы
	 не будут использованы особые настройки колонтитулов.


Для получения параметров верхнего и нижнего колонтитулов первой страницы
 используйте свойства [IEaxHeaders.FirstPageHeader](IEaxHeaders.FirstPageHeader.htm)
 и [IEaxHeaders.FirstPageFooter](IEaxHeaders.FirstPageFooter.htm)
 соответственно.
