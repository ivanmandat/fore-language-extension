# IEaxHeaders


Сборка: Express;


## Описание


Интерфейс IEaxHeaders предназначен
 для работы с коллекцией колонтитулов страниц экспресс-отчета.


## Иерархия наследования


           IEaxHeaders


## Комментарии


Для настройки параметров выбранного колонтитула используйте свойства
 интерфейса [IEaxHeader](../IEaxHeader/IEaxHeader.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DifferentFirstPageHeaderFooter](IEaxHeaders.DifferentFirstPageHeaderFooter.htm)
		 Свойство DifferentFirstPageHeaderFooter
		 определяет использование особого колонтитула для первой страницы.


		 ![](../../Property_Image.gif)
		 [EvenPageFooter](IEaxHeaders.EvenPageFooter.htm)
		 Свойство EvenPageFooter
		 возвращает параметры нижнего колонтитула чётных страниц.


		 ![](../../Property_Image.gif)
		 [EvenPageHeader](IEaxHeaders.EvenPageHeader.htm)
		 Свойство EvenPageHeader
		 возвращает параметры верхнего колонтитула чётных страниц.


		 ![](../../Property_Image.gif)
		 [FirstPageFooter](IEaxHeaders.FirstPageFooter.htm)
		 Свойство FirstPageFooter
		 возвращает параметры нижнего колонтитула первой страницы.


		 ![](../../Property_Image.gif)
		 [FirstPageHeader](IEaxHeaders.FirstPageHeader.htm)
		 Свойство FirstPageHeader
		 возвращает параметры верхнего колонтитула первой страницы.


		 ![](../../Property_Image.gif)
		 [Footer](IEaxHeaders.Footer.htm)
		 Свойство Footer возвращает
		 параметры нижнего колонтитула.


		 ![](../../Property_Image.gif)
		 [Header](IEaxHeaders.Header.htm)
		 Свойство Header возвращает
		 параметры верхнего колонтитула.


		 ![](../../Property_Image.gif)
		 [OddAndEvenPagesHeaderFooter](IEaxHeaders.OddAndEvenPagesHeaderFooter.htm)
		 Свойство OddAndEvenPagesHeaderFooter
		 определяет использование разных колонтитулов для чётных и нечётных
		 страниц.
