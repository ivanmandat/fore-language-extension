# IEaxHeaders.OddAndEvenPagesHeaderFooter


## Синтаксис


OddAndEvenPagesHeaderFooter: Boolean;


## Описание


Свойство OddAndEvenPagesHeaderFooter
 определяет использование разных колонтитулов для чётных и нечётных страниц.


## Комментарии


Допустимые значения:


	- True. Для чётных страниц
	 будут использованы особые настройки колонтитулов;


	- False. Для чётных и
	 нечётных страниц будут использованы одинаковые настройки колонтитулов.


Для получения параметров верхнего и нижнего колонтитулов чётных страниц
 используйте свойства [IEaxHeaders.EvenPageHeader](IEaxHeaders.EvenPageHeader.htm)
 и [IEaxHeaders.EvenPageFooter](IEaxHeaders.EvenPageFooter.htm)
 соответственно.
