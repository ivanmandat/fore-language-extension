# IEaxHeaders.FirstPageFooter


## Синтаксис


FirstPageFooter: [IEaxHeader](../IEaxHeader/IEaxHeader.htm);


## Описание


Свойство FirstPageFooter возвращает
 параметры нижнего колонтитула первой страницы.


## Комментарии


Для получения параметров верхнего колонтитула первой страницы используйте
 свойство [IEaxHeaders.FirstPageHeader](IEaxHeaders.FirstPageHeader.htm).


Для использования настроек особого колонтитула для первой страницы используйте
 свойство [IEaxHeaders.DifferentFirstPageHeaderFooter](IEaxHeaders.DifferentFirstPageHeaderFooter.htm).
