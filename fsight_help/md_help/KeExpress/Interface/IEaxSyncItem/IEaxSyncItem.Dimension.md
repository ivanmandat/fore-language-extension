# IEaxSyncItem.Dimension


## Синтаксис


Dimension: [IEaxDimensionInstance](../IEaxDimensionInstance/IEaxDimensionInstance.htm);


## Описание


Свойство Dimension возвращает
 измерение, с которым связан элемент связи синхронизированных измерений.


## Комментарии


Для получения связи синхронизированных измерений, в которой находится
 элемент, используйте [IEaxSyncItem.SyncGroup](IEaxSyncItem.SyncGroup.htm).
