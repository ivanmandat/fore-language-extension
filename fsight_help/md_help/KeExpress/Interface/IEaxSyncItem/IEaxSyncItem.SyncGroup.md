# IEaxSyncItem.SyncGroup


## Синтаксис


SyncGroup: [IEaxSyncGroup](../IEaxSyncGroup/IEaxSyncGroup.htm);


## Описание


Свойство SyncGroup возвращает
 связь синхронизированных измерений, в которой находится элемент.


## Комментарии


Для получения измерения, с которым связан элемент связи синхронизированных
 измерений, используйте [IEaxSyncItem.Dimension](IEaxSyncItem.Dimension.htm).
