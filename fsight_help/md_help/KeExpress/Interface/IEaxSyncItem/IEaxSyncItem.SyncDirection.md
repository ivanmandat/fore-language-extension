# IEaxSyncItem.SyncDirection


## Синтаксис


SyncDirection: [EaxDimensionSynchronizationDirection](../../Enums/EaxDimensionSynchronizationDirection.htm);


## Описание


Свойство SyncDirection определяет
 тип связи, применяемой для синхронизации измерений.


## Комментарии


По умолчанию связь может как принимать, так и передавать отметку в управляемые
 срезы, то есть свойство SyncDirection
 имеет значение EaxDimensionSynchronizationDirection.ProduceAndConsumeChanges.
