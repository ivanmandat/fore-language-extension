# IEaxSyncItem


Сборка: Express;


## Описание


Интерфейс IEaxSyncItem используется
 для работы с элементом связи [синхронизированных
 измерений](UiReport.chm::/Desktop/Source/Dim/Symc_dimensions.htm).


## Иерархия наследования


           IEaxSyncItem


## Комментарии


Для работы со связью синхронизированных измерений используйте [IEaxSyncGroup](../IEaxSyncGroup/IEaxSyncGroup.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dimension](IEaxSyncItem.Dimension.htm)
		 Свойство Dimension
		 возвращает измерение, с которым связан элемент связи синхронизированных
		 измерений.


		 ![](../../Property_Image.gif)
		 [SyncAttribute](IEaxSyncItem.SyncAttribute.htm)
		 Свойство SyncAttribute
		 определяет атрибут синхронизации элемента связи синхронизированных
		 измерений.


		 ![](../../Property_Image.gif)
		 [SyncDirection](IEaxSyncItem.SyncDirection.htm)
		 Свойство SyncDirection
		 определяет тип связи, применяемой для синхронизации измерений.


		 ![](../../Property_Image.gif)
		 [SyncGroup](IEaxSyncItem.SyncGroup.htm)
		 Свойство SyncGroup
		 возвращает связь синхронизированных измерений, в которой находится
		 элемент.
