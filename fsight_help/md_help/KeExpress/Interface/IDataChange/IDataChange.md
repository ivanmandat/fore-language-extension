# IDataChange


Сборка: Express;


## Описание


Интерфейс IDataChange является
 базовым классом для работы с измененными данными области данных.


## Комментарии


Интерфейс IDataChange является
 базовым для следующих интерфейсов:


	- [IPrxDataChange](../../../KeReport/Interface/IPrxDataChange/IPrxDataChange.htm);


	- [IEaxDataChange](../IEaxDataChange/IEaxDataChange.htm).
