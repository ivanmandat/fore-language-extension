# IEaxDataSources.EndUpdate


## Синтаксис


EndUpdate([Refresh: Boolean = True]);


## Параметры


Refresh. Признак обновления
 коллекции источников данных. Необязательный параметр.


## Описание


Метод EndUpdate возобновляет
 перерисовку коллекции источников данных.


## Комментарии


Значение параметра Refresh
 по умолчанию - True (производить
 обновление).


Приостановка перерисовки коллекции осуществляется при помощи метода
 [IEaxDataSources.BeginUpdate](IEaxDataSources.BeginUpdate.htm)
 приостановил её.
