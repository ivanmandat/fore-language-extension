# IEaxDataSources.Refresh


## Синтаксис


Refresh([PreserveParams: Boolean = False]);


## Параметры


PreserveParams. Признак обновления
 параметров виртуального куба. Необязательный параметр.


## Описание


Метод Refresh осуществляет обновление
 коллекции источников данных и таблицы данных экспресс-отчёта.


## Комментарии


Значение параметра PreserveParams
 по умолчанию - False (не производить
 обновление).
