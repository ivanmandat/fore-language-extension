# IEaxDataSources.BeginUpdate


## Синтаксис


BeginUpdate;


## Описание


Метод BeginUpdate отключает
 любую перерисовку коллекции источников данных в экспресс-отчёте.


## Комментарии


Метод используется при изменении большого количества данных для исключения
 мерцания визуальных компонентов. Возобновление перерисовки осуществляется
 методом [IEaxDataSources.EndUpdate](IEaxDataSources.EndUpdate.htm).
