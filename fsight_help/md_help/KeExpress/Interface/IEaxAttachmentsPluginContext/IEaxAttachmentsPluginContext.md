# IEaxAttachmentsPluginContext


Сборка: Express;


## Описание


Интерфейс IEaxAttachmentsPluginContext
 описывает контекст для работы с вложениями для ячейки.


## Иерархия наследования


           IEaxAttachmentsPluginContext


## Комментарии


Интерфейс не содержит собственных свойств и методов. Контекст создаётся
 и связывается с ячейкой с помощью метода [IEaxGrid.CreateAttachmentsPluginContext](../IEaxGrid/IEaxGrid.CreateAttachmentsPluginContext.htm).
 Полученный контекст указывается в качестве значения свойства [IUiCommandExecutionContext.Data](UiLib.chm::/Interface/IUiCommandExecutionContext/IUiCommandExecutionContext.Data.htm)
 при работе с командой [ShowAttachmentsDialog](UiLib.chm::/Interface/IUiCommandTarget/Command/ShowAttachmentsDialog.htm).


Пример использования приведён в описании метода [IEaxGrid.CreateAttachmentsPluginContext](../IEaxGrid/IEaxGrid.CreateAttachmentsPluginContext.htm).
