# IEaxDataAreaParam.Hidden


## Синтаксис


Hidden: Boolean;


## Описание


Свойство Hidden определяет,
 является ли параметр аналитической области данных скрытым.


## Комментарии


Допустимые значения:


	- True. Параметр аналитической
	 области данных является скрытым;


	- False. Параметр аналитической
	 области данных является видимым.
