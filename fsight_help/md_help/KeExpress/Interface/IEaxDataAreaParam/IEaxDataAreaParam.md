# IEaxDataAreaParam


Сборка: Express;


## Описание


Интерфейс IEaxDataAreaParam
 предназначен для настройки параметра аналитической области данных.


## Иерархия наследования


           [INamedEntity](KeSom.chm::/Interface/INamedEntity/INamedEntity.htm)


           IEaxDataAreaParam


## Комментарии


Для получения параметров аналитической области данных используйте свойства
 и методы интерфейса [IEaxDataAreaParams](../IEaxDataAreaParams/IEaxDataAreaParams.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DataType](IEaxDataAreaParam.DataType.htm)
		 Свойство DataType определяет
		 тип данных параметра аналитической области данных.


		 ![](../../Property_Image.gif)
		 [DefaultValue](IEaxDataAreaParam.DefaultValue.htm)
		 Свойство DefaultValue
		 определяет значение, устанавливаемое для параметра аналитической
		 области данных по умолчанию.


		 ![](../../Property_Image.gif)
		 [Hidden](IEaxDataAreaParam.Hidden.htm)
		 Свойство Hidden определяет,
		 является ли параметр аналитической области данных скрытым.


		 ![](../../Property_Image.gif)
		 [LinkedObject](IEaxDataAreaParam.LinkedObject.htm)
		 Свойство LinkedObject
		 определяет вложенные объекты репозитория.


		 ![](../../Property_Image.gif)
		 [ParamType](IEaxDataAreaParam.ParamType.htm)
		 Свойство ParamType
		 определяет тип параметра аналитической области данных.


		 ![](../../Property_Image.gif)
		 [Slice](IEaxDataAreaParam.Slice.htm)
		 Свойство Slice возвращает
		 срез аналитической области данных, которому принадлежит параметр.


		 ![](../../Property_Image.gif)
		 [Tag](IEaxDataAreaParam.Tag.htm)
		 Свойство Tag не используется
		 компилятором. Можно определить и использовать свойство по своему
		 усмотрению.


		 ![](../../Property_Image.gif)
		 [Value](IEaxDataAreaParam.Value.htm)
		 Свойство Value определяет
		 значение параметра аналитической области данных.
