# IEaxDataAreaParam.DefaultValue


## Синтаксис


DefaultValue: Variant;


## Описание


Свойство DefaultValue определяет
 значение, устанавливаемое для параметра аналитической области данных по
 умолчанию.


## Комментарии


Для определения значения параметра аналитической области данных используйте
 свойство [IEaxDataAreaParam.Value](IEaxDataAreaParam.Value.htm).
