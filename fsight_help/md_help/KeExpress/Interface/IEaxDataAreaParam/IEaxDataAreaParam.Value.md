# IEaxDataAreaParam.Value


## Синтаксис


Value: Variant;


## Описание


Свойство Value определяет значение
 параметра аналитической области данных.


## Комментарии


Для определения значения, устанавливаемого для параметра аналитической
 области данных по умолчанию, используйте свойство [IEaxDataAreaParam.DefaultValue](IEaxDataAreaParam.DefaultValue.htm).
