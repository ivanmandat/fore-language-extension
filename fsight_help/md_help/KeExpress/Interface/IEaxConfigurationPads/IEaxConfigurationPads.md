# IEaxConfigurationPads


Сборка: Express;


## Описание


Интерфейс IEaxConfigurationPads
 содержит свойства и методы для работы с коллекцией вкладок боковой панели
 экспресс-отчета.


## Комментарии


Коллекция вкладок навигационной панели может быть получена в свойстве
 [IEaxCustomConfigurationCategory.Pads](../IEaxCustomConfigurationCategory/IEaxCustomConfigurationCategory.Pads.htm).


## Свойства интерфейса


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IEaxConfigurationPads.Count.htm)


		 Свойство Count возвращает
		 количество вкладок в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IEaxConfigurationPads.Item.htm)


		 Свойство Item возвращает
		 параметры указанной вкладки, индекс которой задается в качестве
		 входного параметра.


## Методы интерфейса


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IEaxConfigurationPads.Add.htm)
		 Метод Add добавляет
		 указанную вкладку в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](IEaxConfigurationPads.Clear.htm)
		 Метод Clear удаляет
		 все вкладки из коллекции.


		 ![](../../Sub_Image.gif)
		 [Insert](IEaxConfigurationPads.Insert.htm)
		 Метод Insert осуществляет
		 вставку указанной вкладки в указанную позицию в коллекции.


		 ![](../../Sub_Image.gif)
		 [PadIndex](IEaxConfigurationPads.PadIndex.htm)
		 Метод PadIndex возвращает
		 индекс вкладки в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IEaxConfigurationPads.Remove.htm)
		 Метод Remove удаляет
		 указанную вкладку из группы.
