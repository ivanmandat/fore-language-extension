# IEaxConfigurationPads.Insert


## Синтаксис


Insert(Index: Integer; Pad: [IEaxConfigurationPad](../IEaxConfigurationPad/IEaxConfigurationPad.htm));


## Параметры


Index. Индекс позиции.


Pad. Вкладка, которую необходимо добавить в коллекцию.


## Описание


Метод Insert осуществляет вставку указанной вкладки в указанную позицию в коллекции.
