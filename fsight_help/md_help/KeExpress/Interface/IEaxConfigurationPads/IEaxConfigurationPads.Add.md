# IEaxConfigurationPads.Add


## Синтаксис


Add(Pad: [IEaxConfigurationPad](../IEaxConfigurationPad/IEaxConfigurationPad.htm));


## Параметры


Pad. Вкладка, которую необходимо добавить в группу.


## Описание


Метод Add добавляет указанную вкладку в коллекцию.


## Комментарии


Вкладка добавляется в конец коллекции. Заголовок вкладки визуально будет отображен ниже всех остальных заголовков.
