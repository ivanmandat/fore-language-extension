# IEaxSyncGroup.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс элемента в связи.


## Описание


Метод Remove удаляет элемент
 связи синхронизированных измерений по его индексу.


## Комментарии


Если элемент связи будет удален, то метод вернет значение True.


Для удаления элемента связи синхронизированных измерений по описанию
 привязанного к нему измерения используйте [IEaxSyncGroup.RemoveByItem](IEaxSyncGroup.RemoveByItem.htm).
