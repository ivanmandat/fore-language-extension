# IEaxSyncGroup.Item


## Синтаксис


Item(Index: Integer): [IEaxSyncItem](../IEaxSyncItem/IEaxSyncItem.htm);


## Параметры


Index. Индекс элемента связи.


## Описание


Свойство Item возвращает элемент
 связи синхронизированных измерений по его индексу.


## Комментарии


Для получения количества элементов связи синхронизированных измерений
 используйте [IEaxSyncGroup.Count](IEaxSyncGroup.Count.htm).
