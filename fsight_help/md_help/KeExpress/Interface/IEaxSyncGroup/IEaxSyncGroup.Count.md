# IEaxSyncGroup.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 элементов связи синхронизированных измерений.


## Комментарии


Для получения элемента связи синхронизированных измерений по его индексу
 используйте свойство [IEaxSyncGroup.Item](IEaxSyncGroup.Item.htm).
