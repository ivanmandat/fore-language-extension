# IEaxSyncGroup


Сборка: Express;


## Описание


Интерфейс IEaxSyncGroup используется
 для работы со связью [синхронизированных
 измерений](UiReport.chm::/Desktop/Source/Dim/Symc_dimensions.htm).


## Иерархия наследования


           IEaxSyncGroup


## Комментарии


Для работы с коллекцией связей [синхронизированных
 измерений](UiReport.chm::/Desktop/Source/Dim/Symc_dimensions.htm) используйте [IEaxSyncGroups](../IEaxSyncGroups/IEaxSyncGroups.htm).


Для работы с элементом связи [синхронизированных
 измерений](UiReport.chm::/Desktop/Source/Dim/Symc_dimensions.htm) используйте [IEaxSyncItem](../IEaxSyncItem/IEaxSyncItem.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Item](IEaxSyncGroup.Item.htm)
		 Свойство Item возвращает
		 элемент связи синхронизированных измерений по его индексу.


		 ![](../../Property_Image.gif)
		 [Count](IEaxSyncGroup.Count.htm)
		 Свойство Count возвращает
		 количество элементов связи синхронизированных измерений.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IEaxSyncGroup.Add.htm)
		 Метод Add добавляет
		 элемент в связь синхронизированных измерений.


		 ![](../../Sub_Image.gif)
		 [Clear](IEaxSyncGroup.Clear.htm)
		 Метод Clear удаляет
		 все элементы из связи синхронизированных измерений.


		 ![](../../Sub_Image.gif)
		 [InGroup](IEaxSyncGroup.InGroup.htm)
		 Метод InGroup определяет,
		 связано ли указанное измерение с элементами связи синхронизированных
		 измерений.


		 ![](../../Sub_Image.gif)
		 [PropogateChanges](IEaxSyncGroup.PropogateChanges.htm)
		 Метод PropogateChanges
		 распространяет изменение отметки на остальные элементы связи синхронизированных
		 измерений.


		 ![](../../Sub_Image.gif)
		 [Remove](IEaxSyncGroup.Remove.htm)
		 Метод Remove удаляет
		 элемент связи синхронизированных измерений по его индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByItem](IEaxSyncGroup.RemoveByItem.htm)
		 Метод RemoveByItem
		 удаляет элемент связи синхронизированных измерений по описанию
		 привязанного к нему измерения.
