# IEaxSyncGroup.PropogateChanges


## Синтаксис


PropogateChanges(Dimension: [IEaxDimensionDescriptor](../IEaxDimensionDescriptor/IEaxDimensionDescriptor.htm));


## Параметры


Dimension. Описание измерения.


## Описание


Метод PropogateChanges распространяет
 изменение отметки на остальные элементы связи синхронизированных измерений.


## Комментарии


Для определения связано ли указанное измерение с элементами связи синхронизированных
 измерений, используйте [IEaxSyncGroup.InGroup](IEaxSyncGroup.InGroup.htm).
