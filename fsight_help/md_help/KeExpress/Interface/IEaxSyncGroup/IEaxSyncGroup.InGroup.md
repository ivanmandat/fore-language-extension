# IEaxSyncGroup.InGroup


## Синтаксис


InGroup(Dimension: [IEaxDimensionDescriptor](../IEaxDimensionDescriptor/IEaxDimensionDescriptor.htm)):
 Boolean;


## Параметры


Dimension. Описание измерения.


## Описание


Метод InGroup определяет, связано
 ли указанное измерение с элементами связи синхронизированных измерений.


## Комментарии


Для распространения изменения отметки на остальные элементы связи синхронизированных
 измерений используйте [IEaxSyncGroup.PropogateChanges](IEaxSyncGroup.PropogateChanges.htm).
