# IEaxSyncGroup.RemoveByItem


## Синтаксис


RemoveByItem(Dimension: [IEaxDimensionDescriptor](../IEaxDimensionDescriptor/IEaxDimensionDescriptor.htm)):
 Boolean;


## Параметры


Dimension. Описание измерения.


## Описание


Метод RemoveByItem удаляет элемент
 связи синхронизированных измерений по описанию привязанного к нему измерения.


## Комментарии


Если элемент связи будет удален, то метод вернет значение True.


Для удаления элемента связи синхронизированных измерений по его индексу
 используйте [IEaxSyncGroup.Remove](IEaxSyncGroup.Remove.htm).
