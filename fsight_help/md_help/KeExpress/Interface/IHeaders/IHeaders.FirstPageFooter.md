# IHeaders.FirstPageFooter


## Синтаксис


FirstPageFooter: [IHeader](../IHeader/IHeader.htm);


## Описание


Свойство FirstPageFooter возвращает
 параметры нижнего колонтитула первой страницы.


## Комментарии


Свойство актуально, если свойству [DifferentFirstPageHeaderFooter](IHeaders.DifferentFirstPageHeaderFooter.htm)
 установлено значение True.
