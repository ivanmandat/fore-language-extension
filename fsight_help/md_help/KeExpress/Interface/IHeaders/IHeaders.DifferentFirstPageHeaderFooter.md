# IHeaders.DifferentFirstPageHeaderFooter


## Синтаксис


DifferentFirstPageHeaderFooter: Boolean;


## Описание


Свойство DifferentFirstPageHeaderFooter
 определяет признак использования особого колонтитула для первой страницы.


## Комментарии


Допустимые значения:


	- True. Для первой страницы
	 будут использованы особые настройки колонтитулов;


	- False. Для первой страницы
	 будут использованы стандартные настройки колонтитулов.
