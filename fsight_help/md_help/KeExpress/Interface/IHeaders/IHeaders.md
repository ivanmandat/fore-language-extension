# IHeaders


Сборка: Express;


## Описание


Интерфейс IHeaders предназначен
 для работы с коллекцией колонтитулов страниц на печати.


## Иерархия наследования


IHeaders


## Комментарии


Интерфейс является базовым для интерфейса [IPrxSheetHeaders](KeReport.chm::/Interface/IPrxSheetHeaders/IPrxSheetHeaders.htm).
 Коллекцию колонтитулов можно получить, используя следующие свойства:


	- [IDashboard.Headers](KeDashboard.chm::/Interface/IDashboard/IDashboard.Headers.htm);


	- [IPrxSheet.HeadersFooters](KeReport.chm::/Interface/IPrxSheet/IPrxSheet.HeadersFooters.htm).


## Свойства


		 Имя свойства

		 [DifferentFirstPageHeaderFooter](IHeaders.DifferentFirstPageHeaderFooter.htm)

		 определяет признак использования особого колонтитула для первой

		 [EvenPageFooter](IHeaders.EvenPageFooter.htm)
		 Свойство EvenPageFooter
		 возвращает параметры нижнего колонтитула чётных страниц.


		 ![](../../Property_Image.gif)
		 [EvenPageHeader](IHeaders.EvenPageHeader.htm)
		 Свойство EvenPageHeader
		 возвращает параметры верхнего колонтитула чётных страниц.


		 ![](../../Property_Image.gif)
		 [FirstPageFooter](IHeaders.FirstPageFooter.htm)
		 Свойство FirstPageFooter
		 возвращает параметры нижнего колонтитула первой страницы.


		 ![](../../Property_Image.gif)
		 [FirstPageHeader](IHeaders.FirstPageHeader.htm)
		 Свойство FirstPageHeader
		 возвращает параметры верхнего колонтитула первой страницы.


		 ![](../../Property_Image.gif)
		 [Footer](IHeaders.Footer.htm)
		 Свойство Footer возвращает
		 параметры нижнего колонтитула.


		 ![](../../Property_Image.gif)
		 [Header](IHeaders.Header.htm)
		 Свойство Header возвращает
		 параметры верхнего колонтитула.


		 ![](../../Property_Image.gif)
		 [OddAndEvenPagesHeaderFooter](IHeaders.OddAndEvenPagesHeaderFooter.htm)
		 Свойство OddAndEvenPagesHeaderFooter
		 определяет признак использования разных колонтитулов для чётных
		 и нечётных страниц.
