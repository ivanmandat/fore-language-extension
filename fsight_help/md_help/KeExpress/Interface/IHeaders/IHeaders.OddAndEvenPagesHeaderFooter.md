# IHeaders.OddAndEvenPagesHeaderFooter


## Синтаксис


OddAndEvenPagesHeaderFooter: Boolean;


## Описание


Свойство OddAndEvenPagesHeaderFooter
 определяет признак использования разных колонтитулов для чётных и нечётных
 страниц.


## Комментарии


Допустимые значения:


	- True. Для чётных страниц
	 будут использованы особые настройки колонтитулов;


	- False. Для чётных страниц
	 будут использованы стандартные настройки колонтитулов.
