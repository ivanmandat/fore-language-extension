# IEaxDataAreaSliceParams.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 параметров аналитической области данных.


## Комментарии


Для получения параметра аналитической области
 данных по его индексу используйте свойство [IEaxDataAreaSliceParams.Item](IEaxDataAreaSliceParams.Item.htm).
