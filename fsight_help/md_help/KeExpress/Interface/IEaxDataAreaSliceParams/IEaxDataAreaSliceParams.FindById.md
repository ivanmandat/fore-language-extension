# IEaxDataAreaSliceParams.FindById


## Синтаксис


FindById(Id: String): [IEaxDataAreaParam](../IEaxDataAreaParam/IEaxDataAreaParam.htm);


## Параметры


Id. Идентификатор параметра
 аналитической области данных.


## Описание


Метод FindById выполняет поиск
 параметра аналитической области данных по его идентификатору.


## Комментарии


Для получения количества параметров аналитической
 области данных в коллекции используйте свойство [IEaxDataAreaSliceParams.Count](IEaxDataAreaSliceParams.Count.htm).
