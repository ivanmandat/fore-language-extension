# IEaxDataAreaSliceParams.Item


## Синтаксис


Item(Index: Integer): [IEaxDataAreaParam](../IEaxDataAreaParam/IEaxDataAreaParam.htm);


## Параметры


Index. Индекс параметра аналитической
 области данных.


## Описание


Свойство Item возвращает параметр
 аналитической области данных по его индексу.


## Комментарии


Для получения индекса указанного параметра
 аналитической области данных используйте метод [IEaxDataAreaSliceParams.IndexOf](IEaxDataAreaSliceParams.IndexOf.htm).
