# IEaxDataAreaSliceParams.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IEaxDataAreaParam](../IEaxDataAreaParam/IEaxDataAreaParam.htm);


## Параметры


Key. Ключ параметра аналитической
 области данных.


## Описание


Метод FindByKey выполняет поиск
 параметра аналитической области данных по его ключу.


## Комментарии


Если параметр с указанным ключом не найден, то метод возвращает значение
 Null.
