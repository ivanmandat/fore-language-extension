# IEaxDataAreaSliceParams


Сборка: Express;


## Описание


Интерфейс IEaxDataAreaSliceParams
 предназначен для работы с параметрами среза аналитической области данных.


## Иерархия наследования


           IEaxDataAreaSliceParams


## Комментарии


Для работы с коллекцией параметров аналитической области данных используйте
 свойства и методы интерфейса [IEaxDataAreaParams](../IEaxDataAreaParams/IEaxDataAreaParams.htm).
