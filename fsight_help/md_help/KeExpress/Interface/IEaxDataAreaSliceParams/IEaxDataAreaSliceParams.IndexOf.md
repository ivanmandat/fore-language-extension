# IEaxDataAreaSliceParams.IndexOf


## Синтаксис

 Integer;

Param. Индекс параметра аналитической
 области данных.

Метод IndexOf возвращает
 индекс указанного параметра аналитической области данных.


## Комментарии


Параметр аналитической области данных передается
 посредством параметра Param.
