# ILanerRevisionLegend.SetRevisions


## Синтаксис


SetRevisions(FirstRevision: Integer; LastRevision:
 Integer);


## Параметры


FirstRevision. Ключ первой
 ревизии диапазона;


LastRevision. Ключ последней
 ревизии диапазона.


## Описание


Метод SetRevisions устанавливает
 диапазон отображаемых в легенде ревизий.
