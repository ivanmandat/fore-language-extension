# ILanerRevisionLegend


Сборка: Express;


## Описание


Интерфейс ILanerRevisionLegend
 содержит свойства и методы для работы с легендой ревизий для рядного режима
 базы данных временных рядов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Color](ILanerRevisionLegend.Color.htm)
		 Свойство Color определяет
		 цвет ячейки для заданной ревизии.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Clear](ILanerRevisionLegend.Clear.htm)
		 Метод Clear очищает
		 диапазон отображаемых в легенде ревизий.


		 ![](../../Sub_Image.gif)
		 [ClearUserColors](ILanerRevisionLegend.ClearUserColors.htm)
		 Метод ClearUserColors
		 сбрасывает пользовательские настройки цветов для ревизий.


		 ![](../../Sub_Image.gif)
		 [SetRevisions](ILanerRevisionLegend.SetRevisions.htm)
		 Метод SetRevisions
		 устанавливает диапазон отображаемых в легенде ревизий.
