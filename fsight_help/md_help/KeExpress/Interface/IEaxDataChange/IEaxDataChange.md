# IEaxDataChange


Сборка: Express;


## Описание


Интерфейс IEaxDataChange предназначен
 для работы с измененными данными таблицы.


## Комментарии


Интерфейс IEaxDataChange поддерживается
 следующими интерфейсами:


	- [IEaxAnalyzer](../IEaxAnalyzer/IEaxAnalyzer.htm);


	- [IEaxGrid](../IEaxGrid/IEaxGrid.htm).
