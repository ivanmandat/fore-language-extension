# IEaxDataAreaViews.Item


## Синтаксис


Item(Index: Integer): [IEaxObject](../IEaxObject/IEaxObject.htm);


## Параметры


Index. Индекс объекта для работы
 с представлением аналитической области данных.


## Описание


Свойство Item возвращает объект
 для работы с представлением аналитической области данных.


## Комментарии


Для получения количества представлений аналитической области данных
 используйте [IEaxDataAreaViews.Count](IEaxDataAreaViews.Count.htm).
