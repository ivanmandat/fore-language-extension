# IEaxDataAreaViews.RemoveByObject


## Синтаксис


RemoveByObject(Value: [IEaxObject](../IEaxObject/IEaxObject.htm)):
 Boolean;


## Параметры


Value. Объект для работы с
 представлением аналитической области данных.


## Описание


Метод RemoveByObject удаляет
 объект для работы с представлением аналитической области данных.


## Комментарии


Если объект будет удален, то метод вернет значение True.


Для создания нового объекта для работы с представлением аналитической
 области данных используйте [IEaxDataAreaViews.AddByType](IEaxDataAreaViews.AddByType.htm).
