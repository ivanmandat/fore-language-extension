# IEaxDataAreaViews.AddByType


## Синтаксис


AddByType(Type: [EaxObjectType](../../Enums/EaxObjectType.htm)):
 [IEaxObject](../IEaxObject/IEaxObject.htm);


## Параметры


Type. Тип создаваемого объекта
 для работы с представлением аналитической области данных.


## Описание


Метод AddByType добавляет в
 коллекцию новый объект для работы с представлением аналитической области
 данных.


## Комментарии


Для удаления объекта для работы с представлением аналитической области
 данных используйте [IEaxDataAreaViews.RemoveByObject](IEaxDataAreaViews.RemoveByObject.htm).


## Пример


Использование метода приведено в примере для [IEaxDataAreaSlices.Add](../IEaxDataAreaSlices/IEaxDataAreaSlices.Add.htm).
