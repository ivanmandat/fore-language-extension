# IEaxDataAreaViews.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс объекта для работы
 с представлением аналитической области данных.


## Описание


Метод Remove удаляет объект
 для работы с представлением аналитической области данных по его индексу.


## Комментарии


Если объект будет удален, то метод вернет значение True.


Для добавления в коллекцию существующего объекта для работы с представлением
 аналитической области данных используйте [IEaxDataAreaViews.Add](IEaxDataAreaViews.Add.htm).
