# IEaxDataAreaViews.FindById


## Синтаксис


FindById(Id: String): [IEaxObject](../IEaxObject/IEaxObject.htm);


## Параметры


Id. Идентификатор объекта.


## Описание


Метод FindById осуществляет
 поиск объекта для работы с представлением аналитической области данных
 по идентификатору.


## Комментарии


Если представление не найдено, то метод вернет значение Null.


Для поиска объекта для работы с представлением аналитической области
 данных по ключу используйте [IEaxDataAreaViews.FindByKey](IEaxDataAreaViews.FindByKey.htm).
