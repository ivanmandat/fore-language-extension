# IEaxDataAreaViews.Slice


## Синтаксис


Slice: [IEaxDataAreaSlice](../IEaxDataAreaSlice/IEaxDataAreaSlice.htm);


## Описание


Свойство Slice возвращает объект
 родитель коллекции представлений аналитической области данных.


## Комментарии


Объектом-родителем является срез аналитической области данных.


Срез данных - это двумерная таблица данных, полученная путём фиксации
 одного или нескольких справочников.
