# IEaxDataAreaViews.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 представлений аналитической области данных.


## Комментарии


Для получения объекта для работы с представлением аналитической области
 данных используйте [IEaxDataAreaViews.Item](IEaxDataAreaViews.Item.htm).
