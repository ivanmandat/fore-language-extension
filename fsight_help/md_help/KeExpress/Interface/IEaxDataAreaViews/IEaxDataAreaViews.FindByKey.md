# IEaxDataAreaViews.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IEaxObject](../IEaxObject/IEaxObject.htm);


## Параметры


Key. Ключ объекта.


## Описание


Метод FindByKey осуществляет
 поиск объекта для работы с представлением аналитической области данных
 по ключу.


## Комментарии


Если представление не найдено, то метод вернет значение Null.


Для поиска объекта для работы с представлением аналитической области
 данных по идентификатору используйте [IEaxDataAreaViews.FindById](IEaxDataAreaViews.FindById.htm).
