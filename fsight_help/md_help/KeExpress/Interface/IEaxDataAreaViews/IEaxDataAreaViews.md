# IEaxDataAreaViews


Сборка: Express;


## Описание


Интерфейс IEaxDataAreaViews
 используется для работы с коллекцией представлений аналитической области
 данных.


## Иерархия наследования


           IEaxDataAreaViews


## Комментарии


Для получения объекта данного типа используйте [IEaxDataArea.Views](../IEaxDataArea/IEaxDataArea.Views.htm).
