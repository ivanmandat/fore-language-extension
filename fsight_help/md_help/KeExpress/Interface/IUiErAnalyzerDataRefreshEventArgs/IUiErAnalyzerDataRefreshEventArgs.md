# IUiErAnalyzerDataRefreshEventArgs


Сборка: Express;


## Описание


Интерфейс IUiErAnalyzerDataRefreshEventArgs
 содержит свойства аргумента события компонента [UiErAnalyzer](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiErAnalyzer.htm),
 происходящего перед обновлением источников данных экспресс отчета.
