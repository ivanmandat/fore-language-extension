# IUiErAnalyzerDataRefreshEventArgs.SaveChangedData


## Синтаксис


SaveChangedData: Boolean;


## Описание


Свойство SaveChangedData определяет,
 сохранять измененные данные перед обновление источника данных экспресс-отчета
 или нет.
