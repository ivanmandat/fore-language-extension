# IEaxDataAreaCellStyle


Сборка: Express;


## Описание


Интерфейс IEaxDataAreaCellStyle
 предназначен для работы со стилями оформления различных элементов аналитической
 области данных.


## Иерархия наследования


           IEaxDataAreaCellStyle


## Комментарии


Стиль оформления возвращают следующие свойства и методы:


	- [IEaxDataAreaHierarchy.ElementStyle](../IEaxDataAreaHierarchy/IEaxDataAreaHierarchy.ElementStyle.htm).
	 Стиль оформления ячеек элемента альтернативной иерархии;


	- [IEaxGridHeaderSettings.TotalStyle](../IEaxGridHeaderSettings/IEaxGridHeaderSettings.TotalStyle.htm).
	 Стиль оформления итогов заголовков;


	- [IEaxGridViewSettings.GetStyleSettings](../IEaxGridViewSettings/IEaxGridViewSettings.GetStyleSettings.htm).
	 Стиль оформления для указанного элемента заголовка.
