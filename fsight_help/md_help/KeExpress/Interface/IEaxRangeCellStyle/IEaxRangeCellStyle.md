# IEaxRangeCellStyle


Сборка: Express;


## Описание


Интерфейс IEaxRangeCellStyle
 предназначен для работы со стилем оформления области в таблице.


## Комментарии


Для работы с коллекцией стилей всей таблицы используйте интерфейс [IEaxRangeCellStyles](../IEaxRangeCellStyles/IEaxRangeCellStyles.htm).
