# IEaxRangeCellStyle.TabRange


## Синтаксис


TabRange: [ITabRange](TabSheet.chm::/Interface/ITabRange/ITabRange.htm);


## Описание


Свойство TabRange возвращает диапазон ячеек, к которым применен стиль.


## Комментарии


Если стиль не применен к диапазону ячеек, то TabRange возвращает значение Null.
