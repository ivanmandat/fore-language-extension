# IEaxInvisibleDimensions.Clear


## Синтаксис


Clear;


## Описание


Метод Clear осуществляет очистку
 списка невидимых измерений. После очистки все измерения становятся видимыми.


## Комментарии


Для получения количества невидимых измерений используйте свойство [IEaxInvisibleDimensions.Count](IEaxInvisibleDimensions.Count.htm).
