# IEaxInvisibleDimensions.Item


## Синтаксис


Item(Index: Integer): Integer;


## Параметры


Index. Индекс измерения в списке
 невидимых.


## Описание


Свойство Item возвращает ключ
 измерения, находящегося в списке невидимых.


## Комментарии


Для получения количества невидимых измерений используйте свойство [IEaxInvisibleDimensions.Count](IEaxInvisibleDimensions.Count.htm).
