# IEaxInvisibleDimensions.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 невидимых измерений экспресс-отчета.


## Комментарии


Для добавления измерения в невидимые используйте метод [IEaxInvisibleDimensions.Add](IEaxInvisibleDimensions.Add.htm).
