# IEaxInvisibleDimensions.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс невидимого измерения.


## Описание


Метод Remove осуществляет удаление
 измерения из списка невидимых. После этого измерение становится видимым.


## Комментарии


Для добавления измерения в невидимые используйте метод [IEaxInvisibleDimensions.Add](IEaxInvisibleDimensions.Add.htm).
