# IEaxInvisibleDimensions.FindByKey


## Синтаксис


FindByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ измерения, по которому
 осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск измерения.


## Комментарии


Возвращает True если измерение
 находится в списке невидимых. Ключ измерения, по которому осуществляется
 поиск, передается посредством параметра Key.
