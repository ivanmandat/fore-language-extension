# IEaxInvisibleDimensions.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ невидимого измерения.


## Описание


Метод RemoveByKey осуществляет
 удаление измерения из списка невидимых. После этого измерение становится
 видимым.


## Комментарии


Для добавления измерения в невидимые используйте метод [IEaxInvisibleDimensions.Add](IEaxInvisibleDimensions.Add.htm).
