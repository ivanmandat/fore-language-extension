# IEaxInvisibleDimensions


Сборка: Express;


## Описание


Интерфейс IEaxInvisibleDimensions
 содержит свойства и методы коллекции невидимых измерений экспресс-отчёта.


## Комментарии


Отметки по невидимым измерениям не будут экспортироваться и отображаться
 в заголовке, колонтитулах экспресс-отчёта. Коллекцию невидимых измерений
 можно получить, используя свойство [IEaxDataAreaPivotSlice.InvisibleDimensions](../IEaxDataAreaPivotSlice/IEaxDataAreaPivotSlice.InvisibleDimensions.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IEaxInvisibleDimensions.Count.htm)
		 Свойство Count возвращает
		 количество невидимых измерений экспресс-отчёта.


		 ![](../../Property_Image.gif)
		 [Item](IEaxInvisibleDimensions.Item.htm)
		 Свойство Item возвращает
		 ключ измерения, находящегося в списке невидимых.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IEaxInvisibleDimensions.Add.htm)
		 Метод Add осуществляет
		 добавление измерения в список невидимых.


		 ![](../../Sub_Image.gif)

		 Метод Clear осуществляет

		 ![](../../Sub_Image.gif)
		 [FindByKey](IEaxInvisibleDimensions.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск измерения по ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](IEaxInvisibleDimensions.Remove.htm)
		 Метод Remove осуществляет
		 удаление измерения из списка невидимых по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IEaxInvisibleDimensions.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление измерения из списка невидимых по ключу.
