# IEaxWorkbookStatistics


Сборка: Express;


## Описание


Интерфейс IEaxWorkbookStatistics
 используется для работы с параметрами [панели статистик](UiDw.chm::/Workbook/UiDw_Description.htm).


## Иерархия наследования


           [IEaxWorkbookObject](../IEaxWorkbookObject/IEaxWorkbookObject.htm)


           IEaxWorkbookStatistics


## Комментарии


Данная панель расположена в рабочей книге базы данных временных рядов.
 Для получения параметров панели используйте свойство [IEaxWorkbook.Statistics](../IEaxWorkbook/IEaxWorkbook.Statistics.htm).
