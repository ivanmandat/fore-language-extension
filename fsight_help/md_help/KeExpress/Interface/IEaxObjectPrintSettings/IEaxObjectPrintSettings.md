# IEaxObjectPrintSettings


Сборка: Express;


## Описание


Интерфейс IEaxObjectPrintSettings
 содержит свойства, определяющие параметры печати объектов экспресс-отчета.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BlackAndWhite](IEaxObjectPrintSettings.BlackAndWhite.htm)
		 Свойство BlackAndWhite
		 определяет признак черно-белой печати объекта.


		 ![](../../Property_Image.gif)
		 [Scale](IEaxObjectPrintSettings.Scale.htm)
		 Свойство Scale определяет
		 масштаб объекта при печати/экспорте.
