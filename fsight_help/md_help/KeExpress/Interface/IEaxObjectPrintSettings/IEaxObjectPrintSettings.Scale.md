# IEaxObjectPrintSettings.Scale


## Синтаксис


Scale: Double;


## Описание


Свойство Scale определяет масштаб
 объекта при печати/экспорте.


## Комментарии


Значение устанавливается от 0,1 до 5 (10-500%). Также зарезервированы
 следующие значения: «0» - разместить на одну страницу, «-1» - подобрать
 по ширине страницы, «-2» - подобрать по высоте страницы.
