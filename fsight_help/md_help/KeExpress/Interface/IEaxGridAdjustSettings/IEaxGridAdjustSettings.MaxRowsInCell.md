# IEaxGridAdjustSettings.MaxRowsInCell


## Синтаксис


MaxRowsInCell: Integer;


## Описание


Свойство MaxRowsInCell определяет максимальное количество строк в ячейке.


## Комментарии


Значение по умолчанию - «8».


MaxRowsInCell учитывается, если [IEaxGridAdjustSettings.Mode](IEaxGridAdjustSettings.Mode.htm) имеет значение [EaxAdjustMode.Auto](../../Enums/EaxAdjustMode.htm).
