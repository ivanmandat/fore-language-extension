# IEaxGridAdjustSettings


Сборка: Express;


## Описание


Интерфейс IEaxGridAdjustSettings
 предназначен для работы с параметрами автоподбора размера ячеек.


## Комментарии


Используя данные настройки, можно осуществить автоподбор ширины, ориентированный
 на гарантированное отображение столбцов области данных при большом количестве
 атрибутов, выведенных в шапке таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AllowAdjust](IEaxGridAdjustSettings.AllowAdjust.htm)
		 Свойство AllowAdjust
		 определяет, будет ли производиться автоподбор размера ячеек.


		 ![](../../Property_Image.gif)
		 [MaxRowsInCell](IEaxGridAdjustSettings.MaxRowsInCell.htm)
		 Свойство MaxRowsInCell
		 определяет максимальное количество строк в ячейке.


		 ![](../../Property_Image.gif)
		 [MinVisibleDataColumns](IEaxGridAdjustSettings.MinVisibleDataColumns.htm)
		 Свойство MinVisibleDataColumns
		 определяет минимальное количество видимых столбцов области данных.


		 ![](../../Property_Image.gif)
		 [Mode](IEaxGridAdjustSettings.Mode.htm)
		 Свойство Mode определяет
		 режим автоподбора размера ячеек.
