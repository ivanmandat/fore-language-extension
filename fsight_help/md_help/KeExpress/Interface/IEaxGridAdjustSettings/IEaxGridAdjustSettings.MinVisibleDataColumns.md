# IEaxGridAdjustSettings.MinVisibleDataColumns


## Синтаксис


MinVisibleDataColumns: Integer;


## Описание


Свойство MinVisibleDataColumns определяет минимальное количество видимых столбцов области данных.


## Комментарии


Значение по умолчанию - «3».


MinVisibleDataColumns учитывается, если [IEaxGridAdjustSettings.Mode](IEaxGridAdjustSettings.Mode.htm) имеет значение [EaxAdjustMode.Auto](../../Enums/EaxAdjustMode.htm).
