# IEaxGridAdjustSettings.Mode


## Синтаксис


Mode: [EaxAdjustMode](../../Enums/EaxAdjustMode.htm);


## Описание


Свойство Mode определяет режим автоподбора размера ячеек.


## Комментарии


Если установлен автоматический режим автоподбора размера ячеек, то его параметры необходимо задать через свойства [IEaxGridAdjustSettings.MaxRowsInCell](IEaxGridAdjustSettings.MaxRowsInCell.htm) и [IEaxGridAdjustSettings.MinVisibleDataColumns](IEaxGridAdjustSettings.MinVisibleDataColumns.htm).
