# IEaxDataAreaParams.Move


## Синтаксис


Move(OldIndex: Integer; NewIndex: Integer);


## Параметры


OldIndex. Индекс позиции параметра,
 который необходимо переместить;


NewIndex. Индекс позиции, в
 которую необходимо переместить параметр.


## Описание


Метод Move перемещает параметр
 аналитической области данных в указанную позицию.


## Комментарии


Для очистки коллекции параметров аналитической области данных используйте
 метод [IEaxDataAreaParams.Clear](IEaxDataAreaParams.Clear.htm).
