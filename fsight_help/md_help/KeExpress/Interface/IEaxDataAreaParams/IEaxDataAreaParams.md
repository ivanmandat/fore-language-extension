# IEaxDataAreaParams


Сборка: Express;


## Описание


Интерфейс IEaxDataAreaParams
 предназначен для работы с коллекцией параметров аналитической области
 данных.


## Иерархия наследования


           [IEaxDataAreaSliceParams](../IEaxDataAreaSliceParams/IEaxDataAreaSliceParams.htm)


           IEaxDataAreaParams


## Комментарии


Для получения коллекции параметров используйте свойство [IEaxDataAreaSlice.Params](../IEaxDataAreaSlice/IEaxDataAreaSlice.Params.htm).
