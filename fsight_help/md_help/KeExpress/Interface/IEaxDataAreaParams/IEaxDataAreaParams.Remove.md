# IEaxDataAreaParams.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс параметра аналитической
 области данных.


## Описание


Метод Remove удаляет параметр
 аналитической области данных из коллекции по его индексу.


## Комментарии


Метод возвращает значение True
 в случае удачного удаления параметра аналитической области данных из коллекции.
