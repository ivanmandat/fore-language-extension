# IEaxGridHeaderSettings.HierarchyIndent


## Синтаксис


HierarchyIndent: Double;


## Описание


Свойство HierarchyIndent определяет
 отступ в раскрывающейся иерархии.


## Комментарии


Значение указывается в миллиметрах.
