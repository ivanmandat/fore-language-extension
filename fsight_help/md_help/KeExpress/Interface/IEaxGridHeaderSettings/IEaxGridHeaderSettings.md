# IEaxGridHeaderSettings


Сборка: Express;


## Описание


Интерфейс IEaxGridHeaderSettings
 предназначен для работы с настройками заголовка.


## Иерархия наследования


           IEaxGridHeaderSettings


## Комментарии


Для получения настроек заголовка приведите результат свойства [IEaxGridViewSettings.GetViewSettings](../IEaxGridViewSettings/IEaxGridViewSettings.GetViewSettings.htm)
 к интерфейсу IEaxGridHeaderSettings.


Используя данные настройки, можно изменить размещение заголовка, поведение
 строк/столбцов, настроить видимость значков сортировки и отступы в раскрывающейся
 иерархии.
