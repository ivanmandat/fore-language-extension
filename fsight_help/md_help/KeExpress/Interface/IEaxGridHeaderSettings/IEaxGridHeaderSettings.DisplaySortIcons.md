# IEaxGridHeaderSettings.DisplaySortIcons


## Синтаксис


DisplaySortIcons: Boolean;


## Описание


Свойство DisplaySortIcons определяет,
 будут ли отображаться [значки
 сортировки](UiAnalyticalArea.chm::/TableView/Layout/Layout.htm).


## Комментарии


Допустимые значения:


	- True. Значки сортировки
	 данных отображаются;


	- False. Значки сортировки
	 данных не отображаются.


В зависимости от того, в каком инструменте продукта «Форсайт. Аналитическая платформа»
 осуществляется настройка заголовка, значение свойства DisplaySortIcons может
 иметь разные значения по умолчанию.
