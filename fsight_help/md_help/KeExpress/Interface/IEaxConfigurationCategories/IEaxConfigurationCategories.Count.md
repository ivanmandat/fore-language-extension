# IEaxConfigurationCategories.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество групп вкладок.


## Комментарии


По умолчанию свойство Count имеет значение 3, и учитывает 3 системные группы вкладок, реализованные для экспресс-отчетов.
