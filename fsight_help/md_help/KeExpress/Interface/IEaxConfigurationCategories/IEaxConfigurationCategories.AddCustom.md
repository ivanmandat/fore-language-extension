# IEaxConfigurationCategories.AddCustom


## Синтаксис


AddCustom(Category: [IEaxCustomConfigurationCategory](../IEaxCustomConfigurationCategory/IEaxCustomConfigurationCategory.htm));


## Параметры


Category. Группа вкладок.


## Описание


Метод AddCustom добавляет пользовательскую группу вкладок.
