# IEaxConfigurationCategories.CustomCategoriesCount


## Синтаксис


CustomCategoriesCount: Integer;


## Описание


Свойство CustomCategoriesCount
 возвращает количество групп вкладок не принадлежащих к системной группе,
 реализованной для экспресс-отчетов.
