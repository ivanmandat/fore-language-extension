# IEaxConfigurationCategories.RemoveByIndex


## Синтаксис


RemoveByIndex(Index: Integer);


## Параметры


Index. Индекс группы вкладок.


## Описание


Метод RemoveByIndex удаляет указанную группу вкладок, индекс которой передается посредством параметра Index.


## Комментарии


Боковая панель по умолчанию содержит три системные группы вкладок, реализованные для экспресс-отчетов, поэтому параметр Index может находиться только в пределах от 3 до ([Count](IEaxConfigurationCategories.Count.htm) -1).


Системные группы вкладок удалить нельзя.
