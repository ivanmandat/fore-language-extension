# IEaxConfigurationCategories


Сборка: Express;


## Описание


Интерфейс IEaxConfigurationCategories
 содержит свойства и методы коллекции групп вкладок боковой панели экспресс-отчета.


## Иерархия наследования


           IEaxConfigurationCategories


## Комментарии


Для работы с коллекцией групп используйте свойство [IEaxConfigurationPanel.Categories](../IEaxConfigurationPanel/IEaxConfigurationPanel.Categories.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IEaxConfigurationCategories.Count.htm)
		 Свойство Count возвращает
		 количество групп вкладок.


		 ![](../../Property_Image.gif)
		 [CustomCategoriesCount](IEaxConfigurationCategories.CustomCategoriesCount.htm)
		 Свойство CustomCategoriesCount
		 возвращает количество групп вкладок не принадлежащих к системной
		 группе, реализованной для экспресс-отчетов.


		 ![](../../Property_Image.gif)
		 [Item](IEaxConfigurationCategories.Item.htm)
		 Свойство Item возвращает
		 параметры указанной группы вкладок.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddCustom](IEaxConfigurationCategories.AddCustom.htm)
		 Метод AddCustom добавляет
		 пользовательскую группу вкладок.


		 ![](../../Sub_Image.gif)
		 [ClearCustom](IEaxConfigurationCategories.ClearCustom.htm)
		 Метод ClearCustom удаляет
		 все группы вкладок.


		 ![](../../Sub_Image.gif)
		 [InsertCustom](IEaxConfigurationCategories.InsertCustom.htm)
		 Метод InsertCustom
		 осуществляет вставку указанной группы в указанную позицию в коллекции.


		 ![](../../Sub_Image.gif)
		 [RemoveByIndex](IEaxConfigurationCategories.RemoveByIndex.htm)
		 Метод RemoveByIndex
		 удаляет указанную группу вкладок, индекс которой передается посредством
		 параметра Index.


		 ![](../../Sub_Image.gif)
		 [RemoveCustom](IEaxConfigurationCategories.RemoveCustom.htm)
		 Метод RemoveCustom
		 удаляет указанную группу вкладок.
