# IEaxSyncGroups.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс связи синхронизированных
 измерений в коллекции.


## Описание


Метод Remove удаляет связь синхронизированных
 измерений по ее индексу.


## Комментарии


Если связь синхронизированных измерений будет удалена, то метод вернет
 значение True.


Для создания связи синхронизированных измерений используйте [IEaxSyncGroups.CreateSyncGroup](IEaxSyncGroups.CreateSyncGroup.htm).
