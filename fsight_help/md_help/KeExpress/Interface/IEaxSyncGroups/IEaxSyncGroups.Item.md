# IEaxSyncGroups.Item


## Синтаксис


Item(Index: Integer): [IEaxSyncGroup](../IEaxSyncGroup/IEaxSyncGroup.htm);


## Параметры


Index. Индекс связи синхронизированных
 измерений.


## Описание


Свойство Item возвращает связь
 синхронизированных измерений по ее индексу.


## Комментарии


Для получения количества связей синхронизированных измерений используйте
 [IEaxSyncGroups.Count](IEaxSyncGroups.Count.htm).
