# IEaxSyncGroups.InGroup


## Синтаксис


InGroup(SyncItem: [IEaxDimensionDescriptor](../IEaxDimensionDescriptor/IEaxDimensionDescriptor.htm)):
 [IEaxSyncGroup](../IEaxSyncGroup/IEaxSyncGroup.htm);


## Параметры


SyncItem. Описание измерения.


## Описание


Метод InGroup возвращает связь
 синхронизированных измерений по указанному описанию измерения.


## Комментарии


Если по указанному описанию связь не будет найдена, то метод вернет
 значение Null.
