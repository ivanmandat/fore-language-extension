# IEaxSyncGroups


Сборка: Express;


## Описание


Интерфейс IEaxSyncGroups используется
 для работы с коллекцией связей [синхронизированных
 измерений](UiReport.chm::/Desktop/Source/Dim/Symc_dimensions.htm).


## Иерархия наследования


           IEaxSyncGroups


## Комментарии


Для работы со связью [синхронизированных
 измерений](UiReport.chm::/Desktop/Source/Dim/Symc_dimensions.htm) используйте [IEaxSyncGroup](../IEaxSyncGroup/IEaxSyncGroup.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IEaxSyncGroups.Count.htm)
		 Свойство Count определяет
		 количество связей синхронизированных измерений.


		 ![](../../Property_Image.gif)
		 [Item](IEaxSyncGroups.Item.htm)
		 Свойство Item возвращает
		 связь синхронизированных измерений по ее индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CreateGroup](IEaxSyncGroups.CreateSyncGroup.htm)
		 Метод CreateGroup создает
		 связь синхронизированных измерений.


		 ![](../../Sub_Image.gif)
		 [GetCompatibleItems](IEaxSyncGroups.GetCompatibleItems.htm)
		 Метод GetCompatibleItems
		 получает коллекцию описаний измерений, к которым подходит указанное
		 описание измерения.


		 ![](../../Sub_Image.gif)
		 [InGroup](IEaxSyncGroups.InGroup.htm)
		 Метод InGroup возвращает
		 связь синхронизированных измерений по указанному описанию измерения.


		 ![](../../Sub_Image.gif)
		 [IsDimensionsCompatible](IEaxSyncGroups.IsDimensions.htm)
		 Метод IsDimensionsCompatible
		 возвращает признак возможности установления связи для указанных
		 измерений.


		 ![](../../Sub_Image.gif)
		 [Remove](IEaxSyncGroups.Remove.htm)
		 Метод Remove удаляет
		 связь синхронизированных измерений по ее индексу.
