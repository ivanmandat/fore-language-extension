# IEaxSyncGroups.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count определяет количество
 связей синхронизированных измерений в коллекции.


## Комментарии


Для получения связи синхронизированных измерений по ее индексу используйте
 [IEaxSyncGroups.Item](IEaxSyncGroups.Item.htm).
