# IEaxAnalyzeCore.UpdateParamProvider


## Синтаксис


UpdateParamProvider;


## Описание


Метод UpdateParamProvider обновляет
 источник данных для параметров экспресс-отчета.


## Комментарии


Данный метод используется после редактирования параметров источника
 данных.


Для получения объекта для работы с параметрами экспресс-отчета используйте
 [IEaxAnalyzeCore.ParamProvider](IEaxAnalyzeCore.ParamProvider.htm).
