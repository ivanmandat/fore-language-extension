# IEaxAnalyzeCore.Sheets


## Синтаксис


Sheets: [IEaxSheets](../IEaxSheets/IEaxSheets.htm);


## Описание


Свойство Sheets возвращает коллекцию
 дочерних листов экспресс-отчета.


## Комментарии


Каждый лист представляет собой отдельный экспресс-отчет, имеющий собственные
 источники данных и настройки объектов: таблицы, диаграммы, карты, матрицы.


По умолчанию коллекция листов пуста. Если в коллекции создать один лист,
 то он будет соответствовать исходному экспресс-отчету. При добавлении
 в коллекцию Sheets более одного
 листа, для исходного экспресс-отчета будут созданы дочерние объекты. Если
 лист основан на существующем экспресс-отчете, то дочерним объектом будет
 являться копия использованного экспресс-отчета. Если лист основан на новом
 экспресс-отчете, то он и будет являться дочерним объектом. При этом в
 коллекцию также будет включена копия исходного экспресс-отчета, которому
 принадлежит коллекция Sheets.
