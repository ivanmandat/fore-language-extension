# IEaxAnalyzeCore.Printable


## Синтаксис


Printable: Boolean;


## Описание


Свойство Printable определяет,
 будет ли экспресс-отчет печататься на принтере.


## Комментарии


Данное свойство актуально, если на базе текущего экспресс-отчета создан
 [лист](../IEaxSheet/IEaxSheet.htm) в каком-либо другом экспресс-отчете
 и производится пакетная печать всей [коллекции
 листов](IEaxAnalyzeCore.Sheets.htm). По умолчанию свойству Printable
 установлено значение True, при
 этом экспресс-отчет будет напечатан. Для отмены печати установите свойству
 Printable значение True.
