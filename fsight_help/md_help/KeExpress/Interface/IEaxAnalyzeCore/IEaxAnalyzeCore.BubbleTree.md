# IEaxAnalyzeCore.BubbleTree


## Синтаксис

## Описание


Свойство BubbleTree возвращает
 объект, содержащий пузырьковое дерево экспресс-отчета.


## Комментарии


Для получения объекта, содержащего плоское дерево используйте свойство
 [IEaxAnalyzeCore.TreeMap](IEaxAnalyzeCore.TreeMap.htm).
