# IEaxAnalyzeCore


Сборка: Express;


## Описание


Интерфейс IEaxAnalyzeCore предназначен
 для работы с экспресс-отчетом.


## Иерархия наследования


IEaxAnalyzeCore

## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [BeginUpdate](IEaxAnalyzeCore.BeginUpdate.htm)


		 Метод BeginUpdate отключает
		 обновление объектов экспресс-отчета.


		 ![](../../Sub_Image.gif)
		 [CanDimBeTimeLine](IEaxAnalyzeCore.CanDimBeTimeLine.htm)


		 Метод CanDimBeTimeLine
		 возвращает признак возможности использования измерения в качестве
		 календарного в отчете и в качестве временной линии карты и матрицы.


		 ![](../../Sub_Image.gif)
		 [ClearDataSourceCache](IEaxAnalyzeCore.ClearDataSourceCache.htm)


		 Метод ClearDataSourceCache
		 очищает объект кеширования источника данных экспресс-отчета.


		 ![](../../Sub_Image.gif)
		 [CorrectScale](IEaxAnalyzeCore.CorrectScale.htm)


		 Метод CorrectScale
		 осуществляет корректировку масштаба относительно размера отображаемой
		 области экспресс-отчета.


		 ![](../../Sub_Image.gif)
		 [CreateDataSourceCache](IEaxAnalyzeCore.CreateDataSourceCache.htm)


		 Метод CreateDataSourceCache
		 создает объект кеширования источника данных экспресс-отчета и
		 возвращает результат его создания.


		 ![](../../Sub_Image.gif)
		 [CreateMultiOlap](IEaxAnalyzeCore.CreateMultiOlap.htm)


		 Метод CreateMultiOlap
		 создает экспресс-отчет, содержащий несколько источников данных.


		 ![](../../Sub_Image.gif)
		 [DrillPoint](IEaxAnalyzeCore.DrillPoint.htm)


		 Метод DrillPoint возвращает
		 информацию о привязке точки ряда данных к данным.


		 ![](../../Sub_Image.gif)
		 [DrillSerie](IEaxAnalyzeCore.DrillSerie.htm)


		 Метод DrillSerie возвращает
		 информацию о привязке ряда к данным.


		 ![](../../Sub_Image.gif)
		 [EndUpdate](IEaxAnalyzeCore.EndUpdate.htm)


		 Метод EndUpdate возобновляет
		 обновление объектов экспресс-отчета.


		 ![](../../Sub_Image.gif)
		 [ExcludeChartElements](IEaxAnalyzeCore.ExcludeChartElements.htm)


		 Метод ExcludeChartElements
		 исключает из отметки указанные элементы измерений.


		 ![](../../Sub_Image.gif)
		 [KeepOnlyChartElements](IEaxAnalyzeCore.KeepOnlyChartElements.htm)


		 Метод KeepOnlyChartElements
		 изменяет отметку таким образом, что доступными остаются только
		 указанные элементы диаграммы.


		 ![](../../Sub_Image.gif)
		 [LoadCubeFromStream](IEaxAnalyzeCore.LoadCubeFromStream.htm)


		 Метод LoadCubeFromStream
		 загружает данные куба в экспресс-отчет из потока.


		 ![](../../Sub_Image.gif)
		 [LoadSettingsFromXml](IEaxAnalyzeCore.LoadSettingsFromXml.htm)


		 Метод LoadSettingsFromXml
		 загружает параметры экспресс-отчёта из xml-документа.


		 ![](../../Sub_Image.gif)
		 [OpenCube](IEaxAnalyzeCore.OpenCube.htm)


		 Метод OpenCube открывает
		 куб, на базе которого строится экспресс-отчет.


		 ![](../../Sub_Image.gif)
		 [OpenCubeEx](IEaxAnalyzeCore.OpenCubeEx.htm)


		 Метод OpenCubeEx открывает
		 куб (с расширенными параметрами), на базе которого строится экспресс-отчет.


		 ![](../../Sub_Image.gif)
		 [OpenMatrixDataSource](IEaxAnalyzeCore.OpenMatrixDataSource.htm)


		 Метод OpenMatrixDataSource
		 осуществляет загрузку в экспресс-отчет данных источника, не являющегося
		 кубом.


		 ![](../../Sub_Image.gif)
		 [RefreshAll](IEaxAnalyzeCore.RefreshAll.htm)


		 Метод RefreshAll обновляет
		 все объекты экспресс-отчета.


		 ![](../../Sub_Image.gif)
		 [RefreshDataSourceCache](IEaxAnalyzeCore.RefreshDataSourceCache.htm)


		 Метод RefreshDataSourceCache
		 обновляет объект кеширования источника данных экспресс-отчета.


		 ![](../../Sub_Image.gif)
		 [SaveCubeToStream](IEaxAnalyzeCore.SaveCubeToStream.htm)


		 Метод SaveCubeToStream
		 сохраняет в поток данные отмеченных элементов измерений экспресс-отчета
		 в виде куба.


		 ![](../../Sub_Image.gif)
		 [SaveSettingsToXml](IEaxAnalyzeCore.SaveSettingsToXml.htm)


		 Метод SaveSettingsToXml
		 выгружает параметры экспресс-отчёта в формате xml.


		 ![](../../Sub_Image.gif)
		 [UpdateParamProvider](IEaxAnalyzeCore.UpdateParamProvider.htm)


		 Метод UpdateParamProvider
		 обновляет источник данных для параметров экспресс-отчета.
