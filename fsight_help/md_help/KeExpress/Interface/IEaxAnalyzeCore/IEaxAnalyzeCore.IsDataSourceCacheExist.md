# IEaxAnalyzeCore.IsDataSourceCacheExist


## Синтаксис


IsDataSourceCacheExist: Boolean;


## Описание


Свойство IsDataSourceCacheExist
 возвращает результат проверки источника экспресс-отчета на наличие у него
 объекта кеширования.


## Комментарии


Свойство возвращает:


	- False, если у источника
	 данных указанного экспресс-отчета нет созданного объекта кеширования;


	- True, если у источника
	 данных указанного экспресс-отчета создан объект кеширования.
