# IEaxAnalyzeCore.BeginUpdate


## Синтаксис


BeginUpdate;


## Описание


Метод BeginUpdate отключает
 обновление объектов экспресс-отчета.


## Комментарии


После вызова метода будет отключено любое обновление таблицы, диаграммы
 и карты экспресс-отчета. Также не будут генерироваться внутренние события
 данных объектов и экспресс-отчета. Данный метод актуально использовать
 при изменении большого количества параметров объектов экспресс-отчета
 для избежания их постоянного обновления, которое может занять длительное
 время. Применение всех изменений и возобновление обновления осуществляется
 после вызова метода [IEaxAnalyzeCore.EndUpdate](IEaxAnalyzeCore.EndUpdate.htm).
