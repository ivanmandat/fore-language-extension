# IEaxAnalyzeCore.ClearDataSourceCache


## Синтаксис


ClearDataSourceCache: Boolean;


## Описание


Метод ClearDataSourceCache очищает
 объект кеширования источника данных экспресс-отчета.


## Комментарии


Метод возвращает результат выполнения операции очищения объекта кеширования
 источника данных экспресс-отчета:


	- True. Операция выполнена
	 успешно;


	- False. Операция не выполнена.
