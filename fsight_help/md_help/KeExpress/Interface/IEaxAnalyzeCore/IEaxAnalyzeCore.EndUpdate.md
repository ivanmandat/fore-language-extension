# IEaxAnalyzeCore.EndUpdate


## Синтаксис


EndUpdate;


## Описание


Метод EndUpdate возобновляет
 обновление объектов экспресс-отчета.


## Комментарии


После вызова метода также будут применены все изменения, произведенные
 в экспресс-отчете между вызовами методов [IEaxAnalyzeCore.BeginUpdate](IEaxAnalyzeCore.BeginUpdate.htm)
 и EndUpdate.
