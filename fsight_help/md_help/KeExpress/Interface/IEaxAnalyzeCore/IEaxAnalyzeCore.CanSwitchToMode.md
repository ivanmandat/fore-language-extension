# IEaxAnalyzeCore.CanSwitchToMode


## Синтаксис


CanSwitchToMode:Boolean;


## Описание


Свойство CanSwitchToMode возвращает
 признак того, что доступна смена режима отображения рабочей книги.


## Комментарии


Допустимые значения:


	- True. Смена режима доступна;


	- False. Смена режима
	 недоступна.


Режим отображения рабочей книги определяет свойство [IEaxAnalyzeCore.Mode](IEaxAnalyzeCore.Mode.htm).
