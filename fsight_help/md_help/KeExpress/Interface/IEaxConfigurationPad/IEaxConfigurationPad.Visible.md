# IEaxConfigurationPad.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет видимость вкладки во время выполнения формы.


## Комментарии


Если свойству установлено значение True, то в области боковой панели будет отображен заголовок вкладки. При щелчке по заголовку вкладка будет развернута и будет доступна связанная с ней панель.
