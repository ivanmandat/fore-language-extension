# IEaxConfigurationPad.Content


## Синтаксис


Content: [IEaxConfigurationPadPanel](../IEaxConfigurationPadPanel/IEaxConfigurationPadPanel.htm);


## Описание


Свойство Content возвращает
 параметры панели, связанной с данной вкладкой.


## Комментарии


Содержимое панели отображается, если текущая вкладка является [активной](../IEaxCustomConfigurationCategory/IEaxCustomConfigurationCategory.ActivePad.htm).
 Панель является фреймом для размещения формы с элементами управления,
 которые будут использоваться при работе с вкладкой.
