# IEaxConfigurationPad


Сборка: Express;


## Описание


Интерфейс IEaxConfigurationPad
 содержит свойства и методы вкладки боковой панели экспресс-отчета.


## Иерархия наследования


           IEaxConfigurationPad


## Комментарии


Свойства вкладки могут быть получены в свойстве [IEaxCustomConfigurationCategory.ActivePad](../IEaxCustomConfigurationCategory/IEaxCustomConfigurationCategory.ActivePad.htm)
 или [IEaxConfigurationPads.Item](../IEaxConfigurationPads/IEaxConfigurationPads.Item.htm).
