# IEaxWorkbookMetaAttributesTree


Сборка: Express;


## Описание


Интерфейс IEaxWorkbookMetaAttributesTree
 используется для работы с параметрами [дерева рядов](UiDw.chm::/Purpose/UiDw_Purpose_Object.htm).


## Иерархия наследования


           [IEaxWorkbookObject](../IEaxWorkbookObject/IEaxWorkbookObject.htm)


           IEaxWorkbookMetaAttributesTree


## Комментарии


Дерево показателей расположено в рабочей книге базы данных временных
 рядов. Для получения параметров дерева используйте свойство [IEaxWorkbook.MetaAttributesTree](../IEaxWorkbook/IEaxWorkbook.MetaAttributesTree.htm).
