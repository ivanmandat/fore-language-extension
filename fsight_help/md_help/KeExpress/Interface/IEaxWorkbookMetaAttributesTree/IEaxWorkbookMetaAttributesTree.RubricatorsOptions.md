# IEaxWorkbookMetaAttributesTree.RubricatorsOptions


## Синтаксис


RubricatorsOptions: [IEaxWorkbookRubricatorOptionsList](../IEaxWorkbookRubricatorOptionsList/IEaxWorkbookRubricatorOptionsList.htm);


## Описание


Свойство RubricatorsOptions
 возвращает коллекцию настроек баз данных временных рядов, которые
 использует дерево показателей.


## Комментарии


Для получения настроек базы данных из коллекции используйте свойство
 [IEaxWorkbookRubricatorOptionsList.Item](../IEaxWorkbookRubricatorOptionsList/IEaxWorkbookRubricatorOptionsList.Item.htm).


## Пример

 настроек дерева объектов рабочей книги](../../Samples/KeExpress_EaxWorkbook_fore.htm)».
