# IEaxBeforeExportEventArgs.Format


## Синтаксис


Format: [IExportFormat](ModDrawing.chm::/Interface/IExportFormat/IExportFormat.htm);


## Описание


Свойство Format возвращает описание
 формата, в который производится экспорт.


## Комментарии


Форматами экспорта в экспресс-отчете являются:


	- *.xls - книга Excel 97-2003;


	- *.xlsx - книга Excel;


	- *.pdf - документ PDF;


	- *.rtf - документ в формате RTF;


	- *.pptx - презентация PowerPoint;


	- *.html - веб-страница;


	- *.mht - веб-страница, архив в одном файле;


	- *.emf - изображение;


	- *.ppcube - куб;


	- *.ppexpress - экспресс-отчёт.
