# IAlgorithmObjectParamControl.AttributeId


## Синтаксис


AttributeId: String;


## Описание


Свойство AttributeId определяет
 идентификатор атрибута, значения которого будут передаваться в параметр
 алгоритма расчёта.


## Комментарии


В качестве значения указывается идентификатор одного из атрибутов справочника,
 на работу с которым настроен параметр [IAlgorithmObjectParamControl.Param](IAlgorithmObjectParamControl.Param.htm).
