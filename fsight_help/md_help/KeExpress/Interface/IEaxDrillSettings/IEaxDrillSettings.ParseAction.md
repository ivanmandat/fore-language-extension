# IEaxDrillSettings.ParseAction


## Синтаксис


ParseAction(Element: Integer): String;


## Параметры


Element. Элемент справочника.


## Описание


Метод ParseAction выполняет
 разбор действия, в котором используется подстановка в виде атрибута измерения,
 и представляет его в текстовом виде.


## Комментарии


Для подготовки действия, в котором участвует подстановка в виде атрибута
 измерения, используйте метод [IEaxDrillSettings.PrepareAction](IEaxDrillSettings.PrepareAction.htm).
