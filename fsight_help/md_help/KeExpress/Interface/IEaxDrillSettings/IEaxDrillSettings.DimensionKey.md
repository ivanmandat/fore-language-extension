# IEaxDrillSettings.DimensionKey


## Синтаксис


DimensionKey: Integer;


## Описание


Свойство DimensionKey возвращает
 ключ измерения, использующего [расшифровку
 элементов измерения](UiAnalyticalArea.chm::/Hyperlinks/Working_with_hyperlinks.htm).


## Комментарии


Для получения ключа среза данных, использующего [расшифровку
 элементов измерения](UiAnalyticalArea.chm::/Hyperlinks/Working_with_hyperlinks.htm), воспользуйтесь [IEaxDrillSettings.SliceKey](IEaxDrillSettings.SliceKey.htm).
