# IEaxDrillSettings.Action


## Синтаксис


Action: String;


## Описание


Свойство Action определяет действие,
 выполняемое при щелчке по гиперссылке.


## Комментарии


Действие, выполняемое при щелчке по гиперссылке, зависит от типа действия,
 которое задается с помощью свойства [IEaxDrillSettings.ActionType](IEaxDrillSettings.ActionType.htm).


В тексте действия доступно использование [подстановок](UiAnalyticalArea.chm::/Hyperlinks/Hyperlinks_sub.htm).
