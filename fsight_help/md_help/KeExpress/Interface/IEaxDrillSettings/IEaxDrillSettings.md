# IEaxDrillSettings


Сборка: Express;


## Описание


Интерфейс IEaxDrillSettings
 используется для работы с настройками [расшифровки
 элементов измерения](UiAnalyticalArea.chm::/Hyperlinks/Working_with_hyperlinks.htm).


## Иерархия наследования


           IEaxDrillSettings


## Комментарии


Для получения объекта данного типа используйте [IEaxGridDimensionSettings.Drill](../IEaxGridDimensionSettings/IEaxGridDimensionSettings.Drill.htm).

		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Action](IEaxDrillSettings.Action.htm)
		 Свойство Action определяет
		 действие, выполняемое при щелчке по гиперссылке.


		 ![](../../Property_Image.gif)
		 [ActionType](IEaxDrillSettings.ActionType.htm)
		 Свойство ActionType

		 [Dimension](IEaxDrillSettings.Dimension.htm)
		 Свойство Dimension

		 ![](../../Property_Image.gif)
		 [DimensionAttribute](IEaxDrillSettings.DimensionAttribute.htm)
		 Свойство DimensionAttribute

		 элементов измерения](UiAnalyticalArea.chm::/Hyperlinks/Working_with_hyperlinks.htm).


		 ![](../../Property_Image.gif)
		 [DimensionAttributes](IEaxDrillSettings.DimensionAttributes.htm)
		 Свойство DimensionAttributes определяет
		 список атрибутов, по которым будет производиться поиск.


		 ![](../../Property_Image.gif)
		 [DimensionIndex](IEaxDrillSettings.DimensionIndex.htm)
		 Свойство DimensionIndex
		 определяет индекс для поиска элементов в [расшифровке
		 элементов измерения](UiAnalyticalArea.chm::/Hyperlinks/Working_with_hyperlinks.htm).


		 ![](../../Property_Image.gif)
		 [DimensionKey](IEaxDrillSettings.DimensionKey.htm)
		 Свойство DimensionKey
		 возвращает ключ измерения, использующего [расшифровку
		 элементов измерения](UiAnalyticalArea.chm::/Hyperlinks/Working_with_hyperlinks.htm).


		 ![](../../Property_Image.gif)
		 [DimensionSheetKey](IEaxDrillSettings.DimensionSheetKey.htm)
		 Свойство DimensionSheetKey
		 возвращает ключ листа, на котором расположена область данных с
		 измерением.


		 ![](../../Property_Image.gif)
		 [Mode](IEaxDrillSettings.Mode.htm)
		 Свойство Mode определяет
		 способ изменения отметки в [расшифровке
		 элементов измерения](UiAnalyticalArea.chm::/Hyperlinks/Working_with_hyperlinks.htm).


		 ![](../../Property_Image.gif)
		 [OpenObjectContext](IEaxDrillSettings.OpenObjectContext.htm)

		 возвращает настройки, передаваемые в открываемый объект.


		 ![](../../Property_Image.gif)
		 [OwnAction](IEaxDrillSettings.OwnAction.htm)
		 Свойство OwnAction
		 определяет наличие пользовательского действия при расшифровке.


		 ![](../../Property_Image.gif)
		 [SheetKey](IEaxDrillSettings.SheetKey.htm)
		 Свойство SheetKey определяет

		 ![](../../Property_Image.gif)
		 [SliceKey](IEaxDrillSettings.SliceKey.htm)
