# IEaxDrillSettings.DimensionIndex


## Синтаксис


DimensionIndex: [IDimIndex](KeDims.chm::/interface/IDimIndex/IDimIndex.htm);


## Описание


Свойство DimensionIndex определяет
 индекс для поиска элементов в [расшифровке
 элементов измерения](UiAnalyticalArea.chm::/Hyperlinks/Working_with_hyperlinks.htm).


## Комментарии


Для использования данного свойства необходимо указать измерение в свойстве
 [IEaxDrillSettings.Dimension](IEaxDrillSettings.Dimension.htm).


Для определения атрибута поиска в индексе используйте [IEaxDrillSettings.DimensionAttribute](IEaxDrillSettings.DimensionAttribute.htm).
