# IEaxDrillSettings.SliceKey


## Синтаксис


SliceKey: Integer;


## Описание


Свойство SliceKey возвращает
 ключ среза данных, использующего [расшифровку
 элементов измерения](UiAnalyticalArea.chm::/Hyperlinks/Working_with_hyperlinks.htm).


## Комментарии


Для получения ключа измерения, использующего [расшифровку
 элементов измерения](UiAnalyticalArea.chm::/Hyperlinks/Working_with_hyperlinks.htm), воспользуйтесь [IEaxDrillSettings.DimensionKey](IEaxDrillSettings.DimensionKey.htm).
