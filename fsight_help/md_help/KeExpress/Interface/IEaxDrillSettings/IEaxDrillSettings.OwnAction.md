# IEaxDrillSettings.OwnAction


## Синтаксис


OwnAction: Boolean;


## Описание


Свойство OwnAction определяет
 наличие пользовательского действия при расшифровке.


## Комментарии


Допустимые значения:


	- True. Будет выполняться
	 настроенное для расшифровки действие;


	- False. Обработка щелчков
	 по гиперссылке будет осуществляться компонентами, отображающими область
	 данных, такими как ReportBox
	 и TabSheetBox.
