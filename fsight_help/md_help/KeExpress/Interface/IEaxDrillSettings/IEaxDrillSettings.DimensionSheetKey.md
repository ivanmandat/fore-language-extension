# IEaxDrillSettings.DimensionSheetKey


## Синтаксис


DimensionSheetKey: Integer;


## Описание


Свойство DimensionSheetKey возвращает
 ключ листа, на котором расположена область данных с измерением.


## Комментарии


Свойство возвращает значение «-1», если лист с установленным измерением
 не найден.


Для определения измерения с [расшифровкой](UiAnalyticalArea.chm::/Hyperlinks/Working_with_hyperlinks.htm)
 используйте свойство [IEaxDrillSettings.Dimension](IEaxDrillSettings.Dimension.htm).
