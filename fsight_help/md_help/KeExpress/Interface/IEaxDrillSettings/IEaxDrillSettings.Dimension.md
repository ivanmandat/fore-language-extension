# IEaxDrillSettings.Dimension


## Синтаксис


Dimension: [IPivotDimension](KePivot.chm::/Interface/IPivotDimension/IPivotDimension.htm);


## Описание


Свойство Dimension определяет
 измерение, использующее [расшифровку
 элементов измерения](UiAnalyticalArea.chm::/Hyperlinks/Working_with_hyperlinks.htm).


## Комментарии


Для определения способа изменения отметки в [расшифровке
 элементов измерения](UiAnalyticalArea.chm::/Hyperlinks/Working_with_hyperlinks.htm) используйте свойство [IEaxDrillSettings.Mode](IEaxDrillSettings.Mode.htm).
