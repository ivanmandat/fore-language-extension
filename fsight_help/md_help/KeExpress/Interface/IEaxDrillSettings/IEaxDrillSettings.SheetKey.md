# IEaxDrillSettings.SheetKey


## Синтаксис


SheetKey: Integer;


## Описание


Свойство SheetKey определяет
 ключ листа, на котором используется [расшифровка
 элементов измерения](UiAnalyticalArea.chm::/Hyperlinks/Working_with_hyperlinks.htm).


## Комментарии


Для многолистового отчета не нужно использовать данное свойство, т.к.
 ключи листов сгенерируются автоматически.
