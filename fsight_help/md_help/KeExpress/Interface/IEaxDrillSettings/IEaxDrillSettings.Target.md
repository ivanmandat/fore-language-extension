# IEaxDrillSettings.Target


## Синтаксис


Target: [TabHyperlinkTarget](TabSheet.chm::/Enums/TabHyperlinkTarget.htm);


## Описание


Свойство Target определяет способ
 загрузки страницы при переходе по гиперссылке.


## Комментарии


Для использования данного свойства, необходимо чтобы [IEaxDrillSettings.ActionType](IEaxDrillSettings.ActionType.htm)
 принимало значение [TabHyperlinkActionTyp.OpenURL](TabSheet.chm::/Enums/TabHyperlinkActionType.htm).
