# IEaxDrillSettings.ActionType


## Синтаксис


ActionType: [TabHyperlinkActionType](TabSheet.chm::/Enums/TabHyperlinkActionType.htm);


## Описание


Свойство ActionType определяет
 тип действия, выполняемого при щелчке по гиперссылке.


## Комментарии


После определения типа действия необходимо задать само действие с помощью
 [IEaxDrillSettings.Action](IEaxDrillSettings.Action.htm).
