# IEaxDrillSettings.OpenObjectInstance


## Синтаксис


OpenObjectInstance(Element: Integer): [IMetabaseObjectInstance](KeSom.chm::/Interface/IMetabaseObjectInstance/IMetabaseObjectInstance.htm);


## Параметры


Element. Ключ измерения.


## Описание


Метод OpenObjectInstance открывает
 объект с заданными параметрами.


## Комментарии


Если в качестве измерения куба используется справочник с [параметрами](UiNavObj.chm::/reference_book/Master_Table_reference_book/parameters.htm),
 то при [расшифровке
 элементов измерения](UiAnalyticalArea.chm::/Hyperlinks/Working_with_hyperlinks.htm) задайте значения параметрам для
 открытия справочника.
