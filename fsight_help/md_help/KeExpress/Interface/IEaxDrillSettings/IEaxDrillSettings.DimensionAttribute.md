# IEaxDrillSettings.DimensionAttribute


## Синтаксис


DimensionAttribute: [IDimAtribute](KeDims.chm::/interface/IDimAttribute/IDimAttribute.htm);


## Описание


Свойство DimensionAttribute
 определяет атрибут поиска в индексе в [расшифровке
 элементов измерения](UiAnalyticalArea.chm::/Hyperlinks/Working_with_hyperlinks.htm).


## Комментарии


Для использования данного свойства необходимо указать измерение в свойстве
 [IEaxDrillSettings.Dimension](IEaxDrillSettings.Dimension.htm).


Для задания списка атрибутов, по которым будет производиться поиск,
 используйте [IEaxDrillSettings.DimensionAttributes](IEaxDrillSettings.DimensionAttributes.htm).


Для определения индекса для поиска элементов в [расшифровке
 элементов измерения](UiAnalyticalArea.chm::/Hyperlinks/Working_with_hyperlinks.htm) используйте [IEaxDrillSettings.DimensionIndex](IEaxDrillSettings.DimensionIndex.htm).
