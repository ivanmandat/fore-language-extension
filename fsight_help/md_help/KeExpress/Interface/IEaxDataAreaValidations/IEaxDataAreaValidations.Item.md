# IEaxDataAreaValidations.Item


## Синтаксис


Item(Index: Integer): [IEaxDataAreaValidation](../IEaxDataAreaValidation/IEaxDataAreaValidation.htm);


## Параметры


Index. Индекс правила валидации.


## Описание


Свойство Item возвращает правило
 валидации по его индексу.


## Комментарии


Для получения индекса правила валидации используйте [IEaxDataAreaValidations.IndexOf](IEaxDataAreaValidations.IndexOf.htm).
