# IEaxDataAreaValidations.Remove


## Синтаксис


Remove(Value: [IEaxDataAreaValidation](../IEaxDataAreaValidation/IEaxDataAreaValidation.htm));


## Параметры


Value. Правило валидации как
 объект.


## Описание


Метод Remove удаляет правило
 валидации из коллекции.


## Комментарии


Для удаления правила валидации по его ключу используйте [IEaxDataAreaValidations.RemoveByKey](IEaxDataAreaValidations.RemoveByKey.htm).
