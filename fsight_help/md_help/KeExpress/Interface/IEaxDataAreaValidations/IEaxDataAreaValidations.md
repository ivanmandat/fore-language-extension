# IEaxDataAreaValidations


Сборка: Express;


## Описание


Интерфейс IEaxDataAreaValidations
 используется для работы с коллекцией правил валидации в аналитической
 области данных.


## Иерархия наследования


           IEaxDataAreaValidations


## Комментарии


Для получения объекта данного типа используйте [IEaxDataAreaSlice.Validations](../IEaxDataAreaSlice/IEaxDataAreaSlice.Validations.htm),
 [IEaxDataAreaValidationChain.Validations](../IEaxDataAreaValidationChain/IEaxDataAreaValidationChain.Validations.htm).
