# IEaxDataAreaValidations.IndexOf


## Синтаксис


IndexOf(Value: [IEaxDataAreaValidation](../IEaxDataAreaValidation/IEaxDataAreaValidation.htm)):
 Integer;


## Параметры


Value. Правило валидации как
 объект.


## Описание


Метод IndexOf возвращает индекс
 указанного правила валидации.


## Комментарии


Для получения правила валидации по его ключу используйте свойство [IEaxDataAreaValidations.Item](IEaxDataAreaValidations.Item.htm).
