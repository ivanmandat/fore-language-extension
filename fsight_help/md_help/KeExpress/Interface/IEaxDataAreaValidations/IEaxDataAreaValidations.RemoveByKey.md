# IEaxDataAreaValidations.RemoveByKey


## Синтаксис


RemoveByKey(Value: Integer): Boolean;


## Параметры


Value. Ключ правила валидации.


## Описание


Метод RemoveByKey удаляет правило
 валидации по его ключу.


## Комментарии


Допустимые значения:


	- True. Правило успешно
	 удалено;


	- False. Правило не удалено.
