# IEaxDataAreaValidations.FindByKey


## Синтаксис


FindByKey(Value: Integer): [IEaxDataAreaValidation](../IEaxDataAreaValidation/IEaxDataAreaValidation.htm);


## Параметры


Value. Ключ правила валидации.


## Описание


Метод FindByKey осуществляет
 поиск правила валидации по его ключу.


## Комментарии


Если правило валидации не найдено, то метод возвращает значение Null.
