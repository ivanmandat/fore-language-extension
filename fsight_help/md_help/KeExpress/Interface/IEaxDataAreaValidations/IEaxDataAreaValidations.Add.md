# IEaxDataAreaValidations.Add


## Синтаксис


Add(Value: Object): [IEaxDataAreaValidation](../IEaxDataAreaValidation/IEaxDataAreaValidation.htm);


## Параметры


Value. Правило валидации как
 объект.


## Описание


Метод Add добавляет правило
 валидации в коллекцию правил.


## Комментарии


Для удаления правила валидации используйте [IEaxDataAreaValidations.Remove](IEaxDataAreaValidations.Remove.htm),
 [IEaxDataAreaValidations.RemoveByKey](IEaxDataAreaValidations.RemoveByKey.htm).
