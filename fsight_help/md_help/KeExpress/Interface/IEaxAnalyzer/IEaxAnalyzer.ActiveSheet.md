# IEaxAnalyzer.ActiveSheet


## Синтаксис


ActiveSheet: [IEaxSheet](../IEaxSheet/IEaxSheet.htm);


## Описание


Свойство ActiveSheet определяет
 активный лист экспресс-отчета.


## Комментарии


Свойство актуально, если для экспресс-отчета в коллекции [IEaxAnalyzeCore.Sheets](../IEaxAnalyzeCore/IEaxAnalyzeCore.Sheets.htm)
 созданы дополнительные листы. По умолчанию в данной коллекции создан только
 один лист, соответствующий текущему экспресс-отчету. Данный лист автоматически
 будет установлен активным.
