# IEaxAnalyzer


Сборка: Express;


## Описание


Интерфейс IEaxAnalyzer содержит
 свойства и методы объекта репозитория - Экспресс-отчет.


## Иерархия наследования


           [IEaxAnalyzeCore](../IEaxAnalyzeCore/IEaxAnalyzeCore.htm)


           IEaxAnalyzer


## Комментарии


Для работы с коллекцией листов экспресс-отчета используйте свойства
 и методы интерфейса [IEaxSheets](../IEaxSheets/IEaxSheets.htm).
