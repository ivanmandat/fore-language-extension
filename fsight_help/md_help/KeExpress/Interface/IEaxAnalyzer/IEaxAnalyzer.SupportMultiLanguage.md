# IEaxAnalyzer.SupportMultiLanguage


## Синтаксис


SupportMultiLanguage: Boolean;


## Описание


Свойство SupportMultiLanguage
 определяет, является ли заголовок экспресс-отчёта мультиязычным.


## Комментарии


Допустимые значения:


	- True. Заголовок отчёта
	 является [мультиязычным](UiExpress.chm::/Setup_express_report/UiExpress_Purpose_Toolbar_Title.htm).
	 Объект «Ресурсы», используемый для хранения переводов, определяет
	 свойство [IEaxAnalyzeCore.Resources](../IEaxAnalyzeCore/IEaxAnalyzeCore.Resources.htm).
	 Если объект не задан, то он будет создан автоматически в качестве
	 дочернего объекта экспресс-отчёта;


	- False. Заголовок отчёта
	 является одноязычным.
