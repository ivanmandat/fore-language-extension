# IEaxDataAreaSource.Load


## Синтаксис


Load;


## Описание


Метод Load загружает источник
 данных аналитической области.


## Комментарии


Метод используется, если


	- у отчёта для свойства [IEaxDataArea.DelayedLoad](../IEaxDataArea/IEaxDataArea.DelayedLoad.htm)
	 установлено значение True;


	- у источника данных для свойства [IEaxDataAreaSource.IsLoaded](IEaxDataAreaSource.IsLoaded.htm)
	 установлено значение True.
