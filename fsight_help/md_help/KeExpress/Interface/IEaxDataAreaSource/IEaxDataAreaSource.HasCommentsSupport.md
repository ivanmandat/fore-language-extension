# IEaxDataAreaSource.HasCommentsSupport


## Синтаксис


HasCommentsSupport: Boolean;


## Описание


Свойство HasCommentsSupport
 возвращает, поддерживает ли источник пользовательские комментарии.


## Комментарии


Свойство возвращает значение True,
 если в дополнительных настройках мастера стандартного куба включено хранение
 комментариев.
