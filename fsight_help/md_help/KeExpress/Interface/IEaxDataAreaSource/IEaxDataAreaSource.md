# IEaxDataAreaSource


Сборка: Express;


## Описание


Интерфейс IEaxDataAreaSource
 предназначен для работы с источником данных аналитической области.


## Иерархия наследования


           [INamedEntity](kesom.chm::/interface/inamedentity/inamedentity.htm)


           IEaxDataAreaSource


## Комментарии


Для работы с коллекцией источников данных аналитической области используйте
 интерфейс [IEaxDataAreaSources](../IEaxDataAreaSources/IEaxDataAreaSources.htm).
