# IDataAreaOwner


Сборка: Express;


## Описание


Интерфейс IDataAreaOwner
 предназначен для работы с областью данных объекта.


## Иерархия наследования


           IDataAreaOwner


## Комментарии


Данный интерфейс позволяет получить доступ к областям данных таких объектов
 как: регламентный отчет, визуализатор аналитической панели, рабочая книга.
