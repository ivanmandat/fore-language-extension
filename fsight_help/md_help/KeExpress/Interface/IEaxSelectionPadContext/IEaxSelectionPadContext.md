# IEaxSelectionPadContext


Сборка: Express;


## Описание


Интерфейс IEaxSelectionPadContext
 предназначен для работы с измерениями, расположенных на вкладках боковой
 панели.


## Иерархия наследования


           [ISystemPadContext](../ISystemPadContext/ISystemPadContext.htm)


           IEaxSelectionPadContext


## Комментарии


Для работы с коллекцией вкладок боковой панели экспресс-отчета используйте
 свойства и методы интерфейса [IEaxConfigurationPads](../IEaxConfigurationPads/IEaxConfigurationPads.htm).
