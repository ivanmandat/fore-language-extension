# IEaxConfigurationCategory.IsSystem


## Синтаксис


IsSystem: Boolean;


## Описание


Свойство IsSystem определяет
 признак принадлежности вкладки к системной группе, реализованной для экспресс-отчетов.


## Комментарии


Если свойство в значении False,
 то вкладка не принадлежит к системной группе.


Если свойство в значении True,
 то вкладка принадлежит к системной группе.
