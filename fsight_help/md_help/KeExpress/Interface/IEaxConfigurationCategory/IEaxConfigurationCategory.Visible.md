# IEaxConfigurationCategory.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет
 признак отображения группы вкладок на боковой панели.


## Комментарии


По умолчанию свойство находится в значении True.


Если свойство в значении False,
 то группа вкладок не будет отображаться на боковой панели.
