# IEaxConfigurationCategory


Сборка: Express;


## Описание


Интерфейс IEaxConfigurationCategory
 содержит свойства группы вкладок боковой панели.


## Иерархия наследования


           IEaxConfigurationCategory


## Комментарии


Свойства группы вкладок могут быть получены в свойстве [IEaxConfigurationCategories.Item](../IEaxConfigurationCategories/IEaxConfigurationCategories.Item.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Caption](IEaxConfigurationCategory.Caption.htm)


		 Свойство Caption определяет
		 заголовок группы вкладок.


		 ![](../../Property_Image.gif)
		 [IsSystem](IEaxConfigurationCategory.IsSystem.htm)


		 Свойство IsSystem определяет
		 признак принадлежности вкладки к системной группе, реализованной
		 для экспресс-отчетов.


		 ![](../../Property_Image.gif)
		 [Visible](IEaxConfigurationCategory.Visible.htm)


		 Свойство Visible определяет
		 признак отображения группы вкладок на боковой панели.
