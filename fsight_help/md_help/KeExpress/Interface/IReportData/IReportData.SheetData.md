# IReportData.SheetData


## Синтаксис


SheetData: [IReportSheetData](../IReportSheetData/IReportSheetData.htm);


## Описание


Свойство SheetData возвращает
 объект для работы с данными листа отчета.


## Комментарии


Данные листа отчета состоят из:


	- коллекции значений параметров источника данных отчета;


	- коллекции источников данных листа отчета;


	- описания листа отчета;


	- идентификатора листа отчета;


	- наименования листа отчета;


	- коллекции значений параметров отчета, заданных пользователем;


	- отметки элементов измерения отчета.
