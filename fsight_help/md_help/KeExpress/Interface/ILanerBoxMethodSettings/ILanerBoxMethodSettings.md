# ILanerBoxMethodSettings


Сборка: Express;


## Описание


Интерфейс ILanerBoxMethodSettings
 предназначен для настройки параметров применения метода преобразования
 к ряду данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CreateDerivedSeries](ILanerBoxMethodSettings.CreateDerivedSeries.htm)


		 Свойство CreateDerivedSeries
		 определяет, добавлять ли в качестве дочерних вычисленные ряды.


		 ![](../../Property_Image.gif)
		 [Param](ILanerBoxMethodSettings.Param.htm)


		 Свойство Param определяет
		 параметры преобразования ряда.


		 ![](../../Property_Image.gif)
		 [TrySelfExecute](ILanerBoxMethodSettings.TrySelfExecute.htm)


		 Свойство TrySelfExecute
		 определяет, создавать ли новый вычислимый ряд или преобразовывать
		 текущий ряд в вычислимый.
