# IEaxDrillSettingsValues.RemoveAt


## Синтаксис


RemoveAt(Key: Integer): Boolean;


## Параметры


Key. Ключ значения [подстановки
 расшифровки](UiAnalyticalArea.chm::/Hyperlinks/Hyperlinks_sub.htm) в коллекции.


## Описание


Метод RemoveAt удаляет значение
 [подстановки
 расшифровки](UiAnalyticalArea.chm::/Hyperlinks/Hyperlinks_sub.htm) из коллекции по заданному ключу.


## Комментарии


Для добавления значения подстановки расшифровки в коллекцию используйте
 метод [IEaxDrillSettingsValues.Add](IEaxDrillSettingsValues.Add.htm).
