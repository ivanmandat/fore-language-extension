# IEaxDrillSettingsValues


Сборка: Express;


## Описание


Интерфейс IEaxDrillSettingsValues
 используется для работы с коллекцией значений [подстановок
 расшифровки](UiAnalyticalArea.chm::/Hyperlinks/Hyperlinks_sub.htm).


## Иерархия наследования


           IEaxDrillSettingsValues


## Комментарии


Для получения коллекции значений подстановок расшифровки используйте
 свойство [IEaxDrillSettings.Values](../IEaxDrillSettings/IEaxDrillSettings.Values.htm).
