# IEaxDrillSettingsValues.GetAt


## Синтаксис


GetAt(Key: Integer): Variant;


## Параметры


Key. Ключ значения подстановки
 расшифровки в коллекции.


## Описание


Метод GetAt возвращает значение
 [подстановки
 расшифровки](UiAnalyticalArea.chm::/Hyperlinks/Hyperlinks_sub.htm) в коллекции по заданному ключу.


## Комментарии


Метод возвращает значение Null, если значение подстановки расшифровки
 по заданному ключу не найдено.
