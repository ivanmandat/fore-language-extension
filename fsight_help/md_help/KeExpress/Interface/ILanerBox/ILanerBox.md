# ILanerBox


Сборка: Express;


## Описание


Интерфейс ILanerBox содержит
 свойства и методы компонента среды разработки - LanerBox.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BorderStyle](ILanerBox.BorderStyle.htm)


		 Свойство BorderStyle
		 определяет стиль границы компонента.


		 ![](../../Property_Image.gif)
		 [DisabledManagedOptions](ILanerBox.DisabledManagedOptions.htm)


		 Свойство DisabledManagedOptions
		 определяет, использовать ли настройки, заданные для компонента
		 LanerBox.


		 ![](../../Property_Image.gif)
		 [Object](ILanerBox.Object.htm)


		 Свойство Object определяет
		 объект, являющийся источником данных компонента.


		 ![](../../Property_Image.gif)
		 [SelectedSeries](ILanerBox.SelectedSeries.htm)


		 Свойство SelectedSeries
		 возвращает указанный ряд, выделенный в компоненте.


		 ![](../../Property_Image.gif)
		 [SelectedSeriesCount](ILanerBox.SelectedSeriesCount.htm)


		 Свойство SelectedSeriesCount
		 возвращает количество рядов, выделенных в компоненте.


		 ![](../../Property_Image.gif)
		 [ShowDimPanel](ILanerBox.ShowDimPanel.htm)


		 Свойство ShowDimPanel
		 определяет, отображать ли панель размерностей в компоненте.


		 ![](../../Property_Image.gif)
		 [ShowMethodsBar](ILanerBox.ShowMethodsBar.htm)


		 Свойство ShowMethodsBar
		 определяет, отображать ли панель моделирования в компоненте.


		 ![](../../Property_Image.gif)
		 [ShowToolbar](ILanerBox.ShowToolbar.htm)


		 Свойство ShowToolbar
		 определяет, отображать ли панель инструментов в компоненте.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ApplyInversion](ILanerBox.ApplyInversion.htm)


		 Метод ApplyInversion
		 применяет указанное преобразование к выделенному вычисляемому
		 ряду.


		 ![](../../Sub_Image.gif)
		 [CanApplyInversion](ILanerBox.CanApplyInversion.htm)


		 Метод CanApplyInversion
		 возвращает признак того, что указанное преобразование можно применить
		 к выделенному ряду.


		 ![](../../Sub_Image.gif)
		 [CanDecreaseLag](ILanerBox.CanDecreaseLag.htm)


		 Метод CanDecreaseLag возвращает
		 признак того, может ли быть уменьшен лаг выделенного ряда.


		 ![](../../Sub_Image.gif)
		 [CanExecuteMethod](ILanerBox.CanExecuteMethod.htm)


		 Метод CanExecuteMethod
		 возвращает признак того, можно ли применить указанный метод преобразования
		 к выделенному ряду/рядам рабочей книги.


		 ![](../../Sub_Image.gif)
		 [CanExecuteOperation](ILanerBox.CanExecuteOperation.htm)


		 Метод CanExecuteOperation
		 возвращает признак того, можно ли применить указанную операцию
		 к рабочей книге, отображаемой в LanerBox.


		 ![](../../Sub_Image.gif)
		 [CanIncreaseLag](ILanerBox.CanIncreaseLag.htm)


		 Метод CanIncreaseLag
		 возвращает признак того, может ли быть увеличен лаг выделенного
		 ряда.


		 ![](../../Sub_Image.gif)
		 [Copy](ILanerBox.Copy.htm)


		 Метод Copy копирует
		 набор рядов рабочей книги в буфер обмена.


		 ![](../../Sub_Image.gif)
		 [Cut](ILanerBox.Cut.htm)


		 Метод Cut вырезает
		 набор рядов рабочей книги в буфер обмена.


		 ![](../../Sub_Image.gif)
		 [DecreaseLag](ILanerBox.DecreaseLag.htm)


		 Метод DecreaseLag выполняет
		 уменьшение лага выделенного ряда.


		 ![](../../Sub_Image.gif)
		 [ExecuteMethodEx](ILanerBox.ExecuteMethodEx.htm)


		 Метод ExecuteMethodEx
		 выполняет преобразование выделенного ряда данных указанным методом
		 с заданными параметрами.


		 ![](../../Sub_Image.gif)
		 [ExecuteOperation](ILanerBox.ExecuteOperation.htm)


		 Метод ExecuteOperation
		 выполняет указанную операцию над рабочей книгой.


		 ![](../../Sub_Image.gif)
		 [GroupSeries](ILanerBox.GroupSeries.htm)


		 Метод GroupSeries объединяет
		 набор рядов рабочей книги в группу.


		 ![](../../Sub_Image.gif)
		 [HitTest](ILanerBox.HitTest.htm)


		 Метод HitTest возвращает
		 область компонента по указанным координатам.


		 ![](../../Sub_Image.gif)
		 [IncreaseLag](ILanerBox.IncreaseLag.htm)


		 Метод IncreaseLag выполняет
		 увеличение лага выделенного ряда.


		 ![](../../Sub_Image.gif)
		 [IsInversionTurnedOn](ILanerBox.IsInversionTurnedOn.htm)


		 Метод IsInversionTurnedOn
		 возвращает признак того, что указанное преобразование уже применено
		 к ряду.


		 ![](../../Sub_Image.gif)
		 [IsOptionSwitchedOn](ILanerBox.IsOptionSwitchedOn.htm)


		 Метод IsOptionSwitchedOn
		 возвращает признак того, применена ли указанная операция в рабочей
		 книге.


		 ![](../../Sub_Image.gif)
		 [Paste](ILanerBox.Paste.htm)


		 Метод Paste вставляет
		 набор рядов в рабочую книгу из буфера обмена.


		 ![](../../Sub_Image.gif)
		 [UngroupSeries](ILanerBox.UngroupSeries.htm)


		 Метод UngroupSeries
		 разъединяет группу рядов в рабочей книге.

		 [Enabled](ModForms.chm::/Interface/IControl/IControl.Enabled.htm)

		 [Font](ModForms.chm::/Interface/IControl/IControl.Font.htm)

		 ![](../../Property_Image.gif)
		 [HelpContext](ModForms.chm::/Interface/IControl/IControl.HelpContext.htm)


		 Свойство HelpContext

		 для данного компонента.


		 ![](../../Property_Image.gif)
		 [Hint](ModForms.chm::/Interface/IControl/IControl.Hint.htm)


		 Свойство Hint определяет
		 текст подсказки, появляющийся при задержке указателя мыши над

		 [HintTimeout](ModForms.chm::/Interface/IControl/IControl.HintTimeout.htm)

		 определяет время отображения всплывающей подсказки.


		 ![](../../Property_Image.gif)
		 [Left](ModForms.chm::/Interface/IControl/IControl.Left.htm)


		 Свойство Left определяет
		 координату левого края компонента в пикселях.


		 ![](../../Property_Image.gif)

		 ![](../../Property_Image.gif)
		 [ParentColor](ModForms.chm::/Interface/IControl/IControl.ParentColor.htm)

		 компонента.


		 ![](../../Property_Image.gif)
		 [ParentFont](ModForms.chm::/Interface/IControl/IControl.ParentFont.htm)


		 Свойство ParentFont

		 ![](../../Property_Image.gif)
		 [ParentShowHint](ModForms.chm::/Interface/IControl/IControl.ParentShowHint.htm)


		 Свойство ParentShowHint

		 [PopupMenu](ModForms.chm::/Interface/IControl/IControl.PopupMenu.htm)

		 [TabOrder](ModForms.chm::/Interface/IControl/IControl.TabOrder.htm)

		 ![](../../Property_Image.gif)
		 [TabStop](ModForms.chm::/Interface/IControl/IControl.TabStop.htm)


		 Свойство TabStop определяет

		 ![](../../Property_Image.gif)
		 [Text](ModForms.chm::/Interface/IControl/IControl.Text.htm)


		 Свойство Text определяет

		 ![](../../Property_Image.gif)
		 [Top](ModForms.chm::/Interface/IControl/IControl.Top.htm)

		 ![](../../Property_Image.gif)
		 [Visible](ModForms.chm::/Interface/IControl/IControl.Visible.htm)

## Методы, унаследованные от [IControl](ModForms.chm::/Interface/IControl/IControl.htm)

		 [ScreenToClient](ModForms.chm::/Interface/IControl/IControl.ScreenToClient.htm)


		 Метод ScreenToClient

		 относительно системы координат компонента.


		 ![](../../Sub_Image.gif)
		 [SetFocus](ModForms.chm::/Interface/IControl/IControl.SetFocus.htm)


		 Метод SetFocus устанавливает
		 фокус на данный компонент.
