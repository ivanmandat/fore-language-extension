# ILanerBox.CanDecreaseLag


## Синтаксис


CanDecreaseLag: Boolean;


## Описание


Метод CanDecreaseLag возвращает
 признак того, может ли быть уменьшен лаг выделенного ряда.


## Комментарии


Допустимые значения:


	- True. Лаг выделенного
	 ряда может быть уменьшен;


	- False. Лаг выделенного
	 ряда не может быть уменьшен.


Для уменьшения лага выделенного ряда используйте метод [ILanerBox.DecreaseLag](ILanerBox.DecreaseLag.htm).
