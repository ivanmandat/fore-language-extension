# ILanerBox.BorderStyle


## Синтаксис


BorderStyle: [ControlBorderStyle](ModForms.chm::/Enums/ControlBorderStyle.htm);


## Описание


Свойство BorderStyle определяет
 стиль границы компонента.


## Комментарии


При размещении компонента LanerBox на форме свойство принимает значение
 [ControlBorderStyle.Single](ModForms.chm::/Enums/ControlBorderStyle.htm),
 т.е. компонент имеет объемный вид.
