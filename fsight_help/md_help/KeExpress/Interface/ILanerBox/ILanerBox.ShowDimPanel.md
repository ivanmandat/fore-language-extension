# ILanerBox.ShowDimPanel


## Синтаксис


ShowDimPanel: Boolean;


## Описание


Свойство ShowDimPanel определяет,
 отображать ли панель размерностей в компоненте.


## Комментарии


Панель размерностей может быть отображена только в многомерном режиме
 отображения данных. Переход в данный режим описан в примере для [ILanerBox.ExecuteOperation](ILanerBox.ExecuteOperation.htm).


Допустимые значения:


	- True. По умолчанию.
	 Панель размерностей отображается в компоненте;


	- False. Панель размерностей
	 не отображается в компоненте.
