# ILanerBox.SelectedSeriesCount


## Синтаксис


SelectedSeriesCount: Integer;


## Описание


Свойство SelectedSeriesCount
 возвращает количество рядов, выделенных в компоненте.


## Комментарии


Свойство возвращает только положительные значения. Если в LanerBox не
 выделен ни один ряд, то SelectedSeriesCount
 вернет нуль.
