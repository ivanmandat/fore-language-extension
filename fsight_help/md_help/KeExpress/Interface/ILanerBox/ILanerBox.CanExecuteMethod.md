# ILanerBox.CanExecuteMethod


## Синтаксис


CanExecuteMethod(Method: [LanerBoxMethod](../../Enums/LanerBoxMethod.htm)):
 Boolean;


## Параметры


Method. Метод преобразования
 ряда.


## Описание


Метод CanExecuteMethod возвращает
 признак того, можно ли применить указанный метод преобразования к выделенному
 ряду/рядам рабочей книги.


## Комментарии


Допустимые значения:


	- True. Можно применять
	 указанный метод преобразования к выделенному ряду/рядам;


	- False. Нельзя применять
	 указанный метод преобразования к выделенному ряду/рядам.


Для выполнения преобразования используйте метод [ILanerBox.ExecuteMethodEx](ILanerBox.ExecuteMethodEx.htm).
