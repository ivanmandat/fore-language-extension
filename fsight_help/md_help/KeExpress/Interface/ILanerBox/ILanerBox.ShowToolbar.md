# ILanerBox.ShowToolbar


## Синтаксис


ShowToolbar: Boolean;


## Описание


Свойство ShowToolbar определяет,
 отображать ли панель инструментов в компоненте.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Панель инструментов отображается в компоненте;


	- False. Панель инструментов
	 не отображается в компоненте.


Внешний вид панели инструментов и её назначение приведены в разделе
 «[Описание
 интерфейса анализа временных рядов](UiDw.chm::/Purpose/UiDw_Purpose_Basic_Window.htm)».
