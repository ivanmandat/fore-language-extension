# ILanerBox.CanIncreaseLag


## Синтаксис


CanIncreaseLag: Boolean;


## Описание


Метод CanIncreaseLag возвращает
 признак того, может ли быть увеличен лаг выделенного ряда.


## Комментарии


Допустимые значения:


	- True. Лаг выделенного
	 ряда может быть увеличен;


	- False. Лаг выделенного
	 ряда не может быть увеличен.


Для увеличения лага выделенного ряда используйте метод [ILanerBox.IncreaseLag](ILanerBox.IncreaseLag.htm).
