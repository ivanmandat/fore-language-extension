# IEaxDataAreaValidationLink.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет,
 будет ли выполняться правило валидации.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Правило валидации будет выполняться;


	- False. Правило валидации
	 не будет выполняться.
