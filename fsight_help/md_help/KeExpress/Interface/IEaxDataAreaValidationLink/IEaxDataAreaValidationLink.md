# IEaxDataAreaValidationLink


Сборка: Express;


## Описание


Интерфейс IEaxDataAreaValidationLink
 используется для работы со ссылками правил валидации в аналитической области
 данных.


## Иерархия наследования


           IEaxDataAreaValidationLink


## Комментарии


Для получения объекта данного типа используйте [IEaxDataAreaValidationChain.Item](../IEaxDataAreaValidationChain/IEaxDataAreaValidationChain.Item.htm).


Количество правил в цепочке диагностического отчета равно количеству
 правил валидаций в аналитической области данных.
