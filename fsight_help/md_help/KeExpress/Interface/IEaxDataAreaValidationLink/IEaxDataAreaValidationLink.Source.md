## Описание


Свойство Source возвращает объект-родитель
 правила валидации.


## Комментарии


Валидация - это проверка данных на соответствие заданным условиям
 и ограничениям. Валидация доступна для временных рядов, экспресс-отчетов,
 таблиц в аналитических панелях, областей данных в регламентных отчетах.


Валидация данных выполняется с помощью правил валидации.
