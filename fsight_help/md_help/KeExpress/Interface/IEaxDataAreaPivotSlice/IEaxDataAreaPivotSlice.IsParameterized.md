# IEaxDataAreaPivotSlice.IsParameterized


## Синтаксис


IsParameterized: Boolean;


## Описание


Свойство IsParameterized возвращает
 признак наличия параметров у источника данных для среза.


## Комментарии


Допустимые значения:


	- True. У источника есть
	 параметры;


	- False. У источника нет
	 параметров.
