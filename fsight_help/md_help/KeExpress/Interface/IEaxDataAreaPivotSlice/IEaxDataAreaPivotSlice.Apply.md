# IEaxDataAreaPivotSlice.Apply


## Синтаксис


Apply(pSlice: [IEaxDataAreaPivotSlice](IEaxDataAreaPivotSlice.htm);
 [Options: [EaxPivotSliceApplyOptions](../../Enums/EaxPivotSliceApplyOptions.htm)
 = 0]);


## Параметры


pSlice. Срез аналитической
 области данных, из которого будут импортированы параметры;


Options. Настройки применения
 импортированных параметров.


## Описание


Метод Apply применяет импортированные
 параметры к срезу аналитической области данных.


## Комментарии


К импортированных параметрам относятся:


	- оформление;


	- настройки расположения измерений;


	- отметка;


	- настройки фильтрации.
