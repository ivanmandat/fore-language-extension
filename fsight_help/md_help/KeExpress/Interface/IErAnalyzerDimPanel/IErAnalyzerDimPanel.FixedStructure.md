# IErAnalyzerDimPanel.FixedStructure


## Синтаксис


FixedStructure: Boolean;


## Описание


Свойство FixedStructure определяет,
 будет ли в панели измерений фиксированная структура измерений куба. Если
 свойство установлено в True то
 менять расположение измерений будет нельзя.
