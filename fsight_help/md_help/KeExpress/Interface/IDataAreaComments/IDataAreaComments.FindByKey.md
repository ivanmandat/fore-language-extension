# IDataAreaComments.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IDataAreaComment](../IDataAreaComment/IDataAreaComment.htm);


## Параметры


Key. Ключ комментария.


## Описание


Метод FindByKey осуществляет
 поиск по ключу и возвращает информацию о комментарии в случае удачного
 поиска.


## Комментарии


Метод возвращает значение Null,
 если комментарий с указанными ключом не найден.
