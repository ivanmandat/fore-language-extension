# IDataAreaComments


Сборка: Express;


## Описание


Интерфейс IDataAreaComments
 предназначен для работы с коллекцией комментариев, созданных для ячейки
 с данными.


## Иерархия наследования


IDataAreaComments


## Комментарии


Коллекцию комментариев можно получить, используя свойства и методы интерфейса
 [IDataAreaCommentManager](../IDataAreaCommentManager/IDataAreaCommentManager.htm).

		 Имя свойства

		 [Count](IDataAreaComments.Count.htm)
