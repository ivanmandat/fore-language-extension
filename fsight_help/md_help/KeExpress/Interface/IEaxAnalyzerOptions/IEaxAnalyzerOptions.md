# IEaxAnalyzerOptions


Сборка: Express;


## Описание


Интерфейс IEaxAnalyzerOptions
 содержит свойства параметров экспресс-отчета.


## Иерархия наследования


           IEaxOptionsBase


           IEaxAnalyzerOptions


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [IgnoreDirtyOnClose](IEaxAnalyzerOptions.IgnoreDirtyOnClose.htm)
		 Свойство IgnoreDirtyOnClose
		 определяет состояние опции «Не спрашивать подтверждения при закрытии
		 измененного отчета».


		 ![](../../Property_Image.gif)
		 [UseSharedEventHandler](IEaxAnalyzerOptions.UseSharedEventHandler.htm)
		 Свойство UseSharedEventHandler
		 определяет, будет ли экспресс-отчет использовать обработчик событий.
