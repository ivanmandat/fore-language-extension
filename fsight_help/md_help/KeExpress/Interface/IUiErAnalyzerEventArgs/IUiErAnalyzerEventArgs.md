# IUiErAnalyzerEventArgs


Сборка: Express;


## Описание


Интерфейс IUiErAnalyzerEventArgs
 содержит свойства аргумента события, происходящего при изменении расположения
 справочников либо при изменении отметки справочника компонента [UiErAnalyzer](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiErAnalyzer.htm).
