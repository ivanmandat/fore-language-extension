# IEaxMapSource


Сборка: Express;


## Описание


Интерфейс IEaxMapSource описывает
 компоненты, которые могут использоваться в качестве источников данных
 для компонента [EaxMapBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/EaxMapBox.htm).


## Комментарии


Данный интерфейс поддерживают следующие компоненты:


	- [UiEaxMap](uidevenv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiEaxMap.htm);


	- [UiErAnalyzer](KeExpress.chm::/Class/UiErAnalyzer/UiErAnalyzer.htm).
