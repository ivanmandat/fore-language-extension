# IEaxVisualizer.CanDrillObjectives


## Синтаксис


CanDrillObjectives(Value: Variant; DrillType: [EaxDrillType](../../Enums/EaxDrillType.htm)): Boolean;


## Параметры


Value. Визуализатор для детализации
 данных;


DrillType. Способ детализации
 данных.


## Описание


Метод CanDrillObjectives возвращает
 возможность детализации данных визуализатора.


## Комментарии


Для детализации данных визуализатора используйте метод [IEaxVisualizer.DrillObjectives](IEaxVisualizer.DrillObjectives.htm).
