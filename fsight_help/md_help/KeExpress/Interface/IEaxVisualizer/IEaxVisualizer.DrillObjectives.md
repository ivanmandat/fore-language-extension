# IEaxVisualizer.DrillObjectives


## Синтаксис


DrillObjectives(Value: Variant; DrillType: [EaxDrillType](../../Enums/EaxDrillType.htm));


## Параметры


Value. Визуализатор для детализации
 данных;


DrillType. Способ детализации
 данных.


## Описание


Метод DrillObjectives выполняет
 детализацию данных визуализатора.


## Комментарии


Для проверки возможности детализации данных визуализатора используйте
 метод [IEaxVisualizer.CanDrillObjectives](IEaxVisualizer.CanDrillObjectives.htm).
