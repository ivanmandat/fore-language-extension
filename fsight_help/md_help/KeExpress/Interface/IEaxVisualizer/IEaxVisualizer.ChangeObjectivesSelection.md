# IEaxVisualizer.ChangeObjectivesSelection


## Синтаксис


ChangeObjectivesSelection(Value: Variant; Operation:
 [EaxChangeSelectionOperation](../../Enums/EaxChangeSelectionOperation.htm));


## Параметры


Value. Визуализатор для отображения
 или исключения;


Operation. Операция над выделенным
 визуализатором.


## Описание


Метод ChangeObjectivesSelection
 исключает или отображает визуализатор.


## Комментарии


Для проверки возможности исключить или отобразить визуализатор используйте
 метод [IEaxVisualizer.CanChangeObjectivesSelection](IEaxVisualizer.CanChangeObjectivesSelection.htm).
