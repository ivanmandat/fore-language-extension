# IEaxVisualizer.CanChangeObjectivesSelection


## Синтаксис


CanChangeObjectivesSelection(Value: Variant; Operation:
 [EaxChangeSelectionOperation](../../Enums/EaxChangeSelectionOperation.htm)):
 Boolean;


## Параметры


Value. Визуализатор для отображения
 или исключения;


Operation. Операция над выделенным
 визуализатором.


## Описание


Метод CanChangeObjectivesSelection
 возвращает возможность исключения или отображения визуализатора.


## Комментарии


Для исключения или отображения визуализатора используйте метод [IEaxVisualizer.ChangeObjectivesSelection](IEaxVisualizer.ChangeObjectivesSelection.htm).
