# EaxConfigurationPanel.OnActivePadChanged


## Синтаксис


OnActivePadChanged(Sender: Object; Args: [IEaxPadChangedArgs](../../Interface/IEaxPadChangedArgs/IEaxPadChangedArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnActivePadChanged наступает
 при активации какой-либо вкладки компонента.
