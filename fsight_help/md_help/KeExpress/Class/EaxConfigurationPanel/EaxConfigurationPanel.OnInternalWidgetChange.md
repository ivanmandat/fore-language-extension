# EaxConfigurationPanel.OnInternalWidgetChange


## Синтаксис


OnInternalWidgetChange(Sender: Object; Args: [IWidgetChangeEventArgs](../../Interface/IWidgetChangeEventArgs/IWidgetChangeEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnInternalWidgetChange
 наступает при изменении каких-либо настроек текущего активного объекта.
