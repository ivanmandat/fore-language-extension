# LanerResultsBoxChartChangedEventArgs


Сборка: Express;


## Описание


Класс LanerResultsBoxChartChangedEventArgs
 предназначен для работы с аргументами событий, происходящих с диаграммой
 экспресс-отчета в компоненте LanerResultsBox.


## Комментарии


Для работы с компонентом среды разработки LanerResultsBox
 используйте свойства и методы интерфейса [ILanerResultsBox](../../Interface/ILanerResultsBox/ILanerResultsBox.htm).
