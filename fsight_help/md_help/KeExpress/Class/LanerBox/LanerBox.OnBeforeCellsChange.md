# LanerBox.OnBeforeCellsChange


## Синтаксис


OnBeforeCellsChange(Sender: Object; Args: [ITabSheetBaseEventArgs](TabSheet.chm::/Interface/ITabSheetBaseEventArgs/ITabSheetBaseEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnBeforeCellsChange
 наступает перед изменением значений в диапазоне ячеек таблицы.
