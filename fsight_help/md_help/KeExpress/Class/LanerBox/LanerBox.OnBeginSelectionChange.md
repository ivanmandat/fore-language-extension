# LanerBox.OnBeginSelectionChange


## Синтаксис


OnBeginSelectionChange(Sender: Object; Args: [ITabSheetBaseEventArgs](TabSheet.chm::/Interface/ITabSheetBaseEventArgs/ITabSheetBaseEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnBeginSelectionChange
 наступает при начале выделения ячеек таблицы.
