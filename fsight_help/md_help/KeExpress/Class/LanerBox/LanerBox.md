# LanerBox


Сборка: Express;


## Описание


Класс LanerBox содержит свойства
 и методы компонента среды разработки - LanerBox.


## Комментарии


Компонент LanerBox предназначен
 для просмотра рабочих книг базы данных временных рядов.
