# LanerBox.OnTableClick


## Синтаксис


OnTableClick(Sender: Object; Args: [ITabSheetBaseEventArgs](TabSheet.chm::/Interface/ITabSheetBaseEventArgs/ITabSheetBaseEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnTableClick наступает
 при щелчке по левой верхней ячейке таблицы (пересечение столбца и строки
 заголовков).


## Комментарии


Щелчок по данной ячейке приводит к выделению всей таблицы.
