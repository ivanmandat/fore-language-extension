# LanerBox.OnRowClick


## Синтаксис


OnRowClick(Sender: Object; Args: [ITabRowBaseEventArgs](TabSheet.chm::/Interface/ITabRowBaseEventArgs/ITabRowBaseEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnRowClick наступает
 при щелчке по заголовку строки таблицы.
