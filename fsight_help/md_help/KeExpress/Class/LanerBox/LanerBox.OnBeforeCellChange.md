# LanerBox.OnBeforeCellChange


## Синтаксис


OnBeforeCellChange(Sender: Object; Args: [ITabCellBeforeChangeBaseEventArgs](TabSheet.chm::/Interface/ITabCellBeforeChangeBaseEventArgs/ITabCellBeforeChangeBaseEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnBeforeCellChange наступает
 перед изменением значения ячейки таблицы.
