# LanerBox.OnCellChange


## Синтаксис


OnCellChange(Sender: Object; Args: [ITabCellChangeBaseEventArgs](TabSheet.chm::/Interface/ITabCellChangeBaseEventArgs/ITabCellChangeBaseEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnCellChange наступает,
 если было произведено изменение текста или значения в ячейке таблицы.
