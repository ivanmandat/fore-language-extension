# LanerBox.OnCellPictureClick


## Синтаксис


OnCellPictureClick(Sender: Object; Args: [ITabCellBaseEventArgs](TabSheet.chm::/Interface/ITabCellBaseEventArgs/ITabCellBaseEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnCellPictureClick наступает
 при щелчке кнопкой мыши по картинке в ячейке таблицы.


## Комментарии


Данное событие обрабатывается только в том случае, если свойство [EnablePictureEvents](TabSheet.chm::/Interface/ITabCellStyle/ITabCellStyle.EnablePictureEvents.htm)
 имеет значение [TriState.OnOption](ForeSys.chm::/Enums/TriState.htm).
