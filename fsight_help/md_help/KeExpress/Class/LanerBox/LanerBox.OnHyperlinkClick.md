# LanerBox.OnHyperlinkClick


## Синтаксис


OnHyperlinkClick(Sender: Object; Args: [ITabCellBaseEventArgs](TabSheet.chm::/Interface/ITabCellBaseEventArgs/ITabCellBaseEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnHyperlinkClick наступает
 при нажатии на гиперссылку в ячейке таблицы.
