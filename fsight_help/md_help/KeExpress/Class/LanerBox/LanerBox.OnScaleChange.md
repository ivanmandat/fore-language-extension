# LanerBox.OnScaleChange


## Синтаксис


OnScaleChange(Sender: Object; Args: [ITabScaleBaseEventArgs](TabSheet.chm::/Interface/ITabScaleBaseEventArgs/ITabScaleBaseEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnScaleChange наступает
 при изменении масштаба таблицы.
