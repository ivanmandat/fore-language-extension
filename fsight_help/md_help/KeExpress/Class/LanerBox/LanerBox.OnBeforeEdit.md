# LanerBox.OnBeforeEdit


## Синтаксис


OnBeforeEdit(Sender: Object; Args: [ITabCellBeforeEditBaseEventArgs](TabSheet.chm::/Interface/ITabCellBeforeEditBaseEventArgs/ITabCellBeforeEditBaseEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnBeforeEdit наступает
 перед началом редактирования ячейки компонента.
