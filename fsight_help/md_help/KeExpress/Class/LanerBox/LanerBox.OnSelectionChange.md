# LanerBox.OnSelectionChange


## Синтаксис


OnSelectionChange(Sender: Object; Args: [ITabSheetBaseEventArgs](TabSheet.chm::/Interface/ITabSheetBaseEventArgs/ITabSheetBaseEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnSelectionChange наступает
 при изменении выделения ячеек таблицы.
