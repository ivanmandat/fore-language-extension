# LanerBox.OnObjectActivate


## Синтаксис


OnObjectActivate(Sender: Object; Args: [ITabObjectBaseEventArgs](TabSheet.chm::/Interface/ITabObjectBaseEventArgs/ITabObjectBaseEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnObjectActivate наступает
 при активации/деактивации объекта таблицы.
