# LanerBox.OnEndSelectionChange


## Синтаксис


OnEndSelectionChange(Sender: Object; Args: [ITabSheetBaseEventArgs](TabSheet.chm::/Interface/ITabSheetBaseEventArgs/ITabSheetBaseEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnEndSelectionChange
 наступает при окончании выделения ячеек таблицы.
