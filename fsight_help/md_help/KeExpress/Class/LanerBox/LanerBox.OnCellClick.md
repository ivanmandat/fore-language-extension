# LanerBox.OnCellClick


## Синтаксис


OnCellClick(Sender: Object; Args: [ITabCellBaseEventArgs](TabSheet.chm::/Interface/ITabCellBaseEventArgs/ITabCellBaseEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnCellClick наступает
 при щелчке кнопкой мыши по ячейке таблицы.
