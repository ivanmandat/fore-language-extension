# LanerBox.OnColumnClick


## Синтаксис


OnColumnClick(Sender: Object; Args: [ITabColumnBaseEventArgs](TabSheet.chm::/Interface/ITabColumnBaseEventArgs/ITabColumnBaseEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnColumnClick наступает
 при щелчке по заголовку столбца таблицы.
