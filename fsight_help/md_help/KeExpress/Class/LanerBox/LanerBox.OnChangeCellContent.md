# LanerBox.OnChangeCellContent


## Синтаксис


OnChangeCellContent(Sender: Object; Args: [ITabCellContentChangeBaseEventArgs](TabSheet.chm::/Interface/ITabCellContentChangeBaseEventArgs/ITabCellContentChangeBaseEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnChangeCellContent
 наступает, если было произведено изменение содержимого в ячейке таблицы.


## Комментарии


В отличие от события [OnCellChange](LanerBox.OnCellChange.htm)
 данное событие происходит, если было изменено значение, формула либо текст
 ячейки.
