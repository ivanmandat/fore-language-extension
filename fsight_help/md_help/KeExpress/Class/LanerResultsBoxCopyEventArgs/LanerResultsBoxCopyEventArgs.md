# LanerResultsBoxCopyEventArgs


Сборка: Express;


## Описание


Класс LanerResultsBoxCopyEventArgs
 содержит свойства аргумента события, происходящего при копировании данных
 со вкладки «Статистические характеристики»
 на панели «Результаты вычислений»
 компонента LanerResultsBox.


## Комментарии


Событие вызывается при выполнении операции копирования из контекстного
 меню панели «Результаты вычислений»
 или при нажатии сочетания клавиш CTRL+C, CTRL+INS.
