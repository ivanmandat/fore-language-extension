# EaxCustomConfigurationCategory


Сборка: Express;


## Описание


Класс EaxCustomConfigurationCategory реализует группу вкладок компонента [EaxConfigurationPanel](uidevenv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/EaxConfigurationPanel.htm).


## Комментарии


Класс EaxCustomConfigurationCategory реализует группу вкладок не принадлежащую системной группе, реализованной для экспресс-отчетов.
