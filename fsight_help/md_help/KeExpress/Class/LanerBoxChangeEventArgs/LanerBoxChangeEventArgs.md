# LanerBoxChangeEventArgs


Сборка: Express;


## Описание


Класс LanerBoxChangeEventArgs
 предназначен для работы с аргументами события, наступающего при изменении
 рабочей книги.


## Комментарии


Используя LanerBoxChangeEventArgs
 можно определить часть рабочей книги, в которой произошли изменения.
