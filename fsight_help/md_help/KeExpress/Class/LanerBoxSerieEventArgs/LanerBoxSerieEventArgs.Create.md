# LanerBoxSerieEventArgs.Create


## Синтаксис


Create(Serie: [ILanerSerie](Laner.chm::/Interface/ILanerSerie/ILanerSerie.htm); [SerieIndex: Integer = -1;][Factor: [IRubricatorFactor](KeCubes.chm::/Interface/IRubricatorFactor/IRubricatorFactor.htm) = Null]);


## Параметры


Serie. Ряд, сгенерировавший событие.


SerieIndex. Индекс ряда, сгенерировавшего событие. Необязательный параметр. Значение по умолчанию - «-1».


Factor. Показатель, на котором основан ряд, сгенерировавший событие. Необязательный параметр. Значение по умолчанию - Null.


## Описание


Конструктор Create создает экземпляр класса для работы с аргументами события.


## Комментарии


Конструктор вызывается автоматически при генерации события.
