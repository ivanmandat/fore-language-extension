# UiErAnalyzer.OnSelectionChanged


## Синтаксис


OnSelectionChanged(Sender: [IUiErAnalyzer](../../Interface/IUiErAnalyzer/IUiErAnalyzer.htm);
 Args: [IUiErAnalyzerEventArgs](../../Interface/IUiErAnalyzerEventArgs/IUiErAnalyzerEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnSelectionChanged наступает
 при изменении отметки измерения источника данных для экспресс-отчета.
