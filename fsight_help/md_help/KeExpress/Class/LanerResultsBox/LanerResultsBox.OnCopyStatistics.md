# LanerResultsBox.OnCopyStatistics


## Синтаксис


OnCopyStatistics(Sender: Object; Args: [ILanerResultsBoxCopyEventArgs](../../Interface/ILanerResultsBoxCopyEventArgs/ILanerResultsBoxCopyEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnCopyStatistics наступает
 при копировании данных со вкладки «Статистические
 характеристики» на панели «Результаты
 вычислений» компонента LanerResultsBox.


## Комментарии


Событие вызывается при выполнении операции копирования из контекстного
 меню панели «Результаты вычислений»
 или при нажатии сочетания клавиш CTRL+C, CTRL+INS.
