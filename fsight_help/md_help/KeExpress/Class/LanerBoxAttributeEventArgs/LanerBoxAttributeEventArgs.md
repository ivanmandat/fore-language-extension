# LanerBoxAttributeEventArgs


Сборка: Express;


## Описание


Класс LanerBoxAttributeEventArgs
 предназначен для работы с аргументами событий, происходящих с атрибутами
 базы данных временных рядов в компоненте LanerBox.


## Комментарии

	- [LanerBox.OnMoveAttribute](../LanerBox/LanerBox.OnMoveAttribute.htm);


	- [LanerBox.OnRemoveAttribute](../LanerBox/LanerBox.OnRemoveAttribute.htm).
