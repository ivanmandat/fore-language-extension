# EaxMapBox.OnTerritoryClick


## Синтаксис


OnTerritoryClick(Sender: Object; Args: [IEaxMapBoxTerritoryEventArgs](../../Interface/IEaxMapBoxTerritoryEventArgs/IEaxMapBoxTerritoryEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnTerritoryClick наступает
 при щелчке кнопкой мыши по какому-либо региону карты.
