# EaxWindowsPosition


## Описание


Перечисление EaxWindowsPosition
 содержит варианты расположения окон объектов экспресс-отчета в области
 отчета.


Используется следующим свойством:


	- [IEaxAnalyzeCore.WindowsPosition](../Interface/IEaxAnalyzeCore/IEaxAnalyzeCore.WindowsPosition.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Maximized. Окно объекта
		 развернуто, одновременно визуально доступен только один из объектов
		 экспресс-отчета.


		 1
		 Vertical. Окна
		 объектов располагаются в виде вертикального ряда.


		 2
		 Horizontal. Окна объектов
		 располагаются в виде горизонтального ряда.
