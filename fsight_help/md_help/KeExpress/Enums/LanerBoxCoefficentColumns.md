# LanerBoxCoefficentColumns


## Описание


Перечисление LanerBoxCoefficentColumns
 используется для оценки модели нелинейной регрессии на вкладке «Коэффициенты» панели «Результаты
 вычислений» для компонента [LanerResultsBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/LanerResultsBox.htm).


Используется следующим свойством:


	- [ILanerResultsBoxCoefficients.ColumnType](../Interface/ILanerResultsBoxCoefficients/ILanerResultsBoxCoefficients.ColumnType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Coefficient. Коэффициент.


		 1
		 Value. Значение.


		 2
		 Error. Стандартная
		 ошибка.


		 3
		 TStatistic. t-статистика.


		 4
		 Probabilty. Вероятность.


		 5
		 Factor. Фактор.


		 6
		 InitApproximation.
		 Начальное приближение.


## Комментарии


Coefficient. Наименование коэффициента.


Value. Оцененное значение коэффициента.


Error. Приближённая величина
 отклонения (вызванного случайностью выборки) оценки коэффициента от истинного
 значения. Чем больше значение стандартной ошибки, тем менее достоверной
 является оценка коэффициента.


TStatistic. Вычисляется как
 отношение оценки коэффициента к его стандартной ошибке. Используется для
 проверки нулевой гипотезы о том, что коэффициент равен нулю, то есть не
 значим в рассматриваемой модели регрессии. Гипотеза проверяется в предположении,
 что остатки имеют нормальное распределение. Проводить интерпретацию значения
 t-статистики легче с помощью значения вероятности, которое приводится
 рядом.


Probabilty. Значение вероятности
 для t-статистики. Нулевая гипотеза о равенстве нулю оценки коэффициента
 отклоняется на выбранном уровне значимости (как правило 0,1, 0,05, 0,01),
 если вероятность меньше, чем уровень значимости.


Factor. Наименование фактора,
 при котором оценивался коэффициент.


InitApproximation. Начальное
 приближение используется в качестве значения, с помощью которого возможно
 более точно оценить модель.
