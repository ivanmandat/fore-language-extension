# AlgorithmCalculationMode


## Описание


Перечисление AlgorithmCalculationMode
 содержит режимы срабатывания алгоритма.


Используется следующими свойствами и методами:


	- [IEaxDataAreaExternalTransformation.CalculationMode](../Interface/IEaxDataAreaExternalTransformation/IEaxDataAreaExternalTransformation.CalculationMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 OnSave. На сохранение
		 данных.


		 1
		 OnChange. На изменение
		 данных.
