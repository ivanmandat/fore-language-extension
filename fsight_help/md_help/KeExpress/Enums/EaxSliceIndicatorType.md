# EaxSliceIndicatorType


## Описание


Перечисление EaxSliceIndicatorType
 используется для определения [типа метрики](UiSelection.chm::/Selection/Dimension_type.htm)
 среза аналитической области данных.


Используется следующими свойствами:


	- [IEaxDataAreaSlice.IndicatorSelection](../Interface/IEaxDataAreaSlice/IEaxDataAreaSlice.IndicatorSelection.htm);


	- [IEaxDataAreaSlice.IndicatorAggregation](../Interface/IEaxDataAreaSlice/IEaxDataAreaSlice.IndicatorAggregation.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 Color. Метрика «Цвет».


		 2
		 Size. Метрика
		 «Размер».


## Комментарии


Для карты значение Size используется
 как метрика «Объем». Данная метрика
 доступна при включенном режиме «[3D](UiMaps.chm::/Configure/MapConfigure.htm#3d)».
