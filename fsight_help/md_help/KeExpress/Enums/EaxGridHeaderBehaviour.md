# EaxGridHeaderBehaviour


## Описание


Перечисление EaxGridHeaderBehaviour
 используется для определения типа поведения строк/столбцов при вставке
 области данных.


Используется следующим свойством:


	- [IEaxGridHeaderSettings.Behaviour](../Interface/IEaxGridHeaderSettings/IEaxGridHeaderSettings.Behaviour.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Fixed. Фиксированный
		 размер строк/столбцов.


		 1
		 Capture. Захват
		 смежных строк/столбцов.


		 2
		 Insert. Вставка
		 новых строк/столбцов.
