# EaxWorkbookChartSerieType


## Описание


Перечисление EaxWorkbookChartSerieType
 используется для определения типа ряда на диаграмме в рабочей книге.


Используется следующими свойствами и методами:


	- [IEaxWorkbookChart.SerieTypeAvailable](../Interface/IEaxWorkbookChart/IEaxWorkbookChart.SerieTypeAvailable.htm);


	- [IEaxWorkbookChart.SerieTypeIndex](../Interface/IEaxWorkbookChart/IEaxWorkbookChart.SerieTypeIndex.htm);


	- [IEaxWorkbookChart.SerieTypeVisible](../Interface/IEaxWorkbookChart/IEaxWorkbookChart.SerieTypeVisible.htm).

## Комментарии


Элементы First и Last
 являются служебными и не используются для определения типа ряда на диаграмме.
