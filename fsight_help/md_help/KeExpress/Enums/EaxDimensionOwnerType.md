# EaxDimensionOwnerType


## Описание


Перечисление EaxDimensionOwnerType
 используется для определения типа источника измерения.


Используется следующим свойством:


	- [IEaxDimensionDescriptor.SliceType](../Interface/IEaxDimensionDescriptor/IEaxDimensionDescriptor.SliceType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 PivotSlice. Срез аналитической
		 области данных.


		 1
		 PrxSlice. Срез
		 обычной области данных.


		 2
		 Visualizer. Визуализатор.
