# EaxWorkbookChartType


## Описание


Перечисление EaxWorkbookChartType используется для определения типа диаграммы в рабочей книге.


Используется следующим свойством:


-
[IEaxWorkbookChart.Type](../Interface/IEaxWorkbookChart/IEaxWorkbookChart.Type.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Simple. Диаграмма для данных в таблице.


 1
 StatMethod. Диаграмма для данных, полученных в результате расчёта статистического метода.


 2
 DetermAggregation. Диаграмма для данных, полученных в результате расчёта агрегации.


 3
 SpliceMethod. Диаграмма для данных, полученных в результате сращивания рядов.
