# LanerBoxManagedOptions


## Описание


Перечисление LanerBoxManagedOptions содержит настройки источника данных и использует компонент LanerBox.


Используется следующим свойством:


-
[ILanerBox.DisabledManagedOptions](../Interface/ILanerBox/ILanerBox.DisabledManagedOptions.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Компонент использует только свои настройки.


 1
 LanerStartDate. Компонент использует дату начала отображения данных, заданную в источнике.


 2
 LanerEndDate. Компонент использует дату окончания отображения данных, заданную в источнике.


## Комментарии


Допускается объединение значений перечисления. Для этого указывайте элементы перечисления через «Or».
