# EaxChangeSelectionOperation


## Описание


Перечисление EaxChangeSelectionOperation
 используется для определения операции над выделенным объектом.


Используется следующими методами:


	- [IEaxVisualizer.CanChangeObjectivesSelection](../Interface/IEaxVisualizer/IEaxVisualizer.CanChangeObjectivesSelection.htm);


	- [IEaxVisualizer.ChangeObjectivesSelection](../Interface/IEaxVisualizer/IEaxVisualizer.ChangeObjectivesSelection.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 Exclude. Исключает
		 выделенный объект.


		 2
		 KeepOnly. Оставляет
		 только выделенный объект.
