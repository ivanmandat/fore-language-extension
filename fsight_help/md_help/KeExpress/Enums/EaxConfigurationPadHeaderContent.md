# EaxConfigurationPadHeaderContent


## Описание


Перечисление EaxConfigurationPadHeaderContent
 содержит типы элементов управления, которые могут присутствовать в заголовке
 панели.


Используется следующим свойством:


	- [IEaxConfigurationPadHeader.Content](../Interface/IEaxConfigurationPadHeader/IEaxConfigurationPadHeader.Content.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Custom. В заголовке
		 отображается элемент управления, заданный в свойстве [CustomContent](../Interface/IEaxConfigurationPadHeader/IEaxConfigurationPadHeader.CustomContent.htm).
		 Если свойство CustomContent
		 не задано, то в заголовке отображается только текст.


		 1
		 CheckBox. В заголовке
		 отображается флажок.
