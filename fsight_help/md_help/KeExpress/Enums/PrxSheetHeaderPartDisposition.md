# PrxSheetHeaderPartDisposition


## Описание


Перечисление PrxSheetHeaderPartDisposition
 содержит способы расположения части заголовка.


Используется следующим свойством:


	- [IHeaderPart.Disposition](../Interface/IHeaderPart/IHeaderPart.Disposition.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Нет.


		 1
		 Left. Левая часть.


		 2
		 Center. Центральная
		 часть.


		 3
		 Right. Правая часть.
