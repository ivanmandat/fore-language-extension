# EaxSliceCopyOptions


## Описание


Перечисление EaxSliceCopyOptions
 используется для определения параметров копирования среза аналитической
 области данных.


Используется следующим методом:


	- [IEaxDataAreaSlice.Copy](../Interface/IEaxDataAreaSlice/IEaxDataAreaSlice.Copy.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 All. Полностью.


		 1
		 NoSelection. Без
		 отметки.


		 2
		 NoSynsIndex.
		 Без синхронизации.


		 3
		 NoSelectionAndSyncIndex.
		 Без отметки и синхронизации.


		 4
		 NoHierarchyModifications.
		 С измерениями в исходном виде.


		 7
		 NoSelectionAndNoSynsIndexAndNoHierarchyModifications.
		 С измерениями в исходном виде, без отметки и синхронизации.


## Комментарии


Полностью. Копия среза будет
 полностью аналогична оригинальному срезу.


Без отметки. Копия среза
 унаследует от оригинального порядок расположения измерений и их синхронизацию.


Без синхронизации. Копия
 среза унаследует от оригинального порядок расположения измерений и отметку
 в измерениях.


Без отметки и синхронизации.
 Копия среза унаследует от оригинального только порядок расположения измерений.


С измерениями в исходном виде.
 Копия среза унаследует от оригинального все измерения в исходном виде
 без порядка их расположения.


С измерениями в исходном виде, без
 отметки и синхронизации. Копия среза унаследует от оригинального
 все измерения в исходном виде с порядком их расположения.
