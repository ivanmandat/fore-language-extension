# EaxPopupSortType


## Описание


Перечисление EaxPopupSortType
 используется для определения типа сортировки элементов измерения.


Используется следующим методом:


	- [IEaxSelectionPadContext.PopupSort](../Interface/IEaxSelectionPadContext/IEaxSelectionPadContext.PopupSort.htm).
