# EaxHeaderTitleContentType


## Описание


Перечисление EaxHeaderTitleContentType
 содержит наименования используемых измерений или атрибутов для [уголка
 таблицы](UiAnalyticalArea.chm::/TableView/Corner_show_dims.htm).


Используется свойством [IEaxGrid.HeaderTitleContentType](../Interface/IEaxGrid/IEaxGrid.HeaderTitleType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 DimensionName. Наименования
		 измерений.


		 2
		 DimensionAttributeName.
		 Наименования атрибутов измерений.


		 4
		 CustomNames. Пользовательское
		 наименование.
