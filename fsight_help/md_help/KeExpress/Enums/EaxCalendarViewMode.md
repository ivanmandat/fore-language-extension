# EaxCalendarViewMode


## Описание


Перечисление EaxCalendarViewMode
 используется для задания настроек режимов отображения календарных измерений.


Используется следующим свойством:


	- [IEaxDataAreaSliceProperties.DimCalendarViewMode](../Interface/IEaxDataAreaSliceProperties/IEaxDataAreaSliceProperties.DimCalendarViewMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 Elements. Режим отображения
		 только по элементам.


		 2
		 Levels. Режим
		 отображения только по уровням.


		 3
		 Full. Режим отображения
		 по уровням и элементам.
