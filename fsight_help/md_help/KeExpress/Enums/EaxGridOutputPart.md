# EaxGridOutputPart


## Описание


Перечисление EaxGridOutputPart
 используется для определения отображаемой области таблицы.


Используется следующим свойством:


	- [IEaxGridViewSettings.OutputPart](../Interface/IEaxGridViewSettings/IEaxGridViewSettings.OutputPart.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Ничего не отображается.


		 1
		 Data. Отображаются
		 только данные.


		 2
		 ColumnsHeader. Отображается
		 только шапка.


		 3
		 NoRowsHeader. Отображается
		 вся таблица, кроме боковика.


		 4
		 RowsHeader. Отображается
		 только боковик.


		 5
		 NoColumnsHeader. Отображается
		 вся таблица, кроме шапки.


		 6
		 NoData. Отображается
		 вся таблица, кроме данных.


		 7
		 Full. По умолчанию.
		 Таблица отображается полностью.
