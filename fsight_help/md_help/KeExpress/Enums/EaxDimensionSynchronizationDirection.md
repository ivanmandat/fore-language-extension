# EaxDimensionSynchronizationDirection


## Описание


Перечисление EaxDimensionSynchronizationDirection
 используется для определения типа связи, применяемой для синхронизации
 измерений.


Используется следующим свойством:


	- [IEaxSyncItem.SyncDirection](../Interface/IEaxSyncItem/IEaxSyncItem.SyncDirection.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 ProduceChangesOnly.
		 Связь только передает отметку в управляемые срезы.


		 2
		 ConsumeChangesOnly.
		 Связь только принимает отметку из срезов, с которыми настроена
		 синхронизация.


		 3
		 ProduceAndConsumeChanges.
		 Связь может как принимать, так и передавать отметку в управляемые
		 срезы.
