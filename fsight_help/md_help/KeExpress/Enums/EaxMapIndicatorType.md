# EaxMapIndicatorType


## Описание


Перечисление EaxMapIndicatorType
 используется для определения типа метрики старого типа карты.


Используется следующими свойствами:


	- [IEaxObject.IndicatorSelection](../Interface/IEaxObject/IEaxObject.IndicatorSelection.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 Color. Цвет территории
		 карты.


		 2
		 Depth. Высота
		 столбца, который формируется для территории, при трехмерном отображении
		 карты.


		 3
		 CompoundBar. Столбиковая
		 диаграмма.


		 4
		 CompoundPie. Круговая
		 диаграмма.


		 5
		 MarkerColor. Цвет маркеров
		 геолокации.


		 6
		 ArrowColor. Цвет стрелок.


		 7
		 ArrowThickness. Толщина
		 стрелок.


## Комментарии


Элементы данного перечисления передаются в свойство [IEaxObject.IndicatorSelection](../Interface/IEaxObject/IEaxObject.IndicatorSelection.htm)
 в качестве параметров.
