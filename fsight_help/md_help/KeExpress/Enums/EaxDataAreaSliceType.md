# EaxDataAreaSliceType


## Описание


Перечисление EaxDataAreaSliceType
 используется для определения типа среза аналитической области данных.


Используется следующими свойствами и методами:


	- [IEaxDataAreaSlices.Add](../Interface/IEaxDataAreaSlices/IEaxDataAreaSlices.Add.htm);


	- [IEaxDataAreaSlice.Type](../Interface/IEaxDataAreaSlice/IEaxDataAreaSlice.Type.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 Pivot. По умолчанию.
		 Таблица.


		 2
		 Laner. Рабочая
		 книга базы данных временных рядов в рядном режиме.


## Комментарии


Pivot. Используется для многомерных
 источников данных.


Laner. Используется для базы
 данных временных рядов в рядном режиме, то есть, если свойство [IEaxAnalyzeCore.Mode](../Interface/IEaxAnalyzeCore/IEaxAnalyzeCore.Mode.htm) принимает значение
 [EaxMode.Series](EaxMode.htm).
