# EaxDimPanelOrientation


## Описание


Перечисление EaxDimPanelOrientation
 содержит варианты расположения измерений в компоненте среды разработки
 - [ErAnalyzerDimPanel](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/ErAnalyzerDimPanel.htm).


Используется следующим свойством:


	- [IErAnalyzerDimPanel.Orientation](../Interface/IErAnalyzerDimPanel/IErAnalyzerDimPanel.Orientation.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Vertical. Располагать
		 вертикально.


		 1
		 Horizontal. Располагать
		 горизонтально.
