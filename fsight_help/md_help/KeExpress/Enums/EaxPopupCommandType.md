# EaxPopupCommandType


## Описание


Перечисление EaxPopupCommandType
 используется для определения действия над сворачивающейся иерархией.


Используется следующим методом:


	- [IEaxSelectionPadContext.PopupCommand](../Interface/IEaxSelectionPadContext/IEaxSelectionPadContext.PopupCommand.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 ExpandAll. Развернуть
		 всю иерархию.


		 1
		 CollapseAll. Свернуть
		 всю иерархию.
