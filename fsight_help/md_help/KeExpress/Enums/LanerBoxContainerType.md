# LanerBoxContainerType


## Описание


Перечисление LanerBoxContainerType содержит типы панели компонента среды разработки LanerResultsBox.


Используется следующими свойствами и методами:


-
[ILanerResultsBox.ContainerVisible](../Interface/ILanerResultsBox/ILanerResultsBox.ContainerVisible.htm);


-
[ILanerResultsBox.RefreshContainer](../Interface/ILanerResultsBox/ILanerResultsBox.RefreshContainer.htm);


-
[ILanerResultsBox.UserFormReplacePanel](../Interface/ILanerResultsBox/ILanerResultsBox.UserFormReplacePanel.htm).
