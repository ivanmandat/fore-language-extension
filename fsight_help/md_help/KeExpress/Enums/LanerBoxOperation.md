# LanerBoxOperation


## Описание


Перечисление LanerBoxOperation содержит операции, совершаемые с рабочей книгой.


Используется следующими свойствами и методами:


-
[ILanerBox.CanExecuteOperation](../Interface/ILanerBox/ILanerBox.CanExecuteOperation.htm);


-
[ILanerBox.ExecuteOperation](../Interface/ILanerBox/ILanerBox.ExecuteOperation.htm);


-
[ILanerBox.IsOptionSwitchedOn](../Interface/ILanerBox/ILanerBox.IsOptionSwitchedOn.htm).
