# EaxGridIndicatorType


## Описание


Перечисление EaxGridIndicatorType
 используется для определения типа метрики таблицы.


Используется следующим свойством:


	- [IEaxObject.IndicatorSelection](../Interface/IEaxObject/IEaxObject.IndicatorSelection.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 Value. Значение ячеек.


		 2
		 BackgroundColor. Цвет
		 фона ячеек.


		 3
		 FontColor. Цвет шрифта
		 ячеек.


		 4
		 FontSize. Размер шрифта
		 ячеек.


		 5
		 ValuePreffix. Префикс
		 для значений ячеек.


		 6
		 ValueSuffix. Суффикс
		 для значений ячеек.


		 7
		 ValueComment. Примечание
		 ячеек.


		 8
		 Footnote. Комментарий
		 к сноскам области данных.


## Комментарии


Префикс - это символьное выражение слева
 от значений ячейки.


Суффикс - это символьное выражение справа от значений ячейки.


Элементы данного перечисления передаются в свойство [IEaxObject.IndicatorSelection](../Interface/IEaxObject/IEaxObject.IndicatorSelection.htm)
 в качестве параметров.
