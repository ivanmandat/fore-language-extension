# EaxInteractiveMode


## Описание


Перечисление EaxInteractiveMode
 содержит режимы интерактивности экспресс-отчета, объединяющие режимы таблицы
 данных и диаграммы. Данный перечислимый тип используется следующими свойствами:


	- [IEaxAnalyzeCore.InteractiveMode](../Interface/IEaxAnalyzeCore/IEaxAnalyzeCore.InteractiveMode.htm);


	- [IEaxAnalyzeCore.IsInteractiveModeAvailable](../Interface/IEaxAnalyzeCore/IEaxAnalyzeCore.IsInteractiveModeAvailable.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Selection. Режим «Выделение»: таблица данных
		 находится в режиме чтения данных, диаграмма находится в режиме
		 выделения объектов диаграммы.


		 1
		 DrillDown. Режим «Проваливание»: таблица данных
		 находится в режиме чтения данных, диаграмма находится в режиме
		 гиперссылок.


		 2
		 AutoScrolling. Режим
		 «Синхронная прокрутка»:
		 таблица данных находится в режиме чтения данных, диаграмма
		 находится в режиме масштабирования.


		 3
		 ZoomAxisX. Режим «Масштабирование оси категорий»:
		 таблица данных находится в режиме чтения данных, диаграмма
		 находится в режиме масштабирования по оси X.
