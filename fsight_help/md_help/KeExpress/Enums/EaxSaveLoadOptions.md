# EaxSaveLoadOptions


## Описание


Перечисление EaxSaveLoadOptions
 содержит параметры загрузки/сохранения экспресс-отчета.


Используется следующим методом:


	- [IEaxAnalyzer.LoadFromStream](../Interface/IEaxAnalyzer/IEaxAnalyzer.LoadFromStream.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Параметры не
		 определены.


		 1
		 NoDataSource. Не загружать
		 источник данных.
