# EaxMode


## Описание


Перечисление EaxMode содержит
 режимы отображения рабочей области базы данных временных рядов.


Используется следующим свойством:


	- [IEaxAnalyzeCore.Mode](../Interface/IEaxAnalyzeCore/IEaxAnalyzeCore.Mode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 MultiDimension. Многомерный
		 режим.


		 1
		 Series. Рядный режим.


## Комментарии


Рядный режим. Значения показателей
 выводятся в таблице по строкам друг под другом. В заголовках строк выводятся
 значения атрибутов, идентифицирующих показатели, в заголовках столбцов
 - календарь.


Многомерный режим. В отличие
 от рядного режима, таблица которого строится в зависимости от добавленных
 показателей, в многомерном режиме таблица строится по отметке в соответствующих
 измерениях. Срезы в данном случае содержат значения атрибутов. Изменение
 отметки в измерениях приводит к добавлению или удалению ряда из таблицы.
 В многомерном режиме измерения можно произвольно перемещать между заголовками
 строк и столбцов. Многомерный режим иначе называется OLAP-представлением
 базы данных временных рядов.
