# EaxOpenCubeOptions


## Описание


Перечисление EaxOpenCubeOptions
 используется для определения режима открытия куба.


Используется следующим методом:


	- [IEaxAnalyzeCore.OpenCubeEx](../Interface/IEaxAnalyzeCore/IEaxAnalyzeCore.OpenCubeEx.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Использовать
		 только данные куба.


		 1
		 DefaultView. Использовать
		 данные и сохраненное представление куба (при наличии сохраненного
		 представления).


		 2
		 DataAndSelection. Использовать
		 данные источника и сохраненную отметку.
