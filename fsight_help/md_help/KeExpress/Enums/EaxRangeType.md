# EaxRangeType


## Описание


Перечисление EaxRangeType содержит
 типы составных частей таблицы экспресс-отчёта.


Используется следующим свойством:


	- [IEaxGrid.SpecificRange](../Interface/IEaxGrid/IEaxGrid.SpecificRange.htm).
