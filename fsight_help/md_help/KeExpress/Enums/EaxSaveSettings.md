# EaxSaveSettings


## Описание


Перечисление EaxSaveSettings
 используется для определения параметров сохранения визуализаторов экспресс-отчёта.


Используется следующим свойством:


	- [IEaxAnalyzeCore.SaveSettingsToXml](../Interface/IEaxAnalyzeCore/IEaxAnalyzeCore.SaveSettingsToXml.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 Chart. Сохраняются
		 параметры диаграммы.


		 2
		 Grid. Сохраняются
		 параметры таблицы.


		 4
		 Map. Сохраняются параметры
		 карты (старая версия).


		 8
		 Speedometer. Сохраняются

		 BubbleChart. Сохраняются
		 параметры пузырьковой диаграммы (старая версия).

		 параметры пузырькового дерева.

		 128
		 MapChart. Сохраняются
		 параметры карты (новая версия).

		 параметры пузырьковой диаграммы (новая версия).

		 AllObjects. Сохраняются
		 параметры всех визуализаторов.

		 IgnoreEVAOnLoad. EVA-атрибуты
		 визуализаторов игнорируются. Данное значение используется только

		 при загрузке параметров визуализаторов экспресс-отчёта.


## Комментарии


EVA-атрибуты - это атрибуты, отвечающие за доступность (Enabled), видимость
 (Visible) и активность (Active) визуализатора.
