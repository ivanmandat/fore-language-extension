# EaxDataAreaSliceCalculationOptions


## Описание


Перечисление EaxDataAreaSliceCalculationOptions
 используется для определения параметров вычисления элементов среза данных.


Используется следующим свойством:


	- [IEaxDataAreaSlice.CalculationOptions](../Interface/IEaxDataAreaSlice/IEaxDataAreaSlice.CalculationOptions.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. По умолчанию.
		 Без использования параметров.


		 1
		 TreatNullsAsZeros. Заменять
		 пустые значения нулями.
