# EaxGridUpdateDataOptions


## Описание


Перечисление EaxGridUpdateDataOptions
 используется для определения варианта обновления.


Используется следующими свойствами и методами:


	- [IEaxGrid.UpdateData](../Interface/IEaxGrid/IEaxGrid.UpdateData.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Default. Обновление
		 данных.


		 1
		 DataRights. Обновление
		 прав доступа.


## Комментарии


Обновление происходит только для переданной отметки.
