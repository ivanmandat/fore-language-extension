# EaxProblemSource


## Описание


Перечисление EaxProblemSource
 используется для определения источника данных для анализа.


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Pivot. В качестве источника
		 данных используется таблица.


		 1
		 Laner. В качестве
		 источника данных используется рабочая книга базы данных временных
		 рядов.
