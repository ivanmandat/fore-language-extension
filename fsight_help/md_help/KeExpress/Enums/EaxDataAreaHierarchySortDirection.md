# EaxDataAreaHierarchySortDirection


## Описание


Перечисление EaxDataAreaHierarchySortDirection
 используется для определения направления сортировки элементов альтернативной
 иерархии.


Используется следующим свойством:


	- [IEaxDataAreaHierarchy.SortDirection](../Interface/IEaxDataAreaHierarchy/IEaxDataAreaHierarchy.SortDirection.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. По умолчанию.
		 Автоматическая сортировка.


		 1
		 Asc. По возрастанию.


		 2
		 Desc. По убыванию.
