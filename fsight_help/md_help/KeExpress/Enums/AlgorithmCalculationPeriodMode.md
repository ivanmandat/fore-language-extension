# AlgorithmCalculationPeriodMode


## Описание


Перечисление AlgorithmCalculationPeriodMode
 содержит варианты формирования отметки, которая будет использоваться для
 расчёта алгоритма.


Используется следующими свойствами и методами:


	- [IEaxDataAreaExternalTransformation.CalculationPeriodMode](../Interface/IEaxDataAreaExternalTransformation/IEaxDataAreaExternalTransformation.CalculationPeriodMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Intersect. Пересечение
		 отметки из алгоритма и среза данных.


		 1
		 FromAlgorithm. Пересечение
		 отметок из алгоритма и среза данных по всем измерениям, кроме
		 календарного измерения. Период расчёта будет взят из алгоритма.


		 2
		 AllFromAlgorithm. Отметка
		 из алгоритма расчёта.
