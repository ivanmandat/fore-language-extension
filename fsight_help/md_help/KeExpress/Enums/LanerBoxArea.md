# LanerBoxArea


## Описание


Перечисление LanerBoxArea содержит области компонента LanerBox.


Используется следующим методом:


-
[ILanerBox.HitTest](../Interface/ILanerBox/ILanerBox.HitTest.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Область не определена.


 1
 Tabsheet. Таблица данных.


 2
 MethodsBar. Панель моделирования.


 3
 Toolbar. Панель инструментов.


 4
 DimPanel. Панель размерностей.


## Комментарии


Панель размерностей отображается только в многомерном режиме компонента LanerBox. Для скрытия/отображения панелей используйте методы:


-
[ILanerBox.ShowDimPanel](../Interface/ILanerBox/ILanerBox.ShowDimPanel.htm);


-
[ILanerBox.ShowMethodsBar](../Interface/ILanerBox/ILanerBox.ShowMethodsBar.htm);


-
[ILanerBox.ShowToolbar](../Interface/ILanerBox/ILanerBox.ShowToolbar.htm).
