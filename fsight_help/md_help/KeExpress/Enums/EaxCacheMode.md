# EaxCacheMode


## Описание


Перечисление EaxCacheMode содержит
 виды механизмов кеширования, которые могут использоваться при работе с
 источниками данных.


Используется следующими свойствами и методами:


	- [IEaxDataAreaSlice.DataSourceCacheMode](../Interface/IEaxDataAreaSlice/IEaxDataAreaSlice.DataSourceCacheMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Механизм кеширования
		 не определён.


		 1
		 Blob. Кеширование
		 In-Memory с использованием [глобального
		 кеша](UiNav.chm::/02_Navigator/Cache_In_Blob.htm).


		 2
		 File. Кеширование In-Memory
		 с использованием файлового кеша.
