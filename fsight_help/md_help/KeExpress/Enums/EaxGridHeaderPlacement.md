# EaxGridHeaderPlacement


## Описание


Перечисление EaxGridHeaderPlacement
 используется для определения типа настройки размещения заголовка.


Используется следующим свойством:


	- [IEaxGridHeaderSettings.Placement](../Interface/IEaxGridHeaderSettings/IEaxGridHeaderSettings.Placement.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Simple. Простое размещение.


		 1
		 SkipDuble. Пропускать
		 повторяющиеся.


		 2
		 Merge. Объединить
		 ячейки.
