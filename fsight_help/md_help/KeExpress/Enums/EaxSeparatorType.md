# EaxSeparatorType


## Описание


Перечисление EaxSeparatorType
 используется для определения типа разделителя между атрибутами таблицы
 в ячейках или их примечаниях.


Используется следующими свойством:


	- [IEaxGrid.IndicatorSeparator](../Interface/IEaxGrid/IEaxGrid.IndicatorSeparator.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Prefix. Разделитель
		 префикса.


		 1
		 AfterPrefix. Разделитель
		 после префикса.


		 2
		 Suffix. Разделитель
		 суффикса.


		 3
		 BeforeSuffix. Разделитель
		 перед суффиксом.


		 4
		 Comment. Разделитель
		 значения в примечании ячеек.


		 5
		 BeforePrefix. Разделитель
		 перед префиксом.


		 6
		 AfterSuffix. Разделитель
		 после суффикса.


		 7
		 BeforeComment. Разделитель
		 перед значением в примечании ячеек.


		 8
		 AfterComment.
		 Разделитель после значения в примечании ячеек.


		 9
		 ValueFootnote. Разделитель
		 для атрибутов в сносках области данных.


		 10
		 HeaderFootnote. Разделитель
		 для атрибутов в сносках заголовков.
