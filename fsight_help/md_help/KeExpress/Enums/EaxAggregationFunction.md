# EaxAggregationFunction


## Описание


Перечисление EaxAggregationFunction
 используется для определения метода агрегации, применяемого для расчета
 отображаемого значения.


Используется следующими свойствами:


	- [IEaxSpeedometer.AggregationFunction](../Interface/IEaxSpeedometer/IEaxSpeedometer.AggregationFunction.htm);


	- [IEaxDataAreaSlice.IndicatorAggregation](../Interface/IEaxDataAreaSlice/IEaxDataAreaSlice.IndicatorAggregation.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Sum. Сумма значений.


		 1
		 Average. Среднее
		 арифметическое значение.


		 2
		 Min. Минимальное значение.


		 3
		 Max. Максимальное значение.


		 4
		 NONE. Агрегация не
		 используется.


## Комментарии


Для спидометра агрегация используется, если в отчете выделено несколько
 рядов с данными.
