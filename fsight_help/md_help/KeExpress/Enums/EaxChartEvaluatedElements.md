# EaxChartEvaluatedElements


## Описание


Перечисление EaxChartEvaluatedElements
 содержит типы вычисляемых элементов, которые могут быть отображены на
 диаграмме вместе с фактическими данными.


Используется следующим свойством:


	- [IEaxChart.EvaluatedElements](../Interface/IEaxChart/IEaxChart.EvaluatedElements.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Не отображать
		 на диаграмме вычисляемые элементы, отображать только фактические
		 данные.


		 1
		 Totals. Отображать
		 фактические данные и рассчитанные итоговые значения.


		 2
		 ParetoOthers. Отображать
		 фактические данные и данные, которые не попали в результирующий
		 набор распределения и были сагрегированны.
