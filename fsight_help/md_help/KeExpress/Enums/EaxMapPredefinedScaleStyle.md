# EaxMapPredefinedScaleStyle


## Описание


Перечисление EaxMapPredefinedScaleStyle
 содержит стандартные стили распределения цветов, которые будут использоваться
 для оформления территорий в зависимости от значений.


Используется следующими свойствами и методами:


	- [IEaxMapChart.AssignPredefinedScaleStyle](../Interface/IEaxMapChart/IEaxMapChart.AssignPredefinedScaleStyle.htm).
