# EaxDataDrillMode


## Описание


Перечисление EaxDataDrillMode
 используется для определения способа изменения отметки в [расшифровке
 элементов измерения](UiAnalyticalArea.chm::/Hyperlinks/Working_with_hyperlinks.htm).


Используется следующим свойством:


	- [IEaxDrillSettings.Mode](../Interface/IEaxDrillSettings/IEaxDrillSettings.Mode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. По умолчанию.
		 Не выбран.


		 1
		 Replace. Замена
		 отметки выбранным элементом.


		 2
		 Invert. Инвертирование
		 отметки для выбранного элемента.


		 4
		 Add. Добавление в отметку
		 выбранного элемента.
