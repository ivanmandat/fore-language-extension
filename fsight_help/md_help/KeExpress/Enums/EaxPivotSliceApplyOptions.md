# EaxPivotSliceApplyOptions


## Описание


Перечисление EaxPivotSliceApplyOptions
 используется для определения настроек применения импортированных параметров
 в срез аналитической области данных.


Используется следующим методом:


	- [IEaxDataAreaPivotSlice.Apply](../Interface/IEaxDataAreaPivotSlice/IEaxDataAreaPivotSlice.Apply.htm).
