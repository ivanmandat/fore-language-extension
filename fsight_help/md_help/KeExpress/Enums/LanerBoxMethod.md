# LanerBoxMethod


## Описание


Перечисление LanerBoxMethod
 содержит методы преобразования ряда данных в рабочей книге.


Используется следующими свойствами и методами:


	- [ILanerBox.CanExecuteMethod](../Interface/ILanerBox/ILanerBox.CanExecuteMethod.htm);


	- [ILanerBox.ExecuteMethodEx](../Interface/ILanerBox/ILanerBox.ExecuteMethodEx.htm).
