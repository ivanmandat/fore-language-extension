# EaxPointChartMode


## Описание


Перечисление EaxPointChartMode
 содержит способы передачи данных при построении точечной
 диаграммы.


Используется следующим свойством:


	- [IEaxChart.PointChartMode](../Interface/IEaxChart/IEaxChart.PointChartMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 XYY. У всех рядов значения
		 по оси Х будут одинаковыми, и будут располагаться либо в верхней
		 строке (при размещении рядов в строках), либо в первом столбце
		 (при размещении рядов в столбцах). В остальных строках / столбцах
		 содержатся значения рядов по оси Y.


		 1
		 XYXY. Значения по осям
		 X и Y будут чередоваться в строках / столбцах (в зависимости от
		 расположения рядов). Если ряды расположены в строках, то чередование
		 будет происходить сверху вниз, а если ряды расположены в столбцах,
		 то слева направо.


		 2
		 YXX. У всех рядов значения
		 по оси Y будут одинаковыми, и будут располагаться либо в верхней
		 строке (при размещении рядов в строках), либо в первом столбце
		 (при размещении рядов в столбцах). В остальных строках / столбцах
		 содержатся значения рядов по оси X.


		 3
		 YXYX. Значения по осям
		 Y и X будут чередоваться в строках / столбцах (в зависимости от
		 расположения рядов). Если ряды расположены в строках, то чередование
		 будет происходить сверху вниз, а если ряды расположены в столбцах,
		 то слева направо.
