# EaxBubbleChartIndicators


## Описание


Перечисление EaxBubbleChartIndicators
 используется для определения типа метрики пузырьковой диаграммы.


Используется следующими свойствами:


	- [IEaxObject.IndicatorSelection](../Interface/IEaxObject/IEaxObject.IndicatorSelection.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 X. Тип метрики «X».


		 1
		 Y. Тип метрики
		 «Y».


		 2
		 Radial. Тип метрики
		 «Размер».


		 3
		 Color. Тип метрики
		 «Цвет».
