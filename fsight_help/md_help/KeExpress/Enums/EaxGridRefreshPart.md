# EaxGridRefreshPart


## Описание


Перечисление EaxGridRefreshPart
 содержит области обновления в таблице регламентного отчета.


Используется следующим методом:


	- [IEaxGrid.RefreshPart](../Interface/IEaxGrid/IEaxGrid.RefreshPart.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 All. Обновлять всю
		 таблицу.


		 1
		 WidthAndHeight. Производить
		 только автоподбор размера ячеек.


		 2
		 TableStyle. Обновлять
		 только чередующийся стиль таблицы.


		 4
		 InternalStyle. Обновлять
		 только стиль данных.
