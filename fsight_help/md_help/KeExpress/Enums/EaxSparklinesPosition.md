# EaxSparklinesPosition


## Описание


Перечисление EaxSparklinesPosition
 содержит области построения спарклайнов в таблице экспресс-отчета.


Используется следующим свойством:


	- [IEaxGrid.SparklinesPosition](../Interface/IEaxGrid/IEaxGrid.SparklinesPosition.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Спарклайны отсутствуют.


		 1
		 Rows. Спарклайны
		 отображаются по строкам.


		 2
		 Columns. Спарклайны
		 отображаются по столбцам.


		 4
		 Totals. Спарклайны
		 отображаются для итогов.
