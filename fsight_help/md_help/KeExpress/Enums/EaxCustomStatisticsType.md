# EaxCustomStatisticsType


## Описание


Перечисление EaxCustomStatisticsType используется для определения пользовательской статистической характеристики.


Используется следующими свойствами и методами:


-
[IEaxStatistics.CustomStatisticsVisible](../Interface/IEaxStatistics/IEaxStatistics.CustomStatisticsVisible.htm).


## Допустимые значения


 Значение
 Краткое описание


 1
 StartDate. Дата начала.


 2
 EndDate. Дата окончания.


 3
 FStartDate. Самая ранняя дата начала.


 4
 LStartDate. Самая поздняя дата начала.


 5
 FEndDate. Самая ранняя дата окончания.


 6
 LEndDate. Самая поздняя дата окончания.


 7
 Scale. Масштаб.
