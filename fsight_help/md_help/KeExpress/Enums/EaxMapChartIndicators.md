# EaxMapChartIndicators


## Описание


Перечисление EaxMapChartIndicators
 используется для определения типа метрики карты, поддерживающей технологию
 OpenGL.


Используется следующими свойствами:


	- [IEaxObject.IndicatorSelection](../Interface/IEaxObject/IEaxObject.IndicatorSelection.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Color. Тип метрики
		 «Цвет».


		 1
		 Height. Тип метрики
		 «Объем» для карты в 3D-режиме.


		 2
		 ArrowColor. Тип метрики
		 «Цвет стрелок» для стрелок.


		 3
		 ArrowWidth. Тип метрики
		 «Размер стрелок» для стрелок.


		 4
		 MarkerColor. Тип метрики
		 «Цвет маркера» для маркеров
		 геолокации.


		 5
		 BubbleColor. Тип метрики
		 «Цвет пузырьков» для пузырьковых
		 показателей.


		 6
		 PieAngle. Тип метрики
		 «Круговой» для круговых
		 показателей.


		 7
		 BubbleRadius. Тип метрики
		 «Размер пузырьков» для
		 пузырьковых показателей.


		 8
		 BarHeight. Тип метрики
		 «Столбиковый» для столбиковых
		 показателей.
