# WbkConfigurationPanelContext


## Описание


Перечисление WbkConfigurationPanelContext
 используется для определения контекста, отображаемого в компоненте WorkbookConfigurationPanel.


Используется следующим свойством:


	- [IWorkbookConfigurationPanel.Context](../Interface/IWorkbookConfigurationPanel/IWorkbookConfigurationPanel.Context.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Контекст не определен.


		 1
		 Workbook. Контекст
		 для настройки рабочей книги в целом.


		 3
		 SourceRow. Контекст
		 для настройки ряда данных.


		 4
		 CalcRow. Контекст
		 для настройки вычисляемого ряда.


		 5
		 DataPoint. Контекст
		 для настройки наблюдения.


		 7
		 Table. Контекст
		 для настройки таблицы данных.


		 8
		 Chart. Контекст для
		 настройки диаграммы.


		 9
		 Map. Контекст для настройки
		 атрибутов.


		 10
		 AttrTree. Контекст
		 для настройки дерева рядов.


		 11
		 AttrTreeSerie. Контекст
		 для настройки ряда из дерева рядов.


		 12
		 Factor. Контекст
		 для настройки факторов.


		 13
		 NoSeries. Контекст
		 для отсутствия рядов данных.


## Комментарии


Контекст - это набор вкладок компонента WorkbookConfigurationPanel,
 предназначенный для настройки какой-либо части рабочей книги.
