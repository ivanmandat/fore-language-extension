# EaxObjectType


## Описание


Перечисление EaxObjectType содержит
 тип объекта экспресс-отчета.


Используется следующими свойствами:


	- [IEaxObject.Type](../Interface/IEaxObject/IEaxObject.Type.htm);


## Допустимые значения


		 Значение
		 Краткое описание


		 -1
		 None. Тип объекта не
		 определен.


		 0
		 Grid. Таблица.


		 1
		 Chart. Диаграмма.


		 2
		 Map. Карта.


		 3
		 BubbleChart. Пузырьковая
		 диаграмма.


		 4
		 Statistics. Панель
		 статистик.


		 5
		 Speedometer. Спидометр.


		 6
		 BubbleTree. Пузырьковое
		 дерево.


		 7
		 TreeMap. Плоское дерево.


		 8
		 MapChart. Карта, использующая
		 для отрисовки технологию OpenGL.
