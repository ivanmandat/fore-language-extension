# EaxChartSingleCellRangeMode


## Описание


Перечисление EaxChartSingleCellRangeMode
 содержит режимы построения диаграммы при одной выделенной ячейке.


Используется следующим свойством:


	- [IEaxChart.SingleCellRangeMode](../Interface/IEaxChart/IEaxChart.SingleCellRangeMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 EntireGrid. Построение
		 графика по всей области данных.


		 1
		 SingleCell. Построение
		 графика по одной ячейке.


		 2
		 SingleRow. Построение
		 графика по одной строке, в которой находится выделенная ячейка.


		 2
		 SingleColumn. Построение
		 графика по одному столбцу, в котором находится выделенная ячейка.
