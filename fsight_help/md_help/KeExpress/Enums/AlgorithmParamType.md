# AlgorithmParamType


## Описание


Перечисление AlgorithmParamType
 содержит типы управления параметром алгоритма расчёта.


Используется следующими свойствами и методами:


	- [IAlgorithmParam.Type](../Interface/IAlgorithmParam/IAlgorithmParam.Type.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Constant. Передавать
		 в параметр константное значение.


		 1
		 ObjectParam. Передавать
		 в параметр алгоритма значение параметра текущего объекта.
