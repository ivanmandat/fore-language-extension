# Введение в сборку Express


В данном разделе представлено описание сборки Express,
 предназначенной для работы с инструментами «[Аналитические
 запросы (OLAP)](uiexpress.chm::/purpose/UiExpress_Purpose.htm)» и «[Анализ
 временных рядов](UiDw.chm::/UiDw_Title.htm)» на языке Fore.
 В интерфейсах сборки реализованы все необходимые средства для настройки
 параметров экспресс-отчёта и рабочей книги, выбора источника данных, работы
 с аналитической областью, настройки визуализаторов.


Основными интерфейсами сборки Express
 являются:


	- [IEaxAnalyzer](../Interface/IEaxAnalyzer/IEaxAnalyzer.htm).
	 Интерфейс содержит свойства и методы для работы с экспресс-отчётом;


	- [IEaxWorkbook](../Interface/IEaxWorkbook/IEaxWorkbook.htm).
	 Интерфейс содержит свойства для работы с рабочей книгой.
