# ChartAutoScaleMode


## Описание


Перечисление ChartAutoScaleMode содержит режимы масштабирования оси.


Используется следующими свойствами и методами:


-
[IChartAxis.AutoScale](../Interface/IChartAxis/IChartAxis.AutoScale.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Масштабируемая шкала не используется.


 1
 Clustering. Кластеризация. При выборе данного режима масштабирования возможно определение параметра масштабирования: [IChartAxis.AutoScaleParameter](../Interface/IChartAxis/IChartAxis.AutoScaleParameter.htm).


 2
 Distributed. Распределённое масштабирование. При выборе данного режима масштабирования возможно определение параметра масштабирования: [IChartAxis.AutoScaleParameter](../Interface/IChartAxis/IChartAxis.AutoScaleParameter.htm).


 3
 Logarithmic. Логарифмическая шкала.


 4
 User. Пользовательская шкала.
