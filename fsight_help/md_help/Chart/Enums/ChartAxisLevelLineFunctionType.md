# ChartAxisLevelLineFunctionType


## Описание


Перечисление ChartAxisLevelLineFunctionType
 используется для определения типа функции, по которой строится линия
 уровня оси значений на диаграмме.


Используется следующим свойством:


	- [IChartAxisLevelLine.ValueFunction](../Interface/IChartAxisLevelLine/IChartAxisLevelLine.ValueFunction.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Constant. Функция не
		 используется.


		 1
		 Max. Максимум.


		 2
		 Min. Минимум.


		 3
		 Summ. Сумма.


		 4
		 Median. Медиана.


		 5
		 Mean. Среднее.
