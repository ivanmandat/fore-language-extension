# ChartImageDrawStyle


## Описание


Перечисление ChartImageDrawStyle содержит типы положения рисунка в заливаемой области.


Используется следующими свойствами и методами:


-
[IChartBackgroundInfo.ImageDrawStyle](../Interface/IChartBackgroundInfo/IChartBackgroundInfo.ImageDrawStyle.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Normal. Разместить как обычно.


 1
 Center. Разместить по центру.


 2
 Tile. Размножить.


 3
 TileFlipX. Размножить с отражением по горизонтали.


 4
 TileFlipY. Размножить с отражением по вертикали.


 5
 TileFlipXY. Размножить с отражением по горизонтали и по вертикали.


 6
 Stretch. Растянуть.
