# ChartLabelPosition


## Описание


Перечисление ChartLabelPosition
 содержит варианты положений подписи данных на столбике, секторе, линии,
 области и точке.


Используется следующим свойством:


	- [IChartLabel.Position](../Interface/IChartLabel/IChartLabel.Position.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Custom. Пользовательское
		 положение подписи.


		 1
		 Center. В центре. Вариант
		 доступен для всех типов рядов.


		 2
		 PeakInside. У вершины
		 внутри. Определяет положение подписи данных для столбика и сектора.


		 3
		 PeakOutside. У вершины
		 снаружи. Определяет положение подписи данных для столбика и сектора.


		 4
		 BaseInside. У основания
		 внутри. Определяет положение подписи данных для столбика и сектора.


		 5
		 Left. Cлева. Определяет
		 положение подписи данных для линии, области и точки.


		 6
		 Top. Cверху. Определяет
		 положение подписи данных для линии, области и точки.


		 7
		 Right. Cправа. Определяет
		 положение подписи данных для линии, области и точки.


		 8
		 Bottom. Cнизу. Определяет
		 положение подписи данных для линии, области и точки.


		 9
		 AutoPlacement. Автоматический
		 режим размещения подписей рядов диаграммы.
