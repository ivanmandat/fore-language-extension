# ChartStyle


## Описание


Перечисление ChartStyle содержит типы шкал диаграммы.


Используется следующими свойствами и методами:


-
[IChart.Style](../Interface/IChart/IChart.Style.htm)


-
[IUiChart.Style](../Interface/IUiChart/IUiChart.Style.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Absolute. Абсолютная шкала.


 1
 Relative. Относительная шкала.


 2
 Percent. Процентная шкала.


 3
 Mixed. Смешанная шкала.
