# ChartDateAxisUnit


## Описание


Перечисление ChartDateAxisUnit
 используется для определения типа подписей оси дат.


Используется следующими свойствами:


	- [IChartDateAxisSettings.BaseUnit](../Interface/IChartDateAxisSettings/IChartDateAxisSettings.BaseUnit.htm);


	- [IChartDateAxisSettings.MajorTickUnit](../Interface/IChartDateAxisSettings/IChartDateAxisSettings.MajorTickUnit.htm);


	- [IChartDateAxisSettings.MinorTickUnit](../Interface/IChartDateAxisSettings/IChartDateAxisSettings.MinorTickUnit.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Auto. Автоматическое
		 определение подписей оси на основе данных источника.


		 1
		 Years. Подписи оси
		 - года.


		 2
		 Month. Подписи
		 оси - месяцы.


		 3
		 Days. Подписи
		 оси - дни.
