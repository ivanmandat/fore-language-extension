# Scene3DProjectionType


## Описание


Перечисление Scene3DProjectionType содержит типы проекций трехмерной сцены.


Используется следующими свойствами и методами:


-
[IScene3DProjection.Type](../Interface/IScene3DProjection/IScene3DProjection.Type.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Perspective. Перспективная проекция.


 1
 Ortho. Ортогональная проекция.
