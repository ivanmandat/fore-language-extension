# ChartLegendAlignment


## Описание


Перечисление ChartLegendAlignment содержит типы размещения легенды на области построения.


Используется следующим свойством:


-
[IChartLegend.Alignment](../Interface/IChartLegend/IChartLegend.Alignment.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Left. Слева.


 1
 Top. Сверху.


 2
 Right. Справа.


 3
 Bottom. Снизу.


 4
 LeftTop. В верхнем левом углу.


 5
 RightTop. В правом верхнем углу.


 6
 LeftBottom. В нижнем левом углу.


 7
 RightBottom. В правом нижнем углу.
