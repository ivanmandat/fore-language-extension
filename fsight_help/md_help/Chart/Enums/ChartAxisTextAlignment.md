# ChartAxisTextAlignment


## Описание


Данный перечислимый тип определяет способ выравнивания текста относительно оси.


Используется следующими свойствами и методами:


-
[IChartAxis.TextAlignment](../Interface/IChartAxis/IChartAxis.TextAlignment.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Near. Ближе к оси.


 1
 Center. По центру.


 2
 Far. Дальше от оси.
