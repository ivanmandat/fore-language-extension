# ChartDisplayMode


## Описание


Перечисление ChartDisplayMode
 используется для определения режима отображения диаграммы.


Используется следующими свойством:


	- [IChart.DisplayMode](../Interface/IChart/IChart.DisplayMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Mode2D. Двумерный режим.
		 Используется для всех типов диаграмм.


		 1
		 Mode3D. Трехмерный
		 режим. Используется для линейных диаграмм, гистограмм, диаграмм
		 с областями, смешанных и круговых диаграмм.


		 2
		 ModeManhattan3D. Трехмерный
		 режим с отображением рядов в виде отдельных столбиков, сгруппированных
		 по категориям. Используется только для объемных гистограмм.
