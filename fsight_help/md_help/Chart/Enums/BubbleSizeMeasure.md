# BubbleSizeMeasure


## Описание


Перечисление BubbleSizeMeasure используется
 для задания величины пузырька.


Используется следующим свойством:


	- [IChartBubbleInfo.RepresentedBubbleSizeMeasure](../Interface/IChartBubbleInfo/IChartBubbleInfo.RepresentedBubbleSizeMeasure.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Area. Задает площадь
		 пузырька.


		 1
		 Width. Задает ширину
		 пузырька.
