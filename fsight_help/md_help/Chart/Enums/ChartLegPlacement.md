# ChartLegPlacement


## Описание


Перечисление ChartLegPlacement содержит варианты размещения элементов легенды.


Используется следующим свойством:


-
[IChartLegend.ItemPlacement](../Interface/IChartLegend/IChartLegend.ItemPlacement.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Column. Элементы размещаются в одну колонку и выравниваются по левому краю.


 1
 InLine. Элементы размещаются в одну строку.


 2
 Auto. Число строк и столбцов рассчитывается автоматически.


 3
 CustomColumns. Число столбцов задается пользователем, используйте свойство [IChartLegend.ColCount](../Interface/IChartLegend/IChartLegend.ColCount.htm).


 4
 CustomRows. Число строк задается пользователем, используйте свойство [IChartLegend.RowCount](../Interface/IChartLegend/IChartLegend.RowCount.htm).
