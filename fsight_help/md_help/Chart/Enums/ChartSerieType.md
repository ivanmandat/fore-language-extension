# ChartSerieType


## Описание


Перечисление ChartSerieType содержит типы ряда. Тип ряда можно изменять только у смешанных и лепестковых диаграмм.


Используется следующими свойствами и методами:


-
[IChartSerie.SerieType](../Interface/IChartSerieBase/IChartSerieBase.SerieType.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Lines. Линейный.


 1
 Bars. Столбиковый. Данный типа ряда нельзя использовать для ряда лепестковой диаграммы.


 2
 Areas. С областями.
