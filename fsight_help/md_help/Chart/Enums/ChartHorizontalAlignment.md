# ChartHorizontalAlignment


## Описание


Перечисление ChartHorizontalAlignment
 содержит типы положения подписи данных по горизонтали.


Используется следующими свойствами и методами:


	- [IChartAxis.TextHorizontalAlignment](../Interface/IChartAxis/IChartAxis.TextHorizontalAlignment.htm);


	- [IChartCaption.HorizontalAlignment](../Interface/IChartCaption/IChartCaption.HorizontalAlignment.htm);


	- [IChartHint.HorizontalAlignment](../Interface/IChartHint/IChartHint.HorizontalAlignment.htm);


	- [IChartLabel.HorizontalHotspotAlignment](../Interface/IChartLabel/IChartLabel.HorizontalHotspotAlignment.htm).
