# ChartAxisStepMode


## Описание


Перечисление ChartAxisStepMode содержит способы расчета делений оси.


Используется следующими свойствами и методами:


-
[IChartAxis.StepMode](../Interface/IChartAxis/IChartAxis.StepMode.htm)


## Допустимые значения


 Значение
 Краткое описание


 1
 Auto. Автоматический расчет делений.


 2
 FixedIntervals. Фиксированное количество интервалов. При выборе данного способа расчета делений необходимо задать свойство [IChartAxis.IntervalsCount](../Interface/IChartAxis/IChartAxis.IntervalsCount.htm).


 3
 FixedStep. Фиксированный шаг (цена) деления. При выборе данного способа расчета делений необходимо задать свойство [IChartAxis.StepValue](../Interface/IChartAxis/IChartAxis.StepValue.htm).


 4
 FixedDistance. Минимальное расстояние между значениями интервала (мм). При выборе данного способа расчета делений необходимо задать свойство [IChartAxis.StepDistance](../Interface/IChartAxis/IChartAxis.StepDistance.htm).
