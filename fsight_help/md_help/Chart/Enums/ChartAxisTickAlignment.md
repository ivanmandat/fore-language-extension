# ChartAxisTickAlignment


## Описание


Перечисление ChartAxisTickAlignment содержит варианты расположения делений относительно осей.


Используется следующими свойствами и методами:


-
[IChartAxis.TickAlignment](../Interface/IChartAxis/IChartAxis.TickAlignment.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Center. Ось пересекает деление по середине (по умолчанию).


 1
 Inside. Деление отображается целиком внутри области построения.


 2
 Outside. Деление отображается целиком снаружи области построения.
