# ChartLegendAxisIcon


## Описание


Перечисление ChartLegendAxisIcon содержит варианты отображения пиктограмм для рядов в легенде.


Используется следующими свойствами и методами:


-
[IChartLegend.AxisIcon](../Interface/IChartLegend/IChartLegend.AxisIcon.htm)


-
[IChartExternLegend.AxisIcon](../Interface/IChartExternLegend/IChartExternLegend.AxisIcon.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Пиктограмма отсутствует.


 1
 Axis. У значка ряда в легенде отображается пиктограмма "основная / дополнительная ось".
