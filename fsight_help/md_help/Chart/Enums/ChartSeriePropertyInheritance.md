# ChartSeriePropertyInheritance


## Описание


Перечисление ChartSeriePropertyInheritance
 содержит настройки ряда, которые наследуются другими рядами.


Используется следующими свойствами и методами:


	- [IChartSerieBase.PropertyInheritance](../Interface/IChartSerieBase/IChartSerieBase.PropertyInheritance.htm).


	- [IChartSerieBase.ResetInheritance](../Interface/IChartSerieBase/IChartSerieBase.ResetInheritance.htm).
