# ChartAxisXPlacement


## Описание


Перечисление ChartAxisXPlacement
 содержит варианты расположения оси категорий.


Используется следующими свойствами и методами:


	- [IChart.AxisXPlacement](../Interface/IChart/IChart.AxisXPlacement.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 BetweenTick. Пересечение
		 в начале категорий.


		 1
		 OnTick. Пересечение
		 на середине категории.
