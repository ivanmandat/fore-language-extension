# TextMultilineType


## Описание


Перечисление TextMultiline используется
 для определения типа переноса текста подписей.


Используется следующим свойством:


	- [IChartAxis.TextMultiline](../Interface/IChartAxis/IChartAxis.TextMultiline.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 NoMultiline. Нет переноса.


		 1
		 MultilineBySymbols.
		 Перенос по символам.


		 2
		 MultilineByWords. Перенос
		 по словам.
