# ChartExtLegPlacement


## Описание


Перечисление ChartExtLegPlacement содержит типы размещения элементов внешней легенды диаграмм.


Используется следующими свойствами и методами:


-
[IChartExternLegend.ItemPlacement](../Interface/IChartExternLegend/IChartExternLegend.ItemPlacement.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Column. Элементы размещаются в одну колонку с выравниванием по левому краю.


 1
 InLine. Элементы размещаются в одну строку.


 2
 Auto. Элементы размещаются автоматически.


 3
 CustomColumns. Элементы размещаются в заданное число столбцов. Количество столбцов определяется свойством [ColCount](../Interface/IChartExternLegend/IChartExternLegend.ColCount.htm).


 4
 CustomRows. Элементы размещаются в заданное число строк. Количество строк определяется свойством [RowCount](../Interface/IChartExternLegend/IChartExternLegend.RowCount.htm).
