# ChartAxisPositionOffsetType


Перечисление ChartAxisPositionOffsetType
 содержит варианты смещения оси.


Используется следующим свойством:


	- [IChartAxis.PositionOffset](../Interface/IChartAxis/IChartAxis.PositionOffset.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Нет смещения.


		 1
		 Auto. Автоматическое
		 смещение. Задает смещение оси равное длине деления внутри оси.


		 2
		 Custom. Заданное смещение.
		 Расстояние задается в свойстве [IChartAxis.PositionOffsetValue](../Interface/IChartAxis/IChartAxis.PositionOffsetValue.htm).
