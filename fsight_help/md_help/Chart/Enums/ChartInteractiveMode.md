# ChartInteractiveMode


## Описание


Перечисление ChartInteractiveMode содержит режимы интерактивности.


Используется следующими свойствами и методами:


-
[IChart.InteractiveMode](../Interface/IChart/IChart.InteractiveMode.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Selection. Режим выделения объектов диаграммы. При двойном щелчке по объекту диаграммы будет выдано окно редактирования формата данного объекта.


 1
 Hyperlinks. Режим гиперссылок.


 2
 ReadOnly. Режим без интерактивности.


 3
 Zoom. Режим масштабирования. По умолчанию производится масштабирование по осям X и Y. Установка типа масштабирования осуществляется с помощью свойства [IChart.ZoomMode](../Interface/IChart/IChart.ZoomMode.htm).


 4
 EditByPoint. Режим редактирования ряда по точкам.


 5
 EditByDrawing. Режим редактирования ряда рисованием.


 6
 AlignObjects. Режим взаимного выравнивания объектов.


 7
 EditLevelLine. Режим редактирования линий уровня.
