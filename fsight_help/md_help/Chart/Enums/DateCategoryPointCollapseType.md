# DateCategoryPointCollapseType


## Описание


Перечисление DateCategoryPointCollapseType
 используется для определения способа агрегации данных оси дат.


Используется следующим свойством:


	- [IChartDateAxisSettings.CategoryPointCollapse](../Interface/IChartDateAxisSettings/IChartDateAxisSettings.CategoryPointCollapse.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Total. Берется сумма
		 значений элементов динамики.


		 1
		 Average. Берется среднее
		 значение элементов динамики.


		 2
		 Minimum. Берется
		 минимальное значение элементов динамики.


		 3
		 Maximum. Берется
		 максимальное значение элементов динамики.


		 4
		 First. Берется первое
		 значение элементов динамики.


		 5
		 Last. Берется последнее
		 значение элементов динамики.


		 6
		 Range. Берется разность
		 между максимальным и минимальным значениями элементов динамики.


		 7
		 Spread. Берется стандартное
		 отклонение по динамике. Стандартное отклонение - это мера того,
		 насколько широко разбросаны значения элементов относительно их
		 среднего.
