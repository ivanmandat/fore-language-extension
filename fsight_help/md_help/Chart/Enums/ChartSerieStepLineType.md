# ChartSerieStepLineType


## Описание


Перечисление ChartSerieStepLineType содержит типы ступенчатого ряда. Ступенька - это изгиб на линии между соседними точками ряда.


Используется следующими свойствами и методами:


-
[IChartSerie.StepLineType](../Interface/IChartSerieBase/IChartSerieBase.StepLineType.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Left. Ступенька располагается слева.


 1
 Center. Ступенька располагается по центру.


 2
 Right. Ступенька располагается справа.
