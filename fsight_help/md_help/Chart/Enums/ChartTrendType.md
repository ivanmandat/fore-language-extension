# ChartTrendType


## Описание


Перечисление ChartTrendType содержит типы линии тренда на графике.


Используется следующими свойствами и методами:


-
[IChartTrendLine.TrendType](../Interface/IChartTrendLine/IChartTrendLine.TrendType.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Linear. Линейный тренд.


 1
 Logarithmic. Логарифмический тренд.


 2
 Exponential. Экспоненциальный тренд.


 3
 Power. Степенной тренд.


 4
 Mean. Среднее.


 5
 Peaks. Пиковые значения.
