# ChartZoomMode


## Описание


Перечисление ChartZoomMode содержит типы масштабирования.


Используется следующими свойствами и методами:


-
[IChart.ZoomMode](../Interface/IChart/IChart.ZoomMode.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Масштабирование не используется, установка этого режима сбрасывает InteractiveMode в ReadOnly.


 1
 AxisX. Масштабирование только по оси X.


 2
 AxisY. Масштабирование только по оси Y.


 3
 AxisXY. Масштабирование по осям X и Y.


 4
 AxisYSec. Масштабирование только по дополнительной оси Y.


 5
 AxisXYSec. Масштабирование по осям: Х и дополнительной Y.


 6
 AxisYYSec. Масштабирование по осям: Y и дополнительной Y.


 7
 AxisAll. Масштабирование по всем осям.
