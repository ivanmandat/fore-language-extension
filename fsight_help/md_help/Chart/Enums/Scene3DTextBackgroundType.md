# Scene3DTextBackgroundType


## Описание


Перечисление Scene3DTextBackgroundType содержит типы оформления фона надписи.


Используется следующими свойствами и методами:


-
[IScene3DTextBackground.Type](../Interface/IScene3DTextBackground/IScene3DTextBackground.Type.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Framing. Обрамление текста.


 1
 Filling. Заливка прямоугольника текста.
