# ChartSecondaryShowMode


## Описание


Перечисление ChartSecondaryShowMode содержит режимы отображения вторичных диаграмм.


Используется следующими свойствами и методами:


-
[IChartCircleExInfo.ShowMode](../Interface/IChartCircleExInfo/IChartCircleExInfo.ShowMode.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 PrimaryAndSecondary. Отображаются основная и дополнительная диаграммы.


 1
 OnlyPrimary. Отображается только основная диаграмма.


 2
 OnlySecondary. Отображается только дополнительная диаграмма.
