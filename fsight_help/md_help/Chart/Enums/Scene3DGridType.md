# Scene3DGridType


## Описание


Перечисление Scene3DGridType
 содержит типы сетки плоскостей трехмерной сцены.


Используется следующими свойствами и методами:


	- [IScene3DGrid.Type](../Interface/IScene3DGrid/IScene3DGrid.Type.htm).
