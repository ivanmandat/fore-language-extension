# ChartOrientation


## Описание


Перечисление ChartOrientation
 содержит типы ориентаций диаграммы.


Используется следующими свойствами и методами:


	- [IChart.Orientation](../Interface/IChart/IChart.Orientation.htm);


	- [IUiChart.Orientation](../Interface/IUiChart/IUiChart.Orientation.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 LeftToRight. Ориентация
		 слева направо.


		 1
		 UpToDown. Ориентация
		 снизу вверх.
