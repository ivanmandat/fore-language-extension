# ChartStockStyle


## Описание


Перечисление ChartStockStyle содержит типы биржевой диаграммы.


Используется следующими свойствами и методами:


-
[IChart.StockStyle](../Interface/IChart/IChart.StockStyle.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Type1. Отображает наборы данных из трех значений (высокий курс, низкий курс, курс закрытия).


 1
 Type2. Отображает наборы данных из четырех значений (курс открытия, высокий курс, низкий курс, курс закрытия).


 2
 Type3. Отображает наборы данных из четырех значений (объем, высокий курс, низкий курс, курс закрытия).


 3
 Type4. Отображает наборы данных из пяти значений (объем, курс открытия, высокий курс, низкий курс, курс закрытия).
