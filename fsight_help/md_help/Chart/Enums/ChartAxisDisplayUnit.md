# ChartAxisDisplayUnit


## Описание


Перечисление ChartAxisDisplayUnit содержит типы цен делений оси.


Используется следующими свойствами и методами:


-
[IChartAxis.DisplayUnit](../Interface/IChartAxis/IChartAxis.DisplayUnit.htm)
