# ChartVerticalAlignment


## Описание


Перечисление ChartVerticalAlignment содержит типы положений подписи данных по вертикали.


Используется следующими свойствами и методами:


-
[IChartLabel.VerticalHotspotAlignment](../Interface/IChartLabel/IChartLabel.VerticalHotspotAlignment.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Top. Выравнивание снизу.


 1
 Middle. Выравнивание по центру.


 2
 Bottom. Выравнивание сверху.


 3
 Justify. Данное значение зарезервировано на будущее.


Примечание. Свойства Top и Bottom не перепутаны, работают так, как описаны. Это связано с тем, что точка отсчета положения подписи данных берется наоборот относительно маркера.
