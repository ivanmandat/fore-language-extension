# ChartAxisGridType


## Описание


Перечисление ChartAxisGridType содержит способы отображения оси.


Используется следующими свойствами и методами:


-
[IChartAxis.GridType](../Interface/IChartAxis/IChartAxis.GridType.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Grid. Отображаются линии сетки (установлено по умолчанию).


 1
 Ticks. Отображаются вспомогательные деления для оси.
