# ChartLegendElementHorizontalAlignment


## Описание


Перечисление ChartLegendElementHorizontalAlignment содержит варианты горизонтального выравнивания элементов в легенде диаграмм.


Используется следующим свойством:


-
[IChartExternLegend.ElementHorizontalAlignment](../Interface/IChartExternLegend/IChartExternLegend.ElementHorizontalAlignment.htm)


## Комментарии


По умолчанию перечисление имеет значение Left.


## Допустимые значения


 Значение
 Краткое описание


 0
 Left. Горизонтальное выравнивание элементов внешней легенды по левому краю.


 1
 Center. Горизонтальное выравнивание элементов внешней легенды по центру.


 2
 Right. Горизонтальное выравнивание элементов внешней легенды по правому краю.
