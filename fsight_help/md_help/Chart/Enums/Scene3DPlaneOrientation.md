# Scene3DPlaneOrientation


## Описание


Перечисление Scene3DPlaneOrientation содержит типы ориентаций уровня относительно плоскостей трехмерной сцены.


Используется следующими свойствами и методами:


-
[IScene3DPlane.Type](../Interface/IScene3DPlane/IScene3DPlane.Type.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 XOYr. Плоскость параллельна плоскости XOY (нормаль ориентирована вдоль оси Z).


 1
 XOYl. Плоскость параллельна плоскости XOY (нормаль ориентирована навстречу оси Z).


 2
 XOZr. Плоскость параллельна плоскости XOZ (нормаль ориентирована вдоль оси Y).


 3
 XOZl. Плоскость параллельна плоскости XOZ (нормаль ориентирована навстречу оси Y).


 4
 YOZr. Плоскость параллельна плоскости YOZ (нормаль ориентирована вдоль оси X).


 5
 YOZl. Плоскость параллельна плоскости YOZ, нормаль ориентирована навстречу оси X.
