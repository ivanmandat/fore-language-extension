# Scene3DSerieType


## Описание


Перечисление Scene3DSerieType содержит типы рядов сцены.


Используется следующими свойствами и методами:


-
[IScene3DSeries.Add](../Interface/IScene3DSeries/IScene3DSeries.Add.htm)


-
[IScene3DSerie.Type](../Interface/IScene3DSerie/IScene3DSerie.Type.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Serie. Undefined (не используется).


 1
 Points. Точки (сферы).


 2
 NamedPoints. Именованные точки (сферы).


 3
 Surface. Поверхности (обычные).


 4
 ConvexHull. Замкнутые поверхности.
