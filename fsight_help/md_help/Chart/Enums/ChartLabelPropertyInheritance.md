# ChartLabelPropertyInheritance


## Описание


Перечисление ChartLabelPropertyInheritance
 используется для задания
 настроек подписей на диаграмме.


Используется следующими свойствами и методами:


	- [IChartLabel.ResetChildrenInheritance](../Interface/IChartLabel/IChartLabel.ResetChildrenInheritance.htm);


	- [IChartLabel.ResetInheritance](../Interface/IChartLabel/IChartLabel.ResetInheritance.htm).
