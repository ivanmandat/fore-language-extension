# ChartAxisType


## Описание


Перечисление ChartAxisType используется
 для определения типа оси.


Используется следующим свойством:


	- [IChartAxis.AxisType](../Interface/IChartAxis/IChartAxis.AxisType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 TextAxis. Автоматическое
		 определение типа оси в зависимости от данных.


		 1
		 BasedOnData. Текстовый
		 тип оси.


		 2
		 DateAxis. Ось дат.
