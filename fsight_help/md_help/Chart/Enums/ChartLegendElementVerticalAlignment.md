# ChartLegendElementVerticalAlignment


## Описание


Перечисление ChartLegendElementVerticalAlignment
 содержит варианты вертикального выравнивания элементов легенды диаграмм.


Используется следующими свойствами и методами:


	- [IChartExternLegend.ElementVerticalAlignment](../Interface/IChartExternLegend/IChartExternLegend.ElementVerticalAlignment.htm);


	- [IChartLegend.ElementVerticalAlignment](../Interface/IChartLegend/IChartLegend.ElementVerticalAlignment.htm).


## Комментарии


По умолчанию перечисление имеет значение Center.


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Top. Вертикальное выравнивание
		 элементов легенды по верхнему краю.


		 1
		 Center. Вертикальное
		 выравнивание элементов легенды по центру.


		 2
		 Bottom. Вертикальное
		 выравнивание элементов легенды по нижнему краю.
