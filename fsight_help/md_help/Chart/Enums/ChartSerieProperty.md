# ChartSerieProperty


## Описание


Перечисление ChartSerieProperty содержит общие свойства ряда.


Используется следующим свойством:


-
[IChartSelectedSeries.IsPropertyEqual](../Interface/IChartSelectedSeries/IChartSelectedSeries.IsPropertyEqual.htm).
