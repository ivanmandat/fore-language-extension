# ChartAxisPosition


## Описание


Перечисление ChartAxisPosition содержит варианты положения оси диаграммы.


Используется следующим свойством:


- [IChartAxis.Position](../Interface/IChartAxis/IChartAxis.Position.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Auto. Автоматическое размещение: пересечение оси с ортогональной осью в минимальном значении/категории.


 1
 Maximum. Пересечение оси с ортогональной осью в максимальном значении/категории. Подписи оси располагаются снаружи области построения.


 2
 Custom. Пересечение оси с ортогональной осью в заданной пользователем точке. Расстояние задается с помощью свойства [IChartAxis.PositionValue](../Interface/IChartAxis/IChartAxis.PositionValue.htm).


## Комментарии


По умолчанию используется значение Auto.
