# IChartAxisInterlineCaption


Сборка: Chart;


## Описание


Интерфейс IChartAxisInterlineCaption
 предназначен для настройки заголовка между линиями уровня.


## Иерархия наследования


           IChartAxisInterlineCaption


## Комментарии


Для задания параметров линии уровня на диаграмме используйте свойства
 интерфейса [IChartAxisLevelLine](../IChartAxisLevelLine/IChartAxisLevelLine.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Font](IChartAxisInterlineCaption.Font.htm)


		 Свойство Font определяет
		 шрифт заголовка между линиями уровня.


		 ![](../../Property_Image.gif)
		 [FontColor](IChartAxisInterlineCaption.FontColor.htm)


		 Свойство FontColor
		 определяет цвет шрифта заголовка между линиями уровня.


		 ![](../../Property_Image.gif)
		 [HorizontalAlignment](IChartAxisInterlineCaption.HorizontalAlignment.htm)


		 Свойство HorizontalAlignment
		 определяет выравнивание текста заголовка между линиями уровня
		 по горизонтали.


		 ![](../../Property_Image.gif)
		 [HorizontalOffset](IChartAxisInterlineCaption.HorizontalOffset.htm)


		 Свойство HorizontalOffset
		 определяет отступ заголовка от линии уровня.


		 ![](../../Property_Image.gif)
		 [Text](IChartAxisInterlineCaption.Text.htm)


		 Свойство Text определяет
		 текст заголовка между линиями уровня.
