# IScene3D.IsDirty


## Синтаксис


IsDirty: Boolean;


## Описание


Свойство IsDirty возвращает
 признак существующих изменений на сцене.


Если были произведены изменения на сцене, то значение свойства принимает
 значение True, в противном случае
 False.


## Пример


В рассматриваемом примере предполагается существование объекта S типа IScene3D.


	Sub Chart3D;

	Var

	    s: IScene3D;

	Begin

	    If s.IsDirty Then Debug.WriteLine("Трехмерная сцена была изменена");

	        Else Debug.WriteLine("Трехмерная сцена не была изменена");

	    End If;

	End Sub Chart3D;


После выполнения примера в окно консоли будет выведена информация о
 наличие изменений трехмерной сцены.
