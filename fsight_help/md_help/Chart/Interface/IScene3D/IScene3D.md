# IScene3D


## Описание


Интерфейс IScene3D содержит
 свойства и методы для определения параметров трехмерной сцены.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Caption](IScene3D.Caption.htm)
		 Свойство Caption возвращает
		 параметры заголовка трехмерной сцены.


		 ![](../../Property_Image.gif)
		 [DisplayedObjects](IScene3D.DisplayedObjects.htm)
		 Свойство DisplayedObjects
		 определяет параметры объектов трехмерной сцены.


		 ![](../../Property_Image.gif)
		 [IsDirty](IScene3D.IsDirty.htm)
		 Свойство IsDirty возвращает
		 признак существующих изменений на сцене.


		 ![](../../Property_Image.gif)
		 [SceneView](IScene3D.SceneView.htm)
		 Свойство SceneView
		 определяет настройки отображения сцены.


		 ![](../../Property_Image.gif)
		 [Series](IScene3D.Series.htm)
		 Свойство Series определяет
		 параметры универсальной коллекции рядов сцены.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [BeginUpdate](IScene3D.BeginUpdate.htm)
		 Метод BeginUpdate устанавливает
		 запрет на обновление сцены.


		 ![](../../Sub_Image.gif)
		 [EndUpdate](IScene3D.EndUpdate.htm)
		 Метод EndUpdate снимает
		 установленный запрет на обновление сцены.


		 ![](../../Sub_Image.gif)
		 [Refresh](IScene3D.Refresh.htm)
		 Метод Refresh осуществляет
		 принудительное обновление изображения трехмерной сцены.
