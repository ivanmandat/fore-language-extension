# IUiChartGroupNameEventArgs.Result


## Синтаксис


Result: String;


## Описание


Свойство Result определяет наименование
 группы на диаграмме.


## Комментарии


Для получения индекса группы используйте [IUiChartGroupNameEventArgs.GroupIndex](IUiChartGroupNameEventArgs.GroupIndex.htm).
