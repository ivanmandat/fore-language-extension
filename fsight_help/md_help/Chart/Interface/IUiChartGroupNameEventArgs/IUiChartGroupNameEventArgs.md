# IUiChartGroupNameEventArgs

## Описание


Интерфейс IUiChartGroupNameEventArgs
 используется для работы с параметрами события [UiChart.OnGetGroupName](../../Class/UiChart/UiChart.OnGetGroupName.htm).


## Иерархия наследования


           [IEventArgs](ForeSys.chm::/Interface/IEventArgs/IEventArgs.htm)


           IUiChartGroupNameEventArgs


## Комментарии


Событие [UiChart.OnGetGroupName](../../Class/UiChart/UiChart.OnGetGroupName.htm)
 позволяет определить наименование группы диаграммы.
