# IChartSerie.ColorIsDefault


## Синтаксис


		ColorIsDefault: Boolean;


## Описание


Свойство ColorIsDefault определяет
 признак использования для элементов ряда цвета из стандартной палитры
 или цвета, заданного пользователем.


## Комментарии


Допустимые значения:


	- True. Для элементов
	 ряда используются цвета из стандартной палитры;


	- False. Для элементов
	 ряда используются цвета, заданные пользователем.


Под элементами ряда понимаются: граница и заливка ряда, линии и маркеры.
