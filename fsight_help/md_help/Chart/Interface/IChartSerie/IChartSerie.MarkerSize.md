# IChartSerie.MarkerSize


## Синтаксис


MarkerSize: Double;


## Описание


Свойство MarkerSize определяет
 размер маркеров у ряда данных.


## Пример


В рассматриваемом примере предполагается, что существует объект Serie типа IChartSerie.


	Sub MarkerS;

	Var

	    Serie : IChartSerie;

	Begin

	    Serie.MarkerSize := 4.5;

	End Sub MarkerS;


После выполнения примера размер маркеров ряда данных на графике изменится
 и будет 4,5 мм.
