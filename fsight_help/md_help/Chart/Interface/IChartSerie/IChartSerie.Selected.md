# IChartSerie.Selected


## Синтаксис


		Selected: Boolean;


## Описание


Свойство Selected определяет
 признак выделения ряда данных.


## Комментарии


Если значение данного свойства True,
 то ряд на графике будет выделен, при значении False
 - нет.
