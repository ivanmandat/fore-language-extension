# IChartSerie.AltColorSetID


## Синтаксис


AltColorSetID: String;


## Описание


Свойство AltColorSetID определяет
 идентификатор альтернативной палитры цветов для ряда диаграммы.


## Комментарии


Альтернативная палитра цветов задается с помощью XML-файла. Цвета из
 альтернативной палитры берутся по порядку.


Если свойство не задано или содержит идентификатор несуществующего набора,
 то используется палитра по умолчанию.
