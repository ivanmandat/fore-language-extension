# IChartSerie


Сборка: Chart;


## Описание


Интерфейс IChartSerie содержит
 свойства и методы для определения параметров ряда данных диаграммы.


## Иерархия наследования


           [IChartAbstractItem](../IChartAbstractItem/IChartAbstractItem.htm)


           [IChartSerieBase](../IChartSerieBase/IChartSerieBase.htm)


           IChartSerie

		 ![](../../Property_Image.gif)

		 ![](../../Property_Image.gif)
		 [MasterSerieIdx](IChartSerie.MasterSerieIdx.htm)

		 определяет индекс дополнительного ряда, который будет являться
		 продолжением основного ряда.

		 [TrendLines](IChartSerie.TrendLines.htm)
