# IChartCircleExInfo


Сборка: Chart;


## Описание


Интерфейс IChartCircleExInfo
 содержит свойства для определения параметров круговой диаграммы с вторичной
 круговой диаграммой или гистограммой.


## Иерархия наследования


           IChartCircleExInfo

		 [BorderPen](IChartCircleExInfo.BorderPen.htm)
		 Свойство BorderPen

		 [DisplayShadow](IChartCircleExInfo.DisplayShadow.htm)

		 Свойство Distance определяет

		 [GroupingRule](IChartCircleExInfo.GroupingRule.htm)
		 Свойство GroupingRule
		 определяет порядок группировки рядов для вторичной диаграммы.


		 ![](../../Property_Image.gif)
		 [GroupingValue](IChartCircleExInfo.GroupingValue.htm)
		 Свойство GroupingValue
		 определяет значение параметра группировки рядов для вторичной
		 диаграммы.

		 Свойство ShowMode определяет
		 режимы отображения вторичной диаграммы.
