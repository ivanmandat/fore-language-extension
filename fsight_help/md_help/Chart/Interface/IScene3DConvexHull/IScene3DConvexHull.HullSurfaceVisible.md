# IScene3DConvexHull.HullSurfaceVisible


## Синтаксис


HullSurfaceVisible: Boolean;


## Описание


Свойство HullSurfaceVisible
 определяет признак отображения оболочки замкнутой поверности. Если значение
 данного свойства True, то поверхность
 оболочки будет отображаться, при значении False
 - нет.


По умолчанию свойство имеет значение True.
