# IScene3DConvexHull.InteriorPointsVisible


## Синтаксис


InteriorPointsVisible: Boolean;


## Описание


Свойство InteriorPointsVisible
 определяет признак отображения точек (сфер) внутри трехмерной поверхности.
 Если значение данного свойства True,
 то точки будут отображаться, при значении False
 - нет.


По умолчанию свойство имеет значение True.
