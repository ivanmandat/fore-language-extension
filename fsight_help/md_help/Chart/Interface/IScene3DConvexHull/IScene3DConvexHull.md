# IScene3DConvexHull


## Описание


Интерфейс IScene3DConvexHull
 содержит свойства для определения параметров трехмерной замнутой поверхности.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DataSource](IScene3DConvexHull.DataSource.htm)
		 Свойство DataSource
		 определяет источник данных для замкнутой поверхности.


		 ![](../../Property_Image.gif)
		 [HullPointsVisible](IScene3DConvexHull.HullPointsVisible.htm)
		 Свойство HullPointsVisible
		 определяет признак отображения вершин (сфер) замкнутой поверхности.


		 ![](../../Property_Image.gif)
		 [HullSurfaceVisible](IScene3DConvexHull.HullSurfaceVisible.htm)
		 Свойство HullSurfaceVisible
		 определяет признак отображения оболочки замкнутой поверхности.


		 ![](../../Property_Image.gif)
		 [InteriorPointsVisible](IScene3DConvexHull.InteriorPointsVisible.htm)
		 Свойство InteriorPointsVisible
		 определяет признак отображения точек (сфер) внутри трехмерной
		 поверхности.


		 ![](../../Property_Image.gif)
		 [Name](IScene3DConvexHull.Name.htm)
		 Свойство Name определяет
		 имя трехмерной поверхности.


		 ![](../../Property_Image.gif)
		 [SurfaceGrid](IScene3DConvexHull.SurfaceGrid.htm)
		 Свойство SurfaceGrid
		 определяет параметры линий, которые соединяют точки трехмерной
		 поверхности.


		 ![](../../Property_Image.gif)
		 [TextStyle](IScene3DConvexHull.TextStyle.htm)
		 Свойство TextStyle
		 позволяет определить параметры надписей имен точек трехмерной
		 поверхности.
