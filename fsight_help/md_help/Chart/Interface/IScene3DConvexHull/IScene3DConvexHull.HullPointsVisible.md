# IScene3DConvexHull.HullPointsVisible


## Синтаксис


HullPointsVisible: Boolean;


## Описание


Свойство HullPointsVisible определяет
 признак отображения вершин (сфер) замкнутой поверхности. Если значение
 данного свойства True, то вершины
 поверхности будут отображаться, при значении False
 - нет.


По умолчанию свойство имеет значение True.
