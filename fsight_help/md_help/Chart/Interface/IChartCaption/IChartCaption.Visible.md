# IChartCaption.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет признак видимости заголовка диаграммы. Если значение данного свойства True, то заголовок диаграммы отображается, при значении False - не отображается.


По умолчанию значение данного свойства True.
