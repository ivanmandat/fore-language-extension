# IChartCaption.WordWrap


## Синтаксис


WordWrap: Boolean;


## Описание


Свойство WordWrap определяет признак переноса по словам заголовка диаграммы. Если значение данного свойства True, то текст заголовка диаграммы будет переноситься по словам, при значении False - этого не происходит.
