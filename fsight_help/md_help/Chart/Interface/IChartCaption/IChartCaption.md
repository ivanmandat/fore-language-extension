# IChartCaption


## Описание


Интерфейс IChartCaption содержит
 свойства для определения параметров заголовка диаграммы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [FormattedText](IChartCaption.FormattedText.htm)
		 Свойство FormattedText
		 определяет форматированный текст заголовка в формате RTF.


		 ![](../../Property_Image.gif)
		 [HorizontalAlignment](IChartCaption.HorizontalAlignment.htm)


		 Свойство HorizontalAlignment
		 определяет выравнивание текста заголовка.


		 ![](../../Property_Image.gif)
		 [Visible](IChartCaption.Visible.htm)
		 Свойство Visible определяет
		 признак видимости заголовка диаграммы.


		 ![](../../Property_Image.gif)
		 [WordWrap](IChartCaption.WordWrap.htm)
		 Свойство WordWrap определяет
		 признак переноса по словам заголовка диаграммы.


## Унаследованные свойства от [IChartPlacedItem](../IChartPlacedItem/IChartPlacedItem.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AutoMargin](../IChartPlacedItem/IChartPlacedItem.AutoMargin.htm)


		 Свойство AutoMargin определяет,
		 будет ли использоваться автоматическая установка отступов для
		 заголовков, осей и легенды диаграммы.


		 ![](../../Property_Image.gif)
		 [Font](../IChartPlacedItem/IChartPlacedItem.Font_.htm)


		 Свойство Font определяет
		 шрифт текста.


		 ![](../../Property_Image.gif)
		 [FontColor](../IChartPlacedItem/IChartPlacedItem.FontColor.htm)


		 Свойство FontColor
		 определяет цвет шрифта.


		 ![](../../Property_Image.gif)
		 [Margin](../IChartPlacedItem/IChartPlacedItem.Margin.htm)


		 Свойство Margin определяет
		 пользовательские величины внешних отступов для заголовка диаграммы,
		 заголовков осей и легенды.


		 ![](../../Property_Image.gif)
		 [Text](../IChartPlacedItem/IChartPlacedItem.Text.htm)


		 Свойство Text позволяет
		 задать текст надписи.


		 ![](../../Property_Image.gif)
		 [TextAngle](../IChartPlacedItem/IChartPlacedItem.TextAngle.htm)


		 Свойство TextAngle
		 определяет угол поворота надписи.
