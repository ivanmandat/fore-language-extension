# IChartCorridor


## Описание


Интерфейс IChartCorridor содержит
 свойства для определения параметров коридора диаграммы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Color](IChartCorridor.Color.htm)
		 Свойство Color определяет
		 цвет монотонной заливки коридора диаграммы.


		 ![](../../Property_Image.gif)
		 [Background](IChartCorridor.Background.htm)
		 Свойство Background
		 определяет параметры заливки коридора диаграммы.


		 ![](../../Property_Image.gif)
		 [DisplayInLegend](IChartCorridor.DisplayInLegend.htm)
		 Свойство DisplayInLegend
		 определяет возможность отображения коридора в легенде диаграмм.


		 ![](../../Property_Image.gif)
		 [GradientFill](IChartCorridor.GradientFill.htm)
		 Свойство GradientFill
		 определяет тип заливки коридора диаграммы.


		 ![](../../Property_Image.gif)
		 [SerieIndex1](IChartCorridor.SerieIndex1.htm)
		 Свойство SerieIndex1
		 определяет индекс первого ряда коридора.


		 ![](../../Property_Image.gif)
		 [SerieIndex2](IChartCorridor.SerieIndex2.htm)
		 Свойство SerieIndex2
		 определяет индекс второго ряда коридора.


		 ![](../../Property_Image.gif)
		 [Text](IChartCorridor.Text.htm)
		 Свойство Text определяет
		 наименование коридора.
