# IChartCorridor.DisplayInLegend


## Синтаксис


DisplayInLegend: Boolean;


## Описание


Свойство DisplayInLegend определяет возможность отображения коридора в легенде диаграмм.


## Комментарии


Коридоры диаграмм могут отображаться как для внутренней, так и для внешней легенд.


Свойство сохраняется вместе с диаграммой.
