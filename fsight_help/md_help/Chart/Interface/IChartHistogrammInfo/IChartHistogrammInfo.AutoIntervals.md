# IChartHistogrammInfo.AutoIntervals


## Синтаксис


AutoIntervals: Boolean;


## Описание


Свойство AutoIntervals определяет признак автоматического расчета интервалов гистограммы.


## Комментарии


Если значение данного свойства True, то производится автоматического расчета интервалов, при значении False не производится.


По умолчанию значение данного свойства True. В случае присвоения данному свойству значения False необходимо задать свойство [Intervals](IChartHistogrammInfo.Intervals.htm).
