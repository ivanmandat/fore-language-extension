# IChartHistogrammInfo


## Описание


Интерфейс IChartHistogrammInfo
 содержит свойства для определения параметров гистограммы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AutoIntervals](IChartHistogrammInfo.AutoIntervals.htm)
		 Свойство AutoIntervals
		 определяет признак автоматического расчета интервалов гистограммы.


		 ![](../../Property_Image.gif)
		 [Intervals](IChartHistogrammInfo.Intervals.htm)
		 Свойство Intervals
		 определяет количество интервалов гистограммы.
