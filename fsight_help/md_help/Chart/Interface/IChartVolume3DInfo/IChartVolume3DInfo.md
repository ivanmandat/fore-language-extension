# IChartVolume3DInfo


## Описание


Интерфейс IChartVolume3DInfo
 содержит свойства для определения параметров объемной диаграммы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [GapDepthCoef](IChartVolume3DInfo.GapDepthCoef.htm)
		 Свойство GapDepthCoef
		 определяет глубину зазора между рядами данных, расположенными
		 перпендикулярно глубине диаграммы.


		 ![](../../Property_Image.gif)
		 [LightContrast](IChartVolume3DInfo.LightContrast.htm)
		 Свойство LightContrast
		 определяет контрастность освещения ( 0 - нет; 1- максимальное).


		 ![](../../Property_Image.gif)
		 [SceneDepthCoef](IChartVolume3DInfo.SceneDepthCoef.htm)
		 Свойство SceneDepthCoef
		 определяет глубину диаграммы.


		 ![](../../Property_Image.gif)
		 [SectorLighting](IChartVolume3DInfo.SectorLighting.htm)
		 Свойство SectorLighting
		 определяет использование освещения каждого сектора для трёхмерных
		 круговых диаграмм.


		 ![](../../Property_Image.gif)
		 [UseContrastAlpha](IChartVolume3DInfo.UseContrastAlpha.htm)
		 Свойство UseContrastAlpha
		 определяет использование альфа-канала при расчете контрастности.


		 ![](../../Property_Image.gif)
		 [UseDepth](IChartVolume3DInfo.UseDepth.htm)
		 Свойство UseDepth определяет
		 принудительное использование глубины (для гистограммы и смешанной
		 диаграммы).
