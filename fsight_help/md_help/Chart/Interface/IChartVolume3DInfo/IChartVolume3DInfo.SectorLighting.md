# IChartVolume3DInfo.SectorLighting


## Синтаксис


SectorLighting: Boolean;


## Описание


Свойство SectorLighting определяет
 использование освещения каждого сектора для трёхмерных круговых диаграмм.


## Комментарии


Если значение данного свойства True,
 то освещение используется, если False
 - не используется.


По умолчанию свойство имеет значение False.
