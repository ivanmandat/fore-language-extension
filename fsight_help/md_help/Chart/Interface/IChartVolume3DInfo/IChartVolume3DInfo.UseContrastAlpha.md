# IChartVolume3DInfo.UseContrastAlpha


## Синтаксис


UseContrastAlpha: Boolean;


## Описание


Свойство UseContrastAlpha определяет
 использование альфа-канала при расчете контрастности.


## Комментарии


Если значение данного свойства True,
 то альфа-канал используется, при значении False
 - нет.


По умолчанию значение данного свойства False.
