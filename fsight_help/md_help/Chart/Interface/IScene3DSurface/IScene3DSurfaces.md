# IScene3DSurface


## Описание


Интерфейс IScene3DSurface содержит
 свойства для определения параметров поверхности представления данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DataSource](IScene3DSurface.DataSource.htm)
		 Свойство DataSource
		 определяет источник данных поверхности.


		 ![](../../Property_Image.gif)
		 [SurfaceGrid](IScene3DSurface.SurfaceGrid.htm)
		 Свойство SurfaceGrid
		 определяет параметры линий сетки поверхности.


## Свойства, унаследованные от [IScene3DPrimitive](../IScene3DPrimitive/IScene3DPrimitive.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BitmapIndex](../IScene3DPrimitive/IScene3DPrimitive.BitmapIndex.htm)
		 Свойство BitmapIndex
		 определяет индекс картинки, которая находится в коллекции текстур
		 IScene3DTextureManager.


		 ![](../../Property_Image.gif)
		 [Color](../IScene3DPrimitive/IScene3DPrimitive.Color.htm)
		 Свойство Color определяет
		 цвет объекта.


		 ![](../../Property_Image.gif)
		 [Selected](../IScene3DPrimitive/IScene3DPrimitive.Selected.htm)
		 Свойство Selected определяет
		 признак выделения объекта.


		 ![](../../Property_Image.gif)
		 [Visible](../IScene3DPrimitive/IScene3DPrimitive.Visible.htm)
		 Свойство Visible определяет
		 видимость объекта.
