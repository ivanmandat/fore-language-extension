# IScene3DLighting


## Описание


Интерфейс IScene3DLighting содержит
 свойства для настройки освещения сцены.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [HighAccuracy](IScene3DLighting.HighAccuracy.htm)
		 Свойство HighAccuracy
		 определяет сглаженный вид, реалистичность изображения трехмерной
		 сцены.


		 ![](../../Property_Image.gif)
		 [UseLighting](IScene3DLighting.UseLighting.htm)
		 Свойство UseLighting
		 определяет использование освещения трехмерной сцены.
