# IChartExporter


## Описание


Интерфейс IChartExporter содержит свойства и методы для экспорта диаграмм.


## Свойства


 Имя свойства
 Краткое описание


 ![](../../Property_Image.gif)
 [Chart](IChartExporter.Chart.htm)
 Свойство Chart определяет диаграмму для экспорта.


 ![](../../Property_Image.gif)
 [PixelHeight](IChartExporter.PixelHeight.htm)
 Свойство PixelHeight определяет высоту диаграммы в пикселях.


 ![](../../Property_Image.gif)
 [PixelWidth](IChartExporter.PixelWidth.htm)
 Свойство PixelWidth определяет ширину диаграммы в пикселях.


 ![](../../Property_Image.gif)
 [Scale](IChartExporter.Scale.htm)
 Свойство Scale определяет масштаб диаграммы для экспорта.
