# IScene3DDisplayedObjects


## Описание


Интерфейс IScene3DDisplayedObjects
 возвращает объекты трехмерной сцены.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ConvexHulls](IScene3DDisplayedObjects.ConvexHulls.htm)


		 Свойство ConvexHulls
		 определяет параметры коллекции трехмерных замкнутых поверхностей.


		 ![](../../Property_Image.gif)
		 [CoordinatePlane](IScene3DDisplayedObjects.CoordinatePlane.htm)


		 Свойство CoordinatePlanes
		 определяет параметры плоскостей системы координат.


		 ![](../../Property_Image.gif)
		 [LevelPlanes](IScene3DDisplayedObjects.LevelPlanes.htm)


		 Свойство LevelPlanes
		 определяет параметры плоскости уровня.


		 ![](../../Property_Image.gif)
		 [NamedSphereGroups](IScene3DDisplayedObjects.NamedSphereGroups.htm)


		 Свойство NamedSphereGroups
		 определяет параметры группы именованных точек.


		 ![](../../Property_Image.gif)
		 [SceneCoordinates](IScene3DDisplayedObjects.SceneCoordinates.htm)
		 Свойство SceneCoordinates
		 определяет параметры системы координат.


		 ![](../../Property_Image.gif)
		 [Spheres](IScene3DDisplayedObjects.Spheres.htm)
		 Свойство Spheres определяет
		 параметры сфер (точек) на сцене.


		 ![](../../Property_Image.gif)
		 [Surfaces](IScene3DDisplayedObjects.Surfaces.htm)
		 Свойство Surfaces определяет
		 поверхности представления данных.


		 ![](../../Property_Image.gif)
		 [TextureManager](IScene3DDisplayedObjects.TextureManager.htm)
		 Свойство TextureManager
		 позволяет работать с текстурами.
