# IChartSeries


Сборка: Chart;


## Описание


Интерфейс IChartSeries содержит
 свойства для определения параметров коллекции рядов диаграммы.


## Иерархия наследования


           IChartSeries


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IChartSeries.Count.htm)
		 Свойство Count возвращает
		 количество рядов диаграммы.


		 ![](../../Property_Image.gif)
		 [DefaultSerie](IChartSeries.DefaultSerie.htm)
		 Свойство DefaultSerie
		 определяет настройки ряда данных по умолчанию.


		 ![](../../Property_Image.gif)
		 [FixedCount](IChartSeries.FixedCount.htm)
		 Свойство FixedCount
		 определяет количество фиксированных (статических) рядов, оформление
		 которых будет сохраняться.


		 ![](../../Property_Image.gif)
		 [Item](IChartSeries.Item.htm)
		 Свойство Item позволяет
		 определять настройки конкретного ряда диаграммы.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AutoplaceLabels](IChartSeries.AutoplaceLabels.htm)
		 Метод AutoplaceLabels
		 осуществляет размещение подписей данных таким образом, чтобы избежать
		 наложения друг на друга.


		 ![](../../Sub_Image.gif)
		 [Move](IChartSeries.Move.htm)
		 Метод Move изменяет
		 индекс ряда, то есть происходит изменение порядка рядов в списке.
