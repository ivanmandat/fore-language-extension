# IChartSerieClickEventArgs


## Описание


Интерфейс IChartSerieClickEventArgs
 возвращает индекс точки и ряда, для которых наступило событие [OnSerieClick](../../Class/ChartBox/ChartBox.OnSerieClick.htm)
 (щелчок кнопкой мыши по ряду на диаграмме), которое возвращает объект
 класса [ChartBox](../../Class/ChartBox/ChartBox.htm).
