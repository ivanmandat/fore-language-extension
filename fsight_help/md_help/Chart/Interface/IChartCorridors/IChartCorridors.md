# IChartCorridors


## Описание


Интерфейс IChartCorridors содержит
 свойства и методы для определения параметров коллекции коридоров диаграммы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Add](IChartCorridors.Add.htm)
		 Свойство Add добавляет
		 на диаграмму новый коридор и возвращает его индекс в коллекции.


		 ![](../../Property_Image.gif)
		 [Count](IChartCorridors.Count.htm)
		 Свойство Count возвращает
		 текущее количество коридоров на диаграмме.


		 ![](../../Property_Image.gif)
		 [Item](IChartCorridors.Item.htm)
		 Свойство Item позволяет
		 определить параметры конкретного коридора диаграммы.


		 ![](../../Property_Image.gif)
		 [Remove](IChartCorridors.Remove.htm)
		 Свойство Remove позволяет
		 удалить конкретный коридор диаграммы.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Clear](IChartCorridors.Clear.htm)
		 Метод Clear позволяет
		 удалять все коридоры диаграммы.
