# IScene3DCoordinatePlanes


## Описание


Интерфейс IScene3DCoordinatePlanes
 содержит свойства для получения плоскости системы координат.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BackPlane](IScene3DCoordinatePlanes.BackPlane.htm)
		 Свойство BackPlane
		 определяет параметры задней стенки сцены (плоскость XOZ).


		 ![](../../Property_Image.gif)
		 [BottomPlane](IScene3DCoordinatePlanes.BottomPlane.htm)
		 Свойство BottomPlane
		 определяет параметры нижней стенки сцены (плоскость XOY).


		 ![](../../Property_Image.gif)
		 [SidePlane](IScene3DCoordinatePlanes.SidePlane.htm)
		 Свойство SidePlane
		 определяет параметры боковой стенки сцены (плоскость YOZ).


		 ![](../../Property_Image.gif)
		 [TextStyles](IScene3DCoordinatePlanes.TextStyles.htm)
		 Свойство TextStyles
		 определяет параметры надписей осей координат.
