# IChartSelectedSeries.Shadow


## Синтаксис


Shadow: [IChartShadow](../IChartShadow/IChartShadow.htm);


## Описание


Свойство Shadow возвращает параметры тени рядов данных. Перед применением параметров тени необходимо, чтобы свойству [DisplayShadow](IChartSelectedSeries.DisplayShadow.htm) было установлено значение True.

Пример использования свойства аналогичен примеру, приведенному в описании свойства [IChartSerie.Shadow](../IChartSerieBase/IChartSerieBase.Shadow.htm).
