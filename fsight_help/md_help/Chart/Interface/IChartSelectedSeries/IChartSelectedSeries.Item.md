# IChartSelectedSeries.Item


## Синтаксис


Item (Index: Integer): [IChartSerie](../IChartSerie/IChartSerie.htm);


## Параметры


Index. Номер ряда на диаграмме. Нумерация начинается с нуля.


## Описание


Свойство Item возвращает параметры ряда на диаграмме по индексу.


## Пример


Пример использования данного свойства приведен в описании свойства [IChartSelectedSeries.RemoveSerie](IChartSelectedSeries.RemoveSerie.htm).
