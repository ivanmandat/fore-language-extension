# IChartSelectedSeries.DisplayInLegend


## Синтаксис


DisplayInLegend: Boolean;


## Описание


Свойство DisplayInLegend определяет признак отображения пиктограмм рядов в легенде. Если значение данного свойства True, то в легенде отображаются пиктограммы рядов, при значение False - нет.
