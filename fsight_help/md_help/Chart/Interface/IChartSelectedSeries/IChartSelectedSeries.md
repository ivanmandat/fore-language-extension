# IChartSelectedSeries


Сборка: Chart;


## Описание


Интерфейс IChartSelectedSeries содержит свойства и методы коллекции выделенных рядов данных диаграммы.


## Свойства


 Имя свойства
 Краткое описание


 ![](../../Property_Image.gif)
 [AllowEdit](IChartSelectedSeries.AllowEdit.htm)
 Свойство AllowEdit определяет возможность редактирования значений ряда прямо на диаграмме.


 ![](../../Property_Image.gif)
 [AtSecondaryAxis](IChartSelectedSeries.AtSecondaryAxis.htm)
 Свойство AtSecondaryAxis определяет, относится ли выделенный ряд к дополнительной оси.


 ![](../../Property_Image.gif)
 [Background](IChartSelectedSeries.Background.htm)
 Свойство Background определяет параметры заливки выделенной коллекции рядов данных.


 ![](../../Property_Image.gif)
 [BorderPen](IChartSelectedSeries.BorderPen.htm)
 Свойство BordenPen определяет параметры линии границы ряда данных.


 ![](../../Property_Image.gif)
 [Count](IChartSelectedSeries.Count.htm)
 Свойство Count возвращает количество выделенных рядов диаграммы.


 ![](../../Property_Image.gif)
 [DisplayInLegend](IChartSelectedSeries.DisplayInLegend.htm)
 Свойство DisplayInLegend определяет признак отображения пиктограмм рядов в легенде.


 ![](../../Property_Image.gif)
 [DisplayMarkers](IChartSelectedSeries.DisplayMarkers.htm)
 Свойство DisplayMarkers определяет признак отображения маркеров у рядов данных.


 ![](../../Property_Image.gif)
 [DisplayShadow](IChartSelectedSeries.DisplayShadow.htm)
 Свойство DisplayShadow определяет признак отображения тени у рядов данных.


 ![](../../Property_Image.gif)
 [EmptyLinePen](IChartSelectedSeries.EmptyLinePen.htm)
 Свойство EmptyLinePen определяет параметры линии диаграммы c отсутствующими данными.


 ![](../../Property_Image.gif)
 [IsEmpty](IChartSelectedSeries.IsEmpty.htm)
 Свойство IsEmpty возвращает признак наличия элементов в коллекции.


 ![](../../Property_Image.gif)
 [IsPropertyEqual](IChartSelectedSeries.IsPropertyEqual.htm)
 Свойство IsPropertyEqual возвращает результат проверки на идентичность значения свойства всех выделенных рядов.


 ![](../../Property_Image.gif)
 [Item](IChartSelectedSeries.Item.htm)
 Свойство Item возвращает параметры ряда на диаграмме по индексу.


 ![](../../Property_Image.gif)
 [LinePen](IChartSelectedSeries.LinePen.htm)
 Свойство LinePen определяет параметры линии выделенных рядов данных диаграммы.


 ![](../../Property_Image.gif)
 [Marker](IChartSelectedSeries.Marker.htm)

 ![](../../Property_Image.gif)
 [MarkerSize](IChartSelectedSeries.MarkerSize.htm)
 Свойство MarkerSize определяет размеры маркеров у выделенных рядов данных диаграммы.


 ![](../../Property_Image.gif)
 [MarkerType](IChartSelectedSeries.MarkerType.htm)
 Свойство MarkerType определяет типы маркеров у выделенных рядов данных диаграммы.


 ![](../../Property_Image.gif)
 [PieDistanceCoef](IChartSelectedSeries.PieDistanceCoef.htm)
 Свойство PieDistanceCoef определяет расстояние от центра круговой диаграммы до крайней дольки ряда.


 ![](../../Property_Image.gif)
 [SerieType](IChartSelectedSeries.SerieType.htm)

 ![](../../Property_Image.gif)
 [Shadow](IChartSelectedSeries.Shadow.htm)
 Свойство Shadow возвращает параметры тени рядов данных.


 ![](../../Property_Image.gif)
 [UseSplineSmoothing](IChartSelectedSeries.UseSplineSmoothing.htm)

## Методы


 Имя метода
 Краткое описание


 ![](../../Sub_Image.gif)
 [Add](IChartSelectedSeries.Add.htm)
 Метод Add добавляет ряд в коллекцию выделенных рядов данных диаграммы по указанному индексу.


 ![](../../Sub_Image.gif)
 [AddSerie](IChartSelectedSeries.AddSerie.htm)
 Метод AddSerie добавляет ряд в коллекцию выделенных рядов данных диаграммы по указанному параметру.


 ![](../../Sub_Image.gif)
 [Clear](IChartSelectedSeries.Clear.htm)
 Метод Clear снимает выделение со всех выделенных рядов диаграммы.


 ![](../../Sub_Image.gif)
 [Remove](IChartSelectedSeries.Remove.htm)
 Метод Remove удаляет ряд из коллекции выделенных рядов данных по указанному индексу.


 ![](../../Sub_Image.gif)
 [RemoveSerie](IChartSelectedSeries.RemoveSerie.htm)
 Метод RemoveSeries удаляет указанные ряды из коллекции выделенных рядов данных диаграммы.


 ![](../../Sub_Image.gif)
 [SelectAll](IChartSelectedSeries.SelectAll.htm)
 Метод SelectAll выделяет все ряды данных, соответствующие заданным параметрам.
