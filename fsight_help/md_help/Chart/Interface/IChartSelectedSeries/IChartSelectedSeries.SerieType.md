# IChartSelectedSeries.SerieType


## Синтаксис


SerieType: [ChartSerieType](../../Enums/ChartSerieType.htm);


## Описание


Свойство SerieType определяет тип ряда данных.


## Комментарии


Данное свойство используется только для рядов смешанных (MultiPle) или лепестковых (Petals) типов диаграмм, причем для ряда лепестковой диаграммы нельзя использовать столбиковый тип ряда (Bars).
