# IChartTrendLines


## Описание


Интерфейс IChartTrendLines содержит
 свойства и методы для определения параметров коллекции линий тренда.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IChartTrendLines.Count.htm)
		 Свойство Count возвращает
		 количество линий трендов.


		 ![](../../Property_Image.gif)
		 [Item](IChartTrendLines.Item.htm)
		 Свойство Item позволяет
		 определять настройки конкретной линии тренда.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IChartTrendLines.Add.htm)
		 Метод Add добавляет
		 линию тренда для ряда данных.


		 ![](../../Sub_Image.gif)
		 [Clear](IChartTrendLines.Clear.htm)
		 Метод Clear удаляет
		 все линии трендов на диаграмме.


		 ![](../../Sub_Image.gif)
		 [Remove](IChartTrendLines.Remove.htm)
		 Метод Remove удаляет
		 линию тренда по заданному индексу.
