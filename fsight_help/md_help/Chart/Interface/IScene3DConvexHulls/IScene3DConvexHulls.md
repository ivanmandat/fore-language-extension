# IScene3DConvexHulls


## Описание


Интерфейс IScene3DConvexHulls
 содержит свойства для определения параметров коллекции трехмерных замкнутых
 поверхностей.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IScene3DConvexHulls.Count.htm)
		 Свойство Count определяет
		 количество трехмерных поверхностей в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IScene3DConvexHulls.Item.htm)
		 Свойство Item возвращает
		 интерфейс IScene3DConvexHull,
		 реализующий свойства трехмерной поверхности, индекс которой передается
		 в качетсве входного параметра Index.
