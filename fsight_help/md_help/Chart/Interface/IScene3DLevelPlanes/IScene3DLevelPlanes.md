# IScene3DLevelPlanes


## Описание


Интерфейс IScene3DLevelPlanes
 содержит свойства для определения параметров коллекции плоскостей указания
 уровня.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IScene3DLevelPlanes.Count.htm)
		 Свойство Count определяет
		 количество плоскостей указания уровня.


		 ![](../../Property_Image.gif)
		 [Item](IScene3DLevelPlanes.Item.htm)
		 Свойство Item возвращает
		 плоскость указания уровня по заданному индексу.
