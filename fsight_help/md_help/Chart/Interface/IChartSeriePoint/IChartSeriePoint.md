# IChartSeriePoint


Сборка: Chart;


## Описание


Интерфейс IChartSeriePoint содержит
 свойства для определения параметров точки ряда данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Background](IChartSeriePoint.Background.htm)
		 Свойство Background
		 определяет параметры заливки для точки ряда данных.


		 ![](../../Property_Image.gif)
		 [Label](IChartSeriePoint.Label.htm)
		 Свойство Label позволяет работать с подписью
		 данных точки ряда данных.


		 ![](../../Property_Image.gif)
		 [Selected](IChartSeriePoint.Selected.htm)
		 Свойство Selected
		 определяет признак выделения точки ряда данных.


		 ![](../../Property_Image.gif)
		 [Total](IChartSeriePoint.Total.htm)
		 Свойство Total определяет
		 признак является ли столбик итогом.
