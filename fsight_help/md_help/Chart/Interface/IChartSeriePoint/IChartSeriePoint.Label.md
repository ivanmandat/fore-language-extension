# IChartSeriePoint.Label


## Синтаксис


Label: [IChartLabel](../IChartLabel/IChartLabel.htm);


## Описание


Свойство Label позволяет работать с подписью данных точки ряда данных.


## Комментарии


Перед использованием свойства необходимо включить отображение подписей данных ([IChartSerie.LabelsEnabled](../IChartSerieBase/IChartSerieBase.LabelsEnabled.htm)).
