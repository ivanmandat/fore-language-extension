# IChartSeriePoint.Background


## Синтаксис


Background: [IChartBackgroundInfo](../IChartBackgroundInfo/IChartBackgroundInfo.htm);


## Описание


Свойство Background определяет параметры заливки для точки ряда данных.


## Комментарии


Заливка точек ряда данных используется на диаграммах: с областями, гистограмме, круговой, смешанной и на частных случаях биржевой.


Для создания градиентной заливки задайте два цвета: начальный и конечный.


Для отображения внесенных изменений на диаграмме необходимо изменить [IChartBackgroundInfo.Type](../IChartBackgroundInfo/IChartBackgroundInfo.Type.htm) на любое значение, кроме [None](../../Enums/ChartBackgroundType.htm).
