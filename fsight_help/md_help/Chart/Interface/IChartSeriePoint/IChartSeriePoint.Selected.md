# IChartSeriePoint.Selected


## Синтаксис


Selected: Boolean;


## Описание


Свойство Selected определяет признак выделения точки ряда данных.


## Комментарии


Если значение данного свойства True, то точка ряда будет выделена, при значении False - нет.
