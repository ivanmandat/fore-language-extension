# IScene3DView


## Описание


Интерфейс IScene3DView содержит
 свойства для настройки отображения трехмерной сцены.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BackgroundColor](IScene3DView.BackgroundColor.htm)
		 Свойство BackgroundColor
		 определяет цвет фона трехмерной сцены.


		 ![](../../Property_Image.gif)
		 [Lighting](IScene3DView.Lighting.htm)
		 Свойство Lighting определяет
		 освещение сцены.


		 ![](../../Property_Image.gif)
		 [SceneViewProperty](IScene3DView.SceneViewProperty.htm)
		 Свойство Scene3DViewProperty
		 определяет настройки вида на сцену.
