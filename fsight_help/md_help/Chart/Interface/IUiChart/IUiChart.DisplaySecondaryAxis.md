# IUiChart.DisplaySecondaryAxis


## Синтаксис


DisplaySecondaryAxis: Boolean;


## Описание


Свойство DisplaySecondaryAxis определяет признак отображения дополнительной оси.


## Комментарии


Если значение данного свойства True, то на графике будет отображаться дополнительная ось. Если значение данного свойства False, то дополнительная ось не отображается.


По умолчанию свойство имеет значение True.
