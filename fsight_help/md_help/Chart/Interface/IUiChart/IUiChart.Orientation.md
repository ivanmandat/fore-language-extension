# IUiChart.Orientation


## Синтаксис


Orientation: [ChartOrientation](../../Enums/ChartOrientation.htm);


## Описание


Свойство Orientation определяет ориентацию диаграммы.


## Комментарии


По умолчанию используется ориентация слева направо.
