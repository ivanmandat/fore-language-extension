# IUiChart.DisplayVolume3D


## Синтаксис


DisplayVolume3D: Boolean;


## Описание


Свойство DisplayVolume3D определяет признак отображения объемного вида диаграммы.


## Комментарии


Если значение данного свойства True, то диаграмма будет представлена в объемном виде. Если значение данного свойства False, то диаграмма будет представлена в двухмерном виде.


По умолчанию свойство имеет значение False.
