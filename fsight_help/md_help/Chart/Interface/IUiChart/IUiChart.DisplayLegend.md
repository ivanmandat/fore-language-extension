# IUiChart.DisplayLegend


## Синтаксис


DisplayLegend: Boolean;


## Описание


Свойство DisplayLegend определяет признак отображения легенды.


## Комментарии


Если значение данного свойства True, то на графике будет отображаться легенда. Если значение данного свойства False, то легенда не отображается.


По умолчанию свойство имеет значение True.
