# IScene3DCoordinates


## Описание


Интерфейс IScene3DCoordinates
 содержит свойства для определения параметров системы координат.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AdditionalAxisValueX](IScene3DCoordinates.AdditionalAxisValueX.htm)
		 Свойство AdditionalAxisValueX
		 возвращает координату по оси X выделенной сферы (точки).


		 ![](../../Property_Image.gif)
		 [AdditionalAxisValueY](IScene3DCoordinates.AdditionalAxisValueY.htm)
		 Свойство AdditionalAxisValueY
		 возвращает координату по оси Y выделенной сферы (точки).


		 ![](../../Property_Image.gif)
		 [AdditionalAxisValueZ](IScene3DCoordinates.AdditionalAxisValueZ.htm)
		 Свойство AdditionalAxisValueZ
		 возвращает координату по оси Z выделенной сферы (точки).


		 ![](../../Property_Image.gif)
		 [AxisStep](IScene3DCoordinates.AxisStep.htm)
		 Свойство AxisStep определяет
		 шаг отображения подписей по основным осям.


		 ![](../../Property_Image.gif)
		 [MaxX](IScene3DCoordinates.MaxX.htm)
		 Свойство MaxX возвращает
		 максимальное значение оси X.


		 ![](../../Property_Image.gif)
		 [MaxY](IScene3DCoordinates.MaxY.htm)
		 Свойство MaxY возвращает
		 максимальное значение оси Y.


		 ![](../../Property_Image.gif)
		 [MaxZ](IScene3DCoordinates.MaxZ.htm)
		 Свойство MaxZ возвращает
		 максимальное значение оси Z.


		 ![](../../Property_Image.gif)
		 [MinX](IScene3DCoordinates.MinX.htm)
		 Свойство MinX возвращает
		 минимальное значение оси X.


		 ![](../../Property_Image.gif)
		 [MinY](IScene3DCoordinates.MinY.htm)
		 Свойство MinY возвращает
		 минимальное значение оси Y.


		 ![](../../Property_Image.gif)
		 [MinZ](IScene3DCoordinates.MinZ.htm)
		 Свойство MinZ возвращает
		 минимальное значение оси Z.


		 ![](../../Property_Image.gif)
		 [NonnumericValueX](IScene3DCoordinates.NonnumericValueX.htm)
		 Свойство NonnumericValueX
		 позволяет задавать пользовательские наименования точек оси Х.


		 ![](../../Property_Image.gif)
		 [NonnumericValueY](IScene3DCoordinates.NonnumericValueY.htm)
		 Свойство NonnumericValueY
		 позволяет задавать пользовательские наименования точек оси Y.


		 ![](../../Property_Image.gif)
		 [NonnumericValueZ](IScene3DCoordinates.NonnumericValueZ.htm)
		 Свойство NonnumericValueZ
		 позволяет задавать пользовательские наименования точек оси Z.


		 ![](../../Property_Image.gif)
		 [UseNumericAxes](IScene3DCoordinates.UseNumericAxes.htm)
		 Свойство UseNumericAxes
		 определяет признак отображения нечисловых значений для осей.
