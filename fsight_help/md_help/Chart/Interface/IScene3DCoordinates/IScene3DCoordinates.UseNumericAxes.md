# IScene3DCoordinates.UseNumericAxes


## Синтаксис


UseNumericAxes: Boolean;


## Описание


Свойство UseNumericAxes определяет
 признак отображения нечисловых значений для осей.


## Комментарии


Свойство применяется для всех координатных осей одновременно.


Если значение свойства True,
 то по осям отображаются числовые значения точек. Если значение свойства
 False, то для осей отображаются
 значения, указанные в свойствах NonnumericValueX,
 NonnumericValueY и NonnumericValueZ.


По умолчанию свойство имеет значение True.
