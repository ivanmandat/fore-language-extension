# IChartLabel.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет признак видимости подписей данных.


## Комментарии


Если значение данного свойства True, то у ряда на графике будут отображены подписи данных, при значении False - этого не происходит.


По умолчанию значение данного свойства False.
