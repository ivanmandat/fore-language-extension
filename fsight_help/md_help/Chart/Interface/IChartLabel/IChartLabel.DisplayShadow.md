# IChartLabel.DisplayShadow


## Синтаксис


DisplayShadow: Boolean;


## Описание


Свойство DisplayShadow определяет, будет ли отображаться тень у подписи данных.


## Комментарии


Если значение данного свойства True, то у подписи данных будет отображаться тень, при значении False - тень не отображается.
