# IChartLabel.AutoPlacement


## Синтаксис


AutoPlacement: Boolean;


## Описание


Свойство AutoPlacement возвращает признак автоматического размещения подписей данных на диаграмме.


## Комментарии


При создании новой диаграммы свойство возвращает True, при перемещении подписи данных либо вызове метода [IChartSeries.AutoplaceLabels](../IChartSeries/IChartSeries.AutoplaceLabels.htm) свойство возвращает False.
