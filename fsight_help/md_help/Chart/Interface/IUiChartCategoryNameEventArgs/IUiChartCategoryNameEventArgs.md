# IUiChartCategoryNameEventArgs


## Описание


Интерфейс IUiChartCategoryNameEventArgs
 предназначен для работы с параметрами события [UiChart.OnGetCategoryName](../../Class/UiChart/UiChart.OnGetCategoryName.htm)
 (формирование наименования точек для оси категорий).
