# IScene3DSphere


## Описание


Интерфейс IScene3DSphere содержит
 свойства для определения параметров сфер (точек).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [R](IScene3DSphere.R.htm)
		 Свойство R определяет
		 радиус сферы (точки). Допустимые значения берутся из диапазона
		 [0, 0.5].


		 ![](../../Property_Image.gif)
		 [Value](IScene3DSphere.Value.htm)
		 Свойство Value определяет
		 координаты сферы (точки).


		 ![](../../Property_Image.gif)
		 [ValueX](IScene3DSphere.ValueX.htm)
		 Свойство ValueX определяет
		 координату сферы (точки) по оси OX.


		 ![](../../Property_Image.gif)
		 [ValueY](IScene3DSphere.ValueY.htm)
		 Свойство ValueY определяет
		 координату сферы (точки) по оси OY.


		 ![](../../Property_Image.gif)
		 [ValueZ](IScene3DSphere.ValueZ.htm)
		 Свойство ValueZ определяет
		 координату сферы (точки) по оси OZ.


## Свойства, унаследованные от [IScene3DPrimitive](../IScene3DPrimitive/IScene3DPrimitive.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BitmapIndex](../IScene3DPrimitive/IScene3DPrimitive.BitmapIndex.htm)
		 Свойство BitmapIndex
		 определяет индекс картинки, которая находится в коллекции текстур
		 IScene3DTextureManager.


		 ![](../../Property_Image.gif)
		 [Color](../IScene3DPrimitive/IScene3DPrimitive.Color.htm)
		 Свойство Color определяет
		 цвет объекта.


		 ![](../../Property_Image.gif)
		 [Selected](../IScene3DPrimitive/IScene3DPrimitive.Selected.htm)
		 Свойство Selected определяет
		 признак выделения объекта.


		 ![](../../Property_Image.gif)
		 [Visible](../IScene3DPrimitive/IScene3DPrimitive.Visible.htm)
		 Свойство Visible определяет
		 видимость объекта.
