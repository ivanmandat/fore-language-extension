## Описание


Интерфейс IUiChartCategoryFormatEventArgs предназначен для работы с параметрами события [UiChart.OnGetCategoryFormat](../../Class/UiChart/UiChart.OnGetCategoryFormat.htm) (формирование формата наименования точек для оси категорий).
