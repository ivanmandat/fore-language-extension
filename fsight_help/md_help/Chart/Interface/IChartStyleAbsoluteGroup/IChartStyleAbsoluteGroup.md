# IChartStyleAbsoluteGroup


Сборка: Chart;


## Описание


Интерфейс IChartStyleAbsoluteGroup
 содержит свойства и методы для определения параметров отображения списка
 групп рядов.


## Иерархия наследования


           IChartStyleAbsoluteGroup


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AbsoluteIndexOf](IChartStyleAbsoluteGroup.AbsoluteIndexOf.htm)
		 Свойство AbsoluteIndexOf
		 возвращает индекс относительной группы рядов диаграммы, к которой
		 принадлежит указанный ряд.


		 ![](../../Property_Image.gif)
		 [Count](IChartStyleAbsoluteGroup.Count.htm)
		 Свойство Count возвращает
		 количество относительных групп рядов в списке групп рядов диаграммы.


		 ![](../../Property_Image.gif)
		 [Item](IChartStyleAbsoluteGroup.Item.htm)
		 Свойство Item возвращает
		 указанную относительную группу рядов.


		 ![](../../Property_Image.gif)
		 [RelativeIndexOf](IChartStyleAbsoluteGroup.RelativeIndexOf.htm)
		 Свойство RelativeIndexOf
		 возвращает индекс указанного ряда в относительной группе рядов,
		 к которой он принадлежит.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [MakeAbsolute](IChartStyleAbsoluteGroup.MakeAbsolute.htm)
		 Метод MakeAbsolute
		 позволяет расположить каждый ряд в отдельную группу.


		 ![](../../Sub_Image.gif)
		 [MakeDefault](IChartStyleAbsoluteGroup.MakeDefault.htm)
		 Метод MakeDefault устанавливает
		 расположение рядов по умолчанию.


		 ![](../../Sub_Image.gif)
		 [MakeRelative](IChartStyleAbsoluteGroup.MakeRelative.htm)
		 Метод MakeRelative
		 позволяет расположить все ряды диаграммы в одной группе.
