# IChartCommonItem.DisplayShadow


## Синтаксис


DisplayShadow: Boolean;


## Описание


Свойство DisplayShadow определяет
 признак отображения тени у элемента графика (ряд, маркер и т.д.). Если
 значение данного свойства True,
 то у объекта на графике будет отображаться тень, при значении False
 - тень не отображается.


По умолчанию значание данного свойства False.
