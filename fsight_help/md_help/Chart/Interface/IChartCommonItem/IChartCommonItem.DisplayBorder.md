# IChartCommonItem.DisplayBorder


## Синтаксис


DisplayBorder: Boolean;


## Описание


Свойство DisplayBorder определяет
 признак отображения обводки. Если значение данного свойства True,
 то обводка будет отображаться, при значение False
 - этого не происходит.


По умолчанию значание данного свойства True.
