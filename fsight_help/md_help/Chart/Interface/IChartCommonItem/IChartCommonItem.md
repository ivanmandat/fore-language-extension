# IChartCommonItem


## Описание


Интерфейс IChartCommonItem содержит
 свойства для определения параметров диаграммы.


Данный интерфейс является базовым для следующих интерфейсов:


	- [IChartPlacedItem](../IChartPlacedItem/IChartPlacedItem.htm)


	- [IChartArea](../IChartArea/IChartArea.htm)


	- [IChartGraphArea](../IChartGraphArea/IChartGraphArea.htm)


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AutoPlacement](IChartCommonItem.AutoPlacement.htm)
		 Свойство AutoPlacement
		 определяет признак авторазмещения.


		 ![](../../Property_Image.gif)
		 [BackgroundInfo](IChartCommonItem.BackgroundInfo.htm)
		 Свойство BackgroundInfo
		 определяет свойства заливки.


		 ![](../../Property_Image.gif)
		 [BorderPen](IChartCommonItem.BorderPen.htm)
		 Свойство BorderPen
		 определяет параметры обводки.


		 ![](../../Property_Image.gif)
		 [DisplayBorder](IChartCommonItem.DisplayBorder.htm)
		 Свойство DisplayBorder
		 определяет признак отображения обводки.


		 ![](../../Property_Image.gif)
		 [DisplayShadow](IChartCommonItem.DisplayShadow.htm)
		 Свойство DisplayShadow
		 определяет признак отображения тени.


		 ![](../../Property_Image.gif)
		 [Rectangle](IChartCommonItem.Rectangle.htm)
		 Свойство Rectangle
		 определяет координаты и размер элемента диаграммы.


		 ![](../../Property_Image.gif)
		 [Shadow](IChartCommonItem.Shadow.htm)
		 Свойство Shadow позволяет
		 определять параметры тени.
