# IChartCommonItem.AutoPlacement


## Синтаксис


AutoPlacement: Boolean;


## Описание


Свойство AutoPlacement определяет
 признак авторазмещения. Если значение данного свойства True,
 то используется авторазмещение объекта, при значение False
 - не используется.


## Комментарии


По умолчанию свойство имеет значение True.
