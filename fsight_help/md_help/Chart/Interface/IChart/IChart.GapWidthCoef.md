# IChart.GapWidthCoef


## Синтаксис


GapWidthCoef: Double;


## Описание


Свойство GapWidthCoef определяет ширину зазора между столбиками гистограммы.


## Комментарии


Устанавливаемая ширина зазора должна находиться в диапазоне [0.0, 5.0].


По умолчанию свойство GapWidthCoef имеет значение 1.5.


## Пример


В рассматриваемом примере предполагается существование объекта Chart типа IChart. Диаграмма должна быть столбикового типа. Для преобразования диаграммы к данному типу следует выполнить команду:


«Chart.Type := ChartType.Bars».


			Sub UserProc;

Begin

    Chart.Type := ChartType.Bars;

    Chart.GapWidthCoef := 0;

End Sub UserProc;


После выполнения примера зазор между столбиками диаграммы будет нулевым.
