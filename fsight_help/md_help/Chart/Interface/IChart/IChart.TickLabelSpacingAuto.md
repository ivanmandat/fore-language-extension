# IChart.TickLabelSpacingAuto


## Синтаксис


TickLabelSpacingAuto: Boolean;


## Описание


Свойство TickLabelSpacingAuto
 определяет, будет ли число категорий между подписями делений рассчитано
 автоматически.


## Комментарии


Допустимые значения:


	- True. Число категорий
	 между подписями делений рассчитывается автоматически;


	- False. Для определения
	 числа категорий между подписями делений используйте свойство [IChart.TickLabelSpacing](IChart.TickLabelSpacing.htm).
