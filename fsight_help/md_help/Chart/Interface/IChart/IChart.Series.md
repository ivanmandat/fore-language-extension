# IChart.Series


## Синтаксис


Series: [IChartSeries](../IChartSeries/IChartSeries.htm);


## Описание


Свойство Series определяет параметры
 рядов.


## Пример


В рассматриваемом примере предполагается, что существует объект Chart типа IChart.


	Sub Series;

	Var

	    Chart : IChart;

	Begin

	    Chart.Series.DisplayLabels := True;

	End Sub Series;


После выполнения примера на всех рядах диаграммы появятся подписи данных.
