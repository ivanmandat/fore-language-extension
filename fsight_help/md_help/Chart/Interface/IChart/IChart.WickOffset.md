# IChart.WickOffset


## Синтаксис


WickOffset: Double;


## Описание


Свойство WickOffset определяет
 смещение маркеров интервала колебания относительно столбика.


## Комментарии


Данное свойство применяется только для диаграммы типа «Гистограмма с
 интервалом колебания». Допустимые значения находятся в интервале [-0.5;
 0.5].


## Пример


В рассматриваемом примере предполагается существование объекта Chart типа IChart.
 Диаграмма должна быть типа «Гистограмма с интервалом колебания». Для преобразования
 диаграммы к данному типу следует выполнить команду:


«Chart.Type := ChartType.Candle;»


	Sub User;

	Var

	    Chart : IChart;

	Begin

	    Chart.WickOffset := 0.3;

	End Sub User;


После выполнения примера маркеры интервала колебаний будут сдвинуты
 относительно центра столбика на заданное расстояние.
