# IChart.RoundJoint


## Синтаксис


RoundJoint: Boolean;


## Описание


Свойство RoundJoint определяет
 возможность скругления углов столбиков диаграммы.


Комментарии


Если значение данного свойства True,
 то углы скругляются, при значении False
 - применяются острые углы.


По умолчанию свойство имеет значение True.


## Пример


В рассматриваемом примере предполагается существование объекта Chart типа IChart.


	Sub User;

	Var

	    Chart : IChart;

	Begin

	    Chart.RoundJoint := False;

	End Sub User;


После выполнения примера скругление углов столбиков рядов будет отключено.
