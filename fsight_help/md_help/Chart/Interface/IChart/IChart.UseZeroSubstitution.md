# IChart.UseZeroSubstitution


## Синтаксис


UseZeroSubstitution: Boolean;


## Описание


Свойство UseZeroSubstitution позволяет настроить замену отсутствующих значений на нули.


## Комментарии


Если значение данного свойства True, то на диаграмме отсутствующие значения будут заменены нулями, при значении False - замены не происходит.


По умолчанию свойство имеет значение False.
