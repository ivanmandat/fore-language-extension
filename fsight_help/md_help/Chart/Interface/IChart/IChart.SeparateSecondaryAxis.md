# IChart.SeparateSecondaryAxis


## Синтаксис


SeparateSecondaryAxis: Boolean;


## Описание


Свойство SeparateSecondaryAxis
 определяет будет ли отображаться дополнительная ось.


## Комментарии


Допустимые значения:


	- True. Дополнительная
	 ось будет отображаться;


	- False. Значение по умолчанию.
	 Дополнительная ось отображаться не будет.
