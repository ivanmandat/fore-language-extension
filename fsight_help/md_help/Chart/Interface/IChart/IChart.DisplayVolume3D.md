# IChart.DisplayVolume3D


## Синтаксис


DisplayVolume3D: Boolean;


## Описание


Свойство DisplayVolume3D определяет признак отображения объемного вида диаграммы.


## Комментарии


Если значение данного свойства True, то диаграмма будет представлена в объемном виде, при значение False - в двумерном виде.


По умолчанию свойство имеет значение False.
