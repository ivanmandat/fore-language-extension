# IChart.DisplaySecondaryAxis


## Синтаксис


DisplaySecondaryAxis: Boolean;


## Описание


Свойство DisplaySecondaryAxis определяет признак отображения дополнительной оси.


## Комментарии


Если значение данного свойства True, то на графике будет отображаться дополнительная ось, при значении False - нет.


По умолчанию свойство имеет значение True.
