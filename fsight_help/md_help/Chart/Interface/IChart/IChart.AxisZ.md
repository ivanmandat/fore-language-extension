# IChart.AxisZ


## Синтаксис


AxisZ: [IChartAxis](../IChartAxis/IChartAxis.htm);


## Описание


Свойство AxisZ возвращает параметры
 оси Z на диаграмме.


## Комментарии


Для получения оси Z необходимо, чтобы свойство [IChart.DisplayZAxis](IChart.DisplayZAxis.htm)
 принимало значение True.


## Пример


Использование свойства приведено в примере для [IChart.DisplayMode](IChart.DisplayMode.htm).
