# IChart.BubbleInfo


## Синтаксис


BubbleInfo: [IChartBubbleInfo](../IChartBubbleInfo/IChartBubbleInfo.htm);


## Описание


Свойство BubbleInfo возвращает
 настройки пузырьковой диаграммы.


## Комментарии


Пузырьковая диаграмма позволяет отображать три набора значений. Третье
 значение определяет размер пузырька.
