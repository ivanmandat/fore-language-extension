# IChart.DisplayZAxis


## Синтаксис


DisplayZAxis: Boolean;


## Описание


Свойство DisplayZAxis определяет,
 будет ли отображаться ось Z на диаграмме.


## Комментарии


Для использования данного свойства необходимо, чтобы [IChart.DisplayMode](IChart.DisplayMode.htm)
 принимал значение [ChartDisplayMode.ModeManhattan3D](../../Enums/ChartDisplayMode.htm).
