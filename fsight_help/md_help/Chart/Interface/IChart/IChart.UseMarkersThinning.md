# IChart.UseMarkersThinning


## Синтаксис


UseMarkersThinning: Boolean;


## Описание


Свойство UseMarkersThinning определяет
 использование автоматического прореживания маркеров.


## Комментарии


Свойство актуально только для линейных диаграмм.


Допустимые значения:


	- True. Автоматическое
	 прореживание маркеров используется;


	- False. Значение по умолчанию.
	 Автоматическое прореживание маркеров не используется.


## Пример


Для выполнения примера добавьте на форму компоненты:


	- UiErAnalyzer, с настроенными свойствами «Active» = true; «Object»
	 = экспресс отчет, содержащий линейную диаграмму;


	- ChartBox1, с настроенным свойством «Source» = идентификатор
	 компонента UiErAnalyzer.


			Sub UserProc();

Begin

    ChartBox1.Chart.UseMarkersThinning := True;

End Sub UserProc;


В результате выполнения примера маркеры на отображаемой диаграмме будут
 автоматически прореживаться.
