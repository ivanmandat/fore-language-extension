# IChart.SecondaryOverlapCoef


## Синтаксис


SecondaryOverlapCoef: Double;


## Описание


Свойство SecondaryOverlapCoef
 определяет величину перекрытия столбцов, построенных по дополнительной
 оси.


## Комментарии


Диапазон допустимых значений: [-1; 1].
