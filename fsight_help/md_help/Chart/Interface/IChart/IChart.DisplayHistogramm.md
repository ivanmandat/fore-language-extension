# IChart.DisplayHistogramm


## Синтаксис


DisplayHistogramm: Boolean;


## Описание


Свойство DisplayHistogramm определяет признак отображения частотной диаграммы.


## Комментарии


Если значение данного свойства True, то на графике будет отображаться частотная диаграмма, при значении False - частотная диаграмма не отображается.


По умолчанию свойство имеет значение False.


Отображение в виде частотной диаграммы доступно для всех типов диаграмм, кроме круговой, лепестковой, биржевой и гистограммы с интервалами колебаний.


Для определения параметров частотной диаграммы используйте свойство [IChart.HistogrammInfo](IChart.HistogrammInfo.htm).
