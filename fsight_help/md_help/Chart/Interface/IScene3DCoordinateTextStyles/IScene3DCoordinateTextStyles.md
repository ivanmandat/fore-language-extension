# IScene3DCoordinateTextStyles


## Описание


Интерфейс IScene3DCoordinateTextStyles
 содержит свойства для настройки стилей надписей по плоскостям системы
 координат.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [TextStyleOX](IScene3DCoordinateTextStyles.TextStyleOX.htm)
		 Свойство TextStyleOX
		 определяет стиль надписей для оси OX.


		 ![](../../Property_Image.gif)
		 [TextStyleOY](IScene3DCoordinateTextStyles.TextStyleOY.htm)
		 Свойство TextStyleOY
		 определяет стиль надписей для оси OY.


		 ![](../../Property_Image.gif)
		 [TextStyleOZ](IScene3DCoordinateTextStyles.TextStyleOZ.htm)
		 Свойство TextStyleOZ
		 определяет стиль надписей для оси OZ.


		 ![](../../Property_Image.gif)
		 [PrecisionOX](IScene3DCoordinateTextStyles.PrecisionOX.htm)
		 Свойство PrecisionOX
		 определяет точность значений по оси OX.


		 ![](../../Property_Image.gif)
		 [PrecisionOY](IScene3DCoordinateTextStyles.PrecisionOY.htm)
		 Свойство PrecisionOY
		 определяет точность значений по оси OY.


		 ![](../../Property_Image.gif)
		 [PrecisionOZ](IScene3DCoordinateTextStyles.PrecisionOZ.htm)
		 Свойство PrecisionOZ
		 определяет точность значений по оси OZ.


		 ![](../../Property_Image.gif)
		 [TextStyleAdditional](IScene3DCoordinateTextStyles.TextStyleAdditional.htm)
		 Свойство TextStyleAdditional
		 определяет стиль надписей координат выделенной сферы (точки).
