# IUiChartSerieGroupNameEventArgs

## Описание


Интерфейс IUiChartSerieGroupNameEventArgs
 используется для работы с параметрами события [UiChart.OnGetSerieGroupName](../../Class/UiChart/UiChart.OnGetSerieGroupName.htm).


## Иерархия наследования


           [IEventArgs](ForeSys.chm::/Interface/IEventArgs/IEventArgs.htm)


           IUiChartSerieGroupNameEventArgs


## Комментарии


Событие [UiChart.OnGetSerieGroupName](../../Class/UiChart/UiChart.OnGetSerieGroupName.htm)
 позволяет определить наименование группы рядов диаграммы.
