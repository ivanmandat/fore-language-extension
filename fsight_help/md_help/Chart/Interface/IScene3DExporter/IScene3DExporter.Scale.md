# IScene3DExporter.Scale


## Синтаксис


Scale: Double;


## Описание


Свойство Scale определяет масштаб трехмерной сцены для экспорта.


## Комментарии


Увеличивает масштаб в заданное количество раз. Свойство может принимать любые значения больше нуля.
