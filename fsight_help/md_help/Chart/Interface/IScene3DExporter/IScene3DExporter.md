# IScene3DExporter


Сборка: Chart;


## Описание


Интерфейс IScene3DExporter предназначен
 для работы с параметрами экспорта трехмерной сцены.


## Иерархия наследования

## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [PixelHeight](IScene3DExporter.PixelHeight.htm)
		 Свойство PixelHeight
		 определяет высоту трехмерной сцены в пикселях.


		 ![](../../Property_Image.gif)
		 [PixelWidth](IScene3DExporter.PixelWidth.htm)
		 Свойство PixelWidth
		 определяет ширину трехмерной сцены в пикселях.


		 ![](../../Property_Image.gif)
		 [Scale](IScene3DExporter.Scale.htm)
		 Свойство Scale определяет
		 масштаб трехмерной сцены для экспорта.


		 ![](../../Property_Image.gif)
		 [Scene3D](IScene3DExporter.Scene3D.htm)
		 Свойство Scene3D определяет
		 трехмерную сцену для экспорта.
