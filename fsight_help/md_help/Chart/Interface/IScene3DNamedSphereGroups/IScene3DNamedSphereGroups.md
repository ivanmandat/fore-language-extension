# IScene3DNamedSphereGroups


## Описание


Интерфейс IScene3DNamedSphereGroups
 содержит свойства для определения параметров коллекции групп именованных
 сфер.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IScene3DNamedSphereGroups.Count.htm)
		 Свойство Count определяет
		 количество групп именованных сфер.


		 ![](../../Property_Image.gif)
		 [Item](IScene3DNamedSphereGroups.Item.htm)
		 Свойство Item возвращает
		 группу именованых точек по заданному индексу.
