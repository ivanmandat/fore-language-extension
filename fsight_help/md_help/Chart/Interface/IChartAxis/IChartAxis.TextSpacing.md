# IChartAxis.TextSpacing


## Синтаксис


		TextSpacing: Double;


## Описание


Свойство TextSpacing определяет
 пользовательское расстояние от подписи до линии оси.


## Комментарии


По умолчанию величина TextSpacing
 равна 2 мм. Использование свойства возможно, если свойству [AutoTextSpacing](IChartAxis.AutoTextSpacing.htm)
 установлено значение False. Если
 величина TextSpacing отрицательная,
 то считается, что расстояние от подписи до оси равно нулю.


Данное свойство сохраняется с диаграммой. Если загружается объект предыдущей
 версии, то для него устанавливаются свойства [TextAlignment](IChartAxis.TextAlignment.htm),
 [AutoTextSpacing](IChartAxis.AutoTextSpacing.htm), [TextSpacing](IChartAxis.TextSpacing.htm)
 и [TextVisible](IChartAxis.TextVisible.htm) по умолчанию.
