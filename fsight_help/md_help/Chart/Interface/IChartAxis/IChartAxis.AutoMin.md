# IChartAxis.AutoMin


## Синтаксис


AutoMin: Boolean;


## Описание


Свойство AutoMin производит
 автоматический расчет минимума на оси. По умолчанию свойство имеет значение
 True. Если значение данного свойства
 False, то минимум не вычисляется
 и его необходимо указать.
