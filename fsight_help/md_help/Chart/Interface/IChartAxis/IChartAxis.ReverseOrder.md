# IChartAxis.ReverseOrder


## Синтаксис


ReverseOrder: Boolean;


## Описание


Свойство ReverseOrder определяет
 признак использования обратного порядка значений (категорий) оси.


## Комментарии


При установке свойства в True
 значения (категории) оси будут расположены в обратном порядке, причем
 перпендикулярная ось перемещается на противоположный край области построения;
 при значении свойства False -
 используется обычный порядок значений (категорий).


По умолчанию свойство имеет значение False.


При применении обратного порядка для оси категорий, ряды будут отображаться
 в обратном порядке.


Данное свойство нельзя использовать для лепестковой диаграммы.


## Пример


	Sub Order;

	Var

	    UiChart1: UiChart;

	    Axisx : IChartAxis;

	Begin

	    Axisx := UiChart1.Chart.AxisX;

	    AxisX.ReverseOrder := True;

	End Sub Order;


После выполнения примера для оси X будет использован обратный порядок
 точек.
