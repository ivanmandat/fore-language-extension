# IChartAxis.AutoMax


## Синтаксис


AutoMax: Boolean;


## Описание


Свойство AutoMax производит
 автоматический расчет максимума на оси. По умолчанию свойство имеет значение
 True. Если значение данного свойства
 False, то максимум не вычисляется
 и его необходимо указать.
