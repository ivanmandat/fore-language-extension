# IChartAxis.ZoomIn


## Синтаксис


ZoomIn(Start: Double; Length: Double);


## Параметры


Start. Начало участка, который
 необходимо увеличить. Данный параметр принимает значения из диапазона
 [0, 1-Length];


Length. Длина участка, который
 будет масштабироваться. Данный параметр принимает значения из диапазона
 [0, 1].


## Описание


Метод ZoomIn позволяет определять
 увеличение масштаба некоторого участка по оси.


## Пример


В рассматриваемом примере предполагается, что существует объект ChartAxis типа IChartAxis.


	Sub Zoom;

	Var

	    ChartAxis : IChartAxis;

	Begin

	    ChartAxis.ZoomIn(0,0.3);

	End Sub Zoom;


После выполнения примера будет увеличен масштаб участка диаграммы, который
 начинается в начале оси и его длина составляет 0,3.
