# IChartAxis.DateAxisSettings


## Синтаксис


DateAxisSettings: [IChartDateAxisSettings](../IChartDateAxisSettings/IChartDateAxisSettings.htm);


## Описание


Свойство DateAxisSettings возвращает
 настройки оси дат.


## Комментарии


Свойство доступно и может быть сохранено только для оси Х. Свойство
 DateAxisSetting учитывается, если
 свойство [IChartAxis.AxisType](IChartAxis.AxisType.htm)
 принимает значение [ChartAxisType.DateAxis](../../Enums/ChartDateAxisUnit.htm).
