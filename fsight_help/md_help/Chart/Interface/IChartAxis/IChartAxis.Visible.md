# IChartAxis.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет признак отображения оси.


## Комментарии


Если значение данного свойства True, то ось отображается, при значении False - нет.


По умолчанию свойство имеет значение True.
