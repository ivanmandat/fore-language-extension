# IChartAxis.MaxJointGap


## Синтаксис


MaxJointGap: Double;


## Описание


Свойство MaxJointGap устанавливает конечное значение точки разрыва оси и рядов диаграммы.


## Комментарии


Значение свойства определяется в пределах от минимального до максимального значений по шкале. Свойство актуально для использования только для оси значений и если свойство [IChartAxis.UseJointGap](IChartAxis.UseJointGap.htm) установлено в True.
