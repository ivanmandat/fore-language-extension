# IChartAxis.MinorTickPen


## Синтаксис

## Описание


Свойство MinorTickPen определяет
 настройки промежуточных делений сетки.


## Комментарии


Для определения настроек промежуточных линий сетки используйте свойство
 [IChartAxis.MinorLinePen](IChartAxis.MinorLinePen.htm).
