# IChartAxis.AxisType


## Синтаксис


AxisType: [ChartAxisType](../../Enums/ChartAxisType.htm);


## Описание


Свойство AxisType определяет
 тип оси на диаграмме.


## Комментарии


Свойство доступно и может быть сохранено только для оси X. Свойство
 AxisType нельзя использовать в
 следующих типах диаграмм:


	- вторичная гистограмма;


	- вторичная круговая;


	- гистограмма;


	- круговая;


	- лепестковая;


	- пузырьковая;


	- точечная.
