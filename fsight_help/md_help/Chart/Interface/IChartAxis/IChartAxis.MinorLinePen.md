# IChartAxis.MinorLinePen


## Синтаксис

## Описание


Свойство MinorLinePen определяет
 настройки промежуточных линий сетки.


## Комментарии


Для определения настроек промежуточных делений сетки используйте свойство
 [IChartAxis.MinorTickPen](IChartAxis.MinorTickPen.htm).
