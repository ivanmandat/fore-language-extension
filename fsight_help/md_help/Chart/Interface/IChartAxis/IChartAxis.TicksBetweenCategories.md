# IChartAxis.TicksBetweenCategories


## Синтаксис


TicksBetweenCategories: Boolean;


## Описание


Свойство TicksBetweenCategories определяет положение линий делений относительно категорий.


## Комментарии


Свойство актуально только для оси категорий (ось X).


По умолчанию свойство имеет значение False и линии делений отображаются по центру категорий. При значении True линии делений отображаются между категориями.
