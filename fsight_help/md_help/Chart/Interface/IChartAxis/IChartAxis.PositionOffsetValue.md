# IChartAxis.PositionOffsetValue


## Синтаксис


PositionOffsetValue: Double;


## Описание


Свойство PositionOffsetValue задает расстояние, на которое будет смещена ось.


## Комментарии


Значение задается в миллиметрах.


При заданном положительном значении ось смещается в противоположенную сторону от области построения. При отрицательном значении смещение происходит внутрь области построения.
