# IScene3DPrimitive.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет
 видимость объекта. Если значение свойства True,
 то объект отображается на сцене, при значении False
 - нет.


По умолчанию свойство имеет значение True.
