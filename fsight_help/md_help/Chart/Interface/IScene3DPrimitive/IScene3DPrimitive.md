# IScene3DPrimitive


## Описание


Интерфейс IScene3DPrimitive
 содержит свойства для определения внешнего вида графических примитивов
 и используется как базовый для интерфейсов:


	- [IScene3DConvexHull](../IScene3DConvexHull/IScene3DConvexHull.htm)


	- [IScene3DGrid](../IScene3DGrid/IScene3DGrid.htm)


	- [IScene3DNamedSphereGroup](../IScene3DNamedSphereGroup/IScene3DNamedSphereGroup.htm)


	- [IScene3DPlane](../IScene3DPlane/IScene3DPlane.htm)


	- [IScene3DSphere](../IScene3DSphere/IScene3DSphere.htm)


	- [IScene3DSurface](../IScene3DSurface/IScene3DSurfaces.htm)
