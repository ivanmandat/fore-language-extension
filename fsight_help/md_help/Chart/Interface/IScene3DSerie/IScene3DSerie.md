# IScene3DSerie


## Описание


Интерфейс IScene3DSerie определяет
 параметры ряда сцены.


## Комментарии


В текущей реализации данный интерфейс следует использовать только для
 чтения свойств объекта. При необходимости его редактирования следует использовать
 специализированную коллекцию (интерфейс).
