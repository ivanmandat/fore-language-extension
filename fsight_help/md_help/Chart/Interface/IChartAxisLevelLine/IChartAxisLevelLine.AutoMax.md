# IChartAxisLevelLine.AutoMax


## Синтаксис


AutoMax: Boolean;


## Описание


Свойство AutoMax определяет,
 будет при производиться автоматический расчет максимума.


## Комментарии


Допустимые значения:


	- True. Автоматический
	 расчет максимума производится;


	- False. Автоматический
	 расчет максимума не производится. Для определения значения максимума
	 используйте свойство [IChartAxisLevelLine.Max](IChartAxisLevelLine.Max.htm).
