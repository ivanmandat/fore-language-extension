# IChartAxisLevelLine.EnableInterlineCaption


## Синтаксис


EnableInterlineCaption: Boolean;


## Описание


Свойство EnableInterlineCaption
 определяет, отображается ли заголовок между линиями уровня.


## Комментарии


Допустимые значения:


	- True. Заголовок между
	 линиями уровня будет отображаться;


	- False. Заголовок
	 между линиями уровня не будет отображаться.


Отображение заголовка настраивается для линии уровня, от которой направлена
 [заливка интервала](IChartAxisLevelLine.InterlineColor.htm)
 между линиями уровня.
