# IChartAxisLevelLine.ValueFunction


## Синтаксис


ValueFunction: [ChartAxisLevelLineFunctionType](../../Enums/ChartAxisLevelLineFunctionType.htm);


## Описание


Свойство ValueFunction определяет,
 какой тип функции будет использован для определения значения линии уровня.


## Комментарии


По умолчанию свойство имеет значение [ChartAxisLevelLineFunctionType.Constant](../../Enums/ChartAxisLevelLineFunctionType.htm), в этом случае для
 определения значения линии уровня используйте свойство [IChartAxisLevelLine.Value](IChartAxisLevelLine.Value.htm).
