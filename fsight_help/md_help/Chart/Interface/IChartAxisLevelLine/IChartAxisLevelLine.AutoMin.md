# IChartAxisLevelLine.AutoMin


## Синтаксис


AutoMin: Boolean;


## Описание


Свойство AutoMin определяет,
 будет ли производиться автоматический расчет минимума.


## Комментарии


Допустимые значения:


	- True. Автоматический
	 расчет минимума производится;


	- False. Автоматический
	 расчет минимума не производится. Для определения значения минимума
	 используйте свойство [IChartAxisLevelLine.Min](IChartAxisLevelLine.Min.htm).
