# IChartAxisLevelLine


Сборка: Chart;


## Описание


Интерфейс IChartAxisLevelLine
 содержит свойства для задания параметров линии уровня на диаграмме.


## Иерархия наследования


           IChartAxisLevelLine


## Комментарии


Для работы с коллекцией линий уровня на диаграмме используйте интерфейс
 [IChartAxisLevelLines](../IChartAxisLevelLines/IChartAxisLevelLines.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AutoMax](IChartAxisLevelLine.AutoMax.htm)


		 Свойство AutoMax определяет,
		 будет ли производиться автоматический расчет максимума.


		 ![](../../Property_Image.gif)
		 [AutoMin](IChartAxisLevelLine.AutoMin.htm)


		 Свойство AutoMin определяет,
		 будет ли производиться автоматический расчет минимума.


		 ![](../../Property_Image.gif)
		 [CalculatedValue](IChartAxisLevelLine.CalculatedValue.htm)


		 Свойство CalculatedValue
		 возвращает значение функции, рассчитывающей линию уровня.


		 ![](../../Property_Image.gif)
		 [DrawUnderSeries](IChartAxisLevelLine.DrawUnderSeries.htm)


		 Свойство DrawUnderSeries
		 определяет порядок расположения линии уровня.


		 ![](../../Property_Image.gif)
		 [EnableInterlineCaption](IChartAxisLevelLine.EnableInterlineCaption.htm)


		 Свойство EnableInterlineCaption
		 определяет, отображается ли заголовок между линиями уровня.


		 ![](../../Property_Image.gif)
		 [Index](IChartAxisLevelLine.Index.htm)


		 Свойство Index возвращает
		 индекс линии уровня.


		 ![](../../Property_Image.gif)
		 [InterlineCaption](IChartAxisLevelLine.InterlineCaption.htm)


		 Свойство InterlineCaption
		 возвращает заголовок между линиями уровня.


		 ![](../../Property_Image.gif)
		 [InterlineColor](IChartAxisLevelLine.InterlineColor.htm)


		 Свойство InterlineColor
		 определяет цвет заливки области от одной до следующей за ней по
		 значению линии уровней диаграммы.


		 ![](../../Property_Image.gif)
		 [Label](IChartAxisLevelLine.Label.htm)


		 Свойство Label определяет
		 показывать подпись или нет.


		 ![](../../Property_Image.gif)
		 [Max](IChartAxisLevelLine.Max.htm)


		 Свойство Max определяет
		 значение максимума.


		 ![](../../Property_Image.gif)
		 [Min](IChartAxisLevelLine.Min.htm)


		 Свойство Min определяет
		 значение минимума.


		 ![](../../Property_Image.gif)
		 [MinMaxRelatesToSecondaryAxis](IChartAxisLevelLine.MinMaxRelatesToSecondaryAxis.htm)


		 Свойство MinMaxRelatesToSecondaryAxis
		 определяет, задаются ли минимальное и максимальное значения по
		 отношению к дополнительной оси.


		 ![](../../Property_Image.gif)
		 [Pen](IChartAxisLevelLine.Pen.htm)


		 Свойство Pen определяет
		 параметры начертания линии уровня.


		 ![](../../Property_Image.gif)
		 [Value](IChartAxisLevelLine.Value.htm)


		 Свойство Value определяет
		 значение линии уровня.


		 ![](../../Property_Image.gif)
		 [ValueFunction](IChartAxisLevelLine.ValueFunction.htm)


		 Свойство ValueFunction
		 определяет, какой тип функции будет использован для определения
		 значения линии уровня.
