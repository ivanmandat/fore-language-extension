# IChartAxisLevelLine.Min


## Синтаксис


Min: Double;


## Описание


Свойство Min определяет значение
 минимума.


## Комментарии


Для определения, будет ли производиться автоматический расчет минимума,
 используйте свойство [IChartAxisLevelLine.AutoMin](IChartAxisLevelLine.AutoMin.htm).
