# IChartAxisLevelLine.CalculatedValue


## Синтаксис


CalculatedValue: Double;


## Описание


Свойство CalculatedValue возвращает
 значение функции, рассчитывающей линию уровня.


## Комментарии


Значение функции указано в свойстве [IChartAxisLevelLine.ValueFunction](IChartAxisLevelLine.ValueFunction.htm),
 если оно не задано, то свойство CalculatedValue
 возвращает значение 0.
