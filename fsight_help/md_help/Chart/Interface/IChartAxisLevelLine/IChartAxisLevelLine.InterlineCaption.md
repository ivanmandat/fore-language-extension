# IChartAxisLevelLine.InterlineCaption


## Синтаксис


InterlineCaption: [IChartAxisInterlineCaption](../IChartAxisInterlineCaption/IChartAxisInterlineCaption.htm);


## Описание


Свойство InterlineCaption возвращает
 заголовок между линиями уровня.


## Комментарии


Формат заголовка настраивается для линии уровня, от которой направлена
 [заливка интервала](IChartAxisLevelLine.InterlineColor.htm)
 между линиями уровня.


Для определения, отображается ли заголовок между линиями уровня, используйте
 свойство [IChartAxisLevelLine.EnableInterlineCaption](IChartAxisLevelLine.EnableInterlineCaption.htm).
