# IChartAxisLevelLine.Max


## Синтаксис


Max: Double;


## Описание


Свойство Max определяет значение
 максимума.


## Комментарии


Для определения, будет ли производиться автоматический расчет максимума,
 используйте свойство [IChartAxisLevelLine.AutoMax](IChartAxisLevelLine.AutoMax.htm).
