# ITrellisChart


Сборка: Chart;


## Описание


Интерфейс ITrellisChart используется
 для настройки диаграммы треллис.


## Иерархия наследования


           ITrellisChart


## Комментарии


Работа с интерфейсом [ITrellisChart](ITrellisChart.htm)
 в Fore недоступна.


Использование [ITrellisChart](ITrellisChart.htm)
 возможно в различных средах разработки приложений с помощью библиотек
 Prognoz.Platform.Interop.Chart.dll и Prognoz.Platform.Interop.Drawing.dll.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ChartSettings](ITrellisChart.ChartSettings.htm)
		 Свойство ChartSettings
		 возвращает общие настройки для всех дочерних диаграмм в диаграмме треллис.


		 ![](../../Property_Image.gif)
		 [ViewFrame](ITrellisChart.ViewFrame.htm)
		 Свойство ViewFrame определяет
		 настройки для области диаграммы треллис.

		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Refresh](ITrellisChart.Refresh.htm)

		 ![](../../Sub_Image.gif)
		 [ResetDataCash](ITrellisChart.ResetDataCash.htm)
		 Метод ResetDataCash
