# ITrellisChart.ChartSettings


## Синтаксис


ChartSettings: [IChart](../IChart/IChart.htm);


## Описание


Свойство ChartSettings возвращает
 общие настройки для всех дочерних диаграмм в диаграмме треллис.


## Комментарии


Работа с интерфейсом [ITrellisChart](ITrellisChart.htm)
 в Fore недоступна.


Использование [ITrellisChart](ITrellisChart.htm)
 возможно в различных средах разработки приложений с помощью библиотек
 Prognoz.Platform.Interop.Chart.dll и Prognoz.Platform.Interop.Drawing.dll.
