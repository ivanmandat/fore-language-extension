# IUiChartRelativeSerieEnabledEventArgs.Result


## Синтаксис


Result: Boolean;


## Описание


Свойство Result определяет,
 будут ли использоваться относительные ряды.


## Комментарии


Допустимые значения:


	- True. Относительные
	 ряды будут использоваться;


	- False. Относительные
	 ряды использоваться не будут.
