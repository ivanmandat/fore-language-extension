## Описание


Интерфейс IUiChartRelativeSerieEnabledEventArgs
 используется для работы с параметрами события [UiChart.OnGetRelativeSerieEnabled](../../Class/UiChart/UiChart.OnGetRelativeSerieEnabled.htm).

 позволяет определить, будут ли использоваться относительные ряды.
