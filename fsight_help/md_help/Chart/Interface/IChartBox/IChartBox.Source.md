# IChartBox.Source


## Синтаксис


Source: [IChartSource](../IChartSource/IChartSource.htm);


## Описание


Свойство Source определяет источник данных для компонента [ChartBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/ChartBox.htm).


## Комментарии


В качестве источника может быть установлен компонент [UiChart](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiChart.htm) либо [UiErAnalyzer](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiErAnalyzer.htm).
