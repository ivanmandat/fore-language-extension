# IChartAbstractItem


## Описание


Интерфейс IChartAbstractItem
 содержит свойства компонентов диаграммы и является базовым для интерфейсов:


	- [IChartAxis](../IChartAxis/IChartAxis.htm);


	- [IChartCommonItem](../IChartCommonItem/IChartCommonItem.htm);


	- [IChartSerieBase](../IChartSerieBase/IChartSerieBase.htm);


	- [IChartLabel](../IChartLabel/IChartLabel.htm).
