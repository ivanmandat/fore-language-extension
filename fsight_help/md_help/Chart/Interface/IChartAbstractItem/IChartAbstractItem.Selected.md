# IChartAbstractItem.Selected


## Синтаксис


Selected: Boolean;


## Описание


Свойство Selected определяет
 признак выделения элемента графика (ряд, маркер и т.д.). Если значение
 данного свойства True, то элемент
 графика будет выделен, при значении False
 - нет.


Примечание.
 Данное свойство не может быть задано для подписей данных по умолчанию
 [IChartSerie.DefaultLabel](../IChartSerieBase/IChartSerieBase.DefaultLabel.htm).
