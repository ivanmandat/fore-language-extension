# IChartArea


## Описание


Интерфейс IChartArea содержит свойства для определения параметров области диаграммы.


## Свойства


 Имя свойства
 Краткое описание


 ![](../../Property_Image.gif)
 [AutoPadding](IChartArea.AutoPadding.htm)
 Свойство AutoPadding определяет величины внутренних отступов от краев диаграммы, устанавливаемые автоматически.


 ![](../../Property_Image.gif)
 [Padding](IChartArea.Padding.htm)
 Свойство Padding определяет пользовательские величины внутренних отступов от краев диаграммы.
