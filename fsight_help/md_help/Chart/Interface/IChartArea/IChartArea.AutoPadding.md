# IChartArea.AutoPadding


## Синтаксис


AutoPadding: Boolean;


## Описание


Свойство AutoPadding определяет величины внутренних отступов от краев диаграммы, устанавливаемые автоматически.


## Комментарии


По умолчанию свойству AutoPadding установлено значение True, при этом величины внутренних отступов от каждого края диаграммы равны 2,5 мм.


Данное свойство сохраняется с диаграммой. Если загружается объект предыдущей версии, то для него устанавливаются свойства AutoPadding и Padding по умолчанию.
