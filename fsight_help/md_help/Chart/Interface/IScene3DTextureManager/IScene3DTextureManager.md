# IScene3DTextureManager


## Описание


Интерфейс IScene3DTextureManager
 содержит свойства и методы для работы с текстурами.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IScene3DTextureManager.Count.htm)
		 Свойство Count возвращает
		 количество текстур в списке.


		 ![](../../Property_Image.gif)
		 [PreviewImage](IScene3DTextureManager.PreviewImage.htm)
		 Свойство PreviewImage
		 позволяет работать с текстурой как с картинкой.


		 ![](../../Property_Image.gif)
		 [PreviewSize](IScene3DTextureManager.PreviewSize.htm)
		 Свойство PreviewSize
		 определяет размер картинки предварительного просмотра текстуры.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Delete](IScene3DTextureManager.Delete.htm)
		 Метод Delete удаляет
		 текстуру из списка по заданному индексу.


		 ![](../../Sub_Image.gif)
		 [LoadFromFile](IScene3DTextureManager.LoadFromFile.htm)
		 Метод LoadFromFile
		 позволяет загрузить текстуры из файла в конец списка текстур.
