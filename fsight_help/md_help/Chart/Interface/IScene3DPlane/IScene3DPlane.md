# IScene3DPlane


## Описание


Интерфейс IScene3DPlane содержит
 свойства для определения параметров плоскости сцены.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Offset](IScene3DPlane.Offset.htm)
		 Свойство Offset определяет
		 смещение уровня относительно начала координат [-0.5; 0.5] (вдоль
		 вектора собственной нормали).


		 ![](../../Property_Image.gif)
		 [Type](IScene3DPlane.Type.htm)
		 Свойство Type определяет
		 ориентацию уровня относительно плоскостей системы координат.


## Свойства, унаследованные от [IScene3DPrimitive](../IScene3DPrimitive/IScene3DPrimitive.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BitmapIndex](../IScene3DPrimitive/IScene3DPrimitive.BitmapIndex.htm)
		 Свойство BitmapIndex
		 определяет индекс картинки, которая находится в коллекции текстур
		 IScene3DTextureManager.


		 ![](../../Property_Image.gif)
		 [Color](../IScene3DPrimitive/IScene3DPrimitive.Color.htm)
		 Свойство Color определяет
		 цвет объекта.


		 ![](../../Property_Image.gif)
		 [Selected](../IScene3DPrimitive/IScene3DPrimitive.Selected.htm)
		 Свойство Selected определяет
		 признак выделения объекта.


		 ![](../../Property_Image.gif)
		 [Visible](../IScene3DPrimitive/IScene3DPrimitive.Visible.htm)
		 Свойство Visible определяет
		 видимость объекта.
