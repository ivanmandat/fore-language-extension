# IChartMarker.DisplayShadow


## Синтаксис


DisplayShadow: Boolean;


## Описание


Свойство DisplayShadow определяет признак отображения тени у маркера.


## Комментарии


Если значение данного свойства True, то у маркеров на графике будет отображена тень, при значении False - тень не отображается.
