# IChartMarker.Size


## Синтаксис


Size: Double;


## Описание


Свойство Size определяет размер
 маркера.


## Пример


В рассматриваемом примере предполагается, что существует объект Marker типа IChartMarker.


	Sub Sz;

	Var

	    Marker: IChartMarker;

	Begin

	    Marker.Size := 9;

	End Sub Sz;


После выполнения примера размер маркеров на графике изменится и будет
 9 мм.
