# IChartMarker


## Описание


Интерфейс IChartMarker содержит
 свойства для определения параметров маркеров диаграммы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BorderPen](IChartMarker.BorderPen.htm)
		 Свойство BordenPen
		 определяет обводку маркера.


		 ![](../../Property_Image.gif)
		 [DisplayShadow](IChartMarker.DisplayShadow.htm)
		 Свойство DisplayShadow
		 определяет признак отображения тени у маркера.


		 ![](../../Property_Image.gif)
		 [FillBrush](IChartMarker.FillBrush.htm)
		 Свойство FillBrush
		 определяет заливку маркера.


		 ![](../../Property_Image.gif)
		 [MarkerType](IChartMarker.MarkerType.htm)
		 Свойство MarkerType
		 определяет тип маркера.


		 ![](../../Property_Image.gif)
		 [Shadow](IChartMarker.Shadow.htm)
		 Свойство Shadow определяет
		 параметры тени у маркера.


		 ![](../../Property_Image.gif)
		 [Size](IChartMarker.Size.htm)
		 Свойство Size определяет
		 размер маркера.


		 ![](../../Property_Image.gif)
		 [Symbol](IChartMarker.Symbol.htm)
		 Свойство Simbol позволяет
		 задать маркер в виде символа.
