# IScene3DNamedSphereGroup


## Описание


Интерфейс IScene3DNamedSphereGroup
 содержит свойства для определения параметров группы именованных точек.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ArrowLength](IScene3DNamedSphereGroup.ArrowLength.htm)
		 Свойство ArrowLength
		 определяет длину стрелки коннектора.


		 ![](../../Property_Image.gif)
		 [ArrowRadius](IScene3DNamedSphereGroup.ArrowRadius.htm)
		 Свойство ArrowRadius
		 определяет радиус раскрытия стрелки коннектора.


		 ![](../../Property_Image.gif)
		 [Name](IScene3DNamedSphereGroup.Name.htm)
		 Свойство Name определяет
		 имя группы.


		 ![](../../Property_Image.gif)
		 [Radius](IScene3DNamedSphereGroup.Radius.htm)
		 Свойство Radius определяет
		 радиус коннектора.


		 ![](../../Property_Image.gif)
		 [SphereCount](IScene3DNamedSphereGroup.SphereCount.htm)
		 Свойство SphereCount
		 определяет количество точек в именованной группе.


		 ![](../../Property_Image.gif)
		 [SphereItem](IScene3DNamedSphereGroup.SphereItem.htm)
		 Свойство SphereItem
		 возвращает точку именованной группы по заданному индексу.


		 ![](../../Property_Image.gif)
		 [StartArrowOffset](IScene3DNamedSphereGroup.StartArrowOffset.htm)
		 Свойство StartArrowOffset
		 определяет смещение стрелки относительно начальной точки коннектора.


		 ![](../../Property_Image.gif)
		 [TextStyle](IScene3DNamedSphereGroup.TextStyle.htm)
		 Свойство TextStyle
		 определяет параметры надписей.
