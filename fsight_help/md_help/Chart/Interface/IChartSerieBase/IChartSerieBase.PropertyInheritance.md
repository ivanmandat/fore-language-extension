# IChartSerieBase.PropertyInheritance


## Синтаксис


		PropertyInheritance: [ChartSeriePropertyInheritance](../../Enums/ChartSeriePropertyInheritance.htm);


## Описание


Свойство PropertyInheritance
 определяет, какие настройки ряда будут наследоваться от настроек ряда
 по умолчанию.


## Комментарии


При создании новых рядов все настройки берутся из настроек [ряда
 по умолчанию](../IChartSeries/IChartSeries.DefaultSerie.htm).
