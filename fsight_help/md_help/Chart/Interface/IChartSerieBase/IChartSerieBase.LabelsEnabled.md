# IChartSerieBase.LabelsEnabled


## Синтаксис


		LabelsEnabled: Boolean;


## Описание


Свойство LabelsEnabled определяет
 отображения подписей данных ряда данных.


## Комментарии


Позволяет включить или отключить отображение подписей данных . Если
 значение данного свойства True,
 то у ряда для всех его точек на графике будут отображаться подписи данных
 , при значении False - подписи
 данных не отображается.
