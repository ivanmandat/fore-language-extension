# IChartSerieBase.DefaultLabel


## Синтаксис


		DefaultLabel: [IChartLabel](../IChartLabel/IChartLabel.htm);


## Описание


Свойство DefaultLabel определяет
 настройки подписей данных ряда по умолчанию.


## Комментарии


При установке настроек подписи по умолчанию свойство [Selected](../IChartAbstractItem/IChartAbstractItem.Selected.htm)
 недоступно, при попытке его использования будет выведено сообщение «Свойство
 поддерживается только для каждой подписи по отдельности».
