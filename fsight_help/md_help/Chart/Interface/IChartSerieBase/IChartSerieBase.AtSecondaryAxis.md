# IChartSerieBase.AtSecondaryAxis


## Синтаксис


		AtSecondaryAxis: Boolean;


## Описание


Свойство AtSecondaryAxis определяет
 отношение ряда к дополнительной оси.


## Комментарии


Если значение данного свойства True,
 то ряд на графике будет относится к дополнительной оси, при значении False ряд относится к общей оси
 других рядов.


По умолчанию свойство имеет значение False.
