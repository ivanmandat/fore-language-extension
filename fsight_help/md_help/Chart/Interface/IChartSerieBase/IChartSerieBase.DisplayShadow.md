# IChartSerieBase.DisplayShadow


## Синтаксис


		DisplayShadow: Boolean;


## Описание


Свойство DisplayShadow определяет
 признак отображения тени у ряда данных.


## Комментарии


Если значение данного свойства True,
 то на графике будет отображена тень ряда, при значении False
 - нет.
