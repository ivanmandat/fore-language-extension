# IChartSerieBase.AllowEdit


## Синтаксис


		AllowEdit: Boolean;


## Описание


Свойство AllowEdit определяет
 признак редактирования значений ряда прямо на диаграмме.


## Комментарии


Если значение данного свойства True,
 то значения ряда можно будет редактировать, если значение False,
 то редактирование значений ряда прямо на диаграмме будет невозможно.


По умолчанию свойство имеет значение True.
