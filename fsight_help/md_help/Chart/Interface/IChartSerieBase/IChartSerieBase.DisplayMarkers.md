# IChartSerieBase.DisplayMarkers


## Синтаксис


		DisplayMarkers: Boolean;


## Описание


Свойство DisplayMarkers определяет
 признак отображения маркеров у ряда данных.


## Комментарии


Если значение данного свойства True,
 то у ряда на графике будут отображаться маркеры, при значение False
 - маркеры не отображаются.
