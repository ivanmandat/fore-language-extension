# IChartSerieBase.Is3DBorder


## Синтаксис


		Is3DBorder: Boolean;


## Описание


Свойство Is3DBorder определяет
 признак использования 3D-стиля границы ряда.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Используется 3D-стиль границы ряда;


	- False. Не используется
	 3D-стиль границы ряда.
