# IChartSerieBase


Сборка: Chart;


## Описание


Интерфейс IChartSerieBase содержит
 базовые свойства и методы различных рядов данных диаграммы.


## Иерархия наследования


           [IChartAbstractItem](../IChartAbstractItem/IChartAbstractItem.htm)


           IChartSerieBase


## Комментарии


Интерфейс является базовым для интерфейса [IChartSerie](../IChartSerie/IChartSerie.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AllowEdit](IChartSerieBase.AllowEdit.htm)
		 Свойство AllowEdit
		 определяет признак редактирования значений ряда прямо на диаграмме.


		 ![](../../Property_Image.gif)
		 [AtSecondaryAxis](IChartSerieBase.AtSecondaryAxis.htm)
		 Свойство AtSecondaryAxis
		 определяет отношение ряда к дополнительной оси.


		 ![](../../Property_Image.gif)
		 [AutoRotateMarker](IChartSerieBase.AutoRotateMarker.htm)
		 Свойство AutoRotateMarker
		 определяет, установлен ли на диаграмме автоматический поворот
		 [маркера](UiDiagrams.chm::/Tuning_format/Format_line/Markers_Setup.htm)
		 для ряда данных.


		 ![](../../Property_Image.gif)
		 [Background](IChartSerieBase.Background.htm)
		 Свойство Background
		 определяет параметры заливки для ряда данных.


		 ![](../../Property_Image.gif)
		 [BorderPen](IChartSerieBase.BorderPen.htm)
		 Свойство BordenPen
		 определяет границу (обводку) ряда данных на графике.


		 ![](../../Property_Image.gif)
		 [DefaultLabel](IChartSerieBase.DefaultLabel.htm)
		 Свойство DefaultLabel
		 определяет настройки подписи данных по умолчанию.


		 ![](../../Property_Image.gif)
		 [DisplayInLegend](IChartSerieBase.DisplayInLegend.htm)
		 Свойство DisplayInLegend
		 определяет признак отображения ряда в легенде.


		 ![](../../Property_Image.gif)
		 [DisplayMarkers](IChartSerieBase.DisplayMarkers.htm)
		 Свойство DisplayMarkers
		 определяет признак отображения маркеров у ряда данных.


		 ![](../../Property_Image.gif)
		 [DisplayShadow](IChartSerieBase.DisplayShadow.htm)
		 Свойство DisplayShadow
		 определяет признак отображения тени у ряда данных.


		 ![](../../Property_Image.gif)
		 [EmptyLinePen](IChartSerieBase.EmptyLinePen.htm)
		 Свойство EmptyLinePen
		 определяет параметры линии ряда данных c отсутствующими данными.


		 ![](../../Property_Image.gif)
		 [Is3DBorder](IChartSerieBase.Is3DBorder.htm)
		 Свойство Is3DBorder
		 определяет признак использования 3D-стиля границы ряда.


		 ![](../../Property_Image.gif)
		 [LabelsEnabled](IChartSerieBase.LabelsEnabled.htm)
		 Свойство LabelsEnabled
		 определяет отображение подписей данных ряда данных.


		 ![](../../Property_Image.gif)
		 [LinePen](IChartSerieBase.LinePen.htm)
		 Свойство LinePen определяет
		 параметры линии графика ряда данных.


		 ![](../../Property_Image.gif)
		 [Marker](IChartSerieBase.Marker.htm)
		 Свойство Marker позволяет
		 работать с маркерами ряда диаграммы.


		 ![](../../Property_Image.gif)
		 [PieDistanceCoef](IChartSerieBase.PieDistanceCoef.htm)
		 Свойство PieDistanceCoef
		 определяет расстояние от центра круговой диаграммы до крайней
		 дольки ряда.


		 ![](../../Property_Image.gif)
		 [PropertyInheritance](IChartSerieBase.PropertyInheritance.htm)
		 Свойство PropertyInheritance
		 определяет, какие настройки ряда будут наследоваться от настроек
		 ряда по умолчанию.


		 ![](../../Property_Image.gif)
		 [SerieType](IChartSerieBase.SerieType.htm)
		 Свойство SerieType

		 ![](../../Property_Image.gif)

		 ![](../../Property_Image.gif)
		 [StepLineType](IChartSerieBase.StepLineType.htm)
		 Свойство StepLineType

		 ![](../../Property_Image.gif)
		 [UseSplineSmoothing](IChartSerieBase.UseSplineSmooth.htm)
		 Свойство UseSplineSmoothing
		 определяет признак сглаживания ряда данных.


		 ![](../../Property_Image.gif)
		 [Visible](IChartSerieBase.Visible.htm)
		 Свойство Visible определяет
		 видимость ряда данных.
