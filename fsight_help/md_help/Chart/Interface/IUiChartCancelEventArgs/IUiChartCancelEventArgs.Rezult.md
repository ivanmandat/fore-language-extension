# IUiChartCancelEventArgs.Result


## Синтаксис


Result: Boolean;


## Описание


Свойство Result определяет признак
 обработки события.


## Комментарии


Если значение данного свойства True,
 то выполняется обработка этого события. Если значение данного свойства
 False, то соответственно не выполняется.


По умолчанию свойство имеет значение False.
