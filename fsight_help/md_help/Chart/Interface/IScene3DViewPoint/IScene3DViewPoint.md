# IScene3DViewPoint


## Описание


Интерфейс IScene3DViewPoint
 содержит свойства для настройки положения и направления камеры сцены.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [HorizontalRotation](IScene3DViewPoint.HorizontalRotation.htm)
		 Свойство HorizontalRotation
		 определяет горизонтальный поворот вокруг сцены.


		 ![](../../Property_Image.gif)
		 [Scale](IScene3DViewPoint.Scale.htm)
		 Свойство Scale определяет
		 масштаб отображения сцены.


		 ![](../../Property_Image.gif)
		 [VerticalRotation](IScene3DViewPoint.VerticalRotation.htm)
		 Свойство VerticalRotation
		 определяет вертикальный поворот вокруг сцены.
