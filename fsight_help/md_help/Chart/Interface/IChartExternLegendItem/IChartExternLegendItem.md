# IChartExternLegendItem


## Описание


Интерфейс IChartExternLegendItem
 содержит свойства элемента внешней легенды диаграмм.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Name](IChartExternLegendItem.Name.htm)
		 Свойство Name возвращает
		 наименование элемента внешней легенды диаграмм.


		 ![](../../Property_Image.gif)
		 [Thumbnail](IChartExternLegendItem.Thumbnail.htm)
		 Свойство Thumbnail
		 возвращает пиктограмму элемента внешней легенды диаграмм.


		 ![](../../Property_Image.gif)
		 [Visible](IChartExternLegendItem.Visible.htm)
		 Свойство Visible определяет
		 признак отображения элемента легенды диаграмм.
