# IChartBackgroundInfo.Type


## Синтаксис


Type: [ChartBackgroundType](../../Enums/ChartBackgroundType.htm);


## Описание


Свойство Type позволяет определять
 тип заливки.


## Комментарии


Для разных [ компонентов
 диаграммы](uidiagrams.chm::/UiDiagrams_basic_concept.htm) доступны разные типы заливки. Каждому
 типу заливки соответствует определенный набор свойств интерфейса [IChartBackgroundInfo](IChartBackgroundInfo.htm).


Значение свойства по умолчанию «None»
 (нет фона).
