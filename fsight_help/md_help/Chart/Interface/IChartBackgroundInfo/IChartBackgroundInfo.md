# IChartBackgroundInfo


## Описание


Интерфейс IChartBackgroundInfo
 содержит свойства, предназначенные для определения параметров заливки
 компонентов диаграммы.


## Комментарии


Набор свойств, который необходимо задать для заливки ряда, зависит от
 установленного [типа заливки](../../Enums/ChartBackgroundType.htm)
 .
