# IScene3DSeriesClickEventArgs


## Описание


Интерфейс IScene3DSeriesClickEventArgs
 возвращает описание графических объектов, для которых наступило событие
 [OnSeriesDoubleClick](../../Class/Chart3DBox/IChart3DBoxEvents.OnSeriesDoubleClick.htm)
 (двойной щелчок мыши по ряду сцены на форме), которое возвращает
 объект класса [Chart3DSeriesClickEventArgs](../../Class/Chart3DSeriesClickEventArgs/Chart3DSeriesClickEventArgs.htm).


## Комментарии


Данный интерфейс работает только с однородными коллекциями графических
 объектов (коллекция плоскостей, коллекция сфер, коллекция поверхностей
 и т.д.).
