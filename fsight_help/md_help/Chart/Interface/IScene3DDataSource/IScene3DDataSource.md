# IScene3DDataSource


## Описание


Интерфейс IScene3DDataSource
 содержит свойства для определения источника данных поверхности.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ArrayX](IScene3DDataSource.ArrayX.htm)
		 Свойство ArrayX определяет
		 все значения по оси OX.


		 ![](../../Property_Image.gif)
		 [ArrayY](IScene3DDataSource.ArrayY.htm)
		 Свойство ArrayY определяет
		 все значения по оси OY.


		 ![](../../Property_Image.gif)
		 [ColumnCount](IScene3DDataSource.ColumnCount.htm)
		 Свойство ColumnCount
		 возвращает число столбцов в источнике данных (данные по оси OX).


		 ![](../../Property_Image.gif)
		 [NewColumn](IScene3DDataSource.NewColumn.htm)
		 Свойство NewColumn
		 добавляет новый столбец в источник данных (значение по оси OX).


		 ![](../../Property_Image.gif)
		 [NewRow](IScene3DDataSource.NewRow.htm)
		 Свойство NewRow добавляет
		 новую строку в источник данных (значение по оси OY).


		 ![](../../Property_Image.gif)
		 [RowCount](IScene3DDataSource.RowCount.htm)
		 Свойство RowCount возвращает
		 число строк в источнике данных (данные по оси OY).


		 ![](../../Property_Image.gif)
		 [ValueX](IScene3DDataSource.ValueX.htm)
		 Свойство ValueX изменяет
		 значение по оси OX в источнике данных.


		 ![](../../Property_Image.gif)
		 [ValueY](IScene3DDataSource.ValueY.htm)
		 Свойство ValueY изменяет
		 значение по оси OY в источнике данных.


		 ![](../../Property_Image.gif)
		 [ValueZ](IScene3DDataSource.ValueZ.htm)
		 Свойство ValueZ определяет
		 значение по оси OZ в источнике данных.
