# IChartAxisLevelLineLabel.MaskText


## Синтаксис


MaskText: String;


## Описание


Свойство MaskText определяет
 маску, в соответствии с которой будет формироваться текст подписи.


## Комментарии


В маске может быть указан любой текст, а также специальные подстановки,
 которые при отображении будут заменены фактическими значениями. Для использования
 доступны следующие подстановки:


	- %Computation. Наименование
	 выбранной функции;


	- %Value. Значение функции.
