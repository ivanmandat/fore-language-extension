# IChartAxisLevelLineLabel


Сборка: Chart;


## Описание


Интерфейс IChartAxisLevelLineLabel
 используется для определения параметров подписи линии уровня.


## Иерархия наследования


           IChartAxisLevelLineLabel


## Комментарии


Для задания параметров линии уровня на диаграмме используйте интерфейс
 [IChartAxisLevelLine](../IChartAxisLevelLine/IChartAxisLevelLine.htm).
