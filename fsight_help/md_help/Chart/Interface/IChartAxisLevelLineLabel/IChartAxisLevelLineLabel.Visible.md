# IChartAxisLevelLineLabel.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет
 признак видимости подписи.


## Комментарии


Допустимые значения:


	- True. Подпись
	 отображается;


	- False.
	 Значение по умолчанию. Подпись не отображается.
