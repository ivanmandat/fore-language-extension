# IUiChartSerieGroupIndexEventArgs

## Описание


Интерфейс IUiChartSerieGroupIndexEventArgs
 используется для работы с параметрами события [UiChart.OnGetSerieGroupIndex](../../Class/UiChart/UiChart.OnGetSerieGroupIndex.htm).


## Иерархия наследования


           [IEventArgs](ForeSys.chm::/Interface/IEventArgs/IEventArgs.htm)

## Комментарии


Событие [UiChart.OnGetSerieGroupIndex](../../Class/UiChart/UiChart.OnGetSerieGroupIndex.htm)
 позволяет определить индекс группы рядов диаграммы.
