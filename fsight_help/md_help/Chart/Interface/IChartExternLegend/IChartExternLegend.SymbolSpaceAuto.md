# IChartExternLegend.SymbolSpaceAuto


## Синтаксис


SymbolSpaceAuto: Boolean;


## Описание


Свойство SymbolSpaceAuto определяет, будет ли расстояние между значком и текстом для элементов внешней легенды установлено автоматически.


## Комментарии


 Если значение данного свойства True, то расстояние между значком и текстом определяется автоматически, при значении False расстояние устанавливается свойством [IChartExternLegend.SymbolSpace](IChartExternLegend.SymbolSpace.htm).
