# IChartExternLegend.ColIntervalAuto


## Синтаксис


ColIntervalAuto: Boolean;


## Описание


Свойство ColIntervalAuto определяет, будет ли расстояние между столбцами внешней легенды установлено автоматически.


## Комментарии


Если значение данного свойства True, то расстояние между столбцами определяется автоматически, при значении False расстояние устанавливается свойством [IChartExternLegend.ColInterval](IChartExternLegend.ColInterval.htm).
