# IChartExternLegend.RowIntervalAuto


## Синтаксис


RowIntervalAuto: Boolean;


## Описание


Свойство RowIntervalAuto определяет, будет ли расстояние между строками внешней легенды установлено автоматически.


## Комментарии


Если значение данного свойства True, то расстояние между строками определяется автоматически, при значении False расстояние устанавливается свойством [IChartExternLegend.RowInterval](IChartExternLegend.RowInterval.htm).
