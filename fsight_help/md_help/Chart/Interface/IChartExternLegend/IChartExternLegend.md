# IChartExternLegend


## Описание


Интерфейс IChartExternLegend
 содержит свойства для определения параметров внешней легенды диаграмм.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AxisIcon](IChartExternLegend.AxisIcon.htm)


		 Свойство AxisIcon определяет
		 признак отображения в легенде значка, который указывает по какой
		 оси расположен ряд (основной или дополнительной).


		 ![](../../Property_Image.gif)
		 [BackgroundInfo](IChartExternLegend.BackgroundInfo.htm)


		 Свойство BackgroundInfo
		 определяет свойства заливки внешней легенды.


		 ![](../../Property_Image.gif)
		 [BorderPen](IChartExternLegend.BorderPen.htm)


		 Свойство BorderPen
		 определяет параметры границ внешней легенды.


		 ![](../../Property_Image.gif)
		 [Chart](IChartExternLegend.Chart.htm)


		 Свойство Chart возвращает
		 объект типа IChart, которому
		 принадлежит элемент легенды.


		 ![](../../Property_Image.gif)
		 [ChartCount](IChartExternLegend.ChartCount.htm)


		 Свойство ChartCount
		 возвращает количество диаграмм, участвующих в построении внешней
		 легенды диаграмм.


		 ![](../../Property_Image.gif)
		 [ColCount](IChartExternLegend.ColCount.htm)


		 Свойство ColCount задает
		 количество столбцов, которые будут использованы для размещения
		 элементов внешней легенды.


		 [![](../../Property_Image.gif)](IChartExternLegend.ColInterval.htm)
		 [ColInterval](IChartExternLegend.ColInterval.htm)


		 Свойство ColInterval
		 задает расстояние между столбцами легенды диаграмм.


		 ![](../../Property_Image.gif)
		 [ColIntervalAuto](IChartExternLegend.ColIntervalAuto.htm)


		 Свойство ColIntervalAuto
		 определяет, будет ли расстояние между столбцами внешней легенды
		 установлено автоматически.


		 ![](../../Property_Image.gif)
		 [ElementHorizontalAlignment](IChartExternLegend.ElementHorizontalAlignment.htm)


		 Свойство ElementHorizontalAlignment определяет
		 тип горизонтального выравнивания элементов внешней легенды.


		 ![](../../Property_Image.gif)
		 [ElementVerticalAlignment](IChartExternLegend.ElementVerticalAlignment.htm)


		 Свойство ElementVerticalAlignment
		 определяет тип вертикального выравнивания элементов внешней легенды.


		 ![](../../Property_Image.gif)
		 [Font](IChartExternLegend.Font.htm)


		 Свойство Font определяет
		 шрифт текста внешней легенды диаграмм.


		 ![](../../Property_Image.gif)
		 [FontColor](IChartExternLegend.FontColor.htm)


		 Свойство FontColor
		 определяет цвет шрифта внешней легенды диаграмм.


		 ![](../../Property_Image.gif)
		 [InternalItems](IChartExternLegend.InternalItems.htm)


		 Свойство InternalItems
		 возвращает коллекцию отображаемых элементов внешней легенды.


		 ![](../../Property_Image.gif)
		 [IsDirty](IChartExternLegend.IsDirty.htm)


		 Свойство IsDirty определяет
		 признак наличия несохраненных изменений во внешней легенде диаграмм.


		 [![](../../Property_Image.gif)](IChartExternLegend.IsMultiline.htm)
		 [IsMultiline](IChartExternLegend.IsMultiline.htm)


		 Свойство IsMultiline
		 определяет, будут ли длинные наименования элементов во внешней
		 легенде расположены в несколько строк.


		 ![](../../Property_Image.gif)
		 [ItemPlacement](IChartExternLegend.ItemPlacement.htm)


		 Свойство ItemPlacement
		 определяет расположение элементов внешней легенды.


		 ![](../../Property_Image.gif)
		 [ItemSeparator](IChartExternLegend.ItemSeparator.htm)


		 Свойство ItemSeparator
		 позволяет задать разделитель текста для объединенных элементов
		 внешней диаграмм.


		 ![](../../Property_Image.gif)
		 [RowCount](IChartExternLegend.RowCount.htm)


		 Свойство RowCount задает
		 количество строк внешней легенды диаграмм.


		 ![](../../Property_Image.gif)
		 [RowInterval](IChartExternLegend.RowInterval.htm)


		 Свойство RowInterval
		 задает расстояние между строками внешней легенды диаграмм.


		 ![](../../Property_Image.gif)
		 [RowIntervalAuto](IChartExternLegend.RowIntervalAuto.htm)


		 Свойство RowIntervalAuto
		 определяет, будет ли расстояние между строками внешней легенды
		 установлено автоматически.


		 ![](../../Property_Image.gif)
		 [SymbolSize](IChartExternLegend.SymbolSize.htm)


		 Свойство SymbolSize
		 определяет размер значка элементов легенды диаграмм.


		 ![](../../Property_Image.gif)
		 [SymbolSpace](IChartExternLegend.SymbolSpace.htm)


		 Свойство SymbolSpace
		 задает расстояние между значком и текстом для элементов легенды
		 диаграмм.


		 ![](../../Property_Image.gif)
		 [SymbolSpaceAuto](IChartExternLegend.SymbolSpaceAuto.htm)


		 Свойство SymbolSpaceAuto
		 определяет, будет ли расстояние между значком и текстом для элементов
		 внешней легенды установлено автоматически.


		 ![](../../Property_Image.gif)
		 [UndoRedo](IChartExternLegend.UndoRedo.htm)


		 Свойство UndoRedo определяет
		 параметры стека изменений (отмены/повтора) внешней легенды диаграммы.


		 ![](../../Property_Image.gif)
		 [UseSerieDesign](IChartExternLegend.UseSerieDesign.htm)


		 Свойство UseSerieDesign
		 определяет признак использования оформления рядов внутренней легенды
		 в соответствии с оформлением рядов на диаграмме.


		 ![](../../Property_Image.gif)
		 [UseSerieIcon](IChartExternLegend.UseSerieIcon.htm)


		 Свойство UseSerieIcon
		 позволяет отображать вместо значков ряда цветовой квадрат.


## Методы


		 Имя свойства
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddChart](IChartExternLegend.AddChart.htm)


		 Метод AddChart осуществляет
		 добавление диаграммы, передаваемой посредством параметра Value, в список диаграмм, подключенных
		 к внешней легенде.


		 ![](../../Sub_Image.gif)
		 [Clear](IChartExternLegend.Clear.htm)


		 Метод Clear осуществляет
		 удаление всех диаграмм из списка подключенных к внешней легенде.


		 ![](../../Sub_Image.gif)
		 [RemoveChart](IChartExternLegend.RemoveChart.htm)


		 Метод RemoveChart осуществляет
		 удаление диаграммы, передаваемой посредством параметра Value,
		 из списка диаграмм, подключенных к внешней легенде.
