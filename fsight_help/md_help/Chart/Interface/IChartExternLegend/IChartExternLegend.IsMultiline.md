# IChartExternLegend.IsMultiline


## Синтаксис


IsMultiline: Boolean;


## Описание


Свойство IsMultiline определяет, будут ли длинные наименования элементов во внешней легенде расположены в несколько строк.


## Комментарии


По умолчанию свойство имеет значение False и наименования элементов внешней легенды располагаются в одну строку.


Если значение данного свойства True, то наименования элементов будут расположены в несколько строк при условии, что текст не может быть размещен в одну строку.
