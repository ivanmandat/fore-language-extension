# IChartExternLegend.RemoveChart


## Синтаксис


RemoveChart (Value: [IChart](../IChart/IChart.htm));


## Параметры


Value. Значение, определяющее диаграмму, которую необходимо удалить из списка диаграмм, подключенных к внешней легенде.


## Описание


Метод RemoveChart осуществляет удаление диаграммы, передаваемой посредством параметра Value, из списка диаграмм, подключенных к внешней легенде.

Использование метода приведено в описании свойства [IChartExternLegend.Chart](IChartExternLegend.Chart.htm).
