# IScene3DSpheres


## Описание


Интерфейс IScene3DSpheres содержит
 свойства для определения параметров коллекции сфер (точек).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IScene3DSpheres.Count_.htm)
		 Свойство Count определяет
		 количество сфер в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IScene3DSpheres.Item.htm)
		 Свойство Item возвращает
		 сферу из коллекции по заданному индексу.
