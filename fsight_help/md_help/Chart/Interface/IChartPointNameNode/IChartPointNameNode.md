# IChartPointNameNode


## Описание


Интерфейс IChartPointNameNode
 содержит свойства для работы с уровнем иерархии для наименований оси категорий.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IChartPointNameNode.Count.htm)


		 Свойство Count возвращает
		 число значений на данном уровне иерархии.


		 ![](../../Property_Image.gif)
		 [Format](IChartPointNameNode.Format.htm)


		 Свойство Format возвращает
		 формат значений элементов данного уровня иерархии.


		 ![](../../Property_Image.gif)
		 [Level](IChartPointNameNode.Level.htm)


		 Свойство Level возвращает
		 текущий уровень иерархии.


		 ![](../../Property_Image.gif)
		 [Name](IChartPointNameNode.Name.htm)


		 Свойство Name возвращает
		 наименование уровня.


		 ![](../../Property_Image.gif)
		 [PointIndex](IChartPointNameNode.PointIndex.htm)


		 Свойство PointIndex
		 возвращает номер точки среди всех точек диаграммы.


		 ![](../../Property_Image.gif)
		 [Required](IChartPointNameNode.Required.htm)


		 Свойство Required определяет,
		 обязательно ли уровень должен отображаться на шкале.


		 ![](../../Property_Image.gif)
		 [SubLevel](IChartPointNameNode.SubLevel.htm)


		 Свойство SubLevel возвращает
		 уровень, расположенный ниже.


		 ![](../../Property_Image.gif)
		 [Value](IChartPointNameNode.Value.htm)


		 Свойство Value возвращает
		 значение уровня.
