## Описание


Интерфейс IUiChartParentSerieIndexEventArgs
 используется для работы с параметрами события [UiChart.OnGetParentSerieIndex](../../Class/UiChart/UiChart.OnGetParentSerieIndex.htm).

 позволяет определить индекс родительского ряда.
