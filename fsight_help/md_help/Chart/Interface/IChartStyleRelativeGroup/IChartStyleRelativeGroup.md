# IChartStyleRelativeGroup


Сборка: Chart;


## Описание


Интерфейс IChartStyleRelativeGroup
 содержит свойства и методы, предназначенные для работы с относительными
 группами рядов диаграммы.


## Иерархия наследования


           IChartStyleRelativeGroup


## Комментарии


Относительная группа формируется суммированием значений рядов. Итоговая
 диаграмма в этом случае выглядит следующим образом:


![](../IChartStyleAbsoluteGroup/MakeRelative.gif)


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IChartStyleRelativeGroup.Count.htm)


		 Свойство Count возвращает
		 количество рядов в указанной относительной группе рядов диаграммы.


		 ![](../../Property_Image.gif)
		 [Item](IChartStyleRelativeGroup.Item.htm)


		 Свойство Item возвращает
		 абсолютный индекс ряда по его индексу в относительной группе рядов
		 диаграммы, к которой он принадлежит.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Move](IChartStyleRelativeGroup.Move.htm)


		 Метод Move перемещает
		 ряд в пределах одной относительной группы рядов диаграммы.


		 ![](../../Sub_Image.gif)
		 [MoveTo](IChartStyleRelativeGroup.MoveTo.htm)


		 Метод MoveTo перемещает
		 ряд из одной относительной группы рядов диаграммы в другую.
