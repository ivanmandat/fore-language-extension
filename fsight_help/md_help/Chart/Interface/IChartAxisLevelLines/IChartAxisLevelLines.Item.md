# IChartAxisLevelLines.Item


## Синтаксис


Item(Index: Integer): [IChartAxisLevelLine](../IChartAxisLevelLine/IChartAxisLevelLine.htm);


## Параметры


Index. Индекс линии уровня.
 Нумерация начинается с нуля.


## Описание


Метод Item возвращает линию
 уровня диаграммы по заданному индексу.
