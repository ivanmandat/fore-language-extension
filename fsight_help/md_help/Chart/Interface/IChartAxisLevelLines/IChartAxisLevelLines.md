# IChartAxisLevelLines


Сборка: Chart;


## Описание


Интерфейс IChartAxisLevelLines
 содержит свойства и методы для задания параметров коллекции линий уровня
 на диаграмме.


## Иерархия наследования


           IChartAxisLevelLines


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IChartAxisLevelLines.Count.htm)
		 Свойство Count возвращает
		 количество линий уровня, расположенных на диаграмме.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IChartAxisLevelLines.Add.htm)
		 Метод Add добавляет
		 новую линию уровня на диаграмму.


		 ![](../../Sub_Image.gif)
		 [Item](IChartAxisLevelLines.Item.htm)
		 Метод Item возвращает
		 линию уровня диаграммы по заданному индексу.


		 ![](../../Sub_Image.gif)
		 [Remove](IChartAxisLevelLines.Remove.htm)
		 Метод Remove удаляет
		 линию уровня диаграммы по заданному индексу.
