# IChartLegend.SymbolSpaceAuto


## Синтаксис


SymbolSpaceAuto: Boolean;


## Описание


Свойство SymbolSpaceAuto возвращает признак автоматического расчёта расстояния между значком легенды и его подписью.


## Комментарии


По умолчанию свойство имеет значение True.
