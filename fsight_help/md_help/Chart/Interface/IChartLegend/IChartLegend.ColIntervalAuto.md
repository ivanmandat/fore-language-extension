# IChartLegend.ColIntervalAuto


## Синтаксис


ColIntervalAuto: Boolean;


## Описание


Свойство ColIntervalAuto возвращает признак автоматического расчёта интервалов для столбцов в легенде.


## Комментарии


По умолчанию свойство имеет значение True.
