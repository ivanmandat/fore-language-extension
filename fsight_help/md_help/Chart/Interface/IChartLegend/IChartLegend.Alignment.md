# IChartLegend.Alignment


## Синтаксис


Alignment: [ChartLegendAlignment](../../Enums/ChartLegendAlignment.htm);


## Описание


Свойство Alignment определяет размещение легенды на области построения. По умолчанию легенда располагается внизу области построения диаграммы.
