# IChartLegend.RowIntervalAuto


## Синтаксис


RowIntervalAuto: Boolean;


## Описание


Свойство RowIntervalAuto возвращает признак автоматического расчёта расстояний между строками в легенде.


## Комментарии


По умолчанию свойство имеет значение True.
