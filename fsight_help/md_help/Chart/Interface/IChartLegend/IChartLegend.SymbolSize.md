# IChartLegend.SymbolSize


## Синтаксис


SymbolSize: [IGxSizeF](ModDrawing.chm::/Interface/IGxSizeF/IGxSizeF.htm);


## Описание


Свойство SymbolSize определяет размер значка элементов легенды диаграмм.


## Комментарии


Размер задается в миллиметрах.
