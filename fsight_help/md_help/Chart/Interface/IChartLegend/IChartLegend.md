# IChartLegend


Сборка: Chart;


## Описание


Интерфейс IChartLegend содержит
 свойства для определения параметров легенды графика.


## Иерархия наследования

           [IChartPlacedItem](ichartplaceditem.chm::/ichartplaceditem.htm)


           IChartLegend


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Alignment](IChartLegend.Alignment.htm)


		 Свойство Alignment
		 определяет размещение легенды на области построения.


		 ![](../../Property_Image.gif)
		 [AxisIcon](IChartLegend.AxisIcon.htm)


		 Свойство AxisIcon определяет
		 признак отображения в легенде значка, который указывает по какой
		 оси расположен ряд (основной или дополнительной).


		 ![](../../Property_Image.gif)
		 [ColCount](IChartLegend.ColCount.htm)


		 Свойство ColCount определяет
		 количество столбцов в легенде.


		 ![](../../Property_Image.gif)
		 [ElementVerticalAlignment](IChartLegend.ElementVerticalAlignment.htm)


		 Свойство ElementVerticalAlignment
		 определяет тип вертикального выравнивания элементов внутренней
		 легенды.


		 ![](../../Property_Image.gif)
		 [IsMultiline](IChartLegend.IsMultiline.htm)


		 Свойство IsMultiline
		 определяет признак переноса текста по словам в легенде.


		 ![](../../Property_Image.gif)
		 [IsOverlapped](IChartLegend.IsOverlapped.htm)


		 Свойство IsOverlapped
		 определяет, будет ли скрыта легенда при недостатке места на диаграмме.


		 ![](../../Property_Image.gif)
		 [ItemPlacement](IChartLegend.ItemPlacement.htm)


		 Свойство ItemPlacement
		 определяет способ размещения элементов легенды.


		 ![](../../Property_Image.gif)
		 [RowCount](IChartLegend.RowCount.htm)


		 Свойство RowCount определяет
		 количество строк в легенде.


		 ![](../../Property_Image.gif)
		 [SymbolSize](IChartLegend.SymbolSize.htm)


		 Свойство SymbolSize
		 определяет размер значка элементов легенды диаграмм.


		 ![](../../Property_Image.gif)
		 [UseSerieDesign](IChartLegend.UseSerieDesign.htm)


		 Свойство UseSerieDesign
		 определяет признак использования оформления рядов внутренней легенды
		 в соответствии с оформлением рядов на диаграмме.
