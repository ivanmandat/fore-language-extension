# IChartLegend.ItemPlacement


## Синтаксис


IChartLegend.ItemPlacement: [ChartLegPlacement](../../Enums/ChartLegPlacement.htm);


## Описание


Свойство ItemPlacement определяет
 способ размещения элементов легенды.


## Комментарии


Для размещения элементов легенды в указанное количество строк/столбцов
 укажите значение [CustomRows](../../Enums/ChartLegPlacement.htm)/[CustomColumns](../../Enums/ChartLegPlacement.htm)
 свойства.


## Пример


Пример использования свойства можно найти в описании свойств [CustomRows](../../Enums/ChartLegPlacement.htm) и [CustomColumns](../../Enums/ChartLegPlacement.htm).
