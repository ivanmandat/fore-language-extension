# IUiChartGetDataValueEventArgs


## Описание


Интерфейс IUiChartGatDataValueEventArgs
 предназначен для работы с параметрами события [UiChart.OnGetDataValue](../../Class/UiChart/UiChart.OnGetDataValue.htm)
 (построение диаграммы).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Editable](IUiChartGetDataValueEventArgs.Editable.htm)


		 Свойство Editable определяет,
		 возможно ли редактирование значения ряда в данной точке.


		 ![](../../Property_Image.gif)
		 [Value](IUiChartGetDataValueEventArgs.Value.htm)


		 Свойство Value используется
		 для задания исходных данных по оси OY.


		 ![](../../Property_Image.gif)
		 [Value2](IUiChartGetDataValueEventArgs.Value2.htm)


		 Свойство Value2 используется
		 для задания исходных данных по оси OX, если тип диаграммы точечный.


		 ![](../../Property_Image.gif)
		 [Value3](IUiChartGetDataValueEventArgs.Value3.htm)


		 Свойство Value3 определяет
		 размер пузырька пузырьковой диаграммы.
