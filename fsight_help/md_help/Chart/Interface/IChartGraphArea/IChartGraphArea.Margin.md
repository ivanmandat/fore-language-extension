# IChartGraphArea.Margin


## Синтаксис


Margin: [IGxRectF](ModDrawing.chm::/Interface/IGxRectF/IGxRectF.htm);


## Описание


Свойство Margin определяет пользовательские
 величины внешних отступов для области построения диаграммы.


## Комментарии


Использование свойства возможно при установке свойству [AutoMargin](IChartGraphArea.AutoMargin.htm)
 значения False. Данное свойство
 сохраняется с диаграммой. Если загружается объект предыдущей версии, то
 для него устанавливаются свойства AutoMargin
 и Margin по умолчанию.
