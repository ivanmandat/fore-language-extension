# IScene3DHullDataSource


## Описание


Интерфейс IScene3DHullDataSource
 содержит свойства и методы для определения источника данных и его параметров
 для замкнутой поверхности.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IScene3DHullDataSource.Count.htm)
		 Свойство Count возвращает
		 количество точек, заданных для построения.


		 ![](../../Property_Image.gif)
		 [PointColor](IScene3DHullDataSource.PointColor.htm)
		 Свойство PointColor
		 определяет цвет точки.


		 ![](../../Property_Image.gif)
		 [PointCoordinate](IScene3DHullDataSource.PointCoordinate.htm)
		 Свойство PointCoordinate
		 определяет координаты точки.


		 ![](../../Property_Image.gif)
		 [PointName](IScene3DHullDataSource.PointName.htm)
		 Свойство PointName
		 определяет имя точки.


		 ![](../../Property_Image.gif)
		 [PointRadius](IScene3DHullDataSource.PointRadius.htm)
		 Свойство PointRadius
		 определяет радиус точки (сферы).
