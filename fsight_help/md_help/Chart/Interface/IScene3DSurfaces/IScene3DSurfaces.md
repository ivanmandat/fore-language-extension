# IScene3DSurfaces


## Описание


Интерфейс IScene3DSurfaces содержит
 свойства для определения параметров коллекции поверхностей представления
 данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IScene3DSurfaces.Count.htm)
		 Свойство Count определяет
		 количество поверхностей данных в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IScene3DSurfaces.Item.htm)
		 Свойство Item возвращает
		 поверхность представления данных по заданному индексу.
