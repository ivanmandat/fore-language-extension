# IChartBubbleInfo


Сборка: Chart;


## Описание


Интерфейс IChartBubbleInfo предназначен
 для настройки пузырьковой диаграммы.


## Иерархия наследования


           IChartBubbleInfo


## Комментарии


Все свойства интерфейса IChartBubbleInfo
 сохраняются вместе с диаграммой.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BubbleSizeScale](IChartBubbleInfo.BubbleSizeScale.htm)


		 Свойство BubbleSizeScale
		 определяет масштаб отображения пузырьков.


		 ![](../../Property_Image.gif)
		 [RepresentedBubbleSizeMeasure](IChartBubbleInfo.RepresentedBubbleSizeMeasure.htm)


		 Свойство RepresentedBubbleSizeMeasure
		 определяет величину пузырька.


		 ![](../../Property_Image.gif)
		 [ShowNegativeBubbles](IChartBubbleInfo.ShowNegativeBubbles.htm)


		 Свойство ShowNegativeBubbles определяет,
		 отображаются ли пузырьки с отрицательными значениями.
