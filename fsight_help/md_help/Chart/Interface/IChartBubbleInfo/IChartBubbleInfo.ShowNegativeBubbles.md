# IChartBubbleInfo.ShowNegativeBubbles


## Синтаксис


ShowNegativeBubbles: Boolean;


## Описание


Свойство ShowNegativeBubbles
 определяет, отображаются ли пузырьки с отрицательными значениями.


## Комментарии


Допустимые значения:


	- True. Пузырьки с отрицательными
	 значениями рисуются. Для определения размера пузырька берется модуль
	 значения. При сплошном, глянцевом или автоматическом типах заливки
	 для заливки пузырька используется серый цвет. При градиентном и неоднородном
	 типах заливки для заливки пузырька используется обратный градиент.
	 При типе заливки «Со штриховкой» меняется фон пузырька и цвет штриховки;


	- False. Значение по умолчанию.
	 Пузырьки с отрицательными значениями не рисуются.
