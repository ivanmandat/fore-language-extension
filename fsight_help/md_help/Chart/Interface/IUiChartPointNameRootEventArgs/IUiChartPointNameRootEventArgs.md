# IUiChartPointNameRootEventArgs


## Описание


Интерфейс IUiChartPointNameRootEventArgs
 предназначен для работы с параметрами события [UiChart.OnGetPointNameRoot](../../Class/UiChart/UiChart.OnGetPointNameRoot.htm)
 (определение корневого элемента для иерархии наименований оси категорий).
