# IChartDateAxisSettings.MajorTickUnit


## Синтаксис


MajorTickUnit: [ChartDateAxisUnit](../../Enums/ChartDateAxisUnit.htm);


## Описание


Свойство MajorTickUnit определяет
 единицы измерения основных делений на оси дат.


## Комментарии


В качестве единиц измерения рассматриваются только дни, месяцы и годы.
