# IChartDateAxisSettings.MajorTiсkStartAt


## Синтаксис


MajorTickStartAt: DateTime;


## Описание


Свойство MajorTickStartAt
 определяет точку оси дат, с которой начинаются основные деления.


## Комментарии


Для настройки единиц измерения и интервала основных делений используйте
 свойства [IChartDateAxisSettings.MajorTickUnit](IChartDateAxisSettings.MajorTickUnit.htm),
 [IChartDateAxisSettings.MajorTickUnitInterval](IChartDateAxisSettings.MajorTickUnitInterval.htm).
