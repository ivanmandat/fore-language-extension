# IChartDateAxisSettings.CrossAxisPositionValue


## Синтаксис


CrossAxisPositionValue: DateTime;


## Описание


Свойство CrossAxisPositionValue
 определяет точку пересечения с осью Y.


## Комментарии


Точка пересечения с осью Y должна находиться в диапазоне минимального
 и максимального значений оси дат. Для задания максимального и минимального
 значений используйте свойства [IChartDateAxisSettings.Max](IChartDateAxisSettings.Max.htm)
 и [IChartDateAxisSettings.Min](IChartDateAxisSettings.Min.htm).
