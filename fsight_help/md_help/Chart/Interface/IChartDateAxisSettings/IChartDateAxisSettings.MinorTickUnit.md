# IChartDateAxisSettings.MinorTickUnit


## Синтаксис


MajorTickUnit: [ChartDateAxisUnit](../../Enums/ChartDateAxisUnit.htm);


## Описание


Свойство MinorTickUnit определяет
 единицы измерения промежуточных делений на оси дат.


## Комментарии


В качестве единиц измерения рассматриваются только дни, месяцы и годы.
