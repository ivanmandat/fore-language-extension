# IChartDateAxisSettings.CategoryPointCollapse


## Синтаксис


CategoryPointCollapse: [DateCategoryPointCollapseType](../../Enums/DateCategoryPointCollapseType.htm);


## Описание


Свойство CategoryPointCollapse
 определяет способ агрегации в точке на оси дат.


## Комментарии


Свойство используется, если требуется агрегация данных с нижнего уровня
 на верхний. Например, на диаграмме представлены данные в дневной динамике,
 а в качестве единиц измерения оси используются месяцы. Таким образом,
 если выбран способ агрегации «Сумма»,
 то дневные данные будут просуммированы для месячной динамики.
