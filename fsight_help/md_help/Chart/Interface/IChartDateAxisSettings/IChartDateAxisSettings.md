# IChartDateAxisSettings


Сборка: Chart;


## Описание


Интерфейс IChartDateAxisSettings
 используется для настройки оси дат.


## Иерархия наследования


           IChartDateAxisSettings


## Комментарии


Используйте интерфейс IChartDateAxisSettings,
 если ось X является осью дат, т.е. если свойство [IChartAxis.AxisType](../IChartAxis/IChartAxis.AxisType.htm),
 принимает значение [ChartAxisType.DateAxis](../../Enums/ChartDateAxisUnit.htm).
