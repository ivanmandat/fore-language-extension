# IChartDateAxisSettings.BaseUnit


## Синтаксис


BaseUnit: [ChartDateAxisUnit](../../Enums/ChartDateAxisUnit.htm);


## Описание


Свойство BaseUnit определяет
 единицы измерения, используемые для оси дат.


## Комментарии


В качестве единиц измерения рассматриваются только дни, месяцы и годы.
