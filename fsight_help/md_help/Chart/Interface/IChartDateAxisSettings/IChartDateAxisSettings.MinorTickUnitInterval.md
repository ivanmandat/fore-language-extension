# IChartDateAxisSettings.MinorTickUnitInterval


## Синтаксис


MinorTickUnitInterval: Integer;


## Описание


Свойство MinorTickUnitInterval
 определяет интервал промежуточных делений на оси дат.


## Комментарии


Для настройки интервала основных делений оси дат используйте свойство
 [IChartDateAxisSettings.MajorTickUnitInterval](IChartDateAxisSettings.MajorTickUnitInterval.htm).
