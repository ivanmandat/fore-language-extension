# IChartDateAxisSettings.MajorTickUnitInterval


## Синтаксис


MajorTickUnitInterval: Integer;


## Описание


Свойство MajorTickUnitInterval
 определяет интервал основных делений на оси дат.


## Комментарии


Для настройки интервала промежуточных делений оси дат используйте свойство
 [IChartDateAxisSettings.MinorTickUnitInterval](IChartDateAxisSettings.MinorTickUnitInterval.htm).
