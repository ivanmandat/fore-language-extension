# IChartExternLegendItems


## Описание


Интерфейс IChartExternLegendItems
 содержит свойства и методы коллекции элементов внешней легенды диаграмм.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IChartExternalLegendItems.Count.htm)
		 Свойство Count возвращает
		 количество элементов внешней легенды диаграмм в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IChartExternLegendItems.Item.htm)
		 Свойство Item возвращает
		 элемент внешней легенды, индекс которого передается в качестве
		 входного параметра Index.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ResetOrder](IChartExternLegendItems.ResetOrder.htm)
		 Метод ResetOrder позволяет
		 установить порядок расположения элементов легенды, установленный по умолчанию.


		 ![](../../Sub_Image.gif)
		 [Split](IChartExternLegendItems.Split.htm)
		 Метод Split позволяет
		 разделить элемент внешней легенды диаграмм, индекс которого передается
		 в качестве входного параметра Index.


		 ![](../../Sub_Image.gif)
		 [SplitAll](IChartExternLegendItems.SplitAll.htm)
		 Метод SplitAll позволяет
		 разделить все объединенные элементы внешней легенды диаграмм.


		 ![](../../Sub_Image.gif)
		 [Swap](IChartExternLegendItems.Swap.htm)
		 Метод Swap позволяет
		 менять местами заданные элементы внешней легенды.


		 ![](../../Sub_Image.gif)
		 [Unify](IChartExternLegendItems.Unify.htm)
		 Метод Unify позволяет
		 объединить элементы внешней легенды, индексы которых передаются
		 в качестве входного параметра Indexes.
