# IScene3DDataSupplier


## Описание


Интерфейс IScene3DDataSupplier
 возвращает параметры ряда сцены.


Примечание.
 Данный интерфейс использовать не рекомендуется, так как в текущей реализации
 предусмотрен только для внутреннего использования.
