# IScene3DDataSupplier.DataValue


## Синтаксис


DataValue(Var ValueX: Array; Var ValueY: Array;
 Var ValueZ: Array; Var ValueC: Array);


## Параметры


ValueX - координаты по оси
 OX;


ValueY - координаты
 по оси OY;


ValueZ - координаты
 по оси OZ;


ValueC - параметр
 зарезервирован на будущее.


## Описание


Метод DataValue позволяет получить
 данные, которые задаются интерфейсами сборки Report.
