# IScene3DTextStyle.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет
 признак видимости надписи. Если значение свойства True,
 то надпись отображается, при значении False
 - нет.


По умолчанию свойство имеет значение True.
