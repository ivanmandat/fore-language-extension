# IScene3DTextStyle


## Описание


Интерфейс IScene3DTextStyle
 содержит свойства для определения стиля надписи.


## Свойства


		 Имя свойства
		 Краткое описание

		 Свойство Background
		 определяет параметры фонового объекта надписи.


		 ![](../../Property_Image.gif)
		 [Color](IScene3DTextStyle.Color.htm)
		 Свойство Color определяет
		 цвет надписи.


		 ![](../../Property_Image.gif)
		 [Font](IScene3DTextStyle.Font.htm)
		 Свойство Font определяет
		 шрифт надписи.


		 ![](../../Property_Image.gif)
		 [Visible](IScene3DTextStyle.Visible.htm)
		 Свойство Visible определяет
		 признак видимости надписи.
