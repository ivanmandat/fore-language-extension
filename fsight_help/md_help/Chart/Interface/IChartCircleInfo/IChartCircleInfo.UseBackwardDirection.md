# IChartCircleInfo.UseBackwardDirection


## Синтаксис


UseBackwardDirection: Boolean;


## Описание


Свойство UseBackwardDirection определяет признак расположения рядов против часовой стрелки от центральной оси круговой диаграммы.


## Комментарии


Допустимые значения:


-
True. Ряды диаграммы располагаются против часовой стрелки относительно центральной оси;


-
False. Значение по умолчанию. Ряды диаграммы располагаются по часовой стрелке относительно центральной оси.


Свойство не актуально для кольцевой диаграммы ([IChart.Type](../IChart/IChart.Type.htm) = ChartType.HoleSize).
