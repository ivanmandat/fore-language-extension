# IChartCircleInfo


Сборка: Chart;


## Описание


Интерфейс IChartCircleInfo содержит
 свойства для определения параметров круговых диаграмм.


## Иерархия наследования


           IChartCircleInfo


## Комментарии


Интерфейс предназначен для работы не только с круговыми, но и с кольцевыми
 диаграммами.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ConcentricDistance](IChartCircleInfo.ConcentricDistance.htm)
		 Свойство ConcentricDistance
		 определяет расстояние между концентрическими окружностями.


		 ![](../../Property_Image.gif)
		 [FixedRadius](IChartCircleInfo.FixedRadius.htm)
		 Свойство FixedRadius
		 определяет радиус круговой диаграммы.


		 ![](../../Property_Image.gif)
		 [HoleSize](IChartCircleInfo.HoleSize.htm)
		 Свойство HoleSize определяет
		 радиус центрального отверстия (в долях) от максимального радиуса
		 диаграммы.


		 ![](../../Property_Image.gif)
		 [SeriesAsRings](IChartCircleInfo.SeriesAsRings.htm)
		 Свойство SeriesAsRings
		 определяет признак отображения рядов в кольцах диаграммы.


		 ![](../../Property_Image.gif)
		 [StartAngle](IChartCircleInfo.StartAngle.htm)
		 Свойство StartAngle
		 определяет угол поворота круговой диаграммы.


		 ![](../../Property_Image.gif)
		 [UseBackwardDirection](IChartCircleInfo.UseBackwardDirection.htm)
		 Свойство UseBackwardDirection
		 определяет признак расположения рядов против часовой стрелки от
		 центральной оси круговой диаграммы.
