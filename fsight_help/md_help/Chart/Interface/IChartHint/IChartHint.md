# IChartHint


Сборка: Chart;


## Описание


Интерфейс IChartHint используется
 для определения формата всплывающих подсказок на диаграмме.


## Иерархия наследования


           IChartHint


## Комментарии


Для отображения всплывающих подсказок используйте свойство [IChart.UseSignatureHint](../IChart/IChart.UseSignatureHint.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BackgroundInfo](IChartHint.BackgroundInfo.htm)


		 Свойство BackgroundInfo
		 определяет параметры заливки всплывающих подсказок на диаграмме.


		 ![](../../Property_Image.gif)
		 [BorderPen](IChartHint.BorderPen.htm)


		 Свойство BorderPen
		 определяет стиль пера для контура всплывающих подсказок на диаграмме.


		 ![](../../Property_Image.gif)
		 [CustomFormat](IChartHint.CustomFormat.htm)


		 Свойство CustomFormat
		 определяет [пользовательский
		 формат](tabsheet.chm::/Interface/ITabCellStyle/ITabCellStyle.CustomFormat.htm) числовых значений всплывающих подсказок
		 на диаграмме.


		 ![](../../Property_Image.gif)
		 [DisplayBorder](IChartHint.DisplayBorder.htm)


		 Свойство DisplayBorder
		 определяет, отображается ли контур всплывающих подсказок на диаграмме.


		 ![](../../Property_Image.gif)
		 [Font](IChartHint.Font.htm)


		 Свойство Font
		 определяет шрифт всплывающих подсказок на диаграмме.


		 ![](../../Property_Image.gif)
		 [FontColor](IChartHint.FontColor.htm)


		 Свойство FontColor
		 определяет цвет шрифта всплывающих подсказок на диаграмме.


		 ![](../../Property_Image.gif)
		 [HorizontalAlignment](IChartHint.HorizontalAlignment.htm)


		 Свойство HorizontalAlignment
		 определяет положение подписи данных всплывающих подсказок по горизонтали
		 на диаграмме.


		 ![](../../Property_Image.gif)
		 [StringFormat](IChartHint.StringFormat.htm)


		 Свойство StringFormat
		 определяет формат отображения подписей данных всплывающих подсказок
		 на диаграмме.


		 ![](../../Property_Image.gif)
		 [UseIndicatorColorForBorder](IChartHint.UseIndicatorColorForBorder.htm)


		 Свойство UseIndicatorColorForBorder
		 определяет, следует ли брать цвет для контура всплывающих подсказок
		 из метрики на диаграмме.
