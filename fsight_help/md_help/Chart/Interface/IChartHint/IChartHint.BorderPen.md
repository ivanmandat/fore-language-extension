# IChartHint.BorderPen


## Синтаксис


BorderPen: [IGxPen](ModDrawing.chm::/Interface/IGxPen/IGxPen.htm);


## Описание


Свойство BorderPen определяет
 стиль пера для контура всплывающих подсказок на диаграмме.


## Комментарии


Для задания цвета контура всплывающих подсказок необходимо, чтобы свойство
 [IChartHint.UseIndicatorColorForBorder](IChartHint.UseIndicatorColorForBorder.htm)
 принимало значение False.
