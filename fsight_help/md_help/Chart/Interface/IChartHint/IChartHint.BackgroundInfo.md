# IChartHint.BackgroundInfo


## Синтаксис


BackgroundInfo: [IChartBackgroundInfo](../IChartBackgroundInfo/IChartBackgroundInfo.htm);


## Описание


Свойство BackgroundInfo
 определяет параметры заливки всплывающих подсказок на диаграмме.


## Комментарии


Для настройки шрифта всплывающих подсказок используйте свойства [IChartHint.Font](IChartHint.Font.htm) и [IChartHint.FontColor](IChartHint.FontColor.htm).
