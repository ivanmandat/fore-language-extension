# IScene3DRadiusDataSupplier.RadiusDataValue


## Синтаксис


RadiusDataValue(Var ValueX: Array; Var ValueY: Array;
 Var ValueZ: Array; Var ValueR: Array; Var ValueC: Array);


## Параметры


ValueX - координаты по оси
 OX;


ValueY - координаты
 по оси OY;


ValueZ - координаты
 по оси OZ;


ValueR
 - радиусы точек;


ValueC - параметр
 зарезервирован на будущее.


## Описание


Метод RadiusDataValue позволяет
 получить координаты точек ряда и их радиусы, которые были заданы интерфейсами
 сборки Report.
