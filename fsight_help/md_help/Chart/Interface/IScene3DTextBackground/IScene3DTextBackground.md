# IScene3DTextBackground


## Описание


Интерфейс IScene3DTextBackground
 содержит свойства для определения фона надписи.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Color](IScene3DTextBackground.Color.htm)
		 Свойство Color определяет
		 цвет фонового объекта надписи.


		 ![](../../Property_Image.gif)
		 [Type](IScene3DTextBackground.Type.htm)
		 Свойство Type определяет
		 тип фона надписи.


		 ![](../../Property_Image.gif)
		 [Visible](IScene3DTextBackground.Visible.htm)
		 Свойство Visible определяет
		 признак видимости фонового объекта надписи.
