# UiChartGetTextEventArgs.Create


## Синтаксис


Create(SerieIndex: Integer; PointIndex: Integer; Text: String; Result: Вoolean);


## Параметры


SerieIndex - номер ряда.


PointIndex - номер точки.


Text - текст подписи данных.


Result - определяет признак обработки события.


## Описание


Метод Create создает аргументы события [UiChart.OnGetLabelText](../UiChart/UiChart.OnGetLabelText.htm). Метод автоматически вызывается при наступлении данного события для создания его аргументов.
