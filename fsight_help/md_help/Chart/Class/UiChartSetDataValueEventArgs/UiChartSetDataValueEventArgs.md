# UiChartSetDataValueEventArgs


## Описание


Класс UiChartSetDataValueEventArgs
 предназначен для работы с параметрами события [UiChart.OnSetDataValue](../UiChart/UiChart.OnSetDataValue.htm)
 (изменение значения точки на графике).
