 Object; Args: [IUiChartGroupCountEventArgs](../../Interface/UiChartGroupCountEventArgs/IUiChartGroupCountEventArgs.htm));

Begin

    //набор операторов

## Параметры


Sender - параметр, возвращающий
 компонент, сгенерировавший событие;


Args - параметр, позволяющий
 определить параметры события.


## Описание


Событие OnGetGroupCount позволяет
 определить количество групп диаграммы.


## Комментарии


Для определения наименования группы диаграммы используйте [UiChart.OnGetGroupName](UiChart.OnGetGroupName.htm).
