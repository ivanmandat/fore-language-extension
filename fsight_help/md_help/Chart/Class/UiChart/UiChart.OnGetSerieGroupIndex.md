## Описание


Событие OnGetSerieGroupIndex
 позволяет определить индекс группы рядов диаграммы.


## Комментарии


Для определения наименования группы рядов диаграммы используйте [UiChart.OnGetSerieGroupName](UiChart.OnGetSerieGroupName.htm).
