# UiChart.OnGetGroupName


## Синтаксис


Sub OnGetGroupName(Sender:
 Object; Args: [IUiChartGroupNameEventArgs](../../Interface/IUiChartGroupNameEventArgs/IUiChartGroupNameEventArgs.htm));

Begin

    //набор операторов

End Sub OnGetGroupName;


## Параметры


Sender - параметр, возвращающий
 компонент, сгенерировавший событие;


Args - параметр, позволяющий
 определить параметры события.


## Описание


Событие OnGetGroupName позволяет
 определить наименование группы диаграммы.


## Комментарии


Для определения количества групп диаграммы используйте [UiChart.OnGetGroupCount](UiChart.OnGetGroupCount.htm).
