# UiChart.OnGetLabelText


## Синтаксис


		UiChart1OnGetLabelText(Sender: Object; Args: [IUiChartGetTextEventArgs](../../Interface/IUiChartGetTextEventArgs/IUiChartGetTextEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnGetLabelText позволяет
 определить пользовательский формат данных, отображенных в подписях данных.


## Комментарии


Событие наступает при отрисовке графика.
