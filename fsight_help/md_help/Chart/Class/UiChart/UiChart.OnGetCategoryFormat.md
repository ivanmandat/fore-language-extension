## Параметры


Sender - параметр, возвращающий компонент, сгенерировавший событие.


Args - параметр, позволяющий определить параметры события.


## Описание


Событие OnGetCategoryFormat позволяет определить формат наименования точек для оси категорий.


Примечание. Наименования точек для оси категорий можно задать, используя событие [UiChart.OnGetCategoryName](UiChart.OnGetCategoryName.htm).
