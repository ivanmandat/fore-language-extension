## Параметры

## Описание


Событие OnGetSerieGroupName
 позволяет определить наименование группы рядов диаграммы.


## Комментарии


Для определения индекса группы рядов диаграммы используйте [UiChart.OnGetSerieGroupIndex](UiChart.OnGetSerieGroupIndex.htm).
