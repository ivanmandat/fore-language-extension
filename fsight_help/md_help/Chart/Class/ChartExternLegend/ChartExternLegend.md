# ChartExternLegend


## Описание


Класс ChartExternLegend реализует свойства и методы, предназначенные для определения параметров внешней легенды диаграмм.


## Свойства объекта класса, унаследованные от [IChartExternLegend](../../Interface/IChartExternLegend/IChartExternLegend.htm)


 Имя свойства
 Краткое описание


 ![](../../Property_Image.gif)
 [AxisIcon](../../Interface/IChartExternLegend/IChartExternLegend.AxisIcon.htm)
 Свойство AxisIcon определяет признак отображения в легенде значка, который указывает по какой оси расположен ряд (основной или дополнительной).


 ![](../../Property_Image.gif)
 [BackgroundInfo](../../Interface/IChartExternLegend/IChartExternLegend.BackgroundInfo.htm)
 Свойство BackgroundInfo определяет свойства заливки внешней легенды.


 ![](../../Property_Image.gif)
 [BorderPen](../../Interface/IChartExternLegend/IChartExternLegend.BorderPen.htm)
 Свойство BorderPen определяет параметры границ внешней легенды.


 ![](../../Property_Image.gif)
 [Chart](../../Interface/IChartExternLegend/IChartExternLegend.Chart.htm)
 Свойство Chart возвращает объект типа IChart, которому принадлежит элемент легенды.


 ![](../../Property_Image.gif)
 [ChartCount](../../Interface/IChartExternLegend/IChartExternLegend.ChartCount.htm)
 Свойство ChartCount возвращает количество диаграмм, участвующих в построении внешней легенды диаграмм.


 ![](../../Property_Image.gif)
 [Font](../../Interface/IChartExternLegend/IChartExternLegend.Font.htm)
 Свойство Font определяет шрифт текста внешней легенды диаграмм.


 ![](../../Property_Image.gif)
 [FontColor](../../Interface/IChartExternLegend/IChartExternLegend.FontColor.htm)
 Свойство FontColor определяет цвет шрифта внешней легенды диаграмм.


 ![](../../Property_Image.gif)
 [InternalItems](../../Interface/IChartExternLegend/IChartExternLegend.InternalItems.htm)
 Свойство InternalItems возвращает коллекцию отображаемых элементов внешней легенды.


 ![](../../Property_Image.gif)
 [IsDirty](../../Interface/IChartExternLegend/IChartExternLegend.IsDirty.htm)
 Свойство IsDirty определяет признак наличия несохраненных изменений во внешней легенде диаграмм.


 ![](../../Property_Image.gif)
 [ItemPlacement](../../Interface/IChartExternLegend/IChartExternLegend.ItemPlacement.htm)
 Свойство ItemPlacement определяет расположение элементов внешней легенды.


 ![](../../Property_Image.gif)
 [ItemSeparator](../../Interface/IChartExternLegend/IChartExternLegend.ItemSeparator.htm)
 Свойство ItemSeparator позволяет задать разделитель текста для объединенных элементов внешней диаграмм.


 ![](../../Property_Image.gif)
 [SymbolSize](../../Interface/IChartExternLegend/IChartExternLegend.SymbolSize.htm)
 Свойство SymbolSize определяет размер пиктограммы для отображения цвета ряда во внешней легенде диаграмм.


 ![](../../Property_Image.gif)
 [UndoRedo](../../Interface/IChartExternLegend/IChartExternLegend.UndoRedo.htm)
 Свойство UndoRedo определяет параметры стека изменений (отмены/повтора) внешней легенды диаграммы.


 ![](../../Property_Image.gif)
 [UseSerieIcon](../../Interface/IChartExternLegend/IChartExternLegend.UseSerieIcon.htm)
 Свойство UseSerieIcon позволяет отображать вместо значков ряда цветовой квадрат.


## Методы объекта класса, унаследованные от [IChartExternLegend](../../Interface/IChartExternLegend/IChartExternLegend.htm)


 Имя метода
 Краткое описание


 ![](../../Sub_Image.gif)
 [AddChart](../../Interface/IChartExternLegend/IChartExternLegend.AddChart.htm)
 Метод AddChart осуществляет добавление диаграммы, передаваемой посредством параметра Value, в список диаграмм, подключенных к внешней легенде.


 ![](../../Sub_Image.gif)
 [Clear](../../Interface/IChartExternLegend/IChartExternLegend.Clear.htm)
 Метод Clear осуществляет удаление всех диаграмм из списка подключенных к внешней легенде.


 ![](../../Sub_Image.gif)
 [RemoveChart](../../Interface/IChartExternLegend/IChartExternLegend.RemoveChart.htm)
 Метод RemoveChart осуществляет удаление диаграммы, передаваемой посредством параметра Value, из списка диаграмм, подключенных к внешней легенде.
