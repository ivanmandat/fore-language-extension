# UiChartGroupNameEventArgs


Сборка: Chart;


## Описание


Класс UiChartGroupNameEventArgs
 реализует параметры события [UiChart.OnGetGroupName](../UiChart/UiChart.OnGetGroupName.htm).


## Комментарии


Событие [UiChart.OnGetGroupName](../UiChart/UiChart.OnGetGroupName.htm)
 позволяет определить наименование группы диаграммы.
