# UiChartGroupNameEventArgs.Create


## Синтаксис


Create(SerieIndex: Integer; Result: String);


## Параметры


SerieIndex.
 Номер ряда;


Result. Наименование группы.


## Описание


Конструктор Create создает параметры
 аргумента события [UiChart.OnGetGroupName](../UiChart/UiChart.OnGetGroupName.htm).


## Комментарии


Метод автоматически вызывается при наступлении данного события для создания
 его аргументов.
