# UiChartPointNameRootEventArgs.Create


## Синтаксис


Create(Result: [IChartPointNameNode](../../Interface/IChartPointNameNode/IChartPointNameNode.htm));


## Параметры


Result - корневой элемент.


## Описание


Конструктор Create создает аргументы события [UiChart.OnGetPointNameRoot](../UiChart/UiChart.OnGetPointNameRoot.htm) (определение корневого элемента для иерархии наименований оси категорий.).


## Комментарии


Метод автоматически вызывается при наступлении данного события для создания его аргументов.
