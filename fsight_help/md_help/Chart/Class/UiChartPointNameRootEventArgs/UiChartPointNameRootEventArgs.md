# UiChartPointNameRootEventArgs


## Описание


Класс UiChartPointNameRootEventArgs
 предназначен для работы с параметрами события [UiChart.OnGetPointNameRoot](../UiChart/UiChart.OnGetPointNameRoot.htm)
 (определение корневого элемента для иерархии наименований оси категорий).
