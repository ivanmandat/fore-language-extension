# Chart3D


## Описание


Класс Chart3D представляет собой
 трехмерную сцену, которая отображается в компоненте [Chart3DBox](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/Chart3DBox.htm).


## Свойства объекта класса, унаследованные от [IScene3D](../../Interface/IScene3D/IScene3D.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Caption](../../Interface/IScene3D/IScene3D.Caption.htm)
		 Свойство Caption возвращает
		 параметры заголовка трехмерной сцены.


		 ![](../../Property_Image.gif)
		 [DisplayedObjects](../../Interface/IScene3D/IScene3D.DisplayedObjects.htm)
		 Свойство DisplayedObjects
		 определяет параметры объектов трехмерной сцены.


		 ![](../../Property_Image.gif)
		 [IsDirty](../../Interface/IScene3D/IScene3D.IsDirty.htm)
		 Свойство IsDirty возвращает
		 признак существующих изменений на сцене.


		 ![](../../Property_Image.gif)
		 [SceneView](../../Interface/IScene3D/IScene3D.SceneView.htm)
		 Свойство SceneView
		 определяет настройки отображения сцены.


		 ![](../../Property_Image.gif)
		 [Series](../../Interface/IScene3D/IScene3D.Series.htm)
		 Свойство Series определяет
		 параметры универсальной коллекции рядов сцены.
