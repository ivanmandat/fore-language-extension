# Chart3DSeriesClickEventArgs


## Описание


Класс Chart3DSeriesClickEventArgs
 возвращает описание графических объектов, для которых наступило событие
 [OnSeriesDoubleClick](../Chart3DBox/IChart3DBoxEvents.OnSeriesDoubleClick.htm)
 (двойной щелчок мыши по ряду сцены на форме).
