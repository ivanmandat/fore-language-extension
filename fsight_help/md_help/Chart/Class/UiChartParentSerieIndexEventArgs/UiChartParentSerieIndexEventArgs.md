# UiChartParentSerieIndexEventArgs


Сборка: Chart;


## Описание


Класс UiChartParentSerieIndexEventArgs
 реализует параметры события [UiChart.OnGetParentSerieIndex](../UiChart/UiChart.OnGetParentSerieIndex.htm).


## Комментарии


Событие [UiChart.OnGetParentSerieIndex](../UiChart/UiChart.OnGetParentSerieIndex.htm)
 позволяет определить индекс родительского ряда.
