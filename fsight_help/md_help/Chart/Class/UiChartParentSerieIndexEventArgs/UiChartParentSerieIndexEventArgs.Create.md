# UiChartParentSerieIndexEventArgs.Create


## Синтаксис


Create(SerieIndex: Integer; Result: Integer);


## Параметры


SerieIndex.
 Индекс родительского ряда диаграммы;


Result. Индекс родительского
 ряда диаграммы.


## Описание


Конструктор Create создает параметры
 аргумента события [UiChart.OnGetParentSerieIndex](../UiChart/UiChart.OnGetParentSerieIndex.htm).


## Комментарии


Метод автоматически вызывается при наступлении данного события для создания
 его аргументов.
