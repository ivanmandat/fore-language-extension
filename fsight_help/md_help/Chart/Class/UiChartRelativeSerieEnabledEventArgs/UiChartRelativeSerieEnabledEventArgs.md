# UiChartRelativeSerieEnabledEventArgs


Сборка: Chart;


## Описание


Класс UiChartRelativeSerieEnabledEventArgs
 реализует параметры события [UiChart.OnGetRelativeSerieEnabled](../UiChart/UiChart.OnGetRelativeSerieEnabled.htm).


## Комментарии


Событие [UiChart.OnGetRelativeSerieEnabled](../UiChart/UiChart.OnGetRelativeSerieEnabled.htm)
 позволяет определить, будут ли использоваться относительные ряды.
