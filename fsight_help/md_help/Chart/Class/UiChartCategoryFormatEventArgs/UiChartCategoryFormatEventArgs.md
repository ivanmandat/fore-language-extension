# UiChartCategoryFormatEventArgs


## Описание


Класс UiChartCategoryFormatEventArgs
 предназначен для работы с параметрами события [UiChart.OnGetCategoryFormat](../UiChart/UiChart.OnGetCategoryFormat.htm)
 (формирование формата наименования точек для оси категорий).
