# TrellisChart


Сборка: Chart;


## Описание


Класс TrellisChart используется
 для настройки диаграммы треллис.


## Комментарии


Работа с данным классом в Fore недоступна.


Использование ITrellisChart
 возможно в различных средах разработки приложений с помощью библиотек
 Prognoz.Platform.Interop.Chart.dll и Prognoz.Platform.Interop.Drawing.dll.
