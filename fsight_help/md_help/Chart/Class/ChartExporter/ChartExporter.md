# ChartExporter


Класс ChartExporter реализует объект, используемый для экспорта диаграммы.

 Имя свойства
 Краткое описание


 ![](../../Property_Image.gif)
 [Chart](../../Interface/IChartExporter/IChartExporter.Chart.htm)
 Свойство Chart определяет диаграмму для экспорта.


 ![](../../Property_Image.gif)
 [PixelHeight](../../Interface/IChartExporter/IChartExporter.PixelHeight.htm)
 Свойство PixelHeight определяет высоту диаграммы в пикселях.


 ![](../../Property_Image.gif)
 [PixelWidth](../../Interface/IChartExporter/IChartExporter.PixelWidth.htm)
 Свойство PixelWidth определяет ширину диаграммы в пикселях.


 ![](../../Property_Image.gif)
 [Scale](../../Interface/IChartExporter/IChartExporter.Scale.htm)
 Свойство Scale определяет масштаб диаграммы для экспорта.
