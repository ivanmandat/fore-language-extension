# UiChartClass


## Описание


Класс UiChartClass реализует
 статические методы для вызова стандартных диалогов настройки диаграммы:


	- параметры диаграммы;


	- формат области диаграммы;


	- тип диаграммы;


	- формат области построения;


	- формат легенды, формат осей;


	- формат ряда данных;


	- коридоры диаграммы.
