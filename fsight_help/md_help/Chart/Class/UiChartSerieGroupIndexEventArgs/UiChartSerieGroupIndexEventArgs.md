# UiChartSerieGroupIndexEventArgs


Сборка: Chart;


## Описание


Класс UiChartSerieGroupIndexEventArgs реализует
 параметры события [UiChart.OnGetSerieGroupIndex](../UiChart/UiChart.OnGetSerieGroupIndex.htm).


## Комментарии


Событие [UiChart.OnGetSerieGroupIndex](../UiChart/UiChart.OnGetSerieGroupIndex.htm)
 позволяет определить индекс группы рядов диаграммы.
