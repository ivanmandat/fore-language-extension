# UiChartSerieGroupIndexEventArgs.Create


## Синтаксис


Create(SerieIndex: Integer; Result: Integer);


## Параметры


SerieIndex.
 Индекс ряда диаграммы;


Result. Индекс ряда диаграммы.


## Описание


Конструктор Create создает параметры
 аргумента события [UiChart.OnGetSerieGroupIndex](../UiChart/UiChart.OnGetSerieGroupIndex.htm).


## Комментарии


Метод автоматически вызывается при наступлении данного события для создания
 его аргументов.
