# UiChartGroupCountEventArgs.Create


## Синтаксис


Create(Result: Integer);


## Параметры


Result. Количество групп.


## Описание


Конструктор Create создает
 параметры аргумента события UiChart.OnGetGroupCount.


## Комментарии


Метод автоматически вызывается при наступлении данного события для создания
 его аргументов.
