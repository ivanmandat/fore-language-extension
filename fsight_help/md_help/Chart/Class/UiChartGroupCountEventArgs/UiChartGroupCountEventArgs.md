# UiChartGroupCountEventArgs


Сборка: Chart;


## Описание


Класс UiChartGroupCountEventArgs
 реализует параметры события [UiChart.OnGetGroupCount](../UiChart/UiChart.OnGetGroupCount.htm).


## Комментарии


Событие [UiChart.OnGetGroupCount](../UiChart/UiChart.OnGetGroupCount.htm)
 позволяет определить количество групп диаграммы.
