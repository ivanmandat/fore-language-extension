# UiChartCategoryNameEventArgs


## Описание


Класс UiChartCategoryNameEventArgs предназначен для работы с параметрами события [UiChart.OnGetCategoryName](../UiChart/UiChart.OnGetCategoryName.htm) (формирование наименования точек для оси категорий).
