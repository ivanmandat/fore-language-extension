# UiChartCategoryNameEventArgs.Create


## Синтаксис


Create(PointIndex: Integer; Result: Variant);


## Параметры


PointIndex - индекс точки, для которой формируется наименование.


Result - наименование.


## Описание


Метод Create создает аргументы события [UiChart.OnGetCategoryName](../UiChart/UiChart.OnGetCategoryName.htm) (формирование наименования точек для оси категорий). Метод автоматически вызывается при наступлении данного события для создания его аргументов.
