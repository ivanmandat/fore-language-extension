# ChartWidget


Сборка: Chart;


## Описание


Класс ChartWidget используется
 для настройки компонента ChartWidget.


## Комментарии


Для активации компонента ChartWidget
 вручную установите в инспекторе объектов среды разработки свойству Active значение True.
