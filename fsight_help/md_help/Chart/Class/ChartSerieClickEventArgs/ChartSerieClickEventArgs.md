# ChartSerieClickEventArgs


## Описание


Класс ChartSerieClickEventArgs
 возвращает индекс точки и ряда, для которых наступило событие [OnSerieClick](../ChartBox/ChartBox.OnSerieClick.htm)
 (щелчок кнопкой мыши по ряду на диаграмме).
