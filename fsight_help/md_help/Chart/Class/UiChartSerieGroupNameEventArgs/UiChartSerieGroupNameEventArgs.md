# UiChartSerieGroupNameEventArgs


Сборка: Chart;


## Описание


Класс UiChartSerieGroupNameEventArgs
 реализует параметры события [UiChart.OnGetSerieGroupName](../UiChart/UiChart.OnGetSerieGroupName.htm).


## Комментарии


Событие [UiChart.OnGetSerieGroupName](../UiChart/UiChart.OnGetSerieGroupName.htm)
 позволяет определить наименование группы рядов диаграммы.
