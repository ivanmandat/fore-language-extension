# Вопросы и ответы


Задачи контейнера, осуществляющие выполнение задач ETL, могут завершиться
 с ошибкой. В зависимости от текста ошибки могут быть различные причины
 её возникновения и пути решения:


	- Ошибка выполнения операции
	 <Наименование>. В источнике данных указан запрос файла. Снимите
	 флажок «Запрашивать файл при выполнении задачи» и повторите выполнение
	 задачи. В настройках одной из следующих задач установлен указанный
	 флажок: Импорт из Access, Импорт из Excel, Импорт из XML или Импорт
	 из текста. Проверьте настройки источников и снимите флажок «Запрашивать
	 файл при выполнении задачи»;


	- При открытии файла <Путь
	 и наименование файла> произошла ошибка (Файл не найден).
	 В задаче ETL в настройках какого-либо источника указан путь к несуществующему
	 файлу. Возможно файл был переименован. Скорректируйте пути к файлам
	 во всех источниках в задаче ETL.


Примечание.
 Актуально только для настольного приложения.
