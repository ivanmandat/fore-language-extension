# Просмотр параметров планировщика задач


Для открытия окна запущенного планировщика задач:


	- дважды щелкните по значку ![](AppSrv.gif)
	 в области уведомлений на панели задач;


	- выполните команду «Параметры»
	 контекстного меню значка ![](AppSrv.gif).


После выполнения данных
 действий будет открыт диалог:


![](AppSrv_Params.png)


В окне планировщика задач доступно:


	- Время запуска. Отображается
	 время, когда был запущен планировщик задач;


	- Параметры. Отображаются
	 [заданные параметры](../3_Work_Tasks/UiAppSrv_Run_Parametrs.htm)
	 планировщика задач;


	- Контейнеры задач. Отображается
	 сервер, репозиторий и идентификатор контейнера запланированных задач.


Для значка доступно контекстное меню, содержащее команды:


	- [Параметры](UiAppSrv_Work_AppSrv_Parameters.htm).
	 Открывает окно планировщика задач;


	- [Обновить
	 настройки](../3_Work_Tasks/Update_config_AppSrv.htm). Проверяет наличие новой версии конфигурационного
	 файла. Если найдена новая версия файла, то осуществляется перезапуск
	 планировщика задач с новыми настройками;


	- [Закрыть](../3_Work_Tasks/Stop_Runnig_Tasks.htm).
	 Останавливает все запущенные задачи и закрывает планировщик задач.


Планировщик задач через заданный интервал времени опрашивает контейнер
 задач и, если подошло время, выполняет задачи. По умолчанию планировщик
 задач проверяет наличие задач каждые 5 минут. В случае если время выполнения
 задач контейнера совпадает, их выполнение происходит параллельно друг
 другу.


При работе с большим количеством задач в планировщике задач предусмотрены
 [особенности распределения потоков
 задач](UiAppSrv_Work_AppSrvXml_Flow.htm).


В окне отображается время запуска планировщика задач, параметры, с которыми
 планировщик был запущен, и список выполняемых контейнеров задач.
