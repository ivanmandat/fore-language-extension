# Создание конфигурационного файла


Для работы планировщик задач использует конфигурационный файл Scheduler.xml,
 содержащий следующую информацию: название репозитория, параметры подключения
 к нему, идентификатор контейнера запланированных задач и настройки почтового
 сервера для отправки уведомлений.


Основные настройки конфигурационного файла [формируются
 автоматически](UiAppSrv_Work_AppSrv_createXML.htm) из контейнера запланированных задач. Сформированный
 конфигурационный файл должен быть расположен:


	- в папке /opt/foresight/fp10.x-biserver/etc
	 в ОС Linux;


	- рядом с исполняемым файлом планировщика Scheduler.exe в папке
	 C:\Program Files\Foresight\Analytics Platform 10.x в ОС Windows.


Примечание.
 При автоматическом создании конфигурационного файла не задаются настройки,
 отвечающие за [рассылку
 результата](configure_file_manual_extend.htm#host), и настройки для [ведения
 журнала операций](configure_file_manual_extend.htm#log). Разделы вносятся вручную при необходимости. Для
 расширения конфигурационного файла обратитесь к разделу «[Расширение
 конфигурационного файла вручную](configure_file_manual_extend.htm)».


При работе с большим количеством задач в планировщике задач предусмотрено
 [распределение всего количества
 потоков](UiAppSrv_Work_AppSrvXml_Flow.htm).
