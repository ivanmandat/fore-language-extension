# Обновление настроек планировщика задач


Обновление настроек планировщика задач необходимо в случае внесения
 изменений в конфигурационный файл.


Для обновления настроек планировщика задач:


	- в веб-приложении сохраните новый конфигурационный файл с именем
	 Scheduler.xml;


	- в настольном приложении:


		- Сохраните новый конфигурационный файл с именем Scheduler.xml.


		- Обновите настройки планировщика задач:


			- нажмите кнопку «Обновить»
			 в окне планировщика задач;


			- выполните команду «Обновить
			 настройки» в контекстном меню значка ![](../1_Work_AppSrv/AppSrv.gif).
