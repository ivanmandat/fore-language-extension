# Создание запланированных задач


Задача - объект контейнера запланированных
 задач. Задача предназначена для выполнения каких-либо действий над объектами
 по заданному расписанию.


Задачи создаются в [контейнере задач](../Getting_Started.htm)
 и могут быть выполнены по расписанию, по условию, по наступлению системного
 или настраиваемого события.


Для выполнения задач [запустите
 планировщик задач](../1_Work_AppSrv/UiAppSrv_Work_Run.htm) на [сервере
 приложений](Setup.chm::/UiWebSetup/AppServer.htm)
 или [BI-сервере](Setup.chm::/UiWebSetup/UiWebSetup_TitlePage.htm).


Контейнер запланированных задач поддерживает создание задач для:


	- вычисления регламентного отчета;


	- расчёта вычисляемого куба;


	- выполнения модуля;


Примечание.
 В задачах выполнения модуля запрещено создание визуальных форм и компонентов.


	- выполнения задачи ETL, задачи импорта/экспорта данных;


	- расчёта задачи моделирования;


Примечание.
 При использовании в задаче моделирования методов [подключенного
 пакета R](uinav.chm::/02_Navigator/CommonSettings/Integration.htm#r) настройте [сервер приложений](Setup.chm::/UiWebSetup/AppServer.htm).


	- выполнения многомерного расчета на сервере БД. Доступно только
	 в настольном приложении;


	- обновления кеша объекта. Доступно только в настольном приложении;


	- обновления поискового индекса. Доступно только в настольном
	 приложении.


Для создания задачи в контейнере запланированных задач выполните шаги:


	- [Создайте
	 задачу](../2_Work_interface/Operations_with_tasks.htm).


	- [Определите
	 наименование и идентификатор задачи](UiAppSrv_Work_Tasks_CreateTask_Task.htm). Заголовок окна соответствует
	 выбранному типу задачи, например: расчет куба, выполнение модуля и
	 т.д.


	- [Задайте
	 периодичность выполнения задачи.](UiAppSrv_Work_Tasks_CreateTask_TimeTable.htm) Для задачи можно настроить различные
	 периоды выполнения: однократное, ежедневное, еженедельное, ежемесячное,
	 а также по наступлению системного или пользовательского события.


Примечание.
 Настройка периодичности опциональна и может быть произведена в зависимости
 от требований задачи. Настройте [расписание](UiAppSrv_Work_Tasks_CreateTask_TimeTable.htm),
 если планируется отправка уведомлений о выполнении задачи по электронной
 почте. При выполнении задач [вручную](view_task_state.htm#manual_task)
 сообщения не отправляются.


	- [Задайте условия выполнения задачи](Conditions.htm).


Примечание.
 Настройка условия опциональна и может быть произведена в зависимости от
 требований задачи.


	- [Настройте
	 объект задачи](UiAppSrv_Work_Tasks_CreateTask_Parameters.htm) и установите значения его параметров.


	- [Настройте
	 обработку результата выполнения задачи](UiAppSrv_Work_Tasks_CreateTask_Rezult.htm).


Примечание.
 Настройка обработки результата опциональна и может быть произведена в
 зависимости от требований задачи. Отправка уведомлений по электронной
 почте доступна только для задач, выполняемых по [расписанию](UiAppSrv_Work_Tasks_CreateTask_TimeTable.htm).
 При запуске задач [вручную](view_task_state.htm#manual_task)
 сообщения не отправляются.


После создания задачи будут доступны [операции
 с задачей](../2_Work_interface/Operations_with_tasks.htm#settingstask) и действия над [объектами
 задачи](../2_Work_interface/Operations_with_tasks.htm#task_object).


Для запуска задач на выполнение
 смотрите статью «[Запуск
 задач на выполнение](../1_Work_AppSrv/UiAppSrv_Work_Run.htm)».
