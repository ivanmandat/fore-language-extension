# Остановка запущенных задач и закрытие планировщика


Для остановки запущенных задач в ОС Linux остановите службу планировщика
 [scheduler.service](../1_Work_AppSrv/UiAppSrv_Work_AppSrv_Service.htm):


sudo systemctl stop scheduler.service


Для закрытия планировщика задач в ОС Windows:


	- нажмите кнопку «Закрыть»
	 в окне планировщика задач;


	- выполните команду «Закрыть»
	 в контекстном меню значка ![](../1_Work_AppSrv/AppSrv.gif).


При нажатии кнопки «Закрыть»
 в окне планировщика задач произойдет закрытие только окна планировщика,
 сам планировщик продолжит работу и будет доступен в области уведомлений.
 Для полного закрытия планировщика задач выполните команду «Закрыть»
 в контекстном меню значка ![](../1_Work_AppSrv/AppSrv.gif).


После остановки запущенных задач при необходимости выполните [обновление
 настроек](Update_config_AppSrv.htm) планировщика задач и [запустите
 задачи на выполнение](../1_Work_AppSrv/UiAppSrv_Work_Run.htm) с необходимыми параметрами.
