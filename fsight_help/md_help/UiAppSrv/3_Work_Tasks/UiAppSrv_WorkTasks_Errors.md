# Ошибки выполнения


Если в процессе выполнения задач возникли ошибки, их можно просмотреть:


	- в журнале операций. Журнал операций ведется, если в [конфигурационном
	 файле](../1_Work_AppSrv/configure_file_manual_extend.htm#log) описан подраздел <LOG>;


	- в письме, если для задачи была настроена отправка [уведомления
	 по электронной почте](UiAppSrv_Work_Tasks_CreateTask_Rezult.htm);


	- на панели «Ошибки выполнения»:


[![](../Opened.gif)![](../Closed.gif)Для
 открытия панели](javascript:TextPopup(this))


	Установите флажок «Ошибки
	 выполнения»:


			- в контекстном меню панели инструментов;


			- в главном меню «Вид
			 > Панели инструментов > Ошибки выполнения».


	После выполнения одного из действий в
	 нижней части окна контейнера задач будет отображена панель «Ошибки выполнения».


![](ExecutionErrors.png)


На панели «Ошибки выполнения»
 отображается название задачи, при выполнении которой возникли ошибки,
 и текст ошибки. В письме и журнале операций дополнительно к этой информации
 отображается время начала и окончания выполнения задачи.
