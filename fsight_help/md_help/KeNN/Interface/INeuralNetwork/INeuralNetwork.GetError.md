# INeuralNetwork.GetError


## Синтаксис


GetError: Integer;


## Описание


Метод GetError возвращает код последней ошибки нейронной сети.


## Комментарии


Коды ошибок:


 Код
 Описание


 0
 Ошибки нет.


 1
 Ошибка распределения памяти.


 2
 Ошибка функции распространения сигнала в сети.


 3
 Ошибка во входных данных.


 4
 Ошибка переполнения.


 5
 Ошибка в параметрах.


 100
 Неопределенная ошибка.


Использование метода приведено в примере для [INeuralNetwork.LearnSOFM](INeuralNetwork.LearnSOFM.htm).
