# INeuralNetwork.GetInputValues


## Синтаксис


GetInputValues: Array;


## Описание


Метод GetInputValues возвращает вещественный массив входных значений сети.


## Комментарии


Число элементов в массиве совпадает с числом входов сети.


Использование метода приведено в примере для [INeuralNetwork.SetInputValues](INeuralNetwork.SetInputValues.htm).
