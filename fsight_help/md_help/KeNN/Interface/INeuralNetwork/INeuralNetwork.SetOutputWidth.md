# INeuralNetwork.SetOutputWidth


## Синтаксис


SetOutputWidth(newWidth: Integer): Integer;


## Параметры


newWidth. Число строк.


## Описание


Метод SetOutputWidth устанавливает число строк в выходном слое самоорганизующейся карты Кохонена.


## Комментарии


Метод SetOutputWidth - это дубликат метода [INeuralNetwork.SetRowWidth](INeuralNetwork.SetRowWidth.htm).
