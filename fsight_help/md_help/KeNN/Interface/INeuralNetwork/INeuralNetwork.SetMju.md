# INeuralNetwork.SetMju


## Синтаксис


SetMju(newMju: Double);


## Параметры


newMju. Коэффициент момента инерции. Значение в диапазоне [0.0; 1.0].


## Описание


Метод SetMju устанавливает коэффициент момента инерции для обучения всех слоев сети обратного распространения.


## Комментарии


Момент инерции определяет, какую часть значения delta (рассчитанного в предыдущую эпоху обучения) использовать в текущую эпоху для коррекции весов синапсов. Если вы не хотите использовать момент инерции, то установите параметр newMju равным нулю.


Использование метода приведено в примере для [INeuralNetwork.LearnBack](INeuralNetwork.LearnBack.htm).
