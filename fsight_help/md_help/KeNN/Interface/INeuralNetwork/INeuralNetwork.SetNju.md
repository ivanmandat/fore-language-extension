# INeuralNetwork.SetNju


## Синтаксис


SetNju(newNju: Double);


## Параметры


SetNju. Скорость обучения сети. Значение в диапазоне [0.0; 1.0].


## Описание


Метод SetNju устанавливает скорость обучения сети с обратным распространением.


## Комментарии


Скорость обучения определяет, насколько будут изменяться веса синапсов в процессе обучения. Обычно данное значение больше «0,5» в начале обучение и стремится к нулю при окончании обучения.


Использование метода приведено в примере для [INeuralNetwork.LearnBack](INeuralNetwork.LearnBack.htm).
