# INeuralNetwork.GetMaximumWeightDelta


## Синтаксис


GetMaximumWeightDelta: Integer;


## Описание


Метод GetMaximumWeightDelta возвращает максимальное значение delta среди всех весов синапсов.


## Комментарии


Метод используется только для сетей обратного распространения.


delta - это значение ошибки, которое будет использовано для изменения веса синапса.


Использование метода приведено в примере для [INeuralNetwork.LearnBack](INeuralNetwork.LearnBack.htm).
