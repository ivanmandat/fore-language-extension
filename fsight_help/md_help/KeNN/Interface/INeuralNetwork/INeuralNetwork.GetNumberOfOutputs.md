# INeuralNetwork.GetNumberOfOutputs


## Синтаксис


GetNumberOfOutputs: Integer;


## Описание


Метод GetNumberOfOutputs возвращает число выходов сети.


## Комментарии


Число выходов сети - это число нейронов в выходном слое. Для самоорганизующейся карты Кохонена оно соответствует числу классов. Также для карты Кохонена выходной слой может быть представлен линейно, либо в виде двухмерной карты. Подробности содержатся в описании методов [INeuralNetwork.GetRowWidth](INeuralNetwork.GetRowWidth.htm) и [INeuralNetwork.SetRowWidth](INeuralNetwork.SetRowWidth.htm).


Использование метода приведено в примере для [INeuralNetwork.GetClosestNeuron](INeuralNetwork.GetClosestNeuron.htm).
