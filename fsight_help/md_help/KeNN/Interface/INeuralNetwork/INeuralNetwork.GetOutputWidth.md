# INeuralNetwork.GetOutputWidth


## Синтаксис


GetOutputWidth: Integer;


## Описание


Метод GetOutputWidth возвращает число строк в выходном слое самоорганизующейся карты Кохонена.


## Комментарии


Метод GetOutputWidth - это дубликат метода [INeuralNetwork.GetRowWidth](INeuralNetwork.GetRowWidth.htm).
