# INeuralNetwork.NormalizeSynapses


## Синтаксис


NormalizeSynapses;


## Описание


Метод NormalizeSynapses нормализует значения весов синапсов всех слоев сети.


## Комментарии


Для нормализации значений весов синапсов определенного слоя используйте [INeuralNetwork.NormalizeSynapsesEx](INeuralNetwork.NormalizeSynapsesEx.htm), а для нормализации входных данных - [INeuralNetwork.NormalizeInputValues](INeuralNetwork.NormalizeInputValues.htm).


Использование метода приведено в примере для [INeuralNetwork.NormalizeInputValues](INeuralNetwork.NormalizeInputValues.htm).
