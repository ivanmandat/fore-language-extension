# INeuralNetwork.SetNjuEx


## Синтаксис


SetNjuEx(newNju: Double; layerIndex: Integer);


## Параметры


SetNju. Скорость обучения слоя сети. Значение в диапазоне [0.0; 1.0].


layerIndex. Индекс слоя.


## Описание


Метод SetNjuEx устанавливает скорость обучения для указанного слоя сети с обратным распространением.


## Комментарии


Подробнее о скорости обучения см. [INeuralNetwork.SetNju](INeuralNetwork.SetNju.htm).


Индексация слоев сети сквозная начинается с нуля.
