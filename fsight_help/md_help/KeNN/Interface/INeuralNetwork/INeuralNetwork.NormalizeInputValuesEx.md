# INeuralNetwork.NormalizeInputValuesEx


## Синтаксис


NormalizeInputValuesEx(layerIndex: Integer): Boolean;


## Параметры


layerIndex. Индекс слоя.


## Описание


Метод NormalizeInputValuesEx
 нормализует входные данные указанного слоя.


## Комментарии


Метод предназначен для внутреннего использования. Не вызывайте метод
 самостоятельно.


Метод возвращает результат выполнения. Допустимые значения:


	- True. Метод завершился
	 успешно;


	- False. При выполнении
	 метода произошла ошибка.
