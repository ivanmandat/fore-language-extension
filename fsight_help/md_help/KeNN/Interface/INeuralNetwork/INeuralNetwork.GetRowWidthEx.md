# INeuralNetwork.GetRowWidthEx


## Синтаксис


GetRowWidthEx(layerIndex: Integer): Integer;


## Параметры


layerIndex. Индекс слоя.


## Описание


Метод GetRowWidthEx возвращает число строк в указанном слое самоорганизующейся карты Кохонена.


## Комментарии


Индексация слоев сквозная и начинается с нуля. Метод GetRowWidthEx подобен методу [INeuralNetwork.GetRowWidth](INeuralNetwork.GetRowWidth.htm).


Использование метода приведено в примере для [INeuralNetwork.SetRowWidthEx](INeuralNetwork.SetRowWidthEx.htm).
