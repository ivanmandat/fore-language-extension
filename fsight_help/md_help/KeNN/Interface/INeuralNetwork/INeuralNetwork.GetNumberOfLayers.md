# INeuralNetwork.GetNumberOfLayers


## Синтаксис


GetNumberOfLayers: Integer;


## Описание


Метод GetNumberOfLayers возвращает число слоев в нейронной сети.


## Комментарии


Обычно сети с обратным распространением ошибки состоят из двух слоев, а самоорганизующейся сети Кохонена из одного.


Использование метода приведено в примере для [INeuralNetwork.InitSynapses](INeuralNetwork.InitSynapses.htm).
