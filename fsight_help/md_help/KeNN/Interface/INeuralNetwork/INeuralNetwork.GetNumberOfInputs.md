# INeuralNetwork.GetNumberOfInputs


## Синтаксис


GetNumberOfInputs: Integer;


## Описание


Метод GetNumberOfInputs возвращает количество входов сети.


## Комментарии


Число входов соответствует числу синапсов в каждом нейроне первого слоя сети.


Использование метода приведено в примере для [INeuralNetwork.SetInputValuesConvex](INeuralNetwork.SetInputValuesConvex.htm).
