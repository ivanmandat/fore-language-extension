# INeuralNetwork.NormalizeSynapsesEx


## Синтаксис


NormalizeSynapsesEx(layerIndex: Integer);


## Параметры


layerIndex. Индекс слоя.


## Описание


Метод NormalizeSynapsesEx нормализует значения весов для синапсов указанного слоя.


## Комментарии


Индексация слоев сквозная и начинается с нуля.


Для нормализации значений весов всех синапсов используйте [INeuralNetwork.NormalizeSynapses](INeuralNetwork.NormalizeSynapses.htm), а для нормализации входных данных - [INeuralNetwork.NormalizeInputValues](INeuralNetwork.NormalizeInputValues.htm).


Использование метода приведено в примере для [INeuralNetwork.NormalizeInputValues](INeuralNetwork.NormalizeInputValues.htm).
