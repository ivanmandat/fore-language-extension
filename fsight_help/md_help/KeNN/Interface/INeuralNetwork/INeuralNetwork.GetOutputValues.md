# INeuralNetwork.GetOutputValues


## Синтаксис


GetOutputValues: Array;


## Описание


Метод GetOutputValues возвращает массив выходных значений сети.


## Комментарии


Для сети обратного распространения данные значения являются результатами работы сети. Для самоорганизующейся сети Кохонена выходные значения могут быть использованы для наблюдением за процессом обучения.


Использование метода приведено в примере для [INeuralNetwork.GetClosestNeuron](INeuralNetwork.GetClosestNeuron.htm).
