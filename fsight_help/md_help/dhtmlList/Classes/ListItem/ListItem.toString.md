# ListItem.toString


## Синтаксис


toString();


## Описание


Метод toString возвращает содержимое компонента [ListItem](../../Components/ListItem/ListItem.htm) в виде строки.


## Пример


Для выполнения примера создайте список, состоящий из трех элементов, как показано на странице «[Конструктор ListItem](Constructor_ListItem.htm)». Третий компонент списка является невидимым (ListItem.Visibility). Создадим кнопку, при нажатии на которую будет вызываться содержимое третьей строки списка.


<button onclick="alert(listitem2.toString())">ShowContent</button>


После выполнения примера при нажатии на кнопку на экран будет выведена строка с содержимым элемента списка с наименованием «listitem2».
