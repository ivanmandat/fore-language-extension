# ListItem.setIsHovered


## Синтаксис


setIsHovered(value);


## Параметры


value. Определяет способ отображения
 строки списка. Если установлено значение true,
 то строка списка подсвечивается аналогично наведению на нее курсора
 мыши. По умолчанию установлено значение false.


## Описание


Метод setIsHovered устанавливает
 признак того, что мышь наведена на указанную строку списка.


## Пример


Для выполнения примера создайте список, состоящий из трех элементов,
 как показано на странице «[Конструктор
 ListItem](Constructor_ListItem.htm)». Создадим кнопку, при нажатии на которую будет вызван
 метод setIsHovered:


<button onclick="listitem.setIsHovered(true)">IsHovered</button>


После выполнения примера при нажатии на кнопку первый элемент списка
 будет подсвечен аналогично строкам, на которые наведен курсор мыши.
