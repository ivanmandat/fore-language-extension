# ListItem.ImageURL


## Синтаксис


ImageURL: String


## Описание


Свойство ImageURL определяет
 путь к изображению предварительного просмотра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setImageURL,
 а возвращается с помощью метода getImageURL.


## Пример


Для выполнения примера необходимо, чтобы было открыто веб-приложение
 «Форсайт. Аналитическая платформа».
 Откройте каталог с расположенным в нём отчётом, настроенным для предварительного
 просмотра. Указанный ниже сценарий нужно запустить в консоли браузера.


Получим данные по элементу списка объектов репозитория:


// Получим экземпляр списка объектов
modObj = PP.App.getModuleObject();
RepBox = modObj.getReportBox();
NavBox = RepBox.getNavigatorBox();
MbObjectsIconView = NavBox.getCurrentModeControl();
exemp = MbObjectsIconView.getItems()[0];
// Получим путь к изображению предварительного просмотра экземпляра
(exemp.getHasPrv() && exemp.getIconType() == "Giant") ? console.log("Путь к изображению: " + exemp.getImageUrl()) : console.log("Предпросмотр не настроен")
В результате в консоль будет выведен путь к изображению предварительного
 просмотра, если предварительный просмотр настроен правильно.
