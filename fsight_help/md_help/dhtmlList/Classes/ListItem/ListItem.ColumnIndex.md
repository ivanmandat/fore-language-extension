# ListItem.ColumnIndex


## Синтаксис


ColumnIndex: Number


## Описание


Свойство ColumnIndex определяет индекс столбца с пиктограммой. Данный индекс используется для расчета смещения пиктограммы по горизонтали относительно края спрайта.


## Комментарии


Значение 0 соответствует первому столбцу, 1 - второму и т.д.
