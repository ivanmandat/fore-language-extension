# ListItem.RowIndex


## Синтаксис


RowIndex: Number


## Описание


Свойство RowIndex определяет индекс ряда с пиктограммой. Данный индекс используется для расчета смещения пиктограммы по вертикали относительно края спрайта.


## Комментарии


Значение 0 соответствует первому ряду, 1 - второму и т.д.
