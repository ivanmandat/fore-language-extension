# ListItem.Selected


## Синтаксис


Selected: Boolean


## Описание


Свойство Selected устанавливает признак того, является ли элемент выделенным.


## Комментарии


Если для свойства установлено значение true, то строка списка является выделенной. По умолчанию установлено значение false.
