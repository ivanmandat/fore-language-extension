# ListItem.getParent


## Синтаксис


getParent();


## Описание


Метод getParent возвращает родительский компонент строки списка.


## Комментарии


Родительским компонентом для строки списка является экземпляр компонента [ListBox](../ListBox/ListBox.htm).
