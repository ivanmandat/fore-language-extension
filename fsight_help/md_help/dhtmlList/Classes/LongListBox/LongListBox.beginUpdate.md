# LongListBox.beginUpdate


## Синтаксис


beginUpdate();


## Описание


Метод beginUpdate отключает перерисовку списка до тех пор, пока не будет вызван метод [endUpdate](LongListBox.endUpdate.htm).

Пример использования метода приведен в описание метода [LongListBox.isAllLoaded](LongListBox.isAllLoaded.htm).
