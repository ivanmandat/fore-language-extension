# LongListBox.getItems


## Синтаксис


getItems(toArray: Boolean);


## Параметры


toArray. Признак, указывающий на возвращение элементов в виде массива. Если значение равно true, то элементы будут возвращены в виде массива, иначе в виде объекта.


## Описание


Метод getItems возвращает загруженные элементы сгруппированного списка.


## Комментарии


Элементы массива - объекты класса PP.Ui.LongListItem.
