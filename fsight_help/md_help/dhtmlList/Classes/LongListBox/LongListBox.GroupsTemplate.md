# LongListBox.GroupsTemplate


## Синтаксис


GroupsTemplate: Array


## Описание


Свойство GroupsTemplate содержит шаблон для заголовков групп списка.


## Комментарии


Заголовком является элемент, для свойства IsGroup которого имеет установлено значение true.


Значение свойства устанавливается из JSON и с помощью метода setGroupsTemplate, а возвращается с помощью метода getGroupsTemplate.
