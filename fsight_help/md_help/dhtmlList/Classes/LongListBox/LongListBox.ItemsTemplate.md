# LongListBox.ItemsTemplate


## Синтаксис


ItemsTemplate: Number


## Описание


Свойство ItemsTemplate содержит шаблон для элементов списка.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setItemsTemplate, а возвращается с помощью метода getItemsTemplate.
