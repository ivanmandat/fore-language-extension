# LongListBox.ItemsHeight


## Синтаксис


ItemsHeight: Number


## Описание


Свойство ItemsHeight содержит высоту элементов списка.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setItemsHeight, а возвращается с помощью метода getItemsHeight.
