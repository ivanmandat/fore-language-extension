# Конструктор LongListBox


## Синтаксис


PP.Ui.LongListBox(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор LongListBox создает экземпляр компонента [LongListBox](../../Components/LongListBox/LongListBox.htm).
