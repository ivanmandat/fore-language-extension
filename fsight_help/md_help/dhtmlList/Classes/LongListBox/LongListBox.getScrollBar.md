# LongListBox.getScrollBar


## Синтаксис


getScrollBar();


## Описание


Метод getScrollBar возвращает
 вертикальную полосу прокрутки сгруппированного списка.


## Комментарии


Метод возвращает объект типа [PP.Ui.ScrollBar](dhtmlUi.chm::/Classes/ScrollBar/ScrollBar.htm).
