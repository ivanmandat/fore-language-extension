# LongListBox.GroupsHeight


## Синтаксис


GroupsHeight: Number


## Описание


Свойство GroupsHeight содержит
 высоту заголовков групп списка.


## Комментарии


Заголовком является элемент, для свойства IsGroup
 которого установлено значение true.


Значение свойства устанавливается из JSON и с помощью метода setGroupsHeight,
 а возвращается с помощью метода getGroupsHeight.
