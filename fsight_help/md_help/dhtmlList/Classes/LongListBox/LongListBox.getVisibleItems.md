# LongListBox.getVisibleItems


## Синтаксис


getVisibleItems();


## Описание


Метод getVisibleItems возвращает массив видимых элементов сгруппированного списка.


## Комментарии


Элементы массива - объекты класса PP.Ui.LongListItem.
