# LongListBox.Groups


## Синтаксис


Groups: Array


## Описание


Свойство Groups содержит массив настроек групп элементов.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setGroups, а возвращается с помощью метода getGroups.
