# LongListBox.getMultiSelect


## Синтаксис


getMultiSelect();


## Описание


Метод getMultiSelect возвращает
 признак множественного выбора элементов сгруппированного списка.


## Комментарии


Метод возвращает значение true,
 если доступен множественный выбор элементов сгруппированного списка, иначе
 - false.


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [LongListBox](../../Components/LongListBox/LongListBox.htm)
 c наименованием «longListBox» (см. «[Пример
 создания компонента LongListBox](../../Components/LongListBox/Example_LongListBox.htm)»). Пример нужно разместить в теге
 <script>. Проверяем доступность множественного выбора элементов
 списка:


// Проверяем доступность множественного выбора в списке
if (longListBox.getMultiSelect()) {
    console.log("Доступен множественный выбор в сгруппированном списке");
} else {
    console.log("Множественный выбор в сгруппированном списке недоступен");
}
В результате в консоль будет выведен результат проверки доступности
 множественного выбора:


Множественный выбор в сгруппированном списке недоступен
