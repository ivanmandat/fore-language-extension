# List.add


## Синтаксис


add (ctrl: PP.Ui.[Control](dhtmlUi.chm::/Classes/control/control.htm),
 fireEvent: Boolean);


## Параметры


ctrl. Элемент управления, который
 необходимо добавить в коллекцию;


fireEvent. Определяет, будет
 ли при вызове метода генерироваться соответствующее событие. Если установлено
 значение true, событие генерируется,
 иначе - не генерируется.


## Описание


Метод add добавляет элемент
 в коллекцию.


## Комментарии


Элемент добавляется в конец коллекции.
