# List.Changing


## Синтаксис


Changing: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие Changed наступает при изменении состояния коллекции (во время любой операции, произведенной над коллекцией: добавление, удаление, вставка и т.д.).
