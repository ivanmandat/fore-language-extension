# List.Changed


## Синтаксис


Changed: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие Changed наступает после редактирования коллекции (после любой операции, произведенной над коллекцией: добавление, удаление, вставка и т.д.).
