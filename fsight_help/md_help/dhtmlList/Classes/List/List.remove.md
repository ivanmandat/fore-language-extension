# List.remove


## Синтаксис


remove (item, withDispose: Boolean, fireEvent: Boolean);


## Параметры


item. Удаляемый объект или
 индекс удаляемого объекта;


withDispose. Если значение
 параметра true, то объект уничтожается,
 а если false, то просто удаляется
 из [списка элементов](List.Items.htm) PP.[List](List.htm);


fireEvent. Определяет, будет
 ли при вызове метода генерироваться соответствующее событие. Если установлено
 значение true, событие генерируется,
 иначе - не генерируется.


## Описание


Метод remove удаляет указанный
 элемент из коллекции.
