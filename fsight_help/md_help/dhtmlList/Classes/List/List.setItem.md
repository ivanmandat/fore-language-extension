# List.setItem


## Синтаксис


setItem (value, index)


## Параметры


value. Элемент,
 который необходимо записать в коллекцию;


index. Индекс
 элемента коллекции.


## Описание


Метод setItem записывает элемент
 в определенную позицию коллекции.
