# List.insertBefore


## Синтаксис


insertBefore (newCtrl, refCtrl)


## Параметры


newCtrl. Добавляемый компонент;


refCtrl. Компонент, перед которым
 необходимо произвести вставку.


## Описание


Метод insertBefore добавляет
 элемент в позицию перед указанным элементом.
