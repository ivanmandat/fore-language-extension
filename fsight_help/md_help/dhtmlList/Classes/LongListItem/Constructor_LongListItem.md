# Конструктор LongListItem


## Синтаксис


PP.Ui.LongListItem(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор LongListItem создает экземпляр компонента [LongListItem](../../Components/LongListItem/LongListItem.htm).


## Пример


Конструктор предназначен для внутреннего использования классом [LongListBox](../LongListBox/LongListBox.htm).
