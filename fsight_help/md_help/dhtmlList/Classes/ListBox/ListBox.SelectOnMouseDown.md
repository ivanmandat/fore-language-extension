# ListBox.SelectOnMouseDown


## Синтаксис


SelectOnMouseDown: Boolean


## Описание


Свойство SelectOnMouseDown определяет
 признак выделения элементов при нажатии на них.


## Комментарии


Допустимые значения:


	- true.
	 По щелчку мыши будет происходить выделение элемента;


	- false.
	 По щелчку мыши выделение элемента происходить не будет (по умолчанию).


Значение свойства устанавливается из JSON и с помощью метода setSelectOnMouseDown,
 а возвращается с помощью метода getSelectOnMouseDown.
