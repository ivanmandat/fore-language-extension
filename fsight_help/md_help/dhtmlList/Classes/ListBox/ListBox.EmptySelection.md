# ListBox.EmptySelection


## Синтаксис


EmptySelection: Boolean;


## Описание


Свойство EmptySelection определяет
 возможность выбора пустого множества элементов.


## Комментарии


Свойство учитывается при множественном выборе элементов списка.


Допустимые значения:


	- true.
	 Возможен выбор пустого множества элементов;


	- false.
	 Выбор пустого множества невозможен (по умолчанию).


Значение свойства устанавливается из JSON и с помощью метода setEmptySelection,
 а возвращается с помощью метода getEmptySelection.
