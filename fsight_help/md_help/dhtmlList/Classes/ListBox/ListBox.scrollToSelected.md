# ListBox.scrollToSelected


## Синтаксис


scrollToSelected();


## Описание


Метод scrollToSelected прокручивает
 список до выделенного элемента списка.


## Комментарии


Для выполнения метода необходимо, чтобы значение свойства [IsScrollToSelected](ListBox.IsScrollToSelected.htm)
 было равно true.
