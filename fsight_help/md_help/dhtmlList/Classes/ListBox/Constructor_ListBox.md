# Конструктор ListBox


## Синтаксис


PP.Ui.ListBox(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор ListBox создает экземпляр компонента [ListBox](../../Components/ListBox/ListBox.htm).
