# ListBox.Items


## Синтаксис


Items: Array


## Описание


Свойство Items содержит массив
 элементов списка.


## Комментарии


Элементы массива являются объектами класса [PP.Ui.ListItem](../ListItem/ListItem.htm).


Значение свойства устанавливается из JSON и с помощью метода setItems,
 а возвращается с помощью метода getItems.
