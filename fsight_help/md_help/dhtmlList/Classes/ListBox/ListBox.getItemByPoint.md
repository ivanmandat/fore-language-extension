# ListBox.getItemByPoint


## Синтаксис


getItemByPoint(x, y);


## Параметры


x. Координата по горизонтали;


y. Координата по вертикали.


## Описание


Метод getItemByPoint возвращает
 элемент списка по заданным координатам.
