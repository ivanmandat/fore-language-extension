# ListBox.addItem


## Синтаксис


addItem(value);


## Параметры


value. Объект, который устанавливается как элемент списка. Это может быть текст, html-разметка, Dom-вершина или визуальный компонент - наследник PP.Ui.[Control](dhtmlUi.chm::/Classes/control/control.htm).


## Описание


Метод addItem добавляет новый элемент в список.
