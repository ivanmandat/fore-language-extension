# ListBox.SelectedIndex


## Синтаксис


SelectedIndex: Number


## Описание


Свойство SelectedIndex определяет индекс выделенной строки списка.


## Комментарии


По умолчанию при создании экземпляра компонента [ListBox](../../Components/ListBox/ListBox.htm) ни одна строка не выделяется, возвращается индекс -1.


## Пример


Для выполнения примера создадим компонент [ListBox](../../Components/ListBox/ListBox.htm) (см. «[Пример создания компонента ListBox](../../Components/ListBox/Example_ListBox.htm)»). Для того чтобы при размещении компонента на html-странице первый элемент списка был выделен, используйте свойство SelectedIndex:


SelectedIndex: 0


После выполнения примера на html-странице будет размещен список, первый элемент которого будет выделен.
