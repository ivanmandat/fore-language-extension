# ListBox.ItemEditing


## Синтаксис


ItemEditing: function (sender, args)


## Параметры


sender. Источник события;


args. Информация
 о событии. Аргумент Item
 возвращает элемент списка, который редактируется.


## Описание


Событие ItemEditing наступает
 при начале редактирования элемента списка.
