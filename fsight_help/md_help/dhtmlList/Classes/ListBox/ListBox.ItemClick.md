# ListBox.ItemClick


## Синтаксис


ItemClick: function (sender, args)


## Параметры


sender. Источник события;


args. Информация
 о событии. Аргумент Item
 возвращает элемент списка, на который произошло нажатие.


## Описание


Событие ItemClick наступает
 при нажатии на элемент списка.


## Коммментарии


Событие срабатывает после отпускания кнопки мыши на том же элементе,
 на который было произведено нажатие.
