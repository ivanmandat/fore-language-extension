# ListBox.ItemDeselected


## Синтаксис


ItemDeselected: function (sender, args)


## Параметры


sender. Источник события;


args. Информация о событии.
 Аргумент ListItem содержит элемент
 списка, с которого была снята отметка.


## Описание


Событие ItemDeselected наступает
 при снятии отметки с элемента списка.


## Комментарии


Событие актуально, если включена возможность [множественного
 выбора](Listbox.Multiselect.htm).

Пример использования события приведен в описании свойства [Listbox.Multiselect](Listbox.Multiselect.htm).
