# ListBox.selectItemById


## Синтаксис


selectItemById(Id: String, fireEvent: true);


## Параметры


Id. Идентификатор элемента;


fireEvent. Признак того, будет
 ли вызвано событие снятия выделения с предыдущего выделенного элемента.
 Если параметр имеет значение true,
 то событие будет вызвано, иначе не будет.


## Описание


Метод selectItemById выделяет
 элемент с указанным идентификатором.
