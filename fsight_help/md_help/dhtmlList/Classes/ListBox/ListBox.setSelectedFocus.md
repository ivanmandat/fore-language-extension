# ListBox.setSelectedFocus


## Синтаксис


setSelectedFocus(value: Number, fireEvent: Boolean,
 oldIndex: Number);


## Параметры


value. Индекс элемента;


fireEvent. Признак того, будет
 ли вызвано событие снятия выделения с предыдущего выделенного элемента.
 Если параметр имеет значение true,
 то событие будет вызвано, иначе не будет.


oldIndex. Индекс предыдущего
 выделенного элемента.


## Описание


Метод setSelectedFocus устанавливает
 фокус по индексу элемента.
