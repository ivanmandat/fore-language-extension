# ListBox.IsScrollToSelected


## Синтаксис


IsScrollToSelected: Boolean


## Описание


Свойство IsScrollToSelected
 определяет признак прокручивания списка до выделенного элемента.


## Комментарии


Допустимые значения:


	- true. По умолчанию.
	 Возможно прокручивание списка до выделенного элемента;


	- false. Прокручивание
	 списка до выделенного элемента невозможно.


Значение свойства устанавливается из JSON и с помощью метода setIsScrollToSelected,
 а возвращается с помощью метода getIsScrollToSelected.
