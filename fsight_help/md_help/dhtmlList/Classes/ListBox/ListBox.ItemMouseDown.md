# ListBox.ItemMouseDown


## Синтаксис


ItemMouseDown: function (sender, args)


## Параметры


sender. Источник события;


args. Информация
 о событии. Аргумент Item
 возвращает элемент списка, на который произошло зажатие.


## Описание


Событие ItemMouseDown наступает
 при зажатии кнопки мыши на элементе.
