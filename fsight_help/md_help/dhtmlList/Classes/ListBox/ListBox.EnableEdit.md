# ListBox.EnableEdit


## Синтаксис


EnableEdit: Boolean;


## Описание


Свойство EnableEdit определяет
 возможность редактирования элементов списка.


## Комментарии


Допустимые значения:


	- true.
	 Редактирование элементов разрешено;


	- false.
	 Редактирование элементов запрещено (по умолчанию).


Значение свойства устанавливается из JSON и с помощью метода setEnableEdit,
 а возвращается с помощью метода getEnableEdit.
