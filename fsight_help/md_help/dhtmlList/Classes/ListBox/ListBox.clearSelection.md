# ListBox.clearSelection


## Синтаксис


clearSelection();


## Описание


Метод clearSelection предназначен для отмены выделения.


## Пример


Для выполнения примера создадим компонент [ListBox](../../Components/ListBox/ListBox.htm) (см."[Пример создания компонента ListBox](../../Components/ListBox/Example_ListBox.htm)") c наименованием «listbox». Создадим html-кнопку с текстом «Сбросить выделение», при нажатии на которую будет вызываться метод clearSelection:


<button onclick="listbox.clearSelection();">Сбросить выделение</button>


Выделите какой-либо элемент созданного списка щелчком мыши. При нажатии на кнопку выделение будет снято.
