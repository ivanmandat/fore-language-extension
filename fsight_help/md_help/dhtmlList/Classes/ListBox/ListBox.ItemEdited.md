# ListBox.ItemEdited


## Синтаксис


ItemEdited: function (sender, args);


## Параметры


sender. Источник события;


args. Информация
 о событии. Аргумент prevValue содержит значение элемента до изменения.
 Аргумент newValue содержит новое значение элемента.


## Описание


Событие ItemEdited наступает
 после завершения редактирования элемента списка.
