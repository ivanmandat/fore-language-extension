# LongListBox


## Описание


Компонент LongListBox представляет
 собой сгруппированный список.


## Комментарии


Компонент реализован классом [LongListBox](../../Classes/LongListBox/LongListBox.htm).


## Пример


Компонент имеет следующий вид:


![](LongListBox.png)


Пример создания сгруппированного списка описан на странице «[Пример
 создания компонента LongListBox](Example_LongListBox.htm)».
