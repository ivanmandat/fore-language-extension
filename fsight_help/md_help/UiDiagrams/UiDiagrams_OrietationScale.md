# Настройка ориентации шкалы


Настройка ориентации шкалы позволяет менять местами оси категорий и
 значений.


Для настройки ориентации шкалы диаграммы выберите вариант ориентации
 шкалы в окне «[Тип диаграммы](Type_diagrams/ChangingChartType.htm)».


Совет. Для быстрой
 настройки ориентации шкалы диаграммы используйте вкладку «[Диаграмма](Params_diagram/UiDiagrams_params_diagram.htm)»
 боковой панели.


[![](opened.gif)![](closed.gif)Для вызова окна
 «Тип диаграммы»](javascript:TextPopup(this))


		- Выполните команду «Тип
		 диаграммы» в контекстном меню диаграммы;


	Примечание.
	 В инструменте «Аналитические панели»
	 выполните команду «Диаграмма >
	 Тип диаграммы» в контекстном меню диаграммы.


		- Выберите пункт «Все типы
		 диаграмм» в раскрывающемся меню кнопки «Тип»,
		 расположенной в группе «Вид»
		 на вкладке «Диаграмма»
		 ленты инструментов.


Примечание.
 Настройка ориентации шкалы диаграммы с помощью окна «Тип диаграммы» доступна только в
 настольном приложении для любого типа диаграмм, кроме [круговой](Type_diagrams/UiDiagrams_round.htm),
 [кольцевой](Type_diagrams/Doughnut_chart.htm), [лепестковой](Type_diagrams/UiDiagrams_petalous.htm),
 [вторичной гистограммы](Type_diagrams/UiDiagrams_secondary_histogram.htm)
 и [вторичной круговой
 диаграммы](Type_diagrams/UiDiagrams_round_secondary.htm).


## Варианты ориентации


Для графиков и диаграмм предусмотрено два варианта ориентации шкал:


	- Слева
	 направо. При ориентации шкалы слева направо график строится
	 по точкам слева направо. Пример отображения [гистограммы](Type_diagrams/UiDiagrams_Bar.htm),
	 ориентированной слева направо (обычной и объемной):


![](Property_diagramm/orientation%20-%2011.gif)


	- Снизу
	 вверх. При ориентации шкалы снизу вверх график строится по
	 точкам снизу вверх. Пример отображения [гистограммы](Type_diagrams/UiDiagrams_Bar.htm),
	 ориентированной снизу вверх (обычной и объемной):


![](Property_diagramm/orientation%20-%2022.gif)
