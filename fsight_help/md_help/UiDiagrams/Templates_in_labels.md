# Шаблоны


Шаблон - подстановка, отображающая
 параметры компонента диаграммы, такие как значение, наименование, номер
 и др.


Шаблоны используются в:


	- [подписях
	 данных](Params_diagram/UiDiagrams__DataTitle.htm) точки ряда, точек одного ряда, всех точек диаграммы;


	- [подсказках](Params_diagram/Tooltips.htm) для всех
	 точек диаграммы (за исключением шаблона %Data);


	- [подписях
	 линий уровня](Tuning_format/Format_Axis/UiDiagrams_FormatAxis_Levels.htm).


Для настройки текста подписи с использованием шаблона нажмите на кнопку
 «Шаблоны», расположенную рядом
 с полем ввода «Текст», «Текст подсказки», «Подпись».


		 Шаблоны текста
		 Определение


		 %Autovalue
		 Автоматический формат значения (установлен по умолчанию).


			- Для всех диаграмм, кроме [круговой](Type_diagrams/UiDiagrams_round.htm),
			 будет использовано значение точки ряда данных.


			- Для [круговой
			 диаграммы](Type_diagrams/UiDiagrams_round.htm), [вторичной
			 круговой диаграммы](Type_diagrams/UiDiagrams_round_secondary.htm) и [вторичной
			 гистограммы](Type_diagrams/UiDiagrams_secondary_histogram.htm) используется отношение значения точки ряда
			 к сумме значений этой же точки всех рядов.


		 %Value
		 Значение точки.


		 %XValue
		 Значение точки по оси X.


		 %YValue
		 Значение точки по оси Y.


		 %Part
		 Значение с накоплением.


		 %Percent
		 Доли в процентах - отношение значения точки ряда к сумме значений
		 этой же точки всех рядов, представленное в процентах.


		 %PercentHundredth
		 Доли - отношение значения точки ряда к сумме значений этой
		 же точки всех рядов.


		 %PercentPart
		 Доли в процентах с накоплением.


		 %PercentPartHundredth
		 Доли с накоплением.


		 %Data
		 Пользовательский формат данных. Настройка формата возможна
		 только с помощью [языка
		 Fore](Chart.chm::/Class/UiChart/UiChart.OnGetLabelText.htm).


		 %SerieName
		 Имя ряда.


		 %SerieNo
		 Номер ряда.


		 %PointName
		 Имя точки.
		Примечание.
		 Если в окне «[Исходные данные](UiReport.chm::/desktop/Diagrams/UiDiagrams_Report_create_source.htm)»
		 было настроено отображение точек на оси категории, то при установке
		 шаблона «%PointName»
		 подписи данных будут отображаться только для выбранных точек.


		 %PointNo
		 Номер точки.


		 %Computation
		 Тип функции в подписи линии уровня.
		Примечание.
		 Шаблон доступен только для подписи [линий
		 уровня](Tuning_format/Format_Axis/UiDiagrams_FormatAxis_Levels.htm).
