# Настройка компонентов диаграммы


Диаграммы представляют собой
 различные формы графического отображения табличных данных и служат для
 их наглядного отображения:


![](Tuning_format/Pics%20tuning/Components.gif)


Диаграмма состоит из набора компонентов, каждый из которых обладает
 определенным набором параметров:


	- [область
	 диаграммы](Tuning_format/UiDiagrams_tuning_area_digramm.htm) включает в себя все компоненты диаграммы;


	- [оси](Params_diagram/UiDiagrams_Axis.htm) служат
	 для упорядочивания данных на диаграмме;


	- [область
	 построения](Tuning_format/uidiagrams_tuning_construction_area_digramm.htm) включает в себя ряды данных диаграммы;


	- [ряды данных](Series.htm) содержит набор значений,
	 на основе которых строится диаграмма;


	- [коридоры диаграммы](UiDiagrams_corridors.htm) отображает
	 часть области построения, находящуюся между двумя рядами линейной
	 диаграммы;


	- [линии тренда](look_and_feel/Report/Trend_Lines.htm)
	 отображают тенденции данных;


	- [заголовки](Params_diagram/UiDiagrams_titles.htm)
	 содержат названия других компонентов диаграммы;


	- [подписи
	 данных](Property_diagramm/UiDiagrams_PropertyDiagramm_Label.htm) содержат сведения о рядах или отдельных точках данных;


	- [всплывающие подсказки](Params_diagram/Tooltips.htm)
	 отображают информацию о точке ряда данных при наведении на нее курсора;


	- [легенда](Property_diagramm/UiDiagrams_PropertyDiagramm_legend.htm)
	 содержит названия и маркеры данных на диаграмме.


## Выделение компонента диаграммы


Для работы с компонентом диаграммы его необходимо выделить.


Для выделения компонента диаграммы в инструменте «Аналитические
 панели»:


	- щёлкните по компоненту диаграммы;


	- выполните команду «Диаграмма
	 > Выделить элемент» в контекстном меню области диаграммы.


Для выделения компонента диаграммы в инструментах «Аналитические
 запросы (OLAP)», «Анализ временных
 рядов»:


	- щёлкните по компоненту диаграммы;


	- выполните команду «Выделить
	 элемент» в контекстном меню области диаграммы.


Для выделения компонента диаграммы в инструменте «Отчеты»:


	- Активируйте
	 диаграмму, щёлкнув по ней.


	- Выделите
	 компонент. Для этого:


		- щёлкните по нему;


		- выполните команду «Выделить
		 элемент» в контекстном меню области диаграммы и выберите
		 элемент;


		- перейдите на вкладку «Диаграмма»
		 ленты инструментов и в раскрывающемся списке, расположенном в
		 группе «Текущий фрагмент»,
		 выберите необходимый элемент диаграммы. Содержание списка аналогично
		 всплывающему меню, которое отображается при выполнении команды
		 контекстного меню «Выделить элемент»
		 диаграммы.


После выполнения команды «Выделить
 элемент» из раскрывающегося списка выберите необходимый элемент:

 ряда](javascript:TextPopup(this))


	После выбора ряда он будет выделен в области построения диаграммы.


	Примечание.
	 В меню отображается первые десять рядов диаграммы.


[![](opened.gif)![](closed.gif)Выбрать ряды](javascript:TextPopup(this))


	Будет открыто окно «Выделить
	 ряды в диаграмме», содержащее список всех рядов диаграммы.
	 В данном списке осуществляется отметка рядов, которые должны быть
	 выделены в области построения диаграммы.


	Для отметки нескольких рядов зажмите клавишу CTRL и последовательно
	 выделите ряды с помощью мыши.


	Для выделения всех рядов в списке:


		- нажмите кнопку «Выделить
		 все»;


		- выполните команду контекстного меню «Выделить
		 все»;


		- нажмите сочетание клавиш CTRL+A.


После выбора необходимого компонента диаграммы он будет выделен в области
 построения диаграммы.
