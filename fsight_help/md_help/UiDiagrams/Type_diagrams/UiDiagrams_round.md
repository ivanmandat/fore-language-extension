# Круговая диаграмма


Круговая диаграмма демонстрирует
 размер элементов одного ряда данных пропорционально сумме элементов.


Каждая группа данных отображается в виде круга, разделённого на секторы,
 угловые размеры которых пропорциональны значениям элементов данных.


Пример отображения круговой диаграммы (обычной и объёмной):


![](Type%20pics/Type_4_Round.png)


В режиме редактирования диаграммы возможно изменение положения крайних
 секторов. Для этого захватите сектор и перетащите его на необходимое расстояние,
 используя механизм Drag&Drop. При этом размер всей диаграммы уменьшится.
 Также в режиме редактирования доступно изменение следующих параметров
 диаграммы:


	- [Формат
	 области диаграммы](../Tuning_format/UiDiagrams_tuning_area_digramm.htm);


	- [Формат
	 области построения](../Tuning_format/uidiagrams_tuning_construction_area_digramm.htm);


	- [Формат ряда данных](../Series.htm);


	- [Формат
	 легенды](../Property_diagramm/UiDiagrams_PropertyDiagramm_legend.htm).


Для данного типа диаграмм доступны следующие возможности:


	- [настройка
	 дополнительных параметров](../Params_diagram/UiDiagrams_extra.htm#additionally_pie);


	- объединение части секторов диаграммы в один. Для этого используются
	 [вторичная гистограмма](UiDiagrams_secondary_histogram.htm)
	 и [вторичная круговая диаграмма](UiDiagrams_round_secondary.htm);


	- отображение [рядов
	 в кольцах](../Params_diagram/UiDiagrams_extra.htm#additionally_pie). Ниже приведены примеры круговой диаграммы, для которой
	 настроено отображение рядов в кольцах (слева) и секторно (справа):


![](Type%20pics/series_in_doughnuts.png)


## Особенности при работе с круговой диаграммой


При исключении ряда данных в круговой диаграмме через легенду по умолчанию
 диаграмма не перестраивается, при этом образуются пустые места:


![](Type%20pics/Type_4_Round_4.png)
Примечание.
 Включить перестроение круговой диаграммы только по включенным в легенде
 элементам можно используя язык [Fore](Chart.chm::/Interface/IChart/IChart.ExcludeInvisibleSeries.htm).
