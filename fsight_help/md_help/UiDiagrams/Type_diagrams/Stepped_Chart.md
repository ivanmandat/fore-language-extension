# Ступенчатая диаграмма


Ступенчатая диаграмма предполагает
 только один вид и отображает абсолютные значения ряда. Ступенька представляет
 собой ломаную линию, соединяющую два соседних значения ряда.


Пример отображения ступенчатой диаграммы:


![](Type%20pics/Stepped_Chart.gif)


В режиме редактирования доступно изменение следующих параметров диаграммы:

	 области диаграммы](../Tuning_format/UiDiagrams_tuning_area_digramm.htm);


	- [Формат
	 области построения](../Tuning_format/uidiagrams_tuning_construction_area_digramm.htm);


	- [Формат осей](../Params_diagram/UiDiagrams_Axis.htm);


	- [Формат ряда данных](../Series.htm);


	- [Формат
	 легенды](../Property_diagramm/UiDiagrams_PropertyDiagramm_legend.htm).


Также для различных рядов данных можно настроить [положение
 ступенек](../Series.htm#stepline) и [тип отображения ряда](../Series.htm#type).
