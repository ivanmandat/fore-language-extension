# Биржевая диаграмма


Биржевая
 диаграмма наиболее часто используется для иллюстрации изменений
 цен на акции. Однако эта диаграмма может использоваться также для вывода
 научных данных. Например, можно использовать биржевые диаграммы для демонстрации
 колебаний дневных или годовых температур.


Для создания биржевой диаграммы необходимо правильно упорядочить выводимые
 данные. Порядок рядов строго определен. Возможны следующие варианты комбинации
 рядов в зависимости от диаграммы:


	- высокий курс, низкий курс, курс закрытия;


	- курс открытия, высокий курс, низкий курс, курс закрытия;


	- объем, высокий курс, низкий курс, курс закрытия;


	- объем, курс открытия, высокий курс, низкий курс, курс закрытия.


По умолчанию на диаграмме:


	- «Высокий курс» и «Низкий курс» отображаются треугольными
	 маркерами;


	- штрих соединяет максимальное и минимальное значение из рядов
	 «Высокий курс», «Низкий
	 курс»;


	- «Курс закрытия» отображается
	 меткой в плоскости штриха;


	- «Объем» отображается
	 в виде столбика, строится по основной оси;


Примечание.
 При использовании ряда «Объем»
 остальные ряды строятся по дополнительной оси. Расположение рядов по осям
 задается автоматически.


	- одновременное использование рядов «Курс
	 открытия» и «Курс закрытия»
	 отображается блоком.


Примечание.
 Если значение ряда «Курс открытия»
 больше значения ряда «Курс закрытия»,
 блок будет закрашен цветом ряда «Курс
 закрытия», если наоборот, блок будет закрашен цветом ряда «Курс открытия».
