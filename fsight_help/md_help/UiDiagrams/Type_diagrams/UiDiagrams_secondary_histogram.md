# Вторичная гистограмма


Вторичная гистограмма - [круговая
 диаграмма](UiDiagrams_round.htm) с частью значений, вынесенных в гистограмму.

Примечание.
 В веб-приложении вторичная гистограмма отображается в виде картинки.

	- [Формат
	 области построения](../Tuning_format/uidiagrams_tuning_construction_area_digramm.htm);


	- [Формат ряда данных](../Series.htm);


	- [Формат
	 легенды](../Property_diagramm/UiDiagrams_PropertyDiagramm_legend.htm).
