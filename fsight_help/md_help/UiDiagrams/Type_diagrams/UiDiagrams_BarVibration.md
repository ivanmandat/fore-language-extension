# Гистограмма с интервалами колебаний


Гистограмма с интервалами колебаний
 используется для отображения данных с интервалами колебаний. Аналогична
 [биржевой](UiDiagrams_birg.htm) диаграмме с 4 рядами c объемом,
 отличается:


	- возможностью задания смещения для каждого интервала колебания
	 относительно центра столбика;


	- возможностью увеличения количества используемых рядов;


	- возможностью отобразить дополнительные ряды по дополнительной
	 оси;


Примечание. Первые четыре ряда строятся по основной
 оси.


	- использованием величины того же порядка, что и остальные ряды, вместо
	 объема.


Примечание.
 Исходные данные должны быть положительными числами.


Для создания гистограммы с интервалами колебаний необходимо правильно
 упорядочить выводимые данные. В качестве исходных данных должны быть заданы
 четыре ряда:


	- 1-ый ряд. Значение показателя,
	 относительно которого происходит колебание;


	- 2-ой ряд. Минимальное
	 значение показателя;


	- 3-ий ряд. Максимальное
	 значение показателя;


	- 4-ый ряд. Текущее значения
	 показателя.

 исходные данные «Реальная среднемесячная
