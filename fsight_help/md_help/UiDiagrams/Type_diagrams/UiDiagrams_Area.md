# Диаграмма с областями


Диаграмма с областями позволяет
 изобразить величину изменений в зависимости от времени и могут использоваться
 для привлечения внимания к суммарному значению в соответствии с трендом.


На диаграмме с областями для каждой группы данных по вертикали откладываются
 отрезки, пропорциональные значениям всех элементов данных. Точки, соответствующие
 одному ряду разных групп, соединяются ломаной линией. Область под ломаной
 отображается с заливкой, установленной для этого элемента данных.
