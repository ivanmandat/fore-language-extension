# Кольцевая диаграмма


Кольцевая диаграмма демонстрирует
 размер элементов одного ряда данных пропорционально сумме элементов.


Каждая группа данных отображается в виде кольца, разделенного на секторы.
 Первый ряд данных отображается в центре диаграммы. Если в подписях данных
 отображаются проценты, то каждое кольцо в сумме должно давать 100 %.


Пример отображения кольцевой диаграммы:


![](Type%20pics/doughnut_chart.gif)


В режиме редактирования диаграммы возможно изменение положения крайних
 секторов. Для этого захватите сектор и перетащите его на необходимое расстояние,
 используя механизм Drag&Drop. При этом размер всей диаграммы уменьшится:


![](Type%20pics/doughnut_chart_sample.gif)

	- [Формат
