# Типы диаграмм


Диаграммы используются для наглядного представления табличных данных.
 Каждый ряд данных на диаграмме обозначается отдельным цветом, указанным
 в [легенде](../Property_diagramm/UiDiagrams_PropertyDiagramm_legend.htm),
 или способом представления. В отчетах могут использоваться следующие типы
 диаграмм:


	- [линейная диаграмма](UiDiagrams_lenear.htm);


	- [гистограмма](UiDiagrams_Bar.htm);


	- [объемная гистограмма с
	 группами](3DHistogram_with_Groups.htm);


	- [диаграмма с областями](UiDiagrams_Area.htm);


	- [смешанная диаграмма](UiDiagrams_mixed_diagram.htm);


	- [точечная диаграмма](UiDiagrams_scatter.htm);


	- [круговая диаграмма](UiDiagrams_round.htm);


	- [вторичная гистограмма](UiDiagrams_secondary_histogram.htm);


	- [вторичная круговая
	 диаграмма](UiDiagrams_round_secondary.htm);


	- [кольцевая диаграмма](Doughnut_chart.htm);


	- [лепестковая диаграмма](UiDiagrams_petalous.htm);


	- [биржевая диаграмма](UiDiagrams_birg.htm);


	- [гистограмма с интервалами
	 колебаний](UiDiagrams_BarVibration.htm);


	- [коробчатая диаграмма](Box_chart.htm);


	- [пузырьковая диаграмма](Bubble_Chart.htm);


	- [ступенчатая диаграмма](Stepped_Chart.htm);


	- [каскадная диаграмма](WaterFall_chart.htm).


При работе с диаграммами в веб-приложении или [конструкторе
 бизнес-приложений](Constructor.chm::/Intro/Designer_business_applications.htm) доступно [подключение
 пользовательских шаблонов диаграмм](DataAnalysis.chm::/Plugins/Custom/CustomDiagram.htm).


Для настройки внешнего вида диаграммы доступно:


	- изменение [типа диаграммы](ChangingChartType.htm);


	- настройка [объемного
	 вида](../Property_diagramm/UiDiagrams_PropertyDiagramm_Volume.htm) диаграммы;


	- создание многорядных и [многоуровневых](../Tuning_format/Format_Axis/UiDiagrams_FormatAxis_Levels.htm)
	 диаграмм;


	- настройка [ориентации
	 шкалы](../UiDiagrams_OrietationScale.htm);


	- настройка расположения [осей](../Params_diagram/UiDiagrams_Axis.htm)
	 диаграммы и другие возможности.
