# Точечная диаграмма


Точечная диаграмма используется
 для демонстрации и сравнения числовых значений: научных, статистических
 или технических данных.


На точечной диаграмме значение каждого элемента измерения отображается
 в виде точки на декартовой плоскости. Для каждого измерения отображается
 свой набор точек, который обозначается отдельным цветом, указанным в [легенде](../Property_diagramm/UiDiagrams_PropertyDiagramm_legend.htm).


Количество измерений, отображаемых на точечной диаграмме, два и более.
