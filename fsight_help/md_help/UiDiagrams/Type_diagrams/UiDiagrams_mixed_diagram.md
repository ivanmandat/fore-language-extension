# Смешанная диаграмма


Смешанная диаграмма позволяет
 для каждой группы данных определить свой тип диаграммы: столбики, линии
 или области. Например, можно объединить график, содержащий данные о ценах,
 с гистограммой, отображающей объемы продаж.


Более подробную информацию о настройке типа ряда данных смотрите в разделе
 «[Ряд данных](../Series.htm)».

	 области построения](../Tuning_format/uidiagrams_tuning_construction_area_digramm.htm);
