# Настройка линии уровня


Для настройки линии уровня используйте вкладку «Уровни»
 окна «Формат оси».
 Настройка линий уровня с помощью окна «Формат
 оси» доступна только в настольном приложении. Для [лепестковой
 диаграммы](../../Type_diagrams/UiDiagrams_petalous.htm) настройка линий уровня с помощью окна «Формат оси» доступна только для
 оси категорий (X).


Совет. Для быстрой
 настройки линии уровня используйте вкладку «Линии
 уровня» боковой панели.


[![](../../opened.gif)![](../../closed.gif)Для вызова окна «Формат
 оси**»](javascript:TextPopup(this))


		- выделите [ось
		 диаграммы](../../UiDiagrams_basic_concept.htm#select_component) и выполните команду «Формат
		 оси» в контекстном меню оси диаграммы;


	Примечание.
	 В инструменте «Аналитические панели»
	 выполните команду «Диаграмма >
	 Формат оси» в контекстном меню оси.


		- дважды щёлкните кнопкой мыши по заголовку выбранной оси.


	Для настройки параметров осей диаграммы в регламентном отчёте:


		- Выделите [ось
		 диаграммы](../../UiDiagrams_basic_concept.htm#select_component).


		- Выберите пункт «Ось
		 X» («Ось Y», «Дополнительная ось») в раскрывающемся
		 списке «Выбор элемента диаграммы
		 для форматирования», расположенном на вкладке «Диаграмма»
		 ленты инструментов.


		- Нажмите кнопку «Стиль
		 фрагмента».


	После выполнения действий будет открыто
	 окно «Формат
	 оси».


Вид вкладки «Уровни» зависит
 от выбранной оси. Например, при выборе оси Y вкладка выглядит следующим
 образом:


![](Pics%20format%20Axis/Axis%20format_6.gif)


На вкладке доступны следующие операции
 с линиями уровня:


[![](../../opened.gif)![](../../closed.gif)Добавление/удаление
 линии уровня](javascript:TextPopup(this))


	Для добавления линии уровня на редактируемую ось значений/категорий
	 нажмите кнопку «Добавить»,
	 после чего в области «Линии уровня»
	 появится новый уровень, имеющий значение предыдущего, которое
	 можно изменить с помощью редактора чисел «Отметка».


	Примечание.
	 При добавлении новой линии уровня автоматически устанавливаются следующие
	 первоначальные параметры: «Отметка»:
	 0,000000, «Тип линии»: сплошная,
	 «Толщина линии»: 1/4 пт.


	Для удаления выбранной линии уровня нажмите кнопку «Удалить».


[![](../../opened.gif)![](../../closed.gif)Установка
 значений линии уровня](javascript:TextPopup(this))


	Для установки значения выбранной линии уровня, перпендикулярной
	 оси Х:


		- для отображения линии уровня из деления, расположенного
		 в категории, укажите категорию в списке «Категория»;


		- для выбора произвольной точки на шкале, из которой будет
		 построена линия уровня, укажите точку из единичного отрезка [0, 1]
		 в редакторе чисел «Отметка».


	Примечание.
	 Шкала оси Х берется в качестве единичного отрезка [0, 1].


	Для установки значения выбранной линии уровня, перпендикулярной
	 оси Y, установите один из переключателей:


		- Константа. Задайте
		 значение линии уровня с помощью клавиатуры или редактора чисел;


		- Функция. В раскрывающемся
		 списке выберите функцию, исходя из значения которой будет строиться
		 линия уровня. Доступны следующие виды функций: «Максимум»,
		 «Минимум», «Сумма»,
		 «Медиана» или «Среднее».


[![](../../opened.gif)![](../../closed.gif)Настройка
 формата линии уровня и заливки интервала](javascript:TextPopup(this))


	Для настройки формата линии уровня выполните последовательность
	 действий в группе параметров «Линия»:


		- В раскрывающемся списке «Тип» выберите [тип
		 линии](../UiDiagrams_tuning_border_and_filling.htm#borders).


		- В раскрывающемся списке «Толщина»
		 задайте толщину линии. При выборе пункта «Пользовательский»
		 будет открыто окно «Толщина линии».
		 Задайте в нем требуемую толщину линии вручную или с помощью редактора
		 чисел.


Примечание.
 Доступен ввод значения [в разных единицах измерения](UiNav.Chm::/GUI/format/different_units.htm), однако значение в строке
 ввода будет всегда отображаться в единицах point (1/72 дюйма).


		- В раскрывающейся палитре выберите цвет линии либо выберите
		 новый цвет в расширенной палитре цветов, вызываемой при нажатии
		 на кнопку «Другой».


		- В редакторе «Прозрачность»
		 задайте процент [прозрачности](../UiDiagrams_tuning_border_and_filling.htm#transparency)
		 линии уровня.


	Для настройки заливки интервала между линиями уровня выполните последовательность
	 действий в группе «Заливка интервала»:


		- Установите флажок «Заливка
		 интервала».


		- В раскрывающейся палитре выберите цвет, которым будет заливаться
		 область до следующей по порядку линии уровня.


		- В редакторе «Прозрачность»
		 задайте процент [прозрачности](../UiDiagrams_tuning_border_and_filling.htm#transparency)
		 области заливки.


[![](../../opened.gif)![](../../closed.gif)Настройка
 отображаемой части линии уровня](javascript:TextPopup(this))


	При работе с диаграммой доступна настройка отображаемой части добавленной
	 линии уровня.


	Для настройки отображаемой части выбранной линии уровня в группе
	 параметров «Ограничить высоту»/«Ограничить длину» установите флажки
	 «От» и/или «До».
	 После этого в соответствующих редакторах укажите значения крайних
	 точек линии уровня. Указываемые значения соответствуют значениям на
	 противоположной оси, параллельно которой будет строиться линия уровня.
	 Если противоположная ось отражает не числовые значения, то значения
	 для линии уровня указываются в диапазоне [0, 1], что соответствует
	 0-100% от размера оси.


	Для линии уровня, перпендикулярной оси категорий (X), доступно
	 построение линии уровня по дополнительной оси. Для этого установите
	 одноимённый флажок.


	Примечание.
	 Построение линии уровня по дополнительной оси доступно, если для какого-либо
	 из рядов данных диаграммы на вкладке «Ряды
	 данных» боковой панели был установлен флажок «[По дополнительной оси](../../Series.htm#secondary_axis)».


	После задания ограничения отображаемой части линии уровня при необходимости
	 настройте [заливку интервала](#format_level_line).


[![](../../opened.gif)![](../../closed.gif)Настройка
 подписи линии уровня](javascript:TextPopup(this))


	Для отображения подписи линии уровня установите флажок «Подпись».


	Для смены или добавления шаблона в текст подписи:


		- Нажмите кнопку «Добавить»,
		 расположенную рядом с многострочным полем ввода.


		- В раскрывающемся списке выберите шаблон текста подписи линии
		 уровня:


			- %Computation.
			 В качестве подписи линии уровня будет использоваться шаблон
			 «Функция».

			В зависимости от [выбранного
			 типа подписи](#set_level_line_value) в области построения диаграммы подпись линии
			 уровня будет отображаться в виде:


				- Константа;


				- <Наименование
				 функции>;


			- %Value. В качестве
			 подписи линии уровня будет использоваться шаблон «Значение».


	При необходимости задайте настройки шрифта. Для этого нажмите кнопку
	 «Шрифт» и в открывшемся
	 окне определите [параметры](../Format_Title/UiDiagrams_FormatTitle_type.htm).


[![](../../opened.gif)![](../../closed.gif)Настройка
 перекрытия рядов данных](javascript:TextPopup(this))


	Для настройки отображения линии уровня поверх рядов данных установите
	 флажок «С перекрытием рядов данных».
	 При снятом флажке линия уровня будет находиться под рядами данных.
	 По умолчанию флажок установлен.


	Ниже приведен пример диаграммы с различными настройками линий уровня:


	![](Pics%20format%20Axis/Level_Example.gif)


	Линия уровня, установленная на отметке «45000», находится под рядами
	 данных. Линия уровня, установленная на отметке «40000» - перекрывает
	 ряды данных.
