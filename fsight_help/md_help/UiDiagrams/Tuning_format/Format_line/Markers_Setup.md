# Настройка маркеров


Маркер - фигура, отображающая
 отдельные точки ряда на диаграмме. Маркеры, соответствующие одному ряду
 данных, имеют одинаковые настройки оформления.


Для настройки маркеров перейдите на вкладку «Маркеры»
 окна «Формат ряда данных».


Настройка доступна только для рядов [линейной](../../Type_diagrams/UiDiagrams_lenear.htm),
 [точечной](../../Type_diagrams/UiDiagrams_scatter.htm), [лепестковой](../../Type_diagrams/UiDiagrams_petalous.htm),
 [биржевой](../../Type_diagrams/UiDiagrams_birg.htm), [коробчатой](../../Type_diagrams/Box_chart.htm)
 диаграмм, [гистограммы
 с интервалами колебаний](../../Type_diagrams/UiDiagrams_BarVibration.htm), для ряда типа «Линия»
 в [смешанной](../../Type_diagrams/UiDiagrams_mixed_diagram.htm)
 диаграмме и для ряда типа «Ступенчатый
 график» в [ступенчатой](../../Type_diagrams/Stepped_Chart.htm)
 диаграмме.


Примечание.
 Настройка маркеров с помощью окна «Формат
 ряда данных» доступна только в настольном приложении.


Совет. Для быстрой
 настройки формата ряда используйте вкладку «[Ряды данных](../../Series.htm)» боковой панели.


[![](../../opened.gif)![](../../closed.gif)Для вызова окна
 «Формат ряда данных»](javascript:TextPopup(this))


		- Выполните команду «Формат
		 ряда данных» в контекстном меню выбранного ряда.


	Примечание.
	 В инструменте «Аналитические панели»
	 выполните команду «Диаграмма >
	 Формат ряда данных» в контекстном меню ряда.


		- Дважды щелкните кнопкой мыши по ряду.


	Для настройки формата ряда диаграммы в
	 регламентном
	 отчете выделите [ряд
	 данных](../../UiDiagrams_basic_concept.htm#select_component). В раскрывающемся списке
	 «Выбор элемента
	 диаграммы для форматирования»,
	 расположенном на вкладке «Диаграмма» ленты инструментов, выберите пункт,
	 соответствующий наименованию ряда данных, и
	 нажмите кнопку «Стиль
	 фрагмента». Откроется окно «Формат ряда данных».


![](format%20row%202_0.gif)


Задайте следующие настройки маркеров данных:


[![](../../opened.gif)![](../../closed.gif)Настройка
 отображения маркеров](javascript:TextPopup(this))


	Для отображения маркеров на ряде данных установите флажок «Отображать маркеры».


[![](../../opened.gif)![](../../closed.gif)Настройка
 типа маркера](javascript:TextPopup(this))


	В раскрывающемся списке выберите тип маркера:


		- Квадрат. Маркеры
		 отображаются в виде квадратов;


		- Ромб. Маркеры отображаются
		 в виде ромбов;


		- Треугольник. Маркеры
		 отображаются в виде треугольников с вершинами, направленными вверх;


		- Обратный треугольник.
		 Маркеры отображаются в виде треугольников с вершинами, направленными
		 вниз;


		- Круг. Маркеры отображаются
		 в виде кругов;


		- Линия. Маркеры отображаются
		 в виде штриха.


	Ниже приведён пример отображения разного типа маркеров:


	![](format%20row%201_1.gif)


	Совет. Для быстрой
	 смены отображения или типа маркера используйте кнопку «Маркеры»,
	 расположенную в группе «Вид»
	 на вкладке «Диаграмма» ленты
	 инструментов.


[![](../../opened.gif)![](../../closed.gif)Настройка
 размера маркеров](javascript:TextPopup(this))


	Задайте размер маркеров с помощью редактора чисел «Размер
	 маркера». По умолчанию размер маркера равен нулю и маркер не
	 отображается.


Примечание.
 Доступен ввод значения [в
 разных единицах измерения](UiNav.Chm::/GUI/format/different_units.htm). В поле ввода редактора отображаются значения
 в единицах измерения, соответствующих текущим региональным настройкам
 операционной системы.


[![](../../opened.gif)![](../../closed.gif)Настройка
 заливки маркеров](javascript:TextPopup(this))


	Для настройки заливки маркеров задайте параметры в группе «Заливка»:


		- Тип заливки. В раскрывающемся
		 списке выберите тип заливки, который будет использоваться для
		 заливки маркеров. Для заливки цветом выберите пункт «Заливка»,
		 для использования штриховки выберите пункт «Штриховка»;


		- Цвет. В раскрывающейся
		 палитре выберите цвет заливки фона маркера либо выберите новый
		 цвет в расширенной палитре цветов, вызываемой при нажатии на кнопку
		 «Другой»;


		- Прозрачность. Определите
		 [прозрачность](../UiDiagrams_tuning_border_and_filling.htm#transparency)
		 фона маркера. 100% соответствует полной прозрачности (фон маркера
		 станет невидимым), 0% соответствует полной непрозрачности. Значение
		 задается с помощью редактора чисел;


		- Штриховка. Группа
		 параметров доступна, если в качестве заливки маркера выбран тип
		 «Штриховка». Определите
		 следующие параметры штриховки:


			- Тип. В раскрывающемся
			 списке выберите тип штриховки маркера;


			- Цвет. В раскрывающейся
			 палитре выберите цвет штриховки фона маркера либо выберите
			 новый цвет в расширенной палитре цветов, вызываемой при нажатии
			 на кнопку «Другой».


[![](../../opened.gif)![](../../closed.gif)Настройка
 границы маркеров](javascript:TextPopup(this))


	Для настройки границ маркеров задайте параметры в группе «Границы»:


		- Тип линии. В раскрывающемся
		 списке выберите тип линии;


		- Толщина линии. В
		 раскрывающемся списке выберите необходимую толщину линии. При
		 выборе пункта «Пользовательский»
		 будет открыто окно «Толщина линии»,
		 в котором с помощью редактора чисел или вручную задайте требуемую
		 толщину линий;


Примечание.
 Доступен ввод значения [в разных единицах измерения](UiNav.Chm::/GUI/format/different_units.htm), однако значение в строке
 ввода будет всегда отображаться в единицах point (1/72 дюйма).


		- Цвет линии. В раскрывающейся
		 палитре выберите цвет линии границы маркера либо выберите новый
		 цвет в расширенной палитре цветов, вызываемой при нажатии на кнопку
		 «Другой»


[![](../../opened.gif)![](../../closed.gif)Настройка
 тени маркеров](javascript:TextPopup(this))


	Для отображения тени маркера установите флажок «Тень».


Далее настройте смещение тени по оси категорий (Х) и смещение тени по
 оси значений (Y) с помощью числовых редакторов «dX»
 и «dY» соответственно. В раскрывающейся
 палитре выберите цвет тени либо выберите новый цвет в расширенной
 палитре цветов, вызываемой при нажатии на кнопку «Другой».


После задания настроек в области «Образец»
 будет отображен маркер, оформленный согласно параметрам, установленным
 на вкладке. Проверьте, соответствует ли внешний вид маркера ожидаемому
 результату.


## Пример отображения маркеров


Ниже приведен пример отображения маркеров. На диаграмме слева у рядов
 данных маркеры скрыты, на диаграмме справа маркеры настроены следующим
 образом:


	- для маркеров ряда «2001»
	 установлен тип маркера «Круг»,
	 задана заливка маркера и тень маркера (значения смещений dX
	 и dY - «1,00 мм» и «0,50
	 мм» соответственно);


	- для маркеров ряда «2002»
	 установлен тип маркера «Треугольник»,
	 задана граница маркера и изменен его размер.
