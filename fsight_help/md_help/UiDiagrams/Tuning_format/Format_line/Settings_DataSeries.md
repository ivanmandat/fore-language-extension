# Общие настройки ряда данных


Для настройки общих параметров [ряда данных](../../Series.htm)
 перейдите на вкладку «Общее»
 окна «Формат ряда данных».


Примечание.
 Общие настройки рядов данных с помощью окна «Формат
 ряда данных» доступны только в настольном приложении.


Совет. Для быстрой
 настройки формата ряда используйте вкладку «[Ряды данных](../../Series.htm)» боковой панели.


[![](../../opened.gif)![](../../closed.gif)Для вызова окна
 «Формат ряда данных»](javascript:TextPopup(this))


		- Выполните команду «Формат
		 ряда данных» в контекстном меню выбранного ряда.


	Примечание.
	 В инструменте «Аналитические панели»
	 выполните команду «Диаграмма >
	 Формат ряда данных» в контекстном меню ряда.


		- Дважды щелкните кнопкой мыши по ряду.


	Для настройки формата ряда диаграммы в
	 регламентном
	 отчете выделите [ряд
	 данных](../../UiDiagrams_basic_concept.htm#select_component). В раскрывающемся списке
	 «Выбор элемента
	 диаграммы для форматирования»,
	 расположенном на вкладке «Диаграмма» ленты инструментов, выберите пункт,
	 соответствующий наименованию ряда данных, и
	 нажмите кнопку «Стиль
	 фрагмента». Откроется окно «Формат ряда данных».


![](format%20row%203_0.gif)


Задайте следующие настройки ряда данных:


[![](../../opened.gif)![](../../closed.gif)Сглаживание
 линейных рядов](javascript:TextPopup(this))


	Сглаживание ряда используется
	 для устранения эффекта ломаной линии, возникающего при построении
	 диаграммы ([линейной](../../Type_diagrams/UiDiagrams_lenear.htm),
	 [точечной](../../Type_diagrams/UiDiagrams_scatter.htm)
	 с соединительными линиями и для ряда типа «Линия»
	 в [смешанной](../../Type_diagrams/UiDiagrams_mixed_diagram.htm)
	 диаграмме).


	Для настройки сглаживания линейных рядов установите флажок:


		- Сглаживать ряд.
		 Флажок располагается на вкладке «Общее»
		 окна «Формат ряда данных».
		 При установленном флажке сглаживание применяется только к выбранному
		 ряду данных диаграммы;


		- Сглаживать ряды.
		 Флажок располагается в окне «[Типы
		 диаграмм](../../Type_diagrams/ChangingChartType.htm)». При установленном флажке сглаживание применяется
		 для всех рядов выбранной диаграммы.


	Ниже приведен пример без использования (слева) и с использованием
	 (справа) сглаживания для ряда линейного типа смешанной диаграммы:


	![](Type%20Line%20example.gif)


[![](../../opened.gif)![](../../closed.gif)Отображение
 в легенде](javascript:TextPopup(this))


	Для отображения ряда в легенде установите флажок «Отображать
	 в легенде». По умолчанию флажок установлен, и ряд отображается
	 в легенде.


[![](../../opened.gif)![](../../closed.gif)Отображение
 тени](javascript:TextPopup(this))


	Для отображения тени ряда установите флажок «Отображать
	 тень».


Далее настройте смещение тени по оси категорий (Х) и смещение тени по
 оси значений (Y) с помощью числовых редакторов «dX»
 и «dY» соответственно. В раскрывающейся
 палитре выберите цвет тени либо выберите новый цвет в расширенной
 палитре цветов, вызываемой при нажатии на кнопку «Другой».


При работе с рядами данными доступны операции:


[![](../../opened.gif)![](../../closed.gif)Использование
 дополнительной оси](javascript:TextPopup(this))


	Дополнительная ось - вспомогательная
	 ось значений, в которую могут быть помещены:


		- значения рядов данных, значительно отличающиеся от значений
		 других рядов;


		- значения рядов, представленные на диаграмме в другом типе
		 данных.


	Для использования дополнительной оси для ряда установите флажок
	 «Отношение к дополнительной оси».


	При расположении ряда по дополнительной оси в легенде можно будет
	 отобразить, по какой оси расположен тот или иной ряд. Для этого установите
	 флажок «Отображать шкалу на пиктограмме
	 ряда» на вкладке «[Общее](../Format_Legend/UiDiagrams_FormatLegend_common.htm#scale_on_legend)»
	 окна «Формат легенды».


[![](../../opened.gif)![](../../closed.gif)Использование
 редактирования](javascript:TextPopup(this))


	Для изменения данных в таблице и в источнике данных с помощью диаграммы
	 установите флажок «Разрешить редактирование».
	 По умолчанию флажок установлен, и доступно [редактирование
	 рядов данных](../../Edit_DataSeries.htm).
