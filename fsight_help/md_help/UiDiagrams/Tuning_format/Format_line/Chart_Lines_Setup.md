# Настройка линий графика


Для настройки линий графика перейдите на вкладку «Линии
 графика» окна «Формат
 ряда данных». Настройка доступна для [линейной](../../Type_diagrams/UiDiagrams_lenear.htm),
 [точечной](../../Type_diagrams/UiDiagrams_scatter.htm), [биржевой](../../Type_diagrams/UiDiagrams_birg.htm) диаграмм
 и для рядов типа «Линия» [смешанной](../../Type_diagrams/UiDiagrams_mixed_diagram.htm),
 [лепестковой](../../Type_diagrams/UiDiagrams_petalous.htm)
 и [ступенчатой](../../Type_diagrams/Stepped_Chart.htm) диаграмм.


Примечание.
 Настройка линий графика с помощью окна «Формат
 ряда данных» доступна только в настольном приложении.


Совет. Для быстрой
 настройки формата ряда используйте вкладку «[Ряды данных](../../Series.htm)» боковой панели.


[![](../../opened.gif)![](../../closed.gif)Для вызова окна
 «Формат ряда данных»](javascript:TextPopup(this))


		- Выполните команду «Формат
		 ряда данных» в контекстном меню выбранного ряда.


	Примечание.
	 В инструменте «Аналитические панели»
	 выполните команду «Диаграмма >
	 Формат ряда данных» в контекстном меню ряда.


		- Дважды щелкните кнопкой мыши по ряду.


	Для настройки формата ряда диаграммы в
	 регламентном
	 отчете выделите [ряд
	 данных](../../UiDiagrams_basic_concept.htm#select_component). В раскрывающемся списке
	 «Выбор элемента
	 диаграммы для форматирования»,
	 расположенном на вкладке «Диаграмма» ленты инструментов, выберите пункт,
	 соответствующий наименованию ряда данных, и
	 нажмите кнопку «Стиль
	 фрагмента». Откроется окно «Формат ряда данных».


![](format%20row%201_0_1.gif)


На вкладке доступны настройки оформления линии
 и пустой линии.


Пустая линия используется в
 случае отсутствия значений на графике. В этом случае два ближайших имеющихся
 значения на диаграмме будут соединены линией с заданными настройками.


Настройки линии и пустой линии совпадают:


[![](../../opened.gif)![](../../closed.gif)Настройка типа
 линии](javascript:TextPopup(this))


В раскрывающемся списке «Тип линии»
 выберите тип линии.


[![](../../opened.gif)![](../../closed.gif)Настройка толщины
 линии](javascript:TextPopup(this))


	В раскрывающемся списке «Толщина линии»
 выберите необходимую толщину линии. При выборе пункта «Пользовательский»
 будет открыто окно «Толщина линии»,
 в котором можно с помощью редактора чисел или вручную задать требуемую
 толщину линий.


Примечание.
 Доступен ввод значения [в разных единицах измерения](UiNav.Chm::/GUI/format/different_units.htm), однако значение в строке
 ввода будет всегда отображаться в единицах point (1/72 дюйма).


[![](../../opened.gif)![](../../closed.gif)Настройка цвета
 линии](javascript:TextPopup(this))


	В раскрывающейся палитре выберите цвет линии либо выберите новый цвет в
 расширенной палитре цветов, вызываемой при нажатии на кнопку «Другой».


	При необходимости настройте [прозрачность](../UiDiagrams_tuning_border_and_filling.htm#transparency)
	 линии.
