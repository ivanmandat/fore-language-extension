# Настройка типа ряда


Для настройки типа ряда данных перейдите на вкладку «Тип
 ряда» окна «Формат
 ряда данных». Настройка доступна только для [смешанной](../../Type_diagrams/UiDiagrams_mixed_diagram.htm),
 [лепестковой](../../Type_diagrams/UiDiagrams_petalous.htm)
 и [ступенчатой](../../Type_diagrams/Stepped_Chart.htm) диаграмм.


Примечание.
 Настройка типа ряда данных с помощью окна «Формат
 ряда данных» доступна только в настольном приложении.


Совет. Для быстрой
 настройки формата ряда используйте вкладку «[Ряды данных](../../Series.htm)» боковой панели.


[![](../../opened.gif)![](../../closed.gif)Для вызова окна
 «Формат ряда данных»](javascript:TextPopup(this))


		- Выполните команду «Формат
		 ряда данных» в контекстном меню выбранного ряда.


	Примечание.
	 В инструменте «Аналитические панели»
	 выполните команду «Диаграмма >
	 Формат ряда данных» в контекстном меню ряда.


		- Дважды щелкните кнопкой мыши по ряду.


	Для настройки формата ряда диаграммы в
	 регламентном
	 отчете выделите [ряд
	 данных](../../UiDiagrams_basic_concept.htm#select_component). В раскрывающемся списке
	 «Выбор элемента
	 диаграммы для форматирования»,
	 расположенном на вкладке «Диаграмма» ленты инструментов, выберите пункт,
	 соответствующий наименованию ряда данных, и
	 нажмите кнопку «Стиль
	 фрагмента». Откроется окно «Формат ряда данных».


![](Type%20Line.gif)


Примечание.
 Для ряда [лепестковой
 ](../../Type_diagrams/UiDiagrams_petalous.htm)и [ступенчатой](../../Type_diagrams/Stepped_Chart.htm)
 диаграммы вкладка содержит только переключатели «Линейный
 тип ряда» и «Тип ряда с областями». Данные
 типы рядов отображают абсолютные значения.


Выберите тип ряда из доступных вариантов:


	- ![](Type%20Line%20line.gif). Линейный тип ряда. Отображает
	 абсолютные значения ряда;


	- ![](Type%20Line%20line_2.gif). Линейный тип ряда с
	 накоплением. Отображает значения с накоплением. Позволяет оценить
	 вклад каждого значения в общую сумму;


	- ![](Type%20Line%20bar.gif). Столбиковый тип ряда. Отображает
	 абсолютные значения ряда;


	- ![](Type%20Line%20bar_2.gif). Столбиковый тип
	 ряда с накоплением. Отображает значения с накоплением. Позволяет
	 оценить вклад каждого значения в общую сумму;


	- ![](Type%20Line%20area.gif). Тип ряда с областями. Отображает
	 абсолютные значения ряда;


	- ![](Type%20Line%20area_2.gif). Тип ряда с областями с
	 накоплением. Отображает значения с накоплением. Позволяет оценить
	 вклад каждого значения в общую сумму.


Для [смешанных
 диаграмм](../../Type_diagrams/UiDiagrams_mixed_diagram.htm) с накоплением существует возможность отображения отдельных
 типов рядов, привязанных к одной оси, как с накоплением, так и без. Это
 позволяет установить/снять накопление со всех рядов одного типа, привязанных
 к одной из осей.


Для [ступенчатых диаграмм](../../Type_diagrams/Stepped_Chart.htm)
 существует возможность настроить положение ступеньки относительно точки
 ряда данных. Для этого в раскрывающемся списке «Ступенька»
 выберите необходимый вариант расположения: слева, по центру или справа.
