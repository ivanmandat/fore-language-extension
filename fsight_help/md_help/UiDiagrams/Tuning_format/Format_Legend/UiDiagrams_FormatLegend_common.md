# Общие настройки оформления легенды


Для настройки общих параметров легенды перейдите на вкладку «Общее»
 окна «Формат легенды».


Совет. Для быстрой
 настройки формата легенды используйте вкладку «[Легенда](../../Property_diagramm/UiDiagrams_PropertyDiagramm_legend.htm)»
 боковой панели.


[![](../../opened.gif)![](../../closed.gif)Для вызова окна
 «Формат легенды»](javascript:TextPopup(this))


		- Выполните команду «Формат
		 легенды» в контекстном меню легенды.


	Примечание.
	 В инструменте «Аналитические панели»
	 выполните команду «Диаграмма >
	 Формат легенды» в контекстном меню легенды.


		- Дважды щелкните кнопкой мыши по легенде.


	Для настройки формата легенды в регламентном
	 отчете [выделите легенду](../../UiDiagrams_basic_concept.htm#select_component). Затем нажмите кнопку
	 «Стиль фрагмента», расположенную
	 в группе «Текущий фрагмент»
	 на вкладке «Диаграмма» ленты
	 инструментов.


![](format_legend_3.png)


На вкладке доступны следующие операции:


[![](../../opened.gif)![](../../closed.gif)Настройка
 ширины и высоты значков легенды](javascript:TextPopup(this))


	Для настройки ширины и высоты значков легенды задайте значение в
	 редакторах чисел «Ширина значка»
	 и «Высота значка» в группе
	 «Значки рядов».


[![](../../opened.gif)![](../../closed.gif)Отображение
 тени](javascript:TextPopup(this))


	Для отображения тени легенды установите флажок «Отображать
	 тень».


Далее настройте смещение тени по оси категорий (Х) и смещение тени по
 оси значений (Y) с помощью числовых редакторов «dX»
 и «dY» соответственно. В раскрывающейся
 палитре выберите цвет тени либо выберите новый цвет в расширенной
 палитре цветов, вызываемой при нажатии на кнопку «Другой».


	При необходимости настройте [прозрачность](../UiDiagrams_tuning_border_and_filling.htm#transparency)
	 тени.


[![](../../opened.gif)![](../../closed.gif)Использование
 оформления рядов](javascript:TextPopup(this))


	Для синхронизации толщины линий и размера маркеров значков рядов
	 в легенде с оформлением рядов в области построения установите флажок
	 «Использовать оформление рядов»
	 в группе «Значки рядов».


	Примечание.
	 Параметр «Использовать оформление
	 рядов» доступен только для легенд диаграмм, содержащих [точечные](../../Type_diagrams/UiDiagrams_scatter.htm)
	 и [линейные](../../Type_diagrams/UiDiagrams_lenear.htm)
	 ряды.


[![](../../opened.gif)![](../../closed.gif)Отображение
 шкалы на пиктограмме ряда](javascript:TextPopup(this))


	Настройка доступна, если хотя бы один ряд данных расположен по дополнительной
	 оси.


	Для отображения у каждого ряда оси, по которой расположен ряд, установите
	 флажок «Отображать шкалу на пиктограмме
	 ряда». После установки флажка в легенде будут отображаться
	 значки осей:


	![](format%20legend%203%20example.gif)


	По умолчанию флажок снят и значки осей не отображаются в легенде.


	Примечание.
	 Расположение ряда по дополнительной оси определяется в окне «[Формат
	 ряда данных](../Format_line/Settings_DataSeries.htm#other_axis)».
