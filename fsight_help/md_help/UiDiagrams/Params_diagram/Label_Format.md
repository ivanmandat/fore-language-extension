# Настройка формата подписей


Настройки формата текста подписи данных задаются в окне «Формат подписей».


Примечание.
 Для [линии
 уровня](../Tuning_format/Format_Axis/UiDiagrams_FormatAxis_Levels.htm) не доступна настройка формата текста подписи данных с помощью
 окна «Формат подписей».


Для вызова окна «Формат
 подписей» нажмите кнопку «Формат»
 в окнах:


	- «Параметры диаграммы»
	 на вкладке «[Подписи
	 данных](UiDiagrams__DataTitle.htm)»;


	- «Формат ряда данных»
	 на вкладке «[Подписи данных](../Property_diagramm/UiDiagrams_PropertyDiagramm_Label.htm)».


	- «Формат точки ряда данных»
	 на вкладке «[Подписи
	 данных](UiDiagrams__DataTitle.htm)»;


Примечание.
 Настройка формата подписей данных с помощью окон «Параметры
 диаграммы», «Формат ряда данных»
 и «Формат подписей» доступны только
 в настольном приложении.


Из окна «Формат подписей» доступны
 следующие настройки подписей данных:


	- [Настройка
	 границы и заливки](../Tuning_format/UiDiagrams_tuning_border_and_filling.htm);


	- [Настройка
	 шрифта](../Tuning_format/Format_Title/UiDiagrams_FormatTitle_type.htm);


	- [Настройка
	 формата числа](UiNav.Chm::/GUI/format/uireport_table_attribute_format.htm).
