# Настройка порядка рядов


Для настройки порядка отображения рядов данных на диаграмме перейдите
 на вкладку «Порядок рядов»
 окна «[Параметры
 диаграммы](UiDiagrams_params_diagram.htm)».


Примечание.
 Настройка порядка рядов данных с помощью окна «Параметры
 диаграммы» доступна только в настольном приложении при снятом флажке
 «[Привязать
 к источнику](UiDiagrams_params_diagram.htm#source_binding)».


Порядок следования рядов устанавливает порядок их отображения в [легенде](../Property_diagramm/UiDiagrams_PropertyDiagramm_legend.htm)
 при настроенном [порядке рядов при перекрытии](#series_overlap_order).


Совет. Для быстрой
 настройки формата ряда используйте вкладку «[Ряды данных](../Series.htm)» боковой панели.


[![](../opened.gif)![](../closed.gif)Для
 вызова окна «Параметры диаграммы»](javascript:TextPopup(this))


	Выполните команду «Параметры
	 диаграммы» в контекстном меню [выделенной
	 диаграммы](../UiDiagrams_basic_concept.htm#select_component).


	Примечание.
	 В инструменте «Аналитические панели»
	 выполните команду «Диаграмма >
	 Параметры диаграммы» в контекстном меню диаграммы.


![](Pics_param/Order%20Series2.gif)


При работе с рядами данными доступны следующие операции:


[![](../opened.gif)![](../closed.gif)Выбор
 порядка рядов при перекрытии](javascript:TextPopup(this))


	В раскрывающемся списке выберите вариант отображения рядов на диаграмме:


		- Прямой порядок.
		 Ряды будут располагаться в том же порядке следования, что и в
		 таблице данных;


		- Обратный порядок.
		 Ряды будут располагаться в обратном порядке от их следования в
		 таблице данных;


		- По среднему арифметическому.
		 Для каждого ряда будет вычислено среднее арифметическое. Ряды
		 будут отсортированы в прямом порядке от полученных значений;


		- Обратный по среднему арифметическому.
		 Для каждого ряда будет вычислено среднее арифметическое. Ряды
		 будут отсортированы в обратном порядке от полученных значений.


	Примечание.
	 Для получения среднего арифметического ряда складываются по модулю
	 все значения точек ряда


		- Обратный, как в Excel.
		 Ряды будут отображаться в обратном порядке от их расположения
		 в таблице данных. При этом ряды будут отсортированы группами по
		 типу: сначала линии, за ними столбцы, за ними области. При установке
		 данного порядка для диаграммы с областями при экспорте в Excel
		 ряды будут отображаются в обратном порядке от их расположения
		 в таблице данных.


	Выбранный порядок рядов данных отобразится в списке «Ряды»,
	 расположенном на вкладке «Порядок
	 рядов» окна «Параметры диаграммы».
	 [Легенда
	 диаграммы](../Property_diagramm/UiDiagrams_PropertyDiagramm_legend.htm) будет перестроена в соответствии с выбранным порядком
	 рядов.


[![](../opened.gif)![](../closed.gif)Настройка
 произвольного порядка рядов](javascript:TextPopup(this))


	Для настройки произвольного порядка рядов используйте список «Ряды», содержащий наименования имеющихся
	 на диаграмме рядов данных. Заданный порядок следования рядов устанавливает
	 порядок их отображения в [легенде](../Property_diagramm/UiDiagrams_PropertyDiagramm_legend.htm)
	 с учётом настроенного [порядка рядов
	 при перекрытии](#series_overlap_order).


	Примечание.
	 Настройка доступна для всех типов диаграмм, кроме типа «[Гистограмма с
	 интервалами колебаний](../Type_diagrams/UiDiagrams_BarVibration.htm)».


	Для изменения положения выбранного в списке ряда:


		- перетащите ряд в необходимую позицию, используя механизм
		 Drag&Drop;


		- нажмите соответствующую кнопку:


			- ![](Pics_param/Button%20OrderS%201.gif) «Переместить выше». Перемещает
			 ряд на позицию выше;


			- ![](Pics_param/Button%20OrderS%202.gif) «Переместить ниже». Перемещает
			 ряд на позицию ниже.


[![](../opened.gif)![](../closed.gif)Сброс
 порядка рядов](javascript:TextPopup(this))


	Для сброса текущего порядка рядов и восстановления порядка рядов
	 по умолчанию нажмите кнопку «Сбросить
	 порядок». После сброса порядок будет соответствовать расположению
	 данных в таблице, на основе которой построена диаграмма.
