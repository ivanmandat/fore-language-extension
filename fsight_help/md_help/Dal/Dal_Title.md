## С чего начать


		 Сборка Dal
		 предназначена для работы с различными СУБД с помощью SQL-запросов.
		 Изучив данный раздел, вы сможете с применением языка Fore
		 создавать соединение с сервером БД, выполнять SQL-запросы в синхронном
		 и асинхронном режиме, работать с объектами БД и данными этих объектов.
		 Более подробно с возможностями блока можно ознакомиться в разделе
		 «[Введение в сборку Dal](Intro/Dal_Intro.htm)».


		Описание ресурсов сборки представлено в следующих подразделах:


			- [Интерфейсы сборки
			 Dal](Interface/Dal_Interface.htm)


			- [Перечисления сборки Dal](Enums/Dal_Enums.htm)


			- [Классы сборки Dal](Class/Dal_Class.htm)


		 Перед изучением сборки Dal рекомендуется ознакомиться
		 с архитектурой СУБД, с которой предстоит работа. Список СУБД,
		 на базе которых может быть построен репозиторий платформы или
		 осуществлено подключение к каким-либо источниками данных, представлен
		 в подразделе «[Поддерживаемые
		 СУБД](Setup.chm::/01_SysReq/database_Support.htm)». Также рекомендуется изучить синтаксис
		 SQL-команд, которые предполагается выполнять из прикладного кода,
		 их отличия при работе с различными СУБД. Данную информацию можно
		 получить в специализированной литературе по соответствующей СУБД.


## Что нового


			- Реализован драйвер для подключения и работы с СУБД SQLite.
			 Для работы с данной СУБД из Fore предназначен класс [DalSQLiteDriver](Class/DalSQLiteDriver/DalSQLiteDriver.htm).


			- Реализован драйвер для подключения и работы с СУБД PostgreSQL.
			 Для работы с данной СУБД из Fore предназначен класс [DalPostgresDriver](Class/DalPostgresDriver/DalPostgresDriver.htm).


			- Реализован драйвер для подключения и работы с СУБД Microsoft
			 SQL Server. Для работы с данной СУБД из Fore предназначен
			 класс [DalMsSql2012Driver](Class/DalMsSql2012Driver/DalMsSql2012Driver.htm).
