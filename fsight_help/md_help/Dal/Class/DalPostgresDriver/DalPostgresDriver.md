# DalPostgresDriver


Сборка: Dal;


## Описание


Класс DalPostgresDriver реализует драйвер, используемый для работы с СУБД PostgreSQL.


## Комментарии


Объект данного класса позволяет создать описание для подключения к СУБД PostgreSQL и дальнейшей работы помощью SQL-запросов.
