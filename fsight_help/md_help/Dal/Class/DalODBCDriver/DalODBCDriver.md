# DalODBCDriver


Сборка: Dal;


## Описание


Класс DalODBCDriver реализует
 ODBC драйвер, используемый для создания соединения с источниками данных.


## Комментарии


ODBC драйвер, который предполагается использовать, должен быть установлен
 в операционной системе.
