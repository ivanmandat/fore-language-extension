# DalSQLiteDriver


Сборка: Dal;


## Описание


Класс DalSQLiteDriver реализует
 драйвер, используемый для работы с СУБД SQLite.


## Комментарии


Объект данного класса позволяет создать описание для подключения к СУБД
 SQLite и дальнейшей работы помощью SQL-запросов.
