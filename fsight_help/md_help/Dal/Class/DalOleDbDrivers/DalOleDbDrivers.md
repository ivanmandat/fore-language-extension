# DalOleDbDrivers


## Описание


Класс DalOleDbDrivers реализует
 коллекцию драйверов OLE DB, используемых для создания соединения с сервером
 БД.


## Свойства объекта класса, унаследованные от [IDalOleDbDrivers](../../Interface/IDalOleDbDrivers/IDalOleDbDrivers.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](../../Interface/IDalOleDbDrivers/IDalOleDbDrivers.Count.htm)
		 Свойство Count возвращает
		 количество драйверов OLE DB, установленных в операционной системе.


		 ![](../../Property_Image.gif)
		 [Driver](../../Interface/IDalOleDbDrivers/IDalOleDbDrivers.Driver.htm)
		 Свойство Driver возвращает
		 настройки OLE DB драйвера с указанным индексом.


		 ![](../../Property_Image.gif)
		 [DriverName](../../Interface/IDalOleDbDrivers/IDalOleDbDrivers.DriverName.htm)
		 Свойство DriverName
		 возвращает наименование OLE DB драйвера с указанным индексом.
