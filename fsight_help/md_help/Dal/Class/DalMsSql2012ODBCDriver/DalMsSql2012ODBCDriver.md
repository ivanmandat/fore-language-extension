# DalMsSql2012ODBCDriver


Сборка: Dal;


## Описание


Класс DalMsSql2012ODBCDriver
 реализует ODBC драйвер, используемый для создания соединения с сервером
 БД (Microsoft SQL Server).


## Комментарии


ODBC драйвер для Microsoft SQL Server должен быть установлен в операционной
 системе.
