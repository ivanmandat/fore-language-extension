		 [DalMsSql2012Driver](DalMsSql2012Driver/DalMsSql2012Driver.htm)

		 БД Microsoft SQL Server.


		 ![](../Class_Image.gif)
		 [DalMsSql2012ODBCDriver](DalMsSql2012ODBCDriver/DalMsSql2012ODBCDriver.htm)
		 Класс DalMsSql2012ODBCDriver
		 реализует ODBC драйвер, используемый для создания соединения с
		 сервером БД (Microsoft SQL Server).


		 ![](../Class_Image.gif)
		 [DalODBCDriver](DalODBCDriver/DalODBCDriver.htm)
		 Класс DalODBCDriver
		 реализует ODBC драйвер, используемый для создания соединения с
		 источниками данных.


		 ![](../Class_Image.gif)
		 [DalOleDbDrivers](DalOleDbDrivers/DalOleDbDrivers.htm)
		 Класс DalOleDbDrivers
		 реализует коллекцию драйверов OLE DB, используемых для создания
		 соединения с сервером БД.


		 ![](../Class_Image.gif)
		 [DalOrcl8Driver](DalOrcl8Driver/DalOrcl8Driver.htm)
		 Класс DalOrcl8Driver
		 реализует драйвер, используемый для создания соединения с сервером
		 БД Oracle.


		 ![](../Class_Image.gif)
		 [DalPostgresDriver](DalPostgresDriver/DalPostgresDriver.htm)
		 Класс DalPostgresDriver
		 реализует драйвер, используемый для работы с СУБД PostgreSQL.


		 ![](../Class_Image.gif)
		 [DalSQLiteDriver](DalSQLiteDriver/DalSQLiteDriver.htm)
		 Класс DalSQLiteDriver
		 реализует драйвер, используемый для работы с СУБД SQLite.
