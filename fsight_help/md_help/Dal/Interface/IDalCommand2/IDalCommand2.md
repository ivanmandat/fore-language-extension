# IDalCommand2


Сборка: Dal;


## Описание


Интерфейс IDalCommand2 содержит
 методы для создания задачи, позволяющей работать с базой данных с помощью
 SQL-запросов в асинхронном режиме.


## Иерархия наследования


IDalCommand2


## Комментарии


Для создания задачи предварительно необходимо создать и настроить команду,
 реализуемую интерфейсом [IDalCommand](../IDalCommand/IDalCommand.htm).
 После настройки созданную команду можно привести к данному интерфейсу
 для создания задачи. Соединение, в рамках которого будет работать команда,
 должно поддерживать асинхронный режим работы. Если текущее соединение
 не поддерживает асинхронный режим, то для получения необходимого соединения
 можно воспользоваться следующими методами:


	- [IDalConnection2.Clone](../IDalConnection2/IDalConnection2.Clone.htm);


	- [ISecurityConnection2.Clone](KeSom.chm::/Interface/ISecurityConnection2/ISecurityConnection2.Clone.htm).
