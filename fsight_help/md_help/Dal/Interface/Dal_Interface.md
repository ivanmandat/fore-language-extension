# Интерфейсы сборки Dal


		 Интерфейс
		 Краткое описание


		 ![](../Interface_Image.gif)
		 [IDalColumns](IDalColumns/IDalColumns.htm)
		 Интерфейс IDalColumns
		 содержит свойства и методы для работы с курсором, содержащим системную
		 информацию о полях таблиц, хранящихся в БД.


		 ![](../Interface_Image.gif)
		 [IDalCommand](IDalCommand/IDalCommand.htm)
		 Интерфейс IDalCommand
		 предназначен для работы с командой, которая осуществляет выполнение
		 SQL-запросов к базе данных.


		 ![](../Interface_Image.gif)
		 [IDalCommand2](IDalCommand2/IDalCommand2.htm)
		 Интерфейс IDalCommand2
		 содержит методы для создания задачи, позволяющей работать с базой
		 данных с помощью SQL-запросов в асинхронном режиме.


		 ![](../Interface_Image.gif)
		 [IDalCommandEx](IDalCommandEx/IDalCommandEx.htm)
		 Интерфейс IDalCommandEx предназначен
		 для работы с базой данных с помощью SQL-запросов на случай возникновения
		 исключительной ситуации.


		 ![](../Interface_Image.gif)
		 [IDalCommandParam](IDalCommandParam/IDalCommandParam.htm)
		 Интерфейс IDalCommandParam
		 содержит свойства параметров SQL-запросов, используемых при выполнении
		 команды на сервере БД.


		 ![](../Interface_Image.gif)
		 [IDalCommandParams](IDalCommandParams/IDalCommandParams.htm)
		 Интерфейс IDalCommandParams
		 управляет коллекцией параметров SQL-запроса для работы с базой
		 данных.


		 ![](../Interface_Image.gif)
		 [IDalCommandTask](IDalCommandTask/IDalCommandTask.htm)
		 Интерфейс IDalCommandTask
		 содержит свойства задачи, предназначенной для работы с базой данных
		 с помощью SQL-запросов в асинхронном режиме.


		 ![](../Interface_Image.gif)
		 [IDalConnection](IDalConnection/IDalConnection.htm)
		 Интерфейс IDalConnection
		 содержит свойства и методы для управления текущим соединением
		 с базой данных.


		 ![](../Interface_Image.gif)
		 [IDalConnection2](IDalConnection2/IDalConnection2.htm)
		 Интерфейс IDalConnection2
		 содержит свойства и методы для управления специализированными
		 соединениями с базой данных.


		 ![](../Interface_Image.gif)
		 [IDalConnectionDescriptor](IDalConnectionDescriptor/IDalConnectionDescriptor.htm)
		 Интерфейс IDalConnectionDescriptor
		 содержит свойства и методы для создания соединения с сервером
		 базы данных.


		 ![](../Interface_Image.gif)
		 [IDalConnectionDescriptor2](IDalConnectionDescriptor2/IDalConnectionDescriptor2.htm)
		 Интерфейс IDalConnectionDescriptor2
		 содержит дополнительные свойства и методы для установки соединения
		 с сервером базы данных.


		 ![](../Interface_Image.gif)
		 [IDalConnectionDescriptorExt](IDalConnectionDescriptorExt/IDalConnectionDescriptorExt.htm)
		 Интерфейс IDalConnectionDescriptorExt
		 позволяет создать соединение с сервером БД, при возникновении
		 ошибки истечения срока действия пароля.


		 ![](../Interface_Image.gif)
		 [IDalConnectionDescriptorParam](IDalConnectionDescriptorParam/IDalConnectionDescriptorParam.htm)
		 Интерфейс IDalConnectionDescriptorParam
		 содержит свойства параметра, используемого для подключения к серверу
		 БД.


		 ![](../Interface_Image.gif)
		 [IDalConnectionDescriptorParams](IDalConnectionDescriptorParams/IDalConnectionDescriptorParams.htm)
		 Интерфейс IDalConnectionDescriptorParams
		 содержит свойства и методы коллекции параметров, используемых
		 для подключения к серверу БД.


		 ![](../Interface_Image.gif)
		 [IDalConnectionExt](IDalConnectionExt/IDalConnectionExt.htm)
		 Интерфейс IDalConnectionExt управляет
		 текущим соединением с базой данных, в случае если при создании
		 соединения произошла ошибка истечения срока действия пароля (для
		 Oracle ORA-28001, для MS SQL 2008 18488).


		 ![](../Interface_Image.gif)
		 [IDalCursor](IDalCursor/IDalCursor.htm)
		 Интерфейс IDalCursor
		 содержит свойства и методы для работы с курсором, полученным в
		 результате выполнения SQL-запроса к БД.


		 ![](../Interface_Image.gif)
		 [IDalCursorField](IDalCursorField/IDalCursorField.htm)
		 Интерфейс IDalCursorField
		 содержит свойства поля курсора, полученного в результате выполнения
		 SQL-запроса.


		 ![](../Interface_Image.gif)
		 [IDalCursorFields](IDalCursorFields/IDalCursorFields.htm)
		 Интерфейс IDalCursorFields
		 содержит свойства для работы с коллекцией полей курсора, полученного
		 в результате выполнения SQL-запроса.


		 ![](../Interface_Image.gif)
		 [IDalDriver](IDalDriver/IDalDriver.htm)
		 Базовый интерфейс IDalDriver,
		 содержащий методы для создания описания соединения с сервером
		 БД.


		 ![](../Interface_Image.gif)
		 [IDalIndexes](IDalIndexes/IDalIndexes.htm)
		 Интерфейс IDalIndexes
		 содержит свойства и методы для работы с курсором, содержащим системную
		 информацию о индексах таблиц, хранящихся в БД.


		 ![](../Interface_Image.gif)
		 [IDalODBCConnectionDescriptor](IDalODBCConnectionDescriptor/IDalODBCConnectionDescriptor.htm)
		 Интерфейс IDalODBCConnectionDescriptor
		 содержит свойства для настройки дополнительных параметров подключения
		 с использованием ODBC драйвера.


		 ![](../Interface_Image.gif)
		 [IDalOleDbDrivers](IDalOleDbDrivers/IDalOleDbDrivers.htm)
		 Интерфейс IDalOleDbDrivers
		 содержит свойства для работы с коллекцией драйверов OLE DB, установленных
		 в операционной системе и используемых для подключения к серверам
		 БД.


		 ![](../Interface_Image.gif)
		 [IDalProcedures](IDalProcedures/IDalProcedures.htm)
		 Интерфейс IDalProcedures
		 содержит свойства и методы для работы с курсором, содержащим системную
		 информацию о процедурах, хранящихся в БД.


		 ![](../Interface_Image.gif)
		 [IDalTables](IDalTables/IDalTables.htm)
		 Интерфейс IDalTables
		 содержит свойства и методы для работы с курсором, содержащим системную
		 информацию о таблицах, хранящихся в БД.


		 ![](../Interface_Image.gif)
		 [IDalTriggers](IDalTriggers/IDalTriggers.htm)
		 Интерфейс IDalTriggers
		 содержит свойства и методы для работы с курсором, содержащим системную
		 информацию о триггерах таблиц, хранящихся в БД.


		 ![](../Interface_Image.gif)
		 [IDalViews](IDalViews/IDalViews.htm)
		 Интерфейс IDalViews
		 содержит свойства и методы для работы с курсором, содержащим системную
		 информацию о представлениях, хранящихся в БД.
