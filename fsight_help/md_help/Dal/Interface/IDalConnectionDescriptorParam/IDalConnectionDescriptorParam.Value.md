# IDalConnectionDescriptorParam.Value


## Синтаксис


Value: String;


## Описание


Свойство Value определяет значение
 параметра, используемое при подключения к серверу БД.


## Комментарии


Значения параметра с [наименованием](IDalConnectionDescriptorParam.Name.htm)
 «Password» будет доступно только на запись.

 метода [IDalConnectionDescriptorParams.Find](../IDalConnectionDescriptorParams/IDalConnectionDescriptorParams.Find.htm).
