# IDalConnectionDescriptorParam


## Описание


Интерфейс IDalConnectionDescriptorParam
 содержит свойства параметра, используемого для подключения к серверу БД.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Mandatory](IDalConnectionDescriptorParam.Mandatory.htm)
		 Свойство Mandatory
		 возвращает True, если
		 параметр, используемый для подключения к серверу БД, является
		 обязательным.


		 ![](../../Property_Image.gif)
		 [Name](IDalConnectionDescriptorParam.Name.htm)
		 Свойство Name возвращает
		 наименование параметра, используемого при подключения к серверу
		 БД.


		 ![](../../Property_Image.gif)
		 [Value](IDalConnectionDescriptorParam.Value.htm)
		 Свойство Value определяет
		 значение параметра, используемое при подключения к серверу БД.
