# IDalConnection2


Сборка: Dal;


## Описание


Интерфейс IDalConnection2 предназначен
 для управления специализированными соединениями с базой данных.


## Комментарии


Интерфейс IDalConnection2 реализован
 для получения возможности создания нового подключения к БД на базе существующего
 с указания типа нового подключения.
