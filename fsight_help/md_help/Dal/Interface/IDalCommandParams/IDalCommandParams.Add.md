# IDalCommandParams.Add


## Синтаксис


Add(Name: String): [IDalCommandParam](../IDalCommandParam/IDalCommandParam.htm);


## Параметры


Name - наименование добавляемого
 параметра.


## Описание


Метод Add добавляет параметр,
 используемый в SQL-запросе.


## Комментарии


Использование данного метода доступно, если свойству [Type](../IDalCommand/IDalCommand.Type.htm)
 установлено значение DalCommandType.StoredProcedure.


## Пример


Использование метода приведено в примере для свойства [IDalCommand.Type](../IDalCommand/IDalCommand.Type.htm).
