# IDalConnectionDescriptorExt


## Описание


Интерфейс IDalConnectionDescriptorExt позволяет создать соединение с сервером БД, при возникновении ошибки истечения срока действия пароля.


## Методы


 Имя метода
 Краткое описание


 ![](../../Sub_Image.gif)
 [CreateConnectionExt](IDalConnectionDescriptorExt.CreateConnectionExt.htm)
 Метод CreateConnectionExt возвращает ошибку или создает соединение с сервером БД, в случае если срок действия пароля пользователя истек.
