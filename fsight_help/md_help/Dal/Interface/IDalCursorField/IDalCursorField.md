# IDalCursorField


## Описание


Интерфейс IDalCursorField содержит
 свойства поля курсора, полученного в результате выполнения SQL-запроса.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DataDomain](IDalCursorField.DataDomain.htm)
		 Свойство DataDomain
		 возвращает тип данных поля, если оно предназначено для хранения
		 многобайтовой информации.


		 ![](../../Property_Image.gif)
		 [GetDbDataType](IDalCursorField.GetDbDataType.htm)
		 Свойство GetDbDataType
		 возвращает тип поля курсора.


		 ![](../../Property_Image.gif)
		 [Name](IDalCursorField.Name.htm)
		 Свойство Name возвращает
		 наименование поля курсора.


		 ![](../../Property_Image.gif)
		 [Precision](IDalCursorField.Precision.htm)
		 Свойство Precision
		 возвращает точность поля курсора.


		 ![](../../Property_Image.gif)
		 [Size](IDalCursorField.Size.htm)
		 Свойство Size возвращает
		 размер поля курсора.


		 ![](../../Property_Image.gif)
		 [Value](IDalCursorField.Value.htm)
		 Свойство Value возвращает
		 значение поля текущей записи в курсоре.
