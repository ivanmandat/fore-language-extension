# IDalCursorField.Size


## Синтаксис


Size: Integer;


## Описание


Свойство Size возвращает размер поля курсора.


## Комментарии


Свойство возвращает следующие значения:


-
для строковых типов данных свойство возвращает размер строки в символах, если это возможно, иначе в байтах.


-
для числовых типов данных с фиксированной точкой свойство возвращает общее количество знаков.


-
для числовых типов данных с плавающей точкой свойство возвращает общее количество знаков.


-
для [Blob](../../Enums/DbDataType.htm) полей свойство возвращает 0.
