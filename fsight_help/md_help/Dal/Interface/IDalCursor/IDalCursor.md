# IDalCursor


## Описание


Интерфейс IDalCursor содержит
 свойства и методы для работы с курсором, полученным в результате выполнения
 SQL-запроса к БД.


## Комментарии


Курсор представляет собой кешированную таблицу, полученную в результате
 выполнения SQL-запроса. Курсор поддерживает перемещение по данным только
 вперед. Курсор не является обновляемым, т.е., если на сервере изменились
 значения в таблицах, то для получения новых данных необходимо повторное
 выполнение SQL-запроса и [создание](../IDalCommand/IDalCommand.CreateCursor.htm)
 нового курсора.


При работе с сервером БД существует ограничение на количество одновременно
 открытых курсоров, поэтому необходимо своевременно закрывать неиспользуемые
 курсоры, используя метод [Close](IDalCursor.Close.htm).
