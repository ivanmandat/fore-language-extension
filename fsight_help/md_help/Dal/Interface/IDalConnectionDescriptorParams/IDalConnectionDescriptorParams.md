# IDalConnectionDescriptorParams


## Описание


Интерфейс IDalConnectionDescriptorParams
 содержит свойства и методы коллекции параметров, используемых для подключения
 к серверу БД.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDalConnectionDescriptorParams.Count.htm)
		 Свойство Count возвращает
		 количество параметров, использыемых для подключения с данным драйвером.


		 ![](../../Property_Image.gif)
		 [Item](IDalConnectionDescriptorParams.Item.htm)
		 Свойство Item возвращает
		 параметр, индекс которого передается посредством параметра Index.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Find](IDalConnectionDescriptorParams.Find.htm)
		 Метод Find осуществляет
		 поиск параметра по наименованию, передаваемому посредством параметра
		 Param.
		Примечание.
		 Метод не предназначен для использования в прикладном коде. Оставлен
		 для совместимости с предыдущими версиями.
