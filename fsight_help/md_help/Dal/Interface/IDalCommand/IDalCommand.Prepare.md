# IDalCommand.Prepare


## Синтаксис


Prepare;


## Описание


Метод Prepare подготавливает
 команду для многократного использования на сервере БД.


Примечание.
 При работе с СУБД Microsoft SQL Server данный метод всегда вызывается
 при выполнении запросов.
