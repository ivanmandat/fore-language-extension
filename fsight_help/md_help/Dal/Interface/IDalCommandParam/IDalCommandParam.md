# IDalCommandParam


## Описание


Интерфейс IDalCommandParam содержит
 свойства параметров SQL-запросов, используемых при выполнении команды
 на сервере БД.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DataDomain](IDalCommandParam.DataDomain.htm)
		 Свойство DataDomain
		 определяет тип данных параметра, если он предназначен для передачи
		 многобайтовой информации.


		 ![](../../Property_Image.gif)
		 [DataType](IDalCommandParam.DataType.htm)
		 Свойство DataType определяет
		 тип данных параметра SQL-запроса.


		 ![](../../Property_Image.gif)
		 [Direction](IDalCommandParam.Direction.htm)
		 Свойство Direction
		 определяет предназначение параметра SQL-запроса.


		 ![](../../Property_Image.gif)
		 [Name](IDalCommandParam.Name.htm)
		 Свойство Name возвращает
		 наименование параметра SQL-запроса, используемого при выполнении
		 команды на сервере БД.


		 ![](../../Property_Image.gif)
		 [Size](IDalCommandParam.Size.htm)
		 Свойство Size определяет
		 максимальную длину значения, которое может принимать параметр.


		 ![](../../Property_Image.gif)
		 [Value](IDalCommandParam.Value.htm)
		 Свойство Value определяет
		 значение параметра SQL-запроса, используемое при выполнении команды
		 на сервере БД.
