# IDalCommandEx.SetCurrentParamsRow


## Синтаксис


SetCurrentParamsRow(Value: Integer);


## Параметры


Value. Индекс набора параметров.


## Описание


Метод SetCurrentParamsRow устанавливает текущий набор значений параметров.


## Комментарии


После установки набора, значения параметров можно определить в коллекции [IDalCommand.Params](../IDalCommand/IDalCommand.Params.htm).
