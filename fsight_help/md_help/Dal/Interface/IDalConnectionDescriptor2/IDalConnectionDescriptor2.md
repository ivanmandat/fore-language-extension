# IDalConnectionDescriptor2


Сборка: Dal;


## Описание


Интерфейс IDalConnectionDescriptor2
 предназначен для определения настроек при установке соединения с сервером
 базы данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ConnectAsPrivilege](IDalConnectionDescriptor2.ConnectAsPrivilege.htm)
		 Свойство ConnectAsPrivilege
		 определяет подключение к серверу баз данных с заданной привилегией.
		Примечание.
		 Свойство не предназначено для использования в прикладном коде.
		 Оставлено для совместимости с предыдущими версиями.


		 ![](../../Property_Image.gif)
		 [ConnectionType](IDalConnectionDescriptor2.ConnectionType.htm)
		 Свойство ConnectionType
		 определяет устанавливаемый тип соединения с базой данных.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [SupportsConnectionType](IDalConnectionDescriptor2.SupportsConnectionType.htm)
		 Метод SupportsConnectionType
		 определяет, поддерживается ли драйвером указанный в параметрах
		 тип соединения.
