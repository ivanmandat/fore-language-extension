# IDalOleDbDrivers


## Описание


Интерфейс IDalOleDbDrivers содержит
 свойства для работы с коллекцией драйверов OLE DB, установленных в операционной
 системе и используемых для подключения к серверам БД.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDalOleDbDrivers.Count.htm)
		 Свойство Count возвращает
		 количество драйверов OLE DB, установленных в операционной системе.


		 ![](../../Property_Image.gif)
		 [Driver](IDalOleDbDrivers.Driver.htm)
		 Свойство Driver возвращает
		 настройки OLE DB драйвера с указанным индексом.


		 ![](../../Property_Image.gif)
		 [DriverName](IDalOleDbDrivers.DriverName.htm)
		 Свойство DriverName
		 возвращает наименование OLE DB драйвера с указанным индексом.
