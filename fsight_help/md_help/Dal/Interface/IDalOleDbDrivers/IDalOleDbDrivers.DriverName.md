# IDalOleDbDrivers.DriverName


## Синтаксис


DriverName(Index: Integer): String;


## Параметры


Index.
 Индекс драйвера в списке.


## Описание


Свойство DriverName возвращает
 наименование OLE DB драйвера с указанным индексом.
