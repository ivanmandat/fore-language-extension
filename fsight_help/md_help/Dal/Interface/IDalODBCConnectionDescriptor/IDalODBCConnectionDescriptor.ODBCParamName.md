# IDalODBCConnectionDescriptor.ODBCParamName


## Синтаксис


		ODBCParamName(Index: Integer): String;


## Параметры


Index. Индекс параметра.


## Описание


Свойство ODBCParamName возвращает
 наименование параметра с указанным индексом.


## Комментарии


Список параметров формируется при задании для них значений с помощью
 свойства [ODBCParams](IDalODBCConnectionDescriptor.ODBCParams.htm)
 или [ODBCParamValue](IDalODBCConnectionDescriptor.ODBCParamValue.htm).
