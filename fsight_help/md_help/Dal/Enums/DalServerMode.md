# DalServerMode


## Описание


Перечисление DalServerMode содержит
 типы серверов СУБД, к которым осуществляется подключение.


Используется следующими свойствами и методами:


	- [IDalConnection2.ServerMode](../Interface/IDalConnection2/IDalConnection2.ServerMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Unknown. Описание.


		 1
		 ORCL. Подключение с
		 использованием драйвера Oracle.


		 2
		 MSSQL. Подключение
		 с использованием драйвера Microsoft SQL Server.


		 5
		 ODBC. Подключение с
		 использованием одного из драйверов ODBC.


		 8
		 Postgres. Подключение
		 с использованием драйвера PostgreSQL.


		 9
		 OleDb. Подключение
		 с использованием драйвера OLE DB, точный тип не определен.


		 10
		 AdoDb. Подключение
		 с использованием драйвера ADOdb.


		 11
		 SQLite. Подключение
		 с использованием драйвера SQLite.


		 12
		 Hive. Подключение с
		 использованием драйвера Hive.


		 13
		 Service. Подключение
		 с использованием веб-сервиса аналитической платформы.


		 15
		 GenericODBC. Подключение
		 к различным источникам данных посредством драйверов ODBC, установленных
		 в операционной системе Windows.


		 16
		 ClickHouse. Подключение
		 с использованием драйвера ODBC для ClickHouse.


		 17
		 MySQL. Подключение
		 с использованием драйвера MySQL.
