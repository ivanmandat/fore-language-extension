# DalConnectionType


## Описание


Перечисление DalConnectionType содержит типы устанавливаемого соединения.


Используется следующими свойствами и методами:


-
[IDalConnection2.Clone](../Interface/IDalConnection2/IDalConnection2.Clone.htm);


-
[IDalConnection2.Type](../Interface/IDalConnection2/IDalConnection2.Type.htm);


-
[IDalConnectionDescriptor2.ConnectionType](../Interface/IDalConnectionDescriptor2/IDalConnectionDescriptor2.ConnectionType.htm);


-
[IDalConnectionDescriptor2.SupportsConnectionType](../Interface/IDalConnectionDescriptor2/IDalConnectionDescriptor2.SupportsConnectionType.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Default. Соединение по умолчанию.


 1
 FastInsert. Соединение для быстрой вставки.


 2
 Async. Соединение для асинхронного выполнения команд.
