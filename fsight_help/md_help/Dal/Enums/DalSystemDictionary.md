# DalSystemDictionary


## Описание


Перечисление DalSystemDictionary содержит типы курсора, полученного в результате выполнения SQL-запроса к БД.


Используется следующими свойствами и методами:


-
[IDalConnection2.Discover](../Interface/IDalConnection2/IDalConnection2.Discover.htm);


-
[ISecurityConnection.Discover](KeSom.chm::/Interface/ISecurityConnection/ISecurityConnection.Discover.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Tables. Курсор, который содержит системную информацию о таблицах, хранящихся в БД.


 1
 Views. Курсор, который содержит системную информацию о представлениях, хранящихся в БД.


 2
 Columns. Курсор, который содержит системную информацию о полях таблиц, хранящихся в БД.


 3
 Indexes. Курсор, который содержит системную информацию об индексах таблиц, хранящихся в БД.


 4
 Trigers. Курсор, который содержит системную информацию о триггерах таблиц, хранящихся в БД.


 5
 Procedures. Курсор, который содержит системную информацию о процедурах, хранящихся в БД.


 6
 FileGroups. Курсор, который содержит системную информацию о файловых группах, созданных в БД.


 7
 ForeignKeys. Курсор, который содержит системную информацию о внешних ключах, созданных для таблиц.


 8
 ForeignLinks. Курсор, который содержит системную информацию о связях, созданных между таблицами.


## Комментарии


ForeignKeys. Возвращаемый курсор содержит следующие поля:


-
TABLE_NAME. Имя таблицы;


-
FK_NAME. Имя внешнего ключа;


-
DELETE_RULE. Метод удаления (1 - каскадное удаление);


-
COLUMN_NAME. Имя поля внешнего ключа;


-
POSITION. Позиция поля в списке полей внешнего ключа;


-
RTABLE_NAME. Имя таблицы, на которую создается внешний ключ;


-
RCOLUMN_NAME. Имя поля внешней таблицы.
