# DalCommandType


## Описание


Перечисление DalCommandType содержит типы выполняемой команды.


Используется следующими свойствами и методами:


-
[IDalCommand.Type](../Interface/IDalCommand/IDalCommand.Type.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Text. Команда используется для выполнения SQL-запроса.


 1
 StoredProcedure. Команда используется для запуска хранимой процедуры или функции.
