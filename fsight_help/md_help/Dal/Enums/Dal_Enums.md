# Перечисления сборки Dal


		 Перечисление
		 Краткое описание


		 ![](../Enums_Image.gif)
		 [DalCommandExecuteType](DalCommandExecuteType.htm)
		 Перечисление DalCommandExecuteType
		 содержит варианты предназначения задачи, которая работает с базой
		 данных с помощью SQL-запросов.


		 ![](../Enums_Image.gif)
		 [DalCommandOption](DalCommandOption.htm)
		 Перечисление DalCommandOption
		 содержит параметры выполнения SQL-запроса.


		 ![](../Enums_Image.gif)
		 [DalCommandType](DalCommandType.htm)
		 Перечисление DalCommandType
		 содержит типы выполняемой команды.


		 ![](../Enums_Image.gif)
		 [DalConnectAsPrivilegeType](DalConnectAsPrivilegeType.htm)
		 Перечисление DalConnectAsPrivilegeType
		 содержит варианты подключения к серверу баз данных.
		Примечание.
		 Перечисление не предназначено для использования в прикладном коде.
		 Оставлено для совместимости с предыдущими версиями.


		 ![](../Enums_Image.gif)
		 [DalConnectionType](DalConnectionType.htm)
		 Перечисление DalConnectionType
		 содержит типы устанавливаемого соединения.


		 ![](../Enums_Image.gif)
		 [DalParamDirection](DalParamDirection.htm)
		 Перечисление DalParamDirection
		 содержит типы предназначения параметра SQL-запроса.


		 ![](../Enums_Image.gif)
		 [DalServerMode](DalServerMode.htm)
		 Перечисление DalServerMode
		 содержит типы серверов СУБД, к которым осуществляется подключение.


		 ![](../Enums_Image.gif)
		 [DalSystemDictionary](DalSystemDictionary.htm)
		 Перечисление DalSystemDictionary
		 содержит типы курсора, полученного в результате выполнения SQL-запроса
		 к БД.


		 ![](../Enums_Image.gif)
		 [DbDataDomain](DbDataDomain.htm)
		 Перечисление DbDataDomain
		 содержит типы данных для хранения многобайтовой информации.


		 ![](../Enums_Image.gif)
		 [DbDataType](DbDataType.htm)
		 Перечисление DbDataType
		 содержит типы данных, используемые в платформе.
