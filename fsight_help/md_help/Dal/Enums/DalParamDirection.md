# DalParamDirection


## Описание


Перечисление DalParamDirection содержит типы предназначения параметра SQL-запроса.


Используется следующими свойствами и методами:


-
[IDalCommandParam.Direction](../Interface/IDalCommandParam/IDalCommandParam.Direction.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Unknown. Неизвестно.


 1
 Input. Параметр, доступный только для ввода значений.


 2
 Output. Параметр, доступный только для вывода значений.


 3
 InputOutput. Параметр, доступный для ввода и вывода значений.


 4
 ReturnValue. Параметр, в который будет помещено значение, возвращаемое хранимой процедурой, встроенной или определенной пользователем функцией. Результат выполнения имеет определенный тип данных.


 8
 RefCursor. Параметр, в который будет помещено значение, возвращаемое хранимой процедурой, встроенной или определенной пользователем функцией. Результатом выполнения является курсор произвольной структуры. Для работы с курсором необходимо привести значение параметра к интерфейсу [IDalCursor](../Interface/IDalCursor/IDalCursor.htm).
Примечание. Возможность получения результата выполнения в виде курсора доступна, если для подключения используется драйвер Oracle 9.x и выше.
