# UiTabSheet


## Описание


Класс UiTabSheet содержит свойства
 компонента [UiTabSheet](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiTabSheet.htm),
 который является источником данных для компонента [TabSheetBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/TabSheetBox.htm).
