# TabScaleEventArgs


## Описание


Класс TabScaleEventArgs предназначен
 для работы с параметрами события [TabSheetBox.OnScaleChange](../TabSheetBox/TabSheetBox.OnScaleChange.htm),
 возникающего при изменении масштаба таблицы.
