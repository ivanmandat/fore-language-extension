# TabSheetBox


Сборка: Tab;


## Описание


Класс TabSheetBox реализует
 компонент среды разработки [TabSheetBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/TabSheetBox.htm),
 используемый для отображения таблиц.


## Комментарии


Для работы с таблицами используйте класс [TabSheet](../TabSheet/TabSheet.htm).
