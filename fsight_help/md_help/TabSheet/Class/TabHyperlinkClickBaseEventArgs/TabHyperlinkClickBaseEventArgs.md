# TabHyperlinkClickBaseEventArgs


Сборка: Tab;


## Описание


Класс TabHyperlinkClickBaseEventArgs реализует
 аргументы событий, связанных с щелчком по гиперссылке в ячейке таблицы
 компонента [LanerBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/LanerBox.htm).
