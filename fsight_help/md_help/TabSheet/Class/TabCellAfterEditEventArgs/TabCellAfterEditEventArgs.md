# TabCellAfterEditEventArgs


## Описание


Класс TabCellAfterEditEventArgs
 предназначен для работы с параметрами события [TabSheetBox.OnAfterEdit](../TabSheetBox/TabSheetBox.OnAfterEdit.htm),
 происходящего после редактирования ячейки.
