# TabCellContentChangeEventArgs


## Описание


Класс TabCellContentChangeEventArgs
 предназначен для работы с параметрами события [TabSheetBox.OnChangeCellContent](../TabSheetBox/TabSheetBox.OnChangeCellContent.htm)
 (изменение содержимого в ячейке таблицы).
