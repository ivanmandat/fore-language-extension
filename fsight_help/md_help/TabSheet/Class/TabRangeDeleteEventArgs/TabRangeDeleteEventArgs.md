# TabRangeDeleteEventArgs


Сборка: Tab;


## Описание


Класс TabRangeDeleteEventArgs
 предназначен для работы с аргументами событий, происходящих при удалении
 диапазона ячеек.


## Комментарии


Для работы с аргументами событий, происходящих при добавлении диапазона ячеек
 используйте класс [TabRangeInsertEventArgs](../TabRangeInsertEventArgs/TabRangeInsertEventArgs.htm).
