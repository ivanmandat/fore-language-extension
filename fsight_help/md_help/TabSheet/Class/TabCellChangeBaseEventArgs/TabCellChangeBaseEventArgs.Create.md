# TabCellChangeBaseEventArgs.Create


## Синтаксис


Create(TabSheet: [ITabSheet](../../Interface/ITabSheet/ITabSheet.htm); Row: Integer; Column: Integer; Value: Variant);


## Параметры


TabSheet - таблица компонента, для которого генерируется событие.


Row - индекс строки, в которой расположена ячейка, для которой генерируется событие.


Column - индекс столбца, в которой расположена ячейка, для которой генерируется событие.


Value - значение, которое установлено в ячейке.


## Описание


Конструктор Create создает аргумент события с указанным значением ячейки таблицы.
