# TabRowEventArgs


## Описание


Класс TabRowEventArgs предназначен
 для работы с параметрами события [TabSheetBox.OnRowClick](../TabSheetBox/TabSheetBox.OnRowClick.htm),
 происходящего при щелчке левой кнопкой мыши по заголовку строки таблицы.
