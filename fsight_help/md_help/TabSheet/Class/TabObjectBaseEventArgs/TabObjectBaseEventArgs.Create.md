# TabObjectBaseEventArgs.Create


## Синтаксис


Create(TabSheet: [ITabSheet](../../Interface/ITabSheet/ITabSheet.htm);
 Object: [ITabObject](../../Interface/ITabObject/ITabObject.htm);
 Action: [TabObjectAction](../../Enums/TabObjectAction.htm));


## Параметры


TabSheet. Таблица компонента,
 для которого генерируется событие.


Object. Объект таблицы, для
 которого генерируется событие.


Action. Действие над объектом.


## Описание


Конструктор Create создает аргумент
 события, наступающего при активации/деактивации объекта таблицы.
