# TabEditorEventArgs.CreateEditorArgs


## Синтаксис


CreateEditorArgs(TabSheet: [ITabSheetBox](../../Interface/ITabSheetBox/ITabSheetBox.htm); Editor: [ITabEditor](../../Interface/ITabEditor/ITabEditor.htm));


## Параметры


TabSheet. Компонент, для которого генерируется событие.


Editor. Параметры редактора ячейки.


## Описание


Конструктор CreateEditorArgs создает аргумент события, возникающего при редактировании текста в ячейке.
