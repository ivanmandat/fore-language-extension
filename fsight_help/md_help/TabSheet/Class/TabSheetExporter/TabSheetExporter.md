# TabSheetExporter


## Описание


Класс TabSheetExporter реализует объект, предназначенный для экспорта таблицы.


## Свойства, унаследованные от [ITabSheetExporter](../../Interface/ITabSheetExporter/ITabSheetExporter.htm)


		 Имя свойства
		 Краткое описание

		 [AutoHeight](../../Interface/ITabSheetExporter/ITabSheetExporter.AutoHeight.htm)


		 Свойство AutoHeight
		 определяет, будет ли при экспорте в форматы HTML (*.html) и RTF
		 (*.rtf) использоваться автоподгон высоты строк.


		 ![](../../Property_Image.gif)
		 [BreakIntoLayouts](../../Interface/ITabSheetExporter/ITabSheetExporter.BreakIntoLayouts.htm)


		 Свойство BreakIntoLayouts
		 определяет, разбивать ли лист на печатные страницы при экспорте
		 в формат RTF (*.rtf).


		 ![](../../Property_Image.gif)
		 [EmfType](../../Interface/ITabSheetExporter/ITabSheetExporter.EmfType.htm)

		 тип создаваемого при экспорте метафайла.


		 ![](../../Property_Image.gif)
		 [ExportCellImages](../../Interface/ITabSheetExporter/ITabSheetExporter.ExportCellImages.htm)


		 Свойство ExportCellImages
		 определяет, будут ли экспортироваться изображения, содержащиеся
		 в ячейках таблицы.


		 ![](../../Property_Image.gif)
		 [ExportFootnotes](../../Interface/ITabSheetExporter/ITabSheetExporter.ExportFootnotes.htm)


		 Свойство ExportFootnotes
		 определяет, будут ли экспортироваться сноски таблицы.


		 ![](../../Property_Image.gif)
		 [ExportFormulas](../../Interface/ITabSheetExporter/ITabSheetExporter.ExportFormulas.htm)


		 Свойство ExportFormulas определяет
		 необходимость экспорта формул, содержащихся в таблице.


		 ![](../../Property_Image.gif)
		 [ExportHiddenCells](../../Interface/ITabSheetExporter/ITabSheetExporter.ExportHiddenCells.htm)


		 Свойство ExportHiddenCells
		 определяет необходимость экспорта скрытых строк/столбцов при экспорте
		 в форматы XLS (*.xls) и ODS (*.ods).


		 ![](../../Property_Image.gif)
		 [ExportObjectAsBitmap](../../Interface/ITabSheetExporter/ITabSheetExporter.ExportObjectAsBitmap.htm)


		 Свойство ExportObjectAsBitmap
		 определяет необходимость экспорта объектов в растровом формате

		 [ExportObjects](../../Interface/ITabSheetExporter/ITabSheetExporter.ExportObjects.htm)

		 определяет необходимость экспорта объектов таблицы.


		 ![](../../Property_Image.gif)
		 [ExportPalette](../../Interface/ITabSheetExporter/ITabSheetExporter.ExportPalette.htm)


		 Свойство ExportPalette

		 ![](../../Property_Image.gif)
		 [Label](../../Interface/ITabSheetExporter/ITabSheetExporter.Label.htm)


		 Свойство Label определяет

		 ![](../../Property_Image.gif)

		 Свойство SheetAccess
