# TabColumnBaseEventArgs.CreateTabColumnArgs


## Синтаксис


CreateTabColumnArgs(TabSheet: [ITabSheet](../../Interface/ITabSheet/ITabSheet.htm); Column: Integer);


## Параметры


TabSheet - таблица компонента, для которого генерируется событие.


Column - индекс столбца, для которого генерируется событие.


## Описание


Конструктор CreateTabColumnArgs создает аргумент события, связанный с указанным столбцом таблицы.
