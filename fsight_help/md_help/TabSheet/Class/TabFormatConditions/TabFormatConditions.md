# TabFormatConditions


## Описание


Класс TabFormatConditions реализует
 коллекцию условных форматов текста.


## Свойства, унаследованные от [ITabFormatConditions](../../Interface/ITabFormatConditions/ITabFormatConditions.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](../../Interface/ITabFormatConditions/ITabFormatConditions.Count.htm)
		 Свойство Count возвращает
		 количество условных форматов в коллекции.


		 ![](../../Property_Image.gif)
		 [IsInUpdate](../../Interface/ITabFormatConditions/ITabFormatConditions.IsInUpdate.htm)
		 Свойство IsInUpdate
		 возвращает признак применения условных форматов.


		 ![](../../Property_Image.gif)
		 [Item](../../Interface/ITabFormatConditions/ITabFormatConditions.Item.htm)
		 Свойство Item возвращает
		 ссылку на условный формат.


## Методы, унаследованные от [ITabFormatConditions](../../Interface/ITabFormatConditions/ITabFormatConditions.htm)


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](../../Interface/ITabFormatConditions/ITabFormatConditions.Add.htm)
		 Метод Add добавляет
		 новый условный формат и возвращает на него ссылку.


		 ![](../../Sub_Image.gif)
		 [AddCollection](../../Interface/ITabFormatConditions/ITabFormatConditions.AddCollection.htm)
		 Метод AddCollection
		 добавляет указанную коллекцию условных форматов.


		 ![](../../Sub_Image.gif)
		 [BeginUpdate](../../Interface/ITabFormatConditions/ITabFormatConditions.BeginUpdate.htm)
		 Метод BeginUpdate отключает
		 вычисление и применение условных форматов.


		 ![](../../Sub_Image.gif)
		 [Clear](../../Interface/ITabFormatConditions/ITabFormatConditions.Clear.htm)
		 Метод Clear очищает
		 коллекцию условных форматов.


		 ![](../../Sub_Image.gif)
		 [EndUpdate](../../Interface/ITabFormatConditions/ITabFormatConditions.EndUpdate.htm)
		 Метод EndUpdate возобновляет
		 вычисление и применение условных форматов.


		 ![](../../Sub_Image.gif)
		 [FindByKey](../../Interface/ITabFormatConditions/ITabFormatConditions.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск условного формата по ключу


		 ![](../../Sub_Image.gif)
		 [Move](../../Interface/ITabFormatConditions/ITabFormatConditions.Move.htm)
		 Метод Move осуществляет
		 перемещение условных форматов в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](../../Interface/ITabFormatConditions/ITabFormatConditions.Remove.htm)
		 Метод Remove удаляет
		 условный формат, индекс которого передается в качестве входного

		 [RemoveCollection](../../Interface/ITabFormatConditions/ITabFormatConditions.RemoveCollection.htm)
		 Метод RemoveCollection
