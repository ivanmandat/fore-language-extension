# TabProtectionFailedEventArgs


## Описание


Класс TabProtectionFailedEventArgs
 предназначен для работы с параметрами события [TabSheetBox.OnProtectionFail](../TabSheetBox/TabSheetBox.OnProtectionFail.htm),
 возникающего при срабатывании защиты таблицы.
