# TabProtectionFailedEventArgs.Create


## Синтаксис


Create(TabSheet: [ITabSheetBox](../../Interface/ITabSheetBox/ITabSheetBox.htm);
 Target: Variant; Message: String);


## Параметры


TabSheet - объект [TabSheetBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/TabSheetBox.htm), для которого возникает
 событие.


Target - параметр не используется.


Message - определяет информационное
 сообщение о запрещенном действии при включенной защите таблицы.


## Описание


Конструктор Create создает аргумент
 события, возникающего при срабатывании защиты таблицы.
