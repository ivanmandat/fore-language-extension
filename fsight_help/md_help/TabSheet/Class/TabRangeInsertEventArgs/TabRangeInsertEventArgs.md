# TabRangeInsertEventArgs


Сборка: Tab;


## Описание


Класс TabRangeInsertEventArgs
 предназначен для работы с аргументами событий, происходящих при добавлении
 диапазона ячеек.


## Комментарии


Для работы с аргументами событий, происходящих при удалении диапазона ячеек
 используйте класс [TabRangeDeleteEventArgs](../TabRangeDeleteEventArgs/TabRangeDeleteEventArgs.htm).
