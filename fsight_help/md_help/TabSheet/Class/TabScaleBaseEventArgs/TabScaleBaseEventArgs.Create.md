# TabScaleBaseEventArgs.Create


## Синтаксис


Create(TabSheet: [ITabSheet](../../Interface/ITabSheet/ITabSheet.htm); Scale: Double);


## Параметры


TabSheet - таблица компонента, для которого генерируется событие.


Scale - масштаб, устанавливаемый в таблице. Допустимое значение параметра [0.25; 4].


## Описание


Конструктор Create создает аргумент события с указанным масштабом таблицы.
