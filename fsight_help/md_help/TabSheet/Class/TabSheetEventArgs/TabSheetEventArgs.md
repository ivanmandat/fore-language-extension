# TabSheetEventArgs


## Описание


Класс TabSheetEventArgs реализует
 аргумент событий компонента [TabSheetBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/TabSheetBox.htm).


## Комментарии


Данный аргумент используется при обработке следующих событий:


	- [OnBeforeCellsChange](../TabSheetBox/TabSheetBox.OnBeforeCellsChange.htm)


	- [OnBeginSelectionChange](../TabSheetBox/TabSheetBox.OnBeginSelectionChange.htm)


	- [OnEndSelectionChange](../TabSheetBox/TabSheetBox.OnEndSelectionChange.htm)


	- [OnSelectionChange](../TabSheetBox/TabSheetBox.OnSelectionChange.htm)


	- [OnTableClick](../TabSheetBox/TabSheetBox.OnTableClick.htm)
