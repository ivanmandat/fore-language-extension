# TabRowBaseEventArgs.CreateTabRowArgs


## Синтаксис


CreateTabRowArgs(TabSheet: [ITabSheet](../../Interface/ITabSheet/ITabSheet.htm); Row: Integer);


## Параметры


TabSheet - таблица компонента, для которого генерируется событие.


Row - индекс строки, для которой генерируется событие.


## Описание


Конструктор CreateTabRowArgs создает аргумент события, связанный с указанной строкой таблицы.
