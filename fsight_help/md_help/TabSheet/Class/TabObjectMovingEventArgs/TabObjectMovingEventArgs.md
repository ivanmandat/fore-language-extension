# TabObjectMovingEventArgs


Сборка: Tab;


## Описание


Класс TabObjectMovingEventArgs
 реализует аргумент события, происходящего при перемещении объекта.


## Комментарии


Для реализации аргумента события, происходящего при каждом изменении
 размера объекта, используйте класс [TabObjectResizingEventArgs](../TabObjectResizingEventArgs/TabObjectResizingEventArgs.htm).
