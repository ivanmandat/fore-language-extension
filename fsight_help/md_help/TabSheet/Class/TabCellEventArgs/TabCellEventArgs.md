# TabCellEventArgs


## Описание


Класс TabCellEventArgs реализует
 аргумент событий, связанных со щелчком мышью по ячейке либо объектам,
 которые могут содержаться в ячейке:


	- [TabSheetBox.OnCellClick](../TabSheetBox/TabSheetBox.OnCellClick.htm);


	- [TabSheetBox.OnCellPictureClick](../TabSheetBox/TabSheetBox.OnCellPictureClick.htm);


	- [TabSheetBox.OnHyperlinkClick](../TabSheetBox/TabSheetBox.OnHyperlinkClick.htm).
