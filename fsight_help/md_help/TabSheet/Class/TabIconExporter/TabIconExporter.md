# TabIconExporter


## Описание


Класс TabIconExporter реализует объект, предназначенный для экспорта значков таблицы.


## Свойства, унаследованные от [ITabIconExporter](../../Interface/ITabIconExporter/ITabIconExporter.htm)

 Краткое описание

 [IconIndex](../../Interface/ITabIconExporter/ITabIconExporter.IconIndex.htm)
 Свойство IconIndex определяет индекс экспортируемого значка.


 ![](../../Property_Image.gif)
 [PixelHeight](../../Interface/ITabIconExporter/ITabIconExporter.PixelHeight.htm)
 Свойство PixelHeight возвращает высоту экспортируемого значка в пикселях.


 ![](../../Property_Image.gif)
 [PixelWidth](../../Interface/ITabIconExporter/ITabIconExporter.PixelWidth.htm)
 Свойство PixelWidth возвращает ширину экспортируемого значка в пикселях.

 [TabSheet](../../Interface/ITabIconExporter/ITabIconExporter.TabSheet.htm)
 Свойство TabSheet определяет таблицу, значок которой необходимо экспортировать.
