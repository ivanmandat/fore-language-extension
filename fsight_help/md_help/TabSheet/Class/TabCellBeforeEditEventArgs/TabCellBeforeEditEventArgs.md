# TabCellBeforeEditEventArgs


## Описание


Класс TabCellBeforeEditEventArgs
 предназначен для работы с параметрами события [TabSheetBox.OnBeforeEdit](../TabSheetBox/TabSheetBox.OnBeforeEdit.htm),
 происходящего перед началом редактирования ячейки.
