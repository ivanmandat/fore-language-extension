# TabCellBeforeEditBaseEventArgs.Create


## Синтаксис


Create(TabSheet: [ITabSheet](../../Interface/ITabSheet/ITabSheet.htm); Row: Integer; Column: Integer; Cancel: Boolean);


## Параметры


TabSheet - таблица компонента, для которого генерируется событие.


Row - индекс строки, в которой расположена ячейка, для которой генерируется событие.


Column - индекс столбца, в которой расположена ячейка, для которой генерируется событие.


Cancel - признак отмены редактирования ячейки.


## Описание


Конструктор Create создает аргумент события с указанным признаком отмены редактирования ячейки.
