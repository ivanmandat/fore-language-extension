# TabCellBeforeEditBaseEventArgs


Сборка: Tab;


## Описание


Класс TabCellBeforeEditBaseEventArgs
 реализует аргумент события, возникающего перед редактированием какой-либо
 ячейки таблицы компонента [LanerBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/LanerBox.htm).
