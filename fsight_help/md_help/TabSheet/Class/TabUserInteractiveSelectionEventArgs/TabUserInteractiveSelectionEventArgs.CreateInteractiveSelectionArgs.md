## Синтаксис


CreateInteractiveSelectionArgs(TabSheet: [ITabSheetBox](../../Interface/ITabSheetBox/ITabSheetBox.htm);

## Параметры


TabSheet. Компонент, для которого
 генерируется событие.


sel. Интерактивный диапазон
 ячеек.


Type. Разрешенные варианты
 изменения интерактивного диапазона.


## Описание


Конструктор CreateInteractiveSelectionArgs
 создает аргумент событий, связанных с изменением интерактивного диапазона
 ячеек.
