# TabInteractiveSelectionEventArgs.CreateInteractiveSelectionArgs


## Синтаксис


CreateInteractiveSelectionArgs(TabSheet: [ITabSheetBox](../../Interface/ITabSheetBox/ITabSheetBox.htm); Range: [ITabRange](../../Interface/ITabRange/ITabRange.htm); Type: [TabInteractiveSelectionType](../../Enums/TabInteractiveSelectionType.htm));


## Параметры


TabSheet. Компонент, для которого генерируется событие.


Range. Диапазон ячеек, для которого генерируется событие.


Type. Тип перемещения ячеек.


## Описание


Конструктор CreateInteractiveSelectionArgs создает экземпляр аргумента событий, происходящих при перемещении выделенной области ячеек в таблице.
