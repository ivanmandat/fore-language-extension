# TabObjectExporter


## Описание


Класс TabObjectExporter реализует объект, предназначенный для экспорта объектов таблицы.


## Свойства, унаследованные от [ITabObjectExporter](../../Interface/ITabObjectExporter/ITabObjectExporter.htm)


 Имя свойства
 Краткое описание


 ![](../../Property_Image.gif)
 [Height](../../Interface/ITabObjectExporter/ITabObjectExporter.Height.htm)
 Свойство Height определяет высоту, которую будет иметь экспортированное изображение объекта.


 ![](../../Property_Image.gif)
 [Object](../../Interface/ITabObjectExporter/ITabObjectExporter.Object.htm)
 Свойство Object определяет объект таблицы, который необходимо экспортировать.


 ![](../../Property_Image.gif)
 [Width](../../Interface/ITabObjectExporter/ITabObjectExporter.Width.htm)
 Свойство Width определяет ширину, которую будет иметь экспортированное изображение объекта.
