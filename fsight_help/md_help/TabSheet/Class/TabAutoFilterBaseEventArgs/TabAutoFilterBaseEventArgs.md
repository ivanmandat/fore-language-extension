## Описание


Класс TabAutoFilterBaseEventArgs
 реализует аргумент события, возникающего при изменении настроек автофильтра
 в таблице компонента [LanerBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/LanerBox.htm).
