# TabAutoFilterBaseEventArgs.Create


## Синтаксис


Create(TabSheet: [ITabSheet](../../Interface/ITabSheet/ITabSheet.htm); Column: Integer; Action: [TabAutoFilterAction](../../Enums/TabAutoFilterAction.htm));


## Параметры


TabSheet. Таблица компонента, для которого генерируется событие;


Column. Индекс столбца, для которого генерируется событие;


Action. Тип выбранного условия автофильтра для события.


## Описание


Конструктор Create создает аргумент события с указанным условием автофильтра.
