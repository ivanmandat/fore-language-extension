# TabCellBeforeChangeEventArgs


Класс TabCellBeforeChangeEventArgs
 предназначен для работы с параметрами события [TabSheetBox.OnBeforeCellChange](../TabSheetBox/TabSheetBox.OnBeforeCellChange.htm),
 происходящего перед изменением значения ячейки таблицы.
