# TabColumnResizeEventArgs.CreateColumnResizeArgs


## Синтаксис


CreateColumnResizeArgs(Sheet: [ITabSheetBox](../../Interface/ITabSheetBox/ITabSheetBox.htm); Column: Integer; ResizeType: [TabRowColumnResizeType](../../Enums/TabRowColumnResizeType.htm));


## Параметры


Sheet. Компонент, для которого генерируется событие.


Column. Столбец, для которого генерируется событие.


ResizeType. Действие, в результате которого изменяется размер столбца.


## Описание


Конструктор CreateColumnResizeArgs создает аргумент события, происходящего после изменения размера столбца таблицы.
