# TabObjectResizingEventArgs


Сборка: Tab;


## Описание


Класс TabObjectResizingEventArgs
 реализует аргумент события, происходящего при каждом изменении размера
 объекта.


## Комментарии


Для реализации аргумента события, происходящего при перемещении объекта,
 используйте класс [TabObjectMovingEventArgs](../TabObjectMovingEventArgs/TabObjectMovingEventArgs.htm).
