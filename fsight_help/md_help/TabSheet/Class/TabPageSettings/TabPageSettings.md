# TabPageSettings


## Описание


Класс TabPageSettings содержит
 свойства и методы параметров страницы таблицы.


## Свойства, унаследованные от [ITabPageSettings](../../Interface/ITabPageSettings/ITabPageSettings.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BlackAndWhite](../../Interface/ITabPageSettings/ITabPageSettings.BlackAndWhite.htm)


		 Свойство BlackAndWhite
		 определяет, будет ли печататься оформление таблицы при печати
		 на принтере.


		 ![](../../Property_Image.gif)
		 [CenterHorizontal](../../Interface/ITabPageSettings/ITabPageSettings.CenterHorizontal.htm)


		 Свойство CenterHorizontal
		 определяет, будет ли таблица при печати центрироваться по горизонтали.


		 ![](../../Property_Image.gif)
		 [CenterVertical](../../Interface/ITabPageSettings/ITabPageSettings.CenterVertical.htm)


		 Свойство CenterVertical
		 определяет, будет ли таблица при печати центрироваться по вертикали.


		 ![](../../Property_Image.gif)
		 [ColumnsBreaks](../../Interface/ITabPageSettings/ITabPageSettings.ColumnsBreaks.htm)


		 Свойство ColumnsBreaks
		 возвращает настройки разрывов по столбцам для текущей страницы.


		 ![](../../Property_Image.gif)
		 [ContinuousColumns](../../Interface/ITabPageSettings/ITabPageSettings.ContinuousColumns.htm)


		 Свойство ContinuousColumns
		 возвращает настройки неразрывного диапазона по столбцам для текущей
		 страницы.


		 ![](../../Property_Image.gif)
		 [ContinuousRows](../../Interface/ITabPageSettings/ITabPageSettings.ContinuousRows.htm)


		 Свойство ContinuousRows
		 возвращает настройки неразрывного диапазона по строкам для текущей
		 страницы.


		 ![](../../Property_Image.gif)
		 [PrintHyperlinks](../../Interface/ITabPageSettings/ITabPageSettings.PrintHyperlinks.htm)


		 Свойство PrintHyperlinks
		 определяет, будут ли экспортироваться гиперссылки в формат HTML.


		 ![](../../Property_Image.gif)
		 [PrintOrder](../../Interface/ITabPageSettings/ITabPageSettings.PrintOrder.htm)


		 Свойство PrintOrder
		 определяет порядок вывода на печать.


		 ![](../../Property_Image.gif)
		 [Range](../../Interface/ITabPageSettings/ITabPageSettings.Range.htm)


		 Свойство Range определяет
		 диапазон печатаемых ячеек.


		 ![](../../Property_Image.gif)
		 [RowsBreaks](../../Interface/ITabPageSettings/ITabPageSettings.RowsBreaks.htm)


		 Свойство RowsBreaks
		 возвращает настройки разрывов по строкам для текущей страницы.


		 ![](../../Property_Image.gif)
		 [Scale](../../Interface/ITabPageSettings/ITabPageSettings.Scale.htm)


		 Свойство Scale возвращает
		 настройки масштаба при печати таблицы.


		 ![](../../Property_Image.gif)
		 [TitleColumns](../../Interface/ITabPageSettings/ITabPageSettings.TitleColumns.htm)


		 Свойство TitleColumns
		 определяет диапазон сквозных столбцов.


		 ![](../../Property_Image.gif)
		 [TitleColumnsScope](../../Interface/ITabPageSettings/ITabPageSettings.TitleColumnsScope.htm)


		 Свойство TitleColumnsScope
		 определяет диапазон окончания действия сквозных столбцов.


		 ![](../../Property_Image.gif)
		 [TitleRows](../../Interface/ITabPageSettings/ITabPageSettings.TitleRows.htm)


		 Свойство TitleRows
		 определяет диапазон сквозных строк.


		 ![](../../Property_Image.gif)
		 [TitleRowsScope](../../Interface/ITabPageSettings/ITabPageSettings.TitleRowsScope.htm)


		 Свойство TitleRowsScope
		 определяет диапазон окончания действия сквозных строк.
