# TabCellBaseEventArgs.CreateTabCellArgs


## Синтаксис


CreateTabCellArgs(TabSheet: [ITabSheet](../../Interface/ITabSheet/ITabSheet.htm); Row: Integer; Column: Integer);


## Параметры


TabSheet - таблица компонента, для которого генерируется событие.


Row - индекс строки, в которой расположена ячейка, для которой генерируется событие.


Column - индекс столбца, в которой расположена ячейка, для которой генерируется событие.


## Описание


Конструктор CreateTabCellArgs создает аргумент событий, связанных с какой-либо ячейкой таблицы.
