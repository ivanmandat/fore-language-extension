# TabTableStyles


## Описание


Класс TabTableStyles содержит
 свойства и методы для работы с коллекцией стилей оформления таблицы.


## Свойства, унаследованные от [ITabTableStyles](../../Interface/ITabTableStyles/ITabTableStyles.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](../../Interface/ITabTableStyles/ITabTableStyles.Count.htm)
		 Свойство Count возвращает
		 количество стилей оформления в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](../../Interface/ITabTableStyles/ITabTableStyles.Item.htm)
		 Свойство Item возвращает
		 указанный стиль оформления из коллекции.
