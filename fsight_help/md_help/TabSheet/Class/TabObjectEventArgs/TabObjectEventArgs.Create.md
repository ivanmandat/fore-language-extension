# TabObjectEventArgs.Create


## Синтаксис


Create(TabSheet: [ITabSheetBox](../../Interface/ITabSheetBox/ITabSheetBox.htm);
 Object: [ITabObject](../../Interface/ITabObject/ITabObject.htm);
 Action: [TabObjectAction](../../Enums/TabObjectAction.htm));


## Параметры


TabSheet -
 объект [TabSheetBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/TabSheetBox.htm), для которого
 возникает событие.


Action - тип действия, совершенного
 над объектом таблицы.


Object - объект, для которого
 возникает событие.


## Описание


Конструктор Create создает аргумент
 события, возникающего при активации/деактивации объекта таблицы.
