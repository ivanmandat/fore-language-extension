# TabObjectEventArgs


## Описание


Класс TabObjectEventArgs предназначен
 для работы с параметрами события [TabSheetBox.OnObjectActivate](../TabSheetBox/TabSheetBox.OnObjectActivate.htm),
 наступающего при активации/деактивации объекта таблицы.
