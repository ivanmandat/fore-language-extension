# TabSheet


## Описание


Класс TabSheet содержит свойства
 и методы таблицы. Таблицы используются в регламентных и экспресс-отчетах,
 а также реализуются компонентами среды разработки [UiTabSheet](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiTabSheet.htm)
 и визуальным компонентом [TabSheetBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/TabSheetBox.htm).
