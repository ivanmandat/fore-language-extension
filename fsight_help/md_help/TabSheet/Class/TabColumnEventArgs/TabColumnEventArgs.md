# TabColumnEventArgs


## Описание


Класс TabColumnEventArgs предназначен
 для работы с параметрами события [TabSheetBox.OnColumnClick](../TabSheetBox/TabSheetBox.OnColumnClick.htm),
 происходящего при щелчке левой кнопкой мыши по заголовку столбца таблицы.
