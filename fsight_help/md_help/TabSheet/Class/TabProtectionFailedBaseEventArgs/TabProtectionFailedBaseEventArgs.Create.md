# TabProtectionFailedBaseEventArgs.Create


## Синтаксис


Create(TabSheet: [ITabSheet](../../Interface/ITabSheet/ITabSheet.htm); Target: Variant; Message: String);


## Параметры


TabSheet - таблица компонента, для которого генерируется событие.


Target - параметр не используется.


Message - сообщение о нарушении защиты.


## Описание


Конструктор Create создает аргумент события с указанным текстом нарушения защиты.
