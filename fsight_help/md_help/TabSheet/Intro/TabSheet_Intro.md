# Введение в сборку Tab


В данном разделе представлено описание сборки Tab,
 содержащей средства для работы с таблицей на языке Fore. Таблица
 может быть получена из отчета или создана пользователем. В интерфейсах
 сборки реализованы все необходимы средства для настройки стиля и внешнего
 вида таблицы, для экспорта и печати таблицы, для работы с данными в таблице
 и для прочих операций с ней.


Более подробно с работой сборки и её иерархией рассказано в разделах:


	- [общие принципы программирования
	 с использованием сборки Tab](Programming_principles.htm);


	- [иерархия сборки Tab](Hierarchy/TabSheet_hierarchy.htm).


Основным интерфейсом сборки Tab
 является интерфейс [ITabSheet](../Interface/ITabSheet/ITabSheet.htm).
