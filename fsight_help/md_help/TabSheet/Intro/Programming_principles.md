# Общие принципы программирования
 с использованием сборки Tab


Сборка Tab предназначена для
 работой с таблицей данных. Таблица может быть создана пользователем или
 получена из регламентного или экспресс отчета.


Программирование на Fore с
 использованием сборки Tab включает
 следующие аспекты:


	- [Получение таблицы](Get_Table.htm).


	- Работа с таблицей:


		- [настройка стиля таблицы](TableStyle.htm);


		- [настройка внешнего вида таблицы](TabView.htm);


		- [применение условного форматирования](FormatConditions.htm);


		- [добавление объектов в таблицу](TabObjects.htm);


		- [поиск данных](TabSearch.htm);


		- [настройка сортировки](CustomSort.htm);


		- [печать таблицы](SheetPrinter.htm);


		- [экспорт таблицы](SheetExporter.htm) и прочие
		 операции с таблицей.
