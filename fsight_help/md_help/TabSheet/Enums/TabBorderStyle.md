# TabBorderStyle


## Описание


Перечисление TabBorderStyle
 содержит типы линий границы ячеек.


Используется следующим свойством:


	- [ITabCellStyle.BorderStyle](../Interface/ITabCellStyle/ITabCellStyle.BorderStyle.htm);


	- [ITabUserInteractiveSelection.BorderStyle](../Interface/ITabUserInteractiveSelection/ITabUserInteractiveSelection.BorderStyle.htm);


	- [ITabRegion.BorderStyle](../Interface/ITabRegion/ITabRegion.BorderStyle.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 -1
		 Undefined.
		 Значение не определено.


		 0
		 Continuous.
		 Непрерывная линия.


		 1
		 Dash.
		 Пунктирная линия.


		 2
		 DashDot. Точка тире.


		 3
		 DashDotDot.
		 Точка точка тире.


		 4
		 Dot. Точка.


		 5
		 Double. Двойная линия.


		 6
		 SlantDashDot.
		 Зарезервировано на будущее. В данный момент при установке данного
		 стиля будет устанавливаться непрерывный стиль линии.


		 7
		 LineStyleNone.
		 Нет линии.
