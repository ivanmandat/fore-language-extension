# TabCellSearchTarget


## Описание


Перечисление TabCellSearchTarget
 содержит варианты свойства ячейки, по которым будет осуществляться поиск.


Используется следующим свойством:


	- [ITabCellSearch.Target](../Interface/ITabCellSearch/ITabCellSearch.Target.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 Value. Поиск
		 по значению в ячейке.


		 2
		 Formula.
		 Поиск по формуле в ячейке.


		 4
		 Text.
		 Поиск по тексту в ячейке.


		 8
		 MetaData. Поиск по
		 метаданным ячейки.
