# TabHyperlinkObjectType


## Описание


Перечисление TabHyperlinkObjectType содержит типы элементов, расположенных в ячейке с гиперссылкой.


Используется следующими свойствами:


-
[IReportHyperlinkClickEventArgs.Type](KeReport.chm::/Interface/IReportHyperlinkClickEventArgs/IReportHyperlinkClickEventArgs.Type.htm);


-
[ITabHyperlinkClickEventArgs.Type](../Interface/ITabHyperlinkClickEventArgs/ITabHyperlinkClickEventArgs.Type.htm).
