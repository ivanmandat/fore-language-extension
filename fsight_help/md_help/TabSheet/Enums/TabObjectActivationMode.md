# TabObjectActivationMode


## Описание


Перечисление TabObjectActivationMode
 содержит режимы активации объектов таблицы.


Используется следующим свойством:


	- [ITabObject.ActivationMode](../Interface/ITabObject/ITabObject.ActivationMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 MouseFly.
		 На движение мышью поверх объекта.


		 1
		 Selection.
		 На выделение объекта.


		 2
		 DoubleClick.
		 На двойной щелчок.


		 3
		 Never.
		 Запрет активации.


		 4
		 OnPreview.
		 При открытии отчёта.
