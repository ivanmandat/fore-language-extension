# TabFootnotesNumberingRule


## Описание


Перечисление TabFootnotesNumberingRule
 содержит правила нумерации сносок при переходе к таблицам других листов.


Используется следующим свойством:


	- [ITabFootnotes.NumberingRule](../Interface/ITabFootnotes/ITabFootnotes.NumberingRule.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Continue. Продолжать
		 нумерацию сносок при переходе к таблицам других листов.


		 1
		 Restart. На каждом
		 листе производится собственная нумерация сносок. Начальное значение
		 указывается в свойстве [ITabFootnotes.StartingNumber](../Interface/ITabFootnotes/ITabFootnotes.StartingNumber.htm).
