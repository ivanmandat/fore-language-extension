# TabInsertShiftDirection


## Описание


Перечисление TabInsertShiftDirection
 используется для определения способа вставки диапазона ячеек.


Используется следующими методами, свойствами, конструкторами и событиями:


	- [ITabRange.Insert](../Interface/ITabRange/ITabRange.Insert.htm);


	- [TabRangeInsertEventArgs.Create](../Class/TabRangeInsertEventArgs/TabRangeInsertEventArgs.Create.htm);


	- [ITabRangeInsertEventArgs.ShiftDirection](../Interface/ITabRangeInsertEventArgs/ITabRangeInsertEventArgs.ShiftDirection.htm);


	- [TabSheetBox.OnAfterInsertRange](../Class/TabSheetBox/TabSheetBox.OnAfterInsertRange.htm);


	- [TabSheetBox.OnBeforeInsertRange](../Class/TabSheetBox/TabSheetBox.OnBeforeInsertRange.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 ShiftToRight. Вставка
		 со сдвигом ячеек вправо.


		 1
		 ShiftDown. Вставка
		 со сдвигом ячеек вниз.
