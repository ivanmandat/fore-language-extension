# TabCellContentChange


## Описание


Перечисление TabCellContentChange
 содержит типы изменения в ячейке таблицы.


Используется следующими свойствами:


	- [ITabCellContentChangeEventArgs.Type](../Interface/ITabCellContentChangeEventArgs/ITabCellContentChangeEventArgs.Type.htm);


	- [IReportCellContentChangeEventArgs.Type](KeReport.chm::/Interface/IReportCellContentChangeEventArgs/IReportCellContentChangeEventArgs.Type.htm).
