# FilterItemState


## Описание


Перечисление FilterItemState
 содержит типы фильтрации.


Используется следующими свойствами и методами:


	- [ITableFilterItem.State](../Interface/ITableFilterItem/ITableFilterItem.State.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Inactive. Фильтрация
		 отключена.


		 1
		 Limit. Фильтрация
		 производится по указанному количеству элементов списка (тип фильтрации
		 первые N).


		 2
		 Condition. Фильтрация
		 производится по выбранному условию.
