# TabFormatWordWrap


## Описание


Перечисление TabFormatWordWrap
 содержит способы переноса текста в ячейках таблицы.


Используется следующим свойством:


	- [ITabCellStyle.WrapText](../Interface/ITabCellStyle/ITabCellStyle.WrapText.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 -1
		 Undefined.
		 Не известно.


		 0
		 None.
		 Не переносить.


		 1
		 BreakWords.
		 Текст по словам.


		 2
		 Syllable.
		 Слово по слогам.
