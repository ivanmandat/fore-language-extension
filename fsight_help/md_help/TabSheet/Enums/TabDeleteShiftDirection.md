# TabDeleteShiftDirection


## Описание


Перечисление TabDeleteShiftDirection
 используется для определения способа удаления диапазона ячеек.


Используется следующими методами, свойствами, конструкторами и событиями:


	- [ITabRange.Delete](../Interface/ITabRange/ITabRange.Delete.htm);


	- [TabRangeDeleteEventArgs.Create](../Class/TabRangeDeleteEventArgs/TabRangeDeleteEventArgs.Create.htm);


	- [ITabRangeDeleteEventArgs.ShiftDirection](../Interface/ITabRangeDeleteEventArgs/ITabRangeDeleteEventArgs.ShiftDirection.htm);


	- [TabSheetBox.OnAfterDeleteRange](../Class/TabSheetBox/TabSheetBox.OnAfterDeleteRange.htm);


	- [TabSheetBox.OnBeforeDeleteRange](../Class/TabSheetBox/TabSheetBox.OnBeforeDeleteRange.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 ShiftToLeft. Удаление
		 со сдвигом ячеек влево.


		 1
		 ShiftToUp. Удаление
		 со сдвигом ячеек вверх.
