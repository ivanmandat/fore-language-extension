# TabViewArea


## Описание


Перечисление TabViewArea содержит
 варианты областей таблицы, в которых расположена точка.


Используется следующим методом:


	- [ITabView.HitTest](../Interface/ITabView/ITabView.HitTest.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Client.
		 Область ячеек.


		 1
		 TopLeftCorner.
		 Верхний левый угол.


		 2
		 ColumnsHeader.
		 Заголовки столбцов.


		 3
		 RowsHeader.
		 Заголовки строк.
