# TabExpanderKind


## Описание


Перечисление TabExpanderKind
 содержит области действия экспандеров.


Используется следующими свойствами и методами:


	- [ITabSheet.ExpanderKind](../Interface/ITabSheet/ITabSheet.ExpanderKind.htm);


	- [ITabSheet.CreateExpander](../Interface/ITabSheet/ITabSheet.CreateExpander.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Columns. Экспандер,
		 раскрывающийся по столбцам.


		 1
		 Rows. Экспандер, раскрывающийся
		 по строкам.
