# FilterRelation


## Описание


Перечисление FilterRelation
 содержит логические связки условия со следующим за ним.


Используется следующим свойством:


	- [ITableFilterCondition.Relation](../Interface/ITableFilterCondition/ITableFilterCondition.Relation.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 And. Объединение по логическому И (And).


		 1
		 Or. Объединение по логическому ИЛИ (Or).
