# TabObjectFlip


## Описание


Перечисление TabObjectFlip содержит
 типы отражения объекта таблицы.


Используется следующим методом:


	- [ITabObject.Flip](../Interface/ITabObject/ITabObject.Flip.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Horizontal. Отражение
		 по горизонтали.


		 1
		 Vertical. Отражение
		 по вертикали.
