# TableFilterFeatures


## Описание


Перечисление TableFilterFeatures
 содержит типы настроек в меню фильтрации в таблице.


Используется следующими свойствами и методами:


	- [ITableFilter.Features](../Interface/ITableFilter/ITableFilter.Features.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Default_. Значение
		 по умолчанию. Отображение всех настроек меню.


		 1
		 HideSorting. Скрытие
		 настроек сортировки.


		 2
		 HideCondition. Скрытие
		 настроек условия «Первые N».


		 4
		 HideSettings. Скрытие
		 настроек фильтрации.
