# TabAccessRights


## Описание


Перечисление TabAccessRights
 содержит варианты прав доступа к данным.


Используется следующим свойством:


	- [ITabCellStyle.AccessRights](../Interface/ITabCellStyle/ITabCellStyle.AccessRights.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 -1
		 Undefined.
		 Права доступа не определены.


		 0
		 NoAccess.
		 Нет прав доступа.


		 1
		 Read.
		 Только право на чтение.


		 2
		 Write.
		 Только право на запись.


		 3
		 FullAccess.
		 По умолчанию. Полный доступ.
