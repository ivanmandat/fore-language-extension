# FilterCondition


## Описание


Перечисление FilterCondition
 содержит условия, по которым будут отбираться данные.


Используется следующими свойством:


	- [ITableFilterCondition.Condition](../Interface/ITableFilterCondition/ITableFilterCondition.Condition.htm)
