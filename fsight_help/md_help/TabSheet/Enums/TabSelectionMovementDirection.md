# TabSelectionMovementDirection


## Описание


Перечисление TabSelectionMovementDirection
 содержит направления перехода выделения после нажатия клавиши ENTER.


Используется следующим свойством:


	- [ITabView.SelectionMovementDirection](../Interface/ITabView/ITabView.SelectionMovementDirection.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Down.
		 Вниз.


		 1
		 Up. Вверх.


		 2
		 Left.
		 Влево.


		 3
		 Right.
		 Вправо.


		 4
		 None. Без перехода.
