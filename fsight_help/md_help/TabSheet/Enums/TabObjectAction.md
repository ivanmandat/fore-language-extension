# TabObjectAction


## Описание


Перечисление TabObjectAction
 содержит типы действия, совершенного над объектом таблицы.


Используется следующим свойством:


	- [ITabObjectEventArgs.Action](../Interface/ITabObjectEventArgs/ITabObjectEventArgs.Action.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Activate.
		 Активация.


		 1
		 Deactivate.
		 Деактивация.
