# TabCustomSortType


## Описание


Перечисление TabCustomSortType
 содержит типы сортировки, применяемой к строкам/столбцам таблицы данных.


Используется следующим свойством:


	- [ITabCustomSortItem.Type](../Interface/ITabCustomSortItem/ITabCustomSortItem.Type.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Value. Сортировка по
		 значению в ячейке.


		 1
		 Text. Сортировка по
		 тексту в ячейке.
