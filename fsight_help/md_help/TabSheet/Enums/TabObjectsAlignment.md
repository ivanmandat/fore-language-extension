# TabObjectsAlignment


## Описание


Перечисление TabObjectsAlignment
 содержит способы выравнивания нескольких выделенных объектов.


Используется следующим методом:


	- [ITabObjects.AlignSelected](../Interface/ITabObjects/ITabObjects.AlignSelected.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Left.
		 По левому краю.


		 1
		 Right.
		 По правому краю.


		 2
		 HorizontalCenter.
		 По горизонтальному центру.


		 3
		 Top. По
		 верхнему краю.


		 4
		 Bottom.
		 По нижнему краю.


		 5
		 VerticalCenter.
		 По вертикальному центру.


		 6
		 DistributeHorizontal.
		 Распределить по горизонтали.


		 7
		 DistributeVertical.
		 Распределить по вертикали.
