# TabPasteMode


## Описание


Перечисление TabPasteMode содержит
 режимы специальной вставки.


Используется следующими методами:


	- [ITabSheet.PasteEx](../Interface/ITabSheet/ITabSheet.PasteEx.htm);


	- [ITabSheet.PasteInRangeEx](../Interface/ITabSheet/ITabSheet.PasteInRangeEx.htm);


	- [ITabRange.PasteFromStreamEx](../Interface/ITabRange/ITabRange.PasteFromStreamEx.htm).
