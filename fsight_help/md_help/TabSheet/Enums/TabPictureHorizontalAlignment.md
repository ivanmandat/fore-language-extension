# TabPictureHorizontalAlignment


## Описание


Перечисление TabPictureHorizontalAlignment содержит варианты ориентаций изображений в ячейке по горизонтали.


Используется следующим свойством:


-
[ITabCellStyle.PictureHorizontalAlignment](../Interface/ITabCellStyle/ITabCellStyle.PictureHorizontalAlignment.htm).


## Допустимые значения


 Значение
 Краткое описание


 -1
 Undefined. Не известно.


 0
 Left. Слева.


 1
 Right. Справа.


 2
 Center. По центру.
