# TabRangeToArrayFlags


## Описание


Перечисление TabRangeToArrayFlags
 содержит флаги для преобразования диапазона.


Используется следующим свойством:


	- [ITabRange.ToDoubleArrayEx](../Interface/ITabRange/ITabRange.ToDoubleArrayEx.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None.
		 Без особенностей.


		 1
		 SkipNonNumber.
		 Исключать нечисловые значения.


		 2
		 ExcludeHidden.
		 Исключать спрятанные ячейки.


		 4
		 IncludeCollapsed.
		 Включать значения свернутых экспандеров.
