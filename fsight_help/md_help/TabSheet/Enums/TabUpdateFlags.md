# TabUpdateFlags


## Описание


Перечисление TabUpdateFlags
 содержит режимы обновления таблицы.


Используется следующими свойствами и методами:


	- [ITabSheet.BeginUpdate](../Interface/ITabSheet/ITabSheet.BeginUpdate.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Default_. Значение
		 по умолчанию.


		 1
		 DisableUndoRedo. Отключение
		 формирования стека изменений.
