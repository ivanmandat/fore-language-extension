# TabObjectChangeType


## Описание


Перечисление TabObjectChangeType
 используется для определения типа изменения, которое происходит с объектом.


Используется следующими свойствами и конструкторами:


	- [ITabObjectChangeEventArgs.Type](../Interface/ITabObjectChangeEventArgs/ITabObjectChangeEventArgs.Type.htm);


	- [TabObjectChangeEventArgs.CreateObjectChangeArgs](../Class/TabObjectChangeEventArgs/TabObjectChangeEventArgs.CreateObjectChangeArgs.htm);


	- [IReportObjectChangeEventArgs](KeReport.chm::/Interface/IReportObjectChangeEventArgs/IReportObjectChangeEventArgs.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Rect. Объект перемещают
		 или изменяют его размер.


		 1
		 Angle. Объект вращают.
