# TableFilterIconState


## Описание


Перечисление TableFilterIconState
 содержит состояния пиктограммы.


Используется следующим свойством:


	- [ITableFilterItem.Icon](../Interface/ITableFilterItem/ITableFilterItem.Icon.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Normal. Состояние,
		 установленное по умолчанию - отсутствует фильтрация и сортировка.


		 1
		 Filtered. Фильтрация
		 включена.


		 2
		 Sorted. Элементы
		 отсортированы (по умолчанию вид сортировки - по убыванию).


		 4
		 Pressed. Кнопка
		 находится в нажатом состоянии.


		 8
		 Ascending. Элементы
		 отсортированы по возрастанию.
