# TabHyperlinkActionType


## Описание


Перечисление TabHyperlinkActionType
 содержит типы действий, выполняемого при щелчке по гиперссылке.


Используется следующими свойствами и методами:


	- [ITabHyperlink.ActionType](../Interface/ITabHyperlink/ITabHyperlink.ActionType.htm);


	- [IPrxDimensionDrill.ActionType](kereport.chm::/interface/iprxdimensiondrill/iprxdimensiondrill.actiontype.htm);


	- [IEaxDrillSettings.ActionType](KeExpress.chm::/Interface/IEaxDrillSettings/IEaxDrillSettings.ActionType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 -1
		 None. Не определено.


		 1
		 OpenFile. Открыть файл.


		 2
		 OpenURL. Открыть ссылку.


		 3
		 GoToSheet. Открыть
		 лист отчета.


		 4
		 ShowRange. Показать
		 диапазон ячеек.


		 5
		 ShowObject. Показать
		 объект в центре экрана.


		 6
		 OpenObject. Открыть
		 объект репозитория.


		 7
		 RunMacros. Выполнить
		 процедуру/функцию.
