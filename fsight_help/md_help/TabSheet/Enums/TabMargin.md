# TabMargin


## Описание


Перечисление TabMargin содержит
 варианты границ, от которых может быть установлен отступ.


Используется следующим свойством:


	- [ITabCellStyle.Margins](../Interface/ITabCellStyle/ITabCellStyle.Margins.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Left.
		 Левая.


		 1
		 Top. Верхняя.


		 2
		 Right.
		 Правая.


		 3
		 Bottom.
		 Нижняя.
