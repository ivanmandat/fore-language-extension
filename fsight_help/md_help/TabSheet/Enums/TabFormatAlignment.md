# TabFormatAlignment


## Описание


Перечисление TabFormatAlignment
 содержит варианты выравниваний текста ячейки по горизонтали.


Используется следующим свойством:


	- [ITabCellStyle.HorizontalAlignment](../Interface/ITabCellStyle/ITabCellStyle.HorizontalAlignment.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 -1
		 Undefined. Не известно.


		 0
		 General. По значению.


		 1
		 Left. По левому краю.


		 2
		 Center. По центру.


		 3
		 Right. По правому краю.


		 4
		 Justify. По ширине.
