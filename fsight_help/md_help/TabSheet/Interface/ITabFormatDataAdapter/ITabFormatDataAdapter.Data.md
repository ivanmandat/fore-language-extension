# ITabFormatDataAdapter.Data


## Синтаксис


Data(Row: Integer; Column: Integer): Variant;


## Параметры


Row. Индекс строки;


Column. Индекс столбца.


## Описание


Свойство Data возвращает данные
 источника условного формата ячеек.

 приведена в примере [ITabFormatScale.TargetType](../ITabFormatScale/ITabFormatScale.TargetType.htm).
