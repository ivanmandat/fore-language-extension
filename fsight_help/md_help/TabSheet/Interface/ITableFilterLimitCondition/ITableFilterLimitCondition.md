# ITableFilterLimitCondition


Сборка: Tab;


## Описание


Интерфейс ITableFilterLimitCondition
 содержит свойства для управления условиями типа фильтрации «Первые
 N».


## Иерархия наследования


           ITableFilterLimitCondition


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITableFilterLimitCondition.Count.htm)
		 Свойство Count определяет
		 количество элементов для отображения.


		 ![](../../Property_Image.gif)
		 [HighestValues](ITableFilterLimitCondition.HighestValues.htm)
		 Свойство HighestValues
		 определяет отображение наибольших или наименьших значений.


		 ![](../../Property_Image.gif)
		 [Percentage](ITableFilterLimitCondition.Percentage.htm)
		 Свойство Percentage
		 определяет способ задания количества элементов для отображения
		 в абсолютном значении или процентах от общего количества.
