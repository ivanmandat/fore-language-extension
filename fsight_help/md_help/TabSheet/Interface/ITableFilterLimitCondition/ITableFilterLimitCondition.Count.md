# ITableFilterLimitCondition.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count определяет количество
 элементов для отображения.


## Комментарии


Для определения способа задания количества элементов для отображения
 используйте свойство [ITableFilterLimitCondition.Percentage](ITableFilterLimitCondition.Percentage.htm).
