# ITabRegions.IndexOf


## Синтаксис


IndexOf(Item: [ITabRegion](../ITabRegion/ITabRegion.htm)):
 Integer;


## Параметры


Item — регион,
 индекс которого необходимо получить.


## Описание


Метод IndexOf возвращает индекс
 региона, который передается в качестве входного параметра Item.
