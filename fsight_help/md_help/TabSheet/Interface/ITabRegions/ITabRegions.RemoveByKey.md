# ITabRegions.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key — ключ
 регион, который необходимо удалить.


## Описание


Метод RemoveByKey удаляет регион,
 ключ которого передается в качестве входного параметра Value.
