# ITabRegions.RemoveAt


## Синтаксис


RemoveAt(Index: Integer): Boolean;


## Параметры


Index — индекс
 региона, который необходимо удалить из коллекции.


## Описание


Метод RemoveAt удаляет регион,
 индекс которого передается в качестве входного параметра Index.
