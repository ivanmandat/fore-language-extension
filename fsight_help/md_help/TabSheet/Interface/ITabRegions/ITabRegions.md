# ITabRegions


Сборка: Tab;


## Описание


Интерфейс ITabRegions предназначен
 для работы с коллекцией регионов таблицы.


## Иерархия наследования


ITabRegions


## Комментарии


Доступ к коллекции регионов предоставляет свойство [ITabSheet.Regions](../ITabSheet/ITabSheet.Regions.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITabRegions.Count.htm)
		 Свойство Count возвращает
		 количество регионов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ITabRegions.Item.htm)
		 Свойство Item возвращает
		 регион.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)

		 ![](../../Sub_Image.gif)
		 [Clear](ITabRegions.Clear.htm)

		 ![](../../Sub_Image.gif)
		 [Remove](ITabRegions.Remove.htm)

		 Метод RemoveAt осуществляет

		 [RemoveByKey](ITabRegions.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
