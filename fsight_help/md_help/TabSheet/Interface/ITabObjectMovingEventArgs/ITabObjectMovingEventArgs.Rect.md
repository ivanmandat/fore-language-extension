# ITabObjectMovingEventArgs.Rect


## Синтаксис


		Rect: [IGxRectF](ModDrawing.chm::/Interface/IGxRectF/IGxRectF.htm);


## Описание


Свойство Rect возвращает координаты
 прямоугольной области, занимаемой объектом.


## Комментарии


Для получения изменяемого объекта используйте свойство [ITabObjectMovingEventArgs.Object](ITabObjectMovingEventArgs.Object.htm).
