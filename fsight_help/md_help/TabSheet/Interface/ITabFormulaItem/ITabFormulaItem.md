# ITabFormulaItem


## Описание


Интерфейс ITabFormulaItem содержит
 базовые свойства и методы области формул.


## Комментарии


В текущей реализации область формул используются в регламентных отчетах.
 Для работы с областью формул регламентного отчета предназначен интерфейс
 [IPrxFormulaIsland](KeReport.chm::/Interface/IPrxFormulaIsland/IPrxFormulaIsland.htm).
