## Описание


Свойство ShiftDirection возвращает
 способ, который использовался при удалении диапазона ячеек.


## Комментарии


Для получения параметров диапазона ячеек используйте свойство [ITabRangeDeleteEventArgs.Range](ITabRangeDeleteEventArgs.Range.htm).
