# ITabRangeDeleteEventArgs.Range


## Синтаксис


Range: [ITabRange](../ITabRange/ITabRange.htm);


## Описание


Свойство Range возвращает параметры
 диапазона ячеек таблицы.


## Комментарии


Для получения способа, который использовался при удалении ячеек таблицы,
 используйте свойство [ITabRangeDeleteEventArgs.ShiftDirection](ITabRangeDeleteEventArgs.ShiftDirection.htm).
