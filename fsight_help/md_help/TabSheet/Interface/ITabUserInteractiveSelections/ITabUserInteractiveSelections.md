# ITabUserInteractiveSelections


Сборка: Tab;


## Описание


Интерфейс ITabUserInteractiveSelections
 содержит свойства и методы для работы с коллекцией интерактивных диапазонов
 ячеек.


## Иерархия наследования


ITabUserInteractiveSelections


## Комментарии


Коллекцию интерактивных диапазонов ячеек можно получить в свойстве [ITabView.UserIntearactiveSelections](../ITabView/ITabView.UserIntearactiveSelections.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITabUserInteractiveSelections.Count.htm)
		 Свойство Count возвращает
		 количество интерактивных диапазонов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ITabUserInteractiveSelections.Item.htm)
		 Свойство Item возвращает
		 настройки интерактивного диапазона с указанным индексом.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ITabUserInteractiveSelections.Add.htm)
		 Метод Add создает интерактивный
		 диапазон и возвращает его настройки.


		 ![](../../Sub_Image.gif)
		 [Clear](ITabUserInteractiveSelections.Clear.htm)
		 Метод Clear удаляет
		 все интерактивные диапазоны из коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](ITabUserInteractiveSelections.Remove.htm)
		 Метод Remove удаляет
		 интерактивный диапазон с указанным индексом и возвращает признак
		 удачного удаления.
