# ITabUserInteractiveSelections.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index.
 Индекс интерактивного диапазона.


## Описание


Метод Remove удаляет интерактивный
 диапазон с указанным индексом и возвращает признак удачного удаления.


## Комментарии


Метод возвращает значение True,
 если интерактивный диапазон был удалён и False,
 если удаление по каким-либо причинам невозможно.
