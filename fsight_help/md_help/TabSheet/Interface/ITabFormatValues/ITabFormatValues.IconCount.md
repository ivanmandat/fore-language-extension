# ITabFormatValues.IconCount


## Синтаксис


IconCount: Integer;


## Описание


Свойство IconCount возвращает
 количество пиктограмм, используемых при форматировании. Актуально, если
 свойству [Style](ITabFormatValues.Style.htm) установлено значение
 IconSets.
