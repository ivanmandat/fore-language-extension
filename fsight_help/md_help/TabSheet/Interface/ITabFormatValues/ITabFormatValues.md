# ITabFormatValues


## Описание


Интерфейс ITabFormatValues содержит
 свойства условного формата, осуществляющего форматирование ячеек на основе
 их значений.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [IconCount](ITabFormatValues.IconCount.htm)
		 Свойство IconCount
		 возвращает количество пиктограмм, используемых при форматировании.


		 ![](../../Property_Image.gif)
		 [IconType](ITabFormatValues.IconType.htm)
		 Свойство IconType определяет
		 тип пиктограмм, используемых при форматирования ячеек.


		 ![](../../Property_Image.gif)
		 [MaxValue](ITabFormatValues.MaxValue.htm)
		 Свойство MaxValue определяет
		 значение условия, по которому будут отбираться ячейки с максимальными
		 значениями.


		 ![](../../Property_Image.gif)
		 [MaxValueColor](ITabFormatValues.MaxValueColor.htm)
		 Свойство MaxValueColor
		 определяет конечный цвет градиента, используемый для оформления
		 ячеек.


		 ![](../../Property_Image.gif)
		 [MaxValueType](ITabFormatValues.MaxValueType.htm)
		 Свойство MaxValueType
		 определяет тип условия, по которому будут отбираться ячейки с
		 максимальными значениями.


		 ![](../../Property_Image.gif)
		 [MidValue](ITabFormatValues.MidValue.htm)
		 Свойство MidValue определяет
		 условие, по которому устанавливается среднее значение для ячеек,
		 попадающих под форматирование.


		 ![](../../Property_Image.gif)
		 [MidValueColor](ITabFormatValues.MidValueColor.htm)
		 Свойство MidValueColor
		 определяет средний цвет градиента либо цвет гистограммы, в зависимости
		 от используемого стиля оформления ячеек.


		 ![](../../Property_Image.gif)
		 [MidValueType](ITabFormatValues.MidValueType.htm)
		 Свойство MidValueType
		 определяет тип условия, по которому устанавливается среднее допустимое
		 значение для ячеек, попадающих под форматирование.


		 ![](../../Property_Image.gif)
		 [MinValue](ITabFormatValues.MinValue.htm)
		 Свойство MinValue определяет
		 условие, по которому устанавливается минимальное допустимое значение
		 для ячеек, попадающих под форматирование.


		 ![](../../Property_Image.gif)
		 [MinValueColor](ITabFormatValues.MinValueColor.htm)
		 Свойство MinValueColor
		 определяет начальный цвет градиента, используемый для оформления
		 ячеек.


		 ![](../../Property_Image.gif)
		 [MinValueType](ITabFormatValues.MinValueType.htm)
		 Свойство MinValueType
		 определяет тип условия, по которому устанавливается минимальное
		 допустимое значение для ячеек, попадающих под форматирование.


		 ![](../../Property_Image.gif)
		 [PointCondition](ITabFormatValues.PointCondition.htm)
		 Свойство PointCondition
		 определяет соотношение, по которому осуществляется отбор значений,
		 удовлетворяющих указанному правилу.


		 ![](../../Property_Image.gif)
		 [PointType](ITabFormatValues.PointType.htm)
		 Свойство PointType
		 определяет тип условия, по которому устанавливается значение для
		 указанного правила.


		 ![](../../Property_Image.gif)
		 [PointValue](ITabFormatValues.PointValue.htm)
		 Свойство PointValue
		 определяет условие, по которому устанавливается значение для ячеек,
		 попадающих под указанное правило.


		 ![](../../Property_Image.gif)
		 [ReverseIcons](ITabFormatValues.ReverseIcons.htm)
		 Свойство ReverseIcons
		 определяет, будет ли использоваться при форматировании обратный
		 порядок пиктограмм.


		 ![](../../Property_Image.gif)
		 [ShowValues](ITabFormatValues.ShowValues.htm)
		 Свойство ShowValues
		 определяет, будут ли отображаться значения в ячейках при использовании
		 условного формата.


		 ![](../../Property_Image.gif)
		 [Style](ITabFormatValues.Style.htm)
		 Свойство Style определяет
		 стиль форматирования ячеек на основе их значений.
