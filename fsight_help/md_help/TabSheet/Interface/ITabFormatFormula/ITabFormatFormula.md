# ITabFormatFormula


## Описание


Интерфейс ITabFormatFormula
 содержит свойства условного формата, осуществляющего форматирование ячеек,
 удовлетворяющих формуле.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Condition](ITabFormatFormula.Condition.htm)
		 Свойство Condition
		 определяет формулу, по которой будут выбираться ячейки, подлежащие
		 форматированию.


		 ![](../../Property_Image.gif)
		 [Style](ITabFormatFormula.Style.htm)
		 Свойство Style определяет
		 стиль оформления, применяемый для ячеек, удовлетворяющих установленному
		 условию.
