# ITabObjectResizingEventArgs.Cancel


## Синтаксис


		Cancel: Boolean;


## Описание


Свойство Cancel определяет,
 отменять ли изменение размеров.


## Комментарии


Допустимые значения:


	- True. Изменение размеров
	 объекта будет отменено;


	- False. Изменение размеров
	 объекта будет выполнено. Значение по умолчанию.
