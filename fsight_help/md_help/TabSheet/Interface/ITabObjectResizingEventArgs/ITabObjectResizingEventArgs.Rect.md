# ITabObjectResizingEventArgs.Rect


## Синтаксис


		Rect: [IGxRectF](ModDrawing.chm::/Interface/IGxRectF/IGxRectF.htm);


## Описание


Свойство Rect возвращает координаты
 прямоугольной области, занимаемой объектом.


## Комментарии


Для получения стороны, которая была изменена, используйте свойство [ITabObjectResizingEventArgs.Side](ITabObjectResizingEventArgs.Side.htm).
