# ITabObjectResizingEventArgs.Side


## Синтаксис


		Side: [TabObjectResizingSide](../../Enums/TabObjectResizingSide.htm);


## Описание


Свойство Side возвращает сторону,
 которая была изменена.


## Комментарии


Для получения координат объекта, используйте свойство [ITabObjectResizingEventArgs.Rect](ITabObjectResizingEventArgs.Rect.htm).
