# ITabObjectResizingEventArgs.Object


## Синтаксис


		Object: [ITabObject](../ITabObject/ITabObject.htm);


## Описание


Свойство Object возвращает изменяемый
 объект.


## Комментарии


Для получения стороны, которая была изменена, используйте свойство [ITabObjectResizingEventArgs.Side](ITabObjectResizingEventArgs.Side.htm).
