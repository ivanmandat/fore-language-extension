# ITableFilterConditions


Сборка: Tab;


## Описание


Интерфейс ITableFilterConditions содержит
 свойства и методы для управления коллекцией условий фильтрации.


## Иерархия наследования


           ITableFilterConditions


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITableFilterConditions.Count.htm)
		 Свойство Count возвращает
		 количество заданных условий фильтрации в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ITableFilterConditions.Item.htm)
		 Свойство Item возвращает
		 условие по его индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ITableFilterConditions.Add.htm)

		 ![](../../Sub_Image.gif)
		 [Clear](ITableFilterConditions.Clear.htm)
		 Метод Clear удаляет
		 все условия фильтрации из коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](ITableFilterConditions.Remove.htm)
		 Метод Remove удаляет
		 условие фильтрации по его индексу.
