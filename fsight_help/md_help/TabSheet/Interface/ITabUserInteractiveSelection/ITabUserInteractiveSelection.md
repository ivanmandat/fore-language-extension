# ITabUserInteractiveSelection


Сборка: Tab;


## Описание


Интерфейс ITabUserInteractiveSelection
 содержит свойства для настройки интерактивного диапазонов ячеек.


## Иерархия наследования


ITabUserInteractiveSelection


## Комментарии


Настройки интерактивного диапазона ячеек можно получить, используя свойства
 и методы интерфейса [ITabUserInteractiveSelections](../ITabUserInteractiveSelections/ITabUserInteractiveSelections.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BorderColor](ITabUserInteractiveSelection.BorderColor.htm)
		 Свойство BorderColor
		 определяет цвет границы интерактивного диапазона.


		 ![](../../Property_Image.gif)
		 [BorderStyle](ITabUserInteractiveSelection.BorderStyle.htm)
		 Свойство BorderStyle
		 определяет стиль линии границы интерактивного диапазона.


		 ![](../../Property_Image.gif)
		 [InteractiveRestrictions](ITabUserInteractiveSelection.InteractiveRestrictions.htm)
		 Свойство InteractiveRestrictions
		 определяет ограничения по возможности изменения границ интерактивного
		 диапазона.


		 ![](../../Property_Image.gif)
		 [Range](ITabUserInteractiveSelection.Range.htm)
		 Свойство Range определяет
		 диапазон ячеек, для которого создается интерактивный диапазон.


		 ![](../../Property_Image.gif)
		 [Visible](ITabUserInteractiveSelection.Visible.htm)
		 Свойство Visible определяет,
		 будет ли отображаться интерактивный диапазон.
