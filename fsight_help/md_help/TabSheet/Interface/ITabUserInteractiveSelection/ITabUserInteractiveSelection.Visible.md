# ITabUserInteractiveSelection.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет,
 будет ли отображаться интерактивный диапазон.


## Комментарии


Допустимые значения:


	- True. Интерактивный
	 диапазон отображается в таблице, доступно его изменение в соответствии
	 с заданными [ограничениями](ITabUserInteractiveSelection.InteractiveRestrictions.htm);


	- False. Интерактивный
	 диапазон не отображается, доступен только в коллекции [ITabView.UserIntearactiveSelections](../ITabView/ITabView.UserIntearactiveSelections.htm).


## Пример


Пример использования приведён в описании метода [ITabUserInteractiveSelections.Add](../ITabUserInteractiveSelections/ITabUserInteractiveSelections.Add.htm).
