# ITabPrintScale


## Описание


Интерфейс ITabPrintScale содержит
 свойства масштаба при печати таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [FitToPages](ITabPrintScale.FitToPages.htm)
		 Свойство FitToPages
		 определяет каким образом устанавливается масштаб при печати таблицы.


		 ![](../../Property_Image.gif)
		 [PageHeight](ITabPrintScale.PageHeight.htm)
		 Свойство PageHeight
		 определяет количество страниц, ограничивающих размер таблицы при
		 печати, по высоте.


		 ![](../../Property_Image.gif)
		 [PageWidth](ITabPrintScale.PageWidth.htm)
		 Свойство PageWidth
		 определяет количество страниц, ограничивающих размер таблицы при
		 печати, по ширине.


		 ![](../../Property_Image.gif)
		 [Value](ITabPrintScale.Value.htm)
		 Свойство Value определяет
		 процентное соотношение, используемое для подгонки масштаба при
		 печати таблицы.
