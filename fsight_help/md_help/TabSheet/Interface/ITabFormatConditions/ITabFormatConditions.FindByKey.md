# ITabFormatConditions.FindByKey


## Синтаксис


FindByKey(Key: Integer): [ITabFormatCondition](../ITabFormatCondition/ITabFormatCondition.htm);


## Параметры


Key - ключ условного формата.


## Описание


Метод FindByKey осуществляет
 поиск условного формата по ключу и возвращает его параметры в случае удачного
 поиска.
