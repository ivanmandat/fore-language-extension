# ITabFormatConditions.IsInUpdate


## Синтаксис


IsInUpdate: Boolean;


## Описание


Свойство IsInUpdate  возвращает
 признак применения условных форматов.


## Комментарии


Допустимые значения:


	- True. Применение условных
	 форматов будет отключено;


	- False. По умолчанию.
	 Применение условных форматов не будет отключено.
