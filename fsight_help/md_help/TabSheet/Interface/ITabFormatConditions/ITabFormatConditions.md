# ITabFormatConditions


## Описание


Интерфейс ITabFormatConditions
 содержит свойства и методы коллекции условных форматов текста.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITabFormatConditions.Count.htm)
		 Свойство Count возвращает
		 количество условных форматов в коллекции.


		 ![](../../Property_Image.gif)
		 [IsInUpdate](ITabFormatConditions.IsInUpdate.htm)
		 Свойство IsInUpdate
		 возвращает признак применения условных форматов.


		 ![](../../Property_Image.gif)
		 [Item](ITabFormatConditions.Item.htm)
		 Свойство Item возвращает
		 ссылку на условный формат.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ITabFormatConditions.Add.htm)
		 Метод Add добавляет
		 новый условный формат и возвращает на него ссылку.


		 ![](../../Sub_Image.gif)
		 [AddCollection](ITabFormatConditions.AddCollection.htm)
		 Метод AddCollection
		 добавляет указанную коллекцию условных форматов.


		 ![](../../Sub_Image.gif)
		 [BeginUpdate](ITabFormatConditions.BeginUpdate.htm)
		 Метод BeginUpdate отключает
		 вычисление и применение условных форматов.


		 ![](../../Sub_Image.gif)
		 [Clear](ITabFormatConditions.Clear.htm)
		 Метод Clear очищает
		 коллекцию условных форматов.


		 ![](../../Sub_Image.gif)
		 [EndUpdate](ITabFormatConditions.EndUpdate.htm)
		 Метод EndUpdate возобновляет
		 вычисление и применение условных форматов.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ITabFormatConditions.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск условного формата по ключу


		 ![](../../Sub_Image.gif)
		 [Move](ITabFormatConditions.Move.htm)
		 Метод Move осуществляет
		 перемещение условных форматов в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](ITabFormatConditions.Remove.htm)
		 Метод Remove удаляет
		 условный формат, индекс которого передается в качестве входного
		 параметра.


		 ![](../../Sub_Image.gif)
		 [RemoveCollection](ITabFormatConditions.RemoveCollection.htm)
		 Метод RemoveCollection
		 осуществляет удаление указанной коллекции условных форматов.
