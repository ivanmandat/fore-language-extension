# ITabFormatConditions.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс
 условного формата, который необходимо удалить.


## Описание


Метод Remove удаляет условный
 формат из коллекции форматов. Индекс формата передается посредством входного
 параметра Index.
