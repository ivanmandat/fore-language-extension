# ITabFormatConditions.RemoveCollection


## Синтаксис


RemoveCollection(Value: [ITabFormatConditions](ITabFormatConditions.htm));


## Параметры


Value - коллекция условных
 форматов, которую необходимо удалить.


## Описание


Метод RemoveCollection осуществляет
 удаление указанной коллекции условных форматов.


Примечание.
 Удаляемая коллекция условных форматов изначально должна быть включена
 в коллекцию, для которой вызывается данный метод, посредством метода [AddCollection](ITabFormatConditions.AddCollection.htm).
