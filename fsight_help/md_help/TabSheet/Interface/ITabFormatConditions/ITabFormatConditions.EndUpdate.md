# ITabFormatConditions.EndUpdate


## Синтаксис


EndUpdate;


## Описание


Метод EndUpdate возобновляет
 вычисление и применение условных форматов.


## Комментарии


Метод возобновляет вычисление и применение условных форматов после использования
 метода [ITabFormatConditions.BeginUpdate](ITabFormatConditions.BeginUpdate.htm).
 При вызове метода также будут применены все изменения, произведённые с
 условными форматами между вызовами методов [ITabFormatConditions.BeginUpdate](ITabFormatConditions.BeginUpdate.htm)
 и ITabFormatConditions.EndUpdate.
