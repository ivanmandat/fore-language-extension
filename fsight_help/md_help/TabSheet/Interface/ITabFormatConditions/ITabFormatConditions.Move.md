# ITabFormatConditions.Move


## Синтаксис


Move(IndexFrom: Integer; IndexTo: Integer);


## Параметры


IndexFrom - индекс позиции
 условного формата, который необходимо переместить.


IndexTo - индекс позиции, в
 которую необходимо переместить.


## Описание


Метод Move осуществляет перемещение
 условных форматов в коллекции.


Используется для изменения приоритета применения условных форматов.
 Форматы, расположенные в начале коллекции, применяются в первую очередь.
