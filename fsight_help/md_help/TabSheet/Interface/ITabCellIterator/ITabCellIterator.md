# ITabCellIterator


## Описание


Интерфейс ITabCellIterator реализует
 свойства и методы для работы с итератором для навигации по ячейкам таблицы,
 содержащим значения.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CurrentCell](ITabCellIterator.CurrentCell.htm)
		 Свойство CurrentCell
		 возвращает ячейку, в которой в данный момент находится итератор.


		 ![](../../Property_Image.gif)
		 [CurrentColumn](ITabCellIterator.CurrentColumn.htm)
		 Свойство CurrentColumn
		 возвращает индекс столбца, в котором в данный момент находится
		 итератор.


		 ![](../../Property_Image.gif)
		 [CurrentRow](ITabCellIterator.CurrentRow.htm)
		 Свойство CurrentRow
		 возвращает индекс строки, в которой в данный момент находится
		 итератор.


		 ![](../../Property_Image.gif)
		 [Order](ITabCellIterator.Order.htm)
		 Свойство Order определяет
		 вариант порядка навигации по элементам в итераторе.


		 ![](../../Property_Image.gif)
		 [Range](ITabCellIterator.Range.htm)
		 Свойство Range
		 задает диапазон ячеек для итератора.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [First](ITabCellIterator.First.htm)
		 Метод First осуществляет
		 переход на начало итератора.


		 ![](../../Sub_Image.gif)
		 [Next](ITabCellIterator.Next.htm)
		 Метод Next осуществляет
		 переход на следующий элемент итератора.
