# ITabCellIterator.Order


## Синтаксис


Order: [TabCellIteratorOrder](../../Enums/TabCellIteratorOrder.htm);


## Описание


Свойство Order определяет вариант порядка навигации по элементам в итераторе.


## Комментарии


Значением свойства являются элементы перечисления [TabCellIteratorOrder](../../Enums/TabCellIteratorOrder.htm). По умолчанию установлено значение None, то есть итератор обходит ячейки по строкам в пределах страницы.
