# ITabEditor.XamlText


## Синтаксис


XamlText: String;


## Описание


Свойство XamlText возвращает отформатированный текст в формате XML, который в текущий момент установлен в редакторе ячейки.


## Комментарии


Если в ячейке установлен неформатированный текст, то свойство вернет пустую строку.
