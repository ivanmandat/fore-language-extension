# ITabObject.ClassId


## Синтаксис


ClassId: String;


## Описание


Свойство ClassId возвращает
 идентификатор класса объекта, расположенного на листе таблицы.


## Комментарии


Данное свойство вернет одно из следующих значений:


	- PrxPicture - изображение;


	- PrxChart - диаграмма;


	- PrxScene3D - 3D-сцена;


	- PrxSpeedometer - спидометр;


	- PrxMap - карта;


	- PrxMapTerritoryInfo - расшифровка территорий карты;


	- PrxMapLegend - легенда карты;


	- PrxShape - автофигура;


	- PrxFlowDocument - форматированный текст;


	- PrxVisualizer - Легенда визуализатора.
