# ITabObject


Сборка: Tab;


## Описание


Интерфейс ITabObject предназначен
 для работы с объектом, расположенном на листе таблицы.


## Иерархия наследования


           ITabObject


## Комментарии


Для работы со всем объектами, расположенными на листе таблицы используйте
 интерфейс [ITabObjects](../ITabObjects/ITabObjects.htm).

		 объект в область, которая передается в качестве входного параметра.

		 [Select_](ITabObject.Select.htm)
