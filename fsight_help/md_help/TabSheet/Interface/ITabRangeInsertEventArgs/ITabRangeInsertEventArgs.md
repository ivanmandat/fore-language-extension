# ITabRangeInsertEventArgs


Сборка: Tab;


## Описание


Интерфейс ITabRangeInsertEventArgs
 предназначен для работы с аргументами событий, происходящих при добавлении
 диапазона ячеек.

## Комментарии


Для работы с аргументами событий, происходящих при удалении диапазона
 ячеек используйте свойства интерфейса [ITabRangeDeleteEventArgs](../ITabRangeDeleteEventArgs/ITabRangeDeleteEventArgs.htm).
