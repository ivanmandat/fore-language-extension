# ITabRangeInsertEventArgs.Range


## Синтаксис


Range: [ITabRange](../ITabRange/ITabRange.htm);


## Описание


Свойство Range возвращает параметры
 диапазона ячеек таблицы.


## Комментарии


Для получения способа, который использовался при добавлении ячеек таблицы,
 используйте свойство [ITabRangeInsertEventArgs.ShiftDirection](ITabRangeInsertEventArgs.ShiftDirection.htm).
