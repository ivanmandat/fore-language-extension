# ITabNamedRegions.RemoveAt


## Синтаксис


RemoveAt(Index: Integer): Boolean;


## Параметры


Index —
 индекс именованного диапазона, который необходимо удалить.


## Описание


Метод RemoveAt удаляет именованный
 диапазон, индекс которого передается в качестве входного параметра Index, и возвращает True
 в случае удачного удаления.
