# ITabNamedRegions


## Описание


Интерфейс ITabNamedRegions управляет
 параметрами коллекции именованных диапазонов таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITabNamedRegions.Count.htm)
		 Свойство Count возвращает
		 количество именованных диапазонов таблицы.


		 ![](../../Property_Image.gif)
		 [Item](ITabNamedRegions.Item.htm)
		 Свойство Item возвращает
		 именованный диапазон таблицы.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ITabNamedRegions.Add.htm)
		 Метод Add добавляет
		 в коллекцию именованный диапазон.


		 ![](../../Sub_Image.gif)
		 [Clear](ITabNamedRegions.Clear.htm)
		 Метод Clear очищает
		 коллекцию именованных диапазонов.


		 ![](../../Sub_Image.gif)
		 [FindByName](ITabNamedRegions.FindByName.htm)
		 Метод FindByName осуществляет
		 поиск именованного диапазона таблицы по наименованию.


		 ![](../../Sub_Image.gif)
		 [Remove](ITabNamedRegions.Remove.htm)
		 Метод Remove удаляет
		 именованный диапазон по наименованию.


		 ![](../../Sub_Image.gif)
		 [RemoveAt](ITabNamedRegions.RemoveAt.htm)
		 Метод RemoveAt удаляет
		 именованный диапазон по индексу.
