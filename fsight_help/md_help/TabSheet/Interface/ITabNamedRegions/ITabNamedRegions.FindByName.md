# ITabNamedRegions.FindByName


## Синтаксис


FindByName(Name: String): [ITabNamedRegion](../ITabNamedRegion/ITabNamedRegion.htm);


## Параметры


Name —
 наименование именованного диапазона, который необходимо получить.


## Описание


Метод FindByName осуществляет
 поиск именованного диапазон таблицы, наименование которого передается
 в качестве входного параметра Name.
