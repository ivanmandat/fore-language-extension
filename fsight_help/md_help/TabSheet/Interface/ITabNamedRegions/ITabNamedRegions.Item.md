# ITabNamedRegions.Item


## Синтаксис


Item(Index: Integer): [ITabNamedRegion](../ITabNamedRegion/ITabNamedRegion.htm);


## Параметры


Index — индекс
 именованного диапазона, который необходимо получить.


## Описание


Свойство Item возвращает именованный
 диапазон таблицы, индекс которого передается в качестве входного параметра
 Index.
