# ITabNamedRegions.Remove


## Синтаксис


Remove(Name: String): Boolean;


## Параметры


Name —
 наименование именованного диапазона, который необходимо удалить.


## Описание


Метод Remove удаляет именованный
 диапазон, наименование которого передается в качестве входного параметра
 Name, и возвращает True
 в случае удачного удаления.
