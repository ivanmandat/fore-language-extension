# ITabNamedRegions.Add


## Синтаксис


Add(Name: String; Range: [ITabRange](../ITabRange/ITabRange.htm)):
 [ITabNamedRegion](../ITabNamedRegion/ITabNamedRegion.htm);


## Параметры


Name — наименование именованного
 диапазона, который необходимо добавить;


Range — диапазон, в котором
 будет располагаться именованный диапазон.


## Описание


Метод Add добавляет в коллекцию
 именованный диапазон, наименование и диапазон которого передается в качестве
 входных параметров Name и Range соответственно.
