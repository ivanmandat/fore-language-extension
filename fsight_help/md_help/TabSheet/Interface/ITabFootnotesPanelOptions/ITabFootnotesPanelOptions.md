# ITabFootnotesPanelOptions


Сборка: Tab;


## Описание


Интерфейс ITabFootnotesPanelOptions
 содержит свойства для настройки параметров панели сносок.


## Иерархия наследования


ITabFootnotesPanelOptions


## Комментарии


Параметры панели сносок можно получить, используя одно из следующих
 свойств:


	- [IEaxGrid.FootnotePanelOptions](KeExpress.chm::/Interface/IEaxGrid/IEaxGrid.FootnotePanelOptions.htm).
	 Параметры панели сносок в экспресс-отчете;


	- [IPrxReportOptions.FootnotePanelOptions](KeReport.chm::/Interface/IPrxReportOptions/IPrxReportOptions.FootnotePanelOptions.htm).
	 Параметры панели сносок в регламентном отчете.
