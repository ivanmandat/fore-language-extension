# ITabFootnotes.StartingNumber


## Синтаксис


StartingNumber: Integer;


## Описание


Свойство StartingNumber определяет
 стартовое значение, начиная с которого производится автоматическая нумерация
 сносок.


## Комментарии


По умолчанию нумерация сносок начинается с единицы.
