# ITabFootnotes.EndUpdate


## Синтаксис


EndUpdate;


## Описание


Метод EndUpdate возобновляет
 обновление сносок.


## Комментарии


При выполнении метода будут применены все изменения, которые были внесены
 в промежутке между вызовами методов [ITabFootnotes.BeginUpdate](ITabFootnotes.BeginUpdate.htm)
 и EndUpdate.
