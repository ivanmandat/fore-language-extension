# ITabFootnotes.UniqueNumbers


## Синтаксис


UniqueNumbers: Boolean;


## Описание


Свойство UniqueNumbers определяет,
 будут ли иметь уникальные номера сноски с одинаковым текстом.


## Комментарии


Допустимые значения:


	- True. Сноски имеют уникальный
	 номер.


	- False. Сноски имеют
	 одинаковый номер.
