# ITabFootnotes.InUpdate


## Синтаксис


InUpdate: Boolean;


## Описание


Свойство InUpdate возвращает
 признак включенного режима запрета обновления сносок.


## Комментарии


Свойство возвращает значение True,
 если был вызван метод [ITabFootnotes.BeginUpdate](ITabFootnotes.BeginUpdate.htm),
 но еще не был вызван метод [ITabFootnotes.EndUpdate](ITabFootnotes.EndUpdate.htm).
