# ITabFootnotes.BeginUpdate


## Синтаксис


BeginUpdate;


## Описание


Метод BeginUpdate отключает
 обновление сносок для экономии времени, связанного с их прорисовкой и
 обновлением данных в сносках.


## Комментарии


При вызове метода BeginUpdate
 останавливаются следующие действия:


	- прорисовка сносок в ячейках таблицы и панели сносок;


	- пересчет номеров сносок, который может быть связан с изменением
	 количества сносок, количества страниц и количества листов.


Все изменения, выполненные после вызова метода BeginUpdate,
 не будут применены до тех пор, пока не будет вызван метод [ITabFootnotes.EndUpdate](ITabFootnotes.EndUpdate.htm).


Проверить, находится ли коллекция в состоянии остановки обновления,
 можно с помощью свойства [ITabFootnotes.InUpdate](ITabFootnotes.InUpdate.htm).
