# ITabFootnotes.CustomMarkFontName


## Синтаксис


CustomMarkFontName: String;


## Описание


Свойство CustomMarkFontName
 определяет наименование шрифта, который будет использоваться для отрисовки
 меток.


## Комментарии


Свойство актуально, если определено свойство [ITabFootnotes.CustomMark](ITabFootnotes.CustomMark.htm).
 В качестве значения укажите наименование одного из шрифтов, установленных
 в операционной системе.
