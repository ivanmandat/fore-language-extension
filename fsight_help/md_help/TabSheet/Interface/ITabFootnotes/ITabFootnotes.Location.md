# ITabFootnotes.Location


## Синтаксис


Location: [TabFootnotesLocation](../../Enums/TabFootnotesLocation.htm);


## Описание


Свойство Location определяет
 расположение сносок при разбивке таблицы на отдельные страницы.


## Комментарии


Значение свойства учитывается при печати/экспорте таблицы, когда производится
 разбивка на отдельные страницы. По умолчанию нумерация сносок продолжается
 при переходе к новым страницам.
