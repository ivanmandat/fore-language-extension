# ITabPageContinuousRanges.ItemPosition


## Синтаксис


ItemPosition(Index: Integer): Integer;


## Параметры


Index. Индекс неразрывного
 диапазона в таблице.


## Описание


Свойство ItemPosition возвращает
 начальную позицию неразрывного диапазона.


## Комментарии


В зависимости от того, с какой коллекцией неразрывных диапазонов ведётся
 работа ([диапазонов
 по столбцам](../ITabPageSettings/ITabPageSettings.ContinuousColumns.htm) или [диапазонов
 по строкам](../ITabPageSettings/ITabPageSettings.ContinuousRows.htm)), будет возвращена начальная позиция по столбцу или строке.
