# ITabPageContinuousRange.Remove


## Синтаксис


Remove(Index: Integer);


## Параметры


Index. Индекс неразрывного
 диапазона в таблице.


## Описание


Метод Remove удаляет в таблице
 указанный неразрывный диапазон.
