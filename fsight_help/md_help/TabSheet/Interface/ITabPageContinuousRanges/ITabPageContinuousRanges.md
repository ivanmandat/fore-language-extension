# ITabPageContinuousRanges


Сборка: Tab;


## Описание


Интерфейс ITabPageContinuousRanges
 содержит свойства и методы для работы с коллекцией неразрывных диапазонов
 таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITabPageContinuousRanges.Count.htm)
		 Свойство Count возвращает
		 количество неразрывных диапазонов в таблице.


		 ![](../../Property_Image.gif)
		 [ItemLenght](ITabPageContinuousRanges.ItemLeght.htm)
		 Свойство ItemLenght
		 возвращает длину указанного неразрывного диапазона.


		 ![](../../Property_Image.gif)
		 [ItemPosition](ITabPageContinuousRanges.ItemPosition.htm)
		 Свойство ItemPosition
		 возвращает начальную позицию неразрывного диапазона.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ITabPageContinuousRanges.Add.htm)
		 Метод Add создаёт в
		 таблице новый неразрывный диапазон.


		 ![](../../Sub_Image.gif)
		 [Assign](ITabPageContinuousRanges.Assign.htm)
		 Метод Assign устанавливает
		 для текущей таблицы неразрывные диапазоны, передаваемые в качестве
		 входного параметра.


		 ![](../../Sub_Image.gif)
		 [Remove](ITabPageContinuousRanges.Remove.htm)
		 Метод Remove удаляет
		 в таблице указанный неразрывный диапазон.


		 ![](../../Sub_Image.gif)
		 [RemoveAll](ITabPageContinuousRanges.RemoveAll.htm)
		 Метод RemoveAll удаляет
		 в таблице все неразрывные диапазоны.
