# ITabPageContinuousRanges.ItemLenght


## Синтаксис


ItemLenght(Index: Integer): Integer;


## Параметры


Index. Индекс неразрывного
 диапазона в таблице.


## Описание


Свойство ItemLenght возвращает
 длину указанного неразрывного диапазона.


## Комментарии


В зависимости от того, с какой коллекцией неразрывных диапазонов ведётся
 работа ([диапазонов
 по столбцам](../ITabPageSettings/ITabPageSettings.ContinuousColumns.htm) или [диапазонов
 по строкам](../ITabPageSettings/ITabPageSettings.ContinuousRows.htm)), будет возвращена длина диапазона по столбцам или строкам.
