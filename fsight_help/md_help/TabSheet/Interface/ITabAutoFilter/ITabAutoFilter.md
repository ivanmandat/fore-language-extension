# ITabAutoFilter


## Описание


Интерфейс ITabAutoFilter управляет
 параметрами автофильтра области таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Enabled](ITabAutoFilter.Enabled.htm)
		 Свойство Enabled определяет
		 активацию автофильтра области таблицы.


		 ![](../../Property_Image.gif)
		 [Range](ITabAutoFilter.Range.htm)
		 Свойство Range определяет
		 область таблицы, в которой будет установлен автофильтр.


## Свойства, унаследованные от [ITableFilter](../ITableFilter/itablefilter.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](../ITableFilter/itablefilter.count.htm)
		 Свойство Count возвращает
		 количество элементов в автофильтре.


		 ![](../../Property_Image.gif)
		 [Features](../ITableFilter/ITableFilter.Features.htm)
		 Свойство Features определяет
		 состав отображаемых настроек в меню фильтрации в таблице.


		 ![](../../Property_Image.gif)
		 [Item](../ITableFilter/itablefilter.item.htm)
		 Свойство Item возвращает
		 параметры указанного элемента фильтрации.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Load](ITabAutoFilter.Load.htm)
		 Свойство Load загружает
		 автофильтр из потока.


		 ![](../../Sub_Image.gif)
		 [Save](ITabAutoFilter.Save.htm)
		 Свойство Save сохраняет
		 автофильтр в поток.
