# ITabCellAfterEditEventArgs


## Описание


Интерфейс ITabCellAfterEditEventArgs
 содержит свойства аргумента события [TabSheetBox.OnAfterEdit](../../Class/TabSheetBox/TabSheetBox.OnAfterEdit.htm),
 происходящего после редактирования значения в ячейке компонента TabSheetBox.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Changed](ITabCellAfterEditEventArgs.Changed.htm)
		 Свойство Changed возвращает
		 True, если в результате
		 редактирования значения в ячейке данные были изменены.
