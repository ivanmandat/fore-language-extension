# ITabOptions.MaxIterations


## Синтаксис


MaxIterations: Integer;


## Описание


Свойство MaxIterations определяет количество итераций, после которых прекратится расчет ячеек с циклическими ссылками.


## Комментарии


Задаваемое значение должно быть неотрицательным числом. По умолчанию используется 100 итераций. Свойство актуально для использования, только если свойство [IterativeCalculation](ITabOptions.IterativeCalculation.htm) установлено в True.
