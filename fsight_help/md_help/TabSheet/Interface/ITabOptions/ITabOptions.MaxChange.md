# ITabOptions.MaxChange


## Синтаксис


MaxChange: Double;


## Описание


Свойство MaxChange определяет относительную погрешность, с которой система сравнивает разницу значений каждой ячейки (с циклическими ссылками) в текущей и предыдущей итерациях. Если изменение значения величины каждой ячейки становится меньше относительной погрешности, то на данной итерации расчет ячеек с циклическими ссылками прекращается.


## Комментарии


Задаваемое значение должно быть неотрицательным числом. По умолчанию относительная погрешность равна 0,001. Свойство актуально для использования, только если свойство [IterativeCalculation](ITabOptions.IterativeCalculation.htm) установлено в True.
