# ITabCellBeforeEditEventArgs


## Описание


Интерфейс ITabCellBeforeEditEventArgs
 содержит свойства аргумента события [TabSheetBox.OnBeforeEdit](../../Class/TabSheetBox/TabSheetBox.OnBeforeEdit.htm),
 происходящего перед началом редактирования значения ячейки компонента
 TabSheetBox.


## Свойства


		 Имя свойства
		 Краткое описание

		 [Cancel](ITabCellBeforeEditEventArgs.Cancel.htm)
		 Свойство Cancel определяет,
		 будет ли производиться редактирование значения ячейки.
