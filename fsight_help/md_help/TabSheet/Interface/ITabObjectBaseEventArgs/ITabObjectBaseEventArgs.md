## Описание


Интерфейс ITabObjectBaseEventArgs
 содержит свойства аргумента события, наступающего при активации/деактивации
 объекта таблицы компонента [LanerBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/LanerBox.htm).
