# ITabFormatCondition


## Описание


Интерфейс ITabFormatCondition
 содержит свойства условного формата ячеек.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AppliedRange](ITabFormatCondition.AppliedRange.htm)
		 Свойство AppliedRange
		 определяет диапазон ячеек, для которых выполняется [условное
		 форматирование](UiAnalyticalArea.chm::/Analysis/Conditional_formatting.htm).


		 ![](../../Property_Image.gif)
		 [DataAdapter](ITabFormatCondition.DataAdapter.htm)
		 Свойство DataAdapter
		 определяет источник данных для условного формата ячеек.


		 ![](../../Property_Image.gif)
		 [Details](ITabFormatCondition.Details.htm)
		 Свойство Details определяет
		 специфические настройки определенного условного формата.


		 ![](../../Property_Image.gif)
		 [Enabled](ITabFormatCondition.Enabled.htm)
		 Свойство Enabled определяет,
		 используется ли текущий условный формат.


		 ![](../../Property_Image.gif)
		 [Key](ITabFormatCondition.Key.htm)
		 Свойство Key возвращает
		 ключ условного формата.


		 ![](../../Property_Image.gif)
		 [Range](ITabFormatCondition.Range.htm)
		 Свойство Range определяет
		 диапазон ячеек таблицы, для которых необходимо установить условный
		 формат.


		 ![](../../Property_Image.gif)
		 [Type](ITabFormatCondition.Type.htm)
		 Свойство Type определяет
		 тип условного формата, применяемого к ячейкам.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AssignPredefinedDataBar](ITabFormatCondition.AssignPredefinedDataBar.htm)
		 Метод AssignPredefinedDataBar
		 загружает в текущий условный формат настройки стандартного условного
		 формата с гистограммой.


		 ![](../../Sub_Image.gif)
		 [AssignPredefinedGradient](ITabFormatCondition.AssignPredefinedGradient.htm)
		 Метод AssignPredefinedGradient
		 загружает в текущий условный формат настройки стандартного условного
		 формата с градиентной заливкой.


		 ![](../../Sub_Image.gif)
		 [AssignPredefinedGrowth](ITabFormatCondition.AssignPredefinedGrowth.htm)
		 Метод AssignPredefinedGradient
		 загружает в текущий условный формат настройки индикатора роста.


		 ![](../../Sub_Image.gif)
		 [AssignPredefinedIcons](ITabFormatCondition.AssignPredefinedIcons.htm)
		 Метод AssignPredefinedIcons
		 загружает в текущий условный формат настройки стандартного условного
		 формата с пиктограммами.


		 ![](../../Sub_Image.gif)
		 [AssignPredefinedScale](ITabFormatCondition.AssignPredefinedScale.htm)
		 Метод AssignPredefinedScale
		 загружает в текущий условный формат параметры стандартной цветовой
		 шкалы.
