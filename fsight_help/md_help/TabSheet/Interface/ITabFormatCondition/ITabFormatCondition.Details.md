# ITabFormatCondition.Details


## Синтаксис


Details: Object;


## Описание


Свойство Details определяет
 специфические настройки определенного условного формата.


## Комментарии


В зависимости от значения свойства [Type](ITabFormatCondition.Type.htm),
 данное свойство должно быть приведено к одному из следующих интерфейсов:


	- [ITabFormatValues](../ITabFormatValues/ITabFormatValues.htm)


	- [ITabFormatCellContent](../ITabFormatCellContent/ITabFormatCellContent.htm)


	- [ITabFormatRankValues](../ITabFormatRankValues/ITabFormatRankValues.htm)


	- [ITabFormatAverage](../ITabFormatAverage/ITabFormatAverage.htm)


	- [ITabFormatDuplicate](../ITabFormatDuplicate/ITabFormatDuplicate.htm)


	- [ITabFormatFormula](../ITabFormatFormula/ITabFormatFormula.htm)


	- [ITabFormatGrowth](../ITabFormatGrowth/ITabFormatGrowth.htm)


	- [ITabFormatScale](../ITabFormatScale/ITabFormatScale.htm)
