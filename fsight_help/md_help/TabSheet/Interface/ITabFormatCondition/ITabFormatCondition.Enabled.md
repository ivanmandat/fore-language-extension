# ITabFormatCondition.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет, используется ли текущий условный формат.


## Комментарии


По умолчанию свойству установлено значение True, при этом условный формат используется для оформления ячеек.


Если свойству установить значение False, то условный формат использоваться не будет. При этом все его настройки сброшены не будут.
