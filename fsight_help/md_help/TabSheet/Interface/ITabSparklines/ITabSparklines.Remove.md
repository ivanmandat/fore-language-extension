# ITabSparklines.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index.
 Индекс спарклайна в коллекции.


## Описание


Метод Remove удаляет указанный
 спарклайн из коллекции.


## Комментарии


Метод возвращает значение True,
 если спарклайн удален удачно и False,
 если удаление по каким-либо причинам невозможно.
