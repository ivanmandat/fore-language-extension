# ITabSparklines.Add


## Синтаксис


Add(Range: [ITabRange](../ITabRange/ITabRange.htm);
 DataRange: [ITabRange](../ITabRange/ITabRange.htm)): [ITabSparklines](ITabSparklines.htm);


## Параметры


Range. Диапазон, в котором
 будут располагаться спарклайны.


DataRange. Диапазон данных,
 на базе которых будут строиться спарклайны.


## Описание


Метод Add добавляет новую коллекцию
 спарклайнов.


## Комментарии


Диапазон, указываемый в параметре Range,
 должен включать в себя ячейки какого-либо одного
 столбца/строки. Если спарклайны добавляются в столбец, то их графики будут
 строиться по данным, которые расположены в строках диапазона DataRange,
 и наоборот, если спарклайны добавляются в строку, то данные для графиков
 берутся из столбцов диапазона DataRange.
 Для графиков спарклайнов будет установлен одинаковый цвет.
