# ITabSparklines


Сборка: Tab;


## Описание


Интерфейс ITabSparklines содержит
 свойства и методы для работы с коллекцией спарклайнов таблицы.


## Иерархия наследования


           ITabSparklines


## Комментарии


Коллекцию спарклайнов можно получить в свойстве [ITabSheet.Sparklines](../ITabSheet/ITabSheet.Sparklines.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITabSparklines.Count.htm)
		 Свойство Count возвращает
		 количество спарклайнов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ITabSparklines.Item.htm)
		 Свойство Item возвращает
		 параметры указанного спарклайна.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ITabSparklines.Add.htm)
		 Метод Add добавляет
		 новую коллекцию спарклайнов.


		 ![](../../Sub_Image.gif)
		 [Clear](ITabSparklines.Clear.htm)
		 Метод Clear удаляет
		 все спарклайны таблицы.


		 ![](../../Sub_Image.gif)
		 [Remove](ITabSparklines.Remove.htm)
		 Метод Remove удаляет
		 указанный спарклайн из коллекции.
