# ITabTableStripe


## Описание


Интерфейс ITabTableStripe содержит
 свойства для определения параметров полосы строк (столбцов) в таблице.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Style](ITabTableStripe.Style.htm)
		 Свойство Style определяет
		 стиль отображения ячеек для полосы строк (столбцов) в таблице.


		 ![](../../Property_Image.gif)
		 [Width](ITabTableStripe.Width.htm)
		 Свойство Width определяет
		 ширину полосы строк (столбцов) в таблице.
