# ITabCellEventArgs


## Описание


Интерфейс ITabCellEventArgs
 содержит свойства аргумента событий, связанных со щелчком мышью по ячейке
 либо объектам, которые могут содержаться в ячейке:


	- [TabSheetBox.OnCellClick](../../Class/TabSheetBox/TabSheetBox.OnCellClick.htm)


	- [TabSheetBox.OnCellPictureClick](../../Class/TabSheetBox/TabSheetBox.OnCellPictureClick.htm)


	- [TabSheetBox.OnHyperlinkClick](../../Class/TabSheetBox/TabSheetBox.OnHyperlinkClick.htm)
