# ITabIconExporter


## Описание


Интерфейс ITabIconExporter определяет
 параметры экспорта значка таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [IconIndex](ITabIconExporter.IconIndex.htm)
		 Свойство IconIndex
		 определяет индекс экспортируемого значка.


		 ![](../../Property_Image.gif)
		 [PixelHeight](ITabIconExporter.PixelHeight.htm)
		 Свойство PixelHeight
		 определяет высоту экспортируемого значка в пикселях.


		 ![](../../Property_Image.gif)
		 [PixelWidth](ITabIconExporter.PixelWidth.htm)
		 Свойство PixelWidth
		 определяет ширину экспортируемого значка в пикселях.


		 ![](../../Property_Image.gif)
		 [TabSheet](ITabIconExporter.TabSheet.htm)
		 Свойство TabSheet определяет
		 таблицу, значок которой необходимо экспортировать.


## Свойства, унаследованные от [IExporter](ModDrawing.chm::/Interface/IExporter/IExporter.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CSSClassName](ModDrawing.chm::/Interface/IExporter/IExporter.CSSClassName.htm)


		 Свойство CSSClassName
		 определяет префикс для элементов таблицы стилей HTML.


		 ![](../../Property_Image.gif)
		 [Encoding](ModDrawing.chm::/Interface/IExporter/IExporter.Encoding.htm)


		 Свойство Encoding устанавливает
		 кодовую страницу таблицы преобразования, в соответствии с которой
		 будет осуществляться экспорт данных.


		 ![](../../Property_Image.gif)
		 [ExportFromWeb](ModDrawing.chm::/Interface/IExporter/IExporter.ExportFromWeb.htm)


		 Свойство ExportFromWeb
		 определяет, какое действие производится с отчетом в веб-приложении
		 - экспорт или открытие в режиме предварительного просмотра.
