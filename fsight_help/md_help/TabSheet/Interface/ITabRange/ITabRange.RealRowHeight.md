# ITabRange.RealRowHeight


## Синтаксис


RealRowHeight: Double;


## Описание


Свойство RealRowHeight возвращает
 реальную высоту для скрытых строк.


## Комментарии


Свойство возвращает высоту, которую имели строки перед тем, как для
 диапазона свойству [Hidden](ITabRange.Hidden.htm) было установлено
 значение [TriState.OnOption](ForeSys.chm::/Enums/TriState.htm).
 Если размер строк был разным, то свойство вернет размер первой верхней
 строки.


Значение измеряется в миллиметрах.
