# ITabRange.StyleForDraw


## Синтаксис


StyleForDraw([RealBorders: Boolean = False]): [ITabCellStyle](../ITabCellStyle/ITabCellStyle.htm);


## Параметры


RealBorders. Признак учёта
 границ, принадлежащих только текущей ячейке.


## Описание


Свойство StyleForDraw возвращает
 стиль ячейки, учитывающий наложение стилей других диапазонов.


## Комментарии


Свойство доступно для диапазона [ITabRange](ITabRange.htm),
 состоящего только из одной ячейки. Для других диапазонов свойство вернет
 значение Null.


Свойство вернёт стиль, полученный путем совмещения стилей различных
 диапазонов, в которые может входить ячейка. Также учитываются [стили](../ITabSheet/ITabSheet.TableStyles.htm),
 созданные для всей таблицы.


Если параметру RealBorders
 установлено значение True, то
 при получении стиля границ ячейки будут учитываться границы соседних ячеек,
 которые являются общими с текущей ячейкой. По умолчанию параметру RealBorders установлено значение
 False, при этом учитываются только
 те параметры границ, которые заданы в стиле текущей ячейки.
