# ITabRange.Part


## Синтаксис


Part(Index: Integer): [ITabRange](ITabRange.htm);


## Параметры


Index. Индекс
 области, которую необходимо получить.


## Описание


Свойство Part возвращает отдельную
 область из всего диапазона, индекс которой передается в качестве входного
 параметра Index.


## Примечание


Свойство доступно для использования только тогда, когда свойство [ITabRange.PartCount](ITabRange.PartCount.htm) вернет значение
 больше нуля.
