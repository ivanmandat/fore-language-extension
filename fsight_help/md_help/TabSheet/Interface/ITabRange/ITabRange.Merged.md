# ITabRange.Merged


## Синтаксис


Merged: [TriState](ForeSys.chm::/Enums/TriState.htm);


## Описание


Свойство Merged возвращает признак
 наличия объединения диапазона.


## Комментарии


Свойство возвращает следующие значения:


	- -1 : OnOption. Весь
	 диапазон является объединенным;


	-  0 : OffOption.
	 Диапазон не содержит объединенных ячеек;


	- -2 : Undefined. Остальные
	 случаи:


		- Диапазон содержит объединенные и не объединенные ячейки;


		- Диапазон содержит часть объединенной ячейки.
