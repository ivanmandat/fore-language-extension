# ITabRange.Location


## Синтаксис


Location: [IGxPointF](ModDrawing.chm::/Interface/IGxPointF/IGxPointF.htm);


## Описание


Свойство Location возвращает
 верхнюю левую точку диапазона, полученную без учета пиксельной подгонки
 размеров строк/столбцов.


## Комментарии


При отрисовке строк/столбцов таблицы в ядре платформы осуществляется
 преобразование миллиметровых размеров в пиксельные. Для того чтобы строки/столбцы
 имеющие одинаковые размеры в миллиметрах, также имели одинаковые размеры
 в пикселях используется специальный механизм пиксельной
 подгонки. Данное свойство не учитывает разницу, которая может возникнуть
 при преобразовании размеров из миллиметров в пиксели. На экране расположение
 этой точки будет отличаться от положения визуальной верхней левой точки
 диапазона.


Примечание.
 Чем дальше расположен диапазон от верхнего левого угла таблицы, тем больше
 будет составлять разница в координатах точки Location
 и визуальной верхней левой точки диапазона.
