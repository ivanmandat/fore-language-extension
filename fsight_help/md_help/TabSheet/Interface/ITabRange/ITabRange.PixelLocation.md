# ITabRange.PixelLocation


## Синтаксис


PixelLocation: [IGxPointF](ModDrawing.chm::/Interface/IGxPointF/IGxPointF.htm);


## Описание


Свойство PixelLocation возвращает
 верхнюю левую точку диапазона в пикселях, полученную с учетом пиксельной
 подгонки размеров строк/столбцов.


## Комментарии


При отрисовке строк/столбцов таблицы в ядре платформы осуществляется
 преобразование миллиметровых размеров в пиксельные. Для того чтобы строки/столбцы
 имеющие одинаковые размеры в миллиметрах, также имели одинаковые размеры
 в пикселях используется специальный механизм пиксельной
 подгонки. Данное свойство учитывает разницу, которая может возникнуть
 при преобразовании размеров из миллиметров в пиксели.
