# ITabRange.Hidden


## Синтаксис


Hidden: [TriState](ForeSys.chm::/Enums/TriState.htm);


## Описание


Свойство Hidden определяет видимость
 строк/столбцов.


## Комментарии


Данное свойство актуально только для тех диапазонов, которые состоят
 из строк либо столбцов.


Если свойству установить значение TriState.OnOption,
 то диапазон будет скрыт. Для возврата отображения диапазона необходимо
 выполнить одно из следующих действий:


	- Установить свойству Hidden
	 значение TriState.OffOption.


	- Установить не нулевые значения ширины/высоты для столбцов/строк
	 диапазона.


Свойство возвращает значение TriState.Undefined
 если текущий диапазон не является диапазоном столбцов или строк, либо
 уже содержит скрытые столбцы/строки.


## Пример


			Var

    TSheet: ITabSheet;

    TRange: ITabRange;

Begin

    TSheet := //Получение параметров таблицы;

    TRange := TSheet.View.Selection.Range;

    If TRange.IsColumn Or TRange.IsRow Then

        TRange.Hidden := TriState.OnOption;

    End If;


При выполнении примера будет получен диапазон, выделенный в текущий
 момент в таблице. Если полученный диапазон состоит только из строк либо
 столбцов, то он будет скрыт.
