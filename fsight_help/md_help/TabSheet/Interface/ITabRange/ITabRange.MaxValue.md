# ITabRange.MaxValue


## Синтаксис


MaxValue([bIgnoreString: Boolean = true]): Variant;


## Параметры


bIgnoreString. Признак игнорирования
 строковых значений.


## Описание


Метод MaxValue ищет наибольшее
 значение в диапазоне.


## Комментарии


Если параметр bIgnoreString принимает значение false,
 и в диапазоне есть текстовое значение, то будет выведено исключение с
 текстом #Н/Д.
