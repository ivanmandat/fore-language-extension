# ITabRange.PartCount


## Синтаксис


PartCount: Integer;


## Описание


Свойство PartCount возвращает
 количество отдельных областей в диапазоне.


## Примечание


Диапазон из отдельных областей формируется, если в таблице были выделены
 ячейки с зажатой клавишей CTRL или диапазон был получен с помощью свойства
 [ITabRange.DistinctCells](ITabRange.DistinctCells.htm).
