# ITabRange.ClearContent


## Синтаксис


ClearContent;


## Описание


Метод ClearContent удаляет все
 данные диапазона без очистки установленного форматирования диапазона.


## Комментарии


Использование метода актуально, если свойство [ITabCellStyle.Locked](../ITabCellStyle/ITabCellStyle.Locked.htm)
 принимает значение по умолчанию, при котором выключена защита ячеек от
 редактирования.
