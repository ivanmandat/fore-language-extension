# ITabRange.Text


## Синтаксис


Text: String;


## Описание


Свойство Text определяет текст
 диапазона ячеек.


## Комментарии


Текст отображается при просмотре ячеек. Если ячейка переведена в режим
 редактирования, то будет отображено значение свойства [Value](ITabRange.Value.htm).
 Если в ячейке используется [редактор](UiNavCommon.chm::/GUI/ValueEditorParameters.htm),
 то свойство Text вернет строковое
 представление заданного в нем значения.


Свойство Text может определять
 текст в формате RTF, если для свойства [ITabRange.IsRtflText](ITabRange.IsRtfText.htm)
 установлено значение True. Информация
 о работе с RTF-редактором представлена в статье «[Использование
 форматированного текста](KnowledgeBase.chm::/01_Fore/knowledgebase_kb000034.htm)». Если задан форматированный
 текст, то фактическое строковое представление этого текста можно получить
 в свойстве [ITabRange.Value](ITabRange.Value.htm).
