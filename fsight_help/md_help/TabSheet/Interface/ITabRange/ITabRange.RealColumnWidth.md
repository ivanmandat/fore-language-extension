# ITabRange.RealColumnWidth


## Синтаксис


RealColumnWidth: Double;


## Описание


Свойство RealColumnWidth возвращает
 реальную ширину для скрытых столбцов.


## Комментарии


Свойство возвращает ширину, которую имели столбцы перед тем, как для
 диапазона свойству [Hidden](ITabRange.Hidden.htm) было установлено
 значение [TriState.OnOption](ForeSys.chm::/Enums/TriState.htm).
 Если размер столбцов был разным, то свойство вернет размер первого левого
 столбца.


Значение измеряется в миллиметрах.
