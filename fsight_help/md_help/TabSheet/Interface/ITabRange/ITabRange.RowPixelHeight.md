# ITabRange.RowPixelHeight


## Синтаксис


RowPixelHeight: Integer;


## Описание


Свойство RowPixelHeight определяет высоту строк диапазона в пикселях.


## Комментарии


Если у строк в диапазоне разная высота, свойство возвращает значение «-1».
