# ITabRange.AdjustHeightEx


## Синтаксис


AdjustHeightEx([MaxHeight: Double = -1;][MinHeight:
 Double = -1][Flag: [TabRangeAdjustHeightFlags](../../Enums/TabRangeAdjustHeightFlags.htm)
 = 0]);


## Параметры


MaxHeight. Максимальная высота
 строки, которая может быть установлена при автоматической подгонке. По
 умолчанию установлено значение "-1", при этом максимальная высота
 ограничивается максимальной высотой содержимого ячеек диапазона. Значение
 параметра указывается в миллиметрах.


MinHeight. Минимальная
 высота строки, которая может быть установлена при автоматической подгонке.
 По умолчанию установлено значение "-1", при этом минимальная
 высота ограничивается минимальной высотой содержимого ячеек диапазона.
 Значение параметра указывается в миллиметрах.


Flag.
 Флаг, определяющий поведение при установленном пользовательском размере.
 По умолчанию установлено значение "0", при этом высота строки
 автоматически подгоняется, независимо от установки пользовательского размера.
 При значении "1" метод AdjustHeightEx не будет подгонять высоту
 строки, если был установлен пользовательский размер строки.


## Описание


Метод AdjustHeightEx выполняет
 автоматическую подгонку высоты строк диапазона в соответствии с содержимым
 ячеек, если не установлен пользовательский размер.


## Комментарии


При установленном параметре Flag
 = "0" метод работает аналогично методу [AdjustHeight](ITabRange.AdjustHeight.htm).


При установленном параметре Flag
 = "1" метод не будет подгонять высоту, если был установлен пользовательский
 размер строки.
