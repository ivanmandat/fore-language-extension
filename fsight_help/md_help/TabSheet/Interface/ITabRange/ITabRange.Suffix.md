# ITabRange.Suffix


## Синтаксис


Suffix: Variant;


## Описание


Свойство Suffix определяет суффикс
 ячейки таблицы.


## Комментарии


Суффикс отображается после значения ячейки таблицы. Не участвует в расчете
 условного форматирования или вычисляемых формулах.
