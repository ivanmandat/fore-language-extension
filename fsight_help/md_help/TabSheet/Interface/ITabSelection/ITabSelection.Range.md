# ITabSelection.Range


## Синтаксис


Range: [ITabRange](../ITabRange/ITabRange.htm);


## Описание


Свойство Range определяет диапазон
 отмеченных ячеек, реализуемый интерфейсом ITabRange.


## Комментарии


При получении значения свойства будет получена ссылка на диапазон выделенных
 ячеек.
