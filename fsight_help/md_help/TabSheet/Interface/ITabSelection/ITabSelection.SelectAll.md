# ITabSelection.SelectAll


## Синтаксис


SelectAll;


## Описание


Метод SelectAll выделяет диапазон
 непустых ячеек, расположенных около активной ячейки/диапазона.


## Комментарий


Если у активной ячейки/диапазона отсутствуют соседние непустые ячейки,
 то выделяется вся таблица (CTRL+A, CTRL + SHIFT + SPACE).
