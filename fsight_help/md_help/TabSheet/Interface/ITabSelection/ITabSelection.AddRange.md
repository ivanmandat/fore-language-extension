# ITabSelection.AddRange


## Синтаксис


AddRange(Range: [ITabRange](../ITabRange/ITabRange.htm));


## Параметры


Range — диапазон,
 который добавится к выделенному диапазону таблицы.


## Описание


Метод AddRange добавляет к выделенному
 диапазону диапазон, который передается в качестве входного параметра Range.
