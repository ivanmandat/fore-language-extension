# ITabFormulaCollection


## Описание


Интерфейс ITabFormulaCollection
 содержит базовые свойства и методы коллекции областей формул таблицы.


## Комментарии


В текущей реализации области формул используются в регламентных отчетах.
 Для работы с коллекцией областей формул регламентного отчета предназначен
 интерфейс [IPrxFormulaIslands](KeReport.chm::/Interface/IPrxFormulaIslands/IPrxFormulaIslands.htm)[IPrxFormulaIslands](../../../kereport/interface/iprxformulaislands/iprxformulaislands.htm).
