# ITabFormatRankValues


## Описание


Интерфейс ITabFormatRankValues
 содержит свойства условного формата, осуществляющего форматирование ячеек
 с наибольшими/наименьшими значениями.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITabFormatRankValues.Count.htm)
		 Свойство Count определяет
		 количество наибольших/наименьших значений ячеек, которые будут
		 форматироваться.


		 ![](../../Property_Image.gif)
		 [Percent](ITabFormatRankValues.Percent.htm)
		 Свойство Percent определяет
		 процент ячеек от выделенной области, которые будут форматироваться.


		 ![](../../Property_Image.gif)
		 [PercentUsed](ITabFormatRankValues.PercentUsed.htm)
		 Свойство PercentUsed
		 определяет, будет ли количество форматируемых ячеек, определяться
		 в процентном соотношении от выделенной области.


		 ![](../../Property_Image.gif)
		 [Style](ITabFormatRankValues.Style.htm)
		 Свойство Style определяет
		 стиль оформления, применяемый для ячеек, удовлетворяющих установленному
		 условию.


		 ![](../../Property_Image.gif)
		 [Type](ITabFormatRankValues.Type.htm)
		 Свойство Type определяет
		 тип значений, которые необходимо форматировать.
