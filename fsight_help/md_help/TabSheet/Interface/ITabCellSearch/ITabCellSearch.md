# ITabCellSearch


Сборка: Tab;


## Описание


Интерфейс ITabCellSearch содержит
 свойства и методы для поиска значения в ячейках.


## Иерархия наследования


           [ITabSearch](../ITabSearch/ITabSearch.htm)


           ITabCellSearch


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CaseSensitive](ITabCellSearch.CaseSensitive.htm)
		 Свойство CaseSensitive
		 определяет, будет ли учитываться регистр символов при поиске.


		 ![](../../Property_Image.gif)
		 [Direction](ITabCellSearch.Direction.htm)
		 Свойство Direction
		 определяет направление, в котором будет производиться поиск.


		 ![](../../Property_Image.gif)
		 [Range](ITabCellSearch.Range.htm)
		 Свойство Range определяет
		 диапазон, в котором будет производиться поиск.


		 ![](../../Property_Image.gif)
		 [RequiredStyle](ITabCellSearch.RequiredStyle.htm)
		 Свойство RequiredStyle
		 определяет, по ячейкам с каким стилем будет производиться поиск.


		 ![](../../Property_Image.gif)
		 [Target](ITabCellSearch.Target.htm)
		 Свойство Target определяет,
		 по каким свойствам ячейки будет осуществляться поиск.


		 ![](../../Property_Image.gif)
		 [Text](ITabCellSearch.Text.htm)
		 Свойство Text определяет
		 текст, который будет искаться в ячейках.


		 ![](../../Property_Image.gif)
		 [WholeWordsOnly](ITabCellSearch.WholeWordsOnly.htm)
		 Свойство WholeWordsOnly
		 определяет, будет ли искаться слово целиком.
