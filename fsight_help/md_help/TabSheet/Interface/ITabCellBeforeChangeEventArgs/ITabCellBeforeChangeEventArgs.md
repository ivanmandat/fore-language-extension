# ITabCellBeforeChangeEventArgs


## Описание


Интерфейс ITabCellBeforeChangeEventArgs
 содержит свойства аргумента события [TabSheetBox.OnBeforeCellChange](../../Class/TabSheetBox/TabSheetBox.OnBeforeCellChange.htm),
 происходящего перед изменением значения ячейки таблицы компонента TabSheetBox.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Value](ITabCellBeforeChangeEventArgs.Value.htm)
		 Свойство Value определяет
		 измененное значение, которое будет содержаться в ячейке после
		 отработки события.
