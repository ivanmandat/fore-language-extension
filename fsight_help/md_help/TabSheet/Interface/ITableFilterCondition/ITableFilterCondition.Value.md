# ITableFilterCondition.Value


## Синтаксис


Value: Variant;


## Описание


Свойство Value определяет значение
 ограничения.


## Комментарии


При задании ограничения в данном свойстве происходит определение типа
 и преобразование значения ограничения к этому типу.
