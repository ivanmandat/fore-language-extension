# ITableFilterCondition


Сборка: Tab;


## Описание


Интерфейс ITableFilterCondition
 содержит свойства и методы для управления условиями фильтрации.


## Иерархия наследования


           ITableFilterCondition


## Комментарии


При описании условия необходимо задать свойства [Condition](ITableFilterCondition.Condition.htm),
 [Relation](ITableFilterCondition.Relation.htm) и одно из свойств
 [Value](ITableFilterCondition.Value.htm) или [TextValue](ITableFilterCondition.TextValue.htm).
