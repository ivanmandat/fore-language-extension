# ITableFilterItem.SortState


## Синтаксис


SortState : [SortDirection](foresys.chm::/enums/sortdirection.htm);


## Описание


Свойство SortState возвращает
 тип установленной сортировки.


## Комментарии


Возможно три варианта сортировки: по возрастанию, по убыванию и без
 сортировки.


Свойство используется только для настольного приложения.
