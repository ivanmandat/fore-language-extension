# ITableFilterItem.IsFiltered


## Синтаксис


IsFiltered(FormattedValue: String; Value: Variant):
 Boolean;


## Параметры


FormattedValue. Параметр
 определяет значение, для которого в ячейке установлен формат;


Value. Параметр определяет
 значение, для которого формат не установлен.


## Описание


Свойство IsFiltered определяет,
 удовлетворяет ли указанное значение настройкам автофильтра, даже в том
 случае, если такого значения в таблице нет.
