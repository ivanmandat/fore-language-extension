# ITableFilterItem


Сборка: Tab;


## Описание


Интерфейс ITableFilterItem содержит
 свойства и методы элементов, определяющих фильтрацию по столбцу.


## Иерархия наследования


           ITableFilterItem


## Комментарии


Для настройки параметров автофильтрации таблиц используйте интерфейс
 [ITableFilter](../ITableFilter/itablefilter.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Conditions](ITableFilterItem.Conditions.htm)
		 Свойство Conditions
		 возвращает коллекцию условий фильтрации.


		 ![](../../Property_Image.gif)
		 [Icon](ITableFilterItem.Icon.htm)
		 Свойство Icon определяет
		 пиктограмму кнопки автофильтра для соответствующего состояния.


		 ![](../../Property_Image.gif)
		 [IsFiltered](ITableFilterItem.IsFiltered.htm)
		 Свойство IsFiltered
		 определяет, удовлетворяет ли указанное значение настройкам автофильтра.


		 ![](../../Property_Image.gif)
		 [LimitCondition](ITableFilterItem.LimitCondition.htm)
		 Свойство LimitCondition
		 определяет условия фильтрации для типа фильтрации «Первые
		 N».


		 ![](../../Property_Image.gif)
		 [SortState](ITableFilterItem.SortState.htm)
		 Свойство SortState
		 возвращает тип установленной сортировки.


		 ![](../../Property_Image.gif)
		 [State](ITableFilterItem.State.htm)


		 Свойство State определяет
		 тип фильтрации.


		 ![](../../Property_Image.gif)
		 [ValuesNumber](ITableFilterItem.ValuesNumber.htm)
		 Свойство ValuesNumber
		 определяет количество уникальных значений, доступных для выбора.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Sort](ITableFilterItem.Sort.htm)
		 Метод Sort определяет
		 направление сортировки, которое будет установлено в столбце.
