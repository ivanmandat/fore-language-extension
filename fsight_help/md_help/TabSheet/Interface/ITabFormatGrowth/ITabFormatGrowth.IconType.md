# ITabFormatGrowth.IconType


## Синтаксис


IconType: [TabConditionIconType](../../Enums/TabConditionIconType.htm);


## Описание


Свойство IconType определяет тип пиктограмм, используемых при форматирования ячеек.


## Комментарии


По умолчанию используется тип пиктограмм - [TabConditionIconType.Arrows](../../Enums/TabConditionIconType.htm).
