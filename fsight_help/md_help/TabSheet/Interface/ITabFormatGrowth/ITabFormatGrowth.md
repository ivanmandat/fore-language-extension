# ITabFormatGrowth


Сборка: Tab;


## Описание


Интерфейс ITabFormatGrowth содержит
 свойства условного формата, позволяющего сформировать индикатор роста
 относительно значений первой строки/столбца диапазона.


## Иерархия наследования


           ITabFormatGrowth


## Комментарии


Параметры условного формата задаются в свойстве [ITabFormatCondition.Details](../ITabFormatCondition/ITabFormatCondition.Details.htm).
