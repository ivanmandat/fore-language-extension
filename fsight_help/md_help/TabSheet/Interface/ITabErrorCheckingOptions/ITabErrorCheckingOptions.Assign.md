# ITabErrorCheckingOptions.Assign


## Синтаксис


Assign(Result: [ITabErrorCheckingOptions](ITabErrorCheckingOptions.htm));


## Параметры


Result. Параметры индикации
 ошибок, в соответствии с которыми будут установлены текущие параметры.


## Описание


Метод Assign заполняет параметры
 индикации ошибок в соответствии параметрами, переданными в Result.


## Комментарии


Для сброса пропущенных ошибок используйте метод [ITabErrorCheckingOptions.ResetSkippedErrors](ITabErrorCheckingOptions.ResetSkippedErrors.htm).
