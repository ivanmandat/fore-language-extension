# ITabErrorCheckingOptions.ResetSkippedErrors


## Синтаксис


ResetSkippedErrors;


## Описание


Метод ResetSkippedErrors осуществляет
 сброс пропущенных ошибок.


## Комментарии


При выполнении метода будет возвращена индикация ячейкам, которые содержали
 ошибки в формулах, но были пропущены пользователем при помощи команды
 «Пропустить ошибку».
