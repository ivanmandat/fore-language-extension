# ITabErrorCheckingOptions.ErrorIndicatorsColor


## Синтаксис


		ErrorIndicatorsColor: [IGxColor](ModDrawing.chm::/Interface/IGxColor/IGxColor.htm);


## Описание


Свойство ErrorIndicatorsColor
 определяет цвет индикатора, отображаемого в ячейках, формулы которых содержат
 ошибки.


## Комментарии


Ячейки, формулы которых имеют ошибки, подсвечиваются цветовым индикатором
 в виде цветного треугольника в верхнем левом углу ячейки. Данное свойство
 определяет цвет этого треугольника. По умолчанию индикатор имеет зеленый
 цвет. Свойство распространяется для всех листов отчета и сохраняется в
 реестре для текущего пользователя.
