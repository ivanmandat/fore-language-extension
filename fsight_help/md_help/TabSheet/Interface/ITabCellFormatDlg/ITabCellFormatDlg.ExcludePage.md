# ITabCellFormatDlg.ExcludePage


## Синтаксис


ExcludePage(Value: [TabFormatDlgPages](../../Enums/TabFormatDlgPages.htm));


## Параметры


Value - страница диалога, которая будет скрыта при отображении диалога форматирования.


## Описание


Метод ExcludePage исключает указанную страницу из списка страниц, отображаемых в диалоге форматирования.


## Комментарии


Для включения страниц в список отображаемых используйте метод [IncludePage](ITabCellFormatDlg.IncludePage.htm).
