# ITabCellFormatDlg


Сборка: Tab;


## Описание


Интерфейс ITabCellFormatDlg
 содержит свойства и методы для работы с диалогом форматирования.


## Иерархия наследования


           ITabCellFormatDlg


## Комментарии


Для работы со стилем ячеек таблицы используйте свойства интерфейса [ITabCellStyle](../ITabCellStyle/ITabCellStyle.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ActivePage](ITabCellFormatDlg.ActivePage.htm)
		 Свойство ActivePage
		 определяет активную страницу диалога форматирования.


		 ![](../../Property_Image.gif)
		 [Style](ITabCellFormatDlg.Style.htm)
		 Свойство Style определяет
		 стиль, настройки которого будут отображены в диалоге форматирования.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [DisableItem](ITabCellFormatDlg.DisableItem.htm)
		 Метод DisableItem блокирует
		 возможность изменения настроек указанного элемента при работе
		 диалога форматирования.


		 ![](../../Sub_Image.gif)
		 [EnableItem](ITabCellFormatDlg.EnableItem.htm)
		 Метод EnableItem снимает
		 запрет на изменение настроек указанного элемента при работе диалога
		 форматирования.


		 ![](../../Sub_Image.gif)
		 [ExcludePage](ITabCellFormatDlg.ExcludePage.htm)
		 Метод ExcludePage исключает
		 указанную страницу из списка страниц, отображаемых в диалоге форматирования.


		 ![](../../Sub_Image.gif)
		 [IncludePage](ITabCellFormatDlg.IncludePage.htm)
		 Метод IncludePage включает
		 указанную страницу в список страниц, отображаемых в диалоге форматирования.


		 ![](../../Sub_Image.gif)
		 [ShowDialog](ITabCellFormatDlg.ShowDialog.htm)
		 Метод ShowDialog отображает
		 диалог форматирования и возвращает результат закрытия данного
		 диалога.
