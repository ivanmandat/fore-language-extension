# ITabCellFormatDlg.DisableItem


## Синтаксис


DisableItem(Value: [TabFormatDlgItems](../../Enums/TabFormatDlgItems.htm));


## Параметры


Value - элемент настроек, который
 необходимо заблокировать.


## Описание


Метод DisableItem блокирует
 возможность изменения настроек указанного элемента при работе диалога
 форматирования.


## Комментарии


Для снятия блокировки используйте метод [EnableItem](ITabCellFormatDlg.EnableItem.htm).
