# ITabCellFormatDlg.Style


## Синтаксис


Style: [ITabCellStyle](../ITabCellStyle/ITabCellStyle.htm);


## Описание


Свойство Style определяет стиль, настройки которого будут отображены в диалоге форматирования.


## Комментарии


После изменения настроек и нажатия в диалоге кнопки «ОК» обновленный стиль будет доступен в данном свойстве. Если при вызове диалога стиль не указан, то будет создан новый стиль с настройками по умолчанию.
