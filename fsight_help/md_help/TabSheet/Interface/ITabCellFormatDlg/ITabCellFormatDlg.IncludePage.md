# ITabCellFormatDlg.IncludePage


## Синтаксис


IncludePage(Value: [TabFormatDlgPages](../../Enums/TabFormatDlgPages.htm));


## Параметры


Value - страница диалога, которая будет доступна при отображении диалога форматирования.


## Описание


Метод IncludePage включает указанную страницу в список страниц, отображаемых в диалоге форматирования.


## Комментарии


Для исключения страниц из списка отображаемых используйте метод [ExcludePage](ITabCellFormatDlg.ExcludePage.htm).
