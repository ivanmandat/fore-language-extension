# ITabCellFormatDlg.EnableItem


## Синтаксис


EnableItem(Value: [TabFormatDlgItems](../../Enums/TabFormatDlgItems.htm));


## Параметры


Value - элемент настроек, который необходимо разблокировать.


## Описание


Метод EnableItem снимает запрет на изменение настроек указанного элемента при работе диалога форматирования.
