# ITabScaleEventArgs


## Описание


Интерфейс ITabScaleEventArgs
 содержит свойства аргумента события [TabSheetBox.OnScaleChange](../../Class/TabSheetBox/TabSheetBox.OnScaleChange.htm),
 возникающего при изменении масштаба таблицы, компонента TabSheetBox.
