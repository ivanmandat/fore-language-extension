# ITabSheet.MaxUsedColumn


## Синтаксис


MaxUsedColumn([WithObjects: Boolean = False]): Integer;


## Параметры


WithObjects – параметр, определяющий
 будут ли учитываться столбцы, в области которых расположены различные
 объекты таблицы. Если в качестве значение передается значение True,
 то данные столбцы будут учитываться.


## Описание


Свойство MaxUsedColumn возвращает
 максимальный индекс используемого столбца таблицы.


## Комментарии


Данное свойство возвращает максимальный индекс столбца, в котором имеются
 визуальные данные, а также могут присутствовать формулы при вычислении
 которых, было получено пустое значение. В зависимости от значения параметра
 WithObjects также могут учитываться
 столбцы, в области которых расположены различные объекты таблицы.
