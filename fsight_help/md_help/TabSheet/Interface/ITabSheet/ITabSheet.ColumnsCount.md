# ITabSheet.ColumnsCount


## Синтаксис


ColumnsCount: Integer;


## Описание


Свойство ColumnsCount определяет
 количество столбцов таблицы.


## Комментарии


При ограничении количества столбцов если вся таблица размещается на
 экране, то не будут отображены полосы прокрутки.


Допустимое значение свойства находится в диапазоне [1, 1048575]. Максимально
 возможное количество столбцов в таблице 1048575.
