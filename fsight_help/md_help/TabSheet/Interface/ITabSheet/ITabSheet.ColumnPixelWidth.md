# ITabSheet.ColumnPixelWidth


## Синтаксис


ColumnPixelWidth(Index: Integer): Integer;


## Параметры


Index. Индекс используемого столбца. Допустимое значение параметра находится в диапазоне [0, [ColumnsCount](ITabSheet.ColumnsCount.htm)).


## Описание


Свойство ColumnPixelWidth определяет ширину столбца в пикселях, индекс которого передается в качестве входного параметра Index.
