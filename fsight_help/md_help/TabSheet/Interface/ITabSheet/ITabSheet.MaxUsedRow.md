# ITabSheet.MaxUsedRow


## Синтаксис


MaxUsedRow([WithObjects: Boolean = False]): Integer;


## Параметры


WithObjects – параметр, определяющий
 будут ли учитываться строки, в области которых расположены различные объекты
 таблицы. Если в качестве значение передается значение Trueто
 данные строки будут учитываться.


## Описание


Свойство MaxUsedRow возвращает
 максимальный индекс используемой строки таблицы.


## Комментарии


Данное свойство возвращает максимальный индекс строки, в которой имеются
 визуальные данные, а также могут присутствовать формулы при вычислении
 которых, было получено пустое значение. В зависимости от значения параметра
 WithObjects также могут учитываться
 строки, в области которых расположены различные объекты таблицы.
