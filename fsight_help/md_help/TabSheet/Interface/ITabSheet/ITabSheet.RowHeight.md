# ITabSheet.RowHeight


## Синтаксис


RowHeight(Index: Integer): Double;


## Параметры


Index. Индекс используемой строки. Допустимое значение параметра находится в диапазоне [0, [RowsCount](ITabSheet.RowsCount.htm)).


## Описание


Свойство RowHeight определяет высоту ячейки в миллиметрах, индекс которой передается в качестве входного параметра Index.
