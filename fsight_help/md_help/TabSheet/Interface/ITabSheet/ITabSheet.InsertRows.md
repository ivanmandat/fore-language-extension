# ITabSheet.InsertRows


## Синтаксис


InsertRows(Row: Integer; Count: Integer);


## Параметры


Row - индекс строки, перед которой добавятся указанное количество строк. Допустимое значение параметра находится в диапазоне [0, [RowsCount](ITabSheet.RowsCount.htm)).


Count - количество добавляемых строк.


## Описание


Метод InsertRows позволяет вставить указанное количество строк (параметр Count) перед строкой, индекс которой передается в качестве входного параметра Row.


## Комментарии


Для корректной работы данного метода должно соблюдаться следующее соотношение параметров: Row + Count <= [RowsCount](ITabSheet.RowsCount.htm).
