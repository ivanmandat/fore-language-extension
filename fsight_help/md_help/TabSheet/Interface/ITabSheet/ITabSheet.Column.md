# ITabSheet.Column


## Синтаксис


Column(Column: Integer): [ITabRange](../ITabRange/ITabRange.htm);


## Параметры


Column. Номер столбца. Допустимое значение параметра находится в диапазоне [0, [ColumnsCount](ITabSheet.ColumnsCount.htm)).


## Описание


Свойство Column возвращает столбец таблицы, номер которого передаются в качестве входного параметра Column.
