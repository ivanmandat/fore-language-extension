# ITabSheet.EndUpdate


## Синтаксис


EndUpdate;


## Описание


Метод EndUpdate возобновляет обычный режим работы таблицы.


## Комментарии


После вызова данного метода в таблице будут включены все функции, отключенные при вызове метода [BeginUpdate](ITabSheet.BeginUpdate.htm).
