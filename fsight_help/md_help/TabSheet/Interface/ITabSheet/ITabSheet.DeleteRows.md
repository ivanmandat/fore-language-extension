# ITabSheet.DeleteRows


## Синтаксис


DeleteRows(Row: Integer; Count: Integer);


## Параметры


Row. Индекс строки, с которой будет начинаться удаление. Допустимое значение параметра находится в диапазоне [0, [RowsCount](ITabSheet.RowsCount.htm)).


Count. Количество удаляемых строк.


## Описание


Метод DeleteRows позволяет удалить указанное количество строк (параметр Count), начиная со строки, индекс которой передается в качестве входного параметра Row.
