# ITabSheet.RowPixelHeight


## Синтаксис


RowPixelHeight(Index: Integer): Integer;


## Параметры


Index. Индекс используемой строки. Допустимое значение параметра находится в диапазоне [0, [RowsCount](ITabSheet.RowsCount.htm)).


## Описание


Свойство RowPixelHeight определяет высоту ячейки в пикселях, индекс которой передается в качестве входного параметра Index.
