# ITabSheet.ColumnWidth


## Синтаксис


ColumnWidth(Index: Integer): Double;


## Параметры


Index - индекс используемого столбца. Допустимое значение параметра находится в диапазоне [0, [ColumnsCount](ITabSheet.ColumnsCount.htm)).


## Описание


Свойство ColumnWidth определяет ширину столбца в миллиметрах, индекс которого передается в качестве входного параметра Index.
