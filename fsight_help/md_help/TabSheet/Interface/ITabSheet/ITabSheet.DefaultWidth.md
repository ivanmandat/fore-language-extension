# ITabSheet.DefaultWidth


## Синтаксис


DefaultWidth: Double;


## Описание


Свойство DefaultWidth определяет ширину столбцов по умолчанию.


## Комментарии


Значение ширины измеряется в миллиметрах. При изменении данного значения, будет изменена ширина всех столбцов, для которых не изменялось значение свойства [ColumnWidth](ITabSheet.ColumnWidth.htm) либо [ColumnPixelWidth](ITabSheet.ColumnPixelWidth.htm).
