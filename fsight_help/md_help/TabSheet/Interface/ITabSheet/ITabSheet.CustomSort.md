# ITabSheet.CustomSort


## Синтаксис


CustomSort:[ITabCustomSort](../ITabCustomSort/ITabCustomSort.htm);


## Описание


Свойство CustomSort определяет параметры настраиваемой сортировки в таблице с использованием различных условий сортировки.


## Пример


Пример использования свойства приведен в описании свойства [ITabCustomSort.CaseSensitive](../ITabCustomSort/ITabCustomSort.CaseSensitive.htm).
