# ITabSheet.ClearPart


## Синтаксис


ClearPart([Part: [TabTableClearPart](../../Enums/TabTableCleanPart.htm) = 7]);


## Параметры


Part - область таблицы, которую необходимо очистить.


## Описание


Метод ClearPart осуществляет очистку заданной области таблицы.


## Комментарии


По умолчанию осуществляется полная очистка таблицы, при этом удаляются все значения и содержимое ячеек, оформления ячеек, строк и столбцов, настройки условных форматов.
