# ITabSheet.ExpanderKind


## Синтаксис


ExpanderKind(Row: Integer; Column: Integer): [TabExpanderKind](../../Enums/TabExpanderKind.htm);


## Параметры


Row. Индекс строки, в которой
 располагается экспандер. Допустимое значение параметра находится в диапазоне
 [0, [RowsCount](ITabSheet.RowsCount.htm)).


Column. Индекс столбца, в котором
 располагается экспандер. Допустимое значение параметра находится в диапазоне
 [0, [ColumnsCount](ITabSheet.ColumnsCount.htm)).


## Описание


Свойство ExpanderKind возвращает
 область действия экспандера, расположенного в ячейке с указанными координатами.
