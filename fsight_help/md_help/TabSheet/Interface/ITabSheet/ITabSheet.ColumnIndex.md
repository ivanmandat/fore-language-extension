# ITabSheet.ColumnIndex


## Синтаксис


ColumnIndex(Column: String): Integer;


## Параметры


Column - наименование столбца.


## Описание


Свойство ColumnIndex возвращает индекс столбца таблицы, наименование которого передается в качестве входного параметра Column.
