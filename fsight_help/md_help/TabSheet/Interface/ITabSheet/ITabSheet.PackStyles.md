# ITabSheet.PackStyles


## Синтаксис


PackStyles;


## Описание


Метод PackStyles удаляет дублирующие
 и неиспользуемые стили внутренней таблицы.


## Комментарии


При настройке оформления для различных диапазонов таблицы в ядре создаются
 отдельные стили оформления. При вызове данного метода осуществляется удаление
 неиспользуемых и дублирующих стилей, имеющих одинаковые настройки оформления.
 Например, если для диапазона ячеек было задано несколько стилей, то при
 вызове данного метода останется только последний добавленный стиль.


Также метод вызывается при вычислении листа, сохранении, вычислении
 и экспорте отчета.
