# ITabSheet.RowsCount


## Синтаксис


RowsCount: Integer;


## Описание


Свойство RowsCount определяет
 количество строк таблицы.


## Комментарии


При ограничении количества строк если вся таблица размещается на экране,
 то не будут отображены полосы прокрутки.


Допустимое значение свойства находится в диапазоне [1, 1048575].Максимально
 возможное количество строк в таблице 1048575.
