# ITabSheet.Footnotes


## Синтаксис


Footnotes: [ITabFootnotes](../ITabFootnotes/ITabFootnotes.htm);


## Описание


Свойство Footnotes возвращает
 коллекцию сносок таблицы.


## Комментарии


Сноски используются для создания каких-либо комментариев к ячейкам таблицы.
 Каждая сноска представляет из себя числовую/текстовую/символьную метку,
 отображаемую в ячейке таблицы рядом с данными, и текстовый комментарий,
 который отображается вместе с меткой в отдельной панели.

Пример использования приведен в описании свойства [ITabFootnote.CustomMark](../ITabFootnote/ITabFootnote.CustomMark.htm).
