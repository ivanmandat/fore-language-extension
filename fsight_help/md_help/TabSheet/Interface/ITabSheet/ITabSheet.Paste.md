# ITabSheet.Paste


## Синтаксис


Paste;


## Описание


Метод Paste вставляет данные
 из буфера обмена в отмеченную область таблицы.


## Комментарии


Числа с дробной частью, скопированные не из «Форсайт. Аналитическая платформа»,
 будут округлены до видимых значений.


Например: число «45,123874», отображаемое в MS Excel как «45,124», будет
 скопировано в «Форсайт. Аналитическая платформа»
 как «45,124».


Используется только в настольном приложении.
