# ITabSheet.ColumnName


## Синтаксис


ColumnName(Column: Integer): String;


## Параметры


Column. Номер столбца. Допустимое значение параметра находится в диапазоне [0, [ColumnsCount](ITabSheet.ColumnsCount.htm)).


## Описание


Свойство ColumnName возвращает адрес столбца таблицы, номер которого передается в качестве входного параметра Column.
