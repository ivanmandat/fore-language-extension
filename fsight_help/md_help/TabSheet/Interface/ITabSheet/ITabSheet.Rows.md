# ITabSheet.Rows


## Синтаксис


Rows(Start: Integer; Finish: Integer): [ITabRange](../ITabRange/ITabRange.htm);


## Параметры


Start. Номер первой строки
 диапазона.


Finish. Номер последней строки
 диапазона.


Допустимое значение всех параметров находится в диапазоне [0, [RowsCount](ITabSheet.RowsCount.htm)).


## Описание


Свойство Rows возвращает строки
 таблицы, диапазон которых передается в качестве входных параметров Start и Finish.
