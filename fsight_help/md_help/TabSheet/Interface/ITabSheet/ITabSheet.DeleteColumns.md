# ITabSheet.DeleteColumns


## Синтаксис


DeleteColumns(Column: Integer; Count: Integer);


## Параметры


Columns. Индекс столбца, с которого будет начинаться удаление. Допустимое значение параметра находится в диапазоне [0, [ColumnsCount](ITabSheet.ColumnsCount.htm)).


Count. Количество удаляемых столбцов.


## Описание


Метод DeleteColumns позволяет удалить указанное количество столбцов (параметр Count), начиная со столбца, индекс которого передается в качестве входного параметра Column.
