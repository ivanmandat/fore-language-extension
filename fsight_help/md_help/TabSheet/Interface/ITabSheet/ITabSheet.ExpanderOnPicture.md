# ITabSheet.ExpanderOnPicture


## Синтаксис


ExpanderOnPicture: Integer;


## Описание


Свойство ExpanderOnPicture определяет
 индекс пиктограммы, соответствующей развернутому экспандеру (![](ExpanderOnPicture.gif)).


## Комментарии


Пиктограмма должна содержаться в коллекции пиктограмм таблицы ([ITabSheet.Images](ITabSheet.Images.htm)).
 Если значение данного свойства не определено или равняется «-1», то будет
 отображаться стандартная пиктограмма.
