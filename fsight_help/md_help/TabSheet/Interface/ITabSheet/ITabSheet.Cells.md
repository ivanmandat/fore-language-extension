# ITabSheet.Cells


## Синтаксис


Cells(


TopRow: Integer;


LeftColumn: Integer;


BottomRow: Integer;


RightColumn: Integer): [ITabRange](../ITabRange/ITabRange.htm);


## Описание


Свойство Cells возвращает диапазон ячеек, координаты которых передаются в качестве входных параметров TopRow, LeftColumn, BottomRow и RightColumn.


## Комментарии


Для получения ячейки в диапазоне таблицы используйте свойство [ITabSheet.ParseCellEx](ITabSheet.ParseCellEx.htm).


## Пример


Примеры использования свойства приведены в описании свойства [ITabRange.Merge](../ITabRange/ITabRange.Merge.htm).
