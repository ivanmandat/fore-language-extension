# ITabSheet.ExpanderOffPicture


## Синтаксис


ExpanderOffPicture: Integer;


## Описание


Свойство ExpanderOffPicture
 определяет индекс пиктограммы, соответствующей свернутому экспандеру (![](ExpanderOffPicture.gif)).


## Комментарии


Пиктограмма должна содержаться в [коллекции
 пиктограмм таблицы](ITabSheet.Images.htm). Если значение данного свойства не определено или
 равняется «-1», то будет отображаться стандартная пиктограмма.
