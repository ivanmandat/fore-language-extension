# ITabSheet.MaxNotEmptyRow


## Синтаксис


MaxNotEmptyRow([WithObjects: Boolean = False]): Integer;


## Параметры


WithObjects – параметр, определяющий будут ли учитываться строки, в области которых расположены различные объекты таблицы. Если в качестве значение передается значение True, то данные строки будут учитываться.


## Описание


Свойство MaxNotEmptyRow возвращает максимальный индекс строки, в которой содержатся данные.


## Комментарии


Данное свойство возвращает максимальный индекс строки, в которой имеются визуальные данные. Свойство не учитывает строки, в которых могут иметься формулы при вычислении которых, было получено пустое значение. В зависимости от значения параметра WithObjects также могут учитываться строки, в области которых расположены различные объекты таблицы.
