# ITabSheet.DefaultHeight


## Синтаксис


DefaultHeight: Double;


## Описание


Свойство DefaultHeight определяет высоту строк по умолчанию.


## Комментарии


Значение высоты измеряется в миллиметрах. При изменении данного значения, будет изменена высота всех строк, для которых не изменялось значение свойства [RowHeight](../ITabRange/ITabRange.RowHeight.htm) либо [RowPixelHeight](ITabSheet.RowPixelHeight.htm).
