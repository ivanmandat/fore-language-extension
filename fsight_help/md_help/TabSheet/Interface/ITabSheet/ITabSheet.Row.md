# ITabSheet.Row


## Синтаксис


Row(Row: Integer): [ITabRange](../ITabRange/ITabRange.htm);


## Параметры


Row - номер строки. Допустимое значение параметра находится в диапазоне [0, [RowsCount](ITabSheet.RowsCount.htm)).


## Описание


Свойство Row возвращает строку таблицы, номер которой передаются в качестве входного параметра Row.
