# ITabSheetPrinter.FirstPageFooterGap


## Синтаксис


		FirstPageFooterGap: Double;


## Описание


Свойство FirstPageFooterGap
 определяет отступ для нижнего колонтитула на первой странице.


## Комментарии


Для определения отступа для заголовка на первой странице используйте
 свойство [ITabSheetPrinter.FirstPageHeaderGap](ITabSheetPrinter.FirstPageHeaderGap.htm).
