# ITabSheetPrinter.DifferentEvenPageGap


## Синтаксис


DifferentEvenPageGap: Boolean;


## Описание


Свойство DifferentEvenPageGap
 определяет, нужно ли учитывать настройки отступа для чётных страниц.


## Комментарии


Для определения отступа для заголовка на чётных страницах используйте
 свойство [ITabSheetPrinter.EvenHeaderGap](ITabSheetPrinter.EvenHeaderGap.htm).
