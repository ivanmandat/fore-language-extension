# ITabSheetPrinter.EvenHeaderGap


## Синтаксис


		EvenHeaderGap: Double;


## Описание


Свойство EvenHeaderGap определяет
 отступ для верхнего колонтитула на чётных страницах.


## Комментарии


Для определения отступа для нижнего колонтитула на чётных страницах
 используйте свойство [ITabSheetPrinter.EvenFooterGap](ITabSheetPrinter.EvenFooterGap.htm).
