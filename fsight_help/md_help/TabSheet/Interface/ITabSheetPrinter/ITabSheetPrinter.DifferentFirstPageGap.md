# ITabSheetPrinter.DifferentFirstPageGap


## Синтаксис


		DifferentFirstPageGap: Boolean;


## Описание


Свойство DifferentFirstPageGap
 определяет, нужно ли учитывать настройки отступа для первой страницы.


## Комментарии


Для определения отступа для заголовка на первой странице используйте
 свойство [ITabSheetPrinter.FirstPageHeaderGap](ITabSheetPrinter.FirstPageHeaderGap.htm).
