# ITabSheetPrinter.EvenFooterGap


## Синтаксис


EvenFooterGap: Double;


## Описание


Свойство EvenFooterGap определяет
 отступ для нижнего колонтитула на чётных страницах.


## Комментарии


Для определения отступа для заголовка на чётных страницах используйте
 свойство [ITabSheetPrinter.EvenHeaderGap](ITabSheetPrinter.EvenHeaderGap.htm).
