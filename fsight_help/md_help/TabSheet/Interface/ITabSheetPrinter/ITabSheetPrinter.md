# ITabSheetPrinter


Сборка: Tab;


## Описание


Интерфейс ITabSheetPrinter содержит
 свойства и методы для печати таблицы.


## Иерархия наследования


           ITabSheetPrinter


## Комментарии


Для экспорта таблицы используйте свойства и методы интерфейса [ITabSheetExporter](../ITabSheetExporter/ITabSheetExporter.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DifferentEvenPageGap](ITabSheetPrinter.DifferentEvenPageGap.htm)
		 Свойство DifferentEvenPageGap
		 определяет, нужно ли учитывать настройки отступа для чётных страниц.


		 ![](../../Property_Image.gif)
		 [DifferentFirstPageGap](ITabSheetPrinter.DifferentFirstPageGap.htm)
		 Свойство DifferentFirstPageGap
		 определяет, нужно ли учитывать настройки отступа для первой страницы.


		 ![](../../Property_Image.gif)
		 [EvenFooterGap](ITabSheetPrinter.EvenFooterGap.htm)
		 Свойство EvenFooterGap
		 определяет отступ для нижнего колонтитула на чётных страницах.


		 ![](../../Property_Image.gif)
		 [EvenHeaderGap](ITabSheetPrinter.EvenHeaderGap.htm)
		 Свойство EvenHeaderGap
		 определяет отступ для верхнего колонтитула на чётных страницах.


		 ![](../../Property_Image.gif)
		 [FirstPageFooterGap](ITabSheetPrinter.FirstPageFooterGap.htm)
		 Свойство FirstPageFooterGap
		 определяет отступ для нижнего колонтитула на первой странице.


		 ![](../../Property_Image.gif)
		 [FirstPageGap](ITabSheetPrinter.FirstPageGap.htm)
		 Свойство FirstPageGap
		 определяет отступ на первой странице.


		 ![](../../Property_Image.gif)
		 [FirstPageHeaderGap](ITabSheetPrinter.FirstPageHeaderGap.htm)
		 Свойство FirstPageHeaderGap
		 определяет отступ для верхнего колонтитула на первой странице.


		 ![](../../Property_Image.gif)
		 [FooterGap](ITabSheetPrinter.FooterGap.htm)
		 Свойство FooterGap
		 определяет отступ для нижнего колонтитула на каждой странице.


		 ![](../../Property_Image.gif)
		 [HeaderGap](ITabSheetPrinter.HeaderGap.htm)
		 Свойство HeaderGap
		 определяет отступ для верхнего колонтитула на каждой странице.


		 ![](../../Property_Image.gif)
		 [PagePrintableRect](ITabSheetPrinter.PagePrintableRect.htm)
		 Свойство PagePrintableRect
		 возвращает диапазон ячеек, который будет напечатан на указанной
		 странице.


		 ![](../../Property_Image.gif)
		 [PageSettings](ITabSheetPrinter.PageSettings.htm)
		 Свойство PageSettings
		 возвращает параметры страницы таблицы.
