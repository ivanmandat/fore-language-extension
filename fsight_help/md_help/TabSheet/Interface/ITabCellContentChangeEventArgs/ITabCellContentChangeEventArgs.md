# ITabCellContentChangeEventArgs


## Описание


Интерфейс ITabCellContentChangeEventArgs
 предназначен для работы с параметрами события [TabSheetBox.OnChangeCellContent](../../Class/TabSheetBox/TabSheetBox.OnChangeCellContent.htm)
 (изменение содержимого в ячейке таблицы).
