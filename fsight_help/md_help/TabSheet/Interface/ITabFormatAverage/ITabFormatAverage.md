# ITabFormatAverage


## Описание


Интерфейс ITabFormatAverage
 содержит свойства условного формата, осуществляющего форматирование ячеек
 со значениями больше/меньше среднего.

		 ![](../../Property_Image.gif)
		 [Type](ITabFormatAverage.Type.htm)
		 Свойство Type определяет
