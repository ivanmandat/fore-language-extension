# ITabCellStyle.Assign


## Синтаксис


Assign(Value: [ITabCellStyle](ITabCellStyle.htm));


## Параметры


Value - стиль, параметры которого необходимо установить для текущего стиля.


## Описание


Метод Assign заполняет параметры текущего стиля параметрами стиля, указанного в качестве входного параметра Value.


## Комментарии


При вызове данного метода, для текущего стиля будут применены все параметры, которые определены для исходного стиля, передаваемого в параметре Value. Параметры, определенные в текущем стиле и неопределенные в исходном, будут сброшены.
