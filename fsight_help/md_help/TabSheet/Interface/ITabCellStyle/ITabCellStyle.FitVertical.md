# ITabCellStyle.FitVertical


## Синтаксис


FitVertical: [TriState](ForeSys.chm::/Enums/TriState.htm);


## Описание


Свойство FitVertical определяет признак масштабирования изображений при изменении высоты ячейки.


## Комментарии


Изображения задаются в свойствах [NormalPicture](ITabCellStyle.NormalPicture.htm), [HoverPicture](ITabCellStyle.HoverPicture.htm) и [PushedPicture](ITabCellStyle.PushedPicture.htm). По умолчанию свойству FitVertical установлено значение [TriState.OffOption](ForeSys.chm::/Enums/TriState.htm). Если свойству FitVertical установить значение [TriState.OnOption](ForeSys.chm::/Enums/TriState.htm), то при изменении высоты ячейки изображения будут масштабироваться пропорционально.
