# ITabCellStyle.Equal


## Синтаксис


Equal(Value: [ITabCellStyle](ITabCellStyle.htm)): Boolean;


## Параметры


Value — стиль, параметры которого необходимо сравнить с параметрами текущего стиля.


## Описание


Метод Equal осуществляет сравнение параметров текущего стиля с параметрами стиля, указанного в качестве входного параметра Value и возвращает True если стили одинаковые.
