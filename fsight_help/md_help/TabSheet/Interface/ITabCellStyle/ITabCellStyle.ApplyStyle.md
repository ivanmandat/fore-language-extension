# ITabCellStyle.ApplyStyle


## Синтаксис


ApplyStyle(Value: [ITabCellStyle](ITabCellStyle.htm));


## Параметры


Value - стиль, параметры которого необходимо применить для текущего стиля.


## Описание


Метод ApplyStyle осуществляет установку параметров текущего стиля в соответствии с параметрами указанного стиля.


## Комментарии


При вызове данного метода для текущего стиля будут применены только те параметры, которые определены для исходного стиля, передаваемого в параметре Value. Параметры, имеющие неопределенные значения в исходном стиле, в текущем стиле переопределены не будут.
