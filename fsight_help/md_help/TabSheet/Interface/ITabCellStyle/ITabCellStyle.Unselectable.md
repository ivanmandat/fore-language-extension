# ITabCellStyle.Unselectable


## Синтаксис


Unselectable: [TriState](foresys.chm::/Enums/TriState.htm);


## Описание


Свойство Unselectable определяет,
 доступна ли ячейка для выделения.


## Комментарии


Допустимые значения:


	- TriState.Undefined.
	 Выделение не определено;


	- TriState.OnOption. Ячейка
	 недоступна для выделения;


	- TriState.OffOption.
	 По умолчанию. Ячейка доступна для выделения.
