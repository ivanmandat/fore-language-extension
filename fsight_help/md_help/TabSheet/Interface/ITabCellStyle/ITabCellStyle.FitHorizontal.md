# ITabCellStyle.FitHorizontal


## Синтаксис


FitHorizontal: [TriState](ForeSys.chm::/Enums/TriState.htm);


## Описание


Свойство FitHorizontal определяет признак масштабирования изображений при изменении ширины ячейки.


## Комментарии


Изображения задаются в свойствах [NormalPicture](ITabCellStyle.NormalPicture.htm), [HoverPicture](ITabCellStyle.HoverPicture.htm) и [PushedPicture](ITabCellStyle.PushedPicture.htm). По умолчанию свойству FitHorizontal установлено значение [TriState.OffOption](ForeSys.chm::/Enums/TriState.htm). Если свойству FitHorizontal установить значение [TriState.OnOption](ForeSys.chm::/Enums/TriState.htm), то при изменении ширины ячейки изображения будут масштабироваться пропорционально.
