# ITableFilter


Сборка: Tab;


## Описание


Интерфейс ITableFilter управляет
 параметрами автофильтрации таблиц и компонентов типа [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm).


## Иерархия наследования


           ITableFilter


## Комментарии


Для настройки фильтрации по столбцу используйте интерфейс [ITableFilterItem](../ITableFilterItem/ITableFilterItem.htm).
