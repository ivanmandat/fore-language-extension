# ITableFilter.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 элементов в автофильтре.


## Комментарии


Для компонента [DataGrid](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/DataGrid.htm)
 количество элементов в автофильтре соответствует количеству столбцов.


Для таблицы, реализуемой интерфейсом [ITabSheet](../ITabSheet/ITabSheet.htm),
 количество элементов в автофильтре определяется количеством столбцов,
 входящих в диапазон указанный в свойстве [ITabAutoFilter.Range](../ITabAutoFilter/ITabAutoFilter.Range.htm).
