# ITabFormatCellContent


## Описание


Интерфейс ITabFormatCellContent
 содержит свойства условного формата, осуществляющего форматирование ячеек
 с определенными значениями.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CellValue](ITabFormatCellContent.CellValue.htm)
		 Свойство CellValue
		 определяет значение, с которым будут сравниваться значения ячеек
		 на соответствие установленному в свойстве [ValueCondition](ITabFormatCellContent.ValueCondition.htm)
		 условию.


		 ![](../../Property_Image.gif)
		 [ContentType](ITabFormatCellContent.ContentType.htm)
		 Свойство ContentType
		 определяет тип содержимого ячеек, для которого ставится условие
		 форматирования.


		 ![](../../Property_Image.gif)
		 [DateCondition](ITabFormatCellContent.DateCondition.htm)
		 Свойство DateCondition
		 определяет условие по дате, которое будет проверяться для форматируемых
		 ячеек.


		 ![](../../Property_Image.gif)
		 [MaxValue](ITabFormatCellContent.MaxValue.htm)
		 Свойство MaxValue определяет
		 максимальное значение, с которым будут сравниваться значения ячеек
		 на соответствие, установленному в свойстве [ValueCondition](ITabFormatCellContent.ValueCondition.htm),
		 условию.


		 ![](../../Property_Image.gif)
		 [MinValue](ITabFormatCellContent.MinValue.htm)
		 Свойство MinValue определяет
		 минимальное значение, с которым будут сравниваться значения ячеек
		 на соответствие установленному в свойстве [ValueCondition](ITabFormatCellContent.ValueCondition.htm)
		 условию.


		 ![](../../Property_Image.gif)
		 [SpecificText](ITabFormatCellContent.SpecificText.htm)
		 Свойство SpecificText
		 определяет текст, с которым будут сравниваться значения ячеек
		 на соответствие установленному в свойстве [TextCondition](ITabFormatCellContent.TextCondition.htm)
		 условию.


		 ![](../../Property_Image.gif)
		 [Style](ITabFormatCellContent.Style.htm)
		 Свойство Style определяет
		 стиль оформления, применяемый для ячеек, удовлетворяющих установленному
		 условию.


		 ![](../../Property_Image.gif)
		 [TextCondition](ITabFormatCellContent.TextCondition.htm)
		 Свойство TextCondition
		 определяет условие по тексту, которое будет проверяться для форматируемых
		 ячеек.


		 ![](../../Property_Image.gif)
		 [ValueCondition](ITabFormatCellContent.ValueCondition.htm)
		 Свойство ValueCondition
		 определяет условие для значений, которое будет проверяться для
		 форматируемых ячеек.
