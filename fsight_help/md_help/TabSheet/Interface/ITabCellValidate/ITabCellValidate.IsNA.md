# ITabCellValidate.IsNA


## Синтаксис


IsNA(Value: Variant): Boolean;


## Параметры


Value. Значение ячейки.


## Описание


Свойство IsNA проверяет значение
 в ячейке на ошибку доступности.


## Комментарии


Допустимые значения:


	- True. Значение в ячейке
	 ошибка доступности, и ячейка не пустая;


	- False. Значение в ячейке
	 не является ошибкой доступности, или ячейка пустая.
