# ITabCellValidate.IsText


## Синтаксис


ITabCellValidate.IsText(Value: Variant): Boolean;


## Параметры


Value. Значение ячейки.


## Описание


Свойство IsText проверяет наличие
 текстового значения в ячейке.


## Комментарии


Допустимые значения:


	- True. Значение в ячейке
	 текстовое, и ячейка не пустая;


	- False. Значение в ячейке
	 не текстовое, или ячейка пустая.
