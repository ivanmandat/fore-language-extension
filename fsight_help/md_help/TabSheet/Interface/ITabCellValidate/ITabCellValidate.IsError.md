# ITabCellValidate.IsError


## Синтаксис


IsError(Value: Variant): Boolean;


## Параметры


Value. Значение ячейки.


## Описание


Свойство IsError проверяет наличие
 ошибок в значении ячейки.


## Комментарии


Допустимые значения:


	- True. Значение в ячейке
	 ошибка;


	- False. Значение в ячейке
	 не является ошибкой.
