# ITabCellValidate.IsBoolean


## Синтаксис


ITabCellValidate.IsBoolean(Value: Variant): Boolean;


## Параметры


Value.
 Значение ячейки.


## Описание


Свойство IsBoolean проверяет
 наличие логического значения в ячейке.


## Комментарии


Допустимые значения:


	- True. Значение в ячейке
	 логическое, и ячейка не пустая;


	- False. Значение в ячейке
	 не логическое, или ячейка пустая.
