# ITabCellValidate.IsNumber


## Синтаксис


ITabCellValidate.IsNumber(Value: Variant): Boolean;


## Параметры


Value. Значение ячейки.


## Описание


Свойство IsNumber проверяет
 наличие числового значения в ячейке.


## Комментарии


Допустимые значения:


	- True. Значение в ячейке
	 числовое, и ячейка не пустая;


	- False. Значение в ячейке
	 не числовое, или ячейка пустая.
