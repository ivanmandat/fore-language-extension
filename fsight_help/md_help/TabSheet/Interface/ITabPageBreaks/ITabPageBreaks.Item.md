# ITabPageBreaks.Item


## Синтаксис


Item(Index: Integer): Integer;


## Параметры


Index. Индекс разрыва в таблице.


## Описание


Свойство Item возвращает место
 разрыва в таблице.


## Комментарии


В зависимости от того, с какой коллекцией разрывов ведётся работа ([разрывы
 по столбцам](../ITabPageSettings/ITabPageSettings.ColumnsBreaks.htm) или [разрывы
 по строкам](../ITabPageSettings/ITabPageSettings.RowsBreaks.htm)), свойство вернёт индекс столбца или строки.
