# ITabPageBreaks


Сборка: Tab;


## Описание


Интерфейс ITabPageBreaks содержит
 свойства и методы для работы с коллекцией разрывов таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITabPageBreaks.Count.htm)
		 Свойство Count возвращает
		 количество разрывов в таблице.


		 ![](../../Property_Image.gif)
		 [Item](ITabPageBreaks.Item.htm)
		 Свойство Item
		 возвращает место разрыва в таблице.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ITabPageBreaks.Add.htm)
		 Метод Add устанавливает
		 новый разрыв в таблице.


		 ![](../../Sub_Image.gif)
		 [Assign](ITabPageBreaks.Assign.htm)
		 Метод Assign
		 устанавливает для текущей таблицы разрывы, передаваемые
		 в качестве входного параметра.


		 ![](../../Sub_Image.gif)
		 [Remove](ITabPageBreaks.Remove.htm)
		 Метод Remove удаляет
		 указанный разрыв в таблице.


		 ![](../../Sub_Image.gif)
		 [RemoveAll](ITabPageBreaks.RemoveAll.htm)
		 Метод RemoveAll удаляет
		 все разрывы в таблице.
