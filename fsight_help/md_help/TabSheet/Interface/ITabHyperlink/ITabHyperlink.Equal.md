# ITabHyperlink.Equal


## Синтаксис


Equal(Value: [ITabHyperlink](ITabHyperlink.htm)): Boolean;


## Параметры


Value — гиперссылка, параметры которой необходимо сравнить с параметрами текущей гиперссылки.


## Описание


Метод Equal осуществляет сравнение параметров текущей гиперссылки с параметрами гиперссылки, указанной в качестве входного параметра Value.
