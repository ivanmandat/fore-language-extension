# ITabHyperlink


## Описание


Интерфейс ITabHyperlink определяет
 параметры гиперссылки в ячейке.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Action](ITabHyperlink.Action.htm)


		 Свойство Action определяет
		 действие, производимое при срабатывании гиперссылки.


		 ![](../../Property_Image.gif)
		 [ActionType](ITabHyperlink.ActionType.htm)


		 Свойство ActionType
		 определяет тип действия, выполняемого при щелчке по гиперссылке.


		 ![](../../Property_Image.gif)
		 [Active](ITabHyperlink.Active.htm)


		 Свойство Active определяет,
		 была ли использована гиперссылка.


		 ![](../../Property_Image.gif)
		 [Color](ITabHyperlink.Color.htm)


		 Свойство Color определяет
		 цвет гиперссылки в ячейке.


		 ![](../../Property_Image.gif)
		 [Enable](ITabHyperlink.Enable.htm)


		 Свойство Enable определяет,
		 будет ли текст ячейки рассматриваться как гиперссылка.


		 ![](../../Property_Image.gif)
		 [SeparateLinkText](ITabHyperlink.SeparateLinkText.htm)


		 Свойство SeparateLinkText
		 определяет использование отдельного текста для гиперссылки.


		 ![](../../Property_Image.gif)
		 [Target](ITabHyperlink.Target.htm)


		 Свойство Target определяет
		 способ загрузки страницы при переходе по ссылке.


		 ![](../../Property_Image.gif)
		 [Text](ITabHyperlink.Text.htm)


		 Свойство Text определяет
		 текст гиперссылки.


		 [![](../../Property_Image.gif)](ITabHyperlink.Underline.htm)
		 [Underline](ITabHyperlink.Underline.htm)


		 Свойство Underline
		 определяет, будет ли гиперссылка подчеркнутой.


		 ![](../../Property_Image.gif)
		 [UseGlobalSettings](ITabHyperlink.UseGlobalSettings.htm)


		 Свойство UseGlobalSettings
		 определяет, использовать ли глобальные настройки стиля для гиперссылки.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Apply](ITabHyperlink.Apply.htm)


		 Метод Apply применяет
		 параметры указанной гиперссылки к текущей.


		 ![](../../Sub_Image.gif)
		 [Assign](ITabHyperlink.Assign.htm)


		 Метод Assign заполняет
		 параметры текущей гиперссылки параметрами другой гиперссылки,
		 указанной в качестве входного параметра.


		 ![](../../Sub_Image.gif)
		 [Equal](ITabHyperlink.Equal.htm)


		 Метод Equal осуществляет
		 сравнение гиперссылок.
