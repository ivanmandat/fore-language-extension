# ITabFormatNumericScale.GetMinValue


## Синтаксис


GetMinValue: Double;


## Описание


Метод GetMinValue возвращает
 минимальное значение числовой шкалы.


## Комментарии


Если минимальное значение числовой шкалы не задано, то метод вернет
 значение «-10».


Для определения максимального значения числовой шкалы используйте свойство
 [ITabFormatNumericScale.GetMaxValue](ITabFormatNumericScale.GetMaxValue.htm).
