# ITabTableStyle


## Описание


Интерфейс ITabTableStyle содержит
 свойства и методы для работы со стилем оформления таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ColumnFirstStripe](ITabTableStyle.ColumnFirstStripe.htm)
		 Свойство ColumnFirstStripe
		 возвращает параметры стиля оформления первой полосы столбцов в
		 таблице.


		 ![](../../Property_Image.gif)
		 [ColumnSecondStripe](ITabTableStyle.ColumnSecondStripe.htm)
		 Свойство ColumnSecondStripe
		 возвращает параметры стиля оформления второй полосы столбцов в
		 таблице.


		 ![](../../Property_Image.gif)
		 [Corner](ITabTableStyle.Corner.htm)
		 Свойство Corner возвращает
		 параметры стиля оформления угла таблицы.


		 ![](../../Property_Image.gif)
		 [Default_](ITabTableStyle.Default_.htm)
		 Свойство Default_ возвращает
		 параметры стиля оформления всей таблицы.


		 ![](../../Property_Image.gif)
		 [FooterColumnStripe](ITabTableStyle.FooterColumnStripe.htm)
		 Свойство FooterColumnStripe
		 возвращает параметры стиля оформления последней полосы строк в
		 таблице.


		 ![](../../Property_Image.gif)
		 [FooterRowStripe](ITabTableStyle.FooterRowStripe.htm)
		 Свойство FooterRowStripe
		 возвращает параметры стиля оформления последней полосы столбцов
		 в таблице.


		 ![](../../Property_Image.gif)
		 [HeaderColumnStripe](ITabTableStyle.HeaderColumnStripe.htm)
		 Свойство HeaderColumnStripe
		 возвращает параметры стиля оформления заголовков столбцов в таблице.


		 ![](../../Property_Image.gif)
		 [HeaderRowStripe](ITabTableStyle.HeaderRowStripe.htm)
		 Свойство HeaderRowStripe
		 возвращает параметры стиля оформления заголовков строк в таблице.


		 ![](../../Property_Image.gif)
		 [IsEmpty](ITabTableStyle.IsEmpty.htm)
		 Свойство IsEmpty определяет,
		 задан ли стиль оформления таблицы.


		 ![](../../Property_Image.gif)
		 [Name](ITabTableStyle.Name.htm)
		 Свойство Name определяет
		 наименование стиля оформления.


		 ![](../../Property_Image.gif)
		 [PredefinedStyle](ITabTableStyle.PredefinedStyle.htm)
		 Свойство PredefinedStyle
		 возвращает значение [чередующегося
		 стиля таблицы](UiReport.chm::/desktop/Table/Work/UiReport_Table_WorkStyle.htm) , используемого на листе отчета.


		 ![](../../Property_Image.gif)
		 [Range](ITabTableStyle.Range.htm)
		 Свойство Range определяет
		 диапазон ячеек, для которых создан стиль оформления.


		 ![](../../Property_Image.gif)
		 [RowFirstStripe](ITabTableStyle.RowFirstStripe.htm)
		 Свойство RowFirstStripe
		 возвращает параметры стиля оформления первой полосы строк в таблице.


		 ![](../../Property_Image.gif)
		 [RowSecondStripe](ITabTableStyle.RowSecondStripe.htm)
		 Свойство RowSecondStripe
		 возвращает параметры стиля оформления второй полосы строк в таблице.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AssignPredefined](ITabTableStyle.AssignPredefined.htm)
		 Метод AssignPredefined
		 осуществляет установку параметров стиля оформления таблицы в соответствии
		 с параметрами указанного стиля.
