# ITabUserInteractiveSelectionChangeEventArgs.Cancel


## Синтаксис


Cancel: Boolean;


## Описание


Свойство Cancel определяет признак
 допустимости изменения интерактивного диапазона.


## Комментарии


Если свойству установить значение True,
 то производимое изменение интерактивного диапазона будет запрещено. По
 умолчанию свойству установлено значение False,
 при этом изменение разрешено.
