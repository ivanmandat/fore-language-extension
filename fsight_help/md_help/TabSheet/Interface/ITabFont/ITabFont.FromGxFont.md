# ITabFont.FromGxFont


## Синтаксис


FromGxFont(Value: [IGxFont](ModDrawing.chm::/Interface/IGxFont/IGxFont.htm));


## Параметры


Value — шрифт,
 из которого нужно получить текущий шрифт.


## Описание


Метод FromGxFont возвращает
 шрифт, который получается из шрифта (IGxFont), указанного в качестве входного
 параметра Value.
