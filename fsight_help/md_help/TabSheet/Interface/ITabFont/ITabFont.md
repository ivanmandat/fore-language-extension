# ITabFont


Сборка: Tab;


## Описание


Интерфейс ITabFont содержит
 свойства и методы для настройки параметров шрифта элементов таблицы.


## Иерархия наследования


           ITabFont


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Angle](ITabFont.Angle.htm)
		 Свойство Angle определяет
		 угол поворота текста.


		 ![](../../Property_Image.gif)
		 [Bold](ITabFont.Bold.htm)
		 Свойство Bold определяет,
		 будет ли использоваться жирный шрифт.


		 ![](../../Property_Image.gif)
		 [Charset](ITabFont.Charset.htm)
		 Свойство Charset определяет
		 кодировку шрифта, используемую при отображении текста.


		 ![](../../Property_Image.gif)
		 [Color](ITabFont.Color.htm)
		 Свойство Color определяет
		 цвет шрифта.


		 ![](../../Property_Image.gif)
		 [Italic](ITabFont.Italic.htm)
		 Свойство Italic определяет
		 курсив шрифта.


		 ![](../../Property_Image.gif)
		 [Name](ITabFont.Name.htm)
		 Свойство Name определяет
		 наименование шрифта.


		 ![](../../Property_Image.gif)
		 [Size](ITabFont.Size.htm)
		 Свойство Size определяет
		 размер шрифта в пунктах.


		 ![](../../Property_Image.gif)
		 [Strikeout](ITabFont.Strikeout.htm)
		 Свойство Strikeout
		 определяет, будет ли шрифт зачеркнутым.


		 ![](../../Property_Image.gif)
		 [Underline](ITabFont.Underline.htm)
		 Свойство Underline
		 определяет, будет ли шрифт подчеркнутый.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Apply](ITabFont.Apply.htm)
		 Метод Apply устанавливает
		 параметры текущего шрифта в соответствии с параметрами указанного
		 шрифта.


		 ![](../../Sub_Image.gif)
		 [AsGxFont](ITabFont.AsGxFont.htm)
		 Метод AsGxFont возвращает
		 шрифт (IGxFont).


		 ![](../../Sub_Image.gif)
		 [Assign](ITabFont.Assign.htm)
		 Метод Assign устанавливает
		 параметры текущего шрифта в соответствии с параметрами указанного
		 шрифта, со сбросом неопределенных параметров.


		 ![](../../Sub_Image.gif)
		 [Equal](ITabFont.Equal.htm)
		 Метод Equal осуществляет
		 сравнение шрифтов.


		 ![](../../Sub_Image.gif)
		 [FromGxFont](ITabFont.FromGxFont.htm)
		 Метод FromGxFont возвращает
		 шрифт, который получается из шрифта (IGxFont), указанного в качестве
		 входного параметра.
