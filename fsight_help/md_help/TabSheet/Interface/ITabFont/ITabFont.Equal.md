# ITabFont.Equal


## Синтаксис


Equal(Value: [ITabFont](ITabFont.htm)):
 Boolean;


## Параметры


Value — шрифт,
 параметры которого необходимо сравнить с параметрами текущего шрифта.


## Описание


Метод Equal осуществляет сравнение
 параметров текущего шрифта с параметрами шрифта, указанного в качестве
 входного параметра Value.
