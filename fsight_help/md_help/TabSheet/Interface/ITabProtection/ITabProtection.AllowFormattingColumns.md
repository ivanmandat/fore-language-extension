# ITabProtection.AllowFormattingColumns


## Синтаксис


AllowFormattingColumns: Boolean;


## Описание


Свойство AllowFormattingColumns
 определяет признак разрешения форматирования столбцов таблицы.


## Комментарии


Если свойству установлено значение True,
 то будет возможность форматирования столбцов таблицы при включенной
 защите таблицы.


Если свойству установлено значение False,
 то в зависимости от значения свойства [UserInterfaceOnly](ITabProtection.UserInterfaceOnly.htm)
 возможность форматирования столбцов будет только на уровне ядра, либо
 данная возможность будет заблокирована полностью.


По умолчанию свойству установлено значение True.
