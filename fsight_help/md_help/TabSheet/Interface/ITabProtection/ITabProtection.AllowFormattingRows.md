# ITabProtection.AllowFormattingRows


## Синтаксис


AllowFormattingRows: Boolean;


## Описание


Свойство AllowFormattingRows
 определяет признак разрешения форматирования строк таблицы.


## Комментарии


Если свойству установлено значение True,
 то будет возможность форматирования строк при включенной защите таблицы.


Если свойству установлено значение False,
 то в зависимости от значения свойства [UserInterfaceOnly](ITabProtection.UserInterfaceOnly.htm)
 возможность форматирования строк будет только на уровне ядра, либо данная
 возможность будет заблокирована полностью.


По умолчанию свойству установлено значение True.
