# ITabProtection.Active


## Синтаксис


Active: Boolean;


## Описание


Свойство Active возвращает признак
 установки защиты таблицы.


## Комментарии


Свойство возвращает значение True,
 если была включена защита таблицы и False,
 если защита выключена.


Для включения защиты используйте метод [Activate](ITabProtection.Activate.htm),
 для отключения - [Deactivate](ITabProtection.Deactivate.htm).
