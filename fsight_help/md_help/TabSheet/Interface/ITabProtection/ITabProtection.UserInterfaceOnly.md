# ITabProtection.UserInterfaceOnly


## Синтаксис


UserInterfaceOnly: Boolean;


## Описание


Свойство UserInterfaceOnly определяет
 признак установки защиты на уровне пользовательского интерфейса.


## Комментарии


Если свойству установлено значение True,
 то защита таблицы будет работать на уровне пользовательского интерфейса
 (в зависимости от установленных настроек будет запрещено удаление строк,
 столбцов, редактирование объектов и т.д. через соответствующие команды
 меню, либо кнопки на панели управления). При этом будет возможно произвести
 соответствующие действия с использованием свойств и методов в ядре.


Если свойству установить значение False,
 то все запрещенные действия будет невозможно произвести как в пользовательском
 интерфейсе, так и через свойства и методы в ядре. При нарушении какого-либо
 параметра защиты будет генерироваться событие [TabSheetBox.OnProtectionFail](../../Class/TabSheetBox/TabSheetBox.OnProtectionFail.htm).


По умолчанию свойству установлено значение True.
