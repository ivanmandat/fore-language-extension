# ITabProtection.Deactivate


## Синтаксис


Deactivate(Password: String);


## Параметры


Password — пароль
 для снятия защиты с таблицы.


## Описание


Метод Deactivate снимает защиту
 таблицы с паролем, который передается в качестве входного параметра Password.
