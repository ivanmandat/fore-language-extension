# ITabProtection.AllowEditingObjects


## Синтаксис


AllowEditingObjects: Boolean;


## Описание


Свойство AllowEditingObjects
 определяет признак разрешения редактирования объектов таблицы.


## Комментарии


Если свойству установлено значение True,
 то будет возможность редактирования объектов при включенной защите
 таблицы. Редактирование включает в себя изменение параметров каких-либо
 объектов и возможность активировать объект.


Если свойству установлено значение False,
 то в зависимости от значения свойства [UserInterfaceOnly](ITabProtection.UserInterfaceOnly.htm)
 возможность редактирования объектов будет только на уровне ядра, либо
 данная возможность будет заблокирована полностью.


По умолчанию свойству установлено значение True.
