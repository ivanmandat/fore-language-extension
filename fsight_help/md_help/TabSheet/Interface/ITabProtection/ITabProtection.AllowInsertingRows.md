# ITabProtection.AllowInsertingRows


## Синтаксис


AllowInsertingRows: Boolean;


## Описание


Свойство AllowInsertingRows
 определяет признак разрешения вставки строк в таблицу.


## Комментарии


Если свойству установлено значение True,
 то будет возможность удаления объектов при включенной защите таблицы.


Если свойству установлено значение False,
 то в зависимости от значения свойства [UserInterfaceOnly](ITabProtection.UserInterfaceOnly.htm)
 возможность удаления объектов будет только на уровне ядра, либо данная
 возможность будет заблокирована полностью.


По умолчанию свойству установлено значение True.
