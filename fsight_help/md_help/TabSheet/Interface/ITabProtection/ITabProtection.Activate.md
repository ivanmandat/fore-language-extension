# ITabProtection.Activate


## Синтаксис


Activate(Password: String);


## Параметры


Password — пароль
 для установки защиты таблицы.


## Описание


Метод Activate включает защиту
 таблицы с паролем, который передается в качестве входного параметра Password.


## Комментарии


После активации защиты пользователю будут доступны те функции, для которых
 в соответствующих свойствах Allow*
 установлено значение True. Функции,
 для которых в свойствах Allow*
 установлено значение False, будут
 доступны только через ядро, либо недоступны полностью. Возможность произвести
 какие-либо запрещенные действия с использованием свойств и методов в ядре
 при включенной защите определяется свойством [UserInterfaceOnly](ITabProtection.UserInterfaceOnly.htm).
