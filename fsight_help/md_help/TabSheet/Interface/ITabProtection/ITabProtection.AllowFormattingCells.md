# ITabProtection.AllowFormattingCells


## Синтаксис


AllowFormattingCells: Boolean;


## Описание


Свойство AllowFormattingCells
 определяет признак разрешения форматирования ячеек таблицы.


## Комментарии


Если свойству установлено значение True,
 то будет возможность форматирования ячеек таблицы при включенной
 защите таблицы.


Если свойству установлено значение False,
 то в зависимости от значения свойства [UserInterfaceOnly](ITabProtection.UserInterfaceOnly.htm)
 возможность форматирования ячеек будет только на уровне ядра, либо данная
 возможность будет заблокирована полностью.


По умолчанию свойству установлено значение True.
