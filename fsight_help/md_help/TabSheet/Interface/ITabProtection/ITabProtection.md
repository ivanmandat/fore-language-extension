# ITabProtection


## Описание


Интерфейс ITabProtection содержит
 свойства и методы параметров защиты таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Active](ITabProtection.Active.htm)
		 Свойство Active возвращает
		 признак установки защиты таблицы.


		 ![](../../Property_Image.gif)
		 [AllowDeletingColumns](ITabProtection.AllowDeletingColumns.htm)
		 Свойство AllowDeletingColumns
		 определяет признак разрешения удаления столбцов из таблицы.


		 ![](../../Property_Image.gif)
		 [AllowDeletingObjects](ITabProtection.AllowDeletingObjects.htm)
		 Свойство AllowDeletingObjects
		 определяет признак разрешения удаления объектов из таблицы.


		 ![](../../Property_Image.gif)
		 [AllowDeletingRows](ITabProtection.AllowDeletingRows.htm)
		 Свойство AllowDeletingRows
		 определяет признак разрешения удаления строк из таблицы.


		 ![](../../Property_Image.gif)
		 [AllowEditingObjects](ITabProtection.AllowEditingObjects.htm)
		 Свойство AllowEditingObjects
		 определяет признак разрешения редактирования объектов таблицы.


		 ![](../../Property_Image.gif)
		 [AllowFiltering](ITabProtection.AllowFiltering.htm)
		 Свойство AllowFiltering
		 определяет признак разрешения использования автофильтра таблицы.


		 ![](../../Property_Image.gif)
		 [AllowFormattingCells](ITabProtection.AllowFormattingCells.htm)
		 Свойство AllowFormattingCells
		 определяет признак разрешения форматирования ячеек таблицы.


		 ![](../../Property_Image.gif)
		 [AllowFormattingColumns](ITabProtection.AllowFormattingColumns.htm)
		 Свойство AllowFormattingColumns
		 определяет признак разрешения форматирования столбцов таблицы.


		 ![](../../Property_Image.gif)
		 [AllowFormattingRows](ITabProtection.AllowFormattingRows.htm)
		 Свойство AllowFormattingRows
		 определяет признак разрешения форматирования строк таблицы.


		 ![](../../Property_Image.gif)
		 [AllowInsertingColumns](ITabProtection.AllowInsertingColumns.htm)
		 Свойство AllowInsertingColumns
		 определяет признак разрешения вставки столбцов в таблицу.


		 ![](../../Property_Image.gif)
		 [AllowInsertingObjects](ITabProtection.AllowInsertingObjects.htm)
		 Свойство AllowInsertingObjects
		 определяет признак разрешения вставки объектов в таблицу.


		 ![](../../Property_Image.gif)
		 [AllowInsertingRows](ITabProtection.AllowInsertingRows.htm)
		 Свойство AllowInsertingRows
		 определяет признак разрешения вставки строк в таблицу.


		 ![](../../Property_Image.gif)
		 [AllowMovingObjects](ITabProtection.AllowMovingObjects.htm)
		 Свойство AllowMovingObjects
		 определяет признак разрешения перемещения объектов таблицы.


		 ![](../../Property_Image.gif)
		 [UserInterfaceOnly](ITabProtection.UserInterfaceOnly.htm)
		 Свойство UserInterfaceOnly
		 определяет признак установки защиты пользовательского интерфейса.

		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Activate](ITabProtection.Activate.htm)

		 ![](../../Sub_Image.gif)

		 Метод Deactivate снимает
