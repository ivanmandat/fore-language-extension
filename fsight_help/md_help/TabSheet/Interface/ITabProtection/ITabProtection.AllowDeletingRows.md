# ITabProtection.AllowDeletingRows


## Синтаксис


AllowDeletingRows: Boolean;


## Описание


Свойство AllowDeletingRows определяет
 признак разрешения удаления строк из таблицы.


## Комментарии


Если свойству установлено значение True,
 то будет возможность удаления строк при включенной защите таблицы.


Если свойству установлено значение False,
 то в зависимости от значения свойства [UserInterfaceOnly](ITabProtection.UserInterfaceOnly.htm)
 возможность удаления строк будет только на уровне ядра, либо данная возможность
 будет заблокирована полностью.


По умолчанию свойству установлено значение True.
