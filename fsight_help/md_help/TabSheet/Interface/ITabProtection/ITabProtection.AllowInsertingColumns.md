# ITabProtection.AllowInsertingColumns


## Синтаксис


AllowInsertingColumns: Boolean;


## Описание


Свойство AllowInsertingColumns
 определяет признак разрешения вставки столбцов в таблицу.


## Комментарии


Если свойству установлено значение True,
 то будет возможность вставки столбцов при включенной защите таблицы.


Если свойству установлено значение False,
 то в зависимости от значения свойства [UserInterfaceOnly](ITabProtection.UserInterfaceOnly.htm)
 возможность вставки столбцов будет только на уровне ядра, либо данная
 возможность будет заблокирована полностью.


По умолчанию свойству установлено значение True.
