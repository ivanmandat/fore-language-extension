# ITabProtection.AllowFiltering


## Синтаксис


AllowFiltering: Boolean;


## Описание


Свойство AllowFiltering определяет
 признак разрешения использования автофильтра таблицы.


## Комментарии


Если свойству установлено значение True,
 то будет возможность использования автофильтра в таблице при включенной
 защите таблицы.


Если свойству установлено значение False,
 то в зависимости от значения свойства [UserInterfaceOnly](ITabProtection.UserInterfaceOnly.htm)
 возможность использования автофильтра будет только на уровне ядра, либо
 данная возможность будет заблокирована полностью.


По умолчанию свойству установлено значение True.
