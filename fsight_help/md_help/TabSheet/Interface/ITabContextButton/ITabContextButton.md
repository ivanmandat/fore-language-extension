# ITabContextButton


Сборка: Tab;


## Описание


Интерфейс ITabContextButton
 используется для работы с контекстной кнопкой выделенной области.


## Иерархия наследования


           ITabContextButton


## Комментарии


Для получения объекта данного типа используйте свойство [ITabView.ContextButton](../ITabView/ITabView.ContextButton.htm).
