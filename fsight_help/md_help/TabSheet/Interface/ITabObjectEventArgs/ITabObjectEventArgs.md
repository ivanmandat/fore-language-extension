# ITabObjectEventArgs


## Описание


Интерфейс ITabObjectEventArgs
 содержит свойства аргумента события [TabSheetBox.OnObjectActivate](../../Class/TabSheetBox/TabSheetBox.OnObjectActivate.htm),
 происходящего при активации/деактивации объекта таблицы.

		 тип действия, совершенного над объектом таблицы.


		 ![](../../Property_Image.gif)
		 [Object](ITabObjectEventArgs.Object.htm)
		 Свойство Object возвращает
		 объект, над которым произведено действие.
