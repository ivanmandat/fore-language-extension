# ITabView.DisplayFootnotesToolTips


## Синтаксис


DisplayFootnotesToolTips: Boolean;


## Описание


Свойство DisplayFootnotesToolTips
 определяет, будет ли отображаться всплывающая подсказка с текстом сноски
 при наведении курсора мыши на номер сноски в области ячейки.


## Комментарии


Допустимые значения:


	- True. Отображать всплывающую
	 подсказку.


	- False. Не отображать
	 всплывающую подсказку.
