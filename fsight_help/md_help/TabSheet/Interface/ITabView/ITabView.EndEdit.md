# ITabView.EndEdit


## Синтаксис


EndEdit(SaveValue: Boolean);


## Параметры


SaveValue. Параметр, определяющий,
 будет ли сохранено введенное значение при выходе из режима редактирования.
 По умолчанию установлено значение True.


## Описание


Метод EndEdit осуществляет
 выход из режима редактирования ячейки.
