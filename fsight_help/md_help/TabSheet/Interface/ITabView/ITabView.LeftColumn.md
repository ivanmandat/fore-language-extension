# ITabView.LeftColumn


## Синтаксис


LeftColumn: Integer;


## Описание


Свойство LeftColumn определяет
 индекс самого левого столбца, находящегося на границе видимого диапазона
 таблицы.


## Комментарии


Если в таблице имеется фиксированная область и прокрутка таблицы не
 осуществлялась, то свойство возвращает индекс столбца, отображаемого на
 границе с фиксированной областью.
