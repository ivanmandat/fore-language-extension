# ITabView.RightColumn


## Синтаксис


RightColumn: Integer;


## Описание


Свойство RightColumn возвращает
 индекс самого правого столбца, находящегося на границе видимого диапазона
 таблицы.


## Комментарии


При наличии фиксированной области - возвращает индекс столбца, расположенного
 на правой границе видимого диапазона таблицы.
