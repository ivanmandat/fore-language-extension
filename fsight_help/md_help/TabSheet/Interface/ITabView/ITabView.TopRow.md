# ITabView.TopRow


## Синтаксис


TopRow: Integer;


## Описание


Свойство TopRow определяет индекс
 самой верхней строки, находящейся на границе видимости таблицы.


## Комментарии


Если в таблице имеется фиксированная область и прокрутка таблицы не
 осуществлялась, то свойство возвращает индекс строки, отображаемой на
 границе с фиксированной областью.
