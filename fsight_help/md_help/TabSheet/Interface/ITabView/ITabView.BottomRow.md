# ITabView.BottomRow


## Синтаксис


BottomRow: Integer;


## Описание


Свойство BottomRow возвращает
 индекс самой нижней строки, находящейся в видимом диапазоне таблицы.


## Комментарии


Если строка видна не полностью, то она не будет учтена.
