# ITabView.DisplayPasteOptions


## Синтаксис


DisplayPasteOptions: Boolean;


## Описание


Свойство DisplayPasteOptions
 определяет необходимость отображения параметров вставки данных из буфера
 обмена (с форматированием либо без него).


## Комментарии


Свойство задается для каждого листа отчета. По умолчанию DisplayPasteOptions
 имеет значение True (отображение
 параметров вставки включено).
