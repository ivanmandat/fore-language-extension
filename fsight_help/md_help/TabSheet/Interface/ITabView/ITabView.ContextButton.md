# ITabView.ContextButton


## Синтаксис


ContextButton: [ITabContextButton](../ITabContextButton/ITabContextButton.htm);


## Описание


Свойство ContextButton возвращает
 объект для работы с контекстной кнопкой выделенной области.


## Комментарии


Для определения, будет ли отображаться контекстная кнопка выделенной
 области, используйте [ITabContextButton.Visible](../ITabContextButton/ITabContextButton.Visible.htm).
