# ITabView.UserIntearactiveSelections


## Синтаксис


UserIntearactiveSelections: [ITabUserInteractiveSelections](../ITabUserInteractiveSelections/ITabUserInteractiveSelections.htm);


## Описание


Свойство UserIntearactiveSelections
 возвращает коллекцию интерактивных диапазонов ячеек.


## Комментарии


Интерактивные диапазоны используются для визуального выделения ячеек
 таблицы. Каждый диапазон имеет собственное оформление границы, также можно
 ограничить возможность изменения интерактивного диапазона.


## Пример


Пример использования приведён в описании метода [ITabUserInteractiveSelections.Add](../ITabUserInteractiveSelections/ITabUserInteractiveSelections.Add.htm).
