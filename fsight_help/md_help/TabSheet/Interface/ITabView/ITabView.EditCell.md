# ITabView.EditCell


## Синтаксис


EditCell(Row: Integer; Column: Integer): Boolean;


## Параметры


Row. Индекс строки. Допустимое значение параметра находится в диапазоне [0, [RowsCount](../ITabSheet/ITabSheet.RowsCount.htm));


Column. Индекс столбца. Допустимое значение параметра находится в диапазоне [0, [ColumnsCount](../ITabSheet/ITabSheet.ColumnsCount.htm)).


## Описание


Метод EditCell переводит в режим редактирования ячейку, адрес которой передается в качестве входных параметров Row и Column.
