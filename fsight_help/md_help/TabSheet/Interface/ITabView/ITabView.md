# ITabView


Сборка: Tab;


## Описание


Интерфейс ITabView содержит
 свойства и методы для настройки внешнего вида электронной таблицы.


## Иерархия наследования


           ITabView


## Комментарии


Доступ к внешнему виду электронной таблицы предоставляет свойство [ITabSheet.View](../ITabSheet/ITabSheet.View.htm).

## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ConvertPtToView](ITabView.ConvertPtToView.htm)


		 Метод ConvertPtToView
		 осуществляет преобразование точки в экранных координатах в точку
		 относительно левого верхнего угла видимой области электронной
		 таблицы.


		 ![](../../Sub_Image.gif)
		 [EditCell](ITabView.EditCell.htm)


		 Метод EditCell переводит
		 ячейку в режим редактирования.


		 ![](../../Sub_Image.gif)
		 [EmulateCellClick](ITabView.EmulateCellClick.htm)


		 Метод EmulateCellClick
		 осуществляет эмуляцию щелчка по указанной ячейке.


		 ![](../../Sub_Image.gif)
		 [EmulateCellPictureClick](ITabView.EmulateCellPictureClick.htm)


		 Метод EmulateCellPictureClick
		 осуществляет эмуляцию щелчка по изображению, расположенному в
		 указанной ячейке.


		 ![](../../Sub_Image.gif)
		 [EmulateHyperlinkClick](ITabView.EmulateHyperlinkClick.htm)


		 Метод EmulateHyperlinkClick
		 осуществляет эмуляцию щелчка по гиперссылке в указанной ячейке.


		 ![](../../Sub_Image.gif)
		 [EndEdit](ITabView.EndEdit.htm)


		 Метод EndEdit осуществляет
		 выход из режима редактирования ячейки.


		 ![](../../Sub_Image.gif)
		 [EnsureRowVisible](ITabView.EnsureRowVisible.htm)


		 Метод EnsureRowVisible
		 прокручивает таблицу таким образом, чтобы заданная строка попала
		 в видимый диапазон.


		 ![](../../Sub_Image.gif)
		 [HitTest](ITabView.HitTest.htm)


		 Метод HitTest возвращает
		 область таблицы, содержащую точку, координаты которой передаются
		 в качестве входных параметров.


		 ![](../../Sub_Image.gif)
		 [HitTestCell](ITabView.HitTestCell.htm)


		 Метод HitTestCell вычисляет
		 индексы строки и столбца для ячейки таблицы, содержащей точку,
		 координаты которой передаются в качестве входных параметров.


		 ![](../../Sub_Image.gif)
		 [HitTestObject](ITabView.HitTestObject.htm)


		 Метод HitTestObject
		 возвращает объект таблицы, содержащий точку, координаты которой
		 передаются в качестве входных параметров.


		 ![](../../Sub_Image.gif)
		 [HitTestRegion](ITabView.HitTestRegion.htm)


		 Метод HitTestRegion
		 возвращает регион таблицы, содержащий точку, координаты которой
		 передаются в качестве входных параметров.


		 ![](../../Sub_Image.gif)
		 [ShowFormatDialogEx](ITabView.ShowFormatDialogEx.htm)


		 Метод ShowFormatDialogEx
		 отображает окно формата для текущего выделения.


		 ![](../../Sub_Image.gif)
		 [ShowFormulaWizard](ITabView.ShowFormulaWizard.htm)


		 Метод ShowFormulaWizard
		 осуществляет вызов окна мастера формул для [TabRange](../ITabRange/ITabRange.htm).
