# ITabView.ActiveRow


## Синтаксис


ActiveRow: Integer;


## Описание


Свойство ActiveRow определяет номер последней строки, в которой был произведен щелчок по гиперссылке.


## Комментарии


Если ни одна из гиперссылок таблицы не была активирована, то свойство возвращает значение «-1».
