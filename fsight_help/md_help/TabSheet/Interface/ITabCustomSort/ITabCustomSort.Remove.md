# ITabCustomSort.RemoveAt


## Синтаксис


RemoveAt(Index: Integer): Boolean;


## Параметры


Index - Параметр определяет индекс области сортировки, которую необходимо удалить.


## Описание


Метод RemoveAt удаляет область сортировки, индекс которой передается в качестве входного параметра Index, и возвращает True в случае удачного удаления.
