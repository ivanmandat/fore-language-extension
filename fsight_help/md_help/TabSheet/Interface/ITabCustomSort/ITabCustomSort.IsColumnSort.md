# ITabCustomSort.IsColumnSort


## Синтаксис


IsColumnSort: Boolean;


## Описание


Свойство IsColumnSort возвращает True, если в таблице настроена сортировка по столбцам.


## Комментарий


Механизм сортировки работает следующим образом: если сортировка происходит по строкам – это означает, что перемещаются значения в строках данного столбца. Если по столбцам, то происходит перестановка значений в столбцах сортируемой строки.
