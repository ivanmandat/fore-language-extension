# ITabCustomSort.Item


## Синтаксис


Item(Index: Integer): [ITabCustomSortItem](../ITabCustomSortItem/ITabCustomSortItem.htm);


## Параметры


Index - Параметр определяет индекс области сортировки, расположенной в таблице данных.


## Описание


Свойство Item возвращает объект, содержащий в себе область сортировки, расположенную в таблице данных.
