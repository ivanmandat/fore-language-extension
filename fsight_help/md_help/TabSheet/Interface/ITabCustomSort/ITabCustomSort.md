# ITabCustomSort


Сборка: Tab;


## Описание


Интерфейс ITabCustomSort содержит
 свойства и методы коллекции областей сортировки строк/столбцов таблицы
 данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CaseSensitive](ITabCustomSort.CaseSensitive.htm)
		 Свойство CaseSensitive
		 определяет, будет ли учитываться регистр символов при сортировке.


		 ![](../../Property_Image.gif)
		 [Count](ITabCustomSort.Count.htm)
		 Свойство Count возвращает
		 количество областей сортировки в коллекции.


		 ![](../../Property_Image.gif)
		 [IsColumnSort](ITabCustomSort.IsColumnSort.htm)
		 Свойство IsColumnSort
		 возвращает True, если
		 в таблице настроена сортировка по столбцам.


		 ![](../../Property_Image.gif)
		 [Item](ITabCustomSort.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий в себе область сортировки, расположенную в
		 таблице данных.


		 ![](../../Property_Image.gif)
		 [Range](ITabCustomSort.Range.htm)
		 Свойство Range определяет
		 область таблицы, в которой будет настроена сортировка.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ITabCustomSort.Add.htm)
		 Метод Add осуществляет
		 добавление новой области сортировки в таблице данных.


		 ![](../../Sub_Image.gif)
		 [Clear](ITabCustomSort.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех областей сортировки таблицы данных.


		 ![](../../Sub_Image.gif)
		 [RemoveAt](ITabCustomSort.Remove.htm)
		 Метод RemoveAt удаляет
		 область сортировки, индекс которой передается в качестве входного
		 параметра Index.


		 ![](../../Sub_Image.gif)
		 [Sort](ITabCustomSort.Sort.htm)
		 Метод Sort осуществляет
		 сортировку заданной области.
