# ITabImages


## Описание


Интерфейс ITabImages содержит
 свойства и методы коллекции картинок таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITabImages.Count.htm)
		 Свойство Count возвращает
		 количество картинок в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ITabImages.Item.htm)
		 Свойство Item возвращает
		 картинку.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ITabImages.Add.htm)
		 Метод Add добавляет
		 картинку.


		 ![](../../Sub_Image.gif)
		 [AddFromImageList](ITabImages.AddFromImageList.htm)
		 Метод AddFromImageList
		 импортирует набор рисунков из компонента ImageList.


		 ![](../../Sub_Image.gif)
		 [Remove](ITabImages.Remove.htm)
		 Метод Remove удаляет
		 указанное изображение из коллекции.


		 ![](../../Sub_Image.gif)
		 [RemoveAll](ITabImages.RemoveAll.htm)
		 Метод RemoveAll очищает
		 всю коллекцию изображений.
