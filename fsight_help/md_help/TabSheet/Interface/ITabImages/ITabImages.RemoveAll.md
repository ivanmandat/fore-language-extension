# ITabImages.RemoveAll


## Синтаксис


RemoveAll([CorrectCellPictureIndexes: Boolean =
 False]);


## Параметры


CorrectCellPictureIndexes.
 Необязательный параметр, указывающий необходимость произвести корректировку
 индексов в свойствах, использующих изображения из данной коллекции.


## Описание


Метод RemoveAll очищает всю
 коллекцию изображений.


## Комментарии


По умолчанию в качестве значения параметра CorrectCellPictureIndexes
 передается значение False, при
 этом корректировка индексов не производится. Во всех свойствах, в которых
 указаны индексы изображений, значения останутся неизменными. Если в качестве
 значения параметра CorrectCellPictureIndexes
 передать значение True, то после
 удаления изображений для всех свойств, которые использовали изображения
 коллекции ITabImages, будут установлены
 значения «-1».
