# ITabImages.Item


## Синтаксис


Item(Index: Integer): [IGxImage](ModDrawing.chm::/Interface/IGxImage/IGxImage.htm);


## Параметры


Index — индекс
 картинки, которую необходимо получить. Индексация начинается с 0.


## Описание


Свойство Item возвращает картинку,
 индекс которой передается в качестве входного параметра Index.
