# ITabImages.Remove


## Синтаксис


Remove(Index: Integer; [CorrectCellPictureIndexes:
 Boolean = False]);


## Параметры


Index.
 Индекс изображения, которое необходимо удалить.


CorrectCellPictureIndexes. Необязательный параметр, указывающий
 необходимость произвести корректировку индексов в свойствах, использующих
 изображения из данной коллекции.


## Описание


Метод Remove удаляет изображение,
 индекс которого передается в качестве входного параметра Index.


## Комментарии


По умолчанию в качестве значения параметра CorrectCellPictureIndexes
 передается значение False, при
 этом корректировка индексов не производится. Во всех свойствах, в которых
 указаны индексы изображений, значения останутся неизменными, но из-за
 уменьшения коллекции указанные индексы будут соответствовать другим изображениям.
 Если в качестве значения параметра CorrectCellPictureIndexes
 передать значение True, то после
 удаления изображения будут скорректированы значения всех свойств, в которых
 указаны индексы изображений. Для свойств, которые использовали изображения
 с индексом Index, будет установлено
 значение «-1».
