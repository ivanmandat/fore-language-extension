# ITabTableStyles


## Описание


Интерфейс ITabTableStyles содержит
 свойства и методы для работы с коллекцией стилей оформления таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITabTableStyles.Count.htm)
		 Свойство Count возвращает
		 количество стилей оформления в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ITabTableStyles.Item.htm)
		 Свойство Item возвращает
		 указанный стиль оформления из коллекции.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ITabTableStyles.Add.htm)
		 Метод Add добавляет
		 стиль в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](ITabTableStyles.Clear.htm)
		 Метод Clear очищает
		 коллекцию стилей таблицы.


		 ![](../../Sub_Image.gif)
		 [FindByName](ITabTableStyles.FindByName.htm)
		 Метод FindByName осуществляет
		 поиск стиля оформления таблицы.


		 ![](../../Sub_Image.gif)
		 [LoadFromXml](ITabTableStyles.LoadFromXml.htm)
		 Метод LoadFromXml
		 загружает стили таблицы из строки в формате XML.


		 ![](../../Sub_Image.gif)
		 [Move](ITabTableStyles.Move.htm)
		 Метод Move осуществляет
		 перемещение стиля в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](ITabTableStyles.Remove.htm)
		 Метод Remove удаляет
		 указанный стиль из коллекции.


		 ![](../../Sub_Image.gif)
		 [SaveToXml](ITabTableStyles.SaveToXml.htm)
		 Метод SaveToXml сохраняет
		 стили оформления таблицы в строку формата XML.
