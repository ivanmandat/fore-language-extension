# ITabTableStyles.FindByName


## Синтаксис


FindByName(Value: String): [ITabTableStyle](../ITabTableStyle/ITabTableStyle.htm);


## Параметры


Value - наименование стиля
 оформления таблицы, по которому осуществляется поиск.


## Описание


Метод FindByName осуществляет
 поиск стиля оформления таблицы по наименованию, и возвращает его параметры
 в случае удачного поиска.
