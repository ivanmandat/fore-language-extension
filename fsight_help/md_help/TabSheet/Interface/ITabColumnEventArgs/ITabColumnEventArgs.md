# ITabColumnEventArgs


## Описание


Интерфейс ITabColumnEventArgs
 содержит свойства аргумента событий, связанных с каким-либо столбцом компонента
 [TabSheetBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/TabSheetBox.htm).


## Комментарии


Данный аргумент используется при обработке следующих событий:


	- [OnColumnClick](../../Class/TabSheetBox/TabSheetBox.OnColumnClick.htm)


	- [OnBeforeColumnResize](../../Class/TabSheetBox/TabSheetBox.OnBeforeColumnResize.htm)
