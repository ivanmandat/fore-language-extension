# ITabPageSettings.CenterVertical


## Синтаксис


CenterVertical: Boolean;


## Описание


Свойство CenterVertical определяет,
 будет ли таблица при печати центрироваться по вертикали.


## Комментарии


Допустимые значения:


	- True. Центрировать по
	 вертикали;


	- False. Не центрировать
	 по вертикали.
