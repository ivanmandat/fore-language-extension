# ITabPageSettings.CenterHorizontal


## Синтаксис


CenterHorizontal: Boolean;


## Описание


Свойство CenterHorizontal определяет,
 будет ли таблица при печати центрироваться по горизонтали.


## Комментарии


Допустимые значения:


	- True. Центрировать по
	 горизонтали;


	- False. Не центрировать
	 по горизонтали.
