# ITabPageSettings.BlackAndWhite


## Синтаксис


BlackAndWhite: Boolean;


## Описание


Свойство BlackAndWhite определяет,
 будет ли печататься оформление таблицы при печати на принтере. Если свойству
 установлено значение True - при
 печати все цветовое оформление сбрасывается, текст и обрамление ячеек
 печатается черным цветом.
