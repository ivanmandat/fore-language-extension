# ITabPageSettings.PrintHyperlinks


## Синтаксис


		PrintHyperlinks: Boolean;


## Описание


Свойство PrintHyperlinks определяет,
 будут ли экспортироваться гиперссылки в формат HTML.


## Комментарии


Допустимые значения:


	- True. Значение
	 по умолчанию. Гиперссылки экспортируются;


	- False. Гиперссылки
	 не экспортируются.


Для определения возможности экспорта гиперссылок в регламентном отчете
 используйте свойство [IPrxReportExporter.ExportHyperlinkOpenURLAction](KeReport.chm::/Interface/IPrxReportExporter/IPrxReportExporter.ExportHyperlinkOpenURLAction.htm).
