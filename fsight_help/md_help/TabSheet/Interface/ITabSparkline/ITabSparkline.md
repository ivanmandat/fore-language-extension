# ITabSparkline


Сборка: Tab;


## Описание


Интерфейс ITabSparkline содержит
 свойства спарклайна таблицы.


## Иерархия наследования


           [ISparkline](ModDrawing.chm::/Interface/ISparkline/ISparkline.htm)


           ITabSparkline


## Комментарии


Параметры отдельного спарклайна можно получить в свойстве [ITabSparklines.Item](../ITabSparklines/ITabSparklines.Item.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DataRange](ITabSparkline.DataRange.htm)
		 Свойство DataRange
		 определяет диапазон ячеек, по данным которых строится график спарклайна.


		 ![](../../Property_Image.gif)
		 [Range](ITabSparkline.Range.htm)
		 Свойство Range определяет
		 ячейку, в которой расположен спарклайн.


## Свойства, унаследованные от [ISparkline](ModDrawing.chm::/Interface/ISparkline/ISparkline.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Color](ModDrawing.chm::/Interface/ISparkline/ISparkline.Color.htm)
		 Свойство Color определяет

		 ![](../../Property_Image.gif)
		 [LineWidth](ModDrawing.chm::/Interface/ISparkline/ISparkline.LineWidth.htm)
		 Свойство LineWidth
		 определяет толщину линии в спарклайне.


		 ![](../../Property_Image.gif)
		 [NullValueBehavior](ModDrawing.chm::/Interface/ISparkline/ISparkline.NullValueBehavior.htm)
		 Свойство NullValueBehavior
		 определяет способ построения спарклайна при наличии пустых данных.
