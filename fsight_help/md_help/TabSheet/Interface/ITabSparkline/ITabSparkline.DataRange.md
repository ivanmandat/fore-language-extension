# ITabSparkline.DataRange


## Синтаксис


DataRange: [ITabRange](../ITabRange/ITabRange.htm);


## Описание


Свойство DataRange определяет
 диапазон ячеек, по данным которых строится график спарклайна.


## Комментарии


Для построения спарклайна указываемый диапазон должен состоять минимум
 из двух ячеек. Значения должны располагаться в одном
 столбце/строке.
