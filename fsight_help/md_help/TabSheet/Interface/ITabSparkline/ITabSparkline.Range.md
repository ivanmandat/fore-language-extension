# ITabSparkline.Range


## Синтаксис


Range: [ITabRange](../ITabRange/ITabRange.htm);


## Описание


Свойство Range определяет ячейку,
 в которой расположен спарклайн.


## Комментарии


В качестве значения свойства указывается диапазон, состоящий из одной
 ячейки таблицы.
