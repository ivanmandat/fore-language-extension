# ITabCellBeforeEditBaseEventArgs.Cancel


## Синтаксис


Cancel: Boolean;


## Описание


Свойство Cancel определяет, будет ли производиться редактирование значения ячейки.


## Комментарии


Если свойству установить значение True, то переход в режим редактирования ячейки будет запрещен.


По умолчанию свойству установлено значение False.
