# ITabSearchResult


## Описание


Интерфейс ITabSearchResult содержит
 свойства и методы результата поиска.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CurrentColumn](ITabSearchResult.CurrentColumn.htm)
		 Свойство CurrentColumn
		 возвращает номер столбца найденной ячейки.


		 ![](../../Property_Image.gif)
		 [CurrentRow](ITabSearchResult.CurrentRow.htm)
		 Свойство CurrentRow
		 возвращает номер строки найденной ячейки.


		 ![](../../Property_Image.gif)
		 [Range](ITabSearchResult.Range.htm)
		 Свойство Range возвращает
		 объект, содержащий ячейку с искомыми данными.
