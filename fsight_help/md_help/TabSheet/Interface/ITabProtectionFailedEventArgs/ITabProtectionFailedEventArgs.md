# ITabProtectionFailedEventArgs


## Описание


Интерфейс ITabCellBeforeChangeEventArgs
 содержит свойства аргумента события [TabSheetBox.OnProtectionFail](../../Class/TabSheetBox/TabSheetBox.OnProtectionFail.htm),
 возникающего при срабатывании защиты таблицы, компонента TabSheetBox.
