# ITabRegion


Сборка: Tab;


## Описание


Интерфейс ITabRegion предназначен
 для настройки параметров региона таблицы.


## Иерархия наследования


ITabRegion


## Комментарии


Стандартными регионами являются области данных и области формул. Для
 получения параметров региона используйте свойства и методы интерфейса
 [ITabRegions](../ITabRegions/ITabRegions.htm).
