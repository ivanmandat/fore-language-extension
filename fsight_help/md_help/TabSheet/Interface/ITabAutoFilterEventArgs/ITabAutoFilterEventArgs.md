# ITabAutoFilterEventArgs


## Описание


Интерфейс ITabAutoFilterEventArgs
 предназначен для работы с параметрами события [TabSheetBox.OnAutoFilter](../../Class/TabSheetBox/TabSheetBox.OnAutoFilter.htm)
 (установка автофильтра таблицы) компонента TabSheetBox.
