# ITabSheetBox.DisplayNumericColumnNames


## Синтаксис


DisplayNumericColumnNames: Boolean;


## Описание


Свойство DisplayNumericColumnNames определяет признак отображения числовых наименований столбцов в компоненте.


## Комментарии


Если свойству установлено значение True, то в компоненте будут отображаться числовые наименования столбцов, и при визуальном конструировании формул новые ссылки на ячейки будут формироваться в стиле R1C1. По умолчанию свойству установлено значение False, при этом используется буквенное обозначение столбцов, и новые ссылки на ячейки формируются в стиле A1.


Примечание. Ссылки на строки/столбцы всегда формируются в стиле A1.
