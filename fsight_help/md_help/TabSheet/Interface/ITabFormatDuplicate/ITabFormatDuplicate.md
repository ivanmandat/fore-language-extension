# ITabFormatDuplicate


## Описание


Интерфейс ITabFormatDuplicate
 содержит свойства условного формата, осуществляющего форматирование ячеек
 с уникальными/дублирующимися значениями.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Style](ITabFormatDuplicate.Style.htm)
		 Свойство Style определяет
		 стиль оформления, применяемый для ячеек, удовлетворяющих установленному
		 условию.


		 ![](../../Property_Image.gif)
		 [Type](ITabFormatDuplicate.Type.htm)
		 Свойство Type определяет
		 тип значений, которые необходимо форматировать.
