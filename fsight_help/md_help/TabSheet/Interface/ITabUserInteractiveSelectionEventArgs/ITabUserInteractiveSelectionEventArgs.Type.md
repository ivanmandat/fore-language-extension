# ITabUserInteractiveSelectionEventArgs.Type


## Синтаксис


Type: [TabUserInteractiveSelectionChangeType](../../Enums/TabUserInteractiveSelectionChangeType.htm);


## Описание


Свойство Type возвращает разрешенные
 варианты изменения интерактивного диапазона.


## Комментарии


Свойство возвращает одно значение или комбинацию значений из элементов
 перечисления [TabUserInteractiveSelectionChangeType](../../Enums/TabUserInteractiveSelectionChangeType.htm)
 соответствующих разрешённым действиям с интерактивным диапазоном.
