# ITabCellContentChangeBaseEventArgs.Value


## Синтаксис


Value: Variant;


## Описание


Свойство Value определяет новое значение, которое будет изменено для ячейки.


## Комментарии


В зависимости от значения свойства [Type](ITabCellContentChangeBaseEventArgs.Type.htm) значение данного свойства будет установлено в качестве [текста](../ITabRange/ITabRange.Text.htm), [значения](../ITabRange/ITabRange.Value.htm) либо [формулы](../ITabRange/ITabRange.Formula.htm) изменяемой ячейки.


## Пример


Пример использования данного свойства приведен в описании свойства [Cancel](ITabCellContentChangeBaseEventArgs.Cancel.htm).
