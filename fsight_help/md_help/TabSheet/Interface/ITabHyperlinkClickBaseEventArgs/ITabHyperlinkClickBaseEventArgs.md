# ITabHyperlinkClickBaseEventArgs


Сборка: Tab;


## Описание


Интерфейс ITabHyperlinkClickBaseEventArgs содержит
 базовые свойства аргументов событий, связанных генерируемых для гиперссылки
 в ячейке таблицы.
