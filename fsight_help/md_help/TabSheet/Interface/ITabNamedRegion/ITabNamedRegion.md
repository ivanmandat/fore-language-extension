# ITabNamedRegion


## Описание


Интерфейс ITabNamedRegion управляет
 параметрами именованного диапазона таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Name](ITabNamedRegion.Name.htm)
		 Свойство Name определяет
		 наименование именованного диапазона таблицы.


## Свойства, унаследованные от [ITabRegion](../ITabRegion/ITabRegion.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Adjustable](../ITabRegion/ITabRegion.Adjustable.htm)
		 Свойство Adjustable
		 определяет изменяемость региона таблицы.


		 ![](../../Property_Image.gif)
		 [BackgroundColor](../ITabRegion/ITabRegion.BackgroundColor.htm)
		 Свойство BackgroundColor
		 определяет цвет фона региона таблицы.


		 ![](../../Property_Image.gif)
		 [BorderColor](../ITabRegion/ITabRegion.BorderColor.htm)
		 Свойство BorderColor
		 определяет цвет границы региона таблицы.


		 ![](../../Property_Image.gif)
		 [BorderStyle](../ITabRegion/ITabRegion.BorderStyle.htm)
		 Свойство BorderStyle
		 определяет тип линии границы региона.


		 ![](../../Property_Image.gif)
		 [BorderWeight](../ITabRegion/ITabRegion.BorderWeight.htm)
		 Свойство BorderWeight
		 определяет толщину линии границы региона.


		 ![](../../Property_Image.gif)
		 [Key](../ITabRegion/ITabRegion.Key.htm)
		 Свойство Key возвращает
		 уникальный ключ региона таблицы.


		 ![](../../Property_Image.gif)
		 [Range](../ITabRegion/ITabRegion.Range.htm)
		 Свойство Range определяет
		 диапазон таблицы, в котором располагается регион.


		 ![](../../Property_Image.gif)
		 [Visible](../ITabRegion/ITabRegion.Visible.htm)
		 Свойство Visible определяет
		 видимость региона таблицы.
