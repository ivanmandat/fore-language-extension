# ITabFootnoteCollection.Add


## Синтаксис


Add(Row: Integer; Column: Integer): [ITabFootnote](../ITabFootnote/ITabFootnote.htm);


## Параметры


Row. Индекс строки, в которой
 расположена ячейка.


Column. Индекс столбца, в котором
 расположена ячейка.


## Описание


Метод Add создает для указанной
 ячейки сноску и возвращает её параметры.
