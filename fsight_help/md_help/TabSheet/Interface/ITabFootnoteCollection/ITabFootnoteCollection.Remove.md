# ITabFootnoteCollection.Remove


## Синтаксис


Remove(Value: [ITabFootnote](../ITabFootnote/ITabFootnote.htm)):
 Boolean;


## Параметры


Value. Сноска, которую необходимо
 удалить из коллекции.


## Описание


Метод Remove удаляет указанную
 сноску из коллекции и возвращает признак удачного удаления.


## Комментарии


Метод возвращает значение True,
 если сноска была удалена из коллекции и False,
 если удаление по каким-либо причинам невозможно.
