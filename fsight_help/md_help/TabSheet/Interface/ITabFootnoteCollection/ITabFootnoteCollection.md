# ITabFootnoteCollection


Сборка: Tab;


## Описание


Интерфейс ITabFootnoteCollection
 содержит базовые свойства и методы коллекции сносок.


## Иерархия наследования


ITabFootnoteCollection


## Комментарии


Интерфейс ITabFootnoteCollection
 является базовым для интерфейса [ITabFootnotes](../ITabFootnotes/ITabFootnotes.htm).
 Также коллекцию сносок возвращает свойство [ITabRange.Footnotes](../ITabRange/ITabRange.Footnotes.htm).

		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITabFootnoteCollection.Count.htm)

		 [IndexOf](ITabFootnoteCollection.IndexOf.htm)
