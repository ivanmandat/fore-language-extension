# ITabFootnoteCollection.RemoveAt


## Синтаксис


RemoveAt(Index: Integer): Boolean;


## Параметры


Index. Индекс сноски, которую
 необходимо удалить.


## Описание


## Описание


Метод RemoveAt удаляет сноску
 с указанным индексом из коллекции и возвращает признак удачного удаления.


## Комментарии


Метод возвращает значение True,
 если сноска была удалена из коллекции и False,
 если удаление по каким-либо причинам невозможно.
