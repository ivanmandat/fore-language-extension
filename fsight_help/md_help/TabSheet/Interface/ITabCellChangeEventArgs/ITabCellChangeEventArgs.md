# ITabCellChangeEventArgs


## Описание


Интерфейс ITabCellChangeEventArgs
 содержит свойства аргумента события [TabSheetBox.OnCellChange](../../Class/TabSheetBox/TabSheetBox.OnCellChange.htm),
 происходящего при изменении значения ячейки таблицы компонента TabSheetBox.
