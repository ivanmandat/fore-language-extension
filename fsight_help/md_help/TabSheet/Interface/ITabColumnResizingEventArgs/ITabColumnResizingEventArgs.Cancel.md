# ITabColumnResizingEventArgs.Cancel


## Синтаксис


Cancel: Boolean;


## Описание


Свойство Cancel определяет допустимость изменения ширины столбца.


## Комментарии


Если свойству установить значение True, то изменение ширины столбца будет запрещено, если False - разрешено.
