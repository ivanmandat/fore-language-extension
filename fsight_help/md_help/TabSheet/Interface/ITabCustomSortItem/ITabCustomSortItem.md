# ITabCustomSortItem


Сборка: Tab;


## Описание


Интерфейс ITabCustomSortItem
 содержит свойства для настройки параметров области сортировки строк/столбцов
 таблицы данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Direction](ITabCustomSortItem.Direction.htm)
		 Свойство Direction
		 определяет направление, в котором будет производиться сортировка.


		 ![](../../Property_Image.gif)
		 [Index](ITabCustomSortItem.Index.htm)
		 Свойство Index определяет
		 относительный индекс строки/столбца диапазона, выбранного для
		 настройки сортировки.


		 ![](../../Property_Image.gif)
		 [Type](ITabCustomSortItem.Type.htm)
		 Свойство Type определяет
		 тип сортировки, применяемой к строкам/столбцам таблицы данных.
