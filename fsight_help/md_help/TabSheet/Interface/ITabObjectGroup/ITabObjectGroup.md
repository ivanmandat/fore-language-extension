# ITabObjectGroup


## Описание


Интерфейс ITabObjectGroup содержит
 свойства и методы для работы с группами объектов, расположенными на листе
 таблицы.


## Комментарии


Для получения интерфейса ITabObjectGroup
 используйте метод [ITabObjects.GroupSelected](../ITabObjects/ITabObjects.GroupSelected.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ChildItem](ITabObjectGroup.ChildItem.htm)


		 Свойство ChildItem
		 возвращает элемент группы, индекс которого передается в качестве
		 входного параметра Index.


		 ![](../../Property_Image.gif)
		 [ChildItemsCount](ITabObjectGroup.ChildItemsCount.htm)


		 Свойство ChildItemsCount
		 возвращает количество элементов группы.
