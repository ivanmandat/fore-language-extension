# ITabFootnote.NumberText


## Синтаксис


NumberText: String;


## Описание


Свойство NumberText возвращает
 номер сноски в виде текста.


## Комментарии


Свойство возвращает [номер сноски](ITabFootnote.Number.htm)
 с учетом [стиля
 цифр](../ITabFootnotes/ITabFootnotes.NumberStyle.htm), который используется для нумерации сносок.

Пример использования приведен в описании свойства [ITabFootnotes.NumberStyle](../ITabFootnotes/ITabFootnotes.NumberStyle.htm).
