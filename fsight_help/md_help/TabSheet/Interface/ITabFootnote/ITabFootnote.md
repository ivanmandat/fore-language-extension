# ITabFootnote


Сборка: Tab;


## Описание


Интерфейс ITabFootnote содержит
 свойства для настройки параметров сноски.


## Иерархия наследования


ITabFootnote


## Комментарии


Параметры сноски можно получить, используя различные свойства и методы
 коллекции [ITabFootnotes](../ITabFootnotes/ITabFootnotes.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Column](ITabFootnote.Column.htm)
		 Свойство Column определяет
		 индекс столбца ячейки, для которой создана сноска.


		 ![](../../Property_Image.gif)
		 [CustomMark](ITabFootnote.CustomMark.htm)
		 Свойство CustomMark
		 определяет пользовательскую метку, которая будет использоваться
		 для обозначения сноски.


		 ![](../../Property_Image.gif)
		 [CustomMarkFontName](ITabFootnote.CustomMarkFontName.htm)
		 Свойство CustomMarkFontName
		 определяет наименование шрифта, который будет использоваться для
		 отрисовки метки.


		 ![](../../Property_Image.gif)
		 [FormattedText](ITabFootnote.FormattedText.htm)
		 Свойство FormattedText
		 возвращает текст сноски в формате RTF.


		 ![](../../Property_Image.gif)
		 [Number](ITabFootnote.Number.htm)
		 Свойство Number возвращает
		 порядковый номер сноски.


		 ![](../../Property_Image.gif)
		 [NumberText](ITabFootnote.NumberText.htm)
		 Свойство NumberText
		 возвращает номер сноски в виде текста.


		 ![](../../Property_Image.gif)
		 [Row](ITabFootnote.Row.htm)
		 Свойство Row определяет
		 индекс строки ячейки, для которой создана сноска.


		 ![](../../Property_Image.gif)
		 [Text](ITabFootnote.Text.htm)
		 Свойство Text определяет
		 текст сноски.
