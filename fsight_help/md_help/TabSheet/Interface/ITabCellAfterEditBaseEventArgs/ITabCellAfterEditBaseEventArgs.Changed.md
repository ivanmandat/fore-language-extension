# ITabCellAfterEditBaseEventArgs.Changed


## Синтаксис


Changed: Boolean;


## Описание


Свойство Changed возвращает признак изменения значения после редактирования ячейки таблицы.


## Комментарии


Свойство возвращает следующие значения:


-
True - значения в ячейке было изменено.


-
False - значение в ячейке не изменилось, либо редактирование было отменено.
