# ITabObjects


Сборка: Tab;


## Описание


Интерфейс ITabObjects содержит
 свойства и методы для работы с коллекцией объектов, расположенных на листе
 таблицы.


## Иерархия наследования


           ITabObjects


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITabObjects.Count.htm)


		 Свойство Count возвращает
		 количество объектов, расположенных на листе таблицы.


		 ![](../../Property_Image.gif)
		 [Item](ITabObjects.Item.htm)


		 Свойство Item возвращает
		 объект листа таблицы по индексу.


		 ![](../../Property_Image.gif)
		 [Object](ITabObjects.Object.htm)


		 Свойство Object возвращает
		 объект листа таблицы по идентификатору.


		 ![](../../Property_Image.gif)
		 [SelectedCount](ITabObjects.SelectedCount.htm)


		 Свойство SelectedCount
		 возвращает количество выделенных объектов.


		 ![](../../Property_Image.gif)
		 [SelectedObject](ITabObjects.SelectedObject.htm)


		 Свойство SelectedObject
		 возвращает выделенный объект.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ITabObjects.Add.htm)


		 Метод Add добавляет
		 на лист таблицы новый объект.


		 ![](../../Sub_Image.gif)
		 [AdjustSelected](ITabObjects.AdjustSelected.htm)


		 Метод AdjustSelected
		 осуществляет подбор размера выделенных объектов.


		 ![](../../Sub_Image.gif)
		 [AlignSelected](ITabObjects.AlignSelected.htm)


		 Метод AlignSelected
		 выравнивает группы объектов относительно видимой области листа.


		 ![](../../Sub_Image.gif)
		 [CenterSelectedAtView](ITabObjects.CenterSelectedAtView.htm)


		 Метод CenterSelectedAtView
		 осуществляет прокрутку таблицы таким образом, чтобы центр выделенных
		 объектов находился в центре видимой области таблицы.


		 ![](../../Sub_Image.gif)
		 [Clear](ITabObjects.Clear.htm)


		 Метод Clear удаляет
		 все объекты, расположенные на листе таблицы.


		 ![](../../Sub_Image.gif)
		 [GroupSelected](ITabObjects.GroupSelected.htm)


		 Метод GroupSelected
		 осуществляет группировку выделенных объектов.


		 ![](../../Sub_Image.gif)
		 [GetObjectIndex](ITabObjects.GetObjectIndex.htm)


		 Метод GetObjectIndex
		 возвращает индекс объекта.


		 ![](../../Sub_Image.gif)
		 [Remove](ITabObjects.Remove.htm)


		 Метод Remove удаляет
		 объект по индексу.


		 ![](../../Sub_Image.gif)
		 [UngroupSelected](ITabObjects.UngroupSelected.htm)


		 Метод UngroupSelected
		 осуществляет разгруппировку выделенных объектов.
