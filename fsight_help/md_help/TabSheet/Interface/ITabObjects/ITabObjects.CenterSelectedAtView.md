# ITabObjects.CenterSelectedAtView


## Синтаксис


CenterSelectedAtView;


## Описание


Метод CenterSelectedAtView осуществляет
 прокрутку таблицы таким образом, чтобы центр выделенных объектов находился
 в центре видимой области таблицы.
