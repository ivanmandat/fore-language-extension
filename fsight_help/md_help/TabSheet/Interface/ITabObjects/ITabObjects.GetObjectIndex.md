# ITabObjects.GetObjectIndex


## Синтаксис


GetObjectIndex(Id: String): Integer;


## Параметры


Id — идентификатор
 объекта, индекс которого необходимо получить.


## Описание


Метод GetObjectIndex возвращает
 индекс объекта, идентификатор которого передается в качестве входного
 параметра Id.
