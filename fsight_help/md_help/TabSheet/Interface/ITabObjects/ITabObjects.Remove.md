# ITabObjects.Remove


## Синтаксис


Remove(Index: Integer);


## Параметры


Index —
 индекс объекта в коллекции объектов, который необходимо удалить.


## Описание


Метод Remove удаляет объект,
 индекс которого передается в качестве входного параметра Index.
