# MapChart.ArrowTextOrientation


Пространство имен: PP


## Описание


Перечисление MapChart.ArrowTextOrientation содержит варианты размещения подписи стрелки.


Используется свойством [MapArrowBase.TextOrientation](../Classes/Map/MapArrowBase/MapArrowBase.TextOrientation.htm).


## Допустимые значения


Значение


Краткое описание


AlongTheArrow


Подпись расположена вдоль стрелки.


Horizontal


Подпись расположена горизонтально.
