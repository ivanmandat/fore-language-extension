# ScaleMode


Пространство имен: PP


## Описание


Перечисление ScaleMode содержит
 режимы расчета шкалы.


Используется свойством [ScaleBase.Mode](dhtmlGraphicBase.chm::/Classes/PP/ScaleBase/ScaleBase.Mode.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 Linear


		 Линейный расчет шкалы.


		 Uniform


		 Равномерный расчет шкалы.


		 Logarithmic


		 Логарифмический расчет шкалы.


		 Spectrum


		 Спектральный расчет шкалы. В разработке.
