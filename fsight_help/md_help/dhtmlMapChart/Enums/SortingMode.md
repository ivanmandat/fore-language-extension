# MapChart.SortingMode


Пространство имен: PP


## Описание


Перечисление MapChart.SortingMode содержит варианты сортировки элементов расшифровки территорий.


Используется свойством [MapTerritoryTranscript.SortingMode](../Classes/Map/MapTerritoryTranscript/MapTerritoryTranscript.SortingMode.htm).


## Допустимые значения


Значение


Краткое описание


Asc


По возрастанию.


Desc


По убыванию.


None


Нет сортировки.
