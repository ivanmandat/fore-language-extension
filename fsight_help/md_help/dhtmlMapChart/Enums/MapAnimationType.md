# MapAnimationType


Пространство имен: PP.Ui.


## Описание


Перечисление MapAnimationType
 содержит режимы воспроизведения анимации столбикового показателя на карте.


Используется следующим свойством:


	- [MapBar.getAnimationType](../Classes/Map/MapBar/MapBar.getAnimationType.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 1


		 Appear.
		 Анимация воспроизводится при отрисовке столбикового показателя
		 карты.


		 2


		 Modify.
		 Анимация воспроизводится при изменении столбикового показателя
		 карты.


		 4


		 Disappear.
		 Анимация воспроизводится при удалении столбикового показателя
		 карты.
