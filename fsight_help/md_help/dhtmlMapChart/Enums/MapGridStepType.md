# MapGridStepType


Пространство имен:
 PP


## Описание


Перечисление MapGridStepType
 содержит режимы построения горизонтальных и вертикальных линий сетки карты.


Используется следующими свойствами:


	- [MapGrid.HorizontalLinesStepType](../Classes/Map/MapGrid/MapGrid.HorizontalLinesStepType.htm);


	- [MapGrid.VerticalLinesStepType](../Classes/Map/MapGrid/MapGrid.VerticalLinesStepType.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 Pixels


		 Расстояния между линиями сетки
		 задаются в пикселях.


		 Relative


		 Расстояния между горизонтальными
		 и вертикальными линиями сетки задаются относительно ширины и высоты
		 сетки.


		 StepsCount


		 Сетка строится по заданному количеству
		 горизонтальных и вертикальных линий.
