# MapChart.HoverMode


Пространство имен: PP


## Описание


Перечисление MapChart.HoverMode
 содержит варианты появления всплывающей подсказки.


Используется свойством [ColorLegendBase.HoverMode](dhtmlUi.chm::/Classes/ColorLegendBase/ColorLegendBase.HoverMode.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 Default


		 Поведение по умолчанию.


		 None


		 Подсказка не отображается.


		 Click


		 Подсказка появляется по щелчку мыши.


		 MouseOver


		 Подсказка появляется при наведении
		 курсора.


		 MouseOverCenter


		 Подсказка отображается в центре
		 элемента при наведении курсора.
