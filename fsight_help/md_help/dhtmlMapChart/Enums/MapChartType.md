# MapChartType


Пространство имен: PP


## Описание


Перечисление MapChartType содержит
 технологии отрисовки топоосновы карты.


Используется свойством [MapChart.Type](../Classes/Map/MapChart/MapChart.Type.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 SVG
		 Технология SVG. Используется по умолчанию.


		 WebGL2D
		 Технология WebGL2D.


		 WebGL
		 Технология WebGL.


		 Sphere
		 Технология Sphere.
