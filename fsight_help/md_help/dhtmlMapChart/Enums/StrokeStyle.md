# MapChart.StrokeStyle


Пространство имен: PP


## Описание


Перечисление MapChart.StrokeStyle
 содержит варианты отображения контуров стрелки (для фигурной стрелки и
 стрелки с заливкой) или варианты отображения линии стрелки (для простой
 стрелки).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 Dash


		 Прерывистая линия.


		 DashDot


		 Линия из точек.


		 DashDotDot


		 Пунктир (тире-точка-тире).


		 Dot


		 Пунктир (тире-точка-точка-тире).


		 Solid


		 Сплошная линия.
