# MapChart.ArrowEffectsMode


Пространство имен: PP.Ui;


## Описание


Перечисление MapChart.ArrowEffectsMode
 содержит варианты подсветки стрелок карты.


Используется свойством [MapChart.ArrowEffectsMode](../Classes/Map/MapChart/MapChart.ArrowEffectsMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 StartShape


		 Подсветка стрелок, выходящих из региона.


		 EndShape


		 Подсветка стрелок, заходящих в регион.


		 Arrow


		 Стрелки становятся прозрачными при выделении любого региона.


		 None


		 Без подсветки.
