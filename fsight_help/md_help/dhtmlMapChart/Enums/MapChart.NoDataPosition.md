# MapChart.NoDataPosition


Пространство имен: PP


## Описание


Перечисление MapChart.NoDataPosition
 содержит варианты размещения элемента легенды «Нет данных» на карте.


Используется следующим свойством:


	- [ColorLegendBase.NoDataPosition](dhtmlUi.chm::/Classes/ColorLegendBase/ColorLegendBase.NoDataPosition.htm).

		 Значение


		 Краткое

		 Перед всеми элементами.


		 After


		 После всех элементов.
