# Visibility


Пространство имен: PP.Ui


## Описание


Перечисление Visibility содержит
 варианты видимости объекта.


Используется следующим свойством:


	- [LegendBase.Visibility](dhtmlGraphicBase.chm::/Classes/LegendBase/LegendBase.Visibility.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 0


		 Collapsed.
		 Объект невидимый.


		 1


		 Visible.
		 Объект видимый.


		 2


		 Hidden.
		 Объект скрыт.


		 3


		 IsDependent.
		 Видимость зависит от внешних параметров, которые уникальны для
		 каждого класса, где используется перечисление.
