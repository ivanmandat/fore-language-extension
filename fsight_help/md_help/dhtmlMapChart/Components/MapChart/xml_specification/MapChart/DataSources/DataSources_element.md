# Элемент DataSources


Элемент DataSources представляет
 собой коллекцию элементов DataSource,
 которые содержат описания источников данных.


Элемент DataSource - экземпляр
 класса [PP.DataSource](dhtmlGraphicBase.chm::/Classes/PP/DataSource/DataSource.htm).


[Пример настройки источников данных](DataSources_exmpl.htm)


## Элемент DataSource


Элемент DataSource содержит
 описание статистического источника данных. Опциональный элемент.


Содержит элемент Dimension.


### Атрибут элемента DataSource


		 Наименование атрибута
		 Тип
		 Описание


		 Id
		 String
		 Идентификатор источника данных.


## Элемент Dimension


Элемент Dimension представляет
 собой массив значений показателей для областей карты.


### Атрибуты элемента массива Dimension


		 Наименование атрибута
		 Тип
		 Описание


		 Id
		 String
		 Идентификатор области карты, для которой хранятся значения
		 показателей в данном измерении.


		 Items
		 Array
		 Перечисление значений показателей (дробных чисел в инвариантной
		 культуре) через пробел (Items=”{12.3
		 11 17 9}”). Перечисление может содержать также слово NULL, обозначающее отсутствие
		 данных для того или иного среза.
