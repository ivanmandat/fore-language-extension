# Настройка словаря имен областей


На странице приведен код настройки словаря имен [областей](../MapShapes/MapShapes_element.htm) в формате [JSON](../../../xml_and_json.htm). Данный элемент должен быть размещен в [файле](../../xml-specification.htm) с настройками карты внутри корневого элемента [MapChart](../MapChart_element.htm).


"RegionsNames":


 {


   "RegionName":


     [


         {


            "@Name": "Ямало-Ненецкий автономный округ",


            "@Id": "RU-YAN"


          },


          {


           "@Name": "Ханты-Мансийский автономный округ",


           "@Id": "RU-KHM"


          },


         {


           "@Name": "Тюменская область",


           "@Id": "RU-TYU"


          },


         {


           "@Name": "Ненецкий автономный округ",


           "@Id": "RU-NEN"


          }


     ]


}


Наименования регионов выводятся в [расшифровку территорий карты](../MapTerritoryTranscript/Territory_exmpl.htm).
