# Элемент RegionsNames


Элемент RegionsNames содержит
 словарь имен областей карты из топоосновы.


[Настройка словаря имен областей](RegionsNames_exmpl.htm)


Включает в себя элемент RegionsName.


## Элемент RegionName


Элемент RegionName представляет
 собой массив элементов - экземпляров класса PP.[RegionName](../../../../../Classes/Map/RegionName/RegionName.htm).


### Атрибуты элемента массива RegionsName


		 Наименование атрибута
		 Тип
		 Описание


		 Id
		 String
		 Идентификатор формы, для которой задается имя.


		 Name
		 String
		 Имя формы.


		 ShowInTranscript
		 Boolean
		 Флаг, указывающий, отображается ли имя в расшифровке территорий.
		 Допустимые значения:


			- true. Отображается;


			- false. Не отображается.


		По умолчанию установлено значение
		 true.
