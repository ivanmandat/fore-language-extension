# Элемент Animation


Элемент Animation содержит описание
 анимации значка.


Примечание.
 На данный момент анимация значков не реализована.


## Атрибуты элемента <Animation>


		 Наименование атрибута
		 Тип
		 Описание


		 AnimationType
		 String
		 Тип анимации. Допустимые значения: Linear.
		 Для будущего использования.


		 ApplySeparately
		 Boolean
		 Определяет, применяются ли настройки по отдельности к отрезкам
		 или ко всей анимации целиком. Допустимые значения:


			- true. В атрибуте
			 StepTime содержится
			 время анимации одного отрезка.


			- false. В атрибуте
			 StepTime должно содержаться
			 время всей анимации, которое, в зависимости от длин, будет
			 распределено по отрезкам.


		 AutoReverse
		 Boolean
		 Определяет, будет ли цикл анимации иметь обратный ход. Допустимые
		 значения:


			- true. Цикл анимации
			 имеет обратный ход;


			- false. Цикл
			 анимации не имеет обратный ход.


		Атрибут актуален, если
		 Loop=”True”.


		 DelayTime
		 Integer
		 Время задержки (остановки) на текущей точке анимации в миллисекундах.


		 Loop
		 Boolean
		 Определяет, необходимо ли зацикливать анимацию. Допустимые
		 значения:


			- true. Необходимо
			 зацикливать анимацию;


			- false. Анимация
			 не зациклена.


		 StepTime
		 Integer
		 Время анимации одного отрезка или анимации целиком в миллисекундах.


		 AnimationPointsList
		 String
		 Список точек анимации. Список содержит строковое описание
		 точек, разделенных пробелами. Определяются относительные или абсолютные
		 значения точек в зависимости от значения атрибута OffsetType
		 значка, содержащего элемент анимации.
