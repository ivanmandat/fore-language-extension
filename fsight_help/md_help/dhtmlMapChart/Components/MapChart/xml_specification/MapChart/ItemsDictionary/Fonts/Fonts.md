# Элемент Font


Элемент Font представляет собой
 коллекцию шрифтов - объектов класса [Font](dhtmlCommon.chm::/Classes/PP/Font/Font.htm).


[Пример настройки коллекции шрифтов](Font_exmpl.htm)


## Атрибуты элемента коллекции шрифтов


		 Наименование атрибута
		 Тип
		 Описание


		 Id
		 String
		 Идентификатор объекта шрифта.


		 Color
		 String
		 Цвет шрифта, может задаваться в следующих форматах:


			- "red";


			- "#000";


			- "RGB(123, 12, 1)".


		 Dictionary
		 Object
		 [Словарь
		 элементов](../ItemsDictionary.htm), в который входит шрифт.


		 FontFamily
		 PP.[Font.Family](dhtmlCommon.chm::/Enums/Font.Family.htm)
		 Название шрифта.


		 IsBold
		 Boolean
		 Определяет,
		 является ли начертание полужирным:


			- true - является;


			- false - не является.


		 IsCapitalized
		 Boolean
		 Определяет,
		 являются ли все символы в тексте прописными:


			- true
			- является;


			- false
			 - не является.


		 IsInferior
		 Boolean
		 Определяет,
		 является ли текст подстрочным:


			- true - является;


			- false - не является.


		 IsItalic
		 Boolean
		 Флаг, указывающий,
		 является ли шрифт курсивным. Допустимые значения:


			- true - курсив;


			- false - не курсив.


		 IsLowerCapitalized
		 Boolean
		 Определяет,
		 используются ли малые прописные символы:


			- true - используются;


			- false - не используются.


		 IsStriken
		 Boolean
		 Определяет,
		 является ли текст зачеркнутым:


			- true - является;


			- false - не является.


		 IsSuperior
		 Boolean
		 Определяет,
		 является ли текст надстрочным:


			- true - является;


			- false - не является.


		 IsUnderline
		 Boolean
		 Определяет,
		 является ли текст подчеркнутым:


			- true - является;


			- false - не является.


		 Size
		 Number
		 Размер шрифта.


		 UseAutoColor
		 Boolean
		 Определяет,
		 используется ли цвет по умолчанию:


			- true - используется черный
			 цвет, атрибут Color
			 неактуален;


			- false - используется цвет
			 шрифта, заданный в атрибуте Color.
