# Создание DHTML-карты


Для создания карты потребуется:


-
файл с топоосновой в формате SVG, например, Russia.svg, World.svg;


-
[файл с json-настройками](xml_specification/xml-specification.htm) карты (создается разработчиком);


-
[html-страница](HTML_layout.htm), которая будет служить контейнером для карты.
