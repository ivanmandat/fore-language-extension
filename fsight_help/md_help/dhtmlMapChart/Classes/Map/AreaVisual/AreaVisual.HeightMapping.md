# AreaVisual.HeightMapping


## Синтаксис


HeightMapping: Array;


## Описание


Свойство HeightMapping определяет
 настройки сопоставления данных высоте элементов территориального показателя.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHeightMapping,
 а возвращается с помощью метода getHeightMapping.


Свойство содержит массив объектов типа [PP.DataMapping](dhtmlGraphicBase.chm::/Classes/PP/DataMapping/DataMapping.htm).
