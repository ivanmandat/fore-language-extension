# MapObject.getIsSettingsCreated


## Синтаксис


getIsSettingsCreated();


## Описание


Метод getIsSettingsCreated возвращает
 признак того, создан ли объект карты на основе настроек пользователя.


## Комментарии


Метод возвращает значение типа Boolean.


Допустимые значения:


	- true. Объект карты создан
	 на основе настроек пользователя;


	- false. Объект карты
	 создан на основе настроек, полученных при разборе топоосновы.
