# MapObject


Пространство имен: PP.Ui;


## Иерархия наследования


[Object](dhtmlCommon.chm::/Classes/object/object.htm)


MapObject


## Описание


Класс MapObject реализует абстрактный
 объект, отображающийся на карте.


## Синтаксис


PP.initClass(PP.Ui.MapObject, [PP.Object](dhtmlCommon.chm::/Classes/object/object.htm),
 "MapObject");
