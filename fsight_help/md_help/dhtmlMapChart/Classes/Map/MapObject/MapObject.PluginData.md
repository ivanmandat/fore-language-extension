# MapObject.PluginData


## Синтаксис


PluginData: Object;


## Описание


Свойство PluginData определяет
 данные внешнего компонента на карте.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setPluginData,
 а возвращается с помощью метода getPluginData.


Свойство предназначено для внутреннего использования.
