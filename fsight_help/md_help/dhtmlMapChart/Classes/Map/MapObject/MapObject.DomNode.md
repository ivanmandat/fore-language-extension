# MapObject.DomNode


## Синтаксис


DomNode: DOMElement;


## Описание


Свойство DomNode определяет
 DOM-вершину, соответствующую объекту на карте.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDomNode,
 а возвращается с помощью метода getDomNode.
