# MapTopobase.changeTopobase


## Синтаксис


changeTopobase(type: [PP.MapChartType](../../../Enums/MapChartType.htm),
 timeIndex: Integer, fireEvent: Boolean);


## Параметры


type. Тип данных;


timeIndex. Индекс применяемой
 топоосновы;


fireEvent. Признак вызова события.


## Описание


Метод changeTopobase меняет
 топооснову в соответствии с другим типом карты.
