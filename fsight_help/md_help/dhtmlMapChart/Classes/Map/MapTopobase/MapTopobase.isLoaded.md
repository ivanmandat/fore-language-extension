# MapTopobase.isLoaded


## Синтаксис


isLoaded();


## Описание


Метод isLoaded определяет, загружена
 ли топооснова по указанному пути.


## Комментарии


Если заданная топооснова загружена, то метод возвращает значение true, иначе - false.
