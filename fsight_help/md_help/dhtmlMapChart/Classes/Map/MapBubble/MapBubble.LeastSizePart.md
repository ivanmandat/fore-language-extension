# MapBubble.LeastSizePart


## Синтаксис


LeastSizePart: Number;


## Описание


Свойство LeastSizePart определяет
 минимальный диаметр пузырька.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLeastSizePart,
 а возвращается с помощью метода getLeastSizePart.
