# MapTerritoryTranscript.ItemsOrder


## Синтаксис


ItemsOrder: [PP.Ui.LegendItemsOrder](dhtmlGraphicBase.chm::/Enums/LegendItemsOrder.htm);


## Описание


Свойство ItemsOrder определяет
 порядок следования элементов в расшифровке территорий карты.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setItemsOrder,
 а возвращается с помощью метода getItemsOrder.


По умолчанию значение свойства равно PP.Ui.LegendItemsOrder.TopToBottomByPages.
