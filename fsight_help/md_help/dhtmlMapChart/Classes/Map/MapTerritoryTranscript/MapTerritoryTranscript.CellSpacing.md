# MapTerritoryTranscript.CellSpacing


## Синтаксис


CellSpacing: Number;


## Описание


Свойство CellSpacing определяет
 отступ элементов в расшифровке территорий карты.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setCellSpacing,
 а возвращается с помощью метода getCellSpacing.


По умолчанию значение свойства равно 5.
