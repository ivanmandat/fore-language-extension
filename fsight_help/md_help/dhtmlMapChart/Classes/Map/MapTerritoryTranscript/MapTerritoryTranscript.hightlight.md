# MapTerritoryTranscript.highlight


## Синтаксис


highlight(shape: [PP.MapShape](../MapShape/MapShape.htm),
 doHighlight: Boolean);


## Параметры


shape. Область слоя карты;


doHighlight. Признак того,
 требуется ли подсветить элемент в расшифровке территорий, соответствующий
 указанной области слоя карты. Допустимые значения:


	- true.
	 Элемент в расшифровке территорий требуется подсветить;


	- false.
	 Элемент в расшифровке территорий не требует подсветки (по умолчанию).


## Описание


Метод highlight подсвечивает
 элемент в расшифровке территорий, соответствующий указанной области слоя
 карты.
