# MapTerritoryTranscript.HighlightBrush


## Синтаксис


HighlightBrush: PP.[Brush](dhtmlCommon.chm::/Classes/PP/Brush/Brush.htm)


## Описание


Свойство HighlightBrush устанавливает кисть, которой будет подсвечиваться выделенный регион.


## Комментарии


Значение свойства - идентификатор [кисти](../../../Components/MapChart/xml_specification/MapChart/ItemsDictionary/Brushes/Brushes_exmpl.htm) из [словаря элементов](../../../Components/MapChart/xml_specification/MapChart/ItemsDictionary/ItemsDictionary.htm) или полное описание кисти.
