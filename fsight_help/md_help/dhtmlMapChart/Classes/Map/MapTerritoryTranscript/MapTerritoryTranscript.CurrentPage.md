# MapTerritoryTranscript.CurrentPage


## Синтаксис


CurrentPage: Integer;


## Описание


Свойство CurrentPage определяет
 текущую страницу в расшифровке территорий карты.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setCurrentPage,
 а возвращается с помощью метода getCurrentPage.


По умолчанию значение свойства равно 0.


Свойство актуально, если для свойства [MapTerritoryTranscript.PageMode](MapTerritoryTranscript.PageMode.htm)
 установлено значение PP.Ui.LegendPageMode.MultiplePages.
