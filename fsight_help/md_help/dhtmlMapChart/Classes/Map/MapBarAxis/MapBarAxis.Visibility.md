# MapBarAxis.Visibility


## Синтаксис


Visibility: [PP.Ui.Visibility](../../../Enums/Visibility.htm);


## Описание


Свойство Visibility определяет
 признак того, отображается ли ось столбикового показателя.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setVisibility,
 а возвращается с помощью метода getVisibility.


По умолчанию свойство равно значению PP.Ui.Visibility.Visible.
