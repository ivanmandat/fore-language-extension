# MapBarAxis.Pen


## Синтаксис


Pen: Type;


## Описание


Свойство Pen определяет перо
 для отрисовки делений оси.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setPen,
 а возвращается с помощью метода getPen.
