# MapBarAxis.prepareToRender


## Синтаксис


prepareToRender(canvas_node: [PP.Ui.Canvas](dhtmlGraphicBase.chm::/Classes/Canvas/Canvas.htm)
 | HTMLElement);


## Параметры


canvas_node. Канва, на которой
 требуется нарисовать ось столбикового показателя. Если в качестве параметра
 указана DOM-вершина, то она преобразуется в SVG-элемент, на которой затем
 будет отрисована ось.


## Описание


Метод prepareToRender отрисовывает
 ось столбикового показателя.
