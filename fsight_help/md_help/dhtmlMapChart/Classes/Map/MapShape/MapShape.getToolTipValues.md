# MapShape.getToolTipValues


## Синтаксис


getToolTipValues(noData: Boolean);


## Параметры


noData. Признак получения значений
 всплывающей подсказки, для которой не определены данные. Допустимые значения:


	- true.
	 Для всплывающей подсказки без данных будут возвращены значения (по
	 умолчанию);


	- false.
	 Для всплывающей подсказки без данных не будут возвращены значения.


## Описание


Метод getToolTipValues возвращает
 значения данных, предназначенных для отображения в всплывающей подсказке
 для области слоя карты.


## Комментарии


Метод возвращает JSON-объект со следующими свойствами: Name
 - наименование области слоя карты, TimelineStep
 - текущая отметка календарного измерения, HeightValue
 - значение, сопоставленное высоте данной области, AreaValue
 - соответствующее ей значение, AreaTitle
 - наименование сопоставления данных области слоя карты.
