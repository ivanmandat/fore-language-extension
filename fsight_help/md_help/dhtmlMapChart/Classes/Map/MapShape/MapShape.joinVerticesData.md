# MapShape.joinVerticesData


## Синтаксис


joinVerticesData();


## Описание


Метод joinVerticesData соединяет добавленные данные вершин, принадлежащих области слоя карты.


## Комментарии


Метод работает в картах с трёхмерной топоосновой, например, с WebGL и Sphere.
