# MapShape.getPoint


## Синтаксис


getPoint(offsetX: Number, offsetY: Number);


## Параметры


offsetX. Отступ от крайней левой точки области слоя карты, выраженный в долях. Значение может принимать значения от 0 до 1;


offsetY. Отступ от крайней верхней точки области слоя карты, выраженный в долях. Значение может принимать значения от 0 до 1.


## Описание


Метод getPoint возвращает точку, принадлежащую области слоя карты, по отступам от её левой и верхней границы.


## Комментарии


Метод вовзращает значение типа PP.Vector3.
