# MapShape.StrokeWidth


## Синтаксис


StrokeWidth: Number;


## Описание


Свойство StrokeWidth определяет
 толщину границы для области слоя карты.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setStrokeWidth,
 а возвращается с помощью метода getStrokeWidth.
