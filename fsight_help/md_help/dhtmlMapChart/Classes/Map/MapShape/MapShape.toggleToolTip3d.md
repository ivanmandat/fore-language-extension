# MapShape.toggleToolTip3d


## Синтаксис


toggleToolTip3d(vec: PP.Vector3);


## Параметры


vec. Координаты расположения всплывающей подсказки в трёхмерной карте.


## Описание


Метод toggleToolTip3d отображает и скрывает всплывающую подсказку для области слоя трёхмерной карты попеременно после каждого вызова.
