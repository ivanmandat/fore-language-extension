# MapShape.toggleTooltip


## Синтаксис


toggleTooltip(x: Number, y: Number);


## Параметры


x. Экранная координата расположения всплывающей подсказки относительно горизонтальной оси карты;


y. Экранная координата расположения всплывающей подсказки относительно вертикальной оси карты.


## Описание


Метод toggleTooltip отображает и скрывает всплывающую подсказу для области слоя карты попеременно после каждого вызова.
