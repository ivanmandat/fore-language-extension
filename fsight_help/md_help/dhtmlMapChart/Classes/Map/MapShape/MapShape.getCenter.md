# MapShape.getCenter


## Синтаксис


getCenter();


## Описание


Метод getCenter возвращает центральную
 точку области слоя карты.


## Комментарии


Если топооснова карты трёхмерная, то метод возвращает значение типа
 PP.Vector3. Если двумерная, то
 значение типа [PP.Point](dhtmlCommon.chm::/Classes/PP/Point/Point.htm).
