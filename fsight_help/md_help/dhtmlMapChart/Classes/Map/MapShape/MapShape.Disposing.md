# MapShape.Disposing


## Синтаксис


Disposing: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие Disposing наступает
 после удаления области слоя карты.


## Пример


Для выполнения примера необходимо наличие на html-странице объекта типа
 [PP.MapShape](MapShape.htm) (см. страницу описания свойства
 [MapShape.Visual](MapShape.Visual.htm)) с наименованием «mapShape».
 Удалим область слоя карты и всплывающую подсказку для неё:


// Обработаем событие удаления области слоя карты
mapShape.Disposing.add(function(sender, args) {
    console.log("Удалена область слоя карты «%s»", sender.getId());
});
// Удаляем область слоя карты
mapShape.dispose();
После выполнения примера область слоя карты и всплывающая подсказка
 были удалены, о чём было выведено соответствующее уведомление в консоли
 браузера:


Удалена область слоя карты «RU-KYA»
