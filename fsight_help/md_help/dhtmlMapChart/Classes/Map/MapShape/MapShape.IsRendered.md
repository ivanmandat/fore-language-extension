# MapShape.IsRendered


## Синтаксис


IsRendered: Boolean;


## Описание


Свойство IsRendered определяет,
 отрисована ли область слоя карты.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsRendered,
 а возвращается с помощью метода getIsRendered.


Допустимые значения:


	- true.
	 Область слоя карты отрисована;


	- false.
	 Область слоя карты не отрисована (по умолчанию).


Свойство предназначено только для чтения.
