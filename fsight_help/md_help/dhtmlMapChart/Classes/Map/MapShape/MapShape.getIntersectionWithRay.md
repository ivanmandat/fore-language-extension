# MapShape.getIntersectionWithRay


## Синтаксис


getIntersectionWithRay(ray: PP.Ray);


## Параметры


ray. Луч, пересекающий область
 слоя карты. Представляет собой JSON-объект с со следующими свойствами:
 start - точка начала луча, dir - точка, определяющая направление луча.


## Описание


Метод getIntersectionWithRay
 возвращает расстояние от начала луча до его пересечения со стрелкой на
 карте


## Комментарии


Метод возвращает значение типа Number.
