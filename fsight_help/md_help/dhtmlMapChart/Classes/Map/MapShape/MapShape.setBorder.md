# MapShape.setBorder


## Синтаксис


setBorder(enabled: Boolean, color: String, opacity: Number);


## Параметры


enabled. Признак того, что граница области слоя карты будет отображена. Если параметр равен значению true, то граница будет отражена, иначе (по умолчания) - не будет;


color. Цвет границы;


opacity. Коэффициент прозрачности границы. Параметр может принимать значение от 0 (полностью прозрачная граница) до 1 (полностью непрозрачная граница).


## Описание


Метод setBorder определяет границу для области слоя карты.
