# MapShape.ParentLayer


## Синтаксис


ParentLayer: PP.[MapLayer](../MapLayer/MapLayer.htm)


## Описание


Свойство ParentLayer устанавливает родительский слой для области карты.


## Комментарии


Значение свойства устанавливается из JSON, а возвращается с помощью метода getParentLayer. Установить значение с помощью метода setParentLayer нельзя.
