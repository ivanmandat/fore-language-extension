# MapShape.isOpaque


## Синтаксис


isOpaque();


## Описание


Метод isOpaque возвращает признак
 того, является ли область слоя карты непрозрачной.


## Комментарии


Метод возвращает значение типа Boolean.


Допустимые значения:


	- true. Область слоя карты
	 непрозрачна;


	- false. Область слоя
	 карты прозрачна.
