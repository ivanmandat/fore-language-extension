# MapGrid.VerticalLinesCount


## Синтаксис


VerticalLinesCount: Number;


## Описание


Свойство VerticalLinesCount определяет количество вертикальных линий сетки.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setVerticalLinesCount.


Данное свойство актуально, если значение свойства [MapGrid.VerticalLinesStepType](MapGrid.VerticalLinesStepType.htm) равно PP.MapGridStepType.StepsCount.
