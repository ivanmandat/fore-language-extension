# MapGrid.VerticalLinesStepType


## Синтаксис


VerticalLinesStepType: [MapGridStepType](../../../Enums/MapGridStepType.htm);


## Описание


Свойство VerticalLinesStepType определяет режим построения вертикальных линий сетки.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setVerticalLinesStepType.
