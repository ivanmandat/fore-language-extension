# MapChart.VerticalLinesStep


## Синтаксис


VerticalLinesStep: Number;


## Описание


Свойство VerticalLinesStep определяет расстояние между вертикальными линиями сетки карты в пикселях.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setVerticalLinesStep.


Данное свойство актуально, если значение свойства [MapGrid.VerticalLinesStepType](MapGrid.VerticalLinesStep.htm) равно PP.MapGridStepType.Pixels или PP.MapGridStepType.Relative.
