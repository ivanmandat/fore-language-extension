# MapGrid.HorizontalLinesStepType


## Синтаксис


HorizontalLinesStepType: [MapGridStepType](../../../Enums/MapGridStepType.htm);


## Описание


Свойство HorizontalLinesStepType определяет режим построения горизонтальных линий сетки.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHorizontalLinesStepType.
