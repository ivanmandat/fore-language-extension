# MapGrid.getIsDrawed


## Синтаксис


getIsDrawed();


## Описание


Метод getIsDrawed определяет, нарисована ли сетка на карте.


## Комментарии


Метод возвращает значение true, если на карте нарисована сетка, и значение false, если не нарисована.
