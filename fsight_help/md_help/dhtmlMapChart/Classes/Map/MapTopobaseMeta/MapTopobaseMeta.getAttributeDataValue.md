# MapTopobaseMeta.getAttributeDataValue


## Синтаксис


getAttributeDataValue(attrName: String, dataId:
 String);


## Параметры


attrName. Наименование атрибута;


dataId. Идентификатор объекта
 данных атрибута.


## Описание


Метод getAttributeDataValue
 возвращает значение объекта данных атрибута по наименованию атрибута и
 идентификатору объекта.


## Комментарии


Метод возвращает значение типа String.
