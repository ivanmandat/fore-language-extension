# MapTopobaseMeta.setGeoItem


## Синтаксис


setGeoItem(geoItem: [PP.MapGeoItem](../MapGeoItem/MapGeoItem.htm));


## Параметры


geoItem. Объект связи географических координат с координатами топоосновы карты.


## Описание


Метод setGeoItem устанавливает объект связи географических координат с координатами топоосновы карты.
