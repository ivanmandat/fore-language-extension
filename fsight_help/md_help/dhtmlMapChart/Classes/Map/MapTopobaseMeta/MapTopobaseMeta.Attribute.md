# MapTopobaseMeta.Attribute


## Синтаксис


Attribute: [PP.MapMetaAttribute](../MapMetaAttribute/MapMetaAttribute.htm);


## Описание


Свойство Attribute определяет
 атрибут для метаданных топоосновы карты.


## Комментарии


Для добавления атрибута в метаданные топоосновы карты используется метод
 setAttribute, в качестве его параметра
 attr указывается добавляемый атрибут.
 Для его получения используется метод getAttribute,
 в котором в качестве параметра name
 необходимо указать наименование возвращаемого атрибута.


Из JSON значение свойства задать нельзя.
