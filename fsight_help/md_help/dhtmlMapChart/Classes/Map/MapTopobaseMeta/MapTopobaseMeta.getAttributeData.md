# MapTopobaseMeta.getAttributeData


## Синтаксис


getAttributeData(attrName: String, dataId: String);


## Параметры


attrName. Наименование атрибута;


dataId. Идентификатор объекта
 данных атрибута.


## Описание


Метод getAttributeData возвращает
 объект данных атрибута по наименованию атрибута и идентификатору объекта.


## Комментарии


Метод возвращает значение типа [PP.MapMetaAttributeData](../MapMetaAttributeData/MapMetaAttributeData.htm).
