# MapFiguredArrow.BasementWidth


## Синтаксис


BasementWidth: Double


## Описание


Свойство BasementWidth устанавливает ширину основания стрелки.


## Комментарии


Значение свойства устанавливается в процентах относительно длины стрелки. Допустимые значения от 0 до 1.
