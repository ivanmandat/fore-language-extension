# MapPie.LeastAnglePart


## Синтаксис


LeastAnglePart: Number;


## Описание


Свойство LeastAnglePart определяет
 минимальный размер сектора в градусах.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLeastAnglePart,
 а возвращается с помощью метода getLeastAnglePart.


По умолчанию значение свойства равно 2.
