# MapPie.LeastRadiusPart


## Синтаксис


LeastRadiusPart: Number;


## Описание


Свойство LeastRadiusPart определяет
 минимальный радиус сектора.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLeastRadiusPart,
 а возвращается с помощью метода getLeastRadiusPart.
