# MapPie.HideNoData


## Синтаксис


HideNoData: Boolean;


## Описание


Свойство HideNoData определяет,
 требуется ли скрывать сектора с нулевым значением.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHideNoData,
 а возвращается с помощью метода getHideNoData.


Допустимые значения:


	- true.
	 Сектора с нулевым значением будут скрыты (по умолчанию);


	- false.
	 Сектора с нулевым значением будут отображены.
