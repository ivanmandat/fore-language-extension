# MapPie.StartAngle


## Синтаксис


StartAngle: Number;


## Описание


Свойство StartAngle определяет
 угол поворота первого сектора кругового показателя.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setStartAngle,
 а возвращается с помощью метода getStartAngle.
