# MapPie.MaxPiesCount


## Синтаксис


MaxPiesCount: Number;


## Описание


Свойство MaxPiesCount определяет
 максимальное количество сегментов в круговом показателе.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMaxPiesCount,
 а возвращается с помощью метода getMaxPiesCount.
