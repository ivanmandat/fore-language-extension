# MapPie.BorderThickness


## Синтаксис


BorderThickness: Number;


## Описание


Свойство BorderThickness устанавливает
 толщину границы секторов в пикселях.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setBorderThickness,
 а возвращается с помощью метода getBorderThickness.
