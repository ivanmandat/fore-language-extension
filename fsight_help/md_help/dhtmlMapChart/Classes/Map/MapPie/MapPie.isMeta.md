# MapPie.isMeta


## Синтаксис


isMeta();


## Описание


Метод isMeta возвращает признак
 наличия нескольких сопоставлений данных у кругового показателя.


## Комментарии


Допустимые значения:


	- true. Для кругового
	 показателя задано более одного сопоставления данных;


	- false. Для кругового
	 показателя задано одно сопоставление данных или не задано ни одного.
