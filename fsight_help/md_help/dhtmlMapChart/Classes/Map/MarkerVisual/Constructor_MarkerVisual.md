# Конструктор MarkerVisual


## Синтаксис


PP.MarkerVisual(settings);


## Параметры


settings. JSON-объект со значениями
 свойств класса.


## Описание


Конструктор MarkerVisual создает
 экземпляр класса MarkerVisual.


## Пример


Конструктор предназначен для внутреннего использования.
