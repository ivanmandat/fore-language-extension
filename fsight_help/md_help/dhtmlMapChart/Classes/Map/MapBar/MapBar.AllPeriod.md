# MapBar.AllPeriod


## Синтаксис


AllPeriod: Boolean;


## Описание


Свойство AllPeriod определяет,
 отображаются ли в столбиковом показателе данные за весь период времени.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setAllPeriod,
 а возвращается с помощью метода getAllPeriod.


Допустимые значения:


	- true.
	 Данные отображаются за весь период времени (по умолчанию);


	- false.
	 Данные отображаются только за текущий период времени.
