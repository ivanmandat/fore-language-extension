# MapBar.BarDistance


## Синтаксис


BarDistance: Number;


## Описание


Свойство BarDistance определяет
 расстояние между столбцами в столбиковом показателе.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setBarDistance,
 а возвращается с помощью метода getBarDistance.
