# MapBar.HideZeroHeight


## Синтаксис


HideZeroHeight: Boolean;


## Описание


Свойство HideZeroHeight определяет,
 требуется ли скрывать столбцы с нулевым значением.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHideZeroHeight,
 а возвращается с помощью метода getHideZeroHeight.


Допустимые значения:


	- true.
	 Столбцы с нулевым значением будут скрыты;


	- false.
	 Столбцы с нулевым значением будут отображены (по умолчанию).
