# MapBar.LeastWidthPart


## Синтаксис


LeastWidthPart: Number;


## Описание


Свойство LeastWidthPart устанавливает
 минимальную ширину столбца.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLeastWidthPart,
 а возвращается с помощью метода getLeastWidthPart.
