# MapBar.Axis


## Синтаксис


Axis: [PP.MapBarAxis](../MapBarAxis/MapBarAxis.htm);


## Описание


Свойство Axis определяет ось
 данных для столбикового представления.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setAxis,
 а возвращается с помощью метода getAxis.
