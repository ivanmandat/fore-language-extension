# MapBar.getAnimationType


## Синтаксис


getAnimationType();


## Описание


Метод getAnimationType возвращает
 режим воспроизведения анимации столбикового показателя карты.


## Комментарии


Метод возвращает значение типа [PP.Ui.MapAnimationType](../../../Enums/MapAnimationType.htm).
