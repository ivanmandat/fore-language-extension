# MapBar.MaxBarsCount


## Синтаксис


MaxBarsCount: Number;


## Описание


Свойство MaxBarsCount определяет
 максимальное количество столбцов в столбиковом показателе.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMaxBarsCount,
 а возвращается с помощью метода getMaxBarsCount.
