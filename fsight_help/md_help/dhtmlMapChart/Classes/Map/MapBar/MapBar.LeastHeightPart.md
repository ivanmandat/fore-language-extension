# MapBar.LeastHeightPart


## Синтаксис


LeastHeightPart: Number;


## Описание


Свойство LeastHeightPart устанавливает
 минимальную высоту столбца.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLeastHeightPart,
 а возвращается с помощью метода getLeastHeightPart.
