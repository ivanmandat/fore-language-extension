# MapBar.IsPercentage


## Синтаксис


IsPercentage: Boolean;


## Описание


Свойство IsPercentage определяет,
 является ли значение столбца процентным показателем.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsPercentage,
 а возвращается с помощью метода getIsPercentage.


Допустимые значения:


	- true.
	 Значение столбца является процентным показателем: ![](../../../Components/MapChart/chartmap15.gif);


	- false.
	 Значение столбца имеет абсолютную величину: ![](../../../Components/MapChart/chartmap14.gif)
	 (по умолчанию).
