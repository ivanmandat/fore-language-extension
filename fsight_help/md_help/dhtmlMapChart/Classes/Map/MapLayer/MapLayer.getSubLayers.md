# MapLayer.getSubLayers


## Синтаксис


getSubLayers();


## Описание


Метод getSubLayers возвращает
 все дочерние слои первого уровня вложенности.


## Комментарии


Метод возвращает JSON-объект, поля которого имеют наименования, равные
 идентификаторам дочерних слоёв карты, и значения, содержащие объекты типа
 [PP.MapLayer](MapLayer.htm).
