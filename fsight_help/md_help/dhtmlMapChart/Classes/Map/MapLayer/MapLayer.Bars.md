# MapLayer.Bars


## Синтаксис


Bars: Array;


## Описание


Свойство Bars определяет столбиковые
 показатели для слоя карты.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setBars,
 а возвращается с помощью метода getBars.
