# MapLayer.updateLabelPositions


## Синтаксис


updateLabelPositions(scale: Number);


## Параметры


scale. Масштаб карты.


## Описание


Метод updateLabelPositions обновляет расположение меток для областей текущего и дочерних слоёв карты.
