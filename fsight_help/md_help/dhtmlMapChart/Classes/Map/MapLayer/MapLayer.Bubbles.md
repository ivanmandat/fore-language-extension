# MapLayer.Bubbles


## Синтаксис


Bubbles: Array;


## Описание


Свойство Bubbles определяет
 пузырьковые показатели для слоя карты.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setBubbles,
 а возвращается с помощью метода getBubbles.
