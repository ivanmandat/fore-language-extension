# MapLayer.Pies


## Синтаксис


Pies: Array;


## Описание


Свойство Pies определяет круговые
 показатели для слоя карты.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setPies,
 а возвращается с помощью метода getPies.
