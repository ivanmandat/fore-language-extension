# MapLayer.Label


## Синтаксис


Label: PP.[MapLabel](../MapLabel/MapLabel.htm)


## Описание


Свойство Label устанавливает метку для слоя.


## Комментарии


Значение свойства - идентификатор метки , описанной в словаре элементов, или полное описание метки формы.
