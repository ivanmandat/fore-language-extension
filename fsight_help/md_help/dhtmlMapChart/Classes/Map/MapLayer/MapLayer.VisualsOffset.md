# MapLayer.VisualsOffset


## Синтаксис


VisualsOffset: Integer;


## Описание


Свойство VisualsOffset определяет
 расстояние между показателями карты, расположенными в одну линию.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setVisualsOffset,
 а возвращается с помощью метода getVisualsOffset.


По умолчанию значение свойства равно 2.
