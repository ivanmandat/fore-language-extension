# MapLayer.setSize


## Синтаксис


setSize(width: Number, height: Number, scale: Number);


## Параметры


width.
 Ширина слоя карты;


height. Высота слоя карты;


scale. Масштаб отображения
 слоя карты.


## Описание


Метод setSize устанавливает
 размеры и масштаб отображения слоя карты.

Метод находится в разработке.
