# MapLayer.getShape


## Синтаксис


getShape(sId: String);


## Параметры


sId. Идентификатор области
 слоя карты.


## Описание


Метод getShape возвращает область
 слоя карты по её идентификатору.


## Комментарии


Метод возвращает значение типа [PP.MapShape](../MapShape/MapShape.htm).
