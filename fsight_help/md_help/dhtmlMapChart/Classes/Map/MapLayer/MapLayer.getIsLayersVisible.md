# MapLayer.getIsLayersVisible


## Синтаксис


getIsLayersVisible();


## Описание


Метод getIsLayersVisible возвращает
 признак того, отображается ли слой карты.


## Комментарии


Метод возвращает значение типа Boolean.


Допустимые значения:


	- true. Слой карты отображается;


	- false. Слой карты скрыт.
