# MapLayer.getSubLayer


## Синтаксис


getSubLayer(id: Number);


## Параметры


id. Идентификатор возвращаемого
 дочернего слоя.


## Описание


Метод getSubLayer возвращает
 дочерний слой карты по его идентификатору.


## Комментарии


Метод возвращает объект типа [PP.MapLayer](MapLayer.htm).
