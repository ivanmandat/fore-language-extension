# MapLayer.getMapBubbleCollection


## Синтаксис


getMapBubbleCollection();


## Описание


Метод getMapBubbleCollection
 возвращает массив пузырьковых показателей карты.


## Комментарии


Метод возвращает массив объектов типа [PP.MapBubble](../MapBubble/MapBubble.htm).
