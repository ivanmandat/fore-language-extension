# MapLayer.Shapes


## Синтаксис


Shapes: Object;


## Описание


Свойство Shapes определяет области
 слоя карты.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setShapes,
 а возвращается с помощью метода getShapes.


Свойство содержит JSON-объект, поля которого имеют наименования, равные
 идентификаторам областей слоя карты, и значения, содержащие объекты типа
 [PP.MapShape](../MapShape/MapShape.htm).
