# MapLayer.updateTooltipPositions


## Синтаксис


updateTooltipPositions(scale: Number);


## Параметры


scale. Масштаб карты.


## Описание


Метод updateTooltipPositions обновляет расположение всплывающей подсказки для области слоя карты.
