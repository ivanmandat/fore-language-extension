# MapLayer.animate


## Синтаксис


animate(step: Number);


## Параметры


step. Доля анимации для воспроизведения. Параметр может принимать числовое значение от 0 до 1 включительно.


## Описание


Метод animate воспроизводит анимацию областей для текущей и дочерних слоёв карты.


## Пример


Выполним пример, приведённый на странице описания метода [MapShape.animate](../MapShape/MapShape.animate.htm), предварительно заменив строку сценария


shape.animate(0.5);

на следующую:


map.getLayer("Regions").animate(0.5);

Результат выполнения примера будет тем же самым.
