# MapLayer.Milestones


## Синтаксис


Milestones: Object;


## Описание


Свойство Milestones определяет
 значки с изображением на карте.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMilestones,
 а возвращается с помощью метода getMilestones.


Свойство содержит JSON-объект, полями которого являются объекты типа
 [PP.MapImageMilestone](../MapImageMilestone/MapImageMilestone.htm).
