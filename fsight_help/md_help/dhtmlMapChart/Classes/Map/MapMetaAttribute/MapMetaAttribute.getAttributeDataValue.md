# MapMetaAttribute.getAttributeDataValue


## Синтаксис


getAttributeDataValue(id: Number);


## Параметры


id. Идентификатор объекта данных
 атрибута.


## Описание


Метод getAttributeDataValue
 возвращает значение объекта данных по его идентификатору.


## Комментарии


Метод возвращает значение типа String.
