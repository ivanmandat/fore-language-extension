# MapMetaAttribute.AttributeData


## Синтаксис


AttributeData: [PP.MapMetaAttributeData](../MapMetaAttributeData/MapMetaAttributeData.htm);


## Описание


Свойство AttributeData определяет
 данные для атрибута.


## Комментарии


Для добавления данных для атрибута используется метод setAttributeData,
 в качестве его параметра data
 указывается добавляемый объект. Для его получения используется метод getAttributeData, в котором в качестве
 параметра id необходимо указать
 идентификатор возвращаемого объекта с данными атрибута.


Из JSON значение свойства задать нельзя.
