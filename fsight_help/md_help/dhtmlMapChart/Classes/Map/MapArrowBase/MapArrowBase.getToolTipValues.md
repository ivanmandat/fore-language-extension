# MapArrowBase.getToolTipValues


## Синтаксис


getToolTipValues();


## Описание


Метод getToolTipValues возвращает
 значения данных, предназначенных для отображения в всплывающей подсказке
 для стрелки на карте.


## Комментарии


Метод возвращает JSON-объект со следующими свойствами: StartShape
 - наименование области слоя карты, из которой следует стрелка, EndShape
 - наименование области слоя карты, в которую следует стрелка, TimelineStep
 - текущее деление временной оси, ColorValue
 - значение, сопоставленное цвету заливки стрелки, ColorTitle
 - наименование сопоставления данных цвету заливки стрелки, WidthValue
 - значение, сопоставленное ширине стрелки, WidthTitle
 - наименование сопоставления данных ширине стрелки.
