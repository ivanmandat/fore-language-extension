# MapArrowBase.PointerLength


## Синтаксис


PointerLength: Double


## Описание


Свойство PointerLength устанавливает высоту указателя стрелки.


## Комментарии


Значение свойства устанавливается в процентах от длины стрелки в диапазоне от 0 до 1.
