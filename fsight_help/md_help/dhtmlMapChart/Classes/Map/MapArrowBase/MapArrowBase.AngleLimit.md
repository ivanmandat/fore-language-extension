# MapArrowBase.AngleLimit


## Синтаксис


AngleLimit: Double;


## Описание


Свойство AngleLimit определяет максимальный
 угол между наконечником стрелки и касательной к кривой в радианах.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setAngleLimit,
 а возвращается с помощью метода getAngleLimit.


По умолчанию значение свойства равно 6.


Данное свойство не учитывается в трёхмерной карте.
