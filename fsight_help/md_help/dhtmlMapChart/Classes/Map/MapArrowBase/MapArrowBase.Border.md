# MapArrowBase.Border


## Синтаксис


Border: PP.Pen | Object;


## Описание


Свойство Border определяет границу
 стрелки на карте.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setBorder,
 а возвращается с помощью метода getBorder.
