# MapArrowBase.Visual


## Синтаксис


Visual: [PP.ArrowVisual](../ArrowVisual/ArrowVisual.htm);


## Описание


Свойство Visual определяет объект
 для сопоставления данных стрелки на карте его свойствам.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setVisual,
 а возвращается с помощью метода getVisual.
