# MapArrowBase.animate


## Синтаксис


animate(step: Number);


## Параметры


step. Отношение требуемой длительности
 воспроизведения анимации к её общей продолжительности. Допустимые значения
 - от 0 до 1 включительно.


## Описание


Метод animate воспроизводит
 анимацию стрелки на карте.


## Пример


Для выполнения примера необходимо наличие на html-странице объекта типа
 [PP.MapArrowBase](MapArrowBase.htm) (см. страницу «[Конструктор
 MapFilledArrow](../MapFilledArrow/Constructor_MapFilledArrow.htm)») с наименованием «mapArrow». Воспроизведём анимацию
 стрелки длительностью, равной половине времени от её общей продолжительности:


mapArrow.animate(0.5);
В результате выполнения примера анимация стрелки была воспроизведена
 за половину времени от её общей продолжительности.
