# MapArrowBase.ToolTip


## Синтаксис


ToolTip: [PP.Ui.ChartTooltipBase](dhtmlBubbleChart.chm::/Classes/BubbleChart/ChartTooltipBase/ChartTooltipBase.htm);


## Описание


Свойство ToolTip определяет
 всплывающую подсказку для стрелки на карте.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setToolTip,
 а возвращается с помощью метода getToolTip.
