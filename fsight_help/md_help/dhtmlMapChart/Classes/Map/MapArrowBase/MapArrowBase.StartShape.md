# MapArrowBase.StartShape


## Синтаксис


StartShape: PP.[MapShape](../MapShape/MapShape.htm)


## Описание


Свойство StartShape устанавливает область карты, с которой начинается стрелка.


## Комментарии


Свойство актуально, если значение свойства [StartPoint](MapArrowBase.StartPoint.htm) не определено.
