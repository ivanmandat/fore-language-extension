# MapArrowBase.getIntersectionWithRay


## Синтаксис


getIntersectionWithRay(ray: PP.Ray);


## Параметры


ray. Луч, для которого определяется
 расстояние от его начала до пересечения со стрелкой на карте.


## Описание


Метод getIntersectionWithRay
 возвращает расстояние от начала луча до его пересечения со стрелкой на
 трёхмерной карте.


## Комментарии


Метод возвращает значение типа Number.
