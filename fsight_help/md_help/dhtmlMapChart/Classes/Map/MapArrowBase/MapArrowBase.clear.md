# MapArrowBase.clear


## Синтаксис


clear();


## Описание


Метод clear удаляет стрелку
 со слоя карты.


## Пример


Для выполнения примера необходимо наличие на html-странице объекта типа
 [PP.MapArrowBase](MapArrowBase.htm) (см. «[Конструктор
 MapFilledArrow](../MapFilledArrow/MapFilledArrow.htm)») с наименованием «mapArrow». Удалим стрелку со слоя
 карты:


clearArrow(mapArrow);
В результате выполнения примера стрелка была удалена.
