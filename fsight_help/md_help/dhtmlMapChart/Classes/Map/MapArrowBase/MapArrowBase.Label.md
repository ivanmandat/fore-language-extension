 | PP.Ui.ChartSVGLabel;


## Описание


Свойство Label определяет подпись
 для стрелки на карте.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLabel,
 а возвращается с помощью метода getLabel.
