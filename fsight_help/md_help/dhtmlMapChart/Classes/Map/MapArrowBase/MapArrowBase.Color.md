# MapArrowBase.Color


## Синтаксис


Color: String


## Описание


Свойство Color устанавливает цвет стрелки в формате HTML.


## Комментарии


Значение свойства может задаваться в следующих форматах:


-
"red";


-
"#000";


-
"RGB(123, 12, 1)".
