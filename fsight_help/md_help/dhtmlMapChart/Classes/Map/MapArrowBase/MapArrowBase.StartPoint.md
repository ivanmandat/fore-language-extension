# MapArrowBase.StartPoint


## Синтаксис


StartPoint: PP.Point


## Описание


Свойство StartPoint устанавливает начальную точку стрелки.


## Комментарии


Свойство актуально, если значение свойства [StartShape](MapArrowBase.StartShape.htm) не определено.


Координаты точки задаются в пикселях.
