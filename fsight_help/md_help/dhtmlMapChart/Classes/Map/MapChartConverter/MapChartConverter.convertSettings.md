# MapChartConverter.convertSettings


## Синтаксис


convertSettings(oldSet: Object);


## Параметры


oldSet. JSON-объект со значениями
 свойств карты в старом формате.


## Описание


Метод convertSettings преобразует
 объект со значениями свойств [карты](../MapChart/MapChart.htm)
 старого формата в объект нового формата.


## Комментарии


Метод возвращает значение типа Object.


Метод предназначен для внутреннего использования.
