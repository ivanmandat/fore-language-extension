# MapVisualItem.Visual


## Синтаксис


Visual: [PP.Visual](dhtmlGraphicBase.chm::/Classes/PP/Visual/Visual.htm);


## Описание


Свойство Visual определяет объект
 для сопоставления данных показателя области слоя карты его измерениям.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setVisual,
 а возвращается с помощью метода getVisual.
