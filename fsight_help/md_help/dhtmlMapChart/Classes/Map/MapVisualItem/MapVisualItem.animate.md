# MapVisualItem.animate


## Синтаксис


animate(step: Number, animationTypes: Array);


## Параметры


step. Отношение требуемой длительности
 воспроизведения анимации к общей продолжительности анимации. Допустимые
 значения - от 0 до 1 включительно;


animationTypes. Массив, определяющий
 режимы воспроизведения анимации. Массив может содержать элементы перечисления [MapAnimationType](../../../Enums/MapAnimationType.htm).


## Описание


Метод animate воспроизводит
 анимацию показателя области слоя карты указанной длительности и в заданных
 режимах.
