# MapVisualItem.SelectedIndex


## Синтаксис


SelectedIndex: Number;


## Описание


Свойство SelectedIndex определяет
 индекс деления временной оси, данные для которого отображает показатель
 области слоя карты.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSelectedIndex,
 а возвращается с помощью метода getSelectedIndex.


По умолчанию значение свойства равно 0.
