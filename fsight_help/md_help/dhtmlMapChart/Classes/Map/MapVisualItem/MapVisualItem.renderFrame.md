# MapVisualItem.renderFrame


## Синтаксис


renderFrame(canvas_parentNode: [PP.Ui.Canvas](dhtmlGraphicBase.chm::/Classes/Canvas/Canvas.htm));


## Параметры


canvas_parentNode. Канва, на
 которой требуется нарисовать показатель области слоя карты.


## Описание


Метод renderFrame отрисовывает
 показатель области слоя карты на указанной канве.
