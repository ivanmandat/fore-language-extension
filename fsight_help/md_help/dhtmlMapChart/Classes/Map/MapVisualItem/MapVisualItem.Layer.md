# MapVisualItem.Layer


## Синтаксис


Layer: Type;


## Описание


Свойство Layer определяет слой
 карты с [областями](../MapShape/MapShape.htm), к одной из которых
 привязан данный визуализатор.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLayer,
 а возвращается с помощью метода getLayer.
