# MapVisualItem.Shape


## Синтаксис


Shape: [PP.MapShape](../MapShape/MapShape.htm);


## Описание


Свойство Shape определяет область
 слоя карты, к которой привязан данный показатель.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setShape,
 а возвращается с помощью метода getShape.
