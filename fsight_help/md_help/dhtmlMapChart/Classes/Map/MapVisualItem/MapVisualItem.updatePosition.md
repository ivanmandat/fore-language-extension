# MapVisualItem.updatePosition


## Синтаксис


updatePosition(x: Number, y: Number);


## Параметры


x. Абсцисса точки. Необязательный
 параметр;


y. Ордината точки. Необязательный
 параметр.


## Описание


Метод updatePosition обновляет
 позицию показателя области слоя карты.


## Комментарии


Если параметры метода не указаны, то позиция показателя обновляется
 с использованием координат области слоя карты, к которой данный визуализатор
 привязан.
