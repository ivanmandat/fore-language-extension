# MapVisualItem.getBoundsRect


## Синтаксис


getBoundsRect();


## Описание


Метод getBoundsRect возвращает
 прямоугольную область, ограничивающую показатель области слоя карты.


## Комментарии


Метод возвращает значение типа [PP.Rect](dhtmlCommon.chm::/Classes/PP/Rect/Rect.htm).
