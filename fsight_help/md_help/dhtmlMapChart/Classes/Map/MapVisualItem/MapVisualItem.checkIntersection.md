# MapVisualItem.checkIntersection


## Синтаксис


checkIntersection(left: Number, top: Number, right:
 Number, bottom: Number, fullEntr: Boolean);


## Параметры


left. Абсцисса левой стороны
 прямоугольной области;


top. Ордината верхней стороны
 прямоугольной области;


right. Абсцисса правой стороны
 прямоугольной области;


bottom. Ордината нижней стороны
 прямоугольной области;


fullEntr. Признак проверки
 на полное вхождение указанной области в какой-либо из столбцов столбикового
 показателя. Допустимые значения:


	- true.
	 Используется проверка на полное вхождение указанной области в какой-либо
	 из столбцов столбикового показателя;


	- false.
	 Проверка на полное вхождение указанной области в какой-либо из столбцов
	 столбикового показателя не используется (по умолчанию).


## Описание


Метод checkIntersection проверяет,
 пересекает ли указанная область столбцы столбикового показателя.


## Комментарии


Метод возвращает идентификаторы главных сопоставлений данных визуализатора
 для тех столбцов показателя, которые пересекает указанная прямоугольная
 область.
