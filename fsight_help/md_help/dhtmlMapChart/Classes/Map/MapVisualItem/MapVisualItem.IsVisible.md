# MapVisualItem.IsVisible


## Синтаксис


IsVisible: Boolean;


## Описание


Свойство IsVisible определяет,
 отображается ли показатель на карте.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsVisible,
 а возвращается с помощью метода getIsVisible.


Допустимые значения:


	- true.
	 Показатель отображается на карте (по умолчанию);


	- false.
	 Показатель не отображается на карте.
