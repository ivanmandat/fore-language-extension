# MapContentMilestone.Interval


## Синтаксис


Interval: Number;


## Описание


Свойство Interval определяет интервал для значка карты.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setInterval, а возвращается - с помощью метода getInterval.
