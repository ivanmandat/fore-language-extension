# MapContentMilestone.AnimationCompleted


## Синтаксис


AnimationCompleted: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие AnimationCompleted генерируется после окончания воспроизведения анимации.


## Комментарии


Данное событие наступает после окончания обратного цикла воспроизведения анимации, если включена его поддержка, иначе - после первого.
