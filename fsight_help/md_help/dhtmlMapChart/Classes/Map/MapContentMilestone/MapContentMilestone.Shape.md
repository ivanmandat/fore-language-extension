# MapContentMilestone.Shape


## Синтаксис


Shape: PP.[MapShape](../MapShape/MapShape.htm)


## Описание


Свойство Shape устанавливает область карты, на которой будет размещаться значок.


## Комментарии


Значение свойства - идентификатор области карты, реализованной классом [MapShape](../MapShape/MapShape.htm).
