# MapContentMilestone.getScreen2DCoord


## Синтаксис


getScreen2DCoord(pos: [PP.Point](dhtmlCommon.chm::/Classes/PP/Point/Point.htm));


## Параметры


pos. Координаты точки относительно топоосновы карты.


## Описание


Метод getScreen2DCoord вовзращает экранные координаты точки по её координатам относительно топоосновы двумерной карты.


## Комментарии


Метод возвращает JSON-объект с полями x и y - экранные координаты точки по горизонтальной и вертикальной оси соответственно.
