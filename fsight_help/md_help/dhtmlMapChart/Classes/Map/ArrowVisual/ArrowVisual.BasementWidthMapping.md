# ArrowVisual.BasementWidthMapping


## Синтаксис


BasementWidthMapping: Array;


## Описание


Свойство BasementWidthMapping
 определяет настройки сопоставления данных ширине основания стрелки на
 карте.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setBasementWidthMapping,
 а возвращается с помощью метода getBasementWidthMapping.


Свойство содержит массив объектов типа [PP.DataMapping](dhtmlGraphicBase.chm::/Classes/PP/DataMapping/DataMapping.htm).
