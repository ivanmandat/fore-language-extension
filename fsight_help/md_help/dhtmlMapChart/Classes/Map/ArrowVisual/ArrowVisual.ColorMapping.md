# ArrowVisual.ColorMapping


## Синтаксис


ColorMapping: Array;


## Описание


Свойство ColorMapping определяет
 настройки сопоставления данных цвету заливки стрелки на карте.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setColorMapping,
 а возвращается с помощью метода getColorMapping.


Свойство содержит массив объектов типа [PP.DataMapping](dhtmlGraphicBase.chm::/Classes/PP/DataMapping/DataMapping.htm).
