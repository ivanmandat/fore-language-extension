# PieVisual.getMappingsCount


## Синтаксис


getMappingsCount();


## Описание


Метод getMappingsCount возвращает
 максимальное количество настроек сопоставления данных для кругового показателя.


## Комментарии


Метод возвращает значение типа Number.
