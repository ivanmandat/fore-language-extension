# PieVisual.RadiusMappings


## Синтаксис


HeightMappings: Array;


## Описание


Свойство RadiusMappings определяет
 настройки сопоставления данных радиусу секторов кругового показателя.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setRadiusMappings,
 а возвращается с помощью метода getRadiusMappings.


Свойство содержит массив объектов типа [PP.DataMapping](dhtmlGraphicBase.chm::/Classes/PP/DataMapping/DataMapping.htm).
