# PieVisual.BorderMappings


## Синтаксис


BorderMappings: Array;


## Описание


Свойство BorderMappings определяет
 настройки сопоставления данных свойствам границы у секторов кругового
 показателя.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setBorderMappings,
 а возвращается с помощью метода getBorderMappings.


Свойство содержит массив объектов типа [PP.DataMapping](dhtmlGraphicBase.chm::/Classes/PP/DataMapping/DataMapping.htm).
