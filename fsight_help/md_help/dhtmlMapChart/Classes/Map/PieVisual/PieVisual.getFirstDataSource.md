## Синтаксис

## Описание


Метод getFirstDataSource возвращает
 источник для первого сопоставления данных кругового показателя.


## Комментарии


Метод возвращает значение типа [PP.DataSource](dhtmlGraphicBase.chm::/Classes/PP/DataSource/DataSource.htm).
