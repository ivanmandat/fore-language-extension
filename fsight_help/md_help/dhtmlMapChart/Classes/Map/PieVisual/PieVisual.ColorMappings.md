# PieVisual.ColorMappings


## Синтаксис


ColorMappings: Array;


## Описание


Свойство ColorMappings определяет
 настройки сопоставления данных цвету заливки секторов кругового показателя.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setColorMappings,
 а возвращается с помощью метода getColorMappings.


Свойство содержит массив объектов типа [PP.DataMapping](dhtmlGraphicBase.chm::/Classes/PP/DataMapping/DataMapping.htm).
