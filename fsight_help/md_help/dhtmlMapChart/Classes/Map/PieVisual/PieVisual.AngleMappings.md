# PieVisual.AngleMappings


## Синтаксис


AngleMappings: Array;


## Описание


Свойство AngleMappings определяет
 настройки сопоставления данных углам секторов кругового показателя.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setAngleMappings,
 а возвращается с помощью метода getAngleMappings.


Свойство содержит массив объектов типа [PP.DataMapping](dhtmlGraphicBase.chm::/Classes/PP/DataMapping/DataMapping.htm).
