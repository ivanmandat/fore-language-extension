# MapChart.getFirstVisual


## Синтаксис


getFirstVisual (typeVisual);


## Параметры


typeVisual. Тип показателя
 карты. Необязательный параметр.


## Описание


Метод getFirstVisual возвращает
 первый показатель карты.


## Комментарии


Как правило, первый показатель карты - это [показатель
 заливки карты](../../../Components/MapChart/xml_specification/MapChart/MapLayers/MapLayer/Visuals/MapAreaVisual_element.htm).
