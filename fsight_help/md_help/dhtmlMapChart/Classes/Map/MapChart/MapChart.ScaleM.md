# MapChart.scale


## Синтаксис


scale(dir);


## Параметры


dir. Number. Направление изменения
 масштаба. При значении больше 0 - масштаб будет увеличен на 10%, при значении
 меньше нуля - уменьшен на 10%.


## Описание


Метод scale увеличивает текущий
 масштаб.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 [MapChart](../../../Components/MapChart/MapChart.htm) с наименованием
 «map» (см. «[Пример
 создания компонента MapChart](../../../Components/MapChart/MapChart_Example.htm)»).


// Увеличим текущий масштаб карты
map.scale(1);
В результате выполнения примера масштаб карты был увеличен на 10%.
