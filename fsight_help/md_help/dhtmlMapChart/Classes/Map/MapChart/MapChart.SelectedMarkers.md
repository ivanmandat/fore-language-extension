# MapChart.SelectedMarkers


## Синтаксис


SelectedMarkers: Object;


## Описание


Свойство SelectedMarkers определяет
 массив выделенных маркеров.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSelectedMarkers,
 а возвращается с помощью метода getSelectedMarkers.


Свойство содержит объект формата: {<[marker_id](../MapMarker/MapMarker.htm)>: true}.


## Пример


Использование свойства аналогично свойству [SelectedBars](MapChart.HoveredBars.htm),
 вместо идентификатора столбика используйте идентификатор пузырькового
 показателя.
