# Конструктор MapChart


## Синтаксис


PP.MapChart(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор MapChart создает экземпляр компонента [MapChart](../../../Components/MapChart/MapChart.htm).
