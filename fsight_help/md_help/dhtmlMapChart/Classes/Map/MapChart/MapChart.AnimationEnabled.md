# MapChart.AnimationEnabled


## Синтаксис


AnimationEnabled: Boolean;


## Описание


Свойство AnimationEnabled управляет разрешением на воспроизведение анимации в карте.


## Комментарии


Значение свойства задаётся из JSON и с помощью метода setAnimationEnabled. По умолчанию свойство будет содержать значение true, если загружена SVG-карта, иначе - значение false.


При равенстве свойства значению true анимация в карте будет воспроизводиться, иначе - не будет.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента [MapChart](../../../Components/MapChart/MapChart.htm) с наименованием «map» (см. «[Пример создания компонента MapChart](../../../Components/MapChart/MapChart_Example.htm)»). Запретим воспроизведение анимации в карте, выполнив в консоли браузера следующую строку кода:


map.setAnimationEnabled(false);

После выполнения примера воспроизведение анимации в карте будет отключено. Чтобы убедиться в этом, нажмём на кнопку воспроизведения анимации временной шкалы. В результате карта будет перерисовываться без плавных переходов.
