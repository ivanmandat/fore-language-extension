# MapChart.SelectedBubbles


## Синтаксис


SelectedBubbles: Object;


## Описание


Свойство SelectedBubbles определяет
 массив выделенных пузырьковых показателей.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSelectedBubbles,
 а возвращается с помощью метода getSelectedBubbles.


Свойство содержит объект формата: {<[bubble_id](../MapBubble/MapBubble.htm)>: true}.


## Пример


Использование свойства аналогично свойству [SelectedBars](MapChart.HoveredBars.htm),
 вместо идентификатора столбика используйте идентификатор пузырькового
 показателя.
