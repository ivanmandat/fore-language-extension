# MapChart.SelectedPies


## Синтаксис


SelectedPies: Object;


## Описание


Свойство SelectedPies определяет
 массив выделенных круговых показателей.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSelectedPies,
 а возвращается с помощью метода getSelectedPies.


Свойство содержит объект формата: {<[pie_id](../MapPie/MapPie.htm)>:
 {<mapping_id>: true}}.


## Пример


Использование свойства аналогично свойству [SelectedBars](MapChart.HoveredBars.htm),
 вместо идентификатора столбика используйте идентификатор кругового показателя.
