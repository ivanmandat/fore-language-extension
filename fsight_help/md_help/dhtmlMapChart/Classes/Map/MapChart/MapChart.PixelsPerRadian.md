# MapChart.PixelsPerRadian


## Синтаксис


PixelsPerRadian: Object;


## Описание


Свойство PixelsPerRadian определяет
 количество радиан, соответствующих одному пикселю карты по горизонтали
 и вертикали.


## Комментарии


Значение свойства устанавливается из JSON и с помощью свойства setPixelsPerRadian.


Свойство содержит JSON-объект с полями x и y - количество радиан в одном
 пикселе по горизонтали и вертикали соответственно.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 [MapChart](../../../Components/MapChart/MapChart.htm) с наименованием
 «map» и компонента [ToolBar](dhtmlUi.chm::/Classes/ToolBar/ToolBar.htm)
 с наименованием «MapType» (см. «[Пример создания
 компонента MapChart](../../../Components/MapChart/MapChart_Example.htm)»). Также требуется, чтобы была
 загружена карта с типом отображения «Sphere» (см. страницу описания свойства
 [MapChart.AmbientLightValue](MapChart.AmbientLightValue.htm)).


Определим количество радиан, соответствующих одному пикселю карты по
 горизонтали и вертикали:


// Определим, сколько радиан соответствует одному пикселу
var pixelsPerRadian = map.getPixelsPerRadian();
console.log("Количество радиан в одном пикселе по горизонтали: " + pixelsPerRadian.x);
console.log("Количество радиан в одном пикселе по вертикали: " + pixelsPerRadian.y);

В результате выполнения примера в консоли браузера было выведено количество
 радиан, соответствующих одному пикселю карты по горизонтали и вертикали:


Количество радиан в одном пикселе по горизонтали:
 0.0063617626240385


Количество радиан в одном пикселе по вертикали:
 0.0038008917619369
