# MapChart.TopobaseHorizontalAlignment


## Синтаксис


TopobaseHorizontalAlignment: [PP.HorizontalAlignment](dhtmlCommon.chm::/Enums/PP.HorizontalAlignment.htm)


## Описание


Свойство TopobaseHorizontalAlignment
 устанавливает тип выравнивания по горизонтали.


## Комментарии


По умолчанию установлено значение «Center».


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 [MapChart](../../../Components/MapChart/MapChart.htm) с наименованием
 «map» (см. «[Пример
 создания компонента MapChart](../../../Components/MapChart/MapChart_Example.htm)»), [xml-файла](../../../Components/MapChart/xml_specification/xml-specification.htm) с
 настройками карты и топоосновы Russia.svg.


map.setTopobaseHorizontalAlignment("Left");


После выполнения примера карта будет расположена в левой части страницы.
