# MapChart.SelectedArrows


## Синтаксис


SelectedArrows: Object;


## Описание


Свойство SelectedArrows определяет
 массив выделенных стрелок.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSelectedArrows,
 а возвращается с помощью метода getSelectedArrows.


Свойство содержит объект формата: {<[arrow_id](../ArrowVisual/ArrowVisual.htm)>: true}.


## Пример


Использование свойства аналогично свойству [SelectedBars](MapChart.HoveredBars.htm),
 вместо идентификатора столбика используйте идентификатор стрелки.
