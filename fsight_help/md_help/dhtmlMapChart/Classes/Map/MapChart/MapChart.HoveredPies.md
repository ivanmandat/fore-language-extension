# MapChart.HoveredPies


## Синтаксис


HoveredPies: Object;


## Описание


Свойство HoveredPies определяет
 массив круговых показателей, для которых установлено наведение.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHoveredPies
 , а возвращается с помощью метода getHoveredPies
 .


Свойство содержит объект формата: {<[pie_id](../MapPie/MapPie.htm)>: {<mapping_id>: true}}.


## Пример


Использование свойства аналогично свойству [HoveredBars](MapChart.HoveredBars.htm),
 вместо идентификатора столбика используйте идентификатор кругового показателя.
