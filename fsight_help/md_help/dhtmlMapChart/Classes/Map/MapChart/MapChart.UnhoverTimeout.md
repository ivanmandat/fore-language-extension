# MapChart.UnhoverTimeout


## Синтаксис


UnhoverTimeout: Number;


## Описание


Свойство UnhoverTimeout определяет
 временной период подсвечивания элементов после отмены наведения.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setUnhoverTimeout,
 а возвращается с помощью метода getUnhoverTimeout.


Значение указывается в миллисекундах, по умолчанию содержит значение
 «300».


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 [MapChart](../../../Components/MapChart/MapChart.htm) с наименованием
 «map» (см. «[Пример
 создания компонента MapChart](../../../Components/MapChart/MapChart_Example.htm)»). Увеличим время подсвечивания элементов:


map.setUnhoverTimeout(1000);
В результате выполнения примера после наведения курсора мыши и отведении
 его в сторону элемент будет подсвечен в течение одной секунды.
