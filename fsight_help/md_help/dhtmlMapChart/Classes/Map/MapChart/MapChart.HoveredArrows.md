# MapChart.HoveredArrows


## Синтаксис


HoveredArrows: Object;


## Описание


Свойство HoveredArrows определяет массив
 стрелок, для которых установлено наведение.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHoveredArrows,
 а возвращается с помощью метода getHoveredArrows.


Свойство содержит объект формата: {<[arrow_id](../ArrowVisual/ArrowVisual.htm)>: true}.


## Пример


Использование свойства аналогично свойству [HoveredBars](MapChart.HoveredBars.htm),
 вместо идентификатора столбика используйте идентификатор стрелки.
