# MapChart.getTopobase


## Синтаксис


getTopobase ();


## Описание


Метод getTopobase возвращает топооснову карты.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента [MapChart](../../../Components/MapChart/MapChart.htm) с наименованием «map» (см. «[Пример создания компонента MapChart](../../../Components/MapChart/MapChart_Example.htm)»).


map.getTopobase().setId("Russia");


После выполнения примера для текущей топоосновы карты будет задан идентификатор «Russia».
