# MapChart.MinScale


## Синтаксис


MinScale: Number;


## Описание


Свойство MinScale определяет минимально допустимый масштаб карты.


## Комментарии


Значение свойства можно установить из JSON и с помощью свойства setMinScale.


По умолчанию свойство содержит значение 0,1.
