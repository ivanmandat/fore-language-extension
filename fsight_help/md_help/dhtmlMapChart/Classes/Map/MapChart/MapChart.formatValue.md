# MapChart.formatValue


## Синтаксис


formatValue(value: Number|String, formatString: String);


## Параметры


value. Исходное значение;


formatString. Формат, в соответствии с которым требуется отформатировать исходное значение.


## Описание


Метод formatValue форматирует значение в соответствии с указанным форматом.


## Комментарии


Метод возвращает значение типа String.
