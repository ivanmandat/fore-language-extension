# MapChart.clearPaper


## Синтаксис


clearPaper();


## Описание


Метод clearPaper очищает карту.


## Комментарии


Данный метод работает только для карты с топоосновой SVG.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента [MapChart](../../../Components/MapChart/MapChart.htm) с наименованием «map» (см. «[Пример создания компонента MapChart](../../../Components/MapChart/MapChart_Example.htm)»). Удалим все области слоя карты с топоосновой SVG:


map.clearPaper();

В результате выполнения примера были удалены все области слоя карты:
