# MapChart.TopobaseVerticalAlignment


## Синтаксис


TopobaseVerticalAlignment: [PP.VerticalAlignment](dhtmlCommon.chm::/Enums/PP.VerticalAlignment.htm)


## Описание


Свойство TopobaseVerticalAlignment
 устанавливает тип выравнивания по вертикали.


## Комментарии


По умолчанию установлено значение «Center».


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 [MapChart](../../../Components/MapChart/MapChart.htm) с наименованием
 «map» (см. «[Пример
 создания компонента MapChart](../../../Components/MapChart/MapChart_Example.htm)»), [xml-файла](../../../Components/MapChart/xml_specification/xml-specification.htm) с
 настройками карты и топоосновы Russia.svg.


map.setTopobaseVerticalAlignment("Bottom");


После выполнения примера карта будет расположена в нижней части страницы.
