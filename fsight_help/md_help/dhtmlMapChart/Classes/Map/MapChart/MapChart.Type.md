# MapChart.Type


## Синтаксис


Type: [PP.MapChartType](../../../Enums/MapChartType.htm);


## Описание


Свойство Type устанавливает технологию отрисовки карты.


## Комментарии


Значения свойства - элементы перечисления [PP.MapChartType](../../../Enums/MapChartType.htm).


По умолчанию установлено значение SVG.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента [MapChart](../../../Components/MapChart/MapChart.htm) с наименованием «map» (см. «[Пример создания компонента MapChart](../../../Components/MapChart/MapChart_Example.htm)»).
