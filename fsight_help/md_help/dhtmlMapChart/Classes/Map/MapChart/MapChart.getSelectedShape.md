# MapChart.getSelectedShape


## Синтаксис


getSelectedShape(x: Number, y: Number);


## Параметры


x. Абсцисса точки;


y. Ордината точки.


## Описание


Метод getSelectedShape вовзращает
 область слоя карты по координатам принадлежащей ей точки или область,
 над которой был проведён указатель мыши в последний раз.


## Комментарии


Метод возвращает значение типа [PP.MapShape](../MapShape/MapShape.htm).


Если при вызове метода не указаны значения обоих параметров, то метод
 возвращает область слоя карты, над которой был проведён указатель мыши
 в последний раз. Иначе метод возвратит область слоя карты по координатам
 принадлежащей ей точки.
