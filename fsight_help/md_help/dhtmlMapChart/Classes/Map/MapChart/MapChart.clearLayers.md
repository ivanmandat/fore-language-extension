# MapChart.clearLayers


## Синтаксис


clearLayers();


## Описание


Метод clearLayers очищает все слои в карте.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента [MapChart](../../../Components/MapChart/MapChart.htm) с наименованием «map» (см. «[Пример создания компонента MapChart](../../../Components/MapChart/MapChart_Example.htm)»). Также требуется, чтобы была загружена карта с топоосновой Google (см. страницу описания свойства [MapChart.afterGoogleLoad](MapChart.afterGoogleLoad.htm)).


Очистим все слои в карте с топоосновой Google:


map.clearLayers();

В результате выполнения примера были очищены все слои в карте с топоосновой Google:
