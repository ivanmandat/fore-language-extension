# MapChart.MaxScale


## Синтаксис


MaxScale: Number;


## Описание


Свойство MaxScale определяет максимально допустимый масштаб карты.


## Комментарии


Значение свойства можно установить из JSON и с помощью свойства setMaxScale.


По умолчанию свойство содержит значение 9000.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента [MapChart](../../../Components/MapChart/MapChart.htm) с наименованием «map» (см. «[Пример создания компонента MapChart](../../../Components/MapChart/MapChart_Example.htm)»). Установим максимальный и минимальный допустимый масштаб карты:


// Установим максимальный масштаб
map.setMaxScale(1.5);
// Установим минимальный масштаб
map.setMinScale(0.7);

В результате выполнения примера масштаб карты можно увеличивать только до 150% и уменьшать до 70%.
