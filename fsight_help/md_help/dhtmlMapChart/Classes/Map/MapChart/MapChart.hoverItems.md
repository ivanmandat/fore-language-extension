# MapChart.hoverItems


## Синтаксис


hoverItems(items: Object, timelineStep: Number);


## Параметры


items. Элементы, которые необходимо
 подсветить;


timelineStep. Шаг временной
 шкалы.


## Описание


Метод hoverItems устанавливает
 подсветку указанных элементов.
