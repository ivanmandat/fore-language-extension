# MapChart.MapMarkers


## Синтаксис


MapMarkers: Object;


## Описание


Свойство MapMarkers определяет
 массив маркеров карты.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMapMarkers,
 а возвращается с помощью метода getMapMarkers.


Свойство содержит объект формата: {<marker_id>:
 [PP.Ui.MapMarker](../MapMarker/MapMarker.htm)}.
