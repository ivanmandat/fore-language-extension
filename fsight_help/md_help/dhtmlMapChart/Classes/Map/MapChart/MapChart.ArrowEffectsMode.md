# MapChart.ArrowEffectsMode


## Синтаксис


ArrowEffectsMode: [PP.Ui.MapChart.ArrowEffectsMode](../../../Enums/MapChart.ArrowEffectsMode.htm);


## Описание


Свойство ArrowEffectsMode определяет
 вариант подсветки стрелок карты.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setArrowEffectsMode,
 а возвращается с помощью метода getArrowEffectsMode.
