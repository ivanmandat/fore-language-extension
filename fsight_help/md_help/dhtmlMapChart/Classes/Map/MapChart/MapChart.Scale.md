# MapChart.Scale


## Синтаксис


Scale: Double


## Описание


Свойство Scale устанавливает масштаб для карты.


## Комментарии


По умолчанию установлено значение 1.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента [MapChart](../../../Components/MapChart/MapChart.htm) с наименованием «map» (см. «[Пример создания компонента MapChart](../../../Components/MapChart/MapChart_Example.htm)»), [xml-файла](../../../Components/MapChart/xml_specification/xml-specification.htm) с настройками карты и топоосновы Russia.svg.


map.setScale(0.5);


После выполнения примера масштаб карты будет изменен: карта уменьшится вдвое.
