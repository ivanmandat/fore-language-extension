# MapChart.sceneToScreenIfVisible


## Синтаксис


sceneToScreenIfVisible(point: PP.Vector3);


## Параметры


point. Точка, координаты которой требуется определить.


## Описание


Метод sceneToScreen возвращает оконные координаты точки, соответствующие мировым, при условии, если она является видимой.


## Комментарии


Метод возвращает значение типа PP.Vector3.
