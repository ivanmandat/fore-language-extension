# MapChart.getTopobaseLeft


## Синтаксис


getTopobaseLeft();


## Описание


Метод getTopobaseLeft возвращает отступ топоосновы от левой границы карты в пикселях.


## Комментарии


Метод возвращает значение типа Number.
