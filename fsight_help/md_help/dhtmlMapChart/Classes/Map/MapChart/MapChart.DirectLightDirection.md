# MapChart.DirectLightDirection


## Синтаксис


DirectLightDirection: PP.Vector3;


## Описание


Свойство DirectLightDirection определяет направление внешнего освещения.


## Комментарии


Значение свойства задаётся из JSON и с помощью метода setAmbientLightValue.


По умолчанию свойство содержит значение типа PP.Vector3, созданного с указанием параметров 0,45, -0.85, -1.


Данное свойство применимо только для карты с топоосновой Sphere.
