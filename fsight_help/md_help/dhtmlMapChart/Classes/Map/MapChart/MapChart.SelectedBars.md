# MapChart.SelectedBars


## Синтаксис


SelectedBars: Object;


## Описание


Свойство SelectedBars определяет
 массив выделенных столбиковых показателей.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSelectedBars,
 а возвращается с помощью метода getSelectedBars.


Свойство содержит объект формата: {<[bar_id](../BarVisual/BarVisual.htm)>:
 {<mapping_id>: true}}.


## Пример


Использование свойства приведено в описании свойства [HoveredBars](MapChart.HoveredBars.htm).
