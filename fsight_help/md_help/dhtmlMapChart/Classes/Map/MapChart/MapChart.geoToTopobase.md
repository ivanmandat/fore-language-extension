# MapChart.geoToTopobase


## Синтаксис


geoToTopobase(lng: number; lat: number);


## Параметры


lng. Координата долготы.


lat. Координата широты.


## Описание


Метод geoToTopobase преобразует
 географические координаты в координаты топоосновы.


## Комментарии


Метод возвращает значение типа PP.Point.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 [MapChart](../../../Components/MapChart/MapChart.htm) с наименованием
 «map» (см. «[Пример
 создания компонента MapChart](../../../Components/MapChart/MapChart_Example.htm)»). Преобразуем географические координаты
 в координаты топоосновы карты:


// Зададим географические координаты
lng = 75.380045583183;
lat = 35.6136504260887;
// Преобразуем координаты и выведем результат в консоль
coord = map.geoToTopobase(lng, lat);
console.debug(coord);
// -> PP.Point {_X: 327.2021399482517, _Y: 627.9895094574584, _Id: "Point6454"}
В результате выполнения примера в консоли браузера были выведены координаты
 топоосновы карты.
