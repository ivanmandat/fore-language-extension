# MapChart.getRegionName


## Синтаксис


getRegionName (id);


## Параметры


id. Идентификатор региона.


## Описание


Метод getRegionName возвращает имя региона по его идентификатору.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента [MapChart](../../../Components/MapChart/MapChart.htm) с наименованием «map» (см. «[Пример создания компонента MapChart](../../../Components/MapChart/MapChart_Example.htm)»).


map.getRegionName("RU-LEN");


После выполнения примера будет возвращено наименование области с идентификатором «RU-LEN». Наименование области содержится в [словаре имен областей](../../../Components/MapChart/xml_specification/MapChart/RegionsNames/RegionsNames_exmpl.htm).
