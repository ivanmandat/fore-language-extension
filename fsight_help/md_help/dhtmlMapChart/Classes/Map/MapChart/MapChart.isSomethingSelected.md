# MapChart.isSomethingSelected


## Синтаксис


isSomethingSelected();


## Описание


Метод isSomethingSelected определяет,
 содержатся ли на карте выделенные элементы.


## Комментарии


Метод возвращает значение true,
 если карта содержит выделенные элементы, иначе - значение false.
