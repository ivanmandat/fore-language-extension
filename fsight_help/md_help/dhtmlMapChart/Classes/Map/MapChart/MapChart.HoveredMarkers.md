# MapChart.HoveredMarkers


## Синтаксис


HoveredMarkers: Object;


## Описание


Свойство HoveredMarkers определяет массив
 маркеров, для которых установлено наведение.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHoveredMarkers,
 а возвращается с помощью метода getHoveredMarkers.


Свойство содержит объект формата: {<[marker_id](../MapMarker/MapMarker.htm)>: true}.


## Пример


Использование свойства аналогично свойству [HoveredBars](MapChart.HoveredBars.htm),
 вместо идентификатора столбика используйте идентификатор пузырькового
 показателя.
