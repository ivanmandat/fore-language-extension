# MapChart.activateSpectrumLegend


## Синтаксис


activateSpectrumLegend(doActivate: Boolean);


## Параметры


doActivate. Признак отображения спектральной легенды. Если параметр равен значению true, то спектральная диаграмма будет отображена на карте, иначе - не будет.


## Описание


Метод activateSpectrumLegend управляет отображением спектральной легенды на карте.
