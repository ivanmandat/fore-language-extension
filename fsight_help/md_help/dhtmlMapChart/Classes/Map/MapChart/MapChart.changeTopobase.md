# MapChart.changeTopobase


## Синтаксис


changeTopobase(topobaseText);


## Параметры


topobaseText. String. Новая
 топооснова. Содержание строки аналогично содержанию файла *.svg.


## Описание


Метод changeTopobase меняет
 топооснову.

Метод предназначен для внутреннего использования.
