# MapPolygonMilestone.Points


## Синтаксис


Points: Array


## Описание


Свойство Points устанавливает коллекцию точек по которой строится многоугольник.


## Комментарии


При перечислении координат точек после запятой не должно быть пробелов, например, "Points”: "10,12 15,15 10,19".
