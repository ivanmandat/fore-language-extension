# MapGeoItem.Longitude


## Синтаксис


Longitude: Number;


## Описание


Свойство Longitude определяет географическую долготу карты.


## Комментарии


Значение свойства устанавливается из JSON и возвращается с помощью метода getLongitude. Задать значение свойства с помощью метода setLongitude нельзя.
