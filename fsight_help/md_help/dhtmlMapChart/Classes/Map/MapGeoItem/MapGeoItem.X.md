# MapGeoItem.X


## Синтаксис


X: Number;


## Описание


Свойство X определяет координату X топоосновы карты.


## Комментарии


Значение свойства устанавливается из JSON и возвращается с помощью метода getX. Задать значение свойства с помощью метода setX нельзя.
