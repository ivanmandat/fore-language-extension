# Конструктор MapGeoItem


## Синтаксис


PP.MapGeoItem(settings: Object);


## Параметры


settings. JSON-объект со значениями свойств класса.


## Описание


Конструктор MapGeoItem создает экземпляр класса [MapGeoItem](MapGeoItem.htm).
