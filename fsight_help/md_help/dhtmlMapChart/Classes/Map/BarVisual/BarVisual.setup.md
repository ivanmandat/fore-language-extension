# BarVisual.setup


## Синтаксис


setup(index: Number);


## Параметры


index. Отметка календарного
 измерения, данные для которой требуется установить.


## Описание


Метод setup устанавливает настройки
 сопоставления данных для указанной отметки календарного измерения.
