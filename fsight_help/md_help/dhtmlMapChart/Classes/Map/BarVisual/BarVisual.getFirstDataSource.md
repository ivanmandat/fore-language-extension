# BarVisual.getFirstDataSource


## Синтаксис


getFirstDataSource();


## Описание


Метод getFirstDataSource возвращает
 источник для первого сопоставления данных столбикового показателя.


## Комментарии


Метод возвращает значение типа [PP.DataSource](dhtmlGraphicBase.chm::/Classes/PP/DataSource/DataSource.htm).
