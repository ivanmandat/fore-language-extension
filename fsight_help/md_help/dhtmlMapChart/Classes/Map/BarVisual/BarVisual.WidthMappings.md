# BarVisual.WidthMappings


## Синтаксис


WidthMappings: Array;


## Описание


Свойство WidthMappings определяет
 настройки сопоставления данных ширине элементов столбикового показателя.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setWidthMappings,
 а возвращается с помощью метода getWidthMappings.


Свойство содержит массив объектов типа [PP.DataMapping](dhtmlGraphicBase.chm::/Classes/PP/DataMapping/DataMapping.htm).
