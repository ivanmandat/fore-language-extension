# BarVisual.getMappingsCount


## Синтаксис


getMappingsCount();


## Описание


Метод getMappingsCount возвращает
 максимальное количество настроек сопоставления данных для столбикового
 показателя.


## Комментарии


Метод возвращает значение типа Number.
