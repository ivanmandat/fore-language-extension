# MapMaster.show3DPanel


## Синтаксис


show3DPanel();


## Описание


Метод show3DPanel** отображает [панель «Индикатор объема»](../Visual3DPanel/MapMaster.Visual3DPanel.htm) [мастера карты](../../Components/MapMaster/MapMaster.htm).


## Комментарии


Чтобы скрыть панель «Индикатор объема», используйте метод [MapMaster.hide3DPanel](MapMaster.hide3DPanel.htm).

Пример использования приведен в описании метода [MapMaster.hide3DPanel](MapMaster.hide3DPanel.htm).
