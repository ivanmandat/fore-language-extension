# AreaVisualPanel.getColorArray


## Синтаксис


getColorArray(firstColor, lastColor, count);


## Параметры


*firstColor.* Начальный цвет;


*lastColor.* Конечный цвет;


*count.* Число желаемых элементов в массиве.


## Описание


Метод getColorArray** возвращает массив случайных цветов в диапазоне.


## Комментарии


null
