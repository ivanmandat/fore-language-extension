# AreaVisualPanel.joinColor


## Синтаксис


joinColor(colorObj, *includeAlpha*);


## Параметры


*colorObj.* Объект цветов со свойствами (a) r g b ;


*includeAlpha.* Определяет, используется ли при здании цвета альфа-канал, отвечающий за прозрачность: true - используется, false - не используется.


## Описание


Метод joinColor** цвет и переводящий его в 16ую СС.


## Комментарии


null
