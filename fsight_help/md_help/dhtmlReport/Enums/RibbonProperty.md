# RibbonProperty


Пространство имен: PP.Prx.Ui


## Описание


Перечисление RibbonProperty содержит свойства ленты инструментов регламентного отчета.


Используется следующим методом [RibbonView.setPropertyValue](../Classes/RegularReport/RibbonView/RibbonView.setPropertyValue.htm).


## Допустимые значения


Значение


Краткое описание


DataSaveEnabled


Определяет активность кнопок в группе «Данные». Если установлено значение true, кнопки активны.


ControlBarSwitchEnabled


Определяет активность кнопки «Элементы управления». Если установлено значение false, кнопка неактивна. По умолчанию установлено значение true, то есть кнопка активна.


ClosedState


Определяет активность кнопок ленты и команд главного меню. По умолчанию установлено значение false, то есть все кнопки и пункты главного меню активны. Если установлено значение true, неактивны все кнопки, кроме кнопок группы «Данные» (они становятся активными, если есть измененные данные в таблице) и все пункты главного меню, кроме «Открыть», «Поделиться» и «Выход».


ControlBarVisible


Определяет видимость панели элементов управления. Если установлено значение true, панель отображается. Если установлено значение false, панель скрыта.
