# DataSaveDlgMode


Пространство имен: PP.Prx.Ui


## Описание


Перечисление DataSaveDlgMode содержит режимы отображения диалога сохранения данных регламентного отчета.


Используется методом [RibbonView.showDataSaveDialog](../Classes/RegularReport/RibbonView/RibbonView.showDataSaveDialog.htm).


## Допустимые значения


Значение


Краткое описание


Full


Диалог содержит кнопки «Да», «Нет», «Отмена». Текст диалога: «Сохранить измененные данные в источник перед выполнением операции?». Режим используется по умолчанию.


NoCancel


Диалог содержит кнопки «Да», «Нет». Текст диалога: «Сохранить измененные данные в источник перед выполнением операции?».


Undo


Диалог содержит кнопки «Да», «Нет». Текст диалога: «Отчет содержит измененные данные. Вы действительно хотите отменить изменения?».
