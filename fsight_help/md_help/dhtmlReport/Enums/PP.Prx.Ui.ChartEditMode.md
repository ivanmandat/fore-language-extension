# ChartEditMode


Пространство имен: PP.Prx.Ui


## Описание


Перечисление ChartEditMode содержит режимы редактирования диаграммы регламентного отчета.


Используется свойством [PrxChartView.EditMode](../Classes/RegularReport/PrxChartView/PrxChartView.EditMode.htm).


## Допустимые значения


Значение


Краткое описание


None


Редактирование отключено.


Point


Редактирование по точкам.


Draw


Редактирование рисованием.
