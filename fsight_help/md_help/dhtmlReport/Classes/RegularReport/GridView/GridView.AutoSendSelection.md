# GridView.AutoSendSelection


## Синтаксис


AutoSendSelection: Boolean


## Описание


Свойство AutoSendSelection определяет, будет ли отправляться на сервер состояние выделения ячеек в таблице регламентного отчета.


## Комментарии


По умолчанию установлено значение true, то есть состояние выделения ячеек таблицы регламентного отчета отправляется на сервер.


Значение свойства устанавливается из конструктора и при помощи метода setAutoSendSelection, а возвращается при помощи метода getAutoSendSelection.
