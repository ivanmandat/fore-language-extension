# GridView.setIsEditable


## Синтаксис


setIsEditable (editMode);


## Параметры


editMode. Определяет, являются ли ячейки таблицы редактируемыми. Если установлено значение true, ячейки являются редактируемыми. Если установлено значение false, редактирование данных в ячейках невозможно.


## Описание


Метод setIsEditable определяет возможность редактирования данных в ячейках таблицы.
