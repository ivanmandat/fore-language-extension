# GridView.refresh


## Синтаксис


refresh ();


## Описание


Метод refresh обновляет содержимое таблицы регламентного отчета.


## Комментарии


Метод аналогичен методу [GridView.refreshAll](GridView.refreshAll.htm).


## Пример


Использование метода аналогично использованию метода [ReportView.refreshAll](../ReportView/ReportView.refreshAll.htm).
