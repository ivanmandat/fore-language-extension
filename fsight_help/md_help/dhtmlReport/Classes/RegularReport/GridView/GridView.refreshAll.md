# GridView.refreshAll


## Синтаксис


refreshAll ();


## Описание


Метод refreshAll обновляет содержимое таблицы регламентного отчета.


## Комментарии


Метод аналогичен методу [GridView.refresh](GridView.refresh.htm).


## Пример


Использование метода аналогично использованию метода [ReportView.refreshAll](../ReportView/ReportView.refreshAll.htm).
