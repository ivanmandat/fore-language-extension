# GridView.getTabObject


## Синтаксис


getTabObject(id);


## Параметры


id. Идентификатор пользовательского
 объекта.


## Описание


Метод getTabObject возвращает
 пользовательский объект по идентификатору.


## Пример


Метод возвращает пользовательский объект, аналогично методу [TabSheetObject.getITabObject()](dhtmlTabSheet.chm::/Classes/TabSheet/TabSheetObject/TabSheetObject.getITabObject.htm).
