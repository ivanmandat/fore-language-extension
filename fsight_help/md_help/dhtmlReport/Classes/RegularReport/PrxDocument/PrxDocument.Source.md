# PrxDocument.Source


## Синтаксис


Source: PP.Prx.[Report](../Report/Report.htm)


## Описание


Свойство Source устанавливает источник для представления текстового листа.


## Комментарии


Свойство устанавливается из JSON или при помощи метода setSource и возвращается при помощи метода getSource.

Пример использования приведен в описании метода [PrxMdService.getDocumentData](../PrxMdService/PrxMdService.getDocumentData.htm).
