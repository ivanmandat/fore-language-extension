# PrxDocument.Service


## Синтаксис


Service: PP.Prx.[PrxMdService](../PrxMdService/PrxMdService.htm)


## Описание


Свойство Service устанавливает сервис для работы с регламентными отчетами для представления текстового листа.


## Комментарии


Свойство устанавливается из JSON или при помощи метода setService и возвращается при помощи метода getService.

Пример использования приведен в описании метода [PrxMdService.getDocumentData](../PrxMdService/PrxMdService.getDocumentData.htm).
