# PrxDocument.refreshAll


## Синтаксис


refreshAll ();


## Описание


Метод refreshAll обновляет весь текстовый лист регламентного отчета.


## Комментарии


Работа метода refreshAll аналогична работе метода [PrxDocument.refresh](PrxDocument.refresh.htm), для которого не задан параметр или задан параметр Recalc.


## Пример


Использование метода аналогично использованию, приведенному в примере для [PrxDocument.refresh](PrxDocument.refresh.htm).
