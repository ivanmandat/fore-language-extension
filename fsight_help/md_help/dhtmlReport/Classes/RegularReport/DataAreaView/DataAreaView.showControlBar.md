# DataAreaView.showControlBar


## Синтаксис


showControlBar (showCtrlBar, updateSize);


## Параметры


showCtrlBar. Определяет, отображается ли панель элементов управления. Если установлено значение true, панель отображается, если false - панель скрыта;


updateSize. Определяет, будет ли изменяться размер таблицы при скрытии/отображении панели элементов. Если установлено значение true, при скрытии/отображении панели изменяется размер таблицы (таблица занимает все пространство области данных). Если установлено значение false, размер таблицы не изменяется. По умолчанию установлено значение true.


## Описание


Метод showControlBar определяет, отображается ли панель элементов управления регламентного отчета.
