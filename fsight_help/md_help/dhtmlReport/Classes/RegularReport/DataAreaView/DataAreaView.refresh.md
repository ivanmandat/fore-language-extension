# DataAreaView.refresh


## Синтаксис


refresh ();


## Описание


Метод refresh обновляет область данных в соответствии с произведенными изменениями.


## Пример


Использование метода аналогично использованию метода [DataAreaView.refreshAll](DataAreaView.refreshAll.htm).
