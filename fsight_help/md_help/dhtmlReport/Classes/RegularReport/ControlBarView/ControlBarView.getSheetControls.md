# ControlBarView.getSheetControls


## Синтаксис


getSheetControls (sheetKey);


## Параметры


sheetKey. Ключ листа отчета, для которого необходимо вернуть элементы управления.


## Описание


Метод getSheetControls возвращает массив элементов управления для указанного листа отчета.
