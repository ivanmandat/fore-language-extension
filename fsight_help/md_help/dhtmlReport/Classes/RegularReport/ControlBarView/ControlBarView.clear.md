# ControlBarView.clear


## Синтаксис


clear (isDispose);


## Параметры


isDispose. Определяет, будут ли удаляемые элементы уничтожены: true- будут, false - не будут.


## Описание


Метод clear очищает панель элементов регламентного отчета.
