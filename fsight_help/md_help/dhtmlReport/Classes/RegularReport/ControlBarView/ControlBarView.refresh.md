# ControlBarView.refresh


## Синтаксис


refresh(args);


## Параметры


args. Определяет, что обновляться будут только значения элементов управления. Если параметр не задан, будет обновляться вся панель (как при использовании метода [ControlBarView.refreshAll](ControlBarView.refreshAll.htm)).


## Описание


Метод refresh используется для выборочного обновления панели элементов управления.


## Комментарии


Если параметр не задан, значит обновляется вся панель (как при использовании метода [ControlBarView.refreshAll](ControlBarView.refreshAll.htm)).
