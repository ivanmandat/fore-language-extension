# ControlBarView.getControl


## Синтаксис


getControl (pos);


## Параметры


pos. Позиция элемента управления.


## Описание


Метод getControl возвращает элемент управления по его позиции.


## Комментарии


Элементы управления реализованы наследниками класса ControlView.
