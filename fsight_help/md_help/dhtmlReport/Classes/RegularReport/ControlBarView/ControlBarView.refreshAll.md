# ControlBarView.refreshAll


## Синтаксис


refreshAll ();


## Описание


Метод refreshAll обновляет всю панель элементов управления.


## Комментарии


Чтобы обновить только значения элементов управления, используйте метод [ControlBarView.refresh](ControlBarView.refresh.htm).
