# Конструктор TableDataSource


## Синтаксис


PP.Prx.TableDataSource (settings);


## Параметры


settings. JSON-объект с настройками компонента.


## Описание


Конструктор TableDataSource создает экземпляр класса [TableDataSource](TableDataSource.htm).


## Пример


Пример использования конструктора приведен в описании свойства [GridView.TableSource](../GridView/GridView.TableSource.htm).
