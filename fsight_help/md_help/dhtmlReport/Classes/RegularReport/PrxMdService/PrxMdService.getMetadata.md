# PrxMdService.getMetadata


## Синтаксис


getMetadata (report, recalc, callback);


## Параметры


report.. [Модель отчета](../Report/Report.htm);


recalc. Определяет, будет ли вычисляться отчет. По умолчанию установлено значение true, то есть отчет вычисляется. Если установлено значение false,отчет не вычисляется;


callback. Процедура обратного вызова.


## Описание


Метод getMetadata возвращает метаданные объекта.
