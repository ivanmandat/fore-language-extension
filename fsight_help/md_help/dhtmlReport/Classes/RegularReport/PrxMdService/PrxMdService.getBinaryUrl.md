# PrxMdService.getBinaryUrl


## Синтаксис


getBinaryUrl (report, objectId)


## Параметры


report. Модель регламентного отчета;


objectId. Идентификатор объекта, ссылку на который необходимо вернуть.


## Описание


Метод getBinaryUrl возвращает ссылку на объект в таблице.


## Комментарии


Метод getBinaryUrl является вспомогательным.
