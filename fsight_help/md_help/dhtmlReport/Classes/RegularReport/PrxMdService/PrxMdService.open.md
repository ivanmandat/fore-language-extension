# PrxMdService.open


## Синтаксис


open (key, callback);


## Параметры


key. Ключ отчета, который необходимо открыть;


callback. Процедура обратного вызова. Необязательный параметр.


## Описание


Метод open открывает регламентный отчет по ключу.


## Комментарии


Для открытия объекта с параметрами используйте метод [PrxMdService.openReport](PrxMdService.openReport.htm).
