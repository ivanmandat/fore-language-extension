# PrxMdService.Export


## Синтаксис


Export (report, args, callback)


## Параметры


report.
 Экспортируемый объект.


args. Параметры экспорта;


callback. Процедура обратного
 вызова;


## Описание


Метод Export осуществляет экспорт
 объекта в файл заданного формата.


## Комментарии


Метод Export помещает результат
 экспорта во внутренний буфер. Забрать результат экспорта из этого буфера
 можно только по отдельному прямому URL вида ..PPService.axd?action=export&key=LCFCKMJNHHCBEOAEMEFJGBGOCDANOGPEOLIDMKAJACDFLGED!M!S!PLCFCKMJNHHCBEOAELNCLHJOPHBFJANKECJGBENKFFADMGFKD!Bin!0&format=xls&d=1342526431868&inline=false


Данный URL специфичен для стандартного веб-приложения «Форсайт. Аналитическая платформа».
