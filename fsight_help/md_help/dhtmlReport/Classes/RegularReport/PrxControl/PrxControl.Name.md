# PrxControl.Name


## Синтаксис


Name: String


## Описание


Свойство Name устанавливает имя элемента управления регламентного отчета.


## Комментарии


Значение свойства устанавливается при помощи метода setName и возвращается при помощи метода getName. Из JSON значение свойства установить нельзя.
