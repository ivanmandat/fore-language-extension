# PrxControl.refresh


## Синтаксис


refresh (args);


## Параметры


args. Параметры обновления элемента управления.


## Описание


Метод refresh обновляет элемент управления.


## Комментарии


Метод refresh является аналогом метода [PrxControl.refreshAll](PrxControl.refreshAll.htm).
