# PrxControl.MetadataChanged


## Синтаксис


MetadataChanged: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие MetadataChanged наступает при измерении метаданных элемента управления регламентного отчета.
