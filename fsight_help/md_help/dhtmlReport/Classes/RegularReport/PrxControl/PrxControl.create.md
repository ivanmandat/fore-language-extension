# PrxControl.create


## Синтаксис


PP.Prx.Ui.PrxControl.create(settings);


## Параметры


settings. JSON-объект с настройками экземпляра класса.


## Описание


Конструктор create создает экземпляр класса [PrxControl](PrxControl.htm).
