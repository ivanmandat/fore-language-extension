# PrxControl.getControl


## Синтаксис


getControl();


## Описание


Метод getControl возвращает класс-наследник класса PP.Ui.[Control](dhtmlUi.chm::/Classes/control/control.htm), который является базовым для выбранного элемента управления регламентного отчета.
