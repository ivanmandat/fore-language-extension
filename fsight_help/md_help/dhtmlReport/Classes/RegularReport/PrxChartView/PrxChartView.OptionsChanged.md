# PrxChartView.OptionsChanged


## Синтаксис


OptionsChanged: function (sender,args);


## Параметры


sender. Источник события;


args. Информация
 о событии.


## Описание


Событие OptionsChanged наступает
 при изменении настроек диаграммы.


## Комментарии


Событие генерируется после выполнения метода [setInstance](PrxChartView.Instance.htm).


## Пример


Пример использования события приведен в описании свойства [PrxChartView.Options](PrxChartView.Options.htm).
