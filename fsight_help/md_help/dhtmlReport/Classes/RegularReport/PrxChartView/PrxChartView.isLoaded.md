# PrxChartView.isLoaded


## Синтаксис


isLoaded ();


## Описание


Метод isLoaded определяет, загружена ли диаграмма регламентного отчета.


## Комментарии


Метод возвращает значение true, если диаграмма загружена, и false - если не загружена.
