# PrxChartView.isImage


## Синтаксис


isImage();


## Описание


Метод isImage определяет, является
 ли диаграмма изображением.


## Комментарии


Если метод возвращает значение true,
 значит диаграмма регламентного отчета является изображением, иначе - диаграмма
 является экземпляром компонента [Chart](dhtmlChart.chm::/Components/Chart/Chart.htm)
 (по умолчанию).
