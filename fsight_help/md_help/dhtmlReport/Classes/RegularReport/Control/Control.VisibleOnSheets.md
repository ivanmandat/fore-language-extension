# Control.VisibleOnSheets


## Синтаксис


VisibleOnSheets: Array;


## Описание


Свойство VisibleOnSheets возвращает
 список листов, для которых отображается элемент управления.


## Комментарии


Свойство возвращается при помощи метода
 getVisibleOnSheets().


Используется для внутренней реализации компонента [ReportBox](../../../Components/RegularReport/ReportBox/ReportBox.htm).
