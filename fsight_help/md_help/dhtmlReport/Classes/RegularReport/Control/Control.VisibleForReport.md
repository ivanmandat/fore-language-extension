# Control.VisibleForReport


## Синтаксис


VisibleForReport: Boolean


## Описание


Свойство VisibleForReport определяет, отображается ли элемент управления для всех листов отчета.


## Комментарии


Если для свойства установлено значение true, значит элемент управления отображается для всех листов отчета.


## Пример


Пример использования свойства приведен в описании события ControlView.ControlChanged.
