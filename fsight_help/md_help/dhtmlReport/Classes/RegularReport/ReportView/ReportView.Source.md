# ReportView.Source


## Синтаксис


Source: PP.Prx.[Report](../Report/Report.htm)


## Описание


Свойство Source устанавливает
 источник для компонента [ReportBox](../../../Components/RegularReport/ReportBox/ReportBox.htm)
 - модель регламентного отчета, реализованную классом [Report](../Report/Report.htm).


## Комментарии


Значение свойства можно установить из JSON и с помощью метода setSource. Возвращается с помощью
 метода getSource.
