# ReportView.getDataView


## Синтаксис


getDataView ();


## Описание


Метод getDataView возвращает представление области данных регламентного отчета.


## Комментарии


Представление области данных регламентного отчета реализовано классом [DataAreaView](../DataAreaView/DataAreaView.htm).
