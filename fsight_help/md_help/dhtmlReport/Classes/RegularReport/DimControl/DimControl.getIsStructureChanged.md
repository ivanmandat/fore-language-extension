# DimControl.getIsStructureChanged


## Синтаксис


getIsStructureChanged(): Boolean;


## Описание


Метод getIsStructureChanged
 возвращает признак изменения структуры справочника элемента управления.


## Комментарии


Используется во внутренней реализации класса.


Возвращает true если после установки
 метаданных была изменена структура справочника.
