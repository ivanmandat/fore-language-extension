# DimControl.getIsContentChanged


## Синтаксис


getIsContentChanged(): Boolean;


## Описание


Метод getIsContentChanged возвращает
 признак изменения содержимого справочника элемента управления.


## Комментарии


Используется во внутренней реализации класса.


Возвращает true если после установки
 метаданных было изменено содержимое справочника.
