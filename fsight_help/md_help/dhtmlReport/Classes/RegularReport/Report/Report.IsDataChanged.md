# Report.IsDataChanged


## Синтаксис


IsDataChanged: Boolean


## Описание


Свойство IsDataChanged определяет, изменялось ли содержимое ячеек таблицы регламентного отчета.


## Комментарии


Если для свойства установлено значение true, содержимое ячеек было изменено, если установлено false - содержимое не изменялось.
