# Report.isAnyTableSheetExists


## Синтаксис


isAnyTableSheetExists();


## Описание


Метод isAnyTableSheetExists определяет, имеются ли в отчете таблицы.


## Комментарии


Если метод возвращает значение true, значит в отчете содержатся таблицы, иначе таблиц нет.


## Пример


Использование метода аналогично использованию метода [Report.isAnyControlVisible](Report.isAnyControlVisible.htm).
