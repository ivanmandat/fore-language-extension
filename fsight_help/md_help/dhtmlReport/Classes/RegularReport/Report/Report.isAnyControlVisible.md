# Report.isAnyControlVisible


## Синтаксис


isAnyControlVisible (sheetKey);


## Параметры


sheetKey. Ключ листа, для которого проверяется наличие элементов управления.


## Описание


Метод isAnyControlVisible определяет наличие элементов управления для указанного листа отчета.


## Комментарии


Если возвращается значение true, значит для указанного листа имеются элементы управления, иначе элементов управления нет.
