# Report.getSheetIndexByKey


## Синтаксис


getSheetIndexByKey (key);


## Параметры


key. Ключ листа отчета, индекс которого нужно вернуть.


## Описание


Метод getSheetIndexByKey возвращает индекс листа по ключу с учетом скрытых листов.
