# Report.Params


## Синтаксис


Params: Array of PP.Mb.[Param](dhtmlMetabase.chm::/Classes/Metabase/Param/Param.htm)


## Описание


Свойство Params задает параметры
 регламентного отчета.


## Комментарии


Значение свойства может быть установлено из JSON или при помощи метода
 setParams, а возвращено при помощи
 метода getParams.


## Пример


Пример использования свойства приведен в описании метода [PrxMdService.openReport](../PrxMdService/PrxMdService.openReport.htm).
