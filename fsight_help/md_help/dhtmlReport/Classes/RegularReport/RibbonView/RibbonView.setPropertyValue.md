# RibbonView.setPropertyValue


## Синтаксис


setPropertyValue (propertyName, value)


## Параметры


propertyName. Имя свойства, элемент перечисления PP.Prx.Ui.[RibbonProperty](../../../Enums/RibbonProperty.htm);


value. Значение свойства (типа Boolean).


## Описание


Метод setPropertyValue устанавливает значения для свойств ленты инструментов регламентного отчета.


## Комментарии


Названия свойств содержатся в перечислении [RibbonProperty](../../../Enums/RibbonProperty.htm). Свойства принимают значения типа Boolean.
