# RibbonView.showDataSaveDialog


## Синтаксис


showDataSaveDialog (args, mode)


## Параметры


args. Информация;


mode. Режим отображения диалога, элемент перечисления [DataSaveDlgMode](../../../Enums/DataSaveDlgMode.htm).


## Описание


Метод showDataSaveDialog открывает диалог для сохранения измененных данных отчета.


## Комментарии


Диалог для сохранения данных в режиме NoCancel (без кнопки «Выход») открывается при нажатии на кнопку ленты «Обновить», если в ячейках таблицы имеются несохраненные данные.


При нажатии на кнопку ленты «Отменить изменения» открывается диалог для сохранения данных в режиме Undo.


По умолчанию диалог открывается в режиме Full.
