# RibbonView.refreshAll


## Синтаксис


refreshAll ();


## Описание


Метод refreshAll обновляет содержимое ленты инструментов.


## Комментарии


Метод аналогичен методу [RibbonView.refresh](RibbonView.refresh.htm).


## Пример


Использование метода аналогично использованию метода [ReportView.refreshAll](../ReportView/ReportView.refreshAll.htm).
