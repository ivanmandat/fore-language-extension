# RibbonView.PropertyChanging


## Синтаксис


PropertyChanging: function (sender, args)


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие PropertyChanging наступает при нажатии на кнопки ленты инструментов.


## Пример


Использование события аналогично использованию события [RibbonView.PropertyChanged](RibbonView.PropertyChanged.htm).
