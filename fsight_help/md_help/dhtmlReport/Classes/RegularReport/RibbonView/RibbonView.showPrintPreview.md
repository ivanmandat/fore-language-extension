# RibbonView.showPrintPreview


## Синтаксис


showPrintPreview (src);


## Параметры


src. Источник регламентного отчета, экземпляр класса PP.Prx.[Report](../Report/Report.htm)


## Описание


Метод showPrintPreview открывает страницу для предварительного просмотра печатной версии отчета.
