# Работа с регламентным отчётом


## Описание


На данной странице приведены компоненты для работы с регламентным отчетом.


## Компоненты


		 Имя компонента
		 Краткое описание


		 ![](../Class_Image.gif)
		 [ControlBar](RegularReport/ControlBar/ControlBar.htm)
		 Компонент ControlBar
		 представляет собой панель элементов управления регламентного отчета.


		 ![](../Class_Image.gif)
		 [DataArea](RegularReport/DataArea/DataArea.htm)
		 Компонент DataArea
		 представляет собой область данных регламентного отчета.


		 ![](../Class_Image.gif)
		 [GridBox](RegularReport/GridBox/GridBox.htm)
		 Компонент GridBox представляет
		 собой таблицу регламентного отчета.


		 ![](../Class_Image.gif)
		 [ReportBox](RegularReport/ReportBox/ReportBox.htm)
		 Компонент ReportBox
		 служит для отображения регламентного отчета.


		 ![](../Class_Image.gif)
		 [RibbonView](RegularReport/RibbonView/RibbonView.htm)
		 Компонент RibbonView
		 представляет собой ленту инструментов регламентного отчета.


		 ![](../Class_Image.gif)
		 [DataSaveDialog](RegularReport/Dialogs/DataSaveDialog.htm)
		 Компонент DataSaveDialog
		 представляет собой диалог для сохранения измененных данных отчета.


		 ![](../Class_Image.gif)
		 [EmfExportDialog](RegularReport/Dialogs/EmfExportDialog.htm)
		 Компонент EmfExportDialog
		 представляет собой диалог для экспорта в формат *.emf.


		 ![](../Class_Image.gif)
		 [ExportDialog](RegularReport/Dialogs/ExportDialog.htm)
		 Компонент ExportDialog
		 реализует базовый диалог экспорта регламентного отчета.


		 ![](../Class_Image.gif)
		 [HtmlExportDialog](RegularReport/Dialogs/HtmlExportDialog.htm)
		 Компонент HtmlExportDialog
		 представляет собой диалог для экспорта регламентного отчета в
		 формат *.html.


		 ![](../Class_Image.gif)
		 [MhtExportDialog](RegularReport/Dialogs/MhtExportDialog.htm)
		 Компонент MhtExportDialog
		 представляет собой диалог для экспорта отчета в формат *.mht.


		 ![](../Class_Image.gif)
		 [ParamsDialog](RegularReport/Dialogs/ParamsDialog.htm)
		 Компонент ParamsDialog
		 представляет собой диалог для изменения параметров отчета.


		 ![](../Class_Image.gif)
		 [PdfExportDialog](RegularReport/Dialogs/PdfExportDialog.htm)
		 Компонент PdfExportDialog
		 представляет собой диалог для экспорта отчета в формат *.pdf.


		 ![](../Class_Image.gif)
		 [RtfExportDialog](RegularReport/Dialogs/RtfExportDialog.htm)
		 Компонент RtfExportDialog
		 представляет собой диалог экспорта отчета в формат *.rtf.


		 ![](../Class_Image.gif)
		 [XlsExportDialog](RegularReport/Dialogs/XlsExportDialog.htm)
		 Компонент XlsExportDialog
		 представляет собой диалог экспорта отчета в формат *.xls.
