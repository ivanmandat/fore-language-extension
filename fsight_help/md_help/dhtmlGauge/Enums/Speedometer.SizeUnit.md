# Speedometer.SizeUnit


Пространство имен: PP.Ui;


## Описание


Перечисление Speedometer.SizeUnit
 содержит типы измерения размеров элементов спидометра.


Используется следующими свойствами и методами:


	- [GaugeArrowBase.setRadiusUnit](../Classes/Speedometer/GaugeArrowBase/GaugeArrowBase.setRadiusUnit.htm).


## Допустимые значения


		 Значение


		 Краткое

		 Абсолютный размер.


		 Relative


		 Относительный размер.
