# SpeedometerSettings


Пространство имен: PP;


## Описание


Перечисление PP. SpeedometerSettings
 содержит варианты сохранения или загрузки установок.


Используется свойством PP.Ui.Speedometer.Settings.


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 View


		 Применять
		 установки только внешнего вида.


		 Values


		 Применять
		 установки только значений.


		 All


		 Применять
		 все установки.
