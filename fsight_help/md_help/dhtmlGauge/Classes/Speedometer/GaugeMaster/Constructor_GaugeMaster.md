# Конструктор GaugeMaster


## Синтаксис


PP.Ui.GaugeMaster(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор GaugeMaster создает экземпляр класса [GaugeMaster](GaugeMaster.htm).
