# GaugeObject.Name


## Синтаксис


Name: String;


## Описание


Свойство Name определяет наименование указателя для шкалы спидометра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setName, а возвращается - с помощью метода getName.
