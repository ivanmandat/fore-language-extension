# GaugeObject.redraw


## Синтаксис


redraw (speedometer, parent);


## Параметры


speedometer. Объект спидометра;


parent. Родительский объект.


## Описание


Метод redraw устанавливает атрибуты
 svg-элементам соответственно свойствам.
