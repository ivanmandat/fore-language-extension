# GaugeObject.paint


## Синтаксис


paint (speedometer, elementN);


## Параметры


speedometer. Объект спидометра;


elementN. Элементы раскраски.


## Описание


Метод paint раскрашивает фон и границу svg элемента соответственно установкам в [Background](GaugeObject.Background.htm), [Border](GaugeObject.Border.htm).
