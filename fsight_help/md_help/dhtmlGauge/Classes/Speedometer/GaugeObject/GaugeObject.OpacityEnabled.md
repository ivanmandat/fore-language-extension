# GaugeObject.OpacityEnabled


## Синтаксис


OpacityEnabled: Boolean;


## Описание


Свойство OpacityEnabled определяет
 признак использования настроек прозрачности элемента.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setOpacityEnabled,
 а возвращается с помощью метода getOpacityEnabled.


Допустимые значения:


	- True. Настройки прозрачности
	 элемента будут использованы;


	- False. Настройки прозрачности
	 элемента не будут использованы.
