# GaugeObject.MajorArrow


## Синтаксис


MajorArrow: [PP.Ui.LineArrow](../LineArrow/LineArrow.htm)|[PP.Ui.SectorArrow](../SectorArrow/SectorArrow.htm)|[PP.Ui.TriangleArrow](../TriangleArrow/TriangleArrow.htm);


## Описание


Свойство MajorArrow определяет основной указатель для шкалы спидометра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMajorArrow, а возвращается - с помощью метода getMajorArrow.
