# GaugeObject.BorderEnabled


## Синтаксис


BorderEnabled: Boolean;


## Описание


Свойство BorderEnabled определяет
 наличие границы элемента спидометра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setBorderEnabled,
 а возвращается с помощью метода getBorderEnabled.


Допустимые значения:


	- True. Элемент спидометра
	 имеет границу;


	- False. Элемент спидометра
	 не имеет границы.
