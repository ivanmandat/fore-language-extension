# GaugeArrowBase.getDimAttributeId


## Синтаксис


getDimAttributeId();


## Описание


Метод getDimAttributeId возвращает идентификатор атрибута измерения, по которому выбираются данные из источника данных.


## Комментарии


Метод возвращает значение типа String.
