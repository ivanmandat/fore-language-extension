# GaugeArrowBase.getAutoResKoef


## Синтаксис


getAutoResKoef();


## Описание


Метод getAutoResKoef возвращает отношение радиуса указателя спидометра к радиусу самого спидометра.


## Комментарии


Метод возвращает значение типа Number.
