# GaugeArrowBase.getScale


## Синтаксис


getScale();


## Описание


Метод getScale возвращает шкалу, в которой находится текущий указатель.


## Комментарии


Метод возвращает значение типа [PP.Ui.GaugeScale](../GaugeScale/GaugeScale.htm).
