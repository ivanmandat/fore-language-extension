# GaugeArrowBase.setRadiusUnit


## Синтаксис


setRadiusUnit(unit: [PP.Ui.Speedometer.SizeUnit](../../../Enums/Speedometer.SizeUnit.htm));


## Параметры


unit. Тип измерения длины указателя.


## Описание


Метод setRadiusUnit устанавливает тип измерения радиуса (длины) указателя.
