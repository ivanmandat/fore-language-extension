# GaugeArrowBase.getIsValid


## Синтаксис


getIsValid();


## Описание


Метод getIsValid возвращает результат проверки значения указателя на корректность.


## Комментарии


Метод возвращает значение типа Boolean. Если метод возвращает значение true, то значение указателя прошло проверку на корректность, иначе - не прошло.
