# GaugeCenterLabel.RealWidth


## Синтаксис


RealWidth: Number;


## Описание


Свойство RealWidth определяет
 фактическую ширину центральной текстовой метки спидометра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setRealWidth,
 а возвращается с помощью метода getRealWidth.
