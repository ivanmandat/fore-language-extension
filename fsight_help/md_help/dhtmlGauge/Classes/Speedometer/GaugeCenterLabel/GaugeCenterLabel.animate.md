# GaugeCenterLabel.animate


## Синтаксис


animate(speedometer: [PP.Ui.Speedometer](../Speedometer/Speedometer.htm),
 parent: [PP.Ui.Control](dhtmlUi.chm::/Classes/control/control.htm),
 isRedraw: Boolean);


## Параметры


speedometer. Спидометр;


parent. Родительский элемент
 спидометра;


isRedraw. Признак необходимости
 перерисовки родительских элементов метки. Необязательный параметр.
 Допустимые значения:


	- True.
	 Необходима перерисовка родительских элементов метки (по умолчанию);


	- False. Перерисовка родительских элементов
	 метки не требуется.


## Описание


Метод animate перерисовывает
 центральную текстовую метку спидометра.
