# GaugeCenterLabel.MaskLabel


## Синтаксис


MaskLabel: String;


## Описание


Свойство MaskLabel определяет
 формат вывода значений в подписи центральной текстовой метки.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMaskLabel,
 а возвращается с помощью метода getMaskLabel.
