# GaugeCenterLabel.RealHeight


## Синтаксис


RealHeight: Number;


## Описание


Свойство RealHeight определяет
 фактическую высоту центральной текстовой метки спидометра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setRealHeight,
 а возвращается с помощью метода getRealHeight.
