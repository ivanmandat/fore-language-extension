# GaugeCenterLabel.AutoSize


## Синтаксис


AutoSize: Boolean;


## Описание


Свойство AutoSize определяет
 признак автоматической установки размеров для центральной текстовой метки.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setAutoSize,
 а возвращается с помощью метода getAutoSize.


Допустимые значения:


	- True. По умолчанию.
	 Осуществляется автоматическая установка размеров для центральной текстовой
	 метки;


	- False. Автоматическая
	 установка размеров для центральной текстовой метки не осуществляется.


Метод setAutoSize содержит необязательный
 параметр _reInit, задающий признак необходимости обновления центральной
 текстовой метки.


Допустимые значения данного параметра:


	- True. Необходимо обновление
	 центральной текстовой метки;


	- False. Обновление центральной
	 текстовой метки не требуется.


Если параметр _reInit не задан, то обновление размеров центральной текстовой
 метки произойдет только в том случае, если значение свойства было изменено
 на противоположное.
