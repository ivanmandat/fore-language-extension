# GaugeTextLabel.TextAlign


## Синтаксис


TextAlign: [PP.HorizontalAlignment](dhtmlCommon.chm::/Enums/PP.HorizontalAlignment.htm)
 || String;


## Описание


Свойство TextAlign определяет
 выравнивание текста меток шкалы спидометра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTextAlign,
 а возвращается с помощью метода getTextAlign.**
