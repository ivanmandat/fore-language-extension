# GaugeScale.MinorTick


## Синтаксис


MinorTick: Array;


## Описание


Свойство MinorTick определяет массив элементов типа PP.Ui.[GaugeMarkerBase](../GaugeMarkerBase/GaugeMarkerBase.htm), которые являются дескрипторами дополнительных маркеров.


## Комментарии


Дополнительный маркер - деления шкалы, к которым нельзя устанавливать метки. Например, для циферблата часов дополнительными метками являются деления обозначающие минуты.
