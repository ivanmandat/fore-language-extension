# GaugeScale.LabelOffset


## Синтаксис


LabelOffset: Number;


## Описание


Свойство LabelOffset определяет смещение текстовых меток шкалы спидометра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLabelOffset, а возвращается с помощью метода getLabelOffset.**
