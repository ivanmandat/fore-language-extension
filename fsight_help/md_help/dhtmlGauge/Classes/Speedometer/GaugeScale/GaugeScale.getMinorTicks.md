# GaugeScale.getMinorTicks


## Синтаксис


getMinorTicks(index: Number);


## Параметры


index. Индекс маркера шкалы. Опциональный параметр. Если индекс не указывается, то метод возвращает массив всех маркеров, иначе - маркер с указанным индексом.


## Описание


Метод getMinorTicks возвращает массив дополнительных маркеров шкалы.


## Комментарии


Метод возвращает массив значений типа [PP.Ui.GaugeMarkerBase](../GaugeMarkerBase/GaugeMarkerBase.htm).
