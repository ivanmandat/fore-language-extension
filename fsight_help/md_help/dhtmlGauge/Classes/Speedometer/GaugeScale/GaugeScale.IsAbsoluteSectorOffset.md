# GaugeScale.IsAbsoluteSectorOffset


## Синтаксис


IsAbsoluteSectorOffset: Boolean;


## Описание


Свойство IsAbsoluteSectorOffset определяет, в каких единицах указывается смещение секторов шкалы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsAbsoluteSectorOffset, а возвращается с помощью метода getIsAbsoluteSectorOffset.**


**Е****сли свойство имеет значение true (по умолчанию), то смещение будет указываться в пикселях, иначе - в процентах.**
