# GaugeScale.setArrow


## Синтаксис


setArrow(value: [PP.Ui.GaugeArrowBase](../GaugeArrowBase/GaugeArrowBase.htm), index: Number)


## Параметры


value. Новый указатель;


index. Индекс заменяемого указателя.


## Описание


Метод setArrow заменяет один из существующих указателей шкалы спидометра.
