# GaugeScale.setLabelsColor


## Синтаксис


setLabelsColor(color: [PP.Color](dhtmlCommon.chm::/Classes/PP/Color/Color.htm)
 || String, index: Number);


## Параметры


color. Цвет текстовых меток.


index. Индекс текстовой метки,
 для которой устанавливается новое значение цвета. Опциональный параметр.
 Если параметр не указан, новый цвет будет установлен для всех текстовых
 меток шкалы.


## Описание


Метод setLabelsColor устанавливает
 цвет текстовых меток шкалы спидометра.
