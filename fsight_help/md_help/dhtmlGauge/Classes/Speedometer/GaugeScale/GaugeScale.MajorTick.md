# GaugeScale.MajorTick


## Синтаксис


MajorTick: Array;


## Описание


Свойство MajorTick определяет массив элементов типа PP.Ui.[GaugeMarkerBase](../GaugeMarkerBase/GaugeMarkerBase.htm), которые являются дескрипторами основных маркеров.


## Комментарии


Основной маркер - деление шкалы, к которому можно устанавливать метки. Например, для циферблата часов основными метками являются деления обозначающие часы.
