# GaugeScale.OuterRadius


## Синтаксис


OuterRadius: PP.[Point](dhtmlCommon.chm::/Classes/PP/Point/Point.htm);


## Описание


Свойство OuterRadius определяет
 радиус шкалы.


## Комментарии


Если значение свойства не задано, то в качестве радиуса шкалы берется
 радиус спидометра.
