# GaugeScale.getRenderInfo


## Синтаксис


getRenderInfo();


## Описание


Метод getRenderInfo возвращает
 данные о размере и позиции шкалы.


## Комментарии


Метод возвращает JSON-объект со следующими полями типа [PP.Point](dhtmlCommon.chm::/Classes/PP/Point/Point.htm):
 center - координаты центра шкалы,
 radius - радиус шкалы.
