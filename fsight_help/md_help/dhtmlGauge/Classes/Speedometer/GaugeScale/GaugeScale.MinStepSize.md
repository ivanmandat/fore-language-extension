# GaugeScale.MinStepSize


## Синтаксис


MinStepSize: Number


## Описание


Свойство MinStepSize определяет
 минимальное допустимое расстояние между [дополнительными
 маркерами](GaugeScale.MinorTick.htm) в пикселях.


## Комментарии


По умолчанию для свойства установлено значение 0.
