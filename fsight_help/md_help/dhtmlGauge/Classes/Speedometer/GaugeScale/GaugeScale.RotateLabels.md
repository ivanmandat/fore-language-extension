# GaugeScale.RotateLabels


## Синтаксис


RotateLabels: Boolean;


## Описание


Свойство RotateLabels определяет статус вращения подписей.


## Комментарии


Если установлено true, вращение осуществляется, false – вращение не осуществляется.
