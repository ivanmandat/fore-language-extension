# GaugeScale.IsAbsoluteTickOffset


## Синтаксис


IsAbsoluteTickOffset: Boolean;


## Описание


Свойство IsAbsoluteTickOffset определяет, в каких единицах указывается смещение отметок шкалы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsAbsoluteTickOffset, а возвращается с помощью метода getIsAbsoluteTickOffset.**


**Если свойство имеет значение true (по умолчанию), то смещение будет указываться в пикселях, иначе - в процентах.**
