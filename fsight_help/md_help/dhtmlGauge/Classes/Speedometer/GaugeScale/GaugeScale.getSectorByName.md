# GaugeScale.getSectorByName


## Синтаксис


getSectorByName(name: String);


## Параметры


name. Имя сектора.


## Описание


Метод getSectorByName возвращает массив секторов с указанным именем.


## Комментарии


Метод возвращает массив значений типа [PP.Ui.GaugeSector](../GaugeSector/GaugeSector.htm).
