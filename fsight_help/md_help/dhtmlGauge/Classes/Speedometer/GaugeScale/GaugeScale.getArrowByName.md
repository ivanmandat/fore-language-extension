# GaugeScale.getArrowByName


## Синтаксис


getArrowByName(name: String)


## Параметры


name. Имя указателя.


## Описание


Метод getArrowByName возвращает массив указателей шкалы спидометра с указанным именем.


## Комментарии


Метод возвращает массив значений типа [PP.Ui.GaugeArrowBase](../GaugeArrowBase/GaugeArrowBase.htm).
