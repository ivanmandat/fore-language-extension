# GaugeScale.getArrowValue


## Синтаксис


getArrowValue(arrow: [PP.Ui.GaugeArrowBase](../GaugeArrowBase/GaugeArrowBase.htm), index: Number);


## Параметры


arrow. Указатель шкалы;


index. Шаг временной шкалы. Опциональный параметр. Если параметр не указан, то возвращается значение указателя для текущего шага временной шкалы.


## Описание


Метод getArrowValue возвращает значение указателя.


## Комментарии


Метод возвращает значение типа Number.
