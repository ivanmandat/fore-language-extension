# GaugeScale.getLabelset


## Синтаксис


getLabelset(index: Number);


## Параметры


index. Индекс текстового узла. Опциональный параметр. Если параметр не указан, будет возвращен массив всех текстовых узлов, иначе - текстовыя узел с указанным индексом.


## Описание


Метод getLabelset возвращает массив текстовых узлов шкалы.


## Комментарии


Метод возвращает массив значений типа SVGTextNode.
