# DropShadow.setAttr


## Синтаксис


setAttr(attrs: Object);


## Параметры


attrs. Объект с атрибутами
 SVG-элемента.


## Описание


Метод setAttr устанавливает
 указанные атрибуты для SVG-элемента, соответствующего тени визуального
 элемента спидометра.
