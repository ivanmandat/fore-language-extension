# DropShadow.getShadowElement


## Синтаксис


getShadowElement();


## Описание


Метод getShadowElement возвращает
 SVG-элемент, соответствующий тени визуального элемента спидометра.


## Комментарии


Метод возвращает объект типа SVGElement.
