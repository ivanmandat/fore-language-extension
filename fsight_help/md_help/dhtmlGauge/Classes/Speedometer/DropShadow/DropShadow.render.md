# DropShadow.render


## Синтаксис


render(parent: SVGElement | VMLElement);


## Параметры


parent. Родительская вершина,
 в которой создаётся SVG-элемент, соответствующий тени визуального элемента
 спидометра.


## Описание


Метод render создаёт SVG-элемент,
 соответствующий тени визуального элемента спидометра.
