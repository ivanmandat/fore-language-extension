# Конструктор Speedometer


## Синтаксис


PP.Ui.Speedometer(setting);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор Speedometer создает экземпляр компонента [Speedometer](../../../Components/Speedometer/Speedometer.htm).
