# Speedometer.getState


## Синтаксис


getState(scaleIndex: Number);


## Параметры


scaleIndex. Индекс шкалы. Если
 параметр не указывается, то будут возвращены настройки для шкалы с индексом
 0.


## Описание


Метод getState возвращает настройки
 шкалы спидометра.


## Комментарии


Метод возвращает объект типа [PP.Ui.PropertyChangedEventArgs](dhtmlMetabase.chm::/Classes/Metabase/PropertyChangedEventArgs/PropertyChangedEventArgs.htm).
