# Speedometer.getOpacityEnabled


## Синтаксис


getOpacityEnabled();


## Описание


Метод getOpacityEnabled возвращает признак использования прозрачности спидометром.


## Комментарии


Метод возвращает значение типа Boolean. Если метод возвращает значение true, то спидометр использует свойство прозрачности, иначе - не использует.
