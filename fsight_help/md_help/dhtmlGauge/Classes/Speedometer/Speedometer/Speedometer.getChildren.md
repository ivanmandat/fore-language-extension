# Speedometer.getChildren


## Синтаксис


getChildren();


## Описание


Метод getChildren возвращает массив дочерних элементов спидометра.


## Комментарии


Метод возвращает массив значений типа [PP.Ui.GaugeObject](../GaugeObject/GaugeObject.htm).
