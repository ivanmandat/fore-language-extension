# Speedometer.paint


## Синтаксис


paint(speedometer: [PP.Ui.Speedometer](Speedometer.htm), elementN: SVGElement);


## Параметры


speedometer. Объект спидометра;


elementN. Элементы раскраски.


## Описание


Метод paint раскрашивает фон и границу SVG-элемента соответственно установкам.
