# Speedometer.BaseHeight


## Синтаксис


BaseHeight: Number;


## Описание


Свойство BaseHeight определяет базовую высоту спидометра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setBaseHeight, а возвращается с помощью метода getBaseHeight.**
