# Speedometer.BaseWidth


## Синтаксис


BaseWidth: Number;


## Описание


Свойство BaseWidth определяет базовую ширину спидометра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setBaseWidth, а возвращается с помощью метода getBaseWidth.**
