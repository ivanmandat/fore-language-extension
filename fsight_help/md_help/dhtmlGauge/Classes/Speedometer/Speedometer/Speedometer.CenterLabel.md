# Speedometer.CenterLabel


## Синтаксис


CenterLabel: [PP.Ui.GaugeCenterLabel](../GaugeCenterLabel/GaugeCenterLabel.htm);


## Описание


Свойство CenterLabel определяет
 центральную текстовую метку спидометра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setCenterLabel,
 а возвращается с помощью метода getCenterLabel.
