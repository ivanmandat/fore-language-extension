# Speedometer.OuterRadius


## Синтаксис


OuterRadius: PP.[Point](dhtmlCommon.chm::/Classes/PP/Point/Point.htm);


## Описание


Свойство OuterRadius определяет
 область отрисовки шкалы спидометра относительно его центра.
