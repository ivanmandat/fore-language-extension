# Speedometer.TimeLineStep


## Синтаксис


TimeLineStep: Number;


## Описание


Свойство TimeLineStep определяет
 текущий шаг временной шкалы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTimeLineStep,
 а возвращается с помощью метода getTimeLineStep.
