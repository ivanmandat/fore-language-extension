# Speedometer.BorderStyle


## Синтаксис


BorderStyle: [PP.Ui.LineTypes](dhtmlUi.chm::/Enums/LineTypes.htm);


## Описание


Свойство BorderStyle определяет
 стиль границ спидометра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setBorderStyle,
 а возвращается с помощью метода getBorderStyle.
