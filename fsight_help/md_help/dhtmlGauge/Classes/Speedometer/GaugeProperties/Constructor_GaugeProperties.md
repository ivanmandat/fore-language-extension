# Конструктор GaugeProperties


## Синтаксис


PP.Ui.GaugeProperties(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор GaugeProperties создает экземпляр класса [GaugeProperties](GaugeProperties.htm).


## Комментарии


Конструктор GaugeProperties вызывается неявно при создании мастера [GaugeMaster](../GaugeMaster/GaugeMaster.htm).
