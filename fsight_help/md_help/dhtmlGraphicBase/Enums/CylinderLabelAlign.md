# CylinderLabelAlign


Пространство имен: PP.Ui


## Описание


Перечисление CylinderLabelAlign
 содержит типы горизонтальных выравниваний подписей оси.


Используется свойством [CylinderHeader.HorizontalAlign](../Classes/Cylinder/CylinderHeader/CylinderHeader.HorizontalAlign.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 Center


		 Выравнивание по центру.


		 Left


		 Выравнивание по левому краю.


		 Right


		 Выравнивание по правому краю.
