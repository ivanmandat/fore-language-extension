# SizeLegendMarkerShape


Пространство имен: PP.Ui;


## Описание


Перечисление SizeLegendMarkerShape
 содержит формы маркеров [легенды
 размеров](../Classes/SizeLegend/SizeLegend.htm).


Используется свойством [SizeLegend.Marker](../Classes/SizeLegend/SizeLegend.Marker.htm).


## Допустимые значения


		 Значение


		 Краткое описание


		 Circle
		 Форма маркера в виде круга.


		 Rectangle
		 Форма маркера в виде квадрата.


		 Trapezoid
		 Форма маркера в виде трапеции.


		 Custom
		 Специальная форма.
