# SpectrumLegendMode


Пространство имен:
 PP.Ui


## Описание


Перечисление SpectrumLegendMode
 содержит режимы отображения элементов [спектральной
 легенды](../Classes/SpectrumLegend/SpectrumLegend.htm).


Используется свойством [SpectrumLegend.Mode](../Classes/SpectrumLegend/SpectrumLegend.Mode.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 IntervalBetweenOnly


		 Режим отображения "между".


		 IntervalLessGreater


		 Режим отображения "меньше-между-больше".


		 Spectrum


		 Градиентный режим отображения.
