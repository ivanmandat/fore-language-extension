# LegendMode


Пространство имен: PP.Ui;


## Описание


Перечисление LegendMode содержит
 режимы отображения элементов легенды.


Используется свойством [IntervalsLegend.Mode](../Classes/IntervalsLegend/IntervalsLegend.Mode.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 BetweenOnly


		 Режим отображения "между".


		 LessGreater


		 Режим отображения "меньше-между-больше".
