# DataMappingType


Пространство имен: PP


## Описание


Перечисление DataMappingType содержит
 типы преобразования данных, сопоставляемых элементам визуализатора.


Используется свойством [DataMapping.Type](../Classes/PP/DataMapping/DataMapping.Type.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 None


		 Преобразование данных отсутствует.


		 Scale


		 Преобразование данных по шкале.


		 Constant


		 Константное преобразование.


		 Transformer


		 Преобразование данных с помощью трансформатора с возможностью
		 задать свой метод для обработки данных.
