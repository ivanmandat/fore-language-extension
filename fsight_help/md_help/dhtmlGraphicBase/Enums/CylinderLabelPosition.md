# CylinderLabelPosition


Пространство имен: PP.Ui


## Описание


Перечисление CylinderLabelPosition
 содержит типы вертикальных выравниваний подписей оси.


Используется свойством [CylinderHeader.VerticalAlign](../Classes/Cylinder/CylinderHeader/CylinderHeader.VerticalAlign.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 Center


		 Выравнивание по центру.


		 Over


		 Выравнивание по верхнему краю.


		 Under


		 Выравнивание по нижнему краю.
