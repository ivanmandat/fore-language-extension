# LegendPageMode


Пространство имен: PP.Ui


## Описание


Перечисление LegendPageMode
 содержит режимы разбиения элементов легенды на страницы.


Используется следующими свойствами:


	- [IntervalsLegend.PageMode](../Classes/IntervalsLegend/IntervalsLegend.PageMode.htm);


	- [MapTerritoryTranscript.PageMode](dhtmlMapChart.chm::/Classes/Map/MapTerritoryTranscript/MapTerritoryTranscript.PageMode.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 MultiplePages


		 Элементы легенды разбиваются на
		 несколько страниц.


		 SinglePage


		 Элементы легенды отображаются
		 на одной странице.


		 SinglePageWithScroll


		 Элементы отображаются на одной
		 странице. Если элементы не помещаются, то для легенды отображается
		 полоса прокрутки.
