# ScaleResultMode


Пространство имен: PP;


## Описание


Перечисление ScaleResultMode содержит
 типы возвращаемых значений шкалы визуализатора.


Используется следующим свойством:


	- [ScaleBase.ResultMode](../Classes/PP/ScaleBase/ScaleBase.ResultMode.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 Intervals


		 В качестве значения шкалы возвращаются элементы
		 легенды визуализатора, имеющей интервалы.


		 IntervalsWithEquality


		 В качестве значения шкалы возвращаются элементы
		 легенды визуализатора, имеющей интервалы, вместе с элементами,
		 соответствующие граничным значениям.


		 Spectrum


		 В качестве значения шкалы возвращаются элементы спектральной
		 легенды.
