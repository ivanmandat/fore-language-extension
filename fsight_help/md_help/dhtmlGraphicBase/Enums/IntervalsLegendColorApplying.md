# IntervalsLegendColorApplying


Пространство имен: PP.Ui;


## Описание


Перечисление IntervalsLegendColorApplying содержит
 режимы цветовой визуализации интервальной легенды.


Используется следующим свойством:


	- [IntervalsLegend.ColorApplying](../Classes/IntervalsLegend/IntervalsLegend.ColorApplying.htm).


## Допустимые значения


		 Значение


		 Краткое описание


		 Marker
		 Цветовая визуализация интервальной легенды применяется к маркеру.


		 Background
		 Цветовая визуализация интервальной легенды применяется к фону.
