# TextDirection


Пространство имён: PP.Ui


## Описание


Перечисление TextDirection определяет
 направление текста.


Используется свойством [CanvasLabel.Direction](../Classes/CanvasLabel/CanvasLabel.Direction.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 BottomToTop


		 Направление текста снизу вверх.


		 LeftToRight


		 Направление текста слева направо.


		 RightToLeft


		 Направление текста справа налево.


		 TopToBottom


		 Направление текста сверху вниз.
