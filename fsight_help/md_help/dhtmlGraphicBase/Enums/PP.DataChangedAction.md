# DataChangedAction


Пространство имен: PP


## Описание


Перечисление DataChangedAction содержит
 варианты действий, приводящих к изменению данных в [источнике](../Classes/PP/DataSource/DataSource.htm).


Используется в аргументах события [DataSource.DataChanged](../Classes/PP/DataSource/DataSource.DataChanged.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 TimeAxisAdded


		 Добавление временной оси.


		 TimeAxisRemoved


		 Удаление временной оси.


		 ItemDataChanged


		 Изменение данных элемента.


		 Cleared


		 Очистка данных источника.


		 AttributeNameChanged


		 Изменение имени атрибута.
