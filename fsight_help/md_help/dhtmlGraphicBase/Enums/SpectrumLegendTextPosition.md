# SpectrumLegendTextPosition


Пространство имен:
 PP.Ui


## Описание


Перечисление SpectrumLegendTextPosition содержит
 позиции текста элементов в [спектральной
 легенде](../Classes/SpectrumLegend/SpectrumLegend.htm).


Используется свойством [SpectrumLegend.TextPosition](../Classes/SpectrumLegend/SpectrumLegend.TextPosition.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 Above


		 Текст располагается над спектром.


		 Under


		 Текст располагается под спектром.
