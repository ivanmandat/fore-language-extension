# AreaSelection.end


## Синтаксис


end();


## Описание


Метод end завершает
 отрисовку рамки выделения.


## Комментарии


Метод возвращает true, если
 была начата отрисовка рамки выделения, или значение false
 в противном случае.


Начало отрисовки рамки выделения производится с помощью метода [AreaSelection.start](AreaSelection.start.htm).
