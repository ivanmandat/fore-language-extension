# AreaSelection.CornerSize


## Синтаксис


CornerSize: Number;


## Описание


Свойство CornerSize определяет
 размер уголков рамки выделения в режиме [PP.Ui.AreaSelectionType.Cornered](../../../Enums/AreaSelectionType.htm).


По умолчанию значение свойства равно 10.


## Комментарии


Значение свойства устанавливается с помощью метода setCornerSize,
 а возвращается с помощью метода getCornerSize.


## Пример


Пример использования приведен в описании свойства [AreaSelection.MinSize](AreaSelection.MinSize.htm).
