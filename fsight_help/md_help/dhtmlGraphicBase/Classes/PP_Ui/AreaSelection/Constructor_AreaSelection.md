# Конструктор AreaSelection


## Синтаксис


PP.Ui.AreaSelection(settings);


## Параметры


settings. JSON-объект со значениями
 свойств класса.


## Описание


Конструктор AreaSelection создает
 экземпляр класса [AreaSelection](AreaSelection.htm).
