# AreaSelection.Type


## Синтаксис


Type: [PP.Ui.AreaSelectionType](../../../Enums/AreaSelectionType.htm);


## Описание


Свойство Type определяет форму
 рамки выделения.


## Комментарии


Значение свойства устанавливается с помощью метода setType,
 а возвращается с помощью метода getType.


По умолчанию для свойства установлено значение [PP.Ui.AreaSelectionType.Simple](../../../Enums/AreaSelectionType.htm).


Метод setType содержит необязательный
 параметр _force, задающий признак
 необходимости принудительной установки нового значения формы рамки
 выделения. Допустимые значения данного параметра:


	- true. Необходима принудительная
	 установка нового значения формы рамки выделения;


	- false. Принудительная
	 установка нового значения формы рамки выделения не требуется.


Если параметр _force не задан
 или равен значению false, то метод
 setType проверяет необходимость
 установки нового значения.
