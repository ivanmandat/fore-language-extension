# PP.Ui.ConfigClass.getSourceTypes


## Синтаксис


getSourceTypes ();


## Описание


Метод getSourceTypes возвращает
 массив типов источников данных, доступных для визуализаторов.


## Комментарии


Метод возвращает массив, который может содержать следующие значения:


		 Значение
		 Краткое описание


		 "Excel"
		 В качестве источника данных могут
		 использоваться файлы в формате *.xls.


		 "ExcelEx"
		 В качестве источника данных могут
		 использоваться файлы в формате *.xlsx.


		 "Access"
		 В качестве источника данных могут
		 использоваться файлы в формате *.mdb.


		 "Xml"
		 В качестве источника данных могут использоваться файлы в формате
		 *.xml.


		 "Text"
		 В качестве источника данных могут использоваться файлы в формате
		 *.txt.


		 "Mb"
		 В качестве источника данных могут использоваться объекты репозитория.


## Пример


Приведенный ниже код выполняется из консоли браузера со страницы стандартного
 веб-приложения «Форсайт. Аналитическая платформа»
 с открытым экспресс-отчетом, регламентным отчетом, рабочей книгой или
 аналитической панелью.


//Получаем настройки веб-приложения
conf = PP.Ui.getConfig();
//Установим в качестве возможного источника данных только объект репозитория
conf.setSourceTypes(["Mb"]);
//Выведем на экран сообщение со списком возможных источников данных
alert("Доступные источники данных: " + conf.getSourceTypes());
После выполнения примера для открытого отчета будет доступно добавление
 в качестве источника данных только объекта репозитория. На экран будет
 выведено сообщение «Доступные источники данных: Mb».
