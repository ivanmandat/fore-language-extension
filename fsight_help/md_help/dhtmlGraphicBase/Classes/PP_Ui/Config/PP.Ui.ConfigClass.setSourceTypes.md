# PP.Ui.ConfigClass.setSourceTypes


## Синтаксис


setSourceTypes (value: Array);


## Параметры


value. Массив доступных источников данных для отчета.


## Описание


Метод setSourceTypes устанавливает массив типов источников данных, доступных для визуализаторов.


## Комментарии


Метод устанавливает массив, который может содержать следующие значения:


 Значение
 Краткое описание


 "Excel"

В качестве источника данных могут использоваться файлы в формате *.xls.


 "ExcelEx"

В качестве источника данных могут использоваться файлы в формате *.xlsx.


 "Access"

В качестве источника данных могут использоваться файлы в формате *.mdb.


 "Xml"
 В качестве источника данных могут использоваться файлы в формате *.xml.


 "Text"
 В качестве источника данных могут использоваться файлы в формате *.txt.


 "Mb"
 В качестве источника данных могут использоваться объекты репозитория.
