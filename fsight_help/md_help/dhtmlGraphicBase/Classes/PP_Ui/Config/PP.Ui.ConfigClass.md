# ConfigClass


Пространство имен: PP.Ui;


## Иерархия наследования


[Object](dhtmlCommon.chm::/Classes/object/object.htm)


ConfigClass


## Описание


Класс ConfigClass содержит глобальные
 настройки визуальных компонентов.


## Комментарии


Объект класса ConfigClass возвращает
 метод [PP.Ui.getConfig](dhtmlUi.chm::/Classes/PP_Ui/PP.Ui.getConfig.htm).


## Синтаксис


PP.initClass(PP.Ui.ConfigClass, PP.Object, "PP.Ui.ConfigClass");
