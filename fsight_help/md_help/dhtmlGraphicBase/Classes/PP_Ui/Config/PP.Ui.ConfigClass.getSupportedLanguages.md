# PP.Ui.ConfigClass.getSupportedLanguages


## Синтаксис


getSupportedLanguages ();


## Описание


Метод getSupportedLanguages возвращает массив поддерживаемых языков.


## Комментарии


Метод возвращает массив кодов поддерживаемых языков в формате LCID.


По умолчанию доступны следующие языки:


 Код
 Язык


1033


Английский.


 1049

Русский.


 2052

Китайский.


 1058
 Украинский.
