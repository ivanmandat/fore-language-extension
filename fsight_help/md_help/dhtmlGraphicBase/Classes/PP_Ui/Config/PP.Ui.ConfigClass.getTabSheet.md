# PP.Ui.ConfigClass.getTabSheet


## Синтаксис


getTabSheet ();


## Описание


Метод getTabSheet возвращает
 параметры таблицы.


## Комментарии


Метод возвращает JSON-объект со следующими полями:


		 Имя поля
		 Тип
		 Краткое описание


		 EnableNativeContextMenu
		 Boolean
		 Определяет, будет ли доступно для таблицы контекстное меню
		 браузера. Допустимые значения:


			- true. Контекстное
			 меню браузера доступно;


			- false. Контекстное
			 меню браузера недоступно (по умолчанию).


		 EnableSelection
		 Boolean
		 Определяет, будет ли доступно выделение ячеек таблицы вручную. Допустимые значения:


			- true. Выделение
			 доступно (по усмолчанию);


			- false. Выделение
			 недоступно.


Значения полей устанавливаются при помощи метода set<Имя
 поля> и возвращаются при помощи метода get<Имя
 поля>.


## Пример


Приведенный ниже код выполняется из консоли браузера со страницы стандартного
 веб-приложения «Форсайт. Аналитическая платформа»
 с открытым экспресс-отчетом, регламентным отчетом, рабочей книгой или
 аналитической панелью.


//Получаем настройки веб-приложения
conf = PP.Ui.getConfig();
//Установим возможность вызова контексного меню браузера для таблицы
conf.getTabSheet().setEnableNativeContextMenu(true);
После выполнения примера контекстное меню браузера будет доступно
 для таблицы.
