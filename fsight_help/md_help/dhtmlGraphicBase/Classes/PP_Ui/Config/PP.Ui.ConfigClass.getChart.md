# PP.Ui.ConfigClass.getChart


## Синтаксис


getChart ();


## Описание


Метод getChart возвращает параметры
 диаграммы.


## Комментарии


Метод возвращает JSON-объект со следующими полями:


		 Имя поля
		 Тип
		 Краткое описание


		 Animation
		 Boolean
		 Определяет, будет ли использоваться анимация рядов при перестройке
		 диаграммы. Допустимые значения:


			- true. Анимация
			 включена;


			- false. Анимация
			 выключена (по умолчанию).


		 ShowLoadConfirm
		 Boolean
		 Определяет, будет ли отображается предупреждение о возмжной
		 продолжительной загрузке, если диаграмма содержит большое количество
		 точек. Допустимые значения:


			- true. Предупреждение
			 отображается (по умолчанию);


			- false. Анимация
			 выключена.


		 EnableCanvas
		 Boolean
		 Определяет, используется ли для отрисовки диаграммы элемент
		 Canvas. Допустимые значения:


			- true. Используется;


			- false. Не используется (по
			 умолчанию).


Значения полей устанавливаются при помощи метода set<Имя
 поля> и возвращаются при помощи метода get<Имя
 поля>.


## Пример


Приведенный ниже код выполняется из консоли браузера со страницы стандартного
 веб-приложения «Форсайт. Аналитическая платформа»
 с открытым экспресс-отчетом, регламентным отчетом, рабочей книгой или
 аналитической панелью.


//Получаем настройки веб-приложения
conf = PP.Ui.getConfig();
//Включаем анимацию диаграммы
conf.getChart().setAnimation(true);
После выполнения примера диаграмма открытого отчета будет отрисовываться
 с использованием анимации.
