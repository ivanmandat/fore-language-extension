# PP.Ui.ConfigClass.setChart


## Синтаксис


setChart (value: JSON);


## Параметры


value. JSON-объект
 с параметрами таблицы.


## Описание


Метод setChart устанавливает
 параметры диаграммы.


## Комментарии


Метод возвращает JSON-объект со следующими полями:


		 Имя поля
		 Тип
		 Краткое описание


		 Animation
		 Boolean
		 Определяет, будет ли использоваться анимация рядов при перестройке
		 диаграммы. Допустимые значения:


			- true. Анимация
			 включена;


			- false. По умолчанию.
			 Анимация выключена.


		 ShowLoadConfirm
		 Boolean
		 Определяет, будет ли отображается предупреждение о возмжной
		 продолжительной загрузке, если диаграмма содержит большое количество
		 точек. Допустимые значения:


			- true. По умолчанию.
			 Предупреждение отображается;


			- false. Анимация
			 выключена.


		 EnableCanvas
		 Boolean
		 Определяет, используется ли для отрисовки диаграммы элемент
		 Canvas. Допустимые значения:


			- true. Используется;


			- false. По умолчанию.
			 Не используется.


## Пример


Приведённый ниже код выполняется из консоли браузера со страницы стандартного
 веб-приложения «Форсайт. Аналитическая платформа»
 с открытым экспресс-отчётом, регламентным отчётом, рабочей книгой или
 аналитической панелью.


//Получаем настройки веб-приложения
conf = PP.Ui.getConfig();
//Включим возможность анимации рядов при перерисовке диаграммы
conf.setChart({Animation: true});
После выполнения примера диаграмма открытого отчета будет отрисовываться
 с использованием анимации.
