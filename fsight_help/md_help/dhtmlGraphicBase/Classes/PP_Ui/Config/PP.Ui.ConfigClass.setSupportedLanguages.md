# PP.Ui.ConfigClass.setSupportedLanguages


## Синтаксис


setSupportedLanguages (value: Array);


## Параметры


value. Массив
 кодов поддерживаемых языков.


## Описание


Метод setSupportedLanguages
 устанавливает массив поддерживаемых языков.


## Комментарии


Метод устанавливает массив кодов поддерживаемых языков в формате LCID.


По умолчанию доступны следующие языки:


		 Код
		 Язык


		 1033


		 Английский.


		 1049
		 Русский.


		 2052
		 Китайский.


		 1058
		 Украинский.


## Пример


Приведенный ниже код выполняется из консоли браузера со страницы стандартного
 веб-приложения «Форсайт. Аналитическая платформа»
 с открытым экспресс-отчетом, регламентным отчетом, рабочей книгой или
 аналитической панелью.


//Получаем настройки веб-приложения
conf = PP.Ui.getConfig();
//Оставляем в списке поддерживаемых языков только английский и русский
conf.setSupportedLanguages([1033, 1049]);
//Выводим на экран сообщение со списком поддерживаемых языков
alert ("Поддерживаемые языки: " + conf.getSupportedLanguages());
После выполнения примера список поддерживаемых языков веб-приложения
 сократится до русского и английского. На экран будет выведено сообщение
 с обновленным списком поддерживаемых языков.
