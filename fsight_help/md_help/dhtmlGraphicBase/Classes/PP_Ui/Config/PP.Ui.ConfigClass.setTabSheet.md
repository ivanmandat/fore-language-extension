# PP.Ui.ConfigClass.setTabSheet


## Синтаксис


setTabSheet (value);


## Параметры


value. JSON-объект
 с параметрами таблицы.


## Описание


Метод setTabSheet устанавливает
 параметры таблицы.


## Комментарии


Метод устанавливает JSON-объект со следующими полями:


		 Имя поля
		 Тип
		 Краткое описание


		 EnableNativeContextMenu
		 Boolean
		 Определяет, будет ли доступно для таблицы контекстное меню
		 браузера. Допустимые значения:


			- true. Контекстное
			 меню браузера доступно;


			- false. По умолчанию.
			 Контекстное меню браузера недоступно.


		 EnableSelection
		 Boolean
		 Определяет, будет ли доступно выделение ячеек таблицы вручную.
		 Допустимые значения:


			- true. По умолчанию.
			 Выделение доступно;


			- false. Выделение
			 недоступно.


## Пример


Приведённый ниже код выполняется из консоли браузера со страницы стандартного
 веб-приложения «Форсайт. Аналитическая платформа»
 с открытым экспресс-отчётом, регламентным отчётом, рабочей книгой или
 аналитической панелью.


//Получаем настройки веб-приложения
conf = PP.Ui.getConfig();
//Установим возможность вызова контекстного меню браузера для таблицы
conf.setTabSheet({EnableNativeContextMenu: true});
После выполнения примера контекстное меню браузера будет доступно
 для таблицы.
