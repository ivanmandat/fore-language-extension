# Canvas.setStrokeStyle


## Синтаксис


setStrokeStyle(style: PP.[Brush](dhtmlCommon.chm::/Classes/PP/Brush/Brush.htm)
 | Object | String, bounds_left: PP.[Rect](dhtmlCommon.chm::/Classes/PP/Rect/Rect.htm)
 | Number, top: Number, width: Number, height: Number);


## Параметры


style. Кисть, объект ее настроек, объект градиента или строковое представление
 цвета;


bounds_left. Прямоугольник
 с размерами градиента кисти или его левая граница. Необязательный параметр;


top. Верхняя граница градиента.
 Необязательный параметр;


width. Ширина градиента. Необязательный
 параметр;


height. Высота градиента. Необязательный
 параметр.


## Описание


Метод setStrokeStyle устанавливает
 стиль границ элементов канвы.
