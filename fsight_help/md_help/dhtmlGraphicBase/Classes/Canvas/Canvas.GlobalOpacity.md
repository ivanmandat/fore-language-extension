# Canvas.GlobalOpacity


## Синтаксис


GlobalOpacity: Number;


## Описание


Свойство GlobalOpacity устанавливает
 глобальную прозрачность элементам канвы.


## Комментарии


Глобальная прозрачность умножается на значение прозрачности, указанной
 в стиле при отрисовке элемента.


Значение свойства устанавливается из JSON и с помощью метода setGlobalOpacity,
 а возвращается с помощью метода getGlobalOpacity.


Значения диапазона прозрачности изменяются от 0 до 1.
