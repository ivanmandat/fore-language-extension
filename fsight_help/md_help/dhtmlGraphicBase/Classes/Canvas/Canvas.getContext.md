# Canvas.getContext


## Синтаксис


getContext();


## Описание


Метод getContext возвращает
 контекст канвы.


## Комментарии


Контекст - объект, который предоставляет методы и свойства для отрисовки
 элементов на канве.


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [Canvas](../../Components/Canvas/Canvas.htm) c наименованием
 «canvas» (см. «[Пример
 создания компонента Canvas](../../Components/Canvas/Example_Canvas.htm)»). Получим параметры шрифта контекста канвы:


// Получаем параметры шрифта контекста канвы
console.log("Параметры шрифта контекста канвы: " + canvas.getContext().font);
В результате в консоль будут выведены параметры шрифта контекста канвы:


Параметры шрифта контекста канвы: 10px sans-serif
