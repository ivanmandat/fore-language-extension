# Canvas.endPath


## Синтаксис


endPath(doClose: Boolean);


## Параметры


doClose. Признак определяет,
 будет ли замкнута полилиния после завершения отрисовки. Если для параметра
 установлено значение true, то
 линия будет замкнута.


## Описание


Метод endPath заканчивает отрисовку
 полилинии.


## Комментарии


Отрисовка полилинии начинается с вызова метода [Canvas.beginPath](Canvas.beginPath.htm).


Полилиния - это непрерывная линия, сформированная из одной или более
 соединённых линий.
