# Canvas.drawCircle


## Синтаксис


drawCircle(cx: Number, cy: Number, r: Number, drawFill:
 Boolean, drawStroke: Boolean);


## Параметры


cx. Координата X центра окружности;


cy. Координата Y центра окружности;


r. Радиус окружности;


drawFill. Определяет, будет
 ли залита окружность. Если для параметра установлено значение true,
 то окружность будет залита;


drawStroke. Определяет, будут
 ли отрисованы границы окружности. Если для параметра установлено значение
 true, то границы будут отрисованы.


## Описание


Метод drawCircle рисует окружность
 на канве.


## Комментарии


Окружность будет отображена на канве после вызова метода [Canvas.flush](Canvas.flush.htm).
