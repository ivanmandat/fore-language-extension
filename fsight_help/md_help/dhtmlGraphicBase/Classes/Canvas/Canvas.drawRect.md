# Canvas.drawRect


## Синтаксис


drawRect(x: Number, y: Number, width: Number, height:
 Number, drawFill: Boolean, drawStroke: Boolean);


## Параметры


cx. Координата X левого верхнего
 угла прямоугольника;


cy. Координата Y левого верхнего
 угла прямоугольника;


width. Ширина прямоугольника;


height. Высота прямоугольника;


drawFill. Определяет, будет
 ли залит прямоугольник. Если для параметра установлено значение true, то прямоугольник будет залит;


drawStroke. Определяет, будут
 ли отрисованы границы прямоугольника. Если для параметра установлено значение
 true, то границы будут отрисованы.


## Описание


Метод drawRect рисует прямоугольник
 на канве.


## Комментарии


Прямоугольник будет отображен на канве после вызова метода [Canvas.flush](Canvas.flush.htm).
