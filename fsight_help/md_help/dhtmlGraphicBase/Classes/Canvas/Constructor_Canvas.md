# Конструктор Canvas


## Синтаксис


PP.Ui.Canvas(settings);


## Параметры


settings. JSON-объект со значениями
 свойств компонента.


## Описание


Конструктор Canvas создает экземпляр
 компонента [Canvas](../../Components/Canvas/Canvas.htm).
