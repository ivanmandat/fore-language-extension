# Canvas.drawLine


## Синтаксис


drawLine(x1: Number, y1: Number, x2: Number, y2:
 Number);


## Параметры


x1. Координата X начальной
 точки линии;


y1. Координата Y начальной
 точки линии;


x2. Координата X конечной точки
 линии;


y2. Координата Y конечной точки
 линии.


## Описание


Метод drawLine рисует линию
 на канве.


## Комментарии


Линия будет отображена на канве после вызова метода [Canvas.flush](Canvas.flush.htm).
