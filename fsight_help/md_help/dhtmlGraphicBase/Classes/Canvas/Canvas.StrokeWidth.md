# Canvas.StrokeWidth


## Синтаксис


StrokeWidth: Number;


## Описание


Свойство StrokeWidth устанавливает
 толщину границ элементов канвы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setStrokeWidth,
 а возвращается с помощью метода getStrokeWidth.
