# Конструктор Pen


## Синтаксис


PP.Pen(settings);


## Параметры


settings. JSON-объект со значениями
 свойств компонента.


## Описание


Конструктор Pen создает экземпляр
 класса [Pen](Pen.htm).

Конструктор предназначен для внутреннего использования.
