# Pen.Style


## Синтаксис


Style: [PP.PenStyle](dhtmlCommon.chm::/Enums/PenStyle.htm);


## Описание


Свойство Style определяет стиль
 отображения границы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setStyle,
 а возвращается с помощью метода getStyle.
