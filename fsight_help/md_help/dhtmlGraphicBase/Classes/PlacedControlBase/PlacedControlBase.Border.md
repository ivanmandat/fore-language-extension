# PlacedControlBase.Border


## Синтаксис


Border: PP.Border;


## Описание


Свойство Border определяет границы визуального элемента управления.


## Комментарии


Значение свойства устанавливается с помощью метода setBorder, а возвращается с помощью метода getBorder. Из JSON значение свойства установить нельзя.


Метод setBorder принимает следующие параметры: value - объект границ, state - состояние элемента управления.
