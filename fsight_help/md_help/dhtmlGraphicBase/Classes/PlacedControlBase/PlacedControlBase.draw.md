# PlacedControlBase.draw


## Синтаксис


draw(freeArea: PP.Rect, container: SVGGElement);


## Параметры


freeArea - Область в родительском контейнере, доступная для размещения;


container - Родительский контейнер.


## Описание


Метод draw рисует текущий визуальный элемент управления.
