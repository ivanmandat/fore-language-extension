# PlacedControlBase.VerticalAlignment


## Синтаксис


VerticalAlignment: [PP.VerticalAlignment](dhtmlCommon.chm::/Enums/PP.VerticalAlignment.htm);


## Описание


Свойство VerticalAlignment определяет
 вертикальное выравнивание элемента управления.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setVerticalAlignment,
 а возвращается с помощью метода getVerticalAlignment.**
