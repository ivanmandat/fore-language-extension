# CanvasHTMLRenderer.ScaleY


## Синтаксис


ScaleY: Number


## Описание


Свойство ScaleY определяет масштаб
 по оси Y рисуемой области.


## Комментарии


Значение свойства устанавливается из JSON или при помощи метода setScaleY, а возвращается при помощи
 метода getScaleY.
