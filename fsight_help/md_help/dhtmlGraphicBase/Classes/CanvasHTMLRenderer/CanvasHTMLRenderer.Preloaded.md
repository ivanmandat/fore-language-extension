# CanvasHTMLRenderer.Preloaded


## Синтаксис


Preloaded: function(sender, args)


## Параметры


sender. Источник события;


args. Информация
 о событии. Аргументы: Images
 - загруженные изображения.


## Описание


Событие Preloaded наступает
 после загрузки всех изображений.
