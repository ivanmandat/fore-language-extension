# CanvasHTMLRenderer.setScale


## Синтаксис


render(value_x: Number, y: Number);


## Параметры


value_x. Величина масштаба
 по оси X;


y. Величина масштаба по оси Y.


## Описание


Метод setScale устанавливает
 масштаб рисуемой области.


## Комментарии


Если не установить значение параметра y,
 тогда по обоим осям будет установлено значение value_x.
