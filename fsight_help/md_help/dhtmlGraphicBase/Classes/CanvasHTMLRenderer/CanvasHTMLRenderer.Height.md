# CanvasHTMLRenderer.Height


## Синтаксис


Height: Number


## Описание


Свойство Height определяет высоту
 рисуемой области.


## Комментарии


Значение свойства устанавливается из JSON или при помощи метода setHeight, а возвращается при помощи
 метода getHeight.
