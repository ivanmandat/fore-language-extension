# CanvasHTMLRenderer.Rendered


## Синтаксис


Rendered: function(sender, args)


## Параметры


sender. Источник события;


args. Информация
 о событии. Аргументы: Canvas
 - область с изображением, Image
 - адрес картинки.


## Описание


Событие Rendered наступает после
 завершения рисования.
