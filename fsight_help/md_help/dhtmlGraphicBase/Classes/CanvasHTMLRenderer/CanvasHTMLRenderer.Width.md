# CanvasHTMLRenderer.Width


## Синтаксис


Width: Number


## Описание


Свойство Width определяет ширину
 рисуемой области.


## Комментарии


Значение свойства устанавливается из JSON или при помощи метода setWidth, а возвращается при помощи
 метода getWidth.
