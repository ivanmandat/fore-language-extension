# CanvasHTMLRenderer.ScaleX


## Синтаксис


ScaleX: Number


## Описание


Свойство ScaleX определяет масштаб
 по оси X рисуемой области.


## Комментарии


Значение свойства устанавливается из JSON или при помощи метода setScaleX, а возвращается при помощи
 метода getScaleX.
