# CanvasHTMLRenderer.TaintTest


## Синтаксис


TaintTest: Boolean


## Описание


Свойство TaintTest определяет
 запуск проверочного теста.


## Комментарии


Допустимые значения:


	- true.
	 Проверочный тест будет запущен (по умолчанию);


	- false.
	 Проверочный тест запущен не будет.


Значение свойства устанавливается из JSON и с помощью метода setTaintTest,
 а возвращается с помощью метода getTaintTest.
