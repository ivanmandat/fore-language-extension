# CanvasHTMLRenderer.RecursiveBoundsCalc


## Синтаксис


RecursiveBoundsCalc: Boolean


## Описание


Свойство RecursiveBoundsCalc
 определяет автоматический расчет размера и позиции DOM-дерева.


## Комментарии


Допустимые значения:


	- true.
	 Будет произведен автоматический расчет размера и позиции;


	- false.
	 Автоматический расчет размера и позиции производиться не будет (по
	 умолчанию).


Значение свойства устанавливается из JSON и с помощью метода setRecursiveBoundsCalc,
 а возвращается с помощью метода getRecursiveBoundsCalc.
