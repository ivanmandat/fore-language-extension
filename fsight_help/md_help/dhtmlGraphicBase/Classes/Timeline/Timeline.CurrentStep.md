# Timeline.CurrentStep


## Синтаксис


CurrentStep: Number


## Описание


Свойство CurrentStep**
 устанавливает и возвращает индекс текущего шага анимации.


## Комментарии


Значение свойства устанавливается с помощью метода setCurrentStep,
 а возвращается с помощью метода getCurrentStep.
 Из JSON значение задать нельзя.


Нумерация шагов анимации начинается с 0.


По умолчанию свойство равно значению 0.
