# Timeline.StepsVisibility


## Синтаксис


StepsVisibility: Array


## Описание


Свойство StepsVisibility** устанавливает и возвращает массив признаков видимости шагов анимации.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setStepsVisibility, а возвращается с помощью метода getStepsVisibility.


Свойство содержит массив значений типа Boolean. Индексы элементов данного массива соответствуют шагам анимации в массиве, который хранится в свойстве [Timeline.StepsNames](Timeline.StepsNames.htm). Если значение элемента равно true, то соответствующий шаг будет отображён, если false, то не будет.
