# Timeline.LastIndexAsInitial


## Синтаксис


LastIndexAsInitial: Boolean;


## Описание


Свойство LastIndexAsInitial
 определяет, первый или последний шаг временной шкалы выбран начальным.


## Комментарии


Значение свойства устанавливается из JSON, возвращается с помощью метода
 getLastIndexAsInitial.


Допустимые значения:


	- True. Значение по умолчанию.
	 Начальным шагом определяется последний шаг временной шкалы;


	- False. Начальным шагом
	 определяется первый шаг временной шкалы.


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [Timeline](Timeline.htm) с наименованием «timeline» (см. «[Пример создания
 компонента Timeline](../../Components/Timeline/Timeline_Example.htm)» ):


// Определим, какой шаг временной шкалы выбран начальным
console.debug(timeline.getLastIndexAsInitial())
В результате выполнения примера в окно консоли было выведено, какой
 шаг временной шкалы был выбран начальным - первый или последний.
