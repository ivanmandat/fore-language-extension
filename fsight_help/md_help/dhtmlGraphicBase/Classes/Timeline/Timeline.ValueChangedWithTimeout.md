# Timeline.ValueChangedWithTimeout


## Синтаксис


ValueChangedWithTimeout: Boolean


## Описание


Свойство ValueChangedWithTimeout** устанавливает и возвращает признак того, что текущий шаг временной шкалы изменяется по истечению времени.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setValueChangedWithTimeout, а возвращается с помощью метода getValueChangedWithTimeout.


Если свойство содержит значение true, то текущий шаг временной шкалы изменяется по истечению времени, в противном случае - значение false.
