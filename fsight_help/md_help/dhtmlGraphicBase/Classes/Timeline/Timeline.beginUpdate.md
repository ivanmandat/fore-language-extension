# Timeline.beginUpdate


## Синтаксис


beginUpdate();


## Описание


Метод beginUpdate отключает
 перерисовку временной шкалы до тех пор, пока не будет вызван метод [endUpdate](Timeline.endUpdate.htm).


## Пример


Пример использования метода приведен в описании метода [TimeLine.removeTick](TimeLine.removeTick.htm).
