# Timeline.StepDuration


## Синтаксис


StepDuration: Number


## Описание


Свойство StepDuration** устанавливает и возвращает длительность шага анимации в миллисекундах.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setStepDuration, а возвращается с помощью метода getStepDuration.


По умолчанию свойство равно 1000.
