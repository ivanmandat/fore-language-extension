# Timeline.getIsStarted


## Синтаксис


getIsStarted();


## Описание


Метод getIsStarted** возвращает признак того, что воспроизведение анимации запущено.


## Комментарии


Метод возвращает значение true, если воспроизведение анимации было запущено, иначе - значение false.
