# Timeline.Inverse


## Синтаксис


Inverse: Boolean


## Описание


Свойство Inverse определяет
 направление шагов временной шкалы.


## Комментарии


Допустимые значения:


	- true. Шаги расположены
	 в обратном направлении;


	- false. По умолчанию. Шаги расположены в прямом
	 направлении.


Значение свойства устанавливается из JSON и с помощью метода setInverse,
 а возвращается с помощью метода getInverse.
