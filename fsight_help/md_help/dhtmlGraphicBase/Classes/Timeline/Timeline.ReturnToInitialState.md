# Timeline.ReturnToInitialState


## Синтаксис


ReturnToInitialState: Boolean


## Описание


Свойство ReturnToInitialState** устанавливает признак возврата временной шкалы в начальное состояние.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setReturnToInitialState, а возвращается с помощью метода getReturnToInitialState.


В начальном состоянии временной шкалы текущим является шаг анимации с индексом 0.


Если свойство равно значению true, то после завершения обхода временной шкалы, текущим снова будет шаг анимации с индексом 0. Если оно равно значению false, то при завершении анимации текущим будет последний шаг шкалы.
