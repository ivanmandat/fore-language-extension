# Timeline.ValueChanged


## Синтаксис


ValueChanged: function(sender, args)


## Параметры


sender. Источник события;


args. Информация о событии.
 Доступный аргумент: TickIndex
 - индекс текущего шага временной шкалы; ContinueAnimation
 - признак продолжения анимации: true
 - анимация продолжается, false
 - анимация остановлена.


## Описание


Событие ValueChanged**
 наступает после смены текущего шага временной шкалы при воспроизведении
 анимации или выбора шага кнопкой мыши.
