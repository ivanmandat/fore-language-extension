# Timeline.TickIndex


## Синтаксис


TickIndex: Number


## Описание


Свойство TickIndex** устанавливает и возвращает индекс текущего шага анимации.


## Комментарии


Данное свойство полностью аналогично свойству [Timeline.CurrentStep](Timeline.CurrentStep.htm).
