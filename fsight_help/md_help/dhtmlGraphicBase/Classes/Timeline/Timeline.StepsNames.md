# Timeline.StepsNames


## Синтаксис


StepsNames: Array


## Описание


Свойство StepsNames** устанавливает и возвращает массив наименований шагов анимации.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setStepsNames, а возвращается с помощью метода getStepsNames.


Свойство содержит массив значений типа String.
