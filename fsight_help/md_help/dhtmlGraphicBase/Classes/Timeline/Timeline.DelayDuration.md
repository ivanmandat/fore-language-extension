# Timeline.DelayDuration


## Синтаксис


DelayDuration: Number


## Описание


Свойство DelayDuration** устанавливает и возвращает длительность задержки анимации в миллисекундах.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDelayDuration, а возвращается с помощью метода getDelayDuration.


По умолчанию свойство равно 500.
