# Конструктор LegendToolTip


## Синтаксис


PP.Ui.LegendToolTip(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор LegendToolTip создает экземпляр класса [LegendToolTip](LegendToolTip.htm).
