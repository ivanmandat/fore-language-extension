# LegendToolTip.LTRB


## Синтаксис


LTRB: [PP.LTRB](dhtmlCommon.chm::/Enums/LTRB.htm);


## Описание


Свойство LTRB определяет
 поворот всплывающей подсказки.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLTRB,
 а возвращается с помощью метода getLTRB.


Метод setLTRB принимает следующие
 параметры: value - направление
 поворота, reflect -
 признак противоположного отражения.
