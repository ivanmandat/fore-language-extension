# MiniMap.ButtonsHeight


## Синтаксис


ButtonsHeight: Number


## Описание


Свойство ButtonsHeight содержит высоту кнопок панели управления масштабом.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setButtonsHeight, а возвращается с помощью метода getButtonsHeight.
