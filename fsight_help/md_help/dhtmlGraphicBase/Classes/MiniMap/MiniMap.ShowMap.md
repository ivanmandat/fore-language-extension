# MiniMap.ShowMap


## Синтаксис


ShowMap: Boolean


## Описание


Свойство ShowMap определяет отображение карты.


## Комментарии


Допустимые значения:


-
true. Карта будет отображена;


-
false. Карта отображена не будет.


Значение свойства устанавливается из JSON и с помощью метода setShowMap, а возвращается с помощью метода getShowMap.
