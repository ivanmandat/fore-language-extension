# MiniMap.KeepInitialBounds


## Синтаксис


KeepInitialBounds: Boolean


## Описание


Свойство KeepInitialBounds определяет сохранение начальных границ карты.


## Комментарии


Допустимые значения:


-
true. Начальные границы карты будут сохранены;


-
false. Начальные границы карты сохранены не будут.


Значение свойства устанавливается из JSON и с помощью метода setKeepInitialBounds, а возвращается с помощью метода getKeepInitialBounds.


Свойство находится в разработке.
