# Конструктор MiniMap


## Синтаксис


PP.Ui.MiniMap(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор MiniMap создает экземпляр класса [MiniMap](MiniMap.htm).
