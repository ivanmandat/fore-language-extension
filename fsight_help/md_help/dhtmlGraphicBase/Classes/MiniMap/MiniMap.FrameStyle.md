# MiniMap.FrameStyle


## Синтаксис


FrameStyle: PP.[Style](dhtmlCommon.chm::/Classes/PP/Style/Style.htm)


## Описание


Свойство FrameStyle содержит
 стиль области, в которой отображается видимая часть.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setFrameStyle,
 а возвращается с помощью метода getFrameStyle.


Свойство находится в разработке.
