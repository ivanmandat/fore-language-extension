# MiniMap.KeepProportions


## Синтаксис


KeepProportions: Boolean


## Описание


Свойство KeepProportions определяет сохранение соотношений сторон изображения при изменении ширины, высоты или после получения нового изображения.


## Комментарии


Допустимые значения:


-
true. Соотношение сторон изображения будет сохранено при изменении высоты, ширины или после получения нового изображения;


-
false. Соотношение сторон изображения не будет сохранено при изменении высоты, ширины или после получения нового изображения.


Значение свойства устанавливается из JSON и с помощью метода setKeepProportions, а возвращается с помощью метода getKeepProportions.
