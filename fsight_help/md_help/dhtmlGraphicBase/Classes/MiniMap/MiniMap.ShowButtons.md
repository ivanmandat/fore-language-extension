# MiniMap.ShowButtons


## Синтаксис


ShowButtons: Boolean


## Описание


Свойство ShowButtons определяет отображение кнопок панели управления масштабом.


## Комментарии


Допустимые значения:


-
true. Кнопки управления масштабом будут отображены;


-
false. Кнопки управления масштабом отображены не будут.


Значение свойства устанавливается из JSON и с помощью метода setShowButtons, а возвращается с помощью метода getShowButtons.
