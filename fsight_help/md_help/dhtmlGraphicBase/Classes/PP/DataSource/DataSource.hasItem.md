# DataSource.hasItem


## Синтаксис


hasItem(id: String);


## Параметры


id. Идентификатор ряда.


## Описание


Метод hasItem определяет, присутствуют ли данные ряда с указанным идентификатором.


## Комментарии


Метод возвращает значение типа Boolean.


Если метод возвращает значение true, то данные присутствуют, иначе - отсутствуют.
