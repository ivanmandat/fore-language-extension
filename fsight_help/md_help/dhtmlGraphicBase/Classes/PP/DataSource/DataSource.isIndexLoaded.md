# DataSource.isIndexLoaded


## Синтаксис


isIndexLoaded(index: Number);


## Параметры


index. Индекс элемента.


## Описание


Метод isIndexLoaded определяет, загружены ли данные для элемента с указанным индексом.


## Комментарии


Если данные загружены, то метод возвращает значение true, иначе - false.
