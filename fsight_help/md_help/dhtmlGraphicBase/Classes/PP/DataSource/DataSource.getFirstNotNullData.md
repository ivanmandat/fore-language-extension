# DataSource.getFirstNotNullData


## Синтаксис


getFirstNotNullData(dimAttrId);


## Параметры


dimAttrId. Идентификатор атрибута.


## Описание


Метод getFirstNotNullData возвращает первое не пустое значение указанного атрибута.


## Комментарии


Метод возвращает значение типа Number.
