# Конструктор DataSource


## Синтаксис


PP.DataSource(settings);


## Параметры


settings. JSON-объект со свойствами
 создаваемого экземпляра класса.


## Описание


Конструктор DataSource создает
 экземпляр класса [DataSource](DataSource.htm).
