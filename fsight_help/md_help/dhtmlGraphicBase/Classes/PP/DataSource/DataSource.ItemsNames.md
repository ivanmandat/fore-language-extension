# DataSource.ItemsNames


## Синтаксис


ItemsNames: Object


## Описание


Свойство ItemsNames определяет объект, содержащий имена рядов данных.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setItemsNames, а возвращается с помощью метода getItemsNames.**


Объект содержит набор идентификаторов, которым присвоены определенные имена.
