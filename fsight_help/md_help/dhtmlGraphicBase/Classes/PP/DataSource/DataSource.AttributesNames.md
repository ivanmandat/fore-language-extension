# DataSource.AttributesNames


## Синтаксис


AttributesNames: Object


## Описание


Свойство AttributesNames определяет объект, содержащий имена атрибутов источника данных.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setAttributesNames, а возвращается с помощью метода getAttributesNames.**


Объект содержит набор идентификаторов, которым присвоены определенные имена.
