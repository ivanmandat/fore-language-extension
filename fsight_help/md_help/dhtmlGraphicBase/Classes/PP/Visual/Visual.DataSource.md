# Visual.DataSource


## Синтаксис


DataSource: PP.DataSource;


## Описание


Свойство DataSource определяет источник данных для объекта их сопоставления.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDataSource.
