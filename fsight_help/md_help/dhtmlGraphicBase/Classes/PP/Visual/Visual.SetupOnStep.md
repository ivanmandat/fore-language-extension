# Visual.SetupOnStep


## Синтаксис


SetupOnStep: Boolean;


## Описание


Свойство SetupOnStep управляет разрешением на обновление настроек сопоставления данных при смене индекса.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSetupOnStep.


Если свойство содержит значение true, то при смене индекса, по которому возвращаются данные из измерения, будут обновлены настройки сопоставления данных, иначе (по умолчанию) - не будут.
