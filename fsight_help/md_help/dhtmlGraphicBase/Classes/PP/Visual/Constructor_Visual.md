# Конструктор Visual


## Синтаксис


PP.Visual(settings: Object);


## Параметры


settings. JSON-объект со значениями свойств экземпляра класса.


## Описание


Конструктор Visual создаёт экземпляр класса [Visual](Visual.htm).

Конструктор предназначен для внутреннего использования.
