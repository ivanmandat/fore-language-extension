# CoordinateTransformer.IncludeZero


## Синтаксис


IncludeZero: Boolean;


## Описание


Свойство IncludeZero определяет
 признак того, разрешено ли использование нуля при преобразовании данных.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIncludeZero,
 а возвращается с помощью метода getIncludeZero.


Допустимые значения:


	- true.
	 Использование нуля разрешено;


	- false.
	 Использование нуля запрещено (по умолчанию).
