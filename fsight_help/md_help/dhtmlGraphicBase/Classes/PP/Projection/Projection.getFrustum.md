# Projection.getFrustum


## Синтаксис


getFrustum(rect: PP.[Rect](dhtmlCommon.chm::/Classes/PP/Rect/Rect.htm));


## Параметры


rect. Прямоугольник.


## Описание


Метод getFrustum предназначен
 для получения усеченной пирамиды в мировых координатах из прямоугольника
 в экранных.
