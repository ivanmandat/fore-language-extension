# DataMapping.AutoSetup


## Синтаксис


AutoSetup: Boolean;


## Описание


Свойство AutoSetup определяет
 признак автоматического сопоставления данных.


## Комментарии


Значение свойства устанавливается из JSON.


Допустимые значения:


	- True. Данные из источника
	 будут автоматически сопоставляться элементам визуализатора;


	- False. Данные из источника
	 не будут автоматически сопоставляться элементам визуализатора.
