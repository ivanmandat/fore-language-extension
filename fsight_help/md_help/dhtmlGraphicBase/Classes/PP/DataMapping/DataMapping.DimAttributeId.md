# DataMapping.DimAttributeId


## Синтаксис


DimAttributeId: String;


## Описание


Свойство DimAttributeId определяет идентификатор атрибута измерения, по которому выбираются данные из источника данных.


## Комментарии


Значение свойства устанавливается из JSON и с помощью свойства setDimAttributeId.
