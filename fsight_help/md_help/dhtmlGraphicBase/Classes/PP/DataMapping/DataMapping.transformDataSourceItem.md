# DataMapping.transformDataSourceItem


## Синтаксис


transformDataSourceItem(data: Number);


## Параметры


data. Данные из источника.


## Описание


Метод transformDataSourceItem
 преобразует данные из источника в элемент шкалы, обработанные данные или
 константу в зависимости от [типа преобразования
 данных](DataMapping.Type.htm).


## Комментарии


Возвращаемое значение зависит от типа преобразования данных, так при
 значении [PP.DataMappingType](../../../Enums/PP.DataMappingType.htm).Scale
 метод возвратит значение типа [PP.SolidColorBrush](dhtmlCommon.chm::/Classes/PP/SolidColorBrush/SolidColorBrush.htm).
