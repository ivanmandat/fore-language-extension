# Конструктор DataMapping


## Синтаксис


PP.DataMapping(settings: Object);


## Параметры


settings. JSON-объект со свойствами создаваемого экземпляра класса.


## Описание


Конструктор DataMapping создает экземпляр класса [DataMapping](DataMapping.htm).


## Пример


Конструктор предназначен для внутреннего использования.
