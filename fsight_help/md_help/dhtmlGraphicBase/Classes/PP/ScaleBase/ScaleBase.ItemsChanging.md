# ScaleBase.ItemsChanging


## Синтаксис


ItemsChanging: PP.Delegate | function (sender, args)


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие ItemsChanging наступает при установке [коллекции элементов](ScaleBase.Items.htm) шкалы или при установке [элемента шкалы, соответствующего отсутствию данных](ScaleBase.NoData.htm).


## Пример


Пример использования события приведен в описании свойства [ScaleBase.NoData](ScaleBase.NoData.htm).
