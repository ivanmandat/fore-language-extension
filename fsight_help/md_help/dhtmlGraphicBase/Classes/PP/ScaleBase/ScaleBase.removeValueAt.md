# ScaleBase.removeValueAt


## Синтаксис


removeValueAt(index: Number);


## Параметры


index. Индекс значения границ интервалов шкалы.


## Описание


Метод removeItemAt удаляет [значение границ интервалов шкалы](ScaleBase.Values.htm) по указанному индексу.
