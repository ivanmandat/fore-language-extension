# ScaleBase.RecalcValues


## Синтаксис


RecalcValues: Boolean


## Описание


Свойство RecalcValues определяет, нужно ли пересчитывать коллекцию значений шкалы.


## Комментарии


Если установлено значение true, то коллекция значений шкалы будет пересчитана, иначе - не будет. Пересчет коллекции значений интервалов нужен, например, при изменении количества элементов шкалы. По умолчанию установлено значение f
