# ScaleBase.EnableEquality


## Синтаксис


EnableEquality: Boolean


## Описание


Свойство
 EnableEquality определяет, учитываются
 ли при построении шкалы равенства или шкала строится исключительно по
 промежуткам.


## Комментарии


Возможные
 значения:


	- true
	 - учитываются равенства;


	- false
	 - строится только по промежуткам.


По
 умолчанию установлено значение false.


Примечание.
 На данный момент нет возможности построения шкалы по равенствам, то есть
 при установке для свойства значений true
 или false шкала строится только
 по промежуткам.
