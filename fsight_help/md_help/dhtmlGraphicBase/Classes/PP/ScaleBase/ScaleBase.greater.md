# ScaleBase.greater


## Синтаксис


greater (index, isInactiveItem);


## Параметры


index. Индекс значения;


isInactiveItem. Определяет, использовать ли [набор данных для подсветки](ScaleBase.ActiveItems.htm).


## Описание


Метод greater возвращает [элемент шкалы](ScaleBase.Items.htm) для значения большего, чем значение, соответствующее заданному индексу.


## Комментарии


Возвращает null, если значение с заданным индексом является максимальным.
