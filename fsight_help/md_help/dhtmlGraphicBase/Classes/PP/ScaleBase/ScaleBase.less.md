# ScaleBase.less


## Синтаксис


less (index, isInactiveItem);


## Параметры


index. Индекс значения;


isInactiveItem. Определяет, использовать ли [набор данных для подсветки](ScaleBase.ActiveItems.htm).


## Описание


Метод less возвращает [элемент шкалы](ScaleBase.Items.htm) для значения меньшего, чем [значение границ интервалов шкалы](ScaleBase.Values.htm) с указанным индексом.


## Комментарии


Возвращает NULL, если значение границ интервалов шкалы с заданным индексом является минимальным.
