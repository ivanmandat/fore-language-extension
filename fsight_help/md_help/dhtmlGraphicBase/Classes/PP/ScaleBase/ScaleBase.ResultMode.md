# ScaleBase.ResultMode


## Синтаксис


ResultMode: [PP.ScaleResultMode](../../../Enums/ScaleResultMode.htm);


## Описание


Свойство ResultMode определяет
 типы возвращаемых значений шкалы визуализатора.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setResultMode,
 а возвращается с помощью метода getResultMode.


По умолчанию значение свойства равно PP.ScaleResultMode.Intervals.
