# Конструктор ScaleBase


## Синтаксис


PP.ScaleBase(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор LegendBase создает экземпляр класса [ScaleBase](ScaleBase.htm).

Конструктор предназначен для внутреннего использования.
