# ScaleBase.Items


## Синтаксис


Items: Arrow


## Описание


Свойство Items устанавливает
 коллекцию элементов шкалы.


## Комментарии


Значением свойства может быть прямое перечисление элементов через пробел
 (Items=”10 12 15”) или перечисление идентификаторов, содержащихся в фигурных
 скобках (Items=”{solid1 solid2 solid3}”).
