# ScaleBase.Values


## Синтаксис


Values: Array


## Описание


Свойство Values устанавливает
 коллекцию значений границ интервалов шкалы.


## Комментарии


Значения устанавливаются в инвариантной культуре, разделенной пробелами.


При использовании свойства Values
 значения шкалы не рассчитываются автоматически на основании данных из
 источника.
