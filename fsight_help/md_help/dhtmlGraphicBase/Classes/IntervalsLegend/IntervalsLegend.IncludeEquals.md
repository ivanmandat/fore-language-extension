# IntervalsLegend.IncludeEquals


## Синтаксис


IncludeEquals: Boolean;


## Описание


Свойство IncludeEquals определяет
 признак необходимости отображения элементов легенды "ровно".


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIncludeEquals,
 а возвращается с помощью метода getIncludeEquals.**


Если свойство содержит значение True,
 то элементы "ровно" будут отображены, иначе - не будут.
