# IntervalsLegend.Mode


## Синтаксис


Mode: [PP.Ui.LegendMode](../../Enums/LegendMode.htm);


## Описание


Свойство Mode определяет
 режим отображения элементов легенды.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMode,
 а возвращается с помощью метода getMode.**
