# IntervalsLegend.MaxHeaderWidth


## Синтаксис


MaxHeaderWidth: Number;


## Описание


Свойство MaxHeaderWidth задает
 максимальную ширину верхнего и нижнего текста легенды в процентах.


## Комментарии


Значение свойства устанавливается из JSON.


Процент рассчитывается от ширины содержимого легенды если выполняются
 условия:


	- свойство [IntervalsLegend.AutoColumnsCount](IntervalsLegend.AutoColumnsCount.htm)
	 принимает значение False;


	- верхний и нижний текст легенды не
	 расположен по центру.
