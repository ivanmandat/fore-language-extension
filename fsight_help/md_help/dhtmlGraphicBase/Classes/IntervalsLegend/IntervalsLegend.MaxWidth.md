# IntervalsLegend.MaxWidth


## Синтаксис


MaxWidth: Number;


## Описание


Свойство MaxWidth задает максимальную
 ширину легенды в процентах.


## Комментарии


Значение свойства устанавливается из JSON.


Процент рассчитывается от ширины заданного размера легенды.
