# IntervalsLegend.CurrentPage


## Синтаксис


CurrentPage: Number;


## Описание


Свойство CurrentPage определяет
текущую страницу легенды.


## Комментарии


Значение свойства устанавливается с помощью метода setCurrentPage,
 а возвращается с помощью метода getCurrentPage.


По умолчанию значение свойства равно 0.
