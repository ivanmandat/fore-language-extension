# IntervalsLegend.ColorApplying


## Синтаксис


ColorApplying: [PP.Ui.IntervalsLegendColorApplying](../../Enums/IntervalsLegendColorApplying.htm);


## Описание


Свойство ColorApplying определяет
 режим цветовой визуализации интервальной легенды.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setColorApplying,
 а возвращается с помощью метода getColorApplying.
