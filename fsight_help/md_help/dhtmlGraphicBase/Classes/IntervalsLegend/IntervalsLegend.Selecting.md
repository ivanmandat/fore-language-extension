# IntervalsLegend.Selecting


## Синтаксис


Selecting: function(sender, args, timeout);


## Параметры


sender. Источник события;


args. Информация о событии;


timeout. Промежуток времени
 в миллисекундах, через который вызовется событие.


## Описание


Событие Selecting наступает
 при выделении элемента легенды.
