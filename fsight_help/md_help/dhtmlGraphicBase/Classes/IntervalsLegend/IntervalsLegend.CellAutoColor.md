# IntervalsLegend.CellAutoColor


## Синтаксис


CellAutoColor: Boolean;


## Описание


Свойство CellAutoColor определяет
 цвет шрифта элементов легенды.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setCellAutoColor,
 а возвращается с помощью метода getCellAutoColor.


Допустимые значения свойства:


	- True. Задает черный
	 цвет шрифта;


	- False. Значение по умолчанию.
	 Автоматически рассчитывает цвет шрифта на основе цвета заливки.
