# IntervalsLegend.Selected


## Синтаксис


Selected: function(sender, args, timeout);


## Параметры


sender. Источник события;


args. Информация о событии;


timeout. Промежуток времени
 в миллисекундах, через который вызовется событие.


## Описание


Событие Selected наступает после
 выделения элемента легенды.
