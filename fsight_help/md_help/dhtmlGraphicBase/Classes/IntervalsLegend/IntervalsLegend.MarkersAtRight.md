# IntervalsLegend.MarkersAtRight


## Синтаксис


MarkersAtRight: Boolean;


## Описание


Свойство MarkersAtRight определяет
 признак отображения маркеров элементов легенды справа от текста.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMarkersAtRight,
 а возвращается с помощью метода getMarkersAtRight.**


**Е**сли свойство содержит
 значение True, то маркеры легенды
 будут отображены справа от текста, иначе - слева от текста.
