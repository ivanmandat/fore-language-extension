# IntervalsLegend.Descending


## Синтаксис


Descending: Boolean;


## Описание


Свойство Descending определяет
 признак сортировки элементов легенды по убыванию.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDescending,
 а возвращается с помощью метода getDescending.**


Если свойство содержит значение True,
 то элементы легенды будут отсортированы по убыванию, иначе (по умолчанию)
 - по возрастанию.
