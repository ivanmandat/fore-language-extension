# IntervalsLegend.ColumnsCount


## Синтаксис


ColumnsCount: Number;


## Описание


Свойство ColumnsCount определяет
 количество столбцов элементов легенды.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setColumnsCount,
 а возвращается с помощью метода getColumnsCount. М**етод getColumnsCount
 возвращает количество видимых столбцов элементов легенды.


Количество столбцов можно установить в том случае, если метод [IntervalsLegend.AutoColumnsCount](IntervalsLegend.AutoColumnsCount.htm)
 принимает значение False.
