# IntervalsLegend.AutoColumnsCount


## Синтаксис


AutoColumnsCount: Boolean;


## Описание


Свойство AutoColumnsCount определяет
 автоматический расчет количества столбцов элементов легенды.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setAutoColumnsCount,
 а возвращается с помощью метода getAutoColumnsCount.


Допустимые значения:


	- True. По умолчанию.
	 Количество столбцов элементов легенды рассчитывается автоматически;


	- False. Автоматический
	 расчет количества столбцов элементов легенды отключен. Для определения
	 количества столбцов элементов легенды используйте свойство [IntervalsLegend.ColumnsCount](IntervalsLegend.ColumnsCount.htm).
