# IntervalsLegend.Items


## Синтаксис


ColorsNames: Array;


## Описание


Свойство Items определяет сопоставления
 названий элементов цветам элементов легенды.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setItems,
 а возвращается с помощью метода getItems.**


Элемент массива представляет собой JSON-объект со следующими полями:
 Text - название элемента, Color - цвет элемента.
