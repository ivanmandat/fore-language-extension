# IntervalsLegend.CellPadding


## Синтаксис


CellPadding: String || [PP.Thickness](dhtmlCommon.chm::/Classes/PP/Thickness/Thickness.htm);


## Описание


Свойство CellPadding определяет
 поля элементов легенды.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setCellPadding,
 а возвращается с помощью метода getCellPadding.
