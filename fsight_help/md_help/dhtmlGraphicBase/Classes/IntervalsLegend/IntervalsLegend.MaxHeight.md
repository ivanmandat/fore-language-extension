# IntervalsLegend.MaxHeight


## Синтаксис


MaxHeight: Number;


## Описание


Свойство MaxHeight задает максимальную
 высоту легенды в процентах.


## Комментарии


Значение свойства устанавливается из JSON.


Процент рассчитывается от высоты заданного размера легенды.
