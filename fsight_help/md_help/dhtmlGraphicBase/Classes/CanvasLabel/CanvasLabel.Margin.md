# CanvasLabel.Margin


## Синтаксис


Margin: Array


## Описание


Свойство Margin определяет внешние
 отступы для подписи на канве.


## Комментарии


Свойство содержит массив из четырёх элементов. Первый элемент соответствует
 отступ сверху, второй - справа, третий - снизу, четвёртый - слева.


Свойство находится в разработке.
