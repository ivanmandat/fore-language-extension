# CanvasLabel.AutoOrientation


## Синтаксис


AutoOrientation: Boolean;


## Описание


Свойство AutoOrientation управляет
 возможностью автоматической ориентации подписи на канве.


## Комментарии


Допустимые значения:


	- true. Автоматическое
	 определение ориентации подписи на канве включено (по умолчанию);


	- false. Автоматическое
	 определение ориентации подписи на канве отключено.


Под ориентацией подписи понимается её вертикальное или горизонтальное
 расположение на канве. Если автоматическое определение ориентации отключено,
 то используется значение свойства [CanvasLabel.Vertical](CanvasLabel.Vertical.htm),
 иначе - не используется.
