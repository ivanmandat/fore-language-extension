# CanvasLabel.draw


## Синтаксис


draw(drawLabel: Boolean, drawText: Boolean);


## Параметры


drawLabel. Признак отрисовки
 границы и заливки подписи. Допустимые значения:


	- true. Для подписи на
	 канве будут отрисованы граница и заливка (по умолчанию);


	- false. Для подписи на
	 канве граница и заливка отрисованы не будут.


drawText. Признак отрисовки
 текста. Допустимые значения:


	- true. Для подписи на
	 канве будет отрисован текст (по умолчанию);


	- false. Для подписи на
	 канве текст отрисован не будет.


## Описание


Метод draw отрисовывает подпись
 на канве.
