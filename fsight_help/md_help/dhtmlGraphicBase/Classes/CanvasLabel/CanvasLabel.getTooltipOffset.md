# CanvasLabel.getTooltipOffset


## Синтаксис


getTooltipOffset();


## Описание


Метод getTooltipOffset рассчитывает
 смещение всплывающей подсказки для подписи на канве.


## Комментарии


Метод возвращает значение типа Number.
