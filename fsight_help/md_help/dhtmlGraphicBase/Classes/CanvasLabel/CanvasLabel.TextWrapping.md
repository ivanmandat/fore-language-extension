# CanvasLabel.TextWrapping


## Синтаксис


TextWrapping: [PP.Ui.TextWrapping](dhtmlUi.chm::/Enums/TextWrapping.htm)


## Описание


Свойство TextWrapping определяет
 режим переноса текста подписи на канве на следующую строку.


## Комментарии


По умолчанию свойство содержит значение PP.Ui.TextWrapping.Hidden.
