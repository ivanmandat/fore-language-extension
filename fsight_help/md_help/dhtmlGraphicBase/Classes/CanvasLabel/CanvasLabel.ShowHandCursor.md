# CanvasLabel.ShowHandCursor


## Синтаксис


ShowHandCursor: Boolean


## Описание


Свойство ShowHandCursor определяет,
 будет ли при наведении на подпись канвы отображаться курсор мыши типа
 «указатель».


## Комментарии


Допустимые значения:


	- true. При наведении
	 на подпись канвы курсор мыши будет иметь вид «указатель» ();


	- false. Будет использоваться
	 курсор, принятый по умолчанию в системе (значение по умолчанию).


Свойство находится в разработке.
