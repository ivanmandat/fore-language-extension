# CanvasLabel.Position


## Синтаксис


Position: Object;


## Описание


Свойство Position определяет
 расположение и размер подписи на канве.


## Комментарии


Свойство содержит JSON-объект со следующими полями: Left - отступ от
 левого края канвы, Top - от верхнего, Width - ширина подписи, Height -
 высота.
