# CanvasLabel.Opacity


## Синтаксис


Opacity: Number


## Описание


Свойство Opacity определяет
 прозрачность заливки подписи на канве.


## Комментарии


Значение свойства устанавливается с помощью метода setOpacity
 и возвращается - с помощью метода getOpacity.
 Из JSON значение задать нельзя.


Принимает значения от 0 до 1.
