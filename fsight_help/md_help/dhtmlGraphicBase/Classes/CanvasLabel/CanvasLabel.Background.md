# CanvasLabel.Background


## Синтаксис


Background: [PP.SolidColorBrush](dhtmlCommon.chm::/Classes/PP/SolidColorBrush/SolidColorBrush.htm)


## Описание


Свойство Background устанавливает
 цвет заливки подписи на канве.


## Комментарии


Значение свойства устанавливается с помощью метода setBackground,
 возвращается - с помощью метода getBackground.
 Из JSON значение задать нельзя.
