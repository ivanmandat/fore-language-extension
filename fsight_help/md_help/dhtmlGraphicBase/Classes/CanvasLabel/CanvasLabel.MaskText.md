# CanvasLabel.MaskText


## Синтаксис


MaskText: String;


## Описание


Свойство MaskText определяет
 маску текста для подписи на канве.


## Комментарии


По умолчанию свойство содержит строку «{Name}».


Свойство находится в разработке.
