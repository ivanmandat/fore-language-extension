# Конструктор LegendBase


## Синтаксис


PP.Ui.LegendBase(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор LegendBase создает экземпляр класса [LegendBase](LegendBase.htm).
