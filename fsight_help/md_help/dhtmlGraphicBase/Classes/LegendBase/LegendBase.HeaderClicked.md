# LegendBase.HeaderClicked


## Синтаксис


HeaderClicked: function(sender, args, timeout);


## Параметры


sender. Источник события;


args. Информация о событии;


timeout. Промежуток времени в миллисекундах, через который вызовется событие.


## Описание


Событие HeaderClicked наступает после нажатия на верхний текст диаграммы.
