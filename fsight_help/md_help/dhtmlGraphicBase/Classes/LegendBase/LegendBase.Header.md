# LegendBase.Header


## Синтаксис


Header: [PP.Ui.LegendHeader](../LegendHeader/LegendHeader.htm);


## Описание


Свойство Header определяет верхний текст легенды.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHeader, а возвращается с помощью метода getHeader.**
