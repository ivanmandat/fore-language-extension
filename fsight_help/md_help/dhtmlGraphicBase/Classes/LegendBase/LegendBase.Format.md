# LegendBase.Format


## Синтаксис


Format: String;


## Описание


Свойство Format определяет формат значений элементов легенды по умолчанию.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setFormat, а возвращается с помощью метода getFormat.**
