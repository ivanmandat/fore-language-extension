# LegendBase.IsOverlap


## Синтаксис


IsOverlap: Boolean;


## Описание


Свойство IsOverlap определяет признак перекрытия легендой рабочей области визуализатора.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsOverlap, а возвращается с помощью метода getIsOverlap.**


Свойство содержит значение True, если легенда перекрывает рабочую область визуализатора, в противном случае - значение False.
