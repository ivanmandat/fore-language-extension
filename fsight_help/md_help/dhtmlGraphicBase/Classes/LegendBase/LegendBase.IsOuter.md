# LegendBase.IsOuter


## Синтаксис


IsOuter: Boolean;


## Описание


Свойство IsOuter определяет признак того, что легенда будет рисоваться извне.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsOuter, а возвращается с помощью метода getIsOuter.**


Если свойство содержит значение True, то в контексте визуализатора будет выделено место под внешнюю легенду, иначе (по умолчанию) - не будет.
