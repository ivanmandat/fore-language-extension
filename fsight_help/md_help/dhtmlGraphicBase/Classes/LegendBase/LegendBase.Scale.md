# LegendBase.Scale


## Синтаксис


Scale: PP.ScaleBase;


## Описание


Свойство Scale определяет шкалу легенды визуализатора.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setScale, а возвращается с помощью метода getScale.**
