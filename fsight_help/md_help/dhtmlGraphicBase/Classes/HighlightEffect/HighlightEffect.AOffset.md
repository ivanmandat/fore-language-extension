# HighlightEffect.AOffset


## Синтаксис


AOffset: Number


## Описание


Свойство AOffset определяет
 смещение значения прозрачности цвета в эффекте подсветки.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setAOffset,
 а возвращается с помощью метода getAOffset.**
