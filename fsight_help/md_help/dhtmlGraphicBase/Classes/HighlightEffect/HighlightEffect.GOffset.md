# HighlightEffect.GOffset


## Синтаксис


GOffset: Number


## Описание


Свойство GOffset определяет
 смещение зеленого компонента цвета в эффекте подсветки.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setGOffset,
 а возвращается с помощью метода getGOffset.**
