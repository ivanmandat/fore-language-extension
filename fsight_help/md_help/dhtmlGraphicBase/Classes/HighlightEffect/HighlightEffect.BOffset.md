# HighlightEffect.BOffset


## Синтаксис


BOffset: Number


## Описание


Свойство BOffset определяет
 смещение синего компонента цвета в эффекте подсветки.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setBOffset,
 а возвращается с помощью метода getBOffset.**
