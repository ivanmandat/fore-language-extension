# HighlightEffect.ROffset


## Синтаксис


ROffset: Number


## Описание


Свойство ROffset определяет
 смещение красного компонента цвета в эффекте подсветки.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setROffset,
 а возвращается с помощью метода getROffset.**
