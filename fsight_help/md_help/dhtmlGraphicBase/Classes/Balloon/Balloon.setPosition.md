# Balloon.setPosition


## Синтаксис


setPosition(x: Number, y: Number, freeArea: PP.Rect,
 color: [PP.Color](dhtmlCommon.chm::/Classes/PP/Color/Color.htm),
 borderColor: [PP.Color](dhtmlCommon.chm::/Classes/PP/Color/Color.htm))
 : [PP.Ui.BubbleTreeAlignment](dhtmlBubbleTree.chm::/Enums/BubbleTreeAlignment.htm);


## Параметры


x. Координата X;


y. Координата Y;


freeArea. Площадь в родительском
 элементе, доступная для размещения. После завершения метода из нее убирается
 площадь, занятая этим компонентом;


color. Цвет фона подсказки;


borderColor. Цвет границы подсказки.


## Описание


Метод setPosition устанавливает
 новую позицию подсказки.


## Комментарии


Метод возвращает сторону, с которой подсказка была расположена - член
 класса [PP.Ui.BubbleTreeAlignment](dhtmlBubbleTree.chm::/Enums/BubbleTreeAlignment.htm).
