# Balloon.TargetY


## Синтаксис


TargetY: Number;


## Описание


Свойство TargetY определяет
 координату Y точки привязки подсказки.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTargetY,
 а возвращается с помощью метода getTargetY.**
