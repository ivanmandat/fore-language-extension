# Balloon.TailLength


## Синтаксис


TailLength: Number;


## Описание


Свойство TailLength определяет
 длину хвоста подсказки.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTailLength,
 а возвращается с помощью метода getTailLength.**


**З**начение свойства задается
 в пикселях.
