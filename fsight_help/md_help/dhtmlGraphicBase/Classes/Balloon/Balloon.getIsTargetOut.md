# Balloon.IsTargetOut


## Синтаксис


IsTargetOut: Boolean;


## Описание


Свойство IsTargetOut определяет
 признак выхода подсказки за пределы свободной области.


## Комментарии


Значение свойства устанавливается из JSON, а возвращается с помощью
 метода getTailPosition.**


Свойство содержит значение true,
 если подсказка выходит за пределы свободной области, иначе - false.
