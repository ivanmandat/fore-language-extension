# Balloon.Offset


## Синтаксис


Offset: Number;


## Описание


Свойство Offset определяет смещение
 от точки привязки.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setOffset,
 а возвращается с помощью метода getOffset.**


**З**начение свойства задается
 в пикселях.
