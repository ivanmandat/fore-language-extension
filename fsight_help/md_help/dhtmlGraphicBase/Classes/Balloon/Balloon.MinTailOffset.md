# Balloon.MinTailOffset


## Синтаксис


MinTailOffset: Number;


## Описание


Свойство MinTailOffset определяет
 минимальное расстояние между подсказкой и её хвостом.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMinTailOffset,
 а возвращается с помощью метода getMinTailOffset.**


Значение свойства задается в пикселях.
