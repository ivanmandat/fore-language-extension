# Balloon.TargetX


## Синтаксис


TargetX: Number;


## Описание


Свойство TargetX определяет
 координату X точки привязки подсказки.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTargetX,
 а возвращается с помощью метода getTargetX.**
