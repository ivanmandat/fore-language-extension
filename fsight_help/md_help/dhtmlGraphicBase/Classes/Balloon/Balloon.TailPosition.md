# Balloon.TailPosition


## Синтаксис


TailPosition: Number;


## Описание


Свойство TailPosition определяет
 смещение хвоста подсказки.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTailPosition,
 а возвращается с помощью метода getTailPosition.**


**Метод** setTailPosition
 принимает следующие параметры: value
 - значение смещения, isAbsolute
 - признак расчета смещения в пикселях, а не в процентах, isInverted
 - признак расчета смещения от правой нижней стороны подсказки.


**По умолчанию свойство содержит
 значение в процентах - «0.3».**
