# CylinderGaugeValue.Fill


## Синтаксис


Fill: [PP.Brush](dhtmlCommon.chm::/Classes/PP/Brush/Brush.htm);


## Описание


Свойство Fill определяет заливку
 значения цилиндра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setFill,
 а возвращается с помощью метода getFill.
