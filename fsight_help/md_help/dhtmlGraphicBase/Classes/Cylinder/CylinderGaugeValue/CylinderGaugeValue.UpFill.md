# CylinderGaugeValue.UpFill


## Синтаксис


UpFill: [PP.Brush](dhtmlCommon.chm::/Classes/PP/Brush/Brush.htm);


## Описание


Свойство UpFill определяет заливку
 основания цилиндра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setUpFill,
 а возвращается с помощью метода getUpFill.
