# CylinderGaugeValue.StartValue


## Синтаксис


StartValue: Number;


## Описание


Свойство StartValue определяет
 координату нижнего основания цилиндра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setStartValue,
 а возвращается с помощью метода getStartValue.
