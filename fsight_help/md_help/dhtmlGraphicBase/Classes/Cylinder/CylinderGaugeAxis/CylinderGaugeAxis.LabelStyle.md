# CylinderGaugeAxis.LabelStyle


## Синтаксис


LabelStyle: [PP.Ui.ChartSVGLabel](dhtmlBubbleChart.chm::/Classes/BubbleChart/ChartSVGLabel/ChartSVGLabel.htm);


## Описание


Свойство LabelStyle определяет
 стиль подписей оси цилиндра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLabelStyle,
 а возвращается с помощью метода geLabelStyle.
