# CylinderGaugeAxis.MajorTicksCount


## Синтаксис


MajorTicksCount: Number;


## Описание


Свойство MajorTicksCount определяет
 количество основных делений оси цилиндра.


## Комментарии


Значение свойства устанавливается из JSON и возвращается с помощью метода
 getMajorTicksCount.
