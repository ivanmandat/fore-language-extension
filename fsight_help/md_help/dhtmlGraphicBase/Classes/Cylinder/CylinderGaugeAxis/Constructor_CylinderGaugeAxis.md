# Конструктор CylinderGaugeAxis


## Синтаксис


PP.Ui.CylinderGaugeAxis(settings);


## Параметры


settings. JSON-объект
 со значениями свойств класса.


## Описание


Конструктор CylinderGaugeAxis создает
 экземпляр класса [CylinderGaugeAxis](CylinderGaugeAxis.htm).
