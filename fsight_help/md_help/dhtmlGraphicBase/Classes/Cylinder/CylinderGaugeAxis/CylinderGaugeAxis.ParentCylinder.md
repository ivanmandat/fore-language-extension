# CylinderGaugeAxis.ParentCylinder


## Синтаксис


ParentCylinder: [PP.Ui.Cylinder](../Cylinder/Cylinder.htm);


## Описание


Свойство ParentCylinder определяет
 цилиндр, к которому принадлежит данная ось.


## Комментарии


Значение свойства устанавливается с помощью JSON, а возвращается с помощью
 метода getParentCylinder.
