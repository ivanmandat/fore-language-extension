# CylinderGaugeAxis.redraw


## Синтаксис


redraw(needClearMinorTicks: Boolean);


## Параметры


needClearMinorTicks. Признак
 необходимости полной перерисовки промежуточных делений оси цилиндра. Необязательный
 параметр. Допустимые значения:


	- true.
	 Необходима полная перерисовка промежуточных делений оси цилиндра;


	- false.
	 Полная перерисовка промежуточных делений оси цилиндра не требуется
	 (по умолчанию).


## Описание


Метод redraw перерисовывает
 ось цилиндра.
