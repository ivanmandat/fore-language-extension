# CylinderGaugeAxis.ShowUnitsForEachValue


## Синтаксис


ShowUnitsForEachValue: Boolean;


## Описание


Свойство ShowUnitsForEachValue
 определяет признак того, следует ли отображать единицу измерения для каждого
 основного деления оси цилиндра.


## Комментарии


Значение свойства устанавливается с помощью метода setShowUnitsForEachValue,
 а возвращается с помощью метода getShowUnitsForEachValue.


Допустимые значения:


	- true.
	 Единица измерения отображается для каждой основной деления оси цилиндра;


	- false.
	 Единица измерения отображается в конце оси (по умолчанию).
