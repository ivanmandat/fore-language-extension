# CylinderGaugeAxis.Position


## Синтаксис


Position: [PP.LTRB](dhtmlCommon.chm::/Enums/LTRB.htm);


## Описание


Свойство Position определяет
 позицию оси относительно цилиндра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setPosition,
 а возвращается с помощью метода getPosition.


Допустимые значения:


	- PP.LTRB.Left.
	 Ось расположена слева от цилиндра (по умолчанию);


	- PP.LTRB.Right.
	 Ось расположена справа от цилиндра.
