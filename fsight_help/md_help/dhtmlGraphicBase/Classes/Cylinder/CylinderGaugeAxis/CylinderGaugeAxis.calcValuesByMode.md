# CylinderGaugeAxis.calcValuesByMode


## Синтаксис


calcValuesByMode();


## Описание


Метод calcValuesByMode возвращает
 значения оси цилиндра, соответствующие режиму построения её делений.


## Комментарии


Режим построения делений оси устанавливается с помощью свойства [CylinderGaugeAxis.ValuesMode](CylinderGaugeAxis.ValuesMode.htm).
