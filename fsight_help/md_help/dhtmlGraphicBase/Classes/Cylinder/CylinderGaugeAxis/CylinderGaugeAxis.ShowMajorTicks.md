# CylinderGaugeAxis.ShowMajorTicks


## Синтаксис


ShowMajorTicks: Boolean;


## Описание


Свойство ShowMajorTicks определяет
 признак отображения основных делений оси цилиндра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setShowMajorTicks,
 а возвращается с помощью метода getShowMajorTicks.


Допустимые значения:


	- true.
	 Основные деления оси цилиндра отображаются (по умолчанию);


	- false.
	 Основные деления оси цилиндра не отображаются.
