# CylinderGaugeAxis.MajorTickStyle


## Синтаксис


MajorTickStyle: [PP.Ui.AxisTick](../AxisTick/AxisTick.htm);


## Описание


Свойство MajorTickStyle определяет
 стиль основных делений оси цилиндра.


## Комментарии


Значение свойства устанавливается с помощью метода setMajorTickStyle,
 а возвращается с помощью метода getMajorTickStyle.
