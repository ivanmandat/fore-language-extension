# CylinderGaugeAxis.MinorTickStyle


## Синтаксис


MinorTickStyle: [PP.Ui.AxisTick](../AxisTick/AxisTick.htm);


## Описание


Свойство MinorTickStyle определяет
 стиль промежуточных делений оси цилиндра.


## Комментарии


Значение свойства устанавливается с помощью метода setMinorTickStyle,
 а возвращается с помощью метода getMinorTickStyle.
