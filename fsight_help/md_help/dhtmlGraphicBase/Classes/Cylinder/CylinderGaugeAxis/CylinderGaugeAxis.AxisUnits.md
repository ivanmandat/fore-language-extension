# CylinderGaugeAxis.AxisUnits


## Синтаксис


AxisUnits: String;


## Описание


Свойство AxisUnits определяет
 единицу измерения оси цилиндра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setAxisUnits,
 а возвращается с помощью метода getAxisUnits.
