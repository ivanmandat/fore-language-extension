# CylinderGaugeAxis.ShowMinorTicks


## Синтаксис


ShowMinorTicks: Boolean;


## Описание


Свойство ShowMinorTicks определяет
 признак отображения промежуточных делений оси цилиндра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setShowMinorTicks,
 а возвращается с помощью метода getShowMinorTicks.


Допустимые значения:


	- true.
	 Промежуточные деления оси цилиндра отображаются (по умолчанию);


	- false.
	 Промежуточные деления оси цилиндра не отображаются.
