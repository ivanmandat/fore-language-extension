# CylinderGaugeAxis.MinorTicksStep


## Синтаксис


MinorTicksStep: Number;


## Описание


Свойство MinorTicksStep определяет
 шаг между промежуточными делениями в единицах оси.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMinorTicksStep,
 а возвращается с помощью метода getMinorTicksStep.


По умолчанию свойство равно 1.
