# Cylinder.Trends


## Синтаксис


Trends: Array;


## Описание


Свойство Trends определяет массив
 линий тренда цилиндра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTrends,
 а возвращается с помощью метода getTrends.


Метод getTrends содержит необязательный
 параметр index, устанавливающий
 индекс линии тренда в массиве. Если индекс задан, метод getTrends
 возвращает элемент массива по указанному индексу, в противном случае
 - метод возвращает весь массив линий тренда цилиндра.


Свойство содержит массив объектов типа [CylinderTrend](../CylinderTrend/CylinderTrend.htm).
