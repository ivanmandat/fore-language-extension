# Cylinder.UseAnimation


## Синтаксис


UseAnimation: Boolean;


## Описание


Свойство UseAnimation определяет
 признак использования анимации при отображении цилиндра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setUseAnimation,
 а возвращается с помощью метода getUseAnimation.


Допустимые значения:


	- true.
	 Анимация используется при отображении цилиндра;


	- false.
	 Анимация не используется при отображении цилиндра (по умолчанию).
