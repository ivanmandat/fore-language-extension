# Cylinder.FlaskHeight


## Синтаксис


FlaskHeight: Number;


## Описание


Свойство FlaskHeight определяет
 высоту цилиндра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setFlaskHeight,
 а возвращается с помощью метода getFlaskHeight.


Метод getFlaskHeight содержит
 необязательный параметр needRel, задающий признак того, что будет возвращено
 отношение высоты цилиндра к высоте области его построения. Допустимые
 значения данного параметра:


	- true. Возвращается отношение
	 высоты цилиндра к высоте области его построения;


	- false. По умолчанию.
	 Возвращается абсолютное значение высоты цилиндра.


По умолчанию свойство равно 0.
