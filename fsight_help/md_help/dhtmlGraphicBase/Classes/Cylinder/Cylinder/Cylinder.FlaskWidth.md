# Cylinder.FlaskWidth


## Синтаксис


FlaskWidth: Number;


## Описание


Свойство FlaskWidth определяет
 ширину цилиндра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setFlaskWidth,
 а возвращается с помощью метода getFlaskWidth.


Метод getFlaskWidth содержит
 необязательный параметр needRel,
 задающий признак того, что будет возвращено отношение ширины цилиндра
 к ширине области его построения. Допустимые значения параметра:


	- true. Возвращается отношение
	 ширины цилиндра к ширине области его построения;


	- false. По умолчанию.
	 Возвращается абсолютное значение ширины цилиндра.


По умолчанию свойство равно 0.
