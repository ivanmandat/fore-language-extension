# Cylinder.AnimationDuration


## Синтаксис


AnimationDuration: Number;


## Описание


Свойство AnimationDuration определяет
 длительность воспроизведения анимации цилиндра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setAnimationDuration,
 а возвращается с помощью метода getAnimationDuration.


Длительность анимации измеряется в миллисекундах. По умолчанию свойство
 равно 0.
