# Cylinder.IsRelativeDimensions


## Синтаксис


IsRelativeDimensions: Boolean;


## Описание


Свойство IsRelativeDimensions определяет,
 являются ли внутренние размеры элементов цилиндра относительными.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsRelativeDimensions,
 а возвращается с помощью метода getIsRelativeDimensions.


Допустимые значения:


	- true.
	 Внутренние размеры элементов цилиндра являются относительными (по
	 умолчанию);


	- false.
	 Внутренние размеры элементов цилиндра не являются относительными.
