# Конструктор CylinderHeader


## Синтаксис


PP.Ui.CylinderHeader(settings);


## Параметры


settings. JSON-объект со значениями
 свойств класса.


## Описание


Конструктор CylinderHeader создает
 экземпляр класса [CylinderHeader](CylinderHeader.htm).
