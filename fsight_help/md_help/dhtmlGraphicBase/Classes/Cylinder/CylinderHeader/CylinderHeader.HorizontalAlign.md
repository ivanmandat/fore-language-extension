# CylinderHeader.HorizontalAlign


## Синтаксис


HorizontalAlign: [PP.Ui.CylinderLabelAlign](../../../Enums/CylinderLabelAlign.htm);


## Описание


Свойство HorizontalAlign определяет
 горизонтальное выравнивание заголовка относительно цилиндра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHorizontalAlign,
 а возвращается с помощью метода getHorizontalAlign.
