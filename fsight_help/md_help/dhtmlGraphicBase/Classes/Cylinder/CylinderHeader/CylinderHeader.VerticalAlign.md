# CylinderHeader.VerticalAlign


## Синтаксис


VerticalAlign: [PP.Ui.CylinderLabelPosition](../../../Enums/CylinderLabelPosition.htm);


## Описание


Свойство VerticalAlign определяет
 вертикальное выравнивание заголовка относительно цилиндра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setVerticalAlign,
 а возвращается с помощью метода getVerticalAlign.
