# Конструктор CylinderTrend


## Синтаксис


PP.Ui.CylinderTrend(settings);


## Параметры


settings. JSON-объект со значениями
 свойств класса.


## Описание


Конструктор CylinderTrend создает
 экземпляр класса [CylinderTrend](CylinderTrend.htm).
