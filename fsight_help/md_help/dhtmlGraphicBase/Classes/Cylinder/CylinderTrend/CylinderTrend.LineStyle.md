# CylinderTrend.LineStyle


## Синтаксис


LineStyle: [PP.Ui.LineTypes](dhtmlUi.chm::/Enums/LineTypes.htm);


## Описание


Свойство LineStyle определяет
 стиль линии тренда цилиндра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLineStyle,
 а возвращается с помощью метода getLineStyle.


По умолчанию свойство имеет значение PP.Ui.LineTypes.Solid.
