# CylinderTrend.LabelStyle


## Синтаксис


LabelStyle: Object;


## Описание


Свойство LabelStyle определяет
 стиль подписи линии тренда.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLabelStyle,
 а возвращается с помощью метода getLabelStyle.


Свойство содержит JSON-объект с полем Text - текст подписи линии тренда.
