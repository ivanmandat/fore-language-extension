# CylinderTrend.Length


## Синтаксис


Length: Number;


## Описание


Свойство Length определяет длину
 линии тренда.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLength,
 а возвращается с помощью метода getLength.
