# AxisTick.setTickStyle


## Синтаксис


setTickStyle(obj: Object);


## Параметры


obj. Стиль деления оси. Параметр
 представляет собой JSON-объект, имеющий следующие поля: Length
 - длина деления, Border - параметры
 границы деления оси.


## Описание


Метод setTickStyle устанавливает
 стиль деления оси.
