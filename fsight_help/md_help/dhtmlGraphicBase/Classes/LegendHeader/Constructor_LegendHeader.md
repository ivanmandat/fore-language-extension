# Конструктор LegendHeader


## Синтаксис


PP.Ui.LegendHeader(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор LegendHeader создает экземпляр класса [LegendHeader](LegendHeader.htm).
