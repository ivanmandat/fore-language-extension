# SizeLegend.MaxRadius


## Синтаксис


MaxRadius: Number;


## Описание


Свойство MaxRadius определяет максимальный радиус маркера.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMaxRadius, а возвращается с помощью метода getMaxRadius.
