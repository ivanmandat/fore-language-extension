# SizeLegend.MinRadius


## Синтаксис


MinRadius: Number;


## Описание


Свойство MinRadius определяет минимальный радиус маркера.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMinRadius, а возвращается с помощью метода getMinRadius.
