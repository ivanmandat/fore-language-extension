# SizeLegend.MarkerColor


## Синтаксис


MarkerColor: [PP.Color](dhtmlCommon.chm::/Classes/PP/Color/Color.htm);


## Описание


Свойство MarkerColor определяет
 цвет маркеров.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMarkerColor,
 а возвращается с помощью метода getMarkerColor.
