# SizeLegend.MinValueFormat


## Синтаксис


MinValueFormat: String;


## Описание


Свойство MinValueFormat определяет формат записи минимального значения маркера.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMinValueFormat, а возвращается с помощью метода getMinValueFormat.
