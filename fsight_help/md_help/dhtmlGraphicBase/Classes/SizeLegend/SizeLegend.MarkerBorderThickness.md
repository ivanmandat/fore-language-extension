# SizeLegend.MarkerBorderThickness


## Синтаксис


MarkerBorderThickness: Number;


## Описание


Свойство MarkerBorderThickness определяет толщину границ маркеров.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMarkerBorderThickness, а возвращается с помощью метода getMarkerBorderThickness.
