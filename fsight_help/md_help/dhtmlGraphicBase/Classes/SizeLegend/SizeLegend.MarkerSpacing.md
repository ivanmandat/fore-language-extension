# SizeLegend.MarkerSpacing


## Синтаксис


MarkerSpacing: Number;


## Описание


Свойство MarkerSpacing определяет отступ маркеров друг от друга.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMarkerSpacing, а возвращается с помощью метода getMarkerSpacing.
