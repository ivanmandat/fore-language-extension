# SizeLegend.MaxValueFormat


## Синтаксис


MaxValueFormat: String;


## Описание


Свойство MaxValueFormat определяет формат записи максимального значения маркера.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMaxValueFormat, а возвращается с помощью метода getMaxValueFormat.
