# SizeLegend.TextSpacing


## Синтаксис


TextSpacing: Number;


## Описание


Свойство TextSpacing определяет отступ текста от маркеров.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTextSpacing, а возвращается с помощью метода getTextSpacing.
