# SpectrumLegend.TextSpacing


## Синтаксис


TextSpacing: Number;


## Описание


Свойство TextSpacing определяет расстояние от разделителей цветовых диапазонов [спектральной легенды](SpectrumLegend.htm) до текста.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTextSpacing, а возвращается с помощью метода getTextSpacing.
