# SpectrumLegend.ItemFormat


## Синтаксис


ItemFormat: String;


## Описание


Свойство ItemFormat определяет формат текста элементов легенды.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setItemFormat, а возвращается с помощью метода getItemFormat.
