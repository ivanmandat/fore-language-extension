# SpectrumLegend.Selecting


## Синтаксис


Selecting: function(sender, args, timeout);


## Параметры


sender. Источник события;


args. Информация о событии;


timeout. Промежуток времени в миллисекундах, через который вызовется событие.


## Описание


Событие Selecting наступает при выделении элемента [спектральной легенды](SpectrumLegend.htm).

Пример использования события представлен в содержимом раздела «Настройки спектральной легенды» на странице описания [конструктора SpectrumLegend](Constructor_SpectrumLegend.htm).
