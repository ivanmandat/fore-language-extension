# SpectrumLegend.MarkColor


## Синтаксис


MarkColor: [PP.Color](dhtmlCommon.chm::/Classes/PP/Color/Color.htm);


## Описание


Свойство MarkColor определяет
 цвет разделителей цветовых диапазонов [спектральной
 легенды](SpectrumLegend.htm).


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMarkColor,
 а возвращается с помощью метода getMarkColor.

 спектральной легенды» на странице описания [конструктора
 SpectrumLegend](Constructor_SpectrumLegend.htm).
