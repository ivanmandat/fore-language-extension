# SpectrumLegend.AreMarksVisible


## Синтаксис


AreMarksVisible: Boolean;


## Описание


Свойство AreMarksVisible определяет признак отображения разделителей цветовых диапазонов [спектральной легенды](SpectrumLegend.htm).


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setAreMarksVisible, а возвращается с помощью метода getAreMarksVisible.


Если свойство имеет значение true, разделители будут отображены, иначе - не будут.
