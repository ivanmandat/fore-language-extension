# SpectrumLegend.SpectrumSize


## Синтаксис


SpectrumSize: Number;


## Описание


Свойство SpectrumSize определяет высоту спектральной полосы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSpectrumSize, а возвращается с помощью метода getSpectrumSize.
