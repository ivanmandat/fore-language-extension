# SpectrumLegend.MarkThickness


## Синтаксис


MarkThickness: Number;


## Описание


Свойство MarkThickness определяет толщину разделителей цветовых диапазонов [спектральной легенды](SpectrumLegend.htm).


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMarkThickness, а возвращается с помощью метода getMarkThickness.


## Пример


Пример использования свойства представлен в содержимом раздела «Настройки спектральной легенды» на странице описания [конструктора SpectrumLegend](Constructor_SpectrumLegend.htm).
