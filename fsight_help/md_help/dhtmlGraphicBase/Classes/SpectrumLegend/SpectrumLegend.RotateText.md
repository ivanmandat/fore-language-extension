# SpectrumLegend.RotateText


## Синтаксис


RotateText: Boolean;


## Описание


Свойство RotateText определяет признак поворота текста элементов легенды при изменении выравниваний.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setRotateText, а возвращается с помощью метода getRotateText.


Если свойство имеет значение true, поворот текста будет осуществлен, иначе - не будет.
