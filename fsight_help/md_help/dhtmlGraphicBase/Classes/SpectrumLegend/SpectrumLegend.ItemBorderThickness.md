# SpectrumLegend.ItemBorderThickness


## Синтаксис


ItemBorderThickness: Number;


## Описание


Свойство ItemBorderThickness определяет толщину границы элементов легенды.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setItemBorderThickness, а возвращается с помощью метода getItemBorderThickness.
