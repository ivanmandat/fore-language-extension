# SpectrumLegend.ItemBorderColor


## Синтаксис


ItemBorderColor: [PP.Color](dhtmlCommon.chm::/Classes/PP/Color/Color.htm);


## Описание


Свойство ItemBorderColor определяет
 цвет границы элементов легенды.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setItemBorderColor,
 а возвращается с помощью метода getItemBorderColor.

 спектральной легенды» на странице описания [конструктора
 SpectrumLegend](Constructor_SpectrumLegend.htm).
