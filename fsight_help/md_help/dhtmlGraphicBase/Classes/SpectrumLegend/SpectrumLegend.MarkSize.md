# SpectrumLegend.MarkSize


## Синтаксис


MarkSize: Number;


## Описание


Свойство MarkSize определяет длину разделителей цветовых диапазонов [спектральной легенды](SpectrumLegend.htm).


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMarkSize, а возвращается с помощью метода getMarkSize.

Пример использования свойства представлен в содержимом раздела «Настройки спектральной легенды» на странице описания [конструктора SpectrumLegend](Constructor_SpectrumLegend.htm).
