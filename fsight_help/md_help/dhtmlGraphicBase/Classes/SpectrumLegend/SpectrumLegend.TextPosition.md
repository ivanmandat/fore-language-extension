# SpectrumLegend.TextPosition


## Синтаксис


TextPosition: [PP.Ui.SpectrumLegendTextPosition](../../Enums/SpectrumLegendTextPosition.htm);


## Описание


Свойство TextPosition определяет позицию текста элемента диаграммы относительно самого элемента.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTextPosition, а возвращается с помощью метода getTextPosition.
