## С чего начать


		 Сборка System
		 содержит интерфейсы, описывающие базовые конструкции языка Fore:


			- Интерфейсы, описывающие исключительные ситуации;


			- Интерфейсы для работы с консолью среды разработки;


			- Интерфейсы для работы с настройками региональных стандартов
			 операционной системы;


			- Интерфейс для работы со сборщиком мусора.


		Также в сборке реализован ряд общих интерфейсов, которые используются
		 в других сборках.


		Классы сборки реализуют типы данных языка Fore,
		 а также матрицы, на которых основана работа с многомерными данными
		 во всех инструментах продукта «Форсайт. Аналитическая платформа».
