# TaskContinuationOptions


## Описание


Перечисление TaskContinuationOptions
 содержит условия, при которых будет запущена на выполнение следующая задача.


Используется следующим методом:


	- [ITask.ContinueWith](../Interface/ITask/ITask.ContinueWith.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Поведение по
		 умолчанию. Задача выполняется асинхронно, независимо от того,
		 как выполнилась предшествующая.


		 65536
		 NotOnRanToCompletion.
		 Задача не выполняется, если предшествующая выполнилась успешно.


		 131072
		 NotOnFaulted. Задача
		 не выполняется, если предшествующая выполнилась с ошибкой.


		 196608
		 OnlyOnCanceled. Задача
		 выполняется, если предшествующая была отменена.


		 262144
		 NotOnCanceled. Задача
		 не выполняется, если предшествующая была отменена.


		 327680
		 OnlyOnFaulted. Задача
		 выполняется, если предшествующая выполнилась с ошибкой.


		 393216
		 OnlyOnRanToCompletion.
		 Задача выполняется, если предшествующая выполнилась успешно.


		 524288
		 ExecuteSynchronously.
		 Задача выполняется синхронно, в том же потоке, что и основная
		 программа, запускающая задачу.
