# SortDirection


## Описание


Перечисление SortDirection содержит
 направления сортировки, которые будут установлены в столбце.


Используется следующими свойствами и методами:


	- [IDimensionViewer.Sort](KeExtCtrls.chm::/Interface/IDimensionViewer/IDimensionViewer.Sort.htm);


	- [IMetaAttributeSetting.SortDirectionType](KeExtCtrls.chm::/Interface/IMetaAttributeSetting/IMetaAttributeSetting.SortDirectionType.htm);


	- [ITreeControl.Sort](modforms.chm::/interface/itreecontrol/itreecontrol.sort.htm);


	- [ITableFilterItem.Sort](tabsheet.chm::/interface/itablefilteritem/itablefilteritem.sort.htm);


	- [ITableFilterItem.SortState](tabsheet.chm::/interface/itablefilteritem/ITableFilterItem.SortState.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Ascending. Сортировка
		 по возрастанию значений.


		 1
		 Descending. Сортировка
		 по убыванию значений.


		 3
		 NoSort. Нет сортировки.
