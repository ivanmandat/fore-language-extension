# StringSplitOption


## Описание


Перечисление StringSplitOption
 содержит виды разделителя, используемого при разбиении символьных строк.


Используется следующим методом:


	- [String.Split](../Class/String/String.Split.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 NoEmpty. Исключать
		 пустые строки из результата. При разбиении символьной строки пустые
		 элементы, создаваемые при нахождении в строке двух и более подряд
		 идущих разделителей, в результирующий массив включены не будут.


		 2
		 Chars. Рассматривать
		 разделитель посимвольно. При разбиении символьной строки в качестве
		 разделителя будет рассматриваться каждый символ строки, указанной
		 в параметре Delimiters
		 метода [Split](../Class/String/String.Split.htm).


		 4
		 String. Рассматривать
		 разделитель единой строкой. При разбиении символьной строки в
		 качестве разделителя будет рассматриваться вся символьная строка,
		 указанная в параметре Delimiters
		 метода [Split](../Class/String/String.Split.htm).


Примечание.
 Комбинацию значений StringSplitOption.Chars и StringSplitOption.String
 использовать запрещено.
