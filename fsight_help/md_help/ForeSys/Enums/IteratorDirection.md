# IteratorDirection


## Описание


Перечисление IteratorDirection
 содержит направления перемещения в итераторе матрицы.


Используется следующим методом:


	- [IMatrixModelIterator.Move](../Interface/IMatrixModelIterator/IMatrixModelIterator.Move.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Перемещение не
		 осуществляется.


		 1
		 First. Переход к первому
		 элементу.


		 2
		 Next. Переход к следующему
		 элементу.


		 3
		 Previous. Переход к
		 предыдущему элементу.


		 4
		 Last. Переход к последнему
		 элементу.
