# RWMutexLockType


## Описание


Перечисление RWMutexLockType
 содержит типы блокировки, для которой предназначен мьютекс.


Используется следующими свойствами и методами:


	- [IRWMutex.LockType](../Interface/IRWMutex/IRWMutex.LockType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Блокировка отсутствует.


		 1
		 Read. Блокировка на
		 чтение.


		 2
		 Write. Блокировка на
		 запись.
