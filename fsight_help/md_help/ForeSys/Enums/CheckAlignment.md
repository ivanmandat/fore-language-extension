# CheckAlignment


## Описание


Перечисление CheckAlignment
 содержит варианты расположения флажков относительно текста в элементах
 управления.


Используется следующими свойствами и методами:


	- [IBindingCheckBox.Align](../Interface/IBindingCheckBox/IBindingCheckBox.Align.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Left. Флажок располагается
		 слева.


		 1
		 Center. Флажок располагается
		 по центру, текст не отображается.


		 2
		 Right. Флажок располагается
		 справа.
