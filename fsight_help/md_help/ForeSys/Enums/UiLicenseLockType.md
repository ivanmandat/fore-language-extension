# UiLicenseLockType


## Описание


Перечисление UiLicenseLockType
 содержит типы привязки лицензии. Данное перечисление используется свойством
 [ILicenseFeatureInfo.LockType](KeSomHost.chm::/Interface/ILicenseFeatureInfo/ILicenseFeatureInfo.LockType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Unlocked.
		 Непривязанная.


		 1
		 ClientLocked.
		 Привязка к клиентскому месту.


		 2
		 ServerLocked.
		 Привязка к серверу.


		 3
		 ClientServerLocked.
		 Привязка к клиентскому месту и к серверу.
