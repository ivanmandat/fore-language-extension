# UiLicenseMode


## Описание


Перечисление UiLicenseMode содержит
 варианты лицензий.


Данное перечисление используется свойством [ILicenseFeatureInfo.LicenseMode](KeSomHost.chm::/Interface/ILicenseFeatureInfo/ILicenseFeatureInfo.LicenseMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Standalone.
		 Автономная. Лицензия на одно рабочее место с привязкой к конкретным
		 параметрам рабочей станции. Устанавливается локально, не требует
		 подключения к сети.


		 1
		 NetWork.
		 Сетевая. Лицензия, позволяющая использование приложения несколькими
		 пользователями в сети. Предоставляет большую гибкость, простоту
		 администрирования.
