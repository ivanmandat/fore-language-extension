# SysEventType


## Описание


Перечисление SysEventType содержит
 категории записи в журнале событий Windows.


Используется следующим методом:


	- [ISysEventLog.Write](../Interface/ISysEventLog/ISysEventLog.Write.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Success. Операция завершилась
		 успешно.


		 1
		 AuditFailure. Операция
		 аудита завершилась неудачно.


		 2
		 AuditSuccess. Операция
		 аудита завершилась успешно.


		 4
		 Error. Ошибка.


		 8
		 Information. Сведения.


		 16
		 Warning. Предупреждение.
