# UiLicenseFeatureType


## Описание


Перечисление UiLicenseFeatureType
 содержит типы единиц лицензирования, используемые в «Форсайт. Аналитическая платформа».


Используется следующими свойствами:


	- [ILicenseRequestSettings.FeatureType](KeSomHost.chm::/Interface/ILicenseRequestSettings/ILicenseRequestSettings.FeatureType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Unknown. Тип лицензии
		 не определён.


		 1
		 PrognozPlatform. Базовая
		 единица лицензирования AnalyticsPlatform, позволяет устанавливать
		 соединение с репозиторием, а также разрешает работу с функциональностью
		 следующих инструментов: навигатор объектов, работа со справочниками
		 НСИ, рабочее пространство, менеджер обновлений, менеджер репозитория.


		 2
		 Study. Зарезервировано
		 для внутреннего использования.


		 3
		 Dashboard. Аналитические
		 панели.
		Примечание.
		 Значение не используется, оставлено для совместимости с предыдущими
		 версиями. Используйте DashboardViewer
		 и DashboardEditor.


		 4
		 Report. Отчётность.

		Примечание.
		 Значение не используется, оставлено для совместимости с предыдущими

		 и ReportEditor.

		 OLAP. Аналитические
		 запросы (OLAP).

		 и прогнозирование.

		 DWDesigner. Конструктор

		 следующие объекты: база данных, импорт данных, все виды кубов,
		 все виды справочников, каталоги ADOMD, таблицы, процедуры, команды
		 СУБД, базы данных временных рядов, репозиторий НСИ.

		 задач.

		 SelfService. Информационная
		 панель и модель данных.


## Комментарии


Каждый тип единицы лицензирования используется для получения прав на
 работу с одним или более инструментом. Более подробно о типах лицензий
 и их взаимодействии читайте в подразделе «[О единицах лицензирования
 «Форсайт. Аналитическая платформа»](Setup.chm::/08_Licensing/AboutFeatures.htm)».
