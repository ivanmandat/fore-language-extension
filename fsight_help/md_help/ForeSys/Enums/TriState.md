# TriState


## Описание


Перечисление TriState содержит
 константы, используемые для указания одного из трех возможных состояний.


## Допустимые значения


		 Значение
		 Краткое описание


		 -2
		 Undefined. Неопределенное
		 состояние.


		 -1
		 OnOption. Включено.


		 0
		 OffOption. Выключено.
