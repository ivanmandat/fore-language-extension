# UiLicenseType


## Описание


Перечисление UiLicenseType содержит
 тип применения лицензии.


Данное перечисление используется свойством [ILicenseFeatureInfo.LicenseType](KeSomHost.chm::/Interface/ILicenseFeatureInfo/ILicenseFeatureInfo.LicenseType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Normal.
		 Ограниченная или неограниченная лицензия по времени действия.


		 1
		 Trial.
		 Ограниченная лицензия по времени действия с момента первого запуска.
