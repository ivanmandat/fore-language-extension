# WebFormResult


## Описание


Перечисление WebFormResult содержит
 результаты закрытия веб-формы, которая была открыта модально.


Используется следующими свойствами и методами:


	- [IWebForm.ShowModal](KeSom.chm::/Interface/IWebForm/IWebForm.ShowModal.htm);


	- [IWebFormComponent.Close](WebForms.chm::/Interface/IWebFormComponent/IWebFormComponent.Close.htm).
