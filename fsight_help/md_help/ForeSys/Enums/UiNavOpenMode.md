# UiNavOpenMode


## Описание


Перечисление UiNavOpenMode содержит
 тип папки, используемой при открытии навигатора объектов, в качестве корневой.


Используется следующими свойствами и методами:


	- Командой [OpenNavigator](UiLib.chm::/Interface/IUiCommandTarget/Command/Command_OpenNavigator.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 PrivateFolder. [Личная
		 папка пользователя](Admin.chm::/03_Admin/Users/Admin_UserFolder.htm). При открытии вкладки навигатора
		 объектов корневой папкой использовать личную папку пользователя.
		 Уровень папок репозитория будет ограничен личной папкой пользователя,
		 которая будет являться корневой.


		 1
		 SetFolder. Заданная
		 папка. При открытии вкладки навигатора объектов корневой папкой
		 использовать заданную папку. Уровень папок репозитория будет ограничен
		 заданной папкой, которая будет являться корневой. Папка задаётся
		 в параметрах команды [OpenNavigator](UiLib.chm::/Interface/IUiCommandTarget/Command/Command_OpenNavigator.htm).
