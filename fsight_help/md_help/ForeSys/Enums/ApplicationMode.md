# ApplicationMode


## Описание


Перечисление ApplicationMode
 содержит режимы подключения к репозиторию.


Используется следующими свойствами и методами:


	- [IMetabaseDefinition.Open](KeSom.chm::/Interface/IMetabaseDefinition/IMetabaseDefinition.Open.htm);


	- [IMetabase.Mode](KeSom.chm::/Interface/IMetabase/IMetabase.Mode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Win. Подключение к
		 репозиторию осуществляется в навигаторе объектов платформы.


		 1
		 Web. Подключение к
		 репозиторию осуществляется посредством Web-интерфейса.
