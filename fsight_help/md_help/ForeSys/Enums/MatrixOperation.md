# MatrixOperation


## Описание


Перечисление MatrixOperation
 содержит операции, которые необходимо произвести для исходной матрицы.


Используется следующими свойствами и методами:


	- [IMatrixModel.OperationConst](../Interface/IMatrixModel/IMatrixModel.OperationConst.htm);


	- [IMatrixModel.OperationMatrix](../Interface/IMatrixModel/IMatrixModel.OperationMatrix.htm).


## Комментарии


Все операции производятся поэлементно. Если матрицы имеют различную
 размерность или разное количество элементов по измерениям, то, в зависимости
 от выполняемой операции, результирующая матрица может быть расширена/усечена.
 При умножении/делении размерности матриц должны быть одинаковы.


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Не производить
		 никаких действий.


		 1
		 Assign. Установить
		 структуру и значения в соответствии с матрицей, передаваемой в
		 качестве параметра.


		 2
		 Append. Добавить в
		 текущую матрицу значения из матрицы, передаваемой в качестве параметра
		 (добавление происходит в начало матрицы, исходные данные заменяются).


		 3
		 Add. Сложение.


		 4
		 Subtract. Вычитание.


		 5
		 Mul. Умножение.


		 6
		 Division. Деление.
