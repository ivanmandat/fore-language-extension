# DimComboSortDirection


## Описание


Перечисление DimComboSortDirection
 содержит варианты сортировки элементов справочника по наименованию.


Используется свойством [IBindingDimCombo.SortDirection](../Interface/IBindingDimCombo/IBindingDimCombo.SortDirection.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Сортировка
		 элементов справочника не производится.


		 1
		 Ascending. Сортировка
		 элементов справочника производится в порядке возрастания по алфавиту
		 или текстовым значениям чисел.


		 2
		 Descending. Сортировка
		 элементов справочника производится в порядке убывания по алфавиту
		 или текстовым значениям чисел.
