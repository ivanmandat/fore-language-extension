# TimeUnits


## Описание


Перечисление TimeUnits содержит
 единицы измерения времени.


Используется следующим свойством:


	- [IViewerUpdateSettings.AutoUpdateIntervalUnits](../Interface/IViewerUpdateSettings/IViewerUpdateSettings.AutoUpdateIntervalUnits.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Seconds. Секунды.


		 1
		 Minutes. Минуты.


		 2
		 Hours. Часы.
