# TaskState


## Описание


Перечисление TaskState содержит значения, возвращающие состояние задачи.


Используется следующим свойством:


-
[ITask.State](../Interface/ITask/ITask.State.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Created. Задача создана, но не выполняется.


 3
 Running. Задача выполняется.


 4
 WaitingForChildrenToComplete. Задача выполнена, производится ожидание выполнения связанных задач.


 5
 RanToCompletion. Задача выполнена успешно.


 6
 Canceled. Задача была отменена.


 7
 Faulted. Задача выполнена с ошибкой.
