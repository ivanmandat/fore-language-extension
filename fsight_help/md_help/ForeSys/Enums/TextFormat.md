# TextFormat


## Описание


Перечисление TextFormat содержит
 форматы текста, между которым доступна конвертация.


Используется следующими свойствами и методами:


	- [ITextConverter.Export](../Interface/ITextConverter/ITextConverter.Export.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Unknown. Формат текста
		 не определен.


		 1
		 Html. HTML-представление
		 текста.


		 2
		 Xaml. XAML-представление
		 текста.


		 3
		 Rtf. RTF-представление
		 текста.
