# RdsElementCardStyle


## Описание


Перечисление RdsElementCardStyle
 содержит типы режима отображения карточки элементов репозитория НСИ.


Используется свойством:


	- [IRdsDatabase.ElementCardStyle](../../KeRds/interface/irdsdatabase/irdsdatabase.rdselementcardstyle.htm);


	- [IDefaultBehaviour.RDSElementCard](KeFore.chm::/Interface/IDefaultBehaviour/IDefaultBehaviour.RDSElementCard.htm).
