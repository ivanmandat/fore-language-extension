# MatrixRankMethod


## Описание


Перечисление MatrixRankMethod
 содержит способы составления матрицы ранжирования.


Используется следующим методом:


	- [IMatrixModel.Rank](../Interface/IMatrixModel/IMatrixModel.Rank.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 NoOrdered. Одинаковый
		 ранг у одинаковых значений, следующий ранг на столько больше,
		 сколько одинаковых элементов (итоговые значения в виде 1,1,3,3,5).


		 1
		 EqualOrdered.
		 Одинаковый ранг у одинаковых значений, следующий ранг на 1 больше
		 (итоговые значения в виде 1,1,2,2,3).


		 2
		 Ordered.
		 Ранг элементов с одинаковыми значениями возрастает в порядке возрастания
		 индекса элемента (итоговые значения в виде 1,2,3,4,5).


		 3
		 InplaceExchange.
		 Поменять местами значения в исходной матрице.
