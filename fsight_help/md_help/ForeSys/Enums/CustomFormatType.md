# CustomFormatType


## Описание


Перечисление CustomFormatType
 содержит список форматов числа в ячейке таблицы.


Используется следующим свойством:


	- [ITabCellStyle.FormatType](TabSheet.chm::/Interface/ITabCellStyle/ITabCellStyle.FormatType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 -1
		 Undefined. Значение
		 используется в случае, если в ячейке таблицы задан формат числа
		 не из списка предустановленных форматов или если свойство запрашивается
		 для нескольких ячеек с различными форматами чисел.


		 0
		 General. Общий.


		 1
		 Number. Числовой.


		 2
		 Date. Дата.


		 3
		 Time. Время.


		 4
		 Percent. Процентный.


		 5
		 Exponential. Экспоненциальный.


		 6
		 Text. Текстовый.


		 7
		 Currency. Денежный
		 (в соответствии с настройками региональных параметров).


		 8
		 CurrencyR. Денежный
		 (рубли).


		 9
		 CurrencyS. Денежный
		 (доллар).


		 10
		 CurrencyE. Денежный
		 (евро).


		 11
		 Additional. Дополнительный.
