# JsonFormatOptions


## Описание


Перечисление JsonFormatOptions
 содержит формат представления JSON-структуры.


Используется следующими свойствами и методами:


	- [IJsonElement.WriteToStream](../Interface/IJsonElement/IJsonElement.WriteToStream.htm);


	- [IJsonElement.WriteToString](../Interface/IJsonElement/IJsonElement.WriteToString.htm);


	- [IJsonDocument.WriteToFile](../Interface/IJsonDocument/IJsonDocument.WriteToFile.htm);


	- [IJsonDocument.WriteToStream](../Interface/IJsonDocument/IJsonDocument.WriteToStream.htm);


	- [IJsonDocument.WriteToString](../Interface/IJsonDocument/IJsonDocument.WriteToString.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Default_. По умолчанию.
		 С отступами и переносами строк.


		 1
		 UnIndented. Без отступов
		 и переносов строк.
