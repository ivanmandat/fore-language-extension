# String.EndsWith_


## Синтаксис


		EndsWith_(Source: String; SubStr: String):
		 Boolean;


## Параметры


Source. Исходная строка;


SubStr. Искомая подстрока.


## Описание


Метод EndsWith_ возвращает признак
 наличия искомой подстроки в конце исходной строки.


## Комментарии


Допустимые значения:


	- True. Исходная строка
	 оканчивается искомой строкой;


	- False. Исходная строка
	 не оканчивается искомой строкой.


Поиск искомой подстроки в исходной строке выполняется с учётом регистра.
 Пробелы в конце строки и подстроки игнорируются.


## Пример


					Sub UserProc;

		Var

		    s1, s2: String;

		    b: Boolean;

		Begin

		    s1 := "Calculator";

		    s2 := "tor";

		    // Проверяем наличие подстроки в конце исходной
		 строки

		    b := String.EndsWith_(s1, s2);

		    Debug.WriteLine("EndsWith_("""  + s1 + "; " + s2 + """) = " + b.ToString);

		End Sub UserProc


В результате выполнения примера в окно консоли будет выведен признак
 наличия подстроки «tor» в конце строки «Calculator».
