# String.SubString


## Синтаксис


SubString(Start: Integer; [Count: Integer = -1]):
 String;


## Параметры


Start. Индекс начального символа
 подстроки в строке;


Count. Количество символов
 в подстроке.


## Описание


Метод SubString возвращает подстроку
 из текущей строки.


## Комментарии


Если параметр Count не задан,
 то метод вернет подстроку, начиная с символа с индексом Start
 и заканчивая последним символом в строке.


## Пример


	Sub Main;

	Var

	    s, s1: String;

	Begin

	    s := "abcdefg";

	    s1 := s.SubString(3, 2);

	End Sub Main;


После выполнения примера в переменной «s1» будет содержаться подстрока
 «de».
