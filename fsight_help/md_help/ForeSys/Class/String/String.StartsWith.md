# String.StartsWith


## Синтаксис


StartsWith(Value: String): Boolean;


## Параметры


Value.
 Проверяемая последовательность символов.


## Описание


Метод StartWith осуществляет
 проверку, начинается ли текущая строка последовательностью символов, передаваемой
 посредством параметра Value.


## Комментарии


Метод возвращает True, если
 строка начинается указанной последовательностью и False
 во всех других случаях.


## Пример


	Sub UserProc;

	Var

	    s: String;

	    b, b1: Boolean;

	Begin

	    s := "abcdefg";

	    b := s.StartsWith("b");

	    b1 := s.StartsWith("a");

	End Sub UserProc;


После выполнения примера в переменных «b» и «b1»
 будут содержаться значения False и True соответственно, так как текущая
 строка начинается с символа «a».
