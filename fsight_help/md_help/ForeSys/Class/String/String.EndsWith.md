# String.EndsWith


## Синтаксис


EndsWith(Value: String): Boolean;


## Параметры


Value. Проверяемая последовательность
 символов.


## Описание


Метод EndsWith осуществляет
 проверку, заканчивается ли текущая строка последовательностью символов,
 передаваемой посредством параметра Value.


## Комментарии


Метод возвращает True, если
 строка заканчивается указанной последовательностью и False
 во всех других случаях.


## Пример


	Sub UserProc;

	Var

	    s: String;

	    b, b1: Boolean;

	Begin

	    s := "abcdefg";

	    b := s.EndsWith("f");

	    b1 := s.EndsWith("g");

	End Sub UserProc;


После выполнения примера в переменных «b» и «b1»
 будут храниться значения False и True соответственно, так как текущая
 строка заканчивается символом «g».
