# String.Join


## Синтаксис


Join(Delimiter: String; Values: Array): String;


## Параметры


Delimiter. Разделитель, используемый
 при объединении;


Values. Строковый массив, содержащий
 набор значений, которые необходимо объединить в одну строку.


## Описание


Метод Join объединяет значения
 всех элементов массива, передаваемого посредством параметра Values,
 в одну строку, используя указанный разделитель Delimiter.


## Пример


	Sub UserProc;

	Var

	    s: String;

	    s1: Array[3] Of String;

	Begin

	    s1[0] := "13";

	    s1[1] := "24";

	    s1[2] := "35";

	    s := String.Join("_", s1);

	End Sub UserProc;


После выполнения примера в переменной «S» будет
 содержаться текст "13_24_35".
