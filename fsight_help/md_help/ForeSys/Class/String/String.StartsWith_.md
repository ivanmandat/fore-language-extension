# String.StartsWith_


## Синтаксис


		StartsWith_(Source: String; SubStr: String):
		 Boolean;


## Параметры


Source. Исходная строка;


SubStr. Искомая подстрока.


## Описание


Метод StartsWith_ возвращает
 признак наличия искомой подстроки в начале исходной строки.


## Комментарии


Допустимые значения:


	- True. Исходная строка
	 начинается с искомой подстроки;


	- False. Исходная строка
	 не начинается с искомой подстроки.


Поиск искомой подстроки в исходной строке выполняется с учётом регистра.
 Пробелы в начале строки и подстроки игнорируются.


## Пример


					Sub UserProc;

		Var

		    s1, s2: String;

		    b: Boolean;

		Begin

		    s1 := "Calculator";

		    s2 := "Calc";

		    // Проверяем наличие
		 искомой подстроки в начале исходной строки

		    b := String.StartsWith_(s1, s2);

		    Debug.WriteLine("StartsWith_(""" + s1 + "; " + s2 + """) = " + b.ToString);

		End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен признак
 наличия подстроки «Calc» в начале строки «Calculator».
