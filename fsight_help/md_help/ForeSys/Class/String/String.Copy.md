# String.Copy


## Синтаксис


Copy(Var Result: String);


## Параметры


Result. Символьная переменная,
 в которую будет помещена копия значения переменной, для которой вызывается
 данный метод.


## Описание


Метод Copy создает копию значения
 текущей строки.


## Пример


	Sub UserProc;

	Var

	    S, S_Copy: String;

	Begin

	    S := "ABCD";

	    S.Copy(S_Copy);

	    S := "";

	End Sub UserProc;


После выполнения примера в переменную «S_Copy»
 будет скопировано содержание строки «S».
