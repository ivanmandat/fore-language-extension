# String


## Описание


Класс String представляет [строковый](Fore.chm::/03_dataTypes/Fore_DataTypes.htm)
 тип данных.

		 ![](../../Sub_Image.gif)
		 [Insert](String.Insert.htm)
		 Метод Insert осуществляет
		 вставку подстроки в указанное место исходной строки.


		 ![](../../Sub_Image.gif)

		 значения всех элементов указанного массива в одну строку, используя
		 указанный разделитель.


		 ![](../../Sub_Image.gif)
		 [Left](String.Left.htm)
		 Метод Left возвращает

		 ![](../../Sub_Image.gif)
		 [Length_](String.Length_.htm)

		 [Mid](String.Mid.htm)
		 Метод Mid возвращает

		 ![](../../Sub_Image.gif)
		 [Min_](String.Min_.htm)
		 Метод Min_ посимвольно
		 сравнивает две строки и возвращает минимальную из них.


		 ![](../../Sub_Image.gif)
		 [PadLeft](String.PadLeft.htm)
		 Метод PadLeft добавляет
		 определенное количество указанных символов слева к исходной строке.


		 ![](../../Sub_Image.gif)
		 [PadRight](String.PadRight.htm)
		 Метод PadRight добавляет
		 определенное количество указанных символов справа к исходной строке.


		 ![](../../Sub_Image.gif)
		 [Remove](String.Remove.htm)
		 Метод Remove удаляет
		 указанное количество символов, начиная с заданной позиции в исходной
		 строке.


		 ![](../../Sub_Image.gif)
		 [Replace](String.Replace.htm)
		 Метод Replace осуществляет
		 поиск указанной подстроки в исходной строке и замену на новое
		 значение.


		 ![](../../Sub_Image.gif)
		 [Right](String.Right.htm)
		 Метод Right возвращает

		 [![](../../Sub_Image.gif)](String.StartsWith_.htm)

		 ![](../../Sub_Image.gif)
		 [StartsWith_](String.StartsWith_.htm)
		 Метод StartsWith_ возвращает
		 признак наличия искомой подстроки в начале исходной строки.


		 ![](../../Sub_Image.gif)
		 [ToLower](String.ToLower.htm)
		 Метод ToLower переводит
		 все символы указанной строки в нижний регистр.


		 ![](../../Sub_Image.gif)
		 [ToUpper](String.ToUpper.htm)
		 Метод ToUpper переводит
		 все символы указанной строки в верхний регистр.


		 ![](../../Sub_Image.gif)
		 [Trim](String.Trim.htm)
		 Метод Trim удаляет
		 все пробелы в начале и в конце указанной строки.


		 ![](../../Sub_Image.gif)
		 [TrimEnd](String.TrimEnd.htm)
		 Метод TrimEnd удаляет
		 все пробелы, имеющиеся в конце строки.


		 ![](../../Sub_Image.gif)
		 [TrimStart](String.TrimStart.htm)
		 Метод TrimStart удаляет
		 все пробелы, имеющиеся в начале строки.
