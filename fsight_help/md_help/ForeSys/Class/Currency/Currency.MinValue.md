# Currency.MinValue


## Синтаксис


MinValue: Currency;


## Описание


Свойство MinValue возвращает
 минимальное значение типа Currency.


## Пример


	Sub UserProc;

	Var

	    c: Currency;

	Begin

	    c := Currency.MinValue;

	End Sub UserProc;


После выполнения примера в переменной «c» будет содержаться минимальное
 значение типа Currency, т.е. значение -922337203685477,5808.
