# Currency.Parse


## Синтаксис


Parse(Value: String): Currency;


## Параметры


Value. Строковое значение,
 которую необходимо преобразовать к типу Currency.


## Описание


Метод Parse преобразует строковое
 значение к типу Currency без проверки на корректность строки.


## Комментарии


Данный метод не учитывает установленные региональные параметры. В значении,
 передаваемом в параметре Value,
 для разделения целой и дробной части должен использоваться символ «.», разделитель разрядов не используется.


Если преобразование не удалось, то генерируется исключительная ситуация.


## Пример


	Sub UserProc;

	Var

	    c: Currency;

	    s: String;

	Begin

	    s := "1234.56789";

	    c := Currency.Parse(s);

	End Sub UserProc;


После выполнения примера в переменной «c» будет содержаться значение
 1234,5679, полученное путем преобразования из указанного строкового значения.
