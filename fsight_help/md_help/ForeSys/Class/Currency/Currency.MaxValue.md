# Currency.MaxValue


## Синтаксис


MaxValue: Currency;


## Описание


Свойство MaxValue возвращает
 максимальное значение типа Currency.


## Пример


	Sub UserProc;

	Var

	    c: Currency;

	Begin

	    c := Currency.MaxValue;

	End Sub UserProc;


После выполнения примера в переменной «c» будет содержаться максимальное
 значение типа Currency, т.е. значение 922337203685477,5807.
