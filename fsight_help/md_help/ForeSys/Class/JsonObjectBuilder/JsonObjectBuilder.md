# JsonObjectBuilder


Сборка: System;


## Описание


Класс JsonObjectBuilder реализует
 новый JSON-объект.


## Комментарии


Для работы с объектом в формате JSON создайте экземпляр класса JsonObjectBuilder
 и приведите полученный объект к интерфейсу [IJsonObjectBuilder](../../Interface/IJsonObjectBuilder/IJsonObjectBuilder.htm).
