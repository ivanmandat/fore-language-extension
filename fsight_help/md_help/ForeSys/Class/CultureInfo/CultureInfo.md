# CultureInfo


## Описание


Класс CultureInfo реализует
 объект, содержащий свойства и методы для работы с региональными настройками
 определенной культуры.

		 Краткое описание

		 Конструктор Create
		 инициализирует настройки языковых и региональных стандартов, идентификатор
		 которых передается в качестве входного параметра.
