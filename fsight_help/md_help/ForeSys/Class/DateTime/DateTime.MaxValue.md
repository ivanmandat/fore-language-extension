# DateTime.MaxValue


## Синтаксис


MaxValue: DateTime;


## Описание


Свойство MaxValue возвращает
 максимальное значение типа DateTime.


## Пример


	Sub UserProc;

	Var

	    d: DateTime;

	Begin

	    d := DateTime.MaxValue;

	End Sub UserProc;


После выполнения примера в переменной «d» будет храниться максимальное
 значение даты - 31.12.9999.
