# DateTime.MinValue


## Синтаксис


MinValue: DateTime;


## Описание


Свойство MinValue возвращает
 минимальное значение типа DateTime.


## Пример


	Sub Main;

	Var

	    d: DateTime;

	Begin

	    d := DateTime.MinValue;

	End Sub Main;


После выполнения примера в переменной «d» будет содержаться минимальное
 значение типа DateTime - «01.01.0100».
