# DateTime.TryParse


## Синтаксис


TryParse(Value: String; Var Result: DateTime): Boolean;


## Параметры


Value. Строка, которую необходимо
 преобразовать к типу DateTime;


Result. Дата, преобразованная
 из строки.


## Описание


Метод TryParse преобразует символьную
 строку к типу DateTime с проверкой на корректность строки.


## Комментарии


Допустимые значения:


	- True. Преобразование
	 удалось. Преобразованное значение помещается в переменную, указанную
	 в параметре Result;


	- False. Преобразование
	 не удалось.


Указываемое значение должны соответствовать допустимому диапазону, который
 установлен для типа данных [DateTime](Fore.chm::/03_dataTypes/Fore_DataTypes.htm).
 При указании даты меньше допустимой автоматически формируется дата, к
 которой добавлено текущее столетие.


## Пример


	Sub UserProc;

	Var

	    d: DateTime;

	    b: Boolean;

	Begin

	    b := DateTime.TryParse("01.01.02", D);

	End Sub UserProc;


Если после выполнения примера в переменной «b» будет содержаться значение
 True, значит преобразование удачное и в переменной «d» будет содержаться
 дата «01.01.02».
