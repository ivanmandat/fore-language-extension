# JsonArrayBuilder


Сборка: System;


## Описание


Класс JsonArrayBuilder реализует
 новый JSON-массив.


## Комментарии


Для работы с массивом создайте экземпляр класса JsonArrayBuilder и приведите
 полученный объект к интерфейсу [IJsonArrayBuilder](../../Interface/IJsonArrayBuilder/IJsonArrayBuilder.htm).
