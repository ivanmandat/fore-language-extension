# Variant.GetPropertyEx


## Синтаксис


GetPropertyEx(Name: String; Parameters: Array):
 Variant;


## Параметры


Name. Наименование свойства,
 значение которого необходимо получить;


Parameters. Дополнительные
 параметры, необходимые для свойства.


## Описание


Метод GetPropertyEx позволяет
 получить значение указанного свойства объекта, хранящегося в переменной,
 используя дополнительные параметры.
