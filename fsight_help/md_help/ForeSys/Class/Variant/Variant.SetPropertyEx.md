# Variant.SetPropertyEx


## Синтаксис


SetPropertyEx(Name:String; Parameters:Array; Value:
 Variant);


## Параметры


Name. Наименование свойства,
 значение которого необходимо установить;


Parameters. Дополнительные
 параметры, необходимые для свойства;


Value. Значение свойства.


## Описание


Метод SetPropertyEx позволяет
 установить значение указанного свойства объекта, хранящегося в переменной,
 используя дополнительные параметры.
