# Variant.SetProperty


## Синтаксис


SetProperty(Name: String; Value: Variant);


## Параметры


Name. Наименование свойства,
 значение которого необходимо установить;


Value. Значение, которого необходимо
 установить.


## Описание


Метод SetProperty позволяет
 установить значение свойства объекта, хранящегося в переменной.
