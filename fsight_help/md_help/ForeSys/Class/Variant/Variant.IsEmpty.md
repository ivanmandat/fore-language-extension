# Variant.IsEmpty


## Синтаксис


IsEmpty: Boolean;


## Описание


Свойство IsEmpty возвращает
 признак содержания в переменной пустого значения.


## Комментарии


Свойство IsEmpty возвращает
 значение True, если для переменной
 установлено [пустое значение](Variant.Empty.htm) и False
 во всех остальных случаях.
