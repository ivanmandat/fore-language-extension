# Array.Sort


## Синтаксис


Sort;


## Описание


Метод Sort осуществляет сортировку
 элементов в массиве по возрастанию.


## Комментарии


Данный метод поддерживается только для одномерных массивов.


## Пример


Добавьте ссылку на системную сборку MathFin.


	Sub UserProc;

	Var

	    Ar: Array[0..20] Of Integer;

	    i: Integer;

	Begin

	    For i := 0 To 20 Do

	        Ar[i] := Math.RandBetweenI(0, 100);

	    End For;

	    Ar.Sort;

	End Sub UserProc;


После выполнения примера в одномерный массив Ar записываются случайные
 целые числа, после этого все значения в массиве сортируются.
