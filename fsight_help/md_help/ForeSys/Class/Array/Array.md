# Array


## Описание


Класс Array реализует свойства
 и методы типа данных [Array](Fore.chm::/03_dataTypes/Fore_DataTypes.htm),
 а также переменных, имеющих этот тип данных.


## Свойства переменных типа Array


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Length](Array.Length.htm)
		 Свойство Length возвращает
		 количество всех элементов в массиве.


		 ![](../../Property_Image.gif)
		 [Rank](Array.Rank.htm)
		 Свойство Rank возвращает
		 количество измерений в массиве.


## Методы переменных типа Array


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ArrayOperation](Array.ArrayOperation.htm)
		 Метод ArrayOperation
		 выполняет арифметическую операцию над элементами массивов.


		 ![](../../Sub_Image.gif)
		 [Concat](Array.Concat.htm)
		 Метод Concat объединяет
		 элементы двух массивов в новый массив путём расширения исходного
		 массива по указанному измерению.


		 ![](../../Sub_Image.gif)
		 [GetLowerBound](Array.GetLowerBound.htm)
		 Метод GetLowerBound
		 возвращает нижнюю границу измерения, номер которого передается
		 в качестве входного параметра.


		 ![](../../Sub_Image.gif)
		 [GetType](Array.GetType.htm)
		 Метод GetType возвращает
		 тип данных, содержащихся в массиве.


		 ![](../../Sub_Image.gif)
		 [GetUpperBound](Array.GetUpperBound.htm)
		 Метод GetUpperBound
		 возвращает верхнюю границу измерения, номер которого передается
		 в качестве входного параметра.


		 ![](../../Sub_Image.gif)
		 [IndexOf](Array.IndexOf.htm)
		 Метод IndexOf осуществляет
		 поиск указанного значения в массиве и возвращает индекс соответствующего
		 элемента при первом вхождении.


		 ![](../../Sub_Image.gif)
		 [LastIndexOf](Array.LastIndexOf.htm)
		 Метод LastIndexOf осуществляет
		 поиск указанного значения в массиве и возвращает индекс соответствующего
		 элемента при последнем вхождении.


		 ![](../../Sub_Image.gif)
		 [Operation](Array.Operation.htm)
		 Метод Operation выполняет
		 арифметическую операцию над элементами массива с указанным значением.


		 ![](../../Sub_Image.gif)
		 [Sort](Array.Sort.htm)
		 Метод Sort осуществляет
		 сортировку элементов в массиве по возрастанию.


		 ![](../../Sub_Image.gif)
		 [ToSimpleArray](Array.ToSimpleArray.htm)
		 Метод ToSimpleArray
		 формирует новый массив путём преобразования исходного массива
		 с типом Variant к массиву, имеющему простой тип данных.


		 ![](../../Sub_Image.gif)
		 [ToVariantArray](Array.ToVariantArray.htm)
		 Метод ToVariantArray
		 формирует новый массив путём преобразования исходного массива
		 к массиву с типом Variant.
