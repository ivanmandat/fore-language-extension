# Guid


Сборка: System;


## Описание


Класс Guid реализует свойства
 и методы типа данных [Guid](Fore.chm::/03_dataTypes/Fore_DataTypes.htm),
 а также переменных, имеющих этот тип данных.


## Методы класса Guid


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [IsGuid](Guid.IsGuid.htm)
		 Метод IsGuid осуществляет
		 проверку, является ли указанное строковое значение уникальным
		 глобальным идентификатором, который можно представить типом данных
		 Guid.


		 ![](../../Sub_Image.gif)
		 [Parse](Guid.Parse.htm)
		 Метод Parse преобразует
		 строковое значение к типу Guid без проверки на корректность строки.


		 ![](../../Sub_Image.gif)
		 [TryParse](Guid.TryParse.htm)
		 Метод TryParse преобразует
		 строковое значение к типу Guid с проверкой на корректность строки.
