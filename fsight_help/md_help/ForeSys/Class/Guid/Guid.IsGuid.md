# Guid.IsGuid


## Синтаксис


IsGuid(Value: String): Boolean;


## Параметры


Value.
 Строковое значение, которое необходимо проверить.


## Описание


Метод IsGuid осуществляет проверку,
 является ли указанное строковое значение уникальным глобальным идентификатором,
 который можно представить типом данных Guid.


## Комментарии


Допустимые значения:


	- True. Указанное значение
	 можно преобразовать к типу данных Guid;


	- False. Преобразование
	 невозможно. Для преобразования используйте метод [Int64.Parse](../Int64/Int64.Parse.htm)
	 или [Int64.TryParse](../Int64/Int64.TryParse.htm).
