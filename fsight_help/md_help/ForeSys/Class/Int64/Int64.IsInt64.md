# Int64.IsInt64


## Синтаксис


IsInt64(Value: String): Boolean;


## Параметры


Value. Строковое значение,
 которое необходимо проверить.


## Описание


Метод IsInt64 осуществляет проверку,
 является ли указанное строковое значение числом, которое можно представить
 типом данных Int64.


## Комментарии


Допустимые значения:


	- True. Указанное значение
	 можно преобразовать к типу данных Int64;


	- False. Преобразование
	 невозможно. Для преобразования используйте метод [Int64.Parse](Int64.Parse.htm)
	 или [Int64.TryParse](Int64.TryParse.htm).
