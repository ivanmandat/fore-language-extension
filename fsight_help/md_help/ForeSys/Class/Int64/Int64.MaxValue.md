# Int64.MaxValue


## Синтаксис


MaxValue: Int64;


## Описание


Свойство MaxValue возвращает
 максимальное допустимое значение.


## Комментарии


Для типа данных Int64 максимальное допустимое значение равно 9223372036854775807.
