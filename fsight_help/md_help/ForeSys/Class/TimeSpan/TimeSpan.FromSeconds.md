# TimeSpan.FromSeconds


## Синтаксис


FromSeconds(Seconds: Double): TimeSpan;


## Параметры


Seconds. Количество секунд
 в интервале времени.


## Описание


Метод FromSeconds формирует
 интервал времени, состоящий из указанного количества секунд.
