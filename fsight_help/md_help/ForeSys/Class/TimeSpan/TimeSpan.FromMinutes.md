# TimeSpan.FromMinutes


## Синтаксис


FromMinutes(Minutes: Double): TimeSpan;


## Параметры


Minutes. Количество минут в
 интервале времени.


## Описание


Метод FromMinutes формирует
 интервал времени, состоящий из указанного количества минут.
