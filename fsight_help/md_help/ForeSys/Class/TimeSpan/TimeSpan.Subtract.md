# TimeSpan.Subtract


## Синтаксис


Subtract(Value: TimeSpan; Decrement: TimeSpan):
 TimeSpan;


## Параметры


Value. Интервал времени, который
 необходимо уменьшить;


Decrement. Интервал времени,
 на который уменьшается значение Value.


## Описание


Метод Subtract уменьшает интервал
 времени на указанное значение.
