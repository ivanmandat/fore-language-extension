# TimeSpan.Add


## Синтаксис


Add(Value: TimeSpan; Increment: TimeSpan): TimeSpan;


## Параметры


Value. Интервал времени, который
 необходимо увеличить;


Increment. Интервал времени,
 на который увеличивается значение Value.


## Описание


Метод Add увеличивает заданный
 интервал времени на указанное значение.
