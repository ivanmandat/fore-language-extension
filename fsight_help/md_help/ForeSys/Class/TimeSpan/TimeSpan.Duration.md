# TimeSpan.Duration


## Синтаксис


Duration: TimeSpan;


## Описание


Свойство Duration возвращает
 абсолютное значение интервала времени.


## Комментарии


Абсолютное значение соответствует интервалу от даты «30.12.1899 0:00:00»
 до текущего временного интервала.
