# TimeSpan.ToString


## Синтаксис


ToString: String;


## Описание


Свойство ToString возвращает
 значение интервала времени, преобразованное к строковому типу.


## Комментарии


Строка будет иметь вид: «<Кол-во дней>.<Кол-во часов>:<Кол-во
 минут>:<Кол-во секунд>.<Кол-во миллисекунд>».
