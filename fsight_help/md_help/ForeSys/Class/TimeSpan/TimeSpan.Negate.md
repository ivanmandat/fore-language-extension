# TimeSpan.Negate


## Синтаксис


Negate: TimeSpan;


## Описание


Свойство Negate возвращает интервал
 времени, обратный данному.


## Комментарии


Интервал времени рассчитанный в обратную сторону от даты «30.12.1899
 0:00:00».
