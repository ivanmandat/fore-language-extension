# TimeSpan.FromMilliseconds


## Синтаксис


FromMilliseconds(Milliseconds: Double): TimeSpan;


## Параметры


Milliseconds. Количество миллисекунд
 в интервале времени.


## Описание


Метод FromMilliseconds формирует
 интервал времени, состоящий из указанного количества миллисекунд.
