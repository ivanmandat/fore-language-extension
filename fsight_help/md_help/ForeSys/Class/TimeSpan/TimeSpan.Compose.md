# TimeSpan.Compose


## Синтаксис


Compose(


Days: Integer;


Hours: Integer;


Minutes: Integer;


Seconds: Integer;


Milliseconds: Integer):
 TimeSpan;


## Параметры


Days. Количество дней в интервале;


Hours. Количество часов в интервале;


Minutes. Количество минут в
 интервале;


Seconds. Количество секунд
 в интервале;


Milliseconds. Количество миллисекунд
 в интервале.


## Описание


Метод Compose формирует интервал
 времени из указанных входных параметров.

Использование свойства приведено в примере для свойства [TimeSpan.Hours](TimeSpan.Hours.htm).
