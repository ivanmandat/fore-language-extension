# TimeSpan.MaxValue


## Синтаксис


MaxValue: TimeSpan;


## Описание


Свойство MaxValue возвращает
 максимальное значение TimeSpan.


## Комментарии


Максимальное значение ограничено максимальным значением, доступным для
 типа Integer, которое указывается в качестве компоненты дни в интервале
 времени, т.е. 2147483647.00:00:00.000.
