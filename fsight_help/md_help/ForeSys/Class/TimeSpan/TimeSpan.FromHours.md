# TimeSpan.FromHours


## Синтаксис


FromHours(Hours: Double): TimeSpan;


## Параметры


Hours.
 Количество часов в интервале времени.


## Описание


Метод FromHours формирует интервала
 времени, состоящий из указанного количества часов.
