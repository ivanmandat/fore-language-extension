# TimeSpan.MinValue


## Синтаксис


MinValue: TimeSpan;


## Описание


Свойство MinValue возвращает
 минимальное значение TimeSpan.


## Комментарии


Минимальное значение ограничено минимальным значением, доступным для
 типа Integer, которое указывается в качестве компоненты дни в интервале
 времени, т.е. -2147483647.00:00:00.000.
