# TimeSpan.Parse


## Синтаксис


Parse(Value: String): TimeSpan;


## Параметры


Value. Значение, которое необходимо
 преобразовать к типу TimeSpan.


## Описание


Метод Parse преобразует строковое
 значение к типу TimeSpan.


## Комментарии


Значение, передаваемое в параметре Value,
 должно иметь следующий формат: «<Кол-во дней>.<Кол-во
 часов>:<Кол-во минут>:<Кол-во секунд>.<Кол-во
 миллисекунд>».


## Пример


	Sub UserProc;

	Var

	    t: TimeSpan;

	Begin

	    t := TimeSpan.Parse("0.00:01:00.000");

	End Sub UserProc;


После выполнения примера в переменной «t» будет содержаться интервал
 времени, полученный путём преобразования из указанного символьного значения.
