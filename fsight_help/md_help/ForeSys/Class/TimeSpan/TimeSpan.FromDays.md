# TimeSpan.FromDays


## Синтаксис


FromDays(Days: Double): TimeSpan;


## Параметры


Days. Количество дней в интервале
 времени.


## Описание


Метод FromDays формирует интервал
 времени, состоящий из указанного количества дней.
