# Integer.MinValue


## Синтаксис


MinValue: Integer;


## Описание


Свойство MinValue возвращает
 минимальное целочисленное значение.


## Пример


	Sub UserProc;

	Var

	    i: Integer;

	Begin

	    i := Integer.MinValue;

	End Sub UserProc;


После выполнения примера в переменной «i» будет
 содержаться минимальное значение типа Integer, т.е. значение -2147483648.
