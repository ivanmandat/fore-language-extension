# Integer.MaxValue


## Синтаксис


MaxValue: Integer;


## Описание


Свойство MaxValue возвращает
 максимальное целочисленное значение.


## Пример


	Sub UserProc;

	Var

	    i: Integer;

	Begin

	    i := Integer.MaxValue;

	End Sub UserProc;


После выполнения примера в переменной «i» будет
 содержаться максимальное значение типа Integer, т.е. значение 2147483647.
