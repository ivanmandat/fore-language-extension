# Integer.IsInteger


## Синтаксис


IsInteger(Value: String): Boolean;


## Параметры


Value. Строковое значение,
 которое необходимо проверить.


## Описание


Метод IsInteger осуществляет
 проверку, является ли указанное строковое значение целым числом.


## Комментарии


Допустимые значения:


	- True. Указанное значение
	 можно преобразовать к целочисленному типу;


	- False. Преобразование
	 невозможно. Для преобразования используйте метод [Integer.Parse](Integer.Parse.htm)
	 или [Integer.TryParse](Integer.TryParse.htm).
