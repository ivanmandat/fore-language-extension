# Integer.TryParse


## Синтаксис


TryParse(Value: String; Var Result: Integer): Boolean;


## Параметры


Value. Строковое значение,
 которое необходимо преобразовать к целочисленному типу;


Result. Переменная, в которую
 будет помещено значение в случае удачного преобразования.


## Описание


Метод TryParse преобразует строковое
 значение к целочисленному типу с проверкой на корректность строки.


## Комментарии


Если преобразование удалось, то метод возвращает значение True
 и преобразованное значение помещается в переменную, указанную в параметре
 Result.


## Пример


	Sub UserProc;

	Var

	    b: Boolean;

	    i: Integer;

	    s: String;

	Begin

	    s := "234";

	    b := Integer.TryParse(s, i);

	End Sub UserProc;


После выполнения примера в случае удачного преобразования
 в переменной «b» будет находиться значение «True», и в переменной «i»
 будет значение 234.
