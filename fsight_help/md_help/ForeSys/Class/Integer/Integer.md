# Integer


## Описание


Класс Integer представляет [целочисленный](Fore.chm::/03_dataTypes/Fore_DataTypes.htm)
 тип данных.

		 Краткое описание

		 проверку, является ли указанное строковое значение целым числом.


		 ![](../../Sub_Image.gif)
		 [Parse](Integer.Parse.htm)
		 Метод Parse преобразует

		 [TryParse](Integer.TryParse.htm)
		 Метод TryParse преобразует

		 строки.
