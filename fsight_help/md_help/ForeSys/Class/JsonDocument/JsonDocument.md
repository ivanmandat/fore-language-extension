# JsonDocument


Сборка: System;


## Описание


Класс JsonDocument реализует
 объект, предназначенный для работы с JSON-структурами.


## Комментарии


Для работы с JSON-структурой создайте экземпляр класса JsonDocument
 и приведите полученный объект к интерфейсу [IJsonDocument](../../Interface/IJsonDocument/IJsonDocument.htm).
