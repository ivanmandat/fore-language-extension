# ExpressionStringHandler


Сборка: System;


## Описание


Класс ExpressionStringHandler
 содержит методы, реализующие события возникающие при преобразовании элементов,
 содержащихся в выражении.


## Методы, унаследованные от [IExpressionStringHandler](../../Interface/IExpressionStringHandler/IExpressionStringHandler.htm)


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [OnTerm](../../Interface/IExpressionStringHandler/IExpressionStringHandler.OnTerm.htm)


		 Метод OnTerm реализует
		 событие, которое наступает при обращении к элементу выражения,
		 содержащему ссылку на данные (либо сами данные).


		 ![](../../Sub_Image.gif)
		 [OnText](../../Interface/IExpressionStringHandler/IExpressionStringHandler.OnText.htm)


		 Метод OnText реализует
		 событие, которое наступает при обращении к элементу, соответствующему
		 символьной части выражения.
