# Boolean.TrueString


## Синтаксис


TrueString: String;


## Описание


Метод TrueString возвращает
 логическое значение True, преобразованное
 к строковому виду.


## Пример


	Sub UserProc;

	Var

	    s: String;

	Begin

	    s := Boolean.TrueString;

	End Sub UserProc;


После выполнения примера в переменной s будет содержаться значение «True».
