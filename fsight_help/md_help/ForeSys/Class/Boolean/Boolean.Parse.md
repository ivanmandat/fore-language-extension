# Boolean.Parse


## Синтаксис


Parse(Value: String): Boolean;


## Параметры


Value.
 Строка, которую необходимо преобразовать
 в логическому типу данных.


## Описание


Метод Parse возвращает логическое
 значение, полученное при преобразовании строкового значения.


## Комментарии


В качестве значения параметра Value
 допустимо указывать следующие значения:


	- True. Для получения
	 логического значения «Истина».


	- False. Для получения
	 логического значения «Ложь».


Для всех остальных значений параметра Value
 методом будет сгенерирована исключительная ситуация.


## Пример


	Sub UserProc;

	Var

	    s: String;

	    b: Boolean;

	Begin

	    s := "True";

	    b := Boolean.Parse(s);

	End Sub UserProc;


После выполнения примера в переменной b будет содержаться логическое
 значение «True», полученное путем преобразования из указанного строкового
 значения.
