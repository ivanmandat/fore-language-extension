# Boolean.FalseString


## Синтаксис


FalseString: String;


## Описание


Метод FalseString возвращает
 логическое значение False, преобразованное
 к строковому виду.


## Пример


	Sub UserProc;

	Var

	    s: String;

	Begin

	    s := Boolean.FalseString;

	End Sub UserProc;


После выполнения примера в переменной s будет содержаться значение «False».
