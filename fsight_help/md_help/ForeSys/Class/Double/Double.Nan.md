# Double.Nan


## Синтаксис


Nan: Double;


## Описание


Свойство NaN возвращает фиксированное
 нечисловое значение.


## Комментарии


Значением этой константы является результат деления нуля на ноль. Данная
 константа возвращается в том случае, если результат операции является
 неопределенным.


Для определения, относится ли значение к нечисловому типу, используйте
 метод [Double.IsNan](Double.IsNan.htm).
