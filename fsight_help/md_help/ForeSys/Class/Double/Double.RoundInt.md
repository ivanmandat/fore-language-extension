# Double.RoundInt


## Синтаксис


RoundInt(Value: Double): Integer;


## Параметры


Value. Значение вещественного
 типа, которое необходимо округлить.


## Описание


Метод RoundInt округляет вещественное
 значение, передаваемое посредством входного параметра Value,
 до целого значения.


## Комментарии


Результатом выполнения метода является значение целого типа.


## Пример


	Sub UserProc;

	Var

	    D: Double;

	    I: Integer;

	Begin

	    D := 234.234;

	    I := Double.RoundInt(D);

	End Sub UserProc;


После выполнения примера в переменной «I» будет
 содержаться значение 234.
