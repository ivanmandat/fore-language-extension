# Double.Epsilon


## Синтаксис


Epsilon: Double;


## Описание


Свойство Epsilon возвращает
 самое минимальное положительное значение вещественного типа, отличное
 от нуля.


## Пример


	Sub UserProc;

	Var

	    D: Double;

	Begin

	    D := Double.Epsilon;

	End Sub UserProc;


После выполнения примера в переменной «D» будет
 находиться самое минимальное значение т.е. 4,94065645841247E-324.
