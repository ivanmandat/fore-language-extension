# Double.FloorInt


## Синтаксис


FloorInt(Value: Double): Integer;


## Параметры


Value. Значение вещественного
 типа, которое необходимо округлить.


## Описание


Метод FloorInt округляет вещественное
 значение, которое передается в качестве входного параметра Value,
 в меньшую сторону до целого значения.


## Комментарии


Результатом выполнения метода является значение целого типа.


## Пример


	Sub UserProc;

	Var

	    D:Double;

	    I:Integer;

	Begin

	    D := 234.734;

	    I := Double.FloorInt(D);

	End Sub UserProc;


После выполнения примера в переменной «I» будет содержаться значение
 234.
