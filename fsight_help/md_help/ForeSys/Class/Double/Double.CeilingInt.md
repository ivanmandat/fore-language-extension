# Double.CeilingInt


## Синтаксис


CeilingInt(Value: Double): Integer;


## Параметры


Value. Значение вещественного типа, которое необходимо округлить.


## Описание


Метод CeilingInt округляет вещественное
 значение, передаваемое посредством входного параметра Value,
 в большую сторону до целого значения.


## Комментарии


Результатом выполнения метода является значение целого типа.


## Пример


	Sub UserProc;

	Var

	    D: Double;

	    I: Integer;

	Begin

	    D := 234.234;

	    I := Double.CeilingInt(D);

	End Sub UserProc;


После выполнения примера в переменной «i» будет содержаться значение
 235.
