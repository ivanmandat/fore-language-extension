		 ![](../../Sub_Image.gif)
		 [CeilingInt](Double.CeilingInt.htm)
		 Метод CeilingInt округляет
		 вещественное значение в большую сторону до целого значения.


		 ![](../../Sub_Image.gif)
		 [Compare](Double.Compare.htm)
		 Метод Compare осуществляет
		 сравнение двух вещественных чисел без учета точности.


		 ![](../../Sub_Image.gif)
		 [Equals](Double.Equals.htm)
		 Метод Equals осуществляет

		 Метод Floor округляет

		 [FloorInt](Double.FloorInt.htm)
		 Метод FloorInt округляет
		 вещественное значение в меньшую сторону до целого значения.

		 [IsInfinity](Double.IsInfinity.htm)
		 Метод IsInfinity осуществляет

		 [IsNan](Double.IsNan.htm)
		 Метод IsNaN осуществляет

		 [IsNegativeInfinity](Double.IsNegativeInfinity.htm)
		 Метод IsNegativeInfinity

		 [IsPositiveInfinity](Double.IsPositiveInfinity.htm)
		 Метод IsPositiveInfinity

		 [TryParse](Double.TryParse.htm)
