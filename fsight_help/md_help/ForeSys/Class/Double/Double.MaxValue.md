# Double.MaxValue


## Синтаксис


MaxValue: Double;


## Описание


Свойство MaxValue возвращает
 максимальное значение типа Double.


## Пример


	Sub UserProc;

	Var

	    D: Double;

	Begin

	    D := Double.MaxValue;

	End Sub UserProc;


После выполнения примера в переменной «D» будет
 храниться максимальное значение типа Double, т.е. значение 1.79769313486232E+308.
