# Double.TryParse


## Синтаксис


TryParse(Value: String; Var Result: Double): Boolean;


## Параметры


Value. Строковое значение,
 которую необходимо преобразовать к вещественному типу;


Result. Переменная, в которую
 будет помещено значение в случае удачного преобразования.


## Описание


Метод TryParse преобразует строковое
 значение к типу Double с проверкой на корректность строки.


## Комментарии


Если преобразование удалось, то метод возвращает значение True
 и преобразованное значение помещается в переменную, указанную в параметре
 Result.


## Пример


	Sub UserProc;

	Var

	    B: Boolean;

	    D: Double;

	    S: String;

	Begin

	    S := "234.234";

	    B := Double.TryParse(S, D);

	End Sub UserProc;


После выполнения примера, если в переменной «B»
 будет содержаться значение True,
 значит преобразование удачное и в переменной «D» будет значение 234,234.
