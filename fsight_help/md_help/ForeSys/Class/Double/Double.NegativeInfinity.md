# Double.NegativeInfinity


## Синтаксис


NegativeInfinity: Double;


## Описание


Свойство NegativeInfinity возвращает
 значение, эквивалентное отрицательной бесконечности.


## Пример


	Sub UserProc;

	Var

	    D: Double;

	Begin

	    D := Double.NegativeInfinity;

	End Sub UserProc;


После выполнения примера в переменной «D» будет
 содержаться значение, эквивалентное отрицательной бесконечности, т.е. значение
 -1.#INF.
