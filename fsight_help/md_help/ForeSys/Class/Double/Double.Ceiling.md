# Double.Ceiling


## Синтаксис


Ceiling(Value: Double): Double;


## Параметры


Value. Значение вещественного
 типа, которое необходимо округлить.


## Описание


Метод Ceiling округляет вещественное
 значение, передаваемое посредством входного параметра Value,
 в большую сторону до целого значения.


## Комментарии


Результатом выполнения метода является значение вещественного типа.


## Пример


	Sub UserProc;

	Var

	    D: Double;

	Begin

	    D := 234.234;

	    D := Double.Ceiling(D);

	End Sub UserProc;


После выполнения примера в переменной «D» будет
 содержаться значение 235.
