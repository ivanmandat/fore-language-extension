# Double.IsNan


## Синтаксис


IsNan(Value: Double): Boolean;


## Параметры


Value. Проверяемое значение
 вещественного типа.


## Описание


Метод IsNaN осуществляет проверку
 эквивалентности значения переменной фиксированному нечисловому значению.


## Комментарии


Допустимые значения:


	- True. Значение переменной
	 эквивалентно фиксированному нечисловому значению;


	- False. Значение
	 переменной не эквивалентно фиксированному нечисловому значению.


Примечание.
 Фиксированным нечисловым значением является результат деления нуля на
 ноль. Данное значение возвращается в том случае, если результат операции
 является неопределенным.
