# Double.Equals


## Синтаксис


Equals(Value1: Double; Value2: Double): Boolean;


## Параметры


Value1. Первое сравниваемое
 значение вещественного типа;


Value2. Второе сравниваемое
 значение вещественного типа.


## Описание


Метод Equals осуществляет сравнение
 и возвращает признак равенства двух вещественных чисел.


## Комментарии


Допустимые значения:


	- True. Числа равны;


	- False. Во всех остальных
	 случаях.
