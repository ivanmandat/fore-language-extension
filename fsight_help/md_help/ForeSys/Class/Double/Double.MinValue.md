# Double.MinValue


## Синтаксис


MinValue: Double;


## Описание


Свойство MinValue возвращает
 минимальное значение типа Double.


## Пример


	Sub UserProc;

	Var

	    D: Double;

	Begin

	    D := Double.MinValue;

	End Sub UserProc;


После выполнения примера в переменной «D» будет
 храниться минимальное значение типа Double, т.е. значение -1.79769313486232E+308.
