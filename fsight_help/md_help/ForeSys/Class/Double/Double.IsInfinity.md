# Double.IsInfinity


## Синтаксис


IsInfinity(Value: Double): Boolean;


## Параметры


Value. Проверяемое значение
 вещественного типа.


## Описание


Метод IsInfinity осуществляет
 проверку эквивалентности значения переменной положительной или отрицательной
 бесконечности.


## Комментарии


Допустимые значения:


	- True. Значение переменной
	 эквивалентно положительной или отрицательной бесконечности;


	- False. Значение переменной
	 не эквивалентно положительной или отрицательной бесконечности.


## Пример


	Sub UserProc;

	Var

	    B: Boolean;

	    D: Double;

	Begin

	    D := Double.PositiveInfinity;

	    B := Double.IsInfinity(D);

	End Sub UserProc;


После выполнения примера в переменной «B» будет
 содержаться признак, определяющий, эквивалентно ли значение переменной
 «D» положительной или отрицательной бесконечности, т.е. значению
 1.#INF или -1.#INF.
