# Char.IsUpper


## Синтаксис


IsUpper: Boolean;


## Описание


Свойство IsUpper возвращает
 признак содержания в переменной значения в верхнем регистре.


## Комментарии


	- True. Значение имеет
	 верхний регистр;


	- False. Во всех остальных
	 случаях.
