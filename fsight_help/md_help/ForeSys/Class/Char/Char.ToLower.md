# Char.ToLower


## Синтаксис


ToLower(Source: Char): Char;


## Параметры


Source.
 Переменная, значение которой необходимо перевести в нижний регистр.


## Описание


Метод ToLower переводит значение
 переменной, передаваемой посредством параметра Source,
 в нижний регистр.
