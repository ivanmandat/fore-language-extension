# Char.Chr


## Синтаксис


Chr(CharCode: Integer): Char;


## Параметры


CharCode.
 Код символа.


## Описание


Свойство Chr возвращает символ,
 код которого передается посредством входного параметра.


## Комментарии


Для получения символа, ASCII-код которого больше 127, в параметр CharCode передайте код символа
 в кодировке Unicode.


## Пример


	Sub UserProc;

	Var

	    s: String;

	Begin

	    s := Char.Chr(100);

	End Sub UserProc;


После выполнения примера в переменной s будет содержаться символ с ASCII-кодом
 - 100.
