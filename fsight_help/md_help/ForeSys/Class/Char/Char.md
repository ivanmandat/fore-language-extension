# Char


## Описание


Класс Char представляет [символьный тип данных](Fore.chm::/03_dataTypes/Fore_DataTypes.htm).


## Свойства класса Char


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Chr](Char.Chr.htm)
		 Свойство Chr возвращает
		 символ, код которого передается посредством входного параметра.


## Методы класса Char


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ASCII](Char.ASCII.htm)
		 Метод ASCII возвращает
		 ASCII-код символа, передаваемого в качестве входного параметра.


		 ![](../../Sub_Image.gif)
		 [ToLower](Char.ToLower.htm)
		 Метод ToLower переводит

		 [ToUpper](Char.ToUpper.htm)
		 Метод ToUpper переводит
		 значение указанной переменной в верхний регистр.
