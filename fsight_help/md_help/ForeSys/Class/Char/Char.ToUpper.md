# Char.ToUpper


## Синтаксис


ToUpper(Source: Char): Char;


## Параметры


Source. Переменная, значение
 которой необходимо перевести в верхний регистр.


## Описание


Метод ToUpper переводит значение
 переменной, передаваемой посредством параметра Source,
 в верхний регистр.
