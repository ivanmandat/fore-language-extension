# Char.Asc


## Синтаксис


Asc: Integer;


## Описание


Свойство Asc возвращает код
 символа.


## Комментарии


Для символов, ASCII-код которых меньше 127, свойство вернет код из таблицы
 ASCII кодов. Для символов, ASCII-код которых больше 127, свойство вернет
 код в кодировке Unicode.


## Пример


	Sub UserProc;

	Var

	    s: String;

	    i: Integer;

	Begin

	    s := "Text example";

	    i := s.Chars(3).Asc;

	End Sub UserProc;


После выполнения примера в переменной i будет содержаться ASCII-код
 символа t, т.е. 84.
