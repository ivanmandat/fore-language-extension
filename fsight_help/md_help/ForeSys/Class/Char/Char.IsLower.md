# Char.IsLower


## Синтаксис


IsLower: Boolean;


## Описание


Свойство IsLower возвращает
 признак содержания в переменной значения в нижнем регистре.


## Комментарии


	- True. Значение имеет
	 нижний регистр;


	- False. Во всех остальных
	 случаях.
