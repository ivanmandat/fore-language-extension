# Object


## Описание


Класс Object является базовым
 для всех классов и типов, используемых в языке Fore.


## Комментарии


Любой объект может храниться в переменных, объявленных с типом данных
 [Object](Fore.chm::/03_dataTypes/Fore_DataTypes.htm).
 Данный класс не реализует собственных свойств и методов.
