# Object.Create


## Синтаксис


Create;


## Описание


Конструктор Create создает экземпляр объекта класса.


## Комментарии


Данный конструктор является базовым и доступен для всех классов объектов.
 Результатом работы конструктора является новый экземпляр объекта того
 класса, для которого он был вызван. Данный конструктор не содержит параметров
 и должен использоваться вместе с ключевым словом [New](Fore.chm::/07_Operations/Operations_of_the_creation_object.htm).


## Пример


			Class A: Object

    //Реализация членов класса

End Class A;


Sub Main;

Var

    Obj: A;

Begin

    Obj := New A.Create;

    //Использование объекта Obj
End Sub Main;


Данный пример является общим примером по созданию экземпляров объектов любых классов.
