# IRegexMatch


Сборка: System;


## Описание


Интерфейс IRegexMatch предназначен
 для работы с результатами отбора с помощью регулярного выражения.


## Иерархия наследования


IRegexMatch


## Комментарии


Результаты отбора с помощью регулярного выражения возвращает метод [IRegex.Match](../IRegex/IRegex.Match.htm).

		 [Success](IRegexMatch.Success.htm)
