# IRegexMatch.Group


## Синтаксис


Group: Array;


## Описание


Свойство Group возвращает массив
 строк, содержащий соответствия подвыражений (групп) шаблона регулярного
 выражения.


## Комментарии


Каждый элемент массива соответствует результату захвата группы в регулярном
 выражении. Первый элемент в массиве содержит полное совпадение, последующие
 элементы - совпадения подвыражений.
