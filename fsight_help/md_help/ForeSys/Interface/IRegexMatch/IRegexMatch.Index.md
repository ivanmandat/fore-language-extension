# IRegexMatch.Index


## Синтаксис


Index: Integer;


## Описание


Свойство Index возвращает позицию
 первого вхождения регулярного выражения в тексте.


## Комментарии


Если текст, соответствующий регулярному выражению не найден, то свойство
 возвращает значение -1.
