# IRegexMatch.Success


## Синтаксис


Success: Boolean;


## Описание


Свойство Success возвращает
 признак корректности выполнения регулярного выражения.


## Комментарии


Свойство возвращает значение True,
 если в методе [IRegex.Match](../IRegex/IRegex.Match.htm) был
 задан корректный шаблон регулярного выражения, который успешно применён
 к тексту. Значение True возвращается,
 даже если не было найдено ни одного соответствия. Значение False
 возвращается, если был задан некорректный шаблон регулярного выражения.
