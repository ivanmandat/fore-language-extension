# IExpressionStringHandler.OnTerm


## Синтаксис


OnTerm(Text: String);


## Параметры


Text. Текст, соответствующий
 распознаваемому элементу выражения.


## Описание


Метод OnTerm реализует событие,
 которое наступает при обращении к элементу выражения, содержащему ссылку
 на данные (либо сами данные).


## Комментарии


Событие срабатывает при обновлении или обращении к какому-либо терму
 в выражении. Текст терма будет доступен в параметре Text.
