# IExpressionStringHandler


## Описание


Интерфейс IExpressionStringHandler
 содержит методы, реализующие события возникающие при преобразовании элементов,
 содержащихся в выражении.


## Комментарии


Методы интерфейса должны быть переопределены в пользовательском классе,
 который является наследником класса [Object](../../Class/Object/Object.htm)
 и данного интерфейса. Если требуется переопределение и отслеживание только
 части событий, то класс может быть унаследован от класса [ExpressionStringHandler](../../Class/ExpressionStringHandler/ExpressionStringHandler.htm).


Созданный экземпляр класс является обработчиком событий, используемым
 при работе с выражениями и может быть указан в качестве значения свойства
 [IExpression.StringHandler](../IExpression/IExpression.StringHandler.htm).
 События наступают при установке или чтении [выражения](../IExpression/IExpression.AsString.htm)
 и могут использоваться для отслеживания правильности разбора выражения
 на термы и текст.
