# IExpressionStringHandler.OnText


## Синтаксис


OnText(Text: String);


## Параметры


Text. Текст, соответствующий
 распознаваемому элементу выражения.


## Описание


Метод OnText реализует событие,
 которое наступает при обращении к элементу, соответствующему символьной
 части выражения.


## Комментарии


Символьная часть не имеет привязки к данным, используется для организации
 выражения. Может содержать в себе различные скобки, знаки операций и прочее.


Событие срабатывает при обращении к какому-либо тексту в выражении.
 Текст будет доступен в параметре Text.
