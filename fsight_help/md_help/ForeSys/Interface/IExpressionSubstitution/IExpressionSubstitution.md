# IExpressionSubstitution


## Описание


Интерфейс IExpressionSubstitution
 содержит свойства подстановки, используемой при составлении выражений.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Reduction](IExpressionSubstitution.Reduction.htm)
		 Свойство Reduction
		 определяет сокращенное обозначение, которое будет использоваться
		 при составлении выражения.


		 ![](../../Property_Image.gif)
		 [Substitution](IExpressionSubstitution.Substitution.htm)
		 Свойство Substitution
		 определяет полное обозначение, которое будет использоваться при
		 расчетах выражения.
