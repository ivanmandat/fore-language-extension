# IViewerUpdateSettings.UpdateByIntervalEnabled


## Синтаксис


UpdateByIntervalEnabled: Boolean;


## Описание


Свойство UpdateByIntervalEnabled
 определяет, будет ли отчет обновляться автоматически с заданным интервалом.


## Комментарии


Допустимые значения:


	- True. Отчет будет обновляться
	 автоматически с заданным интервалом;


	- False. Отчет не будет
	 обновляться автоматически с заданным интервалом.
