# IViewerUpdateSettings.AutoUpdateByData


## Синтаксис


AutoUpdateByData: Boolean;


## Описание


Свойство AutoUpdateByData определяет,
 будет ли отчет обновляться при смене отметки.


## Комментарии


Допустимые значения:


	- True. Отчет будет обновляться
	 при смене отметки;


	- False. Отчет не будет
	 обновляться при смене отметки.
