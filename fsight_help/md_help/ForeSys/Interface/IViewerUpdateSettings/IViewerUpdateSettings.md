# IViewerUpdateSettings


Сборка: System;


## Описание


Интерфейс IViewerUpdateSettings
 предназначен для настройки параметров автообновления отчета.


## Иерархия наследования


IViewerUpdateSettings


## Комментарии


Параметры автообновления возвращают следующие свойства:


	- [IAdhocReport.UpdateSettings](KeAdhoc.chm::/Interface/IAdhocReport/IAdhocReport.UpdateSettings.htm);


	- [IEaxOptionsBase.UpdateSettings](KeExpress.chm::/Interface/IEaxOptionsBase/IEaxBaseOptions.UpdateSettings.htm);


	- [IPrxReportOptions.UpdateSettings](KeReport.chm::/Interface/IPrxReportOptions/IPrxReportOptions.UpdateSettings.htm).
