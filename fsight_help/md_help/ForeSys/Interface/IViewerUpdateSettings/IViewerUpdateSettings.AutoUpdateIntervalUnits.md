# IViewerUpdateSettings.AutoUpdateIntervalUnits


## Синтаксис


AutoUpdateIntervalUnits: [TimeUnits](../../Enums/TimeUnits.htm);


## Описание


Свойство AutoUpdateIntervalUnits
 определяет единицу измерения времени интервала автообновления.


## Комментарии


Для определения интервала времени, через которое будет выполняться автообновление
 отчета, используйте свойство [IViewerUpdateSettings.AutoUpdateInterval](IViewerUpdateSettings.AutoUpdateInterval.htm).
