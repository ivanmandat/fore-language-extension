# IViewerUpdateSettings.AutoUpdateInterval


## Синтаксис


AutoUpdateInterval: Double;


## Описание


Свойство AutoUpdateInterval
 определяет интервал времени, через которое будет выполняться автообновление
 отчета.


## Комментарии


Свойство актуально, если включено свойство [IViewerUpdateSettings.UpdateByIntervalEnabled](IViewerUpdateSettings.UpdateByIntervalEnabled.htm).
