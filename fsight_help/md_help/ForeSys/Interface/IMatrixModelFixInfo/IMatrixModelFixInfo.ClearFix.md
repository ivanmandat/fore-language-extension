# IMatrixModelFixInfo.ClearFix


## Синтаксис


ClearFix(DimIndex: Integer);


## Параметры


DimIndex. Номер измерения,
 по которому необходимо убрать фиксацию.


## Описание


Метод ClearFix осуществляет
 очистку фиксации по указанному измерению.
