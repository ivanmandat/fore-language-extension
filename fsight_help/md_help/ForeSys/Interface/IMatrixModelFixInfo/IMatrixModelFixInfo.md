# IMatrixModelFixInfo


## Описание


Интерфейс IMatrixModelFixInfo
 содержит свойства и методы, необходимые для фиксации измерений в матрице.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMatrixModelFixInfo.Count.htm)
		 Свойство Count возвращает
		 количество измерений, по которым может производится фиксация элементов.


		 ![](../../Property_Image.gif)
		 [DimFixed](IMatrixModelFixInfo.DimFixed.htm)
		 Свойство DimFixed возвращает
		 True, если для указанного
		 измерения имеется фиксация по какому-либо элементу.


		 ![](../../Property_Image.gif)
		 [FixedCount](IMatrixModelFixInfo.FixedCount.htm)
		 Свойство FixedCount
		 возвращает количество измерений матрицы, по которым присутствует
		 фиксация элементов.


		 ![](../../Property_Image.gif)
		 [Item](IMatrixModelFixInfo.Item.htm)
		 Свойство Item определяет
		 индекс элемента, по которому необходимо зафиксировать измерение.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Clear](IMatrixModelFixInfo.Clear.htm)
		 Метод Clear осуществляет
		 очистку фиксации по всем измерениям.


		 ![](../../Sub_Image.gif)
		 [ClearFix](IMatrixModelFixInfo.ClearFix.htm)
		 Метод ClearFix осуществляет
		 очистку фиксации по указанному измерению.
