# IMatrixModelFixInfo.DimFixed


## Синтаксис


DimFixed(DimIndex: Integer): Boolean;


## Параметры


DimIndex. Номер измерения.


## Описание


Свойство DimFixed возвращает
 True, если для указанного измерения
 имеется фиксация по какому-либо элементу.
