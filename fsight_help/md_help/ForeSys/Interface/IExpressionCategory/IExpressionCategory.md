# IExpressionCategory


## Описание


Интерфейс IExpressionCategory
 содержит свойства и методы для работы с категорией подстановок.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IExpressionCategory.Count.htm)
		 Свойство Count возвращает
		 количество подстановок в данной категории.


		 ![](../../Property_Image.gif)
		 [Name](IExpressionCategory.Name.htm)
		 Свойство Name определяет
		 наименование категории подстановок.


		 ![](../../Property_Image.gif)
		 [Substitution](IExpressionCategory.Substitution.htm)
		 Свойство Substitution
		 возвращает параметры подстановки с указанным индексом.


		 ![](../../Property_Image.gif)
		 [SubstitutionByReduction](IExpressionCategory.SubstitutionByReduction.htm)
		 Свойство SubstitutionByReduction
		 возвращает параметры подстановки с указанным сокращенным обозначением.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IExpressionCategory.Add.htm)
		 Метод Add осуществляет
		 добавление новой подстановки.


		 ![](../../Sub_Image.gif)
		 [Remove](IExpressionCategory.Remove.htm)
		 Метод Remove осуществляет
		 удаление указанной подстановки.
