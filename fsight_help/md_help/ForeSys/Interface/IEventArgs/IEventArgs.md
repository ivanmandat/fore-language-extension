# IEventArgs


Сборка: System;


## Описание


Интерфейс IEventArgs является
 базовым интерфейсом аргументов всех событий.


## Иерархия наследования


IEventArgs


## Комментарии


Интерфейс не содержит собственных свойств и методов и используется событиями,
 которые не передают сведения о состоянии обработчику при возникновении
 события.
