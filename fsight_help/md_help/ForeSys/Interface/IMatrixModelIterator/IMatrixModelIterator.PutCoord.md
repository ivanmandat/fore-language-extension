# IMatrixModelIterator.PutCoord


## Синтаксис


PutCoord(Coord: [IMatrixModelCoord](../IMatrixModelCoord/IMatrixModelCoord.htm));


## Параметры


Coord. Координата в матрице.


## Описание


Метод PutCoord осуществляет
 переход в итераторе к позиции, которая соответствует указанной координате
 матрицы.


## Комментарии


Если в матрице не существует элемента с указанной координатой, то матрица
 будет изменена в размере для появления элемента.
