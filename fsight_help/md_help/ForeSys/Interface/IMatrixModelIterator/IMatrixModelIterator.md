# IMatrixModelIterator


## Описание


Интерфейс IMatrixModelIterator
 содержит базовые свойства и методы итератора матрицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [MatrixModel](IMatrixModelIterator.MatrixModel.htm)
		 Свойство MatrixModel
		 возвращает объект-родитель.


		 ![](../../Property_Image.gif)
		 [Valid](IMatrixModelIterator.Valid.htm)
		 Свойство Valid возвращает
		 признак корректности перехода к какому-либо элементу итератора.


		 ![](../../Property_Image.gif)
		 [Value](IMatrixModelIterator.Value.htm)
		 Свойство Value определяет
		 значение элемента, на котором в данный момент находится курсор
		 итератора.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Goto](IMatrixModelIterator.Goto.htm)
		 Метод Goto осуществляет
		 переход в итераторе к позиции, которая соответствует указанной
		 координате матрицы.


		 ![](../../Sub_Image.gif)
		 [Move](IMatrixModelIterator.Move.htm)
		 Метод Move осуществляет
		 перемещение в указанном направлении.


		 ![](../../Sub_Image.gif)
		 [PutCoord](IMatrixModelIterator.PutCoord.htm)
		 Метод PutCoord осуществляет
		 переход в итераторе к позиции, которая соответствует указанной
		 координате матрицы.


		 ![](../../Sub_Image.gif)
		 [PutCurrentPos](IMatrixModelIterator.PutCurrentPos.htm)
		 Метод PutCurrentPos
		 вычисляет текущее положение в матрице.
