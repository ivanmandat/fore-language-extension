# IMatrixModelIterator.Goto


## Синтаксис


Goto(Coord: [IMatrixModelCoord](../IMatrixModelCoord/IMatrixModelCoord.htm));


## Параметры


Coord. Координата в матрице.


## Описание


Метод Goto осуществляет переход
 в итераторе к позиции, которая соответствует указанной координате матрицы.


## Комментарии


Для проверки корректности перехода используйте метод [IMatrixModelIterator.Valid](IMatrixModelIterator.Valid.htm).
