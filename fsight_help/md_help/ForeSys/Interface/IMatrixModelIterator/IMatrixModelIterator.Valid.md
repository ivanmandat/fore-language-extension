# IMatrixModelIterator.Valid


## Синтаксис


Valid: Boolean;


## Описание


Свойство Valid возвращает признак
 корректности перехода к какому-либо элементу итератора.


## Комментарии


Если осуществлен переход к несуществующему элементу, свойство возвращает
 False, иначе True.
