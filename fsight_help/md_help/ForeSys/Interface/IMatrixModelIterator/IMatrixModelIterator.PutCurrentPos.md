# IMatrixModelIterator.PutCurrentPos


## Синтаксис


PutCurrentPos(Result: [IMatrixModelCoord](../IMatrixModelCoord/IMatrixModelCoord.htm));


## Параметры


Result. Параметр, в который
 помещается текущая координата матрицы.


## Описание


Метод PutCurrentPos вычисляет
 текущее положение в матрице.


## Комментарии


После выполнения метода, в параметре Result
 будет содержаться текущая координата матрицы.
