# ILock.Active


## Синтаксис


Active: Boolean;


## Описание


Свойство Active возвращает признак
 включенной блокировки.


## Комментарии


Допустимые значения:


	- True. Блокировка включена.
	 Для отключения блокировки используйте метод [ILock.Unlock](ILock.Unlock.htm);


	- False. Блокировка отключена.
