# IBindingObjectParamValues


Сборка: System;


## Описание


Интерфейс IBindingObjectParamValues
 содержит свойства и методы для определения списка параметров для параметрического
 справочника.


## Иерархия наследования


IBindingObjectParamValues


## Комментарии


Коллекция параметров доступна в свойстве [IBindingDimCombo.ParamValues](../IBindingDimCombo/IBindingDimCombo.ParamValues.htm).
