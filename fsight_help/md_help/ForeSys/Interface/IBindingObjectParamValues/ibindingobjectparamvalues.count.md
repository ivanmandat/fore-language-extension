# IBindingObjectParamValues.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 параметров.


## Комментарии


Количество в списке определяется при формировании свойства [IBindingDimCombo.ParamValues](../IBindingDimCombo/IBindingDimCombo.ParamValues.htm)
 и соответствует количеству параметров в структуре справочника, который
 указывается в свойстве [IBindingDimCombo.Object](../IBindingDimCombo/ibindingdimcombo.object.htm)
 или определяется из строки связи при инициализации настроек редактора
 с помощью метода [IBindingManager.CreateByValue](../IBindingManager/IBindingManager.CreateByValue.htm).
