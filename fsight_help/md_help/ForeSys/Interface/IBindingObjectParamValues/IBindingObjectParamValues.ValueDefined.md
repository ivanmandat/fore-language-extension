# IBindingObjectParamValues.ValueDefined


## Синтаксис


ValueDefined(ItemId: String): Boolean;


## Параметры


ItemId. Идентификатор элемента
 списка.


## Описание


Свойство ValueDefined определяет
 признак наличия параметра с указанными идентификатором в [строке
 связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству ValueDefined установлено
 значение True, то в строке связи,
 описывающей редактор значения, будет присутствовать параметр с указанным
 идентификатором. Значение данного параметра определяет свойство [IBindingObjectParamValues.Value](IBindingObjectParamValues.Value.htm).
