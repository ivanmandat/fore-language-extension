# IJsonArrayBuilder.AddArray


## Синтаксис


		AddArray(Value: Array): [IJsonArrayBuilder](IJsonArrayBuilder.htm);


## Параметры


Value. Массив элементов.


## Описание


Метод AddArray добавляет массив
 значений в создаваемый JSON-массив.
