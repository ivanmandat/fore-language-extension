# IJsonArrayBuilder


Сборка: System;


## Описание


Интерфейс IJsonArrayBuilder
 предназначен для формирования нового массива в формате JSON.


## Иерархия наследования


IJsonArrayBuilder


## Комментарии


Для формирования нового JSON-массива создайте экземпляр класса [JsonArrayBuilder](../../Class/JsonArrayBuilder/JsonArrayBuilder.htm)
 и приведите его к интерфейсу IJsonArrayBuilder.
