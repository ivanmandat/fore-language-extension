# IJsonArrayBuilder.AddInteger


## Синтаксис


		AddInteger(Value: Integer): [IJsonArrayBuilder](IJsonArrayBuilder.htm);


## Параметры


Value. Целочисленное значение.


## Описание


Метод AddInteger добавляет целочисленное
 значение в JSON-массив.
