# ICultureInfo.ThreeLetterISOLanguageName


## Синтаксис


ThreeLetterISOLanguageName: String;


## Описание


Свойство ThreeLetterISOLanguageName
 возвращает 3х-буквенный код языка в соответствии со стандартом ISO 639-2.
