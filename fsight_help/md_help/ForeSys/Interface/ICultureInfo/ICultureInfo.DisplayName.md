# ICultureInfo.DisplayName


## Синтаксис


DisplayName: String;


## Описание


Свойство DisplayName возвращает
 имя культуры в формате <язык полностью> (<страна или регион полностью>)
 на языке локализованной версии .NET Framework.


Примечание.
 Язык установленной версии .NET Framework не зависит от языка, который
 установлен для отображения в операционной системе.
