# ICultureInfo.ThreeLetterWindowsLanguageName


## Синтаксис


ThreeLetterWindowsLanguageName: String;


## Описание


Свойство ThreeLetterWindowsLanguageName
 возвращает трехбуквенный код для языка, определенный в формате Windows
 API.
