# ICultureInfo.LocalizedLanguage


## Синтаксис


LocalizedLanguage: String;


## Описание


Свойство LocalizedLanguage возвращает
 наименование языка культуры на языке локализованной версии .NET Framework.


Примечание.
 Язык установленной версии .NET Framework не зависит от языка, который
 установлен для отображения в операционной системе.
