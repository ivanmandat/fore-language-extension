# ICultureInfo.ParseDate


## Синтаксис


ParseDate(Value: String): DateTime;


## Параметры


Value. Значение,
 приводимое к типу даты.


## Описание


Метод ParseDate преобразует
 указанное значение в дату в соответствии с настройками региональных стандартов.


## Комментарии


Если преобразование невозможно, то генерируется исключительная ситуация.


## Пример


	Sub UserProc;

	Var

	    CI: ICultureInfo;

	    d: DateTime;

	Begin

	    CI := CultureInfo.CurrentUi;

	    d := CI.ParseDate("14 июня 2005");

	End Sub UserProc;


При выполнении примера осуществляется преобразование из указанной строки
 в дату в соответствии с настройками культуры визуального интерфейса текущего
 приложения. Если преобразование пройдет успешно, то в переменной «d» будет
 содержаться дата «14.06.2005».
