# ICultureInfo.TwoLetterISOLanguageName


## Синтаксис


TwoLetterISOLanguageName: String;


## Описание


Свойство TwoLetterISOLanguageName
 возвращает 2х-буквенный код языка в соответствии со стандартом ISO 639-1.
