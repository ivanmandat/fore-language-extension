# ICultureInfo.LocalizedCountry


## Синтаксис


LocalizedCountry: String;


## Описание


Свойство LocalizedCountry возвращает
 наименование страны (региона) культуры на языке локализованной версии
 .NET Framework.


Примечание.
 Язык установленной версии .NET Framework не зависит от языка, который
 установлен для отображения в операционной системе.
