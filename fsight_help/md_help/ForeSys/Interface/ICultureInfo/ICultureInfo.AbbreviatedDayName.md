# ICultureInfo.AbbreviatedDayName


## Синтаксис


AbbreviatedDayName(DayIndex: Integer): String;


## Параметры


DayIndex. Индекс
 дня недели, у которого необходимо узнать сокращенное наименование.


## Описание


Свойство AbbreviatedDayName
 возвращает сокращенное наименование дня недели с указанным индексом.


## Комментарии


Возможны следующие значения параметра DayIndex:


1
 - Понедельник;


2
 - Вторник;


3
 - Среда;


4
 - Четверг;


5
 - Пятница;


6
 - Суббота;


0,
 7 - Воскресенье.
