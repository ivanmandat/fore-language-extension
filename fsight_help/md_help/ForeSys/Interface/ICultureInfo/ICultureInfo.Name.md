# ICultureInfo.Name


## Синтаксис


Name: String;


## Описание


Свойство Name возвращает имя
 культуры в сокращенном виде.


## Комментарии


Свойство формируется по стандарту RFC 1766 в формате <код языка2>-<код
 страны или региона2>, где <код языка2> - код из двух букв в нижнем
 регистре в соответствии с ISO 639-1, а <код страны или региона2>
 - код из двух букв в верхнем регистре в соответствии с ISO 3166. Если
 двухбуквенный код языка недоступен, используется трехбуквенный код в соответствии
 с ISO 639-2, например «div» для культуры, использующей язык дивехи. Если
 культура - нейтральная, то свойство Name
 представляется в формате <код языка2>.


## Пример


	Sub UserProc;

	Var

	    CF: ICultureInfo;

	Begin

	    CF := CultureInfo.UserDefault;

	    Debug.WriteLine(CF.DisplayName);

	    Debug.WriteLine(CF.EnglishName);

	    Debug.WriteLine(CF.Name);

	    Debug.WriteLine(CF.NativeName);

	End Sub UserProc;


При выполнении примера будет получена текущая культура, заданная в операционной
 системе. В консоль среды разработки будут выведены различные наименования
 культуры.
