# ICultureInfo.TryParseDouble


## Синтаксис


TryParseDouble(Value: String; Var Result: Double):
 Boolean;


## Параметры


Value. Значение, приводимое
 к вещественному типу;


Result. Переменная, в которую
 будет помещен результат в случае удачной попытки преобразования.


## Описание


Метод TryParseDouble осуществляет
 проверку на возможность преобразования указанного значения к вещественному
 типу в соответствии с настройками региональных стандартов.


## Комментарии


Если преобразование возможно, то в переменную Result
 будет помещен результат преобразования, и метод вернет значение True, иначе False.


## Пример


	Sub UserProc;

	Var

	    CI: ICultureInfo;

	    D: Double;

	    B: Boolean;

	Begin

	    CI := CultureInfo.Current;

	    B := CI.TryParseDouble("12345,78", D);

	End Sub UserProc;


После выполнения примера в переменной «D» будет содержаться значение
 «123456.78», преобразованное к вещественному типу, а в переменной «B»
 будет содержаться признак удачного преобразования.
