# ITask.Action


## Синтаксис


Action: [ITaskAction](../ITaskAction/ITaskAction.htm);


## Описание


Свойство Action возвращает параметры
 действия, которое выполняется в рамках текущей задачи.


## Комментарии


Данное свойство актуально только для задач, которые выполняют [действия](../ITaskAction/ITaskAction.htm)
 и которые добавлены в очередь выполнения задач с помощью метода [ITask.ContinueWith](ITask.ContinueWith.htm).


Для специфических задач данное свойство возвращает значение Null.
