# IBindingMaskEdit.InputTemplate


## Синтаксис


InputTemplate: String;


## Описание


Свойство InputTemplate определяет
 значение параметра «INPUTTEMPLATE»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Параметр «INPUTTEMPLATE» определяет
 шаблон ввода. Шаблон может содержать любой текст, также в шаблоне с помощью
 символа «_» указываются позиции ввода данных.


Свойство InputTemplate доступно
 для использования, если свойству [IBindingMaskEdit.InputTemplateDefined](IBindingMaskEdit.InputTemplateDefined.htm)
 установлено значение True. При
 изменении свойства InputTemplate
 свойству [IBindingMaskEdit.InputTemplateDefined](IBindingMaskEdit.InputTemplateDefined.htm)
 автоматически устанавливается значение True.
