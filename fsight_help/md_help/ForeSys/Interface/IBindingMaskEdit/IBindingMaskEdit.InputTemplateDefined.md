# IBindingMaskEdit.InputTemplateDefined


## Синтаксис


InputTemplateDefined: Boolean;


## Описание


Свойство InputTemplateDefined
 определяет признак наличия параметра «INPUTTEMPLATE»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству InputTemplateDefined
 установлено значение True, то
 в строке связи, описывающей редактор значения, будет присутствовать параметр
 «INPUTTEMPLATE». Значение данного
 параметра определяет свойство [IBindingMaskEdit.InputTemplate](IBindingMaskEdit.InputTemplate.htm).
