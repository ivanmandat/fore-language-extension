# IBindingMaskEdit.ValidCharsDefined


## Синтаксис


ValidCharsDefined: Boolean;


## Описание


Свойство ValidCharsDefined определяет
 признак наличия параметра «VALIDCHARS»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству ValidCharsDefined
 установлено значение True, то
 в строке связи, описывающей редактор значения, будет присутствовать параметр
 «VALIDCHARS». Значение данного
 параметра определяет свойство [IBindingMaskEdit.ValidChars](IBindingMaskEdit.ValidChars.htm).
