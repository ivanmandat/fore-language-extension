# IBindingMaskEdit.MaskDefined


## Синтаксис


MaskDefined: Boolean;


## Описание


Свойство MaskDefined определяет
 признак наличия параметра «MASK»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству MaskDefined установлено
 значение True, то в строке связи,
 описывающей редактор значения, будет присутствовать параметр «MASK».
 Значение данного параметра определяет свойство [IBindingMaskEdit.Mask](IBindingMaskEdit.Mask.htm).
