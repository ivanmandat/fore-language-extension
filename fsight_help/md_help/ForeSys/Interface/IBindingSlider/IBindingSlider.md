# IBindingSlider


Сборка: System;


## Описание


Интерфейс IBindingSlider
 предназначен для работы с базовыми параметрами редакторе значений в виде
 ползунка.


## Иерархия наследования


           [IBindingValue](../IBindingValue/IBindingValue.htm)


           IBindingSlider


## Комментарии


Доступны следующие виды ползунков:


	- [IBindingDecimalSlider](../IBindingDecimalSlider/IBindingDecimalSlider.htm).
	 Ползунок для десятичных значений;


	- [IBindingFloatSlider](../IBindingFloatSlider/IBindingFloatSlider.htm).
	 Ползунок для вещественных значений;


	- [IBindingIntegerSlider](../IBindingIntegerSlider/IBindingIntegerSlider.htm).
	 Ползунок для целых значений.
