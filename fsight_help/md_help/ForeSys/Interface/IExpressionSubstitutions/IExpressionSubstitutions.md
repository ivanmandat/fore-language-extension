# IExpressionSubstitutions


## Описание


Интерфейс IExpressionSubstitutions
 содержит свойства и методы для работы с коллекцией категорий подстановок.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Category](IExpressionSubstitutions.Category.htm)
		 Свойство Category возвращает
		 коллекцию подстановок, содержащихся в категории с указанным индексом.


		 ![](../../Property_Image.gif)
		 [CategoryByName](IExpressionSubstitutions.CategoryByName.htm)
		 Свойство CategoryByName
		 возвращает коллекцию подстановок, содержащихся в категории с указанным
		 наименованием.


		 ![](../../Property_Image.gif)
		 [Count](IExpressionSubstitutions.Count.htm)
		 Свойство Count возвращает
		 количество категорий подстановок.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IExpressionSubstitutions.Add.htm)
		 Метод Add создает новую
		 категорию подстановок.


		 ![](../../Sub_Image.gif)
		 [Remove](IExpressionSubstitutions.Remove.htm)
		 Метод Remove осуществляет
		 удаление категории со всеми подстановками.
