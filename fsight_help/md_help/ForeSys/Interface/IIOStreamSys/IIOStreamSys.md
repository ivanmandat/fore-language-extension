# IIOStreamSys


Сборка: System;


## Описание


Интерфейс IIOStreamSys является
 базовым для потоков, доступных в платформе.


## Иерархия наследования


IIOStreamSys


## Комментарии


Данный интерфейс не содержит собственных свойств и методов и является
 абстрактным для всех интерфейсов, реализующих различные потоки платформы.


Потоки в платформе реализуются следующими интерфейсами:


	- [IIOStream](ModIo.chm::/Interface/IIOStream/IIOStream.htm).


	- [IMemoryStream](ModIo.chm::/Interface/IMemoryStream/IMemoryStream.htm).


	- [IFileStream](ModIo.chm::/Interface/IFileStream/IFileStream.htm).
