# IMatrixModelSwapInfo


## Описание


Интерфейс IMatrixModelSwapInfo
 содержит свойства и методы необходимые для перестановки измерений в матрице.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMatrixModelSwapInfo.Count.htm)
		 Свойство Count возвращает
		 количество измерений, которые могут быть переставлены в матрице.


		 ![](../../Property_Image.gif)
		 [Item](IMatrixModelSwapInfo.Item.htm)
		 Свойство Item возвращает
		 текущий индекс измерения в матрице.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Swap](IMatrixModelSwapInfo.Swap.htm)
		 Метод Swap осуществляет
		 перестановку измерений.
