# INotFiniteNumberException


## Описание


Интерфейс INotFiniteNumberException
 содержит свойства исключительной ситуации, возникающей когда, в результате
 каких-либо действий, результирующим значением является плюс бесконечность,
 минус бесконечность, либо значение не является числовым (NaN).
