# IUndoRedoStorage


Сборка: System;


## Описание


Интерфейс IUndoRedoStorage предназначен
 для работы со сводной информацией стека изменений.


## Иерархия наследования


IUndoRedoStorage


## Комментарии


Сводную информацию можно получить, используя свойство [IUndoRedo.Storage](../IUndoRedo/IUndoRedo.Storage.htm).
 В сводной информации хранится общее количество действий повтора/отмены,
 доступных для текущего объекта, а также для его дочерних объектов. Например,
 если на листе отчёта расположена диаграмма, то в сводной информации будет
 доступно общее количество действий, произведённых отдельно на листе отчёта
 и отдельно на диаграмме.
