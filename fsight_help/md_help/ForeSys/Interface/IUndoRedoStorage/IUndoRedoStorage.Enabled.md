# IUndoRedoStorage.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled возвращает
 признак доступности стека изменений для пользователя.


## Комментарии


Допустимые значения:


	- True. Стек изменений
	 используется;


	- False. Стек изменений
	 не используется.
