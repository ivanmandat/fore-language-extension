# IBindingCheckBox.Value


## Синтаксис


Value: Boolean;


## Описание


Свойство Value определяет значение
 параметра «VALUE» в [строке
 связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Параметр «VALUE» определяет
 текущее состояние флага редактора. Если параметр имеет значение True, то флаг отмечен,
 иначе отметка с флага снята.


Свойство Value доступно для
 использования, если свойству [IBindingCheckBox.ValueDefined](ibindingcheckbox.valuedefined.htm)
 установлено значение True. При
 изменении свойства Value свойству
 [IBindingCheckBox.ValueDefined](ibindingcheckbox.valuedefined.htm)
 автоматически устанавливается значение True.
