# IBindingCheckBox.AlignDefined


## Синтаксис


AlignDefined: Boolean;


## Описание


Свойство AlignDefined определяет
 признак наличия параметра «ALIGN»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству AlignDefined установлено
 значение True, то в строке связи,
 описывающей редактор значения, будет присутствовать параметр «ALIGN».
 Значение данного параметра определяет свойство [IBindingCheckBox.Align](IBindingCheckBox.Align.htm).
