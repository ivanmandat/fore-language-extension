# IBindingCheckBox.Align


## Синтаксис


Align: [CheckAlignment](../../Enums/CheckAlignment.htm);


## Описание


Свойство Align определяет значение
 параметра «ALIGN» в [строке
 связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Параметр «ALIGN» определяет
 способ выравнивания флажка относительно текста. По умолчанию флажок располагается
 слева. Если флажок будет расположен по центру, то текст отображаться не
 будет.


Свойство Align доступно для
 использования, если свойству [IBindingCheckBox.AlignDefined](IBindingCheckBox.AlignDefined.htm)
 установлено значение True. При
 изменении свойства Align свойству
 [IBindingCheckBox.AlignDefined](IBindingCheckBox.AlignDefined.htm)
 автоматически устанавливается значение True.
