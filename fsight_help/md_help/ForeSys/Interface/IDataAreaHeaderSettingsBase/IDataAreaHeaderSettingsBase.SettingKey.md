# IDataAreaHeaderSettingsBase.SettingKey


## Синтаксис


SettingKey: Integer;


## Описание


Свойство SettingKey возвращает
 ключ для связи с объектом-родителем настроек заголовка в области данных.


## Комментарии


Если у настроек стиля заголовка нет объекта-родителя, то свойство вернет
 значение «0».
