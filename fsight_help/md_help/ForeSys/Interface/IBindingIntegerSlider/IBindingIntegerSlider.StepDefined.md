# IBindingIntegerSlider.StepDefined


## Синтаксис


StepDefined: Boolean;


## Описание


Свойство StepDefined определяет,
 присутствует ли в [строке
 связи](../IBindingValue/IBindingValue.AsString.htm) параметр «STEP».


## Комментарии


Допустимые значения:


	- True. В строке связи
	 присутствует параметр «STEP».
	 Для задания значения данного параметра используйте свойство [IBindingIntegerSlider.Step_](IBindingIntegerSlider.Step_.htm);


	- False. В строке связи
	 отсутствует параметр «STEP».
