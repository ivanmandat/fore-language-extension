# INamedEntityModel


## Описание


Интерфейс INamedEntityModel
 содержит базовые свойства объектов репозитория.


## Комментарии


Данный интерфейс является базовым для большого количества интерфейсов,
 реализующих что-либо в платформе. В определенном случае под объектом может
 пониматься репозиторий, отдельный объект репозитория, либо какой-либо
 дополнительный объект, используемый при работе другого объекта.
