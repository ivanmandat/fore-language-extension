# IExpressionErrorInfo


## Описание


Интерфейс IExpressionErrorInfo
 содержит свойства для получения информации об ошибке, которая может возникнуть
 при разборе некорректного выражения.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ErrorMessage](IExpressionErrorInfo.ErrorMessage.htm)
		 Свойство ErrorMessage
		 возвращает информационное сообщение об ошибке, возникшей при разборе
		 выражения.


		 ![](../../Property_Image.gif)
		 [Exception](IExpressionErrorInfo.Exception.htm)
		 Свойство Exception
		 возвращает ошибку, возникшую при разборе выражения.


		 ![](../../Property_Image.gif)
		 [Position](IExpressionErrorInfo.Position.htm)
		 Свойство Position возвращает
		 номер конечной позиции элемента, являющегося источником ошибки
		 в выражении.
