# IException.Source


## Синтаксис


Source: String;


## Описание


Свойство Source возвращает информацию
 об источнике ошибки.


## Комментарии


Для ошибок, произошедших в прикладном коде, свойство вернет идентификатор
 модуля/формы, в котором произошла ошибка.


Для ошибок, вызвавших текущее исключение и полученных в свойстве [NestedException](IException.NestedException.htm), свойство
 Source вернет наименование файла
 с внутренней реализацией, в котором произошла ошибка.
