# IException


## Описание


Интерфейс IException содержит
 базовые свойства, описывающие исключительную ситуацию, которая может возникнуть
 во время работы в платформе.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Line](IException.Line.htm)
		 Свойство Line возвращает
		 номер строки, при выполнении которой была вызвана исключительная
		 ситуация.


		 ![](../../Property_Image.gif)
		 [Message](IException.Message.htm)
		 Свойство Message возвращает
		 сообщение об ошибке.


		 ![](../../Property_Image.gif)
		 [MessageID](IException.MessageID.htm)
		 Свойство MessageID
		 возвращает цифровой идентификатор ошибки.


		 ![](../../Property_Image.gif)
		 [NestedException](IException.NestedException.htm)
		 Свойство NestedException
		 возвращает ошибку, вызвавшую текущее исключение.


		 ![](../../Property_Image.gif)
		 [Source](IException.Source.htm)
		 Свойство Source возвращает
		 информацию об источнике ошибки.
