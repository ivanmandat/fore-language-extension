# IJsonObjectBuilder.AddElement


## Синтаксис


AddElement(Property_: String; Value: [IJsonElement](../IJsonElement/IJsonElement.htm)):
 [IJsonObjectBuilder](IJsonObjectBuilder.htm);


## Параметры


Property_. Имя элемента;


Value. JSON-элемент.


## Описание


Метод AddElement добавляет элемент
 с другим элементом в качестве значения.
