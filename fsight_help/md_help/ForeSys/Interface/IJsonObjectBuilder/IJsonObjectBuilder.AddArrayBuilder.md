# IJsonObjectBuilder.AddArrayBuilder


## Синтаксис


AddArrayBuilder(Property_: String; Value: [IJsonArrayBuilder](../IJsonArrayBuilder/IJsonArrayBuilder.htm)):
 [IJsonObjectBuilder](IJsonObjectBuilder.htm);


## Параметры


Property_. Имя элемента;


Value. Формируемый JSON-массив,
 который будет являться значением элемента.


## Описание


Метод AddArrayBuilder добавляет
 элемент с JSON-массивом в качестве значения.
