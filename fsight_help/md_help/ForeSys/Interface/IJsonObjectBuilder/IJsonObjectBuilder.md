# IJsonObjectBuilder


Сборка: System;


## Описание


Интерфейс IJsonObjectBuilder
 предназначен для формирования нового объекта в формате JSON.


## Иерархия наследования


IJsonObjectBuilder


## Комментарии


Для формирования нового JSON-объекта создайте экземпляр класса [JsonObjectBuilder](../../Class/JsonObjectBuilder/JsonObjectBuilder.htm)
 и приведите его к интерфейсу IJsonObjectBuilder.
