# IMatrixModel.UpperBound


## Синтаксис


UpperBound(DimIndex: Integer): Integer;


## Параметры


DimIndex. Номер измерения,
 у которого необходимо получить верхнюю границу.


## Описание


Свойство UpperBound возвращает
 верхнюю границу измерения, номер которого передается в качестве входного
 параметра.


## Комментарии


Свойство оставлено для совместимости. При разработке рекомендуется использовать
 свойства [IMatrixModel.UpperBoundI](IMatrixModel.UpperBoundI.htm).
