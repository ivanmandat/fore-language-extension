# IMatrixModel.Clear


## Синтаксис


		Clear;


## Описание


Метод Clear осуществляет очистку
 содержимого элементов матрицы.


## Комментарий


Метод очищает внутренние структуры матрицы, не влияя на данные в базе
 данных.
