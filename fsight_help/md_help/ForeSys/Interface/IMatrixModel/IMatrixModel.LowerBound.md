# IMatrixModel.LowerBound


## Синтаксис


LowerBound(DimIndex: Integer): Integer;


## Параметры


DimIndex. Номер измерения,
 у которого необходимо получить нижнюю границу.


## Описание


Свойство LowerBound возвращает
 нижнюю границу измерения матрицы.


## Комментарии


Свойство оставлено для совместимости. При разработке рекомендуется использовать
 свойства [IMatrixModel.LowerBoundI](IMatrixModel.LowerBoundI.htm).
