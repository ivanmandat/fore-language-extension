# IBindingTreeCombo.SortTypeDefined


## Синтаксис


SortTypeDefined: Boolean;


## Описание


Свойство SortTypeDefined определяет
 признак наличия параметра «SORTTYPE»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству SortTypeDefined
 установлено значение True, то
 в строке связи, описывающей редактор значения, будет присутствовать параметр
 «SORTTYPE». Значение данного параметра
 определяет свойство [IBindingTreeCombo.SortType](IBindingTreeCombo.SortType.htm).
