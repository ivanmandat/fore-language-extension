# IBindingTreeCombo.SortType


## Синтаксис


SortType: Integer;


## Описание


Свойство SortType определяет
 значение параметра «SORTTYPE»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Свойство SortType доступно для
 использования, если свойству [IBindingTreeCombo.SortTypeDefined](IBindingTreeCombo.SortTypeDefined.htm)
 установлено значение True. При
 изменении свойства SortType свойству
 [IBindingTreeCombo.SortTypeDefined](IBindingTreeCombo.SortTypeDefined.htm)
 автоматически устанавливается значение True.
