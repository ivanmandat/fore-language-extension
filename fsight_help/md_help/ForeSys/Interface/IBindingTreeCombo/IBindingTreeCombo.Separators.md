# IBindingTreeCombo.Separators


## Синтаксис


Separators: String;


## Описание


Свойство Separators определяет
 значение параметра «SEPARATORS»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Параметр «SEPARATORS» определяет
 символ-разделитель наименований вершин комбинированного дерева. Если параметр
 не задан, то используется символ-разделитель «|».


Свойство Separators доступно
 для использования, если свойству [IBindingTreeCombo.SeparatorsDefined](IBindingTreeCombo.SeparatorsDefined.htm)
 установлено значение True. При
 изменении свойства Separators
 свойству [IBindingTreeCombo.SeparatorsDefined](IBindingTreeCombo.SeparatorsDefined.htm)
 автоматически устанавливается значение True.
