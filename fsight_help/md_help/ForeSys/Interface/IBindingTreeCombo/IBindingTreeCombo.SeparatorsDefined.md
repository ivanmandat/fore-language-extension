# IBindingTreeCombo.SeparatorsDefined


## Синтаксис


SeparatorsDefined: Boolean;


## Описание


Свойство SeparatorsDefined определяет
 признак наличия параметра «SEPARATORS»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству SeparatorsDefined
 установлено значение True, то
 в строке связи, описывающей редактор значения, будет присутствовать параметр
 «SEPARATORS». Значение данного
 параметра определяет свойство [IBindingTreeCombo.Separators](IBindingTreeCombo.Separators.htm).


## Пример


Пример использования свойства приведен в описании свойства [IBindingTreeCombo.NodesDefined](IBindingTreeCombo.NodesDefined.htm).
