# ISysSettings.Refresh


## Синтаксис


Refresh;


## Описание


Метод Refresh обновляет информацию
 в текущем экземпляре объекта, загружая возможные изменения из файла с
 настройками.


## Комментарии


Метод актуален для использования, если возможна работа с файлом настроек
 нескольких пользователей одновременно. После сохранения изменений одним
 пользователем, данные изменения будут обновлены у других пользователей
 при вызове метода Refresh.
