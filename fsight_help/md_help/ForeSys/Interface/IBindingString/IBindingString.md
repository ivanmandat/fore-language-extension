# IBindingString


Сборка: System;


## Описание


Интерфейс IBindingString содержит
 свойства для определения базовых параметров редакторов, которые меняют
 значения путем ввода/выбора какого-либо текста.


## Иерархия наследования


           [IBindingValue](../IBindingValue/IBindingValue.htm)


           IBindingString


## Комментарии


Интерфейс является базовым для следующих интерфейсов:


	- [IBindingComboBox](../IBindingComboBox/IBindingComboBox.htm);


	- [IBindingEditBox](../IBindingEditBox/IBindingEditBox.htm);


	- [IBindingIdentEdit](../IBindingIdentEdit/IBindingIdentEdit.htm);


	- [IBindingMaskEdit](../IBindingMaskEdit/IBindingMaskEdit.htm);


	- [IBindingTreeCombo](../IBindingTreeCombo/IBindingTreeCombo.htm).
