# IBindingString.MaxLengthDefined


## Синтаксис


MaxLengthDefined: Boolean;


## Описание


Свойство MaxLengthDefined определяет
 признак наличия параметра «MAXLENGTH»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству MaxLengthDefined
 установлено значение True, то
 в строке связи, описывающей редактор значения, будет присутствовать параметр
 «MAXLENGTH». Значение данного
 параметра определяет свойство [IBindingString.MaxLength](IBindingString.MaxLength.htm).

Пример использования приведен в описании
