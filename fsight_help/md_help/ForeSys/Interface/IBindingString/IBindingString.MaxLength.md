# IBindingString.MaxLength


## Синтаксис


MaxLength: Integer;


## Описание


Свойство MaxLength определяет
 значение параметра «MAXLENGTH»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Параметр «MAXLENGTH» определяет
 максимальную длину вводимой строки.


Свойство MaxLength доступно
 для использования, если свойству [IBindingString.MaxLengthDefined](IBindingString.MaxLengthDefined.htm)
 установлено значение True. При
 изменении свойства MaxLength свойству
 [IBindingString.MaxLengthDefined](IBindingString.MaxLengthDefined.htm)
 автоматически устанавливается значение True.
