# IJsonDocument


Сборка: System;


## Описание


Интерфейс IJsonDocument предназначен
 для работы с JSON-структурой.


## Иерархия наследования


IJsonDocument


## Комментарии


Методы интерфейса позволяют загружать и сохранять JSON-структуры из
 файла, потока или символьной строки. Для доступа к элементам JSON-структуры
 используйте свойство [RootElement](IJsonDocument.RootElement.htm).
