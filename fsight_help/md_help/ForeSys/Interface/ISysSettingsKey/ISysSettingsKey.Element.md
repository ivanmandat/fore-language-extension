# ISysSettingsKey.Element


## Синтаксис


Element: Variant;


## Описание


Свойство Element возвращает
 XML-представление текущего элемента.


## Комментарии


Данное свойство возвращает XML-представление описываемое интерфейсом
 IXmlDomElement. Используя свойства и методы, доступные для IXmlDomElement,
 можно изменять настройки, заданные в различных атрибутах/подэлементах
 текущего элемента.
