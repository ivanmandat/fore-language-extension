# IBindingComboBox.ListModeDefined


## Синтаксис


ListModeDefined: Boolean;


## Описание


Свойство ListModeDefined определяет
 признак наличия параметра «LISTMODE»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству ListModeDefined
 установлено значение True, то
 в строке связи, описывающей редактор значения, будет присутствовать параметр
 «LISTMODE». Значение данного параметра
 определяет свойство [IBindingComboBox.ListMode](IBindingComboBox.ListMode.htm).
