# IBindingComboBox.Separators


## Синтаксис


Separators: String;


## Описание


Свойство Separators определяет
 значение параметра «SEPARATORS»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Параметр «SEPARATORS» определяет
 символ-разделитель элементов комбинированного списка, указанных в свойстве
 [IBindingComboBox.Items](IBindingComboBox.Items.htm). По умолчанию
 используется символ-разделитель «|».


Свойство Separators доступно
 для использования, если свойству [IBindingComboBox.SeparatorsDefined](IBindingComboBox.SeparatorsDefined.htm)
 установлено значение True. При
 изменении свойства Separators
 свойству [IBindingComboBox.SeparatorsDefined](IBindingComboBox.SeparatorsDefined.htm)
 автоматически устанавливается значение True.
