# IBroadcast.AddHandler


## Синтаксис


AddHandler(Handler: [IBroadcastEventHandler](../IBroadcastEventHandler/IBroadcastEventHandler.htm);
 [GroupId: Integer = 0]): Integer;


## Параметры


Handler. Обработчик события,
 который назначается объекту.


GroupId. Параметр, необходимый
 для обработки приходящих сообщений.


## Описание


Метод AddHandler предназначен
 для подписки на сообщения [Broadcast](../../Class/Broadcast/Broadcast.htm).
