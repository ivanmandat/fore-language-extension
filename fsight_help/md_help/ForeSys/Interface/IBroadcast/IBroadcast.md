# IBroadcast


Сборка: System;


## Описание


Интерфейс IBroadcast содержит
 методы, позволяющие реализовать возможность обмена сообщениями между толстым
 и тонким клиентом.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddHandler](IBroadcast.AddHandler.htm)
		 Метод AddHandler предназначен
		 для подписки на сообщения [Broadcast](../../Class/Broadcast/Broadcast.htm).


		 ![](../../Sub_Image.gif)
		 [RemoveHandler](IBroadcast.RemoveHandler.htm)
		 Метод RemoveHandler
		 отменяет подписку группы пользователей на отправку сообщений.


		 ![](../../Sub_Image.gif)
		 [Send](IBroadcast.Send.htm)
		 Метод Send предназначен
		 для отправки сообщений группе подписчиков.
