# IJsonElement


Сборка: System;


## Описание


Интерфейс IJsonElement содержит
 базовые свойства и методы для работы с отдельными элементами JSON-структуры.


## Иерархия наследования


IJsonElement


## Комментарии


Интерфейс является базовым для следующих интерфейсов:


	- [IJsonArray](../IJsonArray/IJsonArray.htm);


	- [IJsonObject.](../IJsonObject/IJsonObject.htm)
