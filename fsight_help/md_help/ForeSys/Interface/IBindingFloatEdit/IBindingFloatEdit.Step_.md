# IBindingFloatEdit.Step


## Синтаксис


Step_:
 Double;


## Описание


Свойство Step_ определяет значение
 параметра «STEP» в [строке
 связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Параметр «STEP» определяет шаг
 изменения значения. Изменение значения на величину шага осуществляется
 с помощью кнопок с правой стороны редактора, либо клавишами UP и DOWN.


Свойство Step доступно для использования,
 если свойству [IBindingFloatEdit.StepDefined](IBindingFloatEdit.StepDefined.htm)
 установлено значение True. При
 изменении свойства Step свойству
 [IBindingFloatEdit.StepDefined](IBindingFloatEdit.StepDefined.htm)
 автоматически устанавливается значение True.
