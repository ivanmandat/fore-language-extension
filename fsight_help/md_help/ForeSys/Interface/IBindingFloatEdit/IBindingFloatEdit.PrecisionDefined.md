# IBindingFloatEdit.PrecisionDefined


## Синтаксис


PrecisionDefined: Boolean;


## Описание


Свойство PrecisionDefined определяет
 признак наличия параметра «PRECISION»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству PrecisionDefined
 установлено значение True, то
 в строке связи, описывающей редактор значения, будет присутствовать параметр
 «PRECISION». Значение данного
 параметра определяет свойство [IBindingFloatEdit.Precision](IBindingFloatEdit.Precision.htm).
