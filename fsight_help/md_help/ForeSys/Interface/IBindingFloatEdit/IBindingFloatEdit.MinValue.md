# IBindingFloatEdit.MinValue


## Синтаксис


MinValue:
 Double;


## Описание


Свойство MinValue определяет
 значение параметра «MINVAL» в
 [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Параметр «MINVAL» определяет
 минимальное возможное значение, которое можно будет установить в редакторе.


Свойство MinValue доступно для
 использования, если свойству [IBindingFloatEdit.MinValueDefined](IBindingFloatEdit.MinValueDefined.htm)
 установлено значение True. При
 изменении свойства MinValue свойству
 [IBindingFloatEdit.MinValueDefined](IBindingFloatEdit.MinValueDefined.htm)
 автоматически устанавливается значение True.
