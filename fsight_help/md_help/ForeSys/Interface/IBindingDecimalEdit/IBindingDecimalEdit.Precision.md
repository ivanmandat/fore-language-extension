# IBindingDecimalEdit.Precision


## Синтаксис


Precision: Integer;


## Описание


Свойство Precision определяет
 значение параметра «PRECISION»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Параметр «PRECISION» определяет
 точность значения (количество знаков
 после запятой).


Свойство Precision доступно
 для использования, если свойству [IBindingDecimalEdit.PrecisionDefined](IBindingDecimalEdit.PrecisionDefined.htm)
 установлено значение True. При
 изменении свойства Precision свойству
 [IBindingDecimalEdit.PrecisionDefined](IBindingDecimalEdit.PrecisionDefined.htm)
 автоматически устанавливается значение True.
