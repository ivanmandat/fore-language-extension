# IBindingDecimalEdit.Value


## Синтаксис


Value: Decimal;


## Описание


Свойство Value определяет значение
 параметра «VALUE» в [строке
 связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Параметр «VALUE» определяет
 значение, отображаемое при первом открытии редактора (значение по умолчанию).


Свойство Value доступно для
 использования, если свойству [IBindingDecimalEdit.ValueDefined](IBindingDecimalEdit.ValueDefined.htm)
 установлено значение True. При
 изменении свойства Value свойству
 [IBindingDecimalEdit.ValueDefined](IBindingDecimalEdit.ValueDefined.htm)
 автоматически устанавливается значение True.


## Пример
