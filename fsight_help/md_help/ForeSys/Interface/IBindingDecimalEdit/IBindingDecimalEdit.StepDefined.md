# IBindingDecimalEdit.StepDefined


## Синтаксис


StepDefined: Boolean;


## Описание


Свойство StepDefined определяет
 признак наличия параметра «STEP»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству StepDefined установлено
 значение True, то в строке связи,
 описывающей редактор значения, будет присутствовать параметр «STEP».
 Значение данного параметра определяет свойство [IBindingDecimalEdit.Step_](IBindingDecimalEdit.Step_.htm).
