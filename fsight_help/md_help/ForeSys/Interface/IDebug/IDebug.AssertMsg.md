# IDebug.AssertMsg


## Синтаксис


		AssertMsg(Condition: Boolean; Message: String);


## Параметры


Condition. Условие,
 проверяемое в методе;


Message. Пользовательское
 сообщение об ошибке, выводимое, если условие Condition принимает
 значение False.


## Описание


Метод AssertMsg проверяет условие
 Condition, и выводит
 пользовательское сообщение об ошибке, если условие принимает значение
 False.
