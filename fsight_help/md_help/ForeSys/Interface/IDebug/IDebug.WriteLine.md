# IDebug.WriteLine


## Синтаксис


WriteLine(Value: Variant);


## Параметры


Value. Значение, которое необходимо вывести в
 консоль.


## Описание


Метод WriteLine осуществляет
 вывод значения, передаваемого посредством параметра Value,
 с переводом курсора на новую строку.


## Комментарии


Для корректного вывода в данном параметре должны передаваться простые
 типы данных.


## Пример


	Sub UserProc;

	Var

	    v: Variant;

	Begin

	    v := "Текущие дата и время: ";

	    Debug.WriteLine(v);

	    v := DateTime.Now;

	    Debug.WriteLine(v);

	End Sub UserProc;


В результате выполнения примера, будет выведено сообщение «Текущие дата
 и время: » и текущая дата. Данные выводятся в различных строках.
