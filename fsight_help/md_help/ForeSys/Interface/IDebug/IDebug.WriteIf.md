# IDebug.WriteIf


## Синтаксис


		WriteIf(Condition: Boolean; Value: Variant);


## Параметры


Condition. Условие, при выполнении
 которого выводится значение, передаваемое в параметре Value;


Value. Значение, которое необходимо
 вывести в консоль. Для корректного вывода в данном параметре должны передаваться
 простые типы данных.


## Описание


Метод WriteIf осуществляет вывод
 значения, передаваемого посредством параметра Value,
 без перевода курсора на новую строку.


## Комментарии


Значение выводится, если выполняется условие Condition.


## Пример


Добавьте системную сборку MathFin.


					Sub UserProc;

		Var

		    a, b: Integer;

		Begin

		    a := Math.RandBetweenI(0, 100);

		    b := Math.RandBetweenI(0, 100);

		    Debug.Write("A=" + a.ToString + " B=" + b.ToString + " ");

		    Debug.WriteIf(a > b, "А больше B");

		    Debug.WriteIf(a < b, "B больше A");

		End Sub UserProc;


В результате выполнения примера, будут выведены значения двух сгенерированных
 чисел и сообщение, соответствующее установленным условиям.
