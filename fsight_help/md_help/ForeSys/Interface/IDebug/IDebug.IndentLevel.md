# IDebug.IndentLevel


## Синтаксис


		IndentLevel: Integer;


## Описание


Свойство IndentLevel определяет
 уровень отступа в консоли среды разработки.


## Комментарии


Суммарный отступ определяется по формуле IndentLevel*[IndentSize](IDebug.IndentSize.htm).
