# IDebug.Fail


## Синтаксис


		Fail(Message: String);


## Параметры


Message. Сообщение об ошибке,
 которое необходимо вывести.


## Описание


Метод Fail осуществляет вывод
 сообщения об ошибке, передаваемое посредством параметра Message.


## Комментарии


Сообщения об ошибках выводятся без учета отступов, всегда с начала строки.
