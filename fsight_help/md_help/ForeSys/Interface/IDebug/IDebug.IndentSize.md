# IDebug.IndentSize


## Синтаксис


		IndentSize: Integer;


## Описание


Свойство IndentSize определяет
 шаг одного отступа.


## Комментарии


Шаг устанавливается в пробелах. Суммарный отступ определяется по формуле
 [IndentLevel](IDebug.IndentLevel.htm)*IndentSize.
 По умолчанию свойству установлено значение 4 пробела.
