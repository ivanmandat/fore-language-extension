# IDebug.Assert


## Синтаксис


		Assert(Condition: Boolean);


## Параметры


Condition. Условие, проверяемое
 в методе.


## Описание


Метод Assert проверяет условие
 Condition, и выводит пользовательское
 сообщение об ошибке "Assert failed", если условие принимает
 значение False.
