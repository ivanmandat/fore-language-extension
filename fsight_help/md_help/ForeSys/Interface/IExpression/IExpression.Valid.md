# IExpression.Valid


## Синтаксис


Valid: Boolean;


## Описание


Свойство Valid возвращает признак
 корректного выражения.


## Комментарии


Допустимые значения:


	- True. Выражение корректно
	 и может использоваться для расчета какого-либо значения;


	- False. Выражение содержит
	 ошибку. Информацию об ошибке можно получить в свойстве [IExpression.ErrorInfo](IExpression.ErrorInfo.htm).
