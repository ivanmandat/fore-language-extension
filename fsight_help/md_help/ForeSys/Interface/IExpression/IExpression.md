# IExpression


Сборка: System;


## Описание


Интерфейс IExpression содержит
 свойства, используемые для работы с выражением, по которому вычисляются
 какие-либо значения.


## Иерархия наследования

## Комментарии


Выражение задаётся в строковом виде с помощью свойства [IExpression.AsString](IExpression.AsString.htm). Для
 проверки корректности выражения используйте свойство [IExpression.Valid](IExpression.Valid.htm).

[![](../../Opened.gif)![](../../Closed.gif)Особенности
 выражения, основанного на атрибутах базы данных временных рядов](javascript:TextPopup(this))


	Если выражение основано на атрибутах базы данных временных рядов,
	 то после значений обязательных атрибутов ряда через вертикальную черту
	 (|) можно указать требуемую календарную динамику. Динамика задается
	 следующими мнемониками:


		- A. Годовая;


		- S. Полугодовая;


		- Q. Квартальная;


		- M. Месячная;


		- W. Недельная;


		- D. Дневная.


	После разбора выражения указанная календарная динамика будет содержаться
	 в свойстве [IMsFormulaTransformSlice.Level](KeMs.chm::/Interface/IMsFormulaTransformSlice/IMsFormulaTransformSlice.Level.htm).


	Наименование ряда, основанного на атрибутах базы данных временных
	 рядов, указывается в фигурных скобках, например, {Afganistan|BCA|Q}.


	В данном элементе выражения «Afganistan» и «BCA» - это значения
	 обязательных атрибутов, а «Q» - указание того, что для ряда используется
	 квартальная динамика.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AsString](IExpression.AsString.htm)
		 Свойство AsString определяет
		 выражение, по которому будут вычисляться значения, в виде символьной
		 строки.


		 ![](../../Property_Image.gif)
		 [Empty](IExpression.Empty.htm)
		 Свойство Empty возвращает
		 True, если выражение не
		 установлено.


		 ![](../../Property_Image.gif)
		 [ErrorInfo](IExpression.ErrorInfo.htm)
		 Свойство ErrorInfo
		 возвращает информацию об ошибке, которая может возникнуть при
		 разборе выражения.


		 ![](../../Property_Image.gif)
		 [References](IExpression.References.htm)
		 Свойство References
		 определяет список модулей/форм, содержащих пользовательские функции,
		 используемые для создания выражения.


		 ![](../../Property_Image.gif)
		 [StringHandler](IExpression.StringHandler.htm)
		 Свойство StringHandler
		 определяет объект, используемый для отслеживания преобразования
		 термов, содержащихся в выражении.


		 ![](../../Property_Image.gif)
		 [Substitutions](IExpression.Substitutions.htm)
		 Свойство Substitutions
		 возвращает коллекцию подстановок, используемых в выражениях.


		 ![](../../Property_Image.gif)
		 [ThrowException](IExpression.ThrowException.htm)
		 Свойство ThrowException
		 определяет, будет ли генерироваться исключительная ситуация платформы
		 при наличии ошибок в разбираемом выражении.


		 ![](../../Property_Image.gif)
		 [Valid](IExpression.Valid.htm)
		 Свойство Valid возвращает
		 признак корректного выражения.
