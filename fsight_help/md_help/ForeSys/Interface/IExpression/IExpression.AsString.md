# IExpression.AsString


## Синтаксис


AsString: String;


## Описание


Свойство AsString определяет
 выражение, по которому будут вычисляться значения, в виде символьной строки.


## Комментарии


Для проверки корректности выражения используйте свойство [IExpression.Valid](IExpression.Valid.htm).
