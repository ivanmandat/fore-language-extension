# IMatrixModelCoord.Assign


## Синтаксис


Assign(Coord: [IMatrixModelCoord](IMatrixModelCoord.htm));


## Параметры


Coord. Координата, параметры
 которой необходимо установить для текущей координаты.


## Описание


Метод Assign заполняет параметры
 текущей координаты параметрами входной координаты.
