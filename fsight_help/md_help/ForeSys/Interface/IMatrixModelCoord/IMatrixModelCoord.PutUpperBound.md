# IMatrixModelCoord.PutUpperBound


## Синтаксис


PutUpperBound(DimIndex: Integer);


## Параметры


DimIndex.
 Номер измерения.


## Описание


Метод PutUpperBound для
 указанного параметра координаты устанавливает индекс последнего элемента
 соответствующего измерения матрицы.
