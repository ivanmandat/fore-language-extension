# IMatrixModelCoord.PutLowerBound


## Синтаксис


PutLowerBound(DimIndex: Integer);


## Параметры


DimIndex. Номер измерения.


## Описание


Метод PutLowerBound для указанного
 параметра координаты устанавливает индекс первого элемента соответствующего
 измерения матрицы.
