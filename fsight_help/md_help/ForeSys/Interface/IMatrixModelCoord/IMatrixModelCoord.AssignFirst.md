# IMatrixModelCoord.AssignFirst


## Синтаксис


AssignFirst(Count: Integer; Coord: [IMatrixModelCoord](IMatrixModelCoord.htm));


## Параметры


Count. Количество параметров
 координаты Coord, которые будут установлены для текущей координаты;


Coord.
 Координата, параметры которой необходимо установить для текущей координаты.


## Описание


Метод AssignFirst заполняет
 указанное количество параметров текущей координаты параметрами входной
 координаты.
