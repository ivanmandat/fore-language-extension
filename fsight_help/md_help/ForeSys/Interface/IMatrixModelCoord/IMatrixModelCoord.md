# IMatrixModelCoord


## Описание


Интерфейс IMatrixModelCoord
 содержит свойства и методы для работы с координатами матрицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMatrixModelCoord.Count.htm)
		 Свойство Count возвращает
		 количество параметров координаты.


		 ![](../../Property_Image.gif)
		 [Item](IMatrixModelCoord.Item.htm)
		 Свойство Item определяет
		 индекс элемента в измерении матрицы для указанного параметра координаты.


		 ![](../../Property_Image.gif)
		 [MatrixModel](IMatrixModelCoord.MatrixModel.htm)
		 Свойство MatrixModel
		 возвращает объект-родитель.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Assign](IMatrixModelCoord.Assign.htm)
		 Метод Assign заполняет
		 параметры текущей координаты параметрами входной координаты.


		 ![](../../Sub_Image.gif)
		 [AssignFirst](IMatrixModelCoord.AssignFirst.htm)
		 Метод AssignFirst заполняет
		 указанное количество параметров текущей координаты параметрами
		 входной координаты.


		 ![](../../Sub_Image.gif)
		 [PutFirst](IMatrixModelCoord.PutFirst.htm)
		 Метод PutFirst устанавливает
		 координату первого элемента матрицы.


		 ![](../../Sub_Image.gif)
		 [PutLast](IMatrixModelCoord.PutLast.htm)
		 Метод PutLast устанавливает
		 координату последнего элемента матрицы.


		 ![](../../Sub_Image.gif)
		 [PutLowerBound](IMatrixModelCoord.PutLowerBound.htm)
		 Метод PutLowerBound
		 для указанного параметра координаты устанавливает индекс
		 первого элемента соответствующего измерения матрицы.


		 ![](../../Sub_Image.gif)
		 [PutUpperBound](IMatrixModelCoord.PutUpperBound.htm)
		 Метод PutUpperBound для
		 указанного параметра координаты устанавливает индекс последнего
		 элемента соответствующего измерения матрицы.
