# IBindingDateTimePicker.SpinDefined


## Синтаксис


SpinDefined: Boolean;


## Описание


Свойство SpinDefined определяет
 признак наличия параметра «SPIN»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству SpinDefined установлено
 значение True, то в строке связи,
 описывающей редактор значения, будет присутствовать параметр «SPIN».
 Значение данного параметра определяет свойство [IBindingDateTimePicker.Spin](IBindingDateTimePicker.Spin.htm).
