# IBindingDateTimePicker.CheckBoxFocusDefined


## Синтаксис


CheckBoxFocusDefined: Boolean;


## Описание


Свойство CheckBoxFocusDefined
 определяет признак наличия параметра «SET_FOCUS_ON_CHECK_BOX»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству CheckBoxFocusDefined
 установлено значение True, то
 в строке связи, описывающей редактор значения, будет присутствовать параметр
 «SET_FOCUS_ON_CHECK_BOX». Значение
 данного параметра определяет свойство [IBindingDateTimePicker.CheckBoxFocus](IBindingDateTimePicker.CheckBoxFocus.htm).
