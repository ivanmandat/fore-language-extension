# IBindingDateTimePicker.ValueDefined


## Синтаксис


ValueDefined: Boolean;


## Описание


Свойство ValueDefined определяет
 признак наличия параметра «VALUE»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству ValueDefined установлено
 значение True, то в строке связи,
 описывающей редактор значения, будет присутствовать параметр «VALUE».
 Значение данного параметра определяет свойство [IBindingDateTimePicker.Value](IBindingDateTimePicker.Value.htm).
