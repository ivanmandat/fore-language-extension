# IBindingDateTimePicker.Value


## Синтаксис


Value: DateTime;


## Описание


Свойство Value определяет значение
 параметра «VALUE» в [строке
 связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Параметр «VALUE» определяет
 для редактора значение по умолчанию. Данное значение будет отображено
 в редакторе при его первом открытии.


Свойство Value доступно для
 использования, если свойству [IBindingDateTimePicker.ValueDefined](IBindingDateTimePicker.ValueDefined.htm)
 установлено значение True. При
 изменении свойства Value свойству
 [IBindingDateTimePicker.ValueDefined](IBindingDateTimePicker.ValueDefined.htm)
 автоматически устанавливается значение True.
