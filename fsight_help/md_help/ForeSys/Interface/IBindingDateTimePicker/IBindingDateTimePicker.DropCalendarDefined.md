# IBindingDateTimePicker.DropCalendarDefined


## Синтаксис


DropCalendarDefined: Boolean;


## Описание


Свойство DropCalendarDefined
 определяет признак наличия параметра «DROPCALENDAR»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству DropCalendarDefined
 установлено значение True, то
 в строке связи, описывающей редактор значения, будет присутствовать параметр
 «DROPCALENDAR». Значение данного
 параметра определяет свойство [IBindingDateTimePicker.DropCalendar](IBindingDateTimePicker.DropCalendar.htm).
