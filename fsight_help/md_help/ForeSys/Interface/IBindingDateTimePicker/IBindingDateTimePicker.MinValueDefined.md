# IBindingDateTimePicker.MinValueDefined


## Синтаксис


MinValueDefined: Boolean;


## Описание


Свойство MinValueDefined определяет
 признак наличия параметра «MINVAL»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству MinValueDefined
 установлено значение True, то
 в строке связи, описывающей редактор значения, будет присутствовать параметр
 «MINVAL». Значение данного параметра
 определяет свойство [IBindingDateTimePicker.MinValue](IBindingDateTimePicker.MinValue.htm).
