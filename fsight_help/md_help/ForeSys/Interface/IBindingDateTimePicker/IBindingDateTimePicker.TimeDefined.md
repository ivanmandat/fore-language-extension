# IBindingDateTimePicker.TimeDefined


## Синтаксис


TimeDefined: Boolean;


## Описание


Свойство TimeDefined определяет
 признак наличия параметра «SHOWTIME»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству TimeDefined установлено
 значение True, то в строке связи,
 описывающей редактор значения, будет присутствовать параметр «SHOWTIME».
 Значение данного параметра определяет свойство [IBindingDateTimePicker.Time](IBindingDateTimePicker.Time.htm).
