# IBindingDateTimePicker.MinValue


## Синтаксис


MinValue: DateTime;


## Описание


Свойство MinValue определяет
 значение параметра «MINVAL» в
 [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Параметр «MINVAL» определяет
 минимальное значение, которое можно будет установить в редакторе.


Свойство MinValue доступно для
 использования, если свойству [IBindingDateTimePicker.MinValueDefined](IBindingDateTimePicker.MinValueDefined.htm)
 установлено значение True. При
 изменении свойства MinValue свойству
 [IBindingDateTimePicker.MinValueDefined](IBindingDateTimePicker.MinValueDefined.htm)
 автоматически устанавливается значение True.
