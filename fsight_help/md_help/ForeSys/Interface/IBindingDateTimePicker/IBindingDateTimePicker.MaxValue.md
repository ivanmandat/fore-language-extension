# IBindingDateTimePicker.MaxValue


## Синтаксис


MaxValue: DateTime;


## Описание


Свойство MaxValue определяет
 значение параметра «MAXVAL» в
 [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Параметр «MAXVAL» определяет
 максимальное значение, которое можно будет установить в редакторе.


Свойство MaxValue доступно для
 использования, если свойству [IBindingDateTimePicker.MaxValueDefined](IBindingDateTimePicker.MaxValueDefined.htm)
 установлено значение True. При
 изменении свойства MaxValue свойству
 [IBindingDateTimePicker.MaxValueDefined](IBindingDateTimePicker.MaxValueDefined.htm)
 автоматически устанавливается значение True.
