# IBindingDateTimePicker.CheckBoxFocus


## Синтаксис


CheckBoxFocus: Boolean;


## Описание


Свойство CheckBoxFocus определяет
 значение параметра «SET_FOCUS_ON_CHECK_BOX»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если параметр «SET_FOCUS_ON_CHECK_BOX»
 имеет значение True, то фокус
 будет находиться в области переключателя, иначе фокус будет находиться
 в области значения дня/часа, в зависимости от того, какой редактор настраивается.


Свойство CheckBoxFocus доступно
 для использования, если свойству [IBindingDateTimePicker.CheckBoxFocusDefined](IBindingDateTimePicker.CheckBoxFocusDefined.htm)
 установлено значение True. При
 изменении свойства CheckBoxFocus
 свойству [IBindingDateTimePicker.CheckBoxFocusDefined](IBindingDateTimePicker.CheckBoxFocusDefined.htm)
 автоматически устанавливается значение True.
