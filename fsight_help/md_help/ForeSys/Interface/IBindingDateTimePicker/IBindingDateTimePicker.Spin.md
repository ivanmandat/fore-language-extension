# IBindingDateTimePicker.Spin


## Синтаксис


Spin: Boolean;


## Описание


Свойство Spin определяет значение
 параметра «SPIN» в [строке
 связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если параметр «SPIN» имеет значение
 True, то в редакторе будут присутствовать
 две кнопки для увеличения/уменьшения значений компонент даты/времени.


Свойство Spin доступно для использования,
 если свойству [IBindingDateTimePicker.SpinDefined](IBindingDateTimePicker.SpinDefined.htm)
 установлено значение True. При
 изменении свойства Spin свойству
 [IBindingDateTimePicker.SpinDefined](IBindingDateTimePicker.SpinDefined.htm)
 автоматически устанавливается значение True.
