# IBindingDimCombo.SelectionModeDefined


## Синтаксис


SelectionModeDefined: Boolean;


## Описание


Свойство SelectionModeDefined
 определяет признак наличия параметра «SELECTIONMODE»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству SelectionModeDefined
 установлено значение True, то
 в строке связи, описывающей редактор значения, будет присутствовать параметр
 «SELECTIONMODE». Значение данного
 параметра определяет свойство [IBindingDimCombo.SelectionMode](IBindingDimCombo.SelectionMode.htm).

Пример использования приведен в описании свойства [IBindingDimCombo.DataSourceKeyDefined](ibindingdimcombo.datasourcekeydefined.htm).
