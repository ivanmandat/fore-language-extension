# IBindingDimCombo


Сборка: System;


## Описание


Интерфейс IBindingDimCombo содержит
 свойства для определения параметров типа редактора значений «[Раскрывающийся
 список справочника](UiNav.chm::/GUI/ValueEditorParameters.htm#dimension)».
