# IBindingDimCombo.MultiLevelsDefined


## Синтаксис


MultiLevelsDefined: Boolean;


## Описание


Свойство MultiLevelsDefined
 определяет признак наличия параметра «MULTILEVELS»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству MultiLevelsDefined
 установлено значение True, то
 в строке связи, описывающей редактор значения, будет присутствовать параметр
 «MULTILEVELS». Значение данного
 параметра определяет свойство [IBindingDimCombo.MultiLevels](IBindingDimCombo.MultiLevels.htm).
