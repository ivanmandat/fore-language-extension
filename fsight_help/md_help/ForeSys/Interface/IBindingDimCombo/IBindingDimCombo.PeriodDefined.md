# IBindingDimCombo.PeriodDefined


## Синтаксис


PeriodDefined: Boolean;


## Описание


Свойство PeriodDefined определяет
 признак наличия параметра «PERIOD»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству PeriodDefined
 установлено значение True, то
 в строке связи, описывающей редактор значения, будет присутствовать параметр
 «PERIOD». Значение данного параметра
 определяет свойство [IBindingDimCombo.Period](IBindingDimCombo.Period.htm).
