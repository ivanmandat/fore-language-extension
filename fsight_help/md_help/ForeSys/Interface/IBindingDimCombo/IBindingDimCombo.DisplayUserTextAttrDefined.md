# IBindingDimCombo.DisplayUserTextAttrDefined


## Синтаксис


DisplayUserTextAttrDefined:
 Boolean;


## Описание


Свойство DisplayUserTextAttrDefined
 определяет признак наличия параметра «DISPLAYUSERTEXTATTR»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Допустимые значения:


	- True. В строке связи
	 содержится параметр «DISPLAYUSERTEXTATTR»;


	- False. В строке связи
	 не содержится параметр «DISPLAYUSERTEXTATTR».


Значение параметра «DISPLAYUSERTEXTATTR»
 определяется свойством [IBindingDimCombo.DisplayUserTextAttr](IBindingDimCombo.DisplayUserTextAttr.htm).
