# IBindingDimCombo.StoreAsStringDefined


## Синтаксис


StoreAsStringDefined:
 Boolean;


## Описание


Свойство StoreAsStringDefined
 определяет признак наличия параметра «STOREASSTRING»
 в [строке
 связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Допустимые значения:


	- True. В строке связи
	 содержится параметр «STOREASSTRING»;


	- False. В строке связи
	 не содержится параметр «STOREASSTRING».


Значение параметра «STOREASSTRING»
 определяется свойством [IBindingDimCombo.StoreAsString](IBindingDimCombo.StoreAsString.htm).
