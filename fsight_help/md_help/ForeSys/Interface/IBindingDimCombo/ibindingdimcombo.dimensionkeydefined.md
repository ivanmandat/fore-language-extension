# IBindingDimCombo.DimensionKeyDefined


## Синтаксис


DimensionKeyDefined: Boolean;


## Описание


Свойство DimensionKeyDefined
 определяет признак наличия параметра «DIM_KEY»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству DimensionKeyDefined
 установлено значение True, то
 в строке связи, описывающей редактор значения, будет присутствовать параметр
 «DIM_KEY». Значение данного параметра
 определяет свойство [IBindingDimCombo.DimensionKey](IBindingDimCombo.DimensionKey.htm).
