# IBindingDimCombo.NamespaceDefined


## Синтаксис


NamespaceDefined:
 Boolean;


## Описание


Свойство NamespaceDefined определяет
 признак наличия параметра «NAMESPACE»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Допустимые значения:


	- True. В строке связи
	 содержится параметр «NAMESPACE»;


	- False. В строке связи
	 не содержится параметр «NAMESPACE».


Значение параметра «NAMESPACE»
 определяется свойством [IBindingDimCombo.Namespace_](IBindingDimCombo.Namespace_.htm).
