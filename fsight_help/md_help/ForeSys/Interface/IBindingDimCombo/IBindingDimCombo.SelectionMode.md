# IBindingDimCombo.SelectionMode


## Синтаксис


SelectionMode: [SelectionModeEnum](../../Enums/SelectionModeEnum.htm);


## Описание


Свойство SelectionMode определяет
 значение параметра «SELECTIONMODE»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Параметр «SELECTIONMODE» определяет
 режим отметки элементов справочника.


Свойство SelectionMode доступно
 для использования, если свойству [IBindingDimCombo.SelectionModeDefined](ibindingdimcombo.selectionmodedefined.htm)
 установлено значение True. При
 изменении свойства SelectionMode
 свойству [IBindingDimCombo.SelectionModeDefined](ibindingdimcombo.selectionmodedefined.htm)
 автоматически устанавливается значение True.
