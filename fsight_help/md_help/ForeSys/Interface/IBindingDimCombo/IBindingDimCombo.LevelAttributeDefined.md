# IBindingDimCombo.LevelAttributeDefined


## Синтаксис


LevelAttributeDefined: Boolean;


## Описание


Свойство LevelAttributeDefined
 определяет признак наличия параметра «LEVELATTR»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству LevelAttributeDefined
 установлено значение True, то
 в строке связи, описывающей редактор значения, будет присутствовать параметр
 «LEVELATTR». Значение данного
 параметра определяет свойство [IBindingDimCombo.LevelAttribute](ibindingdimcombo.LevelAttribute.htm).


## Пример


Пример использования приведен в описании свойства [IBindingDimCombo.DataSourceKeyDefined](ibindingdimcombo.datasourcekeydefined.htm).
