# IBindingDimCombo.CustomMultiselectTextDefined


## Синтаксис


CustomMultiselectTextDefined: Boolean;


## Описание


Свойство CustomMultiselectTextDefined
 определяет признак наличия параметра «CUSTOMMULTISELECTTEXT»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству CustomMultiselectTextDefined
 установлено значение True, то
 в строке связи, описывающей редактор значения, будет присутствовать параметр
 «CUSTOMMULTISELECTTEXT». Значение
 данного параметра определяет свойство [IBindingDimCombo.CustomMultiselectText](IBindingDimCombo.CustomMultiselectText.htm).

Пример использования приведен в описании свойства [IBindingDimCombo.DataSourceKeyDefined](ibindingdimcombo.datasourcekeydefined.htm).
