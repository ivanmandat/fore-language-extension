# IBindingDimCombo.Schema


## Синтаксис


Schema: String;


## Описание


Свойство Schema определяет значение
 параметра «SCHEMA» в [строке
 связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Параметр «SCHEMA» определяет
 идентификатор схемы отметки, элементы которой всегда будут отмечены в
 редакторе.


Свойство Schema доступно для
 использования, если свойству [IBindingDimCombo.SchemaDefined](ibindingdimcombo.schemadefined.htm)
 установлено значение True. При
 изменении свойства Schema свойству
 [IBindingDimCombo.SchemaDefined](ibindingdimcombo.schemadefined.htm)
 автоматически устанавливается значение True.


## Пример


Пример использования приведен в описании свойства [IBindingDimCombo.Group](ibindingdimcombo.group.htm).
