# IBindingDimCombo.ModeLevelsDefined


## Синтаксис


ModeLevelsDefined: Boolean;


## Описание


Свойство ModeLevelsDefined определяет
 признак наличия параметра «MODLEVELS»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству ModeLevelsDefined
 установлено значение True, то
 в строке связи, описывающей редактор значения, будет присутствовать параметр
 «MODELEVELS». Значение данного
 параметра определяет свойство [IBindingDimCombo.ModeElements](IBindingDimCombo.ModeElements.htm).

Пример использования приведен в описании свойства [IBindingDimCombo.ModeLevels](IBindingDimCombo.ModeLevels.htm).
