# IBindingDimCombo.ValueAttributeDefined


## Синтаксис


ValueAttributeDefined


## Описание


Свойство ValueAttributeDefined
 определяет признак наличия параметра «ATTRIBUTEVALUE»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству ValueAttributeDefined
 установлено значение True, то
 в строке связи, описывающей редактор значения, будет присутствовать параметр
 «ATTRIBUTEVALUE». Значение данного
 параметра определяет свойство [IBindingDimCombo.ValueAttribute](IBindingDimCombo.ValueAttribute.htm).


## Пример


Пример использования приведен в описании свойства [IBindingDimCombo.DataSourceKeyDefined](ibindingdimcombo.datasourcekeydefined.htm).
