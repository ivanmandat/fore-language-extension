# IBindingDimCombo.GroupDefined


## Синтаксис


GroupDefined: Boolean;


## Описание


Свойство GroupDefined определяет
 признак наличия параметра «GROUP»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству GroupDefined установлено
 значение True, то в строке связи,
 описывающей редактор значения, будет присутствовать параметр «GROUP».
 Значение данного параметра определяет свойство [IBindingDimCombo.Group](ibindingdimcombo.group.htm).

Пример использования приведен в описании свойства [IBindingDimCombo.DataSourceKeyDefined](ibindingdimcombo.datasourcekeydefined.htm).
