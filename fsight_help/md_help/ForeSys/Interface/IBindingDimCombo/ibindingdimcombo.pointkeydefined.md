# IBindingDimCombo.PointKeyDefined


## Синтаксис


PointKeyDefined: Boolean;


## Описание


Свойство PointKeyDefined определяет
 признак наличия параметра «PT_KEY»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Если свойству PointKeyDefined
 установлено значение True, то
 в строке связи, описывающей редактор значения, будет присутствовать параметр
 «PT_KEY». Значение данного параметра
 определяет свойство [IBindingDimCombo.PointKey](IBindingDimCombo.PointKey.htm).
