# IBindingDimCombo.DisplayUserListAttrDefined


## Синтаксис


DisplayUserListAttrDefined: Boolean;


## Описание


Свойство DisplayUserListAttrDefined
 определяет признак наличия параметра «DISPLAYUSERLISTATTR»
 в [строке связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Допустимые значения:


	- True. В строке связи
	 содержится параметр «DISPLAYUSERLISTATTR»;


	- False. В строке связи
	 не содержится параметр «DISPLAYUSERLISTATTR».


Значение параметра «DISPLAYUSERLISTATTR»
 определяется свойством [IBindingDimCombo.DisplayUserListAttr](IBindingDimCombo.DisplayUserListAttr.htm).
