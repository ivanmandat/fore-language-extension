# IBindingDimCombo.Object


## Синтаксис


Object: String;


## Описание


Свойство Object определяет значение
 параметра «ID» в [строке
 связи](../IBindingValue/IBindingValue.AsString.htm).


## Комментарии


Параметр «ID» определяет идентификатор
 справочника репозитория, элементы которого будут отображены в редакторе.


Свойство Object доступно для
 использования, если свойству [IBindingDimCombo.ObjectDefined](ibindingdimcombo.objectdefined.htm)
 установлено значение True. При
 изменении свойства Object свойству
 [IBindingDimCombo.ObjectDefined](ibindingdimcombo.objectdefined.htm)
 автоматически устанавливается значение True.
