# IDataAreaHeaderStyleSettingsBase


Сборка: System;


## Описание


Интерфейс IDataAreaHeaderStyleSettingsBase
 содержит базовые свойства стилей оформления различных заголовков в области
 данных.


## Иерархия наследования


IDataAreaHeaderStyleSettingsBase


## Комментарии


Область данных описывает интерфейс [IEaxDataArea](KeExpress.chm::/Interface/IEaxDataArea/IEaxDataArea.htm).
 Стили оформления заголовков, описываемых интерфейсом IDataAreaHeaderStyleSettingsBase,
 возвращает метод [IEaxGridViewSettings.GetStyleSettings](KeExpress.chm::/Interface/IEaxGridViewSettings/IEaxGridViewSettings.GetStyleSettings.htm).
 К типу IDataAreaHeaderStyleSettingsBase могут быть приведены следующие
 элементы:


	- заголовок измерения, описываемого интерфейсом [IPivotDimension](KePivot.chm::/Interface/IPivotDimension/IPivotDimension.htm);


	- заголовок уровня измерения, описываемого интерфейсом [IPivotDimensionLevel](KePivot.chm::/Interface/IPivotDimensionLevel/IPivotDimensionLevel.htm);


	- заголовок области данных, описываемый интерфейсом [IPivotHeader](KePivot.chm::/Interface/IPivotHeader/IPivotHeader.htm).
