# IDataAreaHeaderStyleSettingsBase.StyleSettingKey


## Синтаксис


StyleSettingKey: Integer;


## Описание


Свойство StyleSettingKey возвращает
 ключ для связи с объектом-родителем настроек стиля оформления заголовка
 в области данных.


## Комментарии


Если у настроек заголовка нет объекта-родителя, то свойство вернет значение
 «0».
