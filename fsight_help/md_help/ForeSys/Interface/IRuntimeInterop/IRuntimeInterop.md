# IRuntimeInterop

## Описание


Интерфейс IRuntimeInterop содержит
 статические свойства класса [RuntimeInterop](../../Class/RuntimeInterop/RuntimeInterop.htm),
 предназначенного для получения информации об ошибках платформы.
