# ITaskAction


Сборка: System;


## Описание


Интерфейс ITaskAction содержит
 метод позволяющий реализовать действие, которое будет выполнено при запуске
 задачи.


## Иерархия наследования


ITaskAction


## Комментарии


Метод [ITask.Action](ITaskAction.Action.htm) должен быть
 переопределен в пользовательском классе. Результат выполнения действия
 будет доступен в свойстве [ITask.Result](../ITask/ITask.Result.htm)
 и будет соответствовать тому значению, которое возвращает метод [ITask.Action](ITaskAction.Action.htm).
