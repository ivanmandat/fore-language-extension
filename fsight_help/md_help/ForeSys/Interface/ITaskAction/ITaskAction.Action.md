# ITaskAction.Action


## Синтаксис


Action(Antecedent: [ITask](../ITask/ITask.htm)):
 Variant;


## Параметры


Antecedent. Задача, после выполнения
 которой была запущена задача выполнения текущего действия.


## Описание


Метод Action реализует действие,
 которое будет выполнено при запуске на выполнение задачи.


## Комментарии


Данный метод должен быть переопределен в пользовательском классе. В
 рамках действия также может быть создана и запущена в асинхронном режиме
 какая-либо задача.


Значение, возвращаемое методом, будет доступно в свойстве [ITask.Result](../ITask/ITask.Result.htm)
 после завершения выполнения задачи.


## Пример


Пример использования данного метода приведен в описании метода [ITask.ContinueWith](../ITask/ITask.ContinueWith.htm).
