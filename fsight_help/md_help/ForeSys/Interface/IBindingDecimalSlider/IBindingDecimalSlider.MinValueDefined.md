# IBindingDecimalSlider.MinValueDefined


## Синтаксис


MinValueDefined: Boolean;


## Описание


Свойство MinValueDefined определяет,
 присутствует ли в [строке
 связи](../IBindingValue/IBindingValue.AsString.htm) параметр «MINVAL».


## Комментарии


Допустимые значения:


	- True. В строке связи
	 присутствует параметр «MINVAL».
	 Для задания значения данного параметра используйте свойство [IBindingDecimalSlider.MinValue](IBindingDecimalSlider.MinValue.htm);


	- False. В строке связи
	 отсутствует параметр «MINVAL».
