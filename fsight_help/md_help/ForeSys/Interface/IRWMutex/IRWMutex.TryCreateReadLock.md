# IRWMutex.TryCreateReadLock


## Синтаксис


TryCreateReadLock: [ILock](../ILock/ILock.htm);


## Описание


Метод TryCreateReadLock осуществляет
 попытку создать объект, используемый для управления блокировкой на чтение
 данных.


## Комментарии


В случае успешного выполнения метод возвращает объект, используемый
 для управления блокировкой на чтение данных. Если создать объект для управления
 блокировкой нельзя, например, уже имеется блокировка на чтение, то метод
 возвращает значение Null.
