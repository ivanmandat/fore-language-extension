# IRWMutex.UnlockRead


## Синтаксис


UnlockRead;


## Описание


Метод UnlockRead снимает блокировку
 на чтение.


## Комментарии


Данный метод должен быть вызван после окончания чтения данных, если
 предварительно вызывался метод [IRWMutex.LockRead](IRWMutex.LockRead.htm).
