# IRWMutex.UnlockWrite


## Синтаксис


UnlockWrite;


## Описание


Метод UnlockWrite снимает блокировку
 на запись.


## Комментарии


Данный метод должен быть вызван после окончания изменения данных, если
 предварительно вызывался метод [IRWMutex.LockWrite](IRWMutex.LockWrite.htm).
