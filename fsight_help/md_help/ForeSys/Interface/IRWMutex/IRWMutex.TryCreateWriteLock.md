# IRWMutex.TryCreateWriteLock


## Синтаксис


TryCreateWriteLock: [ILock](../ILock/ILock.htm);


## Описание


Метод TryCreateWriteLock осуществляет
 попытку создать объект, используемый для управления блокировкой на изменение
 данных.


## Комментарии


В случае успешного выполнения метод возвращает объект, используемый
 для управления блокировкой на изменение данных. Если создать объект для
 управления блокировкой нельзя, например, уже имеется блокировка на изменение,
 то метод возвращает значение Null.
