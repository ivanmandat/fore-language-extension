# IUndoRedo.RedoCount


## Синтаксис


RedoCount: Integer;


## Описание


Свойство RedoCount возвращает
 количество действий, которые хранятся в стеке повтора.


## Комментарии


Количество действий, хранящихся в стеке повтора, ограничивается свойством
 [IUndoRedo.Limit](IUndoRedo.Limit.htm).


## Пример


Пример использования приведен в описании свойства [IUndoRedo.Enabled](IUndoRedo.Enabled.htm).
