# IUndoRedo.UndoState


## Синтаксис


UndoState(Index: Integer): [IUndoRedoState](../IUndoRedoState/IUndoRedoState.htm);


## Параметры


Index. Индекс
 состояния в стеке изменений.


## Описание


Свойство UndoState возвращает
 состояние действия в стеке отмены, передаваемое посредством входного параметра.

Пример использования приведен в описании свойства [IUndoRedo.Enabled](IUndoRedo.Enabled.htm).
