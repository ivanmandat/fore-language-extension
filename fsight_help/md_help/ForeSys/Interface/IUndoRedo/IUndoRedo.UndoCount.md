# IUndoRedo.UndoCount


## Синтаксис


UndoCount: Integer;


## Описание


Свойство UndoCount возвращает
 количество действий, которые хранятся в стеке отмены.


## Комментарии


Количество действий, хранящихся в стеке отмены, ограничивается свойством
 [IUndoRedo.Limit](IUndoRedo.Limit.htm).
