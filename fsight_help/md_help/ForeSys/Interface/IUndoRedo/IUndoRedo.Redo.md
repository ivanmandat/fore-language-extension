# IUndoRedo.Redo


## Синтаксис


Redo([Count: Integer = 1]);


## Параметры


Count. Количество действий.


## Описание


Метод Redo повторяет заданное
 количество отмененных действий.


## Комментарии


Количество действий, доступных для повторения, можно получить в свойстве
 [IUndoRedo.RedoCount](IUndoRedo.RedoCount.htm).


## Пример


Пример использования приведен в описании свойства [IUndoRedo.Enabled](IUndoRedo.Enabled.htm).
