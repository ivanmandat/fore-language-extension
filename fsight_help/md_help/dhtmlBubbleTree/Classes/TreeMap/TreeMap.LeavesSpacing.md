# TreeMap.LeavesSpacing


## Синтаксис


LeavesSpacing: Number


## Описание


Свойство LeavesSpacing** определяет размер отступов между элементами плоского дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**LeavesSpacing**, а возвращается с помощью метода get**LeavesSpacing**.
