# TreeMap.MaxFont


## Синтаксис


MaxFont: Number


## Описание


Свойство MaxFont** определяет максимальный размер шрифта текста элементов плоского дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**MaxFont**, а возвращается с помощью метода get**MaxFont**.
