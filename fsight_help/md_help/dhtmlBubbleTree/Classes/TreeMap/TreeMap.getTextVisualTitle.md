# TreeMap.getTextVisualTitle


## Синтаксис


getTextVisualTitle(returnCached: Boolean);


## Параметры


*returnCached. Признак, определяющий, будет ли возвращено кешированное или новое значение. Если параметр имеет значение true, будет возвращено кешированное значение, иначе - новое.*


## Описание


Метод getTextVisualTitle** возвращает название атрибута показателя текста элементов плоского дерева.


## Комментарии


Метод возвращает значение типа String.
