# TreeMap.MinFont


## Синтаксис


MinFont: Number


## Описание


Свойство MinFont** определяет минимальный размер шрифта текста элементов плоского дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**MinFont**, а возвращается с помощью метода get**MinFont**.
