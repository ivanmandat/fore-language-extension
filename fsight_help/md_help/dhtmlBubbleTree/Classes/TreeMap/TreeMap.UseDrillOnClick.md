# TreeMap.UseDrillOnClick


## Синтаксис


UseDrillOnClick: Boolean;


## Описание


Свойство UseDrillOnClick определяет,
 используется ли детализация по щелчку мыши.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setUseDrillOnClick,
 а возвращается с помощью метода getUseDrillOnClick.


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [TreeMap](../../Components/TreeMap/TreeMap.htm) с наименованием
 «treeMap» (см. «[Пример
 создания компонента TreeMap](../../Components/TreeMap/TreeMap_example.htm)» ):


// Зададим использование детализации по щелчку мыши
treeMap.setUseDrillOnClick(true);
В результате выполнения примера был включен режим детализации по щелчку мыши.
