# TreeMap.BranchesOverflow


## Синтаксис


BranchesOverflow: String;


## Описание


Свойство BranchesOverflow**
 определяет способ сокращения текста в заголовках веток плоского дерева,
 когда размер текста превосходит размеры заголовка.


## Комментарии


Принимает значения из перечисления [PP.Ui.TextWrapping](dhtmlUi.chm::/Enums/TextWrapping.htm).


Значение свойства устанавливается из JSON и с помощью метода setBranches**Overflow**,
 а возвращается с помощью метода getBranches**Overflow**.
