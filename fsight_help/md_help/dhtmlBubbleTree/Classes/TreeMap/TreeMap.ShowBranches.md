# TreeMap.ShowBranches


## Синтаксис


ShowBranches: Boolean;


## Описание


Свойство ShowBranches определяет
 признак отображения ветвей.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setShowBranches
 а возвращается с помощью метода getShowBranches.


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [TreeMap](../../Components/TreeMap/TreeMap.htm) с наименованием
 «treeMap» (см. «[Пример
 создания компонента TreeMap](../../Components/TreeMap/TreeMap_example.htm)» ):


// Зададим отображение ветвей
treeMap.setShowBranches(true);
В результате выполнения примера был включен режим отображения ветвей.
