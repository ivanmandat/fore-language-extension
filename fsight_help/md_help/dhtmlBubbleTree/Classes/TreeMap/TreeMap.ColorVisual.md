# TreeMap.ColorVisual


## Синтаксис


ColorVisual: PP.ColorVisual


## Описание


Свойство ColorVisual** определяет описание показателя, данные из которого применяются для заливки элементов плоского дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**ColorVisual**, а возвращается с помощью метода get**ColorVisual**.
