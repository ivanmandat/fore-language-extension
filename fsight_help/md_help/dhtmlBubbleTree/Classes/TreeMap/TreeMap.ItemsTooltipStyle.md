# TreeMap.ItemsTooltipStyle


## Синтаксис


ItemsTooltipStyle: Object


## Описание


Свойство ItemsTooltipStyle**
 определяет стиль всплывающей подсказки элементов плоского дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**ItemsTooltipStyle****, который в качестве параметра принимает
 JSON-объект с настройками стиля**, а возвращается с помощью метода
 get**ItemsTooltipStyle**.


Свойство содержит JSON-объект с полем Release,
 являющимся объектом типа [PP.Style](dhtmlCommon.chm::/Classes/PP/Style/Style.htm).
