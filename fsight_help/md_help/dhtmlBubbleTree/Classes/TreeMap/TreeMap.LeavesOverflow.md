# TreeMap.LeavesOverflow


## Синтаксис


LeavesOverflow: String


## Описание


Свойство LeavesOverflow** определяет способ отображения текста элемента плоского дерева, когда размер текста превосходит размеры элемента.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**LeavesOverflow**, а возвращается с помощью метода get**LeavesOverflow**.
