# TreeMap.getItemOnCoords


## Синтаксис


getItemOnCoords(x: Number, y: Number);


## Параметры


x. Асбцисса точки;


y. Ордината точки.


## Описание


Метод getItemOnCoords возвращает
 элемент плоского дерева по входящей в него координате.


## Комментарии


Метод возвращает объект типа [PP.Ui.TreeMapItem](../TreeMapItem/TreeMapItem.htm).
