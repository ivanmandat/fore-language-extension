# TreeMap.LeafBorderWidth


## Синтаксис


LeafBorderWidth: Number


## Описание


Свойство LeafBorderWidth** определяет ширину границы элементов плоского дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**LeafBorderWidth**, а возвращается с помощью метода get**LeafBorderWidth**.
