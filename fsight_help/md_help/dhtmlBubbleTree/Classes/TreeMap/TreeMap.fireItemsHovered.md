# TreeMap.fireItemsHovered


## Синтаксис


fireItemsHovered(item: [PP.Ui.TreeMapItem](../TreeMapItem/TreeMapItem.htm));


## Параметры


item. Элемент плоского дерева,
 для которого требуется сгенерировать событие наведения.


## Описание


Метод fireItemsHovered генерирует
 [событие наведения](TreeMap.ItemsHovered.htm) на указанный
 элемент плоского дерева.
