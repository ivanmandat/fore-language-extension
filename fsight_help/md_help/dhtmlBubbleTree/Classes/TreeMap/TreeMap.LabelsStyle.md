# TreeMap.LabelsStyle


## Синтаксис


LabelsStyle: [PP.Style](dhtmlCommon.chm::/Classes/PP/Style/Style.htm)


## Описание


Свойство LabelsStyle**
 определяет стиль подписей элементов плоского дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**LabelsStyle**,
 а возвращается с помощью метода get**LabelsStyle**.
