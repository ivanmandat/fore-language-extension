# TreeMap.VerticalContent


## Синтаксис


VerticalContent: Boolean


## Описание


Свойство VerticalContent** определяет, будет ли текст элемента плоского дерева располагаться вертикально, если высота элемента больше его ширины.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**VerticalContent**, а возвращается с помощью метода get**VerticalContent**.


Если свойство имеет значение true, текст будет располагаться вертикально, иначе - не будет.


По умолчанию свойство имеет значение false.
