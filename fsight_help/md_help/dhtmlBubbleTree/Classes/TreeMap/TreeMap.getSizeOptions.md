# TreeMap.getSizeOptions


## Синтаксис


getSizeOptions();


## Описание


Метод getSizeOptions** возвращает настройки для панели размеров в формате настроек мастера.


## Комментарии


Метод возвращает массив JSON-объектов со следующими полями: Name - имя свойства, Value - значение свойства.
