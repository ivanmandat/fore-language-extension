# TreeMap.Visualizer


## Синтаксис


Visualizer: [PP.Ui.TreeMapItem](../TreeMapItem/TreeMapItem.htm)


## Описание


Свойство Visualizer определяет** визуализатор элемента плоского дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**Visualizer**, а возвращается с помощью метода get**Visualizer****.**


## Пример


Для выполнения примера предполагается наличие на странице компонента [TreeMap](../../Components/TreeMap/TreeMap.htm) с наименованием «treeMap» (см. «[Пример создания компонента TreeMap](../../Components/TreeMap/TreeMap_example.htm)» ). Выведем тип визуализатора элемента диаграммы, цвет фона заголовка первого уровня, цвет шрифта заголовка первого уровня, размер шрифта заголовка первого уровня. Получим и выведем названия атрибутов показателя цветов элементов, показателя высоты элементов, показателя размеров элементов и показателя текста элементов:


// Получим тип визуализатора элемента диаграммы
var visualizerType = treeMap.getVisualizer();
// Создадим новый экземпляр визуализатора
var visualizer = new visualizerType();
// Выведем название класса визуализатора
console.log("Название класса визуализатора: " + visualizer.getTypeName());
// Получим и выведем цвет фона заголовка первого уровня
var backColor = treeMap.getBranchBackColor(1);
console.log("Цвет фона заголовка первого уровня: R:" + backColor.R + " G:" + backColor.G + " B:" + backColor.B);
// Получим и выведем цвет шрифта заголовка первого уровня
var fontColor = treeMap.getBranchFontColor(1);
console.log("Цвет шрифта заголовка первого уровня: R:" + fontColor.R + " G:" + fontColor.G + " B:" + fontColor.B);
// Получим и выведем размер шрифта заголовка первого уровня
var fontSize = treeMap.getBranchFontSize(1);
console.log("Размер шрифта заголовка первого уровня: " + fontSize);
// Получим и выведем название атрибута показателя цветов элементов
console.log("Название атрибута показателя цветов элементов: " + treeMap.getColorVisualTitle(false));
// Получим и выведем название атрибута показателя высоты элементов
console.log("Название атрибута показателя высоты элементов: " + treeMap.getHeightVisualTitle(false));
// Получим и выведем название атрибута показателя размеров элементов
console.log("Название атрибута показателя размеров элементов: " + treeMap.getSizeVisualTitle(false));
// Получим и выведем название атрибута показателя текста элементов
console.log("Название атрибута показателя текста элементов: " + treeMap.getTextVisualTitle(false));

В результате выполнения примера в консоли браузера были выведены следующие данные: тип визуализатора элемента диаграммы, цвет фона заголовка первого уровня, цвет шрифта заголовка первого уровня, размер шрифта заголовка первого уровня, а также названия атрибутов показателя цветов элементов, показателя высоты элементов, показателя размеров элементов и показателя текста элементов:

Цвет фона заголовка первого уровня: R:223 G:225 B:240


Цвет шрифта заголовка первого уровня: R:0 G:9 B:16


Размер шрифта заголовка первого уровня: 12


Название атрибута показателя цветов элементов: Value


Название атрибута показателя высоты элементов: Value


Название атрибута показателя размеров элементов: Value


Название атрибута показателя текста элементов: Value
