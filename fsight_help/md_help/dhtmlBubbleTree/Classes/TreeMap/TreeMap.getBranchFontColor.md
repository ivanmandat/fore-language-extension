# TreeMap.getBranchFontColor


## Синтаксис


getBranchFontColor(level: Number);


## Параметры


*level. Уровень группы элементов диаграммы.*


## Описание


Метод getBranchFontColor** возвращает цвет шрифта заголовка группы элементов плоского дерева по уровню.


## Комментарии


Метод возвращает JSON-объект со следующими полями: R - красная составляющая цвета, G - зеленая составляющая цвета, B - синяя составляющая цвета.
