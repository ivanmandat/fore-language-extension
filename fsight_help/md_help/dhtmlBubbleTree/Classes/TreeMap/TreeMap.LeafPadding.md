# TreeMap.LeafPadding


## Синтаксис


LeafPadding: Number


## Описание


Свойство LeafPadding** определяет размер отступа текста элементов плоского дерева от их границ.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**LeafPadding**, а возвращается с помощью метода get**LeafPadding**.
