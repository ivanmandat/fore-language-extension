# TreeMap.getSceneItems


## Синтаксис


getSceneItems();


## Описание


Метод getSceneItems** возвращает массив элементов, отображённых на плоском дереве в данный момент.


## Комментарии


Метод возвращает массив объектов типа [PP.Ui.TreeMapItem](../TreeMapItem/TreeMapItem.htm).
