# TreeMap.StartHeaderStyle


## Синтаксис


StartHeaderStyle: [PP.Style](dhtmlCommon.chm::/Classes/PP/Style/Style.htm)


## Описание


Свойство StartHeaderStyle**
 определяет стартовый стиль заголовков элементов плоского дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**StartHeaderStyle**,
 а возвращается с помощью метода get**StartHeaderStyle**.
