# TreeMap.EndHeaderStyle


## Синтаксис


EndHeaderStyle: [PP.Style](dhtmlCommon.chm::/Classes/PP/Style/Style.htm)


## Описание


Свойство EndHeaderStyle**
 определяет конечный стиль заголовков элементов плоского дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**EndHeaderStyle**, а возвращается
 с помощью метода get**EndHeaderStyle**.
