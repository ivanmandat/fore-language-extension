# TreeMap.getLegendOptions


## Синтаксис


getLegendOptions();


## Описание


Метод getLegendOptions** возвращает настройки для панели легенды в формате настроек мастера.


## Комментарии


Метод возвращает массив JSON-объектов со следующими полями: Name - имя свойства, Value - значение свойства.
