# TreeMap.LabelAutoColor


## Синтаксис


LabelAutoColor: Boolean


## Описание


Свойство LabelAutoColor** определяет, будет ли цвет текста элементов плоского дерева рассчитываться автоматически или устанавливаться из настроек.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**LabelAutoColor**, а возвращается с помощью метода get**LabelAutoColor**.


Если свойство имеет значение true, цвет текста будет рассчитываться автоматически, иначе он будет браться из настроек.


По умолчанию свойство имеет значение true.
