# TreeMap.SizeVisual


## Синтаксис


SizeVisual: PP.SizeVisual


## Описание


Свойство SizeVisual определяет о**писание показателя, данные из которого применяются для расчета размера элементов плоского дерева.


## Комментарии


Значение свойства устанавливается с помощью метода set**SizeVisual**, а возвращается с помощью метода get**SizeVisual**. Из JSON значение свойства установить нельзя.
