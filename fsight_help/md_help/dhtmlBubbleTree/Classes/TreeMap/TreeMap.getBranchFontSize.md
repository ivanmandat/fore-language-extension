# TreeMap.getBranchFontSize


## Синтаксис


getBranchFontSize(level: Number);


## Параметры


*level. Уровень группы элементов диаграммы.*


## Описание


Метод getBranchFontSize** возвращает размер шрифта заголовка группы элементов плоского дерева по уровню.


## Комментарии


Метод возвращает значение типа Number.
