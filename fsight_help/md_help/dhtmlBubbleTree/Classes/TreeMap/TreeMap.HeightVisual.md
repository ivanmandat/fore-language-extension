# TreeMap.HeightVisual


## Синтаксис


HeightVisual: PP.HeightVisual


## Описание


Свойство HeightVisual** определяет описание показателя, данные из которого применяются для установки высоты элементов плоского дерева в режиме теплокарты.


## Комментарии


Значение свойства устанавливается с помощью метода set**HeightVisual**, а возвращается с помощью метода get**HeightVisual**. Из JSON значение свойства установить нельзя.
