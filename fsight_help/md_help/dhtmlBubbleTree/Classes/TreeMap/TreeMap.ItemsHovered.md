# TreeMap.ItemsHovered


## Синтаксис


ItemsHovered: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.
 Доступный аргумент: Items - JSON-объект,
 поля которого соответствуют идентификаторам элементов плоского дерева.


## Описание


Событие ItemsHovered наступает
 при наведении указателя на элемент плоского дерева.
