# TreeMap.HierarchyDepth


## Синтаксис


HierarchyDepth: Number


## Описание


Свойство HierarchyDepth** определяет отображаемый уровень иерархии элементов плоского дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**HierarchyDepth**, а возвращается с помощью метода get**HierarchyDepth**.
