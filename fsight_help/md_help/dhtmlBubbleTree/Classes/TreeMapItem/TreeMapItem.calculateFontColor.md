# TreeMapItem.calculateFontColor


## Синтаксис


calculateFontColor(color: Object);


## Параметры


*color. Значение цвета.* Объект содержит следующие поля: R - красный компонент цвета, G - зеленый компонент цвета, B - синий компонент цвета.


## Описание


Метод calculateFontColor** рассчитывает значение цвета шрифта текста на основе указанного цвета.


## Комментарии


Метод возвращает значение типа String.
