# TreeMapItem.Level


## Синтаксис


Level: Number


## Описание


Свойство Level** определяет уровень элемента плоского дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**Level**, а возвращается с помощью метода get**Level**.


## Пример


Для выполнения примера предполагается наличие на странице компонента [TreeMap](../../Components/TreeMap/TreeMap.htm) с наименованием «treeMap» (см. «[Пример создания компонента TreeMap](../../Components/TreeMap/TreeMap_example.htm)» ). Получим и выведем значение уровня элемента, значение настройки размера элемента и значение настройки текста элемента:


// Получим элемент диаграммы с индексом 3
var item = treeMap.getSceneItems()[3];
// Отобразим уровень элемента
console.log("Уровень элемента: " + item.getLevel())
// Отобразим значение настройки размера элемента
console.log("Значение настройки размера элемента: " + item.getSize());
// Отобразим значение настройки текста элемента
console.log("Значение настройки текста элемента:" + item.getText());

В результате выполнения примера в консоли браузера были выведены значение уровня элемента, значение настройки размера элемента и значение настройки текста элемента:


Уровень элемента: 3


Значение настройки размера элемента: 71.42857142857129


Значение настройки текста элемента:100
