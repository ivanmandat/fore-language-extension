# TreeMapItem.isCoordsInner


## Синтаксис


isCoordsInner(x: Number, y: Number);


## Параметры


x. Асбцисса точки;


y. Ордината точки.


## Описание


Метод isCoordsInner определяет,
 принадлежит ли точка с указанной координатой заданному элементу плоского
 дерева.


## Комментарии


Метод возвращает следующие значения:


	- true. Точка с указанной
	 координатой принадлежит заданному элементу плоского дерева;


	- false. Точка с указанной
	 координатой не принадлежит данному элементу.
