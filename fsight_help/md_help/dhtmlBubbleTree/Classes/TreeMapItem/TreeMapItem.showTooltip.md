# TreeMapItem.showTooltip


## Синтаксис


showTooltip(coords: Object);


## Параметры


*coords. Координаты всплывающей подсказки.* Объект содержит следующие поля: X - координата X, Y - координата Y.


## Описание


Метод showTooltip** отображает всплывающую подсказку элемента плоского дерева.
