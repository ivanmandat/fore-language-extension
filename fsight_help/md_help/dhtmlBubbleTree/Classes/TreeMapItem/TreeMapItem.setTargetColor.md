# TreeMapItem.setTargetColor


## Синтаксис


setTargetColor(color: [PP.SolidColorBrush](dhtmlCommon.chm::/Classes/PP/SolidColorBrush/SolidColorBrush.htm)
 || String, inactiveOpacity: Number);


## Параметры


color. Цвет, устанавливаемый
 для заданного элемента плоского дерева;


inactiveOpacity. Прозрачность
 неактивного элемента плоского дерева. Принимает значения от 0 до 1.


## Описание


Метод setTargetColor устанавливает
 цвет заливки и прозрачность для заданного элемента плоского дерева.
