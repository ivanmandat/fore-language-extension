# TreeMapItem.build


## Синтаксис


build(position: Object);


## Параметры


*position. Значения размера и позиции элемента.* Объект содержит следующие поля: Top - значение отступа сверху, Left - значение отступа слева, Width - значение ширины, Height - значение высоты.


## Описание


Метод build** перестраивает элемент плоского дерева на основе полученных значений размера и позиции.
