# TreeMapItem.calculateBranchColor


## Синтаксис


calculateBranchColor(color1: String, color2: String,
 level: Number);


## Параметры


*color1. **Значение
 цвета.* Задается в формате "#RRGGBB".


*color2. Значение
 второго цвета.* Задается в формате "#RRGGBB".


*level.
 Уровень заголовка. Определяет пропорции использования цветов. При значении
 6 используется только второй цвет, при значении 0 используется только
 первый цвет.*


## Описание


Метод calculateBranchColor**
 рассчитывает значение цвета заголовка ветки на основе указанных цветов.


## Комментарии


Метод возвращает значение типа String.


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [TreeMap](../../Components/TreeMap/TreeMap.htm) с наименованием
 «treeMap» (см. «[Пример
 создания компонента TreeMap](../../Components/TreeMap/TreeMap_example.htm)» ). Рассчитаем новый цвет границы элемента,
 цвет фона элемента, цвет шрифта текста элемента:


// Получим элемент диаграммы с индексом 3
var item = treeMap.getSceneItems()[3];
// Создадим новые цвета
var color1 = "#252525";
var color2 = "#898900";
// Рассчитаем и выведем цвет заголовка ветки
BrColor = item.calculateBranchColor(color1, color2, 1)
console.log("Рассчитанный цвет заголовка ветки 1 уровня: " + BrColor)

В результате выполнения примера в консоли браузера были выведены новые
 рассчитанные значения цвета заголовка ветки:


Рассчитанный цвет заголовка ветки 1 уровня: "#36361f"
