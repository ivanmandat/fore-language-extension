# TreeMapItem.Size


## Синтаксис


Size: Number


## Описание


Свойство Size** определяет значение настройки размера элемента плоского дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**Size**, а возвращается с помощью метода get**Size**.
