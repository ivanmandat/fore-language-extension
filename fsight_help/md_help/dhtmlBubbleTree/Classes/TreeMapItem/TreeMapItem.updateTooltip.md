# TreeMapItem.updateTooltip


## Синтаксис


updateTooltip(coords: Object, isUpdateSettings: Boolean);


## Параметры


*coords. Координаты всплывающей подсказки.* Объект содержит следующие поля: X - координата X, Y - координата Y;


isUpdateSettings. Флаг, определяющий, будут ли обновляться настройки всплывающей подсказки. Если параметр имеет значение true, настройки будут обновляться, иначе - не будут.


## Описание


Метод updateTooltip** обновляет объект всплывающей подсказки элемента плоского дерева.
