# TreeMapItem.calculateBorderColor


## Синтаксис


calculateBorderColor(color: Object);


## Параметры


*color. Значение цвета.* Объект содержит следующие поля: R - красный компонент цвета, G - зеленый компонент цвета, B - синий компонент цвета.


## Описание


Метод calculateBorderColor** рассчитывает значение цвета границы на основе указанного цвета.


## Комментарии


Метод возвращает значение типа String.


## Пример


Для выполнения примера предполагается наличие на странице компонента [TreeMap](../../Components/TreeMap/TreeMap.htm) с наименованием «treeMap» (см. «[Пример создания компонента TreeMap](../../Components/TreeMap/TreeMap_example.htm)» ). Рассчитаем новый цвет границы элемента, цвет фона элемента, цвет шрифта текста элемента:


// Получим элемент диаграммы с индексом 3
var item = treeMap.getSceneItems()[3];
// Создадим новый объект цвета
var color = {R: 204, G: 80, B:91};
// Рассчитаем и выведем цвет границы элемента
var borderColor = item.calculateBorderColor(color);
console.log("Рассчитанный цвет границы элемента: " + borderColor)
// Рассчитаем и выведем фоновый цвет элемента
var color = item.calculateColor(color);
console.log("Рассчитанный фоновый цвет элемента: " + color)
// Рассчитаем и выведем цвет шрифта текста элемента
var fontColor = item.calculateFontColor(color);
console.log("Рассчитанный цвет шрифта текста элемента: " + fontColor)

В результате выполнения примера в консоли браузера были выведены новые рассчитанные значения цветов границы элемента, фона и шрифта текста элемента:


Рассчитанный цвет границы элемента: #fe828d


Рассчитанный фоновый цвет элемента: #cc505b


Рассчитанный цвет шрифта текста элемента: #FFFFFF
