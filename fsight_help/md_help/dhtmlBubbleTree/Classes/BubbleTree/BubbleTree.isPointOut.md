# BubbleTree.isPointOut


## Синтаксис


isPointOut(x: Number, y: Number);


## Параметры


*x.* Координата X;


*y.* Координата Y.


## Описание


Метод isPointOut** возвращает признак нахождения точки с указанными координатами вне границ пузырькового дерева.


## Комментарии


Метод возвращает значение типа Boolean.
