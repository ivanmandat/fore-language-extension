# BubbleTree.TextVisual


## Синтаксис


TextVisual: PP.TextVisual


## Описание


Свойство TextVisual** **определяет о**писание показателя, данные из которого применятся к подписям элементов пузырькового дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**TextVisual**, а возвращается с помощью метода get**TextVisual**.
