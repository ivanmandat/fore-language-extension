# BubbleTree.OuterLabelsStyle


## Синтаксис


OuterLabelsStyle: Object


## Описание


Свойство OuterLabelsStyle** определяет стиль значений элементов пузырькового дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**OuterLabelsStyle**, а возвращается с помощью метода get**OuterLabelsStyle****.**
