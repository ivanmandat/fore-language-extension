# BubbleTree.getCenter


## Синтаксис


getCenter();


## Описание


Метод getCenter** возвращает координаты центра пузырькового дерева.


## Комментарии


Метод возвращает JSON-объект со следующими полями: X - координата по оси X, Y - координата по оси Y.
