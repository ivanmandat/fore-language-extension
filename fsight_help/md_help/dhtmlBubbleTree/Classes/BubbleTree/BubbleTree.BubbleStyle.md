# BubbleTree.BubbleStyle


## Синтаксис


BubbleStyle: [PP.Ui.VisualizerItemStyle](dhtmlBubbleChart.chm::/Enums/VisualizerItemStyle.htm)


## Описание


Свойство BubbleStyle**
 определяет стиль пузырька пузырькового дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**BubbleStyle**,
 а возвращается с помощью метода get**BubbleStyle****.**
