# BubbleTree.ItemsHovered


## Синтаксис


ItemsHovered: function (sender, args: Object, timeout: Number);


## Параметры


sender. Источник события;


args. Информация о событии.
 Доступный аргумент: Items - JSON-объект, поле которого соответствует идентификатору
 элемента под указателем;


timeout. Промежуток времени
 в миллисекундах, через который вызовется событие.


## Описание


Событие ItemsHovered наступает при наведении указателя на элемент
 пузырькового дерева.
