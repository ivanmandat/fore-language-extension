# BubbleTree.SizeVisual


## Синтаксис


SizeVisual: PP.SizeVisual


## Описание


Свойство SizeVisual** **определяет о**писание показателя, данные из которого применяются для расчета размера элементов пузырькового дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**SizeVisual**, а возвращается с помощью метода get**SizeVisual**.
