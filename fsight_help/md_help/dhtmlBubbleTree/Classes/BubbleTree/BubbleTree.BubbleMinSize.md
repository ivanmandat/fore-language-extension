# BubbleTree.BubbleMinSize


## Синтаксис


BubbleMinSize: Number


## Описание


Свойство BubbleMinSize** определяет минимальный размер пузырька у пузырькового дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**BubbleMinSize**, а возвращается с помощью метода get**BubbleMinSize****.**
