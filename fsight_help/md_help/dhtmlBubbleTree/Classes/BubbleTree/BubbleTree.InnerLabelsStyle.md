# BubbleTree.InnerLabelsStyle


## Синтаксис


InnerLabelsStyle: Object


## Описание


Свойство InnerLabelsStyle** определяет стиль подписей элементов пузырькового дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**InnerLabelsStyle**, а возвращается с помощью метода get**InnerLabelsStyle****.**
