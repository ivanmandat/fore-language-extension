# BubbleTree.selectItems


## Синтаксис


selectItems(items: Object, reset: Boolean);


## Параметры


item. Элементы пузырькового
 дерева для выделения;


reset. Признак того, будет
 ли сброшено выделение всех элементов перед выполнением метода.


## Описание


Метод selectItems выделяет указанные
 элементы пузырькового дерева.
