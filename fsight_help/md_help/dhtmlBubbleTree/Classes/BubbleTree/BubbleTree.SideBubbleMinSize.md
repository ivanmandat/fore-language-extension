# BubbleTree.SideBubbleMinSize


## Синтаксис


SideBubbleMinSize: Number


## Описание


Свойство SideBubbleMinSize** определяет минимальный размер соседних пузырьков у пузырькового дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**SideBubbleMinSize**, а возвращается с помощью метода get**SideBubbleMinSize****.**
