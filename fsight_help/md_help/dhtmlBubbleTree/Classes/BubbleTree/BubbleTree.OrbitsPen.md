# BubbleTree.OrbitsPen


## Синтаксис


OrbitsPen: PP.Pen


## Описание


Свойство OrbitsPen** определяет стиль орбит пузырькового дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**OrbitsPen**, а возвращается с помощью метода get**OrbitsPen****.**
