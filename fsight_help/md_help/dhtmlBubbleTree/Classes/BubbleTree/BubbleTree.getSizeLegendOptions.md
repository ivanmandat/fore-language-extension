# BubbleTree.getSizeLegendOptions


## Синтаксис


getSizeLegendOptions();


## Описание


Метод getSizeLegendOptions возвращает
 настройки для панели легенды размера в формате настроек мастера.


## Комментарии


Метод возвращает массив JSON-объектов со следующими полями: Name
 - имя свойства, Value - значение
 свойства.


## Пример


Использование метода аналогично методу [BubbleTree.getLegendOptions](BubbleTree.getLegendOptions.htm).
