# BubbleTree.fireItemsHovered


## Синтаксис


fireItemsHovered(item: Object);


## Параметры


item. Объект, для которого
 вызывается событие [ItemsHovered](BubbleTree.ItemsHovered.htm).
 Необязательный параметр.


## Описание


Метод fireItemsHovered вызывает
 событие [ItemsHovered](BubbleTree.ItemsHovered.htm) для указанного
 объекта.
