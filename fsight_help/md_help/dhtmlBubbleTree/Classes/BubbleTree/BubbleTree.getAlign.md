# BubbleTree.getAlign


## Синтаксис


getAlign();


## Описание


Метод getAlign** возвращает коэффициенты ориентации пузырькового дерева.


## Комментарии


Метод возвращает JSON-объект со следующими полями: X - коэффициент ориентации по оси X, Y - коэффициент ориентации по оси Y.
