# BubbleTree.getFreeArea


## Синтаксис


getFreeArea();


## Описание


Метод getFreeArea** возвращает свободную область пузырькового дерева, в которой располагаются элементы.


## Комментарии


Метод возвращает объект типа PP.Rect.
