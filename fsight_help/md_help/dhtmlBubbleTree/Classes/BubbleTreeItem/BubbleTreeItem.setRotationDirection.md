# BubbleTreeItem.setRotationDirection


## Синтаксис


setRotationDirection(clockwise: Boolean);


## Параметры


*clockwise.* Признак, определяющий тип направления вращения. Если параметр имеет значение true, вращение будет происходить по часовой стрелке, иначе - против часовой. Если параметр не определен, то вращение будет происходить против часовой стрелки.


## Описание


Метод setRotationDirection** задает направление вращения элемента пузырькового дерева.
