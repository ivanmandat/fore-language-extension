# BubbleTreeItem.Size


## Синтаксис


Size: Number


## Описание


Свойство Size** определяет размер элемента пузырькового дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**Size**, а возвращается с помощью метода get**Size**.
