# BubbleTreeItem.Values


## Синтаксис


Values: Object


## Описание


Свойство Values** определяет
 значения настроек элемента пузырькового дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**Values**,
 а возвращается с помощью метода get**Values**.
