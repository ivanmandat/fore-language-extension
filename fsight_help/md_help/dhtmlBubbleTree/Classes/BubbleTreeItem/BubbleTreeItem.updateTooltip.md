# BubbleTreeItem.updateTooltip


## Синтаксис


updateTooltip(x: Number, y: Number);


## Параметры


*x.
 Координата X элемента подсказки*;


y. *Координата
 Y элемента подсказки.*


## Описание


Метод updateTooltip** обновляет
 объект всплывающей подсказки элемента пузырькового дерева.
