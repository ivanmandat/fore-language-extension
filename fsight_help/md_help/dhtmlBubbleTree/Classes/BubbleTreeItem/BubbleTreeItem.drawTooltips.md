# BubbleTreeItem.drawTooltips


## Синтаксис


drawTooltips(check: Boolean);


## Параметры


check.
 Определяет перекрытие элемента.


## Описание


Метод drawTooltips отрисовывает
 всплывающую подсказку элемента пузырькового дерева.
