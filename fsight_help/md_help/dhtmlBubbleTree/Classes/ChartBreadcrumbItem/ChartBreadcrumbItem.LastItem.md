# ChartBreadcrumbItem.LastItem


## Синтаксис


LastItem: Boolean


## Описание


Свойство LastItem** определяет, является ли текущий элемент цепочки навигации её последним элементом.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**LastItem**, а возвращается с помощью метода get**LastItem**.


Если свойство имеет значение true, элемент является последним, иначе - не последним.
