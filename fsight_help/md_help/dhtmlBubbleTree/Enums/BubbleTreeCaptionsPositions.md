# BubbleTreeCaptionsPositions


Пространство имен: PP.Ui


## Описание


Перечисление BubbleTreeCaptionsPositions содержит варианты расположения подписи для пузырька у [пузырькового дерева](../Components/BubbleTree/BubbleTree.htm).


Используется свойством BubbleTree.CaptionsPosition.


## Допустимые значения


Значение


Краткое описание


Inside


Всегда внутри пузырька.


Above


Всегда над пузырьком.


Below


Всегда под пузырьком.


Auto


Автоматически подбирает наилучшее расположение подписи для пузырька.
