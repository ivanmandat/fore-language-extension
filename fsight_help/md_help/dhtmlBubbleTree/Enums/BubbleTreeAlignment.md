# BubbleTreeAlignment


Пространство имен: PP.Ui


## Описание


Перечисление BubbleTreeAlignment
 содержит варианты выравнивая для [пузырькового
 дерева](../Components/BubbleTree/BubbleTree.htm).


Используется свойством [BubbleTree.Alignment](../Classes/BubbleTree/BubbleTree.Alignment.htm).
