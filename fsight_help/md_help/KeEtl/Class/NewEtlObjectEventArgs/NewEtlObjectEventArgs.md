# NewEtlObjectEventArgs


Сборка: Etl;


## Описание


Класс NewEtlObjectEventArgs
 реализует аргумент события, наступающего при создании нового объекта задачи
 ETL из панелей компонента [EtlObjectPanelBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/EtlObjectPanelBox.htm).
