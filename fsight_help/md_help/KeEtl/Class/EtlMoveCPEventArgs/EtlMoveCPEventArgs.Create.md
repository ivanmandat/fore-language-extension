# EtlMoveCPEventArgs.Create


## Синтаксис


Create(Shape: [IWxShape](Andy.chm::/Interface/IWxShape/IWxShape.htm);
 CPIndex: Integer; Cancel: Boolean);


## Параметры


Shape. Линия, у которой перемещается
 точка соединения;


CPIndex. Индекс точки соединения;


Cancel. Признак возможности
 перемещения.


## Описание


Конструктор Create создаёт аргумент
 события, связанного с перемещением точки соединения.
