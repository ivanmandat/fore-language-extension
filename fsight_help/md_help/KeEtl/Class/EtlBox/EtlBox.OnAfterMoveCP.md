# EtlBox.OnAfterMoveCP


## Синтаксис


OnAfterMoveCP(Sender: Object; Args: [IEtlAttachCPEventArgs](../../Interface/IEtlAttachCPEventArgs/IEtlAttachCPEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnAfterMoveCP наступает
 после перемещения точки соединения линии.
