# EtlObjectPanelBox.OnNewEtlObject


## Синтаксис


OnNewEtlObject(Sender: Object; Args: [INewEtlObjectEventArgs](../../Interface/INewEtlObjectEventArgs/INewEtlObjectEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnNewEtlObject наступает
 при вызове мастера создания нового коннектора/преобразователя.


## Комментарии


Мастер вызывается при щелчке по коннектору/преобразователю на одной
 из панелей компонента.
