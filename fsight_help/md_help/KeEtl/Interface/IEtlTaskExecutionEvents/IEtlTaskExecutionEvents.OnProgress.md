# IEtlTaskExecutionEvents.OnProgress


## Синтаксис


OnProgress(Task: [IEtlTask](../IEtlTask/IEtlTask.htm);
 Progress: Integer);


## Параметры


Task. Задача ETL;


Progress. Прогресс выполнения
 задачи в процентах.


## Описание


Метод OnProgress реализует событие,
 наступающее в процессе выполнения задачи ETL.
