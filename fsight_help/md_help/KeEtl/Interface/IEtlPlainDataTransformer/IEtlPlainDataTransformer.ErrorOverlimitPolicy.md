# IEtlPlainDataTransformer.ErrorOverlimitPolicy


## Синтаксис


ErrorOverlimitPolicy: [EtlErrorOverlimitPolicy](../../Enums/EtlErrorOverlimitPolicy.htm);


## Описание


Свойство ErrorOverlimitPolicy
 определяет действие, выполняемое при превышении максимального количества
 ошибочных записей.


## Комментарии


Количество ошибочных записей задается в свойстве [IEtlPlainDataTransformer.ErrorStopLimit](IEtlPlainDataTransformer.ErrorStopLimit.htm).


## Пример


Пример использования приведен в описании свойства [IEtlPlainDataTransformer.ErrorOutput](IEtlPlainDataTransformer.ErrorOutput.htm).
