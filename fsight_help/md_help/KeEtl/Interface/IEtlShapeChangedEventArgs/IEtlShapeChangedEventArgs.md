# IEtlShapeChangedEventArgs


Сборка: Etl;


## Описание


Интерфейс IEtlShapeChangedEventArgs
 содержит свойства аргумента события, наступающего при изменении фигуры
 компонента [EditBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/EtlBox.htm).


## Иерархия наследования


           [IEventArgs](ForeSys.chm::/Interface/IEventArgs/IEventArgs.htm)


           IEtlShapeChangedEventArgs


## Комментарии


Свойства данного интерфейса используются в событии [EtlBox.OnShapeChanged](../../Class/EtlBox/EtlBox.OnShapeChanged.htm).
