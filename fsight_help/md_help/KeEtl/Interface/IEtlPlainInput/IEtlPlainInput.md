# IEtlPlainInput


## Описание


Интерфейс IEtlPlainInput содержит
 свойства и методы связи и преобразования полей между входящим и текущим
 объектами задачи ETL.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Description](IEtlPlainInput.Description.htm)
		 Свойство Description
		 определяет описание (примечание) входа текущего объекта.


		 ![](../../Property_Image.gif)
		 [Fields](IEtlPlainInput.Fields.htm)
		 Свойство Fields возвращает
		 объект, содержащий список полей входа текущего объекта.


		 ![](../../Property_Image.gif)
		 [Id](IEtlPlainInput.Id.htm)
		 Свойство Id определяет
		 идентификатор входа текущего объекта.


		 ![](../../Property_Image.gif)
		 [Inputs](IEtlPlainInput.Inputs.htm)
		 Свойство Inputs возвращает
		 объект, содержащий родительский список входов.


		 ![](../../Property_Image.gif)
		 [Link](IEtlPlainInput.Link.htm)
		 Свойство Link возвращает
		 ссылку на объект-связку между объектами ETL.


		 ![](../../Property_Image.gif)
		 [Name](IEtlPlainInput.Name.htm)
		 Свойство Name определяет
		 наименование входа текущего объекта.
