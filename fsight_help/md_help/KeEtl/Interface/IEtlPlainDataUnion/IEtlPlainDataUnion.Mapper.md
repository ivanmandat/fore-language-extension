# IEtlPlainDataUnion.Mapper


## Синтаксис


Mapper(InputNumber: Integer): [IEtlPlainFieldsMapper](../IEtlPlainFieldsMapper/IEtlPlainFieldsMapper.htm);


## Параметры


InputNumber.
 Индекс выхода объекта.


## Описание


Свойство Mapper возвращает объект,
 содержащий связь полей между приёмником и источником. Индекс входа передаётся
 посредством параметра InputNumber.
