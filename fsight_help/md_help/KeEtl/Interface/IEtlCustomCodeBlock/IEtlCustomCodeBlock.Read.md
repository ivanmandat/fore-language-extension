# IEtlCustomCodeBlock.Read


## Синтаксис


Read(Reader: [IXmlDomElement](ModXml.chm::/develop.htm));


## Параметры


Reader.
 Данные в формате XML.


## Описание


Метод Read производит чтение
 данных в формате XML.


## Комментарии


В параметре Reader метода будут
 доступны те данные, которые были определены в свойстве [IEtlFixedCodeBlock.UserData](../IEtlFixedCodeBlock/IEtlFixedCodeBlock.UserData.htm)
 при создании пользовательской процедуры или данные, сформированные в свойстве
 [IEtlUserDataContainer.UserData](../IEtlUserDataContainer/IEtlUserDataContainer.UserData.htm)
 в переопределённых интерфейсах [IEtlForeWizardPage](../IEtlForeWizardPage/IEtlForeWizardPage.htm),
 [IEtlForeWizardPages](../IEtlForeWizardPages/IEtlForeWizardPages.htm).
