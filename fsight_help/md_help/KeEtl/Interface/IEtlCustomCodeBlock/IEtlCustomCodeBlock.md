# IEtlCustomCodeBlock


Сборка: Etl;


## Описание


Интерфейс IEtlCustomCodeBlock
 предоставляет методы, которые реализуют шаблон пользовательской процедуры.


## Иерархия наследования


IEtlCustomCodeBlock


## Комментарии


Методы данного интерфейса должны быть переопределены в пользовательском
 классе. Класс задаётся в свойстве [IEtlTemplate.Class](KeFore.chm::/Interface/IEtlTemplate/IEtlTemplate.Class.htm)
 шаблона пользовательской процедуры.
