# IEtlPlainDataSplit.Mapper


## Синтаксис


Mapper(N: Integer): [IEtlPlainFieldsMapper](../IEtlPlainFieldsMapper/IEtlPlainFieldsMapper.htm);


## Параметры


N. Индекс
 выхода.


## Описание


Свойство Mapper возвращает объект,
 содержащий связь полей между приёмником и источником. Индекс выхода передается
 посредством параметра N.
