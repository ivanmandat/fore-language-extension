# IEtlPlainDataBulkConsumer.StopLimit


## Синтаксис


StopLimit: Integer;


## Описание


Свойство StopLimit определяет
 предельное количество ошибочных записей, при превышении которого выполнение
 задачи прекращается.


## Комментарии


Для определения, необходимо ли прекращение выполнения задачи при превышении
 предельного количества ошибочных записей, используйте свойство [IEtlPlainDataBulkConsumer.UseStopLimit](IEtlPlainDataBulkConsumer.UseStopLimit.htm).


## Пример


Использование свойства приведено в примере для свойства [IEtlPlainDataBulkConsumer.EventId](IEtlPlainDataBulkConsumer.EventId.htm).
