# IEtlPlainDataBulkConsumer.ActionOnProblem


## Синтаксис


ActionOnProblem: [EtlActionOnProblem](../../Enums/EtlActionOnProblem.htm);


## Описание


Свойство ActionOnProblem определяет
 способ обработки ошибок, которые могут возникнуть при загрузке данных.


## Комментарии


Для определения имени файла для ошибочных (пропущенных) записей используйте
 свойство [IEtlPlainDataBulkConsumer.InvalidRecsFileName](IEtlPlainDataBulkConsumer.InvalidRecsFileName.htm).
