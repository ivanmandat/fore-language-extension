# IEtlFixedCodeBlock.UserData


## Синтаксис


UserData: [IXmlDomElement](ModXml.chm::/develop.htm);


## Описание


Свойство UserData определяет
 пользовательские данные в формате XML.


## Комментарии


Пользовательские данные передаются в процедуру и будут доступны в методе
 [IEtlCustomCodeBlock.Read](../IEtlCustomCodeBlock/IEtlCustomCodeBlock.Read.htm)
 во время выполнения процедуры.
