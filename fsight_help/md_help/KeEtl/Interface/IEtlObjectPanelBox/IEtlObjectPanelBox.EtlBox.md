# IEtlObjectPanelBox.EtlBox


## Синтаксис


EtlBox: [IEtlBox](../IEtlBox/IEtlBox.htm);


## Описание


Свойство EtlBox определяет компонент,
 с которым будет взаимодействовать текущий компонент [EtlObjectPanelBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/EtlObjectPanelBox.htm).


## Комментарии


В качестве значения свойства необходимо указать компонент [EtlBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/EtlBox.htm),
 который имеется на форме.
