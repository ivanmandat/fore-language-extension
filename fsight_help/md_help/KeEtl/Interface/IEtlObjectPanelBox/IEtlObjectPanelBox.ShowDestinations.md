# IEtlObjectPanelBox.ShowDestinations


## Синтаксис


ShowDestinations: Boolean;


## Описание


Свойство ShowDestinations определяет,
 будет ли в компоненте отображаться вкладка «Приёмники».


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Вкладка «Приёмники» отображается.


	- False. Вкладка «Приёмники»
	 скрыта.
