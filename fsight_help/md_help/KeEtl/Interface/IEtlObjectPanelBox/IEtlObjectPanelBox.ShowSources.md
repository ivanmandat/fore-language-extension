# IEtlObjectPanelBox.ShowSources


## Синтаксис


ShowSources: Boolean;


## Описание


Свойство ShowSources определяет,
 будет ли в компоненте отображаться вкладка «Источники».


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Вкладка «Источники» отображается.


	- False. Вкладка «Источники»
	 скрыта.
