# IEtlObjectPanelBox.ShowOthers


## Синтаксис


ShowOthers: Boolean;


## Описание


Свойство ShowOthers определяет,
 будет ли в компоненте отображаться вкладка «Другие».


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Вкладка «Другие» отображается.


	- False. Вкладка «Другие»
	 скрыта.
