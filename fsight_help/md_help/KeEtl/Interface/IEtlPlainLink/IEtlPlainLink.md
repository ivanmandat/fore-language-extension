# IEtlPlainLink


## Описание


Интерфейс IEtlPlainLink содержит
 свойства и методы связи с полями выхода одного и входа другого объекта.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DestinationObjectInput](IEtlPlainLink.DestinationObjectInput.htm)
		 Свойство DestinationObjectInput
		 определяет вход объекта, в который поступает поток данных.


		 ![](../../Property_Image.gif)
		 [Link](IEtlPlainLink.Link.htm)
		 Свойство Link возвращает
		 привязку полей приёмника к полям источника.


		 ![](../../Property_Image.gif)
		 [SourceObjectOutput](IEtlPlainLink.SourceObjectOutput.htm)
		 Свойство SourceObjectOutput
		 определяет выход объекта, из которого поступает поток данных.
