# IEtlPlainDataProvider.UnpivotProvider


## Синтаксис


UnpivotProvider: IDtUnpivotProvider;


## Описание


Свойство UnpivotProvider возвращает
 параметры кросс-таблицы, в соответствии с которыми будет производиться
 идентификация данных.


## Комментарии


Свойство актуально, если свойству [IEtlPlainDataProvider.UnpivotMode](IEtlPlainDataProvider.UnpivotMode.htm)
 установлено значение True.
