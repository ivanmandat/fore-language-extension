# IEtlPlainRecordSets


## Описание


Интерфейс IEtlPlainRecordSets
 содержит свойства списка наборов плоских данных. Используется в алгоритмах
 пользователя для преобразования данных в задаче ETL.


## Свойства


		 Имя свойства

		 [Count](IEtlPlainRecordSets.Count.htm)

		 количество источников плоских данных.


		 ![](../../Property_Image.gif)
		 [Item](IEtlPlainRecordSets.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий источник плоских данных.
