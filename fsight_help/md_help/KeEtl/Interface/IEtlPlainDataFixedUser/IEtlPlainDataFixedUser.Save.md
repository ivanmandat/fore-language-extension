# IEtlPlainDataFixedUser.Save


## Синтаксис


Save;


## Описание


Метод Save сохраняет произведенные
 изменения с алгоритмом пользователя на основе шаблона.


## Комментарии


Для редактирования алгоритма пользователя на основе шаблона используйте
 [IEtlPlainDataFixedUser.Edit](IEtlPlainDataFixedUser.Edit.htm).
