# IEtlPlainDataFixedUser


Сборка: Etl;


## Описание


Интерфейс IEtlPlainDataFixedUser
 используется для работы с алгоритмом пользователя на основе шаблона.


## Иерархия наследования


           [IEtlObject](../IEtlObject/IEtlObject.htm)


           IEtlPlainDataFixedUser


## Комментарии


Для создания алгоритма пользователя на основе шаблона используйте метод
 [IEtlTask.Create](../IEtlTask/IEtlTask.Create.htm).
