# IEtlCodeBlock.Procedure


## Синтаксис


Procedure: String;


## Описание


Свойство Procedure определяет
 наименование выполняемой процедуры/функции.


## Комментарии


Для использования доступны процедуры/функции без параметров или с параметрами,
 которые принимают значения, имеющие простой тип данных.
