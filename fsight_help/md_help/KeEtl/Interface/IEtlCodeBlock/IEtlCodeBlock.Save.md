# IEtlCodeBlock.Save


## Синтаксис


Save;


## Описание


Метод Save сохраняет изменения
 объекта.


## Комментарии


Метод сохраняет все изменения, которые были произведены после вызова
 метода [Edit](IEtlCodeBlock.Edit.htm).
