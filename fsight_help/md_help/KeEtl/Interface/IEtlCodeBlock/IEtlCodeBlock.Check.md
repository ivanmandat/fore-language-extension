# IEtlCodeBlock.Check


## Синтаксис


Check;


## Описание


Метод Check проверяет, заданы
 ли для текущего объекта необходимые параметры.


## Комментарии


При выполнении метода осуществляется проверка наличия заданных значений
 в свойстве [Module](IEtlCodeBlock.Module.htm) и [Procedure](IEtlCodeBlock.Procedure.htm).
 Если какое-либо свойство не определено, то будет сгенерирована исключительная
 ситуация.
