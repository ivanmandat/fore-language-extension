# IEtlPlainLinkFieldMapping


## Описание


Интерфейс IEtlPlainLinkFieldMapping
 содержит свойства связи полей источника данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Database](IEtlPlainLinkFieldMapping.Database.htm)
		 Свойство Database определяет
		 БД, если тип связи между полями приёмника и источника - Sequence.


		 ![](../../Property_Image.gif)
		 [Field](IEtlPlainLinkFieldMapping.Field.htm)
		 Свойство Field определяет
		 поле источника данных, если тип связи между полями приёмника и
		 источника - Field.


		 ![](../../Property_Image.gif)
		 [Increment](IEtlPlainLinkFieldMapping.Increment.htm)
		 Свойство Increment
		 определяет шаг приращения при автоинкрементном заполнении полей.


		 ![](../../Property_Image.gif)
		 [Sequence](IEtlPlainLinkFieldMapping.Sequence.htm)
		 Свойство Sequence определяет
		 идентификатор сиквенса, если тип связи между полями приёмника
		 и источника - Sequence.


		 ![](../../Property_Image.gif)
		 [StartValue](IEtlPlainLinkFieldMapping.StartValue.htm)
		 Свойство StartValue
		 определяет начальное значение при автоинкрементном заполнении
		 полей.


		 ![](../../Property_Image.gif)
		 [Type](IEtlPlainLinkFieldMapping.Type.htm)
		 Свойство Type определяет
		 тип связи поля источника данных задачи ETL.


		 ![](../../Property_Image.gif)
		 [Value](IEtlPlainLinkFieldMapping.Value.htm)
		 Свойство Value определяет
		 значение константы, если тип связи между полями приёмника и источника
		 - Constant.
