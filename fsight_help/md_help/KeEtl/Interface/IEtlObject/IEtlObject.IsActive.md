# IEtlObject.IsActive


## Синтаксис


IsActive: Boolean;


## Описание


Свойство IsActive возвращает
 признак включения объекта в задачу ETL.


## Комментарии


Допустимые значения:


	- True. Объект включен
	 в задачу ETL;


	- False. Объект не включен
	 в задачу ETL.
