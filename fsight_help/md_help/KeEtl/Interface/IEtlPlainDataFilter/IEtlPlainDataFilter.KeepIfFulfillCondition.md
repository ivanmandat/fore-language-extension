# IEtlPlainDataFilter.KeepIfFulfillCondition


## Синтаксис


KeepIfFulfillCondition: Boolean;


## Описание


Свойство KeepIfFulfillCondition
 определяет условие, при котором записи будут отфильтрованы.


## Комментарии


Допустимые значения:


	- True. Записи, не удовлетворяющие
	 условию, указанному в [FilterIf](IEtlPlainDataFilter.FilterIf.htm),
	 будут отфильтрованы;


	- False. Записи, удовлетворяющие
	 условию, указанному в [FilterIf](IEtlPlainDataFilter.FilterIf.htm),
	 будут отфильтрованы.


По умолчанию свойству установлено значение True.
