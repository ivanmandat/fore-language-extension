# IEtlPlainOutputs.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index.
 Индекс выхода объекта задачи ETL.


## Описание


Метод Remove осуществляет удаление
 выхода объекта задачи ETL, индекс которого передается посредством параметра
 Index.
