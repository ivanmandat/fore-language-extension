# IEtlPlainOutputs


## Описание


Интерфейс IEtlPlainOutputs содержит
 свойства и методы коллекции исходящих источников и преобразований полей.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IEtlPlainOutputs.Count.htm)
		 Свойство Count возвращает
		 количество выходов объекта задачи ETL.


		 ![](../../Property_Image.gif)
		 [EtlObject](IEtlPlainOutputs.ETLObject.htm)
		 Свойство EtlObject
		 возвращает объект ETL.


		 ![](../../Property_Image.gif)
		 [Item](IEtlPlainOutputs.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий в себе связь между исходящим и текущим объектом.


		 ![](../../Property_Image.gif)
		 [MaxObjectCount](IEtlPlainOutputs.MaxObjectCount.htm)
		 Свойство MaxObjectCount
		 возвращает максимальное количество выходов.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IEtlPlainOutputs.Add.htm)
		 Метод Add добавляет
		 выход объекта задачи ETL.


		 ![](../../Sub_Image.gif)
		 [Clear](IEtlPlainOutputs.Clear.htm)
		 Метод Clear осуществляет
		 очистку списка выходов объекта задачи ETL.


		 ![](../../Sub_Image.gif)
		 [Edit](IEtlPlainOutputs.Edit.htm)
		 Метод Edit создает
		 копию объекта для редактирования.


		 ![](../../Sub_Image.gif)
		 [Remove](IEtlPlainOutputs.Remove.htm)
		 Метод Remove осуществляет
		 удаление выхода объекта задачи ETL.


		 ![](../../Sub_Image.gif)
		 [Save](IEtlPlainOutputs.Save.htm)
		 Метод Save сохраняет
		 результаты редактирования.
