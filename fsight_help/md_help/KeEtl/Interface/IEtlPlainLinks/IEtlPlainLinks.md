# IEtlPlainLinks


## Описание


Интерфейс IEtlPlainLinks содержит
 свойства коллекции связей объектов задачи ETL.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IEtlPlainLinks.Count.htm)
		 Свойство Count возвращает
		 количество связей объектов задачи ETL.


		 ![](../../Property_Image.gif)
		 [Item](IEtlPlainLinks.Item.htm)
		 Свойство Item возвращает
		 объект-связку, индекс которого передается в качестве параметра
		 Index.
