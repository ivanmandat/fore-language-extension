# IEtlPlainIndex


## Описание


Интерфейс IEtlPlainIndex содержит
 свойства и методы списка полей для указания индекса для источника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Description](IEtlPlainIndex.Description.htm)
		 Свойство Description
		 определяет описание (примечание) индекса источников данных.


		 ![](../../Property_Image.gif)
		 [FieldCount](IEtlPlainIndex.FieldCount.htm)
		 Свойство FieldCount
		 возвращает количество полей индекса.


		 ![](../../Property_Image.gif)
		 [Name](IEtlPlainIndex.Name.htm)
		 Свойство Name определяет
		 наименование индекса источников данных.


		 ![](../../Property_Image.gif)
		 [PlainInput](IEtlPlainIndex.PlainInput.htm)
		 Свойство PlainInput
		 возвращает объект, содержащий источник данных.


		 ![](../../Property_Image.gif)
		 [PlainInputCount](IEtlPlainIndex.PlainInputCount.htm)
		 Свойство PlainInputCount
		 возвращает количество источников данных для полей индекса.


		 ![](../../Property_Image.gif)
		 [PlainInputField](IEtlPlainIndex.PlainInputField.htm)
		 Свойство PlainInputField
		 определяет поле соответствующего источника данных.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddField](IEtlPlainIndex.AddField.htm)
		 Метод AddField добавляет
		 поле индекса.


		 ![](../../Sub_Image.gif)
		 [ClearFields](IEtlPlainIndex.ClearFields.htm)
		 Метод ClearFields удаляет
		 все поля индекса.


		 ![](../../Sub_Image.gif)
		 [Edit](IEtlPlainIndex.Edit.htm)
		 Метод Edit создает
		 копию объекта для редактирования.


		 ![](../../Sub_Image.gif)
		 [RemoveField](IEtlPlainIndex.RemoveField.htm)
		 Метод RemoveField удаляет
		 поле индекса.


		 ![](../../Sub_Image.gif)
		 [Save](IEtlPlainIndex.Save.htm)
		 Метод Save сохраняет
		 результаты редактирования.
