# IEtlPlainField


## Описание


Интерфейс IEtlPlainField содержит
 свойства и методы поля плоского источника/приёмника данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DataType](IEtlPlainField.DataType.htm)
		 Свойство DataType определяет
		 тип поля плоского источника/приёмника данных.


		 ![](../../Property_Image.gif)
		 [Description](IEtlPlainField.Description.htm)
		 Свойство Description
		 определяет описание (примечание) поля плоского источника/приёмника
		 данных.


		 ![](../../Property_Image.gif)
		 [Fields](IEtlPlainField.Fields.htm)
		 Свойство Fields возвращает
		 объект, содержащий родительский список полей.


		 ![](../../Property_Image.gif)
		 [Id](IEtlPlainField.Id.htm)
		 Свойство Id определяет
		 идентификатор поля плоского источника/приёмника данных.


		 ![](../../Property_Image.gif)
		 [Key](IEtlPlainField.Key.htm)
		 Свойство Key возвращает
		 ключ поля плоского источника/приёмника данных.


		 ![](../../Property_Image.gif)
		 [Name](IEtlPlainField.Name.htm)
		 Свойство Name определяет
		 наименование поля плоского источника/приёмника данных.


		 ![](../../Property_Image.gif)
		 [Precision](IEtlPlainField.Precision.htm)
		 Свойство Precision
		 определяет точность поля источника/приёмника.


		 ![](../../Property_Image.gif)
		 [Size](IEtlPlainField.Size.htm)
		 Свойство Size определяет
		 размер поля источника/приёмника.
