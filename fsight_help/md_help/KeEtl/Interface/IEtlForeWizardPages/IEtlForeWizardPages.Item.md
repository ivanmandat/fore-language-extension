# IEtlForeWizardPages.Item


## Синтаксис


Item(Index: Integer): [IMetabaseObjectDescriptor](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm);


## Параметры


Index. Индекс формы репозитория,
 которая будет использоваться в качестве страницы мастера.


## Описание


Метод Item используется для
 подключения указанной формы репозитория в качестве страницы мастера источника/приёмника
 пользователя.


## Комментарии


Для получения репозитория, в котором находится редактируемый источник/приёмник
 пользователя, используйте [IEtlForeWizardPages.Metabase](IEtlForeWizardPages.Metabase.htm).


Для получения количества страниц мастера источника/приёмника пользователя,
 используйте [IEtlForeWizardPages.Count](IEtlForeWizardPages.Count.htm).
