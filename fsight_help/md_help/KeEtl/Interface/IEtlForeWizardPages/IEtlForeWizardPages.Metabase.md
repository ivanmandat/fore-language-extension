# IEtlForeWizardPages.Metabase


## Синтаксис


Metabase: [IMetabase](KeSom.chm::/Interface/IMetabase/IMetabase.htm);


## Описание


Свойство Metabase возвращает
 репозиторий, в котором производится работа.


## Комментарии


Для получения количества страниц мастера источника/приёмника пользователя,
 используйте [IEtlForeWizardPages.Count](IEtlForeWizardPages.Count.htm).


Для подключения указанной формы репозитория в качестве страницы мастера
 источника/приёмника пользователя используйте [IEtlForeWizardPages.Item](IEtlForeWizardPages.Item.htm).
