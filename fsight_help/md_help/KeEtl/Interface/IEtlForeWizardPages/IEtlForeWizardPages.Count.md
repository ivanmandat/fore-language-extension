# IEtlForeWizardPages.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 новых страниц мастера.


## Комментарии


Для подключения указанной формы репозитория в качестве страницы мастера
 источника/приёмника пользователя используйте [IEtlForeWizardPages.Item](IEtlForeWizardPages.Item.htm).


Для получения репозитория, в котором находится редактируемый источник/приёмник
 пользователя, используйте свойство [IEtlForeWizardPages.Metabase](IEtlForeWizardPages.Metabase.htm).
