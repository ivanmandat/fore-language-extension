# IEtlForeWizardPages.Model


## Синтаксис


Model: [IEtlUserDataContainer](../IEtlUserDataContainer/IEtlUserDataContainer.htm);


## Описание


Свойство Model определяет модель
 метаданных для работы со страницами мастера.


## Комментарии


Свойство доступно только на запись. Свойство может использоваться для
 определения настроек, связанных со всеми окнами мастера. Настройки задаются
 в свойстве [IEtlUserDataContainer.UserData](../IEtlUserDataContainer/IEtlUserDataContainer.UserData.htm).
