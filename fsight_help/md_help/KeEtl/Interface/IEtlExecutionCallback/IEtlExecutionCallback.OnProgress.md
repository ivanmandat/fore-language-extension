# IEtlExecutionCallback.OnProgress


## Синтаксис


OnProgress(Progress: Integer);


## Параметры


Progress. Процент выполнения задачи Etl.


## Описание


Метод OnProgress реализует событие общего статуса выполнения задачи ETL.
