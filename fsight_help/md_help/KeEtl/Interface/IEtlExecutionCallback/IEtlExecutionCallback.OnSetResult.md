# IEtlExecutionCallback.OnSetResult


## Синтаксис


OnSetResult(nTotalRec: Integer, nErrorRec: Integer);


## Параметры


nTotalRec. Количество обработанных записей.


nErrorRec. Количество пропущенных по ошибке записей.


## Описание


Метод OnSetResult реализует событие для установки результата выполнения задачи ETL.
