# IEtlPlainInputs


## Описание


Интерфейс IEtlPlainInputs содержит
 свойства и методы коллекции входящих источников и преобразований полей.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IEtlPlainInputs.Count.htm)
		 Свойство Count возвращает
		 количество входов объекта задачи ETL.


		 ![](../../Property_Image.gif)
		 [EtlObject](IEtlPlainInputs.ETLObject.htm)
		 Свойство EtlObject
		 возвращает объект ETL.


		 ![](../../Property_Image.gif)
		 [Item](IEtlPlainInputs.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий в себе связь между входящим и текущим объектом.


		 ![](../../Property_Image.gif)
		 [MaxObjectCount](IEtlPlainInputs.MaxObjectCount.htm)
		 Свойство MaxObjectCount
		 возвращает максимальное количество входов.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IEtlPlainInputs.Add.htm)
		 Метод Add добавляет
		 вход объекта задачи ETL.


		 ![](../../Sub_Image.gif)
		 [Clear](IEtlPlainInputs.Clear.htm)
		 Метод Clear осуществляет
		 очистку списка входов объекта задачи ETL.


		 ![](../../Sub_Image.gif)
		 [Edit](IEtlPlainInputs.Edit.htm)
		 Метод Edit создает
		 копию объекта и открывает ее для редактирования.


		 ![](../../Sub_Image.gif)
		 [Remove](IEtlPlainInputs.Remove.htm)
		 Метод Remove осуществляет
		 удаление входа объекта задачи ETL.


		 ![](../../Sub_Image.gif)
		 [Save](IEtlPlainInputs.Save.htm)
		 Метод Save сохраняет
		 результаты редактирования.
