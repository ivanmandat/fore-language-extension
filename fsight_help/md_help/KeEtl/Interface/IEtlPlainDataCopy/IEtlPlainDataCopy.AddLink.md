# IEtlPlainDataCopy.AddLink


## Синтаксис


AddLink(ProviderField: String; ConsumerField: String);


## Параметры


ProviderField - идентификатор
 поля источника данных.


ConsumerField - идентификатор
 поля приёмника данных.


## Описание


Метод AddLink добавляет связь
 между указанными полями источника и приёмника данных.


## Комментарии


Для удаления связей между указанными полями источника и приёмника данных
 используйте метод [IEtlPlainDataCopy.RemoveLink](IEtlPlainDataCopy.RemoveLink.htm).
