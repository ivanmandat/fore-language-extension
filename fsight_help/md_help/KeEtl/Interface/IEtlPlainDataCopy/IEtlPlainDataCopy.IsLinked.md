# IEtlPlainDataCopy.IsLinked


## Синтаксис


IsLinked(ProviderField: String; ConsumerField: String):
 Boolean;


## Параметры


ProviderField - идентификатор
 поля источника данных.


ConsumerField - идентификатор
 поля приёмника данных.


## Описание


Метод IsLinked возвращает признак
 того, установлена ли связь между указанными полями источника и приёмника
 данных.


## Комментарии


Допустимые значения:


	- True. Связь установлена;


	- False. Связь отсутствует.
