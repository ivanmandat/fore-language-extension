# IEtlPlainDataCopy.ClearConsumer


## Синтаксис


ClearConsumer: Boolean;


## Описание


Свойство ClearConsumer определяет,
 очищать ли приёмник данных перед копированием.


## Комментарии


Допустимые значения:


	- True. Приёмник данных
	 будет очищен перед копированием;


	- False. Приёмник данных
	 не будет очищен перед копированием.
