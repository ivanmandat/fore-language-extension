# IEtlPlainDataCopy.Edit


## Синтаксис


Edit: [IEtlPlainDataCopy](IEtlPlainDataCopy.htm);


## Описание


Метод Edit создает копию объекта
 для редактирования.


## Комментарии


Для сохранения объекта после редактирования используйте метод [IEtlPlainDataCopy.Save](IEtlPlainDataCopy.Save.htm).
