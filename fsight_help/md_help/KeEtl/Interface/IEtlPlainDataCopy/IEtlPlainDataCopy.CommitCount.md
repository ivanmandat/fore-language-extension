# IEtlPlainDataCopy.CommitCount


## Синтаксис


CommitCount: Integer;


## Описание


Свойство CommitCount определяет
 количество записей, обрабатываемых в рамках одной транзакции.


## Комментарии


По умолчанию в рамках одной транзакции обрабатывается 1000 записей.
