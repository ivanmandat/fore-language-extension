# IEtlForeWizardPage.IsLeaveAllowed


## Синтаксис


IsLeaveAllowed: Boolean;


## Описание


Метод IsLeaveAllowed определяет,
 будет ли активна/не активна кнопка «Далее»
 страницы мастера.


## Комментарии


Допустимые значения:


	- True. Кнопка «Далее» страницы мастера будет активна;


	- False. Кнопка «Далее» страницы мастера будет не
	 активна.


Для определения, будет ли активна/не активна кнопка «Готово»
 страницы мастера источника/приёмника пользователя, используйте [IEtlForeWizardPage.IsDoneAllowed](IEtlForeWizardPage.IsDoneAllowed.htm).
