# IEtlForeWizardPage.Title


## Синтаксис


Title: String;


## Описание


Свойство Title определяет наименование
 новой страницы мастера.


## Комментарии


Для определения модели метаданных для работы со страницей мастера используйте
 свойство [IEtlForeWizardPage.Model](IEtlForeWizardPage.Model.htm).
