# IEtlForeWizardPage.OnSetActive


## Синтаксис


OnSetActive: Boolean;


## Описание


Метод OnSetActive определяет,
 будет ли активна страница мастера.


## Комментарии


Может использоваться для настройки страницы и для запрета перехода на
 нее.


Допустимые значения:


	- True. Страница мастера
	 будет активна, будет возможна ее настройка;


	- False. Страница мастера
	 будет не активна.
