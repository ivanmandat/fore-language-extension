# IEtlForeWizardPage.IsDoneAllowed


## Синтаксис


IsDoneAllowed: Boolean;


## Описание


Метод IsDoneAllowed определяет,
 будет ли активна/не активна кнопка «Готово»
 страницы мастера.


## Комментарии


Допустимые значения:


	- True. Кнопка «Готово» страницы мастера будет активна;


	- False. Кнопка «Готово» страницы мастера будет не
	 активна.


Для определения, будет ли активна/не активна кнопка «Далее»
 страницы мастера источника/приёмника пользователя, используйте [IEtlForeWizardPage.IsLeaveAllowed](IEtlForeWizardPage.IsLeaveAllowed.htm).
