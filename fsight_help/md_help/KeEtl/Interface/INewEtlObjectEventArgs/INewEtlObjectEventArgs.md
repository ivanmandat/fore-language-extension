# INewEtlObjectEventArgs


Сборка: Etl;


## Описание


Интерфейс INewEtlObjectEventArgs
 содержит свойства аргумента события, наступающего при создании нового
 объекта задачи ETL из панелей компонента [EtlObjectPanelBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/EtlObjectPanelBox.htm).


## Иерархия наследования

## Комментарии


Свойства данного интерфейса используются в событии [EtlObjectPanelBox.OnNewEtlObject](../../Class/EtlObjectPanelBox/EtlObjectPanelBox.OnNewEtlObject.htm).
