# INewEtlObjectEventArgs.CmdID


## Синтаксис


CmdID: Integer;


## Описание


Свойство CmdID возвращает цифровой
 идентификатор окна мастера, который был вызван с панелей компонента для
 создания нового коннектора/преобразователя.
