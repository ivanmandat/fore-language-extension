# IEtlMoveCPEventArgs.Cancel


## Синтаксис


Cancel: Boolean;


## Описание


Свойство Cancel определяет признак
 запрета перемещения точки соединения линии.


## Комментарии


Допустимые значения:


	- True. Перемещение точки
	 запрещено;


	- False. По умолчанию.
	 Перемещение точки разрешено.
