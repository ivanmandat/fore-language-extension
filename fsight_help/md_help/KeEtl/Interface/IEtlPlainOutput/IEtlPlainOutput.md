# IEtlPlainOutput


## Описание


Интерфейс IEtlPlainOutput содержит
 свойства и методы связи и преобразования полей между исходящим и текущим
 объектами задачи ETL.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Description](IEtlPlainOutput.Description.htm)
		 Свойство Description
		 определяет описание (примечание) выхода текущего объекта.


		 ![](../../Property_Image.gif)
		 [Fields](IEtlPlainOutput.Fields.htm)
		 Свойство Fields возвращает
		 объект, содержащий список полей выхода текущего объекта.


		 ![](../../Property_Image.gif)
		 [Id](IEtlPlainOutput.Id.htm)
		 Свойство Id определяет
		 идентификатор выхода текущего объекта.


		 ![](../../Property_Image.gif)
		 [Link](IEtlPlainOutput.Link.htm)
		 Свойство Link возвращает
		 ссылку на объект-связку между объектами ETL.


		 ![](../../Property_Image.gif)
		 [Name](IEtlPlainOutput.Name.htm)
		 Свойство Name определяет
		 наименование выхода текущего объекта.


		 ![](../../Property_Image.gif)
		 [Outputs](IEtlPlainOutput.Outputs.htm)
		 Свойство Outputs возвращает
		 объект, содержащий родительский список выходов.


## Методы
