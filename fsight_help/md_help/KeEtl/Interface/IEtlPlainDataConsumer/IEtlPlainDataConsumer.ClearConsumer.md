# IEtlPlainDataConsumer.ClearConsumer


## Синтаксис


ClearConsumer: Boolean;


## Описание


Свойство ClearConsumer определяет, необходимо ли производить очистку приёмника перед добавлением записей. При значении True приёмник будет очищен, при значении False - нет. По умолчанию свойство имеет значение True.
