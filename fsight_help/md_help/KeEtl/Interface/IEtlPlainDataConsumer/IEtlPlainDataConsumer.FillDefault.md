# IEtlPlainDataConsumer.FillDefault


## Синтаксис


FillDefault;


## Описание


Метод FillDefault заполняет поля на основании данных о полях [приёмника](IEtlPlainDataConsumer.Consumer.htm). При возможности [размер](../IEtlPlainField/IEtlPlainField.Size.htm)/[точность](../IEtlPlainField/IEtlPlainField.Precision.htm) полей будет автоматически установлена.
