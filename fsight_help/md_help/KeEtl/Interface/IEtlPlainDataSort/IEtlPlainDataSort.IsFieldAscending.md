# IEtlPlainDataSort.IsFieldAscending


## Синтаксис


IsFieldAscending(Index: Integer): Boolean;


## Параметры


Index. Индекс сортируемого
 поля.


## Описание


Свойство IsFieldAscending определяет
 направление сортировки по указанному полю.
