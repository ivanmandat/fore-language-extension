# IEtlPlainDataForceQueryForFile


Сборка: Etl;


## Описание


Интерфейс IEtlPlainDataForceQueryForFile
 используется для работы с объектом, определяющим, будет ли запрашиваться
 файл для источника данных при выполнении задачи ETL.


## Иерархия наследования


IEtlPlainDataForceQueryForFile


## Комментарии


Данный интерфейс используется для работы с файловыми источниками данных
 задачи ETL. Следующие источники данных, описываемые интерфейсом [IEtlPlainDataProvider](../IEtlPlainDataProvider/IEtlPlainDataProvider.htm),
 могут быть приведены к IEtlPlainDataForceQueryForFile: Access, Excel,
 XML, Текст, JSON.
