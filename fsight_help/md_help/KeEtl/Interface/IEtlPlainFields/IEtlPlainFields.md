# IEtlPlainFields


## Описание


Интерфейс IEtlPlainFields содержит
 свойства и методы коллекции полей входов/выходов объекта задачи ETL.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IEtlPlainFields.Count.htm)
		 Свойство Count возвращает
		 количество полей входа/выхода объекта задачи ETL.


		 ![](../../Property_Image.gif)
		 [Item](IEtlPlainFields.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий поле входа/выхода объекта задачи ETL.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IEtlPlainFields.Add.htm)
		 Метод Add добавляет
		 новое поле входа/выхода объекта задачи ETL.


		 ![](../../Sub_Image.gif)
		 [Clear](IEtlPlainFields.Clear.htm)
		 Метод Clear осуществляет
		 очистку списка полей входа/выхода объекта задачи ETL.


		 ![](../../Sub_Image.gif)
		 [Edit](IEtlPlainFields.Edit.htm)
		 Метод Edit создает
		 копию объекта для редактирования.


		 ![](../../Sub_Image.gif)
		 [Fill](IEtlPlainFields.Fill.htm)
		 Метод Fill осуществляет
		 заполнение списка полей в соответствии с указанным списком.


		 ![](../../Sub_Image.gif)
		 [FindById](IEtlPlainFields.FindById.htm)
		 Метод FindById осуществляет
		 поиск по идентификатору.


		 ![](../../Sub_Image.gif)
		 [MoveDown](IEtlPlainFields.MoveDown.htm)
		 Метод MoveDown осуществляет
		 перемещение поля в списке на одну позицию вниз.


		 ![](../../Sub_Image.gif)
		 [MoveUp](IEtlPlainFields.MoveUp.htm)
		 Метод MoveUp осуществляет
		 перемещение поля в списке на одну позицию вверх.


		 ![](../../Sub_Image.gif)
		 [Remove](IEtlPlainFields.Remove.htm)
		 Метод Remove удаляет
		 поле из списка входа/выхода объекта задачи ETL.


		 ![](../../Sub_Image.gif)
		 [Save](IEtlPlainFields.Save.htm)
		 Метод Save сохраняет
		 результаты редактирования.
