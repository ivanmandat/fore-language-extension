# IEtlTask.Workspace


## Синтаксис


Workspace: [IWxWorkspace](Andy.chm::/Interface/IWxWorkspace/IWxWorkspace.htm);


## Описание


Свойство Workspace возвращает объект, содержащий в себе рабочее пространство задачи ETL.


## Комментарии


Для получения объекта, содержащего все связи между объектами задачи ETL, используйте свойство [IEtlTask.Links](IEtlTask.Links.htm).
