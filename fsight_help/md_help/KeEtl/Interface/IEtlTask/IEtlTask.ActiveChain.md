# IEtlTask.ActiveChain


## Синтаксис


ActiveChain(ConnCompId: Integer): Boolean;


## Параметры


ConnCompId.
 Индекс компоненты связности объекта задачи ETL.


## Описание


Свойство ActiveChain определяет
 состояние цепочки объектов.


## Комментарии


Допустимые значения:


	- True. Цепочка объектов
	 активна;


	- False. Цепочка объектов
	 не активна.
