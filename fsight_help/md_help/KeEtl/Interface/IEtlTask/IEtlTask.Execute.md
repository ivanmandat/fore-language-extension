# IEtlTask.Execute


## Синтаксис


Execute(Callback: [IEtlExecutionCallback](../IEtlExecutionCallback/IEtlExecutionCallback.htm));


## Параметры


Callback.
 Объект, уведомляющий о процессе выполнения задачи ETL.


## Описание


Метод Execute осуществляет выполнение
 задачи ETL.


## Комментарии


Для остановки выполнения задачи ETL используйте [IEtlTask.StopExecute](IEtlTask.StopExecute.htm).
