# IEtlTask.CustomData


## Синтаксис


CustomData: String;


## Описание


Свойство CustomData определяет
 представление задачи ETL в формате JSON.


## Комментарии


Свойство актуально для задач ETL, которые созданы в веб-приложении.
 Свойство хранит в себе настройки задачи ETL, информацию о коннекторах,
 преобразователях и других объектах, добавленных в рабочую область, в формате
 JSON.


Важно. Не
 рекомендуется изменять значение свойства, так как это может нарушить работу
 задачи ETL в веб-приложении.


## Пример


Для выполнения примера в репозитории предполагается наличие задачи ETL
 с идентификатором ETL_TASK. Задача создана в веб-приложении.


Добавьте ссылки на системные сборки: Etl, Metabase.


	Sub UserProc;

	Var

	    Mb: IMetabase;

	    Etl: IEtlTask;

	Begin

	    Mb := MetabaseClass.Active;

	    Etl := Mb.ItemById("ETL_TASK").Bind As IEtlTask;

	    Debug.WriteLine("CustomData: " + Etl.CustomData);

	End Sub UserProc;


При выполнении примера в консоль среды разработки будет выведено представление
 задачи ETL в JSON-формате.
