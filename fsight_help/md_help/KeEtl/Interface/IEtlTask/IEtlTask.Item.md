# IEtlTask.Item


## Синтаксис


Item(Index: Integer): [IEtlObject](../IEtlObject/IEtlObject.htm);


## Параметры


Index — индекс объекта задачи ETL.


## Описание


Свойство Item возвращает объект задачи ETL, индекс которого передается посредством параметра Index.


## Комментарии


Для получения количества объектов в задаче ETL используйте свойство [IEtlTask.Count](IEtlTask.Count.htm).
