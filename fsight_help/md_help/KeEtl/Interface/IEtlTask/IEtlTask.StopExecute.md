# IEtlTask.StopExecute


## Синтаксис


StopExecute;


## Описание


Метод StopExecute останавливает
 выполнение задачи ETL.


## Комментарии


Остановка выполнения задачи ETL будет полезна, например, при большом
 объеме копируемых данных из источника в приёмник, так как операции извлечения
 и вставки больших объемов данных могут расходовать значительное количество
 ресурсов, что может привести к торможению или зависанию операционной системы.


Для осуществления выполнения задачи ETL используйте [IEtlTask.Execute](IEtlTask.Execute.htm).
