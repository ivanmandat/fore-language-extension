# IEtlTask.CanDeactivateObject


## Синтаксис


CanDeactivateObject(Object: [IEtlObject](../IEtlObject/IEtlObject.htm)):
 Boolean;


## Параметры


Object. Объект расчета цепочки.


## Описание


Метод CanDeactivateObject используется
 для определения, исключен ли объект из расчета цепочки.


## Комментарии


Допустимые значения:


	- True. Объект включен
	 в цепочку операций;


	- False. Объект исключен
	 из цепочки операций.
