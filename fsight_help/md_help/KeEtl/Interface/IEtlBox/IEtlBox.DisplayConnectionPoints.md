# IEtlBox.DisplayConnectionPoints


## Синтаксис


DisplayConnectionPoints: Boolean;


## Описание


Свойство DisplayConnectionPoints
 определяет признак отображения точек соединения.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Точки соединения отображаются;


	- False. Точки соединения
	 не отображаются.
