# IEtlBox.DisplayFocusFrame


## Синтаксис


DisplayFocusFrame: Boolean;


## Описание


Свойство DisplayFocusFrame определяет
 признак отображения фокуса.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Фокус отображается.


	- False. Фокус не отображается.
