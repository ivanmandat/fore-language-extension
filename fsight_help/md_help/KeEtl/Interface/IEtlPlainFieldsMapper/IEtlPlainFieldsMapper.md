# IEtlPlainFieldsMapper


## Описание


Интерфейс IEtlPlainFieldsMapper
 содержит свойства и методы связи и преобразования полей между полями входящего
 и текущего объектов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Description](IEtlPlainFieldsMapper.Description.htm)
		 Свойство Description
		 определяет описание связи между полями.


		 ![](../../Property_Image.gif)
		 [Map](IEtlPlainFieldsMapper.Map.htm)
		 Свойство Map возвращает
		 объект, содержащий формулу связи поля выхода с полями входа.


		 ![](../../Property_Image.gif)
		 [Name](IEtlPlainFieldsMapper.Name.htm)
		 Свойство Name определяет
		 наименование связи между полями выхода и входа.
