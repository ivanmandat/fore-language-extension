# EtlInputOutputType


## Описание


Перечисление EtlInputOutputType
 содержит типы входа/выхода объекта задачи ETL.


Используется следующими свойствами и методами:


	- [IEtlObject.InputType](../Interface/IEtlObject/IEtlObject.InputType.htm)


	- [IEtlObject.OutputType](../Interface/IEtlObject/IEtlObject.OutputType.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. нет входа/выхода.


		 1
		 One. Один вход/выход.


		 2
		 Many. Много входов/выходов.
