# EtlAgregateFormula


## Описание


Перечисление EtlAgregateFormula
 содержит агрегационные формулы, используемые для негруппируемых полей
 и правило отбора при удалении дубликатов.


Используется следующими свойствами и методами:


	- [IEtlPlainDataDeduplicate.Rule](../Interface/IEtlPlainDataDeduplicate/IEtlPlainDataDeduplicate.Rule.htm)


	- [IEtlPlainDataGroup.Formula](../Interface/IEtlPlainDataGroup/IEtlPlainDataGroup.Formula.htm)
