# EtlErrorOverlimitPolicy


## Описание


Перечисление EtlErrorOverlimitPolicy
 содержит действия, выполняемые при превышении количества ошибочных записей
 во время преобразования данных.


Используется следующими свойствами и методами:


	- [IEtlPlainDataTransformer.ErrorOverlimitPolicy](../Interface/IEtlPlainDataTransformer/IEtlPlainDataTransformer.ErrorOverlimitPolicy.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Abort. Остановить расчет
		 задачи ETL.


		 1
		 StopOutput. Прекратить
		 вывод ошибочных записей.
