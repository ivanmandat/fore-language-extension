		 [EtlActionOnProblem](EtlActionOnProblem.htm)
		 Перечисление EtlActionOnProblem
		 содержит способы обработки ошибок, которые могут возникнуть при
		 работе с данными.


		 ![](../Enums_Image.gif)
		 [EtlAgregateFormula](EtlAgregateFormula.htm)
		 Перечисление EtlAgregateFormula
		 содержит агрегационные формулы, используемые для негруппируемых
		 полей и правило отбора при удалении дубликатов.


		 ![](../Enums_Image.gif)
		 [EtlErrorOverlimitPolicy](EtlErrorOverlimitPolicy.htm)
		 Перечисление EtlErrorOverlimitPolicy
		 содержит действия, выполняемые при превышении количества ошибочных
		 записей во время преобразования данных.


		 ![](../Enums_Image.gif)
		 [EtlInputOutputType](EtlInputOutputType.htm)
		 Перечисление EtlInputOutputType
		 содержит типы входа/выхода объекта задачи ETL.


		 ![](../Enums_Image.gif)
		 [EtlJoinType](EtlJoinType.htm)
		 Перечисление EtlJoinType
		 содержит типы соединения.


		 ![](../Enums_Image.gif)
		 [EtlObjectType](EtlObjectType.htm)
		 Перечисление EtlObjectType
		 содержит типы объекта задачи Etl.


		 ![](../Enums_Image.gif)
		 [EtlPlainLinkFieldMappingType](EtlPlainLinkFieldMappingType.htm)
		 Перечисление EtlPlainLinkFieldMappingType
		 содержит типы связи поля источника данных задачи ETL.
