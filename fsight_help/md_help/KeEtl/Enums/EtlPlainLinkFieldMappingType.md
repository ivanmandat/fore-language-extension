# EtlPlainLinkFieldMappingType


## Описание


Перечисление EtlPlainLinkFieldMappingType
 содержит типы связи поля источника данных задачи ETL.


Используется следующим свойством:


	- [IEtlPlainLinkFieldMapping.Type](../Interface/IEtlPlainLinkFieldMapping/IEtlPlainLinkFieldMapping.Type.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Field. Поле источника.


		 1
		 Constant. Константа.


		 2
		 CurrentDate. Текущая
		 дата.


		 3
		 Sequence. Идентификатор
		 сиквенса.


		 4
		 Increment. Автоинкремент.
