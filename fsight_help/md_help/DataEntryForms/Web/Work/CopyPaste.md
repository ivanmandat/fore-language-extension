# Работа с данными через буфер обмена


При работе с данными в табличной области через буфер обмена доступны
 операции:


	- [копирование](#copy);


	- [перемещение](#cut);


	- [вставка](#insert).


## Копирование данных в буфер обмена


Для копирования значений ячейки или выделенного диапазона ячеек в буфер
 обмена:


	- выполните команду «Копировать»
	 в контекстном меню ячейки или диапазона ячеек;


	- нажмите сочетание клавиш CTRL+C.


При использовании [автофильтра](FilterSearch.htm#filter)
 диапазона данных на листе формы ввода или табличной области копирование
 диапазона значений в буфер обмена применяется только к видимым строкам/столбцам.


## Перемещение данных в буфер обмена


Для перемещения значений ячейки или выделенного диапазона ячеек в буфер
 обмена:


	- выполните команду «Вырезать»
	 в контекстном меню ячейки или диапазона ячеек;


	- нажмите сочетание клавиш CTRL+X.


При использовании [автофильтра](FilterSearch.htm#filter)
 диапазона данных на листе формы ввода или табличной области перемещение
 диапазона значений в буфер обмена применяется ко всем строкам/столбцам
 независимо от их видимости при фильтрации. Поведение будет одинаковым
 для всего диапазона данных независимо от расположения фильтра.


## Вставка данных из буфера обмена


Для вставки значений ячейки или выделенного диапазона ячеек из буфера
 обмена в текущее место табличной области нажмите сочетание клавиш CTRL+V.


При использовании для вставки:


	- команды контекстного меню ![](b_paste_data.png)
	 «Вставить» ячейки;


	- кнопки  ![](b_paste_data.png) «Вставить»
	 вкладки «Главная» панели инструментов;


	- команды «Правка > ![](b_paste_data.png) Вставить» главного
	 меню.


Будет выведено информационное сообщение о необходимости использования
 сочетания клавиш.


### Особенности вставки данных из буфера обмена


	- Если вставляемый диапазон выходит за пределы табличной области,
	 то вставка осуществится в пределах табличной области.


	- Если для вставки диапазона ячеек выделен диапазон меньшего размера,
	 то размеры диапазона, куда вставляются данные, будут расширены вправо
	 и вниз до размера вставляемого диапазона.


	- Если для вставки диапазона ячеек выделен диапазон большего размера,
	 то будет заполнен только диапазон соответствующий размерам вставляемого
	 диапазона.


	- Если используется [автофильтр](FilterSearch.htm#filter)
	 диапазона данных на листе формы ввода или табличной области:


		- вставка значения из одной ячейки в диапазон ячеек применяется
		 только к видимым строкам/столбцам;


		- вставка диапазона значений применяется ко всем видимым строкам/столбцам.
		 Вставляемый диапазон значений расширяется с учётом размера области
		 копирования, в том числе на скрытые строки/столбцы.


Поведение будет одинаковым для всего диапазона
 данных независимо от расположения фильтра.


### Параметры форматирования при вставке
 из буфера обмена


Расширение «Интерактивные формы ввода
 данных» поддерживает вставку данных при копировании из продуктов:


	- Microsoft Office;


	- LibreOffice;


	- МойОфис;


	- Google Sheets.


Вставка содержимого из буфера на лист формы ввода выполняется:


	- с сохранением исходного форматирования, если форма открыта в
	 режиме редактирования;


	- без сохранения исходного форматирования, если форма открыта
	 в режиме просмотра.


Примечание.
 При вставке данных, скопированных из продуктов «МойОфис» и «Google Sheets»,
 исходное форматирование не сохраняется.
