# Настройка формы ввода под печать


Для настройки формы ввода под печать:


	- Выберите лист формы ввода. Параметры страницы настраиваются
	 при открытии формы ввода на [редактирование](../../Starting/Starting.htm#open)
	 для каждого листа отдельно, кроме параметров [заголовка
	 формы ввода](Heading_params.htm).


	- Выполните [быструю](#quick_setup) или [расширенную](#advanced_setup)
	 настройку параметров страницы.


После выполнения действий форма ввода будет готова для [печати](Tuning_ParamPage.htm#print).


Примечание.
 Перед печатью формы проверьте [настройки
 оформления формы](../../Common/Design.htm).


## Быстрая настройка


Для быстрой настройки параметров страницы активного листа формы ввода
 используйте:


	- кнопки на вкладке «Вид»
	 панели инструментов;


	- группу команд «Разметка страницы»
	 главного меню.


Доступны настройки:


	- ![](../b_page_borders.png) Показать
	 границы. Нажмите кнопку ![](../b_page_borders.png) «Показать границы» или выполните
	 одноимённую команду для отображения границ страниц, выводимых при
	 печати и/или экспорте, в виде пунктирной линии на активном листе формы
	 ввода. Границы страницы устанавливаются в зависимости от ориентации,
	 ширины полей, размеров страницы и разрывов;


	- ![](b_margins.png) Поля. Нажмите кнопку ![](b_margins.png) «Поля»
	 или выполните одноимённую команду для изменения размеров полей в раскрывающемся
	 меню:


		- Обычные. По умолчанию;


		- Широкие;


		- Узкие;


	- ![](b_orientation.png) Ориентация. Нажмите кнопку ![](b_orientation.png) «Ориентация» или выполните одноимённую
	 команду для изменения ориентации страницы в раскрывающемся меню:


		- Книжная. По умолчанию;


		- Альбомная;


	- ![](b_size.png) Размер. Нажмите кнопку ![](b_size.png) «Размер»
	 или выполните одноимённую команду для изменения размера страницы в
	 раскрывающемся меню. Список содержит предустановленные варианты размеров
	 бумаги, такие как A4, A5 и другие. По умолчанию установлен размер
	 A4;


	- ![](../b_breaks.png) Разрывы. Выделите ячейку, которая
	 будет являться левой верхней ячейкой новой границы страницы, и выполните
	 команду «Добавить разрыв»
	 в раскрывающемся меню кнопки ![](../b_breaks.png) «Разрывы»
	 для отображения определённых данных листа на одной странице. После
	 выполнения действия будут отображены разрывы страницы в виде сплошной
	 линии и автоматически включено отображение границ страниц.


Для получения подробной информации о настройке границ и разрывов страниц
 обратитесь к разделу «[Настройка разрывов
 и границ страниц](../Page_Borders_Breaks.htm)».

## Предварительный просмотр


Перед печатью выполните предварительный просмотр и убедитесь, что форма
 имеет требуемый вид.


Для предварительного просмотра:


	- выполните команду «Файл > Печать >
	 Предварительный просмотр» главного меню;


	- нажмите кнопку ![](../../Common/print.png) «Печать»
	 вкладки «Главная» панели инструментов;


	- нажмите сочетание клавиш CTRL+P.


Примечание.
 Сочетание клавиш CTRL+P, как правило, зарезервировано в браузере для печати
 текущей страницы. Чтобы использовать указанное сочетание клавиш для печати
 формы, переопределите его в настройках используемого браузера.


После выполнения одного из действий будет открыт предварительный просмотр
 формы ввода:


![](../../Common/report_print.png)


## Печать формы


Для печати формы в режиме предварительного просмотра:


	- Выберите нужную вкладку с наименованием листа для печати отдельного
	 листа отчёта. Для печати всего отчёта перейдите на вкладку «Отчёт».


	- Нажмите кнопку «Печать»
	 в главном меню. Будет открыт диалог для настройки параметров печати.
	 Порядок работы с диалогом зависит от используемого браузера.
