# Исключение данных из табличной области


При работе с табличной областью доступны операции исключения данных
 с помощью изменения отметки в измерениях по строкам и/или столбцам. Изменение
 отметки в измерениях доступно:


	- [в редакторе
	 табличной области](../Table/Select_dimension_elements.htm);


	- с помощью команд контекстного меню в боковике/шапке табличной
	 области на рабочей области:


		- для исключения строк/столбцов:


			- Выделите один или несколько элементов боковика/шапки.


			- Выполните команду «Исключить»
			 контекстного меню.


		- для исключения строк/столбцов, кроме выделенных:


			- Выделите один или несколько элементов боковика/шапки,
			 данные из которых должны отображаться в табличной области.


			- Выполните команду «Исключить
			 остальные» контекстного меню.


Команды также доступны для элементов одного
 или разных измерений боковика/шапки таблицы, если выполнено [объединение
 измерений](../Table/Merging_Dim.htm#merging) или настроено [поджатие](../Table/Merging_Dim.htm#tucking).
