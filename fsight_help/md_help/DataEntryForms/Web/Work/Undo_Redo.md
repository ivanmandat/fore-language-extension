# Отмена и повтор операции


Для отмены последней операции в форме ввода:


	- нажмите кнопку ![](../Custom_module/b_cancel.png) «Отменить» [главного
	 меню](../Starting/Starting.htm#elements);


	- нажмите сочетание клавиш CTRL+Z.


Если [измененные данные](DataEnter.htm) были сохранены, а
 затем сохранение было отменено, то будут возвращены предыдущие значения.
 Эти значения будут выделены как измененные, и для дальнейшего их использования
 требуется выполнить их сохранение.


Для повтора последней отмененной операции:


	- нажмите кнопку ![](../Custom_module/b_repeat.png) «Повторить» главного меню;


	- нажмите сочетание клавиш CTRL+Y.


Кнопки ![](../Custom_module/b_cancel.png) «Отменить»
 и ![](../Custom_module/b_repeat.png) «Повторить»
 доступны:


	- при [вводе](../Starting/Sheets_Data.htm#entry_editing) данных в ячейки;


	- при [форматировании](../Common/Design.htm) данных;


	- при [вставке](../Starting/Sheets_Data.htm#copying_pasting)/[удалении](../Starting/Sheets_Data.htm#delete) данных;


	- при [автозаполнении](UiNav.chm::/GUI/UiExpress_Tabl_Operation_AutoFillData.htm) ячеек данными;


	- при [добавлении](../Starting/Cell.htm#add_cells)/[удалении](../Starting/Cell.htm#delete_cells) ячеек;


	- при создании/редактировании [формул](../Starting/Sheets_Data.htm#formula);


	- при выполнении операций со [строками](../Starting/Row.htm)/[столбцами](../Starting/Column.htm), кроме операции [закрепления](../Common/Design.htm#freeze)
	 строки/столбца.


При [обновлении](../../Desktop/Common/Refresh.htm) формы
 ввода и расчёте [алгоритмов
 на сохранение](../../Desktop/Table/Calculation_Practices.htm#addalgorithm) очищается история операций и кнопки ![](../Custom_module/b_cancel.png) «Отменить» и ![](../Custom_module/b_repeat.png) «Повторить» [главного
 меню](../Starting/Starting.htm#elements) становятся недоступными.
