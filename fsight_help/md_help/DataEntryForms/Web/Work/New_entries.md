# Добавление новых элементов


Для добавления новых элементов реестра:


	- Выполните одно из действий:


		- нажмите кнопку ![](b_add.png) «Добавить
		 элемент» вкладки «Данные»
		 панели инструментов;


		- нажмите кнопку ![](b_add.png) «Добавить
		 элемент» в раскрывающемся меню команды «Таблица»
		 главного меню;


		- выполните команду «Добавить
		 элемент» в контекстном меню любого элемента [табличной
		 области](../Table/AreaTable.htm).


	- Выберите из списка измерение, в которое будет добавлена запись.
	 Будет открыт объект, настроенный при [расшифровке](../Table/Detail_dimensions.htm).


	- В открывшемся объекте добавьте новый элемент и сохраните данные.


Примечание.
 Сохранение данных доступно, если флажок «[Рассматривать
 пустую отметку как полную](UiNav.chm::/02_Navigator/Repo_Default.htm#MiniTOCBookMark3)» на вкладке «Поведение» окна «Параметры»
 снят.


После выполнения действий в корень измерения будет добавлен новый элемент.


Для удаления добавленных элементов:


	- Выделите элемент.


	- Выполните одно из действий:


		- нажмите кнопку ![](../Starting/delete.png) «Удалить элемент» вкладки «Данные» панели инструментов;


		- нажмите кнопку ![](../Starting/delete.png) «Удалить элемент» в раскрывающемся
		 меню команды «Таблица»
		 главного меню;


		- выполните команду «Удалить
		 элемент» в контекстном меню выделенного элемента.


После выполнения действий выделенный элемент будет удалён. Удалять элементы
 можно только по одному.


[![](../../opened.gif)![](../../closed.gif)Доступность
 кнопок «Добавить элемент» и «Удалить элемент»](javascript:TextPopup(this))


	Кнопка ![](b_add.png) «Добавить элемент» доступна, если:


		- в выделенной табличной области настроена расшифровка измерений
		 при установленном флажке «[Создавать
		 и удалять элементы справочника](../Table/Detail_dimensions.htm#create_element)»;


		- у пользователя есть права на чтение объекта, указанного в расшифровке.


	Кнопка ![](../Starting/delete.png) «Удалить
	 запись» доступна, если:


		- настроена расшифровка измерений при установленном флажке
		 «[Создавать
		 и удалять элементы справочника](../Table/Detail_dimensions.htm#create_element)»;


		- у пользователя есть права на удаление выделенного элемента
		 или элементов этого же уровня соседних измерений.
