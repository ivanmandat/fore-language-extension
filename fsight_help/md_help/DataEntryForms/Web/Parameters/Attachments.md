# Прикрепление вложений к ячейкам


Для пояснения данных в ячейке табличной области доступно прикрепление
 вложений в виде файлов или ссылок. Для прикрепления вложений используйте
 окно «Список вложений»:


[![](../../opened.gif)![](../../closed.gif)Для открытия
 окна](javascript:TextPopup(this))


		- Выделите ячейку области данных.


		- Выполните одно из действий:


			- нажмите кнопку ![](b_attachment.png) «Список
			 вложений» вкладки
			 «Данные» панели инструментов;


			- выполните команду ![](b_attachment.png) «Вложения»
			 контекстного меню;


			- нажмите на пиктограмму ![](b_attachment2.png) в ячейке,
			 если к ней уже были прикреплены вложения.


	Примечание.
	 Прикрепление вложений к ячейкам недоступно при работе с [глобальным кешем](UiNav.chm::/02_Navigator/Cache_In_Blob.htm).


	Кнопка ![](b_attachment.png) «Список
	 вложений» и команда ![](b_attachment.png) «Вложения»
	 доступны если:


		- в качестве источника использован [стандартный
		 куб](UiNavObj.chm::/Cube/CreateCube/Master_Standart/UiMd_Cube_CreateCube_Master_Standart_2.htm#attachment), [представление-куб](UiNavObj.chm::/Cube/CreateCube/Master_Conception/UiMd_Cube_CreateCube_Master_Conception.htm)
		 или [виртуальный
		 куб](UiNavObj.chm::/Cube/CreateCube/Master_Virtual_Cube/UiMd_Cube_CreateCube_Master_Virtual.htm), настроенный для привязки вложений;


		- выделена одна ячейка, принадлежащая области данных, не отображающая
		 значения [вычисляемых
		 элементов](../../../UiAnalyticalArea/Work_with_dimensions/Edited_elements.htm), [агрегированных
		 значений фиксированных измерений](DataEntryForms.chm::/desktop/Table/Table_Area_Structure.htm#aggregation) или [итогов](../../../UiAnalyticalArea/Totals/Calculate_totals.htm);


		- у пользователя есть права на чтение.


![](attachment_settings.png)


В окне указаны все вложения, прикрепленные к данной ячейке. Прикрепить
 можно любое количество вложений.


Для каждого вложения указаны:


	- Тип. Формат файла или
	 ссылка;


	- Наименование. Наименование
	 прикрепленного файла или заданное наименование ссылки;


	- Размер. Размер для файла
	 в байтах, килобайтах или мегабайтах;


	- Дата загрузки. Дата
	 прикрепления или обновления вложения;


	- Пользователь. [Полное
	 наименование пользователя](Admin.chm::/03_Admin/Users/Admin_UserProp_Common.htm#discription), который создал вложение.
	 Если полное наименование не задано, отображается [имя
	 пользователя](Admin.chm::/03_Admin/Users/Admin_UserProp_Common.htm#discription).


Для закрытия окна нажмите кнопку «Закрыть».


Операции с вложениями:


[![](../../opened.gif)![](../../closed.gif)Добавление
 вложения](javascript:TextPopup(this))


	Для выполнения операции у пользователя должны быть [права](Admin.chm::/03_Admin/Admin_AdminObjects.htm)
	 на чтение и редактирование.


	Для прикрепления файла:


		- Нажмите кнопку ![](../Starting/add.png) «Добавить
		 файл». Будет открыто стандартное окно выбора файла.


		- Укажите файл и нажмите кнопку «Открыть».
		 Размер файла не должен превышать 200 МБ. При попытке загрузить
		 файл большего размера будет выведено сообщение о превышении допустимого
		 размера файла.


	Для прикрепления ссылки:


		- Нажмите кнопку ![](../Starting/add.png) «Добавить
		 ссылку». Будет открыто окно «Ссылка»:


	![](reference.png)


		- Укажите:


			- Наименование.
			 Наименование вложения должно нести смысловую нагрузку. Поле
			 не обязательно для заполнения. Если наименование не задано,
			 то вместо наименования в окне «Список
			 вложений» будет отображаться ссылка;


			- Адрес ссылки.


[![](../../opened.gif)![](../../closed.gif)Открытие вложения](javascript:TextPopup(this))


	Для выполнения операции у пользователя должны быть [права](Admin.chm::/03_Admin/Admin_AdminObjects.htm)
	 на чтение.


	Для открытия вложения:


		- содержащего ссылку, нажмите кнопку ![](b_open.png)
		 «Открыть»;


		- щёлкните по необходимому наименованию.


	Примечание.
	 При открытии файла происходит его скачивание в папку загрузок браузера.


[![](../../opened.gif)![](../../closed.gif)Редактирование
 вложения](javascript:TextPopup(this))


	Для выполнения операции у пользователя должны быть [права](Admin.chm::/03_Admin/Admin_AdminObjects.htm)
	 на чтение и редактирование.


	Для редактирования вложения нажмите кнопку ![](../Starting/b_edit.png)
	 «Редактировать». Будет открыто
	 стандартное окно выбора файла или окно «Ссылка».


[![](../../opened.gif)![](../../closed.gif)Сохранение файла](javascript:TextPopup(this))


	Для выполнения операции у пользователя должны быть [права](Admin.chm::/03_Admin/Admin_AdminObjects.htm)
	 на чтение.


	Для сохранения файла на локальный компьютер нажмите кнопку ![](b_download.png) «Загрузить». Файл будет сохранен
	 в папку загрузок браузера.


[![](../../opened.gif)![](../../closed.gif)Удаление вложения](javascript:TextPopup(this))


	Для выполнения операции у пользователя должны быть [права](Admin.chm::/03_Admin/Admin_AdminObjects.htm)
	 на чтение и редактирование.


	Для удаления вложения нажмите кнопку ![](../Common/Delete.png) «Удалить».


Операции с вложениями не влияют на активность кнопки ![](../Starting/saveData.png) «Сохранить данные».


После добавления вложения в ячейке будет отображаться пиктограмма ![](b_attachment2.png).
 При наведении курсора на пиктограмму она становится активной.
