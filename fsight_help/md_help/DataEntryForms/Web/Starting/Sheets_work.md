# Работа с листами


При работе с листами возможны операции:


	- добавление листа;


	- удаление листа;


	- переименование листа;


	- копирование листа;


	- скрытие/отображение листа;


	- [защита листа](Defence.htm);


	- перемещение листа.


## Добавление листа


Для добавления листа нажмите кнопку ![](../Table/add.png) «Добавить лист» или нажмите сочетание
 клавиш SHIFT+F11, расположенную рядом с вкладками листов. Будет добавлен
 лист с наименованием «Лист N»,
 где N - порядковый номер добавляемого листа.


## Удаление листа


Для удаления листа выполните команду «Удалить»
 раскрывающегося меню кнопки ![](deleteSheet.png) «Действия»,
 расположенную справа на вкладке листа. Лист будет удален. В форме ввода
 должен присутствовать хотя бы один лист. Единственный лист удалить нельзя.


## Переименование листа


Для переименования листа выполните команду «Переименовать»
 раскрывающегося меню кнопки ![](deleteSheet.png) «Действия»,
 расположенную справа на вкладке листа. Введите новое имя листа на вкладке.
 Для сохранения имени нажмите клавишу ENTER или щёлкните за пределами вкладки
 листа.


Наименование листа может быть сохранено на разных языках в зависимости
 от текущего языка, выбранного в [окне регистрации](getstarted.chm::/GetStarted/Get_Started.htm).
 Для получения подробной информации обратитесь к разделу «[Настройка
 мультиязычности](UiNav.chm::/Multilanguage/Multilanguage_setting.htm)».


## Копирование листа


Для копирования листа выполните команду «Создать
 копию» раскрывающегося меню кнопки ![](deleteSheet.png) «Действия», расположенную справа
 на вкладке листа. Рядом с копируемым листом будет добавлена его копия -
 новый лист с наименованием по умолчанию. По умолчанию наименование листа
 «Лист N(M)», где N - порядковый
 номер копируемого листа, M - номер копии.


## Скрытие/отображение листа


Для скрытия листа выполните команду «Скрыть»
 раскрывающегося меню кнопки ![](deleteSheet.png) «Действия»,
 расположенную справа на вкладке листа. Лист будет скрыт. В раскрывающемся
 меню кнопки ![](AllSheets.png) «Все листы» скрытые листы обозначены
 пиктограммой ![](hide.png).
 Для отображения скрытого листа выберите его в раскрывающемся списке кнопки
 ![](AllSheets.png) «Все листы».


## Перемещение листа


Для изменения порядка расположения листов наведите курсор на вкладку
 листа. Курсор примет вид указателя ![](../Table/HandMovig.png).
 С помощью механизма Drag&Drop переместите лист в нужную позицию.
