# Настройка данных


Для настройки объединения нескольких значений элементов фиксированных
 измерений табличной области используйте группу параметров «Данные»
 [панели параметров](../Starting/Starting.htm#elements):


[![](../../opened.gif)![](../../closed.gif)Для открытия
 группы параметров «Данные»](javascript:TextPopup(this))


	Для открытия на панели параметров группы параметров «Данные»:


		- Выделите ячейку или диапазон ячеек, принадлежащих табличной
		 области.


		- Нажмите кнопку ![](../Table/BasicPropertiesButton.png) «Параметры» панели инструментов.


		- Выберите тип области «Таблица»
		 в раскрывающемся меню заголовка панели параметров.


![](Format_Table_Data.png)


Задайте агрегацию фиксированных измерений. Агрегация позволяет объединять
 значения элементов, отмеченных в фиксированных измерениях.


Для задания агрегации выберите метод в раскрывающемся списке:


	- Нет. Агрегация не производится.
	 Выбран по умолчанию;


	- Сумма. Суммируются значения
	 элементов фиксированных измерений;


	- Минимум. У элементов
	 фиксированных измерений выбирается минимальное значение;


	- Максимум. У элементов
	 фиксированных измерений выбирается максимальное значение;


	- Из измерений источника;


	- Среднее. Определяется
	 среднее значение элементов фиксированных измерений с учетом пустых
	 значений.
