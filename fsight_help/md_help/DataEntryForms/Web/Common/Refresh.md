# Обновление данных формы ввода


Обновление формы используется если:


	- изменены данные и структура в источниках;


	- изменены настройки оформления табличной области.


В формах ввода доступно обновление данных:


	- ручное;


	- автоматическое.


Примечание.
 Если в форме ввода есть табличная область, для которой была настроена
 [блокировка сохранения
 измененных данных](../Table/Calculation_Practices.htm#block) при сработавшем контроле, и измененные данные, которые
 не были сохранены, то будет выведено предупреждение, что измененные данные
 не будут сохранены, так как не удовлетворяют настройкам контролей. При
 согласии продолжить действие измененные данные будут сброшены и действие
 выполнено. При отказе продолжить действие измененные данные останутся
 в табличной области.


## Ручное обновление


Ручное обновление позволяет:


	- обновлять лист формы ввода.
	 Для обновления данных из источников, отстройки табличной области для
	 активного листа выполните команду «Обновить
	 лист» в раскрывающемся меню кнопки ![](../Starting/refresh.png) «Обновить» вкладки «Главная»
	 панели инструментов;


	- обновлять все листы формы ввода.
	 Для обновления данных из источников, отстройки табличной области для
	 всех листов:


		-  выполните команду «Обновить
		 все листы» в раскрывающемся меню кнопки ![](../Starting/refresh.png) «Обновить» вкладки «Главная»
		 панели инструментов;


		- нажмите клавишу F9. Доступно при редактировании отчёта;


	- переоткрывать
	 источники данных. Выполните команду «Переоткрыть
	 все источники данных» в раскрывающемся меню кнопки ![](../Starting/refresh.png) «Обновить»
	 вкладки «Главная» панели инструментов
	 для применения изменений, произошедших в источнике, например, добавление
	 и удаление элементов, атрибутов.


## Автоматическое обновление


Автоматическое обновление позволяет обновлять данные при изменении отметки
 в параметрах формы или с заданной периодичностью. По умолчанию установлено
 автоматическое обновление при изменении отметки в параметрах формы.


При автоматическом обновлении обновляются данные из источников, отстраивается
 табличная область для всех листов.


Для включения автоматического обновления при изменении отметки в параметрах
 формы установите флажок «При смене отметки»
 в раскрывающемся меню кнопки ![](../Work/refreshAuto.png) «Автоматическое обновление» вкладки
 «Главная» панели инструментов.
 Автоматическое обновление формы ввода будет включено. При каждом изменении
 отметки в параметрах формы будет производиться автоматическое обновление
 формы.


Для периодического обновления данных выполните команду «Задать
 период» в раскрывающемся меню кнопки ![](../Work/refreshAuto.png) «Автоматическое обновление». Будет
 открыт диалог «Период автообновления»:


![](../Work/AutoRefresh.png)


В диалоге задайте период обновления данных в форме ввода. Обновление
 может выполняться каждые N секунд, минут или часов. Максимально возможные
 значения N:


	- 10000 секунд;


	- 1000 минут;


	- 100 часов.


Примечание.
 Если для формы ввода уже настраивался период обновления, то его можно
 выбрать в раскрывающемся меню кнопки ![](../Work/refreshAuto.png) «Автоматическое обновление».


Обновление формы будет выполняться раз в заданный период времени.


Для включения автоматического обновления одновременно при смене отметки
 в параметрах формы и с заданной периодичностью, если период был настроен
 ранее, нажмите кнопку ![](../Work/refreshAuto.png) «Автоматическое
 обновление», если период не был задан, то при нажатии на кнопку
 будет включено автоматическое обновление при смене отметки в параметрах
 формы.


Для отключения периодического обновления данных установите флажок «Не использовать» в раскрывающемся
 меню кнопки ![](../Work/refreshAuto.png) «Автоматическое
 обновление».


Для отключения автоматического обновления переведите кнопку ![](../Work/refreshAuto.png) «Автоматическое
 обновление» в ненажатое состояние, при этом снимется флажок «При смене отметки» и будет установлен
 флажок «Не использовать». Автоматическое
 обновление формы ввода будет отключено.
