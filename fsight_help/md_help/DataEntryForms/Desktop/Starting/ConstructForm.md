# Построение формы ввода


	Построение формы ввода доступно в настольном приложении.


	Шаги для построения формы зависят от типа формы:


		- Карточка. Представляет
		 собой форму ввода с одним листом, содержащую набор полей для ввода
		 или выбора значений атрибутов элемента измерения. Предназначена
		 для добавления и редактирования записей в реестре;


		- Реестровая форма.
		 Представляет собой перечень элементов определенного типа, представленный
		 в табличном виде. В первых столбцах, боковике формы, выводятся
		 наименования элементов измерений, они могут быть расположены в
		 иерархическом порядке. В первых строках, шапке формы, выводятся
		 атрибуты и показатели элементов;


		- Табличная форма.
		 Представляет собой таблицу, в шапке и боковике которой выводятся
		 наименования элементов измерений, атрибутов и элементов измерений
		 показателей;


		- Комбинированная форма.
		 Представляет собой многолистовую форму, состав и содержание листов
		 формируется в зависимости от информационного объекта и его показателей.
		 Листы могут содержать карточки, реестровые и табличные формы,
		 данные могут формироваться автоматически, в том числе, на основе
		 данных других форм.


	Примечание.
	 При редактировании многолистовых форм ввода убедитесь, что установлен
	 флажок «[Загрузка
	 только активного листа](Starting.htm#multiexecutor)» для ускорения работы с формой.


	Показатели - это многомерные
	 наборы данных, объединенные общими аналитическими признаками, служащие
	 для сбора, хранения и отображения информации, объединенной единой
	 тематикой. В качестве показателя может быть выбран любой из доступных
	 типов кубов или база данных временных рядов.


	Измерения - справочники
	 репозитория, из которых состоит многомерная структура показателя.

## Общие шаги по сохранению и копированию формы


	Общими являются завершающие шаги:


		- [Создайте копию формы ввода](../Common/Save.htm),
		 если требуется.


		- Сохраните форму ввода. После внесения
		 изменений в структуру формы, открытую в режиме редактирования,
		 сохраните изменения:


	Для сохранения изменений в форме выполните
	 команду «Форма ввода/вывода >
	 Сохранить» главного меню или согласитесь на сохранение изменений
	 во всплывающем сообщении при закрытии формы.


	Примечание.
	 Если в форме ввода есть табличная область, для которой была настроена
	 [блокировка сохранения
	 измененных данных](../Table/Calculation_Practices.htm#block) при сработавшем контроле, и измененные данные,
	 которые не были сохранены, то будет выведено предупреждение, что измененные
	 данные не будут сохранены, так как не удовлетворяют настройкам контролей.
	 При согласии продолжить действие измененные данные будут сброшены
	 и действие выполнено. При отказе продолжить действие измененные данные
	 останутся в табличной области.


	Закончив построение формы ввода/вывода, перейдите к [работе
	 с готовой формой ввода](../Work/FinishForm.htm).

		Для удаления формы ввода:


			- нажмите кнопку ![](delete.png) «Удалить»
			 в группе «Упорядочить»
			 на вкладке «Главная»
			 ленты инструментов в навигаторе объектов;


			- выполните команду «Удалить»
			 контекстного меню формы в навигаторе объектов;


			- нажмите кнопки CTRL+DELETE, выделив форму в навигаторе
			 объектов.
