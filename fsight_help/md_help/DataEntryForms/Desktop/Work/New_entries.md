# Добавление новых записей


Для добавления новых элементов реестра нажмите кнопку ![](add.png) «Создать
 запись» вкладки «Таблица»
 ленты инструментов и выберите измерение, в которое будет добавляться запись, из
 списка. Условия доступности кнопки указаны [ниже](New_entries.htm#active).


Если при [расшифровке](../Table/Detail_dimensions.htm) измерения
 или первого уровня измерения в качестве объекта указан справочник НСИ,
 то при добавлении записи:


	- в веб-приложении справочник откроется на вкладке «[Описание](UiNavObj.chm::/reference_book/Master_RDS_reference_book/Master_RDS.htm#structure)».
	 Для добавления записи перейдите на вкладку «[Элементы
	 справочника](UiNavObj.chm::/reference_book/Work/Work_Dictionary.htm#rds)»;


	- в настольном приложении новая запись не появляется в форме ввода
	 автоматически. Для отображения новой записи переоткройте форму ввода.


[![](../../opened.gif)![](../../closed.gif)Доступность
 кнопок «Создать запись» и «Удалить запись»](javascript:TextPopup(this))


	Кнопка ![](add.png) «Создать запись» будет доступна,
	 если:


		- в выделенной табличной области настроена расшифровка измерений
		 или первых уровней измерений при установленном флажке «[Создавать
		 и удалять элементы справочника](../Table/Detail_dimensions.htm#create_element)»;


		- у пользователя есть [права](Admin.chm::/02_ComonFunction/Admin_SecManager_CommonFunction.htm)
		 на чтение хотя бы одного объекта, указанного в расшифровке.


	Кнопка ![](../Starting/delete.png) «Удалить
	 запись» будет доступна, если:


		- настроена расшифровка измерений или первых уровней измерений
		 при установленном флажке «[Создавать
		 и удалять элементы справочника](../Table/Detail_dimensions.htm#create_element)»;


		- у пользователя есть [права](Admin.chm::/03_Admin/Admin_ElementAccess.htm#abac)
		 на удаление выделенного элемента или элементов этого же уровня
		 соседних измерений;


		- в боковике/шапке отображены несколько измерений и выделен
		 элемент измерения, у которого отсутствует расшифровка или не установлен
		 флажок «[Создавать
		 и удалять элементы справочника](../Table/Detail_dimensions.htm#create_element)», находящийся правее,
		 если отсутствует [объединение](../Table/Merging_Dim.htm),
		 или ниже, если [объединение](../Table/Merging_Dim.htm)
		 настроено, измерений с расшифровкой. При этом удаляться будет
		 элемент измерения с настроенной расшифровкой, установленным флажком
		 «[Создавать
		 и удалять элементы справочника](../Table/Detail_dimensions.htm#create_element)» и разрешением на удаление,
		 располагающийся выше или левее (в зависимости от настроек объединения)
		 выделенного элемента.


После нажатия кнопки ![](add.png) «Создать
 запись» будет открыт объект, настроенный при расшифровке. Добавьте
 элемент и сохраните данные. Элемент будет добавлен в корень измерения.
 Если расшифровка настроена для всего измерения и для первого уровня измерения,
 то будет использоваться настройка, заданная для всего измерения.


Примечание.
 Сохранение данных доступно, если флажок «[Рассматривать
 пустую отметку как полную](UiNav.chm::/02_Navigator/Repo_Default.htm#MiniTOCBookMark3)» на вкладке «Поведение» окна «Параметры»
 снят.


Для удаления добавленных записей выделите элемент
 и нажмите кнопку ![](../Starting/delete.png) «Удалить
 запись» вкладки «Таблица»
 ленты инструментов. Удалять элементы можно только по одному. Условия доступности
 кнопки указаны [выше](New_entries.htm#active).


Для добавления обработчиков для кнопок ![](add.png) «Создать запись» и ![](../Starting/delete.png) «Удалить
 запись» на вкладке «Таблица»
 ленты инструментов используйте диалог «[Параметры](../Starting/Starting.htm#new_records)».
