# Пользовательские настройки формы ввода


При работе с готовой формой ввода можно изменить и сохранить настройки
 для каждого конкретного пользователя отдельно. Сохраненные настройки для
 конкретного пользователя называются пользовательскими настройками. При
 последующем открытии на просмотр формы ввода будет отображаться в соответствии
 с сохраненными настройками только для данного пользователя.


Примечание.
 Изменение пользовательских настроек формы ввода зависит от значения параметра
 [IsUserDataEnabled](#isuserdataenabled),
 содержащегося в ветке реестра [HKEY_CURRENT_USER\Software\Foresight\Foresight Analytics Platform\10.0\Reporter\Options].


Изначально вид формы ввода при просмотре и редактировании совпадает.
 При сохранении пользовательских настроек вид формы ввода при просмотре
 будет отличаться от вида формы ввода при редактировании.


К пользовательским настройкам относятся:


	- изменение состава отображаемых [показателей](View.htm#indicators)
	 и измерений показателей табличной области;


	- изменение порядка отображения [показателей](View.htm#indicators)
	 и измерений показателей табличной области;


	- изменение значений параметров в табличной области;


Примечание.
 Если при редактировании формы ввода параметр был удален или источник параметра
 изменился, то при просмотре формы ввода сохраненные значения параметра
 будут сброшены.


	- настройка [оформления](../Common/Design.htm#format)
	 табличной области;


	- настройка [вида](View.htm#view) отображения формы
	 ввода;


	- изменение ширины строк и столбцов табличной области вручную;


Примечание.
 Если при редактировании формы ввода была настроена [автоподгонка
 ячеек](../Table/Basic_Properties.htm#parameters), то при просмотре формы ввода будет недоступно сохранение измененной
 ширины строк и столбцов.


	- скрытие/отображение [строк](../Starting/Sheets.htm#rows)
	 и [столбцов](../Starting/Sheets.htm#columns);


	- настройка [фильтрации](FilterSearch.htm) табличной
	 области формы ввода.


Для изменения пользовательских настроек
 в форме ввода создайте параметр IsUserDataEnabled
 типа REG_DWORD со значением «1» в ветке реестра [HKEY_CURRENT_USER\Software\Foresight\Foresight Analytics Platform\10.0\Reporter\Options]
 на локальном компьютере пользователя.


Допустимые значения:


	- 0. По умолчанию. Изменение
	 пользовательских настроек формы ввода недоступно;


	- 1. Изменение пользовательских
	 настроек формы ввода доступно. При открытии формы ввода на просмотр
	 будут отображаться следующие элементы интерфейса:


		- команда «Форма ввода/вывода >
		 Сохранить» в главном меню;


		- кнопка ![](../Common/clear.png) «Сбросить все настройки» на вкладке
		 «Главная» ленты инструментов;


		- вкладка «[Формат](../Common/Design.htm#format)»
		 ленты инструментов.


Важно. Изменение
 пользовательских настроек может привести к снижению производительности
 системы и скорости открытия формы ввода.


Для сохранения измененных пользовательских настроек в форме ввода выполните
 команду «Форма ввода/вывода >
 Сохранить» в главном меню. При последующем открытии формы ввода
 на просмотр будут отображаться последние сохраненные пользовательские
 настройки.


Для сброса всех пользовательских настроек и возврата к исходному виду
 формы ввода нажмите кнопку ![](../Common/clear.png) «Сбросить все настройки» на вкладке
 «Главная» ленты инструментов.
 При просмотре формы ввода будут отображаться последние сохраненные настройки
 при редактировании.


Примечание.
 Кнопка ![](../Common/clear.png) «Сбросить все настройки» всегда активна,
 независимо от изменений пользовательских настроек.
