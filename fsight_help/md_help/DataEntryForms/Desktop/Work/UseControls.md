# Использование интерактивных элементов для управления отметкой


Элементы управления позволяют изменять отметку измерений.


Для отображения панели «Элементы
 управления» установите флажок «Элементы
 управления» на вкладке «Конструктор»
 ленты инструментов. По умолчанию флажок установлен.


Создание элементов управления производится при построении формы ввода
 при добавлении [параметров](../Parameters/Parameters.htm).


Элемент управления отображает выбранное измерение в заданном виде (например,
 в виде раскрывающегося списка), с возможностью поиска по измерению.


Для изменения отображаемых в форме данных выберите другой элемент измерения.


## Использование элементов управления при настроенной альтернативной иерархии


При смене значения параметра, атрибут которого является источником альтернативной
 иерархии для другого параметра:


	- элементы другого параметра меняются в соответствии с настроенными
	 связями. Отметка элементов сохраняется согласно связи между исходным
	 и альтернативным справочником: если в исходном справочнике отмечен
	 элемент, и в альтернативном справочнике есть связанный с ним элемент,
	 то при использовании альтернативного справочника элемент будет отмечен;


	- в случае если значение элемента не является ключом альтернативной
	 иерархии выводится предупреждение, что значение атрибута выбранного
	 элемента не является ключом альтернативной иерархии и будет применена
	 основная иерархия. Для прекращения вывода предупреждения в подобных
	 ситуация в рамках текущей сессии установите флажок «Не
	 показывать больше это сообщение».


Если выбрано несколько элементов параметра, у которых разные значения
 ключей справочников альтернативной иерархии в атрибуте, то в параметре,
 для которого настроена альтернативная иерархия от атрибута первого параметра,
 будут выведены элементы в соответствии с ключом альтернативной иерархии
 атрибута первого выбранного элемента.


Если в справочнике был удален атрибут, который был задан в качестве
 ключа альтернативной иерархии для параметра, то значение параметра в элементе
 управления формы ввода станет пустым. Для назначения другого атрибута
 параметра в качестве ключа альтернативной иерархии используйте [редактор
 параметров](../Parameters/Parameters.htm#properties).
