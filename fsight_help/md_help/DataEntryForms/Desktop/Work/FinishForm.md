# Работа с готовой формой ввода


При работе с готовой формой ввода доступен просмотр и ввод данных, а
 также сохранение [пользовательских настроек](Custom_settings.htm)
 для табличной области формы.


Примечание.
 При работе с многолистовыми формами ввода убедитесь, что установлен флажок
 «[Загрузка
 только активного листа](../Starting/Starting.htm#multiexecutor)» для ускорения работы.


Для удобного заполнения формы ввода используйте быструю навигацию.


[![](../../opened.gif)![](../../closed.gif)Быстрая навигация
 по форме ввода](javascript:TextPopup(this))


	Для быстрой навигации слева направо, сверху вниз по форме ввода
	 при просмотре используйте клавишу TAB:


		- в карточке переход будет осуществляться только по ячейкам,
		 для которых настроен редактор атрибутов;


		- в табличной области переход будет осуществляться только
		 в пределах табличной области.


	При достижении нижней крайней правой ячейки будет осуществлен переход
	 на верхнюю крайнюю левую ячейку.


	Для перемещения в обратном направлении используйте сочетание клавиш
	 SHIFT+TAB.


Для работы с готовой формой ввода выполните шаги:


	- [Обновите данные](../Common/Refresh.htm) формы ввода.


	- [Введите и сохраните данные](DataEnter.htm).


	- [Добавьте новые записи](New_entries.htm).


	- [Введите и согласуйте данные формы ввода](Agreement.htm).


	- [Просмотрите описание формулы](formula_description.htm).


	- [Используйте интерактивные элементы
	 для управления отметкой](UseControls.htm).


	- [Настройте внешний вид](View.htm).


	- [Используйте сортировку и фильтрацию](FilterSearch.htm),
	 если требуется.


	- [Используйте нумерацию и транспонирование](Transpose.htm),
	 если требуется.


	- [Используйте поиск и замену](Find_and_Replace.htm),
	 если требуется.


	- [Прикрепите
	 вложение к ячейке](UiAnalyticalArea.chm::/Attachments.htm), если требуется.


	- [Экспортируйте форму ввода](../Common/Export.htm),
	 если требуется.


	- [Просмотрите и напечатайте
	 форму ввода](../Common/ParamPage.htm), если требуется.


После работы с готовой формой выполните команду «Форма
 ввода/вывода > Выход» в главном меню. Если имеются несохранённые
 изменения, то будет предложено их сохранить.
