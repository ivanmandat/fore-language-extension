# Работа с данными через буфер обмена


При работе с данными в табличной области через буфер обмена доступны
 операции:


	- [копирование](#copy);


	- [перемещение](#cut);


	- [вставка](#insert).


Примечание.
 Работа с буфером обмена между вкладками браузера в веб-приложении доступна
 только с помощью сочетания клавиш. Ограничения могут действовать в зависимости
 от политики безопасности браузера.


## Копирование данных в буфер обмена


Для копирования значений ячейки или выделенного диапазона ячеек в буфер
 обмена:


	- нажмите кнопку ![](../Starting/copy.png) «Копировать», расположенную
	 в группе «Буфер обмена» вкладки
	 «Главная» ленты инструментов;


	- выполните команду «Копировать»
	 в контекстном меню ячейки или диапазона ячеек;


	- нажмите сочетание клавиш CTRL+C или CTRL+INSERT.


## Перемещение данных в буфер обмена


Для перемещения значений ячейки или выделенного диапазона ячеек в буфер
 обмена:


	- нажмите кнопку ![](../Starting/cut.png) «Вырезать», расположенную
	 в группе «Буфер обмена» вкладки
	 «Главная» ленты инструментов;


	- выполните команду «Вырезать»
	 в контекстном меню ячейки или диапазона ячеек;


	- нажмите сочетание клавиш CTRL+X или SHIFT+DELETE.


## Вставка данных из буфера обмена


Для вставки значений ячейки или выделенного диапазона ячеек из буфера
 обмена в текущее место табличной области:


	- нажмите кнопку ![](../Starting/paste.png) «Вставить», расположенную
	 в группе «Буфер обмена» вкладки
	 «Главная» ленты инструментов;


	- выполните команду «Вставить»
	 в контекстном меню ячейки или диапазона ячеек;


	- нажмите сочетание клавиш CTRL+V или SHIFT+INSERT.


### Особенности вставки данных из буфера обмена


	- Если вставляемый диапазон выходит за пределы табличной области,
	 то вставка осуществится в пределах табличной области.


	- Если для вставки диапазона ячеек выделен диапазон меньшего размера,
	 то размеры диапазона, куда вставляются данные, будут расширены вправо
	 и вниз до размера вставляемого диапазона.


	- Если для вставки диапазона ячеек выделен диапазон большего размера,
	 то будет заполнен только диапазон соответствующий размерам вставляемого
	 диапазона.


	- Если для вставки ячейки выделен диапазон, то диапазон будет
	 полностью заполнен данными из ячейки.


### Параметры форматирования при вставке
 из буфера обмена


Расширение «Интерактивные формы ввода
 данных» поддерживает вставку данных при копировании из продуктов:


	- Microsoft Office;


	- LibreOffice;


	- МойОфис;


	- Google Sheets.


При вставке содержимого из буфера на лист формы ввода в правом нижнем
 углу от вставленных ячеек появляется кнопка ![](insert_no_format.png) «Параметры вставки», информирующая
 о наличии собственного оформления у вставленных данных. При нажатии кнопки
 ![](insert_no_format_2.png) «Параметры вставки» отображаются
 варианты вставки:


	- Сохранить исходное форматирование.
	 Данные будут вставлены с использованием исходного форматирования;


	- Вставить без исходного форматирования.
	 Данные будут вставлены с использованием [формата
	 ячеек](../Common/Design.htm#index), в которые осуществляется вставка.


Примечание.
 При вставке данных, скопированных из продуктов «МойОфис» и «Google Sheets»,
 исходное форматирование не сохраняется.


Кнопка с параметрами вставки временно перестаёт отображаться:


	- при перемещении фокуса на другие элементы (например, шрифт,
	 строка формул и т.д.);


	- при открытии окна;


	- при вызове контекстного меню;


	- при переходе в режим редактирования одной из вставленных ячеек
	 и при условии, что ничего не изменялось и/или выход из режима редактирования
	 был осуществлён с помощью клавиши ESC.


Кнопка с параметрами вставки перестаёт отображаться полностью:


	- при изменении содержимого буфера обмена (например, скопированы
	 данные другой ячейки);


	- при переходе в режим редактирования любой ячейки вне области
	 вставки и выхода из режима редактирования, даже если ничего не изменялось;


	- при изменении состояния листа (например, изменение масштаба
	 листа);


	- при нажатии после вставки данных клавиши ESC.
