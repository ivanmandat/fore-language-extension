# Поиск и замена


Используйте поиск и замену, если требуется.


## Поиск


Для поиска данных в форме ввода выполните команду «Поиск >
 Поиск» на вкладке «Главная»
 ленты инструментов. Откроется диалог «Найти
 и заменить». На вкладке «Найти»
 в поле «Найти» введите значение,
 формулу или текст для поиска или выберите из раскрывающегося списка данные,
 которые искали ранее. Если необходимо, задайте дополнительные параметры
 поиска, нажав кнопку «Больше параметров»:


	- Формат. Для выбора формата
	 искомого значения используйте раскрывающееся меню кнопки «Формат»,
	 содержащее команды:


		- Задать формат. Открывает
		 стандартный диалог форматирования, в котором задается нужный формат.
		 Заданный пользователем формат отображается в поле «Образец».
		 Если формат не задан, в поле отображается текст «Формат
		 не задан»;


		- Выбрать формат из ячейки.
		 Скрывает диалог. После выделения ячейки отчета диалог отображается
		 снова, а в поле «Образец»
		 отображается формат выбранной ячейки;


		- Очистить формат поиска.
		 Сбрасывает выбранный формат. В поле «Образец»
		 отображается текст «Формат не
		 задан».


	- Учитывать регистр. Для
	 учёта регистра символов при поиске;


	- Ячейка целиком. Для
	 нахождения ячеек, которые полностью содержат значение, введённое для
	 поиска;


	- Область поиска. Для
	 выбора области поиска. Раскрывающийся список содержит значения: на
	 листе, в отчёте;


	- Порядок поиска. Для
	 выбора порядка поиска. Раскрывающийся список содержит значения: по
	 строкам, по столбцам;


	- Предмет поиска. Для
	 выбора предмета поиска используйте флажки «Текст»
	 или «Формула».


## Замена


Для замены данных в форме ввода выполните команду «Поиск >
 Замена» на вкладке «Главная»
 ленты инструментов. Откроется диалог «Найти
 и заменить». На вкладке «Заменить»
 в поле «Найти» введите значение,
 формулу или текст для поиска или выберите из раскрывающегося списка данные,
 которые искали ранее.


В поле «Заменить на» введите
 значения, формулы или текст, на которые нужно заменить искомое значение.


Если необходимо, задайте дополнительные параметры поиска.


Нажмите на кнопку «Заменить»
 для последовательной замены данных в ячейках таблицы или кнопку «Заменить все» для замены всех найденных
 значений.
