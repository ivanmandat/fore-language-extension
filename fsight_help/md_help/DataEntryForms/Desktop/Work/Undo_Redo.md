# Отмена и повтор операции


Для отмены последней операции в форме ввода:


	- нажмите кнопку ![](../Common/undo.png) «Отменить
	 действие» на панели быстрого доступа ленты инструментов;


	- нажмите сочетание клавиш CTRL+Z.


Если [измененные данные](DataEnter.htm) были сохранены, а
 затем сохранение было отменено, то будут возвращены предыдущие значения.
 Эти значения будут выделены как измененные, и для дальнейшего их использования
 требуется выполнить их сохранение.


Для повтора последней отменённой операции:


	- нажмите кнопку ![](../Common/redo.png) «Повторить
	 действие» на панели быстрого доступа ленты инструментов;


	- нажмите сочетание клавиш CTRL+Y.


Кнопки ![](../Common/undo.png) «Отменить действие» и ![](../Common/redo.png) «Повторить
 действие» на панели быстрого доступа становятся доступными при
 изменении данных.


При [обновлении](../Common/Refresh.htm) формы ввода и расчёте
 [алгоритмов на
 сохранение](../Table/Calculation_Practices.htm#addalgorithm) очищается история операций и кнопки ![](../Common/undo.png) «Отменить действие» и ![](../Common/redo.png) «Повторить
 действие» на панели быстрого доступа становятся недоступными.
