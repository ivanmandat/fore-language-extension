# Выведение атрибута измерения в шапку


При [настройке структуры показателей](../Factors_Structure.htm)
 в мастере табличной области формы ввода переместите измерения или атрибуты
 из области «[Доступные
 показатели](../Factors_Structure.htm)» в область «[Показатели формы ввода](../Factors_Structure.htm)»
 для отображения в шапке:


	- измерений со всеми атрибутами;


	- измерений с выбранными атрибутами;


	- одного и того же измерения с набором атрибутов несколько раз.


Для перемещения измерений и атрибутов из области «Доступные
 показатели»:


	- Измерение. Выделите
	 одно или несколько измерений и с помощью механизма Drag&Drop переместите
	 в область «Показатели формы ввода».
	 В области отобразится измерение со всеми атрибутами;


	- Атрибуты. Выделите один
	 или несколько атрибутов одного или нескольких измерений и с помощью
	 механизма Drag&Drop переместите в область «Показатели
	 формы ввода». В области отобразятся измерения с выделенными
	 атрибутами.


Примечание.
 Перемещение календарного измерения и его атрибутов в область «Показатели
 формы ввода» недоступно.


Для изменения наименования выведенного измерения выполните команду «Переименовать» контекстного меню
 измерения в области «Показатели формы
 ввода» и введите новое наименование.


После выполнения действий [добавьте
 вычисляемые показатели](Adding_Calculated_Indicators.htm) при необходимости.
