# Дополнительные настройки структуры показателей


При [настройке структуры показателей](../Factors_Structure.htm)
 в мастере табличной области формы ввода можно дополнительно настроить
 структуру показателей:


	- [добавьте](#add) доступные показатели в шапку;


	- [измените](#moveindicators) порядок расположения
	 измерений и показателей в шапке;


	- [переименуйте](#rename) измерения или показатели
	 для увеличения смысловой нагрузки наименования;


	- [установите запрет](#ban) ввода значений для
	 показателя;


	- [настройте
	 ограничение](Advanced_Structure_Settings.htm#parameter) доступных значений частного измерения показателя.
	 Настройка выполняется в области «[Показатели
	 формы ввода](../Factors_Structure.htm)»;


	- [настройте расшифровку](../Detail_dimensions.htm)
	 измерений и уровней измерений для настройки перехода от элемента измерения
	 к объекту, детализирующему его значение. Настройка выполняется в области
	 «[Показатели
	 формы ввода](../Factors_Structure.htm)».


## Добавление доступных показателей в шапку


Если требуется отобразить в шапке одни и те же показатели с различными
 отметками в частных измерениях, повторно добавьте доступные показатели
 в шапку. Для этого выберите один или несколько показателей в области «[Доступные
 показатели](../Factors_Structure.htm)» и переместите их с помощью механизма Drag&Drop в область
 «[Показатели
 формы ввода](../Factors_Structure.htm)» в любое место списка.


После перемещения показателей [задайте
 отметку](Selecting_Elements.htm) элементов частных измерений.


## Изменение расположения


Для изменения порядка расположения атрибутов измерений, измерений внутри
 показателя, показателей между собой, всех показателей и общих измерений
 шапки используйте кнопки ![](../moveUp.png) «Переместить
 вверх» и ![](../moveDown.png) «Переместить
 вниз», выбрав измерение/показатель в области «[Показатели
 формы](../Factors_Structure.htm)».


Для изменения расположения измерений показателя:


	- выполните команду «По столбцам»
	 контекстного меню измерения для расположения по столбцам;


	- выполните команду «В фиксированные»
	 контекстного меню измерения для расположения в фиксированных.


## Переименование измерения или показателя


Для увеличения смысловой нагрузки наименования измерения или показателя
 переименуйте его. Для этого выполните команду «Переименовать»
 контекстного меню измерения или показателя в области «[Показатели формы ввода](../Factors_Structure.htm)».


Введите новое наименование и завершите редактирование, нажав клавишу
 ENTER.


Примечание.
 Изменение наименования применяется в рамках выбранной табличной области.


## Запрет ввода значений для показателя


Для запрета ввода значений для показателя в форму установите флажок
 «Запретить ввод значений» в контекстном
 меню показателя в области «[Показатели
 формы](../Factors_Structure.htm)». Флажок становится недоступным при установке флажка
 «[Использовать
 оформление листа](../Basic_Properties.htm#parameters)».


## Ограничение доступных значений частного измерения
 показателя


Для ограничения доступных значений частного измерения показателя [задайте значение
 параметра](../Select_dimension_elements.htm#parameter) измерения. Настройка доступна если в кубе, использованном
 в качестве показателя, настроено [управление
 измерением](UiNavObj.chm::/Cube/CreateCube/Master_Standart/UiMd_Cube_CreateCube_Master_Standart_managed_dim.htm).


Настройка выполняется в области «[Показатели формы ввода](../Factors_Structure.htm)».
