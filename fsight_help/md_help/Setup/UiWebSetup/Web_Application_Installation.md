# Установка веб-приложения


Для работы с веб-приложением продукта «Форсайт. Аналитическая платформа»
 установите последовательно следующие компоненты:


	- [BI-сервер](UiWebSetup_TitlePage.htm).
	 Описание процесса установки выделенного BI-сервера продукта «Форсайт.
	 Аналитическая платформа» на серверах с различными операционными системами,
	 а также дополнительные настройки для формирования отказоустойчивых
	 систем ([сервер
	 состояний](01_BIServer/StateServer/State_Server.htm), [кластер
	 BI-серверов](01_BIServer/Cluster/Cluster_Of_BI_Server.htm), [пул соединений](01_BIServer/PoolConnections.htm));


	- [Серверная
	 часть веб-приложения](03_Setup_Web/Web_server_setup.htm). Описание процесса установки базовых
	 файлов, необходимых для работы веб-приложений, на различных веб-серверах
	 и в различных операционных системах.


Совет. Для быстрой
 установки веб-приложения в ОС Linux (СУБД PostgreSQL) обратитесь к разделу
 «[Установка веб-приложения
 «Форсайт. Аналитическая платформа» в ОС Linux (СУБД PostgreSQL)](Install_Web_on_Linux_PostgreSQL.htm)».
 Для расширенной установки веб-приложения с минимально необходимыми требованиями
 обратитесь к разделу «[Расширенная
 установка веб-приложения «Форсайт. Аналитическая платформа» в ОС Linux
 (СУБД PostgreSQL)](InstallEx_Web_on_Linux_PostgreSQL.htm)».
