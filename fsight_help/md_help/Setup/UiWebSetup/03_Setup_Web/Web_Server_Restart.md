# Перезапуск серверной части веб-приложения


Для перезапуска серверной части веб-приложения:


	- в ОС Linux:


		- Debian-подобные дистрибутивы:


sudo systemctl restart
 apache2-fp10.x-web


		- RedHat-подобные дистрибутивы:


sudo systemctl restart
 httpd-fp10.x-web


		- ALT Linux:


sudo systemctl restart
 httpd2-fp10.x-web


Для получения подробной информации обратитесь
 к [документации
 Apache2](http://httpd.apache.org/docs/2.2/stopping.html);


	- в ОС Windows обратитесь к документации [Internet
	 Information Services](https://docs.microsoft.com/ru-ru/previous-versions/commerce-server/ms957500(v%3dcs.70)).
