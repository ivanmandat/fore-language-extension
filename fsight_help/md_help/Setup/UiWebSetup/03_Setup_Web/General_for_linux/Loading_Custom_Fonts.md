# Загрузка пользовательских шрифтов


По умолчанию в веб-приложении доступен набор шрифтов, установленных
 в операционной системе BI-сервера. Шрифты используются при работе и настройке
 оформления объектов:


	- [Форма
	 ввода/вывода](dataentryforms.chm::/DataEntryForms_Purpose.htm);


	- [Регламентный
	 отчёт](uireport.chm::/UiReport_purpose.htm);


	- [Алгоритм
	 расчёта показателей](calculationalgorithm.chm::/Purpose.htm);


	- [Задача
	 ETL](uietl.chm::/Desktop/01_General_Info/UiETL_General.htm);


	- [Веб-форма](uidevenv.chm::/01_Development_Environment/02_Work_in_Development_Environment/DevEnv_Object/Web_Form.htm);


	- [Таблица
	 стилей](uinav.chm::/StylesTable/StylesTablePurpose.htm);


	- [Рабочее
	 пространство](uiwsp.chm::/1_intro/UiWsp_Intro.htm);


	- [Аналитические панели](UIAdhoc.chm::/UiAdhoc_Purpose.htm).


Для расширения набора шрифтов на BI-сервере доступна загрузка пользовательских
 шрифтов в формате *.ttf. Учитывайте лицензионные и другие ограничения
 пользовательских шрифтов.


Для загрузки пользовательских шрифтов, которые не установлены на BI-сервере:


	- в ОС Linux:


		- Добавьте шрифты в папку /opt/foresight/fp10.x-biserver/share/fonts.


		- Укажите путь до шрифтов в файле /etc/fonts/fonts.conf:


<dir>/opt/foresight/fp10.x-biserver/share/fonts</dir>


	- в ОС Windows установите шрифты для всех пользователей на компьютере:


		- Выполните команду «Установить
		 для всех пользователей» в контекстном меню шрифта.


		- Убедитесь, что шрифт содержится в разделах реестра [HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows
		 NT\CurrentVersion\Fonts], [HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows
		 NT\CurrentVersion\Fonts].


После выполнения действий пользовательские шрифты будут доступны для
 использования в раскрывающемся списке «Шрифт»
 при работе с указанными объектами.


Если на компьютере пользователя не содержатся шрифты, загруженные на
 BI-сервере, то при работе с объектами будет выполнено автоматическое применение
 недостающих шрифтов.
