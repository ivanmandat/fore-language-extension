# Установка серверной части веб-приложения
 на Java


Производите установку от имени пользователя, обладающего правами локального
 администратора компьютера.


Установка серверной части веб-приложения на Java веб-сервер (контейнер
 сервлетов) выполняется с помощью специального архива веб-приложения pp.war.
 Процесс распаковки и настройки зависит от используемой версии Java.


По умолчанию веб-приложение устанавливается по пути:


http://<рабочая_станция>:<порт>/pp


Где:


	- <рабочая_станция>:<порт>.
	 Наименование рабочей станции и порт, по которому функционирует веб-сервер
	 Java;


	- pp. Наименование виртуального
	 каталога, в котором расположено веб-приложение.


Далее следует задать [настройки](PP_config_Java.htm) веб-приложения.


Примечание.
 Расширенные настройки веб-приложения Java задаются в файле web.xml, расположенном
 по пути S\pp\WEB-INF, где S - путь, по которому установлено веб-приложение.
 Описание используемых параметров приведено в документации к спецификации
 Servlet 2.5.
