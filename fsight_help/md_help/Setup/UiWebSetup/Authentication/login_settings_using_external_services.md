# Настройка аутентификации через
 внешние сервисы


«Форсайт. Аналитическая платформа»
 поддерживает авторизацию пользователей, прошедших аутентификацию на внешних
 сервисах. В зависимости от сервиса, может отличаться процесс настройки
 и список задаваемых параметров. Более подробная информация и расширенная
 схема взаимодействия веб-приложения с внешними сервисами представлены
 в следующих подразделах:


	- [OAuth 2.0 или OpenID Connect](OAuth_OpenID_Connect.htm)


	- [JWT-токен](JWT.htm)


	- [HTTP-заголовки](HTTPHeaders.htm)
