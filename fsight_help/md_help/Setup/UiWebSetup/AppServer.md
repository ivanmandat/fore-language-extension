# Сервер приложений


Сервер приложений предназначен для выполнения отложенных ресурсоёмких
 задач, запускаемых с помощью инструмента «[Планировщик
 задач](uiappsrv.chm::/UiAppSrv_purpose.htm)». Располагайте сервер приложений отдельно
 от BI-сервера.


## Установка и настройка сервера приложений


Установка и настройка сервера приложений аналогична установке и настройке
 [BI-сервера](UiWebSetup_TitlePage.htm).


## Настройка сервера при расчёте задачи моделирования


Если при расчёте задачи моделирования используются методы R необходимо:


	- Установить пакеты R на сервере приложений. Особенности установки
	 пакета R при использовании ОС Astra Linux указаны ниже.


Примечание.
 Для получения подробной информации об установке пакета R обратитесь к
 [документации к R](http://cran.r-project.org/manuals.html).


	- Установить пакет Rserve в R. Особенности установки пакета Rserve
	 при использовании ОС Astra Linux указаны ниже.


Примечание.
 Для получения подробной информации об установке пакета Rserve обратитесь
 к [документации к Rserve](http://rforge.net/Rserve/doc.html).


	- Запустить Rserve на сервере.


### Особенности установки пакетов R и Rserve для Astra Linux


Для установки пакетов R и Rserve выполните:


	apt-get install r-base r-base-dev


	apt-get install r-cran-rserve
