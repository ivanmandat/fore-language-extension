# Ручная установка и настройка Apache2
 для доступа к конструктору бизнес-приложений


Ниже приведён пример ручной установки и настройки Apache2 для доступа
 к ресурсам конструктора бизнес-приложений в ОС Linux.


Совет. Для установки
 серверной части конструктора бизнес-приложений в ОС Linux рекомендуется
 использовать дистрибутивы, входящие в [комплект
 поставки продукта «Форсайт. Аналитическая платформа»](../../02_AK_Install/DistributionKit.htm#linux).


Для установки серверной части конструктора бизнес-приложений вручную:


	- Убедитесь, что установлен [BI-сервер](../../UiWebSetup/Install_Web_on_Linux.htm).


	- Создайте каталог, в котором будут располагаться файлы серверной
	 части конструктора бизнес-приложений, например: /opt/foresight/dba/10.x.


	- Скопируйте архив DBA.war из папки Java_DBAWeb, входящей в [комплект поставки](../../02_AK_Install/DistributionKit.htm#linux).


	- Вставьте архив DBA.war в каталог /opt/foresight/dba/10.x и распакуйте его с
	 помощью команды:


unzip DBA.war


После выполнения действий будет установлена серверная часть конструктора
 бизнес-приложений. Задайте дополнительные настройки для Apache2 в зависимости
 от операционной системы, в которой запущен веб-сервер:


	- [на Debian-подобных
	 дистрибутивах](Configure_Apache_HTTP_Server_DBA_Debian.htm);


	- [на RedHat-подобных
	 дистрибутивах](Configure_Apache_HTTP_Server_DBA_RHEL.htm);


	- [на
	 ОС ALT Linux](Configure_Apache_HTTP_Server_DBA_ALT_Linux.htm).
