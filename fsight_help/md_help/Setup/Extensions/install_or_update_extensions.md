# Установка расширений продукта «Форсайт. Аналитическая платформа»


Расширения - это дополнительные инструменты продукта «Форсайт. Аналитическая платформа»,
 которые по умолчанию недоступны, но могут быть установлены отдельно и
 использоваться для расширения функциональности создаваемых систем. Продукт
 «Форсайт. Аналитическая платформа»
 включает в себя следующие расширения: «[Алгоритмы расчёта](CalculationAlgorithm.chm::/Purpose.htm)»,
 «[Управление бизнес-процессами](BPM.chm::/Modeling_of_business_processes.htm)»,
 «[Интерактивные формы ввода данных](DataEntryForms.chm::/DataEntryForms_Purpose.htm)»,
 «[Конструктор бизнес-приложений](Constructor.chm::/Intro/Designer_business_applications.htm)»,
 «[Кеш кубов](UiNav.chm::/02_Navigator/Cache_In_Blob.htm#console)».


Расширение может зависеть от других расширений, поэтому при его установке
 также будут установлены те расширения, от которых оно зависит. Зависимость
 расширений:


Интерактивные формы ввода данных > Конструктор
 бизнес-приложений


Алгоритмы расчёта > Конструктор бизнес-приложений


Управление бизнес-процессами > Конструктор бизнес-приложений


Кеш кубов > Конструктор бизнес-приложений


Таким образом:


	- расширения «Интерактивные
	 формы ввода данных», «Алгоритмы
	 расчёта», «Управление бизнес-процессами»
	 и «Кеш кубов» зависят только
	 от расширения «Конструктор бизнес-приложений»,
	 и при их установке также будет установлено данное расширение;


	- расширение «Конструктор бизнес-приложений»
	 не зависит от других расширений и устанавливается отдельно.


Для использования расширений:


	- [установите](install_extensions_in_repository.htm)
	 расширения в репозиторий;


	- [обновите](Update/update_extensions.htm) уже установленные
	 расширения, если имеются файлы с новой версией;


	- установите и настройте [серверную
	 часть конструктора бизнес-приложений](Backend_Designer_of_Business_Applications.htm) для открытия и работы с готовыми
	 бизнес-приложениями в веб-интерфейсе.


Примечание.
 Установку расширений выполняйте от имени пользователя, являющегося владельцем
 базы данных (схемы).


После установки расширения «Управление
 бизнес-процессами» настройте [серверное
 исполнение бизнес-процессов](BPM.chm::/Desktop/Process/Setting_Server_Process_Execution.htm) по расписанию с помощью
 планировщика задач.
