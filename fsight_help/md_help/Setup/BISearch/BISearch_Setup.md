# Установка и настройка сервиса BI-поиска


BI-поиск - это полнотекстовый
 поиск данных по ключевым словам в структуре многомерных источников репозитория.
 Работа BI-поиска в «Форсайт. Аналитическая платформа»
 реализована на базе поисковой платформы [Apache Solr](http://lucene.apache.org/solr/).
 Данный сервис предоставляет следующие возможности:


	- поиск по источникам с формированием необходимого среза данных
	 (поиск осуществляется по элементам измерений источников);


	- поиск источников по их метаданным (например, по наименованию,
	 идентификатору или описанию);


	- расширения списка индексируемых данных путем создания пользовательских
	 полей (заполнение значений полей осуществляется через макрос на Fore);


	- поиск по свободным измерениям, элементы которых индексируются,
	 но не отображаются в результате поиска (смарт-поиск);


	- управление релевантностью поиска по различным полям.

		- [Добавление языков в Solr](Add_Language.htm).

	 После задания настроек BI-поиск также будет доступен при работе BI-сервера.


	- [Индексация
	 источников](Indexing_DataSource.htm).


Задаваемые настройки определяют, каким образом будет производиться BI-поиск
 (поиск по данным, метаданным, с учетом свободных измерений - смарт-поиск),
 определяется список индексируемых полей, а также релевантность поиска
 по этим полям. При возникновении каких-либо ошибок при настройке, индексации
 и использовании BI-поиска обратитесь к подразделу «[Решение возможных проблем с продуктом
 «Форсайт. Аналитическая платформа»](FAQ.chm::/FAQ_title.htm)».

 программного обеспечения.
