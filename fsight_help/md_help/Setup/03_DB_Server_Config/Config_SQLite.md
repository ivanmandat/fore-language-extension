# Подготовка SQLite


Встраиваемая СУБД SQLite не имеет серверной части. Все необходимое программное
 обеспечение для работы с этой СУБД устанавливается вместе с «Форсайт. Аналитическая платформа»
 (смотрите [Комплект поставки
 продукта «Форсайт. Аналитическая платформа»](../02_AK_Install/DistributionKit.htm)).
 После установки «Форсайт. Аналитическая платформа»
 каких-либо дополнительных настроек производить не нужно, можно сразу приступить
 к [созданию репозитория](../05_RepoMngr/Setup_RepoMngr_Work_Main.htm).


Примечание.
 Для репозитория на основе SQLite работа с продуктом «Форсайт. Аналитическая
 платформа» доступна только в [настольном
 приложении](GetStarted.chm::/Desktop_and_web_application.htm) в однопользовательском режиме.
