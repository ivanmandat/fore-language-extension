# Системные требования к серверу баз данных


Ниже представлены рекомендации к аппаратному обеспечению и требования
 к программному обеспечению для сервера баз данных.


## Аппаратное
 обеспечение


Требования к серверу СУБД определяется типом СУБД и предполагаемым объёмом
 хранимых данных.


Для определения требований к серверу СУБД обращайтесь к разработчикам
 СУБД.


Требования к каналу связи зависят от специфики проекта и предполагаемого
 количества одновременно работающих пользователей.


Ширина канала связи рассчитывается индивидуально для каждой прикладной
 системы на основании результатов нагрузочного тестирования. Для первоначальной
 оценки обратитесь в техническую поддержку по адресу [support@fsight.ru](mailto:support@fsight.ru)
 или через [сервисы
 технической поддержки](https://www.fsight.ru/support/), которые доступны после регистрации на сайте.


## Программное
 обеспечение


		 Программное обеспечение


		 СУБД
		 Серверная часть СУБД:


			- PostgreSQL;


			- Postgres Pro;


			- Microsoft SQL Server;


			- Oracle.


		Важно.
		 Не поддерживается работа с Oracle 12.x, если при установке серверной
		 части была включена опция для поддержки технологии Oracle Multitenant.


		Примечание.
		 Также в «Форсайт. Аналитическая платформа»
		 поддерживается работа с СУБД SQLite, которая не имеет серверной
		 части. Более подробно смотрите в разделе «[Системные
		 требования к клиентскому окружению](Setup_DB_ServerSysReq.htm#client)».

		Для получения подробной информации о поддерживаемых версиях СУБД
		 обратитесь к статье «[Поддерживаемые
		 СУБД](../01_SysReq/database_Support.htm)».


		 Операционная система
		 Для подбора конфигурации операционных систем и процессоров
		 (RISC систем) для сервера следует обратиться к документации по
		 СУБД.


# Системные требования к клиентскому окружению


Ниже представлены системные требования к аппаратному и программному
 обеспечению для клиентского окружения, выполнение которых необходимо для
 корректной установки и работы продукта «Форсайт. Аналитическая платформа».
 Требования к программному обеспечению для работы в настольном приложении
 и веб-приложении отличаются.


## Аппаратное
 обеспечение


		 Минимальные требования


		 Процессор
		 2 ядра, тактовая частота 2.40 ГГц и выше


		 Разрядность платформы
		 64-разрядная


		 Оперативная память
		 4 ГБ и выше


		 Видеокарта
		 Видеокарты с поддержкой Shader Model 3.0, такие как:


			- Nvidia GeForce 6600 и выше;


			- ATI Radeon X1300 и выше;


			- чипсеты Intel G35 (GMA X3500), Intel G4x, Q4x и выше.


		 Жесткий диск
		 Свободный объем от 20 ГБ*


		 Монитор
		 Разрешение экрана 1920х1080, диагональ 22 дюйма и выше


* - Свободный объем указан с учётом возможности использования кеширования
 объектов и данных. По умолчанию временные файлы распаковываются на диск
 с операционной системой. Месторасположение временных файлов определяется
 в переменной среды TMP для ОС Windows или TMPDIR для ОС Linux.


## Программное обеспечение при работе через веб-приложение


		 Программное обеспечение


		 Браузер
		 Поддерживаются не более трёх последних версий следующих
		 браузеров:


			- Яндекс.Браузер;


			- Chromium-Gost;


			- Google Chrome;


			- Mozilla Firefox;


			- Chromium.


		 СУБД
		 Установка клиентской части СУБД не требуется.


		 Операционная система
		 Любые версии операционных систем, на которых работают поддерживаемые
		 версии браузеров, которые указаны выше.


Для работы веб-приложения в браузере Chromium-Gost обратитесь к разделу
 «[Настройка
 веб-приложения для работы в браузере Chromium-Gost](../UiWebSetup/04_FAQ/Setting_Web_ChromiumGost.htm)». Актуально только
 для Astra Linux SE 1.7.


Примечание.
 Веб-приложение не предполагает работу на мобильных устройствах. Для работы
 с продуктом «Форсайт. Аналитическая платформа»
 на мобильных устройствах используйте мобильное приложение «[Форсайт. Мобильный BI](MobileApp.chm::/Mobile_app.htm)[](../../MobileApp/Mobile_app.htm)».

# Системные требования к BI-серверу и серверу приложений


Ниже представлены рекомендации к аппаратному обеспечению и требования
 к программному обеспечению для [BI-сервера](../UiWebSetup/UiWebSetup_TitlePage.htm)
 и [сервера приложений](../UiWebSetup/AppServer.htm), выполнение
 которых необходимо для корректной работы через веб-сервис «Форсайт. Аналитическая платформа».


## Аппаратное
 обеспечение


Аппаратные требования напрямую зависят от:


	- количества одновременно работающих пользователей;


	- объёмов и организации данных в СУБД;


	- объёма запрашиваемых данных при работе пользователя с системой;


	- уровня сложности бизнес-объектов и прикладной логики;


	- операционной системы на сервере;


	- системы виртуализации.


Поддерживаемые протоколы передачи данных: HTTP / HTTPS.


Требования к каналу связи зависят от специфики проекта и предполагаемого
 количества одновременно работающих пользователей.


Аппаратные требования рассчитываются индивидуально для каждой прикладной
 системы на основании результатов нагрузочного тестирования. Для первоначальной
 оценки обратитесь в техническую поддержку по адресу [support@fsight.ru](mailto:support@fsight.ru)
 или через [сервисы
 технической поддержки](https://www.fsight.ru/support/), которые доступны после регистрации на сайте.

# Системные требования к серверу веб-приложения


Ниже представлены рекомендации к аппаратному обеспечению сервера и требования
 к программному обеспечению для корректной работы веб-приложения «Форсайт. Аналитическая платформа».


## Аппаратное
 обеспечение


Выбор сервера веб-приложения во многом определяется предполагаемым количеством
 одновременно работающих пользователей и спецификой проекта.


Поддерживаемые протоколы передачи данных: HTTP / HTTPS.


Требования к каналу связи зависят от специфики проекта и предполагаемого
 количества одновременно работающих пользователей.


Аппаратные требования рассчитываются индивидуально для каждой прикладной
 системы на основании результатов нагрузочного тестирования. Для первоначальной
 оценки обратитесь в техническую поддержку по адресу [support@fsight.ru](mailto:support@fsight.ru)
 или через [сервисы
 технической поддержки](https://www.fsight.ru/support/), которые доступны после регистрации на сайте.


## Программное
 обеспечение


		 Программное обеспечение


		 ОС семейства Windows
		 ОС семейства Linux


		 Операционная система
		 Поддерживаются следующие версии ОС семейства Windows:


				- Windows 10;


				- Windows Server 2012;


				- Windows Server 2012 R2;


				- Windows Server 2016;


				- Windows Server 2019.


		 Поддерживаются следующие версии ОС семейства Linux:


			- Astra Linux;


			- РЕД ОС;


			- Rocky Linux;


			- ALT Linux.


		Подробное описание поддерживаемых версий ОС Linux см. в разделе
		 «[Поддерживаемые
		 версии ОС Linux](../UiWebSetup/02_SystemReq/system_requirments_Linux.htm)».


		 Другое ПО
		 Если требуется использование серверной части веб-приложения
		 (интегрированная доменная аутентификация), то используйте Internet
		 Information Services 8.0, 8.5, 10.


		В обратном случае можно использовать любой веб-сервер, позволяющий
		 отдавать статичные файлы.


		 Если требуется использование серверной части веб-приложения,
		 то используйте Apache2. Также можно использовать Apache Tomcat,
		 если не предполагается использование доменной аутентификации.


		В обратном случае можно использовать любой веб-сервер, позволяющий
		 отдавать статичные файлы.
