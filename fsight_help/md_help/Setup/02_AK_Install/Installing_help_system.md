# Установка справочной системы


Справочная система содержится в [комплекте
 поставки продукта](DistributionKit.htm) и доступна для установки с помощью инсталляторов:


	- в ОС Linux:


		- FP10.9help_ru.run. Локальная справка
		 на русском языке для веб-приложения. Поиск в справочной системе
		 доступен только по ключевым словам в указателе;


		- FP10.9help_en.run. Локальная справка
		 на английском языке для веб-приложения. Поиск в справочной системе
		 доступен только по ключевым словам в указателе;


Примечание.
 Для получения инсталлятора обратитесь в техническую поддержку по адресу
 [support@fsight.ru](mailto:support@fsight.ru) или через [сервисы технической
 поддержки](https://www.fsight.ru/support/), которые доступны после регистрации на сайте.


	- в ОС Windows:


		- FP10Help_RU.msi.
		 Локальная справка на русском языке в виде набора chm-файлов для
		 настольного приложения;


		- FP10Help_EN.msi.
		 Локальная справка на английском языке в виде набора chm-файлов
		 для настольного приложения.


Для получения подробной информации о типах справочной системы обратитесь
 к разделу «[Выбор
 типа справочной системы](uinav.chm::/02_Navigator/UiNav_RepoConfig_Help.htm)».


## Установка справочной системы в ОС Linux


Для установки инсталлятора справочной системы в ОС Linux убедитесь,
 что инсталлятор FP10.9help_ru.run или FP10.9help_en.run
 расположен на компьютере с [установленным
 веб-приложением](../UiWebSetup/Web_Application_Installation.htm) и выполните команду:


sudo <путь
 до инсталлятора>/<наименование
 инсталлятора>


После выполнения действий справочная система будет доступна по адресу:


http://<веб-сервер>:<номер порта>/fp10.x/help/<двухбуквенный код языка>/help.htm


Где:


	- <веб-сервер>.
	 IP-адрес или DNS-имя сервера, на котором установлена [серверная
	 часть веб-приложения](../UiWebSetup/03_Setup_Web/Web_server_setup.htm);


	- <номер
	 порта>. Номер [порта](../01_SysReq/EnviromentRequirements.htm#ports_and_protocols),
	 по которому доступна серверная часть веб-приложения. По умолчанию
	 используется порт 8110;


	- <двухбуквенный
	 код языка>. Двухбуквенный код языка, в зависимости от использованного
	 инсталлятора:


		- ru. Русский язык;


		- en. Английский язык.


## Установка справочной системы в ОС Windows


Для автоматической установки справочной системы в ОС Windows при установке
 «Форсайт. Аналитическая платформа»
 необходимо, чтобы msi-инсталляторы справочной системы были расположены
 в одной папке с инсталлятором «Форсайт. Аналитическая платформа».
 Справочные системы устанавливаются после установки «Форсайт. Аналитическая платформа».
 В списке установленных программ для «Форсайт. Аналитическая платформа»
 и каждой справочной системы будут соответствовать разные записи (см. «Панель управления > Установка
 и удаление программ»). При удалении «Форсайт. Аналитическая платформа»
 справочные системы будут автоматически удалены.


Примечание.
 Если при установке «Форсайт. Аналитическая платформа»
 был выбран русский, украинский или казахский язык, будет автоматически
 установлена русская справка, в ином случае будет установлена английская
 справка. Если нужный инсталлятор справки не найден, будет установлена
 имеющаяся справка, независимо от выбранного языка.


Для отдельной установки справочной системы в ОС Windows запустите msi-инсталлятор
 FP10Help_RU.msi или FP10Help_EN.msi.


Примечание.
 Если в рамках одной транзакции запущенно несколько инсталляторов, то при
 возникновении ошибки хотя бы в одном из них будет произведен откат всех
 установок, в том числе и установки самого продукта «Форсайт. Аналитическая платформа».
 Для успешной установки «Форсайт. Аналитическая платформа»
 через удалённое соединение с помощью RemoteDesktop в папке с инсталлятором
 должны отсутствовать msi-файлы справочной системы. Подробно в разделе «[Особенности установки](Setup_AK_InstallFeature.htm)».
