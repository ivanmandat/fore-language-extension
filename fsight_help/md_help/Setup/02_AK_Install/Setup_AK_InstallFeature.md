# Особенности установки


## Установка через удаленное соединение


Для успешной установки «Форсайт. Аналитическая платформа»
 в ОС Windows через удалённое соединение с помощью RemoteDesktop в папке
 с инсталлятором должны отсутствовать msi-файлы справочной системы. Если
 msi-файлы справочной системы будут расположены на одном уровне с инсталлятором
 продукта «Форсайт. Аналитическая платформа»,
 то может произойти откат установки. Это связано с использованием технологии
 Chained MSI в Windows Installer 4.5, то есть при запуске msi-файлов после
 установки основного приложения.


### Решение


Для корректной удаленной установки «Форсайт. Аналитическая платформа»
 необходимо «Форсайт. Аналитическая платформа»
 и справочную систему устанавливать отдельно. Для отдельной установки «Форсайт. Аналитическая платформа»
 в папке, где расположен основной инсталлятор, должен отсутствовать msi-инсталлятор
 справочной системы. После установки «Форсайт. Аналитическая платформа»
 запустите msi-инсталлятор справочной системы.
