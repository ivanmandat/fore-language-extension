# Комплект поставки продукта
	 «Форсайт. Аналитическая платформа»


	Продукт «Форсайт. Аналитическая платформа»
	 включает дистрибутивы для 64-разрядных ОС Linux и Windows.


	Наименование дистрибутива может содержать номер поставляемой версии
	 и сборки в формате «_<номер версии>.<номер
	 сборки>». Например:


		- дистрибутив foresight-fp10.x-biserver_10.9.62-0~astra~1.7_amd64.deb
		 устанавливает BI-сервер продукта «Форсайт. Аналитическая платформа»
		 версии 10.9,
		 сборка номер 62, для Astra Linux;


		- дистрибутив FP10Full_x64_10.9.62.exe
		 устанавливает полный комплект «Форсайт. Аналитическая платформа»
		 версии 10.9,
		 сборка номер 62, для ОС Windows.


	Примечание.
	 Для получения дистрибутивов продукта «Форсайт. Аналитическая платформа»
	 обратитесь в техническую поддержку по адресу [support@fsight.ru](mailto:support@fsight.ru)
	 или через [сервисы
	 технической поддержки](https://www.fsight.ru/support/), которые доступны после регистрации на сайте.

## Reg-free COM registration


	Предусмотрены дистрибутивы продукта «Форсайт. Аналитическая платформа»,
	 которые используют технологию Reg-free COM registration. Данная технология
	 позволяет устанавливать «Форсайт. Аналитическая платформа»
	 без регистрации в реестре классов объектов и библиотек типов. При
	 выборе конфигурации рекомендуется использовать именно эти инсталляторы.


	Преимущества технологии:


		- при наличии пререквизитов можно производить запуск «Форсайт. Аналитическая платформа»
		 без его установки;


		- на одном рабочем месте можно использовать несколько сборок
		 одного релиза.


	Примечание.
	 В рамках одного релиза нельзя использовать сборки, установленные с
	 помощью дистрибутивов с Reg-free COM registration и дистрибутивов,
	 не использующих эту технологию.


## Дистрибутивы для ОС Windows


	Продукт «Форсайт. Аналитическая платформа»
	 включает дистрибутивы для ОС Windows. Для корректной работы платформы
	 разрядность продукта «Форсайт. Аналитическая платформа»
	 должна совпадать с разрядностью ОС Windows.


	В комплект поставки входят следующие дистрибутивы:


			 Наименование
			 Краткое описание
			 Использование Reg-free COM registration


			 FP10Full_x64.exe
			 Полный комплект настольного приложения «Форсайт. Аналитическая платформа»
			 включает отдельные модули администрирования, инструменты «Аналитические запросы (OLAP)»,
			 «Отчеты», модуль
			 среды разработки и конструктор моделей и алгоритмов.
			 ![](check16.png)


			 FP10Standard_x64.exe
			 Настольное приложение «Форсайт. Аналитическая платформа»
			 включает только средства просмотра экспресс-отчетов, регламентных
			 отчетов, анализа временных рядов, таблиц, справочников и некоторых
			 других объектов, а также средства запуска форм.
			 ![](check16.png)


			 FP10BIServerIIS_x64.exe
			 BI-сервер для веб-сервера IIS. Дистрибутив также включает
			 полный комплект настольного приложения «Форсайт. Аналитическая платформа»
			 и конфигурацию для IIS. Порядок установки описан в подразделе
			 «[Установка
			 BI-сервера в OC Windows](../UiWebSetup/Install_Web_on_Windows.htm#bi_install_windows)».
			Примечание.
			 Для BI-сервера доступны только дистрибутивы, использующие
			 технологию Reg-free COM registration.


			 ![](check16.png)


			 FP10WebServer.exe
			 Веб-приложение «Форсайт. Аналитическая платформа».
			 Более подробное описание можно найти в разделе «[Установка
			 и настройка серверной части веб-приложения](../UiWebSetup/03_Setup_Web/Web_server_setup.htm)».
			 ![](cancel16.png)


			 FP10DevTools_x64.exe
			 [Инструменты разработчика](DevTools.htm). Регистрирует
			 Interop-сборки платформы в глобальном кеше сборок (GAC).
			 ![](cancel16.png)


			 FP10JavaInterops_x64.exe
			 Interop-сборки для работы с «Форсайт. Аналитическая платформа»
			 из кода на языке Java. Также в системную переменную PATH прописывается
			 путь установки «Форсайт. Аналитическая платформа».
			 ![](cancel16.png)


			 FP10PythonInterops_x64.exe
			 Interop-сборки для работы с «Форсайт. Аналитическая платформа»
			 из кода на языке Python. Также в системную переменную PATH
			 прописывается путь установки «Форсайт. Аналитическая платформа».
			 ![](cancel16.png)


			 DBAWebServer.exe
			 Серверная часть [конструктора
			 бизнес-приложений](constructor.chm::/Intro/Designer_business_applications.htm). Для получения подробной
			 информации обратитесь к разделу «[Установка
			 серверной части конструктора бизнес-приложений на IIS](../Extensions/ASP_NET/Install_Backend_Designer_of_Business_Applications_ASP_NET.htm)».
			 ![](cancel16.png)


			 Foresight
			 Add-in for Excel_x64.exe
			 [Надстройка](../AddIn/AddIn_Setup.htm) для Microsoft
			 Excel. Является дополнением настольного приложения.
			 ![](cancel16.png)


			 Foresight
			 Add-in for Word_x64.exe
			 [Надстройка](../AddIn/AddIn_Setup.htm) для Microsoft

			 ![](cancel16.png)


			 Foresight
			 Add-in for PowerPoint_x64.exe
			 [Надстройка](../AddIn/AddIn_Setup.htm) для Microsoft

			 Архив для установки серверной части конструктора бизнес-приложений.
			 Используется при ручной установке и настройке Apache2 для доступа к ресурсам конструктора бизнес-приложений.

			 Для установки серверной части конструктора бизнес-приложений
			 рекомендуется использовать дистрибутив DBAWebServer.exe. Ручной
			 способ установки используется в качестве альтернативного варианта.


			Для получения архива обратитесь в техническую поддержку по
			 адресу [support@fsight.ru](mailto:support@fsight.ru)
			 или через [сервисы
			 технической поддержки](https://www.fsight.ru/support/), которые доступны после регистрации
			 на сайте.


	Условные обозначения:


	![](check16.png) - Reg-free COM registration
	 используется;


	![](cancel16.png) - Reg-free COM registration
	 не используется.


	Перед установкой каких-либо дистрибутивов с Interop-сборками должно
	 быть установлено настольное приложение «Форсайт. Аналитическая платформа».


	При установке Add-in-дистрибутивов дополнительно устанавливаются
	 инструменты разработчика, которые устанавливает дистрибутив FP10DevTools.exe.


	Примечание.
	 При использовании дистрибутивов, не использующих технологию Reg-free
	 COM registration, на один компьютер можно установить только один комплект
	 (FP10Full, FP10Standard, FP10BIServer)
	 одного релиза.


## Инсталляторы справочной системы и пререквизиты


	Комплект поставки продукта может включать:


		- инсталляторы справочной системы:


			- FP10.9help_ru.run. Локальная справка
			 на русском языке для веб-приложения в ОС Linux. Поиск в справочной
			 системе доступен только по ключевым словам в указателе;


			- FP10.9help_en.run. Локальная справка
			 на английском языке для веб-приложения в ОС Linux. Поиск в
			 справочной системе доступен только по ключевым словам в указателе;


	Примечание.
	 Для получения инсталлятора FP10.9help_ru.run обратитесь
	 в техническую поддержку по адресу [support@fsight.ru](mailto:support@fsight.ru)
	 или через [сервисы
	 технической поддержки](https://www.fsight.ru/support/), которые доступны после регистрации на сайте.


			- FP10Help_RU.msi.
			 Локальная справка на русском языке в виде набора chm-файлов
			 для настольного приложения в ОС Windows;


			- FP10Help_EN.msi.
			 Локальная справка на английском языке в виде набора chm-файлов
			 для настольного приложения в ОС Windows.


		- папку ISSetupPrerequisites, содержащую
		 [пререквизиты](../01_SysReq/ISSetupPrerequisites.htm)

	Для получения подробной информации об установке справочной системы
