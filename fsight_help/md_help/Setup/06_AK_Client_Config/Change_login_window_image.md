# Замена заставки при запуске приложения


Для замены заставки, отображаемой при запуске «Форсайт. Аналитическая платформа»
 через файл Cfg.exe:


	- Создайте пользовательское изображение splash.bmp.


	- Разместите данный файл в папке установки «Форсайт. Аналитическая платформа».


Также заставку можно заменить при запуске «Форсайт. Аналитическая платформа»
 из командной строки с использованием [ключа
 splash](../07_AK_Run_Additional/UseKeys.htm#splash).


Примечание.
 Замена заставки при запуске приложения доступна только в настольном приложении.
