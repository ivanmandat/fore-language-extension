# Поддержка работы с системами мониторинга


Настольное приложение, веб-приложение и BI-сервер на базе «Форсайт. Аналитическая платформа»
 поддерживают работу с системами мониторинга.


Список поддерживаемых систем мониторинга:


	- Zabbix.


Готовое решение на базе «Форсайт. Аналитическая платформа»
 может быть интегрировано со средством мониторинга для анализа доступности
 и состояния решения.


Примечание.
 Система мониторинга Zabbix является коммерческим продуктом и не входит
 в комплект поставки «Форсайт. Аналитическая платформа».


При работе с системами мониторинга доступны отслеживания следующие показатели:


	- использование ресурсов процессора сервера;


	- использование оперативной памяти сервера;


	- доступность развернутых приложений;


	- скорость обмена данными с жестким диском.


Установку и настройку систем мониторинга осуществляет системный администратор.


Система мониторинга разворачивается в среде окружения как на компьютерах
 пользователей, так и на серверах, и позволяет работать с «Форсайт. Аналитическая платформа»
 через клиентскую часть, предоставляемую разработчиком системы мониторинга.
 При планировании и развертывании приложения системный администратор окружения
 должен использовать руководства по администрированию, предоставляемые
 производителями соответствующих систем мониторинга.
