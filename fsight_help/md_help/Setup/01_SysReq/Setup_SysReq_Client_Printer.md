# Запуск дистрибутива с параметрами


Инсталляторы FP10Full.exe
 и FP10Standard.exe, FP10BIServer.exe и FP10WebServer.exe можно запустить
 с параметрами.


		 Параметр
		 Описание


		 [INSTALLSHORTCUTS](Setup_SysReq_Client_Printer.htm#install)
		 Определяет необходимость установки ярлыков продукта «Форсайт. Аналитическая платформа».


		 [L](Setup_SysReq_Client_Printer.htm#l)
		 Определяет язык мастера установки.


		 [PP_CHECK_CONTROL_SUMS](Setup_SysReq_Client_Printer.htm#check)
		 Определяет, будет ли при каждом запуске «Форсайт. Аналитическая платформа»
		 проверяться целостность исполняемых файлов и соответствие их содержимого
		 эталону.


		 [PP_CONTROL_SUMS_VISIBLE](Setup_SysReq_Client_Printer.htm#sum)
		 Определяет, будет ли при запуске установки «Форсайт. Аналитическая платформа»
		 отображаться опция «[Проверять
		 контрольные суммы при каждом запуске платформы](../02_AK_Install/Setup_AK_Install.htm#sum)».


		 [PPLICENSESHORTCUT](Setup_SysReq_Client_Printer.htm#shortcut)
		 Определяет необходимость установки ярлыка на менеджер лицензий.


		 [PPPRODUCTLANGUAGE](Setup_SysReq_Client_Printer.htm#language)
		 Определяет язык интерфейса продукта «Форсайт. Аналитическая платформа».


		 [PPUPDATEACTION](Setup_SysReq_Client_Printer.htm#action)
		 Определяет, будут ли при установке удалены все предыдущие версии.


		 [CLONE_WAIT](Setup_SysReq_Client_Printer.htm#wait)
		 Указывает процессу, запустившему установку «Форсайт. Аналитическая платформа»,
		 необходимость ожидать завершение установки.


## Синтаксис и описание параметров


[![](../Opened.gif)![](../Closed.gif)INSTALLSHORTCUTS](javascript:TextPopup(this))


	Определяет необходимость установки ярлыков продукта «Форсайт. Аналитическая платформа».
	 По умолчанию ярлыки устанавливаются, поэтому параметр используется
	 только для отключения установки ярлыков.


	Синтаксис:


	FP10Full.exe
	 /v"INSTALLSHORTCUTS="False""


	При использовании команды будет установлен продукт «Форсайт. Аналитическая платформа»,
	 ярлыки будут отсутствовать.


[![](../Opened.gif)![](../Closed.gif)L](javascript:TextPopup(this))


	Параметр определяет язык мастера установки продукта «Форсайт. Аналитическая платформа».
	 При использовании параметра не будет отображен диалог выбора языка
	 при установке «Форсайт. Аналитическая платформа».
	 Использование параметра не влияет на [язык интерфейса
	 платформы](GetStarted.chm::/Interface/Intro_Language.htm).


	Синтаксис:


	FP10Full.exe
	 /L1049


	Где 1049 - код языка, в данном случае русский.


	Мастер установки будет использовать русский язык, диалог выбора
	 языков не будет отображен.


	Коды языков, которые можно использовать, приведены [ниже](Setup_SysReq_Client_Printer.htm#language).


[![](../Opened.gif)![](../Closed.gif)PP_CHECK_CONTROL_SUMS](javascript:TextPopup(this))


	Определяет, будет ли при каждом запуске «Форсайт. Аналитическая платформа»
	 проверяться целостность исполняемых файлов и соответствие их содержимого
	 эталону. По умолчанию проверка целостности отключена.


	Синтаксис:


	FP10Full.exe
	 /v"PP_CHECK_CONTROL_SUMS=1"


	При использовании команды будет установлен продукт «Форсайт. Аналитическая платформа»
	 и включена [проверка
	 целостности платформы](admin.chm::/04_SecurityPolicy/Integrity_control.htm).


[![](../Opened.gif)![](../Closed.gif)PP_CONTROL_SUMS_VISIBLE](javascript:TextPopup(this))


	Определяет, будет ли при запуске установки «Форсайт. Аналитическая платформа»
	 отображаться опция «[Проверять
	 контрольные суммы при каждом запуске платформы](../02_AK_Install/Setup_AK_Install.htm#sum)».


	Синтаксис:


	FP10Full.exe
	 /v"PP_CONTROL_SUMS_VISIBLE=0"


	При использовании команды флажок «[Проверять
	 контрольные суммы при каждом запуске платформы](../02_AK_Install/Setup_AK_Install.htm#sum)» будет скрыт.
	 Изменения касаются выборочного варианта установки.


[![](../Opened.gif)![](../Closed.gif)PPLICENSESHORTCUT](javascript:TextPopup(this))


	Определяет необходимость установки ярлыка на менеджер лицензий.


	Синтаксис:


	FP10Full.exe
	 /v"PPLICENSESHORTCUT=0"


	При использовании команды будет установлен продукт «Форсайт. Аналитическая платформа»,
	 ярлык на менеджер лицензий будет отсутствовать.


[![](../Opened.gif)![](../Closed.gif)PPPRODUCTLANGUAGE](javascript:TextPopup(this))


	Параметр определяет [язык
	 интерфейса платформы](GetStarted.chm::/GetStarted/Interface/Intro_Language.htm).


	Примечание.
	 Заданная настройка перекрывается настройкой языка интерфейса продукта
	 «Форсайт. Аналитическая платформа»
	 для текущего пользователя.


	Использование параметра не влияет на язык мастера установки.


	Синтаксис:


	FP10Full.exe
	 /v"PPPRODUCTLANGUAGE=1049"


	Где 1049 - код языка, в данном случае русский.


	Можно использовать следующие коды:


		- 1049. Русский язык;


		- 1033. Английский язык;


		- 1034. Испанский язык;


		- 1040. Итальянский язык;


		- 1087. Казахский язык;


		- 2052. Китайский язык (упрощенный);


		- 1031. Немецкий язык;


		- 1058. Украинский язык;


		- 1036. Французский язык;


		- 2070. Португальский язык;


		- 1025. Арабский язык;


		- 14337. Арабский язык (Объединенные Арабские Эмираты).


[![](../Opened.gif)![](../Closed.gif)PPUPDATEACTION](javascript:TextPopup(this))


	Определяет, будут ли при установке удалены все предыдущие версии:


		- PPUPDATEACTION=1
		 - предыдущие установленные версии будут удалены. Значение по умолчанию.


		- PPUPDATEACTION <>
		 1 - предыдущие установленные версии будут оставлены.


	Синтаксис:


	FP10Full.exe
	 /v”PPUPDATEACTION=2”


	При использовании команды будет установлен продукт «Форсайт. Аналитическая платформа»,
	 предыдущие установленные версии будут оставлены.


[![](../Opened.gif)![](../Closed.gif)CLONE_WAIT](javascript:TextPopup(this))


	Указание параметра clone_wait
	 информирует процесс, запустивший установку, о необходимости ожидать
	 завершение установки.


	Примечание.
	 Параметр используется во всех инсталляторах «Форсайт. Аналитическая платформа».


	Синтаксис:


	FP10Full.exe
	 /clone_wait


	"Foresight Add-in for Excel.exe" /clone_wait


	При использовании команд последовательно будут установлены настольное
	 приложение «Форсайт. Аналитическая платформа»
	 и надстройка для работы с Microsoft Excel.


## Дополнительные возможности


Также при установке с параметрами доступны параметры, предусмотренные
 системой создания InstallShield.


Например, чтобы установить платформу в тихом режиме, запустите дистрибутив
 следующим образом:


FP10Full.exe
 /s /v/qn
