# Пререквизиты


В комплект поставки продукта «Форсайт. Аналитическая платформа»
 могут входить пререквизиты – программное обеспечение (ПО), которое потребуется
 для работы «Форсайт. Аналитическая платформа».
 Пререквизиты располагаются в папке ISSetupPrerequisites на одном уровне
 с инсталлятором продукта «Форсайт. Аналитическая платформа».
 Если пререквизиты не были установлены на рабочей станции, то они будут
 установлены автоматически до установки «Форсайт. Аналитическая платформа».


		 Наименование
		 Комментарии


		 Microsoft Visual C++ 2019 Redistributable Package
		 (x64) (vc_redist.x64.exe)
		 Для установки VCRedist 2019 на Windows Server 2012 R2 требуются
		 обновления [KB2975061](http://support.microsoft.com/KB/2975061)
		 и [KB2919355](http://support.microsoft.com/KB/2919355),
		 которые, при необходимости, скачиваются из интернета во время
		 установки.


Пререквизиты, которые не понадобятся при установке «Форсайт. Аналитическая платформа»,
 могут быть удалены из папки пререквизитов.


Если при установке «Форсайт. Аналитическая платформа»
 требуемый пререквизит не был найден, то установка «Форсайт. Аналитическая платформа»
 будет прервана.


Примечание.
 При установке демонстрационной версии
 продукта «Форсайт. Аналитическая платформа»
 пререквизиты не входят в комплект поставки, при необходимости во время
 установки будет предложено скачать необходимые пререквизиты с официальных
 сайтов.
