# Поддержка работы в виртуальных средах


Настольное и веб-приложение на базе «Форсайт. Аналитическая платформа»
 поддерживает установку и работу в различных виртуальных средах. Установка
 в виртуальных средах ничем не отличается от обычной установки [настольного
 приложения](../02_AK_Install/Setup_AK_Install.htm), [веб-приложения](../UiWebSetup/03_Setup_Web/Web_server_setup.htm)
 и [BI-сервера](../UiWebSetup/UiWebSetup_TitlePage.htm).


Список поддерживаемых виртуальных сред:


	- Hyper-V;


	- VMware;


	- VirtualBox;


	- Docker;


	- Citrix Virtual Apps and Desktops (ранее XenApp и XenDesktop);


	- Microsoft Application Virtualization (App-V);


	- zVIRT.


Список поддерживаемых ОС и СУБД при использовании виртуальных сред уточняйте
 на сайтах производителей.


При планировании и развертывании приложения администратор сервера виртуализации
 должен использовать руководства по администрированию, предоставляемые
 производителями соответствующих систем виртуализации.


При работе «Форсайт. Аналитическая платформа»
 под управлением виртуальных сред для активации должен использоваться сервер
 лицензий на базе [Guardant](../08_Licensing/Guardant/License_Server_Guardant.htm)
 или [RMS](../08_Licensing/RMS/LicenseServer.htm).


Виртуальные среды Citrix Virtual Apps and Desktops или Microsoft Azure
 RemoteApp могут использоваться для предоставления готовых решений на базе
 «Форсайт. Аналитическая платформа»
 в виде услуги (SaaS).


Виртуальная среда Microsoft Application Virtualization
 (App-V) может использоваться для подготовки и предоставления настольной
 версии «Форсайт. Аналитическая платформа»
 или готовых решений на базе «Форсайт. Аналитическая платформа»
 в виде виртуальных пакетов. При работе с Microsoft Application Virtualization
 используйте [дистрибутивы](../02_AK_Install/DistributionKit.htm#regfree)
 только на основе технологии Reg-free COM registration.


Виртуальный пакет разворачивается на компьютерах пользователей и позволяет
 работать с «Форсайт. Аналитическая платформа»
 через клиентскую часть виртуальной среды. Использование виртуальных пакетов
 облегчает процесс установки и настройки «Форсайт. Аналитическая платформа»,
 так как все действия производятся один раз на выделенном сервере виртуальной
 среды. Дальнейшее распространение, а также настройка прав доступа, осуществляется
 через Active Directory.
