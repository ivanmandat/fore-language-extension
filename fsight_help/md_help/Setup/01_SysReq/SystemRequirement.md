# Системные требования


В данном подразделе представлены системные требования, которые предъявляются
 к серверам СУБД, клиентским компьютерам и веб-серверам. Всё программное
 обеспечение может быть развёрнуто как на физических серверах, так и в
 [виртуальных средах](virtualization_platform.htm). Развёртывание
 приложений на базе «Форсайт. Аналитическая платформа»
 доступно в ОС Windows и Linux.


Совет. Для получения
 информации о настройках для лучшей производительности «Форсайт. Аналитическая платформа»
 обратитесь к статье «[Рекомендации
 по повышению производительности](Performance_Recommendations.htm)».


## Для работы с настольным приложением


Для работы с настольным приложением [формируется
 система](../GeneralInfo.htm) с использованием программного продукта «Форсайт. Аналитическая платформа»,
 которая предполагает наличие:


	- сервера баз данных, на котором установлена одна из поддерживаемых
	 СУБД;


	- сервера лицензий, используемого для активации продукта «Форсайт. Аналитическая платформа»
	 [сетевой
	 лицензией](../08_Licensing/Admin_Licensing_Variants.htm#network) и контроля количества одновременно работающих пользователей;


	- сервера приложений, используемого для выполнения отложенных
	 задач;


	- клиентских компьютеров, на которых установлено настольное приложение
	 «Форсайт. Аналитическая платформа».


Ознакомьтесь с требованиями к [серверу
 баз данных](../03_DB_Server_Config/Setup_DB_ServerSysReq.htm), [серверу
 лицензий](../08_Licensing/RMS/LicenseServer.htm#system_requirements), [серверу приложений](../UiWebSetup/AppServer.htm),
 [к клиентскому
 компьютеру](../03_DB_Server_Config/Setup_DB_ServerSysReq.htm#client) и с требованиями к [настройкам
 операционной системы](EnviromentRequirements.htm).


## Для работы с веб-приложением


Для работы с веб-приложением [формируется
 система](../GeneralInfo.htm) с использованием программного продукта «Форсайт. Аналитическая платформа»,
 которая предполагает наличие:


	- сервера баз данных, на котором установлена одна из поддерживаемых
	 СУБД;


	- сервера лицензий, используемого для активации продукта «Форсайт. Аналитическая платформа»
	 [сетевой
	 лицензией](../08_Licensing/Admin_Licensing_Variants.htm#network) и контроля количества одновременно работающих пользователей;


	- сервера приложений, используемого для выполнения отложенных
	 задач;


	- BI-сервера, на котором установлен веб-сервис «[BI-сервер](../UiWebSetup/03_Setup_Web/BI_Server_Settings.htm)»;


	- веб-сервера, на котором установлена серверная часть веб-приложения
	 для [ОС Linux](../UiWebSetup/Install_Web_on_Linux.htm)
	 или [ОС Windows](../UiWebSetup/Install_Web_on_Windows.htm);


	- клиентских компьютеров с доступом к сети, в которой расположен
	 веб-сервер.


Примечание.
 [BI-сервер](../UiWebSetup/UiWebSetup_TitlePage.htm), [сервер
 приложений](../UiWebSetup/AppServer.htm) и веб-приложение могут располагаться на одном физическом
 сервере. При выполнении ресурсоёмких задач располагайте сервер приложений
 и BI-сервер на различных физических серверах.


Для корректной установки веб-сервера, сервера приложений и работы веб-приложения
 «Форсайт. Аналитическая платформа»
 ознакомьтесь с [требованиями к настройкам
 операционной системы](EnviromentRequirements.htm).


Ознакомьтесь с системными требованиями к аппаратному и программному
 обеспечению для [сервера
 баз данных](../03_DB_Server_Config/Setup_DB_ServerSysReq.htm), [сервера
 лицензий](../08_Licensing/RMS/LicenseServer.htm#system_requirements), [сервера
 веб-приложения](../03_DB_Server_Config/Setup_DB_ServerSysReq.htm#web), [сервера приложений](../UiWebSetup/AppServer.htm),
 [BI-сервера](../03_DB_Server_Config/Setup_DB_ServerSysReq.htm#bi)
 и [клиентских
 компьютеров](../03_DB_Server_Config/Setup_DB_ServerSysReq.htm#client).
