# Приложения в составе продукта «Форсайт. Аналитическая платформа»


В состав продукта «Форсайт. Аналитическая платформа»
 входят следующие приложения:


	- Форсайт. Аналитическая платформа:


	-


		- Studio.exe. Используется
		 для входа в «Форсайт. Аналитическая платформа»
		 для полноценной работы с объектами репозитория. При запуске появится
		 стандартное окно регистрации с возможностью выбора репозитория;


		- PP.exe. Используется
		 для входа в «Форсайт. Аналитическая платформа»
		 для просмотра объектов репозитория. При запуске появится стандартное
		 окно регистрации с возможностью выбора репозитория;


	- Менеджер безопасности.
	 Открывает [менеджер
	 безопасности](admin.chm::/Admin_Title.htm)
	 в виде отдельного приложения. При разделении ролей администратора
	 информационной безопасности (АИБ) и прикладного администратора менеджер
	 безопасности используется АИБом для выполнения своих функций;


	- Менеджер лицензий. Запускает
	 [менеджер лицензий](../08_Licensing/RMS/Intro_LicMngr.htm);


	- Менеджер репозитория.
	 Запускает [менеджер репозитория](Setup_RepoMngr_Work_Main.htm);


	- Языковые параметры.
	 Открывает диалог настройки [языковых параметров](GetStarted.chm::/Interface/Intro_Language.htm)
	 продукта «Форсайт. Аналитическая платформа».


Примечание.
 Для ОС Linux в состав продукта «Форсайт. Аналитическая платформа»
 входит только менеджер репозитория.


	ОС
	 Linux ОС Windows

		 менеджер репозитория - консольное приложение RepoManager и скрипт
		 RepoManager_start.sh. Приложение запускается с помощью скрипта.

 При [выборочной
 установке](../02_AK_Install/Setup_AK_Install.htm#custom_setup) «Форсайт. Аналитическая платформа»

 при установке компонент будет доступен через [навигатор
 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm).


## Служебные приложения


В состав «Форсайт. Аналитическая платформа»
 входят следующие служебные приложения:


	ОС
	 Linux ОС Windows

		Служебные приложения, входящие в комплект поставки FPBIServer, в ОС
		 Linux:


			- Scheduler. Запускает
			 [планировщик
			 задач](uiappsrv.chm::/uiappsrv_purpose.htm)
			 для выполнения задач контейнера;


			- [PP.Util](Service_Applications/PP_Util.htm).
			 Используется для выполнения различных вспомогательных действий
			 при настройке BI-сервера. Приложение запускается с помощью
			 скрипта PP.Util_start.sh, который входит в состав продукта;


			- echoid. Используется
			 для [получения
			 цифрового отпечатка компьютера](../08_Licensing/RMS/LockingCode.htm) при активации продукта
			 на базе [RMS](../08_Licensing/RMS/Activation_Product_RMS.htm);


			- lsinit. Используется
			 для инициализации подсистемы лицензирования, если лицензии
			 [ограничены
			 по времени](../08_Licensing/Admin_Licensing_Variants.htm);


			- [UpdateManager](Service_Applications/UpdateManager.htm).
			 Используется для [установки
			 обновления](UpdManager.chm::/Admin_UpMBObj_DoUpdate.htm) объектов репозитория. Приложение
			 запускается с помощью скрипта UpdateManager_start.sh, который
			 входит в состав продукта;


			- axis2_http_server.
			 Тестовая версия BI-сервера, предназначенная для отладки и
			 диагностики проблем.


		Важно.
		 Запуск axis2_http_server.exe
		 в промышленной эксплуатации запрещен.


		Служебные приложения, входящие в комплект поставки FP10Full, FP10BIServer,
		 FP10Standard,
		 в ОС Windows:


			- Scheduler.exe.
			 Запускает [планировщик
			 задач](uiappsrv.chm::/uiappsrv_purpose.htm)
			 для выполнения задач контейнера;


			- Cfg.exe. Используется
			 для [обновления продукта «Форсайт. Аналитическая платформа»](../Update/Update.htm),
			 после чего производится вход в систему. Обновление производится,
			 если в источнике есть более поздняя сборка продукта «Форсайт. Аналитическая платформа»,
			 в противном случае осуществляется только вход в систему. После
			 обновления появляется стандартное окно регистрации;


		Примечание.
		 При установленной версии дистрибутива [FP10Full](../02_AK_Install/Setup_AK_Install.htm)
		 вход в систему будет осуществлён через файл запуска Studio.exe,
		 в противном случае вход будет осуществляться через файл PP.exe.


			- Tuner.exe. Используется
			 для [автоматической настройки
			 параметров репозитория](../Update/Tuner.htm);


			- [](Service_Applications/PP_Util.htm)[PP.Util.exe](Service_Applications/PP_Util.htm).
			 Используется для выполнения различных вспомогательных действий
			 при настройке BI-сервера;


			- echoid.exe.
			 Используется для [получения
			 цифрового отпечатка компьютера](../08_Licensing/RMS/LockingCode.htm) при активации продукта
			 на базе [RMS](../08_Licensing/RMS/Activation_Product_RMS.htm);


			- lsinit.exe.
			 Используется для инициализации подсистемы лицензирования,
			 если лицензии [ограничены
			 по времени](../08_Licensing/Admin_Licensing_Variants.htm);


			- axis2_http_server.exe.
			 Тестовая версия BI-сервера, предназначенная для отладки и
			 диагностики проблем.


		Важно.
		 Запуск axis2_http_server.exe
		 в промышленной эксплуатации запрещен.
