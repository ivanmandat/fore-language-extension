# Обновление продукта «Форсайт. Аналитическая платформа»


При выпуске новой версии продукта «Форсайт. Аналитическая платформа»
 требуется выполнить обновление текущей версии:


	- в ОС Linux обновление выполняется для BI-сервера и серверной
	 части веб-приложения. Для получения подробной информации обратитесь
	 к разделу «[Обновление веб-приложения
	 в ОС Linux](Update_Web_Application_Linux.htm)»;


	- в ОС Windows обновление выполняется для настольного приложения,
	 справочных систем, BI-сервера и серверной части веб-приложения. Допустимые
	 варианты обновления:


		- Обновление инсталлятором.
		 Каждый пользователь должен самостоятельно запустить инсталлятор
		 для обновления. Для получения подробной информации обратитесь
		 к разделу «[Обновление инсталлятором](ManualUpdate.htm)»;


		- Автоматическое обновление.
		 Администратор должен разместить файлы обновления в одном источнике,
		 с которого происходит одновременное обновление у нескольких пользователей.
		 Для получения подробной информации обратитесь к разделу «[Автоматическое
		 обновление](AutomaticUpdate.htm)».


Важно.
 Одновременное использование новой и прошлой версии продукта «Форсайт. Аналитическая платформа»
 невозможно, начиная с версии 9.4.
