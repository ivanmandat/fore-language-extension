# Особенности обновления


При [автоматическом обновлении](AutomaticUpdate.htm) в ОС
 Windows необходимо учитывать ряд особенностей:


	- при обновлении внутри одного релиза применяется значение флажка
	 «[Проверять
	 контрольные суммы при каждом запуске платформы](../02_AK_Install/Setup_AK_Install.htm)», указанное при
	 первой установке данного релиза. При обновлении с релиза на релиз
	 установленное значение флажка не наследуется. Для изменения значения
	 флажка при обновлении используйте атрибут [CheckControlSums](StructureCFG.htm#msi)
	 в файле [Cfgconfig.xml](StructureCFG.htm);


	- если при обновлении в рамках одного релиза в источнике обновления
	 лежат msi-инсталляторы установленных ранее справочных систем, то будет
	 произведено их обновление. Для просмотра информации об установленных
	 справочных системах откройте панель управления Windows и перейдите
	 в раздел «Программы > Программы
	 и компоненты»;


	- в ветке реестра [HKEY_CURRENT_USER\SOFTWARE\Foresight\Foresight Analytics Platform\10.0\AutoUpdate\Profiles\10.9] для ОС Windows
	 после обновления создаются следующие ключи:


		- CurrentVersion = 10.9.25098
		 - версия продукта «Форсайт. Аналитическая платформа»;


		- InstallDir = C:\Program Files\Foresight\
		 - папка, в которую установлен «Форсайт. Аналитическая платформа»;


		- Package = FP10Full.msi
		 - комплект установленного продукта «Форсайт. Аналитическая платформа».
		 Для дальнейших обновлений в источнике должен находиться указанный
		 комплект;


		- Dest = C:\Program Files\Foresight\Analytics Platform 10.x\Studio.exe - имя
		 исполняемого файла при запуске Cfg.exe (PP.exe для Standard, Studio.exe
		 для Full).
