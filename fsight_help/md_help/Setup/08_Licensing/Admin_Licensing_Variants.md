# О типе и времени действия лицензии


Для лицензии задается комбинация типа и времени действия.


В таблице приведены типы лицензий и сценарии их применения:


		 Тип лицензии
		 Сценарий использования


		 Сетевая
		 Лицензия для сервера лицензий на базе [Guardant](Guardant/License_Server_Guardant.htm)
		 или [RMS](RMS/LicenseServer.htm) в локальной сети.
		 Сервер лицензий контролирует количество одновременно работающих
		 пользователей, обеспечивает простоту и гибкость администрирования,
		 так как лицензия требуется только для размещения на сервере.

Сетевая лицензия может создаваться без ограничения
		 по количеству одновременно работающих пользователей.

Сетевая лицензия может разрешать получение временной
		 локальной лицензии для автономной работы с приложением вне сетевого
		 окружения в течение 30 дней. Например, для использования приложения
		 на ноутбуках в командировках.


		 Автономная
		 Лицензия на одно рабочее место. Лицензия устанавливается локально
		 на каждом компьютере, не требует подключения к сети.


В таблице приведены варианты времени действия
 лицензии и сценарии их применения:


		 Время действия лицензии
		 Сценарий использования


		 Конкретные даты


		 Временная лицензия разрешает работать с приложением в период,
		 заданный конкретными датами. Лицензия закончится по окончанию
		 этого периода, приложение не будет запускаться, независимо от
		 того, использовалось оно или нет.

Пример лицензии с конкретными датами: лицензия
		 разрешает работу с приложением с 1 июля 2017 по 15 августа 2017.


		 Пробный период


		 Временная лицензия разрешает работать с приложением в течение
		 заданного периода с момента первого запуска.

Пример лицензии с пробным периодом: лицензия
		 разрешает работу с приложением в течение 30 дней с момента первого
		 запуска. Если приложение было запущено первый раз 1 июля 2017,
		 то срок действия лицензии истечет 30 июля 2017.


		 Постоянная


		 У постоянной лицензии нет ограничения времени действия.

Для создания постоянной лицензии требуется [цифровой отпечаток компьютера](RMS/LockingCode.htm).
