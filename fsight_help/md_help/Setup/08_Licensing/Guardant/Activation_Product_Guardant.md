# Активация продукта на базе Guardant


Для активации продукта на базе программного обеспечения Guardant:


	- Установите и настройте [сервер
	 лицензий](License_Server_Guardant.htm).


	- Установите [сервис лицензирования](../Service/Licensing_Service.htm).


	- Активируйте лицензию, полученную в виде серийного номера, в
	 [мастере лицензий Guardant](License_Activation_Guardant.htm).


Примечание.
 Активация продукта на базе программного обеспечения Guardant производится
 только при использовании [сетевой
 лицензии](../Admin_Licensing_Variants.htm#network) для работы веб-приложения в ОС Linux и Windows.


Перед [открытием
 веб-приложения](../../UiWebSetup/03_Setup_Web/Opening_Web_Application.htm) запустите сервис лицензирования licman_guardant. В
 процессе работы с веб-приложением используйте [Guardant
 Control Center](License_Server_Guardant.htm#work) для просмотра захваченных единиц лицензирования и активных
 сессий пользователей.


Для получения подробной информации о программном обеспечении Guardant
 обратитесь к [официальной
 документации](https://dev.guardant.ru/).
