# Получение цифрового отпечатка компьютера


Цифровой отпечаток компьютера используется для создания [постоянной
 лицензии](../Admin_Licensing_Variants.htm#temporary) и последующей её активации на базе [RMS](Activation_Product_RMS.htm).


Для получения цифрового отпечатка компьютера используйте консольное
 приложение echoid в ОС Linux/echoid.exe в ОС Windows. Приложение входит
 в [состав продукта
 «Форсайт. Аналитическая платформа»](../../05_RepoMngr/Setup_RepoMngr_Main.htm)
 и расположено:


	- в папке установки [BI-сервера](../../UiWebSetup/Install_Web_on_Linux.htm)
	 /opt/foresight/fp10.x-biserver/bin
	 в ОС Linux;


	- в папке установки [BI-сервера](../../UiWebSetup/Install_Web_on_Windows.htm)
	 или [настольного
	 приложения](../../02_AK_Install/Setup_AK_Install.htm) C:\Program Files\Foresight\Analytics Platform 10.x\Tools в ОС Windows.


Запуск приложения зависит от используемой операционной системы и [типа лицензии](../Admin_Licensing_Variants.htm).


	ОС
	 Linux ОС Windows

		При
		 использовании сетевой лицензии запустите приложение от имени администратора:


		./echoid


		При использовании автономной лицензии запустите приложение от
		 имени служебного пользователя Apache2:


			- Debian-подобные дистрибутивы:


		sudo -H -u www-data
		 ./echoid


			- RedHat-подобные дистрибутивы:


		sudo -H -u apache
		 ./echoid


			- ALT Linux:


		sudo -H -u apache2
		 ./echoid


		При
		 использовании сетевой или автономной лицензии запустите приложение
		 от имени администратора:


		echoid.exe


После запуска приложения будет получен цифровой отпечаток компьютера
 в зашифрованном виде, например:


12345678912A34567B89123456789C1234567D8912
