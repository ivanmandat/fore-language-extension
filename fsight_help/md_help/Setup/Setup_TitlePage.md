# Установка и настройка продукта «Форсайт. Аналитическая платформа»


	Продукт «Форсайт. Аналитическая платформа»
	 содержит основные взаимосвязанные компоненты:


		- ![](UiWebSetup/web.png) [Веб-приложение](GetStarted.chm::/Desktop_and_web_application.htm);


		- ![](UiWebSetup/desktop.png) [Настольное
		 приложение](GetStarted.chm::/Desktop_and_web_application.htm);


		- ![](UiWebSetup/mobile.png) [Мобильное приложение](MobileApp.chm::/Mobile_app.htm).
		 Приложение представлено для ознакомительных целей.


	Использование программных продуктов на базе «Форсайт. Аналитическая платформа»
	 включает ряд требований и условий, которые необходимо учитывать при
	 планировании закупок программного обеспечения, а также при проектировании
	 прикладных систем.


## Планирование установки


	Перед установкой продукта «Форсайт. Аналитическая платформа»
	 выполните этапы планирования установки:


		- Выберите [конфигурацию системы](GeneralInfo.htm)
		 и ознакомьтесь с принципами их работы.


		- Проверьте соответствие [системным
		 требованиям](01_SysReq/SystemRequirement.htm), которые накладываются на серверы СУБД, веб-серверы
		 и клиентские компьютеры.


		- Выберите [тип
		 аутентификации](UiWebSetup/Authentication/Authentication.htm) пользователей.


		- Ознакомьтесь с процессом [активации](08_Licensing/Admin_Licensing.htm)
		 продуктов и списком [доступных
		 лицензий](08_Licensing/AboutFeatures.htm). Сервер лицензий является обязательной составляющей
		 любой системы, используется для активации программных продуктов
		 на базе «Форсайт. Аналитическая платформа»,
		 а также для контроля доступа к используемым инструментам.


		- Подготовьте [серверную
		 часть СУБД](03_DB_Server_Config/Setup_DB_Server_Config.htm).


	После подготовки системного окружения перейдите к непосредственной
	 установке «Форсайт. Аналитическая платформа».
	 Установка веб- и настольного приложений осуществляется с помощью инсталляторов,
	 входящих в [комплект
	 поставки продукта «Форсайт. Аналитическая платформа»](02_AK_Install/DistributionKit.htm#32_and_64).
	 Во время работы «Форсайт. Аналитическая платформа»
	 использует различное дополнительное программное обеспечение, которое
	 должно быть установлено в операционной системе. Наличие необходимого
	 программного обеспечения проверяется при установке «Форсайт. Аналитическая платформа».
	 При отсутствии какой-либо программы она будет установлена автоматически
	 из комплекта [пререквизитов](01_SysReq/ISSetupPrerequisites.htm).


	Примечание.
	 Рекомендуется использовать инсталляторы, поддерживающие технологию
	 Reg-free COM registration.

## Установка настольного приложения


	Для установки настольного приложения продукта «Форсайт. Аналитическая платформа»
	 в ОС Windows:


		- Ознакомьтесь с [особенностями
		 установки](02_AK_Install/Setup_AK_InstallFeature.htm) настольного приложения.


		- Убедитесь, что установка производится от имени пользователя,
		 обладающего правами локального администратора компьютера.


		- Запустите один из [инсталляторов](02_AK_Install/DistributionKit.htm#32_and_64),
		 входящих в [комплект
		 поставки продукта «Форсайт. Аналитическая платформа»](02_AK_Install/DistributionKit.htm):


			- FP10Full_x64.exe;


			- FP10Standard_x64.exe.


	Примечание.
	 Для корректной работы платформы разрядность продукта «Форсайт. Аналитическая платформа» должна
	 совпадать с разрядностью ОС Windows.


	Процесс установки настольного приложения
	 с помощью инсталлятора представлен в разделе «[Установка
	 настольного приложения продукта «Форсайт. Аналитическая платформа](02_AK_Install/Setup_AK_Install.htm)».


	При необходимости инсталляторы FP10Full_x64.exe и FP10Standard_x64.exe можно запустить
	 с параметрами. Для получения подробной информации обратитесь к разделу
	 «[Запуск дистрибутива
	 с параметрами](01_SysReq/Setup_SysReq_Client_Printer.htm)».


		- [Активируйте продукт
		 «Форсайт. Аналитическая платформа»](08_Licensing/Admin_Licensing.htm).


		- [Создайте
		 репозиторий метаданных](05_RepoMngr/Setup_RepoMngr_CreateRepo.htm).


		- [Создайте
		 служебного пользователя подсистемы безопасности](05_RepoMngr/Setup_RepoMngr_Service_User.htm).


	После выполнения действий будет установлено настольное приложение
	 «Форсайт. Аналитическая платформа».
	 Описание приложений, которые будут доступны после установки, представлено
	 в подразделе «[Приложения
	 в составе продукта «Форсайт. Аналитическая платформа»](05_RepoMngr/Setup_RepoMngr_Main.htm)».


	Также при установке настольного приложения может быть выполнена
	 настройка репозиториев и параметров автообновления. Для получения
	 подробной информации обратитесь к разделу «[Настройка
	 репозиториев и автообновления из файла](Update/Tuner.htm)».


	Настольное приложение может быть запущено из [командной
	 строки](07_AK_Run_Additional/UsingCommandLine.htm) с использованием определённого синтаксиса, например, для
	 автоматического подключения к репозиторию и/или открытия объекта.
	 При запуске платформы из командной строки возможно использование [ключей](07_AK_Run_Additional/UseKeys.htm). Запуск настольного
	 приложения из [командной
	 строки](07_AK_Run_Additional/UsingCommandLine.htm) с различными параметрами используется для формирования
	 и тестирования команд, которые в дальнейшем могут быть сохранены в
	 виде ярлыков и использоваться для автоматического запуска разрабатываемых
	 приложений, а также видоизменять сам процесс запуска.


	При возникновении проблем обратитесь к разделу «[Проблемы при запуске, входе
	 и проверке пользователя](FAQ.chm::/Errors/StartingError.htm)».


	Для начала работы с настольным приложением обратитесь к разделу
	 «[Начало
	 работы с веб-приложением и настольным приложением](GetStarted.chm::/GetStarted/Get_Started.htm)».


## Установка мобильного приложения


	Для установки мобильного приложения «Форсайт. Мобильный BI»:


		- Убедитесь, что операционная система планшета поддерживает:


			- iOS версии 11 и выше;


			- Android версии 7 и выше.


		- Скачайте мобильное приложение «Форсайт. Мобильный BI»
		 на сайте магазина App Store или Google Play.


	После выполнения действий на планшете будет установлено мобильное
	 приложение «Форсайт BI».


	Примечание.
	 Для работы с мобильным приложением должны быть установлены: [веб-](#web)
	 или [настольное приложение](#desktop) продукта «Форсайт. Аналитическая платформа»,
	 продукт «[Форсайт. Мобильная платформа](MobilePlatform.chm::/Installation/MobilePlatform_Setup.htm)[](../MobilePlatform/Installation/MobilePlatform_Setup.htm)».


	Для начала работы с мобильным приложение обратитесь к разделу «[Начало работы
	 с мобильным приложением](MobileApp.chm::/Get_started/Get_started_mobile.htm)».
