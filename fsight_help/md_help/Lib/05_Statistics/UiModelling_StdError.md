# Стандартная ошибка


Стандартная ошибка - это мера ошибки предсказанного значения y для отдельного значения x.


Уравнение для стандартной ошибки предсказанного y имеет следующий вид:


![](../img/StdError.gif)


Где:


-
x, y. Выборочные средние значения;


-
n. Размер выборки.
