# Частота


Частотой статистического распределения является массив ai длины m + 1, где m - длина массива интервалов ar2. В котором:


-
a1. Количество элементов из массива данных ar1 на интервале (-∞, ar2[1]];


-
ai. Количество элементов из массива данных ar1 на интервале (ar2[i - 1], ar2[i]];


-
am + 1. Количество элементов из массива данных ar1 на интервале (ar2[m, +∞].


Для каждого элемента из массива данных ar1 применяется бинарный поиск соответствующего ему интервала.


В случае, когда массив данных пустой – возвращается массив нулей. Когда массив интервалов пустой - возвращается массив из одного элемента равного количеству элементов в массиве данных.
