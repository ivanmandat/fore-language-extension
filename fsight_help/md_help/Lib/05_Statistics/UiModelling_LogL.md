# Функция правдоподобия


Метод максимального правдоподобия - способ построения оценки неизвестного параметра. Он состоит в том, что в качестве «наиболее правдоподобного» значения параметра берут значение ![](../img/teta.gif), максимизирующее вероятность получить при n опытах данную выборку X = (X1, …, Xn). Это значение параметра ![](../img/teta.gif) зависит от выборки.
