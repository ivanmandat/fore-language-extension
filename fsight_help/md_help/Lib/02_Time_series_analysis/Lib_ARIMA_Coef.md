# Оценка коэффициентов модели ARIMA


Модели класса [ARIMA](UiModelling_ARIMA.htm) являются нелинейными.
 Оценка коэффициентов для таких моделей может быть получена с помощью следующих
 методов:


	- [методы, основанные
	 на максимизации функции максимального правдоподобия](ARIMA_Estimation_MaxLikelihood.htm);


	- методы, основанные на минимизации суммы квадратов остатков:


	-


		- к[вазиньютоновский метод](Quasi_Newton.htm);


		- м[етод Левенберга-Марквардта](Levenberg-Marquardt.htm).


Примечание.
 Данные методы также могут использоваться для оценки коэффициентов [нелинейной
 регрессии](../01_Regression_models/UiModelling_NonLinearRegr.htm).
