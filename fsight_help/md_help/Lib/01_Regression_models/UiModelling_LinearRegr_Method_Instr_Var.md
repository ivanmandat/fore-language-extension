# Оценка методом инструментальных переменных


Метод инструментальных переменных является обобщением обычного [метода
 наименьших квадратов](UiModelling_LinearRegr_LSM.htm). Для оценки коэффициентов используется формула:


![](../img/2%20MNK.gif)


Где:


	- ![](../img/2%20MNK%20beta.gif). Оцениваемые коэффициенты;


	- ![](../img/2%20MNKY.gif). Зависимая переменная (моделируемая);


	- ![](../img/2%20MNKX.gif). Матрица независимых переменных
	 (факторы);


	- ![](../img/2%20MNKZ.gif). Матрица инструментальных переменных.
