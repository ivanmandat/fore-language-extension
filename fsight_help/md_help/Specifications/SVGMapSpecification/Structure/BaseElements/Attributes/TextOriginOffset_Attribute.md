# Атрибут TextOriginOffset


Атрибут TextOriginOffset служит
 для указания точки, в которой будет помещена метка. Координаты точки указываются
 в процентном значении относительно размера прямоугольной области, которая
 ограничивает фигуру. Отсчет ведется из левого нижнего угла области.


При конвертации в формат tbs значения данного атрибута будут установлены
 в качестве значений параметра TextOriginOffset
 для элементов карты. Если атрибут TextOriginOffset
 не задан, то для параметров будет указано значение по умолчанию - точка
 0.5,0.5.


## Синтаксис


<fs:Attribute Name="TextOriginOffset"
 Type="point">


    <fs:Data
 Id="RU-MOS" Value="0.3,0.3" />


    ...


</fs:Attribute>
