# Атрибут IdMapping


Атрибут IdMapping содержит значения
 идентификаторов, которые будут связаны с фигурами и по которым, в дальнейшем,
 будет осуществляться связь с элементами измерений. Значения, указываемые
 в дочерних элементах <fs:Data>
 атрибута IdMapping, должны соответствовать
 тем цифровым значениям, которые указаны в идентификаторах фигур. Пример:
 если Id фигуры «O100», то значение
 атрибута Value для элемента <fs:Data> должно быть «100».


При конвертации в формат tbs значения данного атрибута будут установлены
 в качестве значений параметра Id
 для элементов карты. Если атрибут IdMapping
 не задан, то при конвертации значения параметра Id
 будут сгенерированы автоматически.


## Синтаксис


<fs:Attribute Name="IdMapping"
 Type="int">


    <fs:Data
 Id="O1" Value="1" />


    <fs:Data
 Id="O2" Value="2" />


    ...


</fs:Attribute>
