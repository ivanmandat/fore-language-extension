# Атрибут ISO


Атрибут ISO содержит коды стран,
 соответствующие стандарту ISO 3166. Значения данного атрибута должны совпадать
 с идентификаторами, которые определены для фигур.


При конвертации в формат tbs значения данного атрибута будут установлены
 в качестве значений параметра ISO
 для элементов карты. Если атрибут ISO
 не задан, то при конвертации параметр будет отсутствовать полностью.


Примечание.
 Если для карты задан атрибут ISO,
 то связь с элементами измерений будет осуществляться по его значениям,
 иначе по значениям атрибута [IdMapping](IdMapping_Attribute.htm).


## Синтаксис


<fs:Attribute Name="ISO"
 Type="string">


    <fs:Data
 Id="RU-MOS" Value="RU-MOS" />


    ...


</fs:Attribute>
