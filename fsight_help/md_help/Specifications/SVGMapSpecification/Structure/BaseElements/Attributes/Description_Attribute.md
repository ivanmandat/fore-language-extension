# Атрибут Description


Атрибут Description содержит
 описание элементов карты.


При конвертации в формат tbs значения данного атрибута будут установлены
 в качестве значений параметра Description
 для элементов карты. Если атрибут Description
 не задан, то при конвертации параметр Description
 задан не будет.


## Синтаксис


<fs:Attribute Name="Description"
 Type="string">


    <fs:Data
 Id="RU-MOS" Value="Московская область..." />


    ...


</fs:Attribute>
