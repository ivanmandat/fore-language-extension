# Атрибут Name


Атрибут Name содержит наименования
 элементов карты.


При конвертации в формат tbs значения данного атрибута будут установлены
 в качестве значений параметра Name
 для элементов карты. Если атрибут Name
 не задан, то для слоев в качестве значения параметра будут установлены
 идентификаторы, а для фигур значения наименований заданы не будут.


## Синтаксис


<fs:Attribute Name="Name"
 Type="string">


    <fs:Data
 Id="Root" Value="Root" />


    <fs:Data
 Id="Regions" Value="Regions" />


    <fs:Data
 Id="RU-MOS" Value="Московская область" />


    ...


</fs:Attribute>
