# Элемент <svg>


Элемент <svg> является
 корневым элементом топоосновы. В корневом элементе указываются размеры
 топоосновы в пикселях, а также (опционально) атрибут proportionality,
 показывающий отношение высоты к ширине. Также здесь определяется пространство
 имен fs, в котором объявляются
 специфические атрибуты и элементы топоосновы. При указании дополнительных
 атрибутов и элементов топоосновы, отсутствующих в спецификации SVG, необходимо
 будет использовать префикс «fs:».
 Элемент <svg> является обязательным.


Примечание.
 В продукте «Форсайт. Аналитическая платформа»
 поддерживается пространство имён p5,
 используемое в Prognoz Platform.


## Синтаксис


<svg version="1.1" xmlns:fs="http://www.fsight.ru"
 width="853px" height="640px" proportionality="0.75"
 xmlns="http://www.w3.org/2000/svg">


    ...


</svg>


## Атрибуты элемента


		 Атрибут
		 Описание


		 version
		 Номер версии спецификации.


		 xmlns:fs="http://www.fsight.ru"
		 Пространство имен, необходимое для объявления специфических
		 элементов топоосновы.


		 width
		 Ширина топоосновы.


		 height
		 Высота топоосновы.


		 proportionality
		 Необязательный атрибут, указывающий отношение высоты и ширины.


Примечание.
 В таблице указаны атрибуты, которые определены спецификацией формата SVG
 версии 1.1, используемые топоосновой. Наименования всех атрибутов должны
 быть указаны строчными буквами.
