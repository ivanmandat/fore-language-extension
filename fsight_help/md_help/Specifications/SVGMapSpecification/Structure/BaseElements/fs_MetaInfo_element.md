# Элемент <fs:MetaInfo>


Элемент <fs:MetaInfo>
 является последним дочерним элементом элемента [<svg>](svg_element.htm)
 и используется для указания метаданных, относящихся к конкретной фигуре.
 В рамках элемента <fs:MetaInfo>
 могут располагаться следующие дочерние элементы:


	- [<fs:Geo>](fs_Geo_element.htm).
	 Элемент, содержащий информацию о привязке фигур к географическим координатам.


	- [<fs:Attributes>](fs_Attributes_element.htm).
	 Элемент, содержащий пользовательские метаданные, относящиеся к конкретным
	 фигурам.


Элемент <fs:MetaInfo>
 является необязательным. Если он отсутствует, то фигурам не будут сопоставлены
 какие-либо метаданные. Значения, необходимые при конвертации, будут сгенерированы.


## Синтаксис


<fs:MetaInfo>


    <fs:Geo>


        <fs:GeoItem
 Longitude="..." Latitude="..." X="..." Y="..."
 />


        ...


    </fs:Geo>


    <fs:Attributes>


        <fs:Attribute
 Name="ISO" Type="string">


            <fs:Data
 Id="RU-MOS" Value="RU-MOS" />


            ...


        </fs:Attribute>


        ...


    </fs:Attributes>


</fs:MetaInfo>


## Атрибуты элемента


Элемент <fs:MetaInfo>
 не имеет обязательных атрибутов.
