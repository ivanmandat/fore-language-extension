# Элемент <fs:Geo>


Элемент <fs:Geo> является
 дочерним элементом элемента [<fs:MetaInfo>](fs_MetaInfo_element.htm)
 и используется для указания информации о привязке к географическим координатам.


Элемент, в котором описывается привязка к географическим координатам,
 должен называться <fs:GeoItem>.
 Этот элемент является дочерним относительно <fs:Geo>
 и должен содержать следующие атрибуты: Latitude,
 Longitude, X,
 Y и ShapeId
 (опционально).


Примечание.
 Для задания координат используйте стандартную географическую систему координат
 [WGS84](http://www.unoosa.org/pdf/icg/2012/template/WGS_84.pdf),
 основанную на [проекции Меркатора](https://en.wikipedia.org/wiki/Mercator_projection).


Элемент <fs:Geo> необходим
 для последующей конвертации карты в формат [Google
 Maps](https://developers.google.com/maps/documentation/javascript/maptypes#MapCoordinates). Если он не указан, то конвертация в формат Google Maps будет
 невозможна.


Примечание.
 Значения атрибутов Longitude и
 Latitude влияют на совпадение
 границ накладываемых карт при подключении внешних картографических сервисов
 в веб-приложении. При возникновении проблем с геопривязкой обратитесь
 к статье «[Решение
 проблем с геопривязкой при подключении внешних сервисов](DataAnalysis.chm::/Plugins/Problems_geolocation.htm)».


## Синтаксис


<fs:Geo>


    <fs:GeoItem
 Longitude="22.1471203091967" Latitude="44.2498454806746"
 X="0" Y="0" />


    ...


</fs:Geo>


## Атрибуты элемента


		 Атрибут
		 Описание


		 Longitude
		 Географическая долгота точки (в градусах).


		 Latitude
		 Географическая широта точки (в градусах).


		 X
		 Горизонтальная координата точки на карте.


		 Y
		 Вертикальная координата точки на карте.


		 ShapeId
		 Опциональный атрибут соответствия описанной привязки конкретной
		 фигуре на карте, а не всей карте в целом.


Важно. Наименования
 всех атрибутов должны начинаться с заглавной буквы.
