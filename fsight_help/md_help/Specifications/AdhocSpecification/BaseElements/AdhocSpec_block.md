# Элемент <block>


Элемент <block> соответствует
 структурной единице аналитической панели - блоку.


Панель представляет собой набор блоков. Блоком может быть как визуальный
 компонент панели, так и его невизуальные элементы.


Низкоуровневые компоненты могут быть объединены в блоки для повышения
 удобства управления ими как единым объектом, из-за их функциональной близости
 и т.п.


## Синтаксис


<block> может содержать
 набор вложенных элементов <[prop](adhocspec_prop.htm)>,
 определяющих свойства блока.


<block
 name=""
 type=""
 key="">

  <prop>...</prop>

  ...

  <prop>...</prop>

</block>


## Атрибуты элемента


		 Атрибут
		 Описание


		 name
		 Имя блока.


		 key
		 Идентификатор блока.


		 type
		 Тип блока. Допустимые значения:


			- [Dashboard](../Props/block_dashboard.htm).
			 Аналитическая панель;


			- Grid. Элемент-контейнер
			 «Сетка»;


			- [Panel](../Props/adhocspec_panel.htm).
			 Элемент-контейнер «Панель»;


			- [TabPanel](../Props/block_tabpanel.htm).
			 Элемент-контейнер «Закладки»;


			- [Table](../Props/block_table.htm).
			 Визуализатор «Таблица»;


			- Chart. Визуализатор
			 «Диаграмма»;


			- Map. Визуализатор
			 «Карта»;


			- Indicator. Визуализатор
			 «Индикатор»;


			- Bubbles. Визуализатор
			 «Пузырьковая диаграмма»;


			- Report. Блок
			 «Регламентный отчёт»;


			- ExpressReport. Блок
			 «Экспресс-отчёт»;


			- TimeSeries. Блок
			 «Временные ряды»;


			- Control. Блок
			 «Элемент управления»;


			- [Label](../Props/block_label.htm). Блок
			 «Надпись»;


			- [Image](../Props/adhoc_image.htm). Блок
			 «Рисунок»;


			- [Frame](../Props/block_frame.htm). Блок
			 «HTML».
