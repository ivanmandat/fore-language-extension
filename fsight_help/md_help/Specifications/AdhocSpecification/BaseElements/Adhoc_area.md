# Элемент <area>


Элемент <area> предназначен для описания объектов, обладающих свойством панели с абсолютным позиционированием. В панели может быть размещён только один блок. Характеристики самой панели описываются коллекцией соответствующих свойств.


## Синтаксис


<area> может содержать несколько вложенных элементов <[prop](adhocspec_prop.htm)> и не более одного вложенного элемента <[block](AdhocSpec_block.htm)>.


<area key="" tag="">

  <prop>...</prop>

  ...
  <prop>...</prop>

  <block>...</block>

</area>


## Атрибуты элемента


 Атрибут
 Описание


 key
 Идентификатор элемента <area>.


 tag
 Имя свойства.
