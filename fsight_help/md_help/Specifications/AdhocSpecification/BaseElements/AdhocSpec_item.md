# Элемент <item>


Элемент <item> предназначен для обрамления отдельных (как правило, однотипных) элементов в коллекции. Служит для структурирования XML-представления аналитического отчёта.


## Синтаксис


<item> может содержать перечни элементов <[block](AdhocSpec_block.htm)>, <[prop](adhocspec_prop.htm)> или <[area](Adhoc_area.htm)>.


<item>

  <prop>...</prop>
  ...
  <prop>...</prop>
  <block>...</block>
  ...
  <block>...</block>
  <area>...</area>
  ...
  <area>...</area>
</item>
