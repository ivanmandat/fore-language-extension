# Конструктивные элементы аналитической панели


XML-представление аналитической панели может содержать следующие элементы:


-
[<kap>](AdhocSpec_kap.htm). Корневой элемент XML-представления;


-
[<block>](AdhocSpec_block.htm). Предназначен для описания блока аналитической панели;


-
[<prop>](adhocspec_prop.htm). Предназначен для описания свойств блока;


-
[<area>](Adhoc_area.htm). Предназначен для описания элемента-контейнера;


-
[<item>](AdhocSpec_item.htm). Предназначен для обрамления отдельных (как правило, однотипных) элементов в коллекции.


Ниже приведена упрощенная общая структура аналитической панели:


<?xml version="1.0" encoding="UTF-8"?>


<kap>


    <!-- Описание элемента-контейнера -->


    <area>


        <!-- Описание блока -->


        <block>


            <!-- Описание свойств блока -->


          <prop>


              <item>


                <prop />


                <prop />


              </item>


          </prop>


        </block>


    </area>


</kap>
