# Элемент <prop>


Элемент <prop> предназначен
 для описания характеристик и функциональных свойств блока.


Как правило, панель состоит из типовых блоков, которые описываются стандартным
 набором свойств. [Перечень свойств](../Props/adhocspec_props.htm)
 для таких типовых блоков приведён в соответствующем разделе спецификации.


## Синтаксис


По характеру своего содержимого элемент <prop>
 может быть примитивным или структурным.


Тег, описывающий примитивный элемент (как правило, поле), может содержать
 лишь конкретное значение, указываемое в атрибуте val.


<prop
 tag=""
 val=""
 />


Тег, описывающий структурный элемент, может иметь различные варианты
 синтаксиса:


	- Вариант 1.Коллекция
	 элементов <prop>.


<prop tag="">

  <prop>...</prop>

  ...

  <prop>...</prop>

</prop>


	- Вариант 2. Коллекция
	 элементов <prop> c одним
	 элементом <[area](Adhoc_area.htm)>:


<prop
 tag="">

  <prop>...</prop>

  ...

  <prop>...</prop>

  <area>...</area>

</prop>


	- Вариант 3. Коллекция
	 элементов <[area](Adhoc_area.htm)>:


<prop tag="">

  <area>...</area>

  ...

  <area>...</area>

</prop>


	- Вариант 4. Коллекция
	 элементов <[item](AdhocSpec_item.htm)>:


<prop tag="">

  <item>...</item>

  ...

  <item>...</item>

</prop>


## Атрибуты элемента


		 Атрибут
		 Описание


		 tag
		 Имя свойства.


		 val
		 Значение свойства.
