# Элемент <kap>


Элемент <kap> является корневым элементом XML-представления аналитической панели. Всё содержимое панели размещается между начальным и конечным тегом данного элемента.


## Синтаксис


<kap> должен содержать не менее одного внутреннего элемента <[block](AdhocSpec_block.htm)>. Среди элементов <block> должен содержаться только один элемент типа Dashboard.


<kap version="">

  <block>...</block>

</kap>


## Атрибуты элемента


 Атрибут
 Описание


 version
 Номер версии спецификации.
