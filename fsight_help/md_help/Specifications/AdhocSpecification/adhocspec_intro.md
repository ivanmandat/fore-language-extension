# Назначение и основные возможности


Аналитическая панель - один из способов представления данных системы
 поддержки принятия решения, позволяющий отображать сводную аналитическую
 информацию из различных источников посредством визуальных компонентов.


Данная спецификация описывает структуру XML-представления аналитической
 панели, которое может использоваться в качестве внешнего формата для блока
 «[Аналитические
 панели](UiAdhoc.chm::/uiadhoc_purpose.htm)».


Для сохранения аналитической панели в XML-представлении выполните пункт
 контекстного меню «Сохранить в файл»
 в навигаторе объектов.


В разделе «[Конструктивные
 элементы панели](BaseElements/adhoc_baseelements.htm)» приведено описание базовых элементов, из которых
 состоит аналитическая панель. Кратко указано назначение элемента, синтаксис
 его описания в XML-представлении (со схематичным примером), приведён перечень
 атрибутов элемента.


В разделе «[Свойства типовых блоков
 панели](Props/adhocspec_props.htm)» приведены свойства типовых объектов, из которых строится панель.
 Свойства объектов раскрывают его функциональные характеристики. Типы данных,
 используемые при описании свойств объектов аналитической панели, приведены
 в разделе «[Типы данных](Props/AdhocSpec_DataTypes.htm)».


В разделе «[Примеры](Samples/adhoc_samples.htm)» собраны
 примеры XML-описания различных объектов аналитической панели.
