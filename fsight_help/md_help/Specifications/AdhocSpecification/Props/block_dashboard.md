# Аналитическая панель


Блок «Аналитическая панель»
 - обязательный базовый блок, используемый для отображения информации.


		 Свойство
		 Тип данных
		 Описание


		 name
		 Строка
		 Название блока.


		 [background](AdhocSpec_block.htm#background)


		 Однотипное структурное свойство
		 Заливка блока.


		 layouts
		 Коллекция элементов <[area](../BaseElements/Adhoc_area.htm)>
		 Набор панелей, в которых расположены дочерние блоки.


		 defaultBlockStyle
		 Структурное свойство
		 Настройки дочерних блоков, используемые по умолчанию.


		 blockPanel


		 Структурное свойство


		 [background](AdhocSpec_block.htm#background)


		 Однотипное структурное свойство
		 Заливка блока.


		 [decor](AdhocSpec_block.htm#decor)


		 Однотипное структурное свойство
		 Оформление блока.


		 [blockTitle](AdhocSpec_block.htm#blocktitle)


		 Однотипное структурное свойство
		 Заголовок блока.
		Примечание.
		 Атрибут text недоступен.


		 expandable


		 0|1
		 Определяет, доступна ли возможность разворачивать блок на весь
		 экран.


		 [layout](AdhocSpec_block.htm#layout)


		 Однотипное структурное свойство
		 Расположение блока в контейнере.


Примечание.
 При описании настроек дочерних блоков могут указываться не все атрибуты
 соответствующего структурного свойства, а лишь его отдельные элементы.


Блок является аналитической панелью, если описывающий его элемент <[block](../BaseElements/AdhocSpec_block.htm)>
 имеет тип Dashboard.


XML-описание блока приведёно в примере «[Аналитическая
 панель](../Samples/01_Adhoc.htm)».
