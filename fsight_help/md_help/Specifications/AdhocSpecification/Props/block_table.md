# Таблица


Блок «Таблица» предназначен
 для отображения данных в табличной форме на базе источника данных из репозитория.


		 Свойство
		 Тип данных
		 Описание


		 name
		 Строка
		 Название блока.


		 expandable
		 0|1
		 Определяет, доступна ли возможность разворачивать блок на весь
		 экран.


		 [background](AdhocSpec_block.htm#background)


		 Однотипное структурное свойство
		 Заливка блока.


		 [dataSource](AdhocSpec_block.htm#datasource)


		 Однотипное структурное свойство
		 Источник данных.


		 [layout](AdhocSpec_block.htm#layout)


		 Однотипное структурное свойство
		 Расположение блока в контейнере.


		 [decor](AdhocSpec_block.htm#decor)


		 Однотипное структурное свойство
		 Оформление блока.


		 [selectionBus](AdhocSpec_block.htm#selectionbus)


		 Однотипное структурное свойство
		 Параметры шины отметок.


Блок является таблицей, если описывающий его элемент <[block](../BaseElements/AdhocSpec_block.htm)>
 имеет тип Table.


XML-описание блока приведёно в примере «[Закладки.
 Панель. Таблица](../Samples/03_TabPanelTable.htm)».
