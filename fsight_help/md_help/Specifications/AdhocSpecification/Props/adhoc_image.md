# Рисунок


Блок «Рисунок» предназначен
 для отображения статической картинки, загруженной из файла.


		 Свойство
		 Тип данных
		 Описание


		 name
		 Строка
		 Название блока.


		 expandable
		 0|1
		 Определяет, доступна ли возможность разворачивать блок на весь
		 экран.


		 fillImage
		 0|1
		 Определяет, будет ли изображение растягиваться/сжиматься до
		 размеров блока, в котором он выводится.


		 imageData
		 base64
		 Закодированная информация об изображении.


		 [background](AdhocSpec_block.htm#background)


		 Однотипное структурное свойство
		 Заливка блока.


		 [blockTitle](AdhocSpec_block.htm#blocktitle)


		 Однотипное структурное свойство
		 Заголовок блока.


		 [layout](AdhocSpec_block.htm#layout)


		 Однотипное структурное свойство
		 Расположение блока в контейнере.


		 [decor](AdhocSpec_block.htm#decor)


		 Однотипное структурное свойство
		 Оформление блока.


Блок является рисунком, если описывающий его элемент <[block](../BaseElements/AdhocSpec_block.htm)>
 имеет тип Image.


XML-описание блока приведёно в примере «[Информационные
 блоки без источника данных](../Samples/04_BlockOther.htm)».
