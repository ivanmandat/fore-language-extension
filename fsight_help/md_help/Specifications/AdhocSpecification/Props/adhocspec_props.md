# Свойства типовых блоков панели


Аналитическая панель состоит из следующих типовых блоков:


	- [Аналитическая
	 панель](block_dashboard.htm). Обязательный базовый блок панели, используемый
	 для отображения информации;


	- [Закладки](block_tabpanel.htm).
	 Многостраничный элемент-контейнер;


	- [Панель](adhocspec_panel.htm).
	 Элемент-контейнер, который использует абсолютное позиционирование;


	- [Таблица](block_table.htm).
	 Отображает данные в табличной форме на базе источника данных из репозитория;


	- [Рисунок](adhoc_image.htm).
	 Блок, предназначенный для отображения графической информации;


	- [Надпись](block_label.htm).
	 Отображает заданные текстовые данные;


	- [HTML](block_frame.htm).
	 Отображает содержимое, расположенное по указанному URL-адресу.


В разделе приведено описание уникальных и [однотипных
 свойств](AdhocSpec_block.htm) типовых блоков. Типы данных, используемые при описании свойств,
 приведены в разделе «[Типы данных](../adhocspec_intro.htm)».
