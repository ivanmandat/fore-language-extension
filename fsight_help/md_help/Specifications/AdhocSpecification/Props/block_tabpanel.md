# Закладки


Блок «Закладки» - многостраничный
 элемент-контейнер, на страницах которого можно располагать информационные
 блоки, элементы управления и т.д. При работе отображается содержимое какой-либо
 одной страницы. Для переключения между страницами используются вкладки.


		 Свойство
		 Тип данных
		 Описание


		 name
		 Строка
		 Название блока.


		 expandable
		 0|1
		 Определяет, доступна ли возможность разворачивать блок на весь
		 экран.


		 selectedIndex


		 Число
		 Порядковый номер отображаемой (текущей) вкладки. Нумерация
		 вкладок сквозная и начинается с нуля.


		 tabItems


		 Коллекция элементов <[item](../BaseElements/AdhocSpec_item.htm)>
		 Каждый элемент в коллекции соответствует одной вкладке и должен
		 содержать вложенный элемент <[block](../BaseElements/AdhocSpec_block.htm)>
		 типа Panel.


		 [background](AdhocSpec_block.htm#background)


		 Однотипное структурное свойство
		 Заливка блока.


		 [layout](AdhocSpec_block.htm#layout)


		 Однотипное структурное свойство
		 Расположение блока в контейнере.


		 [decor](AdhocSpec_block.htm#decor)


		 Однотипное структурное свойство
		 Оформление блока.


Блок является многостраничным контейнером, если описывающий его элемент
 <[block](../BaseElements/AdhocSpec_block.htm)>
 имеет тип TabPanel.


XML-описание блока приведёно в примере «[Закладки.
 Панель. Таблица](../Samples/03_TabPanelTable.htm)».
