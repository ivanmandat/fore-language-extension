# Панель


Блок «Панель» - элемент-контейнер,
 позволяющий объединить в группу несколько блоков аналитической панели.


		 Свойство
		 Тип данных
		 Описание


		 [background](AdhocSpec_block.htm#background)


		 Однотипное структурное свойство
		 Заливка блока.


		 [decor](AdhocSpec_block.htm#decor)


		 Однотипное структурное свойство
		 Оформление блока.


		 layouts
		 Коллекция элементов <[area](../BaseElements/Adhoc_area.htm)>
		 Набор панелей, в которых расположены блоки (элемент <[block](../BaseElements/AdhocSpec_block.htm)>).


Блок является элементом-контейнером, если описывающий его элемент <[block](../BaseElements/AdhocSpec_block.htm)>
 имеет тип Panel.


XML-описание блока приведёно в примере «[Закладки.
 Панель. Таблица](../Samples/03_TabPanelTable.htm)».
