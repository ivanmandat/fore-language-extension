# Типы данных


Типы данных, используемые при описании свойств блоков аналитической панели:


 Тип данных
 Описание


 base64
 Бинарные данные в виде текстовой строки формата base64.


 Идентификатор
 Строка, обеспечивающая уникальную идентификацию объекта в рамках аналитической панели.


 Ключ
 Ключ объекта репозитория.


 Процент
 Целочисленное значение от 0 до 100, интерпретируемое как процент.


 Размер
 Количество пикселей.


 Ссылка
 Строка, начинающаяся с префикса «http://» и интерпретируемая как сетевой адрес ресурса.


 Строка
 Последовательность из нуля и более символов.


 Цвет
 Цвет в терминах каналов красного, зеленого и синего (RGB). Указываются три пары шестнадцатеричных цифр после символа #, например «#808080».


 Число
 Численный тип данных.


 Шрифт
 Перечислимый тип данных, значениями которого являются системные названия шрифтов.


 0|1
 Логический тип данных. 0 - ложь; 1 - истина.
