## Описание


Компонент MapBox предназначен
 для отображения карт.


## Работа с компонентом


Для работы компонента необходимо в свойстве [Source](Map.chm::/Interface/IMapBox/IMapBox.Source.htm)
 задать источник данных, предоставляющий карту. В качестве значения свойства
 указывается компоненты [UiMap](UiMap.htm) или [UiErAnalyzer](UiErAnalyzer.htm).

		 ![](../../Sub_Image.gif)

		 [ScreenToClient](ModForms.chm::/Interface/IControl/IControl.ScreenToClient.htm)
