# Расширение функциональности приложения


Функциональность создаваемых приложений может быть расширена за счёт
 использования дополнительных возможностей, которые предоставляет «Форсайт. Аналитическая платформа».
 К таким возможностям относятся:


	- [Создание](../04_NavigatorSetting/Classes_Object/UiNav_Classes_Object.htm)
	 собственных классов объектов и экземпляров объектов этих классов.
	 Алгоритм работы таких объектов полностью реализуется в прикладном
	 коде.


	- Настройка и генерация [собственных
	 событий](../04_NavigatorSetting/Classes_Object/Classes_Events.htm). Генерация таких событий производится из прикладного кода
	 или с помощью задач в [планировщике задач](UiAppSrv.chm::/UiAppSrv_purpose.htm).
	 Для отслеживания событий также используется планировщик задач.


	- Использование [глобальных
	 обработчиков](../04_NavigatorSetting/EventHandler.htm) событий для отчётов, обновлений, а также обработка
	 ошибок, которые могут возникнуть при работе прикладного кода.
