# Отмена изменений объектов


Для отмены изменений в локальных [файлах](VSC_Add_Object.htm)
 объектов:


	- Выделите изменённые объекты в [навигаторе
	 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm), навигаторе проекта или на вкладке «Изменения» на панели «[Ожидающие изменения](VSC_Introduction.htm#control)»
	 в среде разработки.


	- Выполните одно из действий:


		- нажмите кнопку ![](../03_Windows_of_Development_Environment/Button/VCS_Undo_Button.gif) «Отменить
		 изменения» на ленте инструментов навигатора объектов или
		 в меню навигатора проекта;


		- выполните команду ![](../03_Windows_of_Development_Environment/Button/VCS_Undo_Button.gif) «Отменить ожидающие изменения»
		 в контекстном меню объектов.


После выполнения действий будут отменены изменения в соответствующих
 локальных файлах объектов.
