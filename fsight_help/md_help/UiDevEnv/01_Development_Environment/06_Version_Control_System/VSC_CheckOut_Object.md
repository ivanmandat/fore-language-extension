# Извлечение объекта для редактирования


Для изменения объекта извлеките версию объекта, если репозиторий [подключен](../../04_NavigatorSetting/VCS/Add_In_VCS.htm)
 к системе управления версиями Team Foundation Server:


	- Выделите заблокированный объект в [навигаторе
	 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm) или навигаторе
	 проекта в среде разработки.


Примечание.
 Пиктограмма заблокированного объекта отмечается соответствующей пиктограммой.
 Например, пиктограмма заблокированного модуля принимает вид ![](blocked.png).


	- Выполните одно из действий:


		- нажмите кнопку ![](../03_Windows_of_Development_Environment/Button/VCS_CheckOut_Button.gif)
		 «Извлечь» на ленте инструментов
		 навигатора объектов или в меню навигатора проекта среды разработки;


		- выполните команду ![](../03_Windows_of_Development_Environment/Button/VCS_CheckOut_Button.gif)
		 «Извлечь для редактирования»
		 в контекстном меню объектов.


После выполнения действий объект будет доступен для редактирования и
 заблокирован для других разработчиков. Вся дальнейшая работа будет осуществляться
 с локальной копией файлов на диске.


Если установлен флажок «[Автоматически
 блокировать объекты для редактирования](UiNav.chm::/02_Navigator/UiNav_BlockObjects.htm)», то
 при извлечении объекта будет происходить автоматическая блокировка и извлечение.
 Если флажок снят, то будет выдаваться соответствующее подтверждение на
 блокировку. Объекты, извлеченные одним пользователем, остаются доступными
 для извлечения другими пользователями.
