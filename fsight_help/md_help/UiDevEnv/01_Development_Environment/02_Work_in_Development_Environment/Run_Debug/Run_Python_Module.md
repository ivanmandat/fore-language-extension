# Запуск Python-модулей


Для Python-модулей доступен запуск без отладки. Логика выполнения кода
 (поиск точки входа, порядок запуска процедур) соответствует той логике,
 которая заложена в языке Python. Если в коде встречается функция print или sys.stdout.write,
 то значение из параметров будет выведено в консоль среды разработки «Форсайт. Аналитическая платформа».


Перед запуском кода осуществляется его компиляция. Если во время компиляции
 возникают какие-либо ошибки, то информация о них будет выведена в окно
 сообщений компилятора.
