# Запуск модулей


Модули, созданные вне сборки,
 могут быть откомпилированы и запущены как отдельная сборка репозитория.


Примечание.
 Возможность компиляции модулей вне сборок определяется в [обработке
 событий](../../../04_NavigatorSetting/EventHandler.htm).


При запуске модуля осуществляется синтаксическая проверка кода. Если
 синтаксические ошибки отсутствуют, то осуществляется компиляция модуля
 и в глобальном пространстве имен модуля будет осуществлён поиск [точки
 входа](../DevEnv_Object/entry_point.htm) - процедуры Main. Если процедура найдена, то она запускается
 на выполнение. Если процедура Main отсутствует или её сигнатура не соответствует
 требуемой, то будет сгенерирована одна из следующих исключительных ситуаций:


	- В сборке нет процедуры Main;


	- Ожидается процедура Main без параметров.
