# Запуск форм


Формы, созданные вне сборки,
 могут быть откомпилированы и запущены как отдельная сборка репозитория.


Примечание.
 Возможность компиляции форм вне сборок определяется в параметрах разработки
 приложений.


При запуске формы осуществляется синтаксическая проверка кода. Если
 синтаксические ошибки отсутствуют, то форма будет откомпилирована и на
 экран будет выведено окно формы.


Используя события формы, можно управлять процессом запуска, инициализировать
 какие-либо объекты, запускать другие формы и т.д.
