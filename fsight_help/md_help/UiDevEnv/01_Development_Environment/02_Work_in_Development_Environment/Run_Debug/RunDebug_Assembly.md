# Запуск сборок


При запуске сборки осуществляется
 синтаксическая проверка кода всех модулей/форм, входящих в данную сборку.
 Если синтаксические ошибки отсутствуют, то в глобальном пространстве имен
 сборки будет осуществлён поиск и запуск [точки
 входа](../DevEnv_Object/entry_point.htm) - процедуры Main. Данная процедура может содержаться в любом
 модуле/форме сборки. В процедуре может производится инициализация каких-либо
 объектов, запуск форм сборки и любые другие действия.
