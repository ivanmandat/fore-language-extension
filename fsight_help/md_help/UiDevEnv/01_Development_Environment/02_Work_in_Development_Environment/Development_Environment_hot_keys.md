# Горячие клавиши


Для ускорения работы в среде разработки наиболее часто выполняемые действия
 продублированы с помощью сочетаний «горячих клавиш». Большая часть сочетаний
 включена в описание действий в главном меню, в контекстном меню либо во
 всплывающих подсказках. Список всех сочетаний «горячих клавиш», используемых
 в среде разработки, представлен ниже.


	Веб-приложение Настольное приложение


				 Сочетание клавиш
				 Краткое описание


				 CTRL+S
				 Сохранение изменений текущего открытого объекта. Доступно
				 также через команду «Файл
				 > Сохранить <наименование
				 объекта>»
				 главного меню.


				 CTRL+Z
				 Отмена последнего внесенного изменения. Доступно также
				 через команду «Правка
				 > Отменить» главного меню.


				 CTRL+Y
				 Повтор отмененного изменения. Доступно также через
				 команду «Правка >
				 Повторить» главного меню.


				 CTRL+C; CTRL+INSERT
				 Копирование кодав буфер обмена. Доступно также через команду
				 «Правка > Копировать»
				 главного меню.


				 CTRL+X; SHIFT+DELETE
				 Удаление кода
				 в буфер обмена. Доступно также через команду «Правка
				 > Вырезать» главного меню.


				 CTRL+V; SHIFT+INSERT
				 Вставка содержимого буфера обмена. Доступно также через
				 команду «Правка >
				 Вставить» главного меню.


				 CTRL+A
				 Выделение всего текста макроса. Доступно
				 также через команду «Правка
				 > Выделить всё» главного меню.


				 CTRL+F; CTRL+SHIFT+F
				 Поиск по коду текущего модуля.
				 Доступно также через команду «Правка
				 > Найти» главного меню.


				 CTRL+H
				 Поиск по коду с заменой. Доступно также через команду
				 «Правка > Заменить»
				 главного меню.


				 CTRL+SHIFT+1/9
				 Снятие/установка закладки с указанным номером.


				 CTRL+1/9
				 Переход к закладке с указанным номером.


				 CTRL+G
				 Переход к указанной строке в окне макроса. Доступно
				 также через команду «Правка
				 > Перейти к строке» главного меню.


				 F1
				 Вызов справки по доступным командам в среде разработки..


				 CTRL+F2
				 Остановка выполнения текущего модуля/веб-формы.
				 Доступно также через команду «Отладка
				 > Прервать выполнение кода» главного меню.


				 F5
				 Установка/снятие точки останова в текущей строке. Доступно
				 также через команду «Установить/Снять
				 точку останова» главного меню.


				 F7
				 Выполнение кода по одному оператору с заходом в вызываемые
				 методы. Доступно также через команду «Отладка
				 > Выполнить пошагово» главного меню.

Доступно при пошаговом выполнении программы.


				 CTRL+F7
				 Вычисление выражения, выделенного в окне макроса либо
				 введённого пользователем. Доступно также через команду
				 «Вычислить выражение»
				 контекстного меню или «Отладка
				 > Вычислить выражение» главного меню.

Доступно при пошаговом выполнении программы.


				 F8
				 Выполнение следующей строки кода без захода в вызываемые
				 методы. Доступно также через команду «Отладка
				 > Перешагнуть» главного меню.

Доступно при пошаговом выполнении программы.


				 F9
				 Запуск на выполнение модуля/веб-формы в режиме отладки.
				 Доступно также через команду «Отладка
				 > Выполнить» главного меню.


				 CTRL+SHIFT+F9
				 Запуск на выполнение модуля/python-модуля/веб-формы
				 без отладки. Доступно также через команду «Отладка
				 > Выполнить без отладки» главного меню.


				 SHIFT+F7


				 Переход из окна макроса к веб-форме. Актуально только
				 для веб-формы. Для перехода от веб-формы к окну макроса
				 используется горячая клавиша F7.


				 SHIFT+F11
				 Продолжение выполнения макроса от текущей позиции до
				 следующей точки останова либо до конца программы. Доступно
				 также через команду «Отладка
				 > Выполнять до возврата» главного меню.

Доступно при пошаговом выполнении программы.


				 F12
				 Переход к описанию выделенного элемента кода. Для системных
				 классов/процедур/функций будет осуществлен переход в [инспектор
				 классов](../03_Windows_of_Development_Environment/Inspector_class.htm) к соответствующему элементу и его описанию.
				 Для пользовательских элементов кода осуществляется переход
				 к месту реализации вызываемых классов/процедур/функций.


				 CTRL+SPACE
				 Вызов IntelliSense в коде.


				 CTRL+SHIFT+SPACE
				 Вызов всплывающей подсказки для члена типа в окне макроса.


				 CTRL+K+C
				 Закомментировать выделенные строки. Доступно также
				 через команду «Правка
				 > Закомментировать выделенные строки» главного
				 меню.


				 CTRL+U
				 Раскомментировать выделенные строки. Доступно также
				 через команду «Правка
				 > Раскомментировать выделенные строки» главного
				 меню.


				 CTRL+L
				 Свернуть/развернуть текущий сегмент кода.


				 CTRL+M
				 Свернуть/развернуть все сегменты кода.


				 CTRL+ALT+N
				 Свернуть/развернуть дочерние сегменты кода.


				 CTRL+K+D
				 Форматировать весь модуль.


				 CTRL+K+F
				 Форматировать выделенный фрагмент.
