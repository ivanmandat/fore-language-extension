# Создание Java-модулей


Java-модуль предназначен для хранения в репозитории кода модулей на
 языке Java.
 Доступно выполнение Java-модуля из прикладного кода на Fore.
 Разработку и отладку самих модулей рекомендуется производить в специализированных
 средах разработки, ориентированных на работу с языком Java. Наполнение
 и выполнение Java-модуля осуществляется только из прикладного кода.


Примечание.
 Работа с Java-модулями доступна только в настольном приложении. Версии
 устанавливаемого программного обеспечения Java должны быть 8 или выше.
 Разрядность Java и «Форсайт. Аналитическая платформа» должна совпадать.
 Для получения подробной информации обратитесь к разделу «[Подключение
 дополнительных модулей к «Форсайт. Аналитическая платформа](UiNav.chm::/02_Navigator/CommonSettings/Integration.htm)».


[![](../../../Opened.gif)![](../../../Closed.gif)Создание Java-модуля](javascript:TextPopup(this))


	Для создания нового Java-модуля в [навигаторе
	 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm):


		- нажмите кнопку «Новый объект
		 > Среда разработки
		 > Java-модуль» в группе «Создать»
		 на вкладке «Главная» ленты
		 инструментов;


		- выполните команду «Создать
		 > Среда разработки > Java-модуль» в контекстном меню
		 навигатора объектов.


Для наполнения Java-модуля напишете код на Fore с использованием синтаксиса
 языка Java или вставьте заранее написанный и протестированный модуль с
 помощью свойства [IJavaModule.Text](KeJava.chm::/Interface/IJavaModule/IJavaModule.Text.htm).


Для использования кода из Java-модуля предназначены ресурсы сборки [Java](KeJava.chm::/KeJava_Title.htm).
 Для выполнения отдельных функций используйте метод [Java.InvokeModule](KeJava.chm::/Class/Java/Java.InvokeModule.htm)
 или [IJavaUtils.InvokeModule](KeJava.chm::/Interface/IJavaUtils/IJavaUtils.InvokeModule.htm),
 для работы с классами используйте конструктор [JavaClassObject.CreateClass](KeJava.chm::/Class/JavaClassObject/JavaClassObject.CreateClass.htm).
