# Разработка прикладного приложения


Для разработки прикладного приложения используйте интегрированный инструмент
 «Среда разработки».


Инструмент «Среда разработки»
 предназначен для проектирования, создания, запуска и отладки пользовательских
 приложений. Разработка приложений возможна на языке программирования [Fore](Fore.chm::/Fore_Title.htm).


Для разработки прикладного приложения:


	- Создайте [сборку](../02_Work_in_Development_Environment/DevEnv_Object/DevEnv_Assembly.htm)
	 для объединения форм и модулей по их функциональному назначению.


	- Создайте [форму](../02_Work_in_Development_Environment/DevEnv_Object/DevEnv_Form.htm)
	 в настольном приложении, [веб-форму](../02_Work_in_Development_Environment/DevEnv_Object/Web_Form.htm)
	 в веб-приложении или [модуль](../02_Work_in_Development_Environment/DevEnv_Object/DevEnv_Module.htm).


Если предполагается использование [дополнительных
 модулей](UiNav.chm::/02_Navigator/CommonSettings/Integration.htm), подключенных к «Форсайт. Аналитическая платформа»:


		- создайте [Python-модули](../02_Work_in_Development_Environment/DevEnv_Object/python_module.htm)
		 для интеграции с Python;


		- создайте [Java-модули](../02_Work_in_Development_Environment/DevEnv_Object/Java_module.htm)
		 для интеграции с Java (доступно только в настольном приложении).


	- Добавьте [код
	 приложения](../03_Windows_of_Development_Environment/Window_Macros.htm). При разработке можно воспользоваться различными [примерами кода на
	 Fore](../../Project_Development/fore_sample.htm).


При необходимости созданные объекты могут
 быть [сохранены
 в файл или загружены из файла](../02_Work_in_Development_Environment/Load_ObjectDE_From_External_File.htm).


	- Произведите [запуск
	 и отладку](../02_Work_in_Development_Environment/Run_Debug.htm) получившегося приложения.


	- [Защитите объекты репозитория](../Objects_protection.htm)
	 от редактирования при необходимости.


После выполнения действий будет разработано прикладное приложение с
 пользовательскими настройками.


Для форм доступна поддержка нескольких языков интерфейса с помощью [ресурсов](../07_MultiLng_Applications/UiDevEnv_MLng_Applications.htm).
