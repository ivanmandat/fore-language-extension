# IRegistryClass


## Описание


Интерфейс IRegistryClass содержит
 статические свойства, предоставляющие возможность для работы с базовыми
 разделами реестра.


## Комментарии


Данный интерфейс содержит статические свойства класса [RegistryClass](../../Class/RegistryClass/RegistryClass.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ClassesRoot](IRegistryClass.ClassesRoot.htm)
		 Свойство ClassesRoot
		 возвращает раздел реестра, содержащий сведения, обеспечивающие
		 открытие необходимой программы при открытии файла.


		 ![](../../Property_Image.gif)
		 [CurrentConfig](IRegistryClass.CurrentConfig.htm)
		 Свойство CurrentConfig
		 возвращает раздел, содержащий сведения о профиле оборудования,
		 используемые локальным компьютером при запуске системы.


		 ![](../../Property_Image.gif)
		 [CurrentUser](IRegistryClass.CurrentUser.htm)
		 Свойство CurrentUser
		 возвращает раздел, содержащий настройки пользователя, вошедшего
		 в систему в настоящий момент.


		 ![](../../Property_Image.gif)
		 [LocalMachine](IRegistryClass.LocalMachine.htm)
		 Свойство LocalMachine
		 возвращает раздел, содержащий настройки, относящиеся к данному
		 компьютеру (для всех пользователей).


		 ![](../../Property_Image.gif)
		 [Users](IRegistryClass.Users.htm)
		 Свойство Users возвращает
		 раздел, содержащий настройки всех профилей пользователей компьютера.
