# IRegistryKey.Name


## Синтаксис


Name: String;


## Описание


Свойство Name возвращает наименование
 раздела реестра.


## Комментарии


Имя подраздела включает абсолютный путь этого подраздела в реестре,
 всегда начинающийся в базовом разделе, например, [HKEY_LOCAL_MACHINE].
