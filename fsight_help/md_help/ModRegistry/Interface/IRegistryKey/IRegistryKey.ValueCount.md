# IRegistryKey.ValueCount


## Синтаксис


ValueCount: Integer;


## Описание


Свойство ValueCount возвращает
 количество параметров в подразделе реестра.


## Комментарии


В каждом подразделе содержится строковый параметр по умолчанию. Данный
 параметр не содержит имя и не включается в общий список параметров подраздела.
 Для работы с параметром по умолчанию необходимо методам записи и чтения
 в качестве наименования параметра передать пустую строку - "".
