# IRegistryKey.Flush


## Синтаксис


Flush;


## Описание


Метод Flush осуществляет запись
 всех атрибутов заданного открытого подраздела реестра на диск.


## Комментарии


Изменения реестра сбрасываются на диск с помощью отложенного сброса.
 Отложенный сброс происходит автоматически и регулярно после заданного
 системой интервала времени. Изменения реестра также сбрасываются на диск
 при завершении работы системы.


Метод Flush необходимо вызывать,
 если требуется абсолютная уверенность в том, что изменения реестра сохранены
 на диске.
