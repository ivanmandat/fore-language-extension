# RegistryClass


## Описание


Класс RegistryClass реализует
 объект, позволяющий работать с одним из базовых разделов реестра.


## Статические свойства класса, унаследованные от [IRegistryClass](../../Interface/IRegistryClass/IRegistryClass.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ClassesRoot](../../Interface/IRegistryClass/IRegistryClass.ClassesRoot.htm)
		 Свойство ClassesRoot
		 возвращает раздел реестра, содержащий сведения, обеспечивающие
		 открытие необходимой программы при открытии файла.


		 ![](../../Property_Image.gif)
		 [CurrentConfig](../../Interface/IRegistryClass/IRegistryClass.CurrentConfig.htm)
		 Свойство CurrentConfig
		 возвращает раздел, содержащий сведения о профиле оборудования,
		 используемые локальным компьютером при запуске системы.


		 ![](../../Property_Image.gif)
		 [CurrentUser](../../Interface/IRegistryClass/IRegistryClass.CurrentUser.htm)
		 Свойство CurrentUser
		 возвращает раздел, содержащий настройки пользователя, вошедшего
		 в систему в настоящий момент.


		 ![](../../Property_Image.gif)
		 [LocalMachine](../../Interface/IRegistryClass/IRegistryClass.LocalMachine.htm)
		 Свойство LocalMachine
		 возвращает раздел, содержащий настройки, относящиеся к данному
		 компьютеру (для всех пользователей).


		 ![](../../Property_Image.gif)
		 [Users](../../Interface/IRegistryClass/IRegistryClass.Users.htm)
		 Свойство Users возвращает
		 раздел, содержащий настройки всех профилей пользователей компьютера.
