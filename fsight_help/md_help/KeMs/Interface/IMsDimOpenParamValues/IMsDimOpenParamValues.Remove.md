# IMsDimOpenParamValues.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемого набора
 параметров открытия.


## Описание


Метод Remove удаляет набор параметров
 открытия из коллекции с указанным индексом.


## Комментарии


Если удаление выполнено успешно, то метод вернет значение True,
 иначе - False.
