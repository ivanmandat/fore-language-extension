# IMsDimOpenParamValues.FindByDimId


## Синтаксис


FindByDimId(Id: Integer): [IMsDimOpenParamValue](../IMsDimOpenParamValue/IMsDimOpenParamValue.htm);


## Параметры


Id. Идентификатор измерения.


## Описание


Метод FindByDimId возвращает
 параметры открытия измерения по идентификатору.


## Комментарии


Если измерение с указанным идентификатором не найдено, то метод возвращает
 значение Null.
