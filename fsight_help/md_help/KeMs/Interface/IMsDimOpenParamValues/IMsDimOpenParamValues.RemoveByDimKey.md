# IMsDimOpenParamValues.RemoveByDimKey


## Синтаксис


RemoveByDimKey(Key: Integer): Boolean;


## Параметры


Key. Ключ измерения.


## Описание


Метод RemoveByDimKey удаляет
 указанный набор параметров открытия из коллекции по ключу измерения.


## Комментарии


Если удаление выполнено успешно, то метод вернет значение True,
 иначе - False.
