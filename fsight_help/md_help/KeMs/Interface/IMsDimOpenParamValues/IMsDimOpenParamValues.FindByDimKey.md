# IMsDimOpenParamValues.FindByDimKey


## Синтаксис


FindByDimKey(Key: Integer): [IMsDimOpenParamValue](../IMsDimOpenParamValue/IMsDimOpenParamValue.htm);


## Параметры


Key. Ключ измерения.


## Описание


Метод FindByDimKey возвращает
 параметры открытия измерения по ключу.


## Комментарии


Если измерение с указанным ключом не найдено, то метод возвращает значение
 Null.
