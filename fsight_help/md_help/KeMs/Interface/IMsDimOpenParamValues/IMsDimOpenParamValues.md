# IMsDimOpenParamValues


Сборка: Ms;


## Описание


Интерфейс IMsDimOpenParamValues
 предназначен для указания параметров, которые будут использоваться при
 открытии измерений в объекте моделирования.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMsDimOpenParamValues.Count.htm)
		 Свойство Count возвращает
		 количество элементов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMsDimOpenParamValues.Item.htm)
		 Свойство Item возвращает
		 указанный набор параметров открытия из коллекции.
