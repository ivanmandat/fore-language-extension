# IMsCalculationChainEntry


Сборка: Ms;


## Описание


Интерфейс IMsCalculationChainEntry
 содержит свойства элементов [цепочки
 расчёта](UiModelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_3_MetaModel/uimodelling_metamodel_2.htm) метамодели.


## Иерархия наследования


           IMsCalculationChainEntry


## Комментарии


Для работы со всей цепочкой расчёта используйте интерфейс [IMsCalculationChainEntries](../IMsCalculationChainEntries/IMsCalculationChainEntries.htm).
