# IMsCalculationChainEntry.Name


## Синтаксис


		Name: String;


## Описание


Свойство Name определяет наименование
 элемента [цепочки
 расчета](UiModelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_3_MetaModel/uimodelling_metamodel_2.htm).


## Комментарии


Если элемент цепочки является [условием](../IMsCalculationChainBranch/IMsCalculationChainBranch.htm),
 то после изменения наименования свойство [IMsCalculationChainBranch.AutoName](../IMsCalculationChainBranch/IMsCalculationChainBranch.AutoName.htm)
 принимает значение False.
