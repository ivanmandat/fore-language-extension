# IMsECMEquation.Included


## Синтаксис


		Included: Boolean;


## Описание


Свойство Included определяет
 признак использования данного уравнения при расчёте модели коррекции ошибок.


## Комментарии


Допустимые значения:


	- True. Данное уравнение
	 будет использоваться при расчёте модели коррекции ошибок;


	- False. Данное уравнение
	 не будет участвовать в расчёте модели коррекции ошибок.
