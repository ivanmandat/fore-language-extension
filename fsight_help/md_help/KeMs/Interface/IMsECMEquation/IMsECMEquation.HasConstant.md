# IMsECMEquation.HasConstant


## Синтаксис


		HasConstant: Boolean;


## Описание


Свойство HasConstant определяет
 признак наличия константы в уравнении модели.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 В уравнении модели присутствует константа;


	- False. В уравнении модели
	 нет констант.
