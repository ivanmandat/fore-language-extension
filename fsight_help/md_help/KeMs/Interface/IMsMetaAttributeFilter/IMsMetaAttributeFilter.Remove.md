# IMsMetaAttributeFilter.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс параметра фильтрации.


## Описание


Метод Remove осуществляет удаление параметров фильтра по индексу. Метод возвращает значение True, если удаление прошло успешно.
