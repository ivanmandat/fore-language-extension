# IMsMetaAttributeFilter.RemoveByKey


## Синтаксис


RemoveByKey(AttributeKey: Integer): Boolean;


## Параметры


AttributeKey. Ключ атрибута.


## Описание


Метод RemoveByKey осуществляет удаление параметров фильтра по ключу атрибута. Метод возвращает значение True, если удаление прошло успешно.
