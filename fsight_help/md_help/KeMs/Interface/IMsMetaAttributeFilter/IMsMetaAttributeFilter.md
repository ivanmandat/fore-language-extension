# IMsMetaAttributeFilter


## Описание


Интерфейс IMsMetaAttributeFilter
 содержит свойства и методы фильтра, основанного на атрибутах.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMsMetaAttributeFilter.Count.htm)
		 Свойство Count возвращает
		 количество атрибутов в фильтре.


		 ![](../../Property_Image.gif)
		 [Item](IMsMetaAttributeFilter.Item.htm)
		 Свойство Item возвращает
		 атрибут из фильтра по его индексу.
