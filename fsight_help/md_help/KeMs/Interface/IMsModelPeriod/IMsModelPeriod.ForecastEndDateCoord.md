# IMsModelPeriod.ForecastEndDateCoord


## Синтаксис


		ForecastEndDateCoord: [IMsModelCoordAttributes](../IMsModelCoordAttributes/IMsModelCoordAttributes.htm);


## Описание


Свойство ForecastEndDateCoord
 определяет настройки координаты, задающей дату окончания периода прогнозирования.


## Комментарии


Координата доступна только если модель находится под [многомерным
 итератором](../IMsCalculationChainMultiDimIterator/IMsCalculationChainMultiDimIterator.htm). При выполнении координата - это текущая итерация многомерного
 итератора, под которым выполняется модель.
