# IMsModelPeriod.IdentificationStartOffset


## Синтаксис


IdentificationStartOffset: Integer;


## Описание


Свойство IdentificationStartOffset
 определяет число периодов, на которое сдвигается дата начала периода идентификации.


## Комментарии


IdentificationStartOffset учитывается,
 если выполняется одно из условий:


	- [IMsModelPeriod.AutoPeriod](IMsModelPeriod.AutoPeriod.htm)
	 использует значение, требующее задания дополнительного числа периодов,
	 например, MsAutoPeriodOptions.IdentStartAsNow;


	- задано свойство [IMsModelPeriod.IdentificationStartOffset](IMsModelPeriod.IdentificationStartOffset.htm).
