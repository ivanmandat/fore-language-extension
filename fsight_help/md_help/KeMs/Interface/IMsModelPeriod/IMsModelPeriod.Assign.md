# IMsModelPeriod.Assign


## Синтаксис


Assign(Period: [IMsModelParams](../IMsModelParams/IMsModelParams.htm));


## Параметры


Period. Копируемый период.


## Описание


Метод Assign копирует
 параметры указанного периода в текущий период.


## Комментарии


Данный метод позволяет полностью скопировать параметры периода, включая
 внутренние свойства.
