# IMsModelPeriod.UseForecastStart


## Синтаксис


UseForecastStart: Boolean;


## Описание


Свойство UseForecastStart
 определяет, использовать ли дату начала прогнозирования.


## Комментарии


Допустимые значения:


	- True. Использовать дату
	 начала прогнозирования;


	- False. Не использовать
	 дату начала прогнозирования.
