# IMsPDLTermOptions


Сборка: Ms;


## Описание


Интерфейс IMsPDLTermOptions
 возвращает параметры терма, соответствующего лаговой переменной.


## Иерархия наследования


           [IMsTermOptions](../IMsTermOptions/IMsTermOptions.htm)


           IMsPDLTermOptions


## Комментарии


IMsPDLTermOptions является наследником
 базового интерфейса [IMsTermOptions](../IMsTermOptions/IMsTermOptions.htm).
