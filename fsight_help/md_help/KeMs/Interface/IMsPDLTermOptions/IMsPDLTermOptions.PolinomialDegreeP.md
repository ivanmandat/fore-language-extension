# IMsPDLTermOptions.PolinomialDegreeP


## Синтаксис


PolinomialDegreeP: Integer;


## Описание


Свойство PolinomialDegreeP определяет степень полинома терма.


## Комментарии


Степень полинома не должна быть больше числа факторов модели.
