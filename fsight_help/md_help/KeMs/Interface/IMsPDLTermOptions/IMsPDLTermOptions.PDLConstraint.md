# IMsPDLTermOptions.PDLConstraint


## Синтаксис


PDLConstraint: [PDLConstraintType](StatLib.chm::/Enums/PDLConstraintType.htm);


## Описание


Свойство PDLConstraint определяет ограничения терма.


## Комментарии


По умолчанию ограничения терма не используются.
