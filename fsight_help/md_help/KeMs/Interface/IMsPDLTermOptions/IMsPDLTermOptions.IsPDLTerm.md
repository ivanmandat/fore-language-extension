# IMsPDLTermOptions.IsPDLTerm


## Синтаксис


IsPDLTerm: Boolean;


## Описание


Свойство IsPDLTerm определяет,
 является ли терм лаговой переменной.


## Комментарии


Допустимые значения:


	- True. Терм является
	 лаговой переменной;


	- False. Терм не является
	 лаговой переменной.
