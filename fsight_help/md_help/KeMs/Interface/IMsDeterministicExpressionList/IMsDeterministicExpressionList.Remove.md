# IMsDeterministicExpressionList.Remove


## Синтаксис


		Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс формулы.


## Описание


Метод Remove удаляет из коллекции
 формулу с указанным индексом.


## Комментарии


При удачном удалении метод возвращает значение True,
 при неудачном - False.
