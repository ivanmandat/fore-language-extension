# IMsDeterministicExpressionList


Сборка: Ms;


## Описание


Интерфейс IMsDeterministicExpressionList
 предназначен для работы с коллекцией формул, имеющих период действия.


## Иерархия наследования


           IMsDeterministicExpressionList


## Комментарии


Коллекцию формул можно получить, используя свойство [IMsDeterministicTransform.ExpressionList](../IMsDeterministicTransform/IMsDeterministicTransform.ExpressionList.htm).

		 Имя свойства
