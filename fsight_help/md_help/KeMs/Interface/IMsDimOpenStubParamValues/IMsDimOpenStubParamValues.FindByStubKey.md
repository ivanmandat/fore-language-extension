# IMsDimOpenStubParamValues.FindByStubKey


## Синтаксис


FindByStubKey(StubKey: Integer): [IMsDimOpenStubParamValue](../IMsDimOpenStubParamValue/IMsDimOpenStubParamValue.htm);


## Параметры


StubKey. Ключ абстрактного
 источника данных.


## Описание


Метод FindByStubKey возвращает
 набор параметров открытия измерений по ключу абстрактного источника данных.


## Комментарии


Если указанная отметка не найдена, то метод вернет значение Null.
