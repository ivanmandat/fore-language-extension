# IMsDimOpenStubParamValues


Сборка: Ms;


## Описание


Интерфейс IMsDimOpenStubParamValues
 предназначен для указания параметров измерений источников данных, которые
 будут использоваться при открытии измерений источников в объекте моделирования.
