# IMsDimOpenStubParamValues.RemoveByStubKey


## Синтаксис


RemoveByStubKey(StubKey: Integer): Boolean;


## Параметры


StubKey. Ключ абстрактного
 источника данных.


## Описание


Метод RemoveByKey удаляет указанный
 набор параметров открытия из коллекции по ключу абстрактного источника
 данных.


## Комментарии


Если удаление выполнено успешно, то метод вернет значение True,
 иначе - False.
