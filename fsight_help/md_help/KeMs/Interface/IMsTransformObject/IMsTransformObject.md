# IMsTransformObject


Сборка: Ms;


## Описание


Интерфейс IMsTransformObject
 предназначен для работы с объектом, используемым методом расчёта.


## Иерархия наследования


           [INamedEntityModel](ForeSys.chm::/Interface/INamedEntityModel/INamedEntityModel.htm)


           IMsTransformObject


## Комментарии


Объект используется для вставки константного значения справочника в
 уравнение расчета.
