# IMsInterindustryBalanceTransform.NewGrossIssueMatrix


## Синтаксис


NewGrossIssueMatrix: [IMsFormulaTerm](../IMsFormulaTerm/IMsFormulaTerm.htm);


## Описание


Свойство NewGrossIssueMatrix
 определяет терм, соответствующий переменной, в которую будет выгружаться
 рассчитанная матрица национального дохода. Используется при расчете III
 квадранта.
