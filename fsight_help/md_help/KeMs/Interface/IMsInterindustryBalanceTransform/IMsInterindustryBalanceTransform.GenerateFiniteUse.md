# IMsInterindustryBalanceTransform.GenerateFiniteUse


## Синтаксис


GenerateFiniteUse: Boolean;


## Описание


Свойство GenerateFiniteUse определяет,
 будет ли рассчитываться матрица конечного использования при расчете модели.


## Комментарии


По умолчанию свойству установлено значение False,
 при этом используется матрица из переменной, установленной в свойстве
 [FiniteUseMatrix](IMsInterindustryBalanceTransform.FiniteUseMatrix.htm).
 Данное свойство актуально при расчете II квадранта.
