# IMsInterindustryBalanceTransform.GenerateMatrixA


## Синтаксис


GenerateMatrixA: Boolean;


## Описание


Свойство GenerateMatrixA определяет,
 будет ли генерироваться матрица А при расчете модели.


## Комментарии


По умолчанию свойству установлено значение True.
 Если свойству установить значение False
 - матрица А будет загружаться из переменной, установленной в свойстве
 [MatrixA](IMsInterindustryBalanceTransform.MatrixA.htm).
