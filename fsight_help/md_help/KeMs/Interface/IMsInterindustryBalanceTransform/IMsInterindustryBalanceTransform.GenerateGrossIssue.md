# IMsInterindustryBalanceTransform.GenerateGrossIssue


## Синтаксис


GenerateGrossIssue: Boolean;


## Описание


Свойство GenerateGrossIssue
 определяет, будет ли рассчитываться матрица национального дохода при расчете
 модели.


## Комментарии


По умолчанию свойству установлено значение False,
 при этом используется матрица из переменной, установленной в свойстве
 [GrossIssueMatrix](IMsInterindustryBalanceTransform.GrossIssueMatrix.htm).
 Данное свойство актуально при расчете III квадранта.
