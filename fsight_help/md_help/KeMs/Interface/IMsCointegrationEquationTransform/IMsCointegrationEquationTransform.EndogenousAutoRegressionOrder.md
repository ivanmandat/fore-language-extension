# IMsCointegrationEquationTransform.EndogenousAutoRegressionOrder


## Синтаксис


		EndogenousAutoRegressionOrder: Array;


## Описание


Свойство EndogenousAutoRegressionOrder
 определяет порядок авторегрессии эндогенной переменной.


## Комментарии


Для задания порядка авторегрессии экзогенной переменной используйте
 свойство [IMsCointegrationEquationTransform.ExogenousAutoRegressionOrder](IMsCointegrationEquationTransform.ExogenousAutoRegressionOrder.htm).
