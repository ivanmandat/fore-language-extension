# IMsCointegrationEquationTransform.ExogenousAutoRegressionOrder


## Синтаксис


		ExogenousAutoRegressionOrder: Array;


## Описание


Свойство ExogenousAutoRegressionOrder
 определяет порядок авторегрессии экзогенной переменной.


## Комментарии


Для задания порядка авторегрессии эндогенной переменной используйте
 свойство [IMsCointegrationEquationTransform.EndogenousAutoRegressionOrder](IMsCointegrationEquationTransform.EndogenousAutoRegressionOrder.htm).
