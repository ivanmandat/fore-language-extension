# IMsDimIteratorLinks.Remove


## Синтаксис


		Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемого связанного
 измерения.


## Описание


Метод Remove удаляет связанное
 измерение с указанным индексом.


## Комментарии


При успешном удалении связанного измерения метод возвращает значение
 True, при неуспешном - False.


Индексация связанных измерений в коллекции сквозная и начинается с нуля.


Для получения количества связанных измерений в коллекции используйте
 свойство [IMsDimIteratorLinks.Count](IMsDimIteratorLinks.Count.htm).
