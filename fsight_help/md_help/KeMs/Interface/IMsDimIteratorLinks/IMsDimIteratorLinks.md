# IMsDimIteratorLinks


Сборка: Ms;


## Описание


Интерфейс IMsDimIteratorLinks
 предназначен для работы с коллекцией связанных измерений в измерении многомерного
 итератора.


## Иерархия наследования


           IMsDimIteratorLinks


## Комментарии


Для получения коллекции связей используйте свойство [IMsDimIteratorDimension.Links](../IMsDimIteratorDimension/IMsDimIteratorDimension.Links.htm).
