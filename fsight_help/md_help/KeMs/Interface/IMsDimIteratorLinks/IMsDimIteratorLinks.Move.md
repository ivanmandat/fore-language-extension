# IMsDimIteratorLinks.Move


## Синтаксис


		Move(FromIndex: Integer; ToIndex: Integer);


## Параметры


FromIndex. Индекс позиции связанного
 измерения, которое необходимо переместить;


ToIndex. Индекс позиции, в
 которую необходимо переместить связанное измерение.


## Описание


Метод Move перемещает связанное
 измерение в указанную позицию в коллекции.


## Комментарии


Если указан несуществующий индекс позиции, то перемещение не выполняется.
