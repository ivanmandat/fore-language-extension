# IMsDimIteratorLinks.RemoveByKey


## Синтаксис


		RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ удаляемого связанного
 измерения.


## Описание


Метод RemoveByKey удаляет связанное
 измерение с указанным ключом.


## Комментарии


При успешном удалении связанного измерения метод возвращает значение
 True, при неуспешном - False.
