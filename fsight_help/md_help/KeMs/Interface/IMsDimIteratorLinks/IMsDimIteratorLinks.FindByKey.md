# IMsDimIteratorLinks.FindByKey


## Синтаксис


		FindByKey(Key: Integer): [IMsDimIteratorLink](../IMsDimIteratorLink/IMsDimIteratorLink.htm);


## Параметры


Key. Ключ измерения.


## Описание


Метод FindByKey выполняет поиск
 связанного измерения с указанным ключом.


## Комментарии


Если связанное измерение не найдено, то метод возвращает значение Null.
