# IMsDimIteratorLinks.IndexOf


## Синтаксис


		IndexOf(Value: [IMsDimIteratorLink](../IMsDimIteratorLink/IMsDimIteratorLink.htm)):
		 Integer;


## Параметры


Value. Связанное измерение,
 индекс которого необходимо узнать.


## Описание


Метод IndexOf возвращает индекс
 указанного связанного измерения.


## Комментарии


Если указанное связанное измерение отсутствует в коллекции, то метод
 возвращает значение «-1».
