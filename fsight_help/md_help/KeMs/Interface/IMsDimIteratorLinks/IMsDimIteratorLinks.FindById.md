# IMsDimIteratorLinks.FindById


## Синтаксис


		FindById(Id: String): [IMsDimIteratorLink](../IMsDimIteratorLink/IMsDimIteratorLink.htm);


## Параметры


Id. Идентификатор измерения.


## Описание


Метод FindById выполняет поиск
 связанного измерения с указанным идентификатором.


## Комментарии


Если связанное измерение не найдено, то метод возвращает значение Null.
