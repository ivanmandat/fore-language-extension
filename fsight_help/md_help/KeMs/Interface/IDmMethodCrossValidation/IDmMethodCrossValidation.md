# IDmMethodCrossValidation


Сборка: Ms;


## Описание


Интерфейс IDmMethodCrossValidation
 предназначен для настройки параметров кросс-валидации.


## Иерархия наследования


           IDmMethodCrossValidation


## Комментарии


Кросс-валидация (англ. Cross-validation) - метод оценки качества классификации.


Для получения параметров кросс-валидации используйте свойство [IDmMethodDetails.CrossValidation](../IDmMethodDetails/IDmMethodDetails.CrossValidation.htm).
