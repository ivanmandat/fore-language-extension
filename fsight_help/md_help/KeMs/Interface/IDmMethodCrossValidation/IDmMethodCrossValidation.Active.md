# IDmMethodCrossValidation.Active


## Синтаксис


		Active: Boolean;


## Описание


Свойство Active определяет
 признак использования кросс-валидации.


## Комментарии


Допустимые значения:


	- True. Кросс-валидация
	 используется;


	- False. Кросс-валидация
	 не используется.
