# IDmMethodCrossValidation.SamplingType


## Синтаксис


		SamplingType: [CrossValidationSamplingType](StatLib.chm::/Enums/CrossValidationSamplingType.htm);


## Описание


Свойство SamplingType
 определяет метод кросс-валидации.


## Комментарии


По умолчанию используется метод кросс-валидация по K
 блокам, то есть свойство имеет значение [CrossValidationSamplingType.Kfold](StatLib.chm::/Enums/CrossValidationSamplingType.htm).
