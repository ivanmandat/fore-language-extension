# IValidationFilter.Kind


## Синтаксис


Kind: [ValidationDetailsKind](../../Enums/ValidationDetailsKind.htm);


## Описание


Свойство Kind определяет тип правила валидации.


## Комментарии


От типа фильтра зависит набор специфических настроек, доступных для правила валидации. Для получения настроек используйте свойство [IValidationFilter.Details](IValidationFilter.Details.htm).
