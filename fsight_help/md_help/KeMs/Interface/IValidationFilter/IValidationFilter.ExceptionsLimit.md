# IValidationFilter.ExceptionsLimit


## Синтаксис


ExceptionsLimit: Integer;


## Описание


Свойство ExceptionsLimit определяет
 пороговое количество исключений, выгружаемых правилом валидации.


## Комментарии


По умолчанию правило валидации выгружает 1000000 исключений.


Если количество найденных исключений превышает пороговое значение, то
 правило генерирует предупреждение, сохраняет найденные исключения и завершает
 работу.


Если ExceptionsLimit = 0,
 то пороговое значение отсутствует и правило выявляет все исключения.


Важно. Если
 пороговое значение отсутствует, то правило может аварийно завершить свою
 работу из-за нехватки оперативной памяти. Найденные исключения сохранены
 не будут.
