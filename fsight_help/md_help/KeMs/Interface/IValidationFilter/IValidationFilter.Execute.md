# IValidationFilter.Execute


## Синтаксис


Execute(Settings: [IValidationExecuteSettings](../IValidationExecuteSettings/IValidationExecuteSettings.htm)): [IValidationExecRun](KeCubes.chm::/Interface/IValidationExecRun/IValidationExecRun.htm);


## Параметры


Settings. Настройки расчета валидации.


## Описание


Метод Execute выполняет расчёт правила валидации.


## Комментарии


После окончания расчёта метод возвращает объект, содержащий результаты выполнения правила.
