# IValidationFilter.Level


## Синтаксис


Level: [DimCalendarLevel](KeDims.chm::/Enums/DimCalendarLevel.htm);


## Описание


Свойство Level определяет календарный уровень для расчета правила.


## Комментарии


Если свойство имеет значение [DimCalendarLevel.None](KeDims.chm::/Enums/DimCalendarLevel.htm), то правило будет рассчитано по всем календарным уровням.
