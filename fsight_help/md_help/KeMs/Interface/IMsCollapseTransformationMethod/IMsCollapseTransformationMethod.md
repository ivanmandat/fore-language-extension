# IMsCollapseTransformationMethod


Сборка: Ms;


## Описание


Интерфейс IMsCollapseTransformationMethod
 содержит свойства для определения параметров преобразования терма методом
 «Коллапс».


## Иерархия наследования


           IMsCollapseTransformationMethod


## Комментарии


Метод осуществляет агрегацию данных с нижнего уровня на верхний. Например,
 данные, рассчитанные по месяцам, необходимо просуммировать для квартальной
 динамики.
