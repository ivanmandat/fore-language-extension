# IMsUORuntimeEquations


Сборка: Ms;


## Описание


Интерфейс IMsUORuntimeEquations
 предназначен для работы с коллекцией уравнений, используемых для расчёта
 значений пользовательской задачи оптимизации.


## Иерархия наследования


IMsUORuntimeEquations


## Комментарии


Коллекцию уравнений можно получить, используя свойство [IMsUORuntime.Equations](../IMsUORuntime/IMsUORuntime.Equations.htm)
 во время расчёта задачи.
