# IMsProblemDetailsArray.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс параметров в
 коллекции.


## Описание


Метод Remove удаляет параметры
 с указанным индексом из коллекции.


## Комментарии


Если удаление выполнено успешно, то метод возвращает значение True, в противном случае - False.
