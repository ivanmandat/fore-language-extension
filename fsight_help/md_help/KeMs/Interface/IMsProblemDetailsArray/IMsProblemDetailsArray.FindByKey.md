# IMsProblemDetailsArray.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IMsProblemDetails](../IMsProblemDetails/IMsProblemDetails.htm);


## Описание


Метод FindByKey выполняет
 поиск параметров в коллекции по указанному ключу.


## Комментарии


Если параметры с указанным ключом не найдены, то метод возвращает значение
 Null.
