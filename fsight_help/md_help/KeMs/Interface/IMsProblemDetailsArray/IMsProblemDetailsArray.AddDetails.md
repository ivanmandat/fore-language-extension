# IMsProblemDetailsArray.AddDetails


## Синтаксис


AddDetails(Value: [IMsProblemDetails](../IMsProblemDetails/IMsProblemDetails.htm);
 Slice: [IMsFormulaTransformSlice](../IMsFormulaTransformSlice/IMsFormulaTransformSlice.htm)):
 Integer;


## Параметры


Value. Параметры задачи, добавляемые
 в коллекцию;


Slice. Срез, для которого добавляются
 параметры.


## Описание


Метод AddDetails добавляет
 указанные параметры в коллекцию.


## Комментарии


В результате выполнения метод возвращает индекс добавленных параметров.
