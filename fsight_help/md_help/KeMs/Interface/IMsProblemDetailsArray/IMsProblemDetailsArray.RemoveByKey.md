# IMsProblemDetailsArray.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ параметров в коллекции.


## Описание


Метод RemoveByKey удаляет
 параметры с указанным ключом из коллекции.


## Комментарии


Если удаление выполнено успешно, то метод возвращает значение True, в противном случае - False.
