# IMsProblemDetailsArray


Сборка: Ms;


## Описание


Интерфейс IMsProblemDetailsArray
 предназначен для работы с коллекцией параметров расчета задачи.


## Иерархия наследования


           [IMsProblemDetailsArray](IMsProblemDetailsArray.htm)


## Комментарии


Для получения коллекции параметров расчета задачи используйте свойство
 [IMsProblem.AdditionalDetails](../IMsProblem/IMsProblem.AdditionalDetails.htm).
