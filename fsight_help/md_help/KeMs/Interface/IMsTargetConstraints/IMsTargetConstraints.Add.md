# IMsTargetConstraints.Add


## Синтаксис


Add: [IMsTargetConstraint](../IMsTargetConstraint/IMsTargetConstraint.htm);


## Описание


Метод Add добавляет ограничение
 в коллекцию.


## Комментарии


Для удаления определённого ограничения из коллекции используйте метод
 [IMsTargetConstraints.Remove](IMsTargetConstraints.Remove.htm),
 для удаления всех ограничений - метод [IMsTargetConstraints.Clear](IMsTargetConstraints.Clear.htm).
