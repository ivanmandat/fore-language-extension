# IMsTargetConstraints.FindByKey


## Синтаксис


FindByKey(Key: Integer): Integer;


## Параметры


Key. Ключ искомого ограничения.


## Описание


Метод FindByKey выполняет
 поиск ограничения в коллекции.


## Комментарии


Если ограничение с указанным ключом найдено, то метод возвращает его
 индекс, если ограничение не найдено - значение «-1».
