# IMsTargetConstraints.Item


## Синтаксис


Item(Index: Integer): [IMsTargetConstraint](../IMsTargetConstraint/IMsTargetConstraint.htm);


## Параметры


Index. Индекс ограничения.


## Описание


Свойство Item возвращает
 указанное ограничение из коллекции.


## Комментарии


Индексация ограничений в коллекции сквозная и начинается с нуля. Количество
 ограничений в коллекции возвращает свойство [IMsTargetConstraints.Count](IMsTargetConstraints.Count.htm).
