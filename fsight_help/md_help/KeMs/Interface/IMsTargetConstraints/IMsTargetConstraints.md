# IMsTargetConstraints


Сборка: Ms;


## Описание


Интерфейс IMsTargetConstraints
 предназначен для работы с коллекцией ограничений целевой функции.


## Иерархия наследования


           IMsTargetConstraints


## Комментарии


Для получения коллекции ограничений целевой функции используйте свойство
 [IMsTargetProblem.Constraints](../IMsTargetProblem/IMsTargetProblem.Constraints.htm).
