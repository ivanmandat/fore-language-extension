# IMsFormulaTransformSlices


## Описание


Интерфейс IMsFormulaTransformSlices
 содержит свойства и методы для работы с коллекцией срезов переменной.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMsFormulaTransformSlices.Count.htm)
		 Свойство Count возвращает
		 количество срезов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMsFormulaTransformSlices.Item.htm)
		 Свойство Item возвращает
		 срез переменной по индексу.

		 Имя метода
		 Краткое описание
