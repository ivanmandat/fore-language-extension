# IMsFormulaTransformSlices.FindByKey


## Синтаксис


FindByKey(SliceKey: Integer): [IMsFormulaTransformSlice](../IMsFormulaTransformSlice/IMsFormulaTransformSlice.htm);


## Параметры


SliceKey. Ключ среза в переменной,
 по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает объект, содержащий срез переменной.
