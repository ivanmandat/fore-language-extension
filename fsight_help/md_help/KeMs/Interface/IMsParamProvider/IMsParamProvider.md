# IMsParamProvider


Сборка: Ms;


## Описание


Интерфейс IMsParamProvider предназначен
 для работы с параметрами объекта.


## Иерархия наследования


           IMsParamProvider


## Комментарии


Для работы с параметрами используются две коллекции: коллекция параметров
 и коллекция значений параметров. Каждому элементу из коллекции параметров
 соответствует элемент из коллекции значений параметров. После добавления/удаления
 элемента из коллекции параметров соответствующий элемент в коллекции значений
 параметров будет автоматически создан/удален.
