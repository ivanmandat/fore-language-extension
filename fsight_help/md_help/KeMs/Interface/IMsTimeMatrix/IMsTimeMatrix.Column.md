# IMsTimeMatrix.Column


## Синтаксис


Column(Index: Integer): Variant;


## Параметры


Index. Индекс столбца матрицы.


## Описание


Метод Column возвращает столбец
 матрицы по его индексу.


## Комментарии


Для получения значения точки ряда по ее индексу используйте свойство
 [IMsTimeMatrix.Item](IMsTimeMatrix.Item.htm).
