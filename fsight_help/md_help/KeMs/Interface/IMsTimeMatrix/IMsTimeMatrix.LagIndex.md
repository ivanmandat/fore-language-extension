# IMsTimeMatrix.LagIndex


## Синтаксис


LagIndex(Index: Integer; Lag: Integer): Integer;


## Параметры


Index. Индекс;


Lag. Лаг.


## Описание


Свойство LagIndex возвращает
 индекс лага.


## Комментарии


Лаг - это сдвиг точек ряда во
 временном периоде. Направление сдвига зависит от значения лага. Если лаг
 отрицательный, то точки переменной сдвигаются назад, если положительный
 - вперед.
