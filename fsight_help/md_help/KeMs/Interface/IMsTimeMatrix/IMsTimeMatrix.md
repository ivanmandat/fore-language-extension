# IMsTimeMatrix


Сборка: Ms;


## Описание


Интерфейс IMsTimeMatrix предназначен
 для работы с матрицей.


## Иерархия наследования


           IMsTimeMatrix


## Комментарии


Матрица представляет собой набор рядов типа [ITimeSeries](../ITimeSeries/ITimeSeries.htm).
