# IMsTimeMatrix.CalcPointCount


## Синтаксис


CalcPointCount: Integer;


## Описание


Свойство CalcPointCount возвращает
 количество точек между индексами начала и конца ряда.


## Комментарии


Для получения индекса начала ряда используйте свойство [IMsTimeMatrix.StartIndex](IMsTimeMatrix.StartIndex.htm).
 Для получения индекса конца ряда используйте свойство [IMsTimeMatrix.EndIndex](IMsTimeMatrix.EndIndex.htm).
