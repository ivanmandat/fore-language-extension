# IMsTimeMatrix.EndCached


## Синтаксис


EndCached;


## Описание


Метод EndCached завершает кеширование
 данных в матрице.


## Комментарии


Для запуска кеширования данных в матрице используйте метод [IMsTimeMatrix.BeginCached](IMsTimeMatrix.BeginCached.htm).
