# IMsTimeMatrix.IndexToDate


## Синтаксис


IndexToDate(Index: Integer): [DateTime](ForeSys.chm::/Class/DateTime/DateTime.htm);


## Параметры


Index. Индекс точки ряда.


## Описание


Метод IndexToDate переводит
 индекс в дату.


## Комментарии


Для перевода даты в индекс используйте метод [IMsTimeMatrix.DateToIndex](IMsTimeMatrix.DateToIndex.htm).
