# IMsTimeMatrix.BeginCached


## Синтаксис


BeginCached;


## Описание


Метод BeginCached запускает
 кеширование данных матрицы.


## Комментарии


Для завершения кеширования данных матрицы используйте метод [IMsTimeMatrix.EndCached](IMsTimeMatrix.EndCached.htm).
