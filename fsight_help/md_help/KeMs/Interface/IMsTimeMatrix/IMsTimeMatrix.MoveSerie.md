# IMsTimeMatrix.MoveSerie


## Синтаксис


MoveSerie(FromIndex: Integer; ToIndex: Integer);


## Параметры


FromIndex. Индекс начальной
 позиции ряда;


ToIndex. Индекс позиции, куда
 необходимо переместить ряд.


## Описание


Метод MoveSerie перемещает ряд
 в указанную позицию.


## Комментарии


Для добавления ряда в матрицу используйте метод [IMsTimeMatrix.AddSerie](IMsTimeMatrix.AddSerie.htm).
