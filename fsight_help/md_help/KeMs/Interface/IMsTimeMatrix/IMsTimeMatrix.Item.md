# IMsTimeMatrix.Item


## Синтаксис


Item(Index: Integer): [ITimeSeries](../ITimeSeries/ITimeSeries.htm);


## Параметры


Index. Индекс точки ряда.


## Описание


Свойство Item возвращает значение
 точки ряда по ее индексу.


## Комментарии


Для получения количества рядов в матрице используйте свойство [IMsTimeMatrix.Count](IMsTimeMatrix.Count.htm).
