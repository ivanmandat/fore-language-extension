# IMsTimeMatrix.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 рядов в матрице.


## Комментарии


Для получения значения точки ряда по ее индексу используйте свойство
 [IMsTimeMatrix.Item](IMsTimeMatrix.Item.htm).
