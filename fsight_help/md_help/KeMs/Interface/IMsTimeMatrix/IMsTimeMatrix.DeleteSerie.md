# IMsTimeMatrix.DeleteSerie


## Синтаксис


DeleteSerie(Index: Integer): Boolean;


## Параметры


Index. Индекс ряда.


## Описание


Метод DeleteSerie удаляет ряд
 из матрицы по его индексу.


## Комментарии


Для добавления ряда в матрицу используйте метод [IMsTimeMatrix.AddSerie](IMsTimeMatrix.AddSerie.htm).
