# IMsDiagnosticTest


Сборка: Ms;


## Описание


Интерфейс IMsDiagnosticTest
 предназначен для работы с диагностическим тестом модели.


## Иерархия наследования


           IMsDiagnosticTest


## Комментарии


Диагностический тест может быть выполнен только для модели [линейной
 регрессии (оценка МНК)](../IMsLinearRegressionTransform/IMsLinearRegressionTransform.htm). Перед выполнением теста настройте его параметры
 с помощью свойства [IMsDiagnosticTest.Settings](IMsDiagnosticTest.Settings.htm).
