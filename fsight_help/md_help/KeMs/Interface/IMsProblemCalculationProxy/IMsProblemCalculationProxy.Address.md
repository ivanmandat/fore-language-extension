# IMsProblemCalculationProxy.Address


## Синтаксис


		Address: String;


## Описание


Свойство Address определяет
 адрес BI-сервера, на котором выполняется удалённый расчёт задачи моделирования.


## Комментарии


Значение свойства учитывается, если свойство [IMsProblemCalculationSettings.RemoteCalc](../IMsProblemCalculationSettings/IMsProblemCalculationSettings.RemoteCalc.htm)
 имеет значение MsProblemRemoteCalc.EnableBySources.
