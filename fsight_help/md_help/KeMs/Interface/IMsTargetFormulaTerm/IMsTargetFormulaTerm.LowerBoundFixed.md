## Описание


Свойство LowerBoundFixed
 определяет тип ограничения снизу.


## Комментарии


Для задания типа ограничения сверху используйте свойство [IMsTargetFormulaTerm.UpperBoundFixed](IMsTargetFormulaTerm.UpperBoundFixed.htm).
