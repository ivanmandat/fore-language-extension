# IMsModelSpace.StartDate


## Синтаксис


StartDate: [DateTime](ForeSys.chm::/Class/DateTime/DateTime.htm);


## Описание


Свойство StartDate определяет
 дату начала календарного периода, используемую для расчетов в контейнере
 моделирования.


## Комментарии


Для определения даты окончания календарного периода, применяемого для
 расчетов, используйте свойство [IMsModelSpace.EndDate](IMsModelSpace.EndDate.htm).
