# IMsModelSpace


Сборка: Ms;


## Описание


Интерфейс IMsModelSpace содержит
 базовые свойства объекта репозитория - [Контейнер
 моделирования](UiModelling.chm::/1_Modelling/UiModelling_First.htm).


## Иерархия наследования


           IMsModelSpace


## Комментарии


Контейнер моделирования - объект репозитория, предназначенный для формирования
 математических моделей процессов (явлений) и проведения расчетов на их
 основе.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Database](IMsModelSpace.Database.htm)
		 Свойство Database определяет
		 базу данных, в которой будут храниться данные моделей, содержащихся
		 в контейнере моделирования.


		 ![](../../Property_Image.gif)
		 [DefaultObject](IMsModelSpace.DefaultObject.htm)
		 Свойство DefaultObject
		 определяет базу данных временных рядов, в которой будут
		 храниться данные контейнера моделирования.


		 ![](../../Property_Image.gif)
		 [EndDate](IMsModelSpace.EndDate.htm)
		 Свойство EndDate определяет
		 дату окончания календарного периода, используемую для расчетов
		 в контейнере моделирования.


		 ![](../../Property_Image.gif)
		 [ScenarioDimension](IMsModelSpace.ScenarioDimension.htm)
		 Свойство ScenarioDimension
		 возвращает коллекцию сценариев контейнера моделирования в виде
		 измерения.


		 ![](../../Property_Image.gif)
		 [ScenarioTree](IMsModelSpace.ScenarioTree.htm)
		 Свойство ScenarioTree
		 возвращает коллекцию сценариев в виде дерева.


		 ![](../../Property_Image.gif)
		 [StartDate](IMsModelSpace.StartDate.htm)
		 Свойство StartDate
		 определяет дату начала календарного периода, используемую для
		 расчетов в контейнере моделирования.


		 ![](../../Property_Image.gif)
		 [UserMethodTemplates](IMsModelSpace.UserMethodTemplates.htm)
		 Свойство UserMethodTemplates
		 возвращает коллекцию шаблонов пользовательских методов, используемых
		 в контейнере моделирования.
