# IMsModelSpace.EndDate


## Синтаксис


EndDate: [DateTime](ForeSys.chm::/Class/DateTime/DateTime.htm);


## Описание


Свойство EndDate определяет
 дату окончания календарного периода, используемую для расчетов в контейнере
 моделирования.


## Комментарии


Для определения даты начала календарного периода, применяемого для расчетов,
 используйте свойство [IMsModelSpace.StartDate](IMsModelSpace.StartDate.htm).
