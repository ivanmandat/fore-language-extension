# IMsModelSpace.Database


## Синтаксис


Database: [IDatabase](KeDb.chm::/Interface/IDatabase/IDatabase.htm);


## Описание


Свойство Database определяет
 базу данных, в которой будут храниться данные моделей, содержащихся в
 контейнере моделирования.


## Комментарии


Данные контейнера моделирования также могут храниться в базе данных
 временных рядов, указанной в свойстве [IMsModelSpace.Database](IMsModelSpace.Database.htm).
