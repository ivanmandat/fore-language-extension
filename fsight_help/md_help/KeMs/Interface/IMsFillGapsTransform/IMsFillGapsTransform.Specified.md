# IMsFillGapsTransform.Specified


## Синтаксис


		Specified: [IMsFormulaTermInfo](../IMsFormulaTermInfo/IMsFormulaTermInfo.htm);


## Описание


Свойство Specified определяет
 образцовый ряд для методов обработки пропусков Pattern (по шаблону) и
 Overlay (значениями другого ряда).


## Комментарии


Метод обработки пропусков определяется свойством [IMsFillGapsTransform.MissingData](IMsFillGapsTransform.MissingData.htm).
