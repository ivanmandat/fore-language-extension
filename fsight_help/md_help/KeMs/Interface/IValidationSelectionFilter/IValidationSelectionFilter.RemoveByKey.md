# IValidationSelectionFilter.RemoveByKey


## Синтаксис


RemoveByKey(SelectionKey: Integer): Boolean;


## Параметры


SelectionKey. Ключ отметки.


## Описание


Метод RemoveByKey удаляет
 из фильтра элемент, соответствующий отметке с указанным ключом.


## Комментарии


Ключ отметки совпадает с ключом справочника, на котором основана отметка.


Если удаление выполнено успешно, то метод возвращает значение True, иначе - False.
