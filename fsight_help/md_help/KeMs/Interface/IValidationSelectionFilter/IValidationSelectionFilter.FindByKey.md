# IValidationSelectionFilter.FindByKey


## Синтаксис


FindByKey(SelectionKey: Integer): [IValidationSelectionValues](../IValidationSelectionValues/IValidationSelectionValues.htm);


## Параметры


SelectionKey. Ключ отметки.


## Описание


Метод FindByKey выполняет поиск
 элемента фильтра по ключу отметки.


## Комментарии


Ключ отметки совпадает с ключом справочника, на котором основана отметка.


Если элемент фильтра не найден, то метод возвращает значение Null.
