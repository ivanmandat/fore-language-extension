# IValidationSelectionFilter.Item


## Синтаксис


Item(Index: Integer): [IValidationSelectionValues](../IValidationSelectionValues/IValidationSelectionValues.htm);


## Параметры


Index. Индекс элемента фильтра.


## Описание


Свойство Item возвращает указанный
 элемент из фильтра.


## Комментарии


Индексация элементов в фильтре сквозная и начинается с нуля. Для получения
 количества элементов в фильтре используйте свойство [IValidationSelectionFilter.Count](IValidationSelectionFilter.Count.htm).
