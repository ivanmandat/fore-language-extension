# IValidationSelectionFilter.FindById


## Синтаксис


FindById(SelectionId: Stirng): [IValidationSelectionValues](../IValidationSelectionValues/IValidationSelectionValues.htm);


## Параметры


SelectionId. Идентификатор
 отметки.


## Описание


Метод FindById выполняет
 поиск элемента фильтра по идентификатору отметки.


## Комментарии


Идентификатор отметки совпадает с идентификатором справочника, на котором
 основана отметка.


Если элемент фильтра не найден, то метод возвращает значение Null.
