# IMsFindFormulaResult


Сборка: Ms;


## Описание


Интерфейс IMsFindFormulaResult
 предназначен для работы с результатом поиска формул.


## Иерархия наследования


           IMsFindFormulaResult


## Комментарии


Результат поиска можно получить, используя свойство [IMsFindFormulaResults.Item](../IMsFindFormulaResults/IMsFindFormulaResults.Item.htm).


Интерфейс является базовым для следующих интерфейсов:


	- [IMsFindFormulaResultBranchCondition](../IMsFindFormulaResultBranchCondition/IMsFindFormulaResultBranchCondition.htm);


	- [IMsFindFormulaResultModel](../IMsFindFormulaResultModel/IMsFindFormulaResultModel.htm).
