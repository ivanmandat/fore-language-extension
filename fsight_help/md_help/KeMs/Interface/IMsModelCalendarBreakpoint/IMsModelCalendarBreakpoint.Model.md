# IMsModelCalendarBreakpoint.Model


## Синтаксис


Model: [IMsModel](../IMsModel/IMsModel.htm);


## Описание


Свойство Model определяет модель, для которой устанавливается точка останова.


## Комментарии


После того как определена модель, необходимо задать календарную точку, при расчете которой будет срабатывать точка останова. Для этого используйте свойство [IMsModelCalendarBreakpoint.CalendarPoint](IMsModelCalendarBreakpoint.CalendarPoint.htm).
