# IMsModelCalendarBreakpoint.RemoveCalendarPointFix


## Синтаксис


RemoveCalendarPointFix;


## Описание


Метод RemoveCalendarPointFix сбрасывает календарную точку для точки останова.


## Комментарии


После вызова данного метода точка останова будет срабатывать для каждой календарной точки расчета. Свойство [IMsModelCalendarBreakpoint.IsCalendarPointFixed](IMsModelCalendarBreakpoint.IsCalendarPointFixed.htm) примет значение False.
