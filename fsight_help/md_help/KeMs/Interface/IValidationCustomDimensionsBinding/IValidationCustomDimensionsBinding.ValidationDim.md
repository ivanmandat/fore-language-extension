# IValidationCustomDimensionsBinding.ValidationDim


## Синтаксис


ValidationDim: [IDimInstance](KeDims.chm::/interface/IDimInstance/IDimInstance.htm);


## Описание


Свойство ValidationDim возвращает
 измерение из сопоставляемого источника данных, по которому выполняется
 валидация.


## Комментарии


Для получения измерения из внешнего источника данных, участвующего в
 сопоставлении, используйте свойство [IValidationCustomDimensionsBinding.ExternalDim](IValidationCustomDimensionsBinding.ExternalDim.htm).
