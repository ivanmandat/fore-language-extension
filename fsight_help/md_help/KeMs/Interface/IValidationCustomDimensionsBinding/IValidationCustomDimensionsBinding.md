# IValidationCustomDimensionsBinding


Сборка: Ms;


## Описание


Интерфейс IValidationCustomDimensionsBinding
 предназначен для работы с привязкой измерений.


## Иерархия наследования


           IValidationCustomDimensionsBinding


## Комментарии


Привязка измерений выполняется при сопоставлении источников данных,
 используемых в правиле валидации типа «Сравнение
 по выражению».


Для добавления новой привязки используйте метод [IValidationCustomMapping.AddBinding](../IValidationCustomMapping/IValidationCustomMapping.AddBinding.htm).
