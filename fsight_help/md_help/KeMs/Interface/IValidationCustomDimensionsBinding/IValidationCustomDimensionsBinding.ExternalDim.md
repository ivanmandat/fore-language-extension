# IValidationCustomDimensionsBinding.ExternalDim


## Синтаксис


ExternalDim: [IDimInstance](KeDims.chm::/interface/IDimInstance/IDimInstance.htm);


## Описание


Свойство ExternalDim определяет
 измерение из внешнего источника данных, участвующего в сопоставлении.


## Комментарии


Для получения измерения из сопоставляемого источника данных, по которому
 выполняется валидация, используйте свойство [IValidationCustomDimensionsBinding.ValidationDim](IValidationCustomDimensionsBinding.ValidationDim.htm).
