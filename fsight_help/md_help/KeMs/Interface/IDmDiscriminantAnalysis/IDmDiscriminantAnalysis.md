# IDmDiscriminantAnalysis


Сборка: Ms;


## Описание


Интерфейс IDmDiscriminantAnalysis
 предназначен для заполнения пропусков с помощью дискриминантного анализа.


## Иерархия наследования

           [IDmTemplateFillDetails](../IDmTemplateFillDetails/IDmTemplateFillDetails.htm)


           IDmDiscriminantAnalysis


## Комментарии


Данный метод выполняет поиск линейной комбинации переменных, наилучшим
 образом разделяющих (дискриминирующих) два или более класса.


Примечание.
 В данный момент заполнение пропусков с помощью дискриминантного анализа
 не реализовано.
