# IMsCustomTerm.SameDate


## Синтаксис


		SameDate: String;


## Описание


Метод SameDate возвращает дату
 начала периода для фиктивной переменной.


## Комментарии


Для получения фиктивной переменной используйте метод [IMsCustomTerm.IsPeriod](IMsCustomTerm.IsPeriod.htm).


Метод не предусматривает использования вне фиктивной переменной.
