# IMsCustomTerm.NoDate


## Синтаксис


		NoDate: String;


## Описание


Метод NoDate применяется для
 обозначения начальной или конечной даты периода идентификации модели.


## Комментарии


Метод применяется в качестве параметра метода [IMsCustomTerm.IsPeriod](IMsCustomTerm.IsPeriod.htm).
