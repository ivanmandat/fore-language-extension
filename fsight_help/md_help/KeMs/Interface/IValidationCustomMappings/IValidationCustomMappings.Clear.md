# IValidationCustomMappings.Clear


## Синтаксис


Clear;


## Описание


Метод Clear очищает коллекцию
 сопоставлений.


## Комментарии


После вызова метода свойство [IValidationCustomMappings.Count](IValidationCustomMappings.Count.htm)
 принимает значение «0».
