# IValidationCustomMappings.Item


## Синтаксис


Item(Index: Integer): [IValidationCustomMapping](../IValidationCustomMapping/IValidationCustomMapping.htm);


## Параметры


Index. Индекс сопоставления
 в коллекции.


## Описание


Свойство Item возвращает
 указанное сопоставление из коллекции.


## Комментарии


Индексация сопоставлений в коллекции сквозная и начинается с нуля. Для
 получения количества сопоставлений в коллекции используйте свойство [IValidationCustomMappings.Count](IValidationCustomMappings.Count.htm).
