# IValidationCustomMappings.FindByKey


## Синтаксис


FindByKey(ExternalRubricatorKey: Integer): [IValidationCustomMapping](../IValidationCustomMapping/IValidationCustomMapping.htm);


## Параметры


ExternalRubricatorKey. Ключ
 внешней базы данных временных рядов, сопоставление для которой нужно найти.


## Описание


Метод FindByKey выполняет поиск
 сопоставления для базы данных временных рядов с указанным ключом.


## Комментарии


Если сопоставления для указанной базы данных временных рядов не найдено,
 то метод возвращает значение Null.


Метод используется, если выполняется сопоставление баз данных временных
 рядов.
