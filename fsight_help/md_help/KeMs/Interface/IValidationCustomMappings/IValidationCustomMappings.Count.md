# IValidationCustomMappings.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает
 количество сопоставлений в коллекции.


## Комментарии


Для получения указанного сопоставления из коллекции используйте свойство
 [IValidationCustomMappings.Item](IValidationCustomMappings.Item.htm).
