# IValidationCustomMappings.AddStub


## Синтаксис


AddStub(ExternalStub: [IVariableStub](KeCubes.chm::/Interface/IVariableStub/IVariableStub.htm)):
 [IValidationCustomMapping](../IValidationCustomMapping/IValidationCustomMapping.htm);


## Параметры


ExternalStub. Внешний источник
 данных, сопоставление с которым добавляется в коллекцию.


## Описание


Метод AddStub добавляет в
 коллекцию сопоставление для указанного источника данных.


## Комментарии


В качестве источника данных может выступать база данных временных рядов
 или стандартный куб, содержащий календарное измерение.
