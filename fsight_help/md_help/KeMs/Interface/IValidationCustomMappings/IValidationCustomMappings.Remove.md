# IValidationCustomMappings.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс сопоставления
 в коллекции.


## Описание


Метод Remove удаляет указанное
 сопоставление из коллекции.


## Комментарии


Если удаление прошло успешно, то метод возвращает значение True,
 иначе - False.


Индексация сопоставлений в коллекции сквозная и начинается с нуля. Для
 получения количества сопоставлений в коллекции используйте свойство [IValidationCustomMappings.Count](IValidationCustomMappings.Count.htm).
