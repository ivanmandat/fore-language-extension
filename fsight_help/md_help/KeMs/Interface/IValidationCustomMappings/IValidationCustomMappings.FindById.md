# IValidationCustomMappings.FindById


## Синтаксис


FindById(ExternalRubricatorId: String): [IValidationCustomMapping](../IValidationCustomMapping/IValidationCustomMapping.htm);


## Параметры


ExternalRubricatorId. Идентификатор
 внешней базы данных временных рядов, сопоставление для которой нужно найти.


## Описание


Метод FindById выполняет
 поиск сопоставления для базы данных временных рядов с указанным идентификатором.


## Комментарии


Если сопоставления для указанной базы данных временных рядов не найдено,
 то метод возвращает значение Null.
