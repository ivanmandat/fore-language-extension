# IMsHodrickPrescottFilterTransform.Power


## Синтаксис


Power:Double;


## Описание


Свойство Power определяет
 значение степени, применяемой в качестве параметра сглаживания.


## Комментарии


Power должно быть неотрицательным числом, минимальное значение - нуль,
 значение по умолчанию - четыре.


Power учитывается, если [IMsHodrickPrescottFilterTransform.SmoothingParameterMode](IMsHodrickPrescottFilterTransform.SmoothingParameterMode.htm)
 имеет значение HPSmoothingParameterModeType.SetByFrequencyRule.
