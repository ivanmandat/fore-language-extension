# IMsCalculationTree


Сборка: Ms;


## Описание


Интерфейс IMsCalculationTree
 содержит свойства для работы с деревом расчёта задачи моделирования.


## Иерархия наследования


           IMsCalculationTree


## Комментарии


Дерево расчёта в иерархической форме представляет последовательность
 расчёта, а также позволяет отображать на какой узел дерева установлена
 точка останова. Оно формируется автоматически исходя из сценариев, цепочки
 расчёта и точек останова задачи. Дерево не может быть изменено пользователем
 напрямую.
