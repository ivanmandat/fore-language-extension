# IDmReports.FindByType


## Синтаксис


		FindByType(Type: [DmReportType](../../Enums/DmReportType.htm)):
		 [IDmReport](../IDmReport/IDmReport.htm);


## Параметры


Type. Тип отчёта.


## Описание


Свойство FindByType возвращает
 отчёт указанного типа.


## Комментарии


Для получения количества отчётов в коллекции используйте свойство [IDmReports.Count](IDmReports.Count.htm).
