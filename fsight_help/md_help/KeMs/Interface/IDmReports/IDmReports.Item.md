# IDmReports.Item


## Синтаксис


		Item(Index: Integer): [IDmReport](../IDmReport/IDmReport.htm);


## Параметры


Index. Индекс отчёта в коллекции.


## Описание


Свойство Item возвращает
 указанный отчёт из коллекции.


## Комментарии


Индексация отчётов в коллекции сквозная и начинается с нуля.


Для получения количества отчётов в коллекции используйте свойство [IDmReports.Count](IDmReports.Count.htm).
