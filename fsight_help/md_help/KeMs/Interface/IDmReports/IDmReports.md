# IDmReports


Сборка: Ms;


## Описание


Интерфейс IDmReports предназначен
 для работы с коллекцией отчётов с результатами интеллектуального анализа
 данных.


## Иерархия наследования


           IDmReports


## Комментарии


Для получения коллекции отчётов используйте метод [IDmMethod.Execute](../IDmMethod/IDmMethod.Execute.htm).
