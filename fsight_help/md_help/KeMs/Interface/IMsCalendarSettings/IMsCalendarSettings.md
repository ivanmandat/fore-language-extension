# IMsCalendarSettings


Сборка: Ms;


## Описание


Интерфейс IMsCalendarSettings
 предназначен для работы с параметрами календарного уровня.


## Иерархия наследования


           [ITsCalendarSettings](KeTransform.chm::/Interface/ITsCalendarSettings/ITsCalendarSettings.htm)


           IMsCalendarSettings


## Комментарии


Параметры календарного уровня для пользовательского метода расчёта возвращает
 свойство [ITimeSeries.CalendarSettings](../ITimeSeries/ITimeSeries.CalendarSettings.htm).
