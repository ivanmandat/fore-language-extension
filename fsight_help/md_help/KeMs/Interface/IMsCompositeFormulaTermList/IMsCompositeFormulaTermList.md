# IMsCompositeFormulaTermList


Сборка: Ms;


## Описание


Интерфейс IMsCompositeFormulaTermList
 содержит свойства и методы для работы с коллекцией составных термов.


## Иерархия наследования


           IMsCompositeFormulaTermList


## Комментарии


Составные термы используются в формулах при расчёте модели методом линейной
 регрессии либо при создании модели коррекции ошибок.
