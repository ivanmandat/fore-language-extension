# IMsUORuntimeControlTerm


Сборка: Ms;


## Описание


Интерфейс IMsUORuntimeControlTerm
 предназначен для работы с управляющей переменной пользовательской задачи
 оптимизации.


## Иерархия наследования


IMsUORuntimeControlTerm


## Комментарии


Управляющую переменную можно получить, используя свойство [IMsUORuntimeControlTerms.Item](../IMsUORuntimeControlTerms/IMsUORuntimeControlTerms.Item.htm)
 во время расчёта задачи.
