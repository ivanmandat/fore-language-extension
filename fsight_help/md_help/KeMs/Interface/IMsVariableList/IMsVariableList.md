# IMsVariableList


## Описание


Интерфейс IMsVariableList содержит
 свойства и методы для работы со списком переменных моделирования.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMsVariableList.Count.htm)
		 Свойство Count возвращает
		 количество переменных моделирования в списке.


		 ![](../../Property_Image.gif)
		 [Item](IMsVariableList.Item.htm)
		 Свойство Item возвращает
		 переменную моделирования.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](IMsVariableList.FindById.htm)
		 Метод FindById осуществляет
		 поиск переменной моделирования по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IMsVariableList.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск переменной моделирования по ключу.
