# IMsVariableList.Item


## Синтаксис


Item(Index: Integer): [IMsVariable](../IMsVariable/IMsVariable.htm);


## Параметры


Index. Индекс переменной моделирования, которую необходимо получить.


## Описание


Свойство Item возвращает переменную моделирования, индекс которой передается посредством параметра Index.
