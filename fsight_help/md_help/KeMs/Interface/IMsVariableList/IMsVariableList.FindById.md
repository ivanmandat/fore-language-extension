# IMsVariableList.FindById


## Синтаксис


FindById(VariableId: String): [IMsVariable](../IMsVariable/IMsVariable.htm);


## Параметры


VariableId. Идентификатор переменной моделирования.


## Описание


Метод FindById осуществляет поиск и возвращает объект, содержащий переменную моделирования. Идентификатор переменной, по которому осуществляется поиск, передается посредством параметра VariableId.
