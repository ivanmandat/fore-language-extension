# IMsVariableList.FindByKey


## Синтаксис


FindByKey(VariableKey: Integer): [IMsVariable](../IMsVariable/IMsVariable.htm);


## Параметры


VariableKey. Ключ переменной моделирования.


## Описание


Метод FindByKey осуществляет поиск и возвращает объект, содержащий переменную моделирования. Ключ переменной, по которому осуществляется поиск, передается посредством параметра VariableKey.
