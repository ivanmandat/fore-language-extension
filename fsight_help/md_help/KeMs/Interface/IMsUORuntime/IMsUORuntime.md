# IMsUORuntime


Сборка: Ms;


## Описание


Интерфейс IMsUORuntime предназначен
 для работы с контекстом расчёта пользовательской задачи оптимизации.


## Иерархия наследования


IMsUORuntime


## Комментарии


Контекст расчёта доступен в событии [IMsUserOptimizationCallback.OnExecute](../IMsUserOptimizationCallback/IMsUserOptimizationCallback.OnExecute.htm).
