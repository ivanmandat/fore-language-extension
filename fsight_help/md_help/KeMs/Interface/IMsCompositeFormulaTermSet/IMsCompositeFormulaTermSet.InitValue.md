# IMsCompositeFormulaTermSet.InitValue


## Синтаксис


		InitValue: Double;


## Описание


Свойство InitValue определяет
 начальное значение составного терма.


## Комментарии


По умолчанию начальное значение не задано и свойство возвращает значение
 Double.Nan.
