# IMsTargetVarConstraints.Add


## Синтаксис


Add: [IMsTargetVarConstraint](../IMsTargetVarConstraint/IMsTargetVarConstraint.htm);


## Описание


Метод Add добавляет ограничение
 в коллекцию.


## Комментарии


Для удаления ограничений из коллекции используйте методы [IMsTargetVarConstraints.Clear](IMsTargetVarConstraints.Clear.htm)
 и [IMsTargetVarConstraints.Remove](IMsTargetVarConstraints.Remove.htm).
