# IMsTargetVarConstraints.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс ограничения в
 коллекции.


## Описание


Метод Remove удаляет указанное
 ограничение из коллекции.


## Комментарии


Если удаление выполнено успешно, то метод возвращает значение True, иначе - False.


Индексация ограничений в коллекции сквозная и начинается с нуля. Для
 получения количества ограничений в коллекции используйте свойство [IMsTargetVarConstraints.Count](IMsTargetVarConstraints.Count.htm).
