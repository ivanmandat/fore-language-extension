# IMsTargetVarConstraints


Сборка: Ms;


## Описание


Интерфейс IMsTargetVarConstraints
 предназначен для работы с коллекцией ограничений переменной.


## Иерархия наследования


           IMsTargetVarConstraints


## Комментарии


Для получения коллекции ограничений используйте свойство [IMsTargetFormulaTerm.VarConstraints](../IMsTargetFormulaTerm/IMsTargetFormulaTerm.VarConstraints.htm).
