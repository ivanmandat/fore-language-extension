# IMsCalculationChainVariable.IsInternal


## Синтаксис


		IsInternal: Boolean;


## Описание


Свойство IsInternal определяет
 признак внутренности переменной по отношению к [цепочке
 расчёта](UiModelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_3_MetaModel/uimodelling_metamodel_2.htm).


## Комментарии


Допустимые значения:


	- True. Переменная внутренняя.
	 Для хранения данных подобные переменные используют внутреннюю БД временных
	 рядов;


	- False. Переменная из
	 внешнего источника данных.
