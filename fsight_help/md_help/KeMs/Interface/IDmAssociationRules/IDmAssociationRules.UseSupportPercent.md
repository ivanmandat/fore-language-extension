# IDmAssociationRules.UseSupportPercent


## Синтаксис


		UseSupportPercent: Boolean;


## Описание


Свойство UseSupportPercent определяет
 единицы измерения значения минимальной поддержки.


## Комментарии


Допустимые значения:


	- True. Значение минимальной
	 поддержки задано в процентах;


	- False. Значение минимальной
	 поддержки задано количеством элементов.


Значение минимальной поддержки определяется свойством [IDmAssociationRules.MinimumSupport](IDmAssociationRules.MinimumSupport.htm).
