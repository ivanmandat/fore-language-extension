# IDmAssociationRules.MinimumSupport


## Синтаксис


		MinimumSupport: Double;


## Описание


Свойство MinimumSupport определяет
 минимальную поддержку.


## Комментарии


Единицы измерения, в которых задаётся данный параметр, определяет свойство
 [IDmAssociationRules.UseSupportPercent](IDmAssociationRules.UseSupportPercent.htm).


Минимальный уровень доверия определяется свойством [IDmAssociationRules.MinimumConfidence](IDmAssociationRules.MinimumConfidence.htm).


Значения для параметров минимальная поддержка и минимальное доверие
 выбираются таким образом, чтобы ограничить количество найденных ассоциаций.
 Если поддержка имеет большое значение, то алгоритмы будут находить ассоциации,
 хорошо известные аналитикам или настолько очевидные, что нет никакого
 смысла проводить такой анализ. С другой стороны, низкое значение поддержки
 ведёт к генерации огромного количества ассоциаций, что, конечно, требует
 существенных вычислительных ресурсов. Тем не менее, большинство интересных
 ассоциаций находится именно при низком значении порога поддержки. Хотя
 слишком низкое значение поддержки ведёт к генерации статистически необоснованных
 ассоциаций.


Для примера рассмотрим утверждение: «75% транзакций, содержащих хлеб,
 также содержат молоко. 3% от общего числа всех транзакций содержат оба
 товара». 75% - это уровень доверия, 3% - это поддержка.
