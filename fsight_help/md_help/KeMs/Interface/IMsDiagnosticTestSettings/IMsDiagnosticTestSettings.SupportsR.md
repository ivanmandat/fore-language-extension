# IMsDiagnosticTestSettings.SupportsR


## Синтаксис


		SupportsR: Boolean;


## Описание


Свойство SupportsR определяет
 признак возможности расчёта диагностического теста с помощью R.


## Комментарии


Допустимые значения:


	- True. Диагностический
	 тест может быть рассчитан с помощью R. Для этого используйте свойство
	 [IMsDiagnosticTestSettings.UseR](IMsDiagnosticTestSettings.UseR.htm);


	- False. Диагностический
	 тест не может быть рассчитан с помощью R.
