# IMsDiagnosticTestSettings.RefreshIncluded


## Синтаксис


		RefreshIncluded;


## Описание


Метод RefreshIncluded обновляет
 индексы объясняющих рядов.


## Комментарии


При вызове метода происходит обновление значения свойства [IMsDiagnosticTestSettings.IncludedExplanatories](IMsDiagnosticTestSettings.IncludedExplanatories.htm)
 на основе тестируемой модели нелинейной регрессии (оценка МНК).
