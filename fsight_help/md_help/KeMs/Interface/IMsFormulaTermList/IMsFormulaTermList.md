# IMsFormulaTermList


Сборка: Ms;


## Описание


Интерфейс IMsFormulaTermList
 предназначен для работы с коллекцией элементов выражения (термов).


## Иерархия наследования


           IMsFormulaTermList


## Комментарии


Для работы с элементом из коллекции используйте интерфейс [IMsFormulaTerm](../IMsFormulaTerm/IMsFormulaTerm.htm).
