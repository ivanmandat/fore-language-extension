# IMsUORuntimeCriterionFunction


Сборка: Ms;


## Описание


Интерфейс IMsUORuntimeCriterionFunction
 предназначен для работы с целевой функцией пользовательской задачи оптимизации.


## Иерархия наследования


IMsUORuntimeCriterionFunction


## Комментарии


Целевую функцию можно получить, используя свойство [IMsUORuntime.CriterionFunction](../IMsUORuntime/IMsUORuntime.CriterionFunction.htm)
 во время расчёта задачи.
