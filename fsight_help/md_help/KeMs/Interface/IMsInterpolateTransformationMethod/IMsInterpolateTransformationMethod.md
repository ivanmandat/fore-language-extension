# IMsInterpolateTransformationMethod


## Описание


Интерфейс IMsInterpolateTransformationMethod
 содержит свойства для определения параметров преобразования терма методом
 «Интерпоялция». Данный метод осуществляет дезагрегацию данных с верхнего
 уровня на нижний. Например, данные квартальной динамики необходимо распределить
 по месяцам.
