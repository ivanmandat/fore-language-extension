# IMsPDLCoefficients


Сборка: Ms;


## Описание


Интерфейс IMsPDLCoefficients
 предназначен для работы с коэффициентами лаговой переменной.


## Иерархия наследования


           [ICoefficients](StatLib.chm::/Interface/ICoefficients/ICoefficients.htm)


           IMsPDLCoefficients


## Комментарии


Перед получением коэффициентов они должны быть идентифицированы. Для
 этого используйте [IMsStochasticMethod.Identify](../IMsStochasticMethod/IMsStochasticMethod.Identify.htm).
