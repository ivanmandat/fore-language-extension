# IValidationCrossDatabaseMappings.FindByTargetId


## Синтаксис


FindByTargetId(Id: String): [IValidationCrossDatabaseMapping](../IValidationCrossDatabaseMapping/IValidationCrossDatabaseMapping.htm);


## Параметры


Id. Идентификатор источника
 данных.


## Описание


Метод FindByTargetId выполняет
 поиск привязки по идентификатору измерения сравниваемого источника данных.


## Комментарии


Если привязка не найдена, то метод возвращает значение Null.
