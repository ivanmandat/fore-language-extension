# IValidationCrossDatabaseMappings.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс элемента коллекции.


## Описание


Метод Remove выполняет удаление элемента из коллекции по его индексу.


## Комментарии


Если после выполнения метод вернул значение True, то удаление прошло успешно. Индексация элементов коллекции сквозная и начинается с нуля. Число элементов в коллекции возвращает свойство [IValidationCrossDatabaseMappings.Count](IValidationCrossDatabaseMappings.Count.htm).
