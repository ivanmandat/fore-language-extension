# IValidationCrossDatabaseMappings.FindByTargetKey


## Синтаксис


FindByTargetKey(Key: Integer): [IValidationCrossDatabaseMapping](../IValidationCrossDatabaseMapping/IValidationCrossDatabaseMapping.htm);


## Параметры


Key. Ключ измерения.


## Описание


Метод FindByTargetKey
 выполняет поиск привязки по ключу измерения сравниваемого источника данных.


## Комментарии


Если привязка не найдена, то метод возвращает значение Null.
