# IValidationCrossDatabaseMappings.Add


## Синтаксис


Add: [IValidationCrossDatabaseMapping](../IValidationCrossDatabaseMapping/IValidationCrossDatabaseMapping.htm);


## Описание


Метод Add добавляет элемент
 в коллекцию.


## Комментарии


После выполнения метод возвращает добавленную привязку измерений, значение
 [IValidationCrossDatabaseMappings.Count](IValidationCrossDatabaseMappings.Count.htm)
 увеличивается на единицу. Для удаления элемента используйте метод [IValidationCrossDatabaseMappings.Remove](IValidationCrossDatabaseMappings.Remove.htm).
