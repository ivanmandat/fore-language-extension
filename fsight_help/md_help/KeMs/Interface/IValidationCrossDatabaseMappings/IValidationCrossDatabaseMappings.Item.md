# IValidationCrossDatabaseMappings.Item


## Синтаксис


Item(Index: Integer): [IValidationCrossDatabaseMapping](../IValidationCrossDatabaseMapping/IValidationCrossDatabaseMapping.htm);


## Параметры


Index. Индекс элемента коллекции.


## Описание


Свойство Item возвращает элемент коллекции по его индексу.


## Комментарии


Индексация элементов коллекции сквозная и начинается с нуля. Число элементов в коллекции возвращает свойство [IValidationCrossDatabaseMappings.Count](IValidationCrossDatabaseMappings.Count.htm).
