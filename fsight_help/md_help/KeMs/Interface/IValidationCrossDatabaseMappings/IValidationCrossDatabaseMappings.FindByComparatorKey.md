# IValidationCrossDatabaseMappings.FindByComparatorKey


## Синтаксис


FindByComparatorKey(Key: Integer): [IValidationCrossDatabaseMapping](../IValidationCrossDatabaseMapping/IValidationCrossDatabaseMapping.htm);


## Параметры


Key. Ключ измерения.


## Описание


Метод FindByComparatorKey
 выполняет поиск привязки по ключу измерения источника данных, с которым
 выполняется сравнение.


## Комментарии


Если привязка не найдена, то метод возвращает значение Null.
