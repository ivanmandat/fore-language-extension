# IValidationCrossDatabaseMappings.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество элементов в коллекции.


## Комментарии


Для добавления элемента в коллекцию используйте метод [IValidationCrossDatabaseMappings.Add](IValidationCrossDatabaseMappings.Add.htm), для удаления элемента - метод [IValidationCrossDatabaseMappings.Remove](IValidationCrossDatabaseMappings.Remove.htm), для удаления всех элементов - метод [IValidationCrossDatabaseMappings.Clear](IValidationCrossDatabaseMappings.Clear.htm).
