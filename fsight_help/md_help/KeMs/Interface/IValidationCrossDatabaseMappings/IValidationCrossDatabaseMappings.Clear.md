# IValidationCrossDatabaseMappings.Clear


## Синтаксис


Clear;


## Описание


Метод Clear удаляет все элементы из коллекции.


## Комментарии


После выполнения метода значение [IValidationCrossDatabaseMappings.Count](IValidationCrossDatabaseMappings.Count.htm) будет равно нулю.


Для удаления одного элемента используйте метод [IValidationCrossDatabaseMappings.Remove](IValidationCrossDatabaseMappings.Remove.htm).
