# IValidationCrossDatabaseMappings.FindByComparatorId


## Синтаксис


FindByComparatorId(Id: String): [IValidationCrossDatabaseMapping](../IValidationCrossDatabaseMapping/IValidationCrossDatabaseMapping.htm);


## Параметры


Id. Идентификатор измерения.


## Описание


Метод FindByComparatorId выполняет
 поиск привязки по идентификатору измерения источника данных, с которым
 выполняется сравнение.


## Комментарии


Если привязка не найдена, то метод возвращает значение Null.
