# IDmMethod


Сборка: Ms;


## Описание


Интерфейс IDmMethod предназначен
 для работы с методами интеллектуального анализа данных.


## Иерархия наследования


           IDmMethod


## Комментарии


Интеллектуальный анализ данных (Data Mining) - это процесс выявления
 скрытых фактов и взаимосвязей в больших массивах данных. Полученные данные
 могут использоваться для принятия решений в различных сферах человеческой
 деятельности.
