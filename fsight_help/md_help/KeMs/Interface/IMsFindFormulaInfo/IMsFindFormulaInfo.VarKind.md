# IMsFindFormulaInfo.VarKind


## Синтаксис


		VarKind: [MsVariableKind](../../Enums/MsVariableKind.htm);


## Описание


Свойство VarKind определяет
 тип переменных, к которым должен относиться [указанный
 источник](IMsFindFormulaInfo.Stub.htm).


## Комментарии


По умолчанию поиск осуществляется по всем переменным.
