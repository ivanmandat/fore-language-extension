# IMsFindFormulaInfo.ParametrizedDims


## Синтаксис


		ParametrizedDims: [IMsParametrizedDimensions](../IMsParametrizedDimensions/IMsParametrizedDimensions.htm);


## Описание


Свойство ParametrizedDims возвращает
 настройки параметрических измерений.


## Комментарии


Свойство актуально, если свойству [IMsFindFormulaInfo.UseParamDims](IMsFindFormulaInfo.UseParamDims.htm)
 установлено значение True. В настройках
 параметрических измерений определите параметр, отметка которого будет
 управлять отметкой измерения.

Использование свойства приведено в примере для метода [IMsFindFormulaInfo.UseParamDims](IMsFindFormulaInfo.UseParamDims.htm).
