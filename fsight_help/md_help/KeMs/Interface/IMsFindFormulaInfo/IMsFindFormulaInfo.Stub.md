# IMsFindFormulaInfo.Stub


## Синтаксис


		Stub: [IVariableStub](KeCubes.chm::/Interface/IVariableStub/IVariableStub.htm);


## Описание


Свойство Stub определяет источник
 данных, на котором должны быть основаны искомые термы в формулах.


## Комментарии


В свойстве [IMsFindFormulaInfo.Selection](IMsFindFormulaInfo.Selection.htm)
 могут быть указаны отдельные элементы измерений источника, которые будут
 искаться в термах формул.


## Пример


Использование свойства приведено в примере для метода [IMsMetaModel.FindFormulas](../IMsMetaModel/IMsMetaModel.FindFormulas.htm).
