# IMsBranchCase.AutoName


## Синтаксис


		AutoName: Boolean;


## Описание


Свойство AutoName определяет
 признак автоматической генерации наименования для ветки условия в [цепочке
 расчёта](UiModelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_3_MetaModel/uimodelling_metamodel_2.htm) метамодели.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Наименование генерируется автоматически;


	- False. Наименование
	 задаётся пользователем вручную. Используйте свойство [IMsBranchCase.Name](IMsBranchCase.Name.htm).


Правила автоматической генерации наименований:


		 Тип условия
		 Наименование, генерируемое автоматически


		 Наименования параметров условия
		 «Если {наименование параметра условия}».


		 Наименования параметров веток условия
		 «Если {наименование параметра ветки условия}».


		 Наименования групп
		 «Если {наименование параметра условия}» или {наименование параметра
		 ветки условия} в группе {наименование группы}».


		 Список элементов, формирующих отмету
		 «Если {наименование параметра условия}» или {наименование параметра
		 ветки условия} в отметке из N элементов <список элементов из
		 отметки>».

Где N - количество элементов в отметке.

Список элементов из отметки формируется следующим
		 образом: если N>6, то <первые три элемента, …, последние
		 три элемента>; если N<=6, то <все элементы>.


		 Наименования параметров, формирующих отметку
		 «Если {наименование параметра условия}» или {наименование параметра
		 ветки условия} в отметке по параметру {наименование параметра,
		 формирующего отметку}».


		 Условие по выражению (для типа [IMsBranchConditionExpression](../IMsBranchConditionExpression/IMsBranchConditionExpression.htm))
		 «Если <условие>».


		 Непустые ряды (для типа [IMsBranchConditionNotEmpty](../IMsBranchConditionNotEmpty/IMsBranchConditionNotEmpty.htm))


			- «Если непустой ряд по параметру {наименование
			 параметра условия}» или {наименование параметра ветки условия};


			- «Если непустой ряд в отметке из N элементов <список
			 элементов из отметки>».


		Где N - количество элементов в отметке.

Список элементов из отметки формируется следующим
		 образом: если N>6, то <первые три элемента, …, последние
		 три элемента>; если N<=6, то <все элементы>.


		 [Ветка
		 условия](../IMsCalculationChainBranch/IMsCalculationChainBranch.DefaultContents.htm), выполняемая если не выполнилась ни одна ветка
		 из коллекции [IMsCalculationChainBranch.CaseList](../IMsCalculationChainBranch/IMsCalculationChainBranch.CaseList.htm).
		 «Иначе».
