# IMsBranchCase.Name


## Синтаксис


		Name: String;


## Описание


Свойство Name определяет наименование
 ветки условия в [цепочке
 расчёта](UiModelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_3_MetaModel/uimodelling_metamodel_2.htm) метамодели.


## Комментарии


Если для ветки условия пользователь задал наименование, то свойство
 [IMsBranchCase.AutoName](IMsBranchCase.AutoName.htm)
 примет значение False.
