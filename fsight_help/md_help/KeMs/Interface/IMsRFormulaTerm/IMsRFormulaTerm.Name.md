# IMsRFormulaTerm.Name


## Синтаксис


Name: String;


## Описание


Свойство Name определяет
 наименование элемента выражения.


## Комментарии


Наименование элемента используется в выражении, по которому рассчитывается
 модель «R». Для задания выражения
 используйте свойство [IMsRTransform.Code](../IMsRTransform/IMsRTransform.Code.htm).
