# IMsDiagnosticTestList.Count


## Синтаксис


		Count: Integer;


## Описание


Свойство Count возвращает число
 тестов в наборе.


## Комментарии


Ёмкость коллекции - это количество элементов, которое может содержать
 коллекция. Значение свойства - это количество элементов, которое в действительности
 содержится в коллекции.


Ёмкость всегда больше или равна значению свойства. Если при добавлении
 элементов в коллекцию значение свойства превышает ёмкость, то она автоматически
 увеличивается.
