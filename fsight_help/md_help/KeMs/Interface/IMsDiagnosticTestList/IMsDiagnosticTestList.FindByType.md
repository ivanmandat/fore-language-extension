# IMsDiagnosticTestList.FindByType


## Синтаксис


		FindByType(Type: [MsDiagnosticTestType](../../Enums/MsDiagnosticTestType.htm)):
		 [IMsDiagnosticTest](../IMsDiagnosticTest/IMsDiagnosticTest.htm);


## Параметры


Type. Тип теста.


## Описание


Свойство FindByType возвращает
 тест указанного типа.


## Комментарии


Свойство не может вернуть значение Null.
