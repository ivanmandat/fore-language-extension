# IMsPooledModelTransform.MissingData


## Синтаксис


MissingData: [IMissingData](StatLib.chm::/Interface/IMissingData/IMissingData.htm);


## Описание


Свойство MissingData возвращает
 объект, содержащий параметры обработки пропусков исходного ряда.


## Комментарии


По умолчанию используется метод обработки пропусков MissingDataMethod.Casewise,
 т.е. пустые значения исключаются из ряда и расчёт ведется без их учета.
