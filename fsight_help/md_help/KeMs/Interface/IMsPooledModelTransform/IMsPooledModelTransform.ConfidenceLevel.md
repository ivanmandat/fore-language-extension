# IMsPooledModelTransform.ConfidenceLevel


## Синтаксис


ConfidenceLevel: Double;


## Описание


Свойство ConfidenceLevel определяет значимость доверительных границ прогнозного ряда.


## Комментарии


Значение ConfidenceLevel должно входить в диапазон (0, 1). Значение по умолчанию - «0,95».
