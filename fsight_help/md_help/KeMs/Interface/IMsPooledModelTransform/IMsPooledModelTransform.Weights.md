# IMsPooledModelTransform.Weights


## Синтаксис


Weights: [IMsFormulaTermSet](../IMsFormulaTermSet/IMsFormulaTermSet.htm);


## Описание


Свойство Weights возвращает веса, используемые при расчете модели.


## Комментарии


По умолчанию веса при расчете модели не используются.
