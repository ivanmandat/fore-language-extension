# IMsPooledModelTransform.ResidualsTermSet


## Синтаксис


ResidualsTermSet: [IMsFormulaTermSet](../IMsFormulaTermSet/IMsFormulaTermSet.htm);


## Описание


Свойство ResidualsTermSet возвращает параметры переменной, в которую будут выгружены данные ряда остатков после расчета модели.


## Комментарии


По умолчанию переменная не задана. Размерности данной и объясняемой переменной должны совпадать. Параметры объясняемой переменной возвращает свойство [IMsPooledModelTransform.Explained](IMsPooledModelTransform.Explained.htm).


Параметры, возвращаемые ResidualsTermSet, представлены в виде набора термов.
