# IMsPooledModelTransform.ConstantValue


## Синтаксис


ConstantValue: Double;


## Описание


Свойство ConstantValue определяет значение константы для модели.


## Комментарии


ConstantValue учитывается, если [IMsPooledModelTransform.ConstantMode](IMsPooledModelTransform.ConstantMode.htm) имеет значение InterceptMode.ManualEstimate.
