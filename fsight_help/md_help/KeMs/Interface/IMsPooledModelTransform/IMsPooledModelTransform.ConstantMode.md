# IMsPooledModelTransform.ConstantMode


## Синтаксис


ConstantMode: [InterceptMode](StatLib.chm::/Enums/InterceptMode.htm);


## Описание


Свойство ConstantMode определяет режим задания константы для модели.


## Комментарии


ConstantMode = InterceptMode.None. Значение по умолчанию. Константа в модели не используется.


ConstantMode = InterceptMode.ManualEstimate. Значение константы задается пользователем с использованием свойства [IMsPooledModelTransform.ConstantValue](IMsPooledModelTransform.ConstantValue.htm).
