# IMsPooledModelTransform.CrossSection


## Синтаксис


CrossSection: [PooledModelCrossSectionType](StatLib.chm::/Enums/PooledModelCrossSectionType.htm);


## Описание


Свойство CrossSection определяет тип модели регрессии на панельных данных.


## Комментарии


Значение по умолчанию PooledModelCrossSectionType.None, т.е. рассматривается объединенная модель регрессии (модель без эффектов).
