# IDmReport


Сборка: Ms;


## Описание


Интерфейс IDmReport предназначен
 для работы с отчётом, содержащим результаты интеллектуального анализа
 данных.


## Иерархия наследования


           IDmReport


## Комментарии


Для работы со всеми отчётами, полученными в результате интеллектуального
 анализа, используйте интерфейс [IDmReports](../IDmReports/IDmReports.htm).
