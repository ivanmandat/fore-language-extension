# IMsVariableTestDataList


Сборка: Ms;


## Описание


Интерфейс IMsVariableTestDataList
 предназначен для работы с коллекцией переменных, для которых рассчитывается
 описательная статистика.


## Иерархия наследования


           IMsVariableTestDataList


## Комментарии


Для получения коллекции переменных, для которых рассчитывается описательная
 статистика, используйте свойство [IMsVariableTestSettings.IncludedSources](../IMsVariableTestSettings/IMsVariableTestSettings.IncludedSources.htm).
