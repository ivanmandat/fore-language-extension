# IMsVariableTestDataList.Add


## Синтаксис


Add(Slice: [IMsFormulaTransformSlice](../IMsFormulaTransformSlice/IMsFormulaTransformSlice.htm); [EntryKey: Integer = -1]): [IMsVariableTestData](../IMsVariableTestData/IMsVariableTestData.htm);


## Параметры


Slice. Срез добавляемой переменной;


EntryKey. Ключ добавляемой переменной.


## Описание


Метод Add добавляет переменную в коллекцию.


## Комментарии


Для удаления переменной из коллекции используйте метод [IMsVariableTestDataList.Remove](IMsVariableTestDataList.Remove.htm).
