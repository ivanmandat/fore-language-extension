# IMsVariableTestDataList.Clear


## Синтаксис


Clear;


## Описание


Метод Clear очищает коллекцию переменных.


## Комментарии


После выполнения метода свойство [IMsVariableTestDataList.Count](IMsVariableTestDataList.Count.htm) принимает значение «0».


Для удаления одной переменной из коллекции используйте метод [IMsVariableTestDataList.Remove](IMsVariableTestDataList.Remove.htm).
