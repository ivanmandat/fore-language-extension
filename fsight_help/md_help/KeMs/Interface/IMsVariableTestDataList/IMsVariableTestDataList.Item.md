# IMsVariableTestDataList.Item


## Синтаксис


Item(Index: Integer): [IMsVariableTestData](../IMsVariableTestData/IMsVariableTestData.htm);


## Параметры


Index. Индекс переменной в коллекции.


## Описание


Свойство Item возвращает указанную переменную из коллекции.


## Комментарии


Индексация переменных сквозная и начинается с нуля. Для получения количества переменных в коллекции используйте свойство [IMsVariableTestDataList.Count](IMsVariableTestDataList.Count.htm).
