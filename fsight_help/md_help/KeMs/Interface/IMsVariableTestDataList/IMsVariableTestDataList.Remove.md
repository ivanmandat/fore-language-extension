# IMsVariableTestDataList.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемой переменной.


## Описание


Метод Remove удаляет указанную переменную из коллекции.


## Комментарии


Индексация переменных сквозная и начинается с нуля. Для получения количества переменных в коллекции используйте свойство [IMsVariableTestDataList.Count](IMsVariableTestDataList.Count.htm).


Метод возвращает значение True, если удаление выполнено успешно. В противном случае метод возвращает значение False.
