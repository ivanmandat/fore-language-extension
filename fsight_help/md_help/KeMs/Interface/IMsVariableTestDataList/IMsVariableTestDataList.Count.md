# IMsVariableTestDataList.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество переменных в коллекции.


## Комментарии


Для получения переменной из коллекции используйте свойство [IMsVariableTestDataList.Item](IMsVariableTestDataList.Item.htm).
