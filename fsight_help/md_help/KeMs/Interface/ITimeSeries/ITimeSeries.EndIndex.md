# ITimeSeries.EndIndex


## Синтаксис


EndIndex: Integer;


## Описание


Свойство EndIndex
 возвращает конечный индекс ряда.


## Комментарии


Для получения количества дат расчета ряда используйте формулу: EndIndex - [ITimeSeries.StartIndex](ITimeSeries.StartIndex.htm) + 1.
