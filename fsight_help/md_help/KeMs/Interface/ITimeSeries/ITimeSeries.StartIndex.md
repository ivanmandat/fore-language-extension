# ITimeSeries.StartIndex


## Синтаксис


StartIndex: Integer;


## Описание


Свойство StartIndex
 возвращает начальный индекс ряда.


## Комментарии


Для получения количества дат расчета ряда используйте формулу: [ITimeSeries.EndIndex](ITimeSeries.EndIndex.htm) - StartIndex + 1.
