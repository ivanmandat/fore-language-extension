# IMsModelSet.FindByKey


## Синтаксис


FindByKey(ModelKey: Integer): [IMsModel](../IMsModel/IMsModel.htm);


## Параметры


ModelKey. Ключ модели.


## Описание


Метод FindByKey осуществляет
 поиск модели по ключу и возвращает её структуру в случае удачного поиска.


## Комментарии


Метод возвращает значение Null,
 если модель отсутствует в коллекции.
