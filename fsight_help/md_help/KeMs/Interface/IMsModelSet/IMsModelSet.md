# IMsModelSet


Сборка: Ms;


## Описание


Интерфейс IMsModelSet предназначен
 для работ коллекцией моделей контейнера моделирования.


## Иерархия наследования


IMsModelSet


## Комментарии


Коллекцию моделей можно получить, используя метод [IMsCalculationChainEntries.FindByCoord](../IMsCalculationChainEntries/IMsCalculationChainEntries.FindByCoord.htm).

		 Имя метода
