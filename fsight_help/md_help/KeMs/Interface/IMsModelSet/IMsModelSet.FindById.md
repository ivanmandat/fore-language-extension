# IMsModelSet.FindById


## Синтаксис


FindById(ModelId: String): [IMsModel](../IMsModel/IMsModel.htm);


## Параметры


ModelId. Идентификатор модели.


## Описание


Метод FindById осуществляет
 поиск модели по идентификатору и возвращает её структуру в случае удачного
 поиска.


## Комментарии


Метод возвращает значение Null,
 если модель отсутствует в коллекции.
