# IMsVariableLoadSettings.ScenarioIncluded


## Синтаксис


ScenarioIncluded(ScenarioKey: Integer): Boolean;


## Параметры


ScenarioKey. Ключ сценария,
 по которому будут загружены данные в переменную моделирования. Если необходимо
 осуществить загрузку по измерению «Факт» - задается значение -1.


## Описание


Свойство ScenarioIncluded определяет,
 будет ли осуществляться загрузка данных по измерению.
