# IMsVariableLoadSettings


## Описание


Интерфейс IMsVariableLoadSettings
 содержит свойства и методы, определяющие параметры загрузки данных в переменную
 с использованием формулы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DimensionFix](IMsVariableLoadSettings.DimensionFix.htm)
		 Свойство DimensionFix
		 возвращает коллекцию фиксированных измерений переменной моделирования.


		 ![](../../Property_Image.gif)
		 [Period](IMsVariableLoadSettings.Period.htm)
		 Свойство Period возвращает
		 период, в рамках которого будет осуществляться загрузка данных
		 в переменную.


		 ![](../../Property_Image.gif)
		 [ScenarioIncluded](IMsVariableLoadSettings.ScenarioIncluded.htm)
		 Свойство ScenarioIncluded
		 определяет, будет ли осуществляться загрузка данных по измерению.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CreateCalculation](IMsVariableLoadSettings.CreateCalculation.htm)
		 Метод CreateCalculation
		 создает объект, содержащий настройки, используемые при расчете
		 значений переменной.
