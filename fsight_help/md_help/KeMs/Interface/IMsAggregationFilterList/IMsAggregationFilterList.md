# IMsAggregationFilterList


Сборка: Ms;


## Описание


Интерфейс IMsAggregationFilterList
 содержит свойства и методы для работы с коллекцией фильтров агрегации.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMsAggregationFilterList.Count.htm)
		 Свойство Count возвращает
		 количество фильтров в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMsAggregationFilterList.Item.htm)
		 Свойство Item возвращает
		 фильтр из коллекции по его индексу.
