# IMsFormula


Сборка: Ms;


## Описание


Интерфейс IMsFormula предназначен
 для работы с методом, используемым для расчёта модели.


## Иерархия наследования


           IMsFormula


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [FreeDimensions](IMsFormula.FreeDimensions.htm)
		 Свойство FreeDimensions
		 возвращает коллекцию измерений, по которым разрешена множественная
		 отметка.


		 ![](../../Property_Image.gif)
		 [Kind](IMsFormula.Kind.htm)
		 Свойство Kind определяет
		 метод, используемый для расчёта модели.


		 ![](../../Property_Image.gif)
		 [Level](IMsFormula.Level.htm)
		 Свойство Level определяет
		 шаг расчёта, используемый при расчёте модели.


		 ![](../../Property_Image.gif)
		 [LevelParam](IMsFormula.LevelParam.htm)
		 Свойство LevelParam
		 определяет параметр, задающий динамику расчёта метода.


		 ![](../../Property_Image.gif)
		 [Method](IMsFormula.Method.htm)
		 Свойство Method возвращает
		 параметры метода, используемого для расчёта модели.
