# IMsObjectParamValues


Сборка: Ms;


## Описание


Интерфейс IMsObjectParamValues
 для работы с коллекцией связей параметров моделирования со значениями
 параметров объекта репозитория для его открытия.


## Иерархия наследования


           IMsObjectParamValues


## Комментарии


Для работы с коллекцией связей параметров моделирования с несколькими
 объектами репозитория используйте интерфейс [IMsObjectParams](../IMsObjectParams/IMsObjectParams.htm).
