# IMsCalculationChainTarget.TargetProblem


## Синтаксис


		TargetProblem: [IMsTargetProblem](../IMsTargetProblem/IMsTargetProblem.htm);


## Описание


Свойство TargetProblem определяет
 целевую задачу, которой принадлежит целевая функция.


## Комментарии


По умолчанию целевая функция принадлежит целевой задаче, указанной при
 вызове метода [IMsCalculationChainEntries.AddTargetProblem](../IMsCalculationChainEntries/IMsCalculationChainEntries.AddTargetProblem.htm).
