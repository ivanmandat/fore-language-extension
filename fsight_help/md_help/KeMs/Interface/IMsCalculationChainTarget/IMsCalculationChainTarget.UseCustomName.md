# IMsCalculationChainTarget.UseCustomName


## Синтаксис


		UseCustomName: Boolean;


## Описание


Свойство UseCustomName
 определяет признак использования пользовательского наименования целевой
 функции.


## Комментарии


Допустимые значения:


	- True. Наименование целевой
	 функции задаётся пользователем;


	- False. Наименование
	 целевой функции формируется автоматически.
