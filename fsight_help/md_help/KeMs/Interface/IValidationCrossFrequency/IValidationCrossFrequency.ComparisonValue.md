# IValidationCrossFrequency.ComparisonValue


## Синтаксис


ComparisonValue: [IValidationComparisonValue](../IValidationComparisonValue/IValidationComparisonValue.htm);


## Описание


Свойство ComparisonValue возвращает
 параметры сравнения.


## Комментарии


Значения для сравнения могут быть как абсолютными, так и относительными.
 Это определяется свойством [IValidationComparisonValue.Percentage](../IValidationComparisonValue/IValidationComparisonValue.Percentage.htm).
