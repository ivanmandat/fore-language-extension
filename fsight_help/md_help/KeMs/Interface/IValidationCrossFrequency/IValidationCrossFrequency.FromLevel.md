# IValidationCrossFrequency.FromLevel


## Синтаксис


FromLevel: [DimCalendarLevel](KeDims.chm::/Enums/DimCalendarLevel.htm);


## Описание


Свойство FromLevel определяет
 сравниваемый уровень.


## Комментарии


Сравниваемый уровень должен иметь меньшую календарную динамику, чем
 динамика правила валидации, заданная свойством [IValidationFilter.Level](../IValidationFilter/IValidationFilter.Level.htm).
