# IMsCalculationChainGroup.Period


## Синтаксис


		Period: [IMsModelPeriod](../IMsModelPeriod/IMsModelPeriod.htm);


## Описание


Свойство Period возвращает параметры
 периода расчёта группы.


## Комментарии


Данное свойство позволяет задать следующие параметры периода расчёта
 группы:


		 Дата начала периода идентификации
		 Дата окончания периода идентификации
		 Дата окончания периода прогнозирования


		 Определённая
		 дата (по умолчанию).
		 Определённая дата (по умолчанию).
		 Определённая дата (по умолчанию).


		 Сегодня плюс/минус
		 k
		 точек.
		 Сегодня плюс/минус k точек.
		 Сегодня плюс/минус k точек.


		 Установка значения
		 по параметру.
		 Установка значения по параметру.
		 Установка значения по параметру.


Если свойство Period используется
 для задания периода расчёта ветки условия, реализованной интерфейсом [IMsCalculationChainBranch](../IMsCalculationChainBranch/IMsCalculationChainBranch.htm), то доступны
 следующие возможности:


		 Дата начала периода идентификации
		 Дата окончания периода идентификации
		 Дата окончания периода прогнозирования


		 Определённая
		 дата (по умолчанию).
		 Определённая дата (по умолчанию).
		 Определённая дата (по умолчанию).


		 Сегодня плюс/минус
		 k
		 точек.
		 Сегодня плюс/минус k точек.
		 Сегодня плюс/минус k точек.


		 Установка значения
		 по параметру.
		 Установка значения по параметру.
		 Установка значения по параметру.


		 Конец идентификации
		 минус k
		 точек.
		 Начало идентификации плюс k
		 точек.
		 Конец идентификации плюс k
		 точек.
