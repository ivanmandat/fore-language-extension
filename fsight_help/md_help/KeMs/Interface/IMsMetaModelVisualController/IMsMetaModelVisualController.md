# IMsMetaModelVisualController


Сборка: Ms;


## Описание


Интерфейс IMsMetaModelVisualController
 используется для поддержки визуального построения метамодели.


## Иерархия наследования


           IMsMetaModelVisualController


## Комментарии


Параметры визуального построения метамодели возвращает свойство [IMsMetaModel.VisualController](../IMsMetaModel/IMsMetaModel.VisualController.htm).
