# IMsMetaModelVisualController.ChainWorkbook


## Синтаксис


ChainWorkbook: [IMetabaseObject](KeSom.chm::/Interface/IMetabaseObject/IMetabaseObject.htm);


## Описание


Свойство ChainWorkbook возвращает внутреннюю рабочую книгу цепочки расчета метамодели.


## Комментарии


Внутреннюю рабочую книгу для внутренних переменных метамодели возвращает свойство [IMsMetaModelVisualController.VariableWorkbook](IMsMetaModelVisualController.VariableWorkbook.htm).
