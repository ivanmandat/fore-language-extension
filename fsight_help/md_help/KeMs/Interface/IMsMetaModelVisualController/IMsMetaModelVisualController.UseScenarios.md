# IMsMetaModelVisualController.UseScenarios


## Синтаксис


UseScenarios: Boolean;


## Описание


Свойство UseScenarios определяет, используется ли сценарное моделирование.


## Комментарии


Допустимые значения:


-
True. Используется сценарное моделирование. Рассматривается несколько альтернативных вариантов развития модели;


-
False. Значение по умолчанию. Сценарное моделирование не используется. Рассматривается единственный вариант развития модели.


Для корректной работы с UseScenarios установите значение свойства [IMsMetaModelVisualController.Problem](IMsMetaModelVisualController.Problem.htm).


Примечание. Для сохранения значения свойства UseScenarios необходимо сохранить задачу, указанную в [IMsMetaModelVisualController.Problem](IMsMetaModelVisualController.Problem.htm).
