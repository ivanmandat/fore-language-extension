# IMsMetaModelVisualController.UpdateVariables


## Синтаксис


UpdateVariables;


## Описание


Метод UpdateVariables обновляет внутренние переменные, содержащиеся в метамодели.


## Комментарии


Для создания внутренней переменной используйте метод [IMsMetaModelVisualController.CreateVariable](IMsMetaModelVisualController.CreateVariable.htm).
