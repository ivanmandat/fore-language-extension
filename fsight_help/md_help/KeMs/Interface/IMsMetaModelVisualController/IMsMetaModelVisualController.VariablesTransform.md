# IMsMetaModelVisualController.VariablesTransform


## Синтаксис


VariablesTransform: [IMsFormulaTransform](../IMsFormulaTransform/IMsFormulaTransform.htm);


## Описание


Свойство VariablesTransform возвращает параметры внутренней модели.


## Комментарии


Внутреннюю базу данных временных рядов для метамодели возвращает свойство [IMsMetaModelVisualController.VariableRubricator](IMsMetaModelVisualController.VariableRubricator.htm).
