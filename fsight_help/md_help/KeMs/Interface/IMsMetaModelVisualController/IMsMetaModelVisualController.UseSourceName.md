# IMsMetaModelVisualController.UseSourceName


## Синтаксис


UseSourceName: Boolean;


## Описание


Свойство UseSourceName
 определяет, отображать ли наименование источника данных в наименовании
 переменных.


## Комментарии


Допустимые значения:


	- True. Наименование источника
	 данных будет отображаться в наименованиях переменных;


	- False. Наименование
	 источника данных не будет отображаться в наименованиях переменных.


Примечание.
 Если пользователь изменил имя переменной, то значение свойства UseSourceName
 не влияет на её наименование.
