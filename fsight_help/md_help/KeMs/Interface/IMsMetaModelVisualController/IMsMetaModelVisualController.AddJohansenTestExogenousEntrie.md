# IMsMetaModelVisualController.AddJohansenTestExogenousEntrie


## Синтаксис


AddJohansenTestExogenousEntrie(EntryKey: Integer): [IMsVariableTestData](../IMsVariableTestData/IMsVariableTestData.htm);


## Параметры


EntryKey. Ключ элемента цепочки расчёта метамодели, который является переменной.


## Описание


Метод AddJohansenTestExogenousEntrie добавляет экзогенную переменную в расчёт теста Йохансена.


## Комментарии


Для получения параметров расчёта теста Йохансена используйте свойство [IMsMetaModelVisualController.JohansenTest](IMsMetaModelVisualController.JohansenTest.htm).
