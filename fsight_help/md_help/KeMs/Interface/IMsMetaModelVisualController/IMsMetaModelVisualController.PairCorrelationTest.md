## Описание


Свойство PairCorrelationTest возвращает параметры расчёта парных коэффициентов корреляции.


## Комментарии


Для настройки параметров расчёта используйте интерфейс [IMsPairCorrelationTestSettings](../IMsPairCorrelationTestSettings/IMsPairCorrelationTestSettings.htm).
