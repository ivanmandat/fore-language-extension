# IMsValueAtRiskTransform.ZeroMean


## Синтаксис


ZeroMean: Boolean;


## Описание


Свойство ZeroMean определяет,
 использовать ли гипотезу о нулевом среднем.


## Комментарии


Допустимые значения:


	- True. Использовать гипотезу
	 о нулевом среднем;


	- False. Не использовать
	 гипотезу о нулевом среднем;
