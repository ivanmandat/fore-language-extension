# IMsValueAtRiskTransform.OrganizationsDimension


## Синтаксис


OrganizationsDimension: [IDimensionModel](KeDims.chm::/interface/IDimensionModel/IDimensionModel.htm);


## Описание


Свойство OrganizationsDimension определяет
 измерение организаций.


## Комментарии


Данное измерение должно входить в:


	- переменную «портфеля»: [IMsValueAtRiskTransform.Portfolio](IMsValueAtRiskTransform.Portfolio.htm);


	- переменную, в которую будут помещены данные расчёта модели:
	 [IMsValueAtRiskTransform.ValueAtRisk](IMsValueAtRiskTransform.ValueAtRisk.htm).

Использование свойства приведено в [IMsValueAtRiskTransform.Portfolio](IMsValueAtRiskTransform.Portfolio.htm).
