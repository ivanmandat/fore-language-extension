# IMsValueAtRiskTransform.InstrumentsDimension


## Синтаксис


InstrumentsDimension: [IDimensionModel](KeDims.chm::/interface/IDimensionModel/IDimensionModel.htm);


## Описание


Свойство InstrumentsDimension определяет
 измерение финансовых инструментов.


## Комментарии


Данное измерение должно входить в:


	- переменную «портфеля»: [IMsValueAtRiskTransform.Portfolio](IMsValueAtRiskTransform.Portfolio.htm);


	- переменную финансовых инструментов: [IMsValueAtRiskTransform.StockPrices](IMsValueAtRiskTransform.StockPrices.htm).
