# IMsValueAtRiskTransform.LambdaEMWA


## Синтаксис


LambdaEMWA: Double;


## Описание


Свойство LambdaEMWA определяет значение лямбды EMWA.


## Комментарии


Значение по умолчанию «0,9».


Свойство DistinguishLongShortPositions используется только в методе «Дельта-нормальный», т.е. [IMsValueAtRiskTransform.MethodType](IMsValueAtRiskTransform.MethodType.htm) = MsValueAtRiskMethodType.DeltaNormal.
