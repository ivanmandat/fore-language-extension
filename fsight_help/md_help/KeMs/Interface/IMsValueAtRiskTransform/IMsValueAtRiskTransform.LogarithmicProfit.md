# IMsValueAtRiskTransform.LogarithmicProfit


## Синтаксис


LogarithmicProfit: Boolean;


## Описание


Свойство LogarithmicProfit определяет,
 использовать ли логарифмическую доходность.


## Комментарии


Допустимые значения:


	- True. Использовать логарифмическую
	 доходность;


	- False. Не использовать
	 логарифмическую доходность.
