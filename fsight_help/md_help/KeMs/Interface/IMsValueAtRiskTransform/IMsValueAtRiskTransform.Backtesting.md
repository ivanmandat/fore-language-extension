# IMsValueAtRiskTransform.Backtesting


## Синтаксис


Backtesting: [IMsValueAtRiskBacktesting](../IMsValueAtRiskBacktesting/IMsValueAtRiskBacktesting.htm);


## Описание


Свойство Backtesting возвращает параметры бэктестинга.


## Комментарии


Процедура бэктестинга производится параллельно с расчётом значения модели VaR. Производить ли бэктестинг, определяет свойство [IMsValueAtRiskBacktesting.Enabled](../IMsValueAtRiskBacktesting/IMsValueAtRiskBacktesting.Enabled.htm).
