# IMsValueAtRiskTransform.MissingData


## Синтаксис


MissingData: [IMissingData](StatLib.chm::/Interface/IMissingData/IMissingData.htm);


## Описание


Свойство MissingData возвращает метод обработки пропусков.


## Комментарии


MissingData учитывается, если [IMsValueAtRiskTransform.UseFillGaps](IMsValueAtRiskTransform.UseFillGaps.htm) установлено в True.

Использование свойства приведено в [IMsValueAtRiskTransform.Portfolio](IMsValueAtRiskTransform.Portfolio.htm).
