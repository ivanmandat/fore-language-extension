# IMsValueAtRiskTransform.UseFillGaps


## Синтаксис


UseFillGaps: Boolean;


## Описание


Свойство UseFillGaps определяет,
 использовать ли обработку пропусков.


## Комментарии


Допустимые значения:


	- True. Использовать обработку
	 пропусков. Метод обработки задаётся в [IMsValueAtRiskTransform.MissingData](IMsValueAtRiskTransform.MissingData.htm);


	- False. Не использовать
	 обработку пропусков.
