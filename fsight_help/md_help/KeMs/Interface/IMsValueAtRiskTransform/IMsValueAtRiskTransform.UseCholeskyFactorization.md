# IMsValueAtRiskTransform.UseCholeskyFactorization


## Синтаксис


UseCholeskyFactorization: Boolean;


## Описание


Свойство UseCholeskyFactorization определяет,
 использовать ли факторизацию Холецкого.


## Комментарии


Допустимые значения:


	- True. Использовать факторизацию
	 Холецкого;


	- False. Не использовать
	 факторизацию Холецкого.


Свойство UseCholeskyFactorization
 используется только в методе «Монте-Карло», т.е. [IMsValueAtRiskTransform.MethodType](IMsValueAtRiskTransform.MethodType.htm)
 = MsValueAtRiskMethodType.MonteCarlo.
