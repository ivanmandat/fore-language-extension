# IMsValueAtRiskTransform.ValueAtRisk


## Синтаксис


ValueAtRisk: [IMsFormulaTermInfo](../IMsFormulaTermInfo/IMsFormulaTermInfo.htm);


## Описание


Свойство ValueAtRisk определяет терм, соответствующий переменной с результатами расчёта метода.


## Комментарии


Переменная, содержащая результаты расчёта метода, должна содержать измерение, указанное в [IMsValueAtRiskTransform.OrganizationsDimension](IMsValueAtRiskTransform.OrganizationsDimension.htm).

Использование свойства приведено в [IMsValueAtRiskTransform.Portfolio](IMsValueAtRiskTransform.Portfolio.htm).
