# IMsValueAtRiskTransform.StockPrices


## Синтаксис


StockPrices: [IMsFormulaTermInfo](../IMsFormulaTermInfo/IMsFormulaTermInfo.htm);


## Описание


Свойство StockPrices определяет терм, соответствующий переменной с данными финансовых инструментах.


## Комментарии


Переменная, содержащая информацию о финансовых инструмента, должна содержать измерение, указанное в [IMsValueAtRiskTransform.InstrumentsDimension](IMsValueAtRiskTransform.InstrumentsDimension.htm).

Использование свойства приведено в [IMsValueAtRiskTransform.Portfolio](IMsValueAtRiskTransform.Portfolio.htm).
