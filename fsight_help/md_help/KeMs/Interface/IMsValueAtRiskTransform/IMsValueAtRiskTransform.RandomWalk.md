# IMsValueAtRiskTransform.RandomWalk


## Синтаксис


RandomWalk: Boolean;


## Описание


Свойство RandomWalk определяет,
 использовать ли гипотезу о случайном блуждании.


## Комментарии


Допустимые значения:


	- True. Использовать гипотезу
	 о случайном блуждании;


	- False. Не использовать
	 гипотезу о случайном блуждании.


Свойство DistinguishLongShortPositions
 используется только в методе «Дельта-нормальный», т.е. [IMsValueAtRiskTransform.MethodType](IMsValueAtRiskTransform.MethodType.htm)
 = MsValueAtRiskMethodType.DeltaNormal.
