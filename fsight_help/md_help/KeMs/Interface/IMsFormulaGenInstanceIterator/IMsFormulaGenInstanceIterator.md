# IMsFormulaGenInstanceIterator


Сборка: Ms;


## Описание


Интерфейс IMsFormulaGenInstanceIterator
 предназначен для работы с итератором по отдельным точкам, в которых производился
 расчёт по формуле.


## Иерархия наследования


           IMsFormulaGenInstanceIterator


## Комментарии


Для получения итератора по отдельным точкам используйте свойство [IMsFormulaGenerator.InstancesIterator](../IMsFormulaGenerator/IMsFormulaGenerator.InstancesIterator.htm).
