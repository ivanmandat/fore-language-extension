# IMsFormulaGenInstanceIterator.Next


## Синтаксис


Next: Boolean;


## Описание


Метод Next осуществляет переход
 к следующей записи в итераторе.


## Комментарии


Если при выполнении метода достигнут конец итератора, метод возвращает
 значение True, иначе - False.
