# IMsFormulaGenInstanceIterator.Eof


## Синтаксис


Eof: Boolean;


## Описание


Метод Eof возвращает признак
 нахождения на последней записи в итераторе.


## Комментарии


Допустимые значения:


	- True. Курсор в итераторе
	 расположен на последней записи;


	- False. Курсор в итераторе
	 расположен не на последней записи.
