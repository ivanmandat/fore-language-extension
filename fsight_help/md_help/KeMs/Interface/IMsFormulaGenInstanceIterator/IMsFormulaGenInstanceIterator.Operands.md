## Синтаксис

## Описание


Свойство Operands возвращает
 итератор, содержащий информацию об операндах формулы, использовавшихся
 в точке расчёта.


## Комментарии


Каждый операнд - это отдельный терм, который содержит параметры, применяемые
 при расчёте формулы.
