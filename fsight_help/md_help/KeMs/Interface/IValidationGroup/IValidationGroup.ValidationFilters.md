# IValidationGroup.ValidationFilters


## Синтаксис


ValidationFilters: [IValidationFilters](../IValidationFilters/IValidationFilters.htm);


## Описание


Свойство ValidationFilters возвращает коллекцию фильтров, входящих в группу правил валидации.


## Комментарии


Для настройки параметров группы правил валидации используйте свойство [IValidationGroup.Params](IValidationGroup.Params.htm).
