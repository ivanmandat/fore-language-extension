# IValidationGroup


Сборка: Ms;


## Описание


Интерфейс IValidationGroup содержит
 свойства и методы для работы с группой правил валидации.


## Иерархия наследования


           IValidationGroup


## Комментарии


Группа правил валидации - это несколько правил валидации, объединенных
 в группу для последовательного выполнения.
