# IValidationGroup.ExceptionsLimit


## Синтаксис


ExceptionsLimit: Integer;


## Описание


Свойство ExceptionsLimit определяет
 пороговое количество исключений, выгружаемых группой правил валидации.


## Комментарии


По умолчанию группа правил выгружает 1000000 исключений.


Если количество найденных исключений превышает пороговое значение, то
 группа генерирует предупреждение, сохраняет найденные исключения и завершает
 работу.


Если ExceptionsLimit = 0,
 то пороговое значение отсутствует и группа правил выявляет все исключения.


Важно. Если
 пороговое значение отсутствует, то группа правил может аварийно завершить
 свою работу из-за нехватки оперативной памяти. Найденные исключения сохранены
 не будут.
