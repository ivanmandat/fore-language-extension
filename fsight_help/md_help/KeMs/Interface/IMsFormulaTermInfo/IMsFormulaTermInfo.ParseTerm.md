# IMsFormulaTermInfo.ParseTerm


## Синтаксис


		ParseTerm(Value: String): Boolean;


## Параметры


Value. Внутреннее представление терма, в соответствии с которым будут
 настроены параметры текущего терма.


## Описание


Метод ParseTerm осуществляет
 разбор внутреннего представления терма и устанавливает параметры текущего
 терма в соответствии с полученными настройками.
