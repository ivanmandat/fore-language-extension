# IMsFormulaTermInfo.Date


## Синтаксис


		Date: DateTime;


## Описание


Свойство Date возвращает дату,
 на которую будут браться данные из указанного среза переменной при использовании
 данного терма.


## Комментарии


Свойство актуально, если свойству [IMsFormulaTermInfo.Type](IMsFormulaTermInfo.Type.htm)
 установлено значение MsFormulaTermType.Date.
