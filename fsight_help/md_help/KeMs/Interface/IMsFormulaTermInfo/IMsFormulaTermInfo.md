# IMsFormulaTermInfo


Сборка: Ms;


## Описание


Интерфейс IMsFormulaTermInfo
 содержит свойства и методы для настройки параметров терма.


## Иерархия наследования


           IMsFormulaTermInfo


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Date](IMsFormulaTermInfo.Date.htm)
		 Свойство Date возвращает
		 дату, на которую будут браться данные из указанного среза переменной
		 при использовании данного терма.


		 ![](../../Property_Image.gif)
		 [DateParamID](IMsFormulaTermInfo.DateParamID.htm)
		 Свойство DateParamID
		 определяет идентификатор параметра, являющегося календарной точкой.


		 ![](../../Property_Image.gif)
		 [DateSlice](IMsFormulaTermInfo.DateSlice.htm)
		 Свойство DateSlice
		 определяет срез данных, используемый при расчётах с данным термом.


		 ![](../../Property_Image.gif)
		 [FormatText](IMsFormulaTermInfo.FormatText.htm)
		 Свойство FormatText
		 возвращает наименование терма в указанном формате.


		 ![](../../Property_Image.gif)
		 [InversionInfo](IMsFormulaTermInfo.InversionInfo.htm)
		 Свойство InversionInfo
		 возвращает параметры начального преобразования, применяемого к
		 терму.


		 ![](../../Property_Image.gif)
		 [Lag](IMsFormulaTermInfo.Lag.htm)
		 Свойство Lag определяет
		 значение лага для терма.


		 ![](../../Property_Image.gif)
		 [Slice](IMsFormulaTermInfo.Slice.htm)
		 Свойство Slice определяет
		 срез переменной, которому соответствует данный терм.


		 ![](../../Property_Image.gif)
		 [TermInnerText](IMsFormulaTermInfo.TermInnerText.htm)
		 Свойство TermInnerText
		 возвращает внутреннее представление терма.


		 ![](../../Property_Image.gif)
		 [TermScreenedText](IMsFormulaTermInfo.TermScreenedText.htm)
		 Свойство TermScreenedText
		 возвращает наименование терма с символами экранирования.


		 ![](../../Property_Image.gif)
		 [TermText](IMsFormulaTermInfo.TermText.htm)
		 Свойство TermText возвращает
		 наименование терма, которое отображается в списке термов.


		 ![](../../Property_Image.gif)
		 [TermTransformationMethod](IMsFormulaTermInfo.TermTransformationMethod.htm)
		 Свойство TermTransformationMethod
		 определяет параметры преобразования терма.


		 ![](../../Property_Image.gif)
		 [Type](IMsFormulaTermInfo.Type.htm)
		 Свойство Type определяет
		 тип терма.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Assign](IMsFormulaTermInfo.Assign.htm)
		 Метод Assign осуществляет
		 установку параметров текущего терма, в соответствии с параметрами
		 указанного терма.


		 ![](../../Sub_Image.gif)
		 [CreateTermTransformationMethod](IMsFormulaTermInfo.CreateTermTransformationMethod.htm)
		 Метод CreateTermTransformationMethod
		 создаёт объект, определяющий параметры преобразования терма.


		 ![](../../Sub_Image.gif)
		 [ParseTerm](IMsFormulaTermInfo.ParseTerm.htm)
		 Метод ParseTerm осуществляет
		 разбор внутреннего представления терма и устанавливает параметры
		 текущего терма в соответствии с полученными настройками.


		 ![](../../Sub_Image.gif)
		 [SetDate](IMsFormulaTermInfo.SetDate.htm)
		 Метод SetDate осуществляет
		 фиксацию даты для терма.


		 ![](../../Sub_Image.gif)
		 [SetDateByLevel](IMsFormulaTermInfo.SetDateByLevel.htm)
		 Метод SetDateByLevel
		 фиксирует дату и календарную динамику для элемента выражения.
