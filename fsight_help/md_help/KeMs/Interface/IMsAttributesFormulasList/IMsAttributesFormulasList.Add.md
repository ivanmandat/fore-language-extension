# IMsAttributesFormulasList.Add


## Синтаксис


		Add: [IMsFormulaTransform](../IMsFormulaTransform/IMsFormulaTransform.htm);


## Описание


Метод Add
 добавляет новый метод расчёта в коллекцию.


## Комментарии


После выполнения будет возвращён добавленный метод расчёта. Метод добавляется
 в конец коллекции, при этом значение свойства [IMsAttributesFormulasList.Count](IMsAttributesFormulasList.Count.htm)
 увеличивается на единицу.
