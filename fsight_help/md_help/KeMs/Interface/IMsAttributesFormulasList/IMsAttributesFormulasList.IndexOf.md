# IMsAttributesFormulasList.IndexOf


## Синтаксис


		IndexOf(Value: [IMsFormulaTransform](../IMsFormulaTransform/IMsFormulaTransform.htm)):
		 Integer;


## Параметры


Value. Метод расчёта, который
 необходимо получить.


## Описание


Метод IndexOf
 возвращает индекс указанного метода.


## Комментарии


Поиск осуществляется по всей коллекции в прямом направлении, начиная
 с первого элемента и заканчивая последним. Если искомый элемент найден,
 то метод возвращает отсчитываемый от нуля индекс первого вхождения Value в диапазон элементов
 коллекции; в противном случае метод возвращает значение «-1».
