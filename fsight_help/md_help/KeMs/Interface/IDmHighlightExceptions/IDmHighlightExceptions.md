# IDmHighlightExceptions


Сборка: Ms;


## Описание


Интерфейс IDmHighlightExceptions
 предназначен для интеллектуального анализа данных методом «Поиск
 исключений».


## Иерархия наследования


           [IDmMethodDetails](../IDmMethodDetails/IDmMethodDetails.htm)


           [IDmAttributesDetails](../IDmAttributesDetails/IDmAttributesDetails.htm)


           IDmHighlightExceptions


## Комментарии


Данный вид анализа определяет «степень исключительности» для каждого
 признака каждого объекта на основании всей совокупности данных и значений
 других признаков. Исходя из полученной характеристики, можно ранжировать
 объекты по их исключительности.
