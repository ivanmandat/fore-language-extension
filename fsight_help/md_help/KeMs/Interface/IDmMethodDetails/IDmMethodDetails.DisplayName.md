# IDmMethodDetails.DisplayName


## Синтаксис


		DisplayName: String;


## Описание


Свойство DisplayName возвращает
 наименование метода.


## Комментарии


Для получения параметров статистического метода, используемого при анализе,
 используйте свойство [IDmMethodDetails.StatMethod](IDmMethodDetails.StatMethod.htm).
