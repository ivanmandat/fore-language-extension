# IDmMethodDetails.CrossValidation


## Синтаксис


		CrossValidation: [IDmMethodCrossValidation](../IDmMethodCrossValidation/IDmMethodCrossValidation.htm);


## Описание


Свойство CrossValidation возвращает
 настройки кросс-валидации.


## Комментарии


По умолчанию используется метод кросс-валидация по K
 блокам, где K равняется трём.
