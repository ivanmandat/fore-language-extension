# IMsObjectParamValue


Сборка: Ms;


## Описание


Интерфейс IMsObjectParamValue
 предназначен для работы со связью параметра моделирования со значением
 параметра объекта репозитория.


## Иерархия наследования


           IMsObjectParamValue


           [NamedEntity](KeSom.chm::/Interface/INamedEntity/INamedEntity.htm)


## Комментарии


Для работы с коллекцией связей параметров моделирования с несколькими
 объектами используйте интерфейс [IMsObjectParams](../IMsObjectParams/IMsObjectParams.htm).
