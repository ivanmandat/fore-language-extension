# IValidationCustomMapping.FindByValidationDimId


## Синтаксис


FindByValidationDimId(ValidationDimId: String):
 [IValidationCustomDimensionsBinding](../IValidationCustomDimensionsBinding/IValidationCustomDimensionsBinding.htm);


## Параметры


ValidationDimId. Идентификатор
 измерения из сопоставляемого источника данных, по которому выполняется
 валидация.


## Описание


Метод FindByValidationDimId выполняет
 поиск привязки по идентификатору измерения из сопоставляемого источника
 данных, по которому выполняется валидация.


## Комментарии


Если привязка для указанного измерения не найдена, то метод возвращает
 значение Null.
