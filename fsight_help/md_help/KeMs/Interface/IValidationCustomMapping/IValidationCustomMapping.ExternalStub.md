# IValidationCustomMapping.ExternalStub


## Синтаксис


ExternalStub: [IVariableStub](KeCubes.chm::/Interface/IVariableStub/IVariableStub.htm);


## Описание


Свойство ExternalStub возвращает
 внешний источник данных, который участвует в сопоставлении.


## Комментарии


Для получения сопоставляемого источника данных, по которому выполняется
 валидация, используйте свойство [IValidationCustomMapping.ValidationStub](IValidationCustomMapping.ValidationStub.htm).
