# IValidationCustomMapping.FindByExternalDimId


## Синтаксис


FindByExternalDimId(ExternalDimId: String): [IValidationCustomDimensionsBinding](../IValidationCustomDimensionsBinding/IValidationCustomDimensionsBinding.htm);


## Параметры


ExternalDimId. Идентификатор
 измерения из внешнего источника данных, участвующего в сопоставлении.


## Описание


Метод FindByExternalDimId выполняет
 поиск привязки по идентификатору измерения из внешнего источника данных.


## Комментарии


Если привязка для указанного измерения не найдена, то метод возвращает
 значение Null.
