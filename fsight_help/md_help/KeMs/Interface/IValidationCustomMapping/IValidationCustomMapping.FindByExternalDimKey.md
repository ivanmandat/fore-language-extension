# IValidationCustomMapping.FindByExternalDimKey


## Синтаксис


FindByExternalDimKey(ExternalDimKey: Integer): [IValidationCustomDimensionsBinding](../IValidationCustomDimensionsBinding/IValidationCustomDimensionsBinding.htm);


## Параметры


ExternalDimKey. Ключ измерения
 из внешнего источника данных, участвующего в сопоставлении.


## Описание


Метод FindByExternalDimKey выполняет
 поиск привязки по ключу измерения из внешнего источника данных.


## Комментарии


Если привязка для указанного измерения не найдена, то метод возвращает
 значение Null.
