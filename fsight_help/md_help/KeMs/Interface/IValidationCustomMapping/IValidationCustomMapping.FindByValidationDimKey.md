# IValidationCustomMapping.FindByValidationDimKey


## Синтаксис


FindByValidationDimKey(ValidationDimKey: Integer):
 [IValidationCustomDimensionsBinding](../IValidationCustomDimensionsBinding/IValidationCustomDimensionsBinding.htm);


## Параметры


ValidationDimKey. Ключ измерения
 из сопоставляемого источника данных, по которому выполняется валидация.


## Описание


Метод FindByValidationDimKey выполняет
 поиск привязки по ключу измерения из сопоставляемого источника
 данных, по которому выполняется валидация.


## Комментарии


Если привязка для указанного измерения не найдена, то метод возвращает
 значение Null.
