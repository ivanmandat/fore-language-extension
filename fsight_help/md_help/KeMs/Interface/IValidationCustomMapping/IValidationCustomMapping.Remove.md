# IValidationCustomMapping.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс привязки.


## Описание


Метод Remove удаляет указанную
 привязку.


## Комментарии


Если удаление прошло успешно, то метод возвращает значение True,
 иначе - False.
