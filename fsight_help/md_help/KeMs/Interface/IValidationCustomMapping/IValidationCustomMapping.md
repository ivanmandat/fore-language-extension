# IValidationCustomMapping


Сборка: Ms;


## Описание


Интерфейс IValidationCustomMapping
 предназначен для работы с сопоставлением источников данных, используемых
 в правиле валидации типа «Сравнение
 по выражению».


## Иерархия наследования


           IValidationCustomMapping


## Комментарии


Для работы со всеми сопоставлениями используйте интерфейс [IValidationCustomMappings](../IValidationCustomMappings/IValidationCustomMappings.htm).

		 Имя свойства

		 [ExternalStub](IValidationCustomMapping.ExternalStub.htm)
