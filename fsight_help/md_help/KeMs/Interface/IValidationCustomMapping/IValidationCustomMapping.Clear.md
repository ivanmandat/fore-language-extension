# IValidationCustomMapping.Clear


## Синтаксис


Clear;


## Описание


Метод Clear очищает привязки
 измерений для сопоставляемых источников данных.


## Комментарии


Для удаления определенных привязок измерений используйте метод [IValidationCustomMapping.Remove](IValidationCustomMapping.Remove.htm).
