# IValidationCustomMapping.ValidationStub


## Синтаксис


ValidationStub: [IVariableStub](KeCubes.chm::/Interface/IVariableStub/IVariableStub.htm);


## Описание


Свойство ValidationStub возвращает
 сопоставляемый источник данных, по которому выполняется валидация.


## Комментарии


Для получения внешнего источника данных, который участвует в сопоставлении,
 используйте свойство [IValidationCustomMapping.ExternalStub](IValidationCustomMapping.ExternalStub.htm).
