# IMsNonLinearRegressionTransform.MaxIteration


## Синтаксис


MaxIteration: Integer;


## Описание


Свойство MaxIteration определяет
 максимальное число итераций, за которое будет производиться расчет оптимальных
 значений коэффициентов.


## Комментарии


Значение по умолчанию: 500.
