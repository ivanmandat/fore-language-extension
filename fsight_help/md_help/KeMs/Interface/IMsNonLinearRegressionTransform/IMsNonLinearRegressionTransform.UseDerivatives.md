# IMsNonLinearRegressionTransform.UseDerivatives


## Синтаксис


UseDerivatives: Boolean;


## Описание


Свойство UseDerivatives определяет,
 будут ли использоваться аналитические производные для повышения точности
 расчета.


## Комментарии


Допустимые значения:


	- True. Аналитические
	 производные используются;


	- False. Значение по умолчанию.
	 Аналитические производные не используются.
