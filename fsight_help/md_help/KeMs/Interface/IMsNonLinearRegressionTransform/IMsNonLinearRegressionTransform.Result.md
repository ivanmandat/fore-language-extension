## Синтаксис

## Описание


Свойство Result возвращает терм,
 соответствующий моделируемой переменной.


## Комментарии


Для получения коллекции термов, используемых для составления уравнения
 модели, используйте свойство [IMsNonLinearRegressionTransform.Operands](IMsNonLinearRegressionTransform.Operands.htm).
