# IMsNonLinearRegressionTransform.MissingData


## Синтаксис


MissingData: [IMissingData](StatLib.chm::/Interface/IMissingData/IMissingData.htm);


## Описание


Свойство MissingData возвращает
 объект, содержащий параметры обработки пропусков исходного ряда.


## Комментарии


По умолчанию используется метод Casewise.
 Пустые значения исключаются из ряда. Расчеты ведутся без их учета.
