# IMsNonLinearRegressionTransform.Coefficients


## Синтаксис


Coefficients: Array;


## Описание


Свойство Coefficients возвращает
 массив, содержащий наименования коэффициентов, входящих в уравнение модели.


## Комментарии


Для получения значений коэффициентов используйте свойство [IMsNonLinearRegressionTransform.CoefficientsData](IMsNonLinearRegressionTransform.CoefficientsData.htm).
