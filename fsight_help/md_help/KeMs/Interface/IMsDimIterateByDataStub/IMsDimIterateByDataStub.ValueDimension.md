# IMsDimIterateByDataStub.ValueDimension


## Синтаксис


ValueDimension: [IMsDimIteratorFilteredDimension](../IMsDimIteratorFilteredDimension/IMsDimIteratorFilteredDimension.htm);


## Описание


Свойство ValueDimension возвращает
 измерение источника данных приёмника, к которому привязано значение фильтрующего
 источника данных для дополнительной фильтрации элементов измерения приёмника
 по значениям указанного атрибута.


## Комментарии


Для привязки значения по атрибуту измерения источника данных приёмника
 используйте метод [IMsDimIterateByDataStub.SetValueDimension](IMsDimIterateByDataStub.SetValueDimension.htm).


Для указания атрибута измерения источника данных приёмника используйте
 свойство [IMsDimIterateByDataStub.ValueDimAttr](IMsDimIterateByDataStub.ValueDimAttr.htm).
