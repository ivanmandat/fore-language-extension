Dimension. Измерение, по которому

Свойство Dimension возвращает
 параметры фильтрации измерения, по которому выполняется итерация.

 используйте свойство [IMsDimIterateByDataStub.Stub](IMsDimIterateByDataStub.Stub.htm).
