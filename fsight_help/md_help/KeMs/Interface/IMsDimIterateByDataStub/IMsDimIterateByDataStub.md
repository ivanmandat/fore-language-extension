# IMsDimIterateByDataStub


Сборка: Ms;


## Описание


Интерфейс IMsDimIterateByDataStub
 предназначен для работы с источником данных, по существующим точкам которых
 будут выполняться итерации.


## Иерархия наследования


           IMsDimIterateByDataStub


## Комментарии


Для работы с коллекцией источников, по существующим точкам которых будут
 выполняться итерации, используйте интерфейс [IMsDimIterateByDataStubs](../IMsDimIterateByDataStubs/IMsDimIterateByDataStubs.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dimension](IMsDimIterateByDataStub.Dimension.htm)


		 Свойство Dimension
		 возвращает параметры фильтрации измерения, по которому выполняется
		 итерация.


		 ![](../../Property_Image.gif)
		 [Stub](IMsDimIterateByDataStub.Stub.htm)


		 Свойство Stub
		 возвращает источник данных, по существующим точкам которого будут
		 выполняться итерации.


		 ![](../../Property_Image.gif)
		 [ValueDimAttr](IMsDimIterateByDataStub.ValueDimAttr.htm)


		 Свойство ValueDimAttr
		 определяет атрибут измерения, по значениям которого будут отфильтрованы
		 элементы измерения приёмника.


		 ![](../../Property_Image.gif)
		 [ValueDimension](IMsDimIterateByDataStub.ValueDimension.htm)


		 Свойство ValueDimension
		 возвращает измерение источника данных приёмника, к которому привязано
		 значение фильтрующего источника данных для дополнительной фильтрации
		 элементов измерения приёмника по значениям указанного атрибута.
