# IMsDimIterateByDataStub.ValueDimAttr


## Синтаксис


ValueDimAttr: String;


## Описание


Свойство ValueDimAttr определяет
 атрибут измерения, по значениям которого будут отфильтрованы элементы
 измерения приёмника.


## Комментарии


Для привязки значения по атрибуту измерения источника данных приёмника
 используйте метод [IMsDimIterateByDataStub.SetValueDimension](IMsDimIterateByDataStub.SetValueDimension.htm).
