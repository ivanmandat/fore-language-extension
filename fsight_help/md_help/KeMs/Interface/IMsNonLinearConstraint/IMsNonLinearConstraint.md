# IMsNonLinearConstraint


## Описание


Интерфейс IMsNonLinearConstraint
 содержит свойства ограничения целевой функции.


## Комментарии


Ограничение целевой функции можно получить, используя свойства и методы
 интерфейса [IMsNonLinearConstraints](../IMsNonLinearConstraints/IMsNonLinearConstraints.htm).
