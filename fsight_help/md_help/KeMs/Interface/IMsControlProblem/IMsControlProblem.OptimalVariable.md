# IMsControlProblem.OptimalVariable


## Синтаксис


		OptimalVariable: [IMsVariable](../IMsVariable/IMsVariable.htm);


## Описание


Свойство OptimalVariable определяет
 переменную контейнера моделирования, в которую будет выгружаться рассчитанное
 оптимальное значение целевой функции.


## Комментарии


Для задания ограничений целевой функции используйте свойство [IMsControlProblem.Constraints](IMsControlProblem.Constraints.htm).
