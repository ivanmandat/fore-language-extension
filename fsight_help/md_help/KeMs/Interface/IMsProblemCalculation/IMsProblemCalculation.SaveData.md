# IMsProblemCalculation.SaveData


## Синтаксис


SaveData: Boolean;


## Описание


Свойство SaveData определяет
 признак выгрузки данных в выходные переменные после завершения расчёта
 [задачи
 моделирования](uimodelling.chm::/2_container_of_modeling/2_3_work_object/2_3_4_problem/uimodelling_problem.htm).


## Комментарии


По умолчанию свойству установлено значение True,
 при этом осуществляется выгрузка данных.
