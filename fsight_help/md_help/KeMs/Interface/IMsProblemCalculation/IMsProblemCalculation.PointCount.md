# IMsProblemCalculation.PointCount


## Синтаксис


PointCount: Integer;


## Описание


Свойство PointCount возвращает
 общее количество шагов, которые будут пройдены при расчёте [задачи
 моделирования](uimodelling.chm::/2_container_of_modeling/2_3_work_object/2_3_4_problem/uimodelling_problem.htm).


## Комментарии


Свойство зависит от количества объектов, участвующих в расчете.
