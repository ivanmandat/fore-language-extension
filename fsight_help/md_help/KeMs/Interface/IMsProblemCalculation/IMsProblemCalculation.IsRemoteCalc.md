# IMsProblemCalculation.IsRemoteCalc


## Синтаксис


		IsRemoteCalc: Boolean;


## Описание


Свойство IsRemoteCalc возвращает
 признак удалённого расчёта [задачи
 моделирования](uimodelling.chm::/2_container_of_modeling/2_3_work_object/2_3_4_problem/uimodelling_problem.htm) на BI-сервере.


## Комментарии


Допустимые значения:


	- True. Задача моделирования
	 рассчитывается удалённо. Параметры удалённого расчёта определяет свойство
	 [IMsProblemCalculationSettings.RemoteCalc](../IMsProblemCalculationSettings/IMsProblemCalculationSettings.RemoteCalc.htm);


	- False. Задача моделирования
	 рассчитывается локально.
