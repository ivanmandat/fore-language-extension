# IMsLinearEquationsTransform.EstimateSimultaneously


## Синтаксис


EstimateSimultaneously(Formula: [IMsFormula](../IMsFormula/IMsFormula.htm)): Boolean;


## Параметры


Formula. Уравнение.


## Описание


Свойство EstimateSimultaneously определяет, используется ли уравнение в системе. True - уравнение включается в систему, False - уравнение исключается из системы.
