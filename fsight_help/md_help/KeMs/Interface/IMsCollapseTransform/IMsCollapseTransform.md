## Описание

 содержит свойства и методы, определяющие параметры модели «Коллапс».

## Комментарии


Данная модель осуществляет агрегацию данных с нижнего уровня на верхний
 и необходима при расчёте задачи, требующей агрегации данных. Например,
 данные, рассчитанные по месяцам, необходимо просуммировать для квартальной
 динамики.


При агрегации данных модель учитывает настройки календарной динамики:
 смещение начала периода относительно его начала/конца.
