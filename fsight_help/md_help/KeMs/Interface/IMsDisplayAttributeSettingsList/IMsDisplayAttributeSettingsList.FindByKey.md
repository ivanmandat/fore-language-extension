# IMsDisplayAttributeSettingsList.FindByKey


## Синтаксис


		FindByKey(DimKey: Integer): [IMsDisplayAttributeSettings](../IMsDisplayAttributeSettings/IMsDisplayAttributeSettings.htm);


## Параметры


DimKey. Ключ измерения.


## Описание


Метод FindByKey выполняет поиск
 настроек атрибутов для измерения с указанным ключом.


## Комментарии


Если настроек атрибутов для указанного измерения не найдено, то метод
 вернёт значение Null.
