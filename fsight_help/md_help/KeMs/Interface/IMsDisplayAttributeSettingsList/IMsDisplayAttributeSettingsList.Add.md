# IMsDisplayAttributeSettingsList.Add


## Синтаксис


		Add(Value: [IDimensionModel](KeDims.chm::/interface/IDimensionModel/IDimensionModel.htm)):
		 [IMsDisplayAttributeSettings](../IMsDisplayAttributeSettings/IMsDisplayAttributeSettings.htm);


## Параметры


Value. Измерение, атрибут которого
 будет отображаться в строковом представлении элемента выражения.


## Описание


Метод Add добавляет параметр
 в коллекцию.


## Комментарии


Для удаления элементов из коллекции используйте методы [IMsDisplayAttributeSettingsList.Clear](IMsDisplayAttributeSettingsList.Clear.htm),
 [IMsDisplayAttributeSettingsList.Remove](IMsDisplayAttributeSettingsList.Remove.htm),
 [IMsDisplayAttributeSettingsList.RemoveByKey](IMsDisplayAttributeSettingsList.RemoveByKey.htm).
