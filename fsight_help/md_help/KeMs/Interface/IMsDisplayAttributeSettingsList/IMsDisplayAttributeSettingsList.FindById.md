# IMsDisplayAttributeSettingsList.FindById


## Синтаксис


		FindById(DimId: String): [IMsDisplayAttributeSettings](../IMsDisplayAttributeSettings/IMsDisplayAttributeSettings.htm);


## Параметры


DimId. Идентификатор измерения.


## Описание


Метод FindById выполняет
 поиск настроек атрибутов для измерения с указанным идентификатором.


## Комментарии


Если настроек атрибутов для указанного измерения не найдено, то метод
 вернёт значение Null.
