# IMsDisplayAttributeSettingsList.Clear


## Синтаксис


		Clear;


## Описание


Метод Clear очищает коллекцию.


## Комментарии


Для удаления одного элемента коллекции используйте методы [IMsDisplayAttributeSettingsList.Remove](IMsDisplayAttributeSettingsList.Remove.htm)
 и [IMsDisplayAttributeSettingsList.RemoveByKey](IMsDisplayAttributeSettingsList.RemoveByKey.htm).
