# IMsDisplayAttributeSettingsList.Remove


## Синтаксис


		Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс элемента в коллекции.


## Описание


Метод Remove удаляет указанный
 элемент из коллекции.


## Комментарии


При успешном удалении элемента метод вернёт значение True,
 при неуспешном - False.
