# IMsDisplayAttributeSettingsList.RemoveByKey


## Синтаксис


		RemoveByKey(DimKey: Integer): Boolean;


## Параметры


DimKey. Ключ измерения.


## Описание


Метод RemoveByKey удаляет настройки
 атрибутов для измерения с указанным ключом.


## Комментарии


При успешном удалении элемента метод вернёт значение True,
 при неуспешном - False.
