# IMsDisplayAttributeSettingsList


Сборка: Ms;


## Описание


Интерфейс IMsDisplayAttributeSettingsList
 предназначен для работы с коллекцией параметров отображения атрибутов
 в строковом представлении элементов выражения.


## Иерархия наследования


           IMsDisplayAttributeSettingsList


## Комментарии


Для получения параметров отображения атрибутов используйте свойство
 [IMsDisplaySettings.DisplayAttributes](../IMsDisplaySettings/IMsDisplaySettings.DisplayAttributes.htm).
