## Описание


Свойство UserOptimizationProblem
 возвращает пользовательскую задачу оптимизации, которой соответствует
 текущий элемент цепочки расчёта.


## Комментарии


Свойство вернёт ту задачу, которая была указана при выполнении метода
 [IMsCalculationChainEntries.AddUserOptimizationProblem](../IMsCalculationChainEntries/IMsCalculationChainEntries.AddUserOptimizationProblem.htm).
