# IMsStringGenerator.ShowFullVariableNames


## Синтаксис


ShowFullVariableNames: Boolean;


## Описание


Свойство ShowFullVariableNames
 определяет, отображать ли полные наименования переменных.


## Комментарии


Допустимые значения:


	- True.
	 Значение по умолчанию. Отображать полные наименования переменных;


	- False.
	 Отображать наименования элементов выражения.
