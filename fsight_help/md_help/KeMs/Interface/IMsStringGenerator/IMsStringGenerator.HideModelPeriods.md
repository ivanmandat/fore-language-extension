# IMsStringGenerator.HideModelPeriods


## Синтаксис


HideModelPeriods: Boolean;


## Описание


Свойство HideModelPeriods определяет, скрывать ли период расчёта в строковом представлении.


## Комментарии


Допустимые значения:


-
True. Период расчёта не включается в строковое представление. Например: Arhentina|NGDP[t] = Arhentina|BCA[t];


-
False. Значение по умолчанию. Период расчёта включается в строковое представление. При генерации строкового представления учитываются параметры и условия, применяемые для определения периода расчёта. Например:


 Способ определения периода расчёта
 Пример строкового представления


 Период расчёта задан определенными датами
 Arhentina|NGDP[t] = Arhentina|BCA[t], 2014A1-2020A1


 Период расчёта задан определенными датами и одна из дат является параметром
 Arhentina|NGDP[t] = Arhentina|BCA[t], 2014A1-{EndDate}


 Начало периода идентификации определяется по условию:


-
Начало данных +/-k периодов;


-
Сегодня +/-k периодов;


-
От окончания идентификации +/-k периодов.


-
Arhentina|NGDP[t] = Arhentina|BCA[t], (Начало данных +/-k)-2020A1


-
Arhentina|NGDP[t] = Arhentina|BCA[t], (Сегодня +/-k)-2020A1


-
Arhentina|NGDP[t] = Arhentina|BCA[t], (От окончания идентификации +/-k)-2020A1


 Окончание периода идентификации определяется по условию:


-
Начало данных +/-k периодов;


-
Окончание данных +/-k периодов;


-
Сегодня +/-k периодов;


-
От начала идентификации +/-k периодов.


-
Arhentina|NGDP[t] = Arhentina|BCA[t], (Начало данных +/-k)-2020A1


-
Arhentina|NGDP[t] = Arhentina|BCA[t], (Окончание данных +/-k)-2020A1


-
Arhentina|NGDP[t] = Arhentina|BCA[t], (Сегодня +/-k)-2020A1


-
Arhentina|NGDP[t] = Arhentina|BCA[t], (От начала идентификации +/-k)-2020A1


 Окончание периода прогнозирования определяется по условию:


-
Окончание данных +/-k периодов;


-
От окончания идентификации +/-k периодов;


-
Сегодня +/-k периодов.


-
Arhentina|NGDP[t] = Arhentina|BCA[t], 2014A1-(Окончание данных +/-k)


-
Arhentina|NGDP[t] = Arhentina|BCA[t], 2014A1-(От окончания идентификации +/-k)


-
Arhentina|NGDP[t] = Arhentina|BCA[t], 2014A1-(Сегодня +/-k)


 Все случаи, когда две даты задаются с помощью условий
 Arhentina|NGDP[t] = Arhentina|BCA[t], (Начало данных +/-k)-(Сегодня +/-k)


 Все случаи, когда одна дата задается параметром, а другая с помощью условия
 Arhentina|NGDP[t] = Arhentina|BCA[t], (Начало данных +/-k)-{EndDate}
