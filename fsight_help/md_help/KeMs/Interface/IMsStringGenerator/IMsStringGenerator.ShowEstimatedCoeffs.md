# IMsStringGenerator.ShowEstimatedCoeffs


## Синтаксис


ShowEstimatedCoeffs: Boolean;


## Описание


Свойство ShowEstimatedCoeffs определяет, отображать ли идентифицированные коэффициенты уравнения.


## Комментарии


Допустимые значения:


-
True. Идентифицированные коэффициенты уравнения отображаются;


-
False. Значение по умолчанию. Идентифицированные коэффициенты уравнения не отображаются.
