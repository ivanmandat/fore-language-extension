# IMsStringGenerator


Сборка: Ms;


## Описание


Интерфейс IMsStringGenerator
 содержит свойства и методы для работы с генератором строкового представления.


## Иерархия наследования


           IMsStringGenerator


## Комментарии


Для получения генератора строкового представления используйте методы:


	- [IMsFormula.CreateStringGenerator](../IMsFormula/IMsFormula.CreateStringGenerator.htm).
	 Генератор для метода расчёта модели;


	- [IMsFormulaTransform.CreateStringGenerator](../IMsFormulaTransform/IMsFormulaTransform.CreateStringGenerator.htm).
	 Генератор для параметров расчёта модели;


	- [IMsModel.CreateStringGenerator](../IMsModel/IMsModel.CreateStringGenerator.htm).
	 Генератор для наименования модели.
