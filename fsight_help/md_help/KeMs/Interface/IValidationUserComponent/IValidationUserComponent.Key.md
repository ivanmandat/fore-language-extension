# IValidationUserComponent.Key


## Синтаксис


Key: Integer;


## Описание


Свойство Key возвращает ключ компонента.


## Комментарии


Ключ - это уникальный идентификатор компонента в коллекции. Значение ключа генерируется автоматически и не может быть изменено пользователем.
