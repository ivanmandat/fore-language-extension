# IValidationUserComponent


Сборка: Ms;


## Описание


Интерфейс IValidationUserComponent
 предназначен для работы с пользовательским компонентом валидации.


## Комментарии


Компонент - это ряд, настройка которого позволяет проводить более эффективный
 анализ возникновения исключения в правиле валидации.


Компонент может настраиваться по отметке или по формуле. Способ настройки
 определяется свойством [IValidationUserComponent.Kind](IValidationUserComponent.Kind.htm).
