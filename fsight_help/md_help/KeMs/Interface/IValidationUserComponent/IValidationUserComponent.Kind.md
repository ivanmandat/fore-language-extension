# IValidationUserComponent.Kind


## Синтаксис


Kind: [ValidationUserComponentKind](../../Enums/ValidationUserComponentKind.htm);


## Описание


Свойство Kind определяет вид
 компонента.


## Комментарии


Вид компонента определяет способ его настройки: по отметке или
 по формуле.


Допустимые значения:


	- ValidationUserComponentKind.Selection.
	 Компонент настраивается по отметке. Используйте свойство [IValidationUserComponent.Selection](IValidationUserComponent.Selection.htm);


	- ValidationUserComponentKind.Transform.
	 Компонент настраивается по формуле. Используйте свойство [IValidationUserComponent.Transform](IValidationUserComponent.Transform.htm).
