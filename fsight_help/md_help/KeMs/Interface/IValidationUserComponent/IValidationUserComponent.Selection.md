# IValidationUserComponent.Selection


## Синтаксис


Selection: [IDimSelectionSet](KeDims.chm::/interface/IDimSelectionSet/IDimSelectionSet.htm);


## Описание


Свойство Selection определяет отметку, по которой настроен компонент.


## Комментарии


Свойство учитывается, если [IValidationUserComponent.Kind](IValidationUserComponent.Kind.htm) = ValidationUserComponentKind.Selection.
