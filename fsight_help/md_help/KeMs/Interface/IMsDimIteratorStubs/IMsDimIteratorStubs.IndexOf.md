# IMsDimIteratorStubs.IndexOf


## Синтаксис


		IndexOf(Value: [IVariableStub](KeCubes.chm::/Interface/IVariableStub/IVariableStub.htm)):
		 Integer;


## Параметры


Value. Источник данных, индекс
 которого необходимо узнать.


## Описание


Метод IndexOf возвращает индекс
 указанного источника данных.


## Комментарии


Если указанный источник данных отсутствует в коллекции, то метод возвращает
 значение «-1».
