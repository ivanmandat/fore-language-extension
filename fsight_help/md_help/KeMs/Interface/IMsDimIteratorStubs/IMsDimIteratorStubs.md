# IMsDimIteratorStubs


Сборка: Ms;


## Описание


Интерфейс IMsDimIteratorStubs
 предназначен для работы с коллекцией источников данных, к которым применяется
 измерение многомерного итератора.


## Иерархия наследования


           IMsDimIteratorStubs


## Комментарии


Для получения коллекции источников данных используйте свойство [IMsDimIteratorDimension.Stubs](../IMsDimIteratorDimension/IMsDimIteratorDimension.Stubs.htm).
