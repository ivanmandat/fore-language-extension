# IMsDimIteratorStubs.Remove


## Синтаксис


		Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемого источника
 данных.


## Описание


Метод Remove удаляет источник
 данных с указанным индексом.


## Комментарии


При успешном удалении источника данных метод возвращает значение True, при неуспешном - False.


Индексация источников данных в коллекции сквозная и начинается с нуля.
 Для получения количества источников данных в коллекции используйте свойство
 [IMsDimIteratorStubs.Count](IMsDimIteratorStubs.Count.htm).
