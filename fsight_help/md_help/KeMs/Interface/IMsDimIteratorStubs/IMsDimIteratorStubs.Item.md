# IMsDimIteratorStubs.Item


## Синтаксис


		Item(Index: Integer): [IVariableStub](KeCubes.chm::/Interface/IVariableStub/IVariableStub.htm);


## Параметры


Index. Индекс источника данных
 в коллекции.


## Описание


Свойство Item возвращает источник
 данных с указанным индексом.


## Комментарии


Индексация источников данных в коллекции сквозная и начинается с нуля.


Для получения количества источников данных в коллекции используйте свойство
 [IMsDimIteratorStubs.Count](IMsDimIteratorStubs.Count.htm).
