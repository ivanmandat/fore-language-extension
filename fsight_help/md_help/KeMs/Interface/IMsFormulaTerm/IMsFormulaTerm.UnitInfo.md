# IMsFormulaTerm.UnitInfo


## Синтаксис


		UnitInfo: [IMsUnitInfo](../IMsUnitInfo/IMsUnitInfo.htm);


## Описание


Свойство UnitInfo возвращает
 данные о единицах измерения терма.


## Комментарии


Единицы измерения используются в термах, основанных на данных базы данных
 временных рядов.


Если единицы измерения не являются обязательным атрибутом базы данных
 временных рядов, то свойство возвращает информацию о том, что единицы
 измерения не определены.
