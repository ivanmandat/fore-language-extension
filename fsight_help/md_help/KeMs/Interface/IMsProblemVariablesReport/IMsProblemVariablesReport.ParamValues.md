# IMsProblemVariablesReport.ParamValues


## Синтаксис


ParamValues: [IMsModelParamValues](../IMsModelParamValues/IMsModelParamValues.htm);


## Описание


Свойство ParamValues возвращает значения параметров, используемых в расчете задачи моделирования.


## Комментарии


Значения параметров доступны только после расчета задачи моделирования.
