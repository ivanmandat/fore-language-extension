# IMsProblemVariablesReport.SetModelSpace


## Синтаксис


SetModelSpace(Value: [IMsModelSpace](../IMsModelSpace/IMsModelSpace.htm));


## Параметры


Value. Контейнер моделирования.


## Описание


Метод SetModelSpace устанавливает
 контейнер моделирования, содержащий задачу, для которой генерируется отчёт.


## Комментарии


Для добавления переменных в отчёт используйте методы [IMsProblemVariablesReport.AddVariable](IMsProblemVariablesReport.AddVariable.htm)
 и [IMsProblemVariablesReport.AddVariableKind](IMsProblemVariablesReport.AddVariableKind.htm).
