# IMsProblemVariablesReport.AddVariable


## Синтаксис


AddVariable(Value: [IVariableStub](KeCubes.chm::/Interface/IVariableStub/IVariableStub.htm)): Integer;


## Параметры


Value. Переменная, которую необходимо добавить в отчет.


## Описание


Метод AddVariable добавляет указанную переменную в отчет.


## Комментарии


Value. Добавляемая переменная должна быть представлена как абстрактный источник данных.


После выполнения AddVariable возвращает индекс добавленной переменной.
