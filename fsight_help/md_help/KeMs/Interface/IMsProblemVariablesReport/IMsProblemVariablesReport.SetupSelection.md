# IMsProblemVariablesReport.SetupSelection


## Синтаксис


SetupSelection(Selection: [IDimSelectionSet](KeDims.chm::/interface/IDimSelectionSet/IDimSelectionSet.htm));


## Параметры


Selection. Отметка, по которой
 будет настроена отметка отчета.


## Описание


Метод SetupSelection настраивает
 отметку отчета.


## Комментарии


Допустимые значения параметра Selection:


	- Selection = Null. Будет
	 создан отчет с частными измерениями;


	- Selection <> Null.
	 Будет создан отчет с составными измерениями.
