# IMsProblemVariablesReport.CreateDataSource


## Синтаксис


CreateDataSource:[IMatrixDataSource](KeMatrix.chm::/Interface/IMatrixDataSource/IMatrixDataSource.htm);


## Описание


Метод CreateDataSource создает источник данных на основе отчета.


## Комментарии


Источник будет пустым, если был создан до добавления в отчет переменных. Для добавления переменных используйте [IMsProblemVariablesReport.AddVariable](IMsProblemVariablesReport.AddVariable.htm) и [IMsProblemVariablesReport.AddVariableKind](IMsProblemVariablesReport.AddVariableKind.htm).
