# IMsProblemVariablesReport


Сборка: Ms;


## Описание


Интерфейс IMsProblemVariablesReport
 предназначен для работы с отчетом по переменным задачи моделирования.


## Иерархия наследования


           IMsProblemVariablesReport


## Комментарии


Для создания отчета используйте метод [IMsProblem.CreateVariablesReport](../IMsProblem/IMsProblem.CreateVariablesReport.htm)
 или [IMsProblemCalculation.CreateVariablesReport](../IMsProblemCalculation/IMsProblemCalculation.CreateVariablesReport.htm).
