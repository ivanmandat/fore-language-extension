# IMsProblemVariablesReport.AddVariableKind


## Синтаксис


AddVariableKind(Kind: [MsVariableKind](../../Enums/MsVariableKind.htm));


## Параметры


Kind. Тип переменной.


## Описание


Метод AddVariableKind добавляет все переменные указанного типа в отчет.


## Комментарии


Используя данный метод в отчет можно добавить сразу все входные, все выходные или все переменные, участвующие в задаче.
