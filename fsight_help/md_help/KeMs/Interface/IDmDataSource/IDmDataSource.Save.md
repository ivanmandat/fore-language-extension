# IDmDataSource.Save


## Синтаксис


		Save;


## Описание


Метод Save сохраняет источник
 данных.


## Комментарии


Для определения признака редактирования данных источника используйте
 свойство [IDmDataSource.SupportsEdit](IDmDataSource.SupportsEdit.htm),
 для определения признака доступности данных только для чтения - свойство
 [IDmDataSource.ReadOnly](IDmDataSource.ReadOnly.htm).
