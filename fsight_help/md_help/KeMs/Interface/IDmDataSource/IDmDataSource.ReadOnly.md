# IDmDataSource.ReadOnly


## Синтаксис


		ReadOnly: Boolean;


## Описание


Свойство ReadOnly определяет признак доступности данных только
 для чтения.


## Комментарии


Допустимые значения:


	- True. Данные доступны
	 только для чтения;


	- False. Данные доступны
	 для чтения и записи.
