# IDmDataSource.SupportsEdit


## Синтаксис


		SupportsEdit: Boolean;


## Описание


Свойство SupportsEdit определяет признак поддержки редактирования
 источником данных.


## Комментарии


Допустимые значения:


	- True. Источник данных
	 поддерживает редактирование;


	- False. Источник данных
	 не поддерживает редактирование.
