# IDmDataSource.AddResultColumn


## Синтаксис


		AddResultColumn(Caption: String);


## Параметры


Caption. Заголовок столбца.


## Описание


Метод AddResultColumn добавляет
 столбец, содержащий результаты анализа.


## Комментарии


Добавление столбца с результатами может использоваться при кластеризации
 данных.
