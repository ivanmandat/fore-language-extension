# IDmDataSource.FieldCount


## Синтаксис


		FieldCount: Integer;


## Описание


Свойство FieldCount возвращает
 количество полей в источнике данных.


## Комментарии


Для получения наименования полей используйте свойство [IDmField.Name](../IDmField/IDmField.Name.htm).


## Пример


Использование свойства приведено в примере для [IDmClusteringDetails.CategoriesCount](../IDmClusteringDetails/IDmClusteringDetails.CategoriesCount.htm).
