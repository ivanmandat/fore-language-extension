# IDmDataSource.Field


## Синтаксис


		Field(Index: Integer): [IDmField](../IDmField/IDmField.htm);


## Параметры


Index. Индекс поля.


## Описание


Свойство Field возвращает указанное
 поле источника данных.


## Комментарии


Индексация полей сквозная и начинается с нуля. Для получения количества
 полей в источнике данных используйте свойство [IDmDataSource.FieldCount](IDmDataSource.FieldCount.htm).


## Пример


Использование свойства приведено в примере для [IDmMethod.Execute](../IDmMethod/IDmMethod.Execute.htm).
