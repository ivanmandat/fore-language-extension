# IDmDataSource.RefreshRecordCount


## Синтаксис


		RefreshRecordCount;


## Описание


Метод RefreshRecordCount обновляет
 количество записей в источнике данных.


## Комментарии


Для получения количества записей в источнике данных используйте свойство
 [IDmDataSource.RecordCount](IDmDataSource.RecordCount.htm).
