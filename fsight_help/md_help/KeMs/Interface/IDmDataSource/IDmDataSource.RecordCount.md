# IDmDataSource.RecordCount


## Синтаксис


		RecordCount: Integer;


## Описание


Свойство RecordCount возвращает
 количество записей в источнике данных.


## Комментарии


Для получения данных из источника используйте свойство [IDmDataSource.Item](IDmDataSource.Item.htm).


Для обновления количества записей в источнике данных используйте метод
 [IDmDataSource.RefreshRecordCount](IDmDataSource.RefreshRecordCount.htm).


Примечание.
 Если работа ведётся с табличным источником данных, то RecordCount
 всегда возвращает значение «0».
