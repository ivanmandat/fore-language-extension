# IDmDataSource.DataInColumns


## Синтаксис


		DataInColumns: Boolean;


## Описание


Свойство DataInColumns определяет признак расположения данных в
 столбцах.


## Комментарии


Допустимые значения:


	- True. Данные расположены
	 в столбцах;


	- False. Данные расположены
	 в строках.
