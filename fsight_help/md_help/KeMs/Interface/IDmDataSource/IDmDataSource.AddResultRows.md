# IDmDataSource.AddResultRows


## Синтаксис


		AddResultRows(Count: Integer);


## Параметры


Count. Количество добавляемых
 строк.


## Описание


Метод AddResultRows добавляет
 строки, содержащие результаты анализа.


## Комментарии


Добавление строк с результатами может использоваться при прогнозировании
 данных.
