# IMsDimensionFilterList.FindByKey


## Синтаксис


		FindByKey(DimKey: Integer): [IMsDimensionFilter](../IMsDimensionFilter/IMsDimensionFilter.htm);


## Параметры


DimKey. Ключ искомого измерения.


## Описание


Метод FindByKey выполняет поиск
 измерения в коллекции по его ключу.


## Комментарии


Если измерение с указанным ключом не найдено, то метод возвращает значение
 Null.
