# IMsDimensionFilterList.RemoveByKey


## Синтаксис


		RemoveByKey(DimKey: Integer): Boolean;


## Параметры


DimKey. Ключ удаляемого измерения.


## Описание


Метод RemoveByKey удаляет измерение
 из коллекции по его ключу.


## Комментарии


При успешном удалении измерения метод возвращает значение True,
 при неуспешном - False.
