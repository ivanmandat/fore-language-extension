# IMsDimensionFilterList


Сборка: Ms;


## Описание


Интерфейс IMsDimensionFilterList
 предназначен для работы с коллекцией измерений, по которым разрешена множественная
 отметка.


## Иерархия наследования


           IMsDimensionFilterList


## Комментарии


Для получения коллекции измерений, по которым разрешена множественная
 отметка, используйте свойство [IMsFormula.FreeDimensions](../IMsFormula/IMsFormula.FreeDimensions.htm).
