# IMsDimensionFilterList.FindById


## Синтаксис


		FindById(DimId: String): [IMsDimensionFilter](../IMsDimensionFilter/IMsDimensionFilter.htm);


## Параметры


DimId. Идентификатор искомого
 измерения.


## Описание


Метод FindById выполняет
 поиск измерения в коллекции по его идентификатору.


## Комментарии


Если измерение с указанным идентификатором не найдено, то метод возвращает
 значение Null.
