# IMsDimensionFilterList.Item


## Синтаксис


		Item(Index: Integer): [IMsDimensionFilter](../IMsDimensionFilter/IMsDimensionFilter.htm);


## Параметры


Index. Индекс измерения в коллекции.


## Описание


Свойство Item возвращает
 измерение из коллекции по его индексу.


## Комментарии


Индексация измерений в коллекции сквозная и начинается с нуля.


Для получения количества измерений в коллекции используйте свойство
 [IMsDimensionFilterList.Count](IMsDimensionFilterList.Count.htm).

Использование свойства приведено в примере для [IMsDimensionFilter.FilterDimension](../IMsDimensionFilter/IMsDimensionFilter.FilterDimension.htm).
