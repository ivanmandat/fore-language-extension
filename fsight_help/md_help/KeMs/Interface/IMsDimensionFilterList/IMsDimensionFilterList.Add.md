# IMsDimensionFilterList.Add


## Синтаксис


		Add(Value: [IDimensionModel](KeDims.chm::/interface/IDimensionModel/IDimensionModel.htm)):
		 [IMsDimensionFilter](../IMsDimensionFilter/IMsDimensionFilter.htm);


## Параметры


Value. Добавляемое измерение.


## Описание


Метод Add добавляет указанное
 измерение в коллекцию.


## Комментарии


Для получения указанного измерения из коллекции используйте свойство
 [IMsDimensionFilterList.Item](IMsDimensionFilterList.Item.htm).
