# IDmForecasting.Level


## Синтаксис


		Level: [DimCalendarLevel](KeDims.chm::/Enums/DimCalendarLevel.htm);


## Описание


Свойство Level определяет календарную
 динамику временного ряда, для которого выполняется прогноз.


## Комментарии


Свойство не учитывается, если для прогнозирования используется Грей-метод.


Для получения метода, применяемого для прогнозирования, используйте
 свойство [IDmForecasting.ForecastingMethod](IDmForecasting.ForecastingMethod.htm).
