# IDmForecasting.Labels


## Синтаксис


		Labels: Integer;


## Описание


Свойство Labels определяет индекс
 поля источника данных, значения которого будут использоваться в качестве
 подписей для оси X.


## Комментарии


Индексация полей сквозная и начинается с нуля.


Для определения источника данных для интеллектуального анализа используйте
 свойство [IDmMethod.InputDataSource](../IDmMethod/IDmMethod.InputDataSource.htm).
