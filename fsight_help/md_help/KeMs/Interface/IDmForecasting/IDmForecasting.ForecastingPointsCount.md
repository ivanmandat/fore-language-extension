# IDmForecasting.ForecastingPointsCount


## Синтаксис


		ForecastingPointsCount: Integer;


## Описание


Свойство ForecastingPointsCount
 определяет число периодов прогнозирования.


## Комментарии


Для определения календарной динамики прогнозирования используйте свойство
 [IDmForecasting.Level](IDmForecasting.Level.htm).
