# IDmForecasting


Сборка: Ms;


## Описание


Интерфейс IDmForecasting применяется
 для выполнения интеллектуального анализа данных «Прогнозирование».


## Иерархия наследования


           [IDmMethodDetails](../IDmMethodDetails/IDmMethodDetails.htm)


           [IDmAttributesDetails](../IDmAttributesDetails/IDmAttributesDetails.htm)


           IDmForecasting


## Комментарии


Данный вид анализа выполняет прогнозирование значений временного ряда.

		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ForecastingMethod](IDmForecasting.ForecastingMethod.htm)
		 Свойство ForecastingMethod
		 возвращает метод, применяемый для прогнозирования значений.


		 ![](../../Property_Image.gif)
		 [ForecastingPointsCount](IDmForecasting.ForecastingPointsCount.htm)
		 Свойство ForecastingPointsCount
		 определяет число периодов прогнозирования.

		 Свойство Labels
		 определяет индекс поля источника данных, значения которого будут
		 использоваться в качестве подписей для оси X.


		 ![](../../Property_Image.gif)
		 [Level](IDmForecasting.Level.htm)
		 Свойство Level
		 определяет календарную динамику временного ряда, для которого
		 выполняется прогноз.
