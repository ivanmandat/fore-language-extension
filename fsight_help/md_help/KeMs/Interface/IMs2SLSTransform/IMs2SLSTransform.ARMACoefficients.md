## Описание


Свойство ARMACoefficients
 возвращает оценённые коэффициенты авторегрессии скользящего среднего.


## Комментарии


Оценка коэффициентов производится на стадии идентификации модели. Для
 этого вызовите метод [IMsStochasticMethod.Identify](../IMsStochasticMethod/IMsStochasticMethod.Identify.htm).
