# IMs2SLSTransform.CoefUncertaintyInSECalc


## Синтаксис


		CoefUncertaintyInSECalc: Boolean;


## Описание


Свойство CoefUncertaintyInSECalc
 определяет признак учёта неопределённости коэффициентов при расчёте доверительных
 границ.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 При расчёте доверительных границ учитывается, что коэффициенты найдены
	 приближённо;


	- False. При расчёте доверительных
	 границ не учитывается, что коэффициенты найдены приближённо.
