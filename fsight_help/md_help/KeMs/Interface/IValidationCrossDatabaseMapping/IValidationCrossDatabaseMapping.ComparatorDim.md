# IValidationCrossDatabaseMapping.ComparatorDim


## Синтаксис


ComparatorDim: [IDimensionModel](KeDims.chm::/interface/IDimensionModel/IDimensionModel.htm);


## Описание


Свойство ComparatorDim
 определяет справочник, на котором строится атрибут в базе данных временных
 рядов, с которой производится сравнение.


## Комментарии


Для задания отметки по данному атрибуту используйте свойство [IValidationCrossDatabaseMapping.ComparatorSelectionMapping](IValidationCrossDatabaseMapping.ComparatorSelectionMapping.htm).
