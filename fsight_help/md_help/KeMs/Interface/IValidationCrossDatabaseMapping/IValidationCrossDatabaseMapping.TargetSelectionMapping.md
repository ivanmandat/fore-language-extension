# IValidationCrossDatabaseMapping.TargetSelectionMapping


## Синтаксис


TargetSelectionMapping: [IDimSelection](KeDims.chm::/interface/IDimSelection/IDimSelection.htm);


## Описание


Свойство TargetSelectionMapping
 определяет отметку измерения в сравниваемом источнике данных.


## Комментарии


Для задания справочника, на основе которого строится данное измерение,
 используйте свойство [IValidationCrossDatabaseMapping.TargetDim](IValidationCrossDatabaseMapping.TargetDim.htm).
