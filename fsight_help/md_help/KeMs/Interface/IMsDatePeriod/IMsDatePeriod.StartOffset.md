# IMsDatePeriod.StartOffset


## Синтаксис


		StartOffset: Integer;


## Описание


Свойство StartOffset определяет
 количество периодов, на которое сдвигается дата начала периода.


## Комментарии


Для задания количества периодов, на которое сдвигается дата окончания
 периода используйте свойство [IMsDatePeriod.EndOffset](IMsDatePeriod.EndOffset.htm).
