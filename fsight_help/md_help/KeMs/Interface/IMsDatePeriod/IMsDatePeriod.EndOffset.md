# IMsDatePeriod.EndOffset


## Синтаксис


		EndOffset: Integer;


## Описание


Свойство EndOffset определяет
 количество периодов, на которое сдвигается дата окончания периода.


## Комментарии


Для задания количества периодов, на которое сдвигается дата начала периода
 используйте свойство [IMsDatePeriod.StartOffset](IMsDatePeriod.StartOffset.htm).
