# IMsDatePeriod.AutoPeriod


## Синтаксис


		AutoPeriod: Integer;


## Описание


Свойство AutoPeriod определяет
 условия, которые используются для вычисления дат начала и окончания периода.


## Комментарии


Условия задаются с помощью перечисления [MsDateAutoPeriodOptions](../../Enums/MsDateAutoPeriodOptions.htm).

Использование свойства приведено в примере для [IMsProblemCalculationSettings.OutputPeriod](../IMsProblemCalculationSettings/IMsProblemCalculationSettings.OutputPeriod.htm).
