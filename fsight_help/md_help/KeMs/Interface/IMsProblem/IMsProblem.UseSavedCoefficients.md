# IMsProblem.UseSavedCoefficients


## Синтаксис


UseSavedCoefficients: Boolean;


## Описание


Свойство UseSavedCoefficients определяет, будут ли использоваться сохраненные коэффициенты моделей при расчете задачи. Если свойству установлено значение True, при расчете задачи будут использоваться сохраненные коэффициенты. По умолчанию установлено значение False.
