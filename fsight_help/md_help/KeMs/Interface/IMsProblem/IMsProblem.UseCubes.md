# IMsProblem.UseCubes


## Синтаксис


UseCubes: Boolean;


## Описание


Свойство UseCubes определяет,
 использовать ли кубы для отчётов по переменным.


## Комментарии


Допустимые значения:


	- True. Кубы используются
	 при создании отчетов по переменным;


	- False. Кубы не используются
	 для отчетов по переменным.


При создании задачи из Fore или веб-приложения свойство по умолчанию
 принимает значение False, при
 создании из настольного приложения - True.
