# IMsGreyForecastTransform.MissingData


## Синтаксис


MissingData: [IMissingData](StatLib.chm::/Interface/IMissingData/IMissingData.htm);


## Описание


Свойство MissingData возвращает объект, содержащий параметры обработки пропусков исходного ряда.


## Комментарии


По умолчанию используется метод [MissingDataMethod.None](StatLib.chm::/Enums/MissingDataMethod.htm), т.е. обработка пропусков не выполняется.


Метод обработки пропусков [MissingDataMethod.Casewise](StatLib.chm::/Enums/MissingDataMethod.htm) не может быть использован из-за смещения данных.
