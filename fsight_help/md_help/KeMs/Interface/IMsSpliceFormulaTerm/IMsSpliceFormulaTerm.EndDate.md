# IMsSpliceFormulaTerm.EndDate


## Синтаксис


EndDate: [DateTime](ForeSys.chm::/Class/DateTime/DateTime.htm);


## Описание


Свойство EndDate
 определяет дату окончания совмещения ряда.


## Комментарии


Для задания даты начала совмещения рядов используйте свойство [IMsSpliceFormulaTerm.StartDate](IMsSpliceFormulaTerm.StartDate.htm).
