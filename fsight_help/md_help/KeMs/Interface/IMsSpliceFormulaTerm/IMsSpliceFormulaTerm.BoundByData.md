# IMsSpliceFormulaTerm.BoundByData


## Синтаксис


BoundByData: Boolean;


## Описание


Свойство BoundByData определяет,
 игнорировать ли даты начала и окончания совмещения ряда.


## Комментарии


Допустимые значения:


	- True. Даты игнорируются;


	- False. Для совмещения
	 рядов используются указанные даты начала ([IMsSpliceFormulaTerm.StartDate](IMsSpliceFormulaTerm.StartDate.htm))
	 и окончания ([IMsSpliceFormulaTerm.EndDate](IMsSpliceFormulaTerm.EndDate.htm)).
