# IMsSpliceFormulaTerm.StartDate


## Синтаксис


StartDate: [DateTime](ForeSys.chm::/Class/DateTime/DateTime.htm);


## Описание


Свойство StartDate
 определяет дату начала совмещения ряда.


## Комментарии


Для задания даты окончания совмещения рядов используйте свойство [IMsSpliceFormulaTerm.EndDate](IMsSpliceFormulaTerm.EndDate.htm).
