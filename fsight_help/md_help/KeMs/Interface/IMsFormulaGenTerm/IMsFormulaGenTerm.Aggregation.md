# IMsFormulaGenTerm.Aggregation


## Синтаксис


Aggregation: [BasicAggregatorOperation](KeMatrix.chm::/Enums/BasicAggregatorOperation.htm);


## Описание


Свойство Aggregation возвращает
 метод агрегации, который использовался при расчёте.


## Комментарии


Свойство актуально, если для операнда задан метод агрегации или для
 расчёта используется модель матричной агрегации. В этом случае в каждой
 [точке](IMsFormulaGenTerm.Point.htm) будет информация о используемых
 при агрегации значениях ([количество](../IMsFormulaGenTermPoint/IMsFormulaGenTermPoint.ValuesCount.htm)
 и [фактические
 значения](../IMsFormulaGenTermPoint/IMsFormulaGenTermPoint.Value.htm)).
