# IMsParametrizedDimension


Сборка: Ms;


## Описание


Интерфейс IMsParametrizedDimension
 предназначен для работы с параметризированным измерением.


## Иерархия наследования


           [INamedEntityModel](ForeSys.chm::/Interface/INamedEntityModel/INamedEntityModel.htm)


           IMsParametrizedDimension


## Комментарии


Для работы со всеми параметризованными измерениями используйте интерфейс
 [IMsParametrizedDimensions](../IMsParametrizedDimensions/IMsParametrizedDimensions.htm).
