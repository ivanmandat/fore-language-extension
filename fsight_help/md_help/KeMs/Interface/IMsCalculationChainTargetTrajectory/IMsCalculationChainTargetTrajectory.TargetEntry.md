## Описание


Свойство TargetEntry возвращает
 целевую функцию, содержащуюся в [цепочке
 расчёта](UiModelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_3_MetaModel/uimodelling_metamodel_2.htm) метамодели, которой принадлежит траектория.


## Комментарии


С помощью полученной целевой функции задаются значения траектории.
