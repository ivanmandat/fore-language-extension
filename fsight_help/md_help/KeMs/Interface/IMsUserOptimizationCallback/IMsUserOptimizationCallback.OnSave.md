# IMsUserOptimizationCallback.OnSave


## Синтаксис


OnSave(Xml: [IXmlDomElement](ModXml.chm::/develop.htm));


## Параметры


Xml. XML-файл, в который будут
 сохранены заданные параметры пользовательской задачи оптимизации.


## Описание


Метод OnSave реализует событие,
 возникающее после сохранения целевой функции.
