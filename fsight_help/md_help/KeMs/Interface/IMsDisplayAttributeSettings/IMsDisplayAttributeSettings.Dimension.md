# IMsDisplayAttributeSettings.Dimension


## Синтаксис


		Dimension: [IDimensionModel](KeDims.chm::/interface/IDimensionModel/IDimensionModel.htm);


## Описание


Свойство Dimension возвращает
 измерение, для атрибутов которого задаются параметры отображения.


## Комментарии


Для задания отображаемого атрибута измерения используйте свойство [IMsDisplayAttributeSettings.AttributeId](IMsDisplayAttributeSettings.AttributeId.htm).
