# IMsDisplayAttributeSettings


Сборка: Ms;


## Описание


Интерфейс IMsDisplayAttributeSettings
 предназначен для работы с параметрами отображения атрибутов в строковом
 представлении элементов выражения.


## Иерархия наследования


           [INamedEntity](KeSom.chm::/Interface/INamedEntity/INamedEntity.htm)


           IMsDisplayAttributeSettings


## Комментарии


Для работы с коллекцией параметров используйте интерфейс [IMsDisplayAttributeSettingsList](../IMsDisplayAttributeSettingsList/IMsDisplayAttributeSettingsList.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AttributeId](IMsDisplayAttributeSettings.AttributeId.htm)
		 Свойство AttributeId
		 определяет идентификатор атрибута, значение которого будет отображаться
		 в строковом представлении элемента выражения.


		 ![](../../Property_Image.gif)
		 [Dimension](IMsDisplayAttributeSettings.Dimension.htm)
		 Свойство Dimension
		 возвращает измерение, для атрибутов которого задаются параметры
		 отображения.
