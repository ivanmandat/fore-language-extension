# IMsDisplayAttributeSettings.AttributeId


## Синтаксис


		AttributeId: String;


## Описание


Свойство AttributeId определяет
 идентификатор атрибута, значение которого будет отображаться в строковом
 представлении элемента выражения.


## Комментарии


Для получения измерения, содержащего атрибут, используйте свойство [IMsDisplayAttributeSettings.Dimension](IMsDisplayAttributeSettings.Dimension.htm).
