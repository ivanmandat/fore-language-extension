# IValidationDateSettings


## Описание


Интерфейс IValidationDateSettings
 содержит свойства для определения параметров даты расчета валидации.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Date](IValidationDateSettings.Date.htm)
		 Свойство Date определяет
		 дату расчета валидации.


		 ![](../../Property_Image.gif)
		 [DateOptions](IValidationDateSettings.DateOptions.htm)
		 Свойство DateOptions
		 определяет способ расчета даты.


		 ![](../../Property_Image.gif)
		 [Lag](IValidationDateSettings.Lag.htm)
		 Свойство Lag определяет
		 количество периодов, на которое должна быть изменена текущая дата.
