# IValidationDateSettings.Lag


## Синтаксис


Lag: Integer;


## Описание


Свойство Lag определяет количество периодов, на которое должна быть изменена текущая дата.


## Комментарии


Данное свойство учитывается, если значение свойства [IValidationDateSettings.DateOptions](IValidationDateSettings.DateOptions.htm) - [ValidationDateOptions.RelativeDate](../../Enums/ValidationDateOptions.htm).
