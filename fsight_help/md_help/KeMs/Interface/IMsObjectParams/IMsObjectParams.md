# IMsObjectParams


Сборка: Ms;


## Описание


Интерфейс IMsObjectParams
 описывает коллекцию связей параметров моделирования с несколькими объектами
 репозитория.


## Иерархия наследования


           IMsObjectParams


## Комментарии


Для работы со связью параметра со значением параметра объекта репозитория
 используйте интерфейс [IMsObjectParamValue](../IMsObjectParamValue/IMsObjectParamValue.htm).
