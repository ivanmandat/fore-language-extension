# IMsBpfResult


Сборка: Ms;


## Описание


Интерфейс IMsBpfResult
 предназначен для работы с результатами расчёта фильтра Бакстера-Кинга.


## Иерархия наследования


           [IMsResultObject](../IMsResultObject/IMsResultObject.htm)


           IMsBpfResult


## Комментарии


Данный интерфейс используется в пользовательских функциях и методах,
 предназначенных для расчёта в детерминированном уравнении.
