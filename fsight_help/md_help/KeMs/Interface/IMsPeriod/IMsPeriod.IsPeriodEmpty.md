# IMsPeriod.IsPeriodEmpty


## Синтаксис


		IsPeriodEmpty: Boolean;


## Описание


Свойство IsPeriodEmpty возвращает
 признак наличия изменений в периоде, внесённых пользователем.


## Комментарии


Свойство возвращает значение True,
 если пользователь менял какие-либо настройки периода. При этом заданные
 настройки будут использоваться во время расчёта.


Свойство вернёт значение False,
 если пользователь не менял настройки периода или был вызван метод [IMsDatePeriod.ClearPeriod](../IMsDatePeriod/IMsDatePeriod.ClearPeriod.htm). При этом
 во время расчёта для периода будут использоваться значения по умолчанию.
