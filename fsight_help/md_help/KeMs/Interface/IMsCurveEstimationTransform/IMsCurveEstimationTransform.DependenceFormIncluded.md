# IMsCurveEstimationTransform.DependenceFormIncluded


## Синтаксис


		DependenceFormIncluded(Index: Integer):
		 Boolean;


## Параметры


Index. Индекс модели.


## Описание


Свойство DependenceFormIncluded
 определяет, какие виды зависимостей будут рассчитываться и в дальнейшем
 оцениваться, выбранным критерием отбора моделей.


## Комментарии


Для выбора необходимой модели рекомендуется использовать перечислимый
 тип [DependenceType](StatLib.chm::/Enums/DependenceType.htm).
