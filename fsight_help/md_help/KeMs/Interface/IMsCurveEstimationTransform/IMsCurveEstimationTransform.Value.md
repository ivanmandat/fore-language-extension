# IMsCurveEstimationTransform.Value


## Синтаксис


		Value: Double;


## Описание


Свойство Value определяет значение
 для экстраполяции.


## Комментарии


Свойство учитывается, если свойство [IMsCurveEstimationTransform.ExtrapolateType](IMsCurveEstimationTransform.ExtrapolateType.htm)
 имеет значение DependenceType.Value.
