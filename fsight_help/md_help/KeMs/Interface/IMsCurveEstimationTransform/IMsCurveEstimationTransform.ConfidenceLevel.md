# IMsCurveEstimationTransform.ConfidenceLevel


## Синтаксис


		ConfidenceLevel: Double;


## Описание


Свойство ConfidenceLevel определяет
 значимость доверительных границ.


## Комментарии


По умолчанию свойству установлено значение 0,95.
