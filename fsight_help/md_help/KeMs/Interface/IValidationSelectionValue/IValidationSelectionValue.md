# IValidationSelectionValue


Сборка: Ms;


## Описание


Интерфейс IValidationSelectionValue
 предназначен для работы с элементом набора данных, по которым выполняется
 валидация.


## Иерархия наследования


           [INamedEntityModel](ForeSys.chm::/Interface/INamedEntityModel/INamedEntityModel.htm)


           IValidationSelectionValue


## Комментарии


Элемент набора данных задаётся отметкой по какому-либо измерению. Для
 работы со всем набором используйте интерфейс [IValidationSelectionValueList](../IValidationSelectionValueList/IValidationSelectionValueList.htm).
