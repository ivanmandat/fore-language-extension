# IMsScenarioTreeEntry


## Описание


Интерфейс IMsScenarioTreeEntry
 содержит базовые свойства элементов дерева сценариев.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Key](IMsScenarioTreeEntry.Key.htm)
		 Свойство Key возвращает
		 ключ элемента дерева сценариев.


		 ![](../../Property_Image.gif)
		 [Name](IMsScenarioTreeEntry.Name.htm)
		 Свойство Name определяет
		 наименование элемента дерева сценариев.


		 ![](../../Property_Image.gif)
		 [ParentEntries](IMsScenarioTreeEntry.ParentEntries.htm)
		 Свойство ParentEntries
		 определяет родительскую ветку, в которой содержится данный элемент.
