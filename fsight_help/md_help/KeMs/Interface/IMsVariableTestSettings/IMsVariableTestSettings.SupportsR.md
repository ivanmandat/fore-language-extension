# IMsVariableTestSettings.SupportsR


## Синтаксис


SupportsR: Boolean;


## Описание


Свойство SupportsR определяет,
 возможен ли расчет описательной статистики с помощью R.


## Комментарии


Допустимые значения:


	- True. Описательная статистика
	 может быть рассчитана с помощью R. Для этого используйте свойство
	 [IMsVariableTestSettings.UseR](IMsVariableTestSettings.UseR.htm);


	- False. Описательная
	 статистика не может быть рассчитана с помощью R.
