# IMsVariableTestSettings.EndDate


## Синтаксис


EndDate: DateTime;


## Описание


Свойство EndDate определяет конечную точку расчёта.


## Комментарии


Для определения начальной точки расчёта используйте свойство [IMsVariableTestSettings.StartDate](IMsVariableTestSettings.StartDate.htm).
