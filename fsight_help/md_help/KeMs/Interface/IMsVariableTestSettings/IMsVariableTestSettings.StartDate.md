# IMsVariableTestSettings.StartDate


## Синтаксис


StartDate: DateTime;


## Описание


Свойство StartDate определяет начальную точку расчёта.


## Комментарии


Для определения конечной точки расчёта используйте свойство [IMsVariableTestSettings.EndDate](IMsVariableTestSettings.EndDate.htm).
