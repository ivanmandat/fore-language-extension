# IMsVariableTestSettings.MissingData


## Синтаксис


MissingData: [IMissingData](StatLib.chm::/Interface/IMissingData/IMissingData.htm);


## Описание


Свойство MissingData возвращает параметры обработки пропусков.


## Комментарии


По умолчанию для обработки пропусков используется метод MissingDataMethod.Casewise, т.е. расчёты ведутся без учета отсутствующих значений.
