# IMsVariableTestSettings.IncludedSources


## Синтаксис


IncludedSources: [IMsVariableTestDataList](../IMsVariableTestDataList/IMsVariableTestDataList.htm);


## Описание


Свойство IncludedSources возвращает коллекцию переменных, для которых рассчитывается описательная статистика.


## Комментарии


Для определения параметров обработки пропусков в данных переменных используйте свойство [IMsVariableTestSettings.MissingData](IMsVariableTestSettings.MissingData.htm).
