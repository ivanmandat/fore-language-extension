# IMsModelParams.FindById


## Синтаксис


FindById(Id: String): [IMsModelParam](../IMsModelParam/IMsModelParam.htm);


## Параметры


Id. Идентификатор искомого параметра.


## Описание


Метод FindById осуществляет поиск параметра в коллекции по идентификатору.


## Комментарии


Если параметр с указанным идентификатором не найден, то метод возвращает значение Null.
