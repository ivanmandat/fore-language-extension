# IMsModelParams.Clear


## Синтаксис


Clear;


## Описание


Метод Clear удаляет все параметры из коллекции.


## Комментарии


После вызова метода значение свойства [IMsModelParams.Count](IMsModelParams.Count.htm) будет обнулено.
