# IMsModelParams.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Синтаксис


Remove(Index: Integer): boolean;


## Параметры


Index. Индекс параметра в коллекции.


## Описание


Метод Remove удаляет параметр из коллекции по индексу.


## Комментарии


Индексация параметров в коллекции сквозная и начинается с нуля. Для получения количества параметров в коллекции используйте свойство [IMsModelParams.Count](IMsModelParams.Count.htm).


Если удаление выполнено успешно, то метод возвращает значение True, в обратном случае - False.
