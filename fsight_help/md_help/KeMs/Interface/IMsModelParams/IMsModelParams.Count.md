# IMsModelParams.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество параметров в коллекции.


## Комментарии


Для получения определенного параметра из коллекции используйте свойство [IMsModelParams.Item](IMsModelParams.Item.htm).
