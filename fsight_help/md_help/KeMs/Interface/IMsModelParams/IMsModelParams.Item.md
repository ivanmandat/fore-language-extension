# IMsModelParams.Item


## Синтаксис


Item(Index: Integer): [IMsModelParam](../IMsModelParam/IMsModelParam.htm);


## Параметры


Item. Индекс параметра в коллекции.


## Описание


Свойство Item возвращает параметр из коллекции по индексу.


## Комментарии


Индексация параметров в коллекции сквозная и начинается с нуля. Для получения количества параметров в коллекции используйте свойство [IMsModelParams.Count](IMsModelParams.Count.htm).
