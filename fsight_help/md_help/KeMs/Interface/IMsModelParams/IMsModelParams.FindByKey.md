# IMsModelParams.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IMsModelParam](../IMsModelParam/IMsModelParam.htm);


## Параметры


Key. Ключ искомого параметра.


## Описание


Метод FindByKey осуществляет поиск параметра в коллекции по ключу.


## Комментарии


Если параметр с указанным ключом не найден, то метод возвращает значение Null.
