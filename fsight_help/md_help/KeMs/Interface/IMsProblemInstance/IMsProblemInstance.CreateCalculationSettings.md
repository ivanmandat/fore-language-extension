# IMsProblemInstance.CreateCalculationSettings


## Синтаксис


CreateCalculationSettings: [IMsProblemCalculationSettings](../IMsProblemCalculationSettings/IMsProblemCalculationSettings.htm);


## Описание


Метод CreateCalculationSettings
 создает объект, в котором содержатся настройки открытого экземпляра задачи
 моделирования для расчёта.


## Комментарии


Для выполнения расчёта открытого экземпляра задачи моделирования
 с заданными настройками используйте метод [IMsProblemInstance.Calculate](IMsProblemInstance.Calculate.htm).
