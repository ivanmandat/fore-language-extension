# IMsDetermAggregationTransform.Rebase


## Синтаксис


		Rebase: Boolean;


## Описание


Свойство Rebase определяет признак
 использования функции Rebase при вычислении агрегации.


## Комментарии


Допустимые значения:


	- True. Функция Rebase
	 используется. В этом случае должно быть определено значение свойства
	 [IMsDetermAggregationTransform.LevelRebaseDate](IMsDetermAggregationTransform.LevelRebaseDate.htm);


	- False. Функция Rebase
	 не используется.
