# IMsDetermAggregationTransform.Level


## Синтаксис


		Level: Boolean;


## Описание


Свойство Level определяет признак
 использования функции Level при вычислении агрегации.


## Комментарии


Допустимые значения:


	- True. Функция Level
	 используется. В этом случае должно быть определено значение свойства
	 [IMsDetermAggregationTransform.LevelRebaseDate](IMsDetermAggregationTransform.LevelRebaseDate.htm);


	- False. Функция Level
	 не используется.
