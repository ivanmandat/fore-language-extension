# IMsFormulaGenTermIterator


Сборка: Ms;


## Описание


Интерфейс IMsFormulaGenTermIterator
 предназначен для работы с итератором по термам, которые используются при
 составлении каких-либо формул.


## Иерархия наследования


           IMsFormulaGenTermIterator


## Комментарии


Для получения итератора по отдельным термам используйте свойство [IMsFormulaGenInstanceIterator.Operands](../IMsFormulaGenInstanceIterator/IMsFormulaGenInstanceIterator.Operands.htm).


Каждый терм содержит параметры, применяемые при расчёте формулы.
