# IMsFormulaGenTermIterator.Eof


## Синтаксис


Eof: Boolean;


## Описание


Метод Eof возвращает признак
 нахождения на последней записи в итераторе.


## Комментарии


Допустимые значения:


	- True. Курсор расположен
	 на последней записи в итераторе;


	- False. Курсор расположен
	 не на последней записи в итераторе.
