# IMsFormulaGenTermIterator.Next


## Синтаксис


Next: Boolean;


## Описание


Метод Next осуществляет переход
 к следующей записи в итераторе.


## Комментарии


Если при выполнении метода достигнут конец итератора, то метод возвращает
 значение True, иначе - False.
