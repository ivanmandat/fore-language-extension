# IMsDisplaySettings.DisplayParentAs


## Синтаксис


		DisplayParentAs: [MsDisplayParent](../../Enums/MsDisplayParent.htm);


## Описание


Свойство DisplayParentAs
 определяет режим отображения родительского элемента в строковом представлении
 элемента выражения.


## Комментарии


По умолчанию отображается наименование родительского элемента.
