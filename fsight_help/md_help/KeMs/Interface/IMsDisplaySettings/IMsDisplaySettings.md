# IMsDisplaySettings


Сборка: Ms;


## Описание


Интерфейс IMsDisplaySettings
 содержит свойства для работы с параметрами отображения объектов.


## Иерархия наследования


           IMsDisplaySettings


## Комментарии


Для получения параметров отображения модели используйте свойство [IMsFormulaTransform.DisplaySettings](../IMsFormulaTransform/IMsFormulaTransform.DisplaySettings.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DisplayAttributes](IMsDisplaySettings.DisplayAttributes.htm)
		 Свойство DisplayAttributes
		 возвращает коллекцию настроек отображения атрибутов в строковом
		 представлении элементов выражения.


		 ![](../../Property_Image.gif)
		 [DisplayParentAs](IMsDisplaySettings.DisplayParentAs.htm)
		 Свойство DisplayParentAs
		 определяет режим отображения родительского элемента в строковом
		 представлении элемента выражения.


		 ![](../../Property_Image.gif)
		 [DisplayTermsAs](IMsDisplaySettings.DisplayTermsAs.htm)
		 Свойство DisplayTermsAs
		 определяет параметры отображения объектов.
