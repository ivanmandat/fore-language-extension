# IMsOptimizationFormulaTerm.InitApproximation


## Синтаксис


InitApproximation: Array;


## Описание


Свойство InitApproximation
 определяет начальные значения терма.


## Комментарии


Если свойству InitApproximation
 присвоить значение Null, то начальные
 значения терма будут определяться автоматически.
