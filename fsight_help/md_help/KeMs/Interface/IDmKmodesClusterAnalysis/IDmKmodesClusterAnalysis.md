# IDmKmodesClusterAnalysis


Сборка: Ms;


## Описание


Интерфейс IDmKmodesClusterAnalysis
 предназначен для кластеризации данных методом «K-мод».

## Комментарии


Исходные данные разбиваются на небольшое число категорий (до 10) таким
 образом, чтобы расстояние между группами было сведено к минимуму. Метод
 работает на категориальных значениях.
