# IMsUserTransformImplementation


## Описание


Интерфейс IMsUserTransformImplementation
 реализует свойства и методы, необходимые для расчета модели с использованием
 пользовательского алгоритма. Данный интерфейс должен быть переопределен
 в пользовательском классе.
