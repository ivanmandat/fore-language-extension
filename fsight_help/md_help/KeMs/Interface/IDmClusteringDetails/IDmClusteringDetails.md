# IDmClusteringDetails


Сборка: Ms;


## Описание


Интерфейс IDmClusteringDetails
 является базовым для работы с методами кластеризации, используемыми для
 интеллектуального анализа данных.


## Иерархия наследования


           [IDmMethodDetails](../IDmMethodDetails/IDmMethodDetails.htm)


           [IDmAttributesDetails](../IDmAttributesDetails/IDmAttributesDetails.htm)


           IDmClusteringDetails


## Комментарии


Данный интерфейс может быть приведён к одному из следующих интерфейсов:


	- [IDmKmodesClusterAnalysis](../IDmKmodesClusterAnalysis/IDmKmodesClusterAnalysis.htm).
	 Кластеризация методом K-мод;


	- [IDmSelfOrganizingMap](../IDmSelfOrganizingMap/IDmSelfOrganizingMap.htm).
	 Кластеризация с помощью самоорганизующихся карт Кохонена.
