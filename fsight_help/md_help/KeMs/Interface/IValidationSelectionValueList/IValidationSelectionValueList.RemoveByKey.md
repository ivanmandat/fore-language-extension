# IValidationSelectionValueList.RemoveByKey


## Синтаксис


RemoveByKey(DimensionKey: Integer): Boolean;


## Параметры


DimensionKey. Ключ справочника.


## Описание


Метод RemoveByKey удаляет
 отметку по ключу справочника.


## Комментарии


Если удаление выполнено успешно, то метод возвращает значение True, иначе - False.
