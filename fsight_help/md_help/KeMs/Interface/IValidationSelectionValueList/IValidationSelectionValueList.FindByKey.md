# IValidationSelectionValueList.FindByKey


## Синтаксис


FindByKey(DimensionKey: Integer): [IValidationSelectionValue](../IValidationSelectionValue/IValidationSelectionValue.htm);


## Параметры


DimensionKey. Ключ справочника.


## Описание


Метод FindByKey выполняет поиск
 отметки по ключу справочника.


## Комментарии


Если отметка не найдена, то метод возвращает значение Null.
