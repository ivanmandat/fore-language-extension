# IValidationSelectionValueList.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс отметки в наборе.


## Описание


Метод Remove удаляет указанную
 отметку из набора.


## Комментарии


Индексация отметок в наборе сквозная и начинается с нуля. Для получения
 количества отметок в наборе используйте свойство [IValidationSelectionValueList.Item](IValidationSelectionValueList.Item.htm).


Если удаление выполнено успешно, то метод возвращает значение True, иначе - False.
