# IValidationSelectionValueList.FindById


## Синтаксис


FindById(DimensionId: String): [IValidationSelectionValue](../IValidationSelectionValue/IValidationSelectionValue.htm);


## Параметры


DimensionId. Идентификатор
 справочника.


## Описание


Метод FindById выполняет
 поиск отметки по идентификатору справочника.


## Комментарии


Если отметка не найдена, то метод возвращает значение Null.
