# IValidationSelectionValueList.Item


## Синтаксис


Item(Index: Integer): [IValidationSelectionValue](../IValidationSelectionValue/IValidationSelectionValue.htm);


## Параметры


Index. Индекс отметки в наборе.


## Описание


Свойство Item возвращает
 заданную отметку из набора.


## Комментарии


Индексация отметок в наборе сквозная и начинается с нуля. Для получения
 количества отметок в наборе используйте свойство [IValidationSelectionValueList.Item](IValidationSelectionValueList.Item.htm).
