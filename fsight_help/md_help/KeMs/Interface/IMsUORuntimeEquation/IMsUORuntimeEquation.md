# IMsUORuntimeEquation


Сборка: Ms;


## Описание


Интерфейс IMsUORuntimeEquation
 предназначен для работы уравнением, которое используется для расчёта значений
 пользовательской задачи оптимизации.


## Иерархия наследования


IMsUORuntimeEquation


## Комментарии


Уравнение можно получить, используя свойство [IMsUORuntimeEquations.Item](../IMsUORuntimeEquations/IMsUORuntimeEquations.Item.htm)
 во время расчёта задачи.
