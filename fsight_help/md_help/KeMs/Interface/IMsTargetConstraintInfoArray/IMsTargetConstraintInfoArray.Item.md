# IMsTargetConstraintInfoArray.Item


## Синтаксис


Item(Index: Integer): [IMsTargetConstraintInfo](../IMsTargetConstraintInfo/IMsTargetConstraintInfo.htm);


## Параметры


Index. Индекс значения.


## Описание


Свойство Item возвращает
 указанное значение из коллекции.


## Комментарии


Индексация элементов в коллекции сквозная и начинается с нуля. Для получения
 количества элементов в коллекции используйте свойство [IMsTargetConstraintInfoArray.Count](IMsTargetConstraintInfoArray.Count.htm).
