# IMsTargetConstraintInfoArray.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемого значения.


## Описание


Метод Remove удаляет указанное
 значение из коллекции.


## Комментарии


Если удаление выполнено успешно, то метод возвращает значение True, иначе - False.


Индексация элементов в коллекции сквозная и начинается с нуля. Для получения
 количества элементов в коллекции используйте свойство [IMsTargetConstraintInfoArray.Count](IMsTargetConstraintInfoArray.Count.htm).
