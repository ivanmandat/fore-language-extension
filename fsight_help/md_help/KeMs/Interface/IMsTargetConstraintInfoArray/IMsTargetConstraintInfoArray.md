# IMsTargetConstraintInfoArray


Сборка: Ms;


## Описание


Интерфейс IMsTargetConstraintInfoArray
 предназначен для работы с коллекцией значений для ограничения целевой
 функции.


## Иерархия наследования


           IMsTargetConstraintInfoArray


## Комментарии


Для получения коллекции значений для ограничений целевой функции используйте
 свойство [IMsTargetConstraint.ConstraintInfoArray](../IMsTargetConstraint/IMsTargetConstraint.ConstraintInfoArray.htm).
