# IMsCalculationChainFilter


Сборка: Ms;


## Описание


Интерфейс IMsCalculationChainFilter
 предназначен для работы с коллекцией элементов, рассчитываемых задачей.


## Иерархия наследования


           IMsCalculationChainFilter


## Комментарии


Для получения коллекции элементов используйте свойство [IMsProblemCalculationSettings.EntriesFilter](../IMsProblemCalculationSettings/IMsProblemCalculationSettings.EntriesFilter.htm).
