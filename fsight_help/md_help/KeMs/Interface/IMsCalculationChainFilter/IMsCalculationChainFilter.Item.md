# IMsCalculationChainFilter.Item


## Синтаксис


		Item(Index: Integer): [IMsCalculationChainEntry](../IMsCalculationChainEntry/IMsCalculationChainEntry.htm);


## Параметры


Index. Индекс элемента коллекции.


## Описание


Свойство Item возвращает
 из коллекции элемент с заданным индексом.


## Комментарии


Индексация элементов в коллекции сквозная и начинается с нуля. Для получения
 количества элементов в коллекции используйте свойство [IMsCalculationChainFilter.Count](IMsCalculationChainFilter.Count.htm).
