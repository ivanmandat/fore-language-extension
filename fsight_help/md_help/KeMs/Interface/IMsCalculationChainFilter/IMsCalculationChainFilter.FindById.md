# IMsCalculationChainFilter.FindById


## Синтаксис


		FindById(Id: String): [IMsCalculationChainEntry](../IMsCalculationChainEntry/IMsCalculationChainEntry.htm);


## Параметры


Id. Идентификатор элемента
 коллекции.


## Описание


Метод FindById выполняет
 поиск в коллекции элемента с заданным идентификатором.


## Комментарии


Если искомый элемент не найден, то метод возвращает значение Null.
