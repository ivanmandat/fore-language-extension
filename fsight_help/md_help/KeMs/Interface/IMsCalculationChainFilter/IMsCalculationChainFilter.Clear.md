# IMsCalculationChainFilter.Clear


## Синтаксис


		Clear;


## Описание


Метод Clear очищает коллекцию
 элементов.


## Комментарии


После вызова метода Clear значение
 свойства [IMsCalculationChainFilter.Count](IMsCalculationChainFilter.Count.htm)
 будет равно нулю.


Для удаления одного элемента из коллекции используйте методы [IMsCalculationChainFilter.Remove](IMsCalculationChainFilter.Remove.htm)
 и [IMsCalculationChainFilter.RemoveByKey](IMsCalculationChainFilter.RemoveByKey.htm).
