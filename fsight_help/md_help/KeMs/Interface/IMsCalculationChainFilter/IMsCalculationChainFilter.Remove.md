# IMsCalculationChainFilter.Remove


## Синтаксис


		Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс элемента коллекции.


## Описание


Метод Remove удаляет из
 коллекции элемент с заданным индексом.


## Комментарии


Индексация элементов в коллекции сквозная и начинается с нуля. Для получения
 количества элементов в коллекции используйте свойство [IMsCalculationChainFilter.Count](IMsCalculationChainFilter.Count.htm).


При успешном удалении метод возвращает значение True,
 при неудачном - False.
