# IMsCalculationChainFilter.RemoveByKey


## Синтаксис


		RemoveByKey(EntryKey: Integer): Boolean;


## Параметры


EntryKey. Ключ элемента коллекции.


## Описание


Метод RemoveByKey удаляет
 из коллекции элемент с заданным ключом.


## Комментарии


При успешном удалении метод возвращает значение True,
 при неудачном - False.
