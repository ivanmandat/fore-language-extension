# IMsCalculationChainFilter.FindByKey


## Синтаксис


		FindByKey(EntryKey: Integer): [IMsCalculationChainEntry](../IMsCalculationChainEntry/IMsCalculationChainEntry.htm);


## Параметры


EntryKey. Ключ элемента коллекции.


## Описание


Метод FindByKey выполняет поиск
 в коллекции элемента с заданным ключом.


## Комментарии


Если искомый элемент не найден, то метод возвращает значение Null.
