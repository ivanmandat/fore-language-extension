# IMsDimIteratorDimensions.Remove


## Синтаксис


		Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемого измерения.


## Описание


Метод Remove удаляет измерение
 с указанным индексом.


## Комментарии


При успешном удалении измерения метод возвращает значение True,
 при неуспешно - False.


Индексация измерений в коллекции сквозная и начинается с нуля.


Для получения количества измерений в коллекции используйте свойство
 [IMsDimIteratorDimensions.Count](IMsDimIteratorDimensions.Count.htm).
