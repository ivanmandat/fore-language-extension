# IMsDimIteratorDimensions.RemoveByKey


## Синтаксис


		RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ удаляемого измерения.


## Описание


Метод RemoveByKey удаляет измерение
 с указанным ключом.


## Комментарии


При успешном удалении измерения метод возвращает значение True,
 при неуспешно - False.
