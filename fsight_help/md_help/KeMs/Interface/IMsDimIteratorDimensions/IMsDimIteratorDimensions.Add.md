# IMsDimIteratorDimensions.Add


## Синтаксис


		Add(Instance: [IDimInstance](KeDims.chm::/interface/IDimInstance/IDimInstance.htm)):
		 [IMsDimIteratorDimension](../IMsDimIteratorDimension/IMsDimIteratorDimension.htm);


## Параметры


Instance. Добавляемое измерение.


## Описание


Метод Add добавляет указанное
 измерение в коллекцию.


## Комментарии


Для удаления указанного измерения из коллекции используйте методы [IMsDimIteratorDimensions.Remove](IMsDimIteratorDimensions.Remove.htm)
 и [IMsDimIteratorDimensions.RemoveByKey](IMsDimIteratorDimensions.RemoveByKey.htm).
