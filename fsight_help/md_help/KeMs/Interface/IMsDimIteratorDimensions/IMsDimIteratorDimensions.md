# IMsDimIteratorDimensions


Сборка: Ms;


## Описание


Интерфейс IMsDimIteratorDimensions
 предназначен для работы с коллекцией измерений, по которым будет выполняться
 итерация.


## Иерархия наследования


           IMsDimIteratorDimensions


## Комментарии


Итерации выполняются многомерным итератором. Для получения коллекции
 измерений, по которым будет выполняться итерация используйте свойство
 [IMsCalculationChainMultiDimIterator.Dimensions](../IMsCalculationChainMultiDimIterator/IMsCalculationChainMultiDimIterator.Dimensions.htm).
