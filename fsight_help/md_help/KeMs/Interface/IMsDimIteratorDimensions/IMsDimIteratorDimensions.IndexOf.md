# IMsDimIteratorDimensions.IndexOf


## Синтаксис


		IndexOf(Value: [IMsDimIteratorDimension](../IMsDimIteratorDimension/IMsDimIteratorDimension.htm)):
		 Integer;


## Параметры


Value. Измерение, индекс которого
 необходимо узнать.


## Описание


Метод IndexOf возвращает
 индекс указанного измерения.


## Комментарии


Если указанное измерение отсутствует в коллекции, то метод возвращает
 значение «-1».
