# IMsDimIteratorDimensions.Item


## Синтаксис


		Item(Index: Integer): [IMsDimIteratorDimension](../IMsDimIteratorDimension/IMsDimIteratorDimension.htm);


## Параметры


Index. Индекс измерения в коллекции.


## Описание


Свойство Item возвращает
 измерение с указанным индексом.


## Комментарии


Индексация измерений в коллекции сквозная и начинается с нуля.


Для получения количества измерений в коллекции используйте свойство
 [IMsDimIteratorDimensions.Count](IMsDimIteratorDimensions.Count.htm).
