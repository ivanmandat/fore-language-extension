# IMsDimIteratorDimensions.FindByKey


## Синтаксис


		FindByKey(Key: Integer): [MsDimIteratorDimension](../IMsDimIteratorDimension/IMsDimIteratorDimension.htm);


## Параметры


Key. Ключ искомого измерения.


## Описание


Метод FindByKey выполняет поиск
 измерения с указанным ключом.


## Комментарии


Если измерение с указанным ключом не найдено, то метод возвращает значение
 Null.
