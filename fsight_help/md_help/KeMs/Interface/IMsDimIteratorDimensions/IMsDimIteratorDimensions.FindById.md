# IMsDimIteratorDimensions.FindById


## Синтаксис


		FindById(Id: String): [MsDimIteratorDimension](../IMsDimIteratorDimension/IMsDimIteratorDimension.htm);


## Параметры


Id. Идентификатор искомого
 измерения.


## Описание


Метод FindById выполняет
 поиск измерения с указанным идентификатором.


## Комментарии


Если измерение с указанным идентификатором не найдено, то метод возвращает
 значение Null.
