# IMsDataLoadingPeriod.EndDate


## Синтаксис


		EndDate: [DateTime](ForeSys.chm::/class/datetime/datetime.htm);


## Описание


Свойство EndDate определяет дату окончания периода.


## Комментарии


Значение свойства учитывается, если свойству [IMsDataLoadingPeriod.UseDefault](IMsDataLoadingPeriod.UseDefault.htm)
 установлено значение False. Значение
 по умолчанию совпадает с окончанием периода прогнозирования задачи моделирования
 (свойство [IMsProblemDetails.Period](../IMsProblemDetails/IMsProblemDetails.Period.htm)).
