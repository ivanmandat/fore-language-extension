# IMsDataLoadingPeriod


Сборка: Ms;


## Описание


Интерфейс IMsDataLoadingPeriod
 предназначен для работы с параметрами периода загрузки данных.


## Иерархия наследования


           IMsDataLoadingPeriod


## Комментарии


Интерфейс служит для оптимизации загрузки данных. Изменяя границы периода,
 можно загружать лишь необходимые данные.
