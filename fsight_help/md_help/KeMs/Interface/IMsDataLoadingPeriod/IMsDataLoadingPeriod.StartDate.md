# IMsDataLoadingPeriod.StartDate


## Синтаксис


		StartDate: [DateTime](ForeSys.chm::/class/datetime/datetime.htm);


## Описание


Свойство StartDate определяет
 дату начала периода.


## Комментарии


Значение свойства учитывается, если свойству [IMsDataLoadingPeriod.UseDefault](IMsDataLoadingPeriod.UseDefault.htm)
 установлено значение False. Значение
 по умолчанию совпадает с началом периода идентификации задачи моделирования
 (свойство [IMsProblemDetails.Period](../IMsProblemDetails/IMsProblemDetails.Period.htm)).
