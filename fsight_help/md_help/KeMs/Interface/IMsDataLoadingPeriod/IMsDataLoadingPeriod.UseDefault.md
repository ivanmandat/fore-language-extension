# IMsDataLoadingPeriod.UseDefault


## Синтаксис


		UseDefault: Boolean;


## Описание


Свойство UseDefault определяет
 признак использования традиционного механизма определения периода загрузки
 данных.


## Комментарии


Допустимые значения:


	- True. Период начала
	 и окончания загрузки данных определяется автоматически во время расчёта
	 задачи;


	- False. Период начала
	 и окончания загрузки определяется значениями свойств [IMsDataLoadingPeriod.StartDate](IMsDataLoadingPeriod.StartDate.htm)
	 и [IMsDataLoadingPeriod.EndDate](IMsDataLoadingPeriod.EndDate.htm).
