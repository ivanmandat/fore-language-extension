# IMsVariableDimensions.FindById


## Синтаксис


FindById(DimId: String): [IMsVariableDimension](../IMsVariableDimension/IMsVariableDimension.htm);


## Параметры


DimId. Идентификатор дополнительного
 измерения, по которому осуществляется поиск.


## Описание


Метод FindById осуществляет
 поиск дополнительного измерения переменной по идентификатору.
