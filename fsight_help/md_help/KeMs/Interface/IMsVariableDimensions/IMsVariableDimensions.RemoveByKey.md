# IMsVariableDimensions.RemoveByKey


## Синтаксис


RemoveByKey(DimKey: Integer): Boolean;


## Параметры


DimKey. Ключ дополнительного
 измерения, который необходимо удалить.


## Описание


Метод RemoveByKey осуществляет
 удаление дополнительного измерения переменной моделирования по ключу.
