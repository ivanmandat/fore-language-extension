# IMsVariableDimensions


## Описание


Интерфейс IMsVariableDimensions
 содержит свойства и методы дополнительных измерений переменной моделирования.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMsVariableDimensions.Count.htm)
		 Свойство Count возвращает
		 количество дополнительных измерений переменной моделирования.


		 ![](../../Property_Image.gif)
		 [Item](IMsVariableDimensions.Item.htm)
		 Свойство Item возвращает
		 дополнительное измерение переменной моделирования.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IMsVariableDimensions.Add.htm)
		 Метод Add осуществляет
		 добавление дополнительного справочника в структуру переменной
		 моделирования.


		 ![](../../Sub_Image.gif)
		 [Clear](IMsVariableDimensions.Clear.htm)
		 Метод Clear осуществляет
		 очистку списка дополнительных измерений переменной моделирования.


		 ![](../../Sub_Image.gif)
		 [FindById](IMsVariableDimensions.FindById.htm)
		 Метод FindById осуществляет
		 поиск дополнительного измерения переменной по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IMsVariableDimensions.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск дополнительного измерения переменной по ключу.


		 ![](../../Sub_Image.gif)
		 [MoveTo](IMsVariableDimensions.MoveTo.htm)
		 Метод MoveTo осуществляет
		 перемещение справочника в списке дополнительных измерений переменной
		 моделирования.


		 ![](../../Sub_Image.gif)
		 [Remove](IMsVariableDimensions.Remove.htm)
		 Метод Remove осуществляет
		 удаление дополнительного измерения переменной моделирования по
		 индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IMsVariableDimensions.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление дополнительного измерения переменной моделирования по
		 ключу.
