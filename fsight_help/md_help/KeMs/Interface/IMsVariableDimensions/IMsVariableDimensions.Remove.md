# IMsVariableDimensions.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс дополнительного
 измерения, который необходимо удалить.


## Описание


Метод Remove осуществляет удаление
 дополнительного измерения переменной моделирования по индексу.
