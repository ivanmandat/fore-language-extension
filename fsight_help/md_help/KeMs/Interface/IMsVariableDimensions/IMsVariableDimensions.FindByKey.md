# IMsVariableDimensions.FindByKey


## Синтаксис


FindByKey(DimKey: Integer): [IMsVariableDimension](../IMsVariableDimension/IMsVariableDimension.htm);


## Параметры


DimKey. Ключ дополнительного
 измерения, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск дополнительного измерения переменной по ключу.
