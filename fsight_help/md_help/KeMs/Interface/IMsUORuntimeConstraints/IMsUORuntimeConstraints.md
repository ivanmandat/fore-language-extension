# IMsUORuntimeConstraints


Сборка: Ms;


## Описание


Интерфейс IMsUORuntimeConstraints
 предназначен для работы с коллекцией ограничений пользовательской задачи
 оптимизации.


## Иерархия наследования


IMsUORuntimeConstraints


## Комментарии


Ограничения можно получить, используя свойство [IMsUORuntime.Constraints](../IMsUORuntime/IMsUORuntime.Constraints.htm)
 во время расчёта задачи.
