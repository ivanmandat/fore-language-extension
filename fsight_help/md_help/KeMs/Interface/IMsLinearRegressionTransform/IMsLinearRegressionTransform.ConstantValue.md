# IMsLinearRegressionTransform.ConstantValue


## Синтаксис


ConstantValue: Double;


## Описание


Свойство ConstantValue определяет значение константы для модели.


## Комментарии


ConstantValue учитывается, если [IMsLinearRegressionTransform.ConstantMode](IMsLinearRegressionTransform.ConstantMode.htm) имеет значение InterceptMode.ManualEstimate.
