# IMsLinearRegressionTransform.CoefUncertaintyInSECalc


## Синтаксис


CoefUncertaintyInSECalc: Boolean;


## Описание


Свойство CoefUncertaintyInSECalc
 определяет, учитывать ли неопределенность коэффициентов при расчете доверительных
 границ.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 При расчете доверительных границ учитывается, что коэффициенты найдены
	 приближенно;


	- False. При расчете доверительных
	 границ не учитывается, что коэффициенты найдены приближенно.
