# IMsDimCustomLinker


Сборка: Ms;


## Описание


Интерфейс IMsDimCustomLinker
 предназначен для работы с пользовательским механизмом связывания измерений.


## Иерархия наследования


           IMsDimCustomLinker


## Комментарии


Механизм связывания измерений должен быть реализован в виде класса,
 реализующего методы интерфейса IMsDimCustomLinker.


Для установки пользовательского механизма связывания измерений используйте
 свойство [IMsDimIteratorLinks.CustomLinker](../IMsDimIteratorLinks/IMsDimIteratorLinks.CustomLinker.htm).
