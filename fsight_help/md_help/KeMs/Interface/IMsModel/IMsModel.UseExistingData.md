# IMsModel.UseExistingData


## Синтаксис


UseExistingData: Boolean;


## Описание


Свойство UseExistingData определяет,
 будут ли использоваться все фактические данные входных переменных при
 расчете модели. По умолчанию свойству установлено значение True.
 Если свойству установить значение False,
 при расчете будут использоваться данные переменных на указанный период
 идентификации.
