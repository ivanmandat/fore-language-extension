# IMsModel.Stochastic


## Синтаксис


Stochastic: Boolean;


## Описание


Свойство Stochastic возвращает True, если для расчета модели используются стохастические методы. К стохастическим методам относятся линейная и не линейная регрессия, модель коррекции ошибок.
