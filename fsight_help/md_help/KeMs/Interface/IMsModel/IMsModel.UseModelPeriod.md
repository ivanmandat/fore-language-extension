# IMsModel.UseModelPeriod


## Синтаксис


UseModelPeriod: Boolean;


## Описание


Свойство UseModelPeriod определяет,
 будут ли при расчете задачи использоваться указанные для модели периоды
 идентификации и прогнозирования.


## Комментарии


По умолчанию свойству установлено значение False,
 при этом используются периоды, указанные для всей задачи.
