# IMsModel.UseAutoPeriod


## Синтаксис


UseAutoPeriod: Boolean;


## Описание


Свойство UseAutoPeriod возвращает
 признак того, что для расчета дат начала/окончания периодов идентификации
 и прогнозирования используются условия.


## Комментарии


Допустимые значения:


	- True. Для вычисления
	 границ периода расчета модели используются условия;


	- False. Границы периода
	 расчета модели заданы пользователем.
