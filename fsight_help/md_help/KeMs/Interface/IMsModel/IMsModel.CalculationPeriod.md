# IMsModel.CalculationPeriod


## Синтаксис


CalculationPeriod: [MsCalculationPeriod](../../Enums/MsCalculationPeriod.htm);


## Описание


Свойство CalculationPeriod определяет
 период действия формулы.


## Комментарии


Свойство применяется только для следующих видов моделей:


	- [детерминированное
	 уравнение](../IMsDeterministicTransform/IMsDeterministicTransform.htm);


	- [R](../IMsRTransform/IMsRTransform.htm);


	- [методы
	 накопления](../IMsCumulativeTransform/IMsCumulativeTransform.htm);


	- [коллапс
	 (поточечный расчёт)](../IMsCollapseTransform/IMsCollapseTransform.htm);


	- [интерполяция](../IMsInterpolateTransform/IMsInterpolateTransform.htm).
