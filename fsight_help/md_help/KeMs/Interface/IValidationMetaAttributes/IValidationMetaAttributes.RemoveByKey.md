# IValidationMetaAttributes.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ атрибута в коллекции.


## Описание


Метод RemoveByKey осуществляет удаление атрибута из коллекции по ключу. Метод возвращает значение True, если удаление прошло успешно.
