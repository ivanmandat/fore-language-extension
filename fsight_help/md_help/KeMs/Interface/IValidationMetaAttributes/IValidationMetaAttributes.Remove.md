# IValidationMetaAttributes.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс атрибута в коллекции.


## Описание


Метод Remove осуществляет удаление атрибута из коллекции по индексу. Метод возвращает значение True, если удаление прошло успешно.
