# IValidationMetaAttributes


## Описание


Интерфейс IValidationMetaAttributes
 содержит свойства и методы для работы с коллекцией атрибутов для выполнения
 валидации.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IValidationMetaAttributes.Count.htm)
		 Свойство Count возвращает
		 количество атрибутов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IValidationMetaAttributes.Item.htm)
		 Свойство Item возвращает
		 атрибут из коллекции по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IValidationMetaAttributes.Add.htm)
		 Метод Add осуществляет
		 добавление атрибута в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](IValidationMetaAttributes.Clear.htm)
		 Метод Clear очищает
		 коллекцию атрибутов.


		 ![](../../Sub_Image.gif)
		 [Remove](IValidationMetaAttributes.Remove.htm)
		 Метод Remove осуществляет
		 удаление атрибута из коллекции по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IValidationMetaAttributes.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление атрибута из коллекции по ключу.
