# IMsResultObject.Residuals


## Синтаксис


Residuals: [ITimeSeries](../ITimeSeries/ITimeSeries.htm);


## Описание


Свойство Residuals возвращает
 ряд остатков.


## Комментарии


Для получения результирующего ряда используйте свойство [IMsResultObject.TimeSeries](IMsResultObject.TimeSeries.htm),
 а для получения модельного ряда - [IMsResultObject.Fitted](IMsResultObject.Fitted.htm).

Использование свойства приведено в примере для [IMsForecastObject.Forecast](../IMsForecastObject/IMsForecastObject.Forecast.htm).
