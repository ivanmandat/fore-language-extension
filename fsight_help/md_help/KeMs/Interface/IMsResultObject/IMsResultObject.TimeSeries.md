# IMsResultObject.TimeSeries


## Синтаксис


TimeSeries: [ITimeSeries](../ITimeSeries/ITimeSeries.htm);


## Описание


Свойство TimeSeries возвращает
 результирующий ряд.


## Комментарии


Для получения модельного ряда используйте свойство [IMsResultObject.Fitted](IMsResultObject.Fitted.htm),
 а для получения ряда остатков - [IMsResultObject.Residuals](IMsResultObject.Residuals.htm).


## Пример


Использование свойства приведено в примере для [IMsForecastObject.Forecast](../IMsForecastObject/IMsForecastObject.Forecast.htm).
