# IMsResultObject


Сборка: Ms;


## Описание


Интерфейс IMsResultObject
 предназначен для работы с результатами расчета метода.


## Иерархия наследования


           IMsResultObject


## Комментарии


Данный интерфейс является базовым для результатов, полученных при расчёте
 метода в детерминированном уравнении.
