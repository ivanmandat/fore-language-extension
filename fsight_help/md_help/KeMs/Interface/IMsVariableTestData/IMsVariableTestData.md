# IMsVariableTestData


Сборка: Ms;


## Описание


Интерфейс IMsVariableTestData
 предназначен для работы с переменной, для которой рассчитывается описательная
 статистика.


## Иерархия наследования


           IMsVariableTestData


## Комментарии


Для работы с коллекцией переменных, для которых рассчитывается описательная
 статистика, используйте свойство [IMsVariableTestSettings.IncludedSources](../IMsVariableTestSettings/IMsVariableTestSettings.IncludedSources.htm).
