# IMsVariableTestData.Lag


## Синтаксис


Lag: String;


## Описание


Свойство Lag определяет
 лаг переменной.


## Комментарии


Лаг - это сдвиг точек переменной во временном периоде. Направление сдвига
 зависит от значения лага. Если лаг отрицательный, то точки переменной
 сдвигаются назад, если положительный - то вперед.


Для задания свойства Lag укажите
 величину сдвига в строковом виде. Например, сдвиг переменной на четыре
 точки вперед:


Lag := "4";
