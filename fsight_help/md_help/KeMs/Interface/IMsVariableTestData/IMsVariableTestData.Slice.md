Свойство Slice определяет срез переменной.


## Комментарии


По умолчанию срез переменной - это срез, указанный при добавлении переменной с помощью метода [IMsVariableTestDataList.Add](../IMsVariableTestDataList/IMsVariableTestDataList.Add.htm).
