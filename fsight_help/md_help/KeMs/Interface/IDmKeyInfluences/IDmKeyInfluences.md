# IDmKeyInfluences


Сборка: Ms;


## Описание


Интерфейс IDmKeyInfluences предназначен
 для интеллектуального анализа данных методом «Ключевые
 факторы».


## Иерархия наследования


           [IDmMethodDetails](../IDmMethodDetails/IDmMethodDetails.htm)


           [IDmAttributesDetails](../IDmAttributesDetails/IDmAttributesDetails.htm)


           [IDmTargetDetails](../IDmTargetDetails/IDmTargetDetails.htm)


           IDmKeyInfluences


## Комментарии


Данный вид анализа оценивает степень влияния каждого фактора на зависимую
 переменную и выявляет наиболее значимые факторы. Метод работает на категориальных
 значениях.


Использование метода «Ключевые факторы»
 приведено в примере для [IDmMethod.Execute](../IDmMethod/IDmMethod.Execute.htm).
