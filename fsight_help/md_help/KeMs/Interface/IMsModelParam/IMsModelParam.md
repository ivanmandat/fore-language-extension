# IMsModelParam


Сборка: Ms;


## Описание


Интерфейс IMsModelParam предназначен
 для работы с параметром метамодели.


## Комментарии


Для работы с коллекцией параметров используйте интерфейс [IMsModelParams](../IMsModelParams/IMsModelParams.htm).

		 Имя свойства

		 ![](../../Property_Image.gif)
		 [ObjectParam](IMsModelParam.ObjectParam.htm)
		 Свойство ObjectParam
		 определяет параметр, с которым будет связан текущий параметр метамодели.


		 ![](../../Property_Image.gif)
		 [SelectionMode](IMsModelParam.SelectionMode.htm)
		 Свойство SelectionMode
