# IMsModelParam.SelectionMode


## Синтаксис


SelectionMode: [SelectionModeEnum](ForeSys.chm::/Enums/SelectionModeEnum.htm);


## Описание


Свойство SelectionMode определяет
 способ отметки элементов справочника.


## Комментарии


Свойство актуально, если свойство [ITsModelParam.ParamType](KeTransform.chm::/Interface/ITsModelParam/ITsModelParam.ParamType.htm)
 принимает значение TsParamType.Selection.
