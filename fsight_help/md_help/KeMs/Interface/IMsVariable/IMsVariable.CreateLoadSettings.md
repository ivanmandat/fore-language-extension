# IMsVariable.CreateLoadSettings


## Синтаксис


CreateLoadSettings: [IMsVariableLoadSettings](../IMsVariableLoadSettings/IMsVariableLoadSettings.htm);


## Описание


Метод CreateLoadSettings создает
 объект, определяющий параметры загрузки данных в переменную моделирования.
 Используется при загрузке данных с помощью формулы.
