# IMsVariable


## Описание


Интерфейс IMsVariable содержит
 свойства объекта контейнера моделирования - [Переменная](UiModelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_1_Value/uimodelling_work_object_value.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Cube](IMsVariable.Cube.htm)
		 Свойство Cube возвращает
		 автокуб, содержащий все данные переменной моделирования.


		 ![](../../Property_Image.gif)
		 [DataFillType](IMsVariable.DataFillType.htm)
		 Свойство DataFillType
		 определяет способ заполнения данными переменной моделирования.


		 ![](../../Property_Image.gif)
		 [Dimensions](IMsVariable.Dimensions.htm)
		 Свойство Dimensions
		 возвращает коллекцию дополнительных измерений переменной моделирования.


		 ![](../../Property_Image.gif)
		 [Modified](IMsVariable.Modified.htm)
		 Свойство Modified не
		 реализовано.


		 ![](../../Property_Image.gif)
		 [Sampling](IMsVariable.Sampling.htm)
		 Свойство Sampling возвращает
		 настройки шага расчета в структуре переменной.


		 ![](../../Property_Image.gif)
		 [ScenarioDimension](IMsVariable.ScenarioDimension.htm)
		 Свойство ScenarioDimension
		 возвращает данные сценарного измерения переменной моделирования.


		 ![](../../Property_Image.gif)
		 [Transform](IMsVariable.Transform.htm)
		 Свойство Transform
		 возвращает объект, содержащий настройки необходимые для использования
		 переменной в различных методах преобразования.


		 ![](../../Property_Image.gif)
		 [Type](IMsVariable.Type.htm)
		 Свойство Type определяет
		 дополнительные параметры переменной моделирования.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CopyData](IMsVariable.CopyData.htm)
		 Метод CopyData осуществляет
		 копирование данных между измерениями переменной.


		 ![](../../Sub_Image.gif)
		 [CreateLoadSettings](IMsVariable.CreateLoadSettings.htm)
		 Метод CreateLoadSettings
		 создает объект, определяющий параметры загрузки данных в переменную
		 моделирования.


		 ![](../../Sub_Image.gif)
		 [Execute](IMsVariable.Execute.htm)
		 Метод Execute осуществляет
		 загрузку данных в переменную.
