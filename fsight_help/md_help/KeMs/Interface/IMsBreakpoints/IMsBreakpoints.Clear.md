# IMsBreakpoints.Clear


## Синтаксис


		Clear;


## Описание


Метод Clear
 удаляет все точки останова из коллекции.


## Комментарии


После вызова метода значение свойства [IMsBreakpoints.Count](IMsBreakpoints.Count.htm)
 становится равным нулю.
