# IMsBreakpoints


Сборка: Ms;


## Описание


Интерфейс IMsBreakpoints содержит
 свойства и методы для работы с коллекцией точек останова.


## Иерархия наследования


           IMsBreakpoints


## Комментарии


Объекты данной коллекции реализованы базовым интерфейсом [IMsBreakpoint](../IMsBreakpoint/IMsBreakpoint.htm) и используются для работы
 с точками останова при расчёте задачи моделирования. Коллекция точек останова
 для задачи моделирования не сохраняется при сохранении самой задачи.
