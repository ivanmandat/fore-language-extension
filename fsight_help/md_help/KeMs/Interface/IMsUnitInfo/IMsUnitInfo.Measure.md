# IMsUnitInfo.Measure


## Синтаксис


Measure: Integer;


## Описание


Свойство Measure возвращает индекс меры.


## Комментарии


Индекс Measure соответствует индексу элемента в справочнике мер и единиц измерения. Справочник возвращает свойство [IMsUnitInfo.UnitsDimension](IMsUnitInfo.UnitsDimension.htm).


Если Measure возвращает «-1», то мера не определена.
