# IMsUnitInfo.UnitsDimension


## Синтаксис


UnitsDimension: [IDimInstance](KeDims.chm::/interface/IDimInstance/IDimInstance.htm);


## Описание


Свойство UnitsDimension возвращает данные справочника мер и единиц измерения.


## Комментарии


Для работы с данными справочника как с внутренним измерением базы данных временных рядов приведите данный интерфейс к интерфейсу [IUnitsDimInstance](KeDims.chm::/interface/IUnitsDimInstance/IUnitsDimInstance.htm).
