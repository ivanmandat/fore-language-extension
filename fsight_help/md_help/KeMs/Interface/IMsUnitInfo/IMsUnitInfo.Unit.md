# IMsUnitInfo.Unit


## Синтаксис


Unit: Integer;


## Описание


Свойство Unit возвращает индекс единицы измерения.


## Комментарии


Индекс Unit соответствует индексу элемента в справочнике мер и единиц измерения. Справочник возвращает свойство [IMsUnitInfo.UnitsDimension](IMsUnitInfo.UnitsDimension.htm).


Если Unit возвращает «-1», то единица измерения не определена.
