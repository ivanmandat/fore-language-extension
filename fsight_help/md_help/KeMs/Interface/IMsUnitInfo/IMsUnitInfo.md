# IMsUnitInfo


Сборка: Ms;


## Описание


Интерфейс IMsUnitInfo предназначен
 для работы с единицами измерения.


## Иерархия наследования


           [INamedEntity](KeSom.chm::/Interface/INamedEntity/INamedEntity.htm)


           IMsUnitInfo


## Комментарии


Информация о единицах измерения может быть получена от термов и наборов
 термов:


	- [IMsCompositeFormulaTerm.UnitInfo](../IMsCompositeFormulaTerm/IMsCompositeFormulaTerm.UnitInfo.htm);


	- [IMsCompositeFormulaTermList.UnitInfo](../IMsCompositeFormulaTermList/IMsCompositeFormulaTermList.UnitInfo.htm);


	- [IMsFormulaTerm.UnitInfo](../IMsFormulaTerm/IMsFormulaTerm.UnitInfo.htm);


	- [IMsFormulaTermList.UnitInfo](../IMsFormulaTermList/IMsFormulaTermList.UnitInfo.htm).
