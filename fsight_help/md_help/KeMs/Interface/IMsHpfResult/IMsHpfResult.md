# IMsHpfResult


Сборка: Ms;


## Описание


Интерфейс IMsHpfResult предназначен
 для работы с результатами расчета фильтра Ходрика-Прескотта.


## Иерархия наследования


           [IMsResultObject](../IMsResultObject/IMsResultObject.htm)


           IMsHpfResult


## Комментарии


Данный интерфейс используется в пользовательских функциях и методах,
 предназначенных для расчета в детерминированном уравнении.
