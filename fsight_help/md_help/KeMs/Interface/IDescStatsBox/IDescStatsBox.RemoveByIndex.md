# IDescStatsBox.RemoveByIndex


## Синтаксис


		RemoveByIndex(Index: Integer);


## Параметры


Index. Индекс переменной в
 списке переменных компонента.


## Описание


Метод RemoveByIndex осуществляет удаление указанной переменной.


## Комментарии


Метод выполняет удаление переменной с индексом Index.
