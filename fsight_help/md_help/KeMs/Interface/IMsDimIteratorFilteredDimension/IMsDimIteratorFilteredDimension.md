# IMsDimIteratorFilteredDimension


Сборка: Ms;


## Описание


Интерфейс IMsDimIteratorFilteredDimension
 предназначен для работы с параметрами фильтрации измерения, по которому
 выполняется итерация.


## Иерархия наследования


           [INamedEntity](KeSom.chm::/Interface/INamedEntity/INamedEntity.htm)


           IMsDimIteratorFilteredDimension


## Комментарии


Для получения параметров фильтрации измерения используйте свойство [IMsDimIterateByDataStub.Dimension](../IMsDimIterateByDataStub/IMsDimIterateByDataStub.Dimension.htm).
