# IMsDimIteratorFilteredDimension.Filter


## Синтаксис


		Filter: [IMsDimIteratorFilter](../IMsDimIteratorFilter/IMsDimIteratorFilter.htm);


## Описание


Свойство Filter возвращает
 фильтр измерения.


## Комментарии


Для связанных измерений фильтр игнорируется.


Для получения фильтруемого измерения используйте свойство [IMsDimIteratorFilteredDimension.Dimension](IMsDimIteratorFilteredDimension.Dimension.htm).

Использование свойства приведено в примере для [IMsCalculationChainMultiDimIterator.IterateByData](../IMsCalculationChainMultiDimIterator/IMsCalculationChainMultiDimIterator.IterateByData.htm).
