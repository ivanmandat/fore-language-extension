# IMsValueAtRiskBacktestingResult


Сборка: Ms;


## Описание


Интерфейс IMsValueAtRiskBacktestingResult
 содержит свойства, возвращающие результаты бэктестинга.


## Иерархия наследования


           IMsValueAtRiskBacktestingResult


## Комментарии


Результаты могут быть получены только после расчета модели Value-At-Risk.
 Их возвращает свойство [IMsValueAtRiskBacktesting.Result](../IMsValueAtRiskBacktesting/IMsValueAtRiskBacktesting.Result.htm).
