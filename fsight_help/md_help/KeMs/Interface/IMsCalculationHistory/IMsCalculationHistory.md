# IMsCalculationHistory


Сборка: Ms;


## Описание


Интерфейс IMsCalculationHistory
 содержит свойства истории расчёта задачи моделирования.


## Иерархия наследования


           IMsCalculationHistory


## Комментарии


История расчёта является составным объектом, содержащим параметры всех
 переменных, моделей, метамодели и задачи, установленные на момент расчёта.
 Для просмотра сохранённых параметров, необходимо осуществлять соответствующие
 преобразования для [дочерних](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.Children.htm)
 объектов объекта «История расчёта».
