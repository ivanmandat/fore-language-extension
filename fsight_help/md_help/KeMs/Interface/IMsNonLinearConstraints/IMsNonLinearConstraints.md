# IMsNonLinearConstraints


## Описание


Интерфейс IMsNonLinearConstraints
 содержит свойства и методы для работы с коллекцией ограничений целевой
 функции.


## Комментарии


Ограничения целевой функции можно получить, используя следующие свойства:


	- [IMsControlProblem.Constraints](../IMsControlProblem/IMsControlProblem.Constraints.htm);


	- [IMsUserOptimizationProblem.Constraints](../IMsUserOptimizationProblem/IMsUserOptimizationProblem.Constraints.htm).
