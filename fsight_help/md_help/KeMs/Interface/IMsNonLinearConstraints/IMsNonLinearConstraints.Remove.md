# IMsNonLinearConstraints.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс ограничения.


## Описание


Метод Remove осуществляет удаление ограничения, индекс которого передается посредством параметра Index.
