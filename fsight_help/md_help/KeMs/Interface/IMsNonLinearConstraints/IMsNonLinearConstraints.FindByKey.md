# IMsNonLinearConstraints.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IMsNonLinearConstraint](../IMsNonLinearConstraint/IMsNonLinearConstraint.htm);


## Параметры


Key. Ключ ограничения.


## Описание


Метод FindByKey осуществляет
 поиск ограничения по ключ и возвращает его настройки в случает удачного
 поиска.


## Комментарии


Метод возвращает значение Null,
 если ограничение с указанным ключом не найдено в коллекции.
