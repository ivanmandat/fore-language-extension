# IValidationStatisticalOutliers.InvertException


## Синтаксис


InvertException: Boolean;


## Описание


Свойство InvertException определяет, инвертировать ли ошибки валидации.


## Комментарии


Допустимые значения:


-
True. Ошибки инвертируются, т.е. выявляются не статистические выбросы, а данные, не выходящие за заданные границы;


-
False. Используется по умолчанию. Ошибками валидации являются статистические выбросы.
