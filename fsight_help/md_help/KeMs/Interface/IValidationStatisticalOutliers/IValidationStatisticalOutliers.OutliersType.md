# IValidationStatisticalOutliers.OutliersType


## Синтаксис


OutliersType: [ValidationStatisticalOutliersType](../../Enums/ValidationStatisticalOutliersType.htm);


## Описание


Свойство OutliersType определяет область данных для выявления статистических выбросов.


## Комментарии


По умолчанию OutliersType = ValidationStatisticalOutliersType.CrossSerie, т.е. статистические выбросы выявляются отдельно для каждого ряда.
