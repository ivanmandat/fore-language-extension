# IValidationStatisticalOutliers.UpperPercentile


## Синтаксис


UpperPercentile: Integer;


## Описание


Свойство UpperPercentile определяет
 вероятность верхней границы выброса.


## Комментарии


Диапазон допустимых значений свойства: [0, 100].


Для задания вероятности нижней границы выброса используйте свойство
 [IValidationStatisticalOutliers.LowerPercentile](IValidationStatisticalOutliers.LowerPercentile.htm).
