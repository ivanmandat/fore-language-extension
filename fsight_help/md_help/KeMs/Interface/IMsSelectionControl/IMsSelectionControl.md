# IMsSelectionControl


Сборка: Ms;


## Описание


Интерфейс IMsSelectionControl
 предназначен для управления параметрами измерений источника данных.


## Иерархия наследования


           IMsSelectionControl


## Комментарии


Настройка управления параметрами измерений доступна для всех [типов
 кубов](UiNavObj.chm::/Cube/CreateCube/UiMd_Cube_CreateCube.htm). Если в кубе имеются параметрические измерения,
 то можно настроить управление параметрами данных измерений с помощью других
 измерений куба. Для получения подробной информации об управлении параметрами
 и настройке управляющих измерений в кубе обратитесь к разделам «[Управление
 параметрами вложенных объектов](UiNavObj.chm::/Cube/CreateCube/Master_Standart/UiMd_Cube_CreateCube_Master_Standart_manage_param.htm)», «[Управляющие
 измерения](UiNavObj.chm::/Cube/CreateCube/Master_Standart/UiMd_Cube_CreateCube_Master_Standart_managed_dim.htm)».
