# IMsSelectionControl.IsActive


## Синтаксис


IsActive: Boolean;


## Описание


Свойство IsActive определяет
 признак управления параметрами измерений источника данных.


## Комментарии


Допустимые значения:


	- True. Управление параметрами
	 измерений источника данных используется;


	- False. Управление параметрами
	 измерений источника данных не используется.


Примечание.
 Для работы с управляющими и управляемыми измерениями перед использованием
 методов [IMsSelectionControl.GetStubSel](IMsSelectionControl.GetStubSel.htm)
 и [IMsSelectionControl.Apply](IMsSelectionControl.Apply.htm)
 свойство должно принимать значение True.
