# IMsModelCalculation


## Описание


Интерфейс IMsModelCalculation
 содержит свойства, определяющие параметры расчета модели.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ClearSeriesOnWarning](IMsModelCalculation.ClearSeriesOnWarning.htm)
		 Свойство ClearSeriesOnWarning
		 определяет, будет ли результирующий ряд пустым при возникновении
		 предупреждения.


		 ![](../../Property_Image.gif)
		 [CurrentPoint](IMsModelCalculation.CurrentPoint.htm)
		 Свойство CurrentPoint
		 определяет точку, в которой производится расчет модели.


		 ![](../../Property_Image.gif)
		 [ParamValues](IMsModelCalculation.ParamValues.htm)
		 Свойство ParamValues
		 возвращает коллекцию значений параметров расчета модели.


		 ![](../../Property_Image.gif)
		 [Period](IMsModelCalculation.Period.htm)
		 Свойство Period возвращает
		 объект, содержащий настройки периодов идентификации и прогнозирования,
		 на которых осуществляется расчет модели.


		 ![](../../Property_Image.gif)
		 [Scenario](IMsModelCalculation.Scenario.htm)
		 Свойство Scenario определяет
		 сценарий, по которому будет рассчитана модель.


		 ![](../../Property_Image.gif)
		 [Variable](IMsModelCalculation.Variable.htm)
		 Свойство Variable возвращает
		 матрицу с данными переменной.
