# IMsModelCalculation.CurrentPoint


## Синтаксис


CurrentPoint: DateTime;


## Описание


Свойство CurrentPoint определяет точку, в которой производится расчет модели. Данное свойство используется при расчете модели с помощью пользовательского метода расчета.
