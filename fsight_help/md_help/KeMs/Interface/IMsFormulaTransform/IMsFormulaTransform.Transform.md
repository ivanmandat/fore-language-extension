# IMsFormulaTransform.Transform


## Синтаксис


Transform(Selector: [IMsFormulaTransformSelector](../IMsFormulaTransformSelector/IMsFormulaTransformSelector.htm)):
 [IMsFormula](../IMsFormula/IMsFormula.htm);


## Параметры


Selector. Объект, определяющий
 срез переменной, по которому необходимо настроить параметры метода расчёта.


## Описание


Свойство Transform возвращает
 объект, позволяющий настроить параметры метода расчёта.


## Комментарии


Если в моделируемой переменной имеются дополнительные измерения, то
 существует возможность для каждого среза определить свой метод расчёта.
