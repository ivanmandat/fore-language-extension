# IMsFormulaTransform.CalculationDirection


## Синтаксис


		CalculationDirection: [MsCalculationDirection](../../Enums/MsCalculationDirection.htm);


## Описание


Свойство CalculationDirection
 определяет направление поточечного режима расчёта.


## Комментарии


Направление расчёта актуально только в задачах трансформации. В задачах
 прогнозирования всегда применяется расчёт вперед.


Режим расчёта определяется свойством [IMsFormulaTransform.CalculationType](IMsFormulaTransform.CalculationType.htm).
