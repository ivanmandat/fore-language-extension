# IMsFormulaTransform.DisplayId


## Синтаксис


		DisplayId: Boolean;


## Описание


Свойство DisplayId определяет
 вид отображаемой информации в формуле терма.


## Комментарии


Если свойству установлено значение True,
 то в формуле отображается идентификатор переменной, иначе наименование.
