# IValidationSelectionValues


Сборка: Ms;


## Описание


Интерфейс IValidationSelectionValues
 предназначен для работы с элементом фильтра, ограничивающего ряды, проверяемые
 правилом валидации.


## Иерархия наследования


           [INamedEntityModel](ForeSys.chm::/Interface/INamedEntityModel/INamedEntityModel.htm)


           IValidationSelectionValues


## Комментарии


Фильтр представляет собой набор отметок по измерениям источника данных
 рядов. Каждый элемент фильтра задаёт отметку одного измерения. Для работы
 со всем фильтром используйте интерфейс [IValidationSelectionFilter](../IValidationSelectionFilter/IValidationSelectionFilter.htm).
