# IMsVariableSampling


## Описание


Интерфейс IMsVariableSampling
 содержит свойства для настройки шага расчета в структуре переменной.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Level](IMsVariableSampling.Level.htm)
		 Свойство Level определяет,
		 будет ли уровень календаря присутствовать в динамике переменной.


		 ![](../../Property_Image.gif)
		 [SourceDimension](IMsVariableSampling.SourceDimension.htm)
		 Свойство SourceDimension
		 возвращает структуру календарного измерения переменной моделирования.
