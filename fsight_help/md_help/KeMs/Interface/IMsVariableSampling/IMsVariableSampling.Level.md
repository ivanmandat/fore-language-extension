# IMsVariableSampling.Level


## Синтаксис


Level(Level: [DimCalendarLevel](KeDims.chm::/Enums/DimCalendarLevel.htm)):
 Boolean;


## Параметры


Level. Уровень календаря, который
 может использоваться в динамике переменной.


## Описание


Свойство Level определяет, будет
 ли уровень календаря присутствовать в динамике переменной.
