# IMsModelCalculationResult


## Описание


Интерфейс IMsModelCalculationResult
 содержит свойства для просмотра результатов расчета модели.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [IdentificationEndDate](IMsModelCalculationResult.IdentificationEndDate.htm)
		 Свойство IdentificationEndDate
		 возвращает дату окончания периода идентификации.


		 ![](../../Property_Image.gif)
		 [IdentificationStartDate](IMsModelCalculationResult.IdentificationStartDate.htm)
		 Свойство IdentificationStartDate
		 возвращает дату начала периода идентификации.


		 ![](../../Property_Image.gif)
		 [Method](IMsModelCalculationResult.Method.htm)
		 Свойство Method возвращает
		 параметры метода, использованного для расчета модели.


		 ![](../../Property_Image.gif)
		 [StatMethod](IMsModelCalculationResult.StatMethod.htm)
		 Свойство StatMethod
		 возвращает параметры статистического метода, использованного для
		 расчета модели.
