# IMsBinaryRegressionTransform.Tolerance


## Синтаксис


		Tolerance: Double;


## Описание


Свойство Tolerance определяет
 точность оптимизации функции правдоподобия.


## Комментарии


Значение Tolerance должно быть
 больше нуля. Значение по умолчанию - «0,001».
