# IMsBinaryRegressionTransform.ConstantValue


## Синтаксис


		ConstantValue: Double;


## Описание


Свойство ConstantValue определяет
 значение константы для модели.


## Комментарии


ConstantValue учитывается, если
 [IMsBinaryRegressionTransform.ConstantMode](IMsBinaryRegressionTransform.ConstantMode.htm)
 имеет значение InterceptMode.ManualEstimate.
