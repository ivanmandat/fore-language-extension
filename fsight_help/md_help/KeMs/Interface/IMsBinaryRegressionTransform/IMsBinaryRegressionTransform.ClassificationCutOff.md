# IMsBinaryRegressionTransform.ClassificationCutOff


## Синтаксис


		ClassificationCutOff: Double;


## Описание


Свойство ClassificationCutOff
 определяет пороговое значение вероятности для отнесения объектов к группе
 «0» или «1».


## Комментарии


Значение ClassificationCutOff
 должно быть больше нуля. Значение по умолчанию - «0,5».
