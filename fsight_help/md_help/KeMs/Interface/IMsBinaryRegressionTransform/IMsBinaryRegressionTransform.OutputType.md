# IMsBinaryRegressionTransform.OutputType


## Синтаксис


		OutputType: [MsBinaryRegressionOutputType](../../Enums/MsBinaryRegressionOutputType.htm);


## Описание


Свойство OutputType определяет
 тип значений, выгружаемых в моделируемый и прогнозные ряды.


## Комментарии


По умолчанию свойство принимает значение MsBinaryRegressionOutputType.Probability,
 т.е. выгружаются вероятностные значения.
