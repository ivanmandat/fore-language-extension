# IMsBinaryRegressionTransform.ConstantMode


## Синтаксис


		ConstantMode: [InterceptMode](StatLib.chm::/Enums/InterceptMode.htm);


## Описание


Свойство ConstantMode определяет
 режим задания константы для модели.


## Комментарии


Если ConstantMode имеет значение
 InterceptMode.ManualEstimate,
 то константа задается с использованием свойства [IMsBinaryRegressionTransform.ConstantValue](IMsBinaryRegressionTransform.ConstantValue.htm).

Использование свойства приведено в примере для [IMsBinaryRegressionTransform.BinaryDistr](IMsBinaryRegressionTransform.BinaryDistr.htm).
