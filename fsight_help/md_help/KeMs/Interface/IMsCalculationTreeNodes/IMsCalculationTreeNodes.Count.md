# IMsCalculationTreeNodes.Count


## Синтаксис


		Count: Integer;


## Описание


Свойство Count возвращает количество
 элементов в коллекции.


## Комментарии


Ёмкость коллекции - это количество элементов, которое может содержать
 коллекция.


Свойство возвращает количество элементов, которое в действительности
 содержится в коллекции. Ёмкость всегда больше или равна значению свойства.
 Если при добавлении элементов в коллекцию значение свойства превышает
 ёмкость, то она автоматически увеличивается.
