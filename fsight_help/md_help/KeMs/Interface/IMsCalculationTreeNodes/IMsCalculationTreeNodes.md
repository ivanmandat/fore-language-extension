# IMsCalculationTreeNodes


Сборка: Ms;


## Описание


Интерфейс IMsCalculationTreeNodes
 содержит свойства для работы с коллекцией элементов дерева расчёта.


## Иерархия наследования


           IMsCalculationTreeNode


## Комментарии


Элементы данной коллекции реализованы интерфейсом [IMsCalculationTreeNode](../IMsCalculationTreeNode/IMsCalculationTreeNode.htm)
 и используются для работы элементами дерева расчёта.
