# IMsDimAttributesList.Add


## Синтаксис


		Add(Attribute: [IDimAttribute](KeDims.chm::/interface/IDimAttribute/IDimAttribute.htm));


## Параметры


Attribute. Атрибут, который
 необходимо добавить в цепочку.


## Описание


Метод Add добавляет в цепочку
 указанный атрибут.


## Комментарии


Значения добавленного атрибута будут использоваться для формирования
 значений параметра.


## Пример


Пример использования приведён в описании свойств [IMsModelPeriod.IdentificationStartDateCoord](../IMsModelPeriod/IMsModelPeriod.IdentificationStartDateCoord.htm),
 [IMsModelPeriod.IdentificationStartDateParam](../IMsModelPeriod/IMsModelPeriod.IdentificationStartDateParam.htm).
