# IMsDimAttributesList


Сборка: Ms;


## Описание


Интерфейс IMsDimAttributesList
 предназначен для настройки цепочки атрибутов, по которым будет формироваться
 значения для параметров.


## Иерархия наследования


           IMsDimAttributesList


## Комментарии


Цепочку атрибутов возвращает свойство [IMsModelAttributesChain.Attributes](../IMsModelAttributesChain/IMsModelAttributesChain.Attributes.htm).
