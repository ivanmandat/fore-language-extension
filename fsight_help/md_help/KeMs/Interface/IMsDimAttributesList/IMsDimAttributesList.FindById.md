# IMsDimAttributesList.FindById


## Синтаксис


		FindById(AttributeId: String): [IDimAttribute](KeDims.chm::/interface/IDimAttribute/IDimAttribute.htm);


## Параметры


AttributeId. Идентификатор
 атрибута.


## Описание


Метод FindById осуществляет
 поиск атрибута в цепочке по идентификатору.


## Комментарии


Метод возвращает параметры атрибута в случае удачного поиска и Null, если атрибут с указанным идентификатором
 не добавлен в цепочку.
