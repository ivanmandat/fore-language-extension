# IMsDimAttributesList.FindByKey


## Синтаксис


		FindByKey(AttributeKey: Integer): [IDimAttribute](KeDims.chm::/interface/IDimAttribute/IDimAttribute.htm);


## Параметры


AttributeKey. Ключ атрибута.


## Описание


Метод FindByKey осуществляет
 поиск атрибута в цепочке по ключу.


## Комментарии


Метод возвращает параметры атрибута в случае удачного поиска и Null, если атрибут с указанным идентификатором
 не добавлен в цепочку.
