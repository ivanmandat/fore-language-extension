# IMsDimAttributesList.Remove


## Синтаксис


		Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс атрибута в цепочке.


## Описание


Метод Remove удаляет атрибут
 из цепочки по индексу.


## Комментарии


При успешном удалении атрибута метод возвращает значение True,
 при неуспешном - False.
