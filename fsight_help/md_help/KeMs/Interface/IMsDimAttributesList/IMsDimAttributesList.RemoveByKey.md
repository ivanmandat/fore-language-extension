# IMsDimAttributesList.RemoveByKey


## Синтаксис


		RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ атрибута в цепочке.


## Описание


Метод RemoveByKey удаляет атрибут
 из цепочки по ключу.


## Комментарии


При успешном удалении атрибута метод возвращает значение True,
 при неуспешном - False.
