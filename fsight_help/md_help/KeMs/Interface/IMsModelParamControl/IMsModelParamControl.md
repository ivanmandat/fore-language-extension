# IMsModelParamControl


## Описание


Интерфейс IMsModelParamControl
 содержит свойства для работы с контролируемым параметром метамодели, содержащейся
 в цепочке расчета.

		 [ControllingParam](IMsModelParamControl.ControllingParam.htm)
		 Свойство ControllingParam
		 определяет параметр, который будет контролироваться.
