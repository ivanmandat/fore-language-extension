# IMsCalculationStats.LoadFromXml


## Синтаксис


LoadFromXml(Xml: [IXmlDomElement](ModXml.chm::/develop.htm));


## Параметры


Xml. XML-файл статистик расчёта.


## Описание


Метод LoadFromXml загружает
 дерево статистик из XML-файла.


## Комментарии


Используется при сравнении статистик расчёта с помощью метода [IMsCalculationStats.Compare](IMsCalculationStats.Compare.htm).


Для сохранения статистик расчёта в XML-файл используйте метод [IMsCalculationStats.SaveToXml](IMsCalculationStats.SaveToXml.htm).
