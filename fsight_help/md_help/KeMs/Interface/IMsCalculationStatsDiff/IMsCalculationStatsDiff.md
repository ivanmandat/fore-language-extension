# IMsCalculationStatsDiff


Сборка: Ms;


## Описание


Интерфейс IMsCalculationStatsDiff
 предназначен для получения информации о различиях значений статистики
 выполнения задачи ETL.


## Иерархия наследования


           IMsCalculationStatsDiff


## Комментарии


Сравнение статистик производит метод [IMsCalculationStats.Compare](../IMsCalculationStats/IMsCalculationStats.Compare.htm).
 Для получения информации о различиях значений после сравнения используйте
 свойство [IMsCalculationStatsDiffList.Item](../IMsCalculationStatsDiffList/IMsCalculationStatsDiffList.Item.htm).
