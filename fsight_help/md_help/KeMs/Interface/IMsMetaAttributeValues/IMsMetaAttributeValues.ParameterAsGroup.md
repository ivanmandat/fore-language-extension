# IMsMetaAttributeValues.ParameterAsGroup


## Синтаксис


ParameterAsGroup: [IMsModelParam](../IMsModelParam/IMsModelParam.htm);


## Описание


Свойство ParameterAsGroup определяет параметр для передачи группы элементов, используемой для расчёта валидации.


## Комментарии


Для задания значения параметра используйте свойство [IValidationExecuteSettings.ParamValues](../IValidationExecuteSettings/IValidationExecuteSettings.ParamValues.htm).
