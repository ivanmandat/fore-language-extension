# IMsModelAttributesChain


Сборка: Ms;


## Описание


Интерфейс IMsModelAttributesChain
 содержит базовые свойства и методы для настройки цепочек атрибутов, используемых
 параметрами.


## Иерархия наследования


           IMsModelAttributesChain


## Комментарии


Интерфейс является базовым для следующих интерфейсов:


	- [IMsModelCoordAttributes](../IMsModelCoordAttributes/IMsModelCoordAttributes.htm);


	- [IMsModelParamAttributes](../IMsModelParamAttributes/IMsModelParamAttributes.htm).
