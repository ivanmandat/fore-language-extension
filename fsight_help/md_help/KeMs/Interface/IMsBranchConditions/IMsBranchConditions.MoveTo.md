# IMsBranchConditions.MoveTo


## Синтаксис


		MoveTo(FromIndex: Integer; ToIndex: Integer);


## Параметры


FromIndex. Индекс позиции,
 из которой перемещается условие расчёта;


ToIndex. Индекс позиции, в
 которую перемещается условие расчёта.


## Описание


Метод MoveTo осуществляет перемещение
 условий расчёта в коллекции.
