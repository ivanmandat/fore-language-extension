# IMsBranchConditions.Remove


## Синтаксис


		Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемого условия.


## Описание


Метод Remove
 удаляет условие из коллекции по индексу.


## Комментарии


Если удаление прошло успешно, метод возвращает
 значение True, иначе - False.
