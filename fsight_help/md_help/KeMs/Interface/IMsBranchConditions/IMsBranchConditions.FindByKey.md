# IMsBranchConditions.FindByKey


## Синтаксис


		FindByKey(Key: Integer): [IMsBranchCondition](../IMsBranchCondition/IMsBranchCondition.htm);


## Параметры


Key. Ключ
 условия расчёта.


## Описание


Метод FindByKey осуществляет
 поиск условия расчёта по ключу.


## Комментарии


Метод возвращает значение Null,
 если условие расчёта с указанным ключом не найдено.
