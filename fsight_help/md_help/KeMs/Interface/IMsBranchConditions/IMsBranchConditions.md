# IMsBranchConditions


Сборка: Ms;


## Описание


Интерфейс IMsBranchConditions
 предназначен для работы с коллекцией условий расчёта ветки условия в [цепочке](UiModelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_3_MetaModel/uimodelling_metamodel_2.htm)
 метамодели.


## Иерархия наследования


           IMsBranchConditions


## Комментарии


Для получения коллекции условий используйте свойство [IMsBranchCase.Conditions](../IMsBranchCase/IMsBranchCase.Conditions.htm).
