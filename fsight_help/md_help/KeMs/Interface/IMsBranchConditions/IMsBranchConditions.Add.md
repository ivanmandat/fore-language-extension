# IMsBranchConditions.Add


## Синтаксис


		Add(Type: [MsBranchConditionType](../../Enums/MsBranchConditionType.htm)):
		 [IMsBranchCondition](../IMsBranchCondition/IMsBranchCondition.htm);


## Параметры


Type. Тип добавляемого условия.


## Описание


Метод Add осуществляет добавление
 условия в коллекцию.


## Комментарии


Для получения указанного условия из коллекции используйте свойство [IMsBranchConditions.Item](IMsBranchConditions.Item.htm).
