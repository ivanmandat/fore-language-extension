# IMsCalculationChainEntries.IndexOf


## Синтаксис


		IndexOf(Value: [IMsCalculationChainEntry](../IMsCalculationChainEntry/IMsCalculationChainEntry.htm)):
		 Integer;


## Параметры


Value. Элемент цепочки расчёта,
 индекс которого необходимо получить.


## Описание


Метод IndexOf возвращает индекс
 элемента [цепочки
 расчёта](UiModelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_3_MetaModel/uimodelling_metamodel_2.htm).


## Комментарии


Для получения элемента цепочки расчёта по его индексу используйте свойство
 [IMsCalculationChainEntries.Item](IMsCalculationChainEntries.Item.htm).
