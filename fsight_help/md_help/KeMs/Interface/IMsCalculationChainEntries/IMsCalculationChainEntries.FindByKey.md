# IMsCalculationChainEntries.FindByKey


## Синтаксис


		FindByKey(EntryKey: Integer): [IMsCalculationChainEntry](../IMsCalculationChainEntry/IMsCalculationChainEntry.htm);


## Параметры


EntryKey. Ключ элемента цепочки
 расчёта, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск элемента [цепочки
 расчёта](UiModelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_3_MetaModel/uimodelling_metamodel_2.htm) по ключу.


## Комментарии


Если элемент не найден, то метод возвращает значение Null.
