# IMsCalculationChainEntries.Item


## Синтаксис


		Item(Index: Integer): [IMsCalculationChainEntry](../IMsCalculationChainEntry/IMsCalculationChainEntry.htm);


## Параметры


Index. Индекс элемента цепочки
 расчёта.


## Описание


Свойство Item возвращает элемент
 [цепочки
 расчёта](UiModelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_3_MetaModel/uimodelling_metamodel_2.htm).


## Комментарии


Количество элементов в коллекции возвращает свойство [IMsCalculationChainEntries.Count](IMsCalculationChainEntries.Count.htm).
 Индексация элементов сквозная и начинается с нуля.
