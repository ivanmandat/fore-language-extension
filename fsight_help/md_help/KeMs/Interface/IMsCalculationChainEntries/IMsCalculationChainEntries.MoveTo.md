# IMsCalculationChainEntries.MoveTo


## Синтаксис


		MoveTo(FromIndex: Integer; ToIndex: Integer);


## Параметры


FromIndex. Индекс позиции элемента,
 который необходимо переместить;


ToIndex. Индекс позиции, в
 которую необходимо переместить элемент.


## Описание


Метод MoveTo перемещает элемент
 [цепочки
 расчёта](UiModelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_3_MetaModel/uimodelling_metamodel_2.htm) в указанную позицию.


## Комментарии


Для получения текущего индекса элемента используйте метод [IMsCalculationChainEntries.IndexOf](IMsCalculationChainEntries.IndexOf.htm).
