# IMsCalculationChainEntries.AddFolder


## Синтаксис


		AddFolder(FolderName: String): [IMsCalculationChainFolder](../IMsCalculationChainFolder/IMsCalculationChainFolder.htm);


## Параметры


FolderName. Наименование папки,
 которую необходимо добавить.


## Описание


Метод AddFolder добавляет папку
 в [цепочку
 расчёта](UiModelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_3_MetaModel/uimodelling_metamodel_2.htm) метамодели.


## Комментарии


Метод возвращает содержимое добавленной папки.


Для удаления папки из цепочки расчёта используйте метод [IMsCalculationChainEntries.Remove](IMsCalculationChainEntries.Remove.htm) или
 [IMsCalculationChainEntries.RemoveByKey](IMsCalculationChainEntries.RemoveByKey.htm).
