# IMsCalculationChainEntries.RemoveByKey


## Синтаксис


		RemoveByKey(EntryKey: Integer): Boolean;


## Параметры


EntryKey. Ключ элемента цепочки
 расчёта.


## Описание


Метод RemoveByKey удаляет элемент
 [цепочки
 расчёта](UiModelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_3_MetaModel/uimodelling_metamodel_2.htm) по ключу.


## Комментарии


Элемент удаляется со всеми дочерними элементами. Если удаление выполнено
 успешно, то метод возвращает значение True,
 иначе - False.
