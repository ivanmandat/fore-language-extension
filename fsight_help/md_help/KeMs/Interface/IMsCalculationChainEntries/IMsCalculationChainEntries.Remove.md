# IMsCalculationChainEntries.Remove


## Синтаксис


		Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс элемента цепочки
 расчёта.


## Описание


Метод Remove удаляет элемент
 [цепочки
 расчёта](UiModelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_3_MetaModel/uimodelling_metamodel_2.htm) по индексу.


## Комментарии


Элемент удаляется со всеми дочерними элементами. Если удаление выполнено
 успешно, то метод возвращает значение True,
 иначе - False.
