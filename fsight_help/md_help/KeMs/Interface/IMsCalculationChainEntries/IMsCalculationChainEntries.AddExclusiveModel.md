# IMsCalculationChainEntries.AddExclusiveModel


## Синтаксис


		AddExclusiveModel: [IMsCalculationChainModel](../IMsCalculationChainModel/IMsCalculationChainModel.htm);


## Описание


Метод AddExclusiveModel добавляет
 внутреннюю модель в [цепочку
 расчёта](UiModelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_3_MetaModel/uimodelling_metamodel_2.htm) метамодели.


## Комментарии


Для удаления внутренней модели из цепочки расчёта используйте метод
 [IMsCalculationChainEntries.Remove](IMsCalculationChainEntries.Remove.htm)
 или [IMsCalculationChainEntries.RemoveByKey](IMsCalculationChainEntries.RemoveByKey.htm).
