# IMsCalculationChainEntries.Visit


## Синтаксис


Visit(Visitor: IMsCalculationChainVisitor;
 Recursive: Boolean): Boolean;


## Параметры


Visitor. Пользовательские обработчики
 для каждого типа элемента цепочки;


Recursive. Признак определения
 задействованности цепочки расчета вложенных метамоделей в процессе
 обхода.


## Описание


Метод Visit выполняет пользовательские
 обработчики для всех элементов цепочки расчета.
