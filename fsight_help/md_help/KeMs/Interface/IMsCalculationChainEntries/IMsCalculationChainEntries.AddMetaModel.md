# IMsCalculationChainEntries.AddMetaModel


## Синтаксис


		AddMetaModel(MetaModel: [IMsMetaModel](../IMsMetaModel/IMsMetaModel.htm)):
		 [IMsCalculationChainMetaModel](../IMsCalculationChainMetaModel/IMsCalculationChainMetaModel.htm);


## Параметры


MetaModel. Метамодель, которую
 необходимо добавить в цепочку расчёта.


## Описание


Метод AddMetaModel добавляет
 метамодель в [цепочку
 расчёта](UiModelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_3_MetaModel/uimodelling_metamodel_2.htm) метамодели.


## Комментарии


Для удаления метамодели из цепочки расчёта используйте метод [IMsCalculationChainEntries.Remove](IMsCalculationChainEntries.Remove.htm) или
 [IMsCalculationChainEntries.RemoveByKey](IMsCalculationChainEntries.RemoveByKey.htm).

Использование свойства приведено в примере для [IMsModelParamControl.ControllingParam](../IMsModelParamControl/IMsModelParamControl.ControllingParam.htm).
