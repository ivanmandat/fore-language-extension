# IMsCalculationChainEntries.AddModel


## Синтаксис


		AddModel(Model: [IMsModel](../IMsModel/IMsModel.htm)):
		 [IMsCalculationChainModel](../IMsCalculationChainModel/IMsCalculationChainModel.htm);


## Параметры


Model. Модель, которую необходимо
 добавить в цепочку расчёта.


## Описание


Метод AddModel добавляет модель
 в [цепочку
 расчёта](UiModelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_3_MetaModel/uimodelling_metamodel_2.htm) метамодели.


## Комментарии


Для удаления модели из цепочки расчёта используйте метод [IMsCalculationChainEntries.Remove](IMsCalculationChainEntries.Remove.htm) или
 [IMsCalculationChainEntries.RemoveByKey](IMsCalculationChainEntries.RemoveByKey.htm).

Использование свойства приведено в примере для [IMsCalculationChainEntry.InheritModelPeriod](../IMsCalculationChainEntry/IMsCalculationChainEntry.InheritModelPeriod.htm).
