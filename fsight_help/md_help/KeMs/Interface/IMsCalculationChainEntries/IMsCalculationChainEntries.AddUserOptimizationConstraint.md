# IMsCalculationChainEntries.AddUserOptimizationConstraint


## Синтаксис


AddUserOptimizationConstraint(UserOptimizationEntry:
 [IMsCalculationChainUserOptimization](../IMsCalculationChainUserOptimization/IMsCalculationChainUserOptimization.htm);
 NonLinearConstraint: [IMsNonLinearConstraint](../IMsNonLinearConstraint/IMsNonLinearConstraint.htm)):
 [IMsCalculationChainUserOptimizationConstraint](../IMsCalculationChainUserOptimizationConstraint/IMsCalculationChainUserOptimizationConstraint.htm);


## Параметры


UserOptimizationEntry. Элемент
 цепочки расчёта, соответствующий пользовательской задаче оптимизации.


NonLinearConstraint. Ограничение
 целевой функции.


## Описание


Метод AddUserOptimizationConstraint
 добавляет в цепочку расчёта ограничение пользовательской задачи оптимизации.
