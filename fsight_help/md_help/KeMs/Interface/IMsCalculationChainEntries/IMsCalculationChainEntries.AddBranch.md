# IMsCalculationChainEntries.AddBranch


## Синтаксис


		AddBranch(BranchName: String): [IMsCalculationChainBranch](../IMsCalculationChainBranch/IMsCalculationChainBranch.htm);


## Параметры


BranchName. Наименование условия.


## Описание


Метод AddBranch
 добавляет условие в [цепочку
 расчёта](UiModelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_3_MetaModel/uimodelling_metamodel_2.htm) метамодели.


## Комментарии


Для удаления условия из цепочки расчёта используйте метод [IMsCalculationChainEntries.Remove](IMsCalculationChainEntries.Remove.htm) или
 [IMsCalculationChainEntries.RemoveByKey](IMsCalculationChainEntries.RemoveByKey.htm).
