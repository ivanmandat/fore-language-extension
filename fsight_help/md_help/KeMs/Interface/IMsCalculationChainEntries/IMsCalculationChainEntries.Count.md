# IMsCalculationChainEntries.Count


## Синтаксис


		Count: Integer;


## Описание


Свойство Count возвращает количество
 элементов, содержащихся в данном подкаталоге [цепочки
 расчёта](UiModelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_3_MetaModel/uimodelling_metamodel_2.htm).


## Комментарии


Для получения элемента из цепочки расчёта используйте свойство [IMsCalculationChainEntries.Item](IMsCalculationChainEntries.Item.htm).
