# IMsScenarioTreeEntries.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс элемента, который необходимо удалить.


## Описание


Метод Remove осуществляет удаление элемента из дерева сценариев, индекс которого передается посредством параметра Index.


Примечание. Элементы из дерева сценариев удаляются со всеми дочерними элементами.
