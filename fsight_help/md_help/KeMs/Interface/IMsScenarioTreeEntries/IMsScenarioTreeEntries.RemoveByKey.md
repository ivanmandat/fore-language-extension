# IMsScenarioTreeEntries.RemoveByKey


## Синтаксис


RemoveByKey(EntryKey: Integer): Boolean;


## Параметры


EntryKey. Ключ элемента, который необходимо удалить.


## Описание


Метод RemoveByKey осуществляет удаление элемента из дерева сценариев, ключ которого передается посредством параметра EntryKey.


Примечание. Элементы из дерева сценариев удаляются со всеми дочерними элементами.
