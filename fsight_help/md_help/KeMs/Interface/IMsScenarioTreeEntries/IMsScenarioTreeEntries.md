# IMsScenarioTreeEntries


Сборка: Ms;


## Описание


Интерфейс IMsScenarioTreeEntries
 содержит свойства и методы для работы с деревом сценариев на данном уровне
 иерархии.


## Иерархия наследования


           IMsScenarioTreeEntries


## Комментарии


Дерево сценариев состоит из элементов, реализуемых интерфейсом [IMsScenarioTreeEntry](../IMsScenarioTreeEntry/IMsScenarioTreeEntry.htm).
