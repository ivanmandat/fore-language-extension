# IMsScenarioTreeEntries.FindByKey


## Синтаксис


FindByKey(EntryKey: Integer): [IMsScenarioTreeEntry](../IMsScenarioTreeEntry/IMsScenarioTreeEntry.htm);


## Параметры


EntryKey. Ключ элемента, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет поиск элемента в дереве сценариев, ключ которого передается посредством параметра EntryKey.
