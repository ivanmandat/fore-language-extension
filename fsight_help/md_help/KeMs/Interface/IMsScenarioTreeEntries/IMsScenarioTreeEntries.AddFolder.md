# IMsScenarioTreeEntries.AddFolder


## Синтаксис


AddFolder(FolderName: String): [IMsScenarioTreeFolder](../IMsScenarioTreeFolder/IMsScenarioTreeFolder.htm);


## Параметры


FolderName. Наименование папки,
 которую необходимо добавить в дерево сценариев.


## Описание


Метод AddFolder осуществляет
 добавление новой папки в дерево сценариев контейнера моделирования.
