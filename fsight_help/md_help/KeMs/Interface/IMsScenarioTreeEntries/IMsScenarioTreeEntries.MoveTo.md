# IMsScenarioTreeEntries.MoveTo


## Синтаксис


MoveTo(FromIndex: Integer; ToIndex: Integer);


## Параметры


FromIndex. Индекс позиции элемента,
 который необходимо переместить;


ToIndex. Индекс позиции, в
 которую необходимо переместить элемент.


## Описание


Метод MoveTo осуществляет перемещение
 элементов в дереве сценариев.
