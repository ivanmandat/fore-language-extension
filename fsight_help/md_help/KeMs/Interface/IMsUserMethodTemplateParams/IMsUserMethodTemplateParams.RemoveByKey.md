# IMsUserMethodTemplateParams.RemoveByKey


## Синтаксис


RemoveByKey(ParamKey: Integer): Boolean;


## Параметры


TemplateKey. Ключ параметра шаблона пользовательского метода.


## Описание


Метод RemoveByKey осуществляет удаление параметра шаблона пользовательского метода, ключ которого передается посредством параметра ParamKey.
