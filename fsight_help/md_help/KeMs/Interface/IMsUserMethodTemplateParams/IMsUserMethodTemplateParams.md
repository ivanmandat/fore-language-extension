# IMsUserMethodTemplateParams


## Описание


Интерфейс IMsUserMethodTemplateParams
 содержит свойства и методы для работы с коллекцией параметров шаблонов
 пользовательских методов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMsUserMethodTemplateParams.Count.htm)
		 Свойство Count возвращает
		 количество параметров шаблона пользовательского метода.


		 ![](../../Property_Image.gif)
		 [Item](IMsUserMethodTemplateParams.Item.htm)
		 Свойство Item возвращает
		 настройки параметра шаблона пользовательского метода.
