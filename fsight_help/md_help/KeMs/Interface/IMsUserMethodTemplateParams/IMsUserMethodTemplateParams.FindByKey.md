# IMsUserMethodTemplateParams.FindByKey


## Синтаксис


FindByKey(ParamKey: Integer): [IMsUserMethodTemplateParam](../IMsUserMethodTemplateParam/IMsUserMethodTemplateParam.htm);


## Параметры


TemplateKey. Ключ параметра шаблона пользовательского метода, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет поиск и возвращает свойства параметра шаблона пользовательского метода, в случае удачного поиска.
