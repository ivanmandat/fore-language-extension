# IMsUserMethodTemplateParams.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс параметра шаблона пользовательского метода.


## Описание


Метод Remove осуществляет удаление параметра шаблона пользовательского метода, индекс которого передается посредством параметра Index.
