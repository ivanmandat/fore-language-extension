# IMsDimensionFilter.Fixed


## Синтаксис


		Fixed: Boolean;


## Описание


Свойство Fixed определяет
 признак фиксации измерения.


## Комментарии


Допустимые значения:


	- True. Измерение зафиксировано;


	- False. Измерение незафиксировано.


Особенности расчёта для зафиксированных и незафиксированных измерений
 приведены в комментарии к свойству [IMsFormula.FreeDimensions](../IMsFormula/IMsFormula.FreeDimensions.htm).
