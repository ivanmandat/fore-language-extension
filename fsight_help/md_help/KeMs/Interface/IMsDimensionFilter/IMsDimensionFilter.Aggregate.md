# IMsDimensionFilter.Aggregate


## Синтаксис


		Aggregate: Boolean;


## Описание


Свойство Aggregate определяет
 признак проведения агрегации по измерению при множественной отметке.


## Комментарии


Допустимые значения:


	- True. Данные по отмеченным
	 элементам измерения агрегируются, после расчёта метода они будут доступны
	 в результирующей матрице по элементу «-2» для данного измерения;


	- False. Агрегация не
	 производится, результат выгружается по всем элементам отметки измерения.
