# IMsDimensionFilter


Сборка: Ms;


## Описание


Интерфейс IMsDimensionFilter
 предназначен для настройки параметров измерения, по которому разрешена
 множественная отметка.


## Иерархия наследования


           [INamedEntity](KeSom.chm::/Interface/INamedEntity/INamedEntity.htm)


           IMsDimensionFilter


## Комментарии


Для работы с коллекцией подобных измерений используйте интерфейс [IMsDimensionFilterList](../IMsDimensionFilterList/IMsDimensionFilterList.htm).
