# IMsScenarioTreeElement


## Описание


Интерфейс IMsScenarioTreeElement
 содержит свойства элемента дерева сценариев, которому соответствует сценарий
 контейнера моделирования.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Scenario](IMsScenarioTreeElement.Scenario.htm)
		 Свойство Scenario возвращает
		 параметры сценария контейнера моделирования.


		 ![](../../Property_Image.gif)
		 [ScenarioDescriptor](IMsScenarioTreeElement.ScenarioDescriptor.htm)
		 Свойство ScenarioDescriptor
		 возвращает описание сценария контейнера моделирования как объекта
		 репозитория.
