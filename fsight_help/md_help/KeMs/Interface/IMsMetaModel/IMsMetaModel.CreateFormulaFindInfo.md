# IMsMetaModel.CreateFormulaFindInfo


## Синтаксис


		CreateFormulaFindInfo: [IMsFindFormulaInfo](../IMsFindFormulaInfo/IMsFindFormulaInfo.htm);


## Описание


Свойство CreateFormulaFindInfo
 возвращает объект, используемый для задания условий поиска формул.


## Комментарии


Поиск формул осуществляется с помощью метода [FindFormulas](IMsMetaModel.FindFormulas.htm).


## Пример


Использование свойства приведено в примере для метода [IMsMetaModel.FindFormulas](IMsMetaModel.FindFormulas.htm).
