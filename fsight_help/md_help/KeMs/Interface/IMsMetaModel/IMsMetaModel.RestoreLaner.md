# IMsMetaModel.RestoreLaner


## Синтаксис


RestoreLaner(Laner: [ILaner](Laner.chm::/Interface/ILaner/ILaner.htm);
 ParamValues: [IMsModelParamValues](../IMsModelParamValues/IMsModelParamValues.htm));


## Параметры


Laner. Рабочая область базы
 данных временных рядов.


ParamValues. Параметры, с которыми
 будет восстановлена метамодель.


## Описание


Метод RestoreLaner восстанавливает
 рабочую область базы данных временных рядов из метамодели.


## Комментарии


Если параметры для метамодели не заданы, то для ParamValues следует
 указывать значение Null.
