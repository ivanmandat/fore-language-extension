# IMsMetaModel.VisualController


## Синтаксис


VisualController: [IMsMetaModelVisualController](../IMsMetaModelVisualController/IMsMetaModelVisualController.htm);


## Описание


Свойство VisualController возвращает параметры визуального построения метамодели.


## Комментарии


С помощью свойства VisualController в метамодели ведется работа с внутренней базой данных временных рядов.
