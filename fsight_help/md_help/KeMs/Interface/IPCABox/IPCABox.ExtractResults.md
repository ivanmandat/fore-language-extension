# IPCABox.ExtractResults


## Синтаксис


ExtractResults;


## Описание


Метод ExtractResults инициализирует процесс выгрузки результатов расчета метода главных компонент. После вызова метода будет выведено окно выгрузки результатов со списком рассчитанных главных компонент.
