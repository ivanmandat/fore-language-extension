# IPCABox.RemoveByIndex


## Синтаксис


RemoveByIndex(Index: Integer);


## Параметры


Index. Индекс переменной в списке переменных компонента.


## Описание


Метод RemoveByIndex осуществляет удаление переменной, индекс которой передается посредством параметра Index, из списка переменных компонента.
