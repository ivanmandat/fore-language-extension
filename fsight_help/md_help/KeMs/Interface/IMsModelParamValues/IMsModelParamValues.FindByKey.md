# IMsModelParamValues.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IMsModelParamValue](../IMsModelParamValue/IMsModelParamValue.htm);


## Параметры


Key. Ключ параметра, значение которого необходимо найти.


## Описание


Метод FindByKey осуществляет поиск значения параметра в коллекции по ключу.


## Комментарии


Если значение для параметра с указанным ключом не найдено, то метод возвращает значение Null.
