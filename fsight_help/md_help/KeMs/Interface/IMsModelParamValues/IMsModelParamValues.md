# IMsModelParamValues


Сборка: Ms;


## Описание


Интерфейс IMsModelParamValues
 предназначен для работы с коллекцией значений параметров метамодели перед
 началом расчета задачи.


## Комментарии


Для работы с коллекцией параметров используйте интерфейс [IMsModelParams](../IMsModelParams/IMsModelParams.htm).
