# IMsModelParamValues.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество параметров в коллекции.


## Комментарии


Для получения определенного значения параметра из коллекции используйте свойство [IMsModelParamValues.Item](IMsModelParamValues.Item.htm).
