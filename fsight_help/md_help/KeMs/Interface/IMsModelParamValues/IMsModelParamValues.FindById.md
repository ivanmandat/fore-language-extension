# IMsModelParamValues.FindById


## Синтаксис


FindById(Id: String): [IMsModelParamValue](../IMsModelParamValue/IMsModelParamValue.htm);


## Параметры


Id. Идентификатор искомого параметра.


## Описание


Метод FindById осуществляет поиск значения параметра в коллекции по идентификатору.


## Комментарии


Если значение для параметра с указанным идентификатором не найдено, то метод возвращает значение Null.
