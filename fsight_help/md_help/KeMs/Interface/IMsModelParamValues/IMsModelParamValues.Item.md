# IMsModelParamValues.Item


## Синтаксис


Item(Index: Integer): [IMsModelParamValue](../IMsModelParamValue/IMsModelParamValue.htm);


## Параметры


Index. Индекс параметра, значение которого необходимо получить.


## Описание


Свойство Item возвращает значение параметра из коллекции по индексу.


## Комментарии


Индексация значений параметров в коллекции сквозная и начинается с нуля. Для получения количества значений параметров в коллекции используйте свойство [IMsModelParamValues.Count](IMsModelParamValues.Count.htm).
