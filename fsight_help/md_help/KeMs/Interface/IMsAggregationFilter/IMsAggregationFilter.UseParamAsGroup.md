# IMsAggregationFilter.UseParamAsGroup


## Синтаксис


		UseParamAsGroup: Boolean;


## Описание


Свойство UseParamAsGroup определяет
 признак передачи группы элементов в качестве значения параметра.


## Комментарии


Допустимые значения:


	- True. Параметр является
	 группой элементов;


	- False. Параметр передает
	 значение одного элемента.


Идентификатор используемого параметра определяет свойство [IMsAggregationFilter.AggregationParamID](IMsAggregationFilter.AggregationParamID.htm).
