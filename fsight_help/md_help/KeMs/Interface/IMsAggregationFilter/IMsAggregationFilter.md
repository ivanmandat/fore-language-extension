# IMsAggregationFilter


Сборка: Ms;


## Описание


Интерфейс IMsAggregationFilter
 содержит свойства для работы с фильтром агрегации.


## Иерархия наследования


           [INamedEntityModel](ForeSys.chm::/Interface/INamedEntityModel/INamedEntityModel.htm)


           IMsAggregationFilter


## Комментарии


Для работы с коллекцией фильтров агрегации используйте интерфейс [IMsAggregationFilterList](../IMsAggregationFilterList/IMsAggregationFilterList.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AdvancedFilter](IMsAggregationFilter.AdvancedFilter.htm)
		 Свойство AdvancedFilter
		 возвращает параметры расширенного фильтра.


		 ![](../../Property_Image.gif)
		 [AggregationGroup](IMsAggregationFilter.AggregationGroup.htm)
		 Свойство AggregationGroup
		 определяет группу, по которой будет производиться агрегация.


		 ![](../../Property_Image.gif)
		 [AggregationParamID](IMsAggregationFilter.AggregationParamID.htm)
		 Свойство AggregationParamID
		 определяет идентификатор параметра, используемого для задания
		 агрегируемых элементов.


		 ![](../../Property_Image.gif)
		 [AggregationSelection](IMsAggregationFilter.AggregationSelection.htm)
		 Свойство AggregationSelection
		 определяет отметку, используемую для задания агрегируемых элементов.


		 ![](../../Property_Image.gif)
		 [FilterDimension](IMsAggregationFilter.FilterDimension.htm)
		 Свойство FilterDimension
		 возвращает фильтруемое измерение.


		 ![](../../Property_Image.gif)
		 [HierarchyKey](IMsAggregationFilter.HierarchyKey.htm)
		 Свойство HierarchyKey
		 определяет ключ агрегации по уровням [альтернативной
		 иерархии](UiNavObj.chm::/Cube/CreateCube/Master_Standart/UiMd_Cube_CreateCube_Master_Standart_7.htm#alt_agr) в коллекции альтернативных иерархий
		 измерения.


		 ![](../../Property_Image.gif)
		 [HierarchyParamAttributes](IMsAggregationFilter.HierarchyParamAttributes.htm)
		 Свойство HierarchyParamAttributes
		 возвращает настройки привязки альтернативной иерархии к атрибутам
		 параметра.


		 ![](../../Property_Image.gif)
		 [InvertSelection](IMsAggregationFilter.InvertSelection.htm)
		 Свойство InvertSelection
		 определяет признак использования инвертированной отметки в фильтре
		 агрегации при расчёте [задачи
		 моделирования](UiModelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_4_Problem/uimodelling_problem.htm).


		 ![](../../Property_Image.gif)
		 [OutputParamID](IMsAggregationFilter.OutputParamID.htm)
		 Свойство OutputParamID
		 определяет параметр, в который будет выгружаться отметка фильтра
		 в каждой итерации расчёта.


		 ![](../../Property_Image.gif)
		 [UseParamAsGroup](IMsAggregationFilter.UseParamAsGroup.htm)
		 Свойство UseParamAsGroup
		 определяет признак передачи группы элементов в качестве значения
		 параметра.
