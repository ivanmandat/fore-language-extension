# IMsTransformStringGenerator


## Описание


Интерфейс IMsTransformStringGenerator
 содержит свойства и методы для работы с генератором строкового представления
 параметров расчета.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Transform](IMsTransformStringGenerator.Transform.htm)


		 Свойство Transform
		 возвращает параметры расчета, для которых генерируется строковое
		 представление.
