# IMsFormulaTransformVariable.SlicesTree


## Синтаксис


SlicesTree(OutputVariable: [IMsFormulaTransformVariable](IMsFormulaTransformVariable.htm)):
 [IMsFormulaTransformSlicesTree](../IMsFormulaTransformSlicesTree/IMsFormulaTransformSlicesTree.htm);


## Параметры


OutputVariable. Параметр не
 используется, оставлен для совместимости. Вместо него необходимо передавать
 значение Null.


## Описание


Свойство SlicesTree возвращает
 объект, позволяющий работать с коллекцией срезов переменной в виде дерева.
