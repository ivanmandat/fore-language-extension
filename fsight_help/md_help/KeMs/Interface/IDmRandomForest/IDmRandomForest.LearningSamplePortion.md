# IDmRandomForest.LearningSamplePortion


## Синтаксис


LearningSamplePortion: Double;


## Описание


Свойство LearningSamplePortion
 определяет долю выборки для обучения дерева.


## Комментарии


Допустимые значения в диапазоне: (0, 1].


Значение по умолчанию: «0,7».
