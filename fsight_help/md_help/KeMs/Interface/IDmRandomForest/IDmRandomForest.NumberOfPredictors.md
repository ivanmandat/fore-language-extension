# IDmRandomForest.NumberOfPredictors


## Синтаксис


NumberOfPredictors: Integer;


## Описание


Свойство NumberOfPredictors
 определяет число признаков, которые формируют случайное дерево.


## Комментарии


Допустимые значения в диапазоне: [1, ∞).


Значение по умолчанию: «4».
