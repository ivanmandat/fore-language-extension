# IValidationMissingMetadata.Attributes


## Синтаксис


Attributes: [IValidationMetaAttributes](../IValidationMetaAttributes/IValidationMetaAttributes.htm);


## Описание


Свойство Attributes возвращает
 коллекцию атрибутов, отсутствие значений которых будет проверять валидация.


## Комментарии


Для корректного расчета валидации необходимо, чтобы коллекция атрибутов
 содержала как минимум один элемент.
