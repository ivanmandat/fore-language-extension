# IMsDiagnosticTestResults.CoefficientsAR


## Синтаксис


		CoefficientsAR: [ICoefficients](StatLib.chm::/Interface/ICoefficients/ICoefficients.htm);


## Описание


Свойство CoefficientsAR возвращает коэффициенты авторегрессии.


## Комментарии


Коэффициенты рассчитываются, если в тестируемой модели задан порядок
 авторегрессии. Для получения наименований коэффициентов используйте свойство
 [IMsDiagnosticTestResults.CoefficientsARNames](IMsDiagnosticTestResults.CoefficientsARNames.htm).
