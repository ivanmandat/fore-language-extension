# IMsDiagnosticTestResults.Error


## Синтаксис


		Error: String;


## Описание


Свойство Error возвращает ошибки,
 возникшие при расчёте диагностического теста.


## Комментарии


Если тест рассчитался без ошибок, то свойство возвращает пустое значение.
