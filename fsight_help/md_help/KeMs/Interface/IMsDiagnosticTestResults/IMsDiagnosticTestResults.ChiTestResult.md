# IMsDiagnosticTestResults.ChiTestResult


## Синтаксис


		ChiTestResult: Boolean;


## Описание


Свойство ChiTestResult возвращает результаты теста по статистике Хи-квадрат.


## Комментарии


Допустимые значения:


	- True. Гипотеза принимается;


	- False. Гипотеза отвергается.


Тестируемая гипотеза отличается в зависимости от типа теста.
