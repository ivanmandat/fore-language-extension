# IMsDiagnosticTestResults.CoefficientsARNames


## Синтаксис


		CoefficientsARNames: Array;


## Описание


Свойство CoefficientsARNames
 возвращает наименование коэффициентов авторегрессии.


## Комментарии


Коэффициенты рассчитываются, если в тестируемой модели задан порядок
 авторегрессии. Для получения значений коэффициентов используйте свойство
 [IMsDiagnosticTestResults.CoefficientsAR](IMsDiagnosticTestResults.CoefficientsAR.htm).
