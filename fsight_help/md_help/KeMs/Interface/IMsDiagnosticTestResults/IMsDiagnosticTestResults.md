# IMsDiagnosticTestResults


Сборка: Ms;


## Описание


Интерфейс IMsDiagnosticTestResults
 предназначен для работы с результатами выполнения диагностического теста.


## Иерархия наследования


           IMsDiagnosticTestResults


## Комментарии


Результаты выполнения теста возвращает метод [IMsDiagnosticTest.Execute](../IMsDiagnosticTest/IMsDiagnosticTest.Execute.htm).
