# IMsDiagnosticTestResults.CoefficientsMANames


## Синтаксис


		CoefficientsMANames: Array;


## Описание


Свойство CoefficientsMANames
 возвращает наименование коэффициентов скользящего среднего.


## Комментарии


Коэффициенты рассчитываются, если в тестируемой модели задан порядок
 скользящего среднего. Для получения значений коэффициентов используйте
 свойство [IMsDiagnosticTestResults.CoefficientsMA](IMsDiagnosticTestResults.CoefficientsMA.htm).
