# IMsDiagnosticTestResults.CoefficientsMA


## Синтаксис


		CoefficientsMA: [ICoefficients](StatLib.chm::/Interface/ICoefficients/ICoefficients.htm);


## Описание


Свойство CoefficientsMA возвращает
 коэффициенты скользящего среднего.


## Комментарии


Коэффициенты рассчитываются, если в тестируемой модели задан порядок
 скользящего среднего. Для получения наименований коэффициентов используйте
 свойство [IMsDiagnosticTestResults.CoefficientsMANames](IMsDiagnosticTestResults.CoefficientsMANames.htm).
