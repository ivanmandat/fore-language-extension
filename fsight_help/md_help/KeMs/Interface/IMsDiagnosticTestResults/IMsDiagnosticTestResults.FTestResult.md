# IMsDiagnosticTestResults.FTestResult


## Синтаксис


		FTestResult: Boolean;


## Описание


Свойство FTestResult возвращает
 результаты теста по статистике Фишера.


## Комментарии


Допустимые значения:


	- True. Гипотеза принимается;


	- False. Гипотеза отвергается.


Тестируемая гипотеза отличается в зависимости от типа теста.
