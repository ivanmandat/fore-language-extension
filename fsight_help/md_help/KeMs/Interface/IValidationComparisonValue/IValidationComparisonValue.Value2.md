# IValidationComparisonValue.Value2


## Синтаксис


Value2: Double;


## Описание


Свойство Value2 определяет второе
 сравниваемое значение.


## Комментарии


Данное свойство учитывается, если значение свойства [IValidationComparisonValue.ComparisonOperator](IValidationComparisonValue.ComparisonOperator.htm)
 - [ValidationComparisonOperator.Between](../../Enums/ValidationComparisonOperator.htm)
 или [ValidationComparisonOperator.Notbetween](../../Enums/ValidationComparisonOperator.htm).
