# IMsStubLoadSelectionValues


Сборка: Ms;


## Описание


Интерфейс IMsStubLoadSelectionValues
 предназначен для работы с коллекцией отметок в источниках данных, используемой
 для загрузки данных при расчете задачи.


## Иерархия наследования


           IMsStubLoadSelectionValues


## Комментарии


Для получения коллекции используйте свойство [IMsProblemCalculationSettings.StubLoadSelections](../IMsProblemCalculationSettings/IMsProblemCalculationSettings.StubLoadSelections.htm).
