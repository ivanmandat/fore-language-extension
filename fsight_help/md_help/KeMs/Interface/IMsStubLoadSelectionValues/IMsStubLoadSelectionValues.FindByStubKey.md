# IMsStubLoadSelectionValues.FindByStubKey


## Синтаксис


FindByStubKey(StubKey: Integer): [IMsStubLoadSelectionValue](../IMsStubLoadSelectionValue/IMsStubLoadSelectionValue.htm);


## Параметры


StubKey. Ключ источника данных.


## Описание


Метод FindByStubKey выполняет
 поиск в коллекции отметки, принадлежащей указанному источнику данных.


## Комментарии


Если указанная отметка не найдена, то метод вернет значение Null.
