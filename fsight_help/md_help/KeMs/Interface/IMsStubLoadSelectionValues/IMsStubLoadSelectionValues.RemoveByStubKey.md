# IMsStubLoadSelectionValues.RemoveByStubKey


## Синтаксис


RemoveByStubKey(StubKey: Integer): Boolean;


## Параметры


StubKey. Ключ источника данных.


## Описание


Метод RemoveByStubKey
 удаляет из коллекции отметку, соответствующую указанному источнику данных.


## Комментарии


Если удаление выполнено успешно, то метод вернет значение True,
 иначе - False.
