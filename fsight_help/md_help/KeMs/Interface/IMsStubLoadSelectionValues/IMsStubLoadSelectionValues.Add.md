# IMsStubLoadSelectionValues.Add


## Синтаксис


Add(StubKey: Integer): [IMsStubLoadSelectionValue](../IMsStubLoadSelectionValue/IMsStubLoadSelectionValue.htm);


## Параметры


StubKey. Ключ источника данных.


## Описание


Метод Add добавляет в
 коллекцию отметку для указанного источника данных.


## Комментарии


Для получения отметки из коллекции используйте свойство [IMsStubLoadSelectionValues.Item](IMsStubLoadSelectionValues.Item.htm).
