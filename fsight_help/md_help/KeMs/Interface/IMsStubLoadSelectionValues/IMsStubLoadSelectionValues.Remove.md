# IMsStubLoadSelectionValues.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс отметки в коллекции.


## Описание


Метод Remove удаляет из
 коллекции отметку с указанным индексом.


## Комментарии


Индексация отметок в коллекции сквозная и начинается с нуля.


Если удаление выполнено успешно, то метод вернет значение True,
 иначе - False.
