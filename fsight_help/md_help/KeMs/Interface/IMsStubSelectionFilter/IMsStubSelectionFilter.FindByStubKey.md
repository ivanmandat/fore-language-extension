# IMsStubSelectionFilter.FindByStubKey


## Синтаксис


FindByStubKey(StubKey: Integer): [IMsStubSelection](../IMsStubSelection/IMsStubSelection.htm);


## Параметры


StubKey. Ключ источника данных.


## Описание


Метод FindByStubKey осуществляет
 поиск фильтра по ключу источника данных.


## Комментарии


Метод возвращает фильтр по отметке в случае удачного поиска и Null, если для источника с указанным
 ключом фильтр еще не создан.
