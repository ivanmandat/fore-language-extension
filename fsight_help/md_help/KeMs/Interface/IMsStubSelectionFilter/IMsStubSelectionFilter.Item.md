# IMsStubSelectionFilter.Item


## Синтаксис


Item(Index: Integer): [IMsStubSelection](../IMsStubSelection/IMsStubSelection.htm);


## Параметры


Index. Индекс источника, для
 которого необходимо получить фильтр по отметке.


## Описание


Свойство Item возвращает фильтр
 по отметке для источника с указанным индексом.
