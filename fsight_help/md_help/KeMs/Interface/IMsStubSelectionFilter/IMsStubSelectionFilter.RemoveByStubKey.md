# IMsStubSelectionFilter.RemoveByStubKey


## Синтаксис


RemoveByStubKey(StubKey: Integer): Boolean;


## Параметры


StubKey. Ключ источника данных.


## Описание


Метод RemoveByStubKey удаляет
 из коллекции фильтр источника данных с указанным ключом.


## Комментарии


Метод возвращает значение True,
 если удаление прошло удачно, и False
 во всех других случаях.
