# IMsStubSelectionFilter.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс источника данных.


## Описание


Метод Remove удаляет из коллекции
 фильтр источника данных с указанным индексом.


## Комментарии


Метод возвращает значение True,
 если удаление прошло удачно и False
 во всех других случаях.
