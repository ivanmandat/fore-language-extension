# IMsDimIteratorAdvancedFilterCondition


Сборка: Ms;


## Описание


Интерфейс IMsDimIteratorAdvancedFilterCondition
 предназначен для работы с условием [расширенного фильтра](uinav.chm::/GUI/UFE/AttributeFiltering.htm).


## Иерархия наследования


           IMsDimIteratorAdvancedFilterCondition


## Комментарии


Для работы с расширенным фильтром используйте интерфейс [IMsDimIteratorAdvancedFilter](../IMsDimIteratorAdvancedFilter/IMsDimIteratorAdvancedFilter.htm).
