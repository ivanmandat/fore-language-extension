# IMsDimIteratorAdvancedFilterCondition.IgnoreEmptyAttributes


## Синтаксис


IgnoreEmptyAttributes: Boolean;


## Описание


Свойство IgnoreEmptyAttributes
 определяет признак учёта пустых значений сравниваемого атрибута измерения.


## Комментарии


Допустимые значения:


	- True. Пустые значения
	 атрибута измерения не учитываются при выполнении фильтрации;


	- False. По умолчанию.
	 Пустые значения атрибута измерения учитываются при выполнении фильтрации.


Для получения подробной информации обратитесь к разделу «[Настройка фильтрации
 по значениям атрибутов](Uinav.chm::/GUI/UFE/AttributeFiltering.htm)».
