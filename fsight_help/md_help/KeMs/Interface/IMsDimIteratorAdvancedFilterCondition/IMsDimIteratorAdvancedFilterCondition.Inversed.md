# IMsDimIteratorAdvancedFilterCondition.Inversed


## Синтаксис


		Inversed: Boolean;


## Описание


Свойство Inversed определяет
 признак обратности условия.


## Комментарии


Допустимые значения:


	- True. Условие является
	 обратным, то есть фильтруются все элементы, не соответствующие условию;


	- False. Условие является
	 прямым, то есть фильтруются все элементы, соответствующие условию.
