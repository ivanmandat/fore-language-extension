# IValidationRevisionMetadataComparison.Revision


## Синтаксис


Revision: Integer;


## Описание


Свойство Revision определяет
 ключ ревизии, с которой сравниваются актуальные метаданные.


## Комментарии


В Revision следует указывать
 ключ только той ревизии, которая является меткой. Если задан ключ «-1»,
 то сравнение осуществляется с последней перед актуальными данными ревизией.
