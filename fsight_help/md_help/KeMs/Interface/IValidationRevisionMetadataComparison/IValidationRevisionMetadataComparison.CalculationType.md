# IValidationRevisionMetadataComparison.CalculationType


## Синтаксис


CalculationType: [MsCalculationType](../../Enums/MsCalculationType.htm);


## Описание


Свойство CalculationType определяет
 способ расчета правила валидации: на уровне ряда или на уровне наблюдений
 ряда.


## Комментарии


При расчете правила в зависимости от значения CalculationType
 исключения будут выгружаться, соответственно, либо на заголовок
 ряда, либо на точки ряда.
