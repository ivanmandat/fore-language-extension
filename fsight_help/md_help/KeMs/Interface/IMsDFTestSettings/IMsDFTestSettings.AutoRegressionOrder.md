# IMsDFTestSettings.AutoRegressionOrder


## Синтаксис


		AutoRegressionOrder: Integer;


## Описание


Свойство AutoRegressionOrder
 определяет порядок авторегрессии.


## Комментарии


Свойство AutoRegressionOrder
 может принимать только неотрицательные значения.
