# IMsDFTestSettings.EquationType


## Синтаксис


		EquationType: [ADFEquationType](StatLib.chm::/Enums/ADFEquationType.htm);


## Описание


Свойство EquationType определяет тип модели, используемой для расчёта теста.


## Комментарии


Для определения применяемого метода дифференцирования используйте свойство
 [IMsDFTestSettings.TestedSeries](IMsDFTestSettings.TestedSeries.htm).
