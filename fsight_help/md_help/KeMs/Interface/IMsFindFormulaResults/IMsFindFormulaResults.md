# IMsFindFormulaResults


Сборка: Ms;


## Описание


Интерфейс IMsFindFormulaResults
 предназначен для работы с коллекцией результатов поиска формул.


## Иерархия наследования


           IMsFindFormulaResults


## Комментарии


Поиск формул осуществляется с помощью метода [IMsMetaModel.FindFormulas](../IMsMetaModel/IMsMetaModel.FindFormulas.htm).
