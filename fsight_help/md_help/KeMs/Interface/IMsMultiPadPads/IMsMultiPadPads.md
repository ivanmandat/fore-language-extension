# IMsMultiPadPads


## Описание


Интерфейс IMsMultiPadPads содержит
 свойства и методы для работы с коллекцией панелей различных компонентов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMsMultiPadPads.Count.htm)
		 Свойство Count возвращает
		 общее количество панелей, которые доступны в компоненте.


		 ![](../../Property_Image.gif)
		 [Item](IMsMultiPadPads.Item.htm)
		 Свойство Item возвращает
		 параметры панели.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](IMsMultiPadPads.FindById.htm)
		 Метод FindById осуществляет
		 поиск панели по идентификатору.
