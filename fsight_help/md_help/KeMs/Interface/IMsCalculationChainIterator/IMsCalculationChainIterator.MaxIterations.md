# IMsCalculationChainIterator.MaxIterations


## Синтаксис


		MaxIterations: Integer;


## Описание


Свойство MaxIterations
 определяет максимальное количество итераций цикла.


## Комментарии


По умолчанию MaxIterations = 500.


Если MaxIterations = -1,
 то цикл выполняется до тех пор, пока не будет выполнено условие, заданное
 в свойстве [IMsCalculationChainIterator.Expression](IMsCalculationChainIterator.Expression.htm).
