# IMsCalculationChainIterator.PostCondition


## Синтаксис


		PostCondition: Boolean;


## Описание


Свойство PostCondition
 определяет признак расчёта цикла в первый раз без проверки условия выполнения
 цикла.


## Комментарии


Допустимые значения:


	- True. Первый раз цикл
	 рассчитывается без проверки условия;


	- False. Первый раз цикл
	 рассчитывается при выполнении условия.


Условие расчета цикла возвращает свойство [IMsCalculationChainIterator.Expression](IMsCalculationChainIterator.Expression.htm).
