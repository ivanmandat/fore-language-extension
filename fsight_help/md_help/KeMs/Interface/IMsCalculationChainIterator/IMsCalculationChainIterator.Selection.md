# IMsCalculationChainIterator.Selection


## Синтаксис


		Selection: [IDimSelection](KeDims.chm::/interface/IDimSelection/IDimSelection.htm);


## Описание


Свойство Selection
 определяет отметку, по которой будет рассчитываться цикл.


## Комментарии


Если определено значение данного свойства,
 то свойство [IMsCalculationChainIterator.ElementGroup](IMsCalculationChainIterator.ElementGroup.htm)
 использовать не следует.
