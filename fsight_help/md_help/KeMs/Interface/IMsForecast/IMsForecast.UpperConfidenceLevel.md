# IMsForecast.UpperConfidenceLevel


## Синтаксис


		UpperConfidenceLevel: [ITimeSeries](../ITimeSeries/ITimeSeries.htm);


## Описание


Свойство UpperConfidenceLevel
 возвращает верхнюю доверительную границу прогнозного ряда.


## Комментарии


Для получения нижней доверительной границы прогнозного ряда используйте
 свойство [IMsForecast.LowerConfidenceLevel](IMsForecast.LowerConfidenceLevel.htm).
