# IMsForecast.Value


## Синтаксис


		Value: [ITimeSeries](../ITimeSeries/ITimeSeries.htm);


## Описание


Свойство Value возвращает
 значения прогнозного ряда.


## Комментарии


Для получения ряда, состоящего из исходного ряда на период идентификации
 и прогнозного ряда на период прогнозирования используйте свойство [IMsForecast.Combined](IMsForecast.Combined.htm).
