# IMsForecast.LowerConfidenceLevel


## Синтаксис


		LowerConfidenceLevel: [ITimeSeries](../ITimeSeries/ITimeSeries.htm);


## Описание


Свойство LowerConfidenceLevel
 возвращает нижнюю доверительную границу прогнозного ряда.


## Комментарии


Для получения верхней доверительной границы прогнозного ряда используйте
 свойство [IMsForecast.UpperConfidenceLevel](IMsForecast.UpperConfidenceLevel.htm).
