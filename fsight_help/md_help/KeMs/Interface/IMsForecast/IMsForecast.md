# IMsForecast


Сборка: Ms;


## Описание


Интерфейс IMsForecast
 предназначен для работы с параметрами прогнозного ряда.


## Иерархия наследования


           IMsForecast


## Комментарии


Для получения прогнозного ряда используйте свойство [IMsForecastObject.Forecast](../IMsForecastObject/IMsForecastObject.Forecast.htm).
