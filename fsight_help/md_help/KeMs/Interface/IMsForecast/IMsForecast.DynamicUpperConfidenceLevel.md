# IMsForecast.DynamicUpperConfidenceLevel


## Синтаксис


		DynamicUpperConfidenceLevel: [ITimeSeries](../ITimeSeries/ITimeSeries.htm);


## Описание


Свойство DynamicUpperConfidenceLevel
 возвращает верхнюю динамическую доверительную границу прогнозного ряда.


## Комментарии


Для получения нижней динамической доверительной границы прогнозного
 ряда используйте свойство [IMsForecast.DynamicLowerConfidenceLevel](IMsForecast.DynamicLowerConfidenceLevel.htm).
