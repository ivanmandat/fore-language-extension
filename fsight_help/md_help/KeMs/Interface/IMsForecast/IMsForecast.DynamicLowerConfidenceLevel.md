# IMsForecast.DynamicLowerConfidenceLevel


## Синтаксис


		DynamicLowerConfidenceLevel: [ITimeSeries](../ITimeSeries/ITimeSeries.htm);


## Описание


Свойство DynamicLowerConfidenceLevel
 возвращает нижнюю динамическую доверительную границу прогнозного ряда.


## Комментарии


Для получения верхней динамической доверительной границы прогнозного
 ряда используйте свойство [IMsForecast.DynamicUpperConfidenceLevel](IMsForecast.DynamicUpperConfidenceLevel.htm).
