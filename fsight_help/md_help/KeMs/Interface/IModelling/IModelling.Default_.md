# IModelling.Default_


## Синтаксис


		Default_: Variant;


## Описание


Метод Default_ возвращает значение
 Null.


## Комментарии


Метод можно использовать для задания пустых значений параметров для
 методов интерфейса [IModelling](IModelling.htm).

Использование метода приведено в примере для [IModelling.Combine](IModelling.Combine.htm).
