# IModelling.None


## Синтаксис


		None: Variant;


## Описание


Метод None возвращает признак
 неиспользования константы.


## Комментарии


None используется в следующих
 методах:


	- [IModelling.Arima](IModelling.Arima.htm);


	- [IModelling.Ols](IModelling.Ols.htm);


	- [IModelling.Tsls](IModelling.Tsls.htm);


	- [IModelling.Expsmooth](IModelling.Expsmooth.htm). В этом случае константа равна нулю.
