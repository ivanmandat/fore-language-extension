# IMsTargetVarConstraint.UseLowerBound


## Синтаксис


UseLowerBound: Boolean;


## Описание


Свойство UseLowerBound определяет,
 используется ли ограничение снизу.


## Комментарии


Допустимые значения:


	- True. Ограничение используется;


	- False. Ограничение не
	 используется.
