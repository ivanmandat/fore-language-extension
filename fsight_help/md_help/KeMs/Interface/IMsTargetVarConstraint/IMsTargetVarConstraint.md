# IMsTargetVarConstraint


Сборка: Ms;


## Описание


Интерфейс IMsTargetVarConstraint
 предназначен для работы с ограничением переменной.


## Иерархия наследования


           [IMsTargetConstraintCommon](../IMsTargetConstraintCommon/IMsTargetConstraintCommon.htm)


           IMsTargetVarConstraint


## Комментарии


Для работы с коллекцией ограничений используйте интерфейс [IMsTargetVarConstraints](../IMsTargetVarConstraints/IMsTargetVarConstraints.htm).
