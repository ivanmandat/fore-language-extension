# IMsTargetVarConstraint.UseUpperBound


## Синтаксис


UseUpperBound: Boolean;


## Описание


Свойство UseUpperBound
 определяет, используется ли ограничение сверху.


## Комментарии


Допустимые значения:


	- True. Ограничение используется;


	- False. Ограничение не
	 используется.
