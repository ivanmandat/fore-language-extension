# IMsArimaTransform.ConstantValue


## Синтаксис


		ConstantValue: Double;


## Описание


Свойство ConstantValue определяет
 значение константы.


## Комментарии


Свойство актуально, если свойству [IMsArimaTransform.ConstantMode](IMsArimaTransform.ConstantMode.htm)
 установлено значение ManualEstimate.
