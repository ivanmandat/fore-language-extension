# IMsCensus2Transform.D12Term


## Синтаксис


		D12Term: [IMsFormulaTermInfo](../IMsFormulaTermInfo/IMsFormulaTermInfo.htm);


## Описание


Свойство D12Term
 определяет терм, в который будет выгружаться тренд-циклическая компонента
 ряда при расчёте задачи моделирования.


## Комментарии


Указанный терм должен использоваться только для выгрузки одного результирующего
 ряда.
