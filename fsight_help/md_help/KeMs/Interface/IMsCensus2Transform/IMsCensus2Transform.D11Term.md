# IMsCensus2Transform.D11Term


## Синтаксис


		D11Term: [IMsFormulaTermInfo](../IMsFormulaTermInfo/IMsFormulaTermInfo.htm);


## Описание


Свойство D11Term
 определяет терм, в который будет выгружаться сезонная корректировка ряда
 при расчёте задачи моделирования.


## Комментарии


Указанный терм должен использоваться только для выгрузки одного результирующего
 ряда.
