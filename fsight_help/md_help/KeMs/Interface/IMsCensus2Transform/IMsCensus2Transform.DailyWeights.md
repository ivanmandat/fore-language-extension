# IMsCensus2Transform.DailyWeights


## Синтаксис


		DailyWeights: Array;


## Описание


Свойство DailyWeights определяет
 весовые коэффициенты дней недели.


## Комментарии


Весовые коэффициенты дней недели учитываются только при расчёте на месячной
 динамике с мультипликативной сезонностью.


Месяцы содержат различное число рабочих дней, это может повлиять на
 окончательный результат анализа временного ряда. Метод X11 даёт возможность
 учесть это обстоятельство введением весовых коэффициентов для дней недели.
 Коэффициенты в сумме должны дать 7. По умолчанию все веса равны 1.


Нормирование весовых коэффициентов дней недели - это изменение их таким
 образом, чтобы в сумме они составляли 7. При расчёте нормирование выполняется
 автоматически, а вызов метода [IMsCensus2Transform.NormalizeDailyWeights](IMsCensus2Transform.NormalizeDailyWeights.htm)
 позволяет выполнить нормирование и получить значения весовых коэффициентов
 дней недели еще до расчёта метода.


Метод X11 поддерживается только под ОС Windows.
