# IMsCensus2Transform.SlidingSpans


## Синтаксис


		SlidingSpans: Boolean;


## Описание


Свойство SlidingSpans определяет
 признак применения скользящих интервалов при расчёте.


## Комментарии


Допустимые значения:


	- True. Скользящие интервалы
	 применяются при расчёте;


	- False. По умолчанию.
	 Скользящие интервалы не применяются при расчёте.
