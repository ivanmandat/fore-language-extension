# IMsCensus2Transform.MovingAverage


## Синтаксис


		MovingAverage: Array;


## Описание


Свойство MovingAverage определяет
 скользящее среднее.


## Комментарии


Размер массива зависит от динамики расчёта формулы [IMsFormula.Level](../IMsFormula/IMsFormula.Level.htm).
 Если выбран шаг расчёта «Кварталы» (Quarter), то размер массива будет
 равен 4, во всех остальных случаях - 12.
