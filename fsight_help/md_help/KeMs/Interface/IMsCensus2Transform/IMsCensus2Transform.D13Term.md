# IMsCensus2Transform.D13Term


## Синтаксис


		D13Term: [IMsFormulaTermInfo](../IMsFormulaTermInfo/IMsFormulaTermInfo.htm);


## Описание


Свойство D13Term
 определяет терм, в который будет выгружаться нерегулярная компонента ряда
 при расчёте задачи моделирования.


## Комментарии


Указанный терм должен использоваться только для выгрузки одного результирующего
 ряда.
