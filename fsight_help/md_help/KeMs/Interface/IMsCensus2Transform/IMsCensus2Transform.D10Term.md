# IMsCensus2Transform.D10Term


## Синтаксис


		D10Term: [IMsFormulaTermInfo](../IMsFormulaTermInfo/IMsFormulaTermInfo.htm);


## Описание


Свойство D10Term определяет
 терм, в который будет выгружаться сезонная составляющая исходного ряда
 при расчёте задачи моделирования.


## Комментарии


Указанный терм должен использоваться только для выгрузки одного результирующего
 ряда.
