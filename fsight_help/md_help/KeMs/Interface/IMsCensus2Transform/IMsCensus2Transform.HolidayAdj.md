# IMsCensus2Transform.HolidayAdj


## Синтаксис


		HolidayAdj: [Census2HolidayAdjType](StatLib.chm::/Enums/Census2HolidayAdjType.htm);


## Описание


Свойство HolidayAdj определяет
 тип поправки на праздничные дни при расчёте.


## Комментарии


По умолчанию поправка на праздничные дни не производится, то есть IMsCensus2Transform.HolidayAdj принимает
 значение [Census2HolidayAdjType.C2hatNone](StatLib.chm::/Enums/Census2HolidayAdjType.htm).
