# IMsCensus2Transform.B1Term


## Синтаксис


		B1Term: [IMsFormulaTermInfo](../IMsFormulaTermInfo/IMsFormulaTermInfo.htm);


## Описание


Свойство B1Term
 определяет терм, в который будет выгружаться ряд после априорной корректировки
 или исходный ряд при расчёте задачи моделирования.


## Комментарии


Указанный терм должен использоваться только для выгрузки одного результирующего
 ряда.
