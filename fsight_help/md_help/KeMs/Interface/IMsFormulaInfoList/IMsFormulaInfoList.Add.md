# IMsFormulaInfoList.Add


## Синтаксис


		Add(FormulaInfo: [IMsFormulaInfo](../IMsFormulaInfo/IMsFormulaInfo.htm));


## Параметры


FormulaInfo. Информация о формуле,
 которая использовалась при расчёте.


## Описание


Метод Add добавляет в коллекцию
 запись с информацией о формуле, которая использовалась при расчёте.
