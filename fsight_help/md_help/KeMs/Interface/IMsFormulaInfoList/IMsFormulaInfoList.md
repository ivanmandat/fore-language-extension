# IMsFormulaInfoList


Сборка: Ms;


## Описание


Интерфейс IMsFormulaInfoList
 предназначен для работы с коллекцией, содержащей информацию о формулах,
 которые использовались для расчёта.


## Иерархия наследования


           IMsFormulaInfoList


## Комментарии


Коллекцию можно получить после расчёта, используя метод [IMsProblemCalculation.GetFormulaInfo](../IMsProblemCalculation/IMsProblemCalculation.GetFormulaInfo.htm).
