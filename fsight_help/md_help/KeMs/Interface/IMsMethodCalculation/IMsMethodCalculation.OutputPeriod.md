# IMsMethodCalculation.OutputPeriod


## Синтаксис


OutputPeriod: [IMsPeriod](../IMsPeriod/IMsPeriod.htm);


## Описание


Свойство OutputPeriod
 определяет параметры периода выгрузки данных.


## Комментарии


Для настройки параметров идентификации и прогнозирования модели используйте
 свойство [IMsMethodCalculation.Period](IMsMethodCalculation.Period.htm).
