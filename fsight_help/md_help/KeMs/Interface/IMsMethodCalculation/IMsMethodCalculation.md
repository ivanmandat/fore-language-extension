# IMsMethodCalculation


Сборка: Ms;


## Описание


Интерфейс IMsMethodCalculation
 содержит свойства, определяющие параметры, необходимые при расчете модели
 с помощью выбранного метода.


## Иерархия наследования


           IMsMethodCalculation


## Комментарии


Для получения параметров расчета модели используйте метод [IMsFormulaTransform.CreateCalculation](../IMsFormulaTransform/IMsFormulaTransform.CreateCalculation.htm).
