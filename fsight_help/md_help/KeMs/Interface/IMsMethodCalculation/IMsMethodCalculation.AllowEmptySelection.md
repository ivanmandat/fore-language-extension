# IMsMethodCalculation.AllowEmptySelection


## Синтаксис


AllowEmptySelection: Boolean;


## Описание


Свойство AllowEmptySelection
 возвращает признак того, допустима ли пустая отметка в измерениях моделируемой
 переменной.


## Комментарии


Допустимые значения:


	- True. В измерениях моделируемой
	 переменной допускается пустая отметка;


	- False. В измерениях
	 моделируемой переменной пустая отметка не допускается.


Если расчет выполняется в инструменте «Моделирование
 и прогнозирование», то по умолчанию свойство AllowEmptySelection
 имеет значение False, если в аналитической
 области данных - True.
