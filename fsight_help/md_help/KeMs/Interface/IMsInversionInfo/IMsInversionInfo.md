# IMsInversionInfo


Сборка: Ms;


## Описание


Интерфейс IMsInversionInfo предназначен для настройки параметров начального преобразования, применяемого к переменной.


## Иерархия наследования


           [ITsInversionInfo](KeTransform.chm::/Interface/ITsInversionInfo/ITsInversionInfo.htm)


           IMsInversionInfo


## Комментарии


Начальное преобразование позволяет скорректировать значения переменной.
