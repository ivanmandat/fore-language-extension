# IMsX11Result


Сборка: Ms;


## Описание


Интерфейс IMsX11Result
 предназначен для работы с результатами расчета метода «X11».


## Иерархия наследования


           [IMsResultObject](../IMsResultObject/IMsResultObject.htm)


           IMsX11Result


## Комментарии


Данный интерфейс используется в пользовательских функциях и методах,
 предназначенных для расчета в детерминированном уравнении.


Метод X11 поддерживается только в ОС Windows.
