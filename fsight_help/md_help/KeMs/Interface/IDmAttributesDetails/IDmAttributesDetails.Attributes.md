# IDmAttributesDetails.Attributes


## Синтаксис


		Attributes: Array;


## Описание


Свойство Attributes определяет
 индексы полей источника данных, содержащего факторы для анализа.


## Комментарии


Индексация полей сквозная и начинается с нуля.


Источник данных для интеллектуального анализа определяет свойство [IDmMethod.InputDataSource](../IDmMethod/IDmMethod.InputDataSource.htm).


## Пример


Использование свойства приведено в примере для [IDmMethod.Execute](../IDmMethod/IDmMethod.Execute.htm).
