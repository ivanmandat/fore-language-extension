# IDmAttributesDetails


Сборка: Ms;


## Описание


Интерфейс IDmAttributesDetails
 предназначен для работы с факторами, используемыми в интеллектуальном
 анализе данных.


## Иерархия наследования


           [IDmMethodDetails](../IDmMethodDetails/IDmMethodDetails.htm)


           IDmAttributesDetails


## Комментарии


Для доступа к работе с факторами приведите объект, возвращаемый свойством
 [IDmMethod.Details](../IDmMethod/IDmMethod.Details.htm)
 к интерфейсу IDmAttributesDetails.
