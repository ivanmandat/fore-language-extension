# IMsCalculationState


Сборка: Ms;


## Описание


Интерфейс IMsCalculationState
 предназначен для работы с информацией о текущем шаге расчёта задачи моделирования.


## Иерархия наследования


           IMsCalculationState


## Комментарии


Для получения информации о текущем шаге расчёта задачи моделирования
 используйте свойство [IMsProblemCalculation.CalculationState](../IMsProblemCalculation/IMsProblemCalculation.CalculationState.htm).
