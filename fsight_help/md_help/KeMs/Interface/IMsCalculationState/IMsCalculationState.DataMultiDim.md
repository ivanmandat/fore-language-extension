# IMsCalculationState.DataMultiDim


## Синтаксис


		DataMultiDim(Slice: [IMsFormulaTransformSlice](../IMsFormulaTransformSlice/IMsFormulaTransformSlice.htm)):
		 [IMatrixDataSource](KeMatrix.chm::/Interface/IMatrixDataSource/IMatrixDataSource.htm);


## Параметры


Slice. Срез переменной, данные
 которого необходимо получить.


## Описание


Свойство DataMultiDim возвращает
 данные текущего шага расчёта с учётом используемых сценариев моделирования.


## Комментарии


Для получения ключа текущего шага расчёта используйте свойство [IMsCalculationState.Scenario](IMsCalculationState.Scenario.htm).
