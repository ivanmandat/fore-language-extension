# IMsTransformSeries.UpperConfidenceLevel


## Синтаксис


UpperConfidenceLevel: Array;


## Описание


Свойство UpperConfidenceLevel
 возвращает массив, содержащий значения верхней доверительной границы.


## Комментарии


Для получения значений нижней доверительной границы используйте свойство
 [IMsTransformSeries.LowerConfidenceLevel](IMsTransformSeries.LowerConfidenceLevel.htm).
