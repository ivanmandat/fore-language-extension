# IMsTransformSeries


Сборка: Ms;


## Описание


Интерфейс IMsTransformSeries
 предназначен для работы с данными по основным рядам модели.


## Иерархия наследования


           IMsTransformSeries


## Комментарии


Значения рядов доступны после расчета модели.
