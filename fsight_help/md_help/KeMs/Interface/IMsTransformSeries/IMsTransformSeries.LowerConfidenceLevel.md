# IMsTransformSeries.LowerConfidenceLevel


## Синтаксис


LowerConfidenceLevel: Array;


## Описание


Свойство LowerConfidenceLevel
 возвращает массив, содержащий значения нижней доверительной границы.


## Комментарии


Для получения значений верхней доверительной границы используйте свойство
 [IMsTransformSeries.UpperConfidenceLevel](IMsTransformSeries.UpperConfidenceLevel.htm).
