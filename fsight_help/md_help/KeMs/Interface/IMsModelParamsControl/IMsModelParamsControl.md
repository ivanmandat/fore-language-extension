# IMsModelParamsControl


## Описание


Интерфейс IMsModelParamsControl
 содержит свойства и методы для работы с коллекцией контролируемых параметров
 метамодели, содержащейся в цепочке расчета.


## Комментарии


Коллекцию параметров можно получить, используя свойства [IMsCalculationChainMetaModel.ParamsControl](../IMsCalculationChainMetaModel/IMsCalculationChainMetaModel.ParamsControl.htm),
 [IValidationFilters.ParamControl](../IValidationFilters/IValidationFilters.ParamControl.htm),
