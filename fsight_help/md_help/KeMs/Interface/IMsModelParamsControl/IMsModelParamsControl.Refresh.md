# IMsModelParamsControl.Refresh


## Синтаксис


Refresh;


## Описание


Метод Refresh обновляет список
 параметров в коллекции.


## Комментарии


Метод может использоваться для обновления коллекции контролируемых параметров
 после обновления коллекции параметров метамодели, описываемой интерфейсом
  [IMsModelParams](../IMsModelParams/IMsModelParams.htm).
