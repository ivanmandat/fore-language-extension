# IDmSelfOrganizingMap


Сборка: Ms;


## Описание


Интерфейс IDmSelfOrganizingMap
 предназначен для кластеризации данных с помощью самоорганизующихся карт
 Кохонена.


## Иерархия наследования

           [IDmAttributesDetails](../IDmAttributesDetails/IDmAttributesDetails.htm)


           [IDmClusteringDetails](../IDmClusteringDetails/IDmClusteringDetails.htm)


           IDmSelfOrganizingMap


## Комментарии


Данный метод группирует объекты с помощью нейронной сети Кохонена. Используется
 обучение без учителя, то есть результат зависит только от структуры входных
 данных. Метод работает на числовых значениях.
