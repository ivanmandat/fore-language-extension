# IMsVariableTestResults.Error


## Синтаксис


Error: String;


## Описание


Свойство Error возвращает ошибки, возникшие при расчёте описательной статистики.


## Комментарии


Если статистика рассчиталась без ошибок, то свойство возвращает пустое значение.
