# IMsVariableTestResults.ResValueMatrix


## Синтаксис


ResValueMatrix: Array;


## Описание


Свойство ResValueMatrix возвращает результаты расчёта описательной статистики для каждой переменной.


## Комментарии


Для получения ошибок, возникших при расчёте используйте свойство [IMsVariableTestResults.Error](IMsVariableTestResults.Error.htm).
