# IMsVariableTestResults


Сборка: Ms;


## Описание


Интерфейс IMsVariableTestResults
 предназначен для работы с результатами расчёта описательной статистики.


## Иерархия наследования


           IMsVariableTestResults


## Комментарии


Для расчёта описательной статистики и получения результатов используйте
 метод [IMsVariableTest.Execute](../IMsVariableTest/IMsVariableTest.Execute.htm).
