# IMsBreakpoint.Key


## Синтаксис


		Key: Integer;


## Описание


Свойство Key
 возвращает ключ точки останова.


## Комментарии


Ключ предназначен для однозначной идентификации точки останова внутри
 коллекции. Значение ключа задаётся автоматически и не может быть изменено.
