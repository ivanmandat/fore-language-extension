# IMsBreakpoint.Name


## Синтаксис


		Name: String;


## Описание


Свойство Name
 определяет наименование точки останова.


## Комментарии


По умолчанию наименование у точки останова отсутствует. Оно не предназначено
 для однозначной идентификации точки останова и поэтому может быть неуникальным.
 Оно может состоять из любой последовательности букв, цифр и символов.
 Желательно, чтобы наименование несло смысловую нагрузку.
