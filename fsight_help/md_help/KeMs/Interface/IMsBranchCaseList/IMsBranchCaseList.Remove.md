# IMsBranchCaseList.Remove


## Синтаксис


		Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемой ветки.


## Описание


Метод Remove
 осуществляет удаление ветки из коллекции по индексу.


## Комментарии


В случае успешного удаления ветки метод
 возвращает значение True.
