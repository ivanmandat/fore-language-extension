# IMsBranchCaseList


Сборка: Ms;


## Описание


Интерфейс IMsBranchCaseList
 предназначен для работы с коллекцией веток условия в [цепочке
 расчёта](UiModelling.chm::/2_Container_of_Modeling/2_3_Work_object/2_3_3_MetaModel/uimodelling_metamodel_2.htm) метамодели.
