# IMsDimOpenStubParamValue


Сборка: Ms;


## Описание


Интерфейс IMsDimOpenStubParamValue
 предназначен для указания параметров измерений одного источника данных,
 которые будут использоваться при открытии измерений в объекте моделирования.
