# IMsPointwiseCollapseTransform.MethodType


## Синтаксис


MethodType: [MsCollapseType](../../Enums/MsCollapseType.htm);


## Описание


Свойство MethodType определяет
 метод агрегации, который необходимо применить к моделируемой переменной.


## Комментарии


По умолчанию используется агрегация выполняется путем суммирования значений
 исходной динамики, то есть MethodType = MsCollapseType.Total.
