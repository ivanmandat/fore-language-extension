# IMsCalculationStatsDiffList


Сборка: Ms;


## Описание


Интерфейс IMsCalculationStatsDiffList
 предназначен для работы с коллекцией, содержащей различающиеся статистики.


## Иерархия наследования


           IMsCalculationStatsDiffList


## Комментарии


Коллекция может быть получена в результате сравнения статистик выполнения
 задачи моделирования. Сравнение производит метод [IMsCalculationStats.Compare](../IMsCalculationStats/IMsCalculationStats.Compare.htm).
