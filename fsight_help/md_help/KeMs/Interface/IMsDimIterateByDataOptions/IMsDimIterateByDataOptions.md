# IMsDimIterateByDataOptions


Сборка: Ms;


## Описание


Интерфейс IMsDimIterateByDataOptions
 предназначен для настройки параметров итерации по данным.


## Иерархия наследования

## Комментарии


Для получения параметров итерации по данным используйте свойство [IMsCalculationChainMultiDimIterator.IterateByData](../IMsCalculationChainMultiDimIterator/IMsCalculationChainMultiDimIterator.IterateByData.htm).
