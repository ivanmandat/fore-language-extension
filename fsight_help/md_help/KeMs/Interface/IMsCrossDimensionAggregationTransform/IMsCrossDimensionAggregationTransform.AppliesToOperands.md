# IMsCrossDimensionAggregationTransform.AppliesToOperands


## Синтаксис


		AppliesToOperands: [IMsFormulaTermList](../IMsFormulaTermList/IMsFormulaTermList.htm);


## Описание


Свойство AppliesToOperands возвращает
 коллекцию элементов выражения для задания компонентов, которые будут агрегироваться
 без учёта обработки пропусков.


## Комментарии


Выражение для задания компонентов, которые будут агрегироваться без
 учёта обработки пропусков, определяет свойство [IMsCrossDimensionAggregationTransform.AppliesToExpression](IMsCrossDimensionAggregationTransform.AppliesToExpression.htm).
