# IMsCrossDimensionAggregationTransform.WeightsExpression


## Синтаксис


		WeightsExpression: [IExpression](ForeSys.chm::/Interface/IExpression/IExpression.htm);


## Описание


Свойство WeightsExpression определяет
 выражение для расчёта весов агрегации.


## Комментарии


Значение WeightsExpression учитывается,
 если выбран [метод взвешенной
 агрегации](../../Enums/MsAgregationMethodType.htm) (например, взвешенная сумма).


Коллекцию элементов, которые могут быть использованы в выражении, возвращает
 свойство [IMsCrossDimensionAggregationTransform.WeightsOperands](IMsCrossDimensionAggregationTransform.WeightsOperands.htm).
