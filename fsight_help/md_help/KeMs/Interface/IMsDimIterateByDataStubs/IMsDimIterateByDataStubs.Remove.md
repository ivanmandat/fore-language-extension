# IMsDimIterateByDataStubs.Remove


## Синтаксис


		Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемого источника
 данных.


## Описание


Метод Remove удаляет источник
 данных с указанным индексом.


## Комментарии


При успешном удалении источника данных метод возвращает значение True, при неуспешном - False.
