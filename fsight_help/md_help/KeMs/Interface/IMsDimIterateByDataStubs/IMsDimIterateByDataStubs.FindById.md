# IMsDimIterateByDataStubs.FindById


## Синтаксис


		FindById(Id: String): [IMsDimIterateByDataStub](../IMsDimIterateByDataStub/IMsDimIterateByDataStub.htm);


## Параметры


Id. Идентификатор искомого
 источника данных.


## Описание


Метод FindById выполняет
 поиск источника данных с заданным идентификатором.


## Комментарии


Если источник данных с заданным идентификатором не найден, то метод
 возвращает значение Null.
