# IMsDimIterateByDataStubs


Сборка: Ms;


## Описание


Интерфейс IMsDimIterateByDataStubs
 предназначен для работы с коллекцией источников данных, по существующим
 точкам которых будут выполняться итерации.


## Иерархия наследования


           IMsDimIterateByDataStubs


## Комментарии


Для получения коллекции источников используйте свойство [IMsDimIterateByDataOptions.Stubs](../IMsDimIterateByDataOptions/IMsDimIterateByDataOptions.Stubs.htm).
