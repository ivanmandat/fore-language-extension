# IMsDimIterateByDataStubs.RemoveByKey


## Синтаксис


		RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ удаляемого источника
 данных.


## Описание


Метод RemoveByKey удаляет
 источник данных с указанным ключом.


## Комментарии


При успешном удалении источника данных метод возвращает значение True, при неуспешном - False.
