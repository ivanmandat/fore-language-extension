# IMsFormulaTransformSlice.Level


## Синтаксис


Level: [DimCalendarLevel](KeDims.chm::/Enums/DimCalendarLevel.htm);


## Описание


Свойство Level определяет календарную
 динамику среза переменной.


## Комментарии


Если динамика среза не задана, то она берется из динамики формулы: [IMsFormula.Level](../IMsFormula/IMsFormula.Level.htm).


Календарная динамика важна при расчете методов агрегации/дезагрегации
 данных.
