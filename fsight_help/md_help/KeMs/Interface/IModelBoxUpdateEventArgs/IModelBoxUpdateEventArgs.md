## Описание


Интерфейс IModelBoxUpdateEventArgs
 содержит свойства аргумента события, возникающего при изменении параметров
 модели, отображаемой в компоненте [ModelBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/ModelBox.htm). Интерфейс
 зарезервирован на будущее.
