# IMsGlobalSettings


## Описание


Интерфейс IMsGlobalSettings
 предназначен для работы с глобальными настройками контейнера моделирования.


## Свойства, унаследованные от [IMsDisplaySettings](../IMsDisplaySettings/IMsDisplaySettings.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DisplayAttributes](../IMsDisplaySettings/IMsDisplaySettings.DisplayAttributes.htm)
		 Свойство DisplayAttributes
		 возвращает коллекцию настроек отображения атрибутов в строковом
		 представлении элементов выражения.


		 ![](../../Property_Image.gif)
		 [DisplayParentAs](../IMsDisplaySettings/IMsDisplaySettings.DisplayParentAs.htm)
		 Свойство DisplayParentAs
		 определяет режим отображения родительского элемента в строковом
		 представлении элемента выражения.


		 ![](../../Property_Image.gif)
		 [DisplayTermsAs](../IMsDisplaySettings/IMsDisplaySettings.DisplayTermsAs.htm)
		 Свойство DisplayTermsAs
		 определяет параметры отображения объектов.
