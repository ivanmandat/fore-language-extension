# IMsMethod.SupportsR


## Синтаксис


SupportsR: Boolean;


## Описание


Свойство SupportsR определяет,
 возможен ли расчет метода с помощью R.


## Комментарии


Допустимые значения:


	- True. Метод может быть
	 рассчитан с помощью R. Для этого используйте свойство [IMsMethod.UseR](IMsMethod.UseR.htm);


	- False. Метод не может
	 быть рассчитан с помощью R.
