# IMsStubSelection.Add


## Синтаксис


Add(Value: [IDimSelectionSet](KeDims.chm::/interface/IDimSelectionSet/IDimSelectionSet.htm);
 DataType: [CubeDataType](KeCubes.chm::/Enums/CubeDataType.htm));


## Параметры


Value. Отметка источника, которая
 формирует множество точек.


DataType. Способ формирования
 данных, которым будут отмечены ячейки.


## Описание


Метод Add добавляет указанную
 отметку источника, формируя множество точек.


## Комментарии


В дальнейшем, при использовании фильтра, будут обрабатываться только
 те точки, которые сформированы переданной отметкой.
