# IMsProblemScenarios.RemoveByKey


## Синтаксис


RemoveByKey(ScenarioKey: Integer): Boolean;


## Параметры


ScenarioKey. Ключ сценария, который необходимо удалить.


## Описание


Метод RemoveByKey удаляет сценарий из коллекции сценариев задачи моделирования. Ключ сценария передается посредством параметра ScenarioKey.
