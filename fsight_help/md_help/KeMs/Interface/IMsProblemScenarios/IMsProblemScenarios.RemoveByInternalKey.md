# IMsProblemScenarios.RemoveByInternalKey


## Синтаксис


RemoveByKey(InrernalKey: Integer): Boolean;


## Параметры


InrernalKey. Внутренний ключ сценария моделирования.


## Описание


Метод RemoveByInternalKey удаляет сценарий моделирования по его внутреннему ключу.


## Комментарии


Если удаление выполнено успешно, то метод возвращает значение True, в обратном случае - False.


Для получения внутреннего ключа сценария моделирования используйте свойство [IMsScenario.InternalKey](../IMsScenario/IMsScenario.InternalKey.htm).
