# IMsProblemScenarios


Сборка: Ms;


## Описание


Интерфейс IMsProblemScenarios
 содержит свойства и методы для работы с коллекцией сценариев задачи моделирования.


## Иерархия наследования


           IMsProblemScenarios


## Комментарии


Для получения коллекции сценариев задачи моделирования используйте свойство
 [IMsProblem.Scenarios](../IMsProblem/IMsProblem.Scenarios.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMsProblemScenarios.Count.htm)
		 Свойство Count возвращает
		 количество сценариев, по которым может осуществляться расчет задачи.


		 ![](../../Property_Image.gif)
		 [Item](IMsProblemScenarios.Item.htm)
		 Свойство Item возвращает
		 сценарий задачи моделирования.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddScenario](IMsProblemScenarios.AddScenario.htm)

		 ![](../../Sub_Image.gif)
		 [Clear](IMsProblemScenarios.Clear.htm)
		 Метод Clear очищает

		 ![](../../Sub_Image.gif)
		 [CopyScenarioData](IMsProblemScenarios.CopyScenarioData.htm)
		 Метод CopyScenarioData
		 копирует данные по сценариям моделирования.


		 ![](../../Sub_Image.gif)
		 [FindByInternalKey](IMsProblemScenarios.FindByInternalKey.htm)

		 ключу.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IMsProblemScenarios.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск сценария в коллекции сценариев задачи моделирования.


		 ![](../../Sub_Image.gif)
		 [MoveTo](IMsProblemScenarios.MoveTo.htm)
		 Метод MoveTo осуществляет
		 перемещение сценариев в списке.


		 ![](../../Sub_Image.gif)
		 [Remove](IMsProblemScenarios.Remove.htm)
		 Метод Remove удаляет

		 ![](../../Sub_Image.gif)
		 [RemoveByInternalKey](IMsProblemScenarios.RemoveByInternalKey.htm)
		 Метод RemoveByInternalKey
		 удаляет сценарий моделирования по его внутреннему ключу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IMsProblemScenarios.RemoveByKey.htm)
		 Метод RemoveByKey удаляет
		 сценарий из коллекции по ключу.
