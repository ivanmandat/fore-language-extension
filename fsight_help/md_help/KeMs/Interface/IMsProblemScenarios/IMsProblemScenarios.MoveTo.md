# IMsProblemScenarios.MoveTo


## Синтаксис


MoveTo(FromIndex: Integer; ToIndex: Integer);


## Параметры


FromIndex. Индекс позиции сценария, который необходимо переместить.


ToIndex. Индекс позиции, в которую необходимо переместить.


## Описание


Метод MoveTo осуществляет перемещение сценариев в списке.
