# IMsProblemScenarios.Item


## Синтаксис


Item(Index: Integer): [IMsScenario](../IMsScenario/IMsScenario.htm);


## Параметры


Index. Индекс сценария.


## Описание


Свойство Item возвращает сценарий, по которому может осуществляться расчет задачи. Индекс сценария передается посредством параметра Index.
