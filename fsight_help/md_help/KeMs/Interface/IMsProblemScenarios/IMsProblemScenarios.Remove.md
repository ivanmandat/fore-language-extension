# IMsProblemScenarios.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс сценария, который необходимо удалить.


## Описание


Метод Remove удаляет сценарий из коллекции сценариев задачи моделирования. Индекс сценария передается посредством параметра Index.
