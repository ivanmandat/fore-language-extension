# IValidationDetails.SelectionTargetData


## Синтаксис


SelectionTargetData: [IValidationSelectionValueList](../IValidationSelectionValueList/IValidationSelectionValueList.htm);


## Описание


Свойство SelectionTargetData
 возвращает объект, определяющий данные, по которым будет выполняться валидация.


## Комментарии


Если значение свойства не задано, то валидация выполняется по всей базе
 данных временных рядов.
