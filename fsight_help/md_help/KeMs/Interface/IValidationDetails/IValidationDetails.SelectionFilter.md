# IValidationDetails.SelectionFilter


## Синтаксис


SelectionFilter: [IValidationSelectionFilter](../IValidationSelectionFilter/IValidationSelectionFilter.htm);


## Описание


Свойство SelectionFilter возвращает
 параметры фильтра, ограничивающего проверяемые ряды.


## Комментарии


Фильтр представляет собой набор отметок по измерениям источника данных
 рядов. Каждый элемент фильтра задаёт отметку одного измерения.


По умолчанию фильтр не задан и валидация выполняется по всем рядам,
 заданным в [IValidationDetails.SelectionTargetData](IValidationDetails.SelectionTargetData.htm).
