# IMsValueAtRiskBacktesting


Сборка: Ms;


## Описание


Интерфейс IMsValueAtRiskBacktesting
 предназначен для работы с параметрами бэктестинга.


## Иерархия наследования


           IMsValueAtRiskBacktesting


## Комментарии


Бэктестинг используется в модели Value-At-Risk. Его параметры возвращает
 свойство [IMsValueAtRiskTransform.Backtesting](../IMsValueAtRiskTransform/IMsValueAtRiskTransform.Backtesting.htm).
