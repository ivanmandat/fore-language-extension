# IMsValueAtRiskBacktesting.Observations


## Синтаксис


Observations: Integer;


## Описание


Свойство Observations определяет количество наблюдений.


## Комментарии


Observations учитывается, если [IMsValueAtRiskBacktesting.Enabled](IMsValueAtRiskBacktesting.Enabled.htm) установлено в True.
