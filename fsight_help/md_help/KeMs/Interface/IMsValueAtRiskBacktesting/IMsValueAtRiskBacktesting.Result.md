# IMsValueAtRiskBacktesting.Result


## Синтаксис


Result(Organization: Integer): [IMsValueAtRiskBacktestingResult](../IMsValueAtRiskBacktestingResult/IMsValueAtRiskBacktestingResult.htm);


## Параметры


Organization. Индекс элемента измерения, соответствующий организации.


## Описание


Свойство Result возвращает результаты бэктестинга.


## Комментарии


Измерение организаций определяет свойство [IMsValueAtRiskTransform.OrganizationsDimension](../IMsValueAtRiskTransform/IMsValueAtRiskTransform.OrganizationsDimension.htm).


Результаты могут быть возвращены только после расчёта модели.
