# IMsValueAtRiskBacktesting.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет,
 использовать ли бэктестинг.


## Комментарии


Допустимые значения:


	- True. Использовать бэктестинг;


	- False. Не использовать
	 бэктестинг.
