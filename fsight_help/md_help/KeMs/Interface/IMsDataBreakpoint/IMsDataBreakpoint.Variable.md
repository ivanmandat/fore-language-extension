# IMsDataBreakpoint.Variable


## Синтаксис


		Variable: [IVariableStub](KeCubes.chm::/Interface/IVariableStub/IVariableStub.htm);


## Описание


Свойство Variable определяет
 источник данных.


## Комментарии


После того как определён источник данных, задайте отметку по всем его
 измерениям с помощью свойства [IMsDataBreakpoint.Selection](IMsDataBreakpoint.Selection.htm).
 Таким образом будут заданы точки данных, в которых проверяется условие
 срабатывания точки останова.
