# IMsDataBreakpoint.Selection


## Синтаксис


		Selection: [IDimSelectionSet](KeDims.chm::/interface/IDimSelectionSet/IDimSelectionSet.htm);


## Описание


Свойство Selection определяет
 отметку элементов источника данных.


## Комментарии


Данное свойство задает отметку по всем измерениям источника данных.
 Источник данных определяется свойством [IMsDataBreakpoint.Variable](IMsDataBreakpoint.Variable.htm).


Таким образом будут получены точки данных, в которых проверяется условие
 срабатывания точки останова.
