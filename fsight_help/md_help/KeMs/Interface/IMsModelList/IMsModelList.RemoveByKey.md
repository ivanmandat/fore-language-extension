# IMsModelList.RemoveByKey


## Синтаксис


RemoveByKey(ModelKey: Boolean): Boolean;


## Параметры


ModelKey. Ключ модели.


## Описание


Метод RemoveByKey удаляет модель из коллекции по ключу. Возвращает значение True, если удаление прошло успешно.
