# IMsModelList


## Описание


Интерфейс IMsModelList предназначен
 для работы с коллекцией моделей.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMsModelList.Count.htm)
		 Свойство Count возвращает
		 число моделей в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMsModelList.Item.htm)
		 Свойство Item возвращает
		 модель из коллекции по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IMsModelList.Add.htm)

		 добавление модели в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](IMsModelList.Clear.htm)
		 Метод Clear осуществляет

		 Метод FindByKey осуществляет

		 [RebindContents](IMsModelList.RebindContents.htm)
		 Метод RebindContents

		 [RemoveByKey](IMsModelList.RemoveByKey.htm)
		 Метод RemoveByKey удаляет
		 модель из коллекции по ключу.
