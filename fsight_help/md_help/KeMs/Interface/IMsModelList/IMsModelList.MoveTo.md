# IMsModelList.MoveTo


## Синтаксис


MoveTo(FromIndex: Integer; ToIndex: Integer);


## Параметры


FromIndex. Индекс позиции модели,
 которую необходимо переместить.


ToIndex. Индекс позиции, в
 которую необходимо переместить модель.


## Описание


Метод MoveTo
 осуществляет перемещение модели в коллекции.
