# IMsModelList.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс модели в коллекции.


## Описание


Метод Remove удаляет модель из коллекции по индексу. Возвращает значение True, если удаление прошло успешно.
