# IMsProblemCalculationCallback.OnModelCalculation


## Синтаксис


OnModelCalculation(Model: [IMsModel](../IMsModel/IMsModel.htm));


## Параметры


Model. Модель, которая будет рассчитана.


## Описание


Метод OnModelCalculation реализует событие, происходящее перед началом расчета модели.
