# IMsProblemCalculationCallback.OnError


## Синтаксис


OnError(Message: String);


## Параметры


Message. Сообщение об ошибке.


## Описание


Метод OnError реализует событие, происходящее при возникновении ошибки во время расчета задачи в контейнере моделирования.
