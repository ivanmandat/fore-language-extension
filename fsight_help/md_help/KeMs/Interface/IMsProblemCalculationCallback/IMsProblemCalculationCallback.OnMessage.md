# IMsProblemCalculationCallback.OnMessage


## Синтаксис


OnMessage(Message: String);


## Параметры


Message. Информационное сообщение, генерируемое контейнером моделирования.


## Описание


Метод OnMessage реализует событие, происходящее при инициализации какого-либо шага при расчете задачи в контейнере моделирования.
