# IMsDimIteratorAdvancedFilter.Item


## Синтаксис


		Item(Index: Integer): [IMsDimIteratorAdvancedFilterCondition](../IMsDimIteratorAdvancedFilterCondition/IMsDimIteratorAdvancedFilterCondition.htm);


## Параметры


Index. Индекс условия.


## Описание


Свойство Item возвращает заданное
 условие расширенного фильтра.


## Комментарии


Индексация условий сквозная и начинается с нуля.


Для получения количества условий используйте свойство [IMsDimIteratorAdvancedFilter.Count](IMsDimIteratorAdvancedFilter.Count.htm).
