# IMsDimIteratorAdvancedFilter.Test


## Синтаксис


		Test(Element: Integer; Calculation: [IMsProblemCalculation](../IMsProblemCalculation/IMsProblemCalculation.htm)):
		 Boolean;


## Параметры


Element. Тестируемый элемент
 справочника;


Calculation. Объект, рассчитывающий
 метамодель, содержащую многомерный итератор.


## Описание


Метод Test проверяет элемент
 на соответствие расширенному фильтру.


## Комментарии


Допустимые значения:


	- True. Элемент соответствует
	 фильтру;


	- False. Элемент не соответствует
	 фильтру.
