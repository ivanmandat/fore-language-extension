# IMsJohansenTestSettings.ModelType


## Синтаксис


ModelType: [ECMType](StatLib.chm::/Enums/ECMType.htm);


## Описание


Свойство ModelType определяет используемый тип модели коррекции ошибок.


## Комментарии


Значение по умолчанию ECMType.NoTrendNoIntercept, т.е. модель без тренда в авторегрессии, без константы в коинтеграционном уравнении.
