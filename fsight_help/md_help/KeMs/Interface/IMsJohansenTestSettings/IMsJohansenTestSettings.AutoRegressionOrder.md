# IMsJohansenTestSettings.AutoRegressionOrder


## Синтаксис


AutoRegressionOrder: String;


## Описание


Свойство AutoRegressionOrder
 определяет порядок авторегрессии.


## Комментарии


Значение AutoRegressionOrder
 может содержать номера или диапазоны порядков авторегрессии,
 разделённые запятыми. Например:


AutoRegressionOrder("1-3,5,7-9");
