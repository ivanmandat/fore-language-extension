# IMsJohansenTestSettings.ExogenousSources


## Синтаксис


ExogenousSources: [IMsVariableTestDataList](../IMsVariableTestDataList/IMsVariableTestDataList.htm);


## Описание


Свойство ExogenousSources возвращает коллекцию экзогенных переменных.


## Комментарии


Для добавления переменной в коллекцию используйте метод [IMsMetaModelVisualController.AddJohansenTestExogenousEntrie](../IMsMetaModelVisualController/IMsMetaModelVisualController.AddJohansenTestExogenousEntrie.htm) или [IMsVariableTestDataList.Add](../IMsVariableTestDataList/IMsVariableTestDataList.Add.htm).


Для получения коллекции эндогенных переменных используйте свойство [IMsVariableTestSettings.IncludedSources](../IMsVariableTestSettings/IMsVariableTestSettings.IncludedSources.htm).
