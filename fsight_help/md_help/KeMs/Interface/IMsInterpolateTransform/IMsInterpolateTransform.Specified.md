# IMsInterpolateTransform.Specified


## Синтаксис


Specified: [IMsFormulaTermInfo](../IMsFormulaTermInfo/IMsFormulaTermInfo.htm);


## Описание


Свойство Specified определяет
 образцовый ряд для интерполяции по шаблону.


## Комментарии


Для интерполяции по шаблону необходимо определить вид шаблона с помощью
 свойства [IMsInterpolateTransform.PatternType](IMsInterpolateTransform.PatternType.htm).
 Метод интерполяции определяется свойством [IMsInterpolateTransform.MethodType](IMsInterpolateTransform.MethodType.htm).
