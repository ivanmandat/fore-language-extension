# IMsInterpolateTransform.MethodType


## Синтаксис


MethodType: [MsInterpolateType](../../Enums/MsInterpolateType.htm);


## Описание


Свойство MethodType определяет
 метод интерполяции, который необходимо применить к моделируемой переменной.


## Комментарии


По умолчанию используется повторная интерполяция, то есть MethodType = MsInterpolateType.Repeat.
