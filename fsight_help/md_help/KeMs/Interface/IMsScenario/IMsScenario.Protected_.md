# IMsScenario.Protected_


## Синтаксис


Protected_: Boolean;


## Описание


Свойство Protected_ определяет,
 является ли сценарий защищенным.


## Комментарии


Допустимые значения:


	- True. Сценарий является
	 защищенным;


	- False. Сценарий не является
	 защищенным.
