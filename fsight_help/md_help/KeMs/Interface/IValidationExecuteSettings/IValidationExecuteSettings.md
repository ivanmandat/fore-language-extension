# IValidationExecuteSettings


Сборка: Ms;


## Описание


Интерфейс IValidationExecuteSettings
 определяет настройки расчета валидации.


## Иерархия наследования


           IValidationExecuteSettings


## Комментарии


Для получения настроек расчета валидации используйте свойство [IValidationFilterModel.Settings](Laner.chm::/Interface/IValidationFilterModel/IValidationFilterModel.Settings.htm).
 Интерфейс [IValidationFilter](../IValidationFilter/IValidationFilter.htm)
 является наследником интерфейса [IValidationFilterModel](Laner.chm::/Interface/IValidationFilterModel/IValidationFilterModel.htm).
