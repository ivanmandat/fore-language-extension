# IValidationExecuteSettings.LoadStart


## Синтаксис


LoadStart: DateTime;


## Описание


Свойство LoadStart определяет
 начало периода загрузки данных.


## Комментарии


Для определения окончания периода загрузки данных используйте свойство
 [IValidationExecuteSettings.LoadEnd](IValidationExecuteSettings.LoadEnd.htm).
