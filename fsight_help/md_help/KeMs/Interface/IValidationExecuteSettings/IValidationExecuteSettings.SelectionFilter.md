# IValidationExecuteSettings.SelectionFilter


## Синтаксис


SelectionFilter: [IValidationSelectionFilter](../IValidationSelectionFilter/IValidationSelectionFilter.htm);


## Описание


Свойство SelectionFilter возвращает
 параметры фильтра, ограничивающего ряды, по которым рассчитывается валидация.


## Комментарии


Фильтр представляет собой набор отметок по атрибутам рядов. Каждый элемент
 фильтра задаёт отметку одного атрибута.


Фильтрация позволяет ограничить проверяемые данные и тем самым сократить
 время расчета валидации.
