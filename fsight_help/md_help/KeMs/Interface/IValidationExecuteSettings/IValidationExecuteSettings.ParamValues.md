# IValidationExecuteSettings.ParamValues


## Синтаксис


ParamValues: [IMsModelParamValues](../IMsModelParamValues/IMsModelParamValues.htm);


## Описание


Свойство ParamValues возвращает значения параметров, используемых при расчёте валидации.


## Комментарии


Для настройки параметрической валидации используйте свойство [IValidationFilter.Params](../IValidationFilter/IValidationFilter.Params.htm); для настройки параметрической группы валидаций - [IValidationGroup.Params](../IValidationGroup/IValidationGroup.Params.htm).
