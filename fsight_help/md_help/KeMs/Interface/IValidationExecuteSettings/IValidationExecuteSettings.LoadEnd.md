## Описание


Свойство LoadEnd определяет
 окончание периода загрузки данных.


## Комментарии


Для определения начала периода загрузки данных используйте свойство
 [IValidationExecuteSettings.LoadStart](IValidationExecuteSettings.LoadStart.htm).
