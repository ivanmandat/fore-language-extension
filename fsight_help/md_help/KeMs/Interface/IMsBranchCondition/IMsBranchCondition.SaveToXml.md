# IMsBranchCondition.SaveToXml


## Синтаксис


		SaveToXml(Element: [IXmlDomElement](ModXml.chm::/develop.htm));


## Параметры


Element. Элемент, в который
 будут записаны параметры условия расчёта ветки.


## Описание


Метод SaveToXml выполняет выгрузку
 параметров условия расчёта ветки в XML-файл.


## Комментарии


Для загрузки параметров условия расчета ветки из XML-файла используйте
 метод [IMsBranchCondition.LoadFromXml](IMsBranchCondition.LoadFromXml.htm).
 Набор методов SaveToXml и [IMsBranchCondition.LoadFromXml](IMsBranchCondition.LoadFromXml.htm)
 предназначен для копирования параметров условия расчёта ветки.
