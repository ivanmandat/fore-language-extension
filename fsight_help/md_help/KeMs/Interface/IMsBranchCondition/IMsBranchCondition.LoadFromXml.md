# IMsBranchCondition.LoadFromXml


## Синтаксис


		LoadFromXml(Element: [IXmlDomElement](ModXml.chm::/develop.htm));


## Параметры


Element. Элемент, из которого
 будут загружены параметры условия расчёта ветки.


## Описание


Метод LoadFromXml выполняет
 загрузку параметров условия расчёта ветки из XML-файла.


## Комментарии


Для выгрузки параметров условия расчета ветки в XML-файл используйте
 метод [IMsBranchCondition.SaveToXml](IMsBranchCondition.SaveToXml.htm).
 Набор методов [IMsBranchCondition.SaveToXml](IMsBranchCondition.SaveToXml.htm)
 и LoadFromXml предназначен
 для копирования параметров условия расчёта ветки.
