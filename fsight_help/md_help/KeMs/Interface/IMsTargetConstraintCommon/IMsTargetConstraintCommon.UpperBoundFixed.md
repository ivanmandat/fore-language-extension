# IMsTargetConstraintCommon.UpperBoundFixed


## Синтаксис


UpperBoundFixed: Boolean;


## Описание


Свойство UpperBoundFixed определяет,
 зафиксирована ли верхняя граница ограничения.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Верхняя граница ограничения зафиксирована и она не может быть изменена
	 при автоматическом подборе;


	- False. Верхняя граница
	 ограничения ослаблена.


Для определения, зафиксирована ли нижняя граница ограничения, используйте
 свойство [IMsTargetConstraintCommon.LowerBoundFixed](IMsTargetConstraintCommon.LowerBoundFixed.htm).
