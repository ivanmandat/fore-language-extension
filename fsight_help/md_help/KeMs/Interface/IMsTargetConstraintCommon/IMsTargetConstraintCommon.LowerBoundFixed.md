# IMsTargetConstraintCommon.LowerBoundFixed


## Синтаксис


LowerBoundFixed: Boolean;


## Описание


Свойство LowerBoundFixed определяет,
 зафиксирована ли нижняя граница ограничения.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Нижняя граница ограничения зафиксирована и она не может быть изменена
	 при автоматическом подборе;


	- False. Нижняя граница
	 ограничения ослаблена.


Для определения, зафиксирована ли верхняя граница ограничения, используйте
 свойство [IMsTargetConstraintCommon.UpperBoundFixed](IMsTargetConstraintCommon.UpperBoundFixed.htm).
