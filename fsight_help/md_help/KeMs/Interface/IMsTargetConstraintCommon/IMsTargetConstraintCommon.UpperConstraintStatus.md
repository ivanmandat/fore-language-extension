# IMsTargetConstraintCommon.UpperConstraintStatus


## Синтаксис


UpperConstraintStatus: [TargetConstraintStatusType](KeCp.chm::/Enums/TargetConstraintStatusType.htm);


## Описание


Свойство UpperConstraintStatus
 возвращает состояние верхней границы ограничения.


## Комментарии


Значение доступно после расчета целевой задачи. Для получения состояния
 нижней границы ограничения используйте свойство [IMsTargetConstraintCommon.LowerConstraintStatus](IMsTargetConstraintCommon.LowerConstraintStatus.htm).
