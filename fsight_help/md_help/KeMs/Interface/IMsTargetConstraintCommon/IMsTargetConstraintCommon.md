# IMsTargetConstraintCommon


Сборка: Ms;


## Описание


Интерфейс IMsTargetConstraintCommon
 предназначен для работы с ограничением в целевой задаче.


## Иерархия наследования


           IMsTargetConstraintCommon


## Комментарии


В целевой задаче ограничения накладываются на управляющие переменные
 и целевую функцию.
