# IModelBoxPad.Expanded


## Синтаксис


		Expanded: Boolean;


## Описание


Свойство Expanded определяет
 признак развёрнутости панели.


## Комментарии


Допустимые значения:


	- True. Панель находится
	 в развёрнутом состоянии;


	- False. Панель находится
	 в свёрнутом состоянии.
