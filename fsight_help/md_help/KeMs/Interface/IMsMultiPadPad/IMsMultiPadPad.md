# IMsMultiPadPad


## Описание


Интерфейс IMsMultiPadPad содержит
 свойства панели компонента.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CanShow](IMsMultiPadPad.CanShow.htm)
		 Свойство CanShow определяет
		 признак отображения панели в области компонента.


		 ![](../../Property_Image.gif)
		 [Expanded](IMsMultiPadPad.Expanded.htm)
		 Свойство Expanded определяет
		 состояние панели.


		 ![](../../Property_Image.gif)
		 [Id](IMsMultiPadPad.Id.htm)
		 Свойство Id возвращает
		 идентификатор панели.
