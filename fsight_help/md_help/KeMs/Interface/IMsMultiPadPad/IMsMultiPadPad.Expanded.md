# IMsMultiPadPad.Expanded


## Синтаксис


Expanded: Boolean;


## Описание


Свойство Expanded определяет
 состояние панели.


## Комментарии


Если свойству установлено значение True,
 то панель находится в развернутом состоянии, иначе панель свернута.
