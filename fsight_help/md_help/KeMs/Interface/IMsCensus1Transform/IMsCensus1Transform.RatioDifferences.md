# IMsCensus1Transform.RatioDifferences


## Синтаксис


		RatioDifferences: Array;


## Описание


Свойство RatioDifferences возвращает
 разность для аддитивной модели сезонности и отношение для мультипликативной
 модели.


## Комментарии


Значение доступно только после расчёта метода.


Для определения модели сезонности используйте свойство [IMsCensus1Transform.Seasonality](IMsCensus1Transform.Seasonality.htm).
