# IMsCensus1Transform.Seasonality


## Синтаксис


		Seasonality: [SeasonalityType](StatLib.chm::/Enums/SeasonalityType.htm);


## Описание


Свойство Seasonality определяет
 используемую модель сезонности.


## Комментарии


Свойство не может принимать значение SeasonalityType.None.
