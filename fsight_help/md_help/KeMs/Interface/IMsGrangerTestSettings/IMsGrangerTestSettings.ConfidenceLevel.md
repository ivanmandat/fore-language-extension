# IMsGrangerTestSettings.ConfidenceLevel


## Синтаксис


ConfidenceLevel: Double;


## Описание


Свойство ConfidenceLevel определяет уровень значимости, при котором гипотеза отвергается.


## Комментарии


Значение по умолчанию - «0.05».
