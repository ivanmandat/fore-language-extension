# IMsDimIteratorFilter


Сборка: Ms;


## Описание


Интерфейс IMsDimIteratorFilter
 предназначен для работы с фильтром измерения в многомерном итераторе.


## Иерархия наследования


           IMsDimIteratorFilter


## Комментарии


Для получения фильтра измерения используйте свойство [IMsDimIteratorFilteredDimension.Filter](../IMsDimIteratorFilteredDimension/IMsDimIteratorFilteredDimension.Filter.htm).
