# IMsParametrizedDimensionAggregation.ByHierarchy


## Синтаксис


ByHierarchy: Boolean;


## Описание


Свойство ByHierarchy определяет,
 выполнять ли агрегацию по иерархии.


## Комментарии


Допустимые значения:


	- True. Выполняется агрегация
	 по иерархии;


	- False. Выполняется агрегация
	 без иерархии.
