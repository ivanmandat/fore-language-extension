# IMsParametrizedDimensionAggregation


Сборка: Ms;


## Описание


Интерфейс IMsParametrizedDimensionAggregation
 предназначен для настройки агрегации параметризированного измерения.


## Иерархия наследования


           [INamedEntityModel](ForeSys.chm::/Interface/INamedEntityModel/INamedEntityModel.htm)


           IMsParametrizedDimensionAggregation


## Комментарии


Для получения настроек агрегации параметризированного измерения используйте
 свойство [IMsParametrizedDimension.Aggregation](../IMsParametrizedDimension/IMsParametrizedDimension.Aggregation.htm).
