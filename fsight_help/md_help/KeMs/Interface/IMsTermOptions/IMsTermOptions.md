# IMsTermOptions


Сборка: Ms;


## Описание


Интерфейс IMsTermOptions определяет дополнительные параметры терма.


## Иерархия наследования


           IMsTermOptions


## Комментарии


IMsTermOptions является базовым для интерфейса [IMsPDLTermOptions](../IMsPDLTermOptions/IMsPDLTermOptions.htm). Получить IMsTermOptions можно используя [IMsCompositeFormulaTerm.CustomOptions](../IMsCompositeFormulaTerm/IMsCompositeFormulaTerm.CustomOptions.htm).


Использование IMsTermOptions приведено в примере для [IMsLinearRegressionTransform.PDLStatCoefficients](../IMsLinearRegressionTransform/IMsLinearRegressionTransform.PDLStatCoefficients.htm).
