# IMsDeterministicExpression.IsActive


## Синтаксис


IsActive: Boolean;


## Описание


Свойство IsActive возвращает
 признак активности формулы.


## Комментарии


Свойство возвращает значение True,
 если текущий период расчёта не превышает дату окончания действия формулы.
