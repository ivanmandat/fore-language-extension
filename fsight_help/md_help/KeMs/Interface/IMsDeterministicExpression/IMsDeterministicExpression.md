# IMsDeterministicExpression


Сборка: Ms;


## Описание


Интерфейс IMsDeterministicExpression
 предназначен для настройки формулы, имеющей период действия.


## Иерархия наследования


           IMsDeterministicExpression


## Комментарии


Формулу, имеющую период действия, можно получить с помощью свойств и
 методов интерфейса [IMsDeterministicExpressionList](../IMsDeterministicExpressionList/IMsDeterministicExpressionList.htm).
