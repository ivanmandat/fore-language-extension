# IModelBox.Execute


## Синтаксис


		Execute;


## Описание


Метод Execute выполняет
 расчёт модели и обновляет компонент ModelBox.


## Комментарии


Для определения признака изменения модели используйте свойство [IModelBox.IsDirty](IModelBox.IsDirty.htm).
