# IMsCensus1Result


Сборка: Ms;


## Описание


Интерфейс IMsCensus1Result предназначен
 для работы с результатами расчёта сезонной составляющей методом Census1.


## Иерархия наследования


           [IMsResultObject](../IMsResultObject/IMsResultObject.htm)


           IMsCensus1Result


## Комментарии


Данный интерфейс используется в пользовательских функциях и методах,
 предназначенных для расчёта в детерминированном уравнении.
