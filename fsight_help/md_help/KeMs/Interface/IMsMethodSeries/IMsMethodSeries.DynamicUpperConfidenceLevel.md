# IMsMethodSeries.DynamicUpperConfidenceLevel


## Синтаксис


DynamicUpperConfidenceLevel: [IMsFormulaTermInfo](../IMsFormulaTermInfo/IMsFormulaTermInfo.htm);


## Описание


Свойство DynamicUpperConfidenceLevel определяет ряд, содержащий верхнюю динамическую доверительную границу.


## Комментарии


Ряд задается в виде терма.


После расчета метода свойство [IMsMethodSeries.IsAvailable](IMsMethodSeries.IsAvailable.htm), вызванное с параметром MsMethodSeriesType.DynamicUpperConfidenceLevel, возвращает признак участия ряда в расчете.
