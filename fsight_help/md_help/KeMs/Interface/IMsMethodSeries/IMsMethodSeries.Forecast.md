# IMsMethodSeries.Forecast


## Синтаксис


Forecast: [IMsFormulaTermInfo](../IMsFormulaTermInfo/IMsFormulaTermInfo.htm);


## Описание


Свойство Forecast определяет прогнозный ряд.


## Комментарии


Ряд задается в виде терма.


После расчета метода свойство [IMsMethodSeries.IsAvailable](IMsMethodSeries.IsAvailable.htm), вызванное с параметром MsMethodSeriesType.Forecast, возвращает признак участия ряда в расчете.
