# IMsMethodSeries.IsAvailable


## Синтаксис


IsAvailable(Type: [MsMethodSeriesType](../../Enums/MsMethodSeriesType.htm)): Boolean;


## Параметры


Type. Тип ряда.


## Описание


Свойство IsAvailable возвращает признак: использовался ли указанный ряд при расчете метода.


## Комментарии


IsAvailable следует использовать только после расчета метода.
