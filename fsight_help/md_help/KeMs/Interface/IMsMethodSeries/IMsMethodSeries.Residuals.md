# IMsMethodSeries.Residuals


## Синтаксис


Residuals: [IMsFormulaTermInfo](../IMsFormulaTermInfo/IMsFormulaTermInfo.htm);


## Описание


Свойство Residuals определяет ряд остатков.


## Комментарии


Ряд задается в виде терма.


После расчета метода свойство [IMsMethodSeries.IsAvailable](IMsMethodSeries.IsAvailable.htm), вызванное с параметром MsMethodSeriesType.Residuals, возвращает признак участия ряда в расчете.
