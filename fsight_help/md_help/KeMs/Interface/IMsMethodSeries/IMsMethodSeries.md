# IMsMethodSeries


Сборка: Ms;


## Описание


Интерфейс IMsMethodSeries предназначен
 для работы с набором возможных рядов, используемых методом при расчете.


## Комментарии


При расчете методы используют различные наборы рядов. Для определения,
 участвовал ли конкретный ряд в расчете, используйте свойство [IsAvailable](IMsMethodSeries.IsAvailable.htm).
