# IMsMethodSeries.SeriesByType


## Синтаксис


SeriesByType(Type: [MsMethodSeriesType](../../Enums/MsMethodSeriesType.htm)): [IMsFormulaTermInfo](../IMsFormulaTermInfo/IMsFormulaTermInfo.htm);


## Параметры


Type. Тип ряда.


## Описание


Свойство SeriesByType позволяет получить ряд заданного типа.


## Комментарии


Ряд будет получен в виде терма и доступен для изменения его параметров.
