# IMsMethodSeries.DynamicLowerConfidenceLevel


## Синтаксис


DynamicLowerConfidenceLevel: [IMsFormulaTermInfo](../IMsFormulaTermInfo/IMsFormulaTermInfo.htm);


## Описание


Свойство DynamicLowerConfidenceLevel определяет ряд, содержащий нижнюю динамическую доверительную границу.


## Комментарии


Ряд задается в виде терма.


После расчета метода свойство [IMsMethodSeries.IsAvailable](IMsMethodSeries.IsAvailable.htm), вызванное с параметром MsMethodSeriesType.DynamicLowerConfidenceLevel, возвращает признак участия ряда в расчете.
