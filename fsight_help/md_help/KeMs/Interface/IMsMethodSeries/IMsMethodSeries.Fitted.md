# IMsMethodSeries.Fitted


## Синтаксис


Fitted: [IMsFormulaTermInfo](../IMsFormulaTermInfo/IMsFormulaTermInfo.htm);


## Описание


Свойство Fitted определяет
 модельный ряд.


## Комментарии


Ряд задается в виде терма.


После расчета метода свойство [IMsMethodSeries.IsAvailable](IMsMethodSeries.IsAvailable.htm),
 вызванное с параметром MsMethodSeriesType.Fitted,
 возвращает признак участия ряда в расчете.
