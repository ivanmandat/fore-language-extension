# IMsMethodSeries.UpperConfidenceLevel


## Синтаксис


UpperConfidenceLevel: [IMsFormulaTermInfo](../IMsFormulaTermInfo/IMsFormulaTermInfo.htm);


## Описание


Свойство UpperConfidenceLevel определяет ряд, содержащий верхнюю доверительную границу.


## Комментарии


Ряд задается в виде терма.


После расчета метода свойство [IMsMethodSeries.IsAvailable](IMsMethodSeries.IsAvailable.htm), вызванное с параметром MsMethodSeriesType.UpperConfidenceLevel, возвращает признак участия ряда в расчете.
