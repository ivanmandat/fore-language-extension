# IMsMethodSeries.LowerConfidenceLevel


## Синтаксис


LowerConfidenceLevel: [IMsFormulaTermInfo](../IMsFormulaTermInfo/IMsFormulaTermInfo.htm);


## Описание


Свойство LowerConfidenceLevel определяет ряд, содержащий нижнюю доверительную границу.


## Комментарии


Ряд задается в виде терма.


После расчета метода свойство [IMsMethodSeries.IsAvailable](IMsMethodSeries.IsAvailable.htm), вызванное с параметром MsMethodSeriesType.LowerConfidenceLevel, возвращает признак участия ряда в расчете.
