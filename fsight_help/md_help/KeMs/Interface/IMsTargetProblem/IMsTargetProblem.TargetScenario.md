# IMsTargetProblem.TargetScenario


## Синтаксис


TargetScenario: [IMsScenario](../IMsScenario/IMsScenario.htm);


## Описание


Свойство TargetScenario определяет
 сценарий целевой задачи.


## Комментарии


Сценарий целевой задачи используется, если задана траектория целевой
 функции. Траектория целевой функции задаётся значениями моделируемой переменной
 в модели целевой функции, возвращаемой свойством [IMsBaseOptimizationProblem.ControlTransform](../IMsBaseOptimizationProblem/IMsBaseOptimizationProblem.ControlTransform.htm).
