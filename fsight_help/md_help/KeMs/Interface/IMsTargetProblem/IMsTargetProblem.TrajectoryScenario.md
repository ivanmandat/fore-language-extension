# IMsTargetProblem.TrajectoryScenario


## Синтаксис


TrajectoryScenario: [IMsScenario](../IMsScenario/IMsScenario.htm);


## Описание


Свойство TrajectoryScenario
 определяет сценарий для траектории целевой функции.


## Комментарии


Сценарий траектории целевой функции используется, если задана траектория
 целевой функции. Траектория целевой функции задаётся значениями моделируемой
 переменной в модели целевой функции, возвращаемой свойством [IMsBaseOptimizationProblem.ControlTransform](../IMsBaseOptimizationProblem/IMsBaseOptimizationProblem.ControlTransform.htm).
