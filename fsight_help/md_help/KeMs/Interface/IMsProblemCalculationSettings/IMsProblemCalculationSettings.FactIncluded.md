# IMsProblemCalculationSettings.FactIncluded


## Синтаксис


FactIncluded: Boolean;


## Описание


Свойство FactIncluded определяет, будет ли производиться расчет задачи по сценарию «Факт».


## Комментарии


Допустимые значения:


-
True. Задача рассчитывается по сценарию «Факт»;


-
False. Задача не рассчитывается по сценарию «Факт».
