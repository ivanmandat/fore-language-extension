# IMsProblemCalculationSettings.FilterWarnings


## Синтаксис


FilterWarnings: Boolean;


## Описание


Свойство FilterWarnings определяет,
 выводить ли дублирующиеся предупреждения.


## Комментарии


Допустимые значения:


	- True. Выводятся все
	 предупреждения, в том числе и дублирующиеся;


	- False. Значение по умолчанию.
	 Дублирующиеся предупреждения удаляются.
