# IMsProblemCalculationSettings.Callback


## Синтаксис


Callback: [IMsProblemCalculationCallback](../IMsProblemCalculationCallback/IMsProblemCalculationCallback.htm);


## Описание


Свойство Callback определяет
 объект, отслеживающий события, происходящие при расчете задачи в контейнере
 моделирования.


## Комментарии


Объект должен быть пользовательского класса. Данный класс должен являться
 наследником класса [MsProblemCalculationCallback](../../Class/ProblemCalculationCallback/ProblemCalculationCallback.htm)
 и содержать его переопределенные методы.
