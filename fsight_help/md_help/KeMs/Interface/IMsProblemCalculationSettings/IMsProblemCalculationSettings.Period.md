# IMsProblemCalculationSettings.Period


## Синтаксис


Period: [IMsModelPeriod](../IMsModelPeriod/IMsModelPeriod.htm);


## Описание


Свойство Period возвращает параметры периодов идентификации и прогнозирования задачи.


## Комментарии


Period служит для настройки данных периодов без необходимости пересохранения задачи. По умолчанию значение свойства совпадает с периодом расчета задачи: Period = [IMsProblemDetails.Period](../IMsProblemDetails/IMsProblemDetails.Period.htm).
