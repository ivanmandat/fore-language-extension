Свойство ParamValues возвращает
 значения параметров метамодели, входящей в задачу.

Для получения метамодели, рассчитываемой задачей, используйте свойство
 [IMsProblem.MetaModel](../IMsProblem/IMsProblem.MetaModel.htm).
