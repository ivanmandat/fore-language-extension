# IMsTargetConstraint


Сборка: Ms;


## Описание


Интерфейс IMsTargetConstraint
 предназначен для работы с ограничением целевой функции.


## Иерархия наследования


           IMsTargetConstraint


## Комментарии


Для работы с коллекцией ограничений целевой функции используйте интерфейс
 [IMsTargetConstraints](../IMsTargetConstraints/IMsTargetConstraints.htm).
