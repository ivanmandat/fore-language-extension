# IMsTargetConstraint.Operands


## Синтаксис


Operands: [IMsFormulaTermList](../IMsFormulaTermList/IMsFormulaTermList.htm);


## Описание


Свойство Operands возвращает
 коллекцию операндов для ограничения целевой функции.


## Комментарии


Из операндов, содержащихся в коллекции, составляется выражение ограничения
 целевой функции. Для задания выражения используйте свойство [IMsTargetConstraint.Expression](IMsTargetConstraint.Expression.htm).
