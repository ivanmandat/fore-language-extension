# IMsTargetConstraint.LowerBoundFixed


## Синтаксис


LowerBoundFixed: [MsBoundFixType](../../Enums/MsBoundFixType.htm);


## Описание


Свойство LowerBoundFixed определяет
 режим фиксации нижней границы ограничения.


## Комментарии


По умолчанию нижняя граница зафиксирована, то есть свойство имеет значение
 [MsBoundFixType.Fixed](../../Enums/MsBoundFixType.htm).


Для определения режима фиксации верхней границы ограничения, используйте
 свойство [IMsTargetConstraint.UpperBoundFixed](IMsTargetConstraint.UpperBoundFixed.htm).
