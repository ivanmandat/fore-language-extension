# IMsTargetConstraint.Expression


## Синтаксис


Expression: [IExpression](ForeSys.chm::/Interface/IExpression/IExpression.htm);


## Описание


Свойство Expression возвращает
 выражение ограничения целевой функции.


## Комментарии


В выражении можно использовать только те операнды, которые содержатся
 в коллекции операндов для ограничения целевой функции. Для работы с коллекцией
 операндов используйте свойство [IMsTargetConstraint.Operands](IMsTargetConstraint.Operands.htm).
