# IMsTargetConstraint.UpperBoundFixed


## Синтаксис


UpperBoundFixed: [MsBoundFixType](../../Enums/MsBoundFixType.htm);


## Описание


Свойство UpperBoundFixed определяет
 режим фиксации верхней границы ограничения.


## Комментарии


По умолчанию верхняя граница зафиксирована, то есть свойство имеет значение
 [MsBoundFixType.Fixed](../../Enums/MsBoundFixType.htm).


Для определения режима фиксации нижней границы ограничения, используйте
 свойство [IMsTargetConstraint.LowerBoundFixed](IMsTargetConstraint.LowerBoundFixed.htm).
