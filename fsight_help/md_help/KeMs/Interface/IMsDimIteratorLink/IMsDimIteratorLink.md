# IMsDimIteratorLink


Сборка: Ms;


## Описание


Интерфейс IMsDimIteratorLink
 предназначен для работы со связанным измерением в измерении многомерного
 итератора.


## Иерархия наследования


           IMsDimIteratorLink


## Комментарии


Для работы со всеми связями измерения многомерного итератора используйте
 интерфейс [IMsDimIteratorLinks](../IMsDimIteratorLinks/IMsDimIteratorLinks.htm).
