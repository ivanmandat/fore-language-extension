# IMsDimIteratorLink.DestinationAttribute


## Синтаксис

## Описание


Свойство DestinationAttribute
 определяет атрибут управляемого измерения.


## Комментарии


Для задания атрибута управляющего измерения используйте свойство [IMsDimIteratorLink.SourceAttribute](IMsDimIteratorLink.SourceAttribute.htm).
