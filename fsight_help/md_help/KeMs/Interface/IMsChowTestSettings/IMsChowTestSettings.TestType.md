# IMsChowTestSettings.TestType


## Синтаксис


		TestType: [ChowTestType](StatLib.chm::/Enums/ChowTestType.htm);


## Описание


Свойство TestType определяет
 тип теста.


## Комментарии


Значение свойства следует выбирать в зависимости от соотношения количества
 элементов в группах:


	- ChowTestType.BreakPoint.
	 Количество элементов в обеих группах примерно одинаковое;


	- ChowTestType.Forecast.
	 Количество элементов в одной из групп существенно превосходит количество
	 элементов в другой.
