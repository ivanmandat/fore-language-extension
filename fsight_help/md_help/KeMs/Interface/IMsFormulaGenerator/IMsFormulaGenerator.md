# IMsFormulaGenerator


Сборка: Ms;


## Описание


Интерфейс IMsFormulaGenerator
 содержит базовые свойства для получения информации о формулах.


## Иерархия наследования


           IMsFormulaGenerator


## Комментарии


Интерфейс является базовым для следующих интерфейсов:


	- [IMsFormulaGenIterator](../IMsFormulaGenIterator/IMsFormulaGenIterator.htm);


	- [IMsFormulaInfo](../IMsFormulaInfo/IMsFormulaInfo.htm).
