# IMsFormulaGenerator.Expression


## Синтаксис


		Expression: String;


## Описание


Свойство Expression возвращает
 выражение формулы, которая использовалось для расчёта.


## Комментарии


[Термы](../IMsFormulaGenTerm/IMsFormulaGenTerm.htm) в выражении
 обозначаются индексами, например {1}.


## Пример


Пример использования приведён в описании метода [IMsProblemCalculation.GetFormulaInfo](../IMsProblemCalculation/IMsProblemCalculation.GetFormulaInfo.htm).
