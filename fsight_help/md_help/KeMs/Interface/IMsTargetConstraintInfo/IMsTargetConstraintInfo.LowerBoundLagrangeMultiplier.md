# IMsTargetConstraintInfo.LowerBoundLagrangeMultiplier


## Синтаксис


LowerBoundLagrangeMultiplier: Double;


## Описание


Свойство LowerBoundLagrangeMultiplier
 возвращает множитель Лагранжа для нижней границы ограничения.


## Комментарии


Множитель Лагранжа характеризует в какой степени значение ограничения
 близко его границе. Значение доступно только после расчета целевой задачи.


Для получения множителя Лагранжа для верхней границы ограничения используйте
 свойство [IMsTargetConstraintInfo.UpperBoundLagrangeMultiplier](IMsTargetConstraintInfo.UpperBoundLagrangeMultiplier.htm).
