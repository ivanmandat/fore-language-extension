# IMsTargetConstraintInfo.OptimalValue


## Синтаксис


OptimalValue: Double;


## Описание


Свойство OptimalValue возвращает
 значение, соответствующее оптимальному решению целевой задачи.


## Комментарии


Значение доступно только после расчета целевой задачи.
