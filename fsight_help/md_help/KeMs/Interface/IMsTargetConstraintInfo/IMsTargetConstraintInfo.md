# IMsTargetConstraintInfo


Сборка: Ms;


## Описание


Интерфейс IMsTargetConstraintInfo
 предназначен для работы со значением ограничения целевой функции.


## Иерархия наследования


           [IMsTargetConstraintCommon](../IMsTargetConstraintCommon/IMsTargetConstraintCommon.htm)


           IMsTargetConstraintInfo


## Комментарии


Для работы с коллекцией значений используйте интерфейс [IMsTargetConstraintInfoArray](../IMsTargetConstraintInfoArray/IMsTargetConstraintInfoArray.htm).
