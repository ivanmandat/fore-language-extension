# IMsTargetConstraintInfo.UpperBoundLagrangeMultiplier


## Синтаксис


UpperBoundLagrangeMultiplier: Double;


## Описание


Свойство UpperBoundLagrangeMultiplier
 возвращает множитель Лагранжа для верхней границы ограничения.


## Комментарии


Множитель Лагранжа характеризует в какой степени значение ограничения
 близко его границе. Значение доступно только после расчета целевой задачи.


Для получения множителя Лагранжа для нижней границы ограничения используйте
 свойство [IMsTargetConstraintInfo.LowerBoundLagrangeMultiplier](IMsTargetConstraintInfo.LowerBoundLagrangeMultiplier.htm).
