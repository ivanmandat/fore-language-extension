# IMsMetaAttributeValueList.FindById


## Синтаксис


FindById(AttributeId: String): [IMsMetaAttributeValue](../IMsMetaAttributeValue/IMsMetaAttributeValue.htm);


## Параметры


AttributeId. Идентификатор
 атрибута, по которому осуществляется поиск.


## Описание


Метод FindById осуществляет
 поиск атрибута в коллекции по заданному идентификатору. Идентификатор,
 по которому осуществляется поиск, передается посредством параметра AttributeId.
