# IDmField.CategoriesCount


## Синтаксис


		CategoriesCount: Integer;


## Описание


Свойство CategoriesCount
 определяет количество категорий, на которые должны быть разбиты данные.


## Комментарии


По умолчанию данные разбиваются на десять категорий.
