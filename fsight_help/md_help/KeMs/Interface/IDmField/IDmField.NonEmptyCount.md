# IDmField.NonEmptyCount


## Синтаксис


		NonEmptyCount: Integer;


## Описание


Свойство NonEmptyCount
 возвращает количество непустых значений, которые содержатся в поле.


## Комментарии


Для выделения пустых значений в отдельную категорию используйте свойство
 [IDmField.TreatNanAsCategory](IDmField.TreatNanAsCategory.htm).
