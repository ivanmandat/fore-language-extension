# IDmField.IsNumerical


## Синтаксис


		IsNumerical: Boolean;


## Описание


Свойство IsNumerical возвращает
 признак содержания числовых данных в поле.


## Комментарии


Допустимые значения:


	- True. Поле содержит
	 числовые данные;


	- False. Поле содержит
	 нечисловые данные.
