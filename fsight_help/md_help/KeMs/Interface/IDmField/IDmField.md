# IDmField


Сборка: Ms;


## Описание


Интерфейс IDmField предназначен
 для работы с полем источника данных.


## Иерархия наследования


           IDmField


## Комментарии


Для получения поля источника данных используйте свойство [IDmDataSource.Field](../IDmDataSource/IDmDataSource.Field.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BinningType](IDmField.BinningType.htm)
		 Свойство BinningType
		 определяет метод расчёта процедуры Binning.


		 ![](../../Property_Image.gif)
		 [CategoriesCount](IDmField.CategoriesCount.htm)
		 Свойство CategoriesCount
		 определяет количество категорий, на которые должны быть разбиты
		 данные.


		 ![](../../Property_Image.gif)
		 [DataSource](IDmField.DataSource.htm)
		 Свойство DataSource
		 возвращает источник данных, которому принадлежит поле.


		 ![](../../Property_Image.gif)
		 [ExplanatoryType](IDmField.ExplanatoryType.htm)
		 Свойство ExplanatoryType
		 определяет тип фактора, влияющего на анализируемый признак.


		 ![](../../Property_Image.gif)
		 [FieldType](IDmField.FieldType.htm)
		 Свойство FieldType
		 определяет тип данных поля.


		 ![](../../Property_Image.gif)
		 [Index](IDmField.Index.htm)
		 Свойство Index
		 возвращает индекс поля.


		 ![](../../Property_Image.gif)
		 [IsNumerical](IDmField.IsNumerical.htm)
		 Свойство IsNumerical
		 возвращает признак содержания числовых данных в поле.


		 ![](../../Property_Image.gif)
		 [Name](IDmField.Name.htm)
		 Свойство Name
		 возвращает наименование поля.


		 ![](../../Property_Image.gif)
		 [NonEmptyCount](IDmField.NonEmptyCount.htm)
		 Свойство NonEmptyCount
		 возвращает количество непустых значений, которые содержатся в
		 поле.


		 ![](../../Property_Image.gif)
		 [TreatNanAsCategory](IDmField.TreatNanAsCategory.htm)
		 Свойство TreatNanAsCategory
		 определяет признак выделения пропусков в отдельную категорию.
