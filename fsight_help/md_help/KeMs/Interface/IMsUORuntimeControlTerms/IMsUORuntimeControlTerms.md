# IMsUORuntimeControlTerms


Сборка: Ms;


## Описание


Интерфейс IMsUORuntimeControlTerms
 предназначен для работы с коллекцией управляющих переменных пользовательской
 задачи оптимизации.


## Иерархия наследования


IMsUORuntimeControlTerms


## Комментарии


Управляющие переменные можно получить, используя свойство [IMsUORuntime.ControlVariables](../IMsUORuntime/IMsUORuntime.ControlVariables.htm)
 во время расчёта задачи.

		 Имя свойства
