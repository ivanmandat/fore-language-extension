# IMsUORuntimeControlTerms.FindByString


## Синтаксис


FindByString(Value: String): Integer;


## Параметры


Value. Строковое представление
 управляемой переменной.


## Описание


Метод FindByString осуществляет
 поиск управляющей переменной по строковому представлению и возвращает
 её индекс в коллекции.


## Комментарии


Метод возвращает значение -1, если управляющая переменная не найдена.
