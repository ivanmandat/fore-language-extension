## Описание


Интерфейс IModelBoxPads содержит
 свойства и методы для работы с коллекцией панелей компонента [ModelBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/ModelBox.htm).
