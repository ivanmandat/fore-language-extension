# IMsDimIteratorAdvancedFilterConditionGroup.RemoveCondition


## Синтаксис


		RemoveCondition(Index: Integer): Boolean;


## Параметры


Index. Индекс условия фильтрации.


## Описание


Метод RemoveCondition удаляет
 заданное условие из группы расширенного фильтра по индексу.


## Комментарии


Для получения индекса условия в группе используйте свойство [IMsDimIteratorAdvancedFilterConditionGroup.IndexOfCondition](IMsDimIteratorAdvancedFilterConditionGroup.IndexOfCondition.htm).
