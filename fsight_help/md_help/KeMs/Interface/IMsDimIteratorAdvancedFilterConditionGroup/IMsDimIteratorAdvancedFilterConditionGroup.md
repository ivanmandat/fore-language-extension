# IMsDimIteratorAdvancedFilterConditionGroup


Сборка: Ms;


## Описание


Интерфейс IMsDimIteratorAdvancedFilterConditionGroup
 предназначен для работы с группой условий [расширенного фильтра](uinav.chm::/GUI/UFE/AttributeFiltering.htm).
