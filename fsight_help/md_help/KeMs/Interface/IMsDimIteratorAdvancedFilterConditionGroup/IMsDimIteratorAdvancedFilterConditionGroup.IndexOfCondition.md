# IMsDimIteratorAdvancedFilterConditionGroup.IndexOfCondition


## Синтаксис


		IndexOfCondition(Condition: [IMsDimIteratorAdvancedFilterCondition](../IMsDimIteratorAdvancedFilterCondition/IMsDimIteratorAdvancedFilterCondition.htm)):
		 Integer;


## Параметры


Condition.
 Условие фильтрации.


## Описание


Свойство IndexOfCondition возвращает
 индекс заданного условия в группе расширенного фильтра.


## Комментарии


Для получения условия в группе по индексу используйте свойство [IMsDimIteratorAdvancedFilterConditionGroup.Item](IMsDimIteratorAdvancedFilterConditionGroup.Item.htm).


Для удаления условия из группы по индексу используйте метод [IMsDimIteratorAdvancedFilterConditionGroup.RemoveCondition](IMsDimIteratorAdvancedFilterConditionGroup.RemoveCondition.htm).
