# IMsDimIteratorAdvancedFilterConditionGroup.Item


## Синтаксис


		Item(Index: Integer): [IMsDimIteratorAdvancedFilterCondition](../IMsDimIteratorAdvancedFilterCondition/IMsDimIteratorAdvancedFilterCondition.htm);


## Параметры


Index.
 Индекс условия фильтрации.


## Описание


Свойство Item возвращает условие
 из группы расширенного фильтра по индексу.


## Комментарии


Для получения индекса условия в группе используйте свойство [IMsDimIteratorAdvancedFilterConditionGroup.IndexOfCondition](IMsDimIteratorAdvancedFilterConditionGroup.IndexOfCondition.htm).
