# IMsFormulaTransformCoord.Scenario


## Синтаксис


Scenario: Integer;


## Описание


Свойство Scenario определяет
 ключ сценария, по которому будет производиться расчет.


## Комментарии


По умолчанию расчет производится по измерению фактов, при этом свойство
 принимает значение «-1».
