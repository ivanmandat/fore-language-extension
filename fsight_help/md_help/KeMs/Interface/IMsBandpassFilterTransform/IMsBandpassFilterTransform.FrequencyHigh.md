# IMsBandpassFilterTransform.FrequencyHigh


## Синтаксис


		FrequencyHigh: Double;


## Описание


Свойство FrequencyHigh определяет
 значение верхней границы периода сезонности.


## Комментарии


Значение по умолчанию зависит от календарной динамики, которая применяется
 для расчёта метода:


		 Динамика
		 Верхняя граница


		 Годовая
		 8


		 Полугодовая
		 16


		 Квартальная
		 32


		 Месячная
		 96


		 Недельная
		 416


		 5-дневная
		 2088


		 7-дневная
		 2920
