# IMsBandpassFilterTransform.FrequencyLow


## Синтаксис


		FrequencyLow: Double;


## Описание


Свойство FrequencyLow определяет
 значение нижней границы периода сезонности.


## Комментарии


Значение по умолчанию зависит от календарной динамики, которая применяется
 для расчёта метода:


		 Динамика
		 Нижняя граница


		 Годовая
		 2


		 Полугодовая
		 3


		 Квартальная
		 6


		 Месячная
		 18


		 Недельная
		 78


		 5-дневная
		 391,5


		 7-дневная
		 547,5
