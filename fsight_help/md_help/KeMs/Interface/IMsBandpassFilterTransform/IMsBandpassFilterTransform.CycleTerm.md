# IMsBandpassFilterTransform.CycleTerm


## Синтаксис


		CycleTerm: [IMsFormulaTermInfo](../IMsFormulaTermInfo/IMsFormulaTermInfo.htm);


## Описание


Свойство CycleTerm определяет
 элемент, соответствующий переменной, в которую будет выгружена циклическая
 составляющая исходного ряда.


## Комментарии


Для указания переменной, в которую будет выгружена нециклическая компонента
 используйте свойство [IMsBandpassFilterTransform.NonCyclicalTerm](IMsBandpassFilterTransform.NonCyclicalTerm.htm).
