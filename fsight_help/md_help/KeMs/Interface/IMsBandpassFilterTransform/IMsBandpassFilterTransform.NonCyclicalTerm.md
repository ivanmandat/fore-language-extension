# IMsBandpassFilterTransform.NonCyclicalTerm


## Синтаксис


		NonCyclicalTerm: [IMsFormulaTermInfo](../IMsFormulaTermInfo/IMsFormulaTermInfo.htm);


## Описание


Свойство NonCyclicalTerm определяет
 элемент, соответствующий переменной, в которую будет выгружена нециклическая
 составляющая исходного ряда.


## Комментарии


Для указания переменной, в которую будет выгружена циклическая компонента,
 используйте свойство [IMsBandpassFilterTransform.CycleTerm](IMsBandpassFilterTransform.CycleTerm.htm).
