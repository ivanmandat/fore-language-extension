# IMsCalculationChainUserOptimizationConstraint


Сборка: Ms;


## Описание


Интерфейс IMsCalculationChainUserOptimizationConstraint
 предназначен для работы с элементом цепочки расчёта метамодели, который
 соответствует ограничению пользовательской задачи оптимизации.


## Иерархия наследования


           [IMsCalculationChainEntry](../IMsCalculationChainEntry/IMsCalculationChainEntry.htm)


           IMsCalculationChainUserOptimizationConstraint


## Комментарии


Элемент цепочки расчёта можно получить, используя свойства и методы
 интерфейса [IMsCalculationChainEntries](../IMsCalculationChainEntries/IMsCalculationChainEntries.htm).
