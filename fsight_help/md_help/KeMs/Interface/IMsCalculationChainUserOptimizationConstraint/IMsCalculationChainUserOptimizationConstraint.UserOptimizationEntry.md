## Синтаксис

## Описание


Свойство UserOptimizationEntry
 возвращает элемент, соответствующий пользовательской задаче оптимизации
 в цепочке расчёта.


## Комментарии


Свойство возвращает тот элемент, который был указан при выполнении метода
 [IMsCalculationChainEntries.AddUserOptimizationConstraint](../IMsCalculationChainEntries/IMsCalculationChainEntries.AddUserOptimizationConstraint.htm).
