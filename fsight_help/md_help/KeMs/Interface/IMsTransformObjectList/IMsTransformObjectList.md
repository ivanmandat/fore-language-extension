# IMsTransformObjectList


Сборка: Ms;


## Описание


Интерфейс IMsTransformObjectList
 содержит свойства и методы для работы с коллекцией объектов, используемых
 методом расчета.


## Иерархия наследования


           IMsTransformObjectList


## Комментарии


Объекты данной коллекции используются для вставки константного значения
 справочника в уравнение расчета. Каждый объект, содержащийся в коллекции,
 является объектом репозитория.
