# IMsFormulaInfo


Сборка: Ms;


## Описание


Интерфейс IMsFormulaInfo предназначен
 для получения информации о формуле, которая использовалась при расчёте.


## Иерархия наследования


           [IMsFormulaGenerator](../IMsFormulaGenerator/IMsFormulaGenerator.htm)


           IMsFormulaInfo


## Комментарии


Информацию о формуле возвращает свойство [IMsFormulaInfoList.Item](../IMsFormulaInfoList/IMsFormulaInfoList.Item.htm).
