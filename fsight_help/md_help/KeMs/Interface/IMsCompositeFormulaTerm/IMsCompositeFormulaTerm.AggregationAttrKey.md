# IMsCompositeFormulaTerm.AggregationAttrKey


## Синтаксис


		AggregationAttrKey: Integer;


## Описание


Свойство AggregationAttrKey
 определяет ключ атрибута, по значениям которого будет рассчитываться агрегация
 для составного терма.
