# IMsCompositeFormulaTerm.CustomOptions


## Синтаксис


		CustomOptions: [IMsTermOptions](../IMsTermOptions/IMsTermOptions.htm);


## Описание


Свойство CustomOptions возвращает
 дополнительные параметры составного терма.


## Комментарии


Свойство используется для задания лаговых переменных для метода [линейной
 регрессии (оценка МНК)](../IMsLinearRegressionTransform/IMsLinearRegressionTransform.htm).
