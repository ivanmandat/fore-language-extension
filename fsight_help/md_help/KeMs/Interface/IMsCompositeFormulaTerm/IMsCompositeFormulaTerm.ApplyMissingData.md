# IMsCompositeFormulaTerm.ApplyMissingData


## Синтаксис


		ApplyMissingData: Boolean;


## Описание


Свойство ApplyMissingData определяет
 признак применения метода обработки пропусков для значений терма.


## Комментарии


Допустимые значения:


	- True. Для значений терма
	 применяется обработка пропусков. Для настройки параметров обработки
	 используйте [IMsCompositeFormulaTerm.MissingData](IMsCompositeFormulaTerm.MissingData.htm);


	- False. По умолчанию.
	 Обработка пропусков не используется.
