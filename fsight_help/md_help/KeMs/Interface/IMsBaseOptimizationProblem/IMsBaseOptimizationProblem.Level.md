## Описание


Свойство Level определяет
 уровень календаря, по которому будет осуществляться расчёт задачи.


## Комментарии


Для задания периодов расчёта задачи используйте свойство [IMsProblemDetails.Period](../IMsProblemDetails/IMsProblemDetails.Period.htm).
