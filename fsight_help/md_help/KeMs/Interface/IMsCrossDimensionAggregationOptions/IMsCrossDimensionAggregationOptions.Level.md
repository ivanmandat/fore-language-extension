# IMsCrossDimensionAggregationOptions.Level


## Синтаксис


		Level: Boolean;


## Описание


Свойство Level
 определяет признак использования функции Level для расчёта агрегации.


## Комментарии


Функция Level позволяет сглаживать полученные данные.


Допустимые значения:


	- True.
	 Использовать функцию Level для расчёта агрегации;


	- False.
	 Не использовать функцию Level.
