# IMsCrossDimensionAggregationOptions.TreatEmptyCompositionRelevanceAs


## Синтаксис


		TreatEmptyCompositionRelevanceAs: Double;


## Описание


Свойство TreatEmptyCompositionRelevanceAs
 определяет значение, которое используется для заполнения пропусков в данных
 порога расчёта агрегации.


## Комментарии


Значение по умолчанию Double.Nan.
