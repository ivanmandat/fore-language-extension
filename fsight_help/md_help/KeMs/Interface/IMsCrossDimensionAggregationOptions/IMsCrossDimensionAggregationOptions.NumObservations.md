# IMsCrossDimensionAggregationOptions.NumObservations


## Синтаксис


		NumObservations: Integer;


## Описание


Свойство NumObservations определяет
 минимальное количество наблюдений в общем периоде агрегируемых данных.


## Комментарии


Свойство актуально, если накладывается ограничение на минимальное количество
 наблюдений в общем периоде агрегируемых данных, т.е. [IMsCrossDimensionAggregationOptions.UseNumObservations](IMsCrossDimensionAggregationOptions.UseNumObservations.htm)
 принимает значение True.


Значение по умолчанию - три наблюдения.
