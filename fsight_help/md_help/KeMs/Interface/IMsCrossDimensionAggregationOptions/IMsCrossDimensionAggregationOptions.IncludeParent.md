# IMsCrossDimensionAggregationOptions.IncludeParent


## Синтаксис


		IncludeParent: Boolean;


## Описание


Свойство IncludeParent
 определяет признак учёта исходного элемента иерархии в агрегации.


## Комментарии


Значение свойства IncludeParent
 будет учитываться в расчёте, если свойство [IMsCrossDimensionAggregationOptions.FollowHierarchy](IMsCrossDimensionAggregationOptions.FollowHierarchy.htm)
 имеет значение True.


Допустимые значения:


	- True. Учитывать исходный
	 элемент иерархии;


	- False. По умолчанию.
	 Не учитывать исходный элемент иерархии.
