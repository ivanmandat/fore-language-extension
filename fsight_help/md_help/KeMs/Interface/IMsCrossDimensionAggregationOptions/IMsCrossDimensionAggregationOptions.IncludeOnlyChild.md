# IMsCrossDimensionAggregationOptions.IncludeOnlyChild


## Синтаксис


		IncludeOnlyChild: Boolean;


## Описание


Свойство IncludeOnlyChild определяет
 признак расчёта агрегации только на один уровень вниз по иерархии.


## Комментарии


Значение свойства будет учитываться в расчёте, если свойство [IMsCrossDimensionAggregationOptions.FollowHierarchy](IMsCrossDimensionAggregationOptions.FollowHierarchy.htm)
 имеет значение True.


Допустимые значения:


	- True. Агрегация рассчитывается
	 только на один уровень вниз по иерархии;


	- False. По умолчанию.
	 Агрегация рассчитывается по всем уровням иерархии.
