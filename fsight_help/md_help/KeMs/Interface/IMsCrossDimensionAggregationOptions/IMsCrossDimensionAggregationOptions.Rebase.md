# IMsCrossDimensionAggregationOptions.Rebase


## Синтаксис


		Rebase: Boolean;


## Описание


Свойство Rebase определяет признак
 использования функции Rebase для расчёта агрегации.


## Комментарии


Функция Rebase позволяет «перебазировать» данные с одной динамики на
 другую.


Допустимые значения:


	- True.
	 Использовать функцию Rebase для расчёта агрегации;


	- False.
	 Не использовать функцию Rebase для расчета агрегации.
