# IMsCrossDimensionAggregationOptions.FillGapsUserMethodParams


## Синтаксис


		FillGapsUserMethodParams: [IMsUserMethodParams](../IMsUserMethodParams/IMsUserMethodParams.htm);


## Описание


Свойство FillGapsUserMethodParams
 возвращает параметры пользовательского метода обработки пропусков.


## Комментарии


Первым параметром пользовательского метода должен передаваться ряд,
 представленный интерфейсом [ITimeSeries](../ITimeSeries/ITimeSeries.htm).
 Остальные параметры опциональные, их задаёт пользователь.
