# IMsCrossDimensionAggregationOptions.FollowHierarchy


## Синтаксис


		FollowHierarchy: Boolean;


## Описание


Свойство FollowHierarchy
 определяет признак учёта иерархии при выполнении расчёта агрегации.


## Комментарии


Допустимые значения:


	- True. Агрегация рассчитывается
	 с учётом иерархии. С помощью свойств [IMsCrossDimensionAggregationOptions.IncludeOnlyChild](IMsCrossDimensionAggregationOptions.IncludeOnlyChild.htm)
	 и [IMsCrossDimensionAggregationOptions.IncludeParent](IMsCrossDimensionAggregationOptions.IncludeParent.htm)
	 доступно задание дополнительных параметров расчёта;


	- False. По умолчанию.
	 Агрегация рассчитывается без учёта иерархии.
