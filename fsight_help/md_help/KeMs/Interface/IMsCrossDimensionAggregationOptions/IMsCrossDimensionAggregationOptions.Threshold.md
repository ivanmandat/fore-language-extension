# IMsCrossDimensionAggregationOptions.Threshold


## Синтаксис


		Threshold: Double;


## Описание


Свойство Threshold
 определяет значение порога для расчёта релевантности агрегации.


## Комментарии


Диапазон допустимых значений: [0, 100].


Если значение данного свойства больше нуля,
 то учитывается значение свойства [IMsCrossDimensionAggregationTransform.WeightsExpression](../IMsCrossDimensionAggregationTransform/IMsCrossDimensionAggregationTransform.WeightsExpression.htm).
