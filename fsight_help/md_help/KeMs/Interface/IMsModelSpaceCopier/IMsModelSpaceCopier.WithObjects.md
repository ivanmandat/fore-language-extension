# IMsModelSpaceCopier.WithObjects


## Синтаксис


WithObjects: Boolean;


## Описание


Свойство WithObjects определяет,
 копировать ли контейнер моделирования с вложенными объектами.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Копировать контейнер с вложенными объектами;


	- False. Копировать контейнер
	 без вложенных объектов. Значение свойства [IMsModelSpaceCopier.WithObjectsData](IMsModelSpaceCopier.WithObjectsData.htm)
	 игнорируется.
