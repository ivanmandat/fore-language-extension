# IMsModelSpaceCopier.WithObjectsData


## Синтаксис


WithObjectsData: Boolean;


## Описание


Свойство WithObjectsData определяет,
 копировать ли контейнер моделирования вместе с данными переменных.


## Комментарии


WithObjectsData учитывается,
 если [IMsModelSpaceCopier.WithObjects](IMsModelSpaceCopier.WithObjects.htm)
 установлено в True.


Допустимые значения:


	- True. По умолчанию.
	 Копировать контейнер вместе с данными переменных моделирования;


	- False. Копировать контейнер
	 с переменными моделирования, но без их данных.
