# IMsExpressionEdit.Rubricator


## Синтаксис


		Rubricator: [IRubricator](KeCubes.chm::/Interface/IRubricator/IRubricator.htm);


## Описание


Свойство Rubricator определяет
 базу данных временных рядов, с атрибутами которой можно работать в компоненте
 [MsExpressionEdit](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/MsExpressionEdit.htm).


## Комментарии


Отметка атрибутов, значения которых будут доступны в редакторе, указывается
 в свойстве [IMsExpressionEdit.SelectionSet](IMsExpressionEdit.SelectionSet.htm).


Также атрибуты могут быть добавлены с помощью метода [IMsExpressionEdit.AddAttribute](IMsExpressionEdit.AddAttribute.htm).
