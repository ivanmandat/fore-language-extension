# IMsExpressionEdit.ClearAttributes


## Синтаксис


		ClearAttributes;


## Описание


Метод ClearAttributes удаляет
 все атрибуты, добавленные для использования в компоненте.


## Комментарии


Метод удаляет все атрибуты, которые были добавлены с помощью метода
 [IMsExpressionEdit.AddAttribute](IMsExpressionEdit.AddAttribute.htm)
 либо получены при установке свойства [IMsExpressionEdit.SelectionSet](IMsExpressionEdit.SelectionSet.htm).
