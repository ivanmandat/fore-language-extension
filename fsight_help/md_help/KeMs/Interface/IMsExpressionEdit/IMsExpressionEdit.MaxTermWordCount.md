# IMsExpressionEdit.MaxTermWordCount


## Синтаксис


		MaxTermWordCount: Integer;


## Описание


Свойство MaxTermWordCount определяет
 максимально возможное количество слов в выражении.


## Комментарии


Значение по умолчанию - одно слово.


Разделитель слов терма определяет свойство [IMsExpressionEdit.LightDelimiters](IMsExpressionEdit.LightDelimiters.htm).
