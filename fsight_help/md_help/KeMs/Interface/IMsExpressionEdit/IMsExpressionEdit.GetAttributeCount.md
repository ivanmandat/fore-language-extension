# IMsExpressionEdit.GetAttributeCount


## Синтаксис


		GetAttributeCount: Integer;


## Описание


Метод GetAttributeCount возвращает
 число атрибутов, доступных для использования в компоненте.


## Комментарии


Для добавления атрибута используйте метод [IMsExpressionEdit.AddAttribute](IMsExpressionEdit.AddAttribute.htm),
 либо задайте свойство [IMsExpressionEdit.SelectionSet](IMsExpressionEdit.SelectionSet.htm).
