# IMsExpressionEdit.HeavyDelimiters


## Синтаксис


		HeavyDelimiters: String;


## Описание


Свойство HeavyDelimiters определяет
 разделитель выражений.


## Комментарии


Значение по умолчанию: «{Табуляция}{Возврат каретки, Перевод строки},./?<>;:"'{[}]~`%^&*()-+=!».
