# IMsExpressionEdit.UseFullVariableNames


## Синтаксис


		UseFullVariableNames: Boolean;


## Описание


Свойство UseFullVariableNames
 определяет признак использования полных наименований операндов.


## Комментарии


Допустимые значения:


	- True. Используются полные
	 наименования операндов;


	- False. Используются
	 сокращённые наименования операндов.
