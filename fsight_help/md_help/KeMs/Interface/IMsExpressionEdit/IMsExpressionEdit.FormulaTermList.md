# IMsExpressionEdit.FormulaTermList


## Синтаксис


		FormulaTermList: [IMsFormulaTermList](../IMsFormulaTermList/IMsFormulaTermList.htm);


## Описание


Свойство FormulaTermList определяет
 коллекцию термов, доступных в компоненте [MsExpressionEdit](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/MsExpressionEdit.htm).


## Комментарии


По умолчанию свойству установлено значение Null,
 при этом в редакторе не доступны какие-либо термы. Если задано свойство
 [IMsExpressionEdit.FormulaTransform](IMsExpressionEdit.FormulaTransform.htm),
 то в данном свойстве могут быть указаны термы соответствующей модели.
