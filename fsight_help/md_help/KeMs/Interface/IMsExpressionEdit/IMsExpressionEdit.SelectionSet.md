# IMsExpressionEdit.SelectionSet


## Синтаксис


		SelectionSet: [IDimSelectionSet](KeDims.chm::/interface/IDimSelectionSet/IDimSelectionSet.htm);


## Описание


Свойство SelectionSet определяет
 отметку атрибутов базы данных временных рядов.


## Комментарии


База данных временных рядов, для атрибутов которой определяется отметка,
 должна быть указана в свойстве [IMsExpressionEdit.Rubricator](IMsExpressionEdit.Rubricator.htm).
 После установки значения свойства SelectionSet
 значения атрибутов будут доступны в компоненте и будут использоваться
 для составления рядов.
