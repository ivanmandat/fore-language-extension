# IMsExpressionEdit.FormulaTransform


## Синтаксис


		FormulaTransform: [IMsFormulaTransform](../IMsFormulaTransform/IMsFormulaTransform.htm);


## Описание


Свойство FormulaTransform определяет
 параметры модели, на которую настроен компонент [MsExpressionEdit](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/MsExpressionEdit.htm).


## Комментарии


Параметры модели могут быть получены в свойстве [IMsModel.Transform](../IMsModel/IMsModel.Transform.htm).


По умолчанию свойству установлено значение Null,
 при этом редактор не связан с какой-либо модель.
