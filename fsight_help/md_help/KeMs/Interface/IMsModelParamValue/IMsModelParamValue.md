# IMsModelParamValue


Сборка: Ms;


## Описание


Интерфейс IMsModelParamValue
 содержит свойства для работы со значением параметра метамодели перед началом
 расчета задачи.


## Комментарии


Для работы с коллекцией значений параметров используйте интерфейс [IMsModelParamValues](../IMsModelParamValues/IMsModelParamValues.htm).
