# IMsOptimizationProblem.NodesCount


## Синтаксис


		NodesCount:
		 Integer;


## Описание


Свойство NodesCount определяет
 количество интервалов, на которые будет разбиваться диапазон, заданный
 для управляющих переменных.


## Комментарии


Значение по умолчанию: 5.


Свойство используется, если в качестве метода расчёта используется перебор
 по сетке. Для задания метода расчёта используйте свойство [IMsOptimizationProblem.MethodType](IMsOptimizationProblem.MethodType.htm).
