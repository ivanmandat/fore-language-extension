# IMsOptimizationProblem.MaxIterationsCount


## Синтаксис


		MaxIterationsCount:
		 Integer;


## Описание


Свойство MaxIterationsCount
 определяет максимальное количество итераций, за которое будет определяться
 оптимальное значение.


## Комментарии


Значение по умолчанию: 100.
