# IMsAggregationGroups


Сборка: Ms;


## Описание


Интерфейс IMsAggregationGroups
 содержит свойства и методы для работы с коллекцией групп агрегации.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMsAggregationGroups.Count.htm)
		 Свойство Count возвращает
		 количество групп в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMsAggregationGroups.Item.htm)
		 Свойство Item возвращает
		 группу из коллекции по её индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IMsAggregationGroups.Add.htm)
		 Метод Add осуществляет
		 добавление группы в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](IMsAggregationGroups.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех групп из коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IMsAggregationGroups.Remove.htm)
		 Метод Remove осуществляет
		 удаление группы из коллекции по её индексу.
