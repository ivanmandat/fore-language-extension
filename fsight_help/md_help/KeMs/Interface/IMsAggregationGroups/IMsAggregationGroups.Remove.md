# IMsAggregationGroups.Remove


## Синтаксис


		Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс
 группы в коллекции.


## Описание


Метод Remove удаляет группы
 из коллекции по её индексу.


## Комментарии


При удачном удалении группы метод возвращает True,
 при неудачном - False.
