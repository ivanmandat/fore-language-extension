# IMsFormulaTransformSlicesTree.Owner


## Синтаксис


Owner(Element: Integer): Integer;


## Параметры


Element. Индекс элемента, для
 которого необходимо определить индекс родительского элемента.


## Описание


Свойство Owner возвращает индекс
 родительского элемента.


## Комментарии


Свойство возвращает 0, если элемент с индексом Element является корневым.
