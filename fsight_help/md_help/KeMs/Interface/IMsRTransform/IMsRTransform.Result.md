# IMsRTransform.Result


## Синтаксис


Result: [IMsFormulaTerm](../IMsFormulaTerm/IMsFormulaTerm.htm);


## Описание


Свойство Result возвращает терм,
 соответствующий моделируемой переменной.


## Комментарии


Для получения элементов выражения, соответствующих входным переменным,
 используйте свойство [IMsRTransform.Inputs](IMsRTransform.Inputs.htm).
