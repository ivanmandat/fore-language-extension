# IMsRTransform.Inputs


## Синтаксис


Inputs: [IMsFormulaTermList](../IMsFormulaTermList/IMsFormulaTermList.htm);


## Описание


Свойство Inputs возвращает список
 термов, соответствующих входным переменным модели.


## Комментарии


Для получения терма, соответствующего выходной переменной, используйте
 свойство [IMsRTransform.Result](IMsRTransform.Result.htm).
