# IMsProblemDetails


Сборка: Ms;


## Описание


Интерфейс IMsProblemDetails
 предназначен для работы с параметрами расчета задачи моделирования.


## Комментарии


Данный интерфейс позволяет настроить параметры прогнозирования и идентификации.
 IMsProblemDetails является базовым для настройки параметров расчета [задач
 оптимизации](../IMsOptimizationProblem/IMsOptimizationProblem.htm).
