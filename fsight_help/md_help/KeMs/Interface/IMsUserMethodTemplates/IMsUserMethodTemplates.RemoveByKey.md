# IMsUserMethodTemplates.RemoveByKey


## Синтаксис


RemoveByKey(TemplateKey: Integer): Boolean;


## Параметры


TemplateKey. Ключ шаблона пользовательского метода.


## Описание


Метод RemoveByKey осуществляет удаление шаблона пользовательского метода, ключ которого передается посредством параметра TemplateKey.
