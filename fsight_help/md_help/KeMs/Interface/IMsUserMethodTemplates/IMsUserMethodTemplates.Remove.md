# IMsUserMethodTemplates.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс шаблона пользовательского метода.


## Описание


Метод Remove осуществляет удаление шаблона пользовательского метода, индекс которого передается посредством параметра Index.
