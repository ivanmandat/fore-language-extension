# IMsUserMethodTemplates


## Описание


Интерфейс IMsUserMethodTemplates
 содержит свойства и методы для работы с коллекцией шаблонов пользовательских
 методов контейнера моделирования.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMsUserMethodTemplates.Count.htm)
		 Свойство Count возвращает
		 количество шаблонов пользовательских методов контейнера моделирования.


		 ![](../../Property_Image.gif)
		 [Item](IMsUserMethodTemplates.Item.htm)
		 Свойство Item возвращает
		 параметры шаблона пользовательского метода.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IMsUserMethodTemplates.Add.htm)
		 Метод Add осуществляет
		 создание нового шаблона пользовательского метода.


		 ![](../../Sub_Image.gif)
		 [Clear](IMsUserMethodTemplates.Clear.htm)
		 Метод Clear осуществляет
		 очистку списка шаблонов пользовательских методов.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IMsUserMethodTemplates.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск шаблона пользовательского метода по ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](IMsUserMethodTemplates.Remove.htm)
		 Метод Remove осуществляет
		 удаление шаблона пользовательского метода по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IMsUserMethodTemplates.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление шаблона пользовательского метода по ключу.
