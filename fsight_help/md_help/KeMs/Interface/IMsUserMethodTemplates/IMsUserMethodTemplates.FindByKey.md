# IMsUserMethodTemplates.FindByKey


## Синтаксис


FindByKey(TemplateKey: Integer): [IMsUserMethodTemplate](../IMsUserMethodTemplate/IMsUserMethodTemplate.htm);


## Параметры


TemplateKey. Ключ шаблона пользовательского метода, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет поиск и возвращает параметры шаблона пользовательского метода, в случае удачного поиска.
