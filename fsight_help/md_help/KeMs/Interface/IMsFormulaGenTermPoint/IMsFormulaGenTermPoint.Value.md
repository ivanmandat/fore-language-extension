# IMsFormulaGenTermPoint.Value


## Синтаксис


Value(Index: Integer): Variant;


## Параметры


Index. Индекс значения.


## Описание


Свойство Value возвращает значение
 терма с указанным индексом.


## Комментарии


Параметр Index принимает значение
 больше нуля, если значение терма формируется путём агрегации данных.


Количество доступных значений возвращает свойство [IMsProblemCalculation.ValuesCount](IMsFormulaGenTermPoint.ValuesCount.htm).
