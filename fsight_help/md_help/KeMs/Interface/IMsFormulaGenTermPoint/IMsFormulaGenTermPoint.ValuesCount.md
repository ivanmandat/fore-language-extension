# IMsFormulaGenTermPoint.ValuesCount


## Синтаксис


ValuesCount: Integer;


## Описание


Свойство ValuesCount возвращает
 количество значений, используемых при формирования значения терма.


## Комментарии


Свойство актуально, если значение терма формируется путём агрегации
 данных. Для термов, которые ссылаются на определённые значения источника,
 свойство вернёт значение 0.
