# IValidationMissingData.Type


## Синтаксис


Type: [ValidationMissingDataType](../../Enums/ValidationMissingDataType.htm);


## Описание


Свойство Type определяет тип
 пропусков.


## Комментарии


По умолчанию выявляются все типы пропусков, то есть свойство имеет значение
 ValidationMissingDataType.All.
