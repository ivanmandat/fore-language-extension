# IMsParametrizedDimensions


## Описание


Интерфейс IMsParametrizedDimensions
 предназначен для работы с коллекцией параметризированных измерений.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMsParametrizedDimensions.Count.htm)
		 Свойство Count возвращает
		 количество измерений в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMsParametrizedDimensions.Item.htm)
		 Свойство Item возвращает
		 измерение из коллекции по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](IMsParametrizedDimensions.FindById.htm)
		 Метод FindById осуществляет
		 поиск измерения в коллекции по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IMsParametrizedDimensions.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск измерения в коллекции по ключу.
