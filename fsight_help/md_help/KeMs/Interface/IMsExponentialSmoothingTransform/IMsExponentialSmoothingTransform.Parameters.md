## Описание


Свойство Parameters возвращает
 параметры метода экспоненциального сглаживания.


## Комментарии


Параметры метода - это набор коэффициентов: «Альфа»,
 «Дельта», «Гамма»,
 «Фи».


Для автоматического подбора значений параметров используйте свойство
 [IMsExponentialSmoothingTransform.AutoSearch](IMsExponentialSmoothingTransform.AutoSearch.htm).
