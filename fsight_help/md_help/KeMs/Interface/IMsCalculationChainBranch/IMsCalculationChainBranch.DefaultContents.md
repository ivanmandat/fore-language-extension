# IMsCalculationChainBranch.DefaultContents


## Синтаксис


		DefaultContents: [IMsCalculationChainEntries](../IMsCalculationChainEntries/IMsCalculationChainEntries.htm);


## Описание


Свойство DefaultContents возвращает
 ветку, которая выполняется, если не выполнилась ни одна ветка из коллекции
 веток условия.


## Комментарии


Для получения коллекции веток условия используйте свойство [IMsCalculationChainBranch.CaseList](IMsCalculationChainBranch.CaseList.htm).
