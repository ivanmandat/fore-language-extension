# IValidationFilters.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ фильтра в коллекции.


## Описание


Метод RemoveByKey осуществляет удаление фильтра из коллекции по ключу. Возвращает True в случае удачного удаления.
