# IValidationFilters.ParamControl


## Синтаксис


ParamControl(Index: Integer): [IMsModelParamsControl](../IMsModelParamsControl/IMsModelParamsControl.htm);


## Параметры


Index. Индекс параметра.


## Описание


Свойство ParamControl определяет набор контролируемых параметров.


## Комментарии


Для получения параметра из набора задайте значение Index. Индексация параметров в наборе сквозная и начинается с нуля.
