# IValidationFilters


Сборка: Ms;


## Описание


Интерфейс IValidationFilters
 содержит свойства и методы для работы с коллекцией фильтров валидации.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IValidationFilters.Count.htm)
		 Свойство Count возвращает
		 количество фильтров в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IValidationFilters.Item.htm)
		 Свойство Item возвращает
		 фильтр из коллекции по его индексу.


		 ![](../../Property_Image.gif)
		 [ParamControl](IValidationFilters.ParamControl.htm)
		 Свойство ParamControl
		 определяет набор контролируемых параметров.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IValidationFilters.Add.htm)
		 Метод Add осуществляет
		 добавление фильтра в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](IValidationFilters.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех фильтров из коллекции.


		 ![](../../Sub_Image.gif)
		 [FindById](IValidationFilters.FindById.htm)
		 Метод FindById осуществляет
		 поиск фильтра в коллекции по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IValidationFilters.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск фильтра в коллекции по ключу.


		 ![](../../Sub_Image.gif)
		 [MoveTo](IValidationFilters.MoveTo.htm)
		 Метод MoveTo осуществляет
		 перемещение фильтра в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IValidationFilters.Remove.htm)
		 Метод Remove осуществляет
		 удаление фильтра из коллекции по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IValidationFilters.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление фильтра из коллекции по ключу.
