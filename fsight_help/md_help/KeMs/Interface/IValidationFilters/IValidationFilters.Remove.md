# IValidationFilters.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс фильтра в коллекции.


## Описание


Метод Remove осуществляет удаление фильтра из коллекции по индексу. Возвращает True в случае удачного удаления.
