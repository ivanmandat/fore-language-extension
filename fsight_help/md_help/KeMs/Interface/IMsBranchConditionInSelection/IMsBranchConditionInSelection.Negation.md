# IMsBranchConditionInSelection.Negation


## Синтаксис


		Negation: Boolean;


## Описание


Свойство Negation определяет
 признак отрицания условия.


## Комментарии


Допустимые значения:


	- True.
	 Условие отрицается. Ветка условия будет рассчитана, если элемент не
	 входит в группу;


	- False.
	 Условие не отрицается.
