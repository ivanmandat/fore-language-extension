# IMsVariables.RemoveByKey


## Синтаксис


RemoveByKey(VariableKey: Integer): Boolean;


## Параметры


VariableKey. Ключ переменной, которую необходимо удалить.


## Описание


Метод RemoveByKey осуществляет удаление переменной и возвращает True в случае удачного удаления.


Примечание. При удалении переменной из коллекции удаляются все элементы, в которых данная переменная используется.
