# IMsVariables.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс переменной, которую необходимо удалить.


## Описание


Метод Remove осуществляет удаление переменной и возвращает True в случае удачного удаления.


Примечание. При удалении переменной из коллекции удаляются все элементы, в которых данная переменная используется.
