# IMsVariables


## Описание


Интерфейс IMsVariables содержит
 свойства и методы коллекции переменных.


## Свойства, унаследованные от [IMsVariableList](../IMsVariableList/IMsVariableList.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](../IMsVariableList/IMsVariableList.Count.htm)
		 Свойство Count возвращает
		 количество переменных моделирования в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](../IMsVariableList/IMsVariableList.Item.htm)
		 Свойство Item возвращает
		 переменную моделирования.
