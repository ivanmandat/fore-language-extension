# IMsVariables.Add


## Синтаксис


Add(Variable: [IMsVariable](../IMsVariable/IMsVariable.htm)):
 Boolean;


## Параметры


Variable. Переменная моделирования,
 которую необходимо добавить в коллекции.


## Описание


Метод Add осуществляет добавление
 переменной моделирования в коллекцию.


Примечание.
 Специфика добавления переменных зависит от области, в которой она применяется.
