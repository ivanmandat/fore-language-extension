# IDmTargetDetails.Target


## Синтаксис


		Target: Integer;


## Описание


Свойство Target определяет
 индекс столбца источника данных, содержащего анализируемую информацию.


## Комментарии


Индексация столбцов сквозная и начинается с нуля.


Для определения источника данных для интеллектуального анализа используйте
 свойство [IDmMethod.InputDataSource](../IDmMethod/IDmMethod.InputDataSource.htm).
