# IDmTargetDetails


Сборка: Ms;


## Описание


Интерфейс IDmTargetDetails предназначен
 для работы с данными, исследуемыми с помощью интеллектуального анализа.


## Иерархия наследования


           [IDmMethodDetails](../IDmMethodDetails/IDmMethodDetails.htm)


           [IDmAttributesDetails](../IDmAttributesDetails/IDmAttributesDetails.htm)


           IDmTargetDetails


## Комментарии


Для доступа к работе с данными приведите объект, возвращаемый свойством
 [IDmMethod.Details](../IDmMethod/IDmMethod.Details.htm),
 к интерфейсу IDmAttributesDetails.
