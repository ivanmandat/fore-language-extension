# IMsForecastObject


Сборка: Ms;


## Описание


Интерфейс IMsForecastObject
 предназначен для работы с результатами расчёта прогнозного метода.


## Иерархия наследования


           [IMsResultObject](../IMsResultObject/IMsResultObject.htm)


           IMsForecastObject


## Комментарии


Данный интерфейс является базовым для результатов, полученных при расчёте
 прогнозного метода в детерминированном уравнении.
