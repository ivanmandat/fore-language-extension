# IMsCalculationTreeNode


Сборка: Ms;


## Описание


Интерфейс IMsCalculationTreeNode
 содержит свойства для работы с элементом дерева расчёта.


## Иерархия наследования


           IMsCalculationTreeNode


## Комментарии


Построение дерева возможно, т.к. каждый его элемент содержит данные
 о своём родителе (свойство [IMsCalculationTreeNode.Parent](IMsCalculationTreeNode.Parent.htm))
 и потомках (свойство [IMsCalculationTreeNode.Children](IMsCalculationTreeNode.Children.htm)).
