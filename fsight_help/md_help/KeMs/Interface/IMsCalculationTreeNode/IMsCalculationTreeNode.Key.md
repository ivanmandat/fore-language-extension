# IMsCalculationTreeNode.Key


## Синтаксис


		Key: Integer;


## Описание


Свойство Key возвращает ключ
 элемента.


## Комментарии


Ключ служит для идентификации элементов внутри веток расчёта задачи.
 Значение ключа присваивается элементу автоматически и не может быть изменено.
