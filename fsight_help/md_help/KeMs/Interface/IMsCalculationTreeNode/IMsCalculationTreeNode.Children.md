# IMsCalculationTreeNode.Children


## Синтаксис


		Children: [IMsCalculationTreeNodes](../IMsCalculationTreeNodes/IMsCalculationTreeNodes.htm);


## Описание


Свойство Children возвращает
 коллекцию потомков для элемента.


## Комментарии


Коллекция представляет собой упорядоченный набор, содержащий потомков
 данного элемента. Если у элемента нет потомков, то свойство возвращает
 пустую коллекцию элементов: [IMsCalculationTreeNodes.Count](../IMsCalculationTreeNodes/IMsCalculationTreeNodes.Count.htm)
 имеет значение ноль.
