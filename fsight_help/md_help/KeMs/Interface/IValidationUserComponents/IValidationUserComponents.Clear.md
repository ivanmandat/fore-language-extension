# IValidationUserComponents.Clear


## Синтаксис


Clear;


## Описание


Метод Clear очищает коллекцию.


## Комментарии


После вызова Clear значение свойства [IValidationUserComponents.Count](IValidationUserComponents.Count.htm) становится равным нулю.
