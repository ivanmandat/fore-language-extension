# IValidationUserComponents


Сборка: Ms;


## Описание


Интерфейс IValidationUserComponents
 предназначен для работы с коллекцией пользовательских компонентов валидации.


## Иерархия наследования


           IEnumerable


           IValidationUserComponents


## Комментарии


Элемент коллекции реализован интерфейсом [IValidationUserComponent](../IValidationUserComponent/IValidationUserComponent.htm).


Компоненты - это ряды, настройка которых позволяет проводить более эффективный
 анализ возникновения исключения в правиле валидации.
