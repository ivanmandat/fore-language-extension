# DataMiningMethod


Сборка: Ms;


## Описание


Класс DataMiningMethod предназначен для работы с методами интеллектуального анализа данных.


## Комментарии


Интеллектуальный анализ данных (Data Mining) - это процесс выявления скрытых фактов и взаимосвязей в больших массивах данных. Полученные данные могут использоваться для принятия решений в различных сферах человеческой деятельности.
