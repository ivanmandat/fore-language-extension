# MsDatePeriod


Сборка: Ms;


## Описание


Класс MsDatePeriod предназначен
 для настройки периода, состоящего из двух дат: начала и окончания периода.


## Комментарии


Даты периода могут быть заданы:


	- конкретными датами.
	 Используйте свойства [IMsDatePeriod.Start](../../Interface/IMsDatePeriod/IMsDatePeriod.Start.htm)
	 и [IMsDatePeriod.End_](../../Interface/IMsDatePeriod/IMsDatePeriod.End_.htm);


	- условиями, которые используются
	 для расчета дат начала и окончания периода. Используйте свойства
	 [IMsDatePeriod.AutoPeriod](../../Interface/IMsDatePeriod/IMsDatePeriod.AutoPeriod.htm), [IMsDatePeriod.StartOffset](../../Interface/IMsDatePeriod/IMsDatePeriod.StartOffset.htm), ;[IMsDatePeriod.EndOffset](../../Interface/IMsDatePeriod/IMsDatePeriod.EndOffset.htm).
