# MsTimeMatrix.CreateLevel


## Синтаксис


CreateLevel(CalendarLevel: [DimCalendarLevel](KeDims.chm::/Enums/DimCalendarLevel.htm));


## Параметры


CalendarLevel. Календарная
 динамика ряда.


## Описание


Конструктор CreateLevel создает
 матрицу с заданной календарной динамикой.


## Комментарии


Для матрицы без дополнительных параметров используйте конструктор по
 умолчанию: [MsTimeMatrix.Create](MsTimeMatrix.Create.htm).
