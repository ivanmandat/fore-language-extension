## Описание


Конструктор Create создает матрицу.


## Комментарии


Для создания матрицы с учётом дополнительных параметров используйте
 конструкторы [MsTimeMatrix.CreateEx](MsTimeMatrix.CreateEx.htm)
 и [MsTimeMatrix.CreateLevel](MsTimeMatrix.CreateLevel.htm).
