# ModelBox.OnUpdate


## Синтаксис


OnUpdate(Sender: Object; Args: [IModelBoxUpdateEventArgs](../../Interface/IModelBoxUpdateEventArgs/IModelBoxUpdateEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnUpdate наступает при
 изменении параметров модели. Событие наступает, если была изменена моделируемая
 переменная, вид модели, используемой для расчета, параметры спецификации
 модели и т.д.
