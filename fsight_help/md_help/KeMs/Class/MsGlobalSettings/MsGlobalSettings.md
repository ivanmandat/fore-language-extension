# MsGlobalSettings


Класс MsGlobalSettings содержит
 свойства для работы с глобальными настройками контейнера моделирования.


## Свойства, унаследованные от [IMsDisplaySettings](../../Interface/IMsDisplaySettings/IMsDisplaySettings.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DisplayAttributes](../../Interface/IMsDisplaySettings/IMsDisplaySettings.DisplayAttributes.htm)
		 Свойство DisplayAttributes
		 возвращает коллекцию настроек отображения атрибутов в строковом
		 представлении элементов выражения.


		 ![](../../Property_Image.gif)
		 [DisplayParentAs](../../Interface/IMsDisplaySettings/IMsDisplaySettings.DisplayParentAs.htm)
		 Свойство DisplayParentAs
		 определяет режим отображения родительского элемента в строковом
		 представлении элемента выражения.


		 ![](../../Property_Image.gif)
		 [DisplayTermsAs](../../Interface/IMsDisplaySettings/IMsDisplaySettings.DisplayTermsAs.htm)
		 Свойство DisplayTermsAs
		 определяет параметры отображения объектов.
