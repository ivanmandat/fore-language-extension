# MsCalendarSettings.Create


## Синтаксис


Create;


## Описание


Конструктор Create создает экземпляр объекта класса.


## Комментарии


Для каждого объекта, имеющего параметры календарного уровня, конструктор вызывается автоматически, поэтому дополнительный его вызов не требуется.
