# MsCalendarSettings


Сборка: Ms;


## Описание


Класс MsCalendarSettings предназначен
 для работы с параметрами календарного уровня.


## Комментарии


Параметры календарного уровня для пользовательского метода расчёта возвращает
 свойство [ITimeSeries.CalendarSettings](../../Interface/ITimeSeries/ITimeSeries.CalendarSettings.htm).
