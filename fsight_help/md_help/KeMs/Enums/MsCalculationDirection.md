# MsCalculationDirection


## Описание


Перечисление MsCalculationDirection
 содержит направления поточечного режима расчёта.


Используется следующим свойством:


	- [IMsFormulaTransform.CalculationDirection](../Interface/IMsFormulaTransform/IMsFormulaTransform.CalculationDirection.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Forward. Расчет от
		 начальной точки к конечной.


		 1
		 Backward. Расчет от
		 конечной точки к начальной.


## Комментарии


При поточечном режиме расчёт ведется последовательно в каждой точке
 переменной:


	- Forward. Сначала рассчитывается
	 выражение по первой точке, затем происходит расчет по второй точке
	 и т.д.;


	- Backward. Сначала рассчитывается
	 выражение по последней точке, затем происходит расчет по предпоследней
	 точке и т.д.
