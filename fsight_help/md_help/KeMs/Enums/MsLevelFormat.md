# MsLevelFormat


## Описание


Перечисление MsLevelFormat используется для определения режима отображения календарной динамики.


Используется следующим свойством:


-
[IMsMetaModelVisualController.LevelFormat](../Interface/IMsMetaModelVisualController/IMsMetaModelVisualController.LevelFormat.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Календарная динамика не отображается.


 1
 Short. Отображается аббревиатура календарной динамики.


 2
 Long. Отображается полное название календарной динамики.


## Комментарии


Возможные аббревиатуры календарной динамики:


-
A. Годовая;


-
S. Полугодовая;


-
Q. Квартальная;


-
M. Месячная;


-
D. Дневная.
