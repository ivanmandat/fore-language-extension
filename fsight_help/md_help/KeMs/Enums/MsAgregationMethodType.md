# MsAgregationMethodType


## Описание


Перечисление MsAgregationMethodType
 содержит методы расчета агрегации. Данный перечислимый тип используется
 следующими свойствами:


	- [IMsCrossDimensionAggregationTransform.AgregationMethod](../Interface/IMsCrossDimensionAggregationTransform/IMsCrossDimensionAggregationTransform.AgregationMethod.htm);


	- [IMsDetermAggregationTransform.AgregationMethod](../Interface/IMsDetermAggregationTransform/IMsDetermAggregationTransform.AgregationMethod.htm).
