# MsCollapseType


## Описание


Перечисление MsCollapseType содержит методы агрегации, применяемые к моделируемой переменной в модели «Коллапс». Данное перечисление используется следующими свойствами:


-
[IModelling.Collapse](../Interface/IModelling/IModelling.Collapse.htm);


-
[IMsCollapseTransform.MethodType](../Interface/IMsCollapseTransform/IMsCollapseTransform.MethodType.htm);


-
[IMsCollapseTransformationMethod.CollapseType](../Interface/IMsCollapseTransformationMethod/IMsCollapseTransformationMethod.CollapseType.htm);


-
[IMsPointwiseCollapseTransform.MethodType](../Interface/IMsPointwiseCollapseTransform/IMsPointwiseCollapseTransform.MethodType.htm);


-
[IValidationCrossFrequency.AggregationType](../Interface/IValidationCrossFrequency/IValidationCrossFrequency.AggregationType.htm).


## Описание


 Значение
 Краткое описание


 0
 Total. Берется сумма значений элементов динамики.


 1
 Average. Берется среднее значение элементов динамики.


 2
 Minimum. Берется минимальное значение элементов динамики.


 3
 Maximum. Берется максимальное значение элементов динамики.


 4
 First. Берется первое имеющееся значение элементов динамики.


 5
 Last. Берется последнее имеющееся значение элементов динамики.


 6
 Range. Берется разность между максимальным и минимальным значениями элементов динамики.


 7
 Spread. Берется стандартное отклонение по динамике. Стандартное отклонение - это мера того, насколько широко разбросаны значения элементов относительно их среднего.
