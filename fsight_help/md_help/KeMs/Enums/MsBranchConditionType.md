# MsBranchConditionType


## Описание


Перечисление MsBranchConditionType
 используется для определения типа условия выполнения ветки условия в цепочке
 расчета метамодели.


Используется следующими свойствами и методами:


	- [IMsBranchCondition.Type](../Interface/IMsBranchCondition/IMsBranchCondition.Type.htm);


	- [IMsBranchConditions.Add](../Interface/IMsBranchConditions/IMsBranchConditions.Add.htm).


## Описание


		 Значение
		 Краткое описание


		 0
		 Scalar. Скалярное условие.


		 1
		 InSelection. Вхождение
		 в отметку элементов.


		 2
		 InGroup. Вхождение
		 в группу элементов.


		 3
		 Expression. Выражение,
		 после вычисления дающее результат логического типа.


		 4
		 NotEmpty. Не пустой
		 ряд.
