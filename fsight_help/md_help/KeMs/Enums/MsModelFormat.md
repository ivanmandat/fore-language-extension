# MsModelFormat


## Описание


Перечисление MsModelFormat используется
 для определения формата объектов.


Используется следующими свойствами


	- [IMsMetaModel.Convert](../Interface/IMsMetaModel/IMsMetaModel.Convert.htm);


	- [IMsProblem.Convert](../Interface/IMsProblem/IMsProblem.Convert.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 WithObject. Объекты
		 являются отдельным объектом репозитория.


		 1
		 WithoutObject. Объекты
		 не являются отдельным объектом репозитория.
