# MsBPFOutputType


## Описание


Перечисление MsBPFOutputType содержит компоненты, выгружаемые после применения фильтра Бакстера-Кинга.


Используется следующими свойствами и методами:


-
[IModelling.Bpf](../Interface/IModelling/IModelling.Bpf.htm);


-
[IMsBandpassFilterTransform.OutputType](../Interface/IMsBandpassFilterTransform/IMsBandpassFilterTransform.OutputType.htm).

 Значение
 Краткое описание


 0

 1
 NonCyclicalSeries. Нециклическая компонента.
