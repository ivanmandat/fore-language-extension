# MsBreakpointKind


## Описание


Перечисление MsBreakpointKind содержит виды точек останова.


Используется следующими свойствами и методами:


-
[IMsBreakpoint.Kind](../Interface/IMsBreakpoint/IMsBreakpoint.Kind.htm);


-
[IMsBreakpoints.Add](../Interface/IMsBreakpoints/IMsBreakpoints.Add.htm).


## Описание


 Значение
 Краткое описание


 0
 ModelCalendar. Календарная точка.


 1
 Data. Точка в данных.


## Комментарии


Для работами с точками останова любого вида используется базовый интерфейс [IMsBreakpoint](../Interface/IMsBreakpoint/IMsBreakpoint.htm). Для работы с календарными точками предназначен интерфейс [IMsModelCalendarBreakpoint](../Interface/IMsModelCalendarBreakpoint/IMsModelCalendarBreakpoint.htm), для работы с точками в данных - интерфейс [IMsDataBreakpoint](../Interface/IMsDataBreakpoint/IMsDataBreakpoint.htm).
