# MsDisplayParent


## Описание


Перечисление MsDisplayParent
 используется для определения режима отображения родительского элемента
 в строковом представлении элемента выражения.


Используется следующим свойством:


	- [IMsDisplaySettings.DisplayParentAs](../Interface/IMsDisplaySettings/IMsDisplaySettings.DisplayParentAs.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Derive. Режим отображения
		 наследуется от родительского объекта.


		 1
		 Id. Отображается идентификатор.


		 16
		 Name. Отображается
		 наименование.


		 256
		 NoDisplay. Родительский
		 элемент не отображается.
