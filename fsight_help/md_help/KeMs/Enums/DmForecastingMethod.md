# DmForecastingMethod


## Описание


Перечисление DmForecastingMethod предназначено для определения метода прогнозирования.


Используется свойством:


-
[IDmForecasting.ForecastingMethod](../Interface/IDmForecasting/IDmForecasting.ForecastingMethod.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 GreyForecast. Грей-метод.


 1
 CurveEstimation. Подбор формы зависимости.


 2
 ExponentialSmoothing. Экспоненциальное сглаживание.
