# MsInversionLag


## Описание


Перечисление MsInversionLag
 содержит периоды, на которых будет осуществляться начальное преобразование
 переменной.


Используется следующими свойствами:


	- [IModelling.Ratio](../Interface/IModelling/IModelling.Ratio.htm).


## Описание


		 Значение
		 Краткое описание


		 0
		 PrecidingValue. Предыдущий
		 период.


		 1
		 PrecidingYear. Соответствующий
		 период предыдущего года.


		 2
		 EndOfPrecedingYear.
		 Конец предыдущего года.
