# MsCalculationType


## Описание


Перечисление MsCalculationType
 используется для определения режима расчета.


Используется следующими свойствами:


	- [IMsCalculationChainGroup.CalculationType](../Interface/IMsCalculationChainGroup/IMsCalculationChainGroup.CalculationType.htm);


	- [IMsFormulaTransform.CalculationType](../Interface/IMsFormulaTransform/IMsFormulaTransform.CalculationType.htm);


	- [IValidationCustom.CalculationType](../Interface/IValidationCustom/IValidationCustom.CalculationType.htm);


	- [IValidationMissingMetadata.CalculationType](../Interface/IValidationMissingMetadata/IValidationMissingMetadata.CalculationType.htm);


	- [IValidationRevisionMetadataComparison.CalculationType](../Interface/IValidationRevisionMetadataComparison/IValidationRevisionMetadataComparison.CalculationType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Pointwise. Расчёт выполняется
		 последовательно в каждой точке ряда: сначала формула рассчитывается
		 по первой точке, затем - по второй точке и т.д.


		 1
		 Serie. Расчёт выполняется
		 одновременно по всем точкам ряда.
