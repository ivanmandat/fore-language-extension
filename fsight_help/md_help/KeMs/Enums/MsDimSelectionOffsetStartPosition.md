# MsDimSelectionOffsetStartPosition


## Описание


Перечисление MsDimSelectionOffsetStartPosition
 используется для определения типа смещения элементов при настройке [связи](CalculationAlgorithm.chm::/Desktop/Calculation_block/Page_Sources.htm#link)
 измерения источника данных с измерением приёмника данных в [блоке
 расчёта](CalculationAlgorithm.chm::/Desktop/Calculation_block/Calculation_unit.htm) или [блоке
 контроля](CalculationAlgorithm.chm::/Desktop/Control_Block/Control_Block.htm).


Используется следующим свойством:


	- [ICalcBlockIteratorStubDimensionLink.StartPosition](KeAlgo.chm::/Interface/ICalcBlockIteratorStubDimensionLink/ICalcBlockIteratorStubDimensionLink.StartPosition.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Current. По умолчанию
		 - текущий элемент. Смещение отметки выполняется для каждого отмеченного
		 элемента, полученного по связи.


		 1
		 First. Первый элемент.
		 Смещение отметки выполняется только для первого отмеченного элемента,
		 полученного по связи. Отметка с остальных элементов снимается.
		 Если по связи получена отметка элементов, принадлежащих разным
		 родителям, то смещение отметки выполняется только для первого
		 дочернего элемента в каждой группе элементов, принадлежащих одному
		 родителю. Отметка с остальных дочерних элементов снимается.
		При использовании данного вида смещения сдвиг может быть только
		 положительным или нулевым.


		 2
		 Last. Последний элемент.
		 Смещение отметки выполняется только для последнего отмеченного
		 элемента, полученного по связи. Отметка с остальных элементов
		 снимается. Если по связи получена отметка элементов, принадлежащих
		 разным родителям, то смещение отметки выполняется только для последнего
		 дочернего элемента в каждой группе элементов, принадлежащих одному
		 родителю. Отметка с остальных дочерних элементов снимается.
		При использовании данного вида смещения сдвиг может быть только
		 отрицательным или нулевым.
