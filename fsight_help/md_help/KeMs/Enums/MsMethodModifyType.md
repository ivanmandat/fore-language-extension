# MsMethodModifyType


## Описание


Перечисление MsMethodModifyType
 содержит способы формирования выходного ряда.


Используется следующими свойством:


	- [IMsDeterministicTransform.ModifyType](../Interface/IMsDeterministicTransform/IMsDeterministicTransform.ModifyType.htm).


## Описание


		 Значение
		 Краткое описание


		 0
		 Replace. Выходной ряд
		 содержит только значения моделируемого ряда.


		 1
		 Update. Выходной ряд
		 содержит все значения моделируемого ряда. Пустые значения выходного
		 ряда заменяются значениями входного ряда.


		 2
		 Extend. Выходной ряд
		 содержит все значения входного ряда. Пустые значения выходного
		 ряда заменяются значениями моделируемого ряда.
