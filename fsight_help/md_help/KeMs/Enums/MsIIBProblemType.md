# MsIIBProblemType


## Описание


Перечисление MsIIBProblemType содержит типы решаемой задачи межотраслевого баланса.


Используется следующим свойством:


- [IMsInterindustryBalanceTransform.ProblemType](../Interface/IMsInterindustryBalanceTransform/IMsInterindustryBalanceTransform.ProblemType.htm).


## Описание


 Значение
 Краткое описание


 0
 DirectProblem - Прямая задача (нахождение конечного использования).


 1
 InverseProblem - Обратная задача (нахождение валового выпуска).
