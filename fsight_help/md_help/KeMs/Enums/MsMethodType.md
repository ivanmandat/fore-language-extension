# MsMethodType


## Описание


Перечисление MsMethodType содержит
 типы метода, используемого для расчета.


Используется следующим свойством:


	- [IMsMethod.Type](../Interface/IMsMethod/IMsMethod.Type.htm).


## Описание


		 Значение
		 Краткое описание


		 0
		 Deterministic. Детерминированный
		 метод. Предназначен для изучения влияния факторов на результативный
		 показатель, связь между которыми имеет функциональный характер.
		 То есть результативный показатель представлен в виде произведения,
		 алгебраической суммы или частного факторов.


		 1
		 Stochastic. Стохастический
		 метод. Предполагает вероятностный характер как прогноза, так и
		 связи между используемыми данными и прогнозными показателями.
		 Вероятность расчета точного прогноза определяется объемом эмпирических
		 данных, используемых при прогнозировании.


		 2
		 Vector. Векторный метод.
		 Предполагает измерение целевого показателя с помощью набора различных
		 частных показателей.
