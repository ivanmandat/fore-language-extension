# MsVariableTestType


## Описание


Перечисление MsVariableTestType используется для определения типа описательной статистики.


Используется следующими свойствами:


-
[IMsMetaModelVisualController.AddTestIncludedEntrie](../Interface/IMsMetaModelVisualController/IMsMetaModelVisualController.AddTestIncludedEntrie.htm);


-
[IMsVariableTest.Type](../Interface/IMsVariableTest/IMsVariableTest.Type.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 DF. Расширенный тест Дики-Фуллера.


 1
 Granger. Тест Гранжера.


 2
 Johansen. Тест Йохансена.


 3
 PairCorrelation. Парные коэффициенты корреляции.
