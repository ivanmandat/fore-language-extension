# MsOutputCensus1SeriesType


## Описание


Перечисление MsOutputCensus1SeriesType
 содержит типы выходного ряда, который необходимо получить при расчёте
 методом Census1.


Используется следующими свойствами и методами:


	- [IModelling.Census1](../Interface/IModelling/IModelling.Census1.htm).


## Допустимые значения


		 Значение

		 0

		 1
		 RatioDifferences. Разность
		 для аддитивной модели сезонности и отношение для мультипликативной
		 модели.

		 5

		 компонента.
