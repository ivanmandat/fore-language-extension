# MsCalculationPeriod


## Описание


Перечисление MsCalculationPeriod
 используется для определения периода действия формулы.


Используется свойством:


	- [IMsModel.CalculationPeriod](../Interface/IMsModel/IMsModel.CalculationPeriod.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Identification. Период
		 идентификации.


		 1
		 Forecast. Период прогнозирования.


		 2
		 Both. Весь период.
