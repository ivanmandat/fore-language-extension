# MsEquationIdentifiableState


## Описание


Перечисление MsEquationIdentifiableState
 содержит критерии идентифицируемости уравнения.


Используется следующим свойством:


	- [IMsLinearEquationsTransform.IdentifiableState](../Interface/IMsLinearEquationsTransform/IMsLinearEquationsTransform.IdentifiableState.htm).


## Описание


		 Значение
		 Краткое описание


		 0
		 NonIdentifiable. Уравнение
		 неидентифицируемо.


		 1
		 Identifiable. Уравнение
		 идентифицируемо.


		 2
		 SuperIdentifiable.
		 Уравнение сверхидентифицируемо.
