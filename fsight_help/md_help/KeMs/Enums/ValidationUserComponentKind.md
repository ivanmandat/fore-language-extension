# ValidationUserComponentKind


## Описание


Перечисление ValidationUserComponentKind содержит виды пользовательского компонента валидации.


Используется следующим свойством:


-
[IValidationUserComponent.Kind](../Interface/IValidationUserComponent/IValidationUserComponent.Kind.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Selection. Компонент настраивается по отметке.


 1
 Transform. Компонент настраивается по формуле.


## Комментарии


Selection. Для настройки отметки используется свойство [IValidationUserComponent.Selection](../Interface/IValidationUserComponent/IValidationUserComponent.Selection.htm).


Transform. Для настройки формулы используется свойство [IValidationUserComponent.Transform](../Interface/IValidationUserComponent/IValidationUserComponent.Transform.htm).
