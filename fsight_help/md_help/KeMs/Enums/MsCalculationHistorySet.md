# MsCalculationHistorySet


## Описание


Перечисление MsCalculationHistorySet
 используется для определения параметров ведения журнала расчета задачи
 моделирования.


Используется следующим свойством:


	- [IMsProblemCalculation.SaveHistoryMask](../Interface/IMsProblemCalculation/IMsProblemCalculation.SaveHistoryMask.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. История не сохраняется.


		 1
		 Log. История сохраняется
		 только в виде журнала.


		 2
		 Models. Сохраняются
		 только копии моделей, участвующих в задаче.


		 4
		 Dependencies. Сохраняются
		 зависимости (только для моделей, основанных на атрибутах базы
		 данных временных рядов).


		 255
		 Full. Сохраняются копии
		 моделей и история в виде журнала.


## Комментарии


Особенности использования значения Dependencies:
 при расчёте задачи моделирования с использованием данного значения в базу
 данных в формате XML сохраняются параметры рассчитываемой модели ([IMsFormulaTransform](../Interface/IMsFormulaTransform/IMsFormulaTransform.htm)). В дальнейшем
 сохранённые параметры используются при расчёте валидации «[Анализ
 отклонения в компонентах](UiValidation.chm::/Validation_types/Component_mismatch.htm)»: они
 загружаются из XML и рассчитываются повторно, затем полученные значения
 сравниваются с сохранёнными значениями рядов.
