# MsWorkbookType


## Описание


Перечисление MsWorkbookType
 используется для определения типа данных в рабочей книге.


Используется следующим свойством:


	- [IMsMetaModelVisualController.Workbook](../Interface/IMsMetaModelVisualController/IMsMetaModelVisualController.Workbook.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Variable. Рабочая книга
		 содержит данные переменных.


		 1
		 Chain. Рабочая книга
		 содержит данные цепочки расчета метамодели.


		 2
		 TargetProblem. Рабочая
		 книга содержит данные целевой функции.


		 3
		 TargetConstraints.
		 Рабочая книга содержит данные ограничений целевой функции.
