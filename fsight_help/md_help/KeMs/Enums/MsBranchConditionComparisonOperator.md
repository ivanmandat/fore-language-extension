# MsBranchConditionComparisonOperator


## Описание


Перечисление MsBranchConditionComparisonOperator
 содержит операторы сравнения в условии ветки, входящей в цепочку расчета
 метамодели.


Используется следующим свойством:


	- [IMsBranchConditionScalar.ComparisonOperator](../Interface/IMsBranchConditionScalar/IMsBranchConditionScalar.ComparisonOperator.htm).


## Описание


		 Значение
		 Краткое описание


		 0
		 Equal. Проверяется
		 равенство значений.


		 1
		 More. Выявляется большее
		 из значений.


		 2
		 Less. Выявляется меньшее
		 из значений.


		 3
		 NotEqual. Проверяется,
		 что значения не равны.


		 4
		 Contains. Проверяется,
		 что первое значение содержит второе.


		 5
		 NotContains. Проверяется,
		 что первое значение не содержит второе.


		 6
		 ContainsIn. Проверяется,
		 что второе значение содержит первое.


		 7
		 NotContainsIn. Проверяется,
		 что второе значение не содержит первое.
