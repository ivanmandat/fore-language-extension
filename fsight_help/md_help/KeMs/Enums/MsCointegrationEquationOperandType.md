# MsCointegrationEquationOperandType


## Описание


Перечисление MsCointegrationEquationOperandType
 содержит группы коинтеграционных связей, в которые входит экзогенная переменная.


Используется следующим свойством:


	- [IMsCointegrationEquationTransform.OperandType](../Interface/IMsCointegrationEquationTransform/IMsCointegrationEquationTransform.OperandType.htm).


## Описание


		 Значение
		 Краткое описание


		 0
		 Common. Группа 2 (регрессионное
		 уравнение). Переменная относится к группе, описывающей краткосрочные
		 коинтеграционные связи.


		 1
		 LongTerm. Группа 1
		 (уравнение долгосрочной динамики). Переменная относится к группе,
		 описывающей долгосрочные коинтеграционные связи.


		 2
		 Both. Переменная входит
		 в группу 1 и 2 одновременно. Тип связи для нее считается неопределенным.
