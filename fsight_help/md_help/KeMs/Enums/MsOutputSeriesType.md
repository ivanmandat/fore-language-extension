# MsOutputSeriesType


## Описание


Перечисление MsOutputSeriesType
 содержит составляющие, полученные после сезонной декомпозиций и корректировки
 данных методом «[X11](Lib.chm::/02_Time_series_analysis/UiModelling_Census2.htm)».


Примечание.
 Метод X11 поддерживается только в ОС Windows.


Используется следующими свойствами и методами:


	- [IModelling.X11](../Interface/IModelling/IModelling.X11.htm).


## Описание


		 Значение
		 Краткое описание


		 0
		 B1. Ряд после априорной
		 корректировки или исходный ряд.


		 1
		 D10. Сезонная составляющая.


		 2
		 D11. Сезонная корректировка
		 ряда.


		 3
		 D12. Тренд-циклическая
		 компонента.


		 4
		 D13. Нерегулярная компонента.
