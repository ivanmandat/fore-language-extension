# DmFieldType


## Описание


Перечисление DmFieldType
 используется для определения типа данных поля.


Используется следующим свойством:


	- [IDmField.FieldType](../Interface/IDmField/IDmField.FieldType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 String. Строковые данные.


		 1
		 Numeric. Числовые данные.


		 2
		 Date. Данные типа «Дата».


		 3
		 Integer. Целочисленные
		 данные.
