# MsBoundFixType


## Описание


Перечисление MsBoundFixType
 используется для определения режима фиксации границы ограничения.


Используется следующими свойствами:


	- [IMsTargetConstraint.LowerBoundFixed](../Interface/IMsTargetConstraint/IMsTargetConstraint.LowerBoundFixed.htm);


	- [IMsTargetConstraint.UpperBoundFixed](../Interface/IMsTargetConstraint/IMsTargetConstraint.UpperBoundFixed.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Undefined. Фиксация
		 границы не задана.


		 1
		 Fixed. Граница зафиксирована.
		 Она не может быть изменена при автоматическом подборе.


		 2
		 NotFixed. Граница ослаблена.
