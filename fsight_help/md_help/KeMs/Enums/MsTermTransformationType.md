# MsTermTransformationType


## Описание


Перечисление MsTermTransformationType содержит методы преобразования терма. Данный перечислимый тип используется следующими свойствами и методами:


-
[IMsFormulaTermInfo.CreateTermTransformationMethod](../Interface/IMsFormulaTermInfo/IMsFormulaTermInfo.CreateTermTransformationMethod.htm);


-
[IMsFormulaTermTransformationMethod.Kind](../Interface/IMsFormulaTermTransformationMethod/IMsFormulaTermTransformationMethod.Kind.htm).


## Описание


 Значение
 Краткое описание


 0
 Collapse. Коллапс (агрегация данных с нижнего уровня на верхний).


 1
 Interpolate. Интерполяция (дезагрегация данных с верхнего уровня на нижний).
