# MsCalculationChainEntryType


## Описание


Перечисление MsCalculationChainEntryType
 используется для определения типа объекта, входящего в цепочку расчёта
 метамодели.


Используется следующим свойством:


	- [IMsCalculationChainEntry.Type](../Interface/IMsCalculationChainEntry/IMsCalculationChainEntry.Type.htm);


	- [IMsFindFormulaResult.Type](../Interface/IMsFindFormulaResult/IMsFindFormulaResult.Type.htm).
