# MsCalculationLogRecordType


## Описание


Перечисление MsCalculationLogRecordType содержит типы записи в журнале расчета задачи.


Используется следующим свойством:


-
[IMsCalculationLogRecord.Type](../Interface/IMsCalculationLogRecord/IMsCalculationLogRecord.Type.htm).


## Описание


 Значение
 Краткое описание


 0
 Information. Информационное сообщение.


 1
 Error. Информация об ошибке.


 2
 Warning. Предупреждение.


 3
 Validation. Нарушение ограничения.
