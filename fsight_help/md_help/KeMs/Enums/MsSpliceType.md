# MsSpliceType


## Описание


Перечисление MsSpliceType содержит методы совмещения рядов.


Используется следующими свойствами и методами:


-
[IModelling.Splice](../Interface/IModelling/IModelling.Splice.htm);


-
[IMsSpliceTransform.MethodType](../Interface/IMsSpliceTransform/IMsSpliceTransform.MethodType.htm).


## Описание


 Значение
 Краткое описание


 0
 Ratio. Метод сращивания данных с темпами роста.


 1
 Butt. Метод сращивания без преобразования данных.


 2
 Pch. Метод сращивания данных с темпами прироста.
