# MsFormulaTermType


## Описание


Перечисление MsFormulaTermType
 содержит типы терма. В соответствии с данным типом будут передаваться
 определенные данные при расчете модели.


Используется следующим свойством:


	- [IMsFormulaTermInfo.Type](../Interface/IMsFormulaTermInfo/IMsFormulaTermInfo.Type.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Pointwise. Поточечный
		 терм.


		 1
		 Date. Терм является
		 значением на указанную дату.


		 2
		 Serie. Терм является
		 рядом.


		 3
		 ParamDate. Терм является
		 календарным параметром.


		 4
		 SliceDate. Терм является
		 срезом данных.


## Комментарии


При расчете модели терм типа Pointwise
 будет передаваться как значение, соответствующее дате точки расчета.


При расчете модели терм типа Date
 будет передаваться как значение, соответствующее указанной точке расчета.


При расчете модели терм типа Serie
 будет передаваться как ряд, соответствующий указанному срезу переменной.


Терм ParamDate используется
 для параметризации выражений и представляет собой дату.


Терм типа SliceDate используется
 для поддержки в выражениях дополнительных атрибутов базы данных временных
 рядов при работе с контейнером моделирования, принадлежащим каталогу.
