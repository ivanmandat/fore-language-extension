# MsFormulaGenerationErrorType


## Описание


Перечисление MsFormulaGenerationErrorType
 содержит типы ошибок, которые могут возникнуть при генерации формул.


Используется следующими свойствами и методами:


	- [IMsFormulaGenInstanceIterator.ErrorType](../Interface/IMsFormulaGenInstanceIterator/IMsFormulaGenInstanceIterator.ErrorType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 NoError. Ошибки отсутствуют,
		 генерация формулы прошла успешно.


		 1
		 CycleFormula. В процессе
		 генерации формулы обнаружена циклическая зависимость.
