# MsCasewise


## Описание


Перечисление MsCasewise используется
 для определения метода обработки пропусков.


Используется следующими свойствами и методами:


	- [IModelling.Arima](../Interface/IModelling/IModelling.Arima.htm);


	- [IModelling.Ecm](../Interface/IModelling/IModelling.Ecm.htm);


	- [IModelling.Expsmooth](../Interface/IModelling/IModelling.Expsmooth.htm);


	- [IModelling.Extrapolate](../Interface/IModelling/IModelling.Extrapolate.htm);


	- [IModelling.GreyForecast](../Interface/IModelling/IModelling.GreyForecast.htm);


	- [IModelling.MedianSmooth](../Interface/IModelling/IModelling.MedianSmooth.htm);


	- [IModelling.MedianSmoothR](../Interface/IModelling/IModelling.MedianSmoothR.htm);


	- [IModelling.MovAvg](../Interface/IModelling/IModelling.MovAvg.htm);


	- [IModelling.Ols](../Interface/IModelling/IModelling.Ols.htm);


	- [IModelling.Tsls](../Interface/IModelling/IModelling.Tsls.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 No. Обработка пропусков
		 не используется.


		 1
		 Yes. Расчёт выполняется
		 с применением обработки пропусков методом Casewise.
