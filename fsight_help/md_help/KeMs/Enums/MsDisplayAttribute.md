# MsDisplayAttribute


## Описание


Перечисление MsDisplayAttribute
 содержит параметры отображения объектов контейнера моделирования.


Используется следующим свойством:


	- [IMsDisplaySettings.DisplayTermsAs](../Interface/IMsDisplaySettings/IMsDisplaySettings.DisplayTermsAs.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Derive. Используются
		 глобальные параметры отображения.


		 1
		 Id. Отображается идентификатор.


		 16
		 Name. Отображается
		 наименование.


		 256
		 UseMnemo. Отображается
		 мнемоника.


## Комментарии


Derive. Данное значение
 нельзя устанавливать для интерфейса [IMsGlobalSettings](../Interface/IMsGlobalSettings/IMsGlobalSettings.htm).
