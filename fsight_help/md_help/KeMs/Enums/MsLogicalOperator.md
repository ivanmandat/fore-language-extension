# MsLogicalOperator


## Описание


Перечисление MsLogicalOperator
 используется для определения логического оператора при объединении условий
 [расширенного
 фильтра](uinav.chm::/GUI/UFE/AttributeFiltering.htm).


Используется следующими свойствами:


	- [IMsDimIteratorAdvancedFilterCondition.ConditionJoin](../Interface/IMsDimIteratorAdvancedFilterCondition/IMsDimIteratorAdvancedFilterCondition.ConditionJoin.htm);


	- [IMsDimIteratorAdvancedFilterConditionGroup.ConditionJoin](../Interface/IMsDimIteratorAdvancedFilterConditionGroup/IMsDimIteratorAdvancedFilterConditionGroup.ConditionJoin.htm).

		 Значение

		 0
