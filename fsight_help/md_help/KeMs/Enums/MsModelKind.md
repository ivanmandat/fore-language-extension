# MsModelKind


## Описание


Перечисление MsModelKind используется
 для определения типа модели.


Используется следующим свойством:


	- [IMsFormulaTransform.Kind](../Interface/IMsFormulaTransform/IMsFormulaTransform.Kind.htm).


## Комментарии


Математическое описание моделей приведено в разделе «[Библиотека
 методов и моделей](Lib.chm::/uimodelling_lib_common.htm)».


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Simple. Стандартная
		 модель.


		 1
		 NonLinearEquations.
		 Система уравнений.


		 2
		 InterindustryBalance.
		 Межотраслевой баланс.


		 3
		 ErrorCorrectionModel.
		 Модель коррекции ошибок.


		 4
		 LinearEquations. Система
		 линейных уравнений.


		 5
		 PanelData. Регрессия
		 на панельных данных.


		 6
		 ValueAtRisk. Модель
		 «Value-At-Risk».


		 7
		 MatrixAggregation.
		 Модель матричной агрегации.
