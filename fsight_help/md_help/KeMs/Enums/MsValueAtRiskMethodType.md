# MsValueAtRiskMethodType


## Описание


Перечисление MsValueAtRiskMethodType содержит варианты расчёта модели Value-At-Risk.


Используется следующим свойством:


-
[IMsValueAtRiskTransform.MethodType](../Interface/IMsValueAtRiskTransform/IMsValueAtRiskTransform.MethodType.htm)

 MonteCarlo. Статистические испытания Монте-Карло.
