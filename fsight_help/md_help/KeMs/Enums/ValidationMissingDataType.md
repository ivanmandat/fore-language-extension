# ValidationMissingDataType


## Описание


Перечисление ValidationMissingDataType используется для определения типа пропусков.


Используется свойством:


-
[IValidationMissingData.Type](../Interface/IValidationMissingData/IValidationMissingData.Type.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 All. Все типы пропусков.


 1
 Leading. Пропуски, предшествующие данным.


 2
 Lagging. Пропуски, следующие за данными.


 3
 InData. Пропуски внутри ряда данных.
