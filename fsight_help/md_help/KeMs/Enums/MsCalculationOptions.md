# MsCalculationOptions


## Описание


Перечисление MsCalculationOptions
 используется для определения опций расчета модели.


Используется следующим свойством:


	- [IMsFormulaTransform.CalculationOption](../Interface/IMsFormulaTransform/IMsFormulaTransform.CalculationOption.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Опция отсутствует.


		 8
		 UseTransformPeriod.
		 Использовать периоды расчета модели.


		 16
		 UseExistingData. Использовать
		 все фактические данные входных переменных.


		 64
		 TreatNullsAsZeros.
		 Заменять пустые значения нулями.


		 16384
		 IntersectPeriods. Выполнять
		 пересечение периодов модели и её объекта-родителя.
