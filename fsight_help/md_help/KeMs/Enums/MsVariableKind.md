# MsVariableKind


## Описание


Перечисление MsVariableKind
 содержит типы переменных, которые необходимо получить. Данный перечислимый
 тип используется следующими свойствами:


	- [IMsProblem.VariableStubs](../Interface/IMsProblem/IMsProblem.VariableStubs.htm);


	- [IMsProblemVariablesReport.AddVariableKind](../Interface/IMsProblemVariablesReport/IMsProblemVariablesReport.AddVariableKind.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 -1
		 Unknow. Тип переменной
		 не определён.


		 0
		 All. Все переменные.


		 1
		 Input. Входные переменные.


		 2
		 Output. Выходные переменные.
