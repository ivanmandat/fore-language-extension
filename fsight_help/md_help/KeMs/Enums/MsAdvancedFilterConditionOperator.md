# MsAdvancedFilterConditionOperator


## Описание


Перечисление MsAdvancedFilterConditionOperator
 используется для определения оператора сравнения для условия фильтрации.


Используется следующим свойством:


	- [IMsDimIteratorAdvancedFilterCondition.Operator_](../Interface/IMsDimIteratorAdvancedFilterCondition/IMsDimIteratorAdvancedFilterCondition.Operator_.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Equal. Значение атрибута
		 должно быть равно указанному значению.


		 1
		 NotEqual. Значение
		 атрибута должно быть не равно указанному значению.


		 2
		 Greater. Значение атрибута
		 должно быть больше указанного значения.


		 3
		 GreaterOrEqual. Значение
		 атрибута должно быть больше или равно указанному значению.


		 4
		 Less. Значение атрибута
		 должно быть меньше указанного значения.


		 5
		 LessOrEqual. Значение
		 атрибута должно быть меньше или равно указанному значению.


		 6
		 Between. Значение атрибута
		 должно быть в указанном диапазоне значений.


		 7
		 In_. Значение атрибута
		 должно быть в указанном массиве значений.


		 8
		 StartsWith. Значение
		 атрибута должно начинаться со строки, указанной в значении. Только
		 для строковых атрибутов.


		 9
		 EndsWith. Значение
		 атрибута должно заканчиваться на строку, указанную в значении.
		 Только для строковых атрибутов.


		 10
		 Contains. Значение
		 атрибута должно содержать подстроку, указанную в значении. Только
		 для строковых атрибутов.
