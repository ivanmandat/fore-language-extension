# MsCensus2OutputType


## Описание


Перечисление MsCensus2OutputType
 используется для определения компоненты, выгружаемой в моделируемую переменную
 после расчёта метода «[X11](Lib.chm::/02_Time_series_analysis/UiModelling_Census2.htm)».


Примечание.
 Метод X11 поддерживается только в ОС Windows.


Используется следующим свойством:


	- [IMsCensus2Transform.OutputType](../Interface/IMsCensus2Transform/IMsCensus2Transform.OutputType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 B1. Ряд после априорной
		 корректировки или исходный ряд.


		 1
		 D10. Сезонная составляющая.


		 2
		 D11. Сезонная корректировка
		 ряда.


		 3
		 D12. Тренд-циклическая
		 компонента.


		 4
		 D13. Нерегулярная компонента.
