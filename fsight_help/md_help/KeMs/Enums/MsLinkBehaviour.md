# MsLinkBehaviour


## Описание


Перечисление MsLinkBehaviour
 используется для определения принципа работы [связи](CalculationAlgorithm.chm::/Desktop/Calculation_block/Page_Sources.htm#link)
 измерения источника данных с измерением приёмника данных в [блоке
 расчёта](CalculationAlgorithm.chm::/Desktop/Calculation_block/Calculation_unit.htm) или [блоке
 контроля](CalculationAlgorithm.chm::/Desktop/Control_Block/Control_Block.htm).


Используется следующим свойством:


	- [ICalcBlockIteratorStubDimensionLink.Behaviour](KeAlgo.chm::/Interface/ICalcBlockIteratorStubDimensionLink/ICalcBlockIteratorStubDimensionLink.Behaviour.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Default_. По умолчанию.
		 В связанном измерении источника данных будут оставлены только
		 те отмеченные элементы, для которых есть соответствующий отмеченный
		 элемент в связанном измерении приёмника данных. Если соответствующий
		 элемент в источнике не найден, то возникнет ошибка в расчёте.


		 1
		 EmptyCoordAsEmptyValue.
		 Пустая точка, если элемент не найден. В связанном измерении источника
		 данных будут оставлены только те отмеченные элементы, для которых
		 есть соответствующий отмеченный элемент в связанном измерении
		 приёмника данных. Если соответствующий элемент в источнике не
		 найден, то вместо него в расчёте будет использоваться значение
		 Null.


		 2
		 FilterDestination.
		 Фильтрация приёмника. В связанном измерении приёмника данных будут
		 оставлены только те отмеченные элементы, для которых есть соответствующий
		 отмеченный элемент в связанном измерении источника данных.
