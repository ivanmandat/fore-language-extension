# MsCumulativeMethodType


## Описание


Перечисление MsCumulativeMethodType используется для определения метода накопления.


Используется следующими свойствами:


-
[IModelling.Cumulative](../Interface/IModelling/IModelling.Cumulative.htm);


-
[IMsCumulativeTransform.CumulativeMethod](../Interface/IMsCumulativeTransform/IMsCumulativeTransform.CumulativeMethod.htm).


## Описание


 Значение
 Краткое описание


 0
 Average. Определяется среднее значение элементов диапазона.


 1
 Max. Определяется максимальное значение элемента диапазона.


 2
 Min. Определяется минимальное значение элемента диапазона.


 3
 Multiply. Определяется произведение значений элементов диапазона.


 4
 StDev. Определяется среднеквадратическое отклонение значений элементов диапазона.


 5
 Sum. Определяется сумма значений элементов диапазона.


 6
 Median. Определяется медиана значений элементов диапазона.


## Комментарии


Метод накопления - метод, с помощью которого будет происходить вычисление значений элементов моделируемого ряда на диапазоне значений входного ряда.
