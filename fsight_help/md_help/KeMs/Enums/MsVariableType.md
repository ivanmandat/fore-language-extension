# MsVariableType


## Описание


Перечисление MsVariableType
 содержит дополнительные параметры переменной моделирования. Перечисление
 является битовым, работа с элементами осуществляется с помощью [логических
 операций](Fore.chm::/07_Operations/Logical_operations.htm).


Используется следующим свойством:


	- [IMsVariable.Type](../Interface/IMsVariable/IMsVariable.Type.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 Loading. Загружать
		 данные в память перед расчетом задачи.


		 2
		 Saving. Выгружать полученные
		 данные после расчета.
