# MsTruncateMode


## Описание


Перечисление MsTruncateMode
 содержит режимы усечения.


Используется следующим методом:


	- [IModelling.Truncate](../Interface/IModelling/IModelling.Truncate.htm).


## Описание


		 Значение
		 Краткое описание


		 0
		 Remove. Отбрасываются
		 все значения ряда между датами начала и окончания усечения.


		 1
		 Keep. Отбрасываются
		 все значения ряда до даты начала усечения и после даты его окончания.
