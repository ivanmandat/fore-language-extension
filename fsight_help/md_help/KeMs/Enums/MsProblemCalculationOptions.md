# MsProblemCalculationOptions


## Описание


Перечисление MsProblemCalculationOptions
 используется для определения дополнительных параметров расчета задачи
 моделирования.


Используется следующим свойством:


	- [IMsProblemCalculationSettings.AdvancedOptions](../Interface/IMsProblemCalculationSettings/IMsProblemCalculationSettings.AdvancedOptions.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Дополнительные
		 параметры не используются.


		 1
		 SkipOutputDataLoad.
		 Не загружать данные по выходным переменным.


		 2
		 CacheCoords. Кешировать
		 координаты срезов переменных.


		 4
		 AllowEmptySelection.
		 Допускается пустая отметка в выходной переменной.


## Комментарии


Для применения нескольких дополнительных параметров указывайте необходимые
 значения перечисления MsProblemCalculationOptions
 через «Or».
