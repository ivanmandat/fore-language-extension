# DmExplanatoryType


## Описание


Перечисление DmExplanatoryType
 используется для определения [типа фактора](DataMining.chm::/Master/Edit_Attribute.htm),
 влияющего на анализируемый признак.


Используется свойством:


	- [IDmField.ExplanatoryType](../Interface/IDmField/IDmField.ExplanatoryType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Continuous. Количественный
		 тип.


		 1
		 Ordered. Порядковый
		 тип.


		 2
		 Categorical. Категориальный
		 тип.
