# MsCensus1OutputType


## Описание


Перечисление MsCensus1OutputType
 используется для определения компоненты, выгружаемой в моделируемую переменную
 после расчёта метода «[Census1](Lib.chm::/02_Time_series_analysis/UiModelling_Census1.htm)».


Используется следующим свойством:


	- [IMsCensus1Transform.OutputType](../Interface/IMsCensus1Transform/IMsCensus1Transform.OutputType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 MA. Сглаженный ряд.


		 1
		 RD. Разность для аддитивной
		 модели сезонности и отношение для мультипликативной модели.


		 2
		 S. Сезонная составляющая.


		 3
		 SA. Сезонная корректировка.


		 4
		 TC. Тренд-циклическая
		 компонента.


		 5
		 I. Нерегулярная компонента.
