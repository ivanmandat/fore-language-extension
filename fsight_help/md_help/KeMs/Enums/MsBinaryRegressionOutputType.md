# MsBinaryRegressionOutputType


## Описание


Перечисление MsBinaryRegressionOutputType
 используется для определения типа значений, выгружаемых в моделируемый
 и прогнозные ряды для модели бинарной регрессии.


Используется следующими свойством:


	- [IMsBinaryRegressionTransform.OutputType](../Interface/IMsBinaryRegressionTransform/IMsBinaryRegressionTransform.OutputType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Probability. Вероятностные
		 значения.


		 1
		 Binary. Бинарные значения.
