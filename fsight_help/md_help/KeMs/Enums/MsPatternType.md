# MsPatternType


## Описание


Перечисление MsPatternType содержит
 виды шаблонов, по которым производится интерполяция.


Используется следующими свойствами:


	- [IModelling.InterpolateP](../Interface/IModelling/IModelling.InterpolateP.htm);


	- [IMsInterpolateTransform.PatternType](../Interface/IMsInterpolateTransform/IMsInterpolateTransform.PatternType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Average. Среднее по
		 элементам.


		 1
		 First. По первому элементу.


		 2
		 Last. По последнему
		 элементу.
