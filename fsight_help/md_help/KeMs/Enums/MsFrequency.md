# MsFrequency


## Описание


Перечисление MsFrequency используется
 для определения календарной динамики для расчетов.


Используется следующими методами:


	- [IMsCustomTerm.IsPeriod](../Interface/IMsCustomTerm/IMsCustomTerm.IsPeriod.htm);


	- [IMsCustomTerm.Seas](../Interface/IMsCustomTerm/IMsCustomTerm.Seas.htm);


	- [IMsCustomTerm.Time](../Interface/IMsCustomTerm/IMsCustomTerm.Time.htm);


	- [IModelling.Collapse](../Interface/IModelling/IModelling.Collapse.htm);


	- [IModelling.Cumulative](../Interface/IModelling/IModelling.Cumulative.htm);


	- [IModelling.Interpolate](../Interface/IModelling/IModelling.Interpolate.htm);


	- [IModelling.InterpolateP](../Interface/IModelling/IModelling.InterpolateP.htm);


	- [IModelling.LevelIndexSeries](../Interface/IModelling/IModelling.LevelIndexSeries.htm);


	- [IModelling.YearSeries](../Interface/IModelling/IModelling.YearSeries.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Undefined. Динамика
		 не определена.


		 1
		 Annual. Годовая динамика.


		 2
		 Semiannual. Полугодовая
		 динамика.


		 4
		 Quarterly. Квартальная
		 динамика.


		 8
		 Monthly. Месячная динамика.


		 16
		 Daily. Дневная динамика.


		 32
		 Weekly. Недельная динамика.
