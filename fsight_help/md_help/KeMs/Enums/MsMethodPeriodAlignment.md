# MsMethodPeriodAlignment


## Описание


Перечисление MsMethodPeriodAlignment
 содержит варианты расчета метода относительно периода.


Используется следующими свойством:


	- [IMsMethod.PeriodAlignment](../Interface/IMsMethod/IMsMethod.PeriodAlignment.htm).


## Описание


		 Значение
		 Краткое описание


		 0
		 Beginning. Метод рассчитывается
		 на начало периода.


		 1
		 End_. Метод рассчитывается
		 на конец периода.
