# MsVariableDataFillType


## Описание


Перечисление MsVariableDataFillType
 содержит возможные способы наполнения данными переменной моделирования.


Используется следующими свойством:


	- [IMsVariable.DataFillType](../Interface/IMsVariable/IMsVariable.DataFillType.htm).


## Описание


		 Значение
		 Краткое описание


		 0
		 Manual. Заполнение
		 данными вручную.


		 1
		 DataSource. Заполнение
		 данными из указанного источника.


		 2
		 Formula. Расчет данных
		 по указанной формуле.
