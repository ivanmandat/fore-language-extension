# MsArrangeMode


## Описание


Перечисление MsArrangeMode используется
 для определения режима автопостроения цепочки расчета метамодели.


Используется следующим свойством:


	- [IMsCalculationChainEntries.ArrangeModels](../Interface/IMsCalculationChainEntries/IMsCalculationChainEntries.ArrangeModels.htm).


## Описание


		 Значение
		 Краткое описание


		 0
		 InsideFolders. По структуре
		 папок. При таком автопостроении папки выстраиваются вперед моделей
		 и в них также производится автопостроение.


		 1
		 DeleteFolders. С
		 нарушением структуры папок. При таком автопостроении все модели
		 всех дочерних папок извлекаются на уровень, для которого производится
		 автопостроение, а папки удаляются.


		 2
		 SplitSlices. Разделять
		 на части многомерные модели с циклами. Если в цепочке расчёта
		 присутствует многомерная модель, содержащая внутренние циклы,
		 то используется ее копия. Копия будет разбита на модели, соответствующие
		 измерениям, для которых задан метод расчёта. Данные модели будут
		 выстроены в порядке зависимости в папке «Цикл
		 N»,
		 где N - порядковый
		 номер цикла.


		 4
		 CreateSystems. Создавать
		 системы при выявлении циклов. Если уравнения линейной регрессии,
		 нелинейной регрессии, детерминированного уравнения образуют цикл,
		 то они объединяются в систему уравнений.


		 8
		 NoCycles. Не выделять
		 циклы в папки. Если в цепочке расчета присутствуют циклы, то модели,
		 образующие цикл, извлекаются из папок, соответствующих циклам,
		 в цепочку расчета вместо самих папок.


## Комментарии


Значение SplitSlices можно
 комбинировать с другими значениями перечисления MsArrangeMode.
 Для этого указывайте необходимые значения через оператор «Or».
