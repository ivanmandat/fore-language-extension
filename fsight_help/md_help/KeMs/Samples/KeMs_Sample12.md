# Настройка глобальных параметров отображения объектов в контейнере моделирования


В данном примере описан механизм установки глобальных параметров отображения
 объектов во всех контейнерах моделирования текущего репозитория.


Добавьте ссылку на системную сборку Ms.


			Sub UserProc;

Var

    GlobalSett: IMsGlobalSettings;

Begin

    GlobalSett := New MsGlobalSettings.Create As IMsGlobalSettings;

    GlobalSett.DisplayTermsAs := MsDisplayAttribute.Id;

End Sub UserProc;


После выполнения примера в текущем репозитории во всех контейнерах моделирования
 для объектов будут отображаться их идентификаторы.
