## С чего начать


		 Сборка
		 Java предназначена для использования прикладного кода, написанного
		 на языке программирования Java, в коде на Fore. Изучив
		 данный раздел, вы сможете с применением языка Fore обращаться
		 и работать с системными классами языка Java, вызывать
		 пользовательские функции Java, получать и обрабатывать результаты
		 их выполнения, работать с различными типами объектов, поддерживаемых
		 в языке Java.


		Более подробно с возможностями блока можно ознакомиться в разделе
		 «[Введение в сборку Java](Intro/Java_Intro.htm)».


		Описание ресурсов сборки представлено в следующих подразделах:


			- [интерфейсы
			 сборки Java](Interface/KeJava_Interface.htm);


			- [перечисления сборки
			 Java](Enums/KeJava_Enums.htm);


			- [классы сборки Java](Class/KeJava_Class.htm).


		 Перед началом работы рекомендуется
		 ознакомиться с разделами справки: «[Среда
		 разработки](UiDevEnv.chm::/01_Development_Environment/01_Purpose_of_the_constructor/Purpose_of_the_constructor.htm)», «[Руководство по языку Fore](Fore.chm::/Fore_Title.htm)».


		Информацию по языку Java можно получить в документации разработчика
		 на сайте [https://www.java.com/ru/download/help/index.html](https://www.java.com/ru/download/help/index.html).


		Также необходимо ознакомиться с правилами формирования сигнатур
		 в формате Java Native Interface (JNI). Документация доступна по
		 адресу [https://docs.oracle.com/javase/8/docs/technotes/guides/jni/spec/jniTOC.html](https://docs.oracle.com/javase/8/docs/technotes/guides/jni/spec/jniTOC.html).
