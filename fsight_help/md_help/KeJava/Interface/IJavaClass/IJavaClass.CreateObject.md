# IJavaClass.CreateObject


## Синтаксис


CreateObject(CtorSig: String, Params: Array): [IJavaClassObject](../IJavaClassObject/IJavaClassObject.htm);


## Параметры


CtorSig. JNI-сигнатура вызываемого
 конструктора класса;


Params. Массив параметров,
 передаваемых в конструктор.


## Описание


Метод CreateObject создаёт экземпляр
 Java-класса путём вызова конструктора класса.
