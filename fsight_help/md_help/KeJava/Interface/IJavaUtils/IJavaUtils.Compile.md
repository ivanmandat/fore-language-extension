# IJavaUtils.Compile


## Синтаксис


Compile(FilePath: String);


## Параметры


FilePath. Путь и наименование
 файла с пользовательскими классами (архива с пользовательскими классами).


## Описание


Метод Compile компилирует указанный
 файл с Java-кодом.


## Комментарии


Файл с расширением *.java компилируется в файл с расширением *.class
 для дальнейшей работы с Java-классом с помощью метода [IJavaUtils.Invoke](IJavaUtils.Invoke.htm).


Если во время компиляции возникнут ошибки, то они могут быть обработаны
 с помощью класса [JavaException](../../Class/JavaException/JavaException.htm).
