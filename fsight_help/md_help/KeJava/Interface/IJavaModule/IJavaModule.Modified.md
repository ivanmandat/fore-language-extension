# IJavaModule.Modified


## Синтаксис


Modified: Boolean;


## Описание


Свойство Modified определяет,
 имеются ли несохранённые изменения в тексте модуля.


## Комментарии


Допустимые значения:


	- True. В тексте модуля
	 имеются несохранённые изменения;


	- False. Текст модуля
	 не изменен.
