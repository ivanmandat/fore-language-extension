# Интерфейсы сборки Java


В рамках сборки Java существуют
 следующие интерфейсы:


		 Интерфейс


		 Краткое описание


		 ![](../Interface_Image.gif)
		 [IJavaClass](IJavaClass/IJavaClass.htm)
		 Интерфейс IJavaClass
		 предназначен для работы с классами на языке Java.


		 ![](../Interface_Image.gif)
		 [IJavaClassObject](IJavaClassObject/IJavaClassObject.htm)
		 Интерфейс IJavaClassObject
		 предназначен для работы с объектом, который является экземпляром
		 Java-класса.


		 ![](../Interface_Image.gif)
		 [IJavaException](IJavaException/IJavaException.htm)
		 Интерфейс IJavaException
		 описывает исключительные ситуации, которые могут возникнуть во
		 время выполнения методов на языке Java.


		 ![](../Interface_Image.gif)
		 [IJavaModule](IJavaModule/IJavaModule.htm)
		 Интерфейс IJavaModule
		 предназначен для работы с объектом среды разработки - [Java-модуль](UiDevEnv.chm::/01_Development_Environment/02_Work_in_Development_Environment/DevEnv_Object/Java_module.htm).


		 ![](../Interface_Image.gif)
		 [IJavaObject](IJavaObject/IJavaObject.htm)
		 Интерфейс IJavaObject
		 содержит базовые свойства Java-объектов.


		 ![](../Interface_Image.gif)
		 [IJavaUtils](IJavaUtils/IJavaUtils.htm)
		 Интерфейс IJavaUtils
		 предназначен для проверки Java-кода на наличие ошибок компиляции
		 и выполнения Java-методов.
