# IJavaClassObject.Field


## Синтаксис


Field(Name: String, Sig: String): Variant;


## Параметры


Name. Наименование поля;


Sig. JNI-сигнатура поля.


## Описание


Свойство Field определяет значение
 поля для экземпляра объекта Java-класса.
