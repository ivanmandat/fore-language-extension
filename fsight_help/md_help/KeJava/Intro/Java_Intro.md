# Введение в сборку Java


Java - кросс-платформенный язык программирования, применяемый для разработки
 в различных областях и сферах деятельности: Android приложения для мобильных
 устройств, серверные и веб-приложения, программные средства разработки,
 системы обработки больших данных, научные приложения и другие. Многофункциональность
 и гибкость языка может значительно расширить возможности приложений, разрабатываемых
 в «Форсайт. Аналитическая платформа».


В данном разделе представлено описание сборки Java, предназначенной
 для выполнения функций, реализованных на языке программирования Java.
 Для работы установите на компьютере Java SE Runtime Environment (JRE)
 и Java SE Development Kit (JDK) версии 8 или выше, произведите настройку.
 Более подробно об установке и настройке читайте в подразделе «[Подключение
 дополнительных модулей к «Форсайт. Аналитическая платформа](UiNav.chm::/02_Navigator/CommonSettings/Integration.htm)».


Основными интерфейсами сборки Java являются:


	- [IJavaUtils](../Interface/IJavaUtils/IJavaUtils.htm).
	 Интерфейс используется для выполнения функций на языке Java.


	- [IJavaClass](../Interface/IJavaClass/IJavaClass.htm).
	 Интерфейс используется для выполнения функций на языке Java, которые
	 реализованы в классах.


	- [IJavaClassObject](../Interface/IJavaClassObject/IJavaClassObject.htm).
	 Интерфейс для работы с экземплярами Java-классов.


Также для выполнения функций Java можно использовать статический метод
 [Invoke](../Class/Java/Java.Invoke.htm)
 или [InvokeModule](../Class/Java/Java.InvokeModule.htm) класса
 [Java](../Class/Java/Java.htm).


Остальные интерфейсы сборки предназначены для работы с объектами, которые
 имеют типы языка Java, и могут быть получены или переданы в функции Java.


Дополнительная информация по работе со сборкой представлена в следующих
 подразделах:
