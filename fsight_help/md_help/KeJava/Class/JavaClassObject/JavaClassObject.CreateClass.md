# JavaClassObject.CreateClass


## Синтаксис


CreateClass(ClassName: String, CstorSig: String,
 Params: Array);


## Параметры


CreateClass. Наименование класса
 в сигнатуре JNI;


CstorSig. JNI-сигнатура конструктора;


Params. Значения параметров,
 передаваемых в конструктор.


## Описание


Конструктор CreateClass создаёт
 объект, являющийся экземпляром Java-класса в соответствии с заданными
 параметрами.
