# Классы сборки Java


В рамках сборки Java существуют
 следующие классы:


		 Класс


		 Краткое описание


		 ![](../Class_Image.gif)
		 [Java](Java/Java.htm)
		 Класс Java содержит
		 статический метод, используемый для выполнения функций на языке
		 Java.


		 ![](../Class_Image.gif)
		 [JavaClass](JavaClass/JavaClass.htm)
		 Класс JavaClass реализует
		 объект, который является аналогом Java-класса.


		 ![](../Class_Image.gif)
		 [JavaClassObject](JavaClassObject/JavaClassObject.htm)
		 Класс JavaClassObject
		 реализует объект, который является экземпляром класса языка Java.


		 ![](../Class_Image.gif)
		 [JavaException](JavaException/JavaException.htm)
		 Класс JavaException
		 реализует экземпляр исключительной ситуации, которая может возникнуть
		 при выполнении функции Java.


		 ![](../Class_Image.gif)
		 [JavaObject](JavaObject/JavaObject.htm)
		 Класс JavaObject реализует
		 Java-объект.


		 ![](../Class_Image.gif)
		 [JavaUtils](JavaUtils/JavaUtils.htm)
		 Класс JavaUtils реализует
		 объект, предназначенный для выполнения функций на языке Java.
