# JavaClass.Create


## Синтаксис


Create(ClassName: String);


## Параметры


ClassName. Наименование класса
 в сигнатуре JNI.


## Описание


Конструктор Create создаёт объект,
 который является аналогом Java-класса с указанным наименованием.
