# Введение в сборку Python


Python - это язык программирования общего назначения, ориентированный
 на разработку приложений с высокой производительностью. Стандартные библиотеки
 Python, а также большое количество подключаемых сторонних модулей, позволяют
 значительно расширять спектр решаемых задач.


В данном разделе представлено описание сборки Python, предназначенной
 для выполнения функций, реализованных на языке программирования Python.
 Для работы установите на компьютере Python версии 3.5 и выше. Более подробно
 об установке читайте в подразделе «[Подключение
 дополнительных модулей к «Форсайт. Аналитическая платформа](UiNav.chm::/02_Navigator/CommonSettings/Integration.htm)».


Основными интерфейсами сборки Python являются:


	- [IPythonUtils](../Interface/IPythonUtils/IPythonUtils.htm).
	 Интерфейс используется для выполнения функций на языке Python.


	- [IPythonClassObject](../Interface/IPythonClassObject/IPythonClassObject.htm). Интерфейс
	 используется для выполнения функций на языке Python, которые реализованы
	 в классах.


Также для выполнения функций Python можно использовать статический метод
 [Invoke](../Class/Python/Python.Invoke.htm)
 или [InvokeModule](../Class/Python/Python.InvokeModule.htm)
 класса [Python](../Class/Python/Python.htm).


Остальные интерфейсы сборки предназначены для работы с объектами, которые
 имеют типы языка Python, и могут быть получены или переданы в функции
 Python.
