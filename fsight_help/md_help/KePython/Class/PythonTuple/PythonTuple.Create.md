# PythonTuple.Create


## Синтаксис


		Create(Params: Array);


## Параметры


Params.
 Массив значений, на базе которых необходимо создать кортеж.


## Описание


Конструктор Create создаёт кортеж
 на основании указанного массива значений.


## Комментарии


В массиве Params могут передаваться
 значения простых типов, другие массивы или различные объекты, имеющие
 типы из языка Python.
