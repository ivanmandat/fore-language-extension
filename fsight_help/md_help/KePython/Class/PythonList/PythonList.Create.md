# PythonList.Create


## Синтаксис


		Create(Params: Array);


## Параметры


Params. Массив значений, на
 базе которых необходимо создать список.


## Описание


Конструктор Create создаёт список
 на основании указанного массива значений.


## Комментарии


В массиве Params могут передаваться
 значения простых типов, другие массивы или различные объекты, имеющие
 типы из языка Python. Если передаётся многомерный массив, то будет сформирован
 многомерный список.
