# PythonDictionary.Create


## Синтаксис


Create(Params: Array);


## Параметры


Params. Массив значений, на
 базе которых необходимо создать словарь.


## Описание


Конструктор Create создаёт словарь
 на основании указанного массива значений.


## Комментарии


В массиве Params должен передаваться
 словарь, который является результатом выполнения функции на языке Python.
 Для создания нового словаря в коде на Fore передайте в качестве параметра
 Params значение Null
 и добавьте необходимые пары ключ-значение с помощью метода [IPythonDictionary.Update](../../Interface/IPythonDictionary/IPythonDictionary.Update.htm).
