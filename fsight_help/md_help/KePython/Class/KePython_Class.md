# Классы сборки Python


		 Класс
		 Краткое описание


		 ![](../Class_Image.gif)
		 [Python](Python/Python.htm)
		 Класс Python содержит
		 статические методы, используемые для выполнения функций на языке
		 Python.


		 ![](../Class_Image.gif)
		 [PythonClassObject](PythonClassObject/PythonClassObject.htm)
		 Класс PythonClassObject
		 реализует объект, который является экземпляром класса языка Python.


		 ![](../Class_Image.gif)
		 [PythonDictionary](PythonDictionary/PythonDictionary.htm)
		 Класс PythonDictionary
		 реализует объект, который является словарём языка Python.


		 ![](../Class_Image.gif)
		 [PythonException](PythonException/PythonException.htm)
		 Класс PythonException
		 реализует экземпляр исключительной ситуации, которая может возникнуть
		 при выполнении функции Python.


		 ![](../Class_Image.gif)
		 [PythonList](PythonList/PythonList.htm)
		 Класс PythonList реализует
		 объект, который является списком языка Python.


		 ![](../Class_Image.gif)
		 [PythonParameter](PythonParameter/PythonParameter.htm)
		 Класс PythonParameter
		 реализует именованный параметр, который может быть передан в функцию
		 Python.


		 ![](../Class_Image.gif)
		 [PythonSerializer](PythonSerializer/PythonSerializer.htm)
		 Класс PythonSerializer
		 реализует объект, предназначенный для сохранения и загрузки Python-модулей.


		 ![](../Class_Image.gif)
		 [PythonSet](PythonSet/PythonSet.htm)
		 Класс PythonSet реализует
		 объект, который является множеством языка Python.


		 ![](../Class_Image.gif)
		 [PythonTuple](PythonTuple/PythonTuple.htm)
		 Класс PythonTuple реализует
		 объект, который является кортежем языка Python.


		 ![](../Class_Image.gif)
		 [PythonUtils](PythonUtils/PythonUtils.htm)
		 Класс PythonUtils реализует
		 объект, предназначенный для выполнения функций на языке Python.


		 ![](../Class_Image.gif)
		 [PythonValueObject](PythonValueObject/PythonValueObject.htm)
		 Класс PythonValueObject
		 реализует объект, который может быть передан в функции на языке
		 Python.
