# PythonValueObject.Create


## Синтаксис


Create(Value:
 Variant);


## Параметры


Value. Значение, на базе которого
 создаётся объект.


## Описание


Конструктор Create создаёт объект
 Python на основании передаваемого значения.


## Комментарии


В качестве значения Value могут
 передаваться значения простых типов (String, Char, Integer, Double, Decimal,
 Currency, Boolean), которые сопоставимы с типами в Python, экземпляры
 [списков](../../Interface/IPythonList/IPythonList.htm), [кортежей](../../Interface/IPythonTuple/IPythonTuple.htm)
 или [экземпляры
 классов Python](../../Interface/IPythonClassObject/IPythonClassObject.htm).
