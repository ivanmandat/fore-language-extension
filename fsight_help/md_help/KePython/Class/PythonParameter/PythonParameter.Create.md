# PythonParameter.Create


## Синтаксис


		Create(Name: String; Value: [IPythonObject](../../Interface/IPythonObject/IPythonObject.htm));


## Параметры


Name. Наименование параметра.
 Наименование является регистрозависимым.


Value. Экземпляр объекта Python,
 который будет являться значением создаваемого параметра.


## Описание


Конструктор Create создаёт объект,
 который является именованным параметром с заданным значением.


## Комментарии


Для выполнения функции используйте методы [IPythonUtils.Invoke](../../Interface/IPythonUtils/IPythonUtils.Invoke.htm),
 [IPythonClassObject.Invoke](../../Interface/IPythonClassObject/IPythonClassObject.Invoke.htm),
 [Python.Invoke](../Python/Python.Invoke.htm).
