## С чего начать


		 Сборка
		 Python предназначена для использования прикладного кода, написанного
		 на языке программирования Python, в коде на Fore. Изучив
		 данный раздел, вы сможете с применением языка Fore вызывать
		 функции Python, получать и обрабатывать результаты их выполнения,
		 работать с различными типами объектов, поддерживаемых в языке
		 Python.


		Более подробно с возможностями блока можно ознакомиться в разделе
		 «[Введение в сборку Python](Intro/Python_Intro.htm)».


		Описание ресурсов сборки представлено в следующих подразделах:


			- [интерфейсы
			 сборки Python](Interface/KePython_Interface.htm);


			- [перечисления сборки
			 Python](Enums/KePython_Enums.htm);


			- [классы сборки Python](Class/KePython_Class.htm);


			- [примеры](Samples/Python_Samples.htm).


		 Перед началом работы рекомендуется
		 ознакомиться с разделами справки: «[Среда
		 разработки](UiDevEnv.chm::/01_Development_Environment/01_Purpose_of_the_constructor/Purpose_of_the_constructor.htm)», «[Руководство по языку Fore](Fore.chm::/Fore_Title.htm)».
