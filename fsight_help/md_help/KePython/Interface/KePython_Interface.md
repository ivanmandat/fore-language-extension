# Интерфейсы
 сборки Python


		 Интерфейс
		 Краткое описание


		 ![](../Interface_Image.gif)
		 [IPythonClassObject](IPythonClassObject/IPythonClassObject.htm)
		 Интерфейс IPythonClassObject
		 предназначен для работы с объектом, который является экземпляром
		 класса на языке Python.


		 ![](../Interface_Image.gif)
		 [IPythonDictionary](IPythonDictionary/IPythonDictionary.htm)
		 Интерфейс IPythonDictionary
		 предназначен для работы с типом данных языка Python - словарь.


		 ![](../Interface_Image.gif)
		 [IPythonException](IPythonException/IPythonException.htm)
		 Интерфейс IPythonException
		 описывает исключительные ситуации, которые могут возникнуть во
		 время выполнения функций на языке Python.


		 ![](../Interface_Image.gif)
		 [IPythonList](IPythonList/IPythonList.htm)
		 Интерфейс IPythonList предназначен
		 для работы с типом данных языка Python - список.


		 ![](../Interface_Image.gif)
		 [IPythonModule](IPythonModule/IPythonModule.htm)
		 Интерфейс IPythonModule
		 предназначен для работы с объектом среды разработки - [Python-модуль](UiDevEnv.chm::/01_Development_Environment/02_Work_in_Development_Environment/DevEnv_Object/python_module.htm).


		 ![](../Interface_Image.gif)
		 [IPythonObject](IPythonObject/IPythonObject.htm)
		 Интерфейс IPythonObject
		 содержит базовые свойства всех объектов, которые могут быть использованы
		 в функциях на языке Python.


		 ![](../Interface_Image.gif)
		 [IPythonParameter](IPythonParameter/IPythonParameter.htm)
		 Интерфейс IPythonParameter
		 предназначен для работы с именованным параметром функции на языке
		 Python.


		 ![](../Interface_Image.gif)
		 [IPythonSerializer](IPythonSerializer/IPythonSerializer.htm)
		 Интерфейс IPythonSerializer
		 содержит методы для сохранения и загрузки Python-модулей.


		 ![](../Interface_Image.gif)
		 [IPythonSerializerLoader](IPythonSerializerLoader/IPythonSerializerLoader.htm)
		 Интерфейс IPythonSerializerLoader
		 содержит свойства и методы для загрузки Python-модулей из файлов.


		 ![](../Interface_Image.gif)
		 [IPythonSet](IPythonSet/IPythonSet.htm)
		 Интерфейс IPythonSet
		 предназначен для работы с типом данных языка Python - множество.


		 ![](../Interface_Image.gif)
		 [IPythonTuple](IPythonTuple/IPythonTuple.htm)
		 Интерфейс IPythonTuple
		 предназначен для работы с типом данных языка Python - кортеж.


		 ![](../Interface_Image.gif)
		 [IPythonUtils](IPythonUtils/IPythonUtils.htm)
		 Интерфейс IPythonUtils
		 предназначен для выполнения функций на языке Python.


		 ![](../Interface_Image.gif)
		 [IPythonValueObject](IPythonValueObject/IPythonValueObject.htm)
		 Интерфейс IPythonValueObject
		 предназначен для работы с объектом, который может быть передан
		 в функции на языке Python.
