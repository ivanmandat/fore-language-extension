# IPythonModule.Modified


## Синтаксис


Modified: Boolean;


## Описание


Свойство Modified определяет,
 имеются ли несохранённые изменения в тексте модуля.


## Комментарии


Свойство возвращает значение True,
 если в тексте имеются изменения и False,
 если текст не изменялся.
