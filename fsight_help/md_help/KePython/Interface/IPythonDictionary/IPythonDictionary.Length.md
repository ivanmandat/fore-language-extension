# IPythonDictionary.Length


## Синтаксис


Length: Integer;


## Описание


Свойство Length возвращает количество
 элементов в словаре.


## Комментарии


Каждый элемент представляет собой пару ключ-значение. Доступ к значению
 осуществляется по ключу.
