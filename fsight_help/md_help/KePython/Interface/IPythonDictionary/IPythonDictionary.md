## Комментарии


Словарь в Python представляет собой неупорядоченную коллекцию различных
 объектов, доступ к которым осуществляется по ключу. Элементы в словаре
 можно изменять. Также можно добавлять новые пары ключ-значение или удалять
 существующие.


Для создания словаря Python предназначен Fore-класс - [PythonDictionary](../../Class/PythonDictionary/PythonDictionary.htm).
