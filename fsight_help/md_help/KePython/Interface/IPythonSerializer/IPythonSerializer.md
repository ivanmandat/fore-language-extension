# IPythonSerializer


Сборка: Python;


## Описание


Интерфейс IPythonSerializer
 содержит методы для сохранения и загрузки Python-модулей.


## Иерархия наследования


IPythonSerializer


## Комментарии


Для работы с Python-модулями:


	- Создайте экземпляр класса [PythonSerializer](../../Class/PythonSerializer/PythonSerializer.htm)
	 и приведите его к интерфейсу IPythonSerializer.


	- Выполните метод [CreateLoaderFromFile](IPythonSerializer.CreateLoaderFromFile.htm)
	 для получения объекта, который осуществляет загрузку Python-модулей
	 из файлов, или [SaveModuleToFile](IPythonSerializer.SaveModuleToFile.htm)
	 для сохранения Python-модуля в файл.
