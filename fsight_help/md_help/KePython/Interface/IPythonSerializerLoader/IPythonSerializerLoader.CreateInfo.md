# IPythonSerializerLoader.CreateInfo


## Синтаксис


CreateInfo: [IMetabaseObjectCreateInfo](KeSom.chm::/Interface/IMetabaseObjectCreateInfo/IMetabaseObjectCreateInfo.htm);


## Описание


Свойство CreateInfo возвращает
 информацию о создаваемом в репозитории объекте.


## Комментарии


Полученная информация может быть использована для изменения настроек
 создаваемого объекта (наименование, идентификатор и другие).
