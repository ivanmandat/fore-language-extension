# IPythonSerializerLoader


Сборка: Python;


## Описание


Интерфейс IPythonSerializerLoader
 содержит свойства и методы для загрузки Python-модулей из файлов.


## Иерархия наследования


IPythonSerializerLoader


## Комментарии


Объект для загрузки Python-модулей можно получить, используя метод [IPythonSerializer.CreateLoaderFromFile](../IPythonSerializer/IPythonSerializer.CreateLoaderFromFile.htm).
