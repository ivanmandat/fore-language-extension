# IPythonParameter


Сборка: Python;


## Описание


Интерфейс IPythonParameter предназначен
 для работы с именованным параметром функции на языке Python.


## Иерархия наследования


IPythonParameter


## Комментарии


[Именованные
 параметры](../../Intro/Use_Assembly_Python_In_Programm.htm#namedparameter) используются, если функция имеет параметры со значениями
 по умолчанию и при выполнении необходимо передать значения только определённых
 параметров.


Для создания нового параметра предназначен Fore-класс - [PythonParameter](../../Class/PythonParameter/PythonParameter.htm).
