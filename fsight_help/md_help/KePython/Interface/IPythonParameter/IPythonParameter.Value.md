# IPythonParameter.Value


## Синтаксис


		Value: [IPythonObject](../IPythonObject/IPythonObject.htm);


## Описание


Свойство Value определяет значение
 параметра.


## Комментарии


Значение параметра должно иметь тот тип, который определён в сигнатуре
 выполняемой функции.
