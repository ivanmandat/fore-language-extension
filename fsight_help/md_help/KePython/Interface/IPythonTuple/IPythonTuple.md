## Комментарии


Кортеж в Python представляет собой неизменяемый [список](../IPythonList/IPythonList.htm).
 Значения элементам в кортеже присваиваются в момент его инициализации
 и в дальнейшем не могут быть изменены. Также не могут быть добавлены новые
 элементы. Кортежи, в отличие от списков, занимают меньше оперативной памяти.


Для создания кортежа Python предназначен Fore-класс - [PythonTuple](../../Class/PythonTuple/PythonTuple.htm).
