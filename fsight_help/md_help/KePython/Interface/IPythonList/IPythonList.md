## Комментарии


Список в Python представляет собой упорядоченную коллекцию объектов
 одинаковых или различных типов. Элементы в списке могут изменять своё
 значение. Также в список могут добавляться новые элементы.


Для создания списка Python предназначен Fore-класс - [PythonList](../../Class/PythonList/PythonList.htm).
