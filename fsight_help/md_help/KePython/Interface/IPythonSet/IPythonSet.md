 для работы с типом данных языка Python - множество.

## Комментарии


Множество в Python представляет собой неупорядоченный набор элементов
 с уникальным значением. Элементы могут быть как добавлены, так и удалены.
 Если элемент уже существует во множестве, то повторно он не добавляется.
 Существующие элементы не могут быть изменены.


Для создания множества Python предназначен Fore-класс - [PythonSet](../../Class/PythonSet/PythonSet.htm).
