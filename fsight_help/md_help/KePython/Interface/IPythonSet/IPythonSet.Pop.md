# IPythonSet.Pop


## Синтаксис


Pop: Variant;


## Описание


Метод Pop удаляет первый элемент
 из множества и возвращает его значение.


## Комментарии


Ввиду того, что множество является неупорядоченным, то первый элемент
 может быть неизвестен.
