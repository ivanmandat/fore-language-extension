# IPythonValueObject


Сборка: Python;


## Описание


Интерфейс IPythonValueObject
 предназначен для работы с объектом, который может быть передан в функции
 на языке Python.


## Иерархия наследования


           [IPythonObject](../IPythonObject/IPythonObject.htm)


           IPythonValueObject


## Комментарии


Объект может содержать значения простых типов, сопоставимых с
 типами данных Fore. Для создания объекта предназначен Fore-класс
 - [PythonValueObject](../../Class/PythonValueObject/PythonValueObject.htm).
