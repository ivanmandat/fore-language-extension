# IPythonUtils.AddFolderToPythonPath


## Синтаксис


		AddFolderToPythonPath(FolderName: String);


## Параметры


FolderName. Путь к папке с
 модулями на языке Python.


## Описание


Метод AddFolderToPythonPath
 добавляет указанную папку в список папок для поиска модулей Python.


## Комментарии


По умолчанию поиск модулей производится в текущей папке, из которой
 была запущена платформа.


Метод AddFolderToPythonPath
 позволяет добавить в список поиска любую папку, к которой у пользователя
 имеется доступ. Для добавления нескольких папок выполните метод несколько
 раз с различными значениями параметра FolderName.
 Если в заданных папках имеются модули с одинаковым именем, а в этих модулях
 расположены функции с одинаковыми наименованиями, то при выполнении будет
 использоваться модуль и функция из папки, которая была добавлена с помощью
 метода AddFolderToPythonPath последней.
