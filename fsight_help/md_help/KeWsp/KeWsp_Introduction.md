# Введение в сборку Workspace


Основными интерфейсами сборки Workspace
 являются:


	- [IUiWorkspace](Interface/IUiWorkspace/IUiWorkspace.htm)
	 - интерфейс, описывающий компонента среды разработки [UiWorkspace](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiWorkspace.htm),
	 который предоставляет пустое рабочее пространство;


	- [IUiWorkspaceObject](Interface/IUiWorkspaceObject/IUiWorkspaceObject.htm)
	 - интерфейс, описывающий компонента среды разработки [UiWorkspaceObject](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiWorkspaceObject.htm),
	 который предоставляет доступ к рабочим пространствам репозитория;


	- [IWorkspaceBox](Interface/IWorkspaceBox/IWorkspaceBox.htm)
	 - интерфейс, описывающий компонента среды разработки [WorkspaceBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/WorkspaceBox.htm),
	 который предназначен для отображения рабочих пространств.
