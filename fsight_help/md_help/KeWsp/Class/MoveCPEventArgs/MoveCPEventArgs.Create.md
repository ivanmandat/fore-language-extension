# MoveCPEventArgs.Create


## Синтаксис


Create(Shape: [IWxShape](Andy.chm::/Interface/IWxShape/IWxShape.htm);
 CPIndex: Integer; Cancel: Boolean);


## Параметры


Shape - объект рабочего пространства,
 конечную точку которого необходимо переместить.


CPIndex - индекс конечной точки,
 которую необходимо переместить.


Cancel - параметр определяющий,
 будет ли происходить перемещение конечных точки.


## Описание


Конструктор Create создает аргумент
 события, происходящего перед перемещением одной из конечных точек линии
 связи.
