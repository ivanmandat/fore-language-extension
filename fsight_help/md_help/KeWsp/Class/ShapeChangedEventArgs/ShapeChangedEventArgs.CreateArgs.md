# ShapeChangedEventArgs.CreateArgs


## Синтаксис


CreateArgs(Shape: [IWxShape](Andy.chm::/Interface/IWxShape/IWxShape.htm);
 ShapeOperation: [WxShapeOperation](Andy.chm::/Enums/WxShapeOperation.htm));


## Параметры


Shape. Фигура рабочего пространства,
 для которой наступает события.


ShapeOperation. Действие с
 фигурой, в результате которого наступает событие.


## Описание


Конструктор CreateArgs создает
 аргумент события, наступающего после каких-либо изменений, связанных с
 фигурой рабочего пространства.
