# IWorkspace.Data


## Синтаксис


Data(Name: String): Variant;


## Параметры


Name. Идентификатор временной
 переменной, в которой будут храниться значения.


## Описание


Свойство Data предназначено
 для хранения пользовательской информации во время выполнения модуля/формы.


## Комментарии


В данном свойстве могут содержаться любые данные.
