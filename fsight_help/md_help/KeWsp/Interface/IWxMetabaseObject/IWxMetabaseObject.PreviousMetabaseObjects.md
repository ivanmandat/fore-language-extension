# IWxMetabaseObject.PreviousMetabaseObjects


## Синтаксис


PreviousMetabaseObjects(Index: Integer): [IWxMetabaseObject](IWxMetabaseObject.htm);


## Параметры


Index -
 индекс объекта, зависящего от данного объекта.


## Описание


Свойство PreviousMetabaseObjects
 возвращает объект, зависящий от данного объекта. Индекс объекта передается
 посредством параметра Index.
