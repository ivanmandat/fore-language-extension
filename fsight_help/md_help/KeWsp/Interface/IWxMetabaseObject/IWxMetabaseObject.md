# IWxMetabaseObject


## Описание


Интерфейс IWxMetabaseObject
 содержит свойства объекта репозитория, отображаемого на рабочем пространстве.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [MetabaseObject](IWxMetabaseObject.MetabaseObject.htm)
		 Свойство MetabaseObject
		 определяет объект репозитория, который будет отображаться в рабочем
		 пространстве.


		 ![](../../Property_Image.gif)
		 [NextMetabaseObjects](IWxMetabaseObject.NextMetabaseObjects.htm)
		 Свойство NextMetabaseObjects
		 возвращает объект, от которого зависит данный объект.


		 ![](../../Property_Image.gif)
		 [NextMetabaseObjectsCount](IWxMetabaseObject.NextMetabaseObjectsCount.htm)
		 Свойство NextMetabaseObjectsCount
		 возвращает количество объектов, от которых зависит данный объект.


		 ![](../../Property_Image.gif)
		 [PreviousMetabaseObjects](IWxMetabaseObject.PreviousMetabaseObjects.htm)
		 Свойство PreviousMetabaseObjects
		 возвращает объект, зависящий от данного объекта.


		 ![](../../Property_Image.gif)
		 [PreviousMetabaseObjectsCount](IWxMetabaseObject.PreviousMetabaseObjectsCount.htm)
		 Свойство PreviousMetabaseObjectsCount
		 возвращает количество объектов, зависящих от данного объекта.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ShowDependencies](IWxMetabaseObject.ShowDependencies.htm)
		 Метод ShowDependencies
		 отображает на рабочем пространстве объекты, от которых зависит
		 данный объект.


		 ![](../../Sub_Image.gif)
		 [ShowDependents](IWxMetabaseObject.ShowDependents.htm)
		 Метод ShowDependents
		 отображает на рабочем пространстве объекты, которые зависят от
		 данного объекта.
