# IWxMetabaseObject.NextMetabaseObjects


## Синтаксис


NextMetabaseObjects(Index: Integer): [IWxMetabaseObject](IWxMetabaseObject.htm);


## Параметры


Index - индекс объекта, от
 которого зависит данный объект.


## Описание


Свойство NextMetabaseObjects
 возвращает объект, от которого зависит данный объект. Индекс объекта передается
 посредством параметра Index.
