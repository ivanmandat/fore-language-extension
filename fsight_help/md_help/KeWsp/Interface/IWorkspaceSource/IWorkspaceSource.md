# IWorkspaceSource


## Описание


Интерфейс IWorkspaceSource реализует
 графический контекст рабочего пространства, являясь, таким образом, источником
 данных для компонента среды разработки [WorkspaceBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/WorkspaceBox.htm).
 Интерфейс не содержит собственных свойств и методов.
