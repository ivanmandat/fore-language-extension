# IUiWorkspaceCancelEventArgs.Cancel


## Синтаксис


Cancel: Boolean;


## Описание


Свойство Cancel определяет,
 будет ли выполняться последнее действие.


## Комментарии


Допустимые значения:


	- True. Последнее действие
	 не будет выполнено;


	- False. Последнее действие
	 будет выполнено.
