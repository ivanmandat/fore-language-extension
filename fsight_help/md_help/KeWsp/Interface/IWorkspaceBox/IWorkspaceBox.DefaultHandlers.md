# IWorkspaceBox.DefaultHandlers


## Синтаксис


DefaultHandlers: Boolean;


## Описание


Свойство DefaultHandlers определяет
 возможность выполнения установленного по умолчанию действия над фигурой
 рабочего пространства.


## Комментарии


Свойство актуально, если на рабочем пространстве присутствуют фигуры,
 для которых установлено свойство [IWxShape.Extension](Andy.chm::/Interface/IWxShape/IWxShape.Extension.htm).


Если свойству DefaultHandlers
 установлено значение True, то
 для фигур будут выполняться те действия, которые определены для них в
 рабочем пространстве. При двойном щелчке по фигуре, которая соответствует
 объекту репозитория, будет произведено открытие соответствующего объекта.
 Если свойству DefaultHandlers
 установлено значение False, то
 все действия обрабатываются в событиях компонента.


По умолчанию свойству установлено значение False.
