# IWxAssemblies


Сборка: Workspace;


## Описание


Интерфейс IWxAssemblies содержит
 коллекцию модулей, определяющих обработчик событий в рабочем пространстве.


## Иерархия наследования


           IWxAssemblies


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IWxAssemblies.Count.htm)
		 Свойство Count возвращает
		 количество модулей в коллекции.


		 ![](../../Property_Image.gif)
		 [EventsAssembly](IWxAssemblies.EventsAssembly.htm)
		 Свойство EventsAssembly
		 определяет модуль для обработчика событий.


		 ![](../../Property_Image.gif)
		 [EventsClass](IWxAssemblies.EventsClass.htm)
		 Свойство EventsClass
		 определяет класс обработчика событий.


		 ![](../../Property_Image.gif)
		 [Item](IWxAssemblies.Item.htm)
		 Свойство Item возвращает
		 модуль, содержащийся в коллекции, по индексу.


		 ![](../../Property_Image.gif)
		 [Workspace](IWxAssemblies.Workspace.htm)
		 Свойство Workspace
		 возвращает рабочее пространство, для которого определяется обработчик
		 событий.
