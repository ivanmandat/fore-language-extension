# IWxAssemblies.Add


## Синтаксис


Add(Assembly: [IMetabaseObjectDescriptor](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm)):
 Integer;


## Параметры


Assembly. Модуль, который необходимо
 включить в коллекцию для определения обработчика событий рабочего пространства.


## Описание


Метод Add добавляет модуль в
 коллекцию.
