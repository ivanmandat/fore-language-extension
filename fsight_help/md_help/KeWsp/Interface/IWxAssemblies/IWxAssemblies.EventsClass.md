# IWxAssemblies.EventsClass


## Синтаксис


EventsClass: String;


## Описание


Свойство EventsClass определяет
 класс обработчика событий.


## Комментарии


Для определения модуля обработчика событий используйте свойство [IWxAssemblies.EventsAssembly](IWxAssemblies.EventsAssembly.htm).


Определяемый класс обработчика событий должен содержаться в заданном
 модуле.
