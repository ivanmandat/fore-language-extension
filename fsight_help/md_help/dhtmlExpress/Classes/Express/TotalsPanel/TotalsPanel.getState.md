# TotalsPanel.getState


## Синтаксис


getState();


## Описание


Метод getState** возвращает объект со значениями элементов управления типа «флажок» на вкладке «Итоги» в json-формате.


## Комментарии


Метод возвращает JSON-объект со следующими полями: byHierarchy - признак «Рассчитывать итоги по иерархии», byLevels - признак «Рассчитывать итоги по уровням», treatEmptyAsZero - признак «Рассматривать пустые значения как нули», includeOwner - признак «Не учитывать значения в вершинах».
