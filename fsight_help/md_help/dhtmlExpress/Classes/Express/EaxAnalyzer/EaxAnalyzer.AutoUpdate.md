# EaxAnalyzer.AutoUpdate


## Синтаксис


AutoUpdate: Boolean


## Описание


Свойство AutoUpdate** управляет автообновлением экспресс-отчета.


## Комментарии


Значение свойства можно установить из JSON и с помощью метода setAutoUpdate, а возвратиться - с помощью метода getAutoUpdate.


Значения свойства по умолчанию true - экспресс-отчет будет обновляться автоматически, если false - то автоматического обновления происходить не будет. По умолчанию значение свойства равно true.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента [ExpressBox](../../../Components/Express/ExpressBox/ExpressBox.htm) с наименованием «expressBox» (см. «[Пример создания компонента ExpressBox](../../../Components/Express/ExpressBox/ExpressBox_Example.htm)»). Обновим экспресс-отчет в случае, если автообновление его отключено, либо выведем сообщение о том, что экспресс-отчет обновляется автоматически:


var eaxAnalyzer = expressBox.getSource();
// Если автообновление экспресс-отчета отключено, то обновим его
if(eaxAnalyzer.getAutoUpdate()) {
    console.log("Экспресс-отчет обновляется автоматически.");
} else {
    expressBox.refreshAll();
    console.log("Экспресс-отчет обновлён.");
}

В результате выполнения примера выяснилось, что автообновление экспресс-отчета включено:


Экспресс-отчет обновляется автоматически.
