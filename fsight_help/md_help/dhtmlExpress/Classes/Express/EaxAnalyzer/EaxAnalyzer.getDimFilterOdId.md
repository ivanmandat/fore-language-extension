# EaxAnalyzer.getDimFilterOdId


## Синтаксис


getDimFilterOdId();


## Описание


Метод getDimFilterOdId**
 возвращает идентификатор фильтра измерений экспресс-отчета.


## Комментарии


Метод возвращает строковый идентификатор вида {идентификатор_сессии}!{идентификатор_объекта}.
