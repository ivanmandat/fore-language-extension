# EaxAnalyzer.TitleChangedData


## Синтаксис


TitleChangedData: Object


## Описание


Свойство TitleChangedData**
 содержит измененные данные заголовка экспресс-отчета.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTitleChangedData,
 а возвращается с помощью метода getTitleChangedData.


Свойство содержит значение в виде JSON-объекта.


## Пример


Свойство предназначено для внутреннего использования, для хранения временных
 настроек данных изменяемого заголовка. Обнуляется после обновления заголовка.
