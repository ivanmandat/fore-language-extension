# Конструктор EaxAnalyzer


## Синтаксис


PP.Exp.EaxAnalyzer(settings);


## Параметры


*settings.* JSON-объект со значениями свойств класса.


## Описание


Конструктор EaxAnalyzer** создает экземпляр класса **EaxAnalyzer**.


## Пример


Конструктор предназначен для внутреннего использования.
