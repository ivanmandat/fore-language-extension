# EaxAnalyzer.getSourceKey


## Синтаксис


getSourceKey(index: Number);


## Параметры


*index.* Индекс источника
 данных в массиве всех источников экспресс-отчёта.


## Описание


Метод getSourceKey** возвращает
 ключ источника данных экспресс-отчёта по его индексу.
