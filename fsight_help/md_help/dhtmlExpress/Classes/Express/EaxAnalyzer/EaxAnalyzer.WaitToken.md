# EaxAnalyzer.WaitToken


## Синтаксис


WaitToken: [Object](dhtmlCommon.chm::/Classes/object/object.htm)


## Описание


Свойство WaitToken** устанавливает и возвращает маркер ожидания по указанному имени.


## Комментарии


Значение свойства устанавливается с помощью метода set**WaitToken**, а возвращается с помощью метода get**WaitToken**. Из JSON значение свойства установить нельзя.


При вызове методов setWaitToken и getWaitToken указывается параметр n - наименование маркера ожидания, задаваемого с помощью элемента перечисления PP.Exp.[Operation](../../../Enums/Operation.htm).


Свойство содержит значение в виде JSON-объекта.
