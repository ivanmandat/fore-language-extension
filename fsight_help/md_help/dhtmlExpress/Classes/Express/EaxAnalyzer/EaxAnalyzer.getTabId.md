# EaxAnalyzer.getTabId


## Синтаксис


getTabId();


## Описание


Метод getTabId** возвращает идентификатор представления данных «Таблица» у экспресс-отчета.


## Комментарии


Метод возвращает JSON-объект с полем id - строковый идентификатор вида {идентификатор_сессии}!{идентификатор_объекта} для таблицы экспресс-отчета.
