# EaxAnalyzer.GaugeChangedData


## Синтаксис


GaugeChangedData: Object;


## Описание


Свойство GaugeChangedData определяет изменённые настройки для контейнера спидометра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setGaugeChangedData, а возвращается с помощью метода getGaugeChangedData.
