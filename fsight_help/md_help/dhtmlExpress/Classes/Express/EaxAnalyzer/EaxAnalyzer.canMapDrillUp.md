# EaxAnalyzer.canMapDrillUp


## Синтаксис


canMapDrillUp();


## Описание


Метод canMapDrillUp** проверяет, возможно ли поднятие на уровень вверх по иерархии отметки в представлении экспресс-отчета «Карта».


## Комментарии


Метод возвращает значение true, если поднятие на уровень вверх по иерархии отметки в представлении «Карта» возможно, и false, если невозможно.


Поднятие отметки на уровень вверх возможно, если количество детализаций территории карты больше нуля.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента [ExpressBox](../../../Components/Express/ExpressBox/ExpressBox.htm) с наименованием «expressBox» (см. «[Пример создания компонента ExpressBox](../../../Components/Express/ExpressBox/ExpressBox_Example.htm)»). Проверим, возможно ли поднятие отметки на уровень вверх по иерархии, и выведем соответствующее сообщение:


var eaxAnalyzer = expressBox.getSource();
if (eaxAnalyzer.canMapDrillUp()) {
    console.log("Поднятие отметки по иерархии в карте возможно.");
} else {
    console.log("Поднятие отметки по иерархии в карте невозможно.");
}

В результате выполнения примера в консоли браузера было выведено сообщение о том, что поднятие отметки на уровень вверх невозможно. Это объясняется тем, что количество детализаций территории загруженной карты равно нулю.
