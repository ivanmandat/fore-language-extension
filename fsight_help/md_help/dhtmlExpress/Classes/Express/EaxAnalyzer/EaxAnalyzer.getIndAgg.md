# EaxAnalyzer.getIndAgg


## Синтаксис


getIndAgg(indName, objectType);


## Параметры


indName. PP.Exp.EaxObjectIndicator.
 Тип метрики.


objectType. [PP.Exp.Ui.EaxObjectType](../../../Enums/EaxObjectType.htm).
 Тип визуализатора.


## Описание


Метод getIndAgg возвращает установленный
 тип агрегации метрики.


## Комментарии


Метод возвращает значение типа [PP.Exp.Ui.EaxAggregationFunction](../../../Enums/EaxAggregationFunction.htm).


## Пример


Использование метода приведено в описании метода [EaxMdService.setIndAgg](../EaxMdService/EaxMdService.setIndAgg.htm).
