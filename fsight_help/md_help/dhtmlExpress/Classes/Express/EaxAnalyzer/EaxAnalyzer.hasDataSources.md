# EaxAnalyzer.hasDataSources


## Синтаксис


hasDataSources();


## Описание


Метод hasDataSources**
 определяет наличие измерений у экспресс-отчета.


## Комментарии


Метод возвращает значение true,
 если у экспресс-отчета присутствуют измерения, и false,
 если отсутствуют.
