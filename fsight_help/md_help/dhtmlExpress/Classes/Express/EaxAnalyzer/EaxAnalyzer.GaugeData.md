# EaxAnalyzer.GaugeData


## Синтаксис


GaugeData: Object;


## Описание


Свойство GaugeData определяет
 настройки для контейнера спидометра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setGaugeData,
 а возвращается с помощью метода getGaugeData.
