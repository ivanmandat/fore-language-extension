# EaxAnalyzer.getChartId


## Синтаксис


getChartId();


## Описание


Метод getChartId** возвращает идентификатор представления данных «Диаграмма» у экспресс-отчета.


## Комментарии


Метод возвращает JSON-объект с полем id - строковый идентификатор вида {идентификатор_сессии}!{идентификатор_объекта} для диаграммы экспресс-отчета.
