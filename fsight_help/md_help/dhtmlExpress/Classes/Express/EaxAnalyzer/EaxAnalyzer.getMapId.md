# EaxAnalyzer.getMapId


## Синтаксис


getMapId();


## Описание


Метод getMapId** возвращает идентификатор представления данных «Карта» у экспресс-отчета.


## Комментарии


Метод возвращает JSON-объект с полем id - строковый идентификатор вида {идентификатор_сессии}!{идентификатор_объекта} для карты экспресс-отчета.
