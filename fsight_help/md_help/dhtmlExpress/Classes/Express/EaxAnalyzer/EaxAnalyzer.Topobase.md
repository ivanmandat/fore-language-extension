# EaxAnalyzer.Topobase


## Синтаксис


Topobase: Object


## Описание


Свойство Topobase** содержит
 топооснову для представления данных «Карта» экспресс-отчета.


## Комментарии


Значение свойства устанавливается с помощью метода setTopobase,
 а возвращается с помощью метода getTopobase.
 Из JSON значение свойства установить нельзя.


При вызове метода setTopobase
 указываются следующие параметры: n - значение в виде JSON-объекта, t -
 индекс топоосновы в массиве топооснов экспресс-отчета, - формат
 топоосновы, указываемый в виде объекта типа [PP.MapChartType](dhtmlMapChart.chm::/Enums/MapChartType.htm).


При вызове метода getTopobase
 указываются следующие параметры: n - индекс топоосновы в массиве
 топооснов экспресс-отчета, t - формат топоосновы, указываемый в
 виде объекта типа [PP.MapChartType](dhtmlMapChart.chm::/Enums/MapChartType.htm).


## Пример


Свойство в настоящий момент не реализовано.
