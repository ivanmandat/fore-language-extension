# EaxAnalyzer.IsSelectionChanged


## Синтаксис


IsSelectionChanged: Boolean


## Описание


Свойство IsSelectionChanged** содержит признак того, изменено ли выделение в таблице экспресс-отчета.


## Комментарии


Значение свойства устанавливается через JSON или с помощью метода setIs**Selection**Changed, а возвращается - при помощи метода getIs**Selection**Changed. При этом получаем значение true, если выделенные элементы экспресс-отчёта были изменены, и false - в противном случае.
