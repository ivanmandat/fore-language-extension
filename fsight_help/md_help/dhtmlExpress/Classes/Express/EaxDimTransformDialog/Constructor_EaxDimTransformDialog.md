# Конструктор EaxDimTransformDialog


## Синтаксис


PP.Exp.Ui.PivotFilterTransformDialog(settings);


## Параметры


settings. JSON-объект со значениями
 свойств компонента.


## Описание


Конструктор EaxDimTransformDialog создает
 экземпляр компонента EaxDimTransformDialog.
