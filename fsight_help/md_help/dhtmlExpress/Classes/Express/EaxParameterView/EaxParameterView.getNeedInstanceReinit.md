# EaxParameterView.getNeedInstanceReinit


## Синтаксис


getNeedInstanceReinit();


## Описание


Метод getNeedInstanceReinit** возвращает признак того, нуждается ли представление в пересоздании при следующем обновлении параметра.


## Комментарии


Метод возвращает значение true, если объект нужно пересоздать, и значение false, если не требуется этого.
