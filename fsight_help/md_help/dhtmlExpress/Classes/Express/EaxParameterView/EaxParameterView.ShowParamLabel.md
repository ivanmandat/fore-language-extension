# EaxParameterView.ShowParamLabel


## Синтаксис


ShowParamLabel: boolean


## Описание


Свойство ShowParamLabel** управляет видимостью подписи у параметра экспресс-отчета.


## Комментарии


Значение свойства ShowParamLabel можно задавать из JSON и при помощи метода setShowParamLabel, а возвращать - при помощи метода getShowParamLabel.


Если свойство принимает значение true, то подпись у параметра экспресс-отчета отображается, если false, то не отображается.
