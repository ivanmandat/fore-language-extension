# EaxParameterView.ParamId


## Синтаксис


ParamId: String


## Описание


Свойство ParamId** устанавливает идентификатор параметра экспресс-отчета.


## Комментарии


Значение свойства ParamId можно задавать из JSON и при помощи метода setParamId, а возвращать - при помощи метода getParamId.


Для создания компонента [EaxParameterBox](../../../Components/Express/EaxParameterBox/EaxParameterBox.htm) можно либо задать свойства [Source](EaxParameterView.Source.htm) и [ParamId](EaxParameterView.ParamId.htm)  (параметр должен существовать в источнике данных), либо задать свойства [ParamSource](EaxParameterView.ParamSource.htm) и Service.
