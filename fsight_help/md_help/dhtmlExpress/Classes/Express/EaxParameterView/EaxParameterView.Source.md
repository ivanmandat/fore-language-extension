# EaxParameterView.Source


## Синтаксис


Source: PP.Exp.EaxDocument


## Описание


Свойство Source** устанавливает источник данных для параметра экспресс-отчета.


## Комментарии


Значение свойства Source можно задавать из JSON и при помощи метода setSource, а возвращать - при помощи метода getSource.


Для создания компонента [EaxParameterBox](../../../Components/Express/EaxParameterBox/EaxParameterBox.htm) можно либо задать свойства Source и [ParamId](EaxParameterView.ParamId.htm) (параметр должен существовать в источнике данных), либо задать свойства [ParamSource](EaxParameterView.ParamSource.htm) и Service.
