# EaxParameterView.setParamVisible


## Синтаксис


setParamVisible(value);


## Параметры


*value.* Признак того, нужно ли отображать параметр. Если принимает значение true, то параметр будет показан на вкладке, если false, то будет скрыт.


## Описание


Метод setParamVisible** управляет видимостью параметра экспресс-отчета.
