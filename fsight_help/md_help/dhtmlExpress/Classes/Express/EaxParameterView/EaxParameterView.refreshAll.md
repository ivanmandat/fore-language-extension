# EaxParameterView.refreshAll


## Синтаксис


refreshAll(reInit);


## Параметры


*reInit.* Признак того, что требуется пересоздание объекта. Если принимает значение true, то объект будет пересоздан, в противном случае - не будет.


## Описание


Метод refreshAll** обновляет все элементы параметра на вкладке «Параметры источников данных».
