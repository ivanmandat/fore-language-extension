# Конструктор EaxDimBarView


## Синтаксис


PP.Exp.Ui.EaxDimBarView(settings);


## Параметры


*settings.* JSON-объект со значениями свойств класса.


## Описание


Конструктор EaxDimBarView** создает экземпляр класса **EaxDimBarView**.
