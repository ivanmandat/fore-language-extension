# EaxDimBarView.EnableDragDim


## Синтаксис


EnableDragDim: boolean


## Описание


Свойство EnableDragDim** определяет, можно ли перемещать измерения.


## Комментарии


Значение свойства EnableDragDim также можно задать при помощи метода setEnableDragDim и возвратить при помощи метода getEnableDragDim.


Если значение свойства равно true, то перемещение измерений разрешено, если равно false, то запрещено.
