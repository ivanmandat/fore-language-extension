# EaxDimBarView.DimensionMoved


## Синтаксис


DimensionMoved: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: MovedKeyDim - измерение, которое нужно переместить, DimPosition - позиция, в которую нужно переместить измерение, Index - индекс расположения в данной измерения.


## Описание


Событие DimensionMoved** наступает при перемещении измерения на панели измерений экспресс-отчета.
