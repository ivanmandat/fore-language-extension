# ChartTypeMenu.getLineChartTypeListBox


## Синтаксис


getLineChartTypeListBox();


## Описание


Метод getLineChartTypeListBox
 возвращает массив значков, соответствующих линейным диаграммам.


## Комментарии


Метод возвращает объект типа [PP.Ui.ListBox](dhtmlList.chm::/Classes/ListBox/ListBox.htm).
