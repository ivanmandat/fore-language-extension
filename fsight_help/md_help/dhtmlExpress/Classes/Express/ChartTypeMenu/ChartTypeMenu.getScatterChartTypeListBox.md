# ChartTypeMenu.getScatterChartTypeListBox


## Синтаксис


getScatterChartTypeListBox();


## Описание


Метод getScatterChartTypeListBox
 возвращает массив значков, соответствующих точечным диаграммам.


## Комментарии


Метод возвращает объект типа [PP.Ui.ListBox](dhtmlList.chm::/Classes/ListBox/ListBox.htm).
