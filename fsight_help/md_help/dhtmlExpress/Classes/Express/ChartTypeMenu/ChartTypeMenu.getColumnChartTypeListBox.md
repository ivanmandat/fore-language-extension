## Синтаксис

## Описание


Метод getColumnChartTypeListBox
 возвращает массив значков, соответствующих гистограммам.


## Комментарии


Метод возвращает значение типа [PP.Ui.ListBox](dhtmlList.chm::/Classes/ListBox/ListBox.htm).
