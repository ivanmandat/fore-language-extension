# ChartTypeMenu.getPieChartTypeListBox


## Синтаксис


getPieChartTypeListBox();


## Описание


Метод getPieChartTypeListBox
 возвращает массив значков, соответствующих круговым диаграммам.


## Комментарии


Метод возвращает объект типа [PP.Ui.ListBox](dhtmlList.chm::/Classes/ListBox/ListBox.htm).
