# ChartTypeMenu.ChartTypeChanged


## Синтаксис


ChartTypeChanged: function(sender, args: Object);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: Type - тип диаграммы, Inverted - способ отображения диаграммы сверху вниз или слева направо, Stacking - способ накопления её значений.


## Описание


Событие ChartTypeChanged генерируется после изменения типа диаграммы.
