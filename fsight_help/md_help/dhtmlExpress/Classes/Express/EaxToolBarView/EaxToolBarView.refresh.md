# EaxToolBarView.refresh


## Синтаксис


refresh(objectType: PP.Exp.Ui.[EaxObjectType](../../../Enums/EaxObjectType.htm));


## Параметры


*objectType.* Указывает, какую именно кнопку надо обновить. Задаётся с помощью элемента перечисления PP.Exp.Ui.[EaxObjectType](../../../Enums/EaxObjectType.htm).


## Описание


Метод refresh** обновляет указанную кнопку в группе других кнопок, используемых для переключения режима отображения данных.
