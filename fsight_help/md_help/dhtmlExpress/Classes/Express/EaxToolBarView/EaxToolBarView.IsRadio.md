# EaxToolBarView.IsRadio


## Синтаксис


IsRadio: boolean


## Описание


Свойство IsRadio** определяет, включен ли режим переключения кнопок.


## Комментарии


Если значение свойства установлено в true, то активным является режим переключения кнопок, если в false, то включен режим выбора кнопок по типу элементов управления типа «флажок». По умолчанию присваивается значение true.


Свойство может задаваться из JSON или с помощью метода setIsRadio и возвращается с помощью метода getIsRadio. В качестве параметра метода setIsRadio указывается признак того, включен ли режим переключения кнопок.
