# EaxToolBarView.setActiveObject


## Синтаксис


setActiveObject(objType: PP.Exp.Ui.[EaxObjectType](../../../Enums/EaxObjectType.htm), fireEvent);


## Параметры


*objType.* Объект, который необходимо сделать активным;


*fireEvent.* Определяет, генерировать ли событие изменения активного объекта


## Описание


Метод setActiveObject** активирует указанный объект.


## Комментарии


В текущем контексте метод активирует указанную кнопку из группы других кнопок, предназначенных для выбора режима отображения данных.
