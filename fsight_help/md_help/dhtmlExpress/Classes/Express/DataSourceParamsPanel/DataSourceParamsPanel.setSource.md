# DataSourceParamsPanel.setSource


## Синтаксис


setSource(value);


## Параметры


*value.* Источник данных для вкладки «Параметры источников данных».


## Описание


Метод setSource** устанавливает источник данных для вкладки «Параметры источников данных».
