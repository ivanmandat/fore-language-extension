# EaxDataView.ObjSizeChanged


## Синтаксис


ObjSizeChanged: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: Type - тип объекта, Size - размеры объекта ( Width - ширина, Height - высота ).


## Описание


Событие ObjSizeChanged** наступает после изменения размера объекта.
