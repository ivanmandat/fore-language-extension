# EaxDataView.DocDataChanged


## Синтаксис


DocDataChanged: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: TabIndex - индекс вкладки, SheetName - наименование листа, SheetKey - ключ листа, Operation - тип операции в документе, вызвавшей данное событие.


## Описание


Событие DocDataChanged** наступает после изменения информациии о документе, загруженном в рабочей области экспресс-отчета.
