# EaxDataView.Rendered


## Синтаксис


Rendered: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие Rendered** генерируется после того, как контейнер для рабочей области экспресс-отчета будет готов к просмотру.
