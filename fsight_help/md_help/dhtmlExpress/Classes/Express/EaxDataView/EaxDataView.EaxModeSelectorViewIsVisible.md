# EaxDataView.EaxModeSelectorViewIsVisible


## Синтаксис


EaxModeSelectorViewIsVisible: boolean


## Описание


Свойство EaxModeSelectorViewIsVisible** задает видимость кнопки для выбора режима отображения представлений данных «Раскладка».


## Комментарии
