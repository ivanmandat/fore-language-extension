# EaxDataView.HeaderContainerVisibility


## Синтаксис


HeaderContainerVisibility: PP.Exp.Ui.[HeaderContainerVisibility](../../../Enums/HeaderContainerVisibility.htm)


## Описание


Свойство HeaderContainerVisibility** задаёт режим отображения контейнера, содержащего заголовок экспресс-отчета и группу кнопок для выбора варианта представления данных.


## Комментарии


Значением свойства **HeaderContainerVisibility** является элемент перечисления PP.Exp.Ui.[HeaderContainerVisibility](../../../Enums/HeaderContainerVisibility.htm).
