# EaxDataView.SheetClick


## Синтаксис


SheetClick: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие SheetClick генерируется при нажатии по заголовку листа экспресс-отчета.
