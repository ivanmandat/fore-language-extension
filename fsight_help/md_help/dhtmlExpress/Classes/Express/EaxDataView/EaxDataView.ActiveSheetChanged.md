# EaxDataView.ActiveSheetChanged


## Синтаксис


ActiveSheetChanged: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: sheetInfo - информация о сменившем активном листе, sheets - набор листов.


## Описание


Событие ActiveSheetChanged** наступает после смены активного листа в рабочей области экспресс-отчета.
