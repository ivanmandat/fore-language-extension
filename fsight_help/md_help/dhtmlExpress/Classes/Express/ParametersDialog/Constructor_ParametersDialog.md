# Конструктор ParametersDialog


## Синтаксис


PP.Exp.Ui.ParametersDialog(settings);


## Параметры


*settings.* JSON-объект со значениями свойств компонента.


## Описание


Конструктор ParametersDialog** создает экземпляр компонента **ParametersDialog**.
