# ViewCatView.refresh


## Синтаксис


refresh();


## Описание


Метод refresh** обновляет вкладку «Вид» ленты инструментов экспресс-отчета.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента ExpressBox с наименованием «expressBox» (см. «[Пример создания компонента ExpressBox](../../../Components/Express/ExpressBox/ExpressBox_Example.htm)»). Далее обновим вкладку «Вид»:


В результате выполнения примера была обновлена вкладка ленты инструментов «Вид».
