# ChartMarkerMenu.Marker


## Синтаксис


Marker: String;


## Описание


Свойство Marker определяет типа
 маркера, выбранного в меню.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMarker(value),
 а возвращается с помощью метода getMarker.


Допустимые значения:


	- none;


	- square;


	- circle;


	- diamond;


	- triangle;


	- triangle-down.
