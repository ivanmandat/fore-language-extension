# ChartMarkerMenu.getChartMarkerListBox


## Синтаксис


getChartMarkerListBox();


## Описание


Метод getChartMarkerListBox
 возвращает список доступных маркеров.


## Комментарии


Метод возвращает объект типа [PP.Ui.ListBox](dhtmlList.chm::/Classes/ListBox/ListBox.htm).
