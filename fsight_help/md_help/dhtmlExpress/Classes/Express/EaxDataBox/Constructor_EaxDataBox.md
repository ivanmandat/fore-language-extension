# Конструктор EaxDataBox


## Синтаксис


PP.Exp.Ui.EaxDataBox(settings);


## Параметры


*settings.* JSON-объект со значениями свойств класса.


## Описание


Конструктор EaxDataBox** создает экземпляр класса **EaxDataBox**.
