# Конструктор EaxMapDefaultSettingsProvider


## Синтаксис


PP.Exp.EaxMapDefaultSettingsProvider(settings);


## Параметры


*settings.* JSON-объект
 со значениями свойств компонента.


## Описание


Конструктор EaxMapDefaultSettingsProvider
 создает экземпляр класса EaxMapDefaultSettingsProvider.
