# Конструктор EaxDimBar


## Синтаксис


PP.Exp.Ui.EaxDimBar(settings);


## Параметры


*settings.* JSON-объект со значениями свойств класса.


## Описание


Конструктор EaxDimBar создает экземпляр компонента EaxDimBar.
