# ExpressView.ViewMode


## Синтаксис


ViewMode: Boolean


## Описание


Свойство ViewMode** определяет,
 открыт ли экспресс-отчёт в режиме просмотра.


## Комментарий


По умолчанию установлено значение false.


Значение свойства возвращается при помощи метода getViewMode.


Режим просмотра отличается от обычного режима отсутствием кнопки «Сохранить»
 в главном меню.
