# Конструктор ChartBox


## Синтаксис


PP.Exp.Ui.ChartBox(settings);


## Параметры


*settings.* JSON-объект со значениями свойств компонента.


## Описание


Конструктор ChartBox** создает экземпляр компонента **ChartBox**.
