# Конструктор XlsExportDialog


## Синтаксис


PP.Exp.Ui.XlsExportDialog(settings);


## Параметры


*settings.* JSON-объект со значениями свойств компонента.


## Описание


Конструктор XlsExportDialog создает экземпляр компонента [XlsExportDialog](../../../Components/Express/XlsExportDialog/XlsExportDialog.htm).
