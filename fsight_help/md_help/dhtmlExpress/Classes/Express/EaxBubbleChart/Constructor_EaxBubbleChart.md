# Конструктор EaxBubbleChart


## Синтаксис


PP.Exp.Ui.EaxBubbleChart(settings);


## Параметры


*settings.* JSON-объект со значениями свойств компонента.


## Описание


Конструктор EaxBubbleChart** создает экземпляр компонента **EaxBubbleChart**.
