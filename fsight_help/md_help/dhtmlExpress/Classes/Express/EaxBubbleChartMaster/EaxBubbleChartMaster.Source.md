# BubbleChartMaster.Source


## Синтаксис


Source: PP.Exp.[EaxDocument](../EaxDocument/EaxDocument.htm)


## Описание


Свойство Source** определяет
 источник данных для пузырьковой диаграммы экспресс-отчета.


## Комментарии


Значение свойства можно установить с помощью метода setSource,
 а возвратить - с помощью метода getSource.
 Из JSON значение свойства установить нельзя.
