# Конструктор BubbleChartMaster


## Синтаксис


PP.Exp.Ui.BubbleChartMaster(settings);


## Параметры


*settings.* JSON-объект
 со значениями свойств компонента.


## Описание


Конструктор BubbleChartMaster**
 создает экземпляр компонента **BubbleChartMaster**.
