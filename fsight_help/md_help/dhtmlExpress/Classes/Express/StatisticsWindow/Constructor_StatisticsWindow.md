# Конструктор StatisticsWindow


## Синтаксис


PP.Exp.Ui.StatisticsWindow(settings);


## Параметры


*settings.* JSON-объект со значениями свойств компонента.


## Описание


Конструктор StatisticsWindow** создает экземпляр компонента **StatisticsWindow**.
