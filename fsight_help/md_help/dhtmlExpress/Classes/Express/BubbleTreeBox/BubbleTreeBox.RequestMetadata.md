# BubbleTreeBox.RequestMetadata


## Синтаксис


RequestMetadata: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступный аргумент: IsReport - признак того, что запрашиваются данные для контейнера пузырькового дерева.


## Описание


Событие RequestMetadata генерируется при запросе метаданных для контейнера пузырькового дерева.
