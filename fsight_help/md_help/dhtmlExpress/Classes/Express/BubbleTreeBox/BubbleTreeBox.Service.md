# BubbleTreeBox.Service


## Синтаксис


Service: [PP.Exp.EaxMdService](../EaxMdService/EaxMdService.htm);


## Описание


Свойство Service определяет
 сервис для работы с контейнером пузырькового дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setService,
 возвращается с помощью метода getService.
