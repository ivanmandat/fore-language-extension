# BubbleTreeBox.StatisticsWindow


## Синтаксис


StatisticsWindow: [PP.Exp.Ui.StatisticsWindow](../StatisticsWindow/StatisticsWindow.htm);


## Описание


Свойство StatisticsWindow определяет для контейнера пузырькового дерева окно «Статистические характеристики».


## Комментарии


Значение свойства устанавливается из JSON и с помощью setStatisticsWindow, а возвращается с помощью метода getStatisticsWindow.


Окно открывается при выполнении команды контекстного меню «Статистические характеристики» или при вызове метода [BubbleTreeBox.showStatisticsWindow](BubbleTreeBox.showStatisticsWindow.htm).
