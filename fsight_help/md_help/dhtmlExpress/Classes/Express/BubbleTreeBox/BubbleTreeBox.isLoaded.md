# BubbleTreeBox.isLoaded


## Синтаксис


isLoaded();


## Описание


Метод isLoaded определяет, загружен ли контейнер для пузырькового дерева.


## Комментарии


Метод возвращает значение true, если контейнер для пузырькового дерева загружен на странице, и значение false, если не загружен.
