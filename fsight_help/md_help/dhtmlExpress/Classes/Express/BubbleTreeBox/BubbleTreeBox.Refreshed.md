# BubbleTreeBox.Refreshed


## Синтаксис


Refreshed: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие Refreshed генерируется после обновления контейнера для пузырькового дерева.
