# BubbleTreeBox.getState


## Синтаксис


getState(arg: Object);


## Параметры


arg. Объект с изначальными настройками.


## Описание


Метод getState возвращает настройки контейнера для пузырькового дерева.


## Комментарии


Метод возвращает объект типа PP.Ui.PropertyChangedEventArgs.
