# BubbleTreeBox.getSizeVisual


## Синтаксис


getSizeVisual();


## Описание


Метод getSizeVisual возвращает настройки сопоставления данных контейнера пузырькового дерева размерам его элементов.


## Комментарии


Метод возвращает объект типа PP.SizeVisual.
