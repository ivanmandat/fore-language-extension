# BubbleTreeBox.getLegends


## Синтаксис


getLegends();


## Описание


Метод getLegends возвращает объект с легендами пузырькового дерева.


## Комментарии


Метод возвращает JSON-объект, свойства которых («Legend0», «Legend1» и так далее) содержат легенды пузырькового дерева.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента [BubbleTreeBox](../../../Components/Express/BubbleTreeBox/BubbleTreeBox.htm) с наименованием «bubbleTreeBox» (см. «[Пример создания компонента BubbleTreeBox](../../../Components/Express/TreeMapBox/TreeMapBox_Example.htm)»). Отобразим размерную легенду, если настройки сопоставления данных размерам элементов дерева определены:


// Получим размерную легенду
var sizeLegend = bubbleTreeBox.getLegends()["Legend1"];
// Определим, доступна ли размерная легенда
if (bubbleTreeBox.getSizeVisual() & sizeLegend) {
    // Отобразим размерную легенду
    sizeLegend.setIsVisible(true);
    // Установим размеры легенды
    sizeLegend.setSize(180, 75);
} else {
    if (sizeLegend) {
        console.log("Размерная легенда доступна.")
    } else {
        console.log("Размерная легенда недоступна.")
    };
    console.log("Настройки сопоставления данных размерам элементов дерева не определены.");
};

Если размерная легенда пузырькового дерева недоступна или не определены настройки сопоставления данных размерам элементов дерева, то после выполнения примера в консоли браузера будут выведены соответствующие уведомления. Если эти настройки определены, то размерная легенда будет отображена.
