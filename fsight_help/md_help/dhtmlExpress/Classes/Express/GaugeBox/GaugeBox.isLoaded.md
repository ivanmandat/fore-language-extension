# GaugeBox.isLoaded


## Синтаксис


isLoaded();


## Описание


Метод isLoaded определяет, загружен ли контейнер для спидометра.


## Комментарии


Метод возвращает значение true, если контейнер для спидометра загружен на странице, и значение false, если не загружен.
