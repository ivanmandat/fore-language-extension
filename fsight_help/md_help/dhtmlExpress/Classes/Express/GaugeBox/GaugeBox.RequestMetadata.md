# GaugeBox.RequestMetadata


## Синтаксис


RequestMetadata: function(sender, args: Object);


## Параметры


sender. Источник события;


args. Информация о событии. Доступный аргумент: IsReport - признак того, что запрашиваются данные для контейнера спидометра.


## Описание


Событие RequestMetadata генерируется при запросе метаданных для контейнера спидометра.
