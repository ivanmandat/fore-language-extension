# GaugeBox.getState


## Синтаксис


getState(arg: Object);


## Параметры


arg. Объект с изначальными настройками.


## Описание


Метод getState возвращает настройки контейнера для спидометра.


## Комментарии


Метод возвращает объект типа PP.Ui.PropertyChangedEventArgs.
