# Конструктор EaxModeSelector


## Синтаксис


PP.Exp.Ui.EaxModeSelector(settings);


## Параметры


*settings.* JSON-объект со значениями свойств компонента.


## Описание


Конструктор EaxModeSelector** создает экземпляр компонента **EaxModeSelector**.
