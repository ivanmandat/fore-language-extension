# Конструктор EaxTableDataSource


## Синтаксис


PP.Exp.EaxTableDataSource(settings);


## Параметры


*settings.* JSON-объект со значениями свойств класса.


## Описание


Конструктор EaxTableDataSource** создает экземпляр класса **EaxTableDataSource**.


## Пример


Конструктор предназначен для внутреннего использования.
