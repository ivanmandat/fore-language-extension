# EaxGrid.AutoSendSelection


## Синтаксис


AutoSendSelection: boolean


## Описание


Свойство AutoSendSelection** определяет, нужно ли после выделения в таблице отправлять изменения сразу на сервер.


## Комментарии


Если значение свойства установлено в true, то изменения в таблице после смены выделения сразу отправляются на сервер, если false - отправления измененных данных сразу не происходит.
