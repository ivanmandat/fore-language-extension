# EaxGrid.MetadataChanged


## Синтаксис


MetadataChanged: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие MetadataChanged** наступает при изменении метаданных для таблицы в рабочей области экспресс-отчета.
