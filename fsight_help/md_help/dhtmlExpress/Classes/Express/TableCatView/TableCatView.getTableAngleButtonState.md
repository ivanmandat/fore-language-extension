# TableCatView.getTableAngleButtonState


## Синтаксис


getTableAngleButtonState();


## Описание


Метод getTableAngleButtonState** возвращает свойства кнопки «Поворот таблицы» на вкладке ленты инструментов «Таблица».


## Комментарии


Метод возвращает значение в виде объекта JSON.
