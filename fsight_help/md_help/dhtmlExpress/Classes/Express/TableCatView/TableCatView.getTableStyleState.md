# TableCatView.getTableStyleState


## Синтаксис


getTableStyleState(sender);


## Параметры


*sender.* Значение, задаваемое с помощью перечисления [PP.Exp.Ui.RibbonTableStyleEnum](../../../Enums/RibbonTableStyleEnum.htm).


## Описание


Метод getTableStyleState** возвращает свойства кнопки «Стиль и оформление» на вкладке ленты инструментов «Таблица».


## Комментарии


Метод возвращает значение в виде объекта JSON.
