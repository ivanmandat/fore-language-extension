# TableCatView.getTableCFormatButton


## Синтаксис


getTableCFormatButton();


## Описание


Метод getTableCFormatButton** возвращает кнопку «Условное форматирование» на вкладке ленты инструментов «Таблица».


## Комментарии


Метод возвращает объект класса PP.Ui.SplitButton.
