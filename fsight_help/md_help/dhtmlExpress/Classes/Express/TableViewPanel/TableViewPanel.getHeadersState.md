# TableViewPanel.getHeadersState


## Синтаксис


getHeadersState();


## Описание


Метод getHeadersState** возвращает состояние элемента управления «Разметка (заголовки столбцов и строк)» на панели.


## Комментарии


Метод возвращает значение true, если выбран пункт «Разметка (заголовки столбцов и строк)» на вкладке «Вид» панели свойств экспресс-отчета, и значение false в противном случае.
