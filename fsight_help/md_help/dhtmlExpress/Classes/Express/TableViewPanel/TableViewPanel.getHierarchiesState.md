# TableViewPanel.getHierarchiesState


## Синтаксис


getHierarchiesState();


## Описание


Метод getHierarchiesState** возвращает состояние элемента управления «Отображение иерархий» на панели.


## Комментарии


Метод возвращает название выбранного пункта в раскрывающемся списке «Отображение иерархий» на вкладке «Вид» панели свойств экспресс-отчета.
