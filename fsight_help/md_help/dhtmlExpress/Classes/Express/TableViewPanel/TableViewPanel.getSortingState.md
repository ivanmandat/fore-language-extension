# TableViewPanel.getSortingState


## Синтаксис


getSortingState();


## Описание


Метод getSortingState** возвращает состояние элемента управления «Способ сортировки» на панели.


## Комментарии


Метод возвращает JSON-объект, содержащий поле @sortDiapason со значением выбранного пункта в раскрывающемся списке «Способ сортировки» на вкладке «Вид» панели свойств экспресс-отчета.
