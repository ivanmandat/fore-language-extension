# TableViewPanel.getAutoFitState


## Синтаксис


getAutoFitState();


## Описание


Метод getAutoFitState** возвращает состояние элемента управления «Автоподгонка ячеек» на панели.


## Комментарии


Метод возвращает JSON-объект со следующими полями: mode - режим автоподгонки ячеек, maxRowsInCell - максимальное количество строк в ячейке, minVisibleDataColumns - минимальное количество видимых столбцов.
