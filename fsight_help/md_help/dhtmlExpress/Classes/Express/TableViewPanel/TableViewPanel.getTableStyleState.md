# TableViewPanel.getTableStyleState


## Синтаксис


getTableStyleState();


## Описание


Метод getTableStyleState** возвращает состояние элемента управления «Стиль оформления» на панели.


## Комментарии


Метод возвращаетJSON-объект, содержащий следующие поля: PredefinedStyle со значением выбранного пункта в раскрывающемся списке «Стиль оформления» (с учетом выбора элемента управления «Чередующиеся строки») , Name со значением «Eax table style».
