# ParametersDataSource.Name


## Синтаксис


Name: string


## Описание


Свойство Name** устанавливает новое название для вкладки «Параметры источников данных» у диалога «Параметры».


## Комментарии


Значение свойства Name можно устанавливать из JSON и с помощью метода setName, а возвращать - при помощи свойства getName.
