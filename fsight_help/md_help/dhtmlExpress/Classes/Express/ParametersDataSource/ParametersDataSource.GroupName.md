# ParametersDataSource.GroupName


## Синтаксис


GroupName: string


## Описание


Свойство GroupName** устанавливает название группы, к которой относится вкладка «Параметры источников данных» у диалога «Параметры».


## Комментарии


Значение свойства GroupName можно устанавливать из JSON и с помощью метода setGroupName, а возвращать - при помощи свойства getGroupName.
