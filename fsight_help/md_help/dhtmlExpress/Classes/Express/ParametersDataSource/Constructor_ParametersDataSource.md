# Конструктор ParametersDataSource


## Синтаксис


PP.Exp.Ui.ParametersDataSource(settings);


## Параметры


*settings.* JSON-объект со значениями свойств компонента.


## Описание


Конструктор ParametersDataSource создает экземпляр компонента ParametersDataSource.
