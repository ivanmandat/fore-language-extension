# Конструктор EaxMapBox


## Синтаксис


PP.Exp.Ui.EaxMapBox(settings);


## Параметры


*settings.* JSON-объект со значениями свойств компонента.


## Описание


Конструктор EaxMapBox** создает экземпляр компонента **EaxMapBox**.
