# Конструктор StatTreeView


## Синтаксис


PP.Exp.Ui.StatTreeView(settings);


## Параметры


*settings.* JSON-объект со значениями свойств компонента.


## Описание


Конструктор StatTreeView** создает экземпляр компонента **StatTreeView**.
