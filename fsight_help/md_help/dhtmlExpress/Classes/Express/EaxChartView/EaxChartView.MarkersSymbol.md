# EaxChartView.MarkersSymbol


## Синтаксис


MarkersSymbol: String;


## Описание


Свойство MarkersSymbol определяет
 форму маркера для рядов данных диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMarkersSymbol,
 а возвращается с помощью метода getMarkersSymbol.


Допустимые значения:


	- circle.
	 Маркер в форме круга (по умолчанию);


	- diamond.
	 Маркер в форме ромба;


	- square.
	 Маркер в форме квадрата;


	- triangle.
	 Маркер в форме треугольника;


	- triangledown,
	 triangle-down. Маркер в форме
	 обратного треугольника.

Использование свойства аналогично использованию свойства [PP.Ui.Chart.MarkersSymbol](dhtmlChart.chm::/Components/Chart/ChartSplineSerie.htm).
