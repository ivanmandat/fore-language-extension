# EaxChartView.SignaturesOptions


## Синтаксис


SignaturesOptions: Object;


## Описание


Свойство SignaturesOptions определяет настройки рядов данных диаграммы.


## Комментарии


Значение свойства устанавливается с помощью метода setSignaturesOptions, а возвращается - с помощью метода getSignaturesOptions.


Метод setSignaturesOptions принимает три параметра: formatter - формат подписей, align - стиль их выравнивания, enabled - признак их доступности.


Метод getSignaturesOptions возвращает JSON-объект с полями formatter, align, enabled.
