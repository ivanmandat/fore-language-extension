# EaxChartView.isLoaded


## Синтаксис


isLoaded();


## Описание


Метод isLoaded** возвращает признак того, что диаграмма экспресс-отчета загружена.


## Комментарии


Если диаграмма экспресс-отчета загружена, то метод возвращает значение true, иначе - значение false.
