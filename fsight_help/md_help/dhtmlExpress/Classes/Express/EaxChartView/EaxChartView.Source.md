# EaxChartView.Source


## Синтаксис


Source: PP.Exp.[EaxDocument](../EaxDocument/EaxDocument.htm)


## Описание


Свойство Source** определяет источник данных для диаграммы экспресс-отчета.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSource, а возвращается с помощью метода getSource.
