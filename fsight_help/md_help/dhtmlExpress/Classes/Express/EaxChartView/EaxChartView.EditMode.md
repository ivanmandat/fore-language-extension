# EaxChartView.EditMode


## Синтаксис


EditMode: PP.Exp.Ui.[ChartEditMode](../../../Enums/ChartEditMode.htm)


## Описание


Свойство EditMode** устанавливает режим редактирования диаграммы экспресс-отчета.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setEditMode, а возвращается с помощью метода getEditMode.


По умолчанию имеет значение PP.Exp.Ui.[ChartEditMode](../../../Enums/ChartEditMode.htm).None.
