# EaxChartView.MarkersEnabled


## Синтаксис


MarkersEnabled: Boolean;


## Описание


Свойство MarkersEnabled определяет
 признак отображения маркеров у рядов данных диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMarkersEnabled,
 а возвращается с помощью метода getMarkersEnabled.


Допустимые значения:


	- true.
	 Маркеры у рядов данных диаграммы отображаются (по умолчанию);


	- false.
	 Маркеры у рядов данных диаграммы не отображаются.


## Пример


Использование свойства аналогично использованию свойства [PP.Ui.Chart.MarkersEnabled](dhtmlChart.chm::/Classes/Chart/Chart/Chart.MarkersEnabled.htm).
