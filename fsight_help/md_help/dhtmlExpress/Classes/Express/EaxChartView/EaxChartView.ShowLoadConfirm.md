# EaxChartView.ShowLoadConfirm


## Синтаксис


ShowLoadConfirm: Boolean;


## Описание


Свойство ShowLoadConfirm устанавливает
 признак отображения сообщения при превышении максимального числа точек
 диаграммы экспресс-отчета.


## Комментарии


Значение свойства устанавливается только из JSON.


Для определения максимального числа точек диаграммы используйте свойство
 [EaxChartView.LoadConfirmThreshold](EaxChartView.LoadConfirmThreshold.htm).


Допустимые значения:


	- true. При превышении
	 максимального числа точек диаграммы будет выведено предупреждающее
	 сообщение;


	- false. При превышении
	 максимального числа точек диаграммы предупреждающего сообщения не
	 будет.
