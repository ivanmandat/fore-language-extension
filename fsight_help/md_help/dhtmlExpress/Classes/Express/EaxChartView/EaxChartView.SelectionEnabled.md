# EaxChartView.SelectionEnabled


## Синтаксис


SelectionEnabled: boolean


## Описание


Свойство SelectionEnabled**
 определяет, доступно ли выделение в диаграмме экспресс-отчета.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSelectionEnabled,
 а возвращается с помощью метода getSelectionEnabled.


Если для свойство установлено значение true,
 то выделение в диаграмме доступно, иначе - не доступно.
