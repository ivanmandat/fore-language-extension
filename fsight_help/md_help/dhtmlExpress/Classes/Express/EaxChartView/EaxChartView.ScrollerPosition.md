# EaxChartView.ScrollerPosition


## Синтаксис


ScrollerPosition: Number;


## Описание


Свойство ScrollerPosition определяет
 позицию элемента прокрутки в диаграмме.


## Комментарии


Значение свойства с помощью метода setScrollerPosition,
 а возвращается с помощью метода getScrollerPosition.
 Из JSON значение задать нельзя.
