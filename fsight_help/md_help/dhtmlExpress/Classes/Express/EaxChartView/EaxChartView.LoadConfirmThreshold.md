# EaxChartView.LoadConfirmThreshold


## Синтаксис


LoadConfirmThreshold: Number;


## Описание


Свойство LoadConfirmThreshold
 устанавливает максимальное число точек диаграммы экспресс-отчета.


## Комментарии


Значение свойства устанавливается только из JSON.


Если число точек диаграммы превысит максимальное, будет выдано предупреждающее
 сообщение о возможной продолжительной загрузке данных.


Значение по умолчанию - «10000».
