# EaxChartView.ShowLegendTooltip


## Синтаксис


ShowLegendTooltip: Boolean


## Описание


Свойство ShowLegendTooltip определяет, будет ли отображаться всплывающая подсказка для элементов легенды.


## Комментарии


По умолчанию для свойства установлено значение true, то есть подсказка отображается.
