# EaxChartView.ScrollerMaxPointsInSight


## Синтаксис


ScrollerMaxPointsInSight: Number;


## Описание


Свойство ScrollerMaxPointsInSight
 определяет максимальное количество шагов временной шкалы, для которых
 будут отображены данные на диаграмме.


## Комментарии


Значение свойства с помощью метода setScrollerMaxPointsInSight,
 а возвращается с помощью метода getScrollerMaxPointsInSight.
 Из JSON значение задать нельзя.
