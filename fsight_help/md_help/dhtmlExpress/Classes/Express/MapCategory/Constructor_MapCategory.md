# Конструктор MapCategory


## Синтаксис


PP.Exp.Ui.MapCategory(settings);


## Параметры


*settings.* JSON-объект со значениями свойств класса.


## Описание


Конструктор MapCategory создает экземпляр класса MapCategory.
