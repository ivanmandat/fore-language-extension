# Конструктор EaxDocument


## Синтаксис


PP.Exp.EaxDocument(settings);


## Параметры


*settings.* JSON-объект со значениями свойств класса.


## Описание


Конструктор EaxDocument** создает экземпляр класса **EaxDocument**.


## Пример


Конструктор предназначен для внутреннего использования.
