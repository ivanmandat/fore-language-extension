# EaxDocument.getSheetName


## Синтаксис


getSheetName(keyOrId);


## Параметры


*keyOrId.* Ключ или идентификатор документа экспресс-отчета.


## Описание


Метод getSheetName** возвращает наименование листа документа экспресс-отчета по его ключу или идентификатору.


## Комментарии


Метод возвращает значение типа String.
