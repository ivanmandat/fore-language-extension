# EaxDocument.DocumentMetadata


## Синтаксис


DocumentMetadata: Object


## Описание


Свойство DocumentMetadata** содержит сведения о документе экспресс-отчета.


## Комментарии


Значение свойства можно установить из JSON и с помощью метода getDocumentMetadata, а возвратить с помощью метода getDocumentMetadata.


Свойство содержит значение в виде JSON-объекта со следующими полями: dirty - признак того, что документ экспресс-отчета был изменен, hasPivot - признак наличия измерения у экспресс-отчета, obInst - экземпляр объекта, sheets - содержит массив листов документа экспресс-отчета, windowsPosition - позиция окна с отчетом.
