# EaxDocument.getSheet


## Синтаксис


getSheet(keyOrId);


## Параметры


*keyOrId.* Ключ или идентификатор документа экспресс-отчета.


## Описание


Метод getSheet** возвращает лист документа экспресс-отчета по его ключу или идентификатору.


## Комментарии


Метод возвращает объект типа PP.Exp.[EaxAnalyzer](../EaxAnalyzer/EaxAnalyzer.htm).
