# TitleBox.refreshSelections


## Синтаксис


refreshSelections(dimKeys);


## Параметры


*dimKeys.* Массив ключей измерений. Необязательный параметр.


## Описание


Метод refreshSelections** обновляет состояние заголовка экспресс-отчета в соответствии с состоянием модели.
