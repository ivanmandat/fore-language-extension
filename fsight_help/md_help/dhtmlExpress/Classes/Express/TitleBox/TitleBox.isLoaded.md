# TitleBox.isLoaded


## Синтаксис


isLoaded();


## Описание


Метод isLoaded** возвращает признак того, что заголовок экспресс-отчета загружен.


## Комментарии


Если заголовок экспресс-отчета загружен, то метод возвращает значение true, иначе - значение false.
