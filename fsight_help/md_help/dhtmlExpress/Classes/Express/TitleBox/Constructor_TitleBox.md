# Конструктор TitleBox


## Синтаксис


PP.Exp.Ui.TitleBox(settings);


## Параметры


*settings.* JSON-объект со значениями свойств класса.


## Описание


Конструктор TitleBox** создает экземпляр класса **TitleBox**.


## Пример


Конструктор предназначен для внутреннего использования.
