# TitleBox.Source


## Синтаксис


Source: PP.Exp.[EaxDocument](../EaxDocument/EaxDocument.htm)


## Описание


Свойство Source** определяет источник данных для заголовка экспресс-отчета.


## Комментарии


Значение свойства можно задать из JSON и при помощи метода setSource, а получить - с помощью метода getSource.


Метод setSource принимает в качестве параметра объект класса PP.Exp.[EaxDocument](../EaxDocument/EaxDocument.htm).
