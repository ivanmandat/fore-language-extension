# TitleBox.EnableEdit


## Синтаксис


EnableEdit: Boolean;


## Описание


Свойство EnableEdit** определяет, разрешено ли редактирование заголовка экспресс-отчёта.


## Комментарии


Значение свойства можно установить из JSON и при помощи метода setEnableEdit, а возвратить - при помощи метода getEnableEdit.


Если значение свойства установлено в true (по умолчанию), то редактирование заголовка разрешено, если в false, то запрещено.
