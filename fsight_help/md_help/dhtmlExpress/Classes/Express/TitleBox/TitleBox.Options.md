# TitleBox.Options


## Синтаксис


Options: Object


## Описание


Свойство Options** устанавливает и возвращает настройки заголовка экспресс-отчета.


## Комментарии


Значение свойства устанавливается при помощи метода setOptions, а возвращается - при помощи метода getOptions. Из JSON значение свойства установить нельзя.


Свойство принимает и возвращает значение в виде объекта JSON со следующими полями: visible - признак видимости заголовка, xamlText - настройки содержимого заголовка в виде строки XAML, xamlTitle - настройки заголовка в виде строки XAML.
