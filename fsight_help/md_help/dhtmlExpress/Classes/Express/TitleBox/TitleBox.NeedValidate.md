# TitleBox.NeedValidate


## Синтаксис


NeedValidate: boolean


## Описание


Свойство NeedValidate** определяет, нуждается ли заголовок в проверке.


## Комментарии


Если свойство имеет значение true, то требуется проверка заголовка, если false, то не требуется.


Значение свойства можно установить из JSON и при помощи метода setNeedValidate, а возвратить - при помощи метода getNeedValidate.
