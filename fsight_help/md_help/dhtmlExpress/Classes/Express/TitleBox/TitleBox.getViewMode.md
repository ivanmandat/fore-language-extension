# TitleBox.getViewMode


## Синтаксис


getViewMode();


## Описание


Метод getViewMode** возвращает значение флага «Режим просмотра» для заголовка экспресс-отчета.


## Комментарии


Метод возвращает значение true, если заголовок экспресс-отчета открыт в режиме просмотра, и false, если он открыт в режиме редактирования.
