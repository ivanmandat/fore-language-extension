# TitleBox.Instance


## Синтаксис


Instance: TitleBox


## Описание


Свойство Instance** возвращает заголовок экспресс-отчета.


## Комментарии


Значение свойства Instance можно задавать только из JSON, а возвращать - при помощи метода getInstance. При этом будет возвращен объект класса PP.Exp.Ui.TitleBox.
