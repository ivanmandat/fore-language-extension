# Конструктор DimViewOptionsDialog


## Синтаксис


PP.Exp.Ui.DimViewOptionsDialog(settings);


## Параметры


*settings.* JSON-объект со значениями свойств компонента.


## Описание


Конструктор DimViewOptionsDialog создает экземпляр компонента DimViewOptionsDialog.
