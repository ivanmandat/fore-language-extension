# DimViewOptionsDialog.ActiveDimMeta


## Синтаксис


ActiveDimMeta: Object


## Описание


Свойство ActiveDimMeta** устанавливает метаданные текущего измерения в диалоге «Настройка размещения».


## Комментарии


Значение свойства в виде JSON-объекта можно установить из JSON и с помощью метода setActiveDimMeta, а возвратить - с помощью метода getActiveDimMeta.


Метод setActiveDimMeta принимает в качестве параметра JSON-объект.
