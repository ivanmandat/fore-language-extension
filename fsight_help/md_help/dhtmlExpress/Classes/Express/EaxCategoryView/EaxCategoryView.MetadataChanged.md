# EaxCategoryView.MetadataChanged


## Синтаксис


MetadataChanged: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие MetadataChanged наступает после изменения метаданных вкладки ленты инструментов экспресс-отчета.
