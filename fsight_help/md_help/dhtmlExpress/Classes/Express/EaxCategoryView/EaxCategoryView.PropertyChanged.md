# EaxCategoryView.PropertyChanged


## Синтаксис


PropertyChanged: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие PropertyChanged наступает после изменения свойства вкладки на ленте инструментов экспресс-отчета.
