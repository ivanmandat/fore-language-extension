# EaxCategoryView.getGrowthIndicatorMenu


## Синтаксис


getGrowthIndicatorMenu();


## Описание


Метод getGrowthIndicatorMenu**
 возвращает меню «Индикатор роста»
 на ленте инструментов экспресс-отчета.


## Комментарии


Метод возвращает объект типа PP.Ui.Menu.
 На ленте инструментов данное меню выглядит следующим образом:
