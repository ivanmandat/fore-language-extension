# EaxCategoryView.getTableCFormatState


## Синтаксис


getTableCFormatState(value);


## Параметры


*value.* Значение, задаваемое с помощью перечисления [PP.Exp.Ui.TabTablePredefinedScale](../../../Enums/TabTablePredefinedScale.htm).


## Описание


Метод getTableCFormatState** возвращает свойства меню ленты инструментов «Условное форматирование».
