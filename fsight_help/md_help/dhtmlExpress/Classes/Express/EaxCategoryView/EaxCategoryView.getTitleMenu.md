# EaxCategoryView.getTitleMenu


## Синтаксис


getTitleMenu();


## Описание


Метод getTitleMenu** возвращает меню «Выбрать данные» на вкладке «Главная» ленты инструментов экспресс-отчета.


## Комментарии


Метод возвращает объект типа PP.Ui.Menu. На ленте инструментов данное меню выглядит следующим образом:
