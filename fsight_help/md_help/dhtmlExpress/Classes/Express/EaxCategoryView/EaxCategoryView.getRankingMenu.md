# EaxCategoryView.getRankingMenu


## Синтаксис


getRankingMenu();


## Описание


Метод getRankingMenu**
 возвращает меню «Отобразить как ранги»
 на ленте инструментов экспресс-отчета.


## Комментарии


Метод возвращает объект типа PP.Ui.Menu.
 На ленте инструментов данное меню выглядит следующим образом:
