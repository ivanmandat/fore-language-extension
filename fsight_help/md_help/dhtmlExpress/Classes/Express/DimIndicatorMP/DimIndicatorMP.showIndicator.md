# DimIndicatorMP.showIndicator


## Синтаксис


showIndicator(indName: String);


## Параметры


*indName.* Ключ индикатора вкладки «Показатели».


## Описание


Метод showIndicator** разворачивает вкладку «Показатели» и фокусируется на элементе измерения, на котором сейчас установлен индикатор.


## Комментарии


Массив индикаторов с их ключами и значениями можно получить с помощью внутреннего свойства _IndicatorElements.
