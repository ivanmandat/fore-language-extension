# DimIndicatorMP.IndicatorElements


## Синтаксис


IndicatorElements: Array


## Описание


Свойство IndicatorElements** задает список индикаторов, определяющих способ отображения данных в рабочей области экспресс-отчета и отображаемых в меню при выборе измерений на вкладке панели свойств «Показатели».


## Комментарии


Свойство **IndicatorElements** задает массив объектов вида: { Key : "<Name>", Value : <Number> }.
