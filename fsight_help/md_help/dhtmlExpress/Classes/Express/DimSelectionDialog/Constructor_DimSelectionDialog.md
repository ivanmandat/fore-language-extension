# Конструктор DimSelectionDialog


## Синтаксис


PP.Exp.Ui.DimSelectionDialog(settings);


## Параметры


*settings.* JSON-объект
 со значениями свойств компонента.


## Описание


Конструктор DimSelectionDialog
 создает экземпляр компонента [DimSelectionDialog](../../../Components/Express/DimSelectionDialog/DimSelectionDialog.htm).
