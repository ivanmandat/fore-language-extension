# DimSelectionDialog.Source


## Синтаксис


Source: [PP.Exp.EaxDocument](../EaxDocument/EaxDocument.htm)


## Описание


Свойство Source** определяет
 источник данных для компонента.


## Комментарии


Значение свойства можно задать из JSON при помощи метода setSource,
 а получить методом getSource.


Метод setSource принимает в
 качестве параметра объект класса [PP.Exp.EaxDocument](../EaxDocument/EaxDocument.htm).
