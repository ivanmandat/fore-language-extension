# EaxModeSelectorView.IsViewModeAvailable


## Синтаксис


IsViewModeAvailable(mode: PP.Exp.EaxViewMode);


## Параметры


*mode.* Режим отображения
 представлений данных.


## Описание


Метод IsViewModeAvailable**
 определяет, доступен ли указанный режим отображения представлений данных.


## Комментарии


Метод возвращает значение true,
 если указанный режим отображения представлений данных доступен и значение
 false - если недоступен.
