# EaxRibbonView.Saved


## Синтаксис


Saved: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: Report - экспресс-отчет, ResponseText - ответное сообщение.


## Описание


Событие Saved наступает при выборе пункта главного меню «Сохранить».
