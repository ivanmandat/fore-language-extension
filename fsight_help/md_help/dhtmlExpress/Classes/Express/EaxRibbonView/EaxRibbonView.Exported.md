# EaxRibbonView.Exported


## Синтаксис


Exported: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: format - наименование формата, в который экспортируется экспресс-отчет.


## Описание


Событие Exported наступает при выборе пункта в меню «Экспорт» на ленте инструментов экспресс-отчета.
