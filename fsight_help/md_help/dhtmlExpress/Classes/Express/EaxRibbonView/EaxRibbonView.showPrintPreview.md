# EaxRibbonView.showPrintPreview


## Синтаксис


showPrintPreview(src);


## Параметры


*src.* Источник данных для экспресс-отчёта, экземпляр класса PP.Exp.EaxDocument.


## Описание


Метод showPrintPreview** открывает страницу для предварительного просмотра печатной версии экспресс-отчета.
