# EaxRibbonView.NewReport


## Синтаксис


NewReport: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: FromFile - признак того, что экспресс-отчёт создаётся из файла.


## Описание


Событие NewReport наступает при выборе пункта меню «Новый отчёт».
