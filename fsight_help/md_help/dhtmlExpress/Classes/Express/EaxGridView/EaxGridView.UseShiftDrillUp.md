# EaxGridView.UseShiftDrillUp


## Синтаксис


UseShiftDrillUp: boolean


## Описание


Свойство UseShiftDrillUp разрешает поднятие отметки на уровень вверх по иерархии при нажатии на кнопку SHIFT.


## Комментарии


Если значение свойства установлено в true, то поднятие отметки на уровень вверх по иерархии при нажатии на кнопку SHIFT разрешено, если false, то запрещено. По умолчанию имеет значение true.
