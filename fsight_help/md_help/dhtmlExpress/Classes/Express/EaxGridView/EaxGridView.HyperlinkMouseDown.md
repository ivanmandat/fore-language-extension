# EaxGridView.HyperlinkMouseDown


## Синтаксис


HyperlinkMouseDown: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: RowIndex, ColIndex - индекс строки и столбца ячейки, в которой была нажата гиперссылка, DrillMethod - направление перемещения отметки по иерархии при нажатии на кнопку SHIFT.


## Описание


Событие HyperlinkMouseDown наступает при нажатии на гиперссылку в таблице рабочей области экспресс-отчета.
