# EaxGridView.VisibleHeaders


## Синтаксис


VisibleHeaders: boolean


## Описание


Свойство VisibleHeaders задает признак видимости заголовков строк и столбцов таблицы в рабочей области экспресс-отчета.


## Описание


Если значение свойства равно true, то заголовки строк и столбцов таблицы будут отображаться; если равно false, то они показаны не будут.
