# EaxGridView.SelectionChanging


## Синтаксис


SelectionChanging: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: Selection - диапазон выделенных ячеек в JSON-формате.


## Описание


Событие SelectionChanging наступает в процессе изменения выделения в таблице рабочей области экспресс-отчета до тех пор, пока кнопка мыши не будет отпущена.
