# EaxGridView.StructureChanged


## Синтаксис


StructureChanged: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: ChangedStruct - структура таблицы.


## Описание


Событие StructureChanged генерируется при изменении структуры таблицы: при изменении размеров строк и столбцов.
