# EaxGridView.DeferSelectionChangedDelay


## Синтаксис


DeferSelectionChangedDelay: Number


## Описание


Свойство DeferSelectionChangedDelay
 определяет задержку перед изменением выделения таблицы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDeferSelectionChangedDelay,
 а возвращается - с помощью метода getDeferSelectionChangedDelay.


Значение по умолчанию: 555 мс.
