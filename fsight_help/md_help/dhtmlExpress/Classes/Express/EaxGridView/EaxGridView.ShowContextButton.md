# EaxGridView.ShowContextButton


## Синтаксис


ShowContextButton: boolean;


## Описание


Свойство ShowContextButton управляет
 видимостью кнопки контекстного меню таблицы в рабочей области экспресс-отчета.


## Комментарии


По умолчанию имеет значение true.
