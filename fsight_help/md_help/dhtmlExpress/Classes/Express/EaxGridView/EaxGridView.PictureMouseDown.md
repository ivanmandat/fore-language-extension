# EaxGridView.PictureMouseDown


## Синтаксис


PictureMouseDown: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие PictureMouseDown наступает при нажатии на пиктограмму внутри ячейки таблицы в рабочей области экспресс-отчета.
