# EaxGridView.RequestMetadata


## Синтаксис


RequestMetadata: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: IsReport - признак того, что необходимы метаданные для экспресс-отчета, IsStatistics - признак того, что необходимы метаданные для статистики.


## Описание


Событие RequestMetadata генерируется при отсутствии метаданных в источнике таблицы.
