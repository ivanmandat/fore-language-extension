# EaxGridView.EditMode


## Синтаксис


EditMode: Boolean;


## Описание


Свойство EditMode** определяет
 режим открытия таблицы рабочей области экспресс-отчёта.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Таблица открывается в режиме редактирования;


	- False. Таблица открывается
	 в режиме просмотра.
