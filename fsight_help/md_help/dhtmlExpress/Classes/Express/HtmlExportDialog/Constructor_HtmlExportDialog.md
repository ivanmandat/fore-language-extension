# Конструктор HtmlExportDialog


## Синтаксис


PP.Exp.Ui.HtmlExportDialog(settings);


## Параметры


*settings.* JSON-объект со значениями свойств компонента.


## Описание


Конструктор HtmlExportDialog** создает экземпляр компонента **HtmlExportDialog**.
