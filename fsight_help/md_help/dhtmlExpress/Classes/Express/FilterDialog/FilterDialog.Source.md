# FilterDialog.Source


## Синтаксис


Source();


## Описание


Свойство Source определяет источник
 данных в диалоге «Фильтр».


## Комментарии


Значение свойства Source можно
 устанавливать из JSON и с помощью метода setSource,
 а возвращать - при помощи свойства getSource.
