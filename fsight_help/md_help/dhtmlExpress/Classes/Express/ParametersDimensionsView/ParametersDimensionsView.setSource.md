# ParametersDimensionsView.setSource


## Синтаксис


setSource(value);


## Параметры


*value.* Значение источника данных.


## Описание


Метод setSource** устанавливает источник данных для компонента.


## Комментарии


Значение метода также можно устанавливать из JSON.
