# Конструктор ParametersDimensionsView


## Синтаксис


PP.Exp.Ui.ParametersDimensionsView(settings);


## Параметры


*settings.* JSON-объект со значениями свойств компонента.


## Описание


Конструктор ParametersDimensionsView** создает экземпляр компонента **ParametersDimensionsView**.
