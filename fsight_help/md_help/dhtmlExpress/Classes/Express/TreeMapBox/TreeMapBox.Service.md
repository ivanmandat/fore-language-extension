# TreeMapBox.Service


## Синтаксис


Service: [PP.Exp.EaxMdService](../EaxMdService/EaxMdService.htm);


## Описание


Метод Service определяет сервис для работы с контейнером плоского дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setService, возвращается с помощью метода getService.
