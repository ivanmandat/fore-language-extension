# TreeMapBox.isLoaded


## Синтаксис


isLoaded();


## Описание


Метод isLoaded определяет, загружен ли контейнер для плоского дерева.


## Комментарии


Метод возвращает значение true, если контейнер для плоского дерева загружен на странице, и значение false, если не загружен.
