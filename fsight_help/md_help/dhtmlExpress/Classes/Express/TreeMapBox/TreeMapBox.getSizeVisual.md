# TreeMapBox.getSizeVisual


## Синтаксис


getSizeVisual();


## Описание


Метод getSizeVisual возвращает настройки сопоставления данных контейнера плоского дерева размерам его элементов.


## Комментарии


Метод возвращает объект типа PP.SizeVisual.
