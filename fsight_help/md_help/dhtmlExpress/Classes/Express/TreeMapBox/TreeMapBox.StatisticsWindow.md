# TreeMapBox.StatisticsWindow


## Синтаксис


StatisticsWindow: [PP.Exp.Ui.StatisticsWindow](../StatisticsWindow/StatisticsWindow.htm);


## Описание


Свойство StatisticsWindow определяет для контейнера плоского дерева окно «Статистические характеристики».


## Комментарии


Значение свойства устанавливается из JSON и с помощью setStatisticsWindow, а возвращается с помощью метода getStatisticsWindow.


Окно открывается при выполнении команды контекстного меню «Статистические характеристики» или при вызове метода [TreeMapBox.showStatisticsWindow](TreeMapBox.showStatisticsWindow.htm).
