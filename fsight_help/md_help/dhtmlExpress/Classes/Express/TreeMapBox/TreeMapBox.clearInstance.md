# TreeMapBox.clearInstance


## Синтаксис


clearInstance();


## Описание


Метод clearInstance удаляет плоское дерево из контейнера.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента [TreeMapBox](../../../Components/Express/TreeMapBox/TreeMapBox.htm) с наименованием «treeMapBox» (см. «[Пример создания компонента TreeMapBox](../../../Components/Express/TreeMapBox/TreeMapBox_Example.htm)»). Удалим плоское дерево из контейнера:


treeMapBox.clearInstance();

В результате выполнения примера всё содержимое контейнера для плоского дерева было удалено.
