# TreeMapBox.RequestMetadata


## Синтаксис


RequestMetadata: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступный аргумент: IsReport - признак того, что запрашиваются данные для контейнера плоского дерева.


## Описание


Событие RequestMetadata генерируется при запросе метаданных для контейнера плоского дерева.
