# EaxMdService.setActiveSheet


## Синтаксис


setActiveSheet(doc, metadata, callback);


## Параметры


*doc.* Документ экспресс-отчета, который нужно сделать активным;


*metadata.* Метаданные;


*callback.* Возвратная функция.


## Описание


Метод setActiveSheet** делает активным указанный лист документа экспресс-отчета.
