# EaxMdService.EaxOpened


## Синтаксис


EaxOpened: function (sender, args);

sender. Источник события;


args. Информация о событии. Доступные аргументы: Analyzer - открытый экспресс-отчет.

Событие EaxOpened** генерируется при открытии экспресс-отчета.
