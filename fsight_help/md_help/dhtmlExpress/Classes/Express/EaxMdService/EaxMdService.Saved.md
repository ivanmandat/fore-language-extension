# EaxMdService.Saved


## Синтаксис


Saved: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: Report - документ экспресс-отчета, ResponseText - текст ответа на данное событие от сервиса.


## Описание


Событие Saved** наступает при сохранении экспресс-отчета.
