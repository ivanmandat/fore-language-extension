# EaxMdService.updateFilterTransformation


## Синтаксис


updateFilterTransformation(report: PP.Exp.[EaxAnalyzer](../EaxAnalyzer/EaxAnalyzer.htm),
 transformation: Object, callback: PP.Delegate);


## Параметры


*report.* Документ экспресс-отчёта;


*transformation.* Обновляемая
 трансформация;


*callback.* Возвратная функция.


## Описание


Метод updateFilterTransformation
 обновляет трансформацию-фильтр экспресс-отчета.

Пример использования метода приведен в описании свойства [EaxMdService.Transformation](EaxMdService.Transformation.htm).
