# EaxMdService.MdLoaded


## Синтаксис


MdLoaded: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.
 Доступные аргументы: Source - загружаемый экспресс-отчет.


## Описание


Событие MdLoaded генерируется
 при загрузке метаданных в экспресс-отчёт.
