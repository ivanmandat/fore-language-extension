# EaxMdService.setTransformationState


## Синтаксис


setTransformationState(report: PP.Exp.[EaxAnalyzer](../EaxAnalyzer/EaxAnalyzer.htm),
 type: PP.Exp.[TransfType](../../../Enums/PP.Exp.TransfType.htm),
 state: PP.Exp.[TransfState](../../../Enums/PP.Exp.TransfState.htm),
 dimKey: Number, dimElKey: Number, callback: PP.Delegate);


## Параметры


*report.* Документ экспресс-отчёта;


*type.* Тип коллекции
 трансформаций;


state. Устанавливаемое состояние
 коллекции;


dimKey. Ключ измерения;


dimElKey. Ключ элемента измерения;


*callback.* Возвратная функция.


## Описание


Метод setTransformationState
 устанавливает состояния коллекции трансформаций, заданной для элемента
 измерения.
