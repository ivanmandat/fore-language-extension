# EaxMdService.Export


## Синтаксис


Export(report: PP.Exp.[EaxDocument](../EaxDocument/EaxDocument.htm), data: Object, callback: PP.Delegate);


## Параметры


*report.* Документ экспресс-отчета;


*data.* Параметры экспорта (exportData - экспортируемые данные, IsPrintRequest - признак запроса вывода на печать);


*callback.* Параметры экспресс-отчета.


## Описание


Метод Export** экспортирует данные экспресс-отчета.
