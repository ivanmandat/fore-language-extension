# EaxMdService.getStatData


## Синтаксис


getStatData(report: PP.Exp.[EaxDocument](../EaxDocument/EaxDocument.htm), callback: PP.Delegate);


## Параметры


*report.* Экспресс-отчет, для данных которого нужно получить статистические характеристики;


*callback.* Возвратная функция на обновление статистических характеристик.


## Описание


Метод getStatData** получает статистические характеристики данных экспресс-отчета.
