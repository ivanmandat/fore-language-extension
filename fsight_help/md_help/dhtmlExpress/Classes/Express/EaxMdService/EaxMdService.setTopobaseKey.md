# EaxMdService.setTopobaseKey


## Синтаксис


setTopobaseKey (report: [PP.Exp.EaxAnalyzer](../EaxAnalyzer/EaxAnalyzer.htm),
 key: Integer, callback: PP.Delegate);


## Параметры


report.
 Активный лист отчета;


key.
 Ключ топоосновы;


callback.
 Процедура обратного вызова.


## Описание


Метод setTopobaseKey устанавливает
 ключ топоосновы для карты экспресс-отчета.

Пример использования приведен в описании метода [EaxMdService.resetTopobase](EaxMdService.resetTopobase.htm).
