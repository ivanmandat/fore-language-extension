# EaxMdService.setTransformationPos


## Синтаксис


setTransformationPos(report: [PP.Exp.EaxAnalyzer](../EaxAnalyzer/EaxAnalyzer.htm),
 tKey: Number, tPos: Number, calculationMode : PP.Exp.TransfCalculationMode,
 callback: PP.Delegate, ext: Object);


## Параметры


*report.* Документ экспресс-отчёта;


*tKey.* Ключ трансформации;


tPos. Новая позиция трансформации;


calculationMode. Позиция вычисления
 относительно расчетов итогов и агрегации. Необязательный параметр;


*callback.* Возвратная функция. Необязательный параметр;


ext. Дополнительная информация
 для передачи в запрос Необязательный параметр.


## Описание


Метод setTransformationPos устанавливает
 трансформацию в новую позицию в коллекции.

Пример использования метода приведен в описании свойства [EaxMdService.Transformation](EaxMdService.Transformation.htm).
