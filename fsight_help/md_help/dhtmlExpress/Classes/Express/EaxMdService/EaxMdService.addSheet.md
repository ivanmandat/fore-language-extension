# EaxMdService.addSheet


## Синтаксис


addSheet(doc: PP.Exp.[EaxDocument](../EaxDocument/EaxDocument.htm), sheetName: String, setIsActive: Boolean, callback: PP.Delegate);


## Параметры


*d*oc. Документ в который добавляется лист;


*sheetName.* Имя добавляемого листа;


*setIsActive.* Сделать ли лист активным;


*callback.* Добавляет новый лист в документ ;


## Описание


Метод addSheet** добавляет новый лист в документ экспресс-отчета.
