# EaxMdService.EaxDocumentOpened


## Синтаксис


EaxDocumentOpened: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: Analyzer - открытый экспресс-отчет, Document - документ экспресс-отчета.


## Описание


Событие EaxDocumentOpened** генерируется при открытии документа экспресс-отчета.
