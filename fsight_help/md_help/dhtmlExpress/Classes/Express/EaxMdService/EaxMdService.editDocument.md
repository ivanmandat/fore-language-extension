# EaxMdService.editDocument


## Синтаксис


editDocument(key: Number, callback: PP.Delegate||function, params);


## Параметры


*key.* Ключ экспресс-отчёта, который необходимо открыть;


*callback.* Возвратная функция, которую необходимо вызвать после открытия отчета;


*params.* Параметры, с которыми необходимо открыть объект.


## Описание


Метод editDocument** открывает документ экспресс-отчет в режиме редактирования.


## Комментарии


Метод возвращает объект типа PP.Exp.[EaxDocument](../EaxDocument/EaxDocument.htm)
