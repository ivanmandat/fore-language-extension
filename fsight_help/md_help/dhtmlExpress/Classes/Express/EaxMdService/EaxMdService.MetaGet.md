# EaxMdService.MetaGet


## Синтаксис


MetaGet: Object;


## Описание


Свойство MetaGet определяет
 метаданные сервиса для работы с экспресс-отчетом.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMetaGet, а возвращается с помощью
 метода getMetaGet.
