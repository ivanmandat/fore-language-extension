# EaxMdService.open


## Синтаксис


open(key: Number, callback: PP.Delegate||function, params);


## Параметры


*key.* Ключ экспресс-отчёта, который необходимо открыть;


*callback.* Возвратная функцию, которую необходимо вызвать после открытия экспресс-отчета;


*params.* Параметры, с которыми необходимо открыть объект.


## Описание


Метод open** открывает экспресс-отчет в режиме чтения.


## Комментарии


Метод возвращает объект типа PP.Exp.[EaxAnalyzer](../EaxAnalyzer/EaxAnalyzer.htm).
