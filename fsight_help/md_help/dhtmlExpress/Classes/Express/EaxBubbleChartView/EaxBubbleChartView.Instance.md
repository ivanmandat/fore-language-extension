# EaxBubbleChartView.Instance


## Синтаксис


Instance: PP.Ui.[EaxBubbleChart](../EaxBubbleChart/EaxBubbleChart.htm)


## Описание


Свойство Instance** содержит ссылку на экземпляр пузырьковой диаграммы экспресс-отчета.


## Комментарии


Значение свойства можно установить с помощью метода setInstance и из JSON, а возвратить - с помощью метода getInstance.
