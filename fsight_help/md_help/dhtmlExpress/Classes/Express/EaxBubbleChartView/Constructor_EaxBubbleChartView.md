# Конструктор EaxBubbleChartView


## Синтаксис


PP.Exp.Ui.EaxBubbleChartView(settings);


## Параметры

## Описание


Конструктор EaxBubbleChartView** создает экземпляр класса **EaxBubbleChartView**.
