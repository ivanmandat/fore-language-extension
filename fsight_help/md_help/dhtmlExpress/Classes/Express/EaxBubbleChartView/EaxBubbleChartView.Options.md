# EaxBubbleChartView.Options


## Синтаксис


Options: Object


## Описание


Свойство Options** содержит настройки представления пузырьковой диаграммы экспресс-отчета.


## Комментарии


Значение свойства можно установить с помощью метода setOptions и из JSON, а возвратить - с помощью метода getOptions.


Свойство содержит JSON-объект с полем BubbleChart.
