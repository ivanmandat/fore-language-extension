# Конструктор TitleCategory


## Синтаксис


PP.Exp.Ui.TitleCategory(settings);


## Параметры


*settings.* JSON-объект со значениями свойств класса.


## Описание


Конструктор TitleCategory создает экземпляр класса TitleCategory.
