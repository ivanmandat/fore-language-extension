# EaxPropertyBar.clearDims


## Синтаксис


clearDims();


## Описание


Метод clearDims** удаляет все измерения.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента EaxPropertyBar с наименованием «eaxPropertyBar» (см. «[Пример совместного размещения EaxPropertyBar и EaxGrid](../../../Components/Express/EaxGrid/EaxGrid_Example.htm)»). Удалим измерения во всех мастерах:


// Очищает измерения во всех мастерах
eaxPropBar.clearDims();

После выполнения данного примера будут удалены измерения во всех мастерах панели свойств.
