# EaxPropertyBar.DataView


## Синтаксис


DataView: Function;


## Описание


Свойство DataView определяет представление данных, которое требуется связать с боковой панелью экспресс-отчёта.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDataView, а возвращается - с помощью метода getDataView.
