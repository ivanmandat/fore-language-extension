# EaxPropertyBar.getTableViewPanel


## Синтаксис


getTableViewPanel();


## Описание


Метод getTableViewPanel возвращает вкладку «Вид», входящую в состав панели свойств экспресс-отчёта.


## Комментарии


Метод возвращает значение типа [PP.Exp.Ui.TableViewPanel](../TableViewPanel/TableViewPanel.htm).
