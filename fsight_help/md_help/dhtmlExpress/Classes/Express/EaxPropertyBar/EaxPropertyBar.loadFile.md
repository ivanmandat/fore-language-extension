# EaxPropertyBar.loadFile


## Синтаксис


loadFile(fileName: String, callback: functions|PP.Delegate, argumentss: Array);


## Параметры


fileName. Наименование загружаемого файла сценария;


callback. Возвратная функция;


argumentss. Массив аргументов.


## Описание


Метод loadFile загружает файл сценария.
