## Описание


Метод getBubbleTreeMaster возвращает
 мастер для настройки пузырькового дерева.


## Комментарии


Метод возвращает значение типа [PP.Ui.BubbleTreeMaster](dhtmlBubbleTree.chm::/Classes/BubbleTreeMaster/BubbleTreeMaster.htm).
