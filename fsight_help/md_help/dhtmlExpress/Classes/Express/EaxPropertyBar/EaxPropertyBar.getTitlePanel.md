# EaxPropertyBar.getTitlePanel


## Синтаксис


getTitlePanel();


## Описание


Метод getTitlePanel** возвращает панель для настройки заголовка «Заголовок».


## Комментарии


Метод возвращает экземпляр класса PP.Exp.Ui.TitleStructurePanel.
