# EaxPropertyBar.getChartMaster


## Синтаксис


getChartMaster(chartView);


## Параметры


*chartView. Необязательный параметр. Является представлением диаграммы.*


## Описание


Метод getChartMaster** возвращает мастер диаграммы.
