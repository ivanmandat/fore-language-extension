# EaxPropertyBar.showChartMaster


## Синтаксис


showChartMaster(itemId: [PP.Exp.Ui.PropertyGroups](../../../Enums/PropertyGroups.htm));


## Параметры


itemId. Идентификатор группы вкладок, которые требуется отобразить в мастере для управления диаграммой.


## Описание


Метод showChartMaster отображает мастер для управления диаграммой.
