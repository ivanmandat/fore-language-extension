# EaxPropertyBar.Service


## Синтаксис


Service: [PP.Exp.EaxMdService](../EaxMdService/EaxMdService.htm);


## Описание


Свойство Service определяет сервис для работы с боковой панелью экспресс-отчёта.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setService, а возвращается - с помощью метода getService.
