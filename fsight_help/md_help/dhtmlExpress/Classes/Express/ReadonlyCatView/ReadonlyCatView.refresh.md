# ReadonlyCatView.refresh


## Синтаксис


refresh();


## Описание


Метод refresh** обновляет вкладку ленты инструментов «Главная» в режиме просмотра экспресс-отчета.


## Пример


Использование данного метода аналогично использованию метода [TitleCatView.refresh](../TitleCatView/TitleCatView.refresh.htm).
