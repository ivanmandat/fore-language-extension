# EaxDimBarMasterPanel.IndexPosition


## Синтаксис


IndexPosition: Number


## Описание


Свойство IndexPosition** устанавливает и возвращает индекс позиции, в которой размещается измерение.


## Комментарии


Значение свойства **IndexPosition** также задается при помощи метода **setIndexPosition** и возвращается при помощи метода g**etIndexPosition**.
