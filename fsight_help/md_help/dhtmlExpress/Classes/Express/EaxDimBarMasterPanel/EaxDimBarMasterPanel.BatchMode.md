# EaxDimBarMasterPanel.BatchMode


## Синтаксис


BatchMode: Boolean;


## Описание


Свойство BatchMode определяет,
 отправляются ли запросы серверу в пакете данных.


## Комментарии


Допустимые значения свойства:


	- true. Запросы отправляются
	 серверу в пакете данных (по умолчанию);


	- false. Запросы не отправляются
	 серверу в пакете данных.


Данное свойство используется для управления отправкой запросов серверу.
 Если оно содержит значение true,
 то сначала требуется включить пакетный режим для формирования пакета данных
 с помощью метода Metabase.beginBatch,
 а затем, чтобы отправить сформированный пакет данных серверу, необходимо
 завершить этот режим с помощью метода Metabase.endBatch.


В случае, если свойство содержит значение false,
 то запросы отправляются серверу автоматически без формирования пакета
 данных.


Значение свойства устанавливается из JSON при помощи метода setBatchMode,
 а возвращается методом getBatchMode.
