# Конструктор EaxRibbon


## Синтаксис


PP.Exp.Ui.EaxRibbon(settings);


## Параметры


*settings.* JSON-объект со значениями свойств класса.


## Описание


Конструктор EaxRibbon создает экземпляр класса EaxRibbon.
