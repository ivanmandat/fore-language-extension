# EaxDimensionTree.EaxDimKey


## Синтаксис


EaxDimKey: Number;


## Описание


Свойство EaxDimKey определяет
 ключ измерения.


## Комментарии


Значение свойства задается через JSON и при помощи метода setEaxDimKey(value,
 dataSourceKey), возвращается при помощи метода getEaxDimKey.


Параметры метода setEaxDimKey:


  value. Ключ измерения;


  dataSourceKey.
 Ключ источника данных.
