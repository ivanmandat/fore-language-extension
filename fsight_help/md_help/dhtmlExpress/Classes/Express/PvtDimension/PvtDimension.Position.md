# PvtDimension.Position


## Синтаксис


Position: [PP.Exp.PvtDimPos](../../../Enums/PvtDimPos.htm);


## Описание


Свойство Position определяет
 расположение измерения.


## Комментарии


Допустимые значения:


	- Left. По строкам.


	- Top. По столбцам.


	- Fixed. Фиксированное измерение.


Значение свойства Position можно
 устанавливать из JSON и с помощью метода setPosition,
 а возвращать - при помощи свойства getPosition.
