# PvtDimension.MultipleAttributesEnabled


## Синтаксис


MultipleAttributesEnabled: Boolean;


## Описание


Свойство MultipleAttributesEnabled
 определяет использование нескольких атрибутов в наименовании.


## Комментарии


Значение свойства MultipleAttributesEnabled
 можно устанавливать из JSON и с помощью метода setMultipleAttributesEnabled,
 а возвращать - при помощи свойства getMultipleAttributesEnabled.


По умолчанию содержит значение true.
