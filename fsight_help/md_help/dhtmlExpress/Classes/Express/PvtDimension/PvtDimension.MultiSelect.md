# PvtDimension.MultiSelect


## Синтаксис


MultiSelect: Boolean;


## Описание


Свойство MultiSelect определяет
 использование режима множественной отметки.


## Комментарии


Значение свойства MultiSelect
 можно устанавливать из JSON и с помощью метода setMultiSelect,
 а возвращать - при помощи свойства getMultiSelect.


По умолчанию содержит значение false.
