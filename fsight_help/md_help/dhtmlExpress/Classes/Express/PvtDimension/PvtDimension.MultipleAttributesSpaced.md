# PvtDimension.MultipleAttributesSpaced


## Синтаксис


MultipleAttributesSpaced: Boolean;


## Описание


Свойство MultipleAttributesSpaced
 определяет, необходимо ли добавлять пробел к разделителю нескольких атрибутов
 в наименовании.


## Комментарии


Значение свойства MultipleAttributesSpaced
 можно устанавливать из JSON и с помощью метода setMultipleAttributesSpaced,
 а возвращать - при помощи свойства getMultipleAttributesSpaced.


По умолчанию содержит значение true.
