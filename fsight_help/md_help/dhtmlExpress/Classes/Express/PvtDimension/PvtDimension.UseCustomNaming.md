# PvtDimension.UseCustomNaming


## Синтаксис


UseCustomNaming: Boolean;


## Описание


Свойство UseCustomNaming определяет
 выражение пользовательского наименования.


## Комментарии


Значение свойства UseCustomNaming
 можно устанавливать из JSON и с помощью метода setUseCustomNaming,
 а возвращать - при помощи свойства getUseCustomNaming.
