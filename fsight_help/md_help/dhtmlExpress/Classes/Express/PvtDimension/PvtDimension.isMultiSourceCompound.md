# PvtDimension.isMultiSourceCompound


## Синтаксис


isMultiSourceCompound(): Boolean;


## Описание


Метод isMultiSourceCompound
 определяет включение в составную отметку более чем одного источника отчета.


## Комментарии


Метод возвращает true, если
 составная отметка включает измерения по нескольким источникам отчета.
