# PvtDimension.MultipleAttributesSeparator


## Синтаксис


MultipleAttributesSeparator: String;


## Описание


Свойство MultipleAttributesSeparator
 определяет разделитель нескольких атрибутов в наименовании.


## Комментарии


Значение свойства MultipleAttributesSeparator
 можно устанавливать из JSON и с помощью метода setMultipleAttributesSeparator,
 а возвращать - при помощи свойства getMultipleAttributesSeparator.


По умолчанию содержит значение "|".
