# PvtDimension.NamingAttributes


## Синтаксис


MultiSelect: Array;


## Описание


Свойство NamingAttributes содержит
 массив идентификаторов атрибутов, используемых в наименовании элементов
 измерения.


## Комментарии


Значение свойства NamingAttributes
 можно устанавливать из JSON и с помощью метода setNamingAttributes,
 а возвращать - при помощи свойства getNamingAttributes.


Массив атрибутов составлен в порядке их следования.
