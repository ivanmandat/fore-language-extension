# PvtDimension.PvtMetadata


## Синтаксис


PvtMetadata: Object;


## Описание


Свойство PvtMetadata определяет
 метаданные измерения.


## Комментарии


Значение свойства PvtMetadata
 можно устанавливать из JSON и с помощью метода setPvtMetadata,
 а возвращать - при помощи свойства getPvtMetadata.


Содержит аналог объекта из wsdl: PvtDim.
