# PvtDimension.IsCanFix


## Синтаксис


IsCanFix: Boolean;


## Описание


Свойство IsCanFix определяет,
 может ли измерение быть частным.


## Комментарии


Значение свойства IsCanFix можно
 устанавливать из JSON и с помощью метода setIsCanFix,
 а возвращать - при помощи свойства getIsCanFix.
