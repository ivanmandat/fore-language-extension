# PvtDimension.IsFixed


## Синтаксис


IsFixed: Boolean;


## Описание


Свойство IsFixed определяет,
 является ли измерение частным.


## Комментарии


Значение свойства IsFixed можно
 устанавливать из JSON и с помощью метода setIsFixed,
 а возвращать - при помощи свойства getIsFixed.
