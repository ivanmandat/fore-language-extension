# EaxMapView.getViewMode


## Синтаксис


getViewMode();


## Описание


Метод getViewMode** возвращает признак того, открыт ли экспресс-отчет в режиме просмотра.


## Комментарии


Метод возвращает значение true, если экспресс-отчет открыт в режиме просмотра, и false, если он открыт в режиме редактирования.
