# EaxMapView.isSLAvailable


## Синтаксис


isSLAvailable();


## Описание


Метод isSLAvailable** определяет, доступен ли Silverlight-объект у карты экспресс-отчета.


## Комментарии


Метод возвращает значение true, если Silverlight-объект доступен, и false в противном случае.
