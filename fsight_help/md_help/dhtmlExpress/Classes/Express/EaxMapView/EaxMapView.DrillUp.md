# EaxMapView.DrillUp


## Синтаксис


DrillUp: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступный аргумент: ShapeId - идентификатор фигуры на карте, с которого переходим на верхний уровень.


## Описание


Событие DrillUp** наступает при возврате на верхний уровень в карте экспресс-отчета.
