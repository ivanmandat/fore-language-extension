# EaxMapView.GoogleLoadError


## Синтаксис


GoogleLoadError: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие GoogleLoadError** наступает при ошибке загрузки карты Google.
