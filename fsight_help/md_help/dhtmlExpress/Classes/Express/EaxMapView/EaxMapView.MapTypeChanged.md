# EaxMapView.MapTypeChanged


## Синтаксис


MapTypeChanged: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступное поле: Type - новый тип карты экспресс-отчета.


## Описание


Событие MapTypeChanged** наступает после изменения типа карты экспресс-отчета.
