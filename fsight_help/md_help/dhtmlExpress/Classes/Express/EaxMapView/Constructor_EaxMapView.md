# Конструктор EaxMapView


## Синтаксис


PP.Exp.Ui.EaxMapView(settings);


## Параметры


*settings.* JSON-объект со значениями свойств класса.


## Описание


Конструктор EaxMapView** создает экземпляр класса **EaxMapView**.


## Пример


Конструктор предназначен для внутреннего использования.
