# EaxMapView.onDropDownStateChanged


## Синтаксис


onDropDownStateChanged(isOpening);


## Параметры


*isOpening.* Признак замены Silverlight-карты на изображение SVG. Если параметр равен true, то данная замена будет осуществлена, а раскрывающиеся списки будут отображены поверх карты, иначе раскрывающийся список будет перекрыт Silverlight-картой.


## Описание


Метод onDropDownStateChanged** переключает Silverlight-карту на изображение SVG-карты с целью отображения раскрывающихся списков поверх неё.


## Комментарии


Примером раскрывающихся списков могут быть меню кнопок на ленте инструментов или контекстное меню карты экспресс-отчета.
