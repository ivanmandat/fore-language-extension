# EaxMapView.isLoaded


## Синтаксис


isLoaded();


## Описание


Метод isLoaded** возвращает признак того, что карта экспресс-отчета загружена.


## Комментарии


Если пузырьковая диаграмма экспресс-отчета загружена, то метод возвращает значение true, иначе - значение false.
