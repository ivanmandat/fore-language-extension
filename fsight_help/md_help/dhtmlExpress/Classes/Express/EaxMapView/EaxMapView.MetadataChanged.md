# EaxMapView.MetadataChanged


## Синтаксис


MetadataChanged: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: ChangedSettings - массив измененных настроек карты экспресс-отчета, Settings - настройки карты.


## Описание


Событие MetadataChanged** наступает при изменении настроек карты экспресс-отчета.
