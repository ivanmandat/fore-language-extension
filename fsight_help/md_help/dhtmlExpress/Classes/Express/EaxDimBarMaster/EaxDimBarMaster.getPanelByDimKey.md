# EaxDimBarMaster.getPanelByDimKey


## Синтаксис


getPanelByDimKey(dimKey: Number, withoutCheck: Boolean);


## Параметры


*d*imKey. Ключ измерения;


*withoutCheck.* Определяет, нужно ли делать проверку на соответствие модели; по умолчанию имеет значение true.


## Описание


Метод getPanelByDimKey** возвращает панель мастера по ключу измерения.
