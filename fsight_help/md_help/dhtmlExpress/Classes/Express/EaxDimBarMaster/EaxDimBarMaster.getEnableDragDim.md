# EaxDimBarMaster.getEnableDragDim


## Синтаксис


getEnableDragDim();


## Описание


Метод getEnableDragDim** возвращает признак того, можно ли перемещать измерения.


## Комментарии


Если перемещение измерений в мастере разрешено, то метод возвращает значение true, если запрещено - то false.
