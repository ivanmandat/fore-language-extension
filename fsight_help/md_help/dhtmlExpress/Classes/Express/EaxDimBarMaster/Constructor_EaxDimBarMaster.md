# Конструктор EaxDimBarMaster


## Синтаксис


PP.Exp.Ui.EaxDimBarMaster(settings);


## Параметры


*settings.* JSON-объект со значениями свойств класса.


## Описание


Конструктор EaxDimBarMaster** создает экземпляр компонента **EaxDimBarMaster**.
