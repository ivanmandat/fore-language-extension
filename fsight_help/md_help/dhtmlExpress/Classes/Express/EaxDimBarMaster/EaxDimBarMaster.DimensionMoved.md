# EaxDimBarMaster.DimensionMoved


## Синтаксис


DimensionMoved: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.
 Доступные аргументы: MovedKeyDim - измерение, которое нужно переместить,
 DimPosition - предыдущая позиция измерения, Index - индекс новой позиции
 измерения.


## Описание


Событие DimensionMoved**
 наступает при перемещении измерения в мастере управления отметкой.
