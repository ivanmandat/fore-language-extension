# Конструктор ParetoDialog


## Синтаксис


PP.Exp.Ui.ParetoDialog(settings);


## Параметры


*settings.* JSON-объект со значениями свойств компонента.


## Описание


Конструктор ParetoDialog создает экземпляр компонента ParetoDialog.
