# Конструктор TableCategory


## Синтаксис


PP.Exp.Ui.TableCategory(settings);


## Параметры


*settings.* JSON-объект со значениями свойств класса.


## Описание


Конструктор TableCategory создает экземпляр класса TableCategory.
