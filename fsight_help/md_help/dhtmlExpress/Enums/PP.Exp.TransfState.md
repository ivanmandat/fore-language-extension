# PP.Exp.TransfState


Пространство имен: PP.Exp;


## Описание


Перечисление TransfState содержит
 состояния коллекции трансформаций элемента измерения.


Используется методом [EaxMdService.setTransformationState](../Classes/Express/EaxMdService/EaxMdService.setTransformationState.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Edit
		 Редактирование коллекции трансформаций.


		 Cancel
		 Отмена изменений коллекции трансформаций.


		 Save
		 Сохранение изменений коллекции трансформаций.
