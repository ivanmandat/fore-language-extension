# RibbonTableEnum


Пространство имен: PP.Exp.Ui;


## Описание


Перечисление RibbonTableEnum содержит
 операции по настройке таблицы экспресс-отчета.


Предназначено для внутреннего использования в событиях.


## Допустимые значения


		 Значение
		 Краткое описание


		 TableAngle
		 Поворот таблицы.


		 TablePareto
		 Распределение 80/20.


		 RibbonTableFilter
		 Фильтр таблицы.


		 RibbonTableNumber
		 Формат числа таблицы.


		 RibbonTableStyle
		 Стиль таблицы.


		 SelectedPartGridStyles
		 Стиль границ выбранных ячеек.


		 InternalGridStyles
		 Стиль внутренних границ.


		 GridStyles
		 Стиль границ.


		 RibbonTableCFormat
		 Условное форматирование.


		 RibbonTableCFormatEdit
		 Настройка условного форматирования.


		 GetSearchResult
		 Поиск.


		 ShowSearchResult
		 Результаты поиска.
