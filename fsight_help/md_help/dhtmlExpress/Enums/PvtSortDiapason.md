# PvtSortDiapason


Пространство имен: PP.Exp.Ui;


## Описание


Перечисление PvtSortDiapason
 содержит виды диапазонов сортировки данных измерений в экспресс-отчёте.


Используется методом [EaxMdService.setTables](../Classes/Express/EaxMdService/EaxMdService.setTables.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Owner
		 По диапазону, определённому пользователем.


		 Level
		 По уровню.


		 Group
		 По группе.


		 All
		 По всем значениям.


		 LevInSlot
		 По измерению в столбце.


		 DimInSlot
		 По уровню в столбце.
