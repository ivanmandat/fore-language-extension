# TextAlignment


Пространство имен: PP.Exp;


## Описание


Перечисление TextAlignment
 содержит варианты выравнивания текста.


Перечисление предназначено для внутреннего использования.


## Допустимые значения


		 Значение
		 Краткое описание


		 -1
		 Custom. Пользовательское
		 расположение.


		 1
		 Left. Выравнивание
		 по левому краю.


		 2
		 Center. Выравнивание
		 по центру.


		 3
		 Right. Выравнивание
		 по правому краю.


		 4
		 Justify. Выравнивание
		 по ширине.
