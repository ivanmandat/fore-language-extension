# EaxDocument.SheetNameError


Пространство имен: PP.Exp;


## Описание


Перечисление EaxDocument.SheetNameError
 содержит описание ошибки при неверном именовании листа документа экспресс-отчета.


Используется методом [EaxDataView.showFault](../Classes/Express/EaxDataView/EaxDataView.showFault.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Coincident
		 Повторяющееся название листа.


		 Empty
		 Пустое название листа.


		 None
		 Нет ошибки.
