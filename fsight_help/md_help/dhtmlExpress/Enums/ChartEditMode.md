# ChartEditMode


Пространство имен: PP.Exp.Ui;


## Описание


Перечисление ChartEditMode
 содержит режимы редактирования данных на диаграмме экспресс-отчета.


Используется свойством [EaxChartView.EditMode](../Classes/Express/EaxChartView/EaxChartView.EditMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 None
		 Редактирование данных не разрешено.


		 Point
		 Редактирование по точкам.


		 Draw
		 Редактирование данных с помощью рисования.
