# RibbonTableNFormatEnum


Пространство имен: PP.Exp.Ui;


## Описание


Перечисление RibbonTableNFormatEnum
 содержит форматы данных в таблице экспресс-отчета.


Используется методом [EaxMdService.setNumberFormat](../Classes/Express/EaxMdService/EaxMdService.setNumberFormat.htm).
