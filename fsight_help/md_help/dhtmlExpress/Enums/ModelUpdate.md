# ModelUpdate


Пространство имен: PP.Exp.Ui;


## Описание


Перечисление ModelUpdate
 содержит способы обновления модели данных.


Используется свойством [PropertyChangedEventArgs.ModelUpdateData](dhtmlMetabase.chm::/Classes/Metabase/PropertyChangedEventArgs/PropertyChangedEventArgs.ModelUpdateData.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 None
		 Обновление модели не требуется.


		 Local
		 Необходимо обновить модель локально.


		 Service
		 Необходимо обновить модель через сервис.
