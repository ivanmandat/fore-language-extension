# IndicatorType


Пространство имен: PP.Exp.Ui;


## Описание


Перечисление IndicatorType
 содержит режимы индикаторов роста.


Используется методами [EaxAnalyzer.getTypeGrowth](../Classes/Express/EaxAnalyzer/EaxAnalyzer.getTypeGrowth.htm)
 и [EaxMdService.setTableGrowth](../Classes/Express/EaxMdService/EaxMdService.setTableGrowth.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Rows
		 Индикатор роста для строк.


		 Columns
		 Индикатор роста для столбцов.
