# EaxSpecialDimType


Пространство имен: PP.Exp;


## Описание


Перечисление EaxSpecialDimType
 содержит специальные типы измерений экспресс-отчёта.


Используется методом:


	- [EaxAnalyzer.getSpecialValidDimKeys](../Classes/Express/EaxAnalyzer/EaxAnalyzer.getSpecialValidDimKeys.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Indicator
		 Измерение в качестве метрики.


		 Marker
		 Измерение в качестве объекта геолокации.


		 None
		 Отсутствует тип.


		 Objective
		 Измерение в качестве объекта наблюдения.


		 SecondaryObjective
		 Измерение в качестве направления стрелок.


		 Timeline
		 Измерение в качестве шкалы времени.
