# ViewTypeUpdate


Пространство имен: PP.Exp.Ui;


## Описание


Перечисление ViewTypeUpdate
 содержит элементы интерфейса, которые необходимо обновить при изменении
 какого-либо свойства в экспресс-отчете.


Используется следующими свойствами, методами и событиями:


	- [EaxBubbleChartMaster.PropertyChanged](../Classes/Express/EaxBubbleChartMaster/EaxBubbleChartMaster.PropertyChanged.htm);


	- [EaxBubbleChartView.PropertyChanged](../Classes/Express/EaxBubbleChartView/EaxBubbleChartView.PropertyChanged.htm);


	- [EaxChartView.DataChanged](../Classes/Express/EaxChartView/EaxChartView.DataChanged.htm);


	- [EaxChartView.PropertyChanged](../Classes/Express/EaxChartView/EaxChartView.PropertyChanged.htm);


	- [EaxGridView.refresh](../Classes/Express/EaxGridView/EaxGridView.refresh.htm);


	- [EaxMapView.PropertyChanged](../Classes/Express/EaxMapView/EaxMapView.PropertyChanged.htm);


	- [EaxTitleView.PropertyChanged](../Classes/Express/TitleBox/TitleBox.PropertyChanged.htm);


	- [EaxToolBarView.ObjectStateChanged](../Classes/Express/EaxToolBarView/EaxToolBarView.ObjectStateChanged.htm);


	- [ParametersDialog.getAllViewsPropChangedObjects](../Classes/Express/ParametersDialog/ParametersDialog.getAllViewsPropChangedObjects.htm);


	- [PropertyChangedEventArgs.TypeUpdateData](dhtmlMetabase.chm::/Classes/Metabase/PropertyChangedEventArgs/PropertyChangedEventArgs.TypeUpdateData.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 DataView
		 Необходимо обновить представление данных экспресс-отчета.


		 None
		 Ничего обновлять не надо.


		 Table
		 Необходимо обновить только таблицу экспресс-отчета.


		 Chart
		 Необходимо обновить только диаграмму экспресс-отчета.


		 BubbleChart
		 Необходимо обновить только пузырьковую диаграмму экспресс-отчета.


		 Map
		 Необходимо обновить только карту экспресс-отчета.


		 PropertyBar
		 Необходимо обновить только боковую панель экспресс-отчета.


		 Ribbon
		 Необходимо обновить только ленту инструментов экспресс-отчета.


		 All
		 Необходимо обновить всё.


		 DimBar
		 Необходимо обновить только панель измерений экспресс-отчета.


		 DimBarSelection
		 Необходимо обновить только выделенные элементы на панели измерений
		 экспресс-отчета.


		 DimBarHighlighting
		 Необходимо обновить только подсветку элементов на панели измерений
		 экспресс-отчета.


		 Title
		 Необходимо обновить только заголовок экспресс-отчета.


		 GridStatistics
		 Необходимо обновить панель статистических характеристик в
		 таблице экспресс-отчета.
