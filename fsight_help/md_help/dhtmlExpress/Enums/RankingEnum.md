# RankingEnum


Пространство имен: PP.Exp.Ui;


## Описание


Перечисление RankingEnum
 содержит способы ранжирования данных в экспресс-отчете.


Используется следующими методами:


	- [EaxMdService.setRanking](../Classes/Express/EaxMdService/EaxMdService.setRanking.htm);


	- [EaxCategoryView.getDataTypeState](../Classes/Express/EaxCategoryView/EaxCategoryView.getDataTypeState.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Absolute
		 По абсолютным значениям.


		 Percentage
		 По процентам.


		 Growth
		 По росту значения.


		 GrowthRate
		 По индикатору роста значения.


		 Change
		 По изменению значения.


		 Rank
		 По рангам.


		 ReverseRank
		 По значениям, обратным рангам.


		 Left
		 По левому краю.


		 Top
		 По верхнему краю.


		 Table
		 По таблице.


		 Row
		 По строкам.


		 Column
		 По столбцам.


		 OtherRanking
		 Другой способ ранжирования.
