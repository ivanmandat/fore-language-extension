# PP.Exp.UnitOption


Пространство имен: PP.Exp;


## Описание


Перечисление UnitOption содержит
 варианты пересчета единиц измерения.


Используется методом [EaxMdService.setAutoUpdate](../Classes/Express/EaxMdService/EaxMdService.setAutoUpdate.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Normalized
		 Нормализовать единицы измерения (привести к одним единицам
		 измерения).


		 Natural
		 Использовать заданные единицы измерения.


		 Selected
		 Использовать первую единицу измерения, которая отмечена в справочнике
		 единиц измерения.


		 Default
		 Использовать единицы измерения по умолчанию.
