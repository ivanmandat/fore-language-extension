# TitleType


Пространство имен: PP.Exp.Ui;


## Описание


Перечисление TitleType
 содержит типы заголовка экспресс-отчета.


Используется методом [EaxMdService.setGridCMenuCornerCell](../Classes/Express/EaxMdService/EaxMdService.setGridCMenuCornerCell.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Simple
		 Заголовок в виде простого текста.


		 Control
		 Заголовок в виде элемента управления.
