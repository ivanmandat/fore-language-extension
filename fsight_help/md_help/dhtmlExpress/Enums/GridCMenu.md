# GridCMenu


Пространство имен: PP.Exp.Ui;


## Описание


Перечисление GridCMenu
 содержит виды контекстного меню у ячеек заголовка-строки/столбца таблицы
 экспресс-отчета.


## Допустимые значения


		 Значение
		 Краткое описание


		 CornerTitle
		 Контекстное меню угловой ячейки таблицы.


		 Cut
		 Вырезать элемент.


		 Copy
		 Копировать элемент.


		 Paste
		 Вставить элемент.


		 KeepElems
		 Оставить только один элемент.


		 ExcludeElems
		 Исключить элементы.


		 DetailElement
		 Детализировать элемент.


		 GenElement
		 Обобщить элемент.


		 FixElement
		 Зафиксировать элемент.


		 FixElementWRep
		 Зафиксировать элемент с заменой.


		 CalcBasedOnElement
		 Добавить по отмеченным с операцией: сумма, среднее, минимум,
		 максимум, мода, медиана, дисперсия.


		 CalcBasedOnElementTotal
		 Добавить итог.


		 CalcBasedOnElementTempTotal
		 Добавить промежуточный итог.


		 DimViewMode
		 Настроить размещение наименований.


		 DimViewAttribute
		 Настроить наименование по атрибутам справочника.


		 DimViewCustom
		 Задать пользовательское наименование.


		 Settings
		 Настройки.


		 Statistics
		 Статистические характеристики.


		 Selection
		 Выбрать элемент.


		 RelationalGroup
		 Группировать [реляционную
		 область данных](UiReport.chm::/desktop/Relational_data_area/UIReport_Relational_data_area.htm).


		 RelationalUngroup
		 Разгруппировать [реляционную
		 область данных](UiReport.chm::/desktop/Relational_data_area/UIReport_Relational_data_area.htm).
