# RibbonTableStyleEnum


Пространство имен: PP.Exp.Ui;


## Описание


Перечисление RibbonTableStyleEnum
 содержит типы стилей таблицы экспресс-отчета.


Используется методом [TableCatView.getTableStyleState](../Classes/Express/TableCatView/TableCatView.getTableStyleState.htm).

		 Name. Именованный стиль.

		 Clear. Пустой стиль.
