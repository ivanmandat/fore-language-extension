# EaxAggregationFunction


Пространство имен: PP.Exp.Ui;


## Описание


Перечисление EaxAggregationFunction
 содержит функции агрегации для индикаторов.


Используется методом [EaxMdService.setIndAgg](../Classes/Express/EaxMdService/EaxMdService.setIndAgg.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 None
		 Агрегация не используется.


		 Average
		 Среднее арифметическое значение.


		 Max
		 Максимальное значение.


		 Mean
		 Среднее математическое ожидание.


		 Median
		 Определяется медиана для агрегируемых значений.


		 Min
		 Минимальное значение.


		 Mode
		 Мода значений.


		 Sum
		 Сумма значений.
