# PvtDimPos


Пространство имен: PP.Exp;


## Описание


Перечисление PvtDimPos
 содержит положения измерения в кубе, на котором построен экспресс-отчёт.


Используется следующими свойствами и методами:


	- [EaxAnalyzer.getDimPosition](../Classes/Express/EaxAnalyzer/EaxAnalyzer.getDimPosition.htm);


	- [EaxAnalyzer.getDimsByPosition](../Classes/Express/EaxAnalyzer/EaxAnalyzer.getDimsByPosition.htm);


	- [EaxDimBarMasterPanel.PvtPosition](../Classes/Express/EaxDimBarMasterPanel/EaxDimBarMasterPanel.PvtPosition.htm);


	- [EaxDimBarMaster.getCurrentState](../Classes/Express/EaxDimBarMaster/EaxDimBarMaster.getCurrentState.htm);


	- [EaxDimBarMasterView.moveDim](../Classes/Express/EaxDimBarMaster/EaxDimBarMaster.moveDim.htm);


	- [EaxDimBarView.getCurrentState](../Classes/Express/EaxDimBarView/EaxDimBarView.getCurrentState.htm);


	- [EaxDimBarView.moveDim](../Classes/Express/EaxDimBarView/EaxDimBarView.moveDim.htm).
