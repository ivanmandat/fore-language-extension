# EaxPropertyViews


Пространство имен: PP.Exp.Ui;


## Описание


Перечисление EaxPropertyViews
 содержит свойства представлений экспресс-отчета.


Используется следующими свойствами и методами:


	- [DataCatView.refresh](../Classes/Express/DataCatView/DataCatView.refresh.htm);


	- [EaxRibbonView.setPropertyValue](../Classes/Express/EaxRibbonView/EaxRibbonView.setPropertyValue.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 DimViewMode
		 Вариант отображения панели измерений в экспресс-отчете.


		 VisiblePropertyBar
		 Видимость панели свойств экспресс-отчета.


		 GridEditMode
		 Режим редактирования таблицы экспресс-отчета.


		 GridChangeData
		 Измененные данные таблицы экспресс-отчета.


		 GridSelection
		 Выделенные элементы таблицы экспресс-отчета.


		 Ranking
		 Отображение данных таблицы экспресс-отчета как ранги.
