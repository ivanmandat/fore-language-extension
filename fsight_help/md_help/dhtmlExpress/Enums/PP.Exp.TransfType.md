# PP.Exp.TransfType


Пространство имен: PP.Exp;


## Описание


Перечисление TransfType содержит
 типы коллекции трансформаций элемента измерения.


Используется методом [EaxMdService.setTransformationState](../Classes/Express/EaxMdService/EaxMdService.setTransformationState.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 CalcTransformations
		 Коллекция трансформаций предназначенная для вычислений.


		 SortTransformations
		 Коллекция трансформаций предназначенная для сортировки.


		 FiterTransformations
		 Коллекция трансформаций предназначенная для фильтрации.
