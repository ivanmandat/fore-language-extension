# DimViewMode


Пространство имен: PP.Exp.Ui;


## Описание


Перечисление DimViewMode
 содержит вариант отображения панели измерений в экспресс-отчете.


Используется свойством [EaxPropertyBarView.DimViewMode](../Classes/Express/EaxPropertyBar/EaxPropertyBar.DimViewMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 New. Каждое измерение
		 находится на отдельной панели типа PP.Ui.[NavigationItem](dhtmlCommon.chm::/Classes/NavigationItem/NavigationItem.htm).


		 1
		 Classic. Измерения
		 находятся в элементах управления типа PP.Ui.[ComboBox](dhtmlUi.chm::/Classes/ComboBox/ComboBox.htm)
		 на одной панели.
