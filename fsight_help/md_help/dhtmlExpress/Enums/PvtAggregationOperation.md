# PvtAggregationOperation


Пространство имен: PP.Exp.Ui;


## Описание


Перечисление PvtAggregationOperation
 содержит функции агрегации данных в фиксированных измерениях экспресс-отчёта.


Используется следующими методами:


	- [EaxMdService.setAgrMethod](../Classes/Express/EaxMdService/EaxMdService.setAgrMethod.htm);


	- [EaxMdService.setMainControlElements](../Classes/Express/EaxMdService/EaxMdService.setMainControlElements.htm).

		 Значение

		 Агрегация отключена.
