# EaxDataSourceOpenMode


Пространство имен: PP.Exp;


## Описание


Перечисление EaxDataSourceOpenMode
 содержит режим извлечения куба.


Используется следующими свойствами и методами:


	- [DataSourcesPanel.OpenMode](../Classes/Express/DataSourcesPanel/DataSourcesPanel.OpenMode.htm);


	- [EaxMdService.setDataSources](../Classes/Express/EaxMdService/EaxMdService.setDataSources.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 None
		 Режим извлечения куба с данными без сохраненного представления.


		 DefaultView
		 Режим извлечения куба с сохраненным представлением данных.


		 DataAndSelection
		 Режим извлечения данных куба и сохраненной отметки.
