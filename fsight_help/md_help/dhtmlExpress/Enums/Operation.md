# Operation


Пространство имен: PP.Exp;


## Описание


Перечисление Operation
 содержит операции, которая могут выполняться в экспресс-отчете.


Используется следующими свойствами и методами:


	- [EaxAnalyzer.clearWaitToken](../Classes/Express/EaxAnalyzer/EaxAnalyzer.clearWaitToken.htm);


	- [EaxAnalyzer.WaitToken](../Classes/Express/EaxAnalyzer/EaxAnalyzer.WaitToken.htm).
