# RibbonAnalyticsEnum


Пространство имен: PP.Exp.Ui;


## Описание


Перечисление RibbonAnalyticsEnum
 содержит методы расширенной аналитики экспресс-отчета.


Используется для определения типа нажатой кнопки на ленте инструментов
 экспресс отчета.
