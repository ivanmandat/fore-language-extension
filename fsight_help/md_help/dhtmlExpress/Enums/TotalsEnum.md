# TotalsEnum


Пространство имен: PP.Exp.Ui;


## Описание


Перечисление TotalsEnum
 содержит список итогов для данных таблицы экспресс-отчета.


Используется методом [EaxMdService.setTotals](../Classes/Express/EaxMdService/EaxMdService.setTotals.htm).
