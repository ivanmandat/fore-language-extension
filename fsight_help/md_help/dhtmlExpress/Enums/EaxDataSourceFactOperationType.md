# EaxDataSourceFactOperationType


Пространство имен: PP.Exp;


## Описание


Перечисление EaxDataSourceFactOperationType
 содержит типы операций с источником данных.


Используется методом [EaxMdService.setDataSourceFactor](../Classes/Express/EaxMdService/EaxMdService.setDataSourceFactor.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Exclude
		 Исключить.


		 Include
		 Добавить.


		 None
		 Нет действия.


		 Set
		 Установить.
