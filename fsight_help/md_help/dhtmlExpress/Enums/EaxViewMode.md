# EaxViewMode


Пространство имен: PP.Exp.Ui;


## Описание


Перечисление EaxViewMode
 содержит режимы размещения представлений данных в рабочей области экспресс-отчёта.


Используется методом [EaxModeSelectorView.GetCurrentViewMode](../Classes/Express/EaxModeSelectorView/EaxModeSelectorView.GetCurrentViewMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Maximized. Размещение
		 представлений данных по всей рабочей области.


		 1
		 Horizontal. Горизонтальное
		 размещение представлений данных.


		 2
		 Vertical. Вертикальное
		 размещение представлений данных.
