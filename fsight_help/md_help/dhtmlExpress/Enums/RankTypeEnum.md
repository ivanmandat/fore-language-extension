# RankTypeEnum


Пространство имен: PP.Exp.Ui;


## Описание


Перечисление RankTypeEnum
 содержит типы ранжирования данных в экспресс-отчете.


Используется методом [EaxMdService.setRanking](../Classes/Express/EaxMdService/EaxMdService.setRanking.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 None
		 Ранжирование отсутствует.


		 Asc
		 Ранжирование по возрастанию значений.


		 Desc
		 Ранжирование по убыванию значений.
