# TabTablePredefinedStyle


Пространство имен: PP.Exp.Ui;


## Описание


Перечисление TabTablePredefinedStyle
 содержит предопределённые стили оформления таблицы экспресс-отчёта.


Используется методом [EaxMdService.setMainTableStyle](../Classes/Express/EaxMdService/EaxMdService.setMainTableStyle.htm).
