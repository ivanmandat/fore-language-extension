# TransfCalculationMode


Пространство имен: PP.Exp;


## Описание


Перечисление TransfCalculationMode
 содержит момент выполнения трансформации в цепочке расчёта.


## Комментарии


Значение передается в метод [EaxMdService.setTransformationPos](../Classes/Express/EaxMdService/EaxMdService.setTransformationPos.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 AfterDataSourceExecute
		 После выполнения расчёта агрегации.


		 AfterCalcTotals
		 После выполнения расчёта итогов.
