# HeaderContainerVisibility


Пространство имен: PP.Exp.Ui;


## Описание


Перечисление HeaderContainerVisibility
 содержит режимы отображения контейнера, содержащего заголовок экспресс-отчёта
 и группу кнопок для выбора варианта представления данных.


Используется методом [EaxDataView.HeaderContainerVisibility](../Classes/Express/EaxDataView/EaxDataView.HeaderContainerVisibility.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 ToolBarOnly
		 Отображение только группы кнопок для выбора варианта представления
		 данных.


		 TitleOnly
		 Отображение только заголовка экспресс-отчёта.


		 HideAll
		 Скрытие всех элементов контейнера.


		 ShowAll
		 Отображение всех элементов контейнера.
