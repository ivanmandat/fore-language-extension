# TitleProperties


Пространство имен: PP.Exp.Ui;


## Описание


Перечисление TitleProperties
 содержит свойства заголовка экспресс-отчета.


Используется следующими свойствами и методами:


	- [EaxPropertyBarView.setPropertyValue](../Classes/Express/EaxPropertyBar/EaxPropertyBar.setPropertyValue.htm);


	- [TitleBox.getPropertyValue](../Classes/Express/TitleBox/TitleBox.getPropertyValue.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 None
		 Свойства отсутствуют.


		 EditMode
		 Режим редактирования.


		 All
		 Все свойства.


		 Font
		 Шрифт.


		 FontColor
		 Цвет шрифта.


		 FontSize
		 Размер шрифта.


		 FontFamily
		 Семейство шрифтов.


		 IsBold
		 Полужирное начертание.


		 IsItalic
		 Курсив.


		 IsUnderline
		 Подчёркивание.


		 HorizontalAlignment
		 Выравнивание по горизонтали.


		 VerticalAlignment
		 Выравнивание по вертикали.


		 BackgroundColor
		 Цвет заливки.


		 Text
		 Текст.


		 TextWrapping
		 Обтекание текстом.


		 Height
		 Высота.


		 TitleType
		 Тип заголовка.


		 UnformattedText
		 Неформатированный текст.


		 Selection
		 Выделение.


		 Visible
		 Видимость.


		 Master
		 Мастер.


		 Blur
		 Размытие.
