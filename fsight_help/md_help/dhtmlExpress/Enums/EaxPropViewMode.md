# EaxPropViewMode


Пространство имен: PP.Exp.Ui;


## Описание


Перечисление EaxPropViewMode
 содержит режим отображения мастеров на панели свойств экспресс-отчета.


Используется методом [EaxPropertyBarView.getMode](../Classes/Express/EaxPropertyBar/EaxPropertyBar.getMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 None
		 Отсутствие отображения мастеров.


		 Widget
		 Режим отображения в виде визуализатора.


		 Selection
		 Режим отображения в виде списков с возможностью выбора элементов.
