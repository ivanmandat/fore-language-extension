# EaxModeSelectorType


Пространство имен: PP.Exp.Ui;


## Описание


Перечисление EaxModeSelectorType
 содержит возможные варианты элементов меню у кнопки для выбора режима
 отображения представлений данных.


Используется следующими свойствами и методами:


	- [EaxDataView.getModeSelectorView](../Classes/Express/EaxDataView/EaxDataView.getModeSelectorView.htm);


	- [EaxModeSelectorView.ModeSelectorState](../Classes/Express/EaxModeSelectorView/EaxModeSelectorView.ModeSelectorState.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Maximized
		 Пункт меню «На закладках».


		 Vertical
		 Пункт меню «Вертикально».


		 Horizontal
		 Пункт меню «Горизонтально».
