# ChartBox


## Описание


Компонент ChartBox реализует
 контейнер для диаграммы экспресс-отчета.


## Комментарии


Компонент ChartBox реализован
 классом [ChartBox](../../../Classes/Express/ChartBox/ChartBox.htm).


## Пример


Компонент ChartBox может иметь
 следующий вид:


![](ChartBox.png)

 и ChartBox](ChartBox_Example.htm)
