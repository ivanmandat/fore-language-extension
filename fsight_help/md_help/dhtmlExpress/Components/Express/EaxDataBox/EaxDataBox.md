# EaxDataBox


## Описание


Компонент EaxDataBox
 реализует контейнер для области данных экспресс-отчета.


## Комментарии


Компонент EaxDataBox
 реализован классом [EaxDataBox](../../../Classes/Express/EaxDataBox/EaxDataBox.htm).

 может иметь следующий вид:
