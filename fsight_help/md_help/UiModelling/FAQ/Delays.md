# Как настроить задержку расчета модели?


В веб-приложении доступно задание задержки перед каждым шагом расчета
 модели. Задержку в расчете можно использовать для наглядности при демонстрации
 расчета.


Задержка расчета настраивается для каждой модели индивидуально.


Для задания задержки расчета:


	- Папка Delays может располагаться по одному из следующих путей:


				- [HKEY_LOCAL_MACHINE\SOFTWARE\Foresight\Foresight Analytics Platform\10.0\10.9\Model\Delays];


				- [HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Foresight\Foresight Analytics Platform\10.0\10.9\Model\Delays];


				- [HKEY_LOCAL_MACHINE\SOFTWARE\FForesight\Foresight Analytics Platform\10.0\10.9\Model\Delays].


	- Создайте параметр типа REG_DWORD. Наименование параметра
	 должно совпадать с идентификатором модели, для которой настраивается
	 задержка расчета.


	- Задайте значение созданного параметра: время задержки в миллисекундах.
	 Значение должно быть больше 200.


В результате перед каждым шагом расчета модели будет выполняться указанная
 задержка.
