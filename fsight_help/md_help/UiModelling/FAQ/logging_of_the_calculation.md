# Как просмотреть отладочную информацию при расчётах?


При использовании многомерного цикла для расчёта задач моделирования
 выводится отладочная информация, отражающая процесс расчёта. Просмотреть
 подобную информацию можно с помощью специализированных программ, например,
 DebugView, которая входит в пакет системных программ [Windows
 Sysinternals](https://docs.microsoft.com/ru-ru/sysinternals/). Если ошибки отсутствуют, то данная информация содержит
 время расчёта каждой отдельной итерации.


При возникновении ошибок выводится следующая информация:


	- Информация о метамодели - наименование, идентификатор, ключ;


	- Информация об отметке, по которой производится расчёт в текущей
	 итерации;


	- Информация об источнике, который используется для фильтрации
	 данных, если фильтрация настроена;


Для вывода дополнительной информации в ветке реестра [HKEY_CURRENT_USER\Software\Foresight\Foresight Analytics Platform\10.0\Debug]
 может быть создан параметр MsDump типа REG_DWORD.
 В зависимости от значений данного параметра выводится следующая информация:


	- MsDump = 1. Полная информация
	 об отметке, по которой производится расчёт. Отметка формируется с
	 учётом фильтрации и значений параметров. Для отмеченных элементов
	 измерений выводится матрица со значениями атрибутов.


При включённой [фильтрации
 по данным](calculationalgorithm.chm::/Desktop/calculation_block/filteringbydata.htm) к полной информации об отметке будет добавлена
 информация о режиме расчёта в зависимости от того, есть ли в источниках
 фильтрации по данным куб-приёмник [блока
 расчёта](calculationalgorithm.chm::/desktop/calculation_block/Calculation_unit.htm). Если нет, то будет применён многопоточный
 расчёт;


	- MsDump = 2. К полной
	 информации об отметке будет добавлена матрица с изменёнными значениями,
	 полученными в результате расчёта;


	- MsDump = 3. К полной
	 информации об отметке и матрице с изменёнными значениями добавляется
	 информация об агрегации данных: отметка, по которой производится агрегация,
	 время расчёта агрегации, результат агрегации.
