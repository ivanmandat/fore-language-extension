# Запуск моделирования и прогнозирования


Запустите настольное приложение «Форсайт. Аналитическая платформа»
 и откройте навигатор объектов:


	- дважды щелкните по объекту «Контейнер
	 моделирования» в дереве папок;


	- выполните команду «Открыть»
	 в контекстном меню контейнера моделирования;


	- выполните команду «Редактировать»
	 в контекстном любого дочернего объекта в контейнере моделирования;


	- выполните
	 команду «Открыть с помощью > Контейнера
	 моделирования» в контекстном меню базы данных временных рядов.
	 Будет создан и открыт контейнер моделирования с выбранной базой данных
	 временных рядов в качестве источника.


Будет открыто [основное
 окно](../2_Container_of_Modeling/2_2_Window_container/UiModelling_Window_container.htm) моделирования и прогнозирования.


## Режимы работы моделирования и прогнозирования


Инструмент «Моделирование и прогнозирование»
 в настольном приложении может работать в следующих режимах:


	- режим на атрибутах.
	 Используется по умолчанию. Модели создаются и рассчитывается на основе
	 данных, полученных по совокупности атрибутов источника данных. В качестве
	 источников данных могут использоваться базы данных временных рядов
	 или кубы;


	- режим на переменных.
	 Модели создаются на основе данных из переменных моделирования. Данный
	 режим используется для поддержки моделей, созданных в предыдущих версиях
	 платформы, где не было режима на атрибутах.


Для выбора режима работы моделирования и прогнозирования используйте
 параметр в реестре: HKEY_CURRENT_USER\Software\Foresight\Foresight Analytics Platform\10.0\10.9\Model\UseMsVariablesUI.
 Параметр может принимать следующие значения:


	- 0. Работа в режиме на
	 атрибутах;


	- 1. Работа в режиме на
	 переменных.


Если параметр типа REG_DWORD отсутствует, то создайте его.
