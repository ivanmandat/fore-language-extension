# Описание интерфейса моделирования
 и прогнозирования


Для открытия контейнера моделирования в настольном приложении:


	- дважды щелкните по контейнеру моделирования;


	- выполните команду «Открыть»
	 в контекстном меню контейнера.


Будет открыто окно контейнера моделирования, например:


![](../../1_modelling/modelling.png)


Окно контейнера моделирования содержит:


	- Главное меню. Содержит
	 основные команды для работы с моделированием и прогнозированием;


	- Панели инструментов.
	 Содержит кнопки, дублирующие основные команды главного меню контейнера
	 моделирования.

	Для отображения или скрытия тех или иных панелей инструментов используйте
	 соответствующие флажки в раскрывающемся меню «Панели
	 инструментов» в главном меню «Вид».

	Для настройки панелей инструментов выполните команду «Панели
	 инструментов > Настройка» в главном меню «Вид».
	 Будет открыт диалог «[Настройка](GetStarted.chm::/Interface/Interface_Description.htm#toolbars)»;


	- [Панель
	 объектов контейнера](uimodelling_window_object.htm). Отображает все объекты контейнера моделирования
	 в виде дерева;


	- [Рабочая
	 область](uimodelling_workspace.htm). Является основной функциональной областью контейнера
	 моделирования и отображает панели управления объектами контейнера;


	- [Панель
	 сценариев моделирования](Scenario_Of_Modeling.htm). Содержит список сценариев, по
	 которым может осуществляться расчет задач.


Совет. Для отображения
 на панелях объектов контейнера и сценариев моделирования идентификаторов
 объектов вместо их наименований установите переключатель «Отображать
 идентификаторы объектов» в главном меню «Вид»,
 для отображения наименований объектов - переключатель «Отображать
 наименования объектов». Также для каждой панели доступны индивидуальные
 настройки отображения.
