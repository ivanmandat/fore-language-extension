# Способ наполнения данными


На панели задаётся способ передачи данных из источника в переменную.


В раскрывающемся списке «Заполнить
 переменную данными» содержится список возможных источников данных.
 Вид панели «Способ наполнения данными»
 изменяется, в зависимости от выбранного метода загрузки данных:


	- [вручную](uimodelling_work_object_value_4_3.htm). Используется
	 по умолчанию;


	- [из источника](uimodelling_work_object_value_4_1.htm);


	- [по формуле](UiModelling_Work_object_Value_4_2.htm).
