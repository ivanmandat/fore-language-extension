# Веса главных компонент


Веса главных компонент - это значения собственных векторов.


![](Weight%20Comp.gif)


Панель состоит из двух раскрывающихся списков и диаграммы. В раскрывающихся
 списках «По оси X» и «По
 оси Y» выбираются главные компоненты, которые будут по осям X и Y
 диаграммы соответственно. На точечной диаграмме отображаются точки по
 факторам. Каждая точка имеет координаты, соответствующие весам, с которыми
 каждый фактор входит в выбранные две главные компоненты.
