# Параметры


Панель предназначена для редактирования основных параметров [метода
 главных компонент](Lib.chm::/04_Other_models/UiModelling_PrincipalComponents.htm):


![](MethPrinComp%20param.gif)


	- Тип объекта анализа.
	 Определяет тип объекта оптимизации:


	-


		- корреляционная матрица исходных данных;


		- стандартизированная матрица исходных данных;


		- матрица сумм квадратов и векторных произведений;


		- ковариационная матрица исходных данных.


	- Использовать весовые коэффициенты.
	 Установка флажка позволяет задать весовые коэффициенты для наблюдений.
	 Нажмите кнопку «Задать». Будет
	 открыто окно для редактирования значений коэффициентов:


![](MethPrinComp%20param%20coeff.gif)


По умолчанию все коэффициенты равны единице.
 Для редактирования значения весового коэффициента наблюдения дважды щелкните
 по нему мышью или нажмите клавишу F2.
