# Коэффициенты стандартизации


Данная панель будет отображена, если на панели «[Параметры](UiModelling_WorkObject_MethPrinComp.htm)» выбран один из следующих типов анализа:


-
Корреляционная матрица исходных данных


-
Стандартизированная матрица исходных данных


В таблице содержатся значения коэффициентов стандартизации для переменных, участвующих в расчете.


![](standardization%20factors.gif)


Если анализируется «Стандартизированная матрица исходных данных», то доступно редактирование значений коэффициентов. Для этого дважды щелкните по значению мышью или нажмите клавишу F2. Для перерасчета метода главных компонент нажмите кнопку «Рассчитать».
