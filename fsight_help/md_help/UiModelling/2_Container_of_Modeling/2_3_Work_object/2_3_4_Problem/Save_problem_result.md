# Сохранение результата расчета задачи в базе данных временных рядов


Если задача моделирования содержится в контейнере моделирования, который
 является внутренним для базы данных ременных рядов, то результаты расчета
 задачи будут сохранены в базе данных временных рядов.


На рисунке ниже приведен пример модели, результаты расчета которой будут
 сохранены в базу данных временных рядов с наименованием «Statistics»:


![](../2_3_2_Model/Standart_Model/Std_Model_Panel_Param_Attr.gif)


Примечание.
 Результат расчета модели может быть сохранен в куб, если база данных временных
 рядов использует этот куб в качестве источника.
