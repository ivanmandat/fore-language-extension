# Тест Гранжера


[Тест
 Гранжера](Lib.chm::/05_Statistics/UiModelling_GrangerTest.htm) предназначен для проверки причинно-следственной
 зависимости временных рядов:


![](Test%20Grenjera.gif)


Параметры теста:


	- Уровень значимости.
	 Может принимать значения из интервала [0, 1];


	- Лаг. Может принимать
	 любые неотрицательные значения;


	- Использовать при расчете подключение
	 к R. Определяет, использовать ли при расчете теста Гранжера
	 статистический пакет R. По умолчанию тест рассчитывается методами
	 продукта «Форсайт. Аналитическая платформа».
	 Для расчета теста с помощью методов R установите данный флажок.


Примечание.
 Возможность доступна, если в «Форсайт. Аналитическая платформа»
 подключен пакет R. Подробнее см. раздел: «[Как
 настроить интеграцию с R?](../../../FAQ/R_Integration.htm)».


После расчета теста в таблицах будут приведены значения вероятностей
 и статистики Фишера.


Полученные значения вероятностей сравниваются с заданным уровнем значимости,
 малое значение вероятности свидетельствует о возможности наличия зависимости
 между переменными.


В зависимости от полученного вывода, ячейки со значениями будут закрашены
 соответствующими цветами:


	- Желтый. Отсутствие причинно-следственной
	 зависимости;


	- Зеленый. Наличие причинно-следственной
	 зависимости.
