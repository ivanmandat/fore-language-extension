# Тест Дики-Фуллера


Тест Дики-Фуллера является средством проверки стационарности временного
 ряда. Результаты расчеты данного теста для каждой исследуемой переменной
 отображаются на соответствующей панели:


![](Uimodelling%20Descriptive%20Test.gif)


	- Тип модели. Тест позволяет
	 работать с уравнениями трех видов, выбор уравнения производится из
	 списка:


	-


		- Без константы и без тренда (описывает процесс случайных
		 блужданий);


		- С константой;


		- С константой и с трендом.


	- Дифференцирование ряда.
	 Задается порядок дифференцирования уравнения. Возможны следующие варианты:


	-


		- Исходный
		 ряд. Дифференцирование не используется;


		- Дифференцированный
		 ряд;


		- Дважды
		 дифференцированный ряд.


	- Порядок авторегрессии.
	 Задается с помощью редактора чисел или вручную.


	- Использовать при расчете подключение
	 к R. Определяет, использовать ли при расчете теста Дики-Фуллера
	 статистический пакет R. По умолчанию тест рассчитывается методами
	 продукта «Форсайт. Аналитическая платформа».
	 Для расчета теста с помощью методов R установите данный флажок.


Примечание.
 Возможность доступна, если в «Форсайт. Аналитическая платформа»
 подключен пакет R. Подробнее см. раздел: «[Как
 настроить интеграцию с R?](../../../FAQ/R_Integration.htm)».


Тест выполняется последовательно для каждой отмеченной переменной.


В таблице приведены результаты теста. Рассчитанная ADF-статистика
 сравнивается с N процентным (N=5%,
 1%) уровнем значимости и на основе данного сравнения делается вывод о
 стационарности/нестационарности ряда. Данный вывод также приведен в таблице.
 Процентный уровень значимости задан и зависит от выбранной модели уравнения
 и от длины исходного ряда.
