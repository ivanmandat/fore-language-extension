# Набор тестов


На панели определяется вид и порядок проводимых расчетов:


![](Uimodelling%20Descriptive%20View.gif)


Расчет теста выполняется, если он отмечен флажком. При снятии флажка тест исключается из расчетов, в рабочей области не будет отображена соответствующая тесту панель.


Для изменения положения в списке выбранного теста используйте кнопки «Вверх» и «Вниз».
