# Набор переменных


На данной панели определяется набор переменных, для которых будут рассчитаны
 описательные статистики:


![](Uimodelling%20Descriptive%20collection%20of%20V.gif)


Описательные статистики будут рассчитаны только для переменных, которые
 отмечены флажками. При снятии флажка переменная будет исключена из расчётов.
 Для работы с отметкой предусмотрены команды контекстного меню:


	- Отметить. Устанавливает
	 флажок;


	- Снять отметку. Снимает
	 флажок;


	- Отметить всё. Устанавливает
	 флажки для всех имеющихся в списке переменных;


	- Снять отметку со всех.
	 Снимает флажки для всех имеющихся в списке переменных.


[![](../../../opened.gif)![](../../../closed.gif)Для добавления
 переменной](javascript:TextPopup(this))


		- перетащите переменную из панели «[Объекты
		 контейнера](../../2_2_Window_container/uimodelling_window_object.htm)»;


		- нажмите кнопку «Добавить»,
		 либо дважды щелкните соответствующую строку панели. Будет открыто
		 окно, в котором следует выбрать переменную.


	При добавлении переменной, имеющей дополнительные измерения, будет
	 открыт диалог «[Изменение
	 размерности](../UiModelling_ChangeDimension.htm)» для их фиксации.


[![](../../../opened.gif)![](../../../closed.gif)Для редактирования
 формулы переменной](javascript:TextPopup(this))


		- нажмите кнопку «Редактировать»;


		- выполните одноименную команду контекстного меню;


		- дважды щёлкните по переменной.


	Будет открыт [редактор выражения](UiNav.Chm::/GUI/ExpressionEditor.htm), в котором задайте формулу переменной.


[![](../../../opened.gif)![](../../../closed.gif)Для задания
 лага переменной](javascript:TextPopup(this))


	Лаг отвечает за сдвиг ряда вперед (положительные значения) или назад
	 (отрицательные значения) на заданное количество точек во временном
	 периоде. Для задания лага выполните команду контекстного меню «Лаг».
	 Будет отображено всплывающее меню:


		- Увеличить. Увеличивает
		 лаг на один шаг;


		- Уменьшить. Уменьшает
		 лаг на один шаг.


	Примечание.
	 Задание лага доступно только для переменных, источник данных которых
	 содержит календарное измерение.


[![](../../../opened.gif)![](../../../closed.gif)Для задания
 преобразования над переменной](javascript:TextPopup(this))


	Выполните команду контекстного меню «[Преобразование](../UiModelling_work_Changes.htm)».


	Примечание.
	 Задание начального преобразования доступно только для переменных,
	 источник данных которых содержит календарное измерение.


[![](../../../opened.gif)![](../../../closed.gif)Для удаления
 выбранных переменных](javascript:TextPopup(this))


		- нажмите кнопку «Удалить»;


		- выполните одноименную команду контекстного меню;


		- нажмите клавишу DELETE.


	Будет отображен диалог подтверждения выполняемого действия.
