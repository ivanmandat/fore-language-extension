# Корреляционная матрица


На панели отображена [корреляция](Lib.chm::/05_Statistics/UiModelling_CoeffCorr.htm) переменных:


![](Uimodelling%20Descriptive%20Correlation.gif)


Ячейки, с полученными значениями, закрашены соответствующими цветами:


-
Желтый. Умеренная взаимосвязь. Значение коэффициента корреляции находится в диапазоне по абсолютному значению от 0,5 до 0,75;


-
Зеленый. Сильная взаимосвязь. Значение коэффициента корреляции находится в диапазоне по абсолютному значению от 0,75 и выше.
