# Диагностические тесты


Для модели «[Линейная
 регрессия (оценка МНК)](../Specification/8_Linear_regression/uimodelling_model_specification_linaer_reg.htm)» доступно выполнение ряда тестов, которые
 проверяют модель на предмет нарушения тех или иных основополагающих гипотез.
 Диагностические тесты помогают проверить: насколько правильно специфицирована
 регрессионная модель.


Для выполнения диагностических тестов откройте на редактирование тестируемую
 модель и выполните команду главного меню «Анализ > Диагностические
 тесты». Будет открыта вкладка, на которой осуществляется выбор
 тестов и настройка их параметров:


![](DiagnosticTest.gif)


На панели «Набор тестов» содержится
 список тестов, которые могут быть выполнены для модели. Установка/снятие
 флажка определяет, будет ли тест выполнен. По умолчанию выполняются все
 тесты (все флажки установлены).


Порядок проводимых тестов и порядок их отображения определяется порядком
 элементов в списке. Изменить порядок можно с помощью команд «Вверх»
 и «Вниз».


Параметры и результаты каждого теста содержатся на отдельной панели,
 название которой совпадает с названием теста. Доступные тесты:


	- [Критерий
	 пропущенных переменных](UiModelling_DiagnTest_OmittedVariables.htm)


	- [Критерий избыточных
	 переменных](UiModelling_DiagnTest_RedundantVariables.htm)


	- [Критерий Годфри
	 автокорреляции остатков](UiModelling_DiagnTest_SerialCorellationLM.htm)


	- [Тест Уайта
	 на гетероскедастичность](UiModelling_DiagnTest_WhiteHeteroskedasticity.htm)


	- [Критерий
	 функциональной формы](UiModelling_DiagnTest_RamseyResset.htm)


	- [Тест на устойчивость Чоу](UiModelling_DiagnTest_Chow.htm)


Во всех панелях с параметрами тестов из тестируемой модели линейной
 регрессии берётся:


	- набор и отметка по объясняющим переменным;


	- наличие/отсутствие константы.


Результатами каждого теста являются набор статистик и решение о том,
 принимается или отвергается та или иная гипотеза.
