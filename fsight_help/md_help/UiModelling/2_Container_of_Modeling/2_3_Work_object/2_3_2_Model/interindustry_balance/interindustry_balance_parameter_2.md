# Параметры


На данной панели определяются основные параметры модели:


![](interindustry%20balance%20parameter%202.gif)


	- Укажите тип решаемой задачи межотраслевого баланса:


		- Прямая задача. Нахождение
		 конечного использования: X ->
		 Y. Для решения используется уравнение: Y = X - A · X. Где:


			- Y. Конечное
			 использование;


			- X. Валовой
			 выпуск;


		- Обратная задача.
		 Нахождение валового выпуска: Y
		 -> X. Для решения используется уравнение: X = (E - A)-1 · Y. Где:


			- Y. Конечное
			 использование;


			- X. Валовой
			 выпуск.


	- Задайте моделируемую переменную:


		- если установлен переключатель «Прямая
		 задача», то в данном разделе появится поле «Конечное
		 использование» (Y),
		 в котором необходимо указать переменную конечного использования;


		- если установлен переключатель «Обратная
		 задача», то в данном разделе появится поле «Валовой
		 выпуск» (X), в
		 котором необходимо указать переменную валового выпуска.


	- Укажите срез отраслей/товаров и
	 услуг:


		- если установлен переключатель «Прямая
		 задача», то необходимо указать срез отраслей/товаров и
		 услуг переменной, который будет расположен «По
		 строкам»;


		- если установлен переключатель «Обратная
		 задача», то необходимо указать срез отраслей/товаров и
		 услуг переменной, который будет расположен «По
		 столбцам».
