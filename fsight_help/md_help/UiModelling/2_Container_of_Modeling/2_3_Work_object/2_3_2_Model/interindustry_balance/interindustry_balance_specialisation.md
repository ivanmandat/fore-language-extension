# Спецификация


Вид панели зависит от выбранной задачи на панели «[Параметры](interindustry_balance_parameter_2.htm)».
 Вверху панели будет указана формула, используемая для решения выбранной
 задачи.


![](interindustry%20balance%20specialisation.gif)


Порядок настройки:


	- Укажите переменную валового/конечного
	 выпуска:


	-


		- для решения прямой задачи укажите валовую переменную (X);


		- для решения обратной задачи укажите конечную переменную
		 (Y).


	- Укажите срез отраслей/товаров
	 и услуг:


	-


		- для прямой задачи укажите срез отраслей/товаров и услуг
		 валовой переменной, который будет расположен «По
		 столбцам»;


		- для обратной задачи укажите срез отраслей/товаров и услуг
		 конечной переменной, который будет расположен «По
		 строкам».


Если у переменной X
 (прямая задача)/Y (обратная задача)
 есть срезы, отсутствующие у переменной Y
 (прямая задача)/X (обратная задача),
 то появится окно «[Изменение
 размерности](../../UiModelling_ChangeDimension.htm)», в котором произведите фиксацию измерений.


	- Для матрицы А необходимо указать
	 способ расчета. Для решения задачи матрицу можно рассчитать
	 или взять из переменной. Для выбора установите соответствующий переключатель:


	-


		- Брать матрицу из переменной.
		 Укажите переменную, из которой будут браться значения и переменную
		 для выгрузки матрицы;


		- Рассчитать матрицу.
		 Укажите переменную, на основе которой будет производиться расчет
		 и переменную для выгрузки матрицы.


При добавлении переменной, которая содержит
 отсутствующие размерности у объясняемой переменной, произведите фиксацию
 в окне «[Изменение
 размерности](../../UiModelling_ChangeDimension.htm)».


	- Если необходим расчет II и/или
	 III квадрантов установите флаги. Для расчета нужного квадранта
	 установите соответствующий флажок. После установки флажка появляется
	 группа параметров, например:


![](interindustry%20balance%20quadrant.gif)


Укажите переменную для выгрузки квадранта.
 Если необходимо, произведите фиксацию в окне «[Изменение
 размерности](../../UiModelling_ChangeDimension.htm)».


Для расчета квадранта можно использовать
 матрицу со структурой конечного дохода (II квадрант)/матрицу со структурой
 национального дохода (III квадрант), либо произвести расчет структуры
 квадранта. Для этого установите переключатель «Рассчитать
 структуру конечного использования» (II квадрант)/«Рассчитать
 структуру конечного национального дохода» (III квадрант), а затем
 укажите переменную, на основе которой будет произведен расчет и переменную,
 в которую будет выгружена полученная матрица.
