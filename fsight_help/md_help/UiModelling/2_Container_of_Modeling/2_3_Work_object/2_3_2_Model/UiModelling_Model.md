# Модель


Модель предназначена для расчета результирующих переменных путем преобразования
 исходных переменных статистическими и математическими методами.


Расчёты ведутся с учетом динамики, так как каждой модели соответствует
 своя динамика: годовая, полугодовая, квартальная и т.д. Также доступен
 расчёт на источниках данных, которые не содержат календарного измерения.
 В данном случае переменные будут рассматриваться в качестве констант.


Если результирующая переменная содержит в себе дополнительные измерения,
 например, территориальное измерение, то можно:


	- настроить независимый расчёт для каждого элемента дополнительных
	 измерений. Например, использовать разные методы, разную динамику;


	- использовать единый расчёт для всех элементов дополнительных
	 измерений.

## Операции с моделью


[![](../../../opened.gif)![](../../../closed.gif)Создание
 модели](javascript:TextPopup(this))


	Для создания модели:


		- выполните команду «Объект
		 > Создать > Модель» главного меню
		 в главном меню контейнера моделирования;


		- выполните команду «Создать > Модель»
		 в контекстном меню панели «[Объекты
		 контейнера](../../2_2_Window_container/uimodelling_window_object.htm)»;


		- нажмите кнопку ![](new_equation2.gif)
		 на панели «[Объекты контейнера](../../2_2_Window_container/uimodelling_window_object.htm)».
		 При нажатии на левую часть кнопки в контейнере моделирования будет
		 создана стандартная модель. Если нажать правую часть кнопки: ![](../../2_2_Window_container/triangle1.gif), то будет отображено меню для выбора
		 типа создаваемой модели.


	В [рабочей
	 области](../../2_2_Window_container/uimodelling_workspace.htm) будут открыты панели для задания параметров создаваемой
	 модели.


[![](../../../opened.gif)![](../../../closed.gif)Расчет
 модели](javascript:TextPopup(this))


	По умолчанию модель рассчитывается автоматически при любом её изменении.
	 Если в ходе расчета модели возникли [предупреждения](Standart_Model/UiModelling_Model_Warning.htm)
	 или [ошибки](Standart_Model/UiModelling_Model_Error.htm),
	 то они будут отображены на соответствующих панелях.


	Если расчет модели занимает много времени, то целесообразно отключить
	 автоматический расчет для более быстрой настройки модели. Для выключения
	 автоматического расчета снимите флажок «Вычислять
	 автоматически» в раскрывающемся меню кнопки ![](../2_3_4_Problem/panel_problem.gif) на
	 панели инструментов.


	Для расчета модели при выключенном автоматическом расчете нажмите
	 кнопку ![](../2_3_4_Problem/panel_problem.gif) на панели инструментов
	 или выполните команду «Рассчитать
	 модель» в главном меню «Модель».


	Примечание.
	 Режим расчета сохраняется в реестре для текущего пользователя и распространяется
	 на все модели во всех контейнерах моделирования во всех репозиториях,
	 с которыми работает пользователь.


[![](../../../opened.gif)![](../../../closed.gif)Экспорт
 модели](javascript:TextPopup(this))


	Для всех[ стандартных
	 моделей](Standart_Model/Standart_Model.htm) и [векторной
	 модели коррекции ошибок (ECM)](error_control/error_control.htm) доступен экспорт в файл или регламентный
	 отчет.


	Для выполнения экспорта используйте команду «Контейнер
	 моделирования > Экспортировать» в главном меню.


	Для получения подробных сведений о экспорте модели обратитесь к
	 разделу «[Экспорт модели](Model_export.htm)».
