# Матрица весов


Панель отображается, если используется метод «[Фильтр
 Бакстера-Кинга](../Specification/UiModelling_Specification_BaxterKingFilter.htm)».


![](Bpf_weights_matrix.gif)


Панель содержит матрицу весов, используемую фильтром Бакстера-Кинга.
 В столбцах расположены рассчитанные значения коэффициентов. Размерность
 матрицы: 1×(q+1), где q - значение параметра «Опережение/лаг»,
 заданного на панели «[Спецификация](../Specification/UiModelling_Specification_BaxterKingFilter.htm)»
 модели «Фильтр Бакстера-Кинга».
