# Ряды (числовые значения)


Панель «Ряды (числовые значения)»
 отображает таблицу, содержащую числовые значения, по которым строятся
 графики на панели «[Графики
 и диаграммы](UiModelling_Model_Fact.htm)»:


![](../numeric_importances.gif)


В заголовках таблицы расположены кнопки:


	- ![](../numeric_importances%20button%202.gif). Нажатие кнопки включает
	 соответствующий ряд в состав диаграммы;


	- ![](../numeric_importances%20button.gif). Нажатие кнопки исключает
	 соответствующий ряд из состава диаграммы.


Для значений рядов, представленных на данной панели, можно настроить
 количество отображаемых десятичных знаков. Выполните команду «Свойства»
 в контекстном меню заголовка панели. Будет открыт диалог «Параметры
 коэффициентов модели», в котором задайте количество отображаемых
 десятичных знаков.


## Особенности отображения данных


	- Динамические доверительные границы прогнозного ряда будут рассчитаны
	 только для моделей «[Линейная регрессия
	 (оценка МНК)](../Specification/8_Linear_regression/uimodelling_model_specification_linaer_reg.htm)» и «[ARIMA](../Specification/1_Arima/uimodelling_model_specification_arima.htm)», если
	 задан порядок авторегрессии.


	- Если источник данных моделируемой переменной не содержит календарное
	 измерение, то:


		- будет отсутствовать столбец «Ряд»;


		- таблица будет содержать только одну строку значений;


		- столбец «Остатки»
		 будет иметь название «Ряд остатков».
