# Настройка перечня характеристик


Окно предназначено для редактирования перечня отображаемых статистических
 характеристик. Для открытия окна выполните команду «Свойства» в контекстном меню
 заголовка панели «[Статистические
 характеристики](statistical_features.htm)».


![](../Property%20StatCharacter.gif)


	- Поля для отображения.
	 Список предназначен для редактирования состава отображаемых полей.
	 Для отображения поля установите напротив него флажок.


	- Статистические характеристики
	 для отображения. Для вывода результатов расчёта характеристики
	 установите напротив неё флажок.


	- Формат числа. В редакторе
	 «Число десятичных знаков»
	 укажите количество десятичных знаков, отображаемых для значений статистических
	 характеристик. Установка флажка «Разделитель
	 групп разрядов» позволяет использовать разделитель групп разрядов
	 - символ, разделяющий группы цифр в целой части числа.


[![](../../../../opened.gif)![](../../../../closed.gif)Для работы с
 отметкой элементов](javascript:TextPopup(this))


	Используйте команды контекстное меню:


		- Отметить. Устанавливается
		 флажок для выбранного элемента;


		- Снять отметку. Снимается
		 флажок для выбранного элемента;


		- Отметить все. Устанавливаются
		 все флажки;


		- Снять отметку со всех.
		 Снимаются все флажки.


[![](../../../../opened.gif)![](../../../../closed.gif)Для изменения
 порядка элементов](javascript:TextPopup(this))


		- используйте кнопки «Вверх»,
		 «Вниз» или одноименные
		 команды контекстного меню;


		- перетащите элемент в требуемое место.
