# Обобщенные результаты модели


Панель отображается, если выбран метод «[Модель
 коррекции ошибок](../Specification/UiModelling_Specification_Error_control.htm)» или «[Векторная
 модель коррекции ошибок (ECM)](../error_control/error_control.htm)».


![](../error_control/error%20control%20general.gif)


На данной панели отображаются четыре регрессионные статистики, описывающие
 модель в целом:


	- Определитель ковариационной
	 матрицы остатков (RC). Характеризует тесноту линейной связи
	 между остатками модели;


	- [Логарифм функции максимального правдоподобия
	 (LogL)](Lib.chm::/UiModelling_LogL.htm). Вычисляется в предположении, что
	 остатки имеют многомерное нормальное распределение. Обычно используется
	 в качестве критерия при выборе из нескольких моделей.


	- [Информационный критерий Акаике (AIC)](Lib.chm::/05_Statistics/Information_Criteria.htm#aic).
	 Используется для выбора наилучшей модели в зависимости от заданных
	 лагов. Предпочтительней модель с меньшим значением критерия.


	- [Информационный критерий Шварца (SC)](Lib.chm::/05_Statistics/Information_Criteria.htm#sc).
	 Аналогичен критерию Акаике. Чем меньше значение критерия, тем лучше
	 модель.
