# Корреляция факторов


Панель отображается, если выбран один из методов:


	- [Детерминированное
	 уравнение](../Specification/4_Deterministic_equation/uimodelling_model_specification_deter.htm);


	- [Линейная
	 регрессия (оценка МНК)](../Specification/8_Linear_regression/uimodelling_model_specification_linaer_reg.htm);


	- [Линейная
	 регрессия (оценка методом инструментальных переменных)](../Specification/8_Linear_regression/uimodelling_model_specification_linaer_reg2.htm);


Панель «Корреляция факторов»
 содержит таблицу со значениями коэффициентов [корреляции](Lib.chm::/05_Statistics/UiModelling_CoeffCorr.htm)
 и точечный график зависимостей переменных:


![](Standart%20Model%20correlation.gif)


Ячейки закрашены соответствующими цветами:


	- Красный. Умеренная взаимосвязь.
	 Значение коэффициента корреляции находится в диапазоне по абсолютному
	 значению от 0,5 до 0,75;


	- Зеленый. Сильная взаимосвязь.
	 Значение коэффициента корреляции находится в диапазоне по абсолютному
	 значению от 0,75 и выше.


Примечание.
 Настройка количества отображаемых десятичных знаков не влияет на главную
 диагональ. Значения коэффициентов корреляции на ней всегда равны единице.


График зависимостей строится для переменных, коэффициент корреляции
 которых отмечен в таблице.
