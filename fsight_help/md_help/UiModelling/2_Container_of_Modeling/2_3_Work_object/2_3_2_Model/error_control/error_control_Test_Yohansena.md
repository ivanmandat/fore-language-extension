# Тест Йохансена


[Тест
 Йохансена](Lib.chm::/05_Statistics/UiModelling_Johansen_test.htm) является средством для проверки коинтеграции
 временных рядов. На панели «Тест Йохансена»
 отображаются расчеты данного теста:


![](Test%20Yohansena.gif)


Для определения количества коинтеграционных связей следует сравнить
 значение отношения максимального правдоподобия
 с N процентным (N=5%, 1%) уровнем
 значимости: если данное значение больше N
 процентного уровня, то предполагают отклонение гипотезы о наличии соответствующего
 количества (нет, 1, 2, … n) коинтеграционных связей в сторону альтернативной
 гипотезы. Альтернативной гипотезой является гипотеза о том, что все ряды
 являются стационарными.


Процентные уровни значимости выводятся в полях «Критическое
 значение 5%» и «Критическое значение
 1%».
