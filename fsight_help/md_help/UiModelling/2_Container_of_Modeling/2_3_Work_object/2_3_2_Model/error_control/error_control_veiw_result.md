# Просмотр результатов


Данная панель позволяет выбирать сценарий, по которому будет производиться
 расчет.


![](error%20control%20view%20result.gif)


По умолчанию выбран сценарий «Факт».


В поле «Уравнение» указывается
 эндогенная переменная, для которой на дополнительных панелях будут отображены
 результаты расчетов.
