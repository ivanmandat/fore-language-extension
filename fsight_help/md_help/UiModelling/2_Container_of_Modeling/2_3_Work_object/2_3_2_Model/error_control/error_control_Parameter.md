# Параметры


На панели редактируются основные параметры модели:


![](error%20control%20parameter.gif)


	- Тип
	 модели коррекции ошибок. Определяет модель коррекции ошибок:


	-


		- без тренда в авторегрессии, без константы в коинтеграционном
		 уравнении;


		- без тренда в авторегрессии, с константой в коинтеграционном
		 уравнении;


		- с константой в авторегрессии и в коинтеграционном уравнении
		 (тренд в исходных данных);


		- с трендом в авторегрессии, с константой и трендом в коинтеграционном
		 уравнении;


		- с квадратичным трендом в авторегрессии, с константой и трендом
		 в коинтеграционном уравнении;


	- Число коинтеграционных связей.
	 Определяется количество коинтеграционных уравнений модели. Это значение
	 должно быть не меньше единицы и меньше числа эндогенных переменных.
	 Точное число коинтеграционных уравнений, как правило, не известно
	 наперёд и подлежит оценке, например, с помощью теста Йохансена;


	- Эндогенные переменные.
	 Добавление, удаление, преобразования над эндогенными переменными аналогично
	 данным [действиям
	 с входными переменными](../Standart_Model/UiModelling_Model_Factor_work.htm) (факторами). Для каждой эндогенной переменной
	 определяется состав экзогенных переменных. При выделении переменной
	 в списке эндогенных ниже будет отображен список ее экзогенных переменных.


	- Порядок авторегрессии.
	 В поле задайте порядок авторегрессии для эндогенных переменных.
	 Введите номера или диапазоны порядка авторегрессии, разделенные запятыми.
	 Диапазон порядка авторегрессии указывается через знак «-».


	- Экзогенные переменные.
	 Добавление, удаление, задание лага, преобразования над экзогенными
	 переменными аналогичны данным [действиям
	 с входными переменными](../Standart_Model/UiModelling_Model_Factor_work.htm) (факторами).


Примечание.
 Деление переменных на эндогенные
 и экзогенные определяется содержательной
 стороной модели. Предполагается, что в каждом уравнении экзогенные переменные
 некоррелированы с ошибкой, а эндогенные - коррелированы с ошибкой.


[![](../../../../opened.gif)![](../../../../closed.gif)Для
 редактирования формулы переменной](javascript:TextPopup(this))


			- дважды щелкните по ней мышью;


			- нажмите кнопку «Редактировать»;


			- выполните одноименную команду контекстного меню.


	Будет открыт [редактор
	 выражения](UiNav.Chm::/GUI/ExpressionEditor.htm).


	- Корректировка прогноза.
	 Установка флажка позволяет задать переменную, используемую для корректировки
	 прогноза. Переменная не включаются в идентифицированное уравнение
	 модели. После установки флажка станет доступна кнопка «Создать»,
	 которая создает переменную (без данных), используемую для корректировки
	 прогноза. Динамика переменной соответствует динамике модели.
	 Созданная переменная располагается в корне контейнера моделирования,
	 имеет наименование «Фактор корректировки
	 прогноза» и автоматически будет открыта для редактирования.


При выборе переменной, количество измерений
 которой не совпадает с количеством измерений моделируемой переменной,
 будет открыт диалог «[Изменение
 размерности](../../UiModelling_ChangeDimension.htm)», в котором необходимо произвести фиксацию по измерениям,
 отсутствующим у моделируемой переменной. Вызвать его повторно можно нажав
 кнопку «Зафиксировать».


Примечание.
 Если родительским объектом контейнера моделирования является база данных
 временных рядов, то для расчета модели используются переменные на атрибутах.
 Порядок работы с ними приведен в разделе «[Параметры
 модели](../Standart_Model/UiModelling_Panel_Param_Attr.htm)». Создание и фиксация переменной недоступны.
