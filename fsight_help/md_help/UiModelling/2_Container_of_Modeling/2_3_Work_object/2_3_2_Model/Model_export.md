# Экспорт модели


Для всех[ стандартных моделей](Standart_Model/Standart_Model.htm)
 и [векторной модели коррекции
 ошибок (ECM)](error_control/error_control.htm) доступен экспорт в файл или регламентный отчет.


Для экспорта модели:


	- Откройте требуемую модель.


	- Выберите пункт «Контейнер моделирования
	 > Экспортировать» в главном меню.


	- Выберите требуемый формат экспорта:


		- В файл. Модель будет
		 экспортирована в файл. Доступные форматы:


			- HTML (*.html);


			- RTF (*.rtf);


			- XLS (*.xls);


			- PDF (*.pdf);


		- В регламентный отчет.
		 Модель будет экспортирована в регламентный отчет.


	- Будет открыт диалог сохранения объекта. Выберите местоположение
	 файла, в который будет экспортирована модель.


В результате экспорта будет создан файл или регламентный отчет, содержащий
 информацию со следующих панелей модели:


	- [Спецификация](Specification/UiModelling_Model_Specification.htm);


	- [Идентифицированное
	 уравнение](Standart_Model/identified_equation.htm);


	- [Статистические
	 характеристики](Standart_Model/statistical_features.htm);


	- [Графики
	 и диаграммы](Standart_Model/UiModelling_Model_Fact.htm);


	- [Ряды (числовые
	 значения)](Standart_Model/UiModelling_Model_Serie.htm);


	- [Факторы](Standart_Model/uimodelling_model_Factor.htm);


	- [Корреляция
	 факторов](Standart_Model/uimodelling_model_correlation.htm);


	- [Предупреждения](Standart_Model/UiModelling_Model_Warning.htm).
