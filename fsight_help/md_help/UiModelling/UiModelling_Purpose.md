# Начало работы с инструментом «Моделирование и прогнозирование»


Для начала работы с инструментом создайте новый или откройте готовый
 контейнер моделирования.


[![](opened.gif)![](closed.gif)Создание нового
 контейнера моделирования](javascript:TextPopup(this))


	Для создания нового контейнера моделирования в [навигаторе
	 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm):


		- нажмите кнопку «Новый объект
		 > Контейнер моделирования» в группе «Создать»
		 на вкладке «Главная» ленты
		 инструментов;


		- выполните команду «Создать
		 > Контейнер моделирования» в контекстном меню навигатора
		 объектов.


[![](opened.gif)![](closed.gif)Открытие готового
 контейнера моделирования](javascript:TextPopup(this))


	Для открытия на редактирование или просмотр готового контейнера
	 моделирования в [навигаторе
	 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm):


		- выделите контейнер моделирования и:


			- нажмите кнопку «Редактировать»
			 в группе «Открыть»
			 на вкладке «Главная»
			 ленты инструментов;


			- выполните команду «Редактировать»
			 в контекстном меню контейнера моделирования;


		- выделите базу данных временных рядов и:


			- нажмите кнопку «Открыть
			 > В контейнере моделирования» в группе «Открыть»
			 на вкладке «Главная»
			 ленты инструментов;


			- выполните команду «Открыть
			 с помощью > Контейнер моделирования» в контекстном
			 меню выбранного объекта.


	При таком способе открытия будет открыт
	 внутренний контейнер моделирования базы данных временных рядов.


Для моделирования и прогнозирования
 в настольном приложении смотрите статью «[Моделирование в настольном
 приложении](1_Modelling/UiModelling_First.htm)».


Для моделирования и прогнозирования
 в веб-приложении смотрите статью «[Моделирование в веб-приложении](Web/WebApplication.htm)».


Если остались вопросы, смотрите
 статью «[Вопросы
 и ответы](FAQ/FAQ.htm)».


Основное окно инструмента «Моделирование
 и прогнозирование»:


![](1_modelling/mainwindow.png)


[![](opened.gif)![](closed.gif)Элементы окна
 контейнера моделирования](javascript:TextPopup(this))


		- [Главное меню](GetStarted.chm::/interface/interface_description.htm#main_menu).
		 Содержит список команд, предназначенных для выполнения различных
		 операций над объектами.


		- [Панели инструментов](GetStarted.chm::/interface/interface_description.htm#toolbars).
		 Содержит кнопки для настройки объектов.


		- Панель объектов.
		 Содержит все объекты контейнера моделирования в виде дерева.


		- Панель сценариев.
		 Содержит список сценариев моделирования, по которым может осуществляться
		 расчёт.


		- Рабочая область.
		 Предназначена для построения моделей.
