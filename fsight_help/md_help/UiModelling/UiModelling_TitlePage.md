# Моделирование и прогнозирование


Для моделирования и прогнозирования предназначен инструмент «Моделирование
 и прогнозирование»:


![](1_modelling/modelling.png)


Инструмент «Моделирование и прогнозирование»
 для подготовленного пользователя и позволяет использовать без ограничений
 все методы «Форсайт. Аналитическая платформа».


Ключевые возможности:


	- описательная статистика данных с использованием различных методов:
	 тест Дики-Фуллера, тест Гранжера и т.п.;


	- сценарное моделирование;


	- конструирование моделей и алгоритмов с помощью любого из методов
	 «Форсайт. Аналитическая платформа»;


	- режим отладочного расчета модели для проверки правильности алгоритмов
	 и корректировки ошибок;


	- написание модельного кода на языках R и Python.
