# Рабочая область


Рабочая область является основной функциональной частью моделирования
 и прогнозирования и предназначена для работы с [моделью](Work/Web_Model_Work.htm):


![](WorkingArea.gif)


Рабочая область содержит:


	- [Область
	 представления данных](UiModeling_w_ViewArea.htm). Предназначена для отображения модели в различных
	 формах;


	- [Дерево
	 рядов](SeriesTree_web.htm). Отображает древовидную структуру элементов открытого источника
	 данных;


	- [Панель
	 результатов](UiModelling_w_ResultPanel.htm). Предназначена для просмотра характеристик объекта,
	 выделенного в области представления данных.
