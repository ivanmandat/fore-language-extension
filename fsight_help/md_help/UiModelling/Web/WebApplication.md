# Моделирование в веб-приложении


В веб-приложении моделирование и прогнозирование выполняется в рамках
 [рабочей области](UiModelling_w_WorkingArea.htm).


Рабочая область - основная функциональная часть инструмента моделирования
 и прогнозирования, позволяющая создать и настроить взаимодействие следующих
 объектов:


	- [модели](Work/Web_Model_Work.htm);


	- [переменных](Work/UiModelling_w_Varable.htm);


	- [уравнений](Work/Web_Equation_Work.htm);


	- [сценариев моделирования](Work/UiModelling_w_Scenario.htm).


«Модель» - объект, выполняющий
 последовательный расчёт уравнений с целью получения выходных данных в
 моделируемых переменных. По окончании расчета результаты выгружаются в
 переменные.


«Переменная» - объект, предназначенный
 для получения исходных данных и выгрузки результатов выполнения расчёта.


«Уравнение» - объект, предназначенный
 для расчета моделируемых переменных путем преобразования исходных переменных
 статистическими и математическими методами. Расчёты ведутся с учетом календарной
 динамики модели (годы, полугодия, кварталы, месяцы, дни).


«Сценарий моделирования» -
 объект, предназначенный для создания в моделируемой переменной ряда, в
 который будут выгружены данные после окончания расчёта.


## Процесс моделирования


В веб-приложении моделирование сводится к созданию [модели](Work/Web_Model_Work.htm).
 В рамках модели создаются [переменные
 моделирования](Work/UiModelling_w_Varable.htm). Если переменные объединены между собой связями, то
 они образуют [уравнение](Work/Web_Equation_Work.htm). Совокупность
 уравнений является моделью, которая может быть рассчитана по одному или
 нескольким сценариям.
