# Запуск моделирования и прогнозирования


Для начала работы с моделированием и прогнозированием в веб-приложении:


	- [Запустите
	 веб-приложение](GetStarted.chm::/GetStarted/Get_Started.htm)
	 и выполните регистрацию. Будет отображена страница приветствия.


	- На странице приветствия выберите инструмент «Моделирование
	 и прогнозирование».


Будет открыто [основное
 окно](../2_Container_of_Modeling/2_2_Window_container/UiModelling_Window_container.htm) моделирования и прогнозирования.


## Режимы работы


Любая модель может быть открыта в следующих режимах:


	- [режим
	 редактирования](Work/UiModelling_w_work.htm). Предназначен для создания и редактирования
	 моделей;


	- [режим
	 просмотра](Work/View_mode.htm). Используется для работы с готовыми моделями.
	 Доступен просмотр параметров и расчет моделей.


Для открытия модели в режиме редактирования:


	- запустите навигатор объектов и выберите требуемую модель:


		- нажмите кнопку ![](Work/b_var_edit.gif) «Редактировать» на вкладке «Главная» ленты инструментов;


		- выполните команду «Редактировать»
		 в контекстном меню модели;


	- запустите инструмент «Моделирование
	 и прогнозирование» и [создайте
	 новую модель](Work/Web_Model_Work.htm#create_of_model) или [откройте
	 существующую](Work/Web_Model_Work.htm#open_a_model);


	- [откройте модель](Work/Web_Model_Work.htm#open_a_model)
	 со специальным параметром «mode»,
	 имеющим значение «edit».


Выбранная модель будет открыта в режиме редактирования. Если у пользователя
 отсутствуют права на редактирование модели, то она будет открыта в режиме
 просмотра.


Для запуска моделирования и прогнозирования в режиме просмотра:


	- запустите навигатор объектов и выберите требуемую модель:


		- нажмите кнопку ![](b_open.gif) «Открыть» на вкладке «Главная»
		 ленты инструментов;


		- выполните команду «Открыть»
		 в контекстном меню модели;


	- [откройте модель](Work/Web_Model_Work.htm#open_a_model)
	 со специальным параметром «mode»,
	 имеющим значение «view».


Выбранная модель будет открыта в режиме просмотра.
