# Настройка преобразования моделируемой переменной


Для настройки преобразования данных моделируемой переменной предназначена
 вкладка «Преобразования», расположенная
 на боковой панели.


[![](../../opened.gif)![](../../closed.gif)Для отображения
 вкладки](javascript:TextPopup(this))


		- Убедитесь, что боковая панель
		 отображается;


		- Выберите моделируемую переменную или одну из связей уравнения;


		- Перейдите на вкладку «Преобразования»
		 в боковой панели.


![](Transformations.gif)


Если требуется преобразование данных моделируемой переменной, то установите
 соответствующий флажок и выберите один из методов.


Описание методов преобразования данных приведено в разделе «[Преобразования
 над переменными или факторами](../../2_Container_of_Modeling/2_3_Work_object/UiModelling_work_Changes.htm)».
