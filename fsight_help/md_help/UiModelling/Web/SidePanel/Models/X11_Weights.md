# Задание весовых коэффициентов


Месяцы содержат различное число рабочих дней, это может повлиять на
 окончательный результат анализа временного ряда. Метод «[X11](w_eq_X11.htm)» даёт возможность учесть это обстоятельство
 введением весовых коэффициентов для дней недели.


Для задания значения весовых коэффициентов для каждого дня недели предназначена
 вкладка «Весовые коэффициенты»,
 расположенная на боковой панели.


[![](../../../opened.gif)![](../../../closed.gif)Для отображения
 вкладки](javascript:TextPopup(this))


		- Убедитесь, что боковая панель отображается;


		- Выберите моделируемую переменную или одну из связей уравнения
		 вида «X11»;


		- Перейдите на вкладку «Весовые
		 коэффициенты» на боковой панели.


![](X11_Weights.gif)


Задайте значение коэффициента для каждого дня недели. По умолчанию весовые
 коэффициенты равны единице.


Для нормирования коэффициентов, т.е. приведения суммы всех коэффициентов
 к семи, нажмите кнопку «Нормировать».
 Нормирование будет выполнено по формуле:


Wi=Wi*7/S,


где:


	- Wi.
	 Вес коэффициента;


	- S.
	 Ненормированная сумма.
