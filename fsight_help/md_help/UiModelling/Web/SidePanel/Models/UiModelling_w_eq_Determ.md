# Детерминированное уравнение


В общем случае, метод выполняет нелинейные преобразования данных.


Для настройки параметров метода используйте вкладку «Уравнение»
 на боковой панели.


[![](../../../opened.gif)![](../../../closed.gif)Для отображения
 вкладки](javascript:TextPopup(this))


		- Убедитесь, что боковая панель отображается;


		- Выберите моделируемую переменную или одну из связей уравнения
		 вида «Детерминированное уравнение»;


		- Перейдите на вкладку «Уравнение»
		 на боковой панели.


![](Determ.gif)


Параметры метода:


	- Период действия формулы.
	 Установите переключатель, соответствующий периоду, на котором будут
	 произведены расчеты:


	-


		- Период идентификации.
		 Формула применяется для значений, расположенных в периоде идентификации.
		 Границы периода задаются на вкладке «[Периоды расчёта](../UiModelling_w_pp_CalcPeriod.htm)»;


		- Период прогнозирования.
		 По умолчанию формула применяется для значений, расположенных в
		 периоде прогнозирования. Границы периода задаются на вкладке «[Периоды
		 расчёта](../UiModelling_w_pp_CalcPeriod.htm)»;


		- Весь период. Расчет
		 по формуле ведется для значений, расположенных в периодах идентификации
		 и прогнозирования.


Формула, по которой рассчитывается уравнение, доступна для редактирования
 на вкладке панели результатов «[Спецификация](../../ResultsPanel/UiModelling_w_rp_specification.htm)».


Примечание.
 Если детерминированное уравнение получено в результате изменения вида
 уравнения линейной/нелинейной регрессии, то в формуле будут присутствовать
 рассчитанные коэффициенты регрессии.
