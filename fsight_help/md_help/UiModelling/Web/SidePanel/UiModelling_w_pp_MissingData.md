# Обработка пропусков


Для выбора метода обработки пропусков, применяемого для расчёта отсутствующих
 значений в переменных, используйте вкладку «Обработка
 пропусков» на боковой панели.


[![](../../opened.gif)![](../../closed.gif)Для отображения
 вкладки](javascript:TextPopup(this))


		- Убедитесь, что боковая панель
		 отображается.


		- Выберите моделируемую переменную или одну из связей уравнения.


		- Перейдите на вкладку «Обработка
		 пропусков» в боковой
		 панели.


![](MissingData.gif)


Если требуется обработка пропусков, то установите флажок «Применить»
 и выберите один из методов:


	- Casewise. Используется
	 по умолчанию. Пустые значения исключаются из ряда. Расчёты ведутся
	 без их учета;


	- Геометрическая интерполяция.
	 Для вычисления пустых значений ряда используется геометрическая интерполяция;


	- Значения заданного ряда.
	 Пустые значения моделируемого ряда заменяются значениями из ряда,
	 указанного в параметре «Ряд со значениями»;


	- Интерполяция кубическим сплайном.
	 Для вычисления пустых значений ряда используется интерполяция кубическим
	 сплайном;


	- Линейная интерполяция.
	 Для вычисления пустых значений ряда используется линейная интерполяция
	 по двум соседним точкам;


	- Линейный тренд. Для
	 вычисления пустых значений ряда используется линейный тренд;


	- По шаблону. Для вычисления
	 пустых значений ряда используется интерполяция по шаблону. В параметре
	 «Шаблонный ряд» укажите переменную,
	 представляющую шаблонный ряд данных;


	- Предыдущее значение.
	 Пустые значения ряда заменяются предыдущим непустым значением;


	- Произвольное значение.
	 Пустые значения ряда заменяются случайным числом из интервала [минимальное значение ряда; максимальное значение ряда];


	- Следующее значение.
	 Пустые значения ряда заменяются следующим непустым значением;


	- Среднее значение по выборке.
	 Пустые значения ряда вычисляются как среднее по выборке;


	- Среднее по N соседним точкам.
	 Пустые значения ряда вычисляются как среднее по N
	 соседним точкам. Количество соседних точек (N)
	 укажите в поле «Параметр»;


	- Темп роста к предыдущему периоду.
	 Пустые значения ряда вычисляются исходя из темпа роста существующих
	 значений к предыдущему периоду. Количество периодов укажите в поле
	 «Параметр»;


	- Темп роста к следующему периоду.
	 Пустые значения ряда вычисляются исходя из темпа роста существующих
	 значений к следующему периоду. Количество периодов укажите в поле
	 «Параметр»;


	- Указанное значение.
	 Пустые значения ряда заменяются числом, указанным в поле «Параметр».


[![](../../opened.gif)![](../../closed.gif)Как задать/очистить
 параметр «Шаблонный ряд»/«Ряд со значениями»?](javascript:TextPopup(this))


	Для задания ряда выберите его в раскрывающемся списке «Шаблонный
	 ряд»/«Ряд со значениями»
	 или:


		- Выберите переменную в области представления данных (представление
		 «[Схема](../UiModeling_w_ViewArea.htm)»).


		- Зажмите кнопку мыши и перетащите переменную в
		 поле «Шаблонный ряд».


		- Отпустите кнопку мыши. Значения переменной будут использованы
		 для заполнения пропусков.


	Для удаления шаблонного ряда/ряда со значениями нажмите кнопку ![](../Work/b_delete.gif).


	Примечание.
	 Если из модели [удалена](../Work/UiModelling_w_work.htm#delete)
	 переменная, используемая в качестве шаблонного ряда или ряда со значениями,
	 то в методе обработки пропусков требуется указать переменную, которая
	 будет использоваться вместо удаленной.
