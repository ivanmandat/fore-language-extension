# Настройка порядка расчета уравнений в модели


Настройка порядка расчета позволяет выстроить уравнения модели в соответствии
 с зависимостями между ними.


Для настройки порядка расчета нажмите кнопку ![](b_CalcOrder.gif) «Порядок расчета» на вкладке ленты
 «Расчет». Отобразится диалог «Порядок расчета», содержащий все
 уравнения модели, выстроенные в порядке расчета:


![](Calculation_Order.gif)


Совокупность уравнений, выстроенных в порядке расчета, является цепочкой
 расчета модели. Для структурирования цепочки расчета применяются папки
 и системы уравнений.


[![](../../opened.gif)![](../../closed.gif)Создание
 папки](javascript:TextPopup(this))


		- Нажмите кнопку «Добавить».


		- В открывшемся меню выполните команду «Папка».


	Папка будет добавлена после выделенного элемента.


[![](../../opened.gif)![](../../closed.gif)Создание
 системы уравнений](javascript:TextPopup(this))


		- Нажмите кнопку «Добавить».


		- В открывшемся меню выполните команду «Система
		 уравнений». Пустая система будет добавлена после выделенного
		 элемента.


		- Перетащите требуемые уравнения в систему. Объединять в систему
		 допускается уравнения следующих типов: [Нелинейная
		 регрессия](../SidePanel/Models/UiModelling_w_eq_NonLinearRegr.htm), [Линейная
		 регрессия](../SidePanel/Models/UiModelling_w_eq_LinearRegr.htm), [Детерминированное
		 уравнение](../SidePanel/Models/UiModelling_w_eq_Determ.htm), [Модель
		 коррекции ошибок](../SidePanel/Models/w_eq_ECM.htm).


	Выбранные уравнения будут объединены в систему уравнений. Подробно
	 см. раздел «[Работа с системой уравнений](Equation_system.htm)».


[![](../../opened.gif)![](../../closed.gif)Изменение
 последовательности элементов в цепочке расчета](javascript:TextPopup(this))


		- Используйте кнопки ![](b_up.gif) «Вверх» и ![](b_down.gif) «Вниз».


		- Перетащите элемент в требуемую позицию.


[![](../../opened.gif)![](../../closed.gif)Автоматическое
 построение цепочки расчета](javascript:TextPopup(this))


	Автоматическое построение позволяет выявить зависимости между уравнениями
	 и выстроить их согласно найденным зависимостям.


	Для выполнения автоматического построения нажмите кнопку «Автоматический порядок». Раскрывающееся
	 меню кнопки позволяет задать дополнительные параметры:


		- С нарушением структуры
		 папок. По умолчанию флажок снят и автоматическое построение
		 выполняется отдельно для каждой папки, входящей в цепочку расчета.
		 Если флажок установлен, то все уравнения из всех папок извлекаются
		 на уровень, для которого производится автоматическое построение,
		 а папки удаляются. Таким образом, настройка порядка расчета выполняется
		 без учета папок.


	В результате, уравнения будут выстроены в порядке зависимости. Если
	 уравнения линейной регрессии, нелинейной регрессии, детерминированного
	 уравнения, модели коррекции ошибок образуют цикл, то они объединяются
	 в систему уравнений.


[![](../../opened.gif)![](../../closed.gif)Переименование
 папки/системы уравнений](javascript:TextPopup(this))


		- Дважды щелкните по требуемому элементу. Наименование перейдет
		 в режим редактирования.


		- Отредактируйте наименование.


		- Нажмите клавишу ENTER.


[![](../../opened.gif)![](../../closed.gif)Удаление
 элементов из цепочки расчета](javascript:TextPopup(this))


		- Выделите требуемый элемент.


		- Нажмите кнопку «Удалить».


	Элемент будет удален.


	Примечание.
	 Удаление доступно только для папок и систем уравнений, не содержащих
	 дочерних элементов.
