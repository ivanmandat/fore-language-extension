# Работа в веб-приложении


Для начала работы [запустите
 инструмент](../../1_Modelling/Start_Modelling.htm) «Моделирование и прогнозирование»
 и создайте новую [модель](Web_Model_Work.htm) или откройте
 существующую.


Для редактирования параметров объекта и просмотра его характеристик
 используйте боковую панель и [панель
 результатов](../UiModelling_w_ResultPanel.htm).


## Базовые операции с объектами и рабочей областью


[![](../../opened.gif)![](../../closed.gif)Удаление
 объекта](javascript:TextPopup(this))


	Для удаления выделенного объекта:


		- нажмите клавишу DELETE;


		- нажмите кнопку ![](b_delete.gif) «Удалить» на вкладке ленты «Главная».


	Будет запрошено подтверждение выполняемого действия. При положительном
	 ответе объект будет удален.


	Примечание.
	 Если удалена переменная, используемая в [методе
	 обработки пропусков](../SidePanel/UiModelling_w_pp_MissingData.htm) в качестве шаблонного ряда или ряда со значениями,
	 то требуется указать переменную, которая будет использоваться вместо
	 удаленной.


[![](../../opened.gif)![](../../closed.gif)Изменение
 размеров объекта](javascript:TextPopup(this))


	Для изменения размеров объекта:


		- Наведите указатель мыши на границу объекта. Указатель мыши
		 должен изменить вид на двунаправленную стрелку.


		- Зажмите кнопку мыши и растяните или сожмите элемент
		 до необходимого размера.


[![](../../opened.gif)![](../../closed.gif)Перемещение
 объектов](javascript:TextPopup(this))


	Для перемещения объекта:


		- Наведите указатель мыши на объект, при этом указатель сменит
		 вид на четырехнаправленную стрелку.


		- Нажмите кнопку мыши и, удерживая её, перетащите
		 объект в нужную позицию.


	Объект будет перемещен.


[![](../../opened.gif)![](../../closed.gif)Перемещение
 по схеме](javascript:TextPopup(this))


	Для перемещения по схеме:


		- Переведите кнопку ![](b_move.gif) «Перемещение» на вкладке «Вид» ленты инструментов в нажатое
		 состояние. Схема будет переведена в режим перемещения.


		- Наведите указатель мыши на схему, при этом указатель сменит
		 вид на четырехнаправленную стрелку.


		- Нажмите кнопку мыши и, удерживая её, переместитесь
		 на нужное место в схеме.


	После перемещения можно продолжать работать с объектами на схеме
	 без выключения режима перемещения.


	Для выключения режима перемещения переведите кнопку ![](b_move.gif) «Перемещение» на вкладке «Вид»
	 ленты инструментов в ненажатое состояние. Для перемещения по схеме
	 можно будет использовать полосы прокрутки.


	Совет. Для быстрого
	 перевода схемы в режим перемещения нажмите и удерживайте клавишу ПРОБЕЛ.
	 Данное действие аналогично переводу кнопки ![](b_move.gif) «Перемещение» в нажатое состояние.
	 Если отпустить клавишу ПРОБЕЛ, то режим перемещения будет выключен.


[![](../../opened.gif)![](../../closed.gif)Выделение
 объектов](javascript:TextPopup(this))


	Для выделения одного объекта щелкните по нему мышью.


	Для выделения нескольких объектов:


		- зажмите клавишу CTRL и последовательно щёлкните по
		 каждому выделяемому объекту;


		- используйте рамку для выделения объектов:


		-


			- Определите группу объектов,
			 которые требуется выделить.


			- Щёлкните и зажмите кнопку мыши за пределами группы выделяемых объектов.


			- Начните перемещать указатель
			 мыши. Будет отображена прямоугольная рамка.


			- Перемещайте указатель до
			 тех пор, пока все требуемые объекты не будут помещены в рамку.


			- Отпустите указатель мыши.
			 Все объекты, полностью помещенные в рамку, будут выделены.


[![](../../opened.gif)![](../../closed.gif)Изменение
 масштаба области представления данных](javascript:TextPopup(this))


	Изменение масштаба доступно только для представления «[Схема](../UiModeling_w_ViewArea.htm#scheme)». По умолчанию
	 установлен масштаб 100%.


	Для изменения масштаба:


		- используйте мини-карту в представлении «[Схема](../UiModeling_w_ViewArea.htm#scheme)».
		 Механизм изменения масштаба с помощью мини-карты приведен в разделе
		 «[Область
		 представления данных](../UiModeling_w_ViewArea.htm)»;


		- зажмите клавишу ALT и прокрутите колесико мыши вверх для
		 увеличения, вниз - для уменьшения масштаба;


		- используйте группу «Масштаб»,
		 расположенную на вкладке «Вид»
		 ленты инструментов:


			- для задания произвольного значения масштаба используйте
			 поле «Масштаб». Допускаются
			 значения масштаба в диапазоне от 10% до 200%;


			- для автоматического подбора значения масштаба используйте
			 раскрывающееся меню кнопки ![](b_Scale.gif)
			 «Подогнать»:


				- По окну.
				 Масштаб будет таким, чтобы в видимой части схемы отображались
				 все объекты;


				- По ширине.
				 Масштаб будет таким, чтобы схема отображалась в полную
				 ширину. Верхние и нижние части схемы могут быть не видны;


				- По высоте.
				 Масштаб будет таким, чтобы схема отображалась в полную
				 высоту. Боковые части схемы могут быть не видны.
