## С чего начать


		 Сборка Net
		 содержит ресурсы для работы с электронной почтой.


		Изучив данный раздел, вы сможете с применением языка Fore
		 формировать и отправлять сообщения электронной почты с
		 помощью почтового клиента, который настроен в операционной системе
		 по умолчанию, или с помощью почтовой службы SMTP.


		Сборка предоставляет ряд общих интерфейсов и перечислений, описывающих
		 различные элементы для работы с электронной почтой, а также две
		 группы [классов](Class/KeNet_Classes.htm), имеющих
		 различия во внутренней реализации:


			- Классы с префиксом Net*
			 - реализованы с использованием .NET Framework, поддерживают
			 работу с электронной почтой только в ОС Windows. Данные классы
			 не рекомендуются к использованию, так как являются устаревшими;


			- Классы с префиксом Curl*
			 - кроссплатформенная реализация для работы с электронной почтой.
			 Рекомендуется к использованию в прикладном коде.


		Описание ресурсов сборки представлено в следующих подразделах:

			- [Классы сборки Net](Class/KeNet_Classes.htm)


			- [Примеры](Samples/Samples.htm)
