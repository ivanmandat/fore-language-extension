# NetMailPriority


## Описание


Перечисление NetMailPriority содержит варианты приоритета, с которым будет отправляться сообщение электронной почты.


Используется следующим свойством:


-
[INetMailMessage.Priority](../Interface/INetMailMessage/INetMailMessage.Priority.htm).


## Допустимые значения


 Значение
 Краткое описание


 1
 High. Высокий.


 2
 Low. Низкий.


 3
 Normal. Обычный.
