# NetHttpCommandType


## Описание


Перечисление NetHttpCommandType
 содержит типы запросов, которые могут быть отправлены сетевым ресурсам.


Используется следующими свойствами и методами:


	- [INetHttpCommand.Command](../Interface/INetHttpCommand/INetHttpCommand.Command.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Post. Отправка Post-запроса,
		 в результате которого сетевой ресурс получает данные.


		 1
		 Get_. Отправка Get-запроса,
		 в результате с сетевого ресурса будут получены данные.
