# INetCollection.RemoveAt


## Синтаксис


RemoveAt(Index: Integer);


## Параметры


Index - индекс элемента коллекции.


## Описание


Метод RemoveAt осуществляет удаление элемента коллекции, индекс которого передается посредством параметра Index.
