# INetCollection


## Описание


Интерфейс INetCollection содержит
 базовые методы коллекций адресов и вложений, используемых при отправке
 электронных почтовых сообщений.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](INetCollection.Count.htm)


		 Свойство Count возвращает
		 количество элементов в коллекции.

		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Clear](INetCollection.Clear.htm)


		 Метод Clear осуществляет

		 ![](../../Sub_Image.gif)
		 [RemoveAt](INetCollection.RemoveAt.htm)


		 Метод RemoveAt осуществляет
