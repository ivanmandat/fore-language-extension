# INetHttpCommandCredential


Сборка: Net;


## Описание


Интерфейс INetHttpCommandCredential
 используется для указания учётных данных, которые необходимы для аутентификации
 пользователя при подключении к сетевым ресурсам.


## Иерархия наследования


INetHttpCommandCredential


## Комментарии


Учётные данные можно получить в свойстве [INetHttpCommand.Credential](../INetHttpCommand/INetHttpCommand.Credential.htm).
