# INetService.Connect


## Синтаксис


Connect: [INetCommand](../INetCommand/INetCommand.htm);


## Описание


Метод Connect осуществляет подключение
 к сетевому ресурсу и возвращает контекст для взаимодействия с ним.


## Комментарии


Перед подключением задайте [URL-адрес](../INetHttpService/INetHttpService.Url.htm)
 сетевого ресурса.
