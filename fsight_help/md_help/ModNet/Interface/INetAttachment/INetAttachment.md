# INetAttachment


## Описание


Интерфейс INetAttachment содержит свойства вложения сообщения электронной почты.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ContentName](INetAttachment.ContentName.htm)


		 Свойство ContentName
		 определяет путь и наименование файла, который необходимо приложить
		 к сообщению электронной почты.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ReleaseResource](INetAttachment.ReleaseResource.htm)


		 Метод ReleaseResource
		 осуществляет освобождение ресурса, используемого в качестве вложения
		 к сообщению электронной почты.
