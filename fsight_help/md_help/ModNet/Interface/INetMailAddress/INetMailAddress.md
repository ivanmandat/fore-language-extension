# INetMailAddress


## Описание


Интерфейс INetMailAddress содержит свойства адреса электронной почты.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Address](INetMailAddress.Address.htm)


		 Свойство Address возвращает
		 адрес электронной почты.


		 ![](../../Property_Image.gif)
		 [DisplayName](INetMailAddress.DisplayName.htm)


		 Свойство DisplayName
		 возвращает наименование, отображаемое вместо самого адреса электронной
		 почты.


		 ![](../../Property_Image.gif)
		 [Host](INetMailAddress.Host.htm)


		 Свойство Host возвращает
		 DNS-суффикс адреса электронной почты.


		 ![](../../Property_Image.gif)
		 [User](INetMailAddress.User.htm)


		 Свойство User возвращает
		 наименование пользователя из адреса электронной почты.
