# INetAttachmentCollection.IndexOf


## Синтаксис


IndexOf(Value: [INetAttachment](../INetAttachment/INetAttachment.htm)): Integer;


## Параметры


Value - вложение, индекс которого необходимо получить.


## Описание


Метод IndexOf возвращает индекс вложения, передаваемого посредством параметра Value.


Если вложение не содержится в коллекции, то метод возвращает значение «-1».
