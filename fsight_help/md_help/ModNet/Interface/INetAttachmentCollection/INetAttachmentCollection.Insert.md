# INetAttachmentCollection.Insert


## Синтаксис


Insert(Index: Integer; Value: [INetAttachment](../INetAttachment/INetAttachment.htm));


## Параметры


Index - индекс вложения, перед которым необходимо добавить вложение Value.


Value - вложение, которое необходимо добавить в коллекцию.


## Описание


Метод Insert осуществляет вставку вложения, передаваемого посредством параметра Value, перед вложением с индексом Index.
