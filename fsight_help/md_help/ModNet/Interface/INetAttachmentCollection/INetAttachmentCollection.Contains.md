# INetAttachmentCollection.Contains


## Синтаксис


Contains(Value: [INetAttachment](../INetAttachment/INetAttachment.htm)): Boolean;


## Параметры


Value - вложение, наличие которого в данной коллекции, необходимо проверить.


## Описание


Метод Contains осуществляет проверку, содержится ли вложение, передаваемое посредством параметра Value, в коллекции и возвращает True, если вложение уже имеется.
