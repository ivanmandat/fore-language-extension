# INetMailMessage


## Описание


Интерфейс INetMailMessage содержит свойства сообщения электронной почты.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attachments](INetMailMessage.Attachments.htm)


		 Свойство Attachments
		 возвращает коллекцию вложений, передаваемых вместе с электронным
		 почтовым сообщением.


		 ![](../../Property_Image.gif)
		 [Bcc](INetMailMessage.Bcc.htm)


		 Свойство Bcc возвращает
		 коллекцию адресов, по которым будет отправлена скрытая копия сообщения
		 электронной почты.


		 ![](../../Property_Image.gif)
		 [Body](INetMailMessage.Body.htm)


		 Свойство Body определяет
		 текст сообщения электронной почты.


		 ![](../../Property_Image.gif)
		 [BodyEncoding](INetMailMessage.BodyEncoding.htm)


		 Свойство BodyEncoding
		 определяет кодовую страницу, используемую для кодировки тела письма.


		 ![](../../Property_Image.gif)
		 [CC](INetMailMessage.CC.htm)


		 Свойство CC возвращает
		 коллекцию адресов, по которым будет отправлена копия сообщения
		 электронной почты.


		 ![](../../Property_Image.gif)
		 [From_](INetMailMessage.From_.htm)


		 Свойство From_ определяет
		 электронный адрес отправителя.


		 ![](../../Property_Image.gif)
		 [IsBodyHtml](INetMailMessage.IsBodyHtml.htm)


		 Свойство IsBodyHtml
		 определяет признак отправки сообщения в формате HTML.


		 ![](../../Property_Image.gif)
		 [Priority](INetMailMessage.Priority.htm)


		 Свойство Priority определяет
		 приоритет сообщения электронной почты.


		 ![](../../Property_Image.gif)
		 [ReplyTo](INetMailMessage.ReplyTo.htm)


		 Свойство ReplyTo определяет
		 адрес, по которому будет отправляться ответ на данное письмо.


		 ![](../../Property_Image.gif)
		 [Sender](INetMailMessage.Sender.htm)


		 Свойство Sender определяет
		 адрес, по которому будет отправлено оповещение с почтового сервера
		 в случае неудачной доставки сообщения.


		 ![](../../Property_Image.gif)
		 [Subject](INetMailMessage.Subject.htm)


		 Свойство Subject определяет
		 заголовок темы сообщения электронной почты.


		 ![](../../Property_Image.gif)
		 [SubjectEncoding](INetMailMessage.SubjectEncoding.htm)


		 Свойство SubjectEncoding
		 определяет кодовую страницу, используемую для кодировки заголовка
		 письма.


		 ![](../../Property_Image.gif)
		 [To_](INetMailMessage.To_.htm)


		 Свойство To_ возвращает
		 коллекцию адресов электронной почты получателей.
