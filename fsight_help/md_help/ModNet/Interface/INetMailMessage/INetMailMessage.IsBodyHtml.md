# INetMailMessage.IsBodyHtml


## Синтаксис


IsBodyHtml: Boolean;


## Описание


Свойство IsBodyHtml определяет
 признак отправки сообщения в формате HTML.


## Комментарии


Если свойству установлено значение True,
 то текст сообщения передается в формате HTML. По умолчанию свойству установлено
 значение False, при этом сообщение
 передается в текстовом виде.
