# INetMailAddressCollection


## Описание


Интерфейс INetMailAddressCollection
 содержит свойства и методы коллекции адресов электронной почты.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Item](INetMailAddressCollection.Item.htm)


		 Свойство Item возвращает
		 адрес электронной почты.
