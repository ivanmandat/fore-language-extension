# INetMailAddressCollection.Item


## Синтаксис


Item(Index: Integer): [INetMailAddress](../INetMailAddress/INetMailAddress.htm);


## Параметры


Index - индекс адреса электронной почты, который необходимо получить.


## Описание


Свойство Item возвращает адрес электронной почты, индекс которой передается посредством параметра Index.
