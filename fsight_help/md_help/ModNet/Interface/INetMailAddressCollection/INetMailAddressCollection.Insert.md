# INetMailAddressCollection.Insert


## Синтаксис


Insert(Index: Integer; Value: [INetMailAddress](../INetMailAddress/INetMailAddress.htm));


## Параметры


Index - индекс электронного адреса, перед которым необходимо добавить адрес Value.


Value - электронный адрес, который необходимо добавить в коллекцию.


## Описание


Метод Insert осуществляет вставку электронного адреса, передаваемого посредством параметра Value, перед адресом с индексом Index.
