# INetMailAddressCollection.ToString


## Синтаксис


ToString: String;


## Описание


Метод ToString возвращает список адресов в коллекции, в виде символьной строки.


## Комментарии


Адреса разделяются "," и представляются в следующем виде: "Наименование" <Адрес>. Если наименование, отображаемое вместо адреса при отправке, не указано - подставляется только адрес.
