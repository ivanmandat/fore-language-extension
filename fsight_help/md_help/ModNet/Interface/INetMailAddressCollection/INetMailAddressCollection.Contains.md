# INetMailAddressCollection.Contains


## Синтаксис


Contains(Value: [INetMailAddress](../INetMailAddress/INetMailAddress.htm)): Boolean;


## Параметры


Value - электронный адрес, наличие которого в данной коллекции, необходимо проверить.


## Описание


Метод Contains осуществляет проверку, содержится ли электронный адрес, передаваемый посредством параметра Value, в коллекции и возвращает True, если адрес уже имеется.
