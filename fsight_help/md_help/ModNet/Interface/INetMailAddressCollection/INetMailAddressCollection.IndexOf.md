# INetMailAddressCollection.IndexOf


## Синтаксис


IndexOf(Value: [INetMailAddress](../INetMailAddress/INetMailAddress.htm)): Integer;


## Параметры


Value - электронный адрес, индекс которого необходимо получить.


## Описание


Метод IndexOf возвращает индекс электронного адреса, передаваемого посредством параметра Value.


Если адрес не содержится в коллекции, то метод возвращает значение «-1».
