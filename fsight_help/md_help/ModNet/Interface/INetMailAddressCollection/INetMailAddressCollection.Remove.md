# INetMailAddressCollection.Remove


## Синтаксис


Remove(Value: [INetMailAddress](../INetMailAddress/INetMailAddress.htm)): Boolean;


## Параметры


Value - электронный адрес, который необходимо удалить.


## Описание


Метод Remove осуществляет удаление электронного адреса из коллекции.
