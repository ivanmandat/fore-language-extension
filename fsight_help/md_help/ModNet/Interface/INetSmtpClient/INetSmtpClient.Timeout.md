# INetSmtpClient.Timeout


## Синтаксис


Timeout: Integer;


## Описание


Свойство Timeout определяет время, в течение которого будет ожидаться отправка сообщения на сервер.


## Комментарии


Значение свойства указывается в миллисекундах. По умолчанию свойству установлено значение 100000 мс.
