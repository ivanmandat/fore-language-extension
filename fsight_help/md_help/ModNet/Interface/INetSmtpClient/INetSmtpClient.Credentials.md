# INetSmtpClient.Credentials


## Синтаксис


Credentials: [INetCredentialsByHost](../INetCredentialsByHost/INetCredentialsByHost.htm);


## Описание


Свойство Credentials возвращает учетные данные, используемые для авторизации на сервере исходящей почты. Учетные данные необходимы, если сервер исходящей почты запрашивает авторизацию.
