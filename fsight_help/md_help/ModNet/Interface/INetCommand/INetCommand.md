# INetCommand


Сборка: Net;


## Описание


Интерфейс INetCommand содержит
 базовые свойства и методы для работы с сетевыми ресурсами.


## Иерархия наследования


INetCommand


## Комментарии


Интерфейс является результатом подключения к сетевому ресурсу с помощью
 метода [INetService.Connect](../INetService/INetService.Connect.htm).
 Для задания дополнительных настроек работы с сетевым ресурсом INetCommand
 может быть приведён к интерфейсу [INetHttpCommand](../INetHttpCommand/INetHttpCommand.htm).
