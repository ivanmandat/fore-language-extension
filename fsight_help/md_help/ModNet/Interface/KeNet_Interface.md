# Интерфейсы сборки Net


		 Интерфейс
		 Краткое описание


		 ![](../Interface_Image.gif)
		 [INetAttachment](INetAttachment/INetAttachment.htm)


		 Интерфейс INetAttachment
		 содержит свойства вложения сообщения электронной почты.


		 ![](../Interface_Image.gif)
		 [INetAttachmentCollection](INetAttachmentCollection/INetAttachmentCollection.htm)


		 Интерфейс INetAttachmentCollection
		 содержит свойства и методы коллекции вложений сообщения электронной
		 почты.


		 ![](../Interface_Image.gif)
		 [INetCollection](INetCollection/INetCollection.htm)


		 Интерфейс INetCollection
		 содержит базовые методы коллекций адресов и вложений, используемых
		 при отправке электронных почтовых сообщений.


		 ![](../Interface_Image.gif)
		 [INetCommand](INetCommand/INetCommand.htm)


		 Интерфейс INetCommand
		 содержит базовые свойства и методы для работы с сетевыми ресурсами.


		 ![](../Interface_Image.gif)
		 [INetCredentialCache](INetCredentialCache/INetCredentialCache.htm)


		 Интерфейс INetCredentialCache
		 предоставляет хранилище для множества учетных записей.


		 ![](../Interface_Image.gif)
		 [INetCredentialsByHost](INetCredentialsByHost/INetCredentialsByHost.htm)


		 Интерфейс INetCredentialsByHost
		 предназначен для обеспечения восстановления учетных данных для
		 почтового сервера.


		 ![](../Interface_Image.gif)
		 [INetHttpCommand](INetHttpCommand/INetHttpCommand.htm)


		 Интерфейс INetHttpCommand
		 предназначен для настройки параметров работы с сетевым ресурсом
		 по протоколу HTTP.


		 ![](../Interface_Image.gif)
		 [INetHttpCommandCredential](INetHttpCommandCredential/INetHttpCommandCredential.htm)


		 Интерфейс INetHttpCommandCredential
		 используется для указания учётных данных, которые необходимы для
		 аутентификации пользователя при подключении к сетевым ресурсам.


		 ![](../Interface_Image.gif)
		 [INetHttpService](INetHttpService/INetHttpService.htm)


		 Интерфейс INetHttpService
		 содержит свойства и методы для работы с сетевыми ресурсами по
		 протоколу HTTP/HTTPS.


		 ![](../Interface_Image.gif)
		 [INetMailAddress](INetMailAddress/INetMailAddress.htm)


		 Интерфейс INetMailAddress
		 содержит свойства адреса электронной почты.


		 ![](../Interface_Image.gif)
		 [INetMailAddressCollection](INetMailAddressCollection/INetMailAddressCollection.htm)


		 Интерфейс INetMailAddressCollection
		 содержит свойства и методы коллекции адресов электронной почты.


		 ![](../Interface_Image.gif)
		 [INetMailMessage](INetMailMessage/INetMailMessage.htm)


		 Интерфейс INetMailMessage
		 содержит свойства сообщения электронной почты.


		 ![](../Interface_Image.gif)
		 [INetNetworkCredential](INetNetworkCredential/INetNetworkCredential.htm)


		 Интерфейс INetNetworkCredential
		 предоставляет учетные данные для схем аутентификации на основе
		 пароля, таких как basic, digest, NTLM и Kerberos.


		 ![](../Interface_Image.gif)
		 [INetService](INetService/INetService.htm)


		 Интерфейс INetService
		 содержит базовый метод для подключения к сетевым ресурсам.


		 ![](../Interface_Image.gif)
		 [INetServiceProxy](INetServiceProxy/INetServiceProxy.htm)


		 Интерфейс INetServiceProxy
		 предназначен для задания настроек прокси-сервера, которые будут
		 использоваться при выполнении внешних запросов в интернет.


		 ![](../Interface_Image.gif)
		 [INetSmtpClient](INetSmtpClient/INetSmtpClient.htm)


		 Интерфейс INetSmtpClient
		 содержит свойства и методы для отправки электронных почтовых сообщений
		 при помощи почтовой службы SMTP.
