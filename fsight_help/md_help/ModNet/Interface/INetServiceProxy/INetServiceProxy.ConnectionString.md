# INetServiceProxy.ConnectionString


## Синтаксис


ConnectionString: String;


## Описание


Свойство ConnectionString возвращает
 полную строку подключения к прокси-серверу.


## Комментарии


Строка подключения формируется на основании настроек, которые задаются
 в различных свойствах текущего интерфейса.
