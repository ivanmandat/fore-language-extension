# INetServiceProxy.NeedAuthentication


## Синтаксис


NeedAuthentication: Boolean;


## Описание


Свойство NeedAuthentication
 определяет, требуется ли аутентификация на прокси-сервере.


## Комментарии


Допустимые значения:


	- True. Требуется аутентификация.
	 Определите также свойства [INetServiceProxy.User](INetServiceProxy.User.htm)
	 и [INetServiceProxy.Password](INetServiceProxy.Password.htm).


	- False. Значение по умолчанию.
	 Аутентификация не требуется.
