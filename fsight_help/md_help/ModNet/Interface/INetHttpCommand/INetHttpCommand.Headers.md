# INetHttpCommand.Headers


## Синтаксис


Headers: [IStringList](ModCollections.chm::/Interface/IStringList/IStringList.htm);


## Описание


Свойство Headers возвращает
 коллекцию, в которой формируется список дополнительных заголовков.


## Комментарии


Дополнительные заголовки могут потребоваться для задания дополнительных
 опций, которые будут обработаны сервером при выполнении запроса.


## Пример


Пример использования приведён в описании свойства [INetHttpCommand.Credential](INetHttpCommand.Credential.htm).
