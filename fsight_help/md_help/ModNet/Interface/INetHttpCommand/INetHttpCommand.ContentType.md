# INetHttpCommand.ContentType


## Синтаксис


ContentType: String;


## Описание


Свойство ContentType возвращает
 тип содержимого, полученного в результате выполнения запроса.


## Комментарии


В качестве значения свойства будет получен MIME-тип содержимого ответа.
