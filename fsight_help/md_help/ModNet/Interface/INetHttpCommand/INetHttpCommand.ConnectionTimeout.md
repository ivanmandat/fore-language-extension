# INetHttpCommand.ConnectionTimeout


## Синтаксис


ConnectionTimeout: Integer;


## Описание


Свойство ConnectionTimeout определяет
 время ожидания подключения к GET-запросу.


## Комментарии


Время задаётся в миллисекундах. Если подключение не будет выполнено
 за указанное время, то его выполнение будет остановлено.
