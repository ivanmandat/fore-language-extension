# INetHttpCommand.Timeout


## Синтаксис


Timeout: Integer;


## Описание


Свойство Timeout определяет
 время ожидания выполнения GET-запроса.


## Комментарии


Время задаётся в миллисекундах. Если запрос не будет выполнен за указанное
 время, то его выполнение будет остановлено.
