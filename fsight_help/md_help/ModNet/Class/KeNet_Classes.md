# Классы сборки Net


		 Класс
		 Краткое описание


		 ![](../Class_Image.gif)
		 [CurlAttachment](CurlAttachment/CurlAttachment.htm)


		 Класс CurlAttachment
		 реализует вложение сообщения электронной почты.


		 ![](../Class_Image.gif)
		 [CurlCredentialCache](CurlCredentialCache/CurlCredentialCache.htm)


		 Класс CurlCredentialCache
		 реализует хранилище учётных записей множества Интернет-ресурсов.


		 ![](../Class_Image.gif)
		 [CurlMailAddress](CurlMailAddress/CurlMailAddress.htm)


		 Класс CurlMailAddress
		 реализует адрес электронной почты, на который необходимо отправить
		 сообщение, либо адрес отправителя.


		 ![](../Class_Image.gif)
		 [CurlMailMessage](CurlMailMessage/CurlMailMessage.htm)


		 Класс CurlMailMessage
		 реализует экземпляр сообщения электронной почты.


		 ![](../Class_Image.gif)
		 [CurlNetworkCredential](CurlNetworkCredential/CurlNetworkCredential.htm)


		 Класс CurlNetworkCredential
		 является базовым классом, который поддерживает учетные данные
		 в схемах аутентификации на основе пароля, таких как Basic, Digest,
		 NTLM и Kerberos.


		 ![](../Class_Image.gif)
		 [CurlSmtpClient](CurlSmtpClient/CurlSmtpClient.htm)


		 Класс CurlSmtpClient
		 реализует объект, позволяющий отправлять электронные почтовые
		 сообщения с помощью библиотеки libcurl по протоколу SMTP.


		 ![](../Class_Image.gif)
		 [NetHttpService](NetHttpService/NetHttpService.htm)


		 Класс NetHttpService
		 реализует объект, предназначенный для работы с сетевыми ресурсами.


		 ![](../Class_Image.gif)
		 [NetServiceProxy](NetServiceProxy/NetServiceProxy.htm)


		 Класс NetServiceProxy
		 реализует объект, предназначенный для задания настроек прокси-сервера
		 в репозитории.
