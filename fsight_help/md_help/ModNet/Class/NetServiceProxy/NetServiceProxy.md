# NetServiceProxy


Сборка: Net;


## Описание


Класс NetServiceProxy реализует
 объект, предназначенный для задания настроек прокси-сервера в репозитории.

		 используется ли прокси-сервер.


		 ![](../../Property_Image.gif)
		 [Address](../../Interface/INetServiceProxy/INetServiceProxy.Address.htm)

		 [ConnectionString](../../Interface/INetServiceProxy/INetServiceProxy.ConnectionString.htm)

		 ![](../../Property_Image.gif)

		 Свойство NoProxyString

		 напрямую, а не через прокси-сервер.

		 Свойство NoProxyStringOriginal

		 напрямую, а не через прокси-сервер.


		 ![](../../Property_Image.gif)
		 [Password](../../Interface/INetServiceProxy/INetServiceProxy.Password.htm)


		 Свойство Password определяет
		 пароль пользователя, используемый при аутентификации на прокси-сервере.


		 ![](../../Property_Image.gif)
		 [Port](../../Interface/INetServiceProxy/INetServiceProxy.Port.htm)


		 Свойство Port определяет
		 порт для подключения к прокси-серверу.


		 ![](../../Property_Image.gif)
		 [User](../../Interface/INetServiceProxy/INetServiceProxy.User.htm)


		 Свойство User определяет
