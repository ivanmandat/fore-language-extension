# CurlNetworkCredential.Create


## Синтаксис


Create(UserName: String; Password: String; Domain: String)


## Параметры


UserName. Имя пользователя;


Password. Пароль пользователя;


Domain. Домен.


## Описание


Конструктор Create создает объект
 для поддержки учётных данных в схемах аутентификации на основе пароля,
 таких как Basic, Digest, NTLM и Kerberos.
