# CurlAttachment


Сборка: Net;


## Описание


Класс CurlAttachment реализует
 вложение сообщения электронной почты.


## Комментарии


Класс представляет собой кроссплатформенную реализацию интерфейса [INetAttachment](../../Interface/INetAttachment/INetAttachment.htm).
 Макросы, реализованные с использованием класса CurlAttachment,
 могут быть запущены на выполнение при работе в операционных системах семейства
 Linux.
