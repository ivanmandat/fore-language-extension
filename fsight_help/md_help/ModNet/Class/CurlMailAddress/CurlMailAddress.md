# CurlMailAddress


Сборка: Net;


## Описание


Класс CurlMailAddress реализует
 адрес электронной почты, на который необходимо отправить сообщение, либо
 адрес отправителя.


## Комментарии


Класс представляет собой кроссплатформенную реализацию интерфейса [INetMailAddress](../../Interface/INetMailAddress/INetMailAddress.htm).
 Макросы, реализованные с использованием класса CurlMailAddress,
 могут быть запущены на выполнение при работе в операционных системах семейства
 Linux.
