# CurlMailMessage


Сборка: Net;


## Описание


Класс CurlMailMessage реализует
 экземпляр сообщения электронной почты.


## Комментарии


Класс представляет собой кроссплатформенную реализацию интерфейса [INetMailMessage](../../Interface/INetMailMessage/INetMailMessage.htm).
 Макросы, реализованные с использованием класса CurlMailMessage,
 могут быть запущены на выполнение при работе в операционных системах семейства
 Linux.
