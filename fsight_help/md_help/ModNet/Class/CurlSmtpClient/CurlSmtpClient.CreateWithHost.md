# CurlSmtpClient.CreateWithHost


## Синтаксис


CreateWithHost(Host: String);


## Параметры


Host. Наименование или IP-адрес
 почтового сервера.


## Описание


Конструктор CreateWithHost создает
 объект, позволяющий отправлять электронные почтовые сообщения.
