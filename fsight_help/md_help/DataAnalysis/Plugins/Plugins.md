# Плагины


Плагин - это независимый модуль, подключаемый для расширения возможностей
 продукта «Форсайт. Аналитическая платформа»:


	- дополнительная визуализация данных и решение пользовательских
	 задач. Например, в инструменте «Отчёты»
	 можно построить аналитическую панель.


Используются следующие типы плагинов:


		- [Преднастроенные
		 плагины](#plugin). Доступны в инструменте «[Аналитические
		 панели](UIAdhoc.chm::/UiAdhoc_Purpose.htm)»;


		- [Пользовательские
		 плагины](#custom_plugin). Доступны в инструментах «[Аналитические
		 панели](UIAdhoc.chm::/UiAdhoc_Purpose.htm)», «[Отчёты](UIReport.chm::/UiReport_purpose.htm)»,
		 «[Интерактивные формы ввода данных](DataEntryForms.chm::/DataEntryForms_Purpose.htm)»;


	- подключение пользовательских шаблонов
	 диаграмм для [гистограммы](UiDiagrams.chm::/Type_diagrams/UiDiagrams_Bar.htm)
	 и [круговой
	 диаграммы](UiDiagrams.chm::/Type_diagrams/UiDiagrams_round.htm).
	 Для подключения пользовательских шаблонов диаграмм обратитесь к разделу
	 «[Подключение
	 пользовательских шаблонов диаграмм](Custom/CustomDiagram.htm)»;


	- подключение внешних картографических
	 сервисов к [карте](UiMaps.chm::/Map.htm): Яндекс, Bing,
	 ArcGis, OpenStreetMap, Google.
	 Для подключения внешних картографических сервисов к карте обратитесь
	 к разделу «[Подключение
	 внешних картографических сервисов к карте](Connecting_External_Services.htm)».


Примечание.
 Использование плагинов доступно только в веб-приложении и [конструкторе
 бизнес-приложений](Constructor.chm::/Intro/Designer_business_applications.htm).


## Преднастроенные плагины


Преднастроенные плагины, доступные для использования в инструменте «[Аналитические
 панели](UIAdhoc.chm::/UiAdhoc_Purpose.htm)» без дополнительной настройки подключения:


	- [Sankey-диаграмма](UiAdhoc.chm::/Blocks/Plugins/Sankey_Chart.htm).
	 Отображает данные в виде потоков и их количественные величины в пропорциональном
	 соотношении друг к другу;


	- [Диаграмма Ганта](UiAdhoc.chm::/Blocks/Plugins/Gantt_Chart.htm). Отображает данные в виде гистограммы,
	 которая позволяет выполнять календарно-сетевое планирование;


	- [Индикатор](UiAdhoc.chm::/Blocks/Plugins/Indicator.htm).
	 Отображает данные в виде отдельных ключевых показателей.


При необходимости для расширения функциональности преднастроенных плагинов
 измените исходные файлы, содержащиеся в папке Plugins/Dashboard, которая
 расположена в папке установки веб-приложения.


## Пользовательские плагины


Для вставки пользовательского плагина в инструментах:


	- [Создайте плагин](Create_Plugin.htm).


Важно.
 Для преднастроенных плагинов зарезервированы идентификаторы: PP.Ui.Dashboard.Sankey
 - «Sankey-диаграмма»; PP.Ui.Dashboard.Gantt
 - «Диаграмма Ганта»; PP.Ui.Dashboard.Indicator
 - «Индикатор». При создании пользовательского
 плагина используйте идентификатор, отличный от зарезервированных.


	- Подключите плагин:


		- [к аналитической
		 панели](Connecting_Plugins_Dashboards.htm);


		- [к регламентному
		 отчёту](Connecting_Plugins_Report.htm);


		- к форме ввода.


Примечание.
 Подключение плагина к форме ввода выполняется аналогично [подключению
 плагина к регламентному отчёту](Connecting_Plugins_Report.htm).


После выполнения действий созданный плагин будет вставлен в соответствующий
 инструмент.
