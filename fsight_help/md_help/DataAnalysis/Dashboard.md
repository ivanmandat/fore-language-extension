# Построение информационных панелей


Для построения информационных панелей предназначен инструмент «Информационная панель»:


![](data_analysis_view.png)


Инструмент лёгкой бизнес-аналитики класса Self-Service BI «Информационная панель»
 предназначен для решения задач бизнес-аналитики, исследования и поиска
 данных, визуализации данных с использованием средств интерактивной деловой
 графики.


Ключевые возможности:


	- извлечение и обработка больших объёмов данных из разнородных
	 источников;


	- формирование визуального представления данных с использованием
	 широкого перечня визуализаторов;


	- создание интерактивных панелей связанных визуализаторов данных.


Для начала работы
 с инструментом смотрите статью «[Начало работы
 с инструментом «Информационная панель](uidashboard.chm::/InformationPanel/Starting.htm)».
