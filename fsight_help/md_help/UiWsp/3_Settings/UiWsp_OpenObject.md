# Открытие объекта рабочего пространства


Для открытия объекта рабочего пространства:


	- выполните команду «Открыть»
	 в контекстном меню объекта;


	- щелкните по объекту два раза кнопкой мыши.


Открытие объекта рабочего пространства позволяет открыть на просмотр
 или редактирование отдельный объект репозитория.


Если на рабочее пространство был добавлен документ, в котором
 сохранён PDF-файл или URL-ссылка, то они будут открыты в новой вкладке
 браузера. Для документов, содержащих другие типы файлов, будет инициализирован
 процесс сохранения файла на диск.
