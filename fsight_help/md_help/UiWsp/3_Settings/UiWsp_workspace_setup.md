# Настройка рабочей области


Рабочая область - контейнер, в котором расположены объекты рабочего
 пространства. Рабочая область является удобным механизмом для:


	- визуального размещения объектов репозитория на едином рабочем
	 поле;


	- организация доступа к функциям и свойствам объектов репозитория;


	- отображение связей между объектами и структуры их
	 зависимостей с использованием фигур и линий.


При работе с объектами на рабочей области доступны следующие режимы:


[![](../Opened.gif)![](../Closed.gif)Режим курсора/указателя](javascript:TextPopup(this))


	Режим курсора/указателя позволяет выделять и перемещать объекты
	 в рабочей области во время редактирования рабочего пространства.


	Режим курсора/указателя используется по умолчанию.


	Для выбора данного режима:


		- в веб-приложении если выбран режим ![](../2_WorkSpace/Objects/b_hand.png)
		 «Рука», то используйте
		 раскрывающийся список для выбора режима  ![](../2_WorkSpace/Objects/b_cursor.png)
		 «Курсор» на [панели
		 инструментов](../2_WorkSpace/UiWsp_WorkSpace_Starting.htm);


		- в настольном приложении:


			- нажмите кнопку ![](../2_WorkSpace/Objects/button_cursor.gif) «Указатель» на
			 вкладке «Главная»
			 ленты инструментов;


			- нажмите сочетание клавиш
			 ALT+1.


[![](../Opened.gif)![](../Closed.gif)Режим
 руки](javascript:TextPopup(this))


	Режим руки предоставляет возможность прокрутки содержимого рабочей
	 области.


	Для выбора режима:


		- выполните одно из действий:


			- в веб-приложении выберите
			 режим ![](../2_WorkSpace/Objects/b_hand.png) «Рука» в
			 раскрывающемся меню режимов на панели инструментов;


			- в настольном приложении
			 нажмите кнопку ![](../2_WorkSpace/Objects/button_hand.gif)
			 «Рука» на вкладке
			 «Главная» ленты инструментов;


		- нажмите сочетание клавиш ALT+6 (доступно только в настольном
		 приложении);


		- зажмите клавишу ПРОБЕЛ при использовании другого режима.


## Перемещение по рабочей области


Для перемещения по рабочей области выполните одно из действий:


	- выберите [режим руки](UiWsp_workspace_setup.htm#hand)
	 на [панели
	 инструментов](../../CalculationAlgorithm/Web/Work/Beginning_of_work.htm) и зажмите кнопку мыши;


	- зажмите клавишу ПРОБЕЛ и кнопку мыши;


	- используйте стрелки на клавиатуре;


	- прокрутите колесо мыши для перемещения рабочей области по вертикали.
	 Для перемещения рабочей области по горизонтали дополнительно зажмите
	 клавишу SHIFT;


	- в веб-приложении используйте миникарту. Для отображения миникарты
	 нажмите кнопку ![](../2_WorkSpace/Objects/b_mini_map.png) «Включить
	 миникарту» на панели настроек отображения рабочей области:


![](../2_WorkSpace/Objects/minimap_example.png)


## Масштабирование рабочей области


Для масштабирования рабочей области:


	- прокрутите колесо мыши с зажатой клавишей CTRL;


	- в веб-приложении используйте панель настроек отображения рабочей
	 области:


![](../2_WorkSpace/Objects/zoom_panel.png)


На панели доступны следующие операции:


		- ![](../2_WorkSpace/Objects/b_adjust_for_all.png) «Подогнать
		 по всем объектам». Изменение масштаба рабочей области по
		 всем объектам;


		- ![](../2_WorkSpace/Objects/b_adjust_for_one.png) «Подогнать
		 по объекту». Изменение масштаба рабочей области по выделенному
		 объекту;


Примечание.
 Если ни один объект на рабочей области не выделен, то кнопка ![](../2_WorkSpace/Objects/b_adjust_for_one.png) «Подогнать по
 объекту» будет недоступна.


		- ![](../2_WorkSpace/Objects/b_scale_plus.png) «Увеличить
		 масштаб». Увеличение масштаба рабочей области на 25%;


		- ![](../2_WorkSpace/Objects/b_scale_minus.png) «Уменьшить
		 масштаб». Уменьшение масштаба рабочей области на 25%.


В рабочей области может быть установлен масштаб
 отображения в диапазоне [1%, 400%]. Для установки стопроцентного масштаба
 нажмите кнопку с отображением текущего масштаба;


	- в настольном приложении выполните одно из действий:


		- выберите из списка или введите значение в поле ![](../2_WorkSpace/MainWindow/button_zoom.gif)
		 главного меню ленты инструментов;


		- нажмите клавишу «+» для увеличения масштаба или клавишу
		 «-» для уменьшения масштаба;


		- нажмите кнопку ![](UiWspPreview_Scale.png) на ленте инструментов
		 для возврата масштаба 100%;


		- нажмите кнопку ![](../2_WorkSpace/MainWindow/UiWsp_ScaleUp.png) на ленте
		 инструментов для приведения размера рабочего пространства под
		 размер окна.
