# Экспорт рабочего пространства


Доступно только в настольном приложении.


Для экспорта рабочего пространства выполните команду главного меню «Документ > Экспорт». Будет открыт
 стандартный диалог сохранения файла. Определите имя, тип и месторасположение
 файла.


Рабочее пространство можно экспортировать в файлы следующих форматов:


	- BMP (*.bmp);


	- PNG (*.png);


	- JPG (*.jpg);


	- GIF (*.gif);


	- TIFF (*.tiff);


	- PDF (*.pdf).


Особенности экспорта:


	- при экспорте рабочего пространства в результирующем файле может
	 быть отображен штамп № 1 в соответствии с Инструкцией № 0126-87, если
	 в качестве политики безопасности используется [мандатный
	 доступ](Admin.chm::/04_SecurityPolicy/Admin_MandatAccess.htm) и для уровня безопасности данного объекта включена [маркировка
	 при экспорте и печати](Admin.chm::/03_Admin/Admin_MandatAccessParam.htm);


	- размеры экспортируемой области будут соответствовать размерам,
	 заданным в [параметрах](UiWsp_WorkSpace_SettingsView.htm)
	 рабочего пространства.
