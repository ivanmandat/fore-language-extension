# Работа с готовым рабочим пространством


Для работы с готовым рабочим пространством выполните шаги:


	- [Измените масштаб рабочего
	 пространства](UiWsp_workspace_setup.htm).


	- Выполните [предварительный
	 просмотр](../2_WorkSpace/UiWsp_Preview.htm) и [настройте параметры
	 страницы](UiWsp_Settings_PageProp.htm) для печати. Доступно только в настольном приложении.


	- [Экспортируйте рабочее пространство](UiWsp_Export.htm). Доступно
	 только в настольном приложении.


	- [Создайте копию рабочего
	 пространства](../2_WorkSpace/UiWsp_CopySave.htm).


	- [Откройте объект рабочего пространства](UiWsp_OpenObject.htm).


После работы с готовым рабочим пространством в настольном приложении
 выполните команду «Документ > Закрыть»
 главного меню или нажмите кнопку ![](../2_WorkSpace/Objects/close_icon.png) «Закрыть» в правом верхнем углу окна.
