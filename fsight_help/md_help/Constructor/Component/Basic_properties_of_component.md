# Объединение функциональности в
 компоненты


Конструктор компонентов предназначен для группировки функциональности,
 используемой в различных бизнес-приложениях, и обеспечения контроля целостности
 при обновлении версий используемых компонентов «Форсайт. Аналитическая платформа».


Работа с компонентом выполняется в настольном приложении продукта «Форсайт. Аналитическая платформа».


Для создания и настройки компонента выполните следующие шаги:


	- Создайте объект «Компонент».


Примечание.
 Для создания пользовательского компонента в репозитории должно быть [установлено
 расширение](setup.chm::/Extensions/install_extensions_in_repository.htm) «Конструктор
 бизнес-приложений» продукта «Форсайт. Аналитическая платформа».


	- [Переместите в компонент](Filling_component.htm)
	 ранее созданные объекты.


	- [Настройте зависимости](Configure_dependencies.htm)
	 компонента от других компонентов продукта «Форсайт. Аналитическая платформа»,
	 необходимых для его стабильной работы.


	- Выберите, какие объекты компонента необходимо сделать [публичными](Setting_public_objects.htm)
	 для использования в различных бизнес-приложениях.


	- Выберите используемые [классы
	 объектов](Adding_user_class.htm), которые относятся к данному компоненту и будут добавлены
	 в репозиторий после установки компонента.


	- [Сформируйте
	 пакет обновлений](Formation_installation_updates_components.htm#forming_updates_components), указав наименование компонента и версию обновления.


	- [Установите
	 сформированный пакет обновлений](Formation_installation_updates_components.htm#install_updates_components) с версией компонента в нужный
	 репозиторий.
