# Настройка базовых свойств


Настройка базовых свойств бизнес-приложения выполняется на странице
 «Базовые настройки» в [мастере
 создания бизнес-приложения](General_Principles_Web.htm).


[![](../Opened.gif)![](../Closed.gif)Для открытия
 мастера](javascript:TextPopup(this))


	Создайте или откройте готовое бизнес-приложение на редактирование.


	Для создания нового бизнес-приложения в [навигаторе
	 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm) нажмите кнопку ![](../Component/b_add.png) «Создать» в главном меню и выберите
	 тип объекта на боковой панели «[Новый объект](UiNav.chm::/02_Navigator/General_Principles_of_Work.htm#add_object)».
	 При создании бизнес-приложения доступна только одна страница мастера
	 - «Базовые настройки».


	Для открытия готового бизнес-приложения на редактирование в [навигаторе
	 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm) выделите бизнес-приложение и выполните
	 одно из действий:


	- выполните команду «Редактировать»
	 в контекстном меню бизнес-приложения;


	- выполните команду «Редактировать
	 в новой вкладке» в контекстном меню бизнес-приложения;


	- выполните команду «Редактировать
	 в новом окне» в контекстном меню бизнес-приложения;


	- нажмите клавишу F4.


	При открытии бизнес-приложения на редактирование доступны все страницы
	 мастера: «Базовые настройки»,
	 «[Разметка
	 и оформление](Layout_and_design_web.htm)», «[Структура
	 навигации](Setting_Navigation_Structure.htm)», «[Дополнительные
	 настройки](Select_Starting_Object_and_Event_Handler.htm)».


![](basic_settings.png)


Задайте параметры:


	- Наименование. Введите
	 наименование бизнес-приложения, которое будет отображаться в навигаторе
	 объектов;


	- Идентификатор. Введите
	 идентификатор бизнес-приложения. Идентификатор представляет собой
	 последовательность букв латинского алфавита, цифр и знаков подчёркивания,
	 начинающаяся с буквы. Идентификатор должен быть уникальным в пределах
	 репозитория;


	- Создать базу данных на основании
	 существующей. Создание базы данных выполняется только при создании
	 бизнес-приложения. База данных используется в качестве дочернего объекта
	 бизнес-приложения для создания реляционных объектов. Установите флажок
	 и выберите в раскрывающемся списке базу данных, на основании настроек
	 подключения которой будет создана база данных бизнес-приложения. Если
	 флажок снят, то будет создана новая база данных с параметрами подключения,
	 которые заданы в [глобальных
	 переменных](UiNav.chm::/02_Navigator/UiNav_GlobalVariables.htm) репозитория. Подключение к базе данных
	 осуществляется автоматически с использованием учётных данных, указанных
	 при входе в репозиторий.


Для быстрого выбора объекта в поле для поиска
 введите его название/идентификатор/ключ, в зависимости от настроек отображения.
 Поиск будет выполняться автоматически по мере ввода текста. Список будет
 содержать объекты, наименования/идентификаторы/ключи которых содержат
 вводимый текст.


Для настройки отображения объектов репозитория
 в списке нажмите кнопку ![](b_view_settings.png) «Отображение
 объекта» и выберите в раскрывающемся меню вариант отображения:


		- Наименование. Объекты
		 отображаются под своими наименованиями. Вариант по умолчанию;


		- Идентификатор. Объекты
		 отображаются под своими идентификаторами;


		- Ключ. Объекты отображаются
		 под своими ключами.


Выбрать можно несколько вариантов. Идентификатор
 и ключ будут указаны в скобках.


Для сброса отметки выбранной базы данных
 нажмите кнопку ![](b_delete_web.png) «Очистить»;


Для сохранения заданных настроек нажмите кнопку «Сохранить».


После настройки базовых свойств создайте или измените [внутреннюю
 структуру бизнес-приложения](Create_Internal_Structure_Business_Application.htm) в [навигаторе
 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm). Если бизнес-приложение открыто на редактирование,
 то перейдите к [настройке разметки
 и оформления](Layout_and_design_web.htm).
