# Визуальное конструирование бизнес-приложений


Для визуального конструирования бизнес-приложений используйте расширение
 «Конструктор бизнес-приложений»
 продукта «Форсайт. Аналитическая платформа»:


![](Application%20interface_main_2.png)


Расширение «Конструктор бизнес-приложений»
 продукта «Форсайт. Аналитическая платформа»
 предназначено для визуального конструирования бизнес-приложений по типовому
 набору шагов.


Ключевые возможности:


	- создание комплексных многоуровневых приложений;


	- типовая последовательность шагов для поэтапного конструирования
	 программных продуктов;


	- поддержка целостности бизнес-приложения;


	- формирование пакетов обновлений;


	- гарантированная доставка приложений между различными системами.


Расширение «Конструктор бизнес-приложений»
 реализует возможность работы:


	- в веб-приложении с объектом «Бизнес-приложение»;


	- в настольном приложении с объектами «Бизнес-приложение»
	 и «Компонент».


Бизнес-приложение предназначено для разработки веб-приложений с использованием
 стандартных инструментов и [расширений
 ](setup.chm::/Extensions/install_or_update_extensions.htm)
 продукта «Форсайт. Аналитическая платформа»,
 а также обеспечения контроля целостности при обновлении версий бизнес-приложений.


Компонент предназначен для объединения функциональности, используемой
 в различных бизнес-приложениях, и обеспечения контроля целостности при
 обновлении версий расширений. Компонент может содержать как стандартные
 инструменты и расширения продукта «Форсайт. Аналитическая платформа», необходимые
 для разработки бизнес-приложений, так и пользовательские объекты.


Для начала работы смотрите статью
 «[Начало работы с расширением «Конструктор
 бизнес-приложений](Component_interface.htm)».


Для построения бизнес-приложения
 в веб-приложении смотрите статью «[Построение
 бизнес-приложения в веб-приложении](../Web/General_Principles_Web.htm)».


Для построения бизнес-приложения
 в настольном приложении смотрите статью «[Построение
 бизнес-приложения в настольном приложении](../Desktop/General_Principles.htm)».


Для работы с готовым бизнес-приложением
 смотрите статью «[Работа
 с готовым бизнес-приложением](../Work/Work_business_application.htm)».


Для создания и работы с компонентом
 в настольном приложении смотрите статью «[Объединение
 функциональности в компоненты](../Component/Basic_properties_of_component.htm)».
