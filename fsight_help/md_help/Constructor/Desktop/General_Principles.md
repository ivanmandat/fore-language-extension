# Построение бизнес-приложения в
 настольном приложении


Построение бизнес-приложения выполняется с помощью конструктора бизнес-приложения.
 Для открытия конструктора [создайте](../Intro/Component_interface.htm#create)
 новое или [откройте](../Intro/Component_interface.htm#open)
 готовое бизнес-приложение на редактирование:


![](Common_parameters.png)


Для построения бизнес-приложения:


	- [Настройте общие параметры](Common_parameters.htm).


	- [Создайте
	 внутреннюю структуру](../Web/Create_Internal_Structure_Business_Application.htm).


	- [Настройте разметку и оформление](Layout_and_design.htm).


	- [Настройте структуру
	 меню](Setting_web_application_structure.htm).


	- [Измените зависимости объектов
	 бизнес-приложения](Edit_Dependencies.htm) от [расширений](setup.chm::/Extensions/install_or_update_extensions.htm)
	 продукта «Форсайт. Аналитическая платформа».


	- [Сформируйте пакет
	 обновлений](Formation_Installation_Updates.htm) для переноса бизнес-приложения в другой репозиторий.
