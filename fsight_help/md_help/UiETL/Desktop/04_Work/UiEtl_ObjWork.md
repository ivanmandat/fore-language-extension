# Настройка рабочей области


Объекты задачи ETL служат для
 отображения схем загрузки и трансформации данных, механизмов, схем и структур
 зависимости объектов в рабочей области.


Рабочая область - контейнер,
 в котором расположены объекты задачи ETL. Рабочая область является удобным
 механизмом для:


	- визуального размещения объектов задачи на едином рабочем поле;


	- организации доступа к свойствам объектов задачи;


	- отображения связей и зависимостей между объектами.

## Операции с рабочей областью


	- Масштабирование. Для
	 выполнения масштабирования рабочей области:


		- укажите процент масштабирования
		 в поле ![](../02_mainWindow/button_zoom.gif) панели
		 инструментов;


		- зажав клавишу CTRL, прокрутите
		 колесико мыши;


		- выполните команду «Вид»
		 в главном меню
		 и выберите вариант масштабирования;


		- выполните команду «Масштаб»
		 в контекстном меню рабочей области и выберите вариант масштабирования;


Примечание.
 При работе в веб-приложении недоступно масштабирование прокруткой колеса
 мыши.


	- Отображение сетки. Для
	 отображения сетки на рабочей области выполните команду «Вид
	 > Отображать сетку» в главном
	 меню.


	- Отображение точек соединения.
	 Для отображения точек соединения выполните команду «Вид
	 > Отображать точки соединения» в главном
	 меню.


	- Отображение пересечений.
	 Для отображения пересечений линий выполните команду «Вид
	 > Отображать пересечения» в главном
	 меню.


## Операции с объектами задачи ETL


[![](../../Opened.gif)![](../../Closed.gif)Создание объекта
 задачи ETL](javascript:TextPopup(this))


	Для создания объекта в рабочей области:


		- выполните команду «Вставка»
		 в главном меню
		 и выберите необходимый объект;


		- дважды щёлкните кнопкой мыши по необходимому объекту на
		 панели объектов;


		- перетащите необходимый объект с панели
		 объектов в рабочую область, используя механизм Drag&Drop.


[![](../../Opened.gif)![](../../Closed.gif)Настройка оформления
 объекта](javascript:TextPopup(this))


	Для настройки оформления выделенного объекта:


		- выполните команду «Формат»
		 в контекстном меню объекта;


		- выполните команду «Формат
		 > Редактировать» в
		 главном меню.


	Будет открыт стандартный [диалог форматирования](UiNav.chm::/GUI/Format.htm).


[![](../../Opened.gif)![](../../Closed.gif)Режимы
 работы с объектами](javascript:TextPopup(this))


	При работе с объектами задачи ETL доступны различные режимы.


	Для быстрой и удобной прокрутки содержимого рабочей области используйте:


		- колёсико мыши;


		- режим руки.


### Режим указателя


	Режим указателя позволяет выделять и перемещать объекты во время
	 редактирования задачи ETL.


	Для выбора данного режима:


		- выполните команду «Вид
		 > Режим указателя» в главном
		 меню;


		- нажмите кнопку ![](../02_mainWindow/button_cursor.gif)
		 на панели инструментов;


		- нажмите сочетание клавиш ALT+1.


	Для выделения группы объектов зажмите клавишу SHIFT и щёлкните необходимые
	 объекты.


	В режиме указателя доступно сгибание линий. Для добавления сгиба
	 линии зажмите клавишу CTRL и кнопкой мыши определите в каком месте
	 линии будет изгиб.


### Режим создания линий


	Данный режим позволяет создавать связи между объектами в виде линий.
	 Для выбора режима:


		- выполните команду «Вид
		 > Режим создания линий» в главном
		 меню;


		- нажмите кнопку ![](../02_mainWindow/button_line.gif)
		 на панели инструментов;


		- нажмите сочетание клавиш ALT+2.


	Для создания произвольной линии перемещайте курсор мыши по рабочей
	 области с зажатыми клавишей CTRL и кнопкой мыши.


### Режим создания связей


	Данный режим позволяет создавать связи между объектами в виде стрелок.


	Для выбора режима:


		- выполните команду «Вид
		 > Режим создания связей» в главном
		 меню;


		- нажмите кнопку ![](../02_mainWindow/UiWsp_ToolBar_CreateLink.gif)
		 на панели инструментов;


		- нажмите сочетание клавиш ALT+3.


	Для создания произвольной линии перемещайте курсор мыши по рабочей
	 области с зажатыми клавишей CTRL и кнопкой мыши.


### Режим создания соединительных линий


	Данный режим позволяет создавать связи между объектами в виде соединительных
	 линий и соединять точки объектов.


	Для выбора режима:


		- выполните команду «Вид
		 > Режим создания соединительных линий» в главном
		 меню;


		- нажмите кнопку ![](../02_mainWindow/UiWsp_ToolBar%20Double%20line.gif)
		 на панели инструментов;


		- нажмите сочетание клавиш ALT+4.


	Для создания произвольной линии перемещайте курсор мыши по рабочей
	 области с зажатыми клавишей CTRL и кнопкой мыши.


### Режим создания кривых линий


	Данный режим позволяет создавать связи между объектами в виде кривых
	 линий.


	Для выбора режима:


		- выполните команду «Вид
		 > Режим создания кривых линий» в главном
		 меню;


		- нажмите кнопку ![](../02_mainWindow/ToolBar_Curve.gif)
		 на панели инструментов;


		- нажмите сочетание клавиш ALT+5.


	Для создания произвольной линии перемещайте курсор мыши по рабочей
	 области с зажатыми клавишей CTRL и кнопкой мыши.


### Режим руки


	Данный режим предоставляет возможность прокрутки содержимого рабочей
	 области.


	Для выбора режима:


		- выполните команду «Вид
		 > Режим руки» в главном
		 меню;


		- нажмите кнопку ![](../02_mainWindow/button_hand.gif)
		 на панели инструментов;


		- нажмите сочетание клавиш ALT+6;


		- зажмите клавишу ПРОБЕЛ при использовании другого режима.


### Режим редактирования точек соединения


	Данный режим позволяет редактировать точки соединения.


	Для выбора режима:


		- выполните команду «Вид
		 > Режим редактирования точек соединения» в главном
		 меню;


		- нажмите кнопку ![](../02_mainWindow/UiWsp_ToolBar_add.gif)
		 на панели инструментов;


		- нажмите сочетание клавиш ALT+7.


	Соединительные точки можно перемещать по контуру объекта. При перемещении
	 с зажатой клавишей CTRL соединительные точки копируются. При этом
	 на контуре объекта может размещаться бесконечное количество точек
	 соединения.


[![](../../Opened.gif)![](../../Closed.gif)Работа с буфером
 обмена](javascript:TextPopup(this))


	Используя буфер обмена можно перемещать и копировать объекты рабочей
	 области.


	Примечание.
	 Работа с буфером обмена ведется с учётом прав доступа. Доступны только
	 те операции, на выполнение которых пользователь имеет права.


	Для копирования выбранных объектов в буфер обмена:


		- выполните команду «Правка
		 > Копировать» в главном
		 меню;


		- выполните команду «Копировать» в
		 контекстном меню объекта;


		- нажмите кнопку ![](../02_mainWindow/UiWsp_ToolBar_Copy.gif)
		 на панели инструментов;


		- нажмите сочетание клавиш CTRL+C или CTRL+INSERT.


	Примечание.
	 Операция недоступна для объекта «[Связь](../03_ETLObjects/05_Links/uietl_links_create.htm)».


	Для перемещения выбранных объектов в буфер обмена:


		- выполните команду «Правка
		 > Вырезать» в главном
		 меню;


		- выполните команду «Вырезать»
		 в контекстном меню объекта;


		- нажмите кнопку ![](../02_mainWindow/UiWsp_ToolBar_Cut.gif)
		 на панели инструментов;


		- нажмите сочетание клавиш CTRL+X или SHIFT+DELETE.


	Примечание.
	 Операция недоступна для объекта «[Связь](../03_ETLObjects/05_Links/uietl_links_create.htm)». При перемещении связи между
	 объектами удаляются.


	Для вставки скопированного/вырезанного объекта из буфера обмена:


		- выполните команду «Правка
		 > Вставить» в главном
		 меню;


		- выполните команду «Вставить»
		 в контекстном меню объекта;


		- нажмите кнопку ![](../02_mainWindow/UiWsp_ToolBar_Paste.gif)
		 на панели инструментов;


		- нажмите сочетание клавиш CTRL+V или SHIFT+INSERT.


	Примечание.
	 Вставка доступна, если в буфере обмена содержатся объекты.

	Каждый объект рабочего пространства может содержать текст.


	Для коннекторов, преобразователей и объектов из группы «Другие»
	 в качестве текста отображается наименование объекта.


	Для [графических
	 примитивов](UiWsp.chm::/2_WorkSpace/Objects/UiWsp_Objects.htm#grafics) или линий, соединяющих примитивы, может
	 быть задан произвольный текст. Для [ввода
	 и редактирования текста](UiWsp.chm::/2_WorkSpace/Objects_Work/UiWsp_ObjectsWork_AddEditText.htm) дважды щёлкните по выбранному
	 примитиву. Будет активирован редактора текста. Для завершения редактирования
	 текста щёлкните за пределами редактора. Текст, созданный для линий,
	 всегда располагается горизонтально. Поворот объекта, содержащего текст,
	 происходит вместе с текстом. Для изменения формата текста выделите
	 его и выполните команду контекстного меню «Формат». Будет открыт [диалог форматирования](UiNav.chm::/GUI/Format.htm).
