# Оформление объектов


По умолчанию все объекты рабочей области имеют одинаковое оформление.
 Для визуального выделения объектов можно менять стиль их оформления.


Параметры оформления каждого объекта настраиваются индивидуально или
 создается [стиль
 оформления](UiNav.chm::/GUI/UiETL_Objects_Styles.htm)
 для группы объектов. Для настройки стилей выполните команду «Формат
 > Стили…» в главном меню. Для применения стиля к объектам выделите
 их и выберите требуемый стиль в раскрывающемся списке «Стиль»,
 расположенном на панели
 инструментов.


Для настройки оформления выделенного объекта:


	- выполните команду «Формат»
	 в контекстном меню объекта;


	- выполните команду «Формат
	 > Редактировать» в
	 главном меню.


Будет открыт диалог «Формат объекта»,
 содержащий вкладки:


	- [Шрифт](UiNav.chm::/GUI/format/UiReport_Table_Attribute_Type.htm);


	- [Заливка](UiNav.chm::/GUI/Format/UiEtl_ObjProp_Fill.htm);


	- [Тень](UiNav.chm::/GUI/Format/UiEtl_ObjProp_Fill.htm);


	- [Размер](UiNav.chm::/GUI/Format/uietl_objprop_size.htm);


	- [Линии](UiNav.chm::/GUI/Format/UiEtl_ObjProp_Lines.htm);


	- [Текстовый
	 блок](UiNav.chm::/GUI/Format/Uietl_ObjProp_Text.htm);


	- [Блок
	 изображения](UiNav.chm::/GUI/Format/UiEtl_ObjProp_Image.htm).
