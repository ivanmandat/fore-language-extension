# Шаблоны пользовательских объектов


Кроме стандартных [источников](../03_ETLObjects/02_Inputs/UiEtl_Inputs.htm)/[приёмников](../03_ETLObjects/03_Outputs/UiEtl_Outputs.htm)/[преобразователей](../03_ETLObjects/04_Transformers/UiEtl_Trfs.htm),
 которые могут использоваться для обработки данных в задаче ETL, могут
 быть реализованы пользовательские объекты, выполняющие аналогичные функции
 по алгоритму прикладного разработчика. Для создания таких объектов необходимо
 предварительно создать шаблон,
 в котором указывается макрос на языке Fore, реализующий соответствующий
 алгоритм обработки данных. В отличии от [источника](../03_ETLObjects/02_Inputs/Uietl_Inputs_User.htm)/[приёмника](../03_ETLObjects/03_Outputs/User/UiEtl_Outputs_User.htm)/[алгоритма](../03_ETLObjects/04_Transformers/09_algorithm/UiEtl_Trfs_Algorithm.htm)
 пользователя, которые настраиваются и используются в конкретной задаче
 ETL, шаблон пользовательского объекта может использоваться в любых задачах
 ETL репозитория. Создание шаблонов и объектов на базе шаблонов осуществляется
 в настольном приложении.


Важно. Изменение
 настроек шаблона повлияет на работу всех задач ETL, где используются объекты
 на базе этого шаблона.


## Работа с шаблонами


[![](../../Opened.gif)![](../../Closed.gif)Создание шаблонов](javascript:TextPopup(this))


	Для создания нового шаблона выполните команду главного меню «Вставка > Источник/Приёмник/Преобразователь/Другие
	 > Создать новый шаблон».


	Будет открыт диалог для настройки нового шаблона соответствующего
	 объекта:


		- [Шаблон источника](Templates/Provider_Template.htm);


		- [Шаблон приёмника](Templates/Consumer_Template.htm);


		- [Шаблон преобразователя](Templates/Transform_Template.htm);


		- [Шаблон процедуры](Templates/Other_Template.htm).


[![](../../Opened.gif)![](../../Closed.gif)Редактирование
 шаблонов](javascript:TextPopup(this))


	Для редактирования существующих шаблонов:


		- В области панели объектов переключитесь на ту панель, для
		 которой создавался шаблон.


		- Найдите необходимый шаблон и выполните для него команду
		 контекстного меню «Редактировать».


	Будет открыт соответствующий диалог для изменения параметров шаблона:


		- [Шаблон источника](Templates/Provider_Template.htm);


		- [Шаблон приёмника](Templates/Consumer_Template.htm);


		- [Шаблон преобразователя](Templates/Transform_Template.htm);


		- [Шаблон процедуры](Templates/Other_Template.htm).


[![](../../Opened.gif)![](../../Closed.gif)Создание объектов
 на базе шаблона](javascript:TextPopup(this))


	Для создания объекта:


		- Перейдите на вкладку, для которой создавался шаблон.


		- Найдите шаблон в списке и дважды щелкните по нему, либо
		 выполните команду «Вставить на
		 рабочее пространство» в контекстном меню шаблона.


	Будет открыт мастер создания нового источника/приёмника/преобразователя
	 на базе шаблона пользователя. В мастере будут содержаться базовые
	 страницы, соответствующие создаваемому объекту.
