# Построение задачи ETL


Построение задачи ETL осуществляется в настольном приложении. Для построения
 задачи ETL выполните шаги:


	- Добавьте один или несколько [коннекторов
	 к источникам данных](02_Inputs/UiEtl_Inputs.htm).


	- Добавьте один или несколько [коннекторов
	 к приёмникам данных](03_Outputs/UiEtl_Outputs.htm).


	- При необходимости добавьте [преобразователи
	 данных](04_Transformers/UiEtl_Trfs.htm).


	- Настройте [связи](05_Links/uietl_links_create.htm)
	 между созданными объектами.


	- Настройте оформление [объектов](../04_Work/01_ObjFormat/UiEtl_ObjFormat.htm)
	 и [рабочей области](../04_Work/UiEtl_ObjWork.htm) задачи
	 ETL.


	- [Выполните](../05_PropertyETL/UIEtl_WorkVisualisation.htm)
	 задачу ETL.
