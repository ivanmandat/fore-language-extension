# Создание преобразователей данных


Преобразователь данных предназначен
 для трансформации данных, поступающих ему на вход, и, в зависимости от
 типа трансформации, выгружает преобразованные данные в один или несколько
 объектов-приёмников. Входами для преобразователей могут служить как объекты-источники
 данных, так и другие объекты-преобразователи. Аналогично, приёмниками
 данных с выходов преобразователей могут выступать как объекты-приёмники,
 так и другие объекты-преобразователи. Длина цепочки преобразований не
 ограничена. Создание и редактирование преобразователей осуществляется
 в настольном приложении.


Если в задаче используются преобразователи удаление дубликатов, сортировка,
 соединение, то будет игнорироваться [пакетная
 обработка данных](../../04_Work/04_Jurnal/UiEtl_Jurnal.htm).

	Для создания преобразователя данных:


		- дважды щёлкните по выбранному объекту на панели «Преобразователи»;


		- выполните команду «Вставка
		 > Преобразователь» и выберите необходимый преобразователь.


	Будет открыто окно мастера создания объекта-преобразователя. Все
	 мастера создания объектов «Преобразователь»
	 содержат ряд общих
	 страниц, настраиваемых для всех объектов одинаково.


[![](../../../Opened.gif)![](../../../Closed.gif)Редактирование
 преобразователя данных](javascript:TextPopup(this))


	Для редактирования преобразователя данных:


		- выполните команду «Редактировать»
		 в контекстном меню преобразователя;


		- дважды щелкните кнопкой мыши по выбранному преобразователю;


		- нажмите клавишу F4.
