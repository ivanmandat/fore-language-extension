# Создание коннекторов к приёмнику
 данных


Все доступные коннекторы располагаются на вкладке «Приёмники».
 Коннекторы предназначены для подключения к внешним приёмникам или объектам
 репозитория и экспорта в них данных. Каждый коннектор хранит в себе настройки
 подключения к определённому приёмнику и связи с другими объектами задачи
 ETL. Создание и редактирование коннекторов осуществляется в настольном
 приложении.


Доступные коннекторы к приёмникам данных:


![](outputs.png)

[


](Repositoriy/UiEtl_Outputs_Repo.htm)[![](../../../Opened.gif)![](../../../Closed.gif)Создание
 коннектора](javascript:TextPopup(this))


	Для создания коннектора к приёмнику данных выполните одно из следующих
	 действий:


		- дважды щелкните по выбранному объекту на панели «Приёмники»;


		- выполните команду главного меню «Вставка
		 > Приёмник» и выберите необходимый объект.


	Будет открыт мастер создания выбранного коннектора. Все мастера
	 содержат ряд общих страниц, настраиваемых для всех объектов одинаково.


[![](../../../Opened.gif)![](../../../Closed.gif)Редактирование
 коннектора](javascript:TextPopup(this))


	Для редактирования коннектора выполните одно из следующих действий:


		- выполните команду «Редактировать»
		 в контекстном меню;


		- дважды щелкните по выбранному объекту;


		- выделите объект и нажмите клавишу F4.


[![](../../../Opened.gif)![](../../../Closed.gif)Просмотр данных](javascript:TextPopup(this))


	Просмотр данных служит для предварительного просмотра данных источника/приёмника
	 задачи ETL,
	 отобранных с помощью запросов или необходимых параметров.


	Примечание.
	 Данная возможность существует для всех приёмников, кроме приёмников:
	 «[Текст](Text/UiEtl_Outputs_Text.htm)»,
	 «[XML](XML/UiEtl_outputs_XML.htm)»,
	 «[Приёмник
	 пользователя](User/UiEtl_Outputs_User.htm)». Для приёмника «[Репозиторий](../02_Inputs/UiEtl_Inputs_Repo.htm)»
	 просмотр данных возможен только, если приёмником данных является объект
	 репозитория «[Документ](UiNavObj.chm::/UiNavObj_document.htm)».


	Подробное описание возможности приведено в разделе «[Просмотр данных](../Page_browse_data.htm)».
