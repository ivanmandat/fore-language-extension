# Создание объектов из  группы
 «Другие»


Группа объектов «Другие» содержит
 дополнительные объекты, реализация которых не позволяет их однозначно
 отнести к [источникам](../02_Inputs/UiEtl_Inputs.htm), [приёмникам](../03_Outputs/UiEtl_Outputs.htm)
 или [преобразователям](../04_Transformers/UiEtl_Trfs.htm).
 Данные объекты предназначены для решения различных специфических задач,
 при этом они могут использоваться как сами по себе, так и в цепочках наряду
 с другим объектами. Создание и редактирование объектов группы «Другие»
 осуществляется в настольном приложении.


Если в задаче используются процедуры или процедуры на основе шаблона,
 то будет игнорироваться [пакетная
 обработка данных](../../04_Work/04_Jurnal/UiEtl_Jurnal.htm).

	Для создания объекта группы «Другие»:


		- дважды щелкните по выбранному объекту на панели «Другие»;


		- выполните команду «Вставка
		 > Другие» в главном меню и выберите необходимый объект.


	Будет открыт мастер создания выбранного объекта. Все мастера создания
	 объектов группы «Другие» содержат
	 ряд общих
	 страниц, настраиваемых для всех объектов одинаково.


[![](../../../Opened.gif)![](../../../Closed.gif)Редактирование
 объекта](javascript:TextPopup(this))


	Для редактирования объекта группы «Другие»:


		- выполните команду «Редактировать»
		 в контекстном меню объекта;


		- дважды щелкните по выбранному объекту;


		- нажмите клавишу F4.
