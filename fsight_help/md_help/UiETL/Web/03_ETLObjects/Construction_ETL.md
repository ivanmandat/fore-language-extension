# Построение задачи ETL


Для построения задачи ETL выполните шаги:


	- Добавьте один или несколько [коннекторов
	 к источникам данных](02_Inputs/UiEtl_Inputs.htm).


	- Добавьте один или несколько [коннекторов
	 к приёмникам данных](03_Outputs/UiEtl_Outputs.htm).


	- При необходимости добавьте [преобразователи
	 данных](04_Transformers/UiEtl_Trfs.htm) или объект из группы «[Другое](Other/Other_objects_group.htm)».


	- Настройте [связи](05_Links/uietl_links_create.htm)
	 между созданными объектами.


	- Задайте [настройки
	 задачи ETL](../06_CreateETL/Relational_ETL_master.htm).


	- [Управляйте объектами
	 задачи ETL](../04_Work/Objects_settings.htm).


	- [Настройте оформление задачи
	 ETL](../04_Work/ETL_Decor.htm).


	- Настройте [рабочую область](../04_Work/UiEtl_ObjWork.htm)
	 при необходимости.


	- [Сохраните
	 задачу ETL](../01_General_Info/UiETL_StartingToWork.htm#save_etl).
