# Существующий объект репозитория


«Существующий объект репозитория»
 - объект, предоставляющий доступ к объекту репозитория с рабочего пространства
 задачи ETL. Объект не участвует в цепочках расчёта, но может использоваться,
 например, для открытия кубов или отчётов, использующих загружаемые с помощью
 задачи ETL данные.


После добавления объекта на рабочее пространство для него будет доступно
 контекстное меню, с помощью которого можно осуществлять различные [операции](../../04_Work/Objects_settings.htm#operations_objetl)
 с объектом.
