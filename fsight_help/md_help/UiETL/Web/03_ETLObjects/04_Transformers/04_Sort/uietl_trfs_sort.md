# Сортировка


	Преобразователь «Сортировка»
	 - объект, выполняющий сортировку данных. На входе объекта один источник,
	 на выходе - один приёмник. Сортировка выполняется по заданному набору
	 полей. Применение этой операции упрощает применение многих алгоритмов.


	При использовании преобразователя «Сортировка»
	 из таблицы:


			 Key
			 Date
			 Value


			 4
			 Summer

			 1

			 5
			 Summer
			 3333


			 2

	Таким образом проводилась сортировка по полю «Key».


## Настройка сортировки


	Для настройки сортировки в преобразователе:


		- Откройте [панель
		 настроек](../UiEtl_Trfs.htm#settings_panel) преобразователя.


		- Нажмите кнопку ![](../02_Union/button_settings.png) «Настройки».


		- На панели дополнительных настроек преобразователя перейдите
		 на вкладку «Настройка сортировки».


	После выполнения действий будет отображена вкладка «Настройка
	 сортировки»:


	![](Sort_Settings.png)


	В списке «Доступные поля»
	 содержатся все входные поля объекта-преобразователя, по которым доступна
	 сортировка. В списке «Выбранные поля»
	 содержатся поля, по которым будет выполняться сортировка. Перемещение
	 полей между списками, а также внутри списка, осуществляется с помощью
	 технологии Drag&Drop.


	Рядом с выбранным полем отображается направление сортировки: ![](sort_ascending.png)
	 - по возрастанию или ![](sort_descending.png) - по убыванию. При щелчке
	 по значку изменяется направление сортировки.


	Важен порядок следования полей, так как именно в этом порядке будет
	 проводиться сортировка. Чем выше расположено поле в списке «Выбранные поля», тем больше у него
	 приоритет при сортировке.
