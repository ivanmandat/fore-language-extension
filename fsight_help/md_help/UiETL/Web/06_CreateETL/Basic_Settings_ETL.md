# Базовые настройки


На странице «Базовые настройки»
 в [мастере редактирования задачи ETL](Relational_ETL_master.htm)
 задаются наименование задачи, её идентификатор и примечание.


![](base_properties.png)


Задайте следующие параметры:


	- Наименование. Введите
	 наименование задачи ETL, которое будет отображаться в [навигаторе
	 объектов](getstarted.chm::/Interface/Interface_Navigator.htm);


Переименование задачи ETL также доступно
 в режиме редактирования. Для этого:


		- Щёлкните по наименованию задачи ETL в области «Базовые
		 настройки». Наименование будет переведено в режим редактирования.


		- Внесите требуемые изменения.


		- Щёлкните за пределами наименования задачи ETL.


	- Идентификатор. Введите
	 идентификатор задачи ETL. Идентификатор представляет собой последовательность
	 букв латинского алфавита, цифр и знаков подчёркивания. Идентификатор
	 должен начинаться с буквы и быть уникальным в рамках репозитория;


	- Примечание. Введите
	 заметки или аннотации к задаче ETL. Поле необязательно для заполнения.
