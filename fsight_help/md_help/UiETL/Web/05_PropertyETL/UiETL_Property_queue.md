# Очерёдность выполнения


На странице «Очередность выполнения»
 в [мастере редактирования
 задачи ETL](../06_CreateETL/Relational_ETL_master.htm) задаются цепочки импорта/экспорта, которые будут выполнены/исключены
 при [выполнении задачи ETL](UIEtl_WorkVisualisation.htm). Задача
 ETL может содержать несколько схем преобразования данных.


![](UiEtl%20_queue.png)


Для изменения порядка выполнения операций используйте кнопки ![](down_button.png) «Вниз»
 и ![](up_button.png) «Вверх».


При выполнения задачи ETL можно включить/выключить цепочки операций. Для включения/выключения
 выбранной операции из выполнения задачи ETL установите/снимите флажок
 слева от наименования. После выполнения одного из действий выбранная операция
 сменит своё состояние на противоположное. Все объекты цепочки, исключенной
 из выполнения задачи ETL, на рабочей области будут полупрозрачными.


По умолчанию все операции выполняются.
