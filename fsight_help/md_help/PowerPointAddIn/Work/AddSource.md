# Выбор источника данных


Для выбора источника данных:


	- Выберите тип отчёта или объекта:


		- в группе «Отчёт»,
		 расположенной на вкладке «FORESIGHT» ленты инструментов,
		 выберите тип отчета: экспресс-отчёт, рабочая книга или регламентный
		 отчёт.


		- в группе «Вставить»,
		 расположенной на вкладке «FORESIGHT» ленты инструментов,
		 выберите тип объекта: таблица, диаграмма или выражение.


	- В раскрывающемся меню кнопки отчёта
	 или объекта выберите источник данных:


		- Из репозитория.
		 Будет открыто [окно
		 репозитория](GetStarted.chm::/GetStarted/Get_Started.htm).
		 Войдите в требуемый репозиторий и выберите источник данных в окне
		 «Выбор объекта»;


Примечание.
 Если при работе с надстройкой было выполнено подключение к репозиторию,
 то при выборе объектов отчёта будет сразу открыто окно выбора объекта.

Если для выражения был выбран вариант, предусматривающий выбор источника
 данных, то после выбора источника данных будет открыт диалог «Вставка
 значений из».


		- Из файла. Будет
		 открыто окно выбора источника данных, сохраненного
		 в виде экспресс-отчета в формате PPEXPRESS (*.ppexpress) или регламентного
		 отчета в формате PPREPORT (*.ppreport);


Примечание.
 Пункт меню недоступен для источника данных «Анализ
 временных рядов».


		- Вставить выражение.
		 В отчет будет вставлено выражение, рассчитанное по пользовательской
		 формуле. Будет открыт диалог «Значение»;


Примечание.
 Пункт меню доступен для типа объекта «Выражение»,
 если при работе с надстройкой уже было выполнено подключение к репозиторию.


		- Известные источники.
		 Если при работе с надстройкой было выполнено подключение к репозиторию,
		 данное меню будет содержать список репозиториев, из которых возможно
		 выбрать источник данных.


После выбора источника данных перейдите к шагу «[Добавление
 объекта или отчёта из репозитория](TableChart_Insert.htm)».
