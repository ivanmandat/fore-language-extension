## Описание


Интерфейс AboutBoxSettings используется
 для работы с классом [AboutBox](../../Classes/Application/AboutBox/AboutBox.htm).


## Комментарии


Значения свойств задаются в [конструкторе
 AboutBox](../../Classes/Application/AboutBox/Constructor_AboutBox.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../property_image.gif)


		 [AnnouncementUrl](AboutBoxSettings.AnnouncementUrl.htm)
		 Свойство AnnouncementUrl
		 определяет адрес, по которому содержится анонс на текущую версию
		 продукта.


		 ![](../../property_image.gif)
		 [ComponentsVersion](AboutBoxSettings.ComponentsVersion.htm)
		 Свойство ComponentsVersion
		 определяет версии установленных [расширений](Setup.chm::/Extensions/install_or_update_extensions.htm)
		 и компонентов.


		 ![](../../property_image.gif)
		 [ContactInfo](AboutBoxSettings.ContactInfo.htm)
		 Свойство ContactInfo
		 определяет контактную информацию продукта.


		 ![](../../property_image.gif)
		 [Copyright](AboutBoxSettings.Copyright.htm)
		 Свойство Copyright
		 определяет авторские права продукта.


		 ![](../../property_image.gif)
		 [ImagePath](AboutBoxSettings.ImagePath.htm)
		 Свойство ImagePath
		 определяет путь к файлам изображений.


		 ![](../../property_image.gif)
		 [LogoUrl](AboutBoxSettings.LogoUrl.htm)
		 Свойство LogoUrl определяет
		 изображение, используемое в качестве логотипа продукта.


		 ![](../../property_image.gif)
		 [ServiceVersion](AboutBoxSettings.ServiceVersion.htm)
		 Свойство ServiceVersion
		 определяет версию сервиса.


		 ![](../../property_image.gif)
		 [Version](AboutBoxSettings.Version.htm)
		 Свойство Version определяет
		 версию веб-приложения.
