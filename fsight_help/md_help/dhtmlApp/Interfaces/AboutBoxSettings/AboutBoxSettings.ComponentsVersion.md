# AboutBoxSettings.ComponentsVersion


## Синтаксис


ComponentsVersion: [


    {


        Name:
 String,


        Version:
 String


    },


    ...


    {


        Name:
 String,


        Version:
 String


    },


];


## Параметры


Name. Наименование расширения
 или компонента;


Version. Версия расширения
 или компонента.


## Описание


Свойство ComponentsVersion определяет
 версии установленных [расширений](Setup.chm::/Extensions/install_or_update_extensions.htm)
 и компонентов.
