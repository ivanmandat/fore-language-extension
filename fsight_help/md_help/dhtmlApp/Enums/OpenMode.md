# OpenMode


Пространство имен: PP.App


## Описание


Перечисление OpenMode содержит
 режимы открытия объекта репозитория.


Используется методом [PP.App.openObject](../Classes/App/PP.App.openObject.htm).


## Допустимые значения


		 Значение


		 Краткое описание


		 view


		 View,
		 Open. Режим просмотра.


		 edit


		 Edit.
		 Режим редактирования.
