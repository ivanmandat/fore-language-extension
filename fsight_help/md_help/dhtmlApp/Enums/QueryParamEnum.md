# QueryParamEnum


Пространство
 имен: PP.App


## Описание


Перечисление QueryParamEnum
 содержит набор параметров адреса веб-приложения.


Используется методами:


	- [PP.App.getQueryParam](../Classes/App/PP.App.getQueryParam.htm);


	- [PP.App.joinQueryParam](../Classes/App/PP.App.joinQueryParam.htm).
