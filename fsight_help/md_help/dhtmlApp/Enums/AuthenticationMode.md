# AuthenticationMode


Пространство имен: PP.App


## Описание


Перечисление AuthenticationMode
 содержит варианты [аутентификации](setup.chm::/UiWebSetup/Authentication/Authentication.htm).


Используется методом[
 PP.App.openMetabase](../Classes/App/PP.App.openMetabase.htm).


## Допустимые значения


		 Значение


		 Краткое описание


		 ignore


		 Ignore.
		 Позволяет пропустить проверку и/или установку авторизационных
		 данных.


		 login


		 Login.
		 Парольная аутентификация.


		 domain


		 Domain.
		 Доменная аутентификация.


		 IntegratedDomain


		 IntegratedDomain.
		 Интегрированная доменная.


		 oauth


		 OAuth.
		 Аутентификация по протоколу OAuth/OpenID Connect.


		 prelogin


		 PreLogin.
		 Позволяет установить соединение с метабазой по уже имеющемуся
		 идентификатору соединения.


		 guest


		 Guest. Гостевой вход.
