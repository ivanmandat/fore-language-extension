# ModuleType


Пространство имен: PP.App


## Описание


Перечисление ModuleType содержит
 возможные типы [модулей](../Classes/Application/ModuleObject/ModuleObject.htm)
 в системе.


Используется следующими методами:


	- [PP.App.getActiveModule](../Classes/App/PP.App.getActiveModule.htm);


	- [PP.App.getModuleUrl](../Classes/App/PP.App.getModuleUrl.htm).
