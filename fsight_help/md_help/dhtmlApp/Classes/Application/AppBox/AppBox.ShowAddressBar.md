# AppBox.ShowAddressBar


## Синтаксис


ShowAddressBar: Boolean;


## Описание


Свойство ShowAddressBar определяет
 признак отображения адресной строки веб-приложения «Форсайт. Аналитическая платформа».


## Комментарии


Значение свойства устанавливается с помощью метода setShowAddressBar,
 а возвращается с помощью метода getShowAddressBar. Из JSON значение свойства установить
 нельзя.** **Если свойство
 имеет значение true, то адресная строка будет отображена,
 иначе - не будет.**
