# AppBox.ModuleHeight


## Синтаксис


ModuleHeight: Number;


## Описание


Свойство ModuleHeight определяет
 высоту контейнера веб-приложения «Форсайт. Аналитическая платформа».


## Комментарии


Значение свойства устанавливается с помощью метода setModuleHeight,
 а возвращается с помощью метода getModuleHeight.
 Из JSON значение свойства установить нельзя.


Цепочка навигации может выглядеть следующим образом:


![](AppBox_ModuleHeight1.png)


## Пример


Для выполнения примера необходимо, чтобы была открыта страница экспресс-отчета
 в веб-приложении «Форсайт. Аналитическая платформа».
 Указанный ниже сценарий должен быть выполнен в консоли браузера.


Скроем цепочку навигации, выведем в консоль браузера значение высоты
 контейнера приложения и идентификатор панели контейнера приложения:


// Получим контейнер приложения
var appBox = PP.App.getAppBox();
// Скроем цепочку навигации
appBox.setShowAddressBar(false);
// Выведем высоту контейнера приложения
console.log("Высота контейнера приложения: " + appBox.getModuleHeight());
// Выведем идентификатор панели контейнера приложения
console.log("Идентификатор панели контейнера приложения: " + appBox.getModuleContainer().getId());
В результате выполнения примера цепочка навигации приложения была скрыта:


![](AppBox_ModuleHeight2.png)


Также в консоли браузера было выведено значение высоты контейнера приложения
 и идентификатор панели контейнера приложения:


Высота контейнера приложения: 651


Идентификатор панели контейнера приложения: Panel88
