# ModuleObject.getSource


## Синтаксис


getSource();


## Описание


Метод getSource возвращает источник
 данных.


## Комментарии


Тип значения, возвращаемого данным методом, зависит от типа модуля.
 Так, например, если загружен экспресс-отчёт, то метод вернёт значение
 [PP.Exp.EaxDocument](dhtmlExpress.chm::/Classes/Express/EaxDocument/EaxDocument.htm).
