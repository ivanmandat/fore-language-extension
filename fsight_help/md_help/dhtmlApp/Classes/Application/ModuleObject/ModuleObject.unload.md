# ModuleObject.unload


## Синтаксис


unload();


## Описание


Метод unload выгружает данные
 из модуля.


## Комментарии


Выгрузка данных и закрытие модуля выполняется на стороне сервера. В
 веб-приложении результат выполнения метода не отображается.
