# Конструктор ContactInfo


## Синтаксис


PP.Application.ContactInfo(settings);


## Параметры


settings. JSON-объект со значениями
 свойств класса.


## Описание


Конструктор ContactInfo создает
 экземпляр компонента ContactInfo.


## Комментарии


Свойства класса, которые можно задать в конструкторе:


		 Имя свойства
		 Тип данных
		 Описание

		 String
		 Поле с адресом.


		 ![](../../../property_image.gif)

		 ![](../../../property_image.gif)

		 Array of String
		 Массив номеров факса.


		 ![](../../../property_image.gif)

		 Array of String

		 ![](../../../property_image.gif)
