# Конструктор AddressBar


## Синтаксис


PP.Application.AddressBar(settings);


## Параметры


settings. JSON-объект со значениями
 свойств компонента.


## Описание


Конструктор AddressBar создает
 экземпляр класса AddressBar.


## Пример


Конструктор предназначен для внутреннего использования.
