# AdmBox.Repository


## Синтаксис


Repository();


## Описание


Свойство Repository определяет
 репозиторий, используемый в менеджере безопасности.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setRepository,
 а возвращается с помощью метода getRepository.
