# PP.App.joinQueryParam


## Синтаксис


joinQueryParam(param: String, val: String, queryString:
 String, sign: String);


## Параметры


param. Параметр системы;


val. Значение параметра;


queryString. Строка запроса;


sign. Знак перед параметром.


## Описание


Метод joinQueryParam подставляет
 указанный параметр в строку запроса.


## Комментарии


Метод возвращает значение типа String.
