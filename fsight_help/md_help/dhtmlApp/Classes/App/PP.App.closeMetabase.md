# PP.App.closeMetabase


## Синтаксис


closeMetabase(callback: Function|PP.Delegate, errorcallback:
 Function|PP.Delegate);


## Параметры


callback. Возвратная функция,
 выполняемая после закрытия соединения с репозиторием;


errorcallback. Возвратная функция,
 выполняемая в случае генерации ошибки.


## Описание


Метод closeMetabase закрывает
 соединение с репозиторием.
