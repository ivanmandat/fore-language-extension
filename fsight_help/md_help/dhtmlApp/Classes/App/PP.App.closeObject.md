# PP.App.closeObject


## Синтаксис


closeObject(mbObject, removeKey, clearStorage);


## Параметры


mbObject. Объект репозитория;


removeKey. Ключ закрываемого
 объекта репозитория;


clearStorage. Признак удаления
 объекта из хранилища браузера. Если параметр равен значению true,
 то объект репозитория будет удалён из хранилища, иначе - не будет.


## Описание


Метод closeObject закрывает
 указанный объект репозитория.
