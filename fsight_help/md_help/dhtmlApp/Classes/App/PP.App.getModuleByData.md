# PP.App.getModuleByData


## Синтаксис


getModuleByData(data: [PP.App.ModuleType](../../Enums/ModuleType.htm));


## Параметры


data. Тип модуля.


## Описание


Метод getModuleByData возвращает
 настройки модуля по его типу.


## Комментарии


Метод возвращает то же значение, что и метод [PP.App.getActiveModule](PP.App.getActiveModule.htm).

Использование метода приведено в примере для [PP.App.openAppModule](PP.App.openAppModule.htm).
