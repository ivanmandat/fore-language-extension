# PP.App.exit


## Синтаксис


exit();


## Описание


Метод exit осуществляет выход
 из приложения.


## Комментарии


После выполнения примера будет загружена страница для входа в приложение.


## Пример


Для выполнения примера необходимо, чтобы был открыт навигатор объектов
 приложения. Указанный ниже сценарий нужно запустить в консоли браузера.


Выполним выход из приложения:


PP.App.exit();
В результате выполнения примера была загружена страница для входа в
 систему:
