# PP.App.getMetabaseConfig


## Синтаксис


getMetabaseConfig(reload: Boolean, settings: Object);


## Параметры


reload. Признак того, что конфигурация
 репозитория будет заново загружена. Повторная загрузка конфигурации будет
 осуществлена в случае, если параметр принимает значение True,
 иначе не будет.


settings. Дополнительные настройки
 репозитория.


## Описание


Метод getMetabaseConfig возвращает
 параметры конфигурации репозитория.


## Комментарии


Метод возвращает JSON-объект, со следующими полями: Id - идентификатор
 репозитория, ConnectionOdId - моникёр подключения к репозиторию, PPServiceUrl
 - адрес сервиса, UserCreds - объект с аутентификационными данными пользователя.
