# PP.App.getImagePath


## Синтаксис


getImagePath(path: String);


## Параметры


path. Строка, добавляемая после
 возвращаемого значения. Необязательный параметр.


## Описание


Метод getImagePath возвращает
 путь к папке со значками.


## Комментарии


Метод возвращает значение типа String.


## Пример


Для выполнения примера необходимо, чтобы был открыт любой экспресс-отчёт.
 Указанный ниже сценарий нужно запустить в консоли браузера.


Определим новый путь к папке со значками, адрес и имя текущей страницы,
 а также список доступных языков:


console.log("Путь к папке со значками: " + PP.App.getImagePath());
// Получим текущий модуль
var activeModule = PP.App.getActiveModule("Olap");
// Получим название страницы, на которой загружен модуль
var moduleUrl = PP.App.getModuleUrl(activeModule.Data);
console.log("Страница с текущим модулем: " + moduleUrl);
В результате выполнения примера в консоли браузера были выведены новый
 путь к папке со значками и имя текущей страницы:


Путь к папке со значками: ../build/img/

Страница с текущим модулем: olap.html
