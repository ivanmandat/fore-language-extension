# PP.App.deleteStorage


## Синтаксис


deleteStorage(key: String, local: Boolean);


## Параметры


key. Ключ параметра в хранилище
 браузера;


local. Признак того, что данные
 находятся в локальном хранилище браузера. Если параметр равен значению
 True, то данные находятся в локальном
 хранилище и сохраняются после закрытия вкладки браузера, иначе эти данные
 доступны только для одной вкладки до тех пор, пока она не будет закрыта.


## Описание


Метод deleteStorage удаляет
 указанный параметр из хранилища браузера.
