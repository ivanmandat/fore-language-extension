# PP.App.setConfig


## Синтаксис


setConfig(value: [PP.AppConfig](dhtmlCommon.chm::/Classes/PP/AppConfig.htm));


## Параметры


Value. Настройки приложения.


## Описание


Метод setConfig устанавливает
 для приложения указанные настройки конфигурации.


## Пример


Для выполнения примера необходимо, чтобы было открыто веб-приложение
 «Форсайт. Аналитическая платформа».
 Указанный ниже сценарий нужно запустить в консоли браузера.


Определим собственный параметр системы, а также установим для интерфейса
 английский язык:


// Установим загруженные настройки
var result = PP.App.loadConfig();
if (result) {
    // Определим конфигурацию системы
    var config = PP.AppConfig;
    // Определим собственный параметр
    config.MyParam = true;
    config.Modules = null;
    // Установим новые настройки конфигурации
    PP.App.setConfig(config);
};
// Выведем значение собственного параметра
console.log("Значение параметра «MyParam»: " + PP.AppConfig.MyParam);
// Установим в качестве текущего английский язык
PP.App.setLocale({
    Locale: PP.CultureNames.en
});
// Определим текущий язык системы
var locale = PP.App.getLocale();
console.log("Текущий язык системы: " + locale);
В результате выполнения примера для интерфейса системы был установлен
 английский язык. Значение собственного параметра и код текущего языка
 были выведены в консоли браузера:


Значение параметра «MyParam»: true


Текущий язык системы: en
