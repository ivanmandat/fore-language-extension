# PP.App.getModuleObject


## Синтаксис


getModuleObject();


## Описание


Метод getModuleObject возвращает
 объект модуля.


## Комментарии


Тип значения, возвращаемого данным методом, зависит от типа модуля.
 Так, например, если загружен экспресс-отчёт, то модуль имеет тип [PP.Application.EaxModuleObject](../Application/EaxModuleObject/EaxModuleObject.htm).
