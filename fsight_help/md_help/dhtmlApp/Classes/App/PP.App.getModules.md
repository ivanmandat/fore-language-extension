# PP.App.getModules


## Синтаксис


getModules();


## Описание


Метод getModules возвращает
 массив элементов приложения.


## Комментарии


Метод возвращает массив значений типа [PP.Ui.MenuItem](dhtmlUi.chm::/Classes/MenuItem/MenuItem.htm).


## Пример


Для выполнения примера откройте веб-приложение «Форсайт. Аналитическая платформа»
 и выполните в консоли браузера:


// Получим массив элементов приложения и выведем его размер
var modules = PP.App.getModules();
console.log("Количество элементов приложения: " + modules.length);
После выполнения примера было выведено значение размера массива элементов
 приложения:


Количество элементов приложения: 8
