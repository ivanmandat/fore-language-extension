# PP.App.getModuleByName


## Синтаксис


getModuleByName(name: String);


## Параметры


name. Наименование модуля.


## Описание


Метод getModuleByName возвращает
 настройки модуля по его наименованию.


## Комментарии


Метод возвращает то же значение, что и метод [PP.App.getActiveModule](PP.App.getActiveModule.htm).


## Пример


Для выполнения примера откройте веб-приложение «Форсайт. Аналитическая платформа».
 Указанный ниже сценарий нужно запустить в консоли браузера.


// Определим настройки модуля экспресс-отчёта
var module = PP.App.getModuleByName("Olap");
console.log(module);
После выполнения примера в консоль были выведены настройки модуля экспресс-отчёта.
