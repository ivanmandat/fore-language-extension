# PP.App.getLocale


## Синтаксис


getLocale();


## Описание


Метод getLocale возвращает текущий
 язык и региональные настройки приложения.


## Комментарии


Метод возвращает элемент перечисления [PP.CultureNames](dhtmlCommon.chm::/Enums/CultureNames.htm).
