# PP.App.isMetabaseOpened


## Синтаксис


isMetabaseOpened();


## Описание


Метод isMetabaseOpened возвращает
 признак того, открыто ли соединение с репозиторием.


## Комментарии


Метод возвращает значение True,
 если открыто соединение с репозиторием, в противном случае - значение
 False.
