# PP.App.getModuleNode


## Синтаксис


getModuleNode();


## Описание


Метод getModuleNode возвращает
 DOM-элемент контейнера, в который загружен текущий модуль.


## Комментарии


Метод возвращает значение типа HTMLNode.
