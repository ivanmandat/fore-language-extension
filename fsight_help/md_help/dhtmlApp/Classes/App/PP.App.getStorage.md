# PP.App.getStorage


## Синтаксис


getStorage(key: String, local: Boolean);


## Параметры


key. Ключ параметра в хранилище
 браузера;


local. Признак того, что данные
 находятся в локальном хранилище браузера. Если параметр принимает значение
 True, то данные находятся в локальном
 хранилище и сохраняются после закрытия вкладки браузера, иначе эти данные
 доступны только для одной вкладки до тех пор, пока она не будет закрыта.


## Описание


Метод getStorage возвращает
 параметр по его ключу в хранилище браузера.


## Комментарии


Метод возвращает значение типа String.
