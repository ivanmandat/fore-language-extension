# PP.App.getModuleUrl


## Синтаксис


getModuleUrl(moduleType: [PP.App.ModuleType](../../Enums/ModuleType.htm),
 settings: Object)


## Параметры


moduleType. Тип модуля;


settings. Дополнительные настройки
 модуля. Необязательный параметр, представляет собой JSON-объект со следующими
 свойствами: queryString - строка запроса, mbObject - объект репозитория,
 OpenMode - режим открытия объекта репозитория, Class - класс стиля, Url
 - адрес модуля.


## Описание


Метод getModuleUrl возвращает
 URL-адрес модуля указанного типа.


## Комментарии


Метод возвращает значение типа String.
