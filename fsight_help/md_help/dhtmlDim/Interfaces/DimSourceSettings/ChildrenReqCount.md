# ChildrenReqCount


## Синтаксис


ChildrenReqCount: Number;


## Описание


Свойство ChildrenReqCount определяет
 количество дочерних вершин, которые по умолчанию будут запрашиваться сервисом.


## Комментарии


Значение свойства задается в конструкторе [DimSource](../../Classes/Metabase/DimSource/Constructor_DimSource.htm)
 или с помощью метода setChildrenReqCount,
 а возвращается с помощью метода getChildrenReqCount.
