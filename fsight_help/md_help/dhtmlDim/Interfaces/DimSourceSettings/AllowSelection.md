# AllowSelection


## Синтаксис


AllowSelection: Boolean


## Описание


Свойство AllowSelection определяет
 возможность установки отметки элементам.


## Комментарии


Допустимые значения:


	- true.
	 Установка отметок элементам возможна;


	- false.
	 Установка отметок элементам невозможна.


Значение свойства устанавливается из JSON и с помощью метода setAllowSelection,
 а возвращается с помощью метода getAllowSelection.
