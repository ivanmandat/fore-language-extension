# AllowHasData


## Синтаксис


AllowHasData: Boolean;


## Описание


Свойство AllowHasData определяет
 возможность загрузки признака наличия данных элементов измерения.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Признак наличия данных элементов измерения загружается;


	- False. Признак наличия
	 данных элементов измерения не загружается.


Значение свойства задается в конструкторе [DimSource](../../Classes/Metabase/DimSource/Constructor_DimSource.htm)
 или с помощью метода setAllowHasData,
 а возвращается с помощью метода getAllowHasData.
