# AllowCalculation


## Синтаксис


AllowCalculation: Boolean;


## Описание


Свойство AllowCalculation определяет
 возможность загрузки признака [вычисляемого
 элемента](UiAnalyticalArea.chm::/Work_with_dimensions/Edited_elements.htm) измерения.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Загрузка вычисляемого элемента измерения доступна;


	- False. Загрузка вычисляемого
	 элемента измерения недоступна.


Значение свойства задается в конструкторе [DimSource](../../Classes/Metabase/DimSource/Constructor_DimSource.htm)
 или с помощью метода setAllowCalculation,
 а возвращается с помощью метода getAllowCalculation.
