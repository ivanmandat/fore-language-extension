# IsAllElsLoaded


## Синтаксис


IsAllElsLoaded: Boolean


## Описание


Свойство IsAllElsLoaded определяет,
 будут ли загружены все элементы справочника вместе с дочерними.


## Комментарии


По умолчанию установлено значение false,
 то есть дочерние элементы не загружаются.
