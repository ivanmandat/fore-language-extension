# LoadAttributeValuesPartial


## Синтаксис


LoadAttributeValuesPartial: Boolean;


## Описание


Свойство LoadAttributeValuesPartial
 определяет возможность частичной загрузки значений атрибутов.


## Комментарии


Допустимые значения:


	- True. Частичная загрузка
	 значений атрибутов доступна;


	- False. По умолчанию.
	 Частичная загрузка значений атрибутов недоступна.


Значение свойства задается в конструкторе [DimSource](../../Classes/Metabase/DimSource/Constructor_DimSource.htm)
 или с помощью метода setLoadAttributeValuesPartial,
 а возвращается с помощью метода getLoadAttributeValuesPartial.
