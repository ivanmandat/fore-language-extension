# AllowAltHierarchyCreation


## Синтаксис


AllowAltHierarchyCreation: Boolean;


## Описание


Свойство AllowAltHierarchyCreation
 определяет возможность создания [альтернативной
 иерархии](UiSelection.chm::/Selection/Hierarchies.htm).


## Комментарии


Допустимые значения:


	- True. Создание альтернативной
	 иерархии доступно;


	- False. Создание альтернативной
	 иерархии недоступно.


Значение свойства задается в конструкторе [DimSource](../../Classes/Metabase/DimSource/Constructor_DimSource.htm)
 или с помощью метода setAllowAltHierarchyCreation,
 а возвращается с помощью метода getAllowAltHierarchyCreation.
