# MultiSelect


## Синтаксис


MultiSelect: Boolean


## Описание


Свойство MultiSelect определяет,
 доступна ли множественная отметка элементов справочника.


## Комментарии


По умолчанию установлено значение true,
 то есть включена возможность множественной отметки элементов.
