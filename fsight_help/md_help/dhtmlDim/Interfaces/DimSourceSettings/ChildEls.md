# ChildEls


## Синтаксис


ChildEls: Array of Object


## Описание


Свойство ChildEls устанавливает
 массив дочерних элементов выбранного элемента справочника.


## Комментарии


Массив дочерних элементов выбранного элемента справочника устанавливается
 при помощи метода setChildEls (key, value)
 и возвращается при помощи метода getChildEls(key).
 В параметре key задается ключ
 родительского элемента.


## Пример


Для выполнения примера предполагается наличие на html-странице компонента
 [DimTree](../../Components/Metabase/DimTree/DimTree.htm) с
 наименованием «dimTree», для которого установлен источник - справочник
 с наименованием «dim» (см. «[Пример
 создания компонента DimTree](../../Components/Metabase/DimTree/DimTree_Example.htm)»). В справочнике должен содержаться элемент
 с ключом «YEARS: 2002». Установим массив дочерних элементов для элемента
 с указанным ключом:


  dim.setChildEls("YEARS: 2002",
 [


  {


      "k": "1",


      "n":
 "Первый элемент",


      "h":
 "1",


      "o":
 "0"


  },


  {


      "k": "2",


      "n":
 "Второй элемент",


      "h":
 "0",


      "o":
 "1"


  },


  {


      "k": "3",


      "n":
 "Третий элемент",


      "h":
 "0",


      "o":
 "1"


  },


  {


      "k": "4",


      "n":
 "Четвертый элемент",


      "h":
 "0",


      "o":
 "0"


  }])


   dimTree.refresh();


После выполнения примера для элемента справочника с ключом «YEARS: 2002»
 будет установлен массив дочерних элементов, который будет отображаться
 в дереве.
