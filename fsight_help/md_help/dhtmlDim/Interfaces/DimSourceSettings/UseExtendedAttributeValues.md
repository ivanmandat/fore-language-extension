# UseExtendedAttributeValues


## Синтаксис


UseExtendedAttributeValues: Boolean;


## Описание


Свойство UseExtendedAttributeValues
 определяет возможность использования расширенного варианта получения значений
 [атрибутов](UiNavObj.chm::/reference_book/Master_RDS_reference_book/Attributes.htm)
 справочника НСИ.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Использование расширенного варианта получения значений [атрибутов](UiNavObj.chm::/reference_book/Master_RDS_reference_book/Attributes.htm)
	 справочника НСИ доступно;


	- False. Использование
	 расширенного варианта получения значений [атрибутов](UiNavObj.chm::/reference_book/Master_RDS_reference_book/Attributes.htm)
	 справочника НСИ недоступно.


Значение свойства задается в конструкторе [DimSource](../../Classes/Metabase/DimSource/Constructor_DimSource.htm)
 или с помощью метода setUseExtendedAttributeValues,
 а возвращается с помощью метода getUseExtendedAttributeValues.
