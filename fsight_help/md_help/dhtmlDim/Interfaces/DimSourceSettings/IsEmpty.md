# IsEmpty


## Синтаксис


IsEmpty: Boolean;


## Описание


Свойство IsEmpty определяет
 наличие элементов в измерении.


## Комментарии


Допустимые значения:


	- true.
	 Элементов в измерении нет;


	- false.
	 В измерении есть элементы.


Значение свойства устанавливается из JSON и с помощью метода setIsEmpty,
 а возвращается с помощью метода getIsEmpty.


## Пример


Для выполнения примера предполагается наличие на html-странице компонента
 [DimTree](../../Components/Metabase/DimTree/DimTree.htm) с
 наименованием «dimTree» (см. «[Пример
 создания компонента DimTree](../../Components/Metabase/DimTree/DimTree_Example.htm)»). Проверим наличие элементов в измерении:


// Проверяем наличие элементов в измерении
if (dim.getIsEmpty()) {
    console.log("В измерении нет элементов");
} else {
    console.log("В измерении есть элементы");
}
В результате в консоль будет выведен результат проверки:


В измерении есть элементы
