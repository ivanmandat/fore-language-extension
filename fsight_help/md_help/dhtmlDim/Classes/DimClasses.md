# Работа с измерением


## Описание


На данной странице приведены классы для работы с измерением.


## Классы


		 Имя класса
		 Краткое описание


		 ![](../Class_Image.gif)
		 [DimensionFilter](Metabase/DimensionFilter/DimensionFilter.htm)
		 Класс DimensionFilter
		 реализует фильтр элементов измерения.


		 ![](../Class_Image.gif)
		 [DimSource](Metabase/DimSource/DimSource.htm)
		 Класс DimSource реализует
		 модель справочника.


		 ![](../Class_Image.gif)
		 [DimSrv](Metabase/DimSrv/DimSrv.htm)
		 Класс DimSrv используется
		 для работы с сервисом для работы со справочниками.


		 ![](../Class_Image.gif)
		 [DimTree](Metabase/DimTree/DimTree.htm)
		 Класс DimTree реализует
		 компонент [DimTree](../Components/Metabase/DimTree/DimTree.htm),
		 представляющий собой дерево, в котором отображаются данные справочника.


## Интерфейсы


		 Имя интерфейса
		 Краткое описание


		 ![](../Interface_Image.gif)
		 [DimSourceSettings](../Interfaces/DimSourceSettings/DimSourceSettings.htm)
		 Интерфейс DimSourceSettings используется
		 для работы с классом [DimSource](Metabase/DimSource/DimSource.htm).


## Перечисления


		 Имя класса
		 Краткое описание


		 ![](../enums_image.gif)
		 [DimComboCaption](../Enums/DimComboCaption.htm)
		 Перечисление DimComboCaption содержит
		 варианты отображения заголовка раскрывающегося списка.


		 ![](../enums_image.gif)
		 [DimElementState](../Enums/DimElementState.htm)
		 Перечисление DimElementState содержит
		 варианты наличия некоторого состояния у элемента измерения (такие
		 как: отметка, наличие данных, принадлежность к группе и др.).


		 ![](../enums_image.gif)
		 [OrderBy](../Enums/OrderBy.htm)
		 Перечисление OrderBy содержит
		 варианты сортировки элементов.
