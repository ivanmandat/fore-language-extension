# DimSrv.DimGetMetadataComplete


## Синтаксис


DimGetMetadataComplete: function(sender, args);


## Параметры


sender. Источник события;


args. Информация
 о событии. Аргументы: Dim -
 открытый справочник; OpArgs - аргументы операции.


## Описание


Событие DimGetMetadataComplete
 наступает после завершения загрузки метаданных справочника.
