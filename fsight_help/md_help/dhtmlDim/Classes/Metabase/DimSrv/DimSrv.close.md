# DimSrv.close


## Синтаксис


close(dim: [PP.Mb.DimSource](../DimSource/DimSource.htm),
 callback: Function);


## Параметры


dim. Справочник, который нужно
 закрыть;


callback. Процедура обратного
 вызова.


## Описание


Метод close закрывает справочник.


## Пример


Для выполнения примера необходимо наличие экземпляра класса [DimSrv](DimSrv.htm)
 с наименованием «dimService» (см. [Конструктор
 DimSrv](Constructor_DimSrv.htm)). Добавим обработчик события закрытия справочника:


// Добавляем обработчик события, наступающего при закрытии справочника
dimService.DimClosed.add(
    function() {
        // Выводим в консоль браузера сообщение о событии
        console.log("Событие DimClosed");
    }
);
// Закрываем справочник
dimService.close(dim);
В результате соединение со справочником было закрыто, в консоль было
 выведено сообщение о закрытии справочника:


Событие DimClosed
