# DimSrv.getIsBatchMode


## Синтаксис


getIsBatchMode();


## Описание


Метод getIsBatchMode возвращает
 признак включения пакетного режима отправки запросов.


## Комментарии


Метод возвращает значение true,
 если включен пакетный режим отправки запросов, иначе - значение false.
