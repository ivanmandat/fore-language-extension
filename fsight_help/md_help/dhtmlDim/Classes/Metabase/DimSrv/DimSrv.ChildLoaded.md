# DimSrv.ChildLoaded


## Синтаксис


ChildLoaded: function(sender, args);


## Параметры


sender. Источник события;


args. Информация
 о событии. Аргументы: Dim -
 открытый справочник; OpArgs - аргументы операции.


## Описание


Событие ChildLoaded наступает
 после загрузки дочерних элементов справочника.
