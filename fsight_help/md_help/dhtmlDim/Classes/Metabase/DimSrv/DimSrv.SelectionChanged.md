# DimSrv.SelectionChanged


## Синтаксис


SelectionChanged: function(sender, args);


## Параметры


sender. Источник события;


args. Информация
 о событии. Аргументы: Dim -
 открытый справочник; OpArgs - аргументы операции отметки элементов.


## Описание


Событие SelectionChanged наступает
 после изменения отметки элементов справочника.

Пример использования события приведен в описании метода [DimSrv.selectByKeys](DimSrv.selectByKeys.htm).
