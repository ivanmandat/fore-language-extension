# DimSource.getLastSelectedText


## Синтаксис


getLastSelectedText (attrIdx);


## Параметры


attrIdx. Индекс или идентификатор
 атрибута, по которому нужно вернуть текст элемента.


## Описание


Метод getLastSelectedText возвращает
 текст последнего выделенного элемента.
