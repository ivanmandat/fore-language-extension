# DimSource.getFirstSelectedText


## Синтаксис


getFirstSelectedText (attrIdx)


## Параметры


attrIdx. Индекс или идентификатор
 атрибута, по которому нужно взять текст элемента.


## Описание


Метод getFirstSelectedText возвращает
 текст первого выделенного элемента.
