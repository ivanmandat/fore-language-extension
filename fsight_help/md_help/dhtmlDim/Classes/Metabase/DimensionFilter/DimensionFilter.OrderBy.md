# DimensionFilter.OrderBy


## Синтаксис


OrderBy: Array;


## Описание


Свойство OrderBy определяет
 настройки сортировки элементов измерения.


## Комментарии


Свойство содержит массив из элементов в формате: : { <AttrId>:
 <[PP.Mb.OrderBy](../../../Enums/OrderBy.htm)> }, где
 AttrId - идентификатор атрибута.


Значение свойства устанавливается из JSON. Возвращается с помощью метода
 getOrderBy().


## Пример


Пример использования приведен в описании свойства [DimensionFilter.OnlySelected](DimensionFilter.OnlySelected.htm).
