# DimensionFilter.OnlyWithData


## Синтаксис


OnlyWithData: Boolean;


## Описание


Свойство OnlyWithData определяет
 фильтрацию только элементов измерения содержащих данные.


## Комментарии


Значение свойства устанавливается из JSON. Возвращается с помощью метода
 getOnlyWithData().

Пример использования приведен в описании свойства [DimensionFilter.OnlySelected](DimensionFilter.OnlySelected.htm).
