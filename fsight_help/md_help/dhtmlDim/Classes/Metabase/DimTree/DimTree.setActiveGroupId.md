# DimTree.setActiveGroupId


## Синтаксис


setActiveGroupId(groupId: String, limitToGroup:
 Boolean);


## Параметры


groupId. Идентификатор группы.


limitToGroup. При значении
 true из отметки будут убраны элементы,
 не входящие в группу. По умолчанию true.


## Описание


Метод setActiveGroupId устанавливает
 идентификатор активной группы элементов справочника.


## Комментарии


Чтобы создать группу элементов, выберите пункт контекстного меню для
 справочника «Группы элементов», в открывшемся окне нажмите кнопку «Добавить»
 и выберите нужную группу элементов справочника. Чтобы узнать идентификатор
 группы элементов, выделите добавленную схему отметки и нажмите на
 кнопку «Сохранить как».


## Пример


Для выполнения примера предполагается наличие на html-странице компонента
 DimCombo с наименованием «dimCombo» (см. «[Пример
 размещения компонента DimCombo](../../../Components/Metabase/DimCombo/DimCombo_Example.htm)»). Предполагается наличие в справочнике
 группы элементов с идентификатором «OBJ628190». Установим для справочника
 активную группу элементов и добавим обработчик события изменения активной
 группы элементов:


// Получим представление дерева
dimTree = dimCombo.getTreeView();
// Добавляем обработчик события изменения активной группы элементов
dimTree.GroupChanging.add(function(sender, args) {
    console.log("Изменяется активная группа элементов");
)};
// Устанавливаем группу элементов
dimTree.setActiveGroupId("OBJ628190");
После выполнения примера для дерева справочника будет установлена группа
 элементов с указанным идентификатором.
