# Разработка веб-приложения с использованием
 DHTML-компонентов


В руководстве представлено описание инструментов разработчика веб-приложения
 «Форсайт. Аналитическая платформа».


![](Development_for_Web.png)


Разработка веб-приложений ведётся в сторонних средах разработки с использованием
 библиотеки DHTML-компонентов продукта «Форсайт. Аналитическая платформа».
 Подробное описание компонентов представлено в разделе «[DHTML-компоненты](dhtml.chm::/DHTML_components.htm)».


В составе «Форсайт. Аналитическая платформа» реализован веб-сервис
 PP.SOM, предназначенный для связи и взаимодействия веб-приложения с объектами
 репозитория. Описание работы приведено в разделе «[Веб-сервис](PPSOMService.chm::/PP7SOMService_Title.htm)».


В разделе «[Спецификации
 компонентов](specifications.chm::/Specifications_Intro.htm)»
 представлены спецификации различных компонентов, используемых в «Форсайт.
 Аналитическая платформа».


Описание решений типовых проблем, с которыми сталкиваются прикладные
 веб-разработчики можно найти в разделе «[База знаний](WebKnowledgeBase.chm::/WebKnowledgeBase_Title.htm)».


Рекомендации к написанию кода для прикладных веб-разработчиков
 можно найти в разделе «[Рекомендации
 к написанию кода](dhtml.chm::/specification/JavaScript_main.htm)».
