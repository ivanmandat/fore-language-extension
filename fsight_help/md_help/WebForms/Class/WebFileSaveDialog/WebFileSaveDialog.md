# WebFileSaveDialog


Сборка: WebForms;


## Описание


Класс WebFileSaveDialog реализует
 компонент веб-форм - [FileSaveDialog](UiDevEnv.chm::/Web_components/FileSaveDialog.htm).

		 ![](../../Property_Image.gif)
		 [FileName](../../Interface/IWebFileDialog/IWebFileDialog.FileName.htm)
		 Свойство FileName определяет
		 наименование открываемого/сохраняемого файла.
