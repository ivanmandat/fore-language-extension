## Описание


Класс WebMessageBox реализует
 пользовательский диалог.


## Комментарии


Для инициализации диалога вызовете статический метод [Show](../../Interface/IWebMessageBox/IWebMessageBox.Show.htm)
 с необходимым набором параметров.
