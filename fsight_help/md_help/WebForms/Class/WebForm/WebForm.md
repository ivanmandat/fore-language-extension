# WebForm

## Описание


Класс WebForm реализует веб-форму.


## Свойства объекта класса, унаследованные от [IWebFormComponent](../../Interface/IWebFormComponent/IWebFormComponent.htm)


		 Имя свойства
		 Краткое описание

		 [Clipboard](../../Interface/IWebFormComponent/IWebFormComponent.Clipboard.htm)
		 Свойство Clipboard
		 возвращает буфер обмена веб-формы.

		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)

		 определяет цвет границы компонента.


		 ![](../../Property_Image.gif)
		 [Color](../../Interface/IWebControl/IWebControl.Color.htm)
		 Свойство Color определяет

		 [Enabled](../../Interface/IWebControl/IWebControl.Enabled.htm)
		 Свойство Enabled определяет

		 [Visible](../../Interface/IWebControl/IWebControl.Visible.htm)
		 Свойство Visible определяет
