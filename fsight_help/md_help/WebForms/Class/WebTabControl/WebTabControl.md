# WebTabControl


Сборка: WebForms;


## Описание


Класс WebTabControl реализует
 компонент веб-форм - [TabControl](UiDevEnv.chm::/Web_components/TabControl.htm).

		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ActiveTab](../../Interface/IWebTabControl/IWebTabControl.ActiveTab.htm)
		 Свойство ActiveTab
		 определяет активную вкладку компонента.


		 ![](../../Property_Image.gif)
		 [Pages](../../Interface/IWebTabControl/IWebTabControl.Pages.htm)
		 Свойство Pages возвращает
		 коллекцию вкладок компонента.
