# WebMessageBoxIcon


## Описание


Перечисление WebMessageBoxIcon
 содержит типы пиктограмм, отображаемых в заголовке диалога и отражающих
 его назначение.


Используется следующими свойствами и методами:


	- [IWebMessageBox.Show](../Interface/IWebMessageBox/IWebMessageBox.Show.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Warning. Предупреждающий
		 диалог - ![](WebMessageBoxIcon_Warning.png).


		 1
		 Information. Информационный
		 диалог - ![](WebMessageBoxIcon_Information.png).


		 2
		 Error. Диалог с сообщением
		 об ошибке - ![](WebMessageBoxIcon_Error.png).


		 3
		 Question. Диалог с
		 запросом подтверждения - ![](WebMessageBoxIcon_Question.png).
