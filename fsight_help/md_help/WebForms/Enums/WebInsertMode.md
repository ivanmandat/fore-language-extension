# WebInsertMode


## Описание


Перечисление WebInsertMode содержит
 варианты вставки различных объектов.


Используется следующими свойствами и методами:


	- [IWebTreeNodes.Add](../Interface/IWebTreeNodes/IWebTreeNodes.Add.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 InsertBefore. Вставить
		 до указанного объекта.


		 1
		 InsertAfter. Вставить
		 после указанного объекта.
