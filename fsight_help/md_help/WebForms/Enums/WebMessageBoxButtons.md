# WebMessageBoxButtons


## Описание


Перечисление WebMessageBoxButtons
 содержит возможные варианты наборов кнопок, которые будут доступны в диалоге.


Используется следующими свойствами и методами:


	- [IWebMessageBox.Show](../Interface/IWebMessageBox/IWebMessageBox.Show.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Ok. Кнопка OK.


		 1
		 OkCancel. Кнопки OK
		 и Отмена.


		 2
		 YesNo. Кнопки Да и
		 Нет.


		 3
		 YesNoCancel. Кнопки
		 Да, Нет и Отмена.
