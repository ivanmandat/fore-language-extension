# Перечисления сборки WebForms


		 Перечисления
		 Краткое описание


		 ![](../Enums_Image.gif)
		 [WebCheckBoxState](WebCheckBoxState.htm)
		 Перечисление WebCheckBoxState используется
		 для определения состояния флажка.


		 ![](../Enums_Image.gif)
		 [WebDialogResult](WebDialogResult.htm)
		 Перечисление WebDialogResult
		 содержит результаты закрытия диалога пользователем.


		 ![](../Enums_Image.gif)
		 [WebInsertMode](WebInsertMode.htm)
		 Перечисление WebInsertMode
		 содержит варианты вставки различных объектов.


		 ![](../Enums_Image.gif)
		 [WebMessageBoxButtons](WebMessageBoxButtons.htm)
		 Перечисление WebMessageBoxButtons
		 содержит возможные варианты наборов кнопок, которые будут доступны
		 в диалоге.


		 ![](../Enums_Image.gif)
		 [WebMessageBoxIcon](WebMessageBoxIcon.htm)
		 Перечисление WebMessageBoxIcon
		 содержит типы пиктограмм, отображаемых в заголовке диалога и отражающих
		 его назначение.
