# WebCheckBoxState


## Описание


Перечисление WebCheckBoxState используется
 для определения состояния флажка.


Используется следующими свойствами и методами:


	- [IWebCheckBox.State](../Interface/IWebCheckBox/IWebCheckBox.State.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Undefined. Состояние
		 флажка не определено.


		 1
		 Unchecked. Флажок не
		 отмечен.


		 2
		 Indeterminate. Промежуточное
		 состояние флажка.


		 3
		 Checked. Флажок отмечен.
