# IWebDateTimePicker.Valid


## Синтаксис


Valid: Boolean;


## Описание


Свойство Valid возвращает признак
 корректности даты, указанной в компоненте.


## Комментарии


Свойство возвращает значение True,
 если в компоненте задано допустимое значение и False,
 если значение выходит за допустимые пределы [[MinValue](IWebDateTimePicker.MinValue.htm),
 [MaxValue](IWebDateTimePicker.MaxValue.htm)].
