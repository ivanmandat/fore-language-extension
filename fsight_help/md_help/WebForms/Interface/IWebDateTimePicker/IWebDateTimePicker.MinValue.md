# IWebDateTimePicker.MinValue


## Синтаксис


MinValue: DateTime;


## Описание


Свойство MinValue определяет
 минимальное значение, которое будет доступно для ввода в компоненте.


## Комментарии


Значение свойства не должно быть больше значения, указываемого в свойстве
 [MaxValue](IWebDateTimePicker.MaxValue.htm). Если свойство
 MinValue не определено, то минимальная
 допустимая дата будет 14.09.1752 00:00:00.
