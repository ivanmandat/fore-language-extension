# IWebDateTimePicker


Сборка: WebForms;


## Описание


Интерфейс IWebDateTimePicker
 содержит свойства и методы компонента веб-форм - [DateTimePicker](UiDevEnv.chm::/Web_components/DateTimePicker.htm).

		 [MaxValue](IWebDateTimePicker.MaxValue.htm)
		 Свойство MaxValue определяет
		 максимальное значение, которое будет доступно для ввода в компоненте.
