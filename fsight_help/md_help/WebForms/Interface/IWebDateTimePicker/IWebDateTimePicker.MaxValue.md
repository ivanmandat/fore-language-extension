# IWebDateTimePicker.MaxValue


## Синтаксис


MaxValue: DateTime;


## Описание


Свойство MaxValue определяет
 максимальное значение, которое будет доступно для ввода в компоненте.


## Комментарии


Значение свойства не должно быть меньше значения, указываемого в свойстве
 [MinValue](IWebDateTimePicker.MinValue.htm). Если свойство
 MaxValue не определено, то максимальная
 допустимая дата будет 18.12.3001 23:59:59.
