# IWebFileOpenDialog.Filter


## Синтаксис


Filter: String;


## Описание


Свойство Filter определяет текст
 фильтра для ограничения списка доступных файлов.


## Комментарии


В фильтре через запятую укажите форматы файлов, которые будут доступны
 при работе диалога, например: .txt, .doc, .docx.
