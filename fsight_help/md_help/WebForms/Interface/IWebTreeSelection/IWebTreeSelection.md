# IWebTreeSelection


Сборка: WebForms;


## Описание


Интерфейс IWebTreeSelection
 предназначен для работы с коллекцией отмеченных вершин.


## Иерархия наследования


IWebTreeSelection


## Комментарии


Коллекцию отмеченных вершин можно получить, используя свойство [IWebTreeControl.Selection](../IWebTreeControl/IWebTreeControl.Selection.htm).
