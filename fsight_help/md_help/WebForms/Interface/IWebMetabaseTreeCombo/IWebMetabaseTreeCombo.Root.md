# IWebMetabaseTreeCombo.Root


## Синтаксис


Root: [IMetabaseObjectDescriptor](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm);


## Описание


Свойство Root определяет корневой
 каталог, содержимое которого будет отображаться в компоненте.


## Комментарии


По умолчанию свойство не определено, при этом в компоненте будет отображаться
 содержимое всего репозитория.


В качестве значения укажите описание какой-либо папки или объекта-контейнера
 репозитория.
