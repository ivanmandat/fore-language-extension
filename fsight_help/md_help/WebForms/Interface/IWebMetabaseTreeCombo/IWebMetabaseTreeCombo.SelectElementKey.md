# IWebMetabaseTreeCombo.SelectElementKey


## Синтаксис


SelectElementKey(Key: Integer);


## Параметры


Key. Ключ объекта, который
 необходимо выделить в компоненте.


## Описание


Метод SelectElementKey выделяет
 в компоненте объект, ключ которого передаётся в качестве входного параметра.
