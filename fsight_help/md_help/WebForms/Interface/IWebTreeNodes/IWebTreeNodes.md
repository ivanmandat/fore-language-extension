# IWebTreeNodes


Сборка: WebForms;


## Описание


Интерфейс IWebTreeNodes предназначен
 для работы с коллекцией вершин дерева.


## Иерархия наследования


IWebTreeNodes


## Комментарии


Коллекцию вершин можно получить, используя свойство [IWebTreeControl.Nodes](../IWebTreeControl/IWebTreeControl.Nodes.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IWebTreeNodes.Count.htm)
		 Свойство Count возвращает
		 количество вершин в коллекции.


		 ![](../../Property_Image.gif)
		 [FirstNode](IWebTreeNodes.FirstNode.htm)
		 Свойство FirstNode
		 возвращает первую вершину в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IWebTreeNodes.Item.htm)
		 Свойство Item возвращает

## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IWebTreeNodes.Add.htm)
		 Метод Add создаёт в
		 дереве новую вершину с заданным текстом.


		 ![](../../Sub_Image.gif)
		 [AddChild](IWebTreeNodes.AddChild.htm)
		 Метод AddChild создаёт
		 в дереве для указанной вершины новую дочернюю вершину с заданным
		 текстом.


		 ![](../../Sub_Image.gif)
		 [AddToRoot](IWebTreeNodes.AddToRoot.htm)
		 Метод AddToRoot создаёт
		 вершину на корневом уровне дерева.


		 ![](../../Sub_Image.gif)
		 [Clear](IWebTreeNodes.Clear.htm)
		 Метод Clear удаляет
		 все вершины из коллекции.


		 ![](../../Sub_Image.gif)
		 [Delete](IWebTreeNodes.Delete.htm)
		 Метод Delete удаляет
		 указанную вершину из коллекции.


		 ![](../../Sub_Image.gif)
		 [DeselectAll](IWebTreeNodes.DeselectAll.htm)
		 Метод DeselectAll снимает
		 выделение со всех вершин.


		 ![](../../Sub_Image.gif)
		 [FindById](IWebTreeNodes.FindById.htm)
		 Метод FindById осуществляет
		 поиск вершины по уникальном цифровому идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByText](IWebTreeNodes.FindByText.htm)
		 Метод FindByText осуществляет
		 поиск вершины по тексту.


		 ![](../../Sub_Image.gif)
		 [SelectAll](IWebTreeNodes.SelectAll.htm)
		 Метод SelectAll выделяет
		 все вершины.
