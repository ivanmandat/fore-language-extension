# IWebTreeNodes.FindById


## Синтаксис


FindById(Index: Integer): [IWebTreeNode](../IWebTreeNode/IWebTreeNode.htm);


## Параметры


Index. Индекс вершины.


## Описание


Метод FindById осуществляет
 поиск вершины по уникальном цифровому идентификатору.


## Комментарии


Метод возвращает значение Null,
 если вершина не найдена. Уникальный цифровой идентификатор генерируется
 при создании вершины и доступен в свойстве [IWebTreeNode.Id](../IWebTreeNode/IWebTreeNode.Id.htm).
