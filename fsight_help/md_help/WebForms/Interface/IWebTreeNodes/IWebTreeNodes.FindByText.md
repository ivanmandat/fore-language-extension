# IWebTreeNodes.FindByText


## Синтаксис


FindByText(Text: String): [IWebTreeNode](../IWebTreeNode/IWebTreeNode.htm);


## Параметры


Text. Текст вершины.


## Описание


Метод FindByText осуществляет
 поиск вершины по тексту.


## Комментарии


Метод возвращает значение Null,
 если вершина не найдена.
