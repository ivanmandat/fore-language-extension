# IWebTreeNodes.AddChild


## Синтаксис


AddChild(Node: [IWebTreeNode](../IWebTreeNode/IWebTreeNode.htm);
 Text: String): [IWebTreeNode](../IWebTreeNode/IWebTreeNode.htm);


## Параметры


Node. Вершина, для которой
 будет создана дочерняя вершина.


Text. Текст вершины.


## Описание


Метод AddChild создаёт в дереве
 для указанной вершины новую дочернюю вершину с заданным текстом.
