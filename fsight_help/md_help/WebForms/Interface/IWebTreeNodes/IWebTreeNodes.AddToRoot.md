# IWebTreeNodes.AddToRoot


## Синтаксис


AddToRoot(Text: String): [IWebTreeNode](../IWebTreeNode/IWebTreeNode.htm);


## Параметры


Text.
 Текст вершины.


## Описание


Метод AddToRoot создаёт вершину
 на корневом уровне дерева.


## Комментарии


Новая вершина добавляется в конце дерева.
