# IWebFormComponent.Close


## Синтаксис


Close([Arguments: [WebFormResult](ForeSys.chm::/Enums/WebFormResult.htm)
 = 0]);


## Параметры


Arguments. Результат закрытия
 веб-формы.


## Описание


Метод Close осуществляет закрытие
 текущей веб-формы.


## Комментарии


Дальнейшее поведение при вызове метода Close
 зависит от значения свойства modal,
 которое задаётся в режиме дизайнера веб-формы:


	- modal = True.
	 Вызов метода приводит к закрытию модальной веб-формы;


	- modal = False.
	 Вызов метода приведёт к закрытию веб-формы и возврату фокуса в навигатор
	 объектов, если веб-форма отображалась в текущей вкладке или закрытию
	 вкладки/окна, если веб-форма отображалась в новой вкладке/окне.


Если веб-формы используются в конструкторе бизнес-приложений:


	- modal = True.
	 Вызов метода приводит к закрытию модальной веб-формы;


	- modal = False
	 и веб-форма открыта в текущей вкладке. Осуществляется закрытие веб-формы
	 и возврат в корень бизнес-приложения;


	- modal = False
	 и веб-форма открыта в новой вкладке/окне. Осуществляется закрытие
	 веб-формы и возврат в корень бизнес-приложения в новой вкладке/окне.


Параметр Arguments используется
 для возврата результата закрытия модальной формы в метод [IWebForm.ShowModal](KeSom.chm::/Interface/IWebForm/IWebForm.ShowModal.htm).


## Пример


Для выполнения примера в репозитории предполагается наличие веб-формы,
 на форме расположены какие-либо компоненты, а также кнопка с наименованием
 Button1.


Добавьте ссылки на системные сборки: WebForms.


	Sub Button1OnClick;

	Begin

	    Self.Close(WebFormResult.OK);

	End Sub Button1OnClick;


При нажатии на кнопку текущая веб-форма будет закрыта с возвратом указанного
 результата закрытия.
