# IWebFloatEdit.Value


## Синтаксис


Value: Double;


## Описание


Свойство Value определяет текущее
 значение, заданное в компоненте.


## Комментарии


Значение должно находиться в диапазоне [[MinValue](IWebFloatEdit.MinValue.htm),
 [MaxValue](IWebFloatEdit.MaxValue.htm)].
