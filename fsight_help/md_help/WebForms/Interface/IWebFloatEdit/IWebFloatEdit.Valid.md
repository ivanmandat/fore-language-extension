# IWebFloatEdit.Valid


## Синтаксис


Valid: Boolean;


## Описание


Свойство Valid возвращает признак
 корректности указанного в компоненте значения.


## Комментарии


Свойство возвращает значение True,
 если в компоненте задано допустимое значение и False,
 если значение выходит за допустимые пределы [MinValue](IWebFloatEdit.MinValue.htm)-[MaxValue](IWebFloatEdit.MaxValue.htm).
