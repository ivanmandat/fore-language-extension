# IWebFloatEdit.MaxValue


## Синтаксис


MaxValue: Double;


## Описание


Свойство MaxValue определяет
 максимальное значение числа, которое можно установить в компоненте.


## Комментарии


В компоненте можно указать любое значение, если [MinValue](IWebFloatEdit.MinValue.htm)
 = MaxValue = 0. Значение будет ограничено допустимыми для типа данных
 Double предельными значениями.
