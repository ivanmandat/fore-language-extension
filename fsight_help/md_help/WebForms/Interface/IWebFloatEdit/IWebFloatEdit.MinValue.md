# IWebFloatEdit.MinValue


## Синтаксис


MinValue: Double;


## Описание


Свойство MinValue определяет
 минимальное значение числа, которое можно установить в компоненте.


## Комментарии


В компоненте можно указать любое значение, если MinValue = [MaxValue](IWebFloatEdit.MaxValue.htm)
 = 0. Значение будет ограничено допустимыми для типа данных Double предельными
 значениями.
