# IWebTextArea.Lines


## Синтаксис


Lines: [IStringList](ModCollections.chm::/Interface/IStringList/IStringList.htm);


## Описание


Свойство Lines определяет коллекцию
 строк, отображаемых в компоненте.


## Пример


Для выполнения примера в репозитории предполагается наличие веб-формы
 и расположенного на ней компонента TextArea с наименованием TextArea1.
 Указанная процедура установлена в качестве обработчика события OnShow
 для веб-формы.


	Sub TESTWebFormOnShow;

	Var

	    Lines: IStringList;

	Begin

	    Lines := New StringList.Create;

	    Lines.Add("Россия, или Российская Федерация (сокр. РФ) — государство в Восточной Европе и Северной Азии.");

	    Lines.Add("Россия — крупнейшее государство в мире, её территория в международно признанных границах составляет 17 098 246 квадратных киллометра.");

	    Lines.Add("Столица — Москва. Государственный язык на всей территории страны — русский, в ряде регионов России также установлены свои государственные и официальные языки.");

	    Lines.Add("Денежная единица — российский рубль.");

	    TextArea1.Lines := Lines;

	End Sub TESTWebFormOnShow;


При запуске веб-формы для компонента TextArea1 будет задан набор отображаемых
 строк.
