# IWebControl.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет
 доступность компонента для пользователя.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Компонент доступен для пользователя.


	- False. Компонент недоступен
	 для пользователя.


Недоступный
 компонент отображается серым цветом, игнорирует события, связанные с мышью
 и клавиатурой.
