# IWebControl.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет
 видимость компонента во время выполнения формы.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Компонент отображается и доступен для использования пользователю.


	- False. Компонент скрыт,
	 но доступен в коде приложения.
