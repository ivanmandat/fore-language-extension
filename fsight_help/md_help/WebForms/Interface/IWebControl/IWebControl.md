# IWebControl


Сборка: WebForms;


## Описание


Интерфейс IWebControl содержит
 базовые свойства и методы всех визуальных компонентов веб-форм.

## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BorderColor](IWebControl.BorderColor.htm)
		 Свойство BorderColor
		 определяет цвет границы компонента.


		 ![](../../Property_Image.gif)
		 [Color](IWebControl.Color.htm)
		 Свойство Color определяет
		 цвет фона компонента.


		 ![](../../Property_Image.gif)
		 [Enabled](IWebControl.Enabled.htm)
		 Свойство Enabled определяет
		 доступность компонента для пользователя.


		 ![](../../Property_Image.gif)
		 [PopupMenu](IWebControl.PopupMenu.htm)
		 Свойство PopupMenu
		 определяет контекстное меню, которое будет вызываться для компонента.


		 ![](../../Property_Image.gif)
		 [Text](IWebControl.Text.htm)
		 Свойство Text определяет
		 строку, идентифицирующую компонент для пользователя.


		 ![](../../Property_Image.gif)
		 [Visible](IWebControl.Visible.htm)
		 Свойство Visible определяет
		 видимость компонента во время выполнения формы.
