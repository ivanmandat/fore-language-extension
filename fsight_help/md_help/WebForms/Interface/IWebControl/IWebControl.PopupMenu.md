# IWebControl.PopupMenu


## Синтаксис


PopupMenu: [IWebPopupMenu](../IWebPopupMenu/IWebPopupMenu.htm);


## Описание


Свойство PopupMenu определяет
 контекстное меню, которое будет вызываться для компонента.


## Комментарии


В качестве значения укажите один из компонентов [PopupMenu](UiDevEnv.chm::/Web_components/PopupMenu.htm),
 которые расположены на веб-форме.
