# IWebControl.Text


## Синтаксис


Text: String;


## Описание


Свойство Text определяет строку,
 идентифицирующую компонент для пользователя.


## Комментарии


По умолчанию Text совпадает
 с наименованием компонента. Для формы свойство Text
 задаёт заголовок. Для управляющих элементов (кнопки, переключатели и другие)
 задаётся текст, отображаемый рядом или в самом элементе управления компонента.
 Для компонентов, предоставляющих возможность редактирования или выбора,
 например [Input](UiDevEnv.chm::/Web_components/Input.htm),
 [ComboBox](UiDevEnv.chm::/Web_components/ComboBox.htm)
 можно указать текст, отображаемый в редакторе по умолчанию.
