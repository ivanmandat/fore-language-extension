# IWebClipboard.Text


## Синтаксис


Text: String;


## Описание


Свойство Text определяет текст,
 копируемый в буфер обмена.


## Комментарии


Свойство доступно только на запись. Вставка из буфера обмена осуществляется
 с помощью средств браузера: сочетание клавиш CTRL+V или соответствующий
 пункт контекстного меню.
