# IWebClipboard


Сборка: WebForms;


## Описание


Интерфейс IWebClipboard предназначен
 для работы с буфером обмена веб-формы.


## Иерархия наследования


IWebClipboard


## Комментарии


Буфер обмена веб-формы можно получить, используя свойство [IWebFormComponent.Clipboard](../IWebFormComponent/IWebFormComponent.Clipboard.htm).
 Доступна только запись в буфер обмена. Вставка из буфера обмена осуществляется
 с помощью средств браузера: сочетание клавиш CTRL+V или соответствующий
 пункт контекстного меню.
