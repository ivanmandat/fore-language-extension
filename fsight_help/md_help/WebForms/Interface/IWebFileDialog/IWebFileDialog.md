# IWebFileDialog


Сборка: WebForms;


## Описание


Интерфейс IWebFileDialog содержит
 базовые свойства и методы компонентов, которые реализуют различные диалоги.


## Иерархия наследования


           [IWebComponent](../IWebComponent/IWebComponent.htm)


           IWebFileDialog


## Комментарии


Интерфейс является базовым для следующих интерфейсов:


	- [IWebFileOpenDialog](../IWebFileOpenDialog/IWebFileOpenDialog.htm);


	- [IWebFileSaveDialog](../IWebFileSaveDialog/IWebFileSaveDialog.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [FileData](IWebFileDialog.FileData.htm)
		 Свойство FileData определяет
		 поток, связанный с открываемым/сохраняемым файлом.


		 ![](../../Property_Image.gif)
		 [FileName](IWebFileDialog.FileName.htm)
		 Свойство FileName определяет
		 наименование открываемого/сохраняемого файла.
