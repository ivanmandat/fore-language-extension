# IWebIntegerEdit.Value


## Синтаксис


Value: Integer;


## Описание


Свойство Value определяет текущее
 значение, заданное в компоненте.


## Комментарии


Значение должно находиться в диапазоне [[MinValue](IWebIntegerEdit.MinValue.htm),
 [MaxValue](IWebIntegerEdit.MaxValue.htm)].
