# IWebIntegerEdit.MinValue


## Синтаксис


MinValue: Integer;


## Описание


Свойство MinValue определяет
 минимальное значение числа, которое можно установить в компоненте.


## Комментарии


В компоненте можно указать любое значение, если MinValue = [MaxValue](IWebIntegerEdit.MaxValue.htm)
 = 0. Значение будет ограничено допустимыми для типа данных Integer предельными
 значениями.
