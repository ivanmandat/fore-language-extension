# IWebIntegerEdit.MaxValue


## Синтаксис


MaxValue: Integer;


## Описание


Свойство MaxValue определяет
 максимальное значение числа, которое можно установить в компоненте.


## Комментарии


В компоненте можно указать любое значение, если [MinValue](IWebIntegerEdit.MinValue.htm)
 = MaxValue = 0. Значение будет ограничено допустимыми для типа данных
 Integer предельными значениями.
