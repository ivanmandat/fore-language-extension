# IWebTreeNode.PreviousNode


## Синтаксис


PreviousNode: [IWebTreeNode](IWebTreeNode.htm);


## Описание


Свойство PreviousNode возвращает
 предыдущую вершину относительно текущей вершины.


## Комментарии


Свойство возвращает значение Null,
 если предыдущая вершина отсутствует.
