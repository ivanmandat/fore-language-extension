# IWebTreeNode.Selected


## Синтаксис


Selected: Boolean;


## Описание


Свойство Selected определяет,
 выделена ли вершина.


## Комментарии


Допустимые значения:


	- True. Вершина выделена;


	- False. Вершина не выделена.
