# IWebTreeNode


Сборка: WebForms;


## Описание


Интерфейс IWebTreeNode предназначен
 для работы с вершиной дерева.


## Иерархия наследования


IWebTreeNode


## Комментарии


Вершину дерева можно получить, используя свойства и методы интерфейса
 [IWebTreeNodes](../IWebTreeNodes/IWebTreeNodes.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Data](IWebTreeNode.Data.htm)
		 Свойство Data предназначено
		 для хранения любых пользовательских данных.


		 ![](../../Property_Image.gif)
		 [Id](IWebTreeNode.Id.htm)
		 Свойство Id возвращает
		 уникальный числовой идентификатор вершины.

		 [NextNode](IWebTreeNode.NextNode.htm)
		 Свойство NextNode возвращает
		 следующую вершину относительно текущей вершины.


		 ![](../../Property_Image.gif)
		 [Parent](IWebTreeNode.Parent.htm)
		 Свойство Parent возвращает
		 родительскую вершину.


		 ![](../../Property_Image.gif)
		 [PreviousNode](IWebTreeNode.PreviousNode.htm)
		 Свойство PreviousNode
		 возвращает предыдущую вершину относительно текущей вершины.


		 ![](../../Property_Image.gif)
		 [Selected](IWebTreeNode.Selected.htm)
		 Свойство Selected определяет,
		 выделена ли вершина.


		 ![](../../Property_Image.gif)
		 [Text](IWebTreeNode.Text.htm)
		 Свойство Text определяет
		 текст вершины.
