# IWebTreeNode.NextNode


## Синтаксис


NextNode: [IWebTreeNode](IWebTreeNode.htm);


## Описание


Свойство NextNode возвращает
 следующую вершину относительно текущей вершины.


## Комментарии


Свойство возвращает значение Null,
 если следующая вершина отсутствует.
