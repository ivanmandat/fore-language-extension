# IWebCommandEventArgs


Сборка: WebForms;


## Описание


Интерфейс IWebCommandEventArgs
 содержит свойства аргумента события [onCommand](UiDevEnv.chm::/01_Development_Environment/02_Work_in_Development_Environment/DevEnv_Object/Web_Form.htm),
 наступающего при отправке веб-форме какой-либо команды.


## Иерархия наследования


IWebCommandEventArgs


## Комментарии


Отправка команд веб-форме осуществляется с помощью метода [IWebForm.SendCommand](KeSom.chm::/Interface/IWebForm/IWebForm.SendCommand.htm).
