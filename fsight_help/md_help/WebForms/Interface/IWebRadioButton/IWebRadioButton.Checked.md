# IWebRadioButton.Checked


## Синтаксис


Checked: Boolean;


## Описание


Свойство Checked определяет,
 выбран ли переключатель.


## Комментарии


Допустимые значения:


	- True. Переключатель
	 выбран.


	- False. Значение по умолчанию.
	 Переключатель не выбран.
