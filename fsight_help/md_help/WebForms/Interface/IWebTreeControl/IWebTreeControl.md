# IWebTreeControl


Сборка: WebForms;


## Описание


Интерфейс IWebTreeControl содержит
 базовые свойства компонентов, отображающих древовидные иерархические структуры.


## Иерархия наследования


           [IWebComponent](../IWebComponent/IWebComponent.htm)


           [IWebControl](../IWebControl/IWebControl.htm)


           IWebTreeControl


## Комментарии


Интерфейс является базовым для следующих интерфейсов:


	- [IWebTreeCombo](../IWebTreeCombo/IWebTreeCombo.htm)


	- [IWebTreeList](../IWebTreeList/IWebTreeList.htm)
