# IWebTreeControl.BeginUpdate


## Синтаксис


BeginUpdate;


## Описание


Метод BeginUpdate блокирует
 обновление дерева элементов компонента.


## Комментарии


Метод используется при внесении многочисленных изменений в структуру
 дерева элементов для избежания мерцания компонента и возможного зависания
 веб-формы. Возобновление обновления дерева элементов осуществляется после
 вызова метода [EndUpdate](IWebTreeControl.EndUpdate.htm).
