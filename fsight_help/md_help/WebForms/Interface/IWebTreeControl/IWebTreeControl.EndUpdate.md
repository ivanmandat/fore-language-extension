# IWebTreeControl.EndUpdate


## Синтаксис


EndUpdate;


## Описание


Метод EndUpdate возобновляет
 обновление дерева элементов компонента.


## Комментарии


Метод вызывается после вызова метода [BeginUpdate](IWebTreeControl.BeginUpdate.htm).
 При выполнении метода EndUpdate
 осуществляется отрисовка всех изменений, которые были внесены между вызовами
 [BeginUpdate](IWebTreeControl.BeginUpdate.htm) и EndUpdate.
