# IWebComboBox


Сборка: WebForms;


## Описание


Интерфейс IWebComboBox содержит
 свойства и методы компонента веб-форм - [ComboBox](UiDevEnv.chm::/Web_components/ComboBox.htm).

		 [ItemIndex](IWebComboBox.ItemIndex.htm)
		 Свойство ItemIndex
