# IWebComboBox.BeginUpdate


## Синтаксис


BeginUpdate;


## Описание


Метод BeginUpdate блокирует
 обновление списка строк компонента.


## Комментарии


Метод используется при внесении многочисленных изменений в коллекцию
 строк для избежания мерцания компонента и возможного зависания веб-формы.
 Возобновление обновления строк осуществляется после вызова метода [EndUpdate](../IWebTreeControl/IWebTreeControl.EndUpdate.htm).
