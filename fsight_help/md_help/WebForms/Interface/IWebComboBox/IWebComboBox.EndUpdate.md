# IWebComboBox.EndUpdate


## Синтаксис


EndUpdate;


## Описание


Метод EndUpdate возобновляет
 обновление списка строк компонента.


## Комментарии


Метод вызывается после вызова метода [BeginUpdate](IWebComboBox.BeginUpdate.htm).
 При выполнении метода EndUpdate
 осуществляется отрисовка всех изменений, которые были внесены между вызовами
 [BeginUpdate](IWebComboBox.BeginUpdate.htm) и EndUpdate.
