# IWebCheckBox.Checked


## Синтаксис


Checked: Boolean;


## Описание


Свойство Checked определяет,
 отмечен ли флажок.


## Комментарии


Допустимые значения:


	- True. Флажок отмечен.


	- False. Значение по умолчанию.
	 Флажок не отмечен.
