# SolidColorBrush.Color


## Синтаксис


Color: String


## Описание


Свойство Color устанавливает цвет заливки.


## Комментарии


Значение свойства - экземпляр класса PP.[Color](../Color/Color.htm) или строка со значением цвета в одном из следующих форматов:


-
"red";


-
"#000";


-
"RGB(123, 12, 1)".
