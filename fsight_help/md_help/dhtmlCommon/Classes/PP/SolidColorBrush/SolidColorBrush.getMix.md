# SolidColorBrush.getMix


## Синтаксис


getMix(brush2: [PP.SolidColorBrush](SolidColorBrush.htm),
 part: Number)


## Параметры


brush2. Кисть со сплошной заливкой,
 с которой требуется выполнить смешивание;


part. Доля использования указываемой
 кисти. Значение может варьироваться от 0 (кисть не применяется) до 1 (применяется
 только данная кисть).


## Описание


Метод getMix возвращает кисть
 со сплошной заливкой, полученной в результате смешивания с указанной
 кистью в заданном соотношении.


## Комментарии


Метод возвращает значение типа [PP.SolidColorBrush](SolidColorBrush.htm).
