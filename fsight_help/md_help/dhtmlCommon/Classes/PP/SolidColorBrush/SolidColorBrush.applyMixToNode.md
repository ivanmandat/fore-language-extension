# SolidColorBrush.applyMixToNode


## Синтаксис


applyMixToNode(node: HTMLElement, brush2: [PP.SolidColorBrush](SolidColorBrush.htm),
 part: Number);


## Параметры


node. DOM-элемент, к которому
 применяется кисть;


brush2. Кисть со сплошной заливкой,
 с которой требуется выполнить смешивание;


part. Доля использования указываемой
 кисти. Значение может варьироваться от 0 (кисть не применяется) до 1 (применяется
 только данная кисть).


## Описание


Метод applyMixToNode применяет
 смешанный цвет кистей со сплошной заливкой к указанному DOM-элементу.
