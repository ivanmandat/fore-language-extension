# PP.initNamespace


## Синтаксис


initNamespace(namespace: Object, name: String, resourceGroup:  String|Array);


## Параметры


*namespace.* Инициализируемое пространство имён;


*name.* Полное наименование пространства имён;


*resourceGroup.* Группа или массив групп ресурсов, которые привязаны к пространству имён.


## Описание


Метод initNamespace** инициализирует пространство имён.
