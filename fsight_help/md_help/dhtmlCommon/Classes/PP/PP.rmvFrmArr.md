# PP.rmvFrmArr


## Синтаксис


rmvFrmArr(arr: Array, item);


## Параметры


*arr.* Исходный массив;


*item.* Элемент, удаляемый из массива.


## Описание


Метод rmvFrmArr** удаляет элемент из массива.


## Комментарии


Метод возвращает значение true, если элемент был успешно удалён из массива, иначе - значение false.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на файл сценария PP.js. Определим массив целых чисел и удалим из него один из элементов:


// Определим массив целых чисел
var elements = [5, 9, 6, 7, 2];
console.log("Исходный массив элементов: [" + elements + "]");
// Удалим из массива элемент 7
PP.rmvFrmArr(elements, 7);
console.log("Изменённый массив элементов: [" + elements + "]");

В результате выполнения примера из заданного массива был удалён элемент 7:


Исходный массив элементов: [5,9,6,7,2]


Изменённый массив элементов: [5,9,6,2]
