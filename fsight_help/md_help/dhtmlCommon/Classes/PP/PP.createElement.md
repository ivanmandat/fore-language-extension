# PP.createElement


## Синтаксис


createElement(parent: HTMLElement, className: String, tagName: String);


## Параметры


*parent.* Родительский элемент;


*className.* Наименование css-класса для создаваемого элемента;


*tagName.* Наименование тега. Необязательный параметр, по умолчанию равен «div».


## Описание


Метод createElement** создаёт DOM-элемент с указанным тегом.


## Комментарии


Метод возвращает значение типа HTMLElement.
