# Date.format


## Синтаксис


PP.Date.format(date, format): String;


## Параметры


date. Date. Дата.


format. String | [PP.Date.Pattern](Date.Pattern.htm).
 Формат, к которому необходимо привести дату.


## Описание


Статический метод format возвращает
 дату в отформатированном виде.


## Комментарии


Метод возвращает объект типа String.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на
 файл сценария PP.js.


// Создадим текущую дату
date = new Date();
// Зададим формат даты
format = PP.Date.Pattern.ShortDate;
// Получим дату в формате короткой даты
newDate = PP.Date.format(date, format);
console.log(newDate);
В результате выполнения примера дата была представлена в укороченном
 формате.
