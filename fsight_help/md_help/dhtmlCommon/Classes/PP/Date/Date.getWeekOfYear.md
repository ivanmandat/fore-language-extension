# Date.getWeekOfYear


## Синтаксис


PP.Date.getWeekOfYear(date): Number;


## Параметры


date. Date. Дата.


## Описание


Статический метод getWeekOfYear
 возвращает номер недели указанной даты.


## Комментарии


Возвращает номер недели, начиная с единицы.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на
 файл сценария PP.js.


// Создадим текущую дату
date = new Date();
// Определим номер недели
week = PP.Date.getWeekOfYear(date);
console.log(week);
В результате выполнения примера был получен номер текущей недели.
