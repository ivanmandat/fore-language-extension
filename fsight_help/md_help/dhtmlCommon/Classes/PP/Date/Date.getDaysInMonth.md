# Date.getDaysInMonth


## Синтаксис


PP.Date.getDaysInMonth(date): Number;


## Параметры


date. Date. Дата.


## Описание


Статический метод getDaysInMonth
 возвращает количество дней в месяце указанной даты.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на
 файл сценария PP.js.


// Создадим текущую дату
date = new Date();
// Получим количество дней
days= PP.Date.getDaysInMonth(date);
console.log(days);
В результате выполнения примера было получено количество дней в текущем
 месяце.
