# Date.getDayOfYear


## Синтаксис


PP.Date.getDayOfYear(date): Number;


## Параметры


date. Date. Дата.


## Описание


Статический метод getDayOfYear
 возвращает номер дня в году.


## Комментарии


Возвращает номер дня, начиная с нуля.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на
 файл сценария PP.js.


// Создадим текущую дату
date = new Date();
// Определим номер дня
day= PP.Date.getDayOfYear(date);
console.log(day);
В результате выполнения примера был определен номер дня текущей даты.
