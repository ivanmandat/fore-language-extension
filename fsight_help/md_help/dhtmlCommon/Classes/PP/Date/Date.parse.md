# Date.parse


## Синтаксис


PP.Date.parse(date, format, strict): Date;


## Параметры


date. String | Number | Date.
 Строковое значение даты.


format. String | [PP.Date.Pattern](Date.Pattern.htm).
 Формат, к которому необходимо привести дату.


strict. Boolean. Проверять
 ли дату на корректность. По умолчанию содержит значение false.


## Описание


Статический метод parse возвращает
 дату из указанной строки по указанному шаблону форматирования.


## Комментарии


Строковое значение даты может быть создано, используя метод [PP.Date.format](Date.format.htm).


Если временная зона не указана, то считается, что время местное.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на
 файл сценария PP.js.


// Создадим дату в строковом виде
date = "2016-09-05T11:24:21.453+05:00";
// Зададим формат даты
format = PP.Date.Pattern.Default;
// Приведем строку к формату даты
newDate = PP.Date.parse(date, format);
console.log(newDate);
В результате выполнения примера из строки был получен объект даты.
