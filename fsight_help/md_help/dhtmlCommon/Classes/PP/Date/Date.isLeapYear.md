# Date.isLeapYear


## Синтаксис


PP.Date.isLeapYear(date): Boolean;


## Параметры


date. Date. Дата.


## Описание


Статический метод isLeapYear
 возвращает является ли год указанной даты високосным.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на
 файл сценария PP.js.


// Создадим текущую дату
date = new Date();
// Определим, является ли год високосным
leap= PP.Date.isLeapYear(date);
console.log(leap ? "Год високосный" : "Год не високосный");
В результате выполнения примера было определено, является ли год високосным.
