# Date.add


## Синтаксис


PP.Date.add(date, interval, value): Date;


## Параметры


date. Date. Дата;


interval. [PP.Date.interval](Date.Interval.htm).
 Единица добавляемого интервала времени;


value. Number. Количество интервалов.


## Описание


Статический метод add добавляет
 к дате интервал времени.


## Комментарии


Метод возвращает объект типа Date.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на
 файл сценария PP.js.


// Создадим текущую дату
date = new Date();
// Добавим к текущей дате 5 дней
newDate = PP.Date.add(date, PP.Date.Interval.Day, 5);
console.log(newDate);
В результате выполнения примера была получена новая дата.
