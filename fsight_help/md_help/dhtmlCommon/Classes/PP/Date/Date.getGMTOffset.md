# Date.getGMTOffset


## Синтаксис


PP.Date.getGMTOffset(date, colon): String;


## Параметры


date. Date. Дата;


colon. Boolean. Требуется ли
 добавление разделителя для часов и минут (двоеточие). Необязательный параметр,
 по умолчанию False.


## Описание


Статический метод getGMTOffset
 возвращает отклонение от времени по Гринвичу.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на
 файл сценария PP.js.


// Создадим текущую дату
date = new Date();
// Получим отклонение от времени по Гринвичу
newDate = PP.Date.getGMTOffset(date, true);
console.log(newDate); // -> +05:00
В результате выполнения примера было получено отклонение по гринвичу.
