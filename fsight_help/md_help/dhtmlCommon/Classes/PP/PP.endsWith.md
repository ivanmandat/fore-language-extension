# PP.endsWith


## Синтаксис


endsWith(str: String, refStr: String);


## Параметры


*str.* Базовая строка;


*refStr.* Подстрока.


## Описание


Метод endsWith** определяет, оканчивается ли строка на указанную подстроку.


## Комментарии


Метод возвращает значение true, если строка оканчивается на указанную подстроку, иначе - значение false.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на
 файл сценария PP.js. Определим, на какую подстроку начинается и какой
 оканчивается базовая строка «Text example»:


// Зададим базовую строку
var baseStr = "Text example";
// Определим две подстроки
var subStr1 = "ple";
var subStr2 = "Tex";
// Определим, на какие подстроки начинается и оканчивается базовая строка
var isStarts = PP.startsWith(baseStr, subStr2) ? "начинается" : "не начинается";
console.log("Строка «" + baseStr + "» " + isStarts + " с подстроки «" + subStr2 + "».");
var isEnds = PP.endsWith(baseStr, subStr1) ? "оканчивается" : "не оканчивается";
console.log("Строка «" + baseStr + "» " + isEnds + " подстрокой «" + subStr1 + "».");

В результате выполнения примера в консоли браузера была выведена информация
 о том, на какую подстроку начинается и какой оканчивается базовая строка
 «Text example»:


Строка «Text example» начинается с подстроки «Tex».


Строка «Text example» оканчивается подстрокой «ple».
