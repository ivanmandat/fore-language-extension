# PP.NotSupportedException


## Синтаксис


NotSupportedException(message: String, self: Object);


## Параметры


*message.* Сообщение, передаваемое исключению. Необязательный параметр, по умолчанию значение не определено;


*self.* Источник исключения. Необязательный параметр, по умолчанию значение не определено.


## Описание


Метод NotSupportedException** представляет исключение, которое выбрасывается, если при каких-либо условиях метод не поддерживается.


## Комментарии


Метод возвращает значение типа String в следующем формате: «NotSupportedException. Source: [Источник исключения] Message: [Передаваемое сообщение]».
