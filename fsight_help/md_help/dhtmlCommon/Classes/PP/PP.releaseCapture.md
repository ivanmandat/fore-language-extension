# PP.releaseCapture


## Синтаксис


releaseCapture(domNode: HTMLElement);


## Параметры


*domNode.* DOM-элемент, у которого необходимо удалить все события. Необязательный параметр, значение по умолчанию не определено.


## Описание


Метод releaseCapture** удаляет сохранённое значение.


## Комментарии


Изначально значение запоминается с помощью метода [PP.setCapture](PP.setCapture.htm).
