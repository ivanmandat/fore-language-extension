# TimeAxis.insertAt


## Синтаксис


insertAt(index: Number, item: Number);


## Параметры


index. Позиция добавляемого элемента;


item. Значение добавляемого элемента.


## Описание


Метод insertAt добавляет элемент массива данных временной оси на позицию с указанным индексом.
