# Конструктор TimeAxis


## Синтаксис


PP.TimeAxis(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор TimeAxis создает экземпляр класса [TimeAxis](TimeAxis.htm).


## Пример


Конструктор предназначен для внутреннего использования.
