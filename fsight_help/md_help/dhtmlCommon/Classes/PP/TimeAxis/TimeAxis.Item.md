# TimeAxis.Item


## Синтаксис


Item: Number;


## Описание


Свойство Item определяет элемент массива данных временной оси.


## Комментарии


Значение свойства устанавливается при помощи метода setItem и возвращается при помощи метода getItem. Из JSON значение свойства установиться нельзя.


Метод setItem принимает следующие параметры: index - индекс элемента, для которого устанавливаются данные, value - значение элемента.


Метода getItem принимает параметр index - индекс возвращаемого элемента.
