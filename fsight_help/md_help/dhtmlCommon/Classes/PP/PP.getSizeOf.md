# PP.getSizeOf


## Синтаксис


getSizeOf(content: HTMLElement);


## Параметры


*content.* DOM-вершина, размеры которой требуется определить.


## Описание


Метод getSizeOf** определяет размеры DOM-вершины.


## Комментарии


Метод возвращает объект JSON-объект со следующими свойствами: Width и Height -- ширина и высота DOM-вершины.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на файл сценария PP.js. Создадим DOM-вершину из строки с разметкой и определим размеры данного элемента:


// Определим строку с разметкой
var markup = "<div id='main'><p>12</p></div>";
// Преобразуем разметку в DOM-вершину
var content = PP.htmlToDOM(markup);
// Определим размеры DOM-вершины
var size = PP.getSizeOf(content);
console.log("Ширина DOM-вершины: " + size.Width);
console.log("Высота DOM-вершины: " + size.Height);

В результате выполнения примера в консоли браузера были выведены ширина и высота созданной DOM-вершины:


Ширина DOM-вершины: 16

Высота DOM-вершины: 52
