# PP.show


## Синтаксис


show(element: HTMLElement);


## Параметры


*element.* DOM-элемент, который требуется отобразить.


## Описание


Метод show** отображает указанный DOM-элемент.


## Комментарии


Для отображения DOM-элемента необходимо, чтобы он был частью DOM-дерева всего документа.


Метод отображает элемент путём удаления у него CSS-класса PPHide.
