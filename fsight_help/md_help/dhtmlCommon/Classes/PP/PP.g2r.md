# PP.g2r


## Синтаксис


g2r(g: Number);


## Параметры


*g.* Угол в градусах.


## Описание


Метод g2r** переводит градусы в радианы.


## Комментарии


Метод возвращает значение типа Number.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылок на файлы сценария PP.js и PP.GraphicsBase.js. Пересчитаем размеры углов 45°, 60° и 90° в радианы:


// Определим функцию для перевода градусов в радианы
var toRadiant = function (g) {
    if (g != null) {
        if (PP.isNumber(g)) {
            /* Если параметр является числом,
               то преобразуем указанное значение в радианы */
            return PP.g2r(g);
        } else {
            /* Иначе выбросим ошибку с сообщением о недопустимом типе аргумента */
            throw PP.ArgumentException("Wrong type of argument", this);
        }
    } else {
        // Параметр функции не должен иметь значение null
        throw PP.ArgumentNullException("Argument can not be null", this);
    }
};
// Переведём градусы в радианы
console.log("45° = " + this.toRadiant(45) + " рад");
console.log("60° = " + this.toRadiant(60) + " рад");
console.log("90° = " + this.toRadiant(90) + " рад");

В результате выполнения примера размеры углов 45°, 60° и 90° были переведены в радианы:


45° = 0.7853981633974483 рад

60° = 1.0471975511965976 рад

90° = 1.5707963267948966 рад
