# PP.ArgumentException


## Синтаксис


ArgumentException(message: String, self: Object);


## Параметры


*message.* Сообщение, передаваемое исключению. Необязательный параметр, по умолчанию значение не определено;


*self.* Источник исключения. Необязательный параметр, по умолчанию значение не определено.


## Описание


Метод ArgumentException** представляет исключение, которое выбрасывается, если один из передаваемых функции параметров является недопустимым.


## Комментарии


Метод возвращает значение типа String в следующем формате: «ArgumentException. Source: [Источник исключения] Message: [Передаваемое сообщение]».
