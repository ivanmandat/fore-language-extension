# PP.calculateMouseY


## Синтаксис


calculateMouseY(event);


## Параметры


*event.* Событие мыши.


## Описание


Метод calculateMouseY** вычисляет ординату текущего положения курсора мыши.


## Комментарии


Метод возвращает значение типа Number.
