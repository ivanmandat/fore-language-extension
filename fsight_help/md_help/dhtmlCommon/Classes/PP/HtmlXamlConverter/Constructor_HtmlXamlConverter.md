# Конструктор HtmlXamlConverter


## Синтаксис


PP.HtmlXamlConverter (settings);


## Параметры


settings. JSON-объект со свойствами экземпляра класса.


## Описание


Конструктор HtmlXamlConverter создает экземпляр класса [HtmlXamlConverter](HtmlXamlConverter.htm).

Пример использования конструктора приведен в описании метода [HtmlXamlConverter.convertToHtml](HtmlXamlConverter.convertToHtml.htm).
