# IEffect.apply


## Синтаксис


apply (shape, doApply)


## Параметры


shape. Область карты, к которой
 будет применен эффект. Экземпляр класса [MapShape](dhtmlMapChart.chm::/Classes/map/mapshape/mapshape.htm);


doApply. Определяет, применяется
 ли эффект. Если установлено значение true,
 эффект применяется, если false
 - не применяется.


## Описание


Метод apply применяет эффект
 к определенной области карты.
