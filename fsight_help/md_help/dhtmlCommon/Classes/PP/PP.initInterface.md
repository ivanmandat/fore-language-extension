# PP.initInterface


## Синтаксис


initInterface(intrface: Function, name: String, baseIntarfaces: Array);


## Параметры


*interface.* Инициализируемый интерфейс;


*name.* Полное наименование интерфейса;


*baseIntarfaces.* Массив родительских интерфейсов.


## Описание


Метод initInterface** инициализирует интерфейс.
