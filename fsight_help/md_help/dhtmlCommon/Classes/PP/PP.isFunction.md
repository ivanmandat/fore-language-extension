# PP.isFunction


## Синтаксис


isFunction(v);


## Параметры


*v.* Исходное значение.


## Описание


Метод isFunction** проверяет, является ли указанное значение функцией.


## Комментарии


Метод возвращает значение true, если указанное значение является функцией, иначе - значение false.
