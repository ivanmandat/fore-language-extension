# PP.delFrom


## Синтаксис


delFrom(jsonObj: Object, arr: Array);


## Параметры


*jsonObj.* Объект, из которого нужно удалить свойства;


*arr.* Массив удаляемых свойств.


## Описание


Метод delFrom** удаляет массив указанных свойств из объекта.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на файл сценария PP.js. Определим объект и затем удалим у него некоторые свойства:


// Определим объект
var chartView = {
    chartType: "pie",
    editMode: 0,
    selectionEnabled: true,
    options: {}
};
// Удалим три свойства из объекта
PP.delFrom(chartView, ["editMode", "selectionEnabled", "options"]);
// Выведем все имеющиеся свойства у объекта
var result = "";
for (var i in chartView) {
    result += chartView[i];
    result += " "
};
console.log("Свойства объекта: " + result);

В результате выполнения примера у созданного объекта были удалены три его последних свойства из четырёх:


Свойства объекта: pie
