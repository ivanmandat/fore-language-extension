# Font.IsSuperior


## Синтаксис


IsSuperior: Boolean


## Описание


Свойство IsSuperior определяет,
 является ли текст надстрочным.


## Комментарии


Если установлено значение true,
 текст будет надстрочным.


По умолчанию установлено значение false.
