# Font.IsUnderline


## Синтаксис


IsUnderline: Boolean


## Описание


Свойство IsUnderline определяет,
 является ли текст подчеркнутым.


## Комментарии


Если установлено значение true,
 текст будет подчеркнутым.


По умолчанию установлено значение false.
