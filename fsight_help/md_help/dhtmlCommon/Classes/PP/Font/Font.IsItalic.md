# Font.IsItalic


## Синтаксис


IsItalic: Boolean


## Описание


Свойство IsItalic определяет,
 используется ли курсивное начертание.


## Комментарии


Если установлено значение true,
 будет использовано курсивное начертание.


По умолчанию установлено значение false.
