# Font.FontFamily


## Синтаксис


FontFamily: [PP.Font.Family](../../../Enums/Font.Family.htm)


## Описание


Свойство FontFamily устанавливает
 название шрифта.


## Комментарии


Доступные название шрифта содержатся в перечислении [Font.Family](../../../Enums/Font.Family.htm).


Задать название шрифта можно двумя способами:


	- FontFamily: "Arial";


	- FontFamily: PP.Font.FontFamily.Arial.
