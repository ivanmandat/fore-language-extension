# Font.IsLowerCapitalized


## Синтаксис


IsLowerCapitalized: Boolean


## Описание


Свойство IsLowerCapitalized
 определяет, используются ли малые прописные символы.


## Комментарии


Если установлено значение true,
 при начертании используются малые
 прописные символы.


По умолчанию установлено значение false.
