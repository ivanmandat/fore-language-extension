# Font.IsCapitalized


## Синтаксис


IsCapitalized: Boolean


## Описание


Свойство IsCapitalized определяет,
 являются ли все символы в тексте прописными.


## Комментарии


Если установлено значение true,
 при начертании используются только ПРОПИСНЫЕ СИМВОЛЫ.


По умолчанию установлено значение false.
