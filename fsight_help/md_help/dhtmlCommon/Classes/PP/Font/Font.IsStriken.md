# Font.IsStriken


## Синтаксис


IsStriken: Boolean


## Описание


Свойство IsStriken определяет,
 является ли текст зачеркнутым.


## Комментарии


Если установлено значение true,
 текст будет зачеркнутым.


По умолчанию установлено значение false.
