# Font.IsBold


## Синтаксис


IsBold: Boolean


## Описание


Свойство IsBold определяет,
 используется ли полужирное начертание.


## Комментарии


Если установлено значение true,
 будет использовано полужирное начертание.


По умолчанию установлено значение false.
