# Font.IsInferior


## Синтаксис


IsInferior: Boolean


## Описание


Свойство IsInferior определяет,
 является ли текст подстрочным.


## Комментарии


Если установлено значение true,
 текст будет подстрочным.


По умолчанию установлено значение false.
