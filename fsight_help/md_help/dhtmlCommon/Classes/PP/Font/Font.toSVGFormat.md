# Font.toSVGFormat


## Синтаксис


toSVGFormat(element: SVGElement)


## Параметры


element. SVG-элемент, для которого
 требуется применить настройки шрифта.


## Описание


Метод toSVGFormat устанавливает
 настройки шрифта для указанного SVG-элемента.
