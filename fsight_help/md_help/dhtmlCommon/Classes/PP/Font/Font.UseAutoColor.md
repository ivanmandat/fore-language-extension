# Font.UseAutoColor


## Синтаксис


UseAutoColor: Boolean


## Описание


Свойство UseAutoColor определяет,
 используется ли цвет по умолчанию.


## Комментарии


Цвет по умолчанию - черный ("#000000").


Если для свойства установлено значение true,
 используется цвет шрифта по умолчанию, при этом значение свойства [Font.Color](Font.Color.htm)
 будет неактуальным.


По умолчанию установлено значение false,
 то есть используются цвета заданные в свойстве [Font.Color](Font.Color.htm).
