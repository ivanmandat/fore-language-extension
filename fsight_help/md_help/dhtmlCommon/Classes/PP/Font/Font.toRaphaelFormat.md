# Font.toRaphaelFormat


## Синтаксис


toRaphaelFormat(element: Element);


## Параметры


element. Raphael-объект, для
 которого требуется применить настройки шрифта.


## Описание


Метод toRaphaelFormat устанавливает
 настройки шрифта для указанного Raphael-объекта.
