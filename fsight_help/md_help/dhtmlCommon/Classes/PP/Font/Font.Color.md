# Font.Color


## Синтаксис


Color: String


## Описание


Свойство Color устанавливает
 цвет шрифта.


## Комментарии


Значение свойства может задаваться в следующих форматах:


	- "red";


	- "#000";


	- "RGB(123, 12, 1)".


Свойство неактуально, если для свойства [Font.UseAutoColor](Font.UseAutoColor.htm)
 установлено значение true.
