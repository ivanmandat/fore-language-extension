# PP.insertToArr


## Синтаксис


insertToArr(arr: Array, item, index: Number);


## Параметры


*arr.* Исходный массив элементов;


*item.* Элемент, который нужно вставить в массив;


*index.* Элемент, перед которым необходимо осуществить вставку.


## Описание


Метод insertToArr** вставляет элемент в массив перед указанным элементом.


## Комментарии


Метод возвращает значение true, если элемент был успешно вставлен в массив, иначе - значение false.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на файл сценария PP.js. Определим массив целых чисел и вставим перед каким-либо элементом новое значение:


// Определим массив целых чисел
var numbers = [5, 9, 6, 7, 2];
console.log("Исходный массив: " + numbers);
// Вставим значение 0 перед элементом 6
PP.arrInsertBefore(numbers, 0, 6);
console.log("Массив после вставки в него элемента: " + numbers);

В результате выполнения примера в исходный массив перед элементом 6 было вставлено значение 0:


Исходный массив: 5,9,6,7,2

Массив после вставки в него элемента: 5,9,0,6,7,2
