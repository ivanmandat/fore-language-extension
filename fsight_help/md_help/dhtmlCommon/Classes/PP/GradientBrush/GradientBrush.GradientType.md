# GradientBrush.GradientType


## Синтаксис


GradientType: Number


## Описание


Свойство GradientType
 определяет направление градиента кисти.


## Комментарии


Свойство применяется для определения направления градиента в браузере
 Internet Explorer.


Допустимые значения*:*


	- 0.
	 Отображается вертикальный градиент (по умолчанию);


	- 1.
	 Отображается горизонтальный градиент.
