# PP.isString


## Синтаксис


isString(v);


## Параметры


*v.* Исходное значение.


## Описание


Метод isString** проверяет, является ли указанное значение строкой.


## Комментарии


Метод возвращает значение true, если указанное значение является строкой, иначе - значение false.
