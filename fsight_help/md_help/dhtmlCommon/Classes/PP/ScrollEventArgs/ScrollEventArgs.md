## Описание


Класс ScrollEventArgs представляет
 собой класс аргументов событий перемещения бегунка по [полосе
 прокрутки](dhtmlUi.chm::/Components/ScrollBar/ScrollBar.htm).


## Синтаксис


PP.initClass(PP.ScrollEventArgs, PP.[Object](../../Object/Object.htm),
 "ScrollEventArgs");
