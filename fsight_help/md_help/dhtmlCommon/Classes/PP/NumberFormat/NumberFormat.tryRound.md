# NumberFormat.tryRound


## Синтаксис


PP.NumberFormat.tryRound (value);


## Параметры


value. Дробное число.


## Описание


Статический метод tryRound округляет дробные числа.


## Пример


Для выполнения примера предполагается наличие на странице ссылки на библиотеку PP.js. Чтобы округлить дробное число, используйте метод tryRound:


PP.NumberFormat.tryRound (1,234);


Результат:


1


Примечание. Метод округляет числа только с запятой в качестве десятичного разделителя.
