# PP.setCurrentCulture


## Синтаксис


setCurrentCulture(value: PP.CultureInfo);


## Параметры


*value.* Язык и региональные
 параметры.


## Описание


Метод setCurrentCulture**
 устанавливает указанный язык и региональные параметры в качестве
 текущих.
