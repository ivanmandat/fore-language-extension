# PP.isPPInstance


## Синтаксис


isPPInstance(instance: Function);


## Параметры


*instance.* Экземпляр класса.


## Описание


Метод isPPInstance** проверяет, является ли объект экземпляром класса, проинициализированным с помощью метода [PP.initClass](PP.initClass.htm).


## Комментарии


Метод возвращает значение true, если указанный объект является экземпляром класса, проинициализированным с помощью метода [PP.initClass](PP.initClass.htm), и значение false, если не является.
