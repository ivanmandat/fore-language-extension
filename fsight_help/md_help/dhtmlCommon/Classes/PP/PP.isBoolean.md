# PP.isBoolean


## Синтаксис


isBoolean(v);


## Параметры


*v.* Исходное значение.


## Описание


Метод isBoolean** проверяет, является ли тип указанного значения булевым.


## Комментарии


Метод возвращает значение true, если тип указанного значения является булевым, иначе - значение false.
