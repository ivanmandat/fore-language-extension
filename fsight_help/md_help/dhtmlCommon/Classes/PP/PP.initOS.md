# PP.initOS


## Синтаксис


initOS(domNode: HTMLElement);


## Параметры


*domNode.* DOM-элемент.


## Описание


Метод initOS** добавляет к
 указанной вершине CSS-класс, соответствующий операционной системе клиента.


## Комментарии


Метод возвращает DOM-вершину, являющуюся значением типа HTMLElement.
