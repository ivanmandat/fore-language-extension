# PP.trim


## Синтаксис


trim(string: String);


## Параметры


*string.* Текст, требующий форматирования.


## Описание


Метод trim** удаляет пробелы из начала и конца строки.


## Комментарии


Метод возвращает значение типа String.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на файл сценария PP.js. Определим строку с пробелами и удалим из них только начальный и конечный:


// Определим строку с пробелами
var str = " pro gnoz ";
console.log("Исходная строка: «" + str + "»");
// Удалим пробелы из начала и конца строки
str = PP.trim(str);
console.log("Строка с удалёнными начальным и конечным пробелами: «" + str + "»");

В результате выполнения примера в консоли браузера были выведены исходная строка и строка без начального и конечного пробела:


Исходная строка: « pro gnoz »

Строка с удалёнными начальным и конечным пробелами: «pro gnoz»
