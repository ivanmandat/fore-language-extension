# PP.isDefined


## Синтаксис


isDefined(v);


## Параметры


*v.* Исходное значение.


## Описание


Метод isDefined** проверяет, определено ли указанное значение.


## Комментарии


Метод возвращает значение true, если указанное значение не определено, иначе - значение false.
