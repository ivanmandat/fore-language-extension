# ZoomOptions.getPointOffset


## Синтаксис


getPointOffset(x: number, y: number, node: HTMLElement);


## Параметры


x. Текущее положение DOM-элемента
 по горизонтали;


y. Текущее положение DOM-элемента
 по вертикали;


node. DOM-элемент, соответствующий
 визуальному компоненту, относительного которого необходимо получить координаты
 элемента.


## Описание


Метод getPointOffset возвращает
 координаты элемента, расположенного относительно визуального компонента,
 с учётом масштаба в пикселях.


## Комментарии


Для получения координат визуального компонента используйте метод [ZoomOptions.getNodeOffset](ZoomOptions.getNodeOffset.htm).
