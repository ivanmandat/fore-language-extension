# ZoomOptions.getNodeOffset


## Синтаксис


getNodeOffset(node: HTMLElement);


## Параметры


node. DOM-элемент, соответствующий
 визуальному компоненту, координаты которого необходимо получить.


## Описание


Метод getNodeOffset возвращает
 координаты визуального компонента, расположенного в видимой области браузера,
 с учётом масштаба в пикселях.


## Комментарии


Для получения координат элемента, расположенного относительно визуального
 компонента, используйте метод [ZoomOptions.getPointOffset](ZoomOptions.getPointOffset.htm).
