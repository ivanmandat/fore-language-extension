# Конструктор ImageBrush


## Синтаксис


PP.ImageBrush(settings);


## Параметры


settings. JSON-объект со значениями свойств экземпляра класса.


## Описание


Конструктор ImageBrush создает экземпляр класса [ImageBrush](ImageBrush.htm).
