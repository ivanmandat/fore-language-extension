# PP.isObject


## Синтаксис


isObject(v);


## Параметры


*v.* Исходное значение.


## Описание


Метод isObject** проверяет, является ли указанное значение объектом.


## Комментарии


Метод возвращает значение true, если указанное значение является объектом, иначе - значение false.
