# PP.hasClass


## Синтаксис


hasClass(element: HTMLElement, value: String);


## Параметры


*element.* Элемент разметки страницы;


*value.* Наименование CSS-класса.


## Описание


Метод hasClass** указывает, есть ли в заданном элементе разметки указанный CSS-класс.


## Комментарии


Метод возвращает значение true, если в заданном элементе разметки существует указанный CSS-класс, иначе - значение false.
