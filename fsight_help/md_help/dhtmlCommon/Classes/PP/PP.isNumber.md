# PP.isNumber


## Синтаксис


isNumber(v, tryParse: Boolean);


## Параметры


*v.* Исходное значение;


*tryParse.* Признак преобразования указанного значения в число с плавающей точкой. Если параметр равен true, то будет выполнена попытка преобразовать заданное значение в число с плавающей точкой, иначе - не будет.


## Описание


Метод isNumber** проверяет, является ли указанное значение числом.


## Комментарии


Метод возвращает значение true, если указанное значение является числом, иначе - значение false.
