# Style.Border


## Синтаксис


Border: PP.Border


## Описание


Свойство Border** устанавливает и возвращает границу для компонента.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setBorder, а возвращается - с помощью метода getBorder.
