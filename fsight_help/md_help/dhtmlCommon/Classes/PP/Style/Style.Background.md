# Style.Background


## Синтаксис


Background: [PP.SolidColorBrush](../SolidColorBrush/SolidColorBrush.htm)


## Описание


Свойство Background** устанавливает и возвращает кисть со сплошной заливкой.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setStyle, а возвращается - с помощью метода getStyle.
