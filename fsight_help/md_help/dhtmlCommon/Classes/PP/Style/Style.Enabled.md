# Style.Enabled


## Синтаксис


Enabled: Boolean


## Описание


Свойство Enabled** определяет возможность применения стиля для компонента.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setEnabled, а возвращается - с помощью метода getEnabled.


Свойство содержит значение true, если применение стиля для компонента доступно, и значение false - в противном случае.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылок на файлы сценария jquery.js, PP.js и файл стилей PP.css. Добавим в документ текстовую область и установим для неё новый стиль, однако применение данного стиля запретим:


// Создадим текстовую область
var textArea = new PP.Ui.TextArea({
    Width: 200,
    Height: 100,
    Content: "2013" // Содержимое
});
// Создадим стиль для текстовой области
var style = {
    Release: new PP.Style({
        // Создадим кисть со сплошной заливкой
        Background: new PP.SolidColorBrush({
            Color: PP.Color.Colors.lightblue
        }),
        Enabled: false // Запретим использование стиля
    })
};
// Установим стиль
textArea.setStyle(style);
// Добавим данную область в документ
textArea.addToNode(document.body);

В результате выполнения примера в документе была создана текстовая область, установлен стиль, окрашивающий область голубым цветом, однако данный стиль применён не был:
