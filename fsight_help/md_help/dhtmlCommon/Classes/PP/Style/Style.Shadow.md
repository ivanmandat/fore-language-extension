# Style.Shadow


## Синтаксис


Shadow: PP.Shadow


## Описание


Свойство Shadow** устанавливает и возвращает тень для компонента.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setShadow, а возвращается - с помощью метода getShadow.
