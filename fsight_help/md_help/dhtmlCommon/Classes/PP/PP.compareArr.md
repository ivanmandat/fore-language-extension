# PP.compareArr


## Синтаксис


compareArr(arr1: Array, arr2: Array);


## Параметры


*arr1.* Исходный массив;


*arr2.* Сравниваемый массив.


## Описание


Метод compareArr** сравнивает два массива.


## Комментарии


Метод возвращает значение true, если количество и значения элементов указанных массивов равны, иначе - значение false.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на файл сценария PP.js. Определим три массива целых чисел и сравним их:


// Определим массивы целых чисел
var numbers1 = [5, 9, 6];
var numbers2 = [5, 9, 6, 7];
var numbers3 = [5, 9, 4];
console.log("Массивы [" + numbers1 + "] и [" + numbers2 + "] " + (PP.compareArr(numbers1, numbers2)? "равны.": "не равны."));
console.log("Массивы [" + numbers1 + "] и [" + numbers3 + "] " + (PP.compareArr(numbers1, numbers3)? "равны.": "не равны."));
console.log("Массивы [" + numbers1 + "] и [" + numbers1 + "] " + (PP.compareArr(numbers1, numbers1)? "равны.": "не равны."));

В результате выполнения примера в консоли браузера были выведены результаты сравнения трех массивов и одного с самим собой:


Массивы [5,9,6] и [5,9,6,7] не равны.

Массивы [5,9,6] и [5,9,4] не равны.

Массивы [5,9,6] и [5,9,6] равны.
