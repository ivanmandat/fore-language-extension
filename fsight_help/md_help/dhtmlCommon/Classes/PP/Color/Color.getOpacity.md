# Color.getOpacity


## Синтаксис


getOpacity();


## Описание


Метод getOpacity возвращает
 значение прозрачности от 0 до 1.


## Пример


Для выполнения примера предполагается наличие компонента [ColorCombo](dhtmlUi.chm::/components/colorcombo/colorcombo.htm)
 с наименованием «colCombo» (см. «[Пример
 создания компонента ColorCombo](dhtmlUi.chm::/components/colorcombo/example_colorcombo.htm)»). Выведем в javascript-консоль значение
 прозрачности текущего цвета:


console.log(colCombo.getColor().getOpacity()


После выполнения примера в консоль браузера будет выведено значение
 прозрачности текущего цвета компонента [ColorCombo](dhtmlUi.chm::/components/colorcombo/colorcombo.htm).
