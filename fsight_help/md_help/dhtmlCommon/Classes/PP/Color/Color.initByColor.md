# Color.initByColor


## Синтаксис


initByColor(color)


## Параметры


color. Цвет, экземпляр [PP.Color](Color.htm).


## Описание


Метод initByColor применяет
 указанный в параметрах цвет к текущему.


## Пример


Для выполнения примера откройте любую страницу с подключенным скриптом
 PP.js. Создадим два новых цвета и скопируем один в другой:


// Создаем новый цвет
c1 = new PP.Color;
// Зададим цвет
c1.setR(200);
c1.setG(230);
c1.setB(150);
// Создаем второй цвет
c2 = new PP.Color;
c2.setR(0);
c2.setG(0);
c2.setB(0);
// Применим первый цвет ко второму
c2.initByColor(c1);
// Получим значение второго цвета
c2;
В результате выполнения примера был получен цвет с2:


PP.Color {_R: 200, _G: 230, _B: 150,
 _A: 255, _H: 0…}
