# Color.brighten


## Синтаксис


brighten(alpha: Number);


## Параметры


alpha. Коэффициент насыщенности
 цвета. Параметр может принимать значение в интервале от 0 (наиболее насыщенный
 цвет) до 1 (наименее насыщенный цвет).


## Описание


Метод brighten изменяет насыщенность
 цвета.

Метод устарел, используйте [Color.brighter](Color.brighter.htm).
