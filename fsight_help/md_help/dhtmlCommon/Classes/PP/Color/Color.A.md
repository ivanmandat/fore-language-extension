# Color.A


## Синтаксис


A: Number


## Описание


Свойство A устанавливает степень
 прозрачности цвета от 0 до 1.


## Комментарии


1- цвет абсолютно непрозрачен, 0 - цвет абсолютно прозрачен.


Значение свойства устанавливается при помощи метода setA
 и возвращается при помощи метода getA.
