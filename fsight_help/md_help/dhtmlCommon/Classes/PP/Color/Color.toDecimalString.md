# Color.toDecimalString


## Синтаксис


toDecimalString(includeAlpha)


## Параметры


includeAlpha. Определяет, выводится ли значение [прозрачности](Color.getOpacity.htm).


## Описание


Метод toDecimalString возвращает строку со значением цвета в формате RGB/RGBA.


## Комментарии


Если для параметра задано значение true, то строка будет иметь вид «rgba(0, 255, 0, 1)».


Если для параметра установлено значение false (по умолчанию), строка будет иметь вид «rgb(0, 255, 0)».
