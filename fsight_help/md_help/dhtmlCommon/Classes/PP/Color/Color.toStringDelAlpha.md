# Color.toStringDelAlpha


## Синтаксис


toStringDelAplha()


## Описание


Метод toStringDelAlpha возвращает
 строку со значением цвета, с учетом прозрачности, в шестнадцатиричном
 формате со знаком решетки в начале.


## Комментарии


Метод возвращает значение цвета, который видит пользователь с учетом
 прозрачности.


## Пример


Для выполнения примера откройте любую страницу, с подключенным скриптом
 PP.js. Создадим новый цвет и получим его строковое значение в шестнадцатиричном
 формате:


// Создаем новый цвет
c1 = new PP.Color;
// Зададим прозрачность и цвета
c1.setA(100);
c1.setR(200);
c1.setG(200);
c1.setB(150);
// Получим в консоли цвет в виде объекта
console.debug(c1);
// Получим стандартное строковое значение
console.debug(c1.toString(true));
// Получим строковое значение, с учетом прозрачности
console.debug(c1.toStringDelAlpha());
В результате выполнения примера было получено 3 разных представления
 одного цвета.


В виде JSON объекта:


PP.Color {_A: 100, _R: 200, _G: 200,
 _B: 150}


В стандартном строковом значении с записанной отдельно прозрачностью:


"#64C8C896"


В строковом значении с учетом прозрачности:


"#E9E9D6"
