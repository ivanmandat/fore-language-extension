# Color.darken


## Синтаксис


darken(alpha: Number);


## Параметры


alpha. Прозрачность цвета.
 Параметр может принимать значение в интервале от 0 до 1.


## Описание


Метод darken затемняет цвет
 на указанное значение прозрачности.
