# String.leftPad


## Синтаксис


PP.String.leftPad(string: String, size: Number,
 character: String);


## Параметры


string. Исходная строка;


size. Минимальная длина строки;


character. Символ, которым
 нужно заполнить пустое пространство слева. Необязательный параметр, по
 умолчанию используется пробел.


## Описание


Статический метод leftPad добавляет
 недостающее количество символов в начало строки.


## Комментарии


Метод возвращает значение типа String.
