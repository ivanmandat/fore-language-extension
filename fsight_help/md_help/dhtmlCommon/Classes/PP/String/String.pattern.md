# String.pattern


## Синтаксис


PP.String.pattern(string: String, replacers: Object,
 caseIndifferent: Boolean, noDataText: String);


## Параметры


string. Шаблон для подстановки,
 содержащий переменные вида %Value1, %Value2 и т. д.;


replacers. JSON-объект с подставляемыми
 значениями, наименования свойств которого совпадают с именами переменных;


caseIndifferent. Признак нечувствительности
 к регистру. Если параметр равен значению true
 (по умолчанию), то при выполнении данного метода регистр букв не учитывается,
 иначе - учитывается. Необязательный параметр;


noDataText. Текст для замены
 значений null или undefined. Необязательный параметр.


## Описание


Статический метод pattern заменяет
 переменные в строке значениями.


## Комментарии


Метод возвращает значение типа String.
