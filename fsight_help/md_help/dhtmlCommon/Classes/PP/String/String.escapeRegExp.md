# String.escapeRegExp


## Синтаксис


PP.String.escapeRegExp(text: String);


## Параметры


text. Текст, требующий экранирования.


## Описание


Статический метод escapeRegExp
 экранирует спецсимволы для использования строки в регулярном выражении.


## Комментарии


Метод возвращает значение типа String.
