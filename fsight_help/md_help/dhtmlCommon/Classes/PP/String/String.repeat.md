# String.repeat


## Синтаксис


PP.String.repeat(pattern: String, count: Number,
 sep: String);


## Параметры


pattern. Шаблон для повторения;


count. Количество повторений;


sep. Разделитель между повторяющимися
 подстроками. Необязательный параметр, по умолчанию подставляется пустая
 строка.


## Описание


Статический метод repeat повторяет
 строку нужное количество раз, при необходимости добавляя разделитель между
 повторяющимися подстроками.


## Комментарии


Метод возвращает значение типа String.
