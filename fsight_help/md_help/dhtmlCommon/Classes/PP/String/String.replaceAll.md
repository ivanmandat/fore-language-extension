# String.replaceAll


## Синтаксис


PP.String.replaceAll(str: String, whatReplace: String,
 thanReplace: String);


## Параметры


str. Строка, в которой необходимо
 произвести замену;


whatReplace. Заменяемая подстрока;


thanReplace. Значение, на которое
 заменяют исходную подстроку.


## Описание


Статический метод replaceAll
 осуществляет замену всех указанных вхождений в строке на заданную подстроку.


## Комментарии


Метод возвращает значение типа String
 - строку, в которой осуществлена замена.
