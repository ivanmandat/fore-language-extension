# String.getRandom


## Синтаксис


PP.String.getRandom(length: Number);


## Параметры


length. Длина строки, которую
 необходимо сформировать.


## Описание


Статический метод getRandomString**
 возвращает строку указанной длины из случайных символов латинского алфавита.


## Комментарии


Метод возвращает значение типа String.


Примечание.
 Данный метод заменяет устаревший метод [PP.getRandomString](../PP.getRandomString.htm).


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на
 файл сценария PP.js. Сгенерируем строку из трёх случайных символов латинского
 алфавита и продублируем её три раза, используя в качестве разделителя
 дефис:


// Зададим строку из трёх символов латинского алфавита
var result = PP.String.getRandom(3);
console.log("Случайная строка: «" + result + "»");
// Повторим строку 3 раза через пробел
result = PP.String.repeat(result, 3, "-");
console.log("Полученная строка: «" + result + "»");
В результате выполнения примера в консоли браузера была выведена случайно
 сгенерированная строка из трёх латинских символов, а также её представление
 после троекратного дублирования через дефис:


Случайная строка: «bfc»


Полученная строка: «bfc-bfc-bfc»
