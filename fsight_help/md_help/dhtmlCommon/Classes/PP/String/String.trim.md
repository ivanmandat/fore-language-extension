# String.trim


## Синтаксис


PP.String.trim(string: String);


## Параметры


string. Текст, требующий форматирования.


## Описание


Статический метод trim удаляет
 пробелы из начала и конца строки.


## Комментарии


Метод возвращает значение типа String.


Примечание.
 Данный метод заменяет устаревший метод [PP.trim](../PP.trim.htm).


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на
 файл сценария PP.js. Определим строку с пробелами и удалим из них только
 начальные и конечные:


// Зададим исходную строку
var sourceStr = "  abc ";
console.log("Исходная строка: «" + sourceStr + "»");
// Получим строку без начальных и конечных пробелов
var result = PP.String.trim(sourceStr);
console.log("Строка без начальных и конечных пробелов: «" + result + "»");
В результате выполнения примера в консоли браузера были выведены исходная
 строка и строка без начальных и конечных пробела:


Исходная строка: « abc »


Строка без начальных и конечных пробелов: «abc»
