# String.unescapeHTML


## Синтаксис


PP.String.unescapeHTML(text: String);


## Параметры


*text.* Текст, требующий разэкранирования.


## Описание


Статический метод unescapeHTML**
 осуществляет разэкранирование символов HTML-разметки в указанной строке.


## Комментарии


Метод возвращает значение типа String.


Экранирование управляющих символов осуществляется с помощью метода [PP.String.escapeHTML](String.escapeHTML.htm).


Примечание.
 Данный метод заменяет устаревший метод [PP.unescapeHTML](../PP.unescapeHTML.htm).
