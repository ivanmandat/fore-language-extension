# PP.startsWith


## Синтаксис


startsWith(str: String, refStr: String);


## Параметры


*str.* Базовая строка;


*refStr.* Подстрока.


## Описание


Метод startsWith** определяет, начинается ли строка на указанную подстроку.


## Комментарии


Метод возвращает значение true, если строка начинается с указанной подстроки, иначе - значение false.
