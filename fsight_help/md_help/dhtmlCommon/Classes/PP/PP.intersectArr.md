# PP.intersectArr


## Синтаксис


intersectArr(arr1: Number, arr2: Number);


## Параметры


*arr1.* Исходный массив;


*arr2.* Массив, пересекающийся
 с исходным.


## Описание


Метод intersectArr** определяет
 элементы пересечения двух массивов.


## Комментарии


Метод возвращает значение типа Array.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на
 файл сценария PP.js. Создадим два массива целых чисел и определим их пересекающиеся
 элементы:


// Определим массивы целых чисел
var num1 = [5, 1, 9, 6];
var num2 = [2, 9, 6, 7, 5];
// Определим элементы пересечения данных массивов
var arr = PP.intersectArr(num1, num2);
if (arr.length > 0) {
    console.log("Элементы пересечения массивов [" + num1 + "] и [" + num2 + "]: [" + arr + "].");
} else {
    console.log("Массивы [" + num1 + "] и [" + num2 + "] не пересекаются.");
};

В результате выполнения примера в консоли браузера были выведены элементы
 пересечения двух заданных массивов:


Элементы пересечения массивов [5,1,9,6] и [2,9,6,7,5]:
 [5,9,6].
