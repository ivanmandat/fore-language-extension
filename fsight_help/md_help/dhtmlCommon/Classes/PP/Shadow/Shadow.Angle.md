# Shadow.Angle


## Синтаксис


Angle: Number;


## Описание


Свойство Angle определяет угол падения тени.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setAngle, а возвращается с помощью метода getAngle.**
