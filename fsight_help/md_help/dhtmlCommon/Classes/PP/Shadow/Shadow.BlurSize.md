# Shadow.BlurSize


## Синтаксис


BlurSize: Number;


## Описание


Свойство BlurSize определяет размер размытия тени.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setBlurSize, а возвращается с помощью метода getBlurSize.**
