# Shadow.EnableOpacity


## Синтаксис


EnableOpacity: Boolean;


## Описание


Свойство EnableOpacity определяет признак использования прозрачности.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setEnableOpacity, а возвращается с помощью метода getEnableOpacity.**


**Е****сли свойство имеет значение true, то тень может быть прозрачной, иначе - не может.**
