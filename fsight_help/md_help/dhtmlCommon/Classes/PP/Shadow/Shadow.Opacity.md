# Shadow.Opacity


## Синтаксис


Opacity: Number;


## Описание


Свойство Opacity определяет
 прозрачность тени.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setOpacity,
 а возвращается с помощью метода getOpacity.**


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 [TextArea](dhtmlUi.chm::/Components/TextArea/TextArea.htm)
 с наименованием «textArea», для которого установлена тень (см. «[Конструктор
 Shadow](Constructor_Shadow.htm)»). Установим прозрачность и новое положение тени, выведем настройки
 тени контейнера и текста в формате CSS:


// Получим параметры тени
var shadow = textArea.getStyle(PP.ControlState.Release).getShadow();
// Установим угол падения тени в 0
shadow.setAngle(0);
// Установим прозрачность тени
shadow.setOpacity(0.6);
// Установим новое положение тени
shadow.setX(15);
shadow.setY(15);
// Обновим стиль текстовой области
textArea.refreshStyle();
// Выведем строку в формате CSS с настройками тени контейнера
console.log("Настройки тени контейнера в формате CSS: " + shadow.toCSSFormat());
// Выведем строку строку в формате CSS с настройками тени текста
console.log("Настройки тени текста в формате CSS: " + shadow.toCSSTextShadowFormat());
В результате выполнения примера была изменена прозрачность и положение
 тени:
