# Shadow.EnableBlur


## Синтаксис


EnableBlur: Boolean;


## Описание


Свойство EnableBlur определяет признак использования размытия.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setEnableBlur, а возвращается с помощью метода getEnableBlur.**


**Если свойство имеет значение true, то размытие будет использоваться, иначе - не будет.**
