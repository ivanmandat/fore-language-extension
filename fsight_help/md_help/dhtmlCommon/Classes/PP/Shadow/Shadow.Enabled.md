# Shadow.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет признак активности тени.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setEnabled, а возвращается с помощью метода getEnabled.**


**Е****сли свойство имеет значение true, то тень будет активна, иначе - не будет.**
