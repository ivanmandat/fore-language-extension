# Shadow.Distance


## Синтаксис


Distance: Number;


## Описание


Свойство Distance определяет сдвиг тени относительно объекта.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDistance, а возвращается с помощью метода getDistance.**
