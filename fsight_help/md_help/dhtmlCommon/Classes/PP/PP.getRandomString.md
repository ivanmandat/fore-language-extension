# PP.getRandomString


## Синтаксис


getRandomString(length: Number);


## Параметры


*length.* Длина строки, которую необходимо сформировать.


## Описание


Метод getRandomString** возвращает строку указанной длины из случайных символов латинского алфавита.


## Комментарии


Метод возвращает значение типа String.
