# Rect.isPointInBox


## Синтаксис


isPointInBox(x: Number, y: Number);


## Параметры


x. Координата x точки;


y. Координата y точки.


## Описание


Метод isPointInBox проверяет
 попадание точки в область прямоугольника.


## Комментарии


Метод возвращает значение true,
 если точка попадает в область прямоугольника, иначе - значение false.


## Пример


Для выполнения примера необходимо наличие в теге <head> html-страницы
 ссылки на файл сценария PP.js. Приведенный ниже код выполняется в консоли
 браузера. Создадим прямоугольник с высотой и шириной 10 пикселей, верхний
 левый угол которого располагается по координатам (10, 10).


// Создаем прямоугольник
var rect = new PP.Rect("10,10,10,10");
Проверим, попадает ли точка с координатами (20, 20) в область прямоугольника:


// Проверяем, попадает ли точка с координатами (20, 20) в область прямоугольника
if (rect.isPointInBox(20, 20)) {
    console.log("Точка находится в области прямоугольника");
} else {
    console.log("Точка находится за пределами прямоугольника");
}
В результате в консоль будет выведен результат проверки попадания точки
 в область прямоугольника:


Точка находится в области прямоугольника
