# Rect.Top


## Синтаксис


Top: Number


## Описание


Свойство Top устанавливает значение
 координаты y левого верхнего угла прямоугольника.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTop,
 а возвращается с помощью метода getTop.
