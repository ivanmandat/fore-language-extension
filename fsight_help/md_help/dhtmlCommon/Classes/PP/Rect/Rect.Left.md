# Rect.Left


## Синтаксис


Left: Number


## Описание


Свойство Left устанавливает
 значение координаты x левого верхнего угла прямоугольника.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLeft,
 а возвращается с помощью метода getLeft.
