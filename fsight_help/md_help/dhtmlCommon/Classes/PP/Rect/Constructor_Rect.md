# Конструктор Rect


## Синтаксис


PP.Rect(string || object)


## Параметры


string. Текстовое
 представление параметров прямоугольника. Строка должна содержать через
 запятую: x-координату левого верхнего левого угла прямоугольника, y-координату
 левого верхнего угла прямоугольника, ширину и высоту.


object.
 Объект с настройками класса.


## Описание


Конструктор Rect создает экземпляр
 класса [Rect](Rect.htm).


## Комментарии


В качестве параметра, вместо строки, может
 принимать объект, содержащий свойства класса, например:


// Создаем прямоугольник
var rect =  new PP.Rect({
  Height: 10,
  Width: 10,
  Left: 10,
  Top: 10
});
