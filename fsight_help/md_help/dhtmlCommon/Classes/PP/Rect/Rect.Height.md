# Rect.Height


## Синтаксис


Height: Number


## Описание


Свойство Height устанавливает
 значение высоты прямоугольника.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHeight,
 а возвращается с помощью метода getHeight.
