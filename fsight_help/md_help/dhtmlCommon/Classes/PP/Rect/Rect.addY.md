# Rect.addY


## Синтаксис


addY(y: Number);


## Параметры


y. Величина растягивания.


## Описание


Метод addX растягивает
 прямоугольник по вертикали на заданную величину.


## Пример


Для выполнения примера необходимо наличие в теге <head> html-страницы
 ссылки на файл сценария PP.js. Приведенный ниже код выполняется в консоли
 браузера. Создадим прямоугольник с высотой и шириной 10 пикселей, верхний
 левый угол которого располагается по координатам (10, 10).


// Создаем прямоугольник
var rect = new PP.Rect("10,10,10,10");
Растянем прямоугольник вверх и вниз на 5 пикселей:


// Растягиваем прямоугольник
rect.addY(5);
// Выводим значение координаты y левого верхнего угла и высоту прямоугольника
console.log('Координата y левого верхнего угла прямоугольника: ' + rect.getTop());
console.log('Высота прямоугольника: ' + rect.getHeight());
В результате в консоль будут выведены значения координаты y левого верхнего
 угла и высота прямоугольника:


Координата y левого верхнего угла прямоугольника:
 5


Высота прямоугольника: 15
