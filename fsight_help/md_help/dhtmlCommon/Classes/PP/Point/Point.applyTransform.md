# Point.applyTransform


## Синтаксис


applyTransform(v: Array);


## Параметры


v. Матрица преобразования.


## Описание


Метод applyTransform предназначен
 для трансформации координат точки.


## Комментарии


Матрица преобразования должна содержать 6 элементов.


## Пример


Для выполнения примера необходимо наличие в теге <head> html-страницы
 ссылки на файл сценария PP.js. Приведенный ниже код выполняется в консоли
 браузера. Создадим точку с координатам (5, 5) и трансформируем её координаты
 с помощью матрицы: [2,0,0,2,0,0]:


// Создаем точку с координатами (5,5)
var point = new PP.Point(5, 5);
// Трансформируем координаты:
point.applyTransform([2,0,0,2,0,0]);
// Выводим результат преобразования
console.log("Координата X: " + point.getX());
console.log("Координата Y: " + point.getY());
В результате в консоль будут выведены новые координаты точки:


Координата X: 10


Координата Y: 10
