# Конструктор Point


## Синтаксис


PP.Point(strX, Y)


## Параметры


[strX](Point.X.htm). Левая координата точки;


[Y](Point.Y.htm). Верхняя координата точки.


## Описание


Конструктор Point создает экземпляр класса [Point](Point.htm).
