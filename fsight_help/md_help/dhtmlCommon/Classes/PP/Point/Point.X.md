# Point.X


## Синтаксис


X: Number


## Описание


Свойство X устанавливает левую координату точки.


## Комментарии


Свойство используется в качестве параметра strX для конструктора [Point](Constructor_Point.htm).


## Пример


Пример использования свойства приведен в описании конструктора [RadialGradientBrush](../RadialGradientBrush/Constructor_RadialGradientBrush.htm).
