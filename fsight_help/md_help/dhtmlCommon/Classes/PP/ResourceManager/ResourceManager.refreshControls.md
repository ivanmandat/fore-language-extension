# ResourceManager.refreshControls


## Синтаксис


refreshControls(callback: PP.Delegate|function);


## Параметры


callback. Возвратная функция.


## Описание


Метод refreshControls обновляет элементы управления, зарегистрированные в менеджере ресурсов.
