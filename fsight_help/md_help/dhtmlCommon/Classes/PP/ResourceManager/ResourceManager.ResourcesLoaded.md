# ResourceManager.ResourcesLoaded


## Синтаксис


ResourcesLoaded: function(sender, args: Object);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие ResourcesLoaded генерируется после загрузки ресурсов из файла.
