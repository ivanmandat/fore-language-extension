# ResourceManager.setResourceList


## Синтаксис


setResourceList(resNamesArray: Array)


## Параметры


resNamesArray. Массив наименований файлов ресурсов (должна быть указана крайняя левая часть названия файла).


## Описание


Метод setResourceList инициализирует список доступных файлов ресурсов.
