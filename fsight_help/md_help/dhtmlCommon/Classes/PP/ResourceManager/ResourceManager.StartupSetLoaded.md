# ResourceManager.StartupSetLoaded


## Синтаксис


StartupSetLoaded: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие StartupSetLoaded генерируется после загрузки всех файлов ресурсов для текущего языка.
