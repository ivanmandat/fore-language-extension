# ResourceManager.addToResourceList


## Синтаксис


addToResourceList(resNamesArray: Array, autoLoad: Boolean);


## Параметры


resNamesArray. Массив наименований файлов ресурсов (должна быть указана крайняя левая часть названия файла);


autoLoad. Признак подгрузки вновь добавленных ресурсов для текущего языка. Если параметр равен значению true, то будут загружены новые ресурсы, иначе - не будут (по умолчанию).


## Описание


Метод addToResourceList добавляет указанный массив к существующему списку наименований файлов ресурсов.
