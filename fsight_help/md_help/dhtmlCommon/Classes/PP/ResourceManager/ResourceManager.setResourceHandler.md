# ResourceManager.setResourceHandler


## Синтаксис


setResourceHandler(value: String);


## Параметры


value. Адрес веб-сервиса, обрабатывающего запросы к менеджеру ресурсов.


## Описание


Метод setResourceHandler устанавливает адрес веб-сервиса, обрабатывающего запросы к менеджеру ресурсов.
