# ResourceManager.ResourceFileLoaded


## Синтаксис


ResourceFileLoaded: function(sender, args: Object);


## Параметры


sender. Источник события;


args. Информация о событии. Допустимые аргументы: Name - крайняя левая часть наименования файла ресурсов, Culture - объект типа [PP.CultureInfo](../CultureInfo/CultureInfo.htm).


## Описание


Событие ResourceFileLoaded генерируется после загрузки файла ресурсов.
