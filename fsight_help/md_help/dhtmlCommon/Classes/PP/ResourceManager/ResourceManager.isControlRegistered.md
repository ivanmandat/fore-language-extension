# ResourceManager.isControlRegistered


## Синтаксис


isControlRegistered(control: [PP.Ui.Control](dhtmlUi.chm::/Classes/control/control.htm));


## Параметры


control. Элемент управления.


## Описание


Метод isControlRegistered проверяет,
 зарегистрирован ли указанный элемент управления в менеджере ресурсов.


## Комментарии


Метод возвращает значение true,
 если указанный элемент управления зарегистрирован в менеджере ресурсов,
 в противном случае - значение false.
