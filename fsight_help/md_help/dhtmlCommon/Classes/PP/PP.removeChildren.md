# PP.removeChildren


## Синтаксис


removeChildren(element: HTMLElement);


## Параметры


*element.* DOM-дерево, все дочерние вершины которого нужно удалить.


## Описание


Метод removeChildren** удаляет все дочерние вершины в указанном DOM-дереве.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на файл сценария PP.js. Создадим DOM-дерево на основе указанной строки с html-разметкой и удалим все его дочерние вершины:


// Определим строку с разметкой
var markup = "<div><p class='plainText'>12</p><span class='plainText'>abc</span></div>";
// Преобразуем разметку в DOM-вершину
var dom = PP.htmlToDOM(markup, false);
console.log("Разметка исходного дерева: " + dom.outerHTML);
// Удалим все дочерние вершины в указанном дереве
PP.removeChildren(dom);
// Получим html-разметку полученного дерева
console.log("Разметка полученного дерева: " + dom.outerHTML);

В результате выполнения примера в консоли браузера была выведена разметка DOM-дерева до и после удаления всех его дочерних вершин:


Разметка исходного дерева: <div><p class="plainText">12</p><span class="plainText">abc</span></div>

Разметка полученного дерева: <div></div>
