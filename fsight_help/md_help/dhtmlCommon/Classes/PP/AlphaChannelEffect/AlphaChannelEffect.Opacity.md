# AlphaChannelEffect.Opacity


## Синтаксис


Opacity: Double


## Описание


Свойство Opacity определяет значение прозрачности, которая будет применена к альфа-каналу кисти фона формы. Значения от 0 до 1.


## Комментарии


Значения от 0 до 1. Чем меньше значение, тем больше прозрачность.
