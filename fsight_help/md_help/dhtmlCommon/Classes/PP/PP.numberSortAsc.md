# PP.numberSortAsc


## Синтаксис


numberSortAsc(a, b);


## Параметры


*a*. Первое значение;


*b*. Второе значение.


## Описание


Метод numberSortAsc представляет функцию для сортировки по возрастанию.


## Комментарии


Метод используется в качестве параметра стандартной JavaScript-функции sort.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на файл сценария PP.js. Определим массив целых чисел и отсортируем их по возрастанию:


// Определим массив целых чисел
var elements = [5, 9, 6, 7, 2];
console.log("Исходный массив элементов: [" + elements + "]");
// Отсортируем элементы массива по возрастанию
elements.sort(PP.numberSortAsc);
console.log("Отсортированный по возрастанию массив элементов: [" + elements + "]");

В результате выполнения примера в консоли браузера были выведены элементы массива, отсортированные по возрастанию:


Исходный массив элементов: [5,9,6,7,2]

Отсортированный по возрастанию массив элементов: [2,5,6,7,9]
