# PP.fixEvent


## Синтаксис


fixEvent(event, elem: HTMLElement);


## Параметры


*event.* Объект события;


*elem.* DOM-элемент, для которого вызвано событие.


## Описание


Метод fixEvent** обеспечивает кроссбраузерную работу указанного события.


## Комментарии


Метод возвращает объект события.
