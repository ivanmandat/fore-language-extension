# PP.isMemberOf


## Синтаксис


isMemberOf(enm, member);


## Параметры


*enm.* Перечисление;


*member.* Проверяемый элемент перечисления.


## Описание


Метод isMemberOf** проверяет, принадлежит ли элемент указанному перечислению.


## Комментарии


Метод возвращает значение true, если элемент принадлежит указанному перечислению, иначе - значение false.
