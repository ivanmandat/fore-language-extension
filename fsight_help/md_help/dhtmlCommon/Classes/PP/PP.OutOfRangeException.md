# PP.OutOfRangeException


## Синтаксис


OutOfRangeException(message: String, self: Object);


## Параметры


*message.* Сообщение, передаваемое исключению. Необязательный параметр, по умолчанию значение не определено;


*self.* Источник исключения. Необязательный параметр, по умолчанию значение не определено.


## Описание


Метод OutOfRangeException** представляет исключение, которое выбрасывается, если указанный аргумент находится вне допустимого диапазона значений.


## Комментарии


Метод возвращает значение типа String в следующем формате: «OutOfRangeException. Source: [Источник исключения] Message: [Передаваемое сообщение]».
