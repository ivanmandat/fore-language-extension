# NativeDelegate.NativeEvent


## Синтаксис


NativeEvent: String;


## Описание


Свойство NativeEvent определяет
 наименование события, для которого устанавливается обработчик.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setNativeEvent,
 а возвращается с помощью метода getNativeEvent.
