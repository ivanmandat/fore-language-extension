# NativeDelegate.ParentControl


## Синтаксис


ParentControl: [PP.Ui.Control](dhtmlUi.chm::/Classes/control/control.htm);


## Описание


Свойство ParentControl определяет
 элемент управления, для события которого устанавливается обработчик.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setParentControl,
 а возвращается с помощью метода getParentControl.
