# NativeDelegate.HandleName


## Синтаксис


HandleName: String;


## Описание


Свойство HandleName определяет
 наименование обработчика события.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHandleName,
 а возвращается с помощью метода getHandleName.
