# NativeDelegate.IsTouchDisabled


## Синтаксис


IsTouchDisabled: Boolean;


## Описание


Свойство IsTouchDisabled определяет,
 генерируется ли событие с помощью касания соответствующего элемента управления.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsTouchDisabled,
 а возвращается с помощью метода getIsTouchDisabled.


Допустимые значения:


	- true.
	 Событие не будет генерироваться с помощью касания соответствующего
	 элемента управления;


	- false.
	 Событие будет генерироваться с помощью касания соответствующего элемента
	 управления (по умолчанию).
