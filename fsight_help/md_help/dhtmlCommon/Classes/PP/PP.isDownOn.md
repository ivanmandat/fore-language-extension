# PP.isDownOn


## Синтаксис


isDownOn(elem: HTMLElement, event);


## Параметры


*elem.* DOM-элемент;


*event.* Событие курсора мыши.


## Описание


Метод isDownOn** определяет, действительно ли курсор мыши находился над элементом в момент события.


## Комментарии


Метод возвращает значение true, если курсор мыши находится над элементом в момент события, и значение false, если не находится.
