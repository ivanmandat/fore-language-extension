# Brush.Opacity


## Cинтаксис


Opacity: Double


## Описание


Свойство Opacity определяет прозрачность заливки.


## Комментарии


Свойство принимает значения от 0 до 1. Чем больше значение, тем меньше прозрачность заливки.
