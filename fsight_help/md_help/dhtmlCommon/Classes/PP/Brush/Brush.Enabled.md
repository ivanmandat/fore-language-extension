# Brush.Enabled


## Синтаксис


Enabled: Boolean


## Описание


Свойство Enabled определяет возможность применения кисти к компоненту.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setEnabled, а возвращается - с помощью метода getEnabled.


Свойство содержит значение true, если применение кисти для компонента доступно, и значение false - если не доступно.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылок на файл сценария PP.js и файл стилей PP.css. Создадим div-элемент и установим для него заливку:


// Создаём div-элемент
var divElem = PP.createElement(document.body);
// Настраиваем стили для данного элемента
divElem.style.cssText = "width: 100px; height: 100px; border: 1px solid rgb(102,102, 102);";
// Определяем кисть и устанавливаем заливку для div-элемента
var brush = new PP.SolidColorBrush();
brush.setColor("#dfdfff");
divElem.style.cssText += brush.toCSSFormat();

В результате выполнения примера в документе был создан div-элемент, для которого была применена сплошная заливка:


![](Brush_Enabled_1.png)


Запретим использование кисти и повторно установим заливку для div-элемента:


// Запрещаем использование кисти
brush.setEnabled(false);
// Устанавливаем заливку div-элемента
divElem.style.cssText += brush.toCSSFormat();

В результате будет установлена пустая заливка:


![](Brush_Enabled_2.png)
