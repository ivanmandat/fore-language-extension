# PP.widthToPixel


## Синтаксис


widthToPixel(len: String);


## Параметры


*len.* Длина в каких-либо единицах.


## Описание


Метод widthToPixel** переводит значения длины различных величин в пиксели.


## Комментарии


В случае успешного выполнения метод возвращает значение типа Number, иначе - значение null.


В качестве параметра задаётся значение с указанием любых единиц измерения, например «1mm».


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на файл сценария PP.js. Переведём значение длины из миллиметров в пиксели:


// Определим значение длины
var length = "5mm";
console.log("Длина в миллиметрах: " + length);
// Переведён заданное значение в пиксели
length = PP.widthToPixel(length);
console.log("Длина в пикселях: " + Math.round(length));

В результате выполнения примера в консоли браузера было выведено значение длины в миллиметрах и в пикселях:


Длина в миллиметрах: 5mm

Длина в пикселях: 19
