# AnimationSettings.UseAnimation


## Синтаксис


UseAnimation: Boolean


## Описание


Свойство UseAnimation определяет признак использования анимации в визуальном компоненте.


## Комментарии


Свойство содержит значение true, если использование анимации в визуальном компоненте разрешено, иначе - значение false.
