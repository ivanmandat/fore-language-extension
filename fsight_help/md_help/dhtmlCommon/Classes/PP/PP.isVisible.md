# PP.isVisible


## Синтаксис


isVisible(element: HTMLElement|PP.Ui.[Control](dhtmlUi.chm::/Classes/control/control.htm));


## Параметры


*element.* DOM-элемент
 или элемент управления.


## Описание


Метод isVisible** проверяет
 видимость указанного элемента.


## Комментарии


Метод не охватывает случай, если значение стиля «Visibility» родительского
 элемента равен «hidden».
