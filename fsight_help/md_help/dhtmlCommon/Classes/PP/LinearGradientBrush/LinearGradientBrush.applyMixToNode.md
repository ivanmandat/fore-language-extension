# LinearGradientBrush.applyMixToNode


## Синтаксис


applyMixToNode(node: HTMLElement, brush2: [PP.LinearGradientBrush](LinearGradientBrush.htm),
 part: Number);


## Параметры


node. DOM-элемент, к которому
 применяется кисть;


brush2. Кисть с линейной градиентной
 заливкой, с которой требуется выполнить смешивание;


part. Доля использования указываемой
 кисти. Значение может варьироваться от 0 (кисть не применяется) до 1 (применяется
 только данная кисть).


## Описание


Метод applyMixToNode применяет
 смешанный цвет кистей с линейной градиентной заливкой к указанному DOM-элементу.
