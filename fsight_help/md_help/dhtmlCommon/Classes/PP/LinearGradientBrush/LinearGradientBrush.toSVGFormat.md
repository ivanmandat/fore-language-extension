# LinearGradientBrush.toSVGFormat


## Синтаксис


toSVGFormat(element);


## Параметры


element. SVG-элемент, к которому
 применяется кисть.


## Описание


Метод toSVGFormat применяет
 кисть к указанному SVG-элементу.
