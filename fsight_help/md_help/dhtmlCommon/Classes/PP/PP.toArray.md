# PP.toArray


## Синтаксис


toArray(obj: Object);


## Параметры


*obj.* Объект, который нужно переместить в массив.


## Описание


Метод toArray перемещает объект в массив.


## Комментарии


Метод возвращает массив, единственный элементов которого является исходный объект.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на файл сценария PP.js. Создадим объект и переместим его в пустой массив:


// Определим объект
var chartView = {
    chartType: "pie"
};
// Поместим данный объект в массив
var objects = PP.toArray(chartView);
console.log("Длина массива: " + objects.length);
console.log("Элементы в массиве:");
for (var i in objects) {
    var object = objects[i];
    console.log(object);
};

В результате выполнения примера в консоли браузера были выведены длина полученного массива и список его элементов:


Длина массива: 1


Элементы в массиве:


Object {chartType: "pie"}
