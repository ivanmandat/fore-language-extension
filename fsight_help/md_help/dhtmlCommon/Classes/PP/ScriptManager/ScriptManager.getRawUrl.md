# ScriptManager.getRawUrl


## Синтаксис


PP.ScriptManager.getRawUrl(url: String);


## Параметры


url. Исходный URL-адрес.


## Описание


Метод getRawUrl очищает URL-адрес от параметров.


## Комментарии


Метод возвращает значение типа String.
