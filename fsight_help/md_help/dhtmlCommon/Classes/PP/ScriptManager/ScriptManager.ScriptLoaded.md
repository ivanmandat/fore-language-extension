# ScriptManager.ScriptLoaded


## Синтаксис


PP.ScriptManager.ScriptLoaded: function(sender, args: Object);


## Параметры


sender. Источник события;


args. Информация о событии. Допустимый аргумент: Src - URL-адрес загруженного сценария.


## Описание


Событие ScriptLoaded генерируется после загрузки файла сценария.
