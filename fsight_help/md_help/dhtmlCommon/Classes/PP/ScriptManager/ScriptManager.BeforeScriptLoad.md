# ScriptManager.BeforeScriptLoad


## Синтаксис


PP.ScriptManager.BeforeScriptLoad: function(sender, args: Object);


## Параметры


sender. Источник события;


args. Информация о событии. Допустимый аргумент: Src - URL-адрес сценария, который должен быть загружен.


## Описание


Событие BeforeScriptLoad генерируется перед загрузкой файла сценария.
