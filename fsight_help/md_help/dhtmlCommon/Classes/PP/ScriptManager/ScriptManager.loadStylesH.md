# ScriptManager.loadStylesH


## Синтаксис


PP.ScriptManager.loadStylesH(urls: Array, handler: String);


## Параметры


urls. Массив URL-адресов файлов со стилями CSS;


handler. Адрес обработчика.


## Описание


Метод loadStylesH загружает файлы со стилями CSS с помощью специального обработчика.


## Комментарии


Метод возвращает значение true, если был загружен хотя бы один файл стиля, и значение false, если не был загружен ни один из файлов.
