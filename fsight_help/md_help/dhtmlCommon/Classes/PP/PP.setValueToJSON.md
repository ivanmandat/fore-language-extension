# PP.setValueToJSON


## Синтаксис


setValueToJSON(value, settings: Object, path: String, justReturn: Boolean);


## Параметры


*value.* Значение, устанавливаемое для заданного свойства;


*settings.* Исходный объект;


*path.* Путь к заданному свойству;


*justReturn.* Признак возврата текущего объекта. Если параметр равен значению true, то метод вернёт текущий объект, иначе ничего получено не будет. Необязательный параметр, по умолчанию его значение не определено.


## Описание


Метод setValueToJSON** устанавливает значение свойства объекта любой вложенности относительно исходного объекта.


## Комментарии


Если какое-либо свойство на пути к указанному полю отсутствует, то метод создаёт его сам.
