# PP.isResourceGroupLoaded


## Синтаксис


isResourceGroupLoaded(name: String);


## Параметры


*name.* Наименование языка.


## Описание


Метод isResourceGroupLoaded**
 проверяет, загружен ли объект с региональными настройками для указанного
 языка.


## Комментарии


Метод возвращает JSON-объект со свойствами, соответствующими региональным
 настройкам конкретного языка, если данные настройки загружены.
