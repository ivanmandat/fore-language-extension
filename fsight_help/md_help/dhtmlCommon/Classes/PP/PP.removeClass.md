# PP.removeClass


## Синтаксис


removeClass(element: HTMLElement, value: String);


## Параметры


*element.* DOM-элемент, для которого требуется удалить CSS-класс;


*value.* Наименование CSS-класса.


## Описание


Метод removeClass** удаляет CSS-класс из указанного элемента.


## Комментарии


Метод очищает наименование CSS-класса в поле «className» указанного DOM-элемента.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на файл сценария PP.js. Создадим DOM-дерево на основе указанной строки с html-разметкой и удалим у него CSS-класс «plainText»:


// Определим строку с разметкой
var markup = "<div class='plainText'>12</div>";
// Преобразуем разметку в DOM-вершину
var dom = PP.htmlToDOM(markup, false);
console.log("Класс исходного элемента: «" + dom.className + "»");
// Удалим CSS-класс из данной вершины
PP.removeClass(dom, dom.className);
console.log("Класс элемента после очистки: «" + dom.className + "»");

В результате выполнения примера в консоли браузера было выведено наименование CSS-класса созданного DOM-элемента до и после очистки данного класса:


Класс исходного элемента: «plainText»

Класс элемента после очистки: «»
