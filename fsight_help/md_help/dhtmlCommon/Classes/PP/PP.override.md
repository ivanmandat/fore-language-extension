# PP.override


## Синтаксис


override(target, source);


## Параметры


*target.* Перезаписываемый элемент;


*source.* Элемент, которым перезаписывается другой элемент.


## Описание


Метод override** перезаписывает один элемент другим.


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на файл сценария PP.js. Создадим два массива целых чисел и перезапишем первый массив вторым:


// Определим массивы целых чисел
var num1 = [5, 1, 9, 6];
console.log("Исходный массив элементов: [" + num1 + "]");
var num2 = [2, 9, 6, 7, 5];
// Перезапишем первый массив вторым
PP.override(num1, num2);
console.log("Перезаписанный массив элементов: [" + num1 + "]");

В результате выполнения примера были созданы два массива и первый из них был перезаписан вторым:


Исходный массив элементов: [5,1,9,6]

Перезаписанный массив элементов: [2,9,6,7,5]
