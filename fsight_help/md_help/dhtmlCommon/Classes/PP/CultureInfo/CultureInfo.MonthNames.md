# CultureInfo.MonthNames


## Синтаксис


MonthNames: Array


## Описание


Свойство MonthNames устанавливает массив полных наименований всех месяцев в году.


## Комментарии


Значение свойства устанавливается напрямую и из JSON. По умолчанию равно следующему массиву строк: ["January", "February", "March", "April", "May", "June", "July", "August", "September", "October", "November", "December"].
