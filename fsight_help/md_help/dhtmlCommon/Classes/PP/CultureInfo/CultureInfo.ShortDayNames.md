# CultureInfo.ShortDayNames


## Синтаксис


ShortDayNames: Array


## Описание


Свойство ShortDayNames устанавливает массив сокращённых наименований дней недели.


## Комментарии


Значение свойства устанавливается напрямую и из JSON. По умолчанию равно следующему массиву строк: ["mo", "tu", "we", "th", "fr", "sa", "su"].
