# CultureInfo.Hour24Mode


## Синтаксис


Hour24Mode: Boolean


## Описание


Свойство Hour24Mode устанавливает 24-часовой режим отображения времени.


## Комментарии


Значение свойства устанавливается напрямую и из JSON, по умолчанию равно true.


Если значение свойства равно true, то используется 24-часовой режим отображения времени, иначе - 12-часовой.
