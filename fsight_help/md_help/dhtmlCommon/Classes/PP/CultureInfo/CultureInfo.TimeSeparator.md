# CultureInfo.TimeSeparator


## Синтаксис


TimeSeparator: String


## Описание


Свойство TimeSeparator определяет разделитель времени.


## Комментарии


Значение свойства устанавливается напрямую и из JSON. По умолчанию равно символу двоеточия.
