# CultureInfo.DisplayName


## Синтаксис


DisplayName: String


## Описание


Свойство DisplayName устанавливает полное наименование языка.


## Комментарии


Значение свойства устанавливается напрямую и из JSON, по умолчанию равно строке «English».
