# CultureInfo.getTimezoneOffsetInMsec


## Синтаксис


getTimezoneOffsetInMsec();


## Описание


Метод getTimezoneOffsetInMsec возвращает смещение текущей временной зоны в миллисекундах.


## Комментарии


Метод возвращает значение типа Number.
