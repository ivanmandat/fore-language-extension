# CultureInfo.getWeekNumber


## Синтаксис


getWeekNumber(date: Date);


## Параметры


date. Дата.


## Описание


Метод getWeekNumber возвращает порядковый номер недели в году, в которую входит указанная дата.


## Комментарии


Метод возвращает значение типа Number.
