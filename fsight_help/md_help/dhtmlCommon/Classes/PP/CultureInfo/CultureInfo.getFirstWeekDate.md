# CultureInfo.getFirstWeekDate


## Синтаксис


getFirstWeekDate(weekNo: Number, fullyear: Number);


## Параметры


weekNo. Порядковый номер недели в году;


fullyear. Год.


## Описание


Метод getFirstWeekDate возвращает дату первого дня в неделе с указанным номером указанного года.


## Комментарии


Метод возвращает значение типа Date.
