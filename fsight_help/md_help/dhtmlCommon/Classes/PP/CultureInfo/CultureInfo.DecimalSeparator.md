# CultureInfo.DecimalSeparator


## Синтаксис


DecimalSeparator: String


## Описание


Свойство DecimalSeparator устанавливает разделитель дробной части.


## Комментарии


Значение свойства устанавливается напрямую и из JSON, по умолчанию содержит символ точки.
