# CultureInfo.CultureName


## Синтаксис


CultureName: [PP.CultureNames](../../../Enums/culturenames.htm)


## Описание


Свойство CultureName устанавливает сокращённое наименование языка.


## Комментарии


Значение свойства устанавливается напрямую и из JSON с использованием перечисления [PP.CultureNames](../../../Enums/culturenames.htm), по умолчанию равно PP.CultureNames.en.
