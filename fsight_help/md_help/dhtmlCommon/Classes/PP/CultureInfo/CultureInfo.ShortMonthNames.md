# CultureInfo.ShortMonthNames


## Синтаксис


ShortMonthNames: Array


## Описание


Свойство ShortMonthNames устанавливает массив наименований всех месяцев в году.


## Комментарии


Значение свойства устанавливается напрямую и из JSON. По умолчанию равно следующему массиву строк: ["Jan", "Feb", "Mar", "Apr", "May", "Jun", "Jul", "Aug", "Sep", "Oct", "Nov", "Dec"].
