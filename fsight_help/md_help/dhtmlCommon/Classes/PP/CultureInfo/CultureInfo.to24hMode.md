# CultureInfo.to24hMode


## Синтаксис


to24hMode(hour: Number, amFlag: Boolean);


## Параметры


hour. Час времени;


amFlag. Признак того, относится ли указанный час времени к первой половине дня. Если значение параметра равно true, то заданный час будет относится к первой половине дня, иначе - ко второй.


## Описание


Метод to24hMode трансформирует указанный час времени из 12-часового режима в 24-часовой.


## Комментарии


Метод возвращает значение типа Number.
