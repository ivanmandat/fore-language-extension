# CultureInfo.IsRTL


## Синтаксис


IsRTL: Boolean


## Описание


Свойство IsRTL устанавливает расположение текста в визуальных компонентах по правому краю.


## Комментарии


Значение свойства устанавливается напрямую и из JSON. По умолчанию для свойства установлено значение false, то есть текст в визуальных компонентах располагается по левому краю. Если свойство содержит значение true, то текст будет располагаться по правому краю.
