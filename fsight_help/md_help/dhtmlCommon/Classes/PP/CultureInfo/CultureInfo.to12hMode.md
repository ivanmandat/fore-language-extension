# CultureInfo.to12hMode


## Синтаксис


to12hMode(hour: Number);


## Параметры


hour. Час времени.


## Описание


Метод to12hMode преобразует указанный час времени из 24-часового формата в 12-часовой.


## Комментарии


Метод возвращает массив. Значение с индексом 0 представляет собой номер часа в 12-часовом формате, а значение с индексом 1 - признак того, относится ли указанный час времени к первой половине дня.
