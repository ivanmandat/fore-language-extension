# CultureInfo.LCID


## Синтаксис


LCID: Number


## Описание


Свойство LCID устанавливает код языка.


## Комментарии


Значение свойства устанавливается напрямую и из JSON. По умолчанию оно равно 1033, что соответствует коду английского языка.
