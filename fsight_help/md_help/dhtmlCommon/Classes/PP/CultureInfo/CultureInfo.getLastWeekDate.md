# CultureInfo.getLastWeekDate


## Синтаксис


getLastWeekDate(weekNo: Number, fullyear: Number);


## Параметры


weekNo. Порядковый номер недели в году;


fullyear. Год.


## Описание


Метод getLastWeekDate возвращает дату последнего дня в неделе с указанным номером указанного года.


## Комментарии


Метод возвращает значение типа Date.
