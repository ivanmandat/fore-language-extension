# CultureInfo.DateOrder


## Синтаксис


DateOrder: String


## Описание


Свойство DateOrder устанавливает порядок формирования даты.


## Комментарии


Значение свойства устанавливается напрямую и из JSON. По умолчанию равно строке «dmy», что соответствует порядку «день-месяц-год».
