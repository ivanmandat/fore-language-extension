# CultureInfo.DateSeparator


## Синтаксис


DateSeparator: String


## Описание


Свойство DateSeparator устанавливает разделитель даты.


## Комментарии


Значение свойства устанавливается напрямую и из JSON, по умолчанию равно символу точки.
