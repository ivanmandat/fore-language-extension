# CultureInfo.IsSundayFirst


## Синтаксис


IsSundayFirst: Boolean


## Описание


Свойство IsSundayFirst определяет, является ли воскресение первым днём недели.


## Комментарии


Значение свойства устанавливается напрямую и из JSON, по умолчанию равно false.


Если свойство содержит значение true, то первым днём недели является воскресение, иначе - не является.
