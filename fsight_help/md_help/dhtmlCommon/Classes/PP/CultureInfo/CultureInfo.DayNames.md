# CultureInfo.DayNames


## Синтаксис


DayNames: Array


## Описание


Свойство DayNames устанавливает массив полных наименований дней недели.


## Комментарии


Значение свойства устанавливается напрямую и из JSON. Свойство содержит массив значений типа String.
