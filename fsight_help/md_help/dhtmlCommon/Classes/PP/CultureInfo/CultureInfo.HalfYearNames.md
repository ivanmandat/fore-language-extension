# CultureInfo.HalfYearNames


## Синтаксис


HalfYearNames: Array


## Описание


Свойство HalfYearNames определяет массив наименований полугодий.


## Комментарии


Значение свойства устанавливается напрямую и из JSON. По умолчанию равно следующему массиву строк: ["I First", "II Second"].
