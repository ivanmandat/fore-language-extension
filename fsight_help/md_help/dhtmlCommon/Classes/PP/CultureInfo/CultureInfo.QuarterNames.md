# CultureInfo.QuarterNames


## Синтаксис


QuarterNames: Array


## Описание


Свойство QuarterNames устанавливает массив наименований кварталов в году.


## Комментарии


Значение свойства устанавливается напрямую и из JSON. По умолчанию равно следующему массиву строк: ["I First", "II Second"].
