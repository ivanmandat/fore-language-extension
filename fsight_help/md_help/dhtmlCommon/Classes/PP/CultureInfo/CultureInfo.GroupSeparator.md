# CultureInfo.GroupSeparator


## Синтаксис


GroupSeparator: String


## Описание


Свойство GroupSeparator устанавливает разделитель разрядов.


## Комментарии


Значение свойства устанавливается напрямую и из JSON, по умолчанию равно пустой строке.
