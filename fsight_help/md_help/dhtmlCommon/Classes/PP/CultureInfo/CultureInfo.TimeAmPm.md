# CultureInfo.TimeAmPm


## Синтаксис


TimeAmPm: Array


## Описание


Свойство TimeAmPm определяет постфиксы для 12-часового режима отображения времени.


## Комментарии


Значение свойства устанавливается напрямую и из JSON. По умолчанию равно массиву строк ["PM", "AM"].


Значение массива с индексом 0 определяет постфикс времени второй половины дня, значение массива с индексом 1 определяет постфикс времени первой половины дня.
