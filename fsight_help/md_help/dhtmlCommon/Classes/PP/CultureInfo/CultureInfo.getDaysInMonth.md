# CultureInfo.getDaysInMonth


## Синтаксис


getDaysInMonth(month: Number, year: Number);


## Параметры


month. Номер месяца;


year. Год.


## Описание


Метод getDaysInMonth возвращает количество дней в заданном месяце указанного года.


## Комментарии


Метод возвращает значение типа Number.
