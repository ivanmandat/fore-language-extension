# CultureInfo.WEEKMS


## Синтаксис


WEEKMS: Number


## Описание


Свойство WEEKMS устанавливает количество миллисекунд в неделе.


## Комментарии


Значение свойства устанавливается напрямую и из JSON. По умолчанию содержит число 604800000.
