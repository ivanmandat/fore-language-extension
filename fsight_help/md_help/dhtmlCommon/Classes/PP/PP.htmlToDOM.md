# PP.htmlToDOM


## Синтаксис


htmlToDOM(html: String, getAllNodes: Boolean, withoutCache: Boolean);


## Параметры


*html.* Разметка, которую нужно преобразовать в DOM-вершину;


*getAllNodes.* Признак того, нужно ли возвращать массив DOM-вершин вместо одной. Если параметр равен значению true, то будет возвращаться массив DOM-вершин, иначе - одна вершина. Необязательный параметр, по умолчанию его значение не определено;


*withoutCache.* Признак того, что кеширование созданных вершин не требуется. Если параметр равен значению true, то DOM-вершины будут закешированы, иначе - не будут. Необязательный параметр, по умолчанию его значение не определено.


## Описание


Метод htmlToDOM** преобразует html-разметку в DOM-вершину.


## Комментарии


Метод возвращает значение типа HTMLElement.
