# PP.unescapeHTML


## Синтаксис


unescapeHTML(text: String);


## Параметры


*text.* Текст, требующий разэкранирования.


## Описание


Метод unescapeHTML** осуществляет разэкранирование символов HTML-разметки в указанной строке.


## Комментарии


Метод возвращает значение типа String.


Экранирование управляющих символов осуществляется с помощью метода [PP.escapeHTML](PP.escapeHTML.htm).


## Пример


Для выполнения примера необходимо наличие на html-странице ссылки на файл сценария PP.js. Определим строку с управляющими символами, экранируем её и затем обратно разэкранируем:


// Определим исходную строку с управляющими символами
var sourceStr = "<div class=\"mark\">12</div>"
console.log("Исходная строка: " + sourceStr);
// Экранируем символы
var escapeStr = PP.escapeHTML(sourceStr);
console.log("Экранированная строка: " + escapeStr);
// Разэкранируем символы
console.log("Разэкранированная строка: " + PP.unescapeHTML(escapeStr));

В результате выполнения примера в консоли браузера была выведена исходная строка с управляющими символами, затем её экранированный и разэкранированный варианты:


Исходная строка: <div class="mark">12</div>

Экранированная строка: &lt;div class="mark"&gt;12&lt;/div&gt;

Разэкранированная строка: <div class="mark">12</div>
