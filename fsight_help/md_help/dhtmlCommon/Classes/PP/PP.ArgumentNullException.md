# PP.ArgumentNullException


## Синтаксис


ArgumentNullException(message: String, self: Object);


## Параметры


*message.* Сообщение, передаваемое исключению. Необязательный параметр, по умолчанию значение не определено;


*self.* Источник исключения. Необязательный параметр, по умолчанию значение не определено.


## Описание


Метод ArgumentNullException** представляет исключение, которое выбрасывается, если один из передаваемых функции параметров равен значению null.


## Комментарии


Метод возвращает значение типа String в следующем формате: «ArgumentNullException. Source: [Источник исключения] Message: [Передаваемое сообщение]».
