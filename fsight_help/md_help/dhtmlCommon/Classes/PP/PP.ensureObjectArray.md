# PP.ensureObjectArray


## Синтаксис


ensureObjectArray(type: [PP.Object](../Object/Object.htm), obj: Array);


## Параметры


*type.* Тип объекта:


*obj.* JSON-объект, который нужно преобразовать, либо объект заданного типа.


## Описание


Метод ensureObjectArray** преобразовывает указанный массив JSON-объектов в массив объектов заданного типа.


## Комментарии


Метод возвращает массив PP-объектов заданного типа.
