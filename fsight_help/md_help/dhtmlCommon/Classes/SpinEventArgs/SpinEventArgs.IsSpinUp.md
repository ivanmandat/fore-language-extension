# SpinEventArgs.IsSpinUp


## Синтаксис


SpinEventArgs.IsSpinUp = Boolean;


## Описание


Свойство IsSpinUp, выполняется
 ли прокручивание вверх.


## Комментарии


Если установлено значение true,
 то выполняется прокручивание при нажатии на кнопку «Вверх». Если установлено
 значение false, то выполняется
 прокручивание при нажатии кнопки «Вниз».
