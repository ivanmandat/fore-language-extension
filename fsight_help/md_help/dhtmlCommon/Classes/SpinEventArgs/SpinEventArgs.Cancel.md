# SpinEventArgs.Cancel


## Синтаксис


SpinEventArgs.Cancel = Boolean;


## Описание


Свойство Cancel определяет,
 что в редактор может быть введен только текст.


## Комментарии


Если установлено значение false,
 то доступен ввод числовых и текстовых значений. Если установлено значение
 true, то доступен ввод только
 текстовых значений.
