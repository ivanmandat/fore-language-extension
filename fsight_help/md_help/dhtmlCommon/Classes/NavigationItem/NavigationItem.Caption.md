# NavigationItem.Caption


## Синтаксис


Caption: String


## Описание


Свойство Caption устанавливает
 и возвращает заголовок вкладки.


## Комментарии


Значение свойства устанавливается с помощью метода setCaption, а возвращается с помощью
 метода getCaption.
 Из JSON значение свойства задать нельзя.
