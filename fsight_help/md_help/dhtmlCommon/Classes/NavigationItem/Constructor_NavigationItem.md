# Конструктор NavigationItem


## Синтаксис


PP.Ui.NavigationItem (settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор NavigationItem создает экземпляр компонента [NavigationItem](../../Components/NavigationItem/NavigationItem.htm).
