# NavigationItem.expand


## Синтаксис


expand ();


## Описание


Метод expand разворачивает панель.


## Пример


Для выполнения примера создайте компонент [NavigationBar](../../Components/NavigationBar/NavigationBar.htm) с наименованием «NB» (см. «[Пример создания компонента NavigationBar](../../Components/NavigationBar/Example_NavigationBar.htm)»). Для того чтобы при загрузке страницы верхняя вкладка была открыта, используйте метод expand:


NB.getItem(0).expand();


После выполнения примера при открытии html-страницы навигатор будет выглядеть следующим образом:
