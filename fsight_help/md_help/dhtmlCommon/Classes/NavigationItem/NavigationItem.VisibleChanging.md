# NavigationItem.VisibleChanging


## Синтаксис


VisibleChanging: function (sender, args)


## Параметры


sender. Источник
 события;


args. Информация о событии.
 Cancel -
 признак отмены установки/снятия флажка для изменения видимости
 вкладки: true - операция отменена,
 false (по умолчанию) - операция
 не отменена.


## Описание


Событие VisibleChanging наступает
 во время изменения видимости вкладки.
