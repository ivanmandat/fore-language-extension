# NavigationItem.getContentSize


## Синтаксис


getContentSize ();


## Описание


Метод getContentSize возвращает размеры содержимого панели.


## Пример


Для выполнения примера создайте компонент [NavigationBar](../../Components/NavigationBar/NavigationBar.htm) с наименованием «NB» (см. «[Пример создания компонента NavigationBar](../../Components/NavigationBar/Example_NavigationBar.htm)»). Чтобы узнать размеры содержимого верхней панели используйте метод getContentSize:


alert (NB.getItem(1).getContentSize());


После выполнения примера на экран будет выведена информация о размерах содержимого верхней панели.
