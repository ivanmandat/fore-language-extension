# WelcomeScreen.getMaxActions


## Синтаксис


getMaxActions();


## Описание


Метод getMaxActions возвращает
 максимальное количество действий в списке в окне приветствия.


## Комментарии


По умолчанию максимальное количество действий равно 5.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 WelcomeScreen с наименованием «wscreen» (см. «Пример создания компонента
 [WelcomeScreen](WelcomeScreen_Example.htm)»). Получим максимальное
 количество действий:


wscreen.getMaxActions();
