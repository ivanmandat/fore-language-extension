# WelcomeScreen.ToolChanged


## Синтаксис


ToolChanged : function (sender, args);


## Параметры


sender. Источник события;


args. Информация
 о событии.


## Описание


Событие ToolChanged наступает
 после смены инструмента в окне приветствия.


## Комментарии


Смена инструмента производится щелчком мыши по еще не открытому
 инструменту.
