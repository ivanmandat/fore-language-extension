# WelcomeScreen.ToolSelected


## Синтаксис


ToolSelected : function (sender, args);


## Параметры


sender. Источник события;


args. Информация
 о событии.


## Описание


Событие ToolSelected наступает
 после повторного выбора инструмента в окне приветствия.


## Комментарии


Выбор инструмента производится щелчком мыши по пиктограмме уже открытого инструмента.
