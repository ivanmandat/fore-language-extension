# WelcomeScreen.ActionSelected


## Синтаксис


ActionSelected : function (sender, args);


## Параметры


sender. Источник события;


args. Информация
 о событии.


## Описание


Событие ActionSelected наступает
 после выбора действия в окне приветствия.


## Комментарии


Действия, доступные в окне приветствия по умолчанию: Открыть документ,
 Новый документ, Импорт данных.
