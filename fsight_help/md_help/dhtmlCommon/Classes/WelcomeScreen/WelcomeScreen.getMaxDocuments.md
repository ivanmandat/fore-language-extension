# WelcomeScreen.getMaxDocuments


## Синтаксис


getMaxDocuments();


## Описание


Метод getMaxDocuments возвращает
 максимальное количество документов в списке последних документов в окне
 приветствия.


## Комментарии


По умолчанию максимальное количество документов равно 10.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 WelcomeScreen с наименованием «wscreen» (см. «Пример создания компонента
 [WelcomeScreen](WelcomeScreen_Example.htm)»). Получим максимальное
 количество действий:


wscreen.getMaxDocuments();
