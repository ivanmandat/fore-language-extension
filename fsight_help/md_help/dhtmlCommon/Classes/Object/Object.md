[IDisposable](http://msdn.microsoft.com/ru-ru/library/system.idisposable.aspx)


[ICloneable](http://msdn.microsoft.com/ru-ru/library/system.icloneable.aspx)


Object


## Синтаксис


PP.initClass(PP.Object, null, "Object");


## Описание


Класс Object является базовым
 классом для всех классов пространства имен PP.
 Все классы в пространстве имен PP
 являются наследниками класса Object.
