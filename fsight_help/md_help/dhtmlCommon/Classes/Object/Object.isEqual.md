# Object.isEqual


## Синтаксис


isEqual(obj: IObject): boolean;


## Параметры


obj. Объект, с которым сравнивается
 текущий объект репозитория.


## Описание


Метод isEqual определяет, равен
 ли заданный объект текущему объекту репозитория.


## Комментарии


Метод возвращает одно из значений:


	- True. Заданный объект
	 равен текущему объекту репозитория;


	- False. Заданный объект
	 не равен текущему объекту репозитория.
