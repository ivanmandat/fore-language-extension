# Object.removeAllEvents


## Синтаксис


removeAllEvents(context: [Object](Object.htm));


## Параметры


context. Контекст, по которому
 удаляются обработчики событий.


## Описание


Метод removeAllEvents удаляет
 все обработчики событий объекта по заданному контексту.
