# Object.isLive


## Синтаксис


isLive(): boolean;


## Описание


Метод isLive определяет действительность
 объекта репозитория.


## Комментарии


Метод возвращает одно из значений:


	- True. Объект действителен;


	- False. Объект недействителен.
