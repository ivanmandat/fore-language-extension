# NavigationBar.insertItem


## Синтаксис


insertItem (item: Object, index: Number || [PP.Ui.NavigationItem](../NavigationItem/NavigationItem.htm));


## Параметры


item. Элемент, вставляемый
 в указанную позицию;


index. Позиция или элемент
 панели, куда вставляется новый элемент.


## Описание


Метод insertItem добавляет элемент
 в панель свойств по индексу или элементу.

Пример использования метода приведен в описании метода [NavigationItem.showScrolls](../NavigationItem/NavigationItem.showScrolls.htm).
