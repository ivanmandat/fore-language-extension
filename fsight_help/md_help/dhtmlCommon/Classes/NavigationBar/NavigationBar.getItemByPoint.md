# NavigationBar.getItemByPoint


## Синтаксис


getItemByPoint(x, y);


## Параметры


x. Координата по горизонтали;


y. Координата по вертикали.


## Описание


Метод getItemByPoint возвращает
 вкладку панели свойств по заданным координатам.
