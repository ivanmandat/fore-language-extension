# NavigationBar.beginUpdate


## Синтаксис


beginUpdate();


## Описание


Метод beginUpdate отключает
 перерисовку панели свойств до тех пор, пока не будет вызван метод [endUpdate](NavigationBar.endUpdate.htm).

Пример использования метода приведен в описании метода [NavigationItem.showScrolls](../NavigationItem/NavigationItem.showScrolls.htm).
