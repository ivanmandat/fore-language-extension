# NavigationBar.ShowBorders


## Синтаксис


ShowBorders: Boolean


## Описание


Свойство ShowBorders определяет, отображаются ли границы компонента.


## Комментарии


Если для свойства установлено значение true, границы компонента отображаются. По умолчанию установлено значение false.
