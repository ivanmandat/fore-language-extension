# Конструктор NavigationBar


## Синтаксис


PP.Ui.NavigationBar (settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор NavigationBar создает экземпляр компонента [NavigationBar](../../Components/NavigationBar/NavigationBar.htm).
