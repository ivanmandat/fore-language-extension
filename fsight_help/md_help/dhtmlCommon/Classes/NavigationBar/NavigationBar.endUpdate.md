# NavigationBar.endUpdate


## Синтаксис


endUpdate();


## Описание


Метод endUpdate включает перерисовку
 панели свойств и обновляет её.


## Комментарии


Использует в связке с методом [beginUpdate](NavigationBar.beginUpdate.htm).

Пример использования метода приведен в описании метода [NavigationItem.showScrolls](../NavigationItem/NavigationItem.showScrolls.htm).
