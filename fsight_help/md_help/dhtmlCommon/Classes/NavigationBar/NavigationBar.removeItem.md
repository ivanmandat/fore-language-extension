# NavigationBar.removeItem


## Синтаксис


removeItem (item, notDispose)


## Параметры


item. Вкладка, которую необходимо удалить. Экземпляр класса [NavigationItem](../NavigationItem/NavigationItem.htm);


notDispose. Определяет, будет ли вкладка удалена полностью.


## Описание


Метод removeItem удаляет указанную вкладку.
