NavigationBar


## Описание


Класс NavigationBar реализует
 компонент [NavigationBar](../../Components/NavigationBar/NavigationBar.htm),
 представляющий собой панель свойств,
 предназначенную для навигации по свойствам активного компонента и управления
 свойствами.


## Синтаксис


PP.initClass(PP.Ui.NavigationBar, PP.Ui.[Control](dhtmlUi.chm::/Classes/control/control.htm),
 "NavigationBar");
