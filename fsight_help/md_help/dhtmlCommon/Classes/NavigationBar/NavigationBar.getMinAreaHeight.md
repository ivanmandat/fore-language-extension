# NavigationBar.getMinAreaHeight


## Синтаксис


getMinAreaHeight();


## Описание


Метод getMinAreaHeight возвращает
 минимальную высоту области панели свойств.


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [NavigationBar](../../Components/NavigationBar/NavigationBar.htm)
 с наименованием «navigationBar» (см. «[Пример
 создания компонента NavigationBar](../../Components/NavigationBar/Example_NavigationBar.htm)»). Получаем минимальную высоту области
 панели свойств:


// Получаем минимальную высота области панели свойств
console.log("Минимальная высота области панели свойств: " + navigationBar.getMinAreaHeight());
В результате в консоль будет выведена минимальная высота области панели
 свойств:


Минимальная высота области панели свойств: 200
