# NavigationBar.IsAlwaysExpanded


## Синтаксис


IsAlwaysExpanded: Boolean


## Описание


Свойство IsAlwaysExpanded определяет, можно ли скрыть открытую вкладку , не раскрывая других вкладок.


## Комментарии


Одновременно не может быть раскрыто больше одной вкладки.


Если для свойства установлено значение true, то после создания компонента одна из вкладок будет развернута (по умолчанию - первая). Если для свойства установлено значение false, то после создания компонента все вкладки будут свернуты. По умолчанию установлено значение true.
