# NavigatorBar.isUpdating


## Синтаксис


isUpdating();


## Описание


Метод isUpdating определяет, обновляется ли в текущий момент времени панель свойств.


## Комментарии


Метод возвращает значение true, если в текущий момент времени обновляется панель свойств, иначе - значение false.


## Пример


Для выполнения примера предполагается наличие на странице компонента [NavigationBar](../../Components/NavigationBar/NavigationBar.htm) с наименованием «NB» (см. «[Пример создания компонента NavigationBar](../../Components/NavigationBar/Example_NavigationBar.htm)»). Проверим, обновляется ли панель свойств в текущий момент времени:


// Установим признак того, что панель обновляется
NB.beginUpdate();
if (NB.isUpdating()) {
    console.log("Панель свойств обновляется");
} else {
    console.log("Панель свойств в текущий момент времени не обновляется");
};
// Снимем признак обновления панели свойств
NB.endUpdate();

В результате выполнения примера в консоли браузера было выведено сообщение о том, что панель свойств обновляется.
