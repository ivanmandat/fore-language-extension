# NavigationBar.addItem


## Синтаксис


addItem (item: [PP.NavigationItem](../NavigationItem/NavigationItem.htm));


## Параметры


item. Вкладка, которая будет добавлена в навигатор.


## Описание


Метод addItem добавляет вкладку с раскрывающейся панелью в навигатор.


## Комментарии


Вкладка с раскрывающейся панелью является экземпляром класса [NavigationItem](../NavigationItem/NavigationItem.htm).


Также панель может быть добавлена в навигатор с помощью свойства [Items](NavigationBar.Items.htm).
