# NavigationBar.getLayoutItems


## Синтаксис


getLayoutItems();


## Описание


Метод getLayoutItems возвращает
 массив элементов панели свойств.


## Комментарии


Элементы массива являются объектами класса [PP.Ui.NavigationItem](../NavigationItem/NavigationItem.htm).
