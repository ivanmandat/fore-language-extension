# NavigationBar.Items


## Синтаксис


Items: Array


## Описание


Свойство Items устанавливает список панелей, входящих в состав компонента.


## Комментарии


Панели могут быть добавлены в компонент также при помощи метода [addItem](NavigationBar.addItem.htm).
