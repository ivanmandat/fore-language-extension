# SpinEdit.ShowButtons


## Синтаксис


ShowButtons: Boolean;


## Описание


Свойство ShowButtons определяет,
 используются ли кнопки для переключения значений в компоненте.


## Комментарии


По умолчанию установлено значение true,
 то есть кнопки переключения значения отображаются.


## Пример


Для выполнения примера создайте компонент [SpinEdit](dhtmlUi.chm::/components/spinedit/spinedit.htm)
 (см. «[Пример
 создания компонента SpinEdit](dhtmlUi.chm::/components/spinedit/example_spinedit.htm)»). Чтобы кнопки переключения значения
 не отображались, используйте свойство ShowButtons:


ShowButtons: false;


После выполнения примера на html-странице будет размещен компонент [SpinEdit](dhtmlUi.chm::/components/spinedit/spinedit.htm)
 без кнопок переключения значения:
