# Конструктор SpinEdit


## Синтаксис


PP.SpinEdit(settings);


## Параметры


settings. JSON-объект со значениями
 свойств компонента.


## Описание


Конструктор SpinEdit создает
 экземпляр компонента [SpinEdit](dhtmlUi.chm::/components/spinedit/spinedit.htm).
