# ImageList.loadImage


## Синтаксис


loadImage();


## Описание


Метод loadImage осуществляет
 загрузку изображения для последующей отрисовки на [канве](dhtmlGraphicBase.chm::/components/canvas/canvas.htm).


## Комментарии


Для отрисовки загруженного изображения на [канве](dhtmlGraphicBase.chm::/components/canvas/canvas.htm)
 используйте метод [ImageList.applyToCanvas](ImageList.applyToCanvas.htm).
