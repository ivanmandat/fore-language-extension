# ImageList.ImageLoaded


## Синтаксис


ImageLoaded: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие ImageLoaded наступает
 после загрузки изображения.


## Комментарии


Перед загрузкой изображения генерируется событие [ImageList.ImageLoading](ImageList.ImageLoading.htm).
