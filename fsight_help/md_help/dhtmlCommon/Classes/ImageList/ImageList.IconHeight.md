# ImageList.IconHeight


## Синтаксис


IconHeight: Number


## Описание


Свойство IconHeight устанавливает
 высоту пиктограммы.


## Комментарии


Значение свойства устанавливается из JSON при [создании
 компонента](dhtmlUi.chm::/Classes/IconListItem/Constructor_IconListItem.htm) и при помощи метода setIconHeight.


## Пример


Значение свойства задается в параметрах [конструктора
 ImageList](Constructor_ImageList.htm) (см. пример на странице «[Конструктор
 IconListItem](dhtmlUi.chm::/Classes/iconlistitem/constructor_iconlistitem.htm)»).
