# Конструктор ImageList


## Синтаксис

## Параметры


settings. JSON-объект со значениями
 свойств компонента.


## Описание


Конструктор ImageList создает
 экземпляр компонента [ImageList](dhtmlUi.chm::/components/imagelist/imagelist.htm).
