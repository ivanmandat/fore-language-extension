# ImageList.ImageLoading


## Синтаксис


ImageLoading: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие ImageLoading
 наступает в начале выполнения загрузки изображения.


## Комментарии


После загрузки изображения генерируется событие [ImageList.ImageLoaded](ImageList.ImageLoaded.htm).
