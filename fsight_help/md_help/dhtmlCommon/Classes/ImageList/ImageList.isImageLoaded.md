# ImageList.isImageLoaded


## Синтаксис


isImageLoaded(): Boolean;


## Описание


Метод isImageLoaded возвращает
 состояние загрузки изображения.


## Комментарии


Допустимые значения:


	- True. Изображение
	 загружено;


	- False. Изображение не
	 загружено.
