# FlexLayoutAlign


Пространство имен: PP;


## Описание


Перечисление FlexLayoutAlign
 содержит способы выравнивания элементов в контейнере с гибким размещением
 элементов.


Используется свойством [FlexLayout.Position](dhtmlUi.chm::/Classes/FlexLayout/FlexLayout.Position.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Left
		 Элементы выравниваются по левому краю контейнера.


		 Top
		 Элементы выравниваются по верхнему краю контейнера.


		 Right
		 Элементы выравниваются по правому краю контейнера.


		 Bottom
		 Элементы выравниваются по нижнему краю контейнера.


		 Middle
		 Элементы выравниваются по середине контейнера, относительно
		 левого и правого краев.


		 Center
		 Элементы выравниваются по центру контейнера.


		 Stretch
		 Элементы растянуться по всему контейнеру.


		 None
		 Элементы не будут выравнены.
