# VerticalAlignment


Пространство имен:
 PP;


## Описание


Перечисление VerticalAlignment содержит
 варианты выравнивания по вертикали.


Используется следующими свойствами:


	- [ChartLabel.Alignment](dhtmlChart.chm::/Classes/ChartLabel/ChartLabel.Alignment.htm);


	- [DataVisualizerContainer.getVerticalAlignment](dhtmlUi.chm::/Classes/DataVisualizerContainer/DataVisualizerContainer.getVerticalAlignment.htm);


	- [MapChart.TopobaseVerticalAlignment](dhtmlMapChart.chm::/Classes/Map/MapChart/MapChart.TopobaseVerticalAlignment.htm);


	- [MapTimeline.getVerticalAlignment](dhtmlMapChart.chm::/Classes/Map/MapTimeline/MapTimeline.getVerticalAlignment.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Top
		 Выравнивание по верхнему краю.


		 Middle
		 Выравнивание по центру по вертикали.


		 Bottom
		 Выравнивание по нижнему краю.


		 Center
		 Выравнивание по центру.


		 Custom
		 Пользовательские настройки.


		 Absolute
		 Абсолютное значение.


		 Relative
		 Относительное значение.


		 None
		 Выравнивание отсутствует.
