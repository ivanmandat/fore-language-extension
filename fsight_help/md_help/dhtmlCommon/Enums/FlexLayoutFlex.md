# FlexLayoutFlex


Пространство имен: PP;


## Описание


Перечисление FlexLayoutFlex
 содержит способы гибкости расположения элементов контейнера.


Используется свойством [FlexLayout.Position](dhtmlUi.chm::/Classes/FlexLayout/FlexLayout.Position.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Even
		 Размер элементов зависит от размера контейнера.


		 Ratio


		 Размер элементов фиксированный и не зависит от размера контейнера.


		 None
		 Гибкость расположения элементов контейнера не задана.
