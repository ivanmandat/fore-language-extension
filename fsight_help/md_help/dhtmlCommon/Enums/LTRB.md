# LTRB


Пространство имен: PP;


## Описание


Перечисление LTRB содержит варианты
 расположения объекта внутри компонента.


Используется следующими свойствами и методами:


	- [Button.ImagePosition](dhtmlUi.chm::/Classes/button/button.imageposition.htm);


	- [Button.MenuArrowPosition](dhtmlUi.chm::/Classes/button/button.menuarrowposition.htm);


	- [Button.MenuDropPosition](dhtmlUi.chm::/Classes/button/button.menudropposition.htm);


	- [LegendToolTip.LTRB](dhtmlGraphicBase.chm::/Classes/LegendToolTip/LegendToolTip.LTRB.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Left. Объект расположен
		 по левому краю.


		 1
		 Top. Объект расположен
		 сверху.


		 2
		 Right. Объект расположен
		 по правому краю.


		 3
		 Bottom. Объект расположен
		 снизу.
