# Sorting


Пространство имен: PP;


## Описание


Перечисление Sorting содержит
 режимы сортировки.


Используется свойством [RatingChart.Sorting](dhtmlChart.chm::/Classes/RatingChart/RatingChart.Sorting.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Ascending
		 Сортировка по возрастанию.


		 Descending
		 Сортировка по убыванию.
