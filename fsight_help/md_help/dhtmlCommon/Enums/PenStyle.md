# PenStyle


Пространство имен: PP;


## Описание


Перечисление PenStyle содержит
 типы линий.


Используется свойством [PP.Pen.Style](dhtmlGraphicBase.chm::/classes/Pen/Pen.Style.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 None
		 Нет границы.


		 Solid
		 Сплошная линия.


		 Dashed
		 Штриховая линия.


		 Dotted
		 Пунктирная линия.


		 ShortDash
		 Короткие штрихи.


		 ShortDot
		 Короткие точки.


		 DashDot
		 Штрих-пунктирная линия.


		 DashDotDot
		 Чередующиеся штрихи и двойные точки.


		 ShortDashDot
		 Чередующиеся короткие штрихи и точки.


		 LongDashDot
		 Чередующиеся длинные штрихи и точки.


		 ShortDashDotDot
		 Чередующиеся короткие штрихи и двойные точки.


		 LongDashDotDot
		 Чередующиеся длинные штрихи и двойные точки.
