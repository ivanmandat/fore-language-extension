# HorizontalAlignment


Пространство имен:
 PP;


## Описание


Перечисление HorizontalAlignment содержит
 варианты выравнивания по горизонтали.


Используется следующими свойствами:


	- [ChartLabel.Alignment](dhtmlChart.chm::/Classes/ChartLabel/ChartLabel.Alignment.htm);


	- [DataVisualizerContainer.getHorizontalAlignment](dhtmlUi.chm::/Classes/DataVisualizerContainer/DataVisualizerContainer.getHorizontalAlignment.htm);


	- [GaugeTextLabel.TextAlign](dhtmlGauge.chm::/Classes/Speedometer/GaugeTextLabel/GaugeTextLabel.TextAlign.htm);


	- [IntervalsLegend.TextHorizontalAlignment](dhtmlGraphicBase.chm::/Classes/IntervalsLegend/IntervalsLegend.TextHorizontalAlignment.htm);


	- [LegendBase.ContentAlignment](dhtmlGraphicBase.chm::/Classes/LegendBase/LegendBase.ContentAlignment.htm);


	- [MapChart.TopobaseHorizontalAlignment](dhtmlMapChart.chm::/Classes/Map/MapChart/MapChart.TopobaseHorizontalAlignment.htm);


	- [TextEditor.Alignment](dhtmlUi.chm::/Classes/TextEditor/TextEditor.Alignment.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Left
		 Выравнивание по левому краю.


		 Center
		 Выравнивание по центру.


		 Right
		 Выравнивание по правому краю.


		 Justify
		 Полное выравнивание по ширине.


		 Custom
		 Пользовательские настройки.


		 Absolute
		 Абсолютное значение.


		 Relative
		 Относительное значение.


		 None
		 Выравнивание отсутствует.
