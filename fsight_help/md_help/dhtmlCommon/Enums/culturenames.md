# CultureNames


Пространство имен: PP;


## Описание


Перечисление CultureNames содержит
 сокращенные наименования языков.


Используется свойством [CultureInfo.CultureName](../Classes/PP/CultureInfo/CultureInfo.CultureName.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 ar
		 Арабский язык.


		 ar-AE
		 Арабский язык (Объединенные Арабские Эмираты).


		 az
		 Азербайджанский язык.


		 de
		 Немецкий язык.


		 en
		 Английский язык.


		 es
		 Испанский язык.


		 fr
		 Французский язык.


		 it
		 Итальянский язык.


		 kk
		 Казахский язык.


		 pt
		 Португальский язык.


		 ru
		 Русский язык.


		 uk
		 Украинский язык.


		 zh-CN
		 Китайский язык.
