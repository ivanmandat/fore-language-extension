# PP.ControlState


Пространство имен: PP;


## Описание


Перечисление PP.ControlState
 содержит варианты состояний визуальных компонентов.


Используется свойством [Control.Style](dhtmlUi.chm::/Classes/Control/Control.Style.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Active
		 Визуальный компонент активен.


		 Disable
		 Визуальный компонент не активен.


		 Hover
		 На визуальный компонент наведен указатель.


		 Press
		 Визуальный компонент нажат.


		 Release
		 Прекращено нажатие визуального компонента.
