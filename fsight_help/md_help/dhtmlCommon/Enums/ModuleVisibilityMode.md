# ModuleVisibilityMode


Пространство имен:
 PP;


## Описание


Перечисление ModuleVisibilityMode содержит
 режимы отображения [модулей](dhtmlApp.chm::/Classes/Application/ModuleObject/ModuleObject.htm)
 в системе.


Используется методом [PP.App.getActiveModule](dhtmlApp.chm::/Classes/App/PP.App.getActiveModule.htm).


## Допустимые значения


		 Значение


		 Краткое описание


		 0


		 Hidden.
		 Скрывает модуль и все его объекты в [навигаторе](dhtmlApp.chm::/Classes/Application/NavModuleObject/NavModuleObject.htm).


		 1


		 Visible.
		 Модуль и его объекты доступны.


		 2


		 ObjectsOnly.
		 Модуль недоступен в списке модулей системы, однако его объекты
		 отображаются в [навигаторе](dhtmlApp.chm::/Classes/Application/NavModuleObject/NavModuleObject.htm).
