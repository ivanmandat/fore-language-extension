# FlexLayoutPack


Пространство имен: PP;


## Описание


Перечисление FlexLayoutPack
 содержит способы вертикального расположения элементов.


Используется свойством [FlexLayout.Position](dhtmlUi.chm::/Classes/FlexLayout/FlexLayout.Position.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 Start
		 Элементы будут расположены в начале контейнера.


		 Center
		 Элементы будут расположены в центре контейнера.


		 End
		 Элементы будут расположены в конце контейнера.


		 None
		 Вертикальное положение элементов не задано.
