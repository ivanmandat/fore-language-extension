# ZoomOptionsSettings


Пространство имен: PP;


## Иерархия наследования


           PP.ObjectSettings


           PP.ZoomOptionsSettings


## Описание


Интерфейс ZoomOptionsSettings используется
 для работы с классом [ZoomOptions](../../Classes/PP/ZoomOptions/ZoomOptions.htm).


## Комментарии


Значения свойств задаются в конструкторе [ZoomOptions](../../Classes/PP/ZoomOptions/Constructor_ZoomOptions.htm)
 или с помощью метода set<имя свойства>,
 а возвращаются с помощью метода get<имя свойства>.


Доступные методы указаны в описании конкретного свойства.
