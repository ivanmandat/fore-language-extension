# Требования для отображения стрелок


Для карты доступно отображение [стрелок](../Configure/Arrows.htm),
 если выполняются следующие условия:


	- В источнике данных карты должны присутствовать два территориальных
	 измерения. Оба территориальных измерения должны соответствовать [основным требованиям](Requirements.htm) для работы с картой.


	- Для карты [указаны роли](Metrics.htm) для следующих
	 измерений: измерение объектов наблюдения (первое территориальное измерение,
	 из которого строится стрелка), измерение направления стрелок (второе
	 территориальное измерение, в которое направлена стрелка), измерение
	 метрик.


	- Настроено [измерение метрик](Metrics.htm#metrics).
	 Сопоставлены элементы измерения метрик и следующие характеристики:


		- Цвет стрелок. Установите
		 переключатель «Цвет стрелок»
		 на вкладке измерения метрик и отметьте элемент, по значениям
		 которого будут закрашиваться стрелки;


		- Толщина стрелок.
		 Установите переключатель «Толщина
		 стрелок» на вкладке измерения метрик и отметьте элемент,
		 по значениям которого будет определяться толщина стрелок. Данная
		 характеристика является необязательной.
