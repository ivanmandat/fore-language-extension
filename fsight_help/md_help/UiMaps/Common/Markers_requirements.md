# Требования для отображения маркеров геолокации


Для карты доступно отображение [маркеров
 геолокации](../Configure/coordinate_marker.htm), если выполняются следующие условия:


	- Источник данных карты содержит единственное измерение объектов
	 геолокации. Данное измерение должно быть построено на основе справочника
	 НСИ, содержащего вещественные атрибуты «LATITUDE» и «LONGITUDE». Атрибуты
	 должны содержать значения широты и долготы для маркеров геолокации.


	- Для карты [указаны роли](Metrics.htm) для следующих
	 измерений: измерение метрик, измерение объектов геолокации.


	- Для заливки маркеров геолокации должно быть настроено [измерение
	 метрик](Metrics.htm#metrics). Сопоставлен элемент измерения метрик и следующая характеристика:


		- Цвет маркера. Установите
		 переключатель «Цвет маркера»
		 на вкладке измерения метрик и отметьте элемент, по значениям которого
		 будут закрашиваться маркеры.


	- Используется карта продукта «Форсайт. Аналитическая платформа»,
	 содержащая атрибут [<fs:Geo>](Specifications.chm::/SVGMapSpecification/Structure/BaseElements/fs_Geo_element.htm),
	 или карта [внешних
	 картографических сервисов](../Configure/Advanced_Features.htm#externalservices) Google или Yandex.


Примечание.
 Отображение маркеров геолокации недоступно в инструменте «Анализ
 временных рядов».
