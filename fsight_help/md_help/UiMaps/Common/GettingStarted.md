# Приступая к работе


Перед работой с картой убедитесь, что:


	- Выполняются [требования для работы
	 с картой](Requirements.htm).


	- Выполнена [настройка измерений для работы
	 с картой](Metrics.htm).


## Основные понятия


При работе с картой рекомендуется ознакомиться с основными понятиями:


	- Легенда. Набор условных
	 цветовых и символьных обозначений, применяемый для отображения на
	 карте различных метрик. Доступны следующие виды легенд:


	-


		- легенда цвета. Предназначена
		 для обозначения значений метрики, по которым [закрашиваются
		 территории](../Configure/Color.htm) карты;


		- легенда столбиковых диаграмм.
		 Предназначена для обозначения значений метрики, по которым закрашиваются
		 [столбиковые диаграммы](../Configure/Diagrams.htm)
		 на карте;


		- легенда круговых диаграмм.
		 Предназначена для обозначения значений метрики, по которым закрашиваются
		 [круговые диаграммы](../Configure/Diagrams.htm) на
		 карте;


		- легенда объема.
		 Предназначена для обозначения значений метрики, по которым определяется
		 [высота территорий](../Configure/Panel_Height.htm)
		 на карте в 3D-режиме;


		- легенда маркеров.
		 Предназначена для обозначения значений метрики, по которым определяется
		 цвет [маркеров геолокации](../Configure/coordinate_marker.htm);


		- легенда цвета стрелок.
		 Предназначена для обозначения значений метрики, по которым определяется
		 цвет [стрелок](../Configure/Arrows.htm);


		- легенда размера стрелок.
		 Предназначена для обозначения значений метрики, по которым определяется
		 размер [стрелок](../Configure/Arrows.htm).


	- Всплывающая подсказка.
	 Предназначена для отображения выбранных показателей при наведении
	 на регион карты. Текст [подсказки](../Configure/Tooltips.htm)
	 задается автоматически или может быть настроен вручную;


	- Расшифровка территорий.
	 Список территорий, отображаемых на карте. Наименования территорий
	 - это наименования элементов территориального измерения;


	- Территориальное измерение.
	 Измерение источника данных содержащее сведения о различных территориях;


	- Измерение метрик. Измерение
	 источника данных, содержащее различные категории данных для территорий;


	- Временная шкала. Элемент
	 для управления отображением того, как изменяются значения метрик с
	 течением времени.
