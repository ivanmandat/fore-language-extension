# Требования для детализации территорий


Для карты доступна [детализация](../Configure/Advanced_Features.htm#drilldown),
 если выполняются следующие условия:


	- Территориальное измерение является иерархическим.


	- Территориальное измерение содержит атрибут целого типа с идентификатором
	 «TOPO_ID».


	- Атрибут «TOPO_ID» содержит ключ карты, используемой для детализации
	 территории, соответствующей элементу территориального измерения.


	- Для каждого уровня иерархии территориального измерения должна
	 быть карта в репозитории.


Например, есть территориальное измерение, содержащее корневой элемент
 «Мир». «Мир» имеет дочерние элементы «Европа» и «Азия». Значения атрибутов
 «TOPO_ID» приведены в таблице:


		 Элементы
		 Значение атрибута «TOPO_ID»


		 Мир
		 100


		 Европа
		 200


		 Азия
		 300


Таким образом, для детализации территории «Европа» будет использоваться
 карта с ключом «200», для территории «Азия» - карта с ключом «300». Для
 обобщения территорий «Европа» и «Азия» будет использоваться карта с ключом
 «100».
