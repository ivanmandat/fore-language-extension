# Основные требования для работы
 с картой


Для корректной работы с картой должны выполняться следующие требования:


	- В репозиторий должна быть [загружена топооснова](Topobase.htm);


	- В репозитории должна быть задана карта, [используемая по умолчанию](UiNav.chm::/02_Navigator/Repo_Default.htm).
	 Требование актуально только для веб-приложения. В настольном приложении
	 при отсутствии карты по умолчанию будет предложено выбрать карту из
	 текущего репозитория;


	- В справочнике территорий, использующимся в качестве территориального
	 измерения источника данных, должен присутствовать строковый атрибут
	 с идентификатором «TERRID». Данный атрибут должен содержать коды территорий
	 по стандарту ISO 3166-1.

	Требования для инструмента «Анализ
	 временных рядов»: в БД временных рядов должен присутствовать
	 обязательный атрибут рядов, ссылающийся на данный справочник территорий;


	Примечание.
	 При [комбинировании
	 территориального измерения](UiSelection.chm::/Selection/Dimension_combine.htm) атрибут «TERRID» будет
	 недоступен. Карта будет доступна, если территориальное измерение не
	 входит в комбинированное.


	- Используемая карта должна содержать строковый атрибут «ISO»
	 со значениями кодов территорий по стандарту ISO 3166-1.

	Если карта не содержит атрибут «ISO», то она должна содержать целочисленный
	 атрибут «ID» со значениями идентификаторов территорий. Справочник
	 территорий должен содержать целочисленный атрибут «TERRID» со значениями,
	 соответствующим значениям атрибута «ID» для карты.

	Информацию о наличии атрибута «ISO» можно получить у разработчика карты.


В зависимости от аспекта работы с картой могут предъявляться следующие
 требования:


	- [требования для детализации
	 территорий](DrillDown_requirements.htm);


	- [требования для отображения контуров
	 территорий](Loop_requirements.htm);


	- [требования для отображения
	 маркеров геолокации](Markers_requirements.htm);


	- [требования для отображения
	 стрелок](Arrows_requirements.htm);


	- [Требования для отображения
	 версионной карты](Version_requirements.htm).
