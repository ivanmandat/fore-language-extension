# Требования для отображения версионной карты


[Версионная карта](../Configure/map_version.htm) доступна,
 если выполняются следующие условия:


	- [Территориальное измерение](Requirements.htm) основано
	 на версионном справочнике НСИ, то есть на справочнике, элементы которого
	 [могут
	 изменяться во времени](UiNavObj.chm::/reference_book/Master_RDS_reference_book/base_settings.htm#change_in_time).


	- Территориальное измерение содержит атрибут целого типа с идентификатором
	 вида «TOPO_ID».


	- Элементы территориального измерения, которые изменяют свои границы,
	 содержат несколько периодов, на которых этот элемент актуален.


	- Для каждого периода актуальности атрибут «TOPO_ID» содержит
	 ключ карты, используемый для визуализации территории на данном периоде.
