# Работа с готовым обновлением


Шаги для работы с готовым обновлением зависят от целей использования:


	- работа с готовым обновлением в [менеджере
	 обновлений](Admin_UpMBObj_RunManager.htm#open_update). При [открытии
	 готового обновления](Admin_UpMBObj_RunManager.htm#open_update) содержимое структуры обновления зависит от
	 процесса обновления при использовании [разделения
	 ролей администраторов](FAQ/Administrator_role_separation.htm) и [разных
	 методов разграничения доступа](FAQ/Access_control_methods.htm). Также при открытии готового обновления
	 существует ряд [особенностей](OpenUpdate/Admin_OpenUpdate.htm);


	- [установка обновления](Admin_UpMBObj_DoUpdate.htm)
	 из [навигатора
	 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm) или [менеджера безопасности](Admin.chm::/admin_title.htm).


Примечание.
 В веб-приложении доступно только [открытие
 готового обновления](Admin_UpMBObj_RunManager.htm#open_update) на просмотр и [установка
 обновления](Admin_UpMBObj_DoUpdate.htm) из навигатора объектов.


## Работа с готовым обновлением в настольном приложении


	- Установите [синхронизацию с репозиторием](AdminUpMGObj_Syn.htm)
	 при необходимости.


	- [Экспортируйте список обновления](Admin_UpMBObj_Export.htm)
	 в текстовый файл при необходимости.


## Установка обновления из навигатора


	- В [навигаторе
	 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm) откройте готовый файл обновления для установки.


	- [Установите обновление](Admin_UpMBObj_DoUpdate.htm).
