# Удаление объектов из обновления


Для удаления объектов из обновления:


	- в мастере обновлений/менеджере обновлений:


		- в веб-приложении нажмите кнопку ![](Delete_button_menu.png) «Удалить объект из обновления»
		 на панели инструментов;


		- в настольном приложении выполните одно из действий:


			- выполните команду «Удалить»
			 в контекстном меню объекта в структуре обновления;


			- нажмите сочетание клавиш CTRL+DELETE;


			- нажмите кнопку ![](Admin_CreateUpdate_DelObjects_Button.gif) «Удалить объект из обновления»,
			 расположенную на панели инструментов;


	- в [навигаторе
	 объектов](GetStarted.chm::/interface/interface_navigator.htm) выполните команду «Удалить
	 из обновления» в контекстном меню объекта.


Для удаления всех объектов из обновления в веб-приложении выполните
 команду «Очистить список обновлений»
 в раскрывающемся меню кнопки ![](Delete_button_menu.png)
 «Удалить объект из обновления».


## Удаление всех объектов одного типа


Для удаления из обновления всех объектов одного типа, например, все
 таблицы, выделите один объект данного типа и выполните команду контекстного
 меню «Удалить все объекты класса "Таблица"».


Примечание.
 Удаление всех объектов одного типа из обновления доступно только в настольном
 приложении.


После выполнения данного действия будет выдан запрос на подтверждение удаления
 из обновления всех объектов данного типа.


Примечание.
 При удалении всех объектов одного типа справочника, дочерние объекты кубов
 не удаляются.
