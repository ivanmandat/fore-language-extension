# Добавление SQL-оператора


SQL-оператор предназначен для формирования новых и изменения уже имеющихся
 пользовательских таблиц.


Для добавления SQL-оператора:


	- выберите тип объекта обновления «SQL-оператор»
	 в раскрывающемся меню кнопки ![](admin_createupdate_addsql_button.gif) «Добавить
	 объект в обновление» на панели инструментов;


	- выполните команду «Добавить
	 SQL-оператор» в контекстном меню [структуры
	 обновления](../Admin_UpMBObj_RunManager.htm). Доступно только в настольном приложении.


После выполнения одного из действий будет открыто окно «[SQL-оператор](Admin_ParamSQL.htm)».


Примечание.
 При разделении [ролей
 администраторов](Admin.chm::/04_SecurityPolicy/Editor_of_Politicy/Security_EditorPoliticy_Adm.htm) создание SQL-оператора доступно только
 прикладному администратору.


## Редактирование параметров SQL-оператора


Для редактирования параметров выбранного SQL-оператора:


	- выполните пункт контекстного меню «Редактировать»;


	- дважды щёлкните по SQL-оператору;


	- нажмите клавишу F4.


После выполнения одного из действий будет открыт диалог «[SQL-оператор](Admin_ParamSQL.htm)».


## Переименование SQL-оператора


Для изменения наименования выбранного SQL-оператора:


	- выполните пункт контекстного меню «Переименовать»;


	- щёлкните по SQL-оператору;


	- нажмите клавишу F2.


После выполнения одного из действий переименуйте SQL-оператор.
