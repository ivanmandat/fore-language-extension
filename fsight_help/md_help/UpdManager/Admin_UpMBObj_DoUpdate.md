# Установка обновления


Установку обновления может производить:


	- администратор ADMIN (владелец схемы);


	- администратор информационной безопасности (АИБ), если [роли
	 администраторов](Admin.chm::/04_SecurityPolicy/Editor_of_Politicy/Security_EditorPoliticy_Adm.htm) разделены;


	- пользователь, входящий во встроенную группу «[Администраторы](Admin.chm::/03_Admin/Groups/Admin_Groups.htm)»;


	- пользователь/член группы, обладающий привилегией «[Доступ
	 к менеджеру обновлений](admin.chm::/04_SecurityPolicy/Admin_Priv.htm#upd_manager)».


Важно. Установка
 обновления метаданных [объектов
 базы данных](Admin.chm::/04_SecurityPolicy/Object_classes.htm#minitocbookmark6) или изменение прав доступа на уровне СУБД
 для этих объектов выполняется только под учетной записью владельца схемы
 - ADMIN. Если установка обновления или изменение прав доступа выполняется
 АИБ или пользователем, то при запросе учетных данных пользователя для
 [авторизации
 в базе данных](UiNavObj.chm::/database/UiDb_database_authorization.htm) необходимо ввести учётные данные владельца
 схемы.


Перед установкой обновления ознакомьтесь с разделом «[Особенности
 установки обновления](UPD_features.htm)».


Для установки обновления выполните одно из действий:


	- в веб-приложении:


		- в [навигаторе
		 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm) выполните команду «Установить
		 обновление» в разделе ![](CreateUpdate/b_update_manager.png) «Менеджер
		 обновлений» на боковой панели;


		- используйте приложение [UpdateManager](Setup.chm::/05_RepoMngr/Service_Applications/UpdateManager.htm);


	- в настольном приложении:


		- в [навигаторе
		 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm) выполните команду главного меню «Навигатор > Обновить объекты репозитория»;


		- в [менеджере безопасности](Admin.chm::/Admin_Title.htm)
		 выполните команду главного меню «Репозиторий
		 > Обновление версии > Произвести».


Примечание.
 При разделении [ролей
 администраторов](Admin.chm::/04_SecurityPolicy/Editor_of_Politicy/Security_EditorPoliticy_Adm.htm) для корректной установки используйте
 [файл обновления](SaveUpdate/Admin_SaveUpdate.htm#in_file)
 соответствующего типа.


После выполнения одного из действий будет открыт:


	- мастер установки обновления в веб-приложении. Для выбора файла
	 обновления нажмите кнопку «Обзор»,
	 будет открыт стандартный диалог выбора файла. Укажите путь к файлу
	 обновления с расширением *.pefx или *.updpack;


	- стандартный диалог выбора файла в настольном приложении. Укажите
	 путь к файлу обновления с расширением *.pefx. После выбора файла нажмите
	 кнопку «Открыть». Будет открыт
	 мастер установки обновления.
