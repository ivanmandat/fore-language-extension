# Субъекты обновления


На вкладке «Субъекты» страницы
 «[Информация](Admin_UpMBObj_DoUpdateFile.htm)»
 отображается список субъектов безопасности из обновления, для которых
 обновляются права доступа, но при проверке субъекты из обновления не были
 сопоставлены с субъектами безопасности репозитория. Субъекты безопасности
 не имеющие прав доступа на объекты обновления, но присутствующие в файле
 обновления отображаются на вкладке «[Объекты
 обновления](Update_Objects.htm)».


Примечание.
 Вкладка «Субъекты» доступна только
 в настольном приложении.


![](UpdateMBObjects/update_subjects.png)


Общее количество субъектов безопасности из обновления, для которых обновляются
 права доступа, отображается в заголовке вкладки.


На вкладке содержится информация о наименовании
 субъекта безопасности, SID, описание субъекта и действие над субъектом.


Для субъектов доступно контекстное меню:


	- Сопоставить с субъектом.
	 Для сопоставления будет открыт стандартный диалог поиска субъекта
	 безопасности. Права несопоставленных субъектов пропускаются при обновлении.
