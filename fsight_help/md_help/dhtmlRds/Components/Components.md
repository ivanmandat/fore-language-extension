# Работа со справочником НСИ


## Описание


На данной странице приведены компоненты для работы со справочником НСИ.


## Компоненты


		 Имя компонента
		 Краткое описание


		 ![](../Class_Image.gif)
		 [CleanUpDialog](Rds/CleanUpDialog/CleanUpDialog.htm)
		 Компонент CleanUpDialog
		 реализует диалог для очистки справочника НСИ.


		 ![](../Class_Image.gif)
		 [DataArea](Rds/DataArea/DataArea.htm)
		 Компонент DataArea
		 реализует область данных справочника НСИ.


		 ![](../Class_Image.gif)
		 [DictionaryBox](Rds/DictionaryBox/DictionaryBox.htm)
		 Компонент DictionaryBox
		 реализует контейнер для справочника НСИ.


		 ![](../Class_Image.gif)
		 [DictionaryTree](Rds/DictionaryTree/DictionaryTree.htm)
		 Компонент DictionaryTree
		 реализует дерево элементов справочника НСИ.


		 ![](../Class_Image.gif)
		 [ElementCard](Rds/ElementCard/ElementCard.htm)
		 Компонент ElementCard
		 реализует карточку элемента дерева справочника НСИ.


		 ![](../Class_Image.gif)
		 [FindBox](Rds/FindBox/FindBox.htm)
		 Компонент FindBox реализует
		 контейнер, предназначенный для поиска элементов справочника НСИ.


		 ![](../Class_Image.gif)
		 [FindSettingsDialog](Rds/FindSettingsDialog/FindSettingsDialog.htm)
		 Компонент FindSettingsDialog
		 реализует диалог настроек параметров поиска по справочнику НСИ.


		 ![](../Class_Image.gif)
		 [ParamsBox](Rds/ParamsBox/ParamsBox.htm)
		 Компонент ParamsBox
		 реализует контейнер параметров справочника.
