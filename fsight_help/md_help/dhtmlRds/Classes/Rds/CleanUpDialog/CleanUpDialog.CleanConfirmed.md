# CleanUpDialog.CleanConfirmed


## Синтаксис


CleanConfirmed: function(sender, args: Object);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: Params - параметры справочника.


## Описание


Событие CleanConfirmed генерируется после нажатия на кнопку «Очистить» в диалоге для очистки справочника.
