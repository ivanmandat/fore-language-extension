# ElementCard.Locale


## Синтаксис


Locale: Number;


## Описание


Свойство Locale определяет код локали карточки элемента дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLocale, а возвращается с помощью метода getLocale.**
