# ElementCard.Mode


## Синтаксис


Mode: [PP.Rds.ElementOperation](../../../Enums/ElementOperation.htm).Insert || [PP.Rds.ElementOperation](../../../Enums/ElementOperation.htm).Update;


## Описание


Свойство Mode определяет режим работы карточки элемента.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMode, а возвращается с помощью метода getMode.**


**Карточка элемента может работать в режиме добавления или в режиме редактирования элемента.**
