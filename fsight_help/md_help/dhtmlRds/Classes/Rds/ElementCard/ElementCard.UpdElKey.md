# ElementCard.UpdElKey


## Синтаксис


UpdElKey: Number;


## Описание


Свойство UpdElKey определяет ключ редактируемого элемента.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setUpdElKey, а возвращается с помощью метода getUpdElKey.**
