# ElementCard.checkValues


## Синтаксис


checkValues();


## Описание


Свойство checkValues проверяет корректность введенных значений.


## Комментарии


Свойство возвращает значение типа Boolean. Если метод возвращает значение true, значит значения корректны, иначе - некорректны.
