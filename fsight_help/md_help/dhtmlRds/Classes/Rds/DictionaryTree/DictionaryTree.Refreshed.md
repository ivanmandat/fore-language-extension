# DictionaryTree.Refreshed


## Синтаксис


Refreshed: function(sender, args, timeout);


## Параметры


sender. Источник события;


args. Информация о событии;


timeout. Промежуток времени в миллисекундах, через который вызовется событие.


## Описание


Событие Refreshed наступает после обновления дерева элементов справочника.
