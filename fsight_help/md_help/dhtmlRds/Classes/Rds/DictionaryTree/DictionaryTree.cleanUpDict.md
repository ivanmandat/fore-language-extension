# DictionaryTree.cleanUpDict


## Синтаксис


cleanUpDict(params: Object);


## Параметры


params. Параметры очистки справочника. Если параметр не указан, но будет очищен весь справочник.


## Описание


Метод cleanUpDict очищает справочник на основе указанных параметров.
