# Конструктор DictionaryTree


## Синтаксис


PP.Rds.Ui.DictionaryTree(settings);


## Параметры


settings. JSON-объект со значениями свойств класса.


## Описание


Конструктор DictionaryTree создает экземпляр класса [DictionaryTree](DictionaryTree.htm).
