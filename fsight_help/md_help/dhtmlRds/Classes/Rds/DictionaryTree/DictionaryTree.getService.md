# DictionaryTree.getService


## Синтаксис


getService();


## Описание


Метод getService возвращает сервис, предназначенный для работы со справочником НСИ.


## Комментарии


Метод возвращает значение типа [PP.Rds.RdsService](../RdsService/RdsService.htm).
