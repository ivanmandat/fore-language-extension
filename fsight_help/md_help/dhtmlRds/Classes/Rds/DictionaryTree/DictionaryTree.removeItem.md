# DictionaryTree.removeItem


## Синтаксис


removeItem(keys: Array, showConfirm: Boolean);


## Параметры


keys. Массив ключей удаляемых элементов;


showConfirm. Признак запроса подтверждения удаления элементов. Если параметр имеет значение true, то запрос будет отображен, иначе - не будет.


## Описание


Метод removeItem удаляет из дерева элементы с указанными ключами и все их дочерние элементы.
