# DictionaryTree.ElementsLocale


## Синтаксис


ElementsLocale: Number;


## Описание


Свойство ElementsLocale определяет код региональных настроек элементов дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setElementsLocale, а возвращается с помощью метода getElementsLocale.**
