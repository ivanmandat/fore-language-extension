# DictionaryTree.pasteClipboard


## Синтаксис


pasteClipboard(key: Number);


## Параметры


key. Ключ элемента дерева, в который необходимо произвести вставку. Если параметр не указан, то вставка будет происходить в последний выделенный элемент.


## Описание


Метод pasteClipboard выполняет операцию вставки элемента из буфера обмена.
