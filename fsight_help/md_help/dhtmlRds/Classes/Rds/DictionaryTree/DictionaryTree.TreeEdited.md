# DictionaryTree.TreeEdited


## Синтаксис


TreeEdited: function(sender, args, timeout);


## Параметры


sender. Источник события;


args. Информация о событии;


timeout. Промежуток времени в миллисекундах, через который вызовется событие.


## Описание


Событие TreeEdited наступает после редактирования дерева элементов справочника.
