# DictionaryTree.refresh


## Синтаксис


refresh(reinit: Boolean);


## Параметры


reinit. Признак повторного открытия справочника перед его обновлением. Если значение параметра равно true, то справочник будет открыт повторно перед обновлением, иначе - не будет.


## Описание


Метод refresh обновляет дерево элементов справочника.
