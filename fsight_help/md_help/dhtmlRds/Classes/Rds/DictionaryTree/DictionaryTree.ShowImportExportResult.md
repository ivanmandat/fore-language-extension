# DictionaryTree.ShowImportExportResult


## Синтаксис


ShowImportExportResult: Boolean;


## Описание


Свойство ShowImportExportResult
 определяет отображение сообщений о результатах импорта или экспорта.


## Комментарии


Значение свойства устанавливается из JSON, а возвращается - с помощью
 метода getShowHidden.


По умолчанию содержит значение true
 - отображать сообщения о результатах импорта и экспорта.
