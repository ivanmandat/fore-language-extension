# DictionaryTree.refreshAll


## Синтаксис


refreshAll(reInit: Boolean);


## Параметры


reInit. Признак переоткрытия справочника. Если параметр имеет значение true, то справочник будет переоткрыт, иначе - не будет.


## Описание


Метод refreshAll обновляет дерево элементов справочника.


## Комментарии


Результат работы метода аналогичен результату работы метода [refresh](DictionaryTree.refresh.htm).
