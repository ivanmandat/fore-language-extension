# DictionaryTree.pushToClipboard


## Синтаксис


pushToClipboard(operation: [PP.Rds.Ui.TreeOperations](../../../Enums/TreeOperations.htm).CopyElements || [PP.Rds.Ui.TreeOperations](../../../Enums/TreeOperations.htm).CutElements, keys: Array);


## Параметры


operation. Операция с буфером обмена;


keys. Ключи элементов, которые необходимо внести в буфер. Если параметр не указан, то в буфер заносятся все выделенные вершины.


## Описание


Метод pushToClipboard помещает указанные элементы дерева в буфер обмена.
