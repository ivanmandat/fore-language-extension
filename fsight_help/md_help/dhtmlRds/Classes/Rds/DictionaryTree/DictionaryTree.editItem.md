# DictionaryTree.editItem


## Синтаксис


editItem(key: Number, dItem: [PP.Rds.DItem](../DItem/DItem.htm));


## Параметры


key. Ключ редактируемого элемента. Если параметр не задан, то используется ключ последнего выделенного элемента.


dItem. Элемент, на который заменяется редактируемый элемент. Если параметр не указан, то будет отображен компонент [ElementCard](../ElementCard/ElementCard.htm) для редактирования элемента.


## Описание


Метод editItem редактирует элемент дерева справочника.
