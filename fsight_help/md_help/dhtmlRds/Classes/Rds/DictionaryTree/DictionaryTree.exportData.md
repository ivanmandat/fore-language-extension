# DictionaryTree.exportData


## Синтаксис


exportData(schemaKey: String);


## Параметры


schemaKey. Ключ схемы экспорта.


## Описание


Метод exportData инициализирует
 процесс экспорта данных по указанной схеме и обновляет дерево.
