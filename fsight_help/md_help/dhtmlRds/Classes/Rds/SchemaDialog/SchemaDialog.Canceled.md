# SchemaDialog.Canceled


## Синтаксис


Canceled: function(sender, args: Object);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие Canceled генерируется
 после нажатия на кнопку «Отмена» в диалоге для выбора схемы экспорта/импорта.
