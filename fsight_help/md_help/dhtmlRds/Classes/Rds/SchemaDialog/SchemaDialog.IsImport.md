# SchemaDialog.IsImport


## Синтаксис


IsImport: Boolean;


## Описание


Свойство IsImport определяет,
 будет ли диалог использован для выбора схемы импорта.


## Комментарии


Значение свойства устанавливается из JSON, а возвращается - с помощью
 метода getIsImport.


По умолчанию содержит значение false
 - диалог будет настроен на выбор схем экспорта.
