# SchemaDialog.Confirmed


## Синтаксис


Confirmed: function(sender, args: Object);


## Параметры


sender. Источник события;


args. Информация о событии.
 Доступные аргументы: Params - параметры справочника.


## Описание


Событие Confirmed генерируется
 после нажатия на кнопку «Ок» в диалоге для выбора схемы экспорта/импорта.
