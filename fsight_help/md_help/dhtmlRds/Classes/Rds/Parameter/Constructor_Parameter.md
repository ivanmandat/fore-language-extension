# Конструктор Parameter


## Синтаксис


PP.Rds.Ui.Parameter(settings);


## Параметры


settings. JSON-объект со значениями свойств класса.


## Описание


Конструктор Parameter создает экземпляр класса [Parameter](Parameter.htm).


## Пример


Конструктор предназначен для внутреннего использования.
