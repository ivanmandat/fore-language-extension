# Dictionary.getBlankElement


## Синтаксис


getBlankElement(key: Number);


## Параметры


key. Ключ, который требуется задать новому элементу.


## Описание


Метод getBlankElement возвращает пустой элемент, количество и порядок атрибутов которого соответствует структуре справочника НСИ.


## Комментарии


Метод возвращает объект типа [PP.Rds.DItem](../DItem/DItem.htm).
