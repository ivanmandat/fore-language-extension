# Dictionary.isPlain


## Синтаксис


isPlain: Boolean;


## Описание


Метод isPlain определяет, содержит ли элемент справочника НСИ дочерние элементы.


## Комментарии


Метод возвращает значение true, если элемент справочника элемент дочерние элементы, и значение false, если не имеет.
