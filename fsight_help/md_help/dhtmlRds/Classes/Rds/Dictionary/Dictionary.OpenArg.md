# Dictionary.OpenArg


## Синтаксис


OpenArg: [PP.Mb.Param](dhtmlMetabase.chm::/Classes/Metabase/Param/Param.htm);


## Описание


Свойство OpenArg определяет
 параметр, с которыми открывается справочник НСИ.


## Комментарии


Значение свойства устанавливается с помощью метода setOpenArg,
 а возвращается - с помощью метода getOpenArg.
 Из JSON значение задать нельзя.


При вызове метода setOpenArg
 используется аргумент argMd - настройки параметра в формате метаданных.


Метод getOpenArg имеет аргумент
 id (идентификатор параметра) и возвращает значение типа [PP.Mb.Param](dhtmlMetabase.chm::/Classes/Metabase/Param/Param.htm).
