# Dictionary.TotalsCount


## Синтаксис


TotalsCount: Number;


## Описание


Свойство TotalsCount определяет общее количество элементов в справочнике НСИ.


## Комментарии


Значение свойства устанавливается с помощью метода setTotalsCount, а возвращается - с помощью метода getTotalsCount. Из JSON значение задать нельзя.
