# Dictionary.IsMultilingual


## Синтаксис


isMultilingual: Boolean;


## Описание


Метод isMultilingual определяет,
 является ли справочник НСИ мультиязычным.


## Комментарии


Метод возвращает значение true,
 если справочник НСИ является мультиязычным, иначе - значение false.
