# Dictionary.isHistorical


## Синтаксис


isHistorical: Boolean;


## Описание


Метод isHistorical определяет,
 является ли справочник НСИ версионным.


## Комментарии


Метод возвращает значение true,
 если справочник НСИ является версионным, иначе - значение false.
