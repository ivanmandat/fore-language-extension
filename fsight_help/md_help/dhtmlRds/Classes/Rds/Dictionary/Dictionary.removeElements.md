# Dictionary.removeElements


## Синтаксис


removeItems(keys: Array);


## Параметры


keys. Массив ключей удаляемых элементов. Элементы массива имеют строковый тип.


## Описание


Метод removeItems удаляет указанные элементы из справочника НСИ.
