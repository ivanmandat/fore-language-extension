# Dictionary.getElem


## Синтаксис


getElem(key: Number);


## Параметры


key. Ключ элемента справочника НСИ.


## Описание


Метод getElem возвращает элемент справочника НСИ по его ключу.


## Комментарии


Метод возвращает значение типа [PP.Rds.DItem](../DItem/DItem.htm).
