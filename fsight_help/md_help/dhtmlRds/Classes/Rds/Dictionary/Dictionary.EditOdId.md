# Dictionary.EditOdId


## Синтаксис


EditOdId: String;


## Описание


Свойство EditOdId определяет моникёр справочника НСИ для редактирования его структуры.


## Комментарии


Значение свойства устанавливается с помощью метода setEditOdId, а возвращается - с помощью метода getEditOdId. Из JSON значение задать нельзя.
