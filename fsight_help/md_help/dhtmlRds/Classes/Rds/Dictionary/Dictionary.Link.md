# Dictionary.Link


## Синтаксис


Link: Object;


## Описание


Свойство Link определяет связь между справочниками НСИ.


## Комментарии


Значение свойства устанавливается с помощью метода setLink, а возвращается - с помощью метода getLink. Из JSON значение задать нельзя.


Метод getLink имеет параметр key - ключ связи.


Свойство устанавливает и возвращает объект связи в формате метаданных, который содержит следующие свойства: attribute - атрибут справочника, по которому установлена связь, refAttribute - атрибут, на который ссылаемся, id - идентификатор связи, k - его ключ, n - наименование и другие.
