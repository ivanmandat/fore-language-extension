# Dictionary.Attribute


## Синтаксис


Attribute: [PP.Rds.Attribute](../Attribute/Attribute.htm);


## Описание


Свойство Attribute определяет атрибут справочника НСИ.


## Комментарии


Значение свойства устанавливается с помощью метода setAttribute, а возвращается - с помощью метода getAttribute. Из JSON значение задать нельзя.


При вызове метода setAttribute используется параметр attr - объект атрибута в формате метаданных.


Метод getAttribute имеет два параметра: key - ключ атрибута, id - идентификатор атрибута. Для получения объекта атрибута достаточно указать один из двух параметров.
