# Dictionary.hasHidden


## Синтаксис


hasHidden();


## Описание


Метод hasHidden определяет, может ли справочник иметь скрытые элементы.


## Комментарии


Метод возвращает значение true, если справочник может иметь скрытые элементы, иначе - значение false.
