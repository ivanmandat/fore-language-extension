# Dictionary.setElement


## Синтаксис


setElement(parentKey: Number, elMd: Object|[PP.Rds.DItem](../DItem/DItem.htm), updateTotalCount: Boolean);


## Параметры


parentKey. Ключ родительского элемента;


elMd. Объект с настройками элемента в формате метаданных или элемент справочника НСИ;


updateTotalCount. Признак обновления счётчика общего количества элементов. Если параметр равен значению true, то счётчик общего количество элемента будет обновлён, иначе - не будет.


## Описание


Метод setElement добавляет или изменяет элемент в справочнике НСИ.


## Комментарии


Если в качестве параметра elMd указан существующий элемент, то он будет обновлён, иначе - будет добавлен в справочник.
