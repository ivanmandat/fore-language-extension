# Dictionary.hasVisibleParams


## Синтаксис


hasVisibleParams();


## Описание


Метод hasVisibleParams определяет, содержит ли справочник НСИ доступные для отображения параметры.


## Комментарии


Метод возвращает значение true, если справочник НСИ содержит доступные для отображения параметры, иначе - значение false.
