# Dictionary.getDefaultTranslation


## Синтаксис


getDefaultTranslation();


## Описание


Метод getDefaultTranslation возвращает для справочника настройки перевода по умолчанию.


## Комментарии


Метод возвращает объект, установленный в свойстве [Dictionary.Translation](Dictionary.Translation.htm).
