# Dictionary.getFilterChildEls


## Синтаксис


getFilterChildEls(key: Number);


## Параметры


key. Ключ родительского элемента справочника НСИ.


## Описание


Метод getFilterChildEls возвращает дочерние отфильтрованные элементы по ключу родительского элемента справочника НСИ.


## Комментарии


Метод возвращает массив объектов типа [PP.Rds.DItem](../DItem/DItem.htm).
