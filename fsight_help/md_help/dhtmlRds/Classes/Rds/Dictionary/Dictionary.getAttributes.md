# Dictionary.getAttributes


## Синтаксис


getAttributes();


## Описание


Метод getAttributes возвращает массив атрибутов справочника НСИ.


## Комментарии


Метод возвращает массив объектов типа [PP.Rds.Attribute](../Attribute/Attribute.htm) по порядку их следования в метаданных.
