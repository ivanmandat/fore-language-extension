# Dictionary.Parameter


## Синтаксис


Parameter: Object;


## Описание


Свойство Parameter определяет параметр справочника НСИ.


## Комментарии


Значение свойства устанавливается с помощью метода setParameter, а возвращается - с помощью метода getParameter. Из JSON значение задать нельзя.


При вызове метода setParameter используется аргумент param - параметр справочника в формате метаданных.


Метод getParameter имеет аргумент key (ключ параметра) и возвращает параметр в формате метаданных.
