# RdsService.getToKeys


## Синтаксис


getToKeys(dict: [PP.Rds.Dictionary](../Dictionary/Dictionary.htm), elKeys: Array, callback: function|PP.Delegate);


## Параметры


dict. Справочник НСИ;


elKeys. Массив ключей элементов, до которых должна быть загружена структура дерева;


callback. Возвратная функция.


## Описание


Метод getToKeys загружает структуру дерева до указанных элементов справочника НСИ.
