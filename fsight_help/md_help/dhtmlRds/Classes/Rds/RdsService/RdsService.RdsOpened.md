# RdsService.RdsOpened


## Синтаксис


RdsOpened: function(sender, args: Object);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: Dictionary - справочник НСИ.


## Описание


Событие RdsOpened генерируется после открытия справочника НСИ.
