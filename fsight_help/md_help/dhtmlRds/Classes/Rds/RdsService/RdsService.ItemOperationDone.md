# RdsService.ItemOperationDone


## Синтаксис


ItemOperationDone: function(sender, args: Object);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: ParentKey - ключ родительского элемента, Operation - тип операции, Keys - ключи копий элементов.


## Описание


Событие ItemOperationDone генерируется после выполнения операции вставки, обновления или удаления элементов справочника НСИ.
