# RdsService.ChildLoaded


## Синтаксис


ChildLoaded(sender, args: Object);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: Dictionary - справочник НСИ, IsFilter - признак того, что фильтр успешно применён, ParentKey - ключ родительского элемента.


## Описание


Событие ChildLoaded генерируется после загрузки дочерних элементов справочника НСИ.
