# RdsService.RdsClosed


## Синтаксис


RdsClosed: function(sender, args: Object);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: Dictionary - справочник НСИ.


## Описание


Событие RdsClosed генерируется после закрытия справочника НСИ.
