# RdsService.getIsBatchMode


## Синтаксис


getIsBatchMode();


## Описание


Метод getIsBatchMode возвращает
 признак того, что сервис находится в режиме формирования пакета данных
 запроса для отправки серверу.


## Комментарии


Метод возвращает значение true,
 если сервис формирует пакет данных запроса, иначе - значение false.
