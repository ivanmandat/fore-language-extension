# RdsService.closeRds


## Синтаксис


closeRds(dict: [PP.Rds.Dictionary](../Dictionary/Dictionary.htm), callback: function|PP.Delegate);


## Параметры


dict. Справочник НСИ, который требуется закрыть;


callback. Возвратная функция.


## Описание


Метод closeRds закрывает указанный справочник.
