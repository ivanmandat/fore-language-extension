# RdsService.removeItems


## Синтаксис


removeItems(dict: [PP.Rds.Dictionary](../Dictionary/Dictionary.htm), elKeys: Array, callback: function|PP.Delegate);


## Параметры


dict. Справочник НСИ, в котором требуется удалить элемент;


elKeys. Массив ключей удаляемых элементов. Элементы массива имеют строковый тип;


callback. Возвратная функция.


## Описание


Метод removeItems удаляет указанные элементы из справочника НСИ.
