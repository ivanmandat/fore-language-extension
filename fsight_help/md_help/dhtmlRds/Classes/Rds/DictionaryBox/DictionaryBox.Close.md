# DictionaryBox.Close


## Синтаксис


Close: function(sender, args: Object);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: ActiveSource - закрываемый справочник НСИ.


## Описание


Событие Close генерируется при закрытии контейнера для справочника НСИ.
