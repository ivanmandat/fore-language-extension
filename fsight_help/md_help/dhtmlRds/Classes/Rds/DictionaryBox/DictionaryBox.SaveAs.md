# DictionaryBox.SaveAs


## Синтаксис


SaveAs: function(sender, args: Object);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: ActiveSource - сохраняемый справочник НСИ.


## Описание


Событие SaveAs генерируется при сохранении данных справочника НСИ с новым именем.
