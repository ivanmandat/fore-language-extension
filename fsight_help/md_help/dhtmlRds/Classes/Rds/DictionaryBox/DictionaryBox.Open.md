# DictionaryBox.Open


## Синтаксис


Open: function(sender, args: Object);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: ActiveSource - открываемый справочник НСИ.


## Описание


Событие Open генерируется при открытии контейнера для справочника НСИ.
