# DictionaryBox.Source


## Синтаксис


Source: Array;


## Описание


Свойство Source определяет для контейнера справочник НСИ.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSource, а возвращается - с помощью метода getSource.


Свойство содержит массив объектов типа [PP.Rds.Dictionary](../Dictionary/Dictionary.htm).
