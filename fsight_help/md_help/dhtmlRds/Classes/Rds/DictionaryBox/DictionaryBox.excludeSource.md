# DictionaryBox.excludeSource


## Синтаксис


excludeSource(value: [PP.Rds.Dictionary](../Dictionary/Dictionary.htm));


## Параметры


value. Справочник НСИ, который нужно удалить из контейнера.


## Описание


Метод excludeSource удаляет из контейнера указанный справочник НСИ.
