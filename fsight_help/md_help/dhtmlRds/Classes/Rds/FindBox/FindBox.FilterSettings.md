# FindBox.FilterSettings


## Синтаксис


FilterSettings: Object;


## Описание


Свойство FilterSettings определяет настройки поиска.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setFilterSettings, а возвращается с помощью метода getFilterSettings.**


**Метод** setFilterSettings принимает следующие параметры: value - настройки поиска, fireEvent - признак вызова события смены настроек, forced - признак вызова события смены настроек даже при отсутствии их изменений.
