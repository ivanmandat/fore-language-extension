# Конструктор FindBox


## Синтаксис


PP.Rds.Ui.FindBox(settings);


## Параметры


settings. JSON-объект со значениями свойств класса.


## Описание


Конструктор FindBox создает экземпляр класса [FindBox](FindBox.htm).
