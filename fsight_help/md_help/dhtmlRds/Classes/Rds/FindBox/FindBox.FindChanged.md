# FindBox.FindChanged


## Синтаксис


FindChanged: function(sender, args, timeout);


## Параметры


sender. Источник события;


args. Информация о событии;


timeout. Промежуток времени в миллисекундах, через который вызовется событие.


## Описание


Событие FindChanged наступает после изменения найденного результата.
