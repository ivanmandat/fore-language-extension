# Attribute.MasterAttrKey


## Синтаксис


MasterAttrKey: Number;


## Описание


Свойство MasterAttrKey определяет для атрибута-перевода ключ главного атрибута, для которого требуется выполнить перевод.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMasterAttrKey, а возвращается - с помощью метода getMasterAttrKey.
