# Attribute.IsTranslation


## Синтаксис


IsTranslation: Boolean;


## Описание


Свойство IsTranslation определяет, доступен ли перевод для атрибута.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsTranslation, а возвращается - с помощью метода getIsTranslation.


По умолчанию свойство содержит значение false.
