# Attribute.ArgKey


## Синтаксис


ArgKey: String;


## Описание


Свойство ArgKey определяет ключ аргумента, управляющего значением атрибута.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setArgKey, а возвращается - с помощью метода getArgKey.
