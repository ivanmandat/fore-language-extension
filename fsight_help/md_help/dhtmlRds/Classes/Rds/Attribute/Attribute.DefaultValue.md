# Attribute.DefaultValue


## Синтаксис


DefaultValue: String;


## Описание


Свойство DefaultValue определяет для атрибута значение по умолчанию.


## Комментарии


Значение свойства устанавливается с помощью метода setDefaultValue, а возвращается - с помощью метода getDefaultValue. Из JSON значение задать нельзя.
