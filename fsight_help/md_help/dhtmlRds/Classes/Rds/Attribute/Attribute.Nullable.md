# Attribute.Nullable


## Синтаксис


Nullable: Boolean;


## Описание


Свойство Nullable определяет, может ли атрибут иметь пустое значение.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setNullable, а возвращается - с помощью метода getNullable.


Если свойство содержит значение true, то атрибут может иметь пустое значение, иначе - не может (по умолчанию).
