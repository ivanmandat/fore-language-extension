# Attribute.WriteAccess


## Синтаксис


WriteAccess: String;


## Описание


Свойство WriteAccess определяет права на изменение атрибута справочника НСИ.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setWriteAccess, а возвращается - с помощью метода getWriteAccess.


Свойство содержит порядковый номер субъекта доступа, которому требуется дать права на изменение атрибута справочника НСИ.
