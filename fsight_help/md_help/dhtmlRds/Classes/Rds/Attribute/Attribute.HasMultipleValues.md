# Attribute.HasMultipleValues


## Синтаксис


HasMultipleValues: Boolean;


## Описание


Свойство HasMultipleValues определяет, может ли иметь атрибут множественные значения.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHasMultipleValues, а возвращается - с помощью метода getHasMultipleValues.


Если свойство содержит значение true, то атрибут может иметь множественные значения, иначе - не может (по умолчанию).
