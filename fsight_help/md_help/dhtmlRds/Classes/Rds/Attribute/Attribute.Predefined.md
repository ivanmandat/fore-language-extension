# Attribute.Predefined


## Синтаксис


Predefined: [PP.Rds.PredefinedAttribute](../../../Enums/PredefinedAttribute.htm);


## Описание


Свойство Predefined определяет, к какому предопределённому атрибуту относится заданный атрибут.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setOwner, а возвращается - с помощью метода getOwner.
