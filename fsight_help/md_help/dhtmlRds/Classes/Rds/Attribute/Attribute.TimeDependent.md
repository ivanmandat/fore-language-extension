# Attribute.TimeDependent


## Синтаксис


TimeDependent: Boolean;


## Описание


Свойство TimeDependent определяет, будет ли в значении атрибута типа «ДатаВремя» отображаться время.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTimeDependent, а возвращается - с помощью метода getTimeDependent.


По умолчанию свойство содержит значение false.
