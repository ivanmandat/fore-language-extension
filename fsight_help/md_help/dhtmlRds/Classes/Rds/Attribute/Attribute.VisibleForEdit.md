# Attribute.VisibleForEdit


## Синтаксис


VisibleForEdit: Number;


## Описание


Свойство VisibleForEdit определяет,
 когда атрибут доступен для редактирования в карточке элемента справочника
 НСИ.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setVisibleForEdit,
 а возвращается - с помощью метода getVisibleForEdit.


Свойство может принимать следующие значения:


		 Значение


		 Краткое
		 описание


		 0


		 Атрибут никогда недоступен для редактирования.


		 1


		 Атрибут всегда доступен для редактирования.


		 2


		 Атрибут доступен для редактирования, если не скрытый.
