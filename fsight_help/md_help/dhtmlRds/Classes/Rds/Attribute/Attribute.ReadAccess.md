# Attribute.ReadAccess


## Синтаксис


ReadAccess: String;


## Описание


Свойство ReadAccess определяет права на просмотр атрибута справочника НСИ.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setReadAccess, а возвращается - с помощью метода getReadAccess.


Свойство содержит порядковый номер субъекта доступа, которому требуется дать права на просмотр атрибута справочника НСИ.
