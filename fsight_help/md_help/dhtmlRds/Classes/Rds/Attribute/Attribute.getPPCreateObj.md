# Attribute.getPPCreateObj


## Синтаксис


getPPCreateObj();


## Описание


Метод getPPCreateObj возвращает информацию об элементе управления, соответствующем атрибуту справочника НСИ.


## Комментарии


Метод возвращает JSON-объект, набор полей которого зависит от типа элемента управления.


## Пример


Для выполнения примера необходимо сначала выполнить пример на странице описания [конструктора Attribute](Constructor_Attribute.htm). Получим информацию об элементе управления, соответствующем атрибуту «Дата последнего изменения»:


var obj = attribute.getPPCreateObj();
console.log("Тип элемента управления: %s", obj.PPType);
console.log("Элемент управления %sможет быть пустым", obj.IsNullable ? "" : "не ");
console.log("Элемент управления %sотображается", obj.IsVisible ? "" : "не ");

В результате выполнения примера в консоли браузера был выведен тип элемента управления, соответствующего атрибуту, а также признаки, может ли он иметь пустое значение и отображаться:


Тип элемента управления: PP.Ui.DateTimePicker


Элемент управления не может быть пустым


Элемент управления отображается
