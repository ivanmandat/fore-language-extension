# Attribute.TranslatedName


## Синтаксис


TranslatedName: String;


## Описание


Свойство TranslatedName определяет текст перевода для наименования атрибута.


## Комментарии


Значение свойства устанавливается с помощью метода setTranslatedName, а возвращается - с помощью метода getTranslatedName. Из JSON значение задать нельзя.


Метод setTranslatedName имеет два атрибута: lcid - код языка, attr - текст перевода.


При вызове метода getTranslatedName используется атрибут lcid.
