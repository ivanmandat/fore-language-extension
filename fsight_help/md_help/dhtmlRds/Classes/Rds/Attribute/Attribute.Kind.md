# Attribute.Kind


## Синтаксис


Kind: [PP.Rds.AttributeKind](../../../Enums/AttributeKind.htm);


## Описание


Свойство Kind определяет вид атрибута справочника НСИ.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsKind, а возвращается - с помощью метода getIsKind.


По умолчанию свойство содержит значение PP.Rds.AttributeKind.Common.
