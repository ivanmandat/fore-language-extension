# Attribute.Tag


## Синтаксис


Tag: String;


## Описание


Свойство Tag определяет пользовательскую информацию об атрибуте справочника НСИ.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTag, а возвращается - с помощью метода getTag.


Свойство не учитывается компилятором, поэтому может быть использовано пользователем по его усмотрению.
