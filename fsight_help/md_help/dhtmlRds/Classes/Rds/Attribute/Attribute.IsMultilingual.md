# Attribute.IsMultilingual


## Синтаксис


IsMultilingual: Boolean;


## Описание


Свойство IsMultilingual определяет,
 является ли атрибут мультиязычным.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIsMultilingual,
 а возвращается - с помощью метода getIsMultilingual.


По умолчанию свойство содержит значение false.
