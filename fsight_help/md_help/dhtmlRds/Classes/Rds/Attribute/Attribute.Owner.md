# Attribute.Owner


## Синтаксис


Owner: [PP.Rds.Dictionary](../Dictionary/Dictionary.htm);


## Описание


Свойство Owner определяет справочник НСИ, которому принадлежит атрибут.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setOwner, а возвращается - с помощью метода getOwner.
