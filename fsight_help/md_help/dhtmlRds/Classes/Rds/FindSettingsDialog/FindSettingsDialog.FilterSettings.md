# FindSettingsDialog.FilterSettings


## Синтаксис


FilterSettings: Object;


## Описание


Свойство FilterSettings определяет настройки окна поиска.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setFilterSettings, а возвращается с помощью метода getFilterSettings.**
