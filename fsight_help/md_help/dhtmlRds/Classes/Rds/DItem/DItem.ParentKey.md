# DItem.ParentKey


## Синтаксис


ParentKey: String;


## Описание


Свойство ParentKey определяет ключ родительского элемента справочника НСИ.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setParentKey, а возвращается - с помощью метода getParentKey.
