# DItem.parseFromMd


## Синтаксис


parseFromMd(md: Object, init: Boolean);


## Параметры


md. Настройки элемента справочника в формате метаданных;


init. Признак того, что элемент будет пересоздан. Если параметр равен значению true, то заново будут установлены значения атрибутов и история их значений, иначе - не будут.


## Описание


Метод parseFromMd считывает настройки элемента справочника НСИ из указанного объекта в формате метаданных.
