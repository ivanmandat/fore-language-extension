# DItem.LinkedData


## Синтаксис


LinkedData: Array;


## Описание


Свойство LinkedData связанные с элементом справочника НСИ данные.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLinkedData, а возвращается - с помощью метода getLinkedData.


Свойство содержит массив объектов в формате метаданных со связанными с элементом справочника НСИ данными.
