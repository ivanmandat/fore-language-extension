# DItem.Key


## Синтаксис


Key: String;


## Описание


Свойство Key определяет ключ элемента справочника НСИ.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setKey, а возвращается - с помощью метода getKey.
