# DItem.Level


## Синтаксис


Level: Number;


## Описание


Свойство Level определяет уровень элемента в справочнике НСИ.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLevel, а возвращается - с помощью метода getLevel.


По умолчанию свойство содержит значение 0.
