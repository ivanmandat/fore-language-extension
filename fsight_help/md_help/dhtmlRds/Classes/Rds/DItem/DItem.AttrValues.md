# DItem.AttrValues


## Синтаксис


AttrValues: Array;


## Описание


Свойство AttrValues определяет значения атрибутов для элемента справочника НСИ.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setAttrValues, а возвращается - с помощью метода getAttrValues.
