# DItem.HasChild


## Синтаксис


HasChild: Boolean;


## Описание


Свойство HasChild определяет, имеет ли элемент справочника НСИ потомков.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHasChild, а возвращается - с помощью метода getHasChild.


По умолчанию свойство содержит значение false.
