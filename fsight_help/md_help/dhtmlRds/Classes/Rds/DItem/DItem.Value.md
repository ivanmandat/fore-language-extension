# DItem.Value


## Синтаксис


Value: String;


## Описание


Свойство Value определяет значение атрибута элемента справочника НСИ по его идентификатору.


## Комментарии


Значение свойства устанавливается с помощью метода setValue, а возвращается - с помощью метода getValue. Из JSON значение задать нельзя.


Метод setValue имеет два параметра: attrId - идентификатор атрибута, value - его значение.


При вызове метода setValue указываются следующие параметры: attrId - идентификатор атрибута, linkedVal - признак того, что будет возвращена ссылка на значение.
