# DItem.Name


## Синтаксис


Name: String;


## Описание


Свойство Name определяет наименование элемента в справочнике НСИ.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setName, а возвращается - с помощью метода getName.


По умолчанию свойство содержит пустую строку.
