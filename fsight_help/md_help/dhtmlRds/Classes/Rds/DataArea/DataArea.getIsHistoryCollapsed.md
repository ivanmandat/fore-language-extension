# DataArea.getIsHistoryCollapsed


## Синтаксис


getIsHistoryCollapsed(key: String);


## Параметры


key. Ключ справочника, вкладка которого будет проверяться.


## Описание


Метод getIsHistoryCollapsed проверяет, свёрнута ли панель с историей значений атрибутов.


## Комментарии


Метод возвращает значение true, если панель с историей значений атрибутов свёрнута, иначе - значение false.
