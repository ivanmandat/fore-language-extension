# DataArea.expandHistory


## Синтаксис


expandHistory(key: String);


## Параметры


key. Ключ справочника, для которого требуется развернуть панель с историей значений атрибутов. Если параметр не указан, то подразумевается справочник на активной вкладке.


## Описание


Метод expandHistory разворачивает панель с историей значений атрибутов.
