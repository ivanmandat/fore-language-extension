# DataArea.collapseHistory


## Синтаксис


collapseHistory(key: String);


## Параметры


key. Ключ справочника, для которого требуется свернуть панель с историей значений атрибутов. Если параметр не указан, то подразумевается справочник на активной вкладке.


## Описание


Метод collapseHistory сворачивает панель с историей значений атрибутов.
