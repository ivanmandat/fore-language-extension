# DataArea.ActiveTabChanged


## Синтаксис


ActiveTabChanged: function(sender, args: Object);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие ActiveTabChanged генерируется при смене активной вкладки в области данных контейнера со справочника НСИ.
