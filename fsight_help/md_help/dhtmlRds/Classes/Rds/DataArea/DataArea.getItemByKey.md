# DataArea.getItemByKey


## Синтаксис


getItemByKey(dictKey: String);


## Параметры


dictKey. Ключ справочника НСИ.


## Описание


Метод getItemByKey возвращает содержимое вкладки с указанным справочником НСИ.


## Комментарии


Метод возвращает JSON-объект со следующими полями: DictTree - дерево элементов справочника НСИ, FindBox - контейнер для их поиска, HistBox -дерево c историей значений атрибутов справочника, ParamBox - контейнер для параметров справочника, TabItem - вкладка со справочником.
