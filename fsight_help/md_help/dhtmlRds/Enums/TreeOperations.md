# TreeOperations


Пространство имен: PP.Rds.Ui


## Описание


Перечисление TreeOperations содержит типы операций, выполняемые над деревом справочника НСИ.


Используется следующими методами:


-
[DictionaryTree.doPredefinedMove](../Classes/Rds/DictionaryTree/DictionaryTree.doPredefinedMove.htm);


-
[DictionaryTree.pushToClipboard](../Classes/Rds/DictionaryTree/DictionaryTree.moveItem.htm).


## Допустимые значения


Значение


Краткое описание


AddElement


Добавление элемента.


EditElement


Редактирование элемента.


DeleteElements


Удаление элементов.


CopyElements


Копирование элементов.


CutElements


Вырезание элементов.


PasteElements


Вставка элементов из буфера обмена.


MoveElementsUp


Перемещение элементов на одну позицию вверх.


MoveElementsDown


Перемещение элементов на одну позицию вниз.


MoveElementsLevelUp


Перемещение элементов на уровень иерархии вверх.
