# ElementOperation


Пространство имен:
 PP.Rds


## Описание


Перечисление ElementOperation содержит
 операции, выполняемые над элементами справочника НСИ.


Используется свойством [ElementCard.Mode](../Classes/Rds/ElementCard/ElementCard.Mode.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 Unknown


		 Неизвестно.


		 Insert


		 Добавление элемента.


		 Update


		 Обновление элемента.


		 Delete


		 Удаление элемента.
