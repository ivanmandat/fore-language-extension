# PredefinedAttribute


Пространство имен: PP.Rds


## Описание


Перечисление PredefinedAttribute содержит идентификаторы предопределённых атрибутов.


Используется свойством [Attribute.Predefined](../Classes/Rds/Attribute/Attribute.Predefined.htm).
