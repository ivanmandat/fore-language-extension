# AttributeKind


Пространство имен: PP.Rds


## Описание


Перечисление AttributeKind содержит
 виды атрибутов справочника НСИ.


Используется свойством [Attribute.Kind](../Classes/Rds/Attribute/Attribute.Kind.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 Common


		 Простой атрибут.


		 Predefined


		 Предопределённый атрибут.


		 Linked


		 Связанный атрибут.


		 Imported


		 Заимствованный атрибут.


		 Calculated


		 Вычисляемый атрибут.
