# ITsModelParamValues.FindByKey


## Синтаксис


FindByKey(Key: Integer): [ITsModelParamValue](../ITsModelParamValue/ITsModelParamValue.htm);


## Параметры


Key. Ключ параметра.


## Описание


Метод FindByKey выполняет поиск
 значения параметра по ключу параметра.


## Комментарии


Если значение для параметра с указанным ключом не найдено, то метод
 возвращает значение Null.
