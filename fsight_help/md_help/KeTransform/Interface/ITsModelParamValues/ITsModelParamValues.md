# ITsModelParamValues


Сборка: Transform;


## Описание


Интерфейс ITsModelParamValues
 предназначен для работы с коллекцией значений параметров.


## Иерархия наследования


           ITsModelParamValues


## Комментарии


Для работы с коллекцией параметров используйте интерфейс [ITsModelParams](../ITsModelParams/ITsModelParams.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Item](ITsModelParamValues.Item.htm)


		 Свойство Item возвращает
		 указанное значение параметра из коллекции.


		 ![](../../Property_Image.gif)
		 [Count](ITsModelParamValues.Count.htm)


		 Свойство Count возвращает
		 количество значений параметров в коллекции.

		 Имя метода
