# ITsModelParamValues.FindById


## Синтаксис


FindById(Id: String): [ITsModelParamValue](../ITsModelParamValue/ITsModelParamValue.htm);


## Параметры


Id. Идентификатор параметра.


## Описание


Метод FindById выполняет поиск
 значения параметра по идентификатору параметра.


## Комментарии


Если значение для параметра с указанным идентификатором не найдено,
 то метод возвращает значение Null.
