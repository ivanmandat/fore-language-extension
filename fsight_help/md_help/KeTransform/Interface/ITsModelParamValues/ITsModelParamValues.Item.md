# ITsModelParamValues.Item


## Синтаксис


Item(Index: Integer): [ITsModelParam](../ITsModelParam/ITsModelParam.htm);


## Параметры


Index. Индекс значения параметра
 в коллекции.


## Описание


Свойство Item возвращает
 указанное значение параметра из коллекции.


## Комментарии


Индексация значений параметров в коллекции сквозная и начинается с нуля.
 Для получения количества значений параметров в коллекции используйте свойство
 [ITsModelParamValues.Count](ITsModelParamValues.Count.htm).
