# ITsModelParamValues.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает
 количество значений параметров в коллекции.


## Комментарии


Для получения определенного значения параметра из коллекции используйте
 свойство [ITsModelParamValues.Item](ITsModelParamValues.Item.htm).
