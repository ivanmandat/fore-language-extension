# ITsCalculationContext.RaiseMessage


## Синтаксис


RaiseMessage(Message: String);


## Параметры


Message. Текст сообщения.


## Описание


Метод RaiseMessage генерирует
 сообщение пользовательского метода.


## Комментарии


После генерации сообщения выполнение метода будет продолжено.
