# ITsCalculationContext.CreateTimeSeries


## Синтаксис


CreateTimeSeries([CalendarLevel: [DimCalendarLevel](KeDims.chm::/Enums/DimCalendarLevel.htm)
 = -1]): Variant;


## Параметры


CalendarLevel. Календарная
 динамика создаваемого ряда;


## Описание


Метод CreateTimeSeries создает
 ряд для расчета пользовательского метода.


## Комментарии


Параметр CalendarLevel необязательный.
 По умолчанию календарная динамика создаваемого ряда, совпадает с календарной
 динамикой родительского объекта.
