# ITsCalculationContext.RaiseError


## Синтаксис


RaiseError(Message: String);


## Параметры


Message. Текст ошибки.


## Описание


Метод RaiseError генерирует
 ошибку пользовательского метода.


## Комментарии


После генерации ошибки выполнение метода прекращается.
