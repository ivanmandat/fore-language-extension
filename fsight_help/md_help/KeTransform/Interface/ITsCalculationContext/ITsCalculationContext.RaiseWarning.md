# ITsCalculationContext.RaiseWarning


## Синтаксис


RaiseWarning(Message: String);


## Параметры


Message. Текст предупреждения.


## Описание


Метод RaiseWarning генерирует
 предупреждение пользовательского метода.


## Комментарии


После генерации предупреждения выполнение метода будет продолжено.
