# ITsCalculationContext.ParentObject


## Синтаксис


ParentObject: [IMetabaseObjectDescriptor](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm);


## Описание


Свойство ParentObject возвращает
 родительский объект контекста пользовательского метода расчета.


## Комментарии


Пользовательские методы расчета применяются в контейнере моделирования
 и базе данных временных рядов.
