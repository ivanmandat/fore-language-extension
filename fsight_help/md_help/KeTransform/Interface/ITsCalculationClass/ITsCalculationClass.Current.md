# ITsCalculationClass.Current


## Синтаксис


Current: [ITsCalculationContext](../ITsCalculationContext/ITsCalculationContext.htm);


## Описание


Свойство Current возвращает
 текущий контекст пользовательского метода расчета.


## Комментарии


Используя контекст можно выводить пользователю сообщения, предупреждения
 и ошибки, возникшие при расчете пользовательского метода.
