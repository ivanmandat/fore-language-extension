# ITsUserData


Сборка: Transform;


## Описание


Интерфейс ITsUserData предназначен
 для хранения любой пользовательской информации.


## Иерархия наследования


ITsUserData


## Комментарии


Пользовательскую информацию возвращают следующие свойства:


	- [ITsCalculationContext.UserData](../ITsCalculationContext/ITsCalculationContext.UserData.htm);


	- [IAlgorithmCalculationDebug.UserData](KeAlgo.chm::/Interface/IAlgorithmCalculationDebug/IAlgorithmCalculationDebug.UserData.htm);


	- [IMsProblemCalculation.UserData](KeMs.chm::/Interface/IMsProblemCalculation/IMsProblemCalculation.UserData.htm).


Каждый элемент с пользовательской информацией имеет уникальный идентификатор.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Data](ITsUserData.Data.htm)
		 Свойство Data определяет
		 любую пользовательскую информацию и сохраняет её в элементе с
		 указанным идентификатором.
