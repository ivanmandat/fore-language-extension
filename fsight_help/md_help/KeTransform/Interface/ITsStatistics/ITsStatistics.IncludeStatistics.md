# ITsStatistics.IncludeStatistics


## Синтаксис


IncludeStatistics(Statistics: [TsStatisticsType](../../Enums/TsStatisticsType.htm)):
 Booelan;


## Параметры


Statistics. Статистическая
 характеристика.


## Описание


Свойство IncludeStatistics определяет,
 рассчитывать ли указанную статистическую характеристику.


## Комментарии


Допустимые значения:


	- True. Рассчитывать статистическую
	 характеристику;


	- False. Не рассчитывать
	 статистическую характеристику.
