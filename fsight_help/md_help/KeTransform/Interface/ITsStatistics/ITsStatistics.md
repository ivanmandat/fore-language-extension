# ITsStatistics


Сборка: Transform;


## Описание


Интерфейс ITsStatistics применяется
 для построения дерева статистических характеристик.


## Иерархия наследования


           ITsStatistics


## Комментарии


Для работы с деревом статистических характеристик предназначен интерфейс
 [ITsStatisticsTree](../ITsStatisticsTree/ITsStatisticsTree.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CalculatedOnly](ITsStatistics.CalculatedOnly.htm)


		 Свойство CalculatedOnly
		 определяет, включать ли в дерево только рассчитанные статистические
		 характеристики.


		 ![](../../Property_Image.gif)
		 [IncludeStatistics](ITsStatistics.IncludeStatistics.htm)


		 Свойство IncludeStatistics
		 определяет, рассчитывать ли указанную статистическую характеристику.


		 ![](../../Property_Image.gif)
		 [SummaryStatistics](ITsStatistics.SummaryStatistics.htm)


		 Свойство SummaryStatistics
		 определяет статистические характеристики для вычисляемых рядов
		 и методов моделирования.


		 ![](../../Property_Image.gif)
		 [Values](ITsStatistics.Values.htm)


		 Свойство Value
		 определяет массив данных, для которого строится дерево статистических
		 характеристик.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Execute](ITsStatistics.Execute.htm)


		 Метод Execute
		 выполняет построение дерева статистических характеристик.
