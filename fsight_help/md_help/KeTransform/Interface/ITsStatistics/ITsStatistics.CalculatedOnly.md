# ITsStatistics.CalculatedOnly


## Синтаксис


CalculatedOnly: Boolean;


## Описание


Свойство CalculatedOnly определяет,
 включать ли в дерево только рассчитанные статистические характеристики.


## Комментарии


Допустимые значения:


	- True. В дерево включаются
	 только те характеристики, значения для которых были рассчитаны;


	- False. В дерево включаются
	 все возможны характеристики.
