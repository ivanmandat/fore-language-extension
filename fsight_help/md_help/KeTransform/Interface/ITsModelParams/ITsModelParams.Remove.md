# ITsModelParams.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемого параметра.


## Описание


Метод Remove удаляет из коллекции
 указанный параметр.


## Комментарии


Индексация параметров в коллекции сквозная и начинается с нуля. Для
 получения количества параметров в коллекции используйте свойство [ITsModelParams.Count](ITsModelParams.Count.htm).


Если удаление выполнено успешно, то метод возвращает значение True, в обратном случае - False.
