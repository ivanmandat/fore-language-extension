# ITsModelParams.FindById


## Синтаксис


FindById(Id: String): [ITsModelParam](../ITsModelParam/ITsModelParam.htm);


## Параметры


Id. Идентификатор параметра.


## Описание


Метод FindById выполняет поиск
 в коллекции параметра по его идентификатору.


## Комментарии


Если параметр с указанным идентификатором не найден, то метод возвращает
 значение Null.
