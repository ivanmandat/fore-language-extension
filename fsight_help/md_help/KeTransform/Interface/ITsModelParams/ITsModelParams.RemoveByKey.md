# ITsModelParams.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ удаляемого параметра.


## Описание


Метод RemoveByKey удаляет
 из коллекции параметр с указанным ключом.


## Комментарии


Если удаление выполнено успешно, то метод возвращает значение True, в обратном случае - False.
