# ITsModelParams.Item


## Синтаксис


Item(Index: Integer): [ITsModelParam](../ITsModelParam/ITsModelParam.htm);


## Параметры


Index. Индекс параметра в коллекции.


## Описание


Свойство Item возвращает
 указанный параметр из коллекции.


## Комментарии


Индексация параметров в коллекции сквозная и начинается с нуля. Для
 получения количества параметров в коллекции используйте свойство [ITsModelParams.Count](ITsModelParams.Count.htm).
