# ITsModelParams


Сборка: Transform;


## Описание


Интерфейс ITsModelParams предназначен
 для работы с коллекцией параметров.


## Иерархия наследования


           ITsModelParams


## Комментарии


Для работы с коллекцией значений параметров используйте интерфейс [ITsModelParamValues](../ITsModelParamValues/ITsModelParamValues.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITsModelParams.Count.htm)


		 Свойство Count возвращает
		 количество параметров в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ITsModelParams.Item.htm)


		 Свойство Item возвращает
		 указанный параметр из коллекции.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ITsModelParams.Add.htm)


		 Метод Add добавляет
		 параметр в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](ITsModelParams.Clear.htm)


		 Метод Clear очищает
		 коллекцию параметров.


		 ![](../../Sub_Image.gif)
		 [FindById](ITsModelParams.FindById.htm)


		 Метод FindById выполняет
		 поиск в коллекции параметра по его идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ITsModelParams.FindByKey.htm)


		 Метод FindByKey выполняет
		 поиск в коллекции параметра по его ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](ITsModelParams.Remove.htm)


		 Метод Remove удаляет
		 из коллекции указанный параметр.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](ITsModelParams.RemoveByKey.htm)


		 Метод RemoveByKey удаляет
		 из коллекции параметр с указанным ключом.
