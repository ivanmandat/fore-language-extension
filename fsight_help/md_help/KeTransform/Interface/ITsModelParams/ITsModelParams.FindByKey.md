# ITsModelParams.FindByKey


## Синтаксис


FindByKey(Key: Integer): [ITsModelParam](../ITsModelParam/ITsModelParam.htm);


## Параметры


Key. Ключ параметра.


## Описание


Метод FindByKey выполняет поиск
 в коллекции параметра по его ключу.


## Комментарии


Если параметр с указанным ключом не найден, то метод возвращает значение
 Null.
