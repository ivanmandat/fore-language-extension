# ITsStatisticsTree.Type


## Синтаксис


Name(Index: Integer): [TsStatisticsTreeNodeType](../../Enums/TsStatisticsTreeNodeType.htm);


## Параметры


Index. Индекс элемента.


## Описание


Свойство Type возвращает
 тип указанного элемента дерева.


## Комментарии


Индексация элементов сквозная и начинается с нуля. Количество элементов
 в дереве возвращает свойство [ITsStatisticsTree.Count](ITsStatisticsTree.Count.htm).
