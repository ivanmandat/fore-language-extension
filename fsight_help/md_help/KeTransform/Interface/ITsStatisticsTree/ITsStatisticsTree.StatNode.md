# ITsStatisticsTree.StatNode


## Синтаксис


StatNode(Index: [TsStatisticsType](../../Enums/TsStatisticsType.htm)):
 Integer;


## Параметры


Index. Статистическая характеристика.


## Описание


Свойство StatNode возвращает
 индекс указанной статистической характеристики.


## Комментарии


Для получения наименования статистической характеристики используйте
 свойство [ITsStatisticsTree.Name](ITsStatisticsTree.Name.htm).
