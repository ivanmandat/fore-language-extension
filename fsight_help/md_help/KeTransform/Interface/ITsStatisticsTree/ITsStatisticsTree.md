# ITsStatisticsTree


Сборка: Transform;


## Описание


Интерфейс ITsStatisticsTree
 предназначен для работы с деревом статистических характеристик.


## Иерархия наследования


           ITsStatisticsTree


## Комментарии


Для построения дерева используйте интерфейс [ITsStatistics](../ITsStatistics/ITsStatistics.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITsStatisticsTree.Count.htm)


		 Свойство Count возвращает
		 количество элементов в дереве статистических характеристик.


		 ![](../../Property_Image.gif)
		 [Data](ITsStatisticsTree.Data.htm)


		 Свойство Data возвращает
		 элемент перечисления, соответствующий указанному элементу дерева.


		 ![](../../Property_Image.gif)
		 [Name](ITsStatisticsTree.Name.htm)


		 Свойство Name возвращает
		 наименование указанного элемента дерева.


		 ![](../../Property_Image.gif)
		 [ParentNode](ITsStatisticsTree.ParentNode.htm)


		 Свойство ParentNode
		 возвращает индекс родительского элемента для указанного элемента
		 дерева.


		 ![](../../Property_Image.gif)
		 [ShortName](ITsStatisticsTree.ShortName.htm)


		 Свойство ShortName
		 возвращает короткое наименование указанного элемента дерева.


		 ![](../../Property_Image.gif)
		 [StatNode](ITsStatisticsTree.StatNode.htm)


		 Свойство StatNode
		 возвращает индекс указанной статистической характеристики.


		 ![](../../Property_Image.gif)
		 [Type](ITsStatisticsTree.Type.htm)


		 Свойство Type возвращает
		 тип указанного элемента дерева.


		 ![](../../Property_Image.gif)
		 [Value](ITsStatisticsTree.Value.htm)


		 Свойство Value возвращает
		 значение указанного элемента дерева.
