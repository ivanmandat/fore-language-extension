# ITsModelParamValue.Hidden


## Синтаксис


Hidden: Boolean;


## Описание


Свойство Hidden возвращает признак
 того, является ли параметр скрытым.


## Комментарии


Допустимые значения:


	- True. Параметр скрытый;


	- False. Параметр не является
	 скрытым.


Если у параметра отсутствует значение по умолчанию, то перед началом
 расчёта задайте значения параметра. Используйте свойство [ITsModelParamValue.Value](ITsModelParamValue.Value.htm).
