# ITsModelParamValue.Value


## Синтаксис


Value: Variant;


## Описание


Свойство Value определяет
 значение параметра.


## Комментарии


Если у параметра отсутствует значение по умолчанию, то свойство Value обязательно должно быть задано
 перед началом расчёта. Для получения значения параметра по умолчанию используйте
 свойство [ITsModelParamValue.DefaultValue](ITsModelParamValue.DefaultValue.htm).
