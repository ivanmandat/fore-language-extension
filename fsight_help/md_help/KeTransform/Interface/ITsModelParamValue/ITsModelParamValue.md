# ITsModelParamValue


Сборка: Transform;


## Описание


Интерфейс ITsModelParamValue
 предназначен для работы со значением параметра.


## Иерархия наследования

           ITsModelParamValue


## Комментарии


Для работы с коллекцией значений параметров используйте интерфейс [ITsModelParamValues](../ITsModelParamValues/ITsModelParamValues.htm).
