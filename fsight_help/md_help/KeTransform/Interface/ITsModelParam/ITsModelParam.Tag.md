# ITsModelParam.Tag


## Синтаксис


Tag: Variant;


## Описание


Свойство Tag определяет
 объект, содержащий дополнительные данные о параметре.


## Комментарии


Свойство не учитывается компилятором и может использоваться пользователем
 по его усмотрению. Например, использовать для хранения данных, которые
 тесно связаны с параметром.
