# ITsModelParam.DefaultValue


## Синтаксис


DefaultValue: Variant;


## Описание


Свойство DefaultValue определяет
 значение параметра по умолчанию.


## Комментарии


По умолчанию значение не задано. Для работы со значением параметра используйте
 интерфейс [ITsModelParamValue](../ITsModelParamValue/ITsModelParamValue.htm).
