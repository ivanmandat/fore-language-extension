# ITsModelParam.Hidden


## Синтаксис


Hidden: Boolean;


## Описание


Свойство Hidden определяет,
 является ли параметр скрытым.


## Комментарии


Допустимые значения:


	- True. Параметр скрытый;


	- False. Параметр не является
	 скрытым.


Если у параметра отсутствует значение по умолчанию, то перед началом
 расчёта задайте значения параметра. Используйте интерфейс [ITsModelParamValue](../ITsModelParamValue/ITsModelParamValue.htm).
