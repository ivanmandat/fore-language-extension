# ITsModelParam


Сборка: Transform;


## Описание


Интерфейс ITsModelParam предназначен
 для работы с параметром.


## Иерархия наследования


           [INamedEntity](KeSom.chm::/Interface/INamedEntity/INamedEntity.htm)


           ITsModelParam


## Комментарии


Для работы с коллекцией параметров используйте интерфейс [ITsModelParams](../ITsModelParams/ITsModelParams.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DataType](ITsModelParam.DataType.htm)


		 Свойство DataType
		 определяет тип данных параметра.


		 ![](../../Property_Image.gif)
		 [DefaultValue](ITsModelParam.DefaultValue.htm)


		 Свойство DefaultValue
		 определяет значение параметра по умолчанию.


		 ![](../../Property_Image.gif)
		 [Hidden](ITsModelParam.Hidden.htm)


		 Свойство Hidden определяет,
		 является ли параметр скрытым.


		 ![](../../Property_Image.gif)
		 [LinkedObject](ITsModelParam.LinkedObject.htm)


		 Свойство LinkedObject
		 определяет справочник, используемый для определения значения параметра.


		 ![](../../Property_Image.gif)
		 [LinkedObjectParams](ITsModelParam.LinkedObjectParams.htm)


		 Свойство LinkedObjectParams
		 возвращает коллекцию значений параметров, используемую для открытия
		 связанного справочника.


		 ![](../../Property_Image.gif)
		 [ParamType](ITsModelParam.ParamType.htm)


		 Свойство ParamType
		 определяет тип параметра.


		 ![](../../Property_Image.gif)
		 [SelectionAttrID](ITsModelParam.SelectionAttrID.htm)


		 Свойство SelectionAttrID
		 определяет идентификатор атрибута измерения, используемого для
		 восстановления отметки.


		 ![](../../Property_Image.gif)
		 [Tag](ITsModelParam.Tag.htm)


		 Свойство Tag
		 определяет объект, содержащий дополнительные данные о параметре.


		 ![](../../Property_Image.gif)
		 [Visible](ITsModelParam.Visible.htm)


		 Свойство Visible определяет,
		 будет ли параметр отображаться на панели параметров.
