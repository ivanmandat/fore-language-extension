# ITsModelParam.SelectionAttrID


## Синтаксис


SelectionAttrID: String;


## Описание


Свойство SelectionAttrID
 определяет идентификатор атрибута измерения, используемого для восстановления
 отметки.


## Комментарии


Для задания справочника, использующегося в качестве измерения атрибута,
 используйте свойство [ITsModelParam.LinkedObject](ITsModelParam.LinkedObject.htm).
