# ITsInversionInfo.Inversion


## Синтаксис


Inversion: [TsInversion](../../Enums/TsInversion.htm);


## Описание


Свойство Inversion определяет
 способ начального преобразования.


## Комментарии


От значения Inversion зависит,
 какие свойства интерфейса [ITsInversionInfo](ITsInversionInfo.htm)
 будут учитываться при расчете.
