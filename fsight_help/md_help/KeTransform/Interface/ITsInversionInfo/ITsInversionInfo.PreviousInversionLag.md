# ITsInversionInfo.PreviousInversionLag


## Синтаксис


PreviousInversionLag: Integer;


## Описание


Свойство PreviousInversionLag
 определяет лаг для периода, на котором осуществляется начальное преобразование.


## Комментарии


Значение по умолчанию «-1». Свойство может принимать только отрицательные
 значения и актуально, если свойству [ITsInversionInfo.InversionLag](ITsInversionInfo.InversionLag.htm)
 установлено значение отличное от [TsInversionLag.EndOfPrecedingYear](../../Enums/TsInversionLag.htm).
