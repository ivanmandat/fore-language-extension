# ITsInversionInfo


Сборка: Transform;


## Описание


Интерфейс ITsInversionInfo предназначен
 для настройки параметров начального преобразования, применяемого к переменной.


## Иерархия наследования


           ITsInversionInfo


## Комментарии


Начальное преобразование позволяет скорректировать значения переменной.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dependence](ITsInversionInfo.Dependence.htm)


		 Свойство Dependence
		 определяет вид тренда, используемого для детрендирования.


		 ![](../../Property_Image.gif)
		 [Inversion](ITsInversionInfo.Inversion.htm)


		 Свойство Inversion
		 определяет способ начального преобразования.


		 ![](../../Property_Image.gif)
		 [InversionLag](ITsInversionInfo.InversionLag.htm)


		 Свойство InversionLag
		 определяет период, на котором осуществляется начальное преобразование.


		 ![](../../Property_Image.gif)
		 [K](ITsInversionInfo.K.htm)


		 Свойство K определяет
		 параметр расчета по методу k-сигм.


		 ![](../../Property_Image.gif)
		 [PreviousInversionLag](ITsInversionInfo.PreviousInversionLag.htm)


		 Свойство PreviousInversionLag
		 определяет лаг для периода, на котором осуществляется начальное
		 преобразование.


		 ![](../../Property_Image.gif)
		 [Seasonality](ITsInversionInfo.Seasonality.htm)


		 Свойство Seasonality
		 определяет вид сезонности, используемой для устранения сезонности.
