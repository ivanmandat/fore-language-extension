# TsInversionLag


## Описание


Перечисление TsInversionLag используется для определения периода, на котором выполняется начальное преобразование переменной (ряда).


Используется следующим свойством:


-
[ILanerSerieDisplayParams.InversionLag](Laner.chm::/Interface/ILanerSerieDisplayParams/ILanerSerieDisplayParams.InversionLag.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 PrecidingValue. Предыдущий период.


 1
 PrecidingYear. Соответствующий период предыдущего года.


 2
 EndOfPrecedingYear. Конец предыдущего года.
