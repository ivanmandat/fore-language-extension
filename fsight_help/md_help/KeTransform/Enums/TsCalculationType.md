# TsCalculationType


## Описание


Перечисление TsCalculationType используется для определения режима расчета метода.


Используется следующим свойством:


-
[ITsCalculationContext.CalculationType](../Interface/ITsCalculationContext/ITsCalculationContext.CalculationType.htm).


## Допустимые значения


 Значение
 Краткое описание


 -1
 Undefined. Метод не определен.


 0
 Pointwise. Поточечный режим расчёта.


 1
 Series. Векторный режим расчёта.


## Комментарии


Pointwise. Используемая точка ряда будет зависеть от даты, для которой производится расчет. Например, в расчете метода для 2000 года будет использоваться значение ряда за 2000 год, в расчете метода для 2001 года - значение ряда за 2001 год и т.д.


Series. В расчёте используются все наблюдения ряда одновременно.
