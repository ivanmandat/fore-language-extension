# TsCalculationDirection


## Описание


Перечисление TsCalculationDirection используется для определения направления поточечного режима расчёта.


Используется следующим свойством:


-
[ITsCalculationContext.CalculationDirection](../Interface/ITsCalculationContext/ITsCalculationContext.CalculationDirection.htm).


## Допустимые значения


 Значение
 Краткое описание


 -1
 Undefined. Направление не определено.


 0
 Forward. Расчет вперед.


 1
 Backward. Расчет назад.
