# TsStatisticsType


## Описание


Перечисление TsStatisticsType
 предназначено для определения используемой статистической характеристики.


Используется следующими свойствами:


	- [IEaxStatistics.AllStatistics](KeExpress.chm::/Interface/IEaxStatistics/IEaxStatistics.AllStatistics.htm);


	- [ITsStatisticsTree.Data](../Interface/ITsStatisticsTree/ITsStatisticsTree.Data.htm);


	- [ITsStatisticsTree.StatNode](../Interface/ITsStatisticsTree/ITsStatisticsTree.StatNode.htm);


	- [ITsStatisticsTree.Type](../Interface/ITsStatisticsTree/ITsStatisticsTree.Type.htm).
