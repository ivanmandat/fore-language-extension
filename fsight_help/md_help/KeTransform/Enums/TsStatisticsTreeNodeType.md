# TsStatisticsTreeNodeType


## Описание


Перечисление TsStatisticsTreeNodeType используется для определения типа элементов в дереве статистических характеристик.


Используется следующими свойствами:


-
[ITsStatisticsTree.Data](../Interface/ITsStatisticsTree/ITsStatisticsTree.Data.htm);


-
[ITsStatisticsTree.Type](../Interface/ITsStatisticsTree/ITsStatisticsTree.Type.htm).


## Допустимые значения


 Значение
 Краткое описание


 1
 Statistics. Статистическая характеристика.


 2
 Category. Категория статистических характеристик.


 3
 Custom. Пользовательское значение.


## Комментарии


Для определения статистической характеристики используйте перечисление [TsStatisticsType](TsStatisticsType.htm), для определения категории характеристики - перечисление [TsStatisticsCategory](TsStatisticsCategory.htm).
