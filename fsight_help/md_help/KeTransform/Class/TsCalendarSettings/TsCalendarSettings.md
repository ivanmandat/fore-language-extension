# TsCalendarSettings


Сборка: Transform;


## Описание


Класс TsCalendarSettings предназначен
 для работы с параметрами календарного уровня.


## Комментарии


Параметры календарного уровня для контекста пользовательского метода
 расчёта возвращает свойство [ITsCalculationContext.CalendarSettings](../../Interface/ITsCalculationContext/ITsCalculationContext.CalendarSettings.htm).
