# TsStatistics


Сборка: Transform;


## Описание


Класс TsStatistics применяется
 для построения дерева статистических характеристик.


## Комментарии


Для работы с деревом статистических характеристик предназначен интерфейс
 [ITsStatisticsTree](../../Interface/ITsStatisticsTree/ITsStatisticsTree.htm).


## Свойства объекта класса, унаследованные от [ITsStatistics](../../Interface/ITsStatistics/ITsStatistics.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CalculatedOnly](../../Interface/ITsStatistics/ITsStatistics.CalculatedOnly.htm)


		 Свойство CalculatedOnly
		 определяет, включать ли в дерево только рассчитанные статистические
		 характеристики.


		 ![](../../Property_Image.gif)
		 [IncludeStatistics](../../Interface/ITsStatistics/ITsStatistics.IncludeStatistics.htm)


		 Свойство IncludeStatistics
		 определяет, рассчитывать ли указанную статистическую характеристику.


		 ![](../../Property_Image.gif)
		 [SummaryStatistics](../../Interface/ITsStatistics/ITsStatistics.SummaryStatistics.htm)


		 Свойство SummaryStatistics
		 определяет статистические характеристики для вычисляемых рядов
		 и методов моделирования.


		 ![](../../Property_Image.gif)
		 [Values](../../Interface/ITsStatistics/ITsStatistics.Values.htm)


		 Свойство Value
		 определяет массив данных, для которого строится дерево статистических
		 характеристик.
