# Взаимодействие с расширениями


В данном разделе представлено описание различных функций, которые предназначены
 для работы с дополнительными расширениями «Форсайт. Аналитическая платформа».
 Функции реализованы на языке Fore, но сформированы таким образом, чтобы
 их можно было вызывать посредством веб-сервиса «Форсайт. Аналитическая платформа».
 Данные функции имеют ту же функциональность, которую предоставляет [API расширений](Assembly.chm::/API_Extentions.htm).


## Вызов функций


Для вызова функций и получения результата их выполнения в веб-сервисе
 используется операция [ForeExec](../Operations/Repository/ForeExec.htm).
 В зависимости от расширения, с которым осуществляется взаимодействие,
 операция должна быть адресована различным модулям.
 Более подробное описание модулей, их составу и назначению, представлено
 в следующих подразделах:


	- [Работа
	 с формами ввода](DataEntryForms/DataEntryFormsServices.htm).
