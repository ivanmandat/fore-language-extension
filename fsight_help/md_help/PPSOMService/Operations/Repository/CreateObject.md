# CreateObject


## Синтаксис


CreateObjectResult CreateObject(OdId tParent, CreateObjectArg
 tArg)


## Параметры


tParent. Моникёр родительского
 объекта, в рамках которого осуществляется выполнение операции.


tArg. Параметры выполнения
 операции.


## Описание


Операция CreateObject создает
 новый объект репозитория.


## Комментарии


В качестве значения параметра tParent
 должен быть указан моникёр папки, либо одного из доступных объектов-контейнеров
 репозитория. К объектам-контейнерам относятся следующие объекты: Репозиторий
 НСИ, БД временных рядов, Контейнер моделирования, Сборка, Контейнер.


В параметре tArg указывается
 информация о создаваемом объекте. Также в нем могут быть указаны параметры
 обновления списка объектов репозитория, которые будут применены после
 выполнения операции.


Результатом работы операции является моникёр и описание созданного объекта,
 а также обновленный список объектов репозитория, если указаны соответствующие
 параметры выполнения операции.
