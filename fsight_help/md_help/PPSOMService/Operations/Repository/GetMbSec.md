# GetMbSec


## Синтаксис


GetMbSecResult GetMbSec(MbId tMbSec, GetMbSecArg
 tArg)


## Параметры


tMbSec. Моникёр соединения
 с репозиторием, метаданные менеджера безопасности которого необходимо
 получить.


tArg. Параметры извлечения
 метаданных.


## Описание


Операция GetMbSec извлекает
 метаданные менеджера безопасности репозитория.


## Комментарии


Операция позволяет получить следующую информацию:


	- Информацию о группах пользователей и пользователях, которые
	 созданы в менеджере безопасности репозитория;


	- Информацию о правах доступа к объектам;


	- Информацию о профилях пользователей репозитория;


	- Настройки политики безопасности;


	- Настройки различных методов разграничения доступа.


Для выполнения операции укажите в поле tMbSec
 моникёр соединения с репозиторием и в поле tArg.pattern
 шаблон, в соответствии с которым будут извлекаться метаданные менеджера
 безопасности. Моникёр может быть получен при выполнении операции [OpenMetabase](OpenMetabase.htm).


Результатом операции будет полученная из менеджера безопасности информация.
