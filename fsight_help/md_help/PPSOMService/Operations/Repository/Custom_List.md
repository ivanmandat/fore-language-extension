# Работа с объектами пользовательских классов


В данном разделе представлено описание операций, предназначенных для
 работы с контейнером пользовательских классов и объектами пользовательских
 классов. Операции позволяют производить следующие действия:


	- открывать контейнер пользовательских классов и работать с коллекцией
	 классов;


	- работать с готовыми объектами пользовательских классов.
