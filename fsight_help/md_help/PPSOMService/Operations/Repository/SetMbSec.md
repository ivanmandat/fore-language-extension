# SetMbSec


## Синтаксис


SetMbSecResult SetMbSec(MbId tMbSec, SetMbSecArg
 tArg)


## Параметры


tMbSec. Моникёр соединения
 с репозиторием, метаданные менеджера безопасности которого необходимо
 изменить.


tArg. Параметры, в соответствии
 с которыми будут изменены метаданные.


## Описание


Операция SetMbSec изменяет метаданные
 менеджера безопасности.


## Комментарии


Операция позволяет производить следующие действия:


	- изменять списки субъектов безопасности;


	- работать с профилями пользователей;


	- настраивать политику безопасности репозитория;


	- настраивать различные методы разграничения доступа;


	- настраивать права доступа к объектам репозитория.


Перед изменением метаданных необходимо получить лицензию для работы
 с менеджером безопасности. Для этого выполните операцию [OpenMbLicense](OpenMbLicense.htm)
 с типом лицензии - Adm.


Для выполнения операции SetMbSec укажите в поле tMbSec
 моникёр соединения с репозиторием и в поле tArg.pattern
 шаблон, в соответствии с которым будут изменяться метаданные. В поле tArg.meta должны быть указаны изменённые
 метаданные.


Результатом выполнения операции будут обновлённые метаданные менеджера
 безопасности, если было определено поле tArg.metaGet.


После завершения работы с менеджером безопасности необходимо освободить
 лицензию. Для этого выполните операцию [CloseMbLicense](CloseMbLicense.htm).
