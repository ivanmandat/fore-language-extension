# GetLocales


## Синтаксис


GetLocalesResult GetLocales()


## Описание


Операция GetLocales получает
 список языков, на которые может осуществляться локализация ресурсов в
 репозитории.


## Комментарии


Операция выполняется без каких-либо параметров.


Результатом операции будет список языков и их уникальные числовые идентификаторы.
