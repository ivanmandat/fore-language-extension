# GetCustomOb


## Синтаксис


GetCustomObResult GetCustomOb(CustomObId tCustomOb,
 GetCustomObArg tArg)


## Параметры


tCustomOb. Моникёр объекта
 пользовательского класса.


tArg. Параметры выполнения
 операции.


## Описание


Операция GetCustomOb извлекает
 метаданные объекта пользовательского класса.


## Комментарии


Для выполнения операции укажите в поле tCustomOb
 моникёр объекта пользовательского класса, а в поле tArg.pattern
 шаблон, в соответствии с которым будут извлекаться метаданные. Моникёр
 может быть получен при выполнении операции [OpenCustomOb](OpenCustomOb.htm).


Результатом работы операции будут полученные метаданные.
