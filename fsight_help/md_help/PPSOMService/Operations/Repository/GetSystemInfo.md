# GetSystemInfo


## Синтаксис


GetSystemInfoResult GetSystemInfo(GetSystemInfoArg
 tArg)


## Параметры


tArg. Параметры выполнения
 операции.


## Описание


Операция GetSystemInfo извлекает
 информацию о BI-сервере.


## Комментарии


Данная операция позволяет получить следующую информацию:


	- проверить наличие соединения с каким-либо репозиторием по имеющемуся
	 моникёру;


	- получить информацию о загрузке процессора на BI-сервере;


	- получить параметры производительности и используемые ресурсы
	 BI-сервера.


Для выполнения операции укажите в поле tArg.metabase
 моникёр соединения с репозиторием, доступ к которому необходимо проверить.
 Если доступ имеется, то будет возвращена информация о версии операционной
 системы BI-сервера. При отсутствии доступа будет получен пустой ответ.
 Результат выполнения операции также зависит от наличия настройки GetSystemInfoExtended
 в [settings.xml](Setup.chm::/UiWebSetup/03_Setup_Web/BI_Server_Settings_XML.htm)/[реестре](setup.chm::/UiWebSetup/03_Setup_Web/BI_Server_Registry_Key.htm).


Для получения информации о процессоре и используемых ресурсах определите
 поля tArg.cpuInfo и tArg.processInfo
 соответственно. Результатом выполнения операции будет полученная информация.
