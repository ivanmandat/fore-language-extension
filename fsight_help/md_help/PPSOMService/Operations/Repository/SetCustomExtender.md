# SetCustomExtender


## Синтаксис


bool SetCustomExtender(CustomObId tCustomExtender,
 SetCustomExtenderArg tArg)


## Параметры


tCustomExtender. Моникёр
 открытого экземпляра контейнера пользовательских классов.


tArg. Параметры выполнения
 операции.


## Описание


Операция SetCustomExtender изменяет
 метаданные контейнера пользовательских классов.


## Комментарии


Операция позволяет создавать, изменять и удалять пользовательские классы.
 Для выполнения операции укажите в поле tCustomExtender
 моникёр открытого экземпляра контейнера пользовательских классов, в поле
 tArg.pattern шаблон, в соответствии
 с которым будут изменяться метаданные, а в поле tArg.meta
 обновлённые метаданные. Моникёр может быть получен при выполнении операции
 [OpenCustomExtender](OpenCustomExtender.htm).


Результатом выполнения операции будут обновлённые метаданные, если при
 выполнении операции определить поле tArg.metaGet.


Для сохранения изменений в контейнере пользовательских классов используйте
 операцию [SaveObject](SaveObject.htm).
