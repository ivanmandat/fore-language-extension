# SetMbUpdate


## Синтаксис


SetMbUpdateResult SetMbUpdate(MbUpdateId tMbUpdate,
 SetMbUpdateArg tArg)


## Параметры


tMbUpdate. Моникёр
 для работы с обновлением.


tArg. Параметры выполнения
 операции.


## Описание


Операция SetMbUpdate изменяет
 настройки обновления.


## Комментарии


Операция позволяет настроить дерево элементов обновления, задать различные
 параметры обновления для отдельных элементов дерева и самого обновления,
 запустить процесс применения обновления, сохранить обновление в файл/объект
 репозитория или загрузить из файла/объекта репозитория, а также экспортировать
 дерево элементов обновления в файл. Для выполнения операции укажите в
 поле tMbUpdate моникёр обновления,
 а в поле tArg параметры применения
 изменений. Моникёр может быть получен при выполнении операции [OpenMbUpdate](OpenMbUpdate.htm). В поле tArg.meta
 указываются вносимые изменения, а в поле tArg.pattern
 шаблон, в соответствии с которым будут применяться изменения. В поле tArg.meta.execute указываются параметры
 выполнения каких-либо действий с обновлением (параметры запуска, сохранения/загрузки
 и другие).


Результатом выполнения операции будет актуальная информация об обновлении,
 если было определено поле tArg.metaGet.
