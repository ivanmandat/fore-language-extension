# GetMbSecSnapshot


## Синтаксис


GetMbSecSnapshotResult GetMbSecSnapshot(MbSecSnapshotId
 tMbSecSnapshot, GetMbSecSnapshotArg tArg)


## Параметры


tMbSecSnapshot. Моникёр
 объекта, предназначенного для работы с резервными копиями политики безопасности.


tArg. Параметры выполнения
 операции.


## Описание


Операция GetMbSecSnapshot получает
 информацию о состоянии объекта, предназначенного для резервного копирования
 политики безопасности.


## Комментарии


Операция актуальна для использования, если предварительно с помощью
 операции [SetMbSecSnapshot](SetMbSecSnapshot.htm)
 был запущен процесс восстановления политики безопасности из бинарного
 файла в асинхронном режиме. Вызов операции GetMbSecSnapshot
 позволяет узнать на какой стадии находится процесс восстановления и какое
 именно действие выполняется в текущий момент.


Для выполнения операции укажите в поле tMbSecSnapshot
 моникёр объекта, предназначенного для резервного копирования политики
 безопасности, а в поле tArg.pattern
 шаблон, в соответствии с которым будет получена информация о состоянии
 объекта.


Результатом работы операции будет запрошенная информация.
