# Специфические операции


В данном разделе представлено описание специфических операций, которые
 нельзя полноценно отнести к определенному инструменту продукта «Форсайт. Аналитическая платформа».
 В список входят операции, которые могут применяться как к какому-либо
 определенному типу объектов, так и к различным типам объектов. Также данные
 операции используются как вспомогательные для выполнения каких-либо промежуточных
 действий.


		 Операция
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ForeExec](ForeExec.htm)
		 Операция ForeExec выполняет
		 методы, реализованные в Fore-модулях.


		 ![](../../Sub_Image.gif)
		 [ForeExecAtomic](ForeExecAtomic.htm)
		 Операция ForeExecAtomic
		 выполняет Fore-методы при работе в интеграционных сервисах.


		 ![](../../Sub_Image.gif)
		 [ForeExecWithState](ForeExecWithState.htm)
		 Операция ForeExecWithState
		 выполняет методы, реализованные в Fore-модулях, с последующим
		 сохранением объектов на сервере состояний.


		 ![](../../Sub_Image.gif)
		 [GetSearch](GetSearch.htm)
		 Операция GetSearch
		 выполняет BI-поиск среди индексированных объектов репозитория.


		 ![](../../Sub_Image.gif)
		 [BatchExec](BatchExec.htm)
		 Операция BatchExec
		 предназначена для пакетного выполнения других операций.


		 ![](../../Sub_Image.gif)
		 [SetExp](SetExp.htm)
		 Операция SetExp используется
		 для формирования термов по передаваемой информации.


		 ![](../../Sub_Image.gif)
		 [GetFormulaEditorInfo](GetFormulaEditorInfo.htm)
		 Операция GetFormulaEditorInfo
		 получения списка функций, которые могут использоваться для составления
		 выражений в различных инструментах продукта «Форсайт. Аналитическая платформа».


		 ![](../../Sub_Image.gif)
		 [GetFonts](GetFonts.htm)
		 Операция GetFonts получает
		 список системных шрифтов, установленных на BI-сервере.


		 ![](../../Sub_Image.gif)
		 [GetSharedFunctions](GetSharedFunctions.htm)
		 Операция GetSharedFunctions
		 получает список пользовательских функций, доступных для использования
		 в формулах.


		 ![](../../Sub_Image.gif)
		 [GetTabStaticFormulas](GetTabStaticFormulas.htm)
		 Операция GetTabStaticFormulas
		 получает список системных функций, доступных для использования
		 в формулах.


		 ![](../../Sub_Image.gif)
		 [GetTreeElements](GetTreeElements.htm)
		 Операция GetTreeElements
		 извлекает дерево метаданных объекта репозитория.


		 ![](../../Sub_Image.gif)
		 [IsFeatureAvailable](IsFeatureAvailable.htm)
		 Операция IsFeatureAvailable
		 проверяет возможность выполнения какой-либо функции в платформе.
