# SetRepoParams


## Синтаксис


RepoParamsResult SetRepoParams(MbId tMb, SetRepoParamsArg
 tArg)


## Параметры


tMb. Моникёр
 соединения с репозиторием.


tArg. Параметры выполнения
 операции.


## Описание


Операция SetRepoParams изменяет
 параметры репозитория.


## Комментарии


Операция позволяет изменить следующие параметры:


	- глобальные переменные;


	- пользовательские классы, события и методы;


	- объекты, используемые по умолчанию;


	- обработчики ошибок и событий;


	- настройки кеширования метаданных.


Для выполнения операции укажите в поле tMb
 моникёр соединения с репозиторием, в поле tArg.pattern
 шаблон, в соответствии с которым будут изменены различные параметры, а
 в поле tArg.meta обновлённые
 значения параметров. Моникёр может быть получен при выполнении операции
 [OpenMetabase](OpenMetabase.htm).
 Если определено поле tArg.metaGet,
 то результатом операции будут обновлённые параметры, которые будут получены
 после их изменения.
