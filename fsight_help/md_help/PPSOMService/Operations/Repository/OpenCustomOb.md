# OpenCustomOb


## Синтаксис


OpenCustomObResult OpenCustomOb(OdId tOb, OpenCustomObArg
 tArg)


## Параметры


tOb. Моникёр открываемого объекта.


tArg. Параметры открытия объекта.


## Описание


Операция OpenCustomOb открывает
 объект пользовательского класса.


## Комментарии


Для выполнения операции укажите в поле tOb
 моникёр открываемого объекта. Моникёр объекта может быть получен при выполнении
 операции [GetObjects](GetObjects.htm).
 Если во время открытия также необходимо получить какие-либо метаданные,
 то определите поле tArg.metaGet.


Результатом операции будет моникёр открытого экземпляра указанного объекта
 и полученные метаданные, если было указано поле tArg.metaGet.


Дальнейшая работа осуществляется с помощью операций [GetCustomOb](GetCustomOb.htm)
 и [SetCustomOb](SetCustomOb.htm).
 Для закрытия открытого экземпляра объекта используйте операцию [CloseCustomOb](CloseCustomOb.htm).
