# SetCustomOb


## Синтаксис


SetCustomObResult SetCustomOb(CustomObId tCustomOb,
 SetCustomObArg tArg)


## Параметры


tCustomOb. Моникёр объекта
 пользовательского класса.


tArg. Параметры выполнения
 операции.


## Описание


Операция SetCustomOb изменяет
 метаданные объекта пользовательского класса.


## Комментарии


Операция SetCustomOb используется
 только при работе с объектами пользовательского класса.


Данная операция позволяет произвести следующие изменения:


	- загрузить данные в объект;


	- осуществить проверку прав пользователя на выполнение какой-либо
	 операции пользовательского класса с занесением записи в протокол доступа.


Для выполнения операции укажите в поле tCustomOb
 моникёр объекта пользовательского класса. Моникёр может быть получен при
 выполнении операции [OpenCustomOb](OpenCustomOb.htm).
 В поле tArg.pattern укажите шаблон,
 в соответствии с которым будут изменяться метаданные, а в поле tArg.meta
 обновленные метаданные объекта.


Результатом работы операции будет моникёр измененного объекта и обновленные
 метаданные, которые могут быть запрошены в шаблоне tArg.metaGet.
