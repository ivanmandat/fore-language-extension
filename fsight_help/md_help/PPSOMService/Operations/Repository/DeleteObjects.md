# DeleteObjects


## Синтаксис


DeleteObjectsResult DeleteObjects(MbId tMb, DeleteObjectsArg
 tArg)


## Параметры


tMb. Моникёр соединения с репозиторием,
 в рамках которого производится удаление объектов.


tArg. Параметры выполнения
 операции.


## Описание


Операция DeleteObjects удаляет
 объекты репозитория.


## Комментарии


Данная операция позволяет удалить один или несколько объектов репозитория.
 Для этого в параметре tArg необходимо
 указать поле objectKey или objects соответственно. Также в
 поле getObjects могут быть указаны
 параметры обновления списка объектов репозитория, которые будут применены
 после выполнения операции. Если параметры обновления указаны, то в поле
 objects результата выполнения
 операции будет доступен обновленный список объектов.


Если удаление объекта(ов) по каким-либо причинам невозможно, то при
 выполнении операции будет сгенерирована исключительная ситуация.
