# OpenAuditLog


## Синтаксис


OpenAuditLogResult OpenAuditLog(MbId tMb, OpenAuditLogArg
 tArg)


## Параметры


tMb. Моникёр соединения с репозиторием,
 протокол доступа которого необходимо открыть.


tArg. Параметры открытия протокола
 доступа.


## Описание


Операция OpenAuditLog открывает
 протокол доступа и возвращает моникёр для доступа к записям.


## Комментарии


Первая операция OpenAuditLog
 должна быть выполнена с указанием в поле tArg.logType
 значения Logons. При этом будет
 получен моникёр, позволяющий получить список подключений к репозитория.
 Используя данный моникёр в операции [GetAuditLog](GetAuditLog.htm)
 можно получить записи о подключениях к репозиторию. Каждая запись имеет
 ключ сессии с репозиторием. Теперь указав в поле tArg.session
 ключ одной из сессий и в поле tArg.logType
 значение Operations с помощью
 операции OpenAuditLog можно получить
 моникёр протокола со списком операций, произведенных над объектами репозитория
 в рамках выбранной сессии. Информацию об операциях над объектами также
 можно получить, используя операцию [GetAuditLog](GetAuditLog.htm).
 Для выполнения операции OpenAuditLog
 также необходимо в параметре tMb
 указать моникёр соединения с репозиторием.
