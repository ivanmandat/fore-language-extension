# Интерактивные визуальные компоненты


В данном разделе представлено описание операций, предназначенных для
 работы с различными визуализаторами. Визуализаторы используются в различных
 инструментах «Форсайт. Аналитическая платформа»
 для графического представления информации. Операции позволяют получать
 и изменять настройки визуализаторов. Доступные операции представлены в
 следующих подразделах:


	- [Таблица](Table/tabsheet_list.htm)


	- [Диаграмма](Chart/chart_list.htm)


	- [Топооснова](Map/map_list.htm)
