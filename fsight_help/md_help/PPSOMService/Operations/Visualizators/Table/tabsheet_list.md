# Таблица


В данном разделе представлено описание операций, предназначенных для
 работы с таблицей. Таблица используется для отображения данных в регламентных
 и экспресс-отчетах, в рабочей книге. Операции позволяют получить параметры
 и данные таблица, изменить их и сохранить обратно.
