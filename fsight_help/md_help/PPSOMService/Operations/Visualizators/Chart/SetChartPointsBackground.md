# SetChartPointsBackground


## Синтаксис


bool SetChartPointsBackground(ChartPointsId mon,
 ChartPointsBackgroundArgs arg)


## Параметры


mon. Моникёр
 для работы с точками ряда диаграммы.


arg. Параметры выполнения операции.


## Описание


Операция SetChartPointsBackground
 изменяет параметры заливки для точек ряда диаграммы.


## Комментарии


Для выполнения операции укажите в поле mon
 моникёр для работы с точками ряда, а в поле arg
 точки ряда и параметры заливки. Моникёр может быть сформирован на основании
 моникёра диаграммы, указываемого в операциях [GetChart](../GetChart.htm),
 [SetChart](../SetChart.htm), [GetChartSeries](../GetChartSeries.htm).
 Моникёр формируется в следующем формате: «Моникёр
 диаграммы»!Series!«Индекс ряда»!Points.


Результатом работы операции будут логическое значение true,
 если изменение параметров заливки завершилось успешно.
