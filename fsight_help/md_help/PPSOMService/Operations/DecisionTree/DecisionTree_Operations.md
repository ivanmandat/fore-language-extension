# Работа с деревом решений


В данном разделе представлено описание операций, предназначенных для
 работы с деревьями решений. Операции позволяют производить следующие действия:


	- настраивать параметры и метрики дерева решений;


	- настраивать различные блоки: блоки условий, действия, решения,
	 делители;


	- формировать связи между блоками;


	- запускать расчёт дерева решений;


	- получать результаты расчёта дерева решений и экспортировать
	 их в таблицу.
