Операции, используемые для запуска и мониторинга процессов:


		 Операция
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ExecuteBpmOperation](ExecuteBpmOperation.htm)
		 Операция ExecuteBpmOperation
		 выполняет различные операции при работе с экземплярами выполняющихся
		 процессов.


		 ![](../../Sub_Image.gif)
		 [GetActiveProcesses](GetActiveProcesses.htm)
		 Операция GetActiveProcesses
		 получает информацию о процессах, которые выполняются в текущий
		 момент в репозитории.


		 ![](../../Sub_Image.gif)
		 [GetBpmMonitoringWorkspace](GetBpmMonitoringWorkspace.htm)
		 Операция GetBpmMonitoringWorkspace
		 получает информацию о структуре процесса во время его выполнения.


		 ![](../../Sub_Image.gif)
		 [GetBpmProcessStartInfo](GetBpmProcessStartInfo.htm)
		 Операция GetBpmProcessStartInfo
		 получает настройки, с которыми может быть запущен процесс или
		 подпроцесс.


		 ![](../../Sub_Image.gif)
		 [GetBpmStepOperations](GetBpmStepOperations.htm)
		 Операция GetBpmStepOperations
		 получает список операций, доступных для шага выполняемого экземпляра
		 процесса.


		 ![](../../Sub_Image.gif)
		 [GetBpmUserTasks](GetBpmUserTasks.htm)
		 Операция GetBpmUserTasks
		 получения списка шагов, доступных пользователю.


		 ![](../../Sub_Image.gif)
		 [GetBpmProcessInstanceJournal](GetBpmProcessInstanceJournal.htm)
		 Операция GetBpmProcessInstanceJournal
		 получает журнал исполнения экземпляра процесса.


		 ![](../../Sub_Image.gif)
		 [GetExecGroupUsers](GetExecGroupUsers.htm)
		 Операция GetExecGroupUsers
		 получает список пользователей, входящих в группу, назначенную
		 в качестве исполнителя для шага процесса.


		 ![](../../Sub_Image.gif)
		 [GetLinkActiveProcess](GetLinkActiveProcess.htm)
		 Операция GetLinkActiveProcess
		 проверяет наличие связи регламентного отчёта и шагов запущенных
		 экземпляров процессов.


		 ![](../../Sub_Image.gif)
		 [GetProcessInstances](GetProcessInstances.htm)
		 Операция GetProcessInstances
		 получает список выполняемых экземпляров процесса.


		 ![](../../Sub_Image.gif)
		 [GetSubprocessInstancesList](GetSubprocessInstancesList.htm)
		 Операция GetSubprocessInstancesList
		 получает список подпроцессов, которые запущены и выполняются в
		 рамках экземпляра процесса.


		 ![](../../Sub_Image.gif)
		 [OpenMonitoringWS](OpenMonitoringWS.htm)
		 Операция OpenMonitoringWS
		 получает URL-адрес страницы мониторинга выполнения экземпляров
		 процесса.


		 ![](../../Sub_Image.gif)
		 [StartBpmProcess](StartBpmProcess.htm)
		 Операция StartBpmProcess
		 запускает процесс на выполнение.
