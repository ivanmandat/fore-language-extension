# Работа с контейнером запланированных задач


В данном разделе представлено описание операций, предназначенных для
 работы с объектом репозитория - Контейнер запланированных задач. Операции
 позволяют производить следующие действия:


	- создавать или удалять задачи;


	- получать и изменять параметры задач;


	- запускать задачи на выполнение;


	- проверять состояние задач и получать результаты их выполнения.


## Операции


		 Операция
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CloseScheduler](CloseScheduler.htm)


		 Операция CloseScheduler
		 закрывает контейнер запланированных задач.


		 ![](../../Sub_Image.gif)
		 [GetScheduler](GetScheduler.htm)


		 Операция GetScheduler
		 извлекает метаданные контейнера запланированных задач.


		 ![](../../Sub_Image.gif)
		 [OpenScheduler](OpenScheduler.htm)


		 Операция OpenScheduler
		 открывает контейнер запланированных задач и возвращает контекст
		 для работы с его содержимым.


		 ![](../../Sub_Image.gif)
		 [SaveSchedulerSettings](SaveSchedulerSettings.htm)


		 Операция SaveSchedulerSettings
		 сохраняет настройки контейнера запланированных задач в файл.


		 ![](../../Sub_Image.gif)
		 [SetScheduler](SetScheduler.htm)


		 Операция SetScheduler
		 изменяет метаданные контейнера запланированных задач.
