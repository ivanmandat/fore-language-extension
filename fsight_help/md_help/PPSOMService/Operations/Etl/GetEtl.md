# GetEtl


## Синтаксис


GetEtlResult GetEtl(EtlId tEtl, GetEtlArg tArg)


## Параметры


tEtl. Моникёр открытого экземпляра
 задачи ETL.


tArg. Параметры извлечения
 метаданных.


## Описание


Операция GetEtl извлекает метаданные
 задачи ETL.


## Комментарии


Данная операция позволяет получить информацию об объектах, которые созданы
 в задаче ETL и осуществляют импорт и экспорт данных. Для выполнения операции
 необходимо в параметре tEtl указать
 моникёр экземпляра задачи ETL и в параметре tArg
 указать параметры, в соответствии с которыми будет извлекаться информация.
 Моникёр может быть получен при выполнении операции [OpenEtl](OpenEtl.htm). В поле tArg.pattern
 указывается шаблон для получения информации о различных объектах задачи.
 Результатом работы операции будут запрошенные метаданные.
