# Работа с задачей ETL


В данном разделе представлено описание операций, предназначенных для
 работы с объектом репозитория - Задача ETL. Операции позволяют получать
 информацию об объектах задачи ETL, создать новые или удалить существующие
 приёмники/источники/преобразователи и другие объекты, настроить их параметры.
 Также можно запустить задачу ETL на выполнение и отслеживать статус выполнения.
