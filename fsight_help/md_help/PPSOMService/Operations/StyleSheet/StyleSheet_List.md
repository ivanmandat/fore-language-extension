# Работа с таблицей стилей


В данном разделе представлено описание операций, предназначенных для
 работы с таблицей стилей. Операции позволяют производить следующие действия:


	- открывать таблицу стилей;


	- получать список имеющихся стилей;


	- создавать новый, клонировать или удалять существующий стиль;


	- задавать настройки стилей.
