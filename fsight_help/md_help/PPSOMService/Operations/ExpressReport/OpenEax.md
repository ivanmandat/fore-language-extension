# OpenEax


## Синтаксис


OpenEaxResult OpenEax(OdId tObject, OpenEaxArg tArg)


## Параметры


tObject. Моникёр
 объекта, являющегося экспресс-отчетом.


tArg.
 Параметры открытия экспресс-отчета.


## Описание


Операция OpenEax открывает экспресс-отчет
 и возвращает контекст для работы с данными и метаданными.


## Комментарии


Для выполнения операции необходимо в параметре tObject
 указать моникёр объекта, являющегося экспресс-отчетом, а в параметре tArg указать параметры открытия.
 Моникёр объекта может быть получен при выполнении операции [GetObjects](../Repository/GetObjects.htm).
 В параметре tArg могут быть указаны
 режим открытия экспресс-отчета (просмотр или редактирование), значения
 параметров, которые будут переданы параметрическим источникам данных,
 и шаблон для извлечения метаданных. В шаблоне указывается та информация,
 которая должна быть получена при выполнении операции. Если экспресс-отчет
 открывается на просмотр без извлечения каких-либо метаданных, то в качестве
 значения параметра tArg может
 быть указана пустая структура OpenEaxArg.


Результатом работы операции будет моникёр открытого экземпляра экспресс-отчета,
 а также метаданные, если при выполнении операции в параметре tArg
 был задан соответствующий шаблон. Дальнейшая работа с открытым экспресс-отчетом
 осуществляется с помощью операций [GetEaxMd](GetEaxMd.htm)
 и [SetEaxMd](SetEaxMd.htm). Для закрытия открытого экземпляра
 экспресс-отчета используйте операцию [CloseEax](CloseEax.htm).


Дальнейшая работа с объектами экспресс-отчета осуществляется с использованием
 следующих операций:


	- [GetTabSheetData](../Visualizators/GetTabSheetData.htm)
	 и [SetTabSheetData](../Visualizators/SetTabSheetData.htm).
	 Для работы с таблицей данных;


	- [GetChart](../Visualizators/GetChart.htm) и [SetChart](../Visualizators/SetChart.htm).
	 Для работы с диаграммой;


	- [GetMap](../Visualizators/GetMap.htm) и [SetMap](../Visualizators/SetMap.htm).
	 Для работы с картой.
