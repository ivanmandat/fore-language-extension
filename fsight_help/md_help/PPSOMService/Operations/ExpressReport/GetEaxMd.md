# GetEaxMd


## Синтаксис


GetEaxMdResult GetEaxMd(EaxId tEax, GetEaxMdArg
 tArg)


## Параметры


tEax.
 Моникёр открытого экземпляра экспресс-отчета.


tArg. Параметры извлечения
 метаданных экспресс-отчета.


## Описание


Операция GetEaxMd извлекает
 метаданные экспресс-отчета.


## Комментарии


Данная операция позволяет получить следующую информацию:


	- Информация об источниках данных;


	- Параметры объектов экспресс-отчета (таблицы, диаграммы, карты,
	 пузырьковой диаграммы);


	- Параметры оформления различных элементов таблицы;


	- Экспортированный отчет в заданный формат.


Также, используя операцию, можно сделать проверку на возможность детализации
 данных, произвести детализацию и получить данные после детализации.


Для выполнения операции необходимо в параметре tEax
 указать моникёр экземпляра экспресс-отчета и в параметре tArg
 указать параметры, в соответствии с которыми будет извлекаться информация.
 Моникёр может быть получен при выполнении операции [OpenEax](OpenEax.htm).
 В поле tArg.pattern указывается шаблон для получения информации о различных
 элементах экспресс-отчета. Результатом работы операции будет запрошенная
 информация. Для получения дополнительной информации об объектах экспресс-отчета
 могут быть использованы следующие операции: [GetTabSheetData](../Visualizators/GetTabSheetData.htm),
 [GetChart](../Visualizators/GetChart.htm), [GetMap](../Visualizators/GetMap.htm).
