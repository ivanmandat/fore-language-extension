# SetEaxMd


## Синтаксис


SetEaxMdResult SetEaxMd(EaxId tEax, SetEaxMdArg
 tArg)


## Параметры


tEax. Моникёр открытого экземпляра экспресс-отчета.


tArg. Параметры изменения метаданных
 экспресс-отчета.


## Описание


Операция SetEaxMd изменяет метаданные
 экспресс-отчета.


## Комментарии


Данная операция позволяет изменить практически все параметры, которые
 могут быть указаны в шаблоне операции (кроме тех, которые доступны для
 использования только с операцией [GetEaxMd](GetEaxMd.htm)).


Для выполнения операции необходимо в параметре tEax
 указать моникёр экземпляра экспресс-отчета и в параметре tArg
 указать параметры, в соответствии с которыми будут изменяться различные
 элементы экспресс-отчета. Моникёр может быть получен при выполнении операции
 [OpenEax](OpenEax.htm). В поле tArg.pattern
 указывается шаблон, в соответствии с которым будет производиться изменение,
 а в поле tArg.meta указываются
 обновленные метаданные. Также для изменения объектов экспресс-отчета могут
 быть использованы следующие операции: [SetTabSheetData](../Visualizators/SetTabSheetData.htm),
 [SetChart](../Visualizators/SetChart.htm).


После внесения изменений для сохранения экспресс-отчета используйте
 операцию [SaveObject](../Repository/SaveObject.htm) или [SaveObjectAs](../Repository/SaveObjectAs.htm).
