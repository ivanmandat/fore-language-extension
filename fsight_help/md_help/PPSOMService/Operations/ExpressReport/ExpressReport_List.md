# Работа с экспресс-отчётами


В данном разделе представлено описание операций, предназначенных для
 работы с объектом репозитория - Экспресс-отчёт. Операции позволяют производить
 следующие действия:


	- открывать экспресс-отчёт на просмотр и редактирование;


	- работать с отдельными листами экспресс-отчёта;


	- получать и изменять различные настройки экспресс-отчёта (сортировка
	 и фильтрация данных, настройка итогов, условного форматирования, подсветка
	 данных, определение параметров заголовка экспресс-отчёта);


	- работать с визуализаторами экспресс-отчёта: таблицей,
	 картой, диаграммой, пузырьковой диаграммой;


	- работать с источниками данных (изменение источника данных, формирование
	 мульти-OLAP'а);


	- работать со стилями различных элементов экспресс-отчёта;


	- работать с внутренней таблицей, которая лежит в основе экспресс-отчёта
	 (изменений данные, изменение порядка расположения измерений и отметки
	 по ним, настройка вычисляемых показателей);


	- производить детализацию данных.
