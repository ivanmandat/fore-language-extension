# Операции веб-сервиса


Сервис PP.SOM содержит операции для работы со всеми блоками продукта
 «Форсайт. Аналитическая платформа».
 В зависимости от того, через какую [точку
 подключения](../Intro/PP7Service_Using.htm) осуществляется работа, для использования будет доступен
 различный набор операций.
