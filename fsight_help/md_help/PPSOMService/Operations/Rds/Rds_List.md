# Работа со справочниками НСИ


В данном разделе представлено описание операций, предназначенных для
 работы со справочниками НСИ и с репозиторием НСИ. Операции позволяют производить
 следующие действия:


	- работать со структурой справочника НСИ (получать и изменять
	 различные элементы структуры: атрибуты, параметры, связи, уровни,
	 параметры перевода);


	- работать с элементами справочников НСИ;


	- работать со структурой репозитория НСИ.
