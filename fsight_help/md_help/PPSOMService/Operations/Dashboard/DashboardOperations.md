# Работа с информационными панелями


В данном разделе представлено описание операций, предназначенных для
 работы с объектом репозитория - [Информационная
 панель](UiDashboard.chm::/InformationPanel/Starting.htm).


## Операции

		 ![](../../Sub_Image.gif)

		 Операция GetDashboardGxPageSettings
		 получает настройки печати информационной панели.


		 ![](../../Sub_Image.gif)
		 [GetDashboardMeta](GetDashboardMeta.htm)
		 Операция GetDashboardMeta
		 получает метаданные информационной панели.


		 ![](../../Sub_Image.gif)
		 [OpenDashboard](OpenDashboard.htm)
		 Операция OpenDashboard
		 открывает информационную панель.


		 ![](../../Sub_Image.gif)
		 [SetDashboardGxPageSettings](SetDashboardGxPageSettings.htm)
		 Операция SetDashboardGxPageSettings

		 ![](../../Sub_Image.gif)
		 [SetDashboardMeta](SetDashboardMeta.htm)
		 Операция SetDashboardMeta
		 изменяет метаданные информационной панели.

		 данных, который редактируется в редакторе выражений.


		 ![](../../Sub_Image.gif)
		 [DashboardFormulaEditorEdit](DashboardFormulaEditorEdit.htm)
		 Операция DashboardFormulaEditorEdit

		 панели или модели данных.
