# Работа с бизнес-приложениями


В данном разделе представлено описание операций, предназначенных для
 работы с бизнес-приложениями. Операции позволяют получать и изменять метаданные
 бизнес-приложений.


## Операции

		 Операция GetDba получает
		 метаданные бизнес-приложения.


		 ![](../../Sub_Image.gif)
		 [OpenDba](OpenDba.htm)
		 Операция OpenDba открывает

		 ![](../../Sub_Image.gif)

		 Операция SaveDba сохраняет

		 ![](../../Sub_Image.gif)

		 Операция SetDba изменяет
