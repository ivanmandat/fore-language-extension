# GetCubeImport


## Синтаксис


GetCubeImportResult GetCubeImport(CubeImportId tCubeImport,
 GetCubeImportArg tArg)


## Параметры


tCubeImport. Моникёр объекта
 импорта данных в куб.


tArg. Параметры выполнения
 операции.


## Описание


Операция GetCubeImport извлекает
 метаданные объекта импорта данных в куб.


## Комментарии


Данная операция позволяет получить различные настройки загрузчика данных
 в куб:


	- параметры источников данных, из которых будет производиться
	 импорт;


	- параметры куба-приёмника;


	- параметры привязки полей источника данных к измерениям и фактам
	 куба;


	- статус выполнения импорта данных.


Для выполнения операции в поле tCubeImport
 укажите моникёр экземпляра объекта импорта и в поле tArg
 укажите параметры, в соответствии с которыми будет извлекаться информация.
 Моникёр может быть получен при выполнении операции [OpenCubeImport](OpenCubeImport.htm).
 В поле tArg.pattern указывается
 шаблон для получения информации о загрузчике данных.
