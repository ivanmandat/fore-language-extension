# SetCube


## Синтаксис


SetCubeResult SetCube(CubeId tCube, SetCubeArg tArg)


## Параметры


tCube. Моникёр открытого экземпляра
 куба.


tArg. Параметры изменения метаданных.


## Описание


Операция SetCube изменяет метаданные
 куба.


## Комментарии


Данная операция позволяет изменять следующие настройки в структуре куба:


	- набор измерений;


	- варианты отображения;


	- параметры управляющих измерений;


	- кубы-источники и параметры фиксации измерений при работе с представление-кубом;


	- агрегацию по измерениям куба.


Для выполнения операции в параметре tCube
 укажите моникёр открытого экземпляра куба и в параметре tArg
 укажите параметры, в соответствии с которыми будут произведены изменения.
 Моникёр может быть получен при выполнении операции [OpenCube](OpenCube.htm).
 Куб должен быть открыт на редактирование. В поле tArg.pattern
 указывается шаблон для изменения структуры куба, а в поле tArg.meta
 указываются обновленные метаданные.


Результатом работы операции будет моникёр измененного куба и запрошенные
 метаданные, которые могут быть указаны в шаблоне tArg.metaGet.


Для сохранения внесенных изменений после операции SetCube
 выполните операцию [SaveObject](../Repository/SaveObject.htm)
 или [SaveObjectAs](../Repository/SaveObjectAs.htm).


Примечание.
 В «Форсайт. Аналитическая платформа»
 изменение структуры доступно только для стандартных кубов.
