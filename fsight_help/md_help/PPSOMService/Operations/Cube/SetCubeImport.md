# SetCubeImport


## Синтаксис


SetCubeImportResult SetCubeImport(CubeImportId tCubeImport,
 SetCubeImportArg tArg)


## Параметры


tCubeImport. Моникёр объекта
 импорта данных в куб.


tArg. Параметры выполнения
 операции.


## Описание


Операция SetCubeImport изменяет
 метаданные объекта импорта данных в куб.


## Комментарии


Данная операция позволяет произвести следующие действия:


	- настроить параметры инициализации различных видов источников
	 данных, из которых будет производиться импорт;


	- указать существующий куб-приёмник или определить параметры создания
	 нового куба;


	- задать параметры привязки полей источника данных к измерениям
	 и фактам куба;


	- определить различные дополнительные настройки, которые будут
	 использоваться при импорте;


	- произвести импорт данных.


Для выполнения операции в поле tCubeImport
 укажите моникёр экземпляра объекта импорта и в поле tArg
 укажите параметры, в соответствии с которыми будут произведены изменения.
 Моникёр может быть получен при выполнении операции [OpenCubeImport](OpenCubeImport.htm).
 В поле tArg.pattern указывается
 шаблон для изменения импорта, а в поле tArg.meta
 указываются обновленные метаданные.


Результатом работы операции будет моникёр объекта импорта, для которого
 выполнялась операция, и запрошенные метаданные, которые могут быть указаны
 в шаблоне tArg.metaGet.
