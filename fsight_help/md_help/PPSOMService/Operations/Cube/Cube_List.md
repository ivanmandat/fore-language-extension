# Работа с кубами


В данном разделе представлено описание операций, предназначенных для
 работы с различными видами кубов репозитория. Операции позволяют получать
 информацию о структуре кубов и данные кубов. Для стандартных кубов поддерживается
 изменение структуры.


## Операции


		 Операция
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CloseCube](CloseCube.htm)


		 Операция CloseCube
		 закрывает экземпляр открытого куба и возвращает признак удачного
		 закрытия.


		 ![](../../Sub_Image.gif)
		 [CloseCubeImport](CloseCubeImport.htm)


		 Операция CloseCubeImport
		 закрывает экземпляр объекта импорта данных в куб и возвращает
		 признак удачного закрытия.


		 ![](../../Sub_Image.gif)
		 [GetCalculatedCubeFormulas](GetCalculatedCubeFormulas.htm)


		 Операция GetCalculatedCubeFormulas
		 получает формулы для вычисляемого куба.


		 ![](../../Sub_Image.gif)
		 [GetCube](GetCube.htm)


		 Операция GetCube извлекает
		 метаданные куба.


		 ![](../../Sub_Image.gif)
		 [GetCubeImport](GetCubeImport.htm)


		 Операция GetCubeImport
		 извлекает метаданные объекта импорта данных в куб.


		 ![](../../Sub_Image.gif)
		 [OpenCube](OpenCube.htm)


		 Операция OpenCube открывает
		 куб репозитория и возвращает контекст для работы со структурой
		 куба.


		 ![](../../Sub_Image.gif)
		 [OpenCubeImport](OpenCubeImport.htm)


		 Операция OpenCubeImport
		 создает объект импорта данных в куб и возвращает контекст для
		 работы с ним.


		 ![](../../Sub_Image.gif)
		 [SaveCalculatedCubeFormulas](SaveCalculatedCubeFormulas.htm)


		 Операция SaveCalculatedCubeFormulas
		 сохраняет изменения в формулах вычисляемого куба.


		 ![](../../Sub_Image.gif)
		 [SetCalculatedCubeFormulas](SetCalculatedCubeFormulas.htm)


		 Операция SetCalculatedCubeFormulas
		 изменяет формулы для вычисляемого куба.


		 ![](../../Sub_Image.gif)
		 [SetCube](SetCube.htm)


		 Операция SetCube изменяет
		 метаданные куба.


		 ![](../../Sub_Image.gif)
		 [SetCubeImport](SetCubeImport.htm)


		 Операция SetCubeImport
		 изменяет метаданные объекта импорта данных в куб.


		 ![](../../Sub_Image.gif)
		 [UpdCubeCache](UpdCubeCache.htm)


		 Операция UpdCubeCache
		 создаёт новый или обновляет существующий кеш In-Memory для указанного
		 куба.
