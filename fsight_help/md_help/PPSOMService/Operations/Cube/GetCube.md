# GetCube


## Синтаксис


GetCubeResult GetCube(CubeId tCube, GetCubeArg tArg)


## Параметры


tCube. Моникёр открытого экземпляра
 куба.


tArg. Параметры извлечения
 метаданных.


## Описание


Операция GetCube извлекает метаданные
 куба.


## Комментарии


Операция позволяет получить информацию об измерениях и вариантах отображения
 куба, параметры управляющих измерений, кубы-источники и параметры фиксации
 измерений при работе с представление-кубом, настройки агрегации по измерениям.
 Для выполнения операции необходимо в параметре tCube
 указать моникёр открытого экземпляра куба и в параметре tArg
 указать параметры, в соответствии с которыми будет извлекаться информация.
 Моникёр может быть получен при выполнении операции [OpenCube](OpenCube.htm).
 В поле tArg.pattern указывается
 шаблон для получения информации о структуре куба, в полях tArg.data
 или tArg.matrix могут быть указаны
 параметры, в соответствии с которыми будут получены данные куба. Результатом
 работы операции будут запрошенные данные и метаданные.
