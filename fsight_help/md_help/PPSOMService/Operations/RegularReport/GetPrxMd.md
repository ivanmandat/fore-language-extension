# GetPrxMd


## Синтаксис


GetPrxMdResult GetPrxMd(PrxId tPrx, GetPrxMdArg
 tArg)


## Параметры


tPrx.
 Моникёр открытого экземпляра регламентного отчета.


tArg.
 Параметры извлечения метаданных регламентного отчета.


## Описание


Операция GetPrxMd извлекает
 метаданные регламентного отчета.


## Комментарии


Данная операция позволяет получить следующую информацию:


	- Информация о листах отчета;


	- Информация об источниках данных отчета;


	- Информация о различных областях данных;


	- Параметры элементов управления отчета;


	- Экспортированный отчет в заданный формат.


Также, используя операцию, можно сделать проверку на привязку ячеек
 к областям данных.


Для выполнения операции необходимо в параметре tPrx
 указать моникёр экземпляра регламентного отчета и в параметре tArg
 указать параметры, в соответствии с которыми будет извлекаться информация.
 Моникёр может быть получен при выполнении операции [OpenPrx](OpenPrx.htm). В поле tArg.pattern
 указывается шаблон для получения информации о различных элементах регламентного
 отчета. Результатом работы операции будут запрошенные метаданные. Для
 получения дополнительной информации о таблицах листов и объектах, которые
 могут быть созданы на листах, используйте следующие операции: [GetTabSheetData](../Visualizators/GetTabSheetData.htm), [GetChart](../Visualizators/GetChart.htm),
 [GetMap](../Visualizators/GetMap.htm).
