# OpenPrx


## Синтаксис


OpenPrxResult OpenPrx(OdId tObject, OpenPrxArg tArg)


## Параметры


tObject. Моникёр
 объекта, являющегося регламентным отчетом.


tArg.
 Параметры открытия регламентного отчета.


## Описание


Операция OpenPrx открывает регламентный
 отчет и возвращает контекст для работы с данными и метаданными.


## Комментарии


Для выполнения операции необходимо в параметре tObject
 указать моникёр объекта, являющегося регламентным отчетом, а в параметре
 tArg указать параметры открытия.
 Моникёр объекта может быть получен при выполнении операции [GetObjects](../Repository/GetObjects.htm). В параметре tArg
 могут быть указаны режим открытия отчета (просмотр или редактирование),
 значения параметров, которые будут переданы параметрическим источникам
 данных, и шаблон для извлечения метаданных. В шаблоне указывается та информация,
 которая должна быть получена при выполнении операции.


Результатом работы операции будет моникёр открытого экземпляра регламентного
 отчета, а также метаданные, если при выполнении операции в параметре tArg был задан соответствующий шаблон.
 Дальнейшая работа с открытым отчетом осуществляется с помощью операций
 [GetPrxMd](GetPrxMd.htm) и [SetPrxMd](SetPrxMd.htm). Для закрытия открытого экземпляра
 регламентного отчета используйте операцию [ClosePrx](ClosePrx.htm).


Для работы с таблицами листов регламентного отчета используйте операции
 [GetTabSheetData](../Visualizators/GetTabSheetData.htm)
 и [SetTabSheetData](../Visualizators/SetTabSheetData.htm).
 Для работы с объектами, которые могут располагаться на листах, используйте
 следующие операции:


	- [GetChart](../Visualizators/GetChart.htm)
	 и [SetChart](../Visualizators/SetChart.htm).
	 Для работы с диаграммой;


	- [GetMap](../Visualizators/GetMap.htm)
	 и [SetMap](../Visualizators/SetMap.htm).
	 Для работы с картой.
