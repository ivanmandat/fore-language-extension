# SetPrxMd


## Синтаксис


SetPrxMdResult SetPrxMd(PrxId tPrx, SetPrxMdArg
 tArg)


## Параметры


tEax. Моникёр открытого экземпляра экспресс-отчета.


tArg. Параметры изменения метаданных
 экспресс-отчета.


## Описание


Операция SetPrxMd изменяет метаданные
 регламентного отчета.


## Комментарии


Данная операция позволяет изменить практически все параметры, которые
 могут быть указаны в шаблоне операции (кроме тех, которые доступны для
 использования только с операцией [GetPrxMd](GetPrxMd.htm)).


Для выполнения операции необходимо в параметре tEax
 указать моникёр экземпляра регламентного отчета и в параметре tArg
 указать параметры, в соответствии с которыми будут изменяться различные
 элементы отчета. Моникёр может быть получен при выполнении операции [OpenPrx](OpenPrx.htm).
 В поле tArg.pattern указывается
 шаблон, в соответствии с которым будет производиться изменение, а в поле
 tArg.meta указываются обновленные
 метаданные. Также для изменения объектов отчета могут быть использованы
 следующие операции: [SetTabSheetData](../Visualizators/SetTabSheetData.htm),
 [SetChart](../Visualizators/SetChart.htm).


После внесения изменений для сохранения регламентного отчета используйте
 операцию [SaveObject](../Repository/SaveObject.htm)
 или [SaveObjectAs](../Repository/SaveObjectAs.htm).
