# GetDbCommand


## Синтаксис


GetDbCommandResult GetDbCommand(DbCommandId tDbCommand,
 GetDbCommandArg tArg)


## Параметры


tDbCommand. Моникёр
 SQL-объекта репозитория.


tArg.
 Параметры извлечения метаданных.


## Описание


Операция GetDbCommand извлекает
 метаданные SQL-объекта репозитория.


## Комментарии


Данная операция позволяет получить следующую информацию:


	- Базу данных репозитория, на работу с которой настроен SQL-объект;


	- Текст выполняемого SQL-запроса;


	- Информацию о параметрах, которые используются для передачи данных
	 в запрос;


	- Результаты выполнения SQL-объекта.


Для выполнения операции необходимо в поле tDbCommand
 указать моникёр экземпляра SQL-объекта и в поле tArg
 указать параметры, в соответствии с которыми будет извлекаться информация.
 Моникёр может быть получен при выполнении операции [OpenDbCommand](OpenDbCommand.htm).
 В поле tArg.pattern указывается
 шаблон, в соответствии с которым будет извлекаться информация. В зависимости
 от типа SQL-объекта, с которым осуществляется работа, для его выполнения
 и получения результатов необходимо определить следующие поля в шаблоне:


	- Для Запроса: Поле rows и при необходимости ограничения
	 извлекаемого набора данных - поле rowsFilter;


	- Для команды СУБД и Процедуры:
	 Поле exec.


Если объект параметрический, то предварительно задайте значения параметров,
 используя операцию [SetDbCommand](SetDbCommand.htm).
 Вычислить объект можно указав поля rows/exec в поле tArg.metaGet
 операции [SetDbCommand](SetDbCommand.htm).


Результатом работы операции будут запрошенные метаданные и результаты
 выполнения.
