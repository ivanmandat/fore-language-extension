# Работа с SQL-объектами репозитория


В данном разделе представлено описание операций, предназначенных для
 работы с различными SQL-объектами репозитория: [Запрос](UiNavObj.chm::/Query/query.htm),
 [Команда
 СУБД](UiNavObj.chm::/CUBD/UiDb_relationa_CUBD.htm), [Процедура](UiNavObj.chm::/procedure/UiDb_relationa_procedure.htm).
 Операции позволяют производить следующие действия:


	- работать со структурой SQL-объектов (получать и изменять запрос
	 и параметры объекта);


	- получать результаты выполнения SQL-объектов.
