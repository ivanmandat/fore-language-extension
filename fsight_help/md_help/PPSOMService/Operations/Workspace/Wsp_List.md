# Работа с рабочим пространством


В данном разделе представлено описание операций, предназначенных для
 работы с объектом репозитория - Рабочее пространство. Операции позволяют
 получать информацию о фигурах рабочего пространства, создать новые или
 удалить существующие фигуры, настраивать их параметры.
