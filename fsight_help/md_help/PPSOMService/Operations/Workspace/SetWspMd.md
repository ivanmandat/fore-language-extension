# SetWspMd


## Синтаксис


SetWspMdResult SetWspMd(WspId tWsp, SetWspMdArg
 tArg)


## Параметры


tWsp. Моникёр
 открытого экземпляра рабочего пространства.


tArg. Параметры выполнения
 операции.


## Описание


Операция SetWspMd изменяет рабочее
 пространство.


## Комментарии


Данная операция позволяет изменить список фигур на рабочем пространстве
 и их свойства. Для выполнения операции укажите в поле tWsp
 моникёр экземпляра рабочего пространства, а в поле tArg
 параметры, в соответствии с которыми будут изменяться фигуры. Моникёр
 может быть получен при выполнении операции [OpenWsp](OpenWsp.htm).


В поле tArg.pattern указывается
 шаблон, в соответствии с которым будет производиться изменение, а в поле
 tArg.meta обновлённые метаданные
 фигур или действия, выполняемые на рабочем пространстве.


После внесения изменений для сохранения рабочего пространства используйте
 операцию [SaveObject](../Repository/SaveObject.htm) или [SaveObjectAs](../Repository/SaveObjectAs.htm).
