# Работа с метаданными объектов


В данном разделе представлено описание операций, предназначенных для
 работы с метаданными объектов репозитория. Операции позволяют производить
 следующие действия:


	- получать внутреннее представление объектов;


	- изменять документы репозитория, загружая в них новые файлы.
