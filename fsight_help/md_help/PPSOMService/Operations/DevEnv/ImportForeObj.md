# ImportForeObj


## Синтаксис


ImportForeObjResult ImportForeObj(string mon, ImportForeArg
 tArg)


## Параметры


mon. Моникёр
 файла, в котором сохранён объект репозитория.


tArg.
 Параметры выполнения операции.


## Описание


Операция ImportForeObj импортирует
 объект среды разработки из внешнего файла в репозиторий.


## Комментарии


Для выполнения операции укажите в параметре mon
 моникёр файла, сохранённого на сервере, а в параметре tArg
 параметры сохранения объекта в репозиторий. Моникёр может быть получен
 после выполнения операции [ExportForeObj](ExportForeObj.htm),
 которая экспортирует объект среды разработки в файл.
