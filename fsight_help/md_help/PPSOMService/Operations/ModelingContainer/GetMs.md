# GetMs


## Синтаксис


GetMsResult GetMs(MsId tMs, GetMsArg tArg)


## Параметры


tMs. Моникёр открытого экземпляра
 контейнера моделирования;


tArg. Параметры выполнения
 операции.


## Описание


Операция GetMs извлекает метаданные
 контейнера моделирования и его дочерних объектов.


## Комментарии


Операция позволяет:


	- создавать новые объекты в контейнере моделирования;


	- получать данные и метаданные объектов;


	- выполнять различные действия над объектами, например, рассчитывать
	 уравнения или модели, получать рассчитанные данные и так далее.


Для выполнения операции укажите в поле tMs
 моникёр контейнера моделирования и в поле tArg
 параметры выполнения операции:


	- в поле tArg.pattern
	 указывается шаблон, в соответствии с которым будут извлекаться метаданные;


	- в поле tArg.filter
	 указывается фильтр, в соответствии с которым будут отбираться только
	 те объекты, с которыми предстоит дальнейшая работа;


	- в поле tArg.execParams
	 указываются параметры выполнения каких-либо действий, которые необходимо
	 выполнить над объектом контейнера моделирования. Сам объект указывается
	 в поле tArg.pattern.Item.


Результатом операции будут запрошенные метаданные или результаты выполнения
 указанных действий.
