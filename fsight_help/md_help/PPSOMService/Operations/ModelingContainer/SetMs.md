# SetMs


## Синтаксис


SetMsResult SetMs(MsId tMs, SetMsArg tArg)


## Параметры


tMs. Моникёр открытого экземпляра
 контейнера моделирования;


tArg. Параметры изменения данных.


## Описание


Операция SetMs изменяет метаданные
 контейнера моделирования и его дочерних объектов.


## Комментарии


Операция позволяет:


	- настраивать параметры объектов;


	- изменять данные и метаданные объектов;


	- выполнять различные действия над объектами, например рассчитывать
	 уравнения или модели, получать рассчитанные данные и другое.


Для выполнения операции укажите в поле tMs
 моникёр контейнера моделирования и в поле tArg
 параметры выполнения операции:


	- в поле tArg.pattern
	 указывается шаблон, в соответствии с которым будут изменяться метаданные;


	- в поле tArg.meta указывается
	 метаданные, которые будут применены к объектам;


	- в поле tArg.execParams
	 указываются параметры выполнения каких-либо действий, которые необходимо
	 выполнить над объектами контейнера моделирования;


	- в поле tArg.metaGet
	 может быть указан шаблон, в соответствии с которым будут получены
	 обновленные метаданные.


Результатом операции будут запрошенные метаданные, если было определено
 поле tArg.metaGet, или результаты
 выполнения каких-либо действий.


Для сохранения внесенных изменений используйте операцию [SaveObject](../Repository/SaveObject.htm).
