# Работа с контейнером моделирования


В разделе представлено описание операций, структур которые они используют,
 предназначенных для работы с объектом репозитория - «Контейнер моделирования».
 Операции позволяют:


	- получать и изменять параметры объектов контейнера моделирования:
	 «Переменная», «Уравнение», «Метамодель», «Модель»;


	- работать со сценариями моделирования;


	- работать с интеллектуальным анализом данных.
