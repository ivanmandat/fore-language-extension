# SetWbkMd


## Синтаксис


SetWbkMdResult SetWbkMd(WbkId tWbk, SetWbkMdArg
 tArg)


## Параметры


tWbk. Моникёр экземпляра рабочей
 книги;


tArg. Параметры выполнения
 операции.


## Описание


Операция SetWbkMd изменяет данные
 и метаданные рабочей книги.


## Комментарии


Для выполнения операции необходимо в поле tWbk
 указать моникёр открытого экземпляра рабочей книги, а в поле tArg
 указать параметры изменения. Моникёр может быть получен при выполнении
 операции [OpenWbk](OpenWbk.htm).
 В поле tArg.pattern указывается
 шаблон, в соответствии с которым будут производиться изменения, а в поле
 tArg.meta обновленные данные и
 метаданные. Для сохранения изменений после операции SetWbkMd
 выполните операцию [SaveObject](../Repository/SaveObject.htm)
 или [SaveObjectAs](../Repository/SaveObjectAs.htm).


Результатом работы операции SetWbkMd
 будет моникёр измененной рабочей книги и запрошенные в шаблоне tArg.metaGet
 данные/метаданные.
