# GetWbkMd


## Синтаксис


GetWbkMdResult GetWbkMd(WbkId tWbk, GetWbkMdArg
 tArg)


## Параметры


tWbk. Моникёр экземпляра рабочей
 книги;


tArg. Параметры выполнения
 операции.


## Описание


Операция GetWbkMd извлекает
 данные и метаданные рабочей книги.


## Комментарии


Для выполнения операции необходимо в поле tWbk
 указать моникёр открытого экземпляра рабочей книги, а в поле tArg
 указать параметры извлечения. Моникёр может быть получен при выполнении
 операции [OpenWbk](OpenWbk.htm).
 Результатом работы операции будут запрошенные данные и метаданные.
