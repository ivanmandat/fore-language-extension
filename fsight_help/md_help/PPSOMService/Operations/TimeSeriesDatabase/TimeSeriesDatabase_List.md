# Работа с базой данных временных рядов


В данном разделе представлено описание операций, предназначенных для
 работы с объектом репозитория - База данных временных рядов (БДВР) и Рабочая
 книга. Операции позволяют производить следующие действия:


	- получить и изменить различные параметры и структуру базы данных
	 временных рядов;


	- работать с иерархией временных рядов и их данными;


	- работать с рабочими книгами, созданными на основе временных
	 рядов.
