# OpenWbk


## Синтаксис


OpenWbkResult OpenWbk(OdId tOb, OpenWbkArg tArg)


## Параметры


tOb. Моникёр объекта репозитория,
 являющегося рабочей книгой;


tArg. Параметры выполнения
 операции.


## Описание


Операция OpenWbk открывает рабочую
 книгу и возвращает контекст для работы с ней.


## Комментарии


Для выполнения операции необходимо в поле tOb
 указать моникёр объекта, являющегося рабочей книгой, а в поле tArg
 указать параметры открытия. В поле tArg
 могут быть указаны режим открытия рабочей книги (просмотр или редактирование),
 значения параметров, которые будут переданы параметрическим источникам
 данных, и параметры для извлечения и изменения данных или метаданных.
 Параметры для извлечения данных/метаданных указываются в поле tArg.metaGet
 или tArg.metaGetActiveSheet. Параметры
 для изменения данных/метаданных указываются в поле tArg.metaSet.
 Результатом работы операции будет моникёр открытого экземпляра рабочей
 книги, а также метаданные, если при выполнении операции был задан соответствующий
 шаблон. Дальнейшая работа с открытой рабочей книгой осуществляется с помощью
 операций [GetWbkMd](GetWbkMd.htm)
 и [SetWbkMd](SetWbkMd.htm).
 Для закрытия открытого экземпляра рабочей книги используйте операцию [CloseWbk](CloseWbk.htm).


Для работы с таблицей рабочей книги используйте операции [GetTabSheetData](../Visualizators/GetTabSheetData.htm) и [SetTabSheetData](../Visualizators/SetTabSheetData.htm),
 для работы с диаграммой рабочей книги используйте операции [GetChart](../Visualizators/GetChart.htm) и [SetChart](../Visualizators/SetChart.htm).
