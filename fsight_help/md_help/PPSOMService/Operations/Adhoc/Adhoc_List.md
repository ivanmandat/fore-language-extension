# Работа с аналитической панелью


В данном разделе представлено описание операций, предназначенных для
 работы с объектом репозитория - Аналитическая панель. Операции позволяют
 получать информацию о структуре аналитической панели и изменять её.


## Операции


		 Операция
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CloseAdHoc](CloseAdHoc.htm)


		 Операция CloseAdHoc
		 закрывает экземпляр открытой аналитической панели.


		 ![](../../Sub_Image.gif)
		 [GetAdHoc](GetAdHoc.htm)


		 Операция GetAdHoc извлекает
		 метаданные аналитической панели.


		 ![](../../Sub_Image.gif)
		 [OpenAdHoc](OpenAdHoc.htm)


		 Операция OpenAdHoc
		 открывает аналитическую панель и возвращает контекст для работы
		 с её содержимым.


		 ![](../../Sub_Image.gif)
		 [SetAdHoc](SetAdHoc.htm)


		 Операция SetAdHoc изменяет
		 метаданные аналитической панели.
