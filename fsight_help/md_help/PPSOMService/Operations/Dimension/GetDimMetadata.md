# GetDimMetadata


## Синтаксис


GetDimMetadataResult GetDimMetadata(DmId tDim, GetDimMetadataArg
 tArg)


## Параметры


tDim. Моникёр открытого экземпляра
 справочника.


tArg. Параметры извлечения
 метаданных справочника.


## Описание


Операция GetDimMetadata извлекает
 метаданные справочника.


## Комментарии


Данная операция позволяет получить следующую информацию:


	- Информацию о структуре справочников (атрибуты, уровни);


	- Информация о дочерних объектах справочников (схемы отметок,
	 группы элементов, альтернативные иерархии);


	- Информация об элементах, которые получены при построении справочника.


	- Информация об отметке элементов.


Для выполнения операции необходимо в параметре tDim
 указать моникёр экземпляра справочника и в параметре tArg
 указать параметры, в соответствии с которыми будет извлекаться информация.
 Моникёр может быть получен при выполнении операции [OpenDim](OpenDim.htm).
 В поле tArg.pattern указывается
 шаблон для получения информации о структуре справочника и его дочерних
 объектах, в поле elsArg указываются
 параметры извлечения информации об элементах справочника. Результатом
 работы операции будут запрошенные данные.
