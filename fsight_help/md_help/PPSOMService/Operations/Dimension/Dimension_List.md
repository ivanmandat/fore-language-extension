# Работа со справочниками


В данном разделе представлено описание операций и структур, предназначенных
 для работы с различными справочниками репозитория. Операции позволяют
 производить следующие действия:


	- Получать информацию о структуре справочника и его дочерних объектов;


	- Получать информацию об элементах справочника;


	- Работать с отметкой элементов;


	- Работать со схемами отметок, группами элементов, альтернативными
	 иерархиями справочника.
