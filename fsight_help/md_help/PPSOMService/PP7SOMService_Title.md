## С чего начать


		 Руководство предназначено
		 для изучения работы с веб-сервисом PP.SOM, который реализован
		 в составе «Форсайт. Аналитическая платформа».


		Изучив данный раздел, вы научитесь с помощью веб-сервиса:


			- подключаться к репозиторию и работать с его
			 объектами;


			- изменять содержимое репозитория, настраивать
			 параметры объектов;


			- работать с менеджером безопасности репозитория,
			 настраивать права доступа к объектам;


			- получать и изменять данные объектов;


			- работать с расширениями «Форсайт. Аналитическая платформа».


		 Перед изучением данного раздела
		 рекомендуем ознакомиться с общими принципами работы с объектами
		 в настольной версии «Форсайт. Аналитическая платформа».
		 Также необходимо изучить системные сборки языка Fore,
		 которые предназначены для работы с теми областями, с которыми
		 предполагается работа посредством веб-сервиса PP.SOM. Знание системных
		 сборок облегчит понимание тех структур и полей, которые используются
		 при выполнении операций веб-сервиса.


		Изучение данного раздела рекомендуется начать с ознакомления
		 со [способом подключения](Intro/PP7Service_Using.htm)
		 к веб-сервису, а также с [общими
		 принципами организации](Intro/PP7Service_GeneralPrinciple.htm) его работы. Если разработка ведется
		 в среде Microsoft Visual Studio, то необходимо ознакомиться с
		 классом SomPortTypeClient,
		 который генерируется при подключении к BI-серверу.


		Для изучения структуры операций используйте веб-приложение -
		 [Консоль веб-сервиса](Intro/WSC_Intro.htm).
