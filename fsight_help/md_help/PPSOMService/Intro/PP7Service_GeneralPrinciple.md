# Общие принципы организации веб-сервиса


Все операции веб-сервиса по выполняемым функциям можно разделить на
 четыре группы:


	- Операции открытия экземпляра
	 объекта

	Данные операции имеют в своем наименовании префикс Open.
	 Операции выполняются при начале работы с каким-либо объектом. Результатом
	 выполнения операций будет моникёр открытого экземпляра объекта. Данный
	 моникёр в дальнейшем используется для получения доступа к данным и
	 метаданным объекта. Также для операций может быть указан шаблон, в
	 соответствии с которым при открытии будут извлечены необходимые данные/метаданные.


	- Операции извлечения данных
	 или метаданных

	Данные операции имеют в своем наименовании префикс Get.
	 Операции выполняются для открытого экземпляра объекта и позволяют
	 получить его данные или метаданные. Для операций указывается шаблон,
	 который позволяет получить только необходимые данные/метаданные.


	- Операции изменения данных или
	 метаданных

	Данные операции имеют в своем наименовании префикс Set.
	 Операции выполняются для открытого экземпляра объекта и позволяют
	 изменить его данные/метаданные. Для выполнения операции указывается
	 шаблон, определяющий что именно будет изменяться, а также новые данные/метаданные.


	- Операции закрытия экземпляров
	 объектов

	Данные операции имеют в своем наименовании префикс Close.
	 Операции используются для закрытия экземпляров объектов и освобождения
	 ресурсов сервера, которые были затрачены на работу с объектами.


Для выполнения базовых операций над объектами (создание, удаление и
