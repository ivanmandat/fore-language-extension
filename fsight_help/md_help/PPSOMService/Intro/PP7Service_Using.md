# Подключение к веб-сервису


Сервис доступен для использования одновременно по двум протоколам: SOAP
 и JSON-RPC. Используемый протокол указывается с помощью параметра Content-Type
 в заголовке http-запроса:


	- application/json. Используется
	 протокол JSON-RPC.


	- text/xml. Используется
	 протокол SOAP.


На базе веб-сервиса PP.SOM организована работа BI-сервера продукта «Форсайт. Аналитическая платформа».
 Адрес для работы с BI-сервером имеет следующий формат:


http://<ServerName | IP-address>[:<Port>][/<VirtualCatalog>]/axis2/services/PP.SOM.Som


Примечание.
 На x64 компьютерах наименование виртуального каталога может содержать
 суффикс - x64.


Для установки BI-сервера должен использоваться [соответствующий
 дистрибутив](Setup.chm::/02_AK_Install/DistributionKit.htm). Более подробно читайте в разделе «[Установка BI-сервера
 продукта «Форсайт. Аналитическая платформа»](Setup.chm::/UiWebSetup/UiWebSetup_TitlePage.htm).


Список операций и их параметров можно получить из сгенерированного wsdl-файла,
 доступного по адресу:


http://<BIServer>/axis2/services/PP.SOM.Som?wsdl


Все JSON-структуры аналогичны XML-структурам, используемым при работе
 с протоколом SOAP.


Важно. Следующие
 части ссылок являются регистрозависимыми и должны быть указаны в соответствующем
 виде:

1) services/PP.SOM.Som.

2) wsdl.
