# Параметры строки адреса консоли веб-сервиса


После установки консоли по умолчанию адрес для доступа будет иметь следующий
 вид: http://localhost/FP_App_v10.x/Wsc/Index.html. Основные
 функции по управлению осуществляются с помощью различных элементов управления,
 доступных в интерфейсе. Также существует ряд дополнительных параметров,
 которые могут быть указаны в строке адреса. Данные параметры отделяются
 от основного адреса с помощью символа «#», между собой параметры разделяются
 символом «&». Пример: http://localhost/FP_App_v10.x/Wsc/Index.html#locale=en.


		 Параметр
		 Описание


		 format
		 Формат представления запроса/ответа.

Допустимые значения:


			- json

			- xml


		 locale
		 Язык, на котором отображается интерфейс консоли и справка по
		 операциям/полям.

Допустимые значения:


			- ru

			- en
