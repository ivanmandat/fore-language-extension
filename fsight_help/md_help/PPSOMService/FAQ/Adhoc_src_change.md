# Как изменить на сервере источник данных для визуализатора аналитической
 панели?


Ниже приведен пример шаблон JSON-запроса с использованием операции [SetEaxMd](../Operations/ExpressReport/SetEaxMd.htm) для изменения
 источника данных для визуализатора аналитической панели. В запросе передается
 моникёры открытого репозитория, аналитической панели и ключ визуализатора.
 Результатом работы операции является ключ измененного визуализатора.


### JSON-запрос:


{

    {

    "SetEaxMd": {

        "tEax": {

            "id": "Моникер_Соединения!M!S!Моникер_Аналитической_Панели!DSO!Ключ_Визуализатора"

        },

        "tArg": {

            "pattern": {

                "dataSources": "Set"

            },

            "meta": {

                "dataSources": {

                    "its": {

                        "it": [{

                            "k": 0,

                            "vis": true,

                            "cube": {

                                "obDesc": {

                                    "n": "",

                                    "i": "",

                                    "k": Ключ_Источника_Визуализатора,

                                    "c": Ключ_Источника_Визуализатора

                                }

                            }

                        }]

                    }

                }

            }

        }

    }

}


### JSON-ответ:


{

  "SetEaxMdResult":

   {

    "id":

     {

      "id": "Моникер_Соединения!M!S!Моникер_Аналитической_Панели!DSO!Ключ_Визуализатора"

     }

   }

}
