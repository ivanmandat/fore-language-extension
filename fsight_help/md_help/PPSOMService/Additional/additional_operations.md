# Дополнительные операции


Кроме [общего списка операций](../Operations/BaseService.htm)
 также имеется ряд операций, которые не используют JSON/SOAP структуры,
 а выполняются при указании их в адресной строке BI-сервера. Список таких
 операций приведен в таблице ниже:


		 Операция
		 Краткое описание


		 [CustomRequestHandler](CustomRequestHandler.htm)
		 Операция CustomRequestHandler
		 предназначена для выполнения Fore-функции с помощью POST-запроса.


		 [GetBin](GetBin.htm)
		 Операция GetBin предназначена
		 для извлечения бинарных данных с сервера с помощью GET-запроса.


		 [GetBinFont](GetBinFont.htm)
		 Операция GetBinFont
		 предназначена для загрузки шрифта с указанным наименованием с
		 BI-сервера на клиентский компьютер с помощью GET-запроса.


		 [InsertPrxPicture](InsertPrxPicture.htm)
		 Операция InsertPrxPicture
		 предназначена для вставки изображения на лист регламентного отчёта
		 с помощью POST-запроса.


		 [LoadTopobase](LoadTopobase.htm)
		 Операция LoadTopobase
		 предназначена для загрузки топоосновы из файла в репозиторий с
		 помощью POST-запроса.


		 [PutBin](PutBin.htm)
		 Операция PutBin предназначена
		 для загрузки бинарных данных на сервер с помощью POST-запроса.


		 [ReloadTopobase](ReloadTopobase.htm)
		 Операция ReloadTopobase
		 предназначена для загрузки топоосновы из файла в существующую
		 топооснову в репозитории с помощью POST-запроса.


		 [SaveTopobase](SaveTopobase.htm)
		 Операция SaveTopobase
		 предназначена для получения топоосновы из репозитория с помощью
		 GET-запроса и сохранения её в файл.
