# GetBinFont


## Синтаксис


[<адрес
 веб-сервиса>](../Intro/PP7Service_Using.htm)/PP.GetBinFont?FileName=<наименование шрифта>


## Описание


Операция GetBinFont предназначена
 для загрузки шрифта с указанным наименованием с BI-сервера на клиентский
 компьютер с помощью GET-запроса.


## Комментарии


Операция имеет единственный параметр fileName,
 в качестве значения которого передаётся наименование загружаемого шрифта.
 Список шрифтов, которые могут быть загружены с BI-сервера, возвращает
 операция [GetFonts](../Operations/Repository/GetFonts.htm).
 В результате операции [GetFonts](../Operations/Repository/GetFonts.htm)
 в поле url можно получить полный
 добавочный адрес для загрузки какого-либо шрифта.
