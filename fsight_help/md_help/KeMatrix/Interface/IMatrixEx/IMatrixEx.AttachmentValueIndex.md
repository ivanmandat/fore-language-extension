# IMatrixEx.AttachmentValueIndex


## Синтаксис


AttachmentValueIndex: Integer;


## Описание


Свойство AttachmentValueIndex
 возвращает индекс вложения.


## Комментарии


Индекс используется для передачи в свойство [IMatrixIterator.Values](../IMatrixIterator/IMatrixIterator.Values.htm)
 для получения ключа элемента справочника НСИ с вложениями матрицы.
