# IMatrixEx.PreserveOriginalValues


## Синтаксис


PreserveOriginalValues: Boolean;


## Описание


Свойство PreserveOriginalValues
 определяет признак хранения исходных значений матрицы.


## Комментарии


Допустимые значения:


	- True. Исходные значения
	 будут сохраняться. Для получения информации об исходных значениях
	 используйте свойство [IMatrixEx.ChangesInfo](IMatrixEx.ChangesInfo.htm);


	- False. Исходные значения
	 сохраняться не будут.
