# IMatrixEx.ScaleValueIndex


## Синтаксис


ScaleValueIndex: Integer;


## Описание


Свойство ScaleValueIndex возвращает
 индекс масштаба.


## Комментарии


Индекс используется для передачи в свойство [IMatrixIterator.Values](../IMatrixIterator/IMatrixIterator.Values.htm)
 для получения величины коэффициента масштаба матрицы.
