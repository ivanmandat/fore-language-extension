# IMatrixEx


Сборка: Matrix;


## Описание


Интерфейс IMatrixEx используется
 для работы с расширенными параметрами матрицы данных.


## Иерархия наследования


           IMatrixEx


## Комментарии


Для получения объекта данного типа необходимо объект типа [IMatrix](../IMatrix/IMatrix.htm) привести к IMatrixEx.
