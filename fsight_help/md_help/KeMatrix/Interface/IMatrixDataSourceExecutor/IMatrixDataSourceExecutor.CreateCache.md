# IMatrixDataSourceExecutor.CreateCache


## Синтаксис


CreateCache: [IMatrixDataSourceExecutorCache](../IMatrixDataSourceExecutorCache/IMatrixDataSourceExecutorCache.htm);


## Описание


Метод CreateCache создает кеш
 данных для источника.


## Комментарии


Кеш данных существует пока источник открыт. Для кеширования данных необходимо
 выполнить метод [Execute](../IMatrixDataSourceExecutorCache/IMatrixDataSourceExecutorCache.Execute.htm).
