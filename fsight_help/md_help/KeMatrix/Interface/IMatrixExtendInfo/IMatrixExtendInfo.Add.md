# IMatrixExtendInfo.Add


## Синтаксис


Add(Value: [IDimSelection](KeDims.chm::/interface/IDimSelection/IDimSelection.htm));


## Параметры


Value. Отметка элементов.


## Описание


Метод Add добавляет измерение матрицы.


## Комментарии


Если добавление прошло успешно, то значение [IMatrixExtendInfo.Count](IMatrixExtendInfo.Count.htm) увеличивается на единицу.
