# IMatrixExtendInfo.Item


## Синтаксис


Item(Index: Integer): [IDimSelection](KeDims.chm::/interface/IDimSelection/IDimSelection.htm);


## Параметры


Index. Индекс измерения.


## Описание


Свойство предназначено только для чтения.


Свойство Item возвращает указанное измерение.


## Комментарии


Индексация измерений сквозная и начинается с нуля. Общее число измерений возвращает свойство [IMatrixExtendInfo.Count](IMatrixExtendInfo.Count.htm).
