# IMatrixTransformer.Source


## Синтаксис


Source: [IMatrix](../IMatrix/IMatrix.htm);


## Описание


Свойство Source определяет матрицу,
 являющуюся источником данных для матричного преобразователя.


## Комментарии


Для получения коллекции временных измерений матричного преобразователя
 используйте [IMatrixTransformer.Dimensions](IMatrixTransformer.Dimensions.htm).
