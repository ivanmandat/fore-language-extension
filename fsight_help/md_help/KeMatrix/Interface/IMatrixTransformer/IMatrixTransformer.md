# IMatrixTransformer


Сборка: Matrix;


## Описание


Интерфейс IMatrixTransformer
 используется для работы с матричным преобразователем.


## Иерархия наследования


IMatrixTransformer


## Комментарии


Матричный преобразователь представляет собой объект, который преобразует
 исходную матрицу по отметкам измерений в новую матрицу с временными измерениями.


Для работы с матричным преобразователем его необходимо создать.
