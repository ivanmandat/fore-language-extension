# IMatrixQuery


Сборка: Matrix;


## Описание


Интерфейс IMatrixQuery предназначен
 для получения информации о данных матрицы.


## Иерархия наследования


IMatrixQuery


## Комментарии


Для создания запроса используйте метод [IMatrixFactory.CreateQuery](../IMatrixFactory/IMatrixFactory.CreateQuery.htm).
