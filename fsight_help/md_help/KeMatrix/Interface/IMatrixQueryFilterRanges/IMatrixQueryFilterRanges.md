# IMatrixQueryFilterRanges


Сборка: Matrix;


## Описание


Интерфейс IMatrixQueryFilterRanges
 предназначен для работы с коллекцией диапазонов фильтрации матрицы.


## Иерархия наследования


           IMatrixQueryFilterRanges


## Комментарии


Для получения коллекции используйте свойство [IMatrixQueryFilter.Ranges](../IMatrixQueryFilter/IMatrixQueryFilter.Ranges.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMatrixQueryFilterRanges.Count.htm)
		 Свойство Count возвращает
		 количество диапазонов фильтрации в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMatrixQueryFilterRanges.Item.htm)
		 Свойство Item возвращает
		 заданный диапазон фильтрации данных матрицы.
