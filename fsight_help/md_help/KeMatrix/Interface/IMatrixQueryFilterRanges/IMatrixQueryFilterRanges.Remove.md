# IMatrixQueryFilterRanges.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс диапазона коллекции.


## Описание


Метод Remove удаляет из коллекции
 указанный диапазон фильтрации.


## Комментарии


Метод возвращает значение True,
 если удаление диапазона фильтрации завершилось успешно.
