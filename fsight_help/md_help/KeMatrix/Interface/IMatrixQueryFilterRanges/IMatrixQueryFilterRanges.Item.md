# IMatrixQueryFilterRanges.Item


## Синтаксис


Item(Index: Integer): [IMatrixQueryFilterRange](../IMatrixQueryFilterRange/IMatrixQueryFilterRange.htm);


## Параметры


Index. Индекс диапазона коллекции.


## Описание


Свойство Item возвращает заданный
 диапазон фильтрации данных матрицы.


## Комментарии


Для добавления нового диапазона в коллекцию используйте метод [IMatrixQueryFilterRanges.Add](IMatrixQueryFilterRanges.Add.htm).
