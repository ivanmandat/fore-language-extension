# IMatrixAggregatorModelHierarchies.FindById


## Синтаксис


FindById(HierarchyId: String): [IMatrixAggregatorModel](../IMatrixAggregatorModel/IMatrixAggregatorModel.htm);


## Параметры


HierarchyId. Идентификатор
 механизма/метода агрегации по уровням альтернативной иерархии.


## Описание


Метод FindById осуществляет
 поиск параметров механизма/метода агрегации по уровням альтернативной
 иерархии в коллекции альтернативных иерархий по указанному идентификатору.


## Комментарии


Для получения идентификатора механизма/метода агрегации по уровням альтернативной
 иерархии в коллекции альтернативных иерархий используйте метод [IMatrixAggregatorModel.HierarchyId](../IMatrixAggregatorModel/IMatrixAggregatorModel.HierarchyId.htm).


Коллекция альтернативных иерархий содержится в папке «[Альтернативные иерархии](UiNavObj.chm::/Cube/CreateCube/Master_Standart/UiMd_Cube_CreateCube_Master_Standart_7.htm#alt_agr)» с доступными для справочника
 уровнями альтернативной иерархии.
