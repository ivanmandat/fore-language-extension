# IMatrixAggregatorModelHierarchies


## Описание


Интерфейс IMatrixAggregatorModelHierarchies
 содержит свойства и методы для работы с агрегацией по уровням [альтернативной
 иерархии](UiNavObj.chm::/Cube/CreateCube/Master_Standart/UiMd_Cube_CreateCube_Master_Standart_7.htm#alt_agr).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMatrixAggregatorModelHierarchies.Count.htm)
		 Свойство Count возвращает
		 количество существующих механизмов/методов агрегации по уровням
		 альтернативной иерархии.


		 ![](../../Property_Image.gif)
		 [Item](IMatrixAggregatorModelHierarchies.Item.htm)
		 Свойство Item возвращает
		 параметры указанного механизма/метода агрегации по уровням альтернативной
		 иерархии.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IMatrixAggregatorModelHierarchies.Add.htm)
		 Метод Add осуществляет
		 добавление новых параметров агрегации альтернативной иерархии
		 справочника по ключу.


		 ![](../../Sub_Image.gif)
		 [AddValue](IMatrixAggregatorModelHierarchies.AddValue.htm)
		 Метод AddValue осуществляет
		 добавление существующих параметров агрегации альтернативной иерархии
		 справочника по ключу.


		 ![](../../Sub_Image.gif)
		 [Clear](IMatrixAggregatorModelHierarchies.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех параметров агрегации по уровням альтернативной иерархии.


		 ![](../../Sub_Image.gif)
		 [FindById](IMatrixAggregatorModelHierarchies.FindById.htm)
		 Метод FindById осуществляет
		 поиск параметров механизма/метода агрегации по уровням альтернативной
		 иерархии в коллекции альтернативных иерархий по указанному идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IMatrixAggregatorModelHierarchies.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск параметров механизма/метода агрегации по уровням альтернативной
		 иерархии в коллекции альтернативных иерархий по указанному ключу.


		 ![](../../Sub_Image.gif)
		 [RemoveById](IMatrixAggregatorModelHierarchies.RemoveById.htm)
		 Метод RemoveById осуществляет
		 удаление параметров механизма/метода агрегации по уровням альтернативной
		 иерархии в коллекции альтернативных иерархий по указанному идентификатору.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IMatrixAggregatorModelHierarchies.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление параметров агрегации по уровням альтернативной иерархии
		 по указанному ключу.
