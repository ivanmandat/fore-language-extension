# IMatrixAggregatorModelHierarchies.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 существующих механизмов/методов агрегации по уровням альтернативной иерархии.


## Комментарии


Свойство доступно только для чтения.


Существующие механизмы/методы агрегации по уровням [альтернативной
 иерархии](UiNavObj.chm::/Cube/CreateCube/Master_Standart/UiMd_Cube_CreateCube_Master_Standart_7.htm#alt_agr) содержатся в папке «Альтернативные
 иерархии» внутри справочников.


Для добавления параметров альтернативной иерархии справочника по ключу
 используйте метод [IMatrixAggregatorModelHierarchies.Add](IMatrixAggregatorModelHierarchies.Add.htm)
 или [IMatrixAggregatorModelHierarchies.AddValue](IMatrixAggregatorModelHierarchies.AddValue.htm).
