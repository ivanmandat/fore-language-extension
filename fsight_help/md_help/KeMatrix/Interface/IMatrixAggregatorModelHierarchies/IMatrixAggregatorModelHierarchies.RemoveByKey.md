# IMatrixAggregatorModelHierarchies.RemoveByKey


## Синтаксис


RemoveByKey(HierarchyKey: Integer): Boolean;


## Параметры


HierarchyKey. Ключ механизма/метода
 агрегации по уровням альтернативной иерархии.


## Описание


Метод RemoveByKey осуществляет
 удаление параметров механизма/метода агрегации по уровням альтернативной
 иерархии в коллекции альтернативных иерархий по указанному ключу.


## Комментарии


Для получения ключа механизма/метода агрегации по уровням альтернативной
 иерархии в коллекции альтернативных иерархий используйте метод [IMatrixAggregatorModel.HierarchyKey](../IMatrixAggregatorModel/IMatrixAggregatorModel.HierarchyKey.htm).


Коллекция альтернативных иерархий содержится в папке «[Альтернативные иерархии](UiNavObj.chm::/Cube/CreateCube/Master_Standart/UiMd_Cube_CreateCube_Master_Standart_7.htm#alt_agr)»
 с доступными для справочника уровнями альтернативной иерархии.
