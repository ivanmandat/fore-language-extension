# IMatrixTransformerDimension.Index


## Синтаксис


Index: Integer;


## Описание


Свойство Index возвращает индекс
 временного измерения матричного преобразователя.


## Комментарии


Для определения данных измерения из коллекции временных измерений матричного
 преобразователя используйте [IMatrixTransformerDimension.Dimension](IMatrixTransformerDimension.Dimension.htm).
