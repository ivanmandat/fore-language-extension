# IMatrixTransformerDimension.MatchCase


## Синтаксис


MatchCase: Boolean;


## Описание


Свойство MatchCase определяет,
 будет ли учитываться регистр для строк.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Регистр для строк будет учитываться;


	- False. Регистр для строк
	 учитываться не будет.
