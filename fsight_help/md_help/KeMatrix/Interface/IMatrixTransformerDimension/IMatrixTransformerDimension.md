# IMatrixTransformerDimension


Сборка: Matrix;


## Описание


Интерфейс IMatrixTransformerDimension
 используется для работы с временным измерением матричного преобразователя.


## Иерархия наследования


IMatrixTransformerDimension


## Комментарии


Для получения объекта данного типа используйте [IMatrixTransformerDimensions.Item](../IMatrixTransformerDimensions/IMatrixTransformerDimensions.Item.htm).
