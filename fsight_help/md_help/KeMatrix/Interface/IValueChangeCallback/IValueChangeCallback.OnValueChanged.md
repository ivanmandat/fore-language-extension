# IValueChangeCallback.OnValueChanged


## Синтаксис


OnValueChanged(Value: [IMatrixCoord](../IMatrixCoord/IMatrixCoord.htm));


## Параметры


Value. Координата, по которой
 изменилось значение в матрице.


## Описание


Метод OnValueChanged реализует
 событие, которое наступает при смене значения в ячейке матрицы.
