# IMatrixDataSource.ReadOnly


## Синтаксис


ReadOnly: Boolean;


## Описание


Свойство ReadOnly возвращает признак открытия источника данных только для чтения. Если свойство возвращает значение True - источник данных открыт только для чтения, иначе будет возможность изменить и сохранить данные.
