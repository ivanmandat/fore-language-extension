# IMatrixDataSource


Сборка: Matrix;


## Описание


Интерфейс IMatrixDataSource
 содержит свойства и методы объекта, представляющего собой абстрактный
 многомерный источник данных.


## Иерархия наследования


            IMatrixDataSource


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ReadOnly](IMatrixDataSource.ReadOnly.htm)
		 Свойство ReadOnly возвращает
		 признак открытия источника данных только для чтения.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CreateDimSelectionSet](IMatrixDataSource.CreateDimSelectionSet.htm)
		 Метод CreateDimSelectionSet
		 создает набор отметок для измерений источника данных.


		 ![](../../Sub_Image.gif)
		 [CreateDimSelectionSetOp](IMatrixDataSource.CreateDimSelectionSetOp.htm)
		 Метод CreateDimSelectionSetOp
		 создает набор отметок для измерений источника данных с учетом
		 дополнительных параметров.


		 ![](../../Sub_Image.gif)
		 [CreateExecutor](IMatrixDataSource.CreateExecutor.htm)
		 Метод CreateExecutor
		 создает объект для работы с данными многомерного источника данных.


		 ![](../../Sub_Image.gif)
		 [Execute](IMatrixDataSource.Execute.htm)
		 Метод Execute осуществляет
		 расчет матрицы с данными в соответствии с указанной отметкой элементов.


		 ![](../../Sub_Image.gif)
		 [SaveData](IMatrixDataSource.SaveData.htm)
		 Метод SaveData осуществляет
		 сохранение матрицы с данными в источник данных.
