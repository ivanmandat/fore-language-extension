# IMatrixFactory.CreateMatrix


## Синтаксис


CreateMatrix(Dimensions: [IDimSelectionSet](KeDims.chm::/interface/IDimSelectionSet/IDimSelectionSet.htm)):
 [IMatrix](../IMatrix/IMatrix.htm);


## Параметры


Dimensions. Отметка измерений.


## Описание


Метод CreateMatrix создает матрицу
 на основе указанной отметки измерений.


## Комментарии


Отметку измерений, на основе которой создана матрица, возвращает свойство
 [IMatrix.Dimensions](../IMatrix/IMatrix.Dimensions.htm).
