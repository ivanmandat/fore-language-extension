# IMatrixFactory


Сборка: Matrix;


## Описание


Интерфейс IMatrixFactory предназначен
 для создания матрицы.

		 Имя метода
		 Краткое описание

		 [CreateExtendInfo](IMatrixFactory.CreateExtendInfo.htm)
		 Метод CreateExtendInfo
		 создает расширенные параметры матрицы данных.


		 ![](../../Sub_Image.gif)
		 [CreateMatrix](IMatrixFactory.CreateMatrix.htm)
		 Метод CreateMatrix
		 создает матрицу на основе указанной отметки измерений.


		 ![](../../Sub_Image.gif)
		 [CreateQuery](IMatrixFactory.CreateQuery.htm)
		 Метод CreateQuery создает
		 запрос на получение информации о существующих данных в указанной
		 матрице.
