# IMatrixIterator.Assign


## Синтаксис


Assign(Iterator: [IMatrixModelIterator](ForeSys.chm::/Interface/IMatrixModelIterator/IMatrixModelIterator.htm));


## Параметры


Iterator - итератор, в соответствии с параметрами которого, будут установлены параметры текущего итератора.


## Описание


Метод Assign осуществляет установку параметров текущего итератора, в соответствии с параметрами итератора, передаваемого посредством параметра Iterator. Оба итератора должны быть созданы на базе одной матрицы.
