# IMatrixTransformerDimensions.Add


## Синтаксис


Add(Index: Integer; Element: Integer): Boolean;


## Параметры


Index. Индекс временного измерения
 в коллекции;


Element. Индекс элемента
 временного измерения.


## Описание


Метод Add добавляет новое временное
 измерение в коллекцию.


## Комментарии


Если временное измерение будет добавлено в коллекцию, то метод вернет
 значение True.


Для удаления временного измерения из коллекции используйте [IMatrixTransformerDimensions.Remove](IMatrixTransformerDimensions.Remove.htm).
