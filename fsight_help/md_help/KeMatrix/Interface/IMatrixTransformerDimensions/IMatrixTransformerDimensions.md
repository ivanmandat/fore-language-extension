# IMatrixTransformerDimensions


Сборка: Matrix;


## Описание


Интерфейс IMatrixTransformerDimensions
 используется для работы с коллекцией временных измерений матричного преобразователя.


## Иерархия наследования


IMatrixTransformerDimensions


## Комментарии


Для получения объекта данного типа используйте [IMatrixTransformer.Dimensions](../IMatrixTransformer/IMatrixTransformer.Dimensions.htm).
