# IMatrixTransformerDimensions.Item


## Синтаксис


Item(Index: Integer): [IMatrixTransformerDimension](../IMatrixTransformerDimension/IMatrixTransformerDimension.htm);


## Параметры


Index. Индекс временного измерения.


## Описание


Свойство Item возвращает временное
 измерение по его индексу.


## Комментарии


Для определения количества временных измерений в коллекции используйте
 [IMatrixTransformerDimensions.Count](IMatrixTransformerDimensions.Count.htm).
