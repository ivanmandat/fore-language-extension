# IMatrixTransformerDimensions.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс временного измерения.


## Описание


Метод Remove удаляет временное
 измерение из коллекции.


## Комментарии


Если временное измерение будет удалено, то метод вернет значение True.


Для добавления временного измерения в коллекцию используйте [IMatrixTransformerDimensions.Add](IMatrixTransformerDimensions.Add.htm).
