# IMatrixTransformerDimensions.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает
 количество временных измерений в коллекции матричного преобразователя.


## Комментарии


Для получения временного измерения по его индексу используйте [IMatrixTransformerDimensions.Item](IMatrixTransformerDimensions.Item.htm).
