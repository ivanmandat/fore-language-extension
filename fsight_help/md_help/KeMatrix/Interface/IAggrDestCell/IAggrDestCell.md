# IAggrDestCell


Сборка: Matrix;


## Описание


Интерфейс IAggrDestCell используется
 для работы с конкретным агрегированным элементом.


## Иерархия наследования


           [IAggrSourceCell](../IAggrSourceCell/IAggrSourceCell.htm)


           IAggrDestCell


## Комментарии


Для получения агрегированного элемента используйте свойство [IAggregationInfoItem.Item](../IAggregationInfoItem/IAggregationInfoItem.Item.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IAggrDestCell.Count.htm)
		 Свойство Count возвращает
		 количество исходных элементов, которые использовались при расчёте
		 агрегированного элемента.


		 ![](../../Property_Image.gif)
		 [Item](IAggrDestCell.Item.htm)
		 Свойство Item возвращает
		 исходный элемент, который использовался при расчёте агрегированного
		 элемента.


		 ![](../../Property_Image.gif)
		 [Operation](IAggrDestCell.Operation.htm)
		 Свойство Operation
		 возвращает метод расчёта основного механизма агрегации, который
		 использовался при расчёте агрегированного элемента.
