# IAggrDestCell.Operation


## Синтаксис


Operation: [BasicAggregatorOperation](../../Enums/BasicAggregatorOperation.htm);


## Описание


Свойство Operation возвращает
 метод расчёта основного механизма агрегации, который использовался при
 расчёте агрегированного элемента.


## Комментарии


Для определения метода расчёта основного механизма агрегации используйте
 свойство [IBasicMatrixLevelAggregator.Operation](../IBasicMatrixLevelAggregator/IBasicMatrixLevelAggregator.Operation.htm).
