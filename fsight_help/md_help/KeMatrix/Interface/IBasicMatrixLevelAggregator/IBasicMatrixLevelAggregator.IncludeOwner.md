# IBasicMatrixLevelAggregator.IncludeOwner


## Синтаксис


IncludeOwner: Boolean;


## Описание


Свойство IncludeOwner определяет
 признак учёта значения на уровне-приёмнике при занесении на него агрегированных
 данных.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 При занесении агрегированных данных на уровень-приёмник будет учитываться
	 значение на этом уровне;


	- False. При занесении
	 агрегированных данных на уровень-приёмник значение на этом уровне
	 учитываться не будет.
