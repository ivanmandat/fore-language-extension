# IBasicMatrixLevelAggregator.IncludeChildren


## Синтаксис


IncludeChildren: Boolean;


## Описание


Свойство IncludeChildren определяет
 признак использования значений непосредственных дочерних элементов при
 агрегации.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 При агрегации будут использоваться значения непосредственных дочерних
	 элементов;


	- False. При агрегации
	 будут использоваться значения элементов уровней, указанных в свойстве
	 [IBasicMatrixLevelAggregator.Include](IBasicMatrixLevelAggregator.Include.htm).
	 Свойство [IBasicMatrixLevelAggregator.Include](IBasicMatrixLevelAggregator.Include.htm)
	 должно принимать значение True.
