# IBasicMatrixLevelAggregator.UseDefaultLevelAggregation


## Синтаксис


UseDefaultLevelAggregation: Boolean;


## Описание


Свойство UseDefaultLevelAggregation
 определяет признак использования настроек агрегации для уровня по умолчанию.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Для агрегации используются параметры, установленные в свойстве [IBasicMatrixAggregator.DefaultLevelAggregation](../IBasicMatrixAggregator/IBasicMatrixAggregator.DefaultLevelAggregation.htm);


	- False. Для агрегации
	 требуется определить необходимые параметры, используя соответствующие
	 свойства.
