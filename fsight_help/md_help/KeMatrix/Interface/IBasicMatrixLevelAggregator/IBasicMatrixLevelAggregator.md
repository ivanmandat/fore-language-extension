# IBasicMatrixLevelAggregator


## Описание


Интерфейс IBasicMatrixLevelAggregator
 содержит свойства, определяющие параметры основного механизма агрегации
 по заданному уровню измерения.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Include](IBasicMatrixLevelAggregator.Include.htm)
		 Свойство Include определяет
		 признак использования данных указанного уровня при агрегации.


		 ![](../../Property_Image.gif)
		 [IncludeChildren](IBasicMatrixLevelAggregator.IncludeChildren.htm)
		 Свойство IncludeChildren
		 определяет признак использования значений непосредственных дочерних
		 элементов при агрегации.


		 ![](../../Property_Image.gif)
		 [IncludeOwner](IBasicMatrixLevelAggregator.IncludeOwner.htm)
		 Свойство IncludeOwner
		 определяет признак учёта значения на уровне-приёмнике при занесении
		 на него агрегированных данных.


		 ![](../../Property_Image.gif)
		 [Level](IBasicMatrixLevelAggregator.Level.htm)
		 Свойство Level возвращает
		 уровень измерения, для которого необходимо настроить параметры
		 агрегации.


		 ![](../../Property_Image.gif)
		 [Operation](IBasicMatrixLevelAggregator.Operation.htm)
		 Свойство Operation
		 определяет метод агрегации, применяемый для элементов выбранного
		 уровня измерения.


		 ![](../../Property_Image.gif)
		 [PreserveExistingData](IBasicMatrixLevelAggregator.PreserveExistingData.htm)
		 Свойство PreserveExistingData
		 определяет признак сохранения неагрегированных данных.


		 ![](../../Property_Image.gif)
		 [PreserveMethod](IBasicMatrixLevelAggregator.PreserveMethod.htm)
		 Свойство PreserveMethod
		 определяет способ сохранения неагрегированных данных.


		 ![](../../Property_Image.gif)
		 [UseDefaultLevelAggregation](IBasicMatrixLevelAggregator.UseDefaultLevelAggregation.htm)
		 Свойство UseDefaultLevelAggregation
		 определяет признак использования настроек агрегации для уровня
		 по умолчанию.
