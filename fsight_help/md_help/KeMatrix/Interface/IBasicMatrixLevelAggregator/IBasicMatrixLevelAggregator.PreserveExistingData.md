# IBasicMatrixLevelAggregator.PreserveExistingData


## Синтаксис


PreserveExistingData: Boolean;


## Описание


Свойство PreserveExistingData
 определяет признак сохранения неагрегированных данных.


## Комментарии


Допустимые значения:


	- True. Неагрегированные
	 данные будут сохраняться;


	- False. По умолчанию.
	 Неагрегированные данные сохраняться не будут.
