# IMatrixWritableProxy.BaseMatrix


## Синтаксис


BaseMatrix: [IMatrix](../IMatrix/IMatrix.htm);


## Описание


Свойство BaseMatrix возвращает
 матрицу исходных данных объекта-посредника.


## Комментарии


Для получения матрицы изменённых данных объекта-посредника используйте
 [IMatrixWritableProxy.ChangeMatrix](IMatrixWritableProxy.ChangeMatrix.htm).
