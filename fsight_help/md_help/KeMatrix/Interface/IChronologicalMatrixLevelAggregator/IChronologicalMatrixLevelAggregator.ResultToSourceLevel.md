# IChronologicalMatrixLevelAggregator.ResultToSourceLevel


## Синтаксис


ResultToSourceLevel: Boolean;


## Описание


Свойство ResultToSourceLevel определяет, будут ли результаты агрегации сохраняться на уровень-источник. По умолчанию свойству установлено значение True.
