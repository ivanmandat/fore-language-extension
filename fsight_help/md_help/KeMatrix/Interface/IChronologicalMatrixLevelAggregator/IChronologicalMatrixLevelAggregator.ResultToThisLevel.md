# IChronologicalMatrixLevelAggregator.ResultToThisLevel


## Синтаксис


ResultToThisLevel: Boolean;


## Описание


Свойство ResultToThisLevel определяет, будут ли результаты агрегации сохраняться на данном уровне. По умолчанию свойству установлено значение True. Данное свойство доступно только через Fore.
