# IChronologicalMatrixLevelAggregator


## Описание


Интерфейс IChronologicalMatrixLevelAggregator
 содержит свойства, определяющие параметры хронологического механизма агрегации
 по заданному уровню.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [IncludeChildren](IChronologicalMatrixLevelAggregator.IncludeChildren.htm)
		 Свойство IncludeChildren
		 определяет, будут ли при агрегации использоваться значения непосредственных
		 дочерних элементов.


		 ![](../../Property_Image.gif)
		 [Level](IChronologicalMatrixLevelAggregator.Level.htm)
		 Свойство Level возвращает
		 уровень измерения, для которого необходимо настроить параметры
		 агрегации.


		 ![](../../Property_Image.gif)
		 [Operation](IChronologicalMatrixLevelAggregator.Operation.htm)
		 Свойство Operation
		 определяет метод агрегации, применяемый для элементов выбранного
		 уровня измерения.


		 ![](../../Property_Image.gif)
		 [ResultToSourceLevel](IChronologicalMatrixLevelAggregator.ResultToSourceLevel.htm)
		 Свойство ResultToSourceLevel
		 определяет, будут ли результаты агрегации сохраняться на уровень-источник.


		 ![](../../Property_Image.gif)
		 [ResultToThisLevel](IChronologicalMatrixLevelAggregator.ResultToThisLevel.htm)
		 Свойство ResultToThisLevel
		 определяет, будут ли результаты агрегации сохраняться на данном
		 уровне.


		 ![](../../Property_Image.gif)
		 [SourceLevel](IChronologicalMatrixLevelAggregator.SourceLevel.htm)
		 Свойство SourceLevel
		 определяет уровень измерения, с которого будут браться данные
		 для агрегации.


		 ![](../../Property_Image.gif)
		 [UseDefaultLevelAggregation](IChronologicalMatrixLevelAggregator.UseDefaultLevelAggregation.htm)
		 Свойство UseDefaultLevelAggregation
		 определяет, будут ли использоваться настройки агрегации для уровня
		 по умолчанию.
