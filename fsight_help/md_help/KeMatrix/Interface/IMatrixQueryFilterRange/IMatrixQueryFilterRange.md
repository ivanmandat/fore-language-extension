# IMatrixQueryFilterRange


Сборка: Matrix;


## Описание


Интерфейс IMatrixQueryFilterRange
 предназначен для работы с диапазоном фильтрации запросов матрицы.


## Иерархия наследования


           IMatrixQueryFilterRange


## Комментарии


Для получения диапазона фильтрации используйте свойство [IMatrixQueryFilterRanges.Item](../IMatrixQueryFilterRanges/IMatrixQueryFilterRanges.Item.htm).
