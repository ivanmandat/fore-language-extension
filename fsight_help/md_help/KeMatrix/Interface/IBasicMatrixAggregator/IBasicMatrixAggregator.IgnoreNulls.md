# IBasicMatrixAggregator.IgnoreNulls


## Синтаксис


IgnoreNulls: [IgnoreNullsState](../../Enums/IgnoreNullsState.htm);


## Описание


Свойство IgnoreNulls определяет
 способ обработки пустых значений при агрегации данных.


## Комментарии


По умолчанию свойству установлено значение [IgnoreNullsState.IgnoreAllNulls](../../Enums/IgnoreNullsState.htm).
