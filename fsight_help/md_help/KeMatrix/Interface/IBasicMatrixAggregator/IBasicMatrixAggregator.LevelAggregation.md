# IBasicMatrixAggregator.LevelAggregation


## Синтаксис


LevelAggregation(Level: [IDimLevel](KeDims.chm::/interface/IDimLevel/IDimLevel.htm)):
 [IBasicMatrixLevelAggregator](../IBasicMatrixLevelAggregator/IBasicMatrixLevelAggregator.htm);


## Параметры


Level - уровень измерения,
 для которого необходимо настроить параметры агрегации.


## Описание


Свойство LevelAggregation возвращает
 параметры агрегации для заданного уровня измерения.
