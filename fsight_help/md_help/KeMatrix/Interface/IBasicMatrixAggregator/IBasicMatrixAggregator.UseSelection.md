# IBasicMatrixAggregator.UseSelection


## Синтаксис


UseSelection: Boolean;


## Описание


Свойство UseSelection определяет,
 будут ли при агрегации использоваться данные отмеченных элементов.


## Комментарии


Если свойству установлено значение True,
 то при агрегации будут использоваться значения только отмеченных элементов,
 иначе значения всех элементов указанных уровней. По умолчанию свойству
 установлено значение False.
