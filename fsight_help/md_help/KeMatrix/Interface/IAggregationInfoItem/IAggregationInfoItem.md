# IAggregationInfoItem


Сборка: Matrix;


## Описание


Интерфейс IAggregationInfoItem
 используется для получения информации об агрегированных элементах.


## Иерархия наследования


           IAggregationInfoItem


## Комментарии


Для получения информации об агрегации используйте свойство [IAggregationInfo.Item](../IAggregationInfo/IAggregationInfo.Item.htm).
