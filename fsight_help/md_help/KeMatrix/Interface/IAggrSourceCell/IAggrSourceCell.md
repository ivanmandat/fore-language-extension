# IAggrSourceCell


Сборка: Matrix;


## Описание


Интерфейс IAggrSourceCell используется
 для работы с конкретным исходным элементом, который использовался при
 расчёте агрегированного элемента.


## Иерархия наследования


           IAggrSourceCell


## Комментарии


Для получения исходного элемента используйте свойство [IAggrDestCell.Item](../IAggrDestCell/IAggrDestCell.Item.htm).
