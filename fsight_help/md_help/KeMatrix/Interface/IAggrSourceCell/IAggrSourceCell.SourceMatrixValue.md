# IAggrSourceCell.SourceMatrixValue


## Синтаксис


SourceMatrixValue: Boolean;


## Описание


Свойство SourceMatrixValue возвращает
 признак получения значения элемента.


## Комментарии


Допустимые значения:


	- True. Значение элемента
	 из исходной матрицы;


	- False. Значение элемента
	 из агрегированной матрицы.
