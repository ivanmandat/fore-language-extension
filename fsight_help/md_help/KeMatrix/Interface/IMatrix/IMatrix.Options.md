# IMatrix.Options


## Синтаксис


Options: Integer;


## Описание


Свойство Options возвращает
 значение флага, используемого для определения значения [масштабирования](UiNavObj.chm::/Units/Units.htm).


## Комментарии


Если в свойстве Options матрицы
 стоит флаг, то в последнее значение матрицы будет записано значение масштабирования
 (Scale).
