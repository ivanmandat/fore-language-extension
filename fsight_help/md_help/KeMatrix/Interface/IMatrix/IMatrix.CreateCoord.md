# IMatrix.CreateCoord


## Синтаксис


CreateCoord: [IMatrixCoord](../IMatrixCoord/IMatrixCoord.htm);


## Описание


Метод CreateCoord создает объект, представляющий собой координату в матрице.


## Комментарии


Для создания объекта, содержащего информацию, необходимую для перестановки измерений с данными матрицы, используйте метод [IMatrix.CreateSwapInfo](IMatrix.CreateSwapInfo.htm).
