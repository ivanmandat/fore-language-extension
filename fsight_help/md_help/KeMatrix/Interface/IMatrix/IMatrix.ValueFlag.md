# IMatrix.ValueFlag


## Синтаксис


ValueFlag: Integer;


## Описание


Свойство ValueFlag определяет
 значение флага, используемого в матрице для отметки изменившихся значений.


## Комментарии


Для изменения флага используйте метод [IMatrix.ChangeValueFlag](IMatrix.ChangeValueFlag.htm).
