# IMatrix.CountV


## Синтаксис


CountV(ValueFlag: Integer): Integer;


## Параметры


ValueFlag. Флаг, которым отмечены
 значения.


## Описание


Свойство CountV возвращает количество
 значений, отмеченных указанным флагом.
