# IMatrix.SetValueBySelection


## Синтаксис


SetValueBySelection(Selection: [IDimSelectionSet](KeDims.chm::/interface/IDimSelectionSet/IDimSelectionSet.htm);
 Value: Variant);


## Параметры


Selection. Отметка, задающая
 изменяемый элемент;


Value. Новое значение элемента.


## Описание


Метод SetValueBySelection
 устанавливает значение для элемента, заданного отметкой.


## Комментарии


Значение элемента может быть строковым или числовым.
