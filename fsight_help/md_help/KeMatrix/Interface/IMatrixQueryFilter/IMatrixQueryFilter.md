# IMatrixQueryFilter


Сборка: Matrix;


## Описание


Интерфейс IMatrixQueryFilter
 предназначен для настройки фильтрации матрицы.


## Иерархия наследования


           IMatrixQueryFilter


## Комментарии


Для получения параметров фильтрации запросов матрицы используйте свойство
 [IMatrixQuery.Filter](../IMatrixQuery/IMatrixQuery.Filter.htm).
