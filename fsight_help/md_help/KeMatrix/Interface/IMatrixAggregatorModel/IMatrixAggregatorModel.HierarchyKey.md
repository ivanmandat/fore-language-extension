# IMatrixAggregatorModel.HierarchyKey


## Синтаксис


HierarchyKey: Integer;


## Описание


Свойство HierarchyKey определяет
 ключ агрегации по уровням [альтернативной
 иерархии](UiNavObj.chm::/Cube/CreateCube/Master_Standart/UiMd_Cube_CreateCube_Master_Standart_7.htm#alt_agr) в коллекции альтернативных иерархий справочника.


## Комментарии


Используется для поиска или удаления параметров агрегации по уровням
 альтернативной иерархии с помощью методов: [IMatrixAggregatorModelHierarchies.FindByKey](../IMatrixAggregatorModelHierarchies/IMatrixAggregatorModelHierarchies.FindByKey.htm),
 [IMatrixAggregatorModelHierarchies.RemoveByKey](../IMatrixAggregatorModelHierarchies/IMatrixAggregatorModelHierarchies.RemoveByKey.htm).


Коллекция альтернативных иерархий содержится в папке «[Альтернативные иерархии](UiNavObj.chm::/Cube/CreateCube/Master_Standart/UiMd_Cube_CreateCube_Master_Standart_7.htm#alt_agr)»
 с доступными для справочника уровнями альтернативной иерархии.
