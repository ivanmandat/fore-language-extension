# IMatrixCoord


## Описание


Интерфейс IMatrixCoord содержит
 свойства и методы для работы с координатой элемента в матрице.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ElementId](IMatrixCoord.ElementId.htm)
		 Свойство ElementId
		 определяет идентификатор элемента измерения, которому соответствует
		 координата в матрице.


		 ![](../../Property_Image.gif)
		 [Matrix](IMatrixCoord.Matrix.htm)
		 Свойство Matrix возвращает
		 исходную матрицу.


## Свойства, унаследованные от [IMatrixModelCoord](ForeSys.chm::/Interface/IMatrixModelCoord/IMatrixModelCoord.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ForeSys.chm::/Interface/IMatrixModelCoord/IMatrixModelCoord.Count.htm)
		 Свойство Count возвращает
		 количество параметров координаты.


		 ![](../../Property_Image.gif)
		 [Item](ForeSys.chm::/Interface/IMatrixModelCoord/IMatrixModelCoord.Item.htm)
		 Свойство Item определяет,
		 для указанного параметра координаты, индекс элемента в измерении
		 матрицы.


		 ![](../../Property_Image.gif)
		 [MatrixModel](ForeSys.chm::/Interface/IMatrixModelCoord/IMatrixModelCoord.MatrixModel.htm)

## Методы, унаследованные от [IMatrixModelCoord](ForeSys.chm::/Interface/IMatrixModelCoord/IMatrixModelCoord.htm)


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Assign](ForeSys.chm::/Interface/IMatrixModelCoord/IMatrixModelCoord.Assign.htm)
		 Метод Assign заполняет
		 параметры текущей координаты, параметрами входной координаты.


		 ![](../../Sub_Image.gif)
		 [AssignFirst](ForeSys.chm::/Interface/IMatrixModelCoord/IMatrixModelCoord.AssignFirst.htm)
		 Метод AssignFirst заполняет
		 указанное количество параметров текущей координаты, параметрами
		 входной координаты.


		 ![](../../Sub_Image.gif)
		 [PutFirst](ForeSys.chm::/Interface/IMatrixModelCoord/IMatrixModelCoord.PutFirst.htm)
		 Метод PutFirst устанавливает
		 координату первого элемента матрицы.


		 ![](../../Sub_Image.gif)
		 [PutLast](ForeSys.chm::/Interface/IMatrixModelCoord/IMatrixModelCoord.PutLast.htm)
		 Метод PutLast устанавливает
		 координату последнего элемента матрицы.


		 ![](../../Sub_Image.gif)
		 [PutLowerBound](ForeSys.chm::/Interface/IMatrixModelCoord/IMatrixModelCoord.PutLowerBound.htm)
		 Метод PutLowerBound
		 устанавливает для указанного параметра координаты, индекс первого
		 элемента соответствующего измерения матрицы.


		 ![](../../Sub_Image.gif)
		 [PutUpperBound](ForeSys.chm::/Interface/IMatrixModelCoord/IMatrixModelCoord.PutUpperBound.htm)
		 Метод PutUpperBound
		 устанавливает для указанного параметра координаты, индекс последнего
		 элемента соответствующего измерения матрицы.
