# Введение в сборку Matrix


Основными интерфейсами сборки Matrix
 являются:


	- [IBasicMatrixAggregator](Interface/IBasicMatrixAggregator/IBasicMatrixAggregator.htm)
	 - интерфейс, содержащий свойства для настройки параметров основного
	 механизма агрегации.


	- [IChronologicalMatrixAggregator](Interface/IChronologicalMatrixAggregator/IChronologicalMatrixAggregator.htm)
	 - интерфейс, содержащий свойства для настройки параметров хронологического
	 механизма агрегации.


	- [IMatrix](Interface/IMatrix/IMatrix.htm) - интерфейс,
	 содержащий свойства и методы для работы с матрицей данных.


	- [IMatrixCoord](Interface/IMatrixCoord/IMatrixCoord.htm)
	 - интерфейс, содержащий свойства и методы для работы с координатой
	 элемента в матрице.
