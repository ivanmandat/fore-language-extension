## С чего начать


		 В данном разделе
		 представлено описание сборки Matrix,
		 содержащей средства для работы с матрицей данных и настройки для
		 различных методов агрегации. Матрицы используется при работе с
		 данными [различных
		 видов кубов](KeCubes.chm::/Interface/ICubeInstanceDestination/ICubeInstanceDestination.Execute.htm), данными [переменных
		 моделирования](KeMs.chm::/Interface/IMsVariable/IMsVariable.Cube.htm).


		Более подробно с возможностями сборки можно ознакомиться в разделе
		 «[Введение в сборку Matrix](KeMatrix_Introduction.htm)».


		Описание ресурсов сборки представлено в следующих подразделах:


			- [Интерфейсы
			 сборки Matrix](Interface/KeMatrix_Interface.htm)


			- [Перечисления сборки
			 Matrix](Enums/KeMatrix_Enums.htm)


			- [Классы сборки Matrix](Class/KeMatrix_Class.htm)


		 Перед изучением сборки Matrix рекомендуется ознакомиться
		 с работой с данными [различных
		 видов кубов](KeCubes.chm::/Interface/ICubeInstanceDestination/ICubeInstanceDestination.Execute.htm), данными [переменных
		 моделирования](KeMs.chm::/Interface/IMsVariable/IMsVariable.Cube.htm).
