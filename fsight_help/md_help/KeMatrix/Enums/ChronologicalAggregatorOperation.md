# ChronologicalAggregatorOperation


## Описание


Перечисление ChronologicalAggregatorOperation используется для определения метода расчёта хронологического механизма агрегации.


Используется следующим свойством:


-
[IChronologicalMatrixLevelAggregator.Operation](../Interface/IChronologicalMatrixLevelAggregator/IChronologicalMatrixLevelAggregator.Operation.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Не осуществлять агрегацию.


 1
 Mean. Среднее хронологическое.


 2
 UniformDisaggregation. Равномерная дезагрегация.
