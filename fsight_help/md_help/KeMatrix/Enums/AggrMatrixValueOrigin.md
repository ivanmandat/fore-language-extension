# AggrMatrixValueOrigin


## Описание


Перечисление AggrMatrixValueOrigin
 используется для определения источника значения исходного элемента при
 выполнении агрегации.


Используется свойством [IAggrSourceCell.ValueOrigin](../Interface/IAggrSourceCell/IAggrSourceCell.ValueOrigin.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Source. Элемент в исходной
		 матрице.


		 1
		 Dest. Элемент в агрегированной
		 матрице.


		 2
		 ExtraSource. Дополнительный
		 элемент в исходной матрице.


		 3
		 ExtraDest. Дополнительный
		 элемент в агрегированной матрице.
