# ExistingDataPreserveMethod


## Описание


Перечисление ExistingDataPreserveMethod используется для определения способа сохранения неагрегированных данных.


Используется следующим свойством:


-
[IBasicMatrixLevelAggregator.PreserveMethod](../Interface/IBasicMatrixLevelAggregator/IBasicMatrixLevelAggregator.PreserveMethod.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Не сохранять.


 1
 Always. Сохранять всегда.


 2
 OnNull. Сохранять, если результат агрегации пустой.
