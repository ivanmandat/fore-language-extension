# GxSystemFontNames


Сборка: Drawing;


## Описание


Класс GxSystemFontNames используется
 для работы с объектом, содержащим коллекцию системных шрифтов.


## Комментарии


Для получения коллекции системных шрифтов необходимо создать новый объект
 с помощью данного класса.
