# GxThemeColors


Сборка: Drawing;


## Описание


Класс GxThemeColors реализует
 статические свойства для получения цветов различных элементов, определенных
 в теме оформления.


## Комментарии


Для изменения оформления, используемого в текущем приложении, используйте
 свойство [IWinApplicationClass.ApplicationSettings](UiLib.chm::/Interface/IWinApplicationClass/IWinApplicationClass.ApplicationSettings.htm).
