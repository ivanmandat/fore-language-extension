# GxImageList


Сборка: Drawing;


## Описание


Класс GxImageList реализует
 объект, предоставляющий возможность работы с коллекцией изображений.


## Свойства объекта класса, унаследованные от [IGxImageList](../../Interface/IGxImageList/IGxImageList.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](../../Interface/IGxImageList/IGxImageList.Count.htm)
		 Свойство Count возвращает
		 количество изображений в коллекции.


		 ![](../../Property_Image.gif)
		 [Height](../../Interface/IGxImageList/IGxImageList.Height.htm)
		 Свойство Height определяет
		 высоту изображений, хранящихся в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](../../Interface/IGxImageList/IGxImageList.Item.htm)
		 Свойство Item возвращает
		 графическое изображение указанной пиктограммы.


		 ![](../../Property_Image.gif)
		 [Width](../../Interface/IGxImageList/IGxImageList.Width.htm)
		 Свойство Width определяет
		 ширину изображений, хранящихся в коллекции.
