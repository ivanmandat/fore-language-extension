# GxIcon


Сборка: Drawing;


## Описание


Класс GxIcon реализует пиктограмму.


## Конструкторы


		 Имя конструктора
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CreateCopy](GxIcon.CreateCopy.htm)
		 Конструктор CreateCopy
		 создает копию пиктограммы, масштабируя ее до указанных размеров.


		 ![](../../Sub_Image.gif)
		 [CreateFromFile](GxIcon.CreateFromFile.htm)
		 Конструктор CreateFromFile
		 создает пиктограмму из указанного файла.


		 ![](../../Sub_Image.gif)
		 [CreateFromFileS](GxIcon.CreateFromFileS.htm)
		 Конструктор CreateFromFileS
		 создает пиктограмму из указанного файла, масштабируя ее до указанных
		 размеров.


		 ![](../../Sub_Image.gif)
		 [CreateFromStream](GxIcon.CreateFromStream.htm)
		 Конструктор CreateFromStream
		 создает пиктограмму из потока.


		 ![](../../Sub_Image.gif)
		 [CreateFromStreamS](GxIcon.CreateFromStreamS.htm)
		 Конструктор CreateFromStreamS
		 создает пиктограмму из потока, масштабируя ее до указанных размеров.
