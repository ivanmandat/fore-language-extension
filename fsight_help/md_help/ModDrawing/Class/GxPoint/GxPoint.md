# GxPoint


Сборка: Drawing;


## Описание


Класс GxPoint реализует точку
 с целочисленными координатами.

		 Имя конструктора
		 Краткое описание


		 ![](../../Sub_Image.gif)

		 Конструктор Create
		 создает новую точку с указанными координатами.


		 ![](../../Sub_Image.gif)
		 [CreateFromPoint](GxPoint.CreateFromPoint.htm)
		 Конструктор CreateFromPoint
		 создает новую точку на основе указанной точки.
