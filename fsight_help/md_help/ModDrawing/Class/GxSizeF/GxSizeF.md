# GxSizeF


Сборка: Drawing;


## Описание


Класс GxSizeF реализует вещественный
 размер.

		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Create](GxSizeF.Create.htm)
		 Конструктор Create
		 создает новый размер с указанными значениями ширины и высоты.


		 ![](../../Sub_Image.gif)
		 [CreateFromPoint](GxSizeF.CreateFromPoint.htm)
		 Конструктор CreateFromPoint
		 создает новый размер на основе указанной точки.


		 ![](../../Sub_Image.gif)
		 [CreateFromSize](GxSizeF.CreateFromSize.htm)
