# GxSize


Сборка: Drawing;


## Описание


Класс GxSize реализует целочисленными
 размер.


## Конструкторы


		 Имя конструктора
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Create](GxSize.Create.htm)
		 Конструктор Create
		 создает новый размер с указанными значениями ширины и высоты.


		 ![](../../Sub_Image.gif)
		 [CreateFromPoint](GxSize.CreateFromPoint.htm)
		 Конструктор CreateFromPoint
		 создает новый размер, на основании расстояния от центра системы
		 координат до точки, передаваемой посредством параметра Point.


		 ![](../../Sub_Image.gif)
		 [CreateFromSize](GxSize.CreateFromSize.htm)
		 Конструктор CreateFromSize
		 создает новый размер на основе указанного размера.
