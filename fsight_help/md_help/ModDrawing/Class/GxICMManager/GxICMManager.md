# GxICMManager


Сборка: Drawing;


## Описание


Класс GxICMManager реализует
 объект, используемый для работы цветовыми профилями принтеров и монитора.


## Свойства объекта класса, унаследованные от [IGxICMManager](../../Interface/IGxICMManager/IGxICMManager.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ICMProfile](../../Interface/IGxICMManager/IGxICMManager.ICMProfile.htm)
		 Свойство ICMProfile
		 определяет наименование цветового профиля, в соответствии с которым
		 будет осуществляться преобразование.


## Методы объекта класса, унаследованные от [IGxICMManager](../../Interface/IGxICMManager/IGxICMManager.htm)


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CheckColor](../../Interface/IGxICMManager/IGxICMManager.CheckColor.htm)
		 Метод CheckColor осуществляет
		 проверку указанного цвета на воспроизводимость при использовании
		 указанного цветового профиля.


		 ![](../../Sub_Image.gif)
		 [GetProfiles](../../Interface/IGxICMManager/IGxICMManager.GetProfiles.htm)
		 Метод GetProfiles возвращает
		 список цветовых профилей, сопоставленных с указанным принтером.


		 ![](../../Sub_Image.gif)
		 [GetScreenProfiles](../../Interface/IGxICMManager/IGxICMManager.GetScreenProfiles.htm)
		 Метод GetScreenProfiles
		 возвращает список цветовых профилей, сопоставленных с монитором.


		 ![](../../Sub_Image.gif)
		 [TranslateColor](../../Interface/IGxICMManager/IGxICMManager.TranslateColor.htm)
		 Метод TranslateColor
		 осуществляет преобразование указанного цвета в соответствии с
		 палитрой указанного цветового профиля.


		 ![](../../Sub_Image.gif)
		 [TranslateImage](../../Interface/IGxICMManager/IGxICMManager.TranslateImage.htm)
		 Метод TranslateImage
		 осуществляет преобразование указанного изображения в соответствии
		 с палитрой указанного цветового профиля.
