# GxColor


Сборка: Drawing;


## Описание


Класс GxColor реализует цвет.


## Конструкторы


		 Имя конструктора
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CreateARGB](GxColor.CreateARGB.htm)
		 Конструктор CreateARGB
		 создает новый цвет в соответствии с указанными значениями степени
		 прозрачности и красной, зеленой и синей компонент.


		 ![](../../Sub_Image.gif)
		 [CreateHSB](GxColor.CreateHSB.htm)
		 Конструктор CreateHSB
		 создает новый цвет в соответствии с указанными значениями тона,
		 насыщенности и яркости цвета.


		 ![](../../Sub_Image.gif)
		 [CreateRGB](GxColor.CreateRGB.htm)
		 Конструктор CreateRGB
		 создает новый цвет в соответствии с указанными значениями красной,
		 зеленой и синей компонент.

## Свойства объекта класса, унаследованные от [IGxColor](../../Interface/IGxColor/IGxColor.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)

		 ![](../../Property_Image.gif)
		 [ARGB](../../Interface/IGxColor/IGxColor.ARGB.htm)
		 Свойство ARGB определяет
		 цвет как 32-битовую величину.


		 ![](../../Property_Image.gif)
		 [B](../../Interface/IGxColor/IGxColor.B.htm)
		 Свойство B возвращает
		 синюю компоненту цвета от 0 до 255.


		 ![](../../Property_Image.gif)
		 [Brightness](../../Interface/IGxColor/IGxColor.Brightness.htm)
		 Свойство Brightness
		 возвращает яркость цвета от 0,0 до 1,0.


		 ![](../../Property_Image.gif)
		 [G](../../Interface/IGxColor/IGxColor.G.htm)
		 Свойство G возвращает
		 зеленую компоненту цвета от 0 до 255.


		 ![](../../Property_Image.gif)
		 [Hue](../../Interface/IGxColor/IGxColor.Hue.htm)
		 Свойство Hue возвращает
		 тон цвета в градусах от 0,0 до 360,0.


		 ![](../../Property_Image.gif)
		 [IsKnownColor](../../Interface/IGxColor/IGxColor.IsKnownColor.htm)
		 Свойство IsKnownColor
		 возвращает True, если
		 цвет является стандартным.


		 ![](../../Property_Image.gif)
		 [IsSystemColor](../../Interface/IGxColor/IGxColor.IsSystemColor.htm)
		 Свойство IsSystemColor
		 возвращает True, если
		 цвет является системным.


		 ![](../../Property_Image.gif)
		 [IsThemeColor](../../Interface/IGxColor/IGxColor.IsThemeColor.htm)
		 Свойство IsThemeColor
		 возвращает значение True,
		 если текущий цвет является цветом элемента, заданного темой оформления.


		 ![](../../Property_Image.gif)
		 [Name](../../Interface/IGxColor/IGxColor.Name.htm)
		 Свойство Name возвращает
		 название цвета.


		 ![](../../Property_Image.gif)
		 [OleValue](../../Interface/IGxColor/IGxColor.OleValue.htm)
		 Свойство OleValue определяет
		 цвет как 32-битовую величину.


		 ![](../../Property_Image.gif)
		 [R](../../Interface/IGxColor/IGxColor.R.htm)
		 Свойство R возвращает
		 красную компоненту цвета от 0 до 255.


		 ![](../../Property_Image.gif)
		 [Saturation](../../Interface/IGxColor/IGxColor.Saturation.htm)
		 Свойство Saturation
		 возвращает насыщенность цвета.


		 ![](../../Property_Image.gif)
		 [Value](../../Interface/IGxColor/IGxColor.Value.htm)
		 Свойство Value возвращает
		 значение цвета как 32-битовую величину.
