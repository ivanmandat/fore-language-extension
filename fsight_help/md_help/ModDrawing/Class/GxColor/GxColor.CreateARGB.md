# GxColor.CreateARGB


## Синтаксис


		CreateARGB(A: Integer, R: Integer, G: Integer,
		 B: Integer);


## Параметры


A. Степень прозрачности цвета.


R. Красная компонента цвета.


G. Синяя компонента цвета.


B. Зеленая компонента цвета.


## Описание


Конструктор CreateARGB создает
 новый цвет в соответствии с указанными значениями степени прозрачности
 и красной, зеленой и синей компонент.


## Комментарии


Для указания параметров используйте целые числа в диапазоне от 0 до
 255. Если параметр A принимает
 значение 0, то используется 100% степень прозрачности цвета (полностью
 прозрачный цвет), если 255 - 0% (полностью непрозрачный цвет).


Особенности использования конструктора при определении цвета заливки
 выделенных ячеек таблицы с помощью свойства [ITabView.SelectionColor](TabSheet.chm::/Interface/ITabView/ITabView.SelectionColor.htm):


	- если параметр A принимает
	 значение строго меньше 255, то не учитывается степень прозрачности
	 цвета, заданная в [настройках
	 репозитория по умолчанию](UiNav.chm::/02_navigator/repo_default.htm) в параметре «[Прозрачность
	 цвета заливки (%)](UiNav.chm::/02_navigator/repo_default.htm#design)»;


	- если параметр A принимает
	 значение 255, то используется степень прозрачности цвета по умолчанию
	 - 95%.
