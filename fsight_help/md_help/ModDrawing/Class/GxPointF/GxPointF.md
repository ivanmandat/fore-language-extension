# GxPointF


Сборка: Drawing;


## Описание


Класс GxPointF реализует точку
 с вещественными координатами.

		 Имя конструктора
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Create](GxPointF.Create.htm)
		 Конструктор Create
		 создает новую точку с указанными координатами.


		 ![](../../Sub_Image.gif)
		 [CreateFromPoint](GxPointF.CreateFromPoint.htm)
		 Конструктор CreateFromPoint
