# GxPrinters


Сборка: Drawing;


## Описание


Класс GxPrinters реализует объект,
 предоставляющий доступ к параметрам принтеров, установленных в операционной
 системе.

		 Имя свойства

		 ![](../../Property_Image.gif)
		 [DefaultPrinter](../../Interface/IGxPrintersClass/IGxPrintersClass.DefaultPrinter.htm)
		 Свойство DefaultPrinter
		 возвращает принтер, установленный в операционной системе по умолчанию.
