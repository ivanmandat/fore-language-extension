# IGxNumericScale


Сборка: Drawing;


## Описание


Интерфейс IGxNumericScale используется
 для работы с числовой шкалой.


## Иерархия наследования


           IGxScale


           IGxNumericScale


## Комментарии


Для получения объекта данного типа необходимо значение, возвращаемое
 свойством IEaxTableStyle.IndicatorScale,
 привести к IGxNumericScale.
