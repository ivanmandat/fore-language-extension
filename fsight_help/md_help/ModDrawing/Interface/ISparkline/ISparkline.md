# ISparkline


Сборка: Drawing;


## Описание


Интерфейс ISparkline содержит
 базовые свойства спарклайнов, используемых в различных объектах репозитория.


## Иерархия наследования


ISparkline


## Комментарии


Спарклайн -
 небольшая диаграмма, отражающая в миниатюре динамику изменения каких-либо
 данных.


Данный интерфейс является базовым для следующих интерфейсов:


	- [ITabSparkline](TabSheet.chm::/Interface/ITabSparkline/ITabSparkline.htm).
