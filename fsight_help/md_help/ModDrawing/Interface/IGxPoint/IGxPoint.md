# IGxPoint


Сборка: Drawing;


## Описание


Интерфейс IGxPoint содержит
 свойства и методы целочисленной точки.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [IsEmpty](IGxPoint.IsEmpty.htm)
		 Свойство IsEmpty возвращает
		 признак соответствия точке с нулевыми координатами.


		 ![](../../Property_Image.gif)
		 [X](IGxPoint.X.htm)
		 Свойство X определяет
		 координату Х точки.


		 ![](../../Property_Image.gif)
		 [Y](IGxPoint.Y.htm)
		 Свойство Y определяет
		 координату Y точки.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Equals](IGxPoint.Equals.htm)
		 Метод Equals проверяет
		 равенство данной точки с точкой, передаваемой в качестве параметра.


		 ![](../../Sub_Image.gif)
		 [Offset](IGxPoint.Offset.htm)
		 Метод Offset осуществляет
		 смещение точки на заданные значения по осям Х и Y.


		 ![](../../Sub_Image.gif)
		 [OffsetP](IGxPoint.OffsetP.htm)
		 Метод OffsetP осуществляет
		 смещение текущей точки на значения координат точки, передаваемой
		 посредством входного параметра.


		 ![](../../Sub_Image.gif)
		 [OffsetS](IGxPoint.OffsetS.htm)
		 Метод OffsetS осуществляет
		 смещение точки на размер, передаваемый посредством входного параметра.


		 ![](../../Sub_Image.gif)
		 [ToPointF](IGxPoint.ToPointF.htm)
		 Метод ToPointF осуществляет
		 преобразование точки в вещественную точку.


		 ![](../../Sub_Image.gif)
		 [ToSize](IGxPoint.ToSize.htm)
		 Метод ToSize осуществляет
		 преобразование точки в размер.
