# IGxSystemFontNames


Сборка: Drawing;


## Описание


Интерфейс IGxSystemFontNames
 предназначен для работы с коллекцией шрифтов, установленных в операционной
 системе.


## Иерархия наследования


           IGxSystemFontNames


## Комментарии


Для получения коллекции системных шрифтов необходимо создать новый объект
 с помощью класса [GxSystemFontNames](../../Class/GxSystemFontNames/GxSystemFontNames.htm).
