# IGxDeviceICMProfiles.Item


## Синтаксис


Item(Index: Integer): String;


## Параметры


Index - индекс цветового профиля.


## Описание


Свойство Item возвращает наименование
 цветового профиля, сопоставленного с устройством.


## Комментарии


Свойство возвращает наименование цветового профиля без расширения.
