# IGxDeviceICMProfiles


Сборка: Drawing;


## Описание


Интерфейс IGxDeviceICMProfiles
 содержит свойства для работы с коллекцией цветовых профилей, сопоставленных
 с устройством.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IGxDeviceICMProfiles.Count.htm)
		 Свойство Count возвращает
		 количество цветовых профилей, сопоставленных с устройством.


		 ![](../../Property_Image.gif)
		 [Item](IGxDeviceICMProfiles.Item.htm)
		 Свойство Item возвращает
		 наименование цветового профиля, сопоставленного с устройством.
