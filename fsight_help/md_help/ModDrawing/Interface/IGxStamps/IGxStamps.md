# IGxStamps


Сборка: Drawing;


## Описание


Интерфейс IGxStamps содержит
 набор штампов маркировки документа.


## Иерархия наследования


IGxStamps


## Комментарии


Содержит штампы маркировки, отличающиеся положением на странице. Условия
 отображения штампов маркировки приведены в [IGxStamp](../IGxStamp/IGxStamp.htm).


Используется интерфейсом [IGxDocument](../IGxDocument/IGxDocument.htm).
