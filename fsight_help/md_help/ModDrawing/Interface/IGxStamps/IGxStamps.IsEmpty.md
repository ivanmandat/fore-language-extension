# IGxStamps.IsEmpty


## Синтаксис


IsEmpty: Boolean;


## Описание


Свойство IsEmpty определяет
 признак отсутствия штампов.


## Комментарии


Возвращает True, если штампы
 отсутствуют.


Возвращает False, если настроен
 как минимум один штамп.
