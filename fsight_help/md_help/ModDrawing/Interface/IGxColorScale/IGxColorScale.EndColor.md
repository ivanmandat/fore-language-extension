# IGxColorScale.EndColor


## Синтаксис


EndColor: [IGxColor](../IGxColor/IGxColor.htm);


## Описание


Свойство EndColor определяет
 конечный цвет цветовой шкалы.


## Комментарии


Для использования данного свойства необходимо, чтобы свойство [IGxColorScale.UseAutoValues](../IGxScale/IGxScale.UseAutoValues.htm)
 принимало значение True.


Для задания начального цвета цветовой шкалы, используйте свойство [IGxColorScale.StartColor](IGxColorScale.StartColor.htm).
