# IGxColorScale


Сборка: Drawing;


## Описание


Интерфейс IGxColorScale используется
 для работы с цветовой шкалой.


## Иерархия наследования


           [IGxScale](../IGxScale/IGxScale.htm)


           IGxColorScale


## Комментарии


Для получения объекта данного типа необходимо значение, возвращаемое
 свойством IEaxTableStyle.IndicatorScale,
 привести к [IGxColorScale](IGxColorScale.htm).
