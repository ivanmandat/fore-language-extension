# IGxThemeColors.BarDkShadow


## Синтаксис


BarDkShadow: [IGxColor](../IGxColor/IGxColor.htm);


## Описание


Свойство BarDkShadow возвращает
 темный цвет тени всплывающих панелей.


## Комментарии


Данный цвет используется при отрисовке границы панели в плавающем состоянии.
 Темный цвет имеют участки тени максимально отдаленные от границы всплывающей
 панели.
