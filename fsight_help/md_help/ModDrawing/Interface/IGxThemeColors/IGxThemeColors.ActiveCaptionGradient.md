# IGxThemeColors.ActiveCaptionGradient


## Синтаксис


ActiveCaptionGradient: [IGxColor](../IGxColor/IGxColor.htm);


## Описание


Свойство ActiveCaptionGradient
 возвращает цвет, являющийся самым светлым цветом градиента в заголовке
 активного окна.


## Комментарии


Свойство возвращает цвет, являющийся конечным цветом градиента, используемого
 для заливки заголовка активного окна. Данный градиент используется при
 отрисовке заголовков форм и всплывающих панелей.


Если используется тема оформления, изменяющая способ отрисовки заголовка
 формы, то данное свойство актуально только для заголовков всплывающих
 панелей.
