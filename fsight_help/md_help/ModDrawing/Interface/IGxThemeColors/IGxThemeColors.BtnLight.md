# IGxThemeColors.BtnLight


## Синтаксис


BtnLight: [IGxColor](../IGxColor/IGxColor.htm);


## Описание


Свойство BtnLight возвращает
 светлый цвет трехмерного элемента.


## Комментарии


Светлый цвет применяется к участкам трехмерного элемента, которые обращены
 к источнику света.
