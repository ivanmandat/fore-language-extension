# IGxThemeColors.BtnHighlight


## Синтаксис


BtnHighlight: [IGxColor](../IGxColor/IGxColor.htm);


## Описание


Свойство BtnHighlight возвращает
 цвет выделения трехмерного элемента.


## Комментарии


Этот цвет применяется к тем участкам трехмерного элемента, которые обращены
 к источнику света.
