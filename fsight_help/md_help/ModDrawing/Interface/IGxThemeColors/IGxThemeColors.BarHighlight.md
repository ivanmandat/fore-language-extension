# IGxThemeColors.BarHighlight


## Синтаксис


BarHighlight: [IGxColor](../IGxColor/IGxColor.htm);


## Описание


Свойство BarHighlight возвращает
 цвет, используемый для отрисовки самой яркой области в границе всплывающей
 панели.


## Комментарии


Данный цвет используется при отрисовке границы панели в плавающем состоянии.
