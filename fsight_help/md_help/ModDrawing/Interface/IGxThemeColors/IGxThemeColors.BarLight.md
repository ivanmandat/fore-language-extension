# IGxThemeColors.BarLight


## Синтаксис


BarLight: [IGxColor](../IGxColor/IGxColor.htm);


## Описание


Свойство BarLight возвращает
 цвет, используемый для отрисовки границы всплывающей панели.


## Комментарии


Данный цвет используется при отрисовке границы панели в плавающем состоянии.
