# IGxThemeColors.InactiveCaption


## Синтаксис


InactiveCaption: [IGxColor](../IGxColor/IGxColor.htm);


## Описание


Свойство InactiveCaption возвращает
 цвет фона заголовка неактивного окна.


## Комментарии


Свойство возвращает цвет, являющийся начальным цветом градиента, используемого
 для заливки заголовка неактивного окна. Данный градиент используется при
 отрисовке заголовков форм и всплывающих панелей.


Если используется тема оформления, изменяющая способ отрисовки заголовка
 формы, то данное свойство актуально только для заголовков всплывающих
 панелей.
