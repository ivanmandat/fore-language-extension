# IGxThemeColors.BarShadow


## Синтаксис


BarShadow: [IGxColor](../IGxColor/IGxColor.htm);


## Описание


Свойство BarShadow возвращает
 цвет тени всплывающей панели.


## Комментарии


Данный цвет используется при отрисовке границы панели в плавающем состоянии.
