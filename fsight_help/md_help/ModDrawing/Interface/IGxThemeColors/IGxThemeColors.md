# IGxThemeColors


Сборка: Drawing;


## Описание


Интерфейс IGxThemeColors содержит
 статические свойства класса [GxThemeColors](../../Class/GxThemeColors/GxThemeColors.htm),
 предназначенные для получения цветов элементов управления в соответствии
 с используемой темой оформления.
