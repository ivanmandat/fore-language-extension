# IGxThemeColors.BtnDkShadow


## Синтаксис


BtnDkShadow: [IGxColor](../IGxColor/IGxColor.htm);


## Описание


Свойство BtnDkShadow возвращает
 темный цвет тени трехмерного элемента.


## Комментарии


Темный цвет тени применяется к тем участкам трехмерного элемента, которые
 имеют самый темный цвет.
