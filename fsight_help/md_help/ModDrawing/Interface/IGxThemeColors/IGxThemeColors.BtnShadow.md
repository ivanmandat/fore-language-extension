# IGxThemeColors.BtnShadow


## Синтаксис


BtnShadow: [IGxColor](../IGxColor/IGxColor.htm);


## Описание


Свойство BtnShadow возвращает
 цвет тени трехмерного элемента.


## Комментарии


Данный цвет применяется к участкам трехмерного элемента, которые обращены
 не к источнику света, а от него.
