# IGxLinearGradientBrush


Сборка: Drawing;


## Описание


Интерфейс IGxLinearGradientBrush
 содержит свойства линейно-градиентной кисти.


## Свойства


		 Имя свойства
		 Краткое описание

		 [Angle](IGxLinearGradientBrush.Angle.htm)
		 Свойство Angle возвращает
		 угол наклона градиента кисти.


		 ![](../../Property_Image.gif)
		 [EndColor](IGxLinearGradientBrush.EndColor.htm)
		 Свойство EndColor возвращает
		 конечный цвет градиента кисти.


		 ![](../../Property_Image.gif)
		 [StartColor](IGxLinearGradientBrush.StartColor.htm)
		 Свойство StartColor
		 возвращает начальный цвет градиента кисти.
