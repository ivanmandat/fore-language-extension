# IGxSize.IsEmpty


## Синтаксис


IsEmpty: Boolean;


## Описание


Свойство IsEmpty возвращает
 признак соответствия размеру с нулевыми шириной и высотой.


## Комментарии


Свойство возвращает значение True,
 если размер имеет нулевые ширину и высоту и False,
 если хотя бы один параметр отличен от нуля.
