# IGxSize


Сборка: Drawing;


## Описание


Интерфейс IGxSize содержит свойства
 и методы размера, заданного значениями целочисленного типа.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Height](IGxSize.Height.htm)
		 Свойство Height определяет
		 высоту размера.


		 ![](../../Property_Image.gif)
		 [IsEmpty](IGxSize.IsEmpty.htm)
		 Свойство IsEmpty возвращает
		 признак соответствия размеру с нулевыми шириной и высотой.


		 ![](../../Property_Image.gif)
		 [Width](IGxSize.Width.htm)
		 Свойство Width определяет
		 ширину размера.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Equals](IGxSize.Equals.htm)
		 Метод Equals проверяет
		 равенство текущего размера с размером, передаваемым посредством
		 входного параметра.


		 ![](../../Sub_Image.gif)
		 [Expand](IGxSize.Expand.htm)
		 Метод Expand осуществляет
		 увеличение размера на указанные значения по ширине и высоте.


		 ![](../../Sub_Image.gif)
		 [ExpandS](IGxSize.ExpandS.htm)
		 Метод ExpandS осуществляет
		 увеличение текущего размера на размер, передаваемый посредством
		 входного параметра.


		 ![](../../Sub_Image.gif)
		 [Reduce](IGxSize.Reduce.htm)
		 Метод Reduce осуществляет
		 уменьшение размера на указанные значения по ширине и высоте.


		 ![](../../Sub_Image.gif)
		 [ReduceS](IGxSize.ReduceS.htm)
		 Метод ReduceS осуществляет
		 уменьшение размера на указанный размер.


		 ![](../../Sub_Image.gif)
		 [ToPoint](IGxSize.ToPoint.htm)
		 Метод ToPoint осуществляет
		 преобразование размера в точку.


		 ![](../../Sub_Image.gif)
		 [ToSizeF](IGxSize.ToSizeF.htm)
		 Метод ToSizeF осуществляет
		 преобразование целочисленного размера в вещественный размер.
