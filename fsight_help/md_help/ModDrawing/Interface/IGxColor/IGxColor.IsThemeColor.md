# IGxColor.IsThemeColor


## Синтаксис


IsThemeColor: Boolean;


## Описание


Свойство IsThemeColor возвращает
 значение True, если текущий цвет
 является цветом элемента, заданного темой оформления.


## Комментарии


Для получения цветов, определенных используемой темой оформления, используйте
 статические свойства класса [GxThemeColors](../../Class/GxThemeColors/GxThemeColors.htm).
