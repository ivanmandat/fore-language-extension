# IGxColor.Value


## Синтаксис


Value: Integer;


## Описание


Свойство Value возвращает значение
 цвета как 32-битовую величину.


## Комментарии


Цветовая величина содержит четыре 8-битовые компоненты. Для компоненты
 [A](IGxColor.A.htm) берется значение 0, поэтому она не используется
 при расчете. Остальные компоненты расположены в следующем порядке: [B](IGxColor.B.htm), [G](IGxColor.G.htm), [R](IGxColor.R.htm).
 Цветовая величина представлена в десятичном виде. При необходимости из
 нее можно получить значения отдельных компонент, выполнив следующие преобразования:


	- Приведите значение Value
	 к шестнадцатиричному виду;


	- Полученное значение будет представлять из себя шесть шестнадцатиричных
	 цифр, разбейте полученное значение на три комбинации по две цифры;


	- Если каждую комбинацию теперь привести к десятичному виду, то
	 они будут соответствовать значениям отдельных компонент: [B](IGxColor.B.htm),
	 [G](IGxColor.G.htm), [R](IGxColor.R.htm).


## Пример


	- Value = 13382451^10
	 = CC3333^16. [B](IGxColor.B.htm) = CC^16 = 204^10, [G](IGxColor.G.htm)
	 = 33^16 = 51^10 , [R](IGxColor.R.htm) = 33^16 = 51^10.
	 Это соответствует оттенку
	 синего цвета.


	- Value = 255^10 = FF^16.
	 [B](IGxColor.B.htm) = 0^16 = 0^10, [G](IGxColor.G.htm)
	 = 0^16 = 0^10 , [R](IGxColor.R.htm) = FF^16 = 255^10. Это
	 соответствует красному
	 цвету.
