# IGxColor


Сборка: Drawing;


## Описание


Интерфейс IGxColor предназначен
 для работы с параметрами цвета.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [A](IGxColor.A.htm)
		 Свойство A возвращает
		 степень прозрачности цвета от 0 до 255.


		 ![](../../Property_Image.gif)
		 [ARGB](IGxColor.ARGB.htm)
		 Свойство ARGB определяет
		 цвет как 32-битовую величину.


		 ![](../../Property_Image.gif)
		 [B](IGxColor.B.htm)
		 Свойство B возвращает
		 синюю компоненту цвета от 0 до 255.


		 ![](../../Property_Image.gif)
		 [Brightness](IGxColor.Brightness.htm)
		 Свойство Brightness
		 возвращает яркость цвета от 0,0 до 1,0.


		 ![](../../Property_Image.gif)
		 [G](IGxColor.G.htm)
		 Свойство G возвращает
		 зеленую компоненту цвета от 0 до 255.


		 ![](../../Property_Image.gif)
		 [Hue](IGxColor.Hue.htm)
		 Свойство Hue возвращает
		 тон цвета в градусах от 0,0 до 360,0.


		 ![](../../Property_Image.gif)
		 [IsKnownColor](IGxColor.IsKnownColor.htm)
		 Свойство IsKnownColor
		 возвращает True, если
		 цвет является стандартным.


		 ![](../../Property_Image.gif)
		 [IsSystemColor](IGxColor.IsSystemColor.htm)
		 Свойство IsSystemColor
		 возвращает True, если
		 цвет является системным.


		 ![](../../Property_Image.gif)
		 [IsThemeColor](IGxColor.IsThemeColor.htm)
		 Свойство IsThemeColor
		 возвращает значение True,
		 если текущий цвет является цветом элемента, заданного темой оформления.


		 ![](../../Property_Image.gif)
		 [Name](IGxColor.Name.htm)
		 Свойство Name возвращает
		 название цвета.


		 ![](../../Property_Image.gif)
		 [OleValue](IGxColor.OleValue.htm)
		 Свойство OleValue определяет
		 цвет как 32-битовую величину.


		 ![](../../Property_Image.gif)
		 [R](IGxColor.R.htm)
		 Свойство R возвращает
		 красную компоненту цвета от 0 до 255.


		 ![](../../Property_Image.gif)
		 [Saturation](IGxColor.Saturation.htm)
		 Свойство Saturation
		 возвращает насыщенность цвета.


		 ![](../../Property_Image.gif)
		 [Value](IGxColor.Value.htm)
		 Свойство Value возвращает
		 значение цвета как 32-битовую величину.
