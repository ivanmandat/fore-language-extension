# IGxColor.ARGB


## Синтаксис


ARGB: Integer;


## Описание


Свойство ARGB определяет цвет как 32-битовую величину.


## Комментарии


Цветовая величина содержит четыре 8-битовые компоненты, взятые в следующем порядке: [A](IGxColor.A.htm), [R](IGxColor.R.htm), [G](IGxColor.G.htm), [B](IGxColor.B.htm).
