# IGxColor.OleValue


## Синтаксис


OleValue: Integer;


## Описание


Свойство OleValue возвращает значение цвета как 32-битовую величину.


## Комментарии


Цветовая величина содержит четыре 8-битовые компоненты. Данное свойство актуально для системных цветов, при этом в качестве значения компоненты [A](IGxColor.A.htm) берется постоянное значение. Остальные компоненты расположены в следующем порядке: [B](IGxColor.B.htm), [G](IGxColor.G.htm), [R](IGxColor.R.htm). Для всех остальных цветов функция возвращает значение, аналогичное [Value](IGxColor.Value.htm).
