# IGxTitle.BlackAndWhite


## Синтаксис


BlackAndWhite: Boolean;


## Описание


Свойство BlackAndWhite определяет
 признак черно-белой печати заголовка.


## Комментарии


По умолчанию свойству установлено значение False,
 при этом печать и экспорт производятся с сохранением всего цветового оформления
 заголовка. Если свойству установить значение True,
 то при печати/экспорте все цвета будут преобразованы в оттенки серого.
 Свойство доступно только на запись и актуально при экспорте/печати объекта,
 которому принадлежит заголовок.
