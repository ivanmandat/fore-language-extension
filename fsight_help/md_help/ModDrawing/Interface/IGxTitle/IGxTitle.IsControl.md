# IGxTitle.IsControl


## Синтаксис


IsControl: Boolean;


## Описание


Свойство IsControl определяет
 признак отображения заголовка как элемента управления.


## Комментарии


Если свойству установлено значение True,
 то заголовок будет работать в режиме элемента управления. Если в заголовке
 присутствуют подстановки для отображения отметки календарного измерения
 или дополнительных измерений, то отметку можно будет изменить. При щелчке
 по наименованию отмеченного элемента будет открыт список элементов измерения,
 в котором можно будет изменить отметку.


Если свойству IsControl установлено
 значение False, то заголовок будет
 работать в обычном режиме отображения текста.
