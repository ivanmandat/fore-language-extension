# IGxTitle.AutoHeight


## Синтаксис


AutoHeight: Boolean;


## Описание


Свойство AutoHeight определяет,
 включена ли автоматическая подгонка высоты заголовка в зависимости от
 содержимого.


## Комментарии


Допустимые значения:


	- True. Значение
	 по умолчанию. Автоматическая подгонка высоты заголовка включена.[ IGxTitle.Height](IGxTitle.Height.htm) определяет максимальную
	 высоту заголовка;


	- False. Автоматическая
	 подгонка высоты заголовка выключена.
