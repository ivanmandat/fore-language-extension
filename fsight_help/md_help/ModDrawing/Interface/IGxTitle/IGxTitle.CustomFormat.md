# IGxTitle.CustomFormat


## Синтаксис


CustomFormat: String;


## Описание


Свойство CustomFormat определяет
 формат числа в значении заголовка.


## Комментарии


Применяется только для настройки [формата
 числа](UiExpress.chm::/Setup_express_report/UiExpress_Purpose_Toolbar_Title.htm#number_format) при вставке значения в заголовок экспресс-отчета.
 В зависимости от выбранного формата используйте в свойстве CustomFormat
 соответствующий «Шаблон» из таблицы:
