# IGxTitle.WrappedText


## Синтаксис


WrappedText: Boolean;


## Описание


Свойство WrappedText определяет,
 переносить ли текст заголовка на следующую строку, если он не помещается
 в одну строку.


## Комментарии


Допустимые значения:


	- True. Значение
	 по умолчанию. Текст заголовка переносится.


	- False. Текст заголовка
	 не переносится.
