# IGxTitle.HtmlTitle


## Синтаксис


HtmlTitle: String;


## Описание


Свойство HtmlTitle возвращает
 текст заголовка, полученный после обработки автоподстановок, на языке
 разметки HTML.


## Комментарии


В тексте допустимо использование подстановок. Список подстановок приведен
 в описании свойства [IGxTitle.Text](IGxTitle.Text.htm).
