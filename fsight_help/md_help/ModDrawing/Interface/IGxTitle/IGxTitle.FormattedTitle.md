# IGxTitle.FormattedTitle


## Синтаксис


FormattedTitle: String;


## Описание


Свойство FormattedTitle возвращает
 текст заголовка, предназначенный для представления в формате RTF.

 форматированного текста](KnowledgeBase.chm::/01_Fore/knowledgebase_kb000034.htm)»
