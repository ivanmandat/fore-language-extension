# IGxTitle.IsTagIncluded


## Синтаксис


IsTagIncluded(Tag: String): Boolean;


## Параметры


Tag. Подстановка,
 наличие которой необходимо определить.


## Описание


Свойство IsTagIncluded определяет,
 присутствует ли в тексте заголовка указанная подстановка.


## Комментарии


При получении значения свойства: если указанная подстановка присутствует
 в тексте, то свойство вернет значение True,
 иначе False.


При установке значения свойства: если свойству установить значение True, то указанная подстановка будет
 добавлена в конец заголовка (если подстановка уже присутствовала, то она
 останется на том же месте). Если свойству установить значение False,
 то указанная подстановка будет исключена из заголовка.


Список допустимых подстановок, указываемых в параметре Tag,
 смотрите в описании свойства [Text](IGxTitle.Text.htm).
