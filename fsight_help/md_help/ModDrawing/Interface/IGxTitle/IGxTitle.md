# IGxTitle


Сборка: Drawing;


## Описание


Интерфейс IGxTitle содержит
 базовые свойства и методы для работы с графическим заголовком объектов.


## Иерархия наследования


IGxTitle


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AutoHeight](IGxTitle.AutoHeight.htm)
		 Свойство AutoHeight определяет,
		 включена ли автоматическая подгонка высоты заголовка в зависимости
		 от содержимого.


		 ![](../../Property_Image.gif)
		 [BackgroundColor](IGxTitle.BackgroundColor.htm)
		 Свойство BackgroundColor
		 определяет цвет фона заголовка.


		 ![](../../Property_Image.gif)
		 [BlackAndWhite](IGxTitle.BlackAndWhite.htm)
		 Свойство BlackAndWhite
		 определяет признак черно-белой печати заголовка.


		 ![](../../Property_Image.gif)
		 [CustomFormat](IGxTitle.CustomFormat.htm)
		 Свойство CustomFormat определяет
		 формат числа в значении заголовка.


		 ![](../../Property_Image.gif)
		 [Font](IGxTitle.Font.htm)
		 Свойство Font определяет
		 шрифт заголовка.


		 ![](../../Property_Image.gif)
		 [FormattedText](IGxTitle.FormattedText.htm)
		 Свойство FormattedText
		 определяет текст заголовка, предназначенный для представления
		 в формате RTF.


		 ![](../../Property_Image.gif)
		 [FormattedTitle](IGxTitle.FormattedTitle.htm)
		 Свойство FormattedTitle
		 возвращает текст заголовка, предназначенный для представления
		 в формате RTF.


		 ![](../../Property_Image.gif)
		 [Height](IGxTitle.Height.htm)
		 Свойство Height определяет
		 высоту области заголовка.


		 ![](../../Property_Image.gif)
		 [HtmlText](IGxTitle.HtmlText.htm)
		 Свойство HtmlText определяет
		 текст заголовка на языке разметки HTML.


		 ![](../../Property_Image.gif)
		 [HtmlTitle](IGxTitle.HtmlTitle.htm)
		 Свойство HtmlTitle
		 возвращает текст заголовка, полученный после обработки автоподстановок,
		 на языке разметки HTML.


		 ![](../../Property_Image.gif)
		 [IsControl](IGxTitle.IsControl.htm)
		 Свойство IsControl
		 определяет признак отображения заголовка как элемента управления.


		 ![](../../Property_Image.gif)
		 [IsTagIncluded](IGxTitle.IsTagIncluded.htm)
		 Свойство IsTagIncluded
		 определяет, присутствует ли в тексте заголовка указанная подстановка.


		 ![](../../Property_Image.gif)
		 [Text](IGxTitle.Text.htm)
		 Свойство Text определяет
		 текст заголовка.


		 ![](../../Property_Image.gif)
		 [TextAlignment](IGxTitle.TextAlignment.htm)
		 Свойство TextAlignment
		 определяет горизонтальное выравнивание текста в заголовке.


		 ![](../../Property_Image.gif)
		 [TextColor](IGxTitle.TextColor.htm)
		 Свойство TextColor
		 определяет цвет текста заголовка.


		 ![](../../Property_Image.gif)
		 [Title](IGxTitle.Title.htm)
		 Свойство Title возвращает
		 текст заголовка, полученный после обработки автоподстановок.


		 ![](../../Property_Image.gif)
		 [VerticalAlignment](IGxTitle.VerticalAlignment.htm)
		 Свойство VerticalAlignment
		 определяет вертикальное выравнивание текста в заголовке.


		 ![](../../Property_Image.gif)
		 [Visible](IGxTitle.Visible.htm)
		 Свойство Visible определяет
		 признак видимости заголовка.


		 ![](../../Property_Image.gif)
		 [WrappedText](IGxTitle.WrappedText.htm)
		 Свойство WrappedText определяет,
		 переносить ли текст заголовка на следующую строку, если он не
		 помещается в одну строку.
