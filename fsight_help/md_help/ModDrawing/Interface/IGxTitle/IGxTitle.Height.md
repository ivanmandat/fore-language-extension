# IGxTitle.Height


## Синтаксис


Height: Double;


## Описание


Свойство Height определяет высоту
 области заголовка.


## Комментарии


Значение высоты указывается в миллиметрах. По умолчанию заголовок имеет
 высоту 5 миллиметров.


При установке свойства [IGxTitle.AutoHeight](IGxTitle.AutoHeight.htm)
 в значение True, Height определяет
 максимальную высоту заголовка.
