# IGxMargins


Сборка: Drawing;


## Описание


Интерфейс IGxMargins содержит
 свойства для определения отступов различных объектов.

		 Свойство Left определяет
		 значение левого отступа.


		 ![](../../Property_Image.gif)
		 [Right](IGxMargins.Right.htm)
