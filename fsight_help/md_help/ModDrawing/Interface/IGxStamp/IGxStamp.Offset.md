# IGxStamp.Offset


## Синтаксис


Offset: [IGxSizeF](../../Class/GxSizeF/GxSizeF.htm);


## Описание


Свойство Offset определяет смещение
 штампа от места расположения по умолчанию.


## Комментарии


Смещение определяется от левого верхнего угла.
