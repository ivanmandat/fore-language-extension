# IGxImage.PixelFormat


## Синтаксис


PixelFormat: [GxPixelFormat](../../Enums/GxPixelFormat.htm);


## Описание


Свойство PixelFormat возвращает формат пикселей изображения.


## Комментарии


Формат пикселей определяет количество бит памяти, связанных с данными одной точки. Также этот формат определяет порядок цветовых составляющих в данных одной точки.
