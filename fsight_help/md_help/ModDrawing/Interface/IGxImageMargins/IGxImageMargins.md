# IGxImageMargins


Сборка: Drawing;


## Описание


Интерфейс IGxImageMargins содержит
 свойства отступов графических изображений.


## Свойства, унаследованные от [IGxMargins](../IGxMargins/IGxMargins.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Bottom](../IGxMargins/IGxMargins.Bottom.htm)
		 Свойство Bottom определяет
		 нижний отступ графического изображения.


		 ![](../../Property_Image.gif)
		 [Left](../IGxMargins/IGxMargins.Left.htm)
		 Свойство Left определяет
		 левый отступ графического изображения.


		 ![](../../Property_Image.gif)
		 [Right](../IGxMargins/IGxMargins.Right.htm)
		 Свойство Right определяет
		 правый отступ графического изображения.


		 ![](../../Property_Image.gif)
		 [Top](../IGxMargins/IGxMargins.Top.htm)
		 Свойство Top определяет
		 верхний отступ графического изображения.
