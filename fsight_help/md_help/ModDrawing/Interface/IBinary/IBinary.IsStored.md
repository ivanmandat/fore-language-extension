# IBinary.IsStored


## Синтаксис


		IsStored: Boolean;


## Описание


Свойство IsStored возвращает
 признак хранения бинарного объекта в хранилище BI-сервера.


## Комментарии


Свойство возвращает значение True,
 если передаваемый бинарный объект сохранён в хранилище BI-сервера и False, если объект не связан с хранилищем
 BI-сервера.


## Пример


Пример использования операции [ForeExec](PPSOMService.chm::/Operations/Repository/ForeExec.htm)
 для работы с прикладными методами и передачи в них различных объектов

 объектов в качестве значений параметров](PPSOMService.chm::/Operations/Repository/ForeExec_Sample/Object_In_Params.htm)».
