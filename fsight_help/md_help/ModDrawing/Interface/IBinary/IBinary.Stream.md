# IBinary.Stream


## Синтаксис


		Stream: [IIOStream](ModIo.chm::/Interface/IIOStream/IIOStream.htm);


## Описание


Свойство Stream возвращает поток,
 связанный с бинарным объектом.


## Комментарии


В полученном потоке курсор будет расположен в конце. Перед тем, как
 будет осуществляться работа, для перехода к началу потока используйте
 метод [IIOStream.Seek](ModIo.chm::/Interface/IIOStream/IIOStream.Seek.htm).


## Пример


Пример использования операции [ForeExec](PPSOMService.chm::/Operations/Repository/ForeExec.htm)
 для работы с прикладными методами и передачи в них различных объектов
 приведён в подразделе «[Передача
 объектов в качестве значений параметров](PPSOMService.chm::/Operations/Repository/ForeExec_Sample/Object_In_Params.htm)».
