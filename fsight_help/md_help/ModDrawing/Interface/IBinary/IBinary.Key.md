## Синтаксис

## Описание


Свойство Key возвращает ключ
 бинарного объекта в хранилище BI-сервера.

 объектов в качестве значений параметров](PPSOMService.chm::/Operations/Repository/ForeExec_Sample/Object_In_Params.htm)».
