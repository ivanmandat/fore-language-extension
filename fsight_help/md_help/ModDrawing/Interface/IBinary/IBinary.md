# IBinary


Сборка: Drawing;


## Описание


Интерфейс IBinary предназначен
 для работы с бинарными данными, которые могут быть переданы в прикладной
 макрос из операций BI-сервера.


## Иерархия наследования


IBinary


## Комментарии


Для выполнения прикладных макросов используется операция [ForeExec](PPSOMService.chm::/Operations/Repository/ForeExec.htm).
