# IGxIcon


Сборка: Drawing;


## Описание


Интерфейс IGxIcon содержит свойства
 и методы пиктограммы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Height](IGxIcon.Height.htm)
		 Свойство Height возвращает
		 высоту пиктограммы в пикселях.


		 ![](../../Property_Image.gif)
		 [Size](IGxIcon.Size.htm)
		 Свойство Size возвращает
		 объект, содержащий размер пиктограммы.


		 ![](../../Property_Image.gif)
		 [Width](IGxIcon.Width.htm)
		 Свойство Width возвращает
		 ширину пиктограммы в пикселях.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Clone](IGxIcon.Clone.htm)
		 Метод Clone создает
		 копию пиктограммы.


		 ![](../../Sub_Image.gif)
		 [SaveToFile](IGxIcon.SaveToFile.htm)
		 Метод SaveToFile осуществляет
		 сохранение пиктограммы в файл.


		 ![](../../Sub_Image.gif)
		 [SaveToStream](IGxIcon.SaveToStream.htm)
		 Метод SaveToStream
		 осуществляет сохранение пиктограммы в поток.


		 ![](../../Sub_Image.gif)
		 [ToBitmap](IGxIcon.ToBitmap.htm)
		 Метод ToBitmap осуществляет
		 преобразование пиктограммы в битовое изображение.
