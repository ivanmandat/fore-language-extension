# IGxSizeF


Сборка: Drawing;


## Описание


Интерфейс IGxSizeF содержит
 свойства и методы размера, заданного значениями вещественного типа.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Height](IGxSizeF.Height.htm)
		 Свойство Height определяет
		 высоту размера.


		 ![](../../Property_Image.gif)
		 [IsEmpty](IGxSizeF.IsEmpty.htm)
		 Свойство IsEmpty возвращает
		 признак соответствия размеру с нулевыми шириной и высотой.


		 ![](../../Property_Image.gif)
		 [Width](IGxSizeF.Width.htm)
		 Свойство Width определяет
		 ширину размера.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Equals](IGxSizeF.Equals.htm)
		 Метод Equals проверяет
		 равенство текущего вещественного размера с размером, передаваемым
		 посредством входного параметра.


		 ![](../../Sub_Image.gif)
		 [Expand](IGxSizeF.Expand.htm)
		 Метод Expand осуществляет
		 увеличение вещественного размера на указанные значения по ширине
		 и высоте.


		 ![](../../Sub_Image.gif)
		 [ExpandS](IGxSizeF.ExpandS.htm)
		 Метод ExpandS осуществляет
		 увеличение текущего вещественного размера на размер, передаваемый
		 посредством входного параметра.


		 ![](../../Sub_Image.gif)
		 [Reduce](IGxSizeF.Reduce.htm)
		 Метод Reduce осуществляет
		 уменьшение размера на указанные значения по ширине и высоте.


		 ![](../../Sub_Image.gif)
		 [ReduceS](IGxSizeF.ReduceS.htm)
		 Метод ReduceS осуществляет
		 уменьшение вещественного размера на указанный размер.


		 ![](../../Sub_Image.gif)
		 [ToPoint](IGxSizeF.ToPoint.htm)
		 Метод ToPoint осуществляет
		 преобразование вещественного размера в вещественную точку.


		 ![](../../Sub_Image.gif)
		 [ToSize](IGxSizeF.ToSize.htm)
		 Метод ToSize осуществляет
		 преобразование вещественного размера в целочисленный размер.
