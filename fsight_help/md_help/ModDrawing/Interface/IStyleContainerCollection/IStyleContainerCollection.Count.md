# IStyleContainerCollection.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает
 количество контейнеров стилей в коллекции.


## Комментарии


Для получения контейнера стилей из коллекции используйте свойство [IStyleContainerCollection.Item](IStyleContainerCollection.Item.htm).
