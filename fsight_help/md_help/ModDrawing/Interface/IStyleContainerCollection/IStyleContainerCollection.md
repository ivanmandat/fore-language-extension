# IStyleContainerCollection


Сборка: Drawing;


## Описание


Интерфейс IStyleContainerCollection
 предназначен для работы с коллекцией контейнеров стилей.


## Иерархия наследования


           IStyleContainerCollection


## Комментарии


Для получения коллекции используйте метод [IStyleSheet.FindByTag](../IStyleSheet/IStyleSheet.FindByTag.htm).
