# IStyleContainerCollection.Item


## Синтаксис


Item(Index: Integer): [IStyleContainer](../IStyleContainer/IStyleContainer.htm);


## Параметры


Index. Индекс контейнера стилей
 в коллекции.


## Описание


Свойство Item возвращает
 указанный контейнер стилей из коллекции.


## Комментарии


Индексация контейнеров стилей в коллекции сквозная и начинается с нуля.
