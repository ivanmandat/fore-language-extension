# IGxColorClass


Сборка: Drawing;


## Описание


Интерфейс IGxColorClass содержит
 статические методы класса [GxColor](../../Class/GxColor/GxColor.htm),
 используемые для доступа к цветам.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FromKnownColor](IGxColorClass.FromKnownColor.htm)
		 Метод FromKnownColor
		 возвращает предопределенный цвет.


		 ![](../../Sub_Image.gif)
		 [FromName](IGxColorClass.FromName.htm)
		 Метод FromName возвращает
		 цвет с указанным наименованием.


		 ![](../../Sub_Image.gif)
		 [FromSystemColor](IGxColorClass.FromSystemColor.htm)
		 Метод FromSystemColor
		 возвращает системный цвет.


		 ![](../../Sub_Image.gif)
		 [FromThemeColor](IGxColorClass.FromThemeColor.htm)
		 Метод FromThemeColor
		 возвращает цвет, предопределенный для указанного элемента текущей
		 темой оформления.


		 ![](../../Sub_Image.gif)
		 [FromValue](IGxColorClass.FromValue.htm)
		 Метод FromValue возвращает
		 объект, содержащий цвет, по значению.
