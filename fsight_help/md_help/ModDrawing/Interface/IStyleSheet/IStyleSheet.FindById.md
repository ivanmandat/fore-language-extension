# IStyleSheet.FindById


## Синтаксис


FindById(Id: String): [IStyleContainer](../IStyleContainer/IStyleContainer.htm);


## Параметры


Id. Идентификатор контейнера
 стиля, по которому осуществляется поиск.


## Описание


Метод FindById осуществляет
 поиск по идентификатору и возвращает контейнер стиля из таблицы стилей.


## Комментарии


Если контейнер стиля с указанным идентификатором отсутствует, то метод
 возвращает значение Null.
