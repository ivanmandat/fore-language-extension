# IStyleSheet.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс контейнера стиля.


## Описание


Метод Remove осуществляет удаление
 контейнера стиля с указанным индексом.


## Комментарии


Метод возвращает значение True,
 если удаление контейнера стиля завершилось успешно и False,
 если удаление контейнера по каким-либо причинам невозможно.
