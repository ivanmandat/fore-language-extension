# IStyleSheet.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IStyleContainer](../IStyleContainer/IStyleContainer.htm);


## Параметры


Key. Ключ контейнера стиля,
 по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск по ключу и возвращает контейнер стиля из таблицы стилей.


## Комментарии


Если контейнер стиля с указанным ключом отсутствует, то метод возвращает
 значение Null.
