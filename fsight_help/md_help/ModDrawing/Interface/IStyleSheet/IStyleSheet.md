# IStyleSheet


Сборка: Drawing;


## Описание


Интерфейс IStyleSheet содержит
 свойства и методы для работы с таблицей стилей.


## Иерархия наследования


           IStyleSheet


## Комментарии


Таблица стилей представляет собой коллекцию внешних стилей, предназначенных
 для настройки стилевого оформления визуализаторов в инструментах продукта
 «Форсайт. Аналитическая платформа»:
 «Аналитические панели», «Аналитические запросы (OLAP)»,
 «Отчеты», «Анализ
 временных рядов».
