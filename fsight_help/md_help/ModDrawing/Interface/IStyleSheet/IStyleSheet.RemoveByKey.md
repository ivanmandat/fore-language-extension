# IStyleSheet.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ контейнера стиля.


## Описание


Метод RemoveByKey осуществляет
 удаление контейнера стиля с указанным ключом.


## Комментарии


Метод возвращает значение True,
 если удаление контейнера стиля завершилось успешно и False,
 если удаление контейнера по каким-либо причинам невозможно.
