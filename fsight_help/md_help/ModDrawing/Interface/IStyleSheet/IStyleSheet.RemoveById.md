# IStyleSheet.RemoveById


## Синтаксис


RemoveById(Id: String): Boolean;


## Параметры


Id. Идентификатор контейнера
 стиля.


## Описание


Метод RemoveById осуществляет
 удаление контейнера стиля с указанным идентификатором.


## Комментарии


Метод возвращает значение True,
 если удаление контейнера стиля завершилось успешно и False,
 если удаление контейнера по каким-либо причинам невозможно.
