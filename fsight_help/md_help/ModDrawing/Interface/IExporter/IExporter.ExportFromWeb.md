# IExporter.ExportFromWeb


## Синтаксис


		ExportFromWeb: Boolean;


## Описание


Свойство ExportFromWeb возвращает
 тип действия, производимого с отчётом в веб-приложении - экспорт или открытие
 в режиме предварительного просмотра.


## Комментарии


Свойство ExportFromWeb актуально
 для отчетов, содержащих FME-карты. Для того, чтобы экспортируемая FME-карта
 корректно отображалась в веб-приложении, следует указывать относительный
 путь до топоосновы. Но при открытии отчета в режиме предварительного просмотра
 по относительному пути настольное приложение не находит топооснову, поэтому
 возникает необходимость указания абсолютного пути.


Учитывая значение свойства ExportFromWeb,
 в событии экспорта [OnBeforeExportReport](KeReport.chm::/Interface/IPrxReportUserEvents/IPrxReportUserEvents.EventOnBeforeExportReport.htm)
 можно задавать абсолютный или относительный путь. Если свойство ExportFromWeb возвращает значение
 True (экспорт), указывается относительный
 путь, если False (предварительный
 просмотр) - указывается абсолютный путь.
