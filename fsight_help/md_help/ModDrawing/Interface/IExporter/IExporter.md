# IExporter


Сборка: Drawing;


## Описание


Интерфейс IExporter содержит
 базовые свойства и методы, используемые при экспорте различных объектов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CSSClassName](IExporter.CSSClassName.htm)
		 Свойство CSSClassName
		 определяет префикс для элементов таблицы стилей HTML.


		 ![](../../Property_Image.gif)
		 [Encoding](IExporter.Encoding.htm)
		 Свойство Encoding устанавливает
		 кодовую страницу таблицы преобразования, в соответствии с которой
		 будет осуществляться экспорт данных.


		 ![](../../Property_Image.gif)
		 [ExportFromWeb](IExporter.ExportFromWeb.htm)
		 Свойство ExportFromWeb
		 возвращает тип действия, производимого с отчётом в веб-приложении
		 - экспорт или открытие в режиме предварительного просмотра.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ExportToFile](IExporter.ExportToFile.htm)
		 Метод ExportToFile
		 осуществляет экспорт данных в файл указанного формата.
