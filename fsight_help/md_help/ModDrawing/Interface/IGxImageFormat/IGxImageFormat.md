# IGxImageFormat


Сборка: Drawing;


## Описание


Интерфейс IGxImageFormat является
 базовым интерфейсом, содержащим структуру формата графического изображения.


## Комментарии


Интерфейс используется при сравнении со стандартными форматами изображений,
 содержащимися в [IGxImageFormatClass](../IGxImageFormatClass/IGxImageFormatClass.htm).
