# IGxImageList


Сборка: Drawing;


## Описание


Интерфейс IGxImageList содержит
 свойства и методы для работы с коллекцией изображений.


## Иерархия наследования


           IGxImageList


## Комментарии


Интерфейс не поддерживает работу с изображениями глубиной цвета 24 бита.
 Для корректной работы с интерфейсом используйте 32-битные изображения.
