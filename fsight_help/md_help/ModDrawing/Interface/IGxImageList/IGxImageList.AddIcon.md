# IGxImageList.AddIcon


## Синтаксис


AddIcon(Icon: [IGxIcon](../IGxIcon/IGxIcon.htm));


## Параметры


Icon.
 Пиктограмма, которую необходимо добавить в коллекцию.


## Описание


Метод AddIcon добавляет пиктограмму,
 передаваемую посредством параметра Icon,
 в коллекцию изображений.
