# IGxImageList.Width


## Синтаксис


Width: Integer;


## Описание


Свойство Width определяет ширину
 изображений, хранящихся в коллекции.


## Комментарии


Данный размер будет использоваться при разбиении на отдельные изображения,
 в случае если загружаемый графический файл имеет большую ширину.
