# IGxImageList.Item


## Синтаксис


Item(Index: Integer): [IGxImage](ModDrawing.chm::/Interface/IGxImage/IGxImage.htm);


## Параметры


Index. Индекс пиктограммы.


## Описание


Свойство Item возвращает графическое
 изображение пиктограммы, индекс которой передается посредством параметра
 Index.
