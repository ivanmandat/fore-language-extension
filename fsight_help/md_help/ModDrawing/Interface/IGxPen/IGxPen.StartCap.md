# IGxPen.StartCap


## Синтаксис


StartCap: [GxLineCap](../../Enums/GxLineCap.htm);


## Описание


Свойство StartCap определяет
 стиль начала линий, нарисованных при помощи пера.


## Комментарии


В качестве значения данного свойства может быть установлено одно из
 значений [GxLineCap](../../Enums/GxLineCap.htm), отличное от
 [GxLineCap.Undefined](../../Enums/GxLineCap.htm) и [GxLineCap.WxArrow](../../Enums/GxLineCap.htm).


Примечание.
 При попытке установить значение [GxLineCap.Undefined](../../Enums/GxLineCap.htm)
 или [GxLineCap.WxArrow](../../Enums/GxLineCap.htm) будет произведена
 автоматическая замена стиля на [GxLineCap.Flat](../../Enums/GxLineCap.htm).
