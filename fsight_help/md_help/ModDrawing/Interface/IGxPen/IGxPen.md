 пера.

## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Alignment](IGxPen.Alignment.htm)
		 Свойство Alignment
		 определяет выравнивание пера.


		 ![](../../Property_Image.gif)
		 [Brush](IGxPen.Brush.htm)
		 Свойство Brush определяет
		 кисть пера.


		 ![](../../Property_Image.gif)
		 [DashCap](IGxPen.DashCap.htm)
		 Свойство DashCap определяет
		 вид окончания пунктиров, ограничивающих пунктирные линии, нарисованные
		 при помощи объекта пера.


		 ![](../../Property_Image.gif)
		 [DashOffset](IGxPen.DashOffset.htm)
		 Свойство DashOffset
		 определяет расстояние от начала линии до начала штрихового шаблона.


		 ![](../../Property_Image.gif)
		 [EndCap](IGxPen.EndCap.htm)
		 Свойство EndCap определяет
		 стиль окончания линий, нарисованных при помощи пера.


		 ![](../../Property_Image.gif)
		 [LineJoin](IGxPen.LineJoin.htm)
		 Свойство LineJoin определяет
		 общий стиль окончаний для двух последовательных линий, нарисованных
		 при помощи пера.


		 ![](../../Property_Image.gif)
		 [MiterLimit](IGxPen.MiterLimit.htm)
		 Свойство MiterLimit
		 определяет ограничение толщины объединения для острых углов.


		 ![](../../Property_Image.gif)
		 [StartCap](IGxPen.StartCap.htm)
		 Свойство StartCap определяет
		 стиль начала линий, нарисованных при помощи пера.
