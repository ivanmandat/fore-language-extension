# IGxPen.Alignment


## Синтаксис


Alignment: [GxPenAlignment](../../Enums/GxPenAlignment.htm);


## Описание


Свойство Alignment определяет
 выравнивание пера.


## Комментарии


Перо имеет ширину. Центральная точка толщины данного пера выравнивается
 относительно рисуемой линии в зависимости от значения выравнивания. Перо
 может быть расположен таким образом, чтобы рисовать внутри линии или по
 центру над линией.
