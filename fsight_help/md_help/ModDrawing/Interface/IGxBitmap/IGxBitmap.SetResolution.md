# IGxBitmap.SetResolution


## Синтаксис


SetResolution(DpiX: Double; DpiY: Double);


## Параметры


DpiX. Ширина изображения.


DpiY. Высота изображения.


## Описание


Метод SetResolution устанавливает
 разрешение изображения.
