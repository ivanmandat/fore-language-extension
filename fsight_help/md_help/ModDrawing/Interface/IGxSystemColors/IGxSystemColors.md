# IGxSystemColors


Сборка: Drawing;


## Описание


Интерфейс IGxSystemColors содержит
 статические свойства класса [GxSystemColors](../../Class/GxSystemColors/GxSystemColors.htm).


## Комментарии


Свойства позволяют получить цвета, определенные для различных элементов
 в операционной системе.
