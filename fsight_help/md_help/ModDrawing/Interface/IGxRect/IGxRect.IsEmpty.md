# IGxRect.IsEmpty


## Синтаксис


IsEmpty: Boolean;


## Описание


Свойство IsEmpty возвращает
 признак соответствия пустому прямоугольнику.


## Комментарии


Свойство возвращает значение True,
 если прямоугольник имеет нулевые координаты и размеры и False,
 если хотя бы один параметр отличен от нуля.
