# IGxRect


Сборка: Drawing;


## Описание


Интерфейс IGxRect содержит свойства
 и методы целочисленного прямоугольника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Bottom](IGxRect.Bottom.htm)
		 Свойство Bottom определяет
		 координату по оси Y для нижней стороны прямоугольника.


		 ![](../../Property_Image.gif)
		 [Height](IGxRect.Height.htm)
		 Свойство Height определяет
		 высоту целочисленного прямоугольника.


		 ![](../../Property_Image.gif)
		 [IsEmpty](IGxRect.IsEmpty.htm)
		 Свойство IsEmpty возвращает
		 признак соответствия пустому прямоугольнику.


		 ![](../../Property_Image.gif)
		 [Left](IGxRect.Left.htm)
		 Свойство Left определяет
		 координату по оси X для левой стороны прямоугольника.


		 ![](../../Property_Image.gif)
		 [Right](IGxRect.Right.htm)
		 Свойство Right определяет
		 координату по оси X для правой стороны прямоугольника.


		 ![](../../Property_Image.gif)
		 [Size](IGxRect.Size.htm)
		 Свойство Size определяет
		 размер целочисленного прямоугольника.


		 ![](../../Property_Image.gif)
		 [Top](IGxRect.Top.htm)
		 Свойство Top определяет
		 координату по оси Y для верхней стороны прямоугольника.


		 ![](../../Property_Image.gif)
		 [Width](IGxRect.Width.htm)
		 Свойство Width определяет
		 ширину целочисленного прямоугольника.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Contains](IGxRect.Contains.htm)
		 Метод Contains проверяет
		 наличие точки с указанными координатами в текущем прямоугольнике.


		 ![](../../Sub_Image.gif)
		 [ContainsP](IGxRect.ContainsP.htm)
		 Метод ContainsP проверяет
		 наличие точки в текущем прямоугольнике.


		 ![](../../Sub_Image.gif)
		 [ContainsR](IGxRect.ContainsR.htm)
		 Метод ContainsR проверяет
		 наличие прямоугольника в текущем прямоугольнике.


		 ![](../../Sub_Image.gif)
		 [Deflate](IGxRect.Deflate.htm)
		 Метод Deflate осуществляет
		 уменьшение размеров целочисленного прямоугольника на указанные
		 значения по ширине и по высоте.


		 ![](../../Sub_Image.gif)
		 [DeflateS](IGxRect.DeflateS.htm)
		 Метод DeflateS осуществляет
		 уменьшение размеров целочисленного прямоугольника на указанный
		 размер.


		 ![](../../Sub_Image.gif)
		 [Equals](IGxRect.Equals.htm)
		 Метод Equals проверяет
		 равенство текущего целочисленного прямоугольника с прямоугольником,
		 передаваемым посредством входного параметра.


		 ![](../../Sub_Image.gif)
		 [Inflate](IGxRect.Inflate.htm)
		 Метод Inflate осуществляет
		 увеличение размеров целочисленного прямоугольника на указанные
		 значения по ширине и по высоте.


		 ![](../../Sub_Image.gif)
		 [InflateS](IGxRect.InflateS.htm)
		 Метод InflateS осуществляет
		 увеличение целочисленного прямоугольника на указанный размер.


		 ![](../../Sub_Image.gif)
		 [Intersect](IGxRect.Intersect.htm)
		 Метод Intersect возвращает
		 прямоугольник, являющийся результатом пересечения текущего прямоугольника
		 и прямоугольника, передаваемого в качестве входного параметра.


		 ![](../../Sub_Image.gif)
		 [IntersectsWith](IGxRect.IntersectsWith.htm)
		 Метод IntersectsWith
		 осуществляет проверку на пересечение текущего прямоугольника и
		 прямоугольника, передаваемого в качестве входного параметра.


		 ![](../../Sub_Image.gif)
		 [Offset](IGxRect.Offset.htm)
		 Метод Offset осуществляет
		 смещение прямоугольника на заданные значения по осям Х и Y.


		 ![](../../Sub_Image.gif)
		 [OffsetP](IGxRect.OffsetP.htm)
		 Метод OffsetP осуществляет
		 смещение целочисленного прямоугольника на значения координат точки,
		 передаваемой посредством входного параметра.


		 ![](../../Sub_Image.gif)
		 [OffsetS](IGxRect.OffsetS.htm)
		 Метод OffsetS осуществляет
		 смещение целочисленного прямоугольника на размер, передаваемый
		 посредством входного параметра.


		 ![](../../Sub_Image.gif)
		 [ToRectF](IGxRect.ToRectF.htm)
		 Метод ToRectF осуществляет
		 преобразование целочисленного прямоугольника в вещественный.
