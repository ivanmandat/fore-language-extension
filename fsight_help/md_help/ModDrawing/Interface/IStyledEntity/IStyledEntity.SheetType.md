# IStyledEntity.SheetType


## Синтаксис


SheetType: [StyleSheetType](../../Enums/StyleSheetType.htm);


## Описание


Свойство SheetType определяет
 тип используемой таблицы стилей.


## Комментарии


Таблица стилей может быть внешней или внутренней. Внешняя таблица стилей
 - это отдельный объект репозитория, внутренняя таблица стилей - это объект,
 содержащийся внутри отчета.
