# IStyledEntity.ContainerKey


## Синтаксис


ContainerKey: Integer;


## Описание


Свойство ContainerKey определяет
 ключ контейнера стилей, в котором содержится стиль.


## Комментарии


Для отвязки стиля от контейнера присвойте свойству значение «-1»;
