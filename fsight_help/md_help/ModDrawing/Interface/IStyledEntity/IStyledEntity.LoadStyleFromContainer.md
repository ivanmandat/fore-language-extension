# IStyledEntity.LoadStyleFromContainer


## Синтаксис


LoadStyleFromContainer(Value: [IStyleContainer](../IStyleContainer/IStyleContainer.htm));


## Параметры


Value. Контейнер, из которого
 импортируется стиль.


## Описание


Метод LoadStyleFromContainer
 импортирует стиль из указанного контейнера.


## Комментарии


Для экспорта стиля в указанный контейнер используйте метод [IStyledEntity.SaveStyleToContainer](IStyledEntity.SaveStyleToContainer.htm).
