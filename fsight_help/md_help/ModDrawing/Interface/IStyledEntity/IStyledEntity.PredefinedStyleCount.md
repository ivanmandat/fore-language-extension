# IStyledEntity.PredefinedStyleCount


## Синтаксис


PredefinedStyleCount: Integer;


## Описание


Свойство PredefinedStyleCount
 возвращает количество стандартных стилей.


## Комментарии


Для определения индекса стандартного стиля используйте свойство [IStyledEntity.PredefinedStyleIndex](IStyledEntity.PredefinedStyleIndex.htm).
