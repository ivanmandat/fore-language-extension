# IStyledEntity.PredefinedStylePreview


## Синтаксис


PredefinedStylePreview(Index: Integer): [IGxImage](../IGxImage/IGxImage.htm);


## Параметры


Index. Индекс стандартного
 стиля.


## Описание


Свойство PredefinedStylePreview
 возвращает изображение, используемое для предварительного просмотра стандартного
 стиля.


## Комментарии


Для получения индекса стандартного стиля используйте свойство [IStyledEntity.PredefinedStyleIndex](IStyledEntity.PredefinedStyleIndex.htm).
