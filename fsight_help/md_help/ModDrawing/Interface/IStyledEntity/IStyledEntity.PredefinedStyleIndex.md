# IStyledEntity.PredefinedStyleIndex


## Синтаксис


PredefinedStyleIndex: Integer;


## Описание


Свойство PredefinedStyleIndex
 определяет индекс стандартного стиля.


## Комментарии


Допустимое значение индекса находится в диапазоне [0, [IStyledEntity.PredefinedStyleCount](IStyledEntity.PredefinedStyleCount.htm)
 - 1].
