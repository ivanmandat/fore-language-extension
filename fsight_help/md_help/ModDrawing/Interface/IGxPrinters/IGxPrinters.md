# IGxPrinters


Сборка: Drawing;


## Описание


Интерфейс IGxPrinters содержит
 свойства коллекции принтеров, установленных в операционной системе.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IGxPrinters.Count.htm)
		 Свойство Count возвращает
		 количество принтеров, установленных в операционной системе.


		 ![](../../Property_Image.gif)
		 [Item](IGxPrinters.Item.htm)
		 Свойство Item возвращает
		 принтер, индекс которого передается посредством входного параметра.
