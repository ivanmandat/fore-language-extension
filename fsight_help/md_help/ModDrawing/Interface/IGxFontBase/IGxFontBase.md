# IGxFontBase


Сборка: Drawing;


## Описание


Интерфейс IGxFontBase используется
 для получения базовых настроек шрифта.


## Иерархия наследования


           IGxFontBase


## Комментарии


Данный интерфейс является базовым для интерфейса [IGxFont](../IGxFont/IGxFont.htm)
 и содержит свойства для работы со шрифтом.
