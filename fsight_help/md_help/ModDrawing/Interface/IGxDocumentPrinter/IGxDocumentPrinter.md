# IGxDocumentPrinter


Сборка: Drawing;


## Описание


Интерфейс IGxDocumentPrinter
 содержит свойства и методы абстрактного принтера документа.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Description](IGxDocumentPrinter.Description.htm)
		 Свойство Description
		 возвращает описание устройства, на котором производится печать.


		 ![](../../Property_Image.gif)
		 [Graphics](IGxDocumentPrinter.Graphics.htm)
		 Свойство Graphics возвращает
		 контекст принтера для рисования.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [EndDocument](IGxDocumentPrinter.EndDocument.htm)
		 Метод EndDocument завершает
		 постановку документа в очередь печати принтера.


		 ![](../../Sub_Image.gif)
		 [NewPage](IGxDocumentPrinter.NewPage.htm)
		 Метод NewPage отправляет
		 на печать новую страницу документа.


		 ![](../../Sub_Image.gif)
		 [NewSection](IGxDocumentPrinter.NewSection.htm)
		 Метод NewSection начинает
		 печать новой секции документа.


		 ![](../../Sub_Image.gif)
		 [ResetDocument](IGxDocumentPrinter.ResetDocument.htm)
		 Метод ResetDocument
		 инициализирует печать нового документа.


		 ![](../../Sub_Image.gif)
		 [SkipSection](IGxDocumentPrinter.SkipSection.htm)
		 Метод SkipSection пропускает
		 печать секции.


		 ![](../../Sub_Image.gif)
		 [StartDocument](IGxDocumentPrinter.StartDocument.htm)
		 Метод StartDocument
		 осуществляет постановку документа в очередь печати принтера.
