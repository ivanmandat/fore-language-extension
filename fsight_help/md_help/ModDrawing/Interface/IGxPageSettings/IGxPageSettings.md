# IGxPageSettings


Сборка: Drawing;


## Описание


Интерфейс IGxPageSettings содержит
 свойства и методы для настройки параметров страницы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Margins](IGxPageSettings.Margins.htm)
		 Свойство Margins возвращает
		 отступы страницы.


		 ![](../../Property_Image.gif)
		 [PaperFormat](IGxPageSettings.PaperFormat.htm)
		 Свойство PaperFormat
		 определяет формат страницы.


		 ![](../../Property_Image.gif)
		 [PaperOrientation](IGxPageSettings.PaperOrientation.htm)
		 Свойство PaperOrientation
		 определяет ориентацию страницы.


		 ![](../../Property_Image.gif)
		 [PaperSize](IGxPageSettings.PaperSize.htm)
		 Свойство PaperSize
		 возвращает размер страницы.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Assign](IGxPageSettings.Assign.htm)
		 Метод Assign устанавливает
		 параметры страницы в соответствии с параметрами, передаваемыми
		 посредством входного параметра.
