# IGxPrintComment


Сборка: Drawing;


## Описание


Интерфейс IGxPrintComment содержит
 свойства настройки печати.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Comment](IGxPrintComment.Comment.htm)
		 Свойство Comment возвращает
		 комментарий к печати.


		 ![](../../Property_Image.gif)
		 [Marking](IGxPrintComment.Marking.htm)
		 Свойство Marking определяет
		 ограничительную пометку.


		 ![](../../Property_Image.gif)
		 [PageCount](IGxPrintComment.PageCount.htm)
		 Свойство PageCount
		 определяет количество печатаемых страниц.


		 ![](../../Property_Image.gif)
		 [Sheets](IGxPrintComment.Sheets.htm)
		 Свойство Sheets определяет
		 листы, которые необходимо напечатать.


		 ![](../../Property_Image.gif)
		 [Succeeded](IGxPrintComment.Succeeded.htm)
		 Свойство Succeeded
		 определяет результат печати.
