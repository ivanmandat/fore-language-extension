# IGxScale.AutoCalculable


## Синтаксис


		AutoCalculable: Boolean;


## Описание


Свойство AutoCalculable определяет
 признак автоматического расчёта значений интервалов.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Значения интервалов шкалы рассчитываются автоматически;


	- False. Значения интервалов
	 шкалы задаются вручную.


Для определения значений интервалов шкалы вручную используйте свойство
 [IGxScale.Item](IGxScale.Item.htm).


Значения рассчитываются по количеству интервалов, заданному с помощью
 свойства [IGxScale.IntervalsCount](IGxScale.IntervalsCount.htm).
