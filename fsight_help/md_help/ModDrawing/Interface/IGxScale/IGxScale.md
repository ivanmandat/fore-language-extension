# IGxScale


Сборка: Drawing;


## Описание


Интерфейс IGxScale содержит
 параметры шкалы.


## Иерархия наследования


           IGxScale


## Комментарии


Шкала соответствует [метрикам](UiSelection.chm::/Selection/Dimension_type.htm)
 визуализатора и позволяет настраивать параметры цветовой и числовой шкалы
 с помощью интерфейсов [IGxColorScale](../IGxColorScale/IGxColorScale.htm)
 и [IGxNumericScale](../IGxNumericScale/IGxNumericScale.htm)
 соответственно.
