# IGxScale.Item


## Синтаксис


		Item(Index: Integer): Double;


## Параметры


Index. Индекс интервала шкалы.


## Описание


Свойство Item определяет значение
 интервала шкалы по индексу.


## Комментарии


Для определения значения интервала шкалы вручную свойство [IGxScale.AutoCalculable](IGxScale.AutoCalculable.htm)
 должно принимать значение False.
 Для автоматического определения значений интервалов шкалы используйте
 свойство [IGxScale.AutoCalculable](IGxScale.AutoCalculable.htm)
 со значением True.


Значения рассчитываются по количеству интервалов, заданному с помощью
 свойства [IGxScale.IntervalsCount](IGxScale.IntervalsCount.htm).
