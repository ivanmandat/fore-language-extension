# IGxPointF.IsEmpty


## Синтаксис


IsEmpty: Boolean;


## Описание


## Описание


Свойство IsEmpty возвращает
 признак соответствия точке с нулевыми координатами.


## Комментарии


Свойство возвращает значение True,
 если точка имеет координаты [0.0, 0.0] и False,
 если точка имеет какие-либо другие координаты.
