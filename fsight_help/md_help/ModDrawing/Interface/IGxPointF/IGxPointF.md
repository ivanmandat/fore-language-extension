# IGxPointF


Сборка: Drawing;


## Описание


Интерфейс IGxPointF содержит
 свойства и методы вещественной точки.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [IsEmpty](IGxPointF.IsEmpty.htm)
		 Свойство IsEmpty
		 возвращает признак соответствия точке с нулевыми координатами.


		 ![](../../Property_Image.gif)
		 [X](IGxPointF.X.htm)
		 Свойство X определяет координату
		 Х вещественной точки.


		 ![](../../Property_Image.gif)
		 [Y](IGxPointF.Y.htm)
		 Свойство Y определяет координату
		 Y вещественной точки.


## Методы


		  Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Equals](IGxPointF.Equals.htm)
		 Метод Equals проверяет
		 равенство данной точки с точкой, передаваемой посредством входного
		 параметра.


		 ![](../../Sub_Image.gif)
		 [Offset](IGxPointF.Offset.htm)
		 Метод Offset осуществляет
		 смещение точки на заданные значения по осям Х и Y.


		 ![](../../Sub_Image.gif)
		 [OffsetP](IGxPointF.OffsetP.htm)
		 Метод OffsetP осуществляет
		 смещение текущей точки на значения координат точки, передаваемой
		 посредством входного параметра.


		 ![](../../Sub_Image.gif)
		 [OffsetS](IGxPointF.OffsetS.htm)
		 Метод OffsetS осуществляет
		 смещение точки на размер, передаваемый посредством входного параметра.


		 ![](../../Sub_Image.gif)
		 [ToPoint](IGxPointF.ToPoint.htm)
		 Метод ToPoint осуществляет
		 преобразование вещественной точки в целочисленную.


		 ![](../../Sub_Image.gif)
		 [ToSize](IGxPointF.ToSize.htm)
		 Метод ToSize осуществляет
		 преобразование вещественной точки в вещественный размер.
