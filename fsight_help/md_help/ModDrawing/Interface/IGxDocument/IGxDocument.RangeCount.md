# IGxDocument.RangeCount


## Синтаксис


RangeCount: Integer;


## Описание


Свойство RangeCount возвращает
 количество диапазонов печати.


## Комментарии


Под диапазоном печати понимается весь документ, текущий лист или выделенный
 диапазон.
