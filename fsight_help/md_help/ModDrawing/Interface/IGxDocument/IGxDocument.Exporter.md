# IGxDocument.Exporter


## Синтаксис


Exporter: [IExporter](../IExporter/IExporter.htm);


## Описание


Свойство Exporter возвращает
 объект, содержащий дополнительные настройки для экспорта определенного
 вида объектов.


## Комментарии


Данное свойство должно быть приведено к определенному интерфейсу, в
 зависимости от экспортируемого объекта.
