# IGxDocument.Label


## Синтаксис


Label: [IGxDocumentLabel](../IGxDocumentLabel/IGxDocumentLabel.htm);


## Описание


Свойство Label возвращает параметры
 штампа, которым маркируются документы при экспорте и печати.


## Комментарии


Условия отображения штампов маркировки приведены в описании интерфейса
 [IGxStamp](../IGxStamp/IGxStamp.htm).
