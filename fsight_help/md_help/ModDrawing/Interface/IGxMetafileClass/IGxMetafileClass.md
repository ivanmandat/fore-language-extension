# IGxMetafileClass


Сборка: Drawing;


## Описание


Интерфейс IGxMetafileClass содержит
 статические методы для доступа к метафайлам.

		 объект, содержащий изображение, полученное из файла.


		 ![](../../Sub_Image.gif)
		 [FromStream](../IGxImageClass/IGxImageClass.FromStream.htm)
