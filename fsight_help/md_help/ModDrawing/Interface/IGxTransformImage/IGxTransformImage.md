# IGxTransformImage


Сборка: Drawing;


## Описание


Интерфейс IGxTransformImage
 содержит свойства и методы изображения, имеющего какие-либо изменения.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Unit](IGxTransformImage.Unit.htm)
		 Свойство Unit определяет
		 единицы измерения, используемые для задания параметров графического
		 изображения.
