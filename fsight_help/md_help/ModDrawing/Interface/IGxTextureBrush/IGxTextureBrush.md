# IGxTextureBrush


Сборка: Drawing;


## Описание


Интерфейс IGxTextureBrush содержит
 свойства текстурной кисти.

		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Image](IGxTextureBrush.Image.htm)
		 Свойство Image возвращает
		 изображение, используемое как текстура.


		 ![](../../Property_Image.gif)
		 [WrapMode](IGxTextureBrush.WrapMode.htm)
		 Свойство WrapMode определяет
		 метод наложения текстуры.
