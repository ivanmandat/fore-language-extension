# IGxRectF.IsEmpty


## Синтаксис


IsEmpty: Boolean;


## Описание


Свойство IsEmpty возвращает
 признак соответствия пустому вещественному прямоугольнику.


## Комментарии


Свойство возвращает значение True,
 если вещественный прямоугольник имеет нулевые координаты и размеры и False, если хотя бы один параметр
 отличен от нуля.
