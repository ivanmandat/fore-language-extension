# IGxRectF.Contains


## Синтаксис


Contains(X: Double; Y: Double): Boolean;


## Параметры


X. Координата Х точки.


Y. Координата Y точки.


## Описание


Метод Contains проверяет наличие
 точки с указанными координатами в текущем прямоугольнике.


## Комментарии


Метод возвращает значение True,
 если точка с координатами X и Y содержится в области вещественного прямоугольника
 и False, ели точка отсутствует.


## Пример


	Sub UserProc;

	Var

	    Rect: IGxRectF;

	Begin

	    Rect := New GxRectF.Create(0, 0, 10.5, 10.5);

	    Debug.WriteLine(Rect.Contains(Math.RandBetween(-10, 20), Math.RandBetween(-10, 20)));

	End Sub UserProc;


При выполнение примера будет создан вещественный прямоугольник. В консоль
 среды разработки будет выведен признак наличия точки со случайными координатами
 в прямоугольнике.
