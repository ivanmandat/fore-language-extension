# IGxDocumentLabel.LabelDescription


## Синтаксис


LabelDescription: String;


## Описание


Свойство LabelDescription возвращает
 описание, которое будет использовано в тексте метки в правом верхнем углу
 первого листа при маркировке объектов при экспорте и печати.


## Комментарии


Данное описание соответствует описанию уровня безопасности, который
 установлен для объекта. Если описание для уровня пусто, то при маркировке
 будет использовано наименование уровня.
