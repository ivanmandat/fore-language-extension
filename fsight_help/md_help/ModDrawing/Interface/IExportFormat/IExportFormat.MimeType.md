# IExportFormat.MimeType


## Синтаксис


MimeType: String;


## Описание


Свойство MimeType определяет
 тип MIME для данного формата.


## Комментарии


Типы MIME используются для сопоставления с выбранным типом файлов.


В качестве значения свойства MIME может использоваться форма приложение/тип,
 где приложение предоставляет
 приложение либо класс приложений, а тип
 предоставляет уникальный тип MIME. Использование прямой косой черты обязательно.
