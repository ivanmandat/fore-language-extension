# IGxICMManager.ICMProfile


## Синтаксис


ICMProfile: String;


## Описание


Свойство ICMProfile определяет
 наименование цветового профиля, в соответствии с которым будет осуществляться
 преобразование.


## Комментарии


В данном свойстве задается наименование цветового профиля без расширения,
 например: «sRGB Color Space Profile».
