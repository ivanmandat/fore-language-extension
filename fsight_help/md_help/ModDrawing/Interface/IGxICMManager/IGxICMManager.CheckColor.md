# IGxICMManager.CheckColor


## Синтаксис


CheckColor(Source: [IGxColor](../IGxColor/IGxColor.htm)):
 Integer;


## Параметры


Source - цвет, который необходимо
 проверить на воспроизводимость.


## Описание


Метод CheckColor осуществляет
 проверку указанного цвета на воспроизводимость при использовании указанного
 цветового профиля.


## Комментарии


Цветовой профиль задается в свойстве [ICMProfile](IGxICMManager.ICMProfile.htm).
 Данный метод возвращает относительное значение в диапазоне от 0 до 255.
 Результирующее значение метода равное 0 означает нахождение воспроизведенного
 цвета в гамме цветов исходного цвета. Значение отличное от 0 указывает
 на то, что воспроизведенный цвет будет расположен за пределами гаммы исходного
 цвета. Относительное значение в пределах от 0 до 255 показывает как значительно
 отклонение цвета от гаммы исходного цвета.
