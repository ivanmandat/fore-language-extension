# IGxBulkSettings.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет,
 будет ли выполняться пакетная операция.


## Комментарии


Если свойству установлено значение True,
 то для соответствующего объекта будет выполняться пакетная печать/экспорт.
 При этом также необходимо будет определить специфические параметры пакетной
 операции.
