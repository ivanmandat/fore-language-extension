 базовые свойства для определения параметров пакетных операций.


## Иерархия наследования


IGxBulkSettings


## Комментарии


Пакетные операции - это операции экспорта и печати для нескольких вариантов
 объектов отчетности. Пакетные операции поддерживаются для регламентных
 и экспресс-отчетов. Варианты отчетов формируются по отметке фиксированных
 измерений. Необходимыми условиями для выполнения пакетной операции является:


	- Для экспресс-отчетов.
	 Наличие общих фиксированных измерений на разных листах отчета.


	- Для регламентных отчетов.
	 Наличие элементов управления, общих для всех листов отчета. Данные
	 элементы управления должны быть настроены на работу с фиксированными
	 измерениями.


Данный интерфейс является базовым для следующих интерфейсов:


	- [IEaxBulkSettings](KeExpress.chm::/Interface/IEaxBulkSettings/IEaxBulkSettings.htm);


	- [IPrxBulkSelection](KeReport.chm::/Interface/IPrxBulkSelection/IPrxBulkSelection.htm).
