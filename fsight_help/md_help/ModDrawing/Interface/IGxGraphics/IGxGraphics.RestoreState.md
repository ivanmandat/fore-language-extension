# IGxGraphics.RestoreState


## Синтаксис


RestoreState(Value: Integer);


## Параметры


Value.
 Индекс сохраненного состояния графического объекта.


## Описание


Метод RestoreState восстанавливает
 состояние графического объекта из стека.


Сохранение состояния графического объекта осуществляется посредством
 метода [SaveState](IGxGraphics.SaveState.htm).
