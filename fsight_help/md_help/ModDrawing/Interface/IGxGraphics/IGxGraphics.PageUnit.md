# IGxGraphics.PageUnit


## Синтаксис


PageUnit: [GxUnit](../../Enums/GxUnit.htm);


## Описание


Свойство PageUnit определяет
 единицы измерения графического объекта.


## Комментарии


В качестве значения данного свойства может быть установлено одно из
 значений [GxUnit](../../Enums/GxUnit.htm), отличное от [GxUnit.World](../../Enums/GxUnit.htm).


Примечание.
 При попытке установить значение [GxUnit.World](../../Enums/GxUnit.htm)
 будет произведена автоматическая замена единиц измерения на [GxUnit.Display](../../Enums/GxUnit.htm).
