# IGxGraphics.MeasureTextRF


## Синтаксис


MeasureTextRF(Text: String; Font: [IGxFont](../IGxFont/IGxFont.htm);
 Rect: [IGxRectF](../IGxRectF/IGxRectF.htm); Format: [IGxStringFormat](../IGxStringFormat/IGxStringFormat.htm)):
 [IGxRectF](../IGxRectF/IGxRectF.htm);


## Параметры


Text. Текст.


Font. Шрифт текста.


Rect. Прямоугольник, задающий
 максимальную область размещения для текста. Если задан прямоугольник с
 нулевыми размерами - осуществляется расчет области, занимаемой указанным
 текстом с учетом установленных параметров.


Format. Формат текста.


## Описание


Метод MeasureTextRF вычисляет
 размер текста с заданным форматом.
