# IGxGraphics


Сборка: Drawing;


## Описание


Интерфейс IGxGraphics содержит
 свойства и методы для работы с графическим контекстом объекта.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CompositingQuality](IGxGraphics.CompositingQuality.htm)
		 Свойство CompositingQuality
		 определяет качество отрисовки изображений.


		 ![](../../Property_Image.gif)
		 [DpiX](IGxGraphics.DpiX.htm)
		 Свойство DpiX возвращает
		 горизонтальное разрешение графического объекта.


		 ![](../../Property_Image.gif)
		 [DpiY](IGxGraphics.DpiY.htm)
		 Свойство DpiY возвращает
		 вертикальное разрешение графического объекта.


		 ![](../../Property_Image.gif)
		 [InterpolationMode](IGxGraphics.InterpolationMode.htm)
		 Свойство InterpolationMode
		 определяет режим интерполяции графического объекта.


		 ![](../../Property_Image.gif)
		 [PageUnit](IGxGraphics.PageUnit.htm)
		 Свойство PageUnit определяет
		 единицы измерения графического объекта.


		 ![](../../Property_Image.gif)
		 [SmoothingMode](IGxGraphics.SmoothingMode.htm)
		 Свойство SmoothingMode
		 определяет режим сглаживания графических примитивов.


		 ![](../../Property_Image.gif)
		 [TextRenderingHint](IGxGraphics.TextRenderingHint.htm)
		 Свойство TextRenderingHint
		 определяет режим вывода текста.
