# IGxGraphics.SetScale


## Синтаксис


SetScale(X: Double, Y: Double);


## Параметры


X. Масштаб по оси X.


Y. Масштаб по оси Y.


## Описание


Метод SetScale устанавливает
 масштаб графического изображения.


## Комментарии


При указании значений X и Y масштаб умножается на соответствующие
 значения, т.е. если в качестве значений параметров указать 2, то масштаб
 изображения будет увеличен в два раза или на 100%. Для уменьшения масштаба
 укажите в качестве значений параметров значения меньше единицы.


Необходимо учитывать, что данный метод изменяет масштаб относительно
 уже установленного масштаба, а не относительно базового масштаба изображения.
 Поэтому после изменения масштаба в два раза для возврата к исходному масштабу
 нужно указать в качестве значения параметров не 1, а 0,5. Также для возврата
 к базовым настройкам изображения можно использовать метод [ResetTransform](IGxGraphics.ResetTransform.htm).
