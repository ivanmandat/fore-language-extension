# IVisStyleProvider


Сборка: Drawing;


## Описание


Интерфейс IVisStyleProvider
 предназначен для доступа к информации о палитрах стиля визуализаторов
 «Диаграмма» и «Пузырьковая диаграмма» через COM-интерфейс.


## Комментарии


Палитра - это набор цветов, которые используется для заливки частей
 визуализатора: рядов, пузырьков, областей и т.д. Палитра по умолчанию
 имеется у всех диаграмм.
