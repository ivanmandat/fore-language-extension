# IVisStyleProvider.PaletteIDs


## Синтаксис


PaletteIDs: Array;


## Описание


Метод PaletteIDs используется
 для получения массива строк с идентификаторами альтернативных наборов
 палитр.


## Комментарии


Для получения массива основных цветов по идентификатору палитры используйте
 метод [IVisStyleProvider.PaletteColors](IVisStyleProvider.PaletteColors.htm).
