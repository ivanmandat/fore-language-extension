# IStyleContainer


Сборка: Drawing;


## Описание


Интерфейс IStyleContainer содержит
 свойства и методы контейнера стиля, содержащегося в таблице стилей.


## Иерархия наследования


           IStyleContainer


## Комментарии


Для получения контейнеров стиля из таблицы стилей используйте свойства
 и методы интерфейса [IStyleSheet](../IStyleSheet/IStyleSheet.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Id](IStyleContainer.Id.htm)
		 Свойство Id определяет
		 идентификатор контейнера стиля.


		 ![](../../Property_Image.gif)
		 [Key](IStyleContainer.Key.htm)
		 Свойство Key возвращает
		 ключ контейнера стиля.


		 ![](../../Property_Image.gif)
		 [Name](IStyleContainer.Name.htm)
		 Свойство Name
		 определяет наименование контейнера.


		 ![](../../Property_Image.gif)
		 [Preview](IStyleContainer.Preview.htm)
		 Свойство Preview
		 определяет сохраненное представление стиля, содержащегося в контейнере.


		 ![](../../Property_Image.gif)
		 [Sheet](IStyleContainer.Sheet.htm)
		 Свойство Sheet возвращает
		 объект-родитель.


		 ![](../../Property_Image.gif)
		 [Tag](IStyleContainer.Tag.htm)
		 Свойство Tag возвращает
		 текстовую метку стиля, загруженного в контейнер.
