# IStyleContainer.Tag


## Синтаксис


Tag: String;


## Описание


Свойство Tag возвращает текстовую
 метку стиля, загруженного в контейнер.


## Комментарии


Если в контейнере отсутствует стиль, то свойство возвращает пустую строку.
