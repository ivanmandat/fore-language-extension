# IStyleContainer.Preview


## Синтаксис

## Описание


Свойство Preview определяет
 сохраненное представление стиля, содержащегося в контейнере.


## Комментарии


Сохраненное представление генерируется автоматически после сохранения
 стиля в контейнер с помощью метода [IStyledEntity.SaveStyleToContainer](../IStyledEntity/IStyledEntity.SaveStyleToContainer.htm).
