# IStyleContainer.Clear


## Синтаксис


Clear([bAll: Boolean = False]);


## Параметры


bAll. Параметр, определяющий, будет ли очищаться свойство [Tag](IStyleContainer.Tag.htm) при очистке контейнера. По умолчанию установлено значение False, при этом свойство [Tag](IStyleContainer.Tag.htm) не очищается.


## Описание


Метод Clear осуществляет очистку контейнера стиля.


## Комментарии


При очистке контейнера во всех свойствах стиля устанавливаются значения по умолчанию.
