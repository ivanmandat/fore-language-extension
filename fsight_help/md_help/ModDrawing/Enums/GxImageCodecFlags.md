# GxImageCodecFlags


## Описание


Перечисление GxImageCodecFlags
 содержит атрибуты кодека изображения.


Используется следующим свойством:


	- [IGxImageFormat.Flags](../Interface/IGxImageFormat/IGxImageFormat.Flags.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 Encoder.
		 Кодек поддерживает кодирование (сохранение).


		 2
		 Decoder.
		 Кодек поддерживает декодирование (чтение).


		 4
		 SupportBitmap.
		 Кодек поддерживает растровые изображения (точечные рисунки).


		 8
		 SupportVector.
		 Кодек поддерживает векторные изображения (метафайлы).


		 16
		 SeekableEncode.
		 Кодировщику требуется поток вывода с возможностью поиска.


		 32
		 BlockingDecode.
		 В процессе декодирования декодер выполняет блокировку.


		 65536
		 Builtin.
		 Кодек встроен в GDI+.


		 131072
		 System.
		 Не используется.


		 262144
		 User.
		 Не используется.
