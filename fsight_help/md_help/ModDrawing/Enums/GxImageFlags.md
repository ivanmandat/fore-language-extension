# GxImageFlags


## Описание


Перечисление GxImageFlags содержит
 параметры изображения.


Используется следующим свойством:


	- [IGxImage.Flags](../Interface/IGxImage/IGxImage.Flags.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None.
		 Информация о формате отсутствует.


		 1
		 Scalable.
		 Данные о точках являются масштабируемыми.


		 2
		 HasAlpha.
		 Данные о точках содержат сведения об альфа-составляющей.


		 4
		 HasTranslucent.
		 Указывает, что данные о точках содержат значения альфа, отличающиеся
		 от 0 (прозрачный) и 255 (непрозрачный).


		 8
		 PartiallyScalable.
		 Данные о точках частично масштабируемы, но существуют некоторые
		 ограничения.


		 16
		 ColorSpaceRGB.
		 Данные о точках используют цветовое пространство RGB.


		 32
		 ColorSpaceCMYK.
		 Данные о точках используют цветовое пространство CMYK.


		 64
		 ColorSpaceGRAY.
		 Данные о точках являются черно-белыми.


		 128
		 ColorSpaceYCBCR.
		 Указывает, что изображение сохранено с помощью цветового пространства
		 YCBCR.


		 256
		 ColorSpaceYCCK.
		 Указывает, что изображение сохранено с помощью цветового пространства
		 YCCK.


		 4096
		 HasRealDPI.
		 Указывает, что в изображении хранится информация о разрешении
		 в точках на дюйм.


		 8192
		 HasRealPixelSize.
		 Указывает, что в изображении хранится размер в точках.


		 65536
		 ReadOnly.
		 Данные о точках доступны только для чтения.


		 131072
		 Caching.
		 Данные о точках могут быть кешированы для более быстрого доступа.
