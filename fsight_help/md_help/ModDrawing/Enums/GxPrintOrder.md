# GxPrintOrder


## Описание


Перечисление GxPrintOrder содержит
 варианты порядка вывода страниц на печать.


Используется следующими свойством:


	- [ITabPageSettings.PrintOrder](TabSheet.chm::/Interface/ITabPageSettings/ITabPageSettings.PrintOrder.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 FromLeftToRight. Слева
		 направо, сверху вниз.


		 1
		 FromTopToBottom. Сверху
		 вниз, слева направо.
