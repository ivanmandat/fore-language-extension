# GxStringAlignment


## Описание


Перечисление GxStringAlignment содержит способы горизонтального выравнивания текста.


Используется следующими свойствами и методами:


-
[IGxStringFormat.Alignment](../Interface/IGxStringFormat/IGxStringFormat.Alignment.htm);


-
[IGxStringFormat.LineAlignment](../Interface/IGxStringFormat/IGxStringFormat.LineAlignment.htm);


-
[IGxTitle.TextAlignment](../Interface/IGxTitle/IGxTitle.TextAlignment.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Near. Текст выравнивается по ближнему краю. При размещении слева направо ближним краем будет левый. При размещении справа налево ближним краем будет правый.


 1
 Center. Текст выравнивается по центру прямоугольника размещения.


 2
 Far. Текст выравнивается по дальнему краю от исходного положения прямоугольника размещения. При размещении слева направо дальним положением будет правое. При размещении справа налево дальним положением будет левое.
