# GxStringTrimming


## Описание


Перечисление GxStringTrimming
 содержит режимы отсечения текста.


Используется следующим свойством:


	- [IGxStringFormat.Trimming](../Interface/IGxStringFormat/IGxStringFormat.Trimming.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Значение определяется
		 ядром системы.


		 1
		 Character. Текст обрезается
		 по ближайшему знаку.


		 2
		 Word. Текст обрезается
		 по ближайшему слову.


		 3
		 EllipsisCharacter.
		 Текст обрезается по ближайшему знаку и в конце обрезанной строки
		 вставляется троеточие.


		 4
		 EllipsisWord. Текст
		 обрезается по ближайшему слову и в конце обрезанной строки вставляется
		 троеточие.


		 5
		 EllipsisPath. Из обрезанных
		 строк удаляется центр и заменяется троеточием. Этот алгоритм сохраняет
		 максимально возможную часть отделенного косой чертой последнего
		 сегмента строки.
