# GxDrawTextFormat


## Описание


Перечисление GxDrawTextFormat
 содержит режим отрисовки текста в листах отчета, автофигурах отчета и
 диаграммах отчета.


Используется следующими свойствами и методами:


	- [IPrxReportOptions.DrawTextFormat](KeReport.chm::/Interface/IPrxReportOptions/IPrxReportOptions.DrawTextFormat.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Default_. Значение
		 по умолчанию. Отрисовка средствами GDI+. Не поддерживает некоторые
		 шрифты, доступные в Wpf.


		 1
		 Wpf. Отрисовка
		 через WPF.

		 Rtf. Отрисовка через
		 RTF.
