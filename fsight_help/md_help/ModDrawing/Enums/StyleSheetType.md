# StyleSheetType


## Описание


Перечисление StyleSheetType
 используется для определения типа таблицы стилей.


Используется следующими свойствами и методами:


	- [IStyledEntity.SheetType](../Interface/IStyledEntity/IStyledEntity.SheetType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 MetabaseObject. Внешняя
		 таблица стилей. Является отдельным объектом репозитория.


		 1
		 InternalObject. Внутренняя
		 таблица стилей. Является объектом, содержащимся внутри отчета.
