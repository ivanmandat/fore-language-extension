# GxKnownColor


## Описание


Перечисление GxKnownColor содержит
 стандартные цвета, которые необходимо получить.


Используется следующим методом:


	- [IGxColorClass.FromKnownColor](../Interface/IGxColorClass/IGxColorClass.FromKnownColor.htm).
