# GxCompositingQuality


## Описание


Перечисление GxCompositingQuality
 содержит варианты качества отрисовки изображений.


Используется следующим свойством:


	- [IGxGraphics.CompositingQuality](../Interface/IGxGraphics/IGxGraphics.CompositingQuality.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 -1
		 Invalid.
		 Недопустимое качество.


		 0
		 Standard.
		 Качество по умолчанию.


		 1
		 HighSpeed.
		 Быстрая низкокачественная компоновка.


		 2
		 HighQuality.
		 Медленная высококачественная компоновка.


		 3
		 GammaCorrected.
		 Используется гамма-коррекция.


		 4
		 AssumeLinear.
		 Предполагаются линейные значения.
