# GxPenAlignment


## Описание


Перечисление GxPenAlignment
 содержит варианты выравнивания пера.


Используется следующим свойством:


	- [IGxPen.Alignment](../Interface/IGxPen/IGxPen.Alignment.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Center. Перо располагается
		 по центру над теоретической линией.


		 1
		 Inset. Перо располагается
		 внутри теоретической линии.
