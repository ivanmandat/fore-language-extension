# GxPaperOrientation


## Описание


Перечисление GxPaperOrientation
 содержит варианты ориентации страницы.


Используется следующим свойством:


	- [IGxPageSettings.PaperOrientation](../Interface/IGxPageSettings/IGxPageSettings.PaperOrientation.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 Portrait. Книжная.


		 2
		 Landscape. Альбомная.
