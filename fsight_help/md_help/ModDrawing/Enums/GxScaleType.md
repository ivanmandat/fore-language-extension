# GxScaleType


## Описание


Перечисление GxScaleType используется
 для определения типа шкалы.


Используется следующим свойством:


	- [IGxScale.Type](../Interface/IGxScale/IGxScale.Type.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Linear. Линейная шкала.
		 Количество интервалов равно количеству значений в диапазоне. Данное
		 количество можно получить в свойстве [IGxScale.IntervalsCount](../Interface/IGxScale/IGxScale.IntervalsCount.htm).


		 1
		 EqualValues. Шкала
		 с заданным количеством интервалов. В каждом интервале шкалы
		 одинаковое количество значений. Количество интервалов задается
		 в свойстве [IGxScale.IntervalsCount](../Interface/IGxScale/IGxScale.IntervalsCount.htm).
