# GxQualityMode


## Описание


Перечисление GxQualityMode содержит варианты общего качества при визуализации графических объектов.


## Допустимые значения


 Значение
 Краткое описание


 -1
 Invalid. Указывает недопустимый режим.


 0
 Standard. Задаёт режим, используемый по умолчанию.


 1
 Low. Задаёт медленную высококачественную визуализацию.


 2
 High. Задаёт быструю высококачественную визуализацию.
