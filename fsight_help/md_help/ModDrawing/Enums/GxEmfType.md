# GxEmfType


## Описание


Перечисление GxEmfType содержит
 типы метафайлов.


Тип метафайла влияет на вид пунктирных линий, экспортируемых в EMF.


Используется следующими свойствами и методами:


	- [IPrxReportExporter.EmfType](KeReport.chm::/Interface/IPrxReportExporter/IPrxReportExporter.EmfType.htm);


	- [IAdhocReportExporter.EmfType](KeAdhoc.chm::/Interface/IAdhocReportExporter/IAdhocReportExporter.EmfType.htm);


	- [ITabSheetExporter.EmfType](TabSheet.chm::/Interface/ITabSheetExporter/ITabSheetExporter.EmfType.htm);


	- [IExAnalyzerExporter.EmfType](KeExpress.chm::/Interface/IExAnalyzerExporter/IExAnalyzerExporter.EmfType.htm);


	- [IMap.SaveMetafileToFile](Map.chm::/Interface/IMap/IMap.SaveMetafileToFile.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 EmfOnly. EMF-файл.


		 1
		 EmfPlusOnly. Файл EMF+.


		 2
		 EmfPlusDual. Файл EMF+
		 Dual.
