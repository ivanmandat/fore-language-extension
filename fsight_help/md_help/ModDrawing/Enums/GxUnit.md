# GxUnit


## Описание


Перечисление GxUnit содержит
 единицы измерения графического объекта.


Используется следующими свойствами и методами:


	- [GxFont.Create](../Class/GxFont/GxFont.Create.htm);


	- [IGxFont.Unit](../Interface/IGxFont/IGxFont.Unit.htm);


	- [IGxGraphics.PageUnit](../Interface/IGxGraphics/IGxGraphics.PageUnit.htm);


	- [IGxGraphics.DrawImageTransparentF](../Interface/IGxGraphics/IGxGraphics.DrawImageTransparentF.htm);


	- [IGxTransformImage.Unit](../Interface/IGxTransformImage/IGxTransformImage.Unit.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 World.
		 В качестве единицы измерения используются единицы мировой системы
		 координат.


		 1
		 Display.
		 В качестве единицы измерения используются единицы измерения устройства
		 отображения. Обычно это точки для видеодисплеев и 1/100 дюйма
		 для принтеров.


		 2
		 Pixel.
		 В качестве единицы измерения используется точка устройства.


		 3
		 Point.
		 В качестве единицы измерения используется пункт (1/72 дюйма).


		 4
		 Inch.
		 В качестве единицы измерения используются дюймы.


		 5
		 Document.
		 В качестве единицы измерения используются единицы документа (1/300
		 дюйма).


		 6
		 Millimeter.
		 В качестве единицы измерения используются миллиметры.
