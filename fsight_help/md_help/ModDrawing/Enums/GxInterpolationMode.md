# GxInterpolationMode


## Описание


Перечисление GxInterpolationMode
 содержит режимы интерполяции графического объекта.


Используется следующим свойством:


	- [IGxGraphics.InterpolationMode](../Interface/IGxGraphics/IGxGraphics.InterpolationMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 -1
		 Invalid. Недопустимый
		 режим.


		 0
		 Standard.
		 Режим, используемый по умолчанию.


		 1
		 LowQuality.
		 Низкокачественная интерполяция.


		 2
		 HighQuality.
		 Высококачественная интерполяция.


		 3
		 Bilinear.
		 Билинейная интерполяция.


		 4
		 Bicubic.
		 Бикубическая интерполяция.


		 5
		 NearestNeighbor.
		 Интерполяция по ближайшим соседним элементам.


		 6
		 HighQualityBilinear.
		 Высококачественная билинейная интерполяция.


		 7
		 HighQualityBicubic.
		 Высококачественная бикубическая интерполяция.
