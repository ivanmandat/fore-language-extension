# GxTextRenderingHint


## Описание


Перечисление GxTextRenderingHint
 содержит режимы вывода текста.


Используется следующим свойством:


	- [IGxGraphics.TextRenderingHint](../Interface/IGxGraphics/IGxGraphics.TextRenderingHint.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 SystemDefault. Все
		 знаки будут отображены с помощью базовых знаков с системным хинтованием,
		 используемым по умолчанию. Текст будет отображен с параметрами
		 сглаживания шрифта, указанными для системы.


		 1
		 SingleBitPerPixelGridFit
		 . Все знаки будут отображены с помощью базовых знаков. Хинтование
		 применяется для повышения качества отображения знаков на основных
		 штрихах и изгибах.


		 2
		 SingleBitPerPixel.
		 Все знаки будут отображены с помощью базовых знаков. Хинтование
		 отключено.


		 3
		 AntiAliasGridFit. Все
		 знаки будут отображены с помощью сглаженных базовых знаков с хинтованием.
		 Качество существенно повышается за счет снижения производительности.


		 4
		 AntiAlias. Все знаки
		 будут отображены с помощью сглаженных базовых знаков без хинтования.
		 Сглаживание позволяет повысить качество. Поскольку функция хинтования
		 отключена, может быть заметна разница в ширине основных штрихов.


		 5
		 ClearTypeGridFit. Все
		 знаки будут отображены с помощью базовых знаков CT с хинтованием.
		 Наилучшее качество. Применяется для раскрытия преимуществ шрифтов
		 ClearType.
