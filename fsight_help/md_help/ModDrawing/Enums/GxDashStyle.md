# GxDashStyle


## Описание


Перечисление GxDashStyle содержит
 стили изображения линий.


Используется следующими свойствами:


	- IGxPen.DashStyle;


	- [IGxPenBase.DashStyle](../Interface/IGxPenBase/IGxPenBase.DashStyle.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 -1
		 Undefined.
		 Стиль не определен.


		 0
		 Solid. Сплошной.


		 1
		 Dash.
		 Штрих.


		 2
		 Dot. Пунктир.


		 3
		 DashDot. Штрих-пунктир.


		 4
		 DashDotDot. Штрих-пунктир-пунктир.


		 5
		 Custom.
		 Пользовательский стиль штриха.


		 10
		 None.
		 Без стиля.
