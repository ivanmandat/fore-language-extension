# GxWrapMode


## Описание


Перечисление GxWrapMode содержит варианты накладки текстуры меньшей по размеру, чем заполняемая область.


Используется следующим свойством:


-
[IGxTextureBrush.WrapMode](../Interface/IGxTextureBrush/IGxTextureBrush.WrapMode.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Tile. Производит наложение текстуры.


 1
 TileFlipX. Разворачивает текстуру горизонтально, а затем производит ее наложение.


 2
 TileFlipY. Разворачивает текстуру вертикально, а затем производит ее наложение.


 3
 TileFlipXY. Разворачивает текстуру горизонтально и вертикально, а затем производит ее наложение.


 4
 Clamp. Фиксирует текстуру на границе объекта.
