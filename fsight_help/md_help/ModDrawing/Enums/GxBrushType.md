# GxBrushType


## Описание


Перечисление GxBrushType содержит
 доступные варианты кистей, которые могут использоваться для отрисовки
 каких-либо объектов.


Используется следующим свойством:


	- [IGxBrush.BrushType](../Interface/IGxBrush/IGxBrush.BrushType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 SolidColor. Сплошная.


		 1
		 HatchFill.
		 Штриховая.


		 2
		 TextureFill.
		 Текстурная.


		 3
		 PathGradient.
		 Цветовой градиент. Данный тип не поддерживается.


		 4
		 LinearGradient.
		 Линейно-градиентная (двухцветный градиент).
