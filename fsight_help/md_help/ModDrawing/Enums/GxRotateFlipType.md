# GxRotateFlipType


## Описание


Перечисление GxRotateFlipType
 содержит типы вращения и отражения изображения.


Используется следующим методом:


	- [IGxImage.RotateFlip](../Interface/IGxImage/IGxImage.RotateFlip.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 RotateNoneFlipNone.
		 Не поворачивать.


		 1
		 Rotate90FlipNone. Поворот
		 на 90 градусов по часовой стрелке.


		 2
		 Rotate180FlipNone.
		 Поворот на 180 градусов по часовой стрелке.


		 3
		 Rotate270FlipNone.
		 Поворот на 270 градусов по часовой стрелке.


		 4
		 RotateNoneFlipX. Отразить
		 слева направо.


		 5
		 Rotate90FlipX. Поворот
		 на 90 градусов по часовой стрелке и отразить слева направо.


		 6
		 Rotate180FlipX. Поворот
		 на 180 градусов по часовой стрелке и отразить слева направо.


		 7
		 Rotate270FlipX. Поворот
		 на 270 градусов по часовой стрелке и отразить слева направо.
