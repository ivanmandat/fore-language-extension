# GxPixelFormat


## Описание


Перечисление GxPixelFormat содержит
 форматы пикселей битового изображения.


Используется следующими свойствами и методами:


	- [IGxImage.PixelFormat](../Interface/IGxImage/IGxImage.PixelFormat.htm);


	- [IGxBitmapClass.CreateNew](../Interface/IGxBitmapClass/IGxBitmapClass.CreateNew.htm).

## Комментарии


Для избежания в дальнейшем проблем с отображением фона создаваемых изображений
 рекомендуется использовать Format24bppRgb для изображений, сохраняемых
 в формате BMP, и Format32bppArgb для изображений, сохраняемых в формате
 PNG.
