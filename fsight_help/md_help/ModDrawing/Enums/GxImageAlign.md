# GxImageAlign


## Описание


Перечисление GxImageAlign содержит
 варианты положения изображения относительно текста.


Используется следующим свойством:


	- [IDataGridStyle.ImageAlign](KeExtCtrls.chm::/Interface/IDataGridStyle/IDataGridStyle.ImageAlign.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Left.
		 С левой стороны.


		 1
		 Right.
		 С правой стороны.


		 2
		 Top. Вверху
		 над текстом.


		 3
		 Bottom.
		 Внизу под текстом.
