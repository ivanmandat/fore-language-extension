# GxVerticalAlignment


## Описание


Перечисление GxVerticalAlignment содержит способы вертикального выравнивания текста.


Используется следующими свойствами и методами:


-
[IGxTitle.VerticalAlignment](../Interface/IGxTitle/IGxTitle.VerticalAlignment.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Top. Текст выравнивается по верхнему краю.


 1
 Center. Текст выравнивается по центру прямоугольника размещения.


 2
 Bottom. Текст выравнивается по нижнему краю.
