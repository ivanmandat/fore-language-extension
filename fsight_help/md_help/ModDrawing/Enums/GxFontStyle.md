# GxFontStyle


## Описание


Перечисление GxFontStyle содержит
 стили шрифта.


Используется следующими свойствами и методами:


	- [GxFont.Create](../Class/GxFont/GxFont.Create.htm);


	- [IGxFontBase.Style](../Interface/IGxFontBase/IGxFontBase.Style.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Regular. Обычный.


		 1
		 Bold. Полужирный.


		 2
		 Italic. Курсив.


		 3
		 BoldItalic. Полужирный
		 курсив.


		 4
		 Underline. Подчеркнутый.


		 8
		 Strikeout. Зачеркнутый.
