# GxHatchStyle


## Описание


Перечисление GxHatchStyle содержит типы штриховки.


Используется следующим свойством:


-
[IGxHatchBrush.HatchStyle](../Interface/IGxHatchBrush/IGxHatchBrush.HatchStyle.htm).
