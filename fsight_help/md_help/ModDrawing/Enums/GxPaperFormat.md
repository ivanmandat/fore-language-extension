# GxPaperFormat


## Описание


Перечисление GxPaperFormat содержит
 форматы страницы.


Используется следующим свойством:


	- [IGxPageSettings.PaperFormat](../Interface/IGxPageSettings/IGxPageSettings.PaperFormat.htm).
