# GxDashCap


## Описание


Перечисление GxDashCap содержит
 виды окончания пунктиров, ограничивающих пунктирные линии.


Используется следующим свойством:


	- [IGxPen.DashCap](../Interface/IGxPen/IGxPen.DashCap.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Flat.
		 Квадратное завершение для обоих концов каждого штриха.


		 2
		 Round.
		 Круглое завершение для обоих концов каждого штриха.


		 3
		 Triangle.
		 Треугольное завершение для обоих концов каждого штриха.
