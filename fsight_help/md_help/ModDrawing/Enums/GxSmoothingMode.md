# GxSmoothingMode


## Описание


Перечисление GxSmoothingMode
 содержит режимы сглаживания графических примитивов.


Используется следующим свойством:


	- [IGxGraphics.SmoothingMode](../Interface/IGxGraphics/IGxGraphics.SmoothingMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 -1
		 Invalid. Недопустимый
		 режим.


		 0
		 Standard. Режим, используемый
		 по умолчанию.


		 1
		 HighSpeed. Быстрая
		 низкокачественная обработка.


		 2
		 HighQuality. Медленное
		 высококачественное изображение.


		 3
		 None. Отсутствие сглаживания.


		 4
		 AntiAlias8x4. Визуализация
		 со сглаживанием 8*4.


		 5
		 AntiAlias8x8. Визуализация
		 со сглаживанием 8*8.
