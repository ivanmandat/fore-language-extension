# TabSheetSettings.CellAnimationColor


## Синтаксис


CellAnimationColor: [PP.Color](dhtmlCommon.chm::/Classes/PP/Color/Color.htm);


## Описание


Свойство CellAnimationColor
 определяет конечный цвет измененной ячейки.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setCellAnimationColor,
 а возвращается с помощью метода getCellAnimationColor.
