# TabSheetSettings.VisibleRowHeaders


## Синтаксис


VisibleRowHeaders: Boolean


## Описание


Свойство VisibleRowHeaders**
 управляет видимостью заголовков строк таблицы.


## Комментарии


Значение свойства устанавливается с помощью метода setVisibleRowHeaders,
 а возвращается - с помощью метода getVisibleRowHeaders.
 Из JSON значение свойства задать нельзя.


Если свойство равно true, то
 заголовки строк таблицы будут отображаться, иначе - не будут.


По умолчанию свойство равно true.
