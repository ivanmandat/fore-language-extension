# TabSheetSettings.FixedRow


## Синтаксис


FixedRow: Number


## Описание


Свойство FixedRow** содержит
 индекс крайней зафиксированной строки таблицы.


## Комментарии


Значение свойства устанавливается с помощью метода setFixedRow
 и возвращается с помощью метода getFixedRow.
 Из JSON значение свойства задать нельзя.
