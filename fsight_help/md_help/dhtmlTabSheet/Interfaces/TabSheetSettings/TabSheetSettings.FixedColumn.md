# TabSheetSettings.FixedColumn


## Синтаксис


FixedColumn: Number


## Описание


Свойство FixedColumn**
 содержит индекс крайнего зафиксированного столбца таблицы.


## Комментарии


Значение свойства устанавливается с помощью метода setFixedColumn
 и возвращается с помощью метода getFixedColumn.
 Из JSON значение свойства задать нельзя.
