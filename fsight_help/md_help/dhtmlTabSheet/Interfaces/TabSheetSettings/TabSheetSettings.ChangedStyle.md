# TabSheetSettings.ChangedStyle


## Синтаксис


ChangedStyle: Object


## Описание


Свойство ChangedStyle**
 возвращает объект со стилем измененных данных.


## Комментарии


Значение свойства можно установить из JSON и с помощью метода setChangedStyle, а возвратить -
 с помощью метода getChangedStyle.
