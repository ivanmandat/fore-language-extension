# TabSheetSettings.IsEditable


## Синтаксис


IsEditable: [PP.Ui.TabSheetEditMode](../../Enums/TabSheetEditMode.htm);


## Описание


Свойство IsEditable** определяет,
 являются ли ячейки таблицы редактируемыми.


## Комментарии


Значение свойства устанавливается из JSON и c помощью метода set**IsEditable**,
 а возвращается - с помощью метода get**IsEditable**.


По умолчанию значение свойства равно PP.Ui.TabSheetEditMode.Enable.
