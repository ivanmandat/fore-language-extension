# TabSheetSettings.EnableMoveObjects


## Синтаксис


EnableMoveObjects: Boolean


## Описание


Свойство EnableMoveObjects**
 возвращает признак доступности объектов для перемещения.


## Комментарии


Значение свойства можно установить из JSON и с помощью метода setEnableMoveObjects, а возвратить
 - с помощью метода getEnableMoveObjects.


Свойство содержит значение true,
 если объекты доступны для перемещения, если false
 - то недоступны.


По умолчанию свойство содержит значение true.
