# TabSheetSettings.FixedLineColor


## Синтаксис


FixedLineColor: String


## Описание


Свойство FixedLineColor**
 определяет цвет линии, отделяющей фиксированные строки и столбцы от нефиксированных.


## Комментарии


Значение свойства устанавливается из JSON или с помощью метода setFixedLineColor, а возвращается
 - с помощью метода getFixedLineColor.


По умолчанию значение свойства равно null.
