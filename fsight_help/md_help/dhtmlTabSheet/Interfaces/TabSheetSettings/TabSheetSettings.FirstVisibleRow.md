# TabSheetSettings.FirstVisibleRow


## Синтаксис


FirstVisibleRow: Number


## Описание


Свойство FirstVisibleRow**
 содержит индекс первой видимой строки в таблице, не считая зафиксированных.


## Комментарии


Значение свойства можно установить из JSON и с помощью метода setFirstVisibleRow, а возвратить
 - с помощью метода getFirstVisibleRow.


По умолчанию значение свойства равно 0.
