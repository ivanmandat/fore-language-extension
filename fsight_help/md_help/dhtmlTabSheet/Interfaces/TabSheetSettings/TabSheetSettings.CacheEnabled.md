# TabSheetSettings.CacheEnabled


## Синтаксис


CacheEnabled: Boolean


## Описание


Свойство CacheEnabled**
 запрещает и разрешает кеширование данных [компонентом](../../Components/TabSheet/TabSheet/TabSheet.htm)
 при работе в асинхронном режиме.


## Комментарии


Значение свойства можно установить из JSON и с помощью метода setCacheEnabled, а возвратить -
 с помощью метода getCacheEnabled.


Свойство принимает значение true,
 если кеширование данных компонентом при работе в асинхронном режиме разрешено,
 в противном случае - запрещено.


По умолчанию свойство содержит значение true.
