# TabSheetSettings.MouseWheelScrollStep


## Синтаксис


MouseWheelScrollStep: Number


## Описание


Свойство MouseWheelScrollStep**
 устанавливает количество строк, на которое прокручивается колесо мыши.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**MouseWheelScrollStep**,
 а возвращается с помощью метода get**MouseWheelScrollStep**.
 Из JSON значение свойства установить нельзя.


По умолчанию значение свойства равно 3.
