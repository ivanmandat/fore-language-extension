# TabSheetSettings.CacheSize


## Синтаксис


CacheSize: Number


## Описание


Свойство CacheSize** содержит
 значение относительного размера кеша для асинхронного режиме.


## Комментарии


Размер кеша считается по следующей формуле: количество пустых ячеек
 с размерами по умолчанию на данном размере таблицы умножается на значение
 N, где N - это значение данного поля.


По умолчанию свойство содержит значение 30.
