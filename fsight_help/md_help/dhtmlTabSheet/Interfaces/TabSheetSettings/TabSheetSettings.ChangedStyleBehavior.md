# TabSheetSettings.ChangedStyleBehavior


## Синтаксис


ChangedStyleBehavior: Boolean


## Описание


Свойство ChangedStyleBehavior**
 устанавливает и возвращает признак применения стиля измененных данных
 для отредактированных ячеек таблицы.


## Комментарии


Значение свойства можно установить из JSON и с помощью метода set**ChangedStyleBehavior**,
 а возвратить - с помощью метода get**ChangedStyleBehavior**.


Свойство содержит значение true,
 если стиль измененных данных применяется для отредактированных ячеек таблицы,
 в противном случае - не применяется.
