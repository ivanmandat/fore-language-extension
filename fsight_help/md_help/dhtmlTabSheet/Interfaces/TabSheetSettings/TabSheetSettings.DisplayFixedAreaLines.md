# TabSheetSettings.DisplayFixedAreaLines


## Синтаксис


DisplayFixedAreaLines: Boolean


## Описание


Свойство DisplayFixedAreaLines**
 управляет видимостью линий, разделяющих фиксированные строки и столбцы
 таблицы от нефиксированных.


## Комментарии


Значение свойства устанавливается с помощью метода **setDisplayFixedAreaLines**
 и возвращается при помощи метода g**etDisplayFixedAreaLines**.
 Из JSON значение свойства установить нельзя.


Свойство содержит значение типа Boolean. Если оно равно true,
 то линии фиксации будут отображены в таблице, если false,
 то не будут.


По умолчанию значение свойства равно true.
