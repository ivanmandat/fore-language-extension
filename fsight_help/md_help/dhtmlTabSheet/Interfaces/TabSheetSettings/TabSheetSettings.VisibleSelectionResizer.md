# TabSheetSettings.VisibleSelectionResizer


## Синтаксис


VisibleSelectionResizer: Boolean;


## Описание


Свойство VisibleSelectionResizer
 определяет признак отображения элемента изменения размера области
 выделения ячеек таблицы.


## Комментарии


Значение свойства можно установить из JSON и с помощью метода setVisibleSelectionResizer, а возвратить
 - с помощью метода getVisibleSelectionResizer.


Если свойство имеет значение true,
 то элемент изменения размера области выделения будет отображен, иначе
 - не будет.
