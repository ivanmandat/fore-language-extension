# TabSheetSettings.UndoLength


## Синтаксис


UndoLength: Number;


## Описание


Свойство UndoLength определяет
 количество операций, запоминающихся для возможности их отмены или возврата.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setUndoLength,
 а возвращается с помощью метода getUndoLength.


По умолчанию значение свойства равно 50.
