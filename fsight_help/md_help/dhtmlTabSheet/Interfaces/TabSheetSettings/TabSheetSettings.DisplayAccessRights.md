# TabSheetSettings.DisplayAccessRights


## Синтаксис


DisplayAccessRights: Boolean


## Описание


Свойство DisplayAccessRights**
 управляет видимостью значка, обозначающего права доступа к ячейкам таблицы.


## Комментарии


Значение свойства устанавливается с помощью метода **setDisplayAccessRights**
 и возвращается при помощи метода g**etDisplayAccessRights**.
 Из JSON значение свойства установить нельзя.


Свойство содержит значение типа Boolean. Если оно равно true,
 то значок, обозначающий права доступа к ячейкам таблицы, будет отображаться,
 если false, то не будет.


По умолчанию значение свойства равно true.
