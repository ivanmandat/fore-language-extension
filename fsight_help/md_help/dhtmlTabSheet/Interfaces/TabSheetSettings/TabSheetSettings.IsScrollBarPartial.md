# TabSheetSettings.IsScrollBarPartial


## Синтаксис


IsScrollBarPartial: Boolean


## Описание


Свойство IsScrollBarPartial**
 определяет тип отображения полос прокрутки.


## Комментарии


Значение свойства устанавливается из JSON и c помощью метода set**Is****ScrollBarPartial**, а возвращается
 - с помощью метода get**Is****ScrollBarPartial**.


Если свойство содержит значение true,
 то полосы прокрутки не будут охватывать строки и столбцы таблицы, содержащие
 номера:


![](../../Classes/TabSheet/TabSheet/TabSheet_VerticalExpandVolume.png)


Иначе полосы прокрутки будут охватывать всю таблицу:


![](../../Classes/TabSheet/TabSheet/TabSheet.addCellControl_2.png)


По умолчанию значение свойство равно false.
