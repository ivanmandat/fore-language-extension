# TabSheetSettings.Zoom


## Синтаксис


Zoom: Number


## Описание


Свойство Zoom** устанавливает
 и возвращает значение масштаба таблицы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setZoom,
 а возвращается с помощью метода getZoom.


По умолчанию свойство содержит значение 1.
