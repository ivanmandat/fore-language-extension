# TabSheetSettings.CellAnimationDuration


## Синтаксис


CellAnimationDuration: Number;


## Описание


Свойство CellAnimationDuration
 определяет длительность анимации измененной ячейки.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setCellAnimationDuration,
 а возвращается с помощью метода getCellAnimationDuration.
