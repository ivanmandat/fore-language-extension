# TabSheetSettings.SelectedCellStyle


## Синтаксис


SelectedCellStyle: String;


## Описание


Свойство SelectedCellStyle определяет
 наименование CSS-класса, стили которого будут применены для ячеек, входящих
 в область выделения таблицы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSelectedCellStyle,
 а возвращается с помощью метода getSelectedCellStyle.
