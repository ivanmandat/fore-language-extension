# TabSheetSettings


Пространство имен**: PP.Ui;


## Иерархия наследования


PP.ObjectSettings


    PP.Ui.ControlSettings


        PP.Ui.TabSheetSettings


## Описание


Интерфейс TabSheetSettings используется
 для работы с классом [TabSheet](../../Classes/TabSheet/TabSheet/TabSheet.htm).


## Комментарии


Значения свойств задаются в [конструкторе
 TabSheet](../../Classes/TabSheet/TabSheet/Constructor_TabSheet.htm) или с помощью метода set<имя свойства>, а возвращаются с помощью метода
 get<имя
 свойства>. Доступные
 методы указаны в описании конкретного свойства.
