# TabSheetSettings.UndoRedoEnable


## Синтаксис


UndoRedoEnable: Boolean;


## Описание


Свойство UndoRedoEnable определяет
 возможность использования команд отмены и возврата изменений.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setUndoRedoEnable,
 а возвращается с помощью метода getUndoRedoEnable.


Если свойство имеет значение true,
 то возможно использовать команды отмены и возврата изменений, иначе -
 эти команды использовать нельзя.
