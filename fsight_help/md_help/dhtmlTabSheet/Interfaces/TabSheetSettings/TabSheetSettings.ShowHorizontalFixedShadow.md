# TabSheetSettings.ShowHorizontalFixedShadow


## Синтаксис


ShowHorizontalFixedShadow: Boolean;


## Описание


Свойство ShowHorizontalFixedShadow
 определяет признак отображения тени под фиксированными строками.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setShowHorizontalFixedShadow,
 а возвращается с помощью метода getShowHorizontalFixedShadow.


Если свойство имеет значение true
 (по умолчанию), то тень под фиксированными строками будет отображена,
 иначе - не будет.
