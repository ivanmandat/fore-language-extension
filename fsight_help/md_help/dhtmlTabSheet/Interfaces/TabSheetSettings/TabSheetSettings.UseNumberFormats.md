# TabSheetSettings.UseNumberFormats


## Синтаксис


UseNumberFormats: Boolean


## Описание


Свойство UseNumberFormats**
 содержит признак использования числового формата в стиле ячейки при редактировании.


## Комментарии


Значение свойства можно установить из JSON и с помощью метода setUseNumberFormats, а возвратить
 с помощью метода getUseNumberFormats.


По умолчанию свойство содержит значение false.
