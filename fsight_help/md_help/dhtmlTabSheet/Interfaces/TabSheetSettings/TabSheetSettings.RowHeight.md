# TabSheetSettings.RowHeight


## Синтаксис


RowHeight: Number


## Описание


Свойство RowHeight** возвращает
 высоту строки таблицы.


## Комментарии


Значение свойства устанавливается с помощью метода setRowHeight
 и возвращается при помощи метода getRowHeight.
 Из JSON значение свойства установить нельзя.


При вызове метода setRowHeight
 указываются два параметра: value - новое значение высоты строки и rowIndex
 - индекс строки, для которой устанавливается высота.


При вызове метода getRowHeight
 указывается параметр rowIndex - индекс строки таблицы, значение высоты
 которой необходимо возвратить.
