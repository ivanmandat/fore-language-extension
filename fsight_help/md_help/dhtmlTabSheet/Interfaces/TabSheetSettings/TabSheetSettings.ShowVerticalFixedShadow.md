# TabSheetSettings.ShowVerticalFixedShadow


## Синтаксис


ShowVerticalFixedShadow: Boolean;


## Описание


Свойство ShowVerticalFixedShadow
 определяет признак отображения тени правее фиксированных столбцов.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setShowVerticalFixedShadow,
 а возвращается с помощью метода getShowVerticalFixedShadow.


Если свойство имеет значение true,
 то тень правее фиксированных столбцов будет отображена, иначе -
 не будет (по умолчанию).
