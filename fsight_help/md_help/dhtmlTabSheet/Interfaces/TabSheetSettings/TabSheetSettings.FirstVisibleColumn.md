# TabSheetSettings.FirstVisibleColumn


## Синтаксис


FirstVisibleColumn: Number


## Описание


Свойство FirstVisibleColumn**
 содержит индекс первого видимого столбца в таблице, не считая зафиксированных.


## Комментарии


Значение свойства можно установить из JSON и с помощью метода setFirstVisibleColumn, а возвратить
 - с помощью метода getFirstVisibleColumn.


По умолчанию значение свойства равно 0.
