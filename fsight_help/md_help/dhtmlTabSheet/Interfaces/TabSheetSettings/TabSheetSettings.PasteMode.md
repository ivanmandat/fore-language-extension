# TabSheetSettings.PasteMode


## Синтаксис


PasteMode: PP.Ui.[PasteMode](../../Enums/PasteMode.htm)


## Описание


Свойство PasteMode** определяет
 режим вставки данных в таблицу.


## Комментарии


Значение свойства задаётся с помощью метода setPasteMode
 и возвращается с помощью метода getPasteMode.
 Из JSON значение свойства установить нельзя.


Параметром метода setPasteMode
 является элемент перечисления [PP.Ui.PasteMode](../../Enums/PasteMode.htm).
