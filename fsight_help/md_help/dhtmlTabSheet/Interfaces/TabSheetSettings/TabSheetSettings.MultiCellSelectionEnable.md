# TabSheetSettings.MultiCellSelectionEnable


## Синтаксис


MultiCellSelectionEnable: Boolean;


## Описание


Свойство MultiCellSelectionEnable
 определяет признак возможности выделения нескольких ячеек таблицы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMultiCellSelectionEnable,
 а возвращается с помощью метода getMultiCellSelectionEnable.


Если свойство имеет значение true
 (по умолчанию), то можно выделять несколько ячеек таблицы, иначе - только
 одну.
