# TabSheetSettings.EnableSelection


## Синтаксис


EnableSelection: Boolean


## Описание


Свойство EnableSelection**
 управляет возможностью выделения ячеек таблицы вручную.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода **setEnableSelection**
 и возвращается при помощи метода g**etEnableSelection**.


Свойство содержит значение типа Boolean. Если оно равно true,
 то выделение ячеек в таблице будет разрешено, если false,
 то не будет.


По умолчанию значение свойства равно true.
