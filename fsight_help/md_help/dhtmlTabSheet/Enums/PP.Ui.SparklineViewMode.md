# PP.Ui.SparklineViewMode


Пространство имен: PP.Ui;


## Описание


Перечисление SparklineViewMode содержит
 режимы отображения данных на спарклайне.


Используется свойством [Sparkline.ViewMode](../Classes/Sparkline/Sparkline.ViewMode.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 Line


		 Данные отображаются в виде линии.


		 Column


		 Данные отображаются в виде столбцов.
