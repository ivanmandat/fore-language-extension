# TabFixedBehaviour


Пространство имен: PP.Ui


## Описание


Перечисление TabFixedBehaviour определяет, каким образом фиксированные строки и столбцы таблицы будут выделяться при нажатии на них.


Используется следующими методами:


	- [TabSheetSettings.FixedBehaviour](../Interfaces/TabSheetSettings/TabSheetSettings.FixedBehaviour.htm);


	- [TabSheet.getCustomFixedBehaviour](../Classes/TabSheet/TabSheet/TabSheet.getCustomFixedBehaviour.htm);


	- [TabSheetMeasures.FixedBehaviour](../Classes/TabSheet/TabSheetMeasures/TabSheetMeasures.FixedBehaviour.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 0


		 Off. Выделяется только
		 ячейка, которая была нажата.


		 2


		 SelectablePart. Выделяется
		 только нажатая строка или столбец без соответствующих заголовков.


		 3


		 SelectableAll. Выделяется
		 нажатая строка или столбец вместе с соответствующими заголовками.
