# TabSheetSelectionAreaType


Пространство имен: PP.Ui;


## Описание


Перечисление TabSheetSelectionAreaType
 содержит типы областей выделения в таблице.


Используется следующими методами:


	- [TabSheetSelection.addRange](../Classes/TabSheet/TabSheetSelection/TabSheetSelection.addRange.htm);


	- [TabSheetSelection.getSelectedAreas](../Classes/TabSheet/TabSheetSelection/TabSheetSelection.getSelectedAreas.htm);


	- [TabSheetSelection.getSelectionType](../Classes/TabSheet/TabSheetSelection/TabSheetSelection.getSelectionType.htm);


	- [TabSheetSelection.removeArea](../Classes/TabSheet/TabSheetSelection/TabSheetSelection.removeArea.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 All


		 Выделение всей таблицы.


		 Common


		 Обычное выделение ячеек таблицы.


		 Column


		 Выделение столбца таблицы.


		 Row


		 Выделение строки таблицы.
