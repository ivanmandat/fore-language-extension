# TabSheetAccessRights


Пространство имен: PP.Ui


## Описание


Перечисление TabSheetAccessRights определяет права доступа к ячейке таблицы.


Используется свойством [TabSheetCellStyle.AccessRights](../Classes/TabSheet/TabSheetCellStyle/TabSheetCellStyle.AccessRights.htm).


## Допустимые значения


Значение


Краткое описание


-1


 Undefined. Права доступа не определены.


0


 NoAccess. Нет доступа к ячейке таблицы.


1


 Read. Права на чтение данных.


2


 Write. Права на запись данных.


3


 FullAcess. Права на полный доступ к ячейке таблицы.
