# TabSheetEditMode


Пространство имен: PP.Ui


## Описание


Перечисление TabSheetEditMode
 определяет возможность редактирования значений в ячейках таблицы.


Используется свойствами и методами:


	- [TabSheetSettings.IsEditable](../Interfaces/TabSheetSettings/TabSheetSettings.IsEditable.htm);


	- [TabSheet.edit](../Classes/TabSheet/TabSheet/TabSheet.edit.htm);


	- [TabSheet.setIsUserEditableRanges](../Classes/TabSheet/TabSheet/TabSheet.setIsUserEditableRanges.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 Enable


		 Редактирование разрешено (по умолчанию).


		 Disable


		 Редактирование запрещено.


		 UserEnabledCell


		 Редактирование возможно только для определённых ячеек.
