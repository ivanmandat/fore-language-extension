# TabSheetObjectType


Пространство имен: PP.Ui


## Описание


Перечисление TabSheetObjectType
 определяет тип объекта в таблице.


Используется свойством [TabSheetObject.ObjectType](../Classes/TabSheet/TabSheetObject/TabSheetObject.ObjectType.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 Image

		 OldObject


		 Устаревший тип объектов, не поддерживаемых в текущей версии.
		 Оставлено для совместимости.


		 ITabObject


		 Пользовательский объект.


		 Silverlight


		 Silverlight-объект.


		 Shape
