# PP.Ui.TabSheetSelection.Type


Пространство имен: PP.Ui


## Описание


Перечисление PP.Ui.TabSheetSelection.Type
 содержит типы выделения в таблице.


Используется методами [EaxMdService.setTabSheetSelection](dhtmlExpress.chm::/Classes/Express/EaxMdService/EaxMdService.setTabSheetSelection.htm)
 и [GridView.getSelectionType](dhtmlMetabase.chm::/Classes/Metabase/GridView/GridView.getSelectionType.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 Cell


		 Выделение одной ячейки таблицы.


		 Common


		 Обычное выделение ячеек таблицы.


		 Column


		 Выделение столбца таблицы.


		 Row


		 Выделение строки таблицы.
