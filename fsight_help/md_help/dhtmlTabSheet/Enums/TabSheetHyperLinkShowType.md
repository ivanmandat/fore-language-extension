# TabSheetHyperLinkShowType


Пространство имен: PP.Ui


## Описание


Перечисление TabSheetHyperLinkShowType содержит режимы отображения содержимого ячеек таблицы в виде ссылок.


Используется методом [TabSheet.showHyperlinksOnRange](../Classes/TabSheet/TabSheet/TabSheet.showHyperlinksOnRange.htm).


## Допустимые значения


Значение


Краткое описание


Default


В виде гиперссылок отображается содержимое всех ячеек таблицы.


All


В виде гиперссылок отображается содержимое всех ячеек таблицы.


None


Содержимое ячеек таблицы, входящих в указанный диапазон, не отображается в виде гиперссылок.
