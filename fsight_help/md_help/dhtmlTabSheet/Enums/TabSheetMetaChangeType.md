# TabSheetMetaChangeType


Пространство имен: PP.Ui


## Описание


Перечисление TabSheetMetaChangeType
 определяет тип изменений свойств таблицы.


Используется событием [TabSheet.MetaChanged](../Classes/TabSheet/TabSheet/TabSheet.MetaChanged.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 Scroll


		 Изменение свойств полос прокрутки.


		 WidthSize


		 Изменение ширины таблицы.


		 HeightSize


		 Изменение высоты таблицы.


		 Measures


		 Изменение свойств измерений таблицы.


		 Model


		 Изменение свойств модели таблицы.
