# PasteMode


Пространство имен: PP.Ui


## Описание


Перечисление PasteMode
 определяет режим вставки данных в таблицу.


Используется свойством [TabSheetSettings.PasteMode](../Interfaces/TabSheetSettings/TabSheetSettings.PasteMode.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 ContentOnly


		 Режим вставки содержимого без форматирования.


		 ContentAndStyle


		 Режим вставки содержимого с форматированием.
