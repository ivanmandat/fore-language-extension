# TabSearchTarget


Пространство
 имен: PP.Ui


## Описание


Перечисление TabSearchTarget содержит свойства ячейки, по которой будет
 осуществляться поиск.


Используется методом [TabSheetDataSource.search](dhtmlMetabase.chm::/Classes/Metabase/TabSheetDataSource/TabSheetDataSource.search.htm).


## Допустимые значения


		 Значение


		 Краткое описание


		 1


		 Value.
		 Поиск по значению в ячейке.


		 4


		 Text.
		 Поиск по тексту в ячейке.
