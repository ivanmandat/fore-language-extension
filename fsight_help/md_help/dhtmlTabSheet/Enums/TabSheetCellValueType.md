# TabSheetCellValueType


Пространство имен: PP.Ui


## Описание


Перечисление TabSheetCellValueType определяет тип значения в ячейке модели таблицы.


Используется свойством [TabSheetCell.ValueType](../Classes/TabSheet/TabSheetCell/TabSheetCell.ValueType.htm).
