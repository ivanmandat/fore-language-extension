# TabSheetCellBorderWeight


Пространство имен: PP.Ui


## Описание


Перечисление TabSheetCellBorderWeight определяет толщину линии границы ячеек таблицы.


Используется свойством [TabSheetCellBorderStyle.RawWeight](../Classes/TabSheet/TabSheetCellBorderStyle/TabSheetCellBorderStyle.RawWeight.htm).


## Допустимые значения


Значение


Краткое описание


-1


 Undefined. Толщина линии не определена.


0


 Thick. Толстая линия.


1


 Medium. Средняя по толщине линяя.


2


 Thin. Тонкая линия (по умолчанию).


3


 Hairline. Более тонкая линия.
