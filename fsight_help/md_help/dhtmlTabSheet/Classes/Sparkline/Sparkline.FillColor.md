# Sparkline.FillColor


## Синтаксис


FillColor: PP.[Color](dhtmlCommon.chm::/Classes/PP/Color/Color.htm)
 | RGB | String


## Описание


Свойство FillColor определяет
 цвет заливки столбцов спарклайна.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setFillColor,
 а возвращается с помощью метода getFillColor.
