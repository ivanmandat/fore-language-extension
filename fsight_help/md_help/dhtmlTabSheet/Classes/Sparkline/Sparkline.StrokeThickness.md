# Sparkline.StrokeThickness


## Синтаксис


StrokeThickness: Number


## Описание


Свойство StrokeThickness определяет
 толщину границ столбцов спарклайна.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setStrokeThickness,
 а возвращается с помощью метода getStrokeThickness.


По умолчанию значение свойства равно 1.
