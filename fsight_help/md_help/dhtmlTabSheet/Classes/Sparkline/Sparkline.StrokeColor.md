# Sparkline.StrokeColor


## Синтаксис


StrokeColor: PP.[Color](dhtmlCommon.chm::/Classes/PP/Color/Color.htm)
 | RGB | String


## Описание


Свойство StrokeColor определяет
 цвет границ столбцов спарклайна.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setStrokeColor,
 а возвращается с помощью метода getStrokeColor.


По умолчанию значение свойства равно #000000.
