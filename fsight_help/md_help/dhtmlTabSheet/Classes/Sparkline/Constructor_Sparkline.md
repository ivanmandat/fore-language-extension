# Конструктор Sparkline


## Синтаксис


PP.Ui.Sparkline(settings);


## Параметры


settings. JSON-объект со значениями
 свойств компонента.


## Описание


Конструктор Sparkline создает
 экземпляр компонента [Sparkline](../../Components/Sparkline/Sparkline.htm).
