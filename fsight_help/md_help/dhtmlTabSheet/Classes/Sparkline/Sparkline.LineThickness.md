# Sparkline.LineThickness


## Синтаксис


LineThickness: Number


## Описание


Свойство LineThickness определяет
 толщину линии спарклайна.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLineThickness,
 а возвращается с помощью метода getLineThickness.


По умолчанию значение свойства равно 3.
