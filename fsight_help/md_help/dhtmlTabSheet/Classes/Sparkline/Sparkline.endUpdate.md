# Sparkline.endUpdate


## Синтаксис


endUpdate();


## Описание


Метод endUpdate включает перерисовку
 спарклайна и обновляет его.


## Комментарии


Использует в связке с методом [beginUpdate](Sparkline.beginUpdate.htm).
