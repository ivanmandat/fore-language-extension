# Sparkline.Values


## Синтаксис


Values: Array


## Описание


Свойство Values определяет массив
 значений спарклайна.


## Комментарии


Элементы массива - числа.


Значение свойства устанавливается из JSON и с помощью метода setValues,
 а возвращается с помощью метода getValues.
