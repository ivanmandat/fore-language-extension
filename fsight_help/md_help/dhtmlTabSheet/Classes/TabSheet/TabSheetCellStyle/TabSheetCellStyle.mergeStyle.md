# TabSheetCellStyle.mergeStyle


## Синтаксис


mergeStyle(mergePart: Object);


## Параметры


*mergePart.* Копируемый объект стиля.


## Описание


Метод mergeStyle** возвращает копию объекта стиля ячейки таблицы.


## Комментарии


Метод возвращает объект типа [TabSheetCellStyle](TabSheetCellStyle.htm).
