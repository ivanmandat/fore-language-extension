# TabSheetCellStyle.AccessRights


## Синтаксис

## Описание


Свойство AccessRights** устанавливает и возвращает права доступа на ячейки таблицы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**AccessRights**, а возвращается с помощью метода get**AccessRights**.
