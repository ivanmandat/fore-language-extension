# TabSheetCellStyle.getBorderByIndex


## Синтаксис


getBorderByIndex(index: PP.Ui.[TabSheetCellBorderIndex](../../../Enums/TabSheetCellBorderIndex.htm));


## Параметры


*index.* Индекс границы ячейки таблицы.


## Описание


Метод getBorderByIndex** возвращает границу по её индексу для ячейки таблицы.


## Комментарии


Метод возвращает объект типа PP.Ui.[TabSheetCellBorderStyle](../TabSheetCellBorderStyle/TabSheetCellBorderStyle.htm).
