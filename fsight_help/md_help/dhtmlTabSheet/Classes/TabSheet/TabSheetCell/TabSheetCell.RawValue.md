# TabSheetCell.RawValue


## Синтаксис


RawValue: String


## Описание


Свойство RawValue** устанавливает
 и возвращает необработанное значение для ячейки модели таблицы.


## Комментарии


Значение свойства устанавливается с помощью метода setRawValue
 и возвращается с помощью метода getRawValue.
 Из JSON значение свойства установить нельзя.
