# TabSheetCell.Hint


## Синтаксис


Hint: String


## Описание


Свойство Hint** устанавливает подсказку для ячейки модели таблицы.


## Комментарии


Значение свойства устанавливается с помощью метода setHint и возвращается с помощью метода getHint. Из JSON значение свойства установить нельзя.
