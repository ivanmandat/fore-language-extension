# TabSheetCell.getCoord


## Синтаксис


getCoord();


## Описание


Метод getCoord** возвращает текущие координаты ячейки модели таблицы.


## Комментарии


Метод возвращает объект типа PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm).
