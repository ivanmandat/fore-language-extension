# TabSheetCell.ValueType


## Синтаксис


ValueType: PP.Ui.[TabSheetCellValueType](../../../Enums/TabSheetCellValueType.htm)


## Описание


Свойство ValueType** содержит тип значения в ячейке модели.


## Комментарии


Значение свойства устанавливается с помощью метода setValueType и возвращается с помощью метода getValueType. Из JSON значение свойства установить нельзя.
