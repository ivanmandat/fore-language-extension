# TabSheetCell.setEnabled


## Синтаксис


setEnabled(value: Boolean);


## Параметры


*value.* Признак доступности ячейки
 модели. Если параметр равен true,
 то ячейка модели будет доступна , иначе - недоступна.


## Описание


Метод setEnabled** управляет
 доступностью ячейки модели таблицы.
