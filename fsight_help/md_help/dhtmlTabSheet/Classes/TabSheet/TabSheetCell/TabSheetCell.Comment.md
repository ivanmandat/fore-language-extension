# TabSheetCell.Comment


## Синтаксис


Comment: String


## Описание


Свойство Comment** устанавливает комментарий для ячейки модели таблицы.


## Комментарии


Значение свойства устанавливается с помощью метода setComment и возвращается с помощью метода getComment. Из JSON значение свойства установить нельзя.
