# TabSheetCell.getIsEnabled


## Синтаксис


getIsEnabled();


## Описание


Метод getIsEnabled** определяет доступность выделенной ячейки.


## Комментарии


Метод возвращает значение true, если выделенная ячейка доступна, и false, если недоступна.
