# TabSheetCell.Style


## Синтаксис


Style: PP.Ui.[TabSheetCellStyle](../TabSheetCellStyle/TabSheetCellStyle.htm)


## Описание


Свойство Style** устанавливает стиль ячейки модели таблицы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setStyle, а возвращается с помощью метода getStyle.
