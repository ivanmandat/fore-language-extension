# TabSheetCell.Value


## Синтаксис


Value: String


## Описание


Свойство Value** устанавливает и возвращает значение для ячейки модели таблицы.


## Комментарии


Значение свойства устанавливается с помощью метода setValue и возвращается с помощью метода getValue. Из JSON значение свойства установить нельзя.
