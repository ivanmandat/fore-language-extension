# DefaultTabSheetDataSource.loadRanges


## Синтаксис


loadRanges(ranges: Array);


## Параметры


*ranges.* Массив диапазонов ячеек, загружаемых в таблицу.


## Описание


Метод loadRanges** загружает диапазоны ячеек в таблицу.
