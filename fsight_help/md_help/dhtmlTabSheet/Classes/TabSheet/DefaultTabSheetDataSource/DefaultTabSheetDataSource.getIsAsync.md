# DefaultTabSheetDataSource.getIsAsync


## Синтаксис


getIsAsync();


## Описание


Метод getIsAsync** определяет, является ли источник данных по умолчанию асинхронным.


## Комментарии


Метод возвращает значение true, если источник данных по умолчанию является асинхронным, в противном случае - значение false.
