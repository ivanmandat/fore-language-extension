## Синтаксис

## Описание


Метод getCoord** возвращает координаты ячейки, содержащей базовый редактор таблицы.


## Комментарии


Метод возвращает значение типа [TabSheeetCoord](../TabSheetCoord/TabSheetCoord.htm).
