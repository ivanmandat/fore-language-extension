# Конструктор TabSheetDefaultEditor


## Синтаксис


PP.Ui.TabSheetDefaultEditor(settings);


## Параметры


*settings.* JSON-объект со значениями свойств компонента.


## Описание


Конструктор TabSheetDefaultEditor** создает экземпляр класса **TabSheetDefaultEditor.**

Конструктор предназначен для внутреннего использования.
