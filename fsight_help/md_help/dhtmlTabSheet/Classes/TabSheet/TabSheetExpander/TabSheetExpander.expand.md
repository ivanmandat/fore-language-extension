# TabSheetExpander.expand


## Синтаксис


expand(noEvent: Boolean);


## Параметры


*noEvent.* Признак того, что событие разворачивания группировки ячеек таблицы сгенерировано не будет. Если параметр равен значению true, то данное вызвано не будет, иначе - будет.


## Описание


Метод expand** разворачивает группировку ячеек таблицы.


## Комментарии


Признак того, что событие сворачивания группировки ячеек таблицы сгенерировано не будет. Если параметр равен значению true, то данное вызвано не будет, иначе - будет.
