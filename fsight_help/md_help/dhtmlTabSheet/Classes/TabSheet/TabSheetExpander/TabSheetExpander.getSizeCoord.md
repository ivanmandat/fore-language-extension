# TabSheetExpander.getSizeCoord


## Синтаксис


getSizeCoord();


## Описание


Метод getSizeCoord** возвращает размер группировки, выраженный в количестве входящих в неё строк и столбцов таблицы.


## Комментарии


Метод возвращает JSON-объект со следующими полями: row - количество строк в группировке, column - количество столбцов.
