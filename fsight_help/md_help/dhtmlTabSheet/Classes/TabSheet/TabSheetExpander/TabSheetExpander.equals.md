# TabSheetExpander.equals


## Синтаксис


equals(expander: PP.Ui.[TabSheetExpander](TabSheetExpander.htm));


## Параметры


*expander.* Группировка ячеек таблицы,


## Описание


Метод equals** сравнивает указанную группировку ячеек таблицы с текущей.


## Комментарии


Если указанная группировка ячеек эквивалентна текущей, то метод возвращает значение true, иначе - false.
