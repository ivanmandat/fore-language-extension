# TabSheetExpander.isCoordInside


## Синтаксис


isCoordInside(coord: PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm));


## Параметры


*coord.* Координаты ячейки таблицы.


## Описание


Метод isCoordInside** определяет, входят ли указанные координаты в группировку ячеек.


## Комментарии


Метод возвращает значение true, если указанные координаты входят в группировку ячеек, иначе - false.
