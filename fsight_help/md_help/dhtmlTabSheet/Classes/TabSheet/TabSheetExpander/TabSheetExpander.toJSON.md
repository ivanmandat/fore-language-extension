# TabSheetExpander.toJSON


## Синтаксис


toJSON();


## Описание


Метод toJSON возвращает
 JSON объект с настройками экспандера.


## Пример


Для выполнения примера необходимо наличие таблицы с экспандером, созданным
 в описании [конструктора TabSheetExpander](Conctructor_TabSheetExander.htm).
 Получим JSON объект первого экспандера:


// Получим JSON объект экспандера
json = expander2.toJSON();
console.log(json);

В результате выполнения примера был получен
 JSON объект с настройками экспандера.
