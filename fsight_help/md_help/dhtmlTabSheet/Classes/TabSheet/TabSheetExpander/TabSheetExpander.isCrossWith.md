# TabSheetExpander.isCrossWith


## Синтаксис


isCrossWith(expander: PP.Ui.[TabSheetExpander](TabSheetExpander.htm));


## Параметры


*expander.* Группировка ячеек таблицы.


## Описание


Метод isCrossWith** определяет, пересекается ли указанная группировка с текущей.


## Комментарии


Метод возвращает значение true, если указанная группировка пересекается с текущей, иначе - значение false.
