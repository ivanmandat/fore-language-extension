# TabSheetExpander.collapse


## Синтаксис


collapse(noEvent: Boolean);


## Параметры


*noEvent.* Признак того, что событие сворачивания группировки ячеек таблицы сгенерировано не будет. Если параметр равен значению true, то данное событие вызвано не будет, иначе - будет.


## Описание


Метод collapse** сворачивает группировку ячеек таблицы.
