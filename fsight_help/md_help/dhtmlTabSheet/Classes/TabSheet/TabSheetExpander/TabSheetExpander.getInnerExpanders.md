# TabSheetExpander.getInnerExpanders


## Синтаксис


getInnerExpanders();


## Описание


Метод getInnerExpanders** возвращает группировки ячеек таблицы, вложенные в текущую группировку.


## Комментарии


Метод возвращает массив объектов типа PP.Ui.[TabSheetExpander](TabSheetExpander.htm).
