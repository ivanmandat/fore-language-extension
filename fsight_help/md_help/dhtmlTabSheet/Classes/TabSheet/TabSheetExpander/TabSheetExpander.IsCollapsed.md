# TabSheetExpander.IsCollapsed


## Синтаксис


IsCollapsed: Boolean


## Описание


Свойство IsCollapsed** определяет, свёрнута ли группировка ячеек таблицы.


## Комментарии


Метод возвращает значение true, если группировка ячеек таблицы свёрнута, и false - в противном случае.
