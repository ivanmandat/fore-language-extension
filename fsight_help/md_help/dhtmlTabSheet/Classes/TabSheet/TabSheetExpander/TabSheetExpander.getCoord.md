# TabSheetExpander.getCoord


## Синтаксис


getCoord();


## Описание


Метод getCoord** возвращает координаты левой верхней ячейки в группировке.


## Комментарии


Метод возвращает значение типа PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm).
