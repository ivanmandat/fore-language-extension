# TabSheetExpander.contains


## Синтаксис


contains(expander: PP.Ui.[TabSheetExpander](TabSheetExpander.htm));


## Параметры


*expander.* Группировка ячеек таблицы, проверяемая на вложенность в текущую группировку.


## Описание


Метод contains** проверяет, является ли указанная группировка ячеек таблицы вложенной в текущую группировку.


## Комментарии


Метод возвращает значение true, если указанная группировка ячеек таблицы является вложенной в текущую группировку, и false - в противном случае.
