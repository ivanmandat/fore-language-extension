# TabSheetExpander.remove


## Синтаксис


remove(onlyDom: Boolean, fromModel: Boolean);


## Параметры


*onlyDom.* Признак удаления группировки только из DOM-дерева;


*fromModel.* Признак удаления группировки из модели таблицы.


## Описание


Метод remove** удаляет группировку ячеек таблицы.
