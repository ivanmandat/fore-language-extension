# TabSheetExpander.getIndexesToExpand


## Синтаксис


getIndexesToExpand();


## Описание


Метод getIndexesToExpand** возвращает индексы строк и столбцов, которые скрываются при сворачивании группировки.


## Комментарии


Метод возвращает JSON-объект со следующими полями: rows - массив индексов строк, columns - массив индексов столбцов, скрытых после сворачивания группировки.
