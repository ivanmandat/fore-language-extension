# TabSheetCellSpan.toJSON


## Синтаксис


toJSON();


## Описание


Метод toJSON возвращает JSON
 объект с настройками диапазона.


## Пример


Для выполнения получим диапазон ячеек, аналогично примеру из описания
 [конструктора TabSheetCellSpan](Consrtuctor_TabSheetCellSpan.htm).
 Получим JSON объект диапазона:


// Получим JSON объект
cellSpan.toJSON();
// -> Object {@L: 1, @T: 0, @W: 0, @H: 2}

В результате выполнения примера был получен JSON объект диапазона.
