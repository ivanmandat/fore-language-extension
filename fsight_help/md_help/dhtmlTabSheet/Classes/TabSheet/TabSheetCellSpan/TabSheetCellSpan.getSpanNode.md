# TabSheetCellSpan.getSpanNode


## Синтаксис


getSpanNode();


## Описание


Метод getSpanNode возвращает
 ноду, в которой отрисован диапазон.


## Пример


Для выполнения получим диапазон ячеек, аналогично примеру из описания
 [конструктора TabSheetCellSpan](Consrtuctor_TabSheetCellSpan.htm).
 Получим JSON объект диапазона:


// Получим ноду
cellSpan.getSpanNode();

В результате выполнения примера была получена нода диапазона.
