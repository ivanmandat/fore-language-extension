# TabSheetCellSpan.getRenderedCoord


## Синтаксис


getRenderedCoord();


## Описание


Метод getRenderedCoord возвращает
 координаты ячеек, в которой отрисован диапазон.


## Пример


Для выполнения получим диапазон ячеек, аналогично примеру из описания
 [конструктора TabSheetCellSpan](Consrtuctor_TabSheetCellSpan.htm).


// Получим координаты ячеек
cellSpan.getRenderedCoord();

В результате выполнения примера были получены координаты ячеек, в которых
 отрисован диапазон.
