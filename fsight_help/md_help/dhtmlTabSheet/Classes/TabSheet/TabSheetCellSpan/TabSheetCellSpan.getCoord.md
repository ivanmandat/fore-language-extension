# TabSheetCellSpan.getCoord


## Синтаксис


getCoord();


## Описание


Метод getCoord** возвращает координаты верхней левой ячейки в диапазоне объединённых ячеек таблицы.


## Комментарии


Метод возвращает объект PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm).
