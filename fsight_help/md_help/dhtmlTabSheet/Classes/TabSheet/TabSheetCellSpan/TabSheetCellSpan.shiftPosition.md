# TabSheetCellSpan.shiftPosition


## Синтаксис


shiftPosition(rows: Number, cols: Number, rowIndex: Number, colIndex: Number);


## Параметры


*r**ows.* Количество строк, на которое будет уменьшена текущая позиция;


*cols.* Количество столбцов, на которое будет уменьшена текущая позиция;


*rowIndex.* Индекс строки верхней левой ячейки в объединении;


*colIndex.* Индекс столбца верхней левой ячейки в объединении.


## Описание


Метод shiftPosition** сдвигает позицию диапазона выделенных ячеек таблицы.
