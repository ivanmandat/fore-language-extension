# TabSheetCellSpan.getSizeCoord


## Синтаксис


getSizeCoord();


## Описание


Метод getSizeCoord** возвращает размер диапазона объединённых ячеек, выраженный в количестве входящих в него строк и столбцов.


## Комментарии


Метод возвращает JSON-объект со следующими полями: row - количество строк, column - количество столбцов в диапазоне объединённых ячеек.
