# TabSheetCellSpan.getRange


## Синтаксис


getRange(ts);


## Параметры


*ts.* Таблица, диапазон ячеек которой нужно получить.


## Описание


Метод getRange** возвращает диапазон ячеек, соответствующий объединению.


## Комментарии


Метод возвращает объект типа PP.Ui.[TabSheetRange](../TabSheetRange/TabSheetRange.htm).
