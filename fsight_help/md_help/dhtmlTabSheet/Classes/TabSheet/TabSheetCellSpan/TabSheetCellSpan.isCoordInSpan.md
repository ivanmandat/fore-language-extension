# TabSheetCellSpan.isCoordInSpan


## Синтаксис


isCoordInSpan(coord: PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm));


## Параметры


*coord.* Координата, для которой определяется вхождение в диапазон объединённых ячеек.


## Описание


Метод isCoordInSpan** определяет, находится ли указанная координата в диапазоне объединённых ячеек.


## Комментарии


Метод возвращает true, если указанная координата входит в диапазон объединённых ячеек, иначе - false.
