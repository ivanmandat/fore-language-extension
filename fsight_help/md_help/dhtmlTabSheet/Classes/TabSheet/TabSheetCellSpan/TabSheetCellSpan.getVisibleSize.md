# TabSheetCellSpan.getVisibleSize


## Синтаксис


getVisibleSize(ts);


## Параметры


*ts.* Таблица, размер объединённого диапазона которой нужно определить.


## Описание


Метод getVisibleSize** возвращает размер диапазона объединённых ячеек в пикселях.


## Комментарии


Метод возвращает JSON-объект со следующими полями: height - высота, width - ширина диапазона объединённых ячеек в пикселях.
