# TabSheetRange.isAllRowsHidden


## Синтаксис


isAllRowsHidden();


## Описание


Метод isAllRowsHidden** определяет, скрыты ли все строки данного диапазона.


## Комментарии


Метод возвращает значение true, если все строки данного диапазона скрыты, и значение false, если хотя бы одна строка не скрыта.
