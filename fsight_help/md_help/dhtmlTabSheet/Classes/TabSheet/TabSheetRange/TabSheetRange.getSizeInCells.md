# TabSheetRange.getSizeInCells


## Синтаксис


getSizeInCells();


## Описание


Метод getSizeInCells** возвращает размер данного диапазона, выраженный в количестве охватываемых им видимых строк и видимых столбцов.


## Комментарии


Метод возвращает JSON-объект со следующими свойствами: rowsCount - количество видимых строк, охватываемых данным диапазоном, columnsCount - количество видимых столбцов.
