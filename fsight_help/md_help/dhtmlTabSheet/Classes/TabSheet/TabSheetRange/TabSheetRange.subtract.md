# TabSheetRange.subtract


## Синтаксис


subtract(range: [PP.Ui.TabSheetRange](TabSheetRange.htm));


## Параметры


range. Вычитаемый диапазон ячеек.


## Описание


Метод subtract** вычитает из данного диапазона ячеек указанный.


## Комментарии


Метод возвращает массив объектов типа [PP.Ui.TabSheetRange](TabSheetRange.htm).
