# TabSheetRange.isRowBelong


## Синтаксис


isRowBelong(rowIndex: Number);


## Параметры


rowIndex. Индекс строки таблицы.


## Описание


Метод isRowBelong** определяет, входит ли строка с указанным индексом в данный диапазон.


## Комментарии


Метод возвращает значение true, если строка с указанным индексом входит в данный диапазон, и значение false, если не входит.
