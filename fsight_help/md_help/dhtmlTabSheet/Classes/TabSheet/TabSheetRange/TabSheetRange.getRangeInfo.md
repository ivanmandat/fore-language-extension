# TabSheetRange.getRangeInfo


## Синтаксис


getRangeInfo();


## Описание


Метод getRangeInfo** возвращает объект с информацией о верхней левой координате данного диапазона.


## Комментарии


Метод возвращает JSON-объект со следующими полями: cellspan - признак вхождения верхней левой координаты в диапазон объединённых ячеек, colHeader - признак её вхождения в заголовок столбцов, fixedColumns - признак вхождения в фиксированный столбец, fixedRows - признак вхождения в фиксированную строку, rowHeader - признак вхождения в заголовок строк, valid - признак допустимости координаты.
