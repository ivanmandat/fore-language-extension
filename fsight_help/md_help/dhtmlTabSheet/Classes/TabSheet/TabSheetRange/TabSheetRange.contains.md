# TabSheetRange.contains


## Синтаксис


contains(range: [PP.Ui.TabSheetRange](TabSheetRange.htm));


## Параметры


*range.* Диапазон ячеек таблицы.


## Описание


Метод contains** определяет, содержит ли данный диапазон указанный диапазон ячеек таблицы.


## Комментарии


Метод возвращает значение true, если данный диапазон ячеек таблицы содержит указанный диапазон, и значение false - в противном случае.
