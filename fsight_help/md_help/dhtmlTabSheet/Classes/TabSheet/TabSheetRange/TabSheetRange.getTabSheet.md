# TabSheetRange.getTabSheet


## Синтаксис


getTabSheet();


## Описание


Метод getTabSheet** возвращает таблицу, к которой принадлежит данный диапазон.


## Комментарии


Метод возвращает объект типа [PP.Ui.TabSheet](../TabSheet/TabSheet.htm).
