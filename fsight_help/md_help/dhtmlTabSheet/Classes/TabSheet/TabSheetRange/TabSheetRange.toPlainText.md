# TabSheetRange.toPlainText


## Синтаксис


toPlainText(onlyVisibleRange: Boolean);


## Параметры


onlyVisibleRange. Признак того, что в текст нужно преобразовать данные только видимого диапазона. Если параметр равен true, то данные диапазона будут преобразованы в текст, иначе - не будут.


## Описание


Метод toPlainText** преобразует данные диапазона в обычный текст.


## Комментарии


Метод возвращает значение типа String.
