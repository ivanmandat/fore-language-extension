# TabSheetRange.rightExpand


## Синтаксис


rightExpand(amount: Number, ignoreInvalid: Boolean);


## Параметры


amount. Количество столбцов, на которое будет выполнено расширение;


ignoreInvalid. Признак игнорирования несуществующих столбцов таблицы. Если параметр равен значению true, то несуществующие столбцы будут игнорированы, если значению false, то не будут.


## Описание


Метод rightExpand** расширяет диапазон ячеек таблицы вправо на указанное количество столбцов.


## Комментарии


Метод возвращает значение типа Number - количество столбцов, на которое было выполнено расширение.
