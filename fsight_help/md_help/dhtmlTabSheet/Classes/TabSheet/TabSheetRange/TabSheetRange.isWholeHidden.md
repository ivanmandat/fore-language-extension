# TabSheetRange.isWholeHidden


## Синтаксис


isWholeHidden();


## Описание


Метод isWholeHidden** определяет, скрыты ли все ячейки данного диапазона.


## Комментарии


Метод возвращает значение true, если все ячейки диапазона скрыты, и значение false, если видима хотя бы одна ячейка диапазона.
