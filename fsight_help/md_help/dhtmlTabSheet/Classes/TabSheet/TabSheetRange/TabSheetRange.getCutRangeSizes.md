# TabSheetRange.getCutRangeSizes


## Синтаксис


getCutRangeSizes();


## Описание


Метод getCutRangeSizes** возвращает размеры диапазонов в пикселях, разъединённые линиями фиксации.


## Комментарии


Метод возвращает JSON-объект со следующими полями: tlSize - размеры (height - высота, width - ширина) верхнего левого, trSize - верхнего правого, blSize - нижнего левого и brSize - нижнего правого диапазона, status - [режим разбития](TabSheetRange.getCutStatus.htm) исходного диапазона.
