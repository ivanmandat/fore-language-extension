# TabSheetRange.eachRealCell


## Синтаксис


eachRealCell(action: function, context: Object);


## Параметры


*action.* Функция, вызываемая для каждой реальной ячейки данного диапазона;


*context.* Контекст, из которого вызван данный метод. Необязательный параметр, по умолчанию равен текущему контексту this.


## Описание


Метод eachRealCell** выполняет указанную функцию для каждой реальной ячейки данного диапазона.


## Комментарии


Под реальной ячейкой в таблице подразумевается непосредственно сама ячейка, а не её отображение в кеше.
