# TabSheetRange.isCuttedRange


## Синтаксис


isCuttedRange();


## Описание


Метод isCuttedRange** определяет, пересекает ли диапазон какая-либо линия фиксации.


## Комментарии


Метод возвращает значение true, если данный диапазон пересекает какая-либо линия фиксации, и значение false, если его не пересекает ни одна из этих линий.
