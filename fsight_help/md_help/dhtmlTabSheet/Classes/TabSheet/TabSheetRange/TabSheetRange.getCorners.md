# TabSheetRange.getCorners


## Синтаксис


getCorners();


## Описание


Метод getCorners** возвращает координаты угловых ячеек диапазона таблицы.


## Комментарии


Метод возвращает JSON-объект со следующими свойствами: tlCoord - координаты верхней левой ячейки, trCoord - верхней правой ячейки, blCoord - нижней левой ячейки, brCoord - нижней правой ячейки. Каждое свойство является объектом типа [PP.Ui.TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm).
