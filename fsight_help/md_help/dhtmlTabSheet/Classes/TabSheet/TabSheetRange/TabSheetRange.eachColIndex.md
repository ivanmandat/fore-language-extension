# TabSheetRange.eachColIndex


## Синтаксис


eachColIndex(action: function, context: Object, reverse: Boolean);


## Параметры


*action.* Функция, вызываемая при каждом обходе индексов столбцов таблицы;


*context.* Контекст, из которого вызван данный метод. Необязательный параметр, по умолчанию равен текущему контексту this;


*reverse.* Признак, определяющий направление обхода по индексам столбцов таблицы. Если параметр равен значению true, то обход начинается с индекса последнего столбца, если false - то наоборот. Необязательный параметр, по умолчанию равен значению false.


## Описание


Метод eachColIndex** выполняет указанную функцию при обходе индексов каждого столбца данного диапазона.
