# TabSheetRange.isVisible


## Синтаксис


isVisible();


## Описание


Метод isVisible** определяет, видима ли хотя бы одна ячейка из данного диапазона.


## Комментарии


Метод возвращает значение true, если хотя бы одна ячейка из данного диапазона видима, и значение false, если все ячейки невидимы.
