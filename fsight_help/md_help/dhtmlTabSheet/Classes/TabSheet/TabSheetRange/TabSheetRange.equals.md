# TabSheetRange.equals


## Синтаксис


equals(range: [PP.Ui.TabSheetRange](TabSheetRange.htm));


## Параметры


*range.* Диапазон таблицы, проверяемый на равенство с данный диапазоном.


## Описание


Метод equals** проверяет равенство двух диапазонов таблицы.


## Комментарии


Метод возвращает значение true, если оба диапазона равны, и значение false - в противном случае.
