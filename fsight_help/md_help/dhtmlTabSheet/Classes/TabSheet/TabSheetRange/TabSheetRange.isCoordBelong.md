# TabSheetRange.isCoordBelong


## Синтаксис


isCoordBelong(coord: PP.Ui.TabSheetCoord);


## Параметры


coord. Координаты ячейки таблицы.


## Описание


Метод isCoordBelong** определяет, принадлежит ли ячейка с указанными координатами данному диапазону.


## Комментарии


Метод возвращает значение true, если ячейка с указанными координатами принадлежит данному диапазону, и значение false, если не принадлежит.
