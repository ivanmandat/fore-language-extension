# TabSheetRange.intersects


## Синтаксис


intersects(range: [PP.Ui.TabSheetRange](TabSheetRange.htm));


## Параметры


range. Пересекаемый диапазон ячеек.


## Описание


Метод intersects** проверяет, пересекает ли указанный диапазон ячеек таблицы данный диапазон.


## Комментарии


Метод возвращает значение true, если указанный диапазон пересекает данный диапазон ячеек таблицы, иначе - значение false.
