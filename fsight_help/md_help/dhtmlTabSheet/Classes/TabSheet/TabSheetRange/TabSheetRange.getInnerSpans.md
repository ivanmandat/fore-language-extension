# TabSheetRange.getInnerSpans


## Синтаксис


getInnerSpans();


## Описание


Метод getInnerSpans** возвращает массив объединённых ячеек , которые попадают в данный диапазон.


## Комментарии


Метод возвращает значение типа [PP.Ui.TabSheetCellSpan](../TabSheetCellSpan/TabSheetCellSpan.htm).
