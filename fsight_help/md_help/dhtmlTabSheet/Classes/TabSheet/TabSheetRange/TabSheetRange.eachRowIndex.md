# TabSheetRange.eachRowIndex


## Синтаксис


eachRowIndex(action: function, context: Object, reverse: Boolean);


## Параметры


*action.* Функция, вызываемая при каждом обходе индексов строк таблицы;


*context.* Контекст, из которого вызван данный метод. Необязательный параметр, по умолчанию равен текущему контексту this;


*reverse.* Признак, определяющий направление обхода по индексам строк таблицы. Если параметр равен значению true, то обход начинается с индекса последней строки, если false - то наоборот. Необязательный параметр, по умолчанию равен значению false.


## Описание


Метод eachRowIndex** выполняет указанную функцию при обходе индексов каждой строки данного диапазона.
