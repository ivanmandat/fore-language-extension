# TabSheetRange.getInnerVirtualSizeInPX


## Синтаксис


getInnerVirtualSizeInPX();


## Описание


Метод getInnerVirtualSizeInPX** возвращает размеры данного диапазона в пикселях.


## Комментарии


Метод возвращает JSON-объект со следующими свойствами: width - ширина и height - высота диапазона.
