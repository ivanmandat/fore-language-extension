# TabSheetRange.getIntersectionWith


## Синтаксис


getIntersectionWith(range: [PP.Ui.TabSheetRange](TabSheetRange.htm));


## Параметры


range. Диапазон ячеек таблицы.


## Описание


Метод getIntersectionWith** возвращает область, общую для данного и указанного диапазона.


## Комментарии


Метод возвращает объект типа [PP.Ui.TabSheetRange](TabSheetRange.htm).
