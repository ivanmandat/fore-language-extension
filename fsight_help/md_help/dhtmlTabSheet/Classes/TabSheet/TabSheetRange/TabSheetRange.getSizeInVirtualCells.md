# TabSheetRange.getSizeInVirtualCells


## Синтаксис


getSizeInVirtualCells();


## Описание


Метод getSizeInVirtualCells** возвращает размер данного диапазона, выраженный в общем количестве охватываемых им строк и столбцов.


## Комментарии


Метод возвращает JSON-объект со следующими свойствами: rowsCount - общее количество строк, охватываемых данным диапазоном, columnsCount - общее количество столбцов.
