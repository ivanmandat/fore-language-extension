# TabSheetRange.setEnabled


## Синтаксис


setEnabled(value);


## Параметры


*value.* Признак доступности диапазона. Если параметр равен true, то все ячейки данного диапазона будут доступны, иначе - недоступны.


## Описание


Метод setEnabled** управляет доступностью ячеек данного диапазона таблицы.
