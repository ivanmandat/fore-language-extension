# TabSheet.showColumns


## Синтаксис


showColumns(colIndex: Number, count: Number);


## Параметры


*colIndex.* Индекс первого отображаемого столбца;


*count.* Количество отображаемых столбцов.


## Описание


Метод showColumns** отображает скрытые столбцы таблицы.
