# TabSheet.getChangedCells


## Синтаксис


getChangedCells();


## Описание


Метод getChangedCells** возвращает массив данных изменённых ячеек таблицы.


## Комментарии


Метод возвращает массив JSON-объектов со следующими полями: CellData - объект со значениями ячейки, V - её новое значение, S - стиль и др.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 [TabSheet](../../../Components/TabSheet/TabSheet/TabSheet.htm)
 с наименованием «tabSheet» (см. «[Пример
 создания компонента TabSheet](../../../Components/TabSheet/TabSheet/TabSheet_Example.htm)»). Изменим значения двух ячеек таблицы
 и очистим всю информацию о произошедших изменениях. Для контроля данных
 действий будем определять общее количество изменённых ячеек до и после
 их редактирования, а также после очистки изменений:


// Изменим значения определенных ячеек
tabSheet.setCellValue("newData1", 0, 0);
tabSheet.setCellValue("newData2", 0, 1);
console.log("Количество измененных ячеек после редактирования: " + tabSheet.getChangedCells().length);
// Получим массив измененных ячеек
var changedCells = tabSheet.getChangedCells();
// Выведем информацию об измененных ячейках
for (var i in changedCells) {
    console.log("Значение ячейки с координатами (" + changedCells[i].L + ", " + changedCells[i].T + "): " + changedCells[i].CellData.V);
}
console.log("Количество измененных ячеек: " + tabSheet.getChangedCells().length);


В результате выполнения примера в консоли браузера будет выведено общее
 количество изменённых ячеек до и после их редактирования, а также
 после очистки изменений:


Количество измененных ячеек до редактирования: 0


Количество измененных ячеек после редактирования:
 2


Значение ячейки с координатами (0, 0): newData1


Значение ячейки с координатами (1, 0): newData2


Количество измененных ячеек: 0
