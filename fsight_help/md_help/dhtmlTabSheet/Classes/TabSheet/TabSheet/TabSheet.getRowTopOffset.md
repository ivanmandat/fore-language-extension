# TabSheet.getRowTopOffset


## Синтаксис


getRowTopOffset(rowIndex: Number);


## Параметры


*rowIndex.* Индекс строки таблицы.


## Описание


Метод getRowTopOffset** возвращает координаты, с которой начинается строка таблицы.


## Комментарии


Метод возвращает объект типа PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm).
