# TabSheet.split


## Синтаксис


split(range: PP.Ui.[TabSheetRange](../TabSheetRange/TabSheetRange.htm));


## Параметры


*range.* Диапазон объединённых ячеек, который нужно разъединить.


## Описание


Метод split** разъединяет объединённые ячейки таблицы.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента [TabSheet](../../../Components/TabSheet/TabSheet/TabSheet.htm) с наименованием «tabSheet» (см. «[Пример создания компонента TabSheet](../../../Components/TabSheet/TabSheet/TabSheet_Example.htm)»). Объединим и разъединим диапазон ячеек:


// Получим диапазон ячеек по заданным координатам
var range = tabSheet.getRange(0, 0, 1, 1);
// Объединим ячейки в указанном диапазоне
tabSheet.merge(range);
var isSplit = confirm("Разъединить ячейки?");
if (isSplit) {
    // Разобъём диапазон объединенённых ячеек
    tabSheet.split(range);
    // Выделим диапазон ячеек
    range.select()
}

В результате выполнения примера были объединены ячейки в диапазоне (0, 0) - (1, 1):


![](TabSheet.split_2.png)


После этого был показан диалог, запрашивающий подтверждение на разъединение диапазона объединённых ячеек. В результате нажатия кнопки «OK» данный диапазон был разбит на отдельные ячейки и выделен:
