# TabSheet.setClassToRange


## Синтаксис


setClassToRange(ranges: Array, cssClass: String, remove: Boolean);


## Параметры


ranges. Массив диапазонов ячеек;


cssClass. Наименование CSS-класса;


remove. Признак удаления существующих настроек.


## Описание


Метод setClassToRange устанавливает настройки указанного CSS-класса для диапазона ячеек.
