# TabSheet.HintChanged


## Синтаксис


HintChanged: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие HintChanged** генерируется при изменении всплывающей подсказки у диапазона ячеек таблицы.
