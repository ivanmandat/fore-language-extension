# TabSheet.getCellControlIndex


## Синтаксис


getCellControlIndex(needCoord: PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm));


## Параметры


*needCoord.* Координата ячейки, в которой расположен требуемый элемент управления.


## Описание


Метод getCellControlIndex** возвращает индекс элемента управления по координатам ячейки, в которой он расположен.


## Комментарии


Метод возвращает значение типа Number.
