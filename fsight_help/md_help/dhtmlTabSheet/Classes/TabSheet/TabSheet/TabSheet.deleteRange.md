# TabSheet.deleteRange


## Синтаксис


deleteRange(range: Array, isCut: Boolean);


## Параметры


*r*ange. Массив с диапазоном ячеек;


*isCut.* Определяет, удалять или вырезать ячейки. Необязательный параметр. Если параметр равен true, то ячейки будут вырезаны, если false, то удалены.


## Описание


Метод deleteRange** удаляет диапазоны ячеек в таблице.
