# TabSheet.endUpdate


## Синтаксис


endUpdate();


## Описание


Метод endUpdate включает перерисовку таблицы и обновляет её.


## Комментарии


Использует в связке с методом [beginUpdate](TabSheet.beginUpdate.htm).
