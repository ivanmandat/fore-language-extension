# TabSheet.parsePlainText


## Синтаксис


parsePlainText(text: String);


## Параметры


*text.* Текст, который нужно десериализовать в формат источника данных.


## Описание


Метод parsePlainText** десериализует текст диапазона таблицы в формат источника данных.


## Комментарии


Метод возвращает значение типа String.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента [TabSheet](../../../Components/TabSheet/TabSheet/TabSheet.htm) с наименованием «tabSheet» (см. «[Пример создания компонента TabSheet](../../../Components/TabSheet/TabSheet/TabSheet_Example.htm)»). Десериализуем текст диапазона таблицы в формат источника данных:


/* Определим входную строку, которая будет анализироваться.
   Для задания значений нескольких ячеек используются escape-последовательности.
   Адрес начальной ячейки равен (0,0)
   Использование escape-последовательности "\n" означает, что текст, следующий за ней,
   станет значением ячейки в следующей строке в этом же столбце
   Использование escape-последовательности "\t" означает, что текст, следующий за ней, станет значением ячейки в следующем столбце в этой же строке
 */
var text = "cell(0, 0)\nCell(0, 1)\nCell(0, 2)\tCell(1, 2)\tCell(2, 2)";
// Получим результат анализа строки
var res = tabSheet.parsePlainText(text);
// Получим массив ячеек
var arr = res.Cells.Cell;
// Выведем значения ячеек
for (var i in arr) {
    console.log("Значение ячейки с координатами (" + arr[i].L + ", " + arr[i].T + "): " + arr[i].CellData.V)
};

В результате выполнения примера был десериализован диапазон ячеек таблицы, а в консоли браузера были выведены их сгенерированные значения:


Значение ячейки с координатами (0, 0): cell(0, 0)

Значение ячейки с координатами (0, 1): Cell(0, 1)

Значение ячейки с координатами (0, 2): Cell(0, 2)

Значение ячейки с координатами (1, 2): Cell(1, 2)

Значение ячейки с координатами (2, 2): Cell(2, 2)
