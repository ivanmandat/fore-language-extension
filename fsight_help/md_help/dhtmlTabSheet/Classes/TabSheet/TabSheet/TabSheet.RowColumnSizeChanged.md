# TabSheet.RowColumnSizeChanged


## Синтаксис


RowColumnSizeChanged: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: newValue - новое значение ширины колонки или высоты строки таблицы, rowIndexes - массив индексов строк, colIndexes - массив индексов столбцов.


## Описание


Событие RowColumnSizeChanged** наступает после изменения размеров строк и столбцов таблицы.
