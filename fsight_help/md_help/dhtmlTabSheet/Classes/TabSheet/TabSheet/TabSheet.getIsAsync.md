# TabSheet.getIsAsync


## Синтаксис


getIsAsync();


## Описание


Метод getIsAsync** возвращает режим работы с источником данных.


## Комментарии


Если режим работы с источником данных является асинхронным, то метод возвращает значение true, если синхронным, то значение false.
