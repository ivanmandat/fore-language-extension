# TabSheet.ensureRangeVisibility


## Синтаксис


ensureRangeVisibility(range: PP.Ui.[TabSheetRange](../TabSheetRange/TabSheetRange.htm));


## Параметры


*range.* Диапазон ячеек, который нужно отобразить в видимой области таблицы.


## Описание


Метод ensureRangeVisibility** изменяет видимую область таблицы так, чтобы отображался указанный диапазон ячеек.
