# TabSheet.getCellSpanUnderCoord


## Синтаксис


getCellSpanUnderCoord(coord: PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm));


## Параметры


*coord.* Координата ячейки.


## Описание


Метод getCellSpanUnderCoord** возвращает диапазон объединённых ячеек по координате ячейки, входящей в него.


## Комментарии


Метод возвращает объект типа PP.Ui.[TabSheetCellSpan](../TabSheetCellSpan/TabSheetCellSpan.htm).
