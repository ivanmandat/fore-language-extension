# TabSheet.getSelectedRanges


## Синтаксис


getSelectedRanges();


## Описание


Метод getSelectedRanges** возвращает все выделенные диапазоны в таблице.


## Комментарии


Метод возвращает массив объектов типа PP.Ui.[TabSheetRange](../TabSheetRange/TabSheetRange.htm).
