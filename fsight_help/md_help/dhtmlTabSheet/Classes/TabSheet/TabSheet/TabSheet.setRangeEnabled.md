# TabSheet.setRangeEnabled


## Синтаксис


setRangeEnabled(range: PP.Ui.[TabSheetRange](../TabSheetRange/TabSheetRange.htm), enabled: Boolean);


## Параметры


*range.* Диапазон ячеек таблицы;


*enabled.* Признак доступности диапазона ячеек. Если значение параметра true, то указанный диапазон будет доступный, иначе - не доступный.


## Описание


Метод setRangeEnabled** управляет доступностью диапазона ячеек.
