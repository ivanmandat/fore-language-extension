# TabSheet.getHeaderCellContentId


## Синтаксис


getHeaderCellContentId(colIndex: Number);


## Параметры


*colIndex.* Индекс столбца таблицы.


## Описание


Метод getHeaderCellContentId** возвращает идентификатор содержимого заголовка столбца таблицы.


## Комментарии


Метод возвращает значение типа String.
