# TabSheet.getCoord


## Синтаксис


getCoord(rowIndex: Number, colIndex: Number);


## Параметры


*rowIndex.* Индекс строки;


*colIndex.* Индекс столбца.


## Описание


Метод getCoord** возвращает
 координаты ячейки по индексу строки и столбца таблицы.


## Комментарии


Метод возвращает значение типа [PP.Ui.TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm).
