# TabSheet.getRowNode


## Синтаксис


getRowNode(rowIndex: Number);


## Параметры


*rowIndex.* Индекс строки таблицы.


## Описание


Метод getRowNode** возвращает DOM-элемент строки таблицы.


## Комментарии


Метод возвращает значение типа HTMLTableRowElement.
