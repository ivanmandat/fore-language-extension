# TabSheet.scrollToRowAsToLast


## Синтаксис


scrollToRowAsToLast(rowIndex: Number);


## Параметры


*rowIndex.* Индекс строки, к которой нужно прокрутить таблицу.


## Описание


Метод scrollToRowAsToLast** выполняет прокрутку таблицы к указанной строке, чтобы она была последней видимой.
