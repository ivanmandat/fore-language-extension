# TabSheet.isRowVisible


## Синтаксис


isRowVisible(rowIndex: Number);


## Параметры


*rowIndex.* Индекс строки таблицы.


## Описание


Метод isRowVisible** определяет видимость строки таблицы.


## Комментарии


Метод возвращает значение true, если указанный столбец таблицы отображён, иначе - значение false.
