# TabSheet.MouseMove


## Синтаксис


MouseMove: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: Event - событие перемещения курсора мыши, Element - DOM-элемент ячейки под курсором мыши, Break - признак отмены события (если равен true, то событие отменено, иначе - разрешено).


## Описание


Событие MouseMove** генерируется при перемещении курсора мыши.
