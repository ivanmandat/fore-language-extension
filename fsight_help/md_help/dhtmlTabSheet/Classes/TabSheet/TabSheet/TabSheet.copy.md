# TabSheet.copy


## Синтаксис


copy(range: PP.Ui.[TabSheetRange](../TabSheetRange/TabSheetRange.htm));


## Параметры


*range.* Диапазон ячеек. Необязательный параметр.


## Описание


Метод copy** возвращает содержимое указанного или выделенного диапазона ячеек таблицы.


## Комментарии


Если параметр не указан, то метод возвращает содержимое выделенного диапазона ячеек.


Возвращаемое значение является строкой.
