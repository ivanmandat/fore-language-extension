# TabSheet.coord


## Синтаксис


coord(rowIndex: Number, colIndex: Number);


## Параметры


*rowIndex.* Индекс строки;


*colIndex.* Индекс столбца.


## Описание


Метод coord** возвращает
 координаты ячейки по индексу строки и столбца таблицы.


## Комментарии


Метод устарел, используйте [TabSheet.getCoord](TabSheet.getCoord.htm).
