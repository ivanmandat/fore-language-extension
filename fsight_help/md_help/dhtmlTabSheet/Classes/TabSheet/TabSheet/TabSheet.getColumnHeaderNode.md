# TabSheet.getColumnHeaderNode


## Синтаксис


getColumnHeaderNode(colIndex: Number);


## Параметры


*colIndex.* Индекс столбца таблицы.


## Описание


Метод getColumnHeaderNode** возвращает DOM-элемент заголовка столбца.


## Комментарии


Метод возвращает значение типа HTMLTableCellElement.
