# TabSheet.hideRows


## Синтаксис


hideRows(rowIndex: Number, count: Number);


## Параметры


*rowIndex.* Индекс первой скрываемой строки;


*count.* Количество скрываемых строк.


## Описание


Метод hideRows** скрывает строки таблицы.
