# TabSheet.hideColumns


## Синтаксис


hideColumns(colIndex: Number, count: Number);


## Параметры


*colIndex.* Индекс первого скрываемого столбца;


*count.* Количество скрываемых столбцов.


## Описание


Метод hideColumns** скрывает столбцы таблицы.
