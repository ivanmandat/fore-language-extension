# TabSheet.getCoordByNode


## Синтаксис


getCoordByNode(tdNode: HTMLTableCellElement);


## Параметры


*tdNode.* DOM-элемент ячейки таблицы.


## Описание


Метод getCoordByNode** возвращает координаты ячейки по её DOM-элементу.


## Комментарии


Метод возвращает объект типа PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm).
