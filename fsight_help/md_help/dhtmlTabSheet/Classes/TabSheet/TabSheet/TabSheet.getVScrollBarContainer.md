# TabSheet.getVScrollBarContainer


## Синтаксис


getVScrollBarContainer();


## Описание


Метод getVScrollBarContainer** возвращает DOM-элемент контейнера вертикальной полосы прокрутки.


## Комментарии


Метод возвращает значение типа HTMLDivElement.
