# TabSheet.getClipboardContainsData


## Синтаксис


getClipboardContainsData();


## Описание


Метод getClipboardContainsData** извлекает содержимое буфера.


## Комментарии


Метод возвращает значение true, если содержимое буфера непусто, иначе - значение false.
