# TabSheet.scrollToColumn


## Синтаксис


scrollToColumn(colIndex: Number);


## Параметры


*colIndex.* Индекс столбца таблицы.


## Описание


Метод scrollToColumn** выполняет прокрутку таблицы к указанному столбцу.


## Комментарии


Метод возвращает значение true, если прокрутка таблицы была выполнена, иначе - значение false.
