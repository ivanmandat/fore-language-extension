# TabSheet.EditingEnd


## Синтаксис


EditingEnd: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.
 Доступные аргументы: wasModify - были ли изменения в ячейке, before -
 массив ячеек таблицы до их редактирования, after - массив ячеек
 таблицы после их редактирования.


## Описание


Событие EditingEnd наступает
 после редактирования ячейки таблицы.


## Комментарии


Данное событие доступно, если существует возможность выделения ячеек
 таблицы.
