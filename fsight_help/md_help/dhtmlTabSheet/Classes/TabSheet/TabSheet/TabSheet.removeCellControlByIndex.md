# TabSheet.removeCellControlByIndex


## Синтаксис


removeCellControlByIndex(index: Number);


## Параметры


*index.* Индекс элемента управления.


## Описание


Метод removeCellControlByIndex** удаляет элемент управления в ячейке таблицы по его индексу.


## Комментарии


Индекс элемента управления можно узнать с помощью метода [TabSheet.getCellControlIndex](TabSheet.getCellControlIndex.htm).


Метод возвращает значение true в случае успешного удаления элемента управления и false - в противном случае.
