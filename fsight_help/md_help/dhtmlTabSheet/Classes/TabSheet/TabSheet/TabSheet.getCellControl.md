# TabSheet.getCellControl


## Синтаксис


getCellControl(needCoord: PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm));


## Параметры


*needCoord.* Координата ячейки, в которой расположен требуемый элемент управления.


## Описание


Метод getCellControl** возвращает элемент управления по координатам ячейки, в которой он расположен.


## Комментарии


Метод возвращает объект типа PP.Ui.[Control](dhtmlUi.chm::/Classes/control/control.htm) или унаследованного от него типа.
