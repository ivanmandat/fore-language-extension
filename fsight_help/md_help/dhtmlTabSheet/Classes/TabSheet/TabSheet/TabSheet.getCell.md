# TabSheet.getCell


## Синтаксис


getCell(rowIndex: Number, colIndex: Number);


## Параметры


*rowIndex.* Индекс строки ячейки;


*colIndex.* Индекс столбца ячейки.


## Описание


Метод getCell** возвращает диапазон, состоящий из одной ячейки.


## Комментарии


Метод возвращает объект типа PP.Ui.[TabSheetRange](../TabSheetRange/TabSheetRange.htm).
