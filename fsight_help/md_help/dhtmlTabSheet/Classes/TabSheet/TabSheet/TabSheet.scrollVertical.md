# TabSheet.scrollVertical


## Синтаксис


scrollVertical(posInPx: Number);


## Параметры


*posInPx.* Количество виртуальных пикселей, на которое нужно осуществить прокрутку..


## Описание


Метод scrollVertical** осуществляет вертикальную прокрутку таблицы.
