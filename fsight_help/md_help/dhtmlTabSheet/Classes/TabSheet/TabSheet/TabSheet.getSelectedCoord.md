# TabSheet.getSelectedCoord


## Синтаксис


getSelectedCoord();


## Описание


Метод getSelectedCoord** возвращает начальные координаты выделенного диапазона в таблице.


## Комментарии


Метод возвращает объект типа PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm).
