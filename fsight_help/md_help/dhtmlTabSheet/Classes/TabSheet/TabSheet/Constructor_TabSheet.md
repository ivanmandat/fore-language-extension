# Конструктор TabSheet


## Синтаксис


PP.Ui.TabSheet(settings);


## Параметры


*settings.* [JSON-объект](../../../Components/TabSheet/TabSheet/TabSheet_JSON.htm)
 со значениями свойств интерфейса [TabSheetSettings](../../../Interfaces/TabSheetSettings/TabSheetSettings.htm).


## Описание


Конструктор TabSheet**
 создает экземпляр компонента **[TabSheet](../../../Components/TabSheet/TabSheet/TabSheet.htm).**
