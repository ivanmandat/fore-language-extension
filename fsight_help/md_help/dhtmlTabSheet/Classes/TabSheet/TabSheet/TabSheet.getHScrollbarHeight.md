# TabSheet.getHScrollbarHeight


## Синтаксис


getHScrollbarHeight();


## Описание


Метод getHScrollbarHeight** возвращает высоту горизонтального полосы прокрутки в пикселях.


## Комментарии


Метод возвращает значение типа Number.
