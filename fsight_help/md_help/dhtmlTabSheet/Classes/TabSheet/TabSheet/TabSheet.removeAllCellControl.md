# TabSheet.removeAllCellControl


## Синтаксис


removeAllCellControl();


## Описание


Метод removeAllCellControl** удаляет все элементы управления в ячейках таблицы.


## Комментарии


Метод возвращает значение true в случае успешного удаления элемента управления и false - в противном случае.
