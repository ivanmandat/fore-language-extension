# TabSheet.getDynamicExpansion


## Синтаксис


getDynamicExpansion();


## Описание


Метод getDynamicExpansion возвращает
 признак использования динамического увеличения количества строк/столбцов
 при достижении последней заполненной ячейки таблицы.


## Комментарии


Для установки признака динамического увеличения количества строк/столбцов
 используйте метод [setDynamicExpansion](TabSheet.setDynamicExpansion.htm)
 или задайте значение свойства [DynamicExpansion](../../../Interfaces/TabSheetSettings/TabSheetSettings.DynamicExpansion.htm)
 в [конструкторе TabSheet](Constructor_TabSheet.htm).
