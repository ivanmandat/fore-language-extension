# TabSheet.Editing


## Синтаксис


Editing: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: Cell - ячейка таблицы, представленная объектом типа PP.Ui.[TabSheetCell](../TabSheetCell/TabSheetCell.htm), Cancel - признак отмены события.


## Описание


Событие Editing наступает в процессе редактирования ячейки таблицы.


## Комментарии


Данное событие доступно, если существует возможность выделения ячеек таблицы.
