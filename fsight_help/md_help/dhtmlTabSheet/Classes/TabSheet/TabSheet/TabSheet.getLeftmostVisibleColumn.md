# TabSheet.getLeftmostVisibleColumn


## Синтаксис


getLeftmostVisibleColumn();


## Описание


Метод getLeftmostVisibleColumn** возвращает столбец, ближайший к левому краю текущей видимой области таблицы.


## Комментарии


Метод возвращает значение типа Number.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента [TabSheet](../../../Components/TabSheet/TabSheet/TabSheet.htm) с наименованием «tabSheet» (см. «[Пример создания компонента TabSheet](../../../Components/TabSheet/TabSheet/TabSheet_Example.htm)»). Выполним горизонтальную и вертикальную прокрутку таблицы, а затем получим координаты ячейки, ближайшей к левому верхнему краю текущей видимой области таблицы:


// Выполним горизонтальную и вертикальную прокрутку таблицы
tabSheet.scrollHorizontal(300);
tabSheet.scrollVertical(200);
// Получим строку, ближайшую к верхнему краю текущей видимой области таблицы
var row = tabSheet.getTopmostVisibleRow();
// Получим столбец, ближайший к левому краю текущей видимой области таблицы
var column = tabSheet.getLeftmostVisibleColumn();
console.log("Координата ячейки, ближайший к левому верхнему краю текущей видимой области таблицы: (" + row + ", " + column + ")");

В результате выполнения примера была выполнена горизонтальная и вертикальная прокрутка таблицы, а в консоли браузера были выведены координаты ячейки, ближайшей к левому верхнему краю текущей видимой области таблицы:


Координата ячейки, ближайший к левому верхнему краю текущей видимой области таблицы: (3, 2)
