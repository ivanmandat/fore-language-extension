# TabSheet.getExpanderUnderCoord


## Синтаксис


getExpanderUnderCoord(coord: PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm));


## Параметры


*coord.* Координата, по которой требуется получить кнопку развёртывания.


## Описание


Метод getExpanderUnderCoord** возвращает кнопку развёртывания по указанной координате.


## Комментарии


Метод возвращает объект типа PP.Ui.[TabSheetExpander](../TabSheetExpander/TabSheetExpander.htm).
