# TabSheet.setRowsHeight


## Синтаксис


setRowsHeight(value: Number, rowIndex: Number, count: Number, fireEvent: Boolean);


## Параметры


*value.* Устанавливаемое значение высоты;


*rowIndex.* Индекс строки, начиная с которой устанавливается новая высота;


*count.* Количество строк;


*fireEvent.* Определяет, генерировать ли событие установки новой высоты для строк таблицы. Если параметр равен значению true, то данное событие будет сгенерировано, иначе - не будет


## Описание


Метод setRowsHeight** устанавливает высоту строк таблицы.
