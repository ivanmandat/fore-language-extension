# TabSheet.paste


## Синтаксис


paste(text: String, rowIndex: Number, colIndex: Number);


## Параметры


*text.* Вставляемый текст;


*rowIndex.* Индекс строки ячейки, в которую вставляется текст;


*colIndex.* Индекс колонки ячейки, в которую вставляется текст.


## Описание


Метод paste** вставляет текст в ячейку с указанными координатами.
