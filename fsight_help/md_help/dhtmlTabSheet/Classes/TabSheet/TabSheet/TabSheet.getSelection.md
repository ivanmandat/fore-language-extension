# TabSheet.getSelection


## Синтаксис


getSelection(ensureExistence: Boolean);


## Параметры


*ensureExistence.* Признак выделения левой верхней ячейки таблицы. Если параметр равен значению true, то будет выделена левая верхняя ячейка таблицы, иначе - не будет.


## Описание


Метод getSelection** возвращает объект выделения в таблице.


## Комментарии


Метод возвращает объект типа PP.Ui.[TabSheetSelection](../TabSheetSelection/TabSheetSelection.htm).
