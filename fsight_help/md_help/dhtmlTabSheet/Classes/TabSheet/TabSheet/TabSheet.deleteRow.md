# TabSheet.deleteRow


## Синтаксис


deleteRow(rowIndex: Number, count: Number, fireEvent: Boolean, callback: PP.Delegate);


## Параметры


*rowIndex*. Индекс первой удаляемой строки таблицы;


*count.* Количество удаляемых строк;


*fireEvent.* *Определяет, генерировать ли событие удаления строк таблицы*. Если параметр равен значению true, то данное событие будет сгенерировано, иначе - не будет;


*callback.* Возвратная функция.


## Описание


Метод deleteRow** удаляет строки таблицы.
