# TabSheet.IsTabSheetTabObjectExist


## Синтаксис


IsTabSheetTabObjectExist();


## Описание


Метод IsTabSheetTabObjectExist возвращает признак наличия объектов в таблице.


## Комментарии


Метод возвращает значение типа Boolean.


Если в таблице присутствуют объекты, то метод возвращает значение true, иначе - false.
