# TabSheet.deleteRows


## Синтаксис


deleteRows(indexes: Array, fireEvent: Boolean, callback: PP.Delegate);


## Параметры


*indexes.* Массив индексов строк, строго расположенных по возрастанию;


*fireEvent.* *Определяет, генерировать ли событие удаления строк таблицы*. Если параметр равен значению true, то данное событие будет сгенерировано, иначе - не будет;


*callback.* Возвратная функция.


## Описание


Метод deleteRows** удаляет массив строк таблицы.
