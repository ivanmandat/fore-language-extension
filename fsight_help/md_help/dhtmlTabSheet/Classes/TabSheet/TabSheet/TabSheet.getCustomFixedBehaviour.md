# TabSheet.getCustomFixedBehaviour


## Синтаксис


getCustomFixedBehaviour();


## Описание


Метод getCustomFixedBehaviour возвращает определённое пользователем поведение фиксированных строк и столбцов таблицы при нажатии на них.


## Комментарии


Метод возвращает элемент перечисления PP.Ui.[TabFixedBehaviour](../../../Enums/TabFixedBehaviour.htm).
