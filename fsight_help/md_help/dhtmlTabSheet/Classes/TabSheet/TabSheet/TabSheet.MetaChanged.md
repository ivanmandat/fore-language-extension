# TabSheet.MetaChanged


## Синтаксис


MetaChanged: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Значение параметра определяется с помощью элемента перечисления PP.Ui.[TabSheetMetaChangeType](../../../Enums/TabSheetMetaChangeType.htm).


## Описание


Событие MetaChanged** генерируется при отсутствии данных в источнике таблицы.
