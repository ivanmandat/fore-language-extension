# TabSheet.getRowHeaderNode


## Синтаксис


getRowHeaderNode(rowIndex: Number);


## Параметры


*rowIndex.* Индекс строки таблицы.


## Описание


Метод getRowHeaderNode** возвращает DOM-элемент заголовка строки.


## Комментарии


Метод возвращает значение типа HTMLTableCellElement.
