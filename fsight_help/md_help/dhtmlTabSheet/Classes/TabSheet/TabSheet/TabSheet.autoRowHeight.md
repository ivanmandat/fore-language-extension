# TabSheet.autoRowHeight


## Синтаксис


autoRowHeight(rowIndex: Number, fully: Boolean);


## Параметры


*rowIndex.* Индекс строки, высоту которой нужно подогнать;


fully. Определяет, будут ли при подгонке высоты строк учитываться невидимые ячейки. По умолчанию установлено значение true, то есть будут учитываться. Если установлено значение false, невидимые ячейки учитываться не будут.


## Описание


Метод autoRowHeight** автоматически подгоняет высоту строк таблицы.
