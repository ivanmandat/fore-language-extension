# TabSheet.getIsEditing


## Синтаксис


getIsEditing();


## Описание


Метод getIsEditing возвращает признак процесса редактирования ячейки таблицы.


## Комментарии


Метод возвращает значение типа Boolean.


Если метод возвращает значение true, то в текущий момент редактируется ячейка таблицы, иначе - не редактируется.
