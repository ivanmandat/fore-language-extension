# TabSheet.isLastVisibleRow


## Синтаксис


isLastVisibleRow(rowIndex: Number);


## Параметры


*rowIndex.* Индекс строки
 таблицы.


## Описание


Метод isLastVisibleRow**
 определяет, является ли указанная строка последней видимой.


## Комментарии


Метод возвращает значение true,
 если указанная строка является последней из всех видимых строк таблицы,
 иначе - значение false.
