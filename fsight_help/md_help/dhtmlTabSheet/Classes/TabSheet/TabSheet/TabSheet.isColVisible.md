# TabSheet.isColVisible


## Синтаксис


isColVisible(colIndex: Number);


## Параметры


*colIndex.* Индекс столбца таблицы.


## Описание


Метод isColVisible** определяет видимость столбца таблицы.


## Комментарии


Метод возвращает значение true, если указанный столбец таблицы отображён, иначе - значение false.
