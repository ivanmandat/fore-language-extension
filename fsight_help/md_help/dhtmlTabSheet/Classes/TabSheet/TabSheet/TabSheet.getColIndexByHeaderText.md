# TabSheet.getColIndexByHeaderText


## Синтаксис


getColIndexByHeaderText(str: String);


## Параметры


str. Текст заголовка.


## Описание


Метод getColIndexByHeaderText возвращает индекс столбца таблицы по тексту его заголовка.


## Комментарии


Метод возвращает значение типа Number.
