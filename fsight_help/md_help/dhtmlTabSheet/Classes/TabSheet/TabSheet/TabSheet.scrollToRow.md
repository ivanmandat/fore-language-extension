# TabSheet.scrollToRow


## Синтаксис


scrollToRow(rowIndex: Number);


## Параметры


*rowIndex.* Индекс строки таблицы.


## Описание


Метод scrollToRow** выполняет прокрутку таблицы к указанной строке.


## Комментарии


Метод возвращает значение true, если прокрутка таблицы была выполнена, иначе - значение false.
