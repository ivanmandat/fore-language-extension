# TabSheet.isColInsideVisibleRange


## Синтаксис


isColInsideVisibleRange(colIndex: Number);


## Параметры


*colIndex.* Индекс столбца таблицы.


## Описание


Метод isColInsideVisibleRange** определяет, находится ли столбец внутри видимого диапазона.


## Комментарии


Метод возвращает значение true, если указанный столбец находится внутри видимого диапазона, иначе - значение false.
