# TabSheet.isRowInsideVisibleRange


## Синтаксис


isRowInsideVisibleRange(rowIndex: Number);


## Параметры


*rowIndex.* Индекс строки таблицы.


## Описание


Метод isRowInsideVisibleRange** определяет, находится ли строка внутри видимого диапазона таблицы.


## Комментарии


Метод возвращает значение true, если указанная строка находится внутри видимого диапазона, иначе - значение false.
