# TabSheet.getTopmostVisibleRow


## Синтаксис


getTopmostVisibleRow();


## Описание


Метод getTopmostVisibleRow** возвращает ближайшую к левому краю текущей видимой области строку.


## Комментарии


Метод возвращает значение типа Number.
