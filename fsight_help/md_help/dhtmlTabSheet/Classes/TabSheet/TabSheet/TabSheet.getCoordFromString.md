# TabSheet.getCoordFromString


## Синтаксис


getCoordFromString(str: String);


## Параметры


str. Координаты в строковом представлении.


## Описание


Метод getCoordFromString возвращает координаты ячейки по указанному строковому представлению.


## Комментарии


Метод возвращает значение типа [PP.Ui.TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm).
