# TabSheet.undoAll


## Синтаксис


undoAll();


## Описание


Метод undoAll** отменяет все последние изменения в таблице, количество которых устанавливается свойством [TabSheetSettings.UndoLength](../../../Interfaces/TabSheetSettings/TabSheetSettings.UndoLength.htm).
