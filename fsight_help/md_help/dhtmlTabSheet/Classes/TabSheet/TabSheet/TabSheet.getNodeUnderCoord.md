# TabSheet.getNodeUnderCoord


## Синтаксис


getNodeUnderCoord(rowIndex: Number, colIndex: Number);


## Параметры


*rowIndex.* Индекс строки ячейки;


*colIndex.* Индекс столбца ячейки.


## Описание


Метод getNodeUnderCoord** возвращает DOM-элемент ячейки таблицы по указанной координате.


## Комментарии


Метод возвращает объект типа HTMLTableCellElement.
