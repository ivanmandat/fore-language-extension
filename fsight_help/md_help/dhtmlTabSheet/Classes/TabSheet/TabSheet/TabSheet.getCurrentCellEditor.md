# TabSheet.getCurrentCellEditor


## Синтаксис


getCurrentCellEditor();


## Описание


Метод getCurrentCellEditor** возвращает текущий активный редактор ячейки.


## Комментарии


Метод возвращает значение типа PP.Ui.[TabSheetDefaultEditor](../TabSheetDefaultEditor/TabSheetDefaultEditor.htm).
