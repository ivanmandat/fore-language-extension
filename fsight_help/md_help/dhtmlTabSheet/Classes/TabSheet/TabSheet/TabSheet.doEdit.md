# TabSheet.doEdit


## Синтаксис


doEdit(newCell: PP.Ui.[TabSheetCell](../TabSheetCell/TabSheetCell.htm), oldCell: PP.Ui.[TabSheetCell](../TabSheetCell/TabSheetCell.htm));


## Параметры


*newCell.* Отредактированная ячейка таблицы;


*oldCell.* Исходная ячейка таблицы.


## Описание


Метод doEdit** генерирует событие таблицы [Edited](TabSheet.Edited.htm).
