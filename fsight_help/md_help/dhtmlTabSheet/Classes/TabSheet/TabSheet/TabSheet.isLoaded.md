# TabSheet.isLoaded


## Синтаксис


isLoaded();


## Описание


Метод isLoaded** определяет, была ли загружена таблица.


## Комментарии


Метод возвращает значение true, если контейнер с таблицей уже загружен, иначе - значение false.
