# TabSheet.getIsAsyncScrolling


## Синтаксис


getIsAsyncScrolling();


## Описание


Метод getIsAsyncScrolling возвращает признак асинхронной работы компонента.


## Комментарии


Метод возвращает значение типа Boolean.


Если метод возвращает значение true, то компонент работает в асинхронном режиме, иначе - в синхронном.
