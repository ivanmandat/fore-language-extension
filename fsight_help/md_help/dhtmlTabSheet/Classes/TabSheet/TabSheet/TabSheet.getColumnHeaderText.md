# TabSheet.getColumnHeaderText


## Синтаксис


getColumnHeaderText(colIndex: Number);


## Параметры


*colIndex.* Индекс столбца таблицы.


## Описание


Метод getColumnHeaderText** возвращает заголовок столбца таблицы.


## Комментарии


Метод возвращает значение типа String.
