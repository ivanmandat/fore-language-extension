# TabSheet.ScrollChanged


## Синтаксис


ScrollChanged: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы:


-
RowIndex - текущая первая видимая строка;


-
ColIndex - текущий первый видимый столбец;


-
VScrollPos - позиция ползунка полосы прокрутки по вертикали;


-
HScrollPos - позиция ползунка полосы прокрутки по горизонтали.


## Описание


Событие ScrollChanged наступает при прокрутке таблицы.
