# TabSheet.parseCellId


## Синтаксис


parseCellId(cellId: String);


## Параметры


*cellId.* Идентификатор ячейки.


## Описание


Метод parseCellId** возвращает координаты ячейки по её идентификатору.


## Комментарии


Метод возвращает значение типа PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm).
