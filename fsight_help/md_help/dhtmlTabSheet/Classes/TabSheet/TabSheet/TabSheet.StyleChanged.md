# TabSheet.StyleChanged


## Синтаксис


StyleChanged: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: newValue - объект типа PP.Ui.[TabSheetCellStyle](../TabSheetCellStyle/TabSheetCellStyle.htm), содержащий информацию о новом стиле ячеек таблицы, oldValue - массив JSON-объектов, содержащих информацию о старых стилях.


## Описание


Событие StyleChanged** генерируется после изменения стиля таблицы.
