# TabSheet.IsTabSheetExpandExist


## Синтаксис


IsTabSheetExpandExist();


## Описание


Метод IsTabSheetExpandExist возвращает признак наличия группировок в таблице.


## Комментарии


Метод возвращает значение типа Boolean.


Если в таблице присутствуют группировки, то метод возвращает значение true, иначе - false.
