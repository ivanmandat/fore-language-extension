# TabSheet.setIsUserEditableRanges


## Синтаксис


setIsUserEditableRanges(ranges: PP.Ui.[TabSheetRange](../TabSheetRange/TabSheetRange.htm));


## Параметры


*ranges.* Редактируемый диапазон ячеек.


## Описание


Метод setIsUserEditableRanges** разрешает редактирование диапазона ячеек в режиме PP.Ui.TabSheetEditMode.UserEnabledCell.
