# TabSheet.getHScrollBarContainer


## Синтаксис


getHScrollBarContainer();


## Описание


Метод getHScrollBarContainer** возвращает DOM-элемент контейнера горизонтальной полосы прокрутки.


## Комментарии


Метод возвращает значение типа HTMLDivElement.
