# TabSheet.getRangeFromString


## Синтаксис


getRangeFromString(str: String);


## Параметры


str. Строковое представление диапазона ячеек.


## Описание


Метод getRangeFromString возвращает диапазон ячеек по строковому представлению.


## Комментарии


Метод возвращает значение типа [PP.Ui.TabSheetRange](../TabSheetRange/TabSheetRange.htm).
