# TabSheet.setSelectionControlPosition


## Синтаксис


setSelectionControlPosition(value: [PP.Ui.TSSelControlPos](dhtmlUi.chm::/Enums/TSSelControlPos.htm));


## Параметры


value. Позиция элемента.


## Описание


Метод setSelectionControlPosition
 устанавливает позицию элемента управления, который будет отображаться
 при выборе ячейки таблицы.
