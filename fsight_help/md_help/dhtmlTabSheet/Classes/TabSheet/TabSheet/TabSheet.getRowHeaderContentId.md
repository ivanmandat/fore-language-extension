# TabSheet.getRowHeaderContentId


## Синтаксис


getRowHeaderContentId(rowIndex: Number);


## Параметры


*rowIndex.* Индекс строки таблицы.


## Описание


Метод getRowHeaderContentId** возвращает идентификатор содержимого заголовка строки таблицы.


## Комментарии


Метод возвращает значение типа String.
