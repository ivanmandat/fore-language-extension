# TabSheet.setCellControls


## Синтаксис


setCellControls(cellControls: Array);


## Параметры


*cellControls.* Массив объектов с элементами управления (свойство Control со значением типа PP.Ui.[Control](dhtmlUi.chm::/Classes/control/control.htm)) и координатами ячеек, в которые они будут добавлены (свойство Coord со значением типа PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm)).


## Описание


Метод setCellControls** добавляет элементы управления в ячейки с указанными координатами.


## Комментарии


Метод возвращает значение true.
