# TabSheet.removeCellControl


## Синтаксис


removeCellControl(coord: PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm));


## Параметры


*coord.* Координата ячейки, в которой расположен удаляемый элемент управления.


## Описание


Метод removeCellControl** удаляет элемент управления по координатам ячейки, в которой он расположен.


## Комментарии


Метод возвращает значение true в случае успешного удаления элемента управления и false - в противном случае.
