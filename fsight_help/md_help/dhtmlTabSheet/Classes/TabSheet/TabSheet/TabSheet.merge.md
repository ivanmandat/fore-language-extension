# TabSheet.merge


## Синтаксис


merge(range: PP.Ui.[TabSheetRange](../TabSheetRange/TabSheetRange.htm));


## Параметры


*range.* Диапазон, ячейки которого необходимо объединить.


## Описание


Метод merge** объединяет ячейки таблицы в указанном диапазоне.
