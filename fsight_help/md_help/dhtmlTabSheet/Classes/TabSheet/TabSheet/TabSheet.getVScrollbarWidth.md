# TabSheet.getVScrollbarWidth


## Синтаксис


getVScrollbarWidth();


## Описание


Метод getVScrollbarWidth** возвращает ширину вертикальной полосы прокрутки в пикселях.


## Комментарии


Метод возвращает значение типа Number.
