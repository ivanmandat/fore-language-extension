# TabSheet.getRange


## Синтаксис


getRange(colIndex: Number, rowIndex1: Number, colIndex2: Number, rowIndex2: Number);


## Параметры


*colIndex1.* Индекс столбца верхней левой ячейки диапазона;


*rowIndex1.* Индекс строки верхней левой ячейки диапазона;


*colIndex2.* Индекс столбца нижней правой ячейки диапазона;


*rowIndex2.* Индекс строки нижней правой ячейки диапазона.


## Описание


Метод getRange** возвращает диапазон ячеек.


## Комментарии


Метод возвращает объект типа PP.Ui.[TabSheetRange](../TabSheetRange/TabSheetRange.htm).
