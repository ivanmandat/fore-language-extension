# TabSheet.getModel


## Синтаксис


getModel();


## Описание


Метод getModel** возвращает модель данных таблицы.


## Комментарии


Метод возвращает объект типа [PP.Ui.TabSheetModel](../TabSheetModel/TabSheetModel.htm).
