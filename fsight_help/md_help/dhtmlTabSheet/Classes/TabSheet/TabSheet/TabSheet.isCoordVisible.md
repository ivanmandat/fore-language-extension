# TabSheet.isCoordVisible


## Синтаксис


isCoordVisible(coord: PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm));


## Параметры


*coord.* Координата таблицы.


## Описание


Метод isCoordVisible** определяет видимость ячейки с указанной координатой в таблице.


## Комментарии


Метод возвращает значение true, если ячейка с указанной координатой отображена, иначе - значение false.
