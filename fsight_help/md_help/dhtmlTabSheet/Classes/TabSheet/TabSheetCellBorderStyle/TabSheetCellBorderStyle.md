# TabSheetCellBorderStyle

TabSheetCellBorderStyle


## Описание


Класс TabSheetCellBorderStyle
 представляет стили границы ячеек таблицы.


## Комментарии


Получить стили границ ячейки можно вызвав метод getBorders()
 для объекта класса [PP.Ui.TabSheetCellStyle](../TabSheetCellStyle/TabSheetCellStyle.htm).


## Синтаксис


PP.initClass(PP.Ui.TabSheetCellBorderStyle, PP.Object,
 "TabSheetCellBorderStyle");
