# TabSheetCellBorderStyle.hasIndex


## Синтаксис


hasIndex(index: PP.Ui.[TabSheetCellBorderIndex](../../../Enums/TabSheetCellBorderIndex.htm));


## Параметры


*index.* Индекс границы ячейки таблицы. Данный индекс можно получить с помощью свойства [TabSheetCellBorderStyle.Index](TabSheetCellBorderStyle.Index.htm).


## Описание


Метод hasIndex** возвращает флаг наличия указанного индекса границы ячейки таблицы.


## Комментарии


Метод возвращает значение true, если указанный индекс границы ячейки таблицы существует, в противном случае - false.
