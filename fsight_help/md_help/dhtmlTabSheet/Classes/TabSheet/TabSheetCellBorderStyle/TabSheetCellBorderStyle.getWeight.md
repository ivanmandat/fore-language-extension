# TabSheetCellBorderStyle.getWidth


## Синтаксис


getWidth(recIndex: PP.Ui.[TabSheetCellBorderIndex](../../../Enums/TabSheetCellBorderIndex.htm));


## Параметры


*recIndex.* Индекс границы ячейки таблицы.
 Данный индекс можно получить с помощью свойства [TabSheetCellBorderStyle.Index](TabSheetCellBorderStyle.Index.htm).


## Описание


Метод getWidth возвращает толщину
 границы ячейки таблицы.


## Комментарии


Метод возвращает значение типа Number.
