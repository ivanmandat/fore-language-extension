# TabSheetCellBorderStyle.RawWidth


## Синтаксис


RawWeight: PP.Ui.[TabSheetCellBorderWeight](../../../Enums/TabSheetCellBorderWeight.htm)


## Описание


Свойство RawWidth** определяет
 толщину границы ячеек таблицы.


## Комментарии


Значение свойства возвращается с помощью метода getRawWidth.
 Из JSON значение задается используя подстановку "@W" в объекте
 Data (см. [конструктор
 TabSheetCellBorderStyle](Constructor_TabSheetCellBorderStyle.htm)).


По умолчанию значение свойства равно [PP.Ui.TabSheetCellBorderWidth](../../../Enums/TabSheetCellBorderWeight.htm).Thin,
 что соответствует толщине границы, равной 2.
