# TabSheetCellBorderStyle.Style


## Синтаксис


Style: PP.Ui.[TabSheetCellBorderLineStyle](../../../Enums/TabSheetCellBorderLineStyle.htm)


## Описание


Свойство Style** определяет
 стиль линии границы ячеек таблицы.


## Комментарии


Значение свойства возвращается с помощью метода getStyle.
 Из JSON значение задается используя подстановку "@S" в объекте
 Data (см. [конструктор
 TabSheetCellBorderStyle](Constructor_TabSheetCellBorderStyle.htm)).


По умолчанию значение свойства равно [PP.Ui.TabSheetCellBorderLineStyle](../../../Enums/TabSheetCellBorderLineStyle.htm).Continuous,
 что соответствует сплошной линии.
