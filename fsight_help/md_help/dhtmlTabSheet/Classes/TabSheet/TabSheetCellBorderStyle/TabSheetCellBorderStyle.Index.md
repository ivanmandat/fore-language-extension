## Синтаксис

## Описание


Свойство Index** определяет
 индекс границы ячеек таблицы.


## Комментарии


Значение свойства возвращается с помощью метода getIndex.
 Из JSON значение задается используя подстановку "@I" в объекте
 Data (см. [конструктор
 TabSheetCellBorderStyle](Constructor_TabSheetCellBorderStyle.htm)).
