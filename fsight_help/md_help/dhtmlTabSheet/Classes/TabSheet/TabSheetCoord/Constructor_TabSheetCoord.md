# Конструктор TabSheetCoord


## Синтаксис


TabSheetCoord(row: Number, col: Number);


## Параметры


*row.* Индекс строки;


*col.* Индекс столбца таблицы.


## Описание


Конструктор TabSheetCoord создает экземпляр класса TabSheetCoord.
