# TabSheetCoord.isLessThan


## Синтаксис


isLessThan(coord: PP.Ui.[TabSheetCoord](TabSheetCoord.htm));


## Параметры


*coord. Координаты для сравнения.*


## Описание


Метод isLessThan** определяет, являются ли текущие координаты меньше, чем указанные.


## Комментарии


Метод возвращает значение true, если текущие координаты меньше указанных, иначе - значение false.
