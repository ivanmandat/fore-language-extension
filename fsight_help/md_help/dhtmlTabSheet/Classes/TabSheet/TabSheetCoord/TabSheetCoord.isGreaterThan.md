# TabSheetCoord.isGreaterThan


## Синтаксис


isGreaterThan(coord: PP.Ui.[TabSheetCoord](TabSheetCoord.htm));


## Параметры


*coord. Координаты для сравнения.*


## Описание


Метод isGreaterThan** определяет, являются ли текущие координаты больше, чем указанные.


## Комментарии


Метод возвращает значение true, если текущие координаты больше указанных, иначе - значение false.
