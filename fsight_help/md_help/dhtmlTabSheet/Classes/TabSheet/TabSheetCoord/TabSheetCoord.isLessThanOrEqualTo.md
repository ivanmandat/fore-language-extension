# TabSheetCoord.isLessThanOrEqualTo


## Синтаксис


isLessThanOrEqualTo(coord: PP.Ui.[TabSheetCoord](TabSheetCoord.htm));


## Параметры


*coord. Координаты для сравнения.*


## Описание


Метод isLessThanOrEqualTo** определяет, являются ли текущие координаты меньше, чем указанные, или равны им.


## Комментарии


Метод возвращает значение true, если текущие координаты меньше указанных или равны им, иначе - значение false.
