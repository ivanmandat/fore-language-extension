# TabSheetCoord.isGreaterThanOrEqualTo


## Синтаксис


isGreaterThanOrEqualTo(coord: PP.Ui.[TabSheetCoord](TabSheetCoord.htm));


## Параметры


*coord. Координаты для сравнения.*


## Описание


Метод isGreaterThanOrEqualTo** определяет, являются ли текущие координаты больше, чем указанные, или равны им.


## Комментарии


Метод возвращает значение true, если текущие координаты больше указанных или равны им, иначе - значение false.
