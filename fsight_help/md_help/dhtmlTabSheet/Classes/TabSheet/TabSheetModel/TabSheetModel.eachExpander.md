# TabSheetModel.eachExpander


## Синтаксис


eachExpander(action: function, context);


## Параметры


*action.* Функция, которую нужно выполнить для каждой группировки ячеек таблицы;


*context.* Контекст, из которого вызван данный метод.


## Описание


Метод eachExpander** выполняет указанную функцию для каждой группировки ячеек таблицы.
