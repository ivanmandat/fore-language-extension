# TabSheetModel.getExpanders


## Синтаксис


getExpanders();


## Описание


Метод getExpanders** возвращает массив группировок ячеек таблицы.


## Комментарии


Метод возвращает массив объектов типа PP.Ui.[TabSheetExpander](../TabSheetExpander/TabSheetExpander.htm).
