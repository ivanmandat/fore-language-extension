# TabSheetModel.getExpandersJSON


## Синтаксис


getExpandersJSON();


## Описание


Метод getExpandersJSON** возвращает массив группировок ячеек таблицы в JSON-формате.


## Комментарии


Метод возвращает массив JSON-объектов со следующими полями: @H - высота группировки ячеек, @IC - признак того, то группировка свёрнута, @L - индекс её левого столбца, @T - индекс её верхней строки, @W - ширина группировки.
