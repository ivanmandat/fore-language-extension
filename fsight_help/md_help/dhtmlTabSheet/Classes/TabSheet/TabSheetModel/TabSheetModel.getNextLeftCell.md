# TabSheetModel.getNextLeftCell


## Синтаксис


getNextLeftCell(coord: PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm));


## Параметры


*coord.* Координаты ячейки таблицы.


## Описание


Метод getNextLeftCell** возвращает ячейку, расположенную на столбец левее от заданной ячейки таблицы.


## Комментарии


Метод возвращает значение типа PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm).
