# TabSheetModel.eachObject


## Синтаксис


eachObject(action: function, context);


## Параметры


*action.* Функция, которую нужно выполнить для каждого объекта в таблице;


*context.* Контекст, из которого вызван данный метод.


## Описание


Метод eachObject** выполняет указанную функцию для каждого объекта в таблице.
