# TabSheetModel.loadRanges


## Синтаксис


loadRanges(ranges);


## Параметры


*ranges.* Array of PP.Ui.TabSheetRange.
 Массив диапазонов.


## Описание


Метод loadRanges загружает для
 указанного диапазона данные с сервера.

Метод предназначен для внутреннего использования.
