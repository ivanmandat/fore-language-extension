# TabSheetModel.getNextBottomCell


## Синтаксис


getNextBottomCell(coord: PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm));


## Параметры


*coord.* Координаты ячейки таблицы.


## Описание


Метод getNextBottomCell** возвращает ячейку, расположенную на строку ниже от заданной ячейки таблицы.


## Комментарии


Метод возвращает значение типа PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm).
