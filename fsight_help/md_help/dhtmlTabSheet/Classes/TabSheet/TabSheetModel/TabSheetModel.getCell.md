# TabSheetModel.getCell


## Синтаксис


getCell(coord: PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm));


## Параметры


*coord.* Координаты возвращаемой
 ячейки.


## Описание


Метод getCell** возвращает
 ячейку таблицы по указанной координате.


## Комментарии


Метод возвращает объект типа PP.Ui.[TabSheetCell](../TabSheetCell/TabSheetCell.htm).
