# TabSheetModel.addCellSpan


## Синтаксис


addCellSpan(value: PP.Ui.[TabSheetCellSpan](../TabSheetCellSpan/TabSheetCellSpan.htm));


## Параметры


*value.* Диапазон объединённых ячеек, который нужно добавить в модель таблицы.


## Описание


Метод addCellSpan** добавляет диапазон объединённых ячеек в модель таблицы.
