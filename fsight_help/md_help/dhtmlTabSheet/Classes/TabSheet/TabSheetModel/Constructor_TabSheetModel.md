# Конструктор TabSheetModel


## Синтаксис


PP.Ui.TabSheetModel(settings);


## Параметры


*settings.* JSON-объект со значениями свойств класса.


## Описание


Конструктор TabSheetModel** создает экземпляр класса **TabSheetModel.**

Конструктор предназначен для внутреннего использования.
