# TabSheetModel.getNextTopCell


## Синтаксис


getNextTopCell(coord: PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm));


## Параметры


*coord.* Координаты ячейки таблицы.


## Описание


Метод getNextTopCell** возвращает ячейку, расположенную на строку выше от заданной ячейки таблицы.


## Комментарии


Метод возвращает значение типа PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm).
