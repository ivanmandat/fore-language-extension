# Конструктор TabSheetObjectSelection


## Синтаксис


PP.Ui.TabSheetObjectSelection(settings);


## Параметры


*settings.* JSON-объект со значениями свойств класса.


## Описание


Конструктор TabSheetObjectSelection** создает экземпляр класса **TabSheetObjectSelection.**
