# TabSheetAdjustmentMgr.autoAdjustColumnWidth


## Синтаксис


autoAdjustColumnWidth(colIndex: Number);


## Параметры


*colIndex.* Индекс колонки, ширину которой нужно подогнать.


## Описание


Метод autoAdjustColumnWidth** осуществляет автоподгонку ширины столбца таблицы.


## Комментарии


Метод возвращает значение true, если ширина указанного столбца таблицы была успешно подобрана, иначе - false.
