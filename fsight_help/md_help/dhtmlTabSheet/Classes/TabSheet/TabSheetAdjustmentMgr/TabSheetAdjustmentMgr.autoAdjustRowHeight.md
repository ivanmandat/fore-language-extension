# TabSheetAdjustmentMgr.autoAdjustRowHeight


## Синтаксис


autoAdjustRowHeight(rowIndex: Number);


## Параметры


*rowIndex.* Индекс строки, высоту которой нужно подогнать.


## Описание


Метод autoAdjustRowHeight** осуществляет автоподгонку высоты строки таблицы.


## Комментарии


Метод возвращает значение true, если высота указанной строки таблицы была успешно подобрана, иначе - false.
