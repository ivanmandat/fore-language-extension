# TabSheetObject.getCoord


## Синтаксис


getCoord();


## Описание


Метод getCoord** возвращает координаты ячейки, в пределах которой расположен левый верхний угол объекта в таблице.


## Комментарии


Метод возвращает объект типа PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm).
