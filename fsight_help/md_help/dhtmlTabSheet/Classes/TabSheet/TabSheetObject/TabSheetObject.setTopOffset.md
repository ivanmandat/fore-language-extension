# TabSheetObject.setTopOffset


## Синтаксис


setTopOffset(top: Number, isVirtual: Boolean);


## Параметры


*left.* Отступ сверху в пикселях.


*isVirtual.* Определяет, какая координата была передана: виртуальная или реальная. Если параметр равен значению true, то была передана виртуальная координата (т. е. из кеша), иначе - реальная.


## Описание


Метод setTopOffset** устанавливает отступ сверху для объекта в таблице.
