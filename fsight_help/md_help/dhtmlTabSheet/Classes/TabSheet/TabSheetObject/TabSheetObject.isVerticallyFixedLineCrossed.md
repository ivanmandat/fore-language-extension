# TabSheetObject.isVerticallyFixedLineCrossed


## Синтаксис


isVerticallyFixedLineCrossed();


## Описание


Метод isVerticallyFixedLineCrossed** определяет, пересекает ли объект в таблице вертикальную линию фиксации.


## Комментарии


Метод возвращает значение true, если объект пересекает вертикальную линию фиксации, иначе - значение false.
