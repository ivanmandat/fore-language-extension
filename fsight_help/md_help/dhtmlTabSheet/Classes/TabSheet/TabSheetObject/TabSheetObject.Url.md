# TabSheetObject.Url


## Синтаксис


Url: String


## Описание


Свойство Url** определяет источник данных объекта в таблице.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setUrl, а возвращается с помощью метода getUrl.


По умолчанию свойство содержит значение null.
