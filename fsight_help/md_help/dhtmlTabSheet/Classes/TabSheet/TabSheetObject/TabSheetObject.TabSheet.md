# TabSheetObject.TabSheet


## Синтаксис


TabSheet: PP.Ui.[TabSheet](../TabSheet/TabSheet.htm).


## Описание


Свойство TabSheet** определяет базовый компонент - [таблицу](../../../Components/TabSheet/TabSheet/TabSheet.htm).


## Комментарии


Значение свойства может быть установлено из JSON через свойство Parent и с помощью метода setTabSheet, а возвращено - с помощью метода getTabSheet.
