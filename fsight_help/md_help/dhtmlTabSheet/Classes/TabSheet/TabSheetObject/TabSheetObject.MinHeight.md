# TabSheetObject.MinHeight


## Синтаксис


MinHeight();


## Описание


Свойство MinHeight определяет
 минимальную высоту объекта.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMinHeight,
 а возвращается с помощью метода getMinHeight.


Если значение свойства Height
 меньше MinHeight, то при создании
 объекта свойство Height принимает
 значение MinHeight.
