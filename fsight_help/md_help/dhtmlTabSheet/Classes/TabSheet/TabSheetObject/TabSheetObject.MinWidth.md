# TabSheetObject.MinWidth


## Синтаксис


MinWidth();


## Описание


Свойство MinWidth определяет
 минимальную ширину объекта.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMinWidth,
 а возвращается с помощью метода getMinWidth.


Если значение свойства Width
 меньше MinWidth, то при создании
 объекта свойство Width принимает
 значение MinWidth.
