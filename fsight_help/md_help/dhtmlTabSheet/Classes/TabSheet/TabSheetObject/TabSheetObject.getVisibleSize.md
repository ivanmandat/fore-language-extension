# TabSheetObject.getVisibleSize


## Синтаксис


getVisibleSize();


## Описание


Метод getVisibleSize** определяет видимый размер объекта в таблице.


## Комментарии


Метод возвращает JSON-объект с полями height и width, определяющими видимые высоту и ширину объекта соответственно.
