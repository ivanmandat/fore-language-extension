# TabSheetObject.LeftColumn


## Синтаксис


LeftColumn: Number


## Описание


Свойство LeftColumn** определяет индекс столбца, к которому привязан верхний левый угол объекта.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setLeftColumn, а возвращается с помощью метода getLeftColumn.
