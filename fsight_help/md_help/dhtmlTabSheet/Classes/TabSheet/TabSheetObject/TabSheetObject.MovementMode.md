# TabSheetObject.MovementMode


## Синтаксис


MovementMode: PP.Ui.TabObjectMovementMode;


## Описание


Свойство MovementMode определяет
 способ перемещения объекта в таблице.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setMovementMode,
 а возвращается с помощью метода getMovementMode.


Свойство принимает значение перечисления PP.Ui.TabObjectMovementMode,
 аналогичного перечислению [TabObjectMovementMode](TabSheet.chm::/Enums/TabObjectMovementMode.htm).
