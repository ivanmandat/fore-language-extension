# TabSheetObject.ObjectType


## Синтаксис


ObjectType([PP.Ui.TabSheetObjectType](../../../Enums/TabSheetObjectType.htm));


## Описание


Свойство ObjectType определяет
 тип объекта в таблице.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setObjectType,
 а возвращается с помощью метода getObjectType.
