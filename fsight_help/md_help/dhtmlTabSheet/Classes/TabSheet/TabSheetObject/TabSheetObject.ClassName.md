# TabSheetObject.ClassName


## Синтаксис


ClassName: String


## Описание


Свойство ClassName** определяет имя класса объекта таблицы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setClassName, а возвращается с помощью метода getClassName.


По умолчанию свойство содержит значение null.
