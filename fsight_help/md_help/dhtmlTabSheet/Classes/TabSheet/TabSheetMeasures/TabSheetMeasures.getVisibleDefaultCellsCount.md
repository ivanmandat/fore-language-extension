# TabSheetMeasures.getVisibleDefaultCellsCount


## Синтаксис


getVisibleDefaultCellsCount();


## Описание


Метод getVisibleDefaultCellsCount** возвращает для видимых ячеек таблицы их количество по умолчанию.


## Комментарии


Метод возвращает значение типа Number.
