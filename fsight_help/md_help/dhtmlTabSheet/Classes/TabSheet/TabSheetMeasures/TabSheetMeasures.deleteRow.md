# TabSheetMeasures.deleteRow


## Синтаксис


deleteRow(rowIndex: Number, count: Number);


## Параметры


*rowIndex.* Индекс первой удаляемой строки таблицы;


*count.* Количество удаляемых строк таблицы.


## Описание


Метод deleteRow** удаляет строки таблицы.
