# TabSheetMeasures.VisibleGrid


## Синтаксис


VisibleGrid: Boolean


## Описание


Свойство VisibleGrid управляет видимостью сетки таблицы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setVisibleGrid, а возвращается - с помощью метода getVisibleGrid.


Если свойство равно true, то сетка таблицы будет отображена, иначе - не будет.


По умолчанию свойство равно true.
