# TabSheetMeasures.removeHiddenRow


## Синтаксис


removeHiddenRow(rowIndex: Number);


## Параметры


*rowIndex. Индекс скрытой строки, которую необходимо отобразить.*


## Описание


Метод removeHiddenRow** отображает скрытую строку таблицы по указанному индексу.
