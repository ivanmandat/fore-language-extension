# TabSheetMeasures.DefaultRowHeight


## Синтаксис


DefaultRowHeight: Number


## Описание


Свойство DefaultRowHeight устанавливает для строк таблицы высоту по умолчанию.


## Комментарии


Значение свойства можно установить из JSON и с помощью метода setDefaultRowHeight, а возвратить - с помощью метода getDefaultRowHeight.


По умолчанию высота строк таблицы равна 25 пикселям.
