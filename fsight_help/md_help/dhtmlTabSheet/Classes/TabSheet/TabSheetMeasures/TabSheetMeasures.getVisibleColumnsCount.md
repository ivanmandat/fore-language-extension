# TabSheetMeasures.getVisibleColumnsCount


## Синтаксис


getVisibleColumnsCount();


## Описание


Метод getVisibleColumnsCount возвращает** количество видимых столбцов таблицы.


## Комментарии


Метод возвращает значение типа Number.
