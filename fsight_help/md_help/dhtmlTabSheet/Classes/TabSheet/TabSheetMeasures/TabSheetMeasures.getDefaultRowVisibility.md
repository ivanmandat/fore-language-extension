# TabSheetMeasures.getDefaultRowVisibility


## Синтаксис


getDefaultRowVisibility();


## Описание


Метод getDefaultRowVisibility** возвращает признак видимости строк по умолчанию.


## Комментарии


Метод возвращает значение true, если по умолчанию строки таблицы отображаются, и false - в противном случае.
