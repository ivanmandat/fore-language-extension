# TabSheetMeasures.getLastColumnToScroll


## Синтаксис


getLastColumnToScroll();


## Описание


Метод getLastColumnToScroll** возвращает индекс последнего столбца, к которому прокручена таблица.


## Комментарии


Метод возвращает значение типа Number.
