# TabSheetMeasures.eachVisibleRowIndex


## Синтаксис


eachVisibleRowIndex(action: function, context: Object, args: Array);


## Параметры


*action. *Функция, вызываемая при обходе индексов каждой видимой строки таблицы;


*context.* Контекст, из которого вызван данный метод. Необязательный параметр, по умолчанию равен текущему контексту this;


*args.* Массив индексов строк таблицы. Необязательный параметр, по умолчанию равен пустому массиву.


## Описание


Метод eachVisibleRowIndex** выполняет указанную функцию при обходе индексов каждой видимой строки таблицы.
