# TabSheetMeasures.getCoordSize


## Синтаксис


getCoordSize(coord: PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm));


## Параметры


*coord. Координаты ячейки, ширину и высоту которой необходимо определить.*


## Описание


Метод getCoordSize** возвращает высоту строки и ширину столбца, соответствующих указанным координатам.


## Комментарии


Метод возвращает JSON-объект со следующими полями: height - высота строки, width - ширина столбца таблицы.
