# TabSheetMeasures.iterateRowHeight


## Синтаксис


iterateRowHeight(action: function, context: Object, reverse: Boolean);


## Параметры


*action.* Функция, вызываемая при каждом обходе значений высоты строк таблицы;


*context.* Контекст, из которого вызван данный метод. Необязательный параметр, по умолчанию равен текущему контексту this


*reverse.* Признак, определяющий направление обхода по значениям высоты строк таблицы. Если параметр равен значению true, то обход начинается со значения высоты последней строки, если false - то наоборот. Необязательный параметр, по умолчанию равен значению false.


## Описание


Метод iterateRowHeight** перебирает значения высоты строк таблицы.
