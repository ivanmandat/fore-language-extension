# TabSheetMeasures.VisibleRowHeaders


## Синтаксис


VisibleRowHeaders: Boolean


## Описание


Свойство VisibleRowHeaders управляет видимостью заголовков строк таблицы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setVisibleRowHeaders, а возвращается - с помощью метода getVisibleRowHeaders.


Если свойство равно true, то заголовки строк таблицы будут отображаться, иначе - не будут.


По умолчанию свойство равно true.
