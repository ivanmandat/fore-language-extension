# TabSheetMeasures.isCoordInsideVisibleRange


## Синтаксис


isCoordInsideVisibleRange(coord: PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm));


## Параметры


*coord. Координаты ячейки таблицы.*


## Описание


Метод isCoordInsideVisibleRange** определяет, находится ли указанная координата внутри видимого диапазона таблицы.


## Комментарии


Метод возвращает значение true, если указанная координата находится внутри видимого диапазона таблицы, иначе - значение false.
