# TabSheetMeasures.getRowPosByIndex


## Синтаксис


getRowPosByIndex(rowIndex: Number);


## Параметры


*rowIndex. Индекс строки в таблице, позицию которой необходимо определить.*


## Описание


Метод getRowPosByIndex** определяет позицию строки в таблице по её индексу.


## Комментарии


Метод возвращает значение типа Number.


Позиция строки соответствует её индексу без учёта прокрутки таблицы по вертикали.
