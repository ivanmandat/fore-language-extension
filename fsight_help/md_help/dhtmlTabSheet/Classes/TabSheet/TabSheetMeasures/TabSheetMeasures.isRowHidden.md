# TabSheetMeasures.isRowHidden


## Синтаксис


isRowHidden(rowIndex: Number);


## Параметры


*rowIndex. Индекс строки таблицы.*


## Описание


Метод isRowHidden** определяет, является ли указанная строка таблицы скрытой.


## Комментарии


Метод возвращает значение true, если указанная строка является скрытой, иначе - значение false.
