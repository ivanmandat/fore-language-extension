# TabSheetMeasures.getNextTopCoord


## Синтаксис


getNextTopCoord(coord: PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm));


## Параметры


*coord. Координаты ячейки таблицы.*


## Описание


Метод getNextTopCoord** возвращает координаты ячейки, расположенной на одну строку выше ячейки с указанными координатами.


## Комментарии


Метод возвращает объект типа PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm).
