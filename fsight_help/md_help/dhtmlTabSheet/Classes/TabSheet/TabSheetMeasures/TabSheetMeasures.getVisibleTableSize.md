# TabSheetMeasures.getVisibleTableSize


## Синтаксис


getVisibleTableSize();


## Описание


Метод getVisibleTableSize** возвращает количество видимых строк и столбцов таблицы, а также их общую высоту и ширину.


## Комментарии


Метод возвращает JSON-объект со следующими полями: row - количество видимых строк в таблице , column - количество видимых столбцов, rowHeight - общая высота всех видимых строк, colWidth - общая ширина всех видимых столбцов.
