# TabSheetMeasures.getLastColFullyVisible


## Синтаксис


getLastColFullyVisible();


## Описание


Метод getLastColFullyVisible** возвращает индекс последнего полностью видимого столбца в таблице.


## Комментарии


Метод возвращает значение типа Number.
