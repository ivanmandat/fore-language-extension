# TabSheetMeasures.getNextRightCoord


## Синтаксис


getNextRightCoord(coord: PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm));


## Параметры


*coord.* *Координаты ячейки таблицы.*


## Описание


Метод getNextRightCoord** возвращает координаты ячейки, расположенной на один столбец правее ячейки с указанными координатами.


## Комментарии


Метод возвращает объект типа PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm).
