# TabSheetMeasures.getLastNonHiddenColIndex


## Синтаксис


getLastNonHiddenColIndex();


## Описание


Метод getLastNonHiddenColIndex** возвращает индекс последнего нескрытого столбца в таблице.


## Комментарии


Метод возвращает значение типа Number.
