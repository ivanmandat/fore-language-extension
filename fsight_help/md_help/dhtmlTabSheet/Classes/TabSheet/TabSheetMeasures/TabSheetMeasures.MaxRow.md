# TabSheetMeasures.MaxRow


## Синтаксис


MaxRow: Number


## Описание


Свойство MaxRow** устанавливает значение максимального количества строк в таблице.


## Комментарии


Значение свойства можно установить из JSON и с помощью метода setMaxRow, а возвратить - с помощью метода getMaxRow.


По умолчанию свойство содержит значение null.
