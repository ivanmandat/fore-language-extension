# TabSheetMeasures.setFixedRow


## Синтаксис


setFixedRow(value: Number);


## Параметры


*value.* Индекс крайней фиксированной строки таблицы.


## Описание


Метод setFixedRow** фиксирует строки таблицы вплоть до строки с указанным индексом.
