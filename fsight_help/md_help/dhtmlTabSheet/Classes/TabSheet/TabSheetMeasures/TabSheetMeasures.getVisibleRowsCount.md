# TabSheetMeasures.getVisibleRowsCount


## Синтаксис


getVisibleRowsCount();


## Описание


Метод getVisibleRowsCount** **возвращает** количество видимых строк таблицы.


## Комментарии


Метод возвращает значение типа Number.
