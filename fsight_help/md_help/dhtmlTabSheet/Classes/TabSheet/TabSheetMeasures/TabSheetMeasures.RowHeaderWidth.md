# TabSheetMeasures.RowHeaderWidth


## Синтаксис


RowHeaderWidth: Number


## Описание


Свойство RowHeaderWidth** устанавливает ширину заголовков строк таблицы.


## Комментарии


Значение свойства можно установить из JSON и с помощью метода set**RowHeaderWidth**, а возвратить - с помощью метода getRowHeaderWidth.


По умолчанию ширина заголовков строк таблицы равна 33.
