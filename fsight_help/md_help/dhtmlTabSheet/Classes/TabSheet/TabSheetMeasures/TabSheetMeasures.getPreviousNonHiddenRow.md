# TabSheetMeasures.getPreviousNonHiddenRow


## Синтаксис


getPreviousNonHiddenRow(rowIndex: Number);


## Параметры


*rowIndex. Индекс строки таблицы.*


## Описание


Метод getPreviousNonHiddenRow** **возвращает** индекс нескрытой строки, расположенной перед указанной строкой таблицы.


## Комментарии


Метод возвращает значение типа Number.
