# Конструктор TabSheetMeasures


## Синтаксис


PP.Ui.TabSheetMeasures(settings);


## Параметры


*settings.* JSON-объект со значениями свойств класса.


## Описание


Конструктор TabSheetMeasures** создает экземпляр класса **TabSheetMeasures.**


## Пример


Конструктор предназначен для внутреннего использования.
