# TabSheetMeasures.removeHiddenColumn


## Синтаксис


removeHiddenColumn(colIndex: Number);


## Параметры


*colIndex. Индекс скрытого столбца, который необходимо отобразить.*


## Описание


Метод removeHiddenColumn** отображает скрытый столбец таблицы по указанному индексу.
