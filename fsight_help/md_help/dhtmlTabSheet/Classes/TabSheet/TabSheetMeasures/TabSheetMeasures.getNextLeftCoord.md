# TabSheetMeasures.getNextLeftCoord


## Синтаксис


getNextLeftCoord(coord: PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm));


## Параметры


*coord. Координаты ячейки таблицы.*


## Описание


Метод getNextLeftCoord** возвращает координаты ячейки, расположенной на один столбец левее ячейки с указанными координатами.


## Комментарии


Метод возвращает объект типа PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm).
