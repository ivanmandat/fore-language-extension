# TabSheetMeasures.setFixedColumn


## Синтаксис


setFixedColumn(value: Number);


## Параметры


*value. Индекс крайнего фиксированного столбца таблицы.*


## Описание


Метод setFixedColumn** фиксирует столбцы таблицы вплоть до столбца с указанным индексом.
