# TabSheetMeasures.isColumnHidden


## Синтаксис


isColumnHidden(colIndex: Number);


## Параметры


*colIndex. Индекс столбца таблицы.*


## Описание


Метод isColumnHidden** определяет, является ли указанный столбец таблицы скрытым.


## Комментарии


Метод возвращает значение true, если указанный столбец является скрытым, иначе - значение false.
