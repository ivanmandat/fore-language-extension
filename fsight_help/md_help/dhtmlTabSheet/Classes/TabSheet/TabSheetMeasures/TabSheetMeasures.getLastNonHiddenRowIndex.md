# TabSheetMeasures.getLastNonHiddenRowIndex


## Синтаксис


getLastNonHiddenRowIndex();


## Описание


Метод getLastNonHiddenRowIndex** возвращает индекс последней нескрытой строки в таблице.


## Комментарии


Метод возвращает значение типа Number.
