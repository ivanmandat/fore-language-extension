# TabSheetMeasures.setRowsHeight


## Синтаксис


setRowsHeight(value: Array);


## Параметры


*value. Массив значений высоты строк таблицы.*


## Описание


Метод setRowsHeight** устанавливает новые значения высоты строк таблицы.
