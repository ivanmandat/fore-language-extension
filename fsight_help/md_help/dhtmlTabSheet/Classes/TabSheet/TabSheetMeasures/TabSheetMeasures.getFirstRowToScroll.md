# TabSheetMeasures.getFirstRowToScroll


## Синтаксис


getFirstRowToScroll();


## Описание


Метод getFirstRowToScroll** возвращает индекс первой строки, начиная с которой прокручена таблица.


## Комментарии


Метод возвращает значение типа Number.
