# TabSheetMeasures.isLastVisibleRowLastNonHidden


## Синтаксис


isLastVisibleRowLastNonHidden();


## Описание


Метод isLastVisibleRowLastNonHidden** определяет, является ли последняя строка видимого диапазона последней нескрытой строкой в таблице.


## Комментарии


Метод возвращает значение true, если последняя строка видимого диапазона таблицы является последней нескрытой строкой, иначе - значение false.
