# TabSheetMeasures.RowResizeEnabled


## Синтаксис


RowResizeEnabled: Boolean;


## Описание


Свойство RowResizeEnabled**
 определяет возможность изменения высоты строки при помощи мыши.


## Комментарии


Значение свойства устанавливается с помощью метода setRow**ResizeEnabled**
 и возвращается при помощи метода getRow**ResizeEnabled**.


При вызове метода setRow**ResizeEnabled**
 указываются два параметра: colIndex - индекс строки таблицы и value (Boolean)
 - возможность изменения высоты строки.


При вызове метода getRow**ResizeEnabled**
 указывается один параметр: colIndex - индекс строки таблицы, возможность
 изменения высоты которой необходимо возвратить.
