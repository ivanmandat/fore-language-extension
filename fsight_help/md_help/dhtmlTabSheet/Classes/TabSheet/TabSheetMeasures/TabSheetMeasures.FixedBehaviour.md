# TabSheetMeasures.FixedBehaviour


## Синтаксис


FixedBehaviour: PP.Ui.[TabFixedBehaviour](../../../Enums/TabFixedBehaviour.htm)


## Описание


Свойство FixedBehaviour указывает, будут ли фиксированные строки и столбцы выделяться полностью при нажатии на них.


## Комментарии


Значение свойства можно установить из JSON и с помощью метода setFixedBehaviour, а возвратить - с помощью метода getFixedBehaviour.


По умолчанию значение свойства равно 0.
