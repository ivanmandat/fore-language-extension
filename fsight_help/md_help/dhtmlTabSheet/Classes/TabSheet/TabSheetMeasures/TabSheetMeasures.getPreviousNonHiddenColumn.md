# TabSheetMeasures.getPreviousNonHiddenColumn


## Синтаксис


getPreviousNonHiddenColumn(colIndex: Number);


## Параметры


*colIndex.* *Индекс столбца таблицы.*


## Описание


Метод getPreviousNonHiddenColumn** **возвращает** индекс нескрытого столбца таблицы, расположенного перед указанным столбцом.


## Комментарии


Метод возвращает значение типа Number.
