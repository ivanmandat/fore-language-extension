# TabSheetMeasures.isLastVisibleColumnLastNonHidden


## Синтаксис


isLastVisibleColumnLastNonHidden();


## Описание


Метод isLastVisibleColumnLastNonHidden** определяет, является ли последний столбец видимого диапазона последним нескрытым столбцом в таблице.


## Комментарии


Метод возвращает значение true, если последний столбец видимого диапазона таблицы является последним нескрытым столбцом, иначе - значение false.
