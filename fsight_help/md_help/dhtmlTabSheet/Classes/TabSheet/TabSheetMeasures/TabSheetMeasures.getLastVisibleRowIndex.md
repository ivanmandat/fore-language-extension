# TabSheetMeasures.getLastVisibleRowIndex


## Синтаксис


getLastVisibleRowIndex();


## Описание


Метод getLastVisibleRowIndex** возвращает индекс последней видимой строки в таблице.


## Комментарии


Метод возвращает значение типа Number.
