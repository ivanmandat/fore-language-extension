# TabSheetMeasures.getNextNonHiddenColumn


## Синтаксис


getNextNonHiddenColumn(colIndex: Number);


## Параметры


*colIndex. Индекс столбца таблицы.*


## Описание


Метод getNextNonHiddenColumn возвращает** индекс нескрытого столбца таблицы, следующего за указанным столбцом.


## Комментарии


Метод возвращает значение типа Number.
