# TabSheetMeasures.RowHeight


## Синтаксис


RowHeight: Number


## Описание


Свойство RowHeight** определяет высоту указанной строки таблицы.


## Комментарии


Значение свойства устанавливается с помощью метода setRowHeight и возвращается при помощи метода getRowHeight. Из JSON значение свойства задать нельзя.


При вызове метода setRowHeight указываются два параметра: value - новое значение высоты строки и rowIndex - индекс данной строки.


При вызове метода getRowHeight указываются два параметра: rowIndex - индекс строки таблицы, значение высоты которой необходимо возвратить, isVirtual - признак, показывающий, обращаемся ли мы к виртуальной строке таблицы или к реальной.
