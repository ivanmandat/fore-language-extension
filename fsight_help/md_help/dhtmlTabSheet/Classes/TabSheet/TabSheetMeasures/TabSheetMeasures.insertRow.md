# TabSheetMeasures.insertRow


## Синтаксис


insertRow(afterRowIndex: Number, size: Number, count: Number);


## Параметры


*afterRowIndex.* Индекс строка, после которой будет осуществлена вставка;


*size.* Высота вставляемых строк;


*count.* Количество строк для вставки.


## Описание


Метод insertRow** вставляет столбцы в таблицу.
