# TabSheetMeasures.getNextNonHiddenRow


## Синтаксис


getNextNonHiddenRow(rowIndex: Number);


## Параметры


*rowIndex.* *Индекс строки таблицы.*


## Описание


Метод getNextNonHiddenRow** **возвращает** индекс нескрытой строки, следующей за указанной строкой таблицы.


## Комментарии


Метод возвращает значение типа Number.
