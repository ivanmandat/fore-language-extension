# TabSheetMeasures.getVisibleRowIndexes


## Синтаксис


getVisibleRowIndexes();


## Описание


Метод getVisibleRowIndexes** возвращает массив индексов видимых строк таблицы.


## Комментарии


Метод возвращает массив значений типа Number.
