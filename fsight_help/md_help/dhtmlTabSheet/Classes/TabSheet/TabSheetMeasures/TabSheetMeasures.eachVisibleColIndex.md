# TabSheetMeasures.eachVisibleColIndex


## Синтаксис


eachVisibleColIndex(action: function, context: Object, args: Array);


## Параметры


*action.* Функция, вызываемая при обходе индексов каждого видимого столбца таблицы;


*context.* Контекст, из которого вызван данный метод. Необязательный параметр, по умолчанию равен текущему контексту this;


*args.* Массив индексов столбцов таблицы. Необязательный параметр, по умолчанию равен пустому массиву.


## Описание


Метод eachVisibleColIndex** выполняет указанную функцию при обходе индексов каждого видимого столбца таблицы.
