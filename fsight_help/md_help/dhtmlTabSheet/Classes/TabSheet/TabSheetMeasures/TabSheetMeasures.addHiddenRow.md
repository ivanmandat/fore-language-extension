# TabSheetMeasures.addHiddenRow


## Синтаксис


addHiddenRow(rowIndex: Number);


## Параметры


*rowIndex. Индекс скрываемой строки.*


## Описание


Метод addHiddenRow** скрывает строку таблицы по указанному индексу.
