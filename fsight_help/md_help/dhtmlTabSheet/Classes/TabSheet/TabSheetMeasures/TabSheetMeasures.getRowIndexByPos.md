# TabSheetMeasures.getRowIndexByPos


## Синтаксис


getRowIndexByPos(rowPosition: Number);


## Параметры


*rowPosition.* *Позиция строки в таблице, индекс которой необходимо определить.*


## Описание


Метод getRowIndexByPos** определяет индекс строки по её позиции в таблице.


## Комментарии


Метод возвращает значение типа Number.


Позиция строки соответствует её индексу без учёта прокрутки таблицы по вертикали.
