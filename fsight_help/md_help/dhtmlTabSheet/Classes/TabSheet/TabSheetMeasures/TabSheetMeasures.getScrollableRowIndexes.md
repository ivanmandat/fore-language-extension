# TabSheetMeasures.getScrollableRowIndexes


## Синтаксис


getScrollableRowIndexes();


## Описание


Метод getScrollableRowIndexes** возвращает индексы строк таблицы, обладающих возможностью прокрутки.


## Комментарии


Метод возвращает массив значений типа Number.
