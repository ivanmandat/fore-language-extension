# TabSheetMeasures.getLastRowToScroll


## Синтаксис


getLastRowToScroll();


## Описание


Метод getLastRowToScroll** возвращает индекс последней строки, к которой прокручена таблица.


## Комментарии


Метод возвращает значение типа Number.
