# TabSheetMeasures


Пространство имен: PP.Ui;


## Иерархия наследования


[Object](dhtmlCommon.chm::/Classes/object/object.htm)


TabSheetSerializableObject


TabSheetMeasures


## Описание


Класс TabSheetMeasures представляет измерения таблицы.


## Синтаксис


PP.initClass(PP.Ui.TabSheetMeasures, PP.Ui.TabSheetSerializableObject, "PP.Ui.TabSheetMeasures");


## Комментарии


Класс предназначен для хранения данных о состоянии таблицы и вычисления вспомогательных значений.
