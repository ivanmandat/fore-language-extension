# TabSheetSelection.getStartCoord


## Синтаксис


getStartCoord();


## Описание


Метод getStartCoord** возвращает начальные координаты выделения в таблице.


## Комментарии


Метод возвращает объект типа PP.Ui.[TabSheetCoord](../TabSheetCoord/TabSheetCoord.htm).
