# Конструктор TabSheetSelection


## Синтаксис


PP.Ui.TabSheetSelection(settings);


## Параметры


*settings.* JSON-объект со значениями свойств класса.


## Описание


Конструктор TabSheetSelection** создает экземпляр класса **TabSheetSelection.**


## Пример


Конструктор предназначен для внутреннего использования.
