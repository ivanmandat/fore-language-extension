# TabSheetSelection.removeArea


## Синтаксис


removeArea(area: Object);


## Параметры


*area. Область выделения, которую
 требуется убрать. Представляет собой* JSON-объект со следующими
 полями: Nodes - массив вершин html-документа, Range - диапазон ячеек в
 выделенной области таблицы, Type - тип выделенной области (равен элементу
 перечисления [PP.Ui.TabSheetSelection.AreaType](../../../Enums/TabSheetSelectionAreaType.htm)).


## Описание


Метод removeArea** убирает
 выделенные области в таблице.
