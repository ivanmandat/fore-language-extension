# Компоненты для работы с таблицей


## Описание


На данной странице приведены компоненты для работы с таблицей.


## Компоненты


		 Имя компонента
		 Краткое описание


		 ![](../class_image.gif)
		 [Sparkline](Sparkline/Sparkline.htm)
		 Компонент Sparkline
		 представляет собой небольшую диаграмму, которая располагается
		 в отдельной ячейке таблицы и предназначена для отображения данных
		 строк или столбцов таблицы.


		 ![](../class_image.gif)
		 [TabSheet](TabSheet/TabSheet/TabSheet.htm)
		 Компонент TabSheet
		 реализует таблицу.
