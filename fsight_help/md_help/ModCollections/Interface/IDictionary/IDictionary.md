## Комментарии


Словарь представляет собой таблицу, содержащую значение элемента и уникальный
 ключ для каждого элемента.


Данный интерфейс является базовым для следующих интерфейсов:


	- [IHashtable](../IHashtable/IHashtable.htm);


	- [ISortedList](../ISortedList/ISortedList.htm);
