# IStringList.AsString


## Синтаксис


AsString: String;


## Описание


Свойство AsString определяет
 массив как текстовую строку, используя разделитель - «{Перевод строки}».


## Комментарии


При задании свойства AsString
 будет сформирован массив путем разбиения указываемой строки с помощью
 разделителя «{Перевод строки}». Имевшиеся до этого в массиве элементы
 при этом будут удалены.


При получении значения свойства AsString
 будет сформирована строка путём объединения элементов массива. Между элементами
 будет вставлен разделитель «{Возврат каретки} + {Перевод строки}».


## Пример


	Sub UserProc;

	Var

	    StrL: IStringList;

	Begin

	    StrL := New StringList.Create;

	    StrL.AsString := "Один" + #13 + #10 + "Два" + #13 + #10 + "Три" + #13 + #10 + "Четыре";

	End Sub UserProc;


После выполнения примера в массив StrL будут занесены 4 элемента из
 текстовой строки.
