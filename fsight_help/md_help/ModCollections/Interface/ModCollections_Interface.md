# Интерфейсы сборки Collections


		 Интерфейс
		 Краткое описание


		 ![](../Interface_Image.gif)
		 [IArrayList](IArrayList/IArrayList.htm)
		 Интерфейс IArrayList
		 содержит свойства и методы для работы с динамическим массивом.


		 ![](../Interface_Image.gif)
		 [IBitArray](IBitArray/IBitArray.htm)
		 Интерфейс IBitArray
		 содержит свойства и методы для работы с динамическим битовым массивом.


		 ![](../Interface_Image.gif)
		 [ICollection](ICollection/ICollection.htm)
		 ICollection -
		 абстрактный интерфейс, содержит базовые свойства и методы всех
		 коллекций.


		 ![](../Interface_Image.gif)
		 [IComparer](IComparer/IComparer.htm)
		 IComparer - абстрактный
		 интерфейс, содержащий методы для сравнения двух объектов.


		 ![](../Interface_Image.gif)
		 [IComparerClass](IComparerClass/IComparerClass.htm)
		 Интерфейс IComparerClass
		 содержит статические свойства класса [Comparer](../Class/Comparer/Comparer.htm),
		 реализующие объекты для сравнения элементов различного типа.


		 ![](../Interface_Image.gif)
		 [IDictionary](IDictionary/IDictionary.htm)
		 IDictionary - абстрактный
		 интерфейс словаря.


		 ![](../Interface_Image.gif)
		 [IEnumerable](IEnumerable/IEnumerable.htm)
		 IEnumerable - абстрактный
		 интерфейс коллекций, поддерживающих перебор элементов.


		 ![](../Interface_Image.gif)
		 [IFreeTextComparer](IFreeTextComparer/IFreeTextComparer.htm)
		 Интерфейс IFreeTextComparer
		 предназначен для свободного сравнения текста по заданным параметрам.


		 ![](../Interface_Image.gif)
		 [IHashtable](IHashtable/IHashtable.htm)
		 Интерфейс IHashtable
		 представляет набор взаимосвязанных ключей и значений, основанных
		 на хеш-коде ключа.


		 ![](../Interface_Image.gif)
		 [IList](IList/IList.htm)
		 IList - абстрактный
		 интерфейс списка.


		 ![](../Interface_Image.gif)
		 [IQueue](IQueue/IQueue.htm)
		 Интерфейс IQueue содержит
		 свойства и методы для работы с очередью элементов.


		 ![](../Interface_Image.gif)
		 [ISortedList](ISortedList/ISortedList.htm)
		 Интерфейс ISortedList
		 содержит свойства и методы для работы с коллекцией пар «Ключ-Значение»
		 отсортированной по значениям ключа.


		 ![](../Interface_Image.gif)
		 [IStack](IStack/IStack.htm)
		 Интерфейс IStack содержит
		 свойства и методы для работы со стеком элементов.


		 ![](../Interface_Image.gif)
		 [IStringList](IStringList/IStringList.htm)
		 Интерфейс IStringList
		 содержит свойства и методы для работы с динамическим массивом
		 строк.


		 ![](../Interface_Image.gif)
		 [IStringMap](IStringMap/IStringMap.htm)
		 Интерфейс IStringMap
		 содержит свойства и методы для работы с картой строковых элементов.
