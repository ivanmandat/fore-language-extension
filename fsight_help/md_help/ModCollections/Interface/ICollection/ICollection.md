# ICollection


## Описание


ICollection - абстрактный интерфейс,
 содержит базовые свойства и методы всех коллекций.


## Иерархия наследования


           [IEnumerable](../IEnumerable/IEnumerable.htm)


           ICollection


## Комментарии


Все коллекции, которые наследуются от ICollection,
 поддерживают работу через цикл [For Each](Fore.chm::/08_Operators/Fore_For.htm).


Данный интерфейс является базовым для следующих интерфейсов:


	- [IArrayList](../IArrayList/IArrayList.htm);


	- [IBitArray](../IBitArray/IBitArray.htm);


	- [IDictionary](../IDictionary/IDictionary.htm);


	- [IList](../IList/IList.htm);


	- [IQueue](../IQueue/IQueue.htm);


	- [IStack](../IStack/IStack.htm);


	- [IStringList](../IStringList/IStringList.htm);


	- [IStringMap](../IStringMap/IStringMap.htm).
