# IComparer


## Описание


IComparer - абстрактный интерфейс,
 содержащий метод для сравнения двух объектов.


## Комментарии


Метод [Compare](IComparer.Compare.htm) используется для сравнения
 данных различных типов. Для разных типов данных в ядре реализованы различные
 алгоритмы сравнения. В зависимости от типа имеющихся данных используйте
 соответствующее [статическое
 свойство](../IComparerClass/IComparerClass.htm) для получения объекта сравнения нужного типа.


Также метод [Compare](IComparer.Compare.htm) может быть переопределен
 в прикладном классе для сравнения значений по [алгоритму
 пользователя](../../Samples/User_Comparer.htm).
