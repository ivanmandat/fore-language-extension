# IQueue


## Описание


Интерфейс IQueue содержит свойства
 и методы для работы с очередью элементов.


## Иерархия наследования


           [IEnumerable](../IEnumerable/IEnumerable.htm)


           [ICollection](../ICollection/ICollection.htm)


           IQueue


## Комментарии


Принцип работы очереди - «Первый пришел - первый ушел». Элементы всегда
 добавляются в конец очереди.
