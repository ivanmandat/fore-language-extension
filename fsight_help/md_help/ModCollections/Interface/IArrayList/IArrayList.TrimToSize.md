# IArrayList.TrimToSize


## Синтаксис


TrimToSize;


## Описание


Метод TrimToSize устанавливает размер массива, равный фактическому количеству элементов в массиве.


## Комментарии


После вызова метода размер массива [Capacity](IArrayList.Capacity.htm) будет установлен в соответствии с фактическим числом элементов в массиве [Count](../ICollection/ICollection.Count.htm).
