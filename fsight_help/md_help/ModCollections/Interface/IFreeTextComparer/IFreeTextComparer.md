# IFreeTextComparer


## Описание


Интерфейс IFreeTextComparer
 предназначен для свободного сравнения текста по заданным параметрам.


## Комментарии


Для получения доступа к определению параметров сравнения приведите объект,
 возвращаемый свойством [IComparerClass.FreeTextComparer](../IComparerClass/IComparerClass.FreeTextComparer.htm),
 к интерфейсу IFreeTextComparer.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ForEachWord](IFreeTextComparer.ForEachWord.htm)


		 Свойство ForEachWord
		 определяет, осуществлять ли сравнение по вхождению как каждого
		 слова в отдельности, так и всех слов одновременно.


		 ![](../../Property_Image.gif)
		 [MatchCase](IFreeTextComparer.MatchCase.htm)


		 Свойство MatchCase
		 определяет, учитывать ли регистр при сравнении.


		 ![](../../Property_Image.gif)
		 [Options](IFreeTextComparer.Options.htm)


		 Свойство Options определяет
		 параметры сравнения.


		 ![](../../Property_Image.gif)
		 [WholeWord](IFreeTextComparer.WholeWord.htm)


		 Свойство WholeWord
		 определяет, осуществлять ли сравнение только строки целиком.
