# IComparerClass


## Описание


Интерфейс IComparerClass содержит
 статические свойства класса [Comparer](../../Class/Comparer/Comparer.htm),
 реализующие объекты для сравнения элементов различного типа.


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DateTimeComparer](IComparerClass.DateTimeComparer.htm)
		 Свойство DateTimeComparer
		 возвращает объект для сравнения элементов типа DateTime.


		 ![](../../Property_Image.gif)
		 [DoubleComparer](IComparerClass.DoubleComparer.htm)
		 Свойство DoubleComparer
		 возвращает объект для сравнения элементов вещественного типа.


		 ![](../../Property_Image.gif)
		 [FreeTextComparer](IComparerClass.FreeTextComparer.htm)
		 Свойство FreeTextComparer
		 возвращает объект для свободного сравнения текста.


		 ![](../../Property_Image.gif)
		 [IntegerComparer](IComparerClass.IntegerComparer.htm)
		 Свойство IntegerComparer
		 возвращает объект для сравнения элементов целого типа.


		 ![](../../Property_Image.gif)
		 [StringCaseInsensitiveComparer](IComparerClass.StringCaseInsensitiveComparer.htm)
		 Свойство StringCaseInsensitiveComparer
		 возвращает объект для сравнения элементов строкового типа без

		 [StringComparer](IComparerClass.StringComparer.htm)
		 Свойство StringComparer
