# IStack


## Описание


Интерфейс IStack содержит свойства
 и методы для работы со стеком элементов.


## Иерархия наследования


           [IEnumerable](../IEnumerable/IEnumerable.htm)


           [ICollection](../ICollection/ICollection.htm)


           IStack


## Комментарии


Принцип работы стека - «Первый пришел - последний ушел». Элементы всегда
 добавляются в начало стека.
