		 Класс ArrayList реализует
		 объект, представляющий из себя динамический массив элементов.


		 ![](../Class_Image.gif)
		 [BitArray](BitArray/BitArray.htm)
		 Класс BitArray реализует
		 объект, представляющий динамический битовый массив.


		 ![](../Class_Image.gif)
		 [Comparer](Comparer/Comparer.htm)
		 Класс Comparer реализует
		 объект, используемый для сравнения элементов различного типа.


		 ![](../Class_Image.gif)
		 [Hashtable](Hashtable/Hashtable.htm)
		 Класс Hashtable реализует
		 объект, представляющий собой коллекцию пар «Ключ-Значение», основанных
		 на хеш-коде ключа.


		 ![](../Class_Image.gif)
		 [Queue](Queue/Queue.htm)
		 Класс Queue реализует
		 объект, представляющий собой очередь элементов.


		 ![](../Class_Image.gif)
		 [SortedList](SortedList/SortedList.htm)
		 Класс SortedList реализует
		 объект, представляющий собой коллекцию пар «Ключ-Значение», отсортированную
		 по значениям ключа.


		 ![](../Class_Image.gif)
		 [Stack](Stack/Stack.htm)
		 Класс Stack реализует
		 объект, представляющий собой стек элементов.


		 ![](../Class_Image.gif)
		 [StringList](StringList/StringList.htm)
		 Класс StringList реализует
		 объект, представляющий собой динамический массив строк.


		 ![](../Class_Image.gif)
		 [StringMap](StringMap/StringMap.htm)
		 Класс StringMap реализует
		 объект, представляющий из себя карту символьных строк.
