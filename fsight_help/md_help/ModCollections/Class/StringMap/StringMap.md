# StringMap


## Описание


Класс StringMap реализует объект, представляющий из себя карту символьных строк.


## Комментарий


В карте строк каждому элементу соответствует уникальный символьный ключ. Сортировка карты строк осуществляется автоматически при добавлении новых элементов. Карта сортируется по возрастанию значений ключа. Работа с элементами осуществляется по ключу. Для получения ключа по индексу элемента используется свойство [Key](../../Interface/IStringMap/IStringMap.Key.htm).
