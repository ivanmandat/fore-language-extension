# TextComparerOptions


## Описание


Перечисление TextComparerOptions
 содержит параметры сравнения текста.


Используется следующим свойством:


	- [IFreeTextComparer.Options](../Interface/IFreeTextComparer/IFreeTextComparer.Options.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 MatchCase. Сравнение
		 с учетом регистра.


		 2
		 WholeWord. Сравнение
		 с учетом полноты строки.


		 4
		 ForEachWord. Сравнение
		 по вхождению как каждого слова в отдельности, так и всех слов
		 одновременно. Если строка состоит из нескольких слов, то ищутся
		 вхождения как по каждому слову в отдельности, так и по вхождению
		 всех слов. При поиске вхождений отбрасываются окончания существительных.


Например, при поиске фразы «атрибуты справочника»
		 будут найдены элементы, содержащие как «атрибуты справочника»,
		 так и «атрибуты», «атрибут», «справочника», «справочник».
