## С чего начать


		 Сборка Dashboard
		 предназначена для работы с объектом репозитория «Информационная
		 панель».


		Изучив данный раздел, Вы сможете с применением языка Fore:


			- получать доступ к структуре информационной панели;


			- работать с визуализаторами, изменять их настройки.


		Описание ресурсов сборки представлено в следующих подразделах:


			- [Интерфейсы
			 сборки Dashboard](Interface/KeDashboard_Interface.htm)


			- [Классы сборки
			 Dashboard](Class/KeDashboard_Class.htm)


		 Перед изучением сборки Dashboard рекомендуется ознакомиться
		 с разделами: «[Среда разработки](UiDevEnv.chm::/01_Development_Environment/01_Purpose_of_the_constructor/Purpose_of_the_constructor.htm)»
		 и «[Руководство
		 по языку Fore](Fore.chm::/Fore_Title.htm)».
