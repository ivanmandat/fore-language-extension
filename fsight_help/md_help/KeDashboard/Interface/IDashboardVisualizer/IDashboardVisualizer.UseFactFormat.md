# IDashboardVisualizer.UseFactFormat


## Синтаксис


UseFactFormat: Boolean;


## Описание


Свойство UseDefaultFormat определяет
 признак использования для фактов формата данных по умолчанию.


## Комментарии


По умолчанию свойству установлено значение True,
 при этом у визуализатора для фактов используется формат данных по умолчанию.
 Если для фактов необходимо использовать разные форматы данных, то установите
 свойству значение False.
