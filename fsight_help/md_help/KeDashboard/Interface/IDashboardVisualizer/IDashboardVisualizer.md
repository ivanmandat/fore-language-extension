# IDashboardVisualizer


Сборка: Dashboard;


## Описание


Интерфейс IDashboardVisualizer
 предназначен для работы с визуализатором информационной панели.


## Иерархия наследования


           [IDashboardBaseVisualObject](../IDashboardBaseVisualObject/IDashboardBaseVisualObject.htm)


           IDashboardVisualizer


## Комментарии


Визуализатор информационной панели можно получить, используя свойства
 и методы коллекции [IDashboardVisualizers](../IDashboardVisualizers/IDashboardVisualizers.htm).
