# IDashboardVisualizers.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IDashboardVisualizer](../IDashboardVisualizer/IDashboardVisualizer.htm);


## Параметры


Key. Ключ визуализатора.


## Описание


Метод FindByKey осуществляет
 поиск визуализатора по указанному ключу.


## Комментарии


Метод возвращает настройки визуализатора в случае удачного поиска и
 Null, если визуализатор с указанным
 ключом отсутствует в коллекции.
