# IDashboardVisualizers.FindById


## Синтаксис


FindById(Id: String): [IDashboardVisualizer](../IDashboardVisualizer/IDashboardVisualizer.htm);


## Параметры


Id. Идентификатор визуализатора.


## Описание


Метод FindById осуществляет
 поиск визуализатора по указанному идентификатору.


## Комментарии


Метод возвращает настройки визуализатора в случае удачного поиска и
 Null, если визуализатор с указанным
 идентификатором отсутствует в коллекции.
