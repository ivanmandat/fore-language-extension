# IDashboardVisualizers


Сборка: Dashboard;


## Описание


Интерфейс IDashboardVisualizers
 предназначен для работы с коллекцией визуализаторов информационной панели.


## Иерархия наследования


IDashboardVisualizers


## Комментарии


Коллекцию визуализаторов можно получить используя свойство [IDashboard.Visualizers](../IDashboard/IDashboard.Visualizers.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDashboardVisualizers.Count.htm)
		 Свойство Count возвращает
		 количество визуализаторов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IDashboardVisualizers.Item.htm)
		 Свойство Item возвращает
		 настройки визуализатор с указанным индексом.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](IDashboardVisualizers.FindById.htm)
		 Метод FindById осуществляет
		 поиск визуализатора по указанному идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IDashboardVisualizers.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск визуализатора по указанному ключу.


		 ![](../../Sub_Image.gif)
		 [IndexOf](IDashboardVisualizers.IndexOf.htm)
		 Метод IndexOf возвращает
		 для указанного визуализатора индекс в коллекции.
