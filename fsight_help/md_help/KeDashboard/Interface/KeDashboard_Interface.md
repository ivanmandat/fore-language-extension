# Интерфейсы сборки Dashboard


		 Интерфейс
		 Краткое описание


		 ![](../Interface_Image.gif)
		 [IDashboard](IDashboard/idashboard.htm)
		 Интерфейс IDashboard
		 содержит свойства и методы объекта репозитория - [Информационная панель](DataAnalysis.chm::/Dashboard.htm).


		 ![](../Interface_Image.gif)
		 [IDashboardBaseVisualObject](IDashboardBaseVisualObject/IDashboardBaseVisualObject.htm)
		 Интерфейс IDashboardBaseVisualObject
		 содержит базовые свойства визуальных объектов информационной панели.


		 ![](../Interface_Image.gif)
		 [IDashboardEventArgs](IDashboardEventArgs/IDashboardEventArgs.htm)
		 Интерфейс IDashboardEventArgs
		 содержит свойства аргумента события, наступающего при открытии
		 информационной панели.


		 ![](../Interface_Image.gif)
		 [IDashboardUserEvents](IDashboardUserEvents/IDashboardUserEvents.htm)
		 Интерфейс IDashboardUserEvents
		 содержит методы, реализующие события информационной панели.


		 ![](../Interface_Image.gif)
		 [IDashboardVisualizer](IDashboardVisualizer/IDashboardVisualizer.htm)
		 Интерфейс IDashboardVisualizer
		 предназначен для работы с визуализатором информационной панели.


		 ![](../Interface_Image.gif)
		 [IDashboardVisualizerCancelEventArgs](IDashboardVisualizerCancelEventArgs/IDashboardVisualizerCancelEventArgs.htm)
		 Интерфейс IDashboardVisualizerCancelEventArgs
		 содержит свойства аргумента события, наступающего перед обновлением
		 визуализатора информационной панели.


		 ![](../Interface_Image.gif)
		 [IDashboardVisualizerEventArgs](IDashboardVisualizerEventArgs/IDashboardVisualizerEventArgs.htm)
		 Интерфейс IDashboardVisualizerEventArgs
		 содержит свойства аргумента событий, связанных с визуализаторами
		 информационной панели.


		 ![](../Interface_Image.gif)
		 [IDashboardVisualizers](IDashboardVisualizers/IDashboardVisualizers.htm)
		 Интерфейс IDashboardVisualizers
		 предназначен для работы с коллекцией визуализаторов информационной
		 панели.
