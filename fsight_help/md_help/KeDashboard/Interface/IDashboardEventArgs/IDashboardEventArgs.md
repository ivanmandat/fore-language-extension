# IDashboardEventArgs


Сборка: Dashboard;


## Описание


Интерфейс IDashboardEventArgs
 содержит свойства аргумента события, наступающего при открытии информационной
 панели.


## Иерархия наследования


           [IEventArgs](ForeSys.chm::/Interface/IEventArgs/IEventArgs.htm)


           IDashboardEventArgs


## Комментарии


Данный аргумент используется при обработке события с помощью метода
 [IDashboardUserEvents.EventOnBeforeOpenDashboard](../IDashboardUserEvents/IDashboardUserEvents.EventOnBeforeOpenDashboard.htm).
