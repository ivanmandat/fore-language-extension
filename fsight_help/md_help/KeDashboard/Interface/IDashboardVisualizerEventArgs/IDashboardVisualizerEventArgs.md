# IDashboardVisualizerEventArgs


Сборка: Dashboard;


## Описание


Интерфейс IDashboardVisualizerEventArgs
 содержит свойства аргумента событий, связанных с визуализаторами информационной
 панели.


## Иерархия наследования


           [IEventArgs](ForeSys.chm::/Interface/IEventArgs/IEventArgs.htm)


           IDashboardVisualizerEventArgs


## Комментарии


Данный аргумент используется при обработке события с помощью метода
 [IDashboardUserEvents.EventOnAfterRefreshVisualizer](../IDashboardUserEvents/IDashboardUserEvents.EventOnAfterRefreshVisualizer.htm).
 Также текущий интерфейс является базовым для интерфейса [IDashboardVisualizerCancelEventArgs](../IDashboardVisualizerCancelEventArgs/IDashboardVisualizerCancelEventArgs.htm).
