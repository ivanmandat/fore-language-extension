# IDashboardUserEvents


Сборка: Dashboard;


## Описание


Интерфейс IDashboardUserEvents
 содержит методы, реализующие события информационной панели.


## Иерархия наследования


IDashboardUserEvents


## Комментарии


Методы данного интерфейса должны быть переопределены в пользовательском
 классе.
