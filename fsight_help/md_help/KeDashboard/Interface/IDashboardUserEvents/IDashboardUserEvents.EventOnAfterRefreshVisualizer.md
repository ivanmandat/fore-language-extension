# IDashboardUserEvents.EventOnAfterRefreshVisualizer


## Синтаксис


EventOnAfterRefreshVisualizer(Args: [IDashboardVisualizerEventArgs](../IDashboardVisualizerEventArgs/IDashboardVisualizerEventArgs.htm));


## Параметры


Args. Аргумент события.


## Описание


Метод EventOnAfterRefreshVisualizer
 реализует событие, наступающее после обновления визуализатора информационной
 панели.


## Комментарии


Метод должен быть переопределён в пользовательском классе. Класс-обработчик
 событий для информационной панели задаётся в свойстве [IDashboard.EventsClass](../IDashboard/IDashboard.EventsClass.htm).
