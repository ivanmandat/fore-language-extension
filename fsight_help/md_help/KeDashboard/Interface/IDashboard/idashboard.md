# IDashboard


Сборка: Dashboard;


## Описание


Интерфейс IDashboard содержит
 свойства и методы объекта репозитория - [Информационная панель](DataAnalysis.chm::/Dashboard.htm).


## Иерархия наследования


IDashboard


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [EventsAssembly](IDashboard.EventsAssembly.htm)
		 Свойство EventsAssembly
		 определяет описание объекта среды разработки, который будет использоваться
		 для обработки событий информационной панели.


		 ![](../../Property_Image.gif)
		 [EventsClass](IDashboard.EventsClass.htm)
		 Свойство EventsClass
		 определяет класс, используемый для обработки событий информационной
		 панели.


		 ![](../../Property_Image.gif)
		 [Headers](IDashboard.Headers.htm)
		 Свойство Headers возвращает
		 настройки колонтитулов для печати информационной панели.


		 ![](../../Property_Image.gif)
		 [MetabaseObject](IDashboard.MetabaseObject.htm)
		 Свойство MetabaseObject
		 возвращает объект репозитория, соответствующий текущей аналитической
		 панели.


		 ![](../../Property_Image.gif)
		 [Visualizers](IDashboard.Visualizers.htm)
		 Свойство Visualizers
		 возвращает коллекцию визуализаторов аналитической панели.
