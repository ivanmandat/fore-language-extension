# IDashboard.EventsClass


## Синтаксис


EventsClass: String;


## Описание


Свойство EventsClass определяет
 класс, используемый для обработки событий информационной панели.


## Комментарии


Указываемый класс должен содержать реализацию методов интерфейса [IDashboardUserEvents](../IDashboardUserEvents/IDashboardUserEvents.htm)
 или класса [DashboardEvents](../../Class/DashboardEvents/DashboardEvents.htm).
