# IDashboardVisualizerCancelEventArgs.Cancel


## Синтаксис


Cancel: Boolean;


## Описание


Свойство Cancel определяет,
 будет ли сгенерировано событие для визуализатора информационной панели.


## Комментарии


Допустимые значения:


	- True. Событие будет
	 отменено;


	- False. По умолчанию.
	 Событие будет сгенерировано для визуализатора.
