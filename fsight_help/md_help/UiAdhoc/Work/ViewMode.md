# Работа с готовой аналитической
 панелью


Перед работой с готовой аналитической панелью в веб-приложении проверьте
 настройки браузера: убедитесь, что для веб-приложения разрешены всплывающие
 окна.


Для работы с готовой аналитической панелью выполните шаги:


	- [Обновите данные аналитической панели](Refresh.htm).


	- [Настройте аналитическую панель
	 с помощью интерактивных элементов управления](InteractiveElements.htm).


	- [Отмена/повтор операции над объектами
	 аналитической панели](Undo_Redo.htm).


	- [Экспортируйте аналитическую
	 панель](../Document/Export.htm).


	- [Выполните предварительный просмотр
	 и печать аналитической панели](PreviewPrint.htm).


	- [Создайте копию аналитической панели](Copy.htm).


	- [Используйте дополнительные
	 сервисы при работе с готовой аналитической панелью](AdvancedServices.htm#link).


	- [Опубликуйте и обсудите аналитическую
	 панель в социальных сетях](Publication.htm).


После работы с готовой аналитической панелью:


	- выполните команду «Документ
	 > Закрыть» в [главном меню](../Launching.htm)
	 для закрытия текущей аналитической панели и продолжения работы с инструментом
	 «Аналитические панели»»;


	- выполните команду «Документ
	 > Выход» в главном меню для закрытия аналитической панели
	 и завершения работы с инструментом «Аналитические
	 панели».


Если имеются несохранённые изменения, то будет предложено их сохранить.
