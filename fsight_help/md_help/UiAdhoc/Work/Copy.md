# Создание копии аналитической панели


Для создания копии аналитической панели выполните команду «Сохранить как» в главном меню. Будет
 отображен диалог для выбора места хранения копии панели в текущем репозитории.


В настольном приложении по умолчанию объект сохраняется со старым именем
 и с автоматически сгенерированным идентификатором.


В веб-приложении необходимо ввести новое имя объекта и, по желанию,
 идентификатор. Если идентификатор не указан, он создается автоматически
 при сохранении.


Важно. Идентификатор
 сохраняемого объекта не должен содержать символы русского алфавита и пробелы.


При создании копии аналитической панели в репозитории создается новая
 аналитическая панель, которая сразу же открывается в том же режиме, в
 котором была открыта исходная аналитическая панель.


Примечание.
 Сохранение копии панели доступно, если у пользователя есть [права](Admin.chm::/03_Admin/Admin_AdminObjects_Discretionary.htm)
 на изменение данного объекта репозитория.
