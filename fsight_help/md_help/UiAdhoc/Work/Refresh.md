# Обновление данных аналитической панели


Для обновления данных аналитической панели нажмите кнопку ![](../Ui/update.png)
 «Обновить», расположенную на вкладке
 «Главная» ленты инструментов.


Для переоткрытия всех источников данных, используемых в аналитической
 панели, выполните команду «Переоткрыть
 источники данных» в раскрывающемся меню кнопки ![](../Ui/update.png)
 «Обновить».


Примечание.
 Возможность переоткрытия источников данных доступна только в настольном
 приложении.
