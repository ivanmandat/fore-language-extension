# Публикация в социальных сетях и обсуждение аналитической панели


Для публикации ссылки на аналитическую панель в социальной сети выполните
 команду «[Поделиться](UiNav.chm::/GUI/Format/Publishing.htm)»
 в главном меню и в отобразившемся меню выберите необходимую социальную
 сеть.


Примечание.
 Возможность доступна только в веб-приложении.


См. также:


[Работа с готовой аналитической панелью](ViewMode.htm)
