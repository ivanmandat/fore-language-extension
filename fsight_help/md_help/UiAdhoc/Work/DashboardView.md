# Просмотр готовой аналитической панели


Для просмотра готовой аналитической панели нажмите кнопку ![](../Document/view.png)
 «Просмотр» на вкладке ленты
 инструментов «Главная» или «Вставить».


Инструмент «Аналитические панели»
 будет переведен в [режим просмотра](../Launching.htm), предназначенный
 для работы с [готовыми аналитическими панелями](ViewMode.htm).


Совет. Перед запуском
 режима просмотра в веб-приложении проверьте настройки браузера: убедитесь,
 что для веб-приложения разрешены всплывающие окна.
