# Отмена/повтор операции над объектами аналитической панели


Для отмены последней операции, совершенной над объектом:


	- нажмите кнопку ![](undo.png) на панели [быстрого
	 доступа](GetStarted.chm::/Interface/Interface_Description.htm#quick_access_toolbar) ленты инструментов;


	- нажмите сочетание клавиш CTRL+Z.


Для повтора последней отмененной операции:


	- нажмите кнопку ![](redo.png) на панели [быстрого
	 доступа](GetStarted.chm::/Interface/Interface_Description.htm#quick_access_toolbar) ленты инструментов;


	- нажмите сочетание клавиш CTRL+Y.


Примечание.
 Если в настольном приложении на панели быстрого доступа ленты инструментов
 не отображаются кнопки ![](undo.png) и ![](redo.png), то
 выполните [настройку
 панели](GetStarted.chm::/Interface/Interface_Description.htm#quick_access_toolbar): добавьте команды «Повторить»
 и «Отменить» из категории «Главная».
