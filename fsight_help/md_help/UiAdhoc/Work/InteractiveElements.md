# Работа с интерактивными элементами управления


Для управления содержимым блоков предназначены интерактивные [элементы
 управления](../Blocks/Controls/Controls.htm), создаваемые на этапе построения аналитической панели.


Для работы с блоками предназначены интерактивные кнопки управления,
 расположенные в правом верхнем углу блока. Кнопки управления позволяют
 быстро выполнять базовые операции по управлению объектом.


Набор доступных кнопок задаётся на этапе построения аналитической панели
 на вкладке «[Блок](../Document/Block_BaseProps.htm#control_buttons)»
 боковой панели.


[![](../Opened.gif)![](../Closed.gif)Для
 отображения кнопок управления](javascript:TextPopup(this))


Наведите указатель мыши на уголок в правом верхнем углу блока:


![](../Document/block_corner.png)


Будут отображены [кнопки управления](../Document/Block_BaseProps.htm)
 объектом, например:


![](../Document/control_buttons.png)


Доступны следующие кнопки:


	- ![](../Document/b_export.png). Отображает меню для выполнения операций
	 с блоками:


		- Печать.
		 Печать выбранного блока без предварительного просмотра;


		- Экспорт.
		 Экспорт блока в выбранный формат. Подробное описание доступных
		 форматов и особенностей экспорта приведено в разделе «[Экспорт
		 аналитической панели](../Document/Export.htm)»;


		- Ссылка на блок.
		 Получение ссылки на блок, позволяющей встроить его в пользовательское
		 веб-приложение или аналитическую панель. Возможность доступна
		 только в веб-приложении, если в файле «[PP.xml](Setup.chm::/UiWebSetup/03_Setup_Web/PP_config_Java.htm)» в
		 разделе <pp> присутствует
		 строка <embed enabled="true" />;


	- ![](../Blocks/Gadgets/Panels/b_dim_menu.png). Отображает меню для быстрой настройки
	 визуализатора. Кнопка доступна только для визуализаторов. Доступные
	 настройки зависят от типа визуализатора и выделенной области визуализатора.
	 Если для выделенной области отсутствуют быстрые настройки, то отображается
	 надпись «Нет доступных действий».


		- Настройки, доступные для [таблицы](../Blocks/Gadgets/Table.htm):


			- детализация и обобщение данных;


			- перемещение данных в буфер обмена и вставка данных из
			 буфера обмена;


			- расчёт статистических характеристик.


		- Настройки, доступные для [диаграммы](../Blocks/Gadgets/Diagram.htm):
		 детализация и обобщение данных.


		- Настройки, доступные для всех визуализаторов: исключение
		 данных из визуализатора.


Подробное описание настроек и способов их
 выполнения приведено в разделе, описывающем настройку соответствующего
 [визуализатора](../Blocks/Gadgets/Gadgets.htm);


	- ![](b_data.png).
	 Отображает данные, на основании которых строится блок;


	- ![](b_maximize.png)
	 / ![](b_minimize.png).
	 Сворачивает и разворачивает блок.
