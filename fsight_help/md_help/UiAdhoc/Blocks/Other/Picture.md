# Вставка рисунка


Блок «Рисунок» предназначен
 для вставки изображений из файла в аналитическую панель. Также доступно
 выполнение какой-либо команды при нажатии на рисунок.


Для добавления рисунка используйте:


	- команду «Рисунок» в
	 раскрывающемся меню кнопки «Другое»
	 на вкладке ленты «Главная»;


	- кнопку «Рисунок» в группе
	 «Другое» на вкладке ленты
	 «Вставка»;


	- команду «Рисунок» в
	 раскрывающемся меню пункта «Новый
	 блок» в контекстном меню аналитической панели.


Отобразится диалог выбора изображения, в котором укажите требуемый файл.
 Для использования доступны изображения в форматах: *.bmp, *.jpeg, *.png,
 *.gif, *.tif.


Пример блоков «[Рисунок](Picture.htm)»,
 «Надпись» и «[Визуализатор](../Gadgets/Gadgets.htm)»:


![](Text.png)


Для настройки параметров блока используйте вкладки «Блок»
 и «Команда» на боковой панели.


[![](../../Opened.gif)![](../../Closed.gif)Для отображения
 вкладок](javascript:TextPopup(this))


		- Убедитесь, что боковая панель отображается.


		- Выделите блок на рабочей области.


		- Перейдите на требуемую вкладку.


## Настройка базовых параметров


Для настройки базовых параметров рисунка используйте вкладку «[Блок](../../Document/Block_BaseProps.htm)»
 на боковой панели.


Помимо параметров, доступных для всех объектов в аналитической панели,
 для рисунка также доступна возможность изменения рисунка. Для этого:


	- в веб-приложении нажмите кнопку ![](b_browse.png) «Обзор»;


	- в настольном приложении нажмите кнопку «Обзор».


Отобразится диалог выбора изображения, в котором укажите требуемый файл.


## Управление командами


Для управления действием, выполняемым при нажатии рисунка, используйте
 вкладку «[Команда](../Controls/Button.htm#button_items_control)»
 на боковой панели:


![](../Controls/button_command.png)


Для рисунка доступны те же команды, что и для элемента управления «[Кнопка](../Controls/Button.htm)».
