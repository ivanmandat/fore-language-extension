# Вставка надписей, иллюстраций и
 HTML-контента


При создании аналитической панели используются блоки для отображения
 текстовой и графической информации произвольной формы:


	- [Надпись](Label.htm).
	 Предназначен для отображения текстовой информации;


	- [Рисунок](Picture.htm).
	 Предназначен для отображения изображений;


	- [HTML](html_link.htm).
	 Предназначен для отображения содержимого какого-либо удаленного ресурса.


Для элементов управления доступны все операции с объектами, приведенные
 в разделах «[Основные
 операции с объектами аналитической панели](../../Document/Layout_mode.htm#objects_setup)» и «[Настройка
 внешнего вида элемента эскиза](../../Document/AdjustingAppearance.htm)».
