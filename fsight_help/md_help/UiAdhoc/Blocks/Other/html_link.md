# Вставка HTML-контента


Блок «HTML» предназначен для
 отображения содержимого, расположенного по заданному URL-адресу.


Для добавления блока используйте:


	- команду «HTML» в раскрывающемся
	 меню кнопки «Другое» на вкладке
	 ленты «Главная»;


	- кнопку «HTML» в группе
	 «Другое» на вкладке ленты
	 «Вставка»;


	- команду «HTML» в раскрывающемся
	 меню пункта «Новый блок» в
	 контекстном меню аналитической панели.


Отобразится диалог «Создание HTML блока»,
 в котором введите URL-адрес требуемого ресурса.

![](html.png)


## Настройка базовых параметров


Для настройки базовых параметров блока «HTML»
 используйте вкладку «[Блок](../../Document/Block_BaseProps.htm)»
 на боковой панели.


[![](../../Opened.gif)![](../../Closed.gif)Для отображения
 вкладки](javascript:TextPopup(this))


		- Убедитесь, что боковая панель отображается.


		- Выделите блок на рабочей области.


		- Перейдите на вкладку «Блок».


Помимо параметров, доступных для всех объектов в аналитической панели,
 для блока «HTML» также доступна
 возможность изменения отображаемого ресурса. Для этого введите требуемый
 URL-адрес ресурса в поле «URL-адрес».
