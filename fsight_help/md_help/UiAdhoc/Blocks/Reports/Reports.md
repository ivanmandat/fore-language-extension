# Вставка и настройка отчётов


В аналитической панели отчёты используются для отображения информации
 из источников без возможности анализа и настройки отображаемых данных.
 Доступные виды отчётов:


	- [Регламентный
	 отчёт](Procedural_report.htm). Отображает данные регламентного отчёта;


	- [Аналитическая
	 панель](Dashboard.htm). Отображает данные аналитической панели;


	- [Экспресс-отчёт](Express_Report.htm).
	 Отображает данные экспресс-отчёта;


	- [Временные
	 ряды](TimeSeries.htm). Отображает данные рабочей книги.


Оформление элементов в отчёте соответствует оформлению, настроенному
 в источнике данных.


Примечание.
 По умолчанию в веб-приложении доступно только создание блоков типа
 «Регламентный отчет», в настольном
 приложении доступно создание блоков типа «Регламентный
 отчет» и «Аналитическая панель».
 Для создания других видов блоков, предназначенных для работы с отчетами,
 требуются дополнительные настройки. Данные настройки не влияют на доступность
 отчётов в сохраненных аналитических панелях. Подробнее см. раздел: «[Как сделать доступным создание блоков,
 предназначенных для работы с отчетами?](../../FAQ/Reports.htm)».


## Операции с отчётами


Для отчётов доступны все операции с объектами, приведенные в разделах
 «[Основные операции
 с объектами аналитической панели](../../Document/Layout_mode.htm#objects_setup)» и «[Настройка
 внешнего вида элемента эскиза](../../Document/AdjustingAppearance.htm)». Также для отчётов доступно:


[![](../../Opened.gif)![](../../Closed.gif)Выбор
 источника данных](javascript:TextPopup(this))


	Для выбора источника данных используйте вкладку «Отчёт»
	 на боковой панели. Для отображения вкладки:


		- Убедитесь, что боковая панель отображается.


		- В рабочей области добавьте новый отчёт или выберите существующий.


		- Перейдите на вкладку «Отчёт».


	На вкладке отображаются все доступные для отчёта источники данных,
	 расположенные в репозитории. Для выбора или смены источника данных
	 выберите соответствующий элемент в дереве объектов:
