# Выбор элементов эскиза для компоновки
 панели


Эскиз определяет взаимное расположение блоков аналитической панели и
 формируется с помощью элементов-контейнеров. Настройка параметров элементов-контейнеров
 выполняется с помощью вкладок на боковой панели.


Доступные элементы-контейнеры:


	- [Сетка](Container_Grid.htm).
	 Набор ячеек в виде таблицы. Каждая ячейка является контейнером для
	 размещения объектов аналитической панели;


	- [Вкладки](Container_Tabs.htm).
	 Многостраничный элемент-контейнер, позволяющий объединить в группу
	 несколько объектов аналитической панели;


	- [Панель](Container_Panel.htm).
	 Одностраничный элемент-контейнер, позволяющий объединить в группу
	 несколько объектов аналитической панели.


Элемент-контейнер может содержать вложенные блоки и вложенные элементы-контейнеры.


## Основные операции с элементами-контейнерами


Для элементов-контейнеров доступны все операции с объектами, приведенные
 в разделе «[Проектирование эскиза
 панели](../Document/Layout_mode.htm)».
