# Управление объектами


Для управления [объектами аналитической
 панели](Layout_mode.htm#objects_setup) используйте вкладку «Навигатор
 блоков» на боковой панели.


[![](../Opened.gif)![](../Closed.gif)Для отображения
 вкладки](javascript:TextPopup(this))


		- Убедитесь, что боковая панель отображается.


		- Щёлкните по пустому пространству в рабочей области.


		- Установите переключатель «Слайд».


	Примечание.
	 Пропустите шаг, если аналитическая панель создана в настольном приложении.
	 При открытии аналитической панели в веб-приложении данный переключатель
	 будет недоступен.


		- Перейдите на вкладку «Навигатор
		 блоков».


![](block_navigator.png)


На вкладке содержится набор объектов, расположенных на рабочей области
 в рамках текущего [слайда](../Presentation/Presentation.htm),
 с указанием типа и адреса объекта. При переходе на другой слайд набор
 объектов меняется.


Если на рабочей области содержатся [элементы-контейнеры](../Blocks/Create_markup.htm)
 с вложенными объектами, то набор объектов будет отображаться в виде дерева,
 вершинами которого являются элементы-контейнеры.


Операции с объектами:


[![](../Opened.gif)![](../Closed.gif)Скрытие/отображение
 объекта](javascript:TextPopup(this))


	Для скрытия объекта:


		- Выделите объект.


		- Нажмите кнопку «Скрыть
		 блок».


	После выполнения действий объект будет скрыт на рабочей области.


	Для отображения скрытого объекта:


		- Выделите объект.


		- Нажмите кнопку «Показать
		 блок».


	После выполнения действий объект будет отображён на рабочей области.


[![](../Opened.gif)![](../Closed.gif)Поиск и навигация
 по объектам](javascript:TextPopup(this))


	Для удобного поиска и навигации по объектам выполните одно из действий:


		- дважды щёлкните по объекту;


		- выделите объект и нажмите кнопку «Показать
		 блок».


	После выполнения одного из действий объект будет выделен на рабочей
	 области и на боковой панели будут открыты настройки объекта.
