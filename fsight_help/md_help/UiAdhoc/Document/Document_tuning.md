# Использование дополнительных сервисов
 в аналитической панели


Аналитическая панель предоставляет следующие дополнительные возможности:


[![](../Opened.gif)![](../Closed.gif)Перевод
 элементов аналитической панели на другие языки](javascript:TextPopup(this))


	Примечание.
	 Доступно только в веб-приложении.


	Аналитические панели могут поддерживать несколько языков пользовательского
	 интерфейса, например, русский и английский. Подобные панели называются
	 мультиязычными.


	Для создания мультиязычной аналитической панели установите флажок
	 «Мультиязычность» на вкладке
	 боковой панели «Документ»
	 и задайте перевод для полей, переводимых на различные языки.


	Для получения подробной информации о создании мультиязычной аналитической
	 панели обратитесь к разделу «[Перевод
	 элементов аналитических панелей на другие языки](UiNav.chm::/Multilanguage/Multilanguage_tuning.htm)».


	Для получения подробной информации о настройке мультиязычности содержимого
	 репозитория и общей системы обратитесь к разделу «[Настройка
	 мультиязычности](uinav.chm::/Multilanguage/Multilanguage_setting.htm)».


[![](../Opened.gif)![](../Closed.gif)Включение
 режима редактирования данных](javascript:TextPopup(this))


	Для разрешения [редактирования табличных данных](Document.htm)
	 установите флажок «Редактирование
	 данных» на вкладке «Документ»
	 боковой панели. Для запрета редактирования снимите флажок «Редактирование
	 данных».


	Примечание.
	 Редактирование данных доступно, если у пользователя есть соответствующие
	 права.


[![](../Opened.gif)![](../Closed.gif)Использование
 встроенной расширенной аналитики](javascript:TextPopup(this))


	При построении аналитической панели доступно использование встроенной
	 расширенной аналитики.


	Для применения расширенной аналитики к данным в выделенной таблице
	 используйте вкладку «Расширенная
	 аналитика» ленты инструментов.


	Для получения подробной информации обратитесь к разделу «[Расширенная
	 аналитика](UiAnalyticalArea.chm::/Analysis/AdvancedAnalytics.htm)».


[![](../Opened.gif)![](../Closed.gif)Управление
 объектами](javascript:TextPopup(this))


	Примечание.
	 Доступно только в веб-приложении.


	Для управления [объектами
	 аналитической панели](Layout_mode.htm#objects_setup) используйте вкладку «Навигатор
	 блоков» на боковой панели.


	Для получения подробной информации обратитесь к разделу «[Управление
	 объектами](Object_Control.htm)».


[![](../Opened.gif)![](../Closed.gif)Работа
 со слайдами](javascript:TextPopup(this))


	Слайды позволяют создать многостраничную аналитическую панель. Для
	 работы со слайдами:


		- [Запустите инструмент «Аналитические
		 панели»](../Launching.htm).


		- [Создайте
		 слайды](../Presentation/Presentation.htm#slide_new).


	В целом, работа с отдельным слайдом аналогична работе с отдельной
	 аналитической панелью.


	Для получения подробной информации о слайдах обратитесь к разделу
	 «[Работа со слайдами](../Presentation/Presentation.htm)».


	Примечание.
	 Работа со слайдами доступна, если аналитическая панель создана в веб-приложении.
	 При открытии аналитической панели со слайдами в настольном приложении
	 будет доступно содержимое только первого слайда.


[![](../Opened.gif)![](../Closed.gif)Публикация
 в социальные сети](javascript:TextPopup(this))


	Примечание.
	 Доступно только в веб-приложении.


	Для публикации ссылки на аналитическую панель в социальной сети
	 выберите пункт «[Поделиться](UiNav.chm::/GUI/Format/Publishing.htm)»
	 в главном меню и в отобразившемся меню выберите необходимую социальную
	 сеть.


[![](../Opened.gif)![](../Closed.gif)Получение
 веб-ссылки](javascript:TextPopup(this))


	Ссылка позволяет встроить аналитическую панель или объект из аналитической
	 панели в пользовательский сайт, веб-приложение или другую аналитическую
	 панель.


	Для получения подробной информации обратитесь к разделу «[Получение
	 веб-ссылки](LinkToDashboard.htm)».
