# Настройка автоматического обновления данных


Автоматическое обновление позволяет обновлять данные при смене отметки
 в блоке или с заданным периодом. По умолчанию автоматическое обновление
 выполняется всякий раз при изменении отметки.


Все настройки обновления данных сохраняются вместе с аналитической панелью.


[![](../Opened.gif)![](../Closed.gif)Для
 автоматического обновления данных при смене отметки](javascript:TextPopup(this))


		- Перейдите на вкладку «Главная»
		 ленты инструментов.


		- Установите флажок «При
		 смене отметки» в раскрывающемся меню кнопки ![](../ui/auto_update.png) «Автоматическое обновление».


	Обновление блока будет выполняться каждый раз при изменении отметки
	 измерений его источника.


	Для отключения режима автоматического обновления при смене отметки
	 снимите флажок «При смене отметки»
	 в раскрывающемся меню кнопки ![](../ui/auto_update.png) «Автоматическое обновление».


[![](../Opened.gif)![](../Closed.gif)Для
 периодического обновления данных](javascript:TextPopup(this))


		- Перейдите на вкладку «Главная»
		 ленты инструментов.


		- Установите переключатель «Задать
		 период» в раскрывающемся меню кнопки ![](../ui/auto_update.png) «Автоматическое обновление».
		 Будет открыт диалог «Период автообновления»:


	![](../ui/update_period.png)


	В диалоге задайте период обновления данных
	 в аналитической панели. Обновление может выполняться каждые N секунд, минут или часов.


	Примечание.
	 Если для аналитической панели уже настраивался период обновления,
	 то его можно выбрать в раскрывающемся меню кнопки ![](../ui/auto_update.png) «Автоматическое обновление».


	Обновление аналитической панели будет выполняться раз в заданный
	 период времени.


	Для отключения периодического обновления данных установите переключатель
	 «Не использовать» в раскрывающемся
	 меню кнопки ![](../ui/auto_update.png) «Автоматическое обновление».


[![](../Opened.gif)![](../Closed.gif)Для
 отключения автоматического обновления](javascript:TextPopup(this))


		- В раскрывающемся меню кнопки ![](../ui/auto_update.png) «Автоматическое обновление» снимите
		 флажок «При смене отметки»
		 и установите переключатель «Не
		 использовать».


		- Переведите кнопку ![](../ui/auto_update.png) «Автоматическое обновление» в
		 ненажатое состояние.


	Автоматическое обновление аналитической панели будет отключено.


	Примечание.
	 Настройки [синхронизации
	 измерений](../Blocks/Dimension_links.htm#dashboard) аналитической панели срабатывают, даже если было отключено
	 автоматическое обновление данных.


	Для установки прежних параметров автоматического обновления нажмите
	 кнопку ![](../ui/auto_update.png) «Автоматическое
	 обновление».


Обновление блоков выполняется следующим образом:


	- если были созданы новые элементы измерения, то они будут добавлены
	 в раскрывающийся иерархический список элементов без отметки;


	- если элементы были удалены, то с них будет снята отметка, они
	 будут удалены из списка элементов измерения. Если был удален отмеченный
	 элемент фиксированного измерения, то отметка будет установлена для
	 следующего за ним элемента в списке;


	- если было создано новое измерение, то оно будет добавлено в
	 список фиксированных измерений, отметка будет установлена для первого
	 элемента;


	- удаленное измерение будет удалено из списка измерений.


## Обновление данных вручную


При выключенном автоматическом обновлении после изменения отметки блоки
 будут пустыми и будут содержать только ссылку «Отобразить
 данные». Для заполнения блока данными:


	- перейдите по ссылке «Отобразить
	 данные» в блоке;


	- нажмите кнопку ![](../Ui/update.png)
	 «Обновить» на вкладке «Данные» ленты инструментов.


Данные будут отображены в блоке.


Совет. Для переоткрытия
 источника данных выделенного блока выполните команду «Переоткрыть
 источники данных» в раскрывающемся меню кнопки ![](../Ui/update.png) «Обновить» на вкладке «Данные»
 ленты инструментов. Возможность доступна только в настольном приложении.
