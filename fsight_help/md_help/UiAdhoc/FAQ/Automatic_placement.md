# Как отключить режим автоматического размещения объектов, используемый
 по умолчанию?


По умолчанию аналитические панели создаются в режиме [автоматического
 размещения](../Document/Layout_mode.htm#automatic_placement) объектов, позволяющего располагать объекты в рабочей области
 без необходимости выполнять выравнивание, привязку и изменение размеров
 создаваемых объектов.


Если требуется создавать аналитические панели с возможностью располагать
 объекты в рабочей области произвольным образом с возможностью дальнейшего
 выравнивания, привязки и изменения размеров создаваемых объектов, то нужно
 выключить режим автоматического размещения объектов. Иными словами, нужно
 включить для всех создаваемых панелей режим [ручного
 размещения](../Document/Layout_mode.htm#manual_placement) объектов.


## Решение для веб-приложения


	- Откройте для редактирования файл [PP.xml](Setup.chm::/UiWebSetup/03_Setup_Web/PP_config_Java.htm).


	- В файл [PP.xml](Setup.chm::/UiWebSetup/03_Setup_Web/PP_config_Java.htm)
	 в раздел <pp> добавьте
	 следующий код:


<modules>
    <dashboard>
        <ReportBox AutoLayoutByDefault="false" />
    </dashboard>
</modules>


	- Сохраните изменения и закройте файл [PP.xml](Setup.chm::/UiWebSetup/03_Setup_Web/PP_config_Java.htm).


	- Перезапустите BI-сервер и серверную часть веб-приложения.


### Пример файла PP.xml, содержащего настройки для создания аналитической
 панели в режиме ручного размещения объектов


<pp>
    <service url="http://localhost/FPBI_App_v10.x/axis2/services/PP.SOM.Som"/>
    <metabase id="FPRepository"/>
    <cultures>
        <culture cultureName="ru"/>
        <culture cultureName="en"/>
    </cultures>
    <Modules>
        <Dashboard>
            <ReportBox AutoLayoutByDefault="false"/>
        </Dashboard>
    </Modules>
</pp>

## Решение для настольного приложения


В настольном приложении для отключения режима автоматического размещения
 объектов пользователь должен иметь права на редактирование системного
 реестра.


	- Откройте реестр для редактирования.


	- Перейдите в ветку реестра [HKEY_CURRENT_USER\SOFTWARE\Foresight\Foresight Analytics Platform\10.0\Dashboard].


	- Найдите параметр AutoLayout
	 типа REG_DWORD. Если параметр отсутствует, то создайте его.


	- Присвойте параметру AutoLayout
	 значение 0.


	- Закройте реестр, сохранив изменения.
