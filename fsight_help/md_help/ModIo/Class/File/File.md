# File


## Описание


Класс File предназначен для работы с файлами.


## Комментарии


Все методы данного класса являются статическими. Используя данные методы можно создавать, удалять, перемещать и изменять файлы, а также работать с атрибутами файлов и параметрами, отвечающими за время создания, изменения и последнего доступа к файлу. Работа с файлами может осуществляться в текстовом и двоичном виде. Если предполагается выполнение множества действий над файлами, то рекомендуется использовать соответствующие методы класса [FileInfo](../FileInfo/FileInfo.htm).
