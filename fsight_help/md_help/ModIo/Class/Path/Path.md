# Path


## Описание


Класс Path предназначен для
 работы с символьными переменными содержащими сведения о пути к файлу или
 папке.


## Статические методы класса, унаследованные от [IPath](../../Interface/IPath/IPath.htm)


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ChangeExtension](../../Interface/IPath/IPath.ChangeExtension.htm)


		 Метод ChangeExtension
		 возвращает строку, содержащую измененные сведения пути.


		 ![](../../Sub_Image.gif)
		 [Combine](../../Interface/IPath/IPath.Combine.htm)


		 Метод Combine осуществляет
		 объединение двух строк пути.


		 ![](../../Sub_Image.gif)
		 [GetDirectoryName](../../Interface/IPath/IPath.GetDirectoryName.htm)


		 Метод GetDirectoryName
		 возвращает для указанного пути сведения о папке.


		 ![](../../Sub_Image.gif)
		 [GetExtension](../../Interface/IPath/IPath.GetExtension.htm)


		 Метод GetExtension
		 возвращает расширение указанной строки пути.


		 ![](../../Sub_Image.gif)
		 [GetFileName](../../Interface/IPath/IPath.GetFileName.htm)


		 Метод GetFileName возвращает
		 имя файла и расширение указанной строки пути.


		 ![](../../Sub_Image.gif)
		 [GetFileNameWithoutExtension](../../Interface/IPath/IPath.GetFileNameWithoutExtension.htm)


		 Метод GetFileNameWithoutExtension
		 возвращает имя файла указанной строки пути без расширения.


		 ![](../../Sub_Image.gif)
		 [GetFullPath](../../Interface/IPath/IPath.GetFullPath.htm)


		 Метод GetFullPath возвращает
		 для указанной строки пути абсолютный путь.


		 ![](../../Sub_Image.gif)
		 [GetPathRoot](../../Interface/IPath/IPath.GetPathRoot.htm)


		 Метод GetPathRoot возвращает
		 путь к корневой папке.


		 ![](../../Sub_Image.gif)
		 [GetTempFileName](../../Interface/IPath/IPath.GetTempFileName.htm)


		 Метод GetTempFileName
		 создает временный файл с уникальным именем и возвращает абсолютный
		 путь к нему.


		 ![](../../Sub_Image.gif)
		 [GetTempPath](../../Interface/IPath/IPath.GetTempPath.htm)


		 Метод GetTempPath возвращает
		 путь временной папки для текущего пользователя системы.


		 ![](../../Sub_Image.gif)
		 [HasExtension](../../Interface/IPath/IPath.HasExtension.htm)


		 Метод HasExtension
		 возвращает True, если
		 путь включает расширение имени файла.


		 ![](../../Sub_Image.gif)
		 [IsPathRooted](../../Interface/IPath/IPath.IsPathRooted.htm)


		 Метод IsPathRooted
		 возвращает признак наличия в указанной строке абсолютного пути
		 к файлу.
