# TextReader


## Описание


Класс TextReader реализует объект,
 позволяющий считывать данные в текстовом формате.


## Конструкторы


		 Имя конструктора
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Create](TextReader.Create.htm)


		 Конструктор Create
		 создает объект, позволяющий считывать данные в текстовом формате.


## Свойства объекта класса, унаследованные от [ITextReader](../../Interface/ITextReader/ITextReader.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Encoding](../../Interface/ITextReader/ITextReader.Encoding.htm)


		 Свойство Encoding определяет
		 кодировку текста для чтения из файла.


		 ![](../../Property_Image.gif)
		 [Eof](../../Interface/ITextReader/ITextReader.Eof.htm)


		 Свойство Eof возвращает
		 признак окончания текста в файле.


		 ![](../../Property_Image.gif)
		 [WordDelimiters](../../Interface/ITextReader/ITextReader.WordDelimiters.htm)

		 определяет разделитель между словами при чтении текстового файла.
