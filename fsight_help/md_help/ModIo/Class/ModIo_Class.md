		 чтения при попытке выйти за пределы потока.

		 [FileNotFoundException](FileNotFoundException/FileNotFoundException.htm)
		 Класс FileNotFoundException
		 реализует экземпляр исключительной ситуации, возникающей при попытке
		 доступа к файлу, отсутствующему на диске.


		 ![](../Class_Image.gif)
		 [FileStream](FileStream/FileStream.htm)
		 Класс FileStream реализует
		 объект, представляющий из себя поток, связанный с файлом.

		 [IOException](IOException/IOException.htm)
		 Класс IOException реализует

		 [PathTooLongException](PathTooLongException/PathTooLongException.htm)

		 системой.
