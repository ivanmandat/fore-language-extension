# Directory


## Описание


Класс Directory предназначен для работы с каталогами файловой системы ПК.


## Комментарии


Все методы данного класса являются статическими. Используя данные методы можно создавать, удалять, перемещать, получать и изменять информацию о каталогах. Если предполагается выполнение множества действий над каталогами, то рекомендуется использовать соответствующие методы класса [DirectoryInfo](../DirectoryInfo/DirectoryInfo.htm).
