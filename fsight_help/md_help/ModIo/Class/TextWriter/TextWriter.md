# TextWriter


## Описание


Класс TextWriter реализует объект,
 позволяющий записывать данные в текстовом формате.


## Конструкторы

		 [Create](TextWriter.Create.htm)


		 Конструктор Create
		 создает объект, позволяющий записывать данные в текстовом формате.
