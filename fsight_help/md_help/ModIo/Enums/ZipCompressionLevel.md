# ZipCompressionLevel


## Описание


Перечисление ZipCompressionLevel
 содержит возможные степени сжатия ZIP-архива.


Используется следующими свойствами и методами:


	- [IZipArchive.CompressionLevel](../Interface/IZipArchive/IZipArchive.CompressionLevel.htm);


	- [ZipArchive.Create](../Class/ZipArchive/ZipArchive.Create.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 -1
		 DefaultCompression.
		 Средняя скорость и степень сжатия.


		 0
		 NoCompression. Без
		 сжатия файлов.


		 1
		 BestSpeed. Быстрое
		 сжатие, степень сжатия будет не оптимальной.


		 9
		 BestCompression. Лучшее
		 сжатие, может потребоваться больше времени на завершение операции
		 сжатия.
