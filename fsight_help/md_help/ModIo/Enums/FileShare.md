# FileShare


## Описание


Перечисление FileShare содержит возможности совместного доступа к открытому файлу.


Используется следующими свойствами и методами:


-
[IFile.Open](../Interface/IFile/IFile.Open.htm)


-
[IFileInfo.Open](../Interface/IFileInfo/IFileInfo.Open.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Exclusive. Эксклюзивная работа с файлом.


 1
 DenyWrite. Запрет записи в файл.


 2
 DenyRead. Запрет чтения из файла.


 3
 DenyNone. Общий доступ.
