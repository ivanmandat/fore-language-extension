# FileOpenMode


## Описание


Перечисление FileOpenMode содержит виды запроса на открытие файла.


Используется следующими свойствами и методами:


-
[IFile.Open](../Interface/IFile/IFile.Open.htm)


-
[IFileInfo.Open](../Interface/IFileInfo/IFileInfo.Open.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Create. Создание файла.


 1
 Read. Открытие файла для чтения.


 2
 Write. Открытие файла для записи.


 3
 ReadWrite. Открытие файла для чтения/записи.
