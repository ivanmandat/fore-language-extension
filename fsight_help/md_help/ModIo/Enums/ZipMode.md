# ZipMode


## Описание


Перечисление ZipMode содержит
 режимы инициализации нового объекта для работы с ZIP-архивами.


Используется следующими свойствами и методами:


	- [ZipArchive.Create](../Class/ZipArchive/ZipArchive.Create.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Create. Создание нового
		 ZIP-архива.


		 1
		 Read. Открытие существующего
		 ZIP-архива с возможностью только просмотра содержимого.


		 2
		 Update. Открытие существующего
		 ZIP-архива с возможностью просмотра и изменения содержимого.
