# SeekOrigin


## Описание


Перечисление SeekOrigin содержит точки потока, используемые для получения нового положения.


Используется следующим методом:


-
[IIOStream.Seek](../Interface/IIOStream/IIOStream.Seek.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Beginning. Смещение указывается относительно начала потока.


 1
 Current. Смещение указывается относительно текущей позиции.


 2
 FromEnd. Смещение указывается относительно конца потока.
