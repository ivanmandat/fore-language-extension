# IPath


## Описание


Интерфейс IPath содержит статические
 методы класса [Path](../../Class/Path/Path.htm).


## Иерархия наследования


IPath


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ChangeExtension](IPath.ChangeExtension.htm)


		 Метод ChangeExtension
		 возвращает строку, содержащую измененные сведения пути.


		 ![](../../Sub_Image.gif)
		 [Combine](IPath.Combine.htm)


		 Метод Combine осуществляет
		 объединение двух строк пути.


		 ![](../../Sub_Image.gif)
		 [GetDirectoryName](IPath.GetDirectoryName.htm)


		 Метод GetDirectoryName
		 возвращает для указанного пути сведения о папке.


		 ![](../../Sub_Image.gif)
		 [GetExtension](IPath.GetExtension.htm)


		 Метод GetExtension
		 возвращает расширение указанной строки пути.


		 ![](../../Sub_Image.gif)
		 [GetFileName](IPath.GetFileName.htm)


		 Метод GetFileName возвращает
		 имя файла и расширение указанной строки пути.


		 ![](../../Sub_Image.gif)
		 [GetFileNameWithoutExtension](IPath.GetFileNameWithoutExtension.htm)


		 Метод GetFileNameWithoutExtension
		 возвращает имя файла указанной строки пути без расширения.


		 ![](../../Sub_Image.gif)
		 [GetFullPath](IPath.GetFullPath.htm)


		 Метод GetFullPath возвращает
		 для указанной строки пути абсолютный путь.


		 ![](../../Sub_Image.gif)
		 [GetPathRoot](IPath.GetPathRoot.htm)


		 Метод GetPathRoot возвращает
		 путь к корневой папке.


		 ![](../../Sub_Image.gif)
		 [GetTempFileName](IPath.GetTempFileName.htm)


		 Метод GetTempFileName
		 создает временный файл с уникальным именем и возвращает абсолютный
		 путь к нему.


		 ![](../../Sub_Image.gif)
		 [GetTempPath](IPath.GetTempPath.htm)


		 Метод GetTempPath возвращает
		 путь временной папки для текущего пользователя системы.


		 ![](../../Sub_Image.gif)
		 [HasExtension](IPath.HasExtension.htm)


		 Метод HasExtension
		 возвращает True, если
		 путь включает расширение имени файла.


		 ![](../../Sub_Image.gif)
		 [IsPathRooted](IPath.IsPathRooted.htm)


		 Метод IsPathRooted
		 возвращает признак наличия в указанной строке абсолютного пути
		 к файлу.
