# IDirectory.GetCurrentDirectory


## Синтаксис


GetCurrentDirectory: String;


## Описание


Метод GetCurrentDirectory возвращает путь к текущему рабочему каталогу приложения.


## Комментарии


Каталог, из которого было запущено приложение, может не совпадать с рабочим каталогом приложения. Для изменения рабочего каталога приложения используйте метод [SetCurrentDirectory](IDirectory.SetCurrentDirectory.htm).
