# IDirectory.Exists


## Синтаксис


Exists(Path: String): Boolean;


## Параметры


Path - путь к каталогу, наличие которого необходимо проверить.


## Описание


Метод Exists осуществляет проверку наличия указанного каталога.


## Комментарии


Метод возвращает значение True, если каталог существует, и False, если каталог отсутствует.
