# ITextWriter.WriteWord


## Синтаксис


WriteWord(Value: String);


## Параметры


Value. Символьное слово, которое необходимо записать.


## Описание


Метод WriteWord осуществляет запись символьных слов в текстовый файл.


## Комментарии


После каждого слова, записанного с помощью метода WriteWord, в текстовый файл также будет добавлен разделитель, указанный в свойстве [WordDelimeter](ITextWriter.WordDelimeter.htm).


## Пример


Пример использования приведен в описании свойства [ITextWriter.WordDelimeter](ITextWriter.WordDelimeter.htm).
