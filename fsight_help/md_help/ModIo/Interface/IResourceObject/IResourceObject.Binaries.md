# IResourceObject.Binaries


## Синтаксис


Binaries: [IResourceBinaries](../IResourceBinaries/IResourceBinaries.htm);


## Описание


Свойство Binaries возвращает
 коллекцию изображений редактора ресурсов.


## Комментарии


Для получения коллекции строк редактора ресурсов используйте свойство
 [IResourceObject.Strings](IResourceObject.Strings.htm).
