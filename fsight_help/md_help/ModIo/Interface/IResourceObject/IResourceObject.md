# IResourceObject


Сборка: IO;


## Описание


Интерфейс IResourceObject содержит
 свойства коллекций объектов редактора ресурсов.


## Иерархия наследования


           IResourceObject


## Комментарии


Объект репозитория «[Ресурсы](UiNav.chm::/Multilanguage/Resources.htm)»
 предназначен для создания мультиязычных систем.
