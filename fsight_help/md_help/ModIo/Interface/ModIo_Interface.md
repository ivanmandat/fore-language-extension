		 [IDirectory](IDirectory/IDirectory.htm)

		 [IDirectoryInfo](IDirectoryInfo/IDirectoryInfo.htm)


		 Интерфейс IDirectoryInfo
		 содержит свойства и методы для работы с характеристиками папок.


		 ![](../Interface_Image.gif)
		 [IDirectoryInfoList](IDirectoryInfoList/IDirectoryInfoList.htm)


		 Интерфейс IDirectoryInfoList
		 содержит свойства коллекции описаний папок.


		 ![](../Interface_Image.gif)
		 [IDirectoryNotFoundException](IDirectoryNotFoundException/IDirectoryNotFoundException.htm)


		 Интерфейс IDirectoryNotFoundException
		 содержит свойства исключительной ситуации, возникающей при попытке
		 доступа к директории, отсутствующей на диске.


		 ![](../Interface_Image.gif)
		 [IEndOfStreamException](IEndOfStreamException/IEndOfStreamException.htm)


		 Интерфейс IEndOfStreamException
		 содержит свойства исключительной ситуации, возникающей во время
		 чтения при попытке выйти за пределы потока.


		 ![](../Interface_Image.gif)
		 [IFile](IFile/IFile.htm)


		 Интерфейс IFile содержит
		 статические методы класса [File](../Class/File/File.htm).


		 ![](../Interface_Image.gif)
		 [IFileInfo](IFileInfo/IFileInfo.htm)


		 Интерфейс IFileInfo
		 содержит свойства и методы для работы с характеристиками файлов.


		 ![](../Interface_Image.gif)
		 [IFileInfoList](IFileInfoList/IFileInfoList.htm)


		 Интерфейс IFileInfoList
		 содержит свойства коллекции описаний файлов.


		 ![](../Interface_Image.gif)
		 [IFileNotFoundException](IFileNotFoundException/IFileNotFoundException.htm)


		 Интерфейс IFileNotFoundException
		 содержит свойства исключительной ситуации, возникающей при попытке
		 доступа к файлу, отсутствующему на диске.


		 ![](../Interface_Image.gif)
		 [IFileStream](IFileStream/IFileStream.htm)


		 Интерфейс IFileStream
		 содержит свойства и методы для работы с файлом в виде последовательности
		 байтов.


		 ![](../Interface_Image.gif)
		 [IFileSystemInfo](IFileSystemInfo/IFileSystemInfo.htm)


		 Интерфейс IFileSystemInfo
		 содержит свойства и методы для работы с системной информацией
		 файлов и папок.


		 ![](../Interface_Image.gif)
		 [IFileSystemInfoList](IFileSystemInfoList/IFileSystemInfoList.htm)


		 Интерфейс IFileSystemInfoList
		 содержит свойства для работы с системной информацией коллекции
		 файлов и папок.


		 ![](../Interface_Image.gif)
		 [IIOException](IIOException/IIOException.htm)


		 Интерфейс IIOException
		 содержит свойства исключительной ситуации, создаваемой при возникновении
		 ошибки ввода-вывода.


		 ![](../Interface_Image.gif)
		 [IIOStream](IIOStream/IIOStream.htm)


		 Интерфейс IIOStream
		 содержит свойства и методы для работы с потоком в виде последовательности
		 байтов.


		 ![](../Interface_Image.gif)
		 [IIOStreamClass](IIOStreamClass/IIOStreamClass.htm)


		 Интерфейс IIOStreamClass
		 содержит статические методы для создания потока из переменных
		 типа Variant.


		 ![](../Interface_Image.gif)
		 [IMemoryStream](IMemoryStream/IMemoryStream.htm)


		 Интерфейс IMemoryStream
		 содержит свойства и методы для работы с потоком, организованным
		 в памяти ПК.


		 ![](../Interface_Image.gif)
		 [IPath](IPath/IPath.htm)


		 Интерфейс IPath содержит
		 статические методы класса [Path](../Class/Path/Path.htm).


		 ![](../Interface_Image.gif)
		 [IPathTooLongException](IPathTooLongException/IPathTooLongException.htm)


		 Интерфейс IPathTooLongException
		 содержит свойства исключительной ситуации, возникающей, когда
		 путь или имя файла больше, чем максимальная длина, определенная
		 системой.


		 ![](../Interface_Image.gif)
		 [IResourceBinaries](IResourceBinaries/IResourceBinaries.htm)


		 Интерфейс IResourceBinaries
		 содержит свойства и методы для работы с изображениями редактора
		 ресурсов.


		 ![](../Interface_Image.gif)
		 [IResourceItems](IResourceItems/IResourceItems.htm)


		 Интерфейс IResourceItems
		 является базовым и содержит свойства и методы для интерфейсов,
		 реализующих объекты редактора ресурсов.


		 ![](../Interface_Image.gif)
		 [IResourceLocale](IResourceLocale/IResourceLocale.htm)


		 Интерфейс IResourceLocale
		 содержит свойства для работы с языком редактора ресурсов.


		 ![](../Interface_Image.gif)
		 [IResourceLocales](IResourceLocales/IResourceLocales.htm)


		 Интерфейс IResourceLocales
		 содержит свойства и методы коллекции языков редактора ресурсов.


		 ![](../Interface_Image.gif)
		 [IResourceObject](IResourceObject/IResourceObject.htm)


		 Интерфейс IResourceObject
		 содержит свойства коллекций объектов редактора ресурсов.


		 ![](../Interface_Image.gif)
		 [IResourceStrings](IResourceStrings/IResourceStrings.htm)


		 Интерфейс IResourceStrings
		 предназначен для работы со строковыми элементами в редакторе ресурсов.

		 ![](../Interface_Image.gif)
		 [ITextWriter](ITextWriter/ITextWriter.htm)


		 Интерфейс ITextWriter
		 содержит свойства и методы для записи данных в текстовом формате.
