# IFileStream


## Описание


Интерфейс IFileStream содержит
 свойства и методы для работы с файлом в виде последовательности байтов.

## Методы, унаследованные от [IIOStream](../IIOStream/IIOStream.htm)


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CopyFrom](../IIOStream/IIOStream.CopyFrom.htm)


		 Метод CopyFrom осуществляет
		 копирование данных из указанного потока в текущий поток.


		 ![](../../Sub_Image.gif)
		 [ReadByte](../IIOStream/IIOStream.ReadByte.htm)


		 Метод ReadByte осуществляет

		 [Seek](../IIOStream/IIOStream.Seek.htm)


		 Метод Seek задает позицию

		 ![](../../Sub_Image.gif)
		 [WriteByte](../IIOStream/IIOStream.WriteByte.htm)


		 Метод WriteByte осуществляет
