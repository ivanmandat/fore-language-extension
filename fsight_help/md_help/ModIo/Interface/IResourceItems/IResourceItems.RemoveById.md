# IResourceItems.RemoveById


## Синтаксис


RemoveById(Id: String): Boolean;


## Параметры


Id. Идентификатор объекта редактора
 ресурсов.


## Описание


Метод RemoveById удаляет объект
 редактора ресурсов по заданному идентификатору.


## Комментарии


Если удаление прошло успешно, то метод возвращает значение True,
 в противном случае - False.
