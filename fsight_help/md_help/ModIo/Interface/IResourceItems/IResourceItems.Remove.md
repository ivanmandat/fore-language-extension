# IResourceItems.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс объекта редактора
 ресурсов.


## Описание


Метод Remove удаляет объект
 редактора ресурсов по заданному индексу.


## Комментарии


Если удаление прошло успешно, то метод возвращает значение True,
 в противном случае - False.
