# IResourceItems.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 строк/изображений в редакторе ресурсов.


## Комментарии


Для удаления всех строк/изображений из ресурсов используйте метод [IResourceItems.Clear](IResourceItems.Clear.htm).
