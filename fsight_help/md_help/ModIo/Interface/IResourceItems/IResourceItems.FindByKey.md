# IResourceItems.FindByKey


## Синтаксис


FindByKey(Key: Integer): Integer;


## Параметры


Key. Ключ объекта редактора
 ресурсов.


## Описание


Метод FindByKey выполняет
 поиск индекса объекта редактора ресурсов по указанному ключу объекта.


## Комментарии


Если объект с указанным ключом не найден, то метод возвращает значение
 «-1».


Для получения ключа объекта по его идентификатору используйте свойство
 [IResourceItems.Key](IResourceItems.Key.htm).
