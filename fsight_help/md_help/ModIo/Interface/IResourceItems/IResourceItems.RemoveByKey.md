# IResourceItems.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ объекта редактора
 ресурсов.


## Описание


Метод RemoveByKey удаляет объект
 редактора ресурсов по заданному ключу.


## Комментарии


Если удаление прошло успешно, то метод возвращает значение True,
 в противном случае - False.
