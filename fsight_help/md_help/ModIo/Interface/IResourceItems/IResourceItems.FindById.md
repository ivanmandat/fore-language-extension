# IResourceItems.FindById


## Синтаксис


FindById(Id: String): Integer;


## Параметры


Id. Идентификатор объекта редактора
 ресурсов.


## Описание


Метод FindById выполняет поиск
 ключа объекта редактора ресурсов по указанному идентификатору объекта.


## Комментарии


Если объект с указанным идентификатором не найден, то метод возвращает
 значение «-1».


Для получения идентификатора объекта по его ключу используйте свойство
 [IResourceItems.Id](IResourceItems.Id.htm).
