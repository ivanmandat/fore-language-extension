# IResourceItems.Clear


## Синтаксис


Clear;


## Описание


Метод Clear удаляет все объекты
 редактора ресурсов.


## Комментарии


После выполнения метода свойство [IResourceLocales.Count](../IResourceLocales/IResourceLocales.Count.htm)
 принимает значение «0».
