# IResourceItems.Add


## Синтаксис


Add([Key: Integer = 0]; [Id: String = ""]):
 Integer;


## Параметры


Key. Ключ добавляемого объекта;


Id. Идентификатор добавляемого
 объекта.


## Описание


Метод Add добавляет объект в
 редактор ресурсов.


## Комментарии


В результате выполнения метод возвращает ключ добавленного объекта.
