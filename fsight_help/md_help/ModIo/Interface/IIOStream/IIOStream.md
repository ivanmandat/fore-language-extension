# IIOStream


## Описание


Интерфейс IIOStream содержит
 свойства и методы для работы с потоком в виде последовательности байтов.


## Иерархия наследования


           [IIOStreamSys](ForeSys.chm::/Interface/IIOStreamSys/IIOStreamSys.htm)


           IIOStream


## Комментарии


Интерфейс является базовым для следующих интерфейсов:


	- [IFileStream](../IFileStream/IFileStream.htm);


	- [IMemoryStream](../IMemoryStream/IMemoryStream.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Position](IIOStream.Position.htm)


		 Свойство Position определяет
		 позицию в потоке.


		 ![](../../Property_Image.gif)
		 [Size](IIOStream.Size.htm)


		 Свойство Size определяет
		 размер потока в байтах.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CopyFrom](IIOStream.CopyFrom.htm)


		 Метод CopyFrom осуществляет
		 копирование данных из указанного потока в текущий поток.


		 ![](../../Sub_Image.gif)
		 [ReadByte](IIOStream.ReadByte.htm)


		 Метод ReadByte осуществляет
		 считывание байта данных из потока.


		 ![](../../Sub_Image.gif)
		 [Seek](IIOStream.Seek.htm)


		 Метод Seek задает позицию
		 в потоке с учетом параметров смещения и позиционирования потока.


		 ![](../../Sub_Image.gif)
		 [WriteByte](IIOStream.WriteByte.htm)


		 Метод WriteByte осуществляет
		 запись байта данных в поток.
