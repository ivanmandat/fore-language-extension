# IZipArchiveEntry.UnpackFileToStream


## Синтаксис


UnpackFileToStream(Stream: [IIOStream](../IIOStream/IIOStream.htm));


## Параметры


Stream. Поток, в который будет
 распакован текущий элемент.


## Описание


Метод UnpackFileToStream распаковывает
 текущий элемент в указанный поток.


## Комментарии


Метод актуален для использования, если текущий элемент соответствует
 файлу.
