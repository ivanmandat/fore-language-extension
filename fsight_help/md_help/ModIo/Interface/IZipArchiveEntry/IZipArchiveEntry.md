# IZipArchiveEntry


Сборка: IO;


## Описание


Интерфейс IZipArchiveEntry содержит
 свойства и методы для работы с элементом ZIP-архива.


## Иерархия наследования


IZipArchiveEntry


## Комментарии


Элемент ZIP-архива можно получить, используя различные свойства и методы
 интерфейса [IZipArchive](../IZipArchive/IZipArchive.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ArchPath](IZipArchiveEntry.ArchPath.htm)
		 Свойство ArchPath возвращает
		 путь до элемента внутри архива.


		 ![](../../Property_Image.gif)
		 [ModificationDate](IZipArchiveEntry.ModificationDate.htm)
		 Свойство ModificationDate
		 возвращает дату и время последней модификации элемента в архиве.


		 ![](../../Property_Image.gif)
		 [Owner](IZipArchiveEntry.Owner.htm)
		 Свойство Owner возвращает
		 архив, в котором создан и хранится текущий элемент.


		 ![](../../Property_Image.gif)
		 [Size](IZipArchiveEntry.Size.htm)
		 Свойство Size возвращает
		 исходный размер файла, которому соответствует текущий элемент
		 архива.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Remove](IZipArchiveEntry.Remove.htm)
		 Метод Remove удаляет
		 текущий элемент из архива.


		 ![](../../Sub_Image.gif)
		 [Rename](IZipArchiveEntry.Rename.htm)
		 Метод Rename переименовывает
		 файл/папку внутри архива.


		 ![](../../Sub_Image.gif)
		 [UnpackFile](IZipArchiveEntry.UnpackFile.htm)
		 Метод UnpackFile распаковывает
		 текущий элемент в указанную папку на диске.


		 ![](../../Sub_Image.gif)
		 [UnpackFileToStream](IZipArchiveEntry.UnpackFileToStream.htm)
		 Метод UnpackFileToStream
		 распаковывает текущий элемент в указанный поток.
