# IZipArchiveEntry.Size


## Синтаксис


Size: Integer;


## Описание


Свойство Size возвращает исходный
 размер файла, которому соответствует текущий элемент архива.


## Комментарии


Размер возвращается в байтах. Свойство актуально только для файлов.
 Если текущий элемент соответствует папке, то свойство возвращает значение
 «0».
