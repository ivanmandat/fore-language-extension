# IResourceLocales.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ языка в ресурсах.


## Описание


Метод RemoveByKey удаляет
 язык с указанным ключом.


## Комментарии


Метод возвращает значение True,
 если удаление прошло успешно, в противном случае - False.
