# IResourceLocales.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс языка в ресурсах.


## Описание


Метод Remove удаляет язык
 с указанным индексом.


## Комментарии


Индексация языков в ресурсах сквозная и начинается с нуля. Для получения
 количества языков в ресурсах используйте свойство [IResourceLocales.Count](IResourceLocales.Count.htm).


Метод возвращает значение True,
 если удаление прошло успешно, в противном случае - False.
