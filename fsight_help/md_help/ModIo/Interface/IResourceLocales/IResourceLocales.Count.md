# IResourceLocales.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 доступных языков в редакторе ресурсов.


## Комментарии


Для получения языка из ресурсов используйте свойство [IResourceLocales.Item](IResourceLocales.Item.htm).
