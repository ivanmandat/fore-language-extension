# IResourceLocales.Clear


## Синтаксис


Clear;


## Описание


Метод Clear удаляет все языки
 из редактора ресурсов.


## Комментарии


После удаления всех языков в ресурсы автоматически будет добавлен язык
 по умолчанию.
