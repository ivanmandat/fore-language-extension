# IResourceLocales.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IResourceLocale](../IResourceLocale/IResourceLocale.htm);


## Параметры


Key. Ключ искомого языка.


## Описание


Метод FindByKey выполняет
 поиск языка ресурсов по заданному ключу.


## Комментарии


Для указания ключа языка используйте значения перечисления [LocaleCodeID](ForeSys.chm::/Enums/LocaleCodeID.htm).


## Пример


Использование метода приведено в примере для [IResourceLocales.Add](IResourceLocales.Add.htm).
