# IResourceLocale.Strings


## Синтаксис


Strings: [IResourceStrings](../IResourceStrings/IResourceStrings.htm);


## Описание


Свойство Strings возвращает
 коллекцию строк редактора ресурсов для заданного языка.


## Комментарии


Для получения коллекции изображений для заданного языка используйте
 свойство [IResourceLocale.Binaries](IResourceLocale.Binaries.htm).
