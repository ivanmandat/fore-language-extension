# IResourceLocale.Binaries


## Синтаксис


Binaries: [IResourceBinaries](../IResourceBinaries/IResourceBinaries.htm);


## Описание


Свойство Binaries возвращает
 коллекцию изображений редактора ресурсов для заданного языка.


## Комментарии


Для получения коллекции строк для заданного языка используйте свойство
 [IResourceLocale.Strings](IResourceLocale.Strings.htm).
