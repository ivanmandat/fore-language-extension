# IResourceLocale


Сборка: IO;


## Описание


Интерфейс IResourceLocale содержит
 свойства для работы с языком редактора ресурсов.


## Иерархия наследования


           IResourceLocale


## Комментарии


Для работы с коллекций языков используйте интерфейс [IResourceLocales](../IResourceLocales/IResourceLocales.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Binaries](IResourceLocale.Binaries.htm)


		 Свойство Binaries возвращает
		 коллекцию изображений редактора ресурсов для заданного языка.


		 ![](../../Property_Image.gif)
		 [Id](IResourceLocale.Id.htm)


		 Свойство Id возвращает
		 идентификатор языка редактора ресурсов.


		 ![](../../Property_Image.gif)
		 [IsCurrent](IResourceLocale.IsCurrent.htm)


		 Свойство IsCurrent
		 определяет является ли язык текущим языком перевода.


		 ![](../../Property_Image.gif)
		 [IsDefault](IResourceLocale.IsDefault.htm)


		 Свойство IsDefault
		 определяет является ли данных язык языком ресурсов по умолчанию.


		 ![](../../Property_Image.gif)
		 [Key](IResourceLocale.Key.htm)


		 Свойство Key возвращает
		 ключ языка редактора ресурсов.


		 ![](../../Property_Image.gif)
		 [Strings](IResourceLocale.Strings.htm)


		 Свойство Strings возвращает
		 коллекцию строк редактора ресурсов для заданного языка.
