# IResourceLocale.Id


## Синтаксис


Id: String;


## Описание


Свойство Id возвращает идентификатор
 языка редактора ресурсов.


## Комментарии


Для получения ключа языка используйте свойство [IResourceLocale.Key](IResourceLocale.Key.htm).
