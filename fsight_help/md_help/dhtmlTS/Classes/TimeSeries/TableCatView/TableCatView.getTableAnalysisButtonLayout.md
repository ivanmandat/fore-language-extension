# TableCatView.getTableAnalysisButtonLayout


## Синтаксис


getTableAnalysisButtonLayout();


## Описание


Метод getTableAnalysisButtonLayout
 возвращает кнопку «Разметка» на ленте инструментов рабочей книги.


## Комментарии


Метод возвращает объект типа [PP.Ui.RibbonButton](dhtmlRibbon.chm::/Classes/RibbonButton/RibbonButton.htm).
