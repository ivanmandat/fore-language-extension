# CustomAnalysisWizard.setHierarchy


## Синтаксис


setHierarchy(value: [PP.TS.MetaHierarchy](../MetaHierarchy/MetaHierarchy.htm));


## Параметры


value. Иерархия метабазы.


## Описание


Метод setHierarchy** устанавливает иерархию метабазы в редакторе выражения рабочей книги.
