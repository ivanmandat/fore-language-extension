# RollbackRevisionDlg.RubName


## Синтаксис


RubName: String


## Описание


Свойство RubName определяет наименование каталога показателей.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setRubName, а возвращается - с помощью метода getRubName. Из JSON значение установить нельзя.
