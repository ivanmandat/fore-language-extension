# WbkValidationTree.RequestMetadata


## Синтаксис


RequestMetadata(sender: Object, args);


## Описание


Событие RequestMetadata** генерируется при отсутствии метаданных в источнике дерева валидации.


## Параметры


sender. Источник события;


args. Информация о событии.
