# Конструктор RevisionsPanel


## Синтаксис


PP.TS.Ui.RevisionsPanel (settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор RevisionsPanel создает экземпляр класса [RevisionsPanel](RevisionsPanel.htm).
