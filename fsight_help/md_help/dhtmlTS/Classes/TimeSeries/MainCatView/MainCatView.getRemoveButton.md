# MainCatView.getRemoveButton


## Синтаксис


getRemoveButton();


## Описание


Метод getRemoveButton возвращает кнопку «Удалить».


## Комментарии


Метод возвращает объект типа PP.Ui.[RibbonButton](dhtmlRibbon.chm::/Classes/RibbonButton/RibbonButton.htm).
