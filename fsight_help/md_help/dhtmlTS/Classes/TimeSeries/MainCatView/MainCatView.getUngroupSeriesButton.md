# MainCatView.getUngroupSeriesButton


## Синтаксис


getUngroupSeriesButton();


## Описание


Метод getUngroupSeriesButton возвращает кнопку «Разгруппировать».


## Комментарии


Метод возвращает объект типа PP.Ui.[RibbonButton](dhtmlRibbon.chm::/Classes/RibbonButton/RibbonButton.htm).
