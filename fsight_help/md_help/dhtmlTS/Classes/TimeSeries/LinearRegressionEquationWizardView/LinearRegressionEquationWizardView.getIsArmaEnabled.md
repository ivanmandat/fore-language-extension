# LinearRegressionEquationWizardView.getIsArmaEnabled


## Синтаксис


getIsArmaEnabled ();


## Описание


Метод getIsArmaEnabled определяет активность полей AR (авторегрессия) и MA (скользящее среднее).


## Комментарии


Метод возвращает true, если хотя бы одно из полей AR(авторегрессия) или MA(скользящее среднее) мастера настройки параметров линейной регрессии активировано, иначе возвращает false.
