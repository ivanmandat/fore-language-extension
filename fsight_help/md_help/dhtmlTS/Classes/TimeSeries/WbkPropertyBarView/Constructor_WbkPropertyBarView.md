# Конструктор WbkPropertyBarView


## Синтаксис


PP.TS.Ui.WbkPropertyBarView(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор WbkPropertyBarView создает экземпляр класса [WbkPropertyBarView](WbkPropertyBarView.htm).

Конструктор предназначен для внутреннего использования.
