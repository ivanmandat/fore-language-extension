# WbkPropertyBarView.deleteExtraRevisionPanels


## Синтаксис


deleteExtraRevisionPanels();


## Описание


Метод deleteExtraRevisionPanels удаляет из панели свойств вкладки, соответствующие источникам (базам данных временных рядов), не содержащимся в рабочей книге.
