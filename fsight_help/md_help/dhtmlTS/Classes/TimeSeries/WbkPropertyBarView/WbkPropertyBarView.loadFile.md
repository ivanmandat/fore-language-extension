# WbkPropertyBarView.loadFile


## Синтаксис


loadFile(fileName: String, callBack: PP.Delegate,
 arguments: Object);


## Параметры


fileName. Имя добавляемого
 файла;


callBack. Возвратная функция;


arguments. Аргументы возвратной
 функции.


## Описание


Метод loadFile загружает файл
 с указанным именем.


## Пример


Метод предназначен для внутреннего использования.
