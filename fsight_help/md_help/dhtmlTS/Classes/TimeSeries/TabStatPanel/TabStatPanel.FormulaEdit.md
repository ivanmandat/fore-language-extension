# TabStatPanel.FormulaEdit


## Синтаксис


FormulaEdit: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие FormulaEdit наступает при нажатии на кнопку «Редактировать» на панели статистик.
