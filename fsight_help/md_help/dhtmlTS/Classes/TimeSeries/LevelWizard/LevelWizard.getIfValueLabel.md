# LevelWizard.getIfValueLabel


## Синтаксис


getIfValueLabel();


## Описание


Метод getIfValueLabel**
 возвращает текст раскрывающегося списка для выбора условия сравнения данных
 с числом в мастере валидации.


## Комментарии


Метод возвращает значение типа [PP.Ui.Label](dhtmlUi.chm::/Classes/Label/Label.htm).
