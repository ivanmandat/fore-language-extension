# WbkDataView.refreshView


## Синтаксис


refreshView(args);


## Параметры


args. Параметр, определяющий,
 что нужно обновить.


## Описание


Метод refreshView обновляет
 указанные элементы рабочей книги.


## Комментарии


Метод refreshView является копией
 метода [WbkDataView.refresh](WbkDataView.refresh.htm).
