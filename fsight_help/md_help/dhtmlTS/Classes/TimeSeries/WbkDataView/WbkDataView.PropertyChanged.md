# WbkDataView.PropertyChanged


## Синтаксис


PropertyChanged: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие PropertyChanged наступает после изменения параметра рабочей книги.


## Пример


Для выполнения примера предполагается наличие на странице компонента [WorkbookBox](../../../Components/TimeSeries/WorkbookBox/WorkbookBox.htm) с наименованием «workbookBox» (см. «[Пример создания компонента WorkbookBox](../../../Components/TimeSeries/WorkbookBox/Component_WorkbookBox.htm)»), наличие в рабочей книге ряда с типом формулы «Нелинейная регрессия». Затем необходимо в обработчике события открытия документа добавить следующий код:


	workbookBox.getDataView().PropertyChanged.add(PP.Delegate(function(){
		alert("Параметры рабочей книги изменены");
	}));

Чтобы сгенерировать данное событие, необходимо нажать на кнопку «Редактировать» на панели статистик. За тем в открывшемся диалоге добавления нелинейной регрессии нажать на кнопку «Ок», при необходимости, предварительно отредактировав формулу. Чтобы кнопка «Редактировать» на панели статистик была доступна, необходимо в рабочей книге выделить ряд с типом формулы «Нелинейная регрессия». После этого будет сгенерировано событие PropertyChanged и будет выведено сообщение: «Параметры рабочей книги изменены».
