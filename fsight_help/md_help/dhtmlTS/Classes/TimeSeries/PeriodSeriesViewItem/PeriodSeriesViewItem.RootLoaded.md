# PeriodSeriesViewItem.RootLoaded


## Синтаксис


RootLoaded(sender: Object, args: Object);


## Параметры


sender. Источник события;


args. Информация о событии. Доступный аргумент: JSON-объект с полем MetaAttrId - идентификатор атрибута ряда.


## Описание


Событие RootLoaded** наступает при загрузке временных рядов в источник панели «Ряды» на панели свойств рабочей книги.
