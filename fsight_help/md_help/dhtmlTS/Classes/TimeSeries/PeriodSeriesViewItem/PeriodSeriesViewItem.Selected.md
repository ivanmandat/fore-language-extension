# PeriodSeriesViewItem.Selected


## Синтаксис


Selected: Boolean


## Описание


Свойство Selected** устанавливает и возвращает признак того, что атрибут «Ряды» отмечен.


## Комментарии


Значение свойства устанавливается с помощью метода setSelected, а возвращается - с помощью метода getSelected. Из JSON значение задать нельзя.


Если свойству указывается значение true, то заголовок панели «Ряды» будет отмечен, иначе не будет.


## Пример


В настоящее время данное свойство не реализовано.
