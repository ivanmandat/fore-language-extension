# LanerBoxView.setNeedUpdateSelection


## Синтаксис


setNeedUpdateSelection (value);


## Параметры


value. Определяет, будет ли обновляться отметка после загрузки таблицы. Если установлено true, будет, иначе - нет.


## Описание


Метод setNeedUpdateSelection определяет, будет ли обновляться отметка после загрузки таблицы.
