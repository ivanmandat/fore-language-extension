# LanerBoxView.removeSeries


## Синтаксис


removeSeries (series, meta, seriesTabSheetIndexes);


## Параметры


series. Ряды;


meta. Ответ сервиса;


seriesTabSheetIndexes. Массив индексов, удаляемых с электронной таблицы строк (столбцов в случае повёрнутой таблицы).


## Описание


Метод removeSeries удаляет ряды.
