# LanerBoxView.adjustSeriesSize


## Синтаксис


adjustSeriesSize (tabSheetStricture, series);


## Параметры


tabSheetStricture. Задает структуру вкладки;


series. Задает ряды, ширину и высоту которых надо подогнать.


## Описание


Метод adjustSeriesSize подгоняет ширину и высоту рядов согласно структуре вкладки.


## Комментарии


Метод не используется, необходим для внутреннего использования.
