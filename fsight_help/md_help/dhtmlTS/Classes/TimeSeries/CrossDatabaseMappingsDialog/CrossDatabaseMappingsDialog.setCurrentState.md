# CrossDatabaseMappingsDialog.setCurrentState


## Синтаксис


setCurrentState(state: Object);


## Параметры


state. JSON-объект со свойством comparatorRubKey, определяющим ключ эталонной базы данных.


## Описание


Метод setCurrentState** устанавливает в диалоге эталонную базу данных временных рядов по её ключу.
