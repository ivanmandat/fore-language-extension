# Конструктор AddAggregationDialog


## Синтаксис


PP.TS.Ui.AddAggregationDialog(settings);


## Параметры


settings. JSON-объект со значениями свойств класса.


## Описание


Конструктор AddAggregationDialog создает экземпляр класса [AddAggregationDialog](AddAggregationDialog.htm).
