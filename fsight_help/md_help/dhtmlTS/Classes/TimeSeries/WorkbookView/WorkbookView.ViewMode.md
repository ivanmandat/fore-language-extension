# WorkbookView.ViewMode


## Синтаксис


ViewMode: Boolean;


## Описание


Свойство ViewMode определяет, находится ли рабочая книга в режиме редактирования.


## Комментарии:


Если установлено значение true, рабочая книга в режиме просмотра, false - в режиме редактирования.


## Пример


Для выполнения примера предполагается наличие на странице компонента [WorkbookBox](../../../Components/TimeSeries/WorkbookBox/WorkbookBox.htm) с наименованием «workbookBox» (см. «[Пример создания компонента WorkbookBox](../../../Components/TimeSeries/WorkbookBox/Component_WorkbookBox.htm)»), также необходимо в обработчике события открытия документа добавить следующий код:


//определим, находится ли рабочая книга в режиме редактирования
if (workbookBox.getViewMode() == true)
	console.log("Книга в режиме просмотра");
else
	console.log("Книга в режиме редактирования");

После выполнения примера в консоль браузера будет выведено сообщение «Книга в режиме просмотра», если рабочая книга находится в режиме просмотра или «Книга в режиме редактирования», если она находится в режиме редактирования.
