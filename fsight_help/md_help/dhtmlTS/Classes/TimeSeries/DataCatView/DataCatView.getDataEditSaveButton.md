# DataCatView.getDataEditSaveButton


## Синтаксис


getDataEditSaveButton();


## Описание


Метод getDataEditSaveButton возвращает кнопку «Сохранить изменения».


## Комментарии


Метод возвращает объект типа PP.Ui.[RibbonButton](dhtmlRibbon.chm::/Classes/RibbonButton/RibbonButton.htm).
