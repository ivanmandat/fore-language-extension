# MetaAttributeBreadCrumbView.refreshAll


## Синтаксис


refreshAll (fireEvent);


## Параметры


fireEvent. Определяет, генерировать ли при вызове метода событие [RequestSelectedItemsChanged](MetaAttributeBreadCrumbView.RequestSelectedItemsChanged.htm). Если установлено значение true (по умолчанию), событие будет генерироваться, иначе – не будет. Необязательный параметр.


## Описание


Метод refreshAll является аналогом метода [refresh](MetaAttributeBreadCrumbView.refresh.htm).
