# MetaAttributeBreadCrumbView.ItemsCaptionType


## Синтаксис


ItemsCaptionType: [PP.TS.BreadcrumbItemsCaptionType](../../../Enums/BreadcrumbItemCaptionType.htm)


## Описание


Свойство ItemsCaptionType определяет
 тип надписи у элемента цепочки навигации в компоненте [MetaAttributeBreadCrumb](../../../Components/TimeSeries/MetaAttributeBreadCrumb/MetaAttributeBreadCrumb.htm).


## Комментарии


Значение свойства устанавливается из JSON.


По умолчанию свойство содержит значение PP.TS.BreadcrumbItemCaptionType.SelectedCount.
