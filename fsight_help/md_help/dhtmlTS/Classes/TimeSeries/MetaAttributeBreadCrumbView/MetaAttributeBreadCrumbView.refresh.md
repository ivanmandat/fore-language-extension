# MetaAttributeBreadCrumbView.refresh


## Синтаксис


refresh (args, fireEvent);


## Параметры


args. Параметры, определяющие, что нужно обновить;


fireEvent. Определяет, генерировать ли при вызове метода событие [RequestSelectedItemsChanged](MetaAttributeBreadCrumbView.RequestSelectedItemsChanged.htm). Если установлено значение true (по умолчанию), событие будет генерироваться, иначе – не будет.


## Описание


Метод refresh вычисляет пункты навигационной цепочки в соответствии с моделью или сохраненным состоянием [Options](MetaAttributeBreadCrumbView.Options.htm), если оно есть.
