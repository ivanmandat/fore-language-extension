# MetaAttribute.IsRTL


## Синтаксис


IsRTL: Boolean;


## Описание


Свойство lsRTL определяет направление письма справа налево.


## Комментарии


Если для свойства установлено значение false (по умолчанию), то атрибуты располагаются слева направо (по левому краю), если установлено значение true – справа налево (по правому краю).
