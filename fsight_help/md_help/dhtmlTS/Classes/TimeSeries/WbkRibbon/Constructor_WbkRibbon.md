# Конструктор WbkRibbon


## Синтаксис


PP.TS.UiWbkRibbon(settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор WbkRibbon создает экземпляр класса [WbkRibbon](WbkRibbon.htm).
