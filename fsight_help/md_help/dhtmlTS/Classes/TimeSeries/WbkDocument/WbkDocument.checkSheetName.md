# WbkDocument.checkSheetName


## Синтаксис


checkSheetName (newSheetName);


## Параметры


newSheetName. Задает наименование листа.


## Описание


Метод checkSheetName проверяет допустимость нового названия листа.


## Комментарии


Метод checkSheetName выполняет проверку уникальности имен, чтобы избежать повторяющихся. При выполнении проверки начальные и завершающие пробелы не учитываются, ограничений на символы, которые может содержать имя листа, нет. В случае успеха проверки метод возвращает true.


## Пример


Для выполнения примера необходимо создать сервис для работы с временными рядами, получить экземпляр рабочей книги из базы данных временных рядов (см. «[WbkDocument.ActiveSheetKey](WbkDocument.ActiveSheetKey.htm)») и в обработчике события открытия документа добавить следующий код:


	var isAcceptable = wbkDocument.checkSheetName("Лист 2");
	console.log("допустимость нового названия листа");
	console.log(isAcceptable);

После выполнения примера в консоль браузера будет выведен результат проверки на допустимость названия нового листа:


допустимость нового названия листа


true
