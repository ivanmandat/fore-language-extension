# TrendWizardView.getMethodType


## Синтаксис


getMethodType ();


## Описание


Метод getMethodType возвращает тип формулы расчета выбранного вычисляемого ряда.


## Комментарии


Метод возвращает элемент перечисления PP.TS.Ui.[FormulaTypeEnum](../../../Enums/FormulaTypeEnum.htm).
