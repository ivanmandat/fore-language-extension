# TablePanel.Mode


## Синтаксис


Mode: [PP.TS.Ui.WbkTablePanMode](../../../Enums/WbkTablePanMode.htm)


## Описание


Свойство Mode определяет режим отображения панели настроек таблицы в рабочей книге.


## Описание


Значение свойства может быть установлено из JSON и с помощью метода setMode, а возвращено - с помощью метода getMode.
