# WbkRibbonView.Expanded


## Синтаксис


Expanded: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие Expanded наступает при разворачивании ленты инструментов.


## Пример


Пример использования события Expanded показан на странице "[WbkRibbonView.Collapsed](WbkRibbonView.Collapsed.htm)"


После выполнения примеров будет раскрыта лента инструментов и на экран будет выведено сообщение: «Лента инструментов временных рядов раскрыта».
