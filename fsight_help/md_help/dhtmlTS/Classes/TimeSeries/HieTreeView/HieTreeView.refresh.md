# HieTreeView.refresh


## Синтаксис


refresh (args);


## Параметры


args. Задает параметры, определяющие, что нужно обновить.


## Описание


Метод refresh обновляет дерево.


## Пример


Для выполнения примера необходимо создать сервис для работы с временными рядами, создать компонент для отображения иерархии каталога показателей с наименованием «maTree» (см. «[Конструктор MetaAttributeTree](../MetaAttributeTree/Constructor_MetaAttributeTree.htm)») и в теге <SCRIPT> добавить следующий код:


	console.log("Обновляем дерево");
	maTree.refresh();

После выполнения примера будет осуществлено обновление дерева, а в консоль браузера будет выведено соответствующее сообщение.
