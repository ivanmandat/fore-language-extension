# HieTreeView.IsBusy


## Синтаксис


IsBusy: Boolean;


## Описание


Свойство IsBusy устанавливает признак того, что в дереве выполняется перезагрузка вершин.


## Комментарии


Свойство IsBusy принимает значение true, если перезагрузка вершин выполняется, false - не выполняется (по умолчанию).


## Пример


Для выполнения примера необходимо создать сервис для работы с временными рядами, создать компонент для отображения иерархии каталога показателей с наименованием «maTree» (см. «[Конструктор MetaAttributeTree](../MetaAttributeTree/Constructor_MetaAttributeTree.htm)») и в теге <SCRIPT> добавить следующий код:


	//Устанавливаем признак перезагрузки узлов
	maTree.setIsBusy(true);
	//Получаем признак того, что в дереве выполняется перезагрузка узлов
	console.log(maTree.getIsBusy());

После выполнения примера в консоль браузера будет выведен признак того, что в дереве выполняется перезагрузка вершин:


true
