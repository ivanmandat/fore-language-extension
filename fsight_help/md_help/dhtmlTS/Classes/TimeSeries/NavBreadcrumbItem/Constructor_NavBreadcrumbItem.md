# Конструктор NavBreadcrumbItem


## Синтаксис


PP.TS.Ui.NavBreadcrumbItem (settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор NavBreadcrumbItem создает экземпляр класса [NavBreadcrumbItem](NavBreadcrumbItem.htm).
