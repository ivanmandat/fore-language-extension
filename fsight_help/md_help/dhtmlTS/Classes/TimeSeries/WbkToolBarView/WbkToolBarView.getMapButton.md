# WbkToolBarView.getMapButton


## Синтаксис


getMapButton();


## Описание


Метод getMapButton возвращает
 кнопку «Карта» компонента [WbkToolBar](../WbkToolBar/WbkToolBar.htm).


## Комментарии


Метод возвращает объект типа PP.Ui.[ToolBarButton](dhtmlUi.chm::/Classes/ToolBarButton/ToolBarButton.htm).
