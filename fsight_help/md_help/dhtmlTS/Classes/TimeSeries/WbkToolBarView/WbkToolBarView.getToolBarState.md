# WbkToolBarView.getToolBarState


## Синтаксис


getToolBarState();


## Описание


Метод getToolBarState возвращает состояние компонента [WbkToolBar](../WbkToolBar/WbkToolBar.htm).


## Комментарии


Метод возвращает JSON-объект со следующими полями: chart - данные о состоянии представления «Диаграмма», grid - данные о состоянии представления «Таблица», map - данные о состоянии представления «Карта».
