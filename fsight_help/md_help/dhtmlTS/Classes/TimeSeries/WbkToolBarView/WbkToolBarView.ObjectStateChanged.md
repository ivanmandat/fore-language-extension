# WbkToolBarView.ObjectStateChanged


## Синтаксис


ObjectStateChanged: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии, представлена классом PP.Mb.Ui.[PropertyChangedEventArgs](dhtmlMetabase.chm::/Classes/Metabase/PropertyChangedEventArgs/PropertyChangedEventArgs.htm).


## Описание


Событие ObjectStateChanged наступает после изменения состояния представления рабочей книги.
