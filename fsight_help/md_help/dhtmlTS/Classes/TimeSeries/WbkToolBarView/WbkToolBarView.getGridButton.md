# WbkToolBarView.getGridButton


## Синтаксис


getGridButton();


## Описание


Метод getGridButton возвращает
 кнопку «Таблица» компонента [WbkToolBar](../WbkToolBar/WbkToolBar.htm).


## Комментарии


Метод возвращает объект типа PP.Ui.[ToolBarButton](dhtmlUi.chm::/Classes/ToolBarButton/ToolBarButton.htm).
