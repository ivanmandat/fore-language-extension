# MetaHierarchy.LeafLoadExtent


## Синтаксис


LeafLoadExtent: Number;


## Описание


Свойство LeafLoadExtent определяет максимальное количество листьевых элементов, которые будут загружены в иерархии.


## Комментарии


Под количеством листьевых элементов подразумевается максимальное количество уровней вложенности, которые будут подгружаться.


## Пример


Для выполнения примера необходимо создать сервис для работы с временными рядами, получить экземпляр рабочей книги из базы данных временных рядов (см. «[MetaHierarchy.ChildEls](MetaHierarchy.ChildEls.htm)») и в обработчике события открытия документа добавить следующий код:


console.log("Получаем максимальное количество листьевых элементов, которые будут загружены в иерархии");
var nMaxLeafs = hie.getLeafLoadExtent();
console.log(nMaxLeafs);

После выполнения примера в консоль браузера будет выведено максимальное количество листьевых элементов, которые будут загружены в иерархии:


65536
