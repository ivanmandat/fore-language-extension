# MetaHierarchy.setAttrProperty


## Синтаксис


setAttrProperty (attrId, propName, propValue);


## Параметры


attrId. Идентификатор атрибута;


propName. Имя свойства;


propValue. Значения свойства.


## Описание


Метод setAttrProperty устанавливает
 значение для свойства атрибута по его идентификатору.


## Пример


Для выполнения примера необходимо создать сервис для работы с временными
 рядами, получить экземпляр рабочей книги из базы данных временных рядов
 (см. «[MetaHierarchy.ChildEls](MetaHierarchy.ChildEls.htm)»),
 а также предполагается наличие атрибута с идентификатором «DL» и в обработчике
 события открытия документа добавить следующий код:


//Ищем на всех уровнях атрибут с идентификатором attrId и устанавливаем для его свойства c именем propName значение propValue
hie.setAttrProperty( "DL", "isLeaf", true);

После выполнения примера для атрибута с идентификатором «DL» для свойства
 isLeaf будет установлено значение true, т.е. указанный уровень будет являться
 листьевым.
