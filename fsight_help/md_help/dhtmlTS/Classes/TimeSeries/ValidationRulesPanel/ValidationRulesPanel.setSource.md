# ValidationRulesPanel.setSource


## Синтаксис


setSource(value: Object);


## Параметры


value. Источник данных.


## Описание


Метод setSource** устанавливает источник данных для панели правил валидации в рабочей книге.


## Комментарии


Источник данных для панели правил валидации также можно установить из JSON при создании экземпляра класса [PP.TS.Ui.ValidationRulesPanel](ValidationRulesPanel.htm).
