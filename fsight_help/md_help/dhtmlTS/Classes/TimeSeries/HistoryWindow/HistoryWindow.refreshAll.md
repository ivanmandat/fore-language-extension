# HistoryWindow.refreshAll


## Синтаксис


refreshAll ();


## Описание


Метод refreshAll обновляет компонент.


## Пример


Для выполнения предполагается наличие на странице компонента [HistoryWindow](../../../Components/TimeSeries/HistoryWindow/HistoryWindow.htm) с наименованием «historyWindow» (см. «[Конструктор HistoryWindow](Constructor_HistoryWindow.htm)») и в обработчике события открытия документа добавить следующий код:


	//Обновляем компонент
	historyWindow.refreshAll();

После выполнения примера компонент [HistoryWindow](../../../Components/TimeSeries/HistoryWindow/HistoryWindow.htm) будет обновлен.
