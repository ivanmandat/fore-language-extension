# SeriesAbstractCatView.getUngroupSeriesButton


## Синтаксис


getUngroupSeriesButton();


## Описание


Метод getUngroupSeriesButton**
 возвращает кнопку «Разгруппировать» ленты инструментов рабочей книги.


## Комментарии


Метод возвращает объект типа [PP.Ui.RibbonButton](dhtmlRibbon.chm::/Components/RibbonButton/RibbonButton.htm).
