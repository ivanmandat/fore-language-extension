# HieService.open


## Синтаксис


open (wbkInstanceId, rubInstanceId, rubKey, callback);


## Параметры


wbkInstanceId. Задает моникёр рабочей книги, по которому будет открыта иерархия главного каталога показателей;


rubInstanceId. Задает моникёр каталога показателей;


rubKey. Задает ключ каталога показателей;


callback. Задает обработчик окончания выполнения операции.


## Описание


Метод open посылает запрос на открытие иерархии рабочей книги по одному из параметров.
