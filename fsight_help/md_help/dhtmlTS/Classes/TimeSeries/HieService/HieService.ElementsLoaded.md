# HieService.ElementsLoaded


## Синтаксис


ElementsLoaded: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие ElementsLoaded наступает по окончанию загрузки дочерних элементов иерархии рядов.
