# HieService.openFromSource


## Синтаксис


openFromSource (wbk, callback);


## Параметры


wbk. Задает значение рабочей книги;


callback. Задает обработчик окончания выполнения операции.


## Описание


Метод openFromSource посылает запрос на открытие иерархии главного каталога показателей рабочей книги.


## Пример


Для выполнения примера необходимо создать сервис для работы с временными рядами c наименованием «hieSrv» (см. «[HieService.getChildEls](HieService.getChildEls.htm)») и в обработчике события открытия документа добавить следующий код:


	//Посылаем запрос на открытие иерархии рабочей книги
	// Создаём сервис для работы с временными рядами
	tsService = new PP.TS.TSService({ Metabase: metabase });
	// Открываем документ
	var wbk;
	tsService.openDocument(RUB_KEY, onWbkOpened);
	function onWbkOpened(sender, args)
	{
		wbk = args.Workbook;
		//Посылаем запрос на открытие иерархии главного каталога показателей рабочей книги
		hieSrv.openFromSource (wbk, PP.Delegate(onFromSourceOpened, this));
		function onFromSourceOpened(){
		console.log("Иерархия главного каталога показателей рабочей книги открыта");
	}
	}

После выполнения примера будет создан сервис для работы с иерархией базы данных временных рядов. Будет открыта иерархия базы данных временных рядов с ключом 1334, при этом в консоль будет выведено соответствующее сообщение. После открытия иерархии базы данных временных рядов будет отправлен запрос на открытие иерархии главного каталога показателей рабочей книги. После этого в консоль будет выведено сообщение «Иерархия главного каталога показателей рабочей книги открыта».
