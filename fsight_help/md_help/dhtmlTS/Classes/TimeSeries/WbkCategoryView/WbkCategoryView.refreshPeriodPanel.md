# WbkCategoryView.refreshPeriodPanel


## Синтаксис


refreshPeriodPanel();


## Описание


Метод refreshPeriodPanel обновляет панель «Период отображения» на ленте инструментов рабочей книги.


## Комментарии


После выполнения данного метода в соответствующих редакторах для ввода даты и времени на панели «Период отображения» будут отображаться первая и последняя даты, доступные в текущем источнике данных.
