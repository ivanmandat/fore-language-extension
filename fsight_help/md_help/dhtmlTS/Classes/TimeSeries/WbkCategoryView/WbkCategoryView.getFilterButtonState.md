# WbkCategoryView.getFilterButtonState


## Синтаксис


getFilterButtonState(sender: [PP.Ui.MenuItem](dhtmlUi.chm::/Classes/MenuItem/MenuItem.htm),
 button: [PP.Ui.SplitButton](dhtmlUi.chm::/Classes/SplitButton/SplitButton.htm));


## Параметры


sender.  Пункт
 меню кнопки «Фильтр» на ленте инструментов рабочей книги;


button.  Кнопка
 «Фильтр».


## Описание


Метод getFilterButtonState получает
 настройки для кнопки «Фильтр» на ленте инструментов рабочей книги.


## Комментарии


Метод возвращает JSON-объект со следующими свойствами: suppressEmpty
 - признак исключения пустых значений, suppressEmptyArea - признак исключения
 всего пустого ряда, определяемый с помощью перечисления [PP.TS.Ui.FilterEnum](../../../Enums/PP.TS.Ui.FilterEnum.htm).
