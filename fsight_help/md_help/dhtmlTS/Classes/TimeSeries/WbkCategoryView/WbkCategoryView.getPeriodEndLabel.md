## Синтаксис

## Описание


Метод getPeriodEndLabel возвращает
 надпись к [редактору
 для ввода даты](dhtmlUi.chm::/Components/DateTimePicker/DateTimePicker.htm) конца периода отображения временных рядов.


## Комментарии


Метод возвращает значение типа [PP.Ui.Label](dhtmlUi.chm::/Classes/Label/Label.htm).
