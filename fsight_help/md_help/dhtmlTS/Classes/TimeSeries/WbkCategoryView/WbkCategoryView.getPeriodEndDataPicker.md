# WbkCategoryView.getPeriodEndDataPicker


## Синтаксис


getPeriodEndDataPicker();


## Описание


Метод getPeriodEndDataPicker
 возвращает [редактор
 для ввода даты и времени](dhtmlUi.chm::/Components/DateTimePicker/DateTimePicker.htm) конца периода отображения временных рядов.


## Комментарии


Метод возвращает значение типа [PP.Ui.DateTimePicker](dhtmlUi.chm::/Components/DateTimePicker/DateTimePicker.htm).
