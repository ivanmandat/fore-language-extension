# WbkCategoryView.getTitleButton


## Синтаксис


getTitleButton();


## Описание


Метод getTitleButton возвращает кнопку «Заголовок» на ленте инструментов рабочей книги.


## Комментарии


Метод возвращает значение типа [PP.Ui.RibbonButton](dhtmlRibbon.chm::/Classes/RibbonButton/RibbonButton.htm).
