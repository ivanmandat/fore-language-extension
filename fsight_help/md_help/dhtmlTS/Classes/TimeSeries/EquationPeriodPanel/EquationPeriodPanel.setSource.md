# EquationPeriodPanel.setSource


## Синтаксис


setSource (source);


## Параметры


source. Задает [рабочую книгу](../Workbook/Workbook.htm).


## Описание


Метод setSource устанавливает рабочую книгу, данные которой отображаются в компоненте.
