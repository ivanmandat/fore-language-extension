# Конструктор ObserveAttributesPanel


## Синтаксис


PP.TS.Ui.ObserveAttributesPanel(settings);


## Параметры

## Описание


Конструктор ObserveAttributesPanel создает экземпляр класса [ObserveAttributesPanel](ObserveAttributesPanel.htm).
