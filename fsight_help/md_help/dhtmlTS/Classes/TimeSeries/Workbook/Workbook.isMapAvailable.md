# Workbook.isMapAvailable


## Синтаксис


isMapAvailable();


## Описание


Метод isMapAvailable определяет,
 доступно ли представление данных «Карта» в рабочей книге.


## Комментарии


Метод возвращает true,
 если карта доступна в рабочей книге, и false,
 если недоступна.


## Пример


Для выполнения предполагается наличие на странице компонента WorkbookBox
 с наименованием «workbookBox» (см. «[Пример
 создания компонента WorkbookBox](../../../Components/TimeSeries/WorkbookBox/Component_WorkbookBox.htm)»). Определим, доступна ли в рабочей
 книге карта:


// Получим источник данных
source = workbookBox.getSource();
// Получим активную страницу - экземпляр класса PP.TS.Workbook
sheet = source.getActiveSheet();
// Определим, доступна ли в рабочей книге карта
console.debug(sheet.isMapAvailable() ? "Карта доступна" : "Карта недоступна")

В результате выполнения примера в консоли браузера будет выведено сообщение
 «Карта доступна», если карта в рабочей книге доступна, и сообщение «Карта
 недоступна» в ином случае.
