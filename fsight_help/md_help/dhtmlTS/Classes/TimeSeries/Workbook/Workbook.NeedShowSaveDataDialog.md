# Workbook.NeedShowSaveDataDialog


## Синтаксис


NeedShowSaveDataDialog: Boolean;


## Описание


Свойство NeedShowSaveDataDialog используется для хранения информации о том, что был изменен ряд (его данные или атрибуты), для которого включено хранение истории.


## Комментарии


Если для свойства установлено значение true, то ряд был изменен, если false - не изменен.


## Пример


Для выполнения примера предполагается наличие на странице компонента [WorkbookBox](../../../Components/TimeSeries/WorkbookBox/WorkbookBox.htm) с наименованием «workbookBox» (см. «[Пример создания компонента WorkbookBox](../../../Components/TimeSeries/WorkbookBox/Component_WorkbookBox.htm)»), также необходимо в обработчике события открытия документа добавить следующий код:


	var setNeedShowSaveDataDialogButt = new PP.Ui.Button({
		ParentNode: document.body, //Родительский узел DOM
		Content: 'Установить признак изменения рабочей книги', //Подпись
		Click: PP.Delegate(onClickSetNeedShow)
	});
	function onClickSetNeedShow()
	{
		//Устанавливаем  измененные данные представления экспресс-отчета «Таблица».
		wbk.setTableChangedData({ c: [] });
		//Устанавливаем признак изменения свойств, данных или аттрибутов рабочей книги
		wbk.setNeedShowSaveDataDialog(true);
	}

После выполнения примера на html-странице будет размещена кнопка с наименованием «Установить признак изменения рабочей книги». При нажатии на эту кнопку для рабочей книги будет установленио свойство, которое содержит информацию о измененных данных представления экспресс-отчета «Таблица», а так же признак изменения свойств, данных или атрибутов рабочей книги. После нажатия на кнопку «Установить признак изменения рабочей книги» нужно нажать на кнопку «Обновить» вкладки «Главная» ленты инструментов рабочей книги, появится диалог подтверждения. Если в диалоге подтверждения нажать кнопку «Да», то появится окно для ввода комментария при сохранении данных. Если кнопку «Установить признак изменения рабочей книги» не нажимать, то при нажатии на кнопку «Обновить» вкладки «Главная» ленты инструментов диалог подтверждения не отобразится.
