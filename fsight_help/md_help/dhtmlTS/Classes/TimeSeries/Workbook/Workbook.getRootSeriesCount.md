# Workbook.getRootSeriesCount


## Синтаксис


getRootSeriesCount() : Number;


## Описание


Метод getRootSeriesCount возвращает
 количество корневых рядов.


## Комментарии


Корневой ряд - ряд не имеющий родителя.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 [WorkbookBox](../../../Components/TimeSeries/WorkbookBox/WorkbookBox.htm)
 с наименованием «workbookBox» (см. «[Пример
 создания компонента WorkbookBox](../../../Components/TimeSeries/WorkbookBox/Component_WorkbookBox.htm)»).


// Получим количество корневых рядов
wbk.getRootSeriesCount();
// -> 3

В результате выполнения примера было получено количество корневых рядов.
