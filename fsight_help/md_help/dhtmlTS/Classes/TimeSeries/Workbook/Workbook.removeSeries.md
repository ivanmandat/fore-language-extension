# Workbook.removeSeries


## Синтаксис


removeSeries (firstIndex, length);


## Параметры


firstIndex. Задает первый элемент массива строк;


length. Задает количество рядов, следующих за рядом с индексом, указанным в в первом параметре.


## Описание


Метод removeSeries удаляет строки таблицы.


## Комментарии


Первый параметр - стартовый элемент массива строк.


Второй параметр - количество элементов которые нужно удалить, начиная со стартового.
