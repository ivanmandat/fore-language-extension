# Workbook.RdsDim


## Синтаксис


RdsDim : PP.Rds.[Dictionary](dhtmlRds.chm::/Classes/Rds/Dictionary/Dictionary.htm);


## Описание


Свойство rdsDim** определяет
 измерение построенное на справочнике НСИ по своему ключу.


## Комментарии


Свойство задается при помощи метода setRdsDim(value,
 rdsKey). Параметры: value
 - измерение атрибута; rdsKey
 - ключ измерения.


Свойство возвращает значение при помощи метода getRdsDim(rdsKey).
 Параметры: rdsKey - ключ измерения;
 Возвращает объект типа PP.Rds.[Dictionary](dhtmlRds.chm::/Classes/Rds/Dictionary/Dictionary.htm).
