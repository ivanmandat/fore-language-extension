# Workbook.getRealIndex


## Синтаксис


getRealIndex (index, serie);


## Параметры


index. Задает индекс ряда с учётом вложенности;


serie. Задает ряд, индекс которого нужно получить.


## Описание


Метод getRealIndex возвращает индекс ряда.


## Комментарии


Индекс считается от первого, без учёта вложенности. В качестве параметров можно задавать либо index, либо serie.
