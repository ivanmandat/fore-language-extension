# TrendWizard.setHierarchy


## Синтаксис


setHierarchy(value: [PP.TS.MetaHierarchy](../MetaHierarchy/MetaHierarchy.htm));


## Описание


Метод setHierarchy** устанавливает иерархию метабазы в мастере для валидации данных путём сравнения трендов временных рядов.
