# ExpressionEditorDialog.Hierarchy


## Синтаксис


Hierarchy: [PP.TS.MetaHierarchy](../MetaHierarchy/MetaHierarchy.htm)


## Описание


Свойство Hierarchy** устанавливает и возвращает иерархию метабазы в редакторе выражения рабочей книги.


## Комментарии


Значение свойства устанавливается с помощью метода setHierarchy, а возвращается с помощью метода getHierarchy. Из JSON значение задать нельзя.
