# ValidationCatView.getPreviousErrorButton


## Синтаксис


getPreviousErrorButton();


## Описание


Метод getPreviousErrorButton** возвращает кнопку «Предыдущая ошибка» на вкладке «Валидация» ленты инструментов рабочей книги.


## Комментарии


Метод возвращает объект типа [PP.Ui.RibbonButton](dhtmlRibbon.chm::/Classes/RibbonButton/RibbonButton.htm).


При нажатии на данную кнопку в таблице рабочей книги будет выделена ячейка, соответствующая предыдущей ошибке валидации относительно текущей.
