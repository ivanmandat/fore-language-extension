# ValidationCatView.setExplanation


## Синтаксис


setExplanation(value: String);


## Параметры


value. Текст, объясняющий ошибку валидации. Необязательный параметр.


## Описание


Метод setExplanation** устанавливает в диалоге текст объяснения для выбранной ошибки валидации.


## Комментарии


Действие данного метода аналогично нажатию [кнопки «Объяснить»](ValidationCatView.getExplainButton.htm) на ленте инструментов рабочей книги. Если параметр не указан, то будет отображено окно без объясняющего текста:
