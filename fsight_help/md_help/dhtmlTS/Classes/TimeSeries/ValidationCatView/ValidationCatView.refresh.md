# ValidationCatView.refresh


## Синтаксис


refresh(propName: [PP.TS.Ui.ControlType](../../../Enums/PP.TS.Ui.ControlType.htm));


## Параметры


propName. Обновляемое свойство. В настоящее время данный параметр не используется.


## Описание


Метод refresh** обновляет вкладку «Валидация» на ленте инструментов рабочей книги.
