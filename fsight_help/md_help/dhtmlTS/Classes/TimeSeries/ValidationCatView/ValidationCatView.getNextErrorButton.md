# ValidationCatView.getNextErrorButton


## Синтаксис


getNextErrorButton();


## Описание


Метод getNextErrorButton** возвращает кнопку «Следующая ошибка» на вкладке «Валидация» ленты инструментов рабочей книги.


## Комментарии


Метод возвращает объект типа [PP.Ui.RibbonButton](dhtmlRibbon.chm::/Classes/RibbonButton/RibbonButton.htm).


При нажатии на данную кнопку в таблице рабочей книги будет выделена ячейка, соответствующая следующей ошибке валидации относительно текущей.
