## Описание


Класс WorkbookViewCtrl реализует
 контроллер для работы с рабочей книгой.


## Комментарии


Класс предназначен для внутреннего использования, создается внутри [WorkbookBox](../WorkbookBox/WorkbookBox.htm).


## Синтаксис


PP.initClass(PP.TS.WorkbookViewCtrl, PP.Ui.[Control](dhtmlUi.chm::/Classes/control/control.htm),
 "PP.TS.WorkbookViewCtrl");
