# ValidationGroupSettingsDialog.setCurrentState


## Синтаксис


setCurrentState(state: Object);


## Параметры


state. Настройки для диалога редактирования группы правил валидации.


## Описание


Метод setCurrentState** устанавливает настройки для диалога редактирования группы правил валидации в [рабочей книге](../Workbook/Workbook.htm).
