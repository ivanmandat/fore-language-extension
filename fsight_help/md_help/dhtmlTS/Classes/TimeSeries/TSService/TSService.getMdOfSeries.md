# TSService.getMdOfSeries


## Синтаксис


getMdOfSeries (wbk, selectedIndexSeries, selectedIndexCols, callback);


## Параметры


wbk. Задает значение [рабочей книги](../Workbook/Workbook.htm);


selectedIndexSeries. Задает массив индексов рядов в таблице;


selectedIndexCols. Задает массив индексов колонок в таблице;


callback. Задает функцию обратного вызова на получение ответа.


## Описание


Метод getMdOfSeries загружает метаданные рядов и колонок.
