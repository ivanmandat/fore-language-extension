# TSService.setMetadata


## Синтаксис


setMetadata (wbk, callback);


## Параметры


wbk. Задает рабочую книгу, экземпляр класса [Workbook](../Workbook/Workbook.htm);


callback. Задает обработчик окончания выполнения операции.


## Описание


Метод setMetadata сохраняет метаданные рабочей книги на сервере.
