# TSService.setSelectedSeries


## Синтаксис


setSelectedSeries (wbk, selectedIndexSeries, selectedIndexCols, callback);


## Параметры


wbk. Задает значение [рабочей книги](../Workbook/Workbook.htm);


selectedIndexSeries. Задает массив индексов рядов в таблице;


selectedIndexCols. Задает массив индексов колонок в таблице;


callback. Задает обработчик завершения операции.


## Описание


Метод setSelectedSeries устанавливает отмеченные ряды рабочей книги.
