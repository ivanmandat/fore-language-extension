# TSService.getSerieRevisions


## Синтаксис


getSerieRevisions (wbk, serie, callback);


## Параметры


wbk. Задает значение [рабочей книги](../Workbook/Workbook.htm);


serie. Задает ряд;


callback. Задает обработчик окончания выполнения операции.


## Описание


Метод getSerieRevisions возвращает ревизии для заданного ряда.


## Комментарии


В методе выполняется двойной запрос, сначала берутся все ревизии для базы данных временных рядов, потом берётся измерение с ревизиями для конкретного ряда, потом из них формируются ревизии для конкретного ряда. Так сделано потому, что в измерении с ревизиями для конкретного ряда возвращается не полная информация о ревизии.
