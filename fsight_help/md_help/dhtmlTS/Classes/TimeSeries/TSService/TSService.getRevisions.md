# TSService.getRevisions


## Синтаксис


getRevisions (wbk, rubKey, callback);


## Параметры


wbk. Задает значение [рабочей книги](../Workbook/Workbook.htm);


rubKey. Задает ключ базы данных временных рядов;


callback. Задает обработчик окончания выполнения операции.


## Описание


Метод getRevisions возвращает все ревизии базы данных временных рядов с ключом rubKey.
