# TSService.openRub


## Синтаксис


openRub (wbk, rubKey, callback);


## Параметры


wbk. Задает значение [рабочей книги](../Workbook/Workbook.htm);


rubKey. Задает значение ключа базы данных временных рядов;


callback. Задает обработчик окончания выполнения операции.


## Описание


Метод openRub открывает базу данных временных рядов по его ключу.

Пример использования метода openRub показан на странице «[ChooseFrequencyDialog.getSelectedKey](../ChooseFrequencyDialog/ChooseFrequencyDialog.getSelectedKey.htm)».
