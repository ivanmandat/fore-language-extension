# TSService.openDLDim


## Синтаксис


openDLDim (wbk, callback);


## Параметры


wbk. Задает значение [рабочей книги](../Workbook/Workbook.htm);


callback. Задает функцию обратного вызова.


## Описание


Метод openDLDim открывает измерение уровней календаря.
