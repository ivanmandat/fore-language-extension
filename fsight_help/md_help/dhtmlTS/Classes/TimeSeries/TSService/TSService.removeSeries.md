# TSService.removeSeries


## Синтаксис


removeSeries (wbk, series, callback);


## Параметры


wbk. Задает значение [рабочей книги](../Workbook/Workbook.htm);


series. Задает ряд;


callback. Задает обработчик окончания выполнения операции.


## Описание


Метод removeSeries удаляет строку таблицы рабочей книги.


## Пример


Пример использования метода removeSeries показан на странице «[LanerBoxView.insertSeries](../LanerBoxView/LanerBoxView.insertSeries.htm)».
