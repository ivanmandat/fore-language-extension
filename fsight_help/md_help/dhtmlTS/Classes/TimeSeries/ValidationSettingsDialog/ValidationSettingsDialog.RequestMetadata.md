# ValidationSettingsDialog.RequestMetadata


## Синтаксис


RequestMetadata(sender: Object, args: [PP.Mb.Ui.PropertyChangedEventArgs](dhtmlMetabase.chm::/Classes/Metabase/PropertyChangedEventArgs/PropertyChangedEventArgs.htm));


## Параметры


sender. Источник события;


args. Информация о событии. Параметр содержит объект типа [PP.Mb.Ui.PropertyChangedEventArgs](dhtmlMetabase.chm::/Classes/Metabase/PropertyChangedEventArgs/PropertyChangedEventArgs.htm), у которого свойство Metadata содержит поле Target - наименование класса, реализующего диалог для редактирования правила валидации данных в рабочей книге.


## Описание


Событие RequestMetadata** генерируется при отсутствии метаданных в источнике панели ошибок валидации в рабочей книге.
