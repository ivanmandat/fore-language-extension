# ValidationSettingsDialog.setCurrentState


## Синтаксис


setCurrentState(state: Object);


## Параметры


state. Настройки правила валидации. Необязательный параметр. Если данный параметр не указан, то по умолчанию в диалоге загружаются значения настроек по умолчанию.


## Описание


Метод setCurrentState** определяет настройки диалога для редактирования правила валидации данных в рабочей книге.
