# CrossFrequencyWizard.selLevels


## Синтаксис


selLevels(levels: Array);


## Параметры


levels. Список календарных уровней. Параметр представляет собой массив объектов типа [PP.TS.Ui.DimCalendarLvl](../../../Enums/DimCalendarLvl.htm).


## Описание


Метод selLevels** устанавливаем список календарных уровней.
