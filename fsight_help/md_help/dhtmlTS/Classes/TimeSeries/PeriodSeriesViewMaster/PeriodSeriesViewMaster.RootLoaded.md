# PeriodSeriesViewMaster.RootLoaded


## Синтаксис


RootLoaded(sender: Object, args: Object);


## Параметры


sender. Источник события;


args. Информация о событии. Доступный аргумент: JSON-объект с полем MetaAttrId - идентификатор атрибута ряда.


## Описание


Событие RootLoaded** наступает при загрузке данных в компонент [PeriodSeriesViewMaster](../../../Components/TimeSeries/PeriodSeriesViewMaster/PeriodSeriesViewMaster.htm).
