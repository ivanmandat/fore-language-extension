## Описание


Свойство Source** устанавливает источник данных для компонента [PeriodSeriesViewMaster](../../../Components/TimeSeries/PeriodSeriesViewMaster/PeriodSeriesViewMaster.htm).


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSource, а возвращается - с помощью метода getSource.
