# EquationWeightCoefficientsPanel.setSource


## Синтаксис


setSource (value);


## Параметры


value. Задает рабочую книгу, экземпляр класса [Workbook](../Workbook/Workbook.htm).


## Описание


Метод setSource устанавливает рабочую книгу, данные которой отображаются в компоненте.
