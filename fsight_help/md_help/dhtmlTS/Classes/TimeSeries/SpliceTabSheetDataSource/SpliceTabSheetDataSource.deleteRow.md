# SpliceTabSheetDataSource.deleteRow


## Синтаксис


deleteRow(index: Number, count: Number);


## Параметры


index. Индекс первой удаляемой строки таблицы;


count. Количество удаляемых строк.


## Описание


Метод deleteRow удаляет строки из таблицы сращиваемых временных рядов.
