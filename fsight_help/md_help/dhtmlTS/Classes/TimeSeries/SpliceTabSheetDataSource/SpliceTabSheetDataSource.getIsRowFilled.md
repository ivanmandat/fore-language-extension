# TabSheetDataSource.getIsRowFilled


## Синтаксис


getIsRowFilled(index: Number);


## Параметры


index. Индекс строки в таблице сращиваемых временных рядов.


## Описание


Метод getIsRowFilled определяет, задан ли временной ряд для указанной строки в таблице сращивания.


## Комментарии


Метод возвращает значение true, если для указанной строки в таблице сращивания задан временной ряд, в противном случае - значение false.
