# SpliceTabSheetDataSource.getFillRowsCount


## Синтаксис


getFillRowsCount();


## Описание


Метод getFillRowsCount возвращает количество строк таблицы, для которых задан временной ряд.


## Комментарии


Метод возвращает значение типа Number.
