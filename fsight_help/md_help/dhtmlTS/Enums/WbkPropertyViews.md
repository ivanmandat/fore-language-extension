# WbkPropertyViews


Пространство имен:
 PP.TS.Ui;


## Описание


Перечисление WbkPropertyViews
 содержит свойства представлений.


Используется следующими свойствами и методами:


	- [ReadonlyCatView.refresh](../Classes/TimeSeries/ReadonlyCatView/ReadonlyCatView.refresh.htm);


	- [FormulasCatView.refresh](../Classes/TimeSeries/FormulasCatView/FormulasCatView.refresh.htm);


	- [DataCatView.refresh](../Classes/TimeSeries/DataCatView/DataCatView.refresh.htm);


	- [WbkCategoryView.setPropertyValue](../Classes/TimeSeries/WbkCategoryView/WbkCategoryView.setPropertyValue.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 DataSourceOpened


		 Открытие источника данных.


		 EnableAnalyticsDataMining


		 Доступность кнопок интеллектуального
		 анализа данных на ленте инструментов.


		 EnableAnalyticsModeling


		 Доступность кнопки «Моделирование
		 и прогнозирование» и «Анализ временных рядов» на ленте инструментов.


		 EnableAnalyticsValidation


		 Доступность кнопок валидации данных на
		 ленте инструментов.


		 EnableFormulasButtons


		 Доступность кнопок формул.


		 EnableTransformationItems


		 Доступность пунктов преобразований.


		 GridChangeData


		 Изменённые данные в таблице.


		 GridSelection


		 Отметка в таблице.


		 SeriesPinned


		 Заблокированные ряды.


		 SourceOpened


		 Открытие источника.


		 UpdateChartEnabled


		 Доступность обновления диаграммы.


		 VisibleBreadCrumb


		 Видимость навигационной цепочки.


		 VisiblePropertyBar


		 Видимость панели свойств.


		 VisibleTitle


		 Видимость заголовка.
