# HPSmoothingParameterModeType


Пространство имен: PP.TS.Ui;


## Описание


Перечисление HPSmoothingParameterModeType содержит способы задания параметра сглаживания для фильтра Ходрика-Прескотта.


Используется свойством [HodrickPrescottFilterWizardView.CurrentState](../Classes/TimeSeries/HodrickPrescottFilterWizardView/HodrickPrescottFilterWizardView.CurrentState.htm).


## Допустимые значения


Значение


Краткое описание


EditDirectly


Параметр сглаживания определяется значением лямбды.


SetByFrequencyRule


Параметр сглаживания определяется значением степени.
