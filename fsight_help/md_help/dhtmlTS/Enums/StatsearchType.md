# StatSearchType


Пространство имен: PP.TS;


## Описание


Перечисление StatSearchType используется для определения метода подбора значений коэффициентов экспоненциального сглаживания.


Используется перечислением [PP.TS.StatCriterionType](StatCriterionType.htm).


## Допустимые значения


Значение


Краткое описание


Grid


Поиск по сетке.


NoSearch


Автоподбор не используется.


Optimal


Метод наилучшей пробы.
