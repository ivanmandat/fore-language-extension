# HieFilterMode


Пространство имен: PP.TS;


## Описание


 HieFilterMode является перечислением режимов фильтрации рядов в иерархии. Перечисление используется свойством [MetaHierarchy.FilterMode](../Classes/TimeSeries/MetaHierarchy/MetaHierarchy.FilterMode.htm).


## Допустимые значения


Значение


Краткое описание


All


Скрывать пустые ряды и папки.


LightWeight


Скрывать пустые ряды.


None


Нет фильтра.
