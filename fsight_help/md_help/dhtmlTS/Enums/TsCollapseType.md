# TsCollapseType


Пространство имен: PP.TS.Ui;


## Описание


Перечисление TsCollapseType содержит тип агрегации по времени.


Используется методом: [AggregationWizardView.getMethodType](../Classes/TimeSeries/AggregationWizardView/AggregationWizardView.getMethodType.htm).


## Допустимые значения


Значение


Краткое описание


Average


Вычисляется среднее значение элементов динамики.


First


Вычисляется первое имеющееся значение элементов динамики.


Last


Вычисляется последнее имеющееся значение элементов динамики.


Maximum


Вычисляется максимальное значение элементов динамики.


Minimum


Вычисляется минимальное значение элементов динамики.


Range


Вычисляется разность между максимальным и минимальным значениями элементов динамики.


Spread


Вычисляется стандартное отклонение по динамике. Стандартное отклонение - это мера того, насколько широко разбросаны значения элементов относительно их среднего.


Total


Вычисляется сумма значений элементов динамики.
