# RibbonTableStyleEnum


Пространство имен: PP.TS.Ui;


## Описание


Перечисление RibbonTableStyleEnum определяет тип стиля таблицы (вкладка риббона Таблица->Стиль).


Используется методом [TableCatView.getTableStyleState](../Classes/TimeSeries/TableCatView/TableCatView.getTableStyleState.htm).


## Допустимые значения


Значение


Краткое описание


Clear


Удалить стиль.


Name


Наименование стиля.


UndefinedStyle


Неопределён.
