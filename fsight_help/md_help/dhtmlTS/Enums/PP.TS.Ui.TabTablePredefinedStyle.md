# TabTablePredefinedStyle


Пространство имен:
 PP.TS.Ui;


## Описание


Перечисление является аналогом перечисления TabTablePredefinedStyle
 из сервисов.


Используется методом [TableCatView.getTableStyleState](../Classes/TimeSeries/TableCatView/TableCatView.getTableStyleState.htm).
