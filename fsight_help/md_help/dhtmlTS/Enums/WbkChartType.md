# WbkChartType


Пространство имен:
 PP.TS;


## Описание


Перечисление WbkChartType содержит
 тип диаграммы в рабочей книге.


Предназначен для внутреннего использования классом: [ChartCategory](../Classes/TimeSeries/ChartCategory/ChartCategory.htm)

		 Значение


		 Краткое
		 описание


		 0
		 Simple. Диаграмма для
		 данных в таблице.


		 1
		 StatMethod. Диаграмма
		 для данных, полученных в результате расчёта статистического метода.


		 2
		 DetermAggregation.
		 Диаграмма для данных, полученных в результате расчёта агрегации.


		 3
		 SpliceMethod. Диаграмма
		 для аднных, полученных в результате сращивания рядов.
