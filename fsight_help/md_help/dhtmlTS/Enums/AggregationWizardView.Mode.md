# AggregationWizardView.Mode


Пространство имен: PP.TS.Ui;


## Описание


Перечисление Mode содержит типы формул.


Используется свойством [AggregationWizardView.CurrentState](../Classes/TimeSeries/AggregationWizardView/AggregationWizardView.CurrentState.htm)


## Допустимые значения


Значение


Краткое описание


Aggregation


Агрегация.


Collapse


Коллапс (векторный расчет). Агрегирует данные с нижнего уровня календаря на верхний.


Interpolate


Интерполяция. Дезагрегирует данные с верхнего календарного уровня на нижний.
