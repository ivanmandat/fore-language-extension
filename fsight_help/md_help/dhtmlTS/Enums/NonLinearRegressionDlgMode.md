# NonLinearRegressionDlgMode


Пространство имен: PP.TS;


## Описание


Перечисление содержит перечисление режимов диалога для нелинейной регрессии.


Используется классом [AddNonLinearRegressionDlg](../Classes/TimeSeries/AddNonLinearRegressionDlg/AddNonLinearRegressionDlg.htm).


## Допустимые значения


Значение


Краткое описание


Add


Режим добавления нового ряда.


Edit


Режим редактирования существующего ряда.
