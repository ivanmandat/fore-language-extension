# MastersEnum


Пространство имен: PP.TS.Ui;


## Описание


Перечисление MastersEnum содержит список мастеров в панели свойств рядного режима.


Используется методом [WbkPropertyBarView.refreshAll](../Classes/TimeSeries/WbkPropertyBarView/WbkPropertyBarView.refreshAll.htm).
