# TsGridDefaultCMenu


Пространство имен:
 PP.TS.Ui;


## Описание


Перечисление TsGridDefaultCMenu содержит типы
 пунктов контекстного меню в таблице.


Используется методом: [LanerBoxView.isEditing](../Classes/TimeSeries/LanerBoxView/LanerBoxView.isEditing.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 Copy


		 Копировать.


		 Cut


		 Вырезать.


		 DeleteSeries


		 Удалить ряды.


		 History


		 История изменений.


		 Paste


		 Вставить.


		 ShowCalcParams


		 Отобразить вкладку «Параметры».


		 ShowMainSeriesProps


		 Отобразить вкладку «Основные».


		 Statistics


		 Статистика.


		 VisibleStatistics


		 Видимость статистики.
