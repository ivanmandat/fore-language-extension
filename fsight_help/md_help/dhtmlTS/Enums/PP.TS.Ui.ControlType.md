# ControlType


Пространство имен: PP.TS.Ui


## Описание


Перечисление ControlType содержит
 типы элементов управления и свойств в рабочей книге.


Предназначен для внутреннего использования, также используется методом
 [ValidationCatView.refresh](../Classes/TimeSeries/ValidationCatView/ValidationCatView.refresh.htm).
