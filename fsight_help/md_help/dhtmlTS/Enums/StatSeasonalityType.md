# StatSeasonalityType


Пространство имен: PP.TS.Ui;


## Описание


Перечисление StatSeasonalityType используется для определения модели сезонности.


Используется следующими свойствами и методами:


-
[ExponentialSmoothingWizardView.CurrentState](../Classes/TimeSeries/ExponentialSmoothingWizardView/ExponentialSmoothingWizardView.CurrentState.htm);


-
[X11EquationWizardView.CurrentState](../Classes/TimeSeries/X11EquationWizardView/X11EquationWizardView.CurrentState.htm);


-
[TrendWizardView.CurrentState](../Classes/TimeSeries/TrendWizardView/TrendWizardView.CurrentState.htm);


## Допустимые значения


Значение


Краткое описание


Additive


Аддитивная модель.


Multiplicative


Мультипликативная модель


None


Сезонность не определена.
