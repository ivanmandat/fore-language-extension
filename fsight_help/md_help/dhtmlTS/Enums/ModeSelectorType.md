# ModeSelectorType

## Описание


Перечисление ModeSelectorType содержит
 возможные варианты расположения элементов меню.


Используется методом [ViewCatView.refresh](../Classes/TimeSeries/ViewCatView/ViewCatView.refresh.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 Horizontal


		 По горизонтали.


		 Maximized


		 На закладках.


		 Vertical


		 По вертикали.
