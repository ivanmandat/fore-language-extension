# ValidationSettingsDialogMode


Пространство имен: PP.TS.Ui


## Описание


Перечисление ValidationSettingsDialogMode содержит режимы отображения диалога для редактирования правила валидации данных в рабочей книге.


Используется методом [ValidationSettingsDialog.setMode](../Classes/TimeSeries/ValidationSettingsDialog/ValidationSettingsDialog.setMode.htm).


## Допустимые значения


Значение


Краткое описание


 Normal
 Нормальный режим отображения.


 Short
 Сокращённый режим отображения.
