# FormulaTypeEnum


Пространство имен: PP.TS.Ui


## Описание


Перечисление FormulaTypeEnum
 содержит типы формул для расчета значений вычисляемых рядов.


Используется следующими методами:


	- [TrendWizardView.getMethodType](../Classes/TimeSeries/TrendWizardView/TrendWizardView.getMethodType.htm);


	- [TSService.updateCalculatedRow](../Classes/TimeSeries/TSService/TSService.updateCalculatedRow.htm).
