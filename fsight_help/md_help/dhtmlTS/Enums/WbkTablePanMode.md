# WbkTablePanMode


Пространство имен: PP.TS.Ui;


## Описание


Перечисление WbkTablePanMode содержит режимы отображения [панели настроек таблицы в рабочей книге](../Classes/TimeSeries/TablePanel/TablePanel.htm).


Используется свойством [TablePanel.Mode](../Classes/TimeSeries/TablePanel/TablePanel.Mode.htm).


## Допустимые значения


Значение


Краткое описание


Filter


Отображение настроек фильтрации данных в таблице.


TableAttr


Отображение настроек атрибутов таблицы.


TableStyle


Отображение настроек стиля оформления таблицы.
