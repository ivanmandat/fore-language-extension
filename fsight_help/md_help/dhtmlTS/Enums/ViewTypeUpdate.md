# ViewTypeUpdate


Пространство имен: PP.TS.Ui


## Описание


Перечисление ViewTypeUpdate содержит
 типы элементов управления и действий в экспресс-отчете.


Используется свойством [PropertyChangedEventArgs.TypeUpdateData](dhtmlMetabase.chm::/Classes/Metabase/PropertyChangedEventArgs/PropertyChangedEventArgs.TypeUpdateData.htm).
