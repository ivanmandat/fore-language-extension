# LnSerieKind


Пространство имен: PP.TS.Ui;


## Описание


Перечисление LnSerieKind используется для определения типа ряда.


Используется статическим методом PP.TS.[isDerivedSerie](../Classes/TimeSeries/PP.TS.isDerivedSerie.htm).


## Допустимые значения


Значение


Краткое описание


Calculate


Вычислимый ряд.


CalculateGroup


Вычислимая группа рядов.


Derived


Дочерний ряд вычисляемого ряда.


Group


Группа рядов.


Source


Ряд, основанный на показателе.


UserGroup


Пользовательская группа рядов.
