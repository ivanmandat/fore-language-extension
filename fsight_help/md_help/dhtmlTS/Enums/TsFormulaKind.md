# TsFormulaKind


Пространство имен:
 PP.TS.Ui;


## Описание


Перечисление TsFormulaKind содержит
 типы рядов.


Используется следующими статическими методами:


	- [isDeterministicSerie](../Classes/TimeSeries/PP.TS.isDeterministicSerie.htm);


	- [isAggregationSerie](../Classes/TimeSeries/PP.TS.isAggregationSerie.htm);


	- [isInteroplationSerie](../Classes/TimeSeries/PP.TS.isInteroplationSerie.htm);


	- [isSmoothingSerie](../Classes/TimeSeries/PP.TS.isSmoothingSerie.htm);


	- [isFillGapsSerie](../Classes/TimeSeries/PP.TS.isFillGapsSerie.htm);


	- [isRegressionSerie](../Classes/TimeSeries/PP.TS.isRegressionSerie.htm);


	- [isPredictionSerie](../Classes/TimeSeries/PP.TS.isPredictionSerie.htm).
