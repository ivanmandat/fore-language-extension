# StatInterceptMode


Пространство имен: PP.TS;


## Описание


Перечисление StatInterceptMode используется для определения режима задания константы.


Используется свойством [ArimaWizardView.CurrentState](../Classes/TimeSeries/ArimaWizardView/ArimaWizardView.CurrentState.htm).


## Допустимые значения


Значение


Краткое описание


AutoEstimate


Значение константы оценивается автоматически.


ManualEstimate


Значение константы задаётся пользователем.


None


Константа не используется.
