# SerieFlags


Пространство имен: PP.TS;


## Описание


Перечисление SerieFlags содержит типы deterministic-методов. Аналог перечисления LnSerieFlags в WSDL.


Используется следующими перечислениями:

[isLogSerie](../Classes/TimeSeries/PP.TS.isLogSerie.htm);


-
[isRoundSerie](../Classes/TimeSeries/PP.TS.isRoundSerie.htm);


-
[isModSerie](../Classes/TimeSeries/PP.TS.isModSerie.htm);


-
[isLagSerie](../Classes/TimeSeries/PP.TS.isLagSerie.htm);


-
[isLeadSerie](../Classes/TimeSeries/PP.TS.isLeadSerie.htm);


-
[isTruncateSerie](../Classes/TimeSeries/PP.TS.isTruncateSerie.htm).
