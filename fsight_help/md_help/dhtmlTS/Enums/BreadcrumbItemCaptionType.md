# BreadcrumbItemCaptionType


Пространство имен: PP.TS


## Описание


Перечисление BreadcrumbItemCaptionType содержит
 типы надписи у элемента цепочки навигации в компоненте [MetaAttributeBreadCrumb](../Components/TimeSeries/MetaAttributeBreadCrumb/MetaAttributeBreadCrumb.htm).


Используется свойством [MetaAttributeBreadCrumbView.ItemsCaptionType](../Classes/TimeSeries/MetaAttributeBreadCrumbView/MetaAttributeBreadCrumbView.ItemsCaptionType.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 SelectedCount


		 Надпись вида: «[Наименование показателя]
		 (Отмечено: [Количество выбранных элементов])».


		 SelectedNames


		 Надпись вида: «[Наименование показателя]
		 ([Первый элемент] .. [Последний элемент] ([Количество выбранных
		 элементов]))».
