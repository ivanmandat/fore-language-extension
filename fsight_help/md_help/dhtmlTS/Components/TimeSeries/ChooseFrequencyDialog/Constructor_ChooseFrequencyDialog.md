# Конструктор ChooseFrequencyDialog


## Синтаксис


PP.TS.Ui.ChooseFrequencyDialog (settings);


## Параметры


settings. JSON-объект со значениями свойств компонента.


## Описание


Конструктор ChooseFrequencyDialog создает экземпляр класса [ChooseFrequencyDialog](ChooseFrequencyDialog.htm).
