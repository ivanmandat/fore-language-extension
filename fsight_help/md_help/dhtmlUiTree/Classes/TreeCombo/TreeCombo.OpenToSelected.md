# TreeCombo.OpenToSelected


## Синтаксис


OpenToSelected: Boolean;


## Описание


Свойство OpenToSelected определяет,
 будет ли прокручиваться раскрывающийся список при открытии к первому выделенному
 элементу.


## Комментарии


Если установлено значение true,
 то при открытии раскрывающийся список будет прокручиваться до первого
 выделенного элемента.


По умолчанию установлено значение false.
