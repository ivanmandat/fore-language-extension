# TreeCombo.showSearch


## Синтаксис


showSearch();


## Описание


Метод showSearch отображает
 строку поиска по дереву.


## Комментарии


Метод раскрывает список компонента, отображает строку поиска и устанавливает
 фокус в строке поиска.
