# TreeCombo.HideOnNodeClick


## Синтаксис


HideOnNodeClick: Boolean


## Описание


Свойство HideOnNodeClick определяет, закрывается ли раскрывающаяся часть компонента [TreeCombo](../../Components/TreeCombo/TreeCombo.htm) при выборе вершины [дерева](../../Components/TreeList/TreeList.htm).


## Комментарии


Если для свойства установлено значение true, раскрывающаяся часть компонента [TreeCombo](../../Components/TreeCombo/TreeCombo.htm) при выборе вершины [дерева](../../Components/TreeList/TreeList.htm) закрывается. По умолчанию установлено значение false.
