# TreeCombo.AutoFillCaption


## Синтаксис


AutoFillCaption: Boolean


## Описание


Свойство AutoFillCaption определяет, отображается ли в заголовке компонента [TreeCombo](../../Components/TreeCombo/TreeCombo.htm) наименование выделенной вершины [дерева](../../Components/TreeList/TreeList.htm).


## Комментарии


Если для свойства установлено значение false в заголовке компонента [TreeCombo](../../Components/TreeCombo/TreeCombo.htm) не отображается наименование выделенной вершины [дерева](../../Components/TreeList/TreeList.htm). По умолчанию установлено значение true.
