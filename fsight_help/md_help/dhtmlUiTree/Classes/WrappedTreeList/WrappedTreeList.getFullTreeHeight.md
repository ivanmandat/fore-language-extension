# WrappedTreeList.getFullTreeHeight


## Синтаксис


getFullTreeHeight();


## Описание


Метод getFullTreeHeight возвращает
 полную высоту дерева с учетом рамки и отступов.


## Пример


Для выполнения примера необходим компонент WrappedTreeList с идентификатором
 «treelist» (см. «[Пример
 создания компонента WrappedTreeList](../../Components/WrappedTreeList/WrappedTreeList_Example.htm)»).


// Получим полную высоту дерева
treelist.getFullTreeHeight();
// -> 552
В результате выполнения примера была получена полная высота дерева.
