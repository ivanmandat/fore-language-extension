		 и [BubbleTreeMaster](dhtmlBubbleTree.chm::/Components/BubbleTreeMaster/BubbleTreeMaster.htm).

## Классы основанные на классе TreeListBase


		 Имя класса
		 Краткое описание


		 ![](../Class_Image.gif)
		 [DynamicTreeList](DynamicTreeList/DynamicTreeList.htm)
		 Класс DynamicTreeList
		 реализует компонент [DynamicTreeList](../Components/DynamicTreeList/DynamicTreeList.htm),
		 предназначенный для отображения дерева наследования объектов с
		 переменной высотой строки и виртуальной прокруткой.


		 ![](../Class_Image.gif)
		 [ListCaption](ListCaption/ListCaption.htm)
		 Класс ListCaption реализует
		 заголовок столбца в деревьях.


		 ![](../Class_Image.gif)
		 [ListCaptions](ListCaptions/ListCaptions.htm)
		 Класс ListCaptions
		 реализует заголовок дерева, состоящий из заголовков столбцов.


		 ![](../Class_Image.gif)
		 [SimpleTreeList](SimpleTreeList/SimpleTreeList.htm)
		 Класс SimpleTreeList
		 реализует компонент [SimpleTreeList](../Components/SimpleTreeList/SimpleTreeList.htm),
		 предназначенный для отображения простого дерева наследования объектов.


		 ![](../Class_Image.gif)
		 [TreeListBase](TreeListBase/TreeListBase.htm)
		 Класс TreeListBase
		 является базовым классом для отображения деревьев.


		 ![](../Class_Image.gif)
		 [TreeListView](TreeListView/TreeListView.htm)
		 Класс TreeListView
		 является базовым классом для визуальных представлений иерархических
		 деревьев.


		 ![](../Class_Image.gif)
		 [WrappedTreeList](WrappedTreeList/WrappedTreeList.htm)
		 Класс WrappedTreeList
		 реализует компонент [WrappedTreeList](../Components/WrappedTreeList/WrappedTreeList.htm),
		 предназначенный для отображения дерева наследования объектов с
		 переменной высотой строки.


		 ![](../Class_Image.gif)
		 [TreeListBox](TreeListBox/TreeListBox.htm)
		 Класс TreeListBox реализует
		 компонент [TreeListBox](../Components/TreeListBox/TreeListBox.htm),
		 предназначенный для отображения дерева наследования объектов с
		 постраничной навигацией.
