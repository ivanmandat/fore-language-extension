# TreeDataVisualizer.ItemClick


## Синтаксис


ItemClick: function(sender, args, timeout);


## Параметры


sender. Источник события;


args. Информация о событии;


timeout. Промежуток времени в миллисекундах, через который вызовется событие.


## Описание


Событие ItemClick наступает при нажатии на элемент диаграммы.
