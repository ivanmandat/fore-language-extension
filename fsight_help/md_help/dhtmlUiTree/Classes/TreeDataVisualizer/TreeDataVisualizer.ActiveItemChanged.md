# TreeDataVisualizer.ActiveItemChanged


## Синтаксис


ActiveItemChanged: function(sender, args, timeout);


## Параметры


sender. Источник события;


args. Информация о событии;


timeout. Промежуток времени в миллисекундах, через который вызовется событие.


## Описание


Событие ActiveItemChanged** наступает при изменении активного элемента диаграммы.
