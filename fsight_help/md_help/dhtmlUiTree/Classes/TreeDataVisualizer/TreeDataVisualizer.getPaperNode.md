# TreeDataVisualizer.getPaperNode


## Синтаксис


getPaperNode();


## Описание


Метод getPaperNode** возвращает SVG-элемент, в котором отрисовывается диаграмма.


## Комментарии


Метод возвращает значение типа SVGElement.
