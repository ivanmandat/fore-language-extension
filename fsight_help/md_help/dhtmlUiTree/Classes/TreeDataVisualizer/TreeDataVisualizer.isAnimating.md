# TreeDataVisualizer.isAnimating


## Синтаксис


isAnimating();


## Описание


Метод isAnimating** возвращает признак воспроизведения анимации диаграммы.


## Комментарии


Метод возвращает значение типа Boolean. Если анимация воспроизводится, то возвращается true, иначе - false.
