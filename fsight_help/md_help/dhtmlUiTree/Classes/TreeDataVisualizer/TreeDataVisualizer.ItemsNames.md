# TreeDataVisualizer.ItemsNames


## Синтаксис


ItemsNames: Object


## Описание


Свойство ItemsNames** определяет
 имена элементов диаграммы.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода set**ItemsNames**,
 а возвращается с помощью метода get**ItemsNames**.
