# TreeModel.getItem


## Синтаксис


getItem(index: String);


## Параметры


index. Индекс возвращаемого элемента.


## Описание


Метод getItem** возвращает элемент модели данных по индексу.


## Комментарии


Метод возвращает значение типа [PP.Ui.TreeItem](../TreeItem/TreeItem.htm).
