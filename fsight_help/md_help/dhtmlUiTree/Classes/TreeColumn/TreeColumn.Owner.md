# TreeColumn.Owner


## Синтаксис


Owner: PP.Ui.[TreeList](../TreeList/TreeList.htm)


## Описание


Свойство Owner устанавливает
 родительский элемент столбца.


## Комментарии


Значение свойства - экземпляр класса PP.Ui.[TreeList](../TreeList/TreeList.htm).


Значение свойства устанавливается из JSON и с помощью метода setOwner,
 а возвращается с помощью метода getOwner.
