# TreeColumn.Type


## Синтаксис


Type: PP.Ui.[TreeColumnType](../../Enums/TreeColumnType.htm)


## Описание


Свойство Type определяет тип
 элементов столбца во время сортировки.


## Комментарии


Значение свойства - элемент перечисления PP.Ui.[TreeColumnType](../../Enums/TreeColumnType.htm).


Значение свойства устанавливается из JSON и с помощью метода setType,
 а возвращается с помощью метода getType.

Пример использования свойства приведен в описании свойства [TreeColumn.FormatMethod](TreeColumn.FormatMethod.htm).
