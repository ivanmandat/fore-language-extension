# TreeColumn.Editable


## Синтаксис


Editable: Boolean


## Описание


Свойство Editable определяет
 возможность редактирования столбца.


## Комментарии


Допустимые значения:


	- true.
	 Редактирование столбца разрешено (по умолчанию);


	- false.
	 Редактирование столбца запрещено.


Значение свойства устанавливается из JSON и с помощью метода setEditable,
 а возвращается с помощью метода getEditable.
