# TreeColumn.VisibleWidth


## Синтаксис


VisibleWidth: Number


## Описание


Свойство VisibleWidth определяет
 видимую ширину столбца дерева.


## Комментарии


Значение свойства устанавливается с помощью метода setVisibleWidth,
 а возвращается с помощью метода getVisibleWidth.
