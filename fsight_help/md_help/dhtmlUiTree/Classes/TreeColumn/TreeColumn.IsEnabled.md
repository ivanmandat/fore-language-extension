# TreeColumn.IsEnabled


## Синтаксис


IsEnabled: Boolean;


## Описание


Свойство IsEnabled определяет,
 активен ли столбец.


## Комментарии


Допустимые значения:


	- true.
	 Столбец активен (по умолчанию);


	- false.
	 Столбец неактивен.


Значение свойства устанавливается из JSON и с помощью метода setIsEnabled,
 а возвращается с помощью метода getIsEnabled.
