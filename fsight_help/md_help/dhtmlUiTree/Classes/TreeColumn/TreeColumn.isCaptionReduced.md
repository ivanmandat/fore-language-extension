# TreeColumn.isCaptionReduced


## Синтаксис


isCaptionReduced();


## Описание


Метод isCaptionReduced возвращает
 признак уменьшения ширины заголовка столбца дерева.


## Комментарии


Метод возвращает значение true,
 если ширина заголовка столбца была уменьшена, иначе - значение false.
