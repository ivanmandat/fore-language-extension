# TreeNode.Focused


## Синтаксис


Focused: Boolean


## Описание


Свойство Focused определяет
 наличие фокуса у вершины дерева.


## Комментарии


Допустимые значения:


	- true. Фокус установлен
	 на вершину;


	- false. Фокус не установлен
	 на вершину.


Значение свойства устанавливается из JSON и с помощью метода setFocused,
 а возвращается с помощью метода getFocused.


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [TreeList](../../Components/TreeList/TreeList.htm) c наименованием
 «treeList» (см. «[Пример
 создания компонента TreeList](../../Components/TreeList/TreeList_example.htm)»). Определяем наличие фокуса у вершины
 с содержимым «Africa»:


// Определяем наличие фокуса у вершины с содержимым «Africa»
if (treeList.findText("Africa").getFocused()) {
    console.log("Фокус на вершине с содержимым «Africa»");
} else {
    console.log("На вершине с содержимым «Africa» фокуса нет");
}
В результате в консоль будет выведен результат проверки наличия фокуса
 у вершины:


На вершине с содержимым «Africa» фокуса нет
