# TreeNode.Class


## Синтаксис


Class: String


## Описание


Свойство Class определяет css-класс
 вершины дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setClass,
 а возвращается с помощью метода getClass.


Для добавления css-класса используется метод [addClass](TreeNode.addClass.htm).


Для удаления css-класса используется метод [removeClass](TreeNode.removeClass.htm).


## Пример


Пример использования свойства приведен в описании метода [TreeNode.removeClass](TreeNode.removeClass.htm).
