# TreeNode.Key


## Синтаксис


Key: String


## Описание


Свойство Key устанавливает ключ вершины.


## Комментарии


Значение свойства является параметром для методов [TreeList.getNodeByKey](../TreeList/TreeList.getNodeByKey.htm) и TreeList.getNodeByKeyGlobal.
