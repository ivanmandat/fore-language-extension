# TreeNode.getRealHeight


## Синтаксис


getRealHeight();


## Описание


Метод getRealHeight возвращает
 реальную высоту вершины.


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [TreeList](../../Components/TreeList/TreeList.htm) c наименованием
 «treeList» (см. «[Пример
 создания компонента TreeList](../../Components/TreeList/TreeList_example.htm)»). Получаем реальную высоту вершины с
 содержимым «Africa»:


// Получаем реальную высоту вершины с содержимым «Africa»
console.log("Реальная высота вершины с содержимым «Africa»: " + treeList.findText("Africa").getRealHeight());
В результате в консоль будет выведена реальная высота вершины с содержимым
 «Africa»:


Реальная высота вершины с содержимым «Africa»: 20
