# TreeNode.selectChild


## Синтаксис


selectChild(fireEvents);


## Параметры


fireEvents. Признак
 вызова событий, после выполнения метода. Чтобы события были вызваны параметр
 должен быть равен значению true, иначе - значению false.


## Описание


Метод selectChild отмечает всех
 потомков вершины.
