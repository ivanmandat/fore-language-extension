# TreeNode.Expanded


## Синтаксис


Expanded: Boolean


## Описание


Свойство Expanded определяет, отображаются ли дочерние элементы вершины.


## Комментарии


Если для свойства установлено значение true, то дочерние элементы вершины отображаются. По умолчанию установлено значение false, то есть вершина отображается в свернутом состоянии.
