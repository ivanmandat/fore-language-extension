# TreeNode.getIndex


## Синтаксис


getIndex();


## Описание


Метод getIndex возвращает индекс
 вершины в дереве.


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [TreeList](../../Components/TreeList/TreeList.htm) c наименованием
 «treeList» (см. «[Пример
 создания компонента TreeList](../../Components/TreeList/TreeList_example.htm)»). Получаем индекс вершины с содержимым
 «Africa»:


// Получаем индекс вершины с содержимым «Africa»
console.log("Индекс вершины с содержимым «Africa»: " + treeList.findText("Africa").getIndex());
В результате в консоль будет выведен индекс вершины с содержимым «Africa»:


Индекс вершины с содержимым «Africa»: 1
