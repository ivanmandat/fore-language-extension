# TreeNode.ImageIndex


## Синтаксис


ImageIndex: Number


## Описание


Свойство ImageIndex определяет
 индекс пиктограммы, расположенной в компоненте [ImageList](dhtmlUi.chm::/Components/imagelist/imagelist.htm).


## Комментарии


Если компонент [ImageList](dhtmlUi.chm::/Components/imagelist/imagelist.htm)
 состоит из нескольких строк и столбцов, необходимо указывать индекс столбца
 и строки.

Пример использования свойства приведен в описании события [TreeList.NodeUnHovered](../TreeList/TreeList.NodeUnHovered.htm).
