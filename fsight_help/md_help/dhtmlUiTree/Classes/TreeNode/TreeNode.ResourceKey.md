# TreeNode.ResourceKey


## Синтаксис


ResourceKey: String


## Описание


Свойство ResourceKey определяет
 ресурсный ключ для вершины дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setResourceKey,
 а возвращается с помощью метода getResourceKey.
