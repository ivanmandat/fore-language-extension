# TreeNode.isLast


## Синтаксис


isLast();


## Описание


Метод isLast определяет, является
 ли вершина последней в своей ветке.


## Комментарии


Метод возвращает значение true,
 если вершина последняя в своей ветке, иначе - значение false.


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [TreeList](../../Components/TreeList/TreeList.htm) c наименованием
 «treeList» (см. «[Пример
 создания компонента TreeList](../../Components/TreeList/TreeList_example.htm)»). Определяем, является ли вершина с
 содержимым «Africa» последней вершиной в своей ветке:


// Определяем, является ли вершина с содержимым «Africa» последней вершиной в своей ветке
if (treeList.findText("Africa").isLast()) {
    console.log("Вершина с содержимым «Africa» последняя в своей ветке");
} else {
    console.log("Вершина с содержимым «Africa» не последняя в своей ветке");
}
В результате в консоль будет выведен результат проверки:


Вершина с содержимым «Africa» последняя в своей
 ветке
