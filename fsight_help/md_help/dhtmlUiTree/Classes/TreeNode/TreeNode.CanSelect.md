# TreeNode.CanSelect


## Синтаксис


CanSelect: Boolean


## Описание


Свойство CanSelect определяет, можно ли выделять вершину.


## Комментарии


Если для свойства установлено значение false, то вершина не может быть выделенной. По умолчанию установлено значение true.
