# TreeNode.Value


## Синтаксис


Value: String


## Описание


Свойство Value определяет значение
 вершины дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setValue,
 а возвращается с помощью метода getValue.
