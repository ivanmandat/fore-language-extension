# TreeNode.isVisible


## Синтаксис


isVisible(partial: Boolean);


## Параметры


partial. Признак частичной
 проверки видимости вершины. Если значение параметра равно true, то видимость
 будет определена частично.


## Описание


Метод isVisible определяет видимость
 вершины.


## Комментарии


Метод возвращает значение true,
 если вершина видима, false, если
 находится за пределами видимости, и null,
 если вершина скрыта или была отфильтрована.


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [TreeList](../../Components/TreeList/TreeList.htm) c наименованием
 «treeList» (см. «[Пример
 создания компонента TreeList](../../Components/TreeList/TreeList_example.htm)»). Определяем видимость вершины с содержимым
 «Africa»:


// Определяем видимость вершины с содержимым «Africa»
if (nodeAfrica.isVisible()) {
    console.log("Вершина с содержимым «Africa» видима");
} else {
    console.log("Вершина с содержимым «Africa» невидима");
}
В результате в консоль будет выведен результат проверки видимости вершины:


Вершина с содержимым «Africa» видима
