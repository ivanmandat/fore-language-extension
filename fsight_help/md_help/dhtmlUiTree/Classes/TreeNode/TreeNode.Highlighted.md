# TreeNode.Highlighted


## Синтаксис


Highlighted: Boolen


## Описание


Свойство Highlighted определяет, применяется ли подсветка для указанной вершины.


## Комментарии


Подсветка возможна, если для свойства [TreeList.EnableHighlight](../TreeList/TreeList.EnableHighlight.htm) установлено значение true.


Цвет подсветки и шрифт текста подсвеченной вершины определяются в свойствах [TreeList.HighlightColor](../TreeList/TreeList.HighlightColor.htm) и [TreeList.HighlightFont](../TreeList/TreeList.HighlightFont.htm) соответственно.
