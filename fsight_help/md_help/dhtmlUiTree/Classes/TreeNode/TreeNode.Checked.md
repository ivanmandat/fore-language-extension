# TreeNode.Checked


## Синтаксис


Checked: Boolean


## Описание


Свойство Checked определяет, установлен ли флаг для вершины.


## Комментарии


Если для свойства установлено значение true, то флаг установлен. По умолчанию установлено значение false.
