# Конструктор DynamicTreeList


## Синтаксис

## Параметры


settings. JSON-объект со значениями
 свойств класса.


## Описание


Конструктор SimpleTreeList создаёт
 экземпляр класса [DynamicTreeList](DynamicTreeList.htm).
