# PropertyMapper.Getter


## Синтаксис


Getter: function(item){ };


## Описание


Свойство Getter определяет функцию
 получения значения.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setGetter, а возвращается с помощью
 метода getGetter.
