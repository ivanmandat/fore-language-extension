## Описание


Класс PropertryMapper реализует
 объект сопоставления данных в деревьях.


## Комментарии


Используется в качестве сопоставления данных для следующих компонентов:


	- PP.Ui.TreeListBox


	- PP.Ui.WrappedTreeList


	- PP.Ui.DynamicTreeList


	- PP.Ui.SimpleTreeList


## Синтаксис


PP.initClass(PP.PropertryMapper, PP.Ui.[Object](dhtmlCommon.chm::/Classes/object/object.htm),
 "PropertryMapper");
