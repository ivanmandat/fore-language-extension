# PropertyMapper.Setter


## Синтаксис


Setter: function(item, value){ };


## Описание


Свойство Setter определяет функцию
 установки значения.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSetter, а возвращается с помощью
 метода getSetter.
