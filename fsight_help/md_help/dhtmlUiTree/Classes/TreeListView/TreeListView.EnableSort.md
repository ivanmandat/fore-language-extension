# TreeListView.EnableSort


## Синтаксис


EnableSort: Boolean;


## Описание


Свойство EnableSort определяет
 возможность сортировки.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setEnableSort,
 а возвращается с помощью метода getEnableSort.


## Пример


Для выполнения примера необходим компонент SimpleTreeList с идентификатором
 «Tree» (см. «[Пример
 создания компонента SimpleTreeList](../../Components/SimpleTreeList/SimpleTreeList_Example.htm)»).


// Выключим возможность сортировки
Tree.setEnableSort(false);
В результате выполнения примера в дереве была выключена возможность
 сортировки.
