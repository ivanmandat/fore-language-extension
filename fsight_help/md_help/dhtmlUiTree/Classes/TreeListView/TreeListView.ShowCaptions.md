# TreeListView.ShowCaptions


## Синтаксис


ShowCaptions: Boolean;


## Описание


Свойство ShowCaptions определяет
 признак отображения заголовков столбцов.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setShowCaptions,
 а возвращается с помощью метода getShowCaptions.


## Пример


Для выполнения примера необходим компонент SimpleTreeList с идентификатором
 «Tree» (см. «[Пример
 создания компонента SimpleTreeList](../../Components/SimpleTreeList/SimpleTreeList_Example.htm)»).


// Скроем заголовки столбцов
Tree.setShowCaptions(false);
В результате выполнения примера заголовки столбцов были скрыты.
