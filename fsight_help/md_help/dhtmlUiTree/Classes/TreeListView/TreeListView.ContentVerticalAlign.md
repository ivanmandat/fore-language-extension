# TreeListView.ContentVerticalAlign


## Синтаксис


ContentVerticalAlign: PP.VerticalAlignment;


## Описание


Свойство ContentVerticalAlign
 определяет вертикальное выравнивание заголовков.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setContentVerticalAlign,
 а возвращается с помощью метода getContentVerticalAlign.


## Пример


Для выполнения примера необходим компонент SimpleTreeList с идентификатором
 «Tree» (см. «[Пример
 создания компонента SimpleTreeList](../../Components/SimpleTreeList/SimpleTreeList_Example.htm)»).


// Установим вертикальное выравнивание
Tree.setContentVerticalAlign(PP.VerticalAlignment.Bottom);
В результате выполнения примера было установлено вертикальное выравнивание.
