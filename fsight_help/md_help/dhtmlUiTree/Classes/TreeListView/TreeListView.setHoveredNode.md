# TreeListView.setHoveredNode


## Синтаксис


setHoveredNode(nodeKey);


## Параметры


nodeKey. Ключ вершины.


## Описание


Метод setHoveredNode устанавливает
 подсветку от наведения мыши указанной вершине.


## Пример


Для выполнения примера необходим компонент SimpleTreeList с идентификатором
 «Tree» (см. «[Пример
 создания компонента SimpleTreeList](../../Components/SimpleTreeList/SimpleTreeList_Example.htm)»).


// Подсветим первую вершину
Tree.setHoveredNode(0)
В результате выполнения примера была установлена подсветка на первую
 вершину.
