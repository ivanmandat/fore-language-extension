# TreeListView.DragAndDropDelay


## Синтаксис


DragAndDropDelay: Number;


## Описание


Свойство DragAndDropDelay определяет
 задержку перед началом включения режима перетаскивания.


## Комментарии


Значение задается в миллисекундах. По умолчанию содержит значение 500.


Значение свойства устанавливается из JSON и с помощью метода setDragAndDropDelay,
 а возвращается с помощью метода getDragAndDropDelay.
