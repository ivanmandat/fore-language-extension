# TreeListView.ScrollStep


## Синтаксис


ScrollStep: Number;


## Описание


Свойство ScrollStep определяет
 шаг прокрутки колесом мыши и автопрокруткой.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setScrollStep,
 а возвращается с помощью метода getScrollStep.


## Пример


Для выполнения примера необходим компонент SimpleTreeList с идентификатором
 «Tree» (см. «[Пример
 создания компонента SimpleTreeList](../../Components/SimpleTreeList/SimpleTreeList_Example.htm)»).


// Зададим шаг прокрутки
Tree.setScrollStep(80);
В результате выполнения примера был задан шаг прокрутки.
