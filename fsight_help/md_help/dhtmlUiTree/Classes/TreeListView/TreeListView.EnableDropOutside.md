# TreeListView.EnableDropOutside


## Синтаксис


EnableDropOutside: Boolean;


## Описание


Свойство EnableDropOutside определяет
 возможность перетаскивания из других деревьев.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setEnableDropOutside,
 а возвращается с помощью метода getEnableDropOutside.
