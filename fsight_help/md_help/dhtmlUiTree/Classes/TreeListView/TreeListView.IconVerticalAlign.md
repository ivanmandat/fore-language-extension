# TreeListView.IconVerticalAlign


## Синтаксис


IconVerticalAlign: PP.Ui.TreeListIconVerticalAlign;


## Описание


Свойство IconVerticalAlign определяет
 вертикальное выравнивание пиктограмм.


## Комментарии


Значение по умолчанию: PP.Ui.TreeListIconVerticalAlign.Top.


Значение свойства устанавливается из JSON и с помощью метода setIconVerticalAlign,
 а возвращается с помощью метода getIconVerticalAlign.


## Пример


Для выполнения примера необходим компонент SimpleTreeList с идентификатором
 «Tree», содержащий пиктограммы (см. «[TreeListBase.NodeImageIndex](../TreeListBase/TreeListBase.NodeImageIndex.htm)»).


// Зададим вертикальное выравнивание пиктограмм
Tree.IconVerticalAlign(PP.Ui.TreeListIconVerticalAlign.Middle);
В результате выполнения примера было задано вертикальное выравнивание
 пиктограмм.
