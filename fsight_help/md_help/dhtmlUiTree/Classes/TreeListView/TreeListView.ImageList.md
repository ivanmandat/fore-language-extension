# TreeListView.ImageList


## Синтаксис


ImageList: PP.ImageList;


## Описание


Свойство ImageList определяет
 изображение с пиктограммами для вершин.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setImageList,
 а возвращается с помощью метода getImageList.
