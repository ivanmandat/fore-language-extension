# TreeListView.EnableDragAndDrop


## Синтаксис


EnableDragAndDrop: Boolean;


## Описание


Свойство EnableDragAndDrop определяет
 активность режима перетаскивания.


## Комментарии


По умолчанию имеет значение false
 - режим перетаскивания выключен.


Значение свойства устанавливается из JSON и с помощью метода setEnableDragAndDrop,
 а возвращается с помощью метода getEnableDragAndDrop.
