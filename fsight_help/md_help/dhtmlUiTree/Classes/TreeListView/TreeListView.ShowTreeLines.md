# TreeListView.ShowTreeLines


## Синтаксис


ShowTreeLines: Boolean;


## Описание


Свойство ShowTreeLines определяет
 отображение линий структуры дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setShowTreeLines,
 а возвращается с помощью метода getShowTreeLines.


## Пример


Для выполнения примера необходим компонент SimpleTreeList с идентификатором
 «Tree» (см. «[Пример
 создания компонента SimpleTreeList](../../Components/SimpleTreeList/SimpleTreeList_Example.htm)»).


// Спрячем линии структуры дерева
Tree.setShowTreeLines(false);
В результате выполнения примера были спрятаны линии структуры дерева.
