# TreeListView.getVisibleNodes


## Синтаксис


getVisibleNodes();


## Описание


Метод getVisibleNodes возвращает
 ключи всех видимых вершин.


## Пример


Для выполнения примера необходим компонент SimpleTreeList с идентификатором
 «Tree» (см. «[Пример
 создания компонента SimpleTreeList](../../Components/SimpleTreeList/SimpleTreeList_Example.htm)»).


// Получим все видимые вершины
Tree.getVisibleNodes();
// -> ["0", "1", "2", "key3"]
В результате выполнения примера были получены ключи видимых вершин.
