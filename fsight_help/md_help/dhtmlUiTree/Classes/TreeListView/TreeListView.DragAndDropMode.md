# TreeListView.DragAndDropMode


## Синтаксис


DragAndDropMode: PP.Ui.[TreeListDragAndDropMode](../../Enums/TreeListDragAndDropMode.htm);


## Описание


Свойство DragAndDropMode определяет
 режим перетаскивания.


## Комментарии


По умолчанию содержит одиночный режим: PP.Ui.TreeListDragAndDropMode.Single.


Значение свойства устанавливается из JSON и с помощью метода setDragAndDropMode,
 а возвращается с помощью метода getDragAndDropMode.
