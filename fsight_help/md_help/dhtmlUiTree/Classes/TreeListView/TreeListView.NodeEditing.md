# TreeListView.NodeEditing


## Синтаксис


NodeEditing: function(sender, args);


## Параметры


sender. Источник события;


args. Информация
 о событии:


	- Node. String. Ключ вершины;


	- Column. Number. Индекс столбца.


## Описание


Событие NodeEditing наступает
 перед редактированием ячейки.
