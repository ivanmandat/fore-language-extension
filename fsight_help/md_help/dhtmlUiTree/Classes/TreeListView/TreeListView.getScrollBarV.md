# TreeListView.ScrollBarV


## Синтаксис


ScrollBarV : [PP.Ui.ScrollBar](dhtmlUi.chm::/Classes/ScrollBar/ScrollBar.htm);


## Описание


Свойство ScrollBarV определяет
 вертикальную полосу прокрутки.


## Комментарии


Значение свойства устанавливается из JSON, а возвращается с помощью
 метода getScrollBarV.


## Пример


Использование свойства аналогично свойству [TreeListView.ScrollBarH](TreeListView.getScrollBarH.htm).
