# TreeListView.DragAndDropIndent


## Синтаксис


DragAndDropIndent: Number;


## Описание


Свойство DragAndDropIndent определяет
 отступ разделительной линии, при использовании режима перетаскивания.


## Комментарии


Значение задается в пикселях.


Значение свойства устанавливается из JSON и с помощью метода setDragAndDropIndent,
 а возвращается с помощью метода getDragAndDropIndent.
