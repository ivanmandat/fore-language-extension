# TreeListBox.ActiveNode


## Синтаксис


ActiveNode: String;


## Описание


Свойство ActiveNode определяет
 ключ раскрытой вершины.


## Пример


Для выполнения примера необходим компонент TreeListBox с идентификатором
 «treelist» (см. «[Пример
 создания компонента TreeListBox](../../Components/TreeListBox/TreeListBox_Example.htm)»).


// Установим активную вершину
treelist.setActiveNode("105000");
В результате выполнения примера была установлена текущая раскрытая вершина.
