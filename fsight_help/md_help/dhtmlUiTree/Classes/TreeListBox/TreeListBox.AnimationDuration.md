# TreeListBox.AnimationDuration


## Синтаксис


AnimationDuration: Number;


## Описание


Свойство AnimationDuration определяет
 скорость анимации перелистывания.


## Комментарии


Значение задается в миллисекундах, значение по умолчанию 500.


## Пример


Для выполнения примера необходим компонент TreeListBox с идентификатором
 «treelist» (см. «[Пример
 создания компонента TreeListBox](../../Components/TreeListBox/TreeListBox_Example.htm)»).


// Установим скорость анимации перелистывания
treelist.setAnimationDuration(1500)
В результате выполнения примера была установлена новая скорость анимации
 перелистывания.
