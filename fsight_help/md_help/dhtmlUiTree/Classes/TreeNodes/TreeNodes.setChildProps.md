# TreeNodes.setChildProps


## Синтаксис


setChildProps(settings: Object, recursive: Boolean);


## Параметры


settings. Объект, описывающий
 устанавливаемые свойства;


recursive. Признак установки свойств всем потомкам вершин. Если необходимо
 установить свойства всем потомкам вершин, то значение параметра должно
 быть true, иначе - false.


## Описание


Метод setChildProps устанавливает
 свойства вершинам и, если необходимо, всем их потомкам.
