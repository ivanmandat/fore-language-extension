# TreeNodes.getItemValue


## Синтаксис


getItemValue(index, name);


## Параметры


index. Индекс вершины, значение
 свойства которой необходимо вернуть;


name. Имя свойства, значение
 которого надо вернуть.


## Описание


Метод getItemValue возвращает
 значение свойства вершин по индексу.


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [TreeList](../../Components/TreeList/TreeList.htm) c наименованием
 «treeList» (см. «[Пример
 создания компонента TreeList](../../Components/TreeList/TreeList_example.htm)»). Получаем текст первой корневой вершины
 дерева:


// Получаем значение первой корневой вершины дерева
console.log("Текст первой корневой вершины дерева: " + treeList.getNodes().getItemValue(0, "Text"));
В результате в консоль будет выведен текст первой корневой вершины:


Текст первой корневой вершины дерева: World
