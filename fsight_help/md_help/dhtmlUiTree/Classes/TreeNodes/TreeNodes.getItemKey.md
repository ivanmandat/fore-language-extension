# TreeNodes.getItemKey


## Синтаксис


getItemKey(index);


## Параметры


index. Индекс вершины, ключ
 которой необходимо вернуть.


## Описание


Метод getItemKey возвращает
 ключ вершины по индексу.


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [TreeList](../../Components/TreeList/TreeList.htm) c наименованием
 «treeList» (см. «[Пример
 создания компонента TreeList](../../Components/TreeList/TreeList_example.htm)»). Получаем ключ первой корневой вершины
 дерева:


// Получаем ключ первой корневой вершины дерева
console.log("Ключ первой корневой вершины дерева: " + treeList.getNodes().getItemKey(0));
В результате в консоль будет выведен ключ первой корневой вершины:


Ключ первой корневой вершины дерева: 0
