# TreeNodes.getCount


## Синтаксис


getCount (index)


## Параметры


index. Индекс вершины, количество
 дочерних вершин которой необходимо вернуть.


## Описание


Метод getCount возвращает количество
 дочерних вершин у указанной вершины.
