# TreeNodes.FilterValue


## Синтаксис


FilterValue: String;


## Описание


Свойство FilterValue определяет значение, по которому осуществляется фильтрация вершин иерархического дерева.


## Комментарии


Значение свойства устанавливается с помощью метода setFilterValue. Из JSON значение задать нельзя.
