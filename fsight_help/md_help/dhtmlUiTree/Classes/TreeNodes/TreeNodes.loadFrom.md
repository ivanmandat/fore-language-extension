# TreeList.loadFrom


## Синтаксис


loadFrom (jsonData);


## Параметры


jsonData. Массив JSON-объектов с полями, соответствующими свойствам класса PP.Ui.[TreeNode](../TreeNode/TreeNode.htm).


## Описание


Метод loadFrom вставляет массив элементов дерева в [вершину](../TreeNode/TreeNode.htm), удаляя все ее дочерние вершины, которые были до этого.
