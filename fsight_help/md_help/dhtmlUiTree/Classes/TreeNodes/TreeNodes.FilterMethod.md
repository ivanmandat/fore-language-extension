# TreeNodes.FilterMethod


## Синтаксис


FilterMethod: Function;


## Описание


Свойство FilterMethod определяет пользовательскую функцию для фильтрации вершин иерархического дерева.


## Комментарии


Значение свойства устанавливается с помощью метода setFilterMethod. Из JSON значение задать нельзя.
