# TreeList.EnableEdit


## Синтаксис


EnableEdit: Boolean


## Описание


Свойство EnableEdit определяет, возможно ли редактирование значений в ячейках таблицы.


## Комментарии


Если для свойства установлено значение true, значит ячейки таблицы можно редактировать. По умолчанию установлено значение false.


Чтобы перейти в режим редактирования, дважды щёлкните по ячейке. Между первым и вторым щелчком должен быть небольшой временной промежуток. Ячейка в режиме редактирования будет иметь следующий вид:


![](../../Components/TreeList/TreeList.EnableEdit.gif)


Чтобы определить, будет ли выделяться текст в режиме редактирования, используйте свойство [TreeList.SelectEditorText](TreeList.SelectEditorText.htm).
