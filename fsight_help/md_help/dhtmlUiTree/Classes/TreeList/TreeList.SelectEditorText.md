# TreeList.SelectEditorText


## Синтаксис


SelectEditorText: Boolean


## Описание


Свойство SelectEditorText определяет, будет ли выделяться редактируемый текст.


## Комментарии


Свойство актуально, если для свойства [TreeList.EnableEdit](TreeList.EnableEdit.htm) установлено значение true.


Чтобы перейти в режим редактирования, дважды щёлкните по ячейке. Между первым и вторым щелчком должен быть небольшой временной промежуток.


Если для свойства установлено значение true, то при переходе в режим редактирования текст будет выделяться:


![](SelectedEditorText_true.gif)


Если установлено значение false (по умолчанию), в режиме редактирования текст не выделяется:


![](SelectedEditorText_false.gif)
