# TreeList.EnableSearch


## Синтаксис


EnableSearch: Boolean


## Описание


Свойство EnableSearch определяет возможность поиска элементов дерева.


## Комментарии


Если для свойства установлено значение true, то возможен поиск элементов дерева, иначе -невозможен.


Чтобы открыть строку поиска, используйте сочетание клавиш CTRL+F. Строка поиска появится в нижней части дерева:
