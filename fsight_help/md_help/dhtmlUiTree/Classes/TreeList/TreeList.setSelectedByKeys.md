# TreeList.setSelectedByKeys


## Синтаксис


setSelectedByKeys (keys, clearSelection, fireEvents);


## Параметры


keys. Массив ключей вершин, которые нужно выделить;


clearSelection. Определяет, снимать ли предыдущее выделение. Если установлено значение true, то предыдущее выделение снимается. По умолчанию предыдущее выделение остается;


fireEvents. Определяет, будет ли генерироваться событие [TreeList.SelectionChanged](TreeList.SelectionChanged.htm) при вызове метода.


## Описание


Метод setSelectedByKeys устанавливает отметку для всех вершин дерева с указанными ключами.


## Пример


Пример использования метода приведен в описании свойства [TreeList.EnableHighlight](TreeList.EnableHighlight.htm).
