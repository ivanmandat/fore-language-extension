# TreeList.SortChanging


## Синтаксис


SortChanging: function (sender, args);


## Параметры


sender. Источник
 события;


args. Информация
 о событии. Доступные аргументы: Column - индекс столбца, у которого меняется
 сортировка; Method - способ сортировки, элемент перечисления
 [PP.SortDirection](dhtmlCommon.chm::/Enums/SortDirection.htm);
 Property - имя свойства, по которому будет проводиться
 сортировка (только для первого столбца); Recursive - признак сортировки дочерних элементов:
 true (по умолчанию) - сортировка будет применена
 ко всем дочерним ветвям, false - дочерние ветви могут иметь собственные
 способы сортировки; Cancel - признак
 отмены установки/снятия флажка для изменения сортировки столбца: true
 - операция отменена, false (по умолчанию) - операция не отменена.


## Описание


Событие SortChanging наступает
 во время изменения сортировки столбца иерархического дерева.


## Комментарии


Сортировка столбцов доступна, если для свойства [TreeList.EnableSortColumns](TreeList.EnableSortColumns.htm)
 установлено значение true.


Чтобы произвести сортировку столбца, необходимо нажать на заголовок
 столбца, который необходимо отсортировать.
