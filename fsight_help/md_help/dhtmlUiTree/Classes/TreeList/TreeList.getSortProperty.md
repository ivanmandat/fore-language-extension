# TreeList.getSortProperty


## Синтаксис


getSortProperty ();


## Описание


Метод getSortProperty возвращает название свойства [вершины](../TreeNode/TreeNode.htm), по которому будет производиться сортировка.


## Комментарии


Свойство, по которому будет производиться сортировка, задается в параметре property метода [TreeList.sort](TreeList.Sort.htm). Например, если выбрано свойство [TreeNode.Checked](../TreeNode/TreeNode.Checked.htm), то вершины будут сортироваться с учетом того, установлен ли флажок.
