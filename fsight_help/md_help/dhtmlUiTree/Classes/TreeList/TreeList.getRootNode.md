# TreeList.getRootNode


## Синтаксис


getRootNode ();


## Описание


Метод getRootNode возвращает объект, ассоциированный с корнем дерева.


## Комментарии


Метод может использоваться для получения всех вершин компонента: treeList.getRootNode().getNodes(). Результат будет аналогичен результату выполнения метода treeList.get[Nodes](TreeList.Nodes.htm)().


## Пример


Пример использования аналогичен использованию метода treeList.get[Nodes](TreeList.Nodes.htm)().
