# TreeList.CaptionVisible


## Синтаксис


CaptionVisible: Boolean


## Описание


Свойство CaptionVisible определяет, отображается ли строка заголовков.


## Комментарии


Если для свойства установлено значение true, строка заголовков отображается, иначе - не отображается. По умолчанию установлено значение false.
