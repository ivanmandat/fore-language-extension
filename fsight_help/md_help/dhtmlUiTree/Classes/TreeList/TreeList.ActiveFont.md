# TreeList.ActiveFont


## Синтаксис


ActiveFont: PP.[Font](dhtmlCommon.chm::/Classes/PP/Font/Font.htm)


## Описание


Свойство ActiveFont определяет
 параметры шрифта активной вершины дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setActiveFont,
 а возвращается с помощью метода getActiveFont.
