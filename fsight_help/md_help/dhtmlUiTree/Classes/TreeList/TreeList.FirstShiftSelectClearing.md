# TreeList.FirstShiftSelectClearing


## Синтаксис


FirstShiftSelectClearing: Boolean


## Описание


Свойство FirstShiftSelectClearing
 устанавливает режим, при котором первое нажатие кнопки мыши по вершине
 с зажатой кнопкой SHIFT снимает выделение со всех вершин, кроме нажатой.


## Комментарии


Допустимые значения:


	- true. Режим активен;


	- false. Режим не активен.


Значение свойства устанавливается из JSON и с помощью метода setFirstShiftSelectClearing,
 а возвращается с помощью метода getFirstShiftSelectClearing.
