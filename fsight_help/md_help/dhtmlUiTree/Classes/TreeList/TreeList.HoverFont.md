# TreeList.HoverFont


## Синтаксис


HoverFont: PP.[Font](dhtmlCommon.chm::/Classes/PP/Font/Font.htm)


## Описание


Свойство HoverFont определяет
 параметры шрифта вершины дерева, на которую наведен курсор.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHoverFont,
 а возвращается с помощью метода getHoverFont.
