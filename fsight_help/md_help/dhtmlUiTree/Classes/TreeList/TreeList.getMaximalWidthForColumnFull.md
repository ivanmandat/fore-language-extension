# TreeList.getMaximalWidthForColumnFull


## Синтаксис


getMaximalWidthForColumnFull (index);


## Параметры


index. Индекс столбца.


## Описание


Метод getMaximalWidthForColumnFull возвращает максимальную ширину содержимого в столбце, во всех строках.
