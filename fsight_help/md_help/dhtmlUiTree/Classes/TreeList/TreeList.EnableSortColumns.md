# TreeList.EnableSortColumns


## Синтаксис


EnableSortColumns: Boolean


## Описание


Свойство EnableSortColumns определяет, возможна ли сортировка в столбцах.


## Комментарии


Если для свойства установлено значение true (по умолчанию), то можно осуществлять сортировку по возрастанию и убыванию при нажатии на заголовок столбца. При этом в заголовке появляются соответствующие пиктограммы:

![](../../Components/TreeList/treelist18.gif) - по возрастанию.
