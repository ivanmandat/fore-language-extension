# TreeList.DblClickExpand


## Синтаксис


DblClickExpand: Boolean


## Описание


Свойство DblClickExpand определяет, раскрывается ли вершина дерева по двойному щелчку мыши.


## Комментарии


Если для свойства установлено значение false, то вершина дерева раскрывается только при одинарном щелчке. Если установлено значение true (по умолчанию), то вершина раскрывается и при двойном, и при одинарном щелчке.
