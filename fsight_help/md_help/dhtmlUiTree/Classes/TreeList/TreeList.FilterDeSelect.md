# TreeList.FilterDeSelect


## Синтаксис


FilterDeSelect: Boolean


## Описание


Свойство FilterDeSelect определяет
 сброс выделения вершин при фильтрации.


## Комментарии


Допустимые значения:


	- true. Во время фильтрации
	 произойдёт сброс выделения вершин;


	- false. По умолчанию. Во время фильтрации сброс
	 выделения вершин не произойдёт.


Значение свойства устанавливается из JSON и с помощью метода setFilterDeSelect,
 а возвращается с помощью метода getFilterDeSelect.
