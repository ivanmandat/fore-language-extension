# TreeList.expandAll


## Синтаксис


expandAll ();


## Описание


Метод expandAll разворачивает все вершины дерева.


## Комментарии


Чтобы свернуть все вершины, используйте метод [TreeList.collapseAll](TreeList.collapseAll.htm).
