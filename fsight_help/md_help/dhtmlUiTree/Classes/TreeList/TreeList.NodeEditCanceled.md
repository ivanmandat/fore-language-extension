# TreeList.NodeEditCanceled


## Синтаксис


NodeEditCanceled: function (sender,args)


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: Node - редактируемая вершина, Column - столбец, в котором осуществляется редактирование, Esc -редактирование отменено при помощи клавиши ESC.


## Описание


Событие NodeEditCanceled наступает при отмене редактирования.


## Комментарии


Событие наступает, если содержимое ячейки еще не было изменено, а только был осуществлен переход в режим редактирования.


Чтобы отменить редактирование, то есть выйти из режима редактирования, щёлкните по пространству вне редактируемой ячейки или нажмите клавишу ESC.
