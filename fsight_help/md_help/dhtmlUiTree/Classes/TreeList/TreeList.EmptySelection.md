# TreeList.EmptySelection


## Синтаксис


EmptySelection: Boolean


## Описание


Свойство EmptySelection определяет, можно ли снимать отметку со всех вершин дерева.


## Комментарии


Если установлено значение true, значит можно снять отметку со всех вершин дерева. По умолчанию установлено значение false, то есть при снятии отметки со всех вершин, одна вершина остается отмеченной.
