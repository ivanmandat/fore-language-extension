# TreeList.getVerticalScroll


## Синтаксис


getVerticalScroll();


## Описание


Метод getVerticalScroll возвращает вертикальную полосу прокрутки.


## Комментарии


Полосы прокрутки появляются, когда размеры содержимого превышают размеры контейнера.
