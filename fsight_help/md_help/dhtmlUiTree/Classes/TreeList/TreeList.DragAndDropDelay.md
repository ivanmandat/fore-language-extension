# TreeList.DragAndDropDelay


## Синтаксис


DragAndDropDelay: Number


## Описание


Свойство DragAndDropDelay определяет, сколько времени нужно продержать курсор над вершиной с одновременным нажатием на кнопку мыши, чтобы активировался режим Drag&Drop.


## Комментарии


Если отпустить курсор до того, как истечет установленное время, режим Drag&Drop активирован не будет.


Свойство актуально, если для [TreeList.EnableDragAndDrop](TreeList.EnableDragAndDrop.htm) установлено значение true.
