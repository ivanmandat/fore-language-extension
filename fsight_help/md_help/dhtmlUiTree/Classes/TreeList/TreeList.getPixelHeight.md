# TreeList.getPixelHeight


## Синтаксис


getPixelHeight();


## Описание


Метод getPixelHeight возвращает
 значение высоты (в пикселях) компонента дерева.


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [TreeList](../../Components/TreeList/TreeList.htm) c наименованием
 «treeListSett» (см. «[Пример
 создания компонента TreeList](../../Components/TreeList/TreeList_example.htm)»). Получаем высоту компонента дерева
 в пикселях:


// Получаем высоту компонента дерева в пикселях
console.log("Высота компонента: " + treeListSett.getPixelHeight());
В результате в консоль будет выведена высота компонента дерева:


Высота компонента: 125
