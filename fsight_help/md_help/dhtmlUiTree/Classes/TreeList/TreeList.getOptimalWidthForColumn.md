# TreeList.getOptimalWidthForColumn


## Синтаксис


getOptimalWidthForColumn(index);


## Параметры


index. Индекс столбца.


## Описание


Метод getOptimalWidthForColumn возвращает оптимальную ширину столбца.


## Комментарии


Оптимальная ширина столбца - это ширина ячейки с самым длинным текстом в указанном столбце.
