# Конструктор TreeList


## Синтаксис


PP.Ui.TreeList(settings);


## Параметры


settings. JSON-объект со значениями
 свойств компонента.


## Описание


Конструктор TreeList создает
 экземпляр компонента [TreeList](../../Components/TreeList/TreeList.htm).
