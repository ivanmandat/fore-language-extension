# TreeList.getHorizontalScroll


## Синтаксис


getHorizontalScroll();


## Описание


Метод getHorizontalScroll возвращает горизонтальную полосу прокрутки.


## Комментарии


Полосы прокрутки появляются, когда размеры содержимого превышают размеры контейнера.
