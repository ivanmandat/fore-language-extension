# TreeList.ToolTipShowing


## Синтаксис


ToolTipShowing: function(sender, args);


## Параметры


sender. Источник события;


args. Информация
 о событии. Доступные аргументы:
 Node - вершина, у которой всплывает подсказка,
 ToolTip - всплывающая подсказка, Column - номер столбца, в котором находится вершина,
 у которой вызывается подсказка.


## Описание


Событие ToolTipShowing наступает,
 когда курсор мыши наведен на вершину иерархического дерева.


## Комментарии


Событие актуально, если для свойства[
 TreeList.ShowtoolTip](TreeList.ShowToolTips.htm) установлено значение true.


## Пример


Пример использования события приведен в описании свойства [TreeList.HoverColor](TreeList.HoverColor.htm).
