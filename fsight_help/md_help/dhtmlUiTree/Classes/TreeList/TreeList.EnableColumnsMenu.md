# TreeList.EnableColumnsMenu


## Синтаксис


EnableColumnsMenu: Boolean


## Описание


Свойство EnableColumnsMenu определяет
 видимость пиктограммы меню в заголовке дерева и возможность вызова меню,
 управляющее видимостью колонок.


## Комментарии


Допустимые значения:


	- true. По умолчанию.
	 Пиктограмма меню дерева отображается и возможен вызов меню;


	- false. Пиктограмма отсутствует и вызвать меню
	 невозможно.


Значение свойства устанавливается из JSON и с помощью метода setEnableColumnsMenu,
 а возвращается с помощью метода getEnableColumnsMenu.
