# TreeList.collapseAll


## Синтаксис


collapseAll ();


## Описание


Метод collapseAll сворачивает все вершины дерева.


## Комментарии


Чтобы развернуть все вершины, используйте метод [TreeList.expandAll](TreeList.expandAll.htm).
