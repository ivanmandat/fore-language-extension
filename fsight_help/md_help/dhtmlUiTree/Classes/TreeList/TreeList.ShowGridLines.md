# TreeList.ShowGridLines


## Синтаксис


ShowGridLines: Boolean


## Описание


Свойство ShowGridLines определяет, отображается ли сетка таблицы.


## Комментарии


Если для свойства установлено значение true, отображается табличная сетка, иначе - не отображается. По умолчанию установлено значение false.
