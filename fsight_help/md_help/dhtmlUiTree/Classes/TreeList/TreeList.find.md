# TreeList.find


## Синтаксис


find (value, column, setFocus, fireEvents);


## Параметры


value. Искомая строка;


column. Индекс столбца, по которому осуществляется поиск. По умолчанию первый столбец (с индексом 0);


setFocus. Определяет, будет ли установлен фокус на найденном элементе. По умолчанию - true, то есть фокус будет установлен;


fireEvents. Определяет, будет ли при выполнении метода генерироваться событие [TreeList.Finding](TreeList.Finding.htm). По умолчанию true, то есть событие генерируется.


## Описание


Метод find осуществляет поиск элемента дерева.


## Комментарии


Поиск возможен, если для свойства [TreeList.EnableSearch](TreeList.EnableSearch.htm) установлено значение true.

Пример использования метода приведен в описании события [TreeList.Finding](TreeList.Finding.htm).
