# TreeList.IgnoreRightPadding


## Синтаксис


IgnoreRightPadding: Boolean


## Описание


Свойство IgnoreRightPadding
 определяет игнорирование правого отступа.


## Комментарии


Допустимые значения:


	- true. Правый отступ
	 будет проигнорирован;


	- false. По умолчанию. Правый отступ не будет
	 проигнорирован.


Значение свойства устанавливается из JSON и с помощью метода setIgnoreRightPadding,
 а возвращается с помощью метода getIgnoreRightPadding.
