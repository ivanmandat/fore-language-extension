# TreeList.SelectedHighlightColor


## Синтаксис


SelectedHighlightColor: PP.[Color](dhtmlCommon.chm::/Classes/PP/Color/Color.htm)||String


## Описание


Свойство SelectedHighlightColor
 определяет цвет подсветки выделенных вершин дерева.


## Комментарии


Свойство актуально, если для свойства [TreeList.EnableSelectedHighlight](TreeList.EnableSelectedHighlight.htm)
 установлено значение true.


## Пример


Пример использования приведен в описании свойства [TreeList.EnableSelectedHighlight](TreeList.EnableSelectedHighlight.htm).
