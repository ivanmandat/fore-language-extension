# TreeList.IsCSMultiSelect


## Синтаксис


IsCSMultiSelect: Boolean


## Описание


Свойство IsCSMultiSelect определяет, включена ли возможность множественной отметки при помощи клавиш SHIFT и CTRL.


## Комментарии


Если для свойства установлено значение true, то возможна множественная отметка вершин дерева при удержании клавиши SHIFT или CTRL. При обычном щелчке выделяется новая вершина, а со старой отметка снимается. Если установлено значение false (по умолчанию), выделить несколько вершин с удержанием клавиши SHIFT или CTRL нельзя.
