# TreeList.SortSequence


## Синтаксис


SortSequence: Array


## Описание


Свойство SortSequence определяет порядок изменения сортировки.


## Комментарии


Свойство актуально, если для свойства [TreeList.EnableSortColumns](TreeList.EnableSortColumns.htm) установлено значение true.


По умолчанию установлен массив [0,1,2], то есть сначала сортировка отсутствует, затем - сортировка по возрастанию, после - по убыванию. Если установлен массив [0,2,1], то сначала сортировка отсутствует, затем по убыванию, после - по возрастанию.


Смена типа сортировки осуществляется при нажатии на заголовок столбца.
