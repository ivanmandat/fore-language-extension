# TreeList.LoadSubNodesOf


## Синтаксис


LoadSubNodesOf: function (sender,arsg)


## Параметры


sender. Источник события;


args. Информация о событии. Доступный аргумент: Node - загруженная вершина.


## Описание


Событие LoadSubNodesOf наступает при загрузке вершины в дерево.


## Комментарии


Событие генерируется при вызове метода TreeList.loadNode.


## Пример


Пример использования события приведен в описании метода TreeList.loadNode.
