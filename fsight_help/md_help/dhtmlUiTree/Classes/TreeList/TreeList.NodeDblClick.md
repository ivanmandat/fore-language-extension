# TreeList.NodeDblClick


## Синтаксис


NodeDblClick: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступный аргумент: Node - выделенная вершина.


## Описание


Событие NodeDblClick наступает при двойном щелчке мыши по элементу списка.
