# TreeList.getAllNodes


## Синтаксис


getAllNodes (includeHidden);


## Параметры


includeHidden. Определяет, будут ли возвращаться скрытые вершины. Если установлено значение true, то метод возвращает все вершины, включая скрытые. Иначе возвращаются только видимые вершины.


## Описание


Метод getAllNodes возвращает одномерный массив, содержащий все вершины дерева с текущим порядком сортировки.
