# TreeList.getMaximalWidthForColumn


## Синтаксис


getMaximalWidthForColumn (index)ж


## Параметры


index. Индекс столбца.


## Описание


Метод getMaximalWidthForColumn возвращает максимальную ширину содержимого в колонке, в видимых строках.
