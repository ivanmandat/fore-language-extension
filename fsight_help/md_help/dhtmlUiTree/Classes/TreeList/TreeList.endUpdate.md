# TreeList.endUpdate


## Синтаксис


endUpdate ();


## Описание


Метод endUpdate обновляет дерево и включает перерисовку.


## Комментарии


Используется в связке с методом [TreeList.beginUpdate](TreeList.beginUpdate.htm).
