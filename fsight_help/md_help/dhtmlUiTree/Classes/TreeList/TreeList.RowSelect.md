# TreeList.RowSelect


## Синтаксис


RowSelect: Boolean


## Описание


Свойство RowSelect определяет, подсвечивается ли при выборе элемента вся строка.


## Комментарии


Если для свойства установлено значение true, при выборе элемента подсвечивается вся строка. Если установлено значение false, подсвечивается только выбранный элемент.


По умолчанию установлено значение false.


## Пример


Для выполнения примера [создайте компонент TreeList](../../Components/TreeList/TreeList_example.htm) с наименованием «treeListSett».


Значение свойства можно установить через JSON-объект:


RowSelect: true


или на уровне API:


treeListSett.setRowSelect(true);
