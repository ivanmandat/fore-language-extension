# TreeList.expandToKeyNode


## Синтаксис


expandToKeyNode (key);


## Параметры


key. Ключ вершины, которую необходимо найти.


## Описание


Метод expandToKeyNode ищет в дереве вершину с указанным ключом и раскрывает дерево на уровне этой вершины.
