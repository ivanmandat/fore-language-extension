# TreeList.EnableHighlight


## Синтаксис


EnableHighlight: Boolean


## Описание


Свойство EnableHighlight определяет, доступна ли подсветка элементов дерева.


## Комментарии


Если для свойства установлено значение true, то доступна подсветка элементов дерева. По умолчанию установлено значение false.
