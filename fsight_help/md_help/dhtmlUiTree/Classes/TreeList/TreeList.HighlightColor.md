# TreeList.HighlightColor


## Синтаксис


HighlightColor: PP.[Color](dhtmlCommon.chm::/Classes/PP/Color/Color.htm)||String


## Описание


Свойство HighlightColor определяет
 цвет подсветки элемента дерева.


## Комментарии


Свойство актуально, если для свойства [TreeList.EnableHighlight](TreeList.EnableHighlight.htm)
 установлено значение true.
