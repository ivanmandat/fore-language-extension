# TreeList.getNodeByKey


## Синтаксис


getNodeByKey (key);


## Параметры


key. Ключ вершины.


## Описание


Метод getNodeByKey возвращает вершину первого уровня по ее ключу.


## Комментарии


Ключ вершины устанавливается при помощи свойства [TreeNode.Key](../TreeNode/TreeNode.Key.htm).
