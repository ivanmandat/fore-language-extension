# TreeList.ContentFont


## Синтаксис


ContentFont: PP.[Font](dhtmlCommon.chm::/Classes/PP/Font/Font.htm)


## Описание


Свойство ContentFont определяет
 параметры шрифта вершин дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setContentFont,
 а возвращается с помощью метода getContentFont.
