# TreeList.ShowToolTips


## Синтаксис


ShowToolTips: Boolean


## Описание


Свойство ShowToolTips определяет
 отображение всплывающих подсказок при наведение курсора мыши на вершину
 дерева.


## Комментарии


Допустимые значения:


	- true. По умолчанию.
	 При наведении на вершину дерева будет отображаться всплывающая подсказка;


	- false. При наведении
	 на вершину дерева всплывающая подсказка отображаться не будет.


Значение свойства устанавливается из JSON и с помощью метода setShowToolTips,
 а возвращается с помощью метода getShowToolTips.
