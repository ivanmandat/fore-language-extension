# TreeList.DragAndDropDetectDistance


## Синтаксис


DragAndDropDetectDistance: Number;


## Описание


Свойство DragAndDropDetectDistance
 определяет дистанцию, после которой включается режим Drag&Drop.


## Комментарии


Дистанция определяется в пикселях.


Значение свойства устанавливается из JSON и с помощью метода setDragAndDropDetectDistance,
 а возвращается с помощью метода getDragAndDropDetectDistance.


Свойство актуально, если для [TreeList.EnableDragAndDrop](TreeList.EnableDragAndDrop.htm)
 установлено значение true.
