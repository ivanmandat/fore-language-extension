# TreeList.HighlightFont


## Синтаксис


HighlightFont: PP.[Font](dhtmlCommon.chm::/Classes/PP/Font/Font.htm)


## Описание


Свойство HighlightFont определяет
 шрифт подсвечиваемой вершины дерева.


## Комментарии


Свойство актуально, если для свойства [TreeList.EnableHighlight](TreeList.EnableHighlight.htm)
 установлено значение true.
