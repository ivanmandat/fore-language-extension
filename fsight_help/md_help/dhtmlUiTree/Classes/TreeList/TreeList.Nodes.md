# TreeList.Nodes


## Синтаксис


Nodes: PP.Ui.[TreeNodes](../TreeNodes/TreeNodes.htm)


## Описание


Свойство Nodes устанавливает
 коллекцию вершин.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setNodes,
 а возвращается с помощью метода getNodes.
