# TreeList.getSortRecursive


## Синтаксис


getSortRecursive ();


## Описание


Метод getSortRecursive определяет, производится ли сортировка в дочерних вершинах.


## Комментарии


Признак того, что сортировка будет производиться в дочерних вершинах, устанавливается в параметре recursive метода [TreeList.sort](TreeList.Sort.htm).
