# TreeList.NodeDrag


## Синтаксис


NodeDrag: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие NodeDrag наступает во время перетаскивания вершины.


## Комментарии


Событие актуально, если для свойства вершины [TreeNode.Draggable](../TreeNode/TreeNode.Draggable.htm) или для свойства [TreeList.EnableDragAndDrop](TreeList.EnableDragAndDrop.htm) дерева установлено значение true.

Пример использования события приведен в описании свойства [TreeNode.Draggable](../TreeNode/TreeNode.Draggable.htm).
