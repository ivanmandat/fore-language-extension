# TreeList.NodeEdited


## Синтаксис


NodeEdited: function(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: Node - редактируемая вершина, Column - столбец, в котором осуществляется редактирование.


## Описание


Событие NodeEdited наступает после редактирования названия вершины.


## Комментарии


Событие актуально, если для свойства [TreeList.EnableEdit](TreeList.EnableEdit.htm) установлено значение true.
