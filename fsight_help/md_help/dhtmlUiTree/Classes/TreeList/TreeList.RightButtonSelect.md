# TreeList.RightButtonSelect


## Синтаксис


RightButtonSelect: Boolean


## Описание


Свойство RightButtonSelect определяет, возможен ли вершины по щелчку правой кнопкой мыши.


## Комментарии


Если для свойства установлено значение true, то выбор вершины может осуществляться как по щелчку левой кнопки мыши, так и правой. Если установлено значение false (по умолчанию), то выбор вершины осуществляется только по щелчку леовй кнопки мыши.
