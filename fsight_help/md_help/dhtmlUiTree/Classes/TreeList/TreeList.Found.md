# TreeList.Found


## Синтаксис


Found: function (sender,args)


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы:


-
Node - найденная вершина, экземпляр PP.Ui.[TreeNode](../TreeNode/TreeNode.htm);


-
Value - значение, введенное в строке поиска;


-
Column - возвращает индекс столбца, по которому осуществлялся поиск;


-
SetFocus - определяет, будет ли установлен фокус на найденном элементе. По умолчанию - true, то есть фокус будет установлен.


## Описание


Событие Found наступает, когда найден элемент дерева.


## Комментарии


Событие актуально, если для свойства [TreeList.EnableSearch](TreeList.EnableSearch.htm) установлено значение true.
