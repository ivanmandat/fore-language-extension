# TreeList.LoadingTimeout


## Синтаксис


LoadingTimeout: Number


## Описание


Свойство LoadingTimeout определяет
 промежуток времени, через который произойдет загрузка дерева.


## Комментарии


Промежуток времени задается в миллисекундах.


Значение свойства устанавливается из JSON и с помощью метода setLoadingTimeout,
 а возвращается с помощью метода getLoadingTimeout.
