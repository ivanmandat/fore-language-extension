# TreeList.MultiSelect


## Синтаксис


MultiSelect: Boolean


## Описание


Свойство MultiSelect определяет возможность множественной отметки элементов.


## Комментарии


Если для свойства установлено значение true, при выборе элемента списка сохраняется выделение выбранных ранее элементов. Если установлено значение false, может быть выделен только один элемент. В этом случае можно выбрать несколько элементов при помощи клавиш «[SHIFT](TreeList.UseShiftSelect.htm)» и «[CTRL](TreeList.UseCtrlSelect.htm)».


По умолчанию установлено значение false.
