# TreeList.KeepHasChild


## Синтаксис


KeepHasChild: Boolean


## Описание


Свойство KeepHasChild определяет
 изменение отображения иерархического дерева и изменение свойства HasChild
 у вершин во время работы режима Drag&Drop.


## Комментарии


Допустимые значения:


	- true. Отображение дерева
	 и свойство hasChild вершин могут меняться во время работы режима Drag&Drop;


	- false. По умолчанию. Отображение дерева и свойство
	 hasChild вершин не меняются во время работы режима Drag&Drop.


Значение свойства устанавливается из JSON и с помощью метода setKeepHasChild,
 а возвращается с помощью метода getKeepHasChild.
