# TreeList.EnableDragAndDrop


## Синтаксис


EnableDragAndDrop: Boolean


## Описание


Свойство EnableDragAndDrop определяет, возможно ли перемещать вершины при помощи механизма Drag&Drop.


## Комментарии


Если для свойства установлено значение true, то можно перемещать вершины при помощи механизма Drag&Drop, иначе - нельзя. По умолчанию установлено значение false.


Чтобы переместить вершину при помощи механизма Drag&Drop, щёлкните по вершине и задержите курсор на некоторое время (время задержки можно определить в свойстве [TreeList.DragAndDropDelay](TreeList.DragAndDropDelay.htm)). Когда вершина будет подсвечена, можно ее перемещать:
