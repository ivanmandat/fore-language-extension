# TreeList.getPixelWidth


## Синтаксис


getPixelWidth();


## Описание


Метод getPixelWidth возвращает
 значение ширины (в пикселях) компонента дерева.


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [TreeList](../../Components/TreeList/TreeList.htm) c наименованием
 «treeListSett» (см. «[Пример
 создания компонента TreeList](../../Components/TreeList/TreeList_example.htm)»). Получаем ширину компонента дерева
 в пикселях:


// Получаем ширину компонента дерева в пикселях
console.log("Ширина компонента: " + treeListSett.getPixelWidth());
В результате в консоль будет выведена ширина компонента дерева:


Ширина компонента: 400
