# TreeList.FilterCaseSensitive


## Синтаксис


FilterCaseSensitive: Boolean


## Описание


Свойство FilterCaseSensitive определяет, будет ли [фильтрация](../TreeNodes/TreeNodes.filter.htm) регистрозависимой.


## Комментарии


Если установлено значение true (по умолчанию), то при фильтрации учитывается регистр вводимых символов, иначе - не учитывается.
