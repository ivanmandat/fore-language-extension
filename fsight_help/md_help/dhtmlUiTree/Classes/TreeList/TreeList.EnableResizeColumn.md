# TreeList.EnableResizeColumn


## Синтаксис


EnableResizeColumn: Boolean


## Описание


Свойство EnableResizeColumn определяет, возможно ли изменять ширину столбцов.


## Комментарии


Если для свойства установлено значение false, изменение ширины столбцов недоступно. Если установлено значение true, можно изменять ширину столбцов при помощи двухконечной стрелки, которая появляется при наведении курсора на границы столбцов в строке заголовков.


По умолчанию установлено значение true.
