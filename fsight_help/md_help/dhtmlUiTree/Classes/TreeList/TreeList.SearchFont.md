# TreeList.SearchFont


## Синтаксис


ActiveFont: PP.[Font](dhtmlCommon.chm::/Classes/PP/Font/Font.htm)


## Описание


Свойство SearchFont определяет
 параметры шрифта искомого текста.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setSearchFont,
 а возвращается с помощью метода getSearchFont.
