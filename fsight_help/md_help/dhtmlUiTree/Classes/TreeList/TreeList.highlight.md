# TreeList.highlight


## Синтаксис


highlight(text: String, caseSensitive: Boolean,
 columns: Array|Number);


## Параметры


text. Подсвечиваемый текст;


caseSensitive. Параметр определяет
 чувствительность к регистру во время поиска вершины. Если необходимо найти
 вершину с учетом регистра, то для свойства должно быть установлено значение
 true, иначе false;


columns. Номер колонки или
 массив колонок, в которых происходит поиск текста и подсвечивание вершины.


## Описание


Метод highlight подсвечивает
 указанный текст в подписях вершин дерева.
