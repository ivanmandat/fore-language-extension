# TreeListBase.getNodeExist


## Синтаксис


getNodeExist(nodeKey);


## Параметры


nodeKey. Number. Ключ вершины.


## Описание


Метод getNodeExist возвращает
 признак существования указанной вершины.


## Пример


Для выполнения примера необходим компонент SimpleTreeList с идентификатором
 «Tree» (см. «[Пример
 создания компонента SimpleTreeList](../../Components/SimpleTreeList/SimpleTreeList_Example.htm)»).


// Получим признак существования вершины
Tree.getNodeExist("key10");
// -> false
В результате выполнения примера будет получен признак существования
 указанной вершины.
