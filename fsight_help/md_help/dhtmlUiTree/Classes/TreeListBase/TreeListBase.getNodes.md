# TreeListBase.getNodes


## Синтаксис


getNodes();


## Описание


Метод getNodes возвращает массив
 предназначенных для отрисовки вершин.


## Пример


Для выполнения примера необходим компонент SimpleTreeList с идентификатором
 «Tree» (см. «[Пример
 создания компонента SimpleTreeList](../../Components/SimpleTreeList/SimpleTreeList_Example.htm)»).


// Получим отрисованные вершины
Tree.getNodes();
// -> ["0", "1", "2", "key3"]
В результате выполнения примера будет получен массив отрисованных вершин.
