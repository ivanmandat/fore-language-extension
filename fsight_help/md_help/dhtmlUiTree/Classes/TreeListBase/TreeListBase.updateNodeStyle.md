# TreeListBase.updateNodeStyle


## Синтаксис


updateNodeStyle(nodeKey);


## Параметры


nodeKey. String. Ключ вершины.


## Описание


Метод updateNodeStyle обновляет
 стили указанной вершины.


## Пример


Для выполнения примера необходим компонент SimpleTreeList с идентификатором
 «Tree» (см. «[Пример
 создания компонента SimpleTreeList](../../Components/SimpleTreeList/SimpleTreeList_Example.htm)»).


// Обновим стили вершины
Tree.updateNodeStyle(2);
В результате выполнения примера будут обновлены стили вершины.
