# TreeListBase.loadNodes


## Синтаксис


loadNodes(nodes, rootKey);


## Параметры


nodes. Array of Object. Массив
 вершин;


rootKey. String. Ключ вершины
 для вставки. Если не указан, вершины будут добавлены в корень.


## Описание


Метод loadNodes загружает вершины
 из объекта.
