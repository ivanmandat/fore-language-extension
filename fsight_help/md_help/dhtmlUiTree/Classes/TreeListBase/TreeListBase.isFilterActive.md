# TreeListBase.isFilterActive


## Синтаксис


isFilterActive();


## Описание


Метод isFilterActive возвращает
 признак использования фильтра.


## Пример


Для выполнения примера необходим компонент SimpleTreeList с идентификатором
 «Tree» (см. «[Пример
 создания компонента SimpleTreeList](../../Components/SimpleTreeList/SimpleTreeList_Example.htm)»).


// Проверим активен ли фильтр
Tree.isFilterActive();
// -> false
В результате выполнения примера будет получен признак использования
 фильтра.
