# TreeListBase.setSort


## Синтаксис


setSort(column, direction, method);


## Параметры


column. Number. Номер столбца;


direction. PP.Ui.TreeSortDirection.
 Направление сортировки;


method. PP.Ui.TreeSortMethod.
 Метод сортировки.


## Описание


Метод setSort устанавливает
 сортировку элементов дерева.
