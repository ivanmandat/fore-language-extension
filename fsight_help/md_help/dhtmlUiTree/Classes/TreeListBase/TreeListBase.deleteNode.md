# TreeListBase.deleteNode


## Синтаксис


deleteNode(nodeKey);


## Параметры


nodeKey. String. Ключ вершины.


## Описание


Метод deleteNode удаляет указанную
 вершину из дерева.


## Пример


Для выполнения примера необходим компонент SimpleTreeList с идентификатором
 «Tree» (см. «[Пример
 создания компонента SimpleTreeList](../../Components/SimpleTreeList/SimpleTreeList_Example.htm)»).


// Удалим вторую вершину
Tree.deleteNode(1)
В результате выполнения примера была удалена вторая вершина.
