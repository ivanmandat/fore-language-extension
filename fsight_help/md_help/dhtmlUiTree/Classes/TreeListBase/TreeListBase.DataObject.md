# TreeListBase.DataObject


## Синтаксис


DataObject: Array;


## Описание


Свойство DataObject определяет
 источник данных дерева.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDataObject,
 а возвращается с помощью метода getDataObject.
