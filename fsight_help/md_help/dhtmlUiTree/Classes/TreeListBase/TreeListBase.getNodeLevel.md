# TreeListBase.getNodeLevel


## Синтаксис


getNodeLevel(nodeKey);


## Параметры


nodeKey. Number. Ключ вершины.


## Описание


Метод getNodeLevel возвращает
 уровень указанной вершины.


## Пример


Для выполнения примера необходим компонент SimpleTreeList с идентификатором
 «Tree» (см. «[Пример
 создания компонента SimpleTreeList](../../Components/SimpleTreeList/SimpleTreeList_Example.htm)»).


// Получим уровень вершины
Tree.getNodeLevel(4);
// -> 1
В результате выполнения примера будет получен уровень указанной вершины.
