# TreeListBase.endUpdate


## Синтаксис


endUpdate(redraw);


## Параметры


redraw. Boolean. Флаг перерисовки
 дерева. По умолчанию true. Необязательный
 параметр.


## Описание


Метод endUpdate снимает флаг
 блокировки обновления.


## Комментарии


Если флаг блокировки равен или меньше 0 - дерево будет обновлено и разрешено
 дальнейшее обновление.
