# TreeListBase.getNodeByPosition


## Синтаксис


getNodeByPosition(index);


## Параметры


index. Number. Порядковый номер
 вершины.


## Описание


Метод getNodeByPosition возвращает
 ключ вершины по её порядковому номеру.


## Пример


Для выполнения примера необходим компонент SimpleTreeList с идентификатором
 «Tree» (см. «[Пример
 создания компонента SimpleTreeList](../../Components/SimpleTreeList/SimpleTreeList_Example.htm)»).


// Получим ключ третьей вершины
Tree.getNodeByPosition(3);
// -> "key3"
В результате выполнения примера будет получен ключ третьей вершины.
