# TreeListBase.preserveSelection


## Синтаксис


preserveSelection(keep, invert);


## Параметры


keep. Boolean. Сохранять текущее
 выделение;


invert. Boolean. Инвертировать
 текущее выделение.


## Описание


Метод preserveSelection запоминает
 текущее выделение для метода setSelectedFromToIndex.


## Пример


Использование метода приведено в описании метода [setSelectedFromToIndex](TreeListBase.setSelectedFromToIndex.htm).
