# TreeListBase.getNodePosition


## Синтаксис


getNodePosition(nodeKey);


## Параметры


nodeKey. Number. Ключ вершины.


## Описание


Метод getNodePosition возвращает
 порядковый номер указанной вершины.


## Пример


Для выполнения примера необходим компонент SimpleTreeList с идентификатором
 «Tree» (см. «[Пример
 создания компонента SimpleTreeList](../../Components/SimpleTreeList/SimpleTreeList_Example.htm)»).


// Получим порядковый номер вершины
Tree.getNodePosition(4);
// -> 1
В результате выполнения примера будет получен порядковый номер указанной
 вершины.
