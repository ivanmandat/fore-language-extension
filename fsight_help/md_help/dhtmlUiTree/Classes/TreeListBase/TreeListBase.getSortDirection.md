# TreeListBase.SortDirection


## Синтаксис


SortDirection: PP.Ui.TreeSortDirection;


## Описание


Свойство SortDirection определяет
 направление сортировки.


## Комментарии


Значение свойства устанавливается через JSON, а возвращается с помощью
 метода getSortDirection.


Значение по умолчанию TreeSortDirection.None.
