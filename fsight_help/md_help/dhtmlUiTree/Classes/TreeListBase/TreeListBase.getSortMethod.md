# TreeListBase.SortMethod


## Синтаксис


SortMethod: PP.Ui.TreeSortMethod;


## Описание


Свойство SortMethod определяет
 метод сортировки.


## Комментарии


Значение свойства устанавливается через JSON, а возвращается с помощью
 метода getSortMethod.


Значение по умолчанию TreeSortMethod.Number.
