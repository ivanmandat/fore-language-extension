# TreeListBase.clearNodes


## Синтаксис


clearNodes();


## Описание


Метод clearNodes удаляет все
 вершины дерева.


## Пример


Для выполнения примера необходим компонент SimpleTreeList с идентификатором
 «Tree» (см. «[Пример
 создания компонента SimpleTreeList](../../Components/SimpleTreeList/SimpleTreeList_Example.htm)»).


// Удалим все вершины дерева
Tree.clearNodes();
В результате выполнения примера все вершины будут удалены.
