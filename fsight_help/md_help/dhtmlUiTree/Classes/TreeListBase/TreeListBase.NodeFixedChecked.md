# TreeListBase.NodeFixedChecked


## Синтаксис


NodeFixedChecked: Boolean;


## Описание


Свойство NodeFixedChecked определяет
 параметр сопоставления данных «FixedChecked» указанной вершины.


## Комментарии


Свойство не используется в текущей реализации класса.


Значение свойства устанавливается с помощью метода setNodeFixedChecked(nodeKey,
 value), а возвращается с помощью метода getNodeFixedChecked(nodeKey).


Параметры:


nodeKey. String. Идентификатор
 вершины;


value. Boolean. Значение.
