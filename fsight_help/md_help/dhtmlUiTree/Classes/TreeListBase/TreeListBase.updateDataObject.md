# TreeListBase.updateDataObject


## Синтаксис


updateDataObject();


## Описание


Метод updateDataObject создает
 пустой объект с данными при его отсутствии.


## Пример


Метод предназначен для внутренней реализации. Для изменения объекта
 с данными используйте свойство [TreeListBase.DataObject](TreeListBase.DataObject.htm).
