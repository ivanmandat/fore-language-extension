# ListCaptions.getVisibleCaptionsCount


## Синтаксис


getVisibleCaptionsCount();


## Описание


Метод getVisibleCaptionsCount
 возвращает количество видимых заголовков.


## Пример


Для выполнения примера необходим компонент SimpleTreeList с идентификатором
 «Tree» (см. «[Пример
 создания компонента SimpleTreeList](../../Components/SimpleTreeList/SimpleTreeList_Example.htm)»).


// Получим заголовки дерева
caps = Tree.getCaptions();
// Получим количества видимых заголовков
caps.getVisibleCaptionsCount();
В результате выполнения примера было получено количество видимых заголовков.
