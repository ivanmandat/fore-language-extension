# ListCaptions.hideMenu


## Синтаксис


hideMenu();


## Описание


Метод hideMenu скрывает меню
 видимости столбцов.


## Пример


Для выполнения примера необходим компонент SimpleTreeList с идентификатором
 «Tree» (см. «[Пример
 создания компонента SimpleTreeList](../../Components/SimpleTreeList/SimpleTreeList_Example.htm)»).


// Получим заголовки дерева
caps = Tree.getCaptions();
// скроем меню видимости столбцов
caps.hideMenu();
В результате выполнения примера была скрыта кнопка меню.
