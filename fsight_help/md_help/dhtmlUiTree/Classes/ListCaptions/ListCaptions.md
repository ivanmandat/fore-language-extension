## Описание


Класс ListCaptions реализует
 заголовок дерева, состоящий из заголовков столбцов.


## Комментарии


Состоит из группы заголовков: [PP.Ui.ListCaption](../ListCaption/ListCaption.htm).


Используется в качестве заголовка для следующих компонентов:


	- [PP.Ui.TreeListBox](../../Components/TreeListBox/TreeListBox.htm)


	- [PP.Ui.WrappedTreeList](../../Components/WrappedTreeList/WrappedTreeList.htm)


	- [PP.Ui.DynamicTreeList](../../Components/DynamicTreeList/DynamicTreeList.htm)


	- [PP.Ui.SimpleTreeList](../../Components/SimpleTreeList/SimpleTreeList.htm)


## Синтаксис


PP.initClass(PP.Ui.ListCaptions, PP.Ui.[Control](dhtmlUi.chm::/Classes/control/control.htm),
 "ListCaptions");
