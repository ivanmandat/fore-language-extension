# ListCaptions.getCaptionsWidth


## Синтаксис


getCaptionsWidth();


## Описание


Метод getCaptionsWidth возвращает
 общую ширину всех заголовков без учета отступов.


## Пример


Для выполнения примера необходим компонент SimpleTreeList с идентификатором
 «Tree» (см. «[Пример
 создания компонента SimpleTreeList](../../Components/SimpleTreeList/SimpleTreeList_Example.htm)»).


// Получим заголовки дерева
caps = Tree.getCaptions();
// Получим ширину заголовков
caps.getCaptionsWidth();
В результате выполнения примера была получена общая ширина заголовков.
