# TreeListDragAndDropMode


Пространство имен: PP.Ui;


## Описание


Перечисление TreeListDragAndDropMode содержит
 варианты работы режима Drag&Drop иерархического дерева.


Используется следующими свойствами и методами:


	- [TreeList.DragAndDropMode](../Classes/TreeList/TreeList.DragAndDropMode.htm);


	- [TreeListView.DragAndDropMode](../Classes/TreeListView/TreeListView.DragAndDropMode.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 SingleDrag


		 В результате удержания мыши на
		 элементе, режим Drag&Drop активируется только на выбранном
		 элементе.


		 MultiDrag


		 В результате удержания мыши на
		 элементе, режим Drag&Drop активируется для всех выделенных
		 элементов.
