# TreeColumnType


Пространство
 имен: PP.Ui;


## Описание


Перечисление TreeColumnType содержит
 типы содержимого столбца иерархического дерева во время сортировки.


Используется свойством [TreeColumn.Type](../Classes/TreeColumn/TreeColumn.Type.htm).


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 0


		 string.
		 Элементы столбца будут сортироваться как строки.


		 1


		 number.
		 Элементы столбца будут сортироваться как числа.
