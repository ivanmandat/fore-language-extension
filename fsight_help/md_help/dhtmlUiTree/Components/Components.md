# Компоненты для работы с деревьями


## Описание


На данной странице приведены компоненты для работы с деревьями.


## Компоненты


		 Имя компонента
		 Краткое описание


		 ![](../Class_Image.gif)
		 [TreeCombo](TreeCombo/TreeCombo.htm)
		 Компонент TreeCombo
		 представляет собой элемент управления для отображения иерархий
		 в виде раскрывающегося списка.


		 ![](../Class_Image.gif)
		 [TreeList](TreeList/TreeList.htm)
		 Компонент TreeList
		 служит для отображения ветвящихся иерархических структур, таких
		 как дерево наследования объектов.


		 ![](../Class_Image.gif)
		 [DynamicTreeList](DynamicTreeList/DynamicTreeList.htm)
		 Компонент DynamicTreeList
		 служит для отображения дерева наследования объектов с переменной
		 высотой строки и виртуальной прокруткой.


		 ![](../Class_Image.gif)
		 [SimpleTreeList](SimpleTreeList/SimpleTreeList.htm)
		 Компонент SimpleTreeList
		 служит для отображения простого дерева наследования объектов.


		 ![](../Class_Image.gif)
		 [TreeListBox](TreeListBox/TreeListBox.htm)
		 Компонент TreeListBox
		 служит для отображения дерева наследования объектов с постраничной
		 навигацией.


		 ![](../Class_Image.gif)
		 [WrappedTreeList](WrappedTreeList/WrappedTreeList.htm)
		 Компонент WrappedTreeList
		 служит для отображения дерева наследования объектов с переменной
		 высотой строки.
