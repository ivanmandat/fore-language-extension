# Арифметические операции


Ниже приведен список арифметических операций в соответствии с приоритетом
 операций, начиная с наиболее приоритетной операции:


	- *. Результат - произведение операндов;


	- /. Результат - частное двух операндов;


	- Mod. Результат
	 - остаток от деления двух операндов;


	- Div. Результат
	 - целая часть от деления двух операндов;


	- +. Результат - сложение операндов;


	- -. Результат - разница операндов.


Операции могут применяться к операндам различного [типа](../03_dataTypes/Fore_DataTypes.htm).
 Если в выражении не присутствует операция «/», то тип результата определяется
 типом операнда, который включает типы других операндов. В случае если
 в выражении присутствует операция «/», то тип возвращаемого выражения
 - вещественный.


При выполнении операций Mod
 и Div для операндов целого типа
 результат также будет целого типа.
