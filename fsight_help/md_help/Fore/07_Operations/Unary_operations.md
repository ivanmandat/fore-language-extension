# Унарные операции


Ниже приведен список унарных операций:


	- «+». Унарный плюс предопределен
	 для всех числовых типов. Результатом использования унарного плюса
	 для числового типа является просто значение операнда.


	- «-». Унарный минус предопределен
	 для всех числовых типов. Результатом использования унарного минуса
	 является значение, полученное вычитанием из нуля значения операнда.


	- Not. Унарная операция
	 Not предопределена для операнда, имеющего логический тип данных. Результат
	 - отрицание операнда.


Унарные операции имеют только один операнд. В выражениях унарные операции
 вычисляются в первую очередь.
