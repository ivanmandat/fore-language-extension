# Логические операции


Ниже приведен список логических операций в соответствии с их приоритетом,
 начиная с наиболее приоритетной операции:


	- And. Результат
	 - логическое "И" операндов.


	- Or. Результат
	 - логическое "ИЛИ" операндов.


	- Xor. Результат
	 - логическое исключающее ИЛИ.


Данные операции применяются к логическим операндам и возвращают логический
 результат.


		 A And B
		 A Or B
		 A Xor B

		 False


Примечание.
 Для логических операций And и
 Or второй операнд не вычисляется,
 если он не влияет на результат операции.
