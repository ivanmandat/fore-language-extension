# Ожидается имя делегата


## Описание


При объявлении события в качестве типа события указан идентификатор
 объекта, не являющегося делегатом.


## Способ устранения


Проверить тип объекта, установленного в качестве типа события.


## Пример


	Class MyClass: Object

	    Event OnUserEvent: Integer;

	End Class MyClass;


При компиляции указанного кода будет сгенерирована ошибка «Ожидается
 имя делегата». Для разрешения ошибки необходимо объявить новый делегат
 и указать его в качестве типа события «OnUserEvent».
