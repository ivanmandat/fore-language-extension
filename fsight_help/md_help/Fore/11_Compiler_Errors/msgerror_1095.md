# Синтаксическая ошибка в символьной константе


## Описание


Символьная константа недопустима, поскольку содержит более одного символа.
 Если символьная константа указывается через код, то возможно значение
 кода выходит за пределы допустимого диапазона.


## Способ устранения


Необходимо указать в качестве значения константы один символ, либо проверить
 значение кода. Значение кода должно находится в диапазоне, допустимом
 для символьного типа данных
 - 0-65535.


## Пример


	Sub Main;

	Const

	    c = #65536;

	    c1 = 'ab';

	Begin


	End Sub Main;


При компиляции указанного кода на каждой строке блока Const будет сгенерирована
 ошибка «Синтаксическая ошибка в символьной константе». В первой константе
 необходимо уменьшить значение, во второй - удалить один символ.
