# Ожидается символ 'символ'


## Описание


В строке кода спецификацией языка Fore предусмотрено наличие указанного символа.


## Способ устранения


Для разрешения ошибки необходимо перед символом, на котором будет сфокусирован курсор, указать символ из текста ошибки. Наиболее часто встречаются следующие варианты ошибки:


-
Ожидается символ ')' - не указана закрывающая скобка после указания параметров процедуры/функции.


-
Ожидается символ ';' - не указан символ ';', являющийся признаком окончания строки кода. Также данный символ используется при описании различных структур.


-
Ожидается символ '.' - после указания идентификатора типа должен стоять знак '.' для получения доступа к членам данного типа.
