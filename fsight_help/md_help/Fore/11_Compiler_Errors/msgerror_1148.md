# После get ожидается идентификатор функции


## Описание


При использовании старого синтаксиса описания свойств после спецификатора
 Get указан идентификатор процедуры,
 а не функции.


## Способ устранения


Проверить тип объекта, указанного после спецификатора Get.
 Указать функцию, синтаксис которой допустим для данного свойства.


## Пример


Class MyClass: Object

    Sub Proc;

    Begin


    End Sub Proc;


    Function Func: Integer;

    Begin

        Return i

    End Function Func;


    Property Item: Integer Get Proc;

End Class MyClass;


При компиляции указанного кода на строке «Property Item: Integer Get Proc;»
 будет сгенерирована ошибка «После get ожидается идентификатор функции».
 Для разрешения ошибки необходимо после спецификатора «Get» указать идентификатор
 функции «Func».
