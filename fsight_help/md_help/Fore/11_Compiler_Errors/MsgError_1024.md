# Константа типа <типДанных> вышла за пределы допустимых значений


## Описание


При определении константы, указанное значение выходит за пределы допустимых
 значений того типа данных, который был автоматически определен для константы.


## Способ устранения


Измените значения константы.


## Пример


	Sub UserProc;

	Const

	    s = 1.8e+308;

	Begin


	End Sub UserProc;


При компиляции указанного кода будет сгенерирована ошибка «Константа
 типа Double вышла за пределы допустимых значений». Для разрешения ошибки
 необходимо уменьшить значение константы т.к. для типа Double верхний предел
 значений это 1,797693e+308.
