# Недоступно для использования в веб


## Описание


При разработке веб-приложения была попытка использования ресурсов, запрещенных
 для использования в веб.


## Способ устранения


В разделе [Работа
 в среде разработки: Способ компиляции сборок](UiDevEnv.chm::/01_Development_Environment/02_Work_in_Development_Environment/Way_to_compiling.htm) представлен
 список ограничений на используемые ресурсы при разработке веб-приложений.
 Ознакомьтесь с данным разделом и внесите соответствующие изменения в код
 программы.
