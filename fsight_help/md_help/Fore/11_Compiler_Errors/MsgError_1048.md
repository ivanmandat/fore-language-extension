# Ошибка: Левая часть оператора := не допускает присваивания

Ошибка: Левая часть оператора := не допускает присваивания
-


# Левая часть оператора := не допускает присваивания


## Описание


В коде встретилось выражение, в котором осуществляется попытка установить
 значение элементу кода, не допускающему присваивание.


## Способ устранения


Измените выражение таким образом, чтобы присваивание осуществлялось
 только элементам, допускающим присваивание.


## Пример


Class TestClass: Object

    Public Sub Test;

    Begin


    End Sub Test;

End Class TestClass;


Sub Main;

Var

    Obj: TestClass;

Begin

    Obj := New TestClass.Create;

    Obj.Test := 1;

End Sub Main;


При компиляции указанного кода будет сгенерирована ошибка «Левая часть
 оператора := не допускает присваивания». Для разрешения ошибки необходимо
 удалить из кода присваивание значения процедуре «Test», либо в качестве
 «Test» объявить свойство, доступное на запись.
