-


# В методе <имяМетода> отсутствует параметр с именем <имяПараметра>


## Описание


При вызове метода с указанием именованных параметров указано имя несуществующего
 параметра.


## Способ устранения


Проверьте синтаксис именованных параметров в определении метода и при
 его вызове.


## Пример


	Sub TestSub(i: Integer; s: String; d: Double);

	Begin


	End Sub TestSub;


	Sub UserProc;

	Begin

	    TestSub(i := 1, dd := 1.1, s := "a");

	End Sub UserProc;


При компиляции указанного кода на строке «TestSub(i := 1, dd := 1.1,
 s := "a");» будет сгенерирована ошибка «В методе TestSub отсутствует
 параметр с именем dd». Для разрешения ошибки необходимо указать корректное
 наименование параметра, т.е. «d».
