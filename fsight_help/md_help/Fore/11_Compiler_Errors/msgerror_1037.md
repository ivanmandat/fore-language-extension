# Операция <имяОперации> недопустима для операндов типа <имяТипа1> и типа <имяТипа2>


## Описание


В выражении используется какая-либо операция, недоступная для операндов указанных типов. Возможно операция применяется для не совместимых типов, либо данная операция не поддерживается одним из операндов.


## Способ устранения


Необходимо проверить совместимость типов. Если типы совместимы, то необходимо проверить поддерживается ли операция каждым типом. Операции, поддерживаемые простыми типами данных, смотрите в описании соответствующих [типов данных](../03_dataTypes/Fore_DataTypes.htm). Для всех типов, являющихся наследником класса [Object](ForeSys.chm::/Class/Object/Object.htm), доступны две операции: "<>" и "=".
