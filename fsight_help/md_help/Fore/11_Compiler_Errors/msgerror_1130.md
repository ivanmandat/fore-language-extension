# Ожидается определение метода, поля или свойства класса


## Описание


В указанном месте кода пропущено описание метода, поля или свойства
 класса.


## Способ устранения


Допишите недостающий фрагмент кода, содержащий описание метода, поля
 или свойства класса.


## Пример


	Class TestClass: Object

	    Final

	End Class TestClass;


При компиляции указанного кода на строке «Final» будет сгенерирована
 ошибка «Ожидается определение метода, поля или свойства класса». Для разрешения
 ошибки необходимо добавить описание метода, поля или свойства класса,
 либо удалить директиву «Final».
