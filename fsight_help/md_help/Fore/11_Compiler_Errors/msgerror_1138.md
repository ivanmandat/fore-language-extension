# Операторы для пользовательских классов не поддерживаются


## Описание


В пользовательском классе встретилось ключевое слово Operator, запрещенное для использования в прикладном коде.


## Способ устранения


Удалите, либо поместите в блок комментариев ключевое слово Operator и весь код, связанный с ним.
