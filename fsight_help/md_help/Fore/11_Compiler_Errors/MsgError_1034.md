# Не удалось загрузить библиотеку <имяБиблиотеки>. Текст ошибки:
 <ошибка>


## Описание


При использовании оператора [Comimport](../06_SyntRules/Fore_Comimport.htm)
 возникла ошибка при загрузке описаний из библиотеки типов. Возможная причина
 ошибки указывается в тексте сообщения.


## Способ устранения


В соответствии с текстом ошибки проверить соответствующую библиотеку
 типов.


## Пример


			Comimport From "{11111111-1111-1111-1111-111111111111}"


End Comimport;


При компиляции указанного кода будет сгенерирована ошибка «Не удалось
 загрузить библиотеку "{11111111-1111-1111-1111-111111111111}".
 Текст ошибки: Библиотека не зарегистрирована.». Для разрешения ошибки
 необходимо изменить указанный GUID библиотеки, либо указать полный путь
 к соответствующему файлу.
