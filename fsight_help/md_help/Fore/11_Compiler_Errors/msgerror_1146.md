# В сборке <имяСборки> используется модуль с именем системного модуля <имяСистемнойСборки>


## Описание


В текущей сборке подключена сборка репозитория, в которой имеется модуль/форма, идентификатор которой совпадает с наименованием одной из системных сборок платформы.


## Способ устранения


Измените идентификатор модуля/формы таким образом, чтобы он отличался от наименований системных сборок.
