# Ожидается идентификатор <идентификатор>


## Описание


Идентификатор в заголовке структуры не совпадает с идентификатором в
 окончании.


## Способ устранения


Проверьте идентификаторы в заголовке и окончании структуры.


## Пример


			Sub UserProc;

Begin


End Sub UserProc1;


При компиляции указанного кода будет сгенерирована ошибка «Ожидается
 идентификатор UserProc». Для разрешения ошибки необходимо в строке «End Sub UserProc1;»
 удалить лишний символ «1».
