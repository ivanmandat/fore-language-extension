# Метод <имяМетода> в базовом классе не найден


## Описание


После ключевого слова INHERITED указан идентификатор метода, который
 не реализован в базовом классе.


## Способ устранения


Проверьте идентификатор вызываемого метода. Возможно область видимости
 метода не допускает его вызов в дочерних классах.


## Пример


			Class BaseClass: Object


End Class BaseClass;


Class TestClass: BaseClass

    Sub Test;

    Begin

        Inherited BaseTest;

    End Sub Test;

End Class TestClass;


При компиляции указанного кода будет сгенерирована ошибка «Метод BaseTest
 в базовом классе не найден». Для разрешения ошибки необходимо в классе
 «BaseClass» описать метод с наименованием «BaseTest».
