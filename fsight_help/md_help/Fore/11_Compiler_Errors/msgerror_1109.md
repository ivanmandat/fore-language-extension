# Сборка <имяСборки1> подключается циклически: <имяСборки1>-<имяСборки2>-<имяСборки1>


## Описание


В сборке2, подключенной к указанной сборке1, уже имеется ссылка на сборку1.


## Способ устранения


В одной из сборок измените список подключенных сборок таким образом,
 чтобы не было циклических ссылок. Если архитектура разрабатываемого приложения
 позволяет, то объедините все необходимые ресурсы (модули/формы/текст классов
 и тому подобное) в одной сборке таким образом, чтобы не было необходимости
 циклически подключать другие сборки.
