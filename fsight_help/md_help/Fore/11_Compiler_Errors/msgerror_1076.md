# Оператор new недопустим для интерфейсов


## Описание


При инициализации переменной после ключевого слова New указан идентификатор
 интерфейса.


## Способ устранения


Измените код, указав при инициализации переменной класс объекта и доступный
 конструктор.


## Пример


Interface IMyInterface


End Interface IMyInterface;


Class MyClass: Object, IMyInterface


End Class MyClass;


Sub UserProc;

Var

    Obj: IMyInterface;

Begin

    Obj := New IMyInterface;

End Sub UserProc;


При компиляции указанного кода на строке «Obj := New IMyInterface;»
 будет сгенерирована ошибка «Оператор new недопустим для интерфейсов».
 Для разрешения ошибки необходимо изменить код для инициализации переменной
 «Obj». Корректный код - «Obj := New MyClass.Create;»
