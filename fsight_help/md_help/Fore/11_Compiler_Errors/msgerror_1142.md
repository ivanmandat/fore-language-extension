# Используется циклическое наследование класса <имяКласса> от самого
 себя


## Описание


При организации иерархии классов выявлено циклическое наследование класса
 от самого себя.


## Способ устранения


Измените иерархию наследования классов таким образом, чтобы не было
 циклических зависимостей.


## Пример


	Class A: C


	End Class A;


	Class B: A


	End Class B;


	Class C: B


	End Class C;


При компиляции указанного кода на строке «Class a: c» будет сгенерирована
 ошибка «Используется циклическое наследование класса A от самого себя».
 Для разрешения ошибки необходимо в описании класса «a» изменить тип, от
 которого наследуется данный класс.
