# По левую сторону от . должен быть объект, класс, перечисление или модуль


## Описание


В блоке кода для константы осуществляется вызов методов, доступных для
 соответствующего типа данных.


## Способ устранения


В среде разработки, если после идентификатора константы установить «.»
 , то в IntelliSense будет сформирован список методов, доступных для соответствующего
 типа данных. Использование данных методов для констант запрещено. Необходимо
 удалить вызов любых методов для константы либо переместить описание константы
 в блок переменных.


## Пример


			Sub UserProc;

Const

    d = 1;

Begin

    d.Parse("1");

End Sub UserProc;


При компиляции указанного кода на строке «d.Parse("1");» будет
 сгенерирована ошибка «По левую сторону от . должен быть объект, класс,
 перечисление или модуль». Для разрешения ошибки необходимо удалить вызов
 метода «Parse» для константы «d».
