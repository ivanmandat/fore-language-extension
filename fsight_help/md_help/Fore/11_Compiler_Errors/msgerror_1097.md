# Идентификатор превышает максимальную допустимую длину 255 символов


## Описание


В соответствии со спецификацией языка Fore максимальная допустимая длина идентификатора элемента кода не должна превышать 255 символов.


## Способ устранения


Измените идентификатор таким образом, чтобы его длина была меньше 255 символов.
