# <имяТипа> не является пространством имен


## Описание


При импорте содержимого пространства имен после оператора Imports указано
 наименование объекта, не являющегося пространством имен.


## Способ устранения


После ключевого слова Imports укажите наименование пространства имен.


## Пример


	Imports String;


	Sub UserProc;

	Begin


	End Sub UserProc;


При компиляции указанного кода будет сгенерирована ошибка «String не
 является пространством имен». Для разрешения ошибки необходимо указать
 наименование существующего пространства имен.
