# Операция <имяОперации> недопустима для операнда типа <имяТипа>


## Описание


Для операнда применяется операция, не предусмотренная для операндов указанного типа спецификацией языка Fore, либо операция должна использоваться в другом контексте. Возможно в выражении пропущен еще один операнд.


## Способ устранения


Необходимо проверить, предусмотрено ли спецификацией использование указанной операции для операндов указанного типа. При необходимости добавьте дополнительные операнды либо измените тип исходного операнда, для которого сгенерирована ошибка.
