# Директива <имяДирективы> недопустима для <имяЧлена>


## Описание


Спецификацией языка Fore запрещено
 использование указанной директивы с указанным членом типа.


## Способ устранения


Измените описание члена типа таким образом, чтобы в описании использовались
 только допустимые директивы.


## Пример


	Delegate Change(Sender: Object);


	Class TestClass: Object

	    Final Event OnChange: Change;

	End Class TestClass;


При компиляции указанного кода будет сгенерирована ошибка «Директива
 final недопустима для события». Для разрешения ошибки необходимо из описания
 события удалить директиву «Filal».
