# Свойство <имяСвойства> доступно только для чтения


## Описание


В коде встретилось выражение, в котором осуществляется попытка установить
 значение свойству, доступному только для чтения.


## Способ устранения


Измените выражение таким образом, чтобы не осуществлялось присвоение
 свойству, доступному только на чтение.


## Пример


			Class TestClass: Object

    Public Shared Property Test: Integer

        Get

        Begin

            Return 1

        End Get

    End Property Test;

End Class TestClass;


Sub Main;

Var

    Obj: TestClass;

Begin

    Obj.Test := 1;

End Sub Main;


При компиляции указанного кода будет сгенерирована ошибка «Свойство
 Test доступно только для чтения». Для разрешения ошибки необходимо в описании
 свойства «Test» добавить спецификатор «Set», либо удалить из выражения
 присвоение значения.
