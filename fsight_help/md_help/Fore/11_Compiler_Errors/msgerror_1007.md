# Выражение не может быть присвоено списку переменных


## Описание


В блоке объявления переменных установлено значение по умолчанию для
 более чем одной переменной.


## Способ устранения


Установка значения по умолчанию доступна только для одной переменной.
 Измените описание переменных таким образом, чтобы значение по умолчанию
 присваивалось каждой переменной индивидуально.


## Пример


	Sub UserProc;

	Var

	    i, j: Integer = 1;

	Begin


	End Sub UserProc;


При компиляции указанного кода будет сгенерирована ошибка «Выражение
 не может быть присвоено списку переменных». Для разрешения ошибки необходимо
 каждую переменную объявить отдельно и указать требуемое значение по умолчанию.
