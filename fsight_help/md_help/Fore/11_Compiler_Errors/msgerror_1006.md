# Константное выражение должно быть не меньше <значение>


## Описание


При указании диапазона допустимых значений левая граница имеет значение
 больше правой.


## Способ устранения


Проверьте значения используемых констант, установите значение не меньше
 указанного значения.


## Пример


	Sub UserProc;

	Const

	    Min = 0;

	    Max = -5;

	Var

	    a: Array[Min..Max];

	Begin


	End Sub UserProc;


При компиляции указанного кода в строке объявления массива на константе
 «Max» будет сгенерирована ошибка «Константное выражение должно быть не
 меньше 0». Для разрешения ошибки необходимо установить соответствующее
 значение для константы «Max».
