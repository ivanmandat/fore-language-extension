# Перечисление должно содержать определение хотя бы одной константы


## Описание


В пользовательском перечислении ни указана ни одна константа.


## Способ устранения


Указать минимум одно значение константы целого типа.


## Пример


			Enum MyEnum


End Enum MyEnum;


При компиляции указанного кода будет сгенерирована ошибка «Перечисление
 должно содержать определение хотя бы одной константы». Для разрешения
 ошибки необходимо указать одно значение константы в перечислении.
