# Недопустимая директива default для статического свойства


## Описание


Свойство не может быть объявлено одновременно с директивами Shared и
 Default.


## Способ устранения


Удалите одну из директив из описания свойства.


## Пример


	Class TestClass: Object

	    i: Integer;

	    Shared Property Item(i: Integer): Integer

	        Get

	        Begin

	            Return i;

	        End Get

	        Set

	        Begin

	            i := Value

	        End Set

	    End Property Item; Default;

	End Class TestClass;


При компиляции указанного кода будет сгенерирована ошибка «Недопустимая
 директива default для статического свойства». Для разрешения ошибки удалите
 из описания свойства директиву «Shared» или «Default».
