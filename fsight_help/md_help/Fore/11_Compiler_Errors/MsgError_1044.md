# Недопустимое использование оператора <оператор> вне цикла


## Описание


В блоке кода вызван оператор, использование которого доступно только
 в рамках какого-либо цикла.


## Способ устранения


Удалите вызов оператора, либо перенесите оператор в какой-либо цикл.


## Пример


	Sub UserProc;

	Begin

	    Break;

	End Sub UserProc;


При компиляции указанного кода будет сгенерирована ошибка «Недопустимое
 использование оператора break вне цикла». Для разрешения ошибки необходимо
 удалить вызов оператора, либо организовать цикл и перенести вызов оператора
 «break» в тело цикла.
