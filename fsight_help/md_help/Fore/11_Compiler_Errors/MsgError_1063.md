# Ожидается имя класса


## Описание


Спецификацией языка Fore в указанном
 месте блока кода предусмотрено указание класса.


## Способ устранения


Измените указанный идентификатор таким образом, чтобы был указан идентификатор
 класса.


## Пример


	Class MyClass: Object


	End Class MyClass;


	Sub UserProc;

	Var

	    Obj1, Obj2: MyClass;

	Begin

	    Obj1 := New MyClass.Create;

	    Obj2 := New Obj1;

	End Sub UserProc;


При компиляции указанного кода на строке «Obj2 := New Obj1;»
 будет сгенерирована ошибка «Ожидается имя класса». Для инициализации переменной
 «Obj2» необходимо указать такой же код, как и для «Obj1».
