# В теле функции не найден оператор return


## Описание


Спецификацией языка Fore определено
 обязательное наличие оператора Return
 в теле функций.


## Способ устранения


В теле функции необходимо дописать оператор Return. Тип возвращаемого
 значения должен соответствовать типу функции.


## Пример


	Class TestClass: Object

	    Function Test: Integer;

	    Begin


	    End Function Test;

	End Class TestClass;


При компиляции указанного кода будет сгенерирована ошибка «В теле функции
 не найден оператор return». Для разрешения ошибки необходимо в теле функции
 «Test» дописать оператор Return и после него какое-либо возвращаемое целочисленное
 значение.
