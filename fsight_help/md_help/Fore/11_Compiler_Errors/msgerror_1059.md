# Ожидается get или set


## Описание


При описании свойства не указан ни один спецификатор, определяющий доступность
 свойства.


## Способ устранения


Указать необходимый спецификатор, определяющий доступность свойства.
 Синтаксис описания свойств представлен в разделе: [Описание
 свойств](../06_SyntRules/Description_Property.htm).


## Пример


			Class MyClass: Object

    Property Item: Integer


    End Property items;

End Class MyClass;


При компиляции указанного кода будет сгенерирована ошибка «Ожидается
 get или set». Для разрешения ошибки необходимо указать требуемый идентификатор
 доступа в соответствии с новым синтаксисом свойств.
