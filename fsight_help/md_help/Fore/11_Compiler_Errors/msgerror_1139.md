# Пользовательский интерфейс не может наследоваться от системного интерфейса


## Описание


Спецификацией языка Fore запрещено
 наследование интерфейсов от системных интерфейсов, описанных в системных
 сборках платформы.


## Способ устранения


Измените иерархию наследования. Исключите наследование пользовательских
 интерфейсов от системных интерфейсов платформы.


## Пример


			Interface IMyInterface: INamedEntityModel


End Interface IMyInterface;


При компиляции указанного кода будет сгенерирована ошибка «Пользовательский
 интерфейс не может наследоваться от системного интерфейса». Для разрешения
 ошибки необходимо в описании интерфейса удалить наследование от интерфейса
 «INamedEntityModel».
