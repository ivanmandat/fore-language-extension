# Ожидается определение блока модуля


## Описание


При компиляции встретился блок кода, который может использоваться только
 внутри какого-либо блока, предусмотренного спецификацией языка Fore.


## Способ устранения


Необходимо дописать объемлющий блок, в котором допустимо использовать
 существующий блок кода.


## Пример


	Select Case i

	End Select;


При компиляции указанного кода будет сгенерирована ошибка «Ожидается
 определение блока модуля». Для разрешения ошибки необходимо реализовать
 процедуру/функций, в которой можно использовать оператор выбора Select
 Case.
