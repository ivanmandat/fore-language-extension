# Ожидается выражение логического типа


## Описание


В операторе спецификацией языка Fore
 предусмотрено использование выражений логического типа, однако указанное
 выражение не может быть неявно приведено к логическому типу.


## Способ устранения


Проверьте тип, который возвращает выражение, используемое в операторе.
 Используйте явное преобразование (оператор [As](../07_Operations/Operations_of_the_creation_object.htm#as)),
 либо специальные методы, возвращающие логические значения.


## Пример


	Sub UserProc;

	Var

	    b: Integer;

	Begin

	    While b Do


	    End While;

	End Sub UserProc;


При компиляции указанного кода будет сгенерирована ошибка «Ожидается
 выражение логического типа». Ошибка возникает, потому что переменная «b»
 имеет тип Integer, который не может быть неявно приведен к типу Boolean.
 Для разрешения ошибки измените тип переменной «b».
