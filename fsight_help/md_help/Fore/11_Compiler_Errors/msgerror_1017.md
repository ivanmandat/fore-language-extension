# Параметр <имяПараметра> уже встречался в списке


## Описание


При вызове процедуры/функции с использованием именованных параметров
 для одного или более параметров значение указано больше одного раза.


## Способ устранения


Удалить дублирующиеся значения параметров. Для каждого параметра должно
 указываться только одно значение. Способы вызова с различными вариантами
 указания параметров представлены в разделе Операторы.


## Пример


	Sub UserProc(a: Integer; b: String);

	Begin


	End Sub UserProc;


	Sub Main;

	Begin

	    UserProc(a := 3, b := "a", b := "b");

	End Sub Main;


При компиляции указанного кода на строке «UserProc(a := 3, b := "a", b := "b");»
 будет сгенерирована ошибка «Параметр b уже встречался в списке». Ошибка
 возникает из-за того, что значение параметра «b» указано два раза.
