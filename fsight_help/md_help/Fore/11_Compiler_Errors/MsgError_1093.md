# Многострочный комментарий не завершен. Ожидается символ }


## Описание


При использовании многострочного комментария забыта закрывающая скобка
 «}».


## Способ устранения


Добавить закрывающую скобку «}» в том месте, где она необходима.


## Пример


	Sub UserProc;

	Begin

	    {Далее идет основной код программы.

	    В данном месте коде осуществляется весь расчет.}

	End Sub UserProc


При компиляции указанного кода будет сгенерирована ошибка «Многострочный
 комментарий не завершен. Ожидается символ }». Для разрешения ошибки необходимо
 добавить закрывающую скобку «}», там где заканчивается обычный текст.
