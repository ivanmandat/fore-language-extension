# Ожидается ключевое слово <ключевоеСлово>


## Описание


В операторе спецификацией языка Fore
 предусмотрено использование указанного ключевого слова.


## Способ устранения


Проверьте формат используемого оператора. Допишите необходимые ключевые
 слова.


## Пример


	Sub UserProc;

	Var

	    i: Integer;

	Begin

	    Select Case i


	    End Select;

	End Sub UserProc;


При компиляции указанного кода будет сгенерирована ошибка «Ожидается
 ключевое слово case». Ошибка возникает, потому что в операторе выбора
 Select case должен быть хотя бы один блок Case.
