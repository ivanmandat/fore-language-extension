# Блок Imports должен предшествовать всем другим блокам


## Описание


Спецификацией языка Fore определено
 указание блока Imports в самом начале модуля/формы.


## Способ устранения


Перенесите блок Imports в самое начало модуля/формы.


## Пример


	Sub UserProc;

	Begin


	End Sub UserProc;


	Imports Space1;


При компиляции указанного кода будет сгенерирована ошибка «Блок Imports
 должен предшествовать всем другим блокам». Для разрешения ошибки необходимо
 указать блок Imports перед процедурой «UserProc».
