# Избыточное количество индексов


## Описание


При обращении к элементу массива указанное количество индексов превышает
 количество индексов, указанных в описании массива.


## Способ устранения


Удалите лишние индексы, указанные при обращении к элементу массива.


## Пример


	Sub UserProc;

	Var

	    s: Array[3, 3] Of Integer;

	Begin

	    s[1, 2, 3] := 3;

	End Sub UserProc;


При компиляции указанного кода на строке «s[1, 2, 3] := 3;»
 будет сгенерирована ошибка «Избыточное количество индексов». Для разрешения
 ошибки необходимо удалить один из лишних индексов.
