# Типы <имяТипа1> и <имяТипа2> не совместимы


## Описание


Результат вычисления выражения имеет <тип2> и не может быть установлен в качестве значения для объекта, имеющего <тип1>. Типы <тип1> и <тип2> неявно не приводимы.


## Способ устранения


Проверьте тип результирующего значения и тип объекта, которому присваивается результат вычисления.
