# Переменная/Константа <наименование> не используется


## Описание


В блоке объявления локальных переменных Var пользовательской процедуры/функции присутствуют неиспользуемые переменные/константы. Возможно, допущена ошибка в написании локальных переменных.


## Способ устранения


Данное сообщение является предупреждением и может быть проигнорировано. Для устранения необходимо проверить наличие орфографических ошибок в наименованиях локальных переменных. Если орфографические ошибки отсутствуют, то необходимо либо удалить неиспользуемые переменные/константы, либо поместить их в область комментариев.
