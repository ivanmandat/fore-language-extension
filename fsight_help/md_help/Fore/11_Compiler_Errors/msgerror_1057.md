# Метод <имяМетода> интерфейса <имяИнтерфейса> не реализован


## Описание


В классе, унаследованном от какого-либо интерфейса, содержится реализация
 не всех членов, описанных в интерфейсе.


## Способ устранения


В классе необходимо дописать реализацию всех членов, структура которых
 описана в интерфейсе.


## Пример


	Interface IMyInterface

	    Sub TestSub(a: Integer; b: String);

	End Interface IMyInterface;


	Class MyClass: Object, IMyInterface


	End Class MyClass;


При компиляции указанного кода будет сгенерирована ошибка «Метод TestSub
 интерфейса IMyInterface не реализован». Ошибка возникает, потому что в
 классе MyClass, который является наследником интерфейса IMyInterface,
 отсутствует реализация метода, описанного в интерфейсе.
