# Неоконченная строка. Ожидается символ <">


## Описание


В символьной строке пропущена закрывающая кавычка.


## Способ устранения


В окончании символьной строки необходимо указать закрывающую кавычку.


## Пример


			Sub UserProc;

Begin

    Debug.WriteLine("Test);

End Sub UserProc;


При компиляции указанного кода будет сгенерирована ошибка «Неоконченная
 строка. Ожидается символ <">». Для разрешения ошибки необходимо
 после слова «Test» указать закрывающую кавычку.
