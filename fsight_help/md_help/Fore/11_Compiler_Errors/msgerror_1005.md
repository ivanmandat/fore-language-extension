# Ожидается выражение


## Описание


Оператор в качестве своего правого операнда требует выражение, однако
 выражение отсутствует.


## Способ устранения


Необходимо добавить допустимое выражение в качестве правого операнда.


## Пример


	Sub Main;

	Var

	    i: Integer;

	Begin

	    i :=;

	End Sub Main;


При компиляции указанного кода на строке «i :=;» будет сгенерирована
 ошибка «Ожидается выражение». Для разрешения ошибки необходимо дописать
 какое-либо выражение, возвращающее целочисленное значение.
