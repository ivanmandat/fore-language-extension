# Определены не все обязательные параметры <метод> <имя>


## Описание


При обращении к какому-либо методу его сигнатурой определено указание
 обязательного набора параметров, но значения одного или более параметров
 отсутствуют.


## Способ устранения


Необходимо указать значения всех обязательных параметров через запятую.
 Указанные значения параметров должны иметь тип, определенный сигнатурой
 вызываемого метода.


## Пример


	Sub Main;

	Var

	    i: Integer;

	Begin

	    i := Integer.Parse;

	End Sub Main;


При компиляции указанного кода на строке «i := Integer.Parse;»
 будет сгенерирована ошибка «Определены не все обязательные параметры метода
 Parse». Для разрешения ошибки необходимо в скобках указать строковое значение
 параметра «Value», который определен в методе «Parse».
