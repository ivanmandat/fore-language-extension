# Ожидается константное выражение


## Описание


В соответствии со спецификацией языка Fore
 при определении констант в выражении можно использовать конкретные значения
 либо ранее определенные константы.


## Способ устранения


Из выражения, используемого для определения значения константы, убрать
 вызов функций, обращение к полям и свойствам.


## Пример


	Sub UserProc;

	Const

	    i = Integer.MaxValue;

	Begin


	End Sub UserProc;


При компиляции указанного кода будет сгенерирована ошибка «Ожидается
 константное выражение». Для разрешения ошибки необходимо в описании константы
 «i» удалить обращение к свойству MaxValue и указать конкретное значение.
