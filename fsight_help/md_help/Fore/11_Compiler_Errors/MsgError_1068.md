# В качестве счетчика цикла ожидается локальная переменная


## Описание


В качестве переменной цикла используется переменная, объявленная за
 пределами текущего члена типа.


## Способ устранения


Используйте в качестве переменной цикла только локальные переменные,
 объявленные в рамках текущего члена типа.


## Пример


Class TestClass: Object

    i: Integer;


    Sub Test;

    Begin

        For i := 0 To 100 Do


        End For;

    End Sub Test;

End Class TestClass;


При компиляции указанного кода будет сгенерирована ошибка «В качестве
 счетчика цикла ожидается локальная переменная». Для разрешения ошибки
 необходимо перенести описание переменной «i» в блок «Var» процедуры «Test».
