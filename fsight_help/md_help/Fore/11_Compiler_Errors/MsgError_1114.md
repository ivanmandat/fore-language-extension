# Оператор new[] допустим только для простых типов


## Описание


Квадратные скобки в сочетании с простыми типами и ключевым словом New
 используются при инициализации массивов. Любое другое сочетание квадратных
 скобок с оператором New недопустимо.


## Способ устранения


Используйте только простые типы при инициализации массива.


## Пример


			Sub UserProc;

Var

    s: Array;

Begin

    s := New Array[3];

End Sub UserProc;


При компиляции указанного кода на строке «s := New Array[3];» будет
 сгенерирована ошибка «Оператор new[] допустим только для простых типов».
 Для разрешения ошибки необходимо изменить тип, используемый при инициализации,
 следующим образом: «s := New Variant[3];».
