# Ожидается переменная типа <тип>


## Описание


В операторе спецификацией языка Fore предусмотрено использование переменной
 указанного типа.


## Способ устранения


Измените тип переменной, либо укажите другую переменную соответствующего
 типа.


## Пример


	Sub UserProc;

	Var

	    i: Double;

	Begin

	    For i := 0 To 100 Do


	    End For;

	End Sub UserProc;


При компиляции указанного кода будет сгенерирована ошибка «Ожидается
 переменная типа Integer». Для разрешения ошибки измените тип переменной
 «i» на «Integer», либо определите новую переменную.
