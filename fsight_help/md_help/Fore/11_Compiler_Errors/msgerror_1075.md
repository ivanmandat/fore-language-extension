# Ожидается строка с GUID библиотеки


## Описание


При использование блока Comimport после ключевого слова указаны данные,
 не являющиеся символьной строкой.


## Способ устранения


После ключевого слова Comimport укажите символьную строку, содержащую
 GUID библиотеки, либо путь к файлу библиотеки.


## Пример


	Comimport From 'c'


	End Comimport;


При компиляции указанного кода будет сгенерирована ошибка «Ожидается
 строка с GUID библиотеки». Для разрешения ошибки после ключевого слова
 Comimport укажите символьную строку с GUID библиотеки, либо путь к файлу
 библиотеки.
