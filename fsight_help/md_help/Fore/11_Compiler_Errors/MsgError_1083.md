# Повторное определение идентификатора


## Описание


В рамках одного пространства имен присутствует более одного описания
 типа (члена типа) с одинаковым идентификатором.


## Способ устранения


Удалите или измените все дублирующиеся идентификаторы.


## Пример


	Sub UsepProc;

	Const

	    i = 100;

	Var

	    i: Integer;

	Begin


	End Sub UsepProc;


При компиляции указанного кода на строке «i: Integer;» будет сгенерирована
 ошибка «Повторное определение идентификатора». Для разрешения ошибки необходимо
 изменить идентификатор переменной, либо константы, либо полностью удалить
 блок объявления констант или переменных.
