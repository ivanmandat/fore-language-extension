# Нельзя наследовать от класса <имяКласса>


## Описание


При наследовании в качестве базового класса указан класс, соответствующий
 простому типу данных. Наследование от простых типов данных запрещено.


## Способ устранения


Измените класс, используемый в качестве базового.


## Пример


			Class TestClass: Variant


End Class TestClass;


При компиляции указанного кода будет сгенерирована ошибка «Нельзя наследовать
 от класса Variant». Для разрешения ошибки необходимо унаследовать пользовательский
 класс от другого класса, например, «Object».
