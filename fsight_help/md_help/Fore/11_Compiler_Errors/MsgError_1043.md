# Недопустимое использование оператора <оператор> в блоке finally


## Описание


В блоке Finally встретился вызов оператора, использование которого запрещено
 в данном блоке спецификацией языка Fore.


## Способ устранения


Удалите вызов недопустимого оператора.


## Пример


			Sub UserProc(Skip: Boolean);

Begin

    Try

        //Проверяемые действия

    Except On e: DirectoryNotFoundException Do

        If Skip Then

            Return;

        Else

            //Обработка ошибки

        End If;

    On e: FileNotFoundException Do

        If Skip Then

            Return;

        Else

            //Обработка ошибки

        End If;

    Finally

        Return;

    End Try;

End Sub UserProc;


При компиляции указанного кода на строке «Return» в блоке «Finally»
 будет сгенерирована ошибка «Недопустимое использование оператора в блоке
 finally». Для разрешения ошибки необходимо удалить оператор «Return» из
 данного блока.
