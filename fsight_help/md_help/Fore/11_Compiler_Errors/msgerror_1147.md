# Ожидается определение метода или свойства интерфейса


## Описание


В описании интерфейса встретилось описание члена типа, недопустимое
 в рамках интерфейса. В рамках интерфейса могут быть описаны только следующие
 члены типов:


	- Процедуры


	- Функции


	- Свойства


## Способ устранения


Перенесите ошибочное описание в реализацию класса, либо удалите полностью.


## Пример


	Delegate CheckDelegate(Obj: Variant; Params: Variant);

	Interface IMyInterface

	    Sub Run;


	    Function Check: Boolean;


	    Property Name: String

	        Get

	        Set

	    End Property Name;


	    Event OnCheck: CheckDelegate;

	End Interface IMyInterface;


При компиляции указанного кода на строке «Event OnCheck: CheckDelegate;»
 будет сгенерирована ошибка «Ожидается определение метода или свойства
 интерфейса». Описание событий недопустимо в рамках интерфейсов, удалите
 соответствующую строку, либо перенесите описание события в реализацию
 класса.
