# Неизвестный идентификатор <идентификатор>


## Описание


В коде встретился вызов нераспознаваемого компилятором идентификатора.


## Способ устранения


Данная ошибка может быть обусловлена множества факторов. Для решения
 ошибки необходимо проверить следующие моменты:


	- в идентификаторе содержится синтаксическая ошибка;


	- переменная/константа с указанным идентификатором не объявлена
	 в соответствующем блоке Var/Const;


	- вызываемый член реализован в сборке, не подключенной к данной
	 сборке;


	- вызываемый член имеет область видимости, не позволяющую его
	 вызов в данном месте кода;


	- вызов члена недопустим в данном контексте (например, вызов конструктора
	 без ключевого слова New).
