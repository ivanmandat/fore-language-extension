# <имяТипа> не является подпрограммой


## Описание


В коде встретился вызов объекта, не являющегося подпрограммой.


## Способ устранения


Проверьте тип вызываемого объекта. Возможно вместо вызова должно быть
 указано выражение.


## Пример


	Sub UserProc;

	Var

	    i: Integer;

	Begin

	    i;

	End Sub UserProc;


При компиляции указанного кода на строке «i;» будет сгенерирована ошибка
 «i не является подпрограммой». Для разрешения ошибки необходимо изменить
 данную строку указав идентификатор какой-либо подпрограммы, либо дописать
 выражение для формирования значения переменной «i».
