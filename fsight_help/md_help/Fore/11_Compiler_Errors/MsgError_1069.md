# Ожидается оператор


## Описание


В указанном месте кода при описании процедуры, функции или свойства
 пропущено ключевое слово End.


## Способ устранения


Допишите недостающий фрагмент кода, содержащий ключевое слово End.


## Пример


			Sub UserProc;

Begin


Sub UserProc;


При компиляции указанного кода на завершающем слове «Sub» будет сгенерирована
 ошибка «Ожидается оператор». Для разрешения ошибки необходимо исправить
 текст завершения процедуры: End Sub UserProc;
