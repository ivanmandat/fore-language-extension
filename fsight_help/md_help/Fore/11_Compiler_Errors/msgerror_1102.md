# Несовместимые размерности массивов


## Описание


При присвоении массивов исходные массивы имеют разные размерности.


## Способ устранения


Для корректного присвоения массивов исходные массивы должны иметь одинаковую
 размерность.


## Пример


	Sub UserProc;

	Var

	    s: Array[3] Of String;

	    s1: Array[3, 3] Of String;

	Begin

	    s := s1;

	End Sub UserProc;


При компиляции указанного кода на строке «s := s1;» будет сгенерирована
 ошибка «Несовместимые размерности массивов». Для разрешения ошибки необходимо
 изменить размерность одного из массивов таком образом, чтобы она совпадала
 с размерностью второго массива, например «s: Array[3, 3] Of String;».
