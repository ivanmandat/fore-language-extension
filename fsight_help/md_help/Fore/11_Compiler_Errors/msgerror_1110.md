# Слишком много методов в импортируемом интерфейсе


## Описание


При импорте содержимого библиотеки типов с помощью оператора [Comimport](../06_SyntRules/Fore_Comimport.htm) в описании одного из интерфейсов превышено допустимое количество членов. Описание импортируемых интерфейсов должно содержать не более 270 различных процедур/функций/свойств.


## Способ устранения


Работа в среде разработки с данной библиотекой типов недопустима. Используйте другую библиотеку типов.
