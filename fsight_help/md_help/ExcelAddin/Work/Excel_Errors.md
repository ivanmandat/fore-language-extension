# Сообщения об ошибках


Раздел содержит информацию о способах устранения возможных ошибок, возникающих
 при попытке выполнить какую-либо операцию.


Текст ошибки:


[![](../Opened.gif)![](../Closed.gif)Выделите хотя
 бы один ряд для вычисления функции](javascript:TextPopup(this))


	Причина появления: возникает
	 при попытке рассчитать метод без исходных данных.


	Решение:


		- В [таблице данных](TimeSeries_Mode/Table_Work.htm)
		 выделите один или несколько рядов.


		- На вкладке «Вычисления»
		 ленты инструментов выполните команду, соответствующую [методу
		 расчёта](TimeSeries_Mode/TimeSeries_Work/Methods/Calculation_Methods.htm).


[![](../Opened.gif)![](../Closed.gif)Нельзя выполнить
 операцию над производными рядами, выберите другие ряды](javascript:TextPopup(this))


	Причина появления: возникает
	 при попытке применить метод расчёта к ряду, являющемуся дочерним рядом
	 по отношению к вычисляемому ряду.


	Решение:


		- В [таблице данных](TimeSeries_Mode/Table_Work.htm)
		 выделите один или несколько рядов, не являющихся дочерними рядами
		 по отношению к вычисляемому ряду.


		- На вкладке «Вычисления»
		 ленты инструментов выполните команду, соответствующую [методу
		 расчёта](TimeSeries_Mode/TimeSeries_Work/Methods/Calculation_Methods.htm).


[![](../Opened.gif)![](../Closed.gif)Этот файл сейчас
 используется. Укажите другое имя файла или закройте файл в другом приложении](javascript:TextPopup(this))


	Причина появления: возникает
	 при попытке вставить из файла в отчёт объект, который уже используется
	 в каком-либо отчёте.


	Решения:


		- закройте отчёт, использующий объект из данного файла. Повторите
		 вставку объекта;


		- выберите для вставки другой объект, сохраненный в виде файла.
