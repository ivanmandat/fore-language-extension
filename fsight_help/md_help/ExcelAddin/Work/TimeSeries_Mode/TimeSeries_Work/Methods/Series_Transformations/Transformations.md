# Преобразования


Для составления формулы расчёта вычисляемого ряда доступно использование
 методов преобразования.


[Математические преобразования](Mathematical_transformations.htm):


		 Функция
		 Краткое описание


		 Лаг
		 Сдвигает ряд вперед на заданное количество точек во временном
		 периоде.


		 Опережение
		 Сдвигает ряд назад на заданное количество точек во временном
		 периоде.


		 Темп прироста
		 Осуществляет расчет темпа прироста в процентах значений ряда
		 к значениям выбранного периода.


		 Разность
		 Осуществляет расчет разности за определённый период.


		 Разность логарифмов
		 Осуществляет расчет разности логарифмов за определённый период.


		 Коэффициент роста
		 Осуществляет расчет коэффициента роста значений ряда за
		 определённый период.


[Методы накопления](Accumulation.htm):


		 Функция
		 Краткое описание


		 Накопленный минимум
		 Преобразует данные с использованием метода накопления «Минимум».


		 Накопленный максимум
		 Преобразует данные с использованием метода накопления «Максимум».


		 Накопленное среднее
		 Преобразует данные с использованием метода накопления «Среднее».


		 Накопленное стандартное отклонение
		 Преобразует данные с использованием метода накопления «Стандартное отклонение».


		 Накопленная сумма
		 Преобразует данные с использованием метода накопления «Сумма».


		 Накопленное умножение
		 Преобразует данные с использованием метода накопления «Произведение».


[Методы обработки пропусков](Missing_Data.htm):


		 Функция
		 Краткое описание


		 Геометрическое заполнение пропусков
		 Осуществляет геометрическое заполнение пропусков в значениях
		 ряда.


		 Линейное заполнение пропусков
		 Осуществляет линейное заполнение пропусков в значениях ряда.


		 Повторное заполнение пропусков
		 Осуществляет заполнение пропусков в значениях ряда предыдущими
		 непустыми значениями.


		 Сплайновое заполнение пропусков
		 Осуществляет сплайновое заполнение пропусков в значениях ряда.


		 Заполнение пропусков заданным значением
		 Осуществляет заполнение пропусков в данных ряда заданным значением.


		 Темп роста
		 Осуществляет заполнение пропусков исходя из темпа роста существующих
		 значений.


		 Темп роста к предыдущему периоду
		 Осуществляет заполнение пропусков исходя из темпа роста существующих
		 значений к предыдущему периоду.


		 Следующее значение
		 Осуществляет заполнение пропусков в данных ряда следующим непустым
		 значением.


		 Темп роста к следующему периоду
		 Осуществляет заполнение пропусков исходя из темпа роста существующих
		 значений к следующему периоду.
