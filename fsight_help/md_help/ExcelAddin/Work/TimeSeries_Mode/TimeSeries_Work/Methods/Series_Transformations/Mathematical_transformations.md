# Математические преобразования


Для составления формулы расчёта вычисляемого ряда доступно использование
 математических преобразований. Входят в группу «[Преобразования](Transformations.htm)».


Доступные математические преобразования:


		 Функция
		 Краткое описание


		 [Лаг](Lag.htm)
		 Сдвигает ряд вперед на заданное количество точек во временном
		 периоде.


		 [Опережение](Lead.htm)
		 Сдвигает ряд назад на заданное количество точек во временном
		 периоде.


		 [Темп прироста](pch.htm)
		 Осуществляет расчёт темпа прироста в процентах значений ряда
		 к значениям выбранного периода.


		 [Разность](diff.htm)
		 Осуществляет расчёт разности соседних значений ряда за определённый
		 период.


		 [Разность логарифмов](dlog.htm)
		 Осуществляет расчёт разности логарифмов соседних точек ряда
		 за определенный период.


		 [Коэффициент роста](Ratio.htm)
		 Осуществляет расчёт коэффициента роста значений ряда за определенный
		 период.
