# Экспонента


Функция «Экспонента» возвращает
 результат возведения числа «е» в степень, определяемую значениями ряда.
 Входит в группу [арифметических функций](Arithmetic.htm).


[![](../../../../../Opened.gif)![](../../../../../Closed.gif)Для применения
 функции](javascript:TextPopup(this))


		- В таблице данных выделите один или несколько рядов.


		- Нажмите кнопку ![](../Arithmetic_button.png) «Арифметика»,
		 расположенную на вкладке «Вычисления» ленты
		 инструментов.


		- В раскрывающемся списке кнопки выберите вариант «Экспонента».


После применения функции в таблицу данных для каждого выделенного ряда
 будет добавлен ряд с наименованием вида «Exp(<Имя_Ряда>)»,
 содержащий результаты расчета. Например:
