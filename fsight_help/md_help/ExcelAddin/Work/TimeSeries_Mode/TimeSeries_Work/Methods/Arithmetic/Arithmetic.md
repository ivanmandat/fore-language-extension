# Арифметика


Для составления формулы расчёта вычисляемого ряда доступно использование
 арифметических функций:


		 Функция
		 Краткое описание


		 [Log](Log.htm)
		 Возвращает логарифм ряда по заданному основанию.


		 [Ln](Ln.htm)
		 Возвращает натуральный логарифм ряда.


		 [Экспонента](Exponent.htm)
		 Возвращает результат возведения числа «е» в степень, определяемую
		 значениями ряда.


		 [Mod](Mod.htm)
		 Возвращает остаток от целочисленного деления значений ряда
		 на заданное число.


		 [Сложение](Summation.htm)
		 Возвращает результат сложения двух и более рядов.


		 [Умножение](Multiplication.htm)
		 Возвращает результат умножения двух и более рядов.


		 [Абсолютное значение](Absolute_value.htm)
		 Возвращает абсолютные значения ряда.


		 [Округление](Rounding.htm)
		 Возвращает значения ряда, округленные до целых чисел.
