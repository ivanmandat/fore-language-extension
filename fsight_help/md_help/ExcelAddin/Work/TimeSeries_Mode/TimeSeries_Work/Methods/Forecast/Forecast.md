# Прогноз


Для составления формулы расчёта вычисляемого ряда доступно использование
 методов прогнозирования:


		 Функция
		 Краткое описание


		 [Тренд](Trend.htm)
		 Осуществляет моделирование значений ряда установленным методом.


		 [Экспоненциальное
		 сглаживание](Exponential_smoothing.htm)
		 Осуществляет моделирование значений ряда методом «Экспоненциального сглаживания».


		 [ARIMA](ARIMA.htm)
		 Осуществляет моделирование значений ряда методом «ARIMA».


		 [Грей-метод](Grey_forecast.htm)
		 Осуществляет моделирование значений ряда «Грей-методом».
