# Настройка параметров экспоненциального сглаживания


Для настройки базовых параметров [экспоненциального
 сглаживания](Exponential_smoothing.htm) используйте вкладку «Параметры»
 на панели свойств.


[![](../../../../../Opened.gif)![](../../../../../Closed.gif)Для отображения
 вкладки](javascript:TextPopup(this))


		- Убедитесь, что панель
		 свойств отображается.


		- Выделите в таблице данных ряд, рассчитанный методом «Экспоненциальное сглаживание».


		- На панели свойств установите переключатель «Ряд».


		- Перейдите на вкладку «Параметры».


![](ExpSmoth_panel.png)


Задайте параметры метода:


[![](../../../../../Opened.gif)![](../../../../../Closed.gif)Сезонный
 эффект](javascript:TextPopup(this))


	Для учёта при расчёте сезонного эффекта:


		- Установите флажок «Сезонный
		 эффект». Если флажок установлен, то при расчёте учитывается
		 [сезонный
		 эффект](Lib.chm::/02_Time_series_analysis/UiModelling_ExpSmooth_season.htm): если в исходном ряде наблюдаются достаточно
		 постоянные периодические отклонения в абсолютном выражении от
		 варьирующего во времени среднего уровня ряда с заранее известным
		 периодом, будет выделена сезонная составляющая. Для отражения
		 сезонной особенности в модели используется коэффициент «Дельта». Временной ряд может
		 быть разложен на систематическую и случайную составляющие.


		- Выберите метод, применяемый для выделения сезонной составляющей
		 исходного ряда:


			- Аддитивный.
			 Используется по умолчанию. Ряд рассматривается как сумма систематической
			 и нерегулярной составляющих;


			- Мультипликативный.
			 Ряд рассматривается как произведение систематической и нерегулярной
			 составляющих.


[![](../../../../../Opened.gif)![](../../../../../Closed.gif)Период
 сезонности](javascript:TextPopup(this))


	Если выбран какой-либо сезонный эффект, задайте длину периода сезонности.
	 Значение по умолчанию - «4».


	Важно.
	 Минимальное число наблюдений ряда должно удовлетворять неравенству:
	 число наблюдений ≥ (длина
	 периода сезонности * 2 + 1). Если данное
	 ограничение не выполняется, то ряд не будет рассчитан и будет заполнен
	 пустыми значениями.


[![](../../../../../Opened.gif)![](../../../../../Closed.gif)Рост](javascript:TextPopup(this))


	Временные ряды часто имеют тенденцию некоторого роста (убывания).
	 Если в модели экспоненциального сглаживания необходимо использовать
	 коэффициент, учитывающий рост («Гамма»):


		- Установите флажок «Рост».


		- Выберите [модель
		 роста](Lib.chm::/02_Time_series_analysis/UiModelling_ExpSmooth_growthhtm.htm):


			- аддитивный;


			- экспоненциальный;


			- затухающий.


[![](../../../../../Opened.gif)![](../../../../../Closed.gif)Альфа.
 Дельта. Гамма. Фи](javascript:TextPopup(this))


	Задайте значения коэффициентов модели. Набор доступных коэффициентов
	 зависит от выбранных параметров [роста](#growth) и [сезонного
	 эффекта](#seas_effect).


	Если установлен переключатель «Подобрать»,
	 то коэффициент оценивается автоматически, иначе значение коэффициента
	 задается вручную в соответствующем редакторе чисел. Для задания параметров
	 автоматической оценки коэффициентов используйте вкладку «[Автоподбор параметров](ExpSmooth_Autofit.htm)» на
	 панели свойств.


	Параметры экспоненциального сглаживания:


		- Альфа. Параметр
		 сглаживания. Необходим для всех моделей экспоненциального сглаживания.
		 Допустимые значения: (0, 1];


		- Дельта. Сезонный
		 сглаживающий параметр. Необходим в сезонных моделях. Допустимые
		 значения: [0, 1];


		- Гамма. Параметр
		 сглаживания тренда. Используется в моделях с линейным и экспоненциальным
		 трендом и в моделях с затухающим трендом в рядах без сезонной
		 составляющей. Допустимые значения: [0, 1];


		- Фи. Параметр модификации
		 тренда. Используется в моделях с затухающим трендом. Влияют на
		 то, как сильно изменения в тренде будут влиять на оценки тренда
		 для последовательных прогнозов. Допустимые значения: (0, 1).


[![](../../../../../Opened.gif)![](../../../../../Closed.gif)Значимость
 доверительных границ](javascript:TextPopup(this))


	Задайте уровень значимости доверительных границ прогнозного ряда.
	 Диапазон допустимых значений: (0, 1). Значение по умолчанию - «0,95».
