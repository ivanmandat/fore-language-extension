# Настройка базовых параметров агрегации по рядам


Для настройки базовых параметров [агрегации
 по рядам](Aggregation_series.htm) используйте вкладку «Параметры»
 на панели свойств:


[![](../../../../../Opened.gif)![](../../../../../Closed.gif)Для отображения
 вкладки](javascript:TextPopup(this))


		- Убедитесь, что панель
		 свойств отображается.


		- Выделите в таблице данных ряд, рассчитанный методом «Агрегация».


		- На панели свойств установите переключатель «Ряд».


		- Перейдите на вкладку «Параметры».


![](Panel_Aggr_param.png)


Параметры на вкладке соответствуют параметрам окна «[Агрегация](Aggregation_series.png)»,
 отображаемого при создании вычисляемого ряда.
