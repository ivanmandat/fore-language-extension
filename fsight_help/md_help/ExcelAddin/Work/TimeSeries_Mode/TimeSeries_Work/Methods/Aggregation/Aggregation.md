# Агрегация


Для составления формулы расчёта вычисляемого ряда доступно использование
 методов агрегации:


		 Функция
		 Краткое описание


		 [Агрегация (по
		 показателям)](Aggregation_series.htm)
		 Агрегирует данные по значениям атрибутов временных рядов.


		 [Агрегация (по
		 времени)](Aggregation_Calendar.htm)
		 Агрегирует данные по уровням календарной динамики временных
		 рядов.


		 [Интерполяция](Interpolation.htm)
		 Рассчитывает данные нижнего уровня на основе данных верхнего
		 уровня.
