# Сглаживание


Для составления формулы расчёта вычисляемого ряда доступно использование
 методов сглаживания:


		 Функция
		 Краткое описание


		 [Скользящее среднее](Moving_Average.htm)
		 Сглаживает ряд методом «Скользящего
		 среднего».


		 [Фильтр Ходрика-Прескотта](Hpf.htm)
		 Сглаживает ряд методом «Фильтр
		 Ходрика-Прескотта».


		 [Фильтр Бакстера-Кинга](Bpf.htm)
		 Сглаживает ряд методом «Фильтр
		 Бакстера-Кинга».


		 [Выделение сезонности (X11)](X11.htm)
		 Осуществляет сезонную декомпозицию и корректировку данных методом
		 «Census2».
		Примечание.
		 Метод X11 поддерживается только в ОС Windows.
