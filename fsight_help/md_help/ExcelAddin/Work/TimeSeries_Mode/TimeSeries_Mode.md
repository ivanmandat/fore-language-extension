# Работа с временными рядами


Режим предназначен для работы с данными, полученными из следующих источников
 данных:


	- стандартные кубы;


	- базы данных временных рядов;


	- рабочие книги.


Процесс работы с надстройкой в режиме анализа временных рядов состоит
 из следующих этапов:


	- Выполните [получение данных](../Excel_Work.htm#get_data).


	- Настройте [таблицу данных](Table_Work.htm).


	- При необходимости добавьте [отдельные
	 значения](../AddFormula.htm#value) или [элементы измерений](../AddFormula.htm#element)
	 из кубов в ячейки на лист рабочей книги


	- Если требуется, добавьте и настройте:


		- [временные](TimeSeries_Work/TimeSeries_Work.htm)
		 и [вычисляемые
		 ряды](TimeSeries_Work/Calculated_Series_work.htm);


		- [карту](Map.htm);


	- Настройте [графическое
	 представление данных](TimeSeries_Work/Chart_work.htm).


	- Если требуется, выполните [валидацию
	 данных](UiAnalyticalArea.chm::/Analysis/Data_validation.htm).


	- Если требуется, выполните [интеллектуальный анализ
	 данных](DataMining.chm::/DataMining_Title.htm).


	- Сохраните [отчёт](../Olap_Mode/SaveOlap.htm)
	 или его [копию](../Olap_Mode/SaveOlap.htm#copy).


Все операции в режиме анализа данных временных рядов выполняются с помощью
 следующих элементов интерфейса:


	- вкладок ленты инструментов: «FORESIGHT», «Таблица»,
	 «Диаграмма», «Вычисления»,
	 «Расширенная аналитика»;


	- панелей: «Панель
	 свойств», «Дерево
	 рядов», «Панель
	 результатов».
