# Редактирование данных


Для редактирования данных в ячейках выделите ячейку на листе отчета
 и введите необходимые данные.


Для сохранения изменённых данных в источник данных нажмите кнопку ![](../../../Interface/save%20date.png) «Сохранить изменения»,
 расположенную на вкладке «Таблица»
 ленты инструментов.


Для отмены изменённых данных нажмите кнопку ![](../../../Interface/undo_changes.png) «Отменить изменения», расположенную
 на вкладке «Таблица» ленты инструментов.


## Настройка формата числа


Для настройки формата числа используйте кнопки, расположенные в группе
 «Формат числа» на вкладке «Таблица» ленты инструментов.
