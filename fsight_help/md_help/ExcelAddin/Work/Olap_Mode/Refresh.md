# Обновление данных отчёта


Для обновления данных на текущем листе отчета нажмите кнопку ![](../b_Refresh.png)
 «Обновить», расположенную в группе «Отчет»
 на вкладке «FORESIGHT»
 ленты инструментов.


## Настройка заголовка отчёта


При работе с отчетами в Microsoft Excel доступна настройка заголовка
 отчета.


Для отображения заголовка отчета:


	- Нажмите кнопку ![](../../Interface/Title.png)«Заголовок», расположенную в группе
	 «Вид» на вкладке «FORESIGHT» ленты инструментов.


	- В раскрывающемся меню кнопки выберите формат заголовка:


		- Простой. Заголовок
		 отображается в виде текста;


		- Элемент управления.
		 Заголовок отображается в виде гиперссылки, при нажатии на которую
		 открывается иерархический список для выбора элемента измерения.


Примечание.
 Формат заголовка «Элемент управления»
 доступен только при использовании аналитических запросов (OLAP).


После выполнения последовательности действий для текущего отчета будет
 отображен заголовок.


Совет. Для быстрого
 отображения заголовка отчета нажмите на верхнюю часть кнопки ![](../../Interface/Title.png)«Заголовок». Для отчета будет отображен
 заголовок в виде текста.


Для скрытия заголовка отчета
 переведите кнопку ![](../../Interface/Title.png)«Заголовок»
 в ненажатое состояние.


Для настройки содержимого заголовка:


	- Переведите заголовок в режим редактирования. Для этого дважды
	 щелкните по нему левой кнопкой мыши. Составляющие части заголовка
	 будут представлены в виде тегов. Например: «&[Name]
	 &[Datasource] &[Date]».


	- В режиме редактирования заголовка можно добавлять текст вручную
	 или использовать подстановки, расположенные в группе «Вставка»
	 на вкладке «Заголовок» ленты
	 инструментов: название отчета, отметка по измерению или текущая дата/время.


Примечание.
 Подстановка «Отметка по измерению»
 доступна только при использовании аналитических запросов (OLAP).


	- Для выхода из режима редактирования установите курсор мыши вне
	 панели заголовка.


Для настройки формата содержимого
 заголовка отчета используйте стандартные раскрывающиеся списки «Шрифт», «Размер»
 и кнопки настройки начертания текста, расположенные в группе «Шрифт»
 на вкладке «Заголовок» ленты инструментов.


## Экспорт отчета


Для экспорта книги Microsoft Excel в экспресс-отчет в формат PPEXPRESS
 (*.ppexpress) или клиентский куб в формате PPCUBE (*.ppcube) нажмите кнопку
 ![](../b_Export.png) «Экспорт»,
 расположенную в группе «Экспорт и печать»
 на вкладке «FORESIGHT»
 ленты инструментов. В диалоге сохранения файла укажите место для сохранения
 результатов экспорта.
