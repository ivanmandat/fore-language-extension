# Работа с аналитическими запросами
 (OLAP)


Режим предназначен для работы с данными, полученными из следующих источников
 данных:


	- все виды кубов;


	- базы данных временных рядов;


	- экспресс-отчёты;


	- файлы в формате *.ppexpress, *.ppcube.


Процесс работы с надстройкой в режиме аналитических запросов состоит
 из следующих этапов:


	- Если требуется, измените и настройте [источник
	 данных](Data_Sources.htm).


	- Если требуется, настройте [отображение
	 измерений](../SettingViewDimension.htm) и выполните [отметку
	 элементов измерений](SelectionDimensionElements.htm).


	- Настройте [таблицу данных](Table_Work/Table_Work.htm).


	- Если требуется, добавьте и настройте [диаграмму](Chart.htm).


	- Если требуется, добавьте и настройте [пузырьковую
	 диаграмму](BubbleChart.htm).


	- Если требуется, добавьте и настройте [пузырьковое
	 дерево](BubbleTree.htm).


	- Если требуется, добавьте и настройте [плоское
	 дерево](TreeMap.htm).


	- При необходимости добавьте [отдельные
	 значения](../AddFormula.htm#value) или [элементы измерений](../AddFormula.htm#element)
	 из кубов в ячейки на лист рабочей книги.


	- Если требуется, [обновите данные](Refresh.htm).


	- Если требуется, настройте [заголовок](Refresh.htm#title)
	 отчёта.


	- При необходимости [экспортируйте отчёт](Refresh.htm#export).


	- Если требуется, выполните [пакетные
	 операции](Batch_Operations/Batch_Operations.htm).


	- Сохраните [отчёт](SaveOlap.htm)
	 или его [копию](SaveOlap.htm#copy).
