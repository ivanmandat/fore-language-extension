# Работа с пузырьковой диаграммой


Пузырьковая диаграмма - точечная
 диаграмма, в которой точки данных заменены пузырьками. Размер пузырьков
 является дополнительным измерением данных. На пузырьковой диаграмме отсутствует
 ось категорий. Горизонтальная и вертикальная оси являются осями значений.


Пузырьковая диаграмма позволяет отобразить наборы из четырех значений:


	- 1-е значение: по оси X;


	- 2-е значение: по оси Y;


	- 3-е значение: размер пузырьков;


	- 4-е значение: цвет пузырьков.


Для начала работы с пузырьковой диаграммой:


	- [Получите данные](../Excel_Work.htm).


	- Нажмите кнопку ![](b_BubbleChart.png) «Пузырьковая
	 диаграмма», расположенную в группе «Вставка»
 на вкладке «FORESIGHT»
 ленты инструментов.


Пузырьковая диаграмма создается на отдельном листе в книге Microsoft
 Excel. Лист с пузырьковой диаграммой вставляется после активного листа
 рабочей книги Microsoft Excel.


Примечание.
 Одна [таблица](Table_Work/Table_Work.htm) может служить источником
 данных только для одной пузырьковой диаграммы.
