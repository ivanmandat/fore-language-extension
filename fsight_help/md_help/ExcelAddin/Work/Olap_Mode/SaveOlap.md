# Сохранение данных отчёта


Для сохранения данных на текущем листе:


	- Нажмите на нижнюю часть кнопки ![](../b_Save.png) «Сохранить»,
	 расположенную в группе «Отчет»
 на вкладке «FORESIGHT»
 ленты инструментов.


	- В раскрывающемся меню кнопки выберите пункт «Сохранить».


## Создание копии отчёта


Для создания копии экспресс-отчета в репозиторий:


	- Нажмите на нижнюю часть кнопки ![](../b_Save.png) «Сохранить»,
	 расположенную в группе «Отчет»
 на вкладке «FORESIGHT»
 ленты инструментов.


	- В раскрывающемся меню кнопки выберите пункт «Сохранить
	 как».


В открывшемся окне [навигатора
 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm) укажите путь для сохранения отчета.
