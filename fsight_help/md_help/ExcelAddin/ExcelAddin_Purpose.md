# Построение отчётов с использованием надстройки Foresight
 Add-in for Excel


Для расширения возможностей приложения Microsoft Excel используйте надстройку
 Foresight Add-in for Excel:


![](Interface/ExcelAddIn.png)


Надстройка Foresight
 Add-in for Excel - это устанавливаемый компонент, который позволяет добавить
 новые возможности к приложению Microsoft Excel.


Примечание.
 Поддерживаются версии Microsoft Office 2016, Microsoft Office 2019​​.


С помощью надстройки можно анализировать данные, создавать аналитические
 записки и отчеты с использованием данных, содержащихся в репозитории платформы.

			 Управление источниками данных:


				- все виды кубов;


				- базы данных временных рядов;

				- стандартные кубы;


				- базы данных временных рядов;


				- рабочие книги.
