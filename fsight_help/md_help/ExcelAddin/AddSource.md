# Выбор источника данных


Для выбора источника данных:


	- В группе «Отчёт», расположенной
	 на вкладке «FORESIGHT»
	 ленты инструментов, выберите тип отчёта:


		- аналитические запросы;


		- анализ временных рядов.


	- В раскрывающемся меню кнопки отчёта выберите источник данных:


		- Из репозитория.
		 Будет открыто [окно
		 регистрации](GetStarted.chm::/GetStarted/Get_Started.htm).
		 Войдите в требуемый репозиторий и укажите источник данных;


		- Из файла. Будет
		 открыто окно выбора источника данных, сохраненного в виде экспресс-отчета
		 в формате PPEXPRESS (*.ppexpress) или клиентского куба в формате
		 (*.ppcube);


Примечание.
 Пункт меню доступен только для типа отчета «Аналитические
 запросы».


		- Известные источники.
		 Если при работе с надстройкой было выполнено подключение к репозиторию,
		 то данное меню будет содержать список репозиториев, из которых
		 можно выбрать источник данных.


	- После выбора источника данных будет
	 отображен диалог для определения места размещения данных. Выберите
	 место размещения данных:


		- имеющийся лист.
		 Данные будут размещены на текущем листе рабочей книги. По умолчанию
		 данные добавляются, начиная с ячейки A1. При необходимости, измените
		 диапазон в соответствующем поле. Для корректной работы надстройки
		 на одном листе рабочей книги Microsoft Excel следует размещать
		 данные только из одного источника;


		- новый лист. Будет
		 создан новый лист рабочей книги для размещения данных.


В результате выполненных действий данные будут добавлены на лист по
 заданным параметрам.


Примечание.
 Если источник данных параметрический, то после выбора места размещения
 для задания значений параметров будет отображен диалог «[Установка значений параметров](uinav.chm::/02_Navigator/General_Principles_of_Work.htm#open_param_object)»,
 в котором задайте значения параметров.
