# ICubeSerializer


## Описание


Интерфейс ICubeSerializer содержит
 методы для инициализации процессов сохранения и загрузки куба из файла.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [FileName](ICubeSerializer.FileName.htm)


		 Свойство FileName определяет
		 имя файла, в который будет сохранен куб.


		 ![](../../Property_Image.gif)
		 [StorageType](ICubeSerializer.StorageType.htm)


		 Свойство StorageType
		 определяет тип хранилища, в который будет помещен в куб.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [LoadFromFile](ICubeSerializer.LoadFromFile.htm)


		 Метод LoadFromFile
		 осуществляет загрузку куба, сохраненного в файл.


		 ![](../../Sub_Image.gif)
		 [PreCheck](ICubeSerializer.PreCheck.htm)


		 Метод PreCheck осуществляет
		 проверку прав на экспорт у данного пользователя.


		 ![](../../Sub_Image.gif)
		 [SaveToFile](ICubeSerializer.SaveToFile.htm)


		 Метод SaveToFile создает
		 файл и инициализирует процесс сохранения куба в файл.
