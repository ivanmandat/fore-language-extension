# ICubeSerializer.FileName


## Синтаксис


FileName: String;


## Описание


Свойство FileName определяет
 имя файла, в который будет сохранен куб.


## Комментарии


Значение не задается при использовании [ICubeSerializer.SaveToFile](ICubeSerializer.SaveToFile.htm).
 Имя файла будет взято из параметра FileName метода.
