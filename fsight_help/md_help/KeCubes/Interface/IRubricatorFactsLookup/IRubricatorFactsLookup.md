# IRubricatorFactsLookup


## Описание


Интерфейс IRubricatorFactsLookup
 содержит свойства и методы, предназначенные для поиска среди наблюдений
 показателей.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Current](IRubricatorFactsLookup.Current.htm)


		 Свойство Current возвращает
		 текущую запись в массиве найденных показателей.
