# IRubricatorFactsLookup.Open


## Синтаксис


Open(Options: [DictionaryCursorOptions](KeRds.chm::/Enums/DictionaryCursorOptions.htm));


## Параметры


Options. Параметр открытия курсора.


## Описание


Метод Open осуществляет открытие курсора, позволяющего перемещаться по данным.


## Комментарии


Для закрытия курсора используйте метод [IRubricatorFactsLookup.Close](IRubricatorFactsLookup.Close.htm).
