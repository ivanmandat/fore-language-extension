# IRubricatorFactor


## Описание


Интерфейс IRubricatorFactor
 содержит свойства показателя, входящего в базу данных временных рядов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CompoundFactor](IRubricatorFactor.CompoundFactor.htm)
		 Свойство CompoundFactor
		 возвращает составной ключ временного ряда.


		 ![](../../Property_Image.gif)
		 [FactData](IRubricatorFactor.FactData.htm)
		 Свойство FactData возвращает
		 данные показателя.


		 ![](../../Property_Image.gif)
		 [Factor](IRubricatorFactor.Factor.htm)
		 Свойство Factor возвращает
		 ключ показателя.


		 ![](../../Property_Image.gif)
		 [Rubricator](IRubricatorFactor.Rubricator.htm)
		 Свойство Rubricator
		 возвращает данные базы, содержащей показатель.


## Методы


		 Имя свойства
		 Краткое описание


		 ![](../../Sub_Image.gif)

		 ![](../../Sub_Image.gif)
		 [GetSubjectEffectiveRights](IRubricatorFactor.GetSubjectEffectiveRights.htm)
