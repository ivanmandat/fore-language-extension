# IVirtualCubeExecuteCallback.OnBeforeExecuteSources


## Синтаксис


		OnBeforeExecuteSources(Value: [IVirtualCubeSourceSelections](../IVirtualCubeSourceSelections/IVirtualCubeSourceSelections.htm));


## Параметры


Value. Коллекция отметок источников
 виртуального куба.


## Описание


Метод OnBeforeExecuteSources
 реализует событие, наступающее перед расчётом источников виртуального
 куба.


## Комментарии


Используя параметр события Value,
 можно получить доступ к отметкам источников и изменить их при необходимости.
