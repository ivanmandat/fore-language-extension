# IVirtualCubeExecuteCallback


Сборка: Cubes;


## Описание


Интерфейс IVirtualCubeExecuteCallback
 предназначен для работы с событиями, которые наступают при расчёте виртуального
 куба.


## Иерархия наследования


IVirtualCubeExecuteCallback


## Комментарии


Методы интерфейса должны быть переопределены в пользовательском классе.
 Экземпляр класса для обработки событий задаётся в свойстве [IVirtualCubeInstance.ExecuteCallback](../IVirtualCubeInstance/IVirtualCubeInstance.ExecuteCallback.htm).
