# ICubeExecuteDataTypes


Сборка: Cubes;


## Описание


Интерфейс ICubeExecuteDataTypes
 предназначен для работы с информацией о способе формирования значений
 в ячейках.


## Иерархия наследования


ICubeExecuteDataTypes


## Комментарии


Информацию о способе формирования значений в ячейках можно получить
 после расчёта в свойстве [ICubeExecuteResult.DataTypes](../ICubeExecuteResult/ICubeExecuteResult.DataTypes.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DataType](ICubeExecuteDataTypes.DataType.htm)


		 Свойство DataType возвращает
		 способ формирования данных в ячейке с указанной координатой.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddCoord](ICubeExecuteDataTypes.AddCoord.htm)


		 Метод AddCoord задаёт
		 способ формирования данных для ячейки с заданной координатой.


		 ![](../../Sub_Image.gif)
		 [AddSelectionSet](ICubeExecuteDataTypes.AddSelectionSet.htm)


		 Метод AddSelectionSet
		 задаёт способ формирования данных для множества ячеек, которые
		 соответствуют указанной отметке.


		 ![](../../Sub_Image.gif)
		 [Clear](ICubeExecuteDataTypes.Clear.htm)


		 Метод Clear очищает
		 список ячеек, для которых был задан способ формирования данных.
