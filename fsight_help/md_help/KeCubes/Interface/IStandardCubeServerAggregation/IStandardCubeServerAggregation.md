# IStandardCubeServerAggregation


## Описание


Интерфейс IStandardCubeServerAggregation
 содержит свойства и методы для работы с агрегацией, рассчитываемой на
 сервере.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AggregationType](IStandardCubeServerAggregation.AggregationType.htm)
		 Свойство AggregationType
		 определяет метод агрегации.


		 ![](../../Property_Image.gif)
		 [CustomAggregateFunction](IStandardCubeServerAggregation.CustomAggregateFunction.htm)
		 Свойство CustomAggregateFunction
		 определяет пользовательский метод расчета агрегации.


		 ![](../../Property_Image.gif)
		 [FactKey](IStandardCubeServerAggregation.FactKey.htm)
		 Свойство FactKey определяет
		 факт, для которого рассчитывается агрегация.
