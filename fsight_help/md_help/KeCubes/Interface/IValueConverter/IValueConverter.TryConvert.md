# IValueConverter.TryConvert


## Синтаксис


TryConvert(Value: Variant; Var Result: Variant):
 Boolean;


## Параметры


Value. Значение, которое надо
 попробовать преобразовать;


Result. Результат преобразования.


## Описание


Метод TryConvert пробует
 выполнить преобразование данных и возвращает признак успешности преобразования.


## Комментарии


Допустимые значения, возвращаемые методом:


	- True. Преобразование
	 выполнено успешно;


	- False. Преобразование
	 выполнить не удалось.
