# IInMemManager.UseCachedDimensions


## Синтаксис


UseCachedDimensions: Boolean;


## Описание


Свойство UseCachedDimensions
 возвращает признак использования кеша In-Memory при работе со справочниками.


## Комментарии


По умолчанию кеш In-Memory для справочников не используется. Управление
 кешем осуществляется через [реестр](Setup.chm::/UiWebSetup/03_Setup_Web/BI_Server_Registry_Key.htm#inmem)/[файл
 settings.xml](Setup.chm::/UiWebSetup/03_Setup_Web/BI_Server_Settings_XML.htm#inmem).
