# IInMemManager.UpdateBySelection


## Синтаксис


		UpdateBySelection(Dest: [ICubeInstanceDestination](../ICubeInstanceDestination/ICubeInstanceDestination.htm);
		 Selection: [IDimSelectionSet](KeDims.chm::/interface/IDimSelectionSet/IDimSelectionSet.htm));


## Параметры


Dest. Отстроенный вариант отображения куба.


Selection. Отметка, в соответствии
 с которой будет осуществлён пересчёт матрицы с данными.


## Описание


Метод UpdateBySelection осуществляет
 обновление данных в кеше в соответствии с заданными параметрами.
