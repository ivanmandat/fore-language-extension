# IInMemManager.ResetMatrix


## Синтаксис


ResetMatrix(Dest: [ICubeInstanceDestination](../ICubeInstanceDestination/ICubeInstanceDestination.htm));


## Параметры


Dest. Отстроенный вариант отображения
 куба.


## Описание


Метод ResetMatrix удаляет из
 кеша матрицу с данными для указанного варианта отображения куба.


## Комментарии


Для создания новой кешируемой матрицы используйте методы [PrepareMatrix](IInMemManager.PrepareMatrix.htm)/[PrepareMatrixPartial](IInMemManager.PrepareMatrixPartial.htm).
