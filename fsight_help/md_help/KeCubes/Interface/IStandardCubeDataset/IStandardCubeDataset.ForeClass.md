# IStandardCubeDataset.ForeClass


## Синтаксис


ForeClass: String;


## Описание


Свойство ForeClass возвращает
 наименование класса, который используется для обработки процесса формирования
 запроса на извлечение данных.


## Комментарии


Установка модуля и класса для обработки процесса формирования запроса
 осуществляется методом [SetCallback](IStandardCubeDataset.SetCallback.htm).
