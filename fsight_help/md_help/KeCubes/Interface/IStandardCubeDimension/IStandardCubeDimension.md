# IStandardCubeDimension


## Описание


Интерфейс IStandardCubeDimension
 содержит свойства измерения стандартного куба.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Aggregator](IStandardCubeDimension.Aggregator.htm)
		 Свойство Aggregator
		 определяет настройки агрегации измерения по указанному факту куба.


		 ![](../../Property_Image.gif)
		 [AggregatorHierarchies](IStandardCubeDimension.AggregatorHierarchies.htm)
		 Свойство AggregatorHierarchies
		 определяет настройки агрегации по уровням [альтернативной
		 иерархии](UiNavObj.chm::/Cube/CreateCube/Master_Standart/UiMd_Cube_CreateCube_Master_Standart_7.htm#alt_agr) для указанного измерения фактов куба.


		 ![](../../Property_Image.gif)
		 [Binding](IStandardCubeDimension.Binding.htm)
		 Свойство Binding возвращает
		 параметры привязки измерения к источникам данных куба.


		 ![](../../Property_Image.gif)
		 [Cube](IStandardCubeDimension.Cube.htm)
		 Свойство Cube возвращает
		 объект-родитель.


		 ![](../../Property_Image.gif)
		 [Dimension](IStandardCubeDimension.Dimension.htm)
		 Свойство Dimension
		 возвращает структуру справочника репозитория, являющегося измерением
		 в кубе.


		 ![](../../Property_Image.gif)
		 [FactBindings](IStandardCubeDimension.FactBindings.htm)
		 Свойство FactBindings
		 возвращает параметры привязки измерения фактов к источникам данных
		 куба.


		 ![](../../Property_Image.gif)
		 [FactDimension](IStandardCubeDimension.FactDimension.htm)
		 Свойство FactDimension
		 определяет, является ли данное измерение - измерением фактов куба.


		 ![](../../Property_Image.gif)
		 [OpenDimension](IStandardCubeDimension.OpenDimension.htm)
		 Свойство OpenDimension
		 определяет открытое измерение куба.


		 ![](../../Property_Image.gif)
		 [QueryStrategy](IStandardCubeDimension.QueryStrategy.htm)
		 Свойство QueryStrategy
		 возвращает параметры стратегии фильтрации по данному измерению
		 куба.
