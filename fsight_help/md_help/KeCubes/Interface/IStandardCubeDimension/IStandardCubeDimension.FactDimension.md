# IStandardCubeDimension.FactDimension


## Синтаксис


FactDimension: Boolean;


## Описание


Свойство FactDimension определяет,
 является ли данное измерение - измерением фактов куба.


## Комментарии


Свойство актуально, если свойству [ExternalFactDimension](../IStandardCube/IStandardCube.ExternalFactDimension.htm)
 установлено значение True.


Для измерения фактов свойство FactDimension
 должно иметь значение True, для
 всех остальных измерений - False.
