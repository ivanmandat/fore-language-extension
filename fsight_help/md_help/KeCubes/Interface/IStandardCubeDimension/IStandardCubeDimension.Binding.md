# IStandardCubeDimension.Binding


## Синтаксис


Binding(Dataset: [IStandardCubeDataset](../IStandardCubeDataset/IStandardCubeDataset.htm)): [IStandardCubeDimensionBinding](../IStandardCubeDimensionBinding/IStandardCubeDimensionBinding.htm);


## Параметры


Dataset - источник данных куба.


## Описание


Свойство Binding возвращает параметры привязки измерения к источникам данных куба.
