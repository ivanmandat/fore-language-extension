# IRubricatorInstance.GetCompoundFactData


## Синтаксис


GetCompoundFactData(FactorKey: Variant; [Options:
 [DictionaryGetDataOptions](KeRds.chm::/Enums/DictionaryGetDataOptions.htm)
 = 0]): [IRubricatorFactData](../IRubricatorFactData/IRubricatorFactData.htm);


## Параметры


FactorKey. Составной ключ временного
 ряда;


Options. Параметр получения
 данных.


## Описание


Метод GetCompoundFactData возвращает
 данные о временном ряде по значению его составного ключа.
