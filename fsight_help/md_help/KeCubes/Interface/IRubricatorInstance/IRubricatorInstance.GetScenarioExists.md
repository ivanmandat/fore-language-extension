# IRubricatorInstance.GetScenarioExists


## Синтаксис


GetScenarioExists: Boolean;


## Описание


Метод GetScenarioExists возвращает
 признак того, применяются ли в базе данных временных рядов сценарии
 моделирования.


## Комментарии


Допустимые значения:


	- True. В БД временных
	 рядов используются сценарии моделирования;


	- False. В БД временных
	 рядов не используются сценарии моделирования.
