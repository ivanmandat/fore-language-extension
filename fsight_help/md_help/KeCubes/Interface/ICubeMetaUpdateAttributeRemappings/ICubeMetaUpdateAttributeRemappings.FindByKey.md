# ICubeMetaUpdateAttributeRemappings.FindByKey


## Синтаксис


FindByKey(Key: Integer): [ICubeMetaUpdateAttributeRemapping](../ICubeMetaUpdateAttributeRemapping/ICubeMetaUpdateAttributeRemapping.htm);


## Параметры


Key. Ключ атрибута показателей
 базы-копии.


## Описание


Метод FindByKey осуществляет
 поиск соответствия в коллекции по ключу атрибута показателей базы-копии.


## Комментарии


Метод возвращает значение Null,
 если настройки соответствия для указанного атрибута не найдены.
