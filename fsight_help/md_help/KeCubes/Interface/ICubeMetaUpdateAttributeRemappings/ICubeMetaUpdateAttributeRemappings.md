# ICubeMetaUpdateAttributeRemappings


## Описание


Интерфейс ICubeMetaUpdateAttributeRemappings
 содержит свойства и методы для работы с коллекцией соответствий атрибутов
 показателей копируемой базы и атрибутов показателей базы-копии.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeMetaUpdateAttributeRemappings.Count.htm)
		 Свойство Count возвращает
		 количество соответствий в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ICubeMetaUpdateAttributeRemappings.Item.htm)
		 Свойство Item возвращает
		 соответствие из коллекции по его индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](ICubeMetaUpdateAttributeRemappings.FindById.htm)
		 Метод FindById осуществляет
		 поиск соответствия в коллекции по идентификатору атрибута показателей
		 базы-копии.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ICubeMetaUpdateAttributeRemappings.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск соответствия в коллекции по ключу атрибута показателей базы-копии.
