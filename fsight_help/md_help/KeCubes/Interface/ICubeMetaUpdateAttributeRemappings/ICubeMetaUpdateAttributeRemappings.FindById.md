# ICubeMetaUpdateAttributeRemappings.FindById


## Синтаксис


FindById(Id: String): [ICubeMetaUpdateAttributeRemapping](../ICubeMetaUpdateAttributeRemapping/ICubeMetaUpdateAttributeRemapping.htm);


## Параметры


Id. Идентификатор атрибута
 показателей базы-копии.


## Описание


Метод FindById осуществляет
 поиск соответствия в коллекции по идентификатору атрибута показателей
 базы-копии.


## Комментарии


Метод возвращает значение Null,
 если настройки соответствия для указанного атрибута не найдены.
