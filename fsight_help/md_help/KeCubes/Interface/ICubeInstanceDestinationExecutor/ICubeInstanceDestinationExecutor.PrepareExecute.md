# ICubeInstanceDestinationExecutor.PrepareExecute


## Синтаксис


PrepareExecute(Selection: [IDimSelectionSet](kedims.chm::/interface/idimselectionset/idimselectionset.htm));


## Параметры


Selection. Отметка, по которой
 должен быть произведен расчет. Если значение данного параметра Null,
 то расчет осуществляется по всем элементам.


## Описание


Метод PrepareExecute подготавливает
 расчёт.
