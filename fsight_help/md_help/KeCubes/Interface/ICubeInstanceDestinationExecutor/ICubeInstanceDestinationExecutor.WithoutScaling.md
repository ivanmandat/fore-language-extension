# ICubeInstanceDestinationExecutor.WithoutScaling


## Синтаксис


WithoutScaling: Boolean;


## Описание


Свойство WithoutScaling определяет
 поведение при использовании [масштабирования](UiNavObj.chm::/Units/Units.htm).


## Комментарии


Если установлено значение True,
 то при выборке из куба результирующая матрица будет возвращать два значения
 для каждой точки. С индексом 0 - само значение. Последний доступный индекс
 - масштаб.
