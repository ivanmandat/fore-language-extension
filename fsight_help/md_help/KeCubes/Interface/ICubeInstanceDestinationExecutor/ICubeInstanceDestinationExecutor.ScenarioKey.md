# ICubeInstanceDestinationExecutor.ScenarioKey


## Синтаксис


ScenarioKey: Integer;


## Описание


Свойство ScenarioKey определяет
 ключ сценарного справочника, по которому будет производиться расчет.


## Комментарии


Для расчёта кубов параметр не используется и равен по умолчанию -1.
 При использовании контейнера моделирования возможен расчёт по различным
 сценариям моделирования кодирующиеся определёнными числовыми значениями.
