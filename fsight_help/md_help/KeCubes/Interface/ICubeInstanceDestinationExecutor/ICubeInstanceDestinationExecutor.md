# ICubeInstanceDestinationExecutor


## Описание


Интерфейс ICubeInstanceDestinationExecutor
 содержит свойства и методы для расчета отстроенного куба.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AffectSelection](ICubeInstanceDestinationExecutor.AffectSelection.htm)
		 Свойство AffectSelection
		 возвращает отметку измерений, изменившихся во время расчета.


		 ![](../../Property_Image.gif)
		 [AggregateDimSelection](ICubeInstanceDestinationExecutor.AggregateDimSelection.htm)
		 Свойство AggregateDimSelection
		 определяет признак исключения измерения из куба при агрегации
		 данных фиксированных измерений.


		 ![](../../Property_Image.gif)
		 [BasicUnit](ICubeInstanceDestinationExecutor.BasicUnit.htm)
		 Свойство BasicUnit
		 определяет единицу измерения для расчета.


		 ![](../../Property_Image.gif)
		 [Callback](ICubeInstanceDestinationExecutor.Callback.htm)
		 Свойство Callback определяет
		 объект, осуществляющий обработку ошибок, которые могут возникнуть
		 при расчёте куба.


		 ![](../../Property_Image.gif)
		 [Destination](ICubeInstanceDestinationExecutor.Destination.htm)
		 Свойство Destination
		 возвращает отстроенный куб-родитель.


		 ![](../../Property_Image.gif)
		 [DimensionsIntegrityCheck](ICubeInstanceDestinationExecutor.DimensionsIntegrityCheck.htm)
		 Свойство DimensionsIntegrityCheck
		 определяет, будет ли перед расчетом производиться проверка построения
		 справочников, входящих в отметку.


		 ![](../../Property_Image.gif)
		 [DrillThrough](ICubeInstanceDestinationExecutor.DrillThrough.htm)
		 Свойство DrillThrough
		 возвращает коллекцию элементов детализации при расчете куба.


		 ![](../../Property_Image.gif)
		 [IncludeAttachments](ICubeInstanceDestinationExecutor.IncludeAttachments.htm)
		 Свойство IncludeAttachments
		 определяет, будет ли при расчёте получена информация о вложениях.


		 ![](../../Property_Image.gif)
		 [Matrix](ICubeInstanceDestinationExecutor.Matrix.htm)
		 Свойство Matrix возвращает
		 результирующую матрицу с данными.


		 ![](../../Property_Image.gif)
		 [ScenarioKey](ICubeInstanceDestinationExecutor.ScenarioKey.htm)
		 Свойство ScenarioKey
		 определяет ключ сценарного справочника, по которому будет производиться
		 расчет.


		 ![](../../Property_Image.gif)
		 [WithoutScaling](ICubeInstanceDestinationExecutor.WithoutScaling.htm)
		 Свойство WithoutScaling
		 определяет поведение при использовании масштабирования.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [PerformExecute](ICubeInstanceDestinationExecutor.PerformExecute.htm)
		 Метод PerformExecute
		 осуществляет расчёт.


		 ![](../../Sub_Image.gif)
		 [PerformExecuteO](ICubeInstanceDestinationExecutor.PerformExecuteO.htm)
		 Метод PerformExecuteO
		 осуществляет расчёт куба в заданном режиме.


		 ![](../../Sub_Image.gif)
		 [PrepareExecute](ICubeInstanceDestinationExecutor.PrepareExecute.htm)
		 Метод PrepareExecute
		 подготавливает расчёт.
