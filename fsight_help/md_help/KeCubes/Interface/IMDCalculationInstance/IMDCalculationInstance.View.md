# IMDCalculationInstance.View


## Синтаксис


View(Source: [ICubeInstanceSource](../ICubeInstanceSource/ICubeInstanceSource.htm)): [ICalculatedCubeFormulasView](../ICalculatedCubeFormulasView/ICalculatedCubeFormulasView.htm);


## Параметры


Source. Куб-источник, для которого необходимо настроить параметры отображения элементов измерений.


## Описание


Свойство View возвращает параметры отображения элементов в измерениях указанного куба-источника и в формулах многомерного расчета.
