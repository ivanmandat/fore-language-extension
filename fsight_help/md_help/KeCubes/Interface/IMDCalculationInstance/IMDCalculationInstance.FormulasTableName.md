# IMDCalculationInstance.FormulasTableName


## Синтаксис


FormulasTableName: String;


## Описание


Свойство FormulasTableName возвращает наименование схемы и физическое имя таблицы, являющейся таблицей формул для многомерного расчета на сервере БД.
