# IMDCalculationInstance.CreateApply


## Синтаксис


CreateApply(Dataset: [IMDCalculationDestinationInstance](../IMDCalculationDestinationInstance/IMDCalculationDestinationInstance.htm)):
 [ICubeSelectionControlApply](../ICubeSelectionControlApply/ICubeSelectionControlApply.htm);


## Параметры


Dataset. Отстроенный приёмник
 данных многомерного расчета на сервере БД.


## Описание


Метод CreateApply создает объект,
 осуществляющий управление параметрами многомерного расчета через отметку
 в измерениях.
