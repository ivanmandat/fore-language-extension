# IRubricatorAutoPeriodDate.Offset


## Синтаксис


Offset: Integer;


## Описание


Свойство Offset определяет
 смещение для расчёта относительной даты.


## Комментарии


Свойство актуально, если [IRubricatorAutoPeriodDate.AutoDateType](IRubricatorAutoPeriodDate.AutoDateType.htm)
 принимает одно из значений:


	- [RubricatorAutoDateType](../../Enums/RubricatorAutoDateType.htm).Now;


	- [RubricatorAutoDateType](../../Enums/RubricatorAutoDateType.htm).DateStart;


	- [RubricatorAutoDateType](../../Enums/RubricatorAutoDateType.htm).DateEnd.
