# IRubricatorAutoPeriodDate.ShiftDateInwardsPeriod


## Синтаксис


ShiftDateInwardsPeriod: Boolean;


## Описание


Свойство ShiftDateInwardsPeriod определяет, смещать ли дату на одну точку внутрь периода, если она выходит за границы периода.


## Комментарии


Допустимые значения:


-
True. Смещать дату;


-
False. Значение по умолчанию. Не смещать дату.
