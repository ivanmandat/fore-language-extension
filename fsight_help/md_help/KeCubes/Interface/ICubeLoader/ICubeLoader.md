# ICubeLoader


## Описание


Интерфейс ICubeLoader содержит
 свойства объекта, предназначенного для загрузки данных в различные хранилища
 репозитория.


## Комментарии


Объекты, которые используют ICubeLoader:
  [Загрузчик
 данных](UiNavObj.chm::/Cube/Work_Cube/Loader/UiMd_Cube_Work_Cube_Loader.htm) и [мастер
 импорта данных](UiNavObj.chm::/Data_import_wizard/Data_import_wizard.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Destination](ICubeLoader.Destination.htm)
		 Свойство Destination
		 определяет куб-приёмник данных для загрузки.


		 ![](../../Property_Image.gif)
		 [FixInfo](ICubeLoader.FixInfo.htm)
		 Свойство FixInfo возвращает
		 параметры фиксации измерений куба-источника, которые не связаны
		 с измерениями в кубе-приёмнике.


		 ![](../../Property_Image.gif)
		 [MapInfo](ICubeLoader.MapInfo.htm)
		 Свойство MapInfo возвращает
		 параметры связи измерений куба-приёмника.


		 ![](../../Property_Image.gif)
		 [Source](ICubeLoader.Source.htm)
		 Свойство Source определяет
		 куб-источник данных для загрузки.
