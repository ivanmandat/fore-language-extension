# ICubeSegmentDimension


Сборка: Cubes;


## Описание


Интерфейс ICubeSegmentDimension
 предназначен для работы с измерением сегмента куба.


## Иерархия наследования


ICubeSegmentDimension


## Комментарии


Измерение сегмента куба можно получить, используя свойства и методы
 интерфейса [ICubeSegmentDimensions](../ICubeSegmentDimensions/ICubeSegmentDimensions.htm).
