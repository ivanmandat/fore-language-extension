# IRubricatorCreatorBindings.Item


## Синтаксис


Item(Index: Integer): [IRubricatorCreatorBinding](../IRubricatorCreatorBinding/IRubricatorCreatorBinding.htm);


## Параметры


Index. Индекс элемента в коллекции.


## Описание


Свойство Item возвращает указанный элемент коллекции.


## Комментарии


Индексация элементов в коллекции сквозная и начинается с нуля. Количество элементов в коллекции возвращает свойство [IRubricatorCreatorBindings.Count](IRubricatorCreatorBindings.Count.htm).
