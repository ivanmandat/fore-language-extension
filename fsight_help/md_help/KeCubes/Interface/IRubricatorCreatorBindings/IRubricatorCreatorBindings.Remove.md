# IRubricatorCreatorBindings.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс элемента коллекции.


## Описание


Метод Remove удаляет элемент с указанным индексом из коллекции привязок.


## Комментарии


Если удаление выполнено успешно, то метод возвращает значение True, иначе - False.
