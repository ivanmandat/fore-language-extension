# IRubricatorCreatorBindings.Add


## Синтаксис

## Описание


Метод Add добавляет новый элемент в коллекцию привязок.


## Комментарии


Общее количество элементов в коллекции возвращает свойство [IRubricatorCreatorBindings.Item](IRubricatorCreatorBindings.Item.htm).
