# IRubricatorCreatorBindings.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IRubricatorCreatorBinding](../IRubricatorCreatorBinding/IRubricatorCreatorBinding.htm);


## Параметры


Key. Ключ элемента коллекции.


## Описание


Метод FindByKey выполняет поиск элемента в коллекции по его ключу.


## Комментарии


Если элемент не найден, то метод возвращает значение Null.
