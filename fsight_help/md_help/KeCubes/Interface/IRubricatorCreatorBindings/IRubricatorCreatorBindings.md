# IRubricatorCreatorBindings


Сборка: Cubes;


## Описание


Интерфейс IRubricatorCreatorBindings
 предназначен для работы с коллекцией привязок атрибутов БД временных рядов
 к полям таблицы данных.


## Иерархия наследования


           IRubricatorCreatorBindings


## Комментарии


Для получения коллекции параметров привязок используйте свойство [IRubricatorCreator.Bindings](../IRubricatorCreator/IRubricatorCreator.Bindings.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Item](IRubricatorCreatorBindings.Item.htm)
		 Свойство Item возвращает
		 указанный элемент коллекции.


		 ![](../../Property_Image.gif)
		 [Count](IRubricatorCreatorBindings.Count.htm)
		 Свойство Count возвращает
		 количество элементов в коллекции.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IRubricatorCreatorBindings.Add.htm)
		 Метод Add добавляет
		 новый элемент в коллекцию привязок.


		 ![](../../Sub_Image.gif)
		 [Clear](IRubricatorCreatorBindings.Clear.htm)
		 Метод Clear очищает

		 ![](../../Sub_Image.gif)
		 [FindByKey](IRubricatorCreatorBindings.FindByKey.htm)
		 Метод FindByKey выполняет
		 поиск элемента в коллекции по его ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](IRubricatorCreatorBindings.Remove.htm)
		 Метод Remove удаляет
		 элемент с указанным индексом из коллекции привязок.
