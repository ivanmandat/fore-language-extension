# IRubricatorCreatorBindings.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество элементов в коллекции.


## Комментарии


Для добавления элементов используйте метод [IRubricatorCreatorBindings.Add](IRubricatorCreatorBindings.Add.htm), для удаления - [IRubricatorCreatorBindings.Remove](IRubricatorCreatorBindings.Remove.htm).
