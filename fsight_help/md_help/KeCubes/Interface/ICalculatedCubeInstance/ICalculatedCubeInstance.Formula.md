# ICalculatedCubeInstance.Formula


## Синтаксис


Formula(Coord: [ICalculatedCubeInstanceCoord](../ICalculatedCubeInstanceCoord/ICalculatedCubeInstanceCoord.htm)): [ICalculatedCubeFormulas](../ICalculatedCubeFormulas/ICalculatedCubeFormulas.htm);


## Параметры


Coord. Координата вычисляемого куба, для которой необходимо получить формулы.


## Описание


Свойство Formula возвращает коллекцию формул, по которым осуществляется вычисление значений, для указанной координаты.
