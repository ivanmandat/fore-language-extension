# ICalculatedCubeInstance


## Описание


Интерфейс ICalculatedCubeInstance
 содержит свойства и методы для работы с отстроенным вычислимым кубом.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Destination](ICalculatedCubeInstance.Destination.htm)
		 Свойство Destination
		 возвращает объект, используемый для работы с данными вычисляемого
		 куба.


		 ![](../../Property_Image.gif)
		 [Formula](ICalculatedCubeInstance.Formula.htm)
		 Свойство Formula возвращает
		 коллекцию формул, по которым осуществляется вычисление значений,
		 для указанной координаты.


		 ![](../../Property_Image.gif)
		 [Formulas](ICalculatedCubeInstance.Formulas.htm)
		 Свойство Formulas возвращает
		 матрицу, содержащую формулы вычисляемого куба в строковом виде.


		 ![](../../Property_Image.gif)
		 [Sources](ICalculatedCubeInstance.Sources.htm)
		 Свойство Sources возвращает
		 коллекцию источников вычисляемого куба.


		 ![](../../Property_Image.gif)
		 [View](ICalculatedCubeInstance.View.htm)
		 Свойство View возвращает
		 параметры отображения элементов измерений в кубе и в формулах
		 куба.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Apply](ICalculatedCubeInstance.Apply.htm)
		 Метод Apply осуществляет
		 применение параметров управления отметкой.


		 ![](../../Sub_Image.gif)
		 [CreateApply](ICalculatedCubeInstance.CreateApply.htm)
		 Метод CreateApply создает
		 объект, используемый для управления отметкой измерений куба.


		 ![](../../Sub_Image.gif)
		 [CreateCoord](ICalculatedCubeInstance.CreateCoord.htm)
		 Метод CreateCoord создает
		 объект, представляющий собой координату в вычисляемом кубе.


		 ![](../../Sub_Image.gif)
		 [CreateExecutor](ICalculatedCubeInstance.CreateExecutor.htm)
		 Метод CreateExecutor
		 создает объект, осуществляющий расчет результирующей матрицы вычисляемого
		 куба.


		 ![](../../Sub_Image.gif)
		 [CreateSourceCoord](ICalculatedCubeInstance.CreateSourceCoord.htm)
		 Метод CreateSourceCoord
		 создает объект, представляющий собой координату в кубе-источнике
		 данных для вычисляемого куба.


		 ![](../../Sub_Image.gif)
		 [FindCoordEntries](ICalculatedCubeInstance.FindCoordEntries.htm)
		 Метод FindCoordEntries
		 осуществляет поиск вхождений заданной координаты в формулы вычисляемого
		 куба.


		 ![](../../Sub_Image.gif)
		 [FindCubeEntries](ICalculatedCubeInstance.FindCubeEntries.htm)
		 Метод FindCubeEntries
		 осуществляет поиск вхождений заданного куба в формулы вычисляемого
		 куба.


		 ![](../../Sub_Image.gif)
		 [FindDimensionEntries](ICalculatedCubeInstance.FindDimensionEntries.htm)
		 Метод FindDimensionEntries
		 осуществляет поиск вхождений заданного справочника в формулы вычисляемого
		 куба.


		 ![](../../Sub_Image.gif)
		 [FindElementEntries](ICalculatedCubeInstance.FindElementEntries.htm)
		 Метод FindElementEntries
		 осуществляет поиск вхождений заданного элемента справочника в
		 формулы вычисляемого куба.


		 ![](../../Sub_Image.gif)
		 [LoadFormulasFromFile](ICalculatedCubeInstance.LoadFormulasFromFile.htm)
		 Метод LoadFormulasFromFile
		 осуществляет загрузку формул вычисляемого куба из файла.


		 ![](../../Sub_Image.gif)
		 [SaveFormulas](ICalculatedCubeInstance.SaveFormulas.htm)
		 Метод SaveFormulas
		 осуществляет сохранение изменений, внесенных в формулы вычисляемого
		 куба.


		 ![](../../Sub_Image.gif)
		 [SaveFormulasToFile](ICalculatedCubeInstance.SaveFormulasToFile.htm)
		 Метод SaveFormulasToFile
		 осуществляет сохранение формул вычисляемого куба в файл.
