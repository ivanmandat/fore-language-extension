# IVariableStubList


## Описание


Интерфейс IVariableStubList
 содержит свойства и методы для работы с коллекцией абстрактных источников
 данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IVariableStubList.Count.htm)
		 Свойство Count возвращает
		 количество источников данных в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IVariableStubList.Item.htm)
		 Свойство Item возвращает
		 источник данных из коллекции по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](IVariableStubList.FindById.htm)
		 Метод FindById осуществляет
		 поиск источника данных в коллекции по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IVariableStubList.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск источника данных в коллекции по ключу.
