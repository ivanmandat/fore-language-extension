# ICubeInstanceSource


## Описание


Интерфейс ICubeInstanceSource
 содержит свойства и методы для работы с отстроенным кубом-источником данных
 для данного куба.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Cube](ICubeInstanceSource.Cube.htm)
		 Свойство Cube возвращает
		 параметры отстроенного куба-источника данных.


		 ![](../../Property_Image.gif)
		 [Source](ICubeInstanceSource.Source.htm)
		 Свойство Source возвращает
		 объект, содержащий свойства и методы для работы с отстроенным
		 кубом-источником, в соответствии с его вариантом отображения.


		 ![](../../Property_Image.gif)
		 [SourceModel](ICubeInstanceSource.SourceModel.htm)
		 Свойство SourceModel
		 возвращает параметры куба-источника данных.
