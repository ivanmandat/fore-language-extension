# ICubeExecuteResult


## Описание


Интерфейс ICubeExecuteResult
 содержит свойства для работы с результатами расчёта куба.


## Комментарии


Результаты расчёта куба возвращает свойство [ICubeClass.ExecuteResult](../ICubeClass/ICubeClass.ExecuteResult.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CoordConvertor](ICubeExecuteResult.CoordConvertor.htm)
		 Свойство CoordConvertor
		 возвращает объект, предназначенный для получения координат в кубе-источнике.


		 ![](../../Property_Image.gif)
		 [DataRights](ICubeExecuteResult.DataRights.htm)
		 Свойство DataRights
		 возвращает права доступа на полученные данные.


		 ![](../../Property_Image.gif)
		 [DataTypes](ICubeExecuteResult.DataTypes.htm)
		 Свойство DataTypes
		 возвращает информацию о том, каким образом сформированы значения
		 в ячейках.


		 ![](../../Property_Image.gif)
		 [Destination](ICubeExecuteResult.Destination.htm)
		 Свойство Destination
		 возвращает отстроенный куб-родитель.
