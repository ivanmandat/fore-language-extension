# ICubeExecuteResult.CoordConvertor


## Синтаксис


CoordConvertor: [ICubeCoordConvertor](../ICubeCoordConvertor/ICubeCoordConvertor.htm);


## Описание


Свойство CoordConvertor возвращает
 объект, предназначенный для получения координат в кубе-источнике.


## Комментарии


Свойство актуально, если расчёт осуществлялся для виртуального куба
 или куба-представления.
