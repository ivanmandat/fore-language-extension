# ICubeExecuteResult.DataTypes


## Синтаксис


DataTypes: [ICubeExecuteDataTypes](../ICubeExecuteDataTypes/ICubeExecuteDataTypes.htm);


## Описание


Свойство DataTypes возвращает
 информацию о том, каким образом сформированы значения в ячейках.


## Комментарии


Значение свойства будет доступно, если при расчёте свойству [ICubeExecuteSetup.UseDataTypes](../ICubeExecuteSetup/ICubeExecuteSetup.UseDataTypes.htm) было
 установлено значение True.
