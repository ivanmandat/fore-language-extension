# ICubeValidationAssertion


Сборка: Cubes;


## Описание


Интерфейс ICubeValidationAssertion
 предназначен для получения информации о нарушении метаданных куба.


## Иерархия наследования


ICubeValidationAssertion


## Комментарии


Информацию о нарушении метаданных куба возвращает свойство [ICubeValidationAssertions.Item](../ICubeValidationAssertions/ICubeValidationAssertions.Item.htm).
