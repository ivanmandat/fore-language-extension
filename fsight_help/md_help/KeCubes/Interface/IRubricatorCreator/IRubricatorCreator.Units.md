# IRubricatorCreator.Units


## Синтаксис


Units: [IRdsDictionary](KeRds.chm::/Interface/IRdsDictionary/IRdsDictionary.htm);


## Описание


Свойство Units определяет справочник единиц измерения для создаваемой БД временных рядов.


## Комментарии


Справочник мер определяет свойство [IRubricatorCreator.Measures](IRubricatorCreator.Measures.htm).
