# IRubricatorCreator.RdsDatabase


## Синтаксис


RdsDatabase: [IRdsDatabase](KeRds.chm::/Interface/IRdsDatabase/IRdsDatabase.htm);


## Описание


Свойство RdsDatabase определяет репозиторий НСИ для создаваемой БД временных рядов.


## Комментарии


Справочники мер и единиц измерения, для создаваемой БД, определяют свойства [IRubricatorCreator.Measures](IRubricatorCreator.Measures.htm) и [IRubricatorCreator.Units](IRubricatorCreator.Units.htm).
