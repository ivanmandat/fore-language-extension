# IRubricatorCreator.SourceTable


## Синтаксис


SourceTable: [ITable](KeDb.chm::/Interface/ITable/ITable.htm);


## Описание


Свойство SourceTable определяет таблицу с данными, на которой будет основана БД временных рядов.


## Комментарии


Если БД временных рядов должна быть основана на стандартном кубе, то используйте метод [IRubricatorCreator.AssignCube](IRubricatorCreator.AssignCube.htm).
