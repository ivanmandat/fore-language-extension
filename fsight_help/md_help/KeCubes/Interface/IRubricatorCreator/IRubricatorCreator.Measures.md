# IRubricatorCreator.Measures


## Синтаксис


Measures: [IRdsDictionary](KeRds.chm::/Interface/IRdsDictionary/IRdsDictionary.htm);


## Описание


Свойство Measures определяет справочник мер для создаваемой БД временных рядов.


## Комментарии


Справочник единиц измерения определяет свойство [IRubricatorCreator.Units](IRubricatorCreator.Units.htm).
