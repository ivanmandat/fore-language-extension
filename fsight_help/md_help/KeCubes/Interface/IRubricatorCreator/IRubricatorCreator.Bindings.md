# IRubricatorCreator.Bindings


## Синтаксис


Bindings: [IRubricatorCreatorBindings](../IRubricatorCreatorBindings/IRubricatorCreatorBindings.htm);


## Описание


Свойство Bindings возвращает коллекцию привязок атрибутов БД временных рядов к полям таблицы данных.


## Комментарии


Таблицу данных определяет свойство [IRubricatorCreator.SourceTable](IRubricatorCreator.SourceTable.htm).
