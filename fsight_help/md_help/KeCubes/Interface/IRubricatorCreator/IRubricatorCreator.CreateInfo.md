# IRubricatorCreator.CreateInfo


## Синтаксис


CreateInfo: [IMetabaseObjectCreateInfo](KeSom.chm::/Interface/IMetabaseObjectCreateInfo/IMetabaseObjectCreateInfo.htm);


## Описание


Свойство CreateInfo определяет параметры создаваемой БД временных рядов, как объекта репозитория.


## Комментарии


Репозиторий, в котором должна быть создана БД временных рядов, определяет свойство [IRubricatorCreator.Metabase](IRubricatorCreator.Metabase.htm).
