# IRubricatorCreator.Metabase


## Синтаксис


Metabase: [IMetabase](KeSom.chm::/Interface/IMetabase/IMetabase.htm);


## Описание


Свойство Metabase определяет репозиторий, в котором должна быть создана БД временных рядов.


## Комментарии


Параметры создаваемой БД временных рядов как объекта репозитория определяет свойство [IRubricatorCreator.CreateInfo](IRubricatorCreator.CreateInfo.htm).
