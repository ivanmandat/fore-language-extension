# IRubricatorCreator.TemplateCalendar


## Синтаксис


TemplateCalendar: [ICalendarDimension](KeDims.chm::/interface/ICalendarDimension/ICalendarDimension.htm);


## Описание


Свойство TemplateCalendar определяет календарный справочник, по параметрам которого будет построено календарное измерение создаваемой БД временных рядов.


## Комментарии


Репозиторий НСИ, для создаваемой БД временных рядов, определяет свойство [IRubricatorCreator.RdsDatabase](IRubricatorCreator.RdsDatabase.htm).
