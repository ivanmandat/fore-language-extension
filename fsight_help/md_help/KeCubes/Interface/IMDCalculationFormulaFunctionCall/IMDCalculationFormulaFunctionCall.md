# IMDCalculationFormulaFunctionCall


## Описание


Интерфейс IMDCalculationFormulaFunctionCall
 содержит свойства и методы операнда формулы, представляющего собой специальную
 функцию.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ArgumentCount](IMDCalculationFormulaFunctionCall.ArgumentCount.htm)
		 Свойство ArgumentCount
		 определяет количество аргументов, используемых в функциях MMin,
		 MMax, MAvg.


		 ![](../../Property_Image.gif)
		 [FunctionCall](IMDCalculationFormulaFunctionCall.FunctionCall.htm)
		 Свойство FunctionCall
		 определяет вид специальной функции, используемой в формуле.
