# ICubeSelectionControlItem.IsEmpty


## Синтаксис


IsEmpty: Boolean;


## Описание


Свойство IsEmpty возвращает
 признак того, установлено ли управляющее измерение куба.


## Комментарии


Свойство возвращает значение True,
 если управляющее измерение определено, и False,
 если не задано.
