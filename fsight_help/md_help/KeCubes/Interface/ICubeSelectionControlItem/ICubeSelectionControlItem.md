# ICubeSelectionControlItem


## Описание


Интерфейс ICubeSelectionControlItem
 содержит свойства и методы для работы с управляющим измерением куба.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ControlledObject](ICubeSelectionControlItem.ControlledObject.htm)
		 Свойство ControlledObject
		 возвращает управляемый объект.


		 ![](../../Property_Image.gif)
		 [ControlledOwner](ICubeSelectionControlItem.ControlledOwner.htm)
		 Свойство ControlledOwner
		 возвращает объект-владелец управляемого измерения.


		 ![](../../Property_Image.gif)
		 [ControlledOwnerKey](ICubeSelectionControlItem.ControlledOwnerKey.htm)
		 Свойство ControlledOwnerKey
		 возвращает ключ объекта-владельца управляемого измерения.


		 ![](../../Property_Image.gif)
		 [ControlledParam](ICubeSelectionControlItem.ControlledParam.htm)
		 Свойство ControlledParam
		 возвращает управляемый параметр.


		 ![](../../Property_Image.gif)
		 [ControllingDim](ICubeSelectionControlItem.ControllingDim.htm)
		 Свойство ControllingDim
		 определяет управляющее измерение куба.


		 ![](../../Property_Image.gif)
		 [Expression](ICubeSelectionControlItem.Expression.htm)
		 Свойство Expression
		 возвращает выражение управления.


		 ![](../../Property_Image.gif)
		 [Index](ICubeSelectionControlItem.Index.htm)
		 Свойство Index возвращает
		 индекс управляющего измерения.


		 ![](../../Property_Image.gif)
		 [IsDimension](ICubeSelectionControlItem.IsDimension.htm)
		 Свойство IsDimension
		 возвращает признак того, является ли управляемый объект измерением
		 куба.


		 ![](../../Property_Image.gif)
		 [IsEmpty](ICubeSelectionControlItem.IsEmpty.htm)
		 Свойство IsEmpty возвращает
		 признак того, установлено ли управляющее измерение куба.


		 ![](../../Property_Image.gif)
		 [Key](ICubeSelectionControlItem.Key.htm)
		 Свойство Key возвращает
		 ключ управляющего измерения.


		 ![](../../Property_Image.gif)
		 [SelectionStrategy](ICubeSelectionControlItem.SelectionStrategy.htm)
		 Свойство SelectionStrategy
		 определяет отметку, которая должна быть выставлена в зависимом
		 справочнике после его перестроения.
