# ICubeValidationAssertions


Сборка: Cubes;


## Описание


Интерфейс ICubeValidationAssertions
 предназначен для работы со списком возможных нарушений метаданных куба.


## Иерархия наследования


ICubeValidationAssertions


## Комментарии


Нарушение метаданных может возникнуть на стадии редактирования куба,
 если заданы какие-либо некорректные параметры. Осуществить проверку куба
 и получить список возможных нарушений можно с помощью метода [ICubeModel.Validate](../ICubeModel/ICubeModel.Validate.htm).
