# ICubeValidationAssertions.IsEmpty


## Синтаксис


		IsEmpty: Boolean;


## Описание


Свойство IsEmpty возвращает
 признак отсутствия нарушений метаданных куба.


## Комментарии


Свойство возвращает значение True,
 если структура метаданных куба корректна и в текущей коллекции отсутствуют
 какие-либо сообщения.
