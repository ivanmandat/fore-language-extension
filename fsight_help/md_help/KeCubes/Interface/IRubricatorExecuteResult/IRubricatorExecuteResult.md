# IRubricatorExecuteResult


## Описание


Интерфейс IRubricatorExecuteResult
 содержит свойства для работы с данными, полученными в результате расчета
 базы данных временных рядов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DataAttributes](IRubricatorExecuteResult.DataAttributes.htm)
		 Свойство DataAttributes
		 возвращает информацию об извлеченных значениях атрибутов для указанного
		 справочника базы данных временных рядов.


		 ![](../../Property_Image.gif)
		 [DataMatrix](IRubricatorExecuteResult.DataMatrix.htm)
		 Свойство DataMatrix
		 возвращает матрицу значений указанного справочника базы данных
		 временных рядов.


		 ![](../../Property_Image.gif)
		 [Dependencies](IRubricatorExecuteResult.Dependencies.htm)
		 Свойство Dependencies
		 возвращает матрицу зависимостей базы данных временных рядов.


		 ![](../../Property_Image.gif)
		 [Factors](IRubricatorExecuteResult.Factors.htm)
		 Свойство Factors возвращает
		 матрицу показателей базы данных временных рядов.


		 ![](../../Property_Image.gif)
		 [Values](IRubricatorExecuteResult.Values.htm)
		 Свойство Values возвращает
		 матрицу значений базы данных временных рядов.
