# IAlternateHierarchyMappings.Item


## Синтаксис


Item(Index: Integer): [IAlternateHierarchyMapping](../IAlternateHierarchyMapping/IAlternateHierarchyMapping.htm);


## Параметры


Index. Индекс соответствия
 в коллекции.


## Описание


Свойство Item возвращает
 указанное соответствие из коллекции.


## Комментарии


Индексация соответствий в коллекции сплошная и начинается с нуля. Для
 получения количества соответствий в коллекции используйте свойство [IAlternateHierarchyMappings.Count](IAlternateHierarchyMappings.Count.htm).
