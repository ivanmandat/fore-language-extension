# IAlternateHierarchyMappings.Clear


## Синтаксис

## Описание


Метод Clear удаляет все
 соответствия из коллекции.


## Комментарии


После вызова метода значение свойства [IAlternateHierarchyMappings.Count](IAlternateHierarchyMappings.Count.htm)
 обнуляется.
