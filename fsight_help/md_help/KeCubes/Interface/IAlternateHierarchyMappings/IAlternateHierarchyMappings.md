# IAlternateHierarchyMappings


Сборка: Cubes;


## Описание


Интерфейс IAlternateHierarchyMappings
 используется для работы с коллекцией соответствий между альтернативными
 иерархиями в справочниках копируемой базы данных временных рядов и альтернативными
 иерархиями в справочниках базы-копии.


## Иерархия наследования


           IAlternateHierarchyMappings


## Комментарии


Для получения коллекции соответствий используйте свойство [ICubeMetaUpdateAttributeRemapping.AlternateHierarchyMappings](../ICubeMetaUpdateAttributeRemapping/ICubeMetaUpdateAttributeRemapping.AlternateHierarchyMappings.htm).


## Свойства


		 Имя свойства

		 ![](../../Property_Image.gif)
		 [Count](IAlternateHierarchyMappings.Count.htm)

## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IAlternateHierarchyMappings.Add.htm)


		 Метод Add добавляет
		 новое соответствие в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](IAlternateHierarchyMappings.Clear.htm)


		 Метод Clear удаляет

		 ![](../../Sub_Image.gif)
		 [FindByKey](IAlternateHierarchyMappings.FindByKey.htm)


		 Метод FindByKey ищет
		 соответствие в коллекции по ключу копируемой альтернативной иерархии.


		 ![](../../Sub_Image.gif)
		 [Remove](IAlternateHierarchyMappings.Remove.htm)
