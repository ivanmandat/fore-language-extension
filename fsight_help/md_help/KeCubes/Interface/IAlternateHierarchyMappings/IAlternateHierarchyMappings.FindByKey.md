# IAlternateHierarchyMappings.FindByKey


## Синтаксис


FindByKey(SourceKey: Integer): [IAlternateHierarchyMapping](../IAlternateHierarchyMapping/IAlternateHierarchyMapping.htm);


## Параметры


SourceKey. Ключ копируемой
 альтернативной иерархии.


## Описание


Метод FindByKey ищет соответствие
 в коллекции по ключу копируемой альтернативной иерархии.


## Комментарии


Если соответствие не найдено, то метод возвращает значение Null.
