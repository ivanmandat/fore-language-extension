# IAlternateHierarchyMappings.FindById


## Синтаксис


FindById(SourceId: Stirng): [IAlternateHierarchyMapping](../IAlternateHierarchyMapping/IAlternateHierarchyMapping.htm);


## Параметры


SourceId. Идентификатор копируемой
 альтернативной иерархии.


## Описание


Метод FindById ищет соответствие
 в коллекции по идентификатору копируемой альтернативной иерархии.


## Комментарии


Если соответствие не найдено, то метод возвращает значение Null.
