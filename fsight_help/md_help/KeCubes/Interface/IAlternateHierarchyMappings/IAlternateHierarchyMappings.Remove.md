# IAlternateHierarchyMappings.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс соответствия
 в коллекции.


## Описание


Метод Remove удаляет указанное
 соответствие из коллекции.


## Комментарии


Если удаление выполнено успешно, то метод вернет значение True,
 в противном случае - False.
