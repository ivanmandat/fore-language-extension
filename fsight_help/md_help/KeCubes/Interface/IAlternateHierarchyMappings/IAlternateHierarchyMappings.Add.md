# IAlternateHierarchyMappings.Add


## Синтаксис


Add(SourceKey: Integer; SourceId: Integer): [IAlternateHierarchyMapping](../IAlternateHierarchyMapping/IAlternateHierarchyMapping.htm);


## Параметры


SourceKey. Ключ альтернативной
 иерархии;


SourceId. Индекс альтернативной
 иерархии.


## Описание


Метод Add добавляет новое
 соответствие в коллекцию.


## Комментарии


Для удаления соответствия из коллекции используйте метод [IAlternateHierarchyMappings.Remove](IAlternateHierarchyMappings.Remove.htm).
