# ICubeLoaderFixInfo.FindByKey


## Синтаксис


FindByKey(Key: Integer): [ICubeLoaderDimensionFix](../ICubeLoaderDimensionFix/ICubeLoaderDimensionFix.htm);


## Параметры


Key - ключ справочника репозитория, соответствующего измерению куба-источника.


## Описание


Метод FindByKey осуществляет поиск и возвращает параметры фиксации измерения, ключ которого передается посредством параметра Key.
