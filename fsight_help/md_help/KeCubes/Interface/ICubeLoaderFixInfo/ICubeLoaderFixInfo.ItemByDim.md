# ICubeLoaderFixInfo.ItemByDim


## Синтаксис


ItemByDim(Dimension: [IDimensionModel](kedims.chm::/interface/idimensionmodel/idimensionmodel.htm)): [ICubeLoaderDimensionFix](../ICubeLoaderDimensionFix/ICubeLoaderDimensionFix.htm);


## Параметры


Dimension - измерение куба-источника, либо справочник репозитория, являющийся измерением в кубе-источнике.


## Описание


Свойство ItemByDim возвращает параметры фиксации измерения, передаваемого посредством параметра Dimension.
