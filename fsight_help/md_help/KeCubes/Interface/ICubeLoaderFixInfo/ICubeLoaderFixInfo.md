# ICubeLoaderFixInfo


## Описание


Интерфейс ICubeLoaderFixInfo
 содержит свойства и методы для фиксации измерений куба-источника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeLoaderFixInfo.Count.htm)
		 Свойство Count возвращает
		 количество измерений, которые необходимо зафиксировать.


		 ![](../../Property_Image.gif)
		 [Item](ICubeLoaderFixInfo.Item.htm)
		 Свойство Item возвращает
		 параметры фиксации измерения.


		 ![](../../Property_Image.gif)
		 [ItemByDim](ICubeLoaderFixInfo.ItemByDim.htm)
		 Свойство ItemByDim
		 возвращает параметры фиксации измерения.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindByKey](ICubeLoaderFixInfo.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск измерения по ключу.
