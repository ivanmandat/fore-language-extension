# ICubeMetaUpdateAttributeRemapping


Сборка: Cubes;


## Описание


Интерфейс ICubeMetaUpdateAttributeRemapping
 содержит свойства и методы для задания соответствия между атрибутом копируемой
 базы данных временных рядов и атрибутом базы-копии.


## Иерархия наследования


           [INamedEntityModel](foresys.chm::/interface/inamedentitymodel/inamedentitymodel.htm)


           ICubeMetaUpdateAttributeRemapping


## Комментарии


Для работы с коллекцией соответствий между атрибутом копируемой базы
 данных временных рядов и атрибутом базы-копии используйте интерфейс [ICubeMetaUpdateAttributeRemappings](../ICubeMetaUpdateAttributeRemappings/ICubeMetaUpdateAttributeRemappings.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AlternateHierarchyMappings](ICubeMetaUpdateAttributeRemapping.AlternateHierarchyMappings.htm)
		 Свойство AlternateHierarchyMappings
		 возвращает параметры соответствия между альтернативными иерархиями
		 в справочниках копируемой базы данных временных рядов и альтернативными
		 иерархиями в справочниках базы-копии.


		 ![](../../Property_Image.gif)
		 [Source](ICubeMetaUpdateAttributeRemapping.Source.htm)
		 Свойство Source определяет
		 идентификатор атрибута показателей копируемой базы, являющегося
		 источником данных для атрибута показателей базы-копии.


		 ![](../../Property_Image.gif)
		 [SourceKey](ICubeMetaUpdateAttributeRemapping.SourceKey.htm)
		 Свойство SourceKey
		 определяет ключ атрибута показателей копируемой базы, являющегося
		 источником данных для атрибута показателей базы-копии.


		 ![](../../Property_Image.gif)
		 [Type](ICubeMetaUpdateAttributeRemapping.Type.htm)
		 Свойство Type определяет
		 способ задания значения атрибута показателей базы-копии.


		 ![](../../Property_Image.gif)
		 [Value](ICubeMetaUpdateAttributeRemapping.Value.htm)
		 Свойство Value определяет
		 значение атрибута показателей базы-копии.
