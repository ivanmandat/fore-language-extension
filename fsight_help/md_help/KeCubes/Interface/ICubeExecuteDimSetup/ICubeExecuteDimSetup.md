# ICubeExecuteDimSetup


## Описание


Интерфейс ICubeExecuteDimSetup
 содержит свойства для работы с параметрами измерения при расчете куба.


## Комментарии


Параметры измерения могут быть получены с использованием одного из свойств/методов
 интерфейса [ICubeExecuteSetup](../ICubeExecuteSetup/ICubeExecuteSetup.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AggregationRoot](ICubeExecuteDimSetup.AggregationRoot.htm)


		 Свойство AggregationRoot
		 возвращает индекс элемента в результирующей матрице куба, по которому
		 содержатся агрегированные данные по элементам текущего измерения.


		 ![](../../Property_Image.gif)
		 [CubeSetup](ICubeExecuteDimSetup.CubeSetup.htm)


		 Свойство CubeSetup
		 возвращает коллекцию параметров измерений при расчете куба.


		 ![](../../Property_Image.gif)
		 [Extension](ICubeExecuteDimSetup.Extension.htm)


		 Свойство Extension
		 возвращает дополнительные настройки, которые будут использоваться
		 при расчете куба.


		 ![](../../Property_Image.gif)
		 [FilterFunction](ICubeExecuteDimSetup.FilterFunction.htm)


		 Свойство FilterFunction
		 определяет параметры функции, которая будет применяться для фильтрации
		 данных.


		 ![](../../Property_Image.gif)
		 [GroupIndex](ICubeExecuteDimSetup.GroupIndex.htm)


		 Свойство GroupIndex
		 определяет индекс группы, в которую входит измерение.


		 ![](../../Property_Image.gif)
		 [Options](ICubeExecuteDimSetup.Options.htm)


		 Свойство Options возвращает
		 опции измерения.


		 ![](../../Property_Image.gif)
		 [Position](ICubeExecuteDimSetup.Position.htm)


		 Свойство Position определяет
		 позицию измерения в группе.


		 ![](../../Property_Image.gif)
		 [Predefined](ICubeExecuteDimSetup.Predefined.htm)


		 Свойство Predefined
		 возвращает предопределенные параметры измерения.


		 ![](../../Property_Image.gif)
		 [Selection](ICubeExecuteDimSetup.Selection.htm)


		 Свойство Selection
		 определяет отметку элементов измерения.


		 ![](../../Property_Image.gif)
		 [Tag](ICubeExecuteDimSetup.Tag.htm)


		 Свойство Tag возвращает
		 тип измерения.
