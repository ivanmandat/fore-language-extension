# ICubeExecuteDimSetup.AggregationRoot


## Синтаксис


AggregationRoot: Integer;


## Описание


Свойство AggregationRoot возвращает индекс элемента в результирующей матрице куба, по которому содержатся агрегированные данные по элементам текущего измерения.


## Комментарии


Значение данного свойства будет доступно после расчета результирующей матрицы куба.


Если свойство возвращает значение «-1», значит в матрице по данному измерению нет агрегированных данных.


Если свойство возвращает заведомо большое значение элемента, значит элемент является фиктивным. Значения по этому элементу доступны только в момент существования рассчитанной матрицы.


Примечание. Данное свойство возвращает индекс элемента, по которому содержатся агрегированные данные, рассчитанные по всем уровням измерения. Если в измерении настроена агрегация по отдельным уровням, то данное свойство также вернет индекс элемента, но по этому элементу в матрице будут содержаться пустые значения.
