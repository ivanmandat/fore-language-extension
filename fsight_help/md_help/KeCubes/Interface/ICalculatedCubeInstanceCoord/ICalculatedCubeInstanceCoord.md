# ICalculatedCubeInstanceCoord


## Описание


Интерфейс ICalculatedCubeInstanceCoord
 содержит свойства и методы для работы с координатой в вычисляемом кубе
 и кубах-источниках.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AsString](ICalculatedCubeInstanceCoord.AsString.htm)


		 Свойство AsString возвращает
		 символьное обозначение координаты, отображаемое в формулах вычисляемого
		 куба.


		 ![](../../Property_Image.gif)
		 [AsTerm](ICalculatedCubeInstanceCoord.AsTerm.htm)


		 Свойство AsTerm возвращает
		 символьное представление терма, соответствующего данной координате
		 и используемое для составления формулы.


		 ![](../../Property_Image.gif)
		 [IsReference](ICalculatedCubeInstanceCoord.IsReference.htm)


		 Свойство IsReference
		 определяет признак передачи координаты по ссылке.


		 ![](../../Property_Image.gif)
		 [MatrixCoord](ICalculatedCubeInstanceCoord.MatrixCoord.htm)


		 Свойство MatrixCoord
		 возвращает координату вычисляемого куба, представленную в виде
		 координаты матрицы.


		 ![](../../Property_Image.gif)
		 [Source](ICalculatedCubeInstanceCoord.Source.htm)


		 Свойство Source возвращает
		 источник данных, которому соответствует данная координата.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [DimVal](ICalculatedCubeInstanceCoord.DimVal.htm)


		 Метод DimVal возвращает
		 значение по текущей координате атрибута измерения назначения.


		 ![](../../Sub_Image.gif)
		 [Shift](ICalculatedCubeInstanceCoord.Shift.htm)


		 Метод Shift осуществляет
		 смещение координаты по календарному измерению с заданным лагом.


		 ![](../../Sub_Image.gif)
		 [ShiftVal](ICalculatedCubeInstanceCoord.ShiftVal.htm)


		 Метод ShiftVal возвращает
		 значение по координате, смещенной относительно данной координаты
		 по календарному измерению с заданным лагом.


		 ![](../../Sub_Image.gif)
		 [ToKey](ICalculatedCubeInstanceCoord.ToKey.htm)


		 Метод ToKey осуществляет
		 преобразование и возвращает уникальный ключ, соответствующий данной
		 координате.


		 ![](../../Sub_Image.gif)
		 [Val](ICalculatedCubeInstanceCoord.Val.htm)


		 Метод Val возвращает
		 значение по текущей координате.
