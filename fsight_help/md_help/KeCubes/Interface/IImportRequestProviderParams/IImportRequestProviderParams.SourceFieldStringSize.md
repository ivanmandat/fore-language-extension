# IImportRequestProviderParams.SourceFieldStringSize


## Синтаксис


SourceFieldStringSize: Integer;


## Описание


Свойство SourceFieldStringSize
 определяет, какой длины будут считаться строковые поля в источнике данных.


## Комментарии


Свойство используется для всех строковых полей, сопоставляемых со справочниками
 НСИ. Значение по умолчанию 255.
