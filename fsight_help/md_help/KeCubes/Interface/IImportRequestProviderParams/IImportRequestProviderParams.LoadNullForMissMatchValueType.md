# IImportRequestProviderParams.LoadNullForMissMatchValueType


## Синтаксис


LoadNullForMissMatchValueType: Boolean;


## Описание


Свойство LoadNullForMissMatchValueType
 определяет, записывать ли значение Null
 при импорте значений, не соответствующих типу поля приёмника.


## Комментарии


Допустимые значения:


	- True. Записывать значение
	 Null;


	- False. Значение по умолчанию.
	 Не записывать значение Null.
