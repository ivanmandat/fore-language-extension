# IImportRequestProviderParams.ScenarioKey


## Синтаксис


ScenarioKey: Integer;


## Описание


Свойство ScenarioKey определяет
 ключ сценария, по которому будут загружаться данные.


## Комментарии


По умолчанию ScenarioKey принимает
 значение «-1», то есть данные загружаются по сценарию «Факт».
