# IImportRequestProviderParams.OverWriteWholeFactorValues


## Синтаксис


OverWriteWholeFactorValues: Boolean;


## Описание


Свойство OverWriteWholeFactorValues
 определяет, перезаписывать ли при импорте весь ряд целиком.


## Комментарии


Допустимые значения:


	- True. Перезаписывать
	 весь ряд целиком;


	- False. Значение по умолчанию.
	 Не перезаписывать весь ряд целиком.
