# IStandardCubeDimensions.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key - ключ измерения.


## Описание


Метод RemoveByKey осуществляет удаление измерения, ключ которого передается посредством параметра Key.
