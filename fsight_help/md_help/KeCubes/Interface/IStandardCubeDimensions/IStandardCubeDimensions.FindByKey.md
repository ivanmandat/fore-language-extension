# IStandardCubeDimensions.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IStandardCubeDimension](../IStandardCubeDimension/IStandardCubeDimension.htm);


## Параметры


Key - ключ измерения, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет поиск измерения куба, ключ которого передается посредством параметра Id.
