# IStandardCubeDimensions.FindById


## Синтаксис


FindById(Id: String): [IStandardCubeDimension](../IStandardCubeDimension/IStandardCubeDimension.htm);


## Параметры


Id - идентификатор измерения, по которому осуществляется поиск.


## Описание


Метод FindById осуществляет поиск измерения куба, идентификатор которого передается посредством параметра Id.
