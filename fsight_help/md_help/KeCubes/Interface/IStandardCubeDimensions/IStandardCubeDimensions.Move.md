# IStandardCubeDimensions.Move


## Синтаксис


Move(IndexFrom: Integer; IndexTo: Integer);


## Параметры


IndexFrom - индекс позиции измерения, которое необходимо переместить.


IndexTo - индекс позиции, в которую необходимо переместить измерение.


## Описание


Метод Move осуществляет перемещение измерения в списке.
