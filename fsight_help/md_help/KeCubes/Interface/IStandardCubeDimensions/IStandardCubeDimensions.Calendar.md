# IStandardCubeDimensions.Calendar


## Синтаксис


Calendar: [IStandardCubeDimension](../IStandardCubeDimension/IStandardCubeDimension.htm);


## Описание


Свойство Calendar возвращает параметры календарного измерения куба.


## Комментарии


Свойство возвращает параметры измерения, которому соответствует календарный справочник. Если в структуре куба создано несколько измерений на базе календарных справочников, то свойство вернет параметры того измерения, которое в списке расположено выше других.
