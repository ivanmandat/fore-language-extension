# IStandardCubeDimensions


## Описание


Интерфейс IStandardCubeDimensions
 содержит свойства и методы для работы с коллекцией измерений стандартного
 куба.


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Calendar](IStandardCubeDimensions.Calendar.htm)


		 Свойство Calendar возвращает
		 параметры календарного измерения куба.


		 ![](../../Property_Image.gif)
		 [Count](IStandardCubeDimensions.Count.htm)


		 Свойство Count возвращает
		 количество измерений в кубе.


		 ![](../../Property_Image.gif)
		 [Item](IStandardCubeDimensions.Item.htm)


		 Свойство Item возвращает
		 измерение куба.


## Методы


		  Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IStandardCubeDimensions.Add.htm)


		 Метод Add осуществляет
		 добавление измерения в куб.


		 ![](../../Sub_Image.gif)
		 [FindById](IStandardCubeDimensions.FindById.htm)


		 Метод FindById осуществляет
		 поиск измерения куба по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IStandardCubeDimensions.FindByKey.htm)


		 Метод FindByKey осуществляет
		 поиск измерения куба по ключу.


		 ![](../../Sub_Image.gif)
		 [Move](IStandardCubeDimensions.Move.htm)


		 Метод Move осуществляет
		 перемещение измерения в списке.


		 ![](../../Sub_Image.gif)
		 [Remove](IStandardCubeDimensions.Remove.htm)


		 Метод Remove осуществляет
		 удаление измерения по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IStandardCubeDimensions.RemoveByKey.htm)


		 Метод RemoveByKey осуществляет
		 удаление измерения по ключу.
