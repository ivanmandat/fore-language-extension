# IStandardCubeDimensions.Item


## Синтаксис


Item(Index: Integer): [IStandardCubeDimension](../IStandardCubeDimension/IStandardCubeDimension.htm);


## Параметры


Index - индекс измерения куба.


## Описание


Свойство Item возвращает измерение куба, индекс которого передается посредством параметра Index.
