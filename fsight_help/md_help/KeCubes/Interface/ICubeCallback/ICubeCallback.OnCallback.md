# ICubeCallback.OnCallback


## Синтаксис


OnCallback(Argument: [ICubeCallbackArgument](../ICubeCallbackArgument/ICubeCallbackArgument.htm));


## Параметры


Argument. Аргумент, содержащий
 информацию об ошибке.


## Описание


Метод OnCallback реализует событие,
 происходящее при возникновении исключительных ситуаций во время работы
 с данными куба.
