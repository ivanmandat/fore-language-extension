# ICubeCallback.Argument


## Синтаксис


Argument: [ICubeCallbackArgument](../ICubeCallbackArgument/ICubeCallbackArgument.htm);


## Описание


Свойство Argument возвращает
 аргумент, содержащий информацию об ошибке, произошедшей при сохранении
 данных.


## Комментарии


Данное свойство используется в обработчике события [OnCallback](ICubeCallback.OnCallback.htm)
 и должно быть переопределено в пользовательском классе.
