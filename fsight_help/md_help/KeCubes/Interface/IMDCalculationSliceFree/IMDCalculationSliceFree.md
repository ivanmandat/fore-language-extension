# IMDCalculationSliceFree


## Описание


Интерфейс IMDCalculationSliceFree
 содержит свойства свободного измерения источника/приёмника данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dimension](IMDCalculationSliceFree.Dimension.htm)
		 Свойство Dimension
		 определяет измерение, данные которого будут использоваться при
		 составлении формул многомерного расчета на сервере БД.


		 ![](../../Property_Image.gif)
		 [Index](IMDCalculationSliceFree.Index.htm)
		 Свойство Index определяет
		 индекс измерения [Dimension](IMDCalculationSliceFree.Dimension.htm),
		 по которому будет осуществляться связь с полями [Fields](../IMDCalculationSlice/IMDCalculationSlice.Fields.htm).


		 ![](../../Property_Image.gif)
		 [Slice](IMDCalculationSliceFree.Slice.htm)
		 Свойство Slice возвращает
		 измерение источника/приёмника данных.


		 ![](../../Property_Image.gif)
		 [Summ](IMDCalculationSliceFree.Summ.htm)
		 Свойство Summ возвращает
		 параметры суммирования по элементам измерения.
