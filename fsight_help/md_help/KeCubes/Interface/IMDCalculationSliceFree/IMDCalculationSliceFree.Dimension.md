# IMDCalculationSliceFree.Dimension


## Синтаксис


Dimension: [IDimensionModel](KeDims.chm::/interface/IDimensionModel/IDimensionModel.htm);


## Описание


Свойство Dimension определяет
 измерение, данные которого будут использоваться при составлении формул
 многомерного расчета на сервере БД.


## Комментарии


Данное измерение будет связано с полями [Fields](../IMDCalculationSlice/IMDCalculationSlice.Fields.htm)
 посредством значений атрибутов измерения, по которому создан индекс [Index](IMDCalculationSliceFree.Index.htm).


## Пример


Использование свойства приведено в примере для [IMDCalculationDataset.Data](../IMDCalculationDataset/IMDCalculationDataset.Data.htm).
