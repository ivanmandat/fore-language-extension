# IMDCalculationSliceFree.Index


## Синтаксис


Index: [IDimIndex](KeDims.chm::/interface/IDimIndex/IDimIndex.htm);


## Описание


Свойство Index определяет индекс
 измерения [Dimension](IMDCalculationSliceFree.Dimension.htm),
 по которому будет осуществляться связь с полями [Fields](../IMDCalculationSlice/IMDCalculationSlice.Fields.htm).


## Комментарии


Данный индекс включает в себя атрибуты измерения, по значениям которых,
 будет осуществляться связь с полями [Fields](../IMDCalculationSlice/IMDCalculationSlice.Fields.htm).
