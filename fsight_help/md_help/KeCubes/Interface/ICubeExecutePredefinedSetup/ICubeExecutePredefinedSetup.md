# ICubeExecutePredefinedSetup


## Описание


Интерфейс ICubeExecutePredefinedSetup
 содержит свойства для работы с предопределенными параметрами измерения.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeExecutePredefinedSetup.Count.htm)
		 Свойство Count возвращает
		 количество параметров измерения.


		 ![](../../Property_Image.gif)
		 [Item](ICubeExecutePredefinedSetup.Item.htm)
		 Свойство Item возвращает
		 параметр измерения по его индексу.


		 ![](../../Property_Image.gif)
		 [MultiSelect](ICubeExecutePredefinedSetup.MultiSelect.htm)
		 Свойство MultiSelect
		 возвращает признак одновременного использования нескольких параметров
		 измерения.


		 ![](../../Property_Image.gif)
		 [Selected](ICubeExecutePredefinedSetup.Selected.htm)
		 Свойство Selected определяет
		 параметр, установленный в данный момент.
