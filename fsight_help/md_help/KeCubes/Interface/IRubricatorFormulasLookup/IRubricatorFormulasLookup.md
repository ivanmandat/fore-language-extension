# IRubricatorFormulasLookup


## Описание


Интерфейс IRubricatorFormulasLookup
 содержит свойства и методы, предназначенные для поиска формул в базе данных
 временных рядов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [WhereKind](IRubricatorFormulasLookup.WhereKind.htm)
		 Свойство WhereKind
		 определяет вид искомой формулы.


		 ![](../../Property_Image.gif)
		 [WhereParentKey](IRubricatorFormulasLookup.WhereParentKey.htm)
		 Свойство WhereParentKey
		 определяет ключ родительской формулы, по которой будет производиться
		 поиск.


		 ![](../../Property_Image.gif)
		 [WhereXML](IRubricatorFormulasLookup.WhereXML.htm)
		 Свойство WhereXML определяет
		 XML код искомой формулы.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [LookupData](IRubricatorFormulasLookup.LookupData.htm)
		 Метод LookupData осуществляет
		 поиск формулы по заданным условиям.
