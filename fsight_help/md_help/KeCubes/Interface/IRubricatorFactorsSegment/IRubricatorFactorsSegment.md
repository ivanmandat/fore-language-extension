# IRubricatorFactorsSegment


## Описание


Интерфейс IRubricatorFactorsSegment
 содержит методы для работы с диапазоном данных базы данных временных рядов.


## Методы

		 возвращает отметку элементов измерения, определяющую диапазон
		 данных.


		 ![](../../Sub_Image.gif)
		 [SetSelection](IRubricatorFactorsSegment.SetSelection.htm)
		 Метод SetSelection
		 устанавливает отметку элементов измерения, определяющую диапазон
		 данных.
