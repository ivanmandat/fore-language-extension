# IAlternateHierarchyMapping.SourceId


## Синтаксис


SourceId: String;


## Описание


Свойство SourceId определяет
 идентификатор альтернативной иерархии в исходном справочнике.


## Комментарии


Для задания ключа альтернативной иерархии в исходном справочнике используйте
 свойство [IAlternateHierarchyMapping.SourceKey](IAlternateHierarchyMapping.SourceKey.htm).
