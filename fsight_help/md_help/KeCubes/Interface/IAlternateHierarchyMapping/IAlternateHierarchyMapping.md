# IAlternateHierarchyMapping


Сборка: Cubes;


## Описание


Интерфейс IAlternateHierarchyMapping
 предназначен для работы с соответствием между альтернативной иерархией
 в справочнике копируемой базы данных временных рядов и альтернативной
 иерархией в справочнике базы-копии.


## Иерархия наследования


           IAlternateHierarchyMapping


## Комментарии


Если альтернативной иерархии из исходного справочника нет в справочнике-приёмнике,
 то необходимо задать «нулевое соотвествие». При задании «нулевого
 соответствия» выбранная альтернативная иерархия в исходном справочнике
 будет сбрасываться.


Для задания «нулевого соответствия» задайте следующие параметры альтернативной
 иерархии в справочнике-приёмнике:


	- ключ иерархии: «-1»;


	- идентификатор иерархии: пустая строка.
