# IAlternateHierarchyMapping.TargetKey


## Синтаксис


TargetKey: Integer;


## Описание


Свойство TargetKey определяет
 ключ альтернативной иерархии в справочнике-приёмнике.


## Комментарии


Для задания идентификатора альтернативной иерархии в справочнике-приёмнике
 используйте свойство [IAlternateHierarchyMapping.TargetId](IAlternateHierarchyMapping.TargetId.htm).
