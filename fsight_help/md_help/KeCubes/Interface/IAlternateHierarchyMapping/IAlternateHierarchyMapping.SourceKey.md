# IAlternateHierarchyMapping.SourceKey


## Синтаксис


SourceKey: Integer;


## Описание


Свойство SourceKey определяет
 ключ альтернативной иерархии в исходном справочнике.


## Комментарии


Для задания идентификатора альтернативной иерархии в исходном справочнике
 используйте свойство [IAlternateHierarchyMapping.SourceId](IAlternateHierarchyMapping.SourceId.htm).
