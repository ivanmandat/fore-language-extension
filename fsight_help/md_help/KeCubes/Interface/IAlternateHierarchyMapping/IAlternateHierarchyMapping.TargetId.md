# IAlternateHierarchyMapping.TargetId


## Синтаксис


TargetId: String;


## Описание


Свойство TargetId определяет
 идентификатор альтернативной иерархии в справочнике-приёмнике.


## Комментарии


Для задания ключа альтернативной иерархии в справочнике-приёмнике используйте
 свойство [IAlternateHierarchyMapping.TargetKey](IAlternateHierarchyMapping.TargetKey.htm).
