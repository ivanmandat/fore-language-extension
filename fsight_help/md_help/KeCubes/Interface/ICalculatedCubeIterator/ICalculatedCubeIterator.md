# ICalculatedCubeIterator


## Описание


Интерфейс ICalculatedCubeIterator
 содержит свойства и методы итератора по формулам, удовлетворяющим условиям
 поиска.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Coord](ICalculatedCubeIterator.Coord.htm)
		 Свойство Coord возвращает
		 координату вычисляемого куба, по которой существуют формулы, содержащие
		 искомый элемент.


		 ![](../../Property_Image.gif)
		 [Current](ICalculatedCubeIterator.Current.htm)
		 Свойство Current возвращает
		 координату вычисляемого куба в виде координаты матрицы, по которой
		 существуют формулы, содержащие искомый элемент.


		 ![](../../Property_Image.gif)
		 [Eof](ICalculatedCubeIterator.Eof.htm)
		 Свойство Eof возвращает
		 True, если курсор находится
		 на последней записи в итераторе.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Next](ICalculatedCubeIterator.Next.htm)
		 Метод Next осуществляет
		 переход к следующей записи в итераторе.
