# ICubeMetaUpdateEx.CopyData


## Синтаксис


CopyData: Boolean;


## Описание


Свойство CopyData определяет,
 нужно ли копировать данные базы данных временных рядов.


## Комментарии


Допустимые значения:


	- True. Данные базы должны
	 быть скопированы;


	- False. База данных временных
	 рядов копируется без данных.
