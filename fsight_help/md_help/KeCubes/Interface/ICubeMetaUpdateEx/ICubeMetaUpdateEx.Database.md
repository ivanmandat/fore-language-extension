# ICubeMetaUpdateEx.Database


## Синтаксис


Database: [IDatabase](KeDb.chm::/Interface/IDatabase/IDatabase.htm);


## Описание


Свойство Database определяет
 отдельную базу данных, в которой будут храниться данные скопированной
 базы данных временных рядов.


## Комментарии


По умолчанию свойство не определено, при этом данные будут сохраняться
 в базе данных, которая задана для [репозитория
 НСИ](ICubeMetaUpdateEx.RdsDatabase.htm).
