# ICubeMetaUpdateEx.RemapBySelectionIndex


## Синтаксис


RemapBySelectionIndex: Boolean;


## Описание


Свойство RemapBySelectionIndex
 определяет, применять ли для копирования данных и сопоставления объектов
 уникальный индекс, используемый для формирования отметки.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Копирование и сопоставление объектов выполняется по индексу, используемому
	 для формирования отметки.

	Данный способ копирования возможен, если все справочники, на которых
	 построена база данных временных рядов имеют уникальный индекс, используемый
	 для формирования отметки. Если хотя бы для одного справочника данный
	 индекс не задан, то копирование и сопоставление объектов выполняется
	 по ключам;


	- False. Копирование и
	 сопоставление объектов выполняется по ключам.
