# ICubeMetaUpdateAdditionalObjectDataSettings


Сборка: Cubes;


## Описание


Интерфейс ICubeMetaUpdateAdditionalObjectDataSettings
 содержит свойства для определения параметров копирования данных объекта.


## Иерархия наследования


ICubeMetaUpdateAdditionalObjectDataSettings


## Комментарии


Свойства данного интерфейса определяют параметры копирования данных
 объектов, которые будут копироваться вместе с базой данных временных рядов.
