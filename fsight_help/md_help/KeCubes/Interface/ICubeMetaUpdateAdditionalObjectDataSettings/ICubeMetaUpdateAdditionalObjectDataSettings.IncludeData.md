# ICubeMetaUpdateAdditionalObjectDataSettings.IncludeData


## Синтаксис


IncludeData: Boolean;


## Описание


Свойство IncludeData определяет признак копирования данных вместе с объектом.


## Комментарии


Если свойству установлено значение True, то объект будет копироваться вместе с данными.
