# ICubeMetaUpdateAdditionalObjectDataSettings.ReferenceConstraintsHandling


## Синтаксис


ReferenceConstraintsHandling: [UpdateDataConstraintsHandlingType](kesom.chm::/enums/updatedataconstraintshandlingtype.htm);


## Описание


Свойство ReferenceConstraintsHandling определяет способ обработки ограничения целостности элементов обновляемого объекта.


## Комментарии


Данное свойство актуально при обновлении вместе с базой данных временных рядов справочников НСИ, на элементы которых могут иметься ссылки из других справочников.


## Пример


Пример использования данного свойства приведен в примере для свойства [PrerequisiteObjects](../ICubeMetaUpdateEx/ICubeMetaUpdateEx.PrerequisiteObjects.htm).
