# ICubeView


Сборка: Cubes;


## Описание


Интерфейс ICubeView содержит
 свойства объекта репозитория - Представление-куб.


## Иерархия наследования


ICubeView


## Комментарии


При создании нового представления-куба для его корректной работы необходимо
 в свойстве [Cube](ICubeView.Cube.htm) указать исходный куб,
 а в свойстве [FixInfo](ICubeView.FixInfo.htm) зафиксировать
 необходимые измерения.
