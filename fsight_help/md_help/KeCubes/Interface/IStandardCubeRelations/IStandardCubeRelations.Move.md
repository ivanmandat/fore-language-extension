# IStandardCubeRelations.Move


## Синтаксис


Move(IndexFrom: Integer; IndexTo: Integer);


## Параметры


IndexFrom - индекс позиции отношения, которое необходимо переместить.


IndexTo - индекс позиции, в которую необходимо переместить отношение стандартного куба.


## Описание


Метод Move осуществляет перемещение отношений в списке.
