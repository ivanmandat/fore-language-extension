# IStandardCubeRelations.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IStandardCubeRelation](../IStandardCubeRelation/IStandardCubeRelation.htm);


## Параметры


Key - ключ отношения стандартного куба.


## Описание


Метод FindByKey осуществляет поиск отношения в стандартном кубе, ключ которого передается посредством параметра Key.
