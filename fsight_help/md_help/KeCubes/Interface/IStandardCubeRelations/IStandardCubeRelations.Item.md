# IStandardCubeRelations.Item


## Синтаксис


Item(Index: Integer): [IStandardCubeRelation](../IStandardCubeRelation/IStandardCubeRelation.htm);


## Параметры


Index - индекс отношения стандартного куба.


## Описание


Свойство Item возвращает отношение стандартного куба, индекс которого передается посредством параметра Index.
