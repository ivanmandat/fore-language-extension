# IStandardCubeRelations


## Описание


Интерфейс IStandardCubeRelations
 содержит свойства и методы для работы с коллекцией отношений в стандартном
 кубе.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IStandardCubeRelations.Count.htm)
		 Свойство Count возвращает
		 количество отношений в стандартном кубе.


		 ![](../../Property_Image.gif)
		 [Item](IStandardCubeRelations.Item.htm)
		 Свойство Item возвращает
		 отношение стандартного куба.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IStandardCubeRelations.Add.htm)
		 Метод Add осуществляет
		 добавление нового отношения в стандартный куб.


		 ![](../../Sub_Image.gif)
		 [Clear](IStandardCubeRelations.Clear.htm)
		 Метод Clear осуществляет
		 очистку списка отношений стандартного куба.


		 ![](../../Sub_Image.gif)
		 [FindById](IStandardCubeRelations.FindById.htm)
		 Метод FindById осуществляет
		 поиск отношения по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IStandardCubeRelations.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск отношения по ключу.


		 ![](../../Sub_Image.gif)
		 [Move](IStandardCubeRelations.Move.htm)
		 Метод Move осуществляет
		 перемещение отношений в списке.


		 ![](../../Sub_Image.gif)
		 [Remove](IStandardCubeRelations.Remove.htm)
		 Метод Remove осуществляет
		 удаление отношения стандартного куба по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IStandardCubeRelations.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление отношения стандартного куба по ключу.
