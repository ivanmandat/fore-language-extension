# IStandardCubeRelations.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key - ключ отношения.


## Описание


Метод RemoveByKey осуществляет удаление отношения стандартного куба, ключ которого передается посредством параметра Key.
