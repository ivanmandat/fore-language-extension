# IStandardCubeRelations.FindById


## Синтаксис


FindById(Id: String): [IStandardCubeRelation](../IStandardCubeRelation/IStandardCubeRelation.htm);


## Параметры


Id - идентификатор отношения стандартного куба.


## Описание


Метод FindById осуществляет поиск отношения в стандартном кубе, идентификатор которого передается посредством параметра Id.
