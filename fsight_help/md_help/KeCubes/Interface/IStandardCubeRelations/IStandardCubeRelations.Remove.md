# IStandardCubeRelations.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс отношения.


## Описание


Метод Remove осуществляет удаление отношения стандартного куба, индекс которого передается посредством параметра Index.
