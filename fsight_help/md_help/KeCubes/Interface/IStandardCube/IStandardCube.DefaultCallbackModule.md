# IStandardCube.DefaultCallbackModule


## Синтаксис


		DefaultCallbackModule: [IMetabaseObjectDescriptor](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm);


## Описание


Свойство DefaultCallbackModule
 возвращает модуль, в котором находится класс для обработки процесса сохранения
 данных в куб.


## Комментарии


Свойство возвращает описание того модуля, который был задан в методе
 [SetDefaultCallback](IStandardCube.SetDefaultCallback.htm)
 при установке обработчика сохранения данных.
