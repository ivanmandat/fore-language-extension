# IStandardCube.DefaultCallbackClass


## Синтаксис


		DefaultCallbackClass: String;


## Описание


Свойство DefaultCallbackClass
 возвращает класс, который используется для обработки процесса сохранения
 данных в куб.


## Комментарии


Свойство возвращает наименование класса, который был задан в методе
 [SetDefaultCallback](IStandardCube.SetDefaultCallback.htm)
 при установке обработчика сохранения данных.
