# IStandardCube.SegmentContainer


## Синтаксис


SegmentContainer: [ICubeSegmentContainer](../ICubeSegmentContainer/ICubeSegmentContainer.htm);


## Описание


Свойство SegmentContainer определяет
 контейнер сегментов для куба.


## Комментарии


Сегменты, созданные в контейнере, будут определять доступ к данным для
 разных пользователей.
