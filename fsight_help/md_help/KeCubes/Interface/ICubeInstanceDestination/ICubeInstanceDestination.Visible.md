# ICubeInstanceDestination.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible возвращает признак скрытого варианта отображения.


## Комментарии


Свойство возвращает значение True, если вариант отображения является видимым и False, если вариант отображения скрыт. Скрытый вариант отображения будет недоступен при просмотре данных куба в экспресс-отчете.
