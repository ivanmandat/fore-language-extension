# ICubeInstanceDestination.SetDimensions


## Синтаксис


SetDimensions(Selection: [IDimSelectionSet](kedims.chm::/interface/idimselectionset/idimselectionset.htm));


## Параметры


Selection - отметка, которую необходимо установить для элементов измерений куба.


## Описание


Метод SetDimensions устанавливает указанный набор измерений в текущий экземпляр открытого куба.
