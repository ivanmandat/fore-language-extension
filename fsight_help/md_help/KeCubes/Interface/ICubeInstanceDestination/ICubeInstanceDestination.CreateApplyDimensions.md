# ICubeInstanceDestination.CreateApplyDimensions


## Синтаксис


CreateApplyDimensions([ApplyMode: [SelectionControlApplyMode](../../Enums/SelectionControlApplyMode.htm) = 1]): [ICubeSelectionControlApply](../ICubeSelectionControlApply/ICubeSelectionControlApply.htm);


## Комментарии


ApplyMode - режим, в котором будет выполняться применение параметров управления отметкой.


## Описание


Метод CreateApplyDimensions создает объект, используемый для управления отметкой измерений куба.


## Комментарии


Данный метод актуально использовать, если в кубе имеются управляющие и управляемые измерения. Применение параметров управления отметкой осуществляется с помощью метода [ApplyDimensions](ICubeInstanceDestination.ApplyDimensions.htm).


## Пример


Пример использования данного метода приведен в примере для метода [ApplyDimensions](ICubeInstanceDestination.ApplyDimensions.htm).
