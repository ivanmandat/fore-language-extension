# ICubeInstanceDestination.Cached


## Синтаксис


Cached: Boolean;


## Описание


Свойство Cached возвращает признак
 наличия в базе репозитория кеша с данными экземпляра куба.


## Комментарии


При получении значения свойства осуществляется обращение к базе репозитория
 и проверка там наличия кешированных данных. Если кешированные данные имеются,
 то они будут извлечены и свойство вернет значение True.
 Если кешированные данные отсутствуют, то свойство вернет значение False.
