# ICubeModelDimensionsEx.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key - ключ измерения, которое необходимо удалить.


## Описание


Метод RemoveByKey осуществляет удаление измерения куба по ключу. Возвращает True, если удаление прошло успешно.
