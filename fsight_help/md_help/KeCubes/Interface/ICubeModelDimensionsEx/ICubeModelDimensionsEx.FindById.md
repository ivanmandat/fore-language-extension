# ICubeModelDimensionsEx.FindById


## Синтаксис


FindById(Id: String): [ICubeModelDimensionEx](../ICubeModelDimensionEx/ICubeModelDimensionEx.htm);


## Параметры


Id - идентификатор измерения.


## Описание


Метод FindById осуществляет поиск и возвращает объект, содержащий расширенные свойства измерения. Идентификатор, по которому осуществляется поиск, передается посредством параметра Id.
