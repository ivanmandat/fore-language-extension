# ICubeModelDimensionsEx


## Описание


Интерфейс ICubeModelDimensionsEx
 содержит свойства и методы для доступа к расширенным свойствам измерений,
 в рамках данного куба.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeModelDimensionsEx.Count.htm)
		 Свойство Count возвращает
		 количество измерений куба.


		 ![](../../Property_Image.gif)
		 [Item](ICubeModelDimensionsEx.Item.htm)
		 Свойство Item возвращает
		 расширенные свойства измерения.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddDimensionEx](IcubemodelDimensionsex.AddDimensionEx.htm)
		 Метод AddDimensionEx
		 осуществляет добавление измерения.


		 ![](../../Sub_Image.gif)
		 [Clear](ICubeModelDimensionsEx.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех измерений из коллекции куба.


		 ![](../../Sub_Image.gif)
		 [FindById](ICubeModelDimensionsEx.FindById.htm)
		 Метод FindById осуществляет
		 поиск по идентификатору и возвращает расширенные свойства измерения
		 куба.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ICubeModelDimensionsEx.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск по ключу и возвращает расширенные свойства измерения куба.


		 ![](../../Sub_Image.gif)
		 [Move](ICubeModelDimensionsEx.Move.htm)
		 Метод Move осуществляет
		 перемещение измерений в коллекции куба.


		 ![](../../Sub_Image.gif)
		 [Remove](ICubeModelDimensionsEx.Remove.htm)
		 Метод Remove осуществляет
		 удаление измерения куба по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](ICubeModelDimensionsEx.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление измерения куба по ключу.
