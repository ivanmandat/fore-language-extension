# ICubeModelDimensionsEx.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс измерения, которое необходимо удалить.


## Описание


Метод Remove осуществляет удаление измерения куба по индексу. Возвращает True, если удаление прошло успешно.
