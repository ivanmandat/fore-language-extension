# ICubeModelDimensionsEx.FindByKey


## Синтаксис


FindByKey(DimensionKey: Integer): [ICubeModelDimensionEx](../ICubeModelDimensionEx/ICubeModelDimensionEx.htm);


## Параметры


DimensionKey - ключ измерения.


## Описание


Метод FindByKey осуществляет поиск и возвращает объект, содержащий расширенные свойства измерения. Ключ, по которому осуществляется поиск, передается посредством параметра DimensionKey.
