# IAutoCubeDimensions


## Описание


Интерфейс IAutoCubeDimensions
 содержит свойства и методы коллекции измерений автоматического куба.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Calendar](IAutoCubeDimensions.Calendar.htm)
		 Свойство Calendar возвращает
		 календарное измерение автоматического куба.


		 ![](../../Property_Image.gif)
		 [Count](IAutoCubeDimensions.Count.htm)
		 Свойство Count возвращает
		 количество измерений в автоматическом кубе.


		 ![](../../Property_Image.gif)
		 [Item](IAutoCubeDimensions.Item.htm)
		 Свойство Item возвращает
		 измерение автоматического куба.


		 ![](../../Property_Image.gif)
		 [ItemByDimension](IAutoCubeDimensions.ItemByDimension.htm)
		 Свойство ItemByDimension
		 возвращает измерения автоматического куба, в основе которого лежит
		 указанный справочник репозитория.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IAutoCubeDimensions.Add.htm)
		 Метод Add осуществляет
		 добавление справочника в коллекцию измерений автоматического куба.


		 ![](../../Sub_Image.gif)
		 [Clear](IAutoCubeDimensions.Clear.htm)
		 Метод Clear осуществляет
		 очистку коллекции измерений автоматического куба.


		 ![](../../Sub_Image.gif)
		 [FindById](IAutoCubeDimensions.FindById.htm)
		 Метод FindById осуществляет
		 поиск измерения автоматического куба по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IAutoCubeDimensions.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск измерения автоматического куба по ключу.


		 ![](../../Sub_Image.gif)
		 [Move](IAutoCubeDimensions.Move.htm)
		 Метод Move осуществляет
		 перемещение измерений в списке автоматического куба.


		 ![](../../Sub_Image.gif)
		 [Remove](IAutoCubeDimensions.Remove.htm)
		 Метод Remove осуществляет
		 удаление измерения автоматического куба по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveById](IAutoCubeDimensions.RemoveById.htm)
		 Метод RemoveById осуществляет
		 удаление измерения автоматического куба по идентификатору.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IAutoCubeDimensions.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление измерения автоматического куба по ключу.
