# IAutoCubeDimensions.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IAutoCubeDimension](../IAutoCubeDimension/IAutoCubeDimension.htm);


## Параметры


Key - ключ измерения, по которому
 осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает объект, содержащий измерения автоматического куба,
 в случае удачного поиска.
