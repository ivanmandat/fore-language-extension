# IAutoCubeDimensions.Item


## Синтаксис


Item(Index: Integer): [IAutoCubeDimension](../IAutoCubeDimension/IAutoCubeDimension.htm);


## Параметры


Index - индекс измерения.


## Описание


Свойство Item возвращает измерение
 автоматического куба, индекс которого передается посредством параметра
 Index.
