# IAutoCubeDimensions.FindById


## Синтаксис


FindById(Id: String): [IAutoCubeDimension](../IAutoCubeDimension/IAutoCubeDimension.htm);


## Параметры


Id - идентификатор измерения,
 по которому осуществляется поиск.


## Описание


Метод FindById осуществляет
 поиск и возвращает объект, содержащий измерения автоматического куба,
 в случае удачного поиска.
