# IAutoCubeDimensions.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key - ключ измерения, которое
 необходимо удалить.


## Описание


Метод RemoveByKey осуществляет
 удаление измерения автоматического куба, ключ которого передается посредством
 параметра Key.


Примечание.
 При удалении справочников, имеющиеся в автоматическом кубе данные будут
 утеряны.
