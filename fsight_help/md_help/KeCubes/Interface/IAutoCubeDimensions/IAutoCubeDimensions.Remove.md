# IAutoCubeDimensions.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс измерения, которое
 необходимо удалить.


## Описание


Метод Remove осуществляет удаление
 измерения автоматического куба, индекс которого передается посредством
 параметра Index.


Примечание.
 При удалении справочников, имеющиеся в автоматическом кубе данные будут
 утеряны.
