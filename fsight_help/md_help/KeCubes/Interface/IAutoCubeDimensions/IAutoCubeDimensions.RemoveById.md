# IAutoCubeDimensions.RemoveById


## Синтаксис


RemoveById(Id: String): Boolean;


## Параметры


Id - идентификато
 измерения, которое необходимо удалить.


## Описание


Метод RemoveById осуществляет
 удаление измерения автоматического куба, идентификатор которого передается
 посредством параметра Id.


Примечание.
 При удалении справочников, имеющиеся в автоматическом кубе данные будут
 утеряны.
