# IMDCalculationFormulaDatePeriod


## Описание


Интерфейс IMDCalculationFormulaDatePeriod
 содержит свойства и методы операнда формулы, представляющего собой период,
 по которому будет рассчитываться значение функции [Shift](UiMd.chm::/Cube/Work_Cube/UiMd_CubeWork_MultivariateAccount_Functions.htm)
 в формуле.


## Свойства


		 Имя свойства

		 ![](../../Property_Image.gif)
		 [DatePeriod](IMDCalculationFormulaDatePeriod.DatePeriod.htm)
		 Свойство DatePeriod
		 определяет период в календаре, по которому будет осуществляться
		 смещение с помощью функции [Shift](UiMd.chm::/Cube/Work_Cube/UiMd_CubeWork_MultivariateAccount_Functions.htm).
