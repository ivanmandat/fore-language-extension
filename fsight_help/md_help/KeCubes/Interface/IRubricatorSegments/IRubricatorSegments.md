# IRubricatorSegments


## Описание


Интерфейс IRubricatorSegments
 содержит свойства и методы, предназначенные для работы с коллекцией сегментов
 данных базы данных временных рядов.


## Свойства, унаследованные от [IRubricatorSegmentsSet](../IRubricatorSegmentsSet/IRubricatorSegmentsSet.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](../IRubricatorSegmentsSet/IRubricatorSegmentsSet.Count.htm)


		 Свойство Count возвращает
		 количество сегментов данных в базе данных временных рядов.


		 ![](../../Property_Image.gif)
		 [Item](../IRubricatorSegmentsSet/IRubricatorSegmentsSet.Item.htm)


		 Свойство Item возвращает
		 указанный сегмент данных из коллекции.


		 ![](../../Property_Image.gif)
		 [Rubricator](../IRubricatorSegmentsSet/IRubricatorSegmentsSet.Rubricator.htm)


		 Свойство Rubricator
		 возвращает данные базы данных временных рядов, содержащей коллекцию
		 сегментов данных.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CreateFactorsSegment](IRubricatorSegments.CreateFactorsSegment.htm)
		 Метод CreateFactorsSegment
		 добавляет диапазон данных в коллекцию.


		 ![](../../Sub_Image.gif)
		 [CreateTimeSegment](IRubricatorSegments.CreateTimeSegment.htm)
		 Метод CreateTimeSegment
		 добавляет временной диапазон в коллекцию.


## Методы, унаследованные от [IRubricatorSegmentsSet](../IRubricatorSegmentsSet/IRubricatorSegmentsSet.htm)


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Clear](../IRubricatorSegmentsSet/IRubricatorSegmentsSet.Clear.htm)


		 Метод Clear осуществляет
		 очистку коллекции сегментов данных.


		 ![](../../Sub_Image.gif)
		 [FindByKey](../IRubricatorSegmentsSet/IRubricatorSegmentsSet.FindByKey.htm)


		 Метод FindByKey осуществляет
		 поиск сегмента в коллекции по ключу.


		 ![](../../Sub_Image.gif)
		 [GetAllowed](../IRubricatorSegmentsSet/IRubricatorSegmentsSet.GetAllowed.htm)


		 Метод GetAllowed позволяет
		 получить отметку элементов измерения, доступных для совершения
		 указанных действий текущим пользователем.


		 ![](../../Sub_Image.gif)
		 [GetEffectiveRights](../IRubricatorSegmentsSet/IRubricatorSegmentsSet.GetEffectiveRights.htm)


		 Метод GetEffectiveRights
		 возвращает эффективные права текущего пользователя на коллекцию
		 сегментов данных.


		 ![](../../Sub_Image.gif)
		 [GetFactSegments](../IRubricatorSegmentsSet/IRubricatorSegmentsSet.GetFactSegments.htm)


		 Метод GetFactSegments
		 возвращает количество диапазонов данных, в которые входит указанный
		 показатель.


		 ![](../../Sub_Image.gif)
		 [GetSubjectAllowed](../IRubricatorSegmentsSet/IRubricatorSegmentsSet.GetSubjectAllowed.htm)


		 Метод GetSubjectAllowed
		 позволяет получить отметку элементов измерения, доступных для
		 совершения указанных действий указанным пользователем.


		 ![](../../Sub_Image.gif)
		 [GetSubjectEffectiveRights](../IRubricatorSegmentsSet/IRubricatorSegmentsSet.GetSubjectEffectiveRights.htm)


		 Метод GetSubjectEffectiveRights
		 возвращает эффективные права указанного пользователя на коллекцию
		 сегментов данных.


		 ![](../../Sub_Image.gif)
		 [GetTimeSegments](../IRubricatorSegmentsSet/IRubricatorSegmentsSet.GetTimeSegments.htm)


		 Метод GetTimeSegments
		 возвращает количество временных диапазонов данных, которые попадают
		 в указанный диапазон.


		 ![](../../Sub_Image.gif)
		 [Remove](../IRubricatorSegmentsSet/IRubricatorSegmentsSet.Remove.htm)


		 Метод Remove удаляет
		 указанный сегмент из коллекции.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](../IRubricatorSegmentsSet/IRubricatorSegmentsSet.RemoveByKey.htm)


		 Метод RemoveByKey удаляет
		 сегмент из коллекции по ключу.
