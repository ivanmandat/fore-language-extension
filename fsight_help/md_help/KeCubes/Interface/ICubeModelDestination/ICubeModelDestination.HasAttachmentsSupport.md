# ICubeModelDestination.HasAttachmentsSupport


## Синтаксис


		HasAttachmentsSupport: Boolean;


## Описание


Свойство HasAttachmentsSupport
 возвращает признак поддержки кубом работы с вложениями.


## Комментарии


Работу со вложениями поддерживают только [стандартные
 кубы](../IStandardCube/IStandardCube.htm). Свойство возвращает значение True,
 если для куба [определён
 справочник](../IStandardCubeDestination/IStandardCubeDestination.AttachmentsStorage.htm), в котором хранятся вложения, и настроена [привязка
 фактов](../IStandardCubeFactBindings/IStandardCubeFactBindings.BindingWithRole.htm).
