# ICubeModelDestination.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет, является ли данный вариант отображения куба скрытым.


## Комментарии


По умолчанию свойству установлено значение True, при этом вариант отображения является видимым. Если свойству установить значение False, то вариант отображения будет скрыт. Скрытый вариант отображения будет недоступен при просмотре данных куба в экспресс-отчете.
