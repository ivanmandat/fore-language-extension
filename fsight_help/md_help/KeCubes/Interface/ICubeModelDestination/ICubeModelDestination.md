# ICubeModelDestination


## Описание


Интерфейс ICubeModelDestination
 содержит свойства варианта отображения куба.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AlternateSource](ICubeModelDestination.AlternateSource.htm)
		 Свойство AlternateSource
		 возвращает параметры альтернативного куба, используемого для хранения
		 данных.


		 ![](../../Property_Image.gif)
		 [Cube](ICubeModelDestination.Cube.htm)
		 Свойство Cube возвращает
		 объект-родитель.


		 ![](../../Property_Image.gif)
		 [Dimensions](ICubeModelDestination.Dimensions.htm)
		 Свойство Dimensions
		 возвращает свойства коллекции измерений куба.


		 ![](../../Property_Image.gif)
		 [DimensionsEx](ICubeModelDestination.DimensionsEx.htm)
		 Свойство DimensionsEx
		 возвращает расширенные свойства измерений куба.


		 ![](../../Property_Image.gif)
		 [HasAttachmentsSupport](ICubeModelDestination.HasAttachmentsSupport.htm)
		 Свойство HasAttachmentsSupport
		 возвращает признак поддержки кубом работы с вложениями.


		 ![](../../Property_Image.gif)
		 [HasChangeHistory](ICubeModelDestination.HasChangeHistory.htm)
		 Свойство HasChangeHistory
		 возвращает признак поддержки ведения истории изменений для данных
		 куба.


		 ![](../../Property_Image.gif)
		 [HasCommentsSupport](ICubeModelDestination.HasCommentsSupport.htm)
		 Свойство HasCommentsSupport
		 возвращает признак поддержки кубом возможности сохранения комментариев
		 к данным.


		 ![](../../Property_Image.gif)
		 [IsDefault](ICubeModelDestination.IsDefault.htm)
		 Свойство IsDefault
		 возвращает True, если
		 данный вариант отображения куба, является вариантом отображения
		 по умолчанию.


		 ![](../../Property_Image.gif)
		 [Visible](ICubeModelDestination.Visible.htm)
		 Свойство Visible определяет,
		 является ли данный вариант отображения куба скрытым.
