# IRubricatorFormulaData


## Описание


Интерфейс IRubricatorFormulaData
 содержит свойства для работы с данными формулы базы данных временных рядов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [FormulaKey](IRubricatorFormulaData.FormulaKey.htm)
		 Свойство FormulaKey
		 возвращает ключ формулы.


		 ![](../../Property_Image.gif)
		 [FormulaXML](IRubricatorFormulaData.FormulaXML.htm)
		 Свойство FormulaXML
		 определяет XML код формулы.


		 ![](../../Property_Image.gif)
		 [Hash](IRubricatorFormulaData.Hash.htm)
		 Свойство Hash возвращает
		 хеш-код формулы.


		 ![](../../Property_Image.gif)
		 [Kind](IRubricatorFormulaData.Kind.htm)
		 Свойство Kind определяет
		 вид формулы.


		 ![](../../Property_Image.gif)
		 [ParentKey](IRubricatorFormulaData.ParentKey.htm)
		 Свойство ParentKey
		 определяет ключ владельца формулы.
