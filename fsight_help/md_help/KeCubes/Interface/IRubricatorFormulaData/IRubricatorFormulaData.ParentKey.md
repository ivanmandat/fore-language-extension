# IRubricatorFormulaData.ParentKey


## Синтаксис


ParentKey: Integer;


## Описание


Свойство ParentKey определяет
 ключ владельца формулы.


## Комментарии


Ключ владельца присутствует только для формулы вида RubricatorFormulaDataKind.Params.
 Вид определяется свойством [IRubricatorFormulaData.Kind](IRubricatorFormulaData.Kind.htm).
