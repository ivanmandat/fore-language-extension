# ICubeExecuteFilterFunctionFilter.Function_


## Синтаксис


Function_: String;


## Описание


Свойство Function_ определяет
 наименование функции фильтрации.


## Комментарии


Свойство реализовано на будущее, если в синтаксисе языка MDX-запросов
 появятся какие-либо дополнительные функции для фильтрации по различным
 условиям.


В текущей реализации свойство имеет значение «FILTER».
