# IValidationExecData


## Описание


Интерфейс IValidationExecData
 содержит свойства для работы с данными о расчете валидации.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ByScenario](IValidationExecData.ByScenario.htm)
		 Свойство ByScenario
		 определяет, рассчитывать ли валидацию в измерении сценариев.


		 ![](../../Property_Image.gif)
		 [Key](IValidationExecData.Key.htm)
		 Свойство Key возвращает
		 ключ расчета валидации.


		 ![](../../Property_Image.gif)
		 [Name](IValidationExecData.Name.htm)
		 Свойство Name определяет
		 наименование расчета валидации.


		 ![](../../Property_Image.gif)
		 [RevisionKey](IValidationExecData.RevisionKey.htm)
		 Свойство RevisionKey
		 возвращает ключ ревизии расчета.


		 ![](../../Property_Image.gif)
		 [Stamp](IValidationExecData.Stamp.htm)
		 Свойство Stamp определяет
		 дату выполнения расчета валидации.


		 ![](../../Property_Image.gif)
		 [ValidationKey](IValidationExecData.ValidationKey.htm)
		 Свойство ValidationKey
		 определяет ключ рассчитанного правила или группы валидаций.


		 ![](../../Property_Image.gif)
		 [Validations](IValidationExecData.Validations.htm)
		 Свойство Validations
		 определяет рассчитанные валидации.
