# IValidationExecData.Validations


## Синтаксис


Validations: Array;


## Описание


Свойство Validations определяет
 ключи рассчитанных валидаций в виде целочисленного массива.


## Комментарии


Если было рассчитано правило валидации, то в массиве будет один элемент,
 если была рассчитана группа валидаций, то число элементов будет равняться
 числу валидаций в группе.
