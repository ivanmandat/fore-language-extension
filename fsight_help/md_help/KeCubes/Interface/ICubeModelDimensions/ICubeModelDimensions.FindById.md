# ICubeModelDimensions.FindById


## Синтаксис


FindById(Id: String): [IDimensionModel](kedims.chm::/interface/idimensionmodel/idimensionmodel.htm);


## Параметры


Id. Идентификато справочника,
 который необходимо найти среди измерений куба.


## Описание


Метод FindById осуществляет
 поиск справочника, идентификатор которого передается посредством входного
 параметра.
