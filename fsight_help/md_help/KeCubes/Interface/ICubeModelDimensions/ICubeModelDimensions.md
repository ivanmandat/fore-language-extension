# ICubeModelDimensions


## Описание


Интерфейс ICubeModelDimensions
 содержит свойства и методы для доступа к свойствам измерений куба.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeModelDimensions.Count.htm)
		 Свойство Count возвращает
		 количество измерений куба.


		 ![](../../Property_Image.gif)
		 [Facts](ICubeModelDimensions.Facts.htm)
		 Свойство Facts определяет
		 справочник, используемый в качестве перечня показателей куба.


		 ![](../../Property_Image.gif)
		 [Item](ICubeModelDimensions.Item.htm)
		 Свойство Item возвращает
		 справочник репозитория, используемый в качестве измерения куба.


		 ![](../../Property_Image.gif)
		 [Units](ICubeModelDimensions.Units.htm)
		 Свойство Units определяет
		 справочник единиц измерения куба.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](ICubeModelDimensions.FindById.htm)
		 Метод FindById осуществляет
		 поиск справочника репозитория среди измерений куба по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ICubeModelDimensions.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск справочника репозитория среди измерений куба по ключу.
