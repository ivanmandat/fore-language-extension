# ICubeModelDimensions.FindByKey


## Синтаксис


FindByKey(DimensionKey: Integer): [IDimensionModel](kedims.chm::/interface/idimensionmodel/idimensionmodel.htm);


## Параметры


DimensionKey. Ключ справочника, который необходимо найти
 среди измерений куба.


## Описание


Метод FindByKey осуществляет
 поиск справочника, ключ которого передается посредством входного параметра.
