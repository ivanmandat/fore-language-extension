# IMDCalculationSliceSumm.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет
 признак использования суммирования значений дочерних элементов по данному
 измерению.


## Комментарии


Если свойству установлено значение True,
 то будет присутствовать возможность использовать суммирование по дочерним
 элементам данного измерения.
