# IMDCalculationSliceSumm


## Описание


Интерфейс IMDCalculationSliceSumm
 содержит свойства для настройки параметров суммирования по данному измерению.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Data](IMDCalculationSliceSumm.Data.htm)
		 Свойство Data определяет
		 источник данных, содержащий список элементов используемых при
		 составлении формул с возможностью суммирования.


		 ![](../../Property_Image.gif)
		 [Enabled](IMDCalculationSliceSumm.Enabled.htm)
		 Свойство Enabled определяет
		 признак использования суммирования значений дочерних элементов
		 по данному измерению.


		 ![](../../Property_Image.gif)
		 [KeyFields](IMDCalculationSliceSumm.KeyFields.htm)
		 Свойство KeyFields
		 определяет список полей, содержащих значения ключевых элементов,
		 по которым будет осуществляться связь с дочерними элементами.


		 ![](../../Property_Image.gif)
		 [LookupFields](IMDCalculationSliceSumm.LookupFields.htm)
		 Свойство LookupFields
		 определяет список полей, по которым будет осуществляться поиск
		 дочерних элементов.


		 ![](../../Property_Image.gif)
		 [Slice](IMDCalculationSliceSumm.Slice.htm)
		 Свойство Slice возвращает
		 параметры измерения.


		 ![](../../Property_Image.gif)
		 [SQL](IMDCalculationSliceSumm.SQL.htm)
		 Свойство SQL определяет
		 SQL-запрос, результатом выполнения которого будет таблица содержащая
		 список элементов, по которым будет осуществляться настройка суммирования
		 в формулах.


		 ![](../../Property_Image.gif)
		 [TableName](IMDCalculationSliceSumm.TableName.htm)
		 Свойство TableName
		 определяет наименование таблицы, содержащей список элементов,
		 по которым будет осуществляться настройка суммирования в формулах.
