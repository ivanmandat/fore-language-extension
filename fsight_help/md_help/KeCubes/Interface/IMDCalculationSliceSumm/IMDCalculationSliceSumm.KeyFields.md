# IMDCalculationSliceSumm.KeyFields


## Синтаксис


KeyFields: String;


## Описание


Свойство KeyFields определяет
 список полей, содержащих значения ключевых элементов, по которым будет
 осуществляться связь с дочерними элементами.


## Комментарии


По значениям данных элементов будет осуществляться поиск дочерних элементов
 в полях [LookupFields](IMDCalculationSliceSumm.LookupFields.htm).
 Поля в списке разделяются знаком ";".
