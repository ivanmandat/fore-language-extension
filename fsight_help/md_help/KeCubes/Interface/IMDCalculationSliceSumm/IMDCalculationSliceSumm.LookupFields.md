# IMDCalculationSliceSumm.LookupFields


## Синтаксис


LookupFields: String;


## Описание


Свойство LookupFields определяет
 список полей, по которым будет осуществляться поиск дочерних элементов.


## Комментарии


В данных полях должны содержаться значения, соответствующие родительским
 элементам, значения которых содержатся в полях [KeyFields](IMDCalculationSliceSumm.KeyFields.htm).
 Поля в списке разделяются знаком ";".
