# ICubeCallbackBeforeSaveArgument


Сборка: Cubes;


## Описание


Интерфейс ICubeCallbackBeforeSaveArgument
 предназначен для получения информации в событии перед сохранением данных
 в куб.


## Иерархия наследования


ICubeCallbackBeforeSaveArgument


## Комментарии


Интерфейс описывает аргумент события [ICubeSaveMatrixCallback.OnBeforeSave](../ICubeSaveMatrixCallback/ICubeSaveMatrixCallback.OnBeforeSave.htm).

		 Имя свойства
