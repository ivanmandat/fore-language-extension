# ICubeLoaderFactBindings


Сборка: Cubes;


## Описание


Интерфейс ICubeLoaderFactBindings
 содержит свойства и методы для работы с коллекцией привязок фактов, настраиваемых
 при загрузке данных.


## Иерархия наследования


ICubeLoaderFactBindings


## Комментарии


Привязки фактов можно получить в свойстве [ICubeLoaderFromSource.FactBindings](../ICubeLoaderFromSource/ICubeLoaderFromSource.FactBindings.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeLoaderFactBindings.Count.htm)


		 Свойство Count возвращает
		 количество параметров привязки фактов в коллекции.


		 ![](../../Property_Image.gif)
		 [CreateExternalDimension](ICubeLoaderFactBindings.CreateExternalDimension.htm)


		 Свойство CreateExternalDimension
		 определяет, создавать ли справочник для хранения фактов.


		 ![](../../Property_Image.gif)
		 [HideTrivialFactDimension](ICubeLoaderFactBindings.HideTrivialFactDimension.htm)


		 Свойство HideTrivialFactDimension
		 определяет, будет ли скрываться измерение фактов при построении
		 куба.


		 ![](../../Property_Image.gif)
		 [Item](ICubeLoaderFactBindings.Item.htm)


		 Свойство Item возвращает
		 указанные параметры привязки факта куба.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ICubeLoaderFactBindings.Add.htm)


		 Метод Add добавляет
		 в коллекцию параметры привязки факта куба.


		 ![](../../Sub_Image.gif)
		 [Clear](ICubeLoaderFactBindings.Clear.htm)


		 Метод Clear удаляет
		 из коллекции все параметры привязки.


		 ![](../../Sub_Image.gif)
		 [Remove](ICubeLoaderFactBindings.Remove.htm)


		 Метод Remove удаляет
		 указанные параметры привязки.
