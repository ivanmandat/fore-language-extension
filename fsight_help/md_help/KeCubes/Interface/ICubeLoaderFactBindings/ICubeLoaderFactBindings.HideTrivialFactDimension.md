# ICubeLoaderFactBindings.HideTrivialFactDimension


## Синтаксис


HideTrivialFactDimension: Boolean;


## Описание


Свойство HideTrivialFactDimension определяет, будет ли скрываться измерение фактов при построении куба.


## Комментарии


Данное свойство актуально, если измерение фактов состоит из одного элемента. Если свойству установлено значение True, измерение фактов будет зафиксировано по первому элементу, и в списке измерений отображаться не будет.
