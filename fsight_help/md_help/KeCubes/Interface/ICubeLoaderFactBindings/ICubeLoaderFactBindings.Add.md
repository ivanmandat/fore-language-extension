# ICubeLoaderFactBindings.Add


## Синтаксис


Add(SourceField: String): [ICubeLoaderFactBinding](../ICubeLoaderFactBinding/ICubeLoaderFactBinding.htm);


## Параметры


SourceField. Наименование поля источника данных, к которому осуществляется привязка факта.


## Описание


Метод Add добавляет в коллекцию параметры привязки факта куба.
