# ICubeLoaderFactBindings.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс параметров привязки факта куба.


## Описание


Метод Remove удаляет параметры привязки, индекс которых передается посредством параметра Index.


## Комментарии


Метод возвращает значение True, если параметры привязки были удалены и False, если удаление невозможно.
