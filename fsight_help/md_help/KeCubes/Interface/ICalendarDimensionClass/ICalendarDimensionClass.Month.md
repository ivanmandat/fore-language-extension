# ICalendarDimensionClass.Month


## Синтаксис


		Month(Calendar: [IDimInstance](kedims.chm::/interface/idiminstance/idiminstance.htm);
		 El: Integer): Integer;


## Параметры


Calendar. Данные календарного
 справочника, по которым выполняется расчет;


El. Индекс элемента, относительно
 которого выполняется расчет.


## Описание


Метод Month возвращает индекс
 элемента, соответствующий месяцу, в который входит указанный элемент.


## Комментарии


Для корректного расчёта календарный справочник должен содержать уровень
 «Месяцы». Если уровень отсутствует, то метод возвращает значение «-1».


Например, если календарный справочник содержит уровни кварталы, месяцы,
 дни и расчёт выполняется для:


	- элемента квартальной динамики, то метод Month
	 возвращает элемент, соответствующий месяцу началу квартала;


	- элемента дневной динамики, то метод Month
	 возвращает элемент, соответствующий месяцу, в который входит день.


## Пример


Использование метода аналогично использованию в примере для [ICalendarDimensionClass.Day](ICalendarDimensionClass.Day.htm).


Примечание.
 Для выполнения примера замените метод Day
 на Month. Убедитесь, что в репозитории
 находятся таблица с данными по месяцам или дням и календарный справочник,
 содержащий уровень «Месяцы», и задайте соответствующие настройки стандартного
 куба.
