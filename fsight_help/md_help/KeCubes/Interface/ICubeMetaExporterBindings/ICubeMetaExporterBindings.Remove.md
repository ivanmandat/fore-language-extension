# ICubeMetaExporterBindings.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс элемента в коллекции.


## Описание


Метод Remove осуществляет удаление элемента из коллекции по индексу. Возвращает значение True, если удаление прошло успешно.
