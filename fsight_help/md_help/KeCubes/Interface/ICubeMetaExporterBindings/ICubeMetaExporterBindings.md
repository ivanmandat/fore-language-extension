# ICubeMetaExporterBindings


## Описание


Интерфейс ICubeMetaExporterBindings
 содержит свойства и методы для работы с коллекцией привязок атрибутов
 базы данных временных рядов с полями приёмника данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeMetaExporterBindings.Count.htm)
		 Свойство Count возвращает
		 количество элементов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ICubeMetaExporterBindings.Item.htm)
		 Свойство Item возвращает
		 элемент коллекции по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ICubeMetaExporterBindings.Add.htm)
		 Метод Add осуществляет
		 добавление элемента в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](ICubeMetaExporterBindings.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех элементов из коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](ICubeMetaExporterBindings.Remove.htm)
		 Метод Remove осуществляет
		 удаление элемента из коллекции по индексу.
