# IImportRequestInstance


Сборка: Cubes;


## Описание


Интерфейс IImportRequestInstance
 содержит свойства и методы, предназначенные для импорта временных рядов.


## Иерархия наследования


           IImportRequestInstance


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DisableTransactions](IImportRequestInstance.DisableTransactions.htm)
		 Свойство DisableTransactions
		 определяет, выполнять ли импорт временных рядов за одну транзакцию.


		 ![](../../Property_Image.gif)
		 [ImportCallback](IImportRequestInstance.ImportCallback.htm)
		 Свойство ImportCallback
		 определяет объект, используемый для обработки событий, возникающих
		 во время импорта временных рядов.


		 ![](../../Property_Image.gif)
		 [ImportResult](IImportRequestInstance.ImportResult.htm)
		 Свойство ImportResult
		 возвращает данные о результатах импорта временных рядов.


		 ![](../../Property_Image.gif)
		 [Provider](IImportRequestInstance.Provider.htm)
		 Свойство Provider возвращает
		 объект, являющийся источником данных.
