# ICubeInstanceDestinations.Item


## Синтаксис


Item(Index: Integer): [ICubeInstanceDestination](../ICubeInstanceDestination/ICubeInstanceDestination.htm);


## Параметры


Index - индекс варианта отображения.


## Описание


Свойство Item возвращает объект, содержащий свойства и методы для работы с отстроенным кубом, в соответствии с вариантом отображения, индекс которого передается посредством параметра Index.
