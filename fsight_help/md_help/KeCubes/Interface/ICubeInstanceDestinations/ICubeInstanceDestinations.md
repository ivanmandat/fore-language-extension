# ICubeInstanceDestinations


## Описание


Интерфейс ICubeInstanceDestinations
 содержит свойства и методы для работы с вариантами отображения отстроенного
 куба.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeInstanceDestinations.Count.htm)
		 Свойство Count возвращает
		 количество вариантов отображения куба.


		 ![](../../Property_Image.gif)
		 [DefaultDestination](ICubeInstanceDestinations.DefaultDestination.htm)
		 Свойство DefaultDestination
		 возвращает объект, содержащий свойства и методы для работы с отстроенным
		 кубом, в соответствии с вариантом отображения, установленным для
		 куба по умолчанию.


		 ![](../../Property_Image.gif)
		 [Item](ICubeInstanceDestinations.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий свойства и методы для работы с отстроенным
		 кубом.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindByKey](ICubeInstanceDestinations.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск и возвращает объект, содержащий свойства и методы для работы
		 с отстроенным кубом, в случае удачного поиска.
