# ICubeInstanceDestinations.FindByKey


## Синтаксис


FindByKey(DestinationKey: Integer): [ICubeInstanceDestination](../ICubeInstanceDestination/ICubeInstanceDestination.htm);


## Параметры


DestinationKey - ключ варианта отображения куба, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет поиск и возвращает объект, содержащий свойства и методы для работы с отстроенным кубом, в случае удачного поиска. Ключ варианта отображения передается посредством параметра DestinationKey.
