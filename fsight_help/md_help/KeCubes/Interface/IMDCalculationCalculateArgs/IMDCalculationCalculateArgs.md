# IMDCalculationCalculateArgs


## Описание


Интерфейс IMDCalculationCalculateArgs
 содержит свойства определяющие параметры многомерного расчёта на сервере
 БД.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AutoSaveSQL](IMDCalculationCalculateArgs.AutoSaveSQL.htm)
		 Свойство AutoSaveSQL
		 определяет, будет ли производиться автоматическое сохранение выполняемого
		 SQL-запроса в файл.


		 ![](../../Property_Image.gif)
		 [CleanType](IMDCalculationCalculateArgs.CleanType.htm)
		 Свойство CleanType
		 определяет способ очистки приёмника данных перед расчётом.


		 ![](../../Property_Image.gif)
		 [DateBegin](IMDCalculationCalculateArgs.DateBegin.htm)
		 Свойство DateBegin
		 определяет дату начала периода расчёта.


		 ![](../../Property_Image.gif)
		 [DateEnd](IMDCalculationCalculateArgs.DateEnd.htm)
		 Свойство DateEnd определяет
		 дату окончания периода расчёта.


		 ![](../../Property_Image.gif)
		 [Recursion](IMDCalculationCalculateArgs.Recursion.htm)
		 Свойство Recursion
		 определяет, будет ли учитываться зависимость формул друг от друга
		 при расчёте.


		 ![](../../Property_Image.gif)
		 [RecursionEnabled](IMDCalculationCalculateArgs.RecursionEnabled.htm)
		 Свойство RecursionEnabled
		 возвращает признак доступности использовать рекурсию в формулах.


		 ![](../../Property_Image.gif)
		 [RecursionSource](IMDCalculationCalculateArgs.RecursionSource.htm)
		 Свойство RecursionSource
		 возвращает индекс источника в многомерном расчёте, который является
		 рекурсивным.


		 ![](../../Property_Image.gif)
		 [SliceFilter](IMDCalculationCalculateArgs.SliceFilter.htm)
		 Свойство SliceFilter
		 возвращает настройку фильтрации элементов, по которым будет производиться
		 расчёт.


		 ![](../../Property_Image.gif)
		 [UpdateType](IMDCalculationCalculateArgs.UpdateType.htm)
		 Свойство UpdateType
		 определяет способ обновления данных при расчёте.
