# ICubeCacheSaver.Global


## Синтаксис


Global: Boolean;


## Описание


Свойство Global определяет признак
 использования общего кеша для куба при работе нескольких пользователей.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 При работе с экземпляром куба будет использоваться один общий кеш.
	 При обращении различных пользователей к данным куба, если необходимый
	 набор данных уже закеширован, то он будет взят из общего кеша.


	- False. При работе с
	 каждым экземпляром куба будет создаваться отдельный кеш.


Данное свойство используется при работе BI-сервера продукта «Форсайт. Аналитическая платформа».
