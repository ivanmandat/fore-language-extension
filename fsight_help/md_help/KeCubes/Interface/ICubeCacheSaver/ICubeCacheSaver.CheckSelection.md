# ICubeCacheSaver.CheckSelection


## Синтаксис


CheckSelection: Boolean;


## Описание


Свойство CheckSelection определяет
 признак проверки отметки, по которой был сохранен кеш.


## Комментарии


Значение свойства используется при построении матрицы с данными куба.


По умолчанию свойству установлено значение False,
 при этом проверка текущей отметки, по которой строится матрица, и отметки,
 по которой сохранен кеш, не осуществляется. Если свойству установить значение
 True, то при построении матрицы
 будет осуществляться сравнение отметок. Если отметки отличаются, то данные
 будут загружены не из кеша, а из базы данных, в которой хранятся данные
 куба.
