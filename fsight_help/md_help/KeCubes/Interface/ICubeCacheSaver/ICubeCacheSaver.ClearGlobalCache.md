# ICubeCacheSaver.ClearGlobalCache


## Синтаксис


ClearGlobalCache;


## Описание


Метод ClearGlobalCache удаляет
 из глобального кеша все кешированные данные для текущего куба.


## Комментарии


Метод удаляет кешированные данные куба, для которого создан текущий
 объект, описываемый интерфейсом ICubeCacheSaver.
