# ICubeCacheSaver.Clear


## Синтаксис


Clear([Key: Integer = -1]): Boolean;


## Параметры


Key. Ключ варианта отображения
 куба.


## Описание


Метод Clear осуществляет очистку
 кеша для заданного [варианта
 отображения](../ICubeModelDestination/ICubeModelDestination.htm) куба.


## Комментарии


Если параметр Key не указан,
 то удаляется кеш для всех [вариантов
 отображения](../ICubeModelDestination/ICubeModelDestination.htm) куба.
