# IMDCalculationDatasetInstance.IsDestination


## Синтаксис


IsDestination: Boolean;


## Описание


Свойство IsDestination возвращает
 признак того, что данный объект является назначением многомерного расчета
 на сервере БД.


## Комментарии


Свойство возвращает значение True,
 если объект является назначением многомерного расчета на сервере БД и
 False, если источником.
