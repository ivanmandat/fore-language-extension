# IMDCalculationDatasetInstance.KeyToCoord


## Синтаксис


KeyToCoord(Key: [IMDCalculationFormulaElementKey](../IMDCalculationFormulaElementKey/IMDCalculationFormulaElementKey.htm); [Coord: [IMatrixCoord](KeMatrix.chm::/Interface/IMatrixCoord/IMatrixCoord.htm) = Null]): [IMatrixCoord](KeMatrix.chm::/Interface/IMatrixCoord/IMatrixCoord.htm);


## Параметры


Key - уникальный ключ элемента, который необходимо преобразовать в координату матрицы.


Coord - необязательный параметр, определяющий координату, свойства которой будут изменены в соответствии со значением указанного уникального ключа.


## Описание


Метод KeyToCoord осуществляет преобразование уникального ключа элемента в координату в источнике/приёмнике данных.
