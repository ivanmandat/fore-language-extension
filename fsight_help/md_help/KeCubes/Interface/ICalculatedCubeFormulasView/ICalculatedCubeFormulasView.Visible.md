# ICalculatedCubeFormulasView.Visible


## Синтаксис


Visible(DimensionKey: Integer): Boolean;


## Параметры


DimensionKey. Ключ измерения, для которого необходимо
 определить видимость наименований элементов в наименованиях термов.


## Описание


Свойство Visible определяет,
 будет ли отображаться наименование элементов по указанному измерению в
 формулах.
