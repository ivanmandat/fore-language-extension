# ICalculatedCubeFormulasView.ViewNaming


## Синтаксис


ViewNaming(DimensionKey: Integer): [IDimAttribute](kedims.chm::/interface/idimattribute/idimattribute.htm);


## Параметры


DimensionKey. Ключ измерения, для которого необходимо
 установить отображаемый в наименованиях элементов, атрибут.


## Описание


Свойство ViewNaming определяет
 атрибут, отображаемый в наименованиях элементов в измерении вычисляемого
 куба.
