# ICalculatedCubeFormulasView.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 измерений вычисляемого куба, для которых возможна настройка отображения
 элементов.


## Комментарии


Настройка возможна для всех незафиксированных измерений.
