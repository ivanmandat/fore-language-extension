# ICalculatedCubeFormulasView


## Описание


Интерфейс ICalculatedCubeFormulasView
 содержит свойства, определяющие параметры отображения элементов измерений
 в формулах куба.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICalculatedCubeFormulasView.Count.htm)
		 Свойство Count возвращает
		 количество измерений вычисляемого куба, для которых возможна настройка
		 отображения элементов.


		 ![](../../Property_Image.gif)
		 [Dimension](ICalculatedCubeFormulasView.Dimension.htm)
		 Свойство Dimension
		 возвращает структуру измерения.


		 ![](../../Property_Image.gif)
		 [FormulaNaming](ICalculatedCubeFormulasView.FormulaNaming.htm)
		 Свойство FormulaNaming
		 определяет атрибут, отображаемый в наименованиях элементов в формулах
		 вычисляемого куба.


		 ![](../../Property_Image.gif)
		 [SourceNaming](ICalculatedCubeFormulasView.SourceNaming.htm)
		 Свойство SourceNaming
		 определяет формат обозначения источника данных в наименованиях
		 элементов в формулах вычисляемого куба.


		 ![](../../Property_Image.gif)
		 [ViewNaming](ICalculatedCubeFormulasView.ViewNaming.htm)
		 Свойство ViewNaming
		 определяет атрибут, отображаемый в наименованиях элементов в измерении
		 вычисляемого куба.


		 ![](../../Property_Image.gif)
		 [Visible](ICalculatedCubeFormulasView.Visible.htm)
		 Свойство Visible определяет,
		 будет ли отображаться наименование элементов по указанному измерению
		 в формулах.
