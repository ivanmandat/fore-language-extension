# ICalculatedCubeFormulasView.FormulaNaming


## Синтаксис


FormulaNaming(DimensionKey: Integer): [IDimAttribute](kedims.chm::/interface/idimattribute/idimattribute.htm);


## Параметры


DimensionKey. Ключ измерения,
 для которого необходимо установить отображаемый в формуле, атрибут.


## Описание


Свойство FormulaNaming определяет
 атрибут, отображаемый в наименованиях элементов в формулах вычисляемого
 куба.
