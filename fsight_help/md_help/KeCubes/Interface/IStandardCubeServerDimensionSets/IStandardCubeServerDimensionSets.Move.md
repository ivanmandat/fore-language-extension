# IStandardCubeServerDimensionSets.Move


## Синтаксис


Move(IndexFrom: Integer; IndexTo: Integer);


## Параметры


IndexFrom - индекс позиции группы измерений, которую необходимо переместить.


IndexTo - индекс позиции, в которую необходимо переместить группу измерений.


## Описание


Метод Move осуществляет перемещение групп измерений в списке.
