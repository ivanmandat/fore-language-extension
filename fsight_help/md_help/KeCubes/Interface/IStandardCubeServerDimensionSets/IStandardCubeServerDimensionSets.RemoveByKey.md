# IStandardCubeServerDimensionSets.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key - ключ удаляемой группы измерений.


## Описание


Метод RemoveByKey осуществляет удаление группы измерений по ключу. Данный метод возвращает значение True, если удаление прошло успешно.
