# IStandardCubeServerDimensionSets


## Описание


Интерфейс IStandardCubeServerDimensionSets
 содержит свойства и методы для работы с группами измерений, по которым
 на сервере рассчитывается агрегация.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IStandardCubeServerDimensionSets.Count.htm)
		 Свойство Count возвращает
		 количество групп измерений.


		 ![](../../Property_Image.gif)
		 [Item](IStandardCubeServerDimensionSets.Item.htm)
		 Свойство Item возвращает
		 группу измерений по индексу.

		 Имя метода
		 Краткое описание
