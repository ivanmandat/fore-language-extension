# IStandardCubeServerDimensionSets.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс удаляемой группы измерений.


## Описание


Метод Remove осуществляет удаление группы измерений по индексу. Данный метод возвращает значение True, если удаление прошло успешно.
