# IMDCalculationFormulasIterator


## Описание


Интерфейс IMDCalculationFormulasIterator реализует итератор, позволяющий перемещаться по формулам многомерного расчета.


## Методы


 Имя метода

 ![](../../Sub_Image.gif)
 [Key](IMDCalculationFormulasIterator.Key.htm)
 Метод Key возвращает ключ текущего элемента.

 Метод Next осуществляет переход на следующую запись в итераторе.
