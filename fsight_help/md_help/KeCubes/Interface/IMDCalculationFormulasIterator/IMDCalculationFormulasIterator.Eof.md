# IMDCalculationFormulasIterator.Eof


## Синтаксис


Eof: Boolean;


## Описание


Метод Eof возвращает признак расположения курсора на последней формуле в итераторе.


## Комментарии


Метод возвращает True, если курсор находится на последней записи в итераторе.
