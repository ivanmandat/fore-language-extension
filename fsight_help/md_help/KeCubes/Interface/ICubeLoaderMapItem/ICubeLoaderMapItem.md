# ICubeLoaderMapItem


## Описание


Интерфейс ICubeLoaderMapItem
 содержит свойства, определяющие параметры связи измерения-приёмника и
 измерений куба-источника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dimension](ICubeLoaderMapItem.Dimension.htm)
		 Свойство Dimension
		 возвращает измерение-приёмник, с которым осуществляется связь
		 измерения-источника.


		 ![](../../Property_Image.gif)
		 [DimIndex](ICubeLoaderMapItem.DimIndex.htm)
		 Свойство DimIndex определяет
		 индекс измерения-приёмника, по которому осуществляется связь с
		 измерением куба-источника.


		 ![](../../Property_Image.gif)
		 [MapDimension](ICubeLoaderMapItem.MapDimension.htm)
		 Свойство MapDimension
		 определяет измерение куба-источника, которое связано с измерением
		 [Dimension](ICubeLoaderMapItem.Dimension.htm).


		 ![](../../Property_Image.gif)
		 [MapIndex](ICubeLoaderMapItem.MapIndex.htm)
		 Свойство MapIndex определяет
		 индекс измерения-источника, по которому осуществляется связь с
		 измерением [Dimension](ICubeLoaderMapItem.Dimension.htm).


		 ![](../../Property_Image.gif)
		 [Method](ICubeLoaderMapItem.Method.htm)
		 Свойство Method определяет
		 метод связи измерений.
