# ICubeModel


## Описание


Интерфейс ICubeModel содержит
 базовые свойства модели куба.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Destinations](ICubeModel.Destinations.htm)
		 Свойство Destinations
		 возвращает коллекцию вариантов отображения куба.


		 ![](../../Property_Image.gif)
		 [LockTableForSave](ICubeModel.LockTableForSave.htm)
		 Свойство LockTableForSave
		 определяет признак блокировки таблицы куба при вставке/обновлении
		 данных.


		 ![](../../Property_Image.gif)
		 [OldSaveMethod](ICubeModel.OldSaveMethod.htm)
		 Свойство OldSaveMethod
		 определяет признак использования старого метода сохранения данных.


		 ![](../../Property_Image.gif)
		 [SelectionControl](ICubeModel.SelectionControl.htm)
		 Свойство SelectionControl
		 возвращает коллекцию управляющих измерений куба.


		 ![](../../Property_Image.gif)
		 [Sources](ICubeModel.Sources.htm)
		 Свойство Sources возвращает
		 коллекцию кубов-источников данных куба.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Validate](ICubeModel.Validate.htm)


		 Метод Validate осуществляет
		 проверку метаданных куба, находящегося на стадии редактирования.
