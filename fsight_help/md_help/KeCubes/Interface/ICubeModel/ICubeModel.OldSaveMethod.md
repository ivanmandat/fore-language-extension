# ICubeModel.OldSaveMethod


## Синтаксис


OldSaveMethod: Boolean;


## Описание


Свойство OldSaveMethod определяет
 признак использования старого метода сохранения данных.


## Комментарии


Допустимые значения:


	- True. Использовать старый
	 метод сохранения данных через операторы Insert/Update.


	- False. Значение по умолчанию.
	 Использовать новый метод сохранения данных через оператор Merge.


Старый метод сохранения данных использовался в версиях Prognoz Platform до 8.0 и оставлен
 для совместимости.
