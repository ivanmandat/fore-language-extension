# ICubeAttachment.AttributeValueByKey


## Синтаксис


		AttributeValueByKey(AttributeKey: Integer):
		 Variant;


## Параметры


AttributeKey. Ключ атрибута
 в справочнике вложений.


## Описание


Свойство AttributeValueByKey
 возвращает значение атрибута с указанным ключом из справочника с вложениями.
