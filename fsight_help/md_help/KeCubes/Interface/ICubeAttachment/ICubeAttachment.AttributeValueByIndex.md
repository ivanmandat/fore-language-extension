# ICubeAttachment.AttributeValueByIndex


## Синтаксис


		AttributeValueByIndex(Index: Integer): Variant;


## Параметры


Index. Индекс атрибута в справочнике
 вложений.


## Описание


Свойство AttributeValueByIndex
 возвращает значение атрибута с указанным индексом из справочника с вложениями.
