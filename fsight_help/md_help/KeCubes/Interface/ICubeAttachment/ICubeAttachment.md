# ICubeAttachment


Сборка: Cubes;


## Описание


Интерфейс ICubeAttachment предназначен
 для работы с вложением, хранящимся по координате куба.


## Иерархия наследования


ICubeAttachment


## Комментарии


Вложение может быть получено с помощью свойств и методов интерфейса
 [ICubeAttachments](../ICubeAttachments/ICubeAttachments.htm).
