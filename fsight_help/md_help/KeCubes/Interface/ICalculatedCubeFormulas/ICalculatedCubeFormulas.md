# ICalculatedCubeFormulas


## Описание


Интерфейс ICalculatedCubeFormulas
 содержит свойства и методы для работы с коллекцией формул, связанных с
 координатой в вычисляемом кубе.


## Комментарии


Коллекцию формул можно получить, используя свойство [ICalculatedCubeInstance.Formula](../ICalculatedCubeInstance/ICalculatedCubeInstance.Formula.htm).

## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ICalculatedCubeFormulas.Add.htm)
		 Метод Add осуществляет
		 добавление новой формулы.


		 ![](../../Sub_Image.gif)
		 [Find](ICalculatedCubeFormulas.Find.htm)
		 Метод Find осуществляет
		 поиск формулы, в период действия которой, попадает указанная дата.


		 ![](../../Sub_Image.gif)
		 [FindPeriod](ICalculatedCubeFormulas.FindPeriod.htm)
		 Метод FindPeriod осуществляет
		 поиск формулы, в период действия которой, попадает указанный период.


		 ![](../../Sub_Image.gif)
		 [Save](ICalculatedCubeFormulas.Save.htm)
		 Метод Save осуществляет
		 сохранение изменений, внесенных в выражения формул по данной координате.


		 ![](../../Sub_Image.gif)
		 [ToKey](ICalculatedCubeFormulas.ToKey.htm)
		 Метод ToKey осуществляет
		 преобразование и возвращает уникальный ключ, соответствующий формулам
		 по данной координате.
