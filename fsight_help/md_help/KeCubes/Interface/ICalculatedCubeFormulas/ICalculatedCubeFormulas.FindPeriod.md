# ICalculatedCubeFormulas.FindPeriod


## Синтаксис


FindPeriod(Start: Integer; DateBegin: DateTime; DateEnd: DateTime): Integer;


## Параметры


Start - индекс формулы, начиная с которой, будет производиться поиск.


DateBegin - начальная дата периода.


DateEnd - конечная дата периода.


## Описание


Метод FindPeriod осуществляет поиск формулы, в период действия которой, попадает указанный период, и возвращает индекс формулы в случае удачного поиска.
