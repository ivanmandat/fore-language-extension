# ICalculatedCubeFormulas.Find


## Синтаксис


Find(DatePeriod: DateTime): [ICalculatedCubeFormula](../ICalculatedCubeFormula/ICalculatedCubeFormula.htm);


## Параметры


DatePeriod - дата, попадающая в период действия формулы.


## Описание


Метод Find осуществляет поиск формулы, в период действия которой, попадает дата, передаваемая посредством параметра DatePeriod.
