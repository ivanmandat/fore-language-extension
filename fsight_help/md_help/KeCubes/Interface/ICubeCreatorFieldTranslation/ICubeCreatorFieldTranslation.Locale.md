# ICubeCreatorFieldTranslation.Locale


## Синтаксис


Locale: [LocaleCodeID](ForeSys.chm::/Enums/LocaleCodeID.htm)


## Описание


Свойство Locale возвращает язык
 перевода.


## Комментарии


По языку определяется атрибут перевода справочника, к которому осуществляется
 привязка. Для создания привязки используйте метод [ICubeCreatorFieldTranslations.Add](../ICubeCreatorFieldTranslations/ICubeCreatorFieldTranslations.Add.htm).
