# ICubeCreatorFieldTranslation


Сборка: Cubes;


## Описание


Интерфейс ICubeCreatorFieldTranslation
 содержит свойства привязки атрибута с переводом к полю источника.


## Иерархия наследования


ICubeCreatorFieldTranslation


## Комментарии


Привязку можно получить, используя свойства и методы интерфейса [ICubeCreatorFieldTranslations](../ICubeCreatorFieldTranslations/ICubeCreatorFieldTranslations.htm).
