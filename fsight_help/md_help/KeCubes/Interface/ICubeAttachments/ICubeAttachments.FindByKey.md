# ICubeAttachments.FindByKey


## Синтаксис


		FindByKey(Element: Integer): [ICubeAttachment](../ICubeAttachment/ICubeAttachment.htm);


## Параметры


Element. Ключ вложения.


## Описание


Метод FindByKey осуществляет
 поиск по ключу и возвращает параметры вложения в случае удачного поиска.


## Комментарии


Метод возвращает значение Null,
 если вложение с указанными ключом не найдено.
