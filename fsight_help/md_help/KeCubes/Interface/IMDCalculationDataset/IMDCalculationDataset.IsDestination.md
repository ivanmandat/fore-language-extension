# IMDCalculationDataset.IsDestination


## Синтаксис


IsDestination: Boolean;


## Описание


Свойство IsDestination возвращает
 признак того, что данный объект является назначением многомерного расчета
 на сервере БД.


## Комментарии


Свойство возвращает следующее значение:


	- True. Объект является
	 назначением многомерного расчета;


	- False. Объект является
	 источником многомерного расчета.
