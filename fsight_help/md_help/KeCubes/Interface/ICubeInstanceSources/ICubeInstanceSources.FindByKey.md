# ICubeInstanceSources.FindByKey


## Синтаксис


FindByKey(Key: Integer): [ICubeInstanceSource](../ICubeInstanceSource/ICubeInstanceSource.htm);


## Параметры


Key - ключ куба-источника данных.


## Описание


Метод FindByKey осуществляет поиск по ключу и возвращает объект, содержащий свойства и методы для работы с отстроенным кубом-источником, в случае удачного поиска. Ключ куба-источника передается посредством параметра Key.
