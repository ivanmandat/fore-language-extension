# ICubeInstanceSources


## Описание


Интерфейс ICubeInstanceSources
 содержит свойства и методы для работы с коллекцией отстроенных кубов,
 являющихся источниками данных для данного куба.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeInstanceSources.Count.htm)
		 Свойство Count возвращает
		 количество кубов-источников.


		 ![](../../Property_Image.gif)
		 [Item](ICubeInstanceSources.Item.htm)
		 Свойство Item возвращает
		 объект, содержащий свойства и методы для работы отстроенным кубом-источником.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](ICubeInstanceSources.FindById.htm)
		 Метод FindById осуществляет
		 поиск по идентификатору и возвращает объект, содержащий свойства
		 и методы для работы с отстроенным кубом-источником, в случае удачного
		 поиска.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ICubeInstanceSources.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск по ключу и возвращает объект, содержащий свойства и методы
		 для работы с отстроенным кубом-источником, в случае удачного поиска.
