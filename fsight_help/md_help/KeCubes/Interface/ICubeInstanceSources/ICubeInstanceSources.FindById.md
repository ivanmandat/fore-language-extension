# ICubeInstanceSources.FindById


## Синтаксис


FindById(Id: String): [ICubeInstanceSource](../ICubeInstanceSource/ICubeInstanceSource.htm);


## Параметры


Id - идентификатор куба-источника данных.


## Описание


Метод FindById осуществляет поиск по идентификатору и возвращает объект, содержащий свойства и методы для работы с отстроенным кубом-источником, в случае удачного поиска.


## Комментарии


Идентификатор куба-источника передается посредством параметра id.
