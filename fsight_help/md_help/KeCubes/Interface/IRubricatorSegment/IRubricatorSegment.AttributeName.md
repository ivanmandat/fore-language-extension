# IRubricatorSegment.AttributeName


## Синтаксис


AttributeName: String;


## Описание


Свойство AttributeName определяет
 наименование атрибута, указывающего измерение сегмента.


## Комментарии


Данное свойство актуально, если сегмент представляет собой диапазон
 данных: [IRubricatorSegment.Type](IRubricatorSegment.Type.htm)
 = [RubricatorSegmentType.Factors](../../Enums/RubricatorSegmentType.htm).
