# IRubricatorSegment


## Описание


Интерфейс IRubricatorSegment
 содержит свойства и методы для работы с сегментом данных базы данных временных
 рядов.


## Комментарии


Сегмент - это подмножество данных базы данных временных рядов, которое
 определяется путем задания отметки для измерения или календаря базы.
