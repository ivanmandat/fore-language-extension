# ICubeSegmentDimensions.FindByDim


## Синтаксис


FindByDim(Dimension: [IDimensionModel](KeDims.chm::/interface/IDimensionModel/IDimensionModel.htm)):
 [ICubeSegmentDimension](../ICubeSegmentDimension/ICubeSegmentDimension.htm);


## Параметры


Dimension. Справочник репозитория.


## Описание


Метод FindByDim осуществляет
 поиск по справочнику и возвращает настройки измерения сегмента куба.


## Комментарии


Метод возвращает значение Null,
 если измерение на основе указанного справочника отсутствует в коллекции.
