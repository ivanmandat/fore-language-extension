# ICubeSegmentDimensions.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ справочника, на базе
 которого построено измерение сегмента куба.


## Описание


Метод RemoveByKey удаляет из
 коллекции измерение, основанное на справочнике с указанным ключом.


## Комментарии


Метод возвращает значение True,
 если измерение удалёно и False,
 если удаление по каким-либо причинам не доступно.
