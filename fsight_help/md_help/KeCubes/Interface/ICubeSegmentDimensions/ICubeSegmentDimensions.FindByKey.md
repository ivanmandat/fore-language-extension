# ICubeSegmentDimensions.FindByKey


## Синтаксис


FindByKey(Key: Integer): [ICubeSegmentDimension](../ICubeSegmentDimension/ICubeSegmentDimension.htm);


## Параметры


Key. Ключ справочника репозитория.


## Описание


Метод FindByKey осуществляет
 поиск по ключу и возвращает настройки измерения сегмента куба.


## Комментарии


Метод возвращает значение Null,
 если измерение на основе справочника с указанным ключом отсутствует в
 коллекции.
