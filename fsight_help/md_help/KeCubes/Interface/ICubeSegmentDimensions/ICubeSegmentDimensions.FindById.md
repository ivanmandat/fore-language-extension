# ICubeSegmentDimensions.FindById


## Синтаксис


FindById(Id: String): [ICubeSegmentDimension](../ICubeSegmentDimension/ICubeSegmentDimension.htm);


## Параметры


Id. Идентификатор справочника
 репозитория.


## Описание


Метод FindById осуществляет
 поиск по идентификатору и возвращает настройки измерения сегмента куба.


## Комментарии


Метод возвращает значение Null,
 если измерение на основе справочника с указанным идентификатором отсутствует
 в коллекции.
