# ICubeSegmentDimensions.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс измерения сегмента
 куба.


## Описание


Метод Remove удаляет из коллекции
 измерение с указанным индексом.


## Комментарии


Метод возвращает значение True,
 если измерение удалёно и False,
 если удаление по каким-либо причинам не доступно.
