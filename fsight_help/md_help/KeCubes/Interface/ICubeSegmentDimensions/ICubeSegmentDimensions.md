# ICubeSegmentDimensions


Сборка: Cubes;


## Описание


Интерфейс ICubeSegmentDimensions
 предназначен для работы с коллекцией измерений сегмента куба.


## Иерархия наследования


ICubeSegmentDimensions


## Комментарии


Коллекцию измерений можно получить, используя свойство [ICubeSegmentContainer.Dimensions](../ICubeSegmentContainer/ICubeSegmentContainer.Dimensions.htm)
 или [ICubeSegment.Dimensions](../ICubeSegment/ICubeSegment.Dimensions.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeSegmentDimensions.Count.htm)
		 Свойство Count возвращает
		 количество измерений в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ICubeSegmentDimensions.Item.htm)
		 Свойство Item возвращает
		 настройки измерения с указанным индексом.


		 ![](../../Property_Image.gif)
		 [SegmentContainer](ICubeSegmentDimensions.SegmentContainer.htm)
		 Свойство SegmentContainer
		 возвращает контейнер, которому соответствует коллекция измерений.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ICubeSegmentDimensions.Add.htm)
		 Метод Add добавляет
		 в коллекцию измерение на основе указанного справочника.


		 ![](../../Sub_Image.gif)
		 [FindByDim](ICubeSegmentDimensions.FindByDim.htm)
		 Метод FindByDim осуществляет
		 поиск по справочнику и возвращает настройки измерения сегмента
		 куба.


		 ![](../../Sub_Image.gif)
		 [FindById](ICubeSegmentDimensions.FindById.htm)
		 Метод FindById осуществляет
		 поиск по идентификатору и возвращает настройки измерения сегмента
		 куба.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ICubeSegmentDimensions.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск по ключу и возвращает настройки измерения сегмента куба.


		 ![](../../Sub_Image.gif)
		 [IndexOf](ICubeSegmentDimensions.IndexOf.htm)
		 Метод IndexOf возвращает
		 индекс измерения сегмента куба в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](ICubeSegmentDimensions.Remove.htm)
		 Метод Remove удаляет
		 из коллекции измерение с указанным индексом.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](ICubeSegmentDimensions.RemoveByKey.htm)
		 Метод RemoveByKey удаляет
		 из коллекции измерение, основанное на справочнике с указанным
		 ключом.
