# IVirtualCubeSourceSelection.Selection


## Синтаксис


		Selection: [IDimSelectionSet](KeDims.chm::/interface/IDimSelectionSet/IDimSelectionSet.htm);


## Описание


Свойство Selection возвращает
 отметку источника.


## Комментарии


Свойство возвращает отметку источника, которая сформирована в соответствии
 с отметкой, заданной для виртуального куба при расчёте. При необходимости
 отметку источника можно изменить.
