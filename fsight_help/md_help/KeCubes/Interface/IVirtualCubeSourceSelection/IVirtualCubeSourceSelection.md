# IVirtualCubeSourceSelection


Сборка: Cubes;


## Описание


Интерфейс IVirtualCubeSourceSelection
 предназначен для работы с отметкой источника виртуального куба, которая
 используется при расчёте куба.


## Иерархия наследования


IVirtualCubeSourceSelection


## Комментарии


Отметку источника можно получить, используя свойства и методы интерфейса
 [IVirtualCubeSourceSelections](../IVirtualCubeSourceSelections/IVirtualCubeSourceSelections.htm).
