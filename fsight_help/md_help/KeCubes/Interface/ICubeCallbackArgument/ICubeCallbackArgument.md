# ICubeCallbackArgument


## Описание


Интерфейс ICubeCallbackArgument
 содержит свойства аргумента события, происходящего при возникновении ошибки
 во время сохранения данных в куб.


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Error](ICubeCallbackArgument.Error.htm)
		 Свойство Error определяет
		 информацию об ошибке.


		 ![](../../Property_Image.gif)
		 [IgnoreAllErrors](ICubeCallbackArgument.IgnoreAllErrors.htm)
		 Свойство IgnoreAllErrors
		 определяет, будут ли пропускаться все ошибки.


		 ![](../../Property_Image.gif)
		 [IgnoreError](ICubeCallbackArgument.IgnoreError.htm)
		 Свойство IgnoreError
		 определяет, будет ли пропускаться текущая ошибка.


		 ![](../../Property_Image.gif)
		 [Options](ICubeCallbackArgument.Options.htm)
		 Свойство не реализовано.
