# IAutoCubeInstance


## Описание


Интерфейс IAutoCubeInstance
 содержит методы для работы с данными автоматического куба.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CopyElementsData](IAutoCubeInstance.CopyElementsData.htm)
		 Метод CopyElementsData
		 осуществляет копирование данных автоматического куба в соответствии
		 с указанными параметрами.


		 ![](../../Sub_Image.gif)
		 [DeleteElementsData](IAutoCubeInstance.DeleteElementsData.htm)
		 Метод DeleteElementsData
		 осуществляет удаление данных из автоматического куба.


		 ![](../../Sub_Image.gif)
		 [UpdateFromSource](IAutoCubeInstance.UpdateFromSource.htm)
		 Метод UpdateFromSource
		 осуществляет обновление данных в автоматическом кубе из куба-источника,
		 в соответствии с установленными параметрами.
