# ICubeDimensionFixInfo.FindByKey


## Синтаксис


FindByKey(Key: Integer): [ICubeDimensionFix](../ICubeDimensionFix/ICubeDimensionFix.htm);


## Параметры


Key - ключ измерения.


## Описание


Метод FindByKey осуществляет поиск измерения, ключ которого передается посредством параметра Key, и возвращает параметры фиксации в случае удачного поиска.
