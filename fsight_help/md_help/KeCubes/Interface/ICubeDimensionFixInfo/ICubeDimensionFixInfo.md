# ICubeDimensionFixInfo


## Описание


Интерфейс ICubeDimensionFixInfo
 содержит свойства и методы для работы с фиксированными измерениями куба.


## Комментарии


Используется для фиксации измерений в виртуальном кубе и представлении-кубе.
