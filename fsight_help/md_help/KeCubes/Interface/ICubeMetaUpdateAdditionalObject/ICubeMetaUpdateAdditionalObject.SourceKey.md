# ICubeMetaUpdateAdditionalObject.SourceKey


## Синтаксис


SourceKey: Integer;


## Описание


Свойство SourceKey определяет ключ копируемого объекта-источника.


## Комментарии


По умолчанию значение данного свойства совпадает с ключом объекта репозитория, который должен быть скопирован вместе с базой данных временных рядов.
