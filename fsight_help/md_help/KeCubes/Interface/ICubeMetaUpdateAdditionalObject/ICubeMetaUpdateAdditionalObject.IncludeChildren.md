# ICubeMetaUpdateAdditionalObject.IncludeChildren


## Синтаксис


IncludeChildren: Boolean;


## Описание


Свойство IncludeChildren определяет признак копирования объекта со всеми его дочерними объектами.


## Комментарии


Если свойству установлено значение True, то объект будет копироваться со всеми дочерними объектами (Например, справочник будет копироваться со всеми схемами отметки, группами элементов и альтернативными иерархиями).
