# ICubeMetaUpdateAdditionalObject.SourceClassId


## Синтаксис


SourceClassId: Integer;


## Описание


Свойство SourceClassId определяет класс копируемого объекта-источника.


## Комментарии


Класс определяется с помощью перечисления [MetabaseObjectClass](kesom.chm::/enums/metabaseobjectclass.htm).
