# ICubeMetaUpdateAdditionalObject.SourceId


## Синтаксис


SourceId: String;


## Описание


Свойство SourceId определяет идентификатор копируемого объекта-источника.


## Комментарии


По умолчанию значение данного свойства совпадает с идентификатором объекта репозитория, который должен быть скопирован вместе с базой данных временных рядов.
