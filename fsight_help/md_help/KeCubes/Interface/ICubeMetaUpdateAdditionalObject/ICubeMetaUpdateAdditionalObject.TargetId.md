# ICubeMetaUpdateAdditionalObject.TargetId


## Синтаксис


TargetId: String;


## Описание


Свойство TargetId определяет идентификатор объекта-копии.


## Комментарии


Значение свойства игнорируется, если указан объект-приёмник, который будет заменён объектом-источником: [ICubeMetaUpdateAdditionalObject.Target](ICubeMetaUpdateAdditionalObject.Target.htm).
