# ICubeInstanceDimensions


## Описание


Интерфейс ICubeInstanceDimensions
 содержит свойства и методы для работы с данными измерений куба.


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeInstanceDimensions.Count.htm)
		 Свойство Count возвращает
		 количество измерений куба.


		 ![](../../Property_Image.gif)
		 [Item](ICubeInstanceDimensions.Item.htm)
		 Свойство Item возвращает
		 данные измерения.


## Методы


		  Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindByKey](ICubeInstanceDimensions.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск измерения по ключу.
