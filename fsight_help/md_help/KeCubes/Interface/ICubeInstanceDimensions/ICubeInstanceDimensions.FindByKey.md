# ICubeInstanceDimensions.FindByKey


## Синтаксис


FindByKey(DimensionKey: Integer): [IDimInstance](kedims.chm::/interface/idiminstance/idiminstance.htm);


## Параметры


DimensionKey - ключ измерения, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет поиск измерения, ключ которого передается посредством параметра DimensionKey, и возвращает данные измерения в случае удачного поиска.
