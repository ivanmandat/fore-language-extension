# IMDCalculationSources


## Описание


Интерфейс IMDCalculationSources
 содержит свойства и методы для работы с коллекцией источников данных многомерного
 расчета на сервере БД.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMDCalculationSources.Count.htm)
		 Свойство Count возвращает
		 количество источников данных многомерного расчета на сервере БД.


		 ![](../../Property_Image.gif)
		 [Item](IMDCalculationSources.Item.htm)
		 Свойство Item возвращает
		 параметры источника данных.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IMDCalculationSources.Add.htm)
		 Метод Add осуществляет
		 добавление реляционного источника данных в многомерный расчет
		 на сервере БД.


		 ![](../../Sub_Image.gif)
		 [AddCube](IMDCalculationSources.AddCube.htm)
		 Метод AddCube осуществляет
		 добавление куба в список источников данных многомерного расчета
		 на сервере БД.


		 ![](../../Sub_Image.gif)
		 [Clear](IMDCalculationSources.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех источников многомерного расчета на сервере БД.


		 ![](../../Sub_Image.gif)
		 [FindById](IMDCalculationSources.FindById.htm)
		 Метод FindById осуществляет
		 поиск источника данных по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IMDCalculationSources.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск источника данных по ключу.


		 ![](../../Sub_Image.gif)
		 [Move](IMDCalculationSources.Move.htm)
		 Метод Move осуществляет
		 перемещение источников данных в списке многомерного расчета на
		 сервере БД.


		 ![](../../Sub_Image.gif)
		 [Remove](IMDCalculationSources.Remove.htm)
		 Метод Remove осуществляет
		 удаление источника данных по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IMDCalculationSources.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление источника данных по ключу.
