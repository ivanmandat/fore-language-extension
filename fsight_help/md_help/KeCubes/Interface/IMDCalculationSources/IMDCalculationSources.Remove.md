# IMDCalculationSources.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс источника данных, который необходимо удалить.


## Описание


Метод Remove осуществляет удаление источника данных, индекс которого передается посредством параметра Index.
