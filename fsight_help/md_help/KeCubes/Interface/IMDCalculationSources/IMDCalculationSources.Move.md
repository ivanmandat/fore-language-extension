# IMDCalculationSources.Move


## Синтаксис


Move(IndexFrom: Integer; IndexTo: Integer);


## Параметры


IndexFrom - индекс позиции источника данных, который необходимо переместить.


IndexTo - индекс позиции, в которую необходимо переместить источник данных.


## Описание


Метод Move осуществляет перемещение источников данных в списке многомерного расчета на сервере БД.
