# IMDCalculationSources.Add


## Синтаксис


Add(Value: [IDatasetModel](KeDb.chm::/Interface/IDatasetModel/IDatasetModel.htm)):
 [IMDCalculationSource](../IMDCalculationSource/IMDCalculationSource.htm);


## Параметры


Value. Реляционный источник
 данных, который необходимо добавить.


## Описание


Метод Add осуществляет добавление
 реляционного источника данных в многомерный расчет на сервере БД.
