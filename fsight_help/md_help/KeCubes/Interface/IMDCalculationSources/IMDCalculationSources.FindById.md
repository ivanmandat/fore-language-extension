# IMDCalculationSources.FindById


## Синтаксис


FindById(Id: String): [IMDCalculationSource](../IMDCalculationSource/IMDCalculationSource.htm);


## Параметры


Id - идентификатор источника данных.


## Описание


Метод FindById осуществляет поиск и возвращает свойства источника данных. Идентификатор источника данных, по которому осуществляется поиск, передается посредством параметра Id.
