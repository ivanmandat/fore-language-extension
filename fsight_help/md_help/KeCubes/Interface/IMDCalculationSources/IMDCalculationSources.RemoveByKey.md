# IMDCalculationSources.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key - ключ источника данных, который необходимо удалить.


## Описание


Метод RemoveByKey осуществляет удаление источника данных, ключ которого передается посредством параметра Key.
