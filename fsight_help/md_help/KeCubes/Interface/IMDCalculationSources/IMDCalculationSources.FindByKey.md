# IMDCalculationSources.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IMDCalculationSource](../IMDCalculationSource/IMDCalculationSource.htm);


## Параметры


Key - ключ источника данных.


## Описание


Метод FindByKey осуществляет поиск и возвращает источник данных. Ключ источника данных, по которому осуществляется поиск, передается посредством параметра Key.
