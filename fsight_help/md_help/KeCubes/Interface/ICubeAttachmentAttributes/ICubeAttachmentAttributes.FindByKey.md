# ICubeAttachmentAttributes.FindByKey


## Синтаксис


		FindByKey(Key: Integer): [ICubeAttachmentAttribute](../ICubeAttachmentAttribute/ICubeAttachmentAttribute.htm);


## Параметры


Key. Ключ атрибута.


## Описание


Метод FindByKey осуществляет
 поиск по ключу и возвращает параметры атрибута в случае удачного поиска.


## Комментарии


Метод возвращает значение Null,
 если атрибут с указанным ключом не найден.
