# ICubeAttachmentAttributes


Сборка: Cubes;


## Описание


Интерфейс ICubeAttachmentAttributes
 предназначен для работы с коллекцией атрибутов справочника, в котором
 хранятся вложения куба.


## Иерархия наследования


ICubeAttachmentAttributes


## Комментарии


Коллекцию атрибутов можно получить в свойстве [ICubeAttachments.Attributes](../ICubeAttachments/ICubeAttachments.Attributes.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeAttachmentAttributes.Count.htm)


		 Свойство Count возвращает
		 количество атрибутов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ICubeAttachmentAttributes.Item.htm)


		 Свойство Index возвращает
		 параметры атрибута с указанными индексом.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](ICubeAttachmentAttributes.FindById.htm)


		 Метод FindById осуществляет
		 поиск по идентификатору и возвращает параметры атрибута в случае
		 удачного поиска.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ICubeAttachmentAttributes.FindByKey.htm)


		 Метод FindByKey осуществляет
		 поиск по ключу и возвращает параметры атрибута в случае удачного
		 поиска.


		 ![](../../Sub_Image.gif)
		 [IndexOf](ICubeAttachmentAttributes.IndexOf.htm)


		 Метод IndexOf возвращает
		 индекс указанного атрибута в коллекции.
