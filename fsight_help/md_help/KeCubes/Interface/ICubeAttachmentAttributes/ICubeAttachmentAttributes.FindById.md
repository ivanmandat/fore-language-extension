# ICubeAttachmentAttributes.FindById


## Синтаксис


		FindById(Id: String): [ICubeAttachmentAttribute](../ICubeAttachmentAttribute/ICubeAttachmentAttribute.htm);


## Параметры


Id. Идентификатор атрибута.


## Описание


Метод FindById осуществляет
 поиск по идентификатору и возвращает параметры атрибута в случае удачного
 поиска.


## Комментарии


Метод возвращает значение Null,
 если атрибут с указанным идентификатором не найден.
