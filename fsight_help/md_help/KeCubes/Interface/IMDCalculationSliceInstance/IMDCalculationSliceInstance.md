# IMDCalculationSliceInstance


## Описание


Интерфейс IMDCalculationSliceInstance
 содержит свойства измерения отстроенного источника/приёмника данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dataset](IMDCalculationSliceInstance.Dataset.htm)
		 Свойство Dataset возвращает
		 источник/приёмник данных.


		 ![](../../Property_Image.gif)
		 [Dimension](IMDCalculationSliceInstance.Dimension.htm)
		 Свойство Dimension
		 возвращает данные справочника, связанного с данным измерением
		 источника данных.


		 ![](../../Property_Image.gif)
		 [FieldCount](IMDCalculationSliceInstance.FieldCount.htm)
		 Свойство FieldCount
		 возвращает количество полей, связанных с данным измерением источника/приёмника
		 данных.


		 ![](../../Property_Image.gif)
		 [Index](IMDCalculationSliceInstance.Index.htm)
		 Свойство Index возвращает
		 объект, позволяющий работать с индексированными данными.


		 ![](../../Property_Image.gif)
		 [IsCalendar](IMDCalculationSliceInstance.IsCalendar.htm)
		 Свойство IsCalendar
		 возвращает признак того, что данноое измерение является календарным.
