# ICubeLoaderDimensionBindings.AddCalendar


## Синтаксис


AddCalendar(SourceField: String; CalendarLevel: [DimCalendarLevel](kedims.chm::/enums/dimcalendarlevel.htm)): [ICubeLoaderDimensionBinding](../ICubeLoaderDimensionBinding/ICubeLoaderDimensionBinding.htm);


## Параметры


SourceField. Наименование поля источника данных, к которому осуществляется привязка измерения;


CalendarLevel. Уровень календаря, для которого создается привязка.


## Описание


Метод AddCalendar добавляет в коллекцию параметры привязки календарного измерения.
