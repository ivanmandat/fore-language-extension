# ICubeLoaderDimensionBindings.Add


## Синтаксис


Add(SourceField: String): [ICubeLoaderDimensionBinding](../ICubeLoaderDimensionBinding/ICubeLoaderDimensionBinding.htm);


## Параметры


SourceField. Наименование поля источника данных, к которому осуществляется привязка измерения.


## Описание


Метод Add добавляет в коллекцию параметры привязки измерения.
