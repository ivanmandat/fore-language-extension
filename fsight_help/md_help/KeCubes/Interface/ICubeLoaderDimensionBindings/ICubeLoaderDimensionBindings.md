# ICubeLoaderDimensionBindings


Сборка: Cubes;


## Описание


Интерфейс ICubeLoaderDimensionBindings
 содержит свойства и методы для работы с коллекцией привязок измерений,
 настраиваемых при загрузке данных.


## Иерархия наследования


ICubeLoaderDimensionBindings


## Комментарии


Привязки измерений можно получить в свойстве [ICubeLoaderFromSource.DimensionBindings](../ICubeLoaderFromSource/ICubeLoaderFromSource.DimensionBindings.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeLoaderDimensionBindings.Count.htm)


		 Свойство Count возвращает
		 количество параметров привязки измерений в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ICubeLoaderDimensionBindings.Item.htm)


		 Свойство Item возвращает
		 указанные параметры привязки измерения.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ICubeLoaderDimensionBindings.Add.htm)


		 Метод Add добавляет
		 в коллекцию параметры привязки измерения.


		 ![](../../Sub_Image.gif)
		 [AddCalendar](ICubeLoaderDimensionBindings.AddCalendar.htm)


		 Метод AddCalendar добавляет
		 в коллекцию параметры привязки календарного измерения.


		 ![](../../Sub_Image.gif)
		 [Clear](ICubeLoaderDimensionBindings.Clear.htm)


		 Метод Clear удаляет
		 из коллекции все параметры привязки.


		 ![](../../Sub_Image.gif)
		 [Remove](ICubeLoaderDimensionBindings.Remove.htm)


		 Метод Remove удаляет
		 указанные параметры привязки.
