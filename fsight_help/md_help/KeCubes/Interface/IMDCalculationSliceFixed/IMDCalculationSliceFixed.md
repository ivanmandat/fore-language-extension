# IMDCalculationSliceFixed


## Описание


Интерфейс IMDCalculationSliceFixed
 содержит свойства зафиксированного измерения источника/приёмника данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Mapping](IMDCalculationSliceFixed.Mapping.htm)
		 Свойство Mapping определяет
		 измерение приёмника данных, который ставится в соответствие фиксируемому
		 измерению источника данных.


		 ![](../../Property_Image.gif)
		 [Slice](IMDCalculationSliceFixed.Slice.htm)
		 Свойство Slice возвращает
		 измерение источника/приёмника данных.
