# ICubeMetaUpdateDictionaryRemapings.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс соответствия
 в коллекции.


## Описание


Метод Remove осуществляет удаление
 соответствия из коллекции по его индексу.


## Комментарии


Метод возвращает значение True,
 если удаление прошло удачно.
