# ICubeMetaUpdateDictionaryRemapings


## Описание


Интерфейс ICubeMetaUpdateDictionaryRemapings
 содержит свойства и методы для работы с коллекцией соответствий объектов
 копируемой базы данных временных рядов и объектов базы-копии.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeMetaUpdateDictionaryRemapings.Count.htm)
		 Свойство Count возвращает
		 количество соответствий в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ICubeMetaUpdateDictionaryRemapings.Item.htm)
		 Свойство Item возвращает
		 соответствие из коллекции по его индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ICubeMetaUpdateDictionaryRemapings.Add.htm)
		 Метод Add добавляет
		 новое соответствие в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](ICubeMetaUpdateDictionaryRemapings.Clear.htm)
		 Метод Clear очищает
		 коллекцию.


		 ![](../../Sub_Image.gif)
		 [FindById](ICubeMetaUpdateDictionaryRemapings.FindById.htm)
		 Метод FindById осуществляет
		 поиск соответствия в коллекции по идентификатору объекта-источника.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ICubeMetaUpdateDictionaryRemapings.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск соответствия в коллекции по ключу объекта-источника.


		 ![](../../Sub_Image.gif)
		 [Remove](ICubeMetaUpdateDictionaryRemapings.Remove.htm)
		 Метод Remove осуществляет
		 удаление соответствия из коллекции по его индексу.
