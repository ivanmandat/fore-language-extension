# IVirtualCubeDimensionJoins.RemoveById


## Синтаксис


RemoveById(Id: String): Boolean;


## Параметры


Id. Идентификатор зависимого
 измерения.


## Описание


Метод RemoveById удаляет сопоставление
 с кубом-источником по его идентификатору.


## Комментарии


Метод возвращает значение True,
 если удаление сопоставление прошло удачно и False,
 если удаление по каким-либо причинам невозможно.
