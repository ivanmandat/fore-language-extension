# IVirtualCubeDimensionJoins.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс куба-источника,
 с измерением которого создано сопоставление.


## Описание


Метод Remove удаляет сопоставление
 с кубом-источником по его индексу в коллекции.


## Комментарии


Метод возвращает значение True,
 если удаление сопоставление прошло удачно и False,
 если удаление по каким-либо причинам невозможно.
