# IVirtualCubeDimensionJoins.FindById


## Синтаксис


FindById(Id: String): [IVirtualCubeDimensionJoin](../IVirtualCubeDimensionJoin/IVirtualCubeDimensionJoin.htm);


## Параметры


Id. Идентификатор куба-источника.


## Описание


Метод FindById осуществляет
 поиск по идентификатору куба-источника и возвращает параметры сопоставления
 с его измерением.


## Комментарии


Метод возвращает параметры сопоставления с указанным кубом-источником
 в случае удачного поиска и Null,
 если для указанного куба сопоставление не создано.
