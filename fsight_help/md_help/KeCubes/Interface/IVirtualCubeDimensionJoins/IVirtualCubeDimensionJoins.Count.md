# IVirtualCubeDimensionJoins.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 измерений, с которыми сопоставлено измерение куба-источника.


## Комментарии


Для сопоставления с новым измерением какого-либо куба-источника используйте
 метод [IVirtualCubeDimensionJoins.Add](IVirtualCubeDimensionJoins.Add.htm).
