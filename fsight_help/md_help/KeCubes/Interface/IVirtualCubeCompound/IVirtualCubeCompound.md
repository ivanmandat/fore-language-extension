# IVirtualCubeCompound


## Описание


Интерфейс IVirtualCubeCompound
 предназначен для работы с составной структурой виртуального куба.


## Комментарии


Для получения всех составных структур виртуального куба используйте
 свойство [IVirtualCube.Compounds](../IVirtualCube/IVirtualCube.Compounds.htm).
