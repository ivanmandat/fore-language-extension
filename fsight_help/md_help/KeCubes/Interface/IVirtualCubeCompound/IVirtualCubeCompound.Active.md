# IVirtualCubeCompound.Active


## Синтаксис


Active: Boolean;


## Описание


Свойство Active возвращает признак
 активности структуры.


## Комментарии


Active =
 True. Предопределенная структура активна.


Active =
 False. Предопределенная структура не активна.


Предопределенная структура фактов активна всегда.


Предопределенный структура календаря/сценария активируется при наличии
 в источниках виртуального куба разных календарей/сценариев.


Остальные структуры активируются при фиксации измерения источника в
 этой структуре.
