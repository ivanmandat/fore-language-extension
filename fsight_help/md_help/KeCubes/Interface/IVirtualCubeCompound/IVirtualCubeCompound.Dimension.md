# IVirtualCubeCompound.Dimension


## Синтаксис


Dimension: [IDimensionModel](KeDims.chm::/interface/IDimensionModel/IDimensionModel.htm);


## Описание


Свойство Dimension возвращает
 составной справочник, соответствующий структуре.


## Комментарии


Dimension может возвращать значение
 Null, если структура не
 активна.
