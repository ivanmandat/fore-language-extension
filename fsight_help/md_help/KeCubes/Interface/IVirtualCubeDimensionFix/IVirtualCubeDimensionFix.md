# IVirtualCubeDimensionFix


## Описание


Интерфейс IVirtualCubeDimensionFix
 предназначен для работы с фиксированным составным измерением виртуального
 куба.


## Иерархия


           IVirtualCubeDimensionFix


## Комментарии


Для работы с параметрами фиксации измерений в виртуальном кубе используйте
 интерфейс [ICubeDimensionFixInfo](../ICubeDimensionFixInfo/ICubeDimensionFixInfo.htm).
