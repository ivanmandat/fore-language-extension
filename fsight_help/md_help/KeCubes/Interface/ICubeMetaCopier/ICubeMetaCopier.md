# ICubeMetaCopier


## Описание


Интерфейс ICubeMetaCopier содержит
 свойства и методы для копирования показателей из базы данных временных
 рядов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Bindings](ICubeMetaCopier.Bindings.htm)


		 Свойство Bindings определяет
		 привязку атрибутов базы-источника с атрибутами базы-приёмника.


		 ![](../../Property_Image.gif)
		 [CalendarLevelSet](ICubeMetaCopier.CalendarLevelSet.htm)


		 Свойство CalendarLevel
		 определяет уровень детализации данных.


		 ![](../../Property_Image.gif)
		 [DestinationRubricator](ICubeMetaCopier.DestinationRubricator.htm)


		 Свойство DestinationRubricator
		 определяет базу-приёмник.


		 ![](../../Property_Image.gif)
		 [DestinationRubricatorInstance](ICubeMetaCopier.DestinationRubricatorInstance.htm)


		 Свойство DestinationRubricatorInstance
		 определяет данные базы-приёмника.


		 ![](../../Property_Image.gif)
		 [ImportObjectKey](ICubeMetaCopier.ImportObjectKey.htm)


		 Свойство ImportObjectKey
		 определяет ключ объекта, выполняющего копирование.


		 ![](../../Property_Image.gif)
		 [Log](ICubeMetaCopier.Log.htm)


		 Свойство Log возвращает
		 отчет о копировании показателей.


		 ![](../../Property_Image.gif)
		 [NewRevisionName](ICubeMetaCopier.NewRevisionName.htm)


		 Свойство NewRevisionName
		 определяет имя создаваемой ревизии.


		 ![](../../Property_Image.gif)
		 [SourceRubricator](ICubeMetaCopier.SourceRubricator.htm)


		 Свойство SourceRubricator
		 определяет базу-источник.


		 ![](../../Property_Image.gif)
		 [SourceRubricatorInstance](ICubeMetaCopier.SourceRubricatorInstance.htm)


		 Свойство SourceRubricatorInstance
		 определяет данные базы-источника.
