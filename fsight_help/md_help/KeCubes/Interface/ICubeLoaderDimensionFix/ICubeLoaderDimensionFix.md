# ICubeLoaderDimensionFix


## Описание


Интерфейс ICubeLoaderDimensionFix
 содержит свойства, определяющие параметры фиксации измерения куба-источника
 в загрузчике данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dimension](ICubeLoaderDimensionFix.Dimension.htm)
		 Свойство Dimension
		 возвращает структуру справочника, лежащего в основе измерения.


		 ![](../../Property_Image.gif)
		 [Instance](ICubeLoaderDimensionFix.Instance.htm)
		 Свойство Instance возвращает
		 данные измерения.


		 ![](../../Property_Image.gif)
		 [Operation](ICubeLoaderDimensionFix.Operation.htm)
		 Свойство Operation
		 определяет метод агрегации данных измерения.


		 ![](../../Property_Image.gif)
		 [Selection](ICubeLoaderDimensionFix.Selection.htm)
		 Свойство Selection
		 возвращает отметку измерения.
