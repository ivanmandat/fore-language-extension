# ICubeExecutorDrillThroughItems


Сборка: Cubes;


## Описание


Интерфейс ICubeExecutorDrillThroughItems
 используется для работы с коллекцией элементов детализации при расчете
 куба.


## Иерархия наследования


ICubeExecutorDrillThroughItems


## Комментарии


Для получения объекта данного типа используйте [ICubeInstanceDestinationExecutor.DrillThrough](../ICubeInstanceDestinationExecutor/ICubeInstanceDestinationExecutor.DrillThrough.htm).
