# IAutoCubeUpdateFilter


## Описание


Интерфейс IAutoCubeUpdateFilter содержит свойства автоматического куба как объекта файла обновления.


## Свойства


 Имя свойства
 Краткое описание


 ![](../../Property_Image.gif)

[ParamsValues](IAutoCubeUpdateFilter.ParamsValues.htm)


 Свойство ParamsValues определяет значения параметров автоматического куба.


 ![](../../Property_Image.gif)

[SelectionSet](IAutoCubeUpdateFilter.SelectionSet.htm)


 Свойство SelectionSet определяет отметку по измерениям автоматического куба.
