# ICubeLoaderResult.SkippedStrings


## Синтаксис


SkippedStrings: [IArrayList](ModCollections.chm::/Interface/IArrayList/IArrayList.htm);


## Описание


Свойство SkippedStrings возвращает
 массив строк, которые были пропущены при загрузке данных.


## Комментарии


Строки при загрузке могут быть пропущены, если для какого-либо измерения
 установлен [признак
 пропуска отсутствующих элементов](../ICubeLoaderDimensionBinding/ICubeLoaderDimensionBinding.SkipLoad.htm) и загрузка прошла только по тем элементам,
 которые уже существуют в измерении.
