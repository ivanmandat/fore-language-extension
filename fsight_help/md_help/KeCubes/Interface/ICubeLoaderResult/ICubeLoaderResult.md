# ICubeLoaderResult


Сборка: Cubes;


## Описание


Интерфейс ICubeLoaderResult
 предназначен для работы с результатами загрузки данных.


## Иерархия наследования


           ICubeLoaderResult


## Комментарии


Для получения результатов после загрузки данных используйте метод [ICubeLoaderFromSource.LoadWithResult](../ICubeLoaderFromSource/ICubeLoaderFromSource.LoadWithResult.htm).
