# ICubeCreatorFact.Dimensions


## Синтаксис


Dimensions: [ICubeCreatorDimensions](../ICubeCreatorDimensions/ICubeCreatorDimensions.htm);


## Описание


Свойство Dimensions возвращает коллекцию измерений, используемую для настройки привязки атрибутов к полям таблицы, которая указана в свойстве [Table](ICubeCreatorFact.Table.htm).


## Комментарии


Свойство актуально, есть для факта в свойстве [Table](ICubeCreatorFact.Table.htm) задана отдельная таблица. Свойство Dimensions будет доступно после вызова метода [InitDimensions](ICubeCreatorFact.InitDimensions.htm).
