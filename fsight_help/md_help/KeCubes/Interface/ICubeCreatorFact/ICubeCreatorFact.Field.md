# ICubeCreatorFact.Field


## Синтаксис


Field: String;


## Описание


Свойство Field определяет идентификатор
 поля таблицы, в котором будут храниться значения факта.


## Комментарии


Таблица задается в свойстве [ICubeCreator.Table](../ICubeCreator/ICubeCreator.Table.htm)
 или [ICubeCreatorFact.Table](ICubeCreatorFact.Table.htm). В
 свойстве Field укажите идентификатор
 поля, тип данных которого соответствует типу данных, хранящихся в кубе
 по текущему факту. Привязка к полям двоичного типа не поддерживается.
