# ICubeCreatorFact.InitDimensions


## Синтаксис


InitDimensions;


## Описание


Метод InitDimensions подготавливает
 измерения для привязки к полям таблицы, которая используется для хранения
 значений факта.


## Комментарии


Метод InitDimensions используется
 если для факта задано свойство [Table](ICubeCreatorFact.Table.htm).
 После вызова метода в свойстве [Dimensions](ICubeCreatorFact.Dimensions.htm)
 можно получить коллекцию измерений и настроить их привязку к полям установленной
 таблицы.
