# ICubeCreatorFact


Сборка: Cubes;


## Описание


Интерфейс ICubeCreatorFact содержит
 свойства и методы для настройки факта, который будет создан в кубе.


## Иерархия наследования


ICubeCreatorFact


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AggregationType](ICubeCreatorFact.AggregationType.htm)
		 Свойство AggregationType
		 определяет тип агрегации на источнике данных.


		 ![](../../Property_Image.gif)
		 [Dimensions](ICubeCreatorFact.Dimensions.htm)
		 Свойство Dimensions
		 возвращает коллекцию измерений, используемую для настройки привязки
		 атрибутов к полям таблицы, которая указана в свойстве [Table](ICubeCreatorFact.Table.htm).


		 ![](../../Property_Image.gif)
		 [FactKey](ICubeCreatorFact.FactKey.htm)
		 Свойство FactKey определяет
		 ключ существующего факта, для которого настраивается привязка.


		 ![](../../Property_Image.gif)
		 [Field](ICubeCreatorFact.Field.htm)
		 Свойство Field определяет
		 идентификатор поля таблицы, в котором будут храниться значения
		 факта.


		 ![](../../Property_Image.gif)
		 [Name](ICubeCreatorFact.Name.htm)
		 Свойство Name определяет
		 наименование факта.


		 ![](../../Property_Image.gif)
		 [Table](ICubeCreatorFact.Table.htm)
		 Свойство Table определяет
		 таблицу репозитория, к которой необходимо привязать факт.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [InitDimensions](ICubeCreatorFact.InitDimensions.htm)
		 Метод InitDimensions
		 подготавливает измерения для привязки к полям таблицы, которая
		 используется для хранения значений факта.
