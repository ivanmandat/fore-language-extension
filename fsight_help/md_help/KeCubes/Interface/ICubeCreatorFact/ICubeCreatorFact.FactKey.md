# ICubeCreatorFact.FactKey


## Синтаксис


FactKey: Variant;


## Описание


Свойство FactKey определяет
 ключ существующего факта, для которого настраивается привязка.


## Комментарии


Свойство используется, если необходимо изменить структуру существующего
 куба, в который осуществляется загрузка данных.
