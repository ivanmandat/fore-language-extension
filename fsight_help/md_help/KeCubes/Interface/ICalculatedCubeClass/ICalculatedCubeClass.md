# ICalculatedCubeClass


## Описание


Интерфейс ICalculatedCubeClass
 содержит статические свойства и методы класса [CalculatedCube](../../Class/CalculatedCube/CalculatedCube.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Current](ICalculatedCubeClass.Current.htm)


		 Свойство Current возвращает
		 текущий вычисляемый куб.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Shift](ICalculatedCubeClass.Shift.htm)


		 Метод Shift осуществляет
		 смещение координаты по календарному измерению с заданным лагом.


		 ![](../../Sub_Image.gif)
		 [ShiftVal](ICalculatedCubeClass.ShiftVal.htm)


		 Метод ShiftVal возвращает
		 значение по координате, смещенной относительно данной координаты
		 по календарному измерению с заданным лагом.


		 ![](../../Sub_Image.gif)
		 [Val](ICalculatedCubeClass.Val.htm)


		 Метод Val возвращает
		 значение по указанной координате.
