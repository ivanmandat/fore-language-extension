# IImportRequestRubricatorParams


## Описание


Интерфейс IImportRequestRubricatorParams
 содержит свойства для работы с параметрами импорта показателей из базы
 данных временных рядов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Bindings](IImportRequestRubricatorParams.Bindings.htm)
		 Свойство Bindings определяет
		 привязку атрибутов базы данных временных рядов с полями источника
		 данных.


		 ![](../../Property_Image.gif)
		 [CalendarLevelSet](IImportRequestRubricatorParams.CalendarLevelSet.htm)
		 Свойство CalendarLevelSet
		 определяет уровень детализации данных при импорте.


		 ![](../../Property_Image.gif)
		 [ImportObjectKey](IImportRequestRubricatorParams.ImportObjectKey.htm)
		 Свойство ImportObjectKey
		 определяет ключ объекта, выполняющего импорт.


		 ![](../../Property_Image.gif)
		 [NewRevisionName](IImportRequestRubricatorParams.NewRevisionName.htm)
		 Свойство NewRevisionName
		 определяет имя создаваемой ревизии.


		 ![](../../Property_Image.gif)
		 [SourceRubricator](IImportRequestRubricatorParams.SourceRubricator.htm)
		 Свойство SourceRubricator
		 определяет импортируемую базу данных временных рядов.


		 ![](../../Property_Image.gif)
		 [SourceRubricatorInstance](IImportRequestRubricatorParams.SourceRubricatorInstance.htm)
		 Свойство SourceRubricatorInstance
		 определяет данные импортируемой базой данных временных рядов.
