# ICubeLoaderFactBinding.FactKey


## Синтаксис


FactKey: Variant;


## Описание


Свойство FactKey определяет
 ключ факта, по которому определяется привязка.


## Комментарии


При создании нового куба свойство должно быть пустым.


ICubeLoaderFactBinding.FactKey
 соответствует свойству [IStandardCubeFactBinding.FactKey](../IStandardCubeFactBinding/IStandardCubeFactBinding.FactKey.htm).
