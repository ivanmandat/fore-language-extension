# ICubeLoaderFactBinding


Сборка: Cubes;


## Описание


Интерфейс ICubeLoaderFactBinding
 содержит свойства для определения параметров привязки факта, которые используются
 при загрузке данных.


## Иерархия наследования


           ICubeLoaderFactBinding


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Converter](ICubeLoaderFactBinding.Converter.htm)


		 Свойство Converter
		 определяет объект, который будет использоваться для преобразования
		 значений фактов.


		 ![](../../Property_Image.gif)
		 [CubeFactAggregationType](ICubeLoaderFactBinding.CubeFactAggregationType.htm)


		 Свойство CubeFactAggregationType
		 определяет тип агрегации, который будет применяться при загрузке
		 данных.


		 ![](../../Property_Image.gif)
		 [FactKey](ICubeLoaderFactBinding.FactKey.htm)


		 Свойство FactKey определяет
		 ключ факта, по которому определяется привязка.


		 ![](../../Property_Image.gif)
		 [MetaAttribute](ICubeLoaderFactBinding.MetaAttribute.htm)


		 Свойство MetaAttribute
		 возвращает атрибуты базы данных временных рядов уровня фактов
		 или наблюдений.


		 ![](../../Property_Image.gif)
		 [NewElementName](ICubeLoaderFactBinding.NewElementName.htm)


		 Свойство NewElementName
		 определяет наименование факта, который будет создан в измерении
		 фактов.


		 ![](../../Property_Image.gif)
		 [SourceField](ICubeLoaderFactBinding.SourceField.htm)


		 Свойство SourceField
		 возвращает наименование поля источника данных, с которым будет
		 связан факт.
