# ICubeLoaderFactBinding.NewElementName


## Синтаксис


NewElementName: String;


## Описание


Свойство NewElementName определяет
 наименование факта, который будет создан в измерении фактов.


## Комментарии


По умолчанию свойство NewElementName
 не определено, при этом создаваемый факт будет иметь такое же наименование,
 как и поле, с которым он связан.
