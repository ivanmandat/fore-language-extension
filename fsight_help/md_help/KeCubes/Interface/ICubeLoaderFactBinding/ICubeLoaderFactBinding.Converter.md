# ICubeLoaderFactBinding.Converter


## Синтаксис


Converter: [IValueConverter](../IValueConverter/IValueConverter.htm);


## Описание


Свойство Converter определяет
 объект, который будет использоваться для преобразования значений фактов.


## Комментарии


Объект, выполняющий преобразование, создаётся пользователем в виде пользовательского
 класса, являющегося потомком интерфейса [IValueConverter](../IValueConverter/IValueConverter.htm).
 Пример реализации преобразователя приведен в описании свойства [IValueConverter.Convert](../IValueConverter/IValueConverter.Convert.htm).
