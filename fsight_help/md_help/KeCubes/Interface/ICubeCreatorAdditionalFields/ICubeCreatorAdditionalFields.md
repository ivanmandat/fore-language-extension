# ICubeCreatorAdditionalFields


Сборка: Cubes;


## Описание


Интерфейс ICubeCreatorAdditionalFields
 содержит свойства и методы для работы с коллекцией полей, которые будут
 присутствовать в результирующей таблице куба, но не будут привязаны к
 измерениям/фактам куба.


## Иерархия наследования


ICubeCreatorAdditionalFields


## Комментарии


Параметры коллекции можно получить в свойстве [ICubeCreator.AdditionalFields](../ICubeCreator/ICubeCreator.AdditionalFields.htm).
