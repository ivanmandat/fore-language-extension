# ICubeExecuteDataRights


## Описание


Интерфейс ICubeExecuteDataRights
 содержит свойства для работы с правами доступа на данные.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CoordMatrix](ICubeExecuteDataRights.CoordMatrix.htm)
		 Свойство CoordMatrix
		 возвращает пустую матрицу для соответствия измерений в координате
		 точки данных.


		 ![](../../Property_Image.gif)
		 [Rights](ICubeExecuteDataRights.Rights.htm)
		 Свойство Rights возвращает
		 фактические права доступа на точку данных.


		 ![](../../Property_Image.gif)
		 [SupportedDataRights](ICubeExecuteDataRights.SupportedDataRights.htm)
		 Свойство SupportedDataRights
		 возвращает запрошенные права доступа на данные.
