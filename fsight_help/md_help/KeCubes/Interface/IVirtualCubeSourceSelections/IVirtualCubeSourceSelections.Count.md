# IVirtualCubeSourceSelections.Count


## Синтаксис


		Count: Integer;


## Описание


Свойство Count возвращает количество
 отметок в коллекции.


## Комментарии


Количество отметок соответствует количеству источников в виртуальном
 кубе.
