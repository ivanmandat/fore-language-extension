# IVirtualCubeSourceSelections


Сборка: Cubes;


## Описание


Интерфейс IVirtualCubeSourceSelections
 предназначен для работы с коллекцией отметок источников виртуального куба.


## Иерархия наследования


IVirtualCubeSourceSelections


## Комментарии


Отметки источников виртуального куба используются в событии [IVirtualCubeExecuteCallback.OnBeforeExecuteSources](../IVirtualCubeExecuteCallback/IVirtualCubeExecuteCallback.OnBeforeExecuteSources.htm).
