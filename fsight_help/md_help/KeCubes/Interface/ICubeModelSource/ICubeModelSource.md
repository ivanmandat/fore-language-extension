# ICubeModelSource


## Описание


Интерфейс ICubeModelSource содержит
 свойства куба-источника данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Cube](ICubeModelSource.Cube.htm)
		 Свойство Cube возвращает
		 структуру куба, являющегося источником данных для данного куба.


		 ![](../../Property_Image.gif)
		 [Source](ICubeModelSource.Source.htm)
		 Свойство Source возвращает
		 вариант отображения куба, являющегося источником данных для данного
		 куба.
