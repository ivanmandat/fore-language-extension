# IRubricatorSegmentsSet


## Описание


Интерфейс IRubricatorSegmentsSet
 содержит общие свойства и методы для работы с коллекцией сегментов базы
 данных временных рядов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IRubricatorSegmentsSet.Count.htm)


		 Свойство Count возвращает
		 количество сегментов данных в базе данных временных рядов.


		 ![](../../Property_Image.gif)
		 [Item](IRubricatorSegmentsSet.Item.htm)


		 Свойство Item возвращает
		 указанный сегмент данных из коллекции.


		 ![](../../Property_Image.gif)
		 [Rubricator](IRubricatorSegmentsSet.Rubricator.htm)


		 Свойство Rubricator
		 возвращает данные базы данных временных рядов, содержащей коллекцию
		 сегментов данных.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Clear](IRubricatorSegmentsSet.Clear.htm)


		 Метод Clear осуществляет
		 очистку коллекции сегментов данных.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IRubricatorSegmentsSet.FindByKey.htm)


		 Метод FindByKey осуществляет
		 поиск сегмента в коллекции по ключу.


		 ![](../../Sub_Image.gif)
		 [GetAllowed](IRubricatorSegmentsSet.GetAllowed.htm)


		 Метод GetAllowed позволяет
		 получить отметку элементов измерения, доступных для совершения
		 указанных действий текущим пользователем.


		 ![](../../Sub_Image.gif)
		 [GetEffectiveRights](IRubricatorSegmentsSet.GetEffectiveRights.htm)


		 Метод GetEffectiveRights
		 возвращает эффективные права текущего пользователя на коллекцию
		 сегментов данных.


		 ![](../../Sub_Image.gif)
		 [GetFactSegments](IRubricatorSegmentsSet.GetFactSegments.htm)


		 Метод GetFactSegments
		 возвращает количество диапазонов данных, в которые входит указанный
		 показатель.


		 ![](../../Sub_Image.gif)
		 [GetSubjectAllowed](IRubricatorSegmentsSet.GetSubjectAllowed.htm)


		 Метод GetSubjectAllowed
		 позволяет получить отметку элементов измерения, доступных для
		 совершения указанных действий указанным пользователем.


		 ![](../../Sub_Image.gif)
		 [GetSubjectEffectiveRights](IRubricatorSegmentsSet.GetSubjectEffectiveRights.htm)


		 Метод GetSubjectEffectiveRights
		 возвращает эффективные права указанного пользователя на коллекцию
		 сегментов данных.


		 ![](../../Sub_Image.gif)
		 [GetTimeSegments](IRubricatorSegmentsSet.GetTimeSegments.htm)


		 Метод GetTimeSegments
		 возвращает количество временных диапазонов данных, которые попадают
		 в указанный диапазон.


		 ![](../../Sub_Image.gif)
		 [Remove](IRubricatorSegmentsSet.Remove.htm)


		 Метод Remove удаляет
		 указанный сегмент из коллекции.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IRubricatorSegmentsSet.RemoveByKey.htm)


		 Метод RemoveByKey удаляет
		 сегмент из коллекции по ключу.
