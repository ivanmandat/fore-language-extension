# ICubeCalendarOptions


## Описание


Интерфейс ICubeCalendarOptions
 содержит свойства для работы с динамикой календаря.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [FirstDayOfWeek](ICubeCalendarOptions.FirstDayOfWeek.htm)
		 Свойство FirstDayOfWeek
		 определяет первый день недели.


		 ![](../../Property_Image.gif)
		 [Hierarchy](ICubeCalendarOptions.Hierarchy.htm)
		 Свойство Hierarchy
		 определяет тип иерархии календаря.


		 ![](../../Property_Image.gif)
		 [Levels](ICubeCalendarOptions.Levels.htm)
		 Свойство Levels определяет
		 тип динамики календаря.


		 ![](../../Property_Image.gif)
		 [RestDays](ICubeCalendarOptions.RestDays.htm)
		 Свойство RestDays определяет
		 выходные дни недели.
