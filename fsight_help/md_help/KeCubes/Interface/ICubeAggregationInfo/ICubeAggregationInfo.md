# ICubeAggregationInfo


## Описание


Интерфейс ICubeAggregationInfo содержит свойства для получения информации о настройках агрегации данных в кубе.


## Свойства


 Имя свойства
 Краткое описание


 ![](../../Property_Image.gif)
 [TotalsDefined](ICubeAggregationInfo.TotalsDefined.htm)
 Свойство TotalsDefined возвращает признак наличия настроек для агрегации данных на сервере.
