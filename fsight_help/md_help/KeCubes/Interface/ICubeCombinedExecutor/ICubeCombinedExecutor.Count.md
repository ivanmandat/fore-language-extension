# ICubeCombinedExecutor.Count


## Синтаксис


		Count: Integer;


## Описание


Свойство Count возвращает количество
 доступных результатов параллельного расчёта кубов.


## Комментарии


Результаты расчёта будут доступны после завершения работы метода [Execute](ICubeCombinedExecutor.Execute.htm). Количество доступных
 результатов зависит от количества добавленных кубов/вариантов отображения
 куба.


## Пример


Примеры использования приведёны в описании методов [ICubeCombinedExecutor.AddCube](ICubeCombinedExecutor.AddCube.htm),
 [ICubeCombinedExecutor.AddCubeDest](ICubeCombinedExecutor.AddCubeDest.htm).
