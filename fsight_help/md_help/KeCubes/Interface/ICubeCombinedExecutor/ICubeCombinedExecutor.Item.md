# ICubeCombinedExecutor.Item


## Синтаксис


		Item(Index: Integer): [ICubeCombinedExecutorItem](../ICubeCombinedExecutorItem/ICubeCombinedExecutorItem.htm);


## Параметры


Index. Индекс результата расчёта.


## Описание


Свойство Item возвращает результат
 расчёта с указанным индексом.


## Комментарии


Результаты расчёта будут доступны после завершения работы метода [Execute](ICubeCombinedExecutor.Execute.htm).

Примеры использования приведёны в описании методов [ICubeCombinedExecutor.AddCube](ICubeCombinedExecutor.AddCube.htm),
 [ICubeCombinedExecutor.AddCubeDest](ICubeCombinedExecutor.AddCubeDest.htm).
