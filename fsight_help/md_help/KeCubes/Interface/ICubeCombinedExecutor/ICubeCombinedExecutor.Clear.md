# ICubeCombinedExecutor.Clear


## Синтаксис


		Clear;


## Описание


Метод Clear очищает список кубов,
 для которых может быть произведён параллельный расчёт.


## Комментарии


Для формирования списка кубов используйте методы [ICubeCombinedExecutor.AddCube](ICubeCombinedExecutor.AddCube.htm),
 [ICubeCombinedExecutor.AddCubeDest](ICubeCombinedExecutor.AddCubeDest.htm).
