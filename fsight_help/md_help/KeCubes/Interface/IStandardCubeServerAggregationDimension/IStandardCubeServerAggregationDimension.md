# IStandardCubeServerAggregationDimension


## Описание


Интерфейс IStandardCubeServerAggregationDimension
 содержит свойства и методы для работы с измерением, по которому на сервере
 рассчитывается агрегация.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dimension](IStandardCubeServerAggregationDimension.Dimension.htm)
		 Свойство Dimension
		 возвращает измерение, по которому рассчитывается агрегация.


		 ![](../../Property_Image.gif)
		 [GroupIndex](IStandardCubeServerAggregationDimension.GroupIndex.htm)
		 Свойство GroupIndex
		 определяет индекс группы, в которую входит измерение.


		 ![](../../Property_Image.gif)
		 [Others](IStandardCubeServerAggregationDimension.Others.htm)
		 Свойство Others определяет,
		 рассчитывать ли агрегацию для элементов измерения, не вошедших
		 в отметку (агрегат Other).


		 ![](../../Property_Image.gif)
		 [Position](IStandardCubeServerAggregationDimension.Position.htm)
		 Свойство Position определяет
		 позицию измерения в группе.


		 ![](../../Property_Image.gif)
		 [Totals](IStandardCubeServerAggregationDimension.Totals.htm)
		 Свойство Totals определяет,
		 рассчитывать ли итоги для измерения.
