# IStandardCubeServerAggregationDimension.Totals


## Синтаксис


Totals: Boolean;


## Описание


Свойство Totals определяет,
 рассчитывать ли итоги для измерения.


## Комментарии


Допустимые значения:


	- True. Рассчитывать итоги;


	- False. Не рассчитывать
	 итоги.
