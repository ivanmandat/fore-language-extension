# IAutoCubeSourceFixInfo


## Описание


Интерфейс IAutoCubeSourceFixInfo
 содержит свойства и методы для работы с коллекцией фиксированных измерений
 куба-источника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IAutoCubeSourceFixInfo.Count.htm)
		 Свойство Count возвращает
		 количество фиксированных измерений.


		 ![](../../Property_Image.gif)
		 [Item](IAutoCubeSourceFixInfo.Item.htm)
		 Свойство Item возвращает
		 фиксированное измерение.


		 ![](../../Property_Image.gif)
		 [ItemByDim](IAutoCubeSourceFixInfo.ItemByDim.htm)
		 Свойство ItemByDim
		 возвращает фиксированное измерение автоматического куба, которое
		 задается параметром Dimension.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindByKey](IAutoCubeSourceFixInfo.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск фиксированного измерения по ключу.
