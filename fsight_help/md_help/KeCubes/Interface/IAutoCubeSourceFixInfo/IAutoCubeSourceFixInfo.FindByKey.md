# IAutoCubeSourceFixInfo.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IAutoCubeSourceDimensionFix](../IAutoCubeSourceDimensionFix/IAutoCubeSourceDimensionFix.htm);


## Параметры


Key - ключ измерения, которое необходимо найти.


## Описание


Метод FindByKey осуществляет поиск фиксированного измерения и возвращает его параметры в случае удачного поиска.
