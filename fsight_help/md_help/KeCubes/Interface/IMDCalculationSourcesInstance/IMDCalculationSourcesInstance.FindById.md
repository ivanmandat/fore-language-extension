# IMDCalculationSourcesInstance.FindById


## Синтаксис


FindById(Id: String): [IMDCalculationSourceInstance](../IMDCalculationSourceInstance/IMDCalculationSourceInstance.htm);


## Параметры


Id - идентификатор источника данных.


## Описание


Метод FindById осуществляет поиск и возвращает отстроенный источник данных. Идентификатор, по которому осуществляется поиск, передается посредством параметра Id.
