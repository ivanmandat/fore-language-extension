# IMDCalculationSourcesInstance


## Описание


Интерфейс IMDCalculationSourcesInstance
 содержит свойства и методы для работы с коллекцией отстроенных источников
 данных, участвующих в многомерном расчете на сервере БД.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMDCalculationSourcesInstance.Count.htm)
		 Свойство Count возвращает
		 количество источников данных.


		 ![](../../Property_Image.gif)
		 [Item](IMDCalculationSourcesInstance.Item.htm)
		 Свойство Item возвращает
		 отстроенный источник данных


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](IMDCalculationSourcesInstance.FindById.htm)
		 Метод FindById осуществляет
		 поиск отстроенного источника данных по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IMDCalculationSourcesInstance.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск отстроенного источника данных по ключу.
