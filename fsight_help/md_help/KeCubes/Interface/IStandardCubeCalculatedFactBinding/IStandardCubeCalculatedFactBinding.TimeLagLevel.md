# IStandardCubeCalculatedFactBinding.TimeLagLevel


## Синтаксис


TimeLagLevel: [DimCalendarLevel](KeDims.chm::/Enums/DimCalendarLevel.htm);


## Описание


Свойство TimeLagLevel определяет
 уровень календаря, по которому необходимо сместить отметку, для расчета
 вычисляемого показателя.

Пример использования приведен в описании свойств [TimeBackwardLag](IStandardCubeCalculatedFactBinding.TimeBackwardLag.htm),
 [TimeForwardLag](IStandardCubeCalculatedFactBinding.TimeForwardLag.htm).
