# IMDCalculationFormulasTable


## Описание


Интерфейс IMDCalculationFormulasTable
 содержит свойства и методы для работы с таблицей формул многомерного расчета
 на сервере БД.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Fields](IMDCalculationFormulasTable.Fields.htm)
		 Свойство Fields возвращает
		 коллекцию полей таблицы, являющейся таблицей формул для многомерного
		 расчета на сервере БД.


		 ![](../../Property_Image.gif)
		 [Table](IMDCalculationFormulasTable.Table.htm)
		 Свойство Table возвращает
		 параметры таблицы, являющейся таблицей формул для многомерного
		 расчета на сервере БД.


		 ![](../../Property_Image.gif)
		 [TableName](IMDCalculationFormulasTable.TableName.htm)
		 Свойство TableName
		 возвращает наименование схемы и физическое имя таблицы, являющейся
		 таблицей формул для многомерного расчета на сервере БД.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Alter](IMDCalculationFormulasTable.Alter.htm)
		 Метод не поддерживается.


		 ![](../../Sub_Image.gif)
		 [Attach](IMDCalculationFormulasTable.Attach.htm)
		 Метод Attach осуществляет
		 подключение указанной таблицы в качестве таблицы формул.


		 ![](../../Sub_Image.gif)
		 [Check](IMDCalculationFormulasTable.Check.htm)
		 Метод Check осуществляет
		 проверку таблицы на пригодность использования в качестве таблицы
		 формул.


		 ![](../../Sub_Image.gif)
		 [Create](IMDCalculationFormulasTable.Create.htm)
		 Метод Create создает
		 новую таблицу формул для многомерного расчета на сервере БД.
