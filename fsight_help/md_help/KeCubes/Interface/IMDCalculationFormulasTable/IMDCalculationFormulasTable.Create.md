# IMDCalculationFormulasTable.Create


## Синтаксис


Create([Table: [ITable](KeDb.chm::/Interface/ITable/ITable.htm)
 = Null]): [ITable](KeDb.chm::/Interface/ITable/ITable.htm);


## Параметры


Table. Необязательный параметр,
 определяющий таблицу, в которой необходимо создать поля для использования
 её в качестве таблицы формул.


## Описание


Метод Create создает новую таблицу
 формул для многомерного расчета на сервере БД.
