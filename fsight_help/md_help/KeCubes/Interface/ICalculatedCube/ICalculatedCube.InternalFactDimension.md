# ICalculatedCube.InternalFactDimension


## Синтаксис


InternalFactDimension: [ICustomDimension](kedims.chm::/interface/icustomdimension/icustomdimension.htm);


## Описание


Свойство InternalFactDimension
 возвращает параметры локального измерения фактов вычисляемого куба.


## Комментарии


Параметры локального измерения фактов будут доступны, если свойству
 [ExternalFactDimension](ICalculatedCube.ExternalFactDimension.htm)
 установлено значение False.

Пример использования приведен в описании свойства [ICalculatedCube.ExternalFactDimension](ICalculatedCube.ExternalFactDimension.htm).
