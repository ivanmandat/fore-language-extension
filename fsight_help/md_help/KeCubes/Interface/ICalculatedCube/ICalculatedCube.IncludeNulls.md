# ICalculatedCube.IncludeNulls


## Синтаксис


IncludeNulls: Boolean;


## Описание


Свойство IncludeNulls определяет,
 будут ли пустые значения включены в результирующую матрицу.


## Комментарии


Допустимые значения:


	- True. Включать пустые
	 значения в результирующую матрицу. Если куб настроен на сохранение
	 данных в [альтернативный
	 куб](../ICubeModelDestination/ICubeModelDestination.AlternateSource.htm) и в этом кубе имеются данные по тем координатам, по которым
	 в вычисляемом кубе будут рассчитаны пустые значения, то при сохранении
	 эти данные будут удалены.


	- False. Значение по умолчанию.
	 Не включать пустые значения в результирующую матрицу.
