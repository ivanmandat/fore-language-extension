# ICalculatedCube.FormulasDestinationVisible


## Синтаксис


FormulasDestinationVisible: Boolean;


## Описание


Свойство FormulasDestinationVisible определяет, будет ли доступен вариант отображения формул куба при просмотре куба в OLAP.


## Комментарии


По умолчанию свойству установлено значение True, при этом вариант отображения формул доступен.
