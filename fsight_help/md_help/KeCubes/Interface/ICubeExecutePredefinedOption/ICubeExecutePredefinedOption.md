# ICubeExecutePredefinedOption


## Описание


Интерфейс ICubeExecutePredefinedOption
 содержит свойства для работы с предопределенным параметром измерения.


## Свойства


		 Имя свойства
		 Краткое описание

		 [Selected](ICubeExecutePredefinedOption.Selected.htm)
		 Свойство Selected определяет,
		 используется ли данный параметр.
