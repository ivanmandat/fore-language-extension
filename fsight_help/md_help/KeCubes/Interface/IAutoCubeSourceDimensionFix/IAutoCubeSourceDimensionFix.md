# IAutoCubeSourceDimensionFix


## Описание


Интерфейс IAutoCubeSourceDimensionFix
 содержит свойства, используемые для фиксации измерения куба-источника.


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dimension](IAutoCubeSourceDimensionFix.Dimension.htm)
		 Свойство Dimension
		 возвращает справочник, который соответствует фиксированному измерению
		 куба-источника.


		 ![](../../Property_Image.gif)
		 [Instance](IAutoCubeSourceDimensionFix.Instance.htm)
		 Свойство Instance возвращает
		 данные справочника, соответствующего фиксированному измерению.


		 ![](../../Property_Image.gif)
		 [Operation](IAutoCubeSourceDimensionFix.Operation.htm)
		 Свойство Operation
		 определяет метод агрегации, применяемый по данным фиксированного
		 измерения.


		 ![](../../Property_Image.gif)
		 [Selection](IAutoCubeSourceDimensionFix.Selection.htm)
		 Свойство Selection
		 возвращает отметку, по которой осуществляется фиксация элементов
		 измерения.
