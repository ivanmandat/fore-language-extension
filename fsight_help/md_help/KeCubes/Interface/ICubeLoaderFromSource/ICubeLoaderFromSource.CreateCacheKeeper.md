# ICubeLoaderFromSource.CreateCacheKeeper


## Синтаксис


CreateCacheKeeper: Boolean;


## Описание


Свойство CreateCacheKeeper определяет
 признак создания хранилища кешированных данных.


## Комментарии


Допустимые значения:


	- True. Для создаваемого
	 объекта также будет создан дочерний объект - хранилище кешированных
	 данных. В него будет сохранена матрица данных по всей отметке измерений.


	- False. Хранилище кешированных
	 данных не создается.
