# ICubeLoaderFromSource.FactBindings


## Синтаксис


FactBindings: [ICubeLoaderFactBindings](../ICubeLoaderFactBindings/ICubeLoaderFactBindings.htm);


## Описание


Свойство FactBindings возвращает параметры привязки фактов куба к полям источника данных.


## Комментарии


На основании каждой привязки при выполнении метода [Load](ICubeLoaderFromSource.Load.htm) в структуре куба будет создан отдельный факт.
