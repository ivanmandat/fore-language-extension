# ICubeLoaderFromSource.Cube


## Синтаксис


Cube: [IStandardCube](../IStandardCube/IStandardCube.htm);


## Описание


Свойство Cube определяет куб, в который будут загружены данные.


## Комментарии


Если свойство не задано, то при загрузке данных будет создан новый стандартный куб.
