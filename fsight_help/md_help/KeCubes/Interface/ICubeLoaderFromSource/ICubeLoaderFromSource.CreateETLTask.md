# ICubeLoaderFromSource.CreateETLTask


## Синтаксис


CreateETLTask: Boolean;


## Описание


Свойство CreateETLTask определяет,
 будет ли создана задача ETL, содержащая цепочку импорта данных из выбранного
 источника в приёмник.


## Комментарии


Допустимые значения:


	- True. Будет создана
	 задача ETL.


	- False. Не будет создана
	 задача ETL.
