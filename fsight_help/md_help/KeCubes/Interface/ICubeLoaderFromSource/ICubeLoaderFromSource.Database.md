# ICubeLoaderFromSource.Database


## Синтаксис


Database: [IDatabaseInstance](kedb.chm::/interface/idatabaseinstance/idatabaseinstance.htm);


## Описание


Свойство Database определяет
 базу данных, в которой будет создана таблица для хранения данных куба.


## Комментарии


Если свойство не определено, то будет использоваться база данных, установленная
 для репозитория [по
 умолчанию](UiNav.chm::/02_Navigator/Repo_Default.htm).
