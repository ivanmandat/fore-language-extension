# IRubricatorFactorIO.ValidationValues


## Синтаксис


ValidationValues: Boolean;


## Описание


Свойство ValidationValues определяет
 возможность сохранения или извлечения данных по валидациям.


## Комментарии


Если свойству установлено значение True,
 то будут производиться операции с данными по валидациям.
