# IRubricatorFactorIO.ValueId


## Синтаксис


ValueId: String;


## Описание


Свойство ValueId определяет
 идентификаторы атрибутов наблюдений, значения по которым необходимо загрузить
 либо сохранить.


## Комментарии


Идентификаторы атрибутов в списке разделяются знаком «;». По умолчанию
 свойство не установлено, при этом будет осуществляться загрузка по атрибуту
 «Значение» (идентификатор атрибута - VL).


Примечание.
 Если при сохранении необходимо пропустить какой-либо атрибут, полученный
 при извлечении данных, то в данном свойстве атрибут не указывается и вместо
 него просто ставится разделитель «;». Например, если для извлечения данных
 указан следующий список атрибутов ValueId:="ATTR1;ATTR2;ATTR3",
 то для сохранения данных только по третьему атрибуту необходимо в данном
 свойстве задать следующее значение ValueId:=";;ATTR3".
