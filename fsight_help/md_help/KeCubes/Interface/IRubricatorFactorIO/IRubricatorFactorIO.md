# IRubricatorFactorIO


Сборка: Cubes;


## Описание


Интерфейс IRubricatorFactorIO
 предназначен для сохранения или извлечения данных базы данных временных
 рядов.


## Иерархия наследования


           IRubricatorFactorIO


## Комментарии


Для сохранения или извлечения данных приведите объект интерфейса [ICubeInstanceDestinationExecutor](../ICubeInstanceDestinationExecutor/ICubeInstanceDestinationExecutor.htm)
 к интерфейсу IRubricatorFactorIO.
