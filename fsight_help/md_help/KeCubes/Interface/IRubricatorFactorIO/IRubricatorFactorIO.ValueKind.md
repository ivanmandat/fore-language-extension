# IRubricatorFactorIO.ValueKind


## Синтаксис


ValueKind: [MetaAttributeKind](KeRds.chm::/Enums/MetaAttributeKind.htm);


## Описание


Свойство ValueKind определяет
 тип атрибута наблюдений, значения по которому необходимо загрузить либо
 сохранить.


## Комментарии


По умолчанию свойству установлено значение [MetaAttributeKind.None](KeRds.chm::/Enums/MetaAttributeKind.htm),
 при этом будет осуществляться загрузка по атрибуту «Значение» (тип атрибута
 [MetaAttributeKind.Value](KeRds.chm::/Enums/MetaAttributeKind.htm)).
