# ICubeMetaLoaderBindings


## Описание


Интерфейс ICubeMetaLoaderBindings
 содержит свойства и методы для работы с коллекцией привязок атрибутов
 базы данных временных рядов с полями источника данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CalendarBinding](ICubeMetaLoaderBindings.CalendarBinding.htm)
		 Свойство CalendarBinding
		 возвращает привязку календарного измерения.


		 ![](../../Property_Image.gif)
		 [Count](ICubeMetaLoaderBindings.Count.htm)
		 Свойство Count возвращает
		 количество элементов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ICubeMetaLoaderBindings.Item.htm)
		 Свойство Item возвращает
		 элемент коллекции по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ICubeMetaLoaderBindings.Add.htm)
		 Метод Add осуществляет
		 добавление элемента в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](ICubeMetaLoaderBindings.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех элементов из коллекции.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ICubeMetaLoaderBindings.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск элемента коллекции по ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](ICubeMetaLoaderBindings.Remove.htm)
		 Метод Remove осуществляет
		 удаление элемента из коллекции по индексу.
