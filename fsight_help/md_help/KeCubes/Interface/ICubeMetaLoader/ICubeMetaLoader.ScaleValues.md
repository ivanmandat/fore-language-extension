# ICubeMetaLoader.ScaleValues


## Синтаксис


ScaleValues: Boolean;


## Описание


Свойство ScaleValues определяет,
 нормализовывать ли данные при импорте.


## Комментарии


При значении False данные каждого
 показателя будут загружаться с масштабом единиц измерения, установленном
 для него. Таким образом, импортированные данные будут неоднородны. При
 значении True загружаемые данные
 будут нормализованы, то есть приведены к одному масштабу единиц измерения.
