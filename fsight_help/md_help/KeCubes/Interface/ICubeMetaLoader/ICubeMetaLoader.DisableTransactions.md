# ICubeMetaLoader.DisableTransactions


## Синтаксис


DisableTransactions: Boolean;


## Описание


Свойство DisableTransactions
 определяет, выполнять ли импорт временных рядов за одну транзакцию.


## Комментарии


Допустимые значения:


	- True. Импорт временных
	 рядов выполняется в рамках одной транзакции. Рекомендуется использовать,
	 если с базой данных временных рядов одновременно работают несколько
	 пользователей. Это позволит избежать получения пользователями неполных
	 данных, извлеченных из базы данных временных рядов во время выполнения
	 импорта. Также следует учитывать, что импорт за одну транзакцию приводит
	 к увеличению дисковой памяти на сервере;


	- False. Значение по умолчанию.
	 Импорт временных рядов выполняется в рамках нескольких транзакций.


Примечание.
 Значение данного свойства не сохраняется.
