# ICubeMetaLoader.DecreasePeriod


## Синтаксис


DecreasePeriod: Boolean;


## Описание


Свойство DecreasePeriod
 определяет, уменьшать ли период временных рядов в базе данных временных
 рядов.


## Комментарии


Допустимые значения:


	- True. Уменьшать период
	 временных рядов;


	- False. Не уменьшать
	 период временных рядов.
