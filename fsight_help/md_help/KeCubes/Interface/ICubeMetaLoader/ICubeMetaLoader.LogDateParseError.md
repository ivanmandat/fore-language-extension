# ICubeMetaLoader.LogDateParseError


## Синтаксис


LogDateParseError: Boolean;


## Описание


Свойство LogDateParseError
 определяет, нужно ли заносить в отчет о загрузке временных рядов информацию
 об ошибке разбора дат.


## Комментарии


Допустимые значения:


	- True. В отчет будут
	 занесены ошибки разбора дат;


	- False. В отчет не будут
	 занесены ошибки разбора дат.


Для получения отчета используйте свойство [ICubeMetaLoader.Log](ICubeMetaLoader.Log.htm).
