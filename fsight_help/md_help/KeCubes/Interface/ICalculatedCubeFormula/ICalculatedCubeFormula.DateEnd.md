# ICalculatedCubeFormula.DateEnd


## Синтаксис


DateEnd: DateTime;


## Описание


Свойство DateEnd возвращает
 дату окончания действия формулы. Актуально, если для вычисляемого куба
 установлено измерение, управляющее выбором периода действия формулы.
