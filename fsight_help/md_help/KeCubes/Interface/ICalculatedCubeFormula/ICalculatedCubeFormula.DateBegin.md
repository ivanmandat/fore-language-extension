# ICalculatedCubeFormula.DateBegin


## Синтаксис


DateBegin: DateTime;


## Описание


Свойство DateBegin возвращает
 дату начала действия формулы.


## Комментарии


Актуально, если для вычисляемого куба установлено измерение, управляющее
 выбором периода действия формулы.
