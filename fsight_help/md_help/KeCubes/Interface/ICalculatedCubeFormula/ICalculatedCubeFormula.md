# ICalculatedCubeFormula


## Описание


Интерфейс ICalculatedCubeFormula
 содержит свойства и методы для работы с формулой, по которой вычисляется
 значение в определённой координате куба.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Data](ICalculatedCubeFormula.Data.htm)
		 Свойство Data может
		 содержать любые пользовательские данные.


		 ![](../../Property_Image.gif)
		 [DateBegin](ICalculatedCubeFormula.DateBegin.htm)
		 Свойство DateBegin
		 возвращает дату начала действия формулы.


		 ![](../../Property_Image.gif)
		 [DateEnd](ICalculatedCubeFormula.DateEnd.htm)
		 Свойство DateEnd возвращает
		 дату окончания действия формулы.


		 ![](../../Property_Image.gif)
		 [Expression](ICalculatedCubeFormula.Expression.htm)
		 Свойство Expression
		 возвращает объект, содержащий выражение формулы.


		 ![](../../Property_Image.gif)
		 [Index](ICalculatedCubeFormula.Index.htm)
		 Свойство Index возвращает
		 индекс формулы в общей коллекции.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ChangePeriod](ICalculatedCubeFormula.ChangePeriod.htm)
		 Метод ChangePeriod
		 осуществляет изменение даты начала и окончания действия формулы
		 с корректировкой периодов соседних формул.


		 ![](../../Sub_Image.gif)
		 [GetTerms](ICalculatedCubeFormula.GetTerms.htm)
		 Метод GetTerms возвращает
		 массив термов, используемых в формуле.


		 ![](../../Sub_Image.gif)
		 [Remove](ICalculatedCubeFormula.Remove.htm)
		 Метод Remove осуществляет
		 удаление текущей формулы.
