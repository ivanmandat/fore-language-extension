# ICubeSegments


Сборка: Cubes;


## Описание


Интерфейс ICubeSegments предназначен
 для работы с коллекцией сегментов куба.


## Иерархия наследования


ICubeSegments


## Комментарии


Для получения коллекции сегментов используйте метод [ICubeSegmentContainer.GetAllSegments](../ICubeSegmentContainer/ICubeSegmentContainer.GetAllSegments.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeSegments.Count.htm)
		 Свойство Count возвращает
		 количество сегментов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ICubeSegments.Item.htm)
		 Свойство Item возвращает
		 сегмент с указанным индексом.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Clear](ICubeSegments.Clear.htm)
		 Метод Clear удаляет
		 все сегменты из коллекции.


		 ![](../../Sub_Image.gif)
		 [CreateSegment](ICubeSegments.CreateSegment.htm)
		 Метод CreateSegment
		 создаёт сегмент куба.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ICubeSegments.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск по ключу и возвращает настройки сегмента в случае удачного
		 поиска.


		 ![](../../Sub_Image.gif)
		 [IndexOf](ICubeSegments.IndexOf.htm)
		 Метод IndexOf возвращает
		 индекс сегмента куба в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](ICubeSegments.Remove.htm)
		 Метод Remove удаляет
		 сегмент с указанным индексом.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](ICubeSegments.RemoveByKey.htm)
		 Метод RemoveByKey удаляет
		 сегмент с указанным ключом.
