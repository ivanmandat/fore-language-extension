# ICubeSegments.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс сегмента в коллекции.


## Описание


Метод Remove удаляет сегмент
 с указанным индексом.


## Комментарии


Метод возвращает значение True,
 если сегмент удалён и False, если
 удаление по каким-либо причинам не доступно.
