# ICubeSegments.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ сегмента.


## Описание


Метод RemoveByKey удаляет сегмент
 с указанным ключом.


## Комментарии


Метод возвращает значение True,
 если сегмент удалён и False, если
 удаление по каким-либо причинам не доступно.
