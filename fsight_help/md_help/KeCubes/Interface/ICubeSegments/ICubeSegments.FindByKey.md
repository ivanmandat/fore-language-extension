# ICubeSegments.FindByKey


## Синтаксис


FindByKey(Key: Integer): [ICubeSegment](../ICubeSegment/ICubeSegment.htm);


## Параметры


Key. Ключ сегмента.


## Описание


Метод FindByKey осуществляет
 поиск по ключу и возвращает настройки сегмента в случае удачного поиска.


## Комментарии


Метод возвращает значение Null,
 если сегмент с указанным ключом не найден в коллекции.
