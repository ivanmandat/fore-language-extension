# ICubeExecuteFilterItem


Сборка: Cubes;


## Описание


Интерфейс ICubeExecuteFilterItem
 содержит свойства для настройки параметров фильтрации по измерению.


## Иерархия наследования


ICubeExecuteFilterItem


## Комментарии


Параметры фильтрации можно получить, используя свойства и методы интерфейса
 [ICubeExecuteFilterSetup](../ICubeExecuteFilterSetup/ICubeExecuteFilterSetup.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dimension](ICubeExecuteFilterItem.Dimension.htm)


		 Свойство Dimension
		 определяет измерение, для которого настраиваются параметры фильтрации.


		 ![](../../Property_Image.gif)
		 [DimensionKey](ICubeExecuteFilterItem.DimensionKey.htm)


		 Свойство DimensionKey
		 определяет ключ измерения, для которого настраиваются параметры
		 фильтрации.


		 ![](../../Property_Image.gif)
		 [Filter](ICubeExecuteFilterItem.Filter.htm)


		 Свойство Filter возвращает
		 коллекцию, которой принадлежит текущий элемент с параметрами фильтрации.


		 ![](../../Property_Image.gif)
		 [Function_](ICubeExecuteFilterItem.Function_.htm)


		 Свойство Function_
		 определяет параметры функции, которая будет применяться для фильтрации
		 данных.


		 ![](../../Property_Image.gif)
		 [GroupIndex](ICubeExecuteFilterItem.GroupIndex.htm)


		 Свойство GroupIndex
		 определяет индекс группы измерений, в рамках которых будет применяться
		 фильтрация.


		 ![](../../Property_Image.gif)
		 [Text](ICubeExecuteFilterItem.Text.htm)


		 Свойство Text определяет
		 параметры функции, которая будет применяться для фильтрации данных,
		 в строковом виде.
