# IStandardCubeDestinations


Сборка: Cubes;


## Описание


Интерфейс IStandardCubeDestinations
 содержит свойства и методы для работы с коллекцией вариантов отображения
 куба.


## Комментарии


Коллекцию вариантов отображения куба можно получить, используя свойство
 [IStandardCube.Destinations](../IStandardCube/IStandardCube.Destinations.htm).
 По умолчанию в коллекции всегда присутствует минимум один вариант отображения,
 который является [вариантом
 отображения по умолчанию](../IStandardCubeDestination/IStandardCubeDestination.IsDefault.htm).
