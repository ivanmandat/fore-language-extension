# IStandardCubeDestinations.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс варианта отображения.


## Описание


Метод Remove осуществляет удаление варианта отображения, индекс которого передается посредством параметра Index.
