# IStandardCubeDestinations.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IStandardCubeDestination](../IStandardCubeDestination/IStandardCubeDestination.htm);


## Параметры


Key - ключ варианта отображения, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет поиск варианта отображения, ключ которого передается посредством параметра Key.
