# IStandardCubeDestinations.FindById


## Синтаксис


FindById(Id: String): [IStandardCubeDestination](../IStandardCubeDestination/IStandardCubeDestination.htm);


## Параметры


Id - идентификатор отображения, по которому осуществляется поиск.


## Описание


Метод FindById осуществляет поиск варианта отображения, идентификатор которого передается посредством параметра Id.
