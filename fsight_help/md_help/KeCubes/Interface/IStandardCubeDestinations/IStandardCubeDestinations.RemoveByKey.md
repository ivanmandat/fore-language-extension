# IStandardCubeDestinations.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key - ключ варианта отображения.


## Описание


Метод RemoveByKey осуществляет удаление варианта отображения, ключ которого передается посредством параметра Key.
