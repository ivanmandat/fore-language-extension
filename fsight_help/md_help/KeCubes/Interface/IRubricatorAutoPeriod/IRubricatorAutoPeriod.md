# IRubricatorAutoPeriod


Сборка: Cubes;


## Описание


Интерфейс IRubricatorAutoPeriod
 предназначен для работы с параметрами временного периода в базе данных
 временных рядов.


## Иерархия наследования


           IRubricatorAutoPeriod


## Комментарии


Используя данный интерфейс, можно задать временной интервал для импорта,
 экспорта и копирования показателей, причём, как в абсолютных, так и в
 относительных величинах.
