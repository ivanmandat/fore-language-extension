# IRubricatorAutoPeriod.End_


## Синтаксис


End_: [IRubricatorAutoPeriodDate](../IRubricatorAutoPeriodDate/IRubricatorAutoPeriodDate.htm);


## Описание


Свойство End_ возвращает параметры окончания периода.


## Комментарии


По умолчанию End_ совпадает с датой окончания данных. Дата окончания периода должна быть больше даты начала периода. Параметры начала периода возвращает свойство [IRubricatorAutoPeriod.Start](IRubricatorAutoPeriod.Start.htm).
