# IRubricatorAutoPeriod.Start


## Синтаксис


Start: [IRubricatorAutoPeriodDate](../IRubricatorAutoPeriodDate/IRubricatorAutoPeriodDate.htm);


## Описание


Свойство Start возвращает параметры начала периода.


## Комментарии


По умолчанию Start совпадает с датой начала данных. Дата начала периода должна быть меньше даты окончания периода. Параметры окончания периода возвращает свойство [IRubricatorAutoPeriod.End_](IRubricatorAutoPeriod.End_.htm).
