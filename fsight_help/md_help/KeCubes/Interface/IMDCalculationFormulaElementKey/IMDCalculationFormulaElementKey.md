# IMDCalculationFormulaElementKey


## Описание


Интерфейс IMDCalculationFormulaElementKey
 содержит свойства уникального ключа, соответствующего элементу в источнике/приёмнике
 данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMDCalculationFormulaElementKey.Count.htm)
		 Свойство Count возвращает
		 количество свободных измерений источника/приёмника данных, по
		 которым необходимо задать координаты элемента.


		 ![](../../Property_Image.gif)
		 [FactIndex](IMDCalculationFormulaElementKey.FactIndex.htm)
		 Свойство FactIndex
		 определяет индекс показателя, по которому будут браться данные.


		 ![](../../Property_Image.gif)
		 [Item](IMDCalculationFormulaElementKey.Item.htm)
		 Свойство Item определяет
		 значение уникального ключа по заданному измерению.


		 ![](../../Property_Image.gif)
		 [SourceIndex](IMDCalculationFormulaElementKey.SourceIndex.htm)
		 Свойство SourceIndex
		 возвращает индекс источника.
