# ICubeDimensionFixEx


Сборка: Cubes;


## Описание


Интерфейс ICubeDimensionFixEx
 содержит дополнительные методы для управления отметкой фиксированных измерений.


## Иерархия наследования


ICubeDimensionFixEx


## Комментарии


Данный интерфейс используется, если в структуре куба-источника имеется
 параметрическое измерение, которое необходимо зафиксировать с определенными
 значениями параметров. При этом параметры измерения не связаны с параметрами
 того куба (виртуального или представления-куба), в котором используется
 куб-источник. Параметры фиксации измерения могут быть получены с помощью
 одного из свойств или методов интерфейса [ICubeDimensionFixInfo](../ICubeDimensionFixInfo/ICubeDimensionFixInfo.htm).
