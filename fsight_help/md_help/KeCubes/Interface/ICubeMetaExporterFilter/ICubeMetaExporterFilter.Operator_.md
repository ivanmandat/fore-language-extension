# ICubeMetaExporterFilter.Operator_


## Синтаксис


Operator_: [OrmComparisonOperator](KeOrm.chm::/Enums/OrmComparisonOperator.htm);


## Описание


Свойство Operator_ определяет
 оператор сравнения, используемый в фильтре.


## Комментарии


По умолчанию используется следующий оператор сравнения: Operator_ = OrmComparisonOperator.In_.
