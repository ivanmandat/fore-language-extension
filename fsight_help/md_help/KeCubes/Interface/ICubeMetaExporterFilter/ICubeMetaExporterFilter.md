# ICubeMetaExporterFilter


Сборка: Cubes;


## Описание


Интерфейс ICubeMetaExporterFilter
 содержит свойства для работы с фильтром экспорта временных рядов.


## Иерархия наследования


           ICubeMetaExporterFilter


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attribute](ICubeMetaExporterFilter.Attribute.htm)
		 Свойство Attribute
		 определяет наименование атрибута временных рядов, по которому
		 будет производиться фильтрация.


		 ![](../../Property_Image.gif)
		 [AttributeKey](ICubeMetaExporterFilter.AttributeKey.htm)
		 Свойство AttributeKey
		 определяет ключ атрибута временных рядов, по которому будет производиться
		 фильтрация.


		 ![](../../Property_Image.gif)
		 [Operator_](ICubeMetaExporterFilter.Operator_.htm)
		 Свойство Operator_
		 определяет оператор сравнения, используемый в фильтре.


		 ![](../../Property_Image.gif)
		 [Selection](ICubeMetaExporterFilter.Selection.htm)
		 Свойство Selection
		 определяет отметку элементов справочника, связанного с атрибутом,
		 по которому необходимо отфильтровать временные ряды.


		 ![](../../Property_Image.gif)
		 [ValuesList](ICubeMetaExporterFilter.ValuesList.htm)
		 Свойство ValuesList
		 определяет значения фильтра атрибута временных рядов.
