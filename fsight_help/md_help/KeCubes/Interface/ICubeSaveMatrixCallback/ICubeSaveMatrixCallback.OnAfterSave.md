# ICubeSaveMatrixCallback.OnAfterSave


## Синтаксис


		OnAfterSave(Argument: [ICubeCallbackSaveArgument](../ICubeCallbackSaveArgument/ICubeCallbackSaveArgument.htm));


## Параметры


Argument. Аргумент, содержащий
 информацию о результатах сохранения данных в куб.


## Описание


Метод OnAfterSave реализует
 событие, наступающее после сохранения данных в куб.
