# ICubeSaveMatrixCallback.OnBeforeSave


## Синтаксис


		OnBeforeSave(Argument: [ICubeCallbackBeforeSaveArgument](../ICubeCallbackBeforeSaveArgument/ICubeCallbackBeforeSaveArgument.htm));


## Параметры


Argument. Аргумент, содержащий
 информацию о сохраняемых данных.


## Описание


Метод OnBeforeSave реализует
 событие, наступающее перед сохранением данных в куб.


## Комментарии


Используя аргумент события, можно получить и при необходимости изменить
 матрицу с данными куба.
