# ICubeMetaCopierBinding.Value


## Синтаксис


Value: Variant;


## Описание


Свойство Value определяет копируемое
 значение атрибута.


## Комментарии


Данное свойство учитывается, если свойство [ICubeMetaCopierBinding.BindingType](ICubeMetaCopierBinding.BindingType.htm)
 имеет значение [CubeMetaCopierBindingType.Const_](../../Enums/CubeMetaCopierBindingType.htm).
