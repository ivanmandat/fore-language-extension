# IAutoCubeDimension


## Описание


Интерфейс IAutoCubeDimension
 содержит свойства измерения автоматического куба.


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Aggregator](IAutoCubeDimension.Aggregator.htm)
		 Свойство Aggregator
		 определяет параметра агрегации по измерению для указанного варианта
		 отображения куба.


		 ![](../../Property_Image.gif)
		 [AggregatorHierarchies](IAutoCubeDimension.AggregatorHierarchies.htm)
		 Свойство AggregatorHierarchies
		 определяет настройки агрегации по уровням [альтернативной
		 иерархии](UiNavObj.chm::/Cube/CreateCube/Master_Standart/UiMd_Cube_CreateCube_Master_Standart_7.htm#alt_agr) для указанного варианта отображения
		 куба.


		 ![](../../Property_Image.gif)
		 [Cube](IAutoCubeDimension.Cube.htm)
		 Свойство Cube возвращает
		 объект-родитель.


		 ![](../../Property_Image.gif)
		 [Dimension](IAutoCubeDimension.Dimension.htm)
		 Свойство Dimension
		 возвращает структуру измерения автоматического куба.


		 ![](../../Property_Image.gif)
		 [DimIndex](IAutoCubeDimension.DimIndex.htm)
		 Свойство DimIndex определяет
		 индекс измерения-приёмника, по которому осуществляется связь между
		 измерением автоматического куба и куба-источника.


		 ![](../../Property_Image.gif)
		 [MapDimension](IAutoCubeDimension.MapDimension.htm)
		 Свойство MapDimension
		 определяет измерение куба-источника, которое связывается с измерением
		 в автоматическом кубе.


		 ![](../../Property_Image.gif)
		 [MapIndex](IAutoCubeDimension.MapIndex.htm)
		 Свойство MapIndex определяет
		 индекс измерения-источника, по которому осуществляется связь между
		 измерением автоматического куба и куба-источника.


		 ![](../../Property_Image.gif)
		 [Method](IAutoCubeDimension.Method.htm)
		 Свойство Method определяет
		 метод связи измерений.
