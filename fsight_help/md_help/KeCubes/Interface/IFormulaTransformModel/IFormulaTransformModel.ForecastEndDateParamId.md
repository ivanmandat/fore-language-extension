# IFormulaTransformModel.ForecastEndDateParamId


## Синтаксис


ForecastEndDateParamId: String;


## Описание


Свойство ForecastEndDateParamId определяет идентификатор параметра, задающего дату окончания периода прогнозирования.


## Комментарии


Для определения параметра, задающего дату начала периода прогнозирования используйте свойство [IFormulaTransformModel.ForecastStartDateParamId](IFormulaTransformModel.ForecastStartDateParamId.htm).
