# IFormulaTransformModel.Transform


## Синтаксис


Transform: Object;


## Описание


Свойство Transform возвращает
 объект, позволяющий настроить параметры метода расчета.


## Комментарий


Для настройки параметров метода расчёта возвращаемый объект должен быть
 приведен к интерфейсу [IMsFormulaTransform](KeMs.chm::/Interface/IMsFormulaTransform/IMsFormulaTransform.htm).

Использование свойства приведено в примере для [IFormulaTransformModel.AddInputVariable](IFormulaTransformModel.AddInputVariable.htm).
