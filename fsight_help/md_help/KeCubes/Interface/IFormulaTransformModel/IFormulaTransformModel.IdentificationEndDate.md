# IFormulaTransformModel.IdentificationEndDate


## Синтаксис


IdentificationEndDate: DateTime;


## Описание


Свойство IdentificationEndDate
 определяет дату окончания периода идентификации.


## Комментарии


Дата начала прогноза вычисляется как IdentificationEndDate
 плюс один период. Недопустимо одновременное использование свойств IdentificationEndDate и [IFormulaTransformModel.ForecastStartDate](IFormulaTransformModel.ForecastStartDate.htm).
