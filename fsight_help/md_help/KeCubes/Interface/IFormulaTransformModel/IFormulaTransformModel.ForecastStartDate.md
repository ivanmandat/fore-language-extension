# IFormulaTransformModel.ForecastStartDate


## Синтаксис


ForecastStartDate: DateTime;


## Описание


Свойство ForecastStartDate определяет
 дату начала периода прогнозирования.


## Комментарии


Для задания даты окончания периода прогнозирования используйте свойство
 [IFormulaTransformModel.ForecastEndDate](IFormulaTransformModel.ForecastEndDate.htm).
