# IFormulaTransformModel.IdentificationStartOffset


## Синтаксис


IdentificationStartOffset: Integer;


## Описание


Свойство IdentificationStartOffset определяет число периодов, на которое сдвигается дата начала периода идентификации.


## Комментарии


IdentificationStartOffset учитывается, если [IFormulaTransformModel.AutoPeriod](IFormulaTransformModel.AutoPeriod.htm) использует одно из следующих значений перечисления [TransformModelAutoPeriodOptions](../../Enums/TransformModelAutoPeriodOptions.htm):


-
IdentStartByFact;


-
IdentStartByIdentEnd;


-
IdentStartAsNow;


-
IdentByFact;


-
AllByFact.
