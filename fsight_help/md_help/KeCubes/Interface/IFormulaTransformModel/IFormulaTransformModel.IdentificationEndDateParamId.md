# IFormulaTransformModel.IdentificationEndDateParamId


## Синтаксис


IdentificationStartDateParamId: String;


## Описание


Свойство IdentificationEndDateParamId определяет идентификатор параметра, задающего дату окончания периода идентификации.


## Комментарии


Для определения параметра, задающего дату начала периода идентификации используйте свойство [IFormulaTransformModel.IdentificationStartDateParamId](IFormulaTransformModel.IdentificationStartDateParamId.htm).
