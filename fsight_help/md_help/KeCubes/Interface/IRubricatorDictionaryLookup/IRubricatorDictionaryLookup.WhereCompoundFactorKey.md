# IRubricatorDictionaryLookup.WhereCompoundFactorKey


## Синтаксис


WhereCompoundFactorKey: Variant;


## Описание


Свойство WhereCompoundFactorKey
 определяет составной ключ искомого временного ряда.


## Комментарии


Массив составных ключей найденных временных рядов возвращает метод [IRubricatorDictionaryLookup.LookupFactorsC](IRubricatorDictionaryLookup.LookupFactorsC.htm).
