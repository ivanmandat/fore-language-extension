# IRubricatorDictionaryLookup.WhereRevisionKey


## Синтаксис


WhereRevisionKey: Integer;


## Описание


Свойство WhereRevisionKey определяет
 ключ ревизии, по которой должен быть осуществлен поиск.


## Комментарии


Если данное свойство не определено, то поиск производится по последней
 ревизии.
