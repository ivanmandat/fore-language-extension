# IRubricatorDictionaryLookup.WhereFactorKey


## Синтаксис


WhereFactorKey: Integer;


## Описание


Свойство WhereFactorKey определяет
 ключ искомого временного ряда.


## Комментарии


Если данное свойство не определено, то поиск производится по всем временным
 рядам.
