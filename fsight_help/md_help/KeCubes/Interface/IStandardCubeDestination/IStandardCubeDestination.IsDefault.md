# IStandardCubeDestination.IsDefault


## Синтаксис


		IsDefault: Boolean;


## Описание


Свойство IsDefault возвращает
 признак соответствия варианту отображения по умолчанию.


## Комментарии


Свойство возвращает значение True,
 если текущий вариант отображения, является вариантом отображения по умолчанию.


Вариант отображения по умолчанию используется при первом открытии куба
 в различных инструментах «Форсайт. Аналитическая платформа».
