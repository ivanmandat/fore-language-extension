# IStandardCubeDestination.HasCommentsSupport


## Синтаксис


HasCommentsSupport: Boolean;


## Описание


Свойство HasCommentsSupport
 возвращает признак поддержки кубом возможности сохранения комментариев
 к данным.


## Комментарии


Свойство возвращает значение True,
 если в дополнительных настройках мастера стандартного куба включено хранение
 комментариев.
