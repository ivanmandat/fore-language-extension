# IStandardCubeDestination.HasAttachmentsSupport


## Синтаксис


		HasAttachmentsSupport: Boolean;


## Описание


Свойство HasAttachmentsSupport
 возвращает признак поддержки работы с вложениями.


## Комментарии


Свойство возвращает значение True,
 если для варианта отображения куба [определён
 справочник](IStandardCubeDestination.AttachmentsStorage.htm), в котором хранятся вложения, и настроена [привязка
 фактов](../IStandardCubeFactBindings/IStandardCubeFactBindings.BindingWithRole.htm).
