# IStandardCubeDestination.HideTrivialFactDimension


## Синтаксис


		HideTrivialFactDimension: Boolean;


## Описание


Свойство HideTrivialFactDimension
 определяет, будет ли скрываться измерение фактов при работе с вариантом
 отображения куба в инструментах «Форсайт. Аналитическая платформа».


## Комментарии


Если свойству установлено значение True,
 то измерение фактов будет скрыто при работе с кубом в различных отчётах
 и аналитических панелях. При этом фактические значения будут отображаться/редактироваться
 по первому факту.


Данное свойство актуально, если измерение фактов состоит из одного элемента.
