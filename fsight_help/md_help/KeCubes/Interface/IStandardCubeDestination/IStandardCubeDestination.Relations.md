# IStandardCubeDestination.Relations


## Синтаксис


		Relations: [IStandardCubeRelations](../IStandardCubeRelations/IStandardCubeRelations.htm);


## Описание


Свойство Relations возвращает
 коллекцию отношений варианта отображения куба.


## Комментарии


Отношения позволяют получать значения фактов путём смещения относительно
 текущей точки расчёта. Используются в формулах [вычисляемых
 фактов](IStandardCubeDestination.CalcBindings.htm) куба.


## Пример


Пример использования приведён в описании свойств [IStandardCubeCalculatedFactBinding.TimeBackwardLag](../IStandardCubeCalculatedFactBinding/IStandardCubeCalculatedFactBinding.TimeBackwardLag.htm),
 [IStandardCubeCalculatedFactBinding.TimeForwardLag](../IStandardCubeCalculatedFactBinding/IStandardCubeCalculatedFactBinding.TimeForwardLag.htm).
