# IStandardCubeDestination.FactDimension


## Синтаксис


		FactDimension: [IStandardCubeDimension](../IStandardCubeDimension/IStandardCubeDimension.htm);


## Описание


Свойство FactDimension возвращает
 параметры измерения фактов варианта отображения куба.


## Комментарии


Свойство возвращает структуру измерения фактов вне зависимости от того,
 используется ли локальное измерение или существующий справочник репозитория.
