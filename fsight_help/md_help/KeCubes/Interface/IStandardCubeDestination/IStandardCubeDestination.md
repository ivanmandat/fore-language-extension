# IStandardCubeDestination


Сборка: Cubes;


## Описание


Интерфейс IStandardCubeDestination
 содержит свойства варианта отображения куба.


## Комментарии


Вариант отображения можно получить, используя свойства и методы интерфейса
 [IStandardCubeDestinations](../IStandardCubeDestinations/IStandardCubeDestinations.htm).
