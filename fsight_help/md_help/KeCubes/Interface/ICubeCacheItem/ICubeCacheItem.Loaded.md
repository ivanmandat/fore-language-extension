# ICubeCacheItem.Loaded


## Синтаксис


Loaded: Boolean;


## Описание


Свойство Loaded возвращает признак
 загрузки матрицы с данными текущего экземпляра кеша в память компьютера.


## Комментарии


Свойство возвращает значение True,
 если матрица с данными загружена в память и False,
 если матрица не загружена и была получена из СУБД.
