# ICubeCacheItem.Busy


## Синтаксис


Busy: Boolean;


## Описание


Свойство Busy возвращает признак
 недоступности экземпляра кеша для записи.


## Комментарии


Свойство возвращает значение True,
 если текущий экземпляр кеша заблокирован на запись какой-либо другой операцией,
 и значение False, если экземпляр
 кеша может быть изменён. Если экземпляр кеша заблокирован, то значения
 других свойств для него могут быть не заполнены и возвращать пустое значение.
