# ICubeCacheItem


Сборка: Cubes;


## Описание


Интерфейс ICubeCacheItem предназначен
 для работы с данными экземпляра кеша.


## Иерархия наследования


ICubeCacheItem


## Комментарии


Экземпляр кеша можно получить, используя свойство [ICubeCacheItems.Item](../ICubeCacheItems/ICubeCacheItems.Item.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Busy](ICubeCacheItem.Busy.htm)
		 Свойство Busy возвращает
		 признак недоступности экземпляра кеша для записи.


		 ![](../../Property_Image.gif)
		 [CustomValue](ICubeCacheItem.CustomValue.htm)
		 Свойство CustomValue
		 возвращает пользовательскую метку, которая задана для текущего
		 экземпляра кеша.


		 ![](../../Property_Image.gif)
		 [Date](ICubeCacheItem.Date.htm)
		 Свойство Date возвращает
		 дату последнего изменения экземпляра кеша.


		 ![](../../Property_Image.gif)
		 [DestinationKey](ICubeCacheItem.DestinationKey.htm)
		 Свойство DestinationKey
		 возвращает ключ варианта отображения куба, для которого сохранён
		 кеш.


		 ![](../../Property_Image.gif)
		 [Hash](ICubeCacheItem.Hash.htm)
		 Свойство Hash возвращает
		 хеш-код экземпляра кеша.


		 ![](../../Property_Image.gif)
		 [Loaded](ICubeCacheItem.Loaded.htm)
		 Свойство Loaded возвращает
		 признак загрузки матрицы с данными текущего экземпляра кеша в
		 память компьютера.


		 ![](../../Property_Image.gif)
		 [ParamValues](ICubeCacheItem.ParamValues.htm)
		 Свойство ParamValues
		 возвращает значения параметров, для которых сохранён экземпляр
		 кеша.


		 ![](../../Property_Image.gif)
		 [SelectionSet](ICubeCacheItem.SelectionSet.htm)
		 Свойство SelectionSet
		 возвращает отметку измерений, по которой был создан экземпляр
		 кеша.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Clear](ICubeCacheItem.Clear.htm)
		 Метод Clear очищает
		 содержимое экземпляра кеша.


		 ![](../../Sub_Image.gif)
		 [GetCubeInstance](ICubeCacheItem.GetCubeInstance.htm)
		 Метод GetCubeInstance
		 возвращает открытый экземпляр куба, для которого сохранён кеш.


		 ![](../../Sub_Image.gif)
		 [GetCubeInstanceDestination](ICubeCacheItem.GetCubeInstanceDestination.htm)
		 Метод GetCubeInstanceDestination
		 возвращает открытый экземпляр варианта отображения куба, для которого
		 сохранён кеш.


		 ![](../../Sub_Image.gif)
		 [GetMatrix](ICubeCacheItem.GetMatrix.htm)
		 Метод GetMatrix возвращает
		 матрицу с данными, сохранёнными в текущем экземпляре кеша.


		 ![](../../Sub_Image.gif)
		 [LoadMatrix](ICubeCacheItem.LoadMatrix.htm)
		 Метод LoadMatrix загружает
		 матрицу с данными, которые сохранены в экземпляр кеша, в память
		 компьютера.


		 ![](../../Sub_Image.gif)
		 [SetCustomValue](ICubeCacheItem.SetCustomValue.htm)
		 Метод SetCustomValue
		 устанавливает пользовательскую метку для текущего экземпляра кеша.
