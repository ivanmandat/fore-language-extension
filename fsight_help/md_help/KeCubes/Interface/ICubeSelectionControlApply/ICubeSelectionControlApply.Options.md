# ICubeSelectionControlApply.Options


## Синтаксис


Options: [CubeSelectionControlOptions](../../Enums/CubeSelectionControlOptions.htm);


## Описание


Свойство Options позволяет применить настройки управления через управляющее измерение в вычисляемом кубе.


## Комментарии


Свойство актуально, если для измерения вычисляемого куба настроено задание значения параметра через выражение в управлении вложенными объектами и одновременно через механизм управляющего измерения.
