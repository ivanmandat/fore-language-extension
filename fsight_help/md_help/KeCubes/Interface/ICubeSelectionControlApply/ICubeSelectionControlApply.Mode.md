# ICubeSelectionControlApply.Mode


## Синтаксис


Mode: [SelectionControlApplyMode](../../Enums/SelectionControlApplyMode.htm);


## Описание


Свойство Mode возвращает режим,
 в котором будет осуществляться применение параметров управления.


## Комментарии


Для применения параметров управления используйте метод [ICalculatedCubeInstance.Apply](../ICalculatedCubeInstance/ICalculatedCubeInstance.Apply.htm).
 Свойство Mode вернет тот режим,
 который был выбран при инициализации объекта для применение параметров
 управления.
