# IStandardCubeServerAggregations


## Описание


Интерфейс IStandardCubeServerAggregations
 содержит свойства и методы для работы с коллекцией рассчитываемых на сервере
 агрегаций.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IStandardCubeServerAggregations.Count.htm)
		 Свойство Count возвращает
		 количество агрегаций в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IStandardCubeServerAggregations.Item.htm)
		 Свойство Item возвращает
		 агрегацию из коллекции по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IStandardCubeServerAggregations.Add.htm)
		 Метод Add осуществляет
		 добавление агрегации в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](IStandardCubeServerAggregations.Clear.htm)
		 Метод Clear осуществляет
		 очистку коллекции агрегаций.


		 ![](../../Sub_Image.gif)
		 [FindById](IStandardCubeServerAggregations.FindById.htm)
		 Метод FindById осуществляет
		 поиск агрегации в коллекции по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IStandardCubeServerAggregations.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск агрегации в коллекции по ключу.


		 ![](../../Sub_Image.gif)
		 [Move](IStandardCubeServerAggregations.Move.htm)
		 Метод Move осуществляет
		 перемещение агрегаций в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IStandardCubeServerAggregations.Remove.htm)
		 Метод Remove осуществляет
		 удаление агрегации из коллекции по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IStandardCubeServerAggregations.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление агрегации из коллекции по ключу.
