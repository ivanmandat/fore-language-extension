# IStandardCubeServerAggregations.Move


## Синтаксис


Move(IndexFrom: Integer; IndexTo: Integer);


## Параметры


IndexFrom - индекс позиции агрегации, которую необходимо переместить.


IndexTo - индекс позиции, в которую необходимо переместить агрегацию.


## Описание


Метод Move осуществляет перемещение агрегаций в коллекции.
