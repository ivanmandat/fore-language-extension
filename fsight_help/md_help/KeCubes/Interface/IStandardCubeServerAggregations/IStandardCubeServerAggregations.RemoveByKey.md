# IStandardCubeServerAggregations.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key - ключ удаляемой агрегации.


## Описание


Метод RemoveByKey осуществляет удаление агрегации из коллекции по ключу. Данный метод возвращает значение True, если удаление прошло успешно.
