# IStandardCubeServerAggregations.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс удаляемой агрегации.


## Описание


Метод Remove осуществляет удаление агрегации из коллекции по индексу. Данный метод возвращает значение True, если удаление прошло успешно.
