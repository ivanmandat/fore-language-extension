# ICubeAttachmentAttribute.Nullable


## Синтаксис


		Nullable: Boolean;


## Описание


Свойство Nullable возвращает
 признак возможности хранения пустых значений по данному атрибуту.


## Комментарии


Свойство возвращает значение True,
 если по атрибуту могут храниться пустые значения.
