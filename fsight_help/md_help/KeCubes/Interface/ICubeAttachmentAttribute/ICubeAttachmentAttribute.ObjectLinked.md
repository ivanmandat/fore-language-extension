# ICubeAttachmentAttribute.ObjectLinked


## Синтаксис


		ObjectLinked: Boolean;


## Описание


Свойство ObjectLinked возвращает
 признак хранения в значениях ссылки на объект репозитория.


## Комментарии


Свойство возвращает значение True,
 если в атрибуте хранится ссылка на объект репозитория.
