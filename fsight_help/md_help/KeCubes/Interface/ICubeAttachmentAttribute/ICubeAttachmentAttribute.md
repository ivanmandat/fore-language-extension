# ICubeAttachmentAttribute


Сборка: Cubes;


## Описание


Интерфейс ICubeAttachmentAttribute
 описывает атрибут справочника, в котором хранятся вложения куба.


## Иерархия наследования


           [INamedEntityModel](ForeSys.chm::/Interface/INamedEntityModel/INamedEntityModel.htm)


           ICubeAttachmentAttribute


## Комментарии


Параметры атрибута можно получить, используя свойства и методы интерфейса
 [ICubeAttachmentAttributes](../ICubeAttachmentAttributes/ICubeAttachmentAttributes.htm).
