# IAutoCubeDestination.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет,
 является ли данный вариант отображения скрытым.


## Комментарии


По умолчанию свойству установлено значение True,
 при этом вариант отображения является видимым.
