# IAutoCubeDestination


## Описание


Интерфейс IAutoCubeDestination
 содержит свойства и методы варианта отображения автоматического куба.


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Cube](IAutoCubeDestination.Cube.htm)
		 Свойство Cube возвращает
		 объект-родитель.


		 ![](../../Property_Image.gif)
		 [IsDefault](IAutoCubeDestination.IsDefault.htm)
		 Свойство IsDefault
		 возвращает True, если
		 данный вариант отображения, является варинтом отображения по умолчанию.


		 ![](../../Property_Image.gif)
		 [Visible](IAutoCubeDestination.Visible.htm)
		 Свойство Visible определяет,
		 является ли данный вариант отображения скрытым.

		  Имя метода
