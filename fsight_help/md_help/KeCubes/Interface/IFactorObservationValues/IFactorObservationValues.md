# IFactorObservationValues


## Описание


Интерфейс IFactorObservationValues
 содержит свойства и методы для работы с записью в таблице данных временных
 рядов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CalendarPoint](IFactorObservationValues.CalendarPoint.htm)
		 Свойство CalendarPoint
		 возвращает дату и время, для которых сделана запись.


		 ![](../../Property_Image.gif)
		 [Explanation](IFactorObservationValues.Explanation.htm)
		 Свойство Explanation
		 определяет объяснение значения временного ряда.


		 ![](../../Property_Image.gif)
		 [FactData](IFactorObservationValues.FactData.htm)
		 Свойство FactData возвращает
		 данные показателя.


		 ![](../../Property_Image.gif)
		 [FactDataCompoundKey](IFactorObservationValues.FactDataCompoundKey.htm)
		 Свойство FactDataCompoundKey
		 возвращает составной ключ временного ряда, которому принадлежит
		 запись.


		 ![](../../Property_Image.gif)
		 [FactDataKey](IFactorObservationValues.FactDataKey.htm)
		 Свойство FactDataKey
		 возвращает ключ временного ряда.
