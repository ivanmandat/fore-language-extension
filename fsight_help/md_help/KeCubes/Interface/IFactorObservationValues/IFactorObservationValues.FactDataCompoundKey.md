# IFactorObservationValues.FactDataCompoundKey


## Синтаксис


FactDataCompoundKey: Variant;


## Описание


Свойство FactDataCompoundKey возвращает составной ключ временного ряда, которому принадлежит запись.


## Комментарии


Признак того, используется ли составной ключ, возвращает свойство [IRubricatorRecord.UseCompoundKey](../IRubricatorRecord/IRubricatorRecord.UseCompoundKey.htm).
