# ICubeDimensionQueryStrategy


## Описание


Интерфейс ICubeDimensionQueryStrategy
 содержит свойства, определяющие стратегию фильтрации запроса по измерению.


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [FilterMethod](ICubeDimensionQueryStrategy.FilterMethod.htm)
		 Свойство FilterMethod
		 определяет метод фильтрации запроса по измерению.


		 ![](../../Property_Image.gif)
		 [RatioOrQuantity](ICubeDimensionQueryStrategy.RatioOrQuantity.htm)
		 Свойство RatioOrQuantity
		 определяет размер отметки (%), либо количество элементов, в зависимости
		 от установленного [метода
		 фильтрации](ICubeDimensionQueryStrategy.FilterMethod.htm).
