# ICubeDimensionQueryStrategy.RatioOrQuantity


## Синтаксис


RatioOrQuantity: Integer;


## Описание


Свойство RatioOrQuantity определяет
 размер отметки (%), либо количество элементов, в зависимости от установленного
 [метода фильтрации](ICubeDimensionQueryStrategy.FilterMethod.htm).


## Комментарии


Более подробное описание о формировании запроса на выборку данных в
 зависимости от выбранной стратегии фильтрации представлено в подразделе
 «[Измерения
 куба: Свойства измерения](UiNavObj.chm::/Cube/CreateCube/Master_Standart/UiMd_Cube_CreateCube_Master_Standart_3.htm#dim_properties)».
