# IStandardCubeDatasets.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс источника данных.


## Описание


Метод Remove осуществляет удаление источника данных куба, индекс которого передается посредством параметра Index.
