# IStandardCubeDatasets.Add


## Синтаксис


Add(Dataset: [IDatasetModel](KeDb.chm::/Interface/IDatasetModel/IDatasetModel.htm)): [IStandardCubeDataset](../IStandardCubeDataset/IStandardCubeDataset.htm);


## Параметры


Dataset - реляционный источник данных репозитория, который необходимо добавить в куб.


## Описание


Метод Add осуществляет добавление реляционного источника данных в куб.
