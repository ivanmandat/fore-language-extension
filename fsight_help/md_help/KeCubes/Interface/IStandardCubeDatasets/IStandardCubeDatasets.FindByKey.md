# IStandardCubeDatasets.FindByKey


## Синтаксис


FindByKey(DatasetKey: Integer): [IStandardCubeDataset](../IStandardCubeDataset/IStandardCubeDataset.htm);


## Параметры


DatasetKey - ключ источника данных, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет поиск и возвращает объект, содержащий источник данных куба, в случае удачного поиска.
