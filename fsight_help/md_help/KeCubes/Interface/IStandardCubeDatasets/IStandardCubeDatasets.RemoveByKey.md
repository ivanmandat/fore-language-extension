# IStandardCubeDatasets.RemoveByKey


## Синтаксис


RemoveByKey(DatasetKey: Integer): Boolean;


## Параметры


DatasetKey - ключ источника данных.


## Описание


Метод RemoveByKey осуществляет удаление источника данных, ключ которого передается посредством параметра DatasetKey.
