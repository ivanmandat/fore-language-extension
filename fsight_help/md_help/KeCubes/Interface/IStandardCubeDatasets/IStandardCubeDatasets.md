# IStandardCubeDatasets


## Описание


Интерфейс IStandardCubeDatasets
 содержит свойства и методы для работы с коллекцией источников данных куба.


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IStandardCubeDatasets.Count.htm)
		 Свойство Count возвращает
		 количество источников данных куба.


		 ![](../../Property_Image.gif)
		 [Item](IStandardCubeDatasets.Item.htm)
		 Свойство Item возвращает
		 источник данных куба.


## Методы


		  Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IStandardCubeDatasets.Add.htm)
		 Метод Add осуществляет
		 добавление реляционного источника данных в куб.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IStandardCubeDatasets.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск источника данных по ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](IStandardCubeDatasets.Remove.htm)
		 Метод Remove осуществляет
		 удаление источника данных по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IStandardCubeDatasets.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление источника данных по ключу.
