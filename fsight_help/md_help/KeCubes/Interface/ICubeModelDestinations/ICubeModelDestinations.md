# ICubeModelDestinations


## Описание


Интерфейс ICubeModelDestinations
 содержит свойства и методы коллекции вариантов отображения куба.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeModelDestinations.Count.htm)
		 Свойство Count возвращает
		 количество вариантов отображения куба.


		 ![](../../Property_Image.gif)
		 [DefaultDestination](ICubeModelDestinations.DefaultDestination.htm)
		 Свойство DefaultDestination
		 возвращает вариант отображения, установленный для куба по умолчанию.


		 ![](../../Property_Image.gif)
		 [Item](ICubeModelDestinations.Item.htm)
		 Свойство Item возвращает
		 вариант отображения куба.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindByKey](ICubeModelDestinations.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск варианта отображения куба по ключу.
