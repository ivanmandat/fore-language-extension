# ICubeModelDestinations.FindByKey


## Синтаксис


FindByKey(DestinationKey: Integer): [ICubeModelDestination](../ICubeModelDestination/ICubeModelDestination.htm);


## Параметры


DestinationKey - ключ варианта отображения куба, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет поиск и возвращает вариант отображения куба, в случае удачного поиска. Ключ варианта отображения передается посредством параметра DestinationKey.
