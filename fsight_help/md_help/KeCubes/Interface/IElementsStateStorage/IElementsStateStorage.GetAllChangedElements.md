# IElementsStateStorage.GetAllChangedElements


## Синтаксис


GetAllChangedElements([StateStorage: [IElementsStateStorage](IElementsStateStorage.htm)
 = Null]): Array;


## Параметры


StateStorage. Параметр определяет
 коллекцию отслеживаемых элементов управления.


## Описание


Метод GetAllChangedElements
 позволяет получить ключи всех элементов, значения которых изменились.


## Комментарии


Если параметр StateStorage =
 Null, то сравнение происходит с актуальным состоянием элементов.
