# IElementsStateStorage.IsElementChanged


## Синтаксис


IsElementChanged([StateStorage: [IElementsStateStorage](IElementsStateStorage.htm)
 = Null]): Boolean;


## Параметры


StateStorage. Параметр определяет
 коллекцию отслеживаемых элементов управления.


## Описание


Метод IsElementChanged определяет,
 изменилось ли состояние элемента.


## Комментарии


Если параметр StateStorage =
 Null, то сравнение происходит с актуальным состоянием элемента.
