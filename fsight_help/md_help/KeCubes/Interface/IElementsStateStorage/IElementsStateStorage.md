# IElementsStateStorage


Сборка: Cubes;


## Описание


Интерфейс IElementsStateStorage содержит методы для работы с отслеживаемыми элементами управления отчета.


## Методы


 Имя метода
 Краткое описание


 ![](../../Sub_Image.gif)
 [ClearAllElements](IElementsStateStorage.ClearAllElements.htm)
 Метод ClearAllElements производит очистку всех элементов.


 ![](../../Sub_Image.gif)
 [ClearElement](IElementsStateStorage.ClearElement.htm)
 Метод ClearElement очищает значение элемента.


 ![](../../Sub_Image.gif)
 [GetAllChangedElements](IElementsStateStorage.GetAllChangedElements.htm)
 Метод GetAllChangedElements позволяет получить ключи всех элементов, значения которых изменились.


 ![](../../Sub_Image.gif)
 [IsElementChanged](IElementsStateStorage.IsElementChanged.htm)
 Метод IsElementChanged определяет, изменилось ли состояние элемента.


 ![](../../Sub_Image.gif)
 [PutAllElements](IElementsStateStorage.PutAllElements.htm)
 Метод PutAllElements сохраняет значение всех элементов управления.


 ![](../../Sub_Image.gif)
 [PutElement](IElementsStateStorage.PutElement.htm)
 Метод PutElement сохраняет значение выбранного элемента.
