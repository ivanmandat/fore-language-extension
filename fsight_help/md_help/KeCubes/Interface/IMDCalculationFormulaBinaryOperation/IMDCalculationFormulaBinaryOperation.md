# IMDCalculationFormulaBinaryOperation


## Описание


Интерфейс IMDCalculationFormulaBinaryOperation
 содержит свойства и методы операнда формулы, представляющего собой бинарную
 операцию.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BinaryOperation](IMDCalculationFormulaBinaryOperation.BinaryOperation.htm)
		 Свойство BinaryOperation
		 определяет вид бинарной операции, которая используется как операнд
		 в формуле.
