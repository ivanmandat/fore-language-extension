# IAutoCubeSource.Transparent


## Синтаксис


Transparent: Boolean;


## Описание


Свойство Transparent определяет,
 будут ли загружаться данные из куба-источника при построении автоматического
 куба.


## Комментарии


По умолчанию свойству установлено значение False,
 при этом данные загружаются из системной таблицы, соответствующей автоматическому
 кубу. При установке значения True данные будут загружаться из установленного
 [куба-источника](IAutoCubeSource.Cube.htm).
