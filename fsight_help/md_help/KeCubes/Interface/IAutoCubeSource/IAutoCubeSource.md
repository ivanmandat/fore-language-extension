# IAutoCubeSource


## Описание


Интерфейс IAutoCubeSource содержит
 свойства источника данных для автоматического куба.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Calendar](IAutoCubeSource.Calendar.htm)
		 Свойство Calendar определяет
		 календарный справочник для куба-источника.


		 ![](../../Property_Image.gif)
		 [Cube](IAutoCubeSource.Cube.htm)
		 Свойство Cube определяет
		 куб-источник, из которого будут загружаться данные в автоматический
		 куб.


		 ![](../../Property_Image.gif)
		 [FixInfo](IAutoCubeSource.FixInfo.htm)
		 Свойство FixInfo возвращает
		 коллекцию фиксированных измерений источника данных.


		 ![](../../Property_Image.gif)
		 [Transparent](IAutoCubeSource.Transparent.htm)
		 Свойство Transparent
		 определяет, будут ли загружаться данные из куба-источника, при
		 построении автоматического куба.
