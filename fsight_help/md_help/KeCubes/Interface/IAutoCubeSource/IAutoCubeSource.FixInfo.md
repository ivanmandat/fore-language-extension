# IAutoCubeSource.FixInfo


## Синтаксис


FixInfo: [IAutoCubeSourceFixInfo](../IAutoCubeSourceFixInfo/IAutoCubeSourceFixInfo.htm);


## Описание


Свойство FixInfo возвращает
 коллекцию фиксированных измерений источника данных.


## Комментарии


В данной коллекции содержатся все измерения куба-источника, которых
 нет в автоматическом кубе.
