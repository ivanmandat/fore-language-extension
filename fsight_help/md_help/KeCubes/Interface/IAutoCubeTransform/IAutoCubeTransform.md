# IAutoCubeTransform


## Описание


Интерфейс IAutoCubeTransform
 содержит свойства и методы, необходимые для преобразования данных автоматического
 куба при изменении позиции измерений в кубе.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dimensions](IAutoCubeTransform.Dimensions.htm)
		 Свойство Dimensions
		 возвращает список измерений автоматического куба после перестановки.


		 ![](../../Property_Image.gif)
		 [MapDimensions](IAutoCubeTransform.MapDimensions.htm)
		 Свойство MapDimensions
		 возвращает исходный список измерений автоматического куба.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Execute](IAutoCubeTransform.Execute.htm)
		 Метод Execute осуществляет
		 преобразование данных, необходимое при изменении позиции измерений
		 в кубе.


		 ![](../../Sub_Image.gif)
		 [GetChanged](IAutoCubeTransform.GetChanged.htm)
		 Метод GetChanged возвращает
		 признак наличия изменений в позициях измерений автоматического
		 куба.
