# IMDCalculationSlice.IsShift


## Синтаксис


IsShift: Boolean;


## Описание


Свойство IsShift определяет,
 будет ли при составлении формул, возможность использовать функцию сдвига
 по дате ([Shift](UiMd.chm::/Cube/Work_Cube/UiMd_CubeWork_MultivariateAccount_Functions.htm)).


## Комментарии


Свойство актуально, если свойству [IsCalendar](IMDCalculationSlice.IsCalendar.htm)
 установлено значение True.
