# IMDCalculationSlice


## Описание


Интерфейс IMDCalculationSlice
 содержит свойства среза источника/приёмника данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dataset](IMDCalculationSlice.Dataset.htm)
		 Свойство Dataset возвращает
		 источник/приёмник данных.


		 ![](../../Property_Image.gif)
		 [Fields](IMDCalculationSlice.Fields.htm)
		 Свойство Fields определяет
		 список полей источника/приёмника данных, связанных с данным измерением.


		 ![](../../Property_Image.gif)
		 [Fixed](IMDCalculationSlice.Fixed.htm)
		 Свойство Fixed возвращает
		 параметры измерения данных, источником/приёмником для которого
		 служит фиксированное измерение.


		 ![](../../Property_Image.gif)
		 [FixType](IMDCalculationSlice.FixType.htm)
		 Свойство FixType определяет
		 тип измерения.


		 ![](../../Property_Image.gif)
		 [Free](IMDCalculationSlice.Free.htm)
		 Свойство Free возвращает
		 параметры измерения данных, источником/приёмником для которого
		 служит свободное измерение.


		 ![](../../Property_Image.gif)
		 [IsCalendar](IMDCalculationSlice.IsCalendar.htm)
		 Свойство IsCalendar
		 определяет признак того, что источником данного измерения является
		 календарное измерение.


		 ![](../../Property_Image.gif)
		 [IsShift](IMDCalculationSlice.IsShift.htm)
		 Свойство IsShift определяет,
		 будет ли при составлении формул, возможность использовать функцию
		 сдвига по дате ([Shift](UiMd.chm::/Cube/Work_Cube/UiMd_CubeWork_MultivariateAccount_Functions.htm)).
