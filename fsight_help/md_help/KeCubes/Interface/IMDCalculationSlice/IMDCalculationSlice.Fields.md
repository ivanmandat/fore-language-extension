# IMDCalculationSlice.Fields


## Синтаксис


Fields: String;


## Описание


Свойство Fields определяет список
 полей источника/приёмника данных, связанных с данным измерением. Поля
 в списке разделяются знаком ";".


В данном списке указываются поля, связываемые с измерением [Dimension](../IMDCalculationSliceFree/IMDCalculationSliceFree.Dimension.htm)
 посредством значений атрибутов, по которым создан индекс [Index](../IMDCalculationSliceFree/IMDCalculationSliceFree.Index.htm)
 измерения.
