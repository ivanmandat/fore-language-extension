# ICubeExecuteFilterSetup.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемого элемента.


## Описание


Метод Remove удаляет элемент
 с параметрами фильтрации из коллекции и возвращает признак удачного удаления.


## Комментарии


Метод возвращает значение True,
 если удаление прошло удачно и False,
 если удаление по каким-либо причинам невозможно.
