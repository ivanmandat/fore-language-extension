# ICubeExecuteFilterSetup


Сборка: Cubes;


## Описание


Интерфейс ICubeExecuteFilterSetup
 содержит свойства и методы для работы с коллекцией фильтров по измерениям.


## Иерархия наследования


ICubeExecuteFilterSetup


## Комментарии


Параметры коллекции можно получить в свойствах [ICubeExecuteSetup.Filter](../ICubeExecuteSetup/ICubeExecuteSetup.Filter.htm)
 или [ICubeExecuteFilterItem.Filter](../ICubeExecuteFilterItem/ICubeExecuteFilterItem.Filter.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeExecuteFilterSetup.Count.htm)


		 Свойство Count возвращает
		 количество элементов с параметрами фильтрации по измерениям.


		 ![](../../Property_Image.gif)
		 [CubeSetup](ICubeExecuteFilterSetup.CubeSetup.htm)


		 Свойство CubeSetup
		 возвращает параметры измерений куба, для которых задаются параметры
		 фильтрации.


		 ![](../../Property_Image.gif)
		 [Item](ICubeExecuteFilterSetup.Item.htm)


		 Свойство Item возвращает
		 параметры фильтрации по измерению для указанного элемента.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ICubeExecuteFilterSetup.Add.htm)


		 Метод Add создает новый
		 элемент с параметрами фильтрации и возвращает его параметры.


		 ![](../../Sub_Image.gif)
		 [Clear](ICubeExecuteFilterSetup.Clear.htm)


		 Метод Clear удаляет
		 все элементы с параметрами фильтрации из коллекции.


		 ![](../../Sub_Image.gif)
		 [Move](ICubeExecuteFilterSetup.Move.htm)


		 Метод Move осуществляет
		 перемещение элементов с параметрами фильтрации в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](ICubeExecuteFilterSetup.Remove.htm)


		 Метод Remove удаляет
		 элемент с параметрами фильтрации из коллекции и возвращает признак
		 удачного удаления.
