# ICubeExecuteFilterSetup.Move


## Синтаксис


Move(IndexFrom: Integer; IndexTo: Integer);


## Параметры


IndexFrom. Индекс позиции,
 из которой необходимо переместить элемент.


IndexTo. Индекс позиции, в
 которую необходимо переместить элемент.


## Описание


Метод Move осуществляет перемещение
 элементов с параметрами фильтрации в коллекции.
