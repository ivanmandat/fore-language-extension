# IStandardCubeInstance.SegmentContainer


## Синтаксис


SegmentContainer: [ICubeSegmentContainer](../ICubeSegmentContainer/ICubeSegmentContainer.htm);


## Описание


Свойство SegmentContainer возвращает
 контейнер сегментов для куба.


## Комментарии


Свойство возвращает контейнер, который был определён в свойстве [IStandardCube.SegmentContainer](../IStandardCube/IStandardCube.SegmentContainer.htm)
 при настройке куба.
