# ICubeCoordConvertor.GetStdCubeSourceCoord


## Синтаксис


GetStdCubeSourceCoord(Coord: Array; Var SourceCoord:
 Array): Integer;


## Параметры


Coord. Массив, содержащий ключи
 элементов измерений, формирующих координату в исходном кубе.


SourceCoord. Массив, в который
 будут помещены ключи элементов измерений, формирующих координату в кубе-источнике.


## Описание


Метод GetStdCubeSourceCoord
 вычисляет координату в кубе-источнике в соответствии с переданной координатой
 в исходном кубе.


## Комментарии


Метод оставлен для совместимости. При разработке рекомендуется использовать
 метод [ICubeCoordConvertor.GetStdCubeSourceCoordI](ICubeCoordConvertor.GetStdCubeSourceCoordI.htm).
