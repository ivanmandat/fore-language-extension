# ICubeCoordConvertor


Сборка: Cubes;


## Описание


Интерфейс ICubeCoordConvertor
 предназначен для получения координат в кубе-источнике.


## Иерархия наследования


ICubeCoordConvertor


## Комментарии


Объект для получения координаты в кубе-источнике можно получить, используя
 свойство [ICubeExecuteResult.CoordConvertor](../ICubeExecuteResult/ICubeExecuteResult.CoordConvertor.htm).
 Интерфейс актуален, если расчёт осуществлялся для виртуального куба или
 куба-представления.
