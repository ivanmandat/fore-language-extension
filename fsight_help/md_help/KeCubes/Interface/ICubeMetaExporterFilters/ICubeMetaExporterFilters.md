# ICubeMetaExporterFilters


## Описание


Интерфейс ICubeMetaExporterFilters
 содержит свойства и методы для работы с коллекцией фильтров экспорта показателей.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeMetaExporterFilters.Count.htm)
		 Свойство Count возвращает
		 количество элементов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ICubeMetaExporterFilters.Item.htm)
		 Свойство Item возвращает
		 элемент коллекции по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ICubeMetaExporterFilters.Add.htm)
		 Метод Add осуществляет
		 добавление элемента в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](ICubeMetaExporterFilters.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех элементов из коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](ICubeMetaExporterFilters.Remove.htm)
		 Метод Remove осуществляет
		 удаление элемента из коллекции по индексу.
