# IRubricatorFactorExecutor.WhereIsEmpty


## Синтаксис


WhereIsEmpty: Boolean;


## Описание


Свойство WhereIsEmpty
 определяет, извлекать ли данные пустых рядов.


## Комментарии


Допустимые значения:


	- True. Извлекать данные
	 пустых рядов;


	- False. Не извлекать
	 данные пустых рядов.
