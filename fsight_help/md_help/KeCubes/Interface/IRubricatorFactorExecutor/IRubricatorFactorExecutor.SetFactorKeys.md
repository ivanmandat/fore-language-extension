# IRubricatorFactorExecutor.SetFactorKeys


## Синтаксис


SetFactorKeys(Value: Array);


## Параметры


Value. Массив ключей временных
 рядов.


## Описание


Метод SetFactorKeys определяет
 ключи временных рядов, которые должны быть рассчитаны.


## Комментарии


Если свойство [IRubricatorFactorIO.MultipleFactors](../IRubricatorFactorIO/IRubricatorFactorIO.MultipleFactors.htm)
 имеет значение False, то рассчитывается
 только показатель, чей ключ располагается первым в массиве.
