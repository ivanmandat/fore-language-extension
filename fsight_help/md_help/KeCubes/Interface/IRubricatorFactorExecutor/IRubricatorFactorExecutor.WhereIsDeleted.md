# IRubricatorFactorExecutor.WhereIsDeleted


## Синтаксис


WhereIsDeleted: Boolean;


## Описание


Свойство WhereIsDeleted определяет,
 извлекать ли данные удаленных рядов.


## Комментарии


Допустимые значения:


	- True. Извлекать данные
	 удаленных рядов;


	- False. Не извлекать
	 данные удаленных рядов.
