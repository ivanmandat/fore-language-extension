# IStandardCubeRelation


## Описание


Интерфейс IStandardCubeRelation
 содержит свойства отношения стандартного куба.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Relation](IStandardCubeRelation.Relation.htm)
		 Свойство Relation возвращает
		 выражение отношения куба по заданному измерению.
