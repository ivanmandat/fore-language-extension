# IMDCalculationFormulasConverter.Convert


## Синтаксис


Convert;


## Описание


Метод Convert осуществляет конвертацию формул из многомерного расчета в вычисляемый куб.


## Комментарии


Перед конвертацией рекомендуется [проверить соответствие](IMDCalculationFormulasConverter.ValidateStructure.htm) структур многомерного расчета и вычисляемого куба.


После конвертации необходимо [сохранить](../ICalculatedCubeInstance/ICalculatedCubeInstance.SaveFormulas.htm) измененные формулы вычисляемого куба.


## Пример


Пример использования свойства приведен в описании метода [CreateFormulasConverter](../IMDCalculationInstance/IMDCalculationInstance.CreateFormulasConverter.htm).
