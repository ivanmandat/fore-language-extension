# IMDCalculationFormulasConverter.CleanBeforeConvert


## Синтаксис


CleanBeforeConvert: Boolean;


## Описание


Свойство CleanBeforeConvert определяет признак очистки списка формул вычисляемого куба перед конвертацией.


## Комментарии


Если свойству установлено значение True, то перед конвертацией будут удалены все формулы в вычисляемом кубе. По умолчанию свойству установлено значение False.
