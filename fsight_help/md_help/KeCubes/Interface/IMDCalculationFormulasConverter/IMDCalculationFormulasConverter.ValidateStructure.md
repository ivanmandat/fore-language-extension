# IMDCalculationFormulasConverter.ValidateStructure


## Синтаксис


ValidateStructure;


## Описание


Метод ValidateStructure осуществляет проверку соответствия структур многомерного расчета и вычисляемого куба.


## Комментарии


Если структуры многомерного расчета и вычисляемого куба различны, то во время выполнения метода будет сгенерирована исключительная ситуация.
