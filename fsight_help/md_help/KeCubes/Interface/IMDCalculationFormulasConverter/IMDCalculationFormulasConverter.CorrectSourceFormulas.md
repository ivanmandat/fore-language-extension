# IMDCalculationFormulasConverter.CorrectSourceFormulas


## Синтаксис


CorrectSourceFormulas: Boolean;


## Описание


Свойство CorrectSourceFormulas определяет, будет ли производиться коррекция формул перед конвертацией в формулы [вычисляемого куба](IMDCalculationFormulasConverter.Cube.htm).


## Комментарии


По умолчанию свойство имеет значение False.
