# IMDCalculationFormulasConverter.BatchMode


## Синтаксис


BatchMode: [ConvertBatchMode](../../Enums/ConvertBatchMode.htm);


## Описание


Свойство BatchMode определяет способ обновления коллекции формул при конвертации.


## Комментарии


По умолчанию обновляются существующие и добавляются отсутствующие формулы.

Пример использования свойства приведен в описании метода [CreateFormulasConverter](../IMDCalculationInstance/IMDCalculationInstance.CreateFormulasConverter.htm).
