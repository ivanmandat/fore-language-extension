# IMDCalculationFormulasConverter.SourceFacts


## Синтаксис


SourceFacts(Index: Integer): Integer;


## Параметры

## Описание


Свойство SourceFacts возвращает индекс измерения, являющегося в указанном источнике измерением показателей.
