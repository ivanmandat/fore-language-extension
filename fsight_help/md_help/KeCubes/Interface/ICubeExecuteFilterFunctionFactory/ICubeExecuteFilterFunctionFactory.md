# ICubeExecuteFilterFunctionFactory


Сборка: Cubes;


## Описание


Интерфейс ICubeExecuteFilterFunctionFactory
 содержит метод для создания новых функций, которые будут использоваться
 для настройки фильтрации данных.


## Иерархия наследования


ICubeExecuteFilterFunctionFactory


## Комментарии


Для использования метода [CreatePredefined](ICubeExecuteFilterFunctionFactory.CreatePredefined.htm)
 приведите объект, описываемый интерфейсом [ICubeExecuteFilterSetup](../ICubeExecuteFilterSetup/ICubeExecuteFilterSetup.htm)
 к интерфейсу ICubeExecuteFilterFunctionFactory.
