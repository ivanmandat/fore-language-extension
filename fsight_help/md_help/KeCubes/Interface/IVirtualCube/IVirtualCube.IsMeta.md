# IVirtualCube.IsMeta


## Синтаксис


IsMeta: Boolean;


## Описание


Свойство IsMeta определяет,
 будет ли использоваться измерение метаиерархии в виртуальном кубе.


## Комментарии


Допустимые значения:


	- True. Измерение метаиерархии
	 будет использоваться в виртуальном кубе;


	- False. По умолчанию.
	 Измерение метаиерархии не будет использоваться в виртуальном кубе.


Измерение метаиерархии - это новый вид составного измерения виртуальных
 кубов.


Измерение метаиерархии может быть одно, при этом оно заменяет обычное
 составное измерение.


Измерение метаиерархии поддерживается только, если у виртуального куба
 один источник - база данных временных рядов.
