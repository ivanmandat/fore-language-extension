# IMDCalculationFormula


## Описание


Интерфейс IMDCalculationFormula
 содержит свойства и методы для работы с формулой, действующей в указанный
 период времени.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DateBegin](IMDCalculationFormula.DateBegin.htm)
		 Свойство DateBegin
		 возвращает дату начала периода действия формулы.


		 ![](../../Property_Image.gif)
		 [DateEnd](IMDCalculationFormula.DateEnd.htm)
		 Свойство DateEnd возвращает
		 дату окончания периода действия формулы.


		 ![](../../Property_Image.gif)
		 [Expression](IMDCalculationFormula.Expression.htm)
		 Свойство Expression
		 возвращает выражение формулы, по которому будет производиться
		 расчет.


		 ![](../../Property_Image.gif)
		 [Index](IMDCalculationFormula.Index.htm)
		 Свойство Index возвращает
		 индекс формулы в списке.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ChangePeriod](IMDCalculationFormula.ChangePeriod.htm)
		 Метод ChangePeriod
		 осуществляет изменение даты начала и окончания периода действия
		 формулы, а также корректировку с периодами действия других формул.


		 ![](../../Sub_Image.gif)
		 [Remove](IMDCalculationFormula.Remove.htm)
		 Метод Remove осуществляет
		 удаление формулы.
