# ICubeMetaExporterBinding.ValueDataTypePrecision


## Синтаксис


ValueDataTypePrecision: Integer;


## Описание


Свойство ValueDataTypePrecision определяет точность данных показателя.


## Комментарии


Данное свойство актуально при экспорте данных в новые таблицы, расположенные на сервере СУБД. Точность имеющихся значений атрибутов не должна превышать значение, указанное в данном свойстве.
