# ICubeLoaderInstance


## Описание


Интерфейс ICubeLoaderInstance
 содержит свойства и методы для перегрузки данных из куба в куб с помощью
 объекта репозитория "Загрузчик в куб".


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Destination](ICubeLoaderInstance.Destination.htm)
		 Свойство Destination
		 возвращает параметры открытого куба-приёмника, используемые для
		 окончательной настройки перед загрузкой данных.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ApplyDimensions](ICubeLoaderInstance.ApplyDimensions.htm)
		 Метод ApplyDimensions
		 осуществляет применение параметров управления отметкой в кубе-приёмнике.


		 ![](../../Sub_Image.gif)
		 [CreateApplyDimensions](ICubeLoaderInstance.CreateApplyDimensions.htm)
		 Метод CreateApplyDimensions
		 создает объект, используемый для управления отметкой измерений
		 куба-приёмника.


		 ![](../../Sub_Image.gif)
		 [Execute](ICubeLoaderInstance.Execute.htm)
		 Метод Execute осуществляет
		 загрузку данных в куб.
