# ITableCreator.DataRow


## Синтаксис


DataRow: Integer;


## Описание


Свойство DataRow определяет индекс строки в источнике данных, начиная с которой будет производиться загрузка данных в таблицу.


## Комментарии


Индексация строк начинается с нуля. По умолчанию свойству установлено значение ноль, при этом загрузка данных начинается с первой строки, следующей за заголовком (если заголовок присутствует в указанном [источнике данных](ITableCreator.Source.htm)).
