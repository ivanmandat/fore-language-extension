# ITableCreator.Fields


## Синтаксис


Fields: Object;


## Описание


Свойство Fields возвращает коллекцию полей источника данных, на основании которых будут созданы поля в таблице.


## Комментарии


Данное свойство возвращает коллекцию полей, описываемую интерфейсом [IDtFieldDefinitions](KeDt.chm::/Interface/IDtFieldDefinitions/IDtFieldDefinitions.htm).
