# ITableCreator.Source


## Синтаксис


Source: Object;


## Описание


Свойство Source определяет источник, на основании которого будет создана таблица.


## Комментарии


В качестве значения данного свойства должен быть указан источник данных, описываемый интерфейсом [IDtProvider](KeDt.chm::/Interface/IDtProvider/IDtProvider.htm). Источник должен быть закрыт. На основании его структуры автоматически будет сформирована структура таблицы. После создания таблицы, также из указанного источника будут загружены данные.
