# ITableCreator.RowCount


## Синтаксис


RowCount: Integer;


## Описание


Свойство RowCount определяет количество строк данных, которые необходимо загрузить в созданную таблицу.


## Комментарии


По умолчанию свойству установлено значение [Integer.MaxValue](ForeSys.chm::/Class/Integer/Integer.MaxValue.htm)., при этом из указанного источника в таблицу будут загружены все данные.
