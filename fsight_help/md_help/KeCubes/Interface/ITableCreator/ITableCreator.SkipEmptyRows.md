# ITableCreator.SkipEmptyRows


## Синтаксис


SkipEmptyRows: Boolean;


## Описание


Свойство SkipEmptyRows определяет
 признак пропуска пустых строк.


## Комментарии


Допустимые значения:


	- True. Пропускать строки,
	 в которых отсутствуют все значения.


	- False. Не пропускать
	 пустые строки, добавлять пустые записи в таблицу (Значение по умолчанию).
