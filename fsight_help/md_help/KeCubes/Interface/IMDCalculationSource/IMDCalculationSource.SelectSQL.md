# IMDCalculationSource.SelectSQL


## Синтаксис


SelectSQL: String;


## Описание


Свойство SelectSQL возвращает SQL-запрос, используемый для выборки данных из таблиц БД.


## Комментарии


Актуально, если в качестве источника данных используется [Запрос](UiDb.chm::/Relational_object/Query/UiDb_relational_query.htm). Если в качестве источника данных установлена [Таблица](UiDb.chm::/Relational_object/Table/UiDb_relational_table.htm), либо [Представление](UiDb.chm::/Relational_object/conception/UiDb_relational_conception.htm) - свойство возвращает наименование схемы и физическое имя таблицы, из которой берутся данные.
