# ICubeAttachmentEdited.AttributeValueByIndex


## Синтаксис


		AttributeValueByIndex(Index: Integer): Variant;


## Параметры


Index.
 Индекс атрибута в справочнике вложений.


## Описание


Свойство AttributeValueByIndex
 определяет значение атрибута с указанным индексом из справочника с вложениями.
