# ICubeAttachmentEdited.AttributeValueByKey


## Синтаксис


		AttributeValueByKey(AttributeKey: Integer):
		 Variant;


## Параметры


AttributeKey.
 Ключ атрибута в справочнике вложений.


## Описание


Свойство AttributeValueByKey
 определяет значение атрибута с указанным ключом из справочника с вложениями.
