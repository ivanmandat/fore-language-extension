# ICubeAttachmentEdited


Сборка: Cubes;


## Описание


Интерфейс ICubeAttachmentEdited
 предназначен для редактирования вложения, хранящегося по координате куба.


## Иерархия наследования


ICubeAttachmentEdited


## Комментарии


Для получения вложения, открытого на редактирование, используйте свойства
 и методы интерфейса [ICubeAttachmentsEdited](../ICubeAttachmentsEdited/ICubeAttachmentsEdited.htm).
