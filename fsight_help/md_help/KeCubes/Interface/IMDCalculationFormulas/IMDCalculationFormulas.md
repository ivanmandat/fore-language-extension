# IMDCalculationFormulas


## Описание


Интерфейс IMDCalculationFormulas
 содержит свойства и методы для работы с коллекцией формул, действующих
 в различные периоды времени.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMDCalculationFormulas.Count.htm)
		 Свойство Count возвращает
		 количество формул.


		 ![](../../Property_Image.gif)
		 [Item](IMDCalculationFormulas.Item.htm)
		 Свойство Item возвращает
		 параметры формулы с указанным индексом.


		 ![](../../Property_Image.gif)
		 [Modified](IMDCalculationFormulas.Modified.htm)
		 Свойство Modified возвращает
		 признак присутствия изменений в формулах.


		 ![](../../Property_Image.gif)
		 [TimeDependent](IMDCalculationFormulas.TimeDependent.htm)
		 Свойство TimeDependent
		 возвращает признак зависимости формул от времени.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IMDCalculationFormulas.Add.htm)
		 Метод Add создаёт новую
		 формулу.


		 ![](../../Sub_Image.gif)
		 [CheckPeriods](IMDCalculationFormulas.CheckPeriods.htm)
		 Метод CheckPeriods
		 осуществляет проверку на наличие формул с пересекающимися периодами
		 действия.


		 ![](../../Sub_Image.gif)
		 [Clear](IMDCalculationFormulas.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех формул.


		 ![](../../Sub_Image.gif)
		 [Remove](IMDCalculationFormulas.Remove.htm)
		 Метод Remove осуществляет
		 удаление формулы с указанным индексом.
