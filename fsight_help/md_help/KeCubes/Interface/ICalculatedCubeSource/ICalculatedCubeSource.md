# ICalculatedCubeSource


## Описание


Интерфейс ICalculatedCubeSource
 содержит свойства источника данных вычисляемого куба.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dimensions](ICalculatedCubeSource.Dimensions.htm)
		 Свойство Dimensions
		 возвращает коллекцию измерений куба-источника данных вычисляемого
		 куба.


		 ![](../../Property_Image.gif)
		 [Source](ICalculatedCubeSource.Source.htm)
		 Свойство Source возвращает
		 структуру источника данных виртуального куба.
