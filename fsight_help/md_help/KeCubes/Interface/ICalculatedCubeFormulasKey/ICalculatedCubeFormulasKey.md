# ICalculatedCubeFormulasKey


## Описание


Интерфейс ICalculatedCubeFormulasKey
 содержит свойства и методы для работы с уникальным ключом, который соответствует
 формулам по какой-либо координате в кубе.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Coord](ICalculatedCubeFormulasKey.Coord.htm)
		 Свойство Coord возвращает
		 уникальный ключ координаты в кубе, которой соответствует данный
		 ключ формул.


		 ![](../../Property_Image.gif)
		 [Cube](ICalculatedCubeFormulasKey.Cube.htm)
		 Свойство Cube возвращает
		 куб, координате в котором, соответствует данный уникальный ключ
		 формул.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ToFormulas](ICalculatedCubeFormulasKey.ToFormulas.htm)
		 Метод ToFormulas осуществляет
		 преобразование и возвращает параметры формул, которым соответствует
		 данный уникальный ключ.
