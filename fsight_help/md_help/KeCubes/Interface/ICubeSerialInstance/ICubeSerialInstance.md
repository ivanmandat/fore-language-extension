# ICubeSerialInstance


## Описание


Интерфейс ICubeSerialInstance
 содержит свойства и методы для сохранения куба в файл.


## Свойства


		 Имя метода
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Data](ICubeSerialInstance.Data.htm)


		 Свойство Data возвращает
		 полные данные для указанного варианта отображения.


## Свойства, унаследованные от [ICubeInstance](../ICubeInstance/ICubeInstance.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Cube](../ICubeInstance/ICubeInstance.Cube.htm)


		 Свойство Cube возвращает
		 параметры модели куба.


		 ![](../../Property_Image.gif)
		 [Destinations](../ICubeInstance/ICubeInstance.Destinations.htm)


		 Свойство Destinations
		 возвращает параметры вариантов отображения отстроенного куба.


		 ![](../../Property_Image.gif)
		 [Sources](../ICubeInstance/ICubeInstance.Sources.htm)


		 Свойство Sources возвращает
		 параметры отстроенных кубов, являющихся источниками данных для
		 данного куба.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Save](ICubeSerialInstance.Save.htm)


		 Метод Save осуществляет
		 сохранение данных куба в файл.


## Методы, унаследованные от [ICubeInstance](../ICubeInstance/ICubeInstance.htm)


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [GetExtensionDispatch](../ICubeInstance/ICubeInstance.GetExtensionDispatch.htm)


		 Метод GetExtensionDispatch
		 осуществляет преобразование к общей форме объекта, от которой
		 в дальнейшем можно перейти к необходимому классу объектов.
