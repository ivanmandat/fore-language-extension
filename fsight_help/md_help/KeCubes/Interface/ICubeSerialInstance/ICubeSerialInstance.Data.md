# ICubeSerialInstance.Data


## Синтаксис


Data(Destination: [ICubeInstanceDestination](../ICubeInstanceDestination/ICubeInstanceDestination.htm)):
 [IMatrix](KeMatrix.chm::/Interface/IMatrix/IMatrix.htm);


## Параметры


Destination. Вариант отображения
 куба, в соответствии с которым осуществляется расчет сохраняемых данных.


## Описание


Свойство Data возвращает полные
 данные для указанного варианта отображения.
