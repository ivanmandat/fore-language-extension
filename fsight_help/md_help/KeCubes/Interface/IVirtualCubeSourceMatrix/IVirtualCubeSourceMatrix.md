# IVirtualCubeSourceMatrix


Сборка: Cubes;


## Описание


Интерфейс IVirtualCubeSourceMatrix
 предназначен для получения информации о матрице с данными, которая рассчитана
 для источника виртуального куба.


## Иерархия наследования


IVirtualCubeSourceMatrix


## Комментарии


Информацию о матрице источника можно получить, используя свойства и
 методы интерфейса [IVirtualCubeSourceMatricesList](../IVirtualCubeSourceMatricesList/IVirtualCubeSourceMatricesList.htm).
