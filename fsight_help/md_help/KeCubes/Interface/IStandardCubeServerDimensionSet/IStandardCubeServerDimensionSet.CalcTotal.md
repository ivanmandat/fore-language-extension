# IStandardCubeServerDimensionSet.CalcTotal


## Синтаксис


CalcTotal: Boolean;


## Описание


Свойство CalcTotal определяет,
 рассчитывать ли итоги для группы измерений.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Рассчитывать итоги;


	- False. Не рассчитывать
	 итоги.
