# IStandardCubeServerDimensionSet


## Описание


Интерфейс IStandardCubeServerDimensionSet
 содержит свойства и методы для работы с группой измерений, по которой
 на сервере рассчитывается агрегация.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CalcOther](IStandardCubeServerDimensionSet.CalcOther.htm)
		 Свойство CalcOther
		 определяет, рассчитывать ли агрегацию для элементов измерений,
		 не вошедших в отметку (агрегат Other).


		 ![](../../Property_Image.gif)
		 [CalcTotal](IStandardCubeServerDimensionSet.CalcTotal.htm)
		 Свойство CalcTotal
		 определяет, рассчитывать ли итоги для группы измерений.


		 ![](../../Property_Image.gif)
		 [Dimensions](IStandardCubeServerDimensionSet.Dimensions.htm)
		 Свойство Dimensions
		 возвращает измерения, входящие в группу.
