# ICubeCreatorFacts


Сборка: Cubes;


## Описание


Интерфейс ICubeCreatorFacts
 содержит свойства и методы для работы с коллекцией фактов создаваемого
 куба.


## Иерархия наследования


ICubeCreatorFacts


## Комментарии


Коллекцию фактов можно получить в свойстве [ICubeCreator.Facts](../ICubeCreator/ICubeCreator.Facts.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeCreatorFacts.Count.htm)
		 Свойство Count возвращает
		 количество фактов в коллекции.


		 ![](../../Property_Image.gif)
		 [CreateExternalDimension](ICubeCreatorFacts.CreateExternalDimension.htm)
		 Свойство CreateExternalDimension
		 определяет, создавать ли справочник для хранения фактов.


		 ![](../../Property_Image.gif)
		 [HideTrivialFactDimension](ICubeCreatorFacts.HideTrivialFactDimension.htm)
		 Свойство HideTrivialFactDimension
		 определяет, будет ли скрываться измерение фактов при построении
		 куба.


		 ![](../../Property_Image.gif)
		 [Item](ICubeCreatorFacts.Item.htm)
		 Свойство Item возвращает
		 параметры указанного факта.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ICubeCreatorFacts.Add.htm)
		 Метод Add создает новый
		 факт в коллекции и возвращает его параметры.


		 ![](../../Sub_Image.gif)
		 [Clear](ICubeCreatorFacts.Clear.htm)
		 Метод Clear удаляет
		 все факты из коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](ICubeCreatorFacts.Remove.htm)
		 Метод Remove удаляет
		 из коллекции указанный факт.
