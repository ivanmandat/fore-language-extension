# ICubeCreatorFacts.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс факта.


## Описание


Метод Remove удаляет из коллекции факт, индекс которого передается посредством параметра Index.


## Комментарии


Метод возвращает значение True, если факт был удален и False, если удаление указанного факта недопустимо.
