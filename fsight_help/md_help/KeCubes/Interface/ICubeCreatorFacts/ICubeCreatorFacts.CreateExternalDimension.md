# ICubeCreatorFacts.CreateExternalDimension


## Синтаксис


CreateExternalDimension: Boolean;


## Описание


Свойство CreateExternalDimension
 определяет, создавать ли справочник для хранения фактов.


## Комментарии


Допустимые значения:


	- True. Для хранения фактов
	 будет создан отдельный справочник;


	- False. Факты будут храниться
	 локально.
