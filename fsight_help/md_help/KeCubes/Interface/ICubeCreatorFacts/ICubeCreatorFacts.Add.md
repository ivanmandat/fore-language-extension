# ICubeCreatorFacts.Add


## Синтаксис


Add(Name: String): [ICubeCreatorFact](../ICubeCreatorFact/ICubeCreatorFact.htm);


## Параметры


Name. Наименование факта.


## Описание


Метод Add создает новый факт в коллекции и возвращает его параметры.
