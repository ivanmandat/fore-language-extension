# ICubeLoaderMapInfo


## Описание


Интерфейс ICubeLoaderMapInfo
 содержит свойства и методы для работы с коллекцией привязок измерений
 куба-приёмника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeLoaderMapInfo.Count.htm)
		 Свойство Count возвращает
		 количество измерений куба-приёмника, которые необходимо связать
		 с измерениями куба-источника.


		 ![](../../Property_Image.gif)
		 [Item](ICubeLoaderMapInfo.Item.htm)
		 Свойство Item возвращает
		 параметры привязки измерения-приёмника.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ClearAll](ICubeLoaderMapInfo.ClearAll.htm)
		 Метод ClearAll осуществляет
		 очистку привязок всех измерений куба-приёмника.


		 ![](../../Sub_Image.gif)
		 [FindByDim](ICubeLoaderMapInfo.FindByDim.htm)
		 Метод FindByDim осуществляет
		 поиск измерения и возвращает параметры привязки измерения-приёмника
		 в случае удачного поиска.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ICubeLoaderMapInfo.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск по ключу и возвращает параметры привязки измерения-приёмника
		 в случае удачного поиска.
