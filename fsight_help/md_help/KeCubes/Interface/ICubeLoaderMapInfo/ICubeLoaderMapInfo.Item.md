# ICubeLoaderMapInfo.Item


## Синтаксис


Item(Index: Integer): [ICubeLoaderMapItem](../ICubeLoaderMapItem/ICubeLoaderMapItem.htm);


## Параметры


Index - индекс измерения-приёмника, для которого необходимо настроить привязку.


## Описание


Свойство Item возвращает параметры привязки измерения-приёмника, индекс которого передается посредством параметра Index.
