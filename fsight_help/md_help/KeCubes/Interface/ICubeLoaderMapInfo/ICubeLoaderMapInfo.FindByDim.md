# ICubeLoaderMapInfo.FindByDim


## Синтаксис


FindByDim(Dimension: [IDimensionModel](kedims.chm::/interface/idimensionmodel/idimensionmodel.htm)): [ICubeLoaderMapItem](../ICubeLoaderMapItem/ICubeLoaderMapItem.htm);


## Параметры


Dimension - измерение, которое необходимо найти в списке измерений куба-приёмника.


## Описание


Метод FindByDim осуществляет поиск и возвращает параметры привязки измерения-приёмника в случае удачного поиска.
