# ICubeLoaderMapInfo.FindByKey


## Синтаксис


FindByKey(Key: Integer): [ICubeLoaderMapItem](../ICubeLoaderMapItem/ICubeLoaderMapItem.htm);


## Параметры


Key - ключ измерения, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет поиск и возвращает параметры привязки измерения-приёмника в случае удачного поиска. Ключ, по которому осуществляется поиск, передается посредством параметра Key.
