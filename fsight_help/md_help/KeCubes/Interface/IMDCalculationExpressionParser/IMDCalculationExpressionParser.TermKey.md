# IMDCalculationExpressionParser.TermKey


## Синтаксис


TermKey(Term: String): [IMDCalculationFormulaElementKey](../IMDCalculationFormulaElementKey/IMDCalculationFormulaElementKey.htm);


## Параметры


Term. Символьное представление
 терма, которое необходимо сопоставить с элементом источника данных.


## Описание


Свойство TermKey сопоставляет
 терму, заданному в виде символьной строки, уникальный ключ элемента источника
 данных.


## Комментарии


При разборе выражения вместо терма будет подставляться сопоставленный
 элемент.
