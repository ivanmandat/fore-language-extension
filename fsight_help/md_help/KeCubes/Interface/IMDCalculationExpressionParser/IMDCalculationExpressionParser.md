# IMDCalculationExpressionParser


## Описание


Интерфейс IMDCalculationExpressionParser
 содержит свойства и методы для распознавания формул из текстовой строки.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Calculation](IMDCalculationExpressionParser.Calculation.htm)
		 Свойство Calculation
		 возвращает данные многомерного расчёта на сервере БД.


		 ![](../../Property_Image.gif)
		 [Expression](IMDCalculationExpressionParser.Expression.htm)
		 Свойство Expression
		 возвращает выражение, полученное в результате разбора строки [Text](IMDCalculationExpressionParser.Text.htm).


		 ![](../../Property_Image.gif)
		 [TermKey](IMDCalculationExpressionParser.TermKey.htm)
		 Свойство TermKey сопоставляет
		 терму, заданному в виде символьной строки, уникальный ключ элемента
		 источника/приёмника данных.


		 ![](../../Property_Image.gif)
		 [TermSum](IMDCalculationExpressionParser.TermSum.htm)
		 Свойство TermSum определяет
		 способ суммирования значений дочерних элементов для элемента,
		 сопоставленного указанному терму.


		 ![](../../Property_Image.gif)
		 [Text](IMDCalculationExpressionParser.Text.htm)
		 Свойство Text определяет
		 текст формулы, которую необходимо разобрать.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ClearTermKeys](IMDCalculationExpressionParser.ClearTermKeys.htm)
		 Метод ClearTermKeys
		 осуществляет сброс всех сопоставлений элементов источников данных
		 с термами.


		 ![](../../Sub_Image.gif)
		 [Parse](IMDCalculationExpressionParser.Parse.htm)
		 Метод Parse осуществляет
		 разбор формулы, заданной в виде текстовой строки.
