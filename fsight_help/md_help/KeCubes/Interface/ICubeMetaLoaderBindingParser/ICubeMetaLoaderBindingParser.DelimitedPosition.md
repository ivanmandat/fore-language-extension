# ICubeMetaLoaderBindingParser.DelimitedPosition


## Синтаксис


DelimitedPosition: Integer;


## Описание


Свойство DelimitedPosition определяет
 порядковый номер разделителя атрибутов в списке.


## Комментарии


Список разделителей определяется свойством [ICubeMetaLoaderBindingParser.DelimiterChars](ICubeMetaLoaderBindingParser.DelimiterChars.htm).
 Свойство DelimitedPosition учитывается,
 если значение [ICubeMetaLoaderBindingParser.ParserType](ICubeMetaLoaderBindingParser.ParserType.htm)
 - [CubeMetaLoaderParserType.Delimited](../../Enums/CubeMetaLoaderParserType.htm).
