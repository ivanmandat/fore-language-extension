# ICubeMetaLoaderBindingParser.StringPositionStart


## Синтаксис


StringPositionStart: Integer;


## Описание


Свойство StringPositionStart
 определяет начальную позицию атрибута в строке.


## Комментарии


Конечная позиция атрибута в строке определяется свойством [ICubeMetaLoaderBindingParser.StringPositionEnd](ICubeMetaLoaderBindingParser.StringPositionEnd.htm).


Свойство StringPositionStart
 учитывается, если значение [ICubeMetaLoaderBindingParser.ParserType](ICubeMetaLoaderBindingParser.ParserType.htm)
 - [CubeMetaLoaderParserType.FixedPosition](../../Enums/CubeMetaLoaderParserType.htm).
