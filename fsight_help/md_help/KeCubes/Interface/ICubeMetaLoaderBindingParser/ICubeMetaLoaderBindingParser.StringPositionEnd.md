# ICubeMetaLoaderBindingParser.StringPositionEnd


## Синтаксис


StringPositionEnd: Integer;


## Описание


Свойство StringPositionEnd определяет
 конечную позицию атрибута в строке.


## Комментарии


Если значение данного свойства «-1», то окончание атрибута совпадает
 с окончанием строки. Начальная позиция атрибута в строке определяется
 свойством [ICubeMetaLoaderBindingParser.StringPositionStart](ICubeMetaLoaderBindingParser.StringPositionStart.htm).


Свойство StringPositionEnd учитывается,
 если значение [ICubeMetaLoaderBindingParser.ParserType](ICubeMetaLoaderBindingParser.ParserType.htm)
 - [CubeMetaLoaderParserType.FixedPosition](../../Enums/CubeMetaLoaderParserType.htm).
