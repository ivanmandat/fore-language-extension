# ICubeMetaLoaderBindingParser


## Описание


Интерфейс ICubeMetaLoaderBindingParser
 содержит свойства для распознавания атрибутов при настройке привязок к
 полю источника данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CaptureGroup](ICubeMetaLoaderBindingParser.CaptureGroup.htm)
		 Свойство CaptureGroup
		 определяет порядковый номер группы в регулярном выражении.


		 ![](../../Property_Image.gif)
		 [DelimitedPosition](ICubeMetaLoaderBindingParser.DelimitedPosition.htm)
		 Свойство DelimitedPosition
		 определяет порядковый номер разделителя атрибутов в списке.


		 ![](../../Property_Image.gif)
		 [DelimiterChars](ICubeMetaLoaderBindingParser.DelimiterChars.htm)
		 Свойство DelimiterChars
		 определяет список разделителей атрибутов.


		 ![](../../Property_Image.gif)
		 [ParserType](ICubeMetaLoaderBindingParser.ParserType.htm)
		 Свойство ParserType
		 определяет порядок распознавания атрибутов в источнике данных.


		 ![](../../Property_Image.gif)
		 [RegExp](ICubeMetaLoaderBindingParser.RegExp.htm)
		 Свойство RegExp определяет
		 регулярное выражение.


		 ![](../../Property_Image.gif)
		 [ReplaceMap](ICubeMetaLoaderBindingParser.ReplaceMap.htm)
		 Свойство ReplaceMap
		 возвращает карту символьных строк, которая определяет замены для
		 загружаемых значений.


		 ![](../../Property_Image.gif)
		 [SkipList](ICubeMetaLoaderBindingParser.SkipList.htm)
		 Свойство SkipList возвращает
		 массив строк, определяющий значения, которые не будут загружены.


		 ![](../../Property_Image.gif)
		 [StringPositionEnd](ICubeMetaLoaderBindingParser.StringPositionEnd.htm)
		 Свойство StringPositionEnd
		 определяет конечную позицию атрибута в строке.


		 ![](../../Property_Image.gif)
		 [StringPositionStart](ICubeMetaLoaderBindingParser.StringPositionStart.htm)
		 Свойство StringPositionStart
		 определяет начальную позицию атрибута в строке.
