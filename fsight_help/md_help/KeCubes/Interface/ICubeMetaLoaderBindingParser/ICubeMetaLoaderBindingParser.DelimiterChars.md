# ICubeMetaLoaderBindingParser.DelimiterChars


## Синтаксис


DelimiterChars: String;


## Описание


Свойство DelimiterChars определяет
 список разделителей атрибутов.


## Комментарии


Данное свойство учитывается, если значение [ICubeMetaLoaderBindingParser.ParserType](ICubeMetaLoaderBindingParser.ParserType.htm)
 - [CubeMetaLoaderParserType.Delimited](../../Enums/CubeMetaLoaderParserType.htm).
