# ICubeCacheItems


Сборка: Cubes;


## Описание


Интерфейс ICubeCacheItems предназначен
 для работы с коллекцией экземпляров кеша.


## Иерархия наследования


ICubeCacheItems


## Комментарии


Экземпляры кеша можно получить, используя методы [ICubeCacheSaver.GetItemsFromDatabase](../ICubeCacheSaver/ICubeCacheSaver.GetItemsFromDatabase.htm),
 [ICubeCacheSaver.GetItemsFromMemory](../ICubeCacheSaver/ICubeCacheSaver.GetItemsFromMemory.htm).
