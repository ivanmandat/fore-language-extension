# IMDCalculationScheduledTask


## Описание


Интерфейс IMDCalculationScheduledTask
 содержит свойства и методы для работы с задачей выполнения многомерного
 расчета на сервере БД.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [SourceCalculation](IMDCalculationScheduledTask.SourceCalculation.htm)
		 Свойство SourceCalculation
		 определяет многомерный расчет на сервере БД, который необходимо
		 выполнить.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [LoadCalculationArgs](IMDCalculationScheduledTask.LoadCalculationArgs.htm)
		 Метод LoadCalculationArgs
		 возвращает параметры выполнения многомерного расчета.


		 ![](../../Sub_Image.gif)
		 [SetCalculationArgs](IMDCalculationScheduledTask.SetCalculationArgs.htm)
		 Метод SetCalculationArgs
		 устанавливает параметры выполнения многомерного расчета.
