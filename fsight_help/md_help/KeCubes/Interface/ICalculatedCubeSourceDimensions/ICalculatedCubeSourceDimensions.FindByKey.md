# ICalculatedCubeSourceDimensions.FindByKey


## Синтаксис


FindByKey(Key: Integer): [ICalculatedCubeDimension](../ICalculatedCubeDimension/ICalculatedCubeDimension.htm);


## Параметры


Key - ключ измерения куба-источника данных.


## Описание


Метод FindByKey осуществляет поиск измерения, кчюч которого, передается посредством параметра Key.
