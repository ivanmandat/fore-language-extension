# ICalculatedCubeSourceDimensions.FindById


## Синтаксис


FindById(Id: String): [ICalculatedCubeDimension](../ICalculatedCubeDimension/ICalculatedCubeDimension.htm);


## Параметры


Id - идентификатор измерения куба-источника данных.


## Описание


Метод FindById осуществляет поиск измерения, идентификатор которого, передается посредством параметра Id.
