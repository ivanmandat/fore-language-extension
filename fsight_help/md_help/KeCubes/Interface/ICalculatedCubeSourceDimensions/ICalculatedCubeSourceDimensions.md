# ICalculatedCubeSourceDimensions


## Описание


Интерфейс ICalculatedCubeSourceDimensions
 содержит свойства и методы для работы с измерениями куба, являющегося
 источником данных для вычисляемого куба.


## Комментарии


Коллекцию измерений куба-источника возвращает свойство [ICalculatedCubeSource.Dimensions](../ICalculatedCubeSource/ICalculatedCubeSource.Dimensions.htm).


Интерфейс ICalculatedCubeSourceDimensions
 является базовым для интерфейса [ICalculatedCubeDimensions](../ICalculatedCubeDimensions/ICalculatedCubeDimensions.htm).
