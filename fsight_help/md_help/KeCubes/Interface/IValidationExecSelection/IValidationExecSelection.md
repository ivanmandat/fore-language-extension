# IValidationExecSelection


## Описание


Интерфейс IValidationExecSelection
 содержит свойства и методы для работы с отметкой, при получении данных
 расчета валидации.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [FactorsCount](IValidationExecSelection.FactorsCount.htm)


		 Свойство FactorsCount
		 возвращает количество показателей, для которых должны быть получены
		 данные.


		 ![](../../Property_Image.gif)
		 [MultipleFactors](IValidationExecSelection.MultipleFactors.htm)


		 Свойство MultipleFactors
		 возвращает признак того, что данные должны быть получены для показателей.


		 ![](../../Property_Image.gif)
		 [Selection](IValidationExecSelection.Selection.htm)


		 Свойство Selection
		 возвращает отметку, по которой должны быть получены данные.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [GetFactorKeys](IValidationExecSelection.GetFactorKeys.htm)


		 Метод GetFactorKeys
		 возвращает массив ключей показателей, для которых должны
		 быть получены данные.
