# IMDCalculationFacts.FindById


## Синтаксис


FindById(Id: String): [IMDCalculationFact](../IMDCalculationFact/IMDCalculationFact.htm);


## Параметры


Id - идентификатор показателя.


## Описание


Метод FindById осуществляет поиск и возвращает свойства показателя источника/приёмника данных. Идентификатор, по которому осуществляется поиск, передается посредством параметра Id.
