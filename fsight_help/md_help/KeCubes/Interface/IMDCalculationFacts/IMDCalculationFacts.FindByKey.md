# IMDCalculationFacts.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IMDCalculationFact](../IMDCalculationFact/IMDCalculationFact.htm);


## Параметры


Key - ключ показателя.


## Описание


Метод FindByKey осуществляет поиск и возвращает свойства показателя источника/приёмника данных. Ключ, по которому осуществляется поиск, передается посредством параметра Key.
