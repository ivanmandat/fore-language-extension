# IMDCalculationFacts.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key - ключ показателя, который необходимо удалить.


## Описание


Метод RemoveByKey осуществляет удаление показателя, ключ которого передается посредством параметра Key.
