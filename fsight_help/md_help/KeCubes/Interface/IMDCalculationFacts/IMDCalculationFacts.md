# IMDCalculationFacts


## Описание


Интерфейс IMDCalculationFacts
 содержит свойства и методы для работы с коллекцией показателей связанных
 с полями источника/приёмника данных, содержащими данные, используемые
 при расчёте.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMDCalculationFacts.Count.htm)
		 Свойство Count возвращает
		 количество показателей источника/приёмника данных.


		 ![](../../Property_Image.gif)
		 [Item](IMDCalculationFacts.Item.htm)
		 Свойство Item возвращает
		 свойства показателя.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IMDCalculationFacts.Add.htm)
		 Метод Add осуществляет
		 создание нового показателя источника/приёмника данных.


		 ![](../../Sub_Image.gif)
		 [Clear](IMDCalculationFacts.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех показателей источника/приёмника данных.


		 ![](../../Sub_Image.gif)
		 [FindById](IMDCalculationFacts.FindById.htm)
		 Метод FindById осуществляет
		 поиск показателя по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IMDCalculationFacts.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск показателя по ключу.


		 ![](../../Sub_Image.gif)
		 [Move](IMDCalculationFacts.Move.htm)
		 Метод Move осуществляет
		 перемещение показателей.


		 ![](../../Sub_Image.gif)
		 [Remove](IMDCalculationFacts.Remove.htm)
		 Метод Remove осуществляет
		 удаление показателя по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IMDCalculationFacts.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление показателя по ключу.
