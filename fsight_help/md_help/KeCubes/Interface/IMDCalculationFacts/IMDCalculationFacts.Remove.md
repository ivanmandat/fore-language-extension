# IMDCalculationFacts.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс показателя, который необходимо удалить.


## Описание


Метод Remove осуществляет удаление показателя, индекс которого передается посредством параметра Index.
