# IMDCalculationSliceFilter


## Описание


Интерфейс IMDCalculationSliceFilter
 содержит свойства для настройки фильтрации элементов, по которым будет
 производиться расчет.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Selection](IMDCalculationSliceFilter.Selection.htm)
		 Свойство Selection
		 определяет отметку элементов по которым будет производиться расчет.


		 ![](../../Property_Image.gif)
		 [SQL](IMDCalculationSliceFilter.SQL.htm)
		 Свойство SQL определяет
		 SQL-условие используемое для отбора элементов, по которым будет
		 производиться расчет, в выбранном измерении.


		 ![](../../Property_Image.gif)
		 [Type](IMDCalculationSliceFilter.Type.htm)
		 Свойство Type определяет
		 способ фильтрации, применяемый для выбора элементов, по которым
		 будет производиться расчет.
