# ICubeCreator.UseNameAttribute


## Синтаксис


UseNameAttribute: Boolean;


## Описание


Свойство UseNameAttribute определяет,
 использовать ли атрибут «Наименование»
 при создании куба.


## Комментарии


Допустимые значения:


	- True. Атрибут «Наименование» используется при
	 создании куба;


	- False. Атрибут «Наименование» не используется при
	 создании куба.
