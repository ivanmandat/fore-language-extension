# ICubeCreator.DictionaryParent


## Синтаксис


DictionaryParent: [IMetabaseObject](kesom.chm::/interface/imetabaseobject/imetabaseobject.htm);


## Описание


Свойство DictionaryParent определяет
 папку, в которой будут создаваться объекты, необходимые для куба.


## Комментарии


В папке будут созданы табличные справочники, календарный справочник
 и прочие объекты.


Если задано значение данного свойства, то куб может быть создан без
 указания репозитория НСИ в свойстве [ICubeCreator.RdsDatabase](ICubeCreator.RdsDatabase.htm).
 Достаточно задать папку и базу данных. Для задания базы данных используйте
 свойство [ICubeCreator.Database](ICubeCreator.Database.htm).
