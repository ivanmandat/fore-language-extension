# IRubricatorTimeSegment


## Описание


Интерфейс IRubricatorTimeSegment
 содержит свойства и методы для работы с временным диапазоном данных базы
 данных временных рядов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DateBegin](IRubricatorTimeSegment.DateBegin.htm)
		 Свойство DateBegin
		 определяет дату начала временного диапазона данных.


		 ![](../../Property_Image.gif)
		 [DateEnd](IRubricatorTimeSegment.DateEnd.htm)
		 Свойство DateEnd определяет
		 дату окончания временного диапазона данных.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [SetPeriod](IRubricatorTimeSegment.SetPeriod.htm)
		 Метод SetPeriod устанавливает
		 границы временного диапазона данных.
