# ICubeModelSources.FindByKey


## Синтаксис


FindByKey(SourceKey: Integer): [ICubeModelSource](../ICubeModelSource/ICubeModelSource.htm);


## Параметры


SourceKey - ключ куба, который необходимо найти среди источников данных данного куба.


## Описание


Метод FindByKey осущестлвяет поиск в коллекции источников данных куба, ключ которого передается посредством параметра SourceKey.
