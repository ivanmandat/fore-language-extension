# ICubeModelSources


## Описание


Интерфейс ICubeModelSources
 содержит свойства и методы для работы с коллекций кубов-источников данных.


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeModelSources.Count.htm)
		 Свойство Count возвращает
		 количество кубов-источников данных для данного куба.


		 ![](../../Property_Image.gif)
		 [Item](ICubeModelSources.Item.htm)
		 Свойство Item возвращает
		 куб-источник данных.


## Методы


		  Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindByKey](ICubeModelSources.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск куба-источника по ключу.
