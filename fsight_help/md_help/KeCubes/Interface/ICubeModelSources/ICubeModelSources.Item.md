# ICubeModelSources.Item


## Синтаксис


Item(Index: Integer): [ICubeModelSource](../ICubeModelSource/ICubeModelSource.htm);


## Параметры


Index - индекс куба-источника данных.


## Описание


Свойство Item возвращает куб-источник данных, индекс которого передается посредством параметра Index.
