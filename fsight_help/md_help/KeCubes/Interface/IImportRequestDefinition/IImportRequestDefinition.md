# IImportRequestDefinition


## Описание


Интерфейс IImportRequestDefinition
 содержит свойства для работы с объектом, содержащим параметры импорта
 показателей.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DestinationRubricator](IImportRequestDefinition.DestinationRubricator.htm)
		 Свойство DestinationRubricator
		 определяет базу данных временных рядов, в которую будут импортированы
		 показатели.


		 ![](../../Property_Image.gif)
		 [DestinationRubricatorInstance](IImportRequestDefinition.DestinationRubricatorInstance.htm)
		 Свойство DestinationRubricatorInstance
		 определяет данные базы данных временных рядов, в которые будут
		 импортированы показатели.


		 ![](../../Property_Image.gif)
		 [ProviderParams](IImportRequestDefinition.ProviderParams.htm)
		 Свойство ProviderParams
		 возвращает объект, содержащий параметры импорта показателей из
		 источника данных (кроме базы данных временных рядов).


		 ![](../../Property_Image.gif)
		 [RubricatorParams](IImportRequestDefinition.RubricatorParams.htm)
		 Свойство RubricatorParams
		 возвращает объект, содержащий параметры импорта показателей из
		 базы данных временных рядов.


		 ![](../../Property_Image.gif)
		 [SourceType](IImportRequestDefinition.SourceType.htm)
		 Свойство SourceType
		 определяет тип источника для импорта.
