# ICubeMetaCopierBindings


## Описание


Интерфейс ICubeMetaCopierBindings
 содержит свойства и методы для работы с коллекцией, определяющие привязку
 атрибутов базы-источника с атрибутами базы-приёмника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeMetaCopierBindings.Count.htm)
		 Свойство Count возвращает
		 количество элементов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ICubeMetaCopierBindings.Item.htm)
		 Свойство Item возвращает
		 элемент коллекции по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ICubeMetaCopierBindings.Add.htm)
		 Метод Add осуществляет
		 добавление элемента в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](ICubeMetaCopierBindings.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех элементов из коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](ICubeMetaCopierBindings.Remove.htm)
		 Метод Remove осуществляет
		 удаление элемента из коллекции по индексу.
