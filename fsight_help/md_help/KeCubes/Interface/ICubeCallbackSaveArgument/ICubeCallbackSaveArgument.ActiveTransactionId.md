# ICubeCallbackSaveArgument.ActiveTransactionId


## Синтаксис


		ActiveTransactionId: String;


## Описание


Свойство ActiveTransactionId
 возвращает идентификатор текущей транзакции.


## Комментарии


Для идентификации транзакций используются статические 128-битные уникальные
 идентификаторы.


Свойство будет доступно, если сохранение данных производится с использованием
 метода [ICubeInstanceStorage.SaveMatrixT](../ICubeInstanceStorage/ICubeInstanceStorage.SaveMatrixT.htm).
