# ICubeCallbackSaveArgument


Сборка: Cubes;


## Описание


Интерфейс ICubeCallbackSaveArgument
 предназначен для получения информации о результатах сохранения данных
 в куб.


## Иерархия наследования


ICubeCallbackSaveArgument


## Комментарии


Интерфейс описывает аргумент события [ICubeSaveMatrixCallback.OnAfterSave](../ICubeSaveMatrixCallback/ICubeSaveMatrixCallback.OnAfterSave.htm).

		 [ActiveTransactionId](ICubeCallbackSaveArgument.ActiveTransactionId.htm)

		 Свойство LogKey возвращает

		 Свойство SecurityConnections
		 возвращает массив соединений с серверами баз данных, в которые

		 ![](../../Property_Image.gif)
		 [ValueFlag](ICubeCallbackSaveArgument.ValueFlag.htm)


		 Свойство ValueFlag
		 возвращает флаг, которым отмечены изменившиеся данные, сохранённые

		 ![](../../Property_Image.gif)
		 [ValueFlags](ICubeCallbackSaveArgument.ValueFlags.htm)


		 Свойство ValueFlags
		 возвращает массив флагов, которыми отмечены изменившиеся данные,
		 сохранённые в куб.


		 ![](../../Property_Image.gif)
		 [ValueMatrix](ICubeCallbackSaveArgument.ValueMatrix.htm)


		 Свойство ValueMatrix
		 возвращает матрицу с данными, которая была передана при сохранении
		 данных в куб.
