# ICubeCallbackSaveArgument.ValueFlags


## Синтаксис


		ValueFlags: Array;


## Описание


Свойство ValueFlags возвращает
 массив флагов, которыми отмечены изменившиеся данные, сохранённые в куб.


## Комментарии


Свойство возвращает тот массив флагов, который был указан в методе [ICubeInstanceStorage.SaveMatrixEx](../ICubeInstanceStorage/ICubeInstanceStorage.SaveMatrixEx.htm).
