# ICubeCallbackSaveArgument.ValueFlag


## Синтаксис


		ValueFlag: Integer;


## Описание


Свойство ValueFlag возвращает
 флаг, которым отмечены изменившиеся данные, сохранённые в куб.


## Комментарии


Свойство возвращает тот флаг, который был указан в методе [ICubeInstanceStorage.SaveMatrix](../ICubeInstanceStorage/ICubeInstanceStorage.SaveMatrix.htm) или
 [ICubeInstanceStorage.SaveMatrixT](../ICubeInstanceStorage/ICubeInstanceStorage.SaveMatrixT.htm).
