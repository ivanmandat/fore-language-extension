# ICubeCreatorDimension.Dimension


## Синтаксис


Dimension: [IDimensionModel](kedims.chm::/interface/idimensionmodel/idimensionmodel.htm);


## Описание


Свойство Dimension возвращает справочник, который соответствует данному измерению.


## Комментарии


Свойство актуально, если измерение включено в коллекцию с помощью метода [ICubeCreatorDimensions.Add](../ICubeCreatorDimensions/ICubeCreatorDimensions.Add.htm). Если текущее измерение создано с помощью метода [ICubeCreatorDimensions.AddNew](../ICubeCreatorDimensions/ICubeCreatorDimensions.AddNew.htm), то свойство Dimension вернет значение Null.
