# ICubeCreatorDimension.CubeDimension


## Синтаксис


CubeDimension: [IStandardCubeDimension](../IStandardCubeDimension/IStandardCubeDimension.htm);


## Описание


Свойство CubeDimension определяет
 измерение куба, для которого настраивается привязка.


## Комментарии


Свойство используется, если необходимо изменить структуру существующего
 куба, в который осуществляется загрузка данных.
