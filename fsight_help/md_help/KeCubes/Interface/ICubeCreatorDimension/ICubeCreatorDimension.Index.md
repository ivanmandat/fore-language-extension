# ICubeCreatorDimension.Index


## Синтаксис


Index: [IDimIndex](kedims.chm::/interface/idimindex/idimindex.htm);


## Описание


Свойство Index определяет индекс справочника, который будет использоваться для привязки к полям наборов данных.


## Комментарии


Если для создателя куба установлено свойство [ICubeCreator.Table](../ICubeCreator/ICubeCreator.Table.htm) или [ICubeCreatorFact.Table](../ICubeCreatorFact/ICubeCreatorFact.Table.htm), то в коллекции [Fields](ICubeCreatorDimension.Fields.htm) каждому атрибуту индекса также необходимо сопоставить поле таблицы.


По умолчанию используется первичный индекс справочника.
