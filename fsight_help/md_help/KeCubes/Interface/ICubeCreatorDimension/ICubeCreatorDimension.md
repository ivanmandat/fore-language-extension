# ICubeCreatorDimension


Сборка: Cubes;


## Описание


Интерфейс ICubeCreatorDimension
 содержит свойства измерения, которое будет добавлено в структуру куба
 при его создании.


## Иерархия наследования


ICubeCreatorDimension


## Комментарии


Данный интерфейс содержит свойства, позволяющие провести базовую настройку,
 необходимую для автоматического создания и привязки измерения.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CubeDimension](ICubeCreatorDimension.CubeDimension.htm)


		 Свойство CubeDimension
		 определяет измерение куба, для которого настраивается привязка.


		 ![](../../Property_Image.gif)
		 [Dimension](ICubeCreatorDimension.Dimension.htm)


		 Свойство Dimension
		 возвращает справочник, который соответствует данному измерению.


		 ![](../../Property_Image.gif)
		 [Fields](ICubeCreatorDimension.Fields.htm)


		 Свойство Fields возвращает
		 коллекцию полей, используемых для привязки измерения.


		 ![](../../Property_Image.gif)
		 [Index](ICubeCreatorDimension.Index.htm)


		 Свойство Index определяет
		 индекс справочника, который будет использоваться для привязки
		 к полям наборов данных.


		 ![](../../Property_Image.gif)
		 [Name](ICubeCreatorDimension.Name.htm)


		 Свойство Name определяет
		 наименование измерения.
