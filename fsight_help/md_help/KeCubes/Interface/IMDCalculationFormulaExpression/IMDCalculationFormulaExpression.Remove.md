# IMDCalculationFormulaExpression.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс операнда, который необходимо удалить.


## Описание


Метод Remove осуществляет удаление операнда, индекс позиции которого передается посредством параметра Index.
