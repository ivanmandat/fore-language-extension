# IMDCalculationFormulaExpression


## Описание


Интерфейс IMDCalculationFormulaExpression
 содержит свойства и методы для работы с выражением формулы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMDCalculationFormulaExpression.Count.htm)
		 Свойство Count возвращает
		 количество операндов в выражении формулы.


		 ![](../../Property_Image.gif)
		 [Item](IMDCalculationFormulaExpression.Item.htm)
		 Свойство Item возвращает
		 операнд выражения функции.


		 ![](../../Property_Image.gif)
		 [Modified](IMDCalculationFormulaExpression.Modified.htm)
		 Свойство Modified возвращает
		 признак присутствия изменений в выражении формулы.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Check](IMDCalculationFormulaExpression.Check.htm)
		 Метод Check осуществляет
		 проверку корректности выражения формулы.


		 ![](../../Sub_Image.gif)
		 [Clear](IMDCalculationFormulaExpression.Clear.htm)
		 Метод Clear осуществляет
		 очистку выражения формулы.


		 ![](../../Sub_Image.gif)
		 [CopyTo](IMDCalculationFormulaExpression.CopyTo.htm)
		 Метод CopyTo осуществляет
		 копирование текущего выражения формулы в заданное выражение.


		 ![](../../Sub_Image.gif)
		 [CreateItem](IMDCalculationFormulaExpression.CreateItem.htm)
		 Метод CreateItem создает
		 новый операнд формулы.


		 ![](../../Sub_Image.gif)
		 [InsertItem](IMDCalculationFormulaExpression.InsertItem.htm)
		 Метод InsertItem осуществляет
		 вставку указанного операнда в выражение формулы.


		 ![](../../Sub_Image.gif)
		 [Remove](IMDCalculationFormulaExpression.Remove.htm)
		 Метод Remove осуществляет
		 удаление операнда расположенного в указанной позиции.
