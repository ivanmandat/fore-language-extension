# IMDCalculationFormulaExpression.CopyTo


## Синтаксис


CopyTo(Value: [IMDCalculationFormulaExpression](IMDCalculationFormulaExpression.htm); [Index: Integer = -1]);


## Параметры


Value - выражение формулы, в которое необходимо скопировать текущее выражение.


Index - необязательный параметр, определяющий позицию вставки текущего выражения. По умолчанию вставка осуществляется в конец выражения Value.


## Описание


Метод CopyTo осуществляет копирование текущего выражения формулы в выражение, передаваемое посредством параметра Value.
