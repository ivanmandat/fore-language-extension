# IMDCalculationFormulaExpression.InsertItem


## Синтаксис


InsertItem(Value: [IMDCalculationFormulaOperand](../IMDCalculationFormulaOperand/IMDCalculationFormulaOperand.htm);
 [Index: Integer = -1]): [IMDCalculationFormulaOperand](../IMDCalculationFormulaOperand/IMDCalculationFormulaOperand.htm);


## Параметры


Value. Операнд, который необходимо
 вставить в выражение формулы.


Index. Необязательный параметр,
 определяющий позицию вставки операнда. По умолчанию операнды вставляются
 в конец выражения.


## Описание


Метод InsertItem осуществляет
 вставку указанного операнда в выражение формулы.
