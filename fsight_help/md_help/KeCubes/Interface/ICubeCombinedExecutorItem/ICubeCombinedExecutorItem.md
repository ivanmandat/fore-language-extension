# ICubeCombinedExecutorItem


Сборка: Cubes;


## Описание


Интерфейс ICubeCombinedExecutorItem
 содержит свойства для получения результатов параллельного расчёта результирующих
 матриц кубов.


## Иерархия наследования


ICubeCombinedExecutorItem


## Комментарии


Параллельный расчёт кубов осуществляется при вызове метода [ICubeCombinedExecutor.Execute](../ICubeCombinedExecutor/ICubeCombinedExecutor.Execute.htm).
 Результат расчёта возвращает свойство [ICubeCombinedExecutor.Item](../ICubeCombinedExecutor/ICubeCombinedExecutor.Item.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Cube](ICubeCombinedExecutorItem.Cube.htm)


		 Свойство Cube возвращает
		 экземпляр куба, для которого производился расчёт.


		 ![](../../Property_Image.gif)
		 [Destination](ICubeCombinedExecutorItem.Destination.htm)


		 Свойство Destination
		 возвращает вариант отображения куба, для которого производился
		 расчёт.


		 ![](../../Property_Image.gif)
		 [Matrix](ICubeCombinedExecutorItem.Matrix.htm)


		 Свойство Matrix возвращает
		 матрицу с данными, полученную в результате расчёта.


		 ![](../../Property_Image.gif)
		 [MultiExecuted](ICubeCombinedExecutorItem.MultiExecuted.htm)


		 Свойство MultiExecuted
		 возвращает признак получения текущего результата при параллельном
		 расчёте.


		 ![](../../Property_Image.gif)
		 [Selection](ICubeCombinedExecutorItem.Selection.htm)


		 Свойство Selection
		 возвращает отметку, в соответствии с которой производился расчёт.
