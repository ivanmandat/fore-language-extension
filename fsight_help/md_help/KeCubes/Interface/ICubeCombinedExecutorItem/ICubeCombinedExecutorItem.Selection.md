# ICubeCombinedExecutorItem.Selection


## Синтаксис


		Selection: [IDimSelectionSet](KeDims.chm::/interface/IDimSelectionSet/IDimSelectionSet.htm);


## Описание


Свойство Selection возвращает
 отметку, в соответствии с которой производился расчёт.


## Комментарии


Свойство вернёт ту отметку, которая была передана в методе [ICubeCombinedExecutor.AddCube](../ICubeCombinedExecutor/ICubeCombinedExecutor.AddCube.htm)
 или [ICubeCombinedExecutor.AddCubeDest](../ICubeCombinedExecutor/ICubeCombinedExecutor.AddCubeDest.htm).
