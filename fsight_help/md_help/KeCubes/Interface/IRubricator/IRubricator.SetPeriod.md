# IRubricator.SetPeriod


## Синтаксис


SetPeriod(StartYear: Integer; EndYear: Integer);


## Параметры


StartYear. Год начала для показателей;


EndYear. Год окончания для показателей.


## Описание


Метод SetPeriod устанавливает период для показателей.


## Комментарии


Текущие года начала и окончания для показателей возвращают свойства [IRubricator.StartYear](IRubricator.StartYear.htm) и [IRubricator.EndYear](IRubricator.EndYear.htm).
