# IRubricator.UseDuplication


## Синтаксис


UseDuplication: Boolean;


## Описание


Свойство UseDuplication определяет,
 множить ли значения атрибута ряда на все точки, отмеченные в выборке.


## Комментарии


Выборка формируется только для атрибутного [варианта
 отображения](../IRubricatorCubeDestination/IRubricatorCubeDestination.htm) базы данных временных рядов. Множить атрибуты ряда на
 всю все точки означает, что значения атрибутов будут доступны для всех
 наблюдений, вне зависимости от того, есть они в базе данных временных
 рядов или нет. Например, в выборку попали наблюдения ряда с 2010 по 2015
 год, а в базе данных временных рядов есть наблюдения ряда только с 2011
 по 2015 год. Матрица данных, полученная для данной выборке, не содержит
 значения для наблюдения за 2010 год. Если не множить значения атрибутов,
 то в выбранном варианте отображения для наблюдения за 2010 год будут отсутствовать
 значения атрибутов. Для избежания данной ситуации рекомендуется множить
 значения атрибутов.


Значения атрибутов ряда множатся только при выборке. Если в выборку
 попали ряды с разной динамикой, то атрибуты ряда будут множиться только
 для того уровня календаря, к которому относится ряд. Групповые атрибуты
 множатся по всем динамикам, если атрибут DL
 не идентифицирующий.


Допустимые значения:


	- True. Значения атрибутов
	 множатся на все точки ряда;


	- False. Значение по умолчанию.
	 Значения атрибутов не множатся.
