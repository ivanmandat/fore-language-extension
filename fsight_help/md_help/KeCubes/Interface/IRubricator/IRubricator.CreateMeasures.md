# IRubricator.CreateMeasures


## Синтаксис


CreateMeasures;


## Описание


Метод CreateMeasures создает
 справочник мер в [репозитории НСИ](IRubricator.Database.htm).
 Данный метод используется на этапе создания базы данных временных рядов,
 в случае если в репозитории НСИ, еще не созданы базовые справочники.
