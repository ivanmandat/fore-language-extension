# IRubricator.EndYear


## Синтаксис


EndYear: Integer;


## Описание


Свойство EndYear возвращает год окончания для показателей.


## Комментарии


Для изменения значения EndYear используйте метод [IRubricator.SetPeriod](IRubricator.SetPeriod.htm).
