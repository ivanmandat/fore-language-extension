# IRubricator.KeepHistory


## Синтаксис


KeepHistory: Boolean;


## Описание


Свойство KeepHistory определяет,
 вести ли историю изменений показателей базы данных временных рядов.


## Комментарии


Допустимые значения:


	- True. История изменений
	 ведется, т.е., база данных временных рядов версионная. Значение
	 по умолчанию при создании базы данных временных рядов;


	- False. История изменений
	 не ведется. Если база данных временных рядов содержит большой объем
	 данных, то отключение версионности позволит оптимизировать работу
	 базы.


Для корректного преобразования версионной базы данных временных рядов
 в неверсионный сожмите все ревизии в одну, используя метод [IRubricatorInstance.CutRevisions](../IRubricatorInstance/IRubricatorInstance.CutRevisions.htm). Если
 ревизии не будут сжаты, то преобразование в неверсионный режим недоступно.
