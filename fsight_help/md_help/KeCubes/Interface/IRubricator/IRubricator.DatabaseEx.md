# IRubricator.DatabaseEx


## Синтаксис


DatabaseEx: [IDatabase](KeDb.chm::/Interface/IDatabase/IDatabase.htm);


## Описание


Свойство DatabaseEx определяет
 базу данных для базы данных временных рядов.


## Комментарии


Если данное свойство не определено, то будет использоваться база данных
 репозитория НСИ, указанного в свойстве [IRubricator.Database](IRubricator.Database.htm).
