# IRubricator.StartYear


## Синтаксис


StartYear: Integer;


## Описание


Свойство StartYear возвращает год начала для показателей.


## Комментарии


Для изменения значения StartYear используйте метод [IRubricator.SetPeriod](IRubricator.SetPeriod.htm).
