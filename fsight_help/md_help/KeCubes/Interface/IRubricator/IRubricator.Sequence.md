# IRubricator.Sequence


## Синтаксис


Sequence: [ISequence](KeDb.chm::/Interface/ISequence/ISequence.htm);


## Описание


Свойство Sequence возвращает объект, генерирующий ключи показателей для базы данных временных рядов.


## Комментарии


Если свойство [IRubricator.UseDistinctSequence](IRubricator.UseDistinctSequence.htm) = False, то Sequence возвращает значение NULL.
