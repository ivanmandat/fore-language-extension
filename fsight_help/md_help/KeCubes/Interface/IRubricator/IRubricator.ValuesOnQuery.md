# IRubricator.ValuesOnQuery


## Синтаксис


ValuesOnQuery: Boolean;


## Описание


Свойство ValuesOnQuery определяет, будет ли справочник значений основан на запросе.


## Комментарии


Допустимые значения:


-
True. Справочник значений основан на запросе. Текст запроса определяет свойство [IMetaAttributes.SelectQueryText](KeRds.chm::/Interface/IMetaAttributes/IMetaAttributes.SelectQueryText.htm), возвращаемое свойством [IRubricator.Values](IRubricator.Values.htm).[Attributes](KeRds.chm::/Interface/IMetaDictionary/IMetaDictionary.Attributes.htm);


-
False. Справочник значений не основан на запросе.
