# IRubricator.CompoundFactorKey


## Синтаксис


CompoundFactorKey: Boolean;


## Описание


Свойство CompoundFactorKey определяет, будут ли временные ряды иметь составной ключ.


## Комментарии


Допустимые значения:


-
True. Временные ряды имеют составной ключ;


-
False. Ключ временных рядов не является составным.
