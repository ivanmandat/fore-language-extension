# IRubricator.AutoUnits


## Синтаксис


AutoUnits: Boolean;


## Описание


Свойство AutoUnits определяет
 наличие измерения единиц измерения в базе данных временных рядов.


## Комментарии


Допустимые значения:


	- True. Измерение присутствует;


	- False. Измерение отсутствует.
