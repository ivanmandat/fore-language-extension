# IRubricator.Truncate


## Синтаксис


Truncate;


## Описание


Метод Truncate осуществляет
 очистку базы данных временных рядов.


## Комментарии


При выполнении метода:


	- очищаются таблицы с данными базы данных временных рядов;


	- сбрасываются [последовательности](IRubricator.Sequence.htm)
	 таблиц. Если у базы данных временных рядов своя последовательность,
	 то она сбрасывается, если не своя – не сбрасывается;


	- перестраиваются индексы в таблицах.


## Пример


Для выполнения примера предполагается наличие в репозитории базы данных
 временных рядов с идентификатором TSDB.


Добавьте ссылки на системные сборки: Cubes, Metabase.


	Sub UserProc;

	Var

	    mb: IMetabase;

	    rub: IRubricator;

	Begin

	    mb := MetabaseClass.Active;

	    rub := mb.ItemById("TSDB").Bind As IRubricator;

	    rub.Truncate;

	End Sub UserProc;


После выполнения будет выполнена очистка базы данных временных рядов.
