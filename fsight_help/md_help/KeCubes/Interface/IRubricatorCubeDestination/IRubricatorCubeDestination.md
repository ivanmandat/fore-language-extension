# IRubricatorCubeDestination


Сборка: Cubes;


## Описание


Интерфейс IRubricatorCubeDestination
 предназначен для работы с вариантом отображения базы данных временных
 рядов.


## Иерархия наследования


IRubricatorCubeDestination


## Комментарии


Для работы с вариантом отображения базы данных временных рядов приведите
 объект интерфейса [ICubeModelDestination](../ICubeModelDestination/ICubeModelDestination.htm)
 к интерфейсу IRubricatorCubeDestination.
