# IStandardCubeFactBinding.AggregatedDataTypePrecision


## Синтаксис


AggregatedDataTypePrecision: Integer;


## Описание


Свойство AggregatedDataTypePrecision
 определяет точность значения (количество знаков после запятой) после агрегации.


## Комментарии


Данное свойство учитывается, если значение свойства [IStandardCubeFactBinding.AggregatedDataType](IStandardCubeFactBinding.AggregatedDataType.htm)
 = [CubeFactBindingAggregatedDataType.FixedPoint](../../Enums/CubeFactBindingAggregatedDataType.htm).
