# IStandardCubeFactBinding.DistinctAggregate


## Синтаксис


DistinctAggregate: Boolean;


## Описание


Свойство DistinctAggregate определяет,
 агрегировать ли идентичные данные.


## Комментарии


Допустимые значения:


	- True. Агрегировать только
	 различные значения;


	- False. Агрегировать
	 все значения.
