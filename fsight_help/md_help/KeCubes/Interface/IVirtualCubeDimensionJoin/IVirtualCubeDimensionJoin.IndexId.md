# IVirtualCubeDimensionJoin.IndexId


## Синтаксис


IndexId: Integer;


## Описание


Свойство IndexId возвращает
 идентификатор индекса, который используется при сопоставлении элементов
 двух измерений.


## Комментарии


Свойство возвращает индекс измерения, ключ которого доступен в свойстве
 [DimensionKey](IVirtualCubeDimensionJoin.DimensionKey.htm).
 По значениям атрибутов, входящих в индекс, осуществляется сопоставление
 элементов одного измерения с элементами другого измерения и передача отметки.
