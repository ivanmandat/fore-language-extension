# IVirtualCubeDimensionJoin


Сборка: Cubes;


## Описание


Интерфейс IVirtualCubeDimensionJoin
 содержит свойства для получения информации о сопоставлении измерений.


## Иерархия наследования


IVirtualCubeDimensionJoin


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CubeId](IVirtualCubeDimensionJoin.CubeId.htm)


		 Свойство CubeId возвращает
		 идентификатор куба.


		 ![](../../Property_Image.gif)
		 [DimensionKey](IVirtualCubeDimensionJoin.DimensionKey.htm)


		 Свойство DimensionKey
		 возвращает ключ измерения, которое используется при сопоставлении.


		 ![](../../Property_Image.gif)
		 [IndexId](IVirtualCubeDimensionJoin.IndexId.htm)


		 Свойство IndexId возвращает
		 идентификатор индекса, который используется при сопоставлении
		 элементов двух измерений.
