# IRubricatorValidationsLookup


## Описание


Интерфейс IRubricatorValidationsLookup
 содержит свойства и методы, предназначенные для организации поиска выполнений
 валидаций.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Current](IRubricatorValidationsLookup.Current.htm)
		 Свойство Current возвращает
		 текущий объект в курсоре.


		 ![](../../Property_Image.gif)
		 [Lookup](IRubricatorValidationsLookup.Lookup.htm)
		 Свойство Lookup возвращает
		 условие поиска.


		 ![](../../Property_Image.gif)
		 [Rubricator](IRubricatorValidationsLookup.Rubricator.htm)
		 Свойство Rubricator
		 возвращает данные базы данных временных рядов.


		 ![](../../Property_Image.gif)
		 [WhereValidation](IRubricatorValidationsLookup.WhereValidation.htm)
		 Свойство WhereValidation
		 определяет ключ валидации.
