# ICalculatedCubeExecutor


## Описание


Интерфейс ICalculatedCubeExecutor
 содержит свойства и методы для расчета результирующей матрицы вычисляемого
 куба по заданным координатам.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [EvaluateOnly](ICalculatedCubeExecutor.EvaluateOnly.htm)


		 Свойство EvaluateOnly
		 определяет, по каким координатам будут вычисляться значения результирующей
		 матрицы.


		 ![](../../Property_Image.gif)
		 [IncludeNulls](ICalculatedCubeExecutor.IncludeNulls.htm)


		 Свойство IncludeNulls
		 определяет признак включения пустых значений в результирующую
		 матрицу.


		 ![](../../Property_Image.gif)
		 [Selected](ICalculatedCubeExecutor.Selected.htm)


		 Свойство Selected определяет
		 признак отметки указанной координаты матрицы.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [DeselectAll](ICalculatedCubeExecutor.DeselectAll.htm)


		 Метод DeselectAll снимает
		 отметку со всех координат в матрице.


		 ![](../../Sub_Image.gif)
		 [Execute](ICalculatedCubeExecutor.Execute.htm)


		 Метод Execute осуществляет
		 расчет результирующей матрицы вычисляемого куба.


		 ![](../../Sub_Image.gif)
		 [Matrix](ICalculatedCubeExecutor.Matrix.htm)


		 Метод Matrix возвращает
		 результирующую матрицу вычисляемого куба.


		 ![](../../Sub_Image.gif)
		 [Select_](ICalculatedCubeExecutor.Select_.htm)


		 Метод Select_ осуществляет
		 отметку указанной координаты в матрице, для дальнейшего вычисления
		 значения по данной координате.
