# ICubeMetaLoaderBinding.FieldValue


## Синтаксис


FieldValue: Variant;


## Описание


Свойство FieldValue определяет
 значение поля источника данных.


## Комментарии


Данное свойство учитывается, если свойство [ICubeMetaLoaderBinding.FieldType](ICubeMetaLoaderBinding.FieldType.htm)
 имеет значение [CubeMetaLoaderFieldType.ConstValue](../../Enums/CubeMetaLoaderFieldType.htm).
