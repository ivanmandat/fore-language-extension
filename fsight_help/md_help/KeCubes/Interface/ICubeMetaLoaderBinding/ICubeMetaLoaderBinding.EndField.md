# ICubeMetaLoaderBinding.EndField


## Синтаксис


EndField: String;


## Описание


Свойство EndField определяет
 последнее поле, если элементы источника располагаются в столбцах (значение
 свойства [ICubeMetaLoaderBinding.ByColumns](ICubeMetaLoaderBinding.ByColumns.htm)
 - True).
