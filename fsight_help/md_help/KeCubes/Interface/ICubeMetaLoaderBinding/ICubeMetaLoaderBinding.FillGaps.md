# ICubeMetaLoaderBinding.FillGaps


## Синтаксис


FillGaps: Boolean;


## Описание


Свойство FillGaps определяет,
 заполнять ли пустоты в значениях, которые образуются при импорте объединенных
 ячеек из формата *.xls. True -
 заполнять пустоты в значениях, False
 - не заполнять.
