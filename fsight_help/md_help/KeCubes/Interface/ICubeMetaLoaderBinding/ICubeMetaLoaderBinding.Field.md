# ICubeMetaLoaderBinding.Field


## Синтаксис


Field: String;


## Описание


Свойство Field определяет поле
 источника данных.


## Комментарии


В зависимости от значения свойства [FieldType](ICubeMetaLoaderBinding.FieldType.htm),
 в свойстве Field указывается наименование
 или индекс поля.
