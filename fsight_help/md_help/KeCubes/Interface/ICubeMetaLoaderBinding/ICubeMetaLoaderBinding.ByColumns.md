# ICubeMetaLoaderBinding.ByColumns


## Синтаксис


ByColumns: Boolean;


## Описание


Свойство ByColumns определяет,
 располагаются ли элементы измерения в столбцах.


## Комментарии


Допустимые значения:


	- True. Элементы расположены
	 в столбцах;


	- False. Элементы расположены
	 построчно.
