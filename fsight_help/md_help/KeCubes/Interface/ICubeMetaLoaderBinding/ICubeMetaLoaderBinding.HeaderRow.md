# ICubeMetaLoaderBinding.HeaderRow


## Синтаксис


HeaderRow: Integer;


## Описание


Свойство HeaderRow
 определяет индекс строки, в которой
 расположен заголовок. Если значение данного свойства «-1», то значение
 заголовков определяется источником данных ([ICubeMetaLoader.Data](../ICubeMetaLoader/ICubeMetaLoader.Data.htm)).
