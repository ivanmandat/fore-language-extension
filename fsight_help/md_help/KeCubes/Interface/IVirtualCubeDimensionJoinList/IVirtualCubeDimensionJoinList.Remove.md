# IVirtualCubeDimensionJoinList.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс измерения в списке,
 параметры сопоставления которого необходимо удалить.


## Описание


Метод Remove удаляет параметры
 сопоставления измерения по индексу в списке.


## Комментарии


Метод возвращает значение True,
 если удаление прошло удачно или False,
 если удаление по каким-либо причинам невозможно.
