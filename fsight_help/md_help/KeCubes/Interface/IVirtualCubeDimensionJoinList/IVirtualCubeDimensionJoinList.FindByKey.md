# IVirtualCubeDimensionJoinList.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IVirtualCubeDimensionJoins](../IVirtualCubeDimensionJoins/IVirtualCubeDimensionJoins.htm);


## Параметры


Key. Ключ измерения.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает параметры сопоставления для измерения с указанным ключом.


## Комментарии


Метод возвращает параметры сопоставления в случае удачного поиска и
 Null, если для указанного измерения
 параметры сопоставления не созданы.
