# IVirtualCubeDimensionJoinList.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ измерения, параметры
 сопоставления которого необходимо удалить.


## Описание


Метод RemoveByKey удаляет параметры
 сопоставления по ключу измерения.


## Комментарии


Метод возвращает значение True,
 если удаление прошло удачно или False,
 если удаление по каким-либо причинам невозможно.
