# IVirtualCubeDimensionJoinList.Move


## Синтаксис


Move(IndexFrom: Integer; IndexTo: Integer);


## Параметры


IndexFrom. Индекс позиции,
 из которой осуществляется перемещение.


IndexTo. Индекс позиции, в
 которую осуществляется перемещение.


## Описание


Метод Move осуществляет перемещение
 параметров сопоставления измерения в списке.
