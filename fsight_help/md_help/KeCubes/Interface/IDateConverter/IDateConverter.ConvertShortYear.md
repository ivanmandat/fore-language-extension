# IDateConverter.ConvertShortYear


## Синтаксис


ConvertShortYear: Boolean;


## Описание


Свойство ConvertShortYear определяет
 признак того, преобразовывать ли краткий формат даты.


## Комментарии


Допустимые значения:


	- True. Преобразовывать
	 формат даты;


	- False. Не преобразовывать.
