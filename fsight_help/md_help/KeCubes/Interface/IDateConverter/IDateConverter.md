# IDateConverter


Сборка: Cubes;


## Описание


Интерфейс IDateConverter
 предназначен для задания формата разбора дат.

## Комментарии


Для создания объекта, предназначенного для разбора дат используйте класс
 [DateConverter](../../Class/DateConverter/DateConverter.htm).
