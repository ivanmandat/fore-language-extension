# IValidationExecRun


## Описание


Интерфейс IValidationExecRun
 содержит свойства для работы с данными расчета валидации.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Adhoc](IValidationExecRun.Adhoc.htm)


		 Свойство Adhoc возвращает
		 признак принадлежности данных к текущей валидации.


		 ![](../../Property_Image.gif)
		 [ByScenario](IValidationExecRun.ByScenario.htm)


		 Свойство ByScenario
		 возвращает признак того, что валидация рассчитана в измерении
		 сценариев.


		 ![](../../Property_Image.gif)
		 [Data](IValidationExecRun.Data.htm)


		 Свойство Data возвращает
		 данные о расчёте валидации.


		 ![](../../Property_Image.gif)
		 [EphemeralFacts](IValidationExecRun.EphemeralFacts.htm)


		 Свойство EphemeralFacts
		 определяет, создавать ли новые ряды с признаком удаленности.


		 ![](../../Property_Image.gif)
		 [Matrix](IValidationExecRun.Matrix.htm)


		 Свойство Matrix возвращает
		 данные расчета валидации.


		 ![](../../Property_Image.gif)
		 [MultipleFactors](IValidationExecRun.MultipleFactors.htm)


		 Свойство MultipleFactors
		 возвращает признак того, получены ли данные для нескольких показателей
		 или для всей базы.


		 ![](../../Property_Image.gif)
		 [Revision](IValidationExecRun.Revision.htm)


		 Свойство Revision определяет
		 ревизию для сохранения данных расчета валидации.
