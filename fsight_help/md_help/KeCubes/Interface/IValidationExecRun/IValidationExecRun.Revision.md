# IValidationExecRun.Revision


## Синтаксис


Revision: [IRubricatorRevision](../IRubricatorRevision/IRubricatorRevision.htm);


## Описание


Свойство Revision определяет ревизию для сохранения данных расчета валидации.


## Комментарии


Данная ревизия будет использоваться, если должны сохраняться несуществующие, но возможные ряды. См. свойство: [IValidationExecRun.EphemeralFacts](IValidationExecRun.EphemeralFacts.htm).
