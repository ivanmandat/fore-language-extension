# IValidationExecRun.Adhoc


## Синтаксис


Adhoc: Boolean;


## Описание


Свойство Adhoc возвращает признак принадлежности данных к текущей валидации.


## Комментарии


Свойство возвращает значение True, если данные относятся к текущей валидации и False, если не относятся.
