# IValidationExecRun.Data


## Синтаксис


Data: [IValidationExecData](../IValidationExecData/IValidationExecData.htm);


## Описание


Свойство Data возвращает
 данные о расчёте валидации.


## Комментарии


Данные о расчёте сохраняются, если правило валидации рассчитывается
 для всей БД временных рядов.


Область расчёта валидации определяет свойство [IValidationExecuteSettings.Laner](KeMs.chm::/Interface/IValidationExecuteSettings/IValidationExecuteSettings.Laner.htm).
