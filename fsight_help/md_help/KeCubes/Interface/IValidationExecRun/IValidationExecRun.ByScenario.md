# IValidationExecRun.ByScenario


## Синтаксис


ByScenario: Boolean;


## Описание


Свойство ByScenario возвращает
 признак того, что валидация рассчитана в измерении сценариев.


## Комментарии


Используйте [IValidationExecData.ByScenario](../IValidationExecData/IValidationExecData.ByScenario.htm)
 для расчета валидации в измерении сценариев.
