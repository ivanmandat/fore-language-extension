# IValidationExecRun.MultipleFactors


## Синтаксис


MultipleFactors: Boolean;


## Описание


Свойство MultipleFactors возвращает
 признак того, получены ли данные для нескольких показателей или для всей
 базы.


## Комментарии


Допустимые значения:


	- True. Данные получены
	 для нескольких показателей;


	- False. Данные получены
	 для всей базы.
