# IStandardCubeFactBindings.BindingWithRole


## Синтаксис


		BindingWithRole(FactKey: Variant; BindingRole:
		 [CubeFactBindingRole](../../Enums/CubeFactBindingRole.htm)):
		 [IStandardCubeFactBinding](../IStandardCubeFactBinding/IStandardCubeFactBinding.htm);


## Параметры


FactKey. Ключ элемента измерения
 фактов, привязку которого необходимо получить.


BindingRole. Роль привязки
 в кубе.


## Описание


Свойство BindingWithRole для
 указанного факта куба возвращает привязку с определённой ролью.

Пример использования приведён в описании метода [IStandardCubeDestination.CreateAndSetAttachmentsStorage](../IStandardCubeDestination/IStandardCubeDestination.CreateAndSetAttachmentsStorage.htm).
