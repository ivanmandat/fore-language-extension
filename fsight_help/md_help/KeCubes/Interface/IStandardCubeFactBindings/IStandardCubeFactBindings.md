# IStandardCubeFactBindings


Сборка: Cubes;


## Описание


Интерфейс IStandardCubeFactBindings
 содержит свойства и методы для работы с привязками фактов куба.


## Комментарии


Если коллекция получена в свойстве [IStandardCubeDestination.CalcBindings](../IStandardCubeDestination/IStandardCubeDestination.CalcBindings.htm),
 то каждая отдельная привязка вычисляемого факта для настройки параметров
 может быть приведена к интерфейсу [IStandardCubeCalculatedFactBinding](../IStandardCubeCalculatedFactBinding/IStandardCubeCalculatedFactBinding.htm).
