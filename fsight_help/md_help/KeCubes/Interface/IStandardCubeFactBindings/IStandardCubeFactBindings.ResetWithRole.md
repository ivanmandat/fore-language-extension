# IStandardCubeFactBindings.ResetWithRole


## Синтаксис


		ResetWithRole(FactKey: Variant; BindingRole:
		 [CubeFactBindingRole](../../Enums/CubeFactBindingRole.htm));


## Параметры


FactKey. Ключ элемента измерения
 фактов, привязку которого необходимо сбросить.


BindingRole. Роль привязки
 в кубе.


## Описание


Метод ResetWithRole осуществляет
 очистку привязки факта куба с заданной ролью.
