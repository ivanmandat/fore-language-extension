# IStandardCubeFactBindings.Binding


## Синтаксис


Binding(FactKey: Variant): [IStandardCubeFactBinding](../IStandardCubeFactBinding/IStandardCubeFactBinding.htm);


## Параметры


FactKey. Ключ элемента измерения фактов, привязку
 которого необходимо получить.


## Описание


Свойство Binding возвращает
 привязку факта, ключ которого передается посредством параметра FactKey.
