# IStandardCubeFactBindings.Reset


## Синтаксис


Reset(FactKey: Variant);


## Параметры


FactKey. Ключ элемента измерения
 фактов, для которого необходимо очистить привязку.


## Описание


Метод Reset осуществляет очистку
 привязки факта куба, ключ которого передается посредством параметра FactKey.
