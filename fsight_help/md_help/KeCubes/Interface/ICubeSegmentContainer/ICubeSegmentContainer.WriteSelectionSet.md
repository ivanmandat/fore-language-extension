# ICubeSegmentContainer.WriteSelectionSet


## Синтаксис


WriteSelectionSet(Segment: [ICubeSegment](../ICubeSegment/ICubeSegment.htm);
 SelectionSet: [IDimSelectionSet](KeDims.chm::/interface/IDimSelectionSet/IDimSelectionSet.htm));


## Параметры


Segment. Сегмент куба.


SelectionSet. Отметка, которую
 необходимо сохранить в сегмент.


## Описание


Метод WriteSelectionSet задаёт
 указанную отметку для указанного сегмента куба.
