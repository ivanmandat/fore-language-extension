# ICubeSegmentContainer.ReadSelectionSet


## Синтаксис


ReadSelectionSet(Segment: [ICubeSegment](../ICubeSegment/ICubeSegment.htm);
 SelectionSet: [IDimSelectionSet](KeDims.chm::/interface/IDimSelectionSet/IDimSelectionSet.htm));


## Параметры


Segment. Сегмент куба.


SelectionSet. Отметка, в которую
 осуществляется копирование отметки сегмента.


## Описание


Метод ReadSelectionSet считывает
 отметку из сегмента в указанную отметку.
