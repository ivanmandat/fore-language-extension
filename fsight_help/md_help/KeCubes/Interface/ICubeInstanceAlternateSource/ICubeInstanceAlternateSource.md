# ICubeInstanceAlternateSource


## Описание


Интерфейс ICubeInstanceAlternateSource
 содержит свойства для работы с данными альтернативного источника данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Destination](ICubeInstanceAlternateSource.Destination.htm)
		 Свойство Destination
		 возвращает данные куба, для которого установлен данный альтернативный
		 источник.


		 ![](../../Property_Image.gif)
		 [Dimension](ICubeInstanceAlternateSource.Dimension.htm)
		 Свойство Dimension
		 возвращает данные измерения, добавленного в список измерений,
		 по координатам которых будут загружаться данные из альтернативного
		 куба.


		 ![](../../Property_Image.gif)
		 [DimensionCount](ICubeInstanceAlternateSource.DimensionCount.htm)
		 Свойство DimensionCount
		 возвращает количество измерений, добавленных в список измерений,
		 по координатам которых будут загружаться данные из альтернативного
		 куба.


		 ![](../../Property_Image.gif)
		 [Source](ICubeInstanceAlternateSource.Source.htm)
		 Свойство Source возвращает
		 данные куба, используемого в качестве альтернативного источника
		 данных.
