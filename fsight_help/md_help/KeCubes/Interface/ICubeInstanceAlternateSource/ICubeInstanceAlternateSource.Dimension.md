# ICubeInstanceAlternateSource.Dimension


## Синтаксис


Dimension(Index: Integer): [IDimInstance](kedims.chm::/interface/idiminstance/idiminstance.htm);


## Параметры


Index - индекс измерения.


## Описание


Свойство Dimension возвращает данные измерения, добавленного в список измерений, по координатам которых будут загружаться данные из альтернативного куба.
