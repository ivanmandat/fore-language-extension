# ICubeInstanceAlternateSource.DimensionCount


## Синтаксис


DimensionCount: Integer;


## Описание


Свойство DimensionCount возвращает количество измерений, добавленных в список измерений, по координатам которых будут загружаться данные из альтернативного куба.
