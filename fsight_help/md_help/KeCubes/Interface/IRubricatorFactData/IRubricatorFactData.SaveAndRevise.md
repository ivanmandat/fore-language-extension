# IRubricatorFactData.SaveAndRevise


## Синтаксис


SaveAndRevise(Options: [SaveRubricatorDataOptions](../../Enums/SaveRubricatorDataOptions.htm)):
 Boolean;


## Параметры


Options. Параметр сохранения
 данных.


## Описание


Метод SaveAndRevise осуществляет
 сохранение данных показателя с заданными параметрами. Возвращает значение
 True, если данные были сохранены
 в новой ревизии.
