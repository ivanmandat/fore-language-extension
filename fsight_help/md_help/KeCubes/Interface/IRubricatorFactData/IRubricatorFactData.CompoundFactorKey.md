# IRubricatorFactData.CompoundFactorKey


## Синтаксис


CompoundFactorKey: Variant;


## Описание


Свойство FactDataCompoundKey возвращает составной ключ временного ряда.


## Комментарии


Признак того, используется ли составной ключ, возвращает свойство [IRubricatorRecord.UseCompoundKey](../IRubricatorRecord/IRubricatorRecord.UseCompoundKey.htm).
