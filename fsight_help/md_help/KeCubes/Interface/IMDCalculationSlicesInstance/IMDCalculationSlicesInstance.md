# IMDCalculationSlicesInstance


## Описание


Интерфейс IMDCalculationSlicesInstance
 содержит свойства и методы для работы с коллекцией измерений отстроенного
 источника/приёмника данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMDCalculationSlicesInstance.Count.htm)
		 Свойство Count возвращает
		 количество измерений источника/приёмника данных.


		 ![](../../Property_Image.gif)
		 [Item](IMDCalculationSlicesInstance.Item.htm)
		 Свойство Item возвращает
		 параметры измерения.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](IMDCalculationSlicesInstance.FindById.htm)
		 Метод FindById осуществляет
		 поиск измерения по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IMDCalculationSlicesInstance.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск измерения по ключу.
