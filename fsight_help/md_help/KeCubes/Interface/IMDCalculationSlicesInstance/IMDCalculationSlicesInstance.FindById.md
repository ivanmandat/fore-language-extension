# IMDCalculationSlicesInstance.FindById


## Синтаксис


FindById(Id: String): [IMDCalculationSliceInstance](../IMDCalculationSliceInstance/IMDCalculationSliceInstance.htm);


## Параметры


Id. Идентификатор
 измерения.


## Описание


Метод FindById осуществляет
 поиск и возвращает параметры измерения источника/приёмника данных. Идентификатор,
 по которому осуществляется поиск, передается посредством параметра Id.
