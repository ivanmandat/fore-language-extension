# ICubeCreatorField.Identifying


## Синтаксис


Identifying: Boolean;


## Описание


Свойство Identifying определяет
 признак использования текущей привязки для идентификации элементов в измерении.


## Комментарии


Если свойству установить значение True,
 то значения [поля](ICubeCreatorField.Name.htm) будут использоваться
 для поиска элементов в измерении по значениям [атрибута](ICubeCreatorField.AttributeId.htm),
 с которым оно связано.


## Пример


Пример использования приведен в описании свойства [ICubeLoaderDimensionBinding.Fields](../ICubeLoaderDimensionBinding/ICubeLoaderDimensionBinding.Fields.htm).
