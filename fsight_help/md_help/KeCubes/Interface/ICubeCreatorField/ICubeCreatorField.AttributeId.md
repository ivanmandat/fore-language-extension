# ICubeCreatorField.AttributeId


## Синтаксис


AttributeId: String;


## Описание


Свойство AttributeId определяет
 идентификатор атрибута измерения, к которому осуществляется привязка.


## Комментарии


Свойство используется, если осуществляется привязка нескольких полей
 источника к нескольким атрибутам одного измерения.
