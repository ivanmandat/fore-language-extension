# ICubeCreatorField


Сборка: Cubes;


## Описание


Интерфейс ICubeCreatorField
 содержит свойства для определения параметров поля, используемого для привязки
 измерения.


## Иерархия наследования


ICubeCreatorField


## Комментарии


Параметры поля можно получить используя свойства или методы коллекции
 [ICubeCreatorFields](../ICubeCreatorFields/ICubeCreatorFields.htm).


## Свойства


		 Имя свойства


		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AttributeId](ICubeCreatorField.AttributeId.htm)


		 Свойство AttributeId
		 определяет идентификатор атрибута измерения, к которому осуществляется
		 привязка.


		 ![](../../Property_Image.gif)
		 [DataType](ICubeCreatorField.DataType.htm)


		 Свойство DataType определяет
		 тип данных, который будут иметь импортируемые значения.


		 ![](../../Property_Image.gif)
		 [Identifying](ICubeCreatorField.Identifying.htm)


		 Свойство Identifying
		 определяет признак использования текущей привязки для идентификации
		 элементов в измерении.


		 ![](../../Property_Image.gif)
		 [Name](ICubeCreatorField.Name.htm)


		 Свойство Name определяет
		 идентификатор поля таблицы, к которому осуществляется привязка
		 атрибута измерения.


		 ![](../../Property_Image.gif)
		 [Translations](ICubeCreatorField.Translations.htm)


		 Свойство Translations
		 возвращает коллекцию привязок атрибутов перевода к полям с переводами
		 на различные языки.
