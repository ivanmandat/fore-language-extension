# ICubeCreatorField.Name


## Синтаксис


Name: String;


## Описание


Свойство Name определяет идентификатор поля таблицы, к которому осуществляется привязка атрибута измерения.


## Комментарии


В качестве значения свойства необходимо указать идентификатор поля таблицы, которая задана в свойстве [ICubeCreator.Table](../ICubeCreator/ICubeCreator.Table.htm) или [ICubeCreatorFact.Table](../ICubeCreatorFact/ICubeCreatorFact.Table.htm).
