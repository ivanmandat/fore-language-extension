# IMDCalculationFormulaElement


## Описание


Интерфейс IMDCalculationFormulaElement
 содержит свойства и методы элемента источника/приёмника данных, являющегося
 операндом в формуле.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Key](IMDCalculationFormulaElement.Key.htm)
		 Свойство Key определяет
		 уникальный ключ, соответствующий элементу, который используется
		 в качестве операнда в формуле.


		 ![](../../Property_Image.gif)
		 [SourceIndex](IMDCalculationFormulaElement.SourceIndex.htm)
		 Свойство SourceIndex
		 возвращает индекс источника, элементу которого соответствует данный
		 операнд.


		 ![](../../Property_Image.gif)
		 [Sum](IMDCalculationFormulaElement.Sum.htm)
		 Свойство Sum определяет
		 способ суммирования.


		 ![](../../Property_Image.gif)
		 [SumInt](IMDCalculationFormulaElement.SumInt.htm)
		 Свойство SumInt определяет
		 способ суммирования в виде десятичного числа.
