# IMDCalculationFormulaElement.SumInt


## Синтаксис


SumInt: Integer;


## Описание


Свойство SumInt определяет способ суммирования в виде десятичного числа. Данное значение рассчитывается автоматически при установке свойства [Sum](IMDCalculationFormulaElement.Sum.htm) и сохраняется в таблицу формул. В дальнейшем по данному значению производится расчет.
