# IVirtualCubeSources.RemoveByKey


## Синтаксис


RemoveByKey(SourceKey: Integer): Boolean;


## Параметры


Index - ключ источника данных виртуального куба.


## Описание


Метод Remove осуществляет удаление источника данных виртуального куба, индекс которого передается посредством параметра Index.
