# IVirtualCubeSources


## Описание


Интерфейс IVirtualCubeSources
 содержит свойства и методы для работы с коллекцией источников данных виртуального
 куба.


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IVirtualCubeSources.Count.htm)
		 Свойство Count возвращает
		 количество источников данных виртуального куба.


		 ![](../../Property_Image.gif)
		 [Item](IVirtualCubeSources.Item.htm)
		 Свойство Item возвращает
		 источник данных виртуального куба


## Методы


		  Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IVirtualCubeSources.Add.htm)
		 Метод Add осуществляет
		 добавление нового источника данных в виртуальный куб.


		 ![](../../Sub_Image.gif)
		 [Clear](IVirtualCubeSources.Clear.htm)
		 Метод Clear очищает
		 список источников данных виртуального куба.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IVirtualCubeSources.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск источника данных виртуального куба по ключу.


		 ![](../../Sub_Image.gif)
		 [FindBySource](IVirtualCubeSources.FindBySource.htm)
		 Метод FindBySource
		 осуществляет поиск куба репозитория среди источников данных виртуального
		 куба.


		 ![](../../Sub_Image.gif)
		 [Remove](IVirtualCubeSources.Remove.htm)
		 Метод Remove осуществляет
		 удаление источника данных виртуального куба по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IVirtualCubeSources.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление источника данных виртуального куба по ключу.
