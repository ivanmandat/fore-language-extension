# IVirtualCubeSources.FindByKey


## Синтаксис


FindByKey(SourceKey: Integer): [IVirtualCubeSource](../IVirtualCubeSource/IVirtualCubeSource.htm);


## Параметры


SourceKey - ключ источника данных, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет поиск и возвращает куб, являющийся источником данных виртуального куба. Ключ, по которому осуществляется поиск, передается посредством параметра SourceKey.
