# IVirtualCubeSources.FindBySource


## Синтаксис


FindBySource(Source: [ICubeModelDestination](../ICubeModelDestination/ICubeModelDestination.htm)): [IVirtualCubeSource](../IVirtualCubeSource/IVirtualCubeSource.htm);


## Параметры


Source - куб репозитория, который необходимо найти среди источников данных виртуального куба.


## Описание


Метод FindBySource осуществляет поиск и возвращает куб, являющийся источником данных виртуального куба. Куб репозитория, который необходимо найти, передается посредством параметра Source.
