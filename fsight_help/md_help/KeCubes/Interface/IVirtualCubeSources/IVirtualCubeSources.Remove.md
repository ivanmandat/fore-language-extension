# IVirtualCubeSources.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс источника данных виртуального куба.


## Описание


Метод Remove осуществляет удаление источника данных виртуального куба, индекс которого передается посредством параметра Index.
