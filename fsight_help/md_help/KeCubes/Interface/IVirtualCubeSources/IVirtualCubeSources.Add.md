# IVirtualCubeSources.Add


## Синтаксис


Add(Source: [ICubeModelDestination](../ICubeModelDestination/ICubeModelDestination.htm)):
 [IVirtualCubeSource](../IVirtualCubeSource/IVirtualCubeSource.htm);


## Параметры


Source. Куб, который необходимо
 добавить в источники данных виртуального куба.


## Описание


Метод Add осуществляет добавление
 нового источника данных в виртуальный куб.
