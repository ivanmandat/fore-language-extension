# IVirtualCubeSources.Item


## Синтаксис


Item(Index: Integer): [IVirtualCubeSource](../IVirtualCubeSource/IVirtualCubeSource.htm);


## Параметры


Index - индекс источника данных виртуального куба.


## Описание


Свойство Item возвращает источник данных виртуального куба, индекс которого передается посредством параметра Index.
