# ICalculatedCubeDimensions.RemoveByKey


## Синтаксис


RemoveByKey(DimensionKey: Integer): Boolean;


## Параметры


DimensionKey. Ключ измерения.


## Описание


Метод RemoveByKey осуществляет
 удаление измерения вычисляемого куба, ключ которого передается в качестве
 входного параметра.


## Комментарии


Метод возвращает значение True,
 если измерение было удалено из коллекции и False,
 если удаление по каким-либо причинам невозможно.
