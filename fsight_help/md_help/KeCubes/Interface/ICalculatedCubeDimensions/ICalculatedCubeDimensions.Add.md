# ICalculatedCubeDimensions.Add


## Синтаксис


Add(Dimension: [IDimensionModel](kedims.chm::/interface/idimensionmodel/idimensionmodel.htm)):
 [ICalculatedCubeDimension](../ICalculatedCubeDimension/ICalculatedCubeDimension.htm);


## Параметры


Dimension. Справочник, который
 необходимо добавить в измерения вычисляемого куба.


## Описание


Метод Add осуществляет добавление
 справочника в коллекцию измерений вычисляемого куба.

Пример использования приведен в описании свойства [ICalculatedCubeDimension.Fixed](../ICalculatedCubeDimension/ICalculatedCubeDimension.Fixed.htm).
