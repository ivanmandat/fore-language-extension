# ICalculatedCubeDimensions.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс измерения.


## Описание


Метод Remove осуществляет удаление
 измерения вычисляемого куба, индекс которого передается в качестве входного
 параметра.


## Комментарии


Метод возвращает значение True,
 если измерение было удалено из коллекции и False,
 если удаление по каким-либо причинам невозможно.
