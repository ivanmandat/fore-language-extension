# ICalculatedCubeDimensions


## Описание


Интерфейс ICalculatedCubeDimensions
 содержит свойства и методы для работы с измерениями вычисляемого куба.


## Комментарии


Коллекцию измерений вычисляемого куба возвращает свойство [ICalculatedCube.Dimensions](../ICalculatedCube/ICalculatedCube.Dimensions.htm).
