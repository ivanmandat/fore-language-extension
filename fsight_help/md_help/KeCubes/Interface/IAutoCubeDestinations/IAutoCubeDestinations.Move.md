# IAutoCubeDestinations.Move


## Синтаксис


Move(IndexFrom: Integer; IndexTo: Integer);


## Параметры


IndexFrom - индекс позиции
 варианта отображения, который необходимо переместить.


IndexTo - индекс позиции, в
 которую необходимо переместить вариант отображения.


## Описание


Метод Move осуществляет перемещение
 вариантов отображения в списке автоматического куба.
