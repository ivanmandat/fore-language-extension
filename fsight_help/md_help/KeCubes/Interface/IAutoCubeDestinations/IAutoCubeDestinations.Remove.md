# IAutoCubeDestinations.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс варианта отображения,
 который необходимо удалить.


## Описание


Метод Remove осуществляет удаление
 варианта отображения автоматического куба, индекс которого передается
 посредством параметра Index.
