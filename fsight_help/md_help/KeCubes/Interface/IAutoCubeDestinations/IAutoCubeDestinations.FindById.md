# IAutoCubeDestinations.FindById


## Синтаксис


FindById(Id: String): [IAutoCubeDestination](../IAutoCubeDestination/IAutoCubeDestination.htm);


## Параметры


Id - идентификатор варианта
 отображения, по которому осуществляется поиск.


## Описание


Метод FindById осуществляет
 поиск и возвращает объект, содержащий вариант отображения автоматического
 куба, в случае удачного поиска.
