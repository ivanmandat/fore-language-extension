# IAutoCubeDestinations


## Описание


Интерфейс IAutoCubeDestinations
 содержит свойства и методы коллекции вариантов отображения автоматического
 куба.


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IAutoCubeDestinations.Count.htm)
		 Свойство Count возвращает
		 количество вариантов отображения автоматического куба.


		 ![](../../Property_Image.gif)
		 [Item](IAutoCubeDestinations.Item.htm)
		 Свойство Item возвращает
		 вариант отображения автоматического куба.


## Методы


		  Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IAutoCubeDestinations.Add.htm)
		 Метод Add создает новый
		 вариант отображения автоматического куба.


		 ![](../../Sub_Image.gif)
		 [FindById](IAutoCubeDestinations.FindById.htm)
		 Метод FindById осуществляет
		 поиск варианта отображения автоматического куба по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IAutoCubeDestinations.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск варианта отображения автоматического куба по ключу.


		 ![](../../Sub_Image.gif)
		 [Move](IAutoCubeDestinations.Move.htm)
		 Метод Move осуществляет
		 перемещение вариантов отображения в списке автоматического куба.


		 ![](../../Sub_Image.gif)
		 [Remove](IAutoCubeDestinations.Remove.htm)
		 Метод Remove осуществляет
		 удаление варианта отображения автоматического куба по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IAutoCubeDestinations.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление варианта отображения автоматического куба по ключу.
