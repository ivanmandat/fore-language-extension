# IAutoCubeDestinations.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IAutoCubeDestination](../IAutoCubeDestination/IAutoCubeDestination.htm);


## Параметры


Key -
 ключ варианта отображения, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает объект, содержащий вариант отображения автоматического
 куба, в случае удачного поиска.
