# ICubeCreatorFields.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс поля.


## Описание


Метод Remove удаляет поле, индекс которого передается посредством параметра Index.


## Комментарии


Метод возвращает значение True, если поле было удалено и False, если удаление поля недопустимо.
