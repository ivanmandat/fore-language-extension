# ICubeCreatorFields


Сборка: Cubes;


## Описание


Интерфейс ICubeCreatorFields
 содержит свойства и методы для работы с коллекцией полей, определяющих
 привязку измерения.


## Иерархия наследования


ICubeCreatorFields


## Комментарии


Коллекция полей доступна в свойстве [ICubeCreatorDimension.Fields](../ICubeCreatorDimension/ICubeCreatorDimension.Fields.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeCreatorFields.Count.htm)


		 Свойство Count возвращает
		 количество полей в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ICubeCreatorFields.Item.htm)


		 Свойство Item возвращает
		 параметры поля.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ICubeCreatorFields.Add.htm)


		 Метод Add добавляет
		 в коллекцию поле, с которым будет связан атрибут измерения.


		 ![](../../Sub_Image.gif)
		 [Clear](ICubeCreatorFields.Clear.htm)


		 Метод Clear удаляет
		 все поля из коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](ICubeCreatorFields.Remove.htm)


		 Метод Remove удаляет
		 указанное поле из коллекции.
