# ICubeCreatorFields.Add


## Синтаксис


Add(Name: String): [ICubeCreatorField](../ICubeCreatorField/ICubeCreatorField.htm);


## Параметры


Name. Наименование поля набора данных, с которым будет связан атрибут измерения.


## Описание


Метод Add добавляет в коллекцию поле, с которым будет связан атрибут измерения.


## Комментарии


В качестве значения параметра Name необходимо указать идентификатор поля таблицы, которая задана в свойстве [ICubeCreator.Table](../ICubeCreator/ICubeCreator.Table.htm) или [ICubeCreatorFact.Table](../ICubeCreatorFact/ICubeCreatorFact.Table.htm).
