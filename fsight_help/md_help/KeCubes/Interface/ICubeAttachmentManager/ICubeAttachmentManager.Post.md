# ICubeAttachmentManager.Post


## Синтаксис


		Post(Iterator: [IMatrixModelIterator](ForeSys.chm::/Interface/IMatrixModelIterator/IMatrixModelIterator.htm);
		 Value: [ICubeAttachmentsEdited](../ICubeAttachmentsEdited/ICubeAttachmentsEdited.htm));


## Параметры


Iterator. Итератор, полученный
 для результирующей матрицы куба.


Value. Коллекция отредактированных
 вложений.


## Описание


Метод Post применяет все изменения
 вложений к результирующей матрице куба, для которой был получен итератор.


## Комментарии


Метод используется после того, как коллекция была отредактирована с
 помощью метода [EditAttachments](ICubeAttachmentManager.EditAttachments.htm).


## Пример


Пример использования приведён в описании метода [ICubeAttachmentManager.EditAttachments](ICubeAttachmentManager.EditAttachments.htm).
