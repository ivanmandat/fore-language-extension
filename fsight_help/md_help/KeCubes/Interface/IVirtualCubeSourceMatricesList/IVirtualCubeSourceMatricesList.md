# IVirtualCubeSourceMatricesList


Сборка: Cubes;


## Описание


Интерфейс IVirtualCubeSourceMatricesList
 предназначен для работы с коллекций матриц, которые рассчитаны для источников
 данных виртуального куба.


## Иерархия наследования


IVirtualCubeSourceMatricesList


## Комментарии


Коллекция матриц источников данных доступна в событии [IVirtualCubeExecuteCallback.OnAfterExecuteSources](../IVirtualCubeExecuteCallback/IVirtualCubeExecuteCallback.OnAfterExecuteSources.htm),
 которое наступает после завершения расчета.
