# IVirtualCubeSourceMatricesList.FindByKey


## Синтаксис


		FindByKey(SourceKey: Integer): [IVirtualCubeSourceMatrix](../IVirtualCubeSourceMatrix/IVirtualCubeSourceMatrix.htm);


## Параметры


SourceKey. Ключ источника виртуального
 куба.


## Описание


Метод FindByKey осуществляет
 поиск по ключу источника и возвращает информацию о матрице источника в
 случае удачного поиска.


## Комментарии


Метод возвращает значение Null,
 если в коллекции отсутствует источник с ключом SourceKey.
