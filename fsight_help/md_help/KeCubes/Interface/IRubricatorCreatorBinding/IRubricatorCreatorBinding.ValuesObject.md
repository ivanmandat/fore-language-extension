# IRubricatorCreatorBinding.ValuesObject


## Синтаксис


ValuesObject: [IMetabaseObjectDescriptor](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm);


## Описание


Свойство ValuesObject определяет справочник, на который ссылается связываемый атрибут БД временных рядов.


## Комментарии


Является ли атрибут ссылкой на справочник, определяет свойство [IRubricatorCreatorBinding.IsDimension](IRubricatorCreatorBinding.IsDimension.htm).

Использование свойства приведено в примере для [IRubricatorCreator.CreateRubricator](../IRubricatorCreator/IRubricatorCreator.CreateRubricator.htm).
