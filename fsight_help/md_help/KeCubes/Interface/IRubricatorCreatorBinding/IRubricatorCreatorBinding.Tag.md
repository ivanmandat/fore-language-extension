# IRubricatorCreatorBinding.Tag


## Синтаксис


Tag: [MetaAttributeTag](KeRds.chm::/Enums/MetaAttributeTag.htm);


## Описание


Свойство Tag определяет вид связываемого атрибута БД временных рядов.


## Комментарии


Тип связываемого атрибута определяет свойство [IRubricatorCreatorBinding.Kind](IRubricatorCreatorBinding.Kind.htm).
