# IRubricatorCreatorBinding.Dictionary


## Синтаксис


Dictionary: [RubricatorDictionary](../../Enums/RubricatorDictionary.htm);


## Описание


Свойство Dictionary определяет тип справочника, который должен хранить значения связываемого атрибута базы данных временных рядов.


## Комментарии


Если связываемый атрибут является атрибутом показателей, то Dictionary = RubricatorDictionary.Facts, если атрибутом наблюдений, то Dictionary = RubricatorDictionary.Values.
