# IRubricatorCreatorBinding.UseField


## Синтаксис


UseField: Boolean;


## Описание


Свойство UseField определяет, использовать ли поле таблицы в качестве источника данных для связываемого атрибута.


## Комментарии


Допустимые значения:


-
True. В качестве источника данных для связываемого атрибута используется поле источника данных, заданное свойством [IRubricatorCreatorBinding.SourceFieldId](IRubricatorCreatorBinding.SourceFieldId.htm);


-
False. В качестве источника данных для связываемого атрибута используется значение, заданное свойством [IRubricatorCreatorBinding.SourceValue](IRubricatorCreatorBinding.SourceValue.htm).
