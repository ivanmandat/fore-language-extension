# IRubricatorCreatorBinding


Сборка: Cubes;


## Описание


Интерфейс IRubricatorCreatorBinding
 предназначен для работы с параметрами привязки атрибута БД временных
 рядов к полю таблицы данных.


## Иерархия наследования


           [INamedEntity](KeSom.chm::/Interface/INamedEntity/INamedEntity.htm)


           IRubricatorCreatorBinding


## Комментарии


Для создания новой привязки используйте метод [IRubricatorCreatorBindings.Add](../IRubricatorCreatorBindings/IRubricatorCreatorBindings.Add.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AttributeId](IRubricatorCreatorBinding.AttributeId.htm)
		 Свойство AttributeId
		 определяет идентификатор связываемого атрибута базы данных
		 временных рядов.


		 ![](../../Property_Image.gif)
		 [AttributeName](IRubricatorCreatorBinding.AttributeName.htm)
		 Свойство AttributeName
		 определяет наименование связываемого атрибута базы данных временных
		 рядов.


		 ![](../../Property_Image.gif)
		 [DataType](IRubricatorCreatorBinding.DataType.htm)
		 Свойство DataType определяет
		 тип данных связываемого атрибута базы данных временных рядов.


		 ![](../../Property_Image.gif)
		 [Dictionary](IRubricatorCreatorBinding.Dictionary.htm)
		 Свойство Dictionary
		 определяет тип справочника, который должен хранить значения
		 связываемого атрибута базы данных временных рядов.


		 ![](../../Property_Image.gif)
		 [Hidden](IRubricatorCreatorBinding.Hidden.htm)
		 Свойство Hidden определяет,
		 является ли связываемый атрибут БД временных рядов скрытым.


		 ![](../../Property_Image.gif)
		 [IncludeInIndex](IRubricatorCreatorBinding.IncludeInIndex.htm)
		 Свойство IncludeInIndex
		 определяет, входит ли связываемый атрибут в состав первичного
		 индекса БД временных рядов.


		 ![](../../Property_Image.gif)
		 [IsDimension](IRubricatorCreatorBinding.IsDimension.htm)
		 Свойство IsDimension
		 определяет, является ли связываемый атрибут ссылкой на справочник.


		 ![](../../Property_Image.gif)
		 [Kind](IRubricatorCreatorBinding.Kind.htm)
		 Свойство Kind определяет
		 тип связываемого атрибута БД временных рядов.


		 ![](../../Property_Image.gif)
		 [Nullable](IRubricatorCreatorBinding.Nullable.htm)
		 Свойство Nullable определяет,
		 может ли связываемый атрибут БД временных иметь пустые значения.


		 ![](../../Property_Image.gif)
		 [SourceFieldId](IRubricatorCreatorBinding.SourceFieldId.htm)
		 Свойство SourceFieldId
		 определяет идентификатор поля-источника в таблице данных.


		 ![](../../Property_Image.gif)
		 [SourceValue](IRubricatorCreatorBinding.SourceValue.htm)
		 Свойство SourceValue
		 определяет значение связываемого атрибута БД временных рядов.


		 ![](../../Property_Image.gif)
		 [Tag](IRubricatorCreatorBinding.Tag.htm)
		 Свойство Tag определяет
		 вид связываемого атрибута БД временных рядов.


		 ![](../../Property_Image.gif)
		 [UseField](IRubricatorCreatorBinding.UseField.htm)
		 Свойство UseField определяет,
		 использовать ли поле таблицы в качестве источника данных для связываемого
		 атрибута.


		 ![](../../Property_Image.gif)
		 [ValuesObject](IRubricatorCreatorBinding.ValuesObject.htm)
		 Свойство ValuesObject
		 определяет справочник, на который ссылается связываемый
		 атрибут БД временных рядов.
