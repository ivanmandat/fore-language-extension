# IRubricatorCreatorBinding.SourceValue


## Синтаксис


SourceValue: Variant;


## Описание


Свойство SourceValue определяет значение связываемого атрибута БД временных рядов.


## Комментарии


Свойство учитывается, если [IRubricatorCreatorBinding.UseField](IRubricatorCreatorBinding.UseField.htm) имеет значение False.


## Пример


Использование свойства приведено в примере для [IRubricatorCreator.CreateRubricator](../IRubricatorCreator/IRubricatorCreator.CreateRubricator.htm).
