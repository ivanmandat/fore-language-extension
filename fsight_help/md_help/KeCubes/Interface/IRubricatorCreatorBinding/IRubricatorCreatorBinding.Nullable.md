# IRubricatorCreatorBinding.Nullable


## Синтаксис


Nullable: Boolean;


## Описание


Свойство Nullable определяет, может ли связываемый атрибут БД временных рядов иметь пустые значения.


## Комментарии


Допустимые значения:


-
True. Атрибут может иметь пустые значения;


-
False. Атрибут не может иметь пустые значения.
