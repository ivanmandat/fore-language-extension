# IRubricatorCreatorBinding.Kind


## Синтаксис


Kind: [MetaAttributeKind](KeRds.chm::/Enums/MetaAttributeKind.htm);


## Описание


Свойство Kind определяет тип связываемого атрибута БД временных рядов.


## Комментарии


Вид связываемого атрибута определяет свойство [IRubricatorCreatorBinding.Tag](IRubricatorCreatorBinding.Tag.htm).
