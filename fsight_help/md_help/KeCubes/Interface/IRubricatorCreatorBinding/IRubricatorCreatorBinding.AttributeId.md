# IRubricatorCreatorBinding.AttributeId


## Синтаксис


AttributeId: String;


## Описание


Свойство AttributeId определяет идентификатор связываемого атрибута базы данных временных рядов.


## Комментарии


Идентификатор поля-источника определяет свойство [IRubricatorCreatorBinding.SourceFieldId](IRubricatorCreatorBinding.SourceFieldId.htm).

Использование свойства приведено в примере для [IRubricatorCreator.CreateRubricator](../IRubricatorCreator/IRubricatorCreator.CreateRubricator.htm).
