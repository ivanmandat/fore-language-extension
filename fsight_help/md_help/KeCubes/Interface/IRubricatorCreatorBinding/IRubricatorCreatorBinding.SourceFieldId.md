# IRubricatorCreatorBinding.SourceFieldId


## Синтаксис


SourceFieldId: String;


## Описание


Свойство SourceFieldId определяет идентификатор поля-источника в таблице данных.


## Комментарии


Идентификатор связываемого атрибута БД временных рядов определяет свойство [IRubricatorCreatorBinding.AttributeId](IRubricatorCreatorBinding.AttributeId.htm).
