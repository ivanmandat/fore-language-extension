# IRubricatorCreatorBinding.DataType


## Синтаксис


DataType: [DbDataType](Dal.chm::/Enums/DbDataType.htm);


## Описание


Свойство DataType определяет тип данных связываемого атрибута базы данных временных рядов.


## Комментарии


Идентификатор связываемого атрибута определяет свойство [IRubricatorCreatorBinding.AttributeId](IRubricatorCreatorBinding.AttributeId.htm).
