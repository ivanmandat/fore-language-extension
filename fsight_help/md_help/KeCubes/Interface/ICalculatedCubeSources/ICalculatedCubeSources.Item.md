# ICalculatedCubeSources.Item


## Синтаксис


Item(Index: Integer): [ICalculatedCubeSource](../ICalculatedCubeSource/ICalculatedCubeSource.htm);


## Параметры


Index - индекс куба-источника данных.


## Описание


Свойство Item возвращает куб-источник данных, индекс которого передается посредством параметра Index.
