# ICalculatedCubeSources


## Описание


Интерфейс ICalculatedCubeSources
 содержит свойства и методы коллекции источников данных вычисляемого куба.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICalculatedCubeSources.Count.htm)
		 Свойство Count возвращает
		 количество кубов-источников данных вычисляемого куба.


		 ![](../../Property_Image.gif)
		 [Item](ICalculatedCubeSources.Item.htm)
		 Свойство Item возвращает
		 куб-источник данных.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ICalculatedCubeSources.Add.htm)
		 Метод Add осуществляет
		 добавление источника данных в вычисляемый куб.


		 ![](../../Sub_Image.gif)
		 [Clear](ICalculatedCubeSources.Clear.htm)
		 Метод Clear осуществляет
		 очистку списка кубов-источников данных вычисляемого куба.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ICalculatedCubeSources.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск куба-источника данных по ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](ICalculatedCubeSources.Remove.htm)
		 Метод Remove осуществляет
		 удаление куба-источника данных по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](ICalculatedCubeSources.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление куба-источника данных по ключу.
