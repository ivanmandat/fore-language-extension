# ICalculatedCubeSources.FindByKey


## Синтаксис


FindByKey(SourceKey: Integer): [ICalculatedCubeSource](../ICalculatedCubeSource/ICalculatedCubeSource.htm);


## Параметры


SourceKey - ключ куба-источника данных, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет поиск куба-источника данных, ключ которого передается посредством параметра SourceKey.
