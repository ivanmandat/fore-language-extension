# ICalculatedCubeSources.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс куба-источника данных.


## Описание


Метод Remove осуществляет удаление куба-источника данных, индекс которого передается посредством параметра Index.
