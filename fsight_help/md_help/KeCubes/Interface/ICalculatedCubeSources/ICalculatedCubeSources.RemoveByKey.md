# ICalculatedCubeSources.RemoveByKey


## Синтаксис


RemoveByKey(SourceKey: Integer): Boolean;


## Параметры


SourceKey - ключ куба-источника данных.


## Описание


Метод RemoveByKey осуществляет удаление куба-источника данных, ключ которого передается посредством параметра SourceKey.
