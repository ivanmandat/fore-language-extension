# ICubeCreatorAdditionalField


Сборка: Cubes;


## Описание


Интерфейс ICubeCreatorAdditionalField
 содержит свойства поля, которое будет создано при создании в таблице куба,
 но не будет связано с измерениями/фактами куба.


## Иерархия наследования


ICubeCreatorAdditionalField


## Комментарии


Параметры поля можно получить, используя свойства и методы интерфейса
 [ICubeCreatorAdditionalFields](../ICubeCreatorAdditionalFields/ICubeCreatorAdditionalFields.htm).
