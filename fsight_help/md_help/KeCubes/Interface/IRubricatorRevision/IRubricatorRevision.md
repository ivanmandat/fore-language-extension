# IRubricatorRevision


## Описание


Интерфейс IRubricatorRevision
 содержит свойства и методы для работы с объектом, являющимся ревизией
 или меткой ревизии.

		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)

		 ![](../../Property_Image.gif)
		 [IsLabel](IRubricatorRevision.IsLabel.htm)
		 Свойство IsLabel возвращает
		 признак того, является ли объект меткой.

		 [Name](IRubricatorRevision.Name.htm)
