# ICubeInstanceStorage.LogKey


## Синтаксис


		LogKey: Decimal;


## Описание


Свойство LogKey возвращает ключ
 уникальной записи в протоколе доступа, созданной после сохранения данных.


## Комментарии


Значение свойства будет доступно после выполнения одного из методов
 SaveMatrix*, если сохранение данных
 прошло удачно.
