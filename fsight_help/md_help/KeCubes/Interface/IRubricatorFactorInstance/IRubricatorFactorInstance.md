# IRubricatorFactorInstance


## Описание


Интерфейс IRubricatorFactorInstance
 содержит свойства и методы для работы с данными показателя.


Примечание.
 Интерфейс IRubricatorFactorInstance
 в данный момент не используется.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Rubricator](IRubricatorFactorInstance.Rubricator.htm)
		 Свойство Rubricator
		 возвращает данные базы, которой принадлежит показатель.
