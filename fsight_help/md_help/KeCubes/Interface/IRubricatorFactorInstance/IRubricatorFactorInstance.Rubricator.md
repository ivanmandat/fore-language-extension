# IRubricatorFactorInstance.Rubricator


## Синтаксис


Rubricator: [IRubricatorInstance](../IRubricatorInstance/IRubricatorInstance.htm);


## Описание


Свойство доступно только для чтения.


Свойство Rubricator возвращает данные базы данных временных рядов, которой принадлежит показатель.


Примечание. Интерфейс IRubricatorFactorInstance в данный момент не используется.
