# IRubricatorFactorInstance.CopyScenarioData


## Синтаксис


CopyScenarioData(ScenarioForm: Integer; ScenarioTо: Integer);


## Параметры


ScenarioForm. Ключ сценария, данные с которого необходимо скопировать;


ScenarioTо. Ключ сценария, на который необходимо скопировать данные.


## Описание


Метод CopyScenarioData осуществляет копирование данных с одного сценария на другой.


Примечание. Интерфейс IRubricatorFactorInstance в данный момент не используется.
