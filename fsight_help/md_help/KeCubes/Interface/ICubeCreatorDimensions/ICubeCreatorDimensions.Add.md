# ICubeCreatorDimensions.Add


## Синтаксис


Add(Dimension: [IDimensionModel](kedims.chm::/interface/idimensionmodel/idimensionmodel.htm)): [ICubeCreatorDimension](../ICubeCreatorDimension/ICubeCreatorDimension.htm);


## Параметры


Dimension. Справочник репозитория, который будет использоваться в качестве измерения создаваемого куба.


## Описание


Метод Add добавляет в коллекцию измерение на базе существующего справочника, передаваемого в параметре Dimension.


## Комментарии


При создании куба указанный справочник будет включен в состав структуры куба.
