# ICubeCreatorDimensions.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс измерения в коллекции.


## Описание


Метод Remove удаляет из коллекции измерение, индекс которого передается посредством параметрам Index.


## Комментарии


Метод возвращает значение True, если удаление измерения прошло удачно и False, если удаление измерения с указанным индексом невозможно.
