# IMDCalculationSlices.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс измерения, который
 необходимо удалить.


## Описание


Метод Remove осуществляет удаление
 измерения, индекс которого передается посредством параметра Index.
