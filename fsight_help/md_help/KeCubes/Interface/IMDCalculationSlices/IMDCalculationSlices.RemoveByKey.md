# IMDCalculationSlices.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ измерения, который
 необходимо удалить.


## Описание


Метод RemoveByKey осуществляет
 удаление измерения, ключ которого передается посредством параметра Key.
