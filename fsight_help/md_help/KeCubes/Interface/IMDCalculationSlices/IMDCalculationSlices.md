# IMDCalculationSlices


## Описание


Интерфейс IMDCalculationSlices
 содержит свойства и методы для работы с коллекцией измерений источника/приёмника
 данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMDCalculationSlices.Count.htm)
		 Свойство Count возвращает
		 количество измерений источника/приёмника данных.


		 ![](../../Property_Image.gif)
		 [Item](IMDCalculationSlices.Item.htm)
		 Свойство Item возвращает
		 параметры измерения.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IMDCalculationSlices.Add.htm)
		 Метод Add осуществляет
		 создание нового измерения источника/приёмника данных.


		 ![](../../Sub_Image.gif)
		 [Clear](IMDCalculationSlices.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех измерений источника/приёмника данных.


		 ![](../../Sub_Image.gif)
		 [FindById](IMDCalculationSlices.FindById.htm)
		 Метод FindById осуществляет
		 поиск измерения по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IMDCalculationSlices.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск измерения по ключу.


		 ![](../../Sub_Image.gif)
		 [Move](IMDCalculationSlices.Move.htm)
		 Метод Move осуществляет
		 перемещение измерений в списке источника/приёмника данных.


		 ![](../../Sub_Image.gif)
		 [Remove](IMDCalculationSlices.Remove.htm)
		 Метод Remove осуществляет
		 удаление измерения по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IMDCalculationSlices.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление измерения по ключу.
