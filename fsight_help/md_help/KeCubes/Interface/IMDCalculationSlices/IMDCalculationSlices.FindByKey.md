# IMDCalculationSlices.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IMDCalculationSlice](../IMDCalculationSlice/IMDCalculationSlice.htm);


## Параметры


Key. Ключ измерения.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает параметры измерения источника/приёмника данных. Ключ,
 по которому осуществляется поиск, передается посредством параметра Key.
