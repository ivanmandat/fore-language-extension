# IMDCalculationSlices.FindById


## Синтаксис


FindById(Id: String): [IMDCalculationSlice](../IMDCalculationSlice/IMDCalculationSlice.htm);


## Параметры


Id. Идентификатор измерения.


## Описание


Метод FindById осуществляет
 поиск и возвращает параметры измерения источника/приёмника данных. Идентификатор,
 по которому осуществляется поиск, передается посредством параметра Id.
