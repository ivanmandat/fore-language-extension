# IStandardCubeServerAggregationDimensionsEx


## Описание


Интерфейс IStandardCubeServerAggregationDimensionsEx
 содержит свойства и методы для работы с коллекцией измерений, по которой
 на сервере будет рассчитываться агрегация.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IStandardCubeServerAggregationDimensionsEx.Count.htm)
		 Свойство Count возвращает
		 количество измерений в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IStandardCubeServerAggregationDimensionsEx.Item.htm)
		 Свойство Item возвращает
		 измерение из коллекции по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](IStandardCubeServerAggregationDimensionsEx.FindById.htm)
		 Метод FindById осуществляет
		 поиск измерения в коллекции по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IStandardCubeServerAggregationDimensionsEx.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск измерения в коллекции по ключу.
