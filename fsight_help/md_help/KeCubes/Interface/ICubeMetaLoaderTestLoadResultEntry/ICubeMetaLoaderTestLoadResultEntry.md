# ICubeMetaLoaderTestLoadResultEntry


## Описание


Интерфейс ICubeMetaLoaderTestLoadResultEntry
 содержит свойства записи отчета о тестовой загрузке данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attribute](ICubeMetaLoaderTestLoadResultEntry.Attribute.htm)
		 Свойство Attribute
		 возвращает наименование атрибута, при тестовой загрузке которого
		 произошла ошибка.


		 ![](../../Property_Image.gif)
		 [Count](ICubeMetaLoaderTestLoadResultEntry.Count.htm)
		 Свойство Count возвращает
		 количество ошибок для данного атрибута и его значения.


		 ![](../../Property_Image.gif)
		 [Value](ICubeMetaLoaderTestLoadResultEntry.Value.htm)
		 Свойство Value возвращает
		 значение атрибута, при тестовой загрузке которого произошла ошибка.
