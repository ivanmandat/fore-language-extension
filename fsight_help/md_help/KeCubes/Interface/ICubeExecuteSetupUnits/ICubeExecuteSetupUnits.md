# ICubeExecuteSetupUnits


Сборка: Cubes;


## Описание


Интерфейс ICubeExecuteSetupUnits
 содержит свойства для указания единиц измерения, в которых будет производиться
 вычисление.


## Иерархия наследования


ICubeExecuteSetupUnits


## Комментарии


К интерфейсу ICubeExecuteSetupUnits
 можно привести предопределенные параметры, которые возвращает свойство
 [ICubeExecuteDimSetup.Predefined](../ICubeExecuteDimSetup/ICubeExecuteDimSetup.Predefined.htm).
 Предопределенные параметры должны быть получены для справочника единиц
 измерения.
