# ICubeModelDimensionEx.IsFact


## Синтаксис


IsFact: Boolean;


## Описание


Свойство IsFact определяет,
 используется ли данный справочник в качестве перечня показателей для куба.


## Комментарии


Если свойству установлено значение True,
 справочник используется в качестве перечня показателей для куба.
