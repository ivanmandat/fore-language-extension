# ICubeModelDimensionEx.IsUnit


## Синтаксис


IsUnit: Boolean;


## Описание


Свойство IsUnit определяет,
 является ли данный справочник, справочником единиц измерения для куба.


## Комментарии


Если свойству установлено значение True,
 справочник является справочником единиц измерения куба.
