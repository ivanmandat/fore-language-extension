# ICubeModelDimensionEx


## Описание


Интерфейс ICubeModelDimensionEx
 содержит расширенные свойства измерения куба.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dimension](ICubeModelDimensionEx.Dimension.htm)
		 Свойство Dimension
		 возвращает структуру справочника.


		 ![](../../Property_Image.gif)
		 [DimensionO](ICubeModelDimensionEx.DimensionO.htm)
		 Свойство DimensionO
		 возвращает объект репозитория, являющийся измерением в данном
		 кубе.


		 ![](../../Property_Image.gif)
		 [DimensionTag](ICubeModelDimensionEx.DimensionTag.htm)
		 Свойство DimensionTag
		 возвращает признак измерения куба.


		 ![](../../Property_Image.gif)
		 [IsFact](ICubeModelDimensionEx.IsFact.htm)
		 Свойство IsFact определяет,
		 используется ли данный справочник в качестве перечня показателей
		 для куба.


		 ![](../../Property_Image.gif)
		 [IsUnit](ICubeModelDimensionEx.IsUnit.htm)
		 Свойство IsUnit определяет,
		 является ли данный справочник, справочником единиц измерения для
		 куба.


		 ![](../../Property_Image.gif)
		 [Name](ICubeModelDimensionEx.Name.htm)
		 Свойство Name определяет
		 наименование измерения в пределах данного куба.
