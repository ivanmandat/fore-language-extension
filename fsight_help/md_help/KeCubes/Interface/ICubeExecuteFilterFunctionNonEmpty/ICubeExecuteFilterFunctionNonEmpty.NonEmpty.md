# ICubeExecuteFilterFunctionNonEmpty.NonEmpty


## Синтаксис


NonEmpty: Boolean;


## Описание


Свойство NonEmpty определяет
 признак необходимости использовать функцию NonEmpty для фильтрации данных.


## Комментарии


Если свойству установить значение True,
 то для фильтрации используется функция «NonEmpty». При этом наименовании
 функции необходимо указать в свойстве [ICubeExecuteFilterFunctionNonEmpty.Function_](ICubeExecuteFilterFunctionNonEmpty.Function_.htm),
 а в свойстве [ICubeExecuteFilterFunctionNonEmpty.Set_](ICubeExecuteFilterFunctionNonEmpty.Set_.htm)
 указать отметку элементов, по которым будут отфильтрованы значения.
