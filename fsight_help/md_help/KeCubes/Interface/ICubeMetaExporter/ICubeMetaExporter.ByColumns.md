# ICubeMetaExporter.ByColumns


## Синтаксис


ByColumns: Boolean;


## Описание


Свойство ByColumns определяет признак того, располагаются ли элементы календаря в столбцах. True - элементы календаря расположены в столбцах, False - элементы календаря расположены построчно.
