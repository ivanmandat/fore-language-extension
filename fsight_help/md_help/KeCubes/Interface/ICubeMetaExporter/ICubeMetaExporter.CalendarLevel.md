# ICubeMetaExporter.CalendarLevel


## Синтаксис


CalendarLevel: [DimCalendarLevel](kedims.chm::/enums/dimcalendarlevel.htm);


## Описание


Свойство CalendarLevel определяет уровень детализации данных. Если значение данного свойства [DimCalendarLevel.None](kedims.chm::/enums/dimcalendarlevel.htm), то данные будут экспортироваться со всех календарных уровней.


Примечание. Экспорт данных со всех календарных уровней возможен, только если элементы календаря расположены построчно (значение свойства [ICubeMetaExporter.ByColumns](ICubeMetaExporter.ByColumns.htm) - False).
