# ICubeMetaExporter


Сборка: Cubes;


## Описание


Интерфейс ICubeMetaExporter
 содержит свойства и методы для экспорта временных рядов.


## Иерархия наследования


           ICubeMetaExporter


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AutoPeriod](ICubeMetaExporter.AutoPeriod.htm)
		 Свойство AutoPeriod
		 возвращает параметры периода экспорта временных рядов.


		 ![](../../Property_Image.gif)
		 [Bindings](ICubeMetaExporter.Bindings.htm)
		 Свойство Bindings определяет
		 привязку атрибутов базы данных временных рядов с полями приёмника
		 данных.


		 ![](../../Property_Image.gif)
		 [ByColumns](ICubeMetaExporter.ByColumns.htm)
		 Свойство ByColumns
		 определяет признак того, располагаются ли элементы календаря в
		 столбцах.


		 ![](../../Property_Image.gif)
		 [CalendarLevel](ICubeMetaExporter.CalendarLevel.htm)
		 Свойство CalendarLevel
		 определяет уровень детализации данных.


		 ![](../../Property_Image.gif)
		 [CalendarLevelSet](ICubeMetaExporter.CalendarLevelSet.htm)
		 Свойство CalendarLevelSet
		 определяет набор календарных динамик, данные по которым должны
		 быть экспортированы.


		 ![](../../Property_Image.gif)
		 [ClearBeforeExport](ICubeMetaExporter.ClearBeforeExport.htm)
		 Свойство ClearBeforeExport
		 определяет, очищать ли приёмник данных перед экспортом.


		 ![](../../Property_Image.gif)
		 [Consumer](ICubeMetaExporter.Consumer.htm)
		 Свойство Consumer определяет
		 объект, являющийся приёмником данных.


		 ![](../../Property_Image.gif)
		 [ExportMode](ICubeMetaExporter.ExportMode.htm)
		 Свойство ExportMode
		 определяет режим экспорта временных рядов.


		 ![](../../Property_Image.gif)
		 [Filters](ICubeMetaExporter.Filters.htm)
		 Свойство Filters определяет
		 параметры фильтрации экспортируемых временных рядов по атрибутам
		 рядов.


		 ![](../../Property_Image.gif)
		 [Rubricator](ICubeMetaExporter.Rubricator.htm)
		 Свойство Rubricator
		 определяет базу, содержащую экспортируемые показатели.


		 ![](../../Property_Image.gif)
		 [ScaleValues](ICubeMetaExporter.ScaleValues.htm)
		 Свойство ScaleValues
		 определяет, нормализовывать ли данные при экспорте.


		 ![](../../Property_Image.gif)
		 [ValueFilters](ICubeMetaExporter.ValueFilters.htm)
		 Свойство ValueFilters
		 определяет параметры фильтрации экспортируемых временных рядов
		 по атрибутам наблюдений.


		 ![](../../Property_Image.gif)
		 [WhereIsEmpty](ICubeMetaExporter.WhereIsEmpty.htm)
		 Свойство WhereIsEmpty
		 определяет, экспортировать ли показатели, для которых установлен
		 признак пустоты.
