# ICalculatedCubeDimension.TimeAttribute


## Синтаксис


TimeAttribute: [IDimAttribute](kedims.chm::/interface/idimattribute/idimattribute.htm);


## Описание


Свойство TimeAttribute определяет
 атрибут измерения, который будет управлять выбором периода действия формулы.


## Комментарии


Устанавливаемый атрибут должен иметь тип - Дата. Свойство актуально,
 если свойству [IsTime](ICalculatedCubeDimension.IsTime.htm)
 установлено значение True.
