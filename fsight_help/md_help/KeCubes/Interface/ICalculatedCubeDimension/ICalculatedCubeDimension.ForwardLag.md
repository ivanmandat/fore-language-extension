# ICalculatedCubeDimension.ForwardLag


## Синтаксис


ForwardLag: Integer;


## Описание


Свойство ForwardLag определяет
 лаг вперед, указывающий количество следующих элементов не входящих в отметку,
 по которым будут извлекаться значения.


## Комментарии


Лаг вперед - это смещение вперед по отметке относительно последнего
 отмеченного элемента. Значение свойства ForwardLag
 указывает для скольких следующих элементов, не входящих в отметку, будет
 извлечено значение.


Значение лага необходимо указывать, если в формулах используется функция
 LAGVAL, которая получает значение
 элемента, смещенного относительно указанного элемента. Если элемент, координата
 которого была вычислена функцией LAGVAL,
 отсутствует в отметке, то функция вернет пустое значение, так как значение
 по этому элементу не извлекается из источника данных. Указание свойств
 [BackwardLag](ICalculatedCubeDimension.BackwardLag.htm)
 и ForwardLag позволяет задать
 необходимость извлечения значений по элементам, отсутствующим в отметке.
