# ICalculatedCubeDimension


## Описание


Интерфейс ICalculatedCubeDimension
 содержит свойства измерения вычисляемого куба и кубов-источников данных
 для вычисляемого куба.


## Комментарии


Измерение куба-источника можно получить, используя свойства и методы
 интерфейса [ICalculatedCubeSourceDimensions](../ICalculatedCubeSourceDimensions/ICalculatedCubeSourceDimensions.htm).


Измерение вычисляемого куба можно получить, используя свойства и методы
 интерфейса [ICalculatedCubeDimensions](../ICalculatedCubeDimensions/ICalculatedCubeDimensions.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BackwardLag](ICalculatedCubeDimension.BackwardLag.htm)


		 Свойство BackwardLag
		 определяет лаг назад, указывающий количество предшествующих элементов
		 не входящих в отметку, по которым будут извлекаться значения.


		 ![](../../Property_Image.gif)
		 [Dimension](ICalculatedCubeDimension.Dimension.htm)


		 Свойство Dimension
		 возвращает структуру справочника, лежащего в основе измерения
		 вычисляемого куба.


		 ![](../../Property_Image.gif)
		 [Expression](ICalculatedCubeDimension.Expression.htm)


		 Свойство Expression
		 возвращает выражение связи между измерениями куба-источника и
		 куба-назначения.


		 ![](../../Property_Image.gif)
		 [Fixed](ICalculatedCubeDimension.Fixed.htm)


		 Свойство Fixed определяет,
		 является ли измерение фиксированным.


		 ![](../../Property_Image.gif)
		 [ForwardLag](ICalculatedCubeDimension.ForwardLag.htm)


		 Свойство ForwardLag
		 определяет лаг вперед, указывающий количество следующих элементов
		 не входящих в отметку, по которым будут извлекаться значения.


		 ![](../../Property_Image.gif)
		 [Index](ICalculatedCubeDimension.Index.htm)


		 Свойство Index определяет
		 индекс, по которому осуществляется связь между измерениями куба-источника
		 и куба-назначения.


		 ![](../../Property_Image.gif)
		 [IsTime](ICalculatedCubeDimension.IsTime.htm)


		 Свойство IsTime определяет,
		 будет ли данное измерение использоваться для управления выбором
		 периода действия формул.


		 ![](../../Property_Image.gif)
		 [TimeAttribute](ICalculatedCubeDimension.TimeAttribute.htm)


		 Свойство TimeAttribute
		 определяет атрибут измерения, который будет управлять выбором
		 периода действия формулы.
