# IVirtualCubeCompounds


## Описание


Интерфейс IVirtualCubeCompounds
 предназначен для работы с коллекцией составных структур виртуального куба.


## Комментарии


Коллекция всегда содержит предопределенные структуры (факты/календарь/сценарии),
 которые нельзя удалить.


## Свойства


		 Имя свойства
		 Краткое описание

		 указанную предопределенную структуру.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IVirtualCubeCompounds.Add.htm)
		 Метод Add добавляет
		 составную структуру в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](IVirtualCubeCompounds.Clear.htm)

		 ![](../../Sub_Image.gif)
		 [FindByDimension](IVirtualCubeCompounds.FindByDimension.htm)

		 ![](../../Sub_Image.gif)

		 поиск структуры по указанному идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IVirtualCubeCompounds.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск структуры по указанному ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](IVirtualCubeCompounds.Remove.htm)
		 Метод Remove удаляет
		 составную структуру из коллекции по ее индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IVirtualCubeCompounds.RemoveByKey.htm)
		 Метод RemoveByKey удаляет
		 составную структуру из коллекции по ее ключу.
