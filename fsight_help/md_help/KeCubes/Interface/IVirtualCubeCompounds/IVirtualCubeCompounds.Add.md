# IVirtualCubeCompounds.Add


## Синтаксис


Add: [IVirtualCubeCompound](../IVirtualCubeCompound/IVirtualCubeCompound.htm);


## Описание


Метод Add добавляет составную структуру в коллекцию.


## Комментарии


Добавляемая структура будет пользовательского типа, т.е. [IVirtualCubeCompound.Predefined](../IVirtualCubeCompound/IVirtualCubeCompound.Predefined.htm) = [VirtualCubeCompoundPredefinedType.Custom](../../Enums/VirtualCubeCompoundPredefinedType.htm).
