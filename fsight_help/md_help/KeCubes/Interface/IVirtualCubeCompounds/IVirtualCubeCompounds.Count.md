# IVirtualCubeCompounds.Count


## Синтаксис


Count: Integer;


## Описание


Свойство доступно только для чтения.


Свойство Count возвращает количество составных структур в коллекции.


## Комментарии


Емкость коллекции - это количество элементов, которое может содержать коллекция. Свойство Count - это количество элементов, которое в действительности содержится в коллекции. Емкость всегда больше или равна значению свойства Count. Если при добавлении элементов в коллекцию значение свойства Count превышает емкость, то она автоматически увеличивается.


Коллекция составных структур всегда содержит три предопределенные структуры (факты/календарь/сценарии).
