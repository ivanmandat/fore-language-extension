# ICalculatedCubeInstanceCoordKey


## Описание


Интерфейс ICalculatedCubeInstanceCoordKey
 содержит свойства и методы для работы с уникальным ключом, который соответствует
 определенной координате в кубе.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Cube](ICalculatedCubeInstanceCoordKey.Cube.htm)
		 Свойство Cube возвращает
		 куб, координате в котором, соответствует данный уникальный ключ.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ToCoord](ICalculatedCubeInstanceCoordKey.ToCoord.htm)
		 Метод ToCoord осуществляет
		 преобразование и возвращает параметры координаты, которой соответствует
		 данный уникальный ключ.
