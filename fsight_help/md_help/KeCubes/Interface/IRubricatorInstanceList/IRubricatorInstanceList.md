# IRubricatorInstanceList


## Описание


Интерфейс IRubricatorInstanceList
 предназначен для работы с коллекцией баз данных временных рядов, являющихся
 источниками данных.


## Комментарии


Данная коллекция не предусматривает добавление и удаление из неё элементов
 пользователем.
