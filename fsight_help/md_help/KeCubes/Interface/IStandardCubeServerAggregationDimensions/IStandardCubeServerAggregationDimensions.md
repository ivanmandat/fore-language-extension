# IStandardCubeServerAggregationDimensions


## Описание


Интерфейс IStandardCubeServerAggregationDimensions
 содержит свойства и методы для работы с измерениями, входящими в группу,
 по которой на сервере рассчитывается агрегация.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CalcOther](IStandardCubeServerAggregationDimensions.CalcOther.htm)
		 Свойство CalcOther
		 определяет, рассчитывать ли агрегацию для элементов измерения,
		 не вошедших в отметку (агрегат Other).


		 ![](../../Property_Image.gif)
		 [CalcTotal](IStandardCubeServerAggregationDimensions.CalcTotal.htm)
		 Свойство CalcTotal
		 определяет, рассчитывать ли итоги для измерения.


		 ![](../../Property_Image.gif)
		 [Count](IStandardCubeServerAggregationDimensions.Count.htm)
		 Свойство Count возвращает
		 количество измерений в группе.


		 ![](../../Property_Image.gif)
		 [Item](IStandardCubeServerAggregationDimensions.Item.htm)
		 Свойство Item возвращает
		 измерение из группы по индексу.
