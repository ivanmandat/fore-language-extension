# IStandardCubeServerAggregationDimensions.CalcTotal


## Синтаксис


CalcTotal(Index: Integer): Boolean;


## Параметры


Index. Индекс измерения.


## Описание


Свойство CalcTotal определяет,
 рассчитывать ли итоги для измерения с указанным индексом.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Рассчитывать итоги;


	- False. Не рассчитывать
	 итоги.
