# IStandardCubeServerAggregationDimensions.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key - ключ удаляемого измерения.


## Описание


Метод RemoveByKey осуществляет удаление измерения из группы по ключу. Данный метод возвращает значение True, если удаление прошло успешно.
