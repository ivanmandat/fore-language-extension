# IStandardCubeServerAggregationDimensions.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс удаляемого измерения.


## Описание


Метод Remove осуществляет удаление измерения из группы по индексу. Данный метод возвращает значение True, если удаление прошло успешно.
