# IVirtualCubeSource


## Описание


Интерфейс IVirtualCubeSource
 содержит свойства источника данных для виртуального куба.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Cube](IVirtualCubeSource.Cube.htm)
		 Свойство Cube определяет
		 куб репозитория, являющийся источником данных для виртуального
		 куба.


		 ![](../../Property_Image.gif)
		 [FixInfo](IVirtualCubeSource.FixInfo.htm)
		 Свойство FixInfo возвращает
		 параметры фиксации измерений куба-источника данных.


		 ![](../../Property_Image.gif)
		 [JoinList](IVirtualCubeSource.JoinList.htm)
		 Свойство JoinList возвращает
		 список сопоставлений измерений текущего источника с измерениями
		 других источников.
