# IRubricatorCopier.DisableTransactions


## Синтаксис


DisableTransactions: Boolean;


## Описание


Свойство DisableTransactions
 определяет, выполнять ли копирование базы данных временных рядов за одну
 транзакцию.


## Комментарии


Допустимые значения:


	- True. Копирование базы
	 данных временных рядов выполняется в рамках одной транзакции. Рекомендуется
	 использовать, если с базой данных временных рядов одновременно работают
	 несколько пользователей. Это позволит избежать получения пользователями
	 неполных данных, извлеченных из базы данных временных рядов во время
	 выполнения копирования. Также следует учитывать, что копирование за
	 одну транзакцию приводит к увеличению дисковой памяти на сервере;


	- False. Значение по умолчанию.
	 Копирование базы данных временных рядов выполняется в рамках нескольких
	 транзакций.


Примечание.
 Значение свойства не сохраняется.
