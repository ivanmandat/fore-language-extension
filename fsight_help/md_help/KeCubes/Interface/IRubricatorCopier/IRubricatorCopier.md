# IRubricatorCopier


Сборка: Cubes;


## Описание


Интерфейс IRubricatorCopier
 используется для копирования базы данных временных рядов.


## Иерархия наследования


           IRubricatorCopier


## Комментарии


Для создания объекта, выполняющего копирование базы данных временных
 рядов, используйте класс [RubricatorCopier](../../Class/RubricatorCopier/RubricatorCopier.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CopyInfo](IRubricatorCopier.CopyInfo.htm)


		 Свойство CopyInfo
		 определяет общие параметры копирования базы данных временных рядов.


		 ![](../../Property_Image.gif)
		 [DisableTransactions](IRubricatorCopier.DisableTransactions.htm)


		 Свойство DisableTransactions
		 определяет, выполнять ли копирование базы данных временных рядов
		 за одну транзакцию.


		 ![](../../Property_Image.gif)
		 [WithObjects](IRubricatorCopier.WithObjects.htm)


		 Свойство WithObjects
		 определяет, копировать ли базу данных временных рядов с данными.
