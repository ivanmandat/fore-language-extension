# IRubricatorCopier.WithObjects


## Синтаксис


WithObjects: Boolean;


## Описание


Свойство WithObjects определяет, копировать ли базу данных временных рядов с данными.


## Комментарии


Допустимые значения:


-
True. Копировать с данными: всё содержимое контейнера базы данных временных рядов, включая данные в таблицах;


-
False. Копировать базу данных временных рядов без данных: только метаданные, структуру и описание объектов (со связями между ними).
