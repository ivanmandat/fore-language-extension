# IStandardCubeDimensionBinding


## Описание


Интерфейс IStandardCubeDimensionBinding
 содержит свойства, определяющие привязку измерений к источникам данных
 куба.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Binding](IStandardCubeDimensionBinding.Binding.htm)
		 Свойство Binding возвращает
		 выражение, определяющее связь между полями источника данных куба
		 и атрибутами измерения, по которым создан индекс измерения (блока
		 измерения).


		 ![](../../Property_Image.gif)
		 [Dimension](IStandardCubeDimensionBinding.Dimension.htm)
		 Свойство Dimension
		 возвращает структуру справочника, лежащего в основе измерения,
		 для которого настраивается привязка.


		 ![](../../Property_Image.gif)
		 [Index](IStandardCubeDimensionBinding.Index.htm)
		 Свойство Index определяет
		 индекс измерения (блока измерения), по которому будет осуществляться
		 связь.
