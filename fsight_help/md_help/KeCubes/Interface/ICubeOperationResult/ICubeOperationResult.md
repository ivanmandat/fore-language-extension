# ICubeOperationResult


## Описание


Интерфейс ICubeOperationResult
 содержит свойства для работы с отчётом о результатах загрузки данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Error](ICubeOperationResult.Error.htm)
		 Свойство Error определяет
		 исключительную ситуацию, возникшую при загрузке данных.


		 ![](../../Property_Image.gif)
		 [FormattedMessage](ICubeOperationResult.FormattedMessage.htm)
		 Свойство FormattedMessage
		 возвращает текст сообщения о результатах загрузки данных.


		 ![](../../Property_Image.gif)
		 [Id](ICubeOperationResult.Id.htm)
		 Свойство Id определяет
		 идентификатор загрузчика данных.


		 ![](../../Property_Image.gif)
		 [Name](ICubeOperationResult.Name.htm)
		 Свойство Name определяет
		 наименование загрузчика данных.


		 ![](../../Property_Image.gif)
		 [RecordsProcessed](ICubeOperationResult.RecordsProcessed.htm)
		 Свойство RecordsProcessed
		 определяет число обработанных записей.


		 ![](../../Property_Image.gif)
		 [Succeeded](ICubeOperationResult.Succeeded.htm)
		 Свойство Succeeded
		 определяет признак успешного завершения загрузки данных.


		 ![](../../Property_Image.gif)
		 [TimeFinish](ICubeOperationResult.TimeFinish.htm)
		 Свойство TimeFinish
		 определяет дату и время окончания загрузки данных.


		 ![](../../Property_Image.gif)
		 [TimeStart](ICubeOperationResult.TimeStart.htm)
		 Свойство TimeStart
		 определяет дату и время начала загрузка данных.
