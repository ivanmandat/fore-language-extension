# ICubeOperationResult.Succeeded


## Синтаксис


Succeeded: Boolean;


## Описание


Свойство Succeeded определяет
 признак успешного завершения загрузки данных.


## Комментарии


Допустимые значения:


	- True. Загрузка данных
	 завершена успешно;


	- False. При загрузке
	 данных произошла ошибка. Информацию об ошибке можно получить, используя
	 свойство [ICubeOperationResult.Error](ICubeOperationResult.Error.htm).
