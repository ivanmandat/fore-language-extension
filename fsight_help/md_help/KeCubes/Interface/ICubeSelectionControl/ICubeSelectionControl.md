# ICubeSelectionControl


## Описание


Интерфейс ICubeSelectionControl
 содержит свойства и методы для работы с коллекцией управляющих измерений
 куба.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeSelectionControl.Count.htm)
		 Свойство Count возвращает
		 количество управляющих измерений куба.


		 ![](../../Property_Image.gif)
		 [IsEmpty](ICubeSelectionControl.IsEmpty.htm)
		 Свойство IsEmpty возвращает
		 признак того, определены ли управляющие измерения куба.


		 ![](../../Property_Image.gif)
		 [Item](ICubeSelectionControl.Item.htm)
		 Свойство Item возвращает
		 измерение из коллекции по индексу.


		 ![](../../Property_Image.gif)
		 [Options](ICubeSelectionControl.Options.htm)
		 Свойство Options позволяет
		 настроить параметры управления при настроенном управлении через
		 вложенные объекты.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Clear](ICubeSelectionControl.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех управляющих измерений из коллекции.


		 ![](../../Sub_Image.gif)
		 [CreateApply](ICubeSelectionControl.CreateApply.htm)
		 Метод CreateApply создает
		 объект, осуществляющий управление параметрами куба через отметку
		 в измерениях.


		 ![](../../Sub_Image.gif)
		 [FindByKey](ICubeSelectionControl.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск управляющего измерения в коллекции по ключу.


		 ![](../../Sub_Image.gif)
		 [FindByParam](ICubeSelectionControl.FindByParam.htm)
		 Метод FindByParam осуществляет
		 поиск измерения в коллекции по параметру и объекту-владельцу.


		 ![](../../Sub_Image.gif)
		 [FindByParamO](ICubeSelectionControl.FindByParamO.htm)
		 Метод FindByParamO
		 осуществляет поиск измерения в коллекции по параметру и по ключу
		 объекта-владельца.


		 ![](../../Sub_Image.gif)
		 [Move](ICubeSelectionControl.Move.htm)
		 Метод Move осуществляет
		 перемещение управляющих измерений.
