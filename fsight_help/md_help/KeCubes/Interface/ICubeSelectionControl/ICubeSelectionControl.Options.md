# ICubeSelectionControl.Options


## Синтаксис


Options: [CubeSelectionControlOptions](../../Enums/CubeSelectionControlOptions.htm);


## Описание


Свойство Options позволяет настроить параметры управления при настроенном управлении через вложенные объекты.


## Комментарий


Свойство актуально, если в кубе настроено задание значений параметров через выражение в управлении вложенными объектами и одновременно через механизм управляющего измерения.
