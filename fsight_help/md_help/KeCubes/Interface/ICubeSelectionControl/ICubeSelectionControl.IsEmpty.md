# ICubeSelectionControl.IsEmpty


## Синтаксис


IsEmpty: Boolean;


## Описание


Свойство IsEmpty возвращает признак того, определены ли управляющие измерения куба. True - управляющие измерения определены, False - управляющие измерения не заданы.
