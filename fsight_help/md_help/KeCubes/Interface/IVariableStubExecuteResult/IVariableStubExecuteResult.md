# IVariableStubExecuteResult


## Описание


Интерфейс IVariableStubExecuteResult
 содержит свойства и методы для работы с результатами загрузки данных из
 источника.


## Комментарии


Результаты загрузки доступны после выполнения метода [IVariableStub.Execute](../IVariableStub/IVariableStub.Execute.htm).
