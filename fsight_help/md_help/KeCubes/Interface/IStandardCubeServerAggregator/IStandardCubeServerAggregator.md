# IStandardCubeServerAggregator


## Описание


Интерфейс IStandardCubeServerAggregator
 содержит свойства и методы, определяющие параметры расчета агрегации на
 сервере.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Aggregations](IStandardCubeServerAggregator.Aggregations.htm)
		 Свойство Aggregations
		 возвращает коллекцию рассчитываемых агрегаций.


		 ![](../../Property_Image.gif)
		 [Dimensions](IStandardCubeServerAggregator.Dimensions.htm)
		 Свойство Dimensions
		 возвращает коллекцию измерений, по которой будет рассчитываться
		 агрегация.
