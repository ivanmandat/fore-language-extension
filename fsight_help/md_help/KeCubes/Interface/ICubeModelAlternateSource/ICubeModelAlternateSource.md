# ICubeModelAlternateSource


## Описание


Интерфейс ICubeModelAlternateSource
 содержит свойства и методы, используемые для настройки параметров альтернативного
 источника данных.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AutoSaveToStorage](ICubeModelAlternateSource.AutoSaveToStorage.htm)
		 Свойство AutoSaveToStorage
		 определяет, будут ли данные автоматически сохраняться в альтернативный
		 куб.


		 ![](../../Property_Image.gif)
		 [Destination](ICubeModelAlternateSource.Destination.htm)
		 Свойство Destination
		 возвращает параметры варианта отображения куба, для которого установлен
		 данный альтернативный источник.


		 ![](../../Property_Image.gif)
		 [Dimension](ICubeModelAlternateSource.Dimension.htm)
		 Свойство Dimension
		 возвращает параметры измерения, добавленного в список измерений,
		 по координатам которых будут загружаться данные из альтернативного
		 куба.


		 ![](../../Property_Image.gif)
		 [DimensionCount](ICubeModelAlternateSource.DimensionCount.htm)
		 Свойство DimensionCount
		 возвращает количество измерений, добавленных в список измерений,
		 по координатам которых будут загружаться данные из альтернативного
		 куба.


		 ![](../../Property_Image.gif)
		 [Source](ICubeModelAlternateSource.Source.htm)
		 Свойство Source определяет
		 куб, используемый в качестве альтернативного источника данных.


		 ![](../../Property_Image.gif)
		 [UseExecute](ICubeModelAlternateSource.UseExecute.htm)
		 Свойство UseExecute
		 определяет, будут ли браться данные из альтернативного куба перед
		 вычислением.


		 ![](../../Property_Image.gif)
		 [UseStorage](ICubeModelAlternateSource.UseStorage.htm)
		 Свойство UseStorage
		 определяет, будет ли для хранения данных использоваться альтернативный
		 куб.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddDimension](ICubeModelAlternateSource.AddDimension.htm)
		 Метод AddDimension
		 осуществляет добавление измерения альтернативного куба в список
		 измерений, по элементам которых будет производиться отбор данных.


		 ![](../../Sub_Image.gif)
		 [RemoveDimension](ICubeModelAlternateSource.RemoveDimension.htm)
		 Метод RemoveDimension
		 осуществляет удаление измерения альтернативного куба из списка
		 измерений, по координатам которых будут загружаться данные из
		 альтернативного куба.
