# ICubeModelAlternateSource.RemoveDimension


## Синтаксис


RemoveDimension(Value: [IDimensionModel](kedims.chm::/interface/idimensionmodel/idimensionmodel.htm)):
 Boolean;


## Параметры


Value. Измерение альтернативного
 куба, которое необходимо удалить из списка.


## Описание


Метод RemoveDimension осуществляет
 удаление измерения альтернативного куба из списка измерений, по координатам
 которых будут загружаться данные из альтернативного куба.
