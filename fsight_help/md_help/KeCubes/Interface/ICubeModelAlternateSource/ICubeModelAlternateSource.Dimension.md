# ICubeModelAlternateSource.Dimension


## Синтаксис


Dimension(Index: Integer): [IDimensionModel](kedims.chm::/interface/idimensionmodel/idimensionmodel.htm);


## Параметры


Index. Индекс измерения.


## Описание


Свойство Dimension возвращает
 параметры измерения, добавленного в список измерений, по координатам которых
 будут загружаться данные из альтернативного куба.
