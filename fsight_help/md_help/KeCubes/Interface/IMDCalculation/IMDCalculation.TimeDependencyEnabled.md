# IMDCalculation.TimeDependencyEnabled


## Синтаксис


TimeDependencyEnabled: Boolean;


## Описание


Свойство TimeDependencyEnabled
 возвращает признак доступности установить для формул зависимость от времени.


## Комментарии


Данное свойство возвращает True,
 если в приёмнике имеется календарное измерение, иначе - False.
