# IMDCalculation.RecursionEnabled


## Синтаксис


RecursionEnabled: Boolean;


## Описание


Свойство RecursionEnabled возвращает
 признак доступности использовать рекурсию в формулах.


## Комментарии


Рекурсия в формулах доступна, если источники и приёмник данных содержат
 одинаковый набор измерений. Для создания рекурсивных формул необходимо
 приёмник данных также добавить в список источников данных многомерного
 расчета на сервере БД.
