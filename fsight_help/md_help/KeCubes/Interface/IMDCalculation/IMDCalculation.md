# IMDCalculation


## Описание


Интерфейс IMDCalculation содержит
 свойства объекта репозитория - [Многомерный
 расчёт на сервере БД](UiNavObj.chm::/Cube/Work_Cube/UiMd_CubeWork_MultivariateAccount.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Database](IMDCalculation.Database.htm)
		 Свойство Database определяет
		 базу данных, в которой хранятся все таблицы источников и приёмника
		 данных, участвующих в расчете.


		 ![](../../Property_Image.gif)
		 [Destination](IMDCalculation.Destination.htm)
		 Свойство Destination
		 возвращает настройки куба-назначения многомерного расчета на сервере
		 БД.


		 ![](../../Property_Image.gif)
		 [FormulasTable](IMDCalculation.FormulasTable.htm)
		 Свойство FormulasTable
		 возвращает настройки таблицы с формулами, используемой при многомерном
		 расчете на сервере БД.


		 ![](../../Property_Image.gif)
		 [RecursionEnabled](IMDCalculation.RecursionEnabled.htm)
		 Свойство RecursionEnabled
		 возвращает признак доступности использовать рекурсию в формулах.


		 ![](../../Property_Image.gif)
		 [SelectionControl](IMDCalculation.SelectionControl.htm)
		 Свойство SelectionControl
		 возвращает параметры управляющих измерений для параметризации
		 многомерного расчета.


		 ![](../../Property_Image.gif)
		 [Sources](IMDCalculation.Sources.htm)
		 Свойство Sources возвращает
		 настройки кубов-источников многомерного расчета на сервере БД.


		 ![](../../Property_Image.gif)
		 [TimeDependencyEnabled](IMDCalculation.TimeDependencyEnabled.htm)
		 Свойство TimeDependencyEnabled
		 возвращает признак доступности установить для формул зависимость
		 от времени.


		 ![](../../Property_Image.gif)
		 [TimeDependent](IMDCalculation.TimeDependent.htm)
		 Свойство TimeDependent
		 определяет признак зависимость формул от времени.
