# IExportRequestParams


Сборка: Cubes;


## Описание


Интерфейс IExportRequestParams
 предназначен для настройки параметров экспорта временных рядов.


## Иерархия наследования


           IExportRequestParams


## Комментарии


Для получения параметров экспорта временных рядов используйте свойство
 [IExportRequestDefinition.Exporter](../IExportRequestDefinition/IExportRequestDefinition.Exporter.htm).
