# IExportRequestParams.Filters


## Синтаксис


Filters: [ICubeMetaExporterFilters](../ICubeMetaExporterFilters/ICubeMetaExporterFilters.htm);


## Описание


Свойство Filters определяет
 параметры фильтрации экспортируемых временных рядов по атрибутам рядов.


## Комментарии


Для ограничения количества экспортируемых строк используйте свойство
 [IExportRequestParams.RowCount](IExportRequestParams.RowCount.htm).
