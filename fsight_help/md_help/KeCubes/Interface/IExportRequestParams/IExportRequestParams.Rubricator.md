# IExportRequestParams.Rubricator


## Синтаксис


Rubricator: [IRubricator](../IRubricator/IRubricator.htm);


## Описание


Свойство Rubricator определяет экспортируемую базу данных временных рядов.


## Комментарии


Для настройки привязки атрибутов базы данных временных рядов с полями приёмника данных используйте свойство [IExportRequestParams.Bindings](IExportRequestParams.Bindings.htm).
