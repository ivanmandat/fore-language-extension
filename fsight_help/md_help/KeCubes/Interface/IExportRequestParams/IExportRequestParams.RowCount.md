# IExportRequestParams.RowCount


## Синтаксис


RowCount: Integer;


## Описание


Свойство RowCount определяет максимальное количество экспортируемых строк.


## Комментарии


Для настройки параметров фильтрации экспортируемых временных рядов используйте свойство [IExportRequestParams.Filters](IExportRequestParams.Filters.htm).
