# IExportRequestParams.ClearBeforeExport


## Синтаксис


ClearBeforeExport: Boolean;


## Описание


Свойство ClearBeforeExport определяет, очищать ли приёмник данных перед экспортом.


## Комментарии


Допустимые значения:


-
True. Очищать приёмник данных;


-
False. Не очищать приёмник данных.
