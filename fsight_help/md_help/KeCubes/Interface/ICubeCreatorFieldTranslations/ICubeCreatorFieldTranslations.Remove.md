# ICubeCreatorFieldTranslations.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс привязки в коллекции.


## Описание


Метод Remove удаляет привязку
 с указанным индексом из коллекции и возвращает признак удачного удаления.
