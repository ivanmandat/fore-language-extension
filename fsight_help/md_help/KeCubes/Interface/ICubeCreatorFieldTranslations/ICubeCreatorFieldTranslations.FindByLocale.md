# ICubeCreatorFieldTranslations.FindByLocale


## Синтаксис


FindByLocale(Locale: [LocaleCodeID](ForeSys.chm::/Enums/LocaleCodeID.htm)):
 [ICubeCreatorFieldTranslation](../ICubeCreatorFieldTranslation/ICubeCreatorFieldTranslation.htm);


## Параметры


Locale. Язык перевода, для
 которого необходимо найти привязку к полю источника.


## Описание


Метод FindByLocale осуществляет
 поиск по языку перевода и возвращает параметры привязки к полю источника.


## Комментарии


Метод возвращает параметры привязки атрибута с переводом, если привязка
 была создана и Null, если для
 атрибута перевода с указанным языком еще не создана привязка к полю источника.
