# ICubeCreatorFieldTranslations


Сборка: Cubes;


## Описание


Интерфейс ICubeCreatorFieldTranslations
 содержит свойства и методы для работы с коллекций привязок атрибутов с
 переводами к полям источника.


## Иерархия наследования


ICubeCreatorFieldTranslations


## Комментарии


Коллекцию привязок можно получить в свойстве [ICubeCreatorField.Translations](../ICubeCreatorField/ICubeCreatorField.Translations.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ICubeCreatorFieldTranslations.Count.htm)


		 Свойство Count возвращает
		 количество привязок в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ICubeCreatorFieldTranslations.Item.htm)


		 Свойство Item возвращает
		 параметры привязки по индексу.


		 ![](../../Property_Image.gif)
		 [Locale](ICubeCreatorFieldTranslations.Locale.htm)


		 Свойство Locale определяет
		 язык, по которому осуществляется поиск атрибута для идентификации
		 элементов.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ICubeCreatorFieldTranslations.Add.htm)


		 Метод Add создает привязку
		 поля источника с атрибутом перевода для указанного языка.


		 ![](../../Sub_Image.gif)
		 [Clear](ICubeCreatorFieldTranslations.Clear.htm)


		 Метод Clear удаляет
		 все привязки из коллекции.


		 ![](../../Sub_Image.gif)
		 [FindByLocale](ICubeCreatorFieldTranslations.FindByLocale.htm)


		 Метод FindByLocale
		 осуществляет поиск по языку перевода и возвращает параметры привязки
		 к полю источника.


		 ![](../../Sub_Image.gif)
		 [Remove](ICubeCreatorFieldTranslations.Remove.htm)


		 Метод Remove удаляет
		 привязку с указанным индексом из коллекции и возвращает признак
		 удачного удаления.
