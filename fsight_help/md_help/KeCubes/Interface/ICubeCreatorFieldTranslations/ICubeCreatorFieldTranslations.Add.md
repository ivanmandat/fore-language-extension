# ICubeCreatorFieldTranslations.Add


## Синтаксис


Add(Locale: [LocaleCodeID](ForeSys.chm::/Enums/LocaleCodeID.htm);
 SourceField: String): [ICubeCreatorFieldTranslation](../ICubeCreatorFieldTranslation/ICubeCreatorFieldTranslation.htm);


## Параметры


Locale. Язык перевода, для
 которого необходимо создать привязку.


SourceField. Идентификатор
 поля источника, в котором хранятся значения переводов на языке Locale.


## Описание


Метод Add создает привязку поля
 источника с атрибутом перевода для указанного языка.


## Комментарии


Атрибут, к которому осуществляется привязка, определяется по языку перевода.
