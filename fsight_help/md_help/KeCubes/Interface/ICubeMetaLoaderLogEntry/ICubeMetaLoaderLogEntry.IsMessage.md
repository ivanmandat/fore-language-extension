# ICubeMetaLoaderLogEntry.IsMessage


## Синтаксис


IsMessage: Boolean;


## Описание


Свойство IsMessage возвращает признак соответствия информационной записи.


## Комментарии


Свойство возвращает значение True, если текущая запись содержит информационное сообщение. К информационным сообщениям относятся сообщения о начале и окончании загрузки показателей.
