# ICubeMetaLoaderLogEntry.IsWarning


## Синтаксис


IsWarning: Boolean;


## Описание


Свойство IsWarning возвращает признак соответствия записи о предупреждении.


## Комментарии


Свойство возвращает значение True, если текущая запись содержит предупреждающее сообщение.
