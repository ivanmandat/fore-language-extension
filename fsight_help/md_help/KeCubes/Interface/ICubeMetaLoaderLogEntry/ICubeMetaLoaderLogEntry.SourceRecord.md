## Синтаксис

## Описание


Свойство SourceRecord возвращает
 номер записи в источнике данных.


## Комментарии


Для получения номера соответствующей записи в приёмнике данных используйте
 свойство [ICubeMetaLoaderLogEntry.DestRecord](ICubeMetaLoaderLogEntry.DestRecord.htm).
