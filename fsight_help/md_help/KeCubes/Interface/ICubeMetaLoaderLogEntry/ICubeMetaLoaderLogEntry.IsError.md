# ICubeMetaLoaderLogEntry.IsError


## Синтаксис


IsError: Boolean;


## Описание


Свойство IsError возвращает признак соответствия записи об ошибке.


## Комментарии


Свойство возвращает значение True, если текущая запись содержит информацию об ошибке.
