# ICubeMetaLoaderLogEntry.ExtraField


## Синтаксис


ExtraField(Index: Integer): String;


## Параметры


Index. Индекс привязки поля.


## Описание


Свойство ExtraField возвращает значение поля, дополнительно добавленного в отчет.


## Комментарии


Для добавления дополнительных полей в отчет используйте метод [ICubeMetaLoaderBinding.AddToLog](../ICubeMetaLoaderBinding/ICubeMetaLoaderBinding.AddToLog.htm).
