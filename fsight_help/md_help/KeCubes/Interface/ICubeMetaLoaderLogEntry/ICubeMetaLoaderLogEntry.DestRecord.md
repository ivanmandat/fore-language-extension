# ICubeMetaLoaderLogEntry.DestRecord


## Синтаксис


DestRecord: Integer;


## Описание


Свойство DestRecord возвращает
 номер записи в приёмнике данных.


## Комментарии


Для получения номера соответствующей записи в источнике данных используйте
 свойство [ICubeMetaLoaderLogEntry.SourceRecord](ICubeMetaLoaderLogEntry.SourceRecord.htm).
