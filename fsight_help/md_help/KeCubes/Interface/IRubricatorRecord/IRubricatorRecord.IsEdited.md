# IRubricatorRecord.IsEdited


## Синтаксис


IsEdited: Boolean;


## Описание


Свойство IsEdited возвращает
 признак того, доступна ли запись для редактирования.


## Комментарии


Допустимые значения:


	- True. Запись доступна
	 для редактирования;


	- False. Запись доступна
	 только для чтения.
