# IRubricatorRecord


## Описание


Интерфейс IRubricatorRecord
 содержит свойства и методы для работы с записью базы данных временных
 рядов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CompoundKey](IRubricatorRecord.CompoundKey.htm)
		 Свойство CompoundKey
		 возвращает составной ключ записи.


		 ![](../../Property_Image.gif)
		 [IsEdited](IRubricatorRecord.IsEdited.htm)
		 Свойство IsEdited возвращает
		 признак того, доступна ли запись для редактирования.


		 ![](../../Property_Image.gif)
		 [Key](IRubricatorRecord.Key.htm)
		 Свойство Key возвращает
		 ключ записи.


		 ![](../../Property_Image.gif)
		 [Record](IRubricatorRecord.Record.htm)
		 Свойство Record возвращает
		 данные записи.


		 ![](../../Property_Image.gif)
		 [Rubricator](IRubricatorRecord.Rubricator.htm)
		 Свойство Rubricator
		 возвращает данные базы данных временных рядов.


		 ![](../../Property_Image.gif)
		 [UseCompoundKey](IRubricatorRecord.UseCompoundKey.htm)
		 Свойство UseCompoundKey
		 возвращает признак того, используется ли составной ключ.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [PersistChanges](IRubricatorRecord.PersistChanges.htm)


		 Метод PersistChanges
		 определяет, производить ли сохранение изменений в базу данных.


		 ![](../../Sub_Image.gif)
		 [Save](IRubricatorRecord.Save.htm)


		 Метод Save осуществляет
		 сохранение записи.


		 ![](../../Sub_Image.gif)
		 [Undo](IRubricatorRecord.Undo.htm)


		 Метод Undo закрывает
		 запись без сохранения изменений.


		 ![](../../Sub_Image.gif)
		 [ValidateRules](IRubricatorRecord.ValidateRules.htm)


		 Метод ValidateRules
		 осуществляет проверку выполнения правил валидации.
