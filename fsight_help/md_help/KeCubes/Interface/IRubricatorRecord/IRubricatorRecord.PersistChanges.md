# IRubricatorRecord.PersistChanges


## Синтаксис


PersistChanges([Force: Boolean = False]);


## Параметры


Force. Параметр, определяющий
 сохранение изменений. Необязательный параметр.


## Описание


Метод PersistChanges определяет,
 производить ли сохранение изменений в базу данных.
