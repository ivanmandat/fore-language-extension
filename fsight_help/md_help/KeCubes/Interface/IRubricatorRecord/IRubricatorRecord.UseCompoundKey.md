# IRubricatorRecord.UseCompoundKey


## Синтаксис


UseCompoundKey: Boolean;


## Описание


Свойство UseCompoundKey возвращает признак того, используется ли составной ключ.


## Комментарии


Допустимые значения:


-
True. Используется составной ключ;


-
False. Составной ключ не используется.
