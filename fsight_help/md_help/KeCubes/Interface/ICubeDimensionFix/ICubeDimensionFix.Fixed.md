# ICubeDimensionFix.Fixed


## Синтаксис


Fixed: Boolean;


## Описание


Свойство Fixed определяет, является
 ли измерение зафиксированным.


## Комментарии


Если свойству установлено значение True,
 то измерение будет зафиксировано. Для фиксированного измерения, используя
 свойство [Selection](ICubeDimensionFix.Selection.htm), необходимо
 определить отметку элементов. По отмеченным элементам будет осуществляться
 агрегация с использованием метода, указанного в свойстве [Operation](ICubeDimensionFix.Operation.htm).


## Пример


Пример использования приведен в описании свойства [ICubeDimensionFix.Operation](ICubeDimensionFix.Operation.htm).
