# ICubeDimensionFix


## Описание


Интерфейс ICubeDimensionFix
 содержит свойства, определяющие параметры фиксации измерения.


## Комментарии


Параметры фиксации измерения могут быть получены с помощью одного из
 свойств или методов интерфейса [ICubeDimensionFixInfo](../ICubeDimensionFixInfo/ICubeDimensionFixInfo.htm).


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dimension](ICubeDimensionFix.Dimension.htm)
		 Свойство Dimension
		 возвращает структуру справочника репозитория, являющегося измерением,
		 которое необходимо зафиксировать.


		 ![](../../Property_Image.gif)
		 [Fixed](ICubeDimensionFix.Fixed.htm)
		 Свойство Fixed определяет,
		 является ли измерение зафиксированным.


		 ![](../../Property_Image.gif)
		 [Instance](ICubeDimensionFix.Instance.htm)
		 Свойство Instance возвращает
		 коллекцию данных измерения.


		 ![](../../Property_Image.gif)
		 [Operation](ICubeDimensionFix.Operation.htm)
		 Свойство Operation
		 определяет метод агрегации применяемый для элементов фиксированного
		 измерения.


		 ![](../../Property_Image.gif)
		 [Selection](ICubeDimensionFix.Selection.htm)
		 Свойство Selection
		 возвращает отметку элементов фиксированного измерения.
