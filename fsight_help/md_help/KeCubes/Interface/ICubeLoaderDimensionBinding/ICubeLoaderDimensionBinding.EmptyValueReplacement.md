# ICubeLoaderDimensionBinding.EmptyValueReplacement


## Синтаксис


EmptyValueReplacement: Variant;


## Описание


Свойство EmptyValueReplacement
 определяет значение, которое будет подставлено вместо пустых значений.


## Комментарии


Свойство актуально, если свойству [LoadEmptyValues](ICubeLoaderDimensionBinding.LoadEmptyValues.htm)
 установлено значение True.
