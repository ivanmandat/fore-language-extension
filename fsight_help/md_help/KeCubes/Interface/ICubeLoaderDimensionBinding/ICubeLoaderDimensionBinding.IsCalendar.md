# ICubeLoaderDimensionBinding.IsCalendar


## Синтаксис


IsCalendar: Boolean;


## Описание


Свойство IsCalendar возвращает
 признак соответствия параметрам привязки для календарного измерения.


## Комментарии


Свойство возвращает значение True,
 если текущие параметры привязки соответствуют календарному измерению и
 False, если обычному измерению
 куба.
