# ICubeLoaderDimensionBinding.Index


## Синтаксис


Index: [IDimIndex](kedims.chm::/interface/idimindex/idimindex.htm);


## Описание


Свойство Index определяет
 индекс измерения, который будет использоваться для привязки.


## Комментарии


Для корректной работы используемый индекс должен включать в себя только
 один атрибут.
