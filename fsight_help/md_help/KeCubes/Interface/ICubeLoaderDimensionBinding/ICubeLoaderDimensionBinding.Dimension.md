# ICubeLoaderDimensionBinding.Dimension


## Синтаксис


Dimension: [IStandardCubeDimension](../IStandardCubeDimension/IStandardCubeDimension.htm);


## Описание


Свойство Dimension определяет измерение куба, для которого настраивается привязка.


## Комментарии


Свойство актуально если импорт осуществляется в существующий объект репозитория.
