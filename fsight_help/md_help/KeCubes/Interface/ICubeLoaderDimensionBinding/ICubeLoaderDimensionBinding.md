# ICubeLoaderDimensionBinding


Сборка: Cubes;


## Описание


Интерфейс ICubeLoaderDimensionBinding
 содержит свойства для определения параметров привязки измерения, которые
 используются при загрузке данных.


## Иерархия наследования


           ICubeLoaderDimensionBinding


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AttributeId](ICubeLoaderDimensionBinding.AttributeId.htm)


		 Свойство AttributeId
		 определяет идентификатор атрибута, по значениям которого будет
		 производиться поиск элементов в измерении при загрузке данных.


		 ![](../../Property_Image.gif)
		 [CalendarLevel](ICubeLoaderDimensionBinding.CalendarLevel.htm)


		 Свойство CalendarLevel
		 определяет уровень календарного измерения, для которого настраивается
		 привязка.


		 ![](../../Property_Image.gif)
		 [ConstValue](ICubeLoaderDimensionBinding.ConstValue.htm)


		 Свойство ConstValue
		 задает фиксированное значение для загрузки.


		 ![](../../Property_Image.gif)
		 [Converter](ICubeLoaderDimensionBinding.Converter.htm)


		 Свойство Converter определяет
		 объект, который используется для разбора загружаемых данных измерения.


		 ![](../../Property_Image.gif)
		 [Dictionary](ICubeLoaderDimensionBinding.Dictionary.htm)


		 Свойство Dictionary
		 определяет справочник репозитория, который будет использован в
		 качестве измерения куба.


		 ![](../../Property_Image.gif)
		 [Dimension](ICubeLoaderDimensionBinding.Dimension.htm)


		 Свойство Dimension
		 определяет измерение куба, для которого настраивается привязка.


		 ![](../../Property_Image.gif)
		 [EmptyValueReplacement](ICubeLoaderDimensionBinding.EmptyValueReplacement.htm)


		 Свойство EmptyValueReplacement
		 определяет значение, которое будет подставлено вместо пустых значений.


		 ![](../../Property_Image.gif)
		 [Fields](ICubeLoaderDimensionBinding.Fields.htm)


		 Свойство Fields возвращает
		 коллекцию привязок полей источника к атрибутам измерения.


		 ![](../../Property_Image.gif)
		 [Index](ICubeLoaderDimensionBinding.Index.htm)


		 Свойство Index
		 определяет индекс измерения, который будет использоваться для
		 привязки.


		 ![](../../Property_Image.gif)
		 [IsCalendar](ICubeLoaderDimensionBinding.IsCalendar.htm)


		 Свойство IsCalendar
		 возвращает признак соответствия параметрам привязки для календарного
		 измерения.


		 ![](../../Property_Image.gif)
		 [LoadConstValue](ICubeLoaderDimensionBinding.LoadConstValue.htm)


		 Свойство LoadConstValue
		 определяет признак загрузки фиксированного значения.


		 ![](../../Property_Image.gif)
		 [LoadEmptyValues](ICubeLoaderDimensionBinding.LoadEmptyValues.htm)


		 Свойство LoadEmptyValues
		 определяет признак загрузки пустых значений.


		 ![](../../Property_Image.gif)
		 [MetaAttribute](ICubeLoaderDimensionBinding.MetaAttribute.htm)


		 Свойство MetaAttribute
		 возвращает атрибуты базы данных временных рядов уровня фактов
		 или наблюдений.


		 ![](../../Property_Image.gif)
		 [NewDictionaryName](ICubeLoaderDimensionBinding.NewDictionaryName.htm)


		 Свойство NewDictionaryName
		 определяет наименование создаваемого справочника, который будет
		 использован в качестве измерения куба.


		 ![](../../Property_Image.gif)
		 [SkipLoad](ICubeLoaderDimensionBinding.SkipLoad.htm)


		 Свойство SkipLoad определяет,
		 будут ли пропускаться элементы, отсутствующие в измерении.


		 ![](../../Property_Image.gif)
		 [SourceField](ICubeLoaderDimensionBinding.SourceField.htm)


		 Свойство SourceField
		 возвращает наименование поля источника данных, с которым будет
		 связано измерение.
