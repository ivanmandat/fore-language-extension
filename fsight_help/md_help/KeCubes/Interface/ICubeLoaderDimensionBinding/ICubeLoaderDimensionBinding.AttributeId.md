# ICubeLoaderDimensionBinding.AttributeId


## Синтаксис


AttributeId: String;


## Описание


Свойство AttributeId определяет идентификатор атрибута, по значениям которого будет производиться поиск элементов в измерении при загрузке данных.


## Комментарии


Во время выполнения метода [Load](../ICubeLoaderFromSource/ICubeLoaderFromSource.Load.htm), после того как все необходимые объекты созданы, осуществляется загрузка данных в таблицу. Загрузка данных из источника в таблицу происходит следующим образом:


-
Из источника считывается запись с данными;


-
В записи поочередно берется значение каждого поля и производится следующее действие:


-


-
Если поле связано с измерением куба, то значение поля ищется среди элементов измерения. Поиск осуществляется по значениям атрибута Наименование либо атрибута, указанного в свойстве [AttributeId](ICubeLoaderDimensionBinding.AttributeId.htm). Если элемент не найден, то он автоматически добавляется в соответствующий справочник. Ключ полученного элемента заносится в соответствующее поле таблицы, которая создана для хранения значений куба;


-
Если поле связано с календарным измерением либо фактом, то значение заносится в соответствующее поле таблицы без изменений.


По умолчанию свойство AttributeId не установлено, при этом поиск производится по наименованиям элементов.
