# ICubeLoaderDimensionBinding.SkipLoad


## Синтаксис


SkipLoad: Boolean;


## Описание


Свойство SkipLoad определяет,
 будут ли пропускаться элементы, отсутствующие в измерении.


## Комментарии


Допустимые значения:


	- True. Элементы, которые
	 отсутствуют в измерении, пропускаются. Загрузка производится только
	 по тем элементам, которые уже существуют в измерении.


	- False. Элементы, отсутствующие
	 в измерении, будут добавлены в него.
