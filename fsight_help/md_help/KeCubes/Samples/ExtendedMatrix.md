# Расширенная матрица


Если в кубе настроены отношения, которые используются при расчете значений
 вычисляемых показателей, или формулы, которые рассчитывают значение в
 текущей координате на основании данных по другой координате, то при получении
 матрицы с данными она будет расширена. Кроме значений, полученных в соответствии
 с указанной отметкой, также будут получены все те значения, которые необходимы
 для вычисления значений.


При работе с такой матрицей итератор можно использовать только
 для просмотра значений, так как ряд значений будет содержать большее
 количество элементов, чем определено отметкой. Как и в примере по [работе
 с данными куба](KeCubes_Sample3.htm), для изменения значений в матрице предварительно необходимо
 получить координату элемента, по которому данные могут быть изменены.
 Только после этого можно изменять значение по полученной координате.
