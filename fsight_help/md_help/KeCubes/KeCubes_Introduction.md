# Введение в сборку Cubes


Основными интерфейсами сборки Cubes
 являются:


	- [IStandardCube](Interface/IStandardCube/IStandardCube.htm).
	 Интерфейс, содержащий свойства и методы для работы со структурой стандартного
	 куба.


	- [ICubeView](Interface/ICubeView/ICubeView.htm).
	 Интерфейс, содержащий свойства и методы для работы со структурой представления-куба.


	- [IVirtualCube](Interface/IVirtualCube/IVirtualCube.htm).
	 Интерфейс, содержащий свойства и методы для работы со структурой виртуального
	 куба.


	- [ICalculatedCube](Interface/ICalculatedCube/ICalculatedCube.htm).
	 Интерфейс, содержащий свойства и методы для работы со структурой вычисляемого
	 куба.


	- [IAutoCube](Interface/IAutoCube/IAutoCube.htm).
	 Интерфейс, содержащий свойства и методы для работы со структурой автоматического
	 куба.


	- [ICubeLoader](Interface/ICubeLoader/ICubeLoader.htm).
	 Интерфейс, содержащий свойства и методы для работы со структурой загрузчика
	 данных из куба в куб.


	- [IMDCalculation](Interface/IMDCalculation/IMDCalculation.htm).
	 Интерфейс, содержащий свойства и методы для работы со структурой многомерного
	 расчета на сервере БД.


	- [IRubricator](Interface/IRubricator/IRubricator.htm).
	 Интерфейс, содержащий свойства и методы для работы со структурой базы
	 данных временных рядов.


	- [IRubricatorFactor](Interface/IRubricatorFactor/IRubricatorFactor.htm).
	 Интерфейс, содержащий свойства и методы для работы со структурой показателя,
	 входящего в базу данных временных рядов.


Интерфейсы для работы с открытыми объектами:


	- [IAutoCubeInstance](Interface/IAutoCubeInstance/IAutoCubeInstance.htm).
	 Интерфейс для работы с данными в автоматическом кубе.


	- [ICalculatedCubeInstance](Interface/ICalculatedCubeInstance/ICalculatedCubeInstance.htm).
	 Интерфейс для настройки формул в вычисляемом кубе.


	- [ICubeInstance](Interface/ICubeInstance/ICubeInstance.htm).
	 Интерфейс содержит свойства и методы отстроенного куба.


	- [ICubeInstanceDestination](Interface/ICubeInstanceDestination/ICubeInstanceDestination.htm).
	 Интерфейс содержит свойства и методы для работы с отстроенным кубом,
	 в соответствии с выбранным вариантом отображения.


	- [ICubeLoaderInstance](Interface/ICubeLoaderInstance/ICubeLoaderInstance.htm).
	 Интерфейс содержит свойства и методы для загрузки данных из куба в
	 куб.


	- [IMDCalculationInstance](Interface/IMDCalculationInstance/IMDCalculationInstance.htm).
	 Интерфейс для настройки формул и запуска многомерного расчета на сервере
	 БД.


	- [ICubeSerialInstance](Interface/ICubeSerialInstance/ICubeSerialInstance.htm).
	 Интерфейс содержит методы для сохранения куба в файл.


	- [IRubricatorInstance](Interface/IRubricatorInstance/IRubricatorInstance.htm).
	 Интерфейс содержит свойства и методы для работы с данными базы данных
	 временных рядов.
