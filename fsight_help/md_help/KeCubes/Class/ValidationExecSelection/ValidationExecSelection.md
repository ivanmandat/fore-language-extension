# ValidationExecSelection


## Описание


Класс ValidationExecSelection
 содержит свойства и методы для работы с отметкой, при получении данных
 расчета валидации.


## Конструкторы


		 Имя конструктора
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CreateFactors](ValidationExecSelection.CreateFactors.htm)


		 Конструктор CreateFactors
		 создает отметку с учетом показателей.


		 ![](../../Sub_Image.gif)
		 [CreateSelection](ValidationExecSelection.CreateSelection.htm)


		 Конструктор CreateSelection
		 создает отметку.


## Свойства, унаследованные от [IValidationExecSelection](../../Interface/IValidationExecSelection/IValidationExecSelection.htm):


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)

		 Свойство FactorsCount
		 возвращает количество показателей, для которых должны быть получены

		 [Selection](../../Interface/IValidationExecSelection/IValidationExecSelection.Selection.htm)
