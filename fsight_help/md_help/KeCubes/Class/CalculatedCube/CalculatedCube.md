# CalculatedCube


## Описание


Класс CalculatedCube реализует
 объект, предоставляющий доступ к данным вычисляемого куба из пользовательских
 макросов.


## Свойства, унаследованные от [ICalculatedCubeClass](../../Interface/ICalculatedCubeClass/ICalculatedCubeClass.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Current](../../Interface/ICalculatedCubeClass/ICalculatedCubeClass.Current.htm)


		 Свойство Current возвращает
		 текущий вычисляемый куб.
