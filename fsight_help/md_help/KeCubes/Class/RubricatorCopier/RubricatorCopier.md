# RubricatorCopier


Сборка: Cubes;


## Описание


Класс RubricatorCopier используется
 для копирования базы данных временных рядов.


## Свойства, унаследованные от [IRubricatorCopier](../../Interface/IRubricatorCopier/IRubricatorCopier.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CopyInfo](../../Interface/IRubricatorCopier/IRubricatorCopier.CopyInfo.htm)


		 Свойство CopyInfo
		 определяет общие параметры копирования базы данных временных рядов.


		 ![](../../Property_Image.gif)
		 [DisableTransactions](../../Interface/IRubricatorCopier/IRubricatorCopier.DisableTransactions.htm)


		 Свойство DisableTransactions
		 определяет, выполнять ли копирование базы данных временных рядов
		 за одну транзакцию.


		 ![](../../Property_Image.gif)
		 [WithObjects](../../Interface/IRubricatorCopier/IRubricatorCopier.WithObjects.htm)


		 Свойство WithObjects
		 определяет, копировать ли базу данных временных рядов с данными.
