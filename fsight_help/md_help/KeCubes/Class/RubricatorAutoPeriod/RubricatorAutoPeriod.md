# RubricatorAutoPeriod


Сборка: Cubes;


## Описание


Класс RubricatorAutoPeriod предназначен
 для работы с параметрами временного периода в базе данных временных рядов.


## Комментарии


Используя данный интерфейс, можно задать временной интервал для импорта,
 экспорта и копирования показателей, причём, как в абсолютных, так и в
 относительных величинах.
