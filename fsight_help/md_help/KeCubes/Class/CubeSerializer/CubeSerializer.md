# CubeSerializer


Сборка: Cubes;


## Описание


Класс CubeSerializer реализует
 объект, используемый для работы с кубом, сохраненным в файл.


## Свойства объекта класса, унаследованные от [ICubeSerializer](../../Interface/ICubeSerializer/ICubeSerializer.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [FileName](../../Interface/ICubeSerializer/ICubeSerializer.FileName.htm)


		 Свойство FileName определяет
		 имя файла, в который будет сохранен куб.


		 ![](../../Property_Image.gif)
		 [StorageType](../../Interface/ICubeSerializer/ICubeSerializer.StorageType.htm)


		 Свойство StorageType
		 определяет тип хранилища, в который будет помещен в куб.


## Методы объекта класса, унаследованные от [ICubeSerializer](../../Interface/ICubeSerializer/ICubeSerializer.htm)


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [LoadFromFile](../../Interface/ICubeSerializer/ICubeSerializer.LoadFromFile.htm)


		 Метод LoadFromFile
		 осуществляет загрузку куба, сохраненного в файл.


		 ![](../../Sub_Image.gif)
		 [PreCheck](../../Interface/ICubeSerializer/ICubeSerializer.PreCheck.htm)


		 Метод PreCheck осуществляет
		 проверку прав на экспорт у данного пользователя.


		 ![](../../Sub_Image.gif)
		 [SaveToFile](../../Interface/ICubeSerializer/ICubeSerializer.SaveToFile.htm)


		 Метод SaveToFile создает
		 файл и инициализирует процесс сохранения куба в файл.
