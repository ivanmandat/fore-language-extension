# CubeMetaUpdateCallback


Сборка: Cubes;


## Описание


Класс CubeMetaUpdateCallback
 предназначен для работы с событиями, которые происходят при транзакциях
 во время копирования базы данных временных рядов.


## Комментарии


Класс должен быть переопределен в пользовательском классе.
