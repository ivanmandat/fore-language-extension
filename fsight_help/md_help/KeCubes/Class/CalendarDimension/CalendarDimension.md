# CalendarDimension


## Описание


Класс CalendarDimension реализует объект, предназначенный для работы с элементами календарного справочника в кубе.


## Комментарии


Статические методы данного класса предназначены для формирования выражений, по которым будут рассчитываться [отношения](UiNavObj.chm::/Cube/CreateCube/Master_Standart/UiMd_Cube_CreateCube_Master_Standart_4.htm) в стандартном кубе.
