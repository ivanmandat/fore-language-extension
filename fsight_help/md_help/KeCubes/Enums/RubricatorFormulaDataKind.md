# RubricatorFormulaDataKind


## Описание


Перечисление RubricatorFormulaDataKind
 содержит виды формулы базы данных временных рядов.


Используется следующими свойствами и методами:


	- [IRubricatorFormulaData.Kind](../Interface/IRubricatorFormulaData/IRubricatorFormulaData.Kind.htm);


	- [IRubricatorFormulasLookup.WhereKind](../Interface/IRubricatorFormulasLookup/IRubricatorFormulasLookup.WhereKind.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 -1
		 Unknown. Вид не определен.


		 0
		 Transform. Формула.


		 2
		 Params. Параметры формулы.
