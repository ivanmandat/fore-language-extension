# MDCalculationFormulaBinaryOperation


## Описание


Перечисление MDCalculationFormulaBinaryOperation
 содержит виды бинарной операции, используемой в формуле многомерного расчета
 в качестве операнда.


Используется следующими свойствами и методами:


	- [IMDCalculationFormulaBinaryOperation.BinaryOperation](../Interface/IMDCalculationFormulaBinaryOperation/IMDCalculationFormulaBinaryOperation.BinaryOperation.htm).
