# CubeAttachmentType


## Описание


Перечисление CubeAttachmentType содержит
 типы вложений, хранящихся по различным координатам куба.


Используется следующими свойствами и методами:


	- [ICubeAttachment.Type](../Interface/ICubeAttachment/ICubeAttachment.Type.htm);


	- [ICubeAttachmentEdited.Type](../Interface/ICubeAttachmentEdited/ICubeAttachmentEdited.Type.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Тип вложения
		 не определён.


		 1
		 Value. Вложение хранится
		 в элементе справочника.


		 2
		 Document. Вложение
		 хранится как отдельный объект типа Документ.


		 3
		 URL. В качестве вложения
		 хранится ссылка на внешний файл.
