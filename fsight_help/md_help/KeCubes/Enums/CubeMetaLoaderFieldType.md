# CubeMetaLoaderFieldType


## Описание


Перечисление CubeMetaLoaderFieldType
 содержит типы полей источника данных.


Используется следующими свойствами и методами:


	- [ICubeMetaLoaderBinding.FieldType](../Interface/ICubeMetaLoaderBinding/ICubeMetaLoaderBinding.FieldType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None.
		 Тип не указан.


		 1
		 Index.
		 Поле определяется по номеру. Номер указывается в свойстве [ICubeMetaLoaderBinding.Field](../Interface/ICubeMetaLoaderBinding/ICubeMetaLoaderBinding.Field.htm).


		 2
		 Name.
		 Поле определяется по наименованию. Наименование указывается в
		 свойстве [ICubeMetaLoaderBinding.Field](../Interface/ICubeMetaLoaderBinding/ICubeMetaLoaderBinding.Field.htm).


		 3
		 ConstValue.
		 Постоянное значение, указанное в свойстве [ICubeMetaLoaderBinding.FieldValue](../Interface/ICubeMetaLoaderBinding/ICubeMetaLoaderBinding.FieldValue.htm).
