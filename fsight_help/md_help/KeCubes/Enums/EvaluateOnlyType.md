# EvaluateOnlyType


## Описание


Перечисление EvaluateOnlyType
 содержит наборы координат, по которым будет осуществляться
 расчёт результирующей матрицы.


Используется следующими свойствами и методами:


	- [ICalculatedCube.EvaluateOnly](../Interface/ICalculatedCube/ICalculatedCube.EvaluateOnly.htm);


	- [ICalculatedCubeExecutor.EvaluateOnly](../Interface/ICalculatedCubeExecutor/ICalculatedCubeExecutor.EvaluateOnly.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 All. Все
		 значения.


		 1
		 Selected. Отмеченные
		 координаты матрицы.


		 2
		 Unselected. Неотмеченные
		 координаты матрицы.


		 3
		 Data. Отмеченные координаты
		 матрицы, по которым имеются данные.
