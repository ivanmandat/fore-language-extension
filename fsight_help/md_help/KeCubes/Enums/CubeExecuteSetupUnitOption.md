# CubeExecuteSetupUnitOption


## Описание


Перечисление CubeExecuteSetupUnitOption
 содержит предопределенные типы единиц измерения, которые используются
 при расчете.


Используется следующими свойствами и методами:


	- [ICubeExecuteSetupUnits.UnitOption](../Interface/ICubeExecuteSetupUnits/ICubeExecuteSetupUnits.UnitOption.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Default_. Единицы измерения
		 по умолчанию (используются нормализованные данные).


		 1
		 Normalized. При расчете
		 используются нормализованные данные (приведенные к одним единицам
		 измерения).


		 2
		 Natural. При расчете
		 используются исходные данные в тех единицах измерения, в которых
		 они заданы.


		 3
		 Selected. При расчете
		 используется первая единица измерения, которая отмечена в справочнике
		 единиц измерения.
