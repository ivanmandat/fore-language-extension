# CubeDataRights


## Описание


Перечисление CubeDataRights
 содержит тип права доступа на данные.


Используется следующими свойствами и методами:


	- [ICubeExecuteDataRights.Rights](../Interface/ICubeExecuteSetup/ICubeExecuteSetup.RequiredDataRights.htm);


	- [ICubeExecuteDataRights.SupportedDataRights](../Interface/ICubeExecuteDataRights/ICubeExecuteDataRights.SupportedDataRights.htm);


	- [ICubeExecuteSetup.RequiredDataRights](../Interface/ICubeExecuteSetup/ICubeExecuteSetup.RequiredDataRights.htm);


	- [ILanerTable.DataRights](Laner.chm::/Interface/ILanerTable/ILanerTable.DataRights.htm);


	- [IPivotTable.DataRights](KePivot.chm::/Interface/IPivotTable/IPivotTable.DataRights.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None.
		 Нет права доступа.


		 1
		 Read.
		 Право на чтение данных.


		 2
		 Write.
		 Право на изменение данных.


		 3
		 ReadWrite.
		 Право на чтение и изменение данных.
