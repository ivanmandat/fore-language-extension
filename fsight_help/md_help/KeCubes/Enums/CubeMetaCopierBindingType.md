# CubeMetaCopierBindingType


## Описание


Перечисление CubeMetaCopierBindingType
 содержит типы привязки атрибутов показателей при их копировании из базы
 данных временных рядов.


Используется следующими свойствами и методами:


	- [ICubeMetaCopierBinding.BindingType](../Interface/ICubeMetaCopierBinding/ICubeMetaCopierBinding.BindingType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 Attribute. Копировать
		 значение атрибута показателей.


		 2
		 Const_. Постоянное
		 значение атрибута показателей.


		 3
		 ValueAttribute. Копировать
		 значение атрибута наблюдений.


		 4
		 ValueConst. Постоянное
		 значение атрибута наблюдений.


## Комментарии


Постоянные значения атрибутов показателей и наблюдений копируются из
 значения свойства [ICubeMetaCopierBinding.Value](../Interface/ICubeMetaCopierBinding/ICubeMetaCopierBinding.Value.htm).
