# ConvertBatchMode


## Описание


Перечисление ConvertBatchMode содержит способы обновления коллекции формул при конвертации.


Используется следующими свойствами и методами:


-
[IMDCalculationFormulasConverter.BatchMode](../Interface/IMDCalculationFormulasConverter/IMDCalculationFormulasConverter.BatchMode.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Undefined. Способ обновления не определен.


 1
 Insert. Только дополнять отсутствующими формулами.


 2
 Update. Только обновлять существующие формулы.


 3
 InsertUpdate. Обновлять существующие и дополнять отсутствующими формулами.
