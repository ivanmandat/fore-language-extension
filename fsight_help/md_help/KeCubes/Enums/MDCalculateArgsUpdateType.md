# MDCalculateArgsUpdateType


## Описание


Перечисление MDCalculateArgsUpdateType
 содержит способы обновления данных в приёмнике при расчете.


Используется следующими свойствами и методами:


	- [IMDCalculationCalculateArgs.UpdateType](../Interface/IMDCalculationCalculateArgs/IMDCalculationCalculateArgs.UpdateType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None.
		 Не изменять значения. Вставлять только новые.


		 1
		 All. Изменять
		 существующие значения. Вставлять новые.


		 2
		 NotNull.
		 Изменять, если вычисленное значение не пустое. В случае использования
		 данного режима необходимо, чтобы формулы для координат куба-назначения
		 были заданы по всем фактам. Для фактов, по которым отсутствуют
		 формулы, значения будут затерты.


		 3
		 NotNullByFacts.
		 Изменять, если вычисленное значение не пустое. В случае использования
		 данного режима, значения будут изменяться для фактов, по которым
		 заданы формулы.
