# ImportRequestSourceType


## Описание


Перечисление ImportRequestSourceType содержит типы источников данных для импорта показателей.


Используется следующим свойством:


-
[IImportRequestDefinition.SourceType](../Interface/IImportRequestDefinition/IImportRequestDefinition.SourceType.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Provider. Источник данных (кроме базы данных временных рядов).


 1
 Rubricator. База данных временных рядов.
