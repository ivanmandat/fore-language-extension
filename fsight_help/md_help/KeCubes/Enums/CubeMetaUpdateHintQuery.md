# CubeMetaUpdateHintQuery


## Описание


Перечисление CubeMetaUpdateHintQuery содержит
 типы запросов, выполняемых при репликации баз данных временных рядов.


Используется следующими свойствами и методами:


	- [ICubeMetaUpdateEx.UseTablockHint](../Interface/ICubeMetaUpdateEx/ICubeMetaUpdateEx.UseTablockHint.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 ExportFromActualData.
		 Запрос экспорта из таблиц актуальных данных.


		 2
		 ExportFromRevision.
		 Запрос экспорта из таблиц ревизионных данных.


		 3
		 ExportFromMultiVal.
		 Запрос экспорта из таблицы множественного атрибута.


		 4
		 FillMapTable. Запрос
		 вставки в таблицу сопоставления элементов справочника.


		 5
		 InsertGroupTable1.
		 Первый запрос, выполняемый при вставке для обновления данных групповых
		 записей.


		 6
		 InsertGroupTable2.
		 Второй запрос, выполняемый при вставке для обновления данных групповых
		 записей.


		 7
		 InsertGroupTable3.
		 Третий запрос, выполняемый при вставке для обновления данных групповых
		 записей.


		 8
		 InsertGroupTable4.
		 Четвёртый запрос, выполняемый при вставке для обновления данных
		 групповых записей.


		 9
		 InsertFactsTable.
		 Запрос вставки для обновления фактов.


		 10
		 InsertValsTable. Запрос
		 вставки для обновления значений.


		 11
		 InsertMultiValMapTable.
		 Запрос вставки для обновления таблицы сопоставления множественного
		 атрибута.


		 12
		 InsertMultiValTable.
		 Запрос вставки для обновления таблицы множественного атрибута.


		 13
		 InsertFactorMapTable.
		 Запрос вставки для обновления таблицы сопоставления рядов (факторов).


		 14
		 InsertFactorfilteredMapTable.
		 Запрос вставки для обновления таблицы сопоставления рядов (факторов)
		 с учётом прав.


		 15
		 InsertGenerateNewFactorKeys.
		 Запрос вставки для генерации новых ключей рядов.


		 16
		 InsertPeriodsTable.
		 Запрос вставки для таблицы периодов.
