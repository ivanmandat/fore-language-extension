# MDCalculationFormulaDatePeriod


## Описание


Перечисление MDCalculationFormulaDatePeriod
 содержит виды периода, по которому будет рассчитываться значение функции
 [Shift](UiMd.chm::/Cube/Work_Cube/UiMd_CubeWork_MultivariateAccount_Functions.htm)
 в формуле.


Используется следующими свойствами и методами:


	- [IMDCalculationFormulaDatePeriod.DatePeriod](../Interface/IMDCalculationFormulaDatePeriod/IMDCalculationFormulaDatePeriod.DatePeriod.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None.
		 Период не определен.


		 66
		 Day. Дни.


		 67
		 Month.
		 Месяцы.


		 68
		 Quart.
		 Кварталы.


		 70
		 Year.
		 Годы.
