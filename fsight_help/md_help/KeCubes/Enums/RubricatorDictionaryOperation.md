# RubricatorDictionaryOperation


## Описание


Перечисление RubricatorDictionaryOperation
 содержит типы операций со справочниками базы данных временных рядов.


Используется следующими свойствами и методами:


	- [IRubricator.GetDictionary](../Interface/IRubricator/IRubricator.GetDictionary.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None.
		 Операция не определена.


		 1
		 Create.
		 Создание справочника.


		 2
		 Edit.
		 Редактирование справочника.


		 16
		 ForEachBasis.
		 Выполнять операцию только для каждого базового справочника.


		 32
		 BasisOnly.
		 Выполнять операцию только для базовых справочников.


		 49
		 CreateBasis.
		 Создание всех базовых справочников.


		 50
		 EditBasis.
		 Редактирование всех базовых справочников.


		 64
		 SaveOnCreate.
		 Сохранение в репозиторий (только для базовых справочников).
