# RubricatorContainerType


## Описание


Перечисление RubricatorContainerType
 содержит типы объектов, образующих базу данных временных рядов.


Используется следующими свойствами и методами:


	- [IRubricator.Container](../Interface/IRubricator/IRubricator.Container.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Unknown.
		 Тип объекта не определен.


		 1
		 Data.
		 Папка данных.


		 2
		 Validations.
		 Папка проверок (валидаций).


		 3
		 Favorites.
		 Папка избранного.


		 4
		 Modelspace.
		 Контейнер моделирования.


		 5
		 UserFavorites.
		 Личная папка.


		 7
		 Database.
		 Репозиторий НСИ.


		 8
		 SegmentsFolder.
		 Папка сегментов.
