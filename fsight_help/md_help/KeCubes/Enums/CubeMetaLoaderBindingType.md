# CubeMetaLoaderBindingType


## Описание


Перечисление CubeMetaLoaderBindingType
 содержит типы привязки атрибутов базы данных временных рядов с полями
 источника данных при импорте показателей.


Используется следующими свойствами и методами:


	- [ICubeMetaLoaderBinding.BindingType](../Interface/ICubeMetaLoaderBinding/ICubeMetaLoaderBinding.BindingType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Привязка не определена.


		 1
		 Attribute. Привязка
		 к атрибуту показателя.


		 2
		 Dimension. Привязка
		 к справочнику.


		 3
		 Calendar. Привязка
		 к календарю.


		 4
		 Value. Привязка к значению
		 атрибута.


		 5
		 Unit. Привязка к единицам
		 измерения.


		 6
		 ValueAttribute.
		 Привязка к атрибуту наблюдения.


		 7
		 CheckAttribute.
		 Привязка для проверки значения атрибута.
