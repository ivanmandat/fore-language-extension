# CubeValidationAssertionSeverity


## Описание


Перечисление CubeValidationAssertionSeverity содержит
 коды важности сообщений о нарушениях структуры метаданных куба.


Используется следующими свойствами и методами:


	- [ICubeValidationAssertion.Severity](../Interface/ICubeValidationAssertion/ICubeValidationAssertion.Severity.htm);


	- [ICubeValidationAssertions.MaxSeverity](../Interface/ICubeValidationAssertions/ICubeValidationAssertions.MaxSeverity.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Нет нарушений.


		 1
		 Info. Информационное
		 сообщение.


		 2
		 Warning. Предупреждающее
		 сообщение.


		 3
		 Error. Сообщение об
		 ошибке в структуре метаданных.
