# SaveRubricatorDataOptions


## Описание


Перечисление SaveRubricatorDataOptions
 содержит опции сохранения данных базы данных временных рядов.


Используется следующими свойствами и методами:


	- [IRubricatorFactData.SaveAndRevise](../Interface/IRubricatorFactData/IRubricatorFactData.SaveAndRevise.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Default_.
		 Обычное сохранение данных.


		 2
		 NoValidateRules.
		 Сохранение без выполнения правил валидации.


		 4
		 NoForce.
		 Сохранение без перезаписи данных.


		 8
		 NoRevise.
		 Сохранение в текущую ревизию.


		 16
		 NoEmptyCheck. Сохранение
		 данных ряда без проверки ряда на пустоту.
