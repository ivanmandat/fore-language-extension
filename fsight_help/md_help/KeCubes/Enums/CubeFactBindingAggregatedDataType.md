# CubeFactBindingAggregatedDataType


## Описание


Перечисление CubeFactBindingAggregatedDataType
 содержит типы данных, к которым может быть преобразовано значение после
 агрегации.


Используется следующими свойствами и методами:


	- [IStandardCubeFactBinding.AggregatedDataType](../Interface/IStandardCubeFactBinding/IStandardCubeFactBinding.AggregatedDataType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None.
		 Не преобразовывать тип данных.


		 1
		 FixedPoint.
		 Число с фиксированной точкой.


		 2
		 FloatingPoint.
		 Число с плавающей точкой.
