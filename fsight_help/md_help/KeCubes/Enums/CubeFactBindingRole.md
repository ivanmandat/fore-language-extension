# CubeFactBindingRole


## Описание


Перечисление CubeFactBindingRole используется
 для определения роли привязки фактов в кубе.


Используется следующими свойствами и методами:


	- [IStandardCubeFactBinding.BindingRole](../Interface/IStandardCubeFactBinding/IStandardCubeFactBinding.BindingRole.htm);


	- [IStandardCubeFactBindings.BindingWithRole](../Interface/IStandardCubeFactBindings/IStandardCubeFactBindings.BindingWithRole.htm);


	- [IStandardCubeFactBindings.ResetWithRole](../Interface/IStandardCubeFactBindings/IStandardCubeFactBindings.ResetWithRole.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Value. Привязка для
		 хранения значений.


		 2
		 Attachment. Привязка
		 для хранения вложений.


		 3
		 Comment. Привязка для
		 хранения комментариев к ячейкам.
