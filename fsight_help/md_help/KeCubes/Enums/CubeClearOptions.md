# CubeClearOptions


## Описание


Перечисление CubeClearOptions
 используется для определения параметров очистки базы данных временных
 рядов.


Используется следующим методом:


	- [IRubricatorInstance.ClearEx](../Interface/IRubricatorInstance/IRubricatorInstance.ClearEx.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Default. Создавать
		 новую ревизию, очищать все указанные атрибуты рядов.


		 1
		 NoCreateRevision. Не
		 создавать новую ревизию.


		 2
		 GroupsOnly. Очищать
		 только групповые атрибуты.


		 4
		 NoCreateTransaction.
		 Не создавать новую транзакцию.
