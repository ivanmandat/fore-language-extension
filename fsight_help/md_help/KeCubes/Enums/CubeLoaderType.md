# CubeLoaderType


## Описание


Перечисление CubeLoaderType
 содержит типы объектов репозитория, в которые будет осуществляться загрузка
 данных.


Используется следующими свойствами и методами:


	- [ICubeLoaderFromSource.Type](../Interface/ICubeLoaderFromSource/ICubeLoaderFromSource.Type.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 Cube. Загрузка в новый/существующий
		 стандартный куб.


		 2
		 Rubricator. Загрузка
		 в новую/существующую базу данных временных рядов.
