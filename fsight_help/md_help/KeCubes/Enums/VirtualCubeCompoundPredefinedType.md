# VirtualCubeCompoundPredefinedType


## Описание


Перечисление VirtualCubeCompoundPredefinedType
 содержит типы предопределенных составных структур виртуального куба.


Используется следующими свойствами и методами:


	- [IVirtualCubeCompound.Predefined](../Interface/IVirtualCubeCompound/IVirtualCubeCompound.Predefined.htm);


	- [IVirtualCubeCompounds.Predefined](../Interface/IVirtualCubeCompounds/IVirtualCubeCompounds.Predefined.htm);


	- [IVirtualCubeDimensionFix.PredefinedCompound](../Interface/IVirtualCubeDimensionFix/IVirtualCubeDimensionFix.PredefinedCompound.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Custom. Пользовательская
		 структура.


		 1
		 Fact. Факты.


		 2
		 Calendar. Календарь.


		 3
		 Scenario. Сценарии.


## Комментарии


Виртуальный куб всегда содержит следующие предопределенные структуры:
 Fact, Calendar,
 Scenario.
