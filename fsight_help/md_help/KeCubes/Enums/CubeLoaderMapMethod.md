# CubeLoaderMapMethod


## Описание


Перечисление CubeLoaderMapMethod
 содержит способы связи между измерением-источником и измерением-приёмником.


Используется следующими свойствами и методами:


	- IAutoCubeDimensionTransform.MapMethod;


	- [IAutoCubeDimension.Method](../Interface/IAutoCubeDimension/IAutoCubeDimension.Method.htm);


	- [ICubeLoaderMapItem.Method](../Interface/ICubeLoaderMapItem/ICubeLoaderMapItem.Method.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None.
		 Связь отсутствует.


		 1
		 ById.
		 Связь по идентификатору.


		 2
		 ByIndex.
		 Связь по индексу.


		 3
		 ByElement.
		 Связь по номеру элемента (без преобразования).
