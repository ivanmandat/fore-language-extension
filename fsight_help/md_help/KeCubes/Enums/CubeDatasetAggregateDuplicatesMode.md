# CubeDatasetAggregateDuplicatesMode


## Описание


Перечисление CubeDatasetAggregateDuplicatesMode содержит
 варианты мест где будет осуществляться агрегация дублирующихся данных.


Используется следующими свойствами и методами:


	- [IStandardCubeDataset.AggregateDuplicatesMode](../Interface/IStandardCubeDataset/IStandardCubeDataset.AggregateDuplicatesMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Server. Агрегация на
		 сервере СУБД.


		 1
		 Client. Агрегация на
		 компьютере-клиенте (используется, если сервер СУБД не поддерживает
		 агрегацию).
