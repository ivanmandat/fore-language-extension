# RubricatorSegmentType


## Описание


Перечисление RubricatorSegmentType
 содержит типы сегмента базы данных временных рядов.


Используется следующими свойствами и методами:


	- [IRubricatorSegment.Type](../Interface/IRubricatorSegment/IRubricatorSegment.Type.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Unknown.
		 Тип сегмента не определен.


		 1
		 Factors.
		 Диапазон данных.


		 2
		 Time.
		 Временной диапазон.
