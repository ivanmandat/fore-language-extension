# CubeMetaLoaderLoadType


## Описание


Перечисление CubeMetaLoaderLoadType
 содержит режимы загрузки показателей.


Используется следующими свойствами и методами:


	- [ICubeMetaLoader.LoadType](../Interface/ICubeMetaLoader/ICubeMetaLoader.LoadType.htm);


	- [IImportRequestProviderParams.LoadType](../Interface/IImportRequestProviderParams/IImportRequestProviderParams.LoadType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None.
		 Режим загрузки не указан.


		 1
		 CreateOnly.
		 При загрузке только создавать новые показатели.


		 2
		 UpdateOnly.
		 При загрузке только обновлять существующие показатели.


		 3
		 CreateAndUpdate.
		 При загрузке создавать новые и обновлять существующие показатели.
