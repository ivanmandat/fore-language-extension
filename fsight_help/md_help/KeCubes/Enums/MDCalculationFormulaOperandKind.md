# MDCalculationFormulaOperandKind


## Описание


Перечисление MDCalculationFormulaOperandKind
 содержит виды операнда, используемого в формуле.


Используется следующими свойствами и методами:


	- [IMDCalculationFormulaExpression.CreateItem](../Interface/IMDCalculationFormulaExpression/IMDCalculationFormulaExpression.CreateItem.htm);


	- [IMDCalculationFormulaOperand.Kind](../Interface/IMDCalculationFormulaOperand/IMDCalculationFormulaOperand.Kind.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Вид
		 операнда не определен.


		 1
		 BinaryOperation.
		 Бинарная операция.


		 2
		 UnaryOperation.
		 Унарная операция.


		 3
		 Delimiter.
		 Разделитель.


		 4
		 FunctionCall.
		 Специальная функция.


		 5
		 ConstantValue.
		 Постоянное значение.


		 6
		 Element.
		 Значение элемента источника/приёмника данных.


		 7
		 DatePeriod.
		 Период времени.
