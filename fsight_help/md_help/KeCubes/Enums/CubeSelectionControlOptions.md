# CubeSelectionControlOptions


## Описание


Перечисление CubeSelectionControlOptions
 содержит варианты отключения управления параметрами через измерение при
 редактировании формул через управляющее измерение.


Используется следующими свойствами и методами:


	- [ICubeSelectionControl.Options](../Interface/ICubeSelectionControl/ICubeSelectionControl.Options.htm);


	- [ICubeSelectionControlApply.Options](../Interface/ICubeSelectionControlApply/ICubeSelectionControlApply.Options.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Default. Отключение
		 управления недоступно.


		 1
		 PermanentParamControl.
		 Не применять повторно управление при настроенном управлении через
		 вложенные объекты.
