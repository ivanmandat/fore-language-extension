# CubeFormulasViewSourceNaming


## Описание


Перечисление CubeFormulasViewSourceNaming
 содержит форматы обозначения источников данных в наименованиях элементов
 при составлении формул.


Используется следующими свойствами и методами:


	- [ICalculatedCubeFormulasView.SourceNaming](../Interface/ICalculatedCubeFormulasView/ICalculatedCubeFormulasView.SourceNaming.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Источник данных
		 никак не обозначен.


		 1
		 Name. Перед наименованием
		 элементов отображается наименование источника данных: <Наименование
		 источника> | <Элемент измерения 1> | <Элемент измерения
		 2>...


		 2
		 Id. Перед наименованием
		 элементов измерений отображается идентификатор источника данных:
		 <Идентификатор источника> | <Элемент измерения 1>
		 | <Элемент измерения 2>...


		 3
		 NameId. Перед наименованием
		 элементов измерений отображается наименование и идентификатор
		 источника данных: <Наименование источника(Идентификатор источника)>
		 | <Элемент измерения 1> | <Элемент измерения 2>...
