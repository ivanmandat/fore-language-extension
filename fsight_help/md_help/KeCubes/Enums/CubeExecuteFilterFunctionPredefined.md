# CubeExecuteFilterFunctionPredefined


## Описание


Перечисление CubeExecuteFilterFunctionPredefined содержит
 типы функций, которые могут использоваться для фильтрации данных.


Используется следующими свойствами и методами:


	- [ICubeExecuteFilterFunction.Predefined](../Interface/ICubeExecuteFilterFunction/ICubeExecuteFilterFunction.Predefined.htm);


	- [ICubeExecuteFilterFunctionFactory.CreatePredefined](../Interface/ICubeExecuteFilterFunctionFactory/ICubeExecuteFilterFunctionFactory.CreatePredefined.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Тип функции не
		 определен.


		 1
		 Top. Функции отбора
		 по наибольшим/наименьшим значениям:


			- TOPCOUNT;

			- TOPPERCENT/BOTTOMPERCENT;

			- TOPSUM/BOTTOMSUM;


		 2
		 Filter. Функция FILTER
		 для фильтрации по заданному условию.


		 3
		 Order. Функции сортировки:


			- ASC;

			- DESC;

			- BASC;

			- BDESC.


		 4
		 NonEmpty. Функция фильтрации
		 пустых значений NON EMPTY.
