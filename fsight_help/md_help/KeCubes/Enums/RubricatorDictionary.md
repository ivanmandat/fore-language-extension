# RubricatorDictionary


## Описание


Перечисление RubricatorDictionary
 содержит типы справочников, содержащих данные базы данных временных рядов.


Используется следующими свойствами и методами:


	- [ICubeMetaUpdateEx.SourceDictionary](../Interface/ICubeMetaUpdateEx/ICubeMetaUpdateEx.SourceDictionary.htm);


	- [IRubricator.GetDictionary](../Interface/IRubricator/IRubricator.GetDictionary.htm);


	- [IRubricator.SetDictionary](../Interface/IRubricator/IRubricator.SetDictionary.htm);


	- [IRubricatorCreatorBinding.Dictionary](../Interface/IRubricatorCreatorBinding/IRubricatorCreatorBinding.Dictionary.htm);


	- [IRubricatorExecuteResult.DataMatrix](../Interface/IRubricatorExecuteResult/IRubricatorExecuteResult.DataMatrix.htm);


	- [IRubricatorExecuteResult.DataAttributes](../Interface/IRubricatorExecuteResult/IRubricatorExecuteResult.DataAttributes.htm);


	- [IRubricatorInstance.GetDictionary](../Interface/IRubricatorInstance/IRubricatorInstance.GetDictionary.htm).
