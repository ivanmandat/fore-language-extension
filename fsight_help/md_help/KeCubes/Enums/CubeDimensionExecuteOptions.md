# CubeDimensionExecuteOptions


## Описание


Перечисление CubeDimensionExecuteOptions
 содержит опции измерения.


Используется следующими свойствами и методами:


	- [ICubeExecuteDimSetup.Options](../Interface/ICubeExecuteDimSetup/ICubeExecuteDimSetup.Options.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None.
		 Опции не определены.


		 1
		 Required.
		 Измерение обязательное.


		 2
		 SupportAggreration.
		 По элементам измерения настроена агрегация данных.


		 4
		 SupportCompound. Измерение
		 может открываться как составное.
