# MDCalculateArgsCleanType


## Описание


Перечисление MDCalculateArgsCleanType
 содержит способы очистки приёмника данных перед расчетом.


Используется следующими свойствами и методами:


	- [IMDCalculationCalculateArgs.CleanType](../Interface/IMDCalculationCalculateArgs/IMDCalculationCalculateArgs.CleanType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None.
		 Не очищать приёмник данных.


		 1
		 Existing.
		 Очистить расчитываемые элементы.


		 2
		 All. Очистить
		 все элементы за указанный период. Если период не указан - очищаются
		 все элементы.
