# MDCalculationSliceFixType


## Описание


Перечисление MDCalculationSliceFixType
 содержит типы измерений источника/приёмника данных.


Используется следующими свойствами и методами:


	- [IMDCalculationSlice.FixType](../Interface/IMDCalculationSlice/IMDCalculationSlice.FixType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Default_.
		 Значение не установлено.


		 1
		 Fixed.
		 Измерение является фиксированным. Для фиксированного измерения
		 источника данных должно быть поставлено в соответствие измерение
		 приёмника данных.


		 2
		 Free.
		 Измерение является свободным.
