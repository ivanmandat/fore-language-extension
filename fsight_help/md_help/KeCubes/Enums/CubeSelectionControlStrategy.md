# CubeSelectionControlStrategy


## Описание


Перечисление CubeSelectionControlOptions
 содержит варианты стратегии управления отметкой.


Используется следующими свойствами и методами:


	- [ICubeSelectionControlItem.SelectionStrategy](../Interface/ICubeSelectionControlItem/ICubeSelectionControlItem.SelectionStrategy.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Default. Отметка тех
		 элементов, которые были отмечены до управления.


		 1
		 SelectAll. Отметка
		 всех элементов в управляемом измерении.


		 2
		 DesalectAll. Сброс
		 отметки со всех элементов в управляемом измерении.


		 3
		 Copy. Отметка тех элементов,
		 которые были отмечены до управления.


		 4
		 SelectFirst. Отметка
		 первого элемента.


		 5
		 SelectLast. Отметка
		 последнего элемента.
