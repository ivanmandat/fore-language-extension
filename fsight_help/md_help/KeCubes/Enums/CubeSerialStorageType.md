# CubeSerialStorageType


## Описание


Перечисление CubeSerialStorageType
 содержит типы хранилища для куба.


Используется следующими свойствами и методами:


	- [ICubeSerializer.StorageType](../Interface/ICubeSerializer/ICubeSerializer.StorageType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Default_. Значение
		 по умолчанию.


		 1
		 SingleFile. Куб будет
		 сохранен в один файл.


		 2
		 MappedFileCatalog.
		 Куб будет сохранен в несколько файлов (по каждому варианту отображения).
