# TransformModelPeriodSet


## Описание


Перечисление TransformModelPeriodSet
 содержит параметры периода расчета.


Используется следующими свойствами и методами:


	- [IFormulaTransformModel.PeriodSet](../Interface/IFormulaTransformModel/IFormulaTransformModel.PeriodSet.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Параметры периода
		 расчета не заданы.


		 1
		 IdentificationStartDate.
		 Начало идентификации данных.


		 2
		 IdentificationEndDate.
		 Окончание идентификации данных.


		 4
		 ForecastStartDate.
		 Начало прогнозирования данных.


		 8
		 ForecastEndDate. Окончание
		 прогнозирования данных.


		 15
		 All. Начало, окончание
		 идентификации данных и начало, окончание прогнозирования данных.


## Комментарии


Для указания нескольких параметров периода расчета указывайте необходимые
 значения с использованием логической операции «Or».
