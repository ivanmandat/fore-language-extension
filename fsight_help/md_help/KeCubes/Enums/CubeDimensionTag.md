# CubeDimensionTag


## Описание


Перечисление CubeDimensionTag
 содержит типы измерений куба.


Используется следующими свойствами и методами:


	- [ICubeExecuteDimSetup.Tag](../Interface/ICubeExecuteDimSetup/ICubeExecuteDimSetup.Tag.htm);


	- [ICubeExecuteSetup.FindByTag](../Interface/ICubeExecuteSetup/ICubeExecuteSetup.FindByTag.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None.
		 Тип измерения не определен.


		 1
		 Calendar.
		 Календарное измерение.


		 2
		 Facts.
		 Измерение фактов.


		 3
		 Units.
		 Единицы измерения.
