# MDCalculationFormulaUnaryOperation


## Описание


Перечисление MDCalculationFormulaUnaryOperation
 содержит виды унарной операции, используемой в формуле в качестве операнда.


Используется следующими свойствами и методами:


	- [IMDCalculationFormulaUnaryOperation.UnaryOperation](../Interface/IMDCalculationFormulaUnaryOperation/IMDCalculationFormulaUnaryOperation.UnaryOperation.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None.
		 Операция не задана.


		 2
		 Not_.
		 Логическое Not.


		 3
		 Minus.
		 Унарный минус.
