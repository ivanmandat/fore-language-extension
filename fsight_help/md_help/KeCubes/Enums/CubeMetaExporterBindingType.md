# CubeMetaExporterBindingType


## Описание


Перечисление CubeMetaExporterBindingType
 содержит типы привязки атрибутов базы данных временных рядов с полями
 приёмника данных при экспорте показателей.


Используется следующими свойствами и методами:


	- [ICubeMetaExporterBinding.BindingType](../Interface/ICubeMetaExporterBinding/ICubeMetaExporterBinding.BindingType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 Attribute.
		 Привязка к атрибуту показателя.


		 2
		 Calendar.
		 Привязка к календарю.


		 3
		 Value.
		 Привязка к значению атрибута.


		 4
		 ValueAttribute.
		 Привязка к атрибуту наблюдения.


		 5
		 Calculated. Вычисляемая
		 привязка, выражение задаётся с помощью свойства [ICubeMetaExporterBinding.Expression](../Interface/ICubeMetaExporterBinding/ICubeMetaExporterBinding.Expression.htm).
