# IFinance.Nominal


## Синтаксис


Nominal(EffectRate: Double; PeriodCount: Integer): Double;


## Параметры


		 Параметры
		 Описание
		 Ограничения


		 EffectRate
		 Фактическая процентная ставка.
		 Должен быть положительным.


		 PeriodCount
		 Количество периодов в году, за которое начисляются сложные
		 проценты.
		 Должен быть положительным.


## Описание


Метод Nominal возвращает номинальную
 годовую ставку, если заданы эффективная (фактическая) ставка и число периодов
 в году, за которые начисляются сложные проценты.
