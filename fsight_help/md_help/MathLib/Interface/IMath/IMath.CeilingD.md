# IMath.CeilingD


## Синтаксис


CeilingD(Value: Decimal; Precision: Decimal): Decimal;


## Параметры


Value. Округляемое
 число высокой точности;


Precision. Кратное
 высокой точности, до которого требуется округлить значение.


## Описание


Метод CeilingD возвращает результат
 округления с избытком до ближайшего числа высокой точности, кратного значению
 точности.


## Комментарии


Для обозначения в массиве чисел типа [Decimal](fore.chm::/03_datatypes/Fore_DataTypes.htm)
 после значения каждого числа необходимо дописать символ «m». Например:
 0.001m.


Если число и точность положительны, значение округляется вверх.


Если число и точность отрицательны, значение округляется вниз.


Если число отрицательно, а точность положительна, значение округляется
 вверх.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


					Sub UserProc;

		Var

		    r1,r2,r3: Decimal;

		Begin

		    r1 := Math.CeilingD(1.554951234587512484m,0.1m);

		    Debug.WriteLine(r1);

		    r2 := Math.CeilingD(-2.5m,-2.365481235841654m);

		    Debug.WriteLine(r2);

		    r3 := Math.CeilingD(-2.55634576215697545m,2.54m);

		    Debug.WriteLine(r3);

		End Sub UserProc;


В результате выполнения примера в окно консоли будут выведены результаты:


	- число и точность положительны: 1.6;


	- число и точность отрицательны: -4.730962471683308;


	- число отрицательно, а точность положительна: -2.54.
