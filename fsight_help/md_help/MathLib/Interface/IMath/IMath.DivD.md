# IMath.DivD


## Синтаксис


DivD(Value: Decimal; Divisor: Decimal): Decimal;


## Параметры


Value. Делимое;


Divisor. Делитель.


## Описание


Метод DivD возвращает целую
 часть частного при делении двух десятичных чисел высокой точности.


## Комментарии


Для обозначения в массиве чисел типа [Decimal](fore.chm::/03_datatypes/Fore_DataTypes.htm)
 после значения каждого числа необходимо дописать символ «m». Например:
 0.001m.


Значение параметра Divisor
 не может быть равно 0.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


					Sub UserProc;

		Var

		    r: Decimal;

		Begin

		    r := Math.DivD(28.1m, 7m);

		    Debug.WriteLine(r);

		End Sub UserProc;


В результате выполнения примера в окно консоли будет выведена целая
 часть частного, равная 4.
