# IMath.Odd


## Синтаксис


Odd(Value: Double): Integer;


## Параметры


Value. Округляемое значение.


## Описание


Метод Odd округляет указанное
 значение до ближайшего нечетного целого.


## Комментарии


Диапазон допустимых значений входного параметра: [MinInteger; MaxInteger].
 MinInteger = -231, MaxInteger = 231 - 1.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Integer;

Begin

    r := Math.Odd(1.5);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен результат
 округления до ближайшего нечетного целого, равный «3».
