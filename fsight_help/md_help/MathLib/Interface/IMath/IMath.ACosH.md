# IMath.ACosH


## Синтаксис


ACosH(Value: Double): Double;


## Параметры


Value. Гиперболический косинус
 угла.


## Описание


Метод ACosH возвращает гиперболический
 арккосинус заданного числа.


## Комментарии


Значение параметра Value не
 может быть пустым и должно быть больше или равно «1».


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.ACosH(10);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен гиперболический
 арккосинус, равный «2.99322284612638».
