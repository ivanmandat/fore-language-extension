# IMath.Rand


## Синтаксис


Rand: Double;


## Описание


Метод Rand возвращает равномерно
 распределенное случайное вещественное число из промежутка [0, 1].


## Комментарии


Для получения случайного числа, находящегося в диапазоне между двумя
 заданными значениями, используйте метод [IMath.RandBetween](IMath.RandBetween.htm).


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.Rand;

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведено случайное
 равномерно распределенное число.
