# IMath.Gcd


## Синтаксис


Gcd(Values: Array): Integer;


## Параметры


Values.
 Массив целых чисел.


## Описание


Метод Gcd возвращает наибольший
 общий делитель.


## Комментарии


Необходимо использовать массив значений типа [Integer](Fore.chm::/03_dataTypes/Fore_DataTypes.htm).


Функция корректно работает как с одномерными, так и с двумерными массивами.


Для получения наименьшего общего кратного используйте метод [IMath.Lcm](IMath.Lcm.htm).


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


					Sub UserProc;

		Var r: Double;

		    arr: Array[5] Of Integer;

		Begin

		    arr[0] := 24;

		    arr[1] := 64;

		    arr[2] := 32;

		    arr[3] := 8;

		    arr[4] := 16;

		    r:= math.Gcd(arr);

		    Debug.WriteLine(r);

		End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен наибольший
 общий делитель, равный 8.
