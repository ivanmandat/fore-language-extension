# IMath.RndSample


## Синтаксис


RndSample(N: Integer; M: Integer): Array;


## Параметры


N. Длина
 массива чисел;


M. Количество чисел для
 выборки.


## Описание


Метод RndSample осуществляет
 псевдослучайную выборку из массива чисел.


## Комментарии


Значение параметра M не должно
 превышать значение параметра N.


Значения параметров M и N должны быть не меньше, чем единица.
