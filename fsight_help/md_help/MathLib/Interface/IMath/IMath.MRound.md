# IMath.MRound


## Синтаксис


MRound(Value: Double; Precision: Double): Double;


## Параметры


Value. Округляемое
 значение;


Precision. Точность округления.


## Описание


Метод MRound возвращает число,
 округленное с указанной точностью.


## Комментарии


Округление производится в том случае, если остаток от деления числа
 на точность больше или равен половине точности.


Значение параметра Precision
 не может быть равно «0».


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.MRound(10, 3);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен результат
 округления, равный «9».
