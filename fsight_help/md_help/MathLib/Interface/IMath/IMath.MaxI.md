# IMath.MaxI


## Синтаксис


MaxI(Values: Array): Integer;


## Параметры


Values. Массив целых чисел.


## Описание


Метод MaxI возвращает наибольшее
 целое значение из заданного набора значений.


## Комментарии


Необходимо использовать массив значений типа [Integer](Fore.chm::/03_dataTypes/Fore_DataTypes.htm).


Функция корректно работает как с одномерными, так и с двумерными массивами.


Для получения наименьшего значения из заданного набора используйте метод
 [IMath.Min](IMath.Min.htm).


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


					Sub UserProc;

		Var r:Double;

		    arr:array[5] Of Integer;

		Begin

		    Arr[0] := 10;

		    Arr[1] := 7;

		    Arr[2] := 9;

		    Arr[3] := 27;

		    Arr[4] := 2;

		    r:= math.MaxI(arr);

		    Debug.WriteLine(r);

		End Sub UserProc;


В результате выполнения примера в окно консоли будет выведено наибольшее
 значение, равное 27.
