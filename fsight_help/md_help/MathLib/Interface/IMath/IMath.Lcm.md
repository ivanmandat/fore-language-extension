# IMath.Lcm


## Синтаксис


Lcm(Values: Array): Double;


## Параметры


Values.
 Массив целых чисел.


## Описание


Метод Lcm возвращает наименьшее
 общее кратное.


## Комментарии


Необходимо использовать массив значений типа [Integer](Fore.chm::/03_dataTypes/Fore_DataTypes.htm).


Функция корректно работает как с одномерными, так и с двумерными массивами.


Для получения наибольшего общего делителя используйте метод [IMath.Gcd](IMath.Gcd.htm).


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


					Sub UserProc;

		Var r:Double;

		    arr:array[5] Of Integer;

		Begin

		    arr[0] := 24;

		    arr[1] := 64;

		    arr[2] := 32;

		    arr[3] := 8;

		    arr[4] := 16;

		    r:= math.Lcm(arr);

		    Debug.WriteLine(r);

		End Sub UserProc;


В результате выполнения примера в окно консоли будет выведено наименьшее
 общее кратное, равное 192.
