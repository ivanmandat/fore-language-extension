# IMath.Cos


## Синтаксис


Cos(Value: Double): Double;


## Параметры


Value.
 Угол в радианах, косинус которого необходимо найти.


## Описание


Метод Cos возвращает косинус
 заданного угла.


## Комментарии


Значение параметра Value должно
 быть задано конечным числом.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.Cos(1.047);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен косинус
 заданного угла, равный «0.50017107459707».
