# IMath.SumXMY2


## Синтаксис


SumXMY2(X: Array; Y: Array): Double;


## Параметры


X. Массив вещественных чисел;


Y. Массив вещественных чисел.


## Описание


Метод SumXMY2 возвращает сумму
 квадратов разностей соответствующих значений в двух массивах.


## Комментарии


Массивы X и Y
 должны быть одинаковой размерности.


Функция корректно работает как с одномерными, так и с двумерными массивами.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

    X: Array[7] Of Double;

    Y: Array[7] Of Double;

Begin

    X[0] := 2; Y[0] := 6;

    X[1] := 3; Y[0] := 5;

    X[2] := 9; Y[0] := 11;

    X[3] := 1; Y[0] := 7;

    X[4] := 8; Y[0] := 6;

    X[5] := 7; Y[0] := 4;

    X[6] := 5; Y[0] := 4;

    r := Math.SumXMY2(X, Y);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведена сумма
 квадратов разностей, равная «233».
