# IMath.SinH


## Синтаксис


SinH(Value: Double): Double;


## Параметры


Value.
 Вещественное число, гиперболический синус которого необходимо найти.


## Описание


Метод SinH возвращает гиперболический
 синус заданного числа.


## Комментарии


Значение параметра Value должно
 быть задано конечным числом.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.SinH(4);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен гиперболический
 синус, равный «27.2899171971278».
