# IMath.TanH


## Синтаксис


TanH(Angle: Double): Double;


## Параметры


Angle.
 Вещественное число, гиперболический тангенс которого необходимо найти.


## Описание


Метод TanH возвращает гиперболический
 тангенс заданного числа.


## Комментарии


Значение параметра не может быть пустым.


Для получения обратного гиперболического тангенса используйте метод
 [IMath.ATanH](IMath.ATanH.htm).


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.TanH(-2);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен гиперболический
 тангенс, равный «-0.964027580075817».
