# IMath.Create


## Синтаксис


Create;


## Описание


Метод Create создает экземпляр объекта интерфейса.


## Комментарии


Для каждого объекта, реализующего математические функции, метод вызывается автоматически, поэтому дополнительный его вызов не требуется.
