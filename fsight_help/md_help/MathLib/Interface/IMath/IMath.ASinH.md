# IMath.ASinH


## Синтаксис


ASinH(Value: Double): Double;


## Параметры


Value.
 Гиперболический синус угла.


## Описание


Метод ASinH возвращает гиперболический
 арксинус заданного числа.


## Комментарии


Значение параметра не может быть пустым.


Для получения гиперболического синуса угла используйте метод [IMath.SinH](IMath.SinH.htm).


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.ASinH(-2.5);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен гиперболический
 арксинус, равный «-1.6472311463711».
