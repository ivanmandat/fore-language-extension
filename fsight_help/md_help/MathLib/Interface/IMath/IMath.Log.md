# IMath.Log


## Синтаксис


Log(Value: Double; Base: Double): Double;


## Параметры


Value.
 Положительное вещественное число, для которого вычисляется
 логарифм;


Base.
 Основание логарифма.


## Описание


Метод Log возвращает логарифм
 числа по заданному основанию.


## Комментарии


Значения входных параметров не могут быть пустыми.


Значение параметра Base должно
 принадлежать промежуткам (0, 1) U (1, inf).


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.Log(8,2);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен результат
 вычисления логарифма, равный «3».
