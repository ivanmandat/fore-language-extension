# IMath.Sqrt


## Синтаксис


Sqrt(Value: Double): Double;


## Параметры


Value.
 Вещественное число.


## Описание


Метод Sqrt возвращает квадратный
 корень из заданного числа.


## Комментарии


Значение параметра Value должно
 быть неотрицательным.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.Sqrt(16);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен квадратный
 корень, равный «4».
