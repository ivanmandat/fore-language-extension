# IMath.ACot


## Синтаксис


ACot(Value: Double): Double;


## Параметры


Value. Котангенс угла.


## Описание


Метод ACot возвращает арккотангенс
 числа.


## Комментарии


Значение параметра не может быть пустым.


Для получения котангенса угла используйте метод [IMath.Cot](IMath.Cot.htm).


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.ACot(2);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен арккотангенс
 заданного угла, равный «0.463647609000806».
