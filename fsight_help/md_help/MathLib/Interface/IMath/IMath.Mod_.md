# IMath.Mod_


## Синтаксис


Mod_(Value: Double; Divisor: Double): Double;


## Параметры


Value. Делимое;


Divisor. Делитель.


## Описание


Метод Mod_ возвращает остаток
 от деления двух десятичных чисел.


## Комментарии


Значения входных параметров не могут быть пустыми.


Значение параметра Divisor
 не может быть равно «0».


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.Mod_(83, 2.6);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен остаток
 от деления, равный «2.4».
