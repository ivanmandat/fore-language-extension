# IMath.ASin


## Синтаксис


ASin(Value: Double): Double;


## Параметры


Value. Синус угла.


## Описание


Метод ASin возвращает арксинус
 числа.


## Комментарии


Значение параметра Value не
 может быть пустым и должно принадлежать промежутку [-1;1].


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.ASin(-0.5);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен арксинус
 заданного угла, равный «-0.523598775598299».
