# IMath.Degrees


## Синтаксис


Degrees(Angle: Double): Double;


## Параметры


Angle.
 Угол в радианах, который необходимо преобразовать в градусы.


## Описание


Метод Degrees возвращает значение,
 преобразованное из радиан в градусы.


## Комментарии


Параметр Angle не может быть
 пустым.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.Degrees(3.14);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен результат
 преобразования радиан в градусы, равный «179.908747671079».
