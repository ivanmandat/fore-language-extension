# IMath.Max


## Синтаксис


Max(Values: Array): Double;


## Параметры


Values. Массив вещественных
 чисел.


## Описание


Метод Max возвращает наибольшее
 значение из заданного набора значений.


## Комментарии


Функция корректно работает как с одномерными, так и с двумерными массивами.


Для получения наименьшего значения из заданного набора используйте метод
 [IMath.Min](IMath.Min.htm).


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var r: Double;

    arr: array[5] Of Double;

Begin

    Arr[0] := 10.8;

    Arr[1] := 7.19;

    Arr[2] := 9.56;

    Arr[3] := 27.96;

    Arr[4] := 2;

    r:= math.Max(arr);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведено наибольшее
 значение, равное «27.96».
