# IMath.Even


## Синтаксис


Even(Value: Double): Integer;


## Параметры


Value.
 Округляемое значение.


## Описание


Метод Even округляет указанное
 значение до ближайшего четного целого.


## Комментарии


Если значение Value положительное,
 то округление выполняется в большую сторону, если отрицательное - в меньшую.


Диапазон допустимых значений входного параметра: [MinInteger; MaxInteger].
 MinInteger = -231,
 MaxInteger = 231-1.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Integer;

Begin

    r := Math.Even(1.5);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен результат
 округления до ближайшего четного целого, равный «2».
