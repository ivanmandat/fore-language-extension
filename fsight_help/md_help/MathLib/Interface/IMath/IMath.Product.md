# IMath.Product


## Синтаксис


Product(Values: Array): Double;


## Параметры


Values.
 Массив вещественных чисел.


## Описание


Метод Product возвращает произведение
 всех чисел заданного массива.


## Комментарии


Если массив вещественных чисел содержит пропуски, то функция вернет
 ошибку.


Функция корректно работает как с одномерными, так и с двумерными массивами.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

    Arr: Array[3] Of Double;

Begin

    Arr[0] := 5;

    Arr[1] := 15;

    Arr[2] := 30;

    r := Math.Product(Arr);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен результат
 произведения, равный «2250».
