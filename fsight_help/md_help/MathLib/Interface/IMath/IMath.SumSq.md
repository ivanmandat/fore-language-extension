# IMath.SumSq


## Синтаксис


SumSq(Values: Array): Double;


## Параметры


Values.
 Массив вещественных чисел.


## Описание


Метод SumSq возвращает сумму
 квадратов аргументов.


## Комментарии


Если массив вещественных чисел содержит пропуски, то функция вернет
 ошибку.


Функция корректно работает как с одномерными, так и с двумерными массивами.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

    Arr: Array[4] Of Double;

Begin

    Arr[0] := -5;

    Arr[1] := 15.08;

    Arr[2] := 30.12;

    Arr[3] := 5.94;

    r := Math.SumSq(Arr);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведена сумма
 квадратов элементов массива, равная «1194.9044».
