# IMath.SumSqD


## Синтаксис


SumSqD(Values: Array): Decimal;


## Параметры


Values. Массив десятичных чисел
 большой точности.


## Описание


Метод SumSqD возвращает сумму
 квадратов десятичных чисел большой точности.


## Комментарии


Необходимо использовать массив значений типа [Variant](fore.chm::/03_datatypes/Fore_DataTypes.htm).


Функция корректно работает как с одномерными, так и с двумерными массивами.


Для получения суммы элементов массива десятичных чисел большой
 точности используйте метод [IMath.SumD](IMath.SumD.htm).


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


					Sub UserProc;

		Var

		    r: Decimal;

		    Arr: Array[4] Of Variant;

		Begin

		    Arr[0] := -5m;

		    Arr[1] := 15.08m;

		    Arr[2] := 30.12m;

		    Arr[3] := 5.94m;

		    r := Math.SumSqD(Arr);

		    Debug.WriteLine(r);

		End Sub UserProc;


В результате выполнения примера в окно консоли будет выведена сумма
 квадратов элементов массива, равная 1194.9044.
