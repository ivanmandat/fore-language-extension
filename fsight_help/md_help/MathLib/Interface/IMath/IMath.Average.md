# IMath.Average


## Синтаксис


Average(Values: Array): Double;


## Параметры


Values. Массив вещественных
 чисел.


## Описание


Метод Average возвращает среднее
 арифметическое аргументов.


## Комментарии


Если массив вещественных чисел содержит пропуски, то функция вернет
 ошибку.


Функция корректно работает как с одномерными, так и с двумерными массивами.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

    Arr: Array[5] Of Double;

Begin

    Arr[0]:=10.6;

    Arr[1]:=7;

    Arr[2] := 9.1;

    Arr[3]:=27.25;

    Arr[4]:=2;

    r := Math.Average(Arr);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведено среднее
 арифметическое аргументов, равное «11.19».
