# IMath.Pi


## Синтаксис


Pi: Double;


## Описание


Метод Pi возвращает математическую
 константу «Пи» (3,14159265358979) с точностью до 14-ого знака после запятой.


## Комментарии


Для получения числа Эйлера, возведенного в указанную степень, используйте
 метод [IMath.Exp](IMath.Exp.htm).


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.Pi;

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведено число
 «Пи», равное «3,14159265358979».
