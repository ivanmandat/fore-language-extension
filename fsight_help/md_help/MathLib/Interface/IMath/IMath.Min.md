# IMath.Min


## Синтаксис


Min(Values: Array): Double;


## Параметры


Values. Массив вещественных
 чисел.


## Описание


Метод Min возвращает наименьшее
 значение из заданного набора значений.


## Комментарии


Функция корректно работает как с одномерными, так и с двумерными массивами.


Для получения наибольшего значения из заданного набора используйте метод
 [IMath.Max](IMath.Max.htm).


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


Sub UserProc;

Var r:Double;

    arr:array[5] Of Double;

Begin

    arr[0]:=10.8;

    arr[1] := 7.19;

    arr[2]:=9.56;

    arr[3]:=27.96;

    arr[4]:=2;

    r:= math.Min(arr);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведено наименьшее
 значение, равное «2».
