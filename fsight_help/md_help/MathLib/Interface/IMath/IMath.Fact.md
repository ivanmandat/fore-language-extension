# IMath.Fact


## Синтаксис


Fact(Value: Integer): Double;


## Параметры


Value.
 Число, для которого вычисляется факториал.


## Описание


Метод Fact возвращает факториал
 числа.


## Комментарии


Значение параметра Value должно
 принадлежать промежутку [0, 170].


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.Fact(5);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен факториал
 числа, равный «120».
