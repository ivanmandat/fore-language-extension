# IMath.ATan


## Синтаксис


ATan(Value: Double): Double;


## Параметры


Value. Тангенс угла.


## Описание


Метод ATan возвращает арктангенс
 числа.


## Комментарии


Значение параметра не может быть пустым.


Для получения тангенса угла используйте метод [IMath.Tan](IMath.Tan.htm).


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var r: Double;

Begin

    r := math.ATan(1);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен арктангенс
 заданного угла, равный «0.785398163397448».
