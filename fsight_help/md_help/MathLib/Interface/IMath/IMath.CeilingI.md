# IMath.CeilingI


## Синтаксис


CeilingI(Value: Integer; Precision: Integer): Integer;


## Параметры


Value. Округляемое
 целое число;


Precision . Целое кратное,
 до которого требуется округлить значение.


## Описание


Метод CeilingI возвращает результат
 округления с избытком до ближайшего целого числа, кратного значению точности.


## Комментарии


Если число и точность положительны, значение округляется вверх.


Если число и точность отрицательны, значение округляется вниз.


Если число отрицательно, а точность положительна, значение округляется
 вверх.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.

Sub UserProc;

Var

    r1,r2,r3: Integer;

Begin

    r1 := Math.CeilingI(7,2);

    Debug.WriteLine(r1);

    r2 := Math.CeilingI(-8,-3);

    Debug.WriteLine(r2);

    r3 := Math.CeilingI(-11,6);

    Debug.WriteLine(r3);

End Sub UserProc;


В результате выполнения примера в окно консоли будут выведены результаты:


	- Число и точность положительны: «8»;


	- Число и точность отрицательны: «-9»;


	- Число отрицательно, а точность положительна: «-6».
