# IMath.Sin


## Синтаксис


Sin(Value: Double): Double;


## Параметры


Value.
 Угол в радианах, синус которого необходимо найти.


## Описание


Метод Sin возвращает синус заданного
 угла.


## Комментарии


Значение параметра Value не
 может быть пустым и должно быть задано конечным числом.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.Sin(Math.Pi/2);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен синус заданного
 угла, равный «1».
