# IMath.FloorI


## Синтаксис


FloorI(Value: Integer; Precision: Integer): Integer;


## Параметры


Value. Округляемое
 целое число;


Precision. Целое кратное,
 до которого требуется округлить значение.


## Описание


Метод FloorI возвращает результат
 округления с недостатком до ближайшего целого числа, кратного значению
 точности.


## Комментарии


Если число и точность положительны, значение округляется вниз, если
 отрицательны - вверх. Если число отрицательно, а точность положительна,
 значение округляется вниз.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var r1,r2,r3:Integer;

Begin

    r1 := math.FloorI(7, 2);

    Debug.WriteLine(r1);

    r2:= math.FloorI(-8,-3);

    Debug.WriteLine(r2);

    r3:= math.FloorI(-11,6);

    Debug.WriteLine(r3);

End Sub UserProc;


В результате выполнения примера в окно консоли будут выведены результаты
 округления:


	- Число и точность положительны: «6»;


	- Число и точность отрицательны: «-6»;


	- Число отрицательно, а точность положительна: «-12».
