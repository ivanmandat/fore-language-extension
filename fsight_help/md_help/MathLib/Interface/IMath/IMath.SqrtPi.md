# IMath.SqrtPi


## Синтаксис


SqrtPi(Value: Double): Double;


## Параметры


Value.
 Вещественное число.


## Описание


Метод SqrtPi возвращает квадратный
 корень из значения выражения «Value * Пи».


## Комментарии


Значение параметра Value должно
 быть неотрицательным.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.SqrtPi(2);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен квадратный
 корень, равный «2.506628274631».
