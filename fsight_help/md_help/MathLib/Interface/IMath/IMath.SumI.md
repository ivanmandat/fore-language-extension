# IMath.SumI


## Синтаксис


SumI(Values: Array): Integer;


## Параметры


Values.
 Массив целых чисел.


## Описание


Метод SumI возвращает сумму
 элементов (целых чисел) указанного массива.


## Комментарии


Необходимо использовать массив значений типа [Integer](Fore.chm::/03_dataTypes/Fore_DataTypes.htm).


Функция корректно работает как с одномерными, так и с двумерными массивами.


Для получения суммы вещественных чисел используйте метод [IMath.Sum](IMath.Sum.htm).


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


					Sub UserProc;

		Var

		    r: Integer;

		    Arr: Array[4] Of Integer;

		Begin

		    Arr[0] := -5;

		    Arr[1] := 158;

		    Arr[2] := 30;

		    Arr[3] := 5;

		    r := math.SumI(Arr);

		    Debug.WriteLine(r);

		End Sub UserProc;


В результате выполнения примера в окно консоли будет выведена сумма
 элементов массива, равная 188.
