# IMath.Combin


## Синтаксис


Combin(SetSize: Integer; SampleSize: Integer): Double;


## Параметры


SetSize.
 Общее количество элементов множества;


SampleSize. Набор элементов,
 выбранных из общего количества элементов множества.


## Описание


Метод Combin возвращает количество
 комбинаций для заданного числа объектов.


## Комментарии


Параметры должны быть целыми и неотрицательными числами.


Значение параметра SampleSize
 не должно превышать значения параметра SetSize.


Комбинацией считается любое множество или подмножество объектов, безотносительно
 к их порядку. Комбинации отличаются от перестановок, для которых порядок
 существен.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.Combin(8,2);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведено количество
 комбинаций, равное «28».
