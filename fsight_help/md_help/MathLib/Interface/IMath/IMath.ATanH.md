# IMath.ATanH


## Синтаксис


ATanH(Value: Double): Double;


## Параметры


Value.
 Гиперболический тангенс угла.


## Описание


Метод ATanH возвращает гиперболический
 арктангенс заданного числа.


## Комментарии


Значение параметра Value не может быть пустым и должно принадлежать
 промежутку (-1;1).


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


Sub UserProc;

Var

    r: Double;

Begin

    r := Math.ATanH(0.76);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен гиперболический
 арктангенс, равный «0.996215082345103».
