# IMath.Sign


## Синтаксис


Sign(Value: Double): Integer;


## Параметры


Value.
 Вещественное число.


## Описание


Метод Sign возвращает знак вещественного
 числа.


## Комментарии


Значение параметра не может быть пустым.


Может возвращать следующие значения:


	- «1» - если число положительное;


	- «0» - если число равно 0;


	- «-1» - если число отрицательное.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.Sign(10);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен знак заданного
 числа «1».
