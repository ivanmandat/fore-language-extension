# IMath.SumD


## Синтаксис


SumD(Values: Array): Decimal;


## Параметры


Values. Массив десятичных чисел
 высокой точности.


## Описание


Метод SumD возвращает сумму
 десятичных чисел высокой точности указанного массива.


## Комментарии


Необходимо использовать массив значений типа [Variant](fore.chm::/03_datatypes/Fore_DataTypes.htm).


Функция корректно работает как с одномерными, так и с двумерными массивами.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


					Sub UserProc;

		Var

		    r: Decimal;

		    Arr: Array[4] Of Variant;

		Begin

		    Arr[0] := -5m;

		    Arr[1] := 15.1524m;

		    Arr[2] := 30.65m;

		    Arr[3] := 5.12m;

		    r := math.SumD(Arr);

		    Debug.WriteLine(r);

		End Sub UserProc;


В результате выполнения примера в окно консоли будет выведена сумма
 элементов массива, равная 45.9224.
