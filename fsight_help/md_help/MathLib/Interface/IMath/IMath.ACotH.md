# IMath.ACotH


## Синтаксис


ACotH(Value: Double): Double;


## Параметры


Value. Гиперболический котангенс
 угла.


## Описание


Метод ACotH возвращает гиперболический
 арккотангенс заданного числа.


## Комментарии


Значение параметра не может быть пустым и должно принадлежать промежутку
 (-inf, -1] U [1, inf).


Для получения арккотангенса угла используйте метод [IMath.ACot](IMath.ACot.htm).


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.ACotH(6);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен гиперболический
 арккотангенс, равный «0.168236118310606».
