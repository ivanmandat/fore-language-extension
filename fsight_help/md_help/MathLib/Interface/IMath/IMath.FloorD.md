# IMath.FloorD


## Синтаксис


FloorD(Value: Decimal; Precision: Decimal): Decimal;


## Параметры


Value. Округляемое число высокой
 точности;


Precision. Кратное высокой
 точности, до которого требуется округлить значение.


## Описание


Метод FloorD возвращает результат
 округления с недостатком до ближайшего числа высокой точности, кратного
 значению точности.


## Комментарии


Для обозначения в массиве чисел типа [Decimal](fore.chm::/03_datatypes/Fore_DataTypes.htm)
 после значения каждого числа необходимо дописать символ «m». Например:
 0.001m.


Если число и точность положительны, значение округляется вниз, если
 отрицательны - вверх. Если число отрицательно, а точность положительна,
 значение округляется вниз.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


					Sub UserProc;

		Var r1,r2,r3:Decimal;

		Begin

		    r1:= math.FloorD(3.726541233655125542365854m,2m);

		    Debug.WriteLine(r1);

		    r2:= math.FloorD(-2.5m,-2.365481235841654m);

		    Debug.WriteLine(r2);

		    r3:= math.FloorD(-2.55634576215697545m,2.54m);

		    Debug.WriteLine(r3);

		End Sub UserProc;


В результате выполнения примера в окно консоли будут выведены результаты
 округления:


	- число и точность положительны: 2;


	- число и точность отрицательны: -2.365481235841654;


	- число отрицательно, а точность положительна: -5.08.
