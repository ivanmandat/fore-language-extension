# IMath.Abs


## Синтаксис


Abs(Value: Double): Double;


## Параметры


Value. Вещественное число,
 абсолютное значение которого необходимо найти.


## Описание


Метод Abs возвращает
 абсолютную величину (модуль) числа.


## Комментарии


Значение параметра не может быть пустым.


Для получения абсолютного значения десятичного числа большой точности
 используйте метод [IMath.AbsD](IMath.AbsD.htm).


Для получения абсолютного значения целого числа используйте метод [IMath.AbsI](IMath.AbsI.htm).


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


  			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.Abs(-4.645);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведено абсолютное
 значение заданной величины «4.645».
