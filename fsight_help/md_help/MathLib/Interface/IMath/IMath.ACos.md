# IMath.ACos


## Синтаксис


ACos(Value: Double): Double;


## Параметры


Value. Косинус угла.


## Описание


Метод ACos возвращает арккосинус
 числа.


## Комментарии


Значение параметра Value не
 может быть пустым и должно принадлежать промежутку [-1;1].


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.ACos(-0.5);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен арккосинус
 заданного числа, равный «2.0943951023932».
