# IMath.Power


## Синтаксис


Power(Value: Double; PowerValue: Double): Double;


## Параметры


Value. Вещественное число,
 возводимое в степень;


PowerValue. Показатель степени.


## Описание


Метод Power возвращает результат
 возведения вещественного числа в степень.


## Комментарии


Значения входных параметров не могут быть пустыми.


Если параметр Value задан отрицательным
 числом, то параметр PowerValue
 должен быть целым, иначе функция вернет ошибку.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.Power(4,1.25);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен результат
 возведения в степень, равный «5.65685424949238».
