# IMath.Tan


## Синтаксис


Tan(Value: Double): Double;


## Параметры


Value.
 Угол в радианах, тангенс которого необходимо найти.


## Описание


Метод Tan возвращает тангенс
 заданного угла.


## Комментарии


Значение параметра Value должно
 быть задано конечным числом.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.Tan(0.785);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен тангенс
 заданного угла, равный «0.999203990105043».
