# IMath.ATan2


## Синтаксис


ATan2(X: Double; Y: Double): Double;


## Параметры


X. Координата точки;


Y. Координата точки.


## Описание


Метод ATan2 возвращает арктангенс
 для заданных координат X и Y.


## Комментарии


Значения параметров не могут быть пустыми.


Оба аргумента X и Y
 одновременно не могут быть равны «0».


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


Sub UserProc;

Var

    r: Double;

Begin

    r := Math.ATan2(1, 1);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен арктангенс
 для заданных координат X и Y, равный «0.785398163397448».
