# IMath.Floor


## Синтаксис


Floor(Value: Double; Precision: Double): Double;


## Параметры


Value. Округляемое вещественное
 число;


Precision. Кратное, до которого
 требуется округлить значение.


## Описание


Метод Floor возвращает результат
 округления с недостатком до ближайшего числа, кратного значению точности.


## Комментарии


Значения параметров не могут быть пустыми.


Если число и точность положительны, значение округляется вниз, если
 отрицательны - вверх. Если число отрицательно, а точность положительна,
 значение округляется вниз.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r1, r2, r3: Double;

Begin

    r1 := Math.Floor(3.7,2);

    Debug.WriteLine(r1);

    r2 := Math.Floor(-2.5,-2);

    Debug.WriteLine(r2);

    r3 := Math.Floor(-2.5,2);

    Debug.WriteLine(r3);

End Sub UserProc;


В результате выполнения примера в окно консоли будут выведены результаты
 округления:


	- Число и точность положительны: «2»;


	- Число и точность отрицательны: «-2»;


	- Число отрицательно, а точность положительна: «-4».
