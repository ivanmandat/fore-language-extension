# IMath.CosH


## Синтаксис


CosH(Value: Double): Double;


## Параметры


Value.
 Вещественное число, гиперболический косинус которого необходимо
 найти.


## Описание


Метод CosH возвращает гиперболический
 косинус заданного числа.


## Комментарии


Значение параметра Value должно
 быть задано конечным числом.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.CosH(4);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен гиперболический
 косинус, равный «27.3082328360165».
