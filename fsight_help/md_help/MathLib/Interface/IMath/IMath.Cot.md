# IMath.Cot


## Синтаксис


Cot(Value: Double): Double;


## Параметры


Value. Угол в радианах, котангенс
 которого необходимо найти.


## Описание


Метод Cot возвращает котангенс
 заданного угла.


## Комментарии


Значение параметра Value не
 может быть пустым или равным «0».


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.Cot(30);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен котангенс
 заданного угла, равный «-0.156119952161659».
