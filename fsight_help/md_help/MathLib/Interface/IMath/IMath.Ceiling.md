# IMath.Ceiling


## Синтаксис


Ceiling(Value: Double; Precision: Double): Double;


## Параметры


Value. Округляемое
 вещественное число;


Precision. Кратное,
 до которого требуется округлить значение.


## Описание


Метод Ceiling возвращает результат
 округления с избытком до ближайшего числа, кратного значению точности.


## Комментарии


Значения параметров не могут быть пустыми.


Если число и точность положительны, значение округляется вверх.


Если число и точность отрицательны, значение округляется вниз.


Если число отрицательно, а точность положительна, значение округляется
 вверх.


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r1, r2, r3: Double;

Begin

    r1 := Math.Ceiling(1.5,0.1);

    Debug.WriteLine(r1);

    r2 := Math.Ceiling(-2.5,-2);

    Debug.WriteLine(r2);

    r3 := Math.Ceiling(-2.5,2);

    Debug.WriteLine(r3);

End Sub UserProc;


В результате выполнения примера в окно консоли будут выведены результаты:


	- Число и точность положительны: «1.5»;


	- Число и точность отрицательны: «-4»;


	- Число отрицательно, а точность положительна: «-2».
