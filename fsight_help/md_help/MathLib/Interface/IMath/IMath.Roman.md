# IMath.Roman


## Синтаксис


Roman(Value: Integer; Form: Integer): String;


## Параметры


Value. Целое число, которое
 необходимо преобразовать;


Form. Число, задающее нужную
 форму записи римского числа.


## Описание


Метод Roman преобразует арабское
 число в римское в текстовом формате.


## Комментарии


Значение параметра Value должно
 принадлежать промежутку [1, 3999].


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: String;

Begin

    r := Math.Roman(9,1);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен результат
 преобразования.
