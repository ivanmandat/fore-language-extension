# IMath.Exp


## Синтаксис


Exp(Value: Double): Double;


## Параметры


Value.
 Показатель степени, в которую возводится число Эйлера.


## Описание


Метод Exp возвращает число Эйлера,
 возведенное в указанную степень.


## Комментарии


Значение параметра не может быть пустым.


Для получения натурального логарифма используйте метод [IMath.Ln](IMath.Ln.htm).


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.Exp(2);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен результат
 возведения числа Эйлера в указанную степень, равный «7.38905609893065».
