# IMath.AbsI


## Синтаксис


AbsI(Value: Integer): Integer;


## Параметры


Value. Целое число, абсолютное
 значение которого необходимо найти.


## Описание


Метод AbsI возвращает абсолютную
 величину (модуль) от целого числа.


## Комментарии


Для получения абсолютного значения десятичного числа большой точности
 используйте метод [IMath.AbsD](IMath.AbsD.htm).


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Decimal;

Begin

    r := Math.AbsI(-4);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведено абсолютное
 значение заданной величины «4».
