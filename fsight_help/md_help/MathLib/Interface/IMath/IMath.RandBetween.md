# IMath.RandBetween


## Синтаксис


RandBetween(Bottom: Double; Top: Double): Double;


## Параметры


Bottom. Нижняя граница диапазона;


Top. Верхняя граница диапазона.


## Описание


Метод RandBetween возвращает
 случайное число, находящееся в диапазоне между двумя заданными значениями.


## Комментарии


Значения входных параметров не могут быть пустыми.


Для получения равномерно распределенного случайного вещественного числа
 из промежутка [0, 1] используйте метод [IMath.Rand](IMath.Rand.htm).


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.RandBetween(0,100);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведено случайное
 число.
