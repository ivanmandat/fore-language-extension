# IMath.MinI


## Синтаксис


MinI(Values: Array): Integer;


## Параметры


Values. Массив целых чисел.


## Описание


Метод MinI возвращает наименьшее
 целое значение из заданного набора значений.


## Комментарии


Необходимо использовать массив значений типа [Integer](Fore.chm::/03_dataTypes/Fore_DataTypes.htm).


Функция корректно работает как с одномерными, так и с двумерными массивами.


Для получения наибольшего значения из заданного набора используйте метод
 [IMath.Max](IMath.Max.htm).


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


					Sub UserProc;

		Var

		    r: Integer;

		    Arr: Array[5] Of Integer;

		Begin

		    Arr[0] := 10;

		    Arr[1] := 7;

		    Arr[2] := 9;

		    Arr[3] := 27;

		    Arr[4] := 2;

		    r := Math.MinI(Arr);

		    Debug.WriteLine(r);

		End Sub UserProc;


В результате выполнения примера в окно консоли будет выведено наименьшее
 значение, равное 2.
