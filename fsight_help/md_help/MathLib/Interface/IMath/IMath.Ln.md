# IMath.Ln


## Синтаксис


Ln(Value: Double): Double;


## Параметры


Value.
 Положительное вещественное число.


## Описание


Метод Ln возвращает натуральный
 логарифм числа.


## Комментарии


Для получения десятичного логарифма используйте метод [IMath.Log10](IMath.Log10.htm).


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


Sub UserProc;

Var r: Double;

Begin

    r := Math.Ln(86);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен результат
 вычисления натурального логарифма, равный «4.45434729625351».
