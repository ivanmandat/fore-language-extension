# IMath.ModD


## Синтаксис


ModD(Value: Decimal; Divisor: Decimal): Decimal;


## Параметры


Value. Делимое;


Divisor. Делитель.


## Описание


Метод ModD возвращает остаток
 от деления двух десятичных чисел высокой точности.


## Комментарии


Для обозначения в массиве чисел типа [Decimal](fore.chm::/03_datatypes/Fore_DataTypes.htm)
 после значения каждого числа необходимо дописать символ «m». Например:
 0.001m.


Значение параметра Divisor
 не может быть равно «0».


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


					Sub UserProc;

		Var

		    r: Decimal;

		Begin

		    r := Math.ModD(83m, 2.61642151234516451348521m);

		    Debug.WriteLine(r);

		End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен остаток
 от деления, равный 1.89093311729990008195849.
