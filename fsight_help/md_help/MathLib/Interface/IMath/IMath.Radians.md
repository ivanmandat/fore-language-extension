# IMath.Radians


## Синтаксис


Radians(Angle: Double): Double;


## Параметры


Angle.
 Величина угла в градусах.


## Описание


Метод Radians возвращает значение,
 преобразованное из градусов в радианы.


## Комментарии


Значение параметра не может быть пустым.


Для преобразования радианов в градусы используйте метод [IMath.Degrees](IMath.Degrees.htm).


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.Radians(270);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен результат
 преобразования, равный «4.71238898038469».
