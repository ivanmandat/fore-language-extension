# IMath.Log10


## Синтаксис


Log10(Value: Double): Double;


## Параметры


Value. Положительное вещественное
 число.


## Описание


Метод Log10 возвращает десятичный
 логарифм числа.


## Комментарии


Значение параметра не может быть пустым.


Для получения натурального логарифма используйте метод [IMath.Ln](IMath.Ln.htm).


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.Log10(86);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен результат
 вычисления десятичного логарифма, равный «1.93449845124357».
