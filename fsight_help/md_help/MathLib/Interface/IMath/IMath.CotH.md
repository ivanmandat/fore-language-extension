# IMath.CotH


## Синтаксис


CotH(Value: Double): Double;


## Параметры


Value. Вещественное число,
 гиперболический котангенс которого необходимо найти.


## Описание


Метод CotH возвращает гиперболический
 котангенс заданного числа.


## Комментарии


Значение параметра Value не
 может быть пустым или равным «0».


## Пример


Для выполнения примера добавьте ссылку на системную сборку MathFin.


			Sub UserProc;

Var

    r: Double;

Begin

    r := Math.CotH(2);

    Debug.WriteLine(r);

End Sub UserProc;


В результате выполнения примера в окно консоли будет выведен гиперболический
 котангенс, равный «1.03731472072755».
