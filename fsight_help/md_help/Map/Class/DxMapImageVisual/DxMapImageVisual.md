# DxMapImageVisual


## Описание


Класс DxMapImageVisual содержит свойства и методы для работы с пиктографическим показателем на слое карты.

		 параметры высоты пиктографического показателя.


		 ![](../../Property_Image.gif)
		 [Image](../../Interface/IMapImageVisual/IMapImageVisual.Image.htm)
		 Свойство Image возвращает
		 параметры пиктограммы показателя.


		 ![](../../Property_Image.gif)
		 [StartAngle](../../Interface/IMapImageVisual/IMapImageVisual.StartAngle.htm)

		 [Width](../../Interface/IMapImageVisual/IMapImageVisual.Width.htm)
