# DxMapIconScale


## Описание


Класс DxMapIconScale содержит свойства и методы для работы с пиктографической шкалой.


## Свойства объекта класса, унаследованные от [IMapIconScale](../../Interface/IMapIconScale/IMapIconScale.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Exact](../../Interface/IMapIconScale/IMapIconScale.Exact.htm)
		 Свойство Exact определяет
		 пиктограмму для значений, которые равны значению деления шкалы.


		 ![](../../Property_Image.gif)
		 [Greater](../../Interface/IMapIconScale/IMapIconScale.Greater.htm)
		 Свойство Greater определяет
		 пиктограмму для значений, которые больше значения деления шкалы.


		 ![](../../Property_Image.gif)
		 [Icon](../../Interface/IMapIconScale/IMapIconScale.Icon.htm)
		 Свойство Icon определяет
		 пиктограмму для заданного значения деления шкалы.


		 ![](../../Property_Image.gif)
		 [Less](../../Interface/IMapIconScale/IMapIconScale.Less.htm)
		 Свойство Less определяет
		 пиктограмму для значений, которые меньше значения деления шкалы.


		 ![](../../Property_Image.gif)
		 [NoData](../../Interface/IMapIconScale/IMapIconScale.NoData.htm)
		 Свойство NoData определяет
		 пиктограмму для отсутствующих значений.
