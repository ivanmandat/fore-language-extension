# DxMapBarVisual


## Описание


Класс DxMapBarVisual содержит свойства и методы для определения параметров столбикового показателя.

		 Свойство BarsCount
		 определяет максимальное количество столбиков показателя.


		 ![](../../Property_Image.gif)
		 [Color](../../Interface/IMapBarVisual/IMapBarVisual.Color.htm)
		 Свойство Color определяет

		 [Fill](../../Interface/IMapBarVisual/IMapBarVisual.Fill.htm)

		 ![](../../Property_Image.gif)
		 [FillColor](../../Interface/IMapBarVisual/IMapBarVisual.FillColor.htm)
		 Свойство FillColor
		 определяет цвет штриховки для столбиков показателя.


		 ![](../../Property_Image.gif)
		 [Height](../../Interface/IMapBarVisual/IMapBarVisual.Height.htm)
		 Свойство Height определяет

		 ![](../../Property_Image.gif)
		 [Percentage](../../Interface/IMapBarVisual/IMapBarVisual.Percentage.htm)

		 ![](../../Property_Image.gif)
		 [Width](../../Interface/IMapBarVisual/IMapBarVisual.Width.htm)
		 Свойство Width определяет

		 ![](../../Property_Image.gif)
		 [StartAngle](../../Interface/IMapBarVisual/IMapBarVisual.StartAngle.htm)
		 Свойство StartAngle
		 определяет начальный угол поворота.
