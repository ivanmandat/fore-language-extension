# Классы сборки Map


В рамках сборки Map существуют
 следующие классы:


		 Класс
		 Краткое описание


		 ![](../Class_Image.gif)
		 [DxMap](DxMap/DxMap.htm)
		 Класс DxMap содержит
		 свойства и методы для определения параметров карты.


		 ![](../Class_Image.gif)
		 [DxMapAreaVisual](DxMapAreaVisual/DxMapAreaVisual.htm)
		 Класс DxMapAreaVisual
		 содержит свойства и методы для работы с картографическим показателем
		 на слое карты.


		 ![](../Class_Image.gif)
		 [DxMapBarVisual](DxMapBarVisual/DxMapBarVisual.htm)
		 Класс DxMapBarVisual
		 содержит свойства и методы для определения параметров столбикового
		 показателя.


		 ![](../Class_Image.gif)
		 [DxMapColorScale](DxMapColorScale/DxMapColorScale.htm)
		 Класс DxMapColorScale
		 определяет параметры цветовой шкалы для карты.


		 ![](../Class_Image.gif)
		 [DxMapFillScale](DxMapFillScale/DxMapFillScale.htm)
		 Класс DxMapFillScale
		 содержит свойства и методы для работы со штриховой шкалой.


		 ![](../Class_Image.gif)
		 [DxMapIconScale](DxMapIconScale/DxMapIconScale.htm)
		 Класс DxMapIconScale
		 содержит свойства и методы для работы с пиктографической шкалой.


		 ![](../Class_Image.gif)
		 [DxMapImageVisual](DxMapImageVisual/DxMapImageVisual.htm)
		 Класс DxMapImageVisual
		 содержит свойства и методы для работы с пиктографическим показателем
		 на слое карты.


		 ![](../Class_Image.gif)
		 [DxMapLegend](DxMapLegend/DxMapLegend.htm)
		 Класс DxMapLegend содержит
		 свойства и методы для работы с легендой карты.


		 ![](../Class_Image.gif)
		 [DxMapMarkerVisual](DxMapMarkerVisual/DxMapMarkerVisual.htm)
		 Класс DxMapMarkerVisual
		 предназначен для работы с маркерами геолокации.


		 ![](../Class_Image.gif)
		 [DxMapNumericScale](DxMapNumericScale/DxMapNumericScale.htm)
		 Класс DxMapNumericScale
		 содержит свойства и методы для работы с числовой шкалой.


		 ![](../Class_Image.gif)
		 [DxMapPieVisual](DxMapPieVisual/DxMapPieVisual.htm)
		 Класс DxMapPieVisual
		 содержит свойства и методы для работы с круговым показателем.


		 ![](../Class_Image.gif)
		 [DxMapTerritoryInfo](DxMapTerritoryInfo/DxMapTerritoryInfo.htm)
		 Класс DxMapTerritoryInfo
		 содержит свойства и методы для работы с расшифровкой территорий.


		 ![](../Class_Image.gif)
		 [DxMapTopobase](DxMapTopobase/DxMapTopobase.htm)
		 Класс DxMapTopobase
		 содержит свойства и методы для работы с топоосновой.


		 ![](../Class_Image.gif)
		 [DxMapView](DxMapView/DxMapView.htm)
		 Класс DxMapView содержит
		 свойства и методы, определяющие параметры отображения карты и
		 объектов, расположенных на карте.


		 ![](../Class_Image.gif)
		 [MapBox](MapBox/MapBox.htm)
		 Класс MapBox реализует
		 компонент среды разработки [MapBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/MapBox.htm),
		 используемый для отображения карт.


		 ![](../Class_Image.gif)
		 [MapBoxTerritoryEventArgs](MapBoxTerritoryEventArgs/MapBoxTerritoryEventArgs.htm)
		 Класс MapBoxTerritoryEventArgs
		 реализует аргумент события, возникающего при нажатии по какому
		 либо региону карты.


		 ![](../Class_Image.gif)
		 [MapExporter](MapExporter/MapExporter.htm)
		 Класс MapExporter содержит
		 свойства и методы для экспорта карт.


		 ![](../Class_Image.gif)
		 [MapWidget](MapWidget/MapWidget.htm)
		 Класс MapWidget используется
		 для настройки компонента MapWidget.


		 ![](../Class_Image.gif)
		 [UiMap](UiMap/UiMap.htm)
		 Класс UiMap реализует
		 объект среды разработки [UiMap](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiMap.htm)
		 , являющийся источником данных для компонента [MapBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/MapBox.htm).
