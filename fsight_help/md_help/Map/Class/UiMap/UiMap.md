# UiMap


Класс UiMap реализует объект
 среды разработки [UiMap](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiMap.htm),
 являющийся источником данных для компонента [MapBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/MapBox.htm).


## Свойства объекта класса, унаследованные от [IUiMap](../../Interface/IUiMap/IUiMap.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Map](../../Interface/IUiMap/IUiMap.Map.htm)
		 Свойство Map определяет
		 карту, к которой необходимо получить доступ.
