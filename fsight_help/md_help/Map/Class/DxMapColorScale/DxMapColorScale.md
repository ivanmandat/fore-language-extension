# DxMapColorScale


## Описание


Класс DxMapColorScale определяет параметры цветовой шкалы для карты.


## Свойства объекта класса, унаследованные от [IMapColorScale](../../Interface/IMapColorScale/IMapColorScale.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Color](../../Interface/IMapColorScale/IMapColorScale.Color.htm)
		 Свойство Color определяет
		 цвет заливки относительно заданного значения деления шкалы.


		 ![](../../Property_Image.gif)
		 [Exact](../../Interface/IMapColorScale/IMapColorScale.Exact.htm)
		 Свойство Exact определяет
		 цвет заливки показателя, значения которого равны значению деления
		 шкалы.


		 ![](../../Property_Image.gif)
		 [Greater](../../Interface/IMapColorScale/IMapColorScale.Greater.htm)
		 Свойство Greater определяет
		 цвет заливки показателя, значения которого больше значения деления
		 шкалы.


		 ![](../../Property_Image.gif)
		 [Less](../../Interface/IMapColorScale/IMapColorScale.Less.htm)
		 Свойство Less определяет
		 цвет заливки показателя, значения которого меньше значения деления
		 шкалы.


		 ![](../../Property_Image.gif)
		 [NoData](../../Interface/IMapColorScale/IMapColorScale.NoData.htm)
		 Свойство NoData определяет
		 цвет заливки для отсутствующих значений.
