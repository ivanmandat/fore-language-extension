# MapHorizontalAlignment


## Описание


Перечисление MapHorizontalAlignment
 используется для горизонтального выравнивания элементов легенды.


Используется следующим свойством:


	- [IMapLegend.ElementsHorizontalAlign](../Interface/IMapLegend/IMapLegend.ElementsHorizontalAlign.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Left. Выравнивание
		 по левому краю.


		 1
		 Center. Выравнивание
		 по центру.


		 2
		 Right. Выравнивание
		 по правому краю.
