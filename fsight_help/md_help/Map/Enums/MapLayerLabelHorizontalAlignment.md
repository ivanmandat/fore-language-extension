# MapLayerLabelHorizontalAlignment


## Описание


Перечисление MapLayerLabelHorizontalAlignment
 используется для горизонтального позиционирования подписи слоя карты относительно
 позиции, заданной в топооснове.


Используется следующим свойством:


	- [IMapLayer.LabelHorizontalAlignment](../Interface/IMapLayer/IMapLayer.LabelHorizontalAlignment.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Left. Подпись слева.


		 1
		 Center. Подпись в центре.


		 2
		 Right. Подпись справа.


		 3
		 None. Подпись отсутствует.
