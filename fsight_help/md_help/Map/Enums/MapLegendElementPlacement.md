# MapLegendElementPlacement


## Описание


Перечисление MapLegendElementPlacement
 используется для определения способа размещения элементов легенды.


Используется следующим свойством:


	- [IMapLegend.ElementPlacement](../Interface/IMapLegend/IMapLegend.ElementPlacement.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Auto. Автоматическое
		 размещение.


		 1
		 CustomColumns. Размещение
		 в несколько столбцов.


		 2
		 CustomRows. Размещение
		 в несколько строк.
