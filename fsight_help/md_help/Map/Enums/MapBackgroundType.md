# MapBackgroundType


## Описание


Перечисление MapBackgroundType
 используется для определения типа заливки фона легенды.


Используется следующим свойством:


	- [IMapBackground.HatchStyle](../Interface/IMapBackground/IMapBackground.HatchStyle.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Без заливки.


		 1
		 Color. Сплошная заливка.


		 2
		 Gradient. Градиентная
		 заливка.


		 3
		 Picture. Рисунок.


		 4
		 Hatch. Штриховка.
