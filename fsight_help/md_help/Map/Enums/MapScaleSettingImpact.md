# MapScaleSettingImpact


## Описание


Перечисление MapScaleSettingImpact содержит варианты положения относительно заданного значения деления шкалы.


Используется следующими свойствами и методами:


-
[IMapColorScale.Color](../Interface/IMapColorScale/IMapColorScale.Color.htm)


-
[IMapFillScale.Fill](../Interface/IMapFillScale/IMapFillScale.Fill.htm)


-
[IMapIconScale.Icon](../Interface/IMapIconScale/IMapIconScale.Icon.htm)


-
[IMapNumericScale.Number](../Interface/IMapNumericScale/IMapNumericScale.Number.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Exact. Равно.


 1
 Less. Меньше.


 2
 Greater. Больше.
