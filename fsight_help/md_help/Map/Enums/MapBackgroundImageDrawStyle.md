# MapBackgroundImageDrawStyle


## Описание


Перечисление MapBackgroundImageDrawStyle
 используется для определения способа размещения изображения в качестве
 фона легенды.


Используется следующим свойством:


	- [IMapBackground.ImageDrawStyle](../Interface/IMapBackground/IMapBackground.ImageDrawStyle.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Normal. По размеру.


		 1
		 Center. По центру.


		 2
		 Stretch. Растянуть.


		 3
		 Tile. Замостить.
