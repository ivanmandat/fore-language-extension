# MapVisualType


## Описание


Перечисление MapVisualType содержит
 типы показателей.


Используется следующим свойством:


	- [IMapVisual.VisualType](../Interface/IMapVisual/IMapVisual.VisualType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Area. Картографический
		 показатель.


		 1
		 Bar. Столбиковый показатель.


		 2
		 Pie. Круговой показатель.


		 3
		 Image. Пиктографический
		 показатель.


		 4
		 3D. 3D-показатель.


		 5
		 Marker. Маркер геолокации.
