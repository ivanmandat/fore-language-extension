# MapXmlSaveMethod


## Описание


Перечисление MapXmlSaveMethod
 содержит метод сохранения данных карты в XML.


Используется следующими свойствами и методами:


	- [IMap.SaveToXml](../Interface/IMap/IMap.SaveToXml.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 All. Сохранять всё.


		 1
		 DataSource. Сохранять
		 только данные.


		 2
		 Visual. Сохранять только
		 оформление.
