# MapLegendMarkerType


## Описание


Перечисление MapLegendMarkerType содержит типы маркера в легенде карты.


Используется следующим свойством:


-
[IMapLegend.MarkerType](../Interface/IMapLegend/IMapLegend.MarkerType.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Rectangle. Квадрат.


 1
 Rhombus. Ромб.


 2
 TriangleUp. Треугольник вершиной вверх.


 3
 TriangleDown. Треугольник вершиной вниз.


 4
 Circle. Круг.
