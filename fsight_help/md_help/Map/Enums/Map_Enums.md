# Перечисления сборки Map


В рамках сборки Map существуют
 следующие перечисления:


		 Перечисление
		 Краткое описание


		 ![](../Enums_Image.gif)
		 [MapArrowShape](MapArrowShape.htm)
		 Перечисление MapArrowShape
		 содержит типы стрелок.


		 ![](../Enums_Image.gif)
		 [MapArrowTextDirection](MapArrowTextDirection.htm)
		 Перечисление MapArrowTextDirection
		 содержит типы ориентаций стрелки.


		 ![](../Enums_Image.gif)
		 [MapAutoScaleMode](MapAutoScaleMode.htm)
		 Перечисление MapAutoScaleMode
		 содержит типы расчета значений шкалы отчета.


		 ![](../Enums_Image.gif)
		 [MapBackgroundImageDrawStyle](MapBackgroundImageDrawStyle.htm)
		 Перечисление MapBackgroundImageDrawStyle
		 используется для определения способа размещения изображения в
		 качестве фона легенды.


		 ![](../Enums_Image.gif)
		 [MapBackgroundType](MapBackgroundType.htm)
		 Перечисление MapBackgroundType
		 используется для определения типа заливки фона легенды.


		 ![](../Enums_Image.gif)
		 [MapHorizontalAlignment](MapHorizontalAlignment.htm)
		 Перечисление MapHorizontalAlignment
		 используется для горизонтального выравнивания элементов легенды.


		 ![](../Enums_Image.gif)
		 [MapLayerLabelHorizontalAlignment](MapLayerLabelHorizontalAlignment.htm)
		 Перечисление MapLayerLabelHorizontalAlignment
		 используется для горизонтального позиционирования подписи слоя
		 карты относительно позиции, заданной в топооснове.


		 ![](../Enums_Image.gif)
		 [MapLayerLabelVerticalAlignment](MapLayerLabelVerticalAlignment.htm)
		 Перечисление MapLayerLabelVerticalAlignment
		 используется для вертикального позиционирования подписи слоя карты
		 относительно позиции, заданной в топооснове.


		 ![](../Enums_Image.gif)
		 [MapLegendElementPlacement](MapLegendElementPlacement.htm)
		 Перечисление MapLegendElementPlacement
		 используется для определения способа размещения элементов легенды.


		 ![](../Enums_Image.gif)
		 [MapLegendMarkerType](MapLegendMarkerType.htm)
		 Перечисление MapLegendMarkerType
		 содержит типы маркера в легенде карты.


		 ![](../Enums_Image.gif)
		 [MapScaleSettingImpact](MapScaleSettingImpact.htm)
		 Перечисление MapScaleSettingImpact
		 содержит варианты положения относительно заданного значения деления
		 шкалы.


		 ![](../Enums_Image.gif)
		 [MapTerritoryInfoSortType](MapTerritoryInfoSortType.htm)
		 Перечисление MapTerritoryInfoSortType
		 содержит варианты порядка сортировки элементов в расшифровке территорий.


		 ![](../Enums_Image.gif)
		 [MapViewSelectionMode](MapViewSelectionMode.htm)
		 Перечисление MapViewSelectionMode
		 содержит режимы выделения территорий карты.


		 ![](../Enums_Image.gif)
		 [MapViewWndMode](MapViewWndMode.htm)
		 Перечисление MapViewWndMode
		 содержит режимы интерактивности карты.


		 ![](../Enums_Image.gif)
		 [MapVisualType](MapVisualType.htm)
		 Перечисление MapVisualType
		 содержит типы показателей.


		 ![](../Enums_Image.gif)
		 [MapXmlSaveMethod](MapXmlSaveMethod.htm)
		 Перечисление MapXmlSaveMethod
		 содержит метод сохранения данных карты в XML.
