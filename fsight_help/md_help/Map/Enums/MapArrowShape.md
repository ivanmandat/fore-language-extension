# MapArrowShape


## Описание


Перечисление MapArrowShape содержит типы стрелок.


Используется следующим свойством:


-
[IMapArrow.Shape](../Interface/IMapArrow/IMapArrow.Shape.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Wire. Стрелка в виде простой линии.
![](../Interface/IMapArrow/Map%20arrow%20Type.gif)


 1
 Simple. Стрелка с заливкой наконечника.
![](../Interface/IMapArrow/Map%20arrow%20Type%202.gif)


 2
 Tailed. Фигурная стрелка.
![](../Interface/IMapArrow/Map%20arrow%20Type%203.gif)
