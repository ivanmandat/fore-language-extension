# MapAutoScaleMode


## Описание


Перечисление MapAutoScaleMode содержит типы расчета значений шкалы отчета.


Используется следующим свойством:


-
[IMapScale.AutoScaleMode](../Interface/IMapScale/IMapScale.AutoScaleMode.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Linear. Линейный.


 1
 Uniform. Равномерный.
