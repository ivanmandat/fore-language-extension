# MapArrowTextDirection


## Описание


Перечисление MapArrowTextDirection содержит типы ориентаций стрелки.


Используется следующим свойством:


-
[IMapArrow.TextDirection](../Interface/IMapArrow/IMapArrow.TextDirection.htm)

 Значение

 Rotated. Вдоль стрелки.
