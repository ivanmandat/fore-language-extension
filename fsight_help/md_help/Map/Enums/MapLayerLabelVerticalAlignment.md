# MapLayerLabelVerticalAlignment


## Описание


Перечисление MapLayerLabelVerticalAlignment
 используется для вертикального позиционирования подписи слоя карты относительно
 позиции, заданной в топооснове.


Используется следующим свойством:


	- [IMapLayer.LabelVerticalAlignment](../Interface/IMapLayer/IMapLayer.LabelVerticalAlignment.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Top. Подпись сверху.


		 1
		 Center. Подпись в центре.


		 2
		 Bottom. Подпись снизу.


		 3
		 None. Подпись отсутствует.
