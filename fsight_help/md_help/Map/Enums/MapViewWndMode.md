# MapViewWndMode


## Описание


Перечисление MapViewWndMode содержит режимы интерактивности карты.


Используется следующим свойством:


-
[IMapView.WndMode](../Interface/IMapView/IMapView.WndMode.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Nothing. Режим отсутствия навигации по карте.


 1
 Drag. Режим перетаскивания карты.


 2
 ZoomIn. Режим увеличения масштаба карты.


 3
 ZoomOut. Режим уменьшения масштаба карты.


 4
 Selection. Режим выбора региона.


 5
 Hilite. Режим подсветки регионов.


 6
 ZoomShape. Режим центрирования на регионе.


 7
 ObjectDrag. Режим перетаскивания внутренних объектов.
