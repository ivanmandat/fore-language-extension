# MapTerritoryInfoSortType


## Описание


Перечисление MapTerritoryInfoSortType содержит варианты порядка сортировки элементов в расшифровке территорий.


Используется следующим свойством:


-
[IMapTerritoryInfo.SortType](../Interface/IMapTerritoryInfo/IMapTerritoryInfo.SortType.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Nothing. Без сортировки.


 1
 Code. Сортировка по идентификатору.


 2
 Name. Сортировка по наименованию.
