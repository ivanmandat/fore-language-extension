# IMapLegend.NoDataElementFirst


## Синтаксис


NoDataElementFirst: Boolean;


## Описание


Свойство NoDataElementFirst
 определяет, отображать ли элемент «Нет данных» первым.


## Комментарии


Допустимые значения:


	- True. Отображать элемент
	 «Нет данных» первым;


	- False. Отображать элемент
	 «Нет данных» последним.
