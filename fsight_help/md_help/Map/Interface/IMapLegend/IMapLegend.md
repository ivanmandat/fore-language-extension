# IMapLegend


## Описание


Интерфейс IMapLegend содержит
 свойства и методы для работы с легендой карты.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AutoPadding](IMapLegend.AutoPadding.htm)


		 Свойство AutoPadding
		 определяет, используются ли в легенде автоматические внутренние
		 отступы.


		 ![](../../Property_Image.gif)
		 [Background](IMapLegend.Background.htm)


		 Свойство Background
		 возвращает параметры фона легенды.


		 ![](../../Property_Image.gif)
		 [BorderPen](IMapLegend.BorderPen.htm)


		 Свойство BorderPen

		 ![](../../Property_Image.gif)
		 [Color](IMapLegend.Color.htm)
