# IMapLegend.ColumnSpace


## Синтаксис


ColumnSpace: Double;


## Описание


Свойство ColumsSpace определяет
 внутренние отступы между столбцами легенды.


## Комментарии


Для использования автоматических внутренних отступов между столбцами
 легенды используйте [IMapLegend.ColumnSpaceAuto](IMapLegend.ColumnSpaceAuto.htm).
