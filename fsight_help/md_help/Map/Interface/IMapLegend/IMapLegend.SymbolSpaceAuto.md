# IMapLegend.SymbolSpaceAuto


## Синтаксис


SymbolSpaceAuto: Boolean;


## Описание


Свойство SymbolSpaceAuto определяет,
 используются ли автоматические внутренние отступы между значком и подписью
 элемента легенды.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Автоматические внутренние отступы между значком и подписью включены;


	- False. Автоматические
	 внутренние отступы между значком и подписью элемента отключены и задаются
	 вручную с помощью [IMapLegend.SymbolSpace](IMapLegend.SymbolSpace.htm).
