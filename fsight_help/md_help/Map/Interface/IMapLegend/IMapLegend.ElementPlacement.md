# IMapLegend.ElementPlacement


## Синтаксис


ElementPlacement: [MapLegendElementPlacement](../../Enums/MapLegendElementPlacement.htm);


## Описание


Свойство ElementPlacement
 определяет способ размещения элементов легенды.


## Комментарии


При выборе способа размещения элементов легенды в несколько столбцов
 используйте свойство [IMapLegend.ColumnCount](IMapLegend.ColumnCount.htm)
 для задания количества столбцов. При выборе способа размещения элементов
 легенды в несколько строк используйте [IMapLegend.RowCount](IMapLegend.RowCount.htm)
 для задания количества строк.
