# IMapLegend.RowSpaceAuto


## Синтаксис


RowSpaceAuto: Boolean;


## Описание


Свойство RowSpaceAuto
 определяет, используются ли автоматические внутренние отступы между строками
 легенды.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Автоматические внутренние отступы между строками легенды включены;


	- False. Автоматические
	 внутренние отступы между строками легенды отключены и задаются вручную
	 с помощью [IMapLegend.RowSpace](IMapLegend.RowSpace.htm).
