# IMapLegend.ColumnSpaceAuto


## Синтаксис


ColumnSpaceAuto: Boolean;


## Описание


Свойство ColumnSpaceAuto
 определяет, используются ли внутренние отступы между столбцами легенды.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Автоматические внутренние отступы между столбцами легенды включены;


	- False. Автоматические
	 внутренние отступы между столбцами легенды отключены и задаются вручную
	 с помощью [IMapLegend.ColumnSpace](IMapLegend.ColumnSpace.htm).
