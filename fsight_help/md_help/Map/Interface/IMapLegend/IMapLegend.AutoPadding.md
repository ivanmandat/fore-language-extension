# IMapLegend.AutoPadding


## Синтаксис


AutoPadding: Boolean;


## Описание


Свойство AutoPadding определяет,
 используются ли в легенде автоматические внутренние отступы.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Автоматические внутренние отступы включены;


	- False. Автоматические
	 внутренние отступы отключены и задаются вручную с помощью [IMapLegend.Padding](IMapLegend.Padding.htm).
