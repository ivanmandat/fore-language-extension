# IMapLegend.NoDataElementVisible


## Синтаксис


NoDataElementVisible: Boolean;


## Описание


Свойство NoDataElementVisible
 определяет, отображать ли элемент «Нет данных».


## Комментарии


Допустимые значения:


	- True. Отображать элемент
	 «Нет данных»;


	- False. Не отображать
	 элемент «Нет данных».
