# IMapLegend.ElementsHorizontalAlign


## Синтаксис


ElementsHorizontalAlign: [MapHorizontalAlignment](../../Enums/MapHorizontalAlignment.htm);


## Описание


Свойство ElementsHorizontalAlign
 определяет способ горизонтального выравнивания элементов легенды


## Комментарии


По умолчанию используется выравнивание по левому краю.
