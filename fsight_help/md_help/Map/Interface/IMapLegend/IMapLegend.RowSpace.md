# IMapLegend.RowSpace


## Синтаксис


RowSpace: Double;


## Описание


Свойство RowSpace определяет
 внутренние отступы между строками легенды.


## Комментарии


Для использования автоматических внутренних отступов между столбцами
 легенды используйте [IMapLegend.RowSpaceAuto](IMapLegend.RowSpaceAuto.htm).
