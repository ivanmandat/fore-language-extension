# IMapLegend.SymbolSpace


## Синтаксис


SymbolSpace: Double;


## Описание


Свойство SymbolSpace определяет
 внутренние отступы между значком и подписью элемента легенды.


## Комментарии


Для использования автоматических внутренних отступов между значком и
 подписью элемента легенды используйте [IMapLegend.SymbolSpaceAuto](IMapLegend.SymbolSpaceAuto.htm).
