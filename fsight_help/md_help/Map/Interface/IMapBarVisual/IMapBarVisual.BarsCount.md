# IMapBarVisual.BarsCount


## Синтаксис


BarsCount: Integer;


## Описание


Свойство BarsCount определяет
 максимальное количество столбиков показателя.


## Комментарии


По умолчанию значение свойства равно нулю (все столбики показателя будут
 отображены).
