# IMapBarVisual.LeastWidthPart


## Синтаксис


LeastWidthPart: Double;


## Описание


Свойство LeastWidthPart определяет
 долю минимальной ширины столбиков показателя.


## Комментарии


Допустимые значения ограничены диапазоном [0..1]. По умолчанию свойство
 имеет значение «0.2».
