# IMapBarVisual


## Описание


Интерфейс IMapBarVisual содержит
 свойства и методы для определения параметров столбикового показателя.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BarsCount](IMapBarVisual.BarsCount.htm)
		 Свойство BarsCount
		 определяет максимальное количество столбиков показателя.


		 ![](../../Property_Image.gif)
		 [Color](IMapBarVisual.Color.htm)
		 Свойство Color определяет
		 параметры заливки столбиков показателя.


		 ![](../../Property_Image.gif)
		 [Fill](IMapBarVisual.Fill.htm)
		 Свойство Fill определяет
		 параметры штриховки для столбиков показателя.


		 ![](../../Property_Image.gif)
		 [FillColor](IMapBarVisual.FillColor.htm)
		 Свойство FillColor
		 определяет цвет штриховки для столбиков показателя.


		 ![](../../Property_Image.gif)
		 [Height](IMapBarVisual.Height.htm)
		 Свойство Height определяет
		 высоту частей показателя.


		 ![](../../Property_Image.gif)
		 [InterBarDistance](IMapBarVisual.InterBarDistance.htm)
		 Свойство InterBarDistance
		 определяет расстояние между столбиками показателя.


		 ![](../../Property_Image.gif)
		 [LeastHeightPart](IMapBarVisual.LeastHeightPart.htm)
		 Свойство LeastHeightPart
		 определяет долю минимальной высоты столбиков показателя.


		 ![](../../Property_Image.gif)
		 [LeastWidthPart](IMapBarVisual.LeastWidthPart.htm)
		 Свойство LeastWidthPart
		 определяет долю минимальной ширины столбиков показателя.


		 ![](../../Property_Image.gif)
		 [Percentage](IMapBarVisual.Percentage.htm)
		 Свойство Percentage
		 определяет использование процентной шкалы для столбикового показателя
		 карты.


		 ![](../../Property_Image.gif)
		 [Width](IMapBarVisual.Width.htm)
		 Свойство Width определяет
		 ширину частей показателя.


		 ![](../../Property_Image.gif)
		 [StartAngle](IMapBarVisual.StartAngle.htm)
		 Свойство StartAngle
		 определяет начальный угол поворота.
