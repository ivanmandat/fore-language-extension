# IMapVisualNumericProperty.Dependent


## Синтаксис


Dependent: Boolean;


## Описание


Свойство Dependent определяет,
 используется ли шкала для параметра.


## Коментарии


Допустимые значения:


	- True. Используется
	 шкала, заданная свойством [IMapVisualNumericProperty.Scale](IMapVisualNumericProperty.Scale.htm),
	 значение свойства [IMapVisualNumericProperty.Value](IMapVisualNumericProperty.Value.htm)
	 игнорируется;


	- False. Используются
	 значение, заданное свойством [IMapVisualNumericProperty.Value](IMapVisualNumericProperty.Value.htm),
	 значение свойства [IMapVisualNumericProperty.Scale](IMapVisualNumericProperty.Scale.htm)
	 игнорируется.
