# IMapBackground.GradientColor


## Синтаксис


GradientColor: [IGxColor](ModDrawing.chm::/Interface/IGxColor/IGxColor.htm);


## Описание


Свойство GradientColor
 определяет цвет градиентной заливки фона легенды.


## Комментарии


Данное свойство актуально, если свойство [IMapBackground.Type](IMapBackground.Type.htm)
 использует тип заливки MapBackgroundType.Gradient.
