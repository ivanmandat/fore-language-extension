# IMapBackground


Сборка: Map;


## Описание


Интерфейс IMapBackground используется
 для настройки фона легенды карты.


## Иерархия наследования


           IMapBackground


## Комментарии


Для получения настроек фона легенды используйте [IMapLegend.Background](../IMapLegend/IMapLegend.Background.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Color](IMapBackground.Color.htm)
		 Свойство Color
		 определяет цвет сплошной заливки фона легенды.


		 ![](../../Property_Image.gif)
		 [GradientAngle](IMapBackground.GradientAngle.htm)
		 Свойство GradientAngle
		 определяет угол градиента заливки легенды.


		 ![](../../Property_Image.gif)
		 [GradientColor](IMapBackground.GradientColor.htm)
		 Свойство GradientColor
		 определяет цвет градиентной заливки фона легенды.


		 ![](../../Property_Image.gif)
		 [HatchStyle](IMapBackground.HatchStyle.htm)
		 Свойство HatchStyle
		 определяет тип штриховки фона легенды.


		 ![](../../Property_Image.gif)
		 [Image](IMapBackground.Image.htm)
		 Свойство Image
		 определяет изображение, которое используется в качестве фона легенды.


		 ![](../../Property_Image.gif)
		 [ImageDrawStyle](IMapBackground.ImageDrawStyle.htm)
		 Свойство ImageDrawStyle
		 определяет способ размещения изображения в качестве фона легенды.


		 ![](../../Property_Image.gif)
		 [Type](IMapBackground.Type.htm)
		 Свойство Type
		 определяет тип заливки фона легенды.
