# IMapBackground.ImageDrawStyle


## Синтаксис


ImageDrawStyle: [MapBackgroundImageDrawStyle](../../Enums/MapBackgroundImageDrawStyle.htm);


## Описание


Свойство ImageDrawStyle определяет
 способ размещения изображения в качестве фона легенды.


## Комментарии


Данное свойство актуально, если в качестве фона легенды задано изображение
 с помощью свойства [IMapBackground.Image](IMapBackground.Image.htm).
