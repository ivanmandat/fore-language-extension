# IMapBackground.Type


## Синтаксис


Type: [MapBackgroundType](../../Enums/MapBackgroundType.htm);


## Описание


Свойство Type определяет
 тип заливки фона легенды.


## Комментарии


Для настройки типа заливки используются определенные свойства:


	- MapBackgroundType.Color
	 - [IMapBackground.Color](IMapBackground.Color.htm);


	- MapBackgroundType.Gradient
	 - [IMapBackground.GradientAngle](IMapBackground.GradientAngle.htm),
	 [IMapBackground.GradientColor](IMapBackground.GradientColor.htm);


	- MapBackgroundType.Hatch
	 - [IMapBackground.HatchStyle](IMapBackground.HatchStyle.htm);


	- MapBackgroundType.Image
	 - [IMapBackground.Image](IMapBackground.Image.htm), [IMapBackground.ImageDrawStyle](IMapBackground.ImageDrawStyle.htm).
