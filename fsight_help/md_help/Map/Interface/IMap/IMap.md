# IMap


## Описание


Интерфейс IMap содержит свойства
 и методы для определения параметров карты.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Color](IMap.Color.htm)


		 Свойство Color определяет
		 цвет фона карты.


		 ![](../../Property_Image.gif)
		 [Font](IMap.Font.htm)


		 Свойство Font определяет
		 параметры шрифта на карте.


		 ![](../../Property_Image.gif)
		 [FontColor](IMap.FontColor.htm)


		 Свойство FontColor
		 определяет цвет шрифта на карте.


		 ![](../../Property_Image.gif)
		 [IsDirty](IMap.IsDirty.htm)


		 Свойство IsDirty определяет,
		 присутствуют ли изменения в параметрах карты.


		 ![](../../Property_Image.gif)
		 [IsEmpty](IMap.IsEmpty.htm)


		 Свойство IsEmpty возвращает
		 признак того, загружена ли карта.


		 ![](../../Property_Image.gif)
		 [IsIndicatorsEnabled](IMap.IsIndicatorsEnabled.htm)


		 Свойство IsIndicatorEnabled
		 определяет, отображаются ли метрики на карте.


		 ![](../../Property_Image.gif)
		 [Layers](IMap.Layers.htm)


		 Свойство Layers возвращает
		 коллекцию слоев карты.


		 ![](../../Property_Image.gif)
		 [TextBackgroundBrush](IMap.TextBackgroundBrush.htm)


		 Свойство TextBackgroundBrush
		 определяет фон текста на карте.


		 ![](../../Property_Image.gif)
		 [TextBorderPen](IMap.TextBorderPen.htm)


		 Свойство TextBorderPen
		 определяет параметры обводки текста на карте.


		 ![](../../Property_Image.gif)
		 [TimePoint](IMap.TimePoint.htm)


		 Свойство TimePoint
		 определяет текущую временную точку временной шкалы.


		 ![](../../Property_Image.gif)
		 [Topobase](IMap.TopoBase.htm)


		 Свойство Topobase определяет
		 топооснову карты.


		 ![](../../Property_Image.gif)
		 [Type](IMap.Type.htm)


		 Свойство Type
		 определяет тип карты.


		 ![](../../Property_Image.gif)
		 [UndoRedo](IMap.UndoRedo.htm)


		 Свойство UndoRedo возвращает
		 параметры стека изменений (отмены/повтора) карты.


		 ![](../../Property_Image.gif)
		 [View](IMap.View.htm)


		 Свойство View определяет
		 настройки отображения карты и объектов, расположенных на карте.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [BeginUpdate](IMap.BeginUpdate.htm)
		 Метод BeginUpdate прекращает
		 отрисовку карты.


		 ![](../../Sub_Image.gif)
		 [EndUpdate](IMap.EndUpdate.htm)
		 Метод EndUpdate возобновляет
		 отрисовку карты.


		 ![](../../Sub_Image.gif)
		 [FindVisual](IMap.FindVisual.htm)
		 Метод FindVisual осуществляет
		 поиск на карте показателя по его наименованию.


		 ![](../../Sub_Image.gif)
		 [GetBitmap](IMap.GetBitMap.htm)
		 Метод GetBitmap возвращает
		 графическое изображение карты по заданным параметрам.


		 ![](../../Sub_Image.gif)
		 [Refresh](IMap.Refresh.htm)
		 Метод Refresh осуществляет
		 сброс кеша данных и обновление карты.


		 ![](../../Sub_Image.gif)
		 [SaveMetafileToFile](IMap.SaveMetafileToFile.htm)
		 Метод SaveMetafileToFile
		 выгружает карту в метафайл заданного типа.


		 ![](../../Sub_Image.gif)
		 [SaveToXml](IMap.SaveToXml.htm)
		 Метод SaveToXml выгружает
		 карту в формат XML.


		 ![](../../Sub_Image.gif)
		 [SaveXAMLToFile](IMap.SaveXAMLToFile.htm)
		 Метод SaveXAMLToFile
		 выгружает карту в XAML-файл (*.xaml).
