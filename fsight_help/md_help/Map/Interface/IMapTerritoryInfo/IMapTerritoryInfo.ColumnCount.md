# IMapTerritoryInfo.ColumnCount


## Синтаксис


ColumnCount: Integer;


## Описание


Свойство ColumnCount определяет
 максимальное количество столбцов в расшифровке.


## Комментарии


Если значение данного свойства равно нулю, то количество столбцов в
 расшифровке устанавливается автоматически.
