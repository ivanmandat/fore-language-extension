# IMapTerritoryInfo


## Описание


Интерфейс IMapTerritoryInfo
 содержит свойства и методы для работы с расшифровкой территорий.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AttributeName](IMapTerritoryInfo.AttributeName.htm)
		 Свойство AttributeName
		 определяет наименование атрибута для отображения расшифровки.


		 ![](../../Property_Image.gif)
		 [BorderPen](IMapTerritoryInfo.BorderPen.htm)
		 Свойство BorderPen
		 определяет параметры границы расшифровки территорий.


		 ![](../../Property_Image.gif)
		 [Color](IMapTerritoryInfo.Color.htm)
		 Свойство Color определяет
		 цвет фона расшифровки территорий.


		 ![](../../Property_Image.gif)
		 [ColumnCount](IMapTerritoryInfo.ColumnCount.htm)
		 Свойство ColumnCount
		 определяет максимальное количество столбцов в расшифровке.


		 ![](../../Property_Image.gif)
		 [DisplayVisibleOnly](IMapTerritoryInfo.DisplayVisibleOnly.htm)
		 Свойство DisplayVisibleOnly
		 определяет порядок отображения элементов в расшифровке.


		 ![](../../Property_Image.gif)
		 [FillByRows](IMapTerritoryInfo.FillByRows.htm)
		 Свойство FillByRows
		 определяет порядок размещения элементов расшифровки.


		 ![](../../Property_Image.gif)
		 [Font](IMapTerritoryInfo.Font.htm)
		 Свойство Font
		 определяет параметры шрифта расшифровки.


		 ![](../../Property_Image.gif)
		 [FontColor](IMapTerritoryInfo.FontColor.htm)
		 Свойство FontColor
		 определяет параметры цвета шрифта расшифровки.


		 ![](../../Property_Image.gif)
		 [IsDirty](IMapTerritoryInfo.IsDirty.htm)
		 Свойство IsDirty определяет,
		 изменялась ли расшифровка территорий.


		 ![](../../Property_Image.gif)
		 [KeyAttributeName](IMapTerritoryInfo.KeyAttributeName.htm)
		 Свойство KeyAttributeName
		 определяет наименование атрибута для отображения идентификатора
		 территории.


		 ![](../../Property_Image.gif)
		 [Layer](IMapTerritoryInfo.Layer.htm)
		 Свойство Layer определяет
		 слой, по которому будет отображаться расшифровка.


		 ![](../../Property_Image.gif)
		 [MarkerCellPadding](IMapTerritoryInfo.MarkerCellPadding.htm)
		 Свойство MarkerCellPadding
		 определяет расстояние между идентификатором и наименованием элемента
		 расшифровки.


		 ![](../../Property_Image.gif)
		 [MaxItemCount](IMapTerritoryInfo.MaxItemCount.htm)
		 Свойство MaxItemCount
		 определяет максимальное количество элементов в расшифровке.


		 ![](../../Property_Image.gif)
		 [SortType](IMapTerritoryInfo.SortType.htm)
		 Свойство SortType определяет
		 параметры сортировки элементов в расшифровке.


		 ![](../../Property_Image.gif)
		 [Visible](IMapTerritoryInfo.Visible.htm)
		 Свойство Visible определяет
		 видимость расшифровки территорий.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [BeginUpdate](IMapTerritoryInfo.BeginUpdate.htm)
		 Метод BeginUpdate прекращает
		 отрисовку расшифровки территорий.


		 ![](../../Sub_Image.gif)
		 [Draw](IMapTerritoryInfo.Draw.htm)
		 Метод Draw выполняет
		 отрисовку указанного прямоугольника расшифровки территорий.


		 ![](../../Sub_Image.gif)
		 [EndUpdate](IMapTerritoryInfo.EndUpdate.htm)
		 Метод EndUpdate возобновляет
		 отрисовку расшифровки территорий.


		 ![](../../Sub_Image.gif)
		 [GetBitmap](IMapTerritoryInfo.GetBitmap.htm)
		 Метод GetBitmap возвращает
		 графическое изображение расшифровки территорий.


		 ![](../../Sub_Image.gif)
		 [HitTestShape](IMapTerritoryInfo.HitTestShape.htm)
		 Метод HitTestShape
		 позволяет получить область карты в заданной точке расшифровки
		 территорий.
