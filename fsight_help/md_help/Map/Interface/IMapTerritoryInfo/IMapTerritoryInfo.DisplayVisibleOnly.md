# IMapTerritoryInfo.DisplayVisibleOnly


## Синтаксис


DisplayVisibleOnly: Boolean;


## Описание


Свойство DisplayVisibleOnly
 определяет порядок отображения элементов в расшифровке.


## Комментарии


Допустимые значения:


	- True. В расшифровке
	 будут указаны только видимые в данный момент на карте территории;


	- False. В расшифровке
	 будут указаны все элементы.
