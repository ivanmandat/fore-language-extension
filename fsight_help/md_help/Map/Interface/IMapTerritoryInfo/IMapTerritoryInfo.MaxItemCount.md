# IMapTerritoryInfo.MaxItemCount


## Синтаксис


MaxItemCount: Integer;


## Описание


Свойство MaxItemCount определяет
 максимальное количество элементов в расшифровке.


## Комментарии


Если значение данного свойства равняется нулю, то в расшифровке будут
 отображаться все элементы.
