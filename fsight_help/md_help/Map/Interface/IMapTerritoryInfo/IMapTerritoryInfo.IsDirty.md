# IMapTerritoryInfo.IsDirty


## Синтаксис


IsDirty: Boolean;


## Описание


Свойство IsDirty определяет,
 изменялась ли расшифровка территорий.


## Комментарии


Допустимые значения:


	- True. Изменения в расшифровке
	 территорий присутствуют;


	- False. Изменения в расшифровке
	 территорий отсутствуют.
