# IMapImageVisual


## Описание


Интерфейс IMapImageVisual содержит
 свойства и методы для работы с пиктографическим показателем на слое карты.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Height](IMapImageVisual.Height.htm)
		 Свойство Height возвращает
		 параметры высоты пиктографического показателя.


		 ![](../../Property_Image.gif)
		 [Image](IMapImageVisual.Image.htm)
		 Свойство Image возвращает
		 параметры пиктограммы показателя.


		 ![](../../Property_Image.gif)
		 [StartAngle](IMapImageVisual.StartAngle.htm)
		 Свойство StartAngle
		 определяет угол поворота показателя.


		 ![](../../Property_Image.gif)
		 [Width](IMapImageVisual.Width.htm)
		 Свойство Width возвращает
		 параметры ширины пиктографического показателя.


## Свойства, унаследованные от [IMapVisual](../IMapVisual/IMapVisual.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AttributeName](../IMapVisual/IMapVisual.AttributeName.htm)
		 Свойство AttributeName
		 определяет наименование атрибута привязки показателя.


		 ![](../../Property_Image.gif)
		 [DataAdapter](../IMapVisual/IMapVisual.DataAdapter.htm)
		 Свойство DataAdapter
		 определяет источник данных для показателя.


		 ![](../../Property_Image.gif)
		 [Name](../IMapVisual/IMapVisual.Name.htm)
		 Свойство Name определяет
		 наименование показателя.


		 ![](../../Property_Image.gif)
		 [ToolTipFormat](../IMapVisual/IMapVisual.ToolTipFormat.htm)
		 Свойство ToolTipFormat
		 определяет формат, в соответствии с которым будут отображаться
		 значения показателя во всплывающей подсказке.


		 ![](../../Property_Image.gif)
		 [Visible](../IMapVisual/IMapVisual.Visible.htm)
		 Свойство Visible определяет
		 видимость показателя.


		 ![](../../Property_Image.gif)
		 [VisualType](../IMapVisual/IMapVisual.VisualType.htm)
		 Свойство VisualType
		 возвращает тип показателя.
