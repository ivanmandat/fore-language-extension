# IMapVisualColorProperty


## Описание


Интерфейс IMapVisualColorProperty
 содержит свойства для работы с параметрами заливки показателя.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DataAdapter](IMapVisualColorProperty.DataAdapter.htm)
		 Свойство DataAdapter
		 определяет источник данных, в соответствии с которым будет производиться
		 заливка.


		 ![](../../Property_Image.gif)
		 [Dependent](IMapVisualColorProperty.Dependent.htm)
		 Свойство Dependent
		 определяет, используется ли шкала для заливки показателя.


		 ![](../../Property_Image.gif)
		 [Scale](IMapVisualColorProperty.Scale.htm)
		 Свойство Scale определяет
		 шкалу для заливки показателя.


		 ![](../../Property_Image.gif)
		 [Values](IMapVisualColorProperty.Values.htm)
		 Свойство Values определяет
		 цвета заливки.
