# IMapVisualColorProperty.Dependent


## Синтаксис


Dependent: Boolean;


## Описание


Свойство Dependent определяет,
 используется ли шкала для заливки показателя.


## Комментарии


Допустимые значения:


	- True. Для
	 заливки используется шкала, заданная свойством [IMapVisualColorProperty.Scale](IMapVisualColorProperty.Scale.htm),
	 значение свойства [IMapVisualColorProperty.Values](IMapVisualColorProperty.Values.htm)
	 игнорируется;


	- False. Для
	 заливки используются цвета, заданные свойством [IMapVisualColorProperty.Values](IMapVisualColorProperty.Values.htm),
	 значение свойства [IMapVisualColorProperty.Scale](IMapVisualColorProperty.Scale.htm)
	 игнорируется.
