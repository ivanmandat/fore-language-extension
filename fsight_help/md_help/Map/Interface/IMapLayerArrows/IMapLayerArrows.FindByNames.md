# IMapLayerArrows.FindByNames


## Синтаксис


FindByNames(OriginID: String; DestID: String): [IMapArrow](../IMapArrow/IMapArrow.htm);


## Параметры


OriginID. Наименование территории,
 которая будет являться для стрелки исходной точкой;


DestID. Наименование территории,
 которая будет являться для стрелки конечной точкой.


## Описание


Метод FindByNames осуществляет
 поиск стрелки по заданным наименованиям соединяемых территорий.
