# IMapLayerArrows.BasementWidth


## Синтаксис


BasementWidth: [IMapVisualNumericProperty](../IMapVisualNumericProperty/IMapVisualNumericProperty.htm);


## Описание


Свойство BasementWidth возвращает
 параметры размера стрелок.


## Комментарии


Для использования свойства необходимо, чтобы свойство [IMapLayerArrows.UseDataMapping](IMapLayerArrows.UseDataMapping.htm)
 принимало значение True.


Для получения параметров заливки стрелок используйте [IMapLayerArrows.ColorMapping](IMapLayerArrows.ColorMapping.htm).
