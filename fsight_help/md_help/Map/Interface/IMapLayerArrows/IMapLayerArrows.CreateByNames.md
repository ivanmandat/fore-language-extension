# IMapLayerArrows.CreateByNames


## Синтаксис


CreateByNames(OriginID: String; DestID: String):
 [IMapArrow](../IMapArrow/IMapArrow.htm);


## Параметры


OriginID. Наименование территории,
 которая будет являться для стрелки исходной точкой;


DestID. Наименование территории,
 которая будет являться для стрелки конечной точкой.


## Описание


Метод CreateByNames добавляет
 в коллекцию стрелку по заданным наименованиям соединяемых территорий.
