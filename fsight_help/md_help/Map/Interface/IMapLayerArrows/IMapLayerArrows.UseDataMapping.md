# IMapLayerArrows.UseDataMapping


## Синтаксис


UseDataMapping: Boolean;


## Описание


Свойство UseDataMapping определяет,
 используются ли параметры размера и заливки для стрелки.


## Комментарии


Допустимые значения:


	- True. Параметры используются;


	- False. Параметры не
	 используются.


Для получения параметров заливки стрелки используйте [IMapLayerArrows.ColorMapping](IMapLayerArrows.ColorMapping.htm).


Для получения параметров размера стрелки используйте [IMapLayerArrows.BasementWidth](IMapLayerArrows.BasementWidth.htm).
