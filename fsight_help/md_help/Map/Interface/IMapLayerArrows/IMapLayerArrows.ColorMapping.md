# IMapLayerArrows.ColorMapping


## Синтаксис


ColorMapping: [IMapVisualColorProperty](../IMapVisualColorProperty/IMapVisualColorProperty.htm);


## Описание


Свойство ColorMapping возвращает
 параметры заливки стрелок.


## Комментарии


Для использования свойства необходимо, чтобы свойство [IMapLayerArrows.UseDataMapping](IMapLayerArrows.UseDataMapping.htm)
 принимало значение True.


Для получения числовых параметров размера стрелок используйте [IMapLayerArrows.BasementWidth](IMapLayerArrows.BasementWidth.htm).
