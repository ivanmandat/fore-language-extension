# IMapLayerArrows


Сборка: Map;


## Описание


Интерфейс IMapLayerArrows содержит
 свойства и методы для определения параметров коллекции стрелок.


## Иерархия наследования


IMapLayerArrows


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BasementWidth](IMapLayerArrows.BasementWidth.htm)
		 Свойство BasementWidth
		 возвращает параметры размера стрелок.


		 ![](../../Property_Image.gif)
		 [ColorMapping](IMapLayerArrows.ColorMapping.htm)
		 Свойство ColorMapping
		 возвращает параметры заливки стрелок.


		 ![](../../Property_Image.gif)
		 [Count](IMapLayerArrows.Count.htm)
		 Свойство Count возвращает
		 размер коллекции стрелок.


		 ![](../../Property_Image.gif)
		 [ToolTipFormat](IMapLayerArrows.ToolTipFormat.htm)
		 Свойство ToolTipFormat определяет
		 формат всплывающей подсказки для стрелок.


		 ![](../../Property_Image.gif)
		 [UseDataMapping](IMapLayerArrows.UseDataMapping.htm)
		 Свойство UseDataMapping определяет,
		 используются ли параметры размера и заливки для стрелки.
