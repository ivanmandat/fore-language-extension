# IMapLayerVisuals


## Описание


Интерфейс IMapLayerVisuals содержит
 свойства и методы для работы с коллекцией показателей слоя карты.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMapLayerVisuals.Count.htm)
		 Свойство Count возвращает
		 количество показателей в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMapLayerVisuals.Item.htm)
		 Свойство Item возвращает
		 показатель из коллекции по его индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IMapLayerVisuals.Add.htm)
		 Метод Add возвращает
		 индекс указанного показателя в коллекции.


		 ![](../../Sub_Image.gif)
		 [AddAreaVisual](IMapLayerVisuals.AddAreaVisual.htm)
		 Метод AddAreaVisual
		 добавляет картографический показатель в коллекцию.


		 ![](../../Sub_Image.gif)
		 [AddBarVisual](IMapLayerVisuals.AddBarVisual.htm)
		 Метод AddBarVisual
		 добавляет столбиковый показатель в коллекцию.


		 ![](../../Sub_Image.gif)
		 [AddImageVisual](IMapLayerVisuals.AddImageVisual.htm)
		 Метод AddImageVisual
		 добавляет пиктографический показатель в коллекцию.


		 ![](../../Sub_Image.gif)
		 [AddMarkerVisual](IMapLayerVisuals.AddMarkerVisual.htm)
		 Метод AddMarkerVisual
		 добавляет маркер геолокации в коллекцию.


		 ![](../../Sub_Image.gif)
		 [AddPieVisual](IMapLayerVisuals.AddPieVisual.htm)
		 Метод AddPieVisual
		 добавляет круговой показатель в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](IMapLayerVisuals.Clear.htm)
		 Метод Clear очищает
		 коллекцию показателей.


		 ![](../../Sub_Image.gif)
		 [Remove](IMapLayerVisuals.Remove.htm)
		 Метод Remove удаляет
		 показатель из коллекции по его индексу.
