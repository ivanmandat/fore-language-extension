# IMapTimeDynamicDataAdapter.TimePointName


## Синтаксис


TimePointName(TimePointIndex: Integer): String;


## Параметры


TimePointIndex. Параметр определяет
 временную точку.


## Описание


Свойство TimePointName возвращает
 наименование временной точки.


## Комментарии


Для получения количества временных точек используйте свойство [IMapTimeDynamicDataAdapter.TimePointsCount](IMapTimeDynamicDataAdapter.TimePointsCount.htm).
