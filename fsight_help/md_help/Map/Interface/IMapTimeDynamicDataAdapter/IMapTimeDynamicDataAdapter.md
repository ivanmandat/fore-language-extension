# IMapTimeDynamicDataAdapter


Сборка: Map;


## Описание


Интерфейс IMapTimeDynamicDataAdapter
 предназначен для получения информации о динамических показателях в
 различных временных точках на слое карты.


## Иерархия наследования


IMapTimeDynamicDataAdapter


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Data](IMapTimeDynamicDataAdapter.Data.htm)
		 Свойство Data возвращает
		 значение динамического временного адаптера в указанной области
		 данных и в указанную временную точку.


		 ![](../../Property_Image.gif)
		 [PartCount](IMapTimeDynamicDataAdapter.PartCount.htm)
		 Свойство PartCount
		 возвращает количество составных частей показателя для временной
		 точки.


		 ![](../../Property_Image.gif)
		 [TimePointName](IMapTimeDynamicDataAdapter.TimePointName.htm)
		 Свойство TimePointName
		 возвращает наименование временной точки.


		 ![](../../Property_Image.gif)
		 [TimePointsCount](IMapTimeDynamicDataAdapter.TimePointsCount.htm)
		 Свойство TimePointsCount
		 возвращает количество временных точек динамического временного
		 адаптера.
