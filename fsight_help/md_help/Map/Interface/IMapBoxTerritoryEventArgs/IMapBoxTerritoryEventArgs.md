# IMapBoxTerritoryEventArgs


## Описание


Интерфейс IMapBoxTerritoryEventArgs
 содержит свойства аргумента событий [OnTerritoryClick](../../Class/MapBox/MapBox.OnTerritoryClick.htm)
 компонента [MapBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/MapBox.htm),
 возникающего при щелчке мышью по какому-либо региону карты.
