# IMapTimeDataAdapter.TimePointName


## Синтаксис


TimePointName(TimePointIndex: Integer): String;


## Параметры


TimePointIndex. Параметр определяет
 временную точку.


## Описание


Свойство TimePointName возвращает
 наименование временной точки.


## Комментарии


Временная точка передается с помощью параметра TimePointIndex.
