# IMapPieVisual


## Описание


Интерфейс IMapPieVisual содержит
 свойства и методы для работы с круговым показателем.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Color](IMapPieVisual.Color.htm)
		 Свойство Color возвращает
		 параметры заливки показателя.


		 ![](../../Property_Image.gif)
		 [Fill](IMapPieVisual.Fill.htm)
		 Свойство Fill возвращает
		 параметры штриховки показателя.


		 ![](../../Property_Image.gif)
		 [FillColor](IMapPieVisual.FillColor.htm)
		 Свойство FillColor
		 возвращает параметры цвета штриховки показателя.


		 ![](../../Property_Image.gif)
		 [LeastRadiusPart](IMapPieVisual.LeastRadiusPart.htm)
		 Свойство LeastRadiusPart
		 определяет долю минимального радиуса.


		 ![](../../Property_Image.gif)
		 [PiesCount](IMapPieVisual.PiesCount.htm)
		 Свойство PiesCount
		 определяет максимальное количество отображаемых секторов показателя.


		 ![](../../Property_Image.gif)
		 [Proportional](IMapPieVisual.Proportional.htm)
		 Свойство Proportional
		 определяет, будет ли показатель пропорциональным.


		 ![](../../Property_Image.gif)
		 [Radius](IMapPieVisual.Radius.htm)
		 Свойство Radius определяет
		 радиус частей показателя.


		 ![](../../Property_Image.gif)
		 [StartAngle](IMapPieVisual.StartAngle.htm)
		 Свойство StartAngle
		 определяет начальный угол поворота показателя.


		 ![](../../Property_Image.gif)
		 [UseInvertedAngles](IMapPieVisual.UseInvertedAngles.htm)
		 Свойство UseInvertedAngles
		 определяет, будут ли углы показателя отображаться в обратную сторону.
