# IMapColorScale.Greater


## Синтаксис


Greater(Index: Integer): [IGxColor](ModDrawing.chm::/Interface/IGxColor/IGxColor.htm);


## Параметры


Index. Индекс деления шкалы.


## Описание


Свойство Greater определяет
 цвет заливки показателя, значения которого больше значения деления шкалы.


## Комментарии


Каждому [значению деления
 шкалы](../IMapScale/IMapScale.Value.htm) соответствует три цветовых деления (меньше, равно, больше).
