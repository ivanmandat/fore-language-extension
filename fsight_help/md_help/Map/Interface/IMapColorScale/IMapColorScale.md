# IMapColorScale


## Описание


Интерфейс IMapColorScale определяет
 параметры цветовой шкалы для карты.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Color](IMapColorScale.Color.htm)
		 Свойство Color определяет
		 цвет заливки относительно заданного значения деления шкалы.


		 ![](../../Property_Image.gif)
		 [Exact](IMapColorScale.Exact.htm)
		 Свойство Exact определяет
		 цвет заливки показателя, значения которого равны значению деления
		 шкалы.


		 ![](../../Property_Image.gif)
		 [Greater](IMapColorScale.Greater.htm)
		 Свойство Greater определяет
		 цвет заливки показателя, значения которого больше значения деления
		 шкалы.


		 ![](../../Property_Image.gif)
		 [Less](IMapColorScale.Less.htm)
		 Свойство Less определяет
		 цвет заливки показателя, значения которого меньше значения деления
		 шкалы.


		 ![](../../Property_Image.gif)
		 [NoData](IMapColorScale.NoData.htm)
		 Свойство NoData определяет
		 цвет заливки для отсутствующих значений.


## Свойства, унаследованные от [IMapScale](../IMapScale/IMapScale.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AutoCalculable](../IMapScale/IMapScale.AutoCalculable.htm)
		 Свойство AutoCalculable
		 определяет признак автоматического расчета [значений](../IMapScale/IMapScale.Value.htm)
		 для шкалы отчета.


		 ![](../../Property_Image.gif)
		 [AutoScaleMode](../IMapScale/IMapScale.AutoScaleMode.htm)
		 Свойство AutoScaleMode
		 определяет тип автоматического расчета шкалы отчета.


		 ![](../../Property_Image.gif)
		 [Count](../IMapScale/IMapScale.Count.htm)
		 Свойство Count определяет
		 количество значений в шкале (разбиений).


		 ![](../../Property_Image.gif)
		 [IsDirty](../IMapScale/IMapScale.IsDirty.htm)
		 Свойство IsDirty возвращает
		 признак изменения шкалы отчета.


		 ![](../../Property_Image.gif)
		 [Value](../IMapScale/IMapScale.Value.htm)
		 Свойство Value определяет
		 значение для деления шкалы.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AutoSetup](IMapColorScale.AutoSetup.htm)
		 Метод AutoSetup производит
		 автонастройку цветовой шкалы: применяет градиентную заливку и
		 производит расчет [значений](../IMapScale/IMapScale.Value.htm)
		 для делений шкалы с сохранением заданного [количества
		 интервалов](../IMapScale/IMapScale.Count.htm).


		 ![](../../Sub_Image.gif)
		 [AutoSetupEx](IMapColorScale.AutoSetupEx.htm)
		 Метод AutoSetupEx производит
		 автонастройку цветовой шкалы: применяет градиентную заливку и
		 производит расчет [значений](../IMapScale/IMapScale.Value.htm)
		 для делений шкалы.
