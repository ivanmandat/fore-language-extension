# IMapLayers


## Описание


Интерфейс IMapLayers содержит
 свойства для определения параметров коллекции слоев карты.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMapLayers.Count.htm)
		 Свойство Count возвращает
		 количество слоев в коллекции.


		 ![](../../Property_Image.gif)
		 [FindById](IMapLayers.FindById.htm)
		 Свойство FindById возвращает
		 слой по его идентификатору.


		 ![](../../Property_Image.gif)
		 [FindByName](IMapLayers.FindByName.htm)
		 Свойство FindByName
		 возвращает слой по его наименованию.


		 ![](../../Property_Image.gif)
		 [Item](IMapLayers.Item.htm)
		 Свойство Item возвращает
		 заданный слой из коллекции.
