# IMapLayers.FindByName


## Синтаксис


FindById(LayerName: String): [IMapLayer](../IMapLayer/IMapLayer.htm);


## Параметры


LayerName. Наименование слоя.


## Описание


Свойство FindByName возвращает
 слой по его наименованию.


## Комментарии


Наименование слоя передается посредством параметра LayerName.
