# IMapArrow.BasementWidth


## Синтаксис


BasementWidth: [IMapVisualNumericProperty](../IMapVisualNumericProperty/IMapVisualNumericProperty.htm);


## Описание


Свойство BasementWidth возвращает
 параметры размера стрелки.


## Комментарии


Для использования свойства необходимо, чтобы свойство [IMapLayerArrows.UseDataMapping](../IMapLayerArrows/IMapLayerArrows.UseDataMapping.htm)
 принимало значение True.


Для получения параметров заливки стрелки используйте [IMapArrow.ColorMapping](IMapArrow.ColorMapping.htm).
