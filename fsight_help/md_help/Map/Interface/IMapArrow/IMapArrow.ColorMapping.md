# IMapArrow.ColorMapping


## Синтаксис


ColorMapping: [IMapVisualColorProperty](../IMapVisualColorProperty/IMapVisualColorProperty.htm);


## Описание


Свойство ColorMapping возвращает
 параметры заливки стрелки.


## Комментарии


Для использования свойства необходимо, чтобы свойство [IMapLayerArrows.UseDataMapping](../IMapLayerArrows/IMapLayerArrows.UseDataMapping.htm)
 принимало значение True.


Для получения числовых параметров размера стрелок используйте [IMapArrow.BasementWidth](IMapArrow.BasementWidth.htm).
