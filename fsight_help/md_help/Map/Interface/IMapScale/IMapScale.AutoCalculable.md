# IMapScale.AutoCalculable


## Синтаксис


AutoCalculable: Boolean;


## Описание


Свойство AutoCalculable определяет
 признак автоматического расчета значений ([Value](IMapScale.Value.htm))
 для шкалы отчета.


## Комментарии


Допустимые значения:


	- True. Происходит автоматический
	 расчет значений для шкалы;


	- False. Автоматический
	 расчет не производится.


При установке автоматического расчета шкалы по умолчанию выбирается
 линейный метод. Для выбора типа расчета используется свойство [IMapScale.AutoScaleMode](IMapScale.AutoScaleMode.htm).
