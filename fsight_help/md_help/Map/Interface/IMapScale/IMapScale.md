# IMapScale


## Описание


Интерфейс IMapScale определяет
 параметры шкал отчета.


## Комментарии


Данный интерфейс является базовым для следующих интерфейсов:


	- [IMapColorScale](../IMapColorScale/IMapColorScale.htm)


	- [IMapFillScale](../IMapFillScale/IMapFillScale.htm)


	- [IMapIconScale](../IMapIconScale/IMapIconScale.htm)


	- [IMapNumericScale](../IMapNumericScale/IMapNumericScale.htm)


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AutoCalculable](IMapScale.AutoCalculable.htm)
		 Свойство AutoCalculable
		 определяет признак автоматического расчета [значений](IMapScale.Value.htm)
		 для шкалы отчета.


		 ![](../../Property_Image.gif)
		 [AutoScaleMode](IMapScale.AutoScaleMode.htm)
		 Свойство AutoScaleMode
		 определяет тип автоматического расчета шкалы отчета.


		 ![](../../Property_Image.gif)
		 [Count](IMapScale.Count.htm)
		 Свойство Count определяет
		 количество значений в шкале (разбиений).


		 ![](../../Property_Image.gif)
		 [IsDirty](IMapScale.IsDirty.htm)
		 Свойство IsDirty возвращает
		 признак изменения шкалы отчета.


		 ![](../../Property_Image.gif)
		 [Value](IMapScale.Value.htm)
		 Свойство Value определяет
		 значение для деления шкалы.


## Методы


		 Имя метода

		 ![](../../Sub_Image.gif)
		 [AutoCalculate](IMapScale.AutoCalculate.htm)
