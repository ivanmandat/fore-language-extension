# IMapScale.Value


## Синтаксис


Value(Index: Integer): Double;


## Параметры


Index. Индекс деления шкалы.


## Описание


Свойство Value определяет значение
 для деления шкалы.


## Комментарии


По заданному значению будет определен цвет/штриховка/число (в зависимости
 от типа шкалы) для значений показателя. При использовании автоматического
 расчета значений шкалы ([AutoCalculate](IMapScale.AutoCalculate.htm))
 данное свойство задавать не следует. Для шкал также реализованы методы
 AutoSetup и/или AutoSetupEx, которые позволяют автоматически рассчитывать
 значения для делений шкалы.
