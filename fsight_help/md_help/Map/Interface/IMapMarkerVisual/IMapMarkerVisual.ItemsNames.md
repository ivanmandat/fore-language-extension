# IMapMarkerVisual.ItemsNames


## Синтаксис


ItemsNames: Array;


## Описание


Свойство ItemsNames определяет
 наименования маркеров геолокации.


## Комментарии


Наименования задаются в виде массива строк. Каждый элемент массива соответствует
 маркеру.
