# IMapMarkerVisual.Color


## Синтаксис


Color: [IMapVisualColorProperty](../IMapVisualColorProperty/IMapVisualColorProperty.htm);


## Описание


Свойство Color возвращает
 параметры цвета маркера геолокации.


## Комментарии


С помощью данного свойства можно настроить шкалу для определения
 цвета маркера или задать определенный цвет.
