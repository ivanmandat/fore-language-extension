# IMapColorCollection


## Описание


Интерфейс IMapColorCollection
 определяет коллекцию цветов показателя. Коллекция цветов актуальна для
 [круговых и столбиковых показателей](../../Enums/MapVisualType.htm).


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMapColorCollection.Count.htm)
		 Свойство Count определяет
		 количество цветов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMapColorCollection.Item.htm)
		 Свойство Item определяет
		 цвет элемента в коллекции по заданному индексу.
