# IMapVisualFillProperty


## Описание


Интерфейс IMapVisualFillProperty
 содержит свойства для работы с параметрами штриховки показателя.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DataAdapter](IMapVisualFillProperty.DataAdapter.htm)
		 Свойство DataAdapter
		 определяет источник данных, в соответствии с которым будет производиться
		 штриховка.


		 ![](../../Property_Image.gif)
		 [Dependent](IMapVisualFillProperty.Dependent.htm)
		 Свойство Dependent
		 определяет, используется ли шкала для штриховки показателя.


		 ![](../../Property_Image.gif)
		 [Scale](IMapVisualFillProperty.Scale.htm)
		 Свойство Scale определяет
		 шкалу для штриховки показателя.


		 ![](../../Property_Image.gif)
		 [Value](IMapVisualFillProperty.Value.htm)
		 Свойство Value определяет
		 тип штриховки показателя.
