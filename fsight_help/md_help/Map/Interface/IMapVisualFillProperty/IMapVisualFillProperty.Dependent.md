# IMapVisualFillProperty.Dependent


## Синтаксис


Dependent: Boolean;


## Описание


Свойство Dependent определяет,
 используется ли шкала для штриховки показателя.


## Комментарии


Допустимые значения:


	- True. Для
	 штриховки используется шкала, заданная свойством [IMapVisualFillProperty.Scale](IMapVisualFillProperty.Scale.htm),
	 значение свойства [IMapVisualFillProperty.Value](IMapVisualFillProperty.Value.htm)
	 игнорируется;


	- False. Для
	 штриховки используются цвета, заданные свойством [IMapVisualFillProperty.Value](IMapVisualFillProperty.Value.htm),
	 значение свойства [IMapVisualFillProperty.Scale](IMapVisualFillProperty.Scale.htm)
	 игнорируется.
