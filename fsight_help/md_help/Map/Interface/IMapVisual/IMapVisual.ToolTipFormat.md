# IMapVisual.ToolTipFormat


## Синтаксис


ToolTipFormat: String;


## Описание


Свойство ToolTipFormat определяет
 формат, в соответствии с которым будут отображаться значения показателя
 во всплывающей подсказке.


## Комментарии


Указываемый в данном свойстве формат не зависит от локализации операционной
 системы. При установке формата в качестве разделителей будут использоваться
 запятые и пробелы. При формировании отображаемых значений будут учитываться
 региональные настройки используемой культуры. Формат формируется с использованием
 зарезервированных кодов. Описание кодов представлено в разделе [Формат
 числа: Все форматы](uinav.chm::/GUI/Format/UiReport_Table_Attribute_Format.htm).
