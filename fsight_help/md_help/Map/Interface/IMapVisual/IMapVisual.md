# IMapVisual


## Описание


Интерфейс IMapVisual содержит
 свойства и методы для работы с показателями на слое карты. Данный интерфейс
 является базовым для различных видов показателей.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AttributeName](IMapVisual.AttributeName.htm)
		 Свойство AttributeName
		 определяет наименование атрибута привязки показателя.


		 ![](../../Property_Image.gif)
		 [DataAdapter](IMapVisual.DataAdapter.htm)
		 Свойство DataAdapter
		 определяет источник данных для показателя.


		 ![](../../Property_Image.gif)
		 [Name](IMapVisual.Name.htm)
		 Свойство Name определяет
		 наименование показателя.


		 ![](../../Property_Image.gif)
		 [ToolTipFormat](IMapVisual.ToolTipFormat.htm)
		 Свойство ToolTipFormat
		 определяет формат, в соответствии с которым будут отображаться
		 значения показателя во всплывающей подсказке.


		 ![](../../Property_Image.gif)
		 [Visible](IMapVisual.Visible.htm)
		 Свойство Visible определяет
		 видимость показателя.


		 ![](../../Property_Image.gif)
		 [VisualType](IMapVisual.VisualType.htm)
		 Свойство VisualType
		 возвращает тип показателя.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [BeginUpdate](IMapVisual.BeginUpdate.htm)
		 Метод BeginUpdate прекращает
		 отрисовку показателя.


		 ![](../../Sub_Image.gif)
		 [EndUpdate](IMapVisual.EndUpdate.htm)
		 Метод EndUpdate возобновляет
		 отрисовку данных.


		 ![](../../Sub_Image.gif)
		 [RefreshData](IMapVisual.RefreshData.htm)
		 Метод RefreshData обновляет
		 показатель.


		 ![](../../Sub_Image.gif)
		 [ResetCache](IMapVisual.ResetCache.htm)
		 Метод ResetCache осуществляет
		 сброс кеша источника данных.
