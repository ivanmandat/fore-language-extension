# IMapTopobaseAttributeNames


## Описание


Интерфейс IMapTopobaseAttributeNames
 содержит свойства для определения параметров коллекции наименований атрибутов
 слоя карты.


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMapTopobaseAttributeNames.Count.htm)
		 Свойство Count возвращает
		 количество наименований атрибутов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMapTopobaseAttributeNames.Item.htm)
		 Свойство Item возвращает
		 наименование атрибута из коллекции.
