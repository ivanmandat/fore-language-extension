# IMapLayer.ToolTipDataAdapter


## Синтаксис


ToolTipDataAdapter: Object;


## Описание


Свойство ToolTipDataAdapter
 определяет источник данных для всплывающей подсказки слоя карты.


## Комментарии


Если для слоя карты установлен флаг MouseTransparent,
 т.е., слой является прозрачным, установка всплывающей подсказки невозможна.
