# IMapLayer.PlaceVisualsOneByOne


## Синтаксис


PlaceVisualsOneByOne: Boolean;


## Описание


Свойство PlaceVisualsOneByOne
 определяет, будут ли столбиковые и круговые показатели располагаться рядом
 на одной линии.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Показатели разных типов будут размещаться рядом на одной линии;


	- False. Показатели разных
	 типов будут накладываться друг на друга.


Для задания расстояния между столбиковыми и круговыми показателями,
 расположенными на одной линии, используйте [IMapLayer.VisualOffset](IMapLayer.VisualOffset.htm).
