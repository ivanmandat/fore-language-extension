# IMapLayer.TextVisible


## Синтаксис


TextVisible: Boolean;


## Описание


Свойство TextVisible определяет
 отображение подписей на слое.


## Комментарии


Допустимые значения:


	- True. Подписи на слое
	 будут отображаться;


	- False. Подписи на слое
	 не будут отображаться.
