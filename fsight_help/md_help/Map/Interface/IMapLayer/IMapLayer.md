# IMapLayer


Сборка: Map;


## Описание


Интерфейс IMapLayer содержит
 свойства для определения параметров слоя карты.


## Иерархия наследования


IMapLayer

		 [AttributeNames](IMapLayer.AttributeNames.htm)

		 возвращает коллекцию наименований атрибутов.

		 возвращает коллекцию дочерних слоев.

		 [Description](IMapLayer.Description.htm)

		 определяет описание слоя.

		 ![](../../Property_Image.gif)
		 [FontColor](IMapLayer.FontColor.htm)
		 Свойство FontColor
		 определяет цвет шрифта подписи слоя карты.


		 ![](../../Property_Image.gif)
		 [Id](IMapLayer.Id.htm)
		 Свойство Id определяет
		 уникальный идентификатор слоя карты.


		 ![](../../Property_Image.gif)
		 [LabelFormat](IMapLayer.LabelFormat.htm)
		 Свойство LabelFormat
		 определяет текст подписи.


		 ![](../../Property_Image.gif)
		 [LabelHorizontalAlignment](IMapLayer.LabelHorizontalAlignment.htm)
		 Свойство LabelHorizontalAlignment
		 определяет положение подписи слоя карты по горизонтали относительно
		 позиции, заданной в топооснове.


		 ![](../../Property_Image.gif)
		 [LabelVerticalAlignment](IMapLayer.LabelVerticalAlignment.htm)
		 Свойство LabelVerticalAlignment
		 определяет положение подписи слоя карты по вертикали относительно
		 позиции, заданной в топооснове.


		 ![](../../Property_Image.gif)
		 [Name](IMapLayer.Name.htm)
		 Свойство Name определяет
		 наименование слоя карты.

		 [Owner](IMapLayer.Owner.htm)
		 Свойство Owner возвращает

		 [PlaceVisualsOneByOne](IMapLayer.PlaceVisualsOneByOne.htm)
		 Свойство PlaceVisualsOneByOne

		 Свойство TerritoryInfoAdapter

		 ![](../../Property_Image.gif)
		 [TextDataAdapter](IMapLayer.TextDataAdapter.htm)

		 определяет источник данных для подписей слоя карты.

		 [TextVisible](IMapLayer.TextVisible.htm)

		 ![](../../Property_Image.gif)
		 [ToolTipVisible](IMapLayer.ToolTipVisible.htm)
		 Свойство ToolTipVisible
		 определяет необходимость отображения всплывающих подсказок для

		 Свойство VisualOffset определяет

		 на одной линии.


		 ![](../../Property_Image.gif)
		 [Visuals](IMapLayer.Visuals.htm)
		 Свойство Visuals возвращает
		 коллекцию показателей слоя.
