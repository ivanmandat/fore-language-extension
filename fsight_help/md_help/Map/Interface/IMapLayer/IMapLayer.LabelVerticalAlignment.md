# IMapLayer.LabelVerticalAlignment


## Синтаксис


LabelVerticalAlignment: [MapLayerLabelVerticalAlignment](../../Enums/MapLayerLabelVerticalAlignment.htm);


## Описание


Свойство LabelVerticalAlignment
 определяет положение подписи слоя карты по вертикали относительно позиции,
 заданной в топооснове.


## Комментарии


Для определения положения подписи слоя карты по горизонтали относительно
 позиции из топоосновы используйте [IMapLayer.LabelHorizontalAlignment](IMapLayer.LabelHorizontalAlignment.htm).
