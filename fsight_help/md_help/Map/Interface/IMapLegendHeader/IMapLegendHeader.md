# IMapLegendHeader


## Описание


Интерфейс IMapLegendHeader содержит
 свойства для работы с верхним и нижним текстом легенды.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Color](IMapLegendHeader.Color.htm)
		 Свойство Color определяет
		 цвет текста.


		 ![](../../Property_Image.gif)
		 [Font](IMapLegendHeader.Font.htm)
		 Свойство Font определяет
		 параметры шрифта текста.


		 ![](../../Property_Image.gif)
		 [Text](IMapLegendHeader.Text.htm)
		 Свойство Text определяет
		 верхний/нижний текст легенды.
