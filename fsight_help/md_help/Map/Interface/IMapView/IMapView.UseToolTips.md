# IMapView.UseToolTips


## Синтаксис


UseToolTips: Boolean;


## Описание


Свойство UseToolTips определяет
 признак использования всплывающих подсказок.


## Комментарии


Допустимые значения:


	- True - всплывающие подсказки
	 используются;


	- False - всплывающие
	 подсказки не используются.
