# IMapView.SelectionFlashTimerInterval


## Синтаксис


SelectionFlashTimerInterval: Integer;


## Описание


Свойство SelectionFlashTimerInterval
 определяет интервал мигания подсветки территорий (в миллисекундах
 на одну фазу).
