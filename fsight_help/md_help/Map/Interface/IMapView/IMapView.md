# IMapView


## Описание


Интерфейс IMapView содержит
 свойства и методы, определяющие параметры отображения карты и объектов,
 расположенных на карте.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ImageMap](IMapView.ImageMap.htm)
		 Свойство ImageMap определяет
		 параметры гиперссылки при щелчке по региону.


		 ![](../../Property_Image.gif)
		 [Legend](IMapView.Legend.htm)
		 Свойство Legend определяет
		 легенду карты.


		 ![](../../Property_Image.gif)
		 [LegendRect](IMapView.LegendRect.htm)
		 Свойство LegendRect
		 определяет область карты, в которой будет располагаться легенда.


		 ![](../../Property_Image.gif)
		 [Map](IMapView.Map.htm)
		 Свойство Map возвращает
		 карту-родитель.


		 ![](../../Property_Image.gif)
		 [SelectionFlashPhasesCount](IMapView.SelectionFlashPhasesCount.htm)
		 Свойство SelectionFlashPhasesCount
		 определяет количество фаз мигания подсветки территорий.


		 ![](../../Property_Image.gif)
		 [SelectionFlashTimerInterval](IMapView.SelectionFlashTimerInterval.htm)
		 Свойство SelectionFlashTimerInterval
		 определяет интервал мигания подсветки территорий.


		 ![](../../Property_Image.gif)
		 [SelectionMode](IMapView.SelectionMode.htm)
		 Свойство SelectionMode
		 определяет режим выделения территорий карты.


		 ![](../../Property_Image.gif)
		 [TerritoryInfo](IMapView.TerritoryInfo.htm)
		 Свойство TerritoryInfo
		 определяет расшифровку территорий карты.


		 ![](../../Property_Image.gif)
		 [TerritoryInfoRect](IMapView.TerritoryInfoRect.htm)
		 Свойство TerritoryInfoRect
		 определяет область карты, в которой будет располагаться расшифровка
		 территорий.


		 ![](../../Property_Image.gif)
		 [UseProportionality](IMapView.UseProportionality.htm)
		 Свойство UseProportionality
		 определяет, будет ли соблюдаться пропорциональность ширины и высоты
		 карты.


		 ![](../../Property_Image.gif)
		 [UseScrollBars](IMapView.UseScrollBars.htm)
		 Свойство UseScrollBars
		 определяет, используются ли полосы прокрутки.


		 ![](../../Property_Image.gif)
		 [UseSelectionFlash](IMapView.UseSelectionFlash.htm)
		 Свойство UseSelectionFlash
		 определяет, используется ли мигание при подсветке территорий.


		 ![](../../Property_Image.gif)
		 [UseToolTips](IMapView.UseToolTips.htm)
		 Свойство UseToolTips
		 определяет признак использования всплывающих подсказок.


		 ![](../../Property_Image.gif)
		 [ViewRect](IMapView.ViewRect.htm)
		 Свойство ViewRect определяет
		 область видимости карты.


		 ![](../../Property_Image.gif)
		 [WndMode](IMapView.WndMode.htm)
		 Свойство WndMode определяет
		 режим интерактивности карты.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [GetImage](IMapView.GetImage.htm)
		 Метод GetImage возвращает
		 графическое изображение указанной области карты и объектов, расположенных
		 на ней.


		 ![](../../Sub_Image.gif)
		 [HitTestShape](IMapView.HitTestShape.htm)
		 Метод HitTestShape
		 позволяет получить область карты в заданной точке карты.


		 ![](../../Sub_Image.gif)
		 [Refresh](IMapView.Refresh.htm)
		 Метод Refresh осуществляет
		 обновление карты и расположенных на ней объектов.


		 ![](../../Sub_Image.gif)
		 [ScrollIntoView](IMapView.ScrollIntoView.htm)
		 Метод ScrollIntoView
		 помещает указанную область слоя карты в область видимости.
