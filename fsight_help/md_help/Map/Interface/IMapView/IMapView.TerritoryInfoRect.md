# IMapView.TerritoryInfoRect


## Синтаксис


TerritoryInfoRect: [IGxRectF](ModDrawing.chm::/Interface/IGxRectF/IGxRectF.htm);


## Описание


Свойство TerritoryInfoRect определяет
 область карты, в которой будет располагаться расшифровка территорий.


## Комментарии


Область задается в виде прямоугольника, размеры которого изменяются
 в пределах от [0..1].
