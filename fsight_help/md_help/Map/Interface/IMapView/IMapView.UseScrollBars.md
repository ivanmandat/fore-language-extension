# IMapView.UseScrollBars


## Синтаксис


UseScrollBars: Boolean;


## Описание


Свойство UseScrollBars определяет,
 используются ли полосы прокрутки.


## Комментарии


Свойство актуально только при работе с формами.


Допустимые значения:


	- True. Полосы прокрутки
	 используются;


	- False. Полосы прокрутки
	 не используются.
