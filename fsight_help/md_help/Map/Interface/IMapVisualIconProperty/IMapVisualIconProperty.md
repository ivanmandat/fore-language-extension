# IMapVisualIconProperty


## Описание


Интерфейс IMapVisualIconProperty
 содержит свойства для работы с параметрами пиктограммы показателя.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DataAdapter](IMapVisualIconProperty.DataAdapter.htm)
		 Свойство DataAdapter
		 определяет источник данных, в соответствии с которым будет производиться
		 построение показателя.


		 ![](../../Property_Image.gif)
		 [Dependent](IMapVisualIconProperty.Dependent.htm)
		 Свойство Dependent
		 определяет, используется ли шкала для показателя.


		 ![](../../Property_Image.gif)
		 [Scale](IMapVisualIconProperty.Scale.htm)
		 Свойство Scale определяет
		 шкалу для показателя.


		 ![](../../Property_Image.gif)
		 [Value](IMapVisualIconProperty.Value.htm)
		 Свойство Value определяет
		 пиктограмму показателя.
