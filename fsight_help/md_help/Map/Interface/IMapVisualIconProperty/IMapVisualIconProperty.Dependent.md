# IMapVisualIconProperty.Dependent


## Синтаксис


Dependent: Boolean;


## Описание


Свойство Dependent определяет,
 используется ли шкала для показателя.


## Комментарии


Допустимые значения:


	- True. Используется
	 шкала, заданная свойством [IMapVisualIconProperty.Scale](IMapVisualIconProperty.Scale.htm),
	 значение свойства [IMapVisualIconProperty.Value](IMapVisualIconProperty.Value.htm)
	 игнорируется;


	- False. Используются
	 пиктограмма, заданная свойством [IMapVisualIconProperty.Value](IMapVisualIconProperty.Value.htm),
	 значение свойства [IMapVisualIconProperty.Scale](IMapVisualIconProperty.Scale.htm)
	 игнорируется.
