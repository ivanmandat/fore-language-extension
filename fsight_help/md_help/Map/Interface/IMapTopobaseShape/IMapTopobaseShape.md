# IMapTopobaseShape


## Описание


Интерфейс IMapTopobaseShape
 содержит свойства для определения параметров области слоя карты.


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AttributeValue](IMapTopobaseShape.AttributeValue.htm)
		 Свойство AttributeValue
		 возвращает значение атрибута области по его наименованию.


		 ![](../../Property_Image.gif)
		 [BarOriginOffset](IMapTopobaseShape.BarOriginOffset.htm)
		 Свойство BarOriginOffset
		 определяет параметры точки привязки столбикового показателя.


		 ![](../../Property_Image.gif)
		 [Description](IMapTopobaseShape.Description.htm)
		 Свойство Description
		 определяет описание области.


		 ![](../../Property_Image.gif)
		 [Id](IMapTopobaseShape.Id.htm)
		 Свойство Id определяет
		 идентификатор области.


		 ![](../../Property_Image.gif)
		 [ImageOriginOffset](IMapTopobaseShape.ImageOriginOffset.htm)
		 Свойство ImageOriginOffset
		 определяет параметры точки привязки пиктографического показателя.


		 ![](../../Property_Image.gif)
		 [Name](IMapTopobaseShape.Name.htm)
		 Свойство Name определяет
		 наименование области.


		 ![](../../Property_Image.gif)
		 [OriginOffset](IMapTopobaseShape.OriginOffset.htm)
		 Свойство OriginOffset
		 определяет параметры точки привязки стрелки.


		 ![](../../Property_Image.gif)
		 [Owner](IMapTopobaseShape.Owner.htm)
		 Свойство Owner возвращает
		 слой, которому принадлежит область.


		 ![](../../Property_Image.gif)
		 [PieOriginOffset](IMapTopobaseShape.PieOriginOffset.htm)
		 Свойство PieOriginOffset
		 определяет параметры точки привязки кругового показателя.


		 ![](../../Property_Image.gif)
		 [Primitives](IMapTopobaseShape.Primitives.htm)
		 Свойство Primitives
		 определяет коллекцию примитивов, из которых состоит область.


		 ![](../../Property_Image.gif)
		 [Text](IMapTopobaseShape.Text.htm)
		 Свойство Text определяет
		 подпись области.


		 ![](../../Property_Image.gif)
		 [TextOriginOffset](IMapTopobaseShape.TextOriginOffset.htm)
		 Свойство TextOriginOffset
		 определяет параметры точки привязки подписи области.
