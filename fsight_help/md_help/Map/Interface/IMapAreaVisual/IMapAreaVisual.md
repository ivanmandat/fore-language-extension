# IMapAreaVisual


## Описание


Интерфейс IMapAreaVisual содержит
 свойства и методы для работы с картографическим показателем на слое карты.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Color](IMapAreaVisual.Color.htm)
		 Свойство Color возвращает
		 параметры заливки показателя.


		 ![](../../Property_Image.gif)
		 [Fill](IMapAreaVisual.Fill.htm)
		 Свойство Fill возвращает
		 параметры штриховки показателя.


		 ![](../../Property_Image.gif)
		 [FillColor](IMapAreaVisual.FillColor.htm)
		 Свойство FillColor
		 возвращает параметры цвета штриховки показателя.


## Свойства, унаследованные от [IMapVisual](../IMapVisual/IMapVisual.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)

		 Свойство AttributeName
		 определяет наименование атрибута привязки показателя.


		 ![](../../Property_Image.gif)
		 [DataAdapter](../IMapVisual/IMapVisual.DataAdapter.htm)
		 Свойство DataAdapter
		 определяет источник данных для показателя.


		 ![](../../Property_Image.gif)
		 [Name](../IMapVisual/IMapVisual.Name.htm)
		 Свойство Name определяет

		 [ToolTipFormat](../IMapVisual/IMapVisual.ToolTipFormat.htm)
		 Свойство ToolTipFormat
		 определяет формат, в соответствии с которым будут отображаться
		 значения показателя во всплывающей подсказке.


		 ![](../../Property_Image.gif)
		 [Visible](../IMapVisual/IMapVisual.Visible.htm)
		 Свойство Visible определяет
		 видимость показателя.


		 ![](../../Property_Image.gif)
		 [VisualType](../IMapVisual/IMapVisual.VisualType.htm)
		 Свойство VisualType
