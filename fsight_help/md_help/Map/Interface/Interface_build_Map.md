# Интерфейсы сборки Map


В рамках сборки Map существуют
 следующие интерфейсы:


		 Интерфейс
		 Краткое описание


		 ![](../Interface_Image.gif)
		 [IMap](IMap/IMap.htm)
		 Интерфейс IMap содержит
		 свойства и методы для определения параметров карты.


		 ![](../Interface_Image.gif)
		 [IMapAreaVisual](IMapAreaVisual/IMapAreaVisual.htm)
		 Интерфейс IMapAreaVisual
		 содержит свойства и методы для работы с картографическим показателем
		 на слое карты.


		 ![](../Interface_Image.gif)
		 [IMapArrow](IMapArrow/IMapArrow.htm)
		 Интерфейс IMapArrow
		 содержит свойства для определения параметров стрелки между территориями.


		 ![](../Interface_Image.gif)
		 [IMapBackground](IMapBackground/IMapBackground.htm)
		 Интерфейс IMapBackground
		 используется для настройки фона легенды карты.


		 ![](../Interface_Image.gif)
		 [IMapBarVisual](IMapBarVisual/IMapBarVisual.htm)
		 Интерфейс IMapBarVisual
		 содержит свойства и методы для определения параметров столбикового
		 показателя.


		 ![](../Interface_Image.gif)
		 [IMapBox](IMapBox/IMapBox.htm)
		 Интерфейс IMapBox содержит
		 свойства и методы компонента среды разработки - [MapBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/MapBox.htm).


		 ![](../Interface_Image.gif)
		 [IMapBoxTerritoryEventArgs](IMapBoxTerritoryEventArgs/IMapBoxTerritoryEventArgs.htm)
		 Интерфейс IMapBoxTerritoryEventArgs
		 содержит свойства аргумента событий [OnTerritoryClick](../Class/MapBox/MapBox.OnTerritoryClick.htm)
		 компонента [MapBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/MapBox.htm),
		 возникающего при щелчке мышью по какому-либо региону карты.


		 ![](../Interface_Image.gif)
		 [IMapColorCollection](IMapColorCollection/IMapColorCollection.htm)
		 Интерфейс IMapColorCollection
		 определяет коллекцию цветов показателя.


		 ![](../Interface_Image.gif)
		 [IMapColorScale](IMapColorScale/IMapColorScale.htm)
		 Интерфейс IMapColorScale
		 определяет параметры цветовой шкалы для карты.


		 ![](../Interface_Image.gif)
		 [IMapDataAdapter](IMapDataAdapter/IMapDataAdapter.htm)
		 Интерфейс IMapDataAdapter
		 содержит свойства для работы с адаптером данных.


		 ![](../Interface_Image.gif)
		 [IMapDynamicDataAdapter](IMapDynamicDataAdapter/IMapDynamicDataAdapter.htm)
		 Интерфейс IMapDynamicDataAdapter
		 содержит свойства для работы с динамическим адаптером данных.


		 ![](../Interface_Image.gif)
		 [IMapFillScale](IMapFillScale/IMapFillScale.htm)
		 Интерфейс IMapFillScale
		 содержит свойства и методы для работы со штриховой шкалой.


		 ![](../Interface_Image.gif)
		 [IMapIconScale](IMapIconScale/IMapIconScale.htm)
		 Интерфейс IMapIconScale
		 содержит свойства и методы для работы с пиктографической шкалой.


		 ![](../Interface_Image.gif)
		 [IMapImageVisual](IMapImageVisual/IMapImageVisual.htm)
		 Интерфейс IMapImageVisual
		 содержит свойства и методы для работы с пиктографическим показателем
		 на слое карты.


		 ![](../Interface_Image.gif)
		 [IMapLayer](IMapLayer/IMapLayer.htm)
		 Интерфейс IMapLayer
		 содержит свойства для определения параметров слоя карты.


		 ![](../Interface_Image.gif)
		 [IMapLayerArrows](IMapLayerArrows/IMapLayerArrows.htm)
		 Интерфейс IMapLayerArrows
		 содержит свойства и методы для определения параметров коллекции
		 стрелок.


		 ![](../Interface_Image.gif)
		 [IMapLayers](IMapLayers/IMapLayers.htm)
		 Интерфейс IMapLayer
		 содержит свойства для определения параметров коллекции слоев карты.


		 ![](../Interface_Image.gif)
		 [IMapLayerVisuals](IMapLayerVisuals/IMapLayerVisuals.htm)
		 Интерфейс IMapLayerVisuals
		 содержит свойства и методы для работы с коллекцией показателей
		 слоя карты.


		 ![](../Interface_Image.gif)
		 [IMapLegend](IMapLegend/IMapLegend.htm)
		 Интерфейс IMapLegend
		 содержит свойства и методы для работы с легендой карты.


		 ![](../Interface_Image.gif)
		 [IMapLegendHeader](IMapLegendHeader/IMapLegendHeader.htm)
		 Интерфейс IMapLegendHeader
		 содержит свойства для работы с верхним и нижним текстом легенды.


		 ![](../Interface_Image.gif)
		 [IMapLegendLabelFormat](IMapLegendLabelFormat/IMapLegendLabelFormat.htm)
		 Интерфейс IMapLegendLabelFormat
		 содержит свойства для работы с параметрами формата подписей легенды.


		 ![](../Interface_Image.gif)
		 [IMapNumericScale](IMapNumericScale/IMapNumericScale.htm)
		 Интерфейс IMapNumericScale
		 содержит свойства и методы для работы с числовой шкалой.


		 ![](../Interface_Image.gif)
		 [IMapMarkerVisual](IMapMarkerVisual/IMapMarkerVisual.htm)
		 Интерфейс IMapMarkerVisual
		 предназначен для работы с маркерами геолокации.


		 ![](../Interface_Image.gif)
		 [IMapPieVisual](IMapPieVisual/IMapPieVisual.htm)
		 Интерфейс IMapPieVisual
		 содержит свойства и методы для работы с круговым показателем.


		 ![](../Interface_Image.gif)
		 [IMapScale](IMapScale/IMapScale.htm)
		 Интерфейс IMapScale
		 определяет параметры шкал отчета. Данный интерфейс является базовым
		 для ряда интерфейсов.


		 ![](../Interface_Image.gif)
		 [IMapSource](IMapSource/IMapSource.htm)
		 Интерфейс IMapSource
		 является промежуточным и используется для передачи данных из компонента
		 [UiMap](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiMap.htm),
		 либо [UiErAnalyzer](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiErAnalyzer.htm)
		 в компонент [MapBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/MapBox.htm).


		 ![](../Interface_Image.gif)
		 [IMapTerritoryInfo](IMapTerritoryInfo/IMapTerritoryInfo.htm)
		 Интерфейс IMapTerritoryInfo
		 содержит свойства и методы для работы с расшифровкой территорий.


		 ![](../Interface_Image.gif)
		 [IMapTimeDataAdapter](IMapTimeDataAdapter/IMapTimeDataAdapter.htm)
		 Интерфейс IMapTimeDataAdapter
		 предназначен для получения информации о показателях в различных
		 временных точках на слое карты.


		 ![](../Interface_Image.gif)
		 [IMapTimeDynamicDataAdapter](IMapTimeDynamicDataAdapter/IMapTimeDynamicDataAdapter.htm)
		 Интерфейс IMapTimeDynamicDataAdapter
		 предназначен для получения информации о динамических показателях в
		 различных временных точках на слое карты.


		 ![](../Interface_Image.gif)
		 [IMapTopobase](IMapTopobase/IMapTopobase.htm)
		 Интерфейс IMapTopobase
		 содержит свойства и методы для работы с топоосновой.


		 ![](../Interface_Image.gif)
		 [IMapTopobaseAttributeNames](IMapTopobaseAttributeNames/IMapTopobaseAttributeNames.htm)
		 Интерфейс IMapTopobaseAttributeNames
		 содержит свойства для определения параметров коллекции наименований
		 атрибутов слоя карты.


		 ![](../Interface_Image.gif)
		 [IMapTopobasePrimitive](IMapTopobasePrimitive/IMapTopobasePrimitive.htm)
		 Интерфейс IMapTopobasePrimitive
		 содержит свойства для определения параметров примитивов, из которых
		 состоит область слоя карты.


		 ![](../Interface_Image.gif)
		 [IMapTopobasePrimitives](IMapTopobasePrimitives/IMapTopobasePrimitives.htm)
		 Интерфейс IMapTopobasePrimitives
		 содержит свойства для определения параметров коллекции примитивов,
		 из которых состоит область слоя карты.


		 ![](../Interface_Image.gif)
		 [IMapTopobaseShape](IMapTopobaseShape/IMapTopobaseShape.htm)
		 Интерфейс IMapTopobaseShape
		 содержит свойства для определения параметров области слоя карты.


		 ![](../Interface_Image.gif)
		 [IMapTopobaseShapes](IMapTopobaseShapes/IMapTopobaseShapes.htm)
		 Интерфейс IMapTopobaseShapes
		 содержит свойства для определения параметров коллекции областей
		 слоя карты.


		 ![](../Interface_Image.gif)
		 [IMapView](IMapView/IMapView.htm)
		 Интерфейс IMapView
		 содержит свойства и методы, определяющие параметры отображения
		 карты и объектов, расположенных на карте.


		 ![](../Interface_Image.gif)
		 [IMapVisual](IMapVisual/IMapVisual.htm)
		 Интерфейс IMapVisual
		 содержит свойства и методы для работы с показателями на слое карты.


		 ![](../Interface_Image.gif)
		 [IMapVisualColorProperty](IMapVisualColorProperty/IMapVisualColorProperty.htm)
		 Интерфейс IMapVisualColorProperty
		 содержит свойства для работы с параметрами заливки показателя.


		 ![](../Interface_Image.gif)
		 [IMapVisualFillProperty](IMapVisualFillProperty/IMapVisualFillProperty.htm)
		 Интерфейс IMapVisualFillProperty
		 содержит свойства для работы с параметрами штриховки показателя.


		 ![](../Interface_Image.gif)
		 [IMapVisualIconProperty](IMapVisualIconProperty/IMapVisualIconProperty.htm)
		 Интерфейс IMapVisualIconProperty
		 содержит свойства для работы с параметрами пиктограммы показателя.


		 ![](../Interface_Image.gif)
		 [IMapVisualNumericProperty](IMapVisualNumericProperty/IMapVisualNumericProperty.htm)
		 Интерфейс IMapVisualNumericProperty
		 содержит свойства для работы с числовыми параметрами показателя.


		 ![](../Interface_Image.gif)
		 [IMapWidget](IMapWidget/IMapWidget.htm)
		 Интерфейс IMapWidget
		 используется для работы с компонентом MapWidget.


		 ![](../Interface_Image.gif)
		 [IUiMap](IUiMap/IUiMap.htm)
		 Интерфейс IUiMap содержит
		 свойства компонента среды разработки [UiMap](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiMap.htm),
		 являющегося источником данных для компонента [MapBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/MapBox.htm).
