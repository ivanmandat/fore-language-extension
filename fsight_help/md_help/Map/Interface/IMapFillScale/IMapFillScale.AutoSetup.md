# IMapFillScale.AutoSetup


## Синтаксис


AutoSetup(Layer: [IMapLayer](../IMapLayer/IMapLayer.htm);
 DataAdapter: Object; Count: Integer);


## Параметры


Layer. Слой, содержащий показатель.


DataAdapter. Источник данных,
 в соответствии с которым будет производиться штриховка.


Count. Количество делений шкалы.


## Описание


Метод AutoSetup осуществляет
 автоматическую настройку шкалы.
