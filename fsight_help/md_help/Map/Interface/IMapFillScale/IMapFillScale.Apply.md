# IMapFillScale.Apply


## Синтаксис


Apply(Value: Double): [GxHatchStyle](ModDrawing.chm::/Enums/GxHatchStyle.htm);


## Параметры


Value. Значение, к которому
 необходимо применить шкалу.


## Описание


Метод Apply применяет шкалу
 к указанному непустому значению и возвращает тип штриховки, соответствующий
 данному значению.
