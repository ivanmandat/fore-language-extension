# IMapFillScale.ApplyEx


## Синтаксис


ApplyEx(Value: Variant): [GxHatchStyle](ModDrawing.chm::/Enums/GxHatchStyle.htm);


## Параметры


Value. Значение, к которому
 необходимо применить шкалу.


## Описание


Метод ApplyEx применяет шкалу
 к указанному значению и возвращает тип штриховки, соответствующий данному
 значению.
