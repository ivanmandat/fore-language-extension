# IMapFillScale


## Описание


Интерфейс IMapFillScale содержит свойства и методы для работы со штриховой шкалой.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Exact](IMapFillScale.Exact.htm)
		 Свойство Exact определяет
		 тип штриховки для значений, которые равны значению деления шкалы.


		 ![](../../Property_Image.gif)
		 [Fill](IMapFillScale.Fill.htm)
		 Свойство Fill определяет
		 тип штриховки для заданного значения деления шкалы.


		 ![](../../Property_Image.gif)
		 [Greater](IMapFillScale.Greater.htm)
		 Свойство Greater определяет
		 тип штриховки для значений, которые больше значения деления шкалы.


		 ![](../../Property_Image.gif)
		 [Less](IMapFillScale.Less.htm)
		 Свойство Less определяет
		 тип штриховки для значений, которые меньше значения деления шкалы.


		 ![](../../Property_Image.gif)
		 [NoData](IMapFillScale.NoData.htm)
		 Свойство NoData определяет
		 тип штриховки для отсутствующих значений.


## Свойства, унаследованные от [IMapScale](../IMapScale/IMapScale.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AutoCalculable](../IMapScale/IMapScale.AutoCalculable.htm)
		 Свойство AutoCalculable
		 определяет признак автоматического расчета [значений](../IMapScale/IMapScale.Value.htm)
		 для шкалы отчета.


		 ![](../../Property_Image.gif)
		 [AutoScaleMode](../IMapScale/IMapScale.AutoScaleMode.htm)
		 Свойство AutoScaleMode
		 определяет тип автоматического расчета шкалы отчета.


		 ![](../../Property_Image.gif)
		 [Count](../IMapScale/IMapScale.Count.htm)
		 Свойство Count определяет
		 количество значений в шкале (разбиений).


		 ![](../../Property_Image.gif)
		 [IsDirty](../IMapScale/IMapScale.IsDirty.htm)
		 Свойство IsDirty возвращает
		 признак изменения шкалы отчета.


		 ![](../../Property_Image.gif)
		 [Value](../IMapScale/IMapScale.Value.htm)
		 Свойство Value определяет
		 значение для деления шкалы.
