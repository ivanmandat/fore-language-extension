# IMapTopobasePrimitives


## Описание


Интерфейс IMapTopobasePrimitives
 содержит свойства для определения параметров коллекции примитивов, из
 которых состоит область слоя карты.


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMapTopobasePrimitives.Count.htm)
		 Свойство Count возвращает
		 количество примитивов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMapTopobasePrimitives.Item.htm)
		 Свойство Item возвращает
		 примитив по заданному индексу.
