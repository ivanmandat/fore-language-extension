# IMapNumericScale


## Описание


Интерфейс IMapNumericScale содержит
 свойства и методы для работы с числовой шкалой.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Exact](IMapNumericScale.Exact.htm)
		 Свойство Exact определяет
		 число для значений, которые равны значению деления шкалы.


		 ![](../../Property_Image.gif)
		 [Greater](IMapNumericScale.Greater.htm)
		 Свойство Greater определяет
		 число для значений, которые больше значения деления шкалы.


		 ![](../../Property_Image.gif)
		 [Less](IMapNumericScale.Less.htm)
		 Свойство Less определяет
		 число для значений, которые меньше значения деления шкалы.


		 ![](../../Property_Image.gif)
		 [NoData](IMapNumericScale.NoData.htm)
		 Свойство NoData определяет
		 число для отсутствующих значений.


		 ![](../../Property_Image.gif)
		 [Number](IMapNumericScale.Number.htm)
		 Свойство Number определяет
		 число для заданного значения деления шкалы.


## Свойства, унаследованные от [IMapScale](../IMapScale/IMapScale.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AutoCalculable](../IMapScale/IMapScale.AutoCalculable.htm)
		 Свойство AutoCalculable
		 определяет признак автоматического расчета [значений](../IMapScale/IMapScale.Value.htm)
		 для шкалы отчета.


		 ![](../../Property_Image.gif)
		 [AutoScaleMode](../IMapScale/IMapScale.AutoScaleMode.htm)
		 Свойство AutoScaleMode
		 определяет тип автоматического расчета шкалы отчета.


		 ![](../../Property_Image.gif)
		 [Count](../IMapScale/IMapScale.Count.htm)
		 Свойство Count определяет
		 количество значений в шкале (разбиений).


		 ![](../../Property_Image.gif)
		 [IsDirty](../IMapScale/IMapScale.IsDirty.htm)
		 Свойство IsDirty возвращает
		 признак изменения шкалы отчета.


		 ![](../../Property_Image.gif)
		 [Value](../IMapScale/IMapScale.Value.htm)
		 Свойство Value определяет
		 значение для деления шкалы.
