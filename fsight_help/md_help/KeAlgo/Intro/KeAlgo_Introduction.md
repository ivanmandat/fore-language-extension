# Введение в сборку Algo


Сборка Algo предназначена для
 работы с расширением «[Алгоритмы расчёта](CalculationAlgorithm.chm::/Purpose.htm)» на языке Fore.


Подробное описание смотрите в разделах:


	- [общие принципы программирования с использованием
	 сборки Algo](KeAlgo_Programming.htm);


	- [иерархия
	 сборки Algo](KeAlgo_Hierarchy.htm).


В интерфейсах сборки реализованы все необходимые средства для работы
 с алгоритмом расчёта.


Основными интерфейсами сборки Algo
 являются:


	- [ICalcAlgorithm](../Interface/ICalcAlgorithm/ICalcAlgorithm.htm);


	- [ICalcBlock](../Interface/ICalcBlock/ICalcBlock.htm);


	- [ICalcAggr](../Interface/ICalcAggr/ICalcAggr.htm);


	- [ICalcFunc](../Interface/ICalcFunc/ICalcFunc.htm);


	- [ICalcLinearOptimizationBlock](../Interface/ICalcLinearOptimizationBlock/ICalcLinearOptimizationBlock.htm);


	- [ICalcValidationBlock](../Interface/ICalcValidationBlock/ICalcValidationBlock.htm).
