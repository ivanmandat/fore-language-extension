## Краткое описание


## С чего начать


		 Сборка Algo
		 предназначена для работы с расширением «[Алгоритмы
		 расчёта](CalculationAlgorithm.chm::/Purpose.htm)» на языке Fore.


		Изучив данный раздел, Вы сможете:


			- настраивать блоки алгоритма
			 расчёта;


			- добавлять и настраивать параметры
			 алгоритма расчёта;


			- настраивать порядок расчёта
			 алгоритма;


			- производить отладку расчёта
			 алгоритма;


			- запускать расчёт алгоритма.


		Более подробное описание возможностей блока приведено в разделе
		 «[Введение в сборку Algo](KeAlgo_Introduction.htm)».


		Описание ресурсов сборки представлено в следующих подразделах:


			- [интерфейсы
			 сборки Algo](../Interface/KeAlgo_Interface.htm);


			- [перечисления сборки
			 Algo](../Enums/KeAlgo_Enums.htm).


		 Перед изучением сборки Algo рекомендуется ознакомиться
		 с разделами: «[Среда разработки](UiDevEnv.chm::/01_Development_Environment/01_Purpose_of_the_constructor/Purpose_of_the_constructor.htm)»,
		 «[Руководство по языку Fore](Fore.chm::/Fore_Title.htm)».


		Для установки расширения обратитесь к разделу «[Установка
		 расширений продукта «Форсайт. Аналитическая платформа](Setup.chm::/Extensions/install_or_update_extensions.htm)».


		Для работы с расширением «Алгоритмы
		 расчёта» обратитесь к разделу «[Автоматизация методик
		 и алгоритмов](CalculationAlgorithm.chm::/Purpose.htm)».
