# CalcObjectType


## Описание


Перечисление CalcObjectType
 используется для определения типа объекта [алгоритма
 расчёта](CalculationAlgorithm.chm::/Desktop/Work/Construction_of_the_calculation_algorithm.htm).


Используется следующими свойствами:


	- [ICalcObject.Type](../Interface/ICalcObject/ICalcObject.Type.htm);


	- [IAlgorithmCalculationExecutor.HasObjectByType](../Interface/IAlgorithmCalculationExecutor/IAlgorithmCalculationExecutor.HasObjectByType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Algorithm. Готовый
		 алгоритм расчёта.


		 1
		 CalcBlock. Блок
		 расчёта.


		 2
		 Aggregation. Блок агрегации.


		 3
		 CalcFunction. Блок
		 функций.


		 4
		 LinearOptimization.
		 Блок линейной оптимизации.


		 5
		 Validation. Блок контроля.
