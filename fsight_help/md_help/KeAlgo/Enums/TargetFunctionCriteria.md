# TargetFunctionCriteria


## Описание


Перечисление TargetFunctionCriteria
 используется для определения [критерия
 целевой функции](CalculationAlgorithm.chm::/Desktop/Linear_optimization_block/TargetFunction_LinOpt.htm) в [блоке
 линейной оптимизации](CalculationAlgorithm.chm::/Desktop/Linear_optimization_block/Linear_optimization_block.htm).


Используется следующим свойством:


	- [ICalcLinearOptimizationBlock.FunctionCriteria](../Interface/ICalcLinearOptimizationBlock/ICalcLinearOptimizationBlock.FunctionCriteria.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Min. Значения управляющих
		 переменных будут подобраны таким образом, чтобы целевая функция
		 имела минимальное значение.


		 1
		 Max. Значения
		 управляющих переменных будут подобраны таким образом, чтобы целевая
		 функция имела максимальное значение.
