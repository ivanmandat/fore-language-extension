# IAlgoFindFormulaResults.AddBranch


## Синтаксис


AddBranch(pBranch: [IMsCalculationChainBranch](KeMs.chm::/Interface/IMsCalculationChainBranch/IMsCalculationChainBranch.htm);
 pCase: [IMsBranchCase](KeMs.chm::/Interface/IMsBranchCase/IMsBranchCase.htm);
 pCond: [IMsBranchCondition](KeMs.chm::/Interface/IMsBranchCondition/IMsBranchCondition.htm));


## Параметры


pBranch. Ветвления, в формулах
 которого найден терм.


pCase. Ветка ветвления, в формулах
 которого найден терм.


pCond. Условие, в котором используется
 найденный терм.


## Описание


Метод AddBranch добавляет в
 коллекцию ветку, использующую искомый терм.
