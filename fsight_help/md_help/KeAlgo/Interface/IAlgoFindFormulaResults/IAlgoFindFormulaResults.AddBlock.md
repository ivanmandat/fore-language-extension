# IAlgoFindFormulaResults.AddBlock


## Синтаксис


AddBlock(pObj: [ICalcObject](../ICalcObject/ICalcObject.htm);
 pModel: [IMsCalculationChainModel](KeMs.chm::/Interface/IMsCalculationChainModel/IMsCalculationChainModel.htm));


## Параметры


pObj. Объект расчёта, в формулах
 которого найден терм.


pModel.
 Модель в цепочке расчёта, в формулах которой найден терм.


## Описание


Метод AddBlock добавляет в коллекцию
 блок, использующий искомый терм.
