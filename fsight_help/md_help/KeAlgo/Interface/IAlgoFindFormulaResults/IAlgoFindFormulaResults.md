# IAlgoFindFormulaResults


Сборка: Algo;


## Описание


Интерфейс IAlgoFindFormulaResults
 предназначен для работы с коллекцией результатов поиска формул.


## Иерархия наследования


IAlgoFindFormulaResults


## Комментарии


Поиск формул осуществляется с помощью метода [ICalcAlgorithm.FindFormulas](../ICalcAlgorithm/ICalcAlgorithm.FindFormulas.htm).
