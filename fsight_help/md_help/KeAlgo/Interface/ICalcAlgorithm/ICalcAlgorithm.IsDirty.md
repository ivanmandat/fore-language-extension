# ICalcAlgorithm.IsDirty


## Синтаксис


IsDirty: Boolean;


## Описание


Свойство IsDirty возвращает
 признак наличия изменений в алгоритме расчёта.


## Комментарии


Допустимые значения:


	- True. Алгоритм расчёта
	 изменен;


	- False. Алгоритм расчёта
	 не изменен.
