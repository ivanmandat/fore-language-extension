# ICalcAlgorithm.Debug


## Синтаксис


Debug(Objs: [ICalcObjectsList](../ICalcObjectsList/ICalcObjectsList.htm)):
 [IAlgorithmCalculationDebug](../IAlgorithmCalculationDebug/IAlgorithmCalculationDebug.htm);


## Параметры


Objs. Список объектов алгоритма
 расчёта.


Примечание.
 Список объектов алгоритма расчёта должен быть скопированным. Для копирования
 объектов используйте метод [ICalcObjectsList.Clone](../ICalcObjectsList/ICalcObjectsList.Clone.htm).


## Описание


Метод Debug выполняет [отладку расчёта алгоритма](CalculationAlgorithm.chm::/Desktop/Work/Debug.htm).
