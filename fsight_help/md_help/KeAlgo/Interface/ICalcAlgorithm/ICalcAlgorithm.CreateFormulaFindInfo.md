# ICalcAlgorithm.CreateFormulaFindInfo


## Синтаксис


CreateFormulaFindInfo: [IMsFormulaInfo](KeMs.chm::/Interface/IMsFormulaInfo/IMsFormulaInfo.htm);


## Описание


Свойство CreateFormulaFindInfo
 возвращает объект, используемый для задания условий поиска формул.


## Комментарии


Поиск формул осуществляется с помощью метода [FindFormulas](ICalcAlgorithm.FindFormulas.htm).

Использование свойства приведено в примере для метода [ICalcAlgorithm.FindFormulas](ICalcAlgorithm.FindFormulas.htm).
