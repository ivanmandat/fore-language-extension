# ICalcAlgorithm.Calculate


## Синтаксис


Calculate([Objs: [ICalcObjectsList](../ICalcObjectsList/ICalcObjectsList.htm)
 = Null]): [IAlgorithmCalculationResult](../IAlgorithmCalculationResult/IAlgorithmCalculationResult.htm);


## Параметры


Objs. Список объектов алгоритма
 расчёта.


## Описание


Метод Calculate выполняет расчёт
 алгоритма.


## Комментарии


При расчёте учитываются значения параметров, которые заданы в свойстве
 [ICalcAlgorithm.ParamValues](ICalcAlgorithm.ParamValues.htm).
 Если параметр Objs не указан,
 то рассчитываются все объекты в алгоритме расчёта.
