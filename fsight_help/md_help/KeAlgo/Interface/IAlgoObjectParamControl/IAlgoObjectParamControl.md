# IAlgoObjectParamControl


Сборка: Algo;


## Описание


Интерфейс IAlgoObjectParamControl
 предназначен для настройки управления параметром объекта через параметр
 алгоритма расчёта.


## Иерархия наследования


           IAlgoObjectParamControl


## Комментарии


Настройки управления параметром можно получить, используя свойства и
 методы интерфейса [IAlgoObjectParamsControl](../IAlgoObjectParamsControl/IAlgoObjectParamsControl.htm).
