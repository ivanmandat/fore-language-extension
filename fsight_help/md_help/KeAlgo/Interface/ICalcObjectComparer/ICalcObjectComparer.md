# ICalcObjectComparer


Сборка: Algo;


## Описание


Интерфейс ICalcObjectComparer
 предназначен для сортировки объектов алгоритма расчёта.


## Иерархия наследования


           ICalcObjectComparer


## Комментарии


Методы данного интерфейса должны быть переопределены в пользовательском
 классе. Также пользовательский класс должен наследоваться и содержать
 реализацию методов интерфейса [IComparer](ModCollections.chm::/Interface/IComparer/IComparer.htm).
