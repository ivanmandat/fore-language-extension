# IAlgoFindFormulaResult


Сборка: Algo;


## Описание


Интерфейс IAlgoFindFormulaResult
 содержит базовые свойства интерфейсов с результатами поиска формул в алгоритме
 расчёта.


## Иерархия наследования


IAlgoFindFormulaResult


## Комментарии


Результат поиска можно получить, используя свойство [IAlgoFindFormulaResults.Item](../IAlgoFindFormulaResults/IAlgoFindFormulaResults.Item.htm).


Интерфейс IAlgoFindFormulaResult
 не имеет собственных свойств и методов. Является базовым для следующих
 интерфейсов:


	- [IAlgoBranchCondFindFormulaResult](../IAlgoBranchCondFindFormulaResult/IAlgoBranchCondFindFormulaResult.htm);


	- [IAlgoCalcBlockFindFormulaResult](../IAlgoCalcBlockFindFormulaResult/IAlgoCalcBlockFindFormulaResult.htm).


Для проверки, к какому из интерфейсов может быть приведён результат
 поиска, используйте операцию Is.
