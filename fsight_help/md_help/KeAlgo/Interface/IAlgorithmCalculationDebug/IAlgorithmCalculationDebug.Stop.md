# IAlgorithmCalculationDebug.Stop


## Синтаксис


Stop;


## Описание


Метод Stop останавливает отладку
 расчёта алгоритма.


## Комментарии


При выполнения метода доступен результат расчёта алгоритма. После работы
 с результатом расчёта сбросьте отладку с помощью метода [IAlgorithmCalculationDebug.Dispose_](IAlgorithmCalculationDebug.Dispose_.htm).
