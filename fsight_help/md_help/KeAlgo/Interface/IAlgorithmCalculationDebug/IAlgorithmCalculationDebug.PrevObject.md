# IAlgorithmCalculationDebug.PrevObject


## Синтаксис


PrevObject: [ICalcObject](../ICalcObject/ICalcObject.htm);


## Описание


Свойство PrevObject возвращает
 предыдущий рассчитанный объект алгоритма расчёта для повторного расчёта.


## Комментарии


Для повторного расчёта предыдущего рассчитанного объекта алгоритма расчёта
 используйте метод [IAlgorithmCalculationDebug.RecalcPrev](IAlgorithmCalculationDebug.RecalcPrev.htm).
