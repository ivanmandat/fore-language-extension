# IAlgorithmCalculationDebug.IsFinished


## Синтаксис


IsFinished: Boolean;


## Описание


Свойство IsFinished возвращает
 признак завершения расчёта алгоритма.


## Комментарии


Допустимые значения:


	- True. Расчёт алгоритма
	 завершен;


	- False. Расчёт алгоритма
	 не завершен.
