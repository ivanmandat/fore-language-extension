# IAlgorithmCalculationDebug.Dispose_


## Синтаксис


Dispose_;


## Описание


Метод Dispose_ завершает отладку
 расчёта алгоритма.


## Комментарии


При выполнения метода результат расчёта алгоритма недоступен. Метод
 используется для сброса отладки после остановки расчёта алгоритма с помощью
 метода [IAlgorithmCalculationDebug.Stop](IAlgorithmCalculationDebug.Stop.htm).
