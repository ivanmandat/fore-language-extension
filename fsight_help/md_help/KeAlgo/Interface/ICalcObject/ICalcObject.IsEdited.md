# ICalcObject.IsEdited


## Синтаксис


IsEdited: Boolean;


## Описание


Свойство IsEdited возвращает
 признак редактирования объекта алгоритма расчёта.


## Комментарии


Допустимые значения:


	- True. Редактирование
	 объекта алгоритма расчёта выполняется;


	- False. Редактирование
	 объекта алгоритма расчёта не выполняется.
