# IAlgorithmParamValue


Сборка: Algo;


## Описание


Интерфейс IAlgorithmParamValue
 предназначен для получения настроек конкретного [параметра
 алгоритма расчёта](CalculationAlgorithm.chm::/Desktop/Work/Parameters_of_the_calculation_algorithm.htm).


## Иерархия наследования


           IAlgorithmParamValue


## Комментарии


Для определения настроек конкретного параметра алгоритма расчёта используйте
 интерфейс [IMsModelParam](kems.chm::/interface/imsmodelparam/imsmodelparam.htm).
