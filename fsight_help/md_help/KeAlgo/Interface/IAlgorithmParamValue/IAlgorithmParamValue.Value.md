# IAlgorithmParamValue.Value


## Синтаксис


Value: Variant;


## Описание


Свойство Value определяет значение
 параметра по умолчанию.


## Комментарии


Указываемое значение должно иметь тот тип, который [определён](IAlgorithmParamValue.ParamType.htm)
 для параметра.
