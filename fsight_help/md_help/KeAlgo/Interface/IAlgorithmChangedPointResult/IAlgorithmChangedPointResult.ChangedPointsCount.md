# IAlgorithmChangedPointResult.ChangedPointsCount


## Синтаксис


ChangedPointsCount: Integer;


## Описание


Свойство ChangedPointsCount
 возвращает количество ячеек в приёмнике данных, изменённых при выполнении
 расчёта блока расчёта.
