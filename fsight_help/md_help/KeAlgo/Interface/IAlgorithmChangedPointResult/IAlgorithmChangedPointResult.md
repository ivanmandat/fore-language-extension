# IAlgorithmChangedPointResult


Сборка: Algo;


## Описание


Интерфейс IAlgorithmChangedPointResult
 предназначен для определения [количества
 изменённых ячеек в приёмнике данных](CalculationAlgorithm.chm::/Desktop/Work/Perform_calculations.htm#change_points_count) при выполнении
 [расчёта](CalculationAlgorithm.chm::/Desktop/Work/Perform_calculations.htm)
 конкретного [блока
 расчёта](CalculationAlgorithm.chm::/Desktop/Calculation_block/Calculation_unit.htm).
