# IAlgorithmParamValueList


Сборка: Algo;


## Описание


Интерфейс IAlgorithmParamValueList
 предназначен для работы с коллекцией [параметров
 алгоритма расчёта](CalculationAlgorithm.chm::/Desktop/Work/Set_up_calculation_parameters.htm).


## Иерархия наследования


           IAlgorithmParamValueList


## Комментарии


Для формирования коллекции параметров алгоритма расчёта используйте
 интерфейс [IMsModelParams](kems.chm::/interface/imsmodelparams/imsmodelparams.htm).
