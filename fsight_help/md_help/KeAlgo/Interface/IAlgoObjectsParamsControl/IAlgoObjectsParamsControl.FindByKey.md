# IAlgoObjectsParamsControl.FindByKey


## Синтаксис


FindByKey(ObjKey: Integer): [IAlgoObjectParamsControl](../IAlgoObjectParamsControl/IAlgoObjectParamsControl.htm);


## Параметры


ObjKey. Ключ объекта репозитория.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает настройки управления параметрами объекта.


## Комментарии


Метод возвращает значение Null,
 если объект с указанным ключом отсутствует в алгоритме расчёта.
