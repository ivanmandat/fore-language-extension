# IAlgoObjectsParamsControl


Сборка: Algo;


## Описание


Интерфейс IAlgoObjectsParamsControl
 предназначен для работы с коллекцией настроек управления параметрами объектов
 репозитория.


## Иерархия наследования


IAlgoObjectsParamsControl


## Комментарии


Настройки управления параметрами объектов репозитория можно получить,
 используя свойство [ICalcAlgorithm.ObjectParamsControl](../ICalcAlgorithm/ICalcAlgorithm.ObjectParamsControl.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IAlgoObjectsParamsControl.Count.htm)
		 Свойство Count возвращает
		 количество объектов репозитория, для которых доступна настройка
		 управления параметрами.


		 ![](../../Property_Image.gif)
		 [Item](IAlgoObjectsParamsControl.Item.htm)
		 Свойство Item возвращает
		 настройки управления параметрами объекта по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IAlgoObjectsParamsControl.Add.htm)
		 Метод Add добавляет
		 в коллекцию настройки управления параметрами для указанного объекта
		 репозитория.


		 ![](../../Sub_Image.gif)
		 [Clear](IAlgoObjectsParamsControl.Clear.htm)
		 Метод Clear очищает
		 коллекцию настроек управления параметрами объектов.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IAlgoObjectsParamsControl.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск и возвращает настройки управления параметрами объекта.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IAlgoObjectsParamsControl.RemoveByKey.htm)
		 Метод RemoveByKey удаляет
		 из коллекции настройки управления параметрами для объекта с указанным
		 ключом.
