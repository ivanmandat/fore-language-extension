# IAlgoObjectsParamsControl.RemoveByKey


## Синтаксис


RemoveByKey(ObjKey: Integer);


## Параметры


ObjKey. Ключ объекта репозитория.


## Описание


Метод RemoveByKey удаляет из
 коллекции настройки управления параметрами для объекта с указанным ключом.
