# IAlgoObjectsParamsControl.Add


## Синтаксис


Add(Desc: [IMetabaseObjectDescriptor](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm)):
 [IAlgoObjectParamsControl](../IAlgoObjectParamsControl/IAlgoObjectParamsControl.htm);


## Параметры


Desc. Описание объекта репозитория,
 для которого будет настраиваться управление параметрами.


## Описание


Метод Add добавляет в коллекцию
 настройки управления параметрами для указанного объекта репозитория.
