# ICalcBlockIteratorStubDimensionLinks


Сборка: Algo;


## Описание


Интерфейс ICalcBlockIteratorStubDimensionLinks
 предназначен для работы с коллекцией [связей](CalculationAlgorithm.chm::/Desktop/Calculation_block/Page_Sources.htm#link)
 измерений источника с измерениями приёмника данных.
