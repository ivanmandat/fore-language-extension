# IAlgorithmCalculationResult.Errors


## Синтаксис


Errors: [IStringList](ModCollections.chm::/Interface/IStringList/IStringList.htm);


## Описание


Свойство Errors возвращает список
 сообщений об ошибках или предупреждениях.


## Комментарии


Список сообщений об ошибках задается с помощью метода [IMsProblemCalculationCallback.OnError](kems.chm::/interface/imsproblemcalculationcallback/imsproblemcalculationcallback.onerror.htm),
 а список сообщений о предупреждениях - с помощью метода [IMsProblemCalculationCallback.OnWarning](kems.chm::/interface/imsproblemcalculationcallback/imsproblemcalculationcallback.onwarning.htm).
