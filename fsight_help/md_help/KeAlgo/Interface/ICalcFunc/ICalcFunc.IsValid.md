# ICalcFunc.IsValid


## Синтаксис


IsValid: Boolean;


## Описание


Свойство IsValid возвращает
 признак корректности параметров функции.


## Комментарии


Допустимые значения:


	- True. Параметры функции
	 корректны;


	- False. Параметры функции
	 не корректны.
