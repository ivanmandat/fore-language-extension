# IAlgorithmChangedPointResults


Сборка: Algo;


## Описание


Интерфейс IAlgorithmChangedPointResults
 предназначен для работы с коллекцией результатов выполнения расчёта [блоков
 расчёта](CalculationAlgorithm.chm::/Desktop/Calculation_block/Calculation_unit.htm), содержащих информацию о [количестве
 изменённых ячеек в приёмнике данных](CalculationAlgorithm.chm::/Desktop/Work/Perform_calculations.htm#change_points_count).
