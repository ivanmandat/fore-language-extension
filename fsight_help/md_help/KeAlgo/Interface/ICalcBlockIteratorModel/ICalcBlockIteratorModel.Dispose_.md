# ICalcBlockIteratorModel.Dispose_


## Синтаксис


Dispose_;


## Описание


Метод Dispose_ вызывается после
 использования объекта для освобождения ресурсов.


## Комментарии


Метод Dispose_ вызывается после
 [сохранения](ICalcBlockIteratorModel.Save.htm) всех изменений,
 освобождает все используемые ресурсы и позволяет избежать утечек памяти.
