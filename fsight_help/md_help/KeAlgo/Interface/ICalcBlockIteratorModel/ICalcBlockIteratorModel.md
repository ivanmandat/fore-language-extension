# ICalcBlockIteratorModel


Сборка: Algo;


## Описание


Интерфейс ICalcBlockIteratorModel
 предназначен для проверки изменений [источников
 данных](CalculationAlgorithm.chm::/Desktop/Calculation_block/Page_Sources.htm) и сохранения изменений в блоке расчёта или блоке
 контроля.
