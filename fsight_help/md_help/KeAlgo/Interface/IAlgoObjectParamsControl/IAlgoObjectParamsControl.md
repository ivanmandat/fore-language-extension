# IAlgoObjectParamsControl


Сборка: Algo;


## Описание


Интерфейс IAlgoObjectParamsControl
 предназначен для работы с коллекцией настроек управления параметрами объекта
 репозитория.


## Иерархия наследования


IAlgoObjectParamsControl


## Комментарии


Настройки управления параметрами объекта репозитория можно получить,
 используя свойства и методы интерфейса [IAlgoObjectsParamsControl](../IAlgoObjectsParamsControl/IAlgoObjectsParamsControl.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IAlgoObjectParamsControl.Count.htm)
		 Свойство Count возвращает
		 количество настроек управления параметрами объекта репозитория.


		 ![](../../Property_Image.gif)
		 [Item](IAlgoObjectParamsControl.Item.htm)
		 Свойство Item возвращает
		 настройки управления параметром с указанным индексом.


		 ![](../../Property_Image.gif)
		 [ObjectDesc](IAlgoObjectParamsControl.ObjectDesc.htm)
		 Свойство ObjectDesc
		 возвращает описание объекта репозитория, для которого настраивается
		 управление параметрами.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IAlgoObjectParamsControl.Add.htm)
		 Метод Add для указанного
		 параметра объекта добавляет настройки управления через параметр
		 алгоритма расчёта.


		 ![](../../Sub_Image.gif)
		 [Clear](IAlgoObjectParamsControl.Clear.htm)
		 Метод Clear очищает
		 коллекцию настроек управления параметрами объекта репозитория.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IAlgoObjectParamsControl.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск и возвращает настройки управления параметром объекта репозитория.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IAlgoObjectParamsControl.RemoveByKey.htm)
		 Метод RemoveByKey для
		 указанного параметра удаляет настройки управления.
