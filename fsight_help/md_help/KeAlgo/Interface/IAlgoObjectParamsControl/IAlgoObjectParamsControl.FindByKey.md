# IAlgoObjectParamsControl.FindByKey


## Синтаксис


FindByKey(ObjParamKey: Integer): [IAlgoObjectParamControl](../IAlgoObjectParamControl/IAlgoObjectParamControl.htm);


## Параметры


ObjParamKey. Ключ параметра
 объекта репозитория.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает настройки управления параметром объекта репозитория.


## Комментарии


Метод возвращает значение Null,
 если для указанного параметра отсутствуют настройки управления.
