# IAlgorithmTimeResults


Сборка: Algo;


## Описание


Интерфейс IAlgorithmTimeResults
 предназначен для работы с коллекцией результатов выполнения [расчёта
 алгоритма](CalculationAlgorithm.chm::/Desktop/Work/Perform_calculations.htm), содержащих информацию о времени вычисления
 объектов расчёта.
