# ICalcObjectsList


Сборка: Algo;


## Описание


Интерфейс ICalcObjectsList предназначен
 для формирования списка объектов алгоритма расчёта.


## Иерархия наследования


           ICalcObjectsList

		 Имя свойства
		 Краткое описание

		 [Count](ICalcObjectsList.Count.htm)

		 ![](../../Property_Image.gif)
		 [Item](ICalcObjectsList.Item.htm)
