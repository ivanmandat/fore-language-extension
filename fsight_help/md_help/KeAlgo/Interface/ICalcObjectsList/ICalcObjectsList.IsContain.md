# ICalcObjectsList.IsContain


## Синтаксис


IsContain(CalcObject: [ICalcObject](../ICalcObject/ICalcObject.htm)):
 Boolean;


## Параметры


CalcObject.
 Объект алгоритма расчёта.


## Описание


Метод IsContain возвращает признак
 наличия указанного объекта в коллекции.


## Комментарии


Допустимые значения:


	- True. Объект содержится
	 в коллекции;


	- False. Объект не содержится
	 в коллекции.
