# IAlgorithmParameterValues


Сборка: Algo;


## Описание


Интерфейс IAlgorithmParameterValues
 предназначен для настройки начала и окончания периода расчёта алгоритма,
 получения коллекции [параметров
 алгоритма расчёта](CalculationAlgorithm.chm::/Desktop/Work/Set_up_calculation_parameters.htm).
