# IAlgorithmSettings.IsDirty


## Синтаксис


IsDirty: Boolean;


## Описание


Свойство IsDirty возвращает
 признак наличия изменений в настройках алгоритма расчёта.


## Комментарии


Допустимые значения:


	- True. Настройки алгоритма
	 расчёта изменены;


	- False. Настройки алгоритма
	 расчёта не изменены.
