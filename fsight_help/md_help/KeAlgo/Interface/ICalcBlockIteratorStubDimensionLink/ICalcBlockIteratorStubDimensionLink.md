# ICalcBlockIteratorStubDimensionLink


Сборка: Algo;


## Описание


Интерфейс ICalcBlockIteratorStubDimensionLink
 предназначен для работы с конкретной [связью](CalculationAlgorithm.chm::/Desktop/Calculation_block/Page_Sources.htm#link)
 измерения источника с измерением приёмника данных.
