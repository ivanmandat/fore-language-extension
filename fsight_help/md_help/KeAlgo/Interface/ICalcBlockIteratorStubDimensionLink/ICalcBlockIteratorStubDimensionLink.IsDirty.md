# ICalcBlockIteratorStubDimensionLink.IsDirty


## Синтаксис


IsDirty: Boolean;


## Описание


Свойство IsDirty возвращает
 признак изменений в связи измерения источника с измерением приёмника данных.


## Комментарии


Допустимые значения:


	- True. Связь измерения
	 источника с измерением приёмника данных изменена;


	- False. Связь измерения
	 источника с измерением приёмника данных не изменена.
