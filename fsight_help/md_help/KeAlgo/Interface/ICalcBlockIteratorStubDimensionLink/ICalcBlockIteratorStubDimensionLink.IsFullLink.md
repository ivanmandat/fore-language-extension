# ICalcBlockIteratorStubDimensionLink.IsFullLink


## Синтаксис


IsFullLink: Boolean;


## Описание


Свойство IsFullLink возвращает
 признак заполнения всех обязательных параметров связи измерения источника
 с измерением приёмника данных.


## Комментарии


К обязательным параметрам связи относятся атрибуты измерений источника
 и приёмника данных, заданные с помощью свойств [ICalcBlockIteratorStubDimensionLink.DestinationAttribute](ICalcBlockIteratorStubDimensionLink.DestinationAttribute.htm)
 и [ICalcBlockIteratorStubDimensionLink.SourceAttribute](ICalcBlockIteratorStubDimensionLink.SourceAttribute.htm).


Допустимые значения:


	- True. Обязательные параметры
	 связи измерения источника с измерением приёмника данных заполнены;


	- False. Обязательные
	 параметры связи измерения источника с измерением приёмника данных
	 не заполнены.
