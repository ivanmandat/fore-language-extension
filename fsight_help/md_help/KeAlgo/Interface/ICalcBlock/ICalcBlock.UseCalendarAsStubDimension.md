# ICalcBlock.UseCalendarAsStubDimension


## Синтаксис


UseCalendarAsStubDimension(Stub: [IVariableStub](KeCubes.chm::/Interface/IVariableStub/IVariableStub.htm)):
 Boolean;


## Параметры


Stub. Календарное измерение.


## Описание


Свойство UseCalendarAsStubDimension
 определяет признак использования календарного измерения в качестве обычного
 измерения.
