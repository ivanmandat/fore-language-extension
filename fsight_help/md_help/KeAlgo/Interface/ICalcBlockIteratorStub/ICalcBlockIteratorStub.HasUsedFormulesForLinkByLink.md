# ICalcBlockIteratorStub.HasUsedFormulesForLinkByLink


## Синтаксис


HasUsedFormulesForLinkByLink: Boolean;


## Описание


Метод HasUsedFormulesForLinkByLink
 возвращает признак использования связей измерений источника с измерениями
 приёмника данных по связанным атрибутам в формулах.


## Комментарии


Допустимые значения:


	- True. В формулах используются
	 связи измерений источника с измерениями приёмника данных по связанным
	 атрибутам;


	- False. В формулах не
	 используются связи измерений источника с измерениями приёмника данных
	 по связанным атрибутам.


Для связи измерений источника с измерениями приёмника данных по связанным
 атрибутам используйте метод [ICalcBlockIteratorStub.LinkByLink](ICalcBlockIteratorStub.LinkByLink.htm).
