# ICalcBlockIteratorStub.LinkById


## Синтаксис


LinkById;


## Описание


Метод LinkByLink связывает измерений
 источника с измерениями приёмника данных по идентификаторам атрибутов.


## Комментарии


Для проверки использования связей измерений источника с измерениями
 приёмника данных по идентификаторам атрибутов в формулах используйте метод
 [ICalcBlockIteratorStub.HasUsedFormulesForLinkById](ICalcBlockIteratorStub.HasUsedFormulesForLinkById.htm).
