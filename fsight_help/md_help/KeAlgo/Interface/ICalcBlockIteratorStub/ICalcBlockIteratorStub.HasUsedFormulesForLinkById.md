# ICalcBlockIteratorStub.HasUsedFormulesForLinkById


## Синтаксис


HasUsedFormulesForLinkById: Boolean;


## Описание


Метод HasUsedFormulesForLinkById
 возвращает признак использования связей измерений источника с измерениями
 приёмника данных по идентификаторам атрибутов в формулах.


## Комментарии


Допустимые значения:


	- True. В формулах используются
	 связи измерений источника с измерениями приёмника данных по идентификаторам
	 атрибутов;


	- False. В формулах не
	 используются связи измерений источника с измерениями приёмника данных
	 по идентификаторам атрибутов.


Для связи измерений источника с измерениями приёмника данных по идентификаторам
 атрибутов используйте метод [ICalcBlockIteratorStub.LinkById](ICalcBlockIteratorStub.LinkById.htm).
