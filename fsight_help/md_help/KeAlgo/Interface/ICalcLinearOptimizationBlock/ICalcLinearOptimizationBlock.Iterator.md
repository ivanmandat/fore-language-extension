# ICalcLinearOptimizationBlock.Iterator


## Синтаксис


Iterator: [IMsCalculationChainMultiDimIterator](KeMs.chm::/Interface/IMsCalculationChainMultiDimIterator/IMsCalculationChainMultiDimIterator.htm);


## Описание


Свойство Iterator возвращает
 многомерный итератор блока линейной оптимизации.


## Комментарии


С помощью многомерного итератора доступна работа с измерениям приёмника
 данных, например, настройка расширенного фильтра. Если в многомерном итераторе
 добавлено измерение, которое используется в целевой функции, то оно будет
 удалено из целевой функции и наоборот.
