# IAlgorithmPeriodControl.OffsetValue


## Синтаксис


OffsetValue: Integer;


## Описание


Свойство OffsetValue определяет
 количество точек во временном периоде, на которое дата начала или окончания
 расчёта будет смещена вперёд или назад относительно значения параметра.
