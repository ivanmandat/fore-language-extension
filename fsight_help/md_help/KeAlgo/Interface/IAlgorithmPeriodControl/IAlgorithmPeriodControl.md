# IAlgorithmPeriodControl


Сборка: Algo;


## Описание


Интерфейс IAlgorithmPeriodControl
 предназначен для [управления
 периодом расчёта алгоритма](CalculationAlgorithm.chm::/Desktop/Work/Calculation_period_management.htm).


## Иерархия наследования


           IAlgorithmPeriodControl


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attribute](IAlgorithmPeriodControl.Attribute.htm)
		 Свойство Attribute
		 определяет атрибут, значение которого будет использоваться в качестве
		 даты начала или окончания расчёта.


		 ![](../../Property_Image.gif)
		 [OffsetLevel](IAlgorithmPeriodControl.OffsetLevel.htm)
		 Свойство OffsetLevel
		 определяет календарную динамику периода расчёта.


		 ![](../../Property_Image.gif)
		 [OffsetValue](IAlgorithmPeriodControl.OffsetValue.htm)
		 Свойство OffsetValue
		 определяет количество точек во временном периоде, на которое дата
		 начала или окончания расчёта будет смещена вперёд или назад относительно
		 значения параметра.


		 ![](../../Property_Image.gif)
		 [Parameter](IAlgorithmPeriodControl.Parameter.htm)
		 Свойство Parameter
		 определяет параметр алгоритма расчёта, который будет управлять
		 периодом расчёта.


		 ![](../../Property_Image.gif)
		 [Visible](IAlgorithmPeriodControl.Visible.htm)
		 Свойство Visible определяет
		 признак отображения даты начала или окончания расчёта на [панели
		 параметров](CalculationAlgorithm.chm::/Desktop/Work/Beginning_of_work.htm).


## Методы


		 Имя метода

		 ![](../../Sub_Image.gif)

		 [Reset](IAlgorithmPeriodControl.Reset.htm)
