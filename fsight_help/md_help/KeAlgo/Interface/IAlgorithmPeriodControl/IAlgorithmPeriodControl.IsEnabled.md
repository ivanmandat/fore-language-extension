# IAlgorithmPeriodControl.IsEnabled


## Синтаксис


IsEnabled: Boolean;


## Описание


Метод IsEnabled возвращает признак
 использования настроенной даты начала или окончания расчёта.


## Комментарии


Допустимые значения:


	- True. Настроенная дата
	 начала или окончания расчёта используется;


	- False. Настроенная дата
	 начала или окончания расчёта не используется.


При изменении параметров даты начала или окончания расчёта метод автоматически
 принимает значение True.
