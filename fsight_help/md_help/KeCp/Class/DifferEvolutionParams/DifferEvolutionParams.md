# DifferEvolutionParams


Сборка: Cp;


## Описание


Класс DifferEvolutionParams
 предназначен для настройки параметров метода дифференциальной эволюции.


## Комментарии


Для задания параметров метода используйте свойство [ICpNonLinearOptimization.DifferEvolutionParams](../../Interface/ICpNonLinearOptimization/ICpNonLinearOptimization.DifferEvolutionParams.htm).
