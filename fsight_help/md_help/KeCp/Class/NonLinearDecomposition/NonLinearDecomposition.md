# NonLinearDecomposition


## Описание


Класс NonLinearDecomposition содержит свойства и методы для расчета нелинейной системы уравнений.

		 ![](../../Property_Image.gif)
		 [MaxIteration](../../Interface/ICpNonLinearDecomposition/ICpNonLinearDecomposition.MaxIteration.htm)
