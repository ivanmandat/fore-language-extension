# Введение в сборку Cp


Основными интерфейсами сборки Cp
 являются:


	- [ICpLinearDecomposition](Interface/ICpLinearDecomposition/ICpLinearDecomposition.htm).
	 Решение системы линейных уравнений;


	- [ICpLinearOptimization](Interface/ICpLinearOptimization/ICpLinearOptimization.htm).
	 Линейная оптимизация;


	- [ICpNonLinearDecomposition](Interface/ICpNonLinearDecomposition/ICpNonLinearDecomposition.htm).
	 Решение системы нелинейных уравнений;


	- [ICpNonLinearOptimization](Interface/ICpNonLinearOptimization/ICpNonLinearOptimization.htm).
	 Нелинейная оптимизация.


Интерфейсы необходимые для внесения исходных данных, расчета и просмотра
 результатов:


	- [ICpMethod](Interface/ICpMethod/ICpMethod.htm).
	 Содержит свойства и методы методов оптимизации;


	- [ICpResults](Interface/ICpResults/ICpResults.htm).
	 Содержит свойства для просмотра результатов расчета;


	- [ILoAddend](Interface/ILoAddend/ILoAddend.htm).
	 Содержит свойства слагаемого уравнения линейной оптимизации;


	- [ILoConstraint](Interface/ILoConstraint/ILoConstraint.htm).
	 Содержит свойства для работы с ограничением линейной оптимизации;


	- [ILoCriterionFunction](Interface/ILoCriterionFunction/ILoCriterionFunction.htm).
	 Содержит свойства для работы с целевой функцией линейной оптимизации;


	- [ILoEquation](Interface/ILoEquation/ILoEquation.htm).
	 Содержит свойства уравнения линейной оптимизации;


	- [ILoResults](Interface/ILoResults/ILoResults.htm).
	 Содержит свойства для работы с результатами расчета метода линейной
	 оптимизации;


	- [ILoVariable](Interface/ILoVariable/ILoVariable.htm).
	 Содержит свойства переменной линейной оптимизации;


	- [INonLoConstraint](Interface/INonLoConstraint/INonLoConstraint.htm).
	 Содержит свойства для работы с ограничением нелинейной оптимизации;


	- [INonLoResults](Interface/INonLoResults/INonLoResults.htm).
	 Содержит свойства для работы с результатами расчета метода нелинейной
	 оптимизации;


	- [INonLoVariable](Interface/INonLoVariable/INonLoVariable.htm).
	 Содержит свойства переменной нелинейной оптимизации;


	- [IVarConstraint](Interface/IVarConstraint/IVarConstraint.htm).
	 Содержит свойства для работы с ограничением переменной нелинейной
	 оптимизации.
