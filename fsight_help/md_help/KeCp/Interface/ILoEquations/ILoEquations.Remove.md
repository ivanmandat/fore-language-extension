# ILoEquations.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс уравнения в коллекции.


## Описание


Метод Remove осуществляет удаление
 уравнения из коллекции по его индексу.


## Комментарии


Метод возвращает True в случае
 удачного удаления.
