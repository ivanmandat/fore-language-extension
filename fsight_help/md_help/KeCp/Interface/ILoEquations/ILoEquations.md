# ILoEquations


## Описание


Интерфейс ILoEquations содержит
 свойства и методы коллекции уравнений линейной оптимизации.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ILoEquations.Count.htm)
		 Свойство Count возвращает
		 количество уравнений в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ILoEquations.Item.htm)
		 Свойство Item возвращает
		 уравнение из коллекции по его индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ILoEquations.Add.htm)
		 Метод Add осуществляет
		 добавление уравнения в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](ILoEquations.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех уравнений из коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](ILoEquations.Remove.htm)
		 Метод Remove осуществляет
		 удаление уравнения из коллекции по его индексу.
