# ILoResults.VarValues


## Синтаксис


VarValues(VarId: String): Array;


## Параметры


VarId.
 Идентификатор переменной.


## Описание


Свойство VarValues возвращает
 вычисленные значения переменной в виде массива.


## Комментарии


Свойство доступно только для чтения.
