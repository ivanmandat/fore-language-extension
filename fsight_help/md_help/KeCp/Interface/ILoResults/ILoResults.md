# ILoResults


## Описание


Интерфейс ILoResults содержит
 свойства для работы с результатами расчета метода линейной оптимизации.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [OptimalValue](ILoResults.OptimalValue.htm)
		 Свойство OptimalValue
		 возвращает оптимальное значение целевой функции.


		 ![](../../Property_Image.gif)
		 [VarValues](ILoResults.VarValues.htm)
		 Свойство VarValues
		 возвращает вычисленные значения переменной в виде массива.


## Свойства, унаследованные от [ICpResults](../ICpResults/ICpResults.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ErrorMsg](../ICpResults/ICpResults.ErrorMsg.htm)
		 Свойство ErrorMsg возвращает
		 информационное сообщение о результатах расчета.


		 ![](../../Property_Image.gif)
		 [Status](../ICpResults/ICpResults.Status.htm)
		 Свойство Status возвращает
		 код ошибки.
