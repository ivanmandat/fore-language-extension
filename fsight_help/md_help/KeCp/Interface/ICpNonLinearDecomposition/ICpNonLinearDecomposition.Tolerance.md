# ICpNonLinearDecomposition.Tolerance


## Синтаксис


Tolerance: Double;


## Описание


Свойство Tolerance определяет
 точность решения.


## Комментарии


Для определения максимального числа итераций, за которое должен осуществиться
 поиск решения, используйте свойство [ICpNonLinearDecomposition.MaxIteration](ICpNonLinearDecomposition.MaxIteration.htm).
