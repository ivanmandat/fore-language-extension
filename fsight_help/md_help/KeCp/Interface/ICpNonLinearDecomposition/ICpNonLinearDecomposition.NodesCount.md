# ICpNonLinearDecomposition.NodesCount


## Синтаксис


NodesCount: Integer;


## Описание


Свойство NodesCount определяет
 количество интервалов, на которые будет разбиваться диапазон, заданный
 для управляющих переменных.


## Комментарии


Для определения максимального числа итераций, за которое должен осуществиться
 поиск решения, используйте свойство [ICpNonLinearDecomposition.MaxIteration](ICpNonLinearDecomposition.MaxIteration.htm).
