# ICpNonLinearDecomposition


## Описание


Интерфейс ICpNonLinearDecomposition
 содержит свойства и методы для расчета нелинейной системы уравнений.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Equations](ICpNonLinearDecomposition.Equations.htm)
		 Свойство Equations
		 определяет уравнения системы.


		 ![](../../Property_Image.gif)
		 [MaxIteration](ICpNonLinearDecomposition.MaxIteration.htm)
		 Свойство MaxIteration
		 определяет максимальное число итераций, за которое должен осуществиться
		 поиск решения.


		 ![](../../Property_Image.gif)
		 [MethodType](ICpNonLinearDecomposition.MethodType.htm)
		 Свойство MethodType
		 определяет метод поиска решения.


		 ![](../../Property_Image.gif)
		 [NodesCount](ICpNonLinearDecomposition.NodesCount.htm)
		 Свойство NodesCount
		 определяет количество интервалов, на которые будет разбиваться
		 диапазон, заданный для управляющих переменных.


		 ![](../../Property_Image.gif)
		 [Tolerance](ICpNonLinearDecomposition.Tolerance.htm)
		 Свойство Tolerance
		 определяет точность решения.


		 ![](../../Property_Image.gif)
		 [Variables](ICpNonLinearDecomposition.Variables.htm)
		 Свойство Variables
		 возвращает переменные нелинейной системы уравнений.
