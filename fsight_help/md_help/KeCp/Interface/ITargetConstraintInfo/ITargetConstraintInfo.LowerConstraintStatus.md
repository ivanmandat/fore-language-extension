# ITargetConstraintInfo.LowerConstraintStatus


## Синтаксис


LowerConstraintStatus: [TargetConstraintStatusType](../../Enums/TargetConstraintStatusType.htm);


## Описание


Свойство LowerConstraintStatus
 возвращает состояние нижней границы ограничения.


## Комментарии


Значение доступно после расчета целевой задачи. Для получения состояния
 верхней границы ограничения используйте свойство [ITargetConstraintInfo.UpperConstraintStatus](ITargetConstraintInfo.UpperConstraintStatus.htm).
