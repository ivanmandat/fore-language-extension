# ITargetConstraintInfo.Include


## Синтаксис


Include: Boolean;


## Описание


Свойство Include определяет,
 активно ли значение границ ограничения.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Значение границ активно;


	- False. Ограничение не
	 используется.
