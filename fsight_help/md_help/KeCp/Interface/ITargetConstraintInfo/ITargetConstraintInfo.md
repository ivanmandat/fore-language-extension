# ITargetConstraintInfo


Сборка: Cp;


## Описание


Интерфейс ITargetConstraintInfo
 предназначен для работы с значением границ нелинейного ограничения.


## Иерархия наследования


           ITargetConstraintInfo


## Комментарии


Для работы с коллекцией значений границ нелинейного ограничения используйте
 интерфейс [ITargetConstraintInfoArray](../ITargetConstraintInfoArray/ITargetConstraintInfoArray.htm).

		 Имя свойства

		 [Include](ITargetConstraintInfo.Include.htm)
