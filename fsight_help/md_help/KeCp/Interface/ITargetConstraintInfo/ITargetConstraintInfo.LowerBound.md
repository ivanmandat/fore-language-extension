# ITargetConstraintInfo.LowerBound


## Синтаксис


LowerBound: Double;


## Описание


Свойство LowerBound определяет
 значение нижней границы ограничения.


## Комментарии


Для задания верхней границы ограничения используйте свойство [ITargetConstraintInfo.UpperBound](ITargetConstraintInfo.UpperBound.htm).
