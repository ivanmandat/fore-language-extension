# ITargetConstraintInfo.UpperBound


## Синтаксис


UpperBound: Double;


## Описание


Свойство UpperBound определяет
 значение верхней границы ограничения.


## Комментарии


Для задания нижней границы ограничения используйте свойство [ITargetConstraintInfo.LowerBound](ITargetConstraintInfo.LowerBound.htm).
