# ITargetConstraintInfo.TimeMoment


## Синтаксис


TimeMoment: Integer;


## Описание


Свойство TimeMoment определяет
 момент времени, к которому относится ограничение.


## Комментарии


По умолчанию свойство имеет значение «0».
