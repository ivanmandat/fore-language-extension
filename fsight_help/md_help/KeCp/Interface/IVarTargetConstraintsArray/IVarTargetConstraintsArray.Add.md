# IVarTargetConstraintsArray.Add


## Синтаксис


Add: [IVarTargetConstraint](../IVarTargetConstraint/IVarTargetConstraint.htm);


## Описание


Метод Add добавляет новое
 ограничение в коллекцию.


## Комментарии


Для удаления ограничения из коллекции используйте метод [IVarTargetConstraintsArray.Remove](IVarTargetConstraintsArray.Remove.htm).
