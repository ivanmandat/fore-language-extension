# IVarTargetConstraintsArray


Сборка: Cp;


## Описание


Интерфейс IVarTargetConstraintsArray
 предназначен для работы с коллекцией ограничений на переменную.


## Иерархия наследования


           IVarTargetConstraintsArray


## Комментарии


Ограничения позволяют указать интервал, в котором должны находиться
 значения переменной. Для получения коллекции ограничений управляющей переменной
 используйте свойство [ITargetControlVariable.Constraints](../ITargetControlVariable/ITargetControlVariable.Constraints.htm),
 фазовой переменной - свойство [ITargetPhaseVariable.Constraints](../ITargetPhaseVariable/ITargetPhaseVariable.Constraints.htm).
