# ICpNonLinearOptimization.DifferEvolutionParams


## Синтаксис


DifferEvolutionParams: [IDifferEvolutionParams](../IDifferEvolutionParams/IDifferEvolutionParams.htm);


## Описание


Свойство DifferEvolutionParams
 определяет параметры метода дифференциальной эволюции.


## Комментарии


Параметры участвуют в расчете, если для поиска решения используется
 метод дифференциальной эволюции, то есть свойство [ICpNonLinearOptimization.MethodType](ICpNonLinearOptimization.MethodType.htm)
 имеет значение [CpNonLinearMethodType.DifferentEvolution](../../Enums/CpNonLinearMethodType.htm).
