# ICpNonLinearOptimization.MaxIterationsCount


## Синтаксис


MaxIterationsCount: Integer;


## Описание


Свойство MaxIterationsCount определяет максимальное число итераций, за которое должен осуществиться поиск решения. Значение по умолчанию - «1500».
