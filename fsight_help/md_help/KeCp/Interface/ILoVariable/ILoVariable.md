# ILoVariable


## Описание


Интерфейс ILoVariable содержит
 свойства переменной линейной оптимизации.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ControlVariable](ILoVariable.ControlVariable.htm)
		 Свойство ControlVariable
		 определяет, является ли данная переменная управляющей.


		 ![](../../Property_Image.gif)
		 [Forestall](ILoVariable.Forestall.htm)
		 Свойство Forestall
		 определяет ожидаемые значения переменной.


		 ![](../../Property_Image.gif)
		 [Id](ILoVariable.Id.htm)
		 Свойство Id возвращает
		 идентификатор переменной.


		 ![](../../Property_Image.gif)
		 [InitApproximation](ILoVariable.InitApproximation.htm)
		 Свойство InitApproximation
		 определяет массив начальных приближений для управляющей переменной.


		 ![](../../Property_Image.gif)
		 [Name](ILoVariable.Name.htm)
		 Свойство Name определяет
		 наименование переменной.


		 ![](../../Property_Image.gif)
		 [Retrospective](ILoVariable.Retrospective.htm)
		 Свойство Retrospective
		 определяет начальные значения переменной.


		 ![](../../Property_Image.gif)
		 [Trajectory](ILoVariable.Trajectory.htm)
		 Свойство Trajectory
		 определяет текущие значения переменной.
