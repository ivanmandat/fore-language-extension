# IVarTargetConstraint.UpperBoundFixed


## Синтаксис


UpperBoundFixed: Boolean;


## Описание


Свойство UpperBoundFixed определяет,
 зафиксирована ли верхняя граница ограничения.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Верхняя граница ограничения зафиксирована;


	- False. Верхняя граница
	 ограничения ослаблена.


Для определения, зафиксирована ли нижняя граница ограничения используйте
 свойство [IVarTargetConstraint.LowerBoundFixed](IVarTargetConstraint.LowerBoundFixed.htm).
