# IVarTargetConstraint.UpperConstraintStatus


## Синтаксис


UpperConstraintStatus: [TargetConstraintStatusType](../../Enums/TargetConstraintStatusType.htm);


## Описание


Свойство UpperConstraintStatus
 возвращает состояние верхней границы ограничения.


## Комментарии


Значение доступно после расчета целевой задачи. Для получения состояния
 нижней границы ограничения используйте свойство [IVarTargetConstraint.LowerConstraintStatus](IVarTargetConstraint.LowerConstraintStatus.htm).
