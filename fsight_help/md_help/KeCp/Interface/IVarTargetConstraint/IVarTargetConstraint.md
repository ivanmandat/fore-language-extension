# IVarTargetConstraint


Сборка: Cp;


## Описание


Интерфейс IVarTargetConstraint
 предназначен для работы с ограничением на переменную.


## Иерархия наследования


           IVarTargetConstraint


## Комментарии


Для работы со всеми ограничениями переменной используйте интерфейс [IVarTargetConstraintsArray](../IVarTargetConstraintsArray/IVarTargetConstraintsArray.htm).
