# IVarTargetConstraint.LowerBound


## Синтаксис


LowerBound: Double;


## Описание


Свойство LowerBound определяет
 значение нижней границы ограничения.


## Комментарии


Для задания верхней границы ограничения используйте свойство [IVarTargetConstraint.UpperBound](IVarTargetConstraint.UpperBound.htm).
