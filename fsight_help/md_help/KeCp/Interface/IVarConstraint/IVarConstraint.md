# IVarConstraint


## Описание


Интерфейс IVarConstraint содержит
 свойства для работы с ограничением переменной нелинейной оптимизации.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Lag](IVarConstraint.Lag.htm)
		 Свойство Lag определяет
		 момент времени, на который накладывается ограничение.


		 ![](../../Property_Image.gif)
		 [LowerBound](IVarConstraint.LowerBound.htm)
		 Свойство LowerBound
		 определяет нижнюю границу ограничения.


		 ![](../../Property_Image.gif)
		 [UpperBound](IVarConstraint.UpperBound.htm)
		 Свойство UpperBound
		 определяет верхнюю границу ограничения.
