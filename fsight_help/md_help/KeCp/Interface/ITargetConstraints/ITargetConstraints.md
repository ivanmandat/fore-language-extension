# ITargetConstraints


Сборка: Cp;


## Описание


Интерфейс ITargetConstraints
 предназначен для работы с коллекцией ограничений нелинейной оптимизации.


## Иерархия наследования


           ITargetConstraints


## Комментарии


Для получения коллекции нелинейных ограничений используйте свойство
 [ICpTargetAdjustment.Constraints](../ICpTargetAdjustment/ICpTargetAdjustment.Constraints.htm).
