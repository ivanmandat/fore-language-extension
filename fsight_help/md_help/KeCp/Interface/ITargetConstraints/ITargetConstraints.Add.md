# ITargetConstraints.Add


## Синтаксис


Add: [ITargetConstraint](../ITargetConstraint/ITargetConstraint.htm);


## Описание


Метод Add добавляет ограничение
 в коллекцию.


## Комментарии


Для удаления ограничения из коллекции используйте метод [ITargetConstraints.Remove](ITargetConstraints.Remove.htm).
