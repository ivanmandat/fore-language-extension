# ITargetConstraints.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс удаляемого ограничения.


## Описание


Метод Remove удаляет из
 коллекции ограничение с указанным индексом.


## Комментарии


Индексация ограничений в коллекции сквозная и начинается с нуля.


Если удаление прошло успешно, то метод возвращает значение True,
 иначе - False.
