# INonLoVariables


## Описание


Интерфейс INonLoVariables содержит
 свойства и методы для работы с коллекцией переменных нелинейной оптимизации.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](INonLoVariables.Count.htm)
		 Свойство Count возвращает
		 количество переменных в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](INonLoVariables.Item.htm)
		 Свойство Item возвращает
		 переменную из коллекции по его индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](INonLoVariables.Add.htm)
		 Метод Add осуществляет
		 добавление переменной в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](INonLoVariables.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех переменных из коллекции.


		 ![](../../Sub_Image.gif)
		 [FindById](INonLoVariables.FindById.htm)
		 Метод FindById осуществляет
		 поиск переменной в коллекции по ее идентификатору.


		 ![](../../Sub_Image.gif)
		 [Remove](INonLoVariables.Remove.htm)
		 Метод Remove осуществляет
		 удаление переменной из коллекции по его индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveById](INonLoVariables.RemoveById.htm)
		 Метод RemoveById осуществляет
		 удаление переменной из коллекции по ее идентификатору.
