# INonLoVariables.RemoveById


## Синтаксис


RemoveById(Id: String): Boolean;


## Параметры


Id. Идентификатор
 переменной.


## Описание


Метод RemoveById осуществляет
 удаление переменной из коллекции по ее идентификатору.


## Комментарии


Возвращает True в случае удачного
 удаления.
