# INonLoResults


Сборка: Cp;


## Описание


Интерфейс INonLoResults содержит
 свойства для работы с результатами расчета метода нелинейной оптимизации.


## Иерархия наследования


           [ICpResults](../ICpResults/ICpResults.htm)


           INonLoResults


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CriterionFunctionTrajectory](INonLoResults.CriterionFunctionTrajectory.htm)
		 Свойство CriterionFunctionTrajectory
		 возвращает траекторию целевой функции, получившуюся в результате
		 расчета метода нелинейной оптимизации.


		 ![](../../Property_Image.gif)
		 [ObjValByIter](INonLoResults.ObjValByIter.htm)
		 Свойство ObjValByIter
		 возвращает промежуточные значения целевой функции для каждой итерации
		 в виде массива.


		 ![](../../Property_Image.gif)
		 [OptimalValue](INonLoResults.OptimalValue.htm)
		 Свойство OptimalValue
		 возвращает оптимальное значение целевой функции.


		 ![](../../Property_Image.gif)
		 [VarValues](INonLoResults.VarValues.htm)
		 Свойство VarValues
		 возвращает вычисленные значения переменной в виде массива.
