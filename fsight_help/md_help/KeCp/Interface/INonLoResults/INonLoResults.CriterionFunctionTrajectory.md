# INonLoResults.CriterionFunctionTrajectory


## Синтаксис


CriterionFunctionTrajectory: Array;


## Описание


Свойство CriterionFunctionTrajectory
 возвращает траекторию целевой функции, получившуюся в результате расчета
 метода нелинейной оптимизации.


## Комментарии


Для получения промежуточных значений целевой функции для каждой итерации
 в виде массива используйте свойство [INonLoResults.ObjValByIter](INonLoResults.ObjValByIter.htm).
