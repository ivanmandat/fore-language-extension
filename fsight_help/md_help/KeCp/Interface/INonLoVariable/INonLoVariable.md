# INonLoVariable


## Описание


Интерфейс INonLoVariable содержит
 свойства переменной нелинейной оптимизации.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CoefficientsOrder](INonLoVariable.CoefficientsOrder.htm)
		 Свойство CoefficientsOrder
		 определяет порядок коэффициентов.


		 ![](../../Property_Image.gif)
		 [Constraints](INonLoVariable.Constraints.htm)
		 Свойство Constraints
		 возвращает ограничения переменной нелинейной оптимизации.


		 ![](../../Property_Image.gif)
		 [ControlVariable](INonLoVariable.ControlVariable.htm)
		 Свойство ControlVariable
		 определяет, является ли данная переменная управляющей.


		 ![](../../Property_Image.gif)
		 [Id](INonLoVariable.Id.htm)
		 Свойство Id возвращает
		 идентификатор переменной.


		 ![](../../Property_Image.gif)
		 [InitApproximation](INonLoVariable.InitApproximation.htm)
		 Свойство InitApproximation
		 определяет массив начальных приближений для управляющей переменной.


		 ![](../../Property_Image.gif)
		 [Name](INonLoVariable.Name.htm)
		 Свойство Name определяет
		 наименование переменной.


		 ![](../../Property_Image.gif)
		 [Retrospective](INonLoVariable.Retrospective.htm)
		 Свойство Retrospective
		 определяет начальные значения переменной.
