# ITargetControlVariablesArray.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс управляющей переменной.


## Описание


Метод Remove удаляет из
 коллекции управляющую переменную с указанным индексом.


## Комментарии


Индексация переменных в коллекции сквозная и начинается с нуля.


Если удаление прошло успешно, то метод возвращает значение True,
 иначе - False.
