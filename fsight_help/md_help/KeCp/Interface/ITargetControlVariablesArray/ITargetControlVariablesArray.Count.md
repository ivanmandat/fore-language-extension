# ITargetControlVariablesArray.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает
 количество управляющих переменных в коллекции.


## Комментарии


Для получения управляющей переменной из коллекции используйте свойство
 [ITargetControlVariablesArray.Item](ITargetControlVariablesArray.Item.htm).
