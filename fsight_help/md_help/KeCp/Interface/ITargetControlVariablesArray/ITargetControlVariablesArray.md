# ITargetControlVariablesArray


Сборка: Cp;


## Описание


Интерфейс ITargetControlVariablesArray
 предназначен для работы с коллекцией управляющих переменных.


## Иерархия наследования


           ITargetControlVariablesArray


## Комментарии


Для получения коллекции управляющих переменных используйте свойство
 [ICpTargetAdjustment.ControlVariables](../ICpTargetAdjustment/ICpTargetAdjustment.ControlVariables.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITargetControlVariablesArray.Count.htm)
		 Свойство Count
		 возвращает количество управляющих переменных в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ITargetControlVariablesArray.Item.htm)
		 Свойство Item
		 возвращает заданную управляющую переменную из коллекции.
