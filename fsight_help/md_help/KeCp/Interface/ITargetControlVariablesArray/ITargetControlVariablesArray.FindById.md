# ITargetControlVariablesArray.FindById


## Синтаксис


FindById(Id: String): [ITargetControlVariable](../ITargetControlVariable/ITargetControlVariable.htm);


## Параметры


Id. Идентификатор искомой управляющей
 переменной.


## Описание


Метод FindById выполняет
 поиск в коллекции управляющей переменной с заданным идентификатором.


## Комментарии


Если переменная с указанным идентификатором не найдена, то метод возвращает
 значение Null.
