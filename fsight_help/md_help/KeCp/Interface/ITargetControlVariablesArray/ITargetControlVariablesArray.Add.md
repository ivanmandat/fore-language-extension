# ITargetControlVariablesArray.Add


## Синтаксис


Add(Id: String): [ITargetControlVariable](../ITargetControlVariable/ITargetControlVariable.htm);


## Параметры


Id. Идентификатор добавляемой
 переменной.


## Описание


Метод Add добавляет управляющую
 переменную в коллекцию.


## Комментарии


Каждая переменная в коллекции должна иметь уникальный идентификатор.
 Наличие двух переменных с одинаковым идентификатором недопустимо.
