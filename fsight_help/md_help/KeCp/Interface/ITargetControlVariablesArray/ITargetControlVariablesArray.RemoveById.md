# ITargetControlVariablesArray.RemoveById


## Синтаксис


RemoveById(Id: String): Boolean;


## Параметры


Index. Идентификатор управляющей
 переменной.


## Описание


Метод RemoveById удаляет из
 коллекции управляющую переменную с указанным идентификатором.


## Комментарии


Если удаление прошло успешно, то метод возвращает значение True,
 иначе - False.
