# ITargetControlVariablesArray.Item


## Синтаксис


Item(Index: Integer): [ITargetControlVariable](../ITargetControlVariable/ITargetControlVariable.htm);


## Параметры


Index. Индекс переменной в
 коллекции.


## Описание


Свойство Item возвращает
 заданную управляющую переменную из коллекции.


## Комментарии


Индексация переменных в коллекции сквозная и начинается с нуля.
