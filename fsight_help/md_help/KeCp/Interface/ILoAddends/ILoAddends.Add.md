# ILoAddends.Add


## Синтаксис


Add(Variable: [ILoVariable](../ILoVariable/ILoVariable.htm)):
 [ILoAddend](../ILoAddend/ILoAddend.htm);


## Параметры


Variable. Переменная линейной
 оптимизации.


## Описание


Метод Add осуществляет добавление
 слагаемого в коллекцию.
