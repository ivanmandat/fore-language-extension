# ILoAddends.FindByVar


## Синтаксис


FindByVar(Variable: [ILoVariable](../ILoVariable/ILoVariable.htm)):
 [ILoAddend](../ILoAddend/ILoAddend.htm);


## Параметры


Variable. Переменная, по которой
 осуществляется поиск слагаемого.


## Описание


Метод FindByVar осуществляет
 поиск слагаемого в коллекции и возвращает объект, содержащий слагаемое
 в случае удачного поиска.


## Комментарии


Поиск осуществляется по переменной, входящей в слагаемое.
