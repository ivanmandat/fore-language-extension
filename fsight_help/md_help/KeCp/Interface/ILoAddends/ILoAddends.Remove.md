# ILoAddends.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс слагаемого в
 коллекции.


## Описание


Метод Remove осуществляет удаление
 слагаемого из коллекции по его индексу.


## Комментарии


Возвращает True в случае удачного
 удаления.
