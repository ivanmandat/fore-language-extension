# ILoAddends


## Описание


Интерфейс ILoAddends содержит
 свойства и методы коллекции слагаемых уравнения линейной оптимизации.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ILoAddends.Count.htm)
		 Свойство Count возвращает
		 количество слагаемых в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ILoAddends.Item.htm)
		 Свойство Item возвращает
		 слагаемое из коллекции по его индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Add](ILoAddends.Add.htm)
		 Метод Add осуществляет
		 добавление слагаемого в коллекцию.


		 ![](../../Property_Image.gif)
		 [Clear](ILoAddends.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех слагаемых из коллекции.


		 ![](../../Property_Image.gif)
		 [FindByVar](ILoAddends.FindByVar.htm)
		 Метод FindByVar осуществляет
		 поиск слагаемого в коллекции.


		 ![](../../Property_Image.gif)
		 [Remove](ILoAddends.Remove.htm)
		 Метод Remove осуществляет
		 удаление слагаемого из коллекции по его индексу.


		 ![](../../Property_Image.gif)
		 [RemoveByVar](ILoAddends.RemoveByVar.htm)
		 Метод RemoveByVar осуществляет
		 удаление слагаемого из коллекции.
