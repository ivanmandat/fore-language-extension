# ILoAddends.Item


## Синтаксис


Item(Index: Integer): [ILoAddend](../ILoAddend/ILoAddend.htm);


## Параметры


Index. Индекс слагаемого в
 коллекции.


## Описание


Свойство Item возвращает слагаемое
 из коллекции по его индексу.


## Комментарии


Свойство доступно только для чтения.
