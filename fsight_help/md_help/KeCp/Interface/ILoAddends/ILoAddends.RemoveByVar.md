# ILoAddends.RemoveByVar


## Синтаксис


RemoveByVar(Variable: [ILoVariable](../ILoVariable/ILoVariable.htm)):
 Boolean;


## Параметры


Variable.
 Переменная, по которой осуществляется удаление слагаемого.


## Описание


Метод RemoveByVar позволяет
 удалить слагаемое, по переменной, входящей в его состав.


## Комментарии


Возвращает True в случае удачного
 удаления.
