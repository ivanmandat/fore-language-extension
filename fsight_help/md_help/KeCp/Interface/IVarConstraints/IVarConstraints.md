# IVarConstraints


## Описание


Интерфейс IVarConstraints содержит
 свойства и методы для работы с коллекцией ограничений переменной нелинейной
 оптимизации.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IVarConstraints.Count.htm)
		 Свойство Count возвращает
		 количество ограничений в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IVarConstraints.Item.htm)
		 Свойство Item возвращает
		 ограничение из коллекции по его индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IVarConstraints.Add.htm)
		 Метод Add осуществляет
		 добавление ограничения в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](IVarConstraints.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех ограничений из коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IVarConstraints.Remove.htm)
		 Метод Remove осуществляет
		 удаление ограничения из коллекции по его индексу.
