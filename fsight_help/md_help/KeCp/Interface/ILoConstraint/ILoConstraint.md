# ILoConstraint


## Описание


Интерфейс ILoConstraint содержит
 свойства для работы с ограничением линейной оптимизации.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Addends](ILoConstraint.Addends.htm)
		 Свойство Addends возвращает
		 коллекцию слагаемых, используемых для создания ограничения.


		 ![](../../Property_Image.gif)
		 [LowerBound](ILoConstraint.LowerBound.htm)
		 Свойство LowerBound
		 определяет нижнюю границу ограничения.


		 ![](../../Property_Image.gif)
		 [UpperBound](ILoConstraint.UpperBound.htm)
		 Свойство UpperBound
		 определяет верхнюю границу ограничения.
