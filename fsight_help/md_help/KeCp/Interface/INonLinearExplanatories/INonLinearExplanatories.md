# INonLinearExplanatories


## Описание


Интерфейс INonLinearExplanatories
 предназначен для работы с коллекцией объясняющих переменных для метода
 нелинейной оптимизации.


## Комментарии


Элементы коллекции реализованы интерфейсом [INonLinearExplanatory](../INonLinearExplanatory/INonLinearExplanatory.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](INonLinearExplanatories.Count.htm)
		 Свойство Count возвращает
		 количество объясняющих переменных в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](INonLinearExplanatories.Item.htm)
		 Свойство Item возвращает
		 переменную из коллекции по ее индексу.
