# INonLinearExplanatories.Add


## Синтаксис


Add: [INonLinearExplanatory](../INonLinearExplanatory/INonLinearExplanatory.htm);


## Описание


Метод Add добавляет переменную
 в коллекцию.


## Комментарии


Add после выполнения возвращает
 добавленную переменную. Переменная добавляется в конец коллекции, при
 этом значение свойства [INonLinearExplanatories.Count](INonLinearExplanatories.Count.htm)
 увеличивается на единицу.
