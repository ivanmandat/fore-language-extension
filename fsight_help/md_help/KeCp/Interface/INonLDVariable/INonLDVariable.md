# INonLDVariable


## Описание


Интерфейс INonLDVariable содержит
 свойства и методы для работы с переменной, входящей в систему нелинейных
 уравнений.


## Свойства


		 Имя метода
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CoefficientsOrderForestall](INonLDVariable.CoefficientsOrderForestall.htm)
		 Свойство CoefficientsOrderForestall
		 определяет порядок коэффициентов для ожидаемых значений.


		 ![](../../Property_Image.gif)
		 [CoefficientsOrderRetrospective](INonLDVariable.CoefficientsOrderRetrospective.htm)
		 Свойство CoefficientsOrderRetrospective
		 определяет порядок коэффициентов для начальных значений.


		 ![](../../Property_Image.gif)
		 [Constraints](INonLDVariable.Constraints.htm)
		 Свойство Constraints
		 возвращает ограничения переменной.


		 ![](../../Property_Image.gif)
		 [ControlVariable](INonLDVariable.ControlVariable.htm)
		 Свойство ControlVariable
		 определяет, является ли данная переменная управляющей.


		 ![](../../Property_Image.gif)
		 [Forestall](INonLDVariable.Forestall.htm)
		 Свойство Forestall
		 определяет ожидаемые значения переменной.


		 ![](../../Property_Image.gif)
		 [Id](INonLDVariable.Id.htm)
		 Свойство Id возвращает
		 идентификатор переменной.


		 ![](../../Property_Image.gif)
		 [InitApproximation](INonLDVariable.InitApproximation.htm)
		 Свойство InitApproximation
		 определяет массив начальных приближений для переменной.


		 ![](../../Property_Image.gif)
		 [Name](INonLDVariable.Name.htm)
		 Свойство Name определяет
		 наименование переменной.


		 ![](../../Property_Image.gif)
		 [Retrospective](INonLDVariable.Retrospective.htm)
		 Свойство Retrospective
		 определяет начальные значения переменной.
