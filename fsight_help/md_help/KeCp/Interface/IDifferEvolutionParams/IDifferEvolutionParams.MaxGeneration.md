# IDifferEvolutionParams.MaxGeneration


## Синтаксис


MaxGeneration: Integer;


## Описание


Свойство MaxGeneration
 определяет число поколений.


## Комментарии


Число поколений должно быть больше нуля.


Значение по умолчанию: 200.
