# IDifferEvolutionParams.PopulationSize


## Синтаксис


PopulationSize: Integer;


## Описание


Свойство PopulationSize
 определяет размер популяции.


## Комментарии


Рекомендуется, чтобы размер популяции соответствовал неравенству: 5*N < PopulationSize
 < 10*N, где N
 - количество переменных в задаче оптимизации.
