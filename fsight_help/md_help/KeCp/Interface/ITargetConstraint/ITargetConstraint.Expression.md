# ITargetConstraint.Expression


## Синтаксис


Expression: String;


## Описание


Свойство Expression определяет
 выражение нелинейного ограничения.


## Комментарии


Для задания значений границ нелинейного ограничения используйте свойство
 [ITargetConstraint.Constraints](ITargetConstraint.Constraints.htm).
