# ITargetConstraint


Сборка: Cp;


## Описание


Интерфейс ITargetConstraint
 предназначен для работы с ограничением нелинейной оптимизации.


## Иерархия наследования


           ITargetConstraint


## Комментарии


Для работы со всеми ограничениями нелинейной оптимизации используйте
 интерфейс [ITargetConstraint](ITargetConstraint.htm).
