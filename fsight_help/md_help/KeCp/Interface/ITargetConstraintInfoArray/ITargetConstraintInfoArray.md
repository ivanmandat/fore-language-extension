# ITargetConstraintInfoArray


Сборка: Cp;


## Описание


Интерфейс ITargetConstraintInfoArray
 предназначен для работы с коллекцией значений границ нелинейного
 ограничения.


## Иерархия наследования


           ITargetConstraintInfoArray


## Комментарии


Для получения коллекции используйте свойство [ITargetConstraint.Constraints](../ITargetConstraint/ITargetConstraint.Constraints.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITargetConstraintInfoArray.Count.htm)
		 Свойство Count
		 возвращает количество значений границ нелинейного ограничения
		 в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ITargetConstraintInfoArray.Item.htm)
		 Свойство Item
		 возвращает из коллекции значение границ ограничения с заданным
		 индексом.
