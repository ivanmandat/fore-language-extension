# ITargetConstraintInfoArray.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс значения границ
 ограничения в коллекции.


## Описание


Метод Remove удаляет из
 коллекции значение границ ограничения с указанным индексом.


## Комментарии


Индексация значения границ ограничения в коллекции сквозная и начинается
 с нуля.


Если удаление прошло успешно, то метод возвращает значение True,
 иначе - False.
