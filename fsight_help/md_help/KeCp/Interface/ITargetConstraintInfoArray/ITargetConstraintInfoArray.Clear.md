# ITargetConstraintInfoArray.Clear


## Синтаксис


Clear;


## Описание


Метод Clear очищает коллекцию
 значений границ нелинейного ограничения.


## Комментарии


После вызова метода Clear значение
 свойства [ITargetConstraintInfoArray.Count](ITargetConstraintInfoArray.Count.htm)
 будет иметь значение ноль.
