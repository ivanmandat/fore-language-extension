# ITargetConstraintInfoArray.Add


## Синтаксис


Add: [ITargetConstraintInfo](../ITargetConstraintInfo/ITargetConstraintInfo.htm);


## Описание


Метод Add добавляет новое
 значение границ нелинейного ограничения в коллекцию.


## Комментарии


Для получения заданного значения границ нелинейного ограничения из коллекции
 используйте свойство [ITargetConstraintInfoArray.Item](ITargetConstraintInfoArray.Item.htm).
