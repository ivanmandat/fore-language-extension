# ITargetConstraintInfoArray.Item


## Синтаксис


Item(Index: Integer): [ITargetConstraintInfo](../ITargetConstraintInfo/ITargetConstraintInfo.htm);


## Параметры


Index. Индекс значения границ
 ограничения в коллекции.


## Описание


Свойство Item возвращает
 из коллекции значение границ ограничения с заданным индексом.


## Комментарии


Индексация значения границ ограничения в коллекции сквозная и начинается
 с нуля.
