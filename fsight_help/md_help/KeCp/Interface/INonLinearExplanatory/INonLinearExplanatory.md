# INonLinearExplanatory


Сборка:Cp ;


## Описание


Интерфейс INonLinearExplanatory
 предназначен для работы с объясняющей переменной для метода нелинейной
 оптимизации.


## Иерархия наследования


           INonLinearExplanatory


## Комментарии


Значения переменной должны быть заданы до расчета метода.
