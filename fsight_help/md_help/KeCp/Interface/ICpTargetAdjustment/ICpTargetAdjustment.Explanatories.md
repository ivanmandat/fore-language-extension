# ICpTargetAdjustment.Explanatories


## Синтаксис


Explanatories: [INonLinearExplanatories](../INonLinearExplanatories/INonLinearExplanatories.htm);


## Описание


Свойство Explanatories
 возвращает коллекцию объясняющих переменных.


## Комментарии


Для получения коллекций фазовых и управляющих переменных используйте
 свойства [ICpTargetAdjustment.PhaseVariables](ICpTargetAdjustment.PhaseVariables.htm)
 и [ICpTargetAdjustment.ControlVariables](ICpTargetAdjustment.ControlVariables.htm).

Использование свойства приведено в примере для [ICpTargetAdjustment.Tolerance](ICpTargetAdjustment.Tolerance.htm).
