# ICpTargetAdjustment.Constraints


## Синтаксис


Constraints: [ITargetConstraints](../ITargetConstraints/ITargetConstraints.htm);


## Описание


Свойство Constraints возвращает
 коллекцию ограничений нелинейной оптимизации.


## Комментарии


Для указания того, что целевая задача должна рассчитываться с нелинейными
 ограничениями, используйте свойство [ICpTargetAdjustment.CalcWithNonLinConstraints](ICpTargetAdjustment.CalcWithNonLinConstraints.htm).

Использование свойства приведено в примере для [ICpTargetAdjustment.Tolerance](ICpTargetAdjustment.Tolerance.htm).
