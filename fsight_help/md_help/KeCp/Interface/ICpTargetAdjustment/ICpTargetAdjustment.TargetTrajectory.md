# ICpTargetAdjustment.TargetTrajectory


## Синтаксис


TargetTrajectory: Array;


## Описание


Свойство TargetTrajectory
 определяет траекторию, которой должна соответствовать целевая функция.


## Комментарии


Траектория задается массивом вещественных чисел.


Для задания целевой функции используйте свойство [ICpTargetAdjustment.CriterionFunction](ICpTargetAdjustment.CriterionFunction.htm).
