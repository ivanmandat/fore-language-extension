# ICpTargetAdjustment.CriterionFunction


## Синтаксис


CriterionFunction: String;


## Описание


Свойство CriterionFunction
 определяет целевую функцию.


## Комментарии


В целевой функции допускается использование фазовых, объясняющих и управляющих
 переменных.
