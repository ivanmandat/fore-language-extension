# ICpTargetAdjustment.AutoSearchType


## Синтаксис


AutoSearchType: [TargetAutoSearchType](../../Enums/TargetAutoSearchType.htm);


## Описание


Свойство AutoSearchType
 определяет тип целевой задачи.


## Комментарии


По умолчанию рассчитывается задача максимизации целевой функции, то
 есть свойство AutoSearchType принимает
 значение [TargetAutoSearchType.Maximum](../../Enums/TargetAutoSearchType.htm).


## Пример


Использование свойства приведено в примере для [ICpTargetAdjustment.Tolerance](ICpTargetAdjustment.Tolerance.htm).
