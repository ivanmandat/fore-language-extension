# ICpTargetAdjustment.AutoAdjustSatisfactoryTolerance


## Синтаксис


AutoAdjustSatisfactoryTolerance: Double;


## Описание


Свойство AutoAdjustSatisfactoryTolerance
 определяет приемлемую точность автоматического подбора границ.


## Комментарии


По умолчанию приемлемая точность равна единице.
