# ICpTargetAdjustment.ControlVariables


## Синтаксис


ControlVariables: [ITargetControlVariablesArray](../ITargetControlVariablesArray/ITargetControlVariablesArray.htm);


## Описание


Свойство ControlVariables
 возвращает коллекцию управляющих переменных.


## Комментарии


Для получения коллекций фазовых и объясняющих переменных используйте
 свойства [ICpTargetAdjustment.PhaseVariables](ICpTargetAdjustment.PhaseVariables.htm)
 и [ICpTargetAdjustment.Explanatories](ICpTargetAdjustment.Explanatories.htm).
