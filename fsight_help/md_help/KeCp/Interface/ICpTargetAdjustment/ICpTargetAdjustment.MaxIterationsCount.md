# ICpTargetAdjustment.MaxIterationsCount


## Синтаксис


MaxIterationsCount: Integer;


## Описание


Свойство MaxIterationsCount
 определяет количество итераций расчета.


## Комментарии


По умолчанию расчет должен быть выполнен за 100 итераций.
