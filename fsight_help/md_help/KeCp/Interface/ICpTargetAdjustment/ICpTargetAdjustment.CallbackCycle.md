# ICpTargetAdjustment.CallbackCycle


## Синтаксис


CallbackCycle: [ICallbackCycle](../ICallbackCycle/ICallbackCycle.htm);


## Описание


Свойство CallbackCycle
 определяет обработчик, используемый для расчета управляющих переменных.


## Комментарии


Для получения коллекции управляющих переменных используйте свойство
 [ICpTargetAdjustment.ControlVariables](ICpTargetAdjustment.ControlVariables.htm).

Использование свойства приведено в примере для [ICpTargetAdjustment.Tolerance](ICpTargetAdjustment.Tolerance.htm).
