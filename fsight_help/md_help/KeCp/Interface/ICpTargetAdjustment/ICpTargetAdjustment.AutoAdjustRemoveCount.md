# ICpTargetAdjustment.AutoAdjustRemoveCount


## Синтаксис


AutoAdjustRemoveCount: Integer;


## Описание


Свойство AutoAdjustRemoveCount
 определяет количество ограничений, удаленных за одну итерацию при автоматическом
 подборе границ.


## Комментарии


По умолчанию за одну итерацию удаляется одно ограничение.


Если AutoAdjustRemoveCount имеет
 значение «0», то это приводит к снятию всех активных ограничений на данной
 итерации.
