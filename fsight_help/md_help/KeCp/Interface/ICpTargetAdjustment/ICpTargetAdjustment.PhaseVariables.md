# ICpTargetAdjustment.PhaseVariables


## Синтаксис


PhaseVariables: [ITargetPhaseVariablesArray](../ITargetPhaseVariablesArray/ITargetPhaseVariablesArray.htm);


## Описание


Свойство PhaseVariables
 возвращает коллекцию фазовых переменных.


## Комментарии


Для получения коллекции управляющих и объясняющих переменных используйте
 свойства [ICpTargetAdjustment.ControlVariables](ICpTargetAdjustment.ControlVariables.htm)
 и [ICpTargetAdjustment.Explanatories](ICpTargetAdjustment.Explanatories.htm).
