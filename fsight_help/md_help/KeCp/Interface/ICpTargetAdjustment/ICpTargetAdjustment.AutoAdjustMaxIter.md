# ICpTargetAdjustment.AutoAdjustMaxIter


## Синтаксис


AutoAdjustMaxIter: Integer;


## Описание


Свойство AutoAdjustMaxIter
 определяет количество циклов для автоматического подбора границ.


## Комментарии


По умолчанию используется один цикл.
