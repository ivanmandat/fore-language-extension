# INonLoConstraints


## Описание


Интерфейс INonLoConstraints
 содержит свойства и методы для работы с коллекцией ограничений нелинейной
 оптимизации.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](INonLoConstraints.Count.htm)
		 Свойство Count возвращает
		 количество ограничений в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](INonLoConstraints.Item.htm)
		 Свойство Item возвращает
		 ограничение из коллекции по его индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](INonLoConstraints.Add.htm)
		 Метод Add осуществляет
		 добавление ограничения в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](INonLoConstraints.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех ограничений из коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](INonLoConstraints.Remove.htm)
		 Метод Remove осуществляет
		 удаление ограничения из коллекции по его индексу.
