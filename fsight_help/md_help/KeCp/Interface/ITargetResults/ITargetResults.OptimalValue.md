# ITargetResults.OptimalValue


## Синтаксис


OptimalValue: Double;


## Описание


Свойство OptimalValue
 возвращает оптимальное значение целевой функции.


## Комментарии


Для получения траектории целевой функции используйте свойство [ITargetResults.CriterionFunctionTrajectory](ITargetResults.CriterionFunctionTrajectory.htm).
