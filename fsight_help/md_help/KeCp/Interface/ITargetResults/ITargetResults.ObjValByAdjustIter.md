# ITargetResults.ObjValByAdjustIter


## Синтаксис


ObjValByAdjustIter: Array;


## Описание


Свойство ObjValByAdjustIter
 возвращает прогресс изменения целевой функции в процессе последовательного
 ослабления границ.


## Комментарии


Прогресс контролируется свойством [ICpTargetAdjustment.AutoAdjustMaxIter](../ICpTargetAdjustment/ICpTargetAdjustment.AutoAdjustMaxIter.htm).


## Пример


Использование свойства приведено в примере для [ICpTargetAdjustment.Tolerance](../ICpTargetAdjustment/ICpTargetAdjustment.Tolerance.htm).
