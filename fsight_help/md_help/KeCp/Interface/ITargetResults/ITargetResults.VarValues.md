# ITargetResults.VarValues


## Синтаксис


VarValues(VarId: String): Array;


## Параметры


VarId. Идентификатор переменной.


## Описание


Свойство VarValues возвращает
 значения фазовых и управляющих переменных.


## Комментарии


Возвращаемые значения управляющих переменных отвечают оптимальному значению
 целевой функции, а фазовых переменных - оптимальному управлению.
