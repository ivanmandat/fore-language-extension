# ITargetResults.ObjValByIter


## Синтаксис


ObjValByIter: Array;


## Описание


Свойство ObjValByIter
 возвращает прогресс изменения целевой функции в процессе решения задачи
 нелинейного программирования.


## Комментарии


Прогресс контролируется свойством [ICpTargetAdjustment.MaxIterationsCount](../ICpTargetAdjustment/ICpTargetAdjustment.MaxIterationsCount.htm).


## Пример


Использование свойства приведено в примере для [ICpTargetAdjustment.Tolerance](../ICpTargetAdjustment/ICpTargetAdjustment.Tolerance.htm).
