# ITargetResults.CriterionFunctionTrajectory


## Синтаксис


CriterionFunctionTrajectory: Array;


## Описание


Свойство CriterionFunctionTrajectory
 возвращает траекторию целевой функции.


## Комментарии


Для получения оптимального значения целевой функции используйте свойство
 [ITargetResults.OptimalValue](ITargetResults.OptimalValue.htm).
