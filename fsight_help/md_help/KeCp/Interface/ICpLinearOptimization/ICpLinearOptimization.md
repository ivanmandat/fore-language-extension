## Описание


Интерфейс ICpLinearOptimization
 содержит свойства и методы линейной оптимизации.

## Комментарии


Для создания объекта, выполняющего линейную оптимизацию, используйте
 класс [LinearOptimization](../../Class/LinearOptimization/LinearOptimization.htm).
