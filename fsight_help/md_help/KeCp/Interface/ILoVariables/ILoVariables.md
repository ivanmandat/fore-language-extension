# ILoVariables


## Описание


Интерфейс ILoVariables содержит
 свойства и методы коллекции переменных линейной оптимизации.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ILoVariables.Count.htm)
		 Свойство Count возвращает
		 количество переменных в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ILoVariables.Item.htm)
		 Свойство Item возвращает
		 переменную из коллекции по его индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ILoVariables.Add.htm)
		 Метод Add осуществляет
		 добавление переменной в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](ILoVariables.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех переменных из коллекции.


		 ![](../../Sub_Image.gif)
		 [FindById](ILoVariables.FindById.htm)
		 Метод FindById осуществляет
		 поиск переменной в коллекции по ее идентификатору.


		 ![](../../Sub_Image.gif)
		 [Remove](ILoVariables.Remove.htm)
		 Метод Remove осуществляет
		 удаление переменной из коллекции по его индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveById](ILoVariables.RemoveById.htm)
		 Метод RemoveById осуществляет
		 удаление переменной из коллекции по ее идентификатору.
