# ILoVariables.Item


## Синтаксис


Item(Index: Integer): [ILoVariable](../ILoVariable/ILoVariable.htm);


## Параметры


Index. Индекс переменной в
 коллекции.


## Описание


Свойство Item возвращает переменную
 из коллекции по его индексу.


## Комментарии


Свойство доступно только для чтения.

Использование свойства приведено в примере для свойства [ILoVariable.Id](../ILoVariable/ILoVariable.Id.htm).
