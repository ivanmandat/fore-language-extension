# ILoVariables.Add


## Синтаксис


Add(Id:String): [ILoVariable](../ILoVariable/ILoVariable.htm);


## Параметры


Id. Идентификатор добавляемой
 переменной.


## Описание


Метод Add осуществляет добавление
 переменной в коллекцию.

Использование метода приведено в примере для свойства [ILoVariable.ControlVariable](../ILoVariable/ILoVariable.ControlVariable.htm).
