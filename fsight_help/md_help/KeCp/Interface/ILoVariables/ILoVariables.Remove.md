# ILoVariables.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс переменной в
 коллекции.


## Описание


Метод Remove осуществляет удаление
 переменной из коллекции по его индексу.


## Комментарии


Возвращает True в случае удачного
 удаления.
