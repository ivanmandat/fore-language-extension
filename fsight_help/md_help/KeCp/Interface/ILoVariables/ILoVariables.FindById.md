# ILoVariables.FindById


## Синтаксис


FindById(Id: String): [ILoVariable](../ILoVariable/ILoVariable.htm);


## Параметры


Id. Идентификатор искомой переменной.


## Описание


Метод FindById осуществляет
 поиск переменной в коллекции по ее идентификатору.

Использование метода приведено в примере для свойства [ILoVariable.ControlVariable](../ILoVariable/ILoVariable.ControlVariable.htm).
