# ICpMethod


## Описание


Интерфейс ICpMethod содержит
 свойства и методы для работы с методами оптимизации. Данный интерфейс
 является базовым для всех методов оптимизации.

		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Extremum](ICpMethod.Extremum.htm)
		 Свойство Extremum определяет
		 тип экстремума, который будет искаться при нахождении оптимального
		 значения.
