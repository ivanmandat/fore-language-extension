# ICallbackCycle


Сборка:Cp ;


## Описание


Интерфейс ICallbackCycle используется
 для пользовательского расчета управляющих переменных нелинейной оптимизации.


## Иерархия наследования


           ICallbackCycle


## Комментарии


Для применения пользовательского расчёта в нелинейной оптимизации используйте
 свойство [ICpNonLinearOptimization.CallbackCycle](../ICpNonLinearOptimization/ICpNonLinearOptimization.CallbackCycle.htm).
