# ICallbackCycle.GetTermValue


## Синтаксис


GetTermValue(EquationNumber: Integer; TimeMoment:
 Integer): Variant;


## Параметры


EquationNumber. Индекс уравнения,
 содержащего переменную.


TimeMoment. Текущая точка переменной.


## Описание


Метод GetTermValue используется
 для получения значения фазовой переменной.


## Комментарии


Для установки значения управляющей переменной используйте метод [ICallbackCycle.SetControlVariableValue](ICallbackCycle.SetControlVariableValue.htm).
