# ILinDecompResults


## Описание


Интерфейс ILinDecompResults
 содержит свойства для работы с результатами вычисления линейной системы
 уравнений.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [VarValues](ILinDecompResults.VarValues.htm)
		 Свойство VarValues
		 возвращает вычисленные значения переменной в виде массива.
