# INonLinerCallBack.GetConstraintTermValue


## Синтаксис


GetConstraintTermValue(TimeMoment: Integer; Term:
 String): Double;


## Параметры


TimeMoment. Значение, определяющее
 необходимый момент времени;


Term. Идентификатор переменной.


## Описание


Метод GetConstraintTermValue
 возвращает значение переменной ограничения в заданный момент времени.
