# INonLinerCallBack.SetControlVariableValue


## Синтаксис


SetControlVariableValue(VariableNumber: Integer;
 TimeMoment: Integer; Term: String): Double;


## Параметры


VariableNumber. Номер управляющей
 переменной;


TimeMoment. Значение, определяющее
 необходимый момент времени;


Term. Идентификатор переменной.


## Описание


Метод SetControlVariableValue
 изменяет значение управляющей переменной в заданный момент времени.
