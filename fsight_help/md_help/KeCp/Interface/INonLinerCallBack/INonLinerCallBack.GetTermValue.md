# INonLinerCallBack.GetTermValue


## Синтаксис


GetTermValue(EquationNumber: Integer; TimeMoment:
 Integer; Term: String): Double;


## Параметры


EquationNumber. Номер уравнения
 нелинейной оптимизации;


TimeMoment. Значение, определяющее
 необходимый момент времени;


Term. Идентификатор переменной.


## Описание


Метод GetTermValue возвращает
 значение переменной уравнения в заданный момент времени.
