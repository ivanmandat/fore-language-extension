# INonLinerCallBack.GetCriterialFuncTermValue


## Синтаксис


GetCriterialFuncTermValue(TimeMoment: Integer; Term:
 String): Double;


## Параметры


TimeMoment.
 Значение, определяющее необходимый момент времени;


Term. Идентификатор переменной.


## Описание


Метод GetCriterialFuncTermValue
 возвращает значение переменной целевой функции в заданный момент времени.
