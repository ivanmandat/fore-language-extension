# ITargetControlVariable.Constraints


## Синтаксис


Constraints: [IVarTargetConstraintsArray](../IVarTargetConstraintsArray/IVarTargetConstraintsArray.htm);


## Описание


Свойство Constraints определяет
 коллекцию ограничений на управляющую переменную.


## Комментарии


Ограничения позволяют указать интервал, в котором должны находиться
 значения управляющей переменной. Количество ограничений должно совпадать
 с количеством начальных значений управляющей переменной.
