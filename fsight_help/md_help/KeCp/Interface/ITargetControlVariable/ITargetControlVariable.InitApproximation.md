# ITargetControlVariable.InitApproximation


## Синтаксис


InitApproximation: Array;


## Описание


Свойство InitApproximation
 определяет начальные приближения управляющей переменной.


## Комментарии


Начальные приближения задаются в виде массива вещественных чисел.
