# ITargetControlVariable.Retrospective


## Синтаксис


Retrospective: Array;


## Описание


Свойство Retrospective
 определяет начальные значения управляющей переменной до начала расчета
 целевой задачи.


## Комментарии


Начальные значения необходимо обязательно задавать, если управляющая
 переменная входит в целевую функцию. Для задания целевой функции используйте
 свойство [ICpTargetAdjustment.CriterionFunction](../ICpTargetAdjustment/ICpTargetAdjustment.CriterionFunction.htm).


## Пример


Использование свойства приведено в примере для [ICpTargetAdjustment.CalcWithNonLinConstraints](../ICpTargetAdjustment/ICpTargetAdjustment.CalcWithNonLinConstraints.htm).
