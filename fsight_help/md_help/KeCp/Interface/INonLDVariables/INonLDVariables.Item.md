# INonLDVariables.Item


## Синтаксис


Item(Index: Integer): [INonLDVariable](../INonLDVariable/INonLDVariable.htm);


## Параметры


Index. Индекс переменной.


## Описание


Свойство Item возвращает переменную
 из коллекции по ее индексу.


## Комментарии


Свойство доступно только для чтения.

Использование свойства приведено в примере для свойства [INonLDVariable.CoefficientsOrderForestall](../INonLDVariable/INonLDVariable.CoefficientsOrderForestall.htm).
