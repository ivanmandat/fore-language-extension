# INonLDVariables


Сборка: Cp;


## Описание


Интерфейс INonLDVariables содержит
 свойства и методы для работы с коллекцией переменных нелинейной системы
 уравнений.


## Иерархия наследования


           INonLDVariables


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](INonLDVariables.Count.htm)
		 Свойство Count возвращает
		 количество переменных в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](INonLDVariables.Item.htm)
		 Свойство Item возвращает
		 переменную из коллекции по ее индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](INonLDVariables.Add.htm)
		 Метод Add осуществляет
		 добавление переменной в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](INonLDVariables.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех переменных из коллекции.


		 ![](../../Sub_Image.gif)
		 [FindById](INonLDVariables.FindById.htm)
		 Метод FindById осуществляет
		 поиск переменной в коллекции по ее идентификатору.


		 ![](../../Sub_Image.gif)
		 [Remove](INonLDVariables.Remove.htm)
		 Метод Remove осуществляет
		 удаление переменной из коллекции по ее индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveById](INonLDVariables.RemoveById.htm)
		 Метод RemoveById осуществляет
		 удаление переменной из коллекции по ее идентификатору.
