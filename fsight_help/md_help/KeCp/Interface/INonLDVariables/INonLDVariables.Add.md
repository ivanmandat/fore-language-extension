# INonLDVariables.Add


## Синтаксис


Add(Id:String): [INonLDVariable](../INonLDVariable/INonLDVariable.htm);


## Параметры


Id. Идентификатор добавляемой
 переменной.


## Описание


Метод Add осуществляет добавление
 переменной в коллекцию.

Использование метода приведено в примере для свойства [INonLDVariable.CoefficientsOrderForestall](../INonLDVariable/INonLDVariable.CoefficientsOrderForestall.htm).
