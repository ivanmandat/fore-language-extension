# ITargetPhaseVariablesArray.Add


## Синтаксис


Add(Id: String): [ITargetPhaseVariable](../ITargetPhaseVariable/ITargetPhaseVariable.htm);


## Параметры


Id. Идентификатор добавляемой
 переменной.


## Описание


Метод Add добавляет фазовую
 переменную в коллекцию.


## Комментарии


Каждая переменная в коллекции должна иметь уникальный идентификатор.
 Наличие двух переменных с одинаковым идентификатором недопустимо.
