# ITargetPhaseVariablesArray.FindById


## Синтаксис


FindById(Id: String): [ITargetPhaseVariable](../ITargetPhaseVariable/ITargetPhaseVariable.htm);


## Параметры


Id. Идентификатор искомой фазовой
 переменной.


## Описание


Метод FindById выполняет поиск
 в коллекции фазовой переменной с заданным идентификатором.


## Комментарии


Если переменная с указанным идентификатором не найдена, то метод возвращает
 значение Null.
