# ITargetPhaseVariablesArray.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс фазовой переменной.


## Описание


Метод Remove удаляет из
 коллекции фазовую переменную с указанным индексом.


## Комментарии


Индексация переменных в коллекции сквозная и начинается с нуля.


Если удаление прошло успешно, то метод возвращает значение True,
 иначе - False.
