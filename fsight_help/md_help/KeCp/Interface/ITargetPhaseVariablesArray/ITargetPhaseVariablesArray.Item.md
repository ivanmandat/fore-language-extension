# ITargetPhaseVariablesArray.Item


## Синтаксис

## Параметры


Index. Индекс переменной в
 коллекции.


## Описание


Свойство Item возвращает
 заданную фазовую переменную из коллекции.


## Комментарии


Индексация переменных в коллекции сквозная и начинается с нуля.
