# ITargetPhaseVariablesArray


Сборка: Cp;


## Описание


Интерфейс ITargetPhaseVariablesArray
 предназначен для работы с коллекцией фазовых переменных.


## Иерархия наследования


           ITargetPhaseVariablesArray


## Комментарии


Для получения коллекции фазовых переменных используйте свойство [ICpTargetAdjustment.PhaseVariables](../ICpTargetAdjustment/ICpTargetAdjustment.PhaseVariables.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITargetPhaseVariablesArray.Count.htm)
		 Свойство Count
		 возвращает количество фазовых переменных в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ITargetPhaseVariablesArray.Item.htm)
		 Свойство Item
		 возвращает заданную фазовую переменную из коллекции.
