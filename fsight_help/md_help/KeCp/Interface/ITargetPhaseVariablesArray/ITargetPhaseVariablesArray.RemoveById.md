# ITargetPhaseVariablesArray.RemoveById


## Синтаксис


RemoveById(Id: String): Boolean;


## Параметры


Index. Идентификатор фазовой
 переменной.


## Описание


Метод RemoveById удаляет из
 коллекции фазовую переменную с указанным идентификатором.


## Комментарии


Если удаление прошло успешно, то метод возвращает значение True,
 иначе - False.
