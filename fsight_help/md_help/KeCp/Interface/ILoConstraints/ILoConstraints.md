# ILoConstraints


## Описание


Интерфейс ILoConstraints содержит
 свойства и методы для работы с коллекцией ограничений линейной оптимизации.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ILoConstraints.Count.htm)
		 Свойство Count возвращает
		 количество ограничений в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ILoConstraints.Item.htm)
		 Свойство Item возвращает
		 ограничение из коллекции по его индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ILoConstraints.Add.htm)
		 Метод Add осуществляет
		 добавление ограничения в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](ILoConstraints.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех ограничений из коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](ILoConstraints.Remove.htm)
		 Метод Remove осуществляет
		 удаление ограничения из коллекции по его индексу.
