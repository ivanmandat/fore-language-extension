# ILoConstraints.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс ограничения в
 коллекции.


## Описание


Метод Remove осуществляет удаление
 ограничения из коллекции по его индексу.


## Комментарии


Метод возвращает True в случае
 удачного удаления.
