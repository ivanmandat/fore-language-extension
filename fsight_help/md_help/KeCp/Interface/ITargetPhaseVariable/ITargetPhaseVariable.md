# ITargetPhaseVariable


Сборка: Cp;


## Описание


Интерфейс ITargetPhaseVariable
 предназначен для работы с фазовой переменной.


## Иерархия наследования


           ITargetPhaseVariable


## Комментарии


Для работы со всеми фазовыми переменными используйте интерфейс [ITargetPhaseVariablesArray](../ITargetPhaseVariablesArray/ITargetPhaseVariablesArray.htm).
