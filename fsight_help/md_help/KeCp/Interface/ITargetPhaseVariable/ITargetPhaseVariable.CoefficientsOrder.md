# ITargetPhaseVariable.CoefficientsOrder


## Синтаксис


CoefficientsOrder: String;


## Описание


Свойство CoefficientsOrder
 определяет порядок коэффициентов переменной.


## Комментарии


Если коэффициентов несколько, то они указываются через точку с запятой
 «;».
