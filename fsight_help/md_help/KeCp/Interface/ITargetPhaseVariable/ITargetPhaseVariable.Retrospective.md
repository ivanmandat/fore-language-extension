# ITargetPhaseVariable.Retrospective


## Синтаксис


Retrospective: Array;


## Описание


Свойство Retrospective
 определяет начальные значения фазовой переменной до начала расчета целевой
 задачи.


## Комментарии


Начальные значения необходимо обязательно задавать, если фазовая переменная
 входит в целевую функцию. Для задания целевой функции используйте свойство
 [ICpTargetAdjustment.CriterionFunction](../ICpTargetAdjustment/ICpTargetAdjustment.CriterionFunction.htm).
