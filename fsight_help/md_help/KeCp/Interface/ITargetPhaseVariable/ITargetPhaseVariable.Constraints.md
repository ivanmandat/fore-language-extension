# ITargetPhaseVariable.Constraints


## Синтаксис


Constraints: [IVarTargetConstraintsArray](../IVarTargetConstraintsArray/IVarTargetConstraintsArray.htm);


## Описание


Свойство Constraints определяет
 коллекцию ограничений на фазовую переменную.


## Комментарии


Ограничения позволяют указать интервал, в котором должны находиться
 значения фазовой переменной. Количество ограничений должно совпадать с
 количеством начальных значений фазовой переменной.
