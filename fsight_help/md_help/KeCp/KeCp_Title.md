## С чего начать


		 В данном разделе
		 представлено описание сборки Cp
		 (Criterion problem - целевая задача). Сборка Cp
		 содержит средства, предназначенные для решения задач линейной,
		 нелинейной оптимизации и для нахождения решения системы линейных
		 уравнений.


		Изучив данный раздел, вы научитесь:


			- создавать и настраивать задачи оптимизации или системы
			 уравнений;


			- настраивать переменные, задавать для них значение;


			- производить расчет и получать результаты.


		Более подробно с возможностями сборки можно ознакомиться в разделе
		 «[Введение в сборку Cp](KeCp_About.htm)».


		Описание ресурсов сборки представлено в следующих подразделах:


			- [Интерфейсы сборки
			 Cp](Interface/KeCp_Interface.htm)


			- [Перечисления сборки Cp](Enums/KeCp_Enums.htm)


			- [Классы сборки Cp](Class/KeCp_Class.htm)


		 Перед началом работы рекомендуется
		 ознакомиться с разделами справки: «[Среда
		 разработки](UiDevEnv.chm::/01_Development_Environment/01_Purpose_of_the_constructor/Purpose_of_the_constructor.htm)», «[Руководство по языку Fore](Fore.chm::/Fore_Title.htm)».
