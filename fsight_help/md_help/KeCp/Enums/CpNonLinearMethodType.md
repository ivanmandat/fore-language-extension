# CpNonLinearMethodType


## Описание


Перечисление CpNonLinearMethodType
 содержит методы расчета задачи оптимального управления. Данное перечисление
 используется следующими свойствами и методами:


	- [ICpNonLinearOptimization.MethodType](../Interface/ICpNonLinearOptimization/ICpNonLinearOptimization.MethodType.htm);


	- [IMsOptimizationProblem.MethodType](kems.chm::/Interface/IMsOptimizationProblem/IMsOptimizationProblem.MethodType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 GridSearch. Перебор
		 по сетке. Это метод поиска по сетке, относящийся к прямым методам
		 минимизации, требует больших вычислительных затрат и для решения
		 практических задач не применяется. Результатом данного метода
		 будет экстремум целевой функции, найденный перебором по сетке,
		 количество узлов которой определяется пользователем. Точность
		 найденного решения зависит от размера сетки.


		 1
		 SequentialQP. Последовательное
		 квадратичное программирование. Это метод с квадратичными подзадачами.
		 Результатом данного метода будет локальный экстремум, определяемый
		 начальными приближениями или экстремум функции, при наличии нелинейных
		 ограничений на переменные.


		 2
		 SequentialSimplex.
		 Последовательный симплекс - это метод безусловной оптимизации.
		 Результатом данного метода будет локальный экстремум, определяемый
		 начальными приближениями.


		 3
		 DifferentEvolution.
		 Метод дифференциальной эволюции. Используется для решения задачи
		 оптимального управления.
