# TargetConstraintStatusType


## Описание


Перечисление TargetConstraintStatusType
 используется для определения состояния границы ограничения.


Используется следующими свойствами и методами:


	- [IMsTargetConstraintCommon.LowerConstraintStatus](kems.chm::/Interface/IMsTargetConstraintCommon/IMsTargetConstraintCommon.LowerConstraintStatus.htm);


	- [IMsTargetConstraintCommon.UpperConstraintStatus](kems.chm::/Interface/IMsTargetConstraintCommon/IMsTargetConstraintCommon.UpperConstraintStatus.htm);


	- [ITargetConstraintInfo.LowerConstraintStatus](../Interface/ITargetConstraintInfo/ITargetConstraintInfo.LowerConstraintStatus.htm);


	- [ITargetConstraintInfo.UpperConstraintStatus](../Interface/ITargetConstraintInfo/ITargetConstraintInfo.UpperConstraintStatus.htm);


	- [IVarTargetConstraint.LowerConstraintStatus](../Interface/IVarTargetConstraint/IVarTargetConstraint.LowerConstraintStatus.htm);


	- [IVarTargetConstraint.UpperConstraintStatus](../Interface/IVarTargetConstraint/IVarTargetConstraint.UpperConstraintStatus.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 NotReached. Граница
		 не достигнута.


		 1
		 Reached. Граница достигнута
		 и не может быть снята.


		 2
		 Disabled. Граница достигнута
		 и снята.
