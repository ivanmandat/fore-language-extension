# LinkType


## Описание


Перечисление LinkType содержит
 типы связей параметров вложенных объектов.


Используется следующими свойствами и методами:


	- [IDefParameterLink.Type](../Interface/IDefParameterLink/IDefParameterLink.Type.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 -1
		 None. Тип связи не
		 определён.


		 0
		 Values. Фиксированные
		 значения для параметра вложенного объекта.


		 1
		 Parameters. В качестве
		 значения для параметра вложенного объекта передавать значение
		 атрибута параметра формы ввода.
