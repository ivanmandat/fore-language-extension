# CalcPeriodMode


## Описание


Перечисление CalcPeriodMode
 содержит области, к которым может применяться алгоритм расчёта.


Используется следующими свойствами и методами:


	- [IDefCalculationAlgorithm.CalcPeriod](../Interface/IDefCalculationAlgorithm/IDefCalculationAlgorithm.CalcPeriod.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 IntersectionSelection.
		 Пересечение отметок алгоритма расчёта и табличной области данных.


		 2
		 FromAlgorithm. Период
		 расчёта из алгоритма.


		 3
		 AllSelectionFromAlgorithm.
		 Вся отметка из алгоритма расчёта.
