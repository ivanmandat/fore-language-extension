# emTabRowInsert


## Описание


Перечисление emTabRowInsert
 содержит варианты поведения строк/столбцов при увеличении области данных.


Используется следующими свойствами и методами:


	- [IDefTableArea.ColumnBehaviour](../Interface/IDefTableArea/IDefTableArea.ColumnBehaviour.htm);


	- [IDefTableArea.RowBehaviour](../Interface/IDefTableArea/IDefTableArea.RowBehaviour.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 fixed. Фиксированное.
		 Количество строк/столбцов не изменяется.


		 1
		 merge. Захват смежных
		 строк/столбцов. При увеличении размеров табличной области информация,
		 расположенная в строках/столбцах ниже/правее, будет перекрываться
		 данными табличной области.


		 2
		 insert. Вставка строк/столбцов.
		 При увеличении размеров табличной области будет добавлено необходимое
		 количество строк/столбцов. Тем самым, информация, расположенная
		 в строках/столбцах ниже/правее, будет смещена вниз/вправо.
