# AltHierarchySourceEnum


## Описание


Перечисление AltHierarchySourceEnum
 содержит способы задания источника альтернативной иерархии.


Используется следующими свойствами и методами:


	- [IDefDimension.HierarchyTypeSource](../Interface/IDefDimension/IDefDimension.HierarchyTypeSource.htm);


	- [IDefParameter.HierarchyTypeSource](../Interface/IDefParameter/IDefParameter.HierarchyTypeSource.htm).
