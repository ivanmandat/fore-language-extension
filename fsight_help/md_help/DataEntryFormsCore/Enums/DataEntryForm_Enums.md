# Перечисления форм ввода


		 Перечисление
		 Краткое описание


		 ![](../Enums_Image.gif)
		 [AltHierarchySourceEnum](AltHierarchySourceEnum.htm)
		 Перечисление AltHierarchySourceEnum
		 содержит способы задания источника альтернативной иерархии.


		 ![](../Enums_Image.gif)
		 [CalcParamSchema](CalcParamSchema.htm)
		 Перечисление CalcParamSchema
		 содержит варианты типа отметки, в соответствии с которым будет
		 формироваться значение, передаваемое параметру алгоритма расчёта.


		 ![](../Enums_Image.gif)
		 [CalcParamType](CalcParamType.htm)
		 Перечисление CalcParamType
		 содержит возможные типы значений, которые будут передаваться параметру
		 алгоритма расчёта.


		 ![](../Enums_Image.gif)
		 [CalcPeriodMode](CalcPeriodMode.htm)
		 Перечисление CalcPeriodMode
		 содержит области, к которым может применяться алгоритм расчёта.


		 ![](../Enums_Image.gif)
		 [CalcType](CalcType.htm)
		 Перечисление CalcType
		 содержит типы, определяющие вариант использования алгоритма
		 расчёта.


		 ![](../Enums_Image.gif)
		 [DimPlace](DimPlace.htm)
		 Перечисление DimPlace
		 содержит варианты расположения измерения.


		 ![](../Enums_Image.gif)
		 [emTabRowInsert](emTabRowInsert.htm)
		 Перечисление emTabRowInsert
		 содержит варианты поведения строк/столбцов при увеличении области
		 данных.


		 ![](../Enums_Image.gif)
		 [LinkType](LinkType.htm)
		 Перечисление LinkType
		 содержит типы связей параметров вложенных объектов.
