# DimPlace


## Описание


Перечисление DimPlace содержит
 варианты расположения измерения.


Используется следующими свойствами и методами:


	- [IDefDimension.Place](../Interface/IDefDimension/IDefDimension.Place.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 side. Заголовок строк.


		 1
		 head. Заголовок столбцов.


		 1
		 notSet. Расположение
		 не определено.


		 2
		 fixed. Фиксированные
		 измерения.


		 3
		 privateDim. Частные
		 измерения.


		 4
		 fixedPrivate. Частные
		 фиксированные измерения.
