# CalcParamType


## Описание


Перечисление CalcParamType содержит
 возможные типы значений, которые будут передаваться параметру алгоритма
 расчёта.


Используется следующими свойствами и методами:


	- [IDefCalculationAlgorithmParameter.Type](../Interface/IDefCalculationAlgorithmParameter/IDefCalculationAlgorithmParameter.Type.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 param. Значение параметра
		 или атрибута параметра формы ввода.


		 1
		 val. Константное значение.


		 2
		 dim_chandedvals. Отметка
		 измерения с изменёнными данными.
