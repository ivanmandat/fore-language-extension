# CalcParamSchema


## Описание


Перечисление CalcParamSchema
 содержит варианты типа отметки, в соответствии с которым будет формироваться
 значение, передаваемое параметру алгоритма расчёта.


Используется следующими свойствами и методами:


	- [IDefCalculationAlgorithmParameter.SelectionMode](../Interface/IDefCalculationAlgorithmParameter/IDefCalculationAlgorithmParameter.SelectionMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 selected. Отмеченные
		 элементы.


		 1
		 first. Первый элемент
		 из отмеченных.


		 2
		 last. Последний элемент
		 из отмеченных.


		 3
		 children. Отмеченные
		 элементы + непосредственные дочерние элементы.


		 4
		 childrenAll. Все дочерние
		 элементы рекурсивно.


		 5
		 onlyChildren. Только
		 непосредственные дочерние элементы без отмеченных родительских
		 элементов.


		 6
		 onlyChildrenAll. Все
		 дочерние элементы рекурсивно без отмеченных родительских элементов.
