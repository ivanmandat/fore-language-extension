# CalcType


## Описание


Перечисление CalcType содержит
 типы, определяющие вариант использования алгоритма расчёта.


Используется следующими свойствами и методами:


	- [IDefCalculationAlgorithm.Type](../Interface/IDefCalculationAlgorithm/IDefCalculationAlgorithm.Type.htm);


	- [IDefCalculationAlgorithmsCollection.Add](../Interface/IDefCalculationAlgorithmsCollection/IDefCalculationAlgorithmsCollection.Add.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 CalcAfterSave. Применять
		 алгоритм расчёта при сохранении данных в форме ввода.


		 2
		 CalcAfterEdit. Применять
		 алгоритм расчёта при изменения данных в форме ввода.
