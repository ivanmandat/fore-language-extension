## С чего начать


		 В разделе представлено
		 описание API, предназначенного для работы с формами ввода.


		Изучив данный раздел, вы сможете:


			- создавать и настраивать формы ввода;


			- работать с областями ввода, показателями, источниками
			 и измерениями;


		Более подробно с возможностями сборки можно ознакомиться в разделе
		 «[Введение в API форм
		 ввода](Intro/DataEntryFormsCore_Intro.htm)».


		Описание ресурсов сборки представлено в следующих подразделах:


			- [Интерфейсы
			 форм ввода](Interface/DataEntryForm_Interfaces.htm)


			- [Перечисления
			 форм ввода](Enums/DataEntryForm_Enums.htm)


			- [Классы форм
			 ввода](Class/DataEntryForm_Classes.htm)


		 Перед изучением API форм ввода
		 рекомендуется ознакомиться с разделами: «[Разработка
		 прикладного приложения](UiDevEnv.chm::/01_Development_Environment/01_Purpose_of_the_constructor/Purpose_of_the_constructor.htm)», «[Руководство по языку Fore](Fore.chm::/Fore_Title.htm)».
		 Также рекомендуется изучить раздел с описанием общего принципа
		 работы с расширением «[Интерактивные
		 формы ввода данных](DataEntryForms.chm::/DataEntryForms_Purpose.htm)».
