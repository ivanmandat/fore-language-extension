# IDefCalculationAlgorithm.MapInfo


## Синтаксис


		MapInfo: [IDefCalculationAlgorithmMapInfoCollection](../IDefCalculationAlgorithmMapInfoCollection/IDefCalculationAlgorithmMapInfoCollection.htm);


## Описание


Свойство MapInfo возвращает
 коллекцию привязок алгоритма расчёта к другим формам ввода.


## Комментарии


Свойство актуально, если алгоритм содержит [блок
 расчёта](IDefCalculationAlgorithm.HasCalcBlock.htm). Привязка к формам ввода используется для детализации формул,
 используемых в алгоритме расчёта.
