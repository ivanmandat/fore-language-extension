# IDefCalculationAlgorithm.HasCalcBlock


## Синтаксис


		HasCalcBlock: Boolean;


## Описание


Свойство HasCalcBlock возвращает
 признак наличия блока расчёта в алгоритме расчёта.


## Комментарии


Свойство возвращает значение True,
 если в алгоритме расчёта имеется блок расчёта и False,
 если алгоритм построен на базе других блоков.
