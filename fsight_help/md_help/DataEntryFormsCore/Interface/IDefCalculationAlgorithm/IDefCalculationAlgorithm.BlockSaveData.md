# IDefCalculationAlgorithm.BlockSaveData


## Синтаксис


		BlockSaveData: Boolean;


## Описание


Свойство BlockSaveData определяет,
 будет ли блокироваться сохранение изменённых данных при сработавшем контроле.


## Комментарии


Свойство актуально, если добавлен алгоритм расчёта с блоком контроля.


Если свойству установлено значение True,
 то сохранение изменённых данных будет заблокировано.


При значении False блокировка
 сохранения изменённых данных не производится.
