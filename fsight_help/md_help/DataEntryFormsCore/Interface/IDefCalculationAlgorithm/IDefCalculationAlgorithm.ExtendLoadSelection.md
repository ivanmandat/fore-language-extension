# IDefCalculationAlgorithm.ExtendLoadSelection


## Синтаксис


		ExtendLoadSelection: Boolean;


## Описание


Свойство ExtendLoadSelection
 определяет, будет ли расширяться отметка при применении алгоритма расчёта.


## Комментарии


Если свойству установлено значение True,
 то отметка будет расширяться таким образом, чтоб были загружены и учитывались
 все значения, указанные при настройке алгоритма, а не только выводимые
 в табличной области.


При значении False расчёт алгоритма
 производится в соответствии с теми данными, которые имеются в табличной
 области.
