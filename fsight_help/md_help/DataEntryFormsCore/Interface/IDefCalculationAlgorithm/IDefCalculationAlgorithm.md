# IDefCalculationAlgorithm


## Описание


Интерфейс IDefCalculationAlgorithm
 предназначен для настройки параметров использования алгоритма расчёта
 в табличной области.


## Иерархия наследования


IDefCalculationAlgorithm


## Комментарии


Параметры использования алгоритма расчёта можно получить, используя
 свойства и методы интерфейса [IDefCalculationAlgorithmsCollection](../IDefCalculationAlgorithmsCollection/IDefCalculationAlgorithmsCollection.htm).
