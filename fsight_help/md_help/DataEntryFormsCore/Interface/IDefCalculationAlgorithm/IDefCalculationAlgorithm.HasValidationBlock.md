# IDefCalculationAlgorithm.HasValidationBlock


## Синтаксис


		HasValidationBlock: Boolean;


## Описание


Свойство HasValidationBlock
 возвращает признак наличия блока контроля в алгоритме расчёта.


## Комментарии


Свойство возвращает значение True,
 если в алгоритме расчёта имеется блок контроля и False,
 если алгоритм построен на базе других блоков.
