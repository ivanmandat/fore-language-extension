# IDefCalculationAlgorithm.ControlStyle


## Синтаксис


		ControlStyle: [ITabCellStyle](TabSheet.chm::/Interface/ITabCellStyle/ITabCellStyle.htm);


## Описание


Свойство ControlStyle определяет
 стиль оформления, применяемый к ячейкам, в которых сработал контроль.


## Комментарии


Свойство актуально, если алгоритм содержит [блок
 контроля](IDefCalculationAlgorithm.HasValidationBlock.htm).
