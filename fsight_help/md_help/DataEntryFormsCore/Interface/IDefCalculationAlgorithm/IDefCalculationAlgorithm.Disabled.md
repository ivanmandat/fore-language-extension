# IDefCalculationAlgorithm.Disabled


## Синтаксис


		Disabled: Boolean;


## Описание


Свойство Disabled определяет,
 включено ли использование алгоритма расчёта.


## Комментарии


Допустимые значения:


	- True. Алгоритм расчёта
	 используется в табличной области;


	- False. Алгоритм расчёта
	 не используется в табличной области.
