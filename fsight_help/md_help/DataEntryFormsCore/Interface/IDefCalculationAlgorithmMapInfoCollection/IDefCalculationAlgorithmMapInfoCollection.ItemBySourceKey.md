# IDefCalculationAlgorithmMapInfoCollection.ItemBySourceKey


## Синтаксис


		ItemBySourceKey(sourceKey: Integer): [IDefCalculationAlgorithmMapInfo](../IDefCalculationAlgorithmMapInfo/IDefCalculationAlgorithmMapInfo.htm);


## Параметры


sourceKey. Ключ приёмника в
 алгоритме расчёта.


## Описание


Свойство ItemBySourceKey возвращает
 параметры привязки алгоритма расчёта к другой форме ввода по ключу приёмника.
