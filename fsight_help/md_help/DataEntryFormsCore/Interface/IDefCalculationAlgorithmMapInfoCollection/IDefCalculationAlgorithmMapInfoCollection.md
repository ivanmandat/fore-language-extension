# IDefCalculationAlgorithmMapInfoCollection


## Описание


Интерфейс IDefCalculationAlgorithmMapInfoCollection
 предназначен для работы с коллекцией привязок алгоритма расчёта к другим
 формам ввода.


## Иерархия наследования


IDefCalculationAlgorithmMapInfoCollection


## Комментарии


Привязка к формам ввода используется для детализации формул, используемых
 в алгоритме расчёта. Коллекцию привязок к формам ввода можно получить,
 используя свойство [IDefCalculationAlgorithm.MapInfo](../IDefCalculationAlgorithm/IDefCalculationAlgorithm.MapInfo.htm).
