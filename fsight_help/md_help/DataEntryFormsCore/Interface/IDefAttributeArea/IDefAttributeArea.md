# IDefAttributeArea


## Описание


Интерфейс IDefAttributeArea
 предназначен для управления атрибутной областью.


## Иерархия наследования


           [IDefInputArea](../IDefInputArea/IDefInputArea.htm)


           IDefAttributeArea


## Комментарии


Атрибутную область можно получить, используя методы интерфейса [IDefInputAreas](../IDefInputAreas/IDefInputAreas.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AdvancedFilters](IDefAttributeArea.AdvancedFilters.htm)
		 Свойство AdvancedFilters
		 возвращает коллекцию расширенных фильтров.


		 ![](../../Property_Image.gif)
		 [AttributeKey](IDefAttributeArea.AttributeKey.htm)
		 Свойство AttributeKey
		 определяет ключ атрибута, значения которого будут выводиться в
		 таблице атрибутной области.


		 ![](../../Property_Image.gif)
		 [ConsiderDefault](IDefAttributeArea.ConsiderDefault.htm)
		 Свойство ConsiderDefault
		 определяет, будет ли сохраняться значение по умолчанию в качестве
		 значения атрибута при создании нового элемента.


		 ![](../../Property_Image.gif)
		 [DefaultValue](IDefAttributeArea.DefaultValue.htm)
		 Свойство DefaultValue
		 определяет значение по умолчанию.


		 ![](../../Property_Image.gif)
		 [DictionaryKey](IDefAttributeArea.DictionaryKey.htm)
		 Свойство DictionaryKey
		 определяет ключ справочника, значения которого будут отображаться
		 в арибутной области.


		 ![](../../Property_Image.gif)
		 [ParameterAttributeKey](IDefAttributeArea.ParameterAttributeKey.htm)
		 Свойство ParameterAttributeKey
		 определяет ключ атрибута параметра, с которым будет связана атрибутная
		 область.


		 ![](../../Property_Image.gif)
		 [ParameterKey](IDefAttributeArea.ParameterKey.htm)
		 Свойство ParameterKey
		 определяет ключ параметра, с которым будет связана атрибутная
		 область.


		 ![](../../Property_Image.gif)
		 [ReadOnly](IDefAttributeArea.ReadOnly.htm)
		 Свойство ReadOnly определяет
		 признак доступности атрибутной области только для чтения.
