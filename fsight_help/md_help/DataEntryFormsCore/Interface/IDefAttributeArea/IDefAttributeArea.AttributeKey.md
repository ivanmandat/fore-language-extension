# IDefAttributeArea.AttributeKey


## Синтаксис


		AttributeKey: Integer;


## Описание


Свойство AttributeKey определяет
 ключ атрибута, значения которого будут выводиться в таблице атрибутной
 области.


## Комментарии


В качестве значения свойства указывается атрибут справочника, который
 определён в свойстве [IDefAttributeArea.DictionaryKey](IDefAttributeArea.DictionaryKey.htm).
