# IDefAttributeArea.DefaultValue


## Синтаксис


		DefaultValue: Variant;


## Описание


Свойство DefaultValue определяет
 значение по умолчанию.


## Комментарии


Тип передаваемого значения должен совпадать с типом указанного в [AttributeKey](IDefAttributeArea.AttributeKey.htm) атрибута
 . Для сброса значения по умолчанию задайте значение Null
 или пустую строку.
