# IDefAttributeArea.ReadOnly


## Синтаксис


		ReadOnly: Boolean;


## Описание


Свойство ReadOnly определяет
 признак доступности атрибутной области только для чтения.


## Комментарии


Допустимые значения:


	- True. Атрибутная область
	 доступна только для чтения;


	- False. Атрибутная область
	 доступна для редактирования.
