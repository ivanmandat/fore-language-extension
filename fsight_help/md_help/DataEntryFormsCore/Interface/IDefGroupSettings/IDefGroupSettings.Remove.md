# IDefGroupSettings.Remove


## Синтаксис


Remove(dimKey: Integer);


## Параметры


dimKey. Ключ измерения боковика,
 настройки группировки которого нужно удалить.


## Описание


Метод Remove удаляет из коллекции
 настройки группировки для измерения с указанным ключом.
