# IDefGroupSettings


## Описание


Интерфейс IDefGroupSettings
 предназначен для настройки группировок измерений боковика.


## Иерархия наследования


IDefGroupSettings


## Комментарии


Настройки группировки возвращает свойство [IDefTableArea.GroupSettings](../IDefTableArea/IDefTableArea.GroupSettings.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDefGroupSettings.Count.htm)
		 Свойство Count возвращает
		 количество настроек группировок в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IDefGroupSettings.Item.htm)
		 Свойство Item возвращает
		 настройки группировки с указанным индексом.


		 ![](../../Property_Image.gif)
		 [ItemByDimKey](IDefGroupSettings.ItemByDimKey.htm)
		 Свойство ItemByDimKey
		 возвращает настройки группировки для измерения с указанным ключом.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IDefGroupSettings.Add.htm)
		 Метод Add для указанного
		 измерения добавляет настройки группировки.


		 ![](../../Sub_Image.gif)
		 [Clear](IDefGroupSettings.Clear.htm)
		 Метод Clear удаляет
		 все настройки группировки из коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IDefGroupSettings.Remove.htm)
		 Метод Remove удаляет
		 из коллекции настройки группировки для измерения с указанным ключом.
