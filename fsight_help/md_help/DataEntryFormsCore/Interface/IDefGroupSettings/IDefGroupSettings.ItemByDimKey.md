# IDefGroupSettings.ItemByDimKey


## Синтаксис


ItemByDimKey(dimKey: Integer): [IDefGroupSetting](../IDefGroupSetting/IDefGroupSetting.htm);


## Параметры


dimKey. Ключ измерения боковика,
 настройки группировки которого нужно получить.


## Описание


Свойство ItemByDimKey возвращает
 настройки группировки для измерения с указанным ключом.
