# IDefDrillDimensionParameterValues


## Описание


Интерфейс IDefDrillDimensionParameterValues
 предназначен для работы с коллекцией настроек управления параметрами объекта,
 открываемого при расшифровке.


## Иерархия наследования


IDefDrillDimensionParameterValues


## Комментарии


Настройки параметров можно получить, используя свойство [IDefDrillDimension.ParameterValues](../IDefDrillDimension/IDefDrillDimension.ParameterValues.htm)
 или [IDefDrillDimensionLevel.ParameterValues](../IDefDrillDimensionLevel/IDefDrillDimensionLevel.ParameterValues.htm).
