# IDefParametersApplyResult.Success


## Синтаксис


		Success: Boolean;


## Описание


Свойство Success возвращает
 признак успешности применения изменения настроек параметров.


## Комментарии


Свойство возвращает значение True,
 если изменения в настройках были применены успешно и False,
 если возникли какие-либо ошибки.
