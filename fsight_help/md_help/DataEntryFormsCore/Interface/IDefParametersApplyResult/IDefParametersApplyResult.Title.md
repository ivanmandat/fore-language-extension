# IDefParametersApplyResult.Title


## Синтаксис


		Title: String;


## Описание


Свойство Title возвращает заголовок
 сообщения об ошибке.


## Комментарии


Свойство актуально, если свойство [Success](IDefParametersApplyResult.Success.htm)
 возвращает значение False.
