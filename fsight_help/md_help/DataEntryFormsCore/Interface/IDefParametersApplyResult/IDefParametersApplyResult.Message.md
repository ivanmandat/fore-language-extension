# IDefParametersApplyResult.Message


## Синтаксис


		Message: String;


## Описание


Свойство Message возвращает
 сообщение об ошибке, которая возникла во время применения настроек параметров.


## Комментарии


Свойство актуально, если свойство [Success](IDefParametersApplyResult.Success.htm)
 возвращает значение False.
