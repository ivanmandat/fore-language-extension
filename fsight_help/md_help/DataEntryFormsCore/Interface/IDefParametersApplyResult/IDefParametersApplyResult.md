# IDefParametersApplyResult


## Описание


Интерфейс IDefParametersApplyResult
 предназначен для получения информации о результате применения изменений
 настроек параметров.


## Иерархия наследования


IDefParametersApplyResult


## Комментарии


Результаты применения изменений настроек параметров возвращает метод
 [IDefParameters.ApplyChanges](../IDefParameters/IDefParameters.ApplyChanges.htm).
