# IDefParameterLinkAttributeInfo


## Описание


Интерфейс IDefParameterLinkAttributeInfo
 предназначен для настройки связи с атрибутом параметра формы ввода.


## Иерархия наследования


IDefParameterLinkAttributeInfo


## Комментарии


Настройки связи с атрибутом параметра формы ввода можно получить, используя
 свойство [IDefParameterLink.Attribute](../IDefParameterLink/IDefParameterLink.Attribute.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AttributeKey](IDefParameterLinkAttributeInfo.AttributeKey.htm)
		 Свойство AttributeKey
		 определяет ключ атрибута параметра, значения которого будут передаваться
		 в параметр вложенного объекта.


		 ![](../../Property_Image.gif)
		 [ParameterId](IDefParameterLinkAttributeInfo.ParameterId.htm)
		 Свойство ParameterId
		 определяет идентификатор параметра формы ввода, с которым осуществляется
		 связь.
