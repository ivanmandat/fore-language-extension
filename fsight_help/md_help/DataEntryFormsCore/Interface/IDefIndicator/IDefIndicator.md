# IDefIndicator


## Описание


Интерфейс IDefIndicator предназначен
 для настройки параметров показателя.


## Иерархия наследования


IDefIndicator


## Комментарии


Параметры показателя можно получить, используя свойства и методы интерфейса
 [IDefIndicators](../IDefIndicators/IDefIndicators.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DefDimensions](IDefIndicator.DefDimensions.htm)
		 Свойство DefDimensions
		 возвращает коллекцию частных измерений показателя.


		 ![](../../Property_Image.gif)
		 [Name](IDefIndicator.Name.htm)
		 Свойство Name определяет
		 наименование показателя.


		 ![](../../Property_Image.gif)
		 [ReadOnly](IDefIndicator.ReadOnly.htm)
		 Свойство ReadOnly определяет,
		 будет ли возможность редактировать данные показателя.


		 ![](../../Property_Image.gif)
		 [Unical](IDefIndicator.Unical.htm)
		 Свойство Unical определяет
		 уникальный ключ показателя.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [SetAttributes](IDefIndicator.SetAttributes.htm)
		 Метод SetAttributes
		 задаёт список атрибутов, отображаемых для показателя.
