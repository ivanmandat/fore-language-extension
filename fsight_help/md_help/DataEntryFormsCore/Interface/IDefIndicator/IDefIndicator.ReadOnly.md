# IDefIndicator.ReadOnly


## Синтаксис


ReadOnly: Boolean;


## Описание


Свойство ReadOnly определяет,
 будет ли возможность редактировать данные показателя.


## Комментарии


Допустимые значения:


	- True. Редактирование
	 данных запрещено;


	- False. Значение по умолчанию.
	 Редактирование данных разрешено.
