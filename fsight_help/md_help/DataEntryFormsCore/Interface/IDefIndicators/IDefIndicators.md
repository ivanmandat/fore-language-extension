# IDefIndicators


## Описание


Интерфейс IDefIndicators предназначен
 для работы с коллекцией показателей табличной области.


## Иерархия наследования


IDefIndicators


## Комментарии


Коллекцию показателей можно получить в свойстве [IDefTableArea.DefIndicators](../IDefTableArea/IDefTableArea.DefIndicators.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDefIndicators.Count.htm)
		 Свойство Count возвращает
		 количество показателей в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IDefIndicators.Item.htm)
		 Свойство Item возвращает
		 параметры показателя с указанным индексом.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddItem](IDefIndicators.AddItem.htm)
		 Метод AddItem создаёт
		 новый показатель.


		 ![](../../Sub_Image.gif)
		 [GetIndicatorByUnical](IDefIndicators.GetIndicatorByUnical.htm)
		 Метод GetIndicatorByUnical
		 возвращает параметры показателя с заданным уникальным ключом.


		 ![](../../Sub_Image.gif)
		 [RemoveItem](IDefIndicators.RemoveItem.htm)
		 Метод RemoveItem удаляет
		 показатель из коллекции по уникальному ключу.


		 ![](../../Sub_Image.gif)
		 [SetIndicatorAsFilter](IDefIndicators.SetIndicatorAsFilter.htm)
		 Метод SetIndicatorAsFilter
		 устанавливает показатель с указанным уникальным ключом в качестве
		 фильтра.


		 ![](../../Sub_Image.gif)
		 [Swap](IDefIndicators.Swap.htm)
		 Метод Swap меняет местами
		 два указанных показателя.
