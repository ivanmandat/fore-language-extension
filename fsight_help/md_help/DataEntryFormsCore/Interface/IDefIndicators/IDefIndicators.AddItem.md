# IDefIndicators.AddItem


## Синтаксис


		AddItem(cubeKey: Integer; [isFilter: Boolean
		 = False]): [IDefIndicator](../IDefIndicator/IDefIndicator.htm);


## Параметры


cubeKey. Ключ куба, на базе
 которого добавляется показатель.


isFilter. Признак добавления
 показателя в качестве фильтра.


## Описание


Метод AddItem создаёт новый
 показатель.
