# IDefIndicators.GetIndicatorByUnical


## Синтаксис


		GetIndicatorByUnical(unical: Integer): [IDefIndicator](../IDefIndicator/IDefIndicator.htm);


## Параметры


unical.
 Уникальный ключ показателя.


## Описание


Метод GetIndicatorByUnical возвращает
 параметры показателя с заданным уникальным ключом.


## Комментарии


Метод возвращает значение Null,
 если показателя с указанным уникальным ключом не существует.
