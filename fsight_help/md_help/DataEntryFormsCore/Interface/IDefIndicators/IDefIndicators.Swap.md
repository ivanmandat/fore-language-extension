# IDefIndicators.Swap


## Синтаксис


		Swap(cubeUnical1: Integer; cubeUnical2:
		 Integer);


## Параметры


cubeUnical1. Уникальный ключ
 первого показателя.


cubeUnical2. Уникальный ключ
 второго показателя.


## Описание


Метод Swap меняет местами два
 указанных показателя.
