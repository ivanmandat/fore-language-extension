# IDefIndicators.SetIndicatorAsFilter


## Синтаксис


SetIndicatorAsFilter([unical: Integer = -1]);


## Параметры


unical. Уникальный ключ показателя.


## Описание


Метод SetIndicatorAsFilter устанавливает
 показатель с указанным уникальным ключом в качестве фильтра.


## Комментарии


По умолчанию в качестве значения параметра unical
 передаётся значение -1, при этом фильтр снимается.
