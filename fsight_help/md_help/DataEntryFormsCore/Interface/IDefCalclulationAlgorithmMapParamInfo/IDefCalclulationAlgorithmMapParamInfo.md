# IDefCalclulationAlgorithmMapParamInfo


## Описание


Интерфейс IDefCalclulationAlgorithmMapParamInfo
 предназначен для настройки привязки параметра текущей формы и формы, с
 которой связан алгоритм расчёта.


## Иерархия наследования

## Комментарии
