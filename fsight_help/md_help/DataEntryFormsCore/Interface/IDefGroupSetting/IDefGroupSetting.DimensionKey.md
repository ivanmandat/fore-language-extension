# IDefGroupSetting.DimensionKey


## Синтаксис


DimensionKey: Integer;


## Описание


Свойство DimensionKey возвращает
 ключ измерения, для которого задаются настройки группировки.


## Комментарии


Свойство вернёт ключ измерения, который был указан при выполнении метода
 [IDefGroupSettings.Add](../IDefGroupSettings/IDefGroupSettings.Add.htm).
