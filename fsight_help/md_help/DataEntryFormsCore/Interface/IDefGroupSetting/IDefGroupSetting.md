# IDefGroupSetting


## Описание


Интерфейс IDefGroupSetting предназначен
 для настройки группировки измерения боковика.


## Иерархия наследования


IDefGroupSetting


## Комментарии


Настройки группировки измерения можно получить, используя свойства и
 методы интерфейса [IDefGroupSettings](../IDefGroupSettings/IDefGroupSettings.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attributes](IDefGroupSetting.Attributes.htm)
		 Свойство Attributes
		 определяет коллекцию атрибутов, по значениям которых будет осуществляться
		 группировка элементов измерения.


		 ![](../../Property_Image.gif)
		 [CreateElemForEmptyAttr](IDefGroupSetting.CreateElemForEmptyAttr.htm)
		 Свойство CreateElemForEmptyAttr
		 определят признак создания группировочного элемента при пустых
		 значениях атрибутов.


		 ![](../../Property_Image.gif)
		 [DimensionKey](IDefGroupSetting.DimensionKey.htm)
		 Свойство DimensionKey
		 возвращает ключ измерения, для которого задаются настройки группировки.


		 ![](../../Property_Image.gif)
		 [LevelNumber](IDefGroupSetting.LevelNumber.htm)
		 Свойство LevelNumber
		 определяет номер уровня измерения, для элементов которого будет
		 включена группировка.
