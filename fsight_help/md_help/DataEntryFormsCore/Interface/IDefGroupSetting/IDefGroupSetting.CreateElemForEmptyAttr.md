# IDefGroupSetting.CreateElemForEmptyAttr


## Синтаксис


CreateElemForEmptyAttr: Boolean;


## Описание


Свойство CreateElemForEmptyAttr
 определят признак создания группировочного элемента при пустых значениях
 атрибутов.


## Комментарии


Допустимые значения:


	- True. Создавать группировочный
	 элемент, если у элемента пустое значение атрибутов;


	- False. Не создавать
	 группировочный элемент, если у элемента пустое значение атрибутов.
