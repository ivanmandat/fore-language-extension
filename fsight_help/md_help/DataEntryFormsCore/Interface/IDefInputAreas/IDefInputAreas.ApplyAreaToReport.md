# IDefInputAreas.ApplyAreaToReport


## Синтаксис


		ApplyAreaToReport(areaId: String);


## Параметры


areaId. Идентификатор области
 ввода.


## Описание


Метод ApplyAreaToReport применяет
 изменения в структуре области ввода с указанными идентификатором к форме
 ввода.


## Комментарии


Данный метод необходимо вызвать после внесения изменений в коллекции
 областей ввода перед завершением работы с формой ввода.
