# IDefInputAreas.CreateTableArea


## Синтаксис


		CreateTableArea(sheetKey: Integer; row:
		 Integer; column: Integer): [IDefTableArea](../IDefInputArea/IDefInputArea.htm);


## Параметры


sheetKey. Ключ листа, на который
 будет добавлена новая табличная область;


row. Индекс строки, начиная
 с которой будет располагаться табличная область;


column. Индекс столбца, начиная
 с которого будет располагаться табличная область.


## Описание


Метод CreateTableArea создаёт
 табличную область и возвращает её параметры.
