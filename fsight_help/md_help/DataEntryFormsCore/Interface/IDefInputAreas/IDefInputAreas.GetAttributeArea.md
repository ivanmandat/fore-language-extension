# IDefInputAreas.GetAttributeArea


## Синтаксис


		GetAttributeArea(areaId): [IDefAttributeArea](../IDefAttributeArea/IDefAttributeArea.htm);


## Параметры


areaId. Идентификатор атрибутной
 области.


## Описание


Метод GetAttributeArea возвращает
 параметры атрибутной области с заданным идентификатором.


## Комментарии


Метод возвращает значение Null,
 если атрибутной области с указанными идентификатором не существует.
