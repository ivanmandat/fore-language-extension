# IDefInputAreas


## Описание


Интерфейс IDefInputAreas предназначен
 для работы с коллекцией областей ввода.


## Иерархия наследования


IDefInputAreas


## Комментарии


Коллекцию областей ввода можно получить в свойстве [IDataEntryForm.InputAreas](../IDataEntryForm/IDataEntryForm.InputAreas.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDefInputAreas.Count.htm)
		 Свойство Count возвращает
		 количество областей ввода.


		 ![](../../Property_Image.gif)
		 [Item](IDefInputAreas.Item.htm)
		 Свойство Item возвращает
		 параметры области ввода с указанным индексом.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ApplyAreaToReport](IDefInputAreas.ApplyAreaToReport.htm)
		 Метод ApplyAreaToReport
		 применяет изменения в структуре области ввода с указанными идентификатором
		 к форме ввода.


		 ![](../../Sub_Image.gif)
		 [CreateAttributeArea](IDefInputAreas.CreateAttributeArea.htm)
		 Метод CreateAttributeArea
		 создаёт атрибутную область и возвращает её параметры.


		 ![](../../Sub_Image.gif)
		 [CreateTableArea](IDefInputAreas.CreateTableArea.htm)
		 Метод CreateTableArea
		 создаёт табличную область и возвращает её параметры.


		 ![](../../Sub_Image.gif)
		 [GetAttributeArea](IDefInputAreas.GetAttributeArea.htm)
		 Метод GetAttributeArea
		 возвращает параметры атрибутной области с заданным идентификатором.


		 ![](../../Sub_Image.gif)
		 [GetTableArea](IDefInputAreas.GetTableArea.htm)
		 Метод GetTableArea
		 возвращает параметры табличной области с заданным идентификатором.


		 ![](../../Sub_Image.gif)
		 [RemoveArea](IDefInputAreas.RemoveArea.htm)
		 Метод RemoveArea удаляет
		 область ввода с указанным идентификатором.


		 ![](../../Sub_Image.gif)
		 [TransposeTableAreaWithGrid](IDefInputAreas.TransposeTableAreaWithGrid.htm)
		 Метод TransposeTableAreaWithGrid
		 транспонирует указанную табличную область и применяет изменения
		 к отчёту формы ввода.
