# IDefInputAreas.RemoveArea


## Синтаксис


		RemoveArea(areaId: String): Boolean;


## Параметры


areaId. Идентификатор области
 ввода.


## Описание


Метод RemoveArea удаляет область
 ввода с указанным идентификатором.


## Комментарии


Метод возвращает значение True,
 если удаление завершилось успешно и False,
 если область ввода по каким-либо причинам не была удалена.
