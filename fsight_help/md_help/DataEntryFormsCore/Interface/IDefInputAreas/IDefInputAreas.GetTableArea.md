# IDefInputAreas.GetTableArea


## Синтаксис


GetTableArea(areaId: String): [IDefTableArea](../IDefTableArea/IDefTableArea.htm);


## Параметры


areaId.
 Идентификатор табличной области.


## Описание


Метод GetTableArea возвращает
 параметры табличной области с заданным идентификатором.


## Комментарии


Метод возвращает значение Null,
 если табличной области с указанными идентификатором не существует.
