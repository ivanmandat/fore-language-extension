# IDefSources


## Описание


Интерфейс IDefSources предназначен
 для работы с коллекцией источников табличной области.


## Иерархия наследования


IDefSources


## Комментарии


Коллекцию источников можно получить в свойстве [IDefTableArea.DefSources](../IDefTableArea/IDefTableArea.DefSources.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDefSources.Count.htm)
		 Свойство Count возвращает
		 количество источников в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IDefSources.Item.htm)
		 Свойство Item возвращает
		 описание источника.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddItem](IDefSources.AddItem.htm)
		 Метод AddItem добавляет
		 куб с указанным ключом в качестве источника для табличной области.


		 ![](../../Sub_Image.gif)
		 [RemoveItem](IDefSources.RemoveItem.htm)
		 Метод RemoveItem удаляет
		 указанный куб-источник из коллекции.
