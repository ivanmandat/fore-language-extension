# IDefSources.AddItem


## Синтаксис


		AddItem(cubeKey: Integer): Boolean;


## Параметры


cubeKey. Ключ куба в репозитории.


## Описание


Метод AddItem добавляет куб
 с указанным ключом в качестве источника для табличной области.
