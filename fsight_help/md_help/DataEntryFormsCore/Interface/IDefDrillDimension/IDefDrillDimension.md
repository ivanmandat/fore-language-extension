# IDefDrillDimension


## Описание


Интерфейс IDefDrillDimension
 предназначен для настройки расшифровки для измерения.


## Иерархия наследования


IDefDrillDimension


## Комментарии


Настройки расшифровки можно получить, используя свойство [IDefDimension.Drill](../IDefDimension/IDefDimension.Drill.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CreateElement](IDefDrillDimension.CreateElement.htm)
		 Свойство CreateElement
		 определяет, будет ли доступна возможность добавления и удаления
		 элементов.


		 ![](../../Property_Image.gif)
		 [DrillDescriptor](IDefDrillDimension.DrillDescriptor.htm)
		 Свойство DrillDescriptor
		 определяет описание объекта репозитория, который будет открыт
		 для расшифровки элемента измерения.


		 ![](../../Property_Image.gif)
		 [DrillLevels](IDefDrillDimension.DrillLevels.htm)
		 Свойство DrillLevels
		 возвращает настройки расшифровки для уровней измерения.


		 ![](../../Property_Image.gif)
		 [LevelIndex](IDefDrillDimension.LevelIndex.htm)
		 Свойство LevelIndex
		 возвращает индекс уровня измерения, для которого настраивается
		 расшифровка.


		 ![](../../Property_Image.gif)
		 [OpenOnSheetKey](IDefDrillDimension.OpenOnSheetKey.htm)
		 Свойство OpenOnSheetKey
		 определяет ключ листа, на котором откроется форма ввода при расшифровке.


		 ![](../../Property_Image.gif)
		 [ParameterValues](IDefDrillDimension.ParameterValues.htm)
		 Свойство ParameterValues
		 возвращает настройки для управления параметрами открываемого объекта.


		 ![](../../Property_Image.gif)
		 [RefreshAfterCreationClose](IDefDrillDimension.RefreshAfterCreationClose.htm)
		 Свойство RefreshAfterCreationClose
		 определяет, будет ли производиться обновление табличной области
		 после закрытия объекта для расшифровки.


		 ![](../../Property_Image.gif)
		 [UserCaption](IDefDrillDimension.UserCaption.htm)
		 Свойство UserCaption
		 определяет пользовательское наименование для кнопки расшифровки.
