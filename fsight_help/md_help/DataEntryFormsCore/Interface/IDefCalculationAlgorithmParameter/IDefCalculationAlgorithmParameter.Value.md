# IDefCalculationAlgorithmParameter.Value


## Синтаксис


		Value: Variant;


## Описание


Свойство Value определяет константное
 значение, передаваемое параметру алгоритма расчёта.


## Комментарии


Свойство используется, если в свойстве [Type](IDefCalculationAlgorithmParameter.Type.htm)
 установлено значение [CalcParamType.val](../../Enums/CalcParamType.htm)
 или [CalcParamType.dim_chandedvals](../../Enums/CalcParamType.htm).
 В качестве значения указывается ключ элемента или отметка справочника,
 с которым связан параметр алгоритма расчёта.
