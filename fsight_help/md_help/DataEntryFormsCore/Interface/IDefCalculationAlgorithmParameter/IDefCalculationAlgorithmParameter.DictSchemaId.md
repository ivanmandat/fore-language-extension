# IDefCalculationAlgorithmParameter.DictSchemaId


## Синтаксис


		DictSchemaId: String;


## Описание


Свойство DictSchemaId определяет
 идентификатор схемы отметки, которая будет применяться к элементам справочника,
 связанного с параметром алгоритма расчёта.
