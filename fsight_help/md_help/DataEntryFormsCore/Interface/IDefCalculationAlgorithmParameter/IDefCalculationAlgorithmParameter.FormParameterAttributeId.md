# IDefCalculationAlgorithmParameter.FormParameterAttributeId


## Синтаксис


		FormParameterAttributeId: String;


## Описание


Свойство FormParameterAttributeId
 определяет идентификатор атрибута параметра формы ввода, значения которого
 будут передаваться в качестве значения параметра алгоритма расчёта.


## Комментарии


Свойство используется, если в свойстве [Type](IDefCalculationAlgorithmParameter.Type.htm)
 установлено значение [CalcParamType.param](../../Enums/CalcParamType.htm).
