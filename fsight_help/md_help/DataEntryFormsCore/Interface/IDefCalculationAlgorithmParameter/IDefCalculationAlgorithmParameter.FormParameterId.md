# IDefCalculationAlgorithmParameter.FormParameterId


## Синтаксис


		FormParameterId: String;


## Описание


Свойство FormParameterId определяет
 идентификатор параметра формы ввода, который будет сопоставлен с параметром
 алгоритма расчёта.


## Комментарии


Свойство используется, если в свойстве [Type](IDefCalculationAlgorithmParameter.Type.htm)
 установлено значение [CalcParamType.param](../../Enums/CalcParamType.htm).
