# IDefCalculationAlgorithmParameter


## Описание


Интерфейс IDefCalculationAlgorithmParameter
 предназначен для настройки привязки параметра алгоритма расчёта к параметру
 формы ввода.


## Иерархия наследования


IDefCalculationAlgorithmParameter


## Комментарии


Привязку параметров можно получить, используя свойства интерфейса [IDefCalculationAlgorithmParametersCollection](../IDefCalculationAlgorithmParametersCollection/IDefCalculationAlgorithmParametersCollection.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AlgorithmParameterId](IDefCalculationAlgorithmParameter.AlgorithmParameterId.htm)
		 Свойство AlgorithmParameterId
		 возвращает идентификатор параметра алгоритма расчёта, для которого
		 настраивается привязка.


		 ![](../../Property_Image.gif)
		 [AlgorithmParameterName](IDefCalculationAlgorithmParameter.AlgorithmParameterName.htm)
		 Свойство AlgorithmParameterName
		 возвращает наименование параметра алгоритма расчёта, для которого
		 настраивается привязка.


		 ![](../../Property_Image.gif)
		 [DictSchemaId](IDefCalculationAlgorithmParameter.DictSchemaId.htm)
		 Свойство DictSchemaId
		 определяет идентификатор схемы отметки, которая будет применяться
		 к элементам справочника, связанного с параметром алгоритма расчёта.


		 ![](../../Property_Image.gif)
		 [FormParameterAttributeId](IDefCalculationAlgorithmParameter.FormParameterAttributeId.htm)
		 Свойство FormParameterAttributeId
		 определяет идентификатор атрибута параметра формы ввода, значения
		 которого будут передаваться в качестве значения параметра алгоритма
		 расчёта.


		 ![](../../Property_Image.gif)
		 [FormParameterId](IDefCalculationAlgorithmParameter.FormParameterId.htm)
		 Свойство FormParameterId
		 определяет идентификатор параметра формы ввода, который будет
		 сопоставлен с параметром алгоритма расчёта.


		 ![](../../Property_Image.gif)
		 [SelectionMode](IDefCalculationAlgorithmParameter.SelectionMode.htm)
		 Свойство SelectionMode
		 определяет способ отметки значения при передаче значения в параметр
		 области расчёта.


		 ![](../../Property_Image.gif)
		 [Type](IDefCalculationAlgorithmParameter.Type.htm)
		 Свойство Type определяет
		 тип значения, передаваемого параметру алгоритма расчёта.


		 ![](../../Property_Image.gif)
		 [Value](IDefCalculationAlgorithmParameter.Value.htm)
		 Свойство Value определяет
		 константное значение, передаваемое параметру алгоритма расчёта.
