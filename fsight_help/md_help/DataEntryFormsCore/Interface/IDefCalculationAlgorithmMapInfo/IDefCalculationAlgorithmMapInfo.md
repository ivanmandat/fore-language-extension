# IDefCalculationAlgorithmMapInfo


## Описание


Интерфейс IDefCalculationAlgorithmMapInfo
 предназначен для настройки параметров привязки алгоритма расчёта к другой
 форме ввода.


## Иерархия наследования


IDefCalculationAlgorithmMapInfo


## Комментарии


Параметры привязки можно получить, используя свойства интерфейса [IDefCalculationAlgorithmMapInfoCollection](../IDefCalculationAlgorithmMapInfoCollection/IDefCalculationAlgorithmMapInfoCollection.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [FormKey](IDefCalculationAlgorithmMapInfo.FormKey.htm)
		 Свойство FormKey определяет
		 ключ формы ввода, с которой будет связан алгоритм расчёта.


		 ![](../../Property_Image.gif)
		 [MapParameters](IDefCalculationAlgorithmMapInfo.MapParameters.htm)
		 Свойство MapParameters
		 возвращает коллекцию привязок параметров текущей формы ввода к
		 параметрам формы ввода, которая привязана к алгоритму расчёта.


		 ![](../../Property_Image.gif)
		 [SourceKey](IDefCalculationAlgorithmMapInfo.SourceKey.htm)
		 Свойство SourceKey
		 возвращает ключ приёмника алгоритма расчёта, связываемого с формой
		 ввода.
