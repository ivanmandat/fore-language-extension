# IDefParameters.BeginUpdate


## Синтаксис


		BeginUpdate;


## Описание


Метод BeginUpdate блокирует
 коллекцию параметров для редактирования.


## Комментарии


По окончанию редактирования коллекции можно будет применить изменения
 с помощью метода [ApplyChanges](IDefParameters.ApplyChanges.htm)
 или отменить их с помощью метода [CancelChanges](IDefParameters.CancelChanges.htm).
