# IDefParameters


## Описание


Интерфейс IDefParameters предназначен
 для работы с коллекцией параметров формы ввода.


## Иерархия наследования


IDefParameters


## Комментарии


Коллекцию параметров можно получить, используя свойство [IDataEntryForm.Parameters](../IDataEntryForm/IDataEntryForm.Parameters.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDefParameters.Count.htm)
		 Свойство Count возвращает
		 количество параметров в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IDefParameters.Item.htm)
		 Свойство Item возвращает
		 настройки параметра с указанным индексом.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IDefParameters.Add.htm)
		 Метод Add создаёт новый
		 параметр формы ввода и возвращает его настройки.


		 ![](../../Sub_Image.gif)
		 [ApplyChanges](IDefParameters.ApplyChanges.htm)
		 Метод ApplyChanges
		 применяет изменения настроек параметров к форме ввода.


		 ![](../../Sub_Image.gif)
		 [BeginUpdate](IDefParameters.BeginUpdate.htm)
		 Метод BeginUpdate блокирует
		 коллекцию параметров для редактирования.


		 ![](../../Sub_Image.gif)
		 [CancelChanges](IDefParameters.CancelChanges.htm)
		 Метод CancelChanges
		 отменяет изменения настроек параметров.


		 ![](../../Sub_Image.gif)
		 [Remove](IDefParameters.Remove.htm)
		 Метод Remove удаляет
		 параметр с указанным ключом.


		 ![](../../Sub_Image.gif)
		 [Swap](IDefParameters.Swap.htm)
		 Метод Swap меняет местами
		 два указанных параметра.
