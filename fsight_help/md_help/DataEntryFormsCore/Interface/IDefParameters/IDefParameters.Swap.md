# IDefParameters.Swap


## Синтаксис


		Swap(Key1: Integer; Key2: Integer);


## Параметры


Key1. Ключ первого параметра.


Key2. Ключ второго параметра.


## Описание


Метод Swap меняет местами два
 указанных параметра.
