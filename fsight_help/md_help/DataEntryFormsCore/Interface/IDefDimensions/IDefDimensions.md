# IDefDimensions


## Описание


Интерфейс IDefDimensions предназначен
 для работы с коллекцией измерений табличной области.


## Иерархия наследования


IDefDimensions


## Комментарии


Коллекцию измерений можно получить в свойстве [IDefTableArea.DefDimensions](../IDefTableArea/IDefTableArea.DefDimensions.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDefDimensions.Count.htm)
		 Свойство Count возвращает
		 количество измерений в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IDefDimensions.Item.htm)
		 Свойство Item возвращает
		 параметры измерения с указанным индексом.


		 ![](../../Property_Image.gif)
		 [ItemByKey](IDefDimensions.ItemByKey.htm)
		 Свойство ItemByKey
		 возвращает параметры измерения по ключу справочника.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddItem](IDefDimensions.AddItem.htm)
		 Метод AddItem добавляет
		 в коллекцию измерение на базе справочника репозитория с указанным
		 ключом.


		 ![](../../Sub_Image.gif)
		 [RemoveItem](IDefDimensions.RemoveItem.htm)
		 Метод RemoveItem удаляет
		 из коллекции измерение с указанным ключом.


		 ![](../../Sub_Image.gif)
		 [Swap](IDefDimensions.Swap.htm)
		 Метод Swap меняет местами
		 два указанных измерения.
