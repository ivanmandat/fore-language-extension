# IDefPrivateDimensions


## Описание


Интерфейс IDefPrivateDimensions
 предназначен для работы с коллекцией частных измерений показателя.


## Иерархия наследования


IDefPrivateDimensions


## Комментарии


Коллекцию частных измерений возвращает свойство [IDefIndicator.DefDimensions](../IDefIndicator/IDefIndicator.DefDimensions.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDefPrivateDimensions.Count.htm)
		 Свойство Count возвращает
		 количество частных измерений в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IDefPrivateDimensions.Item.htm)
		 Свойство Item возвращает
		 параметры указанного частного измерения.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ItemByKey](IDefPrivateDimensions.ItemByKey.htm)
		 Метод ItemByKey возвращает
		 параметры частного измерения с указанным ключом.


		 ![](../../Sub_Image.gif)
		 [Swap](IDefPrivateDimensions.Swap.htm)
		 Метод Swap меняет местами
		 два указанных измерения.
