# IDefPrivateDimensions.Swap


## Синтаксис


		Swap(dimKey1: Integer; dimKey2: Integer);


## Параметры


dimKey1. Ключ первого перемещаемого
 измерения.


dimKey2. Ключ второго перемещаемого
 измерения.


## Описание


Метод Swap меняет местами два
 указанных измерения.
