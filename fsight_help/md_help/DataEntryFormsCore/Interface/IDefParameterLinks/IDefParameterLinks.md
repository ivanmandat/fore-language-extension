# IDefParameterLinks


## Описание


Интерфейс IDefParameterLinks
 предназначен для работы с коллекцией связей параметров вложенных объектов.


## Иерархия наследования


IDefParameterLinks


## Комментарии


Настройки связей можно получить, используя свойство [IDefParameter.ParametersMap](../IDefParameter/IDefParameter.ParametersMap.htm).
