# IDefTableArea.MultiLevelTopHeader


## Синтаксис


		MultiLevelTopHeader: Boolean;


## Описание


Свойство MultiLevelTopHeader
 определяет, будет ли использоваться многоуровневая шапка в табличной области.


## Комментарии


Если свойству установить значение True,
 то в шапке уровни измерений будут отображаться в отдельных строках без
 повторения родительских элементов. По умолчанию свойству установлено значение
 False, при этом элементы одного
 измерения располагаются в одной строке.
