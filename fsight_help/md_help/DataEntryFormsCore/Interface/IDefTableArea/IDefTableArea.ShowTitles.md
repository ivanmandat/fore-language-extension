# IDefTableArea.ShowTitles


## Синтаксис


		ShowTitles: Boolean;


## Описание


Свойство ShowTitles определяет
 состояние флажка «Всегда отображать заголовки».


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Флажок установлен;


	- False. Флажок снят.
