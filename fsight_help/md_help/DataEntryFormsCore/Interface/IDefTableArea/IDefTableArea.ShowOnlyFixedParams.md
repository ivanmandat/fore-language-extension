# IDefTableArea.ShowOnlyFixedParams


## Синтаксис


ShowOnlyFixedParams: Boolean;


## Описание


Свойство ShowOnlyFixedParams
 определяет, будут ли отображаться элементы только тех параметров, которые
 связаны с фиксированными измерениями.


## Комментарии


Свойство актуально, если свойству [ShowParams](IDefTableArea.ShowParams.htm)
 установлено значение True.


По умолчанию свойству ShowOnlyFixedParams
 установлено значение False,
 при этом над табличной областью отображаются все параметры, связанные
 с измерениями. При установке свойству значения False
 над табличной областью будут отображаться только те параметры, которые
 связаны с фиксированными измерениями и не отображаются в табличной области.
