# IDefTableArea.Transposed


## Синтаксис


		Transposed: Boolean;


## Описание


Свойство Transposed определяет,
 транспонирована ли табличная область.


## Комментарии


Если свойству установлено значение True,
 то табличная область будет транспонирована. При этом измерения боковика
 перемещаются в шапку, а измерения шапки - в боковик. При установке свойству
 значения False измерения боковика
 и шапки возвращаются на свои места.


Изменения будут применены после вызова метода [IDefInputAreas.ApplyAreaToReport](../IDefInputAreas/IDefInputAreas.ApplyAreaToReport.htm).
