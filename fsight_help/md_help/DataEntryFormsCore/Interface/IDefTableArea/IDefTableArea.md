# IDefTableArea


## Описание


Интерфейс IDefTableArea предназначен
 для управления табличной областью.


## Иерархия наследования


           [IDefInputArea](../IDefInputArea/IDefInputArea.htm)


           IDefTableArea


## Комментарии


Табличную область можно получить, используя методы интерфейса [IDefInputAreas](../IDefInputAreas/IDefInputAreas.htm).
