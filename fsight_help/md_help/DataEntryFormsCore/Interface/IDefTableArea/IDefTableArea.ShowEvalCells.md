# IDefTableArea.ShowEvalCells


## Синтаксис


ShowEvalCells: Boolean;


## Описание


Свойство ShowEvalCells определяет,
 будет ли использоваться пиктограмма для вычисляемых ячеек.


## Комментарии


По умолчанию свойству установлено значение True,
 при этом для вычисляемых ячеек рядом со значением будет отображаться пиктограмма
 ![](../../cellCountable.png).
 Пиктограмма также будет отображаться, если в ячейке отсутствует рассчитанное
 значение.


Если свойству ShowEvalCells
 установить значение False, то
 пиктограмма для вычисляемых ячеек не будет использоваться.
