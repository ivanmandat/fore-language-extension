# IDefTableArea.UseVirtualCells


## Синтаксис


UseVirtualCells: Boolean;


## Описание


Свойство UseVirtualCells определяет,
 будет ли в табличной области использоваться режим виртуальных ячеек.


## Комментарии


По умолчанию свойству установлено значение True,
 при это во время редактирования формы ввода в табличной области не будут
 отображаться данные.


При установке значения False
 во время редактирования формы ввода табличная область будет загружать
 данные.
