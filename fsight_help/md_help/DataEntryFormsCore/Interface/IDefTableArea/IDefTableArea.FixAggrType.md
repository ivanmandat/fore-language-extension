# IDefTableArea.FixAggrType


## Синтаксис


FixAggrType: [PivotAggregationOperation](KePivot.chm::/Enums/PivotAggregationOperation.htm);


## Описание


Свойство FixAggrType определяет
 метод агрегации данных, применяемый для измерений, расположенных в фиксированной
 области.


## Комментарии


Если задан метод агрегации, то по фиксированным измерениям будет доступна
 множественная отметка.
