# IDefParameter


## Описание


Интерфейс IDefParameter предназначен
 для настройки параметра формы ввода.


## Иерархия наследования


IDefParameter


## Комментарии


Настройки параметра можно получить, используя свойства и методы интерфейса
 [IDefParameters](../IDefParameters/IDefParameters.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AdvancedFilters](IDefParameter.AdvancedFilters.htm)
		 Свойство AdvancedFilters
		 возвращает коллекцию расширенных фильтров параметра.


		 ![](../../Property_Image.gif)
		 [DefaultValueSelection](IDefParameter.DefaultValueSelection.htm)
		 Свойство DefaultValueSelection
		 возвращает отметку, определяющую значения по умолчанию для параметра.


		 ![](../../Property_Image.gif)
		 [Dictionary](IDefParameter.Dictionary.htm)
		 Свойство Dictionary
		 определяет справочник, на базе которого построен параметр.


		 ![](../../Property_Image.gif)
		 [DimensionInstance](IDefParameter.DimensionInstance.htm)
		 Свойство DimensionInstance
		 возвращает открытый экземпляр справочника, на базе которого построен
		 параметр.


		 ![](../../Property_Image.gif)
		 [HierarchyDictKey](IDefParameter.HierarchyDictKey.htm)
		 Свойство HierarchyDictKey
		 определяет ключ справочника, который будет использоваться в качестве
		 альтернативной иерархии.


		 ![](../../Property_Image.gif)
		 [HierarchyParameterAttributeKey](IDefParameter.HierarchyParameterAttributeKey.htm)
		 Свойство HierarchyParameterAttributeKey
		 определяет ключ атрибута, значения которого используются для выбора
		 альтернативной иерархии.


		 ![](../../Property_Image.gif)
		 [HierarchyParameterKey](IDefParameter.HierarchyParameterKey.htm)
		 Свойство HierarchyParameterKey
		 определяет ключ параметра, который будет использоваться для выбора
		 альтернативной иерархии.


		 ![](../../Property_Image.gif)
		 [HierarchyTypeSource](IDefParameter.HierarchyTypeSource.htm)
		 Свойство HierarchyTypeSource
		 определяет способ задания источника альтернативной иерархии.


		 ![](../../Property_Image.gif)
		 [Key](IDefParameter.Key.htm)
		 Свойство Key возвращает
		 ключ параметра.


		 ![](../../Property_Image.gif)
		 [Name](IDefParameter.Name.htm)
		 Свойство Name определяет
		 наименование параметра.


		 ![](../../Property_Image.gif)
		 [Order](IDefParameter.Order.htm)
		 Свойство Order возвращает
		 порядок параметра.


		 ![](../../Property_Image.gif)
		 [ParametersMap](IDefParameter.ParametersMap.htm)
		 Свойство ParametersMap
		 возвращает настройки связей параметров справочника, лежащего в
		 основе текущего параметра формы ввода.


		 ![](../../Property_Image.gif)
		 [SelectionType](IDefParameter.SelectionType.htm)
		 Свойство SelectionType
		 определяет тип отметки параметра.


		 ![](../../Property_Image.gif)
		 [Sheets](IDefParameter.Sheets.htm)
		 Свойство Sheets определяет
		 коллекцию листов, на которых будет отображаться параметр.


		 ![](../../Property_Image.gif)
		 [UseDefaultValue](IDefParameter.UseDefaultValue.htm)
		 Свойство UseDefaultValue
		 определяет, будет ли использоваться значение по умолчанию при
		 пустой отметке параметра.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ClearHierarchy](IDefParameter.ClearHierarchy.htm)
		 Метод ClearHierarchy
		 отключает используемую для параметра альтернативную иерархию.


		 ![](../../Sub_Image.gif)
		 [IncludeAllSheets](IDefParameter.IncludeAllSheets.htm)
		 Метод IncludeAllSheets
		 включает использование параметра на всех листах формы ввода.
