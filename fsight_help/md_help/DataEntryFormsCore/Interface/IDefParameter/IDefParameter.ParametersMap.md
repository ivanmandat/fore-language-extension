# IDefParameter.ParametersMap


## Синтаксис


		ParametersMap: [IDefParameterLinks](../IDefParameterLinks/IDefParameterLinks.htm);


## Описание


Свойство ParametersMap возвращает
 настройки связей параметров справочника, лежащего в основе текущего параметра
 формы ввода.


## Комментарии


Свойство используется если справочник, указанный в свойстве [Dictionary](IDefParameter.Dictionary.htm),
 является параметрическим и для его открытия требуется указать значения
 параметров.
