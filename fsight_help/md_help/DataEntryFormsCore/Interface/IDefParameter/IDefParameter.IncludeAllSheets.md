# IDefParameter.IncludeAllSheets


## Синтаксис


		IncludeAllSheets;


## Описание


Метод IncludeAllSheets включает
 использование параметра на всех листах формы ввода.


## Комментарии


При создании новых листов использование параметра на них будет доступно
 автоматически.
