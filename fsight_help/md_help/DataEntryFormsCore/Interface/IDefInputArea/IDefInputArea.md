# IDefInputArea


## Описание


Интерфейс IDefInputArea содержит
 базовые свойства областей ввода.


## Иерархия наследования


IDefInputArea


## Комментарии


Для доступа к областям ввода используйте свойства и методы интерфейса
 [IDefInputAreas](../IDefInputAreas/IDefInputAreas.htm).


Интерфейс IDefInputArea является
 базовым для интерфейса [IDefTableArea](../IDefTableArea/IDefTableArea.htm).
