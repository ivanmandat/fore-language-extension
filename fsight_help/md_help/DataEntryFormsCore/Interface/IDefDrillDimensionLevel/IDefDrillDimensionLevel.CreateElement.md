# IDefDrillDimensionLevel.CreateElement


## Синтаксис


		CreateElement: Boolean;


## Описание


Свойство CreateElement определяет,
 будет ли доступна возможность добавления и удаления элементов.


## Комментарии


Допустимые значения:


	- True. Будет доступна
	 возможность добавление и удаления элементов в справочник измерения;


	- False. Добавление/удаление
	 элементом недоступно.
