# IDefDrillDimensionLevel.RefreshAfterCreationClose


## Синтаксис


		RefreshAfterCreationClose: Boolean;


## Описание


Свойство RefreshAfterCreationClose
 определяет, будет ли производиться обновление табличной области после
 закрытия объекта для расшифровки.


## Комментарии


Допустимые значения:


	- True. Обновлять табличную
	 область после закрытия объекта для расшифровки;


	- False. Не обновлять
	 табличную область.
