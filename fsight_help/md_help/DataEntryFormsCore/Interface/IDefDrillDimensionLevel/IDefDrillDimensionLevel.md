# IDefDrillDimensionLevel


## Описание


Интерфейс IDefDrillDimensionLevel
 предназначен для настройки расшифровки уровня измерения.


## Иерархия наследования


IDefDrillDimensionLevel


## Комментарии


Настройки расшифровки уровня измерения можно получить, используя свойства
 интерфейса [IDefDrillDimensionLevels](../IDefDrillDimensionLevels/IDefDrillDimensionLevels.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CreateElement](IDefDrillDimensionLevel.CreateElement.htm)
		 Свойство CreateElement
		 определяет, будет ли доступна возможность добавления и удаления
		 элементов.


		 ![](../../Property_Image.gif)
		 [DrillDescriptor](IDefDrillDimensionLevel.DrillDescriptor.htm)
		 Свойство DrillDescriptor
		 определяет описание объекта репозиторий, который будет открыт
		 для расшифровки элемента измерения.


		 ![](../../Property_Image.gif)
		 [LevelIndex](IDefDrillDimensionLevel.LevelIndex.htm)
		 Свойство LevelIndex
		 возвращает индекс уровня измерения, для которого настраивается
		 расшифровка.


		 ![](../../Property_Image.gif)
		 [OpenOnSheetKey](IDefDrillDimensionLevel.OpenOnSheetKey.htm)
		 Свойство OpenOnSheetKey
		 определяет ключ листа, на котором откроется форма ввода при расшифровке.


		 ![](../../Property_Image.gif)
		 [ParameterValues](IDefDrillDimensionLevel.ParameterValues.htm)
		 Свойство ParameterValues
		 возвращает настройки для управления параметрами открываемого объекта.


		 ![](../../Property_Image.gif)
		 [RefreshAfterCreationClose](IDefDrillDimensionLevel.RefreshAfterCreationClose.htm)
		 Свойство RefreshAfterCreationClose
		 определяет, будет ли производиться обновление табличной области
		 после закрытия объекта для расшифровки.
