# IDefCalculationAlgorithmsCollection


## Описание


Интерфейс IDefCalculationAlgorithmsCollection
 предназначен для работы с коллекцией алгоритмов расчёта, применяемых в
 табличной области данных.


## Иерархия наследования


IDefCalculationAlgorithmsCollection


## Комментарии


Доступ к коллекции алгоритмов расчёта предоставляет свойство [IDefTableArea.CalculationAlgorithms](../IDefTableArea/IDefTableArea.CalculationAlgorithms.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDefCalculationAlgorithmsCollection.Count.htm)
		 Свойство Count возвращает
		 количество алгоритмов расчёта, добавленных для использования в
		 табличной области.


		 ![](../../Property_Image.gif)
		 [Item](IDefCalculationAlgorithmsCollection.Item.htm)
		 Свойство Item возвращает
		 параметры использования алгоритма расчёта по его индексу в коллекции.


		 ![](../../Property_Image.gif)
		 [ItemByUnical](IDefCalculationAlgorithmsCollection.ItemByUnical.htm)
		 Свойство ItemByUnical
		 возвращает параметры использования алгоритма расчёта по его уникальному
		 идентификатору.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IDefCalculationAlgorithmsCollection.Add.htm)
		 Метод Add добавляет
		 алгоритм расчёта для использования в табличной области.


		 ![](../../Sub_Image.gif)
		 [MoveTo](IDefCalculationAlgorithmsCollection.MoveTo.htm)
		 Метод MoveTo перемещает
		 алгоритмы расчёта внутри коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IDefCalculationAlgorithmsCollection.Remove.htm)
		 Метод Remove удаляет
		 алгоритм расчёта с указанным уникальным идентификатором из коллекции.
