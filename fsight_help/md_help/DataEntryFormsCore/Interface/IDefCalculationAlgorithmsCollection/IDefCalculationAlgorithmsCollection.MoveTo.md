# IDefCalculationAlgorithmsCollection.MoveTo


## Синтаксис


		MoveTo(oldIndex: Integer; newIndex: Integer);


## Параметры


oldIndex. Индекс позиции, из
 которой перемещается алгоритм расчёта.


newIndex. Индекс позиции, в
 которую перемещается алгоритм расчёта.


## Описание


Метод MoveTo перемещает алгоритмы
 расчёта внутри коллекции.


## Комментарии


Индекс позиции алгоритма расчёта в коллекции определяет порядок его
 применения в табличной области.
