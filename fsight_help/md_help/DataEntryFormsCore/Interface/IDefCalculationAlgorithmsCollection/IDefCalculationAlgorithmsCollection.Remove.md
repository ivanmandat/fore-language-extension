# IDefCalculationAlgorithmsCollection.Remove


## Синтаксис


		Remove(unical: Integer);


## Параметры


unical. Уникальный идентификатор
 алгоритма расчёта.


## Описание


Метод Remove удаляет алгоритм
 расчёта с указанным уникальным идентификатором из коллекции.
