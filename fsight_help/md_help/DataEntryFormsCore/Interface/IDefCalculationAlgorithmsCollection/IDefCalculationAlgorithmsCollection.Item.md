# IDefCalculationAlgorithmsCollection.Item


## Синтаксис


		Item(Index: Integer): [IDefCalculationAlgorithm](../IDefCalculationAlgorithm/IDefCalculationAlgorithm.htm);


## Параметры


Index. Индекс алгоритма расчёта
 в коллекции.


## Описание


Свойство Item возвращает параметры
 использования алгоритма расчёта по его индексу в коллекции.
