# IDefCalculationAlgorithmsCollection.ItemByUnical


## Синтаксис


		ItemByUnical(unical: Intger): [IDefCalculationAlgorithm](../IDefCalculationAlgorithm/IDefCalculationAlgorithm.htm);


## Параметры


unical. Уникальный идентификатор
 алгоритма расчёта.


## Описание


Свойство ItemByUnical возвращает
 параметры использования алгоритма расчёта по его уникальному идентификатору.
