# IDefDrillDimensionParameterValue.Value


## Синтаксис


		Value: Variant;


## Описание


Свойство Value определяет значение,
 передаваемое параметру при расшифровке.


## Комментарии


В зависимости от варианта привязки, указанного в свойстве [DrillType](IDefDrillDimensionParameterValue.DrillType.htm),
 в свойстве Value указывается идентификатор
 атрибута измерения, параметра формы ввода или фиксированное значение.
