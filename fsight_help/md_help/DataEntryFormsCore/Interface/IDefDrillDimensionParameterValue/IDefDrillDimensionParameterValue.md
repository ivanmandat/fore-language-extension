# IDefDrillDimensionParameterValue


## Описание


Интерфейс IDefDrillDimensionParameterValue
 предназначен для настройки управления параметром объекта, открываемого
 при расшифровке.


## Иерархия наследования


IDefDrillDimensionParameterValue


## Комментарии


Настройки управления параметром можно получить, используя свойства интерфейса
 [IDefDrillDimensionParameterValues](../IDefDrillDimensionParameterValues/IDefDrillDimensionParameterValues.htm).
