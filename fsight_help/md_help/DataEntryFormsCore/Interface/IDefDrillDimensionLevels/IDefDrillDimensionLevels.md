# IDefDrillDimensionLevels


## Описание


Интерфейс IDefDrillDimensionLevels
 предназначен для настройки расшифровки уровней измерения.


## Иерархия наследования


IDefDrillDimensionLevels


## Комментарии


Настройки расшифровки уровней измерения можно получить, используя свойство
 [IDefDrillDimension.DrillLevels](../IDefDrillDimension/IDefDrillDimension.DrillLevels.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IDefDrillDimensionLevels.Count.htm)
		 Свойство Count возвращает
		 количество уровней, для которых может быть настроена расшифровка.


		 ![](../../Property_Image.gif)
		 [Item](IDefDrillDimensionLevels.Item.htm)
		 Свойство Item возвращает
		 настройки расшифровки для уровня с указанным индексом.


## Методы


		 Имя метода
