# IDefCalclulationAlgorithmMapParamInfoCollection


## Описание


Интерфейс IDefCalclulationAlgorithmMapParamInfoCollection
 предназначен для работы с коллекцией привязок параметров текущей формы
 ввода к параметрам форм ввода, с которыми связан алгоритм расчёта.


## Иерархия наследования


IDefCalclulationAlgorithmMapParamInfoCollection


## Комментарии


Коллекцию привязок можно получить, используя свойство [IDefCalculationAlgorithmMapInfo.MapParameters](../IDefCalculationAlgorithmMapInfo/IDefCalculationAlgorithmMapInfo.MapParameters.htm).
