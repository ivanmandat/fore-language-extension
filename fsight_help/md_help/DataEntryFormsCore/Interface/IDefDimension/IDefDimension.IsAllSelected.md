# IDefDimension.IsAllSelected


## Синтаксис


		IsAllSelected: Boolean;


## Описание


Свойство IsAllSelected определяет,
 выделены ли все элементы измерения.


## Комментарии


Допустимые значения:


	- True. Выделены все элементы
	 измерения.


	- False. Отметка элементов
	 задана в свойстве [IDefDimension.ElemsSelection](IDefDimension.ElemsSelection.htm).
