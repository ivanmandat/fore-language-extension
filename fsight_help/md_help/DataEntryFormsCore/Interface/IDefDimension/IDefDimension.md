# IDefDimension


## Описание


Интерфейс IDefDimension предназначен
 для настройки параметров измерения.


## Иерархия наследования


IDefDimension


## Комментарии


Параметры измерения можно получить, используя свойства и методы интерфейсов
 [IDefDimensions](../IDefDimensions/IDefDimensions.htm), [IDefPrivateDimensions](../IDefPrivateDimensions/IDefPrivateDimensions.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Drill](IDefDimension.Drill.htm)
		 Свойство Drill возвращает
		 настройки расшифровки для измерения.


		 ![](../../Property_Image.gif)
		 [ElemsSelection](IDefDimension.ElemsSelection.htm)
		 Свойство ElemsSelection
		 определяет отметку элементов измерения.


		 ![](../../Property_Image.gif)
		 [HierarchyDictKey](IDefDimension.HierarchyDictKey.htm)
		 Свойство HierarchyDictKey
		 определяет ключ справочника, который будет использоваться в качестве
		 альтернативной иерархии.


		 ![](../../Property_Image.gif)
		 [HierarchyParameterAttributeKey](IDefDimension.HierarchyParameterAttributeKey.htm)
		 Свойство HierarchyParameterAttributeKey
		 определяет ключ атрибута, значения которого используются для выбора
		 альтернативной иерархии.


		 ![](../../Property_Image.gif)
		 [HierarchyParameterKey](IDefDimension.HierarchyParameterKey.htm)
		 Свойство HierarchyParameterKey
		 определяет ключ параметра, который будет использоваться для выбора
		 альтернативной иерархии.


		 ![](../../Property_Image.gif)
		 [HierarchyTypeSource](IDefDimension.HierarchyTypeSource.htm)
		 Свойство HierarchyTypeSource
		 определяет способ задания источника альтернативной иерархии.


		 ![](../../Property_Image.gif)
		 [IsAllSelected](IDefDimension.IsAllSelected.htm)
		 Свойство IsAllSelected
		 определяет, выделены ли все элементы измерения.

		 [Key](IDefDimension.Key.htm)

		 ключ измерения.


		 ![](../../Property_Image.gif)
		 [Place](IDefDimension.Place.htm)
		 Свойство Place определяет
		 расположение измерения.
