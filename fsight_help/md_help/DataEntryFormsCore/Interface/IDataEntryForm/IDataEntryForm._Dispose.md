# IDataEntryForm._Dispose


## Синтаксис


		_Dispose;


## Описание


Метод _Dispose очищает ссылки
 на объекты.


## Комментарии


Метод необходимо вызывать при окончании работы с формой для уничтожения
 внутренних временных объектов. Это необходимо для исключения случаев утечки
 памяти.
