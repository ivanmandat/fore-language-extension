# IDataEntryForm


## Описание


Интерфейс IDataEntryForm предназначен
 для настройки формы ввода.


## Иерархия наследования


IDataEntryForm


## Комментарии


Для работы с формой ввода создайте экземпляр класса [DataEntryForm](../../Class/DataEntryForm/DataEntryForm.htm)
 с помощью доступных конструкторов или статического метода и полученный
 объект приведите к интерфейсу IDataEntryForm.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [InputAreas](IDataEntryForm.InputAreas.htm)
		 Свойство InputAreas
		 возвращает коллекцию областей ввода.


		 ![](../../Property_Image.gif)
		 [Parameters](IDataEntryForm.Parameters.htm)
		 Свойство Parameters
		 возвращает коллекцию параметров формы ввода.


		 ![](../../Property_Image.gif)
		 [refreshOnEdit](IDataEntryForm.refreshOnEdit.htm)
		 Свойство refreshOnEdit
		 определяет, будет ли обновляться содержимое формы ввода при открытии
		 на редактирование.


		 ![](../../Property_Image.gif)
		 [Report](IDataEntryForm.Report.htm)
		 Свойство Report возвращает
		 отчёт, привязанный к форме ввода.


		 ![](../../Property_Image.gif)
		 [Sheets](IDataEntryForm.Sheets.htm)
		 Свойство Sheets возвращает
		 коллекцию листов формы ввода.
