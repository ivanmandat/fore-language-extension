# IDataEntryForm.refreshOnEdit


## Синтаксис


		refreshOnEdit: Boolean;


## Описание


Свойство refreshOnEdit определяет,
 будет ли обновляться содержимое формы ввода при открытии на редактирование.


## Комментарии


По умолчанию свойству установлено значение False,
 при этом содержимое формы ввода не обновляется. Если свойству установить
 значение True, то при открытии
 формы ввода на редактирование будут обновлены все листы, отстроены области
 данных.
