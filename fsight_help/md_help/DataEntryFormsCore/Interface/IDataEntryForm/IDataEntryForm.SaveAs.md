# IDataEntryForm.SaveAs


## Синтаксис


		SaveAs(name: String; id: String; parentKey:
		 Integer): Boolean;


## Параметры


name. Новое наименование копии
 формы ввода;


id. Новый идентификатор копии
 формы ввода;


parentKey. Ключ родительского
 объекта, в рамках которого будет сохранена копия формы ввода.


## Описание


Метод SaveAs создаёт копию формы
 ввода и сохраняет её в репозитории.
