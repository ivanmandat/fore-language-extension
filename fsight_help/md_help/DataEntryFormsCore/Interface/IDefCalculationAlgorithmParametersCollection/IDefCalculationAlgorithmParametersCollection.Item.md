# IDefCalculationAlgorithmParametersCollection.Item


## Синтаксис


		Item(Index: Integer): [IDefCalculationAlgorithmParameter](../IDefCalculationAlgorithmParameter/IDefCalculationAlgorithmParameter.htm);


## Параметры


Index. Индекс привязки в коллекции.


## Описание


Свойство Item возвращает настройки
 привязки параметра алгоритма расчёта к параметру формы ввода по индексу
 привязки.
