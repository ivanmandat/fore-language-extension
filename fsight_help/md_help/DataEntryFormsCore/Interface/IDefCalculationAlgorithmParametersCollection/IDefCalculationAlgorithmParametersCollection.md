# IDefCalculationAlgorithmParametersCollection


## Описание


Интерфейс IDefCalculationAlgorithmParametersCollection
 предназначен для работы с коллекцией привязок параметров алгоритма расчёта
 к параметрам формы ввода.


## Иерархия наследования


IDefCalculationAlgorithmParametersCollection


## Комментарии


Коллекцию привязок можно получить, используя свойства [IDefCalculationAlgorithm.Parameters](../IDefCalculationAlgorithm/IDefCalculationAlgorithm.Parameters.htm).
