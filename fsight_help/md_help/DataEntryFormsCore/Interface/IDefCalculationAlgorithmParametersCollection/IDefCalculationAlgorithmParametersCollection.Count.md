# IDefCalculationAlgorithmParametersCollection.Count


## Синтаксис


		Count: Integer;


## Описание


Свойство Count возвращает количество
 привязок параметров в коллекции.


## Комментарии


Количество привязок соответствует количеству параметров, которые имеет
 алгоритм расчёта.
