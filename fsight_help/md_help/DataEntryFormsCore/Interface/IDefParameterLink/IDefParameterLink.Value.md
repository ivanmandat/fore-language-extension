# IDefParameterLink.Value


## Синтаксис


		Value: Variant;


## Описание


Свойство Value определяет фиксированное
 значение, которое будет передаваться в параметр связанного объекта.


## Комментарии


Свойство актуально, если свойству [Type](IDefParameterLink.Type.htm)
 установлено значение [LinkType.Values](../../Enums/LinkType.htm).
