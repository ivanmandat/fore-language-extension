# IDefParameterLink


## Описание


Интерфейс IDefParameterLink
 предназначен для настройки связи параметра вложенного объекта.


## Иерархия наследования


IDefParameterLink


## Комментарии


Настройки связи можно получить, используя свойства интерфейса [IDefParameterLinks](../IDefParameterLinks/IDefParameterLinks.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attribute](IDefParameterLink.Attribute.htm)
		 Свойство Attribute
		 возвращает информацию о привязке к атрибуту параметра формы ввода,
		 передаваемого в параметр объекта.


		 ![](../../Property_Image.gif)
		 [ParameterId](IDefParameterLink.ParameterId.htm)
		 Свойство ParameterId
		 возвращает идентификатор параметра вложенного объекта.


		 ![](../../Property_Image.gif)
		 [Type](IDefParameterLink.Type.htm)
		 Свойство Type определяет
		 тип связи параметра вложенного объекта.


		 ![](../../Property_Image.gif)
		 [Value](IDefParameterLink.Value.htm)
		 Свойство Value определяет
		 фиксированное значение, которое будет передаваться в параметр
		 связанного объекта.
