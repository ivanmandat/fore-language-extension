# IDefParameterLink.Type


## Синтаксис


		Type: [LinkType](../../Enums/LinkType.htm);


## Описание


Свойство Type определяет тип
 связи параметра вложенного объекта.


## Комментарии


Если свойству установлено значение [LinkType.Values](../../Enums/LinkType.htm),
 то определите в свойстве [Value](IDefParameterLink.Value.htm)
 фиксированное значение, которое будет передаваться параметру вложенного
 объекта.


Если свойству установлено значение [LinkType.Parameters](../../Enums/LinkType.htm),
 то в свойстве [Attribute](IDefParameterLink.Attribute.htm)
 настройте связь с атрибутом параметра формы ввода.
