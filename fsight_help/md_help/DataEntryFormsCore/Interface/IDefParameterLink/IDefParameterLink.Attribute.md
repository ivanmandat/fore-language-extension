# IDefParameterLink.Attribute


## Синтаксис


		Attribute: [IDefParameterLinkAttributeInfo](../IDefParameterLinkAttributeInfo/IDefParameterLinkAttributeInfo.htm);


## Описание


Свойство Attribute возвращает
 информацию о привязке к атрибуту параметра формы ввода, передаваемого
 в параметр объекта.


## Комментарии


Свойство актуально, если свойству [Type](IDefParameterLink.Type.htm)
 установлено значение [LinkType.Parameters](../../Enums/LinkType.htm).
