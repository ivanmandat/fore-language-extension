# Поиск пользователя


Для поиска пользователя в разделе «[Пользователи](Admin_Users.htm)»:


	- в веб-приложении нажмите кнопку ![](../../02_ComonFunction/Search_button.png) «Поиск» на [панели
	 инструментов](../../01_RunSecManager/Admin_Organizational_Starting.htm);


	- в настольном приложении:


		- выполните команду «Пользователь
		 > Поиск» в главном меню;


		- используйте сочетание клавиш
		 CTRL+F/CTRL+F3.


При выполнении одного из действий появится [диалог поиска](UiNav.chm::/02_Navigator/UiNav_FindObject.htm).
