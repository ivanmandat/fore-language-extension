# Параметры общих свойств


Для изменения общих свойств группы пользователей используйте вкладку
 «Общие свойства» на боковой панели
 «[Свойства](Admin_GroupsCreate.htm#properties)»
 в веб-приложении и в окне «[Свойства
 группы](Admin_GroupsCreate.htm#properties)» в настольном приложении.

	 Настольное
	 приложение


		![](Admin_GroupsCreate_web.png)


		![](Admin_GroupsCreate.png)


Параметры группы пользователей:


	- Наименование. Уникальное
	 наименование группы пользователей среди пользователей и групп пользователей.
	 Наименования групп пользователей полностью регистронезависимые. Создание
	 групп пользователей с одинаковыми наименованиями, отличающихся только
	 регистром написания символов, недоступно;


	- Описание. Описание группы
	 пользователей по желанию администратора;


	- Роль СУБД Oracle. Роль
	 на сервере СУБД. Используется при [доменной
	 авторизации](Admin_GroupsCreateDomain.htm#oracle) пользователей, которые отсутствуют в менеджере безопасности
	 репозитория;


Примечание.
 Поле доступно только в настольном приложении при подключении [доменной
 группы](Admin_GroupsCreateDomain.htm) в менеджере безопасности репозитория на базе СУБД Oracle.


	- [Члены
	 группы](Group_Members.htm). Пользователи, состоящие в группе;


	- [Объекты
	 группы](../Users/Admin_UserObjects.htm). Просмотр прав доступа
	 группы пользователей на объекты репозитория.


Примечание.
 Вкладка «Члены группы» и «Объекты группы» отображаются только
 в веб-приложении при [создании](Admin_GroupsCreate.htm#create)
 и [редактировании](Admin_GroupsCreate.htm#edit) группы пользователей.
 В настольном приложении члены группы формируются на отдельной вкладке
 «[Члены группы](Group_Members.htm)»,
 объекты группы отображаются на панели «[Объекты группы](../Users/Admin_UserObjects.htm)».
