# Удаление группы пользователей


Для удаления выбранных групп пользователей в разделе «[Группы](Admin_Groups.htm)»:


	- в веб-приложении нажмите кнопку ![](../../01_RunSecManager/delete.png) «Удалить» на
	 [панели
	 инструментов](../../01_RunSecManager/Admin_Organizational_Starting.htm);


	- в настольном приложении:


		- выполните команду «Удалить»
		 в контекстном меню;


		- выполните команду «Группа
		 > Удалить»
		 в [главном
		 меню](../../01_RunSecManager/Admin_Organizational_Starting.htm).


Примечание.
 Удаление группы пользователей доступно пользователям, обладающим [привилегией](../../04_SecurityPolicy/Admin_Priv.htm)
 «Изменение прав пользователей, раздача
 ролей, изменение политики». При активном [режиме
 разделения ролей](../../04_SecurityPolicy/Editor_of_Politicy/Security_EditorPoliticy_Adm.htm) и установленном на вкладке «[Контроль
 доступа](../../04_SecurityPolicy/Admin_SecPolicy.htm#access_control)» флажке «Запретить
 удаление групп с правами доступа на объекты репозитория» АИБу будет
 недоступно удаление групп с правами на объекты репозитория.


После выполнения одного из действий появится диалог подтверждения удаления.


Примечание.
 Встроенные группы пользователей: администраторы и пользователи недоступны
 для удаления.
