# Экспорт политики безопасности и прав доступа


При необходимости экспортируйте настройки политики безопасности и прав
 доступа на объекты.


Примечание.
 Экспорт политики безопасности и прав доступа на объекты доступен только
 в настольном приложении.


## Экспорт настроек политики безопасности


Для экспорта настроек политики безопасности:


	- Перейдите в раздел «Редактор
	 политик» [панели
	 навигации](../01_RunSecManager/Admin_Organizational_Starting.htm).


	- Выполните команду «Сервис
	 > Экспортировать настройки политики
	 безопасности» в [главном
	 меню](../01_RunSecManager/Admin_Organizational_Starting.htm).


Примечание.
 Экспорт настроек политики безопасности может производить пользователь,
 обладающий [привилегией](Admin_Priv.htm) «Изменение
 прав пользователей, раздача ролей, изменение политики».


После выполнения действий будет открыт стандартный диалог, в котором
 необходимо указать имя и месторасположение файла. По умолчанию в качестве
 имени файла будет предложено использовать: «Политика
 безопасности <имя репозитория> на <дата>».


По окончании экспорта предлагается открыть сформированный файл. При
 положительном ответе сформированный файл будет открыт в программе просмотра
 файлов CSV, зарегистрированной в ОС.


## Экспорт прав доступа


Для экспорта прав доступа:


	- Перейдите в раздел «Навигатор»
	 [панели
	 навигации](../01_RunSecManager/Admin_Organizational_Starting.htm).


	- Выполните команду «Сервис
	 > Экспортировать права доступа»
	 в главном меню и выберите вариант в раскрывающемся меню:


		- По типу объектов.
		 Права доступа в файле будут представлены по типу объектов. Например,
		 сначала буду приведены права для всех папок, имеющихся в репозитории,
		 затем для всех таблиц и т.д.


		- По папкам; по типу объектов.
		 Права доступа в файле будут представлены в соответствии с имеющейся
		 структурой в репозитории, то есть будет сформирована развернутая
		 иерархия объектов репозитория, где дочерние объекты будут отсортированы
		 по типу. Например, сначала будут приведены права для папки, расположенной
		 в корне; затем будут представлены права для объектов, содержащихся
		 внутри нее, но отсортированных по типу объектов и т.д.


Примечание.
 Экспорт прав доступа может производить пользователь, обладающий привилегией
 «Изменение метки безопасности и списка
 контроля доступа любого объекта».


После выполнения действий будет открыт стандартный диалог, в котором
 необходимо указать имя и месторасположение файла. По умолчанию в качестве
 имени файла будет предложен формат: «Права
 доступа <имя репозитория> на <дата>».


По окончании экспорта предлагается открыть сформированный файл. При
 положительном ответе сформированный файл будет открыт в программе просмотра
 CSV файлов, зарегистрированной в ОС.
