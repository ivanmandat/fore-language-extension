# Сохранение протокола доступа в файл


Для сохранения протокола доступа в файл:


	- в веб-приложении:


		- нажмите кнопку ![](../01_RunSecManager/export.png)
		 «Экспортировать» на [панели
		 инструментов](../01_RunSecManager/Admin_Organizational_Starting.htm). После выполнения действия записи протокола доступа
		 будут экспортированы в файл формата *.csv;


		- выберите формат файла в раскрывающемся меню кнопки ![](../01_RunSecManager/export.png) «Экспортировать»:


			- Экспортировать в CSV;


			- Экспортировать в CEF;


	- в настольном приложении:


		- выполните команду «Протокол
		 доступа > Сохранить в файл > Полный протокол/Текущее представление»
		 в [главном
		 меню](../01_RunSecManager/Admin_Organizational_Starting.htm);


		- используйте сочетание клавиш CTRL+S.


После выполнения действий записи протокола доступа будут экспортированы
 в файл выбранного формата: *.pplog, *.csv, *.cef.


Для получения подробной информации обратитесь к разделу «[Экспорт
 списка](../03_Admin/Admin_AdminObjects_AuditTuning.htm#export)».


Примечание.
 Команда «Протокол доступа > Сохранить
 в файл» переименовывается в «Сохранить
 как», когда открыт [файл
 протокола доступа](Admin_AcessProtocol_ViewSaved.htm).
