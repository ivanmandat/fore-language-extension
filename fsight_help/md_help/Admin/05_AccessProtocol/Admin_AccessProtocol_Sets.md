# Настройка протокола доступа


Для настройки протокола доступа:


	- Перейдите в раздел «[Классы
	 объектов](../04_SecurityPolicy/Admin_Object_Classes.htm)».


	- Откройте диалог «[Параметры
	 управления доступом](../03_Admin/Admin_AdminObjects.htm)»;


	- Перейдите на вкладку «[Аудит](../03_Admin/Admin_AdminObjects_Audit.htm)»;


	- Cформируйте список протоколируемых
	 операций.


После выполнения действий выбранные операции над объектами будут записываться
 в протокол доступа.


Также формирование списка операций, заносимых в протокол доступа, может
 осуществляться для конкретного объекта репозитория.
