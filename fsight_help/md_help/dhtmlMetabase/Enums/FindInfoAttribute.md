# FindInfoAttribute


Пространство имен:
 PP.Mb


## Описание


Перечисление FindInfoAttribute содержит
 атрибуты, по которым будет осуществляться поиск объектов репозитория.


Используется методом [Metabase.findObjects](../Classes/Metabase/Metabase/Metabase.findObjects.htm).


## Допустимые значения


		 Значение


		 Краткое описание


		 None


		 Атрибут не задан. По умолчанию.


		 Name


		 Поиск по имени.


		 Ident


		 Поиск по идентификатору.


		 NameOrIndent


		 Поиск по имени и по идентификатору.


		 Key


		 Поиск по ключу.
