# DmElRelative


Пространство имен:
 PP.Mb


## Описание


Перечисление DmElRelative содержит
 элементы, на которые будет распространяться команда для изменения отметки
 в измерении.


Используется методом [DimSrv.execCommand](dhtmlDim.chm::/Classes/Metabase/DimSrv/DimSrv.execCommand.htm).


## Допустимые значения


		 Значение


		 Краткое описание


		 Unknown


		 Не установлено.


		 Current


		 Выбранный элемент.


		 All


		 Все элементы измерения.


		 Children


		 Дочерние элементы для выбранного.


		 ChildrenRecursive


		 Все дочерние элементы измерения
		 с их дочерними вершинами.
