# DbDataType


Пространство имен: PP.Mb


## Описание


Перечисление DbDataType содержит
 типы данных сервиса.


Используется свойством [Param.Type](../Classes/Metabase/Param/Param.Type.htm).


## Допустимые значения


		 Значение


		 Краткое описание


		 0


		 NoData. Нет данных.


		 1


		 String. Строковые данные.


		 2


		 Integer. Целочисленные
		 данные.


		 3


		 Float. Вещественные
		 значения.


		 4


		 DateTime. Дата и время.


		 5


		 Blob. Массив двоичных
		 или символьных данных.


		 6


		 Boolean. Логические
		 данные.


		 7


		 Date. Дата без времени.
