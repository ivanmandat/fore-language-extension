# SelCommands


Пространство имен:
 PP.Mb


## Описание


Перечисление SelCommands содержит
 команды для отметки элементов измерения.


Используется методом [DimSrv.execCommand](dhtmlDim.chm::/Classes/Metabase/DimSrv/DimSrv.execCommand.htm).


## Допустимые значения


		 Значение


		 Краткое описание


		 Unknown


		 Команда не определена.


		 Select


		 Отметить.


		 Deselect


		 Снять отметку.


		 Invert


		 Инвертировать отметку.


		 Set


		 Отметить элемент по ключу.
