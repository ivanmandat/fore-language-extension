# TreeRelation


Пространство имен:
 PP.Mb


## Описание


Перечисление TreeRelation содержит
 типы отношений между элементами дерева.


Используется методом [DictionaryTree.moveItem](dhtmlRds.chm::/Classes/Rds/DictionaryTree/DictionaryTree.moveItem.htm).


## Допустимые значения


		 Значение


		 Краткое описание


		 Unknown


		 Неизвестно.


		 FirstChild


		 Первый дочерний элемент.


		 LastChild


		 Последний дочерний элемент.


		 NextSibling


		 Следующий одноуровневый элемент.


		 PrevSibling


		 Предыдущий одноуровневый элемент.


		 FirstSibling


		 Первый одноуровневый элемент.


		 LastSibling


		 Последний одноуровневый элемент.
