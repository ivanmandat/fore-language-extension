# DimFilterType


Пространство имен:
 PP.Mb.Ui


## Описание


Перечисление DimFilterType содержит
 варианты фильтрации дерева справочника.


## Допустимые значения


		 Значение


		 Краткое описание


		 Contains


		 В результате фильтрации в дереве
		 остаются элементы, содержащие указанную строку.


		 StartWith


		 В результате фильтрации в дереве
		 остаются элементы, названия которых начинаются с указанной строки.


		 EndWith


		 В результате фильтрации в дереве
		 остаются элементы, названия которых заканчиваются на указанную
		 строку.


		 Exactly


		 В результате фильтрации в дереве
		 остаются элементы, названия которых полностью совпадают с указанной
		 строкой.
