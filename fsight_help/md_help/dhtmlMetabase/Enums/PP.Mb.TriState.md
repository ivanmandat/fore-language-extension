# TriState


Пространство имен:
 PP.Mb


## Описание


Перечисление TriState содержит
 варианты отрисовки элемента управления на панели «[Элементы управления](dhtmlReport.chm::/Classes/RegularReport/ControlBar/ControlBar.htm)»
 регламентного отчета.


Используется свойством [ControlBarView.ControlsRenderState](dhtmlReport.chm::/Classes/RegularReport/ControlBarView/ControlBarView.ControlsRenderState.htm).


## Допустимые значения


		 Значение


		 Краткое описание


		 0


		 OffOption. Элемент
		 управления не создается и не добавляется в DOM-вершину.


		 1


		 OnOption. Созданный
		 элемент управления добавляется в DOM-вершину.


		 2


		 Undefined. Созданный
		 элемент управления не добавляется в DOM-вершину.
