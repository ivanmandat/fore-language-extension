# DrillMethod


Пространство имен: PP.Mb


## Описание


Перечисление DrillMethod содержит
 методы перехода по измерению уровней элементов, которые принимает сервис.


Используется следующими методами:


	- [EaxMdService.setChartDrillDown](../../dhtmlExpress/Classes/Express/EaxMdService/EaxMdService.setChartDrillDown.htm);


	- [EaxMdService.setMapDrillDown](../../dhtmlExpress/Classes/Express/EaxMdService/EaxMdService.setMapDrillDown.htm);


	- [EaxMdService.setGridDrillDown](../../dhtmlExpress/Classes/Express/EaxMdService/EaxMdService.setGridDrillDown.htm).


## Допустимые значения


		 Значение


		 Краткое описание


		 Down


		 Переход на нижний уровень.


		 Up


		 Переход на верхний уровень.


		 ToFixed


		 Фиксирование уровня.


		 IntoFixed


		 Переход в фиксированный уровень.


		 OutOfFixed


		 Переход из фиксированного уровня.


		 DownThrough


		 Детализация с проваливанием в реляционную таблицу.


		 UpThrough


		 Детализация с проваливаем из реляционной таблицы.
