# ObjectCopyOperation


Пространство имен:
 PP.Mb


## Описание


Перечисление ObjectCopyOperation содержит
 операции, которые могут выполняться при сохранении копии объекта.


Используется методом [Metabase.saveObjectAs](../Classes/Metabase/Metabase/Metabase.saveObjectAs.htm).


## Допустимые значения


		 Значение


		 Краткое описание


		 CreateNew


		 Создать новый объект. По умолчанию.


		 MoveToParent


		 Переместить объект.


		 OverwriteExisting


		 Перезаписать существующий объект.


		 ChangeCurrent


		 Изменить существующий объект.
