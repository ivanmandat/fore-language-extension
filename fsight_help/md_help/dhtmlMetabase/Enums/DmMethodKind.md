# DmMethodKind


Пространство имен:
 PP.Mb


## Описание


Перечисление DmMethodKind содержит
 методы интеллектуального анализа данных.


Используется мастером [DataMiningMaster](../Components/Metabase/DataMiningMaster/DataMiningMaster.htm).
