# SourceType

## Описание


Перечисление SourceType содержит
 типы источников данных при импорте.


Используется методом [DataImportService.setCubeImportProps](../Classes/Metabase/DataImportService/DataImportService.setCubeImportProps.htm).

		 Значение


		 Краткое описание


		 Unknown

		 Excel


		 Microsoft Office Excel.


		 ExcelEx


		 Microsoft Office Excel 2010.


		 OleDb


		 Object Linking and Embedding Database.


		 Access


		 Microsoft Office Access.


		 Xml


		 eXtensible Markup Language.


		 Text


		 Текст.


		 Mb


		 Метабаза.
