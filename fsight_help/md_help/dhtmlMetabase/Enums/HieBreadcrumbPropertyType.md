# HieBreadcrumbPropertyType


Пространство имен:
 PP.Mb


## Описание


Перечисление HieBreadcrumbPropertyType содержит
 варианты изменения параметров для дерева измерения.


## Допустимые значения


		 Значение


		 Краткое описание


		 DimContentRefreshed


		 Изменение содержимого дерева измерения.


		 DimReloaded


		 Перезагрузка дерева измерения.


		 DimSelectionChanged


		 Изменение отметки.


		 SelectedItemsChanged


		 Изменение выбранных элементов.


		 TreeNameAttrChanged


		 Изменение отображаемого имени.
