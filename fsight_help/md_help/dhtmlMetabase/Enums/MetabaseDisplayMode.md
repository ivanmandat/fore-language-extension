# MetabaseDisplayMode


Пространство имен:
 PP.Mb.Ui


## Описание


Перечисление MetabaseDisplayMode содержит
 варианты отображения заголовков вершин дерева.


Используется свойством [MetabaseTreeList.DisplayMode](../Classes/Metabase/MetabaseTreeList/MetabaseTreeList.DisplayMode.htm).


## Допустимые значения


		 Значение


		 Краткое описание


		 Names


		 В заголовке отображаются только
		 имена.


		 Idents


		 В заголовке отображаются только
		 идентификаторы.


		 Both


		 В заголовке отображаются имена
		 и идентификаторы.
