# FaultCodes


Пространство имен: PP.Mb;


## Описание


Перечисление FaultCodes содержит
 коды ошибок сервиса SoapFault.


Используется методом [LoginForm.checkPwdPolicy](dhtmlApp.chm::/Classes/Application/LoginForm/LoginForm.checkPwdPolicy.htm).


## Допустимые значения


		 Значение


		 Краткое описание


		 0


		 None.
		 Код ошибки отсутствует.


		 5248


		 USERMUSTCHANGEPASSWORD.
		 Требуется сменить пароль.


		 5265


		 KE_SOM_PWD_LESSMINLENGTH.
		 Требуется установить минимальную длину пароля в символах.


		 5313


		 KE_SOM_USERLOGON_BLOCKED.
		 Требуется заблокировать пользователя после указанного числа неудачных
		 попыток входа.


		 5249


		 KE_SOM_PWD_EQUSERNAME.
		 Пароль и имя пользователя должны быть различными.


		 5262


		 KE_SOM_PWD_EQFULLUSERNAME.
		 Пароль и полное имя пользователя должны быть различными


		 5263


		 KE_SOM_PWD_CHARSDIF.
		 Пароль должен состоять из разных символов.


		 5309


		 KE_SOM_PWD_LETTERCASE.
		 Пароль должен содержать буквы разного регистра.


		 5959


		 KE_SOM_SECCMNT_PASSREQUIREDIGITSANDCHARS_ON.
		 Пароль должен содержать цифры и буквы.


		 5264


		 KE_SOM_PWD_UNIQDEGREE.
		 Не достигнуто минимальное число позиций, в которых новый пароль
		 должен отличаться от старого (%s).


		 5250


		 KE_SOM_PWD_CHARSINVALID.
		 Пароль может содержать только буквы a-z, цифры 0-9, символы '%s'
		 и должен начинаться с буквы или цифры, или '_'.


		 5251


		 KE_SOM_PWD_CHARSDIGSSYMS.
		 Пароль должен содержать хотя бы одну букву a-z, цифру 0-9 и символ
		 '%s'.


		 5821


		 KE_SOM_PWD_MAX_REPEATED_CHARS.
		 Превышено максимально допустимое количество повторяющихся символов
		 в пароле (%s).


		 26893


		 KE_SOM_PWD_EQUSPWD.
		 Требуется выбрать новый пароль, поскольку пользователь уже
		 пользовался данным паролем.


		 5961


		 KE_SOM_SECCMNT_PASSREQUIRENONALPHABETICCHARS_ON.
		 Пароль должен содержать спецсимволы (_, @, #, $, %, &, *,
		 ^ ).


		 5535


		 KE_SOM_PWD_STRINGFORBIDDEN.
		 Пароль не может содержать строку '%s'.


		 5272


		 Warning,
		 INVALIDID. Предупреждение.


		 5985


		 KE_SOM_DBNAME_FAILED.
		 Требуется указать правильное наименование базы данных репозитория.


		 6157


		 INVALID_OBJID.
		 Ошибка.


		 11


		 FILE_UPLOAD_ERROR.
		 Ошибка при импорте файла.


		 6195


		 INVALID_SESSION_MONIKER.
		 Некорректный идентификатор сессии метабазы.


		 3086


		 ID_PILIB_LICEXCEPTION.
		 Отсутствует лицензия на данный инструмент.
