# TranslateMode


Пространство имен:
 PP.Mb


## Описание


Перечисление TranslateMode содержит
 режимы отображения строк для ввода переводов названий объектов на разных
 языках в диалоге «[Переводы](../Classes/Metabase/Dialogs/TranslateDialog/TranslateDialog.htm)».


Используется свойством [TranslateDialog.Mode](../Classes/Metabase/Dialogs/TranslateDialog/TranslateDialog.Mode.htm).


## Допустимые значения


		 Значение


		 Краткое описание


		 Translations


		 В диалоге будут отображаться строки ввода для языков, на
		 которые задан перевод в свойстве [TranslateDialog.Translations](../Classes/Metabase/Dialogs/TranslateDialog/TranslateDialog.Translations.htm).


		 Languages


		 В диалоге будут отображаться строки ввода для языков, заданных
		 в свойстве [TranslateDialog.DisplayLanguages](../Classes/Metabase/Dialogs/TranslateDialog/TranslateDialog.DisplayLanguages.htm).


		 Both


		 В диалоге будут отображаться и строки ввода для языков,
		 на которые задан перевод в свойстве [TranslateDialog.Translations](../Classes/Metabase/Dialogs/TranslateDialog/TranslateDialog.Translations.htm),
		 и строки ввода для языков, заданных в свойстве [TranslateDialog.DisplayLanguages](../Classes/Metabase/Dialogs/TranslateDialog/TranslateDialog.DisplayLanguages.htm).
