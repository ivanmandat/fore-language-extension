# NavigatorBox


## Описание


Компонент NavigatorBox представляет
 собой контейнер для дерева папок и списка объектов репозитория.


## Комментарии


Компонент реализован классом [NavigatorBox](../../../Classes/Metabase/NavigatorBox/NavigatorBox.htm).
 Входит в состав компонента [Navigator](../../../Classes/Metabase/Navigator/Navigator.htm).
