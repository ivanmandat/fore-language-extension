# Работа с репозиторием


## Описание


На данной странице приведены компоненты для работы с репозиторием.


## Компоненты


		 Имя компонента
		 Краткое описание


		 ![](../Class_Image.gif)
		 [DataMiningMaster](Metabase/DataMiningMaster/DataMiningMaster.htm)
		 Компонент DataMiningMaster
		 представляет собой мастер «Интелектуального
		 анализа данных».


		 ![](../Class_Image.gif)
		 [ImportDataWizard](Metabase/ImportDataWizard/ImportDataWizard.htm)
		 Компонент ImportDataWizard
		 представляет собой мастер для импорта данных.


		 ![](../Class_Image.gif)
		 [MetabaseObjectsIconView](Metabase/MetabaseObjectsIconView/MetabaseObjectsIconView.htm)
		 Компонент MetabaseObjectsIconView
		 представляет собой контейнер для списка объектов репозитория в
		 виде пиктограмм.


		 ![](../Class_Image.gif)
		 [MetabaseObjectsListView](Metabase/MetabaseObjectsListView/MetabaseObjectsListView.htm)
		 Компонент MetabaseObjectsListView
		 представляет собой контейнер для отображения [списка
		 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm) выбранной папки в окне навигации репозитория.


		 ![](../Class_Image.gif)
		 [MetabaseTreeList](Metabase/MetabaseTreeList/MetabaseTreeList.htm)
		 Компонент MetabaseTreeList
		 представляет собой иерархическое дерево объектов репозитория.
		 Каждый элемент дерева является ссылкой на соответствующий объект
		 репозитория.


		 ![](../Class_Image.gif)
		 [Navigator](Metabase/Navigator/Navigator.htm)
		 Компонент Navigator
		 представляет собой навигатор объектов репозитория платформы.


		 ![](../Class_Image.gif)
		 [NavigatorBox](Metabase/NavigatorBox/NavigatorBox.htm)
		 Компонент NavigatorBox
		 представляет собой контейнер для дерева папок и списка объектов
		 репозитория.


		 ![](../Class_Image.gif)
		 [NavigatorRibbon](Metabase/NavigatorRibbon/NavigatorRibbon.htm)
		 Компонент NavigatorRibbon
		 представляет собой ленту инструментов [навигатора
		 объектов](Metabase/Navigator/Navigator.htm).


		 ![](../Class_Image.gif)
		 [ChangePasswordDialog](Metabase/Dialogs/ChangePasswordDialog/ChangePasswordDialog.htm)
		 Компонент ChangePasswordDialog
		 представляет собой диалог для смены пароля.


		 ![](../Class_Image.gif)
		 [MetabaseDialogBase](Metabase/Dialogs/MetabaseDialogBase/MetabaseDialogBase.htm)
		 Компонент MetabaseDialogBase**
		 представляет собой базовый диалог с репозиторием.


		 ![](../Class_Image.gif)
		 [MetabaseOpenDialog](Metabase/Dialogs/MetabaseOpenDialog/MetabaseOpenDialog.htm)
		 Компонент **MetabaseOpenDialog**
		 представляет собой диалог открытия объекта из репозитория.


		 ![](../Class_Image.gif)
		 [MetabaseSaveDialog](Metabase/Dialogs/MetabaseSaveDialog/MetabaseSaveDialog.htm)
		 Компонент **MetabaseSaveDialog**
		 представляет собой диалог сохранения объекта в репозитории.


		 ![](../Class_Image.gif)
		 [SearchDialog](Metabase/Dialogs/SearchDialog/SearchDialog.htm)
		 Компонент SearchDialog
		 представляет собой диалог поиска объекта репозитория.


		 ![](../Class_Image.gif)
		 [ServiceUnavailableDialog](Metabase/Dialogs/ServiceUnavailableDialog/ServiceUnavailableDialog.htm)
		 Компонент ServiceUnavailableDialog
		 представляет собой диалог, в котором отображается статус состояния
		 связи с сервером при разрыве соединения.


		 ![](../Class_Image.gif)
		 [TranslateDialog](Metabase/Dialogs/TranslateDialog/TranslateDialog.htm)
		 Компонент TranslateDialog
		 представляет собой диалог для сохранения объекта с именами на
		 разных языках.
