# NavigatorBox.createFolder


## Синтаксис


createFolder(name);


## Параметры


name. Название создаваемой
 папки. По умолчанию создается папка с именем «Папка<ключ
 папки>».


## Описание


Метод createFolder создает папку.


## Комментарии


Метод срабатывает автоматически при нажатии на кнопку [ленты](../../../Components/Metabase/NavigatorRibbon/NavigatorRibbon.htm)
 «Новая папка» на вкладке «Главная».
