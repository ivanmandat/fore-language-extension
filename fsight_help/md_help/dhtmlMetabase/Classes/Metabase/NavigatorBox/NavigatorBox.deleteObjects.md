# NavigatorBox.deleteObjects


## Синтаксис


deleteObjects();


## Описание


Метод deleteObjects удаляет
 выделенные объекты.


## Комментарии


Метод срабатывает при нажатии кнопки [ленты](../../../Components/Metabase/NavigatorRibbon/NavigatorRibbon.htm)
 «Удалить» на вкладке «Главная».
