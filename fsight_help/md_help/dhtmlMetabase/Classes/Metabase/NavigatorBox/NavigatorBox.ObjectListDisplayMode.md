# NavigatorBox.ObjectListDisplayMode


## Синтаксис


ObjectListDisplayMode: PP.MB.Ui.[MetabaseDisplayMode](../../../Enums/MetabaseDisplayMode.htm);


## Описание


Свойство ObjectListDisplayMode
 определяет, что будет отображаться в названии объекта в списке объектов.


## Комментарии


Значение свойства - элемент перечисления [MetabaseDisplayMode](../../../Enums/MetabaseDisplayMode.htm).
