# NavigatorBox.cut


## Синтаксис


cut();


## Описание


Метод cut добавляет выбранные
 объекты в буфер обмена, при этом удаляя их.


## Комментарии


Метод срабатывает при нажатии на кнопку [ленты](../../../Components/Metabase/NavigatorRibbon/NavigatorRibbon.htm)
 «Вырезать» на вкладке «Главная».
