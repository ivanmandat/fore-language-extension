# NavigatorBox.refreshForKey


## Синтаксис


refreshForKey(rootKey, withCache);


## Параметры


rootKey. Ключ папки, содержимое
 которой нужно обновить. Если параметр не задан, то будет обновлена текущая
 папка;


withCache. Если установлено
 значение false (по умолчанию),
 то при обновлении компонента данные берутся из кеша. Если установлено
 значение true, то кеш очищается
 и его содержимое обновляется вместе с компонентом для того, чтобы получить
 наиболее актуальные данные.


## Описание


Метод refreshForKey обновляет
 содержимое папки с заданным ключом.
