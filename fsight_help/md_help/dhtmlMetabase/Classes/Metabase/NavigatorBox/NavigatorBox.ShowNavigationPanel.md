# NavigatorBox.ShowNavigationPanel


## Синтаксис


ShowNavigationPanel: Boolean;


## Описание


Свойство ShowNavigationPanel
 определяет, отображается ли дерево папок.


## Комментарии


Если установлено значение false,
 дерево папок скрыто. По умолчанию установлено значение true,
 то есть дерево папок отображается.
