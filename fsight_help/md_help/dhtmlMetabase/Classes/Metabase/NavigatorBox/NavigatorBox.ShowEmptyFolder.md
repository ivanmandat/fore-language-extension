# NavigatorBox.ShowEmptyFolder


## Синтаксис


ShowEmptyFolder: Boolean;


## Описание


Свойство ShowEmptyFolder определяет,
 будут ли отображаться в дереве пустые папки.


## Комментарии


По умолчанию установлено значение true,
 то есть пустые папки отображаются.

Пример использования аналогичен примеру, приведенному для свойства [MetabaseTreeList.ShowEmptyFolder](../MetabaseTreeList/MetabaseTreeList.ShowEmptyFolder.htm).
