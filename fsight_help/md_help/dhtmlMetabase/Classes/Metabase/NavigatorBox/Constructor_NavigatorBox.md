# Конструктор NavigatorBox


## Синтаксис


NavigatorBox(settings);


## Параметры


settings. JSON-объект со значениями
 свойств компонента.


## Описание


Конструктор NavigatorBox создает
 экземпляр компонента [NavigatorBox](../../../Components/Metabase/NavigatorBox/NavigatorBox.htm).
