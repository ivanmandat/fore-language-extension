# NavigatorBox.ObjectListMultiSelect


## Синтаксис


ObjectListMultiSelect: Boolean;


## Описание


Свойство ObjectListMultiSelect
 устанавливает возможность множественного выбора объектов в списке.


## Комментарии


Если для свойства установлено значение true,
 значит включена возможность множественной отметки объектов в списке. По
 умолчанию установлено значение false,
 то есть данная возможность отключена.


Свойство может использоваться только из JSON в конструкторе компонента.
