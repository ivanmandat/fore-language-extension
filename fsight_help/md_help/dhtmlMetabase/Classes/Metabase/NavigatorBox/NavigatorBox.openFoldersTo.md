# NavigatorBox.openFoldersTo


## Синтаксис


openFoldersTo(key);


## Параметры


key. Ключ объекта, до которого
 нужно раскрыть все папки.


## Описание


Метод openFoldersTo раскрывает
 все папки до заданного объекта.
