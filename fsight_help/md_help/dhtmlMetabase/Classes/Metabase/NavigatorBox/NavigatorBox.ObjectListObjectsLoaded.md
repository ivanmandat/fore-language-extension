# NavigatorBox.ObjectListObjectsLoaded


## Синтаксис


ObjectListObjectsLoaded: function (sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие ObjectListObjectsLoaded
 наступает при загрузке списка объектов репозитория.


## Комментарии


Список объектов репозитория загружается при нажатии на кнопку [ленты](../../../Components/Metabase/NavigatorRibbon/NavigatorRibbon.htm)
 «Обновить».
