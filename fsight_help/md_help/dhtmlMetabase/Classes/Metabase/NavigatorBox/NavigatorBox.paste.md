# NavigatorBox.paste


## Синтаксис


paste();


## Описание


Метод paste осуществляет вставку
 объекта из буфера обмена.


## Комментарии


Метод срабатывает при нажатии на кнопку [ленты](../../../Components/Metabase/NavigatorRibbon/NavigatorRibbon.htm)
 «Вставить» на вкладке «Главная».
