# NavigatorBox.FolderTreeFilter


## Синтаксис


FolderTreeFilter: Array;


## Описание


Свойство FolderTreeFilter определяет,
 какие контейнерные объекты будут отображаться в дереве папок.


## Комментарии


Значение свойства - массив элементов перечисления PP.Mb.Ui.[MetabaseObjectClass](KeSom.chm::/Enums/MetabaseObjectClass.htm).
