# NavigatorBox.ObjectListFilters


## Синтаксис


ObjectListFilters: Array;


## Описание


Свойство ObjectListFilters определяет
 типы объектов, которые будут отображаться в списке объектов.


## Комментарии


Значение свойства - массив элементов перечисления PP.Mb.Ui.[MetabaseObjectClass](KeSom.chm::/Enums/MetabaseObjectClass.htm).
