# NavigatorBox.ObjectSelected


## Синтаксис


ObjectSelected: function (sender, args)


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие ObjectSelected наступает
 при изменении отметки в списке объектов.
