# GridView.setScroll


## Синтаксис


setScroll(scroll: PP.Exp.Ui.TableEnum, value: Boolean)


## Параметры


scroll - Тип полосы прокрутки;


value - Признак отображения
 полосы прокрутки. Если значение равно true,
 то полоса будут отображена, иначе - не будет.


## Описание


Метод setScroll определяет признак
 видимости горизонтально и вертикальной полос прокрутки таблицы.
