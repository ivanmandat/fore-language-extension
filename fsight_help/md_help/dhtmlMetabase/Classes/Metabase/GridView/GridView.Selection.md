# GridView.Selection


## Синтаксис


Selection: Object;


## Описание


Свойство Selection определяет текущее выделение в таблице.


## Комментарии


Значение свойства устанавливается при помощи метода setSelection, а возвращается при помощи метода getSelection. Из JSON значение свойства установить нельзя.


Свойство содержит значение в виде JSON-объекта со следующими полями: range - область выделения в таблице экспресс-отчета, type - тип выделения.
