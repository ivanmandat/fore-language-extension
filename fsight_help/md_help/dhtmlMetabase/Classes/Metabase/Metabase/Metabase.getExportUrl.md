# Metabase.getExportUrl


## Синтаксис


getExportUrl ()


## Описание


Метод getExportUrl возвращает
 url экспорта данных.


## Комментарии


Для установки url экспорта данных используется свойство Metabase.ExportUrl,
 значение которого задается только из JSON.
