# Metabase.UserCreds


## Синтаксис


UserCreds: PP.Mb.[UserCreds](../UserCreds/UserCreds.htm)


## Описание


Свойство UserCreds устанавливает
 данные пользователя для подключения к репозиторию.


## Комментарии


Если данные не указаны, то будет использоваться доменная авторизация.


Свойство может быть установлено только из JSON при [создании](Constructor_Metabase.htm)
 соединения к репозиторию.
