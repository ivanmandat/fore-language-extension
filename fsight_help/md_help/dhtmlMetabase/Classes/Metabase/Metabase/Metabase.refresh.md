# Metabase.refresh


## Синтаксис


refresh (callback, errorCallback)


## Параметры


callback. Процедура обратного
 вызова;


errorCallback. Процедура обратного
 вызова при выполнении метода с ошибками.


## Описание


Метод refresh обновляет соединение
 с репозиторием.
