# Metabase.close


## Синтаксис


close();


## Описание


Метод close закрывает соединение
 с репозиторием.


## Пример


Для выполнения примера предполагается, что создано соединение с репозиторием
 с наименованием mb и сервис для работы с измерениями с наименованием dimSrv
 (см. «[Пример
 размещения компонента DimCombo](dhtmlDim.chm::/Components/Metabase/DimCombo/DimCombo_Example.htm)»).


Добавим кнопку, при нажатии на которую будет закрыто соединение с репозиторием:


var button = new PP.Ui.Button({
    Content: "Закрыть соединение",
    Click: function () {
        mb.close();
    },
    ParentNode: document.getElementById('button'),
});
mb.Closed.add(function () {
    alert("Соединение с репозиторием закрыто")
});

После выполнения примера при нажатии на кнопку будет закрыто соединение
 с репозиторием. На экран будет выведено сообщение «Соединение с репозиторием
 закрыто».
