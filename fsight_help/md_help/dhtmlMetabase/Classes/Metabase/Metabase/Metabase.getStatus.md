# Metabase.getStatus


## Синтаксис


getStatus (callback, errorCallback)


## Параметры


callback. Процедура обратного вызова;


errorCallback. Процедура обратного вызова в случае ошибки.


## Описание


Метод getStatus проверяет состояние сессии репозитория.
