# Metabase.moveObject


## Синтаксис


moveObject (parentKey, moveToParentKey, callback)


## Параметры


parentKey. Ключ объекта, который необходимо переместить;


moveToParentKey. Ключ контейнера, в который будет перемещен объект;


callback. Процедура обратного вызова.


## Описание


Метод moveObject используется для перемещения объекта.


## Пример


Для выполнения примера предполагается наличие на странице соединения с репозиторием с наименованием «mb» (см. «[Пример создания компонента Navigator](../../../Components/Metabase/Navigator/Navigator_Example.htm)»). предполагается наличие в репозитории объекта с ключом 469534 и контейнера с ключом 780083. Выполним перемещение:


mb.moveObject(469534, 780083)


mb.refresh()


После выполнения примера объект с ключом 469534 будет расположен в контейнере с ключом 780083.
