# Metabase.flushCache


## Синтаксис


flushCache(callback, errorCallback)


## Параметры


callback. Процедура обратного вызова;


errorCallback. Процедура обратного вызова при выполнении метода с ошибками.


## Описание


Метод flushCache сбрасывает весь кеш репозитория.
