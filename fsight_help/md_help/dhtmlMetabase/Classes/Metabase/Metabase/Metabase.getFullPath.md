# Metabase.getFullPath


## Синтаксис


getFullPath (objKey, callback);


## Параметры


objKey. Ключ
 объекта, родительские объекты которого нужно запросить;


callback. Процедура
 обратного вызова.


## Описание


Метод getFullPath отправляет
 запрос на получение данных о родительских каталогах для указанного объекта.


## Пример


Для выполнения примера предполагается наличие на html-странице объекта
 для к подключения репозиторию с идентификатором «mb» (см. «[Пример
 создания компонента Navigator](../../../Components/Metabase/Navigator/Navigator_Example.htm)»). Предполагается наличие на странице
 объекта с ключом 2033. Отправим на сервер запрос для получения информации
 о родительских каталогов для объекта с указанным ключом:


mb.getFullPath(2033, function (sender, args) {
    var parentObjects = JSON.parse(args.ResponseText).GetObjectsResult.objects;
    console.log(parentObjects)
});
После выполнения примера в консоль браузера будет выведен массив родительских
 каталогов для указанного объекта. Если указанный объект является контейнерным,
 будет выведен массив родительских и дочерних объектов.
