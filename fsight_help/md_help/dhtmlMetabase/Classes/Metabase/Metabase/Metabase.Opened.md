# Metabase.Opened


## Синтаксис


Opened: function (sender, args)


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие Opened наступает при
 открытии репозитория.


## Комментарии


Для открытия репозитория используется метод [Metabase.open](Metabase.open.htm).
