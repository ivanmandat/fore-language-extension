# Metabase.saveObject


## Синтаксис


saveObject (objOrOdId, callback)


## Параметры


objOrOdId. Моникёр сохраняемого объекта;


callback. Процедура обратного вызова.


## Описание


Метод saveObject используется для сохранения открытого объекта.


## Комментарии


Для выполнения операции необходимо в параметре objOrOdId указать моникер объекта, открытого на редактирование и содержащего какие-либо изменения.


## Пример


Для выполнения примера предполагается наличие на странице соединения с репозиторием с наименованием «metabase» и модели экспресс-отчета с наименованием «eaxAnalyzer». Внесем какое-либо изменение в экспресс-отчет и вызовем метод saveObject:


metabase.saveObject(eaxAnalyzer.getOdId())


После выполнения примера изменения в открытом экспресс-отчете будут сохранены.
