# Metabase.EnableCaching


## Синтаксис


EnableCaching: Boolean


## Описание


Свойство EnableCaching определяет,
 будет ли использоваться кеширование объектов репозитория.


## Комментарии


Если для свойства установлено значение false,
 кеширование объектов репозитория использоваться не будет. По умолчанию
 установлено значение true, то
 есть объекты кешируются.
