# Metabase.getMbSec


## Синтаксис


getMbSec(callback);


## Параметры


callback. Процедура обратного вызова.


## Описание


Метод getMbSec запрашивает дополнительные настройки репозитория.


## Комментарии


К дополнительным параметрам репозитория относятся параметры менеджера безопасности.


## Пример


Для выполнения примера предполагается наличие на странице соединения с репозиторием с наименованием «mb» (см. «[Пример создания компонента Navigator](../../../Components/Metabase/Navigator/Navigator_Example.htm)»). Вернем объект с описанием параметров менеджера безопасности репозитория:


mb.getMbSec(function (sender, args)


{


    var res = JSON.parse(args.ResponseText).GetMbSecResult;


    console.log(res)


})


После выполнения примера в консоль будет выведен объект с описанием параметров менеджера безопасности репозитория.
