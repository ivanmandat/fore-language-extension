# Metabase.deleteObjects


## Синтаксис


deleteObjects (arrayKeysForDelete, callback)


## Параметры


arrayKeysForDelete. Массив ключей объектов, которые необходимо удалить;


callback. Процедура обратного вызова. Необязательный параметр.


## Описание


Метод deleteObjects удаляет объекты репозитория по их ключам.


## Пример


Для выполнения примера предполагается наличие на странице соединения с репозиторием с наименованием «mb» (см. «[Пример создания компонента Navigator](../../../Components/Metabase/Navigator/Navigator_Example.htm)»). Необходимо, чтобы в репозиторию были объект с ключами 782569, 782580, 782579, 782571. Удалим объекты с перечисленными ключами:


mb.deleteObjects([782569, 782580, 782579, 782571])


mb.refresh()


После выполнения примера из репозитория будут удалены объекты с указанными ключами.
