# ServiceRequest.ShowWaiterTimeout


## Синтаксис


ShowWaiterTimeout: Number;


## Описание


Свойство ShowWaiterTimeout определяет
 задержку до отображения индикатора загрузки.


## Комментарии


Значение свойства устанавливается из JSON и при помощи метода setShowWaiterTimeout,
 возвращается с помощью метода getShowWaiterTimeout.


Значение по умолчанию: 0.
