# ServiceRequest.ShowWaiter


## Синтаксис


ShowWaiter: Boolean;


## Описание


Свойство ShowWaiter определяет,
 будет ли отображаться индикатор загрузки.


## Комментарии


Значение свойства устанавливается из JSON и при помощи метода setShowWaiter,
 возвращается с помощью метода getShowWaiter.


Если свойство содержит значение true
 (по умолчанию), то индикатор загрузки будет отображаться, иначе - не будет.
