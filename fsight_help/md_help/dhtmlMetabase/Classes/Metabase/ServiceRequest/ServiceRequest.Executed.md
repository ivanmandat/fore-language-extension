# ServiceRequest.Executed


## Синтаксис


Executed: function (sender, args)


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Событие Executed наступает после
 выполнения запроса к сервису.


## Комментарии


Событие срабатывает только в том случае, если в очереди есть запросы.
