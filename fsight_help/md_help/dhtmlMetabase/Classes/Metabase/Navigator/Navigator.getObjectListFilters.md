# Navigator.getObjectListFilters


## Синтаксис


getObjectListFilters ()


## Описание


Метод getObjectListFilters возвращает типы объектов, которые отображаются в навигаторе.


## Комментарии


Метод возвращает массив элементов перечисления PP.Mb.Ui.[MetabaseObjectClass](KeSom.chm::/Enums/MetabaseObjectClass.htm).

Пример использования метода приведен в описании события [Navigator.ObjectListObjectsLoaded](Navigator.ObjectListObjectsLoaded.htm).
