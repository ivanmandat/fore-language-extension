# Descriptor.TimeStamp


## Синтаксис


TimeStamp: Date;


## Описание


Свойство TimeStamp определяет
 дату и время последнего изменения объекта репозитория.


## Комментарии


Значение свойства задается в конструкторах или с помощью метода setTimeStamp, а возвращается с помощью
 метода getTimeStamp.
