# Descriptor.Metadata


## Синтаксис


Metadata: Object;


## Описание


Свойство определяет метаданные объекта.


## Комментарии


Значение свойства устанавливается из JSON и при помощи метода setMetadata и возвращается при помощи
 метода getMetadata.


Для класса PP.Exp.[EaxAnalyzer](dhtmlExpress.chm::/Classes/Express/EaxAnalyzer/EaxAnalyzer.htm)
 метод setMetadata переопределен.
 Для метода определены следующие параметры:


value. Метаданные отчета.


isDefaultMd. Определяет, содержат
 ли метаданные, указанные в параметре value,
 все необходимые данные для отчета. Если установлено значение true,
 значит метаданные содержат все необходимые данные для отчета. Если установлено
 значение false, будет выполняться
 слияние с уже доступными данными.
