# Descriptor.HasChildren


## Синтаксис


HasChildren: Boolean;


## Описание


Свойство HasChildren определяет
 признак наличия дочерних объектов.


## Комментарии


Допустимые значения:


	- True. Наличие дочерних
	 объектов доступно;


	- False. По умолчанию.
	 Наличие дочерних объектов недоступно.


Значение свойства задается в конструкторах, а возвращается с помощью
 метода getHasChildren.
