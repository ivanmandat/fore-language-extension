# Descriptor.IsShortcut


## Синтаксис


IsShortcut: Boolean;


## Описание


Свойство IsShortcut возвращает
 признак того, что данный объект является ярлыком на другой объект репозитория.


## Комментарии


Допустимые значения:


	- True. Объект является
	 ярлыком на другой объект репозитория;


	- False. По умолчанию.
	 Объект не является ярлыком на другой объект репозитория.


Значение свойства задается в конструкторах, а возвращается с помощью
 метода getIsShortcut.
