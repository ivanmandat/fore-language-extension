# Descriptor.IsLink


## Синтаксис


IsLink: Boolean;


## Описание


Свойство IsLink возвращает
 признак того, что данный объект находится в другом репозитории.


## Комментарии


Допустимые значения:


	- True. Объект находится
	 в другом репозитории;


	- False. По умолчанию.
	 Объект находится в текущем репозитории.


Значение свойства задается в конструкторах, а возвращается с помощью
 метода getIsLink.
