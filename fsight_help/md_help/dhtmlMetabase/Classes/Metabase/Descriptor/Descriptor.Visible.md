# Descriptor.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible возвращает
 признак видимости объекта репозитория.


## Комментарии


Допустимые значения:


	- True. Объект репозитория
	 не отображается;


	- False. По умолчанию.
	 Объект репозитория отображается.


Значение свойства задается в конструкторах, а возвращается с помощью
 метода getVisible.
