# Descriptor.Class


## Синтаксис


Class: Number;


## Описание


Свойство Class определяет класс
 объекта репозитория.


## Комментарии


Значение свойства задается в конструкторах или с помощью метода setClass, а возвращается с помощью
 метода getClass.
