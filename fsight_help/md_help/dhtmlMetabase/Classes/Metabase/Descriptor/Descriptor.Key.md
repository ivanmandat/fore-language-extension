# Descriptor.Key


## Синтаксис


Key: Number;


## Описание


Свойство Key определяет ключ
 объекта репозитория.


## Комментарии


Значение свойства задается в конструкторах или с помощью метода setKey, а возвращается с помощью
 метода getKey.
