# ElsFilter.MbJSONDelegate


## Синтаксис


MbJSONDelegate: PP.Delegate;


## Описание


Свойство MbJSONDelegate определяет
 обработчик генерации JSON'а методом [ElsFilter.getFilterJSON](ElsFilter.getFilterJSON.htm).


## Комментарии


Значение свойства устанавливается из JSON. Значение свойства возвращается
 с помощью метода getMbJSONDelegate().


## Пример


Пример использования приведен в описании свойства [ElsFilter.IncludeParents](ElsFilter.IncludeParents.htm).
