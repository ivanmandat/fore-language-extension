# ElsFilter.isCustomHash


## Синтаксис


isCustomHash(): Boolean;


## Описание


Метод isCustomHash определяет,
 был ли хеш установлен пользователем.


## Комментарии


Возвращает значение true, если
 параметры фильтрации были установлены методом [ElsFilter.setCustomHash](ElsFilter.setCustomHash.htm).
