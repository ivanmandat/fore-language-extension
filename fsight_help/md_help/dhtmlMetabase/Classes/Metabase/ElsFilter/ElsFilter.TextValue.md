# ElsFilter.TextValue


## Синтаксис


TextValue: String;


## Описание


Свойство TextValue определяет
 строку поиска при фильтрации по атрибутам.


## Комментарии


Значение свойства устанавливается из JSON. Значение свойства возвращается
 с помощью метода getTextValue().
