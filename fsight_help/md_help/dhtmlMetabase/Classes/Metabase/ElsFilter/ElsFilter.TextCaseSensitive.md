# ElsFilter.TextCaseSensitive


## Синтаксис


TextCaseSensitive: Boolean;


## Описание


Свойство TextCaseSensitive определяет,
 учитывать ли регистр при фильтрации по атрибутам.


## Комментарии


Значение свойства устанавливается из JSON. Значение свойства возвращается
 с помощью метода getTextCaseSensitive().
