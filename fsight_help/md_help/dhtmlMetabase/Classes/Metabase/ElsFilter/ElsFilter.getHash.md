# ElsFilter.getHash


## Синтаксис


getHash(): String;


## Описание


Метод getHash возвращает хеш-строку
 с параметрами фильтрации.


## Комментарии


Значение строки может принимать вид:


"{"text":{"wholeWordsOnly":false,"caseSensitive":false,"text":"200","attributes":"NAME"},"levels":-1,"includeRoot":false,
"includeParents":true,"includeParentsWithSiblings":false,"elementsGroup":true,"onlySelected":false,"onlyWithData":false}"

## Пример


Пример использования приведен в описании свойства [ElsFilter.IncludeParents](ElsFilter.IncludeParents.htm).
