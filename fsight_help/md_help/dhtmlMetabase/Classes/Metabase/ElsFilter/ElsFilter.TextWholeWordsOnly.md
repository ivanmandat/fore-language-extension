# ElsFilter.TextWholeWordsOnly


## Синтаксис


TextWholeWordsOnly: Boolean;


## Описание


Свойство TextWholeWordsOnly
 определяет поиск только по целым словам при фильтрации элементов.


## Комментарии


Значение свойства устанавливается из JSON. Значение свойства возвращается
 с помощью метода getWholeWordsOnly().
