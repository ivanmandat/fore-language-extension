# ElsFilter.getFilterJSON


## Синтаксис


getFilterJSON(levels, includeRoot): Object;


## Параметры


levels. Number. Количество
 уровней, участвующих в фильтрации. Если не указано, принимает значение
 «-1» и все дерево участвует в фильтрации.


includeRoot. Boolean. Включать
 ли в фильтрацию корневую вершину. По умолчанию false - не включать.


## Описание


Метод getFilterJSON возвращает
 настройки фильтрации в формате tns_PP7_SOM:ElsFilter.
