# ElsFilter.TextAttributes


## Синтаксис


TextAttributes: Array;


## Описание


Свойство TextAttributes определяет
 массив идентификаторов атрибутов, по которым производится фильтрация.


## Комментарии


Значение свойства устанавливается из JSON. Значение свойства возвращается
 с помощью метода getTextAttributes().
