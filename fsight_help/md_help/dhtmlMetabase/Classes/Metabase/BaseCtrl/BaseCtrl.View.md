# BaseCtrl.View


## Синтаксис


View: PP.[Control](dhtmlUi.chm::/Classes/control/control.htm)


## Описание


Свойство View определяет визуальный
 компонент, который используется BI-инструментом.


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [DimTree](dhtmlDim.chm::/Components/Metabase/DimTree/DimTree.htm)
 с наименованием «dimTree» (см. «[Пример
 создания компонента DimTree](dhtmlDim.chm::/Components/Metabase/DimTree/DimTree_Example.htm)»):


dimTree.getDimTreeCtrl().getView()


Метод getView вернет экземпляр
 класса PP.Mb.Ui.[DimTree](dhtmlDim.chm::/Classes/Metabase/DimTree/DimTree.htm).
