# PPService.endSyncXHR


## Синтаксис


endSyncXHR ();


## Описание


Метод endSyncXHR отключает режим синхронных запросов для экземпляра класса [PPService](PPService.htm).


## Комментарии


Метод endSyncXHR используется в паре с методом [PPService.beginSyncXHR](PPService.beginSyncXHR.htm). Эти методы используются, когда требуется остановить поток исполнения javascript-сценариев до получения ответа от веб-сервиса.
