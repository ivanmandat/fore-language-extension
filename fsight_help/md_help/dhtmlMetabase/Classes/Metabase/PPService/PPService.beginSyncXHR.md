# PPService.beginSyncXHR


## Синтаксис


beginSyncXHR ();


## Описание


Метод beginSyncXHR включает режим синхронных запросов для экземпляра класса [PPService](PPService.htm).


## Комментарии


Метод beginSyncXHR используется в паре с методом [PPService.endSyncXHR](PPService.endSyncXHR.htm). Эти методы используются, когда требуется остановить поток исполнения javascript-сценариев до получения ответа от веб-сервиса.
