# PPService.Error


## Синтаксис


Error: function (sender, args)


## Параметры


sender. Источник события;


args. Информация о событии.
 Аргумент ResponseText - текст
 ошибки.


## Описание


Событие Error наступает, если
 во время соединения с репозиторием произошла ошибка.
