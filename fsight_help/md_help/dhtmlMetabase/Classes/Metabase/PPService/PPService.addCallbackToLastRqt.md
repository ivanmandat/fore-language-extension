# PPService.addCallbackToLastRqt


## Синтаксис


addCallbackToLastRqt (callback);


## Параметры


callback. Возвратная функция, которая будет добавлена к последнему отправленному запросу. Возвращает значение true, если обработчик был добавлен, иначе false, то есть активных запросов нет.


## Описание


Метод addCallbackToLastRqt добавляет обработчик к последнему в очереди запросу.
