# ImportDataWizard.DefaultImportFolderDesc


## Синтаксис


DefaultImportFolderDesc: Object;


## Описание


Свойство DefaultImportFolderDesc
 устанавливает дескриптор папки для сохранения объектов импорта по умолчанию.


## Комментарии


Может быть заполнен частично, например: {k: 11574, i: null, n: null,
 c: 0}


где, k - ключ объекта репозитория.
