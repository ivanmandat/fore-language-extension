# Resource.DefaultCulture


## Синтаксис


DefaultCulture: Number;


## Описание


Свойство DefaultCulture определяет
 язык и региональные настройки по умолчанию.


## Комментарии


Язык и региональные настройки по умолчанию применяются, если для другого
 языка или региональных настроек на задано значение.


Значение свойства - [четырехзначный
 цифровой код языка](http://msdn.microsoft.com/en-us/goglobal/bb964664.aspx).
