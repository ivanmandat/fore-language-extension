# TabSheetDataSource.ObjectsAsImage


## Синтаксис


ObjectsAsImage: Boolean


## Описание


Свойство ObjectsAsImage определяет,
 будут ли объекты на табличном листе отображаться как картинки.


## Комментарии


Если для свойства установлено значение true,
 объекты будут отображаться как картинки.
