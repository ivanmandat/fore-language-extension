# TabSheetDataSource.SearchComplited


## Синтаксис


SearchComplited: function (sender, args)


## Параметры


sender. Источник события;


args. Информация о событии.
 Доступные аргументы: SearchResult
 (массив с найденными элементами таблицы, содержащий координаты строк и
 столбцов).


## Описание


Событие SearchComplited наступает
 после поиска данных в таблице.
