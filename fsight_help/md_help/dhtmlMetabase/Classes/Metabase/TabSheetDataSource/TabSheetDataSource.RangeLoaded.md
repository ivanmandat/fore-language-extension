# TabSheetDataSource.RangeLoaded


## Синтаксис


RangeLoaded:  function (sender, args)


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: Data (возвращает объект с описанием структуры таблицы) и Ranges (возвращает объект с описанием выделенных ячеек таблицы).


## Описание


Событие RangeLoaded наступает после загрузки диапазонов таблицы.
