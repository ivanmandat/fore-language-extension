# TabSheetDataSource.MeasuresLoaded


## Синтаксис


MeasuresLoaded: function (sender, args)


## Параметры


sender. Источник события;


args. Информация о событии. Доступные аргументы: Data (возвращает объект с описанием структуры таблицы) и Selection (возвращает объект с описанием выделенных ячеек таблицы).


## Описание


Событие MeasuresLoaded наступает при [загрузке](TabSheetDataSource.loadMeasures.htm) структуры таблицы.


## Пример


Пример использования события приведен в описании метода [TabSheetDataSource.loadMeasures](TabSheetDataSource.loadMeasures.htm).
