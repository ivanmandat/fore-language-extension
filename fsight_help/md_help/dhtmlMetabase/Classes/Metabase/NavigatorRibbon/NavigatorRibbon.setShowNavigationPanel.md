# NavigatorRibbon.setShowNavigationPanel


## Синтаксис


setShowNavigationPanel(showNavigationPanel)


## Параметры


showNavigationPanel. Определяет, нажата или отжата кнопка «Дерево объектов» на вкладке «Вид». По умолчанию кнопка нажата. При установке значения false кнопка отжата.


## Описание


Метод setShowNavigationPanel определяет, находится ли кнопка «Дерево объектов» на вкладке «Вид» в нажатом состоянии.
