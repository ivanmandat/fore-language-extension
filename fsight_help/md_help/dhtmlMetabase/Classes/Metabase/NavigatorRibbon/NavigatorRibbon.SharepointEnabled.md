# NavigatorRibbon.SharepointEnabled


## Синтаксис


SharepointEnabled: Boolean


## Описание


Свойство SharepointEnabled определяет,
 отображается ли кнопка «Обсудить в SharePoint»
 на вкладке «Главная».


## Комментарии


Свойство можно задать только из JSON при [создании
 компонента](Constructor_NavigatorRibbon.htm).


Чтобы кнопка «Обсудить в SharePoint»
 отображалась, установите для свойства значение true.
 По умолчанию установлено значение false,
 то есть кнопка скрыта.
