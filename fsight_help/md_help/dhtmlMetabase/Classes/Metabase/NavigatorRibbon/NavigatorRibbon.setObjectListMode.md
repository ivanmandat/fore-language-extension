# NavigatorRibbon.setObjectListMode


## Синтаксис


setObjectListMode (objectListMode)


## Параметры


objectListMode. Определяет вид отображения списка объектов. Элемент перечисления PP.Mb.Ui.[NavigatorBox.ObjectListMode](../../../Enums/NavigatorBox.ObjectListMode.htm). По умолчанию установлено значение «Table».


## Описание


Метод setObjectListMode определяет, какая из кнопок на вкладке «Вид», «Мелкие значки», «Обычные значки» или «Таблица», находится в нажатом состоянии
