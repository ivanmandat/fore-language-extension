# PropertyChangedEventArgs.isCanceled


## Синтаксис


isCanceled();


## Описание


Метод isCanceled** определяет, отменено ли событие, вызываемое после изменения значения свойства.


## Комментарии


Метод возвращает значение true, если событие, вызываемое после изменения значения свойства, было отменено, в противном случае - значение false.
