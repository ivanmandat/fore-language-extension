## Синтаксис


Callback: PP.Delegate


## Описание


Свойство Callback** определяет возвратную функцию.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setCallback, а возвращается - с помощью метода getCallback.


Возвратная функция будет выполняться по завершении события, вызываемого при изменении значений свойств.
