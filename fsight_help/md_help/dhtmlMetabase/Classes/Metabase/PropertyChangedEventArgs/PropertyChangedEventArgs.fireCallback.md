# PropertyChangedEventArgs.fireCallback


## Синтаксис


fireCallback(sender, args);


## Параметры


sender. Источник события;


args. Информация о событии.


## Описание


Метод fireCallback** единожды вызывает возвратную функцию, определённую в свойстве [Callback](PropertyChangedEventArgs.Callback.htm).
