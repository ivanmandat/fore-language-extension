# MetaHierarchy.IsAttributeFilter


## Синтаксис


IsAttributeFilter: Boolean;


## Описание


Свойство IsAttributeFilter определяет,
 установлен ли фильтр для атрибута с указанным идентификатором.


## Комментарии


Если для свойства установлено значение true,
 значит фильтр установлен, иначе - не установлен.


## Пример


Для выполнения примера предполагается наличие измерения с ключом 0 и
 атрибута с идентификатором «DL». Необходимо создать сервис для работы
 с временными рядами, получить экземпляр рабочей книги из базы данных временных
 рядов, получить экземпляр иерархии метаданных с наименованием «hie» (см.
 «[MetaHierarchy.ChildEls](MetaHierarchy.ChildEls.htm)») и в
 обработчике события открытия документа добавить следующий код:


console.log("Определяем, установлен ли фильтр для атрибута с идентификатором DL");
var isAttrFilter = hie.getIsAttributeFilter("DL");
console.log(isAttrFilter);

После выполнения примера в консоль браузера будет выведен признак установленного
 фильтра:


Определяем, установлен ли фильтр для атрибута с
 идентификатором DL


true
