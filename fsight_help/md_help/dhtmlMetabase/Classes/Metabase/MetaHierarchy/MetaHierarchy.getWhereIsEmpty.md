# MetaHierarchy.getWhereIsEmpty


## Синтаксис


getWhereIsEmpty ();


## Описание


Метод getWhereIsEmpty определяет,
 будут ли загружены элементы, для которых нет данных.


## Пример


Для выполнения примера необходимо создать сервис для работы с временными
 рядами, получить экземпляр рабочей книги из базы данных временных рядов
 (см. «[MetaHierarchy.ChildEls](MetaHierarchy.ChildEls.htm)»)
 и в обработчике события открытия документа добавить следующий код:


console.log("Получаем признак определяющий, загружать ли элементы, для которых установлен признак отсутствия данных");
var isEmpty = hie.getWhereIsEmpty();
console.log(isEmpty);

После выполнения примера в консоль браузера будет выведен признак,
 определяющий загружать ли элементы, для которых установлен признак отсутствия
 данных:


0
