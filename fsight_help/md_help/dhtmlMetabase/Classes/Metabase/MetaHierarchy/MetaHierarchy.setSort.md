# MetaHierarchy.setSort


## Синтаксис


setSort (metaAttrId, isActive, desc, dimAttrId);


## Параметры


metaAttrId. Идентификатор метаатрибута;


isActive. Определяет, необходимо
 ли включить сортировку: если true,
 сортировка включена, если false
 - выключена;


desc. Сортировка: true
 - сортировка по убыванию, false
 - по возрастанию, null - по умолчанию;


dimAttrId. Идентификатор атрибута
 измерения, по значению которого будет выполнена сортировка.


## Описание


Метод setSort устанавливает
 сортировку в дереве рядов.


## Пример


Для выполнения примера необходимо создать сервис для работы с временными
 рядами, получить экземпляр рабочей книги из базы данных временных рядов
 (см. «[MetaHierarchy.ChildEls](MetaHierarchy.ChildEls.htm)»)
 и в обработчике события открытия документа добавить следующий код:


console.log("Устанавливаем сортировку в дереве рядов");
hie.setSort("DL", true, true, 1);

После выполнения примера в консоли браузера будет установлена сортировка
 в дереве рядов.
