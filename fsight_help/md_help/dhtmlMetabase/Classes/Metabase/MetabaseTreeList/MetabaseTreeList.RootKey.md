# MetabaseTreeList.RootKey


## Синтаксис


RootKey: Number


## Описание


Свойство RootKey устанавливает ключ корневой папки.


## Комментарии


По умолчанию корневой папкой является [папка с ключом 0](MetabaseTreeList.getShowRootFolder.htm).
