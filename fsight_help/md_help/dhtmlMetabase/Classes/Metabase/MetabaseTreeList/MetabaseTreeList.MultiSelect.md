# MetabaseTreeListe.MultiSelect


## Синтаксис


MultiSelect: Boolean


## Описание


Свойство MultiSelect определяет, возможен ли множественный выбор элементов дерева.


## Комментарии


Если для свойства установлено значение true, доступен множественный выбор элементов дерева. По умолчанию установлено значение false, то есть можно выбрать только один элемент.
