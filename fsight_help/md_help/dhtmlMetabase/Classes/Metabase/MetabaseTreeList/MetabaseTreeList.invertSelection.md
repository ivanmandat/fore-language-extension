# MetabaseTreeList.invertSelection


## Синтаксис


invertSelection ()


## Описание


Метод invertSelection инвертирует отметку в дереве объектов, то есть отмечает те элементы, которые не были отмечены, и снимает отметку с тех, которые были отмечены.
