# MetabaseTreeList.selectAll


## Синтаксис


selectAll ()


## Описание


Метод selectAll выделяет все вершины дерева.


## Комментарии


При помощи метода выделяются только видимые вершины дерева. Если родительская вершина не развернута, то дочерние вершины не выделяются.
