# MetabaseTreeList.Filter


## Синтаксис


Filter: Array of PP.Mb.[MetabaseObjectClass](KeSom.chm::/Enums/MetabaseObjectClass.htm)


## Описание


Свойство Filter определяет список типов объектов, отображаемых в дереве компонента.


## Комментарии


Значение свойства - массив элементов перечисления PP.Mb.Ui.[MetabaseObjectClass](KeSom.chm::/Enums/MetabaseObjectClass.htm).
