# MetabaseTreeList.refreshAll


## Синтаксис


refreshAll (withCache)


## Параметры


withCache. Если установлено значение false (по умолчанию), то при обновлении компонента данные берутся из кеша. Если установлено значение true, то кеш очищается и его содержимое обновляется вместе с компонентом для того, чтобы получить наиболее актуальные данные.


## Описание


Метод refreshAll обновляет все дерево объектов репозитория.
