# MetabaseTreeList.EmptySelection


## Синтаксис


EmptySelection: Boolean


## Описание


Свойство EmptySelection определяет, можно ли снять отметку полностью со всех элементов дерева.


## Комментарии


Если для свойства установлено значение false, значит нельзя снять отметку полностью со всех элементов дерева. То есть, если в дереве выделен один элемент, снять выделение, щёлкнув по нему, нельзя. По умолчанию установлено значение true, то есть можно снять отметку с единственного выделенного элемента, повторно выбрав его.
