# MetabaseTreeList.getPathToRoot


## Синтаксис


getPathToRoot(key, onlyKey)


## Параметры


key. Ключ
 объекта, путь до которого нужно определить


onlyKey. Определяет,
 будет ли метод возвращать только ключи объектов, в которые вложен выбранный
 объект. Если параметр не задан, возвращается только массив ключей. Если
 установлено значение false, будет возвращаться массив объектов -
 экземпляров класса PP.Mb.[Object](../Object/Object.htm).


## Описание


Метод getPathToRoot возвращает
 массив объектов или их ключей, в которые вложен указанный объект.
