# MetabaseTreeList.clearFilter


## Синтаксис


clearFilter (notRefresh)


## Параметры


notRefresh Определяет, будет ли обновляться дерево. Если параметр не задан, дерево будет обновляться автоматически после очистки фильтра. Если установлено значение true, дерево обновляться не будет.


## Описание


Метод clearFilter очищает список типов объектов, отображающихся в дереве.


## Комментарии


После применения метода в дереве отображаются объекты всех типов.
