# MetabaseTreeList.ShowEmptyFolder


## Синтаксис


ShowEmptyFolder: Boolean


## Описание


Свойство ShowEmptyFolder определяет, будут ли отображаться в дереве пустые папки.


## Комментарии


По умолчанию установлено значение true, то есть пустые папки отображаются.
