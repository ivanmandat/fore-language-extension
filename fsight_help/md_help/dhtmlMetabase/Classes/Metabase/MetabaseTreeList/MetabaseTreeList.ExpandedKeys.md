# MetabaseTreeList.ExpandedKeys


## Синтаксис


ExpandedKeys: Array of Number || Number


## Описание


Свойство ExpandedKeys устанавливает ключ объекта или массив ключей объектов, которые отображаются в дереве как развернутые вершины.


## Комментарии


Значение свойства нельзя задать из JSON в [конструкторе](Constructor_MetabaseTreeList.htm) компонента. Для установки значения используйте метод setExpandedKeys.
