# MetabaseTreeList.ShowAttributes


## Синтаксис


ShowAttributes: Boolean


## Описание


Свойство ShowAttributes определяет, будут ли отображаться столбцы с атрибутами объектов.


## Комментарии


По умолчанию установлено значение true, то есть столбцы отображаются.
