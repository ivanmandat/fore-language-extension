# MetabaseTreeList.getSelectedObjects


## Синтаксис


getSelectedObjects()


## Описание


Метод getSelectedObjects возвращает
 массив выбранных объектов - экземпляров класса PP.Mb.[Object](../Object/Object.htm).


## Комментарии


Возможность множественного выбора объектов устанавливается в свойстве
 [MetabaseTreeList.MultiSelect](MetabaseTreeList.MultiSelect.htm).
