# DataMiningRibbonPanel.Source


## Синтаксис


Source : PP.Mb.DataMiningSource;


## Описание


Свойство Source определяет источник
 анализа данных.


## Комментарии


Значение свойства устанавливается через JSON и при помощи метода setSource, а возвращается при помощи
 метода getSource.
