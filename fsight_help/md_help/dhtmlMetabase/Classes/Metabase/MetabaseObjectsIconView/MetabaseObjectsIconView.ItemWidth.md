# MetabaseObjectsIconView.ItemWidth


## Синтаксис


ItemWidth: Number


## Описание


Свойство ItemWidth определяет
 ширину объектов.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setItemWidth,
 а возвращается с помощью метода getItemWidth.
