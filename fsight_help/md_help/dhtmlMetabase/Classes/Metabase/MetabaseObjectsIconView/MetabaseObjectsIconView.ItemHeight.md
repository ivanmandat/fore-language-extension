# MetabaseObjectsIconView.ItemHeight


## Синтаксис


ItemHeight: Number


## Описание


Свойство ItemHeight определяет
 высоту объектов.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setItemHeight,
 а возвращается с помощью метода getItemHeight.
