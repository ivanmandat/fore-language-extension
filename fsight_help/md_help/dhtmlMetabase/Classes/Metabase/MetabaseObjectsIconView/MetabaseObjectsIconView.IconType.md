# MetabaseObjectsIconView.IconType


## Синтаксис


IconType: [PP.Mb.Ui.NavigatorBox.ObjectListMode](../../../Enums/NavigatorBox.ObjectListMode.htm)


## Описание


Свойство IconType определяет
 вариант отображения списка объектов.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setIconType,
 а возвращается с помощью метода getIconType.
