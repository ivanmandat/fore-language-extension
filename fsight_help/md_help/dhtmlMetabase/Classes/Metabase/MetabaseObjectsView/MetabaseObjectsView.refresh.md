# MetabaseObjectsView.refresh


## Синтаксис


refresh(rootKey, withCache);


## Параметры


rootKey. Number. Ключ объекта
 репозитория, который необходимо обновить. Необязательный параметр. По
 умолчанию обновляет все объекты;


withCache. Booleam. Учитывать
 ли кешированные данные. Необязательный параметр. По умолчанию содержит
 значение true.


## Описание


Метод refresh обновляет состояние
 указанного объекта в компоненте.
