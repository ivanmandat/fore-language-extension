# MetabaseObjectsView.EditOnLoad


## Синтаксис


EditOnLoad : Number;


## Описание


Свойство EditOnLoad определяет
 ключ объекта, который необходимо переименовать при загрузке.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setEditOnLoad,
 а возвращается с помощью метода getEditOnLoad.


## Пример


Свойство предназначено для внутренней реализации.
