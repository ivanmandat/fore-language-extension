# MetabaseObjectsView.getCustomClassObject


## Синтаксис


getCustomClassObject(obj);


## Параметры


obj. [PP.Mb.Object](../Object/Object.htm).
 Объект.


## Описание


Метод getCustomClassObject возвращает
 пользовательский класс объекта.


## Комментарии


Пользовательский класс должен быть задан в свойстве [MetabaseObjectsView.CustomClasses](MetabaseObjectsView.CustomClasses.htm).
