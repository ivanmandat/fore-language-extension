# MetabaseObjectsView.refreshAll


## Синтаксис


refreshAll(withCache);


## Параметры


withCache. Booleam. Учитывать
 ли кешированные данные. Необязательный параметр. По умолчанию содержит
 значение true.


## Описание


Метод refreshAll обновляет состояние
 всех объектов в компоненте.


## Пример


Использование метода аналогично методу [MetabaseObjectsView.refresh](MetabaseObjectsView.refresh.htm),
 примененному к корневому объекту репозитория.
