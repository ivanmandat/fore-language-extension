# MetabaseObjectsView.endUpdate


## Синтаксис


endUpdate();


## Описание


Метод endUpdate включает обратно
 перерисовку компонента, используемого для отображения объектов.


## Комментарии


Метод необходим, для возобновления работы компонента, после использования
 метода [MetabaseObjectsView.beginUpdate](MetabaseObjectsView.beginUpdate.htm).
