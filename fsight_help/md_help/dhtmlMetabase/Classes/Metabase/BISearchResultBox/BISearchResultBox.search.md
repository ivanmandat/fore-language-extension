# BISearchResultBox.search


## Синтаксис


search(query, range);


## Параметры


query. String. Строка поиска;


range. Object. Диапазон количества
 записей. Объект должен иметь следующую структуру: range = { start : 0,
 count: 100}, где start = первый индекс для поиска, count = количество
 объектов для поиска.


## Описание


Метод search производит поиск
 по данным репозитория.
