# BISearchResultBox.formatData


## Синтаксис


formatData(objs);


## Параметры


objs. Array. Массив полученных
 объектов поиска.


## Описание


Метод formatData форматирует
 полученные объекты поиска для отображения в компоненте.


## Комментарии


Метод используется для внутренней реализации метода [BISearchResultBox.search](BISearchResultBox.search.htm).
