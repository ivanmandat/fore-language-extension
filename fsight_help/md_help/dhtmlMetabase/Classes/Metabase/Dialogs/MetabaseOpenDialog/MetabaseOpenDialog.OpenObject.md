# MetabaseOpenDialog.OpenObject


## Синтаксис


OpenObject: function(sender, args);


## Параметры


*sender*. Источник события;


*args*. Информация о событии. Доступный аргумент - JSON-объект
 с полем Objects, являющимся массивом объектов типа [PP.Mb.Object](../../Object/Object.htm).


## Описание


Событие OpenObject** наступает
 при открытии объекта репозитория в соответствующем диалоге.
