# Конструктор MetabaseOpenDialog


## Синтаксис


MetabaseOpenDialog (settings);


## Параметры


settings. JSON-объект со значениями свойств класса.


## Описание


Конструктор MetabaseOpenDialog создает экземпляр класса [MetabaseOpenDialog](MetabaseOpenDialog.htm).
