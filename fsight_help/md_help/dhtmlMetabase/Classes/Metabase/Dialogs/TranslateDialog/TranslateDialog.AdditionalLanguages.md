# TranslateDialog.AdditionalLanguages


## Синтаксис


AdditionalLanguages: Array


## Описание


Свойство AdditionalLanguages устанавливает
 массив кодов языков, которые будут отображаться при нажатии на кнопку
 «Еще язык».


## Комментарии


Значение свойства - массив [четырехзначных
 цифровых кодов языков](http://msdn.microsoft.com/en-us/goglobal/bb964664.aspx).
