# TranslateDialog.DisplayLanguages


## Синтаксис


DisplayLanguages: Array


## Описание


Свойство DisplayLanguages устанавливает
 массив кодов языков, которые будут отображаться в диалоге «Переводы».


## Комментарии


Значение свойства - массив [четырехзначных
 цифровых кодов языков](http://msdn.microsoft.com/en-us/goglobal/bb964664.aspx).
