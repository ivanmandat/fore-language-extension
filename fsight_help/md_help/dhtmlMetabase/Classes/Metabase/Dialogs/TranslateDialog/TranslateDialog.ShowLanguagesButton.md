# TranslateDialog.ShowLanguagesButton


## Синтаксис


ShowLanguagesButton: Boolean


## Описание


Свойство ShowLanguagesButton
 определяет, будет ли доступна кнопка «Еще язык» для выбора дополнительных
 языков.


## Комментарии


Если для свойства установлено значение true,
 кнопка «Еще язык» отображается. Если установлено значение false
 (по умолчанию), кнопка «Еще язык» скрыта.
