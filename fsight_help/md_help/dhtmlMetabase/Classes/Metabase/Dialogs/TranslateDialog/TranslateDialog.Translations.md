# TranslateDialog.Translations


## Синтаксис


Translations: Object


## Описание


Свойство Translations устанавливает
 переводы для указанных языков.


## Комментарии


Значение свойства - JSON-объект. Поля объекта - [четырехзначные
 цифровые кодов языков](http://msdn.microsoft.com/en-us/goglobal/bb964664.aspx), для которых будут установлены строки с переводами.
 Значения полей - строки с наименованиями объектов на указанных языках.
