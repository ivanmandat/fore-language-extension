# Конструктор TranslateDialog


## Синтаксис


PP.Mb.Ui.TranslateDialog (settings);


## Параметры


settings. JSON-объект со значениями
 свойств класса.

Конструктор TranslateDialog
 создает экземпляр компонента TranslateDialog.
