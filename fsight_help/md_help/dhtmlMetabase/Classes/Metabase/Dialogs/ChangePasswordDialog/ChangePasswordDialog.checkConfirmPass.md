# ChangePasswordDialog.checkConfirmPass


## Синтаксис


checkConfirmPass ()


## Описание


Метод checkConfirmPass используется для проверки на совпадение нового пароля с подтвержденным.


## Комментарии


Метод возвращает true, если. Иначе возвращает false.
