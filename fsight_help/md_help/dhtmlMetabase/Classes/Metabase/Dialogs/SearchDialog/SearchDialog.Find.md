# SearchDialog.Find


## Синтаксис


Find: function(sender, args);


## Параметры


*sender*. Источник события;


*args*. Информация о событии. Доступные аргументы - JSON-объект
 с полями:


	- Object. JSON-объект
	 с настройками найденного объекта репозитория;


	- Index. Индекс найденного
	 объекта репозитория (начиная с 0).


## Описание


Событие Find наступает при нажатии
 на кнопку «Найти».
