# SearchDialog.setCurrentFolder


## Синтаксис


setCurrentFolder (val);


## Параметры


val. Ключ
 папки, в которой будет осуществлен поиск.


## Описание


Метод setCurrentFolder устанавливает
 папку, в которой будет осуществляться поиск.


## Комментарии


Чтобы выполнить поиск по текущей папке, необходимо выбрать в раскрывающемся
 списке «Область поиска» пункт «Текущая папка».


## Пример


Для выполнения примера предполагается наличие на странице компонента
 [SearchDialog](../../../../Components/Metabase/Dialogs/SearchDialog/SearchDialog.htm)
 с наименованием «searchDialog» (см. «[Пример
 создания компонент SearchDialog](../../../../Components/Metabase/Dialogs/SearchDialog/SearchDialog_example.htm)»). Предполагается наличие в репозитории
 папки с ключом 2055. Установим текущую папку:


searchDialog.setCurrentFolder(2055);
В раскрывающемся списке «Область поиска» станет доступен пункт «Текущая
 папка». Выберите этот пункт, введите искомое значение и нажмите кнопку
 «Найти»/«Найти все». В консоли будут выведены названия и свойства объектов,
 найденных в указанной папке.
