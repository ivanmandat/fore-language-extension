# SearchDialog.FindAll


## Синтаксис


FindAll: function(sender, args);


## Параметры


*sender*. Источник события;


*args*. Информация о событии. Доступные аргументы - JSON-объект
 с полями:


	- Objects. Массив JSON-объектов
	 с настройками найденных объектов репозитория.


## Описание


Событие FindAll наступает при
 нажатии на кнопку «Найти все».
