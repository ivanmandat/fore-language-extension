# DataImportService.GetObjectsCompleted


## Синтаксис


GetObjectsCompleted: function (sender, args)


## Параметры


sender. Источник события;


args. Информация о событии. Доступный аргумент: Objects - массив полученных JSON-объектов типа Od.


## Описание


Событие GetObjectsCompleted наступает после получения объектов репозитория.
