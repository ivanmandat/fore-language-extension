# DataImportService.ImportCompleted


## Синтаксис


ImportCompleted: function (sender, args)


## Параметры


sender. Источник события;


args. Информация о событии. Доступны следующие аргументы: CubeInfo - возвращает модель объекта импорта данных, Error - определяет, завершился ли импорт с ошибкой, ErrorInfo - информация об ошибке.


## Описание


Событие ImportCompleted наступает при завершении импорта данных.
