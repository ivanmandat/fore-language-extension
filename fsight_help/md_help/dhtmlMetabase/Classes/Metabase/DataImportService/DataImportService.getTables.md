# DataImportService.getTables


## Синтаксис


getTables (cubeImport, callback, errorCallback)


## Параметры


cubeImport. Модель объекта
 импорта в куб, экземпляр класса [PP.Mb.CubeImport](../CubeImport/CubeImport.htm).


callback. Функция обратного вызова.


errorCallback. Функция обратного вызова при ошибке.


## Описание


Метод getTables запрашивает
 метаданные о таблицах объекта импорта.

Метод предназначен для внутреннего использования.
