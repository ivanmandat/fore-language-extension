# Param.getPPCreateObj


## Синтаксис


getPPCreateObj();


## Описание


Метод getPPCreateObj** возвращает свойства параметра в виде JSON-объекта.


## Комментарии


Метод возвращает JSON-объект со следующими полями: Data со свойством Binding - настройки отображения параметра, содержащиеся в [Param.Binding](Param.Binding.htm), IsNullable - признак того, может ли параметр принимать значение null, IsVisible - признак видимости параметра, MaxValue - максимальное значение параметра, MinValue - его минимальное значение, PPType - тип, Value - текущее значение.
