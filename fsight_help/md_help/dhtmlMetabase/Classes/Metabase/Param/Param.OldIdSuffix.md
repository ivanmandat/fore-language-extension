# Param.OdIdSuffix


## Синтаксис


OldIdSuffix: String;


## Описание


Свойство OdIdSuffix определяет
 суффик моникера.


## Комментарии


Свойство используется для обращения к коллекции параметров.


Значение свойства устанавливается из JSON и с помощью метода setOdIdSuffix,
 а возвращается - с помощью метода getOdIdSuffix.
