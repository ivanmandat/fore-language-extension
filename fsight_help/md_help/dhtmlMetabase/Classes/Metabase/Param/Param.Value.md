# Param.Value


## Синтаксис


Value: Boolean|Number|String|Object


## Описание


Свойство Value** определяет значение параметра.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setValue, а возвращается - с помощью метода getValue.
