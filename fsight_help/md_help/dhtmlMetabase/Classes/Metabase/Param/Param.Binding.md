# Param.Binding


## Синтаксис


Binding: String


## Описание


Свойство Binding** устанавливает параметры отображения для заданного параметра объекта репозитория.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setBinding, а возвращается - с помощью метода getBinding.
