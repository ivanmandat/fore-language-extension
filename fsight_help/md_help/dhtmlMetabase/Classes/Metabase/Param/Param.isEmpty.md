# Param.isEmpty


## Синтаксис


isEmpty();


## Описание


Метод isEmpty** возвращает признак того, определено ли значение параметра.


## Комментарии


Метод возвращает false, если значение параметра определено, и true, если не определено, то есть равно undefined.
