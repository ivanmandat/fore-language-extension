# Param.Visible


## Синтаксис


Visible: Boolean


## Описание


Свойство Visible** управляет
 видимостью параметра на [панели
 «Параметры источников данных»](dhtmlExpress.chm::/Classes/Express/DataSourceParamsPanel/DataSourceParamsPanel.htm).


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setVisible,
 а возвращается - с помощью метода getVisible.


Свойство содержит значение true,
 если параметр отображается на панели «Параметры источников данных», и
 false, если не отображается.
