# GxTitle.Title


## Синтаксис


Title: String


## Описание


Свойство Title содержит отформатированное
 значение текста заголовка.


## Комментарии


Значение свойства устанавливается с помощью метода setTitle,
 а возвращается с помощью метода getTitle.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 [ExpressBox](dhtmlExpress.chm::/Components/Express/ExpressBox/ExpressBox.htm)
 с наименованием «expressBox» (см. «[Пример
 создания компонента ExpressBox](dhtmlExpress.chm::/Components/Express/ExpressBox/ExpressBox_Example.htm)»). Для появления заголовка
 необходимо нажать на кнопку «Заголовок» на вкладке «Главная». Получим
 отформатированное значение текста заголовка:


// Получаем заголовок
var title = expressBox.getDataView().getTitleView();
// Получаем отформатированное значение текста заголовка
console.log("Отформатированное значение текста заголовка: " + title.getTitle());
В результате в консоль будет выведено конечное значение текста заголовка:


Отформатированное значение текста заголовка: Данные
 департаментов региональной экономики и программ, ВАЛОВОЙ РЕГИОНАЛЬНЫЙ
 ПРОДУКТ, Значение
