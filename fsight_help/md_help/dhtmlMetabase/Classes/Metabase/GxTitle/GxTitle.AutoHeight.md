# GxTitle.AutoHeight


## Синтаксис


AutoHeight: Boolean


## Описание


Свойство AutoHeight определяет,
 будет ли заголовок автоматически выравниваться по высоте.


## Комментарии


Допустимые значения:


	- true.
	 Заголовок будет автоматически выравнен по высоте;


	- false.
	 Заголовок не будет выравнен по высоте (по умолчанию).


Значение свойства устанавливается из JSON и с помощью метода setAutoHeight,
 а возвращается с помощью метода getAutoHeight.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 [ExpressBox](dhtmlExpress.chm::/Components/Express/ExpressBox/ExpressBox.htm)
 с наименованием «expressBox» (см. «[Пример
 создания компонента ExpressBox](dhtmlExpress.chm::/Components/Express/ExpressBox/ExpressBox_Example.htm)»). Для появления заголовка необходимо
 нажать на кнопку «Заголовок» на вкладке «Главная». Проверим использование
 автоматического выравнивания заголовка по высоте:


// Получаем заголовок
var title = expressBox.getDataView().getTitleView();
// Проверяем использование автоматического выравнивания заголовка по высоте
if (title.getAutoHeight()) {
    console.log("Заголовок автоматически выравнен по высоте");
} else {
    console.log("Заголовок не выравнен по высоте");
}
В результате в консоль будет выведен результат проверки использования
 выравнивания заголовка по высоте:


Заголовок не выравнен по высоте
