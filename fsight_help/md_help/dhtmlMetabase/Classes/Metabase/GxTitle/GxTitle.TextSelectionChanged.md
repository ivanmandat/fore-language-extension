# GxTitle.TextSelectionChanged


## Синтаксис


TextSelectionChanged: function(sender, args)


## Параметры


sender. Источник события;


args. Информация
 о событии. Аргументы: Alignment
 - параметры выравнивания текста; BackgroundColor
 - цвет фона заголовка; FontBgColor
 - цвета фона текста; FontColor
 - цвет текста; FontSize - размер
 текста; FontFamily - наименование
 шрифта; IsBold - признак полужирного
 начертания текста; IsItalic -
 признак курсивного начертания текста; IsUnderline
 - признак подчеркнутого начертания текста.


## Описание


Событие TextSelectionChanged
 наступает после изменения текста в редакторе заголовка.
