# Конструктор GxTitle


## Синтаксис


PP.Mb.Ui.GxTitle(settings);


## Параметры


*settings.* JSON-объект
 со значениями свойств класса.


## Описание


Конструктор GxTitle создает
 экземпляр класса [GxTitle](GxTitle.htm).


## Пример


Конструктор предназначен для внутреннего использования.
