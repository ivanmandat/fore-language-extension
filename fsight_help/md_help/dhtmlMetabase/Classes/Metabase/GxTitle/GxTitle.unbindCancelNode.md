# GxTitle.unbindCancelNode


## Синтаксис


unbindCancelNode(domNode: HTMLElement; remove: Boolean);


## Параметры


domNode. DOM-вершина;


remove. Признак удаления обработчика
 события нажатия мыши на DOM-вершину. Если для параметра установлено значение
 true, то обработчик события нажатия
 мыши будет удален.


## Описание


Метод unbindCancelNode отключает
 режим редактирования заголовка при нажатии на указанную DOM-вершину.
