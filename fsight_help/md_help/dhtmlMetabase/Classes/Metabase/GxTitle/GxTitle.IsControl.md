# GxTitle.IsControl


## Синтаксис


IsControl: Boolean


## Описание


Свойство IsControl определяет,
 является ли заголовок элементом управления.


## Комментарии


Допустимые значения:


	- true.
	 Заголовок является элементом управления (по умолчанию);


	- false.
	 Заголовок не является элементом управления.


Значение свойства устанавливается из JSON и с помощью метода setIsControl,
 а возвращается с помощью метода getIsControl.


## Пример


Для выполнения примера необходимо наличие на html-странице компонента
 [ExpressBox](dhtmlExpress.chm::/Components/Express/ExpressBox/ExpressBox.htm)
 с наименованием «expressBox» (см. «[Пример
 создания компонента ExpressBox](dhtmlExpress.chm::/Components/Express/ExpressBox/ExpressBox_Example.htm)»). Для появления заголовка необходимо
 нажать на кнопку «Заголовок» на вкладке «Главная». Проверим, является
 ли заголовок элементом управления:


// Получаем заголовок
var title = expressBox.getDataView().getTitleView();
// Проверяем, является ли заголовок элементом управления
if (title.getIsControl()) {
    console.log("Заголовок является элементом управления");
} else {
    console.log("Заголовок не является элементом управления");
}
В результате в консоль будет выведен результат проверки:


Заголовок является элементом управления
