# GxTitle.beginEdit


## Синтаксис


beginEdit();


## Описание


Метод beginEdit включает режим
 редактирования.


## Комментарии


Режим редактирования может быть включен, если для свойства [EnableEdit](GxTitle.EnableEdit.htm)
 установлено значение true. Режим
 редактирования выключается при вызове метода [endEdit](GxTitle.endEdit.htm).


## Пример


Пример использования метода приведен в описании свойства [GxTitle.EnableEdit](GxTitle.EnableEdit.htm).
