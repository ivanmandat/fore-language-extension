# GxTitle.setFontColor


## Синтаксис


setFontColor(value: PP.[Color](dhtmlCommon.chm::/Classes/PP/Color/Color.htm)
 || RGB || String);


## Параметры


value. Устанавливаемый цвет.


## Описание


Метод setFontColor устанавливает
 цвет шрифта текста заголовка.


## Комментарии


Метод является аналогом метода [setTextColor](GxTitle.TextColor.htm).


## Пример


Пример использования аналогичен примеру использования метода setTextColor,
 приведенного в описании свойства [GxTitle.TextColor](GxTitle.TextColor.htm).
