# GxTitle.HtmlTitle


## Синтаксис


HtmlTitle: String


## Описание


Свойство HtmlTitle содержит
 отформатированное значение текста заголовка на языке разметки HTML.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setHtmlTitle,
 а возвращается с помощью метода getHtmlTitle.
