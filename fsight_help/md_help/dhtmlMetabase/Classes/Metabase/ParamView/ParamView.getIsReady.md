# ParamView.getIsReady


## Синтаксис


getIsReady ();


## Описание


Метод getIsReady определяет, инициализирован ли элемент управления параметра.


## Комментарии


Метод возвращает true, если элемент управления параметра инициализирован, иначе возвращает false.
