# ParamView.ParamLabel


## Синтаксис


ParamLabel: String


## Описание


Свойство ParamLabel устанавливает подпись для параметра.


## Комментарии


Значение свойства можно установить только из JSON при [создании](Constructor_ParamView.htm) экземпляра класса [ParamView](ParamView.htm).


По умолчанию для параметра установлена подпись, совпадающая с его названием.
