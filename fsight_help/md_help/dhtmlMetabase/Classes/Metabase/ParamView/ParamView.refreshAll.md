# ParamView.refreshAll


## Синтаксис


refreshAll(reInit);


## Параметры


reInit. Определяет, будет ли перерисовываться компонент. Если установлено значение true - будет, если false - не будет (по умолчанию).


## Описание


Метод refreshAll обновляет элемент управления параметром отчета.


## Комментарии


Является аналогом метода [ParamView.refresh](ParamView.refresh.htm).
