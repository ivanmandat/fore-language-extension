# ParamView.refresh


## Синтаксис


refresh (reInit);


## Параметры


reInit. Определяет, будет ли перерисовываться компонент. Если установлено значение true - будет, если false - не будет (по умолчанию).


## Описание


Метод refresh обновляет элемент управления параметром отчета.


## Пример


Для выполнения примера предполагается наличие на странице элемента управления параметром отчета с наименованием «param» (см. пример на странице события [ParamView.LinkedDimOpened](ParamView.LinkedDimOpened.htm)). Добавим кнопку, при нажатии на которую будет обновлен данный элемент управления и при этом он будет перерисован:


var but = new PP.Ui.Button({


    Content: "Обновить",


    ParentNode: document.body,


    Click: function () {


        param.refresh(true)


    }


});


После выполнения примера на страницу будет добавлена кнопка «Обновить», при нажатии на которую будет обновлен раскрывающийся список справочника: если были отмечены какие-либо элементы измерения, то отметка будет снята. Также будут сгенерированы события [ParamView.LinkedDimOpened](ParamView.LinkedDimOpened.htm) и [ParamView.ViewReady](ParamView.ViewReady.htm)
