# HieService.openFromRubKey


## Синтаксис


openFromRubKey(rubKey, wbk, callback);


## Параметры


rubKey. Задает ключ
 каталога показателей, иерархия которого открывается;


wbk. Задает
 значение рабочей книги;


callback. Задает обработчик
 окончания выполнения операции.


## Описание


Метод openFromRubKey посылает
 запрос на открытие иерархии рабочей книги по ключу каталога показателей.
