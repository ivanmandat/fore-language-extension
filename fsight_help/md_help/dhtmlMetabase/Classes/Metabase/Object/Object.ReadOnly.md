# Object.ReadOnly


## Синтаксис


ReadOnly: Boolean;


## Описание


Свойство ReadOnly определяет
 возможность редактирования объекта репозитория.


## Комментарии


Допустимые значения:


	- True. Объект недоступен
	 для редактирования;


	- False. По умолчанию.
	 Объект доступен для редактирования.


Значение свойства задается в конструкторах или с помощью метода setReadOnly, а возвращается с помощью
 метода getReadOnly.
