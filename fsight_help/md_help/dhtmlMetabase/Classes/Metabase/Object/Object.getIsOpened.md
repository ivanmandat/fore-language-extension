# Object.getIsOpened


## Синтаксис


getIsOpened();


## Описание


Метод getIsOpened возвращает
 результат проверки открытия объекта.


## Комментарии


Метод возвращает одно из значений:


	- True. Объект открыт;


	- False. Объект закрыт.
