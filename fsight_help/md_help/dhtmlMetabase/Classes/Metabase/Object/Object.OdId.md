# Object.OdId


## Синтаксис


OdId: OdId;


## Параметры


OdId. Моникёр объекта.


## Описание


Свойство OdId определяет строку
 моникёра объекта репозитория.


## Комментарии


Значение свойства задается в конструкторах или с помощью метода setOdId, а возвращается с помощью
 метода getOdId.
