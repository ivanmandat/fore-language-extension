# Object.PPService


## Синтаксис


PPService: [PP.Mb.PPService](../PPService/PPService.htm);


## Описание


Свойство PPService определяет
 сервис для работы с объектом репозитория.


## Комментарии


Значение свойства задается в конструкторах или с помощью метода setPPService, а возвращается с помощью
 метода getPPService.
