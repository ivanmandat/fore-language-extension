# Object.Names


## Синтаксис


Names: OdName[];


## Параметры


OdName. Массив наименований
 объекта на разных языках.


## Описание


Свойство Names определяет коллекцию
 наименований объекта для разных языков.


## Комментарии


Значение свойства задается в конструкторах или с помощью метода setNames, а возвращается с помощью
 метода getNames.
