# Object.Params


## Синтаксис


Params: OdArg[];


## Параметры


OdArg. Параметры объекта.


## Описание


Свойство Params определяет коллекцию
 параметров объекта репозитория.


## Комментарии


Значение свойства задается в конструкторах или с помощью метода setParams, а возвращается с помощью
 метода getParams.
