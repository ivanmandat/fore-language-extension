# Object.ShareUrl


## Синтаксис


ShareUrl: Sring;


## Описание


Свойство ShareUrl определяет
 URL-адрес объекта репозитория.


## Комментарии


Значение свойства задается в конструкторах или с помощью метода setShareUrl, а возвращается с помощью
 метода getShareUrl.
