# MetabaseDialogBase.removeFilter


## Синтаксис


removeFilter(index: Number);


## Параметры


*index.* Позиция фильтра,
 который требуется удалить.


## Описание


Метод removeFilter** удаляет
 фильтр по его позиции из базового диалога с репозиторием.
