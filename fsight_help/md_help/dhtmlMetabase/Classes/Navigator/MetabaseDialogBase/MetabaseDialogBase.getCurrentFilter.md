# MetabaseDialogBase.getCurrentFilter


## Синтаксис


getCurrentFilter();


## Описание


Метод getCurrentFilter**
 возвращает текущий фильтр в базовом диалоге с репозиторием.


## Комментарии


Метод возвращает JSON-объект со следующими полями: Filters - массив
 идентификаторов фильтров, являющихся элементами перечисления PP.Mb.Ui.[MetabaseObjectClass](KeSom.chm::/Enums/MetabaseObjectClass.htm),
 ResourceKey - ключ ресурса для получения наименования фильтра.
