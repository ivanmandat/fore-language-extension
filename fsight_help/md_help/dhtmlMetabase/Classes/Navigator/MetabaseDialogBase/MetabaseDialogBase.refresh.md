# MetabaseDialogBase.refresh


## Синтаксис


refresh(rootKey: Number, withoutCache: Boolean);


## Параметры


*rootKey.*
 Ключ папки, содержимое которой нужно обновить. Если параметр не указан,
 то будет обновлена текущая папка;


*withoutCache.* Признак
 обновления содержимого папки без обновления кеш-данных. Если установлено
 значение false, то при обновлении
 компонента данные берутся из кеша. Если установлено значение true
 (по умолчанию), то кеш очищается и его содержимое обновляется вместе с
 компонентом для того, чтобы получить наиболее актуальные данные.


## Описание


Метод refresh** обновляет
 содержимое папки по его ключу в базовом диалоге с репозиторием.
