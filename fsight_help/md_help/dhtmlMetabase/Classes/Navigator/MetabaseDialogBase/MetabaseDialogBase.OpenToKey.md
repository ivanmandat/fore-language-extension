# MetabaseDialogBase.OpenToKey


## Синтаксис


OpenToKey: Number|Boolean


## Описание


Свойство OpenToKey** устанавливает
 ключ каталога, который нужно раскрыть в базовом диалоге с репозиторием.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setOpenToKey,
 а возвращается - с помощью метода getOpenToKey.


В случае, если ключ каталога, который нужно раскрыть в базовом диалоге
 с репозиторием, не определён, свойство по умолчанию содержит значение
 false.
