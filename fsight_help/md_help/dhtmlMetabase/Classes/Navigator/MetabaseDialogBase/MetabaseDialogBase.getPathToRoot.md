# MetabaseDialogBase.getPathToRoot


## Синтаксис


getPathToRoot();


## Описание


Метод getPathToRoot** возвращает
 путь от корня репозитория до текущей раскрытой папки в базовом диалоге
 с репозиторием.


## Комментарии


Метод возвращает массив объектов типа [PP.Mb.Object](../../Metabase/Object/Object.htm),
 входящих в путь к текущей раскрытой папке.


Текущая папка в диалоге устанавливается с помощью свойства [MetabaseDialogBase.OpenToKey](MetabaseDialogBase.OpenToKey.htm)
 и метода [MetabaseDialogBase.openFoldersTo](MetabaseDialogBase.openFoldersTo.htm).
