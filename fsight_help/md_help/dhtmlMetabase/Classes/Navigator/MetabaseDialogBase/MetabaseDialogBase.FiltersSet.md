# MetabaseDialogBase.FiltersSet


## Синтаксис


FiltersSet: Array


## Описание


Свойство FiltersSet** определяет
 множество фильтров для диалога.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setFiltersSet,
 а возвращается - с помощью метода getFiltersSet.


Свойство содержит массив JSON-объектов со следующими полями: Filters
 - массив идентификаторов фильтров, являющихся элементами перечисления
 PP.Mb.Ui.[MetabaseObjectClass](KeSom.chm::/Enums/MetabaseObjectClass.htm),
 ResourceKey - ключ ресурса для получения наименования фильтра.
