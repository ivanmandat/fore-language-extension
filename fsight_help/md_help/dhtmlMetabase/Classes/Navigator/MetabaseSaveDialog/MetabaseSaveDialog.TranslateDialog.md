# MetabaseSaveDialog.TranslateDialog


## Синтаксис


TranslateDialog: PP.Mb.Ui.TranslateDialog


## Описание


Свойство TranslateDialog**
 устанавливает диалог для перевода имени сохраняемого объекта на различные
 языки.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setTranslateDialog,
 а возвращается - с помощью метода getTranslateDialog.
