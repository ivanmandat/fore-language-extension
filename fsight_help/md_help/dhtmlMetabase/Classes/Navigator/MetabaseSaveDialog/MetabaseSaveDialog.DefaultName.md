# MetabaseSaveDialog.DefaultName


## Синтаксис


DefaultName: String


## Описание


Свойство DefaultName**
 устанавливает для сохраняемого в репозитории объекта имя по умолчанию.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDefaultName,
 а возвращается - с помощью метода getDefaultName.
