# MetabaseSaveDialog.Names


## Синтаксис


Names: Array


## Описание


Свойство Names** определяет
 имя сохраняемого в репозитории объекта в различных переводах.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setNames,
 а возвращается - с помощью метода getNames.


Свойство содержит массив JSON-объектов, у которого в качестве свойства
 указывается идентификатор языка, а в качестве значения - имя объекта,
 переведённое на данный язык.
