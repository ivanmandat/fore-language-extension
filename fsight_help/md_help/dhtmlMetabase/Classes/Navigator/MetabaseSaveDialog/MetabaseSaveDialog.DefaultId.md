# MetabaseSaveDialog.DefaultId


## Синтаксис


DefaultId: String


## Описание


Свойство DefaultId** устанавливает
 для сохраняемого в репозитории объекта идентификатор по умолчанию.


## Комментарии


Значение свойства устанавливается из JSON и с помощью метода setDefaultId,
 а возвращается - с помощью метода getDefaultId.
