# IRibbonPanels


Сборка: Forms;


## Описание


Интерфейс IRibbonPanels содержит
 свойства и методы для работы с коллекцией панелей, расположенных на вкладке
 ленты.


## Иерархия наследования


IRibbonPanels


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IRibbonPanels.Count.htm)
		 Свойство Count возвращает
		 количество панелей, расположенных на вкладке ленты.


		 ![](../../Property_Image.gif)
		 [Item](IRibbonPanels.Item.htm)
		 Свойство Item возвращает
		 параметры указанной панели.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IRibbonPanels.Add.htm)
		 Метод Add добавляет
		 указанную панель в коллекцию панелей вкладки.


		 ![](../../Sub_Image.gif)
		 [Clear](IRibbonPanels.Clear.htm)
		 Метод Clear удаляет
		 все панели из коллекции.


		 ![](../../Sub_Image.gif)
		 [IndexOf](IRibbonPanels.IndexOf.htm)
		 Метод IndexOf возвращает
		 для указанной панели индекс в коллекции.


		 ![](../../Sub_Image.gif)
		 [Insert](IRibbonPanels.Insert.htm)
		 Метод Insert вставляет
		 указанную панель в указанную позицию в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IRibbonPanels.Remove.htm)
		 Метод Remove удаляет
		 указанную панель из коллекции.


		 ![](../../Sub_Image.gif)
		 [RemoveAt](IRibbonPanels.RemoveAt.htm)
		 Метод RemoveAt удаляет
		 панель из коллекции по индексу.
