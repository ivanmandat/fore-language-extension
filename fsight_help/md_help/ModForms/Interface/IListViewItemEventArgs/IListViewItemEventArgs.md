# IListViewItemEventArgs


## Описание


Интерфейс IListViewItemEventArgs содержит свойства аргументов событий, происходящих при каких-либо действиях над элементом компонента [ListView](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ListView.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Item](IListViewItemEventArgs.Item.htm)


		 Свойство Item возвращает
		 элемент, над которым произведено действие, сгенерировавшее какое-либо
		 событие.
