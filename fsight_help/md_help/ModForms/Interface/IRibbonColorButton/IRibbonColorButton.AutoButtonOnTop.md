# IRibbonColorButton.AutoButtonOnTop


## Синтаксис


AutoButtonOnTop: Boolean;


## Описание


Свойство AutoButtonOnTop определяет место расположения кнопки «Авто».


## Комментарии


Если свойству установлено значение True, то кнопка «Авто» будет расположена вверху списка над категорией цветов «Общие».


Если свойству установлено значение False, то кнопка «Авто» будет расположена внизу списка над кнопкой для вызова диалога выбора другого цвета.
