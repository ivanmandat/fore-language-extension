# IRibbonColorButton.AutoButtonCaption


## Синтаксис


AutoButtonCaption: String;


## Описание


Свойство AutoButtonCaption определяет заголовок кнопки «Авто», предназначенной для выбора цвета по умолчанию.


## Комментарии


Если в качестве заголовка установлена пустая строка, то кнопка «Авто» не отображается.
