# IRibbonColorButton.ColorVisible


## Синтаксис


ColorVisible: Boolean;


## Описание


Свойство ColorVisible определяет
 признак отображения выбранного цвета под пиктограммой кнопки.


## Комментарии


Свойство актуально, если свойству [IRibbonPaletteButton.ButtonMode](../IRibbonPaletteButton/IRibbonPaletteButton.ButtonMode.htm) = True.


Допустимые значения:


	- True. Под пиктограммой
	 кнопки будет отображаться последний выбранный цвет: ![](IRibbonColorButton_ColorVisible_True.gif);


	- False. Под пиктограммой
	 кнопки не будет отображаться последний выбранный цвет: ![](IRibbonColorButton_ColorVisible_False.gif).


Пиктограмма кнопки отображается слева от заголовка, если определено
 свойство [IRibbonButton.ImageIndex](../IRibbonButton/IRibbonButton.ImageIndex.htm).
