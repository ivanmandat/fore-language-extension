# IRibbonColorButton.OtherButtonTooltip


## Синтаксис


OtherButtonTooltip: String;


## Описание


Свойство OtherButtonTooltip определяет всплывающую подсказку для кнопки, предназначенной для вызова диалога выбора дополнительного цвета.
