# IRibbonColorButton.OtherButtonCaption


## Синтаксис


OtherButtonCaption: String;


## Описание


Свойство OtherButtonCaption определяет заголовок кнопки, предназначенной для вызова диалога выбора дополнительного цвета.


## Комментарии


Если в качестве заголовка установлена пустая строка, то кнопка для выбора дополнительного цвета не отображается.
