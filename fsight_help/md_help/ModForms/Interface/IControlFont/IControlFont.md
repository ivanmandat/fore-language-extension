# IControlFont


## Описание


Интерфейс IControlFont содержит
 свойства для настройки шрифта компонента.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Bold](IControlFont.Bold.htm)
		 Свойство Bold определяет,
		 будет ли текст выделяться жирным шрифтом.


		 ![](../../Property_Image.gif)
		 [Color](IControlFont.Color.htm)
		 Свойство Color определяет
		 цвет шрифта компонента.


		 ![](../../Property_Image.gif)
		 [GxFont](IControlFont.GxFont.htm)
		 Свойство GxFont определяет
		 графическое представление шрифта.


		 ![](../../Property_Image.gif)
		 [Italic](IControlFont.Italic.htm)
		 Свойство Italic определяет,
		 будет ли текст выделяться курсивом.


		 ![](../../Property_Image.gif)
		 [Name](IControlFont.Name.htm)
		 Свойство Name определяет
		 наименование используемого шрифта.


		 ![](../../Property_Image.gif)
		 [Size](IControlFont.Size.htm)
		 Свойство Size определяет
		 размер шрифта.


		 ![](../../Property_Image.gif)
		 [Strikethrough](IControlFont.Strikethrough.htm)
		 Свойство Strikethrough
		 определяет, будет ли текст компонента зачеркнутым.


		 ![](../../Property_Image.gif)
		 [Underline](IControlFont.Underline.htm)
		 Свойство Underline
		 определяет, будет ли текст выделяться подчеркиванием.
