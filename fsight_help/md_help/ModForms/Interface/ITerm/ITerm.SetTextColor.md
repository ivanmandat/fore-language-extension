# ITerm.SetTextColor


## Синтаксис


SetTextColor(Value: [IGxColor](ModDrawing.chm::/Interface/IGxColor/IGxColor.htm);
 Start: Integer; Length: Integer);


## Параметры


Value.
 Цвет символа.


Start. Индекс первого символа,
 нумерация с нуля.


Length. Длина сегмента, которому
 будет задан цвет.


## Описание


Метод SetTextColor задает цвет
 символов терма.
