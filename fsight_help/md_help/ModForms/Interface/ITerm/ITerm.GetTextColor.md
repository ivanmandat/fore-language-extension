# ITerm.GetTextColor


## Синтаксис


GetTextColor(Position: integer): [IGxColor](ModDrawing.chm::/Interface/IGxColor/IGxColor.htm);


## Параметры


Position. Индекс символа для
 проверки цвета, нумерация с нуля.


## Описание


Метод GetTextColor возвращает
 цвет символа терма.


## Комментарии


Метод возвращает цвета, заданные свойством [ITerm.Color](ITerm.Color.htm)
 или методом [ITerm.SetTextColor](ITerm.SetTextColor.htm).
