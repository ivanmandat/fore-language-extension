# ITerm.Underlined


## Синтаксис


Underlined: Boolean;


## Описание


Свойство Underlined определяет
 подчеркивание терма.


## Комментарии


При значении True терм будет
 иметь подчеркивание.


По умолчанию содержит значение False.
