# ITerm.ResetTextColor


## Синтаксис


ResetTextColor();


## Описание


Метод ResetTextColor сбрасывает
 цвет символов терма.


## Комментарии


Метод устанавливает каждому символу базовый цвет, указанный в свойстве
 [ITerm.Color](ITerm.Color.htm).
