# ITerm.IsLiteral


## Синтаксис


IsLiteral: Boolean;


## Описание


Свойство IsLiteral определяет,
 является ли терм литеральным.


## Комментарии


Если для свойства установлено значение True,
 терм является литеральным. В том случае если установлено значение False, терм не является литеральным. Если
 терм литеральный, то текст внутри терма доступен для редактирования. Иначе,
 редактировать текст внутри терма нельзя.


По умолчанию установлено значение True.
