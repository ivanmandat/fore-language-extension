# INavigationBarPadChangingEventArgs.Allowed


## Синтаксис


Allowed: Boolean;


## Описание


Свойство Allowed определяет, будет ли произведена активация вкладки навигационной панели.


## Комментарии


Если свойству установить значение True, то вкладка будет активирована, иначе активация вкладки будет отменена.
