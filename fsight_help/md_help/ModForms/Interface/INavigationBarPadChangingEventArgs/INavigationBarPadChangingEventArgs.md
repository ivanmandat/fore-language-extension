## Описание


Интерфейс INavigationBarPadChangingEventArgs
 содержит свойства аргумента события наступающего перед активацией вкладки
 компонента [NavigationBar](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/NavigationBar.htm).
