# IKeyEventArgs


## Описание


Интерфейс IKeyEventArgs содержит
 свойства аргумента событий [OnKeyDown](../IControl/IControl.OnKeyDown.htm)
 и [OnKeyUp](../IControl/IControl.OnKeyUp.htm), происходящих,
 когда были нажаты или отпущены клавиши на активном компоненте.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Key](IKeyEventArgs.Key.htm)
		 Свойство Key определяет
		 код клавиши, которая была нажата или отпущена.


		 ![](../../Property_Image.gif)
		 [ShiftState](IKeyEventArgs.ShiftState.htm)
		 Свойство ShiftState
		 возвращает значение, соответствующее состоянию вспомогательных
		 клавиш (SHIFT, ALT и CTRL) во время выполнения действий с клавиатуры.
