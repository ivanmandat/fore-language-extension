# IDropForm.AnimationType


## Синтаксис


AnimationType: [DropAnimationType](../../Enums/DropAnimationType.htm);


## Описание


Свойство AnimationType определяет тип анимации, применяемый при разворачивании формы.


## Комментарии


При сворачивании формы будет применяться обратный тип анимации.
