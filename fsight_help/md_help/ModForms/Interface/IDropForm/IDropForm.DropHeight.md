# IDropForm.DropHeight


## Синтаксис


DropHeight: Integer;


## Описание


Свойство DropHeight определяет
 высоту области, в которой будет отображаться раскрывающаяся форма.


## Комментарии


При добавлении компонента DropForm
 на форму по умолчанию в данном свойстве будет установлено значение - 240
 пикселей.


Минимальное значение, которое можно установить в данном свойстве - 1.
 При этом фактическая высота области, в которой будет отображаться раскрывающаяся
 форма, будет рассчитана автоматически с учетом настроек операционной системы
 (размеров элементов окна, установленных в настройках экрана, используемой
 темы оформления).


Максимальное значение, которое можно установить в данном свойстве, ограничено
 максимальным значением, доступным для целого типа данных. При этом фактическая
 высота области, в которой будет отображаться раскрывающаяся форма, будет
 ограничена доступной видимой областью экрана.
