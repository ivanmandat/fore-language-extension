# IHotTrackStyles.UnderlineHot


## Синтаксис


UnderlineHot: Boolean;


## Описание


Свойство UnderlineHot определяет возможность выделения элемента, на котором останавливается курсор мыши.


## Комментарии


При установке данного свойства в True, элемент, на котором останавливается курсор мыши, выделяется синим цветом и подчеркиванием. При этом свойство [HandPoint](IHotTrackStyles.HandPoint.htm) должно иметь значение True.
