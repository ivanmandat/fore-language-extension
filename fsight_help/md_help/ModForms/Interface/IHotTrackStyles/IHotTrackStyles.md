# IHotTrackStyles


## Описание


Интерфейс IHotTrackStyles содержит
 свойства для определения стиля подсветки элементов списка [ListView](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ListView.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [HandPoint](IHotTrackStyles.HandPoint.htm)
		 Свойство HandPoint
		 определяет внешний вид курсора и необходимость подсветки элемента,
		 находящегося в фокусе.


		 ![](../../Property_Image.gif)
		 [UnderlineCold](IHotTrackStyles.UnderlineCold.htm)
		 Свойство UnderlineCold
		 определяет возможность выделения элемента, на котором останавливается
		 курсор мыши.


		 ![](../../Property_Image.gif)
		 [UnderlineHot](IHotTrackStyles.UnderlineHot.htm)
		 Свойство UnderlineHot
		 определяет необходимость выделения элементов подчеркиванием.
