# IHotTrackStyles.UnderlineCold


## Синтаксис


UnderlineCold: Boolean;


## Описание


Свойство UnderlineCold определяет, выделяются ли элементы компонента [ListView](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ListView.htm) подчеркиванием.


## Комментарии


Если для свойства установлено значение True, то все элементы компонента [ListView](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ListView.htm) выделяются подчеркиванием.


Если для свойства установлено значение False, то элементы не выделяются подчеркиванием.


Свойство UnderlineCold актуально только в том случае, если для свойства [IHotTrackStyles.HandPoint](IHotTrackStyles.HandPoint.htm) установлено значение True.
