# IHotTrackStyles.HandPoint


## Синтаксис


HandPoint: Boolean;


## Описание


Свойство HandPoint определяет внешний вид курсора над элементом, находящимся в фокусе. При установке данного свойства в True, курсор мыши приобретает форму руки, когда он останавливается над элементом.


## Комментарии


Свойства [UnderlineCold](IHotTrackStyles.UnderlineCold.htm) и [UnderlineHot](IHotTrackStyles.UnderlineHot.htm) актуальны только в том случае, если для свойства HandPoint установлено значение True.
