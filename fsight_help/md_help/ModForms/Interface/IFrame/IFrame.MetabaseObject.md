# IFrame.MetabaseObject


## Синтаксис


MetabaseObject: String;


## Описание


Свойство MetabaseObject определяет
 идентификатор объекта репозитория, который будет подгружен на место, обозначенное
 фреймом, во время выполнения данной формы. В список доступных объектов
 входят все формы текущего репозитория.
