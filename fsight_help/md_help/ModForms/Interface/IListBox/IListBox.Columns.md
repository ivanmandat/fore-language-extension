# IListBox.Columns


## Синтаксис


Columns: Integer;


## Описание


Свойство Columns определяет
 количество столбцов, в которых будут отображаться элементы (строки) списка.


## Комментарии


По умолчанию свойству Columns
 установлено значение 0. При этом элементы списка будут отображаться
 в один столбец в вертикальном направлении, при этом появится вертикальная
 полоса прокрутки. При установке свойству Columns
 значения от 1 до n, элементы списка отображаются в горизонтальном направлении.
