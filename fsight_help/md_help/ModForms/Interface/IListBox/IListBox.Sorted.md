# IListBox.Sorted


## Синтаксис


Sorted: Boolean;


## Описание


Свойство Sorted определяет,
 будут ли элементы (строки) в списке автоматически сортироваться в алфавитном
 порядке.


## Комментарии


Если Sorted = False,
 то элементы (строки) отображаются в том порядке, в котором они были добавлены
 в список. По умолчанию Sorted
 = False.
