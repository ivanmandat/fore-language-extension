# IListBox.MultiSelect


## Синтаксис


MultiSelect: Boolean;


## Описание


Свойство MultiSelect определяет
 возможность множественного выбора элементов (строк) из списка.


## Комментарии


Если свойству установлено значение True,
 то при использовании клавиш SHIFT или CTRL можно будет отметить несколько
 элементов из списка. При значении False
 будет возможность отметить только один элемент (строку) списка.
