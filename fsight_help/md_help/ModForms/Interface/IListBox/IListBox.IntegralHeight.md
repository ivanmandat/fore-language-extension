# IListBox.IntegralHeight


## Синтаксис


IntegralHeight: Boolean;


## Описание


Свойство IntegralHeight определяет,
 будет ли отображаться элемент списка, если он полностью не помещается
 в окно компонента, путем подгонки высоты компонента.


## Комментарии


При установке свойству IntegralHeight
 значения True, компонент изменит
 свою высоту таким образом, чтобы элементы списка были видны полностью.
