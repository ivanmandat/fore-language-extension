# IListBox.Selected


## Синтаксис


Selected(Index: Integer): Boolean;


## Параметры


Index. Параметр, определяющий
 индекс элемента, наличие отметки у которого необходимо проверить.


## Описание


Свойство Selected возвращает
 признак наличия отметки у указанного элемента.


## Комментарии


Свойство возвращает следующие значения:


	- True.
	 Элемент отмечен;


	- False. Элемент не отмечен.
