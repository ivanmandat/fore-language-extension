# IConditionElement.Row


## Синтаксис


Row: Integer;


## Описание


Свойство Row возвращает номер
 строки элемента в компоненте.


## Комментарии


Для получения элемента из группы по индексу используйте свойство [IConditionElements.Item](../IConditionElements/IConditionElements.Item.htm).
