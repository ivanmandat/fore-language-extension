# IListViewColumnEventArgs


## Описание


Интерфейс IListViewColumnEventArgs
 содержит свойства аргумента события, возникающего при нажатии на заголовок колонки компонента [ListView](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ListView.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Column](IListViewColumnEventArgs.Column.htm)


		 Свойство Column возвращает
		 параметры колонки, по заголовку которой произведен щелчок мыши.
