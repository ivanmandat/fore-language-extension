# IMonthCalendar.MaxSelectRange


## Синтаксис


MaxSelectRange: Integer;


## Описание


Свойство MaxSelectRange определяет максимальное количество дней, которые могут быть выделены.


## Комментарии


Свойство актуально, если в компоненте включена возможность выделения диапазона дат (свойству [MultiSelect](IMonthCalendar.MultiSelect.htm) установлено значение True). По умолчанию в диапазоне можно выделить 31 день.
