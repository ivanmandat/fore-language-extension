# IMonthCalendar.ShowToday


## Синтаксис


ShowToday: Boolean;


## Описание


Свойство ShowToday определяет,
 будет ли отображаться текущая дата в нижней части календаря.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 Отображать текущую дату;


	- False. Не отображать
	 текущую дату.
