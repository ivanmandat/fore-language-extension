# IMonthCalendar.EndDate


## Синтаксис


EndDate: DateTime;


## Описание


Свойство EndDate определяет дату, на которой будет заканчиваться выделение.


## Комментарии


При этом учитывается состояние свойства [MultiSelect](IMonthCalendar.MultiSelect.htm), если ему установлено значение False, изменение значения свойства EndDate приведет к изменению значений свойств [BeginDate](IMonthCalendar.BeginDate.htm) и [CurrentDate](../ICommonCalendar/ICommonCalendar.CurrentDate.htm).
