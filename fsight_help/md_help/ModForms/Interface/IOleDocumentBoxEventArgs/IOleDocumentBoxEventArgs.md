# IOleDocumentBoxEventArgs


## Описание


Интерфейс IOleDocumentBoxEventArgs
 содержит свойства аргумента события, происходящего при обработке событий,
 загруженного в [OleDocumentBox](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/OleDocumentBox.htm), объекта.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DispId](IOleDocumentBoxEventArgs.DispId.htm)
		 Свойство DispId возвращает
		 диспетчерский идентификатор события, произошедшего для объекта,
		 загруженного в компонент.


		 ![](../../Property_Image.gif)
		 [Params](IOleDocumentBoxEventArgs.Params.htm)
		 Свойство Params возвращает
		 коллекцию параметров события, произошедшего для объекта.
