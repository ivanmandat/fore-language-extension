# IShortCutCollection


## Описание


Интерфейс IShortCutCollection
 содержит свойства и методы для работы с коллекцией горячих клавиш.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IShortCutCollection.Count.htm)
		 Свойство Count возвращает
		 количество сочетаний горячих клавиш, содержащихся в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IShortCutCollection.Item.htm)
		 Свойство Item определяет
		 значение, соответствующее сочетанию горячих клавиш.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IShortCutCollection.Add.htm)
		 Метод Add осуществляет
		 добавление сочетания горячих клавиш в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](IShortCutCollection.Clear.htm)
		 Метод Clear осуществляет
		 очистку коллекции сочетаний горячих клавиш.


		 ![](../../Sub_Image.gif)
		 [Remove](IShortCutCollection.Remove.htm)
		 Метод Remove осуществляет
		 удаление сочетания горячих клавиш из коллекции.
