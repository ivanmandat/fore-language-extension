# IShortCutCollection.Item


## Синтаксис


Item(Index: Integer): Integer;


## Параметры


Index - индекс сочетания горячих клавиш.


## Описание


Свойство Item определяет значение, соответствующее сочетанию горячих клавиш.


## Комментарии


Для создания сочетания горячих клавиш используйте свойство [MakeShortcut](../IFormClass/IFormClass.MakeShortcut.htm).
