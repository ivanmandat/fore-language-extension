# IRibbonEdit.SearchMode


## Синтаксис


SearchMode: Boolean;


## Описание


Свойство SearchMode определяет
 признак работы элемента в режиме с отображением вспомогательного текста.


## Комментарии


Если свойству установлено значение True,
 то при пустом значении свойства [TextValue](IRibbonEdit.TextValue.htm)
 в элементе управления будет отображен вспомогательный текст [SearchPrompt](IRibbonEdit.SearchPrompt.htm).
 Данный текст будет подсвечен серым цветом и может быть использован для
 информирования пользователя о необходимости ввода какого-либо значения:
