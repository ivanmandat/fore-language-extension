# IRibbonEdit.SearchPrompt


## Синтаксис


SearchPrompt: String;


## Описание


Свойство SearchPrompt определяет
 вспомогательный текст для элемента управления.


## Комментарии


Данное свойство актуально, если свойству [SearchMode](IRibbonEdit.SearchMode.htm)
 установлено значение True.


При пустом значение свойства [TextValue](IRibbonEdit.TextValue.htm)
 значение данного свойства будет отображено в редакторе элемента управления.
 Значение будет подсвечено серым цветом и может использоваться для информирования
 пользователя о необходимости ввода какого-либо значения:


![](IRibbonEdit_SearchMode.gif)
