# IRibbonEdit.ImageIndex


## Синтаксис


ImageIndex: Integer;


## Описание


Свойство ImageIndex определяет
 индекс изображения, которое будет отображаться рядом с редактором текста.


## Комментарии


Источник изображений устанавливается в свойстве [SmallImages](../IRibbonCategory/IRibbonCategory.SmallImages.htm)
 для вкладки, в области которой расположен комбинированный список. Выбранное
 изображение отображается с левой стороны редактора текста.


Если комбинированный список расположен в области [общих
 элементов управления](../IRibbon/IRibbon.TabElements.htm), то источник изображений устанавливается в свойстве
 [IRibbon.Images](../IRibbon/IRibbon.Images.htm).
