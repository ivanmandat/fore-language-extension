# IComboBox.DropDownCount


## Синтаксис


DropDownCount: Integer;


## Описание


Свойство DropDownCount определяет максимальное число элементов, одновременно отображаемых в раскрывающемся списке.


## Комментарии


Свойство не учитывается, если свойству [Style](IComboBox.Style.htm) установлено значение ComboboxStyle.Simple.
