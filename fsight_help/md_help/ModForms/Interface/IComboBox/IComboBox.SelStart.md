# IComboBox.SelStart


## Синтаксис


SelStart: Integer;


## Описание


Свойство SelStart определяет
 позицию, начиная с которой осуществляется выделение текста в компоненте.


## Комментарии


Данное свойство возвращает корректные значения, если компонент имеет
 фокус. Во всех остальных случаях свойство возвращает значение 0.
