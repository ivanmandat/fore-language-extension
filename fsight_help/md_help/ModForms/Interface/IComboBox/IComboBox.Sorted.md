# IComboBox.Sorted


## Синтаксис


Sorted: Boolean;


## Описание


Свойство Sorted определяет признак
 автоматической сортировки строк раскрывающегося списка в алфавитном порядке.


## Комментарии


Если свойству установлено значение True,
 то строки будут автоматически сортироваться в алфавитном порядке. По умолчанию
 свойству установлено значение False.
