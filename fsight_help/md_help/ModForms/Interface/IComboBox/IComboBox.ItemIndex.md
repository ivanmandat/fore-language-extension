# IComboBox.ItemIndex


## Синтаксис


ItemIndex: Integer;


## Описание


Свойство ItemIndex определяет
 индекс выбранного элемента списка.


## Комментарии


По умолчанию свойству установлено значение "-1", при этом
 ни один элемент списка не выбран.
