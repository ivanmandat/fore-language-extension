# IComboBox.MaxLength


## Синтаксис


MaxLength: Integer;


## Описание


Свойство MaxLength определяет
 максимальную длину вводимого текста.


## Комментарии


Актуально в случае, если свойство [Style](IComboBox.Style.htm)
 установлено в значение Simple или DropDown. Если MaxLength
 = 0, то длина текста неограничена.
