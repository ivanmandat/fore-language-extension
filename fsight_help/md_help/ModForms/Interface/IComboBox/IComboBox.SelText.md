# IComboBox.SelText


## Синтаксис


SelText: String;


## Описание


Свойство SelText определяет
 текст, выделенный в компоненте.


## Комментарии


Данное свойство возвращает выделенный текст, расположенный начиная с
 позиции [SelStart](IComboBox.SelStart.htm) и длинной [SelLength](IComboBox.SelLength.htm)
 символов.


При установке данного свойства, если в компоненте имеется выделенный
 текст, то данный текст будет заменен. Если выделение отсутствует, то устанавливаемое
 значение будет добавлено в позицию, в которой расположен курсор.
