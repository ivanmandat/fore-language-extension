# INavigationBarHeader


Сборка: Forms;


## Описание


Интерфейс INavigationBarHeader
 содержит свойства заголовка вкладки, который отображается в навигационной
 панели.


## Иерархия наследования


INavigationBarHeader


## Комментарии


Параметры заголовка могут быть получены в свойстве [INavigationBarPad.Header](../INavigationBarPad/INavigationBarPad.Header.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Content](INavigationBarHeader.Content.htm)


		 Свойство Content определяет
		 тип элемента управления, который будет присутствовать в заголовке.


		 ![](../../Property_Image.gif)
		 [CustomContent](INavigationBarHeader.CustomContent.htm)


		 Свойство CustomContent
		 определяет элемент управления, отображаемый в области заголовка.


		 ![](../../Property_Image.gif)
		 [Title](INavigationBarHeader.Title.htm)


		 Свойство Title определяет
		 текст, отображаемый в заголовке вкладки.


		 ![](../../Property_Image.gif)
		 [TitleSuffix](INavigationBarHeader.TitleSuffix.htm)


		 Свойство TitleSuffix
		 определяет суффикс к тексту заголовка вкладки.


		 ![](../../Property_Image.gif)
		 [Value](INavigationBarHeader.Value.htm)


		 Свойство Value определяет
		 состояние флажка, отображаемого в заголовке.
