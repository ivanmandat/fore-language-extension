# INavigationBarHeader.Value


## Синтаксис


Value: Boolean;


## Описание


Свойство Value определяет состояние
 флажка, отображаемого в заголовке.


## Комментарии


Данное свойство актуально, если свойству [Content](INavigationBarHeader.Content.htm)
 установлено значение [NavigationBarHeaderContent.CheckBox](../../Enums/NavigationBarHeaderContent.htm).
