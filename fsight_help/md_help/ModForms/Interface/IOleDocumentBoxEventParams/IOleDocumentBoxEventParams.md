# IOleDocumentBoxEventParams


## Описание


Интерфейс IOleDocumentBoxEventParams
 содержит свойства коллекции параметров события объекта, загруженного в
 [OleDocumentBox](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/OleDocumentBox.htm).
