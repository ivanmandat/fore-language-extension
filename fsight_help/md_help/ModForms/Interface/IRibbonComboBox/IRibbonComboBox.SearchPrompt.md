# IRibbonComboBox.SearchPrompt


## Синтаксис


SearchPrompt: String;


## Описание


Свойство SearchPrompt определяет
 вспомогательный текст для элемента управления.


## Комментарии


Свойство доступно для использования, если для компонента установлен
 стиль [DropDown](../../Enums/ComboboxStyle.htm) и свойству
 [SearchMode](IRibbonComboBox.SearchMode.htm) установлено значение
 True.


Если в редакторе элемента отсутствует какой-либо текст, то там будет
 отображено значение данного свойства. Значение будет подсвечено серым
 цветом и может использоваться для информирования пользователя о необходимости
 выбора какого-либо элемента, либо вводе какого-либо значения:


![](IRibbonComboBox_SearchMode.gif)
