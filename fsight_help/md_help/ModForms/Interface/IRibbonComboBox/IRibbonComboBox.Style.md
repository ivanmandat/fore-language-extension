# IRibbonComboBox.Style


## Синтаксис


Style: [ComboboxStyle](../../Enums/ComboboxStyle.htm);


## Описание


Свойство Style определяет стиль
 работы компонента.


## Комментарии


При установке стиля [ComboboxStyle.Simple](../../Enums/ComboboxStyle.htm)
 компонент будет работать в режиме [ComboboxStyle.DropDown](../../Enums/ComboboxStyle.htm).
