# IRibbonComboBox.ResizeList


## Синтаксис


ResizeList: Boolean;


## Описание


Свойство ResizeList определяет, будет ли в компоненте возможность изменить размер раскрывающегося списка элементов.


## Комментарии


Если свойству установлено значение True, то после раскрытия списка элементов будет возможность изменить его высоту. Если установлено значение False, то высота раскрывающегося списка элементов фиксирована. При большом количестве элементов будет присутствовать полоса прокрутки.


По умолчанию свойству установлено значение True.
