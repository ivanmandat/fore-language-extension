# IRibbonComboBox.Dropped


## Синтаксис


Dropped: Boolean;


## Описание


Свойство Dropped возвращает признак отображения раскрывающегося списка элемента управления.


## Комментарии


Свойство возвращает значение True, если список элементов раскрыт.
