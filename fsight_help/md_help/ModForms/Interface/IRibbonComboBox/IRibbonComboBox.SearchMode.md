# IRibbonComboBox.SearchMode


## Синтаксис


SearchMode: Boolean;


## Описание


Свойство SearchMode определяет
 признак работы элемента в режиме с отображением вспомогательного текста.


## Комментарии


Свойство доступно для использования, если для компонента установлен
 стиль [DropDown](../../Enums/ComboboxStyle.htm).


Если свойству установлено значение True,
 то пока в редакторе компонента отсутствует какой-либо текст, будет отображен
 вспомогательный текст [SearchPrompt](IRibbonComboBox.SearchPrompt.htm).
 Данный текст будет подсвечен серым цветом и может быть использован для
 информирования пользователя о необходимости выбора какого-либо элемента,
 либо вводе какого-либо значения:
