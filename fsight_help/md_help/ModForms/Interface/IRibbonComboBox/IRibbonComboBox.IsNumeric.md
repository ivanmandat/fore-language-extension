# IRibbonComboBox.IsNumeric


## Синтаксис


IsNumeric: Boolean;


## Описание


Свойство IsNumeric определяет
 признак возможности ввода только числовых значений.


## Комментарии


Допустимые значения:


	- True. Доступен ввод
	 только числовых значений;


	- False. Значение по умолчанию.
	 Доступен ввод любых значений (числа, строки, символы).


Наличие возможности ввода значений в элемент управления зависит от выбранного
 [стиля](IRibbonComboBox.Style.htm) компонента.
