# IRibbonComboBox.DropDownCount


## Синтаксис


DropDownCount: Integer;


## Описание


Свойство DropDownCount определяет
 максимальное количество одновременно отображаемых элементов в открытом
 списке компонента.


## Комментарии


По умолчанию в открытом списке отображается 8 элементов. Если общее
 количество элементов больше чем количество отображаемых элементов, то
 в компоненте добавляется полоса прокрутки.
