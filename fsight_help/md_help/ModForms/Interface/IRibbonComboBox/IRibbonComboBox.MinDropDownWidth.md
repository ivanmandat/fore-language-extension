# IRibbonComboBox.MinDropDownWidth


## Синтаксис


MinDropDownWidth: Integer;


## Описание


Свойство MinDropDownWidth определяет
 минимальную ширину раскрывающегося списка элементов.


## Комментарии


Значение ширины указывается в пикселях. По умолчанию свойству установлено
 значение -1, при этом минимальная ширина вычисляется автоматически в зависимости
 от размера текста элементов и ширины самого компонента.
