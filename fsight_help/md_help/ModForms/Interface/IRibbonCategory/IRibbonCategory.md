# IRibbonCategory


Сборка: Forms;


## Описание


Интерфейс IRibbonCategory содержит
 свойства вкладки на [ленте
 инструментов](GetStarted.chm::/Interface/Interface_Description.htm#customize_ribbon).


## Иерархия наследования


           [IComponent](../IComponent/IComponent.htm)


           IRibbonCategory


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Keys](IRibbonCategory.Keys.htm)
		 Свойство Keys определяет
		 горячую клавишу для быстрого доступа к вкладке на ленте инструментов.


		 ![](../../Property_Image.gif)
		 [LargeImages](IRibbonCategory.LargeImages.htm)
		 Свойство LargeImages
		 определяет компонент, содержащий большие изображения, которые
		 будут использоваться элементами управления данной вкладки на ленте
		 инструментов.


		 ![](../../Property_Image.gif)
		 [Panels](IRibbonCategory.Panels.htm)
		 Свойство Panels возвращает
		 коллекцию панелей, расположенных на текущей вкладке на ленте инструментов.


		 ![](../../Property_Image.gif)
		 [SmallImages](IRibbonCategory.SmallImages.htm)
		 Свойство SmallImages
		 определяет компонент, содержащий маленькие изображения, которые
		 будут использоваться элементами управления данной вкладки на ленте
		 инструментов.


		 ![](../../Property_Image.gif)
		 [Text](IRibbonCategory.Text.htm)
		 Свойство Text определяет
		 заголовок вкладки на ленте инструментов.


		 ![](../../Property_Image.gif)
		 [Visible](IRibbonCategory.Visible.htm)
		 Свойство Visible определяет
		 признак отображения вкладки на ленте инструментов.


## Свойства, унаследованные от [IComponent](../IComponent/IComponent.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ComponentCount](../IComponent/IComponent.ComponentCount.htm)
		 Свойство ComponentCount
		 возвращает количество дочерних компонентов, расположенных на родительском
		 компоненте.


		 ![](../../Property_Image.gif)
		 [Components](../IComponent/IComponent.Components.htm)
		 Свойство Components
		 возвращает дочерний компонент, индекс которого передается посредством
		 входного параметра.


		 ![](../../Property_Image.gif)
		 [Data](../IComponent/IComponent.Data.htm)
		 Свойство Data предназначено
		 для хранения любых пользовательских данных.


		 ![](../../Property_Image.gif)
		 [Name](../IComponent/IComponent.Name.htm)
		 Свойство Name определяет
		 наименование компонента.


		 ![](../../Property_Image.gif)
		 [Tag](../IComponent/IComponent.Tag.htm)
		 Свойство Tag не используется
		 компилятором. Пользователь может изменить значение данного свойства
		 и использовать его по своему усмотрению.
