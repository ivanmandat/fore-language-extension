# ITreeNodeEventArgs


## Описание


Интерфейс ITreeNodeEventArgs
 содержит свойства аргумента событий, происходящих для какой-либо вершины
 дерева компонентов [TreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeCombo.htm)
 и [TreeList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeList.htm).
