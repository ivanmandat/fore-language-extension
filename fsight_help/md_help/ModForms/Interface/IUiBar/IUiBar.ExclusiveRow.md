# IUiBar.ExclusiveRow


## Синтаксис


ExclusiveRow: Boolean;


## Описание


Свойство ExclusiveRow определяет
 возможность расположения в одной строке нескольких компонентов.


## Комментарии


Допустимые значения:


	- True. В одной строке
	 может располагаться только один компонент;


	- False. В одной строке
	 могут располагаться несколько компонентов.
