# IUiBar.Floating


## Синтаксис


Floating: Boolean;


## Описание


Свойство Floating возвращает
 состояние компонента относительно окна формы.


## Комментарии


Свойство возвращает следующие значения:


	- True. Компонент расположен
	 в отдельном окне;


	- False. Компонент расположен
	 в области формы.
