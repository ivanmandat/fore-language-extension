# IUiBar.AllowUndock


## Синтаксис


AllowUndock: Boolean;


## Описание


Свойство AllowUndock определяет
 признак возможности перемещения компонента во время выполнения формы.


## Комментарии


Допустимые значения:


	- True. Компонент может
	 быть взят с формы и перемещен в виде отдельного окна;


	- False. Компонент не
	 может быть перемещен, зафиксирован в области формы.
