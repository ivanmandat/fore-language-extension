# IUiBar.Dock


## Синтаксис


Dock(Edge: [BarDockingEdge](../../Enums/BarDockingEdge.htm));


## Параметры


Edge.
 Область формы, к которой будет привязан компонент.


## Описание


Метод Dock осуществляет привязку
 компонента к форме.


## Комментарии


После выполнения привязки компонента к форме свойству [Floating](IUiBar.Floating.htm)
 будет установлено значение False.
