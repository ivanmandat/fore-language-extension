# IUiBar.Float


## Синтаксис


Float;


## Описание


Метод Float осуществляет расположение
 компонента в отдельном окне.


## Комментарии


После расположения компонента в отдельном окне форме свойству [Floating](IUiBar.Floating.htm)
 будет установлено значение True.
