# IFloatEdit.CorrectOnExit


## Синтаксис


CorrectOnExit: Boolean;


## Описание


Свойство CorrectOnExit определяет, будет ли производиться автоматическая коррекция значения при потери компонентом фокуса.


## Комментарии


Если свойству установлено значение True, то после ввода некорректного значения и при потере компонентом фокуса, значение будет скорректировано. Корректировка производится путем установки в качестве значения одного из значений [MinValue](../IIntegerEdit/IIntegerEdit.MinValue.htm)/[MaxValue](../IIntegerEdit/IIntegerEdit.MaxValue.htm), в зависимости от того, какая из границ допустимых значений была нарушена.


По умолчанию свойству установлено значение False, при этом если введено некорректное значение, то после потери компонентом фокуса значение сброшено не будет. Но оно будет подсвечено красным цветом, что символизирует о выходе за пределы допустимых значений.
