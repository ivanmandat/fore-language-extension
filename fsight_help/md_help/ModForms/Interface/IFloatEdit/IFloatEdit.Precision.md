# IFloatEdit.Precision


## Синтаксис


Precision: Integer;


## Описание


Свойство Precision определяет
 количество разрядов, отображаемых после запятой.


## Комментарии


По умолчанию в компоненте установлена точность 2 знака после запятой.
 Максимальная точность - 12 знаков после запятой.
