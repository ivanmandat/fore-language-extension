# IFloatEdit.TrailingZeros


## Синтаксис


TrailingZeros: Boolean;


## Описание


Свойство TrailingZeros определяет
 признак отображения нулей после запятой, при [Precision](IFloatEdit.Precision.htm)
 больше нуля.


## Комментарии


Если TrailingZeros = False
 и значение компонента целое, то нули после запятой не отображаются.
