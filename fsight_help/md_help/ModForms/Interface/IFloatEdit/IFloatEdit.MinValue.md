# IFloatEdit.MinValue


## Синтаксис


MinValue: Double;


## Описание


Свойство MinValue определяет
 минимальное значение числа, которое можно установить в компоненте.


## Комментарии


При соблюдении следующего условия в компоненте можно указывать любые
 значения: MinValue = [MaxValue](IFloatEdit.MaxValue.htm)
 = 0.
