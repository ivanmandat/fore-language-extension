# IFloatEdit.ThousandSeparator


## Синтаксис


ThousandSeparator: Boolean;


## Описание


Свойство ThousandSeparator определяет,
 будет ли отображаться значение в компоненте с использованием разделителя
 групп разрядов.


## Комментарии


По умолчанию свойству установлено значение False,
 при этом значения отображаются без использования разделителя разрядов.
 При установке значения True для
 отображения значения будет использоваться разделитель групп разрядов,
 заданный в настройках региональных параметров операционной системы.
