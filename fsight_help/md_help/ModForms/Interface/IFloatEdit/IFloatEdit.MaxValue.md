# IFloatEdit.MaxValue


## Синтаксис


MaxValue: Double;


## Описание


Свойство MaxValue определяет максимальное значение числа, которое можно установить в компоненте..


## Комментарии


При соблюдении следующего условия в компоненте можно указывать любые значения: [MinValue](IFloatEdit.MinValue.htm) = MaxValue = 0.
