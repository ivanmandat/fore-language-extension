# IFloatEdit.AllowEmpty


## Синтаксис


AllowEmpty: Boolean;


## Описание


Свойство AllowEmpty определяет возможность наличия пустого значения в компоненте.


## Комментарии


При установке свойства в True пустое значение не подсвечивается красным фоном, при этом свойство [Value](IFloatEdit.Value.htm) возвращает 0, а свойство [Text](../IControl/IControl.Text.htm) - пустую строку.
