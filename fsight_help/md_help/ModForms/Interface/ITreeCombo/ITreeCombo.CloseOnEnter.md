# ITreeCombo.CloseOnEnter


## Синтаксис


CloseOnEnter: Boolean;


## Описание


Свойство CloseOnEnter определяет, будет ли производиться закрытие списка и сохранение отметки при нажатии клавиши ENTER.


## Комментарии


При установке свойству значения True производится закрытие списка с сохранением отметки.
