# ITreeCombo.AnimationType


## Синтаксис


AnimationType: [DropAnimationType](../../Enums/DropAnimationType.htm);


## Описание


Свойство AnimationType определяет тип анимации, применяемый при разворачивании списка компонента.


## Комментарии


При сворачивании списка будет применяться обратный тип анимации.
