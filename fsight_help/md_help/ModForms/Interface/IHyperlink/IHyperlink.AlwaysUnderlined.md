# IHyperlink.AlwaysUnderlined


## Синтаксис


AlwaysUnderlined: Boolean;


## Описание


Свойство AlwaysUnderlined определяет признак наличия у гиперссылки подчеркивающей линии. Если False, то линия будет отображаться только при наведении курсора.
