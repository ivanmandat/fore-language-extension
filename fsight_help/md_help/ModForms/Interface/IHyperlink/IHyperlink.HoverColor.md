# IHyperlink.HoverColor


## Синтаксис


HoverColor: [IGxColor](ModDrawing.chm::/Interface/IGxColor/IGxColor.htm);


## Описание


Свойство HoverColor определяет цвет, которым будет подсвечиваться гиперссылка при наведении на нее курсора мыши. По умолчанию устанавливается красный цвет.
