# IListView.ReadOnly


## Синтаксис


ReadOnly: Boolean;


## Описание


Свойство ReadOnly определяет
 возможность внесения изменений в текст элементов списка.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Редактирование текста элементов в списке запрещено;


Примечание.
 Запрет на редактирование текста элементов не распространяется на редактирование
 других элементов интерфейса.


	- False. Редактирование
	 текста элементов в списке разрешено.


## Пример


Для выполнения примера предполагается наличие на форме компонента Button с наименованием «Button1»
 и компонента ListView с наименованием
 «ListView1».


Добавьте ссылку на системную сборку Forms.


	Sub Button1OnClick(Sender: Object; Args: IMouseEventArgs);

	Begin

	    ListView1.ReadOnly := False;

	End Sub Button1OnClick;


В результате выполнения примера будет разрешено редактирование текста
 элементов.
