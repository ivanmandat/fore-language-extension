# IListView.WrapText


## Синтаксис


WrapText: Boolean;


## Описание


Свойство WrapText определяет, будет ли возможность переноса текста элемента на несколько строк, если он выходит за границы пиктограммы.


## Комментарии


Если свойству WrapText установлено значение True, то текст элемента будет переноситься.
