# IListView.HeaderMenu


## Синтаксис


HeaderMenu: [IPopupMenu](../IPopupMenu/IPopupMenu.htm);


## Описание


Свойство HeaderMenu определяет контекстное меню, вызываемое по щелчку дополнительной кнопки мыши на заголовке колонки компонента.


## Комментарии


Данное свойство используется если свойству [Style](IListView.Style.htm) установлено значение Report.
