# IListView.Arrangement


## Синтаксис


Arrangement: [ListViewIconArrangement](../../Enums/ListViewIconArrangement.htm);


## Описание


Свойство Arrangement определяет способ выравнивания элементов в компоненте. Свойство актуально, если свойству [Style](IListView.Style.htm) установлено значение Icon, либо SmallIcon.
