# IListView.LargeImages


## Синтаксис


LargeImages: [IImageList](../IImageList/IImageList.htm);


## Описание


Свойство LargeImages определяет источник крупных пиктограмм.


## Комментарии


В список доступных значений свойства входят компоненты [ImageList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ImageList.htm) и [GlobalImageList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/GlobalImageList.htm). Крупные изображения для элементов отображаются, если для компонента установлен [стиль](IListView.Style.htm) - [Icon](../../Enums/ListViewStyle.htm). Индексы изображения для элементов задаются в свойстве [IListViewItem.ImageIndex](../IListViewItem/IListViewItem.ImageIndex.htm).
