# IListView.SortType


## Синтаксис


SortType: [ControlSortType](../../Enums/ControlSortType.htm);


## Описание


Свойство SortType определяет тип сортировки для компонента.


## Комментарии


Свойство актуально если [Style](IListView.Style.htm) = Report. Для пользовательской сортировки используется событие [OnCompareItems](../../Class/ListView/ListView.OnCompareItems.htm).
