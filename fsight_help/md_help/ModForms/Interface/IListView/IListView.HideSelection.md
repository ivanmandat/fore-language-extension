# IListView.HideSelection


## Синтаксис


HideSelection: Boolean;


## Описание


Свойство HideSelection определяет признак отображения отметки в компоненте при передаче фокуса другому компоненту.


## Комментарии


Если свойству установлено значение False, то отметка остается видимой. Выделенные элементы подсвечиваются серым цветом. Если свойству установить значение True, то отметка отображаться не будет. По умолчанию свойству установлено значение False.
