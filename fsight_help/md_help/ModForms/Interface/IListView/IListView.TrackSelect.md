# IListView.TrackSelect


## Синтаксис


TrackSelect: Boolean;


## Описание


Свойство TrackSelect определяет, будет ли отмечаться элемент компонента [ListView](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ListView.htm) при удержании над ним курсора мыши.


## Комментарии


Если свойству установлено значение True, то будет производиться выделение элемента, иначе отметки производиться не будет.
