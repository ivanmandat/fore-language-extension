# IListView.ShowColumnHeaders


## Синтаксис


ShowColumnHeaders: Boolean;


## Описание


Свойство ShowColumnHeaders определяет,
 будут ли отображаться заголовки столбцов.


## Комментарии


Если свойству ShowColumnHeaders
 установлено значение True, то
 заголовки столбцов будут отображаться, иначе, столбцы не будут иметь заголовков.


Примечание.
 Свойство ShowColumnHeaders доступно
 только в том случае, если для свойства [ListView.Style](IListView.Style.htm)
 установлено значение Report.
