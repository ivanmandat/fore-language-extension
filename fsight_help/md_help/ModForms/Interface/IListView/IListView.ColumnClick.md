# IListView.ColumnClick


## Синтаксис


ColumnClick: Boolean;


## Описание


Свойство ColumnClick определяет,
 будет ли обрабатываться щелчок по заголовкам столбцов.


## Комментарии


По умолчанию свойству установлено значение True,
 при этом, если щелкнуть по заголовку столбца, то для компонента будет
 генерироваться событие [OnColumnClick](../../Class/ListView/ListView.OnColumnClick.htm).
 При установке значения False щелчок
 по заголовкам столбцов не обрабатывается.
