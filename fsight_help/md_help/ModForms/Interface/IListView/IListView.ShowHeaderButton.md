# IListView.ShowHeaderButton


## Синтаксис


ShowHeaderButton: Boolean;


## Описание


Свойство ShowHeaderButton определяет, отображается ли кнопка вызова меню заголовков столбцов.


## Комментарии


Допустимые значения:


-
True. Кнопка отображается, меню заголовков столбцов отображается при её нажатии. Контекстное меню для заголовка столбцов отсутствует;


-
False. Значение по умолчанию. Кнопка не отображается. Для настройки заголовков столбцов доступно контекстное меню.
