# IListView.ShowHints


## Синтаксис


ShowHints: Boolean;


## Описание


Свойство ShowHints определяет, будет ли появляться всплывающая подсказка при задержке курсора мыши над элементами.


## Комментарии


Если свойству установлено значение True, то при задержке курсора мыши над элементом будет отображена всплывающая подсказка. Текст всплывающей подсказки для элемента устанавливается в свойстве [Hint](../IListViewItem/IListViewItem.Hint.htm). Если свойство [Hint](../IListViewItem/IListViewItem.Hint.htm) не задано, то текст всплывающей подсказки будет соответствовать [тексту](../IListViewItem/IListViewItem.Text.htm) элемента.


Если для компонента установлен стиль [ListViewStyle.Report](../../Enums/ListViewStyle.htm), то текст всплывающей подсказки будет соответствовать тексту в области того столбца, где находится курсор мыши. Всплывающая подсказка будет отображена в том случае, если текст элемента не помещается в область столбца. Всплывающая подсказка, установленная в свойстве [Hint](../IListViewItem/IListViewItem.Hint.htm), отображается при задержке курсора мыши в области первого столбца компонента.


Примечание. Всплывающая подсказка, установленная в свойстве [Hint](../IListViewItem/IListViewItem.Hint.htm) при стиле компонента [ListViewStyle.Report](../../Enums/ListViewStyle.htm), отображается при любой длине текста элемента в области первой колонки.
