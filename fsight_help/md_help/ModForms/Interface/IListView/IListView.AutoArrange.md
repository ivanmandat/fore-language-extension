# IListView.AutoArrange


## Синтаксис


AutoArrange: Boolean;


## Описание


Свойство AutoArrange определяет, будут ли элементы автоматически располагаться правильными рядами при изменении их количества.


## Комментарии


Для автоматического расположения необходимо свойству AutoArrange установить значение True.
