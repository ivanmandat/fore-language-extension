# IListView.GetItemAt


## Синтаксис


GetItemAt(Pt: [IPoint](../IPoint/IPoint.htm); [FullRow: Boolean = False]): [IListViewItem](../IListViewItem/IListViewItem.htm);


## Параметры


Pt - точка, в которой можно получить элемент компонента.


FullRow - необязательный параметр логического типа. В случае установки параметру значения True, получить элемент компонента можно выбрав строку, в которой находится элемент. При установке значения False элемент можно получить только в заданной точке. По умолчанию параметр имеет значение False.


## Описание


Метод GetItemAt позволяет получить элемент компонента в заданной точке.
