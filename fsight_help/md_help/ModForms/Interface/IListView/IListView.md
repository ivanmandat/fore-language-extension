# IListView


## Описание


Интерфейс IListView содержит
 свойства и методы компонента среды разработки - [ListView](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ListView.htm).

## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AdjustWidth](IListView.AdjustWidth.htm)
		 Метод AdjustWidth осуществляет
		 автоматический подгон ширины столбца с учётом его содержимого.

		 Метод GetItemAt позволяет
