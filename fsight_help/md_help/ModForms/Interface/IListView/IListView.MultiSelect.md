# IListView.MultiSelect


## Синтаксис


MultiSelect: Boolean;


## Описание


Свойство MultiSelect определяет
 возможность множественной отметки элементов списка.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 В компоненте доступна множественная отметка элементов;


	- False. В компоненте
	 можно отметить только один элемент.
