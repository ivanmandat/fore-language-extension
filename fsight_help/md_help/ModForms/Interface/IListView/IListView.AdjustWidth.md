# IListView.AdjustWidth


## Синтаксис


AdjustWidth(Column: Integer);


## Параметры


Column. Индекс
 столбца, в котором необходимо осуществить автоматический подгон ширины.


## Описание


Метод AdjustWidth осуществляет
 автоматический подгон ширины столбца с учётом его содержимого.


## Комментарии


При вызове метода ширина подгоняется с учётом всех компонентов, которые
 могут содержаться у элемента (флажок, пиктограмма и пиктограмма состояния,
 текст элемента). Метод актуален, если свойству [Style](IListView.Style.htm)
 установлено значение Report.
