# IListView.SortColumn


## Синтаксис


SortColumn: Integer;


## Описание


Свойство SortColumn определяет индекс колонки, по которой осуществляется сортировка.


## Комментарии


Свойство учитывается если [Style](IListView.Style.htm) установлено в Report.
