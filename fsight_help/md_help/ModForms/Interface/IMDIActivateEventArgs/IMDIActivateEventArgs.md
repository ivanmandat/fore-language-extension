# IMDIActivateEventArgs


## Описание


Интерфейс IMDIActivateEventArgs
 содержит свойства аргумента события, возникающего во время смены активной
 дочерней MDI-формы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Activated](IMDIActivateEventArgs.Activated.htm)


		 Свойство Activated
		 возвращает дочернюю MDI-форму, которая стала активной.


		 ![](../../Property_Image.gif)
		 [Deactivated](IMDIActivateEventArgs.Deactivated.htm)


		 Свойство Deactivated
		 возвращает дочернюю MDI-форму, которая была активной.
