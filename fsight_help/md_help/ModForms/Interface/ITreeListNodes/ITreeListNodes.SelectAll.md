# ITreeListNodes.SelectAll


## Синтаксис


SelectAll([Value: Boolean = True]);


## Параметры


Value - параметр, определяющий состояние отметки. True - отметить все вершины, False - снять отметку со всех вершин.


## Описание


Метод SelectAll позволяет отметить все вершин дерева.
