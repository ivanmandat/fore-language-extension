# ITreeListNodes.Add


## Синтаксис


Add(Node: [ITreeListNode](../ITreeListNode/ITreeListNode.htm);
 Text: String): [ITreeListNode](../ITreeListNode/ITreeListNode.htm);


## Параметры


Node. Вершина, на уровне которой,
 будет добавлена новая вершина дерева.


Text. Текст первого столбца
 добавляемой вершины.


## Описание


Метод Add осуществляет добавление
 новой вершины дерева.


Примечание.
 Длина значения, указываемого в параметре Text,
 не должно превышать 5450 символов.
