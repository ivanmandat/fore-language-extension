# ITreeListNodes.FindById


## Синтаксис


FindById(Id: Integer): [ITreeListNode](../ITreeListNode/ITreeListNode.htm);


## Параметры


Id - идентификатор вершины дерева.


## Описание


Метод FindById осуществляет поиск вершины дерева, идентификатор которой передается посредством параметра Id.
