# ITreeListNodes.BeginUpdate


## Синтаксис


BeginUpdate;


## Описание


Метод BeginUpdate блокирует обновление визуального дерева компонента до тех пор, пока не будет выполнен метод [EndUpdate](ITreeListNodes.EndUpdate.htm).


## Комментарии


Используется при одновременной вставке нескольких вершин дерева для предотвращения мерцания дерева в компоненте.
