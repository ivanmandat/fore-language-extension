# ITreeListNodes.AddChild


## Синтаксис


AddChild(Node: [ITreeListNode](../ITreeListNode/ITreeListNode.htm);
 Text: String): [ITreeListNode](../ITreeListNode/ITreeListNode.htm);


## Параметры


Node. Вершина,
 в иерархию которой, будет добавлена дочерняя вершина дерева.


Text. Текст первого столбца
 добавляемой вершины.


## Описание


Метод AddChild осуществляет
 добавление дочерней вершины для указанной вершины.


Примечание.
 Длина значения, указываемого в параметре Text,
 не должно превышать 5450 символов.
