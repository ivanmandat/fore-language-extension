# ITreeListNodes


## Описание


Интерфейс ITreeListNodes содержит
 свойства и методы коллекции вершин.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITreeListNodes.Count.htm)
		 Свойство Count возвращает
		 количество вершин в коллекции.


		 ![](../../Property_Image.gif)
		 [FirstNode](ITreeListNodes.FirstNode.htm)
		 Свойство FirstNode
		 возвращает первую вершину дерева.


		 ![](../../Property_Image.gif)
		 [TreeControl](ITreeListNodes.TreeControl.htm)
		 Свойство TreeControl
		 возвращает объект-родитель.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ITreeListNodes.Add.htm)
		 Метод Add осуществляет
		 добавление новой вершины дерева.


		 ![](../../Sub_Image.gif)
		 [AddChild](ITreeListNodes.AddChild.htm)
		 Метод AddChild осуществляет
		 добавление дочерней вершины для указанной вершины.


		 ![](../../Sub_Image.gif)
		 [AddChildFirst](ITreeListNodes.AddChildFirst.htm)
		 Метод AddChildFirst
		 осуществляет добавление дочерней вершины в первую позицию для
		 указанной вершины.


		 ![](../../Sub_Image.gif)
		 [AddFirst](ITreeListNodes.AddFirst.htm)
		 Метод AddFirst осуществляет
		 добавление вершины в первую позицию на том уровне, на котором
		 расположена указанная вершина.


		 ![](../../Sub_Image.gif)
		 [BeginUpdate](ITreeListNodes.BeginUpdate.htm)
		 Метод BeginUpdate блокирует
		 обновление визуального дерева компонента.


		 ![](../../Sub_Image.gif)
		 [Clear](ITreeListNodes.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех вершин дерева.


		 ![](../../Sub_Image.gif)
		 [Delete](ITreeListNodes.Delete.htm)
		 Метод Delete осуществляет
		 удаление указанной вершины.


		 ![](../../Sub_Image.gif)
		 [EndUpdate](ITreeListNodes.EndUpdate.htm)
		 Метод EndUpdate отменяет
		 действие метода [BeginUpdate](ITreeListNodes.BeginUpdate.htm).


		 ![](../../Sub_Image.gif)
		 [FindById](ITreeListNodes.FindById.htm)
		 Метод FindById осуществляет
		 поиск вершины дерева по идентификатору.


		 ![](../../Sub_Image.gif)
		 [Insert](ITreeListNodes.Insert.htm)
		 Метод Insert осуществляет
		 вставку новой вершины в позицию, перед указанной вершиной.


		 ![](../../Sub_Image.gif)
		 [SelectAll](ITreeListNodes.SelectAll.htm)
		 Метод SelectAll позволяет
		 отметить все вершины дерева.


		 ![](../../Sub_Image.gif)
		 [UnselectAll](ITreeListNodes.UnselectAll.htm)
		 Метод UnselectAll позволяет
		 снять отметку со всех вершин дерева.
