# ITreeListNodes.AddFirst


## Синтаксис


AddFirst(Node: [ITreeListNode](../ITreeListNode/ITreeListNode.htm);
 Text: String): [ITreeListNode](../ITreeListNode/ITreeListNode.htm);


## Параметры


Node. Вершина, на уровне которой,
 будет добавлена первая вершина дерева.


Text. Текст первого столбца
 добавляемой вершины.


## Описание


Метод AddFirst осуществляет
 добавление вершины в первую позицию на том уровне, на котором расположена
 вершина, передаваемая посредством параметра Node.


Примечание.
 Длина значения, указываемого в параметре Text,
 не должно превышать 5450 символов.
