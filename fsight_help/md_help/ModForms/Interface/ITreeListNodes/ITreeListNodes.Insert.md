# ITreeListNodes.Insert


## Синтаксис


Insert(Node: [ITreeListNode](../ITreeListNode/ITreeListNode.htm);
 Text: String): [ITreeListNode](../ITreeListNode/ITreeListNode.htm);


## Параметры


Node. Вершина, на уровне которой,
 будет добавлена новая вершина дерева.


Text. Текст вставляемой вершины.


## Описание


Метод Insert осуществляет вставку
 новой вершины в позицию, перед вершины, передаваемой посредством параметра
 Node.


Примечание.
 Длина значения, указываемого в параметре Text,
 не должно превышать 5450 символов.
