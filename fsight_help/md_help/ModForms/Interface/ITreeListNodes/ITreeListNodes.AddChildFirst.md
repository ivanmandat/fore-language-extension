# ITreeListNodes.AddChildFirst


## Синтаксис


AddChildFirst(Node: [ITreeListNode](../ITreeListNode/ITreeListNode.htm);
 Text: String): [ITreeListNode](../ITreeListNode/ITreeListNode.htm);


## Параметры


Node. Вершина, в иерархию которой,
 будет добавлена первая дочерняя вершина дерева.


Text. Текст первого столбца
 добавляемой вершины.


## Описание


Метод AddChildFirst осуществляет
 добавление дочерней вершины в первую позицию для вершины, передаваемой
 посредством параметра Node.


Примечание.
 Длина значения, указываемого в параметре Text,
 не должно превышать 5450 символов.
