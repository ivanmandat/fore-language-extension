# IScrollBars.HScrollInfo


## Синтаксис


HScrollInfo: [IScrollInfo](../IScrollInfo/IScrollInfo.htm);


## Описание


Свойство HScrollInfo возвращает информацию о горизонтальной полосе прокрутки.


## Комментарии


Если компонент не имеет горизонтальной полосы прокрутки, то данное свойство вернет объект, имеющий нулевые границы.


## Пример


Пример использования данного свойства приведен в описании свойства [ScrollBars](IScrollBars.ScrollBars.htm).
