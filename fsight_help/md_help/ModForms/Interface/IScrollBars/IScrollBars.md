# IScrollBars


Сборка: Forms;


## Описание


Интерфейс IScrollBars содержит
 свойства для получения информации о полосах прокрутки компонентов.


## Иерархия наследования


IScrollBars


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [HScrollInfo](IScrollBars.HScrollInfo.htm)
		 Свойство HScrollInfo
		 возвращает информацию о горизонтальной полосе прокрутки.


		 ![](../../Property_Image.gif)
		 [ScrollBars](IScrollBars.ScrollBars.htm)
		 Свойство ScrollBars
		 возвращает значение, указывающее наличие полос прокрутки у компонента.


		 ![](../../Property_Image.gif)
		 [VScrollInfo](IScrollBars.VScrollInfo.htm)
		 Свойство VScrollInfo
		 возвращает информацию о вертикальной полосе прокрутки.
