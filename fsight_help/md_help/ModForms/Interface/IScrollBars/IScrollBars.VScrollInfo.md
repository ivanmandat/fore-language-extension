# IScrollBars.VScrollInfo


## Синтаксис


VScrollInfo: [IScrollInfo](../IScrollInfo/IScrollInfo.htm);


## Описание


Свойство VScrollInfo возвращает информацию о вертикальной полосе прокрутки.


## Комментарии


Если компонент не имеет вертикальной полосы прокрутки, то данное свойство вернет объект, имеющий нулевые границы.


## Пример


Пример использования данного свойства приведен в описании свойства [ScrollBars](IScrollBars.ScrollBars.htm).
