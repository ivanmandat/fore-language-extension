# IOleDocumentBox.EnableDocumentEvents


## Синтаксис


EnableDocumentEvents: Boolean;


## Описание


Свойство EnableDocumentEvents определяет, будет ли генерироваться событие [OnOleDocumentEvent](../../Class/OleDocumentBox/OleDocumentBox.OnOleDocumentEvent.htm), используемое для обработки событий объектов, загруженных в компонент [OleDocumentBox](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/OleDocumentBox.htm).
