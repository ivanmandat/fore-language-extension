# IDateTimePickerExEventArgs


## Описание


Интерфейс IDateTimePickerEventArgs
 содержит свойства аргумента событий [OnChange](../../Class/DateTimePickerEx/DateTimePickerEx.OnChange.htm),
 [OnCloseUp](../../Class/DateTimePickerEx/DateTimePickerEx.OnCloseUp.htm),
 [OnDropDown](../../Class/DateTimePickerEx/DateTimePickerEx.OnDropDown.htm)
 компонента [DateTimePickerEx](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/DateTimePickerEx.htm), возникающих
 при различных изменениях в компоненте во время выполнения формы.
