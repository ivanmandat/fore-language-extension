# IWidget.NavBar


## Синтаксис


NavBar: [INavigationBar](../INavigationBar/INavigationBar.htm);


## Описание


Свойство NavBar определяет компонент,
 который будет использоваться для визуального отображения вкладок.


## Комментарии


В качестве значения свойства укажите любой доступный компонент [NavigationBar](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/NavigationBar.htm).
 В указанный компонент будут загружены системные вкладки. Набор вкладок
 и доступные на них настройки зависят от источника текущего компонента.
