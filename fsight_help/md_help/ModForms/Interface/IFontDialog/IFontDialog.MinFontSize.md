# IFontDialog.MinFontSize


## Синтаксис


MinFontSize: Integer;


## Описание


Свойство MinFontSize определяет
 минимальный размер шрифта, который можно будет установить для выбранного
 в диалоге шрифта.


## Комментарии


По умолчанию свойству установлено значение 0, при этом минимальный размер
 ограничивается параметрами выбранного шрифта. Свойство актуально, если
 свойству [LimitSize](../IFontDialogOptions/IFontDialogOptions.LimitSize.htm)
 установлено значение True.
