# IFontDialog.MaxFontSize


## Синтаксис


MaxFontSize: Integer;


## Описание


Свойство MaxFontSize определяет
 максимальный размер шрифта, который можно будет установить для выбранного
 в диалоге шрифта.


## Комментарии


По умолчанию свойству установлено значение 0, при этом максимальный
 размер ограничивается параметрами выбранного шрифта. Свойство актуально,
 если свойству [LimitSize](../IFontDialogOptions/IFontDialogOptions.LimitSize.htm)
 установлено значение True.
