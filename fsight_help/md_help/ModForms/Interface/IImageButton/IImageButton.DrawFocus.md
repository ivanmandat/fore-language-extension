# IImageButton.DrawFocus


## Синтаксис


DrawFocus: Boolean;


## Описание


Свойство DrawFocus определяет признак отображения рамки фокуса вдоль границы компонента.


## Комментарии


Данное свойство актуально, когда кнопка имеет фокус. Если свойству установлено значение True, то вдоль границы компонента будет отрисована рамка фокуса.


Если свойству установлено значение False, то наличие фокуса ни как не отображается.
