# IImageButton.AllowAllUp


## Синтаксис


AllowAllUp: Boolean;


## Описание


Свойство AllowAllUp определяет, могут ли все кнопки, составляющие группу, находиться одновременно в состоянии "отпущена".


## Комментарии


Свойство AllowAllUp можно использовать лишь для кнопок, составляющих группу (свойство [GroupIndex](IImageButton.GroupIndex.htm) каждой кнопки имеет одно и то же ненулевое значение). Если значение [GroupIndex](IImageButton.GroupIndex.htm) равно 0 (кнопка не принадлежит ни к какой группе), изменение значения свойства AllowAllUp не производит никакого действия. Если свойство AllowAllUp имеет значение False (принимается по умолчанию), то одна из кнопок группы должна быть нажатой (значение свойства [Down](IImageButton.Down.htm) равно True). При выполнении щелчка на этой кнопке ее состояние не изменятеся (она по-прежнему будет отображаться в состоянии "нажата"). Кнопка перейдет в состояние "отпущена" только после того, как будет выполнен щелчок на другой кнопке этой группы. Если свойство AllowAllUp имеет значение True, то все кнопки можно нажимать раздельно. При изменении значения свойства AllowAllUp одной из кнопок группы значение AllowAllUp всех остальных кнопок группы также изменяется.
