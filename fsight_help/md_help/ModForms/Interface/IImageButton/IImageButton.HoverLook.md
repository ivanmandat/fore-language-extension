# IImageButton.HoverLook


## Синтаксис


HoverLook: [IImageButtonLook](../IImageButtonLook/IImageButtonLook.htm);


## Описание


Свойство HoverLook возвращает настройки внешнего вида кнопки (изображение и цвет текста надписи), который она примет при прохождении над ней курсора мыши.
