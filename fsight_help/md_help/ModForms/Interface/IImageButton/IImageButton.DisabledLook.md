# IImageButton.DisabledLook


## Синтаксис


DisabledLook: [IImageButtonLook](../IImageButtonLook/IImageButtonLook.htm);


## Описание


Свойство DisabledLook возвращает настройки внешнего вида кнопки (изображение и цвет текста надписи), который она примет в недоступном состоянии.
