# IImageButton.Spacing


## Синтаксис


Spacing: Integer;


## Описание


Свойство Spacing определяет интервал между пиктограммой и текстом компонента.


## Комментарии


Интервал указывается в пикселях. По умолчанию свойству установлено значение 4 пикселя.
