# IImageButton.Images


## Синтаксис


Images: [IImageList](../IImageList/IImageList.htm);


## Описание


Свойство Images определяет наименование компонента [ImageList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ImageList.htm), изображения которого будут использоваться в компоненте.


## Комментарии


В список доступных значений данного свойства входят имена всех компонентов [ImageList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ImageList.htm), расположенных на форме.
