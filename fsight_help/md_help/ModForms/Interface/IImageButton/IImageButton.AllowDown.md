# IImageButton.AllowDown


## Синтаксис


AllowDown: Boolean;


## Описание


Свойство AllowDown определяет,
 будет ли кнопка после первого нажатия оставаться в нажатом состоянии.


## Комментарии


Если свойству установлено значение False,
 то при нажатии кнопка будет возвращаться в исходное состояние. По умолчанию
 установлено значение False.
