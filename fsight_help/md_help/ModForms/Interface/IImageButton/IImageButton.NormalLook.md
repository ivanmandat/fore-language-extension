# IImageButton.NormalLook


## Синтаксис


NormalLook: [IImageButtonLook](../IImageButtonLook/IImageButtonLook.htm);


## Описание


Свойство NormalLook возвращает настройки внешнего вида кнопки (изображение и цвет текста надписи), который она примет в не нажатом состоянии.
