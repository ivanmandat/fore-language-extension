# IImageButton.BorderStyle


## Синтаксис


BorderStyle: [ControlBorderStyle](../../Enums/ControlBorderStyle.htm);


## Описание


Свойство BorderStyle определяет стиль рамки, обрамляющей компонент.


## Комментарии


Если для свойства установлено значение Flat, границы кнопки не отображаются.
