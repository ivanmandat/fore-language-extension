# IListViewColumn.ImageIndex


## Синтаксис


ImageIndex: Integer;


## Описание


Свойство ImageIndex определяет индекс изображения из списка свойства [LargeImages](../IListView/IListView.LargeImages.htm) компонента [ListView](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ListView.htm). Если столбец не содержит изображения, то ImageIndex = -1.
