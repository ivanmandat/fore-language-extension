# IListViewColumn.AutoSize


## Синтаксис


AutoSize: Boolean;


## Описание


Свойство AutoSize определяет
 признак автоматического изменения ширины столбца в зависимости от ширины
 компонента.


## Комментарии


Допустимые значения:


	- True. При изменении
	 ширины других столбцов, ширина текущего столбца будет изменяться автоматически
	 таким образом, чтобы все столбцы поместились в область компонента
	 и по возможности не было горизонтальной полосы прокрутки. При этом
	 также работа компонента осуществляется с учетом следующих моментов:


		- Если свойство AutoSize
		 = True установлено для нескольких столбцов, то их ширина
		 будет изменяться пропорционально;


		- Если суммарная ширина других колонок больше области компонента,
		 то для столбцов с установленным AutoSize
		 будет установлена нулевая ширина. При этом также возможно
		 появление горизонтальной полосы прокрутки;


		- При ручном или программном изменении ширины текущего столбца
		 значение свойства AutoSize не
		 сбрасывается. Автоматический расчет ширины будет произведен при
		 следующем изменении ширины для любого столбца, у которого AutoSize = False.


	- False. Столбец будет
	 иметь ширину, которая определена в свойстве [Width](IListViewColumn.Width.htm)
	 или установлена пользователем вручную.
