# IListViewColumn


## Описание


Интерфейс IListViewColumn содержит
 свойства и методы столбца компонента [ListView](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ListView.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Alignment](IListViewColumn.Alignment.htm)


		 Свойство Alignment
		 устанавливает способ расположения надписей элементов дерева в
		 столбце.


		 ![](../../Property_Image.gif)
		 [AutoSize](IListViewColumn.AutoSize.htm)


		 Свойство AutoSize определяет
		 признак автоматического изменения ширины столбца в зависимости
		 от ширины компонента.


		 ![](../../Property_Image.gif)
		 [Caption](IListViewColumn.Caption.htm)


		 Свойство Caption определяет
		 заголовок столбца компонента.


		 ![](../../Property_Image.gif)
		 [ImageIndex](IListViewColumn.ImageIndex.htm)


		 Свойство ImageIndex
		 определяет номер изображения (начиная с нуля) из списка свойства
		 [LargeImages](../IListView/IListView.LargeImages.htm)
		 компонента.


		 ![](../../Property_Image.gif)
		 [Index](IListViewColumn.Index.htm)


		 Свойство Index определяет
		 индекс текущего столбца.


		 ![](../../Property_Image.gif)
		 [MaxWidth](IListViewColumn.MaxWidth.htm)


		 Свойство MaxWidth определяет
		 максимальную ширину столбца.


		 ![](../../Property_Image.gif)
		 [MinWidth](IListViewColumn.MinWidth.htm)


		 Свойство MinWidth определяет
		 минимальную ширину столбца.


		 ![](../../Property_Image.gif)
		 [SortAscending](IListViewColumn.SortAscending.htm)


		 Свойство SortAscending
		 определяет направление сортировки.


		 ![](../../Property_Image.gif)
		 [Visible](IListViewColumn.Visible.htm)


		 Свойство Visible определяет
		 видимость колонки.


		 ![](../../Property_Image.gif)
		 [Width](IListViewColumn.Width.htm)


		 Свойство Width определяет
		 ширину столбца в пикселях.


## Методы

		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Delete](IListViewColumn.Delete.htm)
