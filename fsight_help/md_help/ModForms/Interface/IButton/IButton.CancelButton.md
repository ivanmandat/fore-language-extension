# IButton.CancelButton


## Синтаксис


CancelButton: Boolean;


## Описание


Свойство CancelButton определяет, будет ли обрабатываться событие [OnClick](../IControl/IControl.OnClick.htm) при нажатии клавиши ESC.


## Комментарии


Если свойству установлено значение True, то нажатие клавиши ESC, в то время, когда фокус находится в рамках родительской формы, будет эквивалентно нажатию на данную кнопку. Целесообразно использовать данное свойство для кнопок «Отменить» в различных диалоговых формах.
