# IButton.DefaultButton


## Синтаксис


DefaultButton: Boolean;


## Описание


Свойство DefaultButton определяет,
 что нажатие пользователем клавиши ENTER будет генерировать событие [OnClick](../IControl/IControl.OnClick.htm), для данной кнопки,
 даже если она в этот момент не находится в фокусе.


## Комментарии


Если в момент нажатия клавиши ENTER в фокусе находится другая кнопка,
 то генерируется событие именно для этой кнопки.
