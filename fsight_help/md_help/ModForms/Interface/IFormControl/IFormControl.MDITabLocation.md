# IFormControl.MDITabLocation


## Синтаксис


MDITabLocation: [MDIFrameTabLocation](../../Enums/MDIFrameTabLocation.htm);


## Описание


Свойство MDITabLocation определяет месторасположение вкладок дочерних MDI-окон в области главной MDI-формы.


## Комментарии


По умолчанию вкладки дочерних MDI-окон отображаются в верхней части главной MDI-формы.
