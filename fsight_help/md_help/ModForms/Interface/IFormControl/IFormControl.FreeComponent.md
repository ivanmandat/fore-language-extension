# IFormControl.FreeComponent


## Синтаксис


FreeComponent(Value: [IComponent](../IComponent/IComponent.htm));


## Параметры


Value - компонент, который необходимо удалить.


## Описание


Метод FreeComponent позволяет визуально удалить компонент, но ссылки на этот компонент останутся.


Примечание. Для удаления ссылок на этот компонент (т.е. очистить память) необходимо удаляемому объекту присвоить значение Null.
