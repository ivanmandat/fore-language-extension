# IFormControl


Сборка: Forms;


## Описание


Интерфейс IFormControl содержит
 свойства и методы пользовательской формы.


## Комментарии


Используется только в настольном приложении.


## Иерархия наследования

           [IControl](../IControl/IControl.htm)

## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ActiveControl](IFormControl.ActiveControl.htm)


		 Свойство ActiveControl
		 определяет компонент, который был последним сфокусированным компонентом
		 на форме.


		 ![](../../Property_Image.gif)
		 [BorderStyle](IFormControl.BorderStyle.htm)


		 Свойство BorderStyle
		 определяет стиль обрамления компонента.


		 ![](../../Property_Image.gif)
		 [Constraints](IFormControl.Constraints.htm)


		 Свойства Constraints
		 задает ограничения на размеры компонента в пикселях.


		 ![](../../Property_Image.gif)
		 [Icon](IFormControl.Icon.htm)


		 Свойство Icon определяет
		 пользовательскую пиктограмму, отображаемую в заголовке формы.


		 ![](../../Property_Image.gif)
		 [MainMenu](IFormControl.MainMenu.htm)


		 Свойство MainMenu определяет
		 главное меню формы.


		 ![](../../Property_Image.gif)
		 [MaxTabLength](IFormControl.MaxTabLength.htm)


		 Свойство MaxTabLength
		 определяет максимальную допустимую длину заголовка дочерней MDI-формы,
		 при превышении которой в заголовке вкладки, соответствующей данной
		 форме, будет отображаться сокращенный заголовок.


		 ![](../../Property_Image.gif)
		 [MDIActive](IFormControl.MDIActive.htm)


		 Сво йство MDIActive
		 определяет дочернюю MDI-форму, активную в данный момент.


		 ![](../../Property_Image.gif)
		 [MDIChildCount](IFormControl.MDIChildCount.htm)


		 Свойство MDIChildCount
		 возвращает количество дочерних MDI-форм.


		 ![](../../Property_Image.gif)
		 [MDIChildren](IFormControl.MDIChildren.htm)


		 Свойство MDIChildren
		 возвращает параметры дочерней MDI-формы.


		 ![](../../Property_Image.gif)
		 [MDITabLocation](IFormControl.MDITabLocation.htm)


		 Свойство MDITabLocation
		 определяет месторасположение вкладок дочерних MDI-окон в области
		 главной MDI-формы.


		 ![](../../Property_Image.gif)
		 [MDITabMenu](IFormControl.MDITabMenu.htm)


		 Свойство MDITabMenu
		 определяет контекстное меню, вызываемое в области вкладок, соответствующих
		 дочерним MDI-формам.


		 ![](../../Property_Image.gif)
		 [ModalResult](IFormControl.ModalResult.htm)


		 Свойство ModalResult
		 определяет результат выполнения формы.


		 ![](../../Property_Image.gif)
		 [ParentWindow](IFormControl.ParentWindow.htm)


		 Свойство ParentWindow
		 возвращает родительское окно.


		 ![](../../Property_Image.gif)
		 [Position](IFormControl.Position.htm)


		 Свойство Position определяет
		 место расположение окна формы на экране.


		 ![](../../Property_Image.gif)
		 [Resources](IFormControl.Resources.htm)


		 Свойство Resources
		 определяет объект-ресурсы, из которого будут браться текстовые
		 и графические значения для компонентов формы.


		 ![](../../Property_Image.gif)
		 [ShowOnTaskbar](IFormControl.ShowOnTaskbar.htm)


		 Свойство ShowOnTaskbar
		 определяет, будет ли отображаться в панели задач значок формы
		 во время выполнения формы.


		 ![](../../Property_Image.gif)
		 [StartupObject](IFormControl.StartupObject.htm)


		 Свойство StartupObject
		 возвращает описание запущенного объекта напрямую от объекта.


		 ![](../../Property_Image.gif)
		 [ToolbarPopupMenu](IFormControl.ToolbarPopupMenu.htm)


		 Свойство ToolbarPopupMenu
		 определяет контекстное меню, которое будет появляться по щелчку
		 дополнительной кнопки мыши в области главного меню или в области
		 панелей инструментов формы.


		 ![](../../Property_Image.gif)
		 [TopMost](IFormControl.TopMost.htm)


		 Свойство TopMost определяет,
		 каким образом будет располагаться форма относительно других окон
		 при выполнении.


		 ![](../../Property_Image.gif)
		 [WindowState](IFormControl.WindowState.htm)


		 Свойство WindowState
		 определяет состояние формы на экране во время запуска.


		 ![](../../Property_Image.gif)
		 [WindowStyle](IFormControl.WindowStyle.htm)


		 Свойство WindowStyle
		 определяет стиль формы.
