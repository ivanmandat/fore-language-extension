# IFormControl.TopMost


## Синтаксис


TopMost: Boolean;


## Описание


Свойство TopMost определяет, каким образом будет располагаться форма относительно других окон при выполнении.


## Комментарии


Если свойству установлено значение True, то форма всегда будет располагаться поверх других окон.
