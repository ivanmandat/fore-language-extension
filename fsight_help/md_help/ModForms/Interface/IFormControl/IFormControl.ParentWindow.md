# IFormControl.ParentWindow


## Синтаксис


ParentWindow: [IWin32Window](../IWin32Window/IWin32Window.htm);


## Описание


Свойство ParentWindow возвращает родительское окно.


## Комментарии


Если для формы установлено родительское окно, то форма отображается поверх этого окна. Для получения параметров родительского окна необходимо значение свойство ParentWindow привести к необходимому типу.


Если родительское окно отсутствует, то свойство возвращает значение Null.
