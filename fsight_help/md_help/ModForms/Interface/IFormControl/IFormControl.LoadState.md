# IFormControl.LoadState


## Синтаксис


LoadState(SectionName: String);


## Параметры


SectionName. Наименование раздела
 в реестре, из которого будут загружаться параметры панелей инструментов,
 всплывающих панелей, главного меню формы. Наименование должно состоять
 из одного или более слов и содержать символы, которые допустимо использовать
 в реестре Windows.


## Описание


Метод LoadState осуществляет
 загрузку параметров элементов управления формы из реестра.


## Комментарии


Поиск раздела SectionName будет
 осуществляться в следующей ветке реестра: [HKEY_CURRENT_USER\Software\Foresight\Foresight Analytics Platform\10.0\<версия продукта
 «Форсайт. Аналитическая платформа»>\Studio.INI\<наименование
 репозитория>\]. Если в данной ветке указанный раздел отсутствует,
 то загрузка настроек произведена не будет.


Из реестра загружаются следующие данные:


	- количество панелей инструментов и всплывающих панелей.


	- расположение и размеры панелей инструментов, всплывающих
	 панелей, главного меню формы.


	- признак видимости панелей инструментов и всплывающих
	 панелей.
