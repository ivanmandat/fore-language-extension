# IFormControl.MDITabMenu


## Синтаксис


MDITabMenu: [IPopupMenu](../IPopupMenu/IPopupMenu.htm);


## Описание


Свойство MDITabMenu определяет
 контекстное меню, вызываемое в области вкладок, соответствующих дочерним
 MDI-формам.


## Комментарии


Данное свойство устанавливается для родительской MDI-формы.


По умолчанию свойству установлено значение Null,
 при этом в области вкладок будет вызываться системное контекстное меню,
 содержащее три пункта:


	- Закрыть окно;


	- Закрыть все окна, кроме текущего;


	- Закрыть все окна.
