# IFormControl.EnterContextHelpMode


## Синтаксис


EnterContextHelpMode;


## Описание


Метод EnterContextHelpMode переводит форму в режим отображения справки.


## Комментарии


Метод эмулирует нажатие кнопки-подсказки «?» при установленном свойстве [BorderStyle](IFormControl.BorderStyle.htm) = DialogWithHelp.
