# IFormControl.MDIIconArrange


## Синтаксис


MDIIconArrange;


## Описание


Метод MDIIconArrange упорядочивает
 все минимизированные дочерние MDI-формы.


## Комментарии


Заголовки форм будут сгруппированы в нижнем левом углу главной родительской
 MDI-формы.
