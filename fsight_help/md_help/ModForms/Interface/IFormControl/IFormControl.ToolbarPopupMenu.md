# IFormControl.ToolbarPopupMenu


## Синтаксис


ToolbarPopupMenu: [IPopupMenu](../IPopupMenu/IPopupMenu.htm);


## Описание


Свойство ToolbarPopupMenu определяет контекстное меню, которое будет появляться по щелчку дополнительной кнопки мыши в области главного меню или в области панелей инструментов формы.


## Комментарии


В список допустимых значений входят имена всех компонентов [PopupMenu](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/PopupMenu.htm), размещенных на форме.
