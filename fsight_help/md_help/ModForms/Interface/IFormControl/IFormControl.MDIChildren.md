# IFormControl.MDIChildren


## Синтаксис


MDIChildren(Index: Integer): [IFormControl](IFormControl.htm);


## Параметры


Index - индекс дочерней MDI-формы.


## Описание


Свойство MDIChildren возвращает параметры дочерней MDI-формы, индекс которой передается посредством параметра Index.
