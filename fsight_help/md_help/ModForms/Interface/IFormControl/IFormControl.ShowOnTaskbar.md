# IFormControl.ShowOnTaskbar


## Синтаксис


ShowOnTaskbar: Boolean;


## Описание


Свойство ShowOnTaskbar определяет, будет ли отображаться в панели задач значок формы во время выполнения формы.


## Комментарии


Если свойству установлено значение True, то в панели задач будет отображаться значок формы, иначе не будет.


Примечание. Свойство может изменяться только на этапе проектирования формы.
