# IFormControl.ShowModal


## Синтаксис


ShowModal: [FormModalResult](../../Enums/FormModalResult.htm);


## Описание


Метод ShowModal позволяет открыть модально форму, для которой вызывается данный метод. После закрытия модальной формы метод возвращает результат выполнения формы.


Примечание. Нежелательно открывать форму модально, если она была создана без владельца. Для отображения равноправных форм нужно использовать свойство Visible.

При попытке открытия немодальной формы из модальной, доступ к элементам вызываемой формы будет отсутствовать. Необходимо учитывать данную особенность при разработке прикладных проектов.
