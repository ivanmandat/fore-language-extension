# IFormControl.EndUpdate


## Синтаксис


EndUpdate;


## Описание


Метод EndUpdate возобновляет перерисовку формы.


## Комментарии


После вызова данного метода будут отрисованы все изменения, произведенные между вызовами методов [BeginUpdate](IFormControl.BeginUpdate.htm) и EndUpdate.
