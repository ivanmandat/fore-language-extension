# IFormControl.MDITile


## Синтаксис


MDITile(Value: [MdiTileType](../../Enums/MdiTileType.htm));


## Параметры


Value - параметр, определяющий в каком виде необходимо упорядочить окна дочерних MDI-форм.


## Описание


Метод MDITile формирует все дочерние окна в виде, указанном посредством параметра Value.
