# IFormControl.Icon


## Синтаксис


Icon: [IGxIcon](ModDrawing.chm::/Interface/IGxIcon/IGxIcon.htm);


## Описание


Свойство Icon определяет пользовательскую
 пиктограмму, отображаемую в заголовке формы.


## Комментарии


По умолчанию отображается стандартная пиктограмма продукта «Форсайт. Аналитическая платформа».
 При загрузке собственной пиктограммы изображение должно иметь размер не
 более чем 16*16 пикселей. Если размер больше, то изображение будет смасштабировано
 до размера 16*16 пикселей.
