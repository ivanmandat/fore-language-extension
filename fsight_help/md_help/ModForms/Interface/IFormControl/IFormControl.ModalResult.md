# IFormControl.ModalResult


## Синтаксис


ModalResult: [FormModalResult](../../Enums/FormModalResult.htm);


## Описание


Свойство ModalResult определяет результат выполнения модальной формы.


Примечание. После закрытия модальной формы, базовые свойства всех компонентов и самой формы сохраняются и доступны до завершения процедуры, в которой модальная форма была создана.
