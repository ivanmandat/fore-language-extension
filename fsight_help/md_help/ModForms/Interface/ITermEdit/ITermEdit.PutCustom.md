# ITermEdit.PutCustom


## Синтаксис


PutCustom(Text: String; Formula: String; [Position: Integer = -1]): Integer;


## Параметры


Text. Параметр, определяющий пользовательское выражение, которое будет помещаться в заданную позицию.


Formula. Параметр, определяющий формулу пользовательского терма.


Position. Необязательный параметр, определяющий позицию, в которую будет вставлено пользовательское выражение.


## Описание


Метод PutCustom осуществляет вставку пользовательского выражения в указанную позицию поля редактирования компонента, после чего возвращает индекс выражения в коллекции.


## Комментарии


По умолчанию выражение вставляется в текущую позицию.
