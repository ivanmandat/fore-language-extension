# ITermEdit.EndUpdate


## Синтаксис


EndUpdate;


## Описание


Метод EndUpdate восстанавливает
 работоспособность внутренних механизмов компонента, которые были отключены
 для ускорения вставки термов в компонент.


## Комментарии


Метод восстанавливает работоспособность тех механизмов, которые были
 отключены путем вызова метода [BeginUpdate](ITermEdit.BeginUpdate.htm).
