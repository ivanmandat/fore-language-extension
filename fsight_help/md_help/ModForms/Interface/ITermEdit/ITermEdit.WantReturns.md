# ITermEdit.WantReturns


## Синтаксис


WantReturns: Boolean;


## Описание


Свойство WantReturns определяет возможность вставки в редактируемый текст символа возврата каретки (нажатие клавиши ENTER).


## Комментарии


По умолчанию свойству установлено значение True, при этом будет возможность вставки символа возврата каретки. Если свойству установить значение False, то нажатие клавиши ENTER будет игнорироваться.
