# ITermEdit.Hyperlink


## Синтаксис


Hyperlink(TermIndex: Integer): Boolean;


## Параметры


TermIndex. Индекс
 выражения.


## Описание


Свойство Hyperlink определяет
 признак отображения выражения в виде гиперссылки.


## Комментарии


Индекс выражения передается посредством параметра TermIndex.
 Актуально, если свойству [EnableHyperlinks](ITermEdit.EnableHyperlinks.htm)
 установлено значение True.
