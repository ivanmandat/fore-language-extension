# ITermEdit.PutLiteral


## Синтаксис


PutLiteral(Text: String; [Position: Integer = -1]): Integer;


## Параметры


Text. Параметр, определяющий литеральное выражение, которое будет вставлено в заданную позицию.


Position. Необязательный параметр, определяющий позицию, в которую будет вставлено литеральное выражение.


## Описание


Метод PutLiteral осуществляет размещение литерального выражения в указанную позицию поля редактирования компонента, после чего возвращает индекс выражения в коллекции. По умолчанию выражение вставляется в текущую позицию.
