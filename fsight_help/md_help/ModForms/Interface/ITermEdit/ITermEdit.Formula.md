# ITermEdit.Formula


## Синтаксис


Formula: String;


## Описание


Свойство Formula определяет формулу выражения.


## Комментарии


Данная формула строится из [формул](../ITerm/ITerm.Formula.htm) всех выражений, добавленных в область редактирования компонента. Для получения текста формулы, отображаемой в компоненте, используйте свойство [Text](../IControl/IControl.Text.htm).
