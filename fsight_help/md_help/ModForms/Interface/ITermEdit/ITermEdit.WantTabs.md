# ITermEdit.WantTabs


## Синтаксис


WantTabs: Boolean;


## Описание


Свойство WantTabs определяет возможность вставки в редактируемый текст символа табуляции (нажатие клавиши TAB).


## Комментарии


По умолчанию свойству установлено значение False, при этом клавиша табуляции игнорируется, при нажатии на нее фокус будет переходить к следующему компоненту. Если свойству установить значение True, то при нажатии на клавишу Tab в редактируемый текст компонента будет вставлен символ табуляции.
