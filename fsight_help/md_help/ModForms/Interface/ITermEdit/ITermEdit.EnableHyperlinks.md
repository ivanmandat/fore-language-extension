# ITermEdit.EnableHyperlinks


## Синтаксис


EnableHyperlinks: Boolean;


## Описание


Свойство EnableHyperlinks определяет возможность отображения выражений в виде гиперссылки.


## Комментарии


При установке свойству значения True появится возможность отображения выражений в виде гиперссылок.
