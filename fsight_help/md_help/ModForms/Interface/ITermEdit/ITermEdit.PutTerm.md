# ITermEdit.PutTerm


## Синтаксис


PutTerm(Term: [ITerm](../ITerm/ITerm.htm); [Position: Integer = -1]): Integer;


## Параметры


Term. Параметр, определяющий выражение, которое будет вставлено в заданную позицию.


Position. Необязательный параметр, определяющий позицию, в которую будет помещаться выражение.


## Описание


Метод PutTerm осуществляет вставку выражения в указанную позицию поля редактирования компонента, после чего возвращает индекс выражения в коллекции. По умолчанию выражение вставляется в текущую позицию.
