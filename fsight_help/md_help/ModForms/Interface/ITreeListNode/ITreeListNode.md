# ITreeListNode


Сборка: Forms;


## Описание


Интерфейс ITreeListNode содержит
 свойства и методы вершин дерева.


## Иерархия наследования


           [ITreeListNodeBase](../ITreeListNodeBase/ITreeListNodeBase.htm)


           ITreeListNode


## Комментарии


Для работы с коллекцией вершин используйте интерфейс [ITreeListNodes](../ITreeListNodes/ITreeListNodes.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Accessibility](ITreeListNode.Accessibility.htm)


		 Свойство Accessibility
		 определяет, будут ли активными элементы дерева.


		 ![](../../Property_Image.gif)
		 [Bold](ITreeListNode.Bold.htm)


		 Свойство Bold определяет,
		 будет ли текст элемента отображаться жирным шрифтом.


		 ![](../../Property_Image.gif)
		 [Checked](ITreeListNode.Checked.htm)


		 Свойство Checked определяет,
		 установлена ли отметка для элемента компонента.


		 ![](../../Property_Image.gif)
		 [CheckState](ITreeListNode.CheckState.htm)


		 Свойство CheckState
		 определяет состояние отметки элемента дерева.


		 ![](../../Property_Image.gif)
		 [ColumnText](ITreeListNode.ColumnText.htm)


		 Свойство ColumnText
		 определяет текст указанного столбца для текущего элемента.


		 ![](../../Property_Image.gif)
		 [Cut](ITreeListNode.Cut.htm)


		 Свойство Cut определяет,
		 будет ли элемент отмечен как вырезанный.


		 ![](../../Property_Image.gif)
		 [Data](ITreeListNode.Data.htm)


		 Свойство Data позволяет
		 сопоставлять данные произвольной структуры элементу дерева.


		 ![](../../Property_Image.gif)
		 [Expanded](ITreeListNode.Expanded.htm)


		 Свойство Expanded определяет
		 какой будет иерархия вершины дерева - развернутой или свернутой.


		 ![](../../Property_Image.gif)
		 [Focused](ITreeListNode.Focused.htm)


		 Свойство Focused определяет,
		 находится ли элемент в фокусе.


		 ![](../../Property_Image.gif)
		 [Id](ITreeListNode.Id.htm)


		 Свойство Id возвращает
		 уникальный числовой идентификатор элемента.


		 ![](../../Property_Image.gif)
		 [ImageIndex](ITreeListNode.ImageIndex.htm)


		 Свойство ImageIndex
		 определяет индекс изображения, которое будет располагаться слева
		 от текста элемента в первом столбце компонента.


		 ![](../../Property_Image.gif)
		 [Index](ITreeListNode.Index.htm)


		 Свойство Index возвращает
		 индекс элемента в списке дочерних элементов родителя.


		 ![](../../Property_Image.gif)
		 [IsDropTarget](ITreeListNode.IsDropTarget.htm)


		 Свойство IsDropTarget
		 определяет признак прохождения в текущий момент над данной вершиной
		 какой-либо другой вершины.


		 ![](../../Property_Image.gif)
		 [Level](ITreeListNode.Level.htm)


		 Свойство Level возвращает
		 индекс уровня, на котором расположена данная вершина.


		 ![](../../Property_Image.gif)
		 [NextNode](ITreeListNode.NextNode.htm)


		 Свойство NextNode возвращает
		 следующий элемент дерева.


		 ![](../../Property_Image.gif)
		 [NextSelectedNode](ITreeListNode.NextSelectedNode.htm)


		 Свойство NextSelectedNode
		 возвращает следующий отмеченный элемент дерева.


		 ![](../../Property_Image.gif)
		 [NextSiblingNode](ITreeListNode.NextSiblingNode.htm)


		 Свойство NextSiblingNode
		 возвращает следующую соседнюю вершину, расположенную на том же
		 уровне, на котором расположена данная вершина.


		 ![](../../Property_Image.gif)
		 [OverlayIndex](ITreeListNode.OverlayIndex.htm)


		 Свойство OverlayIndex
		 определяет индекс покрывающей маски.


		 ![](../../Property_Image.gif)
		 [Parent](ITreeListNode.Parent.htm)


		 Свойство Parent возвращает
		 родительский элемент для данного элемента.


		 ![](../../Property_Image.gif)
		 [PreviousNode](ITreeListNode.PreviousNode.htm)


		 Свойство PreviousNode
		 возвращает предыдущий элемент дерева.


		 ![](../../Property_Image.gif)
		 [PreviousSelectedNode](ITreeListNode.PreviousSelectedNode.htm)


		 Свойство PreviousSelectedNode
		 возвращает предыдущий отмеченный элемент дерева.


		 ![](../../Property_Image.gif)
		 [PreviousSiblingNode](ITreeListNode.PreviousSiblingNode.htm)


		 Свойство PreviousSiblingNode
		 возвращает предыдущую соседнюю вершину, расположенную на том же
		 уровне, на котором расположена данная вершина.


		 ![](../../Property_Image.gif)
		 [Selected](ITreeListNode.Selected.htm)


		 Свойство Selected определяет,
		 является ли элемент выбранным.


		 ![](../../Property_Image.gif)
		 [SelectedIndex](ITreeListNode.SelectedIndex.htm)


		 Свойство SelectedIndex
		 определяет индекс, связывающий элемент с одним из рисунков свойства
		 [Images](../ITreeControl/ITreeControl.Images.htm),
		 который будет отображать выделенные элементы.


		 ![](../../Property_Image.gif)
		 [StateIndex](ITreeListNode.StateIndex.htm)


		 Свойство StateIndex
		 определяет индекс, связывающий элемент с одним из рисунков свойства
		 [StateImages](../ITreeControl/ITreeControl.StateImages.htm),
		 который будет отражать состояние элемента.


		 ![](../../Property_Image.gif)
		 [Text](ITreeListNode.Text.htm)


		 Свойство Text определяет
		 текст первого столбца для текущего элемента.


		 ![](../../Property_Image.gif)
		 [Visible](ITreeListNode.Visible.htm)


		 Свойство Visible определяет,
		 попадает ли элемент в видимую область компонента.

## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Delete](ITreeListNode.Delete.htm)


		 Метод Delete осуществляет
		 удаление текущего элемента со всеми дочерними элементами дерева.


		 ![](../../Sub_Image.gif)
		 [EditText](ITreeListNode.EditText.htm)


		 Метод EditText возвращает
		 значение, определяющее возможность включения режима редактирования.


		 ![](../../Sub_Image.gif)
		 [EndEdit](ITreeListNode.EndEdit.htm)


		 Метод EndEdit позволяет
		 выйти из режима редактирования текста элементов дерева.


		 ![](../../Sub_Image.gif)
		 [IsChild](ITreeListNode.IsChild.htm)


		 Метод IsChild позволяет
		 определить является ли текущий элемент дочерним по отношению к
		 элементу, передаваемому в качестве входного параметра.


		 ![](../../Sub_Image.gif)
		 [IsParent](ITreeListNode.IsParent.htm)


		 Метод IsParent позволяет
		 определить является ли текущий элемент родительским по отношению
		 к элементу, передаваемому в качестве входного параметра.


		 ![](../../Sub_Image.gif)
		 [MakeVisible](ITreeListNode.MakeVisible.htm)


		 Метод MakeVisible осуществляет
		 прокрутку дерева элементов таким образом, чтобы данный элемент
		 попал в видимую область компонента.


		 ![](../../Sub_Image.gif)
		 [MoveTo](ITreeListNode.MoveTo.htm)


		 Метод MoveTo осуществляет
		 перемещение вершины в указанную позицию.


		 ![](../../Sub_Image.gif)
		 [SelectNode](ITreeListNode.SelectNode.htm)


		 Метод SelectNode осуществляет
		 выделение элемента дерева.


		 ![](../../Sub_Image.gif)
		 [SelectSiblings](ITreeListNode.SelectSiblings.htm)


		 Метод SelectSiblings
		 осуществляет выделение элементов уровня, на котором расположен
		 данный элемент.


		 ![](../../Sub_Image.gif)
		 [Sort](ITreeListNode.Sort.htm)


		 Метод Sort выполняет
		 сортировку по содержанию указанного столбца.


		 ![](../../Sub_Image.gif)
		 [SortDefault](ITreeListNode.SortDefault.htm)


		 Метод SortDefault осуществляет
		 сортировку по содержанию столбца, указанного в свойстве [SortColumn](../ITreeControl/ITreeControl.SortColumn.htm).


		 ![](../../Sub_Image.gif)
		 [UnselectNode](ITreeListNode.UnselectNode.htm)


		 Метод UnselectNode
		 осуществляет снятие выделения элемента дерева.


		 ![](../../Sub_Image.gif)
		 [UnselectSiblings](ITreeListNode.UnselectSiblings.htm)


		 Метод UnselectSiblings
		 осуществляет снятие отметки элементов уровня, на котором расположен
		 данный элемент.
