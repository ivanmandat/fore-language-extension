# ITreeListNode.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет, попадает ли элемент в видимую область компонента.


## Комментарии


Если свойству установить значение True, то дерево элементов будет прокручено таким образом, чтобы данный элемент попал в видимую область компонента. Установка свойству значения True эквивалентна вызову метода [MakeVisible](ITreeListNode.MakeVisible.htm).


Примечание. При установке свойству значения False, никаких действий не производится.
