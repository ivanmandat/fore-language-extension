# ITreeListNode.Bold


## Синтаксис


Bold: Boolean;


## Описание


Свойство Bold определяет, будет ли текст элемента отображаться жирным шрифтом.


## Комментарии


Если свойству установлено значение True, то текст выбранного элемента будет выделен жирным шрифтом.
