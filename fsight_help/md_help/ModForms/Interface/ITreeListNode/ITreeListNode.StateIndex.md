# ITreeListNode.StateIndex


## Синтаксис


StateIndex: Integer;


## Описание


Свойство StateIndex определяет индекс, связывающий элемент с одним из рисунков свойства [StateImages](../ITreeControl/ITreeControl.StateImages.htm), который будет отражать состояние элемента.
