# ITreeListNode.ColumnText


## Синтаксис


ColumnText(ColumnIndex: Integer): String;


## Параметры


ColumnIndex. Индекс столбца.


## Описание


Свойство ColumnText определяет
 текст указанного столбца для текущего элемента.


## Комментарии


Устанавливается текст столбца, индекс которого передается посредством
 параметра ColumnIndex. Длина указываемого
 значения не должна превышать 5450 символов.
