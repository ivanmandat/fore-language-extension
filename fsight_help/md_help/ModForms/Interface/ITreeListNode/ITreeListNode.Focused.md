# ITreeListNode.Focused


## Синтаксис


Focused: Boolean;


## Описание


Свойство Focused определяет, находится ли элемент в фокусе.


## Комментарии


Если свойству установлено значение True, элемент находится в фокусе, если False - элемент не активен.
