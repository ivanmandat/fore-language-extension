# ITreeListNode.Checked


## Синтаксис


Checked: Boolean;


## Описание


Свойство Checked определяет,
 установлена ли отметка для элемента компонента.


## Комментарии


Если свойству установлено значение True,
 то элементу будет установлена отметка. При установке False,
 отметки не будет.
