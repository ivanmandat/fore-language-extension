# ITreeListNode.IsParent


## Синтаксис


IsParent(Node: [ITreeListNode](ITreeListNode.htm)): Boolean;


## Параметры


Node - вершина дерева, для которой необходимо проверить, является ли она родительской по отношению данной вершине.


## Описание


Метод IsParent позволяет определить является ли текущий элемент родительским по отношению к элементу, передаваемому в качестве параметра Node.
