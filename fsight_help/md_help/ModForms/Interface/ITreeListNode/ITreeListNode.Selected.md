# ITreeListNode.Selected


## Синтаксис


Selected: Boolean;


## Описание


Свойство Selected определяет, является ли элемент выбранным.


## Комментарии


Если свойству установлено значение True, элемент будет выбранным и будет выделяться синим цветом, в случае установки значения False - выделения не будет.
