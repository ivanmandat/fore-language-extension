# ITreeListNode.SelectNode


## Синтаксис


SelectNode([Recursively: Boolean = False]);


## Параметры


Recursively. Параметр, определяющий рекурсивность выбора элементов. Если параметр принимает значение True, происходит выделение всех элементов, при установке параметру значения False, выделен будет только выбранный элемент.


## Описание


Метод SelectNode осуществляет выделение элемента дерева.
