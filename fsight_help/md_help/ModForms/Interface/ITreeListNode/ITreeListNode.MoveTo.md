# ITreeListNode.MoveTo


## Синтаксис


MoveTo(Node: [ITreeListNode](ITreeListNode.htm);
 Mode: [NodeAttachMode](ForeSys.chm::/Enums/NodeAttachMode.htm));


## Параметры


Node. Элемент
 дерева, относительно которого будет располагаться данный элемент.


Mode. Позиция относительно элемента Node, в которую необходимо поместить данную
 вершину.


## Описание


Метод MoveTo осуществляет перемещение
 элемента дерева в указанную позицию.
