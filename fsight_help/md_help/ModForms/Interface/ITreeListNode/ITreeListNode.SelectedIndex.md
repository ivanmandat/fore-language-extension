# ITreeListNode.SelectedIndex


## Синтаксис


SelectedIndex: Integer;


## Описание


Свойство SelectedIndex определяет индекс, связывающий элемент с одним из рисунков свойства [Images](../ITreeControl/ITreeControl.Images.htm), который будет отображать выделенные элементы.
