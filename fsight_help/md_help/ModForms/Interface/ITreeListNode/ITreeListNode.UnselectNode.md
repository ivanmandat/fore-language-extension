# ITreeListNode.UnselectNode


## Синтаксис


UnselectNode([Recursively: Boolean = False]);


## Параметры


Recursively. Параметр, определяющий рекурсивность отмены выделения элементов. Если параметр принимает значение True, то отмена выделения осуществляется для всех элементов. Если установить параметру значения False, то произойдет отмена выделения только выбранного элемента.


## Описание


Метод UnselectNode осуществляет снятие выделения элемента дерева.
