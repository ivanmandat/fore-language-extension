# ITreeListNode.EditText


## Синтаксис


EditText: Boolean;


## Описание


Метод EditText осуществляет переход в режим редактирования текущего элемента и возвращает признак удачного перехода.


## Комментарии


Метод возвращает значение True, если переход в режим редактирования осуществлен удачно. Переход в режим редактирования возможен, если свойству [ReadOnly](../ITreeControl/ITreeControl.ReadOnly.htm) установлено значение False.
