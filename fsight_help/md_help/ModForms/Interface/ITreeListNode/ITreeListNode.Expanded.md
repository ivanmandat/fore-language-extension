# ITreeListNode.Expanded


## Синтаксис


Expanded: Boolean;


## Описание


Свойство Expanded определяет какой будет иерархия вершины дерева - развернутой или свернутой.


## Комментарии


Если свойству установлено значение True, иерархия будет развернутой, иначе свернутой.
