# ITreeListNode.EndEdit


## Синтаксис


EndEdit(Cancel: Boolean);


## Параметры


Cancel - входной логический параметр, определяющий будут ли отменяться изменения, произведенные пользователем в тексте компонента. Если параметр принимает значение True, изменения текста элемента пользователем игнорируются, при установке параметру значения False, изменения текста элемента пользователем сохраняются.


## Описание


Метод EndEdit позволяет выйти из режима редактирования текста элементов дерева.
