# ITreeListNode.Sort


## Синтаксис


Sort([Column: [ITreeListColumn](../ITreeListColumn/ITreeListColumn.htm) = Null]);


## Параметры


Column - необязательный параметр, определяющий столбец, для которого необходимо выполнить сортировку. Если параметр равен Null, то сортировка осуществится по первому столбцу.


## Описание


Метод Sort выполняет сортировку по содержанию указанного столбца. Столбец, по которому осуществляется сортировка, передается посредством параметра Column.
