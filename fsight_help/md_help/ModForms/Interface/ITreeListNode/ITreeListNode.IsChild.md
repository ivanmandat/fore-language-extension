# ITreeListNode.IsChild


## Синтаксис


IsChild(Node: [ITreeListNode](ITreeListNode.htm); [Immediate: Boolean = False]): Boolean;


## Параметры


Node — входной параметр, определяющий вершину дерева, принадлежность к дочерним элементам которого, будет проверяться.


Immediate — параметр, определяющий какие элементы будут считаться дочерними. Если параметру установлено значение True, дочерними считаются только элементы непосредственно следующего уровня. При установке параметра в False дочерними будут считаться элементы всех уровней ниже текущего.


## Описание


Метод IsChild позволяет определить является ли текущий элемент дочерним по отношению к элементу, передаваемому в качестве параметра Node.
