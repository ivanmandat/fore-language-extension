# ITreeListNode.UnselectSiblings


## Синтаксис


UnselectSiblings([Through: Boolean = False]);


## Параметры


Through - необязательный параметр, определяющий область отмены выделения элементов одного уровня. По умолчанию установлено значение False, при этом осуществляется снятие выделение элементов уровня в рамках общего родительского элемента. При установке значения True осуществляется снятие выделения у всех элементов, расположенных на одном уровне с текущим элементом.


## Описание


Метод UnselectSiblings осуществляет снятие отметки элементов уровня, на котором расположен данный элемент.
