# ITreeListNode.SelectSiblings


## Синтаксис


SelectSiblings([Through: Boolean = False]);


## Параметры


Through - необязательный параметр, определяющий область выделения элементов одного уровня. По умолчанию установлено значение False, при этом осуществляется выделение элементов уровня в рамках общего родительского элемента. При установке значения True осуществляется выделение всех элементов, расположенных на одном уровне с текущим элементом.


## Описание


Метод SelectSiblings осуществляет выделение элементов уровня, на котором расположен данный элемент.
