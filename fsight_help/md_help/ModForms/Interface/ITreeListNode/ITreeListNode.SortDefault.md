# ITreeListNode.SortDefault


## Синтаксис


SortDefault;


## Описание


Метод SortDefault осуществляет сортировку по содержанию столбца, указанного в свойстве [SortColumn](../ITreeControl/ITreeControl.SortColumn.htm). При вызове метода несколько раз направление сортировки изменяется.
