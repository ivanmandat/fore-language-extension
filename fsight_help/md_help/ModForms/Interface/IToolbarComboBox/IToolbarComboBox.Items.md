# IToolbarComboBox.Items


## Синтаксис


Items: [IStringList](ModCollections.chm::/Interface/IStringList/IStringList.htm);


## Описание


Свойство Items определяет коллекцию элементов комбинированного списка.


## Комментарии


Список строк задается через язык Fore или в Редакторе строк. Окно «Редактор строк» вызывается при нажатии на кнопки ![](../knopka_dialog.gif) свойства Items в [Инспекторе Объектов](UiDevEnv.chm::/01_Development_Environment/03_Windows_of_Development_Environment/Object_Inspector.htm).
