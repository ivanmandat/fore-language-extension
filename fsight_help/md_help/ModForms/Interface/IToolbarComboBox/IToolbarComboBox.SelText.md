# IToolbarComboBox.SelText


## Синтаксис


SelText: String;


## Описание


Свойство SelText определяет выделенный в компоненте текст.


## Комментарии


Используется вместе со свойствами [IToolbarComboBox.SelLength](IToolbarComboBox.SelLength.htm) и [IToolbarComboBox.SelStart](IToolbarComboBox.SelStart.htm). Свойство доступно только через язык Fore.
