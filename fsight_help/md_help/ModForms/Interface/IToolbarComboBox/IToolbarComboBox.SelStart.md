# IToolbarComboBox.SelStart


## Синтаксис


SelStart: Integer;


## Описание


Свойство SelStart определяет позицию первого выделенного символа в тексте комбинированного списка. Если выделения нет - определяет позицию курсора.


## Комментарии


Используется вместе со свойствами [IToolbarComboBox.SelLength](IToolbarComboBox.SelLength.htm) и [IToolbarComboBox.SelText](IToolbarComboBox.SelText.htm). Свойство доступно только через язык Fore.
