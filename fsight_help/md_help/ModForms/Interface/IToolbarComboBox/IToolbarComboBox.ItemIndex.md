# IToolbarComboBox.ItemIndex


## Синтаксис


ItemIndex: Integer;


## Описание


Свойство ItemIndex определяет индекс элемента в списке, который выбран и отображается в строке редактирования списка. Если ни один элемент не выбран - возвращает -1.
