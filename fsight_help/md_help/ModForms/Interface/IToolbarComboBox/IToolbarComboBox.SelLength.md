# IToolbarComboBox.SelLength


## Синтаксис


SelLength: Integer;


## Описание


Свойство SelLength определяет количество выделенных символов в строке комбинированного списка.


## Комментарии


Используется вместе со свойствами [IToolbarComboBox.SelStart](IToolbarComboBox.SelStart.htm) и [IToolbarComboBox.SelText](IToolbarComboBox.SelText.htm). Свойство доступно только через язык Fore.
