# IToolbarComboBox.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет доступность компонента для пользователя.


## Комментарии


Если свойству установлено значение False, комбинированный список будет недоступен. Недоступный компонент отображается серым цветом, игнорирует события, связанные с мышью, клавиатурой и событием таймера.
