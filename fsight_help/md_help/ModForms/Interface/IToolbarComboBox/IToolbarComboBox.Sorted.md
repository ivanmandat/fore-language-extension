# IToolbarComboBox.Sorted


## Синтаксис


Sorted: Boolean;


## Описание


Свойство Sorted определяет, будут ли элементы комбинированного списка автоматически сортироваться в алфавитном порядке.


## Комментарии


Если свойству установлено значение True, то происходит автоматическая сортировка, иначе элементы отображаются в том порядке, в котором они были добавлены в список.
