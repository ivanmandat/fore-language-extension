# IToolbarComboBox


## Описание


Интерфейс IToolbarComboBox содержит
 свойства и методы комбинированного списка, расположенного на панели инструментов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BeginGroup](IToolbarComboBox.BeginGroup.htm)


		 Свойство BeginGroup
		 позволяет разделить группы компонентов, расположенных на панели
		 инструментов, визуально вертикальной чертой.


		 ![](../../Property_Image.gif)
		 [DropDownCount](IToolbarComboBox.DropDownCount.htm)


		 Свойство DropDownCount
		 определяет максимальное количество одновременно отображаемых элементов
		 в открытом списке компонента.


		 ![](../../Property_Image.gif)
		 [DroppedDown](IToolbarComboBox.DroppedDown.htm)


		 Свойство DroppedDown
		 возвращает признак, раскрыт ли комбинированный список.


		 ![](../../Property_Image.gif)
		 [Enabled](IToolbarComboBox.Enabled.htm)


		 Свойство Enabled определяет
		 доступность компонента для пользователя.


		 ![](../../Property_Image.gif)
		 [Hint](IToolbarComboBox.Hint.htm)


		 Свойство Hint определяет
		 текст всплывающей подсказки, появляющийся при наведении указателем
		 мыши на список в панели инструментов, во время выполнения формы.


		 ![](../../Property_Image.gif)
		 [ItemCount](IToolbarComboBox.ItemCount.htm)


		 Свойство ItemCount
		 возвращает количество элементов в комбинированном списке.


		 ![](../../Property_Image.gif)
		 [ItemHeight](IToolbarComboBox.ItemHeight.htm)


		 Свойство ItemHeight
		 возвращает высоту элементов в комбинированном списке.


		 ![](../../Property_Image.gif)
		 [ItemIndex](IToolbarComboBox.ItemIndex.htm)


		 Свойство ItemIndex
		 определяет индекс элемента в списке, который выбран и отображается
		 в строке редактирования списка.


		 ![](../../Property_Image.gif)
		 [Items](IToolbarComboBox.Items.htm)


		 Свойство Items определяет
		 коллекцию элементов комбинированного списка.


		 ![](../../Property_Image.gif)
		 [MaxLength](IToolbarComboBox.MaxLength.htm)


		 Свойство MaxLength
		 определяет максимальную длину, вводимого в компонент пользователем,
		 текста.


		 ![](../../Property_Image.gif)
		 [SelLength](IToolbarComboBox.SelLength.htm)


		 Свойство SelLength
		 определяет количество выделенных символов в строке комбинированного
		 списка.


		 ![](../../Property_Image.gif)
		 [SelStart](IToolbarComboBox.SelStart.htm)


		 Свойство SelStart определяет
		 позицию первого выделенного символа в тексте комбинированного
		 списка.


		 ![](../../Property_Image.gif)
		 [SelText](IToolbarComboBox.SelText.htm)


		 Свойство SelText определяет
		 выделенный в компоненте текст.


		 ![](../../Property_Image.gif)
		 [Sorted](IToolbarComboBox.Sorted.htm)


		 Свойство Sorted определяет,
		 будут ли элементы комбинированного списка автоматически сортироваться
		 в алфавитном порядке.


		 ![](../../Property_Image.gif)
		 [Style](IToolbarComboBox.Style.htm)


		 Свойство Style определяет
		 стиль работы комбинированного списка.


		 ![](../../Property_Image.gif)
		 [Text](IToolbarComboBox.Text.htm)


		 Свойство Text определяет
		 строку, которая отображается в окне редактирования до того момента,
		 когда произойдет выбор элемента из раскрывающегося списка.


		 ![](../../Property_Image.gif)
		 [Toolbar](IToolbarComboBox.Toolbar.htm)


		 Свойство Toolbar возвращает
		 родительский компонент [ToolBar](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ToolBar.htm).


		 ![](../../Property_Image.gif)
		 [Visible](IToolbarComboBox.Visible.htm)


		 Свойство Visible определяет
		 видимость комбинированного списка во время выполнения формы.


		 ![](../../Property_Image.gif)
		 [Width](IToolbarComboBox.Width.htm)


		 Свойство Width определяет
		 ширину компонента в пикселях.


## Свойства, унаследованные от [IComponent](../IComponent/IComponent.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ComponentCount](../IComponent/IComponent.ComponentCount.htm)
		 Свойство ComponentCount
		 возвращает количество дочерних компонентов, расположенных на родительском
		 компоненте.


		 ![](../../Property_Image.gif)
		 [Components](../IComponent/IComponent.Components.htm)
		 Свойство Components
		 возвращает дочерний компонент, индекс которого передается посредством
		 входного параметра.


		 ![](../../Property_Image.gif)
		 [Data](../IComponent/IComponent.Data.htm)
		 Свойство Data предназначено
		 для хранения любых пользовательских данных.


		 ![](../../Property_Image.gif)
		 [Name](../IComponent/IComponent.Name.htm)
		 Свойство Name определяет
		 наименование компонента.


		 ![](../../Property_Image.gif)
		 [Tag](../IComponent/IComponent.Tag.htm)
		 Свойство Tag не используется
		 компилятором. Пользователь может изменить значение данного свойства
		 и использовать его по своему усмотрению.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [ClearEdit](IToolbarComboBox.ClearEdit.htm)


		 Метод ClearEdit очищает
		 строку редактирования комбинированного списка.


		 ![](../../Sub_Image.gif)
		 [SelectAll](IToolbarComboBox.SelectAll.htm)


		 Метод SelectAll осуществляет
		 выделение текст в строке редактирования.
