# IControlBar.Float


## Синтаксис


Float;


## Описание


Метод Float переводит панель в плавающее состояние.


## Комментарии


Актуально, если свойству [AllowFloat](IControlBar.AllowFloat.htm) установлено значение True. Если [AllowFloat](IControlBar.AllowFloat.htm)=False, то панель переводится в припаркованное состояние.
