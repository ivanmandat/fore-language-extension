# IControlBar.Id


## Синтаксис


Id: Integer;


## Описание


Свойство Id определяет уникальный идентификатор всплывающей панели.


## Комментарии


Данный идентификатор используется при сохранении настроек, учитывающих расположение всплывающих окон.
