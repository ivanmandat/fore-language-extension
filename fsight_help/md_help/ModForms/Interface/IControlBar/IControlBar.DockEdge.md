# IControlBar.DockEdge


## Синтаксис


DockEdge: [IDockSite](../IDockSite/IDockSite.htm);


## Описание


Свойство DockEdge возвращает настройки, определяющие стороны родительского окна, к которым будет возможность прикрепить всплывающую панель во время выполнения формы.
