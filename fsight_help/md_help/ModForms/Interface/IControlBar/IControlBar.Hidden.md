# IControlBar.Hidden


## Синтаксис


Hidden: Boolean;


## Описание


Свойство Hidden определяет состояние, в котором находится панель.


## Комментарии


Если свойству установлено значение True – панель находится в скрытом состоянии, иначе – в припаркованном.
