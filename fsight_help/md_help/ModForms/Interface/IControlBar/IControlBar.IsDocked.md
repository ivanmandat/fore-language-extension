# IControlBar.IsDocked


## Синтаксис


IsDocked(Bar: [IControlBar](IControlBar.htm)): Boolean;


## Параметры


Bar - всплывающая панель, на группировку с которой необходимо проверить данную панель.


## Описание


Метод IsDocked осуществляет проверку, находится ли данная всплывающая панель в одной группе с панелью, передаваемой посредством параметра Bar.


## Комментарии


Если панели находятся в одной группе, то метод возвращает значение True, иначе False. Группировка панелей осуществляется с помощью метода [DockIn](IControlBar.DockIn.htm).
