# IActionItem.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет признак видимости для данного действия. Данное свойство управляет видимостью всех пунктов меню и кнопок панелей инструментов, для которых подключено данное действие.
