# IActionItem.Hint


## Синтаксис


Hint: String;


## Описание


Свойство Hint определяет текст всплывающей подсказки, связанной с данным действием. Данная подсказка будет отображаться при наведении курсора мыши на все объекты, для которых подключено данное действие.
