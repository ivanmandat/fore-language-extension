# IActionItem.Update


## Синтаксис


Update;


## Описание


Метод Update осуществляет обновление состояния действия.


## Комментарии


При вызове данного метода генерируется событие [OnUpdate](../../Class/ActionItem/ActionItem.OnUpdate.htm), а также событие [OnUpdate](../../Class/ActionList/ActionList.OnUpdate.htm) родительского компонента. В данных событиях можно осуществлять дополнительную подготовку к выполнению дальнейших действий.
