# IActionItem.AutoCheck


## Синтаксис


AutoCheck: Boolean;


## Описание


Свойство AutoCheck определяет,
 будет ли производиться автоматическое изменение состояния флажка при выполнении
 действия.


## Комментарии


Допустимые значения:


	- True. Состояние флажка
	 меняется автоматически.


	- False. Значение по умолчанию.
	 Значение флажка меняется при изменении значения свойства [Checked](IActionItem.Checked.htm).
