# IActionItem.ImageIndex


## Синтаксис


ImageIndex: Integer;


## Описание


Свойство ImageIndex определяет индекс изображения, соответствующего данному действию.


## Комментарии


По умолчанию свойству установлено значение «-1», при этом изображение отсутствует. Источником изображений является компонент [ImageList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ImageList.htm) или [GlobalImageList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/GlobalImageList.htm), имя которого указывается в качестве значения свойства [Images](../IActionList/IActionList.Images.htm) родительского компонент [ActionList](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/ActionList.htm).
