# IActionItem.ShortCut


## Синтаксис


ShortCut: Integer;


## Описание


Свойство ShortCut определяет комбинацию «горячих клавиш», используемых для быстрого вызова данного действия.


## Комментарии


При создании «горячих клавиш» во время проектирования формы следует выбрать в инспекторе объектов свойство ShortCut и нажать необходимое сочетание клавиш.
