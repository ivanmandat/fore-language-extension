# IRibbonMainButton.Image


## Синтаксис


Image: [IGxBitmap](ModDrawing.chm::/Interface/IGxBitmap/IGxBitmap.htm);


## Описание


Свойство Image определяет фоновое
 изображение, которое будет отображено в области кнопки приложения.


## Комментарии


Максимальный размер изображения, которое может быть отображено в области
 кнопки приложения, равен 45*45 пикселей. При установке изображения, размеры
 которого больше, оно будет автоматически промасштабировано и уменьшено
 до указанных размеров.


Примечание.
 Данное свойство актуально только для АК «Прогноз-5». Стиль оформления
 различных версий Prognoz Platform и продукта «Форсайт. Аналитическая платформа»
 не допускает использования изображений в области кнопки приложения.
