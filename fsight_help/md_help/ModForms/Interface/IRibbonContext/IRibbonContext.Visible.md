# IRibbonContext.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет признак отображения контекстной вкладки.


## Комментарии


По умолчанию все контекстные вкладки скрыты. При наступлении определенных событий используя данное свойство можно отобразить контекстную вкладку и все вкладки, которые она в себя включает. Для отображения контекстной вкладки необходимо данному свойству установить значение True, для скрытия - False.
