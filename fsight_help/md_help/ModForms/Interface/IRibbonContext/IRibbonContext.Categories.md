# IRibbonContext.Categories


## Синтаксис


Categories: [IRibbonCategories](../IRibbonCategories/IRibbonCategories.htm);


## Описание


Свойство Categories возвращает коллекцию дочерних вкладок, формирующих содержимое контекстной вкладки.


## Комментарии


При создании новой контекстной вкладки в коллекции Categories автоматически создается одна вкладка.
