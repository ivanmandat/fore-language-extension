# ICalendarColors.TralingTextColor


## Синтаксис


TralingTextColor: [IGxColor](ModDrawing.chm::/Interface/IGxColor/IGxColor.htm);


## Описание


Свойство TralingTextColor определяет цвет чисел предыдущего и следующего месяца.


## Комментарии


Изменение цветового оформления календаря не доступно в операционной системе Windows 7 при использовании темы оформления Aero и в более поздних операционных системах.
