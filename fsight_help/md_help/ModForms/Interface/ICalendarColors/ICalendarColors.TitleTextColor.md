# ICalendarColors.TitleTextColor


## Синтаксис


TitleTextColor: [IGxColor](ModDrawing.chm::/Interface/IGxColor/IGxColor.htm);


## Описание


Свойство TitleTextColor определяет цвет надписи в заголовке календаря.


## Комментарии


Изменение цветового оформления календаря не доступно в операционной системе Windows 7 при использовании темы оформления Aero и в более поздних операционных системах.
