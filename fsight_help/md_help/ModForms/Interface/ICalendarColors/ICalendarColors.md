# ICalendarColors


## Описание


Интерфейс ICalendarColors содержит
 свойства для настройки цветового оформления календаря.


## Комментарии


Изменение цветового оформления календаря не доступно в операционной
 системе Windows 7 при использовании темы оформления Aero и в более поздних
 операционных системах.


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BackgroundColor](ICalendarColors.BackgroundColor.htm)
		 Свойство BackgroundColor
		 определяет цвет фона всего календаря.


		 ![](../../Property_Image.gif)
		 [MonthBackgroundColor](ICalendarColors.MonthBackgroundColor.htm)
		 Свойство MonthBackgroundColor
		 определяет цвет фона, на котором располагаются названия месяцев.


		 ![](../../Property_Image.gif)
		 [TextColor](ICalendarColors.TextColor.htm)
		 Свойство TextColor
		 определяет цвет чисел текущего месяца.


		 ![](../../Property_Image.gif)
		 [TitleBackgroundColor](ICalendarColors.TitleBackgroundColor.htm)
		 Свойство TitleBackgroundColor
		 определяет цвет фона заголовка календаря.


		 ![](../../Property_Image.gif)
		 [TitleTextColor](ICalendarColors.TitleTextColor.htm)
		 Свойство TitleTextColor
		 определяет цвет надписи в заголовке календаря.


		 ![](../../Property_Image.gif)
		 [TralingTextColor](ICalendarColors.TralingTextColor.htm)
		 Свойство TralingTextColor
		 определяет цвет чисел предыдущего и следующего месяца.
