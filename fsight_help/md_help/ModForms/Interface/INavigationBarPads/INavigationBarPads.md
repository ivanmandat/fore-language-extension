# INavigationBarPads


Сборка: Forms;


## Описание


Интерфейс INavigationBarPads
 содержит свойства и методы для работы с коллекцией вкладок навигационной
 панели.


## Иерархия наследования


INavigationBarPads


## Комментарии


Коллекция вкладок навигационной панели может быть получена в свойстве
 [INavigationBar.Pads](../INavigationBar/INavigationBar.Pads.htm).


## Свойства, унаследованные от [INavigationBarPadList](../INavigationBarPadList/INavigationBarPadList.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](../INavigationBarPadList/INavigationBarPadList.Count.htm)
		 Свойство Count возвращает
		 количество вкладок в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](../INavigationBarPadList/INavigationBarPadList.Item.htm)
		 Свойство Item возвращает
		 параметры вкладки с указанным индексом.


		 ![](../../Property_Image.gif)
		 [ItemById](../INavigationBarPadList/INavigationBarPadList.ItemById.htm)
		 Свойство ItemById возвращает
		 параметры вкладки с указанным идентификатором.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](INavigationBarPads.Add.htm)


		 Метод Add добавляет
		 указанную вкладку в коллекцию вкладок текущей навигационной панели.


		 ![](../../Sub_Image.gif)
		 [Clear](INavigationBarPads.Clear.htm)


		 Метод Clear удаляет
		 все вкладки из коллекции.


		 ![](../../Sub_Image.gif)
		 [Insert](INavigationBarPads.Insert.htm)


		 Метод Insert осуществляет
		 вставку указанной вкладки в указанную позицию в коллекции.


		 ![](../../Sub_Image.gif)
		 [PadIndex](INavigationBarPads.PadIndex.htm)


		 Метод PadIndex возвращает
		 индекс вкладки в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](INavigationBarPads.Remove.htm)


		 Метод Remove удаляет
		 указанную вкладку из коллекции.
