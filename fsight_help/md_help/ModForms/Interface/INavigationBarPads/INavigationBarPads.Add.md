# INavigationBarPads.Add


## Синтаксис


Add(Pad: [INavigationBarPad](../INavigationBarPad/INavigationBarPad.htm));


## Параметры


Pad. Вкладка, которую необходимо добавить в коллекцию.


## Описание


Метод Add добавляет указанную вкладку в коллекцию вкладок текущей навигационной панели.


## Комментарии


Вкладка добавляется в конец коллекции. Заголовок вкладки визуально будет отображен ниже всех остальных заголовков.
