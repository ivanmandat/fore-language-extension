# INavigationBarPads.PadIndex


## Синтаксис


PadIndex(Pad: [INavigationBarPad](../INavigationBarPad/INavigationBarPad.htm)): Integer;


## Параметры


Pad. Вкладка, для которой необходимо получить индекс в коллекции.


## Описание


Метод PadIndex возвращает индекс вкладки в коллекции.
