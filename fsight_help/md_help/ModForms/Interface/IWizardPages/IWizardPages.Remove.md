# IWizardPages.Remove


## Синтаксис


Remove(Page: [IWizardPage](../IWizardPage/IWizardPage.htm));


## Параметры


Page - параметр, определяющий страницу, которая будет удалена из коллекции страниц мастера.


## Описание


Метод Remove удаляет страницу из коллекции компонента [Wizard](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/Wizard.htm), которая передается в качестве входного параметра Page.
