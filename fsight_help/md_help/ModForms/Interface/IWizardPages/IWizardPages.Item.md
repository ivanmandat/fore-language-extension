# IWizardPages.Item


## Синтаксис


Item(Index: Integer): [IWizardPage](../IWizardPage/IWizardPage.htm);


## Параметры


Index - индекс страницы, которую необходмо получить.


## Описание


Свойство Item возвращает страницу мастера, индекс которой передается в качестве входного параметра Index.
