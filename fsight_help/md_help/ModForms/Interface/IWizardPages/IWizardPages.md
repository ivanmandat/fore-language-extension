# IWizardPages


## Описание


Интерфейс IWizardPages содержит
 свойства и методы коллекции страниц компонента [Wizard](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/Wizard.htm).


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IWizardPages.Count.htm)
		 Свойство Count возвращает
		 количество страниц.


		 ![](../../Property_Image.gif)
		 [Item](IWizardPages.Item.htm)
		 Свойство Item возвращает
		 страницу мастера.


## Методы


		  Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IWizardPages.Add.htm)
		 Метод Add осуществляет
		 добавление страницы в коллекцию страниц мастера.


		 ![](../../Sub_Image.gif)
		 [Clear](IWizardPages.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех страниц мастера.


		 ![](../../Sub_Image.gif)
		 [Insert](IWizardPages.Insert.htm)
		 Метод Insert позволяет
		 вставить страницу мастера в указанную позицию.


		 ![](../../Sub_Image.gif)
		 [Remove](IWizardPages.Remove.htm)
		 Метод Remove удаляет
		 страницу из коллекции.
