# ISearchEdit.Search


## Синтаксис


Search(Text: String);


## Параметры


Text.
 Текст, который будет искаться в индексированных данных в процессе поиска.


## Описание


Метод Search запускает процесс
 поиска.


## Комментарии


После выполнения метода будет осуществлен поиск указанного текста. Если
 поиск занимает продолжительное время, то при превышении значения, указанного
 в свойстве [Timeout](ISearchEdit.Timeout.htm) для компонента
 будет сгенерировано событие [OnSearch](../../Class/SearchEdit/SearchEdit.OnSearch.htm).
