# ISearchEdit.Timeout


## Синтаксис


Timeout: Integer;


## Описание


Свойство Timeout определяет
 допустимое время поиска.


## Комментарии


Время задается в миллисекундах. Если процесс поиска превысил указанное
 значение, то для компонента будет сгенерировано событие [OnSearch](../../Class/SearchEdit/SearchEdit.OnSearch.htm).
 Данное событие может быть использовано для информирования пользователя
 о продолжительности процесса поиска.
