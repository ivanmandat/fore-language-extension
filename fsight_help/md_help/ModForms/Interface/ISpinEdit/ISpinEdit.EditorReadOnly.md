# ISpinEdit.EditorReadOnly


## Синтаксис


EditorReadOnly: Boolean;


## Описание


Свойство EditorReadOnly определяет признак возможности редактирования текста компонента вручную.


## Комментарии


По умолчанию свойству установлено значение False, при этом доступно изменение текста компонента вручную. При установке свойству значения True изменение текста компонента будет доступно только программным способом.
