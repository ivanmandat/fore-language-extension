# INavigationBarPadList


Сборка: Forms;


## Описание


Интерфейс INavigationBarPadList
 содержит свойства и методы для работы с коллекцией вкладок навигационной
 панели.


## Иерархия наследования


INavigationBarPadList


## Комментарии


Коллекцию вкладок возвращает свойство [IWidget.Pads](../IWidget/IWidget.Pads.htm).
