# IBeginDragEventArgs


## Описание


Интерфейс IBeginDragEventArgs
 содержит свойства аргумента события, возникающего во время начала перетаскивания
 какого-либо объекта на форме.


## Комментарии


Обработка начала перетаскивания объекта осуществляется в событии [OnBeginDrag](../IControl/IControl.OnBeginDrag.htm).
