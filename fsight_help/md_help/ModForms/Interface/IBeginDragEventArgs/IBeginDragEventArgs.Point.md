# IBeginDragEventArgs.Point


## Синтаксис


Point: [IPoint](../IPoint/IPoint.htm);


## Описание


Свойство Point возвращает координаты
 курсора мыши относительно компонента, в области которого инициализируется
 перетаскивание.


## Комментарии


Свойство возвращает координаты курсора относительно верхнего левого
 угла компонента.
