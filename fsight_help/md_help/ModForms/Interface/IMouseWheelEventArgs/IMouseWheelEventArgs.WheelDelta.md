# IMouseWheelEventArgs.WheelDelta


## Синтаксис


WheelDelta: Integer;


## Описание


Свойство WheelDelta возвращает изменение положения колесика мыши.


-
> 0 - колесо прокручено вперед.


-
< 0 - колесо прокручено назад.
