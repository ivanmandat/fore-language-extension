# IRibbonDateTimePicker.Mode


## Синтаксис


Mode: [DateTimePickerExMode](../../Enums/DateTimePickerExMode.htm);


## Описание


Свойство Mode определяет режим
 установки даты в календаре.


## Комментарии


При изменении режима установки даты изменяется внешний вид компонента.
 Установка даты будет осуществляться путем выбора одного из периодов, соответствующих
 режиму. Значение свойства [CurrentDate](IRibbonDateTimePicker.CurrentDate.htm)
 будет соответствовать началу выбранного периода.
