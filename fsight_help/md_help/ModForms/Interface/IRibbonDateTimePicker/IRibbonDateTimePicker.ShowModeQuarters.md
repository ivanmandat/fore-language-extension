# IRibbonDateTimePicker.ShowModeQuarters


## Синтаксис


ShowModeQuarters: Boolean;


## Описание


Свойство ShowModeQuarters
 определяет, будет ли в области компонента отображаться кнопка для включения
 режима установки даты путем выбора квартала в году.


## Комментарии


Допустимые значения:


	- True. Кнопка
	 отображается;


	- False.
	 Кнопка не отображается. Установка режима выбора дня в месяце доступна
	 через свойство [Mode](IRibbonDateTimePicker.Mode.htm).
