# IRibbonDateTimePicker.IsValidDate


## Синтаксис


IsValidDate(Value: DateTime): Boolean;


## Параметры


Value. Значение даты и времени,
 которое необходимо проверить на возможность установки в качестве значения
 для элемента управления.


## Описание


Свойство IsValidDate возвращает
 признак возможности установить значение, передаваемое посредством входного
 параметра.


## Комментарии


Свойство возвращает значение True
 для дат в диапазоне 01.01.1601-31.12.9999, и False
 для всех остальных дат.
