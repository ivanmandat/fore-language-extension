# IRibbonDateTimePicker.DropCalendar


## Синтаксис


DropCalendar: Boolean;


## Описание


Свойство DropCalendar определяет
 возможность вызова раскрывающегося календаря.


## Комментарии


Если свойству установлено значение True,
 то в элементе управления будет присутствовать кнопка, при нажатии на которую
 будет появляться раскрывающийся календарь, используемый для выбора даты.
