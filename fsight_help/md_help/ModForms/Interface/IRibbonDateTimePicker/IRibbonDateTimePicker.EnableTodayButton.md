# IRibbonDateTimePicker.EnableTodayButton


## Синтаксис


EnableTodayButton: Boolean;


## Описание


Свойство EnableTodayButton определяет
 признак отображения в раскрывающемся календаре кнопки для установки текущей
 даты и времени.


## Комментарии


По умолчанию свойству установлено значение True,
 при этом в раскрывающемся календаре будет присутствовать кнопка «Сегодня»
 позволяющая выбрать текущую дату. При установке свойству значения False кнопка будет скрыта.
