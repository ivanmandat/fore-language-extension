# IRibbonDateTimePicker.ShowDate


## Синтаксис


ShowDate: Boolean;


## Описание


Свойство ShowDate определяет
 признак отображения даты в редакторе.


## Комментарии


Если свойству установлено значение True,
 то в компоненте будет отображаться дата.
