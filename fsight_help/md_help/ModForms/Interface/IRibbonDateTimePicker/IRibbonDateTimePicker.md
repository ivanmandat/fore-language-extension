# IRibbonDateTimePicker


Сборка: Forms;


## Описание


Интерфейс IRibbonDateTimePicker
 содержит свойства элемента управления ленты - Редактор даты и времени.


## Иерархия наследования


           [IComponent](../IComponent/IComponent.htm)


           [IBaseRibbonElement](../IBaseRibbonElement/IBaseRibbonElement.htm)


           IRibbonDateTimePicker

		 ![](../../Property_Image.gif)
		 [Alignment](IRibbonDateTimePicker.Alignment.htm)
		 Свойство Alignment
		 определяет способ выравнивания раскрывающегося календаря относительно

		 [Checked](IRibbonDateTimePicker.Checked.htm)

		 определяет возможность вызова раскрывающегося календаря.

		 [DroppedDown](IRibbonDateTimePicker.DroppedDown.htm)

		 ![](../../Property_Image.gif)

		 [IsValidDate](IRibbonDateTimePicker.IsValidDate.htm)

		 [MinDate](IRibbonDateTimePicker.MinDate.htm)

		 управления.


		 ![](../../Property_Image.gif)
		 [Mode](IRibbonDateTimePicker.Mode.htm)

		 ![](../../Property_Image.gif)

		 признак отображения даты в редакторе.

		 [ShowModeDays](IRibbonDateTimePicker.ShowModeDays.htm)
		 Свойство ShowModeDays
		 определяет, будет ли в области компонента отображаться кнопка

		 ![](../../Property_Image.gif)
		 [ShowModeMonths](IRibbonDateTimePicker.ShowModeMonths.htm)
		 Свойство ShowModeMonths
		 определяет, будет ли в области компонента отображаться кнопка
		 для включения режима установки даты путем выбора месяца в году.


		 ![](../../Property_Image.gif)
		 [ShowModeQuarters](IRibbonDateTimePicker.ShowModeQuarters.htm)

		 для включения режима установки даты путем выбора квартала в году.


		 ![](../../Property_Image.gif)
		 [ShowModeWeeks](IRibbonDateTimePicker.ShowModeWeeks.htm)
		 Свойство ShowModeWeeks
		 определяет, будет ли в области компонента отображаться кнопка

		 ![](../../Property_Image.gif)
		 [ShowModeYears](IRibbonDateTimePicker.ShowModeYears.htm)
		 Свойство ShowModeYears
		 определяет, будет ли в области компонента отображаться кнопка

		 ![](../../Property_Image.gif)
		 [ShowSpinButton](IRibbonDateTimePicker.ShowSpinButton.htm)
		 Свойство ShowSpinButton

		 [ShowTime](IRibbonDateTimePicker.ShowTime.htm)
		 Свойство ShowTime определяет
