# IRibbonDateTimePicker.ShowSpinButton


## Синтаксис


ShowSpinButton: Boolean;


## Описание


Свойство ShowSpinButton определяет
 признак отображения кнопок ВВЕРХ и ВНИЗ, используемых для изменения значений
 отдельных компонент даты и времени.


## Комментарии


Допустимые значения:


	- True. Кнопки отображаются;


	- False. Кнопки не отображаются.
