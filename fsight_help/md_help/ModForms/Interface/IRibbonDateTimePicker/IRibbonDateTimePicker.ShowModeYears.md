# IRibbonDateTimePicker.ShowModeYears


## Синтаксис


ShowModeYears: Boolean;


## Описание


Свойство ShowModeYears
 определяет, будет ли в области компонента отображаться кнопка для включения
 режима установки даты путем выбора года.


## Комментарии


Допустимые значения:


	- True. Кнопка
	 отображается;


	- False.
	 Кнопка не отображается. Установка режима выбора года доступна через
	 свойство [Mode](IRibbonDateTimePicker.Mode.htm).
