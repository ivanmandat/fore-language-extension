# IRibbonDateTimePicker.Checked


## Синтаксис


Checked: Boolean;


## Описание


Свойство Checked определяет
 состояние флажка, отвечающего за доступ к редактору даты и времени.


## Комментарии


Допустимые значения:


	- True. Флажок включен,
	 доступно редактирование даты и времени;


	- False. Флажок выключен,
	 редактирование даты и времени запрещено.


Свойство актуально, если свойству [ShowCheckbox](IRibbonDateTimePicker.ShowCheckbox.htm)
 установлено значение True.
