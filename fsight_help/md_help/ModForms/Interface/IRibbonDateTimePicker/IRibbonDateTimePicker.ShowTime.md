# IRibbonDateTimePicker.ShowTime


## Синтаксис


ShowTime: Boolean;


## Описание


Свойство ShowTime определяет
 признак отображения времени в редакторе.


## Комментарии


Если свойству установлено значение True,
 то в компоненте будет отображаться время.
