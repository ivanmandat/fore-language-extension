# IRibbonDateTimePicker.DroppedDown


## Синтаксис


DroppedDown: Boolean;


## Описание


Свойство DroppedDown возвращает
 признак отображения раскрывающегося календаря.


## Комментарии


Свойство возвращает значение True,
 если календарь раскрыт, и False,
 если календарь свернут и дата отображается в поле редактирования элемента
 управления.
