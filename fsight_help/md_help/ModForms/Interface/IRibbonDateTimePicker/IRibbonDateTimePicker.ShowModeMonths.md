# IRibbonDateTimePicker.ShowModeMonths


## Синтаксис


ShowModeMonths: Boolean;


## Описание


Свойство ShowModeMonths
 определяет, будет ли в области компонента отображаться кнопка для включения
 режима установки даты путем выбора месяца в году.


## Комментарии


Допустимые значения:


	- True. Кнопка
	 отображается;


	- False.
	 Кнопка не отображается. Установка режима выбора дня в месяце доступна
	 через свойство [Mode](IRibbonDateTimePicker.Mode.htm).
