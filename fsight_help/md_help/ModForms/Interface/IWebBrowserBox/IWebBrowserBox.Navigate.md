# IWebBrowserBox.Navigate


## Синтаксис


Navigate(Value: String);


## Параметры


Value - URL-адрес загружаемой страницы.


## Описание


Метод Navigate осуществляет загрузку в [WebBrowserBox](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/WebBrowserBox.htm) Web-страницу, адрес которой передается в качестве параметра Value.
