# IListViewEditTextEventArgs.Text


## Синтаксис


Text: String;


## Описание


Свойство Text определяет текст для элемента.


## Комментарии


Данное свойство используется в следующих случаях:


-
Определение текста в событии [OnEdited](../../Class/ListView/ListView.OnEdited.htm), возникающего после редактирования элемента.


-
Определение текста всплывающей подсказки в событии [OnGetHint](../../Class/ListView/ListView.OnGetHint.htm), возникающего при появлении всплывающей подсказки над элементом.
