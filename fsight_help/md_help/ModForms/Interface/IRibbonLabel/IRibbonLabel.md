# IRibbonLabel


Сборка: Forms;


## Описание


Интерфейс IRibbonLabel содержит свойства элемента управления ленты - Статический текст.


## Иерархия наследования


           [IComponent](../IComponent/IComponent.htm)


           [IBaseRibbonElement](../IBaseRibbonElement/IBaseRibbonElement.htm)


           IRibbonLabel


## Комментарии


Данный элемент управления используется для вывода надписей, которые могут использоваться в качестве заголовков для панелей, либо разделителя групп в коллекциях других элементов управления.
