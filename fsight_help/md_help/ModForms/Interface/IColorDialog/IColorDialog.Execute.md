# IColorDialog.Execute


## Синтаксис


Execute: Boolean;


## Описание


Метод Execute инициализирует
 диалог выбора цвета, после чего возвращает признак удачного выбора.


## Комментарии


Метод возвращает True, если
 была нажата кнопка «Применить», иначе возвращает False.
