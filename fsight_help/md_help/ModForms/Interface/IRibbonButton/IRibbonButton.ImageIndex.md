# IRibbonButton.ImageIndex


## Синтаксис


ImageIndex: Integer;


## Описание


Свойство ImageIndex определяет
 индекс маленького изображения, которое будет отображаться на кнопке рядом
 с текстом.


## Комментарии


Источник изображений устанавливается в свойстве [SmallImages](../IRibbonCategory/IRibbonCategory.SmallImages.htm)
 для вкладки, в области которой расположена кнопка.


Если кнопка расположена в области [общих
 элементов управления](../IRibbon/IRibbon.TabElements.htm), то источник изображений устанавливается в свойстве
 [IRibbon.Images](../IRibbon/IRibbon.Images.htm).


Примечание.
 Изображение, установленное в данном свойстве, будет отображаться если
 для элемента управления создан ярлык на [панели
 быстрого доступа](../IRibbonQuickAccessToolbar/IRibbonQuickAccessToolbar.htm).
