# IRibbonButton


Сборка: Forms;


## Описание


Интерфейс IRibbonButton содержит свойства элемента управления ленты - Кнопка.


## Иерархия наследования


           [IComponent](../IComponent/IComponent.htm)


           [IBaseRibbonElement](../IBaseRibbonElement/IBaseRibbonElement.htm)


           IRibbonButton


## Комментарии


Данный интерфейс описывает различные кнопки ленты: обычные кнопки с текстовым заголовком, графические кнопки, кнопки с раскрывающимся меню.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Action](IRibbonButton.Action.htm)


		 Свойство Action определяет
		 действие, ассоциируемое с кнопкой ленты.


		 ![](../../Property_Image.gif)
		 [Down](IRibbonButton.Down.htm)


		 Свойство Down определяет
		 состояние элемента управления.


		 ![](../../Property_Image.gif)
		 [ImageIndex](IRibbonButton.ImageIndex.htm)


		 Свойство ImageIndex
		 определяет индекс маленького изображения, которое будет отображаться
		 на кнопке рядом с текстом.


		 ![](../../Property_Image.gif)
		 [LargeImageIndex](IRibbonButton.LargeImageIndex.htm)


		 Свойство LargeImageIndex
		 определяет индекс большого изображения, которое будет отображаться
		 на кнопке рядом с текстом.


		 ![](../../Property_Image.gif)
		 [SplitButton](IRibbonButton.SplitButton.htm)


		 Свойство SplitButton
		 определяет признак разделения кнопки при наличии подэлементов.


		 ![](../../Property_Image.gif)
		 [SubItems](IRibbonButton.SubItems.htm)


		 Свойство SubItems возвращает
		 коллекцию элементов управления, которые формируют раскрывающееся
		 меню кнопки.
