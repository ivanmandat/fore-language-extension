# IRibbonButton.SubItems


## Синтаксис


SubItems: [IRibbonElementsCollection](../IRibbonElementsCollection/IRibbonElementsCollection.htm);


## Описание


Свойство SubItems возвращает коллекцию элементов управления, которые формируют раскрывающееся меню кнопки.


## Комментарии


Раскрывающееся меню строится на базе следующих элементов управления:


-
[Кнопка](IRibbonButton.htm).


-
[Кнопка для вызова диалога выбора цвета](../IRibbonColorButton/IRibbonColorButton.htm).


-
[Текстовая подпись](../IRibbonLabel/IRibbonLabel.htm).


-
[Кнопка для вызова комбинированного меню](../IRibbonPaletteButton/IRibbonPaletteButton.htm).


-
[Разделитель](../IRibbonSeparator/IRibbonSeparator.htm).
