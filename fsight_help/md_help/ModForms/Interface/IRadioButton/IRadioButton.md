# IRadioButton


## Описание


Интерфейс IRadioButton содержит
 свойства и методы компонента среды разработки - [RadioButton](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/RadioButton.htm).

		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Alignment](IRadioButton.Alignment.htm)


		 Свойство Aligment определяет
		 расположение переключателя в области компонента.


		 ![](../../Property_Image.gif)
		 [Checked](IRadioButton.Checked.htm)

		 ![](../../Property_Image.gif)
		 [Enabled](../IControl/IControl.Enabled.htm)

		 [Font](../IControl/IControl.Font.htm)

		 [Hint](../IControl/IControl.Hint.htm)
