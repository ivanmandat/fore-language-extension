# IPoint


## Описание


Интерфейс IPoint содержит свойства,
 определяющие координаты экранной точки.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [X](IPoint.X.htm)
		 Свойство X определяет
		 горизонтальную координату относительно экрана.


		 ![](../../Property_Image.gif)
		 [Y](IPoint.Y.htm)
		 Свойство Y определяет
		 вертикальную координату относительно экрана.
