# ITreeNodeGetHintEventArgs


## Описание


Интерфейс ITreeNodeGetHintEventArgs
 содержит свойства аргумента события, возникающего при появлении всплывающей
 подсказки у элемента, над которым остановился курсор мыши.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Text](ITreeNodeGetHintEventArgs.Text.htm)


		 Свойство Text определяет
		 текст всплывающей подсказки для элемента, на который наведен курсор
		 мыши.
