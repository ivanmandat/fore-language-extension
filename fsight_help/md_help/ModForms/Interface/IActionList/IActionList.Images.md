# IActionList.Images


## Синтаксис


Images: [IImageList](../IImageList/IImageList.htm);


## Описание


Свойство Images определяет компонент,
 изображения которого будут использоваться в действиях текущего компонента.


## Комментарии


В инспекторе объектов в список доступных значений данного свойства входят
 имена всех компонентов [ImageList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ImageList.htm),
 либо [GlobalImageList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/GlobalImageList.htm),
 расположенных на форме.
