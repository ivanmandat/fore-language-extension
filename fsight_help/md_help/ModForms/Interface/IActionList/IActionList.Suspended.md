# IActionList.Suspended


## Синтаксис


Suspended: Boolean;


## Описание


Свойство Suspended определяет состояние компонента и его действий.


## Комментарии


По умолчанию свойству установлено значение False, при этом для компонента и всех его действий будут генерироваться события. Если свойству установить значение True, то события генерироваться не будут.


Данное свойство не влияет на изменение параметров действий компонента. При их изменении соответствующие изменения будут отражены у всех компонентов, использующих действия.
