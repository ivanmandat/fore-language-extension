# IActionList.AddAction


## Синтаксис


AddAction(Action: [IActionItem](../IActionItem/IActionItem.htm));


## Параметры


Action. Действие, которое необходимо
 добавить в список действий данного компонента.


## Описание


Метод AddAction осуществляет
 добавление действия в список действий данного компонента.


## Комментарии


Метод может использоваться для переноса действий из коллекции действий
 одного компонента в коллекцию другого компонента. Если действие создается
 динамически, то оно должно быть добавлено в коллекцию какого-либо компонента
 до его первого использования.
