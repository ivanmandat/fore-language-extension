# IDateTimePickerEventArgs


## Описание


Интерфейс IDateTimePickerEventArgs
 содержит свойства аргумента событий [OnChange](../../Class/DateTimePicker/DateTimePicker.OnChange.htm),
 [OnCloseUp](../../Class/DateTimePicker/DateTimePicker.OnCloseUp.htm),
 [OnDropDown](../../Class/DateTimePicker/DateTimePicker.OnDropDown.htm)
 компонента [DateTimePicker](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/DateTimePicker.htm),
 возникающих при различных изменениях в компоненте во время выполнения
 формы.
