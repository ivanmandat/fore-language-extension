# IRibbonPanel.CenterColumnsVertically


## Синтаксис


CenterColumnsVertically: Boolean;


## Описание


Свойство CenterColumnsVertically
 определяет, будут ли элементы управления выровнены по центру относительно
 вертикали панели.


## Комментарии


Если свойству установлено значение True,
 то все элементы управления будут выровнены по центру относительно вертикали
 панели. Если свойству установлено значение False,
 то все элементы управления будут выровнены относительно верхней стороны
 панели.


По умолчанию свойству установлено значение False.
