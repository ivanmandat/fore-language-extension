# IRibbonPanel.LaunchButtonEnabled


## Синтаксис


LaunchButtonEnabled: Boolean;


## Описание


Свойство LaunchButtonEnabled
 определяет признак отображения кнопки запуска на панели.


## Комментарии


Если свойству установлено значение True,
 то с правой стороны от наименования панели будет отображена кнопка запуска.
 По умолчанию свойству установлено значение False,
 при этом кнопка запуска не отображается.


![](IRibbonPanel_LaunchButton.gif)


Кнопка запуска может быть использована для вызова различных окон, содержащих
 дополнительные настройки, связанные с данной панелью.
