# ITreeControlCompareEventArgs


## Описание


Интерфейс ITreeControlCompareEventArgs
 содержит свойства аргумента события, наступающего при пользовательской
 сортировке элементов компонентов [TreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeCombo.htm)
 или [TreeList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeList.htm).
