# ITreeControlCompareEventArgs.Compare


## Синтаксис


Compare: Integer;


## Описание


Свойство Compare устанавливает признак сортировки.


## Комментарии


Для свойства должно быть установлено одно из следующих значений:


-
0 - если элементы равны.


-
меньше 0 - если [LeftItem](ITreeControlCompareEventArgs.LeftItem.htm) > [RightItem](ITreeControlCompareEventArgs.RightItem.htm).


-
больше 0 - если [LeftItem](ITreeControlCompareEventArgs.LeftItem.htm) < [RightItem](ITreeControlCompareEventArgs.RightItem.htm).
