# OnChange


## Синтаксис


OnChange(Sender: Object; Args: [ICommonCalendarEventArgs](../ICommonCalendarEventArgs/ICommonCalendarEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnChange наступает после
 изменения даты, внесенных с помощью мыши или клавиатуры.
