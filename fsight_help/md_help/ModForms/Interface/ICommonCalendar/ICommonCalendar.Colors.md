# ICommonCalendar.Colors


## Синтаксис


Colors: [ICalendarColors](../ICalendarColors/ICalendarColors.htm);


## Описание


Свойство Colors возвращает настройки цветового оформления календаря.


## Комментарии


Изменение цветового оформления календаря не доступно в операционной системе Windows 7 при использовании темы оформления Aero и в более поздних операционных системах.
