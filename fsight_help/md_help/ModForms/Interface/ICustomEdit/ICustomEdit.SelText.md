# ICustomEdit.SelText


## Синтаксис


SelText: String;


## Описание


Свойство SelText определяет
 текст, выделенный в компоненте.


## Комментарии


При получении значения свойства SelText
 будет возвращён текст, который выделен в компоненте.


При установке значения свойства SelText
 поведение зависит от наличия уже выделенного в компоненте текста:


	- В компоненте выделен какой-либо текст. Выделенный текст будет
	 заменён значением, которое устанавливается в SelText.


	- В компоненте отсутствует выделенный текст. Задаваемое в SelText значение будет добавлено
	 в позицию, в которой расположен курсор ([SelStart](ICustomEdit.SelStart.htm)).


После установки значения SelText
 выделение в компоненте будет снято. При этом само свойство SelText
 будет возвращать пустое значение, [SelLength](ICustomEdit.SelLength.htm)
 будет равно нулю, а значение свойства [SelStart](ICustomEdit.SelStart.htm)
 будет изменено с учётом длины текста, который был указан в качестве значения
 для SelText.
