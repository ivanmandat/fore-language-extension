# ICustomEdit.ClearUndo


## Синтаксис


ClearUndo;


## Описание


Метод ClearUndo используется для фиксации изменений свойства [Text](../IControl/IControl.Text.htm).


## Комментарии


После вызова этого метода свойство [CanUndo](ICustomEdit.CanUndo.htm) устанавливается в False и буфер доступных изменений метода [Undo](ICustomEdit.Undo.htm) становится пустым.
