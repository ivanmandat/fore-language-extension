# ICustomEdit.AutoSelect


## Синтаксис


AutoSelect: Boolean;


## Описание


Свойство AutoSelect определяет,
 будет ли выделяться текст компонента при получении компонентом фокуса.


## Комментарии


Допустимые значения:


	- True. Выделять текст
	 компонента;


	- False. Не выделять текст
	 компонента.


Примечание.
 Значение свойства учитывается, если компонент получает фокус при табуляции
 по различным элементам управления формы с клавиатуры, и не учитывается,
 если установка фокуса осуществляется с помощью мыши.
