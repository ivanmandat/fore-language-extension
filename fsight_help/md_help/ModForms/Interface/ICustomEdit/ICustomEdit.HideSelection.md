# ICustomEdit.HideSelection


## Синтаксис


HideSelection: Boolean;


## Описание


Свойство HideSelection определяет, будет ли выделенный текст компонента оставаться таковым при передаче фокуса другому компоненту.


## Комментарии


Если свойство установлено в False, то текст будет оставаться выделенным. По умолчанию HideSelection = True.
