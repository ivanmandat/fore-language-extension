# OnChange


## Синтаксис


OnChange(Sender: Object; Args: IEventArgs);


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnChange наступает после
 изменений свойства [Text](../IControl/IControl.Text.htm) компонента
 редактирования.


## Комментарии


Это событие позволяет сразу отреагировать на изменения, вносимые пользователем
 в компонент редактирования текста.
