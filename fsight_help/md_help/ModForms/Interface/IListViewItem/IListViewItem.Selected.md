# IListViewItem.Selected


## Синтаксис


Selected: Boolean;


## Описание


Свойство Selected определяет признак отметки данного элемента.


## Комментарии


Если свойство равно True, элемент будет выбранным и будет выделяться синим цветом, иначе - выделения не будет.
