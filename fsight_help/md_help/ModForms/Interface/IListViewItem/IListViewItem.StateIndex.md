# IListViewItem.StateIndex


## Синтаксис


StateIndex: Integer;


## Описание


Свойство StateIndex определяет индекс, связывающий элемент с одним из рисунков свойства [StateImages](../IListView/IListView.StateImages.htm) компонента, который будет отображать состояние элемента.
