# IListViewItem.Focused


## Синтаксис


Focused: Boolean;


## Описание


Свойство Focused определяет, находится ли элемент компонента в фокусе.


## Комментарии


Если установлено значение True, элемент находится в фокусе, иначе - элемент не активен.
