# IListViewItem.Cut


## Синтаксис


Cut: Boolean;


## Описание


Свойство Cut определяет, будет
 ли элемент отмечен как вырезанный.


## Комментарии


Cut не копирует элемент в буфер
 обмена и не удаляет его из списка, поэтому эти операции должно производить
 разрабатываемое приложение. Если свойству Cut
 установлено значение True, то
 пиктограмма элемента будет иметь более светлый вид, сигнализируя о том,
 что элемент был вырезан.
