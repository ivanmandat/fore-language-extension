# IListViewItem.Checked


## Синтаксис


Checked: Boolean;


## Описание


Свойство Checked определяет,
 установлена ли отметка флажка для элемента.


## Комментарии


Если свойству установлено значение True,
 то элементу будет установлена отметка, иначе отметки не будет. Свойство
 актуально, если свойство [CheckBoxes](../IListView/IListView.Checkboxes.htm)
 = True.
