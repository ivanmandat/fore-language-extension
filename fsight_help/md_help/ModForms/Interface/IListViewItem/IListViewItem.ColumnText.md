# IListViewItem.ColumnText


## Синтаксис


ColumnText(ColumnIndex: Integer): String;


## Параметры


ColumnIndex - индекс столбца, текст которого необходимо определить.


## Описание


Свойство ColumnText определяет для данного элемента текст столбца, индекс которого передается посредством параметра ColumnIndex.
