# IListViewItem


## Описание


Интерфейс IListViewItem содержит
 свойства и методы элемента компонента [ListView](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ListView.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Checked](IListViewItem.Checked.htm)


		 Свойство Checked определяет,
		 установлена ли отметка флажка для элемента.


		 ![](../../Property_Image.gif)
		 [ColumnText](IListViewItem.ColumnText.htm)


		 Свойство ColumnText
		 определяет текст заданного столбца элемента.


		 ![](../../Property_Image.gif)
		 [Cut](IListViewItem.Cut.htm)


		 Свойство Cut определяет,
		 будет ли элемент отмечен как вырезанный.


		 ![](../../Property_Image.gif)
		 [Data](IListViewItem.Data.htm)


		 Свойство Data предназначено
		 для хранения любых пользовательских данных.


		 ![](../../Property_Image.gif)
		 [Focused](IListViewItem.Focused.htm)


		 Свойство Focused определяет,
		 находится ли элемент компонента в фокусе.


		 ![](../../Property_Image.gif)
		 [Hint](IListViewItem.Hint.htm)


		 Свойство Hint определяет
		 текст всплывающей подсказки для элемента.


		 ![](../../Property_Image.gif)
		 [ImageIndex](IListViewItem.ImageIndex.htm)


		 Свойство ImageIndex
		 определяет индекс изображения, которое будет располагаться слева
		 от элемента.


		 ![](../../Property_Image.gif)
		 [Indent](IListViewItem.Indent.htm)


		 Свойство Indent определяет
		 отступ элемента от левой границы колонки компонента.


		 ![](../../Property_Image.gif)
		 [Index](IListViewItem.Index.htm)


		 Свойство Index возвращает
		 индекс элемента в общем списке элементов.


		 ![](../../Property_Image.gif)
		 [IsVisible](IListViewItem.IsVisible.htm)


		 Свойство IsVisible
		 возвращает признак расположения элемента в видимой области компонента.


		 ![](../../Property_Image.gif)
		 [ListView](IListViewItem.ListView.htm)


		 Свойство ListView возвращает
		 объект-родитель.


		 ![](../../Property_Image.gif)
		 [OverlayIndex](IListViewItem.OverlayIndex.htm)


		 Свойство OverlayIndex
		 определяет индекс покрывающей маски.


		 ![](../../Property_Image.gif)
		 [Selected](IListViewItem.Selected.htm)


		 Свойство Selected определяет
		 признак отметки данного элемента.


		 ![](../../Property_Image.gif)
		 [StateIndex](IListViewItem.StateIndex.htm)


		 Свойство StateIndex
		 определяет индекс изображения [StateImages](../IListView/IListView.StateImages.htm)
		 компонента, который будет отображать состояние элемента.


		 ![](../../Property_Image.gif)
		 [Text](IListViewItem.Text.htm)


		 Свойство Text определяет
		 текст элемента компонента.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CancelEdit](IListViewItem.CancelEdit.htm)


		 Метод CancelEdit осуществляет
		 выход из режима редактирования элемента с сохранением изменений.


		 ![](../../Sub_Image.gif)
		 [Delete](IListViewItem.Delete.htm)


		 Метод Delete осуществляет
		 удаление элемента компонента.


		 ![](../../Sub_Image.gif)
		 [EditText](IListViewItem.EditText.htm)


		 Метод EditText переводит
		 элемент списка в режим редактирования текста.


		 ![](../../Sub_Image.gif)
		 [MakeVisible](IListViewItem.MakeVisible.htm)


		 Метод MakeVisible осуществляет
		 прокрутку дерева элементов таким образом, чтобы данный элемент
		 попал в видимую область компонента.
