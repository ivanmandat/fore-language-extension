# IListViewItem.Hint


## Синтаксис


Hint: String;


## Описание


Свойство Hint определяет текст всплывающей подсказки для элемента.


## Комментарии


Всплывающая подсказка для элемента отображается, если свойству [ShowHints](../IListView/IListView.ShowHints.htm) компонента установлено значение True.


По умолчанию свойство Hint не задано, при этом текст всплывающей подсказки соответствует [тексту](IListViewItem.Text.htm) элемента.


Если для компонента установлен стиль [ListViewStyle.Report](../../Enums/ListViewStyle.htm), то всплывающая подсказка, указанная в свойстве Hint, будет отображена при задержке курсора мыши в области первого столбца компонента.
