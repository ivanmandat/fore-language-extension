# IWizardSplash.Background


## Синтаксис


Background: [IGxImage](ModDrawing.chm::/Interface/IGxImage/IGxImage.htm);


## Описание


Свойство Background позволяет определить фон компонента [WizardSplash](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/WizardSplash.htm).


Примечание. В режиме разработки данное свойство можно также изменить через пункт контекстного меню "Фон..." комнонента [WizardSplash](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/WizardSplash.htm).
