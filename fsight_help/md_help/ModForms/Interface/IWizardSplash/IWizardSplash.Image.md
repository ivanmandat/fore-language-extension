# IWizardSplash.Image


## Синтаксис


Image: [IGxImage](ModDrawing.chm::/Interface/IGxImage/IGxImage.htm);


## Описание


Свойство Image позволяет определить изображение, которое будет помещаться в общую заставку мастера.


Примечание. В режиме разработки данное свойство можно также изменить через пункт контекстного меню «Рисунок...» комнонента [WizardSplash](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/WizardSplash.htm).
