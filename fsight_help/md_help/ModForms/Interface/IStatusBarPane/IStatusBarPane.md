# IStatusBarPane


## Описание


Интерфейс IStatusBarPane содержит
 свойства панели компонента [StatusBar](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/StatusBar.htm).


## Свойства


		  Имя свойства

		 ![](../../Property_Image.gif)
		 [ImageIndex](IStatusBarPane.ImageIndex.htm)
		 Свойство ImageIndex
		 определяет индекс изображения, отображаемого слева от текста панели.


		 ![](../../Property_Image.gif)
		 [StatusBar](IStatusBarPane.StatusBar.htm)
		 Свойство StatusBar
		 возвращает объект-родитель.


		 ![](../../Property_Image.gif)
		 [Text](IStatusBarPane.Text.htm)
		 Свойство Text определяет
		 текст, отображаемый в панели.


		 ![](../../Property_Image.gif)
		 [Width](IStatusBarPane.Width.htm)
		 Свойство Width определяет
		 ширину панели компонента [StatusBar](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/StatusBar.htm),
		 в пикселях.
