# IRibbonElementsCollection.RemoveAt


## Синтаксис


RemoveAt(Index: Integer);


## Параметры


Index. Индекс элемента управления.


## Описание


Метод RemoveAt удаляет элемент управления из коллекции по индексу.


## Комментарии


При удалении какого-либо элемента управления все остальные элементы будут смещены как в коллекции, так и визуально.
