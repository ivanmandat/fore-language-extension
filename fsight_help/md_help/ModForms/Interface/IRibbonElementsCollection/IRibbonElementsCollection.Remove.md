# IRibbonElementsCollection.Remove


## Синтаксис


Remove(Element: [IBaseRibbonElement](../IBaseRibbonElement/IBaseRibbonElement.htm));


## Параметры


Element. Элемент управления, который необходимо удалить из коллекции.


## Описание


Метод Remove удаляет указанный элемент управления из коллекции.


## Комментарии


При удалении какого-либо элемента управления все остальные элементы будут смещены как в коллекции, так и визуально.
