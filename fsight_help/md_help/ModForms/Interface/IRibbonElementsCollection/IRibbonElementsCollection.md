# IRibbonElementsCollection


Сборка: Forms;


## Описание


Интерфейс IRibbonElementsCollection
 содержит свойства и методы для работы с коллекциями элементов управления
 различных областей ленты.


## Иерархия наследования


           [IBaseRibbonElements](../IBaseRibbonElements/IBaseRibbonElements.htm)


           IRibbonElementsCollection


## Комментарии


Получить коллекции элементов можно используя следующие свойства:


	- [IRibbon.TabElements](../IRibbon/IRibbon.TabElements.htm).
	 Общие элементы управления.


	- [IRibbonButton.SubItems](../IRibbonButton/IRibbonButton.SubItems.htm).
	 Элементы управления, связанные с какой-либо кнопкой.


	- [IRibbonGroup.Elements](../IRibbonGroup/IRibbonGroup.Elements.htm).
	 Элементы управления, сгруппированные в одну группу.


	- [IRibbonMainButton.Buttons](../IRibbonMainButton/IRibbonMainButton.Buttons.htm).
	 Кнопки, связанные с кнопкой приложения.


	- [IRibbonPanel.Elements](../IRibbonPanel/IRibbonPanel.Elements.htm).
	 Элементы управления определенной панели.


	- [IRibbonQuickAccessToolbar.Elements](../IRibbonQuickAccessToolbar/IRibbonQuickAccessToolbar.Elements.htm).
	 Элементы управления, расположенные на панели быстрого доступа.
