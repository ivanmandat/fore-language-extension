# IRibbonElementsCollection.IndexOf


## Синтаксис


IndexOf(Item: [IBaseRibbonElement](../IBaseRibbonElement/IBaseRibbonElement.htm)): Integer;


## Параметры


Item. Элемент управления, индекс которого необходимо получить.


## Описание


Метод IndexOf возвращает для указанного элемента управления индекс в коллекции.


## Комментарии


Индексы элементов в коллекции соответствуют визуальному расположению элементов на ленте. Организация визуального расположения элементов ведется сверху вниз, слева направо.
