# IPageControlSheets


## Описание


Интерфейс IPageControlSheets
 содержит свойства и методы коллекции страниц компонента [PageControl](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/PageControl.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IPageControlSheets.Count.htm)


		 Свойство Count возвращает
		 количество страниц в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IPageControlSheets.Item.htm)


		 Свойство Item возвращает
		 параметры указанной страницы компонента.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IPageControlSheets.Add.htm)


		 Метод Add добавляет
		 новую страницу в коллекцию страниц компонента.


		 ![](../../Sub_Image.gif)
		 [Clear](IPageControlSheets.Clear.htm)


		 Метод Clear удаляет
		 все страницы компонента.


		 ![](../../Sub_Image.gif)
		 [Insert](IPageControlSheets.Insert.htm)


		 Метод Insert осуществляет
		 вставку новой страницы в указанную позицию в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IPageControlSheets.Remove.htm)


		 Метод Remove удаляет
		 указанную страницу компонента.


		 ![](../../Sub_Image.gif)
		 [RemoveAt](IPageControlSheets.RemoveAt.htm)


		 Метод RemoveAt удаляет
		 из коллекции страницу с указанным индексом.
