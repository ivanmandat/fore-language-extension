# IPageControlSheets.RemoveAt


## Синтаксис


RemoveAt(Index: Integer);


## Параметры


Index.
 Индекс страницы в коллекции.


## Описание


Метод RemoveAt удаляет из коллекции
 страницу с указанным индексом.


## Комментарии


Значение параметра Index находится
 в диапазоне [0, [Count](IPageControlSheets.Count.htm)-1].
