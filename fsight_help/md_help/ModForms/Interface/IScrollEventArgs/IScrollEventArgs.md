# IScrollEventArgs


Сборка: Forms;


## Описание


Интерфейс IScrollEventArgs содержит
 свойства аргумента событий, связанных с изменением положения ползунка
 полос прокрутки компонентов.


## Иерархия наследования


           [IEventArgs](ForeSys.chm::/Interface/IEventArgs/IEventArgs.htm)


           IScrollEventArgs


## Комментарии


Данный аргумент используется в следующих событиях:


	- [OnHScroll](../IControl/IControl.OnHScroll.htm)


	- [OnVScroll](../IControl/IControl.OnVScroll.htm)
