# IListViewItems.Add


## Синтаксис


Add(Text: String:): [IListViewItem](../IListViewItem/IListViewItem.htm);


## Параметры


Text - текст элемента, который будет добавлен.


## Описание


Метод Add осуществляет добавление нового элемента в конец списка. Текст элемента передается посредством параметра Text.
