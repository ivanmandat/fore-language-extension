# IListViewItems.BeginUpdate


## Синтаксис


BeginUpdate;


## Описание


Метод BeginUpdate блокирует обновление визуального списка элементов в компоненте до тех пор, пока не будет выполнен метод [EndUpdate](IListViewItems.EndUpdate.htm).


## Комментарии


Используется при одновременной вставке нескольких элементов для предотвращения мерцания списка элементов в компоненте.
