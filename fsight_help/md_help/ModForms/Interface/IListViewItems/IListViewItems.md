# IListViewItems


## Описание


Интерфейс IListViewItems содержит
 свойства и методы коллекции элементов компонента [ListView](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ListView.htm).


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IListViewItems.Count.htm)


		 Свойство Count возвращает
		 количество элементов в списке.


		 ![](../../Property_Image.gif)
		 [Item](IListViewItems.Item.htm)


		 Свойство Item возвращает
		 элемент списка.


		 ![](../../Property_Image.gif)
		 [ListView](IListViewItems.ListView.htm)


		 Свойство ListView возвращает
		 объект-родитель.


## Методы


		  Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IListViewItems.Add.htm)


		 Метод Add осуществляет
		 добавление нового элемента в конец списка.


		 ![](../../Sub_Image.gif)
		 [BeginUpdate](IListViewItems.BeginUpdate.htm)


		 Метод BeginUpdate блокирует
		 обновление визуального списка элементов в компоненте до тех пор,
		 пока не будет выполнен метод [EndUpdate](IListViewItems.EndUpdate.htm).


		 ![](../../Sub_Image.gif)
		 [Clear](IListViewItems.Clear.htm)


		 Метод Clear очищает
		 список элементов компонента.


		 ![](../../Sub_Image.gif)
		 [Delete](IListViewItems.Delete.htm)


		 Метод Delete осуществляет
		 удаление элемента компонента.


		 ![](../../Sub_Image.gif)
		 [EndUpdate](IListViewItems.EndUpdate.htm)


		 Метод EndUpdate отменяет
		 действие метода [BeginUpdate](IListViewItems.BeginUpdate.htm).


		 ![](../../Sub_Image.gif)
		 [Insert](IListViewItems.Insert.htm)


		 Метод Insert осуществляет
		 вставку нового элемента в указанную позицию.
