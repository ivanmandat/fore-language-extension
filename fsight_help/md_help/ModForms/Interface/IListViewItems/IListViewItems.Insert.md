# IListViewItems.Insert


## Синтаксис


Insert(Index: Integer; Text: String): [IListViewItem](../IListViewItem/IListViewItem.htm);


## Параметры


Index - индекс элемента, после которого будет вставлен новый элемент.


Text - текст нового элемента.


## Описание


Метод Insert осуществляет вставку нового элемента, после элемента индекс которого передается посредством параметра Index.


## Комментарии


Текст элемента передается посредством параметра Text.
