# IFormClass


## Описание


Интерфейс IFormClass содержит
 статические свойства класса [Form](../../Class/Form/Form.htm),
 реализующего пользовательскую форму.


## Комментарии


Используется только в настольном приложении.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Active](IFormClass.Active.htm)
		 Свойство Active возвращает
		 параметры формы, активной в данный момент.


		 ![](../../Property_Image.gif)
		 [MakeShortcut](IFormClass.MakeShortcut.htm)
		 Свойство MakeShortcut
		 возвращает значение, соответствующее комбинации «горячих
		 клавиш», которую необходимо назначить для какого-либо действия.
