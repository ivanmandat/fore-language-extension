# IImageButtonMargins.Top


## Синтаксис


Top: Integer;


## Описание


Свойство Top определяет границу
 области текста компонента относительно правой границы компонента. Размер
 задается в пикселях. По умолчанию свойству установлено значение 0, при
 этом граница текста совпадает с границей компонента.
