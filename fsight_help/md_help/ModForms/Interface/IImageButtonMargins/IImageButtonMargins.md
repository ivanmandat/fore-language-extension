# IImageButtonMargins


## Описание


Интерфейс IImageButtonMargins
 содержит свойства для определения границ текста внутри области компонента
 [ImageButton](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ImageButton.htm) относительно границ компонента.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Bottom](IImageButtonMargins.Bottom.htm)
		 Свойство Bottom определяет
		 границу области текста компонента относительно нижней границы
		 компонента.


		 ![](../../Property_Image.gif)
		 [Left](IImageButtonMargins.Left.htm)
		 Свойство Left определяет
		 границу области текста компонента относительно левой границы компонента.


		 ![](../../Property_Image.gif)
		 [Right](IImageButtonMargins.Right.htm)
		 Свойство Right определяет
		 границу области текста компонента относительно правой границы
		 компонента.


		 ![](../../Property_Image.gif)
		 [Top](IImageButtonMargins.Top.htm)
		 Свойство Top определяет
		 границу области текста компонента относительно правой границы
		 компонента.
