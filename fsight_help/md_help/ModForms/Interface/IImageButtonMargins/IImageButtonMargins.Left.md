# IImageButtonMargins.Left


## Синтаксис


Left: Integer;


## Описание


Свойство Left определяет границу
 области текста компонента относительно левой границы компонента. Размер
 задается в пикселях. По умолчанию свойству установлено значение 0, при
 этом граница текста совпадает с границей компонента.
