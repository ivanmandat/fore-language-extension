# ITermList


## Описание


Интерфейс ITermList содержит
 свойства и методы для работы с коллекцией термов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITermList.Count.htm)
		 Свойство Count возвращает
		 количество термов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ITermList.Item.htm)
		 Свойство Item возвращает
		 свойства указанного терма.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CreateCustom](ITermList.CreateCustom.htm)
		 Метод CreateCustom
		 создает пользовательский терм.


		 ![](../../Sub_Image.gif)
		 [CreateLiteral](ITermList.CreateLiteral.htm)
		 Метод CreateLiteral
		 создает литеральный терм.


		 ![](../../Sub_Image.gif)
		 [IndexOf](ITermList.IndexOf.htm)
		 Метод IndexOf возвращает
		 индекс терма в коллекции.
