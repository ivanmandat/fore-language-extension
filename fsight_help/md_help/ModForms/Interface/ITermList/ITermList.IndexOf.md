# ITermList.IndexOf


## Синтаксис


IndexOf(Term: [ITerm](../ITerm/ITerm.htm)): Integer;


## Параметры


Term — терм, индекс которого необходимо получить.


## Описание


Метод IndexOf возвращает индекс терма в коллекции. Терм передается в качестве входного параметра Term.
