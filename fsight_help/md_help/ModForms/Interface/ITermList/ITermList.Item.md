# ITermList.Item


## Синтаксис


Item(Index: Integer): [ITerm](../ITerm/ITerm.htm);


## Параметры


Index — индекс терма в коллекции.


## Описание


Свойство Item возвращает свойства терма, индекс которого передается в качестве входного параметра Index.
