# ICtrlGroupConditions.Ungroup


## Синтаксис


Ungroup;


## Описание


Метод Ungroup разгруппировывает
 отмеченную группу элементов.


## Комментарии


Группа элементов доступна для разгруппировки, если отмечены все элементы
 группы. Для определения массива отмеченных элементов используйте свойство
 [ICtrlGroupConditions.SelectedRows](ICtrlGroupConditions.SelectedRows.htm).


Для определения возможности разгруппировки группы элементов используйте
 метод [ICtrlGroupConditions.CanUngroup](ICtrlGroupConditions.CanUngroup.htm).


Для разгруппировки группы элементов по индексам используйте метод [ICtrlGroupConditions.UngroupByIndexes](ICtrlGroupConditions.UngroupByIndexes.htm).
