# ICtrlGroupConditions.Groups


## Синтаксис


Groups: [IConditionGroups](../IConditionGroups/IConditionGroups.htm);


## Описание


Свойство Groups определяет коллекцию
 групп элементов.


## Комментарии


Группа элементов может состоять из набора сгруппированных элементов
 или одного элемента.


Для группировки отмеченных элементов используйте метод [ICtrlGroupConditions.Group](ICtrlGroupConditions.Group.htm).


Для группировки элементов по индексам используйте метод [ICtrlGroupConditions.GroupByIndexes](ICtrlGroupConditions.GroupByIndexes.htm).
