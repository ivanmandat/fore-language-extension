# ICtrlGroupConditions.GroupByIndexes


## Синтаксис


GroupByIndexes(Indexes: Array);


## Параметры


Indexes. Массив индексов элементов
 компонента.


Примечание.
 Индексы должны быть заданы последовательно.


## Описание


Метод GroupByIndexes группирует
 элементы по индексам.


## Комментарии


Для определения возможности группировки элементов используйте метод
 [ICtrlGroupConditions.CanGroup](ICtrlGroupConditions.CanGroup.htm).


Для группировки отмеченных элементов используйте метод [ICtrlGroupConditions.Group](ICtrlGroupConditions.Group.htm).
