# ICtrlGroupConditions.UngroupByIndexes


## Синтаксис


UngroupByIndexes(Indexes: Array);


## Параметры


Indexes. Массив индексов элементов
 компонента.


Примечание.
 Индексы должны быть заданы последовательно.


## Описание


Метод UngroupByIndexes разгруппировывает
 группу элементов по индексам.


## Комментарии


Для определения возможности разгруппировки группы элементов используйте
 метод [ICtrlGroupConditions.CanUngroup](ICtrlGroupConditions.CanUngroup.htm).


Для разгруппировки отмеченной группы элементов используйте метод [ICtrlGroupConditions.Ungroup](ICtrlGroupConditions.Ungroup.htm).
