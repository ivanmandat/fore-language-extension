# IFileDialog.FilterIndex


## Синтаксис


FilterIndex: Integer;


## Описание


Свойство FilterIndex определяет
 какой синтаксис файла, установленный в свойстве [Filter](IFileDialog.Filter.htm),
 будет устанавливаться по умолчанию при открытии диалога.


## Комментарии


Индексация начинается с 1. Если значение FilterIndex
 выходит за пределы, то по умолчанию берется первый синтаксис файла указанного
 в свойстве [Filter](IFileDialog.Filter.htm).
