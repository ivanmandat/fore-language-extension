# IFileDialog.DefaultExt


## Синтаксис


DefaultExt: String;


## Описание


Свойство DefaultExt определяет расширение файла, которое будет использовано по умолчанию данным компонентом диалога.


## Комментарии


Это расширение автоматически будет добавлено к выбранному имени файла, если только оно не содержит зарегестрированного расширения. Строка расширения должна содержать не более 3 символов.
