# IFileDialog.DereferenceLinks


## Синтаксис


DereferenceLinks: Boolean;


## Описание


Свойство DereferenceLinks определяет
 значение, которое возвращает диалог при выборе ярлыка.


## Комментарии


Если DereferenceLinks = True, то при выборе ярлыка вернется
 путь к объекту, указанному в ярлыке, иначе вернется путь к самому ярлыку.
