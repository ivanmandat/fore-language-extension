# IFileDialog.CheckPathExists


## Синтаксис


CheckPathExists: Boolean;


## Описание


Свойство CheckPathExists определяет
 реакцию системы при попытке пользователя выбрать файл в несуществующем
 каталоге.


## Комментарии


Если свойству установлено значение True,
 то при указании пользователем неправильного пути генерируется сообщение
 об ошибке.
