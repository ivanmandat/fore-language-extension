# ISplitterEventArgs


## Описание


Интерфейс ISplitterEventArgs
 содержит свойства аргумента события, возникающего при изменении размеров
 у компонентов, связанных с компонентом [Splitter](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/Splitter.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [NewSize](ISplitterEventArgs.NewSize.htm)


		 Свойство NewSize определяет
		 новый размер компонента, с которым связан компонент [Splitter](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/Splitter.htm).


		 ![](../../Property_Image.gif)
		 [Resize](ISplitterEventArgs.Resize.htm)


		 Свойство Resize определяет,
		 будет ли производиться изменение размеров компонента, связанного
		 с компонентом [Splitter](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/Splitter.htm).
