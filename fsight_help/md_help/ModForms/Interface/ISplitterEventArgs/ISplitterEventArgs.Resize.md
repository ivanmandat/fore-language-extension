# ISplitterEventArgs.Resize


## Синтаксис


Resize: Boolean;


## Описание


Свойство Resize определяет, будет ли производиться изменение размеров компонента, связанного с компонентом [Splitter](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/Splitter.htm).


## Комментарии


По умолчанию свойству установлено значение True, при этом изменение размеров разрешено. Если свойству установить значение False, то изменение размеров компонентов будет запрещено.
