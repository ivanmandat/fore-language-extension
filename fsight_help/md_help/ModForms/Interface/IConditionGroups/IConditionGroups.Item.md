# IConditionGroups.Item


## Синтаксис


Item(Index: Integer): [IConditionGroup](../IConditionGroup/IConditionGroup.htm);


## Параметры


Index. Индекс группы элементов
 в коллекции.


## Описание


Свойство Item возвращает группу
 элементов по индексу.


## Комментарии


Для определения коллекции групп элементов используйте свойство [ICtrlGroupConditions.Groups](../ICtrlGroupConditions/ICtrlGroupConditions.Groups.htm).

Использование свойства приведено в примере для [ICtrlGroupConditions.InsertRow](../ICtrlGroupConditions/ICtrlGroupConditions.InsertRow.htm).
