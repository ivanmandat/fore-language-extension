# IConditionGroups.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 групп элементов.

Для определения коллекции групп элементов используйте свойство [ICtrlGroupConditions.Groups](../ICtrlGroupConditions/ICtrlGroupConditions.Groups.htm).
