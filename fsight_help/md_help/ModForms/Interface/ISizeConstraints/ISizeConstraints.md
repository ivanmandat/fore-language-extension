# ISizeConstraints


## Описание


Интерфейс ISizeConstraints содержит
 свойства, задающее ограничение на максимальный и минимальный размер формы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [MaxHeight](ISizeConstraints.MaxHeight.htm)
		 Свойство MaxHeight
		 определяет максимально возможную высоту формы.


		 ![](../../Property_Image.gif)
		 [MaxWidth](ISizeConstraints.MaxWidth.htm)
		 Свойство MaxWidth определяет
		 максимально возможную ширину формы.


		 ![](../../Property_Image.gif)
		 [MinHeight](ISizeConstraints.MinHeight.htm)
		 Свойство MinHeight
		 определяет минимально возможную высоту формы.


		 ![](../../Property_Image.gif)
		 [MinWidth](ISizeConstraints.MinWidth.htm)
		 Свойство MinWidth определяет
		 минимально возможную ширину формы.
