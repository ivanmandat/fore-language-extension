# IMouseEventArgs.Button


## Синтаксис


Button: Integer;


## Описание


Свойство Button возвращает код
 кнопки мыши, которой было осуществлено какое-либо действие.


## Комментарии


Свойство возвращает следующие значения:


	- -1 - событие не связано с кнопками мыши.


	- 0 – левая кнопка мыши.


	- 1 – правая кнопка мыши.


	- 2 – средняя кнопка мыши (колёсико).
