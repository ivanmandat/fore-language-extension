# IMouseEventArgs


## Описание


Интерфейс IMouseEventArgs содержит
 свойства аргумента события, возникающего во время выполнения действий
 мышью.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Button](IMouseEventArgs.Button.htm)
		 Свойство Button возвращает
		 код кнопки мыши, которой было осуществлено какое-либо действие.


		 ![](../../Property_Image.gif)
		 [Cancel](IMouseEventArgs.Cancel.htm)
		 Свойство Cancel определяет
		 признак отмены дальнейшего действия, связанного с мышью.


		 ![](../../Property_Image.gif)
		 [pPoint](IMouseEventArgs.pPoint.htm)
		 Свойство pPoint возвращает
		 координаты курсора мыши в клиентской части компонента во время
		 выполнения действий мышью.


		 ![](../../Property_Image.gif)
		 [ShiftState](IMouseEventArgs.ShiftState.htm)
		 Свойство ShiftState
		 возвращает значение, соответствующее состоянию вспомогательных
		 клавиш (SHIFT, ALT и CTRL) и кнопок мыши во время выполнения действий мышью.
