# IListViewActionEventArgs


## Описание


Интерфейс IListViewActionEventArgs
 содержит свойства аргументов событий, происходящих перед каким-либо действием
 над элементами компонента [ListView](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ListView.htm).

		 Свойство Allowed определяет
		 признак разрешения действия.
