# ITreeList.FilterStyle


## Синтаксис


FilterStyle: [TreeComboFilterStyle](../../Enums/TreeComboFilterStyle.htm);


## Описание


Свойство FilterStyle определяет
 режим построения дерева отфильтрованных элементов.


## Комментарии


Во время поиска при свернутом раскрывающемся списке компонента, для
 составления списка найденных элементов используется фильтрация исходного
 списка. Режим построения дерева отфильтрованных элементов определяется
 данным свойством.
