# IDateTimePickerEx.ShowSpinButton


## Синтаксис


ShowSpinButton: Boolean;


## Описание


Свойство ShowSpinButton определяет признак отображения кнопок SpinUp и SpinDown, используемых для изменения значений даты и времени.
