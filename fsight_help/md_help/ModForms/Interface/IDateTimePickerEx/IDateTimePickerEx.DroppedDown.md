# IDateTimePickerEx.DroppedDown


## Синтаксис


DroppedDown: Boolean;


## Описание


Свойство DroppedDown возвращает признак отображения раскрывающегося календаря компонента.


## Комментарии


Свойство возвращает значение True, если раскрывающийся календарь раскрыт, либо False, если календарь свернут и дата отображается в поле редактирования компонента.
