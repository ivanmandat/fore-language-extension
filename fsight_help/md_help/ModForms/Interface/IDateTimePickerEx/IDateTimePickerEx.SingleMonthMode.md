# IDateTimePickerEx.SingleMonthMode


## Синтаксис


SingleMonthMode: Boolean;


## Описание


Свойство SingleMonthMode определяет
 признак отделения чисел месяца от номеров недель.


## Комментарии


Если свойству установлено значение False,
 то между номерами недель и числами месяца в раскрывающемся календаре будет
 отображаться разделительная линия.
