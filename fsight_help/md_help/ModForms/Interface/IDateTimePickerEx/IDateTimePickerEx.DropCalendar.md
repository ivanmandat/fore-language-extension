# IDateTimePickerEx.DropCalendar


## Синтаксис


DropCalendar: Boolean;


## Описание


Свойство DropCalendar определяет
 возможность вызова в компоненте раскрывающегося календаря.


## Комментарии


Если свойству установлено значение True,
 то в компоненте будет присутствовать кнопка ![](../knopka_drop.gif), при
 нажатии на которую будет появляться раскрывающийся календарь, используемый
 для выбора даты.
