# IDateTimePickerEx.ShowModeWeeks


## Синтаксис


ShowModeWeeks: Boolean;


## Описание


Свойство ShowModeWeeks определяет, будет ли в области компонента отображаться кнопка для включения режима установки даты путем выбора недели в полугодии.


## Комментарии


Допустимые значения:


-
True. Кнопка отображается.


-
False. Кнопка не отображается. Установка режима выбора недели в полугодии доступна через свойство [Mode](IDateTimePickerEx.Mode.htm).
