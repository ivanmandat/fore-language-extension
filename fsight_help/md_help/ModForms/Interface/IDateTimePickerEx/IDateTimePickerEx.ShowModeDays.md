# IDateTimePickerEx.ShowModeDays


## Синтаксис


ShowModeDays: Boolean;


## Описание


Свойство ShowModeDays определяет, будет ли в области компонента отображаться кнопка для включения режима установки даты путем выбора дня в месяце.


## Комментарии


Допустимые значения:


-
True. Кнопка отображается.


-
False. Кнопка не отображается. Установка режима выбора дня в месяце доступна через свойство [Mode](IDateTimePickerEx.Mode.htm).
