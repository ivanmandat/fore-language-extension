# IDateTimePickerEx.ShowModeQuarters


## Синтаксис


ShowModeQuarters: Boolean;


## Описание


Свойство ShowModeQuarters определяет, будет ли в области компонента отображаться кнопка для включения режима установки даты путем выбора квартала в году.


## Комментарии


Допустимые значения:


-
True. Кнопка отображается.


-
False. Кнопка не отображается. Установка режима выбора квартала в году доступна через свойство [Mode](IDateTimePickerEx.Mode.htm).
