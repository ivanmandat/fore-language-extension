# IDateTimePickerEx.ShowModeHalfYears


## Синтаксис


ShowModeHalfYears: Boolean;


## Описание


Свойство ShowModeHalfYears определяет, будет ли в области компонента отображаться кнопка для включения режима установки даты путем выбора полугодия.


## Комментарии


Допустимые значения:


-
True. Кнопка отображается.


-
False. Кнопка не отображается. Установка режима выбора полугодия доступна через свойство [Mode](IDateTimePickerEx.Mode.htm).
