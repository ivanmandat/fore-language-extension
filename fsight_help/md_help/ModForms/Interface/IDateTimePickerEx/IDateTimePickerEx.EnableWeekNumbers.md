# IDateTimePickerEx.EnableWeekNumbers


## Синтаксис


EnableWeekNumbers: Boolean;


## Описание


Свойство EnableWeekNumbers определяет признак отображения в раскрывающемся календаре номеров недель.


## Комментарии


Если свойству установлено значение True, то в раскрывающемся календаре будут отображаться номера недель.
