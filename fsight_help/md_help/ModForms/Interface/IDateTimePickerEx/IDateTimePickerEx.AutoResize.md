# IDateTimePickerEx.AutoResize


## Синтаксис


AutoResize: Boolean;


## Описание


Свойство AutoResize определяет
 признак автоматического подгона размеров компонента.


## Комментарии


Если свойству установлено значение True,
 то размеры компонента будут подогнаны таким образом, чтобы в поле редактирования
 компонента помещалось значение выбранной даты в соответствии с настройками
 отображения даты компонента (формат даты, шрифт и т.д.).
