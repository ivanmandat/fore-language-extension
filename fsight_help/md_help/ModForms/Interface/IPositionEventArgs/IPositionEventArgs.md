# IPositionEventArgs


## Описание


Интерфейс IPositionEventArgs
 содержит свойства аргумента событий, возникающих при перемещении компонента,
 либо при изменении его размеров.


## Комментарии


События, использующие данный аргумент:


	- [OnControlMove](../IControl/IControl.OnControlMove.htm)
	 - возникает при перемещении компонента.


	- [OnControlResize](../IControl/IControl.OnControlResize.htm)
	 - возникает при изменении размеров компонента.
