# IPositionEventArgs.Height


## Синтаксис


Height: Integer;


## Описание


Свойство Height возвращает высоту компонента после изменения его размеров.


## Комментарии


Данное свойство используется в обработчике события [OnControlResize](../IControl/IControl.OnControlResize.htm).
