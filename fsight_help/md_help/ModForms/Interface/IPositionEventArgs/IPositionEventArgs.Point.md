# IPositionEventArgs.Point


## Синтаксис


Point: [IPoint](../IPoint/IPoint.htm);


## Описание


Свойство Point возвращает координаты верхнего левого угла компонента после перемещения.


## Комментарии


Координаты угла возвращаются относительно системы координат родительского компонента. Для формы координаты возвращаются относительно экрана. Данное свойство используется в обработчике события [OnControlMove](../IControl/IControl.OnControlMove.htm).
