# IPositionEventArgs.Width


## Синтаксис


Width: Integer;


## Описание


Свойство Width возвращает ширину компонента после изменения его размеров.


## Комментарии


Данное свойство используется в обработчике события [OnControlResize](../IControl/IControl.OnControlResize.htm).
