# IPageControlSheet.TabVisible


## Синтаксис


TabVisible: Boolean;


## Описание


Свойство TabVisible определяет признак видимости данной вкладки страницы.


## Комментарии


Вкладка будет видимой, если данному свойству установлено значение True.
