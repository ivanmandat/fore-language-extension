# IPageControlSheet.ImageIndex


## Синтаксис


ImageIndex: Integer;


## Описание


Свойство ImageIndex определяет индекс изображения, отображаемого в заголовке вкладки страницы.


## Комментарии


Источником изображений является компонент, указанный в качестве значения свойства [Images](../IPageControl/IPageControl.Images.htm) родительского компонента [PageControl](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/PageControl.htm).
