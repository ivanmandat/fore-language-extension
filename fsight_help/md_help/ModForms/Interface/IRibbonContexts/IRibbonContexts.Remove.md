# IRibbonContexts.Remove


## Синтаксис


Remove(Element: [IRibbonContext](../IRibbonContext/IRibbonContext.htm));


## Параметры


Element. Контекстная вкладка, которую необходимо удалить.


## Описание


Метод Remove удаляет указанную контекстную вкладку из коллекции.
