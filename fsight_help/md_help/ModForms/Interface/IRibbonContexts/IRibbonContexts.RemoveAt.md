# IRibbonContexts.RemoveAt


## Синтаксис


RemoveAt(Index: Integer);


## Параметры


Index. Индекс контекстной вкладки.


## Описание


Метод RemoveAt удаляет контекстную вкладку из коллекции по индексу.
