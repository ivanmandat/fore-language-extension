# IRibbonContexts.IndexOf


## Синтаксис


IndexOf(Item: [IRibbonContext](../IRibbonContext/IRibbonContext.htm)): Integer;


## Параметры


Item. Контекстная вкладка, индекс которой необходимо получить.


## Описание


Метод IndexOf возвращает для указанной контекстной вкладки индекс в коллекции.
