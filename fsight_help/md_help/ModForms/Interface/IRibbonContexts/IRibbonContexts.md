# IRibbonContexts


Сборка: Forms;


## Описание


Интерфейс IRibbonContexts содержит
 свойства и методы для работы с коллекцией контекстных вкладок ленты.


## Иерархия наследования


IRibbonContexts


## Комментарии


Контекстные вкладки ленты отображаются при наступлении определенных
 условий, например когда в приложении выделен какой-либо объект. По умолчанию
 все контекстные вкладки скрыты. Для управления видимостью конкретной контекстной
 вкладкой используйте свойство [Visible](../IRibbonContext/IRibbonContext.Visible.htm).

## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IRibbonContexts.Add.htm)


		 Метод Add добавляет
		 указанную контекстную вкладку в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](IRibbonContexts.Clear.htm)


		 Метод Clear удаляет
		 все контекстные вкладки ленты.


		 ![](../../Sub_Image.gif)
		 [IndexOf](IRibbonContexts.IndexOf.htm)


		 Метод IndexOf возвращает
		 для указанной контекстной вкладки индекс в коллекции.


		 ![](../../Sub_Image.gif)
		 [Insert](IRibbonContexts.Insert.htm)


		 Метод Insert вставляет
		 указанную контекстную вкладку в указанную позицию в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IRibbonContexts.Remove.htm)


		 Метод Remove удаляет
		 указанную контекстную вкладку из коллекции.


		 ![](../../Sub_Image.gif)
		 [RemoveAt](IRibbonContexts.RemoveAt.htm)


		 Метод RemoveAt удаляет
		 контекстную вкладку из коллекции по индексу.
