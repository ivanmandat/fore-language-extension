# INavigationBar.Border


## Синтаксис


Border: [ControlBorderType](../../Enums/ControlBorderType.htm);


## Описание


Свойство Border определит границы, которые будут отрисованы для компонента.


## Комментарии


В качестве значения необходимо указать какое-либо значение, либо комбинацию значений, перечислимого типа [ControlBorderType](../../Enums/ControlBorderType.htm). При указании комбинации значения разделяются ключевым словом [Or](Fore.chm::/07_Operations/Logical_operations.htm). По умолчанию свойству установлено значение [ControlBorderType.All](../../Enums/ControlBorderType.htm).
