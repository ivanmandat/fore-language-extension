# INavigationBar.EndUpdate


## Синтаксис


EndUpdate;


## Описание


Метод EndUpdate возобновляет
 перерисовку содержимого компонента.


## Комментарии


После вызова данного метода будут отрисованы все изменения, произведенные
 между вызовами методов [BeginUpdate](INavigationBar.BeginUpdate.htm)
 и EndUpdate.
