# INavigationBar.BeginUpdate


## Синтаксис


BeginUpdate;


## Описание


Метод BeginUpdate отключает
 перерисовку содержимого компонента.


## Комментарии


Возобновление перерисовки компонента и применение всех изменений осуществляется
 методом [EndUpdate](INavigationBar.EndUpdate.htm).
