# INavigationBar.ActivePad


## Синтаксис


ActivePad: [INavigationBarPad](../INavigationBarPad/INavigationBarPad.htm);


## Описание


Свойство ActivePad определяет активную вкладку компонента.


## Комментарии


Для активной вкладки отображается содержимое связанной с ней панели. При активации какой-либо вкладки содержимое остальных вкладок будет скрыто. В качестве значения для свойства ActivePad можно указать одну из доступных вкладок коллекции [Pads](INavigationBar.Pads.htm).
