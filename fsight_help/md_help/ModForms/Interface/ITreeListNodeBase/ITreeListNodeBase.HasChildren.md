# ITreeListNodeBase.HasChildren


## Синтаксис


HasChildren: Boolean;


## Описание


Свойство HasChildren определяет признак наличия дочерних элементов у данного элемента.


## Комментарии


Свойство возвращает значение True, если элемент имеет дочерние элементы и False, если не имеет. Если элемент не имеет дочерних элементов и данному свойству установить значение True, то около элемента будет отображаться знак раскрытия иерархии (+), но раскрыть иерархию будет нельзя.
