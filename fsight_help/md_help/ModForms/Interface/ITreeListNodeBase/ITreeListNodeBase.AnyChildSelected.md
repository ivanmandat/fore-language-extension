# ITreeListNodeBase.AnyChildSelected


## Синтаксис


AnyChildSelected: Boolean;


## Описание


Метод AnyChildSelected возвращает признак наличия выделенных дочерних элементов.


## Комментарии


Метод возвращает значение True, если выделен хотя бы один дочерний элемент.
