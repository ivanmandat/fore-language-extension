# ITreeListNodeBase


## Описание


Интерфейс ITreeListNodeBase
 содержит базовые свойства вершин дерева.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITreeListNodeBase.Count.htm)
		 Свойство Count возвращает
		 количество дочерних элементов.


		 ![](../../Property_Image.gif)
		 [FirstChild](ITreeListNodeBase.FirstChild.htm)
		 Свойство FirstChild
		 возвращает параметры первого дочернего элемента.


		 ![](../../Property_Image.gif)
		 [HasChildren](ITreeListNodeBase.HasChildren.htm)
		 Свойство HasChildren
		 определяет признак наличия дочерних элементов у данного элемента.


		 ![](../../Property_Image.gif)
		 [Item](ITreeListNodeBase.Item.htm)
		 Свойство Item возвращает
		 параметры указанного дочернего элемента.


		 ![](../../Property_Image.gif)
		 [LastChild](ITreeListNodeBase.LastChild.htm)
		 Свойство LastChild
		 возвращает параметры последнего дочернего элемента.


		 ![](../../Property_Image.gif)
		 [TreeControl](ITreeListNodeBase.TreeControl.htm)
		 Свойство TreeControl
		 возвращает объект-родитель.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AnyChildSelected](ITreeListNodeBase.AnyChildSelected.htm)
		 Метод AnyChildSelected
		 возвращает признак наличия выделенных дочерних элементов.


		 ![](../../Sub_Image.gif)
		 [Collapse](ITreeListNodeBase.Collapse.htm)
		 Метод Collapse осуществляет
		 сворачивание структуры дочерних элементов дерева.


		 ![](../../Sub_Image.gif)
		 [DeleteChildren](ITreeListNodeBase.DeleteChildren.htm)
		 Метод DeleteChildren
		 осуществляет удаление всех дочерних элементов текущего элемента.


		 ![](../../Sub_Image.gif)
		 [Expand](ITreeListNodeBase.Expand.htm)
		 Метод Expand осуществляет
		 развертывание иерархии дерева дочерних элементов.


		 ![](../../Sub_Image.gif)
		 [SelectChildren](ITreeListNodeBase.SelectChildren.htm)
		 Метод SelectChildren
		 осуществляет отметку дочерних элементов.


		 ![](../../Sub_Image.gif)
		 [UnselectChildren](ITreeListNodeBase.UnselectChildren.htm)
		 Метод UnselectChildren
		 осуществляет снятие отметки дочерних элементов.
