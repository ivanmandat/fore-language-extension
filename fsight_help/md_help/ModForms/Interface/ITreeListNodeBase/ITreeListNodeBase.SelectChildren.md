# ITreeListNodeBase.SelectChildren


## Синтаксис


SelectChildren([Recursively: Boolean = False]);


## Параметры


Recursively. Параметр, определяющий рекурсивность выделения дочерних элементов дерева. Если параметр принимает значение True, то происходит выделение всех дочерних элементов. При установке параметру значения False произойдет выделение только непосредственных дочерних элементов, расположенных на один уровень ниже текущего элемента.


## Описание


Метод SelectChildren осуществляет отметку дочерних элементов.
