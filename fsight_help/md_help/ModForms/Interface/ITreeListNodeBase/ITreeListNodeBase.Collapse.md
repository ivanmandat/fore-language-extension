# ITreeListNodeBase.Collapse


## Синтаксис


Collapse([Recursively: Boolean = False]);


## Параметры


Recursively. Параметр, определяющий рекурсивность сворачивания. Если параметр принимает значение True, то происходит сворачивание всей иерархии. При установке параметру значения False произойдет сворачивание иерархии только выбранного элемента.


## Описание


Метод Collapse осуществляет сворачивание структуры дочерних элементов дерева.
