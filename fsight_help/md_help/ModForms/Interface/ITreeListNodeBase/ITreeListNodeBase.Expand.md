# ITreeListNodeBase.Expand


## Синтаксис


Expand([Recurse: Boolean = False]);


## Параметры


Recurse - параметр, определяющий рекурсивность разворачивания. Если параметр принимает значение True, то происходит развертывание всей иерархии всех дочерних элементов. При установке параметру значения False произойдет развертывание иерархии только выбранного элемента.


## Описание


Метод Expand осуществляет развертывание иерархии дерева дочерних элементов.
