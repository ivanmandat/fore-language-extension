# IRibbonFloatEdit.MinValue


## Синтаксис


MinValue: Double;


## Описание


Свойство MinValue определяет
 минимальное значение, которое можно будет установить в редакторе.


## Комментарии


Если MinValue = [MaxValue](IRibbonFloatEdit.MaxValue.htm)
 = 0, то минимальное значение
 ограничено минимальным значением, доступным для типа данных Double.
