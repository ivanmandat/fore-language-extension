# IRibbonFloatEdit.Value


## Синтаксис


Value: Double;


## Описание


Свойство Value определяет значение,
 отображаемое в редакторе.


## Комментарии


Указываемое значение должно находиться в диапазоне [[MinValue](IRibbonFloatEdit.MinValue.htm);
 [MaxValue](IRibbonFloatEdit.MaxValue.htm)].
