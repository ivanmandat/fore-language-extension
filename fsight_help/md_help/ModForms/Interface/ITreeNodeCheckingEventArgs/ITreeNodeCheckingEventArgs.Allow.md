# ITreeNodeCheckingEventArgs.Allow


## Синтаксис


Allow: Boolean;


## Описание


Свойство Allow определяет возможность
 изменения отметки флажка у элемента.


## Комментарии


Если свойству установлено значение True,
 то будет возможность изменить отметку флажка элемента, иначе не будет.
