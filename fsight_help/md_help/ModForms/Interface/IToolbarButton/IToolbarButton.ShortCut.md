# IToolbarButton.ShortCut


## Синтаксис


ShortCut: Integer;


## Описание


Свойство ShortCut определяет
 комбинацию «горячих клавиш», используемых для быстрого доступа к кнопке.


## Комментарии


При создании «горячих клавиш» во время проектирования формы следует
 выбрать в инспекторе объектов свойство ShortCut
 и нажать необходимое сочетание клавиш.


Если кнопка использует объект [Action](IToolbarButton.Action.htm),
 то значение свойства ShortCut
 кнопки автоматически определяется значением свойства [ShortCut](../IActionItem/IActionItem.ShortCut.htm)
 объекта [Action](IToolbarButton.Action.htm).
