# IToolbarButton.Text


## Синтаксис


Text: String;


## Описание


Свойство Text определяет наименование кнопки в панели инструментов [ToolBar](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ToolBar.htm), которое будет отображаться в том случае, если свойству [ShowCaptions](../IToolbar/IToolbar.ShowCaptions.htm) установлено значение True.


## Комментарии


Если кнопка использует объект [Action](IToolbarButton.Action.htm), то значение свойства Text кнопки автоматически определяется значением свойства [Caption](../IActionItem/IActionItem.Caption.htm) объекта [Action](IToolbarButton.Action.htm).
