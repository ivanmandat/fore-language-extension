# IToolbarButton.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет видимость кнопки на панели инструментов во время выполнения формы.


## Комментарии


Если кнопка использует объект [Action](IToolbarButton.Action.htm), то значение свойства Visible кнопки автоматически определяется значением свойства [Visible](../IActionItem/IActionItem.Visible.htm) объекта [Action](IToolbarButton.Action.htm).
