# IToolbarButton.MenuMode


## Синтаксис


MenuMode: Boolean;


## Описание


Свойство MenuMode определяет
 признак вызова контекстного меню, установленного в свойстве [PopupMenu](IToolbarButton.PopupMenu.htm),
 при нажатии на саму кнопку.


## Комментарии


По умолчанию свойству установлено значение False,
 при этом контекстное меню вызывается при нажатии на дополнительную кнопку
 ![](../knopka_drop.gif),
 отображаемую справа. Если свойству установить значение True,
 то контекстное меню будет вызываться при нажатии на саму кнопку.


Актуально, если установлено свойство [PopupMenu](IToolbarButton.PopupMenu.htm).
