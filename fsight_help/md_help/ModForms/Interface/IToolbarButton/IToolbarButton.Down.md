# IToolbarButton.Down


## Синтаксис


Down: Boolean;


## Описание


Свойство Down определяет в каком состоянии находится кнопка.


## Комментарии


Если свойству установлено значение True, то кнопка отображается в нажатом состоянии, иначе в не нажатом. По умолчанию свойству установлено значение False.
