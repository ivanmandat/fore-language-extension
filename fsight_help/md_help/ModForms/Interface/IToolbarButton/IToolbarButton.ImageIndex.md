# IToolbarButton.ImageIndex


## Синтаксис


ImageIndex: Integer;


## Описание


Свойство ImageIndex определяет индекс изображения из списка свойства [Images](../IToolbar/IToolbar.Images.htm) панели инструментов [ToolBar](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ToolBar.htm).


## Комментарии


Если кнопка не содержит изображения, то значение свойства ImageIndex равно -1.
