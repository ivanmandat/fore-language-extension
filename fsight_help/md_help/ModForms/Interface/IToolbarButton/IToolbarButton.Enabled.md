# IToolbarButton.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет доступность компонента для пользователя.


## Комментарии


Если свойству установлено значение False, то компонент становится недоступным, отображается серым цветом, игнорирует события, связанные с мышью, клавиатурой и событием таймера.


Если кнопка использует объект [Action](IToolbarButton.Action.htm), то значение свойства Enabled кнопки автоматически определяется значением свойства [Enabled](../IActionItem/IActionItem.Enabled.htm) объекта [Action](IToolbarButton.Action.htm).
