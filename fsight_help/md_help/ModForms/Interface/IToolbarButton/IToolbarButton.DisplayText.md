# IToolbarButton.DisplayText


## Синтаксис


DisplayText: Boolean;


## Описание


Свойство DisplayText определяет признак отображения текста кнопки справа от пиктограммы кнопки на панели инструментов.


## Комментарии


Актуально, если свойству [ShowCaptions](../IToolbar/IToolbar.ShowCaptions.htm) установлено значение False.
