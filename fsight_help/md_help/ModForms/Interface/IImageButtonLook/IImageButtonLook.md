# IImageButtonLook


Сборка: Forms;


## Описание


Интерфейс IImageButtonLook содержит
 свойства, определяющие внешний вид компонента [ImageButton](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ImageButton.htm) в различных
 состояниях.


## Иерархия наследования


           IImageButtonLook


## Комментарии


Для получения настроек определенных состояний компонента [ImageButton](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ImageButton.htm) используются
 следующие свойства:


	- [IImageButton.DisabledLook](../IImageButton/IImageButton.DisabledLook.htm)
	 - параметры отображения кнопки в недоступном состоянии.


	- [IImageButton.HoverLook](../IImageButton/IImageButton.HoverLook.htm)
	 - параметры отображения кнопки при наведении на нее курсора мыши.


	- [IImageButton.NormalLook](../IImageButton/IImageButton.NormalLook.htm)
	 - параметры отображения кнопки в обычном состоянии.


	- [IImageButton.PressedLook](../IImageButton/IImageButton.PressedLook.htm)
	 - параметры отображения кнопки в нажатом состоянии.
