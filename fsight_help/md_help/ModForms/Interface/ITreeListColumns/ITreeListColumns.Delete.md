# ITreeListColumns.Delete


## Синтаксис


Delete(ColumnIndex: Integer);


## Параметры


ColumnIndex - индекс столбца, который необходимо удалить.


## Описание


Метод Delete осуществляет удаление столбца, индекс которого передается посредством параметра ColumnIndex.
