# ITreeListColumns


## Описание


Интерфейс ITreeListColumns содержит
 свойства и методы для работы с коллекцией столбцов компонентов [TreeList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeList.htm)
 и [TreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeCombo.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](ITreeListColumns.Count.htm)
		 Свойство Count возвращает
		 количество столбцов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](ITreeListColumns.Item.htm)
		 Свойство Item возвращает
		 параметры указанного столбца.


		 ![](../../Property_Image.gif)
		 [TreeControl](ITreeListColumns.TreeControl.htm)
		 Свойство TreeControl
		 возвращает объект-родитель.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](ITreeListColumns.Add.htm)
		 Метод Add осуществляет
		 добавление нового столбца.


		 ![](../../Sub_Image.gif)
		 [Clear](ITreeListColumns.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех столбцов из коллекции.


		 ![](../../Sub_Image.gif)
		 [Delete](ITreeListColumns.Delete.htm)
		 Метод Delete позволяет
		 удалить столбец.
