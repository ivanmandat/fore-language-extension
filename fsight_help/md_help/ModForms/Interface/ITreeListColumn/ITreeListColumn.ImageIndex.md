# ITreeListColumn.ImageIndex


## Синтаксис


ImageIndex: Integer;


## Описание


Свойство ImageIndex определяет индекс изображения из списка свойства [Images](../ITreeControl/ITreeControl.Images.htm) компонента [TreeList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeList.htm).


## Комментарии


Если столбец не содержит изображения, то значение свойства равно "-1".
