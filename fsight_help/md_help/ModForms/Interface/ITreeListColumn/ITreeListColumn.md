# ITreeListColumn


## Описание


Интерфейс ITreeListColumn содержит
 свойства и методы для работы со столбцом компонентов [TreeList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeList.htm)
 и [TreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeCombo.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Alignment](ITreeListColumn.Alignment.htm)


		 Свойство Alignment
		 определяет способ выравнивания текста заголовка и элементов в
		 столбце.


		 ![](../../Property_Image.gif)
		 [AutoSize](ITreeListColumn.AutoSize.htm)


		 Свойство AutoSize определяет
		 признак автоматического изменения ширины столбца в зависимости
		 от ширины компонента.


		 ![](../../Property_Image.gif)
		 [Caption](ITreeListColumn.Caption.htm)


		 Свойство Caption определяет
		 текст заголовка столбца.


		 ![](../../Property_Image.gif)
		 [ImageIndex](ITreeListColumn.ImageIndex.htm)


		 Свойство ImageIndex
		 определяет индекс изображения.


		 ![](../../Property_Image.gif)
		 [Index](ITreeListColumn.Index.htm)


		 Свойство Index определяет
		 индекс текущего столбца.


		 ![](../../Property_Image.gif)
		 [MaxWidth](ITreeListColumn.MaxWidth.htm)


		 Свойство MaxWidth определяет
		 максимальную ширину столбца.


		 ![](../../Property_Image.gif)
		 [MinWidth](ITreeListColumn.MinWidth.htm)


		 Свойство MinWidth определяет
		 минимальную ширину столбца.


		 ![](../../Property_Image.gif)
		 [Position](ITreeListColumn.Position.htm)
		 Свойство Position определяет
		 позицию столбца.


		 ![](../../Property_Image.gif)
		 [SortAscending](ITreeListColumn.SortAscending.htm)


		 Свойство SortAscending
		 определяет направление сортировки.


		 ![](../../Property_Image.gif)
		 [Visible](ITreeListColumn.Visible.htm)


		 Свойство Visible определяет
		 видимость колонки во время выполнения формы.


		 ![](../../Property_Image.gif)
		 [Width](ITreeListColumn.Width.htm)


		 Свойство Width определяет
		 ширину столбца в пикселях.
