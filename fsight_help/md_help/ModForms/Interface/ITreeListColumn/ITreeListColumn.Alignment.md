# ITreeListColumn.Alignment


## Синтаксис


Alignment: [TextAlignment](../../Enums/TextAlignment.htm);


## Описание


Свойство Alignment определяет способ выравнивания текста заголовка и элементов в столбце.


## Комментарии


По умолчанию текст заголовков и элементов выравнивается по левому краю.


Если значение свойства Alignment устанавливается для первого столбца, то выравнивание будет изменено только для текста заголовка. Для элементов выравнивание останется по левому краю.
