# IFontDialogOptions.NoOEMFonts


## Синтаксис


NoOEMFonts: Boolean;


## Описание


Свойство NoOEMFonts определяет
 признак отображения OEM шрифтов.


## Комментарии


По умолчанию свойству установлено значение False.
 Если свойству установить значение True,
 то в списке шрифтов диалога OEM шрифты отображаться не будут. Данное свойство
 является аналогом свойства [NoVectorFonts](IFontDialogOptions.NoVectorFonts.htm).
