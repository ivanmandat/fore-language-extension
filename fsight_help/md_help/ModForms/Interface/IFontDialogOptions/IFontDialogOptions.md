# IFontDialogOptions


## Описание


Интерфейс IFontDialogOptions
 предназначен для настройки параметров отображения компонента среды разработки
 - [FontDialog](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/FontDialog.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AnsiOnly](IFontDialogOptions.AnsiOnly.htm)
		 Свойство AnsiOnly определяет
		 признак отображения только тех шрифтов, которые используют набор
		 символов Windows.


		 ![](../../Property_Image.gif)
		 [ApplyButton](IFontDialogOptions.ApplyButton.htm)
		 Свойство ApplyButton
		 определяет признак отображения в диалоге кнопки "Применить".


		 ![](../../Property_Image.gif)
		 [FixedPitchOnly](IFontDialogOptions.FixedPitchOnly.htm)
		 Свойство FixedPitchOnly
		 определяет признак отображения только моноширинных шрифтов.


		 ![](../../Property_Image.gif)
		 [ForceFontExist](IFontDialogOptions.ForceFontExist.htm)
		 Свойство ForceFontExist
		 определяет признак возможности выбора шрифта только из списка
		 диалога.


		 ![](../../Property_Image.gif)
		 [LimitSize](IFontDialogOptions.LimitSize.htm)
		 Свойство LimitSize
		 определяет признак ограничения доступных размеров шрифта при вызове
		 диалога.


		 ![](../../Property_Image.gif)
		 [NoFaceSel](IFontDialogOptions.NoFaceSel.htm)
		 Свойство NoFaceSel
		 определяет признак выбора текущего шрифта компонента в списке
		 шрифтов диалога.


		 ![](../../Property_Image.gif)
		 [NoOEMFonts](IFontDialogOptions.NoOEMFonts.htm)
		 Свойство NoOEMFonts
		 определяет признак отображения OEM шрифтов.


		 ![](../../Property_Image.gif)
		 [NoSizeSel](IFontDialogOptions.NoSizeSel.htm)
		 Свойство NoSizeSel
		 определяет признак выбора размера текущего шрифта в списке размеров
		 диалога.


		 ![](../../Property_Image.gif)
		 [NoStyleSel](IFontDialogOptions.NoStyleSel.htm)
		 Свойство NoStyleSel
		 определяет признак выбора текущего стиля начертания шрифта в списке
		 стилей начертания диалога.


		 ![](../../Property_Image.gif)
		 [NoVectorFonts](IFontDialogOptions.NoVectorFonts.htm)
		 Свойство NoVectorFonts
		 определяет признак отображения векторных шрифтов в списке шрифтов
		 диалога.


		 ![](../../Property_Image.gif)
		 [ScalableOnly](IFontDialogOptions.ScalableOnly.htm)
		 Свойство ScalableOnly
		 определяет признак отображения масштабируемых шрифтов в списке
		 шрифтов диалога.


		 ![](../../Property_Image.gif)
		 [ShowEffects](IFontDialogOptions.ShowEffects.htm)
		 Свойство ShowEffects
		 определяет признак отображения элементов, позволяющих задавать
		 параметры цвета, положения, зачеркивания и подчеркивания текста.


		 ![](../../Property_Image.gif)
		 [TrueTypeOnly](IFontDialogOptions.TrueTypeOnly.htm)
		 Свойство TrueTypeOnly
		 определяет признак отображения TrueType шрифтов.
