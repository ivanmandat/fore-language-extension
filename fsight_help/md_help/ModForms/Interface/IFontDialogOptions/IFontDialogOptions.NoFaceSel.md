# IFontDialogOptions.NoFaceSel


## Синтаксис


NoFaceSel: Boolean;


## Описание


Свойство NoFaceSel определяет
 признак выбора текущего шрифта компонента в списке шрифтов диалога.


## Комментарии


Если свойству установить значение True,
 то при открытии диалога поле выбора шрифта будет пустым. По умолчанию
 свойству установлено значение False,
 при этом в диалоге будет выбран шрифт компонента (Шрифт, установленный
 в свойстве [Font](../IFontDialog/IFontDialog.Font.htm)).
