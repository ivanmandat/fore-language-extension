# IFontDialogOptions.NoVectorFonts


## Синтаксис


NoVectorFonts: Boolean;


## Описание


Свойство NoVectorFonts определяет
 признак отображения векторных шрифтов в списке шрифтов диалога.


## Комментарии


Если свойству установить значение True,
 то в списке шрифтов диалога векторные шрифты отображаться не будут. По
 умолчанию свойству установлено значение False.
 Данное свойство является аналогом свойства [NoOEMFonts](IFontDialogOptions.NoOEMFonts.htm).
