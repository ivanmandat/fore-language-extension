# IFontDialogOptions.TrueTypeOnly


## Синтаксис


TrueTypeOnly: Boolean;


## Описание


Свойство TrueTypeOnly определяет признак отображения TrueType шрифтов.


## Комментарии


По умолчанию свойству установлено значение False. Если свойству установить значение True, то в списке шрифтов диалога будут отображаться только TrueType шрифты.
