# IFontDialogOptions.AnsiOnly


## Синтаксис


AnsiOnly: Boolean;


## Описание


Свойство AnsiOnly определяет
 признак отображения только тех шрифтов, которые используют набор символов
 Windows.


## Комментарии


По умолчанию свойству установлено значение False.
 Если свойству установить значение True,
 то в списке шрифтов диалога не будут отображаться символьные шрифты.
