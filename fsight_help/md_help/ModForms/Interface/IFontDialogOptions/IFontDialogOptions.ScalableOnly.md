# IFontDialogOptions.ScalableOnly


## Синтаксис


ScalableOnly: Boolean;


## Описание


Свойство ScalableOnly определяет признак отображения масштабируемых шрифтов в списке шрифтов диалога.


## Комментарии


По умолчанию свойству установлено значение False. Если свойству установить значение True, то в списке шрифтов диалога масштабируемые шрифты отображаться не будут.
