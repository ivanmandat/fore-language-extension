# IFontDialogOptions.FixedPitchOnly


## Синтаксис


FixedPitchOnly: Boolean;


## Описание


Свойство FixedPitchOnly определяет
 признак отображения только моноширинных шрифтов (шрифты с одинаковым размером
 всех букв).


## Комментарии


По умолчанию свойству установлено значение False.
 Если свойству установить значение True,
 то в списке шрифтов диалога не будут отображаться пропорциональные шрифты.
