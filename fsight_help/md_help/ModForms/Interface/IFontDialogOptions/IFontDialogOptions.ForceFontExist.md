# IFontDialogOptions.ForceFontExist


## Синтаксис


ForceFontExist: Boolean;


## Описание


Свойство ForceFontExist определяет
 признак возможности выбора шрифта только из списка диалога.


## Комментарии


Если свойству установить значение True,
 то при указании отсутствующего в списке шрифта и нажатии на кнопку ОК будет выдаваться информационное
 сообщение о невозможности выбора данного шрифта. По умолчанию свойству
 установлено значение False, при
 этом после указания отсутствующего шрифта и нажатия кнопки ОК
 диалог будет закрыт, но шрифт изменен не будет.
