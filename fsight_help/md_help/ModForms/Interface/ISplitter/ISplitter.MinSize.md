# ISplitter.MinSize


## Синтаксис


MinSize: Integer;


## Описание


Свойство MinSize определяет минимальное расстояние, которое должно оставаться между компонентом [Splitter](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/Splitter.htm) и краем контейнера, в котором расположены элементы управления.
