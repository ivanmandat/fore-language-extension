# ISplitter.Beveled


## Синтаксис


Beveled: Boolean;


## Описание


Свойство Beveled определяет, будет ли отображаться рамка для компонента [Splitter](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/Splitter.htm).


## Комментарии


Если свойству установить значение True, то компонент будет заключен в рамку, иначе рамка отображаться не будет.
