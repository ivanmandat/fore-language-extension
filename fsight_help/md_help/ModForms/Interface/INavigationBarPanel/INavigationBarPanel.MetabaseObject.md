# INavigationBarPanel.MetabaseObject


## Синтаксис


MetabaseObject: String;


## Описание


Свойство MetabaseObject определяет идентификатор формы, содержимое которое будет отображено в области панели.


## Комментарии


В качестве значения свойства необходимо указать идентификатор любой формы репозитория. Если содержимое формы превышает размеры панели, то в области панели автоматически будут созданы полосы прокрутки.
