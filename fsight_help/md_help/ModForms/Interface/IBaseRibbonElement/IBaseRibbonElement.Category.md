# IBaseRibbonElement.Category


## Синтаксис


Category: [IRibbonCategory](../IRibbonCategory/IRibbonCategory.htm);


## Описание


Свойство Category возвращает вкладку, в области которой расположен элемент управления.


## Комментарии


Если элемент управления не принадлежит какой-либо вкладке, то свойство возвращает значение Null.
