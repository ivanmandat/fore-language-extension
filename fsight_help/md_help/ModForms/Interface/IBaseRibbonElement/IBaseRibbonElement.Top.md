# IBaseRibbonElement.Top


## Синтаксис


Top: Integer;


## Описание


Свойство Top возвращает расстояние от верхней границы родительского компонента до верхней границы текущего элемента управления.


## Комментарии


Расстояние измеряется в пикселях.
