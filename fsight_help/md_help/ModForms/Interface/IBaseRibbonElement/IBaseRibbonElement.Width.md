# IBaseRibbonElement.Width


## Синтаксис


Width: Integer;


## Описание


Свойство Width возвращает ширину элемента управления.


## Комментарии


Размер ширины элемента управления рассчитывается компонентом [Ribbon](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/Ribbon.htm) автоматически и зависит от расположения элемента управления, количества, расположения и настроек соседних элементов управления.


Ширина элемента управления измеряется в пикселях.


Для ряда элементов существует возможность установить фиксированную ширину. Для этого предназначены следующие свойства:


-
[IRibbonComboBox.InnerWidth](../IRibbonComboBox/IRibbonComboBox.InnerWidth.htm);


-
[IRibbonEdit.InnerWidth](../IRibbonEdit/IRibbonEdit.InnerWidth.htm);


-
[IRibbonProgressBar.InnerWidth](../IRibbonProgressBar/IRibbonProgressBar.InnerWidth.htm);


-
[IRibbonSlider.InnerWidth](../IRibbonSlider/IRibbonSlider.InnerWidth.htm).
