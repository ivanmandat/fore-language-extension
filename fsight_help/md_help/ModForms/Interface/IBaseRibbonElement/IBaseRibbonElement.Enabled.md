# IBaseRibbonElement.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет доступность элемента управления для пользователя.


## Комментарии


Если свойству установлено значение True, то элемент управления будет доступен для пользователя.
