# IBaseRibbonElement.Highlighted


## Синтаксис


Highlighted: Boolean;


## Описание


Свойство Highlighted возвращает признак подсветки текущего элемента управления.


## Комментарии


Свойство возвращает значение True, если текущий элемент управления подсвечен. Элемент подсвечивается в тот момент, когда в его области находится курсор мыши.
