# IBaseRibbonElement.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет признак видимости элемента управления.


## Комментарии


Если свойству установлено значение True, то элемент управления будет отображен на ленте. Если свойству установить значение False, то элемент управления будет скрыт.
