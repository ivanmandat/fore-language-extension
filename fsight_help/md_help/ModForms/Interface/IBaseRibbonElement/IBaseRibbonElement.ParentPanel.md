# IBaseRibbonElement.ParentPanel


## Синтаксис


ParentPanel: [IRibbonPanel](../IRibbonPanel/IRibbonPanel.htm);


## Описание


Свойство ParentPanel возвращает панель, на которой расположен элемент управления.


## Комментарии


Если элемент управления не располагается на панели, то свойство возвращает значение Null.
