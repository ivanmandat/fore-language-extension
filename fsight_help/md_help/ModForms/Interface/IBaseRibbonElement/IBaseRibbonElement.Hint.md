# IBaseRibbonElement.Hint


## Синтаксис


Hint: String;


## Описание


Свойство Hint определяет текст всплывающей подсказки для элемента управления.


## Комментарии


Всплывающая подсказка отображается при задержке указателя мыши в области элемента управления.
