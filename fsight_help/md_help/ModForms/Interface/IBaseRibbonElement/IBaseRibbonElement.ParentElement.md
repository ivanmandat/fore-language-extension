# IBaseRibbonElement.ParentElement


## Синтаксис


ParentElement: [IBaseRibbonElement](IBaseRibbonElement.htm);


## Описание


Свойство ParentElement возвращает родительский элемент управления.


## Комментарии


Если текущий элемент управления не является дочерним по отношению к какому-либо другому элементу управления, то свойство возвращает значение Null.
