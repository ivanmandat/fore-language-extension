# IBaseRibbonElement.ParentGroup


## Синтаксис


ParentGroup: [IRibbonGroup](../IRibbonGroup/IRibbonGroup.htm);


## Описание


Свойство ParentGroup возвращает группу, в которую включен данный элемент управления.


## Комментарии


Свойство возвращает значение Null, если элемент управления не включен в какую-либо группу.
