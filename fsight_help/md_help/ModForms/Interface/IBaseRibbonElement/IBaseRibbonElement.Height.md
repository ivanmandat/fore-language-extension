# IBaseRibbonElement.Height


## Синтаксис


Height: Integer;


## Описание


Свойство Height возвращает высоту элемента управления.


## Комментарии


Размер высоты элемента управления рассчитывается компонентом [Ribbon](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/Ribbon.htm) автоматически и зависит от расположения элемента управления, количества, расположения и настроек соседних элементов управления.


Высота измеряется в пикселях.
