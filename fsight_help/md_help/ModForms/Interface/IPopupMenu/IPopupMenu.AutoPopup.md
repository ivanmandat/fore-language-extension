# IPopupMenu.AutoPopup


## Синтаксис


AutoPopup: Boolean;


## Описание


Свойство AutoPopup определяет признак автоматического появления контекстного меню по щелчку дополнительной кнопки мыши.


## Комментарии


Если свойству установлено значение False, то для отображения контекстного меню необходимо использовать метод [Popup](IPopupMenu.Popup.htm). По умолчанию свойству установлено значение True.
