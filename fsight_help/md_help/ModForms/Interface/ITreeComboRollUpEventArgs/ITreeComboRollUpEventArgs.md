# ITreeComboRollUpEventArgs


## Описание


Интерфейс ITreeComboEventArgs
 содержит свойства аргумента события, возникающего при свертывании раскрывающегося
 списка компонента [TreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeCombo.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Cancel](ITreeComboRollUpEventArgs.Cancel.htm)


		 Свойство Cancel возвращает
		 признак закрытия раскрывающегося списка компонента по клавише
		 ESC.
