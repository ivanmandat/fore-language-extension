# ITreeComboRollUpEventArgs.Cancel


## Синтаксис


Cancel: Boolean;


## Описание


Свойство Cancel возвращает признак
 закрытия раскрывающегося списка компонента по клавише ESC.


## Комментарии


Свойство возвращает значение True,
 если закрытие списка осуществлено в результате нажатия клавиши ESC и False во всех остальных случаях.
