# ITrackBar.SliderVisible


## Синтаксис


SliderVisible: Boolean;


## Описание


Свойство SliderVisible определяет, будет ли видимым указатель компонента. Если свойству установлено значение True, указатель отображается.


По умолчанию свойство имеет значение True.
