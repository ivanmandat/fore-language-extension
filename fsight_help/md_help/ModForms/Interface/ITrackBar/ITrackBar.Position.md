# ITrackBar.Position


## Синтаксис


Position: Integer;


## Описание


Свойство Position определяет текущее положение ползунка.


## Комментарии


Значение свойства Position может изменяться в пределах, задаваемых значениями свойств [Min](ITrackBar.Min.htm) и [Max](ITrackBar.Max.htm).
