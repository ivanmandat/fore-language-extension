# ITrackBar.Frequency


## Синтаксис


Frequency: Integer;


## Описание


Свойство Frequency определяет
 значение, используемое при расчете количества интервалов и места расположения
 меток в области компонента.


## Комментарии


Свойство Frequency используется,
 если свойство [TickStyle](ITrackBar.TickStyle.htm)
 = Auto. Количество интервалов, располагаемых между метками, вычисляется
 по формуле: ([Max](ITrackBar.Max.htm)-[Min](ITrackBar.Min.htm))/Frequency. Если разница ([Max](ITrackBar.Max.htm)-[Min](ITrackBar.Min.htm))
 не кратна значению Frequency,
 то количество интервалов будет округлено в большую сторону до ближайшего
 целого. Место расположение меток вычисляется автоматически, симметрия
 расположения может быть не соблюдена.
