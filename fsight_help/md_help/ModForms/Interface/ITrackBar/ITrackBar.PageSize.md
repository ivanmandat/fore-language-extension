# ITrackBar.PageSize


## Синтаксис


PageSize: Integer;


## Описание


Свойство PageSize определяет
 смещение ползунка при нажатии клавиш PAGE UP, PAGE DOWN или при щелчке
 мышью по краям шкалы.
