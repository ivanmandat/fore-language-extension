# IDateTimePicker.Reset


## Синтаксис


Reset([Empty: Boolean = True]);


## Параметры


Empty — необязательный параметр, определяющий очистку компонента. Если Empty = False, то значение даты восстанавливается, True - очищается.


## Описание


Метод Reset осуществляет очистку даты компонента. Способ очистки определяется значением параметра Empty.
