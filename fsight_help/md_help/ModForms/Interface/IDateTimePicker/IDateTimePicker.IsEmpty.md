# IDateTimePicker.IsEmpty


## Синтаксис


IsEmpty: Boolean;


## Описание


Свойство IsEmpty возвращает
 признак изменения даты в компоненте.


## Комментарии


Свойство возвращает значение True,
 если после вызова метода [Reset](IDateTimePicker.Reset.htm),
 либо нажатия клавиши DELETE при [AllowEmpty](IDateTimePicker.AllowEmpty.htm)=True, значение даты (времени) компонента
 не изменялось. Значение даты (времени) считается изменившимся, если изменены
 все три компоненты даты (времени).
