# IDateTimePicker.Checked


## Синтаксис


Checked: Boolean;


## Описание


Свойство Checked определяет,
 отмечен ли флажок компонента.


## Комментарии


Свойство актуально, если свойству [ShowCheckBox](IDateTimePicker.ShowCheckbox.htm)
 установлено значение True.


Допустимое значение:


	- True. Флажок установлен,
	 в компоненте могут быть изменены дата/время;


	- False. Флажок снят,
	 доступ для изменения даты/времени запрещен.
