# IDateTimePicker.DateFormat


## Синтаксис


DateFormat: [CalendarDateFormat](../../Enums/CalendarDateFormat.htm);


## Описание


Свойство DateFormat определяет
 формат отображения даты в компоненте.


## Комментарии


Свойство актуально в том случае, если свойству [Kind](IDateTimePicker.Kind.htm)
 компонента установлено значение Date.
