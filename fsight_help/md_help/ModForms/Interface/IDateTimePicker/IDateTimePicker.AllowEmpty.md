# IDateTimePicker.AllowEmpty


## Синтаксис


AllowEmpty: Boolean;


## Описание


Свойство AllowEmpty определяет возможность очистки даты нажатием кнопки "Del".


## Комментарии


Если свойство установлено в True, то будет возможность очистить дату кнопкой "Del". По умолчанию AllowEmpty = False.
