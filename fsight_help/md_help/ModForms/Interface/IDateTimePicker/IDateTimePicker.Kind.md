# IDateTimePicker.Kind


## Синтаксис


Kind: [DateTimePickerKind](../../Enums/DateTimePickerKind.htm);


## Описание


Свойство Kind определяет, что должен отображать компонент: дату или время.


## Комментарии


При отображении даты - свойству [DateMode](IDateTimePicker.DateMode.htm) автоматически устанавливается значение ComboBox, при отображении времени [DateMode](IDateTimePicker.DateMode.htm) = UpDown.
