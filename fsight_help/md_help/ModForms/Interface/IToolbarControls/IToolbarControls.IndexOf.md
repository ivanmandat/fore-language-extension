# IToolbarControls.IndexOf


## Синтаксис


IndexOf(Item: [IComponent](../IComponent/IComponent.htm)): Integer;


## Параметры


Item - компонент, расположенный на панели инструментов.


## Описание


Метод IndexOf возвращает индекс компонента, расположенного на панели инструментов.


## Комментарии


Если компонент не расположен на данной панели инструментов, то метод возвращает значение «-1».
