# IToolbarControls.Item


## Синтаксис


Item(Index: Integer): [IComponent](../IComponent/IComponent.htm);


## Параметры


Index - индекс компонента, расположенного на панели инструментов.


## Описание


Свойство Item возвращает параметры компонента, расположенного на панели инструментов.


## Комментарии


Индекс компонента передается посредством параметра Index.
