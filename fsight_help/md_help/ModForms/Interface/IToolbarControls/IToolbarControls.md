# IToolbarControls


## Описание


Интерфейс IToolbarControls содержит
 свойства и методы для работы с коллекцией элементов управления, расположенных
 на панели инструментов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IToolbarControls.Count.htm)
		 Свойство Count возвращает
		 количество компонентов, расположенных на панели инструментов.


		 ![](../../Property_Image.gif)
		 [Item](IToolbarControls.Item.htm)
		 Свойство Item возвращает
		 параметры компонента, расположенного на панели инструментов.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IToolbarControls.Add.htm)
		 Метод Add осуществляет
		 добавление компонента на панель инструментов.


		 ![](../../Sub_Image.gif)
		 [Clear](IToolbarControls.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех компонентов, расположенных на панели инструментов.


		 ![](../../Sub_Image.gif)
		 [Delete](IToolbarControls.Delete.htm)
		 Метод Delete осуществляет
		 удаление компонента с панели инструментов.


		 ![](../../Sub_Image.gif)
		 [IndexOf](IToolbarControls.IndexOf.htm)
		 Метод IndexOf возвращает
		 индекс компонента, расположенного на панели инструментов.


		 ![](../../Sub_Image.gif)
		 [Insert](IToolbarControls.Insert.htm)
		 Метод Insert осуществляет
		 вставку компонента в указанную позицию на панели инструментов.
