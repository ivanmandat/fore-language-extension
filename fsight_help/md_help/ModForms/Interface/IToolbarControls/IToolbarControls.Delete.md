# IToolbarControls.Delete


## Синтаксис


Delete(Value: Integer);


## Параметры


Value - индекс компонента, который необходимо удалить.


## Описание


Метод Delete осуществляет удаление компонента с панели инструментов. Индекс компонента передается посредством параметра Value.
