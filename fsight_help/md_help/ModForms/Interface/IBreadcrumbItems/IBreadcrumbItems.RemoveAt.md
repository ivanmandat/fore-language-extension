# IBreadcrumbItems.RemoveAt


## Синтаксис


RemoveAt(Index: Integer);


## Параметры


Index - индекс позиции элемента, который необходимо удалить.


## Описание


Метод RemoveAt осуществляет удаление элемента, индекс которого передается посредством параметра Index.
