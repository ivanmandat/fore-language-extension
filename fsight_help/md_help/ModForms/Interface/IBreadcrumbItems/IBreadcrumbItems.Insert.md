# IBreadcrumbItems.Insert


## Синтаксис


Insert(Index: Integer): [IBreadcrumbItem](../IBreadcrumbItem/IBreadcrumbItem.htm);


## Параметры


Index - индекс позиции, в которую осуществляется вставка нового элемента. Индексация элементов начинается с 0.


## Описание


Метод Insert осуществляет вставку нового элемента в указанную позицию. Индекс позиции передается посредством параметра Index.
