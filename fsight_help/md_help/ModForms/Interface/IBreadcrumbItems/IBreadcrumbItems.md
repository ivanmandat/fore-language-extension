# IBreadcrumbItems


## Описание


Интерфейс IBreadcrumbItems содержит
 свойства и методы для работы с коллекцией элементов компонента [Breadcrumb](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/Breadcrumb.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IBreadcrumbItems.Count.htm)
		 Свойство Count возвращает
		 количество элементов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IBreadcrumbItems.Item.htm)
		 Свойство Item возвращает
		 параметры указанного элемента.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IBreadcrumbItems.Add.htm)
		 Метод Add создает новый
		 элемент компонента [Breadcrumb](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/Breadcrumb.htm).


		 ![](../../Sub_Image.gif)
		 [Clear](IBreadcrumbItems.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех элементов коллекции.


		 ![](../../Sub_Image.gif)
		 [Insert](IBreadcrumbItems.Insert.htm)
		 Метод Insert осуществляет
		 вставку нового элемента в указанную позицию.


		 ![](../../Sub_Image.gif)
		 [Move](IBreadcrumbItems.Move.htm)
		 Метод Move осуществляет
		 перемещение указанного элемента в указанную позицию.


		 ![](../../Sub_Image.gif)
		 [RemoveAt](IBreadcrumbItems.RemoveAt.htm)
		 Метод RemoveAt осуществляет
		 удаление указанного элемента.
