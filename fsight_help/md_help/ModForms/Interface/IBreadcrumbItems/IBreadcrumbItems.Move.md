# IBreadcrumbItems.Move


## Синтаксис


Move(IndexFrom: Integer; IndexTo: Integer);


## Параметры


IndexFrom - индекс позиции элемента, который необходимо переместить.


IndexTo - индекс позиции, в которую необходимо переместить элемент.


## Описание


Метод Move осуществляет перемещение указанного элемента в указанную позицию.
