# IFileSaveDialog.CreatePrompt


## Синтаксис


CreatePrompt: Boolean;


## Описание


Свойство CreatePrompt определяет,
 будет ли выводиться дополнительный предупреждающий диалог при попытке
 сохранения в несуществующий файл.


## Комментарии


По умолчанию свойству установлено значение False,
 при этом дополнительный диалог не выводится. При выборе несуществующего
 файла разрешено его создание и после закрытия диалога сохранения метод
 [Execute](../IFileDialog/IFileDialog.Execute.htm) вернет значение
 True.


Если свойству CreatePrompt установить
 значение True, то дополнительный
 диалог будет выводится. Если в диалоге нажать кнопку «Да», то будет разрешено
 создание нового файла, диалог сохранения будет закрыт и метод [Execute](../IFileDialog/IFileDialog.Execute.htm)
 вернет значение True. Если в диалоге
 нажать кнопку «Нет», то фокус будет возвращен в диалог сохранения для
 выбора другого файл.
