# IFileSaveDialog.OverwritePrompt


## Синтаксис


OverwritePrompt: Boolean;


## Описание


Свойство OverwritePrompt определяет,
 будет ли выводиться дополнительный предупреждающий диалог при попытке
 сохранения в существующий файл.


## Комментарии


По умолчанию свойству установлено значение True,
 при этом дополнительный диалог выводится. Если в диалоге нажать кнопку
 «Да», то будет разрешена перезапись выбранного файла, диалог сохранения
 будет закрыт и метод [Execute](../IFileDialog/IFileDialog.Execute.htm) вернет
 значение True. Если в диалоге
 нажать кнопку «Нет», то фокус будет возвращен в диалог сохранения для
 выбора другого файла.


Если свойству OverwritePrompt
 установить значение False, то
 дополнительный диалог не выводится. При выборе существующего файла его
 перезапись разрешена и после закрытия диалога сохранения метод [Execute](../IFileDialog/IFileDialog.Execute.htm) вернет значение True.
