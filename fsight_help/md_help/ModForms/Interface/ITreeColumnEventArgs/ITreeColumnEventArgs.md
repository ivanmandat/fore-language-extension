# ITreeColumnEventArgs


## Описание


Интерфейс ITreeColumnEventArgs
 содержит свойства аргумента события, происходящего при щелчке основной
 кнопкой мыши по заголовку столбца компонентов [TreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeCombo.htm)
 или [TreeList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeList.htm).
