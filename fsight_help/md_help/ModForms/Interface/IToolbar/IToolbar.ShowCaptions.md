# IToolbar.ShowCaptions


## Синтаксис


ShowCaptions: Boolean;


## Описание


Свойство ShowCaption определяет, будут ли отображаться заголовки на кнопках, расположенных на [ToolBar](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ToolBar.htm).


## Комментарии


Если свойству ShowCaption установлено значение True, кнопки в панели инструментов [ToolBar](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ToolBar.htm) будут отображаться с заголовками, иначе без заголовков.
