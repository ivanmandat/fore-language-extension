# IToolbar.AllowClose


## Синтаксис


AllowClose: Boolean;


## Описание


Свойство AllowClose определяет возможность закрытия панели инструментов [ToolBar](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ToolBar.htm) в случае, когда панель не привязана к форме, а расположена независимо в области формы.


## Комментарии


Если свойству установлено значение True, панель инструментов можно закрыть, иначе - такой возможности нет.
