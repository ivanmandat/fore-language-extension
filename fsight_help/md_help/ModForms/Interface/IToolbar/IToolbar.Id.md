# IToolbar.Id


## Синтаксис


Id: Integer;


## Описание


Свойство Id определяет уникальный идентификатор панели инструментов.


## Комментарии


Данный идентификатор используется при сохранении настроек, учитывающих расположение панелей инструментов.
