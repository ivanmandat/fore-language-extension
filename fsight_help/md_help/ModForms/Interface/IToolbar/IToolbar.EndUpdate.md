# IToolbar.EndUpdate


## Синтаксис


EndUpdate;


## Описание


Метод EndUpdate отключает режим запрета перерисовки компонента [ToolBar](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ToolBar.htm).


## Комментарии


Метод используется для избежания мерцания компонента при изменении его свойств. Данный метод используется совместно с методом [BeginUpdate](IToolbar.BeginUpdate.htm).
