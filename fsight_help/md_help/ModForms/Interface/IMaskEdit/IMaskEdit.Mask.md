# IMaskEdit.Mask


## Синтаксис


Mask: String;


## Описание


Свойство Mask определяет маску
 поля ввода, указывающую доступные данные при вводе.


## Комментарии


Символы маски указываются в позициях, предназначенных для ввода данных.
 Данные позиции определяются в [шаблоне](IMaskEdit.InputTemplate.htm)
 с использованием символа, указанного в свойстве [PlaceholderChar](IMaskEdit.PlaceholderChar.htm).


Символы, определяющие доступные данные при вводе:


		 Символ
		 Значение в маске


		 a/A
		 Возможен ввод букв и цифр.


		 d/D
		 Возможен ввод только цифр.


		 c/C
		 Возможен ввод только букв.
