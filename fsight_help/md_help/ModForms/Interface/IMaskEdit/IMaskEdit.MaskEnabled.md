# IMaskEdit.MaskEnabled


## Синтаксис


MaskEnabled: Boolean;


## Описание


Свойство MaskEnabled определяет,
 будет ли компонент работать в режиме ввода данных в соответствии с установленным
 [шаблоном](IMaskEdit.InputTemplate.htm) и [маской](IMaskEdit.Mask.htm)
 ввода.


## Комментарии


Если свойству установлено значение True
 - данные вводятся в соответствии с установленным шаблоном и маской ввода,
 иначе компонент работает аналогично компоненту [EditBox](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/EditBox.htm).
