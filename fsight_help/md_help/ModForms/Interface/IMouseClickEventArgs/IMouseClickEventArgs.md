# IMouseClickEventArgs


## Описание


Интерфейс IMouseClickEventArgs
 содержит свойства аргумента события, возникающего, когда была нажата или
 отпущена кнопка мыши.


## Свойства, унаследованные от [IMouseEventArgs](../IMouseEventArgs/IMouseEventArgs.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Button](../IMouseEventArgs/IMouseEventArgs.Button.htm)
		 Свойство Button возвращает
		 код кнопки мыши, которой было осуществлено какое-либо действие.


		 ![](../../Property_Image.gif)
		 [Cancel](../IMouseEventArgs/IMouseEventArgs.Cancel.htm)
		 Свойство Cancel определяет
		 признак отмены дальнейшего действия, связанного с мышью.


		 ![](../../Property_Image.gif)
		 [pPoint](../IMouseEventArgs/IMouseEventArgs.pPoint.htm)
		 Свойство pPoint возвращает
		 координаты курсора мыши в клиентской части компонента во время
		 выполнения действий мышью.


		 ![](../../Property_Image.gif)
		 [ShiftState](../IMouseEventArgs/IMouseEventArgs.ShiftState.htm)
		 Свойство ShiftState
		 возвращает значение, соответствующее состоянию вспомогательных
		 клавиш (SHIFT, ALT и CTRL) и кнопок мыши во время выполнения действий мышью.
