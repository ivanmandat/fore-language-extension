# IConditionGroup.Operator_


## Синтаксис


Operator_: [ConditionOperatorType](../../Enums/ConditionOperatorType.htm);


## Описание


Свойство Operator_ возвращает
 оператор, который используется в выражении группы элементов.


## Комментарии


Если группа состоит из нескольких элементов, то возвращается оператор
 только первого элемента.


Для получения группы элементов по индексу используйте свойство [IConditionGroups.Item](../IConditionGroups/IConditionGroups.Item.htm).
