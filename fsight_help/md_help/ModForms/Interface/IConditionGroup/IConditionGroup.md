# IConditionGroup


## Иерархия наследования


IConditionGroup


## Описание


Интерфейс IConditionGroup используется
 для работы с группой элементов компонента [CtrlGroupConditions](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/CtrlGroupConditions.htm).


## Комментарии


Группа элементов может состоять из набора сгруппированных элементов
 или одного элемента.
