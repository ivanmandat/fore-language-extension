# IMemo.WantTabs


## Синтаксис


WantTabs: Boolean;


## Описание


Свойство WantTabs определяет возможность вставки в редактируемый текст символа табуляции (нажатие клавиши TAB).


## Комментарии


По умолчанию свойству установлено значение False, при этом клавиша табуляции игнорируется.
