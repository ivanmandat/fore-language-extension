# IMemo.ScrollBars


## Синтаксис


ScrollBars: [ControlScrollStyle](../../Enums/ControlScrollStyle.htm);


## Описание


Свойство ScrollBars определяет
 наличие полос прокрутки текста в поле компонента.


## Комментарии


По умолчанию ScrollBars = None.
