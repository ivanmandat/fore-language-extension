# IMemo.WordWrap


## Синтаксис


WordWrap: Boolean;


## Описание


Свойство WordWrap определяет будут ли переноситься слова, превышающие длину компонента, на новую строку.


## Комментарии


Если свойство имеет значение True, то слова будут переноситься на новую строку.


Примечание. Слова будут переноситься на новую строку только при отсутствии горизонтальной полосы прокрутки.
