# IDocumentPrinter.Source


## Синтаксис


Source: [IDocumentSource](../IDocumentSource/IDocumentSource.htm);


## Описание


Свойство Source определяет источник, в котором находится документ для печати.


## Комментарии


В качестве источника могут быть установлены компоненты [UiReport](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiReport.htm), [UiWorkspace](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiWorkspace.htm), [UiErAnalyzer](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiErAnalyzer.htm).
