# ICommandEventArgs


## Описание


Интерфейс ICommandEventArgs
 содержит свойства аргумента события [OnCommand](../../Class/Form/Form.OnCommand.htm),
 наступающего при отправке форме какой-либо команды.

		 [Argument](ICommandEventArgs.Argument.htm)

		 ![](../../Property_Image.gif)
		 [Command](ICommandEventArgs.Command.htm)

		 Свойство Result определяет
