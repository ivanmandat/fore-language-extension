# ICommandEventArgs.Result


## Синтаксис


Result: Variant;


## Описание


Свойство Result определяет значение
 результата выполнения команды, посланной форме.


## Комментарии


Используется, если сообщения отправлено посредством метода [SendCommand](../IFormControl/IFormControl.SendCommand.htm).
