# IProgressBar.Smooth


## Синтаксис


Smooth: Boolean;


## Описание


Свойство Smooth определяет вид
 индикатора.


## Комментарии


Если свойству установлено значение True,
 то индикатор отображается в виде монолитного прямоугольника, длина которого
 определяется значением свойства [Position](IProgressBar.Position.htm),
 иначе индикатор отображается в виде нескольких разделенных прямоугольников.


Примечание.
 При работе в операционных системах, использующих тему оформления Aero
 (Windows Vista, Windows 7), используется плавная отрисовка полосы индикатора
 процесса. Свойство Smooth при
 этом не влияет на вид индикатора.
