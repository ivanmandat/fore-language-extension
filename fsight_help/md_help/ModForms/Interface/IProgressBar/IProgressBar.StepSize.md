# IProgressBar.StepSize


## Синтаксис


StepSize: Integer;


## Описание


Свойство StepSize определяет величину, на которую будет смещен индикатор при вызове метода [StepIt](IProgressBar.StepIt.htm).


## Комментарии


Значение свойства StepSize может быть как положительным, так и отрицательным.
