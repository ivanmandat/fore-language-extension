# IProgressBar.Position


## Синтаксис


Position: Integer;


## Описание


Свойство Position определяет
 текущее положения индикатора выполнения.


## Комментарии


Значение Position должно лежать
 в промежутке между значениями свойств [Min](IProgressBar.Min.htm)
 и [Max](IProgressBar.Max.htm).
