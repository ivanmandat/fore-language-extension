# IProgressBar.StepBy


## Синтаксис


StepBy(Delta: Integer);


## Параметры


Delta - числовое значение, определяющее шаг перемещения.


## Описание


Метод StepBy осуществляет изменение позиции индикатора на величину, указанную во входном параметре Delta.
