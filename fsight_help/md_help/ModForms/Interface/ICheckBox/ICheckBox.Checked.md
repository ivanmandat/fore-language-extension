# ICheckBox.Checked


## Синтаксис


Checked: Boolean;


## Описание


Свойство Checked определяет
 состояние флажка компонента.


## Комментарии


Если свойству установить значение True,
 то флажок будет отмечен, если False
 - не отмечен.


Если для компонента свойство [AllowGrayed](ICheckBox.AllowGrayed.htm)
 имеет значение True, то для флажка
 можно будет установить промежуточное состояние. Для этого необходимо свойству
 [State](ICheckBox.State.htm) установить
 значение [CheckBoxState.Grayed](../../Enums/CheckBoxState.htm).
