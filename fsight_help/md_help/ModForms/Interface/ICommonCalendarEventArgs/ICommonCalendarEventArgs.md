# ICommonCalendarEventArgs


## Описание


Интерфейс ICommonCalendarEventArgs
 содержит свойства аргумента события [OnChange](../ICommonCalendar/ICommonCalendar.OnChange.htm),
 наступающего при изменении значения календаря.
