# IPPButtonGroup.Remove


## Синтаксис


Remove(pButton: [IPPButton](../IPPButton/IPPButton.htm));


## Параметры


pButton. Кнопка, которую необходимо удалить из группы.


## Описание


Метод Remove удаляет указанную кнопку из группы.
