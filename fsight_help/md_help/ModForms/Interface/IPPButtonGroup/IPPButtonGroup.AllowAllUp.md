# IPPButtonGroup.AllowAllUp


## Синтаксис


AllowAllUp: Button;


## Описание


Свойство AllowAllUp определяет,
 могут ли все кнопки, составляющие группу, находиться одновременно в ненажатом
 состоянии.


## Комментарии


По умолчанию свойству AllowAllUp
 установлено значение False, при
 этом в группе какая-либо кнопка всегда должна находиться в нажатом состоянии.


Если свойству установить значение True,
 то будет возможность перевести все кнопки в ненажатое состояние. Изменение
 состояний кнопок осуществляется путем щелчка по ним, либо нажатием
 клавиши ПРОБЕЛ в то время, когда кнопка имеет фокус. Также управлять состоянием
 кнопок можно изменяя значение свойства [IPPSwitchButton.Down](../IPPSwitchButton/IPPSwitchButton.Down.htm).
