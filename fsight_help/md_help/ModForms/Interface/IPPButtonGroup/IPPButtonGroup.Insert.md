# IPPButtonGroup.Insert


## Синтаксис


Insert(Index: Integer; pButton: [IPPButton](../IPPButton/IPPButton.htm));


## Параметры


Index.
 Индекс позиции, в которую необходимо вставить кнопку.


pButton.
 Кнопка, которая будет добавлена в группу.


## Описание


Метод Insert вставляет кнопку
 в указанную позицию в группе.


## Комментарии


В качестве значения параметра pButton
 необходимо указать кнопку, реализуемую классом [PPSwitchButton](../../Class/PPSwitchButton/PPSwitchButton.htm).
 При добавлении кнопки в группу в качестве родительского компонента для
 нее автоматически будет установлен текущий компонент PPButtonGroup.
