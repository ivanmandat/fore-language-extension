# IPPButtonGroup.RemoveByIndex


## Синтаксис


RemoveByIndex(Index: Integer);


## Параметры


Index. Индекс кнопки, которую необходимо удалить из группы.


## Описание


Метод RemoveByIndex удаляет кнопку из группы по индексу.
