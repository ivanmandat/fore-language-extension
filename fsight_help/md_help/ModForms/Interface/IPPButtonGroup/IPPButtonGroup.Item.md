# IPPButtonGroup.Item


## Синтаксис


Item(Index: Integer): [IPPButton](../IPPButton/IPPButton.htm);


## Параметры


Index. Индекс кнопки в группе.


## Описание


Свойство Item возвращает параметры кнопки, индекс которой передается посредством параметра Index.
