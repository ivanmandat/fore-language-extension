# IMDICloseEventArgs


## Описание


Интерфейс IMDICloseEventArgs
 содержит свойства аргумента события, возникающего в момент закрытия дочерней
 MDI-формы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Allow](IMDICloseEventArgs.Allow.htm)


		 Свойство Allow определяет
		 возможность закрытия дочерней MDI-формы.


		 ![](../../Property_Image.gif)
		 [MDIChild](IMDICloseEventArgs.MDIChild.htm)


		 Свойство MDIChild возвращает
		 закрываемую дочернюю MDI-форму.
