# IWizardButtons.Done


## Синтаксис


Done: [IButton](../IButton/IButton.htm);


## Описание


Свойство Done определяет кнопку для выхода из мастера с сохранением изменений.


## Комментарии


Для данной кнопки будет установлен текст «Готово».


В список доступных компонентов входят все компоненты [Button](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/Button.htm) и [ImageButton](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ImageButton.htm), расположенные на форме.
