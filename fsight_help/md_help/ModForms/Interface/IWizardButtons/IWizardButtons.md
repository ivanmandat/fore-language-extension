# IWizardButtons


## Описание


Интерфейс IWizardButtons содержит
 свойства, определяющие кнопки формы, используемые для навигации по страницам
 компонента [Wizard](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/Wizard.htm).


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Back](IWizardButtons.Back.htm)


		 Свойство Back определяет
		 кнопку, при нажатии на которую происходит переход на предыдущую
		 страницу мастера.


		 ![](../../Property_Image.gif)
		 [Cancel](IWizardButtons.Cancel.htm)


		 Свойство Cancel определяет
		 кнопку для выхода из мастера без сохранения изменений.


		 ![](../../Property_Image.gif)
		 [Done](IWizardButtons.Done.htm)


		 Свойство Done определяет
		 кнопку для выхода из мастера с сохранением изменений.


		 ![](../../Property_Image.gif)
		 [Next](IWizardButtons.Next.htm)


		 Свойство Next определяет
		 кнопку, при нажатии на которую происходит переход на следующую
		 страницу мастера.
