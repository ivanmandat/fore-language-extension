# IWizardButtons.Back


## Синтаксис


Back: [IButton](../IButton/IButton.htm);


## Описание


Свойство Back определяет кнопку, при нажатии на которую происходит переход на предыдущую страницу мастера.


## Комментарии


Для данной кнопки будет установлен текст «< Назад». Смена доступности кнопки происходит автоматически.


В список доступных компонентов входят все компоненты [Button](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/Button.htm) и [ImageButton](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ImageButton.htm), расположенные на форме.
