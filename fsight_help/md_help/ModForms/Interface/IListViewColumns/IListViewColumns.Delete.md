# IListViewColumns.Delete


## Синтаксис


Delete(ColumnIndex: Integer);


## Параметры


ColumnIndex. Индекс столбца, который необходимо удалить.


## Описание


Метод Delete осуществляет удаление столбца, индекс которого передается в качестве входного параметра ColumnIndex.
