# IListViewColumns


## Описание


Интерфейс IListViewColumns содержит
 свойства и методы коллекции столбцов компонента [ListView](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ListView.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IListViewColumns.Count.htm)


		 Свойство Count возвращает
		 количество столбцов компонента.


		 ![](../../Property_Image.gif)
		 [Item](IListViewColumns.Item.htm)


		 Свойство Item возвращает
		 параметры указанного столбца компонента.


		 ![](../../Property_Image.gif)
		 [ListView](IListViewColumns.ListView.htm)


		 Свойство ListView возвращает
		 объект-родитель.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IListViewColumns.Add.htm)


		 Метод Add осуществляет
		 добавление нового столбца.


		 ![](../../Sub_Image.gif)
		 [Clear](IListViewColumns.Clear.htm)


		 Метод Clear осуществляет
		 удаление всех столбцов компонента.


		 ![](../../Sub_Image.gif)
		 [Delete](IListViewColumns.Delete.htm)


		 Метод Delete осуществляет
		 удаление указанного столбца компонента.
