# IKeyPreviewEventArgs


## Описание


Интерфейс IKeyPreviewEventArgs
 содержит свойства аргумента события [OnKeyPreview](../IControl/IControl.OnKeyPreview.htm),
 происходящего перед событиями, связанными с нажатием клавиш.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Cancel](IKeyPreviewEventArgs.Cancel.htm)
		 Свойство Cancel определяет
		 признак происхождения дальнейшего события, связанного с нажатием
		 клавиш.


		 ![](../../Property_Image.gif)
		 [KeyMessage](IKeyPreviewEventArgs.KeyMessage.htm)
		 Свойство KeyMessage
		 возвращает значение, указывающее в результате чего было сгенерировано
		 событие [OnKeyPreview](../IControl/IControl.OnKeyPreview.htm).


## Свойства, унаследованные от [IKeyEventArgs](../IKeyEventArgs/IKeyEventArgs.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Key](../IKeyEventArgs/IKeyEventArgs.Key.htm)
		 Свойство Key определяет
		 код клавиши, которая была нажата или отпущена.


		 ![](../../Property_Image.gif)
		 [ShiftState](../IKeyEventArgs/IKeyEventArgs.ShiftState.htm)
		 Свойство ShiftState
		 возвращает значение, соответствующее состоянию вспомогательных
		 клавиш (SHIFT, ALT и CTRL) во время выполнения действий с клавиатуры.
