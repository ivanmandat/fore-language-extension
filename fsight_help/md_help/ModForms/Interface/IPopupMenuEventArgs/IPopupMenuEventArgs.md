# IPopupMenuEventArgs


## Описание


Интерфейс IPopupMenuEventArgs
 содержит свойства аргумента события, возникающего при вызове контекстного
 меню.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Control](IPopupMenuEventArgs.Control.htm)


		 Свойство Control возвращает
		 компонент, над которым было вызвано контекстное меню.


		 ![](../../Property_Image.gif)
		 [Point](IPopupMenuEventArgs.Point.htm)


		 Свойство Point возвращает
		 точку в области компонента [Control](IPopupMenuEventArgs.Control.htm),
		 в которой было вызвано контекстное меню.
