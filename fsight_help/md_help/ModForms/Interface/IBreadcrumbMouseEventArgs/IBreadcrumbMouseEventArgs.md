# IBreadcrumbMouseEventArgs


## Описание


Интерфейс IBreadcrumbMouseEventArgs содержит свойства аргумента событий, возникающих, когда был осуществлен щелчок кнопкой мыши по элементу, либо по дополнительной кнопке элемента компонента.
