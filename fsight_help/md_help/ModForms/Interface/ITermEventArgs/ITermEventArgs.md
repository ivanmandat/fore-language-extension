# ITermEventArgs


## Описание


Интерфейс ITermEventArgs содержит
 свойства аргумента события, происходящего при щелчке по гиперссылке какого-либо
 терма.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Term](ITermEventArgs.Term.htm)


		 Свойство Term возвращает
		 терм, гиперссылка которого была выбрана.
