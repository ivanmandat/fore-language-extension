# IControl.ShowHint


## Синтаксис


ShowHint: Boolean;


## Описание


Свойство ShowHint определяет, будет ли появляться всплывающая подсказка при задержке курсора мыши над компонентом.


## Комментарии


Всплывающая подсказка появляется при задержке курсора мыши над компонентом в том случае, если ShowHint = True, и задан текст подсказки в свойстве [Hint](IControl.Hint.htm). Также всплывающая подсказка может появляться при ShowHint = True и не заданном тексте подсказки, если [ParentShowHint](IControl.ParentShowHint.htm) = True, а в родительском компоненте установлено ShowHint = True, и задан текст подсказки в свойстве [Hint](IControl.Hint.htm).


По умолчанию установлено значение True.
