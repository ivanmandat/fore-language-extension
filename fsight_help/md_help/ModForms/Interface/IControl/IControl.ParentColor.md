# IControl.ParentColor


## Синтаксис


ParentColor: Boolean;


## Описание


Свойство ParentColor определяет, будет ли для компонента заимствован цвет родительского компонента (компонента, на котором он расположен).


## Комментарии


Если свойству ParentColor установлено значение True, цвет компонента будет соответствовать родительскому.
