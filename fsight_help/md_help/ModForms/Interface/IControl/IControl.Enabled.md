# IControl.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет доступность компонента для пользователя.


## Комментарии


Чтобы компонент не был доступен, следует установить свойству Enabled значение False. Недоступный компонент отображается серым цветом. Он игнорирует события, связанные с мышью, клавиатурой и события таймера. Чтобы восстановить доступность компонента необходимо установить Enabled значение True, после этого компонент начинает нормально отображаться и реагировать на действия пользователя.
