# IControl.Align


## Синтаксис


Align: [ControlAlign](../../Enums/ControlAlign.htm);


## Описание


Свойство Align определяет выравнивание компонента в рамках содержащего его родительского компонента.


## Комментарии


В качестве родительского компонента может выступать форма, компоненты GroupBox, Panel и т.п. компоненты, представляющие собой контейнер для других компонентов.


В зависимости от значения свойства Align поведение компонента при изменении размеров родительского компонента будет различным:


-
размеры компонента будут неизменными;


-
компонент будет занимать всю доступную площадь родительского компонента;


-
компонент будет располагаться, занимая всю область вдоль верхнего, нижнего, левого или правого края родительского компонента.


По умолчанию свойству установлено значение ControlAlign.None, при этом размеры компонента остаются неизменными.
