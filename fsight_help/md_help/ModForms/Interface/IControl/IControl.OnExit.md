# OnExit


## Синтаксис


OnExit(Sender: Object; Args: [IEventArgs](ForeSys.chm::/Interface/IEventArgs/IEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnExit наступает в момент
 потери фокуса компонентом (передача фокуса другому компоненту).


## Комментарии


Событие не наступает при переключении между формами или приложениями.


При переключениях между компонентами, расположенными в разных контейнерах,
 событие OnExit наступает сначала для компонента, а затем для контейнера,
 в котором содержится компонент.


Использование события OnExit недоступно
 для компонента [Label](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/Label.htm).
