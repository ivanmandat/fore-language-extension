# IControl.Left


## Синтаксис


Left: Integer;


## Описание


Свойство Left определяет координату левого края компонента в пикселях.


## Комментарии


За начало отсчета берется левая граница родительского компонента (формы, панели). Для формы отсчет ведется от левого края экрана.
