# IControl.Top


## Синтаксис


Top: Integer;


## Описание


Свойство Top определяет координату верхнего края компонента в пикселях.


## Комментарии


За нулевую точку берется верхняя граница клиентской области родительского компонента (формы, панели). Для формы отсчет ведется от верхнего края экрана.
