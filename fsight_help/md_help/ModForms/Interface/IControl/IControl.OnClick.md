# OnClick


## Синтаксис


OnClick(Sender: Object; Args: [IMouseEventArgs](../IMouseEventArgs/IMouseEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnClick наступает, если
 пользователь щёлкнул в области компонента.


## Комментарии


Событие происходит также, если:


	- Для компонентов, изменяющих свое состояние, нажата
	 клавиша ПРОБЕЛ, когда компонент находится в фокусе;


	- Нажата клавиша ENTER, и активная форма имеет кнопку
	 по умолчанию (кнопку, для которой значение свойства [DefaultButton](../IButton/IButton.DefaultButton.htm)
	 установлено в True);


	- Нажата клавиша ESC, и активная форма имеет кнопку
	 прерывания (кнопку, для которой значение свойства [CancelButton](../IButton/IButton.CancelButton.htm)
	 установлено в True);


	- Нажаты клавиши быстрого доступа («горячая клавиша»)
	 к компоненту.


Если событие было сгенерировано без использования мыши, то свойство
 аргумента события [Button](../IMouseEventArgs/IMouseEventArgs.Button.htm)
 вернет значение «-1», а в свойстве
 [pPoint](../IMouseEventArgs/IMouseEventArgs.pPoint.htm) будут
 содержаться координаты [0, 0].
