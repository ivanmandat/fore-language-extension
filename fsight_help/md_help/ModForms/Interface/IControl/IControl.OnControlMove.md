# OnControlMove


## Синтаксис


OnControlMove(Sender: Object; Args: [IPositionEventArgs](../IPositionEventArgs/IPositionEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnControlMove наступает
 при перемещении компонента.


## Комментарии


Используя свойство аргумента события [Point](../IPositionEventArgs/IPositionEventArgs.Point.htm)
 можно получить новые координаты верхнего левого угла компонента.
