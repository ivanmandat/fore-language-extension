# IControl.ParentFont


## Синтаксис


ParentFont: Boolean;


## Описание


Свойство ParentFont определяет, будет ли для данного компонента использоваться шрифт (установленный с помощью свойства Font) родительского компонента - компонента, на котором он расположен.


## Комментарии


Если у всех компонентов формы свойство ParentFont имеет значение True, то все атрибуты шрифта каждого компонента будут определятся свойством [Font](IControl.Font.htm) формы. Изменение свойства [Font](IControl.Font.htm) какого-либо компонента влечет за собой автоматическое изменение свойства ParentFont в состояние False.
