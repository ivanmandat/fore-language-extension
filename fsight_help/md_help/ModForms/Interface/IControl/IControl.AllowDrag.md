# IControl.AllowDrag


## Синтаксис


AllowDrag: Boolean;


## Описание


Свойство AllowDrag определяет возможность взять у компонента перетаскиваемый объект.


## Комментарии


При установке свойству AllowDrag значения True компонент имеет возможность отдать перетаскиваемые объекты, при установке cвойству значения False перетаскивание объектов с этого компонента запрещено.
