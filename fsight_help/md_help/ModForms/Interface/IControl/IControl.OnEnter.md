# OnEnter


## Синтаксис


OnEnter(Sender: Object; Args: [IEventArgs](ForeSys.chm::/Interface/IEventArgs/IEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnEnter наступает в
 момент получения фокуса компонентом.


## Комментарии


Событие не наступает при переключении между формами или приложениями.


При переключениях между компонентами, расположенными в разных контейнерах,
 событие OnEnter наступает сначала
 для контейнера, а затем для компонента, находящегося внутри контейнера.


Использование события OnEnter недоступно
 для компонента [Label](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/Label.htm).
