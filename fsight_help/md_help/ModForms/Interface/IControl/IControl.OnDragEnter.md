# OnDragEnter


## Синтаксис


OnDragEnter(Sender: Object; Args: [IDragEventArgs](../IDragEventArgs/IDragEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnDragEnter наступает,
 когда перетаскиваемый объект входит в область данного компонента.
