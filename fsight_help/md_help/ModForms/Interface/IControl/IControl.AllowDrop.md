# IControl.AllowDrop


## Синтаксис


AllowDrop: Boolean;


## Описание


Свойство AllowDrop определяет, будет ли возможность у компонента принять перетаскиваемый объект.


## Комментарии


При установке свойству AllowDrop значения True компонент имеет возможность принимать перетаскиваемые объекты, при установке значения False перетаскивание объектов на этот компонент запрещено.
