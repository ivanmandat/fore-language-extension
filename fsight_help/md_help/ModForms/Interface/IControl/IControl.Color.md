# IControl.Color


## Синтаксис


Color: [IGxColor](ModDrawing.chm::/Interface/IGxColor/IGxColor.htm);


## Описание


Свойство Color определяет цвет
 фона компонента.


## Комментарии


В инспекторе объектов для данного свойства реализован раскрывающийся
 список значений. Данный список содержит четыре группы цветов:


![](IControl_Color.gif)


	- Стандартные цвета. Группа
	 цветов с фиксированными значениями [R](ModDrawing.chm::/Interface/IGxColor/IGxColor.R.htm),
	 [G](ModDrawing.chm::/Interface/IGxColor/IGxColor.G.htm),
	 [B](ModDrawing.chm::/Interface/IGxColor/IGxColor.B.htm)
	 компонент. Список данных цветов соответствует списку перечисления
	 [GxKnownColor](ModDrawing.chm::/Enums/GxKnownColor.htm).


	- Цвета темы Windows.
	 Группа цветов, зависящих от используемой в операционной системе цветовой
	 схемы. Список данных цветов соответствует списку перечисления [GxSystemColor](ModDrawing.chm::/Enums/GxSystemColor.htm).


	- Цвета темы платформы.
	 Группа цветов, зависящих от используемой темы оформления. Список данных
	 цветов соответствует списку перечисления [GxThemeColor](ModDrawing.chm::/Enums/GxThemeColor.htm).


	- Другие. Группа, содержащая
	 пользовательские цвета. Пункт «Выбрать
	 цвет» позволяет открыть стандартный диалог выбора цвета. Пункт
	 «Ввести HEX-код цвета» позволяет вручную указать код цвета. Список
	 пользовательских цветов сохраняется в реестре операционной системы
	 и может содержать не более пятидесяти значений. При превышении данного
	 количества цвета, которые были добавлены ранее, будут удалены. Если
	 на форме используется пользовательский цвет, отсутствующий в списке
	 «Другие», то при открытии
	 формы он будет добавлен в список автоматически.
