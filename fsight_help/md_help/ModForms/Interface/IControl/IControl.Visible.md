# IControl.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет видимость компонента во время выполнения формы.


## Комментарии


Если свойству установлено значение False, то компонент становится невидимым. Если свойство Visible установлено в False (True) для родительского компонента - контейнера, то вместе с компонентом - контейнером становятся невидимыми (видимыми) все дочерние компоненты. С помощью данного свойства можно размещать на одном и том же месте контейнера несколько компонентов, соответствующих различным режимам работы формы, и, в нужное время, делать видимым нужный компонент.
