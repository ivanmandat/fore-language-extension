# OnControlResize


## Синтаксис


OnControlResize(Sender: Object; Args: [IPositionEventArgs](../IPositionEventArgs/IPositionEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnControlResize наступает
 при изменении размеров у компонента.


## Комментарии


Используя свойства аргумента события [Width](../IPositionEventArgs/IPositionEventArgs.Width.htm)
 и [Height](../IPositionEventArgs/IPositionEventArgs.Height.htm)
 можно получить новые значения высоты и ширины компонента.
