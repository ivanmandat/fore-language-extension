# IControl.ParentShowHint


## Синтаксис


ParentShowHint: Boolean;


## Описание


Свойство ParentShowHint предназначено для разрешения/запрета отображения всплывающего окна подсказки родительского компонента (свойство [Hint](IControl.Hint.htm)) для всех компонентов контейнера или формы.


## Комментарии


Если ParentShowHint = True, то разрешение/запрет отображения всплывающего окна подсказки определяется свойством [ShowHint](IControl.ShowHint.htm) родительского компонента.


Если ParentShowHint = False, то разрешение/запрет отображения всплывающего окна подсказки определяется свойством [ShowHint](IControl.ShowHint.htm) самого компонента.
