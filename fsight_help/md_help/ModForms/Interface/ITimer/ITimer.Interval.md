# ITimer.Interval


## Синтаксис


Interval: Integer;


## Описание


Свойство Interval определяет интервал времени (в милисекундах), по истечении которого генерируется событие [OnTimer](../../Class/Timer/Timer.OnTimer.htm).


## Комментарии


По умолчанию свойству установлено значение 1000. Если установлено значение 0, то событие [OnTimer](../../Class/Timer/Timer.OnTimer.htm) генерироваться не будет.
