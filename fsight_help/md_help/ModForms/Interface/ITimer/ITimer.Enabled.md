# ITimer.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойству Enabled определяет активность таймера. Если свойству установлено значение True, то таймер работает и через определенный интервал времени, установленный в свойстве [Interval](ITimer.Interval.htm) автоматически генерируется событие [OnTimer](../../Class/Timer/Timer.OnTimer.htm).
