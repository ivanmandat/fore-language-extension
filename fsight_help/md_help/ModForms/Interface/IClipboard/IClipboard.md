# IClipboard


## Описание


Интерфейс IClipboard содержит
 статические свойства и методы для работы с буфером обмена, реализуемым
 классом [Clipboard](../../Class/Clipboard/Clipboard.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Text](IClipboard.Text.htm)


		 Свойство Text определяет
		 текст, содержащийся в буфере обмена.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [HasText](IClipboard.HasText.htm)


		 Метод HasText возвращает
		 признак наличия в буфере обмена текстовой информации.
