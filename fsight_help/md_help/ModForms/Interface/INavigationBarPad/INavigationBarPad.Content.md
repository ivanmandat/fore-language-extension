# INavigationBarPad.Content


## Синтаксис


Content: [INavigationBarPanel](../INavigationBarPanel/INavigationBarPanel.htm);


## Описание


Свойство Content возвращает параметры панели, связанной с данной вкладкой.


## Комментарии


Содержимое панели отображается, если текущая вкладка является [активной](../INavigationBar/INavigationBar.ActivePad.htm). Панель является фреймом для размещения формы с элементами управления, которые будут использоваться при работе с вкладкой.
