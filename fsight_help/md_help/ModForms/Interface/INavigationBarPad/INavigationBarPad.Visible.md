# INavigationBarPad.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет признак видимости вкладки в области навигационной панели.


## Комментарии


Если свойству установлено значение True, то в области навигационной панели будет отображен [заголовок](INavigationBarPad.Header.htm) вкладки. При щелчке по заголовку вкладка будет развернута и будет доступна связанная с ней [панель](INavigationBarPad.Content.htm).
