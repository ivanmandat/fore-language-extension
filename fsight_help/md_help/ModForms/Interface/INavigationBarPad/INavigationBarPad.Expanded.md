# INavigationBarPad.Expanded


## Синтаксис


Expanded: Boolean;


## Описание


Свойство Expanded определяет, развернута ли текущая вкладка.


## Комментарии


Если свойству установить значение True, то текущая вкладка будет развернута. При этом будет отображена [панель](INavigationBarPad.Content.htm), связанная с вкладкой. Все остальные вкладки будут свернуты.
