# INavigationBarPad.IsSystem


## Синтаксис


IsSystem: Boolean;


## Описание


Свойство IsSystem возвращает
 признак соответствия системной вкладке.


## Комментарии


Свойство возвращает значение True,
 если текущая вкладка соответствует системной вкладке, используемой в «Форсайт. Аналитическая платформа»
 для редактирования параметров какого-либо объекта. Компонент будет отображать
 системные вкладки, если он задан в свойстве [IWidget.NavBar](../IWidget/IWidget.NavBar.htm)
 какого-либо компонента, который реализует набор системных вкладок.
