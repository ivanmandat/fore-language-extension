# INavigationBarPad


Сборка: Forms;


## Описание


Интерфейс INavigationBarPad
 содержит свойства вкладки, отображаемой в области навигационной панели.


## Иерархия наследования


INavigationBarPad


## Комментарии


Свойства вкладки могут быть получены в свойстве [INavigationBar.ActivePad](../INavigationBar/INavigationBar.ActivePad.htm),
 [INavigationBarPadList.Item](../INavigationBarPadList/INavigationBarPadList.Item.htm)
 или [INavigationBarPadList.ItemById](../INavigationBarPadList/INavigationBarPadList.ItemById.htm).
