# IRibbon.Contexts


## Синтаксис


Contexts: [IRibbonContexts](../IRibbonContexts/IRibbonContexts.htm);


## Описание


Свойство Contexts возвращает коллекцию контекстных вкладок.


## Комментарии


Контекстные вкладки отображаются при наступлении определенных условий, например, когда выделен какой-либо объект. Управление отображением контекстных вкладок осуществляется с помощью свойства [Visible](../IRibbonContext/IRibbonContext.Visible.htm).

Пример использования приведен в описании метода [IRibbonContexts.Add](../IRibbonContexts/IRibbonContexts.Add.htm).
