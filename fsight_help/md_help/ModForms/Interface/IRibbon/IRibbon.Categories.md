# IRibbon.Categories


## Синтаксис


Categories: [IRibbonCategories](../IRibbonCategories/IRibbonCategories.htm);


## Описание


Свойство Categories возвращает коллекцию вкладок, доступных на ленте.


## Комментарии


Данная коллекция включает в себя вкладки, которые отображаются на ленте всегда.
