# IRibbon.MainCategory


## Синтаксис


MainCategory: [IRibbonCategory](../IRibbonCategory/IRibbonCategory.htm);


## Описание


Свойство MainCategory возвращает параметры главного меню ленты.


## Комментарии


Для настройки главного меню ленты используются два свойства: MainCategory и [MainButton](IRibbon.MainButton.htm). Свойство MainCategory позволяет настроить базовые свойства через свойства вкладки, например указать коллекцию изображений, которая будет использоваться элементами меню, либо задать горячие клавиши для вызова меню. Свойство [MainButton](IRibbon.MainButton.htm) позволяет настроить содержимое меню.
