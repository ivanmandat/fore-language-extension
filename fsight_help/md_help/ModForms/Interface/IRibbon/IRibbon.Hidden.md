# IRibbon.Hidden


## Синтаксис


Hidden: Boolean;


## Описание


Свойство Hidden определяет признак
 скрытости ленты.


## Комментарии


Допустимые значения:


	- True. Лента скрыта,
	 отображаются только [кнопка приложения](IRibbon.MainButton.htm),
	 заголовки [вкладок](IRibbon.Categories.htm) и [контекстных
	 вкладок](IRibbon.Contexts.htm).


	- False. Отображаются
	 все элементы ленты.
