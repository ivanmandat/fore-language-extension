# IRibbon.FindById


## Синтаксис


FindById(Id: Integer): [IBaseRibbonElement](../IBaseRibbonElement/IBaseRibbonElement.htm);


## Параметры


Id. Идентификатор элемента управления.


## Описание


Метод FindById осуществляет поиск элемента управления по числовому идентификатору.


## Комментарии


При создании ленты с использованием редактора, либо при динамическом создании элементов управления, идентификаторы генерируются автоматически. Диапазон идентификаторов начинается со значения 35000. Метод FindById актуально использовать, если структура ленты [загружена](IRibbon.LoadFromXML.htm) из XML файла и значения идентификаторов предопределены для конкретных элементов управления.


Метод FindById возвращает параметры элемента управления в случае удачного поиска и Null, если элемент управления не найден.
