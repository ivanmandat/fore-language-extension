# IRibbon.MainButton


## Синтаксис


MainButton: [IRibbonMainButton](../IRibbonMainButton/IRibbonMainButton.htm);


## Описание


Свойство MainButton возвращает параметры кнопки приложения, используемой для вызова главного меню.


## Комментарии


Кнопка приложения - это большая кнопка, расположенная в верхнем левом углу ленты. При нажатии на данную кнопку выводится главное меню. Для настройки главного меню ленты используются два свойства: [MainCategory](IRibbon.MainCategory.htm) и MainButton. Свойство [MainCategory](IRibbon.MainCategory.htm) позволяет настроить базовые свойства через свойства вкладки, например указать коллекцию изображений, которая будет использоваться элементами меню, либо задать горячие клавиши для вызова меню. Свойство MainButton позволяет настроить содержимое меню.
