# IRibbonChangeEventArgs


Сборка: Forms;


## Описание


Интерфейс IRibbonChangeEventArgs
 содержит свойства аргумента события [Ribbon.OnChange](../../Class/Ribbon/Ribbon.OnChange.htm),
 возникающего при использовании каких-либо элементов управления ленты.
