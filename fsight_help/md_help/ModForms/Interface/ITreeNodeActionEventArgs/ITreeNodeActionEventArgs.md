# ITreeNodeActionEventArgs


## Описание


Интерфейс ITreeNodeActionEventArgs
 содержит свойства аргумента событий, возникающих перед каким-либо действием
 над вершиной дерева компонентов [TreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeCombo.htm)
 и [TreeList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeList.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AllowAction](ITreeNodeActionEventArgs.AllowAction.htm)


		 Свойство AllowAction
		 определяет возможность произвести какое-либо действие над вершиной
		 дерева.


		 ![](../../Property_Image.gif)
		 [Column](ITreeNodeActionEventArgs.Column.htm)


		 Свойство Column возвращает
		 номер колонки, в которой осуществляется переход в режим редактирования
		 элемента.
