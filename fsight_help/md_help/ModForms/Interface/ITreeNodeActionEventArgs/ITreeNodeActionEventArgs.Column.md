# ITreeNodeActionEventArgs.Column


## Синтаксис


Column: Integer;


## Описание


Свойство Column возвращает номер колонки, в которой осуществляется переход в режим редактирования элемента.


## Комментарии


Свойство актуально только для события [OnEditing](../ITreeControl/ITreeControl.OnEditing.htm), во всех других событиях возвращает значение "-1".
