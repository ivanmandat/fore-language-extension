# ITreeNodeActionEventArgs.AllowAction


## Синтаксис


AllowAction: Boolean;


## Описание


Свойство AllowAction определяет возможность произвести какое-либо действие над вершиной дерева. Если свойству установлено значение True, то действие будет разрешено, иначе - запрещено.
