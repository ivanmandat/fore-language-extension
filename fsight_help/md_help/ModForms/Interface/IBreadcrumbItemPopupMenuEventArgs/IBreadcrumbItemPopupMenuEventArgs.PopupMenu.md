# IBreadcrumbItemPopupMenuEventArgs.PopupMenu


## Синтаксис


PopupMenu: [IPopupMenu](../IPopupMenu/IPopupMenu.htm);


## Описание


Свойство PopupMenu определяет всплывающее меню, которое будет вызвано для соответствующего элемента.


## Комментарии


В качестве значения данного свойства должен быть указан какой-либо компонент [PopupMenu](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/PopupMenu.htm).
