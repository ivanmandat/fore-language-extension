# IBreadcrumb.DeleteButton


## Синтаксис


DeleteButton: Boolean;


## Описание


Свойство DeleteButton определяет
 признак отображения кнопки, используемой для удаления всех элементов компонента.


## Комментарии


По умолчанию свойству установлено значение True,
 при этом в левой части компонента отображается кнопка ![](../knopka_del.gif),
 при нажатии на которую будут удалены все элементы компонента.
