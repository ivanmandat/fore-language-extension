# IBreadcrumb.Suffix


## Синтаксис


Suffix: String;


## Описание


Свойство Suffix определяет вспомогательный
 текст, отображаемый в области компонента после цепочки навигации.


## Комментарии


Вспомогательный текст отображается между элементами цепочки навигации
 и правой стороной компонента, либо [кнопкой
 истории](KeExtCtrls.chm::/Interface/IMetaAttributesBreadcrumb/IMetaAttributesBreadcrumb.HistoryButton.htm). Для отображения используется серый курсивный
 шрифт. Если все элементы не помещаются в область компонента и появляются
 кнопки прокрутки списка, то вспомогательный текст будет скрыт.
