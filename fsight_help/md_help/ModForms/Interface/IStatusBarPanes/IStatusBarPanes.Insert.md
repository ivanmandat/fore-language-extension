# IStatusBarPanes.Insert


## Синтаксис


Insert(Index: Integer): [IStatusBarPane](../IStatusBarPane/IStatusBarPane.htm);


## Параметры


Index - индекс панели, после которой будет вставлен новый элемент.


## Описание


Метод Insert осуществляет вставку новой панели в компонент [StatusBar](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/StatusBar.htm) после панели, индекс которой передается посредством входного параметра Index.
