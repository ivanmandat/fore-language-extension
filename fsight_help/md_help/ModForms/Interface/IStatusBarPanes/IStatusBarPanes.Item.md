# IStatusBarPanes.Item


## Синтаксис


Item(Index: Integer): [IStatusBarPane](../IStatusBarPane/IStatusBarPane.htm);


## Параметры


Index - индекс панели, которую необходимо получить.


## Описание


Свойство Item возвращает панель компонента [StatusBar](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/StatusBar.htm), индекс которой передается посредством параметра Index.
