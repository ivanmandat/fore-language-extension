# IWizardPage.ShowDescription


## Синтаксис


ShowDescription: Boolean;


## Описание


Свойство ShowDescription определяет видимость заголовка и [описания](IWizardPage.Description.htm) страницы мастера.


## Комментарии


По умолчанию свойству установлено значение True, при этом заголовок и описание страницы отображаются.


Если свойству ShowDescription установить значение False, то заголовок и описание будут скрыты, а содержимое страницы будет сдвинуто вверх на размер, равный высоте области, которую занимают заголовок с описанием.
