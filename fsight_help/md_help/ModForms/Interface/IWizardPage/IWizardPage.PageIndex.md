# IWizardPage.PageIndex


## Синтаксис


PageIndex: Integer;


## Описание


Свойство PageIndex определяет индекс страницы мастера.


## Комментарии


Индекс определяет очередность перехода к страницы. Индексация начинается с нуля. При создании новых страницы индекс присваивается автоматически. При изменении индекса также будет изменено расположение ссылки на страницу в компоненте [WizardSplash](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/WizardSplash.htm). Значение индекса должно быть в диапазоне [0, [Count](../IWizardPages/IWizardPages.Count.htm)-1].
