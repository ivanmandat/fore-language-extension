# IWizardPage.Description


## Синтаксис


Description: String;


## Описание


Свойство Description определяет описание страницы мастера.


## Комментарии


Описание располагается ниже заголовка страницы. В описании может быть указана любая текстовая информация, описывающая содержимое текущей страницы.


По умолчанию в свойстве Description установлено значение «Описание». Если в качестве значения задать пустую строку, то размер области, отведенной для заголовка и описания, будет уменьшен на размер описания.
