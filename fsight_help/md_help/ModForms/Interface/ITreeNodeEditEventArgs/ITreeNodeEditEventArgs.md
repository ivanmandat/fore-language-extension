# ITreeNodeEditEventArgs


## Описание


Интерфейс ITreeNodeEditEventArgs
 содержит свойства аргумента события, возникающего после выхода из режима
 редактирования элемента.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Column](ITreeNodeEditEventArgs.Column.htm)


		 Свойство Column возвращает
		 индекс колонки, в которой осуществлялось редактирование элемента.


		 ![](../../Property_Image.gif)
		 [Text](ITreeNodeEditEventArgs.Text.htm)


		 Свойство Text определяет
		 текст элемента в колонке, в которой осуществлялось редактирование
		 элемента.
