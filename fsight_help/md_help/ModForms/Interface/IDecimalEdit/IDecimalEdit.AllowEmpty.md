# IDecimalEdit.AllowEmpty


## Синтаксис


AllowEmpty: Boolean;


## Описание


Свойство AllowEmpty
 определяет возможность наличия пустого значения в компоненте.


## Комментарии


Если свойству установлено значение True,
 то пустое значение в компоненте не будет подсвечиваться как недопустимое
 (красным цветом). При этом свойство [Value](IDecimalEdit.Value.htm)
 возвращает 0, а свойство [Text](../IControl/IControl.Text.htm)
 - пустую строку. По умолчанию свойству AllowEmpty
 установлено значение False, при этом пустое значение в компоненте подсвечивается
 как недопустимое.
