# IDecimalEdit.StepSize


## Синтаксис


StepSize: Decimal;


## Описание


Свойство StepSize определяет
 величину, на которую будет изменяться значение введенное в поле редактирования,
 при использовании кнопок SpinUp
 и SpinDown для изменения значения.


## Комментарии


По умолчанию для компонента установлен шаг 0,00000000000001.
