# IDecimalEdit.MinValue


## Синтаксис


MinValue: Decimal;


## Описание


Свойство MinValue
 определяет минимальное значение числа, которое можно установить в компоненте.


## Комментарии


При соблюдении следующего условия в компоненте можно указывать любые
 значения: MinValue
 = [MaxValue](IDecimalEdit.MaxValue.htm)
 = 0.
