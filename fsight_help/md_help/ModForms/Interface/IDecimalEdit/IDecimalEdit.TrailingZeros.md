# IDecimalEdit.TrailingZeros


## Синтаксис


TrailingZeros: Boolean;


## Описание


Свойство TrailingZeros определяет
 признак отображения нулей после запятой, при [Precision](IDecimalEdit.Precision.htm)
 больше нуля.


## Комментарии


Если TrailingZeros = False
 и значение компонента целое, то нули после запятой не отображаются. По
 умолчанию свойству установлено значение True.
