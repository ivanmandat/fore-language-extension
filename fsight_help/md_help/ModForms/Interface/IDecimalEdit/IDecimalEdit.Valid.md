# IDecimalEdit.Valid


## Синтаксис


Valid: Boolean;


## Описание


Свойство Valid возвращает корректность
 вещественного числа, введенного в поле редактирования.


## Комментарии


Если введеное значение некорректно, то поле редактирования подсвечивается
 красным цветом и свойство Valid
 возвращает значение False.
