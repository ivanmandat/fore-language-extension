# IDecimalEdit.Precision


## Синтаксис


Precision: Integer;


## Описание


Свойство Precision определяет
 количество разрядов, отображаемых после запятой.


## Комментарии


По умолчанию в компоненте установлена точность - 14 знаков после запятой.
 Максимальная точность - 18 знаков после запятой.
