# IDecimalEdit.ThousandSeparator


## Синтаксис


ThousandSeparator: Boolean;


## Описание


Свойство ThousandSeparator
 определяет, будет ли отображаться значение в компоненте с использованием
 разделителя групп разрядов.


## Комментарии


Для разделения групп разрядов используется разделитель, установленный
 в системе. По умолчанию свойству установлено значение False,
 при этом значения отображаются без использования разделителя разрядов.
