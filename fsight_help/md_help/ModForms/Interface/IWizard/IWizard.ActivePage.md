# IWizard.ActivePage


## Синтаксис


ActivePage: [IWizardPage](../IWizardPage/IWizardPage.htm);


## Описание


Свойство ActivePage определяет активную страницу мастера.


## Комментарии


В списке доступных значений свойства отображаются имена всех страниц (компонент WizardPage), существующих в рамках текущего компонента [Wizard](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/Wizard.htm).
