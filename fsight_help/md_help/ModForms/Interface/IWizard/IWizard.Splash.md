# IWizard.Splash


## Синтаксис


Splash: [IWizardSplash](../IWizardSplash/IWizardSplash.htm);


## Описание


Свойство Splash возвращает общую заставку мастера [WizardSplash](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/WizardSplash.htm), отображающая все ссылки на страницы мастера. Свойство доступно только через Fore.
