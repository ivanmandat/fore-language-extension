# ITreeNodeGetImageEventArgs


## Описание


Интерфейс ITreeNodeGetImageEventArgs
 содержит свойства аргумента событий, возникающих при получении изображения
 и изображения состояния элемента дерева компонентов [TreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeCombo.htm)
 и [TreeList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeList.htm).
