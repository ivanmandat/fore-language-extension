# IImageList.AddIcon


## Синтаксис


AddIcon(Icon: [IGxIcon](ModDrawing.chm::/Interface/IGxIcon/IGxIcon.htm));


## Параметры


Icon. Пиктограмма, которую необходимо добавить
 в коллекцию.


## Описание


Метод AddIcon добавляет пиктограмму,
 передаваемую посредством параметра Icon,
 в коллекцию компонента.
