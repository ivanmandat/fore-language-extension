# IImageList.Height


## Синтаксис


Height: Integer;


## Описание


Свойство Height определяет высоту
 изображений, хранящихся в коллекции компонента.


## Комментарии


Данный размер будет использоваться при разбиении на отдельные изображения,
 в случае если загружаемый графический файл имеет большую высоту.
