# IImageList.LoadFromStream2


## Синтаксис


LoadFromStream2(Stream: [IIOStream](ModIo.chm::/Interface/IIOStream/IIOStream.htm));


## Параметры


Stream. Поток, из которого осуществляется загрузка изображений.


## Описание


Метод LoadFromStream2 осуществляет загрузку изображений из потока с восстановлением информации о полупрозрачности.


## Комментарии


Данный метод позволяет загрузить изображения, сохраненные в поток с помощью метода [SaveToStream2](IImageList.SaveToStream2.htm).
