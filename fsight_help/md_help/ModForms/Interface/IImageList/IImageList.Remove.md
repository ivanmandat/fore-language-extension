# IImageList.Remove


## Синтаксис


Remove(Index: Integer);


## Параметры


Index - индекс изображения из коллекции, которое необходимо удалить.


## Описание


Метод Remove удаляет изображение из коллекции, индекс которого передается в качестве входного параметра Index.
