# ITrackBarEventArgs


## Описание


Интерфейс ITrackBarEventArgs
 содержит свойства аргумента события, происходящего при изменении положения
 ползунка компонента [TrackBar](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TrackBar.htm).
