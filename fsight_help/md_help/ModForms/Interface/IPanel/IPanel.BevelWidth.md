# IPanel.BevelWidth


## Синтаксис


BevelWidth: Integer;


## Описание


Свойство BevelWidth определяет ширину фаски, создающей эффект объемности рамок. Размер указывается в пикселях. По умолчанию устанавливается значение 1.
