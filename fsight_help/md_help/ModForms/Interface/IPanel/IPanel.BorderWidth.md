# IPanel.BorderWidth


## Синтаксис


BorderWidth: Integer;


## Описание


Свойство BorderWidth определяет расстояние между наружной и внутренней рамками компонента [Panel](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/Panel.htm). Расстояние указывается в пикселях. По умолчанию установлено значение 0.
