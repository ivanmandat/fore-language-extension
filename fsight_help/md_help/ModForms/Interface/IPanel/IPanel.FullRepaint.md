# IPanel.FullRepaint


## Синтаксис


FullRepaint: Boolean;


## Описание


Свойство FullRepaint определяет
 способ перерисовки панели при изменении размеров панели.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 При изменении размеров панели будут перерисованы область панели со
	 всеми её дочерними компонентами и границы панели.


	- False.
	 При изменении размеров панели границы панели перерисованы не будут.
