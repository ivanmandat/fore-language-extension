# IRibbonCategories


Сборка: Forms;


## Описание


Интерфейс IRibbonCategories
 содержит свойства и методы для работы с коллекцией вкладок ленты.


## Иерархия наследования


IRibbonCategories


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IRibbonCategories.Count.htm)


		 Свойство Count возвращает
		 количество вкладок ленты.


		 ![](../../Property_Image.gif)
		 [Item](IRibbonCategories.Item.htm)


		 Свойство Item возвращает
		 параметры указанной вкладки.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IRibbonCategories.Add.htm)


		 Метод Add добавляет
		 указанную вкладку в коллекцию вкладок ленты.


		 ![](../../Sub_Image.gif)
		 [Clear](IRibbonCategories.Clear.htm)


		 Метод Clear удаляет
		 все вкладки ленты.


		 ![](../../Sub_Image.gif)
		 [IndexOf](IRibbonCategories.IndexOf.htm)


		 Метод IndexOf возвращает
		 для указанной вкладки индекс в коллекции.


		 ![](../../Sub_Image.gif)
		 [Insert](IRibbonCategories.Insert.htm)


		 Метод Insert вставляет
		 указанную вкладку в указанную позицию в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IRibbonCategories.Remove.htm)


		 Метод Remove удаляет
		 указанную вкладку из коллекции.


		 ![](../../Sub_Image.gif)
		 [RemoveAt](IRibbonCategories.RemoveAt.htm)


		 Метод RemoveAt удаляет
		 вкладку из коллекции по индексу.
