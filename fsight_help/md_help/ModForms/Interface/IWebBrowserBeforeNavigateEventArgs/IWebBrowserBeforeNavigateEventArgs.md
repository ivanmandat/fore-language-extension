# IWebBrowserBeforeNavigateEventArgs


## Описание


Интерфейс IWebBrowserBeforeNavigateEventArgs
 содержит свойства аргумента события [OnBeforeNavigate2](../../Class/WebBrowserBox/WebBrowserBox.OnBeforeNavigate2.htm),
 наступающего перед тем, как будет совершен переход к какой-либо Web-странице.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Cancel](IWebBrowserBeforeNavigateEventArgs.Cancel.htm)


		 Свойство Cancel определяет,
		 будет ли осуществляться переход к указанной Web-странице.


		 ![](../../Property_Image.gif)
		 [Flags](IWebBrowserBeforeNavigateEventArgs.Flags.htm)


		 Свойство находится в стадии разработки.


		 ![](../../Property_Image.gif)
		 [Headers](IWebBrowserBeforeNavigateEventArgs.Headers.htm)


		 Свойство Headers возвращает
		 дополнительные заголовки, посылаемые серверу.


		 ![](../../Property_Image.gif)
		 [TargetFrameName](IWebBrowserBeforeNavigateEventArgs.TargetFrameName.htm)


		 Свойство TargetFrameName
		 возвращает наименование фрейма, который используется для отображения
		 ресурсов.
