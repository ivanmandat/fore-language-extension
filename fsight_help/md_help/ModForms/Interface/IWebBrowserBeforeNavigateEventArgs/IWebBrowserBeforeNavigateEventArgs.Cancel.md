# IWebBrowserBeforeNavigateEventArgs.Cancel


## Синтаксис


Cancel: Boolean;


## Описание


Свойство Cancel определяет, будет ли осуществляться переход к указанной Web-странице. Если свойству установлено значение True - переход к странице будет запрещен.
