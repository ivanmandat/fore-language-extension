# IWebBrowserBeforeNavigateEventArgs.Headers


## Синтаксис


Headers: String;


## Описание


Свойство Headers возвращает дополнительные заголовки, посылаемые серверу.


## Комментарии


Заголовки могут содержать такие данные, как действие, требуемое из сервера, тип данных, передаваемых серверу, или код статуса.
