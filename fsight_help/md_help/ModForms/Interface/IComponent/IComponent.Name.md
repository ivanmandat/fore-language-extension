# IComponent.Name


## Синтаксис


Name: String;


## Описание


Свойство Name определяет наименование
 компонента, которое используется в дальнейшем.


## Комментарии


Свойство задается только в процессе проектирования формы и не должно
 изменяться во время ее выполнения. По умолчанию каждому компоненту присваивается
 имя, состоящие из наименования класса компонента и номера по порядку (нумерация
 начинается с 1).
