# IComponent.Tag


## Синтаксис


Tag: Integer;


## Описание


Свойство Tag не используется
 компилятором. Его значение по умолчанию равно 0.


## Комментарии


Пользователь может изменить значение свойства Tag
 и использовать его по своему усмотрению.
