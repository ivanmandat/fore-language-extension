# IBreadcrumbActionItemEventArgs.Allow


## Синтаксис


Allow: Boolean;


## Описание


Свойство Allow определяет возможность произвести какое-либо действие над элементом компонента. Если свойству установлено значение True, то действие будет разрешено, иначе - запрещено.
