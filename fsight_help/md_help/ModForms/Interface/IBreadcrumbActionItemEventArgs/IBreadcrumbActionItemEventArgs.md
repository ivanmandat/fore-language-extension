# IBreadcrumbActionItemEventArgs


## Описание


Интерфейс IBreadcrumbActionItemEventArgs
 содержит свойства аргумента событий, происходящих перед каким-либо действием
 над элементом компонента.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Allow](IBreadcrumbActionItemEventArgs.Allow.htm)
		 Свойство Allow определяет
		 возможность произвести какое-либо действие над элементом компонента.


		 ![](../../Property_Image.gif)
		 [Index](IBreadcrumbActionItemEventArgs.Index.htm)
		 Свойство Index возвращает
		 индекс элемента, над которым будет произведено какое-либо действие.
