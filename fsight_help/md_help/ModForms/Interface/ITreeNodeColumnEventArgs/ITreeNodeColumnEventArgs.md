# ITreeNodeColumnEventArgs


## Описание


Интерфейс ITreeNodeColumnEventArgs
 содержит свойства аргумента событий, связанных с определенной колонкой
 компонента.


Используется при обработке следующих событий:


	- [OnColumnClick](../ITreeControl/ITreeControl.OnColumnClick.htm)


	- [OnHover](../ITreeControl/ITreeControl.OnHover.htm)
