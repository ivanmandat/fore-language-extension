# IImageBox.TransparentColor


## Синтаксис


TransparentColor: [IGxColor](ModDrawing.chm::/Interface/IGxColor/IGxColor.htm);


## Описание


Свойство TransparentColor определяет цвет для прозрачности изображения. Актуально если свойству [Transparent](IImageBox.Transparent.htm) установлено значение True.
