# IImageBox.AutoSize


## Синтаксис


AutoSize: Boolean;


## Описание


Свойство AutoSize определяет, будет ли автоматически устанавливаться размер окна компонента в соответствии с размером изображения.


## Комментарии


Если свойству AutoSize установлено значение True, то размер компонента подгоняется под размерамы размещенного в нем изображения, иначе размер компонента остается фиксированным, при этом если размеры компонента меньше размеров изображения - отображается только та часть изображения, которая помещается в данную область. По умолчанию данное свойство имеет значение False.
