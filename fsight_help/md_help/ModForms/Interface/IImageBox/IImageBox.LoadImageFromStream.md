# IImageBox.LoadImageFromStream


## Синтаксис


LoadImageFromStream(Stream: [IIOStream](ModIo.chm::/Interface/IIOStream/IIOStream.htm));


## Параметры


Stream - поток, из которого осуществляется загрузка изображения.


## Описание


Метод LoadImageFromStream осуществляет загрузку изображения из потока.
