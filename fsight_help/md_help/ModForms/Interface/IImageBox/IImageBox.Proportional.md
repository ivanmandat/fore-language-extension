# IImageBox.Proportional


## Синтаксис


Proportional: Boolean;


## Описание


Свойство Proportional определяет соблюдение пропорций рисунка в окне компонента [ImageBox](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ImageBox.htm).


## Комментарии


Если свойству Proportional установлено значение True, то размер рисунка будет пропорционально изменен и будет соответствовать размерам окна компонента, иначе отображаемая часть рисунка будет ограничиваться размерами окна компонента [ImageBox](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ImageBox.htm).
