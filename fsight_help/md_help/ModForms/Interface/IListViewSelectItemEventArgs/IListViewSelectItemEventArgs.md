# IListViewSelectItemEventArgs


## Описание


Интерфейс IListViewSelectItemEventArgs
 содержит свойства аргумента события, происходящего при смене отметки у
 какого-либо элемента компонента [ListView](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ListView.htm).
