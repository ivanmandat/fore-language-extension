# IListViewSelectItemEventArgs.Selected


## Синтаксис


Selected: Boolean;


## Описание


Свойство Selected возвращает признак отметки элемента. Свойство возвращает значение True, если элемент был отмечен и False, если отметка с элемента была снята.
