# IKeyPressEventArgs


## Описание


Интерфейс IKeyPressEventArgs
 содержит свойства аргумента события [OnKeyPress](../IControl/IControl.OnKeyPress.htm),
 происходящего во время нажатия клавиши.

		 Краткое описание

		 [Key](IKeyPressEventArgs.Key.htm)


		 Свойство Key определяет
		 символ клавиши, которая была нажата.
