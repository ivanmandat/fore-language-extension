# ICloseQueryEventArgs


Сборка: Forms;


## Описание


Интерфейс ICloseQueryEventArgs
 содержит свойства аргумента события [OnCloseQuery](../../Class/Form/Form.OnCloseQuery.htm),
 наступающего в тот момент, когда форма получает команду на закрытие.
