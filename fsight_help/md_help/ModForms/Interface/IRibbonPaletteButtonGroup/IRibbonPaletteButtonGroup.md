# IRibbonPaletteButtonGroup


Сборка: Forms;


## Описание


Интерфейс IRibbonPaletteButtonGroup
 содержит свойства группы графических кнопок, отображаемых при раскрытии
 [комбинированного
 меню](../IRibbonPaletteButton/IRibbonPaletteButton.htm).
