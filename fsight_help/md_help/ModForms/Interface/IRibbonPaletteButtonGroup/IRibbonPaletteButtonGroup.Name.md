# IRibbonPaletteButtonGroup.Name


## Синтаксис


Name: String;


## Описание


Свойство Name определяет наименование
 группы графических кнопок.


## Комментарии


Наименование будет отображаться в комбинированном меню над самими графическими
 кнопками группы.
