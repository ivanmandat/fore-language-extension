# IDragEventArgs


## Описание


Интерфейс IDragEventArgs содержит
 свойства аргумента событий, происходящих при перетаскивании объектов с
 использованием механизма Drag&Drop.


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AllowedEffect](IDragEventArgs.AllowedEffect.htm)


		 Свойство AllowedEffect
		 возвращает вид курсора при текущем выполнении Drag&Drop.


		 ![](../../Property_Image.gif)
		 [Data](IDragEventArgs.Data.htm)


		 Свойство Data возвращает
		 компонент, перетаскивание которого осуществляется.


		 ![](../../Property_Image.gif)
		 [Effect](IDragEventArgs.Effect.htm)


		 Свойство Effect определяет
		 тип перетаскивания.


		 ![](../../Property_Image.gif)
		 [KeyState](IDragEventArgs.KeyState.htm)


		 Свойство KeyState возвращает
		 текущее состояние клавиш SHIFT, CTRL, ALT, а также состояние кнопок
		 мыши.


		 ![](../../Property_Image.gif)
		 [Point](IDragEventArgs.Point.htm)


		 Свойство Point возвращает
		 координаты курсора мыши в клиентской части компонента.
