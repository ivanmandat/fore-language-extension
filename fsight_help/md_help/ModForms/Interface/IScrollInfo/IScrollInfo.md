# IScrollInfo


Сборка: Forms;


## Описание


Интерфейс IScrollInfo содержит
 свойства для получения информации о полосе прокрутки.


## Иерархия наследования


IScrollInfo


## Комментарии


Для получения информации о полосах прокрутки используйте свойство [HScrollInfo](../IScrollBars/IScrollBars.HScrollInfo.htm)
 либо [VScrollInfo](../IScrollBars/IScrollBars.VScrollInfo.htm).


## Свойства


		 Имя свойства
		 Краткое описание

		 [Max](IScrollInfo.Max.htm)

		 [Min](IScrollInfo.Min.htm)
		 Свойство Min возвращает

		 [Page](IScrollInfo.Page.htm)
		 Свойство Page возвращает

		 Свойство Pos определяет
		 позицию ползунка полосы прокрутки.


		 ![](../../Property_Image.gif)
		 [TrackPos](IScrollInfo.TrackPos.htm)
		 Свойство TrackPos определяет
		 позицию ползунка в процессе прокрутки.
