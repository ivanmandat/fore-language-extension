# IScrollInfo.Pos


## Синтаксис


Pos: Integer;


## Описание


Свойство Pos определяет позицию ползунка полосы прокрутки.


## Комментарии


Допустимым значением позиции ползунка является значение, находящееся в диапазоне [[Min](IScrollInfo.Min.htm); [Max](IScrollInfo.Max.htm) - Maximum([Page](IScrollInfo.Page.htm)-1, 0)].


## Пример


Пример использования данного свойства приведен в описании свойства [ScrollBars](../IScrollBars/IScrollBars.ScrollBars.htm).
