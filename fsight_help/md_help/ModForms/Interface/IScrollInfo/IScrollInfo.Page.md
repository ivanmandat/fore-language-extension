# IScrollInfo.Page


## Синтаксис


Page: Integer;


## Описание


Свойство Page возвращает размер страницы.


## Комментарии


Данный размер используется полосами прокрутки для пропорционального изменения размера ползунка при изменении размеров компонента, либо прокручиваемой области внутри компонента.


## Пример


Пример использования данного свойства приведен в описании свойства [ScrollBars](../IScrollBars/IScrollBars.ScrollBars.htm).
