# IScrollInfo.TrackPos


## Синтаксис


TrackPos: Integer;


## Описание


Свойство TrackPos определяет позицию ползунка в процессе прокрутки.


## Комментарии


В текущей реализации значение данного свойства совпадает со значением свойства [Pos](IScrollInfo.Pos.htm).
