# ILabel.WordWrap


## Синтаксис


WordWrap: Boolean;


## Описание


Свойство WordWrap определяет,
 будут ли переноситься слова длинной надписи, превышающие длину компонента,
 на новую строку.


## Комментарии


Допустимые значения:


	- True. Переносить слова
	 на новую строку;


	- False. Не переносить
	 слова на новую строку.
