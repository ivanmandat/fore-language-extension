# IFolderBrowserDialog.RootFolder


## Синтаксис


RootFolder: [ShellSpecialFolder](../../Enums/ShellSpecialFolder.htm);


## Описание


Свойство RootFolder позволяет выбрать стандартную папку, которая будет корневой при открытии стандартного окна Windows открытия папки. По умолчанию свойству установлено значение DesktopDirectory.
