# IFolderBrowserDialog.SelectedPath


## Синтаксис


SelectedPath: String;


## Описание


Свойство SelectedPath указывает пользовательскую папку, которая будет искаться в системной папке (указанной с помощью свойства [RootFolder](IFolderBrowserDialog.RootFolder.htm)) и фокусироваться при открытии стандартного окна Windows открытия папки.


## Комментарии


Если данная директория не найдена в указанной системной папке, то при открытии диалога будет фокусироваться указанная системная папка. При удачном выполнении диалога - возвращает абсолютный путь, к выбранной пользователем, папке.
