# IFolderBrowserDialog.ShowNewFolderButton


## Синтаксис


ShowNewFolderButton: Boolean;


## Описание


Свойство ShowNewFolderButton
 определяет, будет ли отображаться кнопка создания новой папки «Создать
 папку».


## Комментарии


Если свойству установлено значение True,
 то кнопка создания новой папки будет отображаться в окне «Обзор папок».
 При значении False кнопка отображаться
 не будет.
