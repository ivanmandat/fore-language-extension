# IFolderBrowserDialog.Execute


## Синтаксис


Execute: Boolean;


## Описание


Метод Execute позволяет вызывать
 стандартное окно Windows открытия папки.


## Комментарии


В случае, если вызов произошел успешно, возвращает True,
 в противном случае - False.
