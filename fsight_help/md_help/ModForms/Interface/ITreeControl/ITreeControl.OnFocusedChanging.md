# ITreeControl.OnFocusedChanging


## Синтаксис


OnFocusedChanging(Sender: Object; Args: [ITreeNodeActionEventArgs](../ITreeNodeActionEventArgs/ITreeNodeActionEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnFocusedChanging наступает
 перед передачей фокуса другой вершине компонента.
