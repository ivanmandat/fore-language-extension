# ITreeControl.OnAdded


## Синтаксис


OnAdded(Sender: Object; Args: [ITreeNodeEventArgs](../ITreeNodeEventArgs/ITreeNodeEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnAdded наступает при
 добавлении нового элемента.
