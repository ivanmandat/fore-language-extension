# ITreeControl.OnUnselecting


## Синтаксис


OnUnselecting(Sender: Object; Args: [ITreeNodeActionEventArgs](../ITreeNodeActionEventArgs/ITreeNodeActionEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnUnselecting наступает
 перед попыткой снять отметку у элемента.
