# ITreeControl.OnEdited


## Синтаксис


OnEdited(Sender: Object; Args: [ITreeNodeEditEventArgs](../ITreeNodeEditEventArgs/ITreeNodeEditEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnEdited наступает при
 выходе из режима редактирования содержимого поля текущей записи компонента.


## Комментарии


Данное событие наступает только в том случае, если произошло событие
 [OnEditing](ITreeControl.OnEditing.htm).
