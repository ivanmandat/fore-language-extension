# ITreeControl.AdjustWidth


## Синтаксис


AdjustWidth(Column: Integer);


## Параметры


Column. Индекс колонки, в которой
 необходимо осуществить автоматический подгон ширины.


## Описание


Метод AdjustWidth осуществляет
 автоматический подгон ширины колонки с учетом её содержимого.


## Комментарии


Индекс колонки передается посредством параметра Column.
 При вызове метода ширина подгоняется с учетом наличия экспандера, флажка,
 пиктограммы и пиктограммы состояния, текста элемента.
