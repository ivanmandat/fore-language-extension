# ITreeControl.SortDefault


## Синтаксис


SortDefault;


## Описание


Метод SortDefault осуществляет
 сортировку элементов в компоненте.


## Комментарии


Метод используется, если определены свойства [SortType](ITreeControl.SortType.htm),
 [Sorted](ITreeControl.Sorted.htm)
 и [SortColumn](ITreeControl.SortColumn.htm).
 Сортировка осуществляется по возрастанию значений.
