# ITreeControl.OnUnselected


## Синтаксис


OnUnselected(Sender: Object; Args: [ITreeNodeEventArgs](../ITreeNodeEventArgs/ITreeNodeEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnUnselected наступает
 после снятия отметки у вершины.
