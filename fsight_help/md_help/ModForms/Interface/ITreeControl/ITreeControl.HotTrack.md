# ITreeControl.HotTrack


## Синтаксис


HotTrack: Boolean;


## Описание


Свойство HotTrack определяет
 признак выделения подчеркиванием элемента, над которым проходит указатель
 мыши.


## Комментарии


Если свойству установлено значение True,
 то будет осуществляться выделение подчеркиванием элементов, над которыми
 будет проходить указатель мыши. При значении False
 элементы выделяться подчеркиванием не будут.


Также при наведении указателя мыши элемент может быть выделен цветом,
 за это отвечает свойство [HighlightTrack](ITreeControl.HighlightTrack.htm).
