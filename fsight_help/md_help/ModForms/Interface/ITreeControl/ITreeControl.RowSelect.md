# ITreeControl.RowSelect


## Синтаксис


RowSelect: Boolean;


## Описание


Свойство RowSelect определяет признак выделения выбранного элемента подсветкой всей строки.


## Комментарии


Если свойству установлено значение True, то подсветкой будет выделяться вся строка, в которой находится выбранный элемент, иначе будет выделяться только сам элемент.


Свойство RowSelect актуально только в том случае, если для свойства [ShowColumnHeaders](ITreeControl.ShowColumnHeaders.htm) установлено значение True.
