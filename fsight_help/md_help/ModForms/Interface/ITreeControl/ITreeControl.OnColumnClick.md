# ITreeControl.OnColumnClick


## Синтаксис


OnColumnClick(Sender: Object; Args: [ITreeNodeColumnEventArgs](../ITreeNodeColumnEventArgs/ITreeNodeColumnEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnColumnClick наступает
 при щелчке кнопкой мыши по заголовку столбца компонента.
