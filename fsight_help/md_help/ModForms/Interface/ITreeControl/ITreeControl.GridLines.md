# ITreeControl.GridLines


## Синтаксис


GridLines: Boolean;


## Описание


Свойство GridLines определяет, будет ли отображаться сетка у компонентов [TreeList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeList.htm) и [TreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeCombo.htm).


## Комментарии


Если свойству установлено значение True, то сетка отображается, иначе сетки не будет.
