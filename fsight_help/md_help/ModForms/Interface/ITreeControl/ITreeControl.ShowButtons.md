# ITreeControl.ShowButtons


## Синтаксис


ShowButtons: Boolean;


## Описание


Свойство ShowButtons определяет, будут ли отображаться кнопки разворачивания и сворачивания дерева потомков вершин.


## Комментарии


Если свойству установлено значение True, слева от вершины будут располагаться кнопки «+» и «-», иначе разворачивание и сворачивание дерева потомков вершин будет производиться двойным щелчком мыши по вершине.
