# ITreeControl.OnCollapsed


## Синтаксис


OnCollapsed(Sender: Object; Args: [ITreeNodeEventArgs](../ITreeNodeEventArgs/ITreeNodeEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnCollapsed наступает
 после свертывания иерархии потомков вершины дерева.


## Комментарии


Данное событие наступает только в том случае, если произошло событие
 [OnCollapsing](ITreeControl.OnCollapsing.htm).
