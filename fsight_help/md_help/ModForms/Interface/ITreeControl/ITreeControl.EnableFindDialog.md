# ITreeControl.EnableFindDialog


## Синтаксис


EnableFindDialog: Boolean;


## Описание


Свойство EnableFindDialog определяет, будет ли вызываться стандартный диалог поиска элементов при нажатии сочетания клавиш CTRL+F или F3.


## Комментарии


По умолчанию свойству установлено значение True, при этом диалог вызывается. Данное свойство актуально, если необходимо переназначить сочетание клавиш CTRL+F и F3 для выполнения каких либо других действий.
