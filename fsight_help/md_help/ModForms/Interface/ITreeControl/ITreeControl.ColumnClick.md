# ITreeControl.ColumnClick


## Синтаксис


ColumnClick: Boolean;


## Описание


Свойство ColumnClick определяет возможность отображения заголовков столбцов компонентов [TreeList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeList.htm) и [TreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeCombo.htm) в виде кнопок.


## Комментарии


Если свойству ColumnClick установлено значение True, заголовок принимает форму кнопки, нажатие на которую будет разрешено, при этом генерируется событие [OnColumnClick](ITreeControl.OnColumnClick.htm). При установке значения False, нажатие запрещено.
