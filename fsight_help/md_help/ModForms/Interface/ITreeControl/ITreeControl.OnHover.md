# ITreeControl.OnHover


## Синтаксис


OnHover(Sender:
 Object; Args:
 [ITreeNodeColumnEventArgs](../ITreeNodeColumnEventArgs/ITreeNodeColumnEventArgs.htm));


## Параметры


Sender.
 Параметр, возвращающий компонент, сгенерировавший событие.


Args.
 Параметр, позволяющий определить параметры события.


## Описание


Событие OnHover
 наступает в момент прохождения курсора мыши над элементом и отметки подчеркиванием
 этого элемента компонента.


## Комментарии


Необходимым условием для наступления события является установка свойству
 [HotTrack](ITreeControl.HotTrack.htm)
 значения True.
