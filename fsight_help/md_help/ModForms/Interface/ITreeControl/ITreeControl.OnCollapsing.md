# ITreeControl.OnCollapsing


## Синтаксис


OnCollapsing(Sender: Object; Args: [ITreeNodeActionEventArgs](../ITreeNodeActionEventArgs/ITreeNodeActionEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnCollapsing наступает
 перед свертыванием иерархии потомков вершины дерева.
