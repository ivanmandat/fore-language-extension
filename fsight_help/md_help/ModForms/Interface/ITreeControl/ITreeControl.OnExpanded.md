# ITreeControl.OnExpanded


## Синтаксис


OnExpanded(Sender: Object; Args: [ITreeNodeEventArgs](../ITreeNodeEventArgs/ITreeNodeEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnExpanded наступает
 после развертывания иерархии потомков вершины дерева.


## Комментарии


Данное событие наступает только в том случае, если произошло событие
 [OnExpanding](ITreeControl.OnExpanding.htm).
