# ITreeControl.OnChecked


## Синтаксис


OnChecked(Sender: Object; Args: [ITreeNodeCheckedEventArgs](../ITreeNodeCheckedEventArgs/ITreeNodeCheckedEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnChecked наступает
 после изменения состояния флажка у элемента.


## Комментарии


Событие актуально, если свойству [CheckBoxes](ITreeControl.Checkboxes.htm)
 установлено значение True.
