# ITreeControl.OnGetHint


## Синтаксис


OnGetHint(Sender: Object; Args: [ITreeNodeGetHintEventArgs](../ITreeNodeGetHintEventArgs/ITreeNodeGetHintEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnGetHint наступает
 при появлении всплывающей подсказки над элементом, над которым остановился
 курсор мыши.
