# ITreeControl.OnSelecting


## Синтаксис


OnSelecting(Sender: Object; Args: [ITreeNodeActionEventArgs](../ITreeNodeActionEventArgs/ITreeNodeActionEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnSelecting наступает
 перед попыткой установить отметку у вершины.
