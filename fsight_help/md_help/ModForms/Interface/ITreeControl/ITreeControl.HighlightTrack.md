# ITreeControl.HighlightTrack


## Синтаксис


HighlightTrack: Boolean;


## Описание


Свойство HighlightTrack определяет,
 подсвечивать ли элемент при наведении указателя мыши.


## Комментарии


Допустимые значения:


	- True. Значение по умолчанию.
	 При наведении указателя мыши элемент будет подсвечен за счет изменения
	 цвета его фона;


	- False. При наведении
	 указателя мыши элемент не подсвечивается.


Также при наведении указателя мыши элемент может быть подсвечен подчеркиванием,
 за это отвечает свойство [HotTrack](ITreeControl.HotTrack.htm).
