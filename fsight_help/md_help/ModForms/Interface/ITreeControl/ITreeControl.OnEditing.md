# ITreeControl.OnEditing


## Синтаксис


OnEditing(Sender: Object; Args: [ITreeNodeActionEventArgs](../ITreeNodeActionEventArgs/ITreeNodeActionEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnEditing наступает
 при переходе в режим редактирования содержимого поля текущей записи компонента.
