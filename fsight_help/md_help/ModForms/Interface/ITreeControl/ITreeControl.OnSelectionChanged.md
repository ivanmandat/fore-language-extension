# ITreeControl.OnSelectionChanged


## Синтаксис


OnSelectionChanged(Sender: Object; Args: [ITreeControlEventArgs](../ITreeControlEventArgs/ITreeControlEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnSelectionChanged наступает
 при изменении отметки у вершины компонента.


## Комментарии


Данное событие наступает в том случае, если произошло событие [OnSelected](ITreeControl.OnSelected.htm)
 или [OnUnselected](ITreeControl.OnUnselected.htm).
