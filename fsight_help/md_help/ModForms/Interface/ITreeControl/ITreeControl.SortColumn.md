# ITreeControl.SortColumn


## Синтаксис


SortColumn: Integer;


## Описание


Свойство SortColumn определяет
 индекс столбца, по которому будет осуществляться сортировка.


## Комментарии


Свойство используется вместе со свойствами [Sorted](ITreeControl.Sorted.htm)
 и [SortType](ITreeControl.SortType.htm).
 Сортировка осуществляется при выполнении метода [SortDefault](ITreeControl.SortDefault.htm).
