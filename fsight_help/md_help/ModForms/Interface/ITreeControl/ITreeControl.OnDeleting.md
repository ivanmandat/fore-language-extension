# ITreeControl.OnDeleting


## Синтаксис


OnDeleting(Sender: Object; Args: [ITreeNodeEventArgs](../ITreeNodeEventArgs/ITreeNodeEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnDeleting наступает
 при удалении элемента.
