# ITreeControl.OnFocusedChanged


## Синтаксис


OnFocusedChanged(Sender: Object; Args: [ITreeControlEventArgs](../ITreeControlEventArgs/ITreeControlEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnFocusedChanged наступает
 после передачи фокуса другой вершине компонента.
