# ITreeControl.OnChecking


## Синтаксис


OnChecking(Sender: Object; Args: [ITreeNodeCheckingEventArgs](../ITreeNodeCheckingEventArgs/ITreeNodeCheckingEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnChecking наступает
 перед изменением состояния флажка у элемента.


## Комментарии


Событие актуально, если свойству [CheckBoxes](ITreeControl.Checkboxes.htm)
 установлено значение True.
