# ITreeControl.ShowLines


## Синтаксис


ShowLines: Boolean;


## Описание


Свойство ShowLines определяет, будет ли отображаться линия, соединяющая вершину с ее потомками.


## Комментарии


Если свойству установлено значение True, линии будут соединять вершины с их потомками.
