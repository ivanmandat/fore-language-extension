# IConditionElements.Item


## Синтаксис


Item(Index: Integer): [IConditionElement](../IConditionElement/IConditionElement.htm);


## Параметры


Index. Индекс элемента группы
 в коллекции.


## Описание


Свойство Item возвращает элемент
 из группы по индексу.


## Комментарии


Для получения коллекции элементов группы используйте свойство [IConditionGroup.Elements](../IConditionGroup/IConditionGroup.Elements.htm).


## Пример


Использование свойства приведено в примере для [ICtrlGroupConditions.InsertRow](../ICtrlGroupConditions/ICtrlGroupConditions.InsertRow.htm).
