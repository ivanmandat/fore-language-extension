# IMenuButton.Menu


## Синтаксис


Menu: [IPopupMenu](../IPopupMenu/IPopupMenu.htm);


## Описание


Свойства Menu определяет пользовательское меню, которое будет отображаться при нажатии на кнопку.


## Комментарии


В список доступных значений данного свойства входят имена всех компонентов [PopupMenu](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/PopupMenu.htm), расположенных на форме.
