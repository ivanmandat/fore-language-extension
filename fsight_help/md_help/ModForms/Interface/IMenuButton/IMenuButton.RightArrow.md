# IMenuButton.RightArrow


## Синтаксис


RightArrow: Boolean;


## Описание


Свойство RightArrow определяет направление стрелок и соответственно расположение всплывающего списка.


## Комментарии


При установке свойству значения True всплывающий список будет располагаться справа от компонента, иначе - снизу от компонента.
