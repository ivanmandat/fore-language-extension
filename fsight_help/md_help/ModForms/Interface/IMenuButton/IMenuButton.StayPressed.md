# IMenuButton.StayPressed


## Синтаксис


StayPressed: Boolean;


## Описание


Свойство StayPressed определяет, будет ли нажата кнопка до момента закрытия всплывающего меню.


## Комментарии


При нажатии на кнопку будет вызвано всплывающее меню. Если данному свойству установлено значение True, то после вызова меню сама кнопка будет оставаться в нажатом состоянии до момента закрытия меню. Если данному свойству установлено значение False, то после вызова меню сама кнопка вернется в исходное состояние. По умолчанию свойству установлено значение False.
