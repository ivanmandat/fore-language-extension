# IRibbonPaletteButtonGroups


Сборка: Forms;


## Описание


Интерфейс IRibbonPaletteButtonGroups
 содержит свойства и методы для работы с коллекцией групп графических кнопок,
 используемых в [комбинированном
 меню](../IRibbonPaletteButton/IRibbonPaletteButton.htm).


## Иерархия наследования


IRibbonPaletteButtonGroups


## Комментарии


Коллекция может быть получена в свойстве [IRibbonPaletteButton.Groups](../IRibbonPaletteButton/IRibbonPaletteButton.Groups.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IRibbonPaletteButtonGroups.Count.htm)
		 Свойство Count возвращает
		 количество групп графических кнопок в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IRibbonPaletteButtonGroups.Item.htm)
		 Свойство Item возвращает
		 параметры указанной группы графических кнопок.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IRibbonPaletteButtonGroups.Add.htm)
		 Метод Add добавляет
		 в коллекцию указанную группу графических кнопок.


		 ![](../../Sub_Image.gif)
		 [Clear](IRibbonPaletteButtonGroups.Clear.htm)
		 Метод Clear удаляет
		 все группы графических кнопок.


		 ![](../../Sub_Image.gif)
		 [Remove](IRibbonPaletteButtonGroups.Remove.htm)
		 Метод Remove удаляет
		 группу графических кнопок по индексу группы в коллекции.
