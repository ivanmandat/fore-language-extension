# IRibbonPaletteButtonGroups.Remove


## Синтаксис


Remove(Index: Integer);


## Параметры


Index. Индекс группы, которую
 необходимо удалить.


## Описание


Метод Remove удаляет группу
 графических кнопок по индексу группы в коллекции.
