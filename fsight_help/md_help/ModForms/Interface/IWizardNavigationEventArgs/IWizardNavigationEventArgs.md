# IWizardNavigationEventArgs


## Описание


Интерфейс IWizardNavigationEventArgs
 содержит свойства аргумента событий, возникающих при переходе по смене
 страницы в компоненте [Wizard](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/Wizard.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [GotoPage](IWizardNavigationEventArgs.GotoPage.htm)


		 Свойство GotoPage определяет
		 индекс страницы, к которой осуществляется переход.
