# IRibbonSlider.VisibleButtons


## Синтаксис


VisibleButtons: Boolean;


## Описание


Свойство VisibleButtons определяет
 признак отображения в элементе управления дополнительных кнопок, которые
 будут использоваться для изменения значения на фиксированную величину.


## Комментарии


Если свойству установлено значение True,
 то по краям элемента управления будут отображены дополнительные кнопки
 ![](Plus_Minus_Buttons.gif). Данные кнопку могут быть использованы
 для изменения текущего значения на 10.


Если свойству установлено значение False,
 то дополнительные кнопки отображаться не будут. Изменить значение можно
 будет с помощью бегунка элемента управления, либо путем явного указания
 значения свойства [Value](IRibbonSlider.Value.htm).
