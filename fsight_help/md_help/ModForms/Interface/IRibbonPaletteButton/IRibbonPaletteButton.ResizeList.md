# IRibbonPaletteButton.ResizeList


## Синтаксис


ResizeList: [RibbonPaletteResize](../../Enums/RibbonPaletteResize.htm);


## Описание


Свойство ResizeList определяет способ изменения размеров комбинированного меню.


## Комментарии


По умолчанию свойству установлено значение [RibbonPaletteResize.None](../../Enums/RibbonPaletteResize.htm), при этом размер раскрытой части меню рассчитывается автоматически таким образом, чтобы все графические кнопки были видны.
