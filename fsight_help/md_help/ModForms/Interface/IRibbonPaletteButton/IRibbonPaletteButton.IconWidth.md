# IRibbonPaletteButton.IconWidth


## Синтаксис


IconWidth: Integer;


## Описание


Свойство IconWidth определяет ширину, в соответствии с которой будет осуществляться разбивка и формирование изображений для основного набора графических кнопок.


## Комментарии


По умолчанию ширина и высота выставляются равными ширине и высоте изображений, содержащихся в компоненте, который задан в свойстве [Images](IRibbonPaletteButton.Images.htm). Свойство IconWidth актуально использовать, если в коллекцию компонента загружено одно или несколько больших изображений, которые необходимо разбить. Значение свойства IconWidth указывается после задания свойства [Images](IRibbonPaletteButton.Images.htm).
