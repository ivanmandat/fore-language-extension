# IRibbonPaletteButton.ColumnCount


## Синтаксис


ColumnCount: Boolean;


## Описание


Свойство ColumnCount определяет количество колонок, в которых будет располагаться набор основных графических кнопок.


## Комментарии


По умолчанию свойству установлено значение «0», при этом кнопки будут располагаться в одной колонке горизонтально. При достижении ограничения по длине, предусмотренного для раскрывающегося меню, кнопки будут перенесены в следующий ряд.


При использовании колонок в каждой отдельной колонке будет расположена только одна графическая кнопка. Кнопки будут расположены в ряд с переносом на следующую строку.
