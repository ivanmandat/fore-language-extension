# IRibbonPaletteButton.SelectedItem


## Синтаксис


SelectedItem: Integer;


## Описание


Свойство SelectedItem определяет
 индекс последней выбранной кнопки из основного набора графических кнопок.


## Комментарии


Основной набор графических кнопок расположен вверху раскрывающегося
 меню и отделен от других элементов управления разделителем. Используя
 данное свойство в событии [Ribbon.OnChange](../../Class/Ribbon/Ribbon.OnChange.htm)
 можно определить, какая из графических кнопок данного элемента управления
 была использована.


Если в качестве значения указать «-1», то ни одна кнопка не будет выбрана.
