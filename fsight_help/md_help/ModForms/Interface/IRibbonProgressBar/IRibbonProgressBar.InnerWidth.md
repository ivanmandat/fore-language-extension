# IRibbonProgressBar.InnerWidth


## Синтаксис


InnerWidth: Integer;


## Описание


Свойство InnerWidth определяет ширину элемента управления.


## Комментарии


Для элементов управления ширина рассчитывается автоматически с учетом количества и параметров соседних элементов управления. Данное свойство позволяет установить фиксированную ширину для текущего элемента управления. Ширина задается в пикселях, по умолчанию индикатор имеет ширину 100 пикселей.
