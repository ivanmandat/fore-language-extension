# IListViewCompareEventArgs.Compare


## Синтаксис


Compare: Integer;


## Описание


Свойство Compare устанавливает признак сортировки. 0 - если равны, меньше 0 - если [LeftItem](IListViewCompareEventArgs.LeftItem.htm) < [RightItem](IListViewCompareEventArgs.RightItem.htm), больше 0 - если [LeftItem](IListViewCompareEventArgs.LeftItem.htm) > [RightItem](IListViewCompareEventArgs.RightItem.htm).
