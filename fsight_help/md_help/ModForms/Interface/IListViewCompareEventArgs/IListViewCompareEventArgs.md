# IListViewCompareEventArgs


## Описание


Интерфейс IListViewCompareEventArgs
 содержит свойства аргумента события, наступающего при сравнении двух элементов,
 во время сортировки в колонке компонента [ListView](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ListView.htm).

		 Имя свойства

		 ![](../../Property_Image.gif)
		 [Compare](IListViewCompareEventArgs.Compare.htm)

		 [RightItem](IListViewCompareEventArgs.RightItem.htm)


		 Свойство RightItem
