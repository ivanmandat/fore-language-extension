# IRibbonQuickAccessToolbar


Сборка: Forms;


## Описание


Интерфейс IRibbonQuickAccessToolbar
 содержит свойства для работы с панелью быстрого доступа.


## Иерархия наследования


           [IComponent](../IComponent/IComponent.htm)


           IRibbonQuickAccessToolbar


## Комментарии


Панель быстрого доступа включает в себя элементы управления, которые
 дублируют какие-либо основные, либо часто используемые команды. Для получения
 параметров панели быстрого доступа доступны используйте свойство [IRibbon.QuickAccessToolbar](../IRibbon/IRibbon.QuickAccessToolbar.htm).
