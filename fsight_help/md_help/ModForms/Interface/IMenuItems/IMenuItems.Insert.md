# IMenuItems.Insert


## Синтаксис


Insert(Index: Integer; Value: [IMenuItem](../IMenuItem/IMenuItem.htm));


## Параметры


Index - параметр, определяющий индекс вставляемого элемента MenuItem.


Value - параметр, определяющий пункт меню, который будет вставлен в компонент.


## Описание


Метод Insert вставляет пункт меню, передаваемый посредством параметра Value, в позицию, индекс которой передается в качестве входного параметра Index.
