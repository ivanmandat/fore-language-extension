# IMenuItems.Add


## Синтаксис


Add(Value: [IMenuItem](../IMenuItem/IMenuItem.htm));


## Параметры


Value - параметр, определяющий пункт меню, который будет добавлен в компонент.


## Описание


Метод Add добавляет в конец списка пункт меню, передаваемый посредством параметра Value.
