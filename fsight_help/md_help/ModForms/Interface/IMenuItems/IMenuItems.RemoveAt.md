# IMenuItems.RemoveAt


## Синтаксис


RemoveAt(Index: Integer);


## Параметры


Index - индекс пункта меню.


## Описание


Метод RemoveAt удаляет пункт меню, индекс которого передается посредством входного параметра Index.
