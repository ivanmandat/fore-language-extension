# IMenuItems.IndexOf


## Синтаксис


IndexOf(Item: [IMenuItem](../IMenuItem/IMenuItem.htm)): Integer;


## Параметры


Item - пункт меню, индекс которого необходимо узнать.


## Описание


Метод IndexOf возвращает индекс указанного пункта меню в коллекции. Если указанного пункта меню в подменю нет, то функция возвращает значение -1.
