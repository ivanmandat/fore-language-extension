# IMenuItems.Remove


## Синтаксис


Remove(Item: [IMenuItem](../IMenuItem/IMenuItem.htm));


## Параметры


Item - пункт меню, который необходимо удалить.


## Описание


Метод Remove удаляет пункт меню, передаваемый посредством входного параметра Item.
