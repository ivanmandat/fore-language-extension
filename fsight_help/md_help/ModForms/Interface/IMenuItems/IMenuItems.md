# IMenuItems


## Описание


Интерфейс IMenuItems содержит
 свойства и методы коллекции элементов меню компонентов [MainMenu](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/MainMenu.htm)
 и [PopupMenu](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/PopupMenu.htm).


## Свойства


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMenuItems.Count.htm)
		 Свойство Count возвращает
		 количество пунктов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMenuItems.Item.htm)
		 Свойство Item возвращает
		 пункт меню.


## Методы


		  Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IMenuItems.Add.htm)
		 Метод Add добавляет
		 в конец списка пункт меню


		 ![](../../Sub_Image.gif)
		 [Clear](IMenuItems.Clear.htm)
		 Метод Clear осуществляет
		 очистку меню.


		 ![](../../Sub_Image.gif)
		 [IndexOf](IMenuItems.IndexOf.htm)
		 Метод IndexOf возвращает
		 индекс указанного пункта меню в коллекции.


		 ![](../../Sub_Image.gif)
		 [Insert](IMenuItems.Insert.htm)
		 Метод Insert вставляет
		 пункт меню в указанную позицию.


		 ![](../../Sub_Image.gif)
		 [Remove](IMenuItems.Remove.htm)
		 Метод Remove удаляет
		 пункт меню.


		 ![](../../Sub_Image.gif)
		 [RemoveAt](IMenuItems.RemoveAt.htm)
		 Метод RemoveAt удаляет
		 пункт меню по индексу.
