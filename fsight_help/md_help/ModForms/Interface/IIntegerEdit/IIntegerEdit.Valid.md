# IIntegerEdit.Valid


## Синтаксис


Valid: Boolean;


## Описание


Свойство Valid возвращает признак корректности целого числа, введенного в поле редактирования.


## Комментарии


Если Valid = False, то поле редактирования подсвечивается красным цветом. Данное свойство доступно только для чтения и используется только через язык Fore.
