# IIntegerEdit.AllowEmpty


## Синтаксис


AllowEmpty: Boolean;


## Описание


Свойство AllowEmpty определяет признак наличия пустого значения в компоненте. Если свойству установлено значение True - пустое значение не подсвечивается красным фоном, при этом свойство [Value](IIntegerEdit.Value.htm) возвращает 0, а свойство [Text](../IControl/IControl.Text.htm) - пустую строку.
