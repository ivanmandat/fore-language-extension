# IIntegerEdit.MaxValue


## Синтаксис


MaxValue: Integer;


## Описание


Свойство MaxValue определяет максимальное значение числа, которое можно установить в компоненте.


## Комментарии


При соблюдении следующего условия в компоненте можно указывать любые значения: [MinValue](IIntegerEdit.MinValue.htm) = MaxValue = 0.
