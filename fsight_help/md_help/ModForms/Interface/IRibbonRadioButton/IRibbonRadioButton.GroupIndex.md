# IRibbonRadioButton.GroupIndex


## Синтаксис


GroupIndex: Integer;


## Описание


Свойство GroupIndex определяет индекс группы, в которую входит текущий элемент управления.


## Комментарии


Если для нескольких переключателей установлено одинаковое значение данного свойства, то переключатели являются сгруппированными в одну группу. Возможен выбор только одного из переключателей, входящих в группу.
