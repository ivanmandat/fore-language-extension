# IBreadcrumbItem.Data


## Синтаксис


Data: Variant;


## Описание


Свойство Data предназначено для хранения пользовательских данных простых типов.


## Комментарии


Значение данного свойства используется при [сохранении](../IBreadcrumb/IBreadcrumb.SaveState.htm)/[загрузке](../IBreadcrumb/IBreadcrumb.LoadState.htm) состояния элементов компонента. Это накладывает определенные ограничения на данные, которые могут храниться в данном свойстве. В свойстве можно хранить данные только простых типов. Хранение коллекций, массивов, объектов системных и пользовательских классов и других данных, не поддерживающих сериализацию в XML, недопустимо.
