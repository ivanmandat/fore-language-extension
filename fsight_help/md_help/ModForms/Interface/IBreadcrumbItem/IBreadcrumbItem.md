# IBreadcrumbItem


## Описание


Интерфейс IBreadcrumbItem содержит
 свойства элемента компонента [Breadcrumb](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/Breadcrumb.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Breadcrumb](IBreadcrumbItem.Breadcrumb.htm)


		 Свойство Breadcrumb
		 возвращает объект-родитель.


		 ![](../../Property_Image.gif)
		 [ButtonLeft](IBreadcrumbItem.ButtonLeft.htm)


		 Свойство находится в стадии разработки.


		 ![](../../Property_Image.gif)
		 [Data](IBreadcrumbItem.Data.htm)


		 Свойство Data предназначено
		 для хранения пользовательских данных простых типов.


		 ![](../../Property_Image.gif)
		 [DownButton](IBreadcrumbItem.DownButton.htm)


		 Свойство DownButton
		 определяет признак отображения рядом с элементом дополнительной
		 кнопки.


		 ![](../../Property_Image.gif)
		 [Height](IBreadcrumbItem.Height.htm)


		 Свойство Height возвращает
		 высоту области элемента в пикселях.


		 ![](../../Property_Image.gif)
		 [Hint](IBreadcrumbItem.Hint.htm)


		 Свойство Hint определяет
		 текст всплывающей подсказки, отображаемой при наведении указателя
		 мыши на элемент.


		 ![](../../Property_Image.gif)
		 [Index](IBreadcrumbItem.Index.htm)


		 Свойство Index возвращает
		 индекс элемента в коллекции элементов компонента [Breadcrumb](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/Breadcrumb.htm).


		 ![](../../Property_Image.gif)
		 [Left](IBreadcrumbItem.Left.htm)


		 Свойство Left возвращает
		 расстояние от левого края компонента до левой границы области
		 элемента.


		 ![](../../Property_Image.gif)
		 [Permanent](IBreadcrumbItem.Permanent.htm)


		 Свойство Permanent
		 определяет, будет ли фиксирован элемент компонента.


		 ![](../../Property_Image.gif)
		 [Text](IBreadcrumbItem.Text.htm)


		 Свойство Text определяет
		 текст элемента.


		 ![](../../Property_Image.gif)
		 [Top](IBreadcrumbItem.Top.htm)


		 Свойство Top возвращает
		 расстояние от верхнего края компонента до верхней границы области
		 элемента.


		 ![](../../Property_Image.gif)
		 [Width](IBreadcrumbItem.Width.htm)


		 Свойство Width возвращает
		 ширину области элемента в пикселях.
