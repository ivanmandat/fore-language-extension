# IAnchors


## Описание


Интерфейс IAnchors содержит
 свойства для определения привязки компонента к границам родительского
 компонента.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Bottom](IAnchors.Bottom.htm)
		 Свойство Bottom определяет
		 процент привязки компонента к нижнему краю родительского компонента.


		 ![](../../Property_Image.gif)
		 [Left](IAnchors.Left.htm)
		 Свойство Left определяет
		 процент привязки компонента к левому краю родительского компонента.


		 ![](../../Property_Image.gif)
		 [Right](IAnchors.Right.htm)
		 Свойство Right определяет
		 процент привязки компонента к правому краю родительского компонента.


		 ![](../../Property_Image.gif)
		 [Top](IAnchors.Top.htm)
		 Свойство Top определяет
		 процент привязки компонента к верхнему краю родительского компонента.
