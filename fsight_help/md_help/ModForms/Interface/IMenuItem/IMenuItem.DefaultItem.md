# IMenuItem.DefaultItem


## Синтаксис


DefaultItem: Boolean;


## Описание


Свойство DefaultItem определяет возможность выполнения по умолчанию данного пункта меню при двойном щелчке мышью на родительском подменю.


## Комментарии


Пункт меню, выполняемый по умолчанию отмечается полужирным начертанием.


Подменю может иметь только один пункт, имеющий значение свойства DefaultItem = True. Если устанавливается это значение для второго пункта подменю, то для предыдущего пункта DefaultItem автоматически изменится на False.
