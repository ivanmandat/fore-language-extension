# IMenuItem.CloseOnCommand


## Синтаксис


CloseOnCommand: Boolean;


## Описание


Свойство CloseOnCommand определяет признак закрытия меню после выбора данного элемента меню.


## Комментарии


По умолчанию свойству установлено значение True, при этом после выбора элемента меню само меню закрывается. Если свойству установлено значение False, то после выбора элемента меню само меню остается раскрытым.
