# IMenuItem.Checked


## Синтаксис


Checked: Boolean;


## Описание


Свойство Checked определяет
 состояние флажка, отображаемого слева от пункта меню.


## Комментарии


По умолчанию значение свойства установлено False
 (флажок выключен).


Если пункт меню использует объект [Action](IMenuItem.Action.htm),
 то значение свойства Checked пункта
 меню автоматически определяется значением свойства [Checked](../IActionItem/IActionItem.Checked.htm)
 объекта [Action](IMenuItem.Action.htm).
