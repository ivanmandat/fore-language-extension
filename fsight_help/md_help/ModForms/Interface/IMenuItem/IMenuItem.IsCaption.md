# IMenuItem.IsCaption


## Синтаксис


IsCaption: Boolean;


## Описание


Свойство IsCaption определяет,
 является ли текущий элемент заголовком.


## Комментарии


Если свойству установлено значение True,
 то текущий элемент будет отображаться
 как заголовок в меню. Текст заголовка отображается с отступом влево на
 более темном фоне. Заголовок не может иметь дочерних элементов и не может
 быть выбран при работе с меню. По умолчанию свойству установлено значение
 False.
