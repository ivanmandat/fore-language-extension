# IMenuItem.ShortCut


## Синтаксис


ShortCut: Integer;


## Описание


Свойство ShortCut определяет
 комбинацию «горячих клавиш» для быстрого выбора пункта меню.


## Комментарии


При создании «горячих клавиш» во время проектирования формы следует
 выбрать в инспекторе объектов свойство ShortCut
 и нажать необходимое сочетание клавиш.


Если пункт меню использует объект [Action](IMenuItem.Action.htm),
 то значение свойства ShortCut
 пункта меню автоматически определяется значением свойства [ShortCut](../IActionItem/IActionItem.ShortCut.htm)
 объекта [Action](IMenuItem.Action.htm).
