# IMenuItem.Text


## Синтаксис


Text: String;


## Описание


Свойство Text определяет строку, идентифицирующую компонент для пользователя.


## Комментарии


В данном свойстве указывается текст, поясняющую пользователю назначение данного компонента. Для того чтобы пункт меню был разделителем, необходимо свойству Text присвоить значение «-».


Если пункт меню использует объект [Action](IMenuItem.Action.htm), то значение свойства Text пункта меню автоматически определяется значением свойства [Caption](../IActionItem/IActionItem.Caption.htm) объекта [Action](IMenuItem.Action.htm).


Примечание. Происходит автоматическое скрытие разделителей, находящихся в начале и в конце меню. Также скрывается один из двух разделителей, идущих подряд.
