# IMenuItem.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет видимость пункта меню во время выполнения формы.


## Комментарии


Если пункт меню использует объект [Action](IMenuItem.Action.htm), то значение свойства Visible пункта меню автоматически определяется значением свойства [Visible](../IActionItem/IActionItem.Visible.htm) объекта [Action](IMenuItem.Action.htm).
