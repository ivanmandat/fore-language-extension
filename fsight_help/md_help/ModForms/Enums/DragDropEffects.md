# DragDropEffects


## Описание


Перечисление DragDropEffects содержит Drag&Drop эффекты, применяемые в определенной области компонентов.


Используется следующими свойствами и методами:


-
[IControl.DoDragDrop](../Interface/IControl/IControl.DoDragDrop.htm)


-
[IDragEventArgs.AllowedEffect](../Interface/IDragEventArgs/IDragEventArgs.AllowedEffect.htm)


-
[IDragEventArgs.Effect](../Interface/IDragEventArgs/IDragEventArgs.Effect.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Место, над которым механизм перетаскивания прекращается, не получает данные.


 1
 Copy. Место, над которым механизм перетаскивания прекращается, получает копию данных. Источник данных остается без изменений.


 2
 Move. Место, над которым механизм перетаскивания прекращается, получает копию данных. Источник данных очищается.


 4
 Link. В том месте, над которым механизм перетаскивания прекращается, создается ссылка к оригинальным данным.


 -2147483648
 Scroll. Место, над которым механизм перетаскивания прекращается, начинает или продолжает скроллироваться, в зависимости от предыдущего состояния. Это значение используется для добавления других значений.
