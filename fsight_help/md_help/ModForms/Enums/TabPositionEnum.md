# TabPositionEnum


## Описание


Перечисление TabPositionEnum
 содержит места расположения заголовков вкладок.


Используется следующим свойством:


	- [IPageControl.TabPosition](../Interface/IPageControl/IPageControl.TabPosition.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Top. Заголовки располагаются
		 вдоль верхней стороны компонента.


		 1
		 Bottom. Заголовки располагаются
		 вдоль нижней стороны компонента.


		 2
		 Left. Заголовки располагаются
		 вдоль левой стороны компонента.


		 3
		 Right. Заголовки располагаются
		 вдоль правой стороны компонента.
