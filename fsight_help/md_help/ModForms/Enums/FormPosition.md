# FormPosition


## Описание


Перечисление FormPosition содержит
 места расположения окна формы на экране.


Используется следующим свойством:


	- [IFormControl.Position](../Interface/IFormControl/IFormControl.Position.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Designe. Расположение
		 формы и её размеры после открытия соответствуют установленным
		 в процессе создания формы.


		 1
		 ScreenCenter. Выравнивание
		 по центру относительно рабочей области.


		 2
		 DesktopCenter. Выравнивание
		 по центру относительно всего рабочего стола.


		 3
		 OwnerFormCenter. Выравнивание
		 в горизонтальном направлении по центру родительской формы.
