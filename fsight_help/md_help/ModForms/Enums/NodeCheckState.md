# NodeCheckState


## Описание


Перечисление NodeCheckState содержит состояние отметки вершины дерева.


Используется следующими свойствами и методами:


-
[IDimensionViewer.CheckState](KeExtCtrls.chm::/Interface/IDimensionViewer/IDimensionViewer.CheckState.htm)


-
[ITreeListNode.CheckState](../Interface/ITreeListNode/ITreeListNode.CheckState.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Unchecked. Отмечена.


 1
 Checked. Не отмечена.


 2
 Indeterminate. Неопределенное состояние отметки.
