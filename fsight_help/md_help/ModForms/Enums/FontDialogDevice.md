# FontDialogDevice


## Описание


Перечисление FontDialogDevice содержит шрифты, которые будут входить в список доступных шрифтов при инициализации диалога выбора шрифта.


Используется следующим свойством:


-
[IFontDialog.Device](../Interface/IFontDialog/IFontDialog.Device.htm)


## Допустимые значения


 Значение
 Краткое описание


 1
 Screen. Экранные шрифты. Шрифты, предназначенные для отображения на экране монитора. Как правило, экранным шрифтам сопутствуют шрифты PostScript, используемые для печати на PostScript-совместимых принтерах.


 2
 Printer. Шрифты принтера. Шрифты, хранящиеся в принтере и предназначенные для использования только на нем.


 3
 Both. Все шрифты.
