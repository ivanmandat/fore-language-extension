# ControlSortType


## Описание


Перечисление ControlSortType
 содержит типы сортировки, применяемой к элементам компонентов.


Используется следующими свойствами и методами:


	- [IDimensionViewer.SortType](KeExtCtrls.chm::/Interface/IDimensionViewer/IDimensionViewer.SortType.htm);


	- [ITreeControl.SortType](../Interface/ITreeControl/ITreeControl.SortType.htm);


	- [IMetaAttributeSetting.SortType](KeExtCtrls.chm::/Interface/IMetaAttributeSetting/IMetaAttributeSetting.SortType.htm);


	- [IListView.SortType](../Interface/IListView/IListView.SortType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Нет сортировки.


		 1
		 Text. Сортировка по
		 тексту элементов. Все значения элементов рассматриваются как строковые.


		 2
		 Custom. Пользовательская
		 сортировка через событие.
