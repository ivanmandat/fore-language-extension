# TrackBarOrientation


## Описание


Перечисление TrackBarOrientation
 содержит варианты ориентации компонента [TrackBar](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TrackBar.htm)
 на форме.


Используется следующим свойством:


	- [ITrackBar.Orientation](../Interface/ITrackBar/ITrackBar.Orientation.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Horizontal. Компонент
		 располагается горизонтально.


		 1
		 Vertical. Компонент
		 располагается вертикально.
