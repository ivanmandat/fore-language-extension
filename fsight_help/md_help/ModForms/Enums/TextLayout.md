# TextLayout


## Описание


Перечисление TextLayout содержит
 варианты выравнивания текста по вертикали внутри области компонента.


Используется следующим свойством:


	- [ILabel.Layout](../Interface/ILabel/ILabel.Layout.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Top. Выравнивание текста
		 по верхнему краю.


		 1
		 Center. Выравнивание
		 текста по центру.


		 2
		 Bottom. Выравнивание
		 текста по нижнему краю.
