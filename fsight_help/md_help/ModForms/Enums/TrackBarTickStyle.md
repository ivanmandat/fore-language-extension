# TrackBarTickStyle


## Описание


Перечисление TrackBarTickStyle
 содержит стили нанесения меток.


Используется следующим свойством:


	- [ITrackBar.TickStyle](../Interface/ITrackBar/ITrackBar.TickStyle.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Меток нет.


		 1
		 Auto. Количество отображаемых
		 интервалов и место расположения меток вычисляется автоматически,
		 в зависимости от значений свойств [Min](../Interface/ITrackBar/ITrackBar.Min.htm),
		 [Max](../Interface/ITrackBar/ITrackBar.Max.htm) и [Frequence](../Interface/ITrackBar/ITrackBar.Frequency.htm).


		 2
		 Manual. Наносятся начальная
		 и конечная метка. Место расположения остальных меток задается
		 с помощью метода [SetTick](../Interface/ITrackBar/ITrackBar.SetTick.htm).
