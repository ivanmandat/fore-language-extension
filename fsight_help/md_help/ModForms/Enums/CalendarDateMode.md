# CalendarDateMode


## Описание


Перечисление CalendarDateMode
 содержит режимы выбора даты.


Используется следующим свойством:


	- [IDateTimePicker.DateMode](../Interface/IDateTimePicker/IDateTimePicker.DateMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 ComboBox. Компонент
		 изображается в виде поля со списком. Раскрывающийся список содержит
		 календарь, в котором пользователь может выбрать дату.


		 1
		 UpDown. Компонент изображается
		 в виде счетчика.
