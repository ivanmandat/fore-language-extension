# MonthStyle


## Описание


Перечисление MonthStyle содержит стили отображения наименования месяца поле редактирования компонента [DateTimePickerEx](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/DateTimePickerEx.htm).


Используется следующим свойством:


-
[IDateTimePickerEx.MonthFormat](../Interface/IDateTimePickerEx/IDateTimePickerEx.MonthFormat.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Short. Сокращенное наименование месяца (01. янв. 2000).


 1
 Long. Полное наименование месяца (01. Январь .2000).


 2
 Numeric. Числовое обозначение месяца (01.01.2000).
