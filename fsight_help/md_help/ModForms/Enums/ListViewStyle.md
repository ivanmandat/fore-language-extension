# ListViewStyle


## Описание


Перечисление ListViewStyle содержит стили отображения элементов компонента [ListView](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ListView.htm).


Используется следующими свойствами и методами:


-
[IListView.Style](../Interface/IListView/IListView.Style.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Icon. Крупные пиктограммы.


 1
 SmallIcon. Мелкие пиктограммы.


 2
 List. Список пиктограмм.


 3
 Report. Таблица пиктограмм.
