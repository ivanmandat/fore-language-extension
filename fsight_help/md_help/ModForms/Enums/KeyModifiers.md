# KeyModifiers


## Описание


Перечисление содержит модификаторы клавиш, которые могут использоваться при составлении комбинаций клавиш.


Используется следующим свойством:


-
[IFormClass.MakeShortcut](../Interface/IFormClass/IFormClass.MakeShortcut.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Дополнительная клавиша не используется.


 1
 Shift. Используется с клавишей SHIFT.


 2
 Ctrl. Используется с клавишей CTRL.


 4
 Alt. Используется с клавишей ALT.
