# DragDropFormat


## Описание


Перечисление DragDropFormat содержит форматы, в которых будут передаваться данные при перетаскивании. Актуально при работе с объектами репозитория.


Используется следующим методом:


-
[IControl.DoDragDrop](../Interface/IControl/IControl.DoDragDrop.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 User. Передавать в навигатор объектов данные в том виде, в каком они указаны в параметре Data.


 1
 MetabaseObjectList. Если в качестве значения параметра Data передается коллекция описаний объектов, то навигатором объектов данная коллекция будет восприниматься как коллекция соответствующих объектов репозитория.
