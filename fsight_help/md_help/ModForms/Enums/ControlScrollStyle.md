# ControlScrollStyle


## Описание


Перечисление ControlScrollStyle
 содержит варианты расположения полос прокрутки в компоненте.


Используется следующими свойствами и методами:


	- [IMemo.ScrollBars](../Interface/IMemo/IMemo.ScrollBars.htm)


	- [IScrollBars.ScrollBars](../Interface/IScrollBars/IScrollBars.ScrollBars.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Полосы прокрутки
		 отсутствуют.


		 1
		 Horizontal. Наличие
		 у компонента горизонтальной полосы прокрутки.


		 2
		 Vertical. Наличие у
		 компонента вертикальной полосы прокрутки.


		 3
		 Both. Наличие у компонента
		 двух (вертикальной и горизонтальной) полос прокрутки.
