# ControlBorderType


## Описание


Перечисление ControlBorderType содержит типы границ, которые могут быть отрисованы для компонентов.


Используется следующим свойством:


-
[INavigationBar.Border](../Interface/INavigationBar/INavigationBar.Border.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Границы отсутствуют.


 1
 Left. Отображается левая граница.


 2
 Top. Отображается верхняя граница.


 4
 Right. Отображается правая граница.


 8
 Bottom. Отображается нижняя граница.


 15
 All. Отображаются все границы.
