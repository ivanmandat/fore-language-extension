# DaysOfWeek


## Описание


Перечисление DaysOfWeek содержит дни, с которых будет начинаться неделя при отображении раскрывающегося календаря компонента [DateTimePickerEx](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/DateTimePickerEx.htm).


Используется следующим свойством:


-
[IDateTimePickerEx.FirstDayOfWeek](../Interface/IDateTimePickerEx/IDateTimePickerEx.FirstDayOfWeek.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Sunday. Понедельник.


 1
 Monday. Вторник.


 2
 Tuesday. Среда.


 3
 Wednesday. Четверг.


 4
 Thursday. Пятница.


 5
 Friday. Суббота.


 6
 Saturday. Воскресенье.
