# DateTimePickerKind


## Описание


Перечисление DateTimePickerKind содержит типы данных, которые будут присутствовать в компоненте [DateTimePicker](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/DateTimePicker.htm).


Используется следующим свойством:


-
[IDateTimePicker.Kind](../Interface/IDateTimePicker/IDateTimePicker.Kind.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Date. Компонент служит для ввода или выбора даты.


 1
 Time. Компонент служит для ввода или выбора времени.
