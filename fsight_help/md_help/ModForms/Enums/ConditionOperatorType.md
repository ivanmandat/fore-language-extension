# ConditionOperatorType


## Описание


Перечисление ConditionOperatorType
 используется для определения оператора объединения при работе с элементами
 компонента [CtrlGroupConditions](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/CtrlGroupConditions.htm).


Используется следующими свойствами и методами:


	- [ICtrlGroupConditions.AddRow](../Interface/ICtrlGroupConditions/ICtrlGroupConditions.AddRow.htm);


	- [ICtrlGroupConditions.InsertRow](../Interface/ICtrlGroupConditions/ICtrlGroupConditions.InsertRow.htm);


	- [IConditionGroup.Operator_](../Interface/IConditionGroup/IConditionGroup.Operator_.htm);


	- [IConditionElement.Operator_](../Interface/IConditionElement/IConditionElement.Operator_.htm).
