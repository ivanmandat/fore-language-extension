# ControlBorderStyle


## Описание


Перечисление ControlBorderStyle
 содержит стили границы различных компонентов.


## Допустимые значения


		 Значение


		 Краткое
		 описание


		 0


		 None. Компонент
		 не имеет рамки.


		![](ControlBorderStyle_None.gif)


		 1


		 Single. Компонент
		 имеет объемный вид.


		![](ControlBorderStyle_Single.gif)


		 2


		 Flat. Компонент
		 имеет рамку, толщиной в один пиксель.


		![](ControlBorderStyle_Flat.gif)
