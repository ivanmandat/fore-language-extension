# DateTimePickerExMode


## Описание


Перечисление DateTimePickerExMode
 содержит режимы выбора даты в календаре.


Используется следующими свойствами и методами:


	- [IDateTimePickerEx.Mode](../Interface/IDateTimePickerEx/IDateTimePickerEx.Mode.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Days. Выбор даты по
		 дню в месяце.


		 1
		 Weeks. Выбор даты по
		 номеру недели в рамках полугодия.
		Примечание.
		 Номер недели определяется согласно ГОСТ ИСО 8601-2001: Неделя
		 начинается с понедельника и идентифицируется своим порядковым
		 номером в году. Первой календарной неделей года считают первую
		 неделю, содержащую первый четверг текущего года.


		 2
		 Months. Выбор даты
		 по месяцу в году.


		 3
		 Quarters. Выбор даты
		 по кварталу в году.


		 4
		 HalfYears. Выбор даты
		 по полугодию.


		 5
		 Years. Выбор даты по
		 годам.
