# PopupAlignment


## Описание


Перечисление PopupAlignment содержит места появления контекстного меню относительно курсора мыши.


Используется следующим свойством:


-
[IPopupMenu.Alignment](../Interface/IPopupMenu/IPopupMenu.Alignment.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Left. Меню располагается справа, верхний левый угол совпадает с курсором мыши.


 2
 Right. Меню располагается слева, верхний правый угол совпадает с курсором мыши.
