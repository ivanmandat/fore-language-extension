# ControlAlign


## Описание


Перечисление ControlAlign содержит варианты выравнивания компонента в рамках содержащего его родительского компонента.


Используется следующим свойством:


-
[IControl.Align](../Interface/IControl/IControl.Align.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Выравнивание отсутствует.


 1
 Top. Выравнивание в соответствии с верхней частью родительского компонента.


 2
 Bottom. Выравнивание в соответствии с нижней частью родительского компонента.


 3
 Left. Выравнивание в соответствии с левой частью родительского компонента.


 4
 Right. Выравнивание в соответствии с правой частью родительского компонента.


 5
 Client. Выравнивание в соответствии со свободной клиентской областью родительского компонента.
