# MouseButton


## Описание


Перечисление MouseButton содержит код кнопки мыши, задействованной в каком-либо событии.


Используется следующим свойством:


-
[IBreadcrumbMouseEventArgs.Button](../Interface/IBreadcrumbMouseEventArgs/IBreadcrumbMouseEventArgs.Button.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Left. Левая кнопка мыши.


 1
 Right. Правая кнопка мыши.


 2
 Middle. Средняя кнопка мыши (ролик).
