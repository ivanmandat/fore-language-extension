# NavigationBarHeaderContent


## Описание


Перечисление NavigationBarHeaderContent содержит типы элементов управления, которые могут присутствовать в заголовке панели.


Используется следующими свойствами и методами:


-
[INavigationBarHeader.Content](../Interface/INavigationBarHeader/INavigationBarHeader.Content.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Custom. В заголовке отображается элемент управления, заданный в свойстве [CustomContent](../Interface/INavigationBarHeader/INavigationBarHeader.CustomContent.htm). Если свойство [CustomContent](../Interface/INavigationBarHeader/INavigationBarHeader.CustomContent.htm) не задано, то в заголовке отображается только текст.


 1
 CheckBox. В заголовке отображается флажок.
