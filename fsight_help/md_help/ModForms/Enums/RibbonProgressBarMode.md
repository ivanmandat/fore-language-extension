# RibbonProgressBarMode


## Описание


Перечисление RibbonProgressBarMode
 содержит режимы отображения индикатора процесса.


Используется следующим свойством:


	- [IRibbonProgressBar.Mode](../Interface/IRibbonProgressBar/IRibbonProgressBar.Mode.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Normal. Обычный режим.
		 При изменении значения свойства [Value](../Interface/IRibbonProgressBar/IRibbonProgressBar.Value.htm)
		 будет отображаться индикатор в виде непрерывной полосы.

![](RibbonProgressBarMode_Normal.gif)


		 1
		 Infinite. Бесконечный
		 режим. При изменении значения свойства [Value](../Interface/IRibbonProgressBar/IRibbonProgressBar.Value.htm)
		 будет отображаться переливающийся индикатор, символизирующий длительный
		 процесс.

![](RibbonProgressBarMode_Infinite.gif)


Визуальное отображение переливающегося индикатора в бесконечном режиме
 доступно при использовании следующих тем оформления:


	- Microsoft Office 2007.


	- Microsoft Office 2010.


	- Carbon.


	- Windows 7 (при использовании операционной системы Windows 7
	 и включенной темы оформления Aero).


При других темах оформления визуальное отображение индикатора в бесконечном
 режиме будет совпадать с индикатором в обычном режиме.
