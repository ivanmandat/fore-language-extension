# LeftRight


## Описание


Перечисление LeftRight тип содержит
 варианты расположения текста в области компонентов [CheckBox](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/CheckBox.htm)
 и [RadioButton](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/RadioButton.htm).


Используется следующими свойствами и методами:


	- [ICheckBox.Alignment](../Interface/ICheckBox/ICheckBox.Alignment.htm).


	- [IRadioButton.Alignment](../Interface/IRadioButton/IRadioButton.Alignment.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Left. Текст расположен
		 в левой части, флажок - в правой части компонента.


		 2
		 Right. Текст расположен
		 в правой части, флажок - в левой части компонента.
