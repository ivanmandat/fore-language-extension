# ProgressBarOrientation


## Описание


Перечисление ProgressBarOrientation содержит варианты ориентации компонента [ProgressBar](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ProgressBar.htm) на форме.


Используется следующим свойством:


-
[IProgressBar.Orientation](../Interface/IProgressBar/IProgressBar.Orientation.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Horizontal. Индикатор расположен горизонтально.


 1
 Vertical. Индикатор расположен вертикально.
