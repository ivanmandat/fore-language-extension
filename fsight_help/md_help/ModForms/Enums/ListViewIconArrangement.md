# ListViewIconArrangement


## Описание


Перечисление ListViewIconArrangement содержит способы выравнивания элементов в компоненте [ListView](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ListView.htm).


Используется следующим свойством:


-
[IListView.Arrangement](../Interface/IListView/IListView.Arrangement.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Top. Элементы выравниваются по верхнему краю компонента.


 1
 Left. Элементы выравниваются по левому краю компонента.
