# MdiTileType


## Описание


Перечисление MdiTileType содержит типы расположения дочерних окон в родительской MDI-форме.


Используется следующим методом:


-
[IFormControl.MDITile](../Interface/IFormControl/IFormControl.MDITile.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Vertical. Вертикальное расположение окон.


 1
 Horizontal. Горизонтальное расположение окон.


 2
 SkipDisabled. Вертикальное расположение всех не заблокированных окон (Окон, для которых [Enable](../Interface/IControl/IControl.Enabled.htm) = True).
