# DropAnimationType


## Описание


Перечисление DropAnimationType содержит типы анимаций, применяемые при разворачивании объектов.


## Комментарии


При необходимости можно использовать комбинации различных видов анимаций. Для составления комбинации необходимо использовать логическую операцию Or.


Используется следующими свойствами и методами:


-
[IDropForm.AnimationType](../Interface/IDropForm/IDropForm.AnimationType.htm)


-
[ITreeCombo.AnimationType](../Interface/ITreeCombo/ITreeCombo.AnimationType.htm)


-
[IDimensionCombo.AnimationType](KeExtCtrls.chm::/Interface/IDimensionCombo/IDimensionCombo.AnimationType.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Анимация не используется.


 1
 LeftToRight. Раскрытие происходит слева направо.


 2
 RightToLeft. Раскрытие происходит справа налево.


 4
 TopToBottom. Раскрытие происходит сверху вниз.


 8
 BottomToTop. Раскрытие происходит снизу вверх.


 16
 Center. Раскрытие происходит из центра.


 262144
 Slide. Эффект выезжающего объекта. Используется только в сочетании с другими видами анимации.


 524288
 Blend. Раскрытие происходит с изменением прозрачности.
