# BevelShape


## Описание


Перечисление BevelShape содержит внешние виды компонента [Bevel](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/Bevel.htm).


Используется следующим свойством:


-
[IBevel.Shape](../Interface/IBevel/IBevel.Shape.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Box. Компонент отображается как выпуклый или вдавленный блок, в зависимости от значения свойства Style.


 1
 Frame. Компонент обрамляется приподнятой или «утопленной» рамкой.


 2
 TopLine. Отображается только верхняя граница компонента.


 3
 BottomLine. Отображается только нижняя граница компонента.


 4
 LeftLine. Отображается только левая граница компонента.


 5
 RightLine. Отображается только правая граница компонента.


 6
 Spacer. Компонент никак не выделяется.
