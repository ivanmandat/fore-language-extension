# FindModeType


## Описание


Перечисление FindModeType содержит режимы отметки найденных элементов.


Используется следующими свойствами и методами:


-
[ITreeControl.FindMode](../Interface/ITreeControl/ITreeControl.FindMode.htm)


-
[IDimensionViewer.FindMode](KeExtCtrls.chm::/Interface/IDimensionViewer/IDimensionViewer.FindMode.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Select_. Найденный элемент будет сфокусирован и выделен. Установленная в компоненте отметка при этом сбрасывается.


 1
 Focus. Найденный элемент будет сфокусирован. Установленная в компоненте отметка сохраняется.
