# FormModalResult


## Описание


Перечисление FormModalResult содержит результаты выполнения модальной формы.


Используется следующими свойствами и методами:


-
[IButton.ModalResult](../Interface/IButton/IButton.ModalResult.htm)


-
[IFormControl.ModalResult](../Interface/IFormControl/IFormControl.ModalResult.htm)


-
[IFormControl.ShowModal](../Interface/IFormControl/IFormControl.ShowModal.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 None. При закрытии модальной формы, результат «None».


 1
 OK. При закрытии модальной формы, результат «OK».


 2
 Cancel. При закрытии модальной формы, результат «Cancel».


 3
 Abort. При закрытии модальной формы, результат «Abort».


 4
 Retry. При закрытии модальной формы, результат «Retry».


 5
 Ignore. При закрытии модальной формы, результат «Ignore».


 6
 Yes. При закрытии модальной формы, результат «Yes».


 7
 No. При закрытии модальной формы, результат «No».


 8
 All. При закрытии модальной формы, результат «All».
