# RibbonElementOrientation


## Описание


Перечисление RibbonElementOrientation содержит варианты ориентации элемента управления.


Используется следующим свойством:


-
[IRibbonSlider.Orientation](../Interface/IRibbonSlider/IRibbonSlider.Orientation.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Horizontal. Элемент управления расположен горизонтально.


 1
 Vertical. Элемент управления расположен вертикально.
