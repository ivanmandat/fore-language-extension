# RibbonPaletteResize


## Описание


Перечисление RibbonPaletteResize содержит способы изменения размера комбинированного меню.


Используется следующим свойством:


-
[IRibbonPaletteButton.ResizeList](../Interface/IRibbonPaletteButton/IRibbonPaletteButton.ResizeList.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Размеры комбинированного меню изменять нельзя.


 1
 Vertical. Список комбинированного меню можно изменить по вертикали.


 2
 Both. Список комбинированного меню можно изменить вниз и в правую сторону.
