# ComboboxStyle


## Описание


Перечисление ComboboxStyle содержит стили работы комбинированных списков.


Используется следующими свойствами и методами:


-
[IComboBox.Style](../Interface/IComboBox/IComboBox.Style.htm)


-
[IToolbarComboBox.Style](../Interface/IToolbarComboBox/IToolbarComboBox.Style.htm)


-
[IRibbonComboBox.Style](../Interface/IRibbonComboBox/IRibbonComboBox.Style.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Simple. В компоненте присутствует поле редактирования. Ниже этого поля всегда отображается список элементов компонента (Для отображения списка необходимо увеличить высоту компонента).
Пользователь может ввести новое значение или выбрать существующий элемент. При наборе текста также производится поиск и фокусирование на элементе с подходящим текстом.


 1
 DropDown. Комбинированный список со строками одинаковой высоты и окном редактирования, позволяющим пользователю вводить собственный или редактировать уже имеющийся текст.


 2
 DropDownList. Раскрывающийся список со строками одинаковой высоты, не содержащий окно редактирования. Пользователь может выбирать только одно значение из заранее созданного списка строк (свойство компонента [Items](../Interface/IComboBox/IComboBox.Items.htm)). Пользователь не может вводить собственный, или редактировать уже имеющийся, текст.
