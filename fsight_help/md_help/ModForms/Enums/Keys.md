# Keys


## Описание


Перечисление Keys содержит коды клавиш клавиатуры.


Используется следующими свойствами и методами:


-
[IWinApplicationClass.GetAsyncKeyState](UiLib.chm::/Interface/IWinApplicationClass/IWinApplicationClass.GetAsyncKeyState.htm)


-
[IWinApplicationClass.GetKeyState](UiLib.chm::/Interface/IWinApplicationClass/IWinApplicationClass.GetKeyState.htm)


-
[IFormClass.MakeShortcut](../Interface/IFormClass/IFormClass.MakeShortcut.htm)
