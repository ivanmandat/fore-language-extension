# RibbonMainButtonColorStyle


## Описание


Перечисление RibbonMainButtonColorStyle
 содержит стили графического оформления кнопки приложения.


Используется следующими свойствами и методами:


	- [IRibbonMainButton.ColorStyle](../Interface/IRibbonMainButton/IRibbonMainButton.ColorStyle.htm).
