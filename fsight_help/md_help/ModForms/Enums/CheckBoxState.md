# CheckBoxState


## Описание


Перечисление CheckBoxState содержит
 состояния флажка компонента [CheckBox](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/CheckBox.htm).


Используется следующим свойством:


	- [ICheckBox.State](../Interface/ICheckBox/ICheckBox.State.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Unchecked. Флажок не
		 отмечен.


		 1
		 Checked. Флажок отмечен.


		 2
		 Grayed. Промежуточное
		 состояние флажка.
