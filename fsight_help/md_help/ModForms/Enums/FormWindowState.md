# FormWindowState


## Описание


Перечисление FormWindowState содержит варианты состояния формы на экране во время запуска.


Используется следующим свойством:


-
[IFormControl.WindowState](../Interface/IFormControl/IFormControl.WindowState.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Normal. Обычное состояние формы.


 1
 Minimized. Форма свернута.


 2
 Maximized. Форма развернута на весь экран.
