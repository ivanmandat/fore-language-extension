# BevelStyle


## Описание


Перечисление BevelStyle содержит стили оформления компонента [Bevel](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/Bevel.htm).


Используется следующим свойством:


-
[IBevel.Style](../Interface/IBevel/IBevel.Style.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Lowered. Компонент отображается вдавленным.


 1
 Raised. Компонент отображается выпуклым.
