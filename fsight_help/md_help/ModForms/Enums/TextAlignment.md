# TextAlignment


## Описание


Перечисление TextAlignment содержит
 способы выравнивания текста в компоненте.


Используется следующими свойствами и методами:


	- [IDimensionViewerColumn.Alignment](KeExtCtrls.chm::/Interface/IDimensionViewerColumn/IDimensionViewerColumn.Alignment.htm)


	- [ILabel.Alignment](../Interface/ILabel/ILabel.Alignment.htm)


	- [IListViewColumn.Alignment](../Interface/IListViewColumn/IListViewColumn.Alignment.htm)


	- [IMemo.Alignment](../Interface/IMemo/IMemo.Alignment.htm)


	- [IPanel.Alignment](../Interface/IPanel/IPanel.Alignment.htm)


	- [IRdsDictionaryBoxColumn.Alignment](KeRds.chm::/Interface/IRdsDictionaryBoxColumn/IRdsDictionaryBoxColumn.Alignment.htm)


	- [ITreeListColumn.Alignment](../Interface/ITreeListColumn/ITreeListColumn.Alignment.htm)


	- [IRibbonComboBox.TextAlign](../Interface/IRibbonComboBox/IRibbonComboBox.TextAlign.htm)


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Left.
		 Выравнивание текста по левому краю.


		 1
		 Center.
		 Выравнивание текста по центру.


		 2
		 Right.
		 Выравнивание текста по правому краю.
