# FormWindowStyle


## Описание


Перечисление FormWindowStyle
 содержит стили формы.


Используется следующим свойством:


	- [IFormControl.WindowStyle](../Interface/IFormControl/IFormControl.WindowStyle.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Normal. Обычная Windows
		 форма.


		 1
		 MDIFrame. Форма является
		 главной MDI-формой.


		 2
		 MDIChild. Форма является
		 дочерней MDI-формой.
