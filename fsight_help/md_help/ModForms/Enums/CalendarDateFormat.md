# CalendarDateFormat


## Описание


Перечисление CalendarDateFormat содержит форматы даты, в соответствии с которыми будет отображаться значение.


Используется следующим свойством:


-
[IDateTimePicker.DateFormat](../Interface/IDateTimePicker/IDateTimePicker.DateFormat.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Long. Отображает дату в формате дд. месяц. гг (Пример: 31 мая 2004г.)


 1
 Short. Отображает дату в формате дд.мм.гг (Пример: 31.07.04)
