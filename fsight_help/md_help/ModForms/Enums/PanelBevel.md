# PanelBevel


## Описание


Перечисление PanelBevel содержит эффекты, применяемый для границ компонента [Panel](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/Panel.htm).


Используется следующими свойствами и методами:


-
[IPanel.BevelInner](../Interface/IPanel/IPanel.BevelInner.htm)


-
[IPanel.BevelOuter](../Interface/IPanel/IPanel.BevelOuter.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Эффект отсутствует.


 1
 Lowered. Вдавленная рамка.


 2
 Raised. Выпуклая рамка.
