# OperationTextLayout


## Описание


Перечисление OperationTextLayout содержит места расположения таймера процесса относительно текста.


Используется следующим методом:


-
[IFormControl.BeginOperation](../Interface/IFormControl/IFormControl.BeginOperation.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 TextOnly. Отображается только текст.


 1
 TextTime. Таймер процесса отображается справа от текста.


 2
 TimeText. Таймер процесса отображается слева от текста.
