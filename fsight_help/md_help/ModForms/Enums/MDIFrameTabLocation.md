# MDIFrameTabLocation


## Описание


Перечисление MDIFrameTabLocation содержит варианты месторасположения вкладок дочерних MDI-окон в области главной MDI-формы.


Используется следующим свойством:


-
[IFormControl.MDITabLocation](../Interface/IFormControl/IFormControl.MDITabLocation.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Top. Вкладки располагаются в верхней части главной MDI-формы.


 1
 Bottom. Вкладки располагаются в нижней части главной MDI-формы.
