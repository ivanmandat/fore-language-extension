# ShellSpecialFolder


## Описание


Перечисление содержит список системных папок, которые будут корневыми при открытии стандартного окна Windows открытия папки.


Используется следующим свойством:


-
[IFolderBrowserDialog.RootFolder](../Interface/IFolderBrowserDialog/IFolderBrowserDialog.RootFolder.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 ApplicationData. Каталог, выполняющий функции общего резервирования для данных приложения текущего подвижного пользователя.
Подвижный пользователь работает на нескольких компьютерах сети. Профиль подвижного пользователя хранится на сервере сети и загружается в систему при подключении пользователя.


 1
 CommonApplicationData. Каталог, выполняющий функции общего хранилища для данных приложения, используемого всеми пользователями.


 2
 CommonProgramFiles. Каталог для компонентов, общих для приложений.


 3
 Cookies. Каталог, служащий общим хранилищем файлов «cookies».


 4
 Desktop. Логический рабочий стол, а не физическое местоположение файлов системы.


 5
 DesktopDirectory. Каталог, используемый для физического хранения объектов файла рабочего стола.
Следует различать этот каталог и папку рабочего стола, которая является виртуальной папкой.


 6
 Favorites. Каталог, служащий в качестве общего хранилища для избранных элементов пользователя.


 7
 History. Каталог, служащий общим хранилищем элементов журнала Интернета.


 8
 InternetCache. Каталог, служащий общим хранилищем временных файлов Интернет.


 9
 LocalApplicationData. Каталог, выполняющий функции общего хранилища для данных приложения, используемых текущим неподвижным пользователем.


 10
 MyComputer. Папка «Мой компьютер».


 11
 MyMusic. Папка «Моя музыка».


 12
 MyPictures - Папка «Мои фотографии».


 13
 Personal. Каталог, служащий в качестве общего хранилища для документов.


 14
 ProgramFiles. Каталог файлов программ.


 15
 Programs. Каталог, содержащий группы программ пользователя.


 16
 Recent. Каталог, содержащий ярлыки на последние используемые документы.


 17
 SendTo. Каталог, содержащий пункт меню «Отправить».


 18
 StartMenu. Каталог, содержащий пункты меню «Пуск».


 19
 Startup. Каталог, соответствующий группе программ автозагрузки пользователя.


 20
 System. Системный каталог Windows.


 21
 Templates. Каталог, служащий в качестве общего хранилища для шаблонов документов.
