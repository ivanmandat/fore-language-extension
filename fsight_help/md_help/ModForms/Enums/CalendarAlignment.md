# CalendarAlignment


## Описание


Перечисление CalendarAlignment содержит варианты выравнивания раскрывающегося календаря.


Используется следующим свойством:


-
[IDateTimePicker.Alignment](../Interface/IDateTimePicker/IDateTimePicker.Alignment.htm)


-
[IDateTimePickerEx.Alignment](../Interface/IDateTimePickerEx/IDateTimePickerEx.Alignment.htm)


## Допустимые значения


 Значение
 Краткое описание


 0
 Left. Раскрывающийся календарь выравнивается по левому краю компонента.


 1
 Right. Раскрывающийся календарь выравнивается по правому краю компонента.
