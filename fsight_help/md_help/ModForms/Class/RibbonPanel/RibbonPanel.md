# RibbonPanel


Сборка: Forms;


## Описание


Класс RibbonPanel реализует элемент ленты - Панель.


## Свойства объекта класса, унаследованные от [IRibbonPanel](../../Interface/IRibbonPanel/IRibbonPanel.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CenterColumnsVertically](../../Interface/IRibbonPanel/IRibbonPanel.CenterColumnsVertically.htm)
		 Свойство CenterColumnsVertically
		 определяет, будут ли элементы управления выровнены по центру относительно
		 вертикали панели.


		 ![](../../Property_Image.gif)
		 [Elements](../../Interface/IRibbonPanel/IRibbonPanel.Elements.htm)
		 Свойство Elements возвращает
		 коллекцию элементов управления, расположенных на панели.


		 ![](../../Property_Image.gif)
		 [ImageIndex](../../Interface/IRibbonPanel/IRibbonPanel.ImageIndex.htm)
		 Свойство ImageIndex
		 определяет индекс изображения, которое будет связано с панелью.


		 [![](../../Property_Image.gif)](../../Interface/IRibbonPanel/IRibbonPanel.Keys.htm)
		 [Keys](../../Interface/IRibbonPanel/IRibbonPanel.Keys.htm)
		 Свойство Keys определяет
		 горячую клавишу, которая будет использоваться для быстрого доступа
		 к панели.


		 ![](../../Property_Image.gif)
		 [LaunchButton](../../Interface/IRibbonPanel/IRibbonPanel.LaunchButton.htm)
		 Свойство LaunchButton
		 возвращает параметры кнопки запуска, расположенной на панели.


		 ![](../../Property_Image.gif)
		 [LaunchButtonEnabled](../../Interface/IRibbonPanel/IRibbonPanel.LaunchButtonEnabled.htm)
		 Свойство LaunchButtonEnabled
		 определяет признак отображения кнопки запуска на панели.


		 ![](../../Property_Image.gif)
		 [Text](../../Interface/IRibbonPanel/IRibbonPanel.Text.htm)
		 Свойство Text определяет
		 заголовок панели.


		 ![](../../Property_Image.gif)
		 [Visible](../../Interface/IRibbonPanel/IRibbonPanel.Visible.htm)
		 Свойство Visible определяет
		 отображение панели.
