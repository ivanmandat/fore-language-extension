# ColorDialog


## Описание


Класс ColorDialog реализует
 компонент среды разработки [ColorDialog](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/ColorDialog.htm),
 предоставляющий стандартный диалога выбора цвета из палитры цветов.


## Комментарии


Используется только в настольном приложении.
