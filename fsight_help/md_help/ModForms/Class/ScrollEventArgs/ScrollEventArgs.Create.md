# ScrollEventArgs.Create


## Синтаксис


Create(Code: [ScrollBarCode](../../Enums/ScrollBarCode.htm); Position: Integer);


## Параметры


Code - код действия, в результате которого генерируется событие.


Position - позиция ползунка полосы прокрутки.


## Описание


Конструктор Create создает аргумент событий, связанных с изменением положения ползунка полос прокрутки компонентов.
