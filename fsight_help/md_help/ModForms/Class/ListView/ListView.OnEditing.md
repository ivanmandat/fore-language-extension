# ListView.OnEditing


## Синтаксис


OnEditing(Sender: Object; Args: [IListViewActionEventArgs](../../Interface/IListViewActionEventArgs/IListViewActionEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnEditing наступает
 при переходе в режим редактирования текущего элемента компонента.
