# ListView.OnFocusChange


## Синтаксис


OnFocusChange(Sender: Object; Args: [IListViewEventArgs](../../Interface/IListViewEventArgs/IListViewEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnFocusChange происходит
 при передаче фокуса другому элементу компонента.
