# ListView.OnChecked


## Синтаксис


OnChecked(Sender: Object; Args: [IListViewItemEventArgs](../../Interface/IListViewItemEventArgs/IListViewItemEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnChecked наступает
 после отметки элемента списка.


## Комментарии


Событие происходит, если свойству [CheckBoxes](../../Interface/IListView/IListView.Checkboxes.htm)
 установлено значение True.
