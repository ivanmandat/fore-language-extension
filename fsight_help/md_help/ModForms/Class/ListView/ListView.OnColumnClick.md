# ListView.OnColumnClick


## Синтаксис


OnColumnClick(Sender: Object; Args: [IListViewColumnEventArgs](../../Interface/IListViewColumnEventArgs/IListViewColumnEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnColumnClick наступает
 при щелчке по заголовку одного из столбцов компонента.
