# ListView.OnDeletion


## Синтаксис


OnDeletion(Sender: Object; Args: [IListViewItemEventArgs](../../Interface/IListViewItemEventArgs/IListViewItemEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnDeletion происходит
 при удалении элемента компонента.
