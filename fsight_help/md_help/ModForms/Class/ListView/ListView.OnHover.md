# ListView.OnHover


## Синтаксис


OnHover(Sender: Object; Args: [IListViewItemEventArgs](../../Interface/IListViewItemEventArgs/IListViewItemEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnHover наступает в
 момент прохождения курсора мыши над элементом и отметки подчеркиванием
 элемента компонента.


## Комментарии


Событие происходит, если свойству [TrackSelect](../../Interface/IListView/IListView.TrackSelect.htm)
 установлено значение True.
