# ListView.OnSelectItem


## Синтаксис


OnSelectItem(Sender: Object; Args: [IListViewSelectItemEventArgs](../../Interface/IListViewSelectItemEventArgs/IListViewSelectItemEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnSelectItem наступает
 при изменении отметки у элемента компонента.
