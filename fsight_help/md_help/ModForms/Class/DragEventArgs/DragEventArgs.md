# DragEventArgs


## Описание


Класс DragEventArgs реализует
 аргумент событий, происходящих при перетаскивании объектов с использованием
 механизма Drag&Drop.

		 Краткое описание


		 ![](../../Sub_Image.gif)

		 создает аргумент событий, происходящих при перетаскивании объектов
