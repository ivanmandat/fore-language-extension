# RibbonCategory


Сборка: Forms;


## Описание


Класс RibbonCategory реализует элемент ленты - Вкладка.


## Свойства объекта класса, унаследованные от [IRibbonCategory](../../Interface/IRibbonCategory/IRibbonCategory.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Keys](../../Interface/IRibbonCategory/IRibbonCategory.Keys.htm)
		 Свойство Keys определяет
		 горячую клавишу для быстрого доступа к вкладке на ленте инструментов.


		 ![](../../Property_Image.gif)
		 [LargeImages](../../Interface/IRibbonCategory/IRibbonCategory.LargeImages.htm)
		 Свойство LargeImages
		 определяет компонент, содержащий большие изображения, которые
		 будут использоваться элементами управления данной вкладки на ленте
		 инструментов.


		 ![](../../Property_Image.gif)
		 [Panels](../../Interface/IRibbonCategory/IRibbonCategory.Panels.htm)
		 Свойство Panels возвращает
		 коллекцию панелей, расположенных на текущей вкладке на ленте инструментов.


		 ![](../../Property_Image.gif)
		 [SmallImages](../../Interface/IRibbonCategory/IRibbonCategory.SmallImages.htm)
		 Свойство SmallImages
		 определяет компонент, содержащий маленькие изображения, которые
		 будут использоваться элементами управления данной вкладки на ленте
		 инструментов.


		 ![](../../Property_Image.gif)
		 [Text](../../Interface/IRibbonCategory/IRibbonCategory.Text.htm)
		 Свойство Text определяет
		 заголовок вкладки на ленте инструментов.


		 ![](../../Property_Image.gif)
		 [Visible](../../Interface/IRibbonCategory/IRibbonCategory.Visible.htm)
		 Свойство Visible определяет
		 признак отображения вкладки на ленте инструментов.
