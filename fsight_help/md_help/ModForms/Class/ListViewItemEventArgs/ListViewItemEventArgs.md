# ListViewItemEventArgs


## Описание


Класс ListViewItemEventArgs
 реализует аргумент событий, происходящих при каких-либо действиях над
 элементом компонента [ListView](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ListView.htm).
