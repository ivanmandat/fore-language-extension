# ListViewItemEventArgs.CreateArgs


## Синтаксис


CreateArgs(ListView: [IListView](../../Interface/IListView/IListView.htm); Item: [IListViewItem](../../Interface/IListViewItem/IListViewItem.htm));


## Параметры


ListView - компонент, для которого генерируется событие.


Item - элемент компонента, для которого генерируется событие.


## Описание


Конструктор CreateArgs создает аргумент событий, происходящих при каких-либо действиях над элементом компонента [ListView](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ListView.htm).
