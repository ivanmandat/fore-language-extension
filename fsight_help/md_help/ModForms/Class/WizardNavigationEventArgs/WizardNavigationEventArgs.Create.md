# WizardNavigationEventArgs.Create


## Синтаксис


Create(GotoPage: Integer);


## Параметры


GotoPage - индекс страницы, к которой осуществляется переход.


## Описание


Конструктор Create создает аргумент событий, возникающих при смене страницы в компоненте.
