# WebBrowserBox.OnDocumentComplete


## Синтаксис


OnDocumentComplete(Sender: Object; Args: [IWebBrowserUrlEventArgs](../../Interface/IWebBrowserUrlEventArgs/IWebBrowserUrlEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnDocumentComplete наступает
 после завершения загрузки Web-страницы.
