# WebBrowserBox.OnBeforeNavigate2


## Синтаксис


OnBeforeNavigate2(Sender: Object; Args: [IWebBrowserBeforeNavigateEventArgs](../../Interface/IWebBrowserBeforeNavigateEventArgs/IWebBrowserBeforeNavigateEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnBeforeNavigate2 наступает
 перед тем, как будет совершен переход к какой-либо Web-странице.
