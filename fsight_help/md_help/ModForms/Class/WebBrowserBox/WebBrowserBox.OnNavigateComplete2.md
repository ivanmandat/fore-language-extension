# WebBrowserBox.OnNavigateComplete2


## Синтаксис


OnNavigateComplete2(Sender: Object; Args: [IWebBrowserUrlEventArgs](../../Interface/IWebBrowserUrlEventArgs/IWebBrowserUrlEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnNavigateComplete2
 наступает после завершения перехода к новой Web-странице.
