# NavigationBarPad


Сборка: Forms;


## Описание


Класс NavigationBarPad реализует вкладку компонента [NavigationBar](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/NavigationBar.htm).


## Комментарии


Объекты данного класса используются в коллекции [INavigationBar.Pads](../../Interface/INavigationBar/INavigationBar.Pads.htm).
