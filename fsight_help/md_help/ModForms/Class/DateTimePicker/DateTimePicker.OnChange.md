# DateTimePicker.OnChange


## Синтаксис


OnChange(Sender: Object; Args: [IDateTimePickerEventArgs](../../Interface/IDateTimePickerEventArgs/IDateTimePickerEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnChange возникает при
 изменении даты/времени в компоненте [DateTimePicker](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/DateTimePicker.htm).
