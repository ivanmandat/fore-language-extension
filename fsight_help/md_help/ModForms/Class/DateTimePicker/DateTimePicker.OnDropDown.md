# DateTimePicker.OnDropDown


## Синтаксис


OnDropDown(Sender: Object; Args: [IDateTimePickerEventArgs](../../Interface/IDateTimePickerEventArgs/IDateTimePickerEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnDropDown наступает
 непосредственно после развертывания раскрывающегося календаря.
