# Breadcrumb


## Описание


Класс Breadcrumb реализует компонент
 среды разработки [Breadcrumb](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/Breadcrumb.htm),
 используемый для создания цепочки навигации.
