# Breadcrumb.OnBeforeRemoveItem


## Синтаксис

## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnBeforeRemoveItem наступает
 перед удалением элемента компонента [Breadcrumb](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/Breadcrumb.htm).
