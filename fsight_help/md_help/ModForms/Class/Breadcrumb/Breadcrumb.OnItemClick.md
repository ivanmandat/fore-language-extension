# Breadcrumb.OnItemClick


## Синтаксис


OnItemClick(Sender: Object; Args: [IBreadcrumbMouseEventArgs](../../Interface/IBreadcrumbMouseEventArgs/IBreadcrumbMouseEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnItemClick наступает
 при щелчке кнопкой мыши по элементу компонента [Breadcrumb](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/Breadcrumb.htm).
