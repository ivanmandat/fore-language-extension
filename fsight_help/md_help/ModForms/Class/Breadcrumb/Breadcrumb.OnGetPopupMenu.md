# Breadcrumb.OnGetPopupMenu


## Синтаксис


OnGetPopupMenu(Sender: Object; Args: [IBreadcrumbItemPopupMenuEventArgs](../../Interface/IBreadcrumbItemPopupMenuEventArgs/IBreadcrumbItemPopupMenuEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnGetPopupMenu наступает
 при вызове контекстного меню для элемента компонента.


## Комментарии


Контекстное меню вызывается при щелчке дополнительной кнопкой мыши в
 области элемента, либо при нажатии кнопки элемента, а также при нажатии
 дополнительной кнопки, отображаемой после кнопки удаления элементов.
