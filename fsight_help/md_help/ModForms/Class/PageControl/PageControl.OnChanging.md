# PageControl.OnChanging


## Синтаксис


OnChanging(Sender: [IPageControl](../../Interface/IPageControl/IPageControl.htm);
 Args: [ICancelEventArgs](../../Interface/ICancelEventArgs/ICancelEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnChanging наступает
 в момент перехода с одной вкладки на другую.
