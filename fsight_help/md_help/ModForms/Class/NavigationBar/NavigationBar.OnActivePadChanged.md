# NavigationBar.OnActivePadChanged


## Синтаксис


OnActivePadChanged(Sender: [INavigationBar](../../Interface/INavigationBar/INavigationBar.htm); Args: [INavigationBarPadEventArgs](../../Interface/INavigationBarPadEventArgs/INavigationBarPadEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnActivePadChanged наступает
 после смены активной вкладки навигационной панели.
