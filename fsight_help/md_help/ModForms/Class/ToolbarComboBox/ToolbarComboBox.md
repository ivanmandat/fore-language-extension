# ToolbarComboBox


## Описание


Класс ToolbarComboBox реализует
 комбинированный список, который может располагаться на панели инструментов
 [ToolBar](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ToolBar.htm).


## Свойства, унаследованные от [IToolbarComboBox](../../Interface/IToolbarComboBox/IToolbarComboBox.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [BeginGroup](../../Interface/IToolbarComboBox/IToolbarComboBox.BeginGroup.htm)


		 Свойство BeginGroup
		 позволяет разделить группы компонентов, расположенных на панели
		 инструментов, визуально вертикальной чертой.


		 ![](../../Property_Image.gif)
		 [DropDownCount](../../Interface/IToolbarComboBox/IToolbarComboBox.DropDownCount.htm)


		 Свойство DropDownCount
		 определяет максимальное количество одновременно отображаемых элементов
		 в открытом списке компонента.


		 ![](../../Property_Image.gif)
		 [DroppedDown](../../Interface/IToolbarComboBox/IToolbarComboBox.DroppedDown.htm)


		 Свойство DroppedDown
		 возвращает признак, раскрыт ли комбинированный список.


		 ![](../../Property_Image.gif)
		 [Enabled](../../Interface/IToolbarComboBox/IToolbarComboBox.Enabled.htm)


		 Свойство Enabled определяет
		 доступность компонента для пользователя.


		 ![](../../Property_Image.gif)
		 [Hint](../../Interface/IToolbarComboBox/IToolbarComboBox.Hint.htm)


		 Свойство Hint определяет
		 текст всплывающей подсказки, появляющийся при наведении указателем
		 мыши на список в панели инструментов, во время выполнения формы.


		 ![](../../Property_Image.gif)
		 [ItemCount](../../Interface/IToolbarComboBox/IToolbarComboBox.ItemCount.htm)


		 Свойство ItemCount
		 возвращает количество элементов в комбинированном списке.


		 ![](../../Property_Image.gif)
		 [ItemHeight](../../Interface/IToolbarComboBox/IToolbarComboBox.ItemHeight.htm)


		 Свойство ItemHeight
		 возвращает высоту элементов в комбинированном списке.


		 ![](../../Property_Image.gif)
		 [ItemIndex](../../Interface/IToolbarComboBox/IToolbarComboBox.ItemIndex.htm)


		 Свойство ItemIndex
		 определяет индекс элемента в списке, который выбран и отображается
		 в строке редактирования списка.


		 ![](../../Property_Image.gif)
		 [Items](../../Interface/IToolbarComboBox/IToolbarComboBox.Items.htm)


		 Свойство Items определяет
		 коллекцию элементов комбинированного списка.


		 ![](../../Property_Image.gif)
		 [MaxLength](../../Interface/IToolbarComboBox/IToolbarComboBox.MaxLength.htm)


		 Свойство MaxLength
		 определяет максимальную длину, вводимого в компонент пользователем,
		 текста.


		 ![](../../Property_Image.gif)
		 [SelLength](../../Interface/IToolbarComboBox/IToolbarComboBox.SelLength.htm)


		 Свойство SelLength
		 определяет количество выделенных символов в строке комбинированного
		 списка.


		 ![](../../Property_Image.gif)
		 [SelStart](../../Interface/IToolbarComboBox/IToolbarComboBox.SelStart.htm)


		 Свойство SelStart определяет
		 позицию первого выделенного символа в тексте комбинированного
		 списка.


		 ![](../../Property_Image.gif)
		 [SelText](../../Interface/IToolbarComboBox/IToolbarComboBox.SelText.htm)


		 Свойство SelText определяет
		 выделенный в компоненте текст.


		 ![](../../Property_Image.gif)
		 [Sorted](../../Interface/IToolbarComboBox/IToolbarComboBox.Sorted.htm)


		 Свойство Sorted определяет,
		 будут ли элементы комбинированного списка автоматически сортироваться
		 в алфавитном порядке.


		 ![](../../Property_Image.gif)
		 [Style](../../Interface/IToolbarComboBox/IToolbarComboBox.Style.htm)


		 Свойство Style определяет
		 стиль работы комбинированного списка.


		 ![](../../Property_Image.gif)
		 [Text](../../Interface/IToolbarComboBox/IToolbarComboBox.Text.htm)


		 Свойство Text определяет
		 строку, которая отображается в окне редактирования до того момента,
		 когда произойдет выбор элемента из раскрывающегося списка.


		 ![](../../Property_Image.gif)
		 [Toolbar](../../Interface/IToolbarComboBox/IToolbarComboBox.Toolbar.htm)


		 Свойство Toolbar возвращает
		 родительский компонент [ToolBar](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ToolBar.htm).


		 ![](../../Property_Image.gif)
		 [Visible](../../Interface/IToolbarComboBox/IToolbarComboBox.Visible.htm)


		 Свойство Visible определяет
		 видимость комбинированного списка во время выполнения формы.


		 ![](../../Property_Image.gif)
		 [Width](../../Interface/IToolbarComboBox/IToolbarComboBox.Width.htm)


		 Свойство Width определяет
		 ширину компонента в пикселях.

## События


		 Имя события
		 Краткое описание


		 ![](../../Event_Image.gif)
		 [OnChange](ToolbarComboBox.OnChange.htm)
		 Событие OnChange наступает
		 при изменении текста в строке редактирования в результате прямого
		 редактирования текста или в результате выбора из списка.


		 ![](../../Event_Image.gif)
		 [OnDropDown](ToolbarComboBox.OnDropDown.htm)
		 Событие OnDropDown
		 происходит в момент развертывания комбинированного списка.


		 ![](../../Event_Image.gif)
		 [OnEnter](ToolbarComboBox.OnEnter.htm)
		 Событие OnEnter наступает
		 в момент получения фокуса компонентом.


		 ![](../../Event_Image.gif)
		 [OnExit](ToolbarComboBox.OnExit.htm)
		 Событие OnExit наступает
		 в момент потери фокуса компонентом.


		 ![](../../Event_Image.gif)
		 [OnItemSelect](ToolbarComboBox.OnItemSelect.htm)
		 Событие OnItemSelect
		 наступает во время выбора элемента из списка.


		 ![](../../Event_Image.gif)
		 [OnKeyPress](ToolbarComboBox.OnKeyPress.htm)
		 Событие OnKeyPress
		 наступает, если компонент находится в фокусе, при нажатии пользователем
		 символьной клавиши.
