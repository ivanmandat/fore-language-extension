# ToolbarComboBox.OnChange


## Синтаксис


OnChange(Sender: Object; Args: [IEventArgs](ForeSys.chm::/Interface/IEventArgs/IEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnChange наступает при
 изменении текста в строке редактирования в результате прямого редактирования
 текста или в результате выбора из списка.
