# ToolbarComboBox.OnDropDown


## Синтаксис


OnDropDown(Sender: Object; Args: [IEventArgs](ForeSys.chm::/Interface/IEventArgs/IEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnDropDown происходит
 в момент развертывания раскрывающегося списка.
