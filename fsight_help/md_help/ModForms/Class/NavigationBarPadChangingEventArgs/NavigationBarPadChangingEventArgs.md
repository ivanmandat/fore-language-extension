# NavigationBarPadChangingEventArgs


Сборка: Forms;


## Описание


Класс NavigationBarPadChangingEventArgs
 реализует аргумент события [OnActivePadChanging](../NavigationBar/NavigationBar.OnActivePadChanging.htm),
 наступающего перед активацией вкладки компонента [NavigationBar](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/NavigationBar.htm).
