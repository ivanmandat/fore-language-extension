# CommonCalendarEventArgs.Create


## Синтаксис


Create(Calendar: [ICommonCalendar](../../Interface/ICommonCalendar/ICommonCalendar.htm));


## Параметры


Calendar - календарь, для которого необходимо сгенерировать событие


## Описание


Конструктор Create создает аргумент события [OnChange](../../Interface/ICommonCalendar/ICommonCalendar.OnChange.htm), возникающего при изменении даты в календаре.
