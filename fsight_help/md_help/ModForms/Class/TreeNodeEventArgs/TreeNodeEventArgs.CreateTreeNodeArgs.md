# TreeNodeEventArgs.CreateTreeNodeArgs


## Синтаксис


CreateTreeNodeArgs(TreeControl: [ITreeControl](../../Interface/ITreeControl/ITreeControl.htm); Node: [ITreeListNode](../../Interface/ITreeListNode/ITreeListNode.htm));


## Параметры


TreeControl - компонент, для которого генерируется событие.


Node - вершина, для которой генерируется событие.


## Описание


Конструктор CreateTreeNodeArgs создает аргумент событий, происходящих для какой-либо вершины дерева компонентов [TreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeCombo.htm) и [TreeList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeList.htm).
