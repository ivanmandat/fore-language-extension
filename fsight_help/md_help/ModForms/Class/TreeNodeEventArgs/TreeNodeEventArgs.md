# TreeNodeEventArgs


## Описание


Класс TreeNodeEventArgs реализует
 аргумент событий, происходящих для какой-либо вершины дерева компонентов
 [TreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeCombo.htm)
 и [TreeList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeList.htm).
