# RibbonPaletteButtonGroup


Сборка: Forms;


## Описание


Класс RibbonPaletteButtonGroup
 реализует группу графических кнопок для [комбинированных
 меню](../../Interface/IRibbonPaletteButton/IRibbonPaletteButton.htm) ленты.
