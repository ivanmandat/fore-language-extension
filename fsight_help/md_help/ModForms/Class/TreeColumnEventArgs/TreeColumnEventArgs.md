# TreeColumnEventArgs


## Описание


Класс TreeColumnEventArgs реализует
 аргумент события, происходящего по щелчку мыши на заголовок
 столбца компонентов [TreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeCombo.htm)
 или [TreeList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeList.htm).
