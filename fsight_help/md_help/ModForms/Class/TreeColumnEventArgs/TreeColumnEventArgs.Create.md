# TreeColumnEventArgs.Create


## Синтаксис


Create(TreeControl: [ITreeControl](../../Interface/ITreeControl/ITreeControl.htm); Column: [ITreeListColumn](../../Interface/ITreeListColumn/ITreeListColumn.htm));


## Параметры


TreeControl - компонент для которого генерируется событие.


Column - колонка компонента, для которой генерируется событие.


## Описание


Конструктор Create создает аргумент события, происходящего по щелчку на заголовок столбца компонентов [TreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeCombo.htm) или [TreeList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeList.htm).
