# WebBrowserUrlEventArgs.CreateArgs


## Синтаксис


CreateArgs(Url: String);


## Параметры


Url - Url-адрес страницы, к которой осуществляется переход.


## Описание


Конструктор CreateArgs создает аргумент событий компонента [WebBrowserBox](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/WebBrowserBox.htm), возникающих при переходе по Web-страницам.
