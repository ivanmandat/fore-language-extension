# TreeNodeCheckedEventArgs.CreateTreeNodeCheckedEventArgs


## Синтаксис


CreateTreeNodeCheckedEventArgs(TreeControl: [ITreeControl](../../Interface/ITreeControl/ITreeControl.htm);
 Node: [ITreeListNode](../../Interface/ITreeListNode/ITreeListNode.htm);
 Value: Boolean);


## Параметры


TreeControl. Компонент, для
 которого генерируется событие.


Node. Вершина, для которой
 генерируется событие.


Value. Значение отметки флажка
 элемента.


## Описание


Конструктор CreateTreeNodeCheckedEventArgs
 создает аргумент события, возникающего после изменение отметки флажка
 элемента.
