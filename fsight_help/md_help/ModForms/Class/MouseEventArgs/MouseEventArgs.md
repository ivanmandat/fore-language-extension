# MouseEventArgs


## Описание


Класс MouseEventArgs реализует
 аргумент событий, возникающих во время выполнения действий мышью.


## Конструкторы


		 Имя конструктора
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CreateArgs](MouseEventArgs.CreateArgs.htm)


		 Конструктор CreateArgs
		 создает аргумент событий, возникающих во время выполнения действий
		 мышью.


## Свойства, унаследованные от [IMouseEventArgs](../../Interface/IMouseEventArgs/IMouseEventArgs.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Button](../../Interface/IMouseEventArgs/IMouseEventArgs.Button.htm)
		 Свойство Button возвращает
		 код кнопки мыши, которой было осуществлено какое-либо действие.


		 ![](../../Property_Image.gif)
		 [Cancel](../../Interface/IMouseEventArgs/IMouseEventArgs.Cancel.htm)
		 Свойство Cancel определяет
		 признак отмены дальнейшего действия, связанного с мышью.


		 ![](../../Property_Image.gif)
		 [pPoint](../../Interface/IMouseEventArgs/IMouseEventArgs.pPoint.htm)
		 Свойство pPoint возвращает
		 координаты курсора мыши в клиентской части компонента во время
		 выполнения действий мышью.


		 ![](../../Property_Image.gif)
		 [ShiftState](../../Interface/IMouseEventArgs/IMouseEventArgs.ShiftState.htm)
		 Свойство ShiftState
		 возвращает значение, соответствующее состоянию вспомогательных
		 клавиш (SHIFT, ALT и CTRL) и кнопок мыши во время выполнения действий мышью.
