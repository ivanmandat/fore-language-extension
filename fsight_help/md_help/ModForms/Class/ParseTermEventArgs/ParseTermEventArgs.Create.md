# ParseTermEventArgs.Create


## Синтаксис


Create(Result: [ITerm](../../Interface/ITerm/ITerm.htm); Text: String);


## Параметры


Result - результирующий терм.


Text - текст, преобразуемый в терм.


## Описание


Конструктор Create создает аргумент события, возникающего при преобразовании текста в терм.
