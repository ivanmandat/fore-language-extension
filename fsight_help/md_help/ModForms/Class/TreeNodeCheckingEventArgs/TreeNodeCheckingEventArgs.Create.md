# TreeNodeCheckingEventArgs.Create


## Синтаксис


Create(TreeControl: [ITreeControl](../../Interface/ITreeControl/ITreeControl.htm);
 Node: [ITreeListNode](../../Interface/ITreeListNode/ITreeListNode.htm);
 Value: Boolean; Allow: Boolean);


## Параметры


TreeControl. Компонент, для
 которого генерируется событие.


Node. Вершина, для которой
 генерируется событие.


Value. Значение отметки флажка,
 которое будет установлено.


Allow. Признак изменения отметки
 флажка.


## Описание


Конструктор Create создает аргумент
 события, возникающего перед изменение отметки флажка элемента.
