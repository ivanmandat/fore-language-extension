# TreeCombo.OnRollUp


## Синтаксис


OnRollUp(Sender: Object; Args: [ITreeComboRollUpEventArgs](../../Interface/ITreeComboRollUpEventArgs/ITreeComboRollUpEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnRollUp наступает при
 сворачивании списка компонента [TreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeCombo.htm).
