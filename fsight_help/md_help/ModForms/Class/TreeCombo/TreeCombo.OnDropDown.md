# TreeCombo.OnDropDown


## Синтаксис


OnDropDown(Sender: Object; Args: [ITreeComboEventArgs](../../Interface/ITreeComboEventArgs/ITreeComboEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnDropDown наступает
 при разворачивании списка компонента [TreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeCombo.htm).
