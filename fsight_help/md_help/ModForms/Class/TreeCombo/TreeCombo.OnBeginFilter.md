# TreeCombo.OnBeginFilter


## Синтаксис


OnBeginFilter(Sender: Object; Args: [ITreeSearchEventArgs](../../Interface/ITreeSearchEventArgs/ITreeSearchEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnBeginFilter наступает
 в момент начала фильтрации списка элементов компонента.


## Комментарии


Если компонент находится в фокусе но его раскрывающийся список закрыт,
 то при наборе сочетания символов осуществляется поиск и фильтрация списка
 элементов. В отфильтрованный список попадают только те элементы, которые
 в своем тексте содержат набранное сочетание символов, а также их родительские
 элементы. После фильтрации полученный список будет отображен для возможности
 выбора элементов.


Событие OnBeginFilter наступает
 после окончания набора символов в момент начала фильтрации. Используя
 свойство [SearchText](../../Interface/ITreeSearchEventArgs/ITreeSearchEventArgs.SearchText.htm)
 аргумента события можно получить текст, по которому фильтруются элементы.
