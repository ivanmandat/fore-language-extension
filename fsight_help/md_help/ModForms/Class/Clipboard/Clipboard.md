# Clipboard


## Описание


Класс Clipboard реализует объект,
 позволяющий работать с буфером обмена Windows.


## Свойства, унаследованные от [IClipboard](../../Interface/IClipboard/IClipboard.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Text](../../Interface/IClipboard/IClipboard.Text.htm)


		 Свойство Text определяет
		 текст, содержащийся в буфере обмена.

		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [HasText](../../Interface/IClipboard/IClipboard.HasText.htm)


		 Метод HasText возвращает
		 признак наличия в буфере обмена текстовой информации.
