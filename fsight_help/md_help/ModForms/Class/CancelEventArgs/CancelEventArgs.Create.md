# CancelEventArgs.Create


## Синтаксис


Create(Allowed: Boolean);


## Параметры


Allowed. Признак выполнения
 действия.


## Описание


Конструктор Create создает аргумент
 события, в котором присутствует возможность отменить какое-либо действие.


## Комментарии


Допустимые значения параметра:


	- True. Действие выполняется;


	- False. Действие будет
	 отменено.
