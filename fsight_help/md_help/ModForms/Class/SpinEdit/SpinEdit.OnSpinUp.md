# SpinEdit.OnSpinUp


## Синтаксис


OnSpinUp(Sender: [ISpinEdit](../../Interface/ISpinEdit/ISpinEdit.htm);
 Args: [IValueChangedEventArgs](../../Interface/IValueChangedEventArgs/IValueChangedEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавшую событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnSpinUp происходит
 непосредственно при нажатии кнопки SpinUp в поле редактирования.
