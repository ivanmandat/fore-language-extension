# SpinEdit.OnSpinDown


## Синтаксис


OnSpinDown(Sender: [ISpinEdit](../../Interface/ISpinEdit/ISpinEdit.htm);
 Args: [IValueChangedEventArgs](../../Interface/IValueChangedEventArgs/IValueChangedEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnSpinDown происходит
 непосредственно при нажатии кнопки SpinDown в поле редактирования.
