# OleDocumentBoxEventArgs


## Описание


Класс OleDocumentBoxEventArgs
 реализует аргумент, описывающий событие, которое необходимо сгенерировать
 для загруженного в [OleDocumentBox](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/OleDocumentBox.htm)
 объекта.
