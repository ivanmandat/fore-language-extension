# TreeControlCompareEventArgs.Create


## Синтаксис


Create(ListView: [ITreeList](../../Interface/ITreeList/ITreeList.htm); Column: [ITreeListColumn](../../Interface/ITreeListColumn/ITreeListColumn.htm); LeftItem: [ITreeListNode](../../Interface/ITreeListNode/ITreeListNode.htm); RightItem: [ITreeListNode](../../Interface/ITreeListNode/ITreeListNode.htm));


## Параметры


ListView - компонент, для которого генерируется событие.


Column - колонка компонента, для которой генерируется событие.


LeftItem - первый сравниваемый элемент.


RightItem - второй сравниваемый элемент.


## Описание


Конструктор Create создает аргумент события, наступающего при пользовательской сортировке элементов компонентов [TreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeCombo.htm) или [TreeList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeList.htm).
