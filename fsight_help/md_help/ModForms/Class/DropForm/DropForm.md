# DropForm


## Описание


Класс DropForm реализует компонент
 среды разработки [DropForm](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/DropForm.htm),
 используемый для создания раскрывающихся форм.


## Свойства, унаследованные от [IDropForm](../../Interface/IDropForm/IDropForm.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AnimationType](../../Interface/IDropForm/IDropForm.AnimationType.htm)
		 Свойство AnimationType
		 определяет тип анимации, применяемый при разворачивании формы.


		 ![](../../Property_Image.gif)
		 [DropHeight](../../Interface/IDropForm/IDropForm.DropHeight.htm)
		 Свойство DropHeight
		 определяет высоту области, в которой будет отображаться раскрывающаяся
		 форма.


		 ![](../../Property_Image.gif)
		 [DropWidth](../../Interface/IDropForm/IDropForm.DropWidth.htm)
		 Свойство DropWidth
		 определяет ширину области, в которой будет отображаться раскрывающаяся
		 форма.


		 ![](../../Property_Image.gif)
		 [Form](../../Interface/IDropForm/IDropForm.Form.htm)
		 Свойство Form возвращает
		 параметры раскрывающейся формы.


		 ![](../../Property_Image.gif)
		 [MetabaseObject](../../Interface/IDropForm/IDropForm.MetabaseObject.htm)
		 Свойство MetabaseObject
		 определяет идентификатор формы репозитория, которая будет использоваться
		 в качестве раскрывающейся формы.


		 ![](../../Property_Image.gif)
		 [Parent](../../Interface/IDropForm/IDropForm.Parent.htm)
		 Свойство Parent определяет
		 родительскую форму для компонента.


		 ![](../../Property_Image.gif)
		 [SizeGrip](../../Interface/IDropForm/IDropForm.SizeGrip.htm)
		 Свойство SizeGrip
		 определяет, будет ли в компоненте отображаться кнопка изменения
		 размеров окна.
