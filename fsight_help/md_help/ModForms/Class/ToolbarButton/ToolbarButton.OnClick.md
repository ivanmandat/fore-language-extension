# ToolbarButton.OnClick


## Синтаксис


OnClick(Sender: Object; Args: [IEventArgs](ForeSys.chm::/Interface/IEventArgs/IEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnClick наступает, если
 пользователь нажал и отпустил кнопку мыши на кнопке, расположенной на
 панели инструментов.
