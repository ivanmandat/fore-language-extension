# Ribbon


Сборка: Forms;


## Описание


Класс Ribbon реализует компонент
 среды разработки [Ribbon](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/Ribbon.htm),
 предназначенный для создания модульной ленты.


## Комментарии


Модульная лента представляет собой современный интерфейс, объединяющий
 в себе главное меню и панели инструментов формы.
