# CommandEventArgs


## Описание


Класс CommandEventArgs реализует
 аргумент события [OnCommand](../Form/Form.OnCommand.htm), возникающего
 во время отправки форме какой-либо команды.


## Конструкторы


		 Имя конструктора
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Create](CommandEventArgs.Create.htm)


		 Конструктор Create
		 создает аргумент события, возникающего во время отправления команды
		 форме.
