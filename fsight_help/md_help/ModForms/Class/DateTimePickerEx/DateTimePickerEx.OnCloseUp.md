# DateTimePickerEx.OnCloseUp


## Синтаксис


OnCloseUp(Sender: Object; Args: [IDateTimePickerExEventArgs](../../Interface/IDateTimePickerExEventArgs/IDateTimePickerExEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnCloseUp наступает
 после сворачивания раскрывающегося календаря.
