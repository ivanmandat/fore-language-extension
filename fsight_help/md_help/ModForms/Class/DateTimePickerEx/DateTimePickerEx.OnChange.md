# DateTimePickerEx.OnChange


## Синтаксис


OnChange(Sender: Object; Args: [IDateTimePickerExEventArgs](../../Interface/IDateTimePickerExEventArgs/IDateTimePickerExEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnChange возникает при
 изменении даты/времени в компоненте [DateTimePickerEx](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/DateTimePickerEx.htm).
