# SplitterEventArgs


## Описание


Класс SplitterEventArgs реализует
 аргумент события, возникающего при изменении размеров компонента, привязанному
 к компоненту [Splitter](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/Splitter.htm).
