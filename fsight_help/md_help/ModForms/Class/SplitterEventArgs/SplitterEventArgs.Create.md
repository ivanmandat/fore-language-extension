# SplitterEventArgs.Create


## Синтаксис


Create(NewSize: Integer; Resize: Boolean);


## Параметры


NewSize - новый размер компонента.


Resize - признак изменения размеров компонента.


## Описание


Конструктор Create создает аргумент события, возникающего при изменении размеров компонента, привязанному к компоненту [Splitter](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/Splitter.htm).
