# ActionList.OnUpdate


## Синтаксис


OnUpdate(Sender: [IActionItem](../../Interface/IActionItem/IActionItem.htm);
 Args: [IEventArgs](ForeSys.chm::/Interface/IEventArgs/IEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 действие компонента [ActionList](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/ActionList.htm),
 которое обновилось.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnUpdate наступает при
 обновлении действий компонента [ActionList](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/ActionList.htm).
