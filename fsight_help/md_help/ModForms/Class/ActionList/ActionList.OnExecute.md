# ActionList.OnExecute


## Синтаксис


OnExecute(Sender: [IActionItem](../../Interface/IActionItem/IActionItem.htm);
 Args: [IEventArgs](ForeSys.chm::/Interface/IEventArgs/IEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 действие компонента [ActionList](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/ActionList.htm),
 которое произошло.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnExecute наступает
 при выполнении какого-либо действия компонента [ActionList](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/ActionList.htm).
