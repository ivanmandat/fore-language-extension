# CloseQueryEventArgs


Сборка: Forms;


## Описание


Класс CloseQueryEventArgs реализует
 аргумент события [OnCloseQuery](../Form/Form.OnCloseQuery.htm),
 наступающего в тот момент, когда форма получает команду на закрытие.
