# CloseQueryEventArgs.Create


## Синтаксис


Create(Allowed: Boolean);


## Параметры


Allowed. Параметр, определяющий признак закрытия формы. Если передается значение True, то форма будет закрыта.


## Описание


Конструктор Create создает аргумент события [OnCloseQuery](../Form/Form.OnCloseQuery.htm), с указанием признака закрытия.
