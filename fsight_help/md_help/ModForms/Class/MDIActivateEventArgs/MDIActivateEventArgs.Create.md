# MDIActivateEventArgs.Create


## Синтаксис


Create(Activated: [IFormControl](../../Interface/IFormControl/IFormControl.htm);
 Deactivated: [IFormControl](../../Interface/IFormControl/IFormControl.htm));


## Параметры


Activated. Дочерняя MDI-форма,
 которая стала активной.


Deactivated. Дочерняя MDI-форма,
 которая была активной.


## Описание


Конструктор Create создает аргумент
 события, наступающего во время смены активной дочерней MDI-формы.
