# Splitter


## Описание


Класс Splitter реализует компонент
 среды разработки [Splitter](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/Splitter.htm),
 предназначенный для разделения формы на области, размеры которых можно
 изменять в процессе работы.


## Свойства, унаследованные от [ISplitter](../../Interface/ISplitter/ISplitter.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AutoSnap](../../Interface/ISplitter/ISplitter.AutoSnap.htm)


		 Свойство AutoSnap определяет,
		 будет ли присутствовать возможность установить размер компонента,
		 к которому привязан текущий компонент [Splitter](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/Splitter.htm),
		 меньше размера указанного в свойстве [MinSize](../../Interface/ISplitter/ISplitter.MinSize.htm).


		 ![](../../Property_Image.gif)
		 [Beveled](../../Interface/ISplitter/ISplitter.Beveled.htm)


		 Свойство Beveled определяет,
		 будет ли отображаться рамка для компонента [Splitter](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/Splitter.htm).


		 ![](../../Property_Image.gif)
		 [MinSize](../../Interface/ISplitter/ISplitter.MinSize.htm)


		 Свойство MinSize определяет
		 минимальное расстояние, которое должно оставаться между компонентом
		 [Splitter](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/Splitter.htm)
		 и краем контейнера, в котором расположены элементы управления.


		 ![](../../Property_Image.gif)
		 [Style](../../Interface/ISplitter/ISplitter.Style.htm)


		 Свойство Style определяет
		 стиль отображения компонента [Splitter](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/Splitter.htm)
		 при изменении размеров связанных компонентов.
