# TreeNodeActionEventArgs


## Описание


Класс TreeNodeActionEventArgs
 реализует аргумент событий, возникающих перед каким-либо действием над
 вершиной дерева компонентов [TreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeCombo.htm)
 и [TreeList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeList.htm).
