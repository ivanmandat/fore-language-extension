# MenuItem.OnClick


## Синтаксис


Name(Sender: Object; Args: [IEventArgs](ForeSys.chm::/Interface/IEventArgs/IEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnClick наступает, если
 пользователь нажал кнопку мыши, когда указатель мыши находился на пункте
 меню.
