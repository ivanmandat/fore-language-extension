# TreeNodeGetImageEventArgs.Create


## Синтаксис


Create(Node: [ITreeListNode](../../Interface/ITreeListNode/ITreeListNode.htm); Result: Integer);


## Параметры


Node - вершина, для которой генерируется событие.


Result - индекс изображения, отображаемого рядом с элементом.


## Описание


Конструктор Create создает аргумент событий, возникающих при получении изображения и изображения состояния элемента дерева компонентов [TreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeCombo.htm) и [TreeList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeList.htm).
