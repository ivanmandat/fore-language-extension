# TreeNodeGetImageEventArgs


## Описание


Класс TreeNodeGetImageEventArgs
 реализует аргумент событий, возникающих при получении изображения и изображения
 состояния элемента дерева компонентов [TreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeCombo.htm)
 и [TreeList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeList.htm).
