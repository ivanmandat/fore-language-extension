# RibbonContext


Сборка: Forms;


## Описание


Класс RibbonContext реализует элемент ленты - Контекстная вкладка.


## Свойства объекта класса, унаследованные от [IRibbonContext](../../Interface/IRibbonContext/IRibbonContext.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Categories](../../Interface/IRibbonContext/IRibbonContext.Categories.htm)


		 Свойство Categories
		 возвращает коллекцию дочерних вкладок, формирующих содержимое
		 контекстной вкладки.


		 ![](../../Property_Image.gif)
		 [Id](../../Interface/IRibbonContext/IRibbonContext.Id.htm)


		 Свойство Id возвращает
		 идентификатор контекстной вкладки.


		 ![](../../Property_Image.gif)
		 [Text](../../Interface/IRibbonContext/IRibbonContext.Text.htm)


		 Свойство Text определяет
		 текст заголовка контекстной вкладки.


		 ![](../../Property_Image.gif)
		 [Visible](../../Interface/IRibbonContext/IRibbonContext.Visible.htm)


		 Свойство Visible определяет
		 признак отображения контекстной вкладки.
