# ListViewCompareEventArgs


## Описание


Класс ListViewCompareEventArgs
 реализует аргумент события, наступающего при сравнении двух элементов,
 во время сортировки в колонке компонента [ListView](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ListView.htm).
