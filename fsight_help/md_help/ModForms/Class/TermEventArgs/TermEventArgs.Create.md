# TermEventArgs.Create


## Синтаксис


Create(Term: [ITerm](../../Interface/ITerm/ITerm.htm));


## Параметры


Term - терм, по гиперссылке которого осуществляется щелчок.


## Описание


Конструктор Create создает аргумент события, происходящего при щелчке по гиперссылке какого-либо терма компонента.
