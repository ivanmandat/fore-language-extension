# TermEventArgs


## Описание


Класс TermEventArgs реализует
 аргумент события, происходящего при щелчке по гиперссылке какого-либо
 терма компонента [TermEdit](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TermEdit.htm)
 и [TermEditEx](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TermEditEx.htm).
