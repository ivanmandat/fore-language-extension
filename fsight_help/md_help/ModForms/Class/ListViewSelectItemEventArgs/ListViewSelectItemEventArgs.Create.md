# ListViewSelectItemEventArgs.Create


## Синтаксис


Create(ListView: [IListView](../../Interface/IListView/IListView.htm); Item: [IListViewItem](../../Interface/IListViewItem/IListViewItem.htm); Selected: Boolean);


## Параметры


ListView. Компонент, для которого генерируется событие.


Item. Элемент компонента, для которого генерируется событие.


Selected. Признак отметки элемента.


## Описание


Конструктор Create создает аргумент события, происходящего при смене отметки у какого-либо элемента компонента.
