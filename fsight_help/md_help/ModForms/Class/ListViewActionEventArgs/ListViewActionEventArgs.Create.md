# ListViewActionEventArgs.Create


## Синтаксис


Create(ListView: [IListView](../../Interface/IListView/IListView.htm); Item: [IListViewItem](../../Interface/IListViewItem/IListViewItem.htm); Allowed: Boolean);


## Параметры


ListView - компонент, для которого генерируется событие.


Item - элемент компонента, над которым происходит какое либо действие.


Allowed - признак разрешения действия.


## Описание


Конструктор Create создает аргумент событий, происходящих перед каким-либо действием над элементами компонента.
