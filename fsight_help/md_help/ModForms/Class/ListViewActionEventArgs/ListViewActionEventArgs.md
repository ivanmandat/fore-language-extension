# ListViewActionEventArgs


## Описание


Класс ListViewActionEventArgs
 реализует аргумент событий, происходящих перед каким-либо действием над
 элементами компонента [ListView](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ListView.htm).
