# FolderBrowserDialog


## Описание


Класс FolderBrowserDialog реализует
 компонент среды разработки [FolderBrowserDialog](UiDevEnv.chm::/02_Components_constructor_forms/01_Standart_Components/FolderBrowserDialog.htm),
 предоставляющий стандартный диалог Windows открытия папки.


## Комментарии


Используется только в настольном приложении.


## Свойства, унаследованные от [IFolderBrowserDialog](../../Interface/IFolderBrowserDialog/IFolderBrowserDialog.htm)


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Description](../../Interface/IFolderBrowserDialog/IFolderBrowserDialog.Description.htm)


		 Свойство Description
		 определяет заголовок диалога, в котором будет выбираться папка.


		 ![](../../Property_Image.gif)
		 [RootFolder](../../Interface/IFolderBrowserDialog/IFolderBrowserDialog.RootFolder.htm)


		 Свойство RootFolder
		 позволяет выбрать стандартную папку, которая будет корневой при
		 открытии стандартного окна Windows открытия папки.


		 ![](../../Property_Image.gif)
		 [SelectedPath](../../Interface/IFolderBrowserDialog/IFolderBrowserDialog.SelectedPath.htm)


		 Свойство SelectedPath
		 указывает пользовательскую папку, которая будет искаться в системной
		 папке и фокусироваться при открытии стандартного окна
		 Windows открытия папки.


		 ![](../../Property_Image.gif)
		 [ShowNewFolderButton](../../Interface/IFolderBrowserDialog/IFolderBrowserDialog.ShowNewFolderButton.htm)


		 Свойство ShowNewFolderButton
		 определяет, будет ли отображаться кнопка создания новой папки
		 «Создать папку».
