# ImageList


## Описание


Класс ImageList реализует компонент
 среды разработки [ImageList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ImageList.htm),
 предназначенный для хранения коллекции пиктограмм.


## Свойства, унаследованные от [IImageList](../../Interface/IImageList/IImageList.htm)


		  Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](../../Interface/IImageList/IImageList.Count.htm)
		 Свойство Count возвращает
		 количество изображений, загруженных в коллекцию компонента.


		 ![](../../Property_Image.gif)
		 [Height](../../Interface/IImageList/IImageList.Height.htm)
		 Свойство Height определяет
		 высоту изображений, хранящихся в коллекции компонента.


		 ![](../../Property_Image.gif)
		 [Icon](../../Interface/IImageList/IImageList.Icon.htm)
		 Свойство Icon возвращает
		 указанное графическое изображение в виде пиктограммы.


		 ![](../../Property_Image.gif)
		 [Item](../../Interface/IImageList/IImageList.Item.htm)
		 Свойство Item возвращает
		 указанное графическое изображение.


		 ![](../../Property_Image.gif)
		 [Width](../../Interface/IImageList/IImageList.Width.htm)
		 Свойство Width определяет
		 ширину изображений, хранящихся в коллекции компонента.

## Методы, унаследованные от [IImageList](../../Interface/IImageList/IImageList.htm)


		  Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](../../Interface/IImageList/IImageList.Add.htm)
		 Метод Add добавляет
		 изображение в коллекцию компонента.


		 ![](../../Sub_Image.gif)
		 [AddIcon](../../Interface/IImageList/IImageList.AddIcon.htm)
		 Метод AddIcon добавляет
		 указанную пиктограмму в коллекцию компонента.


		 ![](../../Sub_Image.gif)
		 [Clear](../../Interface/IImageList/IImageList.Clear.htm)
		 Метод Clear осуществляет
		 очистку коллекции изображений компонента.


		 ![](../../Sub_Image.gif)
		 [LoadFromFile](../../Interface/IImageList/IImageList.LoadFromFile.htm)
		 Метод LoadFromFile
		 осуществляет загрузку изображения в компонент из файла.


		 ![](../../Sub_Image.gif)
		 [LoadFromStream](../../Interface/IImageList/IImageList.LoadFromStream.htm)
		 Метод LoadFromStream
		 осуществляет загрузку изображения в компонент из потока.


		 ![](../../Sub_Image.gif)
		 [LoadFromStream2](../../Interface/IImageList/IImageList.LoadFromStream2.htm)
		 Метод LoadFromStream2
		 осуществляет загрузку изображений из потока с восстановлением
		 информации о полупрозрачности.


		 ![](../../Sub_Image.gif)
		 [Remove](../../Interface/IImageList/IImageList.Remove.htm)
		 Метод Remove удаляет
		 изображение из коллекции.


		 ![](../../Sub_Image.gif)
		 [SaveToDoc](../../Interface/IImageList/IImageList.SaveToDoc.htm)
		 Метод SaveToDoc сохраняет
		 все изображения из компонента в файл.


		 ![](../../Sub_Image.gif)
		 [SaveToStream](../../Interface/IImageList/IImageList.SaveToStream.htm)
		 Метод SaveToStream
		 сохраняет все изображения из компонента в поток.


		 ![](../../Sub_Image.gif)
		 [SaveToStream2](../../Interface/IImageList/IImageList.SaveToStream2.htm)
		 Метод SaveToStream2
		 сохраняет изображения в поток с сохранением информации о полупрозрачности.


		 ![](../../Sub_Image.gif)
		 [SetOverlay](../../Interface/IImageList/IImageList.SetOverlay.htm)
		 Метод SetOverlay идентифицирует
		 изображение в списке как изображение, которое может быть использовано
		 как покрывающая маска.
