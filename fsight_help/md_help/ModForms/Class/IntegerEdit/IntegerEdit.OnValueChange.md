# IntegerEdit.OnValueChange


## Синтаксис


OnValueChange(Sender: Object; Args: [IEventArgs](ForeSys.chm::/Interface/IEventArgs/IEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnValueChange присходит
 при изменении значения в поле редактирования.
