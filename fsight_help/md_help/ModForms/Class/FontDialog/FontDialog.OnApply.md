# FontDialog.OnApply


## Синтаксис


OnApply(Sender: Object; Args: [IEventArgs](ForeSys.chm::/Interface/IEventArgs/IEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnApply наступает, когда
 пользователь нажимает кнопку "Применить" диалога выбора шрифта.
