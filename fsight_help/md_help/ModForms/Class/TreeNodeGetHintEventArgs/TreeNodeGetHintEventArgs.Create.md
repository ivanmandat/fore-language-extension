# TreeNodeGetHintEventArgs.Create


## Синтаксис


Create(TreeControl: [ITreeControl](../../Interface/ITreeControl/ITreeControl.htm); Node: [ITreeListNode](../../Interface/ITreeListNode/ITreeListNode.htm); Column: Integer; Text: String);


## Параметры


TreeControl - компонент, для которого сгенерировано событие.


Node - вершина, для которой сгенерировано событие.


Column - колонка, в области которой расположен курсор мыши.


Text - текст всплывающей подсказки для элемента.


## Описание


Конструктор Create создает аргумент события, возникающего при появлении всплывающей подсказки у элемента, над которым остановился курсор мыши.
