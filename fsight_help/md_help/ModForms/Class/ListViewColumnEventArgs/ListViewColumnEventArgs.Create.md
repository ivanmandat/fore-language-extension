# ListViewColumnEventArgs.Create


## Синтаксис


Create(ListView: [IListView](../../Interface/IListView/IListView.htm); Column: [IListViewColumn](../../Interface/IListViewColumn/IListViewColumn.htm));


## Параметры


ListView. Компонент, для которого генерируется событие.


Column. Колонка, для которой генерируется событие щелчка по заголовку.


## Описание


Конструктор Create создает аргумент события, возникающего при щелчке по заголовку колонки компонента.
