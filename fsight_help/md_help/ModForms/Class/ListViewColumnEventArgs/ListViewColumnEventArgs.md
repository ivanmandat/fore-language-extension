# ListViewColumnEventArgs


## Описание


Класс ListViewColumnEventArgs
 реализует аргумент события, возникающего при нажатии на заголовок колонки компонента [ListView](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ListView.htm).
