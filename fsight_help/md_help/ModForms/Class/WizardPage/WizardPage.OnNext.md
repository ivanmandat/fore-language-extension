# WizardPage.OnNext


## Синтаксис


OnNext(Sender: Object; Args: [IWizardNavigationEventArgs](../../Interface/IWizardNavigationEventArgs/IWizardNavigationEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnNext наступает после
 нажатия на кнопку "Далее".
