# WizardPage.OnSetActive


## Синтаксис


OnSetActive(Sender: Object; Args: [ICancelEventArgs](../../Interface/ICancelEventArgs/ICancelEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnSetActive наступает
 при активации страницы мастера.
