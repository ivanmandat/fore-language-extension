# WizardPage.OnGetLeaveAllowed


## Синтаксис


OnGetLeaveAllowed(Sender: Object; Args: [ICancelEventArgs](../../Interface/ICancelEventArgs/ICancelEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnGetLeaveAllowed наступает
 при попытке перейти на другую страницу мастера.
