# WizardPage.OnBack


## Синтаксис


OnBack(Sender: Object; Args: [IWizardNavigationEventArgs](../../Interface/IWizardNavigationEventArgs/IWizardNavigationEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnBack наступает после
 нажатия на кнопку "Назад".
