# TreeNodeColumnEventArgs


## Описание


Класс TreeNodeColumnEventArgs
 реализует аргумент событий, происходящих при прохождении курсора мыши
 над элементом в рамках какой-либо колонки компонентов [TreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeCombo.htm)
 и [TreeList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeList.htm).
