# TreeNodeColumnEventArgs.CreateTreeNodeColumnArgs


## Синтаксис


CreateTreeNodeColumnArgs(TreeControl: [ITreeControl](../../Interface/ITreeControl/ITreeControl.htm); Node: [ITreeListNode](../../Interface/ITreeListNode/ITreeListNode.htm); Column: Integer);


## Параметры


TreeControl - компонент, для которого генерируется событие.


Node - вершина, для которой генерируется событие.


Column - индекс колонки компонента.


## Описание


Конструктор CreateTreeNodeColumnArgs создает аргумент событий, происходящих при прохождении курсора мыши над элементом в рамках какой-либо колонки компонентов [TreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeCombo.htm) и [TreeList](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeList.htm).
