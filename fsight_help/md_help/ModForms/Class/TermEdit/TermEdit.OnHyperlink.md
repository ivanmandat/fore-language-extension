# TermEdit.OnHyperlink


## Синтаксис


OnHyperlink(Sender: [ITermEdit](../../Interface/ITermEdit/ITermEdit.htm);
 Args: [ITermEventArgs](../../Interface/ITermEventArgs/ITermEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnHyperlink наступает
 при щелчке по термовой гиперссылке.
