# TermEdit.OnParseTerm


## Синтаксис


OnParseTerm(Sender: [ITermEdit](../../Interface/ITermEdit/ITermEdit.htm);
 Args: [IParseTermEventArgs](../../Interface/IParseTermEventArgs/IParseTermEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnParseTerm возникает
 во время преобразования текста редактора в термы(выражения) и знаки операции.
