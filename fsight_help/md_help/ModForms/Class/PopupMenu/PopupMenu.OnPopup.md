# PopupMenu.OnPopup


## Синтаксис


PopupMenuOnPopup(Sender: Object; Args: [IPopupMenuEventArgs](../../Interface/IPopupMenuEventArgs/IPopupMenuEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие;


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnPopup наступает непосредственно
 перед появлением контекстного меню.
