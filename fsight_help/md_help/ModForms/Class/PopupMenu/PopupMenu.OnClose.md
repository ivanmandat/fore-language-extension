# PopupMenu.OnClose


## Синтаксис


OnClose(Sender: Object; Args: [IEventArgs](ForeSys.chm::/Interface/IEventArgs/IEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnClose наступает после
 закрытия контекстного меню.


## Комментарии


Событие наступает вне зависимости, был ли выбран какой-либо пункт меню,
 либо вызов меню был отменен.
