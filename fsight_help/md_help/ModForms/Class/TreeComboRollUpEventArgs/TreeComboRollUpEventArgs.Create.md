# TreeComboRollUpEventArgs.Create


## Синтаксис


Create(Combo: [ITreeCombo](../../Interface/ITreeCombo/ITreeCombo.htm); Cancel: Boolean);


## Параметры


Combo - компонент, для которого генерируется событие.


Cancel - признак закрытия раскрывающегося списка.


## Описание


Конструктор Create создает аргумент события, возникающего при свертывании комбинированного списка компонента [TreeCombo](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/TreeCombo.htm).
