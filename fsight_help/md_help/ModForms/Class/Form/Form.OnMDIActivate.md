# Form.OnMDIActivate


## Синтаксис


OnMDIActivate(Sender: Object; Args: [IMDIActivateEventArgs](../../Interface/IMDIActivateEventArgs/IMDIActivateEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnMDIActivate возникает
 во время смены активной дочерней MDI-формы. Данное событие должно обрабатываться
 у дочерних MDI-форм.
