# Form.OnClose


## Синтаксис


OnClose(Sender: Object; Args: [IEventArgs](ForeSys.chm::/Interface/IEventArgs/IEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnClose наступает непосредственно
 перед закрытием формы, после выполнения события [OnCloseQuery](Form.OnCloseQuery.htm).
 Обычно оно используется для изменения стандартного поведения формы при
 закрытии.
