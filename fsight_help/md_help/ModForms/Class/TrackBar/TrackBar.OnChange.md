# TrackBar.OnChange


## Синтаксис


OnChange(Sender: Object; Args: [ITrackBarEventArgs](../../Interface/ITrackBarEventArgs/ITrackBarEventArgs.htm));


## Параметры


Sender. Параметр, возвращающий
 компонент, сгенерировавший событие.


Args. Параметр, позволяющий
 определить параметры события.


## Описание


Событие OnChange наступает при
 изменении положения ползунка.
