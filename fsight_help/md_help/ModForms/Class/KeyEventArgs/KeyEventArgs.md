# KeyEventArgs


## Описание


Класс KeyEventArgs
 реализует аргумент событий [OnKeyDown](../../Interface/IControl/IControl.OnKeyDown.htm)
 и [OnKeyUp](../../Interface/IControl/IControl.OnKeyUp.htm).
