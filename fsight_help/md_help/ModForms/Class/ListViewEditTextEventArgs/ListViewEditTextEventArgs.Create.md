# ListViewEditTextEventArgs.Create


## Синтаксис


Create(ListView: [IListView](../../Interface/IListView/IListView.htm); Item: [IListViewItem](../../Interface/IListViewItem/IListViewItem.htm); Text: String);


## Параметры


ListView. Компонент, для которого генерируется событие.


Item. Элемент компонента, для которого генерируется событие.


Text. Текст для элемента.


## Описание


Конструктор Create создает аргумент события, возникающего при редактировании элемента, либо при изменении всплывающей подсказки для элементов компонента.
