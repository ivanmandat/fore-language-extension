# ListViewEditTextEventArgs


## Описание


Класс ListViewEditTextEventArgs
 реализует аргумент события, возникающего при редактировании элемента,
 либо при изменении всплывающей подсказки для элементов компонента [ListView](UiDevEnv.chm::/02_Components_constructor_forms/02_Additional_components/ListView.htm).
