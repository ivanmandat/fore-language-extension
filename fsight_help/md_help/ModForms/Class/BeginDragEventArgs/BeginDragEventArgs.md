# BeginDragEventArgs


## Описание


Класс BeginDragEventArgs реализует
 аргумент события [OnBeginDrag](../../Interface/IControl/IControl.OnBeginDrag.htm),
 наступающего при начале перетаскивания какого-либо объекта от компонента.
