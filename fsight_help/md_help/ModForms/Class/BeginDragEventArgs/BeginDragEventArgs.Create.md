# BeginDragEventArgs.Create


## Синтаксис


Create(Point: [IPoint](../../Interface/IPoint/IPoint.htm));


## Параметры


Point - координата в клиентской части компонента во время начала перетаскивания объекта от компонента.


## Описание


Конструктор Create создает аргумент события [OnBeginDrag](../../Interface/IControl/IControl.OnBeginDrag.htm), наступающего при начале перетаскивания какого-либо объекта от компонента.
