# Как настроить автоматическую генерацию мнемоник?


Мнемоника - это уникальное значение, генерируемое на основе значений
 атрибутов временного ряда и используемое в качестве наименования ряда.


Для автоматической генерации мнемоник задайте правила формирования наименований
 временных рядов с помощью языка Fore. Используйте свойство [IMetaDictionary.AttributeMnemoNamings](KeRds.chm::/Interface/IMetaDictionary/IMetaDictionary.AttributeMnemoNamings.htm).


После выполнения данных действий мнемоники рядов будут генерироваться
 автоматически по заданным правилам.


Совет. Для использования
 мнемоник в качестве идентификаторов временных рядов установите флажок
 «Ряды имеют идентификатор» на
 странице «[Атрибуты](UiNavObj.chm::/TimeSeriesDatabase/TS_Attributes.htm)»
 в мастере базы данных временных рядов.
