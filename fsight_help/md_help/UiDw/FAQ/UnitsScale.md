# Как изменяются значения временных рядов при масштабировании данных?


Масштабирование - это приведение данных, хранящихся в БД временных рядов,
 к одному масштабу единиц измерения (нормализация). Масштабирование может
 быть выполнено при импорте и экспорте данных. По умолчанию масштабирование
 не выполняется.


Особенности масштабирования при последовательном экспорте и импорте
 данных:


	- если масштабирование выполняется при экспорте и импорте, то
	 данные временных рядов не нормализуются, т.к. масштабирование
	 выполняется дважды;


	- если масштабирование выполняется только при экспорте, то выгружаемые
	 данные будут изменены. Данные, импортируемые в БД временных рядов,
	 уже являются нормализованными;


	- если масштабирование выполняется только при импорте, то экспортируемые
	 данные остаются без изменений. Нормализация выполняется при загрузке
	 данных в БД временных рядов;


	- если при экспорте и импорте данных масштабирование не выполняется,
	 то данные временных рядов остаются без изменений.


Примечание.
 Масштабирование выполняется независимо от того, являются ли единицы измерения
 обязательным атрибутом.
