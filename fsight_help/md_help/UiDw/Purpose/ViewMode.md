# Работа с готовой рабочей книгой


Для работы с готовой рабочей книгой выполните шаги:


	- Перейдите в [навигатор
	 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm).


	- Выберите рабочую книгу.


	- Выполните команду контекстного меню «Открыть»
	 рабочей книги.


Выбранная рабочая книга будет открыта в режиме просмотра. Для работы
 будет доступна только вкладка «Главная»
 на ленте инструментов.


Примечание.
 Если у пользователя есть [права](Admin.chm::/03_Admin/Admin_AdminObjects_Discretionary.htm)
 на изменение объекта репозитория, то рабочая книга будет открыта в [режиме редактирования](UiWeb_FacCatSeries.htm).


Основные операции, доступные в режиме просмотра:


	- [операции
	 с рабочей книгой](../Workbook/UiDw_Woorkbook.htm):


	-


		- переключение между [представлениями
		 данных](../Workbook/UiDw_Mode.htm). Набор доступных представлений зависит от настроек,
		 выполненных в [режиме редактирования](UiWeb_FacCatSeries.htm);


		- отображение/скрытие [представлений
		 данных](../Workbook/UiDw_Mode.htm);


		- обновление рабочей книги;


		- печать рабочей книги;


		- публикация рабочей книги;


		- переключение между листами рабочей книги;


	- [операции
	 с временными рядами](../Object_Index/UiDw_Factor_New.htm):


	-


		- редактирование данных;


		- сохранение/отмена изменений;


		- просмотр статистических характеристик временного ряда/наблюдений;


	- [операции
	 с таблицей данных](../Workbook/UiDw_Series.htm):


	-


		- сортировка значений;


		- поиск и замена данных;


		- изменение масштаба таблицы;


		- применение расширенной аналитики;


	- [операции
	 с диаграммой](../Workbook/Chart_work.htm):


	-


		- скрытие/отображение ряда;


		- редактирование данных на диаграмме;


	- [операции
	 с картой](../Workbook/Map_Work.htm):


	-


		- фильтрация отображаемых данных;


		- изменение масштаба карты;


		- работа с анимацией.


Выполнение операций в режиме просмотра совпадает с выполнением операций
 в режиме редактирования. Подробное описание операций и способов их выполнения
 приведено в соответствующих разделах.
