# Запуск и режимы работы


Инструмент анализа временны́х рядов может работать в следующих режимах:


	- режим редактирования.
	 Используется по умолчанию. Предназначен для анализа данных. Доступная
	 функциональность:


	-


		- редактирование данных и значений атрибутов временны́х рядов;


		- создание временны́х и вычисляемых рядов;


		- применение различных статистических методов и т.д.


	- режим просмотра. Используется,
	 если у пользователя отсутствуют [права](Admin.chm::/03_Admin/Admin_AdminObjects_Discretionary.htm)
	 на изменение [рабочей книги](../Workbook/UiDw_Woorkbook.htm).
	 Доступны все возможности режима редактирования, кроме возможности
	 [сохранить](../Workbook/UiDw_Woorkbook.htm#save) рабочую
	 книгу, но доступно [сохранение
	 копии](../Workbook/UiDw_Woorkbook.htm#save_as) рабочей книги.


С данными из БД временны́х рядов доступна работа в следующих инструментах:


	- [Аналитические
	 запросы (OLAP)](UiExpress.chm::/purpose/UiExpress_Purpose.htm):


		- в настольном приложении.
		 Выполните команду контекстного меню «Открыть
		 с помощью > Аналитические запросы (OLAP)» для БД временны́х
		 рядов или выполните команду «В
		 аналитических запросах (OLAP)» в раскрывающемся меню кнопки
		 «Открыть», расположенной
		 на вкладке «Главная» ленты
		 инструментов навигатора объектов;


		- в веб-приложении.
		 Выполните команду «Открыть в
		 "Экспресс-анализе"» в контекстном меню БД временны́х
		 рядов или в раскрывающемся меню кнопки «Открыть»,
		 расположенной на вкладке «Главная»
		 ленты инструментов навигатора объектов.


	- [Моделирование и прогнозирование](UiModelling.chm::/1_Modelling/UiModelling_First.htm). Возможность доступна
	 только в настольном приложении. Выполните команду контекстного меню
	 «Открыть с помощью > Контейнера
	 моделирования» для БД временны́х рядов или выполните
	 команду «В контейнере моделирования»
	 в раскрывающемся меню кнопки «Открыть»,
	 расположенной на вкладке «Главная»
	 ленты инструментов навигатора объектов.


## Запуск в настольном приложении


Для запуска инструмента анализа временны́х рядов откройте навигатор
 объектов:


	- выделите объект репозитория «База
	 данных временных рядов» и:


		- дважды щелкните по объекту «База
		 данных временных рядов»;


		- выполните команду «Открыть»
		 в контекстном меню БД временны́х рядов;


		- нажмите кнопку «Открыть»,
		 расположенную на вкладке «Главная»
		 ленты инструментов;


		- выполните команду контекстного меню «Открыть
		 с помощью > Анализ временных рядов» для БД временны́х
		 рядов.


Будет создана и [открыта](UiDw_Purpose_Basic_Window.htm)
 рабочая книга с выбранной базой временны́х рядов в качестве источника
 данных. Инструмент анализа временны́х рядов будет работать в режиме редактирования;


	- выделите объект репозитория «Рабочая
	 книга» и:


		- выполните команду «Открыть»
		 в контекстном меню рабочей книги;


		- нажмите кнопку «Открыть»,
		 расположенную на вкладке «Главная»
		 ленты инструментов.


Будет открыта выбранная рабочая книга. Если
 у пользователя присутствуют [права](Admin.chm::/03_Admin/Admin_AdminObjects_Discretionary.htm)
 на изменение рабочей книги, то инструмент анализа временны́х рядов будет
 работать в режиме редактирования, если [права](Admin.chm::/03_Admin/Admin_AdminObjects_Discretionary.htm)
 на изменение рабочей книги отсутствуют - в режиме просмотра.


## Запуск в веб-приложении


[Запустите
 веб-приложение](GetStarted.chm::/GetStarted/Get_Started.htm),
 выполните авторизацию и:


	- на странице с окном приветствия выберите инструмент «Анализ
	 временны́х рядов»;


	- выполните команду главного меню «Навигатор
	 объектов > Анализ временных рядов»;


	- запустите навигатор объектов, выделите объект репозитория «База данных временных рядов» и:


		- выполните команду контекстного меню «Открыть
		 "Анализ временных рядов"» для БД временны́х рядов;


		- нажмите кнопку «Открыть»,
		 расположенную на вкладке «Главная»
		 ленты инструментов.


Будет создана и [открыта](UiDw_Purpose_Basic_Window.htm)
 рабочая книга с выбранной базой временны́х рядов в качестве источника
 данных.


Инструмент анализа временны́х рядов будет
 работать в режиме редактирования;


	- запустите навигатор объектов, выделите объект репозитория «Рабочая книга» и выполните команду
	 контекстного меню «Открыть».

	Будет открыта выбранная рабочая книга. Если у пользователя присутствуют
	 [права](Admin.chm::/03_Admin/Admin_AdminObjects_Discretionary.htm)
	 на изменение рабочей книги, то инструмент анализа временны́х рядов
	 будет работать в режиме редактирования, если [права](Admin.chm::/03_Admin/Admin_AdminObjects_Discretionary.htm)
	 на изменение рабочей книги отсутствуют - в режиме просмотра.
