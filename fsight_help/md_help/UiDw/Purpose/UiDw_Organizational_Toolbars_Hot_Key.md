# Горячие клавиши


		 Сочетание клавиш
		 Краткое описание


		 CTRL+X
		 Вырезает выбранные объекты в буфер обмена.


		 CTRL+C
		 Копирует выбранные объекты в буфер обмена.


		 CTRL+V
		 Вставляет объекты из буфера обмена.


		 CTRL+F
		 Вызывает окно поиска и замены данных.


		 F5
		 Обновляет рабочую книгу.


		 CTRL+W
		 Закрывает рабочую книгу.


		 CTRL+P
		 Вызывает окно печати.


		 CTRL+A

CTRL+SHIFT+ПРОБЕЛ
		 Выделяет всю таблицу. Первое нажатие приводит к выделению диапазона
		 данных около активной ячейки/диапазона, повторное нажатие приводит
		 к выделению всех ячеек таблицы.


		 CTRL+ПРОБЕЛ
		 Выделить столбец листа.


		 SHIFT+ПРОБЕЛ
		 Выделить строку листа.
