# Анализ временных рядов


Для анализа временных рядов предназначен инструмент «Анализ
 временных рядов»:


![](purpose/timeseries.png)


Инструмент «Анализ временных рядов»
 для исследования данных, изменяющихся во времени.


Ключевые возможности:


	- размещение временных рядов на едином табличном пространстве
	 с возможностью объединения и графического представления;


	- проверка качества данных для моделирования: выявление выбросов,
	 пропусков и т.п.;


	- группировка, сортировка и фильтрация временных рядов, условное
	 форматирование, расчет статистических характеристик;


	- построение новых временных рядов с помощью разнообразных моделей:
	 математические формулы, экстраполяция, факторный анализ и т.д.;


	- сохранение настроенных расчётов в виде моделей.
