# Настройка параметров метода расчёта


Для настройки параметров метода расчёта вычисляемого ряда используйте
 группу вкладок «Ряд» на боковой
 панели.


[![](../../Opened.gif)![](../../Closed.gif)Для отображения
 группы вкладок](javascript:TextPopup(this))


		- Убедитесь, что боковая панель отображается.


		- Выделите вычисляемый ряд в таблице данных.


		- Установите переключатель «Ряд»
		 на боковой панели.


	Совет. Для быстрого
	 отображения группы вкладок «Ряд»
	 на боковой панели выполните команду «Показать
	 параметры» в контекстном меню выбранного ряда.


Набор вкладок зависит от выбранного метода расчёта. Для работы с большинством
 методов используйте вкладки на боковой панели:


	- Параметры. Предназначена
	 для настройки основных параметров расчёта. Вид вкладки зависит от
	 используемого метода расчёта;


	- [Периоды
	 расчёта](Forecast/UiDw_cs_calc_param.htm). Предназначена для редактирования периода расчёта вычисляемого
	 ряда;


	- [Обработка
	 пропусков](UiDw_cs_MisingData_Method.htm). Предназначена для выбора метода обработки пропусков,
	 применяемого для расчёта отсутствующих значений в переменных.

[![](../../Opened.gif)![](../../Closed.gif)Агрегация](javascript:TextPopup(this))


			 Функция


			 Краткое описание


			 [Агрегация
			 по показателям](Aggregation/UiDw_cs_Aggregation.htm)


			 Агрегирует данные по значениям атрибутов временных рядов.


			 [Агрегация
			 по времени](Aggregation/UiDw_cs_TotalCollapseFrequency.htm)


			 Агрегирует данные по уровням календарной динамики временных
			 рядов.


			 Агрегация по времени (минимум).
			 Агрегирует данные с нижнего уровня на верхний путем нахождения
			 минимального значения элементов динамики.


			 Агрегация по времени (максимум).
			 Агрегирует данные с нижнего уровня на верхний путем
			 нахождения максимального значения элементов динамики.


			 Агрегация по времени (первый
			 элемент). Агрегирует данные с нижнего уровня на верхний
			 путем нахождения первого имеющегося значения элементов динамики.


			 Агрегация по времени (последний
			 элемент). Агрегирует данные с нижнего уровня на верхний
			 путем нахождения последнего имеющегося значения элементов
			 динамики.


			 Агрегация по времени (среднее).
			 Агрегирует данные с нижнего уровня на верхний путем нахождения
			 среднего значения элементов динамики.


			 Агрегация по времени (стандартное
			 отклонение). Агрегирует данные с нижнего уровня на
			 верхний путем нахождения стандартного отклонения по динамике.


			 Агрегация по времени (сумма).
			 Агрегирует данные с нижнего уровня на верхний путем нахождения
			 суммы значений элементов динамики.


			 [Интерполяция
			 по времени](Aggregation/UiDw_cs_ProportionalInterpolation.htm)


			 Интерполяция по времени
			 (повторная). Дезагрегирует данные с верхнего уровня
			 на нижний с применением повторной интерполяции.


			 Интерполяция по времени
			 (пропорциональная). Дезагрегирует данные с верхнего
			 уровня на нижний с применением пропорциональной интерполяции.


	Примечание.
	 Метод «Агрегация по показателям»
	 доступен только в настольном приложении.

	[Методы
	 заполнения пропусков](Transformations/UiDw_cs_MissingData_Geometric.htm):


			 Функция
			 Краткое описание


			 Геометрическая интерполяция
			 Для вычисления пустых значений ряда используется геометрическая
			 интерполяция.


			 Интерполяция кубическим сплайном
			 Для вычисления пустых значений ряда используется интерполяция
			 кубическими сплайнами.


			 Линейная интерполяция
			 Для вычисления пустых значений ряда используется линейная
			 интерполяция по двум соседним точкам.


			 Линейный тренд
			 Для вычисления пустых значений ряда используется линейный
			 тренд.


			 Предыдущее значение
			 Пустые значения ряда заменяются предыдущим непустым значением.


			 Следующее значение
			 Пустые значения ряда заменяются следующим непустым значением.


			 Среднее значение по выборке
			 Пустые значения ряда вычисляются как среднее по выборке.


			 Среднее по N
			 соседним точкам
			 Пустые значения ряда вычисляются как среднее по N
			 соседним точкам.


			 Темп роста к предыдущему периоду
			 Пустые значения ряда вычисляются исходя из темпа роста
			 существующих значений к предыдущему периоду.


			 Темп роста к следующему периоду
			 Пустые значения ряда вычисляются исходя из темпа роста
			 существующих значений к следующему периоду.


			 Указанное значение
			 Пустые значения ряда заменяются указанным числом.


	Примечание.
	 Методы «Линейный тренд»,
	 «Среднее значение по выборке»,
	 «Среднее по N соседним точкам»
	 доступны только в настольном приложении.
