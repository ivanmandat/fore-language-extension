# Лаг


Метод сдвигает ряд вперед на заданное количество точек во временном
 периоде. Входит в группу методов, осуществляющих преобразования над рядами.


[![](../../../Opened.gif)![](../../../Closed.gif)Для применения
 метода](javascript:TextPopup(this))


		- Выделите один или несколько рядов в таблице данных;


		- Выполните команду «Временные >
		 Лаг» в раскрывающемся меню кнопки ![](b_transf.gif) «Преобразования» на вкладке ленты
		 «Вычисления».


После применения метода в рабочей книге на основе каждого выделенного
 ряда будет создан вычисляемый ряд с наименованием вида «<Имя_Ряда>[t-<Значение_лага>]»,
 содержащий результаты расчета. Например:


![](Lag.gif)


## Настройка параметров расчёта. Вкладка «Параметры»


Для настройки параметров расчёта используйте вкладку «Параметры»
 на боковой панели.


[![](../../../Opened.gif)![](../../../Closed.gif)Для отображения
 вкладки](javascript:TextPopup(this))


		- Убедитесь, что боковая панель отображается;


		- Выделите в таблице данных ряд, рассчитанный методом «Лаг»;


		- Установите переключатель «Ряд»
		 на боковой панели;


		- Перейдите на вкладку «Параметры».


![](Lag_param.gif)


Параметры метода:


	- Смещение. Укажите значение
	 лага. Лаг может принимать только положительные значения. По умолчанию
	 значение лага равняется единице.
