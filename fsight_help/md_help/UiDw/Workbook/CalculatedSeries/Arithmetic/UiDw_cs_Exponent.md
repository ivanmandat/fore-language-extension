# Экспонента


Метод «Экспонента» вычисляет
 результат возведения числа «е» в степень, определяемую значениями ряда.
 Входит в группу арифметических методов.


[![](../../../Opened.gif)![](../../../Closed.gif)Для применения
 метода](javascript:TextPopup(this))


		- Выделите один или несколько рядов в таблице данных;


		- Выполните команду «Экспонента»
		 в раскрывающемся меню кнопки «Арифметика»
		 ![](b_arithm.gif) на вкладке ленты «Вычисления».


После применения метода в рабочей книге на основе каждого выделенного
 ряда будет создан вычисляемый ряд с наименованием вида «Exp(<Имя_Ряда>)»,
 содержащий результаты расчета. Например:
