# Импорт данных


Для импорта данных из внешних источников в БД временных рядов предназначен
 специальный дочерний объект «Объект импорта».
 Работа с объектом импорта доступна только в настольном приложении.


Возможности объекта импорта:


	- импорт из:


	-


		- БД временных рядов;


		- OLE DB;


		- Microsoft Excel (*.xls; *.xlsx);


		- Microsoft Access (*.mdb; *.accdb);


		- Microsoft XML (*.xml);


		- текстовые файлы (*.txt; *.csv);


		- буфера обмена;


	- масштабирование данных;


	- настройка на данные без предварительной подготовки исходных
	 файлов;


	- импорт составных значений с возможностью их разделения и т.д.


Для начала импорта данных откройте существующий или создайте новый объект
 импорта.


[![](../Opened.gif)![](../Closed.gif)Для создания
 нового объекта импорта](javascript:TextPopup(this))


		- Выберите БД временных рядов в навигаторе объектов:


			- в дереве объектов репозитория;


			- дважды щелкните в списке
			 объектов.


		- Выполните одну из следующих команд:


			- в раскрывающемся меню кнопки ![](../new_object.png)
			 «Новый объект >
			 Схема импорта БД временных
			 рядов» на вкладке ленты «[Главная](GetStarted.chm::/Interface/Interface_Description.htm#customize_ribbon)»;


			- в контекстного меню списка объектов «Создать
			 > Схема импорта БД временных
			 рядов».


	Откроется мастер импорта данных.


[![](../Opened.gif)![](../Closed.gif)Для открытия
 готового объекта импорта](javascript:TextPopup(this))


		- дважды щёлкните по объекту импорта;


		- выполните команду «Открыть» в
		 контекстном меню объекта импорта.


	Будет открыт мастер импорта данных.


Импорт состоит из следующих этапов:


	- [настройка источника](Data_Source.htm);


	- [привязка данных](Data_Binding.htm);


	- [сопоставление свободных
	 атрибутов](Non-Binding_Attributes.htm);


	- [выполнение импорта](Options_Review.htm);


	- [просмотр результатов](Results.htm).
