# Экспорт данных


Для экспорта данных из БД временных рядов предназначен специальный дочерний
 объект «Объект экспорта». Работа
 с объектом экспорта доступна только в настольном приложении.


Возможности объекта экспорта:


	- экспорт в:


	-


		- OLE DB;


		- Microsoft Excel (*.xls; *.xlsx);


		- Microsoft Access (*.mdb; *.accdb);


		- Microsoft XML (*.xml);


		- текстовые файлы (*.txt; *.csv);


	- масштабирование данных;


	- экспорт данных по состоянию на определенную ревизию;


	- выборочный экспорт.


Для начала экспорта данных создайте новый или откройте готовый
 объект экспорта.


[![](../Opened.gif)![](../Closed.gif)Для создания
 нового объекта экспорта](javascript:TextPopup(this))


		- Выберите БД временных рядов в навигаторе объектов:


			- в дереве объектов репозитория;


			- дважды щелкните в списке
			 объектов.


		- Выполните одну из следующих команд:


			- в раскрывающемся меню кнопки «Новый
			 объект > Схема
			 экспорта БД временных рядов» на вкладке ленты «[Главная](GetStarted.chm::/Interface/Interface_Description.htm#customize_ribbon)»;


			- в контекстном меню списка объектов «Создать
			 > Схема экспорта БД временных
			 рядов».


	Откроется мастер экспорта данных.


[![](../Opened.gif)![](../Closed.gif)Для открытия
 готового объекта экспорта](javascript:TextPopup(this))


		- дважды щёлкните по объекту экспорта;


		- выполните команду «Открыть» в
		 контекстном меню объекта экспорта.


	Будет открыт мастер экспорта данных.


Экспорт состоит из следующих этапов:


	- [выбор приёмника данных](UiDw_Export2.htm);


	- [привязка данных](UiDw_Export3.htm);


	- [выбор данных](UiDw_Export4.htm);


	- [выполнение экспорта](UiDw_Export5.htm).
