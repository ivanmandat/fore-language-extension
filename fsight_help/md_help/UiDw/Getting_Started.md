# Начало работы с инструментом «Анализ временных рядов»


Для начала работы создайте новую или откройте готовую рабочую книгу.


[![](Opened.gif)![](Closed.gif)Создание новой
 рабочей книги](javascript:TextPopup(this))


	Для создания новой рабочей книги:


		- в [навигаторе
		 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm):


			- в веб-приложении:


				- нажмите кнопку ![](purpose/add_web.png)
				 «Создать» в главном
				 меню и выберите тип объекта на боковой панели «[Новый объект](UiNav.chm::/02_Navigator/General_Principles_of_Work.htm#add_object)»;


				- выберите базу данных временных рядов и выполните
				 команду ![](purpose/b_open_web.png) «Открыть»
				 в контекстном меню выбранного объекта;


			- в настольном приложении:


				- выполните команду «Новый
				 объект > Рабочая книга» в группе «Создать»
				 на вкладке «Главная»
				 ленты инструментов;


				- выполните команду «Создать
				 > Рабочая
				 книга» в контекстном меню навигатора объектов.


				- выберите базу данных временных рядов и нажмите кнопку
				 ![](purpose/b_open.png) «Открыть
				 > В анализе временных рядов» в группе «Открыть» на вкладке
				 «Главная» ленты
				 инструментов;


				- выберите базу данных временных рядов и выполните
				 команду «Открыть с помощью
				 > Анализ временных рядов» в контекстном меню
				 выбранного объекта;


		- в инструменте «Анализ
		 временных рядов» выполните команду «Новая
		 рабочая книга» в главном меню «Рабочая
		 книга».


[![](Opened.gif)![](Closed.gif)Открытие готовой
 рабочей книги](javascript:TextPopup(this))


	Для открытия на редактирование или просмотр готовой рабочей
	 книги:


		- в [навигаторе
		 объектов](GetStarted.chm::/Interface/Interface_Navigator.htm):


			- в веб-приложении:


				- выполните команду ![](purpose/b_open_web.png) «Открыть» в контекстном
				 меню объекта;


				- дважды щёлкните по объекту;


				- нажмите клавишу ENTER;


			- в настольном приложении:


				- выделите объект и нажмите кнопку ![](purpose/b_open.png)
				 «Открыть», расположенную
				 в группе «Открыть»
				 на вкладке «Главная»
				 ленты инструментов;


				- выполните команду «Открыть»
				 в контекстном меню объекта;


				- дважды щёлкните по объекту;


		- в инструменте «Анализ
		 временных рядов» выполните команду «Открыть»
		 в главном меню «Рабочая книга».
		 В открывшемся диалоге выберите рабочую книгу из навигатора объектов.


Примечание.
 По умолчанию работа с инструментом «Анализ
 временных рядов» в веб-приложении недоступна, данный тип объектов
 не отображается в [навигаторе](GetStarted.chm::/Interface/Interface_Navigator.htm).
 Для доступа к инструменту задайте поле [hideClasses](setup.chm::/UiWebSetup/03_Setup_Web/PP_config_Java.htm#metabase_config) в
 [config.json](setup.chm::/UiWebSetup/03_Setup_Web/PP_config_Java.htm#config_json).


Основное окно инструмента «Анализ
 временных рядов»:


![](Purpose/window.png)


[![](Opened.gif)![](Closed.gif)Элементы
 окна рабочей книги](javascript:TextPopup(this))


		- [Главное меню](GetStarted.chm::/interface/interface_description.htm#main_menu).
		 Содержит список команд, предназначенных для выполнения различных
		 операций над рабочей книгой.


		- [Лента инструментов](GetStarted.chm::/interface/interface_description.htm#customize_ribbon).
		 Содержит кнопки для построения и настройки рабочей книги.


		- [Боковая панель](GetStarted.chm::/interface/interface_description.htm#side_panel).
		 Содержит настройки параметров временных рядов и управления измерениями
		 источников данных.


		- [Панель
		 быстрого доступа](GetStarted.chm::/Interface/Interface_Description.htm#quick_access_toolbar).
		 Содержит набор часто используемых команд.


		- Дерево рядов. Содержит
		 древовидную структуру рядов открытого источника данных.


		- Панель статистик.
		 Предназначена для просмотра характеристик объекта, выделенного
		 в рабочей области


		- Рабочая область.
		 Предназначена для построения рабочей книги.
