# IMetaDictionaryInstance


## Описание


Интерфейс IMetaDictionaryInstance
 предназначен для работы с данными о базе данных временных рядов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dictionary](IMetaDictionaryInstance.Dictionary.htm)


		 Свойство Dictionary
		 возвращает родительский объект.


		 ![](../../Property_Image.gif)
		 [Scheme](IMetaDictionaryInstance.Scheme.htm)


		 Свойство Scheme возвращает
		 данные базы данных временных рядов, использующей системную таблицу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CreateLookup](IMetaDictionaryInstance.CreateLookup.htm)


		 Метод CreateLookup
		 создает объект, осуществляющий поиск элементов базы данных временных
		 рядов по заданным значениям атрибутов.


		 ![](../../Sub_Image.gif)
		 [DefaultHierarchy](IMetaDictionaryInstance.DefaultHierarchy.htm)


		 Метод DefaultHierarchy
		 возвращает данные иерархии, установленной по умолчанию для справочника.


		 ![](../../Sub_Image.gif)
		 [GetData](IMetaDictionaryInstance.GetData.htm)


		 Метод GetData возвращает
		 данные базы данных временных рядов для заданного элемента.


		 ![](../../Sub_Image.gif)
		 [GetDataO](IMetaDictionaryInstance.GetDataO.htm)


		 Метод GetDataO возвращает
		 данные базы данных временных рядов для заданного элемента с заданными
		 параметрами.


		 ![](../../Sub_Image.gif)
		 [OpenHierarchy](IMetaDictionaryInstance.OpenHierarchy.htm)


		 Метод OpenHierarchy
		 открывает иерархию элементов базы данных временных рядов.
