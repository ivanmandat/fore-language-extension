# IRdsParamValues


## Описание


Интерфейс IRdsParamValues содержит
 свойства и методы для работы с коллекций значений параметров, используемых
 при открытии справочников НСИ.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IRdsParamValues.Count.htm)


		 Свойство Count возвращает
		 количество параметров.


		 ![](../../Property_Image.gif)
		 [CurDate](IRdsParamValues.CurDate.htm)


		 Свойство CurDate определяет
		 значение системного параметра - Дата актуальности.


		 ![](../../Property_Image.gif)
		 [Item](IRdsParamValues.Item.htm)


		 Свойство Item возвращает
		 значение параметра.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CreateCopy](IRdsParamValues.CreateCopy.htm)

		 [FindById](IRdsParamValues.FindById.htm)

		 Метод FindByKey осуществляет
