# IRdsParamValues.FindById


## Синтаксис


FindById(Id: String): [IRdsParamValue](../IRdsParamValue/IRdsParamValue.htm);


## Параметры


Id - идентификатор параметра,
 по которому осуществляется поиск.


## Описание


Метод FindById осуществляет
 поиск и возвращает значение параметра в случае удачного поиска. Идентификатор
 параметра, по которому осуществляется поиск, передается посредством параметра
 Id.
