# IRdsParamValues.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IRdsParamValue](../IRdsParamValue/IRdsParamValue.htm);


## Параметры


Key -
 ключ параметра, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает значение параметра в случае удачного поиска. Ключ параметра,
 по которому осуществляется поиск, передается посредством параметра Key.
