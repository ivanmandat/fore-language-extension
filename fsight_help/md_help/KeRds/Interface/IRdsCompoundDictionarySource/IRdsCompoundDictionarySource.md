# IRdsCompoundDictionarySource


## Описание


Интерфейс IRdsCompoundDictionarySource
 содержит свойства источника данных составного справочника НСИ.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dictionary](IRdsCompoundDictionarySource.Dictionary.htm)


		 Свойство Dictionary
		 возвращает объект-родитель.


		 ![](../../Property_Image.gif)
		 [Mappings](IRdsCompoundDictionarySource.Mappings.htm)


		 Свойство Mappings определяет
		 привязку атрибута составного справочника к атрибуту справочника-источника.


		 ![](../../Property_Image.gif)
		 [Source](IRdsCompoundDictionarySource.Source.htm)


		 Свойство Source определяет
		 справочник НСИ, используемый в качестве источника данных для данного
		 составного справочника.
