# IRdsImportSchema.TableName


## Синтаксис


TableName: String;


## Описание


Свойство TableName определяет
 таблицу репозитория, из которой будут импортироваться данные.


## Комментарии


Для импорта данных с помощью SQL-запроса в свойстве TableName
 укажите текст запроса. Учтите, что регистр идентификаторов полей не изменяется.
 В тексте запроса используйте регистр, совпадающий с регистром идентификаторов
 в таблице БД.
