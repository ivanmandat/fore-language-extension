# IRdsImportSchema.Mappings


## Синтаксис


Mappings(Attribute: [IRdsAttribute](../IRdsAttribute/IRdsAttribute.htm)): [IRdsImportSchemaAttribute](../IRdsImportSchemaAttribute/IRdsImportSchemaAttribute.htm);


## Параметры


Attribute. Атрибут справочника НСИ, для которого необходимо определить привязку к полям таблицы импорта.


## Описание


Свойство Mappings возвращает параметры привязки атрибутов справочника НСИ к полям таблицы, из которой осуществляется импорт данных.
