# IMetaAttributeGroupFilter


Сборка: Rds;


## Описание


Интерфейс IMetaAttributeGroupFilter
 предназначен для работы с фильтром группы атрибутов.


## Иерархия наследования


           IMetaAttributeGroupFilter


## Комментарии


Фильтр задает отметку значений атрибутов и определяет набор показателей,
 входящих в группу. Фильтр группы возвращает свойство [IMetaAttributeGroup.Filter](../IMetaAttributeGroup/IMetaAttributeGroup.Filter.htm).
