# IMetaAttributeGroupFilter.IsEmpty


## Синтаксис


IsEmpty: Boolean;


## Описание


Свойство IsEmpty возвращает
 признак того, установлена ли отметка фильтра.


## Комментарии


Если свойство возвращает значение True,
 то отметка фильтра установлена.
