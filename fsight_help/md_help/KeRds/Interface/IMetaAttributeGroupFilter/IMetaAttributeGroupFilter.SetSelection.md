# IMetaAttributeGroupFilter.SetSelection


## Синтаксис


SetSelection(Value: [IDimSelectionSet](KeDims.chm::/interface/IDimSelectionSet/IDimSelectionSet.htm));


## Параметры


Value. Отметка элементов.


## Описание


Метод SetSelection устанавливает
 отметку элементов в фильтр.


## Комментарии


Если отметка элементов пустая или параметр Value
 = Null, то фильтрация очищается.
