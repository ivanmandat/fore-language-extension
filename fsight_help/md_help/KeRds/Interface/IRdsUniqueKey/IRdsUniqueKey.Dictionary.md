# IRdsUniqueKey.Dictionary


## Синтаксис


Dictionary: [IRdsDictionary](../IRdsDictionary/IRdsDictionary.htm);


## Описание


Свойство Dictionary возвращает
 справочник НСИ, которому принадлежит уникальный ключ.


## Комментарии


Для получения всех уникальных ключей справочника используйте свойство
 [IRdsDictionary.UniqueKeys](../IRdsDictionary/IRdsDictionary.UniqueKeys.htm).
