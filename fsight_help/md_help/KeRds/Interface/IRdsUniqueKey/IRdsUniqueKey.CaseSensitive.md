# IRdsUniqueKey.CaseSensitive


## Синтаксис


CaseSensitive: Boolean;


## Описание


Свойство CaseSensitive
 определяет, чувствителен ли уникальный ключ к регистру.


## Комментарии


Свойство используется, если уникальный ключ содержится в справочнике,
 используемом в качестве измерения.


Допустимые значения:


	- True. Уникальный ключ
	 чувствителен к регистру;


	- False. Уникальный ключ
	 регистронезависимый.
