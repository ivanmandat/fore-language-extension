# IMetaHierarchyLevels


## Описание


Интерфейс IMetaHierarchyLevels
 содержит свойства и методы для работы с коллекцией уровней иерархии.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMetaHierarchyLevels.Count.htm)


		 Свойство Count возвращает
		 количество уровней в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMetaHierarchyLevels.Item.htm)


		 Свойство Item возвращает
		 уровень из коллекции по индексу.


		 ![](../../Property_Image.gif)
		 [Leaf](IMetaHierarchyLevels.Leaf.htm)


		 Свойство Leaf возвращает
		 листьевой уровень коллекции.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](IMetaHierarchyLevels.FindById.htm)


		 Метод FindById осуществляет
		 поиск уровня в коллекции по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IMetaHierarchyLevels.FindByKey.htm)


		 Метод FindByKey осуществляет
		 поиск уровня в коллекции по ключу.


		 ![](../../Sub_Image.gif)
		 [IndexOf](IMetaHierarchyLevels.IndexOf.htm)


		 Метод IndexOf возвращает
		 индекс заданного уровня в коллекции.


		 ![](../../Sub_Image.gif)
		 [OpenDimensions](IMetaHierarchyLevels.OpenDimensions.htm)


		 Метод OpenDimensions
		 осуществляет открытие измерения.
