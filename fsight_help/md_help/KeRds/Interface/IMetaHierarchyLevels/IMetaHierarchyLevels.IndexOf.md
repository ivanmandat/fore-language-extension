# IMetaHierarchyLevels.IndexOf


## Синтаксис


IndexOf(Value: [IMetaHierarchyLevel](../IMetaHierarchyLevel/IMetaHierarchyLevel.htm)):
 Integer;


## Параметры


Value - уровень иерархии.


## Описание


Метод IndexOf возвращает индекс
 заданного уровня в коллекции. Если уровень отсутствует в коллекции, то
 метод возвращает значение «-1».
