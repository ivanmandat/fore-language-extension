# IRdsDictionaryElementHistory.Version


## Синтаксис


Version(Index: Integer): Integer;


## Параметры


Index. Индекс записи в истории элемента.


## Описание


Свойство Version возвращает код версии элемента. Текущая версия имеет код - "0".
