# IRdsLinkedBinding.Reference


## Синтаксис


Reference: [IRdsAttribute](../IRdsAttribute/IRdsAttribute.htm);


## Описание


Свойство Reference определяет
 атрибут справочника, с которым осуществляется связь.


## Комментарии


Связь будет установлена с использованием атрибута, которому соответствует
 привязка, и атрибутом, который указан в свойстве Reference.

Пример использования приведен в описании метода [IRdsLoaderBindings.Add](../IRdsLoaderBindings/IRdsLoaderBindings.Add.htm).
