# IRdsDictionary.Hierarchical


## Синтаксис


Hierarchical: Boolean;


## Описание


Свойство Hierarchical определяет
 признак иерархического справочника.


## Комментарии


Если значение данного свойства True,
 то справочник будет иерархический, если False
 - справочник будет одноуровневый. По умолчанию свойство имеет значение
 True.
