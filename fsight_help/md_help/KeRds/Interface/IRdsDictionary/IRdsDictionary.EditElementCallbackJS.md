# IRdsDictionary.EditElementCallbackJS


## Синтаксис


		EditElementCallbackJS: String;


## Описание


Свойство EditElementCallbackJS
 устанавливает JS-функцию для замены стандартного вида диалога «[Свойства
 элемента справочника](UiNavObj.chm::/reference_book/Master_RDS_reference_book/Element_Card.htm)»
 на пользовательский в веб-приложении.


## Комментарии


При установке свойства корректность функции не проверяется.


Для удаления установленной функции установите у свойства пустое значение.
