# IRdsDictionary.DatabaseInstance


## Синтаксис


DatabaseInstance: [IDatabaseInstance](KeDb.chm::/Interface/IDatabaseInstance/IDatabaseInstance.htm);


## Описание


Свойство DatabaseInstance возвращает содержимое базы данных, в которой хранятся данные справочника НСИ.


## Комментарии


Для получения свойств базы данных, в которой хранятся данные справочника НСИ, используйте свойство [IRdsDictionary.Database](IRdsDictionary.Database.htm).
