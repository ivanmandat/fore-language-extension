# IRdsDictionary.HasMandatoryAccess


## Синтаксис


HasMandatoryAccess: Boolean;


## Описание


Свойство HasMandatoryAccess
 определяет, используется ли мандатный контроль доступа для элементов
 справочника НСИ.


## Комментарии


Допустимые значения:


	- True. Для элементов
	 справочника НСИ используется мандатный контроль доступа;


	- False. Мандатный контроль
	 доступа не применяется для элементов справочника НСИ.
