# IRdsDictionary.ResetDefaultElement


## Синтаксис


ResetDefaultElement;


## Описание


Метод ResetDefaultElement сбрасывает
 установку элемента по умолчанию.


## Комментарии


Для установки элемента по умолчанию используйте свойство [IRdsDictionary.DefaultElementKey](IRdsDictionary.DefaultElementKey.htm).
