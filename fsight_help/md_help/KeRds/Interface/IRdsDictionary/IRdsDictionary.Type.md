# IRdsDictionary.Type


## Синтаксис


Type: [RdsDictionaryType](../../Enums/RdsDictionaryType.htm);


## Описание


Свойство Type возвращает тип
 справочника НСИ.


## Комментарии


Если свойство вернуло значение RdsDictionaryType.MetaDictionaryRds,
 то работа ведется с табличным справочником НСИ.
