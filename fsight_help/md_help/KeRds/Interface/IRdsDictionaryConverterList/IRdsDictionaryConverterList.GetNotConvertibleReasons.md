# IRdsDictionaryConverterList.GetNotConvertibleReasons


## Синтаксис


GetNotConvertibleReasons(Dictionary: Integer): [IRdsNotConvertibleReasons](../IRdsNotConvertibleReasons/IRdsNotConvertibleReasons.htm);


## Параметры


Dictionary. Ключ справочника.


## Описание


Метод GetNotConvertibleReasons
 позволяет выяснить причину, по которой справочник НСИ не был преобразован
 в табличный справочник НСИ.


## Комментарии


Для получения описания справочников НСИ, которые были преобразованы
 в табличные справочники НСИ, используйте свойство [IRdsDictionaryConverterList.ConvertibleDictionaries](IRdsDictionaryConverterList.ConvertibleDictionaries.htm).
