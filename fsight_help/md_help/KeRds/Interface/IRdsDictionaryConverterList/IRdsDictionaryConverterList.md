# IRdsDictionaryConverterList


Сборка: Rds;


## Описание


Интерфейс IRdsDictionaryConverterList
 предназначен для работы со списком справочников НСИ, которые необходимо
 преобразовать в табличные справочники НСИ.


## Иерархия наследования


           IRdsDictionaryConverterList


## Комментарии


Для преобразования справочника НСИ в табличный справочник НСИ используйте
 свойства и методы интерфейса [IRdsDictionaryConverter](../IRdsDictionaryConverter/IRdsDictionaryConverter.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ConvertibleDictionaries](IRdsDictionaryConverterList.ConvertibleDictionaries.htm)


		 Свойство ConvertibleDictionaries
		 возвращает описание справочников НСИ, которые были преобразованы
		 в табличные справочники НСИ.


		 ![](../../Property_Image.gif)
		 [NotConvertibleDictionaries](IRdsDictionaryConverterList.NotConvertibleDictionaries.htm)


		 Свойство NotConvertibleDictionaries
		 возвращает описание справочников НСИ, которые не были преобразованы
		 в табличные справочники НСИ.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [GetNotConvertibleReasons](IRdsDictionaryConverterList.GetNotConvertibleReasons.htm)


		 Метод GetNotConvertibleReasons
		 позволяет выяснить причину, по которой справочник НСИ не был преобразован
		 в табличный справочник НСИ.
