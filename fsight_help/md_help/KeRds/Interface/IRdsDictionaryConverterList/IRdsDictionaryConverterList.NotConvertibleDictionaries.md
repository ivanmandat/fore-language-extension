# IRdsDictionaryConverterList.NotConvertibleDictionaries


## Синтаксис


NotConvertibleDictionaries: [IMetabaseObjectDescriptors](KeSom.chm::/Interface/IMetabaseObjectDescriptors/IMetabaseObjectDescriptors.htm);


## Описание


Свойство NotConvertibleDictionaries
 возвращает описание справочников НСИ, которые не были преобразованы в
 табличные справочники НСИ.


## Комментарии


Для получения описания справочников НСИ, которые были преобразованы
 в табличные справочники НСИ, используйте свойство [IRdsDictionaryConverterList.ConvertibleDictionaries](IRdsDictionaryConverterList.ConvertibleDictionaries.htm).
