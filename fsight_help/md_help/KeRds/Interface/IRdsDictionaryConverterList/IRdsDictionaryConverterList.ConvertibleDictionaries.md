# IRdsDictionaryConverterList.ConvertibleDictionaries


## Синтаксис


ConvertibleDictionaries: [IMetabaseObjectDescriptors](KeSom.chm::/Interface/IMetabaseObjectDescriptors/IMetabaseObjectDescriptors.htm);


## Описание


Свойство ConvertibleDictionaries
 возвращает описание справочников НСИ, которые были преобразованы в табличные
 справочники НСИ.


## Комментарии


Для получения описания справочников НСИ, которые не были преобразованы
 в табличные справочники НСИ, используйте свойство [IRdsDictionaryConverterList.NotConvertibleDictionaries](IRdsDictionaryConverterList.NotConvertibleDictionaries.htm).
