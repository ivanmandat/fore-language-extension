# IRdsDatabase


Сборка: Rds;


## Описание


Интерфейс IRdsDatabase содержит
 свойства и методы объекта репозитория «[Репозиторий
 НСИ](UiNavObj.chm::/repository_RDS/RepositoryRDS.htm)».


## Иерархия наследования


IRdsDatabase


## Комментарии


При создании нового репозитория НСИ для его корректной работы необходимо
 задать свойство [Database](IRdsDatabase.Database.htm).
