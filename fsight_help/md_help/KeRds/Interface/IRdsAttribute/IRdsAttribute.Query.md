# IRdsAttribute.Query


## Синтаксис


Query: String;


## Описание


Свойство Query определяет
 текст запроса, по которому рассчитывается значение вычисляемого атрибута.


## Комментарии


Для определения, является ли атрибут вычисляемым, используйте свойство
 [IRdsAttribute.Calculated](IRdsAttribute.Calculated.htm).


В тексте запроса можно использовать маску #TABLE#,
 которая при выполнении запроса заменяется на имя таблицы табличного справочника
 НСИ. Корректность запроса не проверяется.


Например: select t1."KEY"
 from test1_data.T2522 t1 where t1."KEY"=#TABLE#."KEY".


Если в запросе присутствуют поля основной таблицы и заимствованные поля,
 то перед основными полями необходимо использовать маску #TABLE#.
