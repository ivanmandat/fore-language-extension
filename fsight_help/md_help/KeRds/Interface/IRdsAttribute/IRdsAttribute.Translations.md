# IRdsAttribute.Translations


## Синтаксис


Translations: [IRdsAttributeTranslations](../IRdsAttributeTranslations/IRdsAttributeTranslations.htm);


## Описание


Свойство доступно только для чтения.


Свойство Translations возвращает
 коллекцию настроек, содержащих параметры перевода наименования атрибута
 справочника НСИ на различные языки.


## Комментарии


Изначально в коллекции содержится настройка для языка, установленного
 по умолчанию для репозитория пользователя. Чтобы переводить наименование
 атрибута справочника на различные языки необходимо установить значение
 True в свойства [IRdsDictionaryTranslations.IsOn](../IRdsDictionaryTranslations/IRdsDictionaryTranslations.IsOn.htm)
 и [IRdsAttributeTranslations.IsOn](../IRdsAttributeTranslations/IRdsAttributeTranslations.IsOn.htm).
