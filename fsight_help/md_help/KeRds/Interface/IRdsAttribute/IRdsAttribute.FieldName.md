# IRdsAttribute.FieldName


## Синтаксис


FieldName: String;


## Описание


Свойство FieldName определяет наименование поля таблицы, в котором будут храниться значения атрибута. Данное свойство может использоваться при настройке схем импорта и экспорта данных справочника НСИ.
