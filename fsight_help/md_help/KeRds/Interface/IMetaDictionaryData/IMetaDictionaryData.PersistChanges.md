# IMetaDictionaryData.PersistChanges


## Синтаксис


PersistChanges([Force: Boolean = False]);


## Параметры


Force -
 параметр, определяющий сохранение изменений. Необязательный параметр.
 Значение по умолчанию - False
 (не сохранять).


## Описание


Метод PersistChanges определяет,
 производить ли сохранение изменений в базу данных.
