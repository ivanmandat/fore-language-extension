# IMetaDictionaryData


## Описание


Интерфейс IMetaDictionaryData
 содержит свойства и методы для работы с данными базы данных временных
 рядов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dictionary](IMetaDictionaryData.Dictionary.htm)


		 Свойство Dictionary
		 возвращает данные о базе данных временных рядов.


		 ![](../../Property_Image.gif)
		 [Record](IMetaDictionaryData.Record.htm)


		 Свойство Record возвращает
		 значения атрибутов базы данных временных рядов.


		 ![](../../Property_Image.gif)
		 [SD](IMetaDictionaryData.SD.htm)


		 Свойство SD возвращает
		 поле дескриптора безопасности. Находится в стадии разработки.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AsObject](IMetaDictionaryData.AsObject.htm)


		 Метод AsObject возвращает
		 временный объект для редактирования дескриптора безопасности.
		 Находится в стадии разработки.


		 ![](../../Sub_Image.gif)
		 [Close](IMetaDictionaryData.Close.htm)


		 Метод Close осуществляет
		 закрытие данных без сохранения.


		 ![](../../Sub_Image.gif)
		 [EvaluateRule](IMetaDictionaryData.EvaluateRule.htm)


		 Метод EvaluateRule
		 осуществляет проверку указанного правила справочника.


		 ![](../../Sub_Image.gif)
		 [EvaluateRules](IMetaDictionaryData.EvaluateRules.htm)


		 Метод EvaluateRules
		 осуществляет проверку всех правил справочника.


		 ![](../../Sub_Image.gif)
		 [PersistChanges](IMetaDictionaryData.PersistChanges.htm)


		 Метод PersistChanges
		 определяет, производить ли сохранение изменений в базу данных.


		 ![](../../Sub_Image.gif)
		 [Refresh](IMetaDictionaryData.Refresh.htm)


		 Метод Refresh осуществляет
		 обновление данных.


		 ![](../../Sub_Image.gif)
		 [Save](IMetaDictionaryData.Save.htm)


		 Метод Save осуществляет
		 сохранение данных.
