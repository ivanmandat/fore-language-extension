# IRdsSortAttributes.FindByRefAttribute


## Синтаксис


FindByRefAttribute: (Attribute: [IRdsAttribute](../IRdsAttribute/IRdsAttribute.htm); RefAttribute:
 [IRdsAttribute](../IRdsAttribute/IRdsAttribute.htm)): [IRdsSortAttribute](../IRdsSortAttribute/IRdsSortAttribute.htm);


## Параметры


Attribute. Атрибут справочника
 НСИ, по которому осуществляется поиск;


RefAttribute. Атрибут связанного
 справочника НСИ, по которому осуществляется поиск.


## Описание


Метод FindByRefAttribute позволяет
 осуществить поиск атрибута сортировки по связанному атрибуту, который
 передается посредством параметра RefAttribute.
