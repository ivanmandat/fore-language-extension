# IRdsSortAttributes


## Описание


Интерфейс IRdsSortAttributes
 содержит свойства и методы для работы с коллекцией атрибутов сортировки
 элементов справочника НСИ.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IRdsSortAttributes.Count.htm)


		 Свойство Count возвращает
		 количество атрибутов сортировки в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IRdsSortAttributes.Item.htm)


		 Свойство Item возвращает
		 атрибут сортировки из коллекции по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IRdsSortAttributes.Add.htm)


		 Метод Add добавляет
		 сортировку по указанному атрибуту с заданным направлением сортировки.


		 ![](../../Sub_Image.gif)
		 [AddRefAttribute](IRdsSortAttributes.AddRefAttribute.htm)


		 Метод AddRefAttribute
		 добавляет сортировку для атрибутов, участвующих в связях, и позволяет
		 указать атрибут из другого справочника для сортировки по его значениям.


		 ![](../../Sub_Image.gif)
		 [Clear](IRdsSortAttributes.Clear.htm)


		 Метод Clear осуществляет
		 удаление всех атрибутов сортировки из коллекции.


		 ![](../../Sub_Image.gif)
		 [FindByAttribute](IRdsSortAttributes.FindByAttribute.htm)


		 Метод FindByAttribute
		 осуществляет поиск атрибута сортировки по атрибуту справочника.


		 ![](../../Sub_Image.gif)
		 [FindByRefAttribute](IRdsSortAttributes.FindByRefAttribute.htm)


		 Метод FindByRefAttribute
		 позволяет осуществить поиск атрибута сортировки по связанному
		 атрибуту.


		 ![](../../Sub_Image.gif)
		 [LoadFrom](IRdsSortAttributes.LoadFrom.htm)
		 Метод LoadFrom загружает
		 параметры сортировки значений атрибутов из XML.


		 ![](../../Sub_Image.gif)
		 [Remove](IRdsSortAttributes.Remove.htm)


		 Метод Remove осуществляет
		 удаление атрибута сортировки, индекс которого передается посредством
		 параметра Index.


		 ![](../../Sub_Image.gif)
		 [SaveTo](IRdsSortAttributes.SaveTo.htm)
		 Метод SaveTo сохраняет
		 параметры сортировки значений атрибутов в формате XML.
