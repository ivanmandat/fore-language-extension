# IRdsSortAttributes.FindByAttribute


## Синтаксис


FindByAttribute(Attribute: [IRdsAttribute](../IRdsAttribute/IRdsAttribute.htm);
 StartIndex: Integer=0): [IRdsSortAttribute](../IRdsSortAttribute/IRdsSortAttribute.htm);


## Параметры


Attribute. Атрибут справочника
 НСИ, по которому осуществляется поиск;


StartIndex. Начальный индекс.
 Необязательный параметр, значение по умолчанию «0».


## Описание


Метод FindByAttribute осуществляет
 поиск атрибута сортировки по атрибуту справочника. Атрибут справочника
 НСИ, по которому осуществляется поиск, передается посредством параметра
 Attribute.
