# IMetaDataMemberCollection.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс записи в коллекции.


## Описание


Метод Remove осуществляет удаление
 записи из коллекции по индексу. Метод возвращает значение True,
 если удаление прошло успешно.
