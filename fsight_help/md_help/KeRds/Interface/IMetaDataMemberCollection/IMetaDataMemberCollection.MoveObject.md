# IMetaDataMemberCollection.MoveObject


## Синтаксис


MoveObject(Item: [IMetaDataMember](../IMetaDataMember/IMetaDataMember.htm);
 Position: Integer): Boolean;


## Параметры


Item
 - запись, которую необходимо переместить.


Position - индекс позиции,
 в которую необходимо переместить запись.


## Описание


Метод MoveObject осуществляет
 перемещение указанной записи в коллекции. Метод возвращает значение True, если перемещение прошло успешно.
