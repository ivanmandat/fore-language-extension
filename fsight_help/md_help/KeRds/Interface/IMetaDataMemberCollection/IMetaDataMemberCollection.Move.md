# IMetaDataMemberCollection.Move


## Синтаксис


Move(IndexFrom: Integer; IndexTo: Integer): Boolean;


## Параметры


IndexFrom - индекс позиции
 записи, которую необходимо переместить.


IndexTo - индекс позиции, в
 которую необходимо переместить запись.


## Описание


Метод Move осуществляет перемещение
 записи в коллекции по индексам. Метод возвращает значение True,
 если перемещение прошло успешно.
