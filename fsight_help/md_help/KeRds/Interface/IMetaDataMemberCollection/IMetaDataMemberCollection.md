# IMetaDataMemberCollection


## Описание


Интерфейс IMetaDataMemberCollection
 содержит свойства и методы для работы с коллекцией записей системной таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMetaDataMemberCollection.Count.htm)


		 Свойство Count возвращает
		 количество записей в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMetaDataMemberCollection.Item.htm)


		 Свойство Item возвращает
		 запись из коллекции по индексу.


## Свойства, унаследованные от [IMetaDataMember](../IMetaDataMember/IMetaDataMember.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attribute](../IMetaDataMember/IMetaDataMember.Attribute.htm)


		 Свойство Attribute
		 возвращает атрибут записи.


		 ![](../../Property_Image.gif)
		 [Modified](../IMetaDataMember/IMetaDataMember.Modified.htm)


		 Свойство Modified возвращает
		 признак того, была ли запись изменена.


		 ![](../../Property_Image.gif)
		 [UnmodifiedValue](../IMetaDataMember/IMetaDataMember.UnmodifiedValue.htm)


		 Свойство UnmodifiedValue
		 возвращает значение записи до изменения.


		 ![](../../Property_Image.gif)
		 [Value](../IMetaDataMember/IMetaDataMember.Value.htm)


		 Свойство Value определяет
		 значение записи.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IMetaDataMemberCollection.Add.htm)


		 Метод Add осуществляет

		 ![](../../Sub_Image.gif)
		 [Clear](IMetaDataMemberCollection.Clear.htm)


		 Метод Clear осуществляет

		 Метод MoveObject осуществляет

		 ![](../../Sub_Image.gif)
		 [Remove](IMetaDataMemberCollection.Remove.htm)


		 Метод Remove осуществляет
		 удаление записи из коллекции по индексу.
