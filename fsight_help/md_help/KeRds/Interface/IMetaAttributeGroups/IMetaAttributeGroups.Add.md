# IMetaAttributeGroups.Add


## Синтаксис


Add: [IMetaAttributeGroup](../IMetaAttributeGroup/IMetaAttributeGroup.htm);


## Описание


Метод Add добавляет группу
 атрибутов.


## Комментарии


Группа атрибутов предназначена для создания группового атрибута. Значение
 данного атрибута изменяется для всех показателей, входящих в группу.
