# IRdsImportedBinding.LinkBinding


## Синтаксис


LinkBinding: [IRdsLinkedBinding](../IRdsLinkedBinding/IRdsLinkedBinding.htm);


## Описание


Свойство LinkBinding определяет
 привязку атрибута, используемого для связи со справочником, из которого
 заимствуется атрибут.
