# IRdsNotConvertibleReasonConverterConstraint


Сборка: Rds;


## Описание


Интерфейс IRdsNotConvertibleReasonConverterConstraint
 предназначен для работы с причинами, по которым справочник НСИ не может
 быть преобразован в табличный справочник НСИ, связанными с ограничениями
 преобразователя.


## Иерархия наследования


           [IRdsNotConvertibleReason](../IRdsNotConvertibleReason/IRdsNotConvertibleReason.htm)

## Комментарии


Для работы с причиной, по которой справочник НСИ не может быть преобразован
 в табличный справочник НСИ, используйте свойства и методы интерфейса [IRdsNotConvertibleReason](../IRdsNotConvertibleReason/IRdsNotConvertibleReason.htm).
