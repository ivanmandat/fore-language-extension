# IRdsParamsControlInstance


## Описание


Интерфейс IRdsParamsControlInstance
 содержит свойства для управления значениями параметров связанного справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ControlState](IRdsParamsControlInstance.ControlState.htm)


		 Свойство ControlState
		 возвращает текущее состояние объекта, осуществляющего управление
		 значениями параметров связанного справочника.


		 ![](../../Property_Image.gif)
		 [Data](IRdsParamsControlInstance.Data.htm)


		 Свойство Data возвращает
		 данные атрибутов справочника, значения которых берутся в качестве
		 значений для параметров связанного справочника.


		 ![](../../Property_Image.gif)
		 [Link](IRdsParamsControlInstance.Link.htm)


		 Свойство Link возвращает
		 связь со справочником, параметрами которого необходимо управлять.


		 ![](../../Property_Image.gif)
		 [ReferenceDictionary](IRdsParamsControlInstance.ReferenceDictionary.htm)


		 Свойство ReferenceDictionary
		 возвращает данные связанного справочника, параметрами которого
		 осуществляется управление.
