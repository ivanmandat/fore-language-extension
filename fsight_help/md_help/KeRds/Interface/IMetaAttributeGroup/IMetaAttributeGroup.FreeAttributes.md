# IMetaAttributeGroup.FreeAttributes


## Синтаксис


FreeAttributes: [IMetaAttributesSet](../IMetaAttributesSet/IMetaAttributesSet.htm);


## Описание


Свойство FreeAttributes определяет
 набор свободных атрибутов, входящих в группу.


## Комментарии


Каждая группа должна иметь минимум один свободный атрибут. Набор должен
 состоять из атрибутов, которые входят в состав первичного индекса каталога.
 Данный индекс возвращает свойство [IMetaIndex.Primary](../IMetaIndex/IMetaIndex.Primary.htm).
