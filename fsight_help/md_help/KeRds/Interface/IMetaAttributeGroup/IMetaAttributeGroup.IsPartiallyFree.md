# IMetaAttributeGroup.IsPartiallyFree


## Синтаксис


IsPartiallyFree: Boolean;


## Описание


Свойство IsPartiallyFree возвращает
 признак того, что в группе присутствует свободный атрибут.


## Комментарии


Если свойство вернуло значение True,
 значит в группе присутствует свободный атрибут. Каждая группа должна иметь
 минимум один свободный атрибут.
