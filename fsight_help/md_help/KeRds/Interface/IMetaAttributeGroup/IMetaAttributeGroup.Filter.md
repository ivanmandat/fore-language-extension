# IMetaAttributeGroup.Filter


## Синтаксис


Filter: [IMetaAttributesSet](../IMetaAttributesSet/IMetaAttributesSet.htm);


## Описание


Свойство Filter возвращает
 фильтр группы.


## Комментарии


Фильтр задает отметку значений первичных/свободных атрибутов и определяет
 набор показателей, входящих в группу.
