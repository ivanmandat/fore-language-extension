# IMetaAttributeGroup.PrimaryAttributes


## Синтаксис


PrimaryAttributes: [IMetaAttributesSet](../IMetaAttributesSet/IMetaAttributesSet.htm);


## Описание


Свойство PrimaryAttributes определяет
 набор первичных атрибутов, входящих в группу.


## Комментарии


Набор должен состоять из атрибутов, которые входят в состав первичного
 индекса каталога. Данный индекс возвращает свойство [IMetaIndex.Primary](../IMetaIndex/IMetaIndex.Primary.htm).
