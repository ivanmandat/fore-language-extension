# IRdsReplicator.Merge


## Синтаксис


Merge([Callback: [IRdsMergeCallback](../IRdsMergeCallback/IRdsMergeCallback.htm) = Null]);


## Параметры


Callback. Объект, используемый для обработки исключительных ситуаций, которые могут возникнуть во время синхронизации.


## Описание


Метод Merge осуществляет синхронизацию справочников.


## Комментарии


Для загрузки данных синхронизируемых справочников используйте метод [IRdsReplicator.LoadFromFile](IRdsReplicator.LoadFromFile.htm).
