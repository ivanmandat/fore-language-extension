# IRdsReplicator


## Описание


Интерфейс IRdsReplicator содержит
 свойства и методы для синхронизации справочников репозитория НСИ.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dictionaries](IRdsReplicator.Dictionaries.htm)


		 Свойство Dictionaries
		 возвращает коллекцию справочников, данные которых необходимо синхронизировать.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [LoadFromFile](IRdsReplicator.LoadFromFile.htm)


		 Метод LoadFromFile
		 осуществляет загрузку данных синхронизируемых справочников.


		 ![](../../Sub_Image.gif)
		 [Merge](IRdsReplicator.Merge.htm)


		 Метод Merge осуществляет
		 синхронизацию справочников.


		 ![](../../Sub_Image.gif)
		 [SaveToFile](IRdsReplicator.SaveToFile.htm)


		 Метод SaveToFile осуществляет
		 сохранение данных синхронизируемых справочников в файл.
