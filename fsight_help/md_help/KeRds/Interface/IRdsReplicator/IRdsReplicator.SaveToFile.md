# IRdsReplicator.SaveToFile


## Синтаксис


SaveToFile(FileName: String);


## Параметры


FileName. Файл, в который будут сохранены данные синхронизируемых справочников.


## Описание


Метод SaveToFile осуществляет сохранение данных синхронизируемых справочников в файл.


## Комментарии


Путь и наименование файла, в который сохраняются данные, передается посредством параметра FileName.
