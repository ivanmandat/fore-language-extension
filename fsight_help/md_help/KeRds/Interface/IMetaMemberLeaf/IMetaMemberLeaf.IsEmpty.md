# IMetaMemberLeaf.IsEmpty


## Синтаксис


IsEmpty: Boolean;


## Описание


Свойство IsEmpty определяет,
 установлен ли для элемента признак пустоты.


## Комментарии


Признак пустоты применяется для показателей базы данных временных рядов
 и зависит от значения атрибута показателей «Пустой» (идентификатор - «EMPT»).


Если у показателя все значения в БД на актуальную ревизию являются пустыми
 (отсутствуют или равны Null),
 то у него устанавливается признак пустоты. Новый показатель считается
 пустым, пока в него не сохранили данные.
