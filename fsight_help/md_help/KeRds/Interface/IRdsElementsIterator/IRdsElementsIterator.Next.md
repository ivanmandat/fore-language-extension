# IRdsElementsIterator.Next


## Синтаксис


Next: Boolean;


## Описание


Метод Next переводит итератор
 на следующий элемент и возвращает признак доступности последующего перехода.


## Комментарии


Если метод возвращает True,
 то доступен дальнейший переход итератора. Если метод возвращает False, то итератор достиг последнего
 элемента. Для перевода итератора к первому элементу используйте метод
 [IRdsElementsIterator.First](IRdsElementsIterator.First.htm).
