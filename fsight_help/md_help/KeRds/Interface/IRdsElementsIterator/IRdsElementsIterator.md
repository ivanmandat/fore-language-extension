# IRdsElementsIterator


Сборка: Rds;


## Описание


Интерфейс IRdsElementsIterator
 используется для работы с итератором, предназначенным для навигации по
 элементам справочника.


## Иерархия наследования


           IRdsElementsIterator


## Комментарии


Итератор используется для навигации по справочнику, содержащему большое
 количество элементов.
