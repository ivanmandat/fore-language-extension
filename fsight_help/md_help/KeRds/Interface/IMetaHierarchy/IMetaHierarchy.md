# IMetaHierarchy


## Описание


Интерфейс IMetaHierarchy содержит
 свойства для работы с иерархией базы данных временных рядов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attributes](IMetaHierarchy.Attributes.htm)


		 Свойство Attributes
		 возвращает атрибуты, входящие в иерархию.


		 ![](../../Property_Image.gif)
		 [CustomWhere](IMetaHierarchy.CustomWhere.htm)


		 Свойство CustomWhere
		 возвращает дополнительные условия иерархии.


		 ![](../../Property_Image.gif)
		 [LeafLoadExtent](IMetaHierarchy.LeafLoadExtent.htm)


		 Свойство LeafLoadExtent
		 накладывает ограничение на количество загружаемых листьевых элементов.


		 ![](../../Property_Image.gif)
		 [Levels](IMetaHierarchy.Levels.htm)


		 Свойство Levels возвращает
		 уровни иерархии.


		 ![](../../Property_Image.gif)
		 [WhereIsEmpty](IMetaHierarchy.WhereIsEmpty.htm)


		 Свойство WhereIsEmpty
		 определяет, присутствуют ли в иерархии записи, для которых
		 установлен признак пустоты.
