# IMetaAttributeNamings


## Описание


Интерфейс IMetaAttributeNamings
 содержит свойства и методы для работы с коллекцией наименований атрибутов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMetaAttributeNamings.Count.htm)
		 Свойство Count возвращает
		 количество наименований в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMetaAttributeNamings.Item.htm)
		 Свойство Item возвращает
		 наименование из коллекции по индексу.


		 ![](../../Property_Image.gif)
		 [NamingDelimiter](IMetaAttributeNamings.NamingDelimiter.htm)
		 Свойство NamingDelimiter
		 определяет разделитель наименований атрибутов.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IMetaAttributeNamings.Add.htm)
		 Метод Add осуществляет
		 добавление наименования атрибута в коллекцию.


		 ![](../../Sub_Image.gif)
		 [FindById](IMetaAttributeNamings.FindById.htm)
		 Метод FindById осуществляет
		 поиск наименования в коллекции по идентификатору атрибута.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IMetaAttributeNamings.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск наименования в коллекции по ключу атрибута.


		 ![](../../Sub_Image.gif)
		 [GetNames](IMetaAttributeNamings.GetNames.htm)
		 Метод GetNames позволяет
		 получить наименование элементов базы данных временных рядов по
		 заданным значениям.


		 ![](../../Sub_Image.gif)
		 [GetNamesRaw](IMetaAttributeNamings.GetNamesRaw.htm)
		 Метод GetNamesRaw позволяет
		 получить наименование элементов базы данных временных рядов по
		 заданным значениям и измерениям.


		 ![](../../Sub_Image.gif)
		 [GetRecordMnemo](IMetaAttributeNamings.GetRecordMnemo.htm)
		 Метод GetRecordMnemo
		 возвращает мнемонику элемента базы данных временных рядов.


		 ![](../../Sub_Image.gif)
		 [GetRecordName](IMetaAttributeNamings.GetRecordName.htm)
		 Метод GetRecordName
		 возвращает наименование элемента базы данных временных рядов.


		 ![](../../Sub_Image.gif)
		 [Move](IMetaAttributeNamings.Move.htm)
		 Метод Move осуществляет
		 перемещение наименований атрибутов в коллекции по индексам.


		 ![](../../Sub_Image.gif)
		 [Remove](IMetaAttributeNamings.Remove.htm)
		 Метод Remove осуществляет
		 удаление наименования из коллекции.
