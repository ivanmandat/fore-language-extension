# IMetaAttributeNamings.Move


## Синтаксис


Move(IndexFrom: Integer; IndexTo: Integer);


## Параметры


IndexFrom. Индекс позиции наименования,
 которое необходимо переместить.


IndexTo. Индекс позиции, в
 которую необходимо переместить наименование.


## Описание


Метод Move осуществляет перемещение
 наименований атрибутов в коллекции по индексам.
