# IMetaAttributeNamings.Remove


## Синтаксис


Remove(Attribute: [IMetaAttributeNaming](../IMetaAttributeNaming/IMetaAttributeNaming.htm));


## Параметры


Attribute. Наименование, которое
 необходимо удалить.


## Описание


Метод Remove осуществляет удаление
 наименования из коллекции.


## Комментарии


Метод возвращает значение True,
 если удаление прошло успешно.
