# IMetaAttributeNamings.Add


## Синтаксис


Add(Attribute: [IMetaAttribute](../IMetaAttribute/IMetaAttribute.htm)):
 [IMetaAttributeNaming](../IMetaAttributeNaming/IMetaAttributeNaming.htm);


## Параметры


Attribute. Атрибут, наименование
 которого добавляется в коллекцию.


## Описание


Метод Add осуществляет добавление
 наименования атрибута в коллекцию.
