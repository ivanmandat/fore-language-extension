# IMetaAttributeNamings.GetNames


## Синтаксис


GetNames(Values: [IMatrixModel](ForeSys.chm::/Interface/IMatrixModel/IMatrixModel.htm);
 [Names: [IMatrixModel](ForeSys.chm::/Interface/IMatrixModel/IMatrixModel.htm)
 = Null; ][ValueIndex: Integer
 = 0]);


## Параметры


Values. Матрица значений.


Names. Матрица наименований.
 Необязательный параметр. По умолчанию данный параметр не определен.


ValueIndex. Индекс значения
 в матрице. Необязательный параметр. По умолчанию матрица значений обрабатывается
 с первого элемента.


## Описание


Метод GetNames позволяет получить
 наименование элементов базы данных временных рядов по заданным значениям.
