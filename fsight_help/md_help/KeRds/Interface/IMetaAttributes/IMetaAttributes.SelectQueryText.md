# IMetaAttributes.SelectQueryText


## Синтаксис


SelectQueryText: String;


## Описание


Свойство SelectQueryText определяет
 текст запроса, используемого для получения значений атрибутов.


## Комментарии


Свойство учитывается, если [IMetaAttributes.UseQuery](IMetaAttributes.UseQuery.htm) = True.
