# IMetaAttributes.UseQuery


## Синтаксис


UseQuery: Boolean;


## Описание


Свойство UseQuery определяет,
 использовать ли запрос для получения значений атрибутов.


## Комментарии


Допустимые значения:


	- True. Для получения
	 значений атрибутов используется запрос, заданный свойством [IMetaAttributes.SelectQueryText](IMetaAttributes.SelectQueryText.htm);


	- False. Значения атрибутов
	 загружаются из таблицы данных.
