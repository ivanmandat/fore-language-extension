# IMetaAttributes.KeepTableNativeNameOnMetabaseUpdate


## Синтаксис


		KeepTableNativeNameOnMetabaseUpdate: Boolean;


## Описание


Свойство KeepTableNativeNameOnMetabaseUpdate
 определяет признак использования сохранённого физического имени дочерней
 таблицы справочника НСИ при обновлении репозитория.


## Комментарии


Допустимые значения:


	- True. Сохранённое физическое
	 имя таблицы используется при обновлении;


	- False. Сохранённое физическое
	 имя таблицы не используется при обновлении. Значение по умолчанию.
