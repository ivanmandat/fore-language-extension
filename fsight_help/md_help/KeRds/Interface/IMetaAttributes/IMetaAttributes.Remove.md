# IMetaAttributes.Remove


## Синтаксис


Remove(Attribute: [IMetaAttribute](../IMetaAttribute/IMetaAttribute.htm);
 [RemoveField: Boolean = True]);


## Параметры


Attribute - атрибут, которой
 необходимо удалить из коллекции.


RemoveField - параметр, определяющий,
 удалять ли поле для хранения значений атрибута из таблицы. Необязательный
 параметр. Значение по умолчанию True
 - удалять поле из таблицы.


## Описание


Метод Remove осуществляет удаление
 атрибута из коллекции.
