# IMetaMemberChildren


## Описание


Интерфейс IMetaMemberChildren
 содержит свойства и методы для работы с потомками элемента иерархии.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [IsEmpty](IMetaMemberChildren.IsEmpty.htm)


		 Свойство IsEmpty возвращает
		 признак того, присутствуют ли элементы в данном объекте.

		 [Hierarchy](../IMetaHierarchyPosition/IMetaHierarchyPosition.Hierarchy.htm)

		 возвращает иерархию справочника.


		 ![](../../Property_Image.gif)
		 [Level](../IMetaHierarchyPosition/IMetaHierarchyPosition.Level.htm)


		 Свойство Level возвращает

		 ![](../../Property_Image.gif)
		 [LevelIndex](../IMetaHierarchyPosition/IMetaHierarchyPosition.LevelIndex.htm)
