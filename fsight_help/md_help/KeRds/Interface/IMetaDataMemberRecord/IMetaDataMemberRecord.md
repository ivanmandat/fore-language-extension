# IMetaDataMemberRecord


## Описание


Интерфейс IMetaDataMemberRecord
 содержит свойства для работы со сложной записью системной таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [IsEdited](IMetaDataMemberRecord.IsEdited.htm)


		 Свойство IsEdited возвращает
		 признак того, доступна ли запись для редактирования.


		 ![](../../Property_Image.gif)
		 [IsPermanent](IMetaDataMemberRecord.IsPermanent.htm)


		 Свойство IsPermanent
		 возвращает признак того, сохранены ли данные записи в базе данных.


		 ![](../../Property_Image.gif)
		 [Key](IMetaDataMemberRecord.Key.htm)


		 Свойство Key возвращает
		 ключ записи.


		 ![](../../Property_Image.gif)
		 [Members](IMetaDataMemberRecord.Members.htm)


		 Свойство Members возвращает
		 коллекцию, образующую данную запись.


## Методы


		 Имя свойства
		 Краткое описание

		 [SetMembersRecord](IMetaDataMemberRecord.SetMembersRecord.htm)
