# IRdsParamTypes.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IRdsParamType](../IRdsParamType/IRdsParamType.htm);


## Параметры


Key -
 ключ типа параметра, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает тип параметра, в случае удачного поиска. Ключ, по которому
 осуществляется поиск, передается посредством параметра Key.
