# IRdsParamTypes.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс
 типа параметра.


## Описание


Метод Remove осуществляет удаление
 типа параметра, индекс которого передается посредством параметра Index и возвращает True
 в случае удачного удаления.
