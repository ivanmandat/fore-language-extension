# IRdsParamTypes.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key -
 ключ типа параметра.


## Описание


Метод RemoveByKey осуществляет
 удаление типа параметра, ключ которого передается посредством параметра
 Key и возвращает True
 в случае удачного удаления.
