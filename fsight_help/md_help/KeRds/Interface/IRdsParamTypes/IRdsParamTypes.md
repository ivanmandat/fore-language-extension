# IRdsParamTypes


## Описание


Интерфейс IRdsParamTypes содержит
 свойства и методы коллекции типов параметров, содержащихся в определенной
 папке. Интерфейс является устаревшим.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IRdsParamTypes.Count.htm)


		 Свойство Count возвращает
		 количество типов параметров, содержащихся в данной папке.


		 ![](../../Property_Image.gif)
		 [Item](IRdsParamTypes.Item.htm)


		 Свойство Item возвращает
		 тип параметра.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IRdsParamTypes.Add.htm)


		 Метод Add осуществляет
		 добавление нового типа параметров.


		 ![](../../Sub_Image.gif)
		 [Clear](IRdsParamTypes.Clear.htm)


		 Метод Clear осуществляет
		 удаление всех типов параметров, содержащихся в данной папке.


		 ![](../../Sub_Image.gif)
		 [FindById](IRdsParamTypes.FindById.htm)


		 Метод FindById осуществляет
		 поиск типа параметра по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IRdsParamTypes.FindByKey.htm)


		 Метод FindByKey осуществляет
		 поиск типа параметра по ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](IRdsParamTypes.Remove.htm)


		 Метод Remove осуществляет
		 удаление типа параметра по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IRdsParamTypes.RemoveByKey.htm)


		 Метод RemoveByKey осуществляет
		 удаление типа параметра по ключу.
