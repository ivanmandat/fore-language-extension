# IRdsParamTypes.FindById


## Синтаксис


FindById(Id: String): [IRdsParamType](../IRdsParamType/IRdsParamType.htm);


## Параметры


Id - идентификатор
 типа параметра, по которому осуществляется поиск.


## Описание


Метод FindById осуществляет
 поиск и возвращает тип параметра, в случае удачного поиска. Идентификатор,
 по которому осуществляется поиск, передается посредством параметра Id.
