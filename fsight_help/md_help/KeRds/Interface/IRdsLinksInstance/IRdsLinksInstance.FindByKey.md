# IRdsLinksInstance.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IRdsLinkInstance](../IRdsLinkInstance/IRdsLinkInstance.htm);


## Параметры


Key - ключ связи, по которому
 осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает связь справочника НСИ в случае удачного поиска. Ключ
 связи, по которому осуществляется поиск, передается посредством параметра
 Key.
