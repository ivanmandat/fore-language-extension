# IRdsLinksInstance


## Описание


Интерфейс IRdsLinksInstance
 содержит свойства и методы для работы с коллекцией связей открытого справочника
 НСИ.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IRdsLinksInstance.Count.htm)
		 Свойство Count возвращает
		 количество связей справочника НСИ.


		 ![](../../Property_Image.gif)
		 [Item](IRdsLinksInstance.Item.htm)
		 Свойство Item возвращает
		 свойства связи справочника НСИ.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](IRdsLinksInstance.FindById.htm)
		 Метод FindById осуществляет
		 поиск связи справочника НСИ по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IRdsLinksInstance.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск связи справочника НСИ по ключу.
