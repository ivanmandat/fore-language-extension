# IRdsLinksInstance.FindById


## Синтаксис


FindById(Id: String): [IRdsLinkInstance](../IRdsLinkInstance/IRdsLinkInstance.htm);


## Параметры


Id - идентификатор связи, по
 которому осуществляется поиск.


## Описание


Метод FindById осуществляет
 поиск и возвращает связь справочника НСИ в случае удачного поиска. Идентификатор
 связи, по которому осуществляется поиск, передается посредством параметра
 Id.
