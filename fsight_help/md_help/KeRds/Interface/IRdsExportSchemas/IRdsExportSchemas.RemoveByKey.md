# IRdsExportSchemas.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key -
 ключ схемы экспорта справочника НСИ.


## Описание


Метод RemoveByKey осуществляет
 удаление схемы экспорта справочника НСИ, ключ которой передается посредством
 параметра Key, и возвращает True в случае удачного удаления.
