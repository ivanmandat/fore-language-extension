# IRdsExportSchemas.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index -
 индекс схемы экспорта справочника НСИ.


## Описание


Метод Remove осуществляет удаление
 схемы экспорта справочника НСИ, индекс которой передается посредством
 параметра Index, и возвращает
 True в случае удачного удаления.
