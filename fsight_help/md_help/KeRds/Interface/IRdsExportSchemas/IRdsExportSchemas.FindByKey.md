# IRdsExportSchemas.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IRdsExportSchema](../IRdsExportSchema/IRdsExportSchema.htm);


## Параметры


Key -
 ключ схемы экспорта справочника НСИ, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает схему экспорта в случае удачного поиска. Ключ схемы
 экспорта, по которому осуществляется поиск, передается посредством параметра
 Key.
