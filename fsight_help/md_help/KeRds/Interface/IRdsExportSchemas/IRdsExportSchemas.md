# IRdsExportSchemas


## Описание


Интерфейс IRdsExportSchemas
 содержит свойства и методы для работы с коллекцией схем экспорта справочника
 НСИ.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IRdsExportSchemas.Count.htm)


		 Свойство Count возвращает
		 количество схем экспорта справочника НСИ.


		 ![](../../Property_Image.gif)
		 [Item](IRdsExportSchemas.Item.htm)


		 Свойство Item возвращает
		 схему экспорта справочника НСИ.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IRdsExportSchemas.Add.htm)


		 Метод Add осуществляет
		 создание новой схемы экспорта.


		 ![](../../Sub_Image.gif)
		 [Clear](IRdsExportSchemas.Clear.htm)


		 Метод Clear осуществляет
		 удаление всех схем экспорта справочника НСИ.


		 ![](../../Sub_Image.gif)
		 [FindById](IRdsExportSchemas.FindById.htm)


		 Метод FindById осуществляет
		 поиск схемы экспорта по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IRdsExportSchemas.FindByKey.htm)


		 Метод FindByKey осуществляет
		 поиск схемы экспорта по ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](IRdsExportSchemas.Remove.htm)


		 Метод Remove осуществляет
		 удаление схемы экспорта по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IRdsExportSchemas.RemoveByKey.htm)


		 Метод RemoveByKey осуществляет
		 удаление схемы экспорта по ключу.
