# IRdsExportSchemas.FindById


## Синтаксис


FindById(Id: String): [IRdsExportSchema](../IRdsExportSchema/IRdsExportSchema.htm);


## Параметры


Id - идентификатор
 схемы экспорта справочника НСИ, по которому осуществляется поиск.


## Описание


Метод FindById осуществляет
 поиск и возвращает схему экспорта в случае удачного поиска. Идентификатор
 схемы экспорта, по которому осуществляется поиск, передается посредством
 параметра Id.
