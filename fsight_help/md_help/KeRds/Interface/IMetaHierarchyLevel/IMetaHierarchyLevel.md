# IMetaHierarchyLevel


## Описание


Интерфейс IMetaHierarchyLevel
 содержит свойства и методы для работы с уровнем иерархии базы данных временных
 рядов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attributes](IMetaHierarchyLevel.Attributes.htm)


		 Свойство Attributes
		 возвращает набор атрибутов уровня иерархии.


		 ![](../../Property_Image.gif)
		 [AttributeSetup](IMetaHierarchyLevel.AttributeSetup.htm)


		 Свойство AttributeSetup
		 возвращает настройки атрибута иерархии по индексу.


		 ![](../../Property_Image.gif)
		 [Flat](IMetaHierarchyLevel.Flat.htm)


		 Свойство Flat определяет,
		 содержит ли данный уровень иерархию элементов.


		 ![](../../Property_Image.gif)
		 [IncludeAll](IMetaHierarchyLevel.IncludeAll.htm)


		 Свойство IncludeAll
		 определяет, включает ли данный уровень все элементы.


		 ![](../../Property_Image.gif)
		 [IsLeaf](IMetaHierarchyLevel.IsLeaf.htm)


		 Свойство IsLeaf возвращает
		 признак того, является ли данный уровень листьевым.


		 ![](../../Property_Image.gif)
		 [LightWeight](IMetaHierarchyLevel.LightWeight.htm)


		 Свойство LightWeight
		 определяет, выполняется ли запрос на получение дочерних
		 элементов уровня иерархии.


		 ![](../../Property_Image.gif)
		 [Next](IMetaHierarchyLevel.Next.htm)


		 Свойство Next возвращает
		 следующий уровень в иерархии.
