# IMetaHierarchyLevel.IncludeAll


## Синтаксис


IncludeAll: Boolean;


## Описание


Свойство IncludeAll определяет,
 включает ли данный уровень все элементы.


## Комментарии


Допустимые значения:


	- True. Уровень включает
	 все элементы. Допускается использовать, если [IMetaHierarchy.WhereIsEmpty](../IMetaHierarchy/IMetaHierarchy.WhereIsEmpty.htm)
	 <> Undefined;


	- False. Уровень не включает все
	 элементы.
