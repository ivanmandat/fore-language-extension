# IRdsLink


## Описание


Интерфейс IRdsLink содержит
 свойства связи справочника НСИ с другим справочником.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attribute](IRdsLink.Attribute.htm)


		 Свойство Attribute
		 определяет связываемый атрибут данного справочника НСИ.


		 ![](../../Property_Image.gif)
		 [DeleteConstraint](IRdsLink.DeleteConstraint.htm)


		 Свойство DeleteConstraint
		 определяет поведение при попытке удаления элементов в связанном
		 справочнике.


		 ![](../../Property_Image.gif)
		 [Dictionary](IRdsLink.Dictionary.htm)


		 Свойство Dictionary
		 возвращает объект-родитель.


		 ![](../../Property_Image.gif)
		 [DisplayAttribute](IRdsLink.DisplayAttribute.htm)


		 Свойство DisplayAttribute
		 определяет атрибут связываемого справочника, значения которого
		 будут использоваться для отображения значений связываемого атрибута
		 текущего справочника.


		 ![](../../Property_Image.gif)
		 [DisplayFormat](IRdsLink.DisplayFormat.htm)


		 Свойство DisplayFormat
		 определяет формат отображения, который будет применяться для значений
		 атрибута [DisplayAttribute](IRdsLink.DisplayAttribute.htm).


		 ![](../../Property_Image.gif)
		 [Group](IRdsLink.Group.htm)


		 Свойство Group определяет
		 группу элементов, которые доступны для связи.


		 ![](../../Property_Image.gif)
		 [ParamsControl](IRdsLink.ParamsControl.htm)


		 Свойство ParamsControl
		 возвращает настройки привязки параметров.


		 ![](../../Property_Image.gif)
		 [Reference](IRdsLink.Reference.htm)


		 Свойство Reference
		 определяет атрибут справочника, с которым осуществляется связь.
