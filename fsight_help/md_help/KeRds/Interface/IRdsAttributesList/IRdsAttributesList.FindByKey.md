# IRdsAttributesList.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IRdsAttribute](../IRdsAttribute/IRdsAttribute.htm);


## Параметры


Key - ключ атрибута.


## Описание


Метод FindByKey осуществляет
 поиск атрибута в коллекции по ключу и возвращает его параметры в случае
 удачного поиска.


## Комментарии


Метод возвращает значение Null
 если атрибут не найден.
