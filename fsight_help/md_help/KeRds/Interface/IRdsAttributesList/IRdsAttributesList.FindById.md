# IRdsAttributesList.FindById


## Синтаксис


FindById(Id: String): [IRdsAttribute](../IRdsAttribute/IRdsAttribute.htm);


## Параметры


Id - идентификатор атрибута.


## Описание


Метод FindById осуществляет
 поиск атрибута в коллекции по идентификатору и возвращает его параметры
 в случае удачного поиска.


## Комментарии


Метод возвращает значение Null
 если атрибут не найден.
