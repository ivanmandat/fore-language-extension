# IRdsAttributesList.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс атрибута, который
 необходимо удалить.


## Описание


Метод Remove осуществляет удаление
 атрибута из коллекции по индексу и возвращает признак удачного удаления.


## Комментарии


Метод возвращает значение True,
 если удаление прошло успешно.
