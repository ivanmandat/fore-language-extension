# IRdsAttributesList.Attributes


## Синтаксис


Attributes: String;


## Описание


Свойство Attributes определяет
 список атрибутов, по которым будет производиться поиск в виде символьной
 строки.


## Комментарии


Идентификаторы атрибутов разделяются знаком «;». Данное свойство используется
 для смены списка атрибутов, по которым осуществляется поиск.
