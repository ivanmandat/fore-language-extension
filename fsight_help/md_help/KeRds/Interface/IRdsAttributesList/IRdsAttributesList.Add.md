# IRdsAttributesList.Add


## Синтаксис


Add(Value: [IRdsAttribute](../IRdsAttribute/IRdsAttribute.htm));


## Параметры


Value - атрибут, который необходимо
 добавить в коллекцию.


## Описание


Метод Add осуществляет добавление
 указанного атрибута в коллекцию.


## Комментарии


После добавления атрибута в коллекцию его значения будут использовать
 при поиске элементов.
