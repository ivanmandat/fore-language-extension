# IRdsAttributesList


Сборка: Rds;


## Описание


Интерфейс IRdsAttributesList
 содержит свойства и методы для работы с коллекцией атрибутов, по значениям
 которых будет производиться поиск элементов.


## Иерархия наследования


           IRdsAttributesList


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attributes](IRdsAttributesList.Attributes.htm)


		 Свойство Attributes
		 определяет список атрибутов, по которым будет производиться поиск
		 в виде символьной строки.


		 ![](../../Property_Image.gif)
		 [Count](IRdsAttributesList.Count.htm)


		 Свойство Count возвращает
		 количество атрибутов в коллекции.


		 ![](../../Property_Image.gif)
		 [Dictionary](IRdsAttributesList.Dictionary.htm)


		 Свойство Dictionary
		 определяет справочник, из атрибутов которого строится данная коллекция.


		 ![](../../Property_Image.gif)
		 [Item](IRdsAttributesList.Item.htm)


		 Свойство Item возвращает
		 параметры атрибута, добавленного в коллекцию.


		 ![](../../Property_Image.gif)
		 [Probe](IRdsAttributesList.Probe.htm)


		 Свойство Probe возвращает
		 коллекцию атрибутов реализуемую интерфейсом [IDimSimpleAttrProbe](KeDims.chm::/interface/IDimSimpleAttrProbe/IDimSimpleAttrProbe.htm).


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)

		 Метод Clear осуществляет

		 Метод FindById осуществляет

		 ![](../../Sub_Image.gif)
		 [FindByKey](IRdsAttributesList.FindByKey.htm)


		 Метод FindByKey осуществляет

		 [Remove](IRdsAttributesList.Remove.htm)


		 Метод Remove осуществляет
		 удаление атрибута из коллекции по индексу.

		 Метод RemoveByKey осуществляет
