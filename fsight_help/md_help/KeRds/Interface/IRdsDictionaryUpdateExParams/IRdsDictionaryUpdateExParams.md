# IRdsDictionaryUpdateExParams


## Описание


Интерфейс IRdsDictionaryUpdateExParams
 предназначен для определения параметров обновления справочника НСИ.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [PreserveAccessAttributes](IRdsDictionaryUpdateExParams.PreserveAccessAttributes.htm)


		 Свойство PreserveAccessAttributes
		 определяет, будут ли обновлены значения атрибутов, отвечающих
		 за права доступа к элементам справочника НСИ.
