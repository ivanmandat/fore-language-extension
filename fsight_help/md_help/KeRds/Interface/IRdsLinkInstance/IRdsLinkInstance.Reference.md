# IRdsLinkInstance.Reference


## Синтаксис


Reference: [IRdsAttributeInstance](../IRdsAttributeInstance/IRdsAttributeInstance.htm);


## Описание


Свойство Reference возвращает
 атрибут справочника, с которым осуществляется связь.


## Комментарии


В качестве данного атрибута должен быть установлен атрибут [Key](../IRdsAttributes/IRdsAttributes.Key.htm)
 справочника, с которым осуществляется связь.
