# IRdsLinkInstance


## Описание


Интерфейс IRdsLinkInstance содержит
 свойства для работы со связью открытого справочника НСИ.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attribute](IRdsLinkInstance.Attribute.htm)
		 Свойство Attribute
		 возвращает связываемый атрибут данного справочника НСИ.


		 ![](../../Property_Image.gif)
		 [IsControlAttribute](IRdsLinkInstance.IsControlAttribute.htm)
		 Свойство IsControlAttribute
		 проверяет, используется ли указанный атрибут для управления значениями
		 параметра связанного справочника.


		 ![](../../Property_Image.gif)
		 [ParamsControl](IRdsLinkInstance.ParamsControl.htm)
		 Свойство ParamsControl
		 возвращает объект, осуществляющий управление значениями параметров
		 связанного справочника путем изменения значений атрибутов текущего
		 справочника.


		 ![](../../Property_Image.gif)
		 [Reference](IRdsLinkInstance.Reference.htm)
		 Свойство Reference
		 возвращает атрибут справочника, с которым осуществляется связь.


		 ![](../../Property_Image.gif)
		 [ReferenceDictionary](IRdsLinkInstance.ReferenceDictionary.htm)
		 Свойство ReferenceDictionary
		 возвращает данные связанного справочника.
