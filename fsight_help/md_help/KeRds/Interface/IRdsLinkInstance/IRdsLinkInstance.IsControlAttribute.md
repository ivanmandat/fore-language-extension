# IRdsLinkInstance.IsControlAttribute


## Синтаксис


IsControlAttribute(Attribute: [IRdsAttributeInstance](../IRdsAttributeInstance/IRdsAttributeInstance.htm)):
 Boolean;


## Параметры


Attribute. Атрибут
 справочника.


## Описание


Свойство IsControlAttribute
 проверяет, используется ли указанный атрибут для управления значениями
 параметра связанного справочника.


## Комментарии


Свойство возвращает True, если
 атрибут Attribute используется
 для управления значениями параметра связанного справочника, в обратном
 случае - False.
