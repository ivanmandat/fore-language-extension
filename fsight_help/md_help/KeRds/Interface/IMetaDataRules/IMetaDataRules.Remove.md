# IMetaDataRules.Remove


## Синтаксис


Remove(Attribute: [IMetaDataRule](../IMetaDataRule/IMetaDataRule.htm));


## Параметры


Attribute. Правило, которое
 необходимо удалить.


## Описание


Метод Remove осуществляет удаление
 указанного правила из коллекции.
