# IMetaDataRules


## Описание


Интерфейс IMetaDataRules содержит
 свойства и методы для работы с коллекцией правил справочников базы данных
 временных рядов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMetaDataRules.Count.htm)


		 Свойство Count возвращает
		 количество правил в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMetaDataRules.Item.htm)


		 Свойство Item возвращает
		 параметры правила по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IMetaDataRules.Add.htm)


		 Метод Add осуществляет
		 добавление нового правила в коллекцию.


		 ![](../../Sub_Image.gif)
		 [FindById](IMetaDataRules.FindById.htm)


		 Метод FindById осуществляет
		 поиск правила по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IMetaDataRules.FindByKey.htm)


		 Метод FindByKey осуществляет
		 поиск правила по ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](IMetaDataRules.Remove.htm)


		 Метод Remove осуществляет
		 удаление указанного правила из коллекции.
