# IMetaDictionaryLookup.EmptyMarkAttribute


## Синтаксис


EmptyMarkAttribute: String;


## Описание


Свойство EmptyMarkAttribute
 определяет идентификатор атрибута, содержащего признак пустоты показателя.


## Комментарии


Для поиска в базе данных временных рядов с учетом признака пустоты показателей
 используйте свойство [IMetaDictionaryLookup.WhereIsEmpty](IMetaDictionaryLookup.WhereIsEmpty.htm).
