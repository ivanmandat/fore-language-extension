# IMetaAttributesCollection


## Описание


Интерфейс IMetaAttributesCollection
 содержит свойства и методы для работы с коллекцией атрибутов, входящих
 в индекс базы данных временных рядов.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMetaAttributesCollection.Count.htm)


		 Свойство Count возвращает
		 количество атрибутов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMetaAttributesCollection.Item.htm)


		 Свойство Item возвращает
		 атрибут из коллекции по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IMetaAttributesCollection.Add.htm)


		 Метод Add осуществляет
		 добавление атрибута в коллекцию.


		 ![](../../Sub_Image.gif)
		 [FindById](IMetaAttributesCollection.FindById.htm)


		 Метод FindById осуществляет
		 поиск атрибута в коллекции по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IMetaAttributesCollection.FindByKey.htm)


		 Метод FindByKey осуществляет
		 поиск атрибута в коллекции по ключу.


		 ![](../../Sub_Image.gif)
		 [Move](IMetaAttributesCollection.Move.htm)


		 Метод Move осуществляет
		 перемещение атрибута в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IMetaAttributesCollection.Remove.htm)


		 Метод Remove осуществляет
		 удаление указанного атрибута из коллекции.
