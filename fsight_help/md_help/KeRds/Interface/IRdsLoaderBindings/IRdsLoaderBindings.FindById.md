# IRdsLoaderBindings.FindById


## Синтаксис


FindById(Id: String): [IRdsBaseBinding](../IRdsBaseBinding/IRdsBaseBinding.htm);


## Параметры


Id. Идентификатор привязки.


## Описание


Метод FindById осуществляет
 поиск привязки по идентификатору, передаваемому посредством параметра
 Id.


## Комментарии


Метод возвращает параметры привязки в случае удачного поиска и Null, если привязка с указанным
 идентификатором отсутствует.
