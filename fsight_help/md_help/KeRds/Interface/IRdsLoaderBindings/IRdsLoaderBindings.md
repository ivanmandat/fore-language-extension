# IRdsLoaderBindings


Сборка: Rds;


## Описание


Интерфейс IRdsLoaderBindings
 содержит свойства и методы для работы с коллекцией привязок, используемых
 при создании справочников НСИ и загрузке в них данных.


## Иерархия наследования


IRdsLoaderBindings


## Комментарии


Коллекцию привязок можно получить в свойстве [IMetaRdsLoader.Bindings](../IMetaRdsLoader/IMetaRdsLoader.Bindings.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IRdsLoaderBindings.Count.htm)
		 Свойство Count возвращает
		 количество привязок в коллекции.


		 ![](../../Property_Image.gif)
		 [HasMultipleBindings](IRdsLoaderBindings.HasMultipleBindings.htm)
		 Свойство HasMultipleBindings
		 возвращает признак наличия привязок для атрибутов, имеющих множественные
		 значения.


		 ![](../../Property_Image.gif)
		 [Item](IRdsLoaderBindings.Item.htm)
		 Свойство Item возвращает
		 параметры указанной привязки.


		 ![](../../Property_Image.gif)
		 [KeyBinding](IRdsLoaderBindings.KeyBinding.htm)
		 Свойство KeyBinding
		 возвращает параметры привязки для системного атрибута - Ключ.


		 ![](../../Property_Image.gif)
		 [NameBinding](IRdsLoaderBindings.NameBinding.htm)
		 Свойство NameBinding
		 возвращает параметры привязки для системного атрибута - Наименование.


		 ![](../../Property_Image.gif)
		 [OrderBinding](IRdsLoaderBindings.OrderBinding.htm)
		 Свойство OrderBinding
		 возвращает параметры привязки для системного атрибута - Порядок.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IRdsLoaderBindings.Add.htm)
		 Метод Add создает привязку
		 с заданным типом.


		 ![](../../Sub_Image.gif)
		 [AddByAttribute](IRdsLoaderBindings.AddByAttribute.htm)
		 Метод AddByAttribute
		 создает привязку для указанного атрибута.


		 ![](../../Sub_Image.gif)
		 [Clear](IRdsLoaderBindings.Clear.htm)
		 Метод Clear удаляет
		 все привязки из коллекции.


		 ![](../../Sub_Image.gif)
		 [FindById](IRdsLoaderBindings.FindById.htm)
		 Метод FindById осуществляет
		 поиск привязки по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IRdsLoaderBindings.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск привязки по ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](IRdsLoaderBindings.Remove.htm)
		 Метод Remove удаляет
		 указанную привязку из коллекции и возвращает признак удачного
		 удаления.
