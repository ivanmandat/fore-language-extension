# IRdsLoaderBindings.HasMultipleBindings


## Синтаксис


HasMultipleBindings: Boolean;


## Описание


Свойство HasMultipleBindings
 возвращает признак наличия привязок для атрибутов, имеющих множественные
 значения.


## Комментарии


Свойство возвращает значение True,
 если в коллекции имеется привязка как минимум для одного атрибута, имеющего
 множественные значения, и False
 во всех остальных случаях.
