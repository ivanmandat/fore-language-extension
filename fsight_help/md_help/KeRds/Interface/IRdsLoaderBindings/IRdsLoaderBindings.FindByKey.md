# IRdsLoaderBindings.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IRdsBaseBinding](../IRdsBaseBinding/IRdsBaseBinding.htm);


## Параметры


Key. Ключ привязки.


## Описание


Метод FindByKey осуществляет
 поиск привязки по ключу, передаваемому посредством параметра Key.


## Комментарии


Метод возвращает параметры привязки в случае удачного поиска и Null, если привязка с указанным
 ключом отсутствует.
