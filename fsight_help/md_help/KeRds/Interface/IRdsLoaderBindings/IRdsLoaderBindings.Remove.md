# IRdsLoaderBindings.Remove


## Синтаксис


Remove(Binding: [IRdsBaseBinding](../IRdsBaseBinding/IRdsBaseBinding.htm)):
 Boolean;


## Параметры


Binding. Привязка, которую
 необходимо удалить из коллекции.


## Описание


Метод Remove удаляет указанную
 привязку из коллекции и возвращает признак удачного удаления.


## Комментарии


Метод возвращает значение True,
 если удаление привязки прошло удачно и False,
 если удаление не возможно.
