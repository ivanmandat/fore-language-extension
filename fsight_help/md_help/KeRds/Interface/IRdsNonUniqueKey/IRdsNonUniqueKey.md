# IRdsNonUniqueKey


Сборка: Rds;


## Описание


Интерфейс IRdsNonUniqueKey предназначен
 для работы с неуникальными ключами справочника НСИ.


## Иерархия наследования


           [INamedEntity](KeSom.chm::/Interface/INamedEntity/INamedEntity.htm)


           IRdsNonUniqueKey


## Комментарии


IRdsNonUniqueKey также используется
 для работы с неуникальными ключами табличного справочника НСИ.


Неуникальный ключ справочника НСИ можно использовать в качестве индекса
 альтернативного справочника ([IDimHierarchy.SourceIndex](KeDims.chm::/interface/IDimHierarchy/IDimHierarchy.SourceIndex.htm)).
