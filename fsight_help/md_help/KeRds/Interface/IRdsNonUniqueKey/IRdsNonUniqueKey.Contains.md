# IRdsNonUniqueKey.Contains


## Синтаксис


Contains(Attribute: [IRdsAttribute](../IRdsAttribute/IRdsAttribute.htm)):
 Boolean;


## Параметры


Attribute. Атрибут справочника
 НСИ, для которого необходимо определить вхождение в неуникальный ключ.


## Описание


Свойство Contains определяет,
 входит ли указанный атрибут в неуникальный ключ справочника НСИ.


## Комментарии


Допустимые значения:


	- True. Атрибут входит
	 в состав неуникального ключа;


	- False. Атрибут не входит
	 в состав неуникального ключа.
