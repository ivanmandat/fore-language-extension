# IRdsNonUniqueKey.Dictionary


## Синтаксис


Dictionary: [IRdsDictionary](../IRdsDictionary/IRdsDictionary.htm);


## Описание


Свойство Dictionary возвращает
 справочник НСИ, которому принадлежит неуникальный ключ.


## Комментарии


Для определения, входит ли указанный атрибут в неуникальный ключ справочника
 НСИ, используйте свойство [IRdsNonUniqueKey.Contains](IRdsNonUniqueKey.Contains.htm).
