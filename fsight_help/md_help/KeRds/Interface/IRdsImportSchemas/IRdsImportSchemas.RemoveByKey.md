# IRdsImportSchemas.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key - ключ схемы импорта справочника
 НСИ.


## Описание


Метод RemoveByKey осуществляет
 удаление схемы импорта справочника НСИ, ключ которой передается посредством
 параметра Key,
 и возвращает True в случае удачного
 удаления.
