# IRdsImportSchemas.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IRdsImportSchema](../IRdsImportSchema/IRdsImportSchema.htm);


## Параметры


Key -
 ключ схемы импорта справочника НСИ, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает схему импорта в случае удачного поиска. Ключ схемы
 импорта, по которому осуществляется поиск, передается посредством параметра
 Key.
