# IRdsImportSchemas.FindById


## Синтаксис


FindById(Id: String): [IRdsImportSchema](../IRdsImportSchema/IRdsImportSchema.htm);


## Параметры


Id - идентификатор схемы импорта
 справочника НСИ, по которому осуществляется поиск.


## Описание


Метод FindById осуществляет
 поиск и возвращает схему импорта в случае удачного поиска. Идентификатор
 схемы импорта, по которому осуществляется поиск, передается посредством
 параметра Id.
