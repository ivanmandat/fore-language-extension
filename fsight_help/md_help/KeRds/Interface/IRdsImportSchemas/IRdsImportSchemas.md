# IRdsImportSchemas


## Описание


Интерфейс IRdsImportSchemas содержит свойства и методы для работы с коллекцией схем импорта справочника НСИ.


## Свойства


 Имя свойства
 Краткое описание


 ![](../../Property_Image.gif)

[Count](IRdsImportSchemas.Count.htm)


 Свойство Count возвращает количество схем импорта справочника НСИ.


 ![](../../Property_Image.gif)

[Item](IRdsImportSchemas.Item.htm)


 Свойство Item возвращает схему импорта справочника НСИ.


## Методы


 Имя метода
 Краткое описание


 ![](../../Sub_Image.gif)

[Add](IRdsImportSchemas.Add.htm)


 Метод Add осуществляет создание новой схемы импорта.


 ![](../../Sub_Image.gif)

[Clear](IRdsImportSchemas.Clear.htm)


 Метод Clear осуществляет удаление всех схем импорта справочника НСИ.


 ![](../../Sub_Image.gif)

[FindById](IRdsImportSchemas.FindById.htm)


 Метод FindById осуществляет поиск схемы импорта по идентификатору.


 ![](../../Sub_Image.gif)

[FindByKey](IRdsImportSchemas.FindByKey.htm)


 Метод FindByKey осуществляет поиск схемы импорта по ключу.


 ![](../../Sub_Image.gif)

[Remove](IRdsImportSchemas.Remove.htm)


 Метод Remove осуществляет удаление схемы импорта по индексу.


 ![](../../Sub_Image.gif)

[RemoveByKey](IRdsImportSchemas.RemoveByKey.htm)


 Метод RemoveByKey осуществляет удаление схемы импорта по ключу.
