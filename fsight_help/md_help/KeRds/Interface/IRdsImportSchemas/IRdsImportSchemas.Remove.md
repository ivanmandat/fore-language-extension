# IRdsImportSchemas.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс схемы импорта
 справочника НСИ.


## Описание


Метод Remove осуществляет удаление
 схемы импорта справочника НСИ, индекс которой передается посредством параметра
 Index, и возвращает True
 в случае удачного удаления.
