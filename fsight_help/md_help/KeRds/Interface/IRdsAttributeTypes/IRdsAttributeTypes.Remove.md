# IRdsAttributeTypes.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс типа атрибута.


## Описание


Метод Remove осуществляет удаление
 типа атрибута, индекс которого передается посредством параметра Index и возвращает True,
 в случае удачного удаления.
