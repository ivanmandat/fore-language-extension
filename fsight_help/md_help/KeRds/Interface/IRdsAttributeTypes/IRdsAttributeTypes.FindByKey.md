# IRdsAttributeTypes.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IRdsAttributeType](../IRdsAttributeType/IRdsAttributeType.htm);


## Параметры


Key - ключ типа атрибута, по
 которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает тип атрибута, в случае удачного поиска. Ключ, по которому
 осуществляется поиск, передается посредством параметра Key.
