# IRdsAttributeTypes.FindById


## Синтаксис


FindById(Id: String): [IRdsAttributeType](../IRdsAttributeType/IRdsAttributeType.htm);


## Параметры


Id - идентификатор
 типа атрибута, по которому осуществляется поиск.


## Описание


Метод FindById осуществляет
 поиск и возвращает тип атрибута, в случае удачного поиска. Идентификатор,
 по которому осуществляется поиск, передается посредством параметра Id.
