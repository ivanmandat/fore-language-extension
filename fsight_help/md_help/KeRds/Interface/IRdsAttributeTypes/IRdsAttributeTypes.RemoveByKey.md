# IRdsAttributeTypes.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key - ключ типа атрибута.


## Описание


Метод RemoveByKey осуществляет
 удаление типа атрибута, ключ которого передается посредством параметра
 Key и возвращает True,
 в случае удачного удаления.
