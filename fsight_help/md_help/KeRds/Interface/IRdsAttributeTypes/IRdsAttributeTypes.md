# IRdsAttributeTypes


## Описание


Интерфейс IRdsAttributeTypes
 содержит свойства и методы коллекции типов атрибутов, содержащихся в определенной
 папке.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IRdsAttributeTypes.Count.htm)


		 Свойство Count возвращает
		 количество типов атрибутов, содержащихся в данной папке.


		 ![](../../Property_Image.gif)
		 [Item](IRdsAttributeTypes.Item.htm)


		 Свойство Item возвращает
		 тип атрибута.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IRdsAttributeTypes.Add.htm)


		 Метод Add осуществляет
		 добавление нового типа атрибутов.


		 ![](../../Sub_Image.gif)
		 [Clear](IRdsAttributeTypes.Clear.htm)


		 Метод Clear осуществляет
		 удаление всех типов атрибутов, содержащихся в данной папке.


		 ![](../../Sub_Image.gif)
		 [FindById](IRdsAttributeTypes.FindById.htm)


		 Метод FindById осуществляет
		 поиск типа атрибута по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IRdsAttributeTypes.FindByKey.htm)


		 Метод FindByKey осуществляет
		 поиск типа атрибута по ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](IRdsAttributeTypes.Remove.htm)


		 Метод Remove осуществляет
		 удаление типа атрибута по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IRdsAttributeTypes.RemoveByKey.htm)


		 Метод RemoveByKey осуществляет
		 удаление типа атрибута по ключу.
