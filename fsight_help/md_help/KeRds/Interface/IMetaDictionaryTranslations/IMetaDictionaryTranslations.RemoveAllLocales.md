# IMetaDictionaryTranslations.RemoveAllLocales


## Синтаксис


RemoveAllLocales;


## Описание


Метод RemoveAllLocales очищает
 коллекцию настроек, содержащую параметры перевода атрибутов базы данных
 временных рядов на различные языки.


## Комментарии


После вызова метода значение свойства [IMetaDictionaryTranslations.LocalesCount](IMetaDictionaryTranslations.LocalesCount.htm)
 становится равным нулю. При этом не удаляется язык, на котором была создана
 база данных временных рядов.
