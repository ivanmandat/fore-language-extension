# IMetaDictionaryTranslations.IsOn


## Синтаксис


IsOn: Boolean;


## Описание


Свойство IsOn определяет, переводить
 ли атрибуты базы данных временных рядов на различные языки.


## Комментарии


От значения данного свойства зависит, учитывается ли при работе с базой
 данных временных рядов коллекция настроек [IMetaDictionary.Translations](../IMetaDictionary/IMetaDictionary.Translations.htm).


Допустимые значения:


	- True. Атрибуты базы
	 данных могут быть переведены на различные языки;


	- False. Значение по умолчанию.
	 Атрибуты базы данных предназначены для работы на одном языке и не
	 требуют перевода.
