# IMetaDictionaryTranslations.LocalesCount


## Синтаксис


LocalesCount: Integer;


## Описание


Свойство LocalesCount
 возвращает количество языков, на которые переводятся атрибуты базы данных
 временных рядов.


## Комментарии


Свойство доступно только для чтения.


В свойстве не учитывается язык, на котором была создана база данных
 временных рядов.
