# IMetaDictionaryTranslations.RemoveLocale


## Синтаксис


RemoveLocale(Value: [LocaleCodeID](ForeSys.chm::/Enums/LocaleCodeID.htm));


## Параметры


Value.
 Уникальный идентификатор языка перевода.


## Описание


Метод RemoveLocale удаляет настройку
 для указанного языка из коллекции, содержащей параметры перевода атрибутов
 базы данных временных рядов на различные языки.


## Комментарии


В качестве параметра Value
 нельзя указывать значение [LocaleCodeID.None](ForeSys.chm::/Enums/LocaleCodeID.htm).
