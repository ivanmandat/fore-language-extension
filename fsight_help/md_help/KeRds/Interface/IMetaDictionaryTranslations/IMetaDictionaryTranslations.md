# IMetaDictionaryTranslations


## Описание


Интерфейс IMetaDictionaryTranslations
 предназначен для работы с коллекцией настроек, содержащих параметры перевода
 справочников базы данных временных рядов на различные языки.


## Комментарии


Коллекцию настроек можно получить в свойстве [IMetaDictionary.Translations](../IMetaDictionary/IMetaDictionary.Translations.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [IsOn](IMetaDictionaryTranslations.IsOn.htm)
		 Свойство IsOn определяет,
		 переводить ли атрибуты базы данных временных рядов на различные
		 языки.


		 ![](../../Property_Image.gif)
		 [Locale](IMetaDictionaryTranslations.Locale.htm)
		 Свойство Locale возвращает
		 уникальный числовой идентификатор языка, для которого предназначена
		 настройка с указанным индексом.


		 ![](../../Property_Image.gif)
		 [LocalesCount](IMetaDictionaryTranslations.LocalesCount.htm)
		 Свойство LocalesCount возвращает количество языков, на
		 которые переводятся атрибуты базы данных временных рядов.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AddLocale](IMetaDictionaryTranslations.AddLocale.htm)
		 Метод AddLocale добавляет
		 настройку для указанного языка в коллекцию.


		 ![](../../Sub_Image.gif)
		 [RemoveAllLocales](IMetaDictionaryTranslations.RemoveAllLocales.htm)
		 Метод RemoveAllLocales
		 очищает коллекцию настроек.


		 ![](../../Sub_Image.gif)
		 [RemoveLocale](IMetaDictionaryTranslations.RemoveLocale.htm)
		 Метод RemoveLocale
		 удаляет настройку для указанного языка из коллекции.
