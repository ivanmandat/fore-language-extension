# IMetaDictionaryTranslations.AddLocale


## Синтаксис


AddLocale(Value: [LocaleCodeID](ForeSys.chm::/Enums/LocaleCodeID.htm));


## Параметры


Value.
 Уникальный идентификатор языка перевода.


## Описание


Метод AddLocale добавляет настройку
 для указанного языка в коллекцию настроек, содержащую параметры перевода
 атрибутов базы данных временных рядов на различные языки.


## Комментарии


В качестве параметра Value
 нельзя указывать значение [LocaleCodeID.None](ForeSys.chm::/Enums/LocaleCodeID.htm).
