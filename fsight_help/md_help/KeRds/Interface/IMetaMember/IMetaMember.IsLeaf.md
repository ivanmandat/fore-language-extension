# IMetaMember.IsLeaf


## Синтаксис


IsLeaf: Boolean;


## Описание


Свойство IsLeaf возвращает признак
 того, является ли данный элемент листьевым. Листьевой элемент - это элемент,
 у которого отсутствуют элементы-потомки. True
 - элемент является листьевым.
