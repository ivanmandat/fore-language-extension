# IMetaAttributeNaming


## Описание


Интерфейс IMetaAttributeNaming
 содержит свойства для работы с наименованием атрибута.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attribute](IMetaAttributeNaming.Attribute.htm)
		 Свойство Attribute
		 возвращает атрибут, для которого задано наименование.


		 ![](../../Property_Image.gif)
		 [Dictionary](IMetaAttributeNaming.Dictionary.htm)
		 Свойство Dictionary
		 возвращает базу данных временных рядов, которой принадлежит атрибут.


		 ![](../../Property_Image.gif)
		 [MnemoPrefix](IMetaAttributeNaming.MnemoPrefix.htm)
		 Свойство MnemoPrefix
		 определяет префикс мнемоники.


		 ![](../../Property_Image.gif)
		 [MnemoSuffix](IMetaAttributeNaming.MnemoSuffix.htm)
		 Свойство MnemoSuffix
		 определяет суффикс мнемоники.


		 ![](../../Property_Image.gif)
		 [NamingFormat](IMetaAttributeNaming.NamingFormat.htm)
		 Свойство NamingFormat
		 определяет формат наименования атрибута.
