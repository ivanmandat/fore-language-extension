# IMetaHierarchyPosition


## Описание


Интерфейс IMetaHierarchyPosition
 содержит свойства для работы с позицией иерархии.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Hierarchy](IMetaHierarchyPosition.Hierarchy.htm)


		 Свойство Hierarchy
		 возвращает иерархию справочника.


		 ![](../../Property_Image.gif)
		 [Level](IMetaHierarchyPosition.Level.htm)


		 Свойство Level возвращает
		 уровень, которому принадлежит позиция иерархии.


		 ![](../../Property_Image.gif)
		 [LevelIndex](IMetaHierarchyPosition.LevelIndex.htm)


		 Свойство LevelIndex
		 возвращает индекс уровня, которому принадлежит позиция иерархии.


		 ![](../../Property_Image.gif)
		 [Offset](IMetaHierarchyPosition.Offset.htm)


		 Свойство Offset возвращает
		 число уровней от вершины иерархии до текущей позиции.


		 ![](../../Property_Image.gif)
		 [Parent](IMetaHierarchyPosition.Parent.htm)


		 Свойство Parent возвращает
		 объект-владелец позиции иерархии.
