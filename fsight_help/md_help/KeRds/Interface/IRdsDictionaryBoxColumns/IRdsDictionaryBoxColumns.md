# IRdsDictionaryBoxColumns


## Описание


Интерфейс IRdsDictionaryBoxColumns
 содержит свойства коллекции столбцов справочника НСИ, отображаемого в
 компоненте [RdsDictionaryBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/RdsDictionaryBox.htm).
