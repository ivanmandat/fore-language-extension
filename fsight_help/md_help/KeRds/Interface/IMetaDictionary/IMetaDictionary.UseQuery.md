# IMetaDictionary.UseQuery


## Синтаксис


UseQuery: Boolean;


## Описание


Свойство UseQuery определяет,
 использовать ли запрос для получения значений временных рядов.


## Комментарии


Допустимые значения:


	- True. Для получения
	 значений временных рядов используется запрос, заданный свойством [IMetaAttributes.SelectQueryText](../IMetaAttributes/IMetaAttributes.SelectQueryText.htm);


	- False. Значения временных
	 рядов загружаются из таблицы данных.
