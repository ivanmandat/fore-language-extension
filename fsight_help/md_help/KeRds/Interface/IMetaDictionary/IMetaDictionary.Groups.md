# IMetaDictionary.Groups


## Синтаксис


Groups: [IMetaAttributeGroups](../IMetaAttributeGroups/IMetaAttributeGroups.htm);


## Описание


Свойство Groups возвращает
 коллекцию групп атрибутов.


## Комментарии


Если в базе данных временных рядов создана хотя бы одна группа, то нельзя
 удалять, сбрасывать или менять структуру первичного индекса, возвращаемого
 свойством [IMetaIndex.Primary](../IMetaIndex/IMetaIndex.Primary.htm).
