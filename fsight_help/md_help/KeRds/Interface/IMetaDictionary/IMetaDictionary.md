# IMetaDictionary


Сборка: Rds;


## Описание


Интерфейс IMetaDictionary предназначен
 для работы с системным метасправочником базы данных временных рядов.


## Иерархия наследования


           IMetaDictionary


## Комментарии


Для получения подробной информации о системных метасправочниках базы
 данных временных рядов обратитесь к разделу «[Особенности
 хранения данных в базе данных временных рядов](KnowledgeBase.chm::/01_Fore/knowledgebase_kb000036.htm)».
