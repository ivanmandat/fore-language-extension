# IRdsAttributes.Hidden


## Синтаксис


Hidden: [IRdsAttribute](../IRdsAttribute/IRdsAttribute.htm);


## Описание


Свойство Hidden возвращает системный
 атрибут справочника, определяющего видимость элементов.


## Комментарии


Данный атрибут имеет логический тип данных, не может содержать пустые
 и множественные значения. По умолчанию является скрытым. Идентификатор
 атрибута - "HIDDEN". Данный атрибут доступен, если свойству
 [ManageVisibility](../IRdsDictionary/IRdsDictionary.ManageVisibility.htm)
 установлено значение True. Если
 в качестве значения данного атрибута установлено значение True
 - элемент является невидимым, иначе - видимым. По умолчанию для всех элементов
 установлено значение False.


В основном данный атрибут применяется в справочниках, учитывающих изменение
 элементов во времени. Изменяя значение данного атрибута, можно манипулировать
 отображением значений элементов в определенных периодах. Доступ к скрытым
 элементам возможен только по ключу, с помощью свойства [ItemElement](../IRdsDictionaryElements/IRdsDictionaryElements.ItemElement.htm).
