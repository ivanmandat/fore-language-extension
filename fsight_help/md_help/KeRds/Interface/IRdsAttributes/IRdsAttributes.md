# IRdsAttributes


## Описание


Интерфейс IRdsAttributes содержит
 свойства и методы для работы с коллекцией атрибутов справочника НСИ.

## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IRdsAttributes.Add.htm)
		 Метод Add создает пользовательский
		 атрибут.


		 ![](../../Sub_Image.gif)
		 [Clear](IRdsAttributes.Clear.htm)
		 Метод Clear осуществляет
		 удаление всех пользовательских атрибутов справочника НСИ.


		 ![](../../Sub_Image.gif)
		 [FindById](IRdsAttributes.FindById.htm)
		 Метод FindById осуществляет
		 поиск атрибута справочника НСИ по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IRdsAttributes.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск атрибута справочника НСИ по ключу.


		 ![](../../Sub_Image.gif)
		 [Move](IRdsAttributes.Move.htm)
		 Метод Move осуществляет
		 перемещение атрибутов в списке, изменяя тем самым порядок отображения
		 значений атрибутов в дереве элементов.


		 ![](../../Sub_Image.gif)
		 [Remove](IRdsAttributes.Remove.htm)
		 Метод Remove осуществляет
		 удаление атрибута по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IRdsAttributes.RemoveByKey.htm)
		 Метод RemoveByKey осуществляет
		 удаление атрибута по ключу.
