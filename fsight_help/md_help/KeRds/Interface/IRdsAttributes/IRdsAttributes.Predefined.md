# IRdsAttributes.Predefined


## Синтаксис


Predefined(tPredefined: [RdsPredefinedAttribute](../../Enums/RdsPredefinedAttribute.htm)):
 [IRdsAttribute](../IRdsAttribute/IRdsAttribute.htm);


## Параметры


tPredefined - назначение системного
 атрибута, который необходимо получить.


## Описание


Свойство Predefined возвращает
 системный атрибут справочника НСИ, назначение которого передается посредством
 параметра tPredefined.
