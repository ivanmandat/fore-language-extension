# IRdsAttributes.MandatoryAccess


## Синтаксис


MandatoryAccess: [IRdsAttribute](../IRdsAttribute/IRdsAttribute.htm);


## Описание


Свойство MandatoryAccess возвращает
 системный атрибут справочника, определяющий параметры мандатного доступа
 для элементов справочника НСИ.


## Комментарии


Атрибут доступен, если [IRdsDictionary.HasMandatoryAccess](../IRdsDictionary/IRdsDictionary.HasMandatoryAccess.htm) = True.


Атрибут имеет строковый тип данных, может содержать пустые значения.
 По умолчанию является скрытым. Идентификатор атрибута - «MANDATORYACCESS».
