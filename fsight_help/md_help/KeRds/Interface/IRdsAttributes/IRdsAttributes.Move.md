# IRdsAttributes.Move


## Синтаксис


Move(IndexFrom: Integer; IndexTo: Integer);


## Параметры


IndexFrom. Индекс атрибута,
 который необходимо переместить;


IndexTo. Индекс позиции, в
 которую необходимо переместить измерение.


## Описание


Метод Move перемещает атрибуты
 в списке, изменяя тем самым порядок отображения значений атрибутов в дереве
 элементов.
