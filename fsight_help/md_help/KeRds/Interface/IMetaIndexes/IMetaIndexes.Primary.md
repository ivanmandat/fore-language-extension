# IMetaIndexes.Primary


## Синтаксис


Primary: [IMetaIndex](../IMetaIndex/IMetaIndex.htm);


## Описание


Свойство Primary определяет
 первичный индекс.


## Комментарии


Атрибуты, образующие первичный индекс, могут входить в состав групп
 атрибутов. Для работы с группами атрибутов используйте интерфейс [IMetaAttributeGroup](../IMetaAttributeGroup/IMetaAttributeGroup.htm).
