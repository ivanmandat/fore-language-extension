# IMetaIndexes


## Описание


Интерфейс IMetaIndexes содержит
 свойства и методы для работы с коллекцией индексов системной таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMetaIndexes.Count.htm)


		 Свойство Count возвращает
		 количество индексов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMetaIndexes.Item.htm)


		 Свойство Item возвращает
		 элемент из коллекции по индексу.


		 ![](../../Property_Image.gif)
		 [Primary](IMetaIndexes.Primary.htm)


		 Свойство Primary определяет
		 первичный индекс.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IMetaIndexes.Add.htm)


		 Метод Add осуществляет
		 добавление индекса в коллекцию.


		 ![](../../Sub_Image.gif)
		 [FindById](IMetaIndexes.FindById.htm)


		 Метод FindById осуществляет
		 поиск индекса в коллекции по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IMetaIndexes.FindByKey.htm)


		 Метод FindByKey осуществляет
		 поиск индекса в коллекции по ключу.


		 ![](../../Sub_Image.gif)
		 [Move](IMetaIndexes.Move.htm)


		 Метод Move осуществляет
		 перемещение индекса в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IMetaIndexes.Remove.htm)


		 Метод Remove осуществляет
		 удаление указанного индекса из коллекции.
