# IRdsPredefinedBinding.PredefinedType


## Синтаксис


PredefinedType: [RdsPredefinedAttribute](../../Enums/RdsPredefinedAttribute.htm);


## Описание


Свойство PredefinedType определяет
 системный атрибут справочника, которому соответствует данная привязка.


## Комментарии


Нельзя задавать следующие значения: None, Version, InDate, OutDate,
 GUID, ReadAccess, WriteAccess, DeleteAccess, AccessAccess, SourceDictionary,
 SourceElement.
