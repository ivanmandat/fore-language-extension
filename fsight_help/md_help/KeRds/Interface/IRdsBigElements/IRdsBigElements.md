# IRdsBigElements


Сборка: Rds;


## Описание


Интерфейс IRdsBigElements используется
 для работы со справочниками, поддерживающими большое количество элементов.


## Иерархия наследования


IRdsBigElements


## Комментарии


Свойство [IRdsDictionary.IsBigDictionary](../IRdsDictionary/IRdsDictionary.IsBigDictionary.htm),
 определяет поддерживает ли справочник работу с большим количеством элементов.
