# IRdsDictionaryTranslations.IsOn


## Синтаксис


IsOn: Boolean;


## Описание


Свойство IsOn определяет, переводить
 ли справочник на различные языки.


## Комментарии


От значения данного свойства зависит: учитывается ли при работе со справочником
 НСИ коллекция настоек [IRdsDictionary.Translations](../IRdsDictionary/IRdsDictionary.Translations.htm).


True - справочник может быть
 переведен на различные языки; False
 - справочник предназначен для работы на одном языке и не требует перевода.
 Значение по умолчанию - False.
