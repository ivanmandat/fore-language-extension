# IRdsDictionaryTranslations.DefaultLocale


## Синтаксис


DefaultLocale: [LocaleCodeID](ForeSys.chm::/Enums/LocaleCodeID.htm);


## Описание


Свойство DefaultLocale возвращает
 язык по умолчанию для справочника.


## Комментарии


Если перевод справочника для текущего языка репозитория не предусмотрен,
 то справочник отображается на языке, возвращаемом данным свойством.
