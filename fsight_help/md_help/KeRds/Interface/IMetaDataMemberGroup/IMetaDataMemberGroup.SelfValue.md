# IMetaDataMemberGroup.SelfValue


## Синтаксис


SelfValue: Variant;


## Описание


Свойство доступно только для чтения.


Свойство SelfValue возвращает
 значение группового атрибута.


## Комментарии


При изменении значения группового атрибута для одного показателя значение
 данного атрибута для всех остальных показателей, входящих в группу, будет
 изменено автоматически.
