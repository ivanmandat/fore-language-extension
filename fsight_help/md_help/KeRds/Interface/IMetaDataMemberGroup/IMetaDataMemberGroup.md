# IMetaDataMemberGroup


Сборка: Rds;


## Описание


Интерфейс IMetaDataMemberGroup
 предназначен для работы со значением группового атрибута.


## Иерархия наследования


           IMetaDataMemberGroup


## Комментарии


Если атрибут имеет группу, то его значение, реализованное интерфейсом
 [IMetaDataMember](../IMetaDataMember/IMetaDataMember.htm),
 можно привести к интерфейсу IMetaDataMemberGroup.
