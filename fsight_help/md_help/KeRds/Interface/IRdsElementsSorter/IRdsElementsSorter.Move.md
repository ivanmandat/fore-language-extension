# IRdsElementsSorter.Move


## Синтаксис


Move: (SourceKey: Integer; TargetKey: Integer; MoveType:
 [RdsMoveType](../../Enums/RdsMoveType.htm));


## Параметры


SourceKey. Ключ
 элемента, который будет перемещен.


TargetKey. Ключ
 элемента, относительно которого будет произведено перемещение.


MoveType. Позиция,
 в которую перемещается элемент справочника НСИ.


## Описание


Метод Move осуществляет перестановку
 элемента, передаваемого посредством параметра SourceKey,
 в указанную позицию относительно элемента, передаваемого посредством параметра
 TargetKey.
