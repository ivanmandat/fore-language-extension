# IRdsElementsSorter.Parent


## Синтаксис


Parent: Integer;


## Описание


Свойство Parent возвращает ключ
 элемента справочника НСИ, для которого создан объект, осуществляющий сортировку.


## Комментарии


При создании объекта, осуществляющего сортировку всего справочника,
 свойство вернет значение «-2»,
 так как это значение является ключом корневой вершины.
