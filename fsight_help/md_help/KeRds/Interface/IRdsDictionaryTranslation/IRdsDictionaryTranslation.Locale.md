# IRdsDictionaryTranslation.Locale


## Синтаксис


Locale: [LocaleCodeID](ForeSys.chm::/Enums/LocaleCodeID.htm);


## Описание


Свойство Locale возвращает уникальный
 идентификатор языка перевода.


## Комментарии


Свойство возвращает значение, определяемое перечислением [LocaleCodeID](ForeSys.chm::/Enums/LocaleCodeID.htm).
 Locale не может вернуть значение
 LocaleCodeID.None.
