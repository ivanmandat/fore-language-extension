# IRdsDictionaryTranslation.Name


## Синтаксис


Name: String;


## Описание


Свойство Name определяет
 наименование справочника на настраиваемом языке.


## Комментарии


Наименование, установленное свойством Name,
 будет отображаться для справочника, если свойство [IRdsDictionaryTranslations.IsOn](../IRdsDictionaryTranslations/IRdsDictionaryTranslations.IsOn.htm)
 установлено в True.


## Пример


Использование свойства приведено в примере для [IRdsDictionaryTranslations.Count](../IRdsDictionaryTranslations/IRdsDictionaryTranslations.Count.htm).
