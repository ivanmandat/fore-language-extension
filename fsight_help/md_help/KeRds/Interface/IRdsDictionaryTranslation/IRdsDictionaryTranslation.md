# IRdsDictionaryTranslation


## Описание


Интерфейс IRdsDictionaryTranslation
 предназначен для работы с параметрами перевода справочника НСИ на определенный
 язык.


## Комментарии


Параметры перевода включают:


	- уникальный идентификатор настраиваемого языка перевода;


	- признак того, является ли настраиваемый язык языком по умолчанию
	 для справочника;


	- наименование справочника на настраиваемом языке.


Изначально в справочнике НСИ содержится только настройка для языка,
 установленного по умолчанию для репозитория пользователя.
