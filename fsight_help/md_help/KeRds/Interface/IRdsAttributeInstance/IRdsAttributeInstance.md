# IRdsAttributeInstance


## Описание


Интерфейс IRdsAttributeInstance
 содержит свойства для работы с атрибутом открытого справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attribute](IRdsAttributeInstance.Attribute.htm)
		 Свойство Attribute
		 возвращает свойства атрибута.


		 ![](../../Property_Image.gif)
		 [HasReadAccess](IRdsAttributeInstance.HasReadAccess.htm)
		 Свойство HasReadAccess
		 возвращает признак наличия прав у текущего пользователя на чтение
		 значений элементов по данному атрибуту.


		 ![](../../Property_Image.gif)
		 [HasWriteAccess](IRdsAttributeInstance.HasWriteAccess.htm)
		 Свойство HasWriteAccess
		 возвращает признак наличия прав у текущего пользователя на изменение
		 значений элементов по данному атрибуту.


		 ![](../../Property_Image.gif)
		 [IsControl](IRdsAttributeInstance.IsControl.htm)
		 Свойство IsControl
		 возвращает True, если
		 данный атрибут используется для управления значениями параметра
		 связанного справочника.


		 ![](../../Property_Image.gif)
		 [Link](IRdsAttributeInstance.Link.htm)
		 Свойство Link возвращает
		 связь с другим справочником, в которой используется данный атрибут.
