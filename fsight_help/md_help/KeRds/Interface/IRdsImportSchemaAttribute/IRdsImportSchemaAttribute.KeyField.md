# IRdsImportSchemaAttribute.KeyField


## Синтаксис


KeyField: String;


## Описание


Свойство KeyField определяет наименование поля таблицы [DetailTable](IRdsImportSchemaAttribute.DetailTable.htm), содержащего ключи элементов, для которых определены множественные значения атрибута [Attribute](IRdsImportSchemaAttribute.Attribute.htm).
