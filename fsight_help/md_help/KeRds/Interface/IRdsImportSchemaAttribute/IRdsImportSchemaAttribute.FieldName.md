# IRdsImportSchemaAttribute.FieldName


## Синтаксис


FieldName: String;


## Описание


Свойство FieldName определяет
 наименование поля таблицы, с которым связывается атрибут справочника НСИ.


## Комментарии


Если атрибут имеет множественные значения, в данном свойстве указывается
 поле таблицы [TableName](../IRdsImportSchema/IRdsImportSchema.TableName.htm)
 с которым осуществляется связь поля [KeyField](IRdsImportSchemaAttribute.KeyField.htm)
 из таблицы [DetailTable](IRdsImportSchemaAttribute.DetailTable.htm).
