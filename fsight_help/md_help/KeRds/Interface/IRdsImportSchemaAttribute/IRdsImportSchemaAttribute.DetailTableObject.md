# IRdsImportSchemaAttribute.DetailTableObject


## Синтаксис


DetailTableObject: [IMetabaseObjectDescriptor](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm);


## Описание


Свойство DetailTableObject определяет
 таблицу репозитория, из которой импортируются данные множественных значений
 атрибута справочника.


## Комментарии


Свойство является обязательным. Также для корректного импорта необходимо
 задать свойство [IRdsImportSchemaAttribute.DetailTable](IRdsImportSchemaAttribute.DetailTable.htm).
