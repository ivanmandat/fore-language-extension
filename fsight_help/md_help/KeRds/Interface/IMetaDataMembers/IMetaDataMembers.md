# IMetaDataMembers


## Описание


Интерфейс IMetaDataMembers содержит
 свойства и методы для работы с коллекцией записей системной таблицы без
 возможности добавления и удаления.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IMetaDataMembers.Count.htm)


		 Свойство Count возвращает
		 количество записей в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IMetaDataMembers.Item.htm)


		 Свойство Item возвращает
		 запись из коллекции по индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](IMetaDataMembers.FindById.htm)


		 Метод FindById осуществляет
		 поиск записи в коллекции по идентификатору атрибута.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IMetaDataMembers.FindByKey.htm)


		 Метод FindByKey осуществляет
		 поиск записи в коллекции по ключу.


		 ![](../../Sub_Image.gif)
		 [FindByKind](IMetaDataMembers.FindByKind.htm)


		 Метод FindByKind осуществляет
		 поиск записи в коллекции по типу атрибута.


		 ![](../../Sub_Image.gif)
		 [FindByKindBasis](IMetaDataMembers.FindByKindBasis.htm)


		 Метод FindByKindBasis
		 осуществляет поиск записи в коллекции по типу атрибута и базису.


		 ![](../../Sub_Image.gif)
		 [FindByTag](IMetaDataMembers.FindByTag.htm)


		 Метод FindByTag осуществляет
		 поиск записи в коллекции по виду атрибута.


		 ![](../../Sub_Image.gif)
		 [FindByTagBasis](IMetaDataMembers.FindByTagBasis.htm)


		 Метод FindByTagBasis
		 осуществляет поиск записи в коллекции по виду атрибута и базису.
