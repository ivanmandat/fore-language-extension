# IMetaIndex


Сборка: Rds;


## Описание


Интерфейс IMetaIndex содержит
 свойства и методы для работы с индексом системной таблицы.


## Иерархия наследования


           [INamedEntity](KeSom.chm::/Interface/INamedEntity/INamedEntity.htm)


           IMetaIndex


## Комментарии


Для работы со всеми индексами системной таблицы используйте интерфейс
 [IMetaIndexes](../IMetaIndexes/IMetaIndexes.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attributes](IMetaIndex.Attributes.htm)


		 Свойство Attributes
		 возвращает коллекцию атрибутов, входящих в индекс.


		 ![](../../Property_Image.gif)
		 [AttributesSet](IMetaIndex.AttributesSet.htm)


		 Свойство AttributesSet
		 определяет набор атрибутов, образующих индекс.


		 ![](../../Property_Image.gif)
		 [Enabled](IMetaIndex.Enabled.htm)


		 Свойство Enabled определяет,
		 включен ли индекс.


		 ![](../../Property_Image.gif)
		 [Primary](IMetaIndex.Primary.htm)


		 Свойство Primary определяет,
		 является ли индекс первичным.


		 ![](../../Property_Image.gif)
		 [Unique](IMetaIndex.Unique.htm)


		 Свойство Unique определяет,
		 является ли индекс уникальным.
