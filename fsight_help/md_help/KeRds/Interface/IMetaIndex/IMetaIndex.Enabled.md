# IMetaIndex.Enabled


## Синтаксис


Enabled: Boolean;


## Описание


Свойство Enabled определяет,
 включен ли индекс.


## Комментарии


Допустимые значения:


	- True. Индекс включен;


	- False. Индекс отключен
	 и физически не создается.


Данное свойство можно использовать для создания неуникальных индексов
 в дочерней таблице табличного справочника НСИ. Для этого:


	- Создайте или получите существующий неуникальный ключ в табличном
	 справочнике НСИ.


	- Приведите полученный ключ к интерфейсу IMetaIndex.


	- Установите значение свойства Enabled
	 в True.


	- Сохраните изменения.


В результате в дочерней таблице табличного справочника НСИ будет создан
 неуникальный индекс на основе неуникального ключа из табличного справочника
 НСИ. Данный индекс позволят быстрее выполнять запросы.


Примечание.
 Если неуникальный ключ в табличном справочнике НСИ содержит заимствованные
 атрибуты или атрибуты с множественными значениями, то создание неуникального
 индекса на его основе невозможно.


Для получения примера обратитесь к разделу «[Создание
 неуникального индекса в таблице табличного справочника НСИ](../../Samples/KeRds_Sample11.htm)».
