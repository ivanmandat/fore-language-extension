# IMetaIndex.AttributesSet


## Синтаксис


AttributesSet: [IMetaAttributesSet](../IMetaAttributesSet/IMetaAttributesSet.htm);


## Описание


Свойство AttributesSet определяет
 набор атрибутов, образующих индекс.


## Комментарии


Признак того, является ли индекс уникальным возвращает свойство [IMetaIndex.Unique](IMetaIndex.Unique.htm);
 признак того, является ли индекс первичным - свойство [IMetaIndex.Primary](IMetaIndex.Primary.htm).
