# IMetaDataMemberReference


## Описание


Интерфейс IMetaDataMemberReference содержит свойства для работы со ссылкой на запись системной таблицы.


## Свойства, унаследованные от [IMetaDataMember](../IMetaDataMember/IMetaDataMember.htm)


 Имя свойства
 Краткое описание


 ![](../../Property_Image.gif)

[Attribute](../IMetaDataMember/IMetaDataMember.Attribute.htm)


 Свойство Attribute возвращает атрибут записи.


 ![](../../Property_Image.gif)

[Modified](../IMetaDataMember/IMetaDataMember.Modified.htm)


 Свойство Modified возвращает признак того, была ли запись изменена.


 ![](../../Property_Image.gif)

[UnmodifiedValue](../IMetaDataMember/IMetaDataMember.UnmodifiedValue.htm)


 Свойство UnmodifiedValue возвращает значение записи до изменения.


 ![](../../Property_Image.gif)

[Value](../IMetaDataMember/IMetaDataMember.Value.htm)


 Свойство Value определяет значение записи.
