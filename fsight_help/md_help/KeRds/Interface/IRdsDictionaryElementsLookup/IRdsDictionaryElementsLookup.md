# IRdsDictionaryElementsLookup


## Описание


Интерфейс IRdsDictionaryElementsLookup
 содержит свойства и методы для поиска элементов в справочнике НСИ.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AttributeOptions](IRdsDictionaryElementsLookup.AttributeOptions.htm)


		 Свойство AttributeOptions
		 определяет настройки поиска по указанному атрибуту.


		 ![](../../Property_Image.gif)
		 [Attributes](IRdsDictionaryElementsLookup.Attributes.htm)


		 Свойство Attributes
		 определяет список атрибутов, по которым будет производиться поиск.


		 ![](../../Property_Image.gif)
		 [Data](IRdsDictionaryElementsLookup.Data.htm)


		 Свойство Data возвращает
		 объект, содержащий искомые данные элементов справочника НСИ.


		 ![](../../Property_Image.gif)
		 [Options](IRdsDictionaryElementsLookup.Options.htm)


		 Свойство Options определяет
		 общие настройки поиска, применяемые по всем атрибутам.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Lookup](IRdsDictionaryElementsLookup.Lookup.htm)


		 Метод Lookup осуществляет
		 поиск и возвращает массив, содержащий ключи элементов, удовлетворяющих
		 условиям поиска.


		 ![](../../Sub_Image.gif)
		 [LookupList](IRdsDictionaryElementsLookup.LookupList.htm)


		 Метод LookupList осуществляет
		 поиск и возвращает массив, содержащий элементы, удовлетворяющие
		 условиям поиска.
