# IRdsParam


## Описание


Интерфейс IRdsParam содержит
 свойства параметра справочника НСИ.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attribute](IRdsParam.Attribute.htm)


		 Свойство Attribute
		 определяет атрибут справочника, значениями которого управляет
		 данный параметр.


		 ![](../../Property_Image.gif)
		 [Dictionary](IRdsParam.Dictionary.htm)


		 Свойство Dictionary
		 возвращает объект-родитель.


		 ![](../../Property_Image.gif)
		 [Predefined](IRdsParam.Predefined.htm)


		 Свойство Predefined
		 возвращает значение True,
		 если параметр является системным.


		 ![](../../Property_Image.gif)
		 [PredefinedType](IRdsParam.PredefinedType.htm)


		 Свойство PredefinedType
		 возвращает назначение системного параметра.


		 ![](../../Property_Image.gif)
		 [SkipOnNull](IRdsParam.SkipOnNull.htm)


		 Свойство SkipOnNull
		 определяет возможность отключения фильтрации при пустом значении
		 параметра.


		 ![](../../Property_Image.gif)
		 [Type](IRdsParam.Type.htm)


		 Свойство Type определяет
		 тип параметра. Свойство является устаревшим.
