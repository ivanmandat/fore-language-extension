# IMetaAttributeTranslations


## Описание


Интерфейс IMetaAttributeTranslations
 содержит параметры перевода атрибута базы данных временных рядов на различные
 языки.


## Комментарии


Параметры перевода атрибутов можно получить в свойстве [IMetaAttribute.Translations](../IMetaAttribute/IMetaAttribute.Translations.htm).
