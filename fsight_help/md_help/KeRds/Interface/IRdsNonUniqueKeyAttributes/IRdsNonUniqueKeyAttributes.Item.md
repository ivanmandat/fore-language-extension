# IRdsNonUniqueKeyAttributes.Item


## Синтаксис


		Item(Index: Integer): [IRdsNonUniqueKeyAttribute](../IRdsNonUniqueKeyAttribute/IRdsNonUniqueKeyAttribute.htm);


## Параметры


Index. Индекс атрибута.


## Описание


Свойство Item возвращает атрибут
 неуникального ключа справочника НСИ.


## Комментарии


Для получения количества атрибутов неуникального ключа справочника НСИ
 в коллекции используйте [IRdsNonUniqueKeyAttributes.Count](IRdsNonUniqueKeyAttributes.Count.htm).
