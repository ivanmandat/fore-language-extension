# IRdsNonUniqueKeyAttributes


Сборка: Rds;


## Описание


Интерфейс IRdsNonUniqueKeyAttributes
 содержит свойства и методы для работы с коллекцией атрибутов неуникального
 ключа справочника НСИ.


## Иерархия наследования


IRdsNonUniqueKeyAttributes


## Комментарии


Для работы с атрибутом неуникального ключа справочника НСИ используйте
 интерфейс [IRdsNonUniqueKeyAttribute](../IRdsNonUniqueKeyAttribute/IRdsNonUniqueKeyAttribute.htm).
