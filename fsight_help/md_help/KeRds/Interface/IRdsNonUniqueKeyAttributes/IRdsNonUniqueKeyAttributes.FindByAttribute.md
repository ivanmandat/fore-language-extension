# IRdsNonUniqueKeyAttributes.FindByAttribute


## Синтаксис


		FindByAttribute(Attribute: [IRdsAttribute](../IRdsAttribute/IRdsAttribute.htm)):
		 Integer;


## Параметры


Attribute. Атрибут справочника,
 по которому осуществляется поиск.


## Описание


Метод FindByAttribute осуществляет
 поиск и возвращает атрибут неуникального ключа справочника НСИ в случае
 удачного поиска.


## Комментарии


Для работы с атрибутом уникального ключа используйте интерфейс [IRdsUniqueKeyAttribute](../IRdsUniqueKeyAttribute/IRdsUniqueKeyAttribute.htm).
