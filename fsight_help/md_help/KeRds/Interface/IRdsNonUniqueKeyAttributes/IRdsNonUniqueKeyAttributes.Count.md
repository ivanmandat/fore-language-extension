# IRdsNonUniqueKeyAttributes.Count


## Синтаксис


		Count: Integer;


## Описание


Свойство Count возвращает количество
 атрибутов неуникального ключа справочника НСИ.


## Комментарии


Для получения атрибута неуникального ключа справочника НСИ по его индексу
 используйте [IRdsNonUniqueKeyAttributes.Item](IRdsNonUniqueKeyAttributes.Item.htm).
