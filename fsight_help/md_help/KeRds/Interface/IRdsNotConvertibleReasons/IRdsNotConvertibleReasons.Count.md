# IRdsNotConvertibleReasons.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 причин, по которым справочник НСИ не может быть преобразован в табличный
 справочник НСИ.


## Комментарии


Для получения причины, по которой справочник НСИ не может быть преобразован
 в табличный справочник НСИ, по её индексу используйте свойство [IRdsNotConvertibleReasons.Item](IRdsNotConvertibleReasons.Item.htm).
