# IRdsNotConvertibleReasons.Item


## Синтаксис


Item(Index: Integer): [IRdsNotConvertibleReason](../IRdsNotConvertibleReason/IRdsNotConvertibleReason.htm);


## Параметры


Index. Индекс причины.


## Описание


Свойство Item возвращает причину,
 по которой справочник НСИ не может быть преобразован в табличный справочник
 НСИ, по её индексу.


## Комментарии


Для получения количества причин, по которым справочник НСИ не может
 быть преобразован в табличный справочник НСИ, используйте свойство [IRdsNotConvertibleReasons.Count](IRdsNotConvertibleReasons.Count.htm).
