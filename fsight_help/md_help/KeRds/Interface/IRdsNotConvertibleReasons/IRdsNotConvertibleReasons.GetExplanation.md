# IRdsNotConvertibleReasons.GetExplanation


## Синтаксис


GetExplanation: Array;


## Описание


Метод GetExplanation позволяет
 получить объяснение всех причин, по которым справочник НСИ не может быть
 преобразован в табличный справочник НСИ.


## Комментарии


Для получения количества причин, по которым справочник НСИ не может
 быть преобразован в табличный справочник НСИ, используйте свойство [IRdsNotConvertibleReasons.Count](IRdsNotConvertibleReasons.Count.htm).
