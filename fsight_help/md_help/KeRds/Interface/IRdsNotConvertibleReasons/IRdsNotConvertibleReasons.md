# IRdsNotConvertibleReasons


Сборка: Rds;


## Описание


Интерфейс IRdsNotConvertibleReasons
 предназначен для работы с коллекцией причин, по которым справочник НСИ
 не может быть преобразован в табличный справочник НСИ.


## Иерархия наследования


           IRdsNotConvertibleReasons


## Комментарии


Для работы с причиной, по которой справочник НСИ не может быть преобразован
 в табличный справочник НСИ, используйте свойства и методы интерфейса [IRdsNotConvertibleReason](../IRdsNotConvertibleReason/IRdsNotConvertibleReason.htm).
 Ограничения и особенности преобразования справочников перечислены в интерфейсе
 [IRdsDictionaryConverter](../IRdsDictionaryConverter/IRdsDictionaryConverter.htm).
