# IMetaAttributesSet.FindByValuesObject


## Синтаксис


FindByValuesObject(ObjectKey: Integer): [IMetaAttribute](../IMetaAttribute/IMetaAttribute.htm);


## Параметры


ObjectKey. Ключ объекта, на
 который ссылается атрибут.


## Описание


Метод FindByValuesObject осуществляет
 поиск атрибута по ключу объекта, на который ссылается атрибут.


## Пример


Использование метода приведено в примере для [IMetaDictionary.AttributeMnemoNamings](../IMetaDictionary/IMetaDictionary.AttributeMnemoNamings.htm).
