# IMetaAttributesSet.IndexOf


## Синтаксис


IndexOf(Attribute: [IMetaAttribute](../IMetaAttribute/IMetaAttribute.htm)):
 Integer;


## Параметры


Attribute - атрибут.


## Описание


Метод IndexOf возвращает индекс
 атрибута в наборе. Если атрибут отсутствует в наборе, то метод возвращает
 значение «-1».
