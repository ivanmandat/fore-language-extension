# IMetaMembersTree


## Описание


Интерфейс IMetaMembersTree содержит
 свойства и методы для работы с деревом элементов системного справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CurrentDepth](IMetaMembersTree.CurrentDepth.htm)


		 Свойство CurrentDepth
		 возвращает число уровней иерархии от корня справочника.


		 ![](../../Property_Image.gif)
		 [CurrentPosition](IMetaMembersTree.CurrentPosition.htm)


		 Свойство CurrentPosition
		 возвращает текущую позицию в иерархии элементов.


## Свойства, унаследованные от [IMetaMembersSet](../IMetaMembersSet/IMetaMembersSet.htm)


		 Имя метода
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CurrentOffset](../IMetaMembersSet/IMetaMembersSet.CurrentOffset.htm)


		 Свойство CurrentOffset
		 возвращает число уровней иерархии от корня справочника до набора
		 элементов.


## Методы, унаследованные от [IMetaMembersSet](../IMetaMembersSet/IMetaMembersSet.htm)


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Current](../IMetaMembersSet/IMetaMembersSet.Current.htm)


		 Метод Current возвращает
		 текущий элемент в наборе.


		 ![](../../Sub_Image.gif)
		 [Eof](../IMetaMembersSet/IMetaMembersSet.Eof.htm)


		 Метод Eof возвращает
		 признак того, является ли текущий элемент последним в наборе.


		 ![](../../Sub_Image.gif)
		 [Next](../IMetaMembersSet/IMetaMembersSet.Next.htm)


		 Метод Next осуществляет
		 переход к следующему элементу в наборе.


		 ![](../../Sub_Image.gif)
		 [Reset](../IMetaMembersSet/IMetaMembersSet.Reset.htm)


		 Метод Reset сбрасывает
		 текущее положение курсора в наборе элементов.
