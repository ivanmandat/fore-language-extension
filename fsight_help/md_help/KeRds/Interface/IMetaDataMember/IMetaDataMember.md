# IMetaDataMember


## Описание


Интерфейс IMetaDataMember содержит
 базовые свойства для работы с записью системной таблицы.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attribute](IMetaDataMember.Attribute.htm)


		 Свойство Attribute
		 возвращает атрибут записи.


		 ![](../../Property_Image.gif)
		 [Modified](IMetaDataMember.Modified.htm)


		 Свойство Modified возвращает
		 признак того, была ли запись изменена.


		 ![](../../Property_Image.gif)
		 [UnmodifiedValue](IMetaDataMember.UnmodifiedValue.htm)


		 Свойство UnmodifiedValue
		 возвращает значение записи до изменения.


		 ![](../../Property_Image.gif)
		 [Value](IMetaDataMember.Value.htm)


		 Свойство Value определяет
		 значение записи.
