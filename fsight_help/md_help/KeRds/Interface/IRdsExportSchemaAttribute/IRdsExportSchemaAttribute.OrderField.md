# IRdsExportSchemaAttribute.OrderField


## Синтаксис


OrderField: String;


## Описание


Свойство OrderField определяет наименование поля таблицы [DetailTable](IRdsExportSchemaAttribute.DetailTable.htm), в котором будут содержаться номера значений.
