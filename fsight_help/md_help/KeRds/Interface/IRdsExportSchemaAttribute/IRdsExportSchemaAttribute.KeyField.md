# IRdsExportSchemaAttribute.KeyField


## Синтаксис


KeyField: String;


## Описание


Свойство KeyField определяет наименование поля таблицы [DetailTable](IRdsExportSchemaAttribute.DetailTable.htm), в котором будут содержаться ключи элементов имеющих множественные значения атрибута [Attribute](IRdsExportSchemaAttribute.Attribute.htm).
