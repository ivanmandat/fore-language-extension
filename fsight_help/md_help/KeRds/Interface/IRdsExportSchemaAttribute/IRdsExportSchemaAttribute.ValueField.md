# IRdsExportSchemaAttribute.ValueField


## Синтаксис


ValueField: String;


## Описание


Свойство ValueField определяет наименование поля таблицы [DetailTable](IRdsExportSchemaAttribute.DetailTable.htm), содержащего ключи элементов связанного справочника, из которых формируется множественное значение по атрибуту [Attribute](IRdsExportSchemaAttribute.Attribute.htm).
