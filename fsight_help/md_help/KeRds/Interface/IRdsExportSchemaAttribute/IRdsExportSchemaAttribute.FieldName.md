# IRdsExportSchemaAttribute.FieldName


## Синтаксис


FieldName: String;


## Описание


Свойство FieldName определяет
 наименование поля таблицы, с которым связывается атрибут справочника НСИ.


## Комментарии


Если атрибут имеет множественные значения, в данном свойстве указывается
 поле таблицы [TableName](../IRdsExportSchema/IRdsExportSchema.TableName.htm),
 с которым осуществляется связь поля [KeyField](IRdsExportSchemaAttribute.KeyField.htm)
 из таблицы [DetailTable](IRdsExportSchemaAttribute.DetailTable.htm).
