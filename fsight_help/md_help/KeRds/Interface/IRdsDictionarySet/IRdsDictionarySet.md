# IRdsDictionarySet


## Описание


Интерфейс IRdsDictionarySet
 содержит свойства и методы для работы с коллекцией справочников, данные
 которых необходимо синхронизировать.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IRdsDictionarySet.Count.htm)


		 Свойство Count возвращает
		 количество справочников, данные которых необходимо синхронизировать.


		 ![](../../Property_Image.gif)
		 [Item](IRdsDictionarySet.Item.htm)


		 Свойство Item возвращает
		 справочник НСИ, входящий в список синхронизации.
