# IRdsDictionarySet.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс справочника НСИ.


## Описание


Метод Remove удаляет справочник НСИ из списка синхронизации. Индекс справочника передается посредством параметра Index.
