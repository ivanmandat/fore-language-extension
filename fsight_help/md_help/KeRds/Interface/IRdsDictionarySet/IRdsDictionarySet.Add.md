# IRdsDictionarySet.Add


## Синтаксис


Add(Value: [IRdsDictionary](../IRdsDictionary/IRdsDictionary.htm)): Integer;


## Параметры


Value. Справочник НСИ, который необходимо добавить в список синхронизации.


## Описание


Метод Add осуществляет добавление справочника НСИ в список синхронизации.
