# IRdsDictionarySet.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ справочника НСИ.


## Описание


Метод RemoveByKey удаляет справочник НСИ из списка синхронизации. Ключ справочника передается посредством параметра Key.
