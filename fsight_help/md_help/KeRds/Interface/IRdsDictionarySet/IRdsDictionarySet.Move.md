# IRdsDictionarySet.Move


## Синтаксис


Move(IndexFrom: Integer; IndexTo: Integer);


## Параметры


IndexFrom. Индекс позиции справочника НСИ, который необходимо переместить.


IndexTo. Индекс позиции, в которую необходимо переместить справочник.


## Описание


Метод Move осуществляет перемещение справочника НСИ в списке синхронизации.
