# IRdsDictionarySet.FindById


## Синтаксис


FindById(Id: String): [IRdsDictionary](../IRdsDictionary/IRdsDictionary.htm);


## Параметры


Id. Идентификатор справочника НСИ, по которому осуществляется поиск.


## Описание


Метод FindById осуществляет поиск справочника НСИ в списке синхронизации. Идентификатор, по которому осуществляется поиск, передается посредством параметра Id.
