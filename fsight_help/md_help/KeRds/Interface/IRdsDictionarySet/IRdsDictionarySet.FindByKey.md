# IRdsDictionarySet.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IRdsDictionary](../IRdsDictionary/IRdsDictionary.htm);


## Параметры


Key. Ключ справочника НСИ, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет поиск справочника НСИ в списке синхронизации. Ключ, по которому осуществляется поиск, передается посредством параметра Key.
