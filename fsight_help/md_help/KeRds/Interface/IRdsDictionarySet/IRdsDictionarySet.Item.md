# IRdsDictionarySet.Item


## Синтаксис


Item(Index: Integer): [IRdsDictionary](../IRdsDictionary/IRdsDictionary.htm);


## Параметры


Index. Индекс справочника в списке синхронизации.


## Описание


Свойство Item возвращает справочник НСИ, входящий в список синхронизации. Индекс справочника передается посредством параметра Index.
