# IRdsDictionaryBatchDelete.CloseDate


## Синтаксис


CloseDate: DateTime;


## Описание


Свойство CloseDate определяет
 дату окончания действия значений элементов версионного справочника НСИ.


## Комментарии


Версионный справочник НСИ - это справочник, элементы которого могут
 изменяться во времени, т.е. для каждого элемента указан диапазон действия
 значения: дата начала действия и дата окончания действия.


Дата окончания действия значения элементов учитывается, если пакет создан
 для версионного справочника с ручным управлением периодами.
