# IRdsParams


## Описание


Интерфейс IRdsParams содержит
 свойства и методы коллекции параметров справочника НСИ.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IRdsParams.Count.htm)


		 Свойство Count возвращает
		 количество параметров справочника НСИ.


		 ![](../../Property_Image.gif)
		 [CurDate](IRdsParams.CurDate.htm)


		 Свойство CurDate возвращает
		 системный параметр справочника, назначение которого - Дата актуальности.


		 ![](../../Property_Image.gif)
		 [Item](IRdsParams.Item.htm)


		 Свойство Item возвращает
		 параметр справочника НСИ.


		 ![](../../Property_Image.gif)
		 [Predefined](IRdsParams.Predefined.htm)


		 Свойство Predefined
		 возвращает системный параметр справочника НСИ.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IRdsParams.Add.htm)


		 Метод Add создает новый
		 параметр справочника НСИ.


		 ![](../../Sub_Image.gif)
		 [Clear](IRdsParams.Clear.htm)


		 Метод Clear осуществляет
		 удаление всех параметров справочника НСИ.


		 ![](../../Sub_Image.gif)
		 [CreateValues](IRdsParams.CreateValues.htm)


		 Метод CreateValues
		 создает объект, управляющий значениями параметров, необходимых
		 для открытия справочника.


		 ![](../../Sub_Image.gif)
		 [FindByAttribute](IRdsParams.FindByAttribute.htm)


		 Метод FindByAttribute
		 осуществляет поиск управляющего параметра для указанного атрибута.


		 ![](../../Sub_Image.gif)
		 [FindById](IRdsParams.FindById.htm)


		 Метод FindById осуществляет
		 поиск параметра по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IRdsParams.FindByKey.htm)


		 Метод FindByKey осуществляет
		 поиск параметра по ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](IRdsParams.Remove.htm)


		 Метод Remove осуществляет
		 удаление параметра по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IRdsParams.RemoveByKey.htm)


		 Метод RemoveByKey осуществляет
		 удаление параметра по ключу.
