# IRdsParams.FindByAttribute


## Синтаксис


FindByAttribute(Attribute: [IRdsAttribute](../IRdsAttribute/IRdsAttribute.htm)):
 [IRdsParam](../IRdsParam/IRdsParam.htm);


## Параметры


Attribute - атрибут справочника,
 по которому осуществляется поиск.


## Описание


Метод FindByAttribute осуществляет
 поиск и возвращает параметр справочника НСИ в случае удачного поиска.
 Атрибут, для которого необходимо найти управляющий параметра, передается
 посредством параметра Attribute.
