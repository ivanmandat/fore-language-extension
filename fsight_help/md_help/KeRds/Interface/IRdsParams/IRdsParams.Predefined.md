# IRdsParams.Predefined


## Синтаксис


Predefined(tPredef: [RdsPredefinedParam](../../Enums/RdsPredefinedParam.htm)):
 [IRdsParam](../IRdsParam/IRdsParam.htm);


## Параметры


tPredef - назначение системного
 параметра, который необходимо получить.


## Описание


Свойство Predefined возвращает
 системный параметр справочника НСИ, назначение которого передается посредством
 параметра tPredef.
