# IRdsParams.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IRdsParam](../IRdsParam/IRdsParam.htm);


## Параметры


Key -
 ключ параметра, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает параметр справочника НСИ в случае удачного поиска.
 Ключ, по которому осуществляется поиск, передается посредством параметра
 Key.
