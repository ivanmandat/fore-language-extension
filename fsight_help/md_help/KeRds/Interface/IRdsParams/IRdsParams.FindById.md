# IRdsParams.FindById


## Синтаксис


FindById(Id: String): [IRdsParam](../IRdsParam/IRdsParam.htm);


## Параметры


Id - идентификатор параметр,
 по которому осуществляется поиск.


## Описание


Метод FindById осуществляет
 поиск и возвращает параметр справочника НСИ в случае удачного поиска.
 Идентификатор, по которому осуществляется поиск, передается посредством
 параметра Id.
