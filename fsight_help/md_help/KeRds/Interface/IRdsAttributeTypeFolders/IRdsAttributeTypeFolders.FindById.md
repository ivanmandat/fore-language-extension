# IRdsAttributeTypeFolders.FindById


## Синтаксис


FindById(Id: String): [IRdsAttributeTypeFolder](../IRdsAttributeTypeFolder/IRdsAttributeTypeFolder.htm);


## Параметры


Id - идентификатор папки, по
 которому осуществляется поиск.


## Описание


Метод FindById осуществляет
 поиск и возвращает содержимое папки, в случае удачного поиска. Идентификатор,
 по которому осуществляется поиск, передается посредством параметра Id.
