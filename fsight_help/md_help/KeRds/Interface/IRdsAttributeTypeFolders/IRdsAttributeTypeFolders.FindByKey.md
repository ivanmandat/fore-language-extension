# IRdsAttributeTypeFolders.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IRdsAttributeTypeFolder](../IRdsAttributeTypeFolder/IRdsAttributeTypeFolder.htm);


## Параметры


Key -
 ключ папки, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает содержимое папки, в случае удачного поиска. Ключ, по
 которому осуществляется поиск, передается посредством параметра Key
