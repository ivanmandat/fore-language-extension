# IRdsAttributeTypeFolders


## Описание


Интерфейс IRdsAttributeTypeFolders
 содержит свойства и методы коллекции папок типов атрибутов репозитория
 НСИ.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IRdsAttributeTypeFolders.Count.htm)


		 Свойство Count возвращает
		 количество папок в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IRdsAttributeTypeFolders.Item.htm)


		 Свойство Item возвращает
		 содержимое папки.


## Методы


		 Имя метода
		 Краткое описание

		 ![](../../Sub_Image.gif)
