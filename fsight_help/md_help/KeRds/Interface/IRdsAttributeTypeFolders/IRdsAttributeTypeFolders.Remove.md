# IRdsAttributeTypeFolders.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс папки.


## Описание


Метод Remove осуществляет удаление
 папки, индекс которой передается посредством параметра Index
 и возвращает True, в случае удачного
 удаления. Папка удаляется со всеми дочерними папками и типами атрибутов.
