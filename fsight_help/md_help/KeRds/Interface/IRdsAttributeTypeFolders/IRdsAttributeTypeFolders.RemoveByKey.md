# IRdsAttributeTypeFolders.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key - ключ папки.


## Описание


Метод RemoveByKey осуществляет
 удаление папки, ключ которой передается посредством параметра Key
 и возвращает True, в случае удачного
 удаления.
