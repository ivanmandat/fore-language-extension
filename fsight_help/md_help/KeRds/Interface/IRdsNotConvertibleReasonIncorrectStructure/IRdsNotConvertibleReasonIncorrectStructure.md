# IRdsNotConvertibleReasonIncorrectStructure


Сборка: Rds;


## Описание


Интерфейс IRdsNotConvertibleReasonIncorrectStructure
 предназначен для работы с причинами, по которым справочник НСИ не может
 быть преобразован в табличный справочник НСИ, связанными с некорректной
 структурой справочника.

## Комментарии


Для работы с причиной, по которой справочник НСИ не может быть преобразован
 в табличный справочник НСИ, используйте свойства и методы интерфейса [IRdsNotConvertibleReason](../IRdsNotConvertibleReason/IRdsNotConvertibleReason.htm).
