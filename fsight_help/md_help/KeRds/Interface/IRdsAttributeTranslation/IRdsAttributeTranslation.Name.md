# IRdsAttributeTranslation.Name


## Синтаксис


Name: String;


## Описание


Свойство Name определяет наименование
 атрибута для языка настройки.


## Комментарии


Наименование, установленное свойством Name,
 будет отображаться для атрибута справочника, если свойство [IRdsDictionaryTranslations.IsOn](../IRdsDictionaryTranslations/IRdsDictionaryTranslations.IsOn.htm)
 установлено в True.
