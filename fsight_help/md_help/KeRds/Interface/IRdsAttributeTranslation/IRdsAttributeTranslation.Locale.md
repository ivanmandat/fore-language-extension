# IRdsAttributeTranslation.Locale


## Синтаксис


Locale: [LocaleCodeID](ForeSys.chm::/Enums/LocaleCodeID.htm);


## Описание


Свойство доступно только для чтения.


Свойство Locale возвращает
 идентификатор языка, для которого предназначена настройка.


## Комментарии


Свойство возвращает значение, определяемое перечислением [LocaleCodeID](ForeSys.chm::/Enums/LocaleCodeID.htm).
 Locale не может вернуть значение
 LocaleCodeID.None.
