# IMetaAttributeGroupsSet.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer):Boolean;


## Параметры


Key. Ключ группы атрибутов.


## Описание


Метод RemoveByKey удаляет
 из коллекции группу атрибутов с указанным ключом.


## Комментарии


Если метод вернул значение True,
 то удаление группы прошло успешно.
