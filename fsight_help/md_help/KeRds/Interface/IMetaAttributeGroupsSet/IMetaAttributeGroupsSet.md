# IMetaAttributeGroupsSet


## Описание


Интерфейс IMetaAttributeGroupsSet
 предназначен для работы с коллекцией групп атрибутов справочника без возможности
 добавления атрибута.


## Иерархия наследования


           IMetaAttributeGroupsSet


## Комментарии


Для добавления группы атрибутов используйте метод [IMetaAttributeGroups.Add](../IMetaAttributeGroups/IMetaAttributeGroups.Add.htm).
