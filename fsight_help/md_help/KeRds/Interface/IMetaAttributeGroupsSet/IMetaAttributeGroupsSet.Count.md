# IMetaAttributeGroupsSet.Count


## Синтаксис


Count: Integer;


## Описание


Свойство доступно только для чтения.


Свойство Count возвращает
 количество элементов в коллекции групп атрибутов.


## Комментарии


Для добавления элемента в коллекцию используйте метод [IMetaAttributeGroups.Add](../IMetaAttributeGroups/IMetaAttributeGroups.Add.htm).
 Для удаления элементов предназначены методы [IMetaAttributeGroupsSet.Clear](IMetaAttributeGroupsSet.Clear.htm),
 [IMetaAttributeGroupsSet.Remove](IMetaAttributeGroupsSet.Remove.htm),
 [IMetaAttributeGroupsSet.RemoveByKey](IMetaAttributeGroupsSet.RemoveByKey.htm).
