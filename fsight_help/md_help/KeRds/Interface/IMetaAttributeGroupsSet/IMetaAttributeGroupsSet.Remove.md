# IMetaAttributeGroupsSet.Remove


## Синтаксис


Remove(Index: Integer):Boolean;


## Параметры


Index. Индекс удаляемого элемента.


## Описание


Метод Remove удаляет указанный
 элемент из коллекции.


## Комментарии


Если метод вернул значение True,
 то удаление элемента прошло успешно.


Индексация элементов коллекции сквозная и начинается с нуля. Число элементов
 в коллекции возвращает свойство [IMetaAttributeGroupsSet.Count](IMetaAttributeGroupsSet.Count.htm).
