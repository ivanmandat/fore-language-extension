# IMetaAttributeGroupsSet.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IMetaAttributeGroup](../IMetaAttributeGroup/IMetaAttributeGroup.htm);


## Параметры


Key.
 Ключ группы атрибутов.


## Описание


Метод FindByKey выполняет
 поиск в коллекции группы атрибутов с указанным ключом.


## Комментарии


После выполнения метод возвращает группу атрибутов с указанным ключом.
 Если группа не найдена, то метод возвращает значение Null.
