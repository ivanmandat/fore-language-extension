# IRdsDatabaseInstance


## Описание


Интерфейс IRdsDatabaseInstance
 содержит свойства и методы для работы с данными репозитория НСИ.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AttributeTypeFolders](IRdsDatabaseInstance.AttributeTypeFolders.htm)


		 Свойство AttributeTypeFolders
		 возвращает параметры корневой папки справочника типов атрибутов
		 репозитория НСИ.


		 ![](../../Property_Image.gif)
		 [AttributeTypes](IRdsDatabaseInstance.AttributeTypes.htm)


		 Свойство AttributeTypes
		 возвращает содержимое корневой папки справочника типов атрибутов
		 репозитория НСИ.


		 ![](../../Property_Image.gif)
		 [Database](IRdsDatabaseInstance.Database.htm)


		 Свойство Database возвращает
		 параметры репозитория НСИ.


		 ![](../../Property_Image.gif)
		 [DatabaseInstance](IRdsDatabaseInstance.DatabaseInstance.htm)


		 Свойство DatabaseInstance
		 возвращает содержимое базы данных, в которой хранятся данные справочников
		 НСИ.


		 ![](../../Property_Image.gif)
		 [ParamTypeFolders](IRdsDatabaseInstance.ParamTypeFolders.htm)


		 Свойство ParamTypeFolders
		 возвращает параметры корневой папки справочника типов параметров
		 репозитория НСИ.


		 ![](../../Property_Image.gif)
		 [ParamTypes](IRdsDatabaseInstance.ParamTypes.htm)


		 Свойство ParamTypes
		 возвращает содержимое корневой папки справочника типов параметров
		 репозитория НСИ.
