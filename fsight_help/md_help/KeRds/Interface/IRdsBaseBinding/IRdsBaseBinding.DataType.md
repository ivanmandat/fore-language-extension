# IRdsBaseBinding.DataType


## Синтаксис


DataType: [DbDataType](Dal.chm::/Enums/DbDataType.htm);


## Описание


Свойство DataType определяет
 тип данных атрибута.


## Комментарии


Данное свойство актуально, если привязка используется при создании нового
 справочника. Указанный в свойстве DataType
 тип данных будет установлен для атрибута при его создании.
