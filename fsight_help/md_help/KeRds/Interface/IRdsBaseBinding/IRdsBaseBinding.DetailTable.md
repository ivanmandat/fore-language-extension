# IRdsBaseBinding.DetailTable


## Синтаксис


DetailTable: String;


## Описание


Свойство DetailTable определяет
 физическое имя таблицы, из которой будут загружаться множественные значения
 атрибута.


## Комментарии


Свойство актуально, если свойству [HasMultipleValues](IRdsBaseBinding.HasMultipleValues.htm)
 установлено значение True.
