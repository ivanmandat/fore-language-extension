# IRdsBaseBinding.DetailTableObject


## Синтаксис


DetailTableObject: [IMetabaseObjectDescriptor](KeSom.chm::/Interface/IMetabaseObjectDescriptor/IMetabaseObjectDescriptor.htm);


## Описание


Свойство DetailTableObject определяет
 описание таблицы репозитория, из которой будут загружаться множественные
 значения атрибута.


## Комментарии


Свойство актуально, если свойству [HasMultipleValues](IRdsBaseBinding.HasMultipleValues.htm)
 установлено значение True.
