# IRdsBaseBinding.Nullable


## Синтаксис


Nullable: Boolean;


## Описание


Свойство Nullable определяет,
 могут ли по атрибуту, для которого создана привязка, храниться пустые
 значения.


## Комментарии


Данное свойство актуально, если привязка используется при создании нового
 справочника. Указанное значение свойства Nullable
 будет установлено для атрибута при его создании.
