# IRdsDictionaryElementData.Modified


## Синтаксис


Modified(AttributeKey: Integer): Boolean;


## Параметры


AttributeKey - ключ атрибута,
 по которому необходимо определить наличие изменений.


## Описание


Свойство Modified возвращает
 признак наличия изменений данных элемента по атрибуту, ключ которого передается
 посредством параметра AttributeKey.
