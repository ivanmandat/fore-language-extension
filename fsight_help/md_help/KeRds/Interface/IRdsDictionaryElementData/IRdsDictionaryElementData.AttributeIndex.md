# IRdsDictionaryElementData.AttributeIndex


## Синтаксис


AttributeIndex(AttributeKey: Integer): Integer;


## Параметры


AttributeKey -
 ключ атрибута, индекс которого необходимо получить.


## Описание


Свойство AttributeIndex возвращает
 индекс атрибута, ключ которого передается посредством параметра AttributeKey.
