# IRdsDictionaryElementData.Value


## Синтаксис


Value(Index: Integer): Variant;


## Параметры


Index. Индекс атрибута справочника,
 по которому необходимо определить значение.


## Описание


Свойство Value определяет значение
 элемента по атрибуту, индекс которого передается посредством параметра
 Index.
