# IRdsDictionaryElementData


Сборка: Rds;


## Описание


Интерфейс IRdsDictionaryElementData
 содержит свойства для доступа к данным атрибутов элемента справочника
 НСИ.


## Иерархия наследования


           IRdsDictionaryElementData
