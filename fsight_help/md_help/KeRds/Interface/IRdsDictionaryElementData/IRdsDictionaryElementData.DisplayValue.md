# IRdsDictionaryElementData.DisplayValue


## Синтаксис


DisplayValue(Index: Integer): Variant;


## Параметры


Index - индекс атрибута, по
 которому необходимо получить отображаемое значение.


## Описание


Свойство DisplayValue возвращает
 значение, которое будет отображаться по атрибуту с индексом Index
 для элемента.
