# IRdsDictionaryElementData.ValueModified


## Синтаксис


ValueModified(Index: Integer): Boolean;


## Параметры


Index -
 ключ атрибута, по которому необходимо определить наличие изменений.


## Описание


Свойство ValueModified возвращает
 признак наличия изменений данных элемента по атрибуту, индекс которого
 передается посредством параметра Index.
