# IMetaDataRule.Condition


## Синтаксис


Condition: [IOrmConditions](KeOrm.chm::/Interface/IOrmConditions/IOrmConditions.htm);


## Описание


Свойство Condition возвращает коллекцию условий, в соответствии с которыми будут отбираться данные, для которых необходимо проверить условия правила.


## Комментарии


Условия правила задаются в свойстве [Rule](IMetaDataRule.Rule.htm).
