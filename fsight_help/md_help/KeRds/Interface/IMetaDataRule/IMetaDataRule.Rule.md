# IMetaDataRule.Rule


## Синтаксис


Rule: [IOrmConditions](KeOrm.chm::/Interface/IOrmConditions/IOrmConditions.htm);


## Описание


Свойство Rule возвращает коллекцию условий правила, которые необходимо проверить для данных.


## Комментарии


Условия, заданные в данном свойстве, будут проверяться только для тех данных, которые соответствуют условиям отбора. Условия отбора данных задаются в свойстве [Condition](IMetaDataRule.Condition.htm).
