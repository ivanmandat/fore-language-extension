# IRdsSegments.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IRdsSegment](../IRdsSegment/IRdsSegment.htm);


## Параметры


Key - ключ сегмента, по которому
 осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает сегмент справочника НСИ в случае удачного поиска. Ключ,
 по которому осуществляется поиск, передается посредством параметра Key.
