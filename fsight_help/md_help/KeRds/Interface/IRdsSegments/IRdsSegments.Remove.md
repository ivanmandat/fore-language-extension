# IRdsSegments.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс сегмента.


## Описание


Метод Remove осуществляет удаление
 сегмента, индекс которого передается посредством параметра Index
 и возвращает True в случае удачного
 удаления.
