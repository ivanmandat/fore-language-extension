# IRdsSegments.FindById


## Синтаксис


FindById(Id: String): [IRdsSegment](../IRdsSegment/IRdsSegment.htm);


## Параметры


Id - идентификатор сегмента,
 по которому осуществляется поиск.


## Описание


Метод FindById осуществляет
 поиск и возвращает сегмент справочника НСИ в случае удачного поиска. Идентификатор,
 по которому осуществляется поиск, передается посредством параметра Id.
