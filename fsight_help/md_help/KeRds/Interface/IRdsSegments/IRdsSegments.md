# IRdsSegments


## Описание


Интерфейс IRdsSegments содержит
 свойства и методы для работы с коллекцией сегментов справочника НСИ.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IRdsSegments.Count.htm)


		 Свойство Count возвращает
		 количество сегментов справочника НСИ.


		 ![](../../Property_Image.gif)
		 [Item](IRdsSegments.Item.htm)


		 Свойство Item возвращает
		 сегмент справочника НСИ.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IRdsSegments.Add.htm)


		 Метод Add создает сегмент
		 справочника НСИ.


		 ![](../../Sub_Image.gif)
		 [Clear](IRdsSegments.Clear.htm)


		 Метод Clear осуществляет
		 удаление всех сегментов справочника НСИ.


		 ![](../../Sub_Image.gif)
		 [FindById](IRdsSegments.FindById.htm)


		 Метод FindById осуществляет
		 поиск сегмента справочника НСИ по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IRdsSegments.FindByKey.htm)


		 Метод FindByKey осуществляет
		 поиск сегмента справочника НСИ по ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](IRdsSegments.Remove.htm)


		 Метод Remove осуществляет
		 удаление сегмента по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IRdsSegments.RemoveByKey.htm)


		 Метод RemoveByKey осуществляет
		 удаление сегмента по ключу.
