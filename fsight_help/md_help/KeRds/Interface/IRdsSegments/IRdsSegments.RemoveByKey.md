# IRdsSegments.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key - ключ сегмента справочника
 НСИ.


## Описание


Метод RemoveByKey осуществляет
 удаление сегмента, ключ которого передается посредством параметра Key и возвращает True
 в случае удачного удаления.
