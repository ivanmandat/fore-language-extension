# IMetaAttribute.Translations


## Синтаксис


Translations: [IMetaAttributeTranslations](../IMetaAttributeTranslations/IMetaAttributeTranslations.htm);


## Описание


Свойство Translations возвращает
 параметры перевода атрибутов базы данных временных рядов на различные
 языки.


## Комментарии


Изначально в коллекции содержится настройка для языка, установленного для
 репозитория по умолчанию. Чтобы переводить наименование атрибута справочника
 на различные языки необходимо установить значение True
 в свойство [IMetaDictionaryTranslations.IsOn](../IMetaDictionaryTranslations/IMetaDictionaryTranslations.IsOn.htm).


Свойство доступно только для чтения.
