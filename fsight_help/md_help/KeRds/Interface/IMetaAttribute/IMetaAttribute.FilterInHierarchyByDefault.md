# IMetaAttribute.FilterInHierarchyByDefault


## Синтаксис


FilterInHierarchyByDefault: [IDimSelection](KeDims.chm::/interface/IDimSelection/IDimSelection.htm);


## Описание


Свойство FilterInHierarchyByDefault
 определяет отметку, используемую для атрибута по умолчанию.


## Комментарии


Свойство актуально, если атрибут участвует в иерархии по умолчанию,
 то есть свойство [IMetaAttribute.IsInHierarchyByDefault](IMetaAttribute.IsInHierarchyByDefault.htm)
 имеет значение True.
