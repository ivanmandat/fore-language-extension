# IMetaAttribute.Hidden


## Синтаксис


Hidden: Boolean;


## Описание


Свойство Hidden определяет, является ли данный атрибут скрытым.


## Комментарии


Если Hidden = True, то атрибут является скрытым, т.е. он будет участвовать в построении справочника, но при этом в справочнике отображаться не будет.
