# IMetaAttribute.InDefaultHierarchy


## Синтаксис


InDefaultHierarchy: Boolean;


## Описание


Метод InDefaultHierarchy возвращает признак того, входит ли атрибут в иерархию, заданную по умолчанию. True - атрибут входит в иерархию, заданную по умолчанию.
