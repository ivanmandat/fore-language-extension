# IMetaAttribute.OrderInHierarchyByDefault


## Синтаксис


OrderInHierarchyByDefault: Integer;


## Описание


Свойство OrderInHierarchyByDefault
 определяет порядок атрибута в иерархии по умолчанию.


## Комментарии


Свойство актуально, если атрибут участвует в иерархии по умолчанию,
 то есть свойство [IMetaAttribute.IsInHierarchyByDefault](IMetaAttribute.IsInHierarchyByDefault.htm)
 имеет значение True.


Нумерация атрибутов в порядке по умолчанию сплошная и начинается с нуля.
