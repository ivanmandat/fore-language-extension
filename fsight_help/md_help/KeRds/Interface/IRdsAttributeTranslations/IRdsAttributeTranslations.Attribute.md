# IRdsAttributeTranslations.Attribute


## Синтаксис


Attribute(Locale: [LocaleCodeID](ForeSys.chm::/Enums/LocaleCodeID.htm)):
 [IRdsAttribute](../IRdsAttribute/IRdsAttribute.htm);


## Параметры


Locale. Уникальный числовой
 идентификатор языка.


## Описание


Свойство Attribute определяет
 дополнительный атрибут, предназначенный для перевода.


## Комментарии


В качестве параметра Locale
 нельзя указывать значение LocaleCodeID.None.


Для языка справочника, используемого по умолчанию (свойство [IRdsDictionaryTranslations.DefaultLocale](../IRdsDictionaryTranslations/IRdsDictionaryTranslations.DefaultLocale.htm)),
 данное свойство должно иметь значение Null.
