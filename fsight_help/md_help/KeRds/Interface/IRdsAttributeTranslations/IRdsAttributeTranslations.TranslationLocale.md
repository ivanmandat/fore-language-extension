# IRdsAttributeTranslations.TranslationLocale


## Синтаксис


TranslationLocale: [LocaleCodeID](ForeSys.chm::/Enums/LocaleCodeID.htm);


## Описание


Свойство доступно только для чтения.


Свойство TranslationLocale возвращает
 идентификатор языка, для перевода на который предназначен атрибут.


## Комментарии


Свойство возвращает значение, определяемое перечислением [LocaleCodeID](ForeSys.chm::/Enums/LocaleCodeID.htm).
 TranslationLocale не может вернуть
 значение LocaleCodeID.None.
