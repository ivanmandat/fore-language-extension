# IRdsMergeCallback.OnProgress


## Синтаксис


OnProgress(Progress: Integer; Total: Integer);


## Параметры


Progress. Количество этапов, пройденных в процессе синхронизации.


Total. Общее количество этапов процесса синхронизации.


## Описание


Метод OnProgress реализует событие общего статуса процесса синхронизации справочников НСИ.
