# IRdsDictionaryBox.RootImageIndex


## Синтаксис


RootImageIndex: Integer;


## Описание


Свойство RootImageIndex определяет
 индекс пиктограммы для корневого элемента компонента.


## Комментарии


Значение по умолчанию «-1», т.е. пиктограмма не задана.
