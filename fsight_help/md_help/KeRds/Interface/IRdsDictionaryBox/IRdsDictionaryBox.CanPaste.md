# IRdsDictionaryBox.CanPaste


## Синтаксис


CanPaste: Boolean;


## Описание


Метод CanPaste возвращает признак
 доступности вставки элемента из буфера обмена.


## Комментарии


Допустимые значения:


	- True. Вставка элемента
	 доступна. Для выполнения вставки используйте метод [IRdsDictionaryBox.Paste](IRdsDictionaryBox.Paste.htm);


	- False. Вставка элемента
	 недоступна.
