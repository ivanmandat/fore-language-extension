# IRdsDictionaryBox.Copy


## Синтаксис


Copy;


## Описание


Метод Copy копирует выделенный
 элемент в буфер обмена.


## Комментарии


Признак того, доступно ли копирование элемента, возвращает метод [IRdsDictionaryBox.CanCopy](IRdsDictionaryBox.CanCopy.htm).
