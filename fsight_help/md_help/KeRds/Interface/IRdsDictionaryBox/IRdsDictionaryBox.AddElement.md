# IRdsDictionaryBox.AddElement


## Синтаксис


AddElement: Integer;


## Описание


Метод AddElement вызывает стандартный диалог создания нового элемента справочника НСИ и возвращает ключ созданного элемента в случае положительного ответа, иначе - «-1».
