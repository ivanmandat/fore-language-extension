# IRdsDictionaryBox.CanElementDown


## Синтаксис


CanElementDown: Boolean;


## Описание


Свойство CanElementDown возвращает признак возможности перемещения выделенных элементов на одну позицию вниз.


## Комментарии


Свойство возвращает True, если перемещение возможно, и False в противоположном случае.
