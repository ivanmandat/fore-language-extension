# IRdsDictionaryBox.CanElementLevelUp


## Синтаксис


CanElementLevelUp: Boolean;


## Описание


Свойство CanElementLevelUp возвращает признак возможности перемещения выделенных элементов на один уровень вверх.


## Комментарии


Свойство возвращает True, если перемещение возможно, и False в противоположном случае.
