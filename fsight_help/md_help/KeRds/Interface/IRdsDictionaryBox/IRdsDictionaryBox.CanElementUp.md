# IRdsDictionaryBox.CanElementUp


## Синтаксис


CanElementUp: Boolean;


## Описание


Свойство CanElementUp возвращает признак возможности перемещения выделенных элементов на одну позицию вверх.


## Комментарии


Свойство возвращает True, если перемещение возможно, и False в противоположном случае.
