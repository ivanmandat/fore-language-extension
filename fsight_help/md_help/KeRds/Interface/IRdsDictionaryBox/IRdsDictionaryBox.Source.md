# IRdsDictionaryBox.Source


## Синтаксис


Source: [IRdsDictionarySource](../IRdsDictionarySource/IRdsDictionarySource.htm);


## Описание


Свойство Source определяет источник
 данных для компонента [RdsDictionaryBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/RdsDictionaryBox.htm).


## Комментарии


В качестве источника могут быть установлены любые компоненты [UiRdsDictionary](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/UiRdsDictionary.htm),
 имеющиеся на форме.
