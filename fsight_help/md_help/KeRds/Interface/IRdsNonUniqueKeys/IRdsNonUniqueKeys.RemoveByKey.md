# IRdsNonUniqueKeys.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ неуникального ключа
 справочника НСИ.


## Описание


Метод RemoveByKey осуществляет
 удаление неуникального ключа по ключу в справочнике НСИ.


## Комментарии


Для поиска неуникального ключа по ключу используйте метод [IRdsNonUniqueKeys.FindByKey](IRdsNonUniqueKeys.FindByKey.htm).
