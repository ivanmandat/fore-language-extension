# IRdsNonUniqueKeys.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 неуникальных ключей справочника НСИ.


## Комментарии


Для получения неуникального ключа справочника НСИ используйте свойство
 [IRdsNonUniqueKeys.Item](IRdsNonUniqueKeys.Item.htm).
