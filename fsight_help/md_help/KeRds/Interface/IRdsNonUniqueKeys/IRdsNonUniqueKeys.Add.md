# IRdsNonUniqueKeys.Add


## Синтаксис

## Описание


Метод Add создает новый неуникальный
 ключ справочника НСИ.


## Комментарии


Для удаления неуникального ключа по индексу используйте метод [IRdsNonUniqueKeys.Remove](IRdsNonUniqueKeys.Remove.htm).
