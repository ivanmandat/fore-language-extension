# IRdsNonUniqueKeys.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index. Индекс неуникального
 ключа.


## Описание


Метод Remove осуществляет удаление
 неуникального ключа по индексу в справочнике НСИ.


## Комментарии


Для удаления неуникального ключа по ключу используйте метод [IRdsNonUniqueKeys.RemoveByKey](IRdsNonUniqueKeys.RemoveByKey.htm).
