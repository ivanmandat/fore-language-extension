# IRdsNonUniqueKeys.Item


## Синтаксис


Item(Index: Integer): [IRdsNonUniqueKey](../IRdsNonUniqueKey/IRdsNonUniqueKey.htm);


## Параметры


Index. Индекс неуникального
 ключа справочника НСИ.


## Описание


Свойство Item возвращает неуникальный
 ключ справочника НСИ.


## Комментарии


Для получения количества неуникальных ключей справочника НСИ используйте
 свойство [IRdsNonUniqueKeys.Count](IRdsNonUniqueKeys.Count.htm).
