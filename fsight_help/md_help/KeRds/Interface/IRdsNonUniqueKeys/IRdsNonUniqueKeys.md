# IRdsNonUniqueKeys


Сборка: Rds;


## Описание


Интерфейс IRdsNonUniqueKeys
 предназначен для работы с коллекцией неуникальных ключей справочника НСИ.


## Иерархия наследования


           IRdsNonUniqueKeys


## Комментарии


Для работы с неуникальными ключами справочника НСИ используйте интерфейс
 [IRdsNonUniqueKey](../IRdsNonUniqueKey/IRdsNonUniqueKey.htm).
