# IRdsNonUniqueKeys.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IRdsNonUniqueKey](../IRdsNonUniqueKey/IRdsNonUniqueKey.htm);


## Параметры


Key. Ключ неуникального ключа,
 по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск неуникального ключа по ключу в справочнике НСИ.


## Комментарии


Для удаления неуникального ключа по ключу используйте метод [IRdsNonUniqueKeys.RemoveByKey](IRdsNonUniqueKeys.RemoveByKey.htm).
