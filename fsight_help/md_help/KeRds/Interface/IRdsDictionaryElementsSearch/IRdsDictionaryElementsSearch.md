# IRdsDictionaryElementsSearch


Сборка: Rds;


## Описание


Интерфейс IRdsDictionaryElementsSearch
 содержит свойства и методы для поиска элементов справочника НСИ с расширенными
 настройками поиска.


## Иерархия наследования


           IRdsDictionaryElementsSearch


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attributes](IRdsDictionaryElementsSearch.Attributes.htm)
		 Свойство Attributes
		 возвращает список атрибутов, по значениям которых будет осуществляться
		 поиск.


		 ![](../../Property_Image.gif)
		 [Condition](IRdsDictionaryElementsSearch.Condition.htm)
		 Свойство Condition
		 возвращает коллекцию условий расширенного поиска.

		 [Elements](IRdsDictionaryElementsSearch.Elements.htm)
		 Свойство Elements возвращает
		 коллекцию элементов, по значениям атрибутов которых будет производиться
		 поиск.


		 ![](../../Property_Image.gif)
		 [IsIterator](IRdsDictionaryElementsSearch.IsIterator.htm)
		 Свойство IsIterator
		 возвращает признак того, что поиск выполняется среди элементов,
		 полученных из итератора справочника.


		 ![](../../Property_Image.gif)
		 [SelectSearchValue](IRdsDictionaryElementsSearch.SelectSearchValue.htm)
		 Свойство SelectSearchValue
		 определяет условие поиска по значению.
