# IRdsDictionaryElementsSearch.IsIterator


## Синтаксис


IsIterator: Boolean;


## Описание


Свойство IsIterator возвращает
 признак того, что поиск выполняется среди элементов, полученных из итератора
 справочника.


## Комментарии


Допустимые значения:


	- True. Поиск выполняется
	 среди элементов, полученных из итератора справочника;


	- False. Поиск выполняется
	 по всем элементам справочника.


Для установки итератора справочника используйте метод [IRdsDictionaryElementsSearch.SetIterator](IRdsDictionaryElementsSearch.SetIterator.htm).
