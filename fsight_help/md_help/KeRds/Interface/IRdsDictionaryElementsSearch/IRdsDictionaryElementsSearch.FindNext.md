# IRdsDictionaryElementsSearch.FindNext


## Синтаксис


FindNext(Start: Integer): Integer;


## Параметры


Start. Ключ элемента, начиная
 с которого будет производиться поиск.


## Описание


Метод FindNext осуществляет
 поиск следующего элемента, удовлетворяющего условиям поиска и возвращает
 ключ найденного элемента в случае удачного поиска.


## Комментарии


Метод возвращает значение «-1» если элемент не найден.
