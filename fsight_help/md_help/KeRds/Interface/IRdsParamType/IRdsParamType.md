# IRdsParamType


## Описание


Интерфейс IRdsParamType содержит
 свойства типа параметров репозитория НСИ. Интерфейс является устаревшим.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Dictionary](IRdsParamType.Dictionary.htm)


		 Свойство Dictionary
		 определяет справочник НСИ, в котором содержится данный тип параметров.


		 ![](../../Property_Image.gif)
		 [Folder](IRdsParamType.Folder.htm)


		 Свойство Folder определяет
		 папку, в которой хранится данный тип параметров.
