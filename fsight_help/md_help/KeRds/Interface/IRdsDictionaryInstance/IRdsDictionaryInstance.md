# IRdsDictionaryInstance


Сборка: Rds;


## Описание


Интерфейс IRdsDictionaryInstance
 содержит свойства и методы для работы с данными справочника НСИ.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attributes](IRdsDictionaryInstance.Attributes.htm)
		 Свойство Attributes
		 возвращает параметры атрибутов открытого справочника.


		 ![](../../Property_Image.gif)
		 [BigElements](IRdsDictionaryInstance.BigElements.htm)
		 Свойство BigElements
		 возвращает объект, предназначенный для работы со справочниками,
		 поддерживающими большое количество элементов.


		 ![](../../Property_Image.gif)
		 [Dictionary](IRdsDictionaryInstance.Dictionary.htm)
		 Свойство Dictionary
		 возвращает параметры справочника НСИ.


		 ![](../../Property_Image.gif)
		 [Elements](IRdsDictionaryInstance.Elements.htm)
		 Свойство Elements возвращает
		 данные элементов справочника НСИ.


		 ![](../../Property_Image.gif)
		 [FetchAll](IRdsDictionaryInstance.FetchAll.htm)
		 Свойство FetchAll определяет,
		 загружать ли в кеш значения системных и пользовательских атрибутов
		 вместе.


		 ![](../../Property_Image.gif)
		 [Links](IRdsDictionaryInstance.Links.htm)
		 Свойство Links возвращает
		 данные связей справочника НСИ.


		 ![](../../Property_Image.gif)
		 [ParamValues](IRdsDictionaryInstance.ParamValues.htm)
		 Свойство ParamValues
		 возвращает коллекцию значений параметров, с которыми открыт справочник
		 НСИ.


		 ![](../../Property_Image.gif)
		 [ShowHidden](IRdsDictionaryInstance.ShowHidden.htm)
		 Свойство ShowHidden
		 определяет, включать ли скрытые элементы в данные справочника.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [AssignElements](IRdsDictionaryInstance.AssignElements.htm)

		 ![](../../Sub_Image.gif)
		 [CreateBatch](IRdsDictionaryInstance.CreateBatch.htm)
		 Метод CreateBatch создает

		 ![](../../Sub_Image.gif)
		 [ClearData](IRdsDictionaryInstance.ClearData.htm)
		 Метод ClearData удаляет
		 все элементы из справочника НСИ.


		 ![](../../Sub_Image.gif)
		 [CreateDimInstance](IRdsDictionaryInstance.CreateDimInstance.htm)
		 Метод CreateDimInstance
		 возвращает данные справочника на основе текущего набора элементов

		 ![](../../Sub_Image.gif)
		 [CreateElementData](IRdsDictionaryInstance.CreateElementData.htm)

		 Метод CreateLookup

		 по заданным значениям атрибутов.


		 ![](../../Sub_Image.gif)
		 [CreateSearch](IRdsDictionaryInstance.CreateSearch.htm)
		 Метод CreateSearch
		 создает объект, осуществляющий поиск элементов справочника НСИ
		 с расширенными возможностями поиска.


		 ![](../../Sub_Image.gif)
		 [Data](IRdsDictionaryInstance.Data.htm)
		 Метод Data возвращает
		 данные указанного элемента.


		 ![](../../Sub_Image.gif)
		 [Delete](IRdsDictionaryInstance.Delete.htm)

		 ![](../../Sub_Image.gif)
		 [ExportData](IRdsDictionaryInstance.ExportData.htm)
		 Метод ExportData осуществляет
		 экспорт данных справочника НСИ в соответствии с выбранной схемой
		 экспорта.


		 ![](../../Sub_Image.gif)
		 [ImportData](IRdsDictionaryInstance.ImportData.htm)
		 Метод ImportData осуществляет
		 импорт данных справочника НСИ в соответствии с выбранной схемой
		 импорта.


		 ![](../../Sub_Image.gif)
		 [Insert](IRdsDictionaryInstance.Insert.htm)
		 Метод Insert вставляет
		 новый элемент в справочник в соответствии с указанными параметрами.


		 ![](../../Sub_Image.gif)
		 [Update](IRdsDictionaryInstance.Update.htm)
		 Метод Update обновляет
		 данные указанного элемента.
