# IRdsDictionaryInstance.BigElements


## Синтаксис


BigElements: [IRdsBigElements](../IRdsBigElements/IRdsBigElements.htm);


## Описание


Свойство BigElements возвращает
 объект, предназначенный для работы со справочниками, поддерживающими большое
 количество элементов.


## Комментарии


Свойство актуально только для табличных справочников НСИ.


Для определения, поддерживает ли справочник работу с большим количеством
 элементов, используйте свойство [IRdsDictionary.IsBigDictionary](../IRdsDictionary/IRdsDictionary.IsBigDictionary.htm).
