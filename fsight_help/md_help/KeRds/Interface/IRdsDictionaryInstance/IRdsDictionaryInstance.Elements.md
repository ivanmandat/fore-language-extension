# IRdsDictionaryInstance.Elements


## Синтаксис


Elements: [IRdsDictionaryElements](../IRdsDictionaryElements/IRdsDictionaryElements.htm);


## Описание


Свойство Elements возвращает
 данные элементов справочника НСИ.


## Комментарии


По умолчанию доступны значения только системных атрибутов. Для получения
 значений системных и пользовательских атрибутов присвойте свойству [IRdsDictionaryInstance.FetchAll](IRdsDictionaryInstance.FetchAll.htm)
 значение True.
