# IRdsLinks.FindByAttribute


## Синтаксис


FindByAttribute(Attribute: [IRdsAttribute](../IRdsAttribute/IRdsAttribute.htm)):
 [IRdsLink](../IRdsLink/IRdsLink.htm);


## Параметры


Attribute - атрибут справочника
 НСИ, по которому осуществляется поиск.


## Описание


Метод FindByAttribute осуществляет
 поиск и возвращает связь справочника НСИ в случае удачного поиска.


## Комментарии


Атрибут справочника НСИ, по которому осуществляется поиск, передается
 посредством параметра Attribute.
