# IRdsLinks.FindById


## Синтаксис


FindById(Id: String): [IRdsLink](../IRdsLink/IRdsLink.htm);


## Параметры


Id - идентификатор связи, по
 которому осуществляется поиск.


## Описание


Метод FindById осуществляет
 поиск и возвращает связь справочника НСИ в случае удачного поиска.


## Комментарии


Идентификатор связи, по которому осуществляется поиск, передается посредством
 параметра Id.
