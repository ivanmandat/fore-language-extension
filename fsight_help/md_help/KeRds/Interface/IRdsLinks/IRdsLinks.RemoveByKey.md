# IRdsLinks.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key - ключ связи справочника
 НСИ, которую необходимо удалить.


## Описание


Метод RemoveByKey осуществляет
 удаление связи справочника НСИ, ключ которой передается посредством параметра
 Key, и возвращает True
 в случае удачного удаления.
