# IRdsLinks.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IRdsLink](../IRdsLink/IRdsLink.htm);


## Параметры


Key -
 ключ связи, по которому осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает связь справочника НСИ в случае удачного поиска.


## Комментарии


Ключ связи, по которому осуществляется поиск, передается посредством
 параметра Key.
