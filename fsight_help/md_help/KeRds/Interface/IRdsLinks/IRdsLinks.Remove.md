# IRdsLinks.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс связи справочника
 НСИ, которую необходимо удалить.


## Описание


Метод Remove осуществляет удаление
 связи справочника НСИ, индекс которой передается посредством параметра
 Index, и возвращает True
 в случае удачного удаления.
