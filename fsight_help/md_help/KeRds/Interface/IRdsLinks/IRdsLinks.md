# IRdsLinks


## Описание


Интерфейс IRdsLinks содержит
 свойства и методы коллекции связей справочника НСИ.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IRdsLinks.Count.htm)


		 Свойство Count возвращает
		 количество связей справочника НСИ.


		 ![](../../Property_Image.gif)
		 [Item](IRdsLinks.Item.htm)


		 Свойство Item возвращает
		 связь справочника НСИ.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IRdsLinks.Add.htm)


		 Метод Add осуществляет
		 создание связи справочника НСИ.


		 ![](../../Sub_Image.gif)
		 [Clear](IRdsLinks.Clear.htm)


		 Метод Clear осуществляет
		 удаление всех связей справочника НСИ.


		 ![](../../Sub_Image.gif)
		 [FindByAttribute](IRdsLinks.FindByAttribute.htm)


		 Метод FindByAttribute
		 осуществляет поиск связи справочника НСИ по связываемому атрибуту.


		 ![](../../Sub_Image.gif)
		 [FindById](IRdsLinks.FindById.htm)


		 Метод FindById осуществляет
		 поиск связи справочника НСИ по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IRdsLinks.FindByKey.htm)


		 Метод FindByKey осуществляет
		 поиск связи справочника НСИ по ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](IRdsLinks.Remove.htm)


		 Метод Remove осуществляет
		 удаление связи справочника НСИ по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IRdsLinks.RemoveByKey.htm)


		 Метод RemoveByKey осуществляет
		 удаление связи справочника НСИ по ключу.
