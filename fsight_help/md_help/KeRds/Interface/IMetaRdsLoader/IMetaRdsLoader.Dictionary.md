# IMetaRdsLoader.Dictionary


## Синтаксис


Dictionary: [IRdsDictionary](../IRdsDictionary/IRdsDictionary.htm);


## Описание


Свойство Dictionary определяет
 справочник НСИ, в который необходимо загрузить данные.


## Комментарии


В качестве значения свойства необходимо указать любой справочник НСИ.
 Если свойство не задано, то после выполнения метода [IMetaRdsLoader.CreateObject](IMetaRdsLoader.CreateObject.htm)
 будет создан новый справочник. Его параметры также будут доступны в свойстве
 Dictionary.
