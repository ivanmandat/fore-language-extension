# IMetaRdsLoader.TableSource


## Синтаксис


TableSource: [ITable](KeDb.chm::/Interface/ITable/ITable.htm);


## Описание


Свойство TableSource определяет
 таблицу репозитория, которая будет являться источником данных для справочника
 НСИ.


## Комментарии


Указываемая таблица должна располагаться в той базе данных, в которой
 расположен обновляемый/создаваемый справочник НСИ.
