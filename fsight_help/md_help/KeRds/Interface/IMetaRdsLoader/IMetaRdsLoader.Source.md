# IMetaRdsLoader.Source


## Синтаксис


Source: Object;


## Описание


Свойство Source определяет источник
 данных для справочника НСИ.


## Комментарии


В качестве значения данного свойства должен быть указан источник данных,
 описываемый интерфейсом [IDtProvider](KeDt.chm::/Interface/IDtProvider/IDtProvider.htm).
 Источник должен быть закрыт.
