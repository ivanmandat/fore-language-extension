# IMetaAttributeHierarchySetup


## Описание


Интерфейс IMetaAttributeHierarchySetup
 содержит свойства для настройки атрибута иерархии.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Attribute](IMetaAttributeHierarchySetup.Attribute.htm)


		 Свойство Attribute
		 возвращает настраиваемый атрибут.


		 ![](../../Property_Image.gif)
		 [Dimension](IMetaAttributeHierarchySetup.Dimension.htm)


		 Свойство Dimension
		 определяет измерение атрибута.


		 ![](../../Property_Image.gif)
		 [IncludeSelectedOnly](IMetaAttributeHierarchySetup.IncludeSelectedOnly.htm)


		 Свойство IncludeSelectedOnly
		 определяет, включать ли в иерархию только отмеченные элементы
		 измерения.


		 ![](../../Property_Image.gif)
		 [NamingFormat](IMetaAttributeHierarchySetup.NamingFormat.htm)


		 Свойство NamingFormat
		 определяет формат наименования атрибута.


		 ![](../../Property_Image.gif)
		 [Selection](IMetaAttributeHierarchySetup.Selection.htm)


		 Свойство Selection
		 определяет отметку измерения атрибута.
