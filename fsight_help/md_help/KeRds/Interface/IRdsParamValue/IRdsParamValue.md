# IRdsParamValue


## Описание


Интерфейс IRdsParamValue содержит
 свойства для указания значения параметра, с которым осуществляется открытие
 справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Param](IRdsParamValue.Param.htm)


		 Свойство Param возвращает
		 свойства параметра справочника НСИ.


		 ![](../../Property_Image.gif)
		 [PredefinedType](IRdsParamValue.PredefinedType.htm)


		 Свойство PredefinedType
		 возвращает назначение системного параметра.


		 ![](../../Property_Image.gif)
		 [Value](IRdsParamValue.Value.htm)


		 Свойство Value определяет
		 значение параметра, с которым будет осуществляться открытие справочника.
