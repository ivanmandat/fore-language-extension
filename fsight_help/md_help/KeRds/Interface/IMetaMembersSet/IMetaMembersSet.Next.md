# IMetaMembersSet.Next


## Синтаксис


Next;


## Описание


Метод Next осуществляет переход
 к следующему элементу в наборе.


## Комментарии


Для проверки, является ли текущий элемент последним в наборе, используйте
 метод [IMetaMembersSet.Eof](IMetaMembersSet.Eof.htm).
