# IMetaMembersSet


Сборка: Rds;


## Описание


Интерфейс IMetaMembersSet предназначен
 для работы с набором элементов системного справочника.


## Иерархия наследования


           IMetaMembersSet


## Комментарии


Для работы с деревом элементов системного справочника используйте свойства
 интерфейса [IMetaMembersTree](../IMetaMembersTree/IMetaMembersTree.htm).


## Свойства


		 Имя метода
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CurrentOffset](IMetaMembersSet.CurrentOffset.htm)


		 Свойство CurrentOffset
		 возвращает число уровней иерархии от корня справочника до набора
		 элементов.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Current](IMetaMembersSet.Current.htm)


		 Метод Current возвращает
		 текущий элемент в наборе.


		 ![](../../Sub_Image.gif)
		 [Eof](IMetaMembersSet.Eof.htm)


		 Метод Eof возвращает
		 признак, является ли текущий элемент последним в наборе.


		 ![](../../Sub_Image.gif)
		 [Next](IMetaMembersSet.Next.htm)


		 Метод Next осуществляет
		 переход к следующему элементу в наборе.


		 ![](../../Sub_Image.gif)
		 [Reset](IMetaMembersSet.Reset.htm)


		 Метод Reset сбрасывает
		 текущее положение курсора в наборе элементов.
