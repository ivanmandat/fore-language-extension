# IMetaMembersSet.Eof


## Синтаксис


Eof: Boolean;


## Описание


Метод Eof возвращает признак,
 является ли текущий элемент последним в наборе.


## Комментарии


Допустимые значения:


	- True. Курсор находится
	 на последней записи;


	- False. Конец данных
	 еще не достигнут.
