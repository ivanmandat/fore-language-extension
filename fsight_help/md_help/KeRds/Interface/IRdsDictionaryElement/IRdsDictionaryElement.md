# IRdsDictionaryElement


Сборка: Rds;


## Описание


Интерфейс IRdsDictionaryElement
 содержит свойства и методы элемента справочника НСИ.


## Иерархия наследования


           [IRdsDictionaryElementData](../IRdsDictionaryElementData/IRdsDictionaryElementData.htm)


           IRdsDictionaryElement


## Комментарии


В справочниках НСИ поддерживаются ключи элементов, входящие в диапазон
 от 0 до 2 млрд.

## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [CreateSorter](IRdsDictionaryElement.CreateSorter.htm)


		 Метод CreateSorter
		 создает объект, осуществляющий сортировку дочерних элементов выбранного
		 элемента справочника НСИ.


		 ![](../../Sub_Image.gif)
		 [Delete](IRdsDictionaryElement.Delete.htm)


		 Метод Delete осуществляет
		 удаление текущего элемента.


		 ![](../../Sub_Image.gif)
		 [DeleteRaw](IRdsDictionaryElement.DeleteRaw.htm)


		 Метод DeleteRaw осуществляет
		 удаление текущего элемента в соответствии с заданными опциями
		 удаления.


		 ![](../../Sub_Image.gif)
		 [InsertChild](IRdsDictionaryElement.InsertChild.htm)


		 Метод InsertChild создает
		 новый дочерний элемент для данного элемента.


		 ![](../../Sub_Image.gif)
		 [InsertChildCopy](IRdsDictionaryElement.InsertChildCopy.htm)


		 Метод InsertChildCopy
		 создает новый дочерний элемент для данного элемента на основе
		 данных другого элемента.


		 ![](../../Sub_Image.gif)
		 [Move](IRdsDictionaryElement.Move.htm)


		 Метод Move осуществляет
		 перестановку текущего элемента с указанным элементом.


		 ![](../../Sub_Image.gif)
		 [MoveElement](IRdsDictionaryElement.MoveElement.htm)


		 Метод MoveElement осуществляет
		 перемещение элемента справочника НСИ в позицию, указанную относительно
		 заданного элемента.


		 ![](../../Sub_Image.gif)
		 [MoveTo](IRdsDictionaryElement.MoveTo.htm)


		 Метод MoveTo осуществляет
		 перемещение элемента справочника НСИ в позицию, указанную относительно
		 элемента с заданным ключом.


		 ![](../../Sub_Image.gif)
		 [SetParent](IRdsDictionaryElement.SetParent.htm)


		 Метод SetParent осуществляет
		 изменение родительского элемента для текущего элемента.


		 ![](../../Sub_Image.gif)
		 [Update](IRdsDictionaryElement.Update.htm)


		 Метод Update осуществляет
		 обновление данных элемента.
