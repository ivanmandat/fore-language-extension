# IRdsDictionaryElement.HasAccess


## Синтаксис


HasAccess(Attribute: [RdsAccessAttribute](../../Enums/RdsAccessAttribute.htm)): Boolean;


## Параметры


Attribute. Вид атрибута доступа, определяющего права, наличие которых необходимо проверить для текущего пользователя.


## Описание


Свойство HasAccess возвращает признак наличия прав у текущего пользователя. Вид атрибута доступа, по которому осуществляется проверка наличия прав, передается посредством параметра Attribute.
