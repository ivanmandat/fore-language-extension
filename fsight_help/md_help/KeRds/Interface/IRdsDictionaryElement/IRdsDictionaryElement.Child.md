# IRdsDictionaryElement.Child


## Синтаксис


Child(ChildIndex: Integer): [IRdsDictionaryElement](IRdsDictionaryElement.htm);

ChildIndex. Индекс дочернего элемента.


## Описание


Свойство Child возвращает дочерний элемент, индекс которого передается посредством параметра ChildIndex.
