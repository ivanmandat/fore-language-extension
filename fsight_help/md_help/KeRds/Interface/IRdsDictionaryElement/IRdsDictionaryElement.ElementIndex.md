# IRdsDictionaryElement.ElementIndex


## Синтаксис


ElementIndex: Integer;


## Описание


Свойство ElementIndex возвращает
 индекс элемента.


## Комментарии


ElementIndex всегда возвращает
 неотрицательное число. Если ElementIndex
 = 0, то данный элемент является корневым элементом справочника.
