# IRdsUniqueKeys.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IRdsUniqueKey](../IRdsUniqueKey/IRdsUniqueKey.htm);


## Параметры


Key. Ключ искомого уникального
 ключа.


## Описание


Метод FindByKey осуществляет
 поиск уникального ключа по ключу в справочнике НСИ.


## Комментарии


Если уникальный ключ не найден, то метод возвращает значение Null.
