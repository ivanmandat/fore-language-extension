# IRdsUniqueKeys.FindById


## Синтаксис


FindById(Id: String): [IRdsUniqueKey](../IRdsUniqueKey/IRdsUniqueKey.htm);


## Параметры


Id. Идентификатор уникального
 ключа, по которому осуществляется поиск.


## Описание


Метод FindById осуществляет
 поиск и возвращает уникальный ключ справочника НСИ в случае удачного поиска.
 Идентификатор, по которому осуществляется поиск, передается посредством
 параметра Id.
