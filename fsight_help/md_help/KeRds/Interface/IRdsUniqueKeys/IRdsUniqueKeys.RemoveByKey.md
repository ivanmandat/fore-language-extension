# IRdsUniqueKeys.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key. Ключ уникального ключа
 справочника НСИ.


## Описание


Метод RemoveByKey осуществляет
 удаление уникального ключа по ключу в справочнике НСИ.


## Комментарии


Если удаление выполнено успешно, то метод возвращает значение True, в обратном случае - False.
