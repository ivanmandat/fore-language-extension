# IMetaRdsExportSchema.MappingsCount


## Синтаксис


MappingsCount: Integer;


## Описание


Свойство MappingsCount возвращает
 количество привязок атрибутов к полям источника данных.


## Комментарии


Для добавления привязки используйте метод [IMetaRdsExportSchema.AddMapping](IMetaRdsExportSchema.AddMapping.htm),
 для удаления - метод [IMetaRdsExportSchema.RemoveMapping](IMetaRdsExportSchema.RemoveMapping.htm).
