# IMetaRdsExportSchema.SortAttributes


## Синтаксис


SortAttributes: [IRdsSortAttributes](../IRdsSortAttributes/IRdsSortAttributes.htm);


## Описание


Свойство SortAttributes возвращает
 параметры сортировки, применяемые к экспортируемым элементам справочника.


## Комментарии


По умолчанию сортировка экспортируемых элементов не выполняется.
