# IMetaRdsExportSchema.Filter


## Синтаксис


Filter: [IRdsDictionaryFilterConditions](../IRdsDictionaryFilterConditions/IRdsDictionaryFilterConditions.htm);


## Описание


Свойство Filter возвращает
 фильтр, применяемый к экспортируемым элементам справочника.


## Комментарии


По умолчанию фильтрация не производится и экспортируются все элементы.
