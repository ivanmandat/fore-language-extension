# IMetaRdsExportSchema.AddMapping


## Синтаксис


AddMapping: [IRdsExportSchemaAttribute](../IRdsExportSchemaAttribute/IRdsExportSchemaAttribute.htm);


## Описание


Метод AddMapping создает привязку
 атрибута справочника к полю приёмника данных.


## Комментарии


Для удаления привязки атрибута используйте метод [IMetaRdsExportSchema.RemoveMapping](IMetaRdsExportSchema.RemoveMapping.htm).
