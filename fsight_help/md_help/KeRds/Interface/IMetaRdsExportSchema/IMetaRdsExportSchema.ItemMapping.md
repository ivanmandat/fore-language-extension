# IMetaRdsExportSchema.ItemMapping


## Синтаксис


ItemMapping(Index: Integer): [IRdsExportSchemaAttribute](../IRdsExportSchemaAttribute/IRdsExportSchemaAttribute.htm);


## Параметры


Index. Индекс привязки.


## Описание


Свойство ItemMapping возвращает
 указанную привязку атрибута к полю источника данных.


## Комментарии


Количество привязок возвращает свойство [IMetaRdsExportSchema.MappingsCount](IMetaRdsExportSchema.MappingsCount.htm).
