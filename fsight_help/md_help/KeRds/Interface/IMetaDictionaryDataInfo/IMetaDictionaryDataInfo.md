# IMetaDictionaryDataInfo


Сборка: Rds;


## Описание


Интерфейс IMetaDictionaryDataInfo
 предназначен для работы с информацией о показателе базы данных временных
 рядов.


## Иерархия наследования


           IMetaDictionaryDataInfo


## Комментарии


Для получения информации о показателе приведите объект интерфейса [IMetaDictionaryData](../IMetaDictionaryData/IMetaDictionaryData.htm)
 или [IRubricatorFactData](KeCubes.chm::/Interface/IRubricatorFactData/IRubricatorFactData.htm)
 к интерфейсу IMetaDictionaryDataInfo.
