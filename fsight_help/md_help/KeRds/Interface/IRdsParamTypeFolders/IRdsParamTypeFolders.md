# IRdsParamTypeFolders


## Описание


Интерфейс IRdsParamTypeFolders
 содержит свойства и методы коллекции папок типов параметров репозитория
 НСИ. Интерфейс является устаревшим.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IRdsParamTypeFolders.Count.htm)


		 Свойство Count возвращает
		 количество папок в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IRdsParamTypeFolders.Item.htm)


		 Свойство Item возвращает
		 содержимое папки.
