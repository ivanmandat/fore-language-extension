# IRdsDictionaryBatch.Execute


## Синтаксис


Execute: Integer;


## Описание


Метод Execute выполняет действие,
 предусмотренное пакетом.


## Комментарии


После выполнения метод возвращает число обработанных элементов пакета.
 Число элементов, к которым было применено действия возвращает свойство
 [IRdsDictionaryBatch.Count](IRdsDictionaryBatch.Count.htm).


Действие, выполняемое над элементами пакета, зависит от типа пакета.
 Тип пакета возвращает свойство [IRdsDictionaryBatch.BatchType](IRdsDictionaryBatch.BatchType.htm).
