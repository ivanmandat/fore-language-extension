# IRdsDictionaryFilterConditions.FindByAttribute


## Синтаксис


FindByAttribute(Attribute: [IRdsAttribute](../IRdsAttribute/IRdsAttribute.htm)):
 [IRdsDictionaryFilterCondition](../IRdsDictionaryFilterCondition/IRdsDictionaryFilterCondition.htm);


## Параметры


Attribute. Атрибут.


## Описание


Метод FindByAttribute возвращает
 фильтр для заданного атрибута.


## Комментарии


Если фильтрация по заданному атрибуту не выполняется, то метод возвращает
 значение Null.
