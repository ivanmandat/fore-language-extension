# IRdsDictionaryFilterConditions.SaveTo


## Синтаксис


SaveTo(Element: Variant);


## Параметры


Element. Объект, в который
 будут сохранены параметры фильтра в виде XML-кода.


## Описание


Метод SaveTo сохраняет коллекцию
 фильтров в формате XML.


## Комментарии


Для загрузки параметров фильтра из XML используйте [IRdsDictionaryFilterConditions.LoadFrom](IRdsDictionaryFilterConditions.LoadFrom.htm).
