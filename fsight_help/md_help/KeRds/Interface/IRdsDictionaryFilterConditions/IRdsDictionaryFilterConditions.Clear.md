# IRdsDictionaryFilterConditions.Clear


## Синтаксис


Clear;


## Описание


Метод Clear очищает коллекцию
 фильтров.


## Комментарии


После вызова Clear значение
 свойства [IRdsDictionaryFilterConditions.Count](IRdsDictionaryFilterConditions.Count.htm)
 становится равным нулю.
