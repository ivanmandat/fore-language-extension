# IRdsDictionaryFilterConditions


Сборка: Rds;


## Описание


Интерфейс IRdsDictionaryFilterConditions
 предназначен для работы с коллекцией фильтров, по которым отбираются элементы
 справочника.


## Иерархия наследования


           IEnumerable


           IRdsDictionaryFilterConditions


## Комментарии


Каждый фильтр в коллекции реализован интерфейсом [IRdsDictionaryFilterCondition](../IRdsDictionaryFilterCondition/IRdsDictionaryFilterCondition.htm).
