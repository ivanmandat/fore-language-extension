# IRdsNotConvertibleReason.Reason


## Синтаксис


Reason: [RdsNotConvertibleReasonCode](../../Enums/RdsNotConvertibleReasonCode.htm);


## Описание


Свойство Reason возвращает числовой
 идентификатор причины, по которой справочник НСИ не может быть преобразован
 в табличный справочник НСИ.


## Комментарии


Для выполнения объяснения причины, по которой справочник НСИ не может
 быть преобразован в табличный справочник НСИ, используйте метод [IRdsNotConvertibleReason.GetExplanation](IRdsNotConvertibleReason.GetExplanation.htm).
