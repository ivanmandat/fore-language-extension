# IRdsNotConvertibleReason.GetExplanation


## Синтаксис


GetExplanation: String;


## Описание


Метод GetExplanation выполняет
 объяснение причины, по которой справочник НСИ не может быть преобразован
 в табличный справочник НСИ.


## Комментарии


Для получения числового идентификатора причины, по которой справочник
 НСИ не может быть преобразован в табличный справочник НСИ, используйте
 свойство [IRdsNotConvertibleReason.Reason](IRdsNotConvertibleReason.Reason.htm).
