# IRdsNotConvertibleReason


Сборка: Rds;


## Описание


Интерфейс IRdsNotConvertibleReason
 предназначен для работы с причиной, по которой справочник НСИ не может
 быть преобразован в табличный справочник НСИ.


## Иерархия наследования


           IRdsNotConvertibleReason


## Комментарии


Для работы с коллекцией причин, по которым справочник НСИ не может быть
 преобразован в табличный справочник НСИ, используйте свойства и методы
 интерфейса [IRdsNotConvertibleReasons](../IRdsNotConvertibleReasons/IRdsNotConvertibleReasons.htm).
 Ограничения и особенности преобразования справочников перечислены в интерфейсе
 [IRdsDictionaryConverter](../IRdsDictionaryConverter/IRdsDictionaryConverter.htm).
