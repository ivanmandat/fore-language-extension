# IRdsParamsControl


## Описание


Интерфейс IRdsParamsControl
 содержит свойства и методы для настройки привязки параметров параметрического
 справочника при установке с ним связи.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Expression](IRdsParamsControl.Expression.htm)
		 Свойство Expression
		 возвращает выражение, определяющее постоянное значение параметра
		 связываемого справочника.


		 ![](../../Property_Image.gif)
		 [Link](IRdsParamsControl.Link.htm)
		 Свойство Link возвращает
		 объект-родитель.


		 ![](../../Property_Image.gif)
		 [ParamAttribute](IRdsParamsControl.ParamAttribute.htm)
		 Свойство ParamAttribute
		 определяет привязку параметра связываемого справочника с атрибутом
		 текущего справочника, через значения которого будет осуществляться
		 управление.


		 ![](../../Property_Image.gif)
		 [ReferenceDictionary](IRdsParamsControl.ReferenceDictionary.htm)
		 Свойство ReferenceDictionary
		 возвращает связываемый справочник.


		 ![](../../Property_Image.gif)
		 [ValueSelect](IRdsParamsControl.ValueSelect.htm)
		 Свойство ValueSelect
		 определяет алгоритм выбора значения атрибута при множественной
		 отметке для [привязки](UiNavObj.chm::/reference_book/Master_RDS_reference_book/Link_Property.htm#add)
		 значений атрибута редактируемого справочника к параметру связываемого
		 справочника.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Reset](IRdsParamsControl.Reset.htm)
		 Метод Reset осуществляет
		 сброс привязки всех параметров параметрического справочника.
