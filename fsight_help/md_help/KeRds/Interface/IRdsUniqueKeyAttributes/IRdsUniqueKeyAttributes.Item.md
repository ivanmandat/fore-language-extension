# IRdsUniqueKeyAttributes.Item


## Синтаксис


Item(Index: Integer): [IRdsUniqueKeyAttribute](../IRdsUniqueKeyAttribute/IRdsUniqueKeyAttribute.htm);


## Параметры


Index. Индекс атрибута.


## Описание


Свойство Item возвращает атрибут
 уникального ключа справочника НСИ.


## Комментарии


Для получения количества атрибутов уникального ключа справочника НСИ,
 в коллекции используйте [IRdsUniqueKeyAttributes.Count](IRdsUniqueKeyAttributes.Count.htm).
