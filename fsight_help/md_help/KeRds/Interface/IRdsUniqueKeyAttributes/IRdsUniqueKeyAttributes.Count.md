# IRdsUniqueKeyAttributes.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 атрибутов уникального ключа справочника НСИ.


## Комментарии


Для получения атрибута уникального ключа справочника НСИ, по его индексу
 используйте [IRdsUniqueKeyAttributes.Item](IRdsUniqueKeyAttributes.Item.htm).
