# IRdsUniqueKeyAttributes.Move


## Синтаксис


Move(IndexFrom: Integer, IndexTo: Integer);


## Параметры


IndexFrom. Индекс позиции,
 из которой необходимо переместить атрибут;


IndexTo. Индекс позиции, в
 которую необходимо переместить атрибут.


## Описание


Метод Move осуществляет перемещение
 атрибута внутри коллекции.


## Комментарии


Для работы с атрибутом уникального ключа используйте интерфейс [IRdsUniqueKeyAttribute](../IRdsUniqueKeyAttribute/IRdsUniqueKeyAttribute.htm).
