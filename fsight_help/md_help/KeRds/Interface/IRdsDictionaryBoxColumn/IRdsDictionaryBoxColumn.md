# IRdsDictionaryBoxColumn


## Описание


Интерфейс IRdsDictionaryBoxColumn
 содержит свойства столбца справочника НСИ, отображаемого в компоненте
 [RdsDictionaryBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/RdsDictionaryBox.htm).
