# IRdsDictionaryElementDependents.IsEmptyWithChildren


## Синтаксис


IsEmptyWithChildren: Boolean;


## Описание


Метод IsEmptyWithChildren осуществляет
 проверку наличия зависимостей от элемента [Element](IRdsDictionaryElementDependents.Element.htm),
 а также зависимостей от всех дочерних элементов.


## Комментарии


Данный метод возвращает значение True,
 если отсутствуют зависимости от текущего элемента и от всех дочерних элементов.
 Может применяться для быстрой проверки наличия зависимостей при каких-либо
 действиях с коллекцией.
