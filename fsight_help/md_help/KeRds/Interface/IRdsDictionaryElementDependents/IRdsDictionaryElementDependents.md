# IRdsDictionaryElementDependents


Сборка: Rds;


## Описание


Интерфейс IRdsDictionaryElementDependents
 содержит свойства и методы для работы с коллекцией зависимостей от элемента
 справочника НСИ.


## Иерархия наследования


           IRdsDictionaryElementDependents


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Element](IRdsDictionaryElementDependents.Element.htm)


		 Свойство Element возвращает
		 родительский элемент, для которого получена коллекция зависимостей.


		 ![](../../Property_Image.gif)
		 [IsEmpty](IRdsDictionaryElementDependents.IsEmpty.htm)


		 Метод IsEmpty осуществляет
		 проверку наличия зависимостей от элемента [Element](IRdsDictionaryElementDependents.Element.htm).


		 ![](../../Property_Image.gif)
		 [IsEmptyWithChildren](IRdsDictionaryElementDependents.IsEmptyWithChildren.htm)


		 Метод IsEmptyWithChildren
		 осуществляет проверку наличия зависимостей от элемента [Element](IRdsDictionaryElementDependents.Element.htm),
		 а также зависимостей от всех дочерних элементов.


		 ![](../../Property_Image.gif)
		 [Objects](IRdsDictionaryElementDependents.Objects.htm)


		 Свойство Objects возвращает
		 коллекцию описаний объектов, зависящих от элемента [Element](IRdsDictionaryElementDependents.Element.htm).


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Edit](IRdsDictionaryElementDependents.Edit.htm)


		 Метод Edit открывает
		 коллекцию зависимостей от элемента на редактирование.


		 ![](../../Sub_Image.gif)
		 [Save](IRdsDictionaryElementDependents.Save.htm)


		 Метод Save осуществляет
		 сохранение изменений в коллекции зависимостей.


		 ![](../../Sub_Image.gif)
		 [SelectAllWithChildren](IRdsDictionaryElementDependents.SelectAllWithChildren.htm)


		 Метод SelectAllWithChildren
		 возвращает коллекцию зависимостей от элемента [Element](IRdsDictionaryElementDependents.Element.htm)
		 и всех его дочерних элементов.
