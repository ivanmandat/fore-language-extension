# IRdsDictionaryElementDependents.IsEmpty


## Синтаксис


IsEmpty: Boolean;


## Описание


Метод IsEmpty осуществляет проверку
 наличия зависимостей от элемента [Element](IRdsDictionaryElementDependents.Element.htm).


## Комментарии


Данный метод возвращает значение True,
 если текущая коллекция зависимостей, полученная для элемента, пуста. Может
 применяться для быстрой проверки наличия зависимостей при каких-либо действиях
 с коллекцией.
