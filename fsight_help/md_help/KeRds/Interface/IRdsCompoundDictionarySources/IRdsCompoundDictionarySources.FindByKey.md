# IRdsCompoundDictionarySources.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IRdsCompoundDictionarySource](../IRdsCompoundDictionarySource/IRdsCompoundDictionarySource.htm);


## Параметры


Key - ключ источника, по которому
 осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает источник в случае удачного поиска. Ключ, по которому
 осуществляется поиск, передается посредством параметра Key.
