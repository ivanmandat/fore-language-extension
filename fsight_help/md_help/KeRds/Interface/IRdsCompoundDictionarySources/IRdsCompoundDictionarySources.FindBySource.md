# IRdsCompoundDictionarySources.FindBySource


## Синтаксис


FindBySource(Source: [IRdsDictionary](../IRdsDictionary/IRdsDictionary.htm)):
 [IRdsCompoundDictionarySource](../IRdsCompoundDictionarySource/IRdsCompoundDictionarySource.htm);


## Параметры


Source - справочник НСИ, который
 необходимо найти в коллекции источников данного составного справочника.


## Описание


Метод FindBySource осуществляет
 поиск и возвращает источник в случае удачного поиска. Справочник НСИ,
 который необходимо найти в коллекции источников, передается посредством
 параметра Source.
