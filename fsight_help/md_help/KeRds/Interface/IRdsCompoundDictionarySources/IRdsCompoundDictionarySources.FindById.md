# IRdsCompoundDictionarySources.FindById


## Синтаксис


FindById(Id: String): [IRdsCompoundDictionarySource](../IRdsCompoundDictionarySource/IRdsCompoundDictionarySource.htm);


## Параметры


Id - идентификатор источника,
 по которому осуществляется поиск.


## Описание


Метод FindById осуществляет
 поиск и возвращает источник в случае удачного поиска. Идентификатор, по
 которому осуществляется поиск, передается посредством параметра Id.
