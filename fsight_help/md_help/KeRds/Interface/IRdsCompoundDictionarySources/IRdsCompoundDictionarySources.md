# IRdsCompoundDictionarySources


## Описание


Интерфейс IRdsCompoundDictionarySources
 содержит свойства и методы для работы с коллекцией источников составного
 справочника НСИ.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IRdsCompoundDictionarySources.Count.htm)


		 Свойство Count возвращает
		 количество источников у данного составного справочника НСИ.


		 ![](../../Property_Image.gif)
		 [Item](IRdsCompoundDictionarySources.Item.htm)


		 Свойство Item возвращает
		 источник данных.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IRdsCompoundDictionarySources.Add.htm)


		 Метод Add осуществляет
		 создание источника данных.


		 ![](../../Sub_Image.gif)
		 [Clear](IRdsCompoundDictionarySources.Clear.htm)


		 Метод Clear осуществляет
		 удаление всех источников данных.


		 ![](../../Sub_Image.gif)
		 [FindById](IRdsCompoundDictionarySources.FindById.htm)


		 Метод FindById осуществляет
		 поиск источника по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IRdsCompoundDictionarySources.FindByKey.htm)


		 Метод FindByKey осуществляет
		 поиск источника по ключу.


		 ![](../../Sub_Image.gif)
		 [FindBySource](IRdsCompoundDictionarySources.FindBySource.htm)


		 Метод FindBySource
		 осуществляет поиск справочника среди источников.


		 ![](../../Sub_Image.gif)
		 [Remove](IRdsCompoundDictionarySources.Remove.htm)


		 Метод Remove осуществляет
		 удаление источника по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IRdsCompoundDictionarySources.RemoveByKey.htm)


		 Метод RemoveByKey осуществляет
		 удаление источника по ключу.
