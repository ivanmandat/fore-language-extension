# IRdsAttributesInstance


## Описание


Интерфейс IRdsAttributesInstance
 содержит свойства и методы для работы с коллекцией атрибутов открытого
 справочника.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IRdsAttributesInstance.Count.htm)
		 Свойство Count возвращает
		 количество атрибутов.


		 ![](../../Property_Image.gif)
		 [Item](IRdsAttributesInstance.Item.htm)
		 Свойство Item возвращает
		 атрибут справочника.


		 ![](../../Property_Image.gif)
		 [Predefined](IRdsAttributesInstance.Predefined.htm)
		 Свойство Predefined
		 возвращает указанный системный атрибут справочника НСИ.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [FindById](IRdsAttributesInstance.FindById.htm)
		 Метод FindById осуществляет
		 поиск атрибута справочника НСИ по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IRdsAttributesInstance.FindByKey.htm)
		 Метод FindByKey осуществляет
		 поиск атрибута справочника НСИ по ключу.
