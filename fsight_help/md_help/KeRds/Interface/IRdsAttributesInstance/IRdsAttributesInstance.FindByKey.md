# IRdsAttributesInstance.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IRdsAttributeInstance](../IRdsAttributeInstance/IRdsAttributeInstance.htm);


## Параметры


Key - ключ атрибута, по которому
 осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает атрибут справочника НСИ в случае удачного поиска. Ключ,
 по которому осуществляется поиск, передается посредством параметра Key.
