# IRdsLevels


## Описание


Интерфейс IRdsLevels содержит
 свойства и методы для работы с коллекцией уровней справочника НСИ.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IRdsLevels.Count.htm)


		 Свойство Count возвращает
		 количество уровней справочника НСИ.


		 ![](../../Property_Image.gif)
		 [Item](IRdsLevels.Item.htm)


		 Свойство Item возвращает
		 уровень справочника НСИ.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IRdsLevels.Add.htm)


		 Метод Add добавляет
		 уровень в справочник НСИ.


		 ![](../../Sub_Image.gif)
		 [FindById](IRdsLevels.FindById.htm)


		 Метод FindById осуществляет
		 поиск уровня справочника НСИ по идентификатору.


		 ![](../../Sub_Image.gif)
		 [FindByKey](IRdsLevels.FindByKey.htm)


		 Метод FindByKey осуществляет
		 поиск уровня справочника НСИ по ключу.


		 ![](../../Sub_Image.gif)
		 [Remove](IRdsLevels.Remove.htm)


		 Метод Remove осуществляет
		 удаление уровня справочника НСИ по индексу.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IRdsLevels.RemoveByKey.htm)


		 Метод RemoveByKey осуществляет
		 удаление уровня справочника НСИ по ключу.
