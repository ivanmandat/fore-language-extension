# IRdsLevels.FindById


## Синтаксис


FindById(Id: String): [IRdsLevel](../IRdsLevel/IRdsLevel.htm);


## Параметры


Id - идентификатор
 уровня, по которому осуществляется поиск.


## Описание


Метод FindById осуществляет
 поиск и возвращает уровень справочника НСИ, в случае удачного поиска.
 Идентификатор, по которому осуществляется поиск, передается посредством
 параметра Id.
