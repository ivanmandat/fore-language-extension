# IRdsLevels.RemoveByKey


## Синтаксис


RemoveByKey(Key: Integer): Boolean;


## Параметры


Key - ключ уровня справочника
 НСИ.


## Описание


Метод RemoveByKey осуществляет
 удаление уровня справочника НСИ, ключ которого передается посредством
 параметра Key, и возвращает True в случае удачного удаления.
