# IRdsLevels.FindByKey


## Синтаксис


FindByKey(Key: Integer): [IRdsLevel](../IRdsLevel/IRdsLevel.htm);


## Параметры


Key - ключ уровня, по которому
 осуществляется поиск.


## Описание


Метод FindByKey осуществляет
 поиск и возвращает уровень справочника НСИ, в случае удачного поиска.
 Ключ, по которому осуществляется поиск, передается посредством параметра
 Key.
