# IRdsLevels.Remove


## Синтаксис


Remove(Index: Integer): Boolean;


## Параметры


Index - индекс уровня справочника
 НСИ.


## Описание


Метод Remove осуществляет удаление
 уровня справочника НСИ, индекс которого передается посредством параметра
 Index, и возвращает True
 в случае удачного удаления.
