# IRdsDictionaryElements.Segment


## Синтаксис


Segment(Element: Integer): [IRdsSegment](../IRdsSegment/IRdsSegment.htm);


## Параметры


Element - ключ элемента для которого необходимо определить сегмент справочника.


## Описание


Свойство Segment определяет сегмент для элемента справочника НСИ, ключ которого передается посредством параметра Element.
