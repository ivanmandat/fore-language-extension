# IRdsDictionaryElements.Parent


## Синтаксис


Parent(Element: Integer): Integer;


## Параметры


Element - ключ элемента.


## Описание


Свойство Parent возвращает ключ родительского элемента для элемента, ключ которого передается посредством параметра Element.
