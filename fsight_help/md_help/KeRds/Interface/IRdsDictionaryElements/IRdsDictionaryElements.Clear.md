# IRdsDictionaryElements.Clear


## Синтаксис


Clear([ForceClearReferences: Boolean = False]);


## Параметры


ForceClearReferences - необязательный параметр, определяющий, будет ли осуществляться принудительное удаление элементов если на них существуют ссылки в других справочника, связанных с данным. По умолчанию свойству установлено значение False, при этом если есть ссылки на элементы - генерируется исключительная ситуация. Если свойству установить значение True - элементы будут удаляться принудительно, не приводя к удалению связанных элементов в других справочниках.


## Описание


Метод Clear осуществляет удаление всех элементов коллекции с учетом параметра ForceClearReferences.
