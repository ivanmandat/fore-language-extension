# IRdsDictionaryElements.CreateSorter


## Синтаксис


CreateSorter:(ParentKey: Integer): [IRdsElementsSorter](../IRdsElementsSorter/IRdsElementsSorter.htm);


## Параметры


ParentKey. Ключ родительского
 элемента, для которого осуществляется сортировка дочерних элементов.


## Описание


Метод CreateSorter создает объект,
 осуществляющий сортировку дочерних элементов элемента, передаваемого посредством
 параметра ParentKey.
