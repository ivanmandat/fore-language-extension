# IRdsDictionaryElements.ItemElement


## Синтаксис


ItemElement(Element: Integer): [IRdsDictionaryElement](../IRdsDictionaryElement/IRdsDictionaryElement.htm);


## Параметры


Element - ключ элемента. Корень дерева элементов имеет ключ - "-2".


## Описание


Свойство ItemElement возвращает элемент справочника НСИ, ключ которого передается посредством параметра Element.
