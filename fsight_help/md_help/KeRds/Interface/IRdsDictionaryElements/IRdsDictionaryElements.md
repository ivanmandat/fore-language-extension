# IRdsDictionaryElements


Сборка: Rds;


## Описание


Интерфейс IRdsDictionaryElements
 содержит свойства и методы для работы с коллекцией элементов справочника
 НСИ.


## Иерархия наследования


           IRdsDictionaryElements


## Комментарии


В справочниках НСИ поддерживаются ключи элементов, входящие в диапазон
 от 0 до 2 млрд.
