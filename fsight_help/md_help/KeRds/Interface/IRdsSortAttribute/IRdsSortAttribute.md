# IRdsSortAttribute


## Описание


Интерфейс IRdsSortAttribute
 предназначен для работы с атрибутами
 сортировки элементов справочника НСИ.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Ascending](IRdsSortAttribute.Ascending.htm)


		 Свойство Ascending
		 определяет направление сортировки элементов справочника НСИ.


		 ![](../../Property_Image.gif)
		 [Attribute](IRdsSortAttribute.Attribute.htm)


		 Свойство Attribute
		 возвращает атрибут сортировки элементов справочника НСИ.


		 ![](../../Property_Image.gif)
		 [Index](IRdsSortAttribute.Index.htm)


		 Свойство Index возвращает
		 индекс атрибута сортировки в коллекции.


		 ![](../../Property_Image.gif)
		 [RefAttribute](IRdsSortAttribute.RefAttribute.htm)


		 Свойство RefAttribute
		 определяет связанный атрибут сортировки элементов справочника
		 НСИ.
