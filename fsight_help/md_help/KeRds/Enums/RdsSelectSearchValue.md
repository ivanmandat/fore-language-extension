# RdsSelectSearchValue


## Описание


Перечисление RdsSelectSearchValue
 содержит варианты поиска по значению.


Используется следующим свойством:


	- [IRdsDictionaryElementsSearch.SelectSearchValue](../Interface/IRdsDictionaryElementsSearch/IRdsDictionaryElementsSearch.SelectSearchValue.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 DisplayValue. Ищет
		 по значению, которое отображается в дереве элементов справочника.


		 1
		 Value. Ищет по значению,
		 которое лежит в основной таблице справочника.
