# RdsAccessAttribute


## Описание


Перечисление RdsAccessAttribute
 содержит виды атрибута доступа, по которому необходимо получить права
 для указанного элемента.


Используется следующими свойствами:


	- [IRdsDictionaryElements.Access](../Interface/IRdsDictionaryElements/IRdsDictionaryElements.Access.htm);


	- [IRdsDictionaryElements.AccessStr](../Interface/IRdsDictionaryElements/IRdsDictionaryElements.AccessStr.htm);


	- [IRdsDictionaryElements.HasAccess](../Interface/IRdsDictionaryElements/IRdsDictionaryElements.HasAccess.htm);


	- [IRdsDictionaryElement.Access](../Interface/IRdsDictionaryElement/IRdsDictionaryElement.Access.htm);


	- [IRdsDictionaryElement.AccessStr](../Interface/IRdsDictionaryElement/IRdsDictionaryElement.AccessStr.htm);


	- [IRdsDictionaryElement.HasAccess](../Interface/IRdsDictionaryElement/IRdsDictionaryElement.HasAccess.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Права для элемента
		 не определены.


		 1
		 ReadAccess. Права на
		 просмотр данных элемента.


		 2
		 WriteAccess. Права
		 на изменение данных элемента.


		 3
		 DeleteAccess. Права
		 на удаление элемента.


		 4
		 AccessAccess. Права
		 на изменение прав доступа к элементу.
