# RdsLoaderBindingType


## Описание


Перечисление RdsLoaderBindingType
 содержит типы привязок атрибутов и полей источника данных.


Используется следующим свойством:


	- [IRdsBaseBinding.BindingType](../Interface/IRdsBaseBinding/IRdsBaseBinding.BindingType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Тип привязки
		 не определен.


		 1
		 Predefined. Привязка
		 для системного атрибута.


		 2
		 TranslationAttribute.
		 Привязка для атрибута, содержащего значения на определенном языке.

		Примечание.
		 Не поддерживается, оставлено для совместимости.


		 3
		 UserAttribute. Привязка
		 для пользовательского атрибута.


		 4
		 Linked. Привязка для
		 атрибута, используемого для связи с другими справочниками.


		 5
		 Imported. Привязка
		 для заимствованного атрибута.
