# RdsElementsLookupOptions


## Описание


Перечисление RdsElementsLookupOptions содержит параметры поиска элементов справочника НСИ.


Используется следующими свойствами:


-
[IRdsDictionaryElementsLookup.AttributeOptions](../Interface/IRdsDictionaryElementsLookup/IRdsDictionaryElementsLookup.AttributeOptions.htm);


-
[IRdsDictionaryElementsLookup.Options](../Interface/IRdsDictionaryElementsLookup/IRdsDictionaryElementsLookup.Options.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Default_. По умолчанию.


		 1
		 NoDefault. Не по умолчанию.


		 2
		 Partial. По части значения.


		 4
		 CaseInsensitive. Без
		 учета регистра.


		 8
		 IgnoreParams. Без учета
		 значений параметров.


		 16
		 NoCheckDataType. Не
		 проверять тип искомого значения.
