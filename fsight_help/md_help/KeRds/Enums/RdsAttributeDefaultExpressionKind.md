# RdsAttributeDefaultExpressionKind


## Описание


Перечисление RdsAttributeDefaultExpressionKind
 содержит варианты момента вычисления значения атрибута по умолчанию.


Используется следующими свойствами и методами:


	- [IRdsAttribute.DefaultExpressionKind](../Interface/IRdsAttribute/IRdsAttribute.DefaultExpressionKind.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Не задано.


		 1
		 EvaluateOnCreate. Вычислять
		 только при создании элемента.


		 2
		 EvaluateOnEdit. Вычислять
		 при создании и редактировании элемента.


## Комментарии


По умолчанию используется значение None.
