# DictionaryParamsControlState


## Описание


Перечисление DictionaryParamsControlState
 содержит текущие состояния объекта, осуществляющего управление параметрами
 связанного справочника.


Используется следующим свойством:


	- [IRdsParamsControlInstance.ControlState](../Interface/IRdsParamsControlInstance/IRdsParamsControlInstance.ControlState.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 NotControlled. Значения
		 параметров не управляются атрибутами текущего справочника.


		 1
		 Ready. Значения параметров
		 заданы и не изменились.


		 2
		 NotReady. Заданы не
		 все значения параметров.


		 3
		 ReadyModified. Значения
		 параметров заданы и были изменены.
