# MetaMembersGetOptions


## Описание


Перечисление MetaMembersGetOptions
 содержит параметры получения элементов иерархии базы данных временных
 рядов.


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Параметры по
		 умолчанию.


		 1
		 AsTree. Дерево всех
		 дочерних элементов.


		 2
		 IncludeAll. Все элементы.


		 4
		 IncludeExistingOnly.
		 Только существующие элементы.


		 8
		 HierarchyFlat. Элементы
		 без учета иерархии.


		 16
		 HierarchyOriginal.
		 Элементы с учетом иерархии.


		 32
		 LightWeight. Без запроса
		 количества дочерних элементов.


## Комментарии


Использование значения LightWeight
 позволяет увеличить скорость получения элементов иерархии.
