## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Получить данные
		 для чтения.


		 1
		 Edit. Получить данные
		 для редактирования.


		 2
		 Existing. Получить
		 только существующие данные.


		 3
		 EditExisting. Получить
		 для редактирования только существующие данные.


		 8
		 Create. Создать данные.


		 16
		 ByIdentity. Получить
		 данные по ключу справочника.


		 32
		 Temporary. Получить
		 временный объект, без возможности сохранения.
