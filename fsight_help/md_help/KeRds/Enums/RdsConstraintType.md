# RdsConstraintType


## Описание


Перечисление RdsConstraintType содержит варианты поведения при попытке удаления элементов в справочнике, на которые имеются ссылки в других справочниках.


Используется следующим свойством:


-
[IRdsLink.DeleteConstraint](../Interface/IRdsLink/IRdsLink.DeleteConstraint.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Restrict. При попытке удалить элементы, на которые есть ссылки в других справочниках, будет происходить ограничение доступа и генерироваться исключительная ситуация.


 1
 Cascade. При удалении элементов, на которые есть ссылки в других справочниках, также будут удаляться связанные элементы.


 2
 None. При удалении элементов, на которые есть ссылки в других справочниках, будет удаляться ссылка в связанных элементах.


 3
 SetNull. При удалении элементов, на которые есть ссылки в других справочниках, в атрибутах связанных элементов будет устанавливаться пустое значение.


 4
 MakeHidden. При удалении элементов, на которые есть ссылки в других справочниках, связанные элементы будет помечены как скрытые.
