# RdsAttributeKind


## Описание


Перечисление RdsAttributeKind содержит виды атрибута справочника НСИ.


Используется следующим свойством:


-
[IRdsAttribute.Kind](../Interface/IRdsAttribute/IRdsAttribute.Kind.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 Predefined. Системный атрибут.


 1
 Common. Пользовательский атрибут.


 2
 Linked. Связанный атрибут.


 3
 Imported. Заимствованный атрибут.


 4
 Calculated. Вычислимый атрибут. Данный тип не поддерживается.
