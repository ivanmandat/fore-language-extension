# DictionaryCursorOptions


## Описание


Перечисление DictionaryCursorOptions
 содержит способы получения курсора.


Используется следующими свойствами и методами:


	- [IMetaDictionaryLookup.Open](../Interface/IMetaDictionaryLookup/IMetaDictionaryLookup.Open.htm);


	- [IRubricatorFactsLookup.Open](KeCubes.chm::/Interface/IRubricatorFactsLookup/IRubricatorFactsLookup.Open.htm);


	- [IRubricatorValidationsLookup.Open](KeCubes.chm::/Interface/IRubricatorValidationsLookup/IRubricatorValidationsLookup.Open.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Только для чтения.


		 1
		 Edit. Для редактирования.
