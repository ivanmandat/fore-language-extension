# RdsDictionaryElementDataOptions


## Описание


Перечисление RdsDictionaryElementDataOptions
 содержит параметры изменения значения атрибута элемента.


Используется следующим методом:


	- [IRdsDictionaryElementData.SetAttributeValue](../Interface/IRdsDictionaryElementData/IRdsDictionaryElementData.SetAttributeValue.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Default. Разрешено
		 изменение значений только простых атрибутов.


		 1
		 AllowChangeParamAttribute.
		 Разрешено изменение значения атрибута, связанного с параметром.
