# DictionaryTimeDependencyType


## Описание


Перечисление DictionaryTimeDependencyType содержит режимы, в которых будет формироваться период действия элемента.


Используется следующим свойством:


-
[IRdsDictionary.TimeDependency](../Interface/IRdsDictionary/IRdsDictionary.TimeDependency.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Учет изменения элементов во времени не ведется.


 1
 TimeReadOnly. Элементы могут изменяться во времени. Автоматическое формирование периода действия элементов.


 2
 TimeMutable. Элементы могут изменяться во времени. Ручное формирование периода действия элементов.
