# SubsetOperation


## Описание


Перечисление SubsetOperation содержит операции построения набора атрибутов.


Данное перечисление используется следующими свойствами и методами:


-
[IMetaAttributesSet.BaseOperation](../Interface/IMetaAttributesSet/IMetaAttributesSet.BaseOperation.htm);


-
[IMetaAttributesSet.CreateSubset](../Interface/IMetaAttributesSet/IMetaAttributesSet.CreateSubset.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 SelectBase. Выбрать указанные атрибуты из базового набора.


 1
 Select_. Выбрать указанные атрибуты.


 2
 Clone. Создать копию указанных атрибутов.
