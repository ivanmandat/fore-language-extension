# RdsPredefinedParam


## Описание


Перечисление RdsPredefinedParam содержит назначения системного параметра.


Используется следующими свойствами:


-
[IRdsParam.Predefined](../Interface/IRdsParam/IRdsParam.Predefined.htm);


-
[IRdsParams.Predefined](../Interface/IRdsParams/IRdsParams.Predefined.htm);


-
[IRdsParamValue.PredefinedType](../Interface/IRdsParamValue/IRdsParamValue.PredefinedType.htm).


## Допустимые значения


 Значение
 Краткое описание


 0
 None. Не системный параметр.


 1
 CurDate. Дата актуальности.
