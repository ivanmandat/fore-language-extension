# RdsParamsControlValueSelect


## Описание


Перечисление RdsParamsControlValueSelect
 содержит алгоритмы выбора значения при множественной отметке для [привязки](UiNavObj.chm::/reference_book/Master_RDS_reference_book/Link_Property.htm#add)
 значений атрибута редактируемого справочника к параметру связываемого
 справочника.


Используется свойством [IRdsParamsControl.ValueSelect](../Interface/IRdsParamsControl/IRdsParamsControl.ValueSelect.htm).


## Комментарии


Алгоритм выбора значения при множественной отметке определяет значение,
 которое будет передаваться в параметр связываемого справочника.


Для получения подробной информации обратитесь к разделу «[Создание
 привязки](UiNavObj.chm::/reference_book/Master_RDS_reference_book/Link_Property.htm#add)».
