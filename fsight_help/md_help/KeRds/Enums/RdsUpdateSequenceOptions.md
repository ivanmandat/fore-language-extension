# RdsUpdateSequenceOptions


## Описание


Перечисление RdsUpdateSequenceOptions
 используется для определения типа сдвига.


Используется следующими методами:


	- [IMetaDictionary.UpdateSequence](../Interface/IMetaDictionary/IMetaDictionary.UpdateSequence.htm);


	- [IRdsDictionary.UpdateSequence](../Interface/IRdsDictionary/IRdsDictionary.UpdateSequence.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 1
		 Key. Сдвиг по наибольшему
		 значению ключа.


		 2
		 Order. Сдвиг по наибольшему
		 значению порядка.
