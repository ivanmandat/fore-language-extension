# RdsCheckExecuteDeleteBatch


## Описание


Перечисление RdsCheckExecuteDeleteBatch
 содержит типы конфликтов, которые могут произойти при работе с пакетами
 элементов.


Используется следующим методом:


	- [IRdsBatchCallback.CheckExecute](../Interface/IRdsBatchCallback/IRdsBatchCallback.CheckExecute.htm).
