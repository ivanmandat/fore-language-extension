# RdsReferenceDisplayFormat


## Описание


Перечисление RdsReferenceDisplayFormat
 содержит форматы отображения значений связанных атрибутов.


Используется следующим свойством:


	- [IRdsLink.DisplayFormat](../Interface/IRdsLink/IRdsLink.DisplayFormat.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Unknown. Формат не
		 задан. По умолчанию отображается значение атрибута.


		 1
		 ValueOnly. Только значение
		 атрибута.


		 2
		 KeyOnly. Только ключ
		 соответствующего элемента.


		 3
		 KeyAndValue. Ключ и
		 значение в формате «Ключ (Значение)».
