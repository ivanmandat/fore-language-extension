# RdsUpdateDetailTableMode


## Описание


Перечисление RdsUpdateDetailTableMode
 содержит способы обновления записей в таблице, использующейся для экспорта
 значений множественного атрибута.


Используется следующим свойством:


	- [IRdsExportSchemaAttribute.UpdateDetailTableMode](../Interface/IRdsExportSchemaAttribute/IRdsExportSchemaAttribute.UpdateDetailTableMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Recreate. Пересоздать
		 записи.


		 1
		 Update. Обновить записи.


## Комментарии


Пересоздать записи. При экспорте
 сначала из таблицы удаляются все записи, а затем производится экспорт
 значений множественного атрибута.


Обновить записи. При экспорте
 производится сравнение существующих записей в таблице и значений множественного
 атрибута. Если в таблице не найдена запись, соответствующая значению множественного
 атрибута, то она будет туда добавлена. Если в таблице найдена запись,
 которая не соответствуют ни одному из значений множественного атрибута,
 то она будет удалена. Если в таблице найдена запись, соответствующая значению
 множественного атрибута, то она будет обновлена.
