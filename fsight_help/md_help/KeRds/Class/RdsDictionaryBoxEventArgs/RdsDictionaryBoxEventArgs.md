# RdsDictionaryBoxEventArgs


## Описание


Класс RdsDictionaryBoxEventArgs
 реализует аргумент событий, возникающих при каком-либо действии с элементами
 справочника НСИ, отображаемыми в компоненте [RdsDictionaryBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/RdsDictionaryBox.htm).
