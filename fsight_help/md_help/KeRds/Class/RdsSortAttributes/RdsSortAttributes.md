# RdsSortAttributes


Сборка: Rds;


## Описание


Класс RdsSortAttributes содержит
 свойства и методы для работы с коллекцией атрибутов сортировки элементов
 справочника НСИ.


## Свойства объекта класса, унаследованные от [IRdsSortAttributes](../../Interface/IRdsSortAttributes/IRdsSortAttributes.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](../../Interface/IRdsSortAttributes/IRdsSortAttributes.Count.htm)


		 Свойство Count возвращает
		 количество атрибутов сортировки в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](../../Interface/IRdsSortAttributes/IRdsSortAttributes.Item.htm)


		 Свойство Item возвращает
		 атрибут сортировки из коллекции по индексу.


## Методы объекта класса, унаследованные от [IRdsSortAttributes](../../Interface/IRdsSortAttributes/IRdsSortAttributes.htm)


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](../../Interface/IRdsSortAttributes/IRdsSortAttributes.Add.htm)


		 Метод Add добавляет
		 сортировку по указанному атрибуту с заданным направлением сортировки.


		 ![](../../Sub_Image.gif)
		 [AddRefAttribute](../../Interface/IRdsSortAttributes/IRdsSortAttributes.AddRefAttribute.htm)


		 Метод AddRefAttribute
		 добавляет сортировку для атрибутов, участвующих в связях, и позволяет
		 указать атрибут из другого справочника для сортировки по его значениям.


		 ![](../../Sub_Image.gif)
		 [Clear](../../Interface/IRdsSortAttributes/IRdsSortAttributes.Clear.htm)


		 Метод Clear осуществляет
		 удаление всех атрибутов сортировки из коллекции.


		 ![](../../Sub_Image.gif)
		 [FindByAttribute](../../Interface/IRdsSortAttributes/IRdsSortAttributes.FindByAttribute.htm)


		 Метод FindByAttribute
		 осуществляет поиск атрибута сортировки по атрибуту справочника.


		 ![](../../Sub_Image.gif)
		 [FindByRefAttribute](../../Interface/IRdsSortAttributes/IRdsSortAttributes.FindByRefAttribute.htm)


		 Метод FindByRefAttribute
		 позволяет осуществить поиск атрибута сортировки по связанному
		 атрибуту.


		 ![](../../Sub_Image.gif)
		 [LoadFrom](../../Interface/IRdsSortAttributes/IRdsSortAttributes.LoadFrom.htm)
		 Метод LoadFrom загружает
		 параметры сортировки значений атрибутов из XML.


		 ![](../../Sub_Image.gif)
		 [Remove](../../Interface/IRdsSortAttributes/IRdsSortAttributes.Remove.htm)


		 Метод Remove осуществляет
		 удаление атрибута сортировки, индекс которого передается посредством
		 параметра Index.


		 ![](../../Sub_Image.gif)
		 [SaveTo](../../Interface/IRdsSortAttributes/IRdsSortAttributes.SaveTo.htm)
		 Метод SaveTo сохраняет
		 параметры сортировки значений атрибутов в формате XML.
