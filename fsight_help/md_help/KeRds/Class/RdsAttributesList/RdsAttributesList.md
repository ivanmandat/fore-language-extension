# RdsAttributesList


Сборка: Rds;


## Описание


Класс RdsAttributesList реализует
 коллекцию атрибутов, по значениям которых может производиться поиск.


## Конструкторы


		 Имя конструктора
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Create](RdsAttributesList.Create.htm)


		 Конструктор Create
		 создает новую коллекцию атрибутов, по значениям которых может
		 осуществляться поиск элементов.
