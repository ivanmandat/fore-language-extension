# RdsAttributesList.Create


## Синтаксис


Create(Dictionary: [IRdsDictionary](../../Interface/IRdsDictionary/IRdsDictionary.htm);
 Attributes: String);


## Параметры


Dictionary - справочник, из
 атрибутов которого будет формироваться коллекция.


Attributes - список атрибутов,
 которые необходимо включить в коллекцию. Идентификаторы атрибутов разделяются
 знаком «;».


## Описание


Конструктор Create создает новую
 коллекцию атрибутов, по значениям которых может осуществляться поиск элементов.
