# MetaMembersSet


Сборка: Rds;


## Описание


Класс MetaMembersSet предназначен
 для работы с набором элементов системного справочника.


## Свойства класса, унаследованные от [IMetaMembersSet](../../Interface/IMetaMembersSet/IMetaMembersSet.htm)


		 Имя метода
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CurrentOffset](../../Interface/IMetaMembersSet/IMetaMembersSet.CurrentOffset.htm)


		 Свойство CurrentOffset
		 возвращает число уровней иерархии от корня справочника до набора
		 элементов.
