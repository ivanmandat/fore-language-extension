# RdsMergeCallback.OnProgress


## Синтаксис


OnProgress(Current: Integer; Total: Integer);


## Параметры


Current. Количество этапов, пройденных в процессе синхронизации.


Total. Общее количество этапов процесса синхронизации.


## Описание


Метод OnProgress реализует событие общего статуса процесса синхронизации справочников НСИ.
