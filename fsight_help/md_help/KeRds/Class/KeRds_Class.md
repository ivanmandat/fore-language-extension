# Классы сборки Rds


		 Класс
		 Краткое описание


		 ![](../Class_Image.gif)
		 [MetaMembersSet](MetaMembersSet/MetaMembersSet.htm)
		 Класс MetaMembersSet
		 предназначен для работы с набором элементов системного справочника.


		 ![](../Class_Image.gif)
		 [MetaRdsLoader](MetaRdsLoader/MetaRdsLoader.htm)
		 Класс MetaRdsLoader
		 реализует объект используемый для автоматического создания справочников
		 НСИ и загрузки в них данных.


		 ![](../Class_Image.gif)
		 [RdsAttributesList](RdsAttributesList/RdsAttributesList.htm)
		 Класс RdsAttributesList
		 реализует коллекцию атрибутов, по значениям которых может производиться
		 поиск.


		 ![](../Class_Image.gif)
		 [RdsDictionaryBox](RdsDictionaryBox/RdsDictionaryBox.htm)
		 Класс RdsDictionaryBox
		 реализует компонент среды разработки [RdsDictionaryBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/RdsDictionaryBox.htm),
		 используемый для отображения данных справочников НСИ.


		 ![](../Class_Image.gif)
		 [RdsDictionaryBoxEventArgs](RdsDictionaryBoxEventArgs/RdsDictionaryBoxEventArgs.htm)
		 Класс RdsDictionaryBoxEventArgs
		 реализует аргумент событий, возникающих при каком-либо действии
		 с элементами справочника НСИ, отображаемыми в компоненте [RdsDictionaryBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/RdsDictionaryBox.htm).


		 ![](../Class_Image.gif)
		 [RdsDictionaryConverter](RdsDictionaryConverter/RdsDictionaryConverter.htm)
		 Класс RdsDictionaryConverter
		 используется для преобразования справочника НСИ в табличный справочник
		 НСИ.


		 ![](../Class_Image.gif)
		 [RdsDictionaryFilterConditions](RdsDictionaryFilterConditions/RdsDictionaryFilterConditions.htm)
		 Класс RdsDictionaryFilterConditions
		 предназначен для работы с коллекцией фильтров, по которым отбираются
		 элементы справочника.


		 ![](../Class_Image.gif)
		 [RdsMergeCallback](RdsMergeCallback/RdsMergeCallback.htm)
		 Класс RdsMergeCallback
		 реализует объект, используемый для отслеживания событий, возникающих
		 при синхронизации справочников в репозитории НСИ.


		 ![](../Class_Image.gif)
		 [RdsSortAttributes](RdsSortAttributes/RdsSortAttributes.htm)
		 Класс RdsSortAttributes
		 содержит свойства и методы для работы с коллекцией атрибутов сортировки
		 элементов справочника НСИ.
