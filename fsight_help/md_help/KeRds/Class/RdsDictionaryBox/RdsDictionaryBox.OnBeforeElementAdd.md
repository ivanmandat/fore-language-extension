# RdsDictionaryBox.OnBeforeElementAdd


## Синтаксис


OnBeforeElementAdd(Sender: Object; Args: [IRdsDictionaryBoxEventArgs](../../Interface/IRdsDictionaryBoxEventArgs/IRdsDictionaryBoxEventArgs.htm));


## Параметры


Sender - параметр, возвращающий
 компонент, сгенерировавший событие.


Args - параметр, позволяющий
 определить параметры события.


## Описание


Событие OnBeforeElementAdd наступает
 перед добавлением нового элемента в компоненте [RdsDictionaryBox](UiDevEnv.chm::/02_Components_constructor_forms/03_Components_of_the_access_to_data/RdsDictionaryBox.htm).
