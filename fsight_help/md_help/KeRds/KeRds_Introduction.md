# Введение в сборку Rds


Основными интерфейсами сборки Rds
 являются:


	- [IMetaDictionary](Interface/IMetaDictionary/IMetaDictionary.htm)
	 - интерфейс, предназначенный для работы со структурой базы данных
	 временных рядов.


	- [IRdsDatabase](Interface/IRdsDatabase/IRdsDatabase.htm)
	 - интерфейс, содержащий свойства и методы для работы с репозиторием
	 НСИ.


	- [IRdsDictionary](Interface/IRdsDictionary/IRdsDictionary.htm)
	 - интерфейс, содержащий свойства и методы для работы со справочником
	 НСИ.


	- [IRdsCompoundDictionary](Interface/IRdsCompoundDictionary/IRdsCompoundDictionary.htm)
	 - интерфейс, содержащий свойства и методы для работы с составным справочником
	 НСИ.


	- [IRdsDictionaryBox](Interface/IRdsDictionaryBox/IRdsDictionaryBox.htm)
	 - интерфейс, содержащий свойства и методы для работы с компонентом
	 среды разработки RdsDictionaryBox.
