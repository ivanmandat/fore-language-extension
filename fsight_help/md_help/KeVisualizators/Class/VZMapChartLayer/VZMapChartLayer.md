# VZMapChartLayer


Сборка: Visualizators;


## Описание


Класс VZMapChartLayer используется
 для реализации объекта для работы со слоем карты.


## Комментарии


Для реализации объекта для работы с коллекцией слоев карты используйте
 класс VZMapChartLayers.

		 Имя свойства

		 ![](../../Property_Image.gif)
		 [PlaceVisualsOneByOne](../../Interface/IVZMapChartLayer/IVZMapChartLayer.PlaceVisualsOneByOne.htm)


		 Свойство PlaceVisualsOneByOne

		 рядом на одной линии на слое карты.

		 [VisualsOffset](../../Interface/IVZMapChartLayer/IVZMapChartLayer.VisualsOffset.htm)


		 Свойство VisualsOffset
		 определяет расстояние между визуализаторами разных типов, расположенными
		 на одной линии на слое карты.
