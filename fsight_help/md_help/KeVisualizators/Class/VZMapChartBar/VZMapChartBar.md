# VZMapChartBar


Сборка: Visualizators;


## Описание


Класс VZMapChartBar реализует
 объект для работы со [столбиковым
 показателем](UiMaps.chm::/Configure/Diagrams.htm)
 на слое карты.


## Комментарии


Столбиковый показатель позволяет
 отобразить дополнительную информацию о территориях, например соотношение
 значений показателя.
