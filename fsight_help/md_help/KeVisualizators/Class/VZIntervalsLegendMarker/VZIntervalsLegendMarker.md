# VZIntervalsLegendMarker


Сборка: Visualizators;


## Описание


Класс VZIntervalsLegendMarker
 реализует объект для работы с маркером легенды.


## Комментарии


Маркер легенды - это фигура
 или изображение, расположенная перед расшифровкой значений.
