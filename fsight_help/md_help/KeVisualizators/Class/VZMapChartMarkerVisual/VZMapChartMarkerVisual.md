# VZMapChartMarkerVisual


Сборка: Visualizators;


## Описание


Класс VZMapChartMarkerVisual
 реализует объект для работы с визуальным представлением маркера геолокации
 на карте.


## Комментарии


Маркер геолокации - это точка,
 привязанная к территории. Маркеры позволяют отображать на карте места,
 необходимые пользователю, например: расположение магазинов или станций
 метро.
