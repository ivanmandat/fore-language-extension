# VZLegends


## Описание


Класс VZLegends реализует объект, предназначенный для работы с коллекцией легенд визуализатора.


## Свойства объекта класса, унаследованные от [IVZLegends](../../Interface/IVZLegends/IVZLegends.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](../../Interface/IVZLegends/IVZLegends.Count.htm)


		 Свойство Count определяет
		 количество легенд в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](../../Interface/IVZLegends/IVZLegends.Item.htm)


		 Свойство Item определяет
		 легенду в коллекции легенд визуализатора по её индексу.


## Методы объекта класса, унаследованные от [IVZLegends](../../Interface/IVZLegends/IVZLegends.htm)


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](../../Interface/IVZLegends/IVZLegends.Add.htm)


		 Метод Add добавляет
		 указанную легенду в коллекцию легенд визуализатора.


		 ![](../../Sub_Image.gif)
		 [Clear](../../Interface/IVZLegends/IVZLegends.Clear.htm)


		 Метод Clear полностью
		 очищает коллекцию легенд визуализатора.


		 ![](../../Sub_Image.gif)
		 [Remove](../../Interface/IVZLegends/IVZLegends.Remove.htm)


		 Метод Remove удаляет
		 легенду визуализатора по ее индексу.
