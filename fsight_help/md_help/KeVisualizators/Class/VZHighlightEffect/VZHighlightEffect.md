# VZHighlightEffect


Сборка: Visualizators;


## Описание


Класс VZHighlightEffect реализует
 объект для работы с эффектом засветления элемента управления.


## Комментарии


Эффект засветления уменьшает
 яркость выделенного элемента управления.


Для эффекта засветления используются три составляющие цвета (RGB) и
 прозрачность (A).
