# VZMapChartBarAxis


Сборка: Visualizators;


## Описание


Класс VZMapChartBarAxis реализует
 объект для работы с осью Х [столбикового показателя](UiMaps.chm::/Configure/Diagrams.htm)
 на слое карты.


## Комментарии


Ось Х увеличивает информативность столбикового показателя.
