# VZBubbleChartTrendLine


## Описание


Класс VZBubbleChartTrendLine
 реализует объект, предназначенный для работы с линией тренда [пузырьковой
 диаграммы](../../Interface/IVZBubbleChart/IVZBubbleChart.htm).
