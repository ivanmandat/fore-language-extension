# VZMapChartPie


Сборка: Visualizators;


## Описание


Класс VZMapChartPie реализует
 объект для работы с [круговым
 показателем](UiMaps.chm::/Configure/Diagrams.htm)
 на слое карты.


## Комментарии


Круговой показатель позволяет
 отобразить дополнительную информацию о территориях, например соотношение
 значений показателя.
