# VZResourceImage


## Описание


Класс VZResourceImage реализует
 объект, предназначенный для работы с ресурсами размерной легенды визуализатора.


## Свойства объекта класса, унаследованные от [IVZResourceImage](../../Interface/IVZResourceImage/IVZResourceImage.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Bitmap](../../Interface/IVZResourceImage/IVZResourceImage.Bitmap.htm)
		 Свойство Bitmap определяет
		 изображение, которое будет использоваться в качестве маркера элемента
		 легенды.


		 ![](../../Property_Image.gif)
		 [ResourceId](../../Interface/IVZResourceImage/IVZResourceImage.ResourceId.htm)
		 Свойство ResourceId
		 определяет идентификатор ресурса, который будет использоваться
		 в качестве маркера элемента легенды.
