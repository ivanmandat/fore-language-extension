# VZMapChartLabelStyle


Сборка: Visualizators;


## Описание


Класс VZMapChartLabelStyle
 используется для реализации объекта для работы со стилем подписей слоя
 карты.


## Комментарии


Данный класс использует свойства, которые определяют положение подписей
 слоя карты относительно позиции из топоосновы.
