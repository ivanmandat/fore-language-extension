# VZTimeLine


## Описание


Класс VZTimeLine реализует объект, предназначенный для работы с временной шкалой визуализатора.


## Свойства объекта класса, унаследованные от [IVZTimeLine](../../Interface/IVZTimeLine/IVZTimeLine.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CurrentIndex](../../Interface/IVZTimeLine/IVZTimeLine.CurrentIndex.htm)


		 Свойство CurrentIndex
		 определяет индекс текущего шага временной шкалы.


		 ![](../../Property_Image.gif)
		 [DelayDuration](../../Interface/IVZTimeLine/IVZTimeLine.DelayDuration.htm)


		 Свойство DelayDuration
		 определяет длительность задержки анимации визуализатора на каждом
		 шаге временной шкалы.


		 ![](../../Property_Image.gif)
		 [Inverse](../../Interface/IVZTimeLine/IVZTimeLine.Inverse.htm)


		 Свойство Inverse определяет,
		 отображается ли временная шкала в прямом или реверсивном виде.


		 ![](../../Property_Image.gif)
		 [IsOverlap](../../Interface/IVZTimeLine/IVZTimeLine.IsOverlap.htm)


		 Свойство IsOverlap
		 возвращает признак того, что временная шкала не уменьшает рабочую
		 область визуализатора.


		 ![](../../Property_Image.gif)
		 [LastIndexAsInitial](../../Interface/IVZTimeLine/IVZTimeLine.LastIndexAsInitial.htm)


		 Свойство LastIndexAsInitial
		 определяет, будет ли при открытии экспресс-отчета выполняться
		 переход на последний шаг временной шкалы при отрицательном значении
		 индекса.


		 ![](../../Property_Image.gif)
		 [ReturnToInitialState](../../Interface/IVZTimeLine/IVZTimeLine.ReturnToInitialState.htm)


		 Свойство ReturnToInitialState
		 устанавливает признак возврата временной шкалы в начальное состояние.


		 ![](../../Property_Image.gif)
		 [StepDuration](../../Interface/IVZTimeLine/IVZTimeLine.StepDuration.htm)


		 Свойство StepDuration
		 определяет длительность воспроизведения анимации визуализатора
		 при переходе от одного шага временной шкалы к другому.


		 ![](../../Property_Image.gif)
		 [StepsNames](../../Interface/IVZTimeLine/IVZTimeLine.StepsNames.htm)


		 Свойство StepsNames
		 возвращает наименования шагов временной шкалы.
