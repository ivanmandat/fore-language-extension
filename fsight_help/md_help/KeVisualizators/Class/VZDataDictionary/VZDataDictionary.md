# VZDataDictionary


Сборка: Visualizators;


## Описание


Класс VZDataDictionary реализует
 объект, предназначенный для работы со словарем данных.


## Комментарии


Для работы с массивом данных используйте класс [VZDataArray](../VZDataArray/VZDataArray.htm).
