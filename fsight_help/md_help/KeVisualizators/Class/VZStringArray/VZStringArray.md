# VZStringArray


Сборка: Visualizators;


## Описание


Класс VZStringArray реализует
 объект, предназначенный для работы с массивом строковых переменных.


## Комментарии


Для работы с массивом вещественных чисел с двоичной точностью используйте
 класс [VZDoubleArray](../VZDoubleArray/VZDoubleArray.htm).
