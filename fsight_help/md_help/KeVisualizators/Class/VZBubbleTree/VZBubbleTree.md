# VZBubbleTree


## Описание


Класс VZBubbleTree реализует
 объект, предназначенный для работы с пузырьковым деревом.


## Свойства объекта класса, унаследованные от [IVZBubbleTree](../../Interface/IVZBubbleTree/IVZBubbleTree.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Alignment](../../Interface/IVZBubbleTree/IVZBubbleTree.Alignment.htm)


		 Свойство Alignment
		 определяет тип выравнивания элементов в пузырьковом дереве.


		 ![](../../Property_Image.gif)
		 [AnimationTime](../../Interface/IVZBubbleTree/IVZBubbleTree.AnimationTime.htm)


		 Свойство AnimationTime
		 определяет длительность анимации элементов пузырькового дерева
		 в секундах.


		 ![](../../Property_Image.gif)

		 Свойство Captions определяет
		 подписи к элементам пузырькового дерева.


		 ![](../../Property_Image.gif)
		 [ColorVisual](../../Interface/IVZBubbleTree/IVZBubbleTree.ColorVisual.htm)


		 Свойство ColorVisual
		 определяет визуализатор сопоставления данных пузырькового дерева
		 цветам заливки их дочерних элементов.


		 ![](../../Property_Image.gif)
		 [InnerLabelsStyle](../../Interface/IVZBubbleTree/IVZBubbleTree.InnerLabelsStyle.htm)


		 Свойство InnerLabelsStyle
		 определяет стиль подписей внутри элементов пузырькового дерева.


		 ![](../../Property_Image.gif)
		 [ItemsToolTipStyle](../../Interface/IVZBubbleTree/IVZBubbleTree.ItemsToolTipStyle.htm)


		 Свойство ItemsToolTipStyle
		 возвращает объект для работы со стилем всплывающих подсказок.


		 ![](../../Property_Image.gif)
		 [LabelAutoColor](../../Interface/IVZBubbleTree/IVZBubbleTree.LabelAutoColor.htm)


		 Свойство LabelAutoColor
		 определяет, используется ли автоматический цвет шрифта всплывающих
		 подсказок.


		 ![](../../Property_Image.gif)
		 [OrbitsPen](../../Interface/IVZBubbleTree/IVZBubbleTree.OrbitsPen.htm)


		 Свойство OrbitsPen
		 определяет перо линии орбит в пузырьковом дереве.


		 ![](../../Property_Image.gif)
		 [OuterLabelsStyle](../../Interface/IVZBubbleTree/IVZBubbleTree.OuterLabelsStyle.htm)


		 Свойство OuterLabelsStyle
		 определяет стиль внешних подписей для элементов пузырькового дерева.


		 ![](../../Property_Image.gif)
		 [Palette](../../Interface/IVZBubbleTree/IVZBubbleTree.Palette.htm)


		 Свойство Palette определяет
		 палитру цветов пузырькового дерева.


		 ![](../../Property_Image.gif)
		 [SizeVisual](../../Interface/IVZBubbleTree/IVZBubbleTree.SizeVisual.htm)


		 Свойство SizeVisual
		 определяет визуализатор для определения размеров дочерних элементов
		 пузырькового дерева.


		 ![](../../Property_Image.gif)
		 [TextVisual](../../Interface/IVZBubbleTree/IVZBubbleTree.TextVisual.htm)


		 Свойство TextVisual
		 определяет визуализатор текста.


		 ![](../../Property_Image.gif)
		 [UseItemColorForToolTipBorder](../../Interface/IVZBubbleTree/IVZBubbleTree.UseItemColorForToolTipBorder.htm)


		 Свойство UseItemColorForToolTipBorder
		 определяет, будет ли использован цвет элемента для оформления
		 границы всплывающей подсказки при наведении на элемент мышкой.
