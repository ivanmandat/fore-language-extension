# VZLegendItems


Сборка: Visualizators;


## Описание


Класс VZLegendItems реализует
 объект для работы с коллекцией пунктов легенды визуализатора.


## Комментарии


Для реализации объекта для работы с пунктом легенды визуализатора используйте
 [VZLegendItem](../VZLegendItem/VZLegendItem.htm).
