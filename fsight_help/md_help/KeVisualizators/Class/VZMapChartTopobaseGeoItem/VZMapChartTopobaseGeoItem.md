# VZMapChartTopobaseGeoItem


Сборка: Visualizators;


## Описание


Класс VZMapChartTopobaseGeoItem
 реализует объект, предназначенный для работы с привязкой топоосновы к
 географическим координатам.


## Комментарии


Топооснова - набор графических слоев информации, наложенных друг на
 друга.


Географические координаты определяют положение точки на земной поверхности.
