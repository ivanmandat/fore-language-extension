# VZMapChartArrowStyle


Сборка: Visualizators;


## Описание


Класс VZMapChartArrowStyle используется
 для реализации объекта для работы со стилем стрелки на карте.


## Комментарии


Стрелка - это объект карты привязанный
 к двум территориям: из которой начинается стрелка и в которую
 направляется стрелка. Стрелки позволяют визуализировать
 направление движения каких-либо потоков между территориями, например:
 экспорт или импорт товаров.
