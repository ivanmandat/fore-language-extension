# VZLegendItem


Сборка: Visualizators;


## Описание


Класс VZLegendItem реализует
 объект для работы с пунктом легенды визуализатора.


## Комментарии


Для реализации объекта для работы с коллекцией пунктов легенды визуализатора
 используйте [VZLegendItems](../VZLegendItems/VZLegendItems.htm).
