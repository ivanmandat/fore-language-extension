# VZBubbleTreeItem


## Описание


Класс VZBubbleTreeItem реализует
 объект, предназначенный для работы с элементами [пузырькового
 дерева](../../Interface/IVZBubbleTree/IVZBubbleTree.htm).


## Свойства объекта класса, унаследованные от [IVZTreeItem](../../Interface/IVZTreeItem/IVZTreeItem.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ID](../../Interface/IVZTreeItem/IVZTreeItem.ID.htm)
		 Свойство ID вовзращает
		 идентификатор элемента дерева визуализатора.


		 ![](../../Property_Image.gif)
		 [Items](../../Interface/IVZTreeItem/IVZTreeItem.Items.htm)
		 Свойство Items определяет
		 дочерние элементы у заданного элемента дерева визуализатора.


		 ![](../../Property_Image.gif)
		 [Parent](../../Interface/IVZTreeItem/IVZTreeItem.Parent.htm)
		 Свойство Parent возвращает
		 родителя для заданного элемента дерева визуализатора.


		 ![](../../Property_Image.gif)
		 [Selected](../../Interface/IVZTreeItem/IVZTreeItem.Selected.htm)
		 Свойство Selected определяет,
		 используется ли выделение области дерева визуализатора.
