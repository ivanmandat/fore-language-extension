# Требования


Для успешного выполнения примеров, приведённых в описании системной
 сборки Visualizators, необходимо
 выполнить ряд требований к экспресс-отчёту:


	- Источник данных, на котором основан экспресс-отчёт,
	 должен содержать в своей структуре территориальное измерение (в структуре
	 измерения должен присутствовать атрибут с идентификатором TERRID);


	- Для отображения и корректной работы визуализаторов
	 необходимо выполнение [системных
	 требований к видеокарте](Setup.chm::/03_DB_Server_Config/Setup_DB_ServerSysReq.htm#hardware_requirements_frontend_deskapplication).
