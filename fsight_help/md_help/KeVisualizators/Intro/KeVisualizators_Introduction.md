# Введение в сборку Visualizators


В данном разделе представлено описание сборки Visualizators,
 содержащей средства для работы на языке Fore с визуализаторами данных
 экспресс-отчёта.


Визуализаторы предназначены для отображения данных экспресс-отчёта в
 удобном для восприятия представлении - в виде карты, пузырьковой диаграммы,
 пузырькового и плоского дерева.


Основными интерфейсами сборки Visualizators
 являются:


	- [IVZBubbleTree](../Interface/IVZBubbleTree/IVZBubbleTree.htm)
	 - интерфейс предназначен для работы с пузырьковым деревом;


	- [IVZMapChart](../Interface/IVZMapChart/IVZMapChart.htm)
	 - интерфейс предназначен для работы с картой;


	- [IVZTreeMap](../Interface/IVZTreeMap/IVZTreeMap.htm)
	 - интерфейс предназначен для работы с плоским деревом;


	- [IVZDataVisualizer](../Interface/IVZDataVisualizer/IVZDataVisualizer.htm)
	 - базовый интерфейс для всех визуализаторов.


Сборка Visualizators содержит
 также интерфейсы, предназначенные для работы с составными элементами визуализаторов.
 Основными среди них являются:


	- [IVZLegendBase](../Interface/IVZLegendBase/IVZLegendBase.htm)
	 - базовый интерфейс, предназначенный для работы с легендами визуализаторов;


	- [IVZTimeLine](../Interface/IVZTimeLine/IVZTimeLine.htm)
	 - интерфейс содержит свойства и методы для работы со временной шкалой
	 визуализаторов.
