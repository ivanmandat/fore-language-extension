# VisualizatorControlState


## Описание


Перечисление VisualizatorControlState
 используется для определения состояния элемента управления.


Используется следующими методами:


	- [IVZControlStyle.GetBackground](../Interface/IVZControlStyle/IVZControlStyle.GetBackground.htm);


	- [IVZControlStyle.SetBackground](../Interface/IVZControlStyle/IVZControlStyle.SetBackground.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 ControlStateReleased.
		 Элемент управления освобождён.


		 1
		 ControlStateHover.
		 На элемент управления наведён указатель мыши.


		 2
		 ControlStatePressed.
		 Элемент управления нажат.


		 3
		 ControlStateCount.
		 Количество состояний элемента управления.
