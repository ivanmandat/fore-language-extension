# VisualizatorSizeMarkerShape


## Описание


Перечисление VisualizatorSizeMarkerShape
 используется для определения фигуры, используемой в качестве маркера элемента
 легенды с интервалами.


Используется следующими свойствами:


	- [IVZIntervalsLegend.Marker](../Interface/IVZIntervalsLegend/IVZIntervalsLegend.Marker.htm);


	- [IVZSizeLegend.Marker](../Interface/IVZSizeLegend/IVZSizeLegend.Marker.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Circle. Круг.


		 1
		 Rectangle. Прямоугольник.


		 2
		 Trapezoid. Трапеция.


		 3
		 Custom. Фигура или
		 изображение, заданные пользователем.


		 4
		 Line. Линия.
