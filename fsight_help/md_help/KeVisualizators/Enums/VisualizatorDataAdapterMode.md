# VisualizatorDataAdapterMode


## Описание


Перечисление VisualizatorDataAdapterMode
 используется для определения режима преобразования данных в адаптере для
 визуализатора.


Используется следующими свойствами и методами:


	- [IVZDataNormalizeAdapter.NormalizeMode](../Interface/IVZDataNormalizeAdapter/IVZDataNormalizeAdapter.NormalizeMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Linear. Линейное преобразование.


		 1
		 Logarithm. Логарифмическое
		 преобразование.
