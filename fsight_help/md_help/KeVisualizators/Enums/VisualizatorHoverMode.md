# VisualizatorHoverMode


## Описание


Перечисление VisualizatorHoverMode используется
 для определения режима наведения на элементы визуализатора.


Используется свойствами:


	- [IVZColorLegendBase.HoverMode](../Interface/IVZColorLegendBase/IVZColorLegendBase.HoverMode.htm);


	- [IVZToolTipStyle.HoverMode](../Interface/IVZToolTipStyle/IVZToolTipStyle.HoverMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Click. Режим обработки
		 события при нажатии на элемент визуализатора.


		 1
		 MouseOver. Режим обработки
		 события, при котором указатель мыши появляется над областью элемента
		 визуализатора.


		 2
		 MouseOverCenter. Режим
		 обработки события, при котором указатель мыши появляется над центральной
		 областью элемента визуализатора.
