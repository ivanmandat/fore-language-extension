# VisualizatorNoDataPosition


## Описание


Перечисление VisualizatorNoDataPosition
 используется для определения расположения элемента, соответствующего отсутствующим
 данным, в визуализаторе.


Используется следующим свойством:


	- [IVZColorLegendBase.NoDataPosition](../Interface/IVZColorLegendBase/IVZColorLegendBase.NoDataPosition.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Before. Расположение
		 перед остальными элементами визуализатора.


		 1
		 After. Расположение
		 после всех элементов визуализатора.
