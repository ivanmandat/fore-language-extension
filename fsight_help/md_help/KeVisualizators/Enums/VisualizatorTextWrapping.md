# VisualizatorTextWrapping


## Описание


Перечисление VisualizatorTextWrapping
 используется для определения типа переноса текста.


Используется следующими свойствами и методами:


	- [IVZLabelStyle.TextWrapping](../Interface/IVZLabelStyle/IVZLabelStyle.TextWrapping.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 WrapEllipsis.
		 Перенос с подстановкой многоточия.


		 1
		 NoWrap. Перенос не
		 осуществляется.


		 2
		 CharWrap. Перенос по
		 символам.


		 3
		 WordWrap. Перенос по
		 словам.


		 4
		 Hidden. Перенос скрыт.


		 5
		 WordWrapEllipsis. Перенос
		 по словам с подстановкой многоточия.
