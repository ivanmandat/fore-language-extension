# VisualizatorDataMappingType


## Описание


Перечисление VisualizatorDataMappingType используется
 для определения типа преобразования данных.


Используется следующим свойством:


	- [IVZDataSourceMapping.MappingType](../Interface/IVZDataSourceMapping/IVZDataSourceMapping.MappingType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 DataMappingNone. Преобразование
		 данных отсутствует.


		 1
		 DataMappingScale. Преобразование
		 данных по шкале.


		 2
		 DataMappingConstant.
		 Константное преобразование.


		 3
		 DataMappingTransformer.
		 Преобразование данных с возможностью задания метода для обработки
		 данных.
