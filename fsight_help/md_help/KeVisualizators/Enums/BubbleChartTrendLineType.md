# BubbleChartTrendLineType


## Описание


Перечисление BubbleChartTrendLineType
 используется для определения типа линии тренда пузырьковой диаграммы.


Используется свойством [IVZBubbleChartTrendLine.Type](../Interface/IVZBubbleChartTrendLine/IVZBubbleChartTrendLine.Type.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Linear. Линейный тренд.


		 1
		 Power. Степенной тренд.


		 2
		 Exponential. Экспоненциальный
		 тренд.


		 3
		 Logarithmic. Логарифмический
		 тренд.


		 4
		 Mean. Среднее.
