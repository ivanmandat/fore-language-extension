# VisualizatorTerritoryTranscriptSelectEvent


## Описание


Перечисление VisualizatorTerritoryTranscriptSelectEvent используется
 для определения режима выбора элементов в расшифровке территорий.


Используется следующими свойством:


	- [IVZTerritoryTranscript.SelectEvent](../Interface/IVZTerritoryTranscript/IVZTerritoryTranscript.SelectEvent.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 SelectOnHover. Выбор
		 элемента при наведении на него курсора мыши.


		 1
		 SelectOnClick. Выбор
		 элемента при нажатии на него.
