# VisualizatorLegendMode


## Описание


Перечисление VisualizatorLegendMode
 используется для определения режима отображения легенды с интервалами.


Используется свойством [IVZIntervalsLegend.Mode](../Interface/IVZIntervalsLegend/IVZIntervalsLegend.Mode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 BetweenOnly. Отображаются
		 только интервалы с обеими границами.


		 1
		 LessGreater. Отображаются
		 интервалы с обеими границами или хотя бы одной границей.
