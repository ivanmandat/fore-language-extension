# VisualizerLabelHorizontalAlignment


## Описание


Перечисление VisualizerLabelHorizontalAlignment используется
 для позиционирования подписей слоя карты по горизонтали относительно позиции
 из топоосновы.


Используется следующим свойством:


	- [IVZMapChartLabelStyle.HorizontalAlignment](../Interface/IVZMapChartLabelStyle/IVZMapChartLabelStyle.HorizontalAlignment.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Left. Подпись слева.


		 1
		 Center. Подпись по
		 центру.


		 2
		 Right. Подпись справа.
