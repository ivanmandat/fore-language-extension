# VisualizersResourceId


## Описание


Перечисление VisualizersResourceId используется
 для определения идентификатора ресурса визуализатора.


Используется свойством:


	- [IVZResourceImage.ResourceId](../Interface/IVZResourceImage/IVZResourceImage.ResourceId.htm).
