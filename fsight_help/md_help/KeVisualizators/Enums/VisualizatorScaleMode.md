# VisualizatorScaleMode


## Описание


Перечисление VisualizatorScaleMode
 используется для определения режима отображения шкалы.


Используется свойством [IVZDataScale.Mode](../Interface/IVZDataScale/IVZDataScale.Mode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 ScaleLinear. Линейная
		 шкала.


		 1
		 ScaleUniform. Равномерная
		 шкала.


		 2
		 ScaleLogarithmic. Логарифмическая
		 шкала. При использовании данного типа задайте основание логарифма
		 в [IVZDataScale.LogarithmBase](../Interface/IVZDataScale/IVZDataScale.LogarithmBase.htm).


		 3
		 ScaleSpectrum. Спектральная
		 шкала.
