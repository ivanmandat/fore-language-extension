# MapChartArrowTextOrientation


## Описание


Перечисление MapChartArrowTextOrientation используется
 для определения типа расположения подписи стрелки на карте.


Используется следующими свойством:


	- [IVZMapChartArrowStyle.TextOrientation](../Interface/IVZMapChartArrowStyle/IVZMapChartArrowStyle.TextOrientation.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Horizonatal. Горизонтальное
		 расположение.


		 1
		 AlongTheArrow. Расположение
		 вдоль стрелки.
