# VisualizatorHistoryModeEnum


## Описание


Перечисление VisualizatorHistoryModeEnum
 используется для определения режима отображения истории визуализатора.


Используется следующими свойством:


	- [IVZBubbleChart.ShowHistoryMode](../Interface/IVZBubbleChart/IVZBubbleChart.ShowHistoryMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. По умолчанию.
		 Без отображения.


		 1
		 Over. При наведении
		 указателя мыши.


		 2
		 Click. При щелчке мыши.


		 3
		 Always. Постоянное
		 отображение.
