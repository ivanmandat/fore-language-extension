# VisualizatorScaleType


## Описание


Перечисление VisualizatorScaleType используется
 для определения типа шкалы.


Используется свойством [IVZDataScale.Type](../Interface/IVZDataScale/IVZDataScale.Type.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Brush. Цветовая шкала.


		 1
		 Double. Числовая шкала.
