# VisualizatorBubbleTreeAlignment


## Описание


Перечисление VisualizatorBubbleTreeAlignment используется
 для определения типа выравнивания элементов в пузырьковом дереве.


Используется следующими свойствами и методами:


	- [IVZBubbleTree.Alignment](../Interface/IVZBubbleTree/IVZBubbleTree.Alignment.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Right. Выравнивание
		 по правому краю.


		 1
		 Bottom. Выравнивание
		 по нижнему краю.


		 2
		 Left. Выравнивание
		 по левому краю.


		 3
		 Top. Выравнивание по
		 верхнему краю.
