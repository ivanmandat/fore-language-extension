# VisualizatorHorizontalAlignment


## Описание


Перечисление VisualizatorHorizontalAlignment
 используется для определения расположения элемента по горизонтали.


Используется следующим свойством:


	- [IVZPlacedControlBase.HorizontalAlignment](../Interface/IVZPlacedControlBase/IVZPlacedControlBase.HorizontalAlignment.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Right. Расположение
		 по правому краю.


		 1
		 Center. Расположение
		 по центру.


		 2
		 Left. Расположение
		 по левому краю.


		 3
		 None. Элемент будет
		 скрыт.
