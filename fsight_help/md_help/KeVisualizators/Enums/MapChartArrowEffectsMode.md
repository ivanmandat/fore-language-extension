# MapChartArrowEffectsMode


## Описание


Перечисление MapChartArrowEffectsMode используется
 для определения режима подсветки стрелок карты.


Используется следующим свойством:


	- [IVZMapChart.ArrowEffectsMode](../Interface/IVZMapChart/IVZMapChart.ArrowEffectsMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Без подсветки.


		 1
		 StartShape. Подсветка
		 стрелок, выходящих из региона.


		 2
		 EndShape. Подсветка
		 стрелок, заходящих в регион.


## Комментарии


Подсветка стрелок происходит при наведении указателя мыши на регион,
 в который заходят или выходят стрелки.
