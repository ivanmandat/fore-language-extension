# VisualizatorVerticalAlignment


## Описание


Перечисление VisualizatorVerticalAlignment
 используется для определения расположения элемента по вертикали.


Используется следующими свойствами:


	- [IVZLegendBase.Header](../Interface/IVZLegendBase/IVZLegendBase.Header.htm);


	- [IVZLegendBase.Footer](../Interface/IVZLegendBase/IVZLegendBase.Footer.htm);


	- [IVZPlacedControlBase.VerticalAlignment](../Interface/IVZPlacedControlBase/IVZPlacedControlBase.VerticalAlignment.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Bottom. Расположение
		 по нижнему краю.


		 1
		 Center. Расположение
		 по центру.


		 2
		 Top. Расположение по
		 верхнему краю.


		 3
		 None. Элемент будет
		 скрыт.
