# VisualizerLabelVerticalAlignment


## Описание


Перечисление VisualizerLabelVerticalAlignment используется
 для позиционирования подписей слоя карты по вертикали относительно позиции
 из топоосновы.


Используется следующим свойством:


	- [IVZMapChartLabelStyle.VerticalAlignment](../Interface/IVZMapChartLabelStyle/IVZMapChartLabelStyle.VerticalAlignment.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Top. Подпись сверху.


		 1
		 Center. Подпись по
		 центру.


		 2
		 Bottom. Подпись снизу.
