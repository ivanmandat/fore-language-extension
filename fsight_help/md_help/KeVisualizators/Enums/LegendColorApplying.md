# LegendColorApplying


## Описание


Перечисление LegendColorApplying используется
 для определения типа заливки области, в которой находятся пункты легенды
 визуализатора.


Используется следующим свойством:


	- [IVZIntervalsLegend.ColorApplying](../Interface/IVZIntervalsLegend/IVZIntervalsLegend.ColorApplying.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Background. Заливка
		 области, в которой находятся пункты легенды.


		 1
		 Marker. Заливка маркера.
