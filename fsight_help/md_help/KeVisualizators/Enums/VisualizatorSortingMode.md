# VisualizatorSortingMode


## Описание


Перечисление VisualizatorSortingMode используется
 для определения режима сортировки элементов.


Используется следующим свойством:


	- [IVZTerritoryTranscript.SortingMode](../Interface/IVZTerritoryTranscript/IVZTerritoryTranscript.SortingMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 None. Без сортировки.


		 1
		 Asc. Сортировка по
		 возрастанию.


		 2
		 Desc. Сортировка по
		 убыванию.
