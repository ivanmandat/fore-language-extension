# MapChartArrowType


## Описание


Перечисление MapChartArrowType используется
 для определения типа стрелки на карте.


Используется следующими свойством:


	- [IVZMapChartArrowStyle.ArrowType](../Interface/IVZMapChartArrowStyle/IVZMapChartArrowStyle.ArrowType.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Simple. Обычная. Линия стрелки одинаковая
		 по ширине в начале и в конце.


		 1
		 Filled. С заливкой.
		 Линия стрелки одинаковая по ширине в начале и в конце,
		 обладает внутренним пространством, которое можно залить необходимым
		 цветом.


		 2
		 Figured. Фигурная.
		 Линия стрелки сужается от конца к началу, обладает внутренним
		 пространством, которое можно залить необходимым цветом.
