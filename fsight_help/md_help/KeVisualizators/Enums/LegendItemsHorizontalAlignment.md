# LegendItemsHorizontalAlignment


## Описание


Перечисление LegendItemsHorizontalAlignment используется
 для определения горизонтального выравнивания пунктов легенды визуализатора.


Используется следующим свойством:


	- [IVZIntervalsLegend.HorizontalCellsAligment](../Interface/IVZIntervalsLegend/IVZIntervalsLegend.HorizontalCellsAligment.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Center. Выравнивание
		 по центру.


		 1
		 Left. Выравнивание
		 по левому краю.


		 2
		 Right. Выравнивание
		 по правому краю.
