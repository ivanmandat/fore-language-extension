# VisualizatorBubbleTreeRenderMode


## Описание


Перечисление VisualizatorBubbleTreeRenderMode используется
 для определения режима отрисовки пузырькового дерева.


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Mode2DAnimating. Отрисовка
		 двухмерного пузырькового дерева с поддержкой анимации.


		 1
		 Mode2D. Отрисовка двухмерного
		 пузырькового дерева без поддержки анимации.
