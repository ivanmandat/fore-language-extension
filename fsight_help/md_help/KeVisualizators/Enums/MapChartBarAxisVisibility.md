# MapChartBarAxisVisibility


## Описание


Перечисление MapChartBarAxisVisibility используется
 для определения типа видимости оси Х [столбикового показателя](UiMaps.chm::/Configure/Diagrams.htm)
 на слое карты.


Используется следующим свойством:


	- [IVZMapChartBar.Axis](../Interface/IVZMapChartBar/IVZMapChartBar.Axis.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 Collapsed. Ось Х не
		 видна.


		 1
		 Visible. Ось Х видна.


		 2
		 IsDependent. Ось Х
		 видна при наличии отрицательных значений.
