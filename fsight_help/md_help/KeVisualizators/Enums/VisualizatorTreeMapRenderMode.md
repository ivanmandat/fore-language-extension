# VisualizatorTreeMapRenderMode


## Описание


Перечисление VisualizatorTreeMapRenderMode
 используется для определения режима визуализации [плоского
 дерева](../Interface/IVZTreeMap/IVZTreeMap.htm).


Используется следующими свойствами и методами:


	- [IVZTreeMap.RenderMode](../Interface/IVZTreeMap/IVZTreeMap.RenderMode.htm).


## Допустимые значения


		 Значение
		 Краткое описание


		 0
		 RM3D. Режим трехмерной
		 визуализации.


		 1
		 RM2D. Режим двумерной
		 визуализации.
