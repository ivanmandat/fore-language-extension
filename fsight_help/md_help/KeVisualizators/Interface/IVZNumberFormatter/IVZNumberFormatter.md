# IVZNumberFormatter


Сборка: Visualizators;


## Описание


Интерфейс IVZNumberFormatter
 предназначен для определения формата числовых данных визуализатора.


## Иерархия наследования


IVZNumberFormatter


## Комментарии


Объект типа IVZNumberFormatter
 может быть получен в свойстве [IVZBaseFormatter.NumberFormatter](../IVZBaseFormatter/IVZBaseFormatter.NumberFormatter.htm).
