# IVZNumberFormatter.LocalFormatObject


## Синтаксис


LocalFormatObject(Value: Variant, Format: String,
 Var outStr: String): Boolean;


## Параметры


Value. Числовое значение;


Format. [Шаблон](ModDrawing.chm::/Interface/IGxTitle/IGxTitle.CustomFormat.htm)
 формата числа;


outStr. Параметр, в котором
 хранится сконвертированное числовое значение в заданном формате.


## Описание


Метод LocalFormatObject возвращает
 результат конвертации числа в заданный формат.


## Комментарии


Допустимые значения:


	- True. Число сконвертировано
	 в заданный формат;


	- False. Число не сконвертировано
	 в заданный формат.
