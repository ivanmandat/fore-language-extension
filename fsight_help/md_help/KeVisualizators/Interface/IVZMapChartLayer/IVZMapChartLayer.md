# IVZMapChartLayer


Сборка: Visualizators;


## Описание


Интерфейс IVZMapChartLayer
 используется для настройки слоя карты.


## Иерархия наследования


           [IVZMapChartBaseObject](../IVZMapChartBaseObject/IVZMapChartBaseObject.htm)


           IVZMapChartLayer


## Комментарии


Для получения интерфейса [IVZMapChartLayer](IVZMapChartLayer.htm)
 используйте свойство IVZMapChartLayers.Item.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Bars](IVZMapChartLayer.Bars.htm)


		 Свойство Bars возвращает
		 массив [столбиковых
		 показателей](UiMaps.chm::/Configure/Diagrams.htm)
		 на слое карты.


		 ![](../../Property_Image.gif)
		 [Bubbles](IVZMapChartLayer.Bubbles.htm)


		 Свойство Bubbles возвращает
		 массив пузырьковых показателей на слое карты.


		 ![](../../Property_Image.gif)
		 [LabelsStyle](IVZMapChartLayer.LabelsStyle.htm)


		 Свойство LabelsStyle
		 определяет стиль подписей слоя карты.


		 ![](../../Property_Image.gif)
		 [Pies](IVZMapChartLayer.Pies.htm)


		 Свойство Pies возвращает
		 массив [круговых
		 показателей](UiMaps.chm::/Configure/Diagrams.htm)
		 на слое карты.


		 ![](../../Property_Image.gif)
		 [PlaceVisualsOneByOne](IVZMapChartLayer.PlaceVisualsOneByOne.htm)


		 Свойство PlaceVisualsOneByOne
		 определяет, будут ли визуализаторы разных типов располагаться
		 рядом на одной линии на слое карты.


		 ![](../../Property_Image.gif)
		 [VisualsOffset](IVZMapChartLayer.VisualsOffset.htm)


		 Свойство VisualsOffset
		 определяет расстояние между визуализаторами разных типов, расположенными
		 на одной линии на слое карты.
