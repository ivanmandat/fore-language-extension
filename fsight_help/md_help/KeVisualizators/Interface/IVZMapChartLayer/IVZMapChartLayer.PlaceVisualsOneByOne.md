# IVZMapChartLayer.PlaceVisualsOneByOne


## Синтаксис


PlaceVisualsOneByOne: Boolean;


## Описание


Свойство PlaceVisualsOneByOne
 определяет, будут ли визуализаторы разных типов располагаться рядом на
 одной линии на слое карты.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Визуализаторы разных типов будут размещаться рядом на одной линии
	 на слое карты;


	- False. Визуализаторы
	 разных типов будут накладываться друг на друга.


Виды визуализаторов на карте:


	- [круговые показатели](IVZMapChartLayer.Pies.htm);


	- [столбиковые показатели](IVZMapChartLayer.Bars.htm);


	- [пузырьковые показатели](IVZMapChartLayer.Bubbles.htm).


Для задания расстояния между визуализаторами разных типов, расположенными
 на одной линии на слое карты, используйте [IVZMapChartLayer.VisualsOffset](IVZMapChartLayer.VisualsOffset.htm).
