# IVZDataNormalizeAdapter.MinValue


## Синтаксис


MinValue: Double;


## Описание


Свойство MinValue определяет
 нижнюю границу интервала, к которому приводятся данные.


## Комментарии


Для определения минимального значения интервала используйте свойство
 [IVZDataNormalizeAdapter.MinCutValue](IVZDataNormalizeAdapter.MinCutValue.htm).
