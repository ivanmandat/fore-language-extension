# IVZDataNormalizeAdapter.MinCutValue


## Синтаксис


MinCutValue: Double;


## Описание


Свойство MinCutValue определяет
 минимальное значение интервала.


## Комментарии


Значения, которые меньше минимального значения, становятся равными ему.
