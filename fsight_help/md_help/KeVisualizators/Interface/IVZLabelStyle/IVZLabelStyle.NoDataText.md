# IVZLabelStyle.NoDataText


## Синтаксис


NoDataText: Variant;


## Описание


Свойство NoDataText определяет
 текст, отображаемый при отсутствии данных.


## Комментарии


Для определения маски текста используйте [IVZLabelStyle.MaskText](IVZLabelStyle.MaskText.htm).
