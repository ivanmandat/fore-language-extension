# IVZTreeModel.SetActiveItem


## Синтаксис


SetActiveItem(Value: String);


## Параметры


Value. Элемент дерева, который требуется сделать активным.


## Описание


Метод SetActiveItem устанавливает указанный элемент дерева в качестве текущего в визуализаторе иерархической структуры.
