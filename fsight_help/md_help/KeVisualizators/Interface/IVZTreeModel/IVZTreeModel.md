# IVZTreeModel


Сборка: Visualizators;


## Описание


Интерфейс IVZTreeModel предназначен
 для работы с моделью визуализатора иерархической структуры.


## Иерархия наследования


IVZTreeModel


## Комментарии


Параметры модели дерева визуализатора могут быть получены в свойстве
 [IVZTreeDataVisualizer.Model](../IVZTreeDataVisualizer/IVZTreeDataVisualizer.Model.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ActiveItem](IVZTreeModel.ActiveItem.htm)


		 Свойство ActiveItem
		 определяет активный элемент в визуализаторе с иерархической структурой.


		 ![](../../Property_Image.gif)
		 [RootItem](IVZTreeModel.RootItem.htm)


		 Свойство RootItem определяет
		 корневой элемент в визуализаторе с иерархической структурой.

		 Имя метода
