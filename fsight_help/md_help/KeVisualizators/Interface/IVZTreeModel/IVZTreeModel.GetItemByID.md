# IVZTreeModel.GetItemByID


## Синтаксис


GetItemByID(Value: String): [IVZTreeItem](../IVZTreeItem/IVZTreeItem.htm);


## Параметры


Value. Идентификатор элемента дерева, который требуется сделать активным.


## Описание


Метод GetItemByID возвращает элемент дерева по его идентификатору.
