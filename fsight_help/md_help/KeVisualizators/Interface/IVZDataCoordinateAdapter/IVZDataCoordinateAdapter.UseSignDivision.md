# IVZDataCoordinateAdapter.UseSignDivision


## Синтаксис


UseSignDivision: Boolean;


## Описание


Свойство UseSignDivision определяет,
 будут ли данные в преобразователе обрабатываться по модулю.


## Комментарии


Допустимые значения:


	- True. Данные будут обрабатываться
	 по модулю;


	- False. По умолчанию.
	 Данные не будут обрабатываться по модулю.


Для определения, будут ли в преобразователе исключаться нулевые значения,
 используйте [IVZDataCoordinateAdapter.IncludeZero](IVZDataCoordinateAdapter.IncludeZero.htm).
