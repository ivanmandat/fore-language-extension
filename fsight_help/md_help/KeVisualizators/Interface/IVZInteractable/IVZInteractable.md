# IVZInteractable


Сборка: Visualizators;


## Описание


Интерфейс IVZInteractable предназначен
 для работы с функциями интерактивности.


## Иерархия наследования


           IVZInteractable


## Комментарии


Под интерактивностью понимается взаимодействие пользователя с объектами
 интерфейса с помощью мыши.
