# IVZChartGrid


Сборка: Visualizators;


## Описание


Интерфейс IVZChartGrid предназначен
 для работы с координатной сеткой диаграммы.


## Иерархия наследования


           [IVZChartObject](../IVZChartObject/IVZChartObject.htm)


           IVZChartGrid


## Комментарии


Для настройки параметров главных линий сетки используйте свойство [IVZChartAxis.PrimaryGrid](../IVZChartAxis/IVZChartAxis.PrimaryGrid.htm),
 для настройки параметров неосновных линий сетки - [IVZChartAxis.SecondaryGrid](../IVZChartAxis/IVZChartAxis.SecondaryGrid.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AutoLineCount](IVZChartGrid.AutoLineCount.htm)
		 Свойство AutoLineCount
		 определяет признак использования автоматического расчёта количества
		 интервалов между главными линиями сетки.


		 ![](../../Property_Image.gif)
		 [LineCount](IVZChartGrid.LineCount.htm)
		 Свойство LineCount
		 определяет количество интервалов между линиями сетки вручную.


		 ![](../../Property_Image.gif)
		 [Pen](IVZChartGrid.Pen.htm)
		 Свойство Pen определяет
		 параметры начертания линий сетки.


		 ![](../../Property_Image.gif)
		 [Visible](IVZChartGrid.Visible.htm)
		 Свойство Visible определяет
		 признак отображения линий сетки на диаграмме.
