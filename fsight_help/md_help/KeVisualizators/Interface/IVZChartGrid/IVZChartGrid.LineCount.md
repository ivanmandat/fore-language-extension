# IVZChartGrid.LineCount


## Синтаксис


LineCount: Integer;


## Описание


Свойство LineCount определяет
 количество интервалов между линиями сетки вручную.


## Комментарии


Значение свойства учитывается, если отключён автоматический расчёт количества
 интервалов. Свойство [IVZChartGrid.AutoLineCount](IVZChartGrid.AutoLineCount.htm)
 должно принимать значение False.
