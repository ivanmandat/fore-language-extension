# IVZChartGrid.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет
 признак отображения линий сетки на диаграмме.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Линии сетки отображаются на диаграмме;


	- False. Линии сетки не
	 отображаются на диаграмме.
