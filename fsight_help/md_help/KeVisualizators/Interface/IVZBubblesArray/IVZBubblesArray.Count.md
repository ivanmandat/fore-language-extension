# IVZBubblesArray.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 пузырьковых показателей в коллекции на слое карты.


## Комментарии


Для получения пузырькового показателя по его индексу используйте [IVZBubblesArray.Item](IVZBubblesArray.Item.htm).
