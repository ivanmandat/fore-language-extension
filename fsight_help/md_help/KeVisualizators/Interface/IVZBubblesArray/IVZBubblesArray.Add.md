# IVZBubblesArray.Add


## Синтаксис


Add(Value: IVZMapChartBubble): Integer;


## Параметры


Value. Пузырьковый показатель.


## Описание


Метод Add добавляет указанный
 пузырьковый показатель в коллекцию на слое карты.


## Комментарии


Для удаления пузырькового показателя из коллекции по его индексу используйте
 [IVZBubblesArray.Remove](IVZBubblesArray.Remove.htm).


Для очищения коллекции пузырьковых показателей используйте [IVZBubblesArray.Clear](IVZBubblesArray.Clear.htm).
