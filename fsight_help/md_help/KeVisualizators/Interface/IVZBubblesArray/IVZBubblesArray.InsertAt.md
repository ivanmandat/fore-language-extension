# IVZBubblesArray.InsertAt


## Синтаксис


InsertAt(Index: Integer; Value: IVZMapChartBubble);


## Параметры


Index. Индекс позиции, в которую
 необходимо вставить пузырьковый показатель;


Value. Пузырьковый показатель.


## Описание


Метод InsertAt вставляет указанный
 пузырьковый показатель в указанную позицию в коллекции.


## Комментарии


Для добавления пузырькового показателя в коллекцию используйте [IVZBubblesArray.Add](IVZBubblesArray.Add.htm).
