# IVZBubblesArray.Remove


## Синтаксис


Remove(Index: Integer);


## Описание


Метод Remove удаляет пузырьковый
 показатель из коллекции по его индексу.


## Комментарии


Для добавления пузырькового показателя в коллекцию используйте [IVZBubblesArray.Add](IVZBubblesArray.Add.htm).
