# IVZBubblesArray.Item


## Синтаксис


Item(Index: Integer): [IVZMapChartBubble](../IVZMapChartBubble/IVZMapChartBubble.htm);


## Параметры


Index. Индекс пузырькового
 показателя.


## Описание


Свойство Item возвращает
 пузырьковый показатель по его индексу.


## Комментарии


Для получения количества столбиковых показателей в коллекции используйте
 [IVZBubblesArray.Count](IVZBubblesArray.Count.htm).
