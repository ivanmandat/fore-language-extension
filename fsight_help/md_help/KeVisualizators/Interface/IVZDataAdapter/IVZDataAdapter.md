# IVZDataAdapter


Сборка: Visualizators;


## Описание


Интерфейс IVZDataAdapter предназначен
 для преобразования данных в визуализаторах.


## Иерархия наследования


IVZDataAdapter


## Комментарии


Данный интерфейс является базовым для следующих интерфейсов:


	- [IVZDataNormalizeAdapter](../IVZDataNormalizeAdapter/IVZDataNormalizeAdapter.htm).


Настройки преобразователя данных визуализатора могут быть получены в
 свойстве [IVZDataSourceMapping.DataTransformer](../IVZDataSourceMapping/IVZDataSourceMapping.DataTransformer.htm).
