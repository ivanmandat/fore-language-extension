# IVZBreadcrumb


Сборка: Visualizators;


## Описание


Интерфейс IVZBreadcrumb предназначен
 для работы с цепочкой навигации.


## Иерархия наследования


           [IVZPlacedControlBase](../IVZPlacedControlBase/IVZPlacedControlBase.htm)


           IVZBreadcrumb


## Комментарии


Цепочка навигации выглядит следующим образом:


![](IVZBreadcrumb.png)


Параметры цепочки навигации могут быть получены в свойстве [IVZTreeMap.Breadcrumb](../IVZTreeMap/IVZTreeMap.Breadcrumb.htm).
