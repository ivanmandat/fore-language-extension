# IVZLegendItems


Сборка: Visualizators;


## Описание


Интерфейс VZLegendItems используется
 для работы с коллекцией пунктов легенды визуализатора.


## Иерархия наследования


           VZLegendItems


## Комментарии


Для получения объекта данного типа используйте:


	- [IEaxGrid.Legends](KeExpress.chm::/Interface/IEaxGrid/IEaxGrid.Legends.htm)


	- [IVZColorLegendBase.Items](../IVZColorLegendBase/IVZColorLegendBase.Items.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IVZLegendItems.Count.htm)


		 Свойство Count возвращает
		 количество пунктов легенды в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IVZLegendItems.Item.htm)


		 Свойство Item возвращает
		 пункт легенды визуализатора по его индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IVZLegendItems.Add.htm)


		 Метод Add добавляет
		 пункт легенды визуализатора в коллекцию.


		 ![](../../Sub_Image.gif)
		 [Clear](IVZLegendItems.Clear.htm)


		 Метод Clear очищает
		 коллекцию пунктов легенды визуализатора.


		 ![](../../Sub_Image.gif)
		 [InsertAt](IVZLegendItems.InsertAt.htm)


		 Метод InsertAt вставляет
		 указанный пункт легенды визуализатора в указанную позицию в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IVZLegendItems.Remove.htm)


		 Метод Remove удаляет
		 пункт легенды визуализатора по его индексу.
