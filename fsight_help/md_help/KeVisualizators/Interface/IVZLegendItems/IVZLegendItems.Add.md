# IVZLegendItems.Add


## Синтаксис


Add(Value: [IVZLegendItem](../IVZLegendItem/IVZLegendItem.htm);):
 Integer;


## Параметры


Value. Легенда визуализатора.


## Описание


Метод Add добавляет пункт легенды
 визуализатора в коллекцию.


## Комментарии


Для удаления пункта легенды визуализатора по его индексу используйте
 [IVZLegendItems.Remove](IVZLegendItems.Remove.htm).


Для очищения коллекции пунктов легенды визуализатора используйте [IVZLegendItems.Clear](IVZLegendItems.Clear.htm).
