# IVZLegendItems.Remove


## Синтаксис


Remove(Index: Integer);


## Параметры


Index. Индекс пункта легенды
 визуализатора.


## Описание


Метод Remove удаляет пункт легенды
 визуализатора по его индексу.


## Комментарии


Для добавления пункта легенды в коллекцию используйте [IVZLegendItems.Add](IVZLegendItems.Add.htm).
