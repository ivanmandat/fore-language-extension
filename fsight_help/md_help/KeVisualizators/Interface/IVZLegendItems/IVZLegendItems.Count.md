# IVZLegendItems.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 пунктов легенды в коллекции.


## Комментарии


Для получения пункта легенды по его индексу используйте [IVZLegendItems.Item](IVZLegendItems.Item.htm).
