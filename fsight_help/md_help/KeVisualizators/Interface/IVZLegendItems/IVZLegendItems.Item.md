# IVZLegendItems.Item


## Синтаксис


Item(Index: Integer): [IVZLegendItem](../IVZLegendItem/IVZLegendItem.htm);


## Параметры


Index. Индекс пункта легенды
 визуализатора.


## Описание


Свойство Item возвращает пункт
 легенды визуализатора по его индексу.


## Комментарии


Для получения количества пунктов легенды в коллекции используйте [IVZLegendItems.Count](IVZLegendItems.Count.htm).
