# IVZLegendItems.InsertAt


## Синтаксис


InsertAt(Index: Integer; Value: [IVZLegendItem](../IVZLegendItem/IVZLegendItem.htm));


## Параметры


Index. Индекс позиции, в которую
 необходимо вставить пункт легенды визуализатора;


Value. Пункт легенды визуализатора.


## Описание


Метод InsertAt вставляет указанный
 пункт легенды визуализатора в указанную позицию в коллекции.


## Комментарии


Для добавления пункта легенды в коллекцию используйте [IVZLegendItems.Add](IVZLegendItems.Add.htm).
