# IVZChartAxis.ShowLabels


## Синтаксис


ShowLabels: Boolean;


## Описание


Cвойство ShowLabels определяет, будут ли отображаться подписи на оси
 диаграммы.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Подписи отображаются на оси диаграммы;


	- False. Подписи не отображаются.
