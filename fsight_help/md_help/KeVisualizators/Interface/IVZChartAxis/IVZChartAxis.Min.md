# IVZChartAxis.Min


## Синтаксис


Min: Double;


## Описание


Свойство Min определяет минимальное
 значение на оси диаграммы.


## Комментарии


Значение, заданное в свойстве, учитывается в том случае, если отключено
 автоопределение минимального значения (свойство [IVZChartAxis.AutoMin](IVZChartAxis.AutoMin.htm)
 = false).
