# IVZChartAxis.AutoMin


## Синтаксис


AutoMin: Boolean;


## Описание


Свойство AutoMin определяет,
 будет ли автоматически рассчитано и установлено минимальное значение на
 оси диаграммы.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Минимальное значение на оси диаграммы рассчитывается и устанавливается
	 автоматически;


	- False. Минимальное значение
	 на оси диаграммы задается пользователем с помощью свойства [IVZChartAxis.Min](IVZChartAxis.Min.htm).


Для округления значений на оси диаграммы и автоматического расчёта интервалов
 между главными линиями сетки задайте свойству [IVZChartGrid.AutoLineCount](../IVZChartGrid/IVZChartGrid.AutoLineCount.htm)
 значение True.
