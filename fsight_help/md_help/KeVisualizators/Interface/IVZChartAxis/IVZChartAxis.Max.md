# IVZChartAxis.Max


## Синтаксис


Max: Double;


## Описание


Свойство Max определяет максимальное
 значение на оси диаграммы.


## Комментарии


Значение, заданное в свойстве, учитывается в том случае, если отключено
 автоопределение максимального значения (свойство [IVZChartAxis.AutoMax](IVZChartAxis.AutoMax.htm)
 = false).
