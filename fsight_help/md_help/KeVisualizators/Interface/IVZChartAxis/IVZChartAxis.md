# IVZChartAxis


Сборка: Visualizators;


## Описание


Интерфейс IVZChartAxis предназначен
 для работы с параметрами оси [пузырьковой
 диаграммы](../IVZBubbleChart/IVZBubbleChart.htm).


## Иерархия наследования


           [IVZChartObject](../IVZChartObject/IVZChartObject.htm)


           IVZChartAxis


## Комментарии


Параметры оси диаграммы могут быть получены в свойствах [IVZBubbleChart.AxisX](../IVZBubbleChart/IVZBubbleChart.AxisX.htm)
 и [IVZBubbleChart.AxisY](../IVZBubbleChart/IVZBubbleChart.AxisY.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AutoMax](IVZChartAxis.AutoMax.htm)


		 Свойство AutoMax определяет,
		 будет ли автоматически рассчитано и установлено максимальное значение
		 на оси диаграммы.


		 ![](../../Property_Image.gif)
		 [AutoMin](IVZChartAxis.AutoMin.htm)


		 Свойство AutoMin определяет,
		 будет ли автоматически рассчитано и установлено минимальное значение
		 на оси диаграммы.


		 ![](../../Property_Image.gif)
		 [Caption](IVZChartAxis.Caption.htm)


		 Свойство Caption определяет
		 заголовок оси диаграммы.


		 ![](../../Property_Image.gif)
		 [DisplayUnit](IVZChartAxis.DisplayUnit.htm)


		 Свойство DisplayUnit
		 определяет масштаб оси диаграммы.


		 ![](../../Property_Image.gif)
		 [LabelStyle](IVZChartAxis.LabelStyle.htm)


		 Свойство LabelStyle
		 определяет стиль подписей на оси диаграммы.


		 ![](../../Property_Image.gif)
		 [Max](IVZChartAxis.Max.htm)


		 Свойство Max определяет
		 максимальное значение на оси диаграммы.


		 ![](../../Property_Image.gif)
		 [Min](IVZChartAxis.Min.htm)


		 Свойство Min определяет
		 минимальное значение на оси диаграммы.


		 ![](../../Property_Image.gif)
		 [Pen](IVZChartAxis.Pen.htm)


		 Свойство Pen определяет
		 перо, которым рисуется ось.


		 ![](../../Property_Image.gif)
		 [PrimaryGrid](IVZChartAxis.PrimaryGrid.htm)


		 Свойство PrimaryGrid
		 определяет стиль главных линий сетки диаграммы.


		 ![](../../Property_Image.gif)
		 [SecondaryGrid](IVZChartAxis.SecondaryGrid.htm)


		 Свойство SecondaryGrid
		 определяет стиль неосновных линий сетки диаграммы.


		 ![](../../Property_Image.gif)
		 [ShowLabels](IVZChartAxis.ShowLabels.htm)


		 Cвойство ShowLabels
		 определяет, будут ли отображаться подписи на оси диаграммы.


		 ![](../../Property_Image.gif)
		 [TicksPen](IVZChartAxis.TicksPen.htm)


		 Свойство TicksPen определяет
		 перо линий делений.
