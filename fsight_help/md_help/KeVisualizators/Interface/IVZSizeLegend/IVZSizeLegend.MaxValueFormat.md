# IVZSizeLegend.MaxValueFormat


## Синтаксис


MaxValueFormat: Variant;


## Описание


Свойство MaxValueFormat определяет
 формат записи максимального значения у размерной легенды визуализатора.


## Комментарии


Для определения формата записи минимального значения у размерной легенды
 визуализатора используйте свойство [IVZSizeLegend.MinValueFormat](IVZSizeLegend.MinValueFormat.htm).
