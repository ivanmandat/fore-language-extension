# IVZSizeLegend.MinValueFormat


## Синтаксис


MinValueFormat: Variant;


## Описание


Свойство MinValueFormat определяет
 формат записи минимального значения у размерной легенды визуализатора.


## Комментарии


Для определения формата записи максимального значения у размерной
 легенды визуализатора используйте свойство [IVZSizeLegend.MaxValueFormat](IVZSizeLegend.MaxValueFormat.htm).
