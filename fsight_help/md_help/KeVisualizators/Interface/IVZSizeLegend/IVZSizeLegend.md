# IVZSizeLegend


Сборка: Visualizators;


## Описание


Интерфейс IVZSizeLegend предназначен
 для работы с размерной легендой визуализатора.


## Иерархия наследования


           [IVZPlacedControlBase](../IVZPlacedControlBase/IVZPlacedControlBase.htm)


           [IVZLegendBase](../IVZLegendBase/IVZLegendBase.htm)


           IVZSizeLegend


## Комментарии


Параметры размерной легенды могут быть получены в свойстве [IVZLegends.Item](../IVZLegends/IVZLegends.Item.htm) после приведения к
 типу IVZSizeLegend.


Размерная легенда может иметь следующий вид:

## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Marker](IVZSizeLegend.Marker.htm)


		 Свойство Marker определяет
		 фигуру, используемую в качестве маркера элемента размерной легенды.


		 ![](../../Property_Image.gif)
		 [MaxValue](IVZSizeLegend.MaxValue.htm)


		 Свойство MaxValue определяет
		 максимальное значение для размерной шкалы визуализатора.


		 ![](../../Property_Image.gif)
		 [MaxValueFormat](IVZSizeLegend.MaxValueFormat.htm)


		 Свойство MaxValueFormat
		 определяет формат записи максимального значения у размерной легенды
		 визуализатора.


		 ![](../../Property_Image.gif)
		 [MinValue](IVZSizeLegend.MinValue.htm)


		 Свойство MinValue определяет
		 минимальное значение для размерной шкалы визуализатора.


		 ![](../../Property_Image.gif)
		 [MinValueFormat](IVZSizeLegend.MinValueFormat.htm)


		 Свойство MinValueFormat
		 определяет формат записи минимального значения у размерной легенды
		 визуализатора.


		 ![](../../Property_Image.gif)
		 [TextSpacing](IVZSizeLegend.TextSpacing.htm)


		 Свойство TextSpacing
		 определяет отступ подписей от графических элементов размерной
		 легенды визуализатора.
