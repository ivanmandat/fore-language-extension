# IVZBaseFormatter


Сборка: Visualizators;


## Описание


Интерфейс IVZBaseFormatter предназначен
 для форматирования данных визуализатора.


## Иерархия наследования


IVZBaseFormatter


## Комментарии


Объект типа IVZBaseFormatter
 может быть получен в свойстве [IVZBaseDataVisualizer.Formatter](../IVZBaseDataVisualizer/IVZBaseDataVisualizer.Formatter.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [NumberFormatter](IVZBaseFormatter.NumberFormatter.htm)
		 Свойство NumberFormatter
		 определяет формат числовых данных визуализатора.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Format](IVZBaseFormatter.Format.htm)
		 Метод Format заменяет
		 каждый элемент формата в указанной строке текстовым эквивалентом
		 значения соответствующего объекта из указанного массива.


		 ![](../../Sub_Image.gif)
		 [FormatById](IVZBaseFormatter.FormatById.htm)
		 Метод FormatById заменяет
		 каждый элемент формата в указанной строке текстовым эквивалентом
		 значения соответствующего объекта для указанного элемента визуализатора.


		 ![](../../Sub_Image.gif)
		 [FormatByParam](IVZBaseFormatter.FormatByParam.htm)
		 Метод FormatByParam
		 заменяет элемент формата в указанной строке текстовым эквивалентом
		 значения указанного объекта.


		 ![](../../Sub_Image.gif)
		 [FormatByParams](IVZBaseFormatter.FormatByParams.htm)
		 Метод FormatByParams
		 заменяет каждый элемент формата в указанной строке текстовым эквивалентом
		 значений соответствующих объектов.
