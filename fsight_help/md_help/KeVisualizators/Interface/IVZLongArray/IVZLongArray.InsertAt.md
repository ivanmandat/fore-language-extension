# IVZLongArray.InsertAt


## Синтаксис


InsertAt(Index: Integer, Value: Integer);


## Параметры


Index. Индекс добавляемого элемента;


Value. Значение добавляемого элемента.


## Описание


Метод InsertAt добавляет новый элемент в массив по указанному индексу.
