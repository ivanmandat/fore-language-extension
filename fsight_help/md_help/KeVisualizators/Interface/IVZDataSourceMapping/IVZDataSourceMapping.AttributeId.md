# IVZDataSourceMapping.AttributeId


## Синтаксис


AttributeId: String;


## Описание


Свойство AttributeId определяет
 идентификатор измерения, по которому выбираются данные из источника данных.


## Комментарии


Для определения источника данных используйте свойство [IVZDataSourceMapping.DataSource](IVZDataSourceMapping.DataSource.htm).
