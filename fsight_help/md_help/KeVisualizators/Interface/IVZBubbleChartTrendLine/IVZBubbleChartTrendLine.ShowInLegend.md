# IVZBubbleChartTrendLine.ShowInLegend


## Синтаксис


ShowInLegend: Boolean;


## Описание


Свойство ShowInLegend определяет,
 будет ли показана линия тренда в легенде диаграммы.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Линия тренда отображается в легенде диаграммы;


	- False. Линия тренда
	 не отображается в легенде диаграммы.
