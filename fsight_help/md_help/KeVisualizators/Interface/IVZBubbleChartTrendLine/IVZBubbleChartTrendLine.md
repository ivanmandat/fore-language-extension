# IVZBubbleChartTrendLine


Сборка: Visualizators;


## Описание


Интерфейс IVZBubbleChartTrendLine
 предназначен для работы с линией тренда [пузырьковой
 диаграммы](../IVZBubbleChart/IVZBubbleChart.htm).


## Иерархия наследования


           [IVZChartObject](../IVZChartObject/IVZChartObject.htm)


           IVZBubbleChartTrendLine

		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Name](IVZBubbleChartTrendLine.Name.htm)
		 Свойство Name определяет
		 пользовательское наименование линии тренда в легенде.


		 ![](../../Property_Image.gif)
		 [Pen](IVZBubbleChartTrendLine.Pen.htm)
		 Свойство Pen определяет
		 перо, которым рисуется линия тренда.

		 [ShowInLegend](IVZBubbleChartTrendLine.ShowInLegend.htm)
		 Свойство ShowInLegend
		 определяет, будет ли показана линия тренда в легенде диаграммы.


		 ![](../../Property_Image.gif)
		 [Type](IVZBubbleChartTrendLine.Type.htm)
		 Свойство Type определяет
		 тип линии тренда.


		 ![](../../Property_Image.gif)
		 [Visible](IVZBubbleChartTrendLine.Visible.htm)
		 Свойство Visible определяет
		 видимость линии тренда.
