# IVZControlStyle.SetBorderPen


## Синтаксис


SetBorderPen(state: [VisualizatorControlState](../../Enums/VisualizatorControlState.htm), Value: IGxPenBase);


## Параметры


state. Состояние элемента управления;


Value. Перо, описывающее стиль границы элемента управления.


## Описание


Метод SetBorderPen устанавливает перо, определяющее стиль границ элемента управления для указанного состояния.
