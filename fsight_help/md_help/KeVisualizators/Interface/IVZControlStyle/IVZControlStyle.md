# IVZControlStyle


Сборка: Visualizators;


## Описание


Интерфейс IVZControlStyle предназначен
 для работы со стилем элементов управления.


## Иерархия наследования


IVZControlStyle


## Комментарии


Данный интерфейс является базовым для следующих интерфейсов:


	- [IVZLabelStyle](../IVZLabelStyle/IVZLabelStyle.htm).


Параметры стиля элементов управления могут быть получены в свойстве
 [IVZPlacedControlBase.Style](../IVZPlacedControlBase/IVZPlacedControlBase.Style.htm).

		 [Background](IVZControlStyle.Background.htm)

		 [Margin](IVZControlStyle.Margin.htm)
