# IVZControlStyle.GetBorderPen


## Синтаксис


GetBorderPen(state: [VisualizatorControlState](../../Enums/VisualizatorControlState.htm)): IGxPenBase;


## Параметры


state. Состояние элемента управления.


## Описание


Метод GetBorderPen возвращает перо, определяющее стиль границ элемента управления для указанного состояния.
