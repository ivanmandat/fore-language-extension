# IVZControlStyle.GetBackground


## Синтаксис


GetBackground(state: [VisualizatorControlState](../../Enums/VisualizatorControlState.htm)): IGxBrush;


## Параметры


state. Состояние элемента управления.


## Описание


Метод GetBackground возвращает кисть, определяющую фон элемента управления для указанного состояния.
