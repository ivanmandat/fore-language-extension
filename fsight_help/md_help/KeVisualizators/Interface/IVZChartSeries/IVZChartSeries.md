# IVZChartSeries


Сборка: Visualizators;


## Описание


Интерфейс IVZChartSeries предназначен
 для работы с коллекцией рядов данных [пузырьковой
 диаграммы](UiVisualizators.chm::/BubbleChart/BubbleChart.htm).


## Иерархия наследования


IVZChartSeries


## Комментарии


Коллекция рядов данных пузырьковой диаграммы может быть получена с помощью
 свойства [IVZBubbleChart.Series](../IVZBubbleChart/IVZBubbleChart.Series.htm).
