# IVZChartSeries.ItemById


## Синтаксис


ItemById(Value: String): [IVZChartSerie](../IVZChartSerie/IVZChartSerie.htm);


## Параметры


Value. Идентификатор ряда данных
 пузырьковой диаграммы.


## Описание


Свойство ItemById возвращает
 ряд данных по идентификатору.


## Комментарии


Для определения идентификатора ряда данных пузырьковой диаграммы используйте
 свойство [IVZChartSerie.Id](../IVZChartSerie/IVZChartSerie.Id.htm).
