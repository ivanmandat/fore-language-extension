# IVZColorVisual


Сборка: Visualizators;


## Описание


Интерфейс IVZColorVisual предназначен
 для сопоставления данных визуализатора цветам заливки его дочерних элементов.


## Иерархия наследования


           [IVZDataVisual](../IVZDataVisual/IVZDataVisual.htm)


           IVZColorVisual


## Комментарии


Настройки сопоставления данных визуализатора цветам заливки его дочерних
 элементов могут быть получены в следующих свойствах:


	- [IVZBubbleTree.ColorVisual](../IVZBubbleTree/IVZBubbleTree.ColorVisual.htm);


	- [IVZTreeMap.ColorVisual](../IVZTreeMap/IVZTreeMap.ColorVisual.htm).
