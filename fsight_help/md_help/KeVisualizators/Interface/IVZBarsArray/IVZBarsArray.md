# IVZBarsArray


Сборка: Visualizators;


## Описание


Интерфейс IVZBarsArray используется
 для работы с массивом [столбиковых показателей](UiMaps.chm::/Configure/Diagrams.htm)
 на слое карты.


## Иерархия наследования


           IVZBarsArray


## Комментарии


Столбиковый показатель позволяет
 отобразить дополнительную информацию о территориях, например соотношение
 значений показателя.


Для получения массива [столбиковых
 показателей](UiMaps.chm::/Configure/Diagrams.htm)
 на слое карты используйте [IVZMapChartLayer.Bars](../IVZMapChartLayer/IVZMapChartLayer.Bars.htm).

## Методы


		 Имя метода
		 Краткое описание

		 ![](../../Sub_Image.gif)
