# IVZBarsArray.Item


## Синтаксис


Item(Index: Integer): [IVZMapChartBar](../IVZMapChartBar/IVZMapChartBar.htm);


## Параметры


Index. Индекс столбикового
 показателя.


## Описание


Свойство Item возвращает
 [столбиковый
 показатель](UiMaps.chm::/Configure/Diagrams.htm)
 по его индексу.


## Комментарии


Для получения количества столбиковых показателей в коллекции используйте
 [IVZBarsArray.Count](IVZBarsArray.Count.htm).
