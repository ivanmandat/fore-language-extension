# IVZBarsArray.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 [столбиковых
 показателей](UiMaps.chm::/Configure/Diagrams.htm)
 в коллекции на слое карты.


## Комментарии


Для получения столбикового показателя по его индексу используйте [IVZBarsArray.Item](IVZBarsArray.Item.htm).
