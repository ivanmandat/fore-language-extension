## Описание


Метод Clear очищает коллекцию
 [столбиковых
 показателей](UiMaps.chm::/Configure/Diagrams.htm)
 на слое карты.

Для добавления столбикового показателя в коллекцию используйте [IVZBarsArray.Add](IVZBarsArray.Add.htm).
