# IVZBarsArray.Remove


## Синтаксис


Remove(Index: Integer);


## Описание


Метод Remove удаляет [столбиковый показатель](UiMaps.chm::/Configure/Diagrams.htm)
 из коллекции по его индексу.


## Комментарии


Для добавления столбикового показателя в коллекцию используйте [IVZBarsArray.Add](IVZBarsArray.Add.htm).
