# IVZBarsArray.Add


## Синтаксис


Add(Value: [IVZMapChartBar](../IVZMapChartBar/IVZMapChartBar.htm)):
 Integer;


## Параметры


Value. Столбиковый показатель.


## Описание


Метод Add добавляет указанный
 [столбиковый
 показатель](UiMaps.chm::/Configure/Diagrams.htm)
 в коллекцию на слое карты.


## Комментарии


Для удаления столбикового показателя из коллекции по его индексу используйте
 [IVZBarsArray.Remove](IVZBarsArray.Remove.htm).


Для очищения коллекции столбиковых показателей используйте [IVZBarsArray.Clear](IVZBarsArray.Clear.htm).
