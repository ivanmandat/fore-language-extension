# IVZTimeLine


Сборка: Visualizators;


## Описание


Интерфейс IVZTimeLine предназначен
 для работы с временной шкалой визуализатора.


## Иерархия наследования


           [IVZPlacedControlBase](../IVZPlacedControlBase/IVZPlacedControlBase.htm)


           IVZTimeLine


## Комментарии


Временная шкала визуализатора может иметь следующий вид:


![](IVZTimeLine.png)


Для определения параметров шкалы используйте свойство [IVZDataVisualizer.TimeLine](../IVZDataVisualizer/IVZDataVisualizer.TimeLine.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [CurrentIndex](IVZTimeLine.CurrentIndex.htm)


		 Свойство CurrentIndex
		 определяет индекс текущего шага временной шкалы.


		 ![](../../Property_Image.gif)
		 [DelayDuration](IVZTimeLine.DelayDuration.htm)


		 Свойство DelayDuration
		 определяет длительность задержки анимации визуализатора на каждом
		 шаге временной шкалы.


		 ![](../../Property_Image.gif)
		 [Inverse](IVZTimeLine.Inverse.htm)


		 Свойство Inverse определяет,
		 отображается ли временная шкала в прямом или реверсивном виде.


		 ![](../../Property_Image.gif)
		 [IsOverlap](IVZTimeLine.IsOverlap.htm)


		 Свойство IsOverlap
		 возвращает признак того, что временная шкала не уменьшает рабочую
		 область визуализатора.


		 ![](../../Property_Image.gif)
		 [LastIndexAsInitial](IVZTimeLine.LastIndexAsInitial.htm)


		 Свойство LastIndexAsInitial
		 определяет, будет ли при открытии экспресс-отчета выполняться
		 переход на последний шаг временной шкалы при отрицательном значении
		 индекса.


		 ![](../../Property_Image.gif)
		 [ReturnToInitialState](IVZTimeLine.ReturnToInitialState.htm)


		 Свойство ReturnToInitialState
		 устанавливает признак возврата временной шкалы в начальное состояние.


		 ![](../../Property_Image.gif)
		 [StepDuration](IVZTimeLine.StepDuration.htm)


		 Свойство StepDuration
		 определяет длительность воспроизведения анимации визуализатора
		 при переходе от одного шага временной шкалы к другому.


		 ![](../../Property_Image.gif)
		 [StepsNames](IVZTimeLine.StepsNames.htm)


		 Свойство StepsNames
		 возвращает наименования шагов временной шкалы.
