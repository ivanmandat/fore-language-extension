# IVZTimeLine.IsOverlap


## Синтаксис


IsOverlap: Boolean;


## Описание


Свойство IsOverlap возвращает
 признак того, что временная шкала не уменьшает рабочую область визуализатора.


## Комментарии


Свойство содержит значение True,
 если временная шкала накладывается на рабочую область визуализатора, в
 противном случае - значение False.


По умолчанию свойство имеет значение False.
