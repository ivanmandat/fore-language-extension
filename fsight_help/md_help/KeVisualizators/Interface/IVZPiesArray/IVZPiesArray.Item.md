# IVZPiesArray.Item


## Синтаксис


Item(Index: Integer): [IVZMapChartPie](../IVZMapChartPie/IVZMapChartPie.htm);


## Параметры


Index. Индекс кругового показателя.


## Описание


Свойство Item возвращает
 круговой показатель по его индексу.


## Комментарии


Для получения количества круговых показателей в коллекции используйте
 свойство [IVZPiesArray.Count](IVZPiesArray.Count.htm).
