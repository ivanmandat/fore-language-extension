# IVZPiesArray.Remove


## Синтаксис


Remove(Index: Integer);


## Описание


Метод Remove удаляет круговой
 показатель из коллекции по его индексу.


## Комментарии


Для добавления кругового показателя в коллекцию используйте метод [IVZPiesArray.Add](IVZPiesArray.Add.htm).
