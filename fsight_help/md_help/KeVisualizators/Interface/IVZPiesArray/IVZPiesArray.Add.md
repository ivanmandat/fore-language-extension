# IVZPiesArray.Add


## Синтаксис


Add(Value: [IVZMapChartPie](../IVZMapChartPie/IVZMapChartPie.htm)):
 Integer;


## Параметры


Value. Круговой показатель.


## Описание


Метод Add добавляет указанный
 круговой показатель в коллекцию на слое карты.


## Комментарии


Для удаления кругового показателя из коллекции по его индексу используйте
 метод [IVZPiesArray.Remove](IVZPiesArray.Remove.htm).


Для очищения коллекции круговых показателей используйте метод [IVZPiesArray.Clear](IVZPiesArray.Clear.htm).
