# IVZPiesArray.Count


## Синтаксис


Count: Integer;


## Описание


Свойство Count возвращает количество
 круговых показателей в коллекции на слое карты.


## Комментарии


Для получения кругового показателя по его индексу используйте свойство
 [IVZPiesArray.Item](IVZPiesArray.Item.htm).
