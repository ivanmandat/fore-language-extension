# IVZPiesArray.Clear


## Синтаксис


Clear;


## Описание


Метод Clear очищает коллекцию
 круговых показателей на слое карты.


## Комментарии


Для добавления кругового показателя в коллекцию используйте метод [IVZPiesArray.Add](IVZPiesArray.Add.htm).
