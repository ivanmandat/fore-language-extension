# IVZPiesArray.InsertAt


## Синтаксис


InsertAt(Index: Integer; Value: [IVZMapChartPie](../IVZMapChartPie/IVZMapChartPie.htm));


## Параметры


Index. Индекс позиции, в которую
 необходимо вставить круговой показатель;


Value. Круговой показатель.


## Описание


Метод InsertAt вставляет указанный
 круговой показатель в указанную позицию в коллекции.


## Комментарии


Для добавления кругового показателя в коллекцию используйте метод [IVZMapChartArrows.Add](../IVZMapChartArrows/IVZMapChartArrows.Add.htm).
