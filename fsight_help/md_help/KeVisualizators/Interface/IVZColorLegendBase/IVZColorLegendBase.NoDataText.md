# IVZColorLegendBase.NoDataText


## Синтаксис


NoDataText: Variant;


## Описание


Свойство NoDataText определяет
 подпись для элемента, соответствующего отсутствующим данным, в визуализаторе.


## Комментарии


Для определения формата записи интервалов с обеими границами используйте
 [IVZColorLegendBase.BetweenFormat](IVZColorLegendBase.BetweenFormat.htm).
