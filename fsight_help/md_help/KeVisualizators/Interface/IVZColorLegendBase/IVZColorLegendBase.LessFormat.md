# IVZColorLegendBase.LessFormat


## Синтаксис


LessFormat: Variant;


## Описание


Свойство LessFormat определяет
 формат записи интервалов с правой границей.


## Комментарии


Для определения формата записи интервалов с левой границей используйте
 [IVZColorLegendBase.LessFormat](IVZColorLegendBase.LessFormat.htm).
