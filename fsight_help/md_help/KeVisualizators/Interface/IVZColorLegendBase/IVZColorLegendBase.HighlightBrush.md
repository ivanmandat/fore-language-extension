# IVZColorLegendBase.HighlightBrush


## Синтаксис


HighlightBrush: [IGxBrush](ModDrawing.chm::/Interface/IGxBrush/IGxBrush.htm);


## Описание


Свойство HighlightBrush определяет
 кисть для подсветки элементов легенды визуализатора.


## Комментарии


Для включения/отключения подсветки элементов легенды используйте свойство
 [IVZColorLegendBase.DoHighlight](IVZColorLegendBase.DoHighlight.htm).


Для задания действия, при выполнении
 которого будет подсвечиваться элемент легенды, используйте свойство
 [IVZColorLegendBase.HoverMode](IVZColorLegendBase.HoverMode.htm).
