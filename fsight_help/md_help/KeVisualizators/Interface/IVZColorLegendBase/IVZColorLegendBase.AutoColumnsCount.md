# IVZColorLegendBase.AutoColumnsCount


## Синтаксис


AutoColumnsCount: Boolean;


## Описание


Свойство AutoColumnsCount определяет,
 будет ли количество столбцов для пунктов цветовой легенды устанавливаться
 автоматически.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Количество столбцов в легенде будет устанавливаться автоматически;


	- False. Количество столбцов
	 будет задаваться пользователем. Задается в свойстве [IVZIntervalsLegend.ColumnsCount](../IVZIntervalsLegend/IVZIntervalsLegend.ColumnsCount.htm)
	 после приведения интерфейса [IVZColorLegendBase](IVZColorLegendBase.htm) к
	 [IVZIntervalsLegend](../IVZIntervalsLegend/IVZIntervalsLegend.htm).


Для получения коллекции пунктов цветовой легенды используйте [IVZColorLegendBase.Items](IVZColorLegendBase.Items.htm).
