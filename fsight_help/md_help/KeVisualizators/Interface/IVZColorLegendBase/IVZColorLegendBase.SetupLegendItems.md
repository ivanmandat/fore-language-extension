# IVZColorLegendBase.SetupLegendItems


## Синтаксис


SetupLegendItems(dataMapping: [IVZBaseDataSourceMapping](../IVZBaseDataSourceMapping/IVZBaseDataSourceMapping.htm));


## Параметры


dataMapping. Настройки сопоставления
 данных из источника.


## Описание


Метод SetupLegendItems задаёт
 настройки сопоставления данных из источника для цветовой шкалы легенды
 визуализатора.


## Комментарии


Метод позволяет устанавливать элементы цветовой шкалы легенды независимо
 от данных на родительском визуализаторе, если свойство [IVZLegendBase.IsDependent](../IVZLegendBase/IVZLegendBase.IsDependent.htm)
 принимает значение False.
