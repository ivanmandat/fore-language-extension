# IVZColorLegendBase.GreaterFormat


## Синтаксис


GreaterFormat: Variant;


## Описание


Свойство GreaterFormat определяет
 формат записи интервалов с левой границей.


## Комментарии


Для определения формата записи интервалов с правой границей используйте
 [IVZColorLegendBase.LessFormat](IVZColorLegendBase.LessFormat.htm).
