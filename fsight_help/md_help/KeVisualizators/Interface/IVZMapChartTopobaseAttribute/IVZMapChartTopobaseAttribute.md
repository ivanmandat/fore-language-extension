# IVZMapChartTopobaseAttribute


Сборка: Visualizators;


## Описание


Интерфейс IVZMapChartTopobaseAttribute
 предназначен для работы с атрибутом топоосновы карты.


## Иерархия наследования


           IVZMapChartTopobaseAttribute


## Комментарии


Топооснова - набор графических слоёв информации, наложенных друг на
 друга.


Для получения интерфейса используется свойство IVZMapChartTopobaseAttributesDictionary.Item.
