# IVZBubbleChart.ShowHistoryMode


## Синтаксис


ShowHistoryMode: [VisualizatorHistoryModeEnum](../../Enums/VisualizatorHistoryModeEnum.htm);


## Описание


Свойство ShowHistoryMode определяет
 режим отображения истории перемещения пузырьков.


## Комментарии


Для использования данного свойства необходимо, чтобы свойство [IVZBubbleChart.ShowBubbleHistory](IVZBubbleChart.ShowBubbleHistory.htm) принимало
 значение True.
