# IVZBubbleChart.ShowHistoryPreviousStepsOnly


## Синтаксис


ShowHistoryPreviousStepsOnly: Boolean;


## Описание


Свойство ShowHistoryPreviousStepsOnly
 определяет, будет ли отображаться история до текущей позиции (года).


## Комментарии


Допустимые значения:


	- True. История до текущей
	 позиции (года) будет отображаться;


	- False. По умолчанию.
	 История до текущей позиции (года) отображаться не будет.


Для использования данного свойства необходимо, чтобы свойство [IVZBubbleChart.ShowBubbleHistory](IVZBubbleChart.ShowBubbleHistory.htm) принимало
 значение True.
