# IVZBubbleChart.UseZoomOnWheel


## Синтаксис


UseZoomOnWheel: Boolean;


## Описание


Свойство UseZoomOnWheel определяет
 возможность изменения масштаба пузырьковой диаграммы при прокрутке колесика
 мыши.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Изменение масштаба пузырьковой диаграммы при прокрутке колесика мыши
	 возможно;


	- False. Изменение масштаба
	 пузырьковой диаграммы при прокрутке колесика мыши невозможно;
