# IVZBubbleChart.ShowBubbleHistoryLabels


## Синтаксис


ShowBubbleHistoryLabels: Boolean;


## Описание


Свойство ShowBubbleHistoryLabels
 определяет, будут ли отображаться годовые подсказки у пузырьков при включенной
 истории перемещения.


## Комментарии


Допустимые значения:


	- True. Годовые подсказки
	 у пузырьков при включенной истории перемещения будут отображаться;


	- False. По умолчанию.
	 Годовые подсказки у пузырьков при включенной истории перемещения отображаться
	 не будут.


Для использования данного свойства необходимо, чтобы свойство [IVZBubbleChart.ShowBubbleHistory](IVZBubbleChart.ShowBubbleHistory.htm) принимало
 значение True.
