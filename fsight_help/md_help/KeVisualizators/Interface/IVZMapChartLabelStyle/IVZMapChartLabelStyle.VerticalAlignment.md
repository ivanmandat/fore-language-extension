# IVZMapChartLabelStyle.VerticalAlignment


## Синтаксис


VerticalAlignment: [VisualizerLabelVerticalAlignment](../../Enums/VisualizerLabelVerticalAlignment.htm);


## Описание


Свойство VerticalAlignment
 определяет положение подписей слоя карты по вертикали относительно позиции
 из топоосновы.


## Комментарии


Для определения положения подписи слоя карты по горизонтали используйте
 свойство [IVZMapChartLabelStyle.HorizontalAlignment](IVZMapChartLabelStyle.HorizontalAlignment.htm).
