# IVZResourceImage


Сборка: Visualizators;


## Описание


Интерфейс IVZResourceImage предназначен
 для работы с ресурсами размерной легенды визуализатора.


## Иерархия наследования


IVZResourceImage


## Комментарии


Ресурсы задаются для пиктограмм максимального и минимального значений
 размерной легенды с помощью свойств [IVZSizeLegendMarker.BigImage](../IVZSizeLegendMarker/IVZSizeLegendMarker.BigImage.htm),
 [IVZSizeLegendMarker.SmallImage](../IVZSizeLegendMarker/IVZSizeLegendMarker.SmallImage.htm).


Параметры размерной легенды могут быть получены в свойстве [IVZLegends.Item](../IVZLegends/IVZLegends.Item.htm) после приведения
 к типу [IVZSizeLegend](../IVZSizeLegend/IVZSizeLegend.htm).


Для изменения маркера элемента размерной легенды используйте свойство
 [IVZResourceImage.ResourceId](IVZResourceImage.ResourceId.htm)
 или [IVZResourceImage.Bitmap](IVZResourceImage.Bitmap.htm).
