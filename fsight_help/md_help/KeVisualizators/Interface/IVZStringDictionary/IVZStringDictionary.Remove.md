# IVZStringDictionary.Remove


## Синтаксис


Remove(Value: String);


## Параметры


Value. Ключ элемента, удаляемого
 из словаря строковых данных.


## Описание


Метод Remove удаляет элемент
 из словаря строковых данных по указанному ключу.
