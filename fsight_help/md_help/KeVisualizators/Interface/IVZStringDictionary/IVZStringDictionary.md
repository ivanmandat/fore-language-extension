# IVZStringDictionary


Сборка: Visualizators;


## Описание


Интерфейс IVZStringDictionary
 предназначен для работы со словарём строковых данных.


## Иерархия наследования


           IVZStringDictionary


## Комментарии


Для получения словаря строковых данных используйте свойство [IVZMapChartTopobaseAttribute.Data](../IVZMapChartTopobaseAttribute/IVZMapChartTopobaseAttribute.Data.htm).
