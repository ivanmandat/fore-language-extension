# IVZStringDictionary.Add


## Синтаксис


Add(Key: String; Value: String);


## Параметры


Key. Ключ элемента, добавляемого
 в словарь строковых данных;


Value. Строковые данные, добавляемые
 в словарь.


## Описание


Метод Add добавляет новый элемент
 в словарь строковых данных.
