# IVZSizeLegendMarker


Сборка: Visualizators;


## Описание


Интерфейс IVZSizeLegendMarker
 предназначен для работы с маркером размерной легенды визуализатора.


## Иерархия наследования


IVZSizeLegendMarker


## Комментарии


Параметры размерной легенды могут быть получены в свойстве [IVZLegends.Item](../IVZLegends/IVZLegends.Item.htm) после приведения к
 типу [IVZSizeLegend](../IVZSizeLegend/IVZSizeLegend.htm).
