# IVZHighlightEffect.ROffset


## Синтаксис


ROffset: Integer;


## Описание


Свойство ROffset определяет
 интенсивность красной составляющей цвета.


## Комментарии


Допустимые значения находятся в диапазоне [0, 255].


Для определения интенсивности зеленой составляющей цвета используйте
 [IVZHighlightEffect.GOffset](IVZHighlightEffect.GOffset.htm).


Для определения интенсивности синей составляющей цвета используйте [IVZHighlightEffect.BOffset](IVZHighlightEffect.BOffset.htm).


Для определения уровня прозрачности цвета используйте [IVZHighlightEffect.AOffset](IVZHighlightEffect.AOffset.htm).
