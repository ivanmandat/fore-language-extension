# IVZHighlightEffect


Сборка: Visualizators;


## Описание


Интерфейс IVZHighlightEffect
 используется для работы с эффектом засветления элемента управления.


## Иерархия наследования


           [IVZEffect](../IVZEffect/IVZEffect.htm)


           IVZHighlightEffect


## Комментарии


Эффект засветления уменьшает
 яркость выделенного элемента управления.


Для эффекта используются три составляющие цвета (RGB) и прозрачность
 (A).


Интерфейс IVZHighlightEffect
 можно использовать для настройки эффектов через [IVZDataVisualizer](../IVZDataVisualizer/IVZDataVisualizer.htm)
 (HoverActiveEffects, HoverInactiveEffects, SelectActiveEffects, SelectInactiveEffects)
 и[ IVZMapChartShape](../IVZMapChartShape/IVZMapChartShape.htm)
 (Effects).
