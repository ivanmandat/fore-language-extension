# IVZNumericVisual


Сборка: Visualizators;


## Описание


Интерфейс IVZNumericVisual предназначен
 для сопоставления данных визуализатора числовым значениям.


## Иерархия наследования


           [IVZDataVisual](../IVZDataVisual/IVZDataVisual.htm)


           IVZNumericVisual


## Комментарии


Настройки для сопоставления данных визуализатора числовым значениям
 могут быть получены в следующих свойствах:


	- [IVZBubbleTree.SizeVisual](../IVZBubbleTree/IVZBubbleTree.SizeVisual.htm);


	- [IVZTreeMap.HeightVisual](../IVZTreeMap/IVZTreeMap.HeightVisual.htm);


	- [IVZTreeMap.SizeVisual](../IVZTreeMap/IVZTreeMap.SizeVisual.htm).
