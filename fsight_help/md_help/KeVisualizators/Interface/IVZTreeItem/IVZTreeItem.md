# IVZTreeItem


Сборка: Visualizators;


## Описание


Интерфейс IVZTreeItem предназначен
 для работы с элементами дерева визуализатора.


## Иерархия наследования


IVZTreeItem


## Комментарии


Данный интерфейс является базовым для следующих интерфейсов:


	- [IVZBubbleTreeItem](../IVZBubbleTreeItem/IVZBubbleTreeItem.htm);


	- [IVZTreeMapItem](../IVZTreeMapItem/IVZTreeMapItem.htm).


Для получения параметров визуализатора используйте следующие свойства:


	- [IVZTreeItem.Parent](IVZTreeItem.Parent.htm);


	- [IVZTreeItems.Item](../IVZTreeItems/IVZTreeItems.Item.htm);


	- [IVZTreeModel.RootItem](../IVZTreeModel/IVZTreeModel.RootItem.htm);


	- [IVZTreeModel.ActiveItem](../IVZTreeModel/IVZTreeModel.ActiveItem.htm);


	- [IVZTreeModel.GetItemByID](../IVZTreeModel/IVZTreeModel.GetItemByID.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ID](IVZTreeItem.ID.htm)
		 Свойство ID вовзращает
		 идентификатор элемента дерева визуализатора.


		 ![](../../Property_Image.gif)
		 [Items](IVZTreeItem.Items.htm)
		 Свойство Items определяет
		 дочерние элементы у заданного элемента дерева визуализатора.


		 ![](../../Property_Image.gif)
		 [Parent](IVZTreeItem.Parent.htm)
		 Свойство Parent возвращает
		 родителя для заданного элемента дерева визуализатора.


		 ![](../../Property_Image.gif)
		 [Selected](IVZTreeItem.Selected.htm)
		 Свойство Selected определяет,
		 используется ли выделение области дерева визуализатора.
