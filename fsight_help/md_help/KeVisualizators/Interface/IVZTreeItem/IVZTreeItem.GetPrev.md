# IVZTreeItem.GetPrev


## Синтаксис


GetPrev(CircleSearch: Boolean): [IVZTreeItem](IVZTreeItem.htm);


## Параметры


CircleSearch. Признак определения
 последнего дочернего элемента при использовании метода для первого дочернего
 элемента дерева визуализатора.


## Описание


Метод GetPrev возвращает предыдущий
 дочерний элемент дерева визуализатора.


## Комментарии


Допустимые значения параметра CircleSearch:


	- True. Возвращается последний
	 дочерний элемент;


	- False. Возвращается
	 значение Null.
