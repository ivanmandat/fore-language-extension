# IVZTreeItem.GetNext


## Синтаксис


GetNext(CircleSearch: Boolean): [IVZTreeItem](IVZTreeItem.htm);


## Параметры


CircleSearch. Признак определения
 первого дочернего элемента при использовании метода для последнего дочернего
 элемента дерева визуализатора.


## Описание


Метод GetNext возвращает следующий
 дочерний элемент дерева визуализатора.


## Комментарии


Допустимые значения параметра CircleSearch:


	- True. Возвращается первый
	 дочерний элемент;


	- False. Возвращается
	 значение Null.
