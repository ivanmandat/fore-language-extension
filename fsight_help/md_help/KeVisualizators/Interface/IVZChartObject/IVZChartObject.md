# IVZChartObject


Сборка: Visualizators;


## Описание


Интерфейс IVZChartObject является
 базовым интерфейсом для работы с элементами диаграммы.


## Комментарии


Интерфейс не содержит собственных свойств и методов.


Для работы с областью диаграммы используйте интерфейс [IVZChartArea](../IVZChartArea/IVZChartArea.htm).


Для работы с координатной сеткой диаграммы используйте интерфейс
 [IVZChartGrid](../IVZChartGrid/IVZChartGrid.htm).


Для работы с параметрами оси [пузырьковой
 диаграммы](UiVisualizators.chm::/BubbleChart/BubbleChart.htm) используйте интерфейс [IVZChartAxis](../IVZChartAxis/IVZChartAxis.htm).


Для работы с рядом данных пузырьковой диаграммы используйте интерфейс
 [IVZChartSerie](../IVZChartSerie/IVZChartSerie.htm).


Для работы с линией тренда пузырьковой диаграммы используйте интерфейс
 [IVZBubbleChartTrendLine](../IVZBubbleChartTrendLine/IVZBubbleChartTrendLine.htm).
