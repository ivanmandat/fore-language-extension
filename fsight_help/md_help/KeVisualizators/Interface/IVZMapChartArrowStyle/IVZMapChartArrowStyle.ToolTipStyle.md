# IVZMapChartArrowStyle.ToolTipStyle


## Синтаксис


ToolTipStyle: [IVZToolTipStyle](../IVZToolTipStyle/IVZToolTipStyle.htm);


## Описание


Свойство ToolTipStyle определяет
 стиль всплывающей подсказки стрелки на карте.


## Комментарии


В качестве стиля всплывающей
 подсказки стрелки допустимо использование стиля всплывающей подсказки
 любого визуализатора.
