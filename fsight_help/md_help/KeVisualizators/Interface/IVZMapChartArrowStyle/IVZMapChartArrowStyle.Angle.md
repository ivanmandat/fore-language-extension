# IVZMapChartArrowStyle.Angle


## Синтаксис


Angle: Double;


## Описание


Свойство Angle определяет угол
 изгиба стрелки на карте.


## Комментарии


Чем больше угол изгиба (по модулю), тем более выгнутой будет стрелка.
 Допустимые значения [-89; 89].


Для определения типа стрелки на карте используйте [IVZMapChartArrowStyle.ArrowType](IVZMapChartArrowStyle.ArrowType.htm).
