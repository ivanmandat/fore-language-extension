# IVZMapChartArrowStyle


Сборка: Visualizators;


## Описание


Интерфейс IVZMapChartArrowStyle
 используется для работы со стилем стрелки на карте.


## Иерархия наследования


           IVZMapChartArrowStyle


## Комментарии


Стиль стрелки включает: формат стрелки, формат линии стрелки, формат
 подписи стрелки.
