# IVZMapChartArrowStyle.Border


## Синтаксис

## Описание


Свойство Border определяет параметры
 пера для контура стрелки на карте.


## Комментарии


Для определения визуального представления стрелки на карте используйте
 [IVZMapChartArrowStyle.ArrowVisual](IVZMapChartArrowStyle.ArrowVisual.htm).
