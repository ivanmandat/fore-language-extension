# IVZMapChartArrowStyle.PointerWidth


## Синтаксис


PointerWidth: Double;


## Описание


Свойство PointerWidth определяет
 ширину наконечника стрелки на карте.


## Комментарии


Если свойство [IVZMapChartArrowStyle.IsRelativeWidth](IVZMapChartArrowStyle.IsRelativeWidth.htm)
 принимает значение True, то значения
 свойства IVZMapChartArrowStyle.PointerLength
 задаются в относительных координатах - [0, 1]. Если - False,
 то значения свойства задаются в пикселях [0, 100].


Для определения длины наконечника стрелки на карте используйте [IVZMapChartArrowStyle.PointerLength](IVZMapChartArrowStyle.PointerLength.htm).
