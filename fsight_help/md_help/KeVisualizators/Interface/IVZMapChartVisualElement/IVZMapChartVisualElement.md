# IVZMapChartVisualElement


Сборка: Visualizators;


## Описание


Интерфейс IVZMapChartVisualElement
 используется для работы с дополнительным визуальным элементом на карте.


## Иерархия наследования


           [IVZMapChartBaseObject](../IVZMapChartBaseObject/IVZMapChartBaseObject.htm)


           IVZMapChartVisualElement


## Комментарии


К дополнительным визуальным элементам на карте относятся:


	- [круговые
	 показатели](UiMaps.chm::/Configure/Diagrams.htm);


	- [столбиковые
	 показатели](UiMaps.chm::/Configure/Diagrams.htm);


	- [пузырьковые
	 показатели](../IVZBubblesArray/IVZBubblesArray.htm).
