# IVZMapChartArrowVisual


Сборка: Visualizators;


## Описание


Интерфейс IVZMapChartArrowVisual
 используется для работы с визуальным представлением стрелки на карте.


## Иерархия наследования


           [IVZDataVisual](../IVZDataVisual/IVZDataVisual.htm)


           IVZMapChartArrowVisual


## Комментарии


Для получения визуального представления стрелки на карте используйте
 [IVZMapChartArrowStyle.ArrowVisual](../IVZMapChartArrowStyle/IVZMapChartArrowStyle.ArrowVisual.htm).
