# IVZMapChartArrowVisual.BasementWidthMapping


## Синтаксис


BasementWidthMapping: [IVZDataSourceMapping](../IVZDataSourceMapping/IVZDataSourceMapping.htm);


## Описание


Свойство BasementWidthMapping
 определяет настройки сопоставления размеру стрелок на карте.


## Комментарии


Для определения настройки сопоставления данных цвету стрелок на карте
 используйте [IVZMapChartArrowVisual.BackgroundMapping](IVZMapChartArrowVisual.BackGroundMapping.htm).
