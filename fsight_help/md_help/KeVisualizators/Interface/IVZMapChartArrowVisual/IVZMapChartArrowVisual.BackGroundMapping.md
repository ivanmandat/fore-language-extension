# IVZMapChartArrowVisual.BackgroundMapping


## Синтаксис


BackgroundMapping: [IVZDataSourceMapping](../IVZDataSourceMapping/IVZDataSourceMapping.htm);


## Описание


Свойство BackgroundMapping определяет
 настройки сопоставления данных цвету стрелок на карте.


## Комментарии


Для определения настройки сопоставления данных размеру стрелок на карте
 используйте [IVZMapChartArrowVisual.BasementWidthMapping](IVZMapChartArrowVisual.BasementWidthMapping.htm).
