# IVZPalette


Сборка: Visualizators;


## Описание


Интерфейс IVZPalette предназначен
 для работы с палитрой цветов для заливки фона визуализатора.


## Иерархия наследования


IVZPalette


## Комментарии


Параметры палитры цветов для заливки фона визуализатора могут быть получены
 в следующих свойствах:


	- [IVZBubbleTree.Palette](../IVZBubbleTree/IVZBubbleTree.Palette.htm);


	- [IVZTreeMap.Palette](../IVZTreeMap/IVZTreeMap.Palette.htm).
