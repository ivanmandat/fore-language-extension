# IVZPalette.BackgroundArray


## Синтаксис


BackgroundArray: [IVZDataArray](../IVZDataArray/IVZDataArray.htm);


## Описание


Свойство BackgroundArray возвращает массив цветов для заливки фона визуализатора.


## Комментарии


Свойство содержит массив объектов типа [IGxColor](ModDrawing.chm::/Interface/IGxColor/IGxColor.htm).
