# IVZChartArea


Сборка: Visualizators;


## Описание


Интерфейс IVZChartArea предназначен
 для работы с областью построения диаграммы.


## Иерархия наследования


           [IVZChartObject](../IVZChartObject/IVZChartObject.htm)


           IVZChartArea


## Комментарии


Параметры области построения диаграммы можно получить, используя свойство
 [IVZBubbleChart.ChartArea](../IVZBubbleChart/IVZBubbleChart.ChartArea.htm).
