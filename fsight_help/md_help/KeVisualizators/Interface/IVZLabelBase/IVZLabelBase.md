# IVZLabelBase


Сборка: Visualizators;


## Описание


Интерфейс IVZLabelBase представляет
 собой базовый интерфейс для работы с меткой в визуализаторе.


## Иерархия наследования


           IVZLabelBase


## Комментарии


Данный интерфейс является базовым для интерфейса [IVZLegendHeader](../IVZLegendHeader/IVZLegendHeader.htm).
