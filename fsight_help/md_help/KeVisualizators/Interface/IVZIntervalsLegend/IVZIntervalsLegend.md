# IVZIntervalsLegend


Сборка: Visualizators;


## Описание


Интерфейс IVZIntervalsLegend
 предназначен для работы с легендой визуализатора, имеющей интервалы.


## Иерархия наследования


[IVZPlacedControlBase](../IVZPlacedControlBase/IVZPlacedControlBase.htm)


    [IVZLegendBase](../IVZLegendBase/IVZLegendBase.htm)


        [IVZColorLegendBase](../IVZColorLegendBase/IVZColorLegendBase.htm)


            IVZIntervalsLegend


## Комментарии


Параметры легенды с интервалами могут быть получены в свойстве [IVZLegends.Item](../IVZLegends/IVZLegends.Item.htm)
 после приведения к типу IVZIntervalsLegend.

		 Свойство CellSpacing

		 легенды с интервалами.

		 Свойство ColorApplying
		 определяет тип заливки области, в которой находятся пункты легенды
		 визуализатора.


		 ![](../../Property_Image.gif)
		 [ColumnsCount](IVZIntervalsLegend.ColumnsCount.htm)
		 Свойство ColumnsCount
		 определяет количество столбцов у легенды с интервалами.

		 Свойство EqualFormat
		 определяет формат подписи интервалов, включающих в себя границы,
		 в легенде с интервалами.


		 ![](../../Property_Image.gif)
		 [HorizontalCellsAligment](IVZIntervalsLegend.HorizontalCellsAligment.htm)
		 Свойство HorizontalCellsAligment
		 определяет тип горизонтального выравнивания пунктов легенды визуализатора.


		 ![](../../Property_Image.gif)

		 определяет признак использования интервалов с границами в легенде
		 с интервалами.


		 ![](../../Property_Image.gif)
		 [Marker](IVZIntervalsLegend.Marker.htm)

		 маркер легенды.


		 ![](../../Property_Image.gif)

		 Свойство Mode определяет
		 режим отображения легенды с интервалами.


		 ![](../../Property_Image.gif)
		 [TextHorizontalAlignment](IVZIntervalsLegend.TextHorizontalAlignment.htm)
		 Свойство TextHorizontalAlignment
		 определяет горизонтальное выравнивание подписи у элемента легенды
		 с интервалами.


		 ![](../../Property_Image.gif)
		 [TextSpacing](IVZIntervalsLegend.TextSpacing.htm)
		 Свойство TextSpacing
