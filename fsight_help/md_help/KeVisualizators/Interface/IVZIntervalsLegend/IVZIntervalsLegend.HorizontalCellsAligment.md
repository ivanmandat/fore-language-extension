# IVZIntervalsLegend.HorizontalCellsAligment


## Синтаксис


HorizontalCellsAligment: [LegendItemsHorizontalAlignment](../../Enums/LegendItemsHorizontalAlignment.htm);


## Описание


Свойство HorizontalCellsAligment
 определяет тип горизонтального выравнивания пунктов легенды визуализатора.


## Комментарии


Для определения горизонтального выравнивания текста в пункте легенды
 используйте [IVZIntervalsLegend.TextHorizontalAlignment](IVZIntervalsLegend.TextHorizontalAlignment.htm).
