# IVZIntervalsLegend.TextHorizontalAlignment


## Синтаксис


TextHorizontalAlignment: [GxStringAlignment](moddrawing.chm::/Enums/GxStringAlignment.htm);


## Описание


Свойство TextHorizontalAlignment
 определяет горизонтальное выравнивание подписи у элемента легенды с интервалами.


## Комментарии


Для определения типа горизонтального выравнивания пунктов легенды визуализатора
 используйте [IVZIntervalsLegend.HorizontalCellsAligment](IVZIntervalsLegend.HorizontalCellsAligment.htm).
