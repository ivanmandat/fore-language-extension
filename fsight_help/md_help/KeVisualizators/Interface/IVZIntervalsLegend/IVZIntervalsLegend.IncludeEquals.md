# IVZIntervalsLegend.IncludeEquals


## Синтаксис


IncludeEquals: Boolean;


## Описание


Свойство IncludeEquals определяет
 признак использования интервалов с границами в легенде с интервалами.


## Комментарии


Свойство содержит значение True,
 если в легенде с интервалами используется нестрогое неравенство, в противном
 случае - значение False.
