# IVZIntervalsLegend.ColorApplying


## Синтаксис


ColorApplying: [LegendColorApplying](../../Enums/LegendColorApplying.htm);


## Описание


Свойство ColorApplying
 определяет тип заливки области, в которой находятся пункты легенды визуализатора.


## Комментарии


Для определения цвета для пункта легенды визуализатора используйте [IVZLegendItem.Color](../IVZLegendItem/IVZLegendItem.Color.htm).
