# IVZMapChartBar.LeastHeightPart


## Синтаксис


LeastHeightPart: Double;


## Описание


Свойство LeastHeightPart определяет
 минимальную высоту столбиков [столбикового показателя](UiMaps.chm::/Configure/Diagrams.htm)
 на слое карты.
