# IVZMapChartBar.MaxBarsCount


## Синтаксис


MaxBarsCount: Integer;


## Описание


Свойство MaxBarsCount определяет
 максимальное количество отображаемых столбиков [столбикового показателя](UiMaps.chm::/Configure/Diagrams.htm)
 на слое карты.
