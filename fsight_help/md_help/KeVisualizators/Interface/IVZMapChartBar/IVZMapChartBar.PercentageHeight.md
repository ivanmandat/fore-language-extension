# IVZMapChartBar.PercentageHeight


## Синтаксис


PercentageHeight: Double;


## Описание


Свойство PercentageHeight определяет
 высоту делений [столбикового
 показателя](UiMaps.chm::/Configure/Diagrams.htm)
 на слое карты при использовании процентного накопления.


## Комментарии


Для использования процентного накопления установите свойству [IVZMapChartBar.IsPercentage](IVZMapChartBar.IsPercentage.htm)
 значение True.
