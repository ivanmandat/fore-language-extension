# IVZMapChartBar.IsAllPeriod


## Синтаксис


IsAllPeriod: Boolean;


## Описание


Свойство IsAllPeriod определяет
 признак отображения данных в [столбиковом показателе](UiMaps.chm::/Configure/Diagrams.htm)
 на слое карты за весь период.


## Комментарии


Допустимые значения:


	- True. Данные отображаются
	 за весь период;


	- False. По умолчанию.
	 Данные отображаются по годам.
