# IVZMapChartBar.LeastWidthPart


## Синтаксис


LeastWidthPart: Double;


## Описание


Свойство LeastWidthPart определяет
 минимальную ширину столбиков [столбикового показателя](UiMaps.chm::/Configure/Diagrams.htm)
 на слое карты.
