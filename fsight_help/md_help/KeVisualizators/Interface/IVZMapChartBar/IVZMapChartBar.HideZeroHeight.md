# IVZMapChartBar.HideZeroHeight


## Синтаксис


HideZeroHeight: Boolean;


## Описание


Свойство HideZeroHeight определяет
 признак скрытия столбиков нулевой высоты в [столбиковом показателе](UiMaps.chm::/Configure/Diagrams.htm)
 на слое карты.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Столбики нулевой высоты будут скрыты;


	- False. Столбики нулевой
	 высоты будут отображаться.
