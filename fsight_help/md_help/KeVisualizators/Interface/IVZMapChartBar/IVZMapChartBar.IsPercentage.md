# IVZMapChartBar.IsPercentage


## Синтаксис


IsPercentager: Boolean;


## Описание


Свойство IsPercentager определяет
 признак использования процентного накопления в [столбиковом показателе](UiMaps.chm::/Configure/Diagrams.htm)
 на слое карты.


## Комментарии


Допустимые значения:


	- True. Для столбиков
	 используется процентное накопление;


	- False. Для столбиков
	 не используется процентное накопление.


Для настройки высоты делений при использовании процентного накопления
 используйте свойство [IVZMapChartBar.PercentageHeight](IVZMapChartBar.PercentageHeight.htm).
