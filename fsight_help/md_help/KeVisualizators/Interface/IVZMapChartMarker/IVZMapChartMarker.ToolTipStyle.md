# IVZMapChartMarker.ToolTipStyle


## Синтаксис


ToolTipStyle: [IVZToolTipStyle](../IVZToolTipStyle/IVZToolTipStyle.htm);


## Описание


Свойство ToolTipStyle определяет
 стиль всплывающих подсказок маркера геолокации.


## Комментарии


В качестве стиля всплывающих подсказок маркера геолокации допустимо
 использование стиля всплывающих подсказок любого визуализатора.
