# IVZLegendBase.FooterLine


## Синтаксис


FooterLine: [IGxPenBase](ModDrawing.chm::/Interface/IGxPenBase/IGxPenBase.htm);


## Описание


Свойство FooterLine определяет
 объект для работы с линией нижнего колонтитула легенды визуализатора.


## Комментарии


Для определения объекта для работы с линией верхнего колонтитула легенды
 визуализатора используйте [IVZLegendBase.HeaderLine](IVZLegendBase.HeaderLine.htm).
