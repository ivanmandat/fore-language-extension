# IVZLegendBase.Header


## Синтаксис


Header: [IVZLegendHeader](../IVZLegendHeader/IVZLegendHeader.htm);


## Описание


Свойство Header определяет заголовок легенды.


## Комментарии


Свойство Header отличается от свойства [Footer](IVZLegendBase.Footer.htm) тем, поле [VerticalAlignment](../IVZPlacedControlBase/IVZPlacedControlBase.VerticalAlignment.htm) у его значения должно быть равно элементу Top перечисления [VisualizatorVerticalAlignment](../../Enums/VisualizatorVerticalAlignment.htm).
