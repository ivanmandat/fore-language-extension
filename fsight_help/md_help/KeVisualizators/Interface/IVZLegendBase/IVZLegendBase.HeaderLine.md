# IVZLegendBase.HeaderLine


## Синтаксис


HeaderLine: [IGxPenBase](ModDrawing.chm::/Interface/IGxPenBase/IGxPenBase.htm);


## Описание


Свойство HeaderLine определяет
 объект для работы с линией верхнего колонтитула легенды визуализатора.


## Комментарии


Для определения объекта для работы с линией нижнего колонтитула легенды
 визуализатора используйте [IVZLegendBase.FooterLine](IVZLegendBase.FooterLine.htm).
