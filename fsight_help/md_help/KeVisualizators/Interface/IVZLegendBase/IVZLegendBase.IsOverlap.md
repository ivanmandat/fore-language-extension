# IVZLegendBase.IsOverlap


## Синтаксис


IsOverlap: Boolean;


## Описание


Свойство IsOverlap определяет признак того, что легенда не уменьшает рабочую область визуализатора.


## Комментарии


Свойство содержит значение True, если легенда не уменьшает рабочую область визуализатора, то есть накладывается на неё, в противном случае - значение False
