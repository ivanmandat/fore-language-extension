# IVZLegendBase.Footer


## Синтаксис


Footer: [IVZLegendHeader](../IVZLegendHeader/IVZLegendHeader.htm);


## Описание


Свойство Footer определяет нижний колонтитул легенды.


## Комментарии


Свойство Footer отличается от свойства [Header](IVZLegendBase.Header.htm) тем, что поле [VerticalAlignment](../IVZPlacedControlBase/IVZPlacedControlBase.VerticalAlignment.htm) у его значения должно быть равно элементу Bottom перечисления [VisualizatorVerticalAlignment](../../Enums/VisualizatorVerticalAlignment.htm).
