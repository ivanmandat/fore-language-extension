# IVZLegendBase


Сборка: Visualizators;


## Описание


Интерфейс IVZLegendBase представляет
 собой базовый интерфейс для работы с легендой визуализатора.


## Иерархия наследования


           [IVZPlacedControlBase](../IVZPlacedControlBase/IVZPlacedControlBase.htm)


           IVZLegendBase


## Комментарии


Данный интерфейс является базовым для следующих интерфейсов:


	- [IVZColorLegendBase](../IVZColorLegendBase/IVZColorLegendBase.htm);


	- [IVZSizeLegend](../IVZSizeLegend/IVZSizeLegend.htm).


Для определения параметров легенды визуализатора
 используйте свойство [IVZLegends.Item](../IVZLegends/IVZLegends.Item.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [DefaultFormat](IVZLegendBase.DefaultFormat.htm)


		 Свойство DefaultFormat
		 определяет формат чисел легенды.


		 ![](../../Property_Image.gif)
		 [Footer](IVZLegendBase.Footer.htm)


		 Свойство Footer определяет
		 нижний колонтитул легенды.


		 ![](../../Property_Image.gif)
		 [FooterLine](IVZLegendBase.FooterLine.htm)


		 Свойство FooterLine определяет
		 объект для работы с линией нижнего колонтитула легенды визуализатора.


		 ![](../../Property_Image.gif)
		 [Header](IVZLegendBase.Header.htm)


		 Свойство Header определяет
		 заголовок легенды.


		 ![](../../Property_Image.gif)
		 [HeaderLine](IVZLegendBase.HeaderLine.htm)


		 Свойство HeaderLine
		 определяет объект для работы с линией верхнего колонтитула легенды
		 визуализатора.


		 ![](../../Property_Image.gif)
		 [IsDependent](IVZLegendBase.IsDependent.htm)


		 Свойство IsDependent
		 определяет зависимость легенды от данных.


		 ![](../../Property_Image.gif)
		 [IsOverlap](IVZLegendBase.IsOverlap.htm)


		 Свойство IsOverlap
		 определяет признак того, что легенда не уменьшает рабочую область
		 визуализатора.
