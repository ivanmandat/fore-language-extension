# IVZMultiplyDataSource.GetAllLevelDataDictionary


## Синтаксис


GetAllLevelDataDictionary(attrId: String; level:
 Integer): [IVZDataDictionary](../IVZDataDictionary/IVZDataDictionary.htm);


## Параметры


attrId. Идентификатор атрибута;


level. Уровень временной шкалы.


## Описание


Метод GetAllLevelDataDictionary
 возвращает все данные для заданного атрибута на заданном уровне временной
 шкалы.


## Комментарии


Для получения всех данных заданного элемента используйте метод [IVZMultiplyDataSource.GetAllObjectData](IVZMultiplyDataSource.GetAllObjectData.htm).
