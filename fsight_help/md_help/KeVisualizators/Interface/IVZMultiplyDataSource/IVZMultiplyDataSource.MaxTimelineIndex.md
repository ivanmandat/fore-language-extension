# IVZMultiplyDataSource.MaxTimelineIndex


## Синтаксис


MaxTimelineIndex: Integer;


## Описание


Свойство MaxTimelineIndex возвращает
 максимальную вместимость измерений.


## Комментарии


Для получения идентификатора источника данных визуализатора используйте
 свойство [IVZMultiplyDataSource.ID](IVZMultiplyDataSource.ID.htm).
