# IVZMultiplyDataSource.GetAllObjectData


## Синтаксис


GetAllObjectData(objectId: String; attrId: String):
 [IVZDataArray](../IVZDataArray/IVZDataArray.htm);


## Параметры


objectId. Идентификатор элемента из
 измерения объектов наблюдения;


attrId. Идентификатор атрибута.


## Описание


Метод GetAllObjectData возвращает
 все данные для элемента и атрибута по всей временной шкале.


## Комментарии


Для получения всех данных для заданного атрибута на заданном уровне
 временной шкалы используйте метод [IVZMultiplyDataSource.GetAllLevelDataDictionary](IVZMultiplyDataSource.GetAllLevelDataDictionary.htm).
