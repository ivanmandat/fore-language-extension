# IVZMultiplyDataSource.GetDataEx


## Синтаксис


GetDataEx(objectId: String, attrId: String, valIndx:
 Integer): Variant;


## Параметры


objectId. Идентификатор измерения;


attrId. Идентификатор показателя;


valIndx. Индекс элемента.


## Описание


Метод GetDataEx возвращает данные
 элемента заданного измерения по его индексу и идентификатору показателя.


## Комментарии


Для получения данных с атрибутом по умолчанию используйте метод [IVZMultiplyDataSource.GetData](IVZMultiplyDataSource.GetData.htm).
