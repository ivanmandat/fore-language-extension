# IVZMultiplyDataSource.GetAllLevelData


## Синтаксис


GetAllLevelData(level: Integer): [IVZDataArray](../IVZDataArray/IVZDataArray.htm);


## Параметры


level. Уровень временной шкалы.


## Описание


Метод GetAllLevelData возвращает
 все данные атрибута для заданного уровня временной шкалы.


## Комментарии


Если уровень временной шкалы равен «-1», метод возвращает данные по
 всей временной шкале.
