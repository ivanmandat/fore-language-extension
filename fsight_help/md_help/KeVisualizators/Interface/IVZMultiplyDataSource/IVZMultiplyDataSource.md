# IVZMultiplyDataSource


Сборка: Visualizators;


## Описание


Интерфейс IVZMultiplyDataSource
 предназначен для работы с множественным источником данных визуализатора.


## Иерархия наследования


           IVZMultiplyDataSource


## Комментарии


Для определения параметров множественного источника данных используйте
 свойство [IVZDataSourceMapping.DataSource](../IVZDataSourceMapping/IVZDataSourceMapping.DataSource.htm).

## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [GetAllLevelData](IVZMultiplyDataSource.GetAllLevelData.htm)


		 Метод GetAllLevelData
		 возвращает все данные атрибута для заданного уровня временной
		 шкалы.


		 ![](../../Sub_Image.gif)
		 [GetAllLevelDataDictionary](IVZMultiplyDataSource.GetAllLevelDataDictionary.htm)


		 Метод GetAllLevelDataDictionary
		 возвращает все данные для заданного атрибута на заданном уровне
		 временной шкалы.


		 ![](../../Sub_Image.gif)
		 [GetAllLevelDataEx](IVZMultiplyDataSource.GetAllLevelDataEx.htm)


		 Метод GetAllLevelDataEx
		 возвращает все данные атрибута для заданного показателя и уровня
		 временной шкалы.


		 ![](../../Sub_Image.gif)
		 [GetAllObjectData](IVZMultiplyDataSource.GetAllObjectData.htm)


		 Метод GetAllObjectData
		 возвращает все данные для элемента и атрибута по всей временной
		 шкале.


		 ![](../../Sub_Image.gif)
		 [GetAttributes](IVZMultiplyDataSource.GetAttributes.htm)


		 Метод GetAttributes
		 возвращает идентификаторы всех атрибутов источников данных.


		 ![](../../Sub_Image.gif)
		 [GetData](IVZMultiplyDataSource.GetData.htm)


		 Метод GetData возвращает
		 данные элемента заданного измерения по его индексу.


		 ![](../../Sub_Image.gif)
		 [GetDataEx](IVZMultiplyDataSource.GetDataEx.htm)


		 Метод GetDataEx возвращает
		 данные элемента заданного измерения по его индексу и идентификатору
		 показателя.


		 ![](../../Sub_Image.gif)
		 [GetMaxValue](IVZMultiplyDataSource.GetMaxValue.htm)


		 Метод GetMaxValue
		 возвращает максимальное значение заданной метрики по индексу временной
		 шкалы.


		 ![](../../Sub_Image.gif)
		 [GetMinValue](IVZMultiplyDataSource.GetMinValue.htm)


		 Метод GetMinValue
		 возвращает минимальное значение заданной метрики по индексу временной
		 шкалы.


		 ![](../../Sub_Image.gif)
		 [GetObjects](IVZMultiplyDataSource.GetObjects.htm)


		 Метод GetObjects возвращает
		 идентификаторы всех элементов источника данных.
