# IVZMapChartShapes


Сборка: Visualizators;


## Описание


Интерфейс IVZMapChartShapes
 предназначен для работы с коллекцией областей слоя карты.


## Иерархия наследования


IVZMapChartShapes


## Комментарии


Доступ к коллекции областей предоставляет свойство IVZMapChartLayer.Shapes.


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [AllKeys](IVZMapChartShapes.AllKeys.htm)
		 Свойство AllKeys возвращает
		 массив ключей областей слоя карты.


		 ![](../../Property_Image.gif)
		 [Count](IVZMapChartShapes.Count.htm)
		 Свойство Count возвращает
		 количество областей слоя карты в коллекции.


		 ![](../../Property_Image.gif)
		 [ItemByKey](IVZMapChartShapes.ItemByKey.htm)
		 Свойство ItemByKey
		 возвращает настройки области слоя карты по ключу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IVZMapChartShapes.Add.htm)
		 Метод Add добавляет
		 в коллекцию указанную область слоя карты.


		 ![](../../Sub_Image.gif)
		 [Clear](IVZMapChartShapes.Clear.htm)
		 Метод Clear очищает
		 текущую коллекцию областей слоя карты.


		 ![](../../Sub_Image.gif)
		 [RemoveByKey](IVZMapChartShapes.RemoveByKey.htm)
		 Метод RemoveByKey удаляет
		 из коллекции область с указанным ключом.
