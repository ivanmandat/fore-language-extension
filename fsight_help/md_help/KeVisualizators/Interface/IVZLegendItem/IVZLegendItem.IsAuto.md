# IVZLegendItem.IsAuto


## Синтаксис


IsAuto: Boolean;


## Описание


Свойство IsAuto определяет будет
 ли отображаться пункт легенды визуализатора.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Пункт легенды визуализатора отображается в легенде;


	- False. Пункт легенды
	 визуализатора не отображается.
