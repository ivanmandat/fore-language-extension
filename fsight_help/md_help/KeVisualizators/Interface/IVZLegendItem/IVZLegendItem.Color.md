# IVZLegendItem.Color


## Синтаксис


Color: Variant;


## Описание


Свойство Color определяет цвет
 для пункта легенды визуализатора.


## Комментарии


Для определения текста для пункта легенды визуализатора используйте
 [IVZLegendItem.Text](IVZLegendItem.Text.htm).
