# IVZLegendItem


Сборка: Visualizators;


## Описание


Интерфейс IVZLegendItem используется
 для работы с пунктом легенды визуализатора.


## Иерархия наследования


           IVZLegendItem


## Комментарии


Для получения объекта данного типа используйте [IVZLegendItems.Item](../IVZLegendItems/IVZLegendItems.Item.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Color](IVZLegendItem.Color.htm)
		 Свойство Color определяет
		 цвет для пункта легенды визуализатора.


		 ![](../../Property_Image.gif)
		 [IsAuto](IVZLegendItem.IsAuto.htm)
		 Свойство IsAuto определяет
		 отображение пункта в легенде визуализатора.


		 ![](../../Property_Image.gif)
		 [Marker](IVZLegendItem.Marker.htm)
		 Свойство Marker определяет
		 фигуру или изображение, используемое в качестве маркера легенды.


		 ![](../../Property_Image.gif)
		 [Text](IVZLegendItem.Text.htm)
		 Свойство Text определяет
		 текст для пункта легенды визуализатора.
