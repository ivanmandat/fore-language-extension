# IVZLegendItem.Text


## Синтаксис


Text: String;


## Описание


Свойство Text определяет текст
 для пункта легенды визуализатора.


## Комментарии


Для определения цвета пункта легенды визуализатора используйте [IVZLegendItem.Color](IVZLegendItem.Color.htm).
