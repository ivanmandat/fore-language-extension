# IVZMapChartArrow.StartShapeId


## Синтаксис


StartShapeId: String;


## Описание


Свойство StartShapeId определяет
 идентификатор территории, из которой выходит стрелка.


## Комментарии


Для определения идентификатора территории, в которую входит стрелка
 используйте [IVZMapChartArrow.EndShapeId](IVZMapChartArrow.EndShapeId.htm).
