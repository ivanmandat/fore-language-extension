# IVZMapChartArrow.EndShapeId


## Синтаксис


EndShapeId: String;


## Описание


Свойство EndShapeId определяет
 идентификатор территории, в которую входит стрелка.


## Комментарии


Для определения идентификатора территории, из которой выходит стрелка
 используйте [IVZMapChartArrow.StartShapeId](IVZMapChartArrow.StartShapeId.htm).
