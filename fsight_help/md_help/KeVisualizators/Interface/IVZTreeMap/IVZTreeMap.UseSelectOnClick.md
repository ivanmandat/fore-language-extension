# IVZTreeMap.UseSelectOnClick


## Синтаксис


UseSelectOnClick: Boolean;


## Описание


Свойство UseSelectOnClick определяет,
 используется ли щелчок кнопкой мыши для выделения родительских элементов.


## Комментарии


Допустимые значения:


	- True. Щелчок мыши используется
	 для выделения родительских элементов;


	- False. Щелчок мыши не
	 используется для выделения родительских элементов;
