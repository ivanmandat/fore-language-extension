# IVZTreeMap.CaptionHeight


## Синтаксис


CaptionHeight: Double;


## Описание


Свойство CaptionHeight определяет
 высоту области заголовков ветвей [плоского дерева](IVZTreeMap.htm).


## Комментарии


Значение высоты указывается в пикселях.
