# IVZTreeMap.RenderMode


## Синтаксис


RenderMode: [VisualizatorTreeMapRenderMode](../../Enums/VisualizatorTreeMapRenderMode.htm);


## Описание


Свойство RenderMode определяет
 режим визуализации плоского дерева.


## Комментарии


Данное свойство актуально, если свойство [IVZTreeMap.View3D](IVZTreeMap.View3D.htm)
 принимает значение true.
