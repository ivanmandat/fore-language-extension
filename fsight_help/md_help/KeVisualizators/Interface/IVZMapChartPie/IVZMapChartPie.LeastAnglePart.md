# IVZMapChartPie.LeastAnglePart


## Синтаксис


LeastAnglePart: Double;


## Описание


Свойство LeastAnglePart определяет
 наименьший угол сектора [кругового
 показателя](UiMaps.chm::/Configure/Diagrams.htm)
 на карте.


## Комментарии


Допустимые значения находятся в диапазоне [1, 180].
