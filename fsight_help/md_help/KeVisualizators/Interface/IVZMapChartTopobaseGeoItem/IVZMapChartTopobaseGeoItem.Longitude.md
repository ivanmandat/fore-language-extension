# IVZMapChartTopobaseGeoItem.Longitude


## Синтаксис


Longitude: Double;


## Описание


Свойство Longitude определяет
 значение географической долготы.


## Комментарии


Географическая долгота - одна из координат, которыми определяется положение
 места на Земле. Значение задается в градусах. Диапазон значений [-180; 180].


Восточные долготы принято считать положительными, западные - отрицательными.
