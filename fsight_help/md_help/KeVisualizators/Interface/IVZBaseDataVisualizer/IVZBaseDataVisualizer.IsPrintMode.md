# IVZBaseDataVisualizer.IsPrintMode


## Синтаксис


IsPrintMode: Boolean;


## Описание


Свойство IsPrintMode определяет,
 активен ли режим печати.


## Комментарии


Допустимые значения:


	- True. Режим печати активен;


	- False. По умолчанию.
	 Режим печати не активен.
