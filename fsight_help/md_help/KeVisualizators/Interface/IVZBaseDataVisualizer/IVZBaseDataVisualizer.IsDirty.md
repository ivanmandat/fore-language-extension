# IVZBaseDataVisualizer.IsDirty


## Синтаксис


IsDirty: Boolean;


## Описание


Свойство IsDirty определяет,
 присутствуют ли изменения в визуализаторе.


## Комментарии


Допустимые значения:


	- True. Изменения в визуализаторе
	 присутствуют. Перед закрытием отчета будет предложено сохранить изменения,
	 даже, если никаких изменений не производилось;


	- False. Изменения в визуализаторе
	 отсутствуют. Перед закрытием отчета будет предложено сохранить изменения,
	 только, если они производились.
