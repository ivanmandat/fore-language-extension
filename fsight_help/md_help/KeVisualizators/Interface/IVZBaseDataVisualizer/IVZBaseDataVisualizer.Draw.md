# IVZBaseDataVisualizer.Draw


## Синтаксис


Draw(Graphics: [IGxGraphics](ModDrawing.chm::/Interface/IGxGraphics/IGxGraphics.htm);

    RectF: [IGxRectF](ModDrawing.chm::/Interface/IGxRectF/IGxRectF.htm);

    Dpi: Double;

    BlackAndWhite: Boolean);


## Параметры


Graphics. Графический объект;


RectF. Прямоугольная область
 графического объекта;


Dpi. Разрешающая способность
 (пикселей на дюйм);


BlackAndWhite. Определяет,
 будет ли рисунок черно-белым;


## Описание


Метод Draw отрисовывает
 визуализатор на графический объект.


## Комментарии


Для получения растрового изображения визуализатора используйте [IVZBaseDataVisualizer.GetBitmap](IVZBaseDataVisualizer.GetBitmap.htm).
