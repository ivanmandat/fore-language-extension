# IVZBaseDataVisualizer.EnableAnimating


## Синтаксис


EnableAnimating: Boolean;


## Описание


Свойство EnableAnimating определяет
 включена ли анимация визуализатора.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Анимация включена;


	- False. Анимация выключена.
