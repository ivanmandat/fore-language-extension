# IVZBaseDataVisualizer.GetBitmap


## Синтаксис


GetBitmap(Width: Integer; Height: Integer; Dpi:
 Double): [IGxBitmap](ModDrawing.chm::/Interface/IGxBitmap/IGxBitmap.htm);


## Параметры


Width. Количество пикселей
 по горизонтали;


Height. Количество пикселей
 по вертикали;


Dpi. Разрешающая способность
 (количество пикселей на дюйм).


## Описание


Метод GetBitmap получает растровое
 изображение визуализатора.


## Комментарии


Для отрисовки визуализатора на графический объект используйте [IVZBaseDataVisualizer.Draw](IVZBaseDataVisualizer.Draw.htm).
