# IVZMapChart.ArrowEffectsMode


## Синтаксис


ArrowEffectsMode: [MapChartArrowEffectsMode](../../Enums/MapChartArrowEffectsMode.htm);


## Описание


Свойство ArrowEffectsMode определяет
 режим подсветки стрелок карты.


## Комментарии


Если свойство принимает значение:


	- [MapChartArrowEffectsMode.StartShape](../../Enums/MapChartArrowEffectsMode.htm),
	 то при наведении курсора мыши подсвечиваются стрелки, выходящие из
	 региона;


	- [MapChartArrowEffectsMode.EndShape](../../Enums/MapChartArrowEffectsMode.htm),
	 то при наведении курсора мыши подсвечиваются стрелки, заходящие в
	 регион.
