# IVZLegends.Remove


## Синтаксис


Remove(Index: Integer);


## Параметры


Index. Индекс легенды визуализатора.


## Описание


Метод Remove удаляет легенду
 визуализатора по ее индексу.


## Комментарии


Для добавления легенды визуализатора используйте [IVZLegends.Add](IVZLegends.Add.htm).
