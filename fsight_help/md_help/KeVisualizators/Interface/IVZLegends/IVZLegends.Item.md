# IVZLegends.Item


## Синтаксис


Item(Index: Integer): [IVZLegendBase](../IVZLegendBase/IVZLegendBase.htm);


## Параметры


Index. Индекс легенды в коллекции.


## Описание


Свойство Item определяет легенду в коллекции легенд визуализатора по её индексу.
