# IVZLegends


Сборка: Visualizators;


## Описание


Интерфейс IVZLegends предназначен
 для работы с коллекцией легенд визуализатора.


## Иерархия наследования


IVZLegends


## Комментарии


Коллекция шкал визуализатора может быть получена в свойстве [IVZDataVisualizer.Legends](../IVZDataVisualizer/IVZDataVisualizer.Legends.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IVZLegends.Count.htm)


		 Свойство Count определяет
		 количество легенд в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IVZLegends.Item.htm)


		 Свойство Item определяет
		 легенду в коллекции легенд визуализатора по её индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IVZLegends.Add.htm)


		 Метод Add добавляет
		 указанную легенду в коллекцию легенд визуализатора.


		 ![](../../Sub_Image.gif)
		 [Clear](IVZLegends.Clear.htm)


		 Метод Clear полностью
		 очищает коллекцию легенд визуализатора.


		 ![](../../Sub_Image.gif)
		 [Remove](IVZLegends.Remove.htm)


		 Метод Remove удаляет
		 легенду визуализатора по ее индексу.
