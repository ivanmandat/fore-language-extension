# IVZTreeDataVisualizer


Сборка: Visualizators;


## Описание


Интерфейс IVZTreeDataVisualizer
 предназначен для работы с визуализаторами с древовидной структурой.


## Иерархия наследования


           [IVZDataVisualizer](../IVZDataVisualizer/IVZDataVisualizer.htm)


           IVZTreeDataVisualizer


## Комментарии


Данный интерфейс является базовым для всех визуализаторов с древовидной
 структурой:


	- [IVZBubbleTree](../IVZBubbleTree/IVZBubbleTree.htm);


	- [IVZTreeMap](../IVZTreeMap/IVZTreeMap.htm).
