# IVZMapChartShape


Сборка: Visualizators;


## Описание


Интерфейс IVZMapChartShape предназначен
 для работы с областью слоя карты.


## Иерархия наследования


           [IVZMapChartBaseObject](../IVZMapChartBaseObject/IVZMapChartBaseObject.htm)


           IVZMapChartShape


## Комментарии


Для настройки слоя карты используйте свойства интерфейса [IVZMapChartLayer](../IVZMapChartLayer/IVZMapChartLayer.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Background](IVZMapChartShape.Background.htm)
		 Свойство Background
		 определяет кисть для заливки фона области слоя карты.


		 ![](../../Property_Image.gif)
		 [Border](IVZMapChartShape.Border.htm)
		 Свойство Border
		 определяет параметры пера для границы области слоя карты.


		 ![](../../Property_Image.gif)
		 Effects
		 Свойство Effects определяет
		 эффекты для области слоя карты.


		 ![](../../Property_Image.gif)
		 Height
		 Свойство Height определяет
		 высоту области слоя карты.


		 ![](../../Property_Image.gif)
		 [IsStrokedShape](IVZMapChartShape.IsStrokedShape.htm)
		 Свойство IsStrokedShape
		 определяет режим отображения области слоя карты с контуром и без
		 заливки фона.


		 ![](../../Property_Image.gif)
		 [Name](IVZMapChartShape.Name.htm)
		 Свойство Name определяет
		 наименование области слоя карты.


		 ![](../../Property_Image.gif)
		 [ParentLayer](IVZMapChartShape.ParentLayer.htm)
		 Свойство ParentLayer
		 определяет родительский слой карты.


		 ![](../../Property_Image.gif)
		 [Selected](IVZMapChartShape.Selected.htm)
		 Свойство Selected определяет,
		 будет ли отмечена область карты.


		 ![](../../Property_Image.gif)
		 [ShapeId](IVZMapChartShape.ShapeId.htm)
		 Свойство ShapeId определяет
		 идентификатор области слоя карты.


		 ![](../../Property_Image.gif)
		 ShowVisuals
		 Свойство ShowVisuals
		 определяет отображение всплывающей подсказки при наведении на
		 область слоя карты.


		 ![](../../Property_Image.gif)
		 SphereData
		 Свойство SphereData
		 определяет параметры сферической карты.


		 ![](../../Property_Image.gif)
		 TriangulateData
		 Свойство TriangulateData
		 определяет параметры триангулированной карты.
