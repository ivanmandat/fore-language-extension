# IVZMapChartShape.Border


## Синтаксис


Border: [IGxPenBase](ModDrawing.chm::/Interface/IGxPenBase/IGxPenBase.htm);


## Описание


Свойство Border определяет
 параметры пера для границы области слоя карты.


## Комментарии


Данное свойство применимо только для двухмерной карты. Изменение параметров
 пера для трехмерной карты недоступно.
