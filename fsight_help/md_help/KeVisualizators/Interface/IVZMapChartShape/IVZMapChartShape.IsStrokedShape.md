# IVZMapChartShape.IsStrokedShape


## Синтаксис


IsStrokedShape: Boolean;


## Описание


Свойство IsStrokedShape определяет
 режим отображения области слоя карты с контуром и без заливки фона.


## Комментарии


Допустимые значения:


	- True. Область слоя карты
	 отображается с контуром и без заливки фона;


	- False. Область слоя
	 карты отображается с контуром и c заливкой фона по умолчанию.
