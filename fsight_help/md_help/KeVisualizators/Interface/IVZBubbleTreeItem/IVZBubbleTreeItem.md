# IVZBubbleTreeItem


Сборка: Visualizators;


## Описание


Интерфейс IVZBubbleTreeItem
 предназначен для работы с элементами [пузырькового
 дерева](../IVZBubbleTree/IVZBubbleTree.htm).


## Иерархия наследования


           [IVZTreeItem](../IVZTreeItem/IVZTreeItem.htm)


           IVZBubbleTreeItem


## Комментарии


Параметры элемента пузырькового дерева могут быть получены в свойствах
 [IVZTreeModel.ActiveItem](../IVZTreeModel/IVZTreeModel.ActiveItem.htm)
 и [IVZTreeModel.RootItem](../IVZTreeModel/IVZTreeModel.RootItem.htm)
 после приведения к типу IVZBubbleTreeItem.
