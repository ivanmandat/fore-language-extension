# IVZDataDictionary.Remove


## Синтаксис


Remove(Value: String);


## Параметры


Value. Ключ удаляемого элемента.


## Описание


Метод Remove удаляет элемент
 словаря по указанному ключу.


## Комментарии


Для удаления всех элементов словаря используйте метод [IVZDataDictionary.Clear](IVZDataDictionary.Clear.htm).
