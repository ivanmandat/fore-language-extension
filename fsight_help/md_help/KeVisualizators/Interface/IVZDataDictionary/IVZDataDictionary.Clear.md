# IVZDataDictionary.Clear


## Синтаксис


Clear;


## Описание


Метод Clear удаляет все элементы
 словаря.


## Комментарии


Для удаления элемента словаря по его ключу используйте метод [IVZDataDictionary.Remove](IVZDataDictionary.Remove.htm).
