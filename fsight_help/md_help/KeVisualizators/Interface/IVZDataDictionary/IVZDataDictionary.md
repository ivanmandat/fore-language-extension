# IVZDataDictionary


Сборка: Visualizators;


## Описание


Интерфейс IVZDataDictionary
 предназначен для работы со словарем данных.


## Иерархия наследования


           IVZDataDictionary


## Комментарии


Для работы с массивом данных используйте интерфейс [IVZDataArray](../IVZDataArray/IVZDataArray.htm).
