# IVZDataDictionary.Item


## Синтаксис


Item(Key: String): Variant;


## Параметры


Key. Ключ элемента, который
 требуется определить.


## Описание


Свойство Item определяет элемент
 словаря по его ключу.


## Комментарии


Для получения количества элементов в словаре используйте свойство [IVZDataDictionary.Count](IVZDataDictionary.Count.htm).


## Пример


Использование свойства приведено в примере для метода [IVZDataDictionary.Add](IVZDataDictionary.Add.htm).
