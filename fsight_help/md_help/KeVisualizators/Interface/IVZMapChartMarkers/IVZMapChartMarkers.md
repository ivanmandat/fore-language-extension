# IVZMapChartMarkers


Сборка: Visualizators;


## Описание


Интерфейс IVZMapChartMarkers
 используется для работы с коллекцией маркеров геолокации.


## Иерархия наследования


           IVZMapChartMarkers


## Комментарии


Маркер геолокации - это точка, привязанная к территории. Маркеры позволяют
 отображать на карте места, необходимые пользователю, например: расположение
 магазинов или станций метро.


Для получения объекта данного типа используйте [IVZMapChart.Markers](../IVZMapChart/IVZMapChart.Markers.htm).

## Методы


		 Имя метода
