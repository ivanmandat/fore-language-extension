# IVZMapChartMarkers.InsertAt


## Синтаксис


InsertAt(Index: Integer; Value: [IVZMapChartMarker](../IVZMapChartMarker/IVZMapChartMarker.htm));


## Параметры


Index. Индекс позиции, в которую
 необходимо вставить маркер геолокации;


Value. Маркер геолокации.


## Описание


Метод InsertAt вставляет указанный
 маркер геолокации в указанную позицию в коллекции.


## Комментарии


Для добавления маркера геолокации в коллекцию используйте [IVZMapChartMarkers.Add](IVZMapChartMarkers.Add.htm).
