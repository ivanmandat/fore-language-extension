# IVZMapChartMarkers.Add


## Синтаксис


Add(Value: [IVZMapChartMarker](../IVZMapChartMarker/IVZMapChartMarker.htm)):
 Integer;


## Параметры


Value. Маркер геолокации.


## Описание


Метод Add добавляет маркер геолокации
 в коллекцию.


## Комментарии


Для удаления маркера геолокации из коллекции используйте [IVZMapChartMarkers.Remove](IVZMapChartMarkers.Remove.htm).
