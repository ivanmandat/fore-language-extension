# IVZMapChartMarkers.Remove


## Синтаксис


Remove(Index: Integer);


## Параметры


Index. Индекс маркера геолокации.


## Описание


Метод Remove удаляет маркер
 геолокации из коллекции по его индексу.


## Комментарии


Для добавления маркера геолокации в коллекцию используйте [IVZMapChartMarkers.Add](IVZMapChartMarkers.Add.htm).
