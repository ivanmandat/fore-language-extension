# IVZBubbleTree.LabelAutoColor


## Синтаксис


LabelAutoColor: Boolean;


## Описание


Свойство LabelAutoColor определяет,
 используется ли стандартный цвет шрифта всплывающих подсказок.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Используется стандартный цвет шрифта;


	- False. Стандартный
	 цвет шрифта не используется. Цвет задается пользователем вручную.
