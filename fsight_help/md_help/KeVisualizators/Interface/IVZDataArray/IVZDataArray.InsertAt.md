# IVZDataArray.InsertAt


## Синтаксис


InsertAt(Index: Integer, Value: Variant);


## Параметры


Index. Индекс добавляемого элемента;


Value. Значение добавляемого элемента.


## Описание


Метод InsertAt добавляет новый элемент в массив данных по указанному индексу.
