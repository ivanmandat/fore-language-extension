# IVZChartSerie


Сборка: Visualizators;


## Описание


Интерфейс IVZChartSerie предназначен
 для работы с рядом данных [пузырьковой
 диаграммы](UiVisualizators.chm::/BubbleChart/BubbleChart.htm).


## Иерархия наследования


           [IVZChartObject](../IVZChartObject/IVZChartObject.htm)


           IVZChartSerie


## Комментарии


Параметры ряда данных пузырьковой диаграммы могут быть получены с помощью
 свойств [IVZChartSeries.Item](../IVZChartSeries/IVZChartSeries.Item.htm),
 [IVZChartSeries.ItemById](../IVZChartSeries/IVZChartSeries.ItemById.htm).
