# IVZDataScale.AutoCalculable


## Синтаксис


AutoCalculable: Boolean;


## Описание


Свойство AutoCalculable определяет
 признак того, что шаги шкалы будут рассчитаны автоматически.


## Комментарии


Значение свойства равно True,
 если шаги шкалы рассчитываются автоматически, и False
 - в противном случае.


По умолчанию используется значение True.


При значении True для расчета
 шкалы в соответствии с выбранным методом используйте [IVZDataScale.ConstructScaleElements](IVZDataScale.ConstructScaleElements.htm).
