# IVZDataScale.Greater


## Синтаксис


Greater(Index: Integer, useInactiveItem: Boolean): Variant;


## Параметры


Index. Индекс элемента шкалы;


useInactiveItem. Признак использования неактивного элемента. Если свойство равно значению True, то будет учитываться неактивный элемент, иначе - не будет.


## Описание


Свойство Greater возвращает элемент шкалы визуализатора со значением, большим, чем значение элемента с указанным индексом.


## Комментарии


Свойство возвращает кисть, описываемую интерфейсом [IGxBrush](ModDrawing.chm::/Interface/IGxBrush/IGxBrush.htm), или числовое значение в зависимости от [типа шкалы](IVZDataScale.Type.htm).
