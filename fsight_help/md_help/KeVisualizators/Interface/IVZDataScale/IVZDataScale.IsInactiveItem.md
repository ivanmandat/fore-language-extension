# IVZDataScale.IsInactiveItem


## Синтаксис


IsInactiveItem(Index: Integer): Boolean;


## Параметры


Index. Индекс элемента шкалы.


## Описание


Свойство IsInactiveItem определяет,
 является ли указанный элемент шкалы визуализатора неактивным.


## Комментарии


При значение True указанный
 интервал в легенде неактивный, при значении False
 интервал активный.


По умолчанию используется значение False.


Цвет для неактивного элемента визуализатора задается через свойство
 [IVZDataScale.InactiveItem](IVZDataScale.InactiveItem.htm).
