# IVZDataScale.EnableEquality


## Синтаксис


EnableEquality: Boolean;


## Описание


Свойство EnableEquality определяет
 признак использования строгого равенства в элементах шкалы визуализатора.


## Комментарии


При значении True в шкалу визуализатора
 будут включены строгие равенства, при значении False
 строгих равенств в шкале не будет.


По умолчанию используется значение False.
