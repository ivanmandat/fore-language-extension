# IVZDataScale.Mode


## Синтаксис


Mode: [VisualizatorScaleMode](../../Enums/VisualizatorScaleMode.htm);


## Описание


Свойство Mode определяет режим
 расчета шкалы визуализатора.


## Комментарии


После задания значения режима расчета шкалы необходимо выполнить перерасчет
 шкалы с помощью метода [IVZDataScale.ConstructScaleElements](IVZDataScale.ConstructScaleElements.htm).


Если была включена логарифмическая шкала (ScaleLogarithmic), то задайте
 основание логарифма через свойство [IVZDataScale.LogarithmBase](IVZDataScale.LogarithmBase.htm).
