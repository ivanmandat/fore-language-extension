# IVZDataScale.GetValuesPair


## Синтаксис


GetValuesPair(Index: Integer, Var MinValue: Double, Var MaxValue: Double);


## Параметры


Index. Индекс интервала;


MinValue. Начальное значение интервала;


MaxValue. Конечное значение интервала.


## Описание


Метод GetValuesPair возвращает начальное и конечное значение интервала шкалы визуализатора.
