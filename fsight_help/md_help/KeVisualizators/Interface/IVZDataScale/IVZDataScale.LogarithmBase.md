# IVZDataScale.LogarithmBase


## Синтаксис


LogarithmBase: Double;


## Описание


Свойство LogarithmBase определяет
 основание логарифма для логарифмической шкалы визуализатора.


## Комментарии


Логарифмическая шкала задается с помощью [VisualizatorScaleMode](../../Enums/VisualizatorScaleMode.htm).ScaleLogarithmic
 через свойство [IVZDataScale.Mode](IVZDataScale.Mode.htm).


По умолчанию используется основание равное двум.
