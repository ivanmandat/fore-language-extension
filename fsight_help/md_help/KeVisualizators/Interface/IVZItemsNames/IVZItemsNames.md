# IVZItemsNames


Сборка: Visualizators;


## Описание


Интерфейс IVZItemsNames предназначен
 для работы с коллекцией наименований элементов визуализатора.


## Иерархия наследования


           IVZItemsNames


## Комментарии


Коллекция наименований элементов визуализатора может быть получена в
 следующих свойствах:


	- [IVZBubbleTree.Captions](../IVZBubbleTree/IVZBubbleTree.Captions.htm);


	- [IVZTreeMap.Captions](../IVZTreeMap/IVZTreeMap.Captions.htm).
