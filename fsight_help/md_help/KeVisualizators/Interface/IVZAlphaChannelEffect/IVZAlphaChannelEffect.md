# IVZAlphaChannelEffect


Сборка: Visualizators;


## Описание


Интерфейс IVZAlphaChannelEffect
 предназначен для работы с эффектом прозрачности.


## Иерархия наследования


           [IVZEffect](../IVZEffect/IVZEffect.htm)


           IVZAlphaChannelEffect


## Комментарии


Интерфейс можно использовать для настройки эффектов через [IVZDataVisualizer](../IVZDataVisualizer/IVZDataVisualizer.htm)
 (HoverActiveEffects, HoverInactiveEffects, SelectActiveEffects, SelectInactiveEffects)
 и[ IVZMapChartShape](../IVZMapChartShape/IVZMapChartShape.htm)
 (Effects).
