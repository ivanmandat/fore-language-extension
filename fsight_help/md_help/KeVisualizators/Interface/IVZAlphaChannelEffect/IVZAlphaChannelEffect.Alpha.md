# IVZAlphaChannelEffect.Alpha


## Синтаксис


Alpha: Double;


## Описание


Свойство Alpha определяет степень
 прозрачности.


## Комментарии


Допустимые значения находятся в диапазоне [0, 1]: 1 - цвет абсолютно
 непрозрачен, 0 - цвет абсолютно прозрачен.


По умолчанию свойство принимает значение равное 1.
