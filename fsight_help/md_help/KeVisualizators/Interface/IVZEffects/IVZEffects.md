# IVZEffects


Сборка: Visualizators;


## Описание


Интерфейс IVZEffects предназначен
 для работы с коллекцией эффектов ([IVZEffect](../IVZEffect/IVZEffect.htm)),
 использующихся для подсветки элемента управления.


## Иерархия наследования


IVZEffects


## Комментарии


Данный интерфейс используется в [IVZDataVisualizer](../IVZDataVisualizer/IVZDataVisualizer.htm)
 (HoverActiveEffects, HoverInactiveEffects, SelectActiveEffects, SelectInactiveEffects)
 и [IVZMapChartShape](../IVZMapChartShape/IVZMapChartShape.htm)
 (Effects).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IVZEffects.Count.htm)


		 Свойство Count возвращает
		 количество эффектов в коллекции.


		 ![](../../Property_Image.gif)
		 [Item](IVZEffects.Item.htm)


		 Свойство Item возвращает
		 эффект из коллекции эффектов по указанному индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IVZEffects.Add.htm)

		 ![](../../Sub_Image.gif)

		 эффект в указанную позицию в коллекции.


		 ![](../../Sub_Image.gif)
		 [Remove](IVZEffects.Remove.htm)
