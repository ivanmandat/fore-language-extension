# IVZEffects.Item


## Синтаксис


Item(Index: Integer): [IVZEffect](../IVZEffect/IVZEffect.htm);


## Параметры


Index. Индекс эффекта, который
 требуется получить.


## Описание


Свойство Item возвращает эффект из коллекции
 эффектов по указанному индексу.
