# IVZEffects.Remove


## Синтаксис


Remove(Index:
 Integer);


## Параметры


*Index*.
 Индекс эффекта, удаляемого из коллекции.


## Описание


Метод Remove удаляет из коллекции
 эффект по указанному индексу.
