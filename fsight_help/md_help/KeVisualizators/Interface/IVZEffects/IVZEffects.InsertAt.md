# IVZEffects.InsertAt


## Синтаксис


InsertAt(Index: Integer, Value: [IVZEffect](../IVZEffect/IVZEffect.htm));


## Параметры


Index. Индекс позиции, в которую
 необходимо добавить эффект;


Value. Эффект, добавляемый
 в коллекцию.


## Описание


Метод InsertAt добавляет эффект
 в указанную позицию в коллекции.


## Комментарии


При вставке эффекта в позицию происходит сдвиг эффектов, которые находились,
 начиная с позиции вставки, на один шаг вниз.
