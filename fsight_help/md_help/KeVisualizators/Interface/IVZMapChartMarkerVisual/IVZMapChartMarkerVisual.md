# IVZMapChartMarkerVisual


Сборка: Visualizators;


## Описание


Интерфейс IVZMapChartMarkerVisual
 используется для работы с графическим представлением маркера геолокации.


## Иерархия наследования


           [IVZDataVisual](../IVZDataVisual/IVZDataVisual.htm)


           IVZMapChartMarkerVisual


## Комментарии


Маркер геолокации - это точка,
 привязанная к территории. Маркеры позволяют отображать на карте места,
 необходимые пользователю, например: расположение магазинов или станций
 метро.


Для получения графического представления маркера геолокации используйте
 [IVZMapChartMarker.MarkerVisual](../IVZMapChartMarker/IVZMapChartMarker.MarkerVisual.htm).
