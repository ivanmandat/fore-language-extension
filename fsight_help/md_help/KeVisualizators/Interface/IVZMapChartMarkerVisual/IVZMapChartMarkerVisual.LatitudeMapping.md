# IVZMapChartMarkerVisual.LatitudeMapping


## Синтаксис


LatitudeMapping: [IVZDataSourceMapping](../IVZDataSourceMapping/IVZDataSourceMapping.htm);


## Описание


Свойство LatitudeMapping возвращает
 объект для настройки сопоставления данных географической широте маркера
 геолокации на карте.


## Комментарии


Для получения настройки сопоставления данных географической долготе
 маркера геолокации на карте используйте [IVZMapChartMarkerVisual.LongitudeMapping](IVZMapChartMarkerVisual.LongitudeMapping.htm).


Для получения настройки сопоставления данных цвету маркера геолокации
 на карте используйте [IVZMapChartMarkerVisual.BackgroundMapping](IVZMapChartMarkerVisual.BackgroundMapping.htm).
