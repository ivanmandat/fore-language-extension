# IVZMapChartArrows.Remove


## Синтаксис


Remove(Index: Integer);


## Параметры


Index.
 Индекс стрелки.


## Описание


Метод Remove удаляет стрелку
 из коллекции по ее индексу.


## Комментарии


Для очищения коллекции стрелок используйте [IVZMapChartArrows.Clear](IVZMapChartArrows.Clear.htm).


Для добавления стрелки в коллекцию используйте [IVZMapChartArrows.Add](IVZMapChartArrows.Add.htm).
