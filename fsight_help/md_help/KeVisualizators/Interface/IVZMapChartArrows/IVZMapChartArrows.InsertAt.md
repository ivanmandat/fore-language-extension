# IVZMapChartArrows.InsertAt


## Синтаксис


InsertAt(Index: Integer; Value: [IVZMapChartArrow](../IVZMapChartArrow/IVZMapChartArrow.htm));


## Параметры


Index. Индекс позиции, в которую
 необходимо вставить стрелку;


Value. Стрелка.


## Описание


Метод InsertAt вставляет указанную
 стрелку в указанную позицию в коллекции.


## Комментарии


Для добавления стрелки в коллекцию используйте [IVZMapChartArrows.Add](IVZMapChartArrows.Add.htm).
