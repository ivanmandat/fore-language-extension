# IVZStringArray.Remove


## Синтаксис


Remove(Index: Integer);


## Параметры


Index. Индекс удаляемого элемента.


## Описание


Метод Remove удаляет элемент
 массива по указанному индексу.


## Комментарии


Для удаления всех элементов массива используйте метод [IVZStringArray.Clear](IVZStringArray.Clear.htm).
