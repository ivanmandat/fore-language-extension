# IVZStringArray


Сборка: Visualizators;


## Описание


Интерфейс IVZStringArray
 предназначен для работы с массивом строковых переменных.


## Иерархия наследования


           IVZStringArray


## Комментарии


Для работы с массивом вещественных чисел с двоичной точностью используйте
 интерфейс [IVZDoubleArray](../IVZDoubleArray/IVZDoubleArray.htm).
