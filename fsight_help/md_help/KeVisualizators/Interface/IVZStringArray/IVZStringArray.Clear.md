# IVZStringArray.Clear


## Синтаксис


Clear;


## Описание


Метод Clear удаляет все элементы
 массива.


## Комментарии


Для удаления определенного элемента массива используйте метод [IVZStringArray.Remove](IVZStringArray.Remove.htm).
