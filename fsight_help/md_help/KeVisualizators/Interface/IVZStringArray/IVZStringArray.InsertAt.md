# IVZStringArray.InsertAt


## Синтаксис


InsertAt(Index: Integer, Value: String);


## Параметры


Index. Индекс добавляемого
 элемента;


Value. Значение добавляемого
 элемента.


## Описание


Метод InsertAt добавляет новый
 элемент в массив по указанному индексу.


## Комментарии


Для удаления элемента массива по его индексу используйте метод [IVZStringArray.Remove](IVZStringArray.Remove.htm).
