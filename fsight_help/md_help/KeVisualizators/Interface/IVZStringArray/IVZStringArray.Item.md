# IVZStringArray.Item


## Синтаксис


Item(Index: Integer): String;


## Параметры


Index. Индекс элемента, который
 требуется определить.


## Описание


Свойство Item определяет элемент
 массива по его индексу и возвращает его ключ.


## Комментарии


Для получения количества элементов в массиве используйте свойство [IVZStringArray.Count](IVZStringArray.Count.htm).
