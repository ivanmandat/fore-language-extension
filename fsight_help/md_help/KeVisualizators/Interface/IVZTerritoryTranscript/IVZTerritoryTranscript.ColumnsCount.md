# IVZTerritoryTranscript.ColumnsCount


## Синтаксис


ColumnsCount: Integer;


## Описание


Свойство ColumnsCount определяет
 количество столбцов в расшифровке территорий.


## Комментарии


Для определения максимального количества элементов в расшифровке территорий
 используйте [IVZTerritoryTranscript.MaxItemsCount](IVZTerritoryTranscript.MaxItemsCount.htm).
