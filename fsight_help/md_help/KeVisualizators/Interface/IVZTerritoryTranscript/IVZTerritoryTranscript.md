# IVZTerritoryTranscript


Сборка: Visualizators;


## Описание


Интерфейс IVZTerritoryTranscript
 используется для работы с расшифровкой территорий.


## Иерархия наследования


           [IVZPlacedControlBase](../IVZPlacedControlBase/IVZPlacedControlBase.htm)


           IVZTerritoryTranscript


## Комментарии


Для включения расшифровки территорий на карте воспользуйтесь унаследованным
 свойством [IVZPlacedControlBase.Visible](../IVZPlacedControlBase/IVZPlacedControlBase.Visible.htm).
