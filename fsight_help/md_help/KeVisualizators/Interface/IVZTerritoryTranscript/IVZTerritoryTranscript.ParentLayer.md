## Синтаксис

## Описание


Свойство ParentLayer определяет
 родительский слой для расшифровки территорий.


## Комментарии


Для определения стиля элементов в расшифровке территорий используйте
 [IVZTerritoryTranscript.ItemsStyle](IVZTerritoryTranscript.ItemsStyle.htm).
