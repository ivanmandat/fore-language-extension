# IVZTerritoryTranscript.MaxItemsCount


## Синтаксис


MaxItemsCount: Integer;


## Описание


Свойство MaxItemsCount определяет
 максимальное количество элементов в расшифровке территорий.


## Комментарии


Для определения количества столбцов в расшифровке территорий используйте
 [IVZTerritoryTranscript.ColumnsCount](IVZTerritoryTranscript.ColumnsCount.htm).
