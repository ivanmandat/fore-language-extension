# IVZTerritoryTranscript.IsOverlap


## Синтаксис


IsOverlap: Boolean;


## Описание


Свойство IsOverlap определяет,
 не уменьшает ли расшифровка территорий масштаб карты.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Расшифровка территорий не уменьшает масштаб карты;


	- False. Расшифровка территорий
	 уменьшает масштаб карты.
