# IVZMapChartBaseObject.Visible


## Синтаксис


Visible: Boolean;


## Описание


Свойство Visible определяет,
 будет ли отображаться объект на карте.


## Комментарии


Допустимые значения:


	- True. По умолчанию.
	 Объект будет отображаться;


	- False. Объект отображаться
	 не будет.


К объектам карты относятся: слои, регионы, стрелки, показатели, маркеры
 геолокации.
