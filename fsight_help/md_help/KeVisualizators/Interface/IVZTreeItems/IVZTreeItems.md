# IVZTreeItems


Сборка: Visualizators;


## Описание


Интерфейс IVZTreeItems предназначен
 для работы с коллекцией элементов дерева визуализатора.


## Иерархия наследования


IVZTreeItems


## Комментарии


Коллекция элементов дерева визуализатора может быть получена в свойстве
 [IVZTreeItem.Items](../IVZTreeItem/IVZTreeItem.Items.htm).


## Свойства


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [Count](IVZTreeItems.Count.htm)


		 Свойство Count возвращает
		 размер коллекции элементов пузырькового дерева.


		 ![](../../Property_Image.gif)
		 [Item](IVZTreeItems.Item.htm)


		 Свойство Item возвращает
		 элемент из коллекции элементов дерева по указанному индексу.


## Методы


		 Имя метода
		 Краткое описание


		 ![](../../Sub_Image.gif)
		 [Add](IVZTreeItems.Add.htm)


		 Метод Add добавляет
		 указанный элемент в коллекцию элементов дерева визуализатора.


		 ![](../../Sub_Image.gif)
		 [Clear](IVZTreeItems.Clear.htm)


		 Метод Clear очищает
		 коллекцию элементов дерева визуализатора.


		 ![](../../Sub_Image.gif)
		 [Remove](IVZTreeItems.Remove.htm)


		 Метод Remove удаляет
		 из коллекции элемент дерева по указанному индексу.
