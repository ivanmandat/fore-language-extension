# IVZTreeItems.Add


## Синтаксис


Add(Value: [IVZTreeItem](../IVZTreeItem/IVZTreeItem.htm)):
 Integer;


## Параметры


Value. Элемент дерева, добавляемый
 в коллекцию.


## Описание


Метод Add добавляет указанный
 элемент в коллекцию элементов дерева визуализатора.


## Комментарии


Метод возвращает индекс добавленного в коллекцию элемента.
