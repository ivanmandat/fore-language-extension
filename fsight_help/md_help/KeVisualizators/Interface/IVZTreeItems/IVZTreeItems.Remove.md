# IVZTreeItems.Remove


## Синтаксис


Remove(Index: Integer);


## Параметры


Index. Индекс элемента дерева, удаляемого из коллекции.


## Описание


Метод Remove удаляет из коллекции элемент дерева по указанному индексу.
