# IVZIntervalsLegendMarker


Сборка: Visualizators;


## Описание


Интерфейс IVZIntervalsLegendMarker
 используется для работы с маркером легенды.


## Иерархия наследования


           IVZIntervalsLegendMarker


## Комментарии


Маркер легенды - это фигура
 или изображение, расположенная перед расшифровкой значений, например:


![](../IVZColorLegendBase/IVZColorLegendBase_Scale.png)
