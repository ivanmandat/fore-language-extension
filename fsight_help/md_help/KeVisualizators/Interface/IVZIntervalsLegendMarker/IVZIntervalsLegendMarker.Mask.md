# IVZIntervalsLegendMarker.Mask


## Синтаксис


Mask: IVZResourceImage;


## Описание


Свойство Mask определяет объект
 для работы с маской растрового изображения, который используется в качестве
 маркера легенды.


## Комментарии


Для определения объекта для работы с растровым изображением, использующегося
 в качестве маркера легенды, используйте [IVZIntervalsLegendMarker.Image](IVZIntervalsLegendMarker.Image.htm).


Примечание.
 Используйте растровые изображения, размеры которых равны по ширине и высоте.
