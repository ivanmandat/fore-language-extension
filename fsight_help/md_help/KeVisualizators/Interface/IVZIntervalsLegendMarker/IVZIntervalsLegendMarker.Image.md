# IVZIntervalsLegendMarker.Image


## Синтаксис


Image: IVZResourceImage;


## Описание


Свойство Image определяет объект
 для работы с растровым изображением, который используется в качестве маркера
 легенды.


## Комментарии


Для определения объекта для работы с маской растрового изображения,
 использующегося в качестве маркера легенды, используйте [IVZIntervalsLegendMarker.Mask](IVZIntervalsLegendMarker.Mask.htm).


Примечание.
 Используйте растровые изображения, размеры которых равны по ширине и высоте.
