# IVZMapChartBarAxis.Visibility


## Синтаксис


Visibility: [MapChartBarAxisVisibility](../../Enums/MapChartBarAxisVisibility.htm);


## Описание


Свойство Visibility определяет
 тип видимости оси [столбикового
 показателя](UiMaps.chm::/Configure/Diagrams.htm)
 на слое карты.


## Комментарии


Для определения базовых настроек пера оси [столбикового показателя](UiMaps.chm::/Configure/Diagrams.htm)
 на слое карты используйте [IVZMapChartBarAxis.Pen](IVZMapChartBarAxis.Pen.htm).
