# IVZMapChartBarAxis.Pen


## Синтаксис


Pen: [IGxPenBase](ModDrawing.chm::/Interface/IGxPenBase/IGxPenBase.htm);


## Описание


Свойство Pen определяет базовые
 настройки пера оси [столбикового
 показателя](UiMaps.chm::/Configure/Diagrams.htm)
 на слое карты.


## Комментарии


Для определения типа видимости оси [столбикового показателя](UiMaps.chm::/Configure/Diagrams.htm)
 на слое карты используйте [IVZMapChartBarAxis.Visibility](IVZMapChartBarAxis.Visibility.htm).
