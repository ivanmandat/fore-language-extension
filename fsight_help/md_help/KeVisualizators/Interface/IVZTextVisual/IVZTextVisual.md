# IVZTextVisual


Сборка: Visualizators;


## Описание


Интерфейс TextVisual предназначен
 для сопоставления данных визуализатора текстовым значениям.


## Иерархия наследования


           [IVZDataVisual](../IVZDataVisual/IVZDataVisual.htm)


           IVZTextVisual


## Комментарии


Настройки для сопоставления данных визуализатора текстовым значениям
 могут быть получены в следующих свойствах:


	- [IVZBubbleTree.TextVisual](../IVZBubbleTree/IVZBubbleTree.TextVisual.htm);


	- [IVZTreeMap.TextVisual](../IVZTreeMap/IVZTreeMap.TextVisual.htm).
