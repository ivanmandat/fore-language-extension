# IVZMapChartTopobaseMetaInfo.GeoRightTop


## Синтаксис


GeoRightTop: [IVZMapChartTopobaseGeoItem](../IVZMapChartTopobaseGeoItem/IVZMapChartTopobaseGeoItem.htm);


## Описание


Свойство GeoRightTop определяет
 правую верхнюю точку привязки топоосновы к географическим координатам.


## Комментарии


Для определения левой нижней точки привязки топоосновы к географическим
 координатам используйте [IVZMapChartTopobaseMetaInfo.GeoLeftBottom](IVZMapChartTopobaseMetaInfo.GeoLeftBottom.htm)
