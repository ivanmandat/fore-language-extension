# IVZMapChartTopobaseMetaInfo


Сборка: Visualizators;


## Описание


Интерфейс IVZMapChartTopobaseMetaInfo
 используется для работы с параметрами привязки топоосновы к географическим
 координатам.


## Иерархия наследования


           IVZMapChartTopobaseMetaInfo


## Комментарии


Топооснова - набор графических слоев информации, наложенных друг на
 друга.


Для получения объекта данного типа используйте IVZMapChartTopobase.Meta.
