# IVZMapChartTopobaseMetaInfo.GeoLeftBottom


## Синтаксис


GeoLeftBottom: [IVZMapChartTopobaseGeoItem](../IVZMapChartTopobaseGeoItem/IVZMapChartTopobaseGeoItem.htm);


## Описание


Свойство GeoLeftBottom определяет
 левую нижнюю точку привязки топоосновы к географическим координатам.


## Комментарии


Для определения правой верхней точки привязки топоосновы к географическим
 координатам используйте [IVZMapChartTopobaseMetaInfo.GeoRightTop](IVZMapChartTopobaseMetaInfo.GeoRightTop.htm).
