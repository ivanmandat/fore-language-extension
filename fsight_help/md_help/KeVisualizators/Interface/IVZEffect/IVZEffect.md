# IVZEffect


Сборка: Visualizators;


## Описание


Интерфейс IVZEffect предназначен
 для работы с эффектом, использующимся для подсветки элемента управления.


## Иерархия наследования


IVZEffect


## Комментарии


Данный интерфейс не имеет собственных свойств и методов.


Данный интерфейс является базовым для следующих интерфейсов:


	- [IVZBrushEffect](../IVZBrushEffect/IVZBrushEffect.htm);


	- [IVZBorderEffect](../IVZBorderEffect/IVZBorderEffect.htm);


	- [IVZAlphaChannelEffect](../IVZAlphaChannelEffect/IVZAlphaChannelEffect.htm);


	- [IVZHighlightEffect](../IVZHighlightEffect/IVZHighlightEffect.htm).


Для работы с коллекцией эффектов используйте интерфейс [IVZEffects](../IVZEffects/IVZEffects.htm).
