# IVZTreeMapItem


Сборка: Visualizators;


## Описание


Интерфейс IVZTreeMapItem предназначен
 для работы с элементом [плоского
 дерева](../IVZTreeMap/IVZTreeMap.htm).


## Иерархия наследования


           [IVZTreeItem](../IVZTreeItem/IVZTreeItem.htm)


           IVZTreeMapItem


## Свойства, унаследованные от [IVZTreeItem](../IVZTreeItem/IVZTreeItem.htm)


		 Имя свойства
		 Краткое описание


		 ![](../../Property_Image.gif)
		 [ID](../IVZTreeItem/IVZTreeItem.ID.htm)
		 Свойство ID вовзращает
		 идентификатор элемента дерева визуализатора.


		 ![](../../Property_Image.gif)
		 [Items](../IVZTreeItem/IVZTreeItem.Items.htm)
		 Свойство Items определяет
		 дочерние элементы у заданного элемента дерева визуализатора.


		 ![](../../Property_Image.gif)
		 [Parent](../IVZTreeItem/IVZTreeItem.Parent.htm)
		 Свойство Parent возвращает
		 родителя для заданного элемента дерева визуализатора.


		 ![](../../Property_Image.gif)
		 [Selected](../IVZTreeItem/IVZTreeItem.Selected.htm)
		 Свойство Selected определяет,
		 используется ли выделение области дерева визуализатора.
