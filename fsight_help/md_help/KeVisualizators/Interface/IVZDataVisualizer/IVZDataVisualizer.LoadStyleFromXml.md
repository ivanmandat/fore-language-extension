# IVZDataVisualizer.LoadStyleFromXml


## Синтаксис


LoadStyleFromXml(Value: String);


## Параметры


Value. Параметры стиля визуализатора в XML-формате.


## Описание


Метод LoadStyleFromXml загружает XML-параметры стиля визуализатора.
