# IVZDataVisualizer.SelectInactiveEffects


## Синтаксис


SelectInactiveEffects: [IVZEffects](../IVZEffects/IVZEffects.htm);


## Описание


Свойство SelectInactiveEffects
 определяет эффекты, которые применяются для неактивных элементов, которые
 не попали в выделение.


## Комментарии


Для настройки эффектов используйте свойства интерфейсов [IVZBrushEffect](../IVZBrushEffect/IVZBrushEffect.htm),
 [IVZBorderEffect](../IVZBorderEffect/IVZBorderEffect.htm),
 [IVZAlphaChannelEffect](../IVZAlphaChannelEffect/IVZAlphaChannelEffect.htm).
